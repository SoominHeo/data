<p><b>ALZip</b> is an <a href="/wiki/File_archiver" title="File archiver">archive</a> and <a href="/wiki/Data_compression" title="Data compression">compression</a> utility from <a href="/wiki/ESTsoft" title="ESTsoft">ESTsoft</a> for <a href="/wiki/Microsoft_Windows" title="Microsoft Windows">Microsoft Windows</a>. ALZip is a file compression program that can unzip 40 different zip file archives. ALZip can zip files into 8 different archives such as ZIP, EGG, TAR and others. In ALZip version 8, ALZip’s own EGG format, that Unicode and other powerful function supported, can be used.</p>

<h2><span class="mw-headline" id="History">History</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=ALZip&amp;action=edit&amp;section=1" title="Edit section: History">edit</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>Originally ALZip was developed in 1999 as an internal application at the <a href="/wiki/South_Korea" title="South Korea">South Korean</a> <a class="mw-redirect" href="/wiki/Software_company" title="Software company">software company</a> ESTsoft in response to employee frustration with using the English interface in <a href="/wiki/WinZip" title="WinZip">WinZip</a>.</p>
<p>The Korean interface was immediately well received<sup class="reference" id="cite_ref-rel1999c_1-0"><a href="#cite_note-rel1999c-1">[1]</a></sup> and later that year ALZip was publicly released<sup class="reference" id="cite_ref-rel1999a_2-0"><a href="#cite_note-rel1999a-2">[2]</a></sup><sup class="reference" id="cite_ref-rel1999b_3-0"><a href="#cite_note-rel1999b-3">[3]</a></sup><sup class="reference" id="cite_ref-rel2003_4-0"><a href="#cite_note-rel2003-4">[4]</a></sup> as <a href="/wiki/Freeware" title="Freeware">freeware</a>.<sup class="reference" id="cite_ref-rel200211d_5-0"><a href="#cite_note-rel200211d-5">[5]</a></sup> In just over a year ALZip became the most popular zip program in Korea<sup class="reference" id="cite_ref-rel1999e_6-0"><a href="#cite_note-rel1999e-6">[6]</a></sup> and by December 2001 was the top downloaded software<sup class="reference" id="cite_ref-rel2001a_7-0"><a href="#cite_note-rel2001a-7">[7]</a></sup><sup class="reference" id="cite_ref-rel2001b_8-0"><a href="#cite_note-rel2001b-8">[8]</a></sup> in the country. By 2004 it had reached a 70% <a href="/wiki/Market_share" title="Market share">market share</a> in South Korea.<sup class="reference" id="cite_ref-9"><a href="#cite_note-9">[9]</a></sup> The popularity of the software and the <a class="new" href="/w/index.php?title=ALZ_(file_format)&amp;action=edit&amp;redlink=1" title="ALZ (file format) (page does not exist)">ALZ</a> archive format played a large role in earning ESTsoft a place in the Digital Innovation Awards Top 100 Companies in South Korea<sup class="reference" id="cite_ref-rel2001d_10-0"><a href="#cite_note-rel2001d-10">[10]</a></sup> with the software taking special mention.</p>
<p>Early in its development ALZip introduced the ALZ file format to deal with file size limitations of the <a class="mw-redirect" href="/wiki/ZIP_(file_format)" title="ZIP (file format)">ZIP</a> file format. ALZ compression has no theoretical upper limit for <a href="/wiki/File_size" title="File size">file sizes</a>, as they are only limited by <a href="/wiki/Operating_system" title="Operating system">operating system</a> limits or the amount of available <a class="mw-redirect" href="/wiki/Computer_storage" title="Computer storage">storage</a>. The file format has become common in Korea.</p>
<p>The first English version was released in 2002. Since then, support for over 20 languages has been added.</p>
<h2><span class="mw-headline" id="Licensing_changes">Licensing changes</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=ALZip&amp;action=edit&amp;section=2" title="Edit section: Licensing changes">edit</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>Originally ALZip was released as pure <a href="/wiki/Freeware" title="Freeware">freeware</a>, however in October 2001 changed to free for home use, with government usage requiring a <a href="/wiki/Software_license" title="Software license">software license</a>. In April 2002, business license was also introduced,<sup class="reference" id="cite_ref-rel2003_4-1"><a href="#cite_note-rel2003-4">[4]</a></sup> commercial usage.<sup class="reference" id="cite_ref-ref200306a_11-0"><a href="#cite_note-ref200306a-11">[11]</a></sup> Licensing was based on the <a href="/wiki/Honor_system" title="Honor system">honor system</a>, and there were no <a class="mw-redirect" href="/wiki/Nag_screen" title="Nag screen">nag screens</a>.</p>
<p>With the version 7 release, the license was changed from freeware to <a href="/wiki/Adware" title="Adware">adware</a>, displaying downloaded banner ads, and finally beginning 1 December 2008, all new releases are <a href="/wiki/Shareware" title="Shareware">shareware</a> requiring a paid license<sup class="reference" id="cite_ref-12"><a href="#cite_note-12">[12]</a></sup> except in its original Korean language, which requires no licensing fee.</p>
<p>With the version 8.51, from 22 August 2012 ALZip has changed the license the product that was paid is licensed free of charge. On the download page of ALZip is a serial number to copy and paste when you start AlZip, in order to have the license of the software free of charge.the news is available in this site <a class="external free" href="http://www.altools.com/Home/News.aspx?&amp;articleIdx=59&amp;m=3" rel="nofollow">http://www.altools.com/Home/News.aspx?&amp;articleIdx=59&amp;m=3</a></p>
<h2><span class="mw-headline" id="Naming_and_mascots">Naming and mascots</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=ALZip&amp;action=edit&amp;section=3" title="Edit section: Naming and mascots">edit</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>The name "ALZip" was chosen, as the "AL" part is a transliteration from the Korean "aljip" (<a href="/wiki/Hangul" title="Hangul">Hangul</a>: 알집), literally <i>EggZip</i>. Other ALTools feature similar egghead cartoon characters as mascots for each program.</p>
<p>Additionally, ALZip integrates into the "New Folder" function of <a class="mw-redirect" href="/wiki/Windows_Explorer" title="Windows Explorer">Windows Explorer</a>, where new folders are created with the option of using custom icons and names.<sup class="reference" id="cite_ref-rel2003_4-2"><a href="#cite_note-rel2003-4">[4]</a></sup></p>
<h2><span class="mw-headline" id="Issues_on_file_format">Issues on file format</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=ALZip&amp;action=edit&amp;section=4" title="Edit section: Issues on file format">edit</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>In 2003, there was a controversy over ALZip's own compression file format ALZ, when a developer of another Korean compression utility, "빵집" insisted that ALZip users were forced to use the ALZip archiver when decompressing ALZ files. He also insisted that ESTsoft does not offer decompression <a class="mw-redirect" href="/wiki/Library_(software)" title="Library (software)">libraries</a>, and third-party archiver developers have used <a href="/wiki/Reverse_engineering" title="Reverse engineering">reverse engineering</a> to develop their own algorithms.<sup class="reference" id="cite_ref-13"><a href="#cite_note-13">[13]</a></sup></p>
<p>The current version of "빵집" supports ALZ format.<sup class="reference" id="cite_ref-14"><a href="#cite_note-14">[14]</a></sup> and, in ALZip 8.0 beta version, ESTsoft made some format. *.EGG</p>
