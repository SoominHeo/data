<h1 class="firstHeading" id="firstHeading" lang="en">Banach algebra</h1>
<div class="mw-body-content" id="bodyContent">
<div id="siteSub">From Wikipedia, the free encyclopedia</div>
<div id="contentSub"></div>
<div class="mw-jump" id="jump-to-nav">
					Jump to:					<a href="#mw-head">navigation</a>, 					<a href="#p-search">search</a>
</div>
<div class="mw-content-ltr" dir="ltr" id="mw-content-text" lang="en"><script>function mfTempOpenSection(id){var block=document.getElementById("mf-section-"+id);block.className+=" open-block";block.previousSibling.className+=" open-block";}</script><p>In <a href="/wiki/Mathematics" title="Mathematics">mathematics</a>, especially <a href="/wiki/Functional_analysis" title="Functional analysis">functional analysis</a>, a <b>Banach algebra</b>, named after <a href="/wiki/Stefan_Banach" title="Stefan Banach">Stefan Banach</a>, is an <a href="/wiki/Associative_algebra" title="Associative algebra">associative algebra</a> <i>A</i> over the <a href="/wiki/Real_number" title="Real number">real</a> or <a href="/wiki/Complex_number" title="Complex number">complex</a> numbers (or over a <a class="mw-redirect" href="/wiki/Nonarchimedean_field" title="Nonarchimedean field">non-Archimedean</a> complete <a class="new" href="/w/index.php?title=Normed_field&amp;action=edit&amp;redlink=1" title="Normed field (page does not exist)">normed field</a>) that at the same time is also a <a href="/wiki/Banach_space" title="Banach space">Banach space</a>, i.e. normed and complete. The algebra multiplication and the Banach space norm are required to be related by the following inequality:</p>

This property is found in the real and complex numbers; for instance, |-6×5| ≤ |-6|×|5|.</p>
<p>If in the above we relax <a href="/wiki/Banach_space" title="Banach space">Banach space</a> to <a class="mw-redirect" href="/wiki/Normed_space" title="Normed space">normed space</a> the analogous structure is called a <b>normed algebra</b>.</p>
<p>A Banach algebra is called "unital" if it has an <a href="/wiki/Identity_element" title="Identity element">identity element</a> for the multiplication whose norm is 1, and "commutative" if its multiplication is <a class="mw-redirect" href="/wiki/Commutative" title="Commutative">commutative</a>. However, this is not the case all the time. For example, one cannot define all the trigonometric functions in a Banach algebra without identity.</p>
<p>The theory of real Banach algebras can be very different from the theory of complex Banach algebras. For example, the <a class="mw-redirect" href="/wiki/Spectrum_of_an_operator" title="Spectrum of an operator">spectrum</a> of an element of a nontrivial complex Banach algebra can never be empty, whereas in a real Banach algebra it could be empty for some elements.</p>
<p>Banach algebras can also be defined over fields of <a href="/wiki/P-adic_number" title="P-adic number">p-adic numbers</a>. This is part of <a href="/wiki/P-adic_analysis" title="P-adic analysis">p-adic analysis</a>.</p>

<p>The prototypical example of a Banach algebra is <span><span class="mwe-math-mathml-inline mwe-math-mathml-a11y" style="display: none;"><math xmlns="http://www.w3.org/1998/Math/MathML">

More generally, every C*-algebra is a Banach algebra.</p>
<ul>
<li>The set of real (or complex) numbers is a Banach algebra with norm given by the <a href="/wiki/Absolute_value" title="Absolute value">absolute value</a>.</li>
<li>The set of all real or complex <i>n</i>-by-<i>n</i> <a href="/wiki/Matrix_(mathematics)" title="Matrix (mathematics)">matrices</a> becomes a <a class="mw-redirect" href="/wiki/Unital_algebra" title="Unital algebra">unital</a> Banach algebra if we equip it with a sub-multiplicative <a href="/wiki/Matrix_norm" title="Matrix norm">matrix norm</a>.</li>
<li>Take the Banach space <b>R</b><sup><i>n</i></sup> (or <b>C</b><sup><i>n</i></sup>) with norm ||<i>x</i>|| = max |<i>x</i><sub><i>i</i></sub>| and define multiplication componentwise: (<i>x</i><sub>1</sub>,...,<i>x</i><sub><i>n</i></sub>)(<i>y</i><sub>1</sub>,...,<i>y</i><sub><i>n</i></sub>) = (<i>x</i><sub>1</sub><i>y</i><sub>1</sub>,...,<i>x</i><sub><i>n</i></sub><i>y</i><sub><i>n</i></sub>).</li>
<li>The <a href="/wiki/Quaternion" title="Quaternion">quaternions</a> form a 4-dimensional real Banach algebra, with the norm being given by the absolute value of quaternions.</li>
<li>The algebra of all bounded real- or complex-valued functions defined on some set (with pointwise multiplication and the <a class="mw-redirect" href="/wiki/Supremum" title="Supremum">supremum</a> norm) is a unital Banach algebra.</li>
<li>The algebra of all bounded <a class="mw-redirect" href="/wiki/Continuous_function_(topology)" title="Continuous function (topology)">continuous</a> real- or complex-valued functions on some <a href="/wiki/Locally_compact_space" title="Locally compact space">locally compact space</a> (again with pointwise operations and supremum norm) is a Banach algebra.</li>
<li>The algebra of all <a class="mw-redirect" href="/wiki/Continuous_function_(topology)" title="Continuous function (topology)">continuous</a> <a class="mw-redirect" href="/wiki/Linear_transformation" title="Linear transformation">linear</a> operators on a Banach space E (with functional composition as multiplication and the <a href="/wiki/Operator_norm" title="Operator norm">operator norm</a> as norm) is a unital Banach algebra. The set of all <a href="/wiki/Compact_operator" title="Compact operator">compact operators</a> on E is a closed ideal in this algebra.</li>
<li>If <i>G</i> is a <a class="mw-redirect" href="/wiki/Locally_compact" title="Locally compact">locally compact</a> <a href="/wiki/Hausdorff_space" title="Hausdorff space">Hausdorff</a> <a href="/wiki/Topological_group" title="Topological group">topological group</a> and μ its <a href="/wiki/Haar_measure" title="Haar measure">Haar measure</a>, then the Banach space L<sup>1</sup>(<i>G</i>) of all μ-integrable functions on <i>G</i> becomes a Banach algebra under the <a href="/wiki/Convolution" title="Convolution">convolution</a> <i>xy</i>(<i>g</i>) = ∫ <i>x</i>(<i>h</i>) <i>y</i>(<i>h</i><sup>−1</sup><i>g</i>) dμ(<i>h</i>) for <i>x</i>, <i>y</i> in L<sup>1</sup>(<i>G</i>).</li>
<li><a href="/wiki/Uniform_algebra" title="Uniform algebra">Uniform algebra</a>: A Banach algebra that is a subalgebra of the complex algebra C(X) with the supremum norm and that contains the constants and separates the points of X (which must be a compact Hausdorff space).</li>
<li><a href="/wiki/Uniform_algebra" title="Uniform algebra">Natural Banach function algebra</a>: A uniform algebra whose all characters are evaluations at points of X.</li>
<li><a href="/wiki/C*-algebra" title="C*-algebra">C*-algebra</a>: A Banach algebra that is a closed *-subalgebra of the algebra of bounded operators on some <a href="/wiki/Hilbert_space" title="Hilbert space">Hilbert space</a>.</li>
<li><a href="/wiki/Measure_algebra" title="Measure algebra">Measure algebra</a>: A Banach algebra consisting of all <a href="/wiki/Radon_measure" title="Radon measure">Radon measures</a> on some <a href="/wiki/Locally_compact_group" title="Locally compact group">locally compact group</a>, where the product of two measures is given by <a href="/wiki/Convolution" title="Convolution">convolution</a>.</li>
</ul>
<h2><span class="mw-headline" id="Properties">Properties</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Banach_algebra&amp;action=edit&amp;section=2" title="Edit section: Properties">edit</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>Several <a class="mw-redirect" href="/wiki/List_of_functions" title="List of functions">elementary functions</a> which are defined via <a href="/wiki/Power_series" title="Power series">power series</a> may be defined in any unital Banach algebra; examples include the <a href="/wiki/Exponential_function" title="Exponential function">exponential function</a> and the <a class="mw-redirect" href="/wiki/Trigonometric_function" title="Trigonometric function">trigonometric functions</a>, and more generally any <a href="/wiki/Entire_function" title="Entire function">entire function</a>. (In particular, the exponential map can be used to define <a href="/wiki/Abstract_index_group" title="Abstract index group">abstract index groups</a>.) The formula for the <a href="/wiki/Geometric_series" title="Geometric series">geometric series</a> remains valid in general unital Banach algebras. The <a href="/wiki/Binomial_theorem" title="Binomial theorem">binomial theorem</a> also holds for two commuting elements of a Banach algebra.</p>
<p>The set of <a class="mw-redirect" href="/wiki/Invertible_element" title="Invertible element">invertible elements</a> in any unital Banach algebra is an <a href="/wiki/Open_set" title="Open set">open set</a>, and the inversion operation on this set is continuous, (and hence homeomorphism) so that it forms a <a href="/wiki/Topological_group" title="Topological group">topological group</a> under multiplication.</p>
<p>If a Banach algebra has unit <b>1</b>, then <b>1</b> cannot be a <a href="/wiki/Commutator#Ring_theory" title="Commutator">commutator</a>; i.e., <span><span class="mwe-math-mathml-inline mwe-math-mathml-a11y" style="display: none;"><math xmlns="http://www.w3.org/1998/Math/MathML">

This is because <i>xy</i> and <i>yx</i> have the same <a class="mw-redirect" href="/wiki/Spectrum_of_an_operator" title="Spectrum of an operator">spectrum</a> except possibly <i>0</i>.</p>
<p>The various algebras of functions given in the examples above have very different properties from standard examples of algebras such as the reals. For example:</p>
<ul>
<li>Every real Banach algebra which is a <a href="/wiki/Division_algebra" title="Division algebra">division algebra</a> is isomorphic to the reals, the complexes, or the quaternions. Hence, the only complex Banach algebra which is a division algebra is the complexes. (This is known as the <a class="mw-redirect" href="/wiki/Gelfand-Mazur_theorem" title="Gelfand-Mazur theorem">Gelfand-Mazur theorem</a>.)</li>
<li>Every unital real Banach algebra with no <a href="/wiki/Zero_divisor" title="Zero divisor">zero divisors</a>, and in which every <a href="/wiki/Principal_ideal" title="Principal ideal">principal ideal</a> is <a href="/wiki/Closed_set" title="Closed set">closed</a>, is isomorphic to the reals, the complexes, or the quaternions.</li>
<li>Every commutative real unital <a href="/wiki/Noetherian_ring" title="Noetherian ring">Noetherian</a> Banach algebra with no zero divisors is isomorphic to the real or complex numbers.</li>
<li>Every commutative real unital Noetherian Banach algebra (possibly having zero divisors) is finite-dimensional.</li>
<li>Permanently singular elements in Banach algebras are <a class="mw-redirect" href="/wiki/Topological_divisior_of_zero" title="Topological divisior of zero">topological divisors of zero</a>, <i>i.e.</i>, considering extensions <i>B</i> of Banach algebras <i>A</i> some elements that are singular in the given algebra <i>A</i> have a multiplicative inverse element in a Banach algebra extension <i>B</i>. Topological divisors of zero in <i>A</i> are permanently singular in all Banach extension <i>B</i> of <i>A</i>.</li>
</ul>

<p>The norm of a normal element <i>x</i> of a C*-algebra coincides with its spectral radius. This generalizes an analogous fact for normal operators.</p>
<p>Let <i>A</i>  be a complex unital Banach algebra in which every non-zero element <i>x</i> is invertible (a division algebra). For every <i>a</i> ∈ <i>A</i>, there is <i>λ</i> ∈ <b>C</b> such that <i>a</i> − <i>λ</i><b>1</b> is not invertible (because the spectrum of <i>a</i> is not empty) hence <i>a</i> = <i>λ</i><b>1</b> : this algebra <i>A</i> is naturally isomorphic to <b>C</b> (the complex case of the Gelfand-Mazur theorem).</p>

<p>As an algebra, a unital commutative Banach algebra is <a href="/wiki/Semisimple_algebra" title="Semisimple algebra">semisimple</a> (i.e., its <a href="/wiki/Jacobson_radical" title="Jacobson radical">Jacobson radical</a> is zero) if and only if its Gelfand representation has trivial kernel. An important example of such an algebra is a commutative C*-algebra. In fact, when <i>A</i> is a commutative unital C*-algebra, the Gelfand representation is then an isometric *-isomorphism between <i>A</i> and <i>C</i>(Δ(<i>A</i>)) .<sup class="reference" id="cite_ref-2"><a href="#cite_note-2">[2]</a></sup></p>
