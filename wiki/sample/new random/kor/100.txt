<p><b>덴마크 왕세자빈, 몽페사트 백작 부인 메리</b>(<span style="font-size: smaller;"><a href="/wiki/%EB%8D%B4%EB%A7%88%ED%81%AC%EC%96%B4" title="덴마크어">덴마크어</a>:</span> <span lang="da" xml:lang="da">kronprinsesse af Danmark, grevinde af Monpezat; født Mary Elizabeth Donaldson</span>, <a href="/wiki/1972%EB%85%84" title="1972년">1972년</a> <a href="/wiki/2%EC%9B%94_5%EC%9D%BC" title="2월 5일">2월 5일</a> ~ )는 <a href="/wiki/%EB%8D%B4%EB%A7%88%ED%81%AC" title="덴마크">덴마크</a>의 현 <a href="/wiki/%EC%99%95%EC%84%B8%EC%9E%90%EB%B9%88" title="왕세자빈">왕세자빈</a>이다. 결혼하기 이전에 이름은 <b>메리 엘리자베스 도널드슨</b>(<span style="font-size: smaller;"><a href="/wiki/%EC%98%81%EC%96%B4" title="영어">영어</a>:</span> <span lang="en" xml:lang="en">Mary Elizabeth Donaldson</span>)이었다.</p>
<h2><span class="mw-headline" id=".EC.83.9D.EC.95.A0">생애</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%8D%B4%EB%A7%88%ED%81%AC_%EC%99%95%EC%84%B8%EC%9E%90%EB%B9%88_%EB%A9%94%EB%A6%AC&amp;action=edit&amp;section=1" title="부분 편집: 생애">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p><a href="/wiki/%EC%98%A4%EC%8A%A4%ED%8A%B8%EB%A0%88%EC%9D%BC%EB%A6%AC%EC%95%84" title="오스트레일리아">오스트레일리아</a> <a href="/wiki/%ED%83%9C%EC%A6%88%EB%A9%94%EC%9D%B4%EB%8B%88%EC%95%84_%EC%A3%BC" title="태즈메이니아 주">태즈메이니아 주</a> <a href="/wiki/%ED%98%B8%EB%B0%94%ED%8A%B8" title="호바트">호바트</a>의 <a href="/wiki/%EC%8A%A4%EC%BD%94%ED%8B%80%EB%9E%9C%EB%93%9C" title="스코틀랜드">스코틀랜드계</a> 가정에서 태어났다. 그녀의 아버지인 <a class="new" href="/w/index.php?title=%EC%A1%B4_%EB%8F%84%EB%84%90%EB%93%9C%EC%8A%A8&amp;action=edit&amp;redlink=1" title="존 도널드슨 (없는 문서)">존 도널드슨</a>은 공학자로, 딸 메리가 결혼할 당시에는 <a href="/wiki/%EB%8C%80%ED%95%9C%EB%AF%BC%EA%B5%AD" title="대한민국">대한민국</a>의 과학 특성화 대학인 <a href="/wiki/%ED%95%9C%EA%B5%AD%EA%B3%BC%ED%95%99%EA%B8%B0%EC%88%A0%EC%9B%90" title="한국과학기술원">한국과학기술원</a>의 교수로 재직 중이었다. <a href="/wiki/1994%EB%85%84" title="1994년">1994년</a>에 태즈메이니아 대학을 졸업하고 부동산 중개업자가 되었다. <a href="/wiki/2000%EB%85%84" title="2000년">2000년</a> <a href="/wiki/%EC%8B%9C%EB%93%9C%EB%8B%88" title="시드니">시드니</a>에서 열린 <a href="/wiki/2000%EB%85%84_%ED%95%98%EA%B3%84_%EC%98%AC%EB%A6%BC%ED%94%BD" title="2000년 하계 올림픽">하계 올림픽</a>에서 덴마크의 <a href="/wiki/%EB%8D%B4%EB%A7%88%ED%81%AC_%EC%99%95%EC%84%B8%EC%9E%90_%ED%94%84%EB%A0%88%EB%8D%B0%EB%A6%AC%ED%81%AC" title="덴마크 왕세자 프레데리크">프레데리크 왕세자</a>를 만나 연인 관계로 발전하였다. <a href="/wiki/2004%EB%85%84" title="2004년">2004년</a> <a href="/wiki/5%EC%9B%94_14%EC%9D%BC" title="5월 14일">5월 14일</a> 덴마크 의회의 승인을 받은 뒤에 코펜하겐에서 왕세자와 결혼식을 올렸다.</p>
<p><a href="/wiki/2005%EB%85%84" title="2005년">2005년</a> <a href="/wiki/11%EC%9B%94_15%EC%9D%BC" title="11월 15일">11월 15일</a>, 코펜하겐 대학병원에서 장남 크리스티안을 낳았다. <a href="/wiki/2012%EB%85%84" title="2012년">2012년</a> <a href="/wiki/5%EC%9B%94_10%EC%9D%BC" title="5월 10일">5월 10일</a>, 대한민국을 방문하여 전쟁기념관 방문, 국무총리 오찬, 글로벌 녹색성자 서밋(GGGS) 2012행사에 참석, 대통령 오찬에 참석하고, 2012년 여수세계박람회 개막식에 참석하였다.</p>
