<p><b>한일 병합 조약</b>(<small><a class="mw-redirect" href="/wiki/%ED%95%9C%EA%B5%AD_%ED%95%9C%EC%9E%90" title="한국 한자">한국 한자</a>: </small>韓日倂合條約) 또는 <b>한일 병합에 관한 조약</b>(<span style="font-size: smaller;"><a href="/wiki/%EC%9D%BC%EB%B3%B8%EC%96%B4" title="일본어">일본어</a>:</span> <span lang="ja"><span lang="ja"><ruby><rb>韓国併合ニ関スル条約</rb> <rp>(</rp><rt>かんこくへいごうにかんするじょうやく</rt><rp>)</rp></ruby></span></span> <small>칸코쿠헤이고니칸스루조야쿠<sup>[<a href="/wiki/%EC%9C%84%ED%82%A4%EB%B0%B1%EA%B3%BC:%EC%9D%BC%EB%B3%B8%EC%96%B4%EC%9D%98_%ED%95%9C%EA%B8%80_%ED%91%9C%EA%B8%B0" title="위키백과:일본어의 한글 표기">*</a>]</sup></small>)은 <a href="/wiki/1910%EB%85%84" title="1910년">1910년</a> <a href="/wiki/8%EC%9B%94_22%EC%9D%BC" title="8월 22일">8월 22일</a>에 조인되어 <a href="/wiki/8%EC%9B%94_29%EC%9D%BC" title="8월 29일">8월 29일</a> 발효된 <a href="/wiki/%EB%8C%80%ED%95%9C%EC%A0%9C%EA%B5%AD" title="대한제국">대한제국</a>과 <a href="/wiki/%EC%9D%BC%EB%B3%B8_%EC%A0%9C%EA%B5%AD" title="일본 제국">일본 제국</a> 사이에 일방적인 위력에 의해 이루어진 합병조약(合倂條約)이다. <b>한일 합방 조약</b>(<small><a class="mw-redirect" href="/wiki/%ED%95%9C%EA%B5%AD_%ED%95%9C%EC%9E%90" title="한국 한자">한국 한자</a>: </small>韓日合邦条約)이라고도 불린다.<sup class="reference" id="cite_ref-1"><a href="#cite_note-1">[1]</a></sup> 대한제국의 내각총리대신 <a href="/wiki/%EC%9D%B4%EC%99%84%EC%9A%A9" title="이완용">이완용</a>과 제3대 한국 통감인 <a href="/wiki/%EB%8D%B0%EB%9D%BC%EC%9A%B0%EC%B9%98_%EB%A7%88%EC%82%AC%ED%83%80%EC%BC%80" title="데라우치 마사타케">데라우치 마사타케</a>가 형식적인 회의를 거쳐 조약을 통과시켰으며, 조약의 공포는 <a href="/wiki/8%EC%9B%94_29%EC%9D%BC" title="8월 29일">8월 29일</a>에 이루어져 대한제국은 일본 제국의 식민지가 되었다. <a href="/wiki/%ED%95%9C%EA%B5%AD" title="한국">한국</a>에서는 <b>국권피탈</b>(<small><a class="mw-redirect" href="/wiki/%ED%95%9C%EA%B5%AD_%ED%95%9C%EC%9E%90" title="한국 한자">한국 한자</a>: </small>國權被奪), <b>경술국치</b>(<small><a class="mw-redirect" href="/wiki/%ED%95%9C%EA%B5%AD_%ED%95%9C%EC%9E%90" title="한국 한자">한국 한자</a>: </small>庚戌國恥) 등으로 호칭한다.</p>
<p><a href="/wiki/%EC%9D%84%EC%82%AC%EC%A1%B0%EC%95%BD" title="을사조약">을사조약</a> 이후 급격하게 기울던 <a href="/wiki/%EB%8C%80%ED%95%9C%EC%A0%9C%EA%B5%AD" title="대한제국">대한제국</a>은 결국 <a href="/wiki/%EC%9D%BC%EB%B3%B8_%EC%A0%9C%EA%B5%AD" title="일본 제국">일본 제국</a>에 강제 편입되었고, <a href="/wiki/%EC%9D%BC%EC%A0%9C_%EA%B0%95%EC%A0%90%EA%B8%B0" title="일제 강점기">일제 강점기</a>가 시작되었다. 한편 병탄 조약 직후 <a href="/wiki/%ED%99%A9%ED%98%84" title="황현">황현</a>, <a href="/wiki/%EB%AF%BC%EC%98%81%ED%99%98" title="민영환">민영환</a>, <a href="/wiki/%ED%95%9C%EA%B7%9C%EC%84%A4" title="한규설">한규설</a>, <a href="/wiki/%EC%9D%B4%EC%83%81%EC%84%A4" title="이상설">이상설</a> 등 일부 지식인과 관료층은 이에 극렬히 반대하여 자결하거나 독립운동을 전개하였다. 한일 병탄 직후 14만 명이 독립운동에 참여하였다.<sup class="reference" id="cite_ref-shin216_2-0"><a href="#cite_note-shin216-2">[2]</a></sup></p>

한일 병탄 조약시 전권위임장. 관례와는 다르게 순종의 이름(坧)이 서명에 들어갔다. 그러나 坧은 순종의 친필이 아니다.</div>

<a href="/wiki/%EC%B0%BD%EB%8D%95%EA%B6%81" title="창덕궁">창덕궁</a> <a href="/wiki/%EB%8C%80%EC%A1%B0%EC%A0%84" title="대조전">대조전</a>에 있는 흥복헌. 1910년 8월 22일 이곳에서 한일 병합 조약을 찬성하는 마지막 어전회의가 열렸다. 옛 건물은 1917년 화재로 소실되었고 현재 건물을 1920년 중건한 것이다.</div>
</div>
</div>
<p>일본 제국은 병탄의 방침을 <a href="/wiki/1909%EB%85%84" title="1909년">1909년</a> <a href="/wiki/7%EC%9B%94_6%EC%9D%BC" title="7월 6일">7월 6일</a> 내각회의에서 이미 확정해 놓고 있던 상태였다. 다만 부작용을 최소화하고 국제적 명분을 얻는 일만 남겨두었다. 일본 제국 정부는 <a href="/wiki/%EC%9D%BC%EC%A7%84%ED%9A%8C" title="일진회">일진회</a> 고문 <a class="new" href="/w/index.php?title=%EC%8A%A4%EA%B8%B0%EC%95%BC%EB%A7%88_%EC%8B%9C%EA%B2%8C%EB%A7%88%EB%A3%A8&amp;action=edit&amp;redlink=1" title="스기야마 시게마루 (없는 문서)">스기야마 시게마루</a>(杉山茂丸)에게 ‘병합청원’의 시나리오를 준비시키고 있었다. <a href="/wiki/%EC%86%A1%EB%B3%91%EC%A4%80" title="송병준">송병준</a>은 이에 앞서 <a href="/wiki/1909%EB%85%84" title="1909년">1909년</a> <a href="/wiki/2%EC%9B%94" title="2월">2월</a> 일본 제국으로 건너가 매국흥정을 벌였다. 여러 차례 <a href="/wiki/%EC%9D%B4%ED%86%A0_%ED%9E%88%EB%A1%9C%EB%B6%80%EB%AF%B8" title="이토 히로부미">이토 히로부미</a>에게 ‘합병’을 역설한바 있었으나 일본 제국 측의 병탄 계획 때문에 일이 늦어지게 되자 직접 일본 제국으로 건너가서 <a href="/wiki/%EA%B0%80%EC%93%B0%EB%9D%BC_%EB%8B%A4%EB%A1%9C" title="가쓰라 다로">가쓰라 다로</a>(桂太郞) 수상 등 일본 제국의 조야 정객들을 상대로 ‘합병’을 흥정하기에 이른 것이다.</p>
<p>한편 <a href="/wiki/%EC%9D%B4%EC%99%84%EC%9A%A9" title="이완용">이완용</a>은 송병준의 이런 활동을 눈치채고 통감부 외사국장 <a class="new" href="/w/index.php?title=%EA%B3%A0%EB%A7%88%EC%93%B0_%EB%AF%B8%EB%8F%84%EB%A6%AC&amp;action=edit&amp;redlink=1" title="고마쓰 미도리 (없는 문서)">고마쓰 미도리</a>(小松緑)와 조선 병탄 문제의 교섭에 나섰다. 이완용은 일본어를 할 줄 모르기 때문에 일본 제국에 유학했던 <a href="/wiki/%EC%9D%B4%EC%9D%B8%EC%A7%81" title="이인직">이인직</a>을 심복 비서로 삼아 미도리와 교섭에 나서도록 했다. 이 무렵 통감부에서는 이완용 내각을 와해시키고 그와 대립 관계에 있던 송병준으로 하여금 내각을 구성하도록 할 것이라는 소문을 퍼뜨리고 있었다. 두 사람의 충성 경쟁을 부추기려는 전술이었다.</p>
<p>송병준 내각이 성립된다면 보복당할 우려가 있을 뿐만 아니라, 합방의 주역을 빼앗길 것을 두려워한 이완용은 “현 내각이 붕괴되어도 그보다 더 친일적인 내각이 나올 수 없다.”면서 자기 휘하의 내각이 조선 합방 조약을 맺을 수 있음을 자진해서 통감부에 알렸다.</p>
<p>이런 시나리오를 연출하면서 일본 제국은 점차 ‘병탄’의 시기가 무르익어가고 있다고 판단, 시게마루를 내세우고 <a href="/wiki/%EC%9D%B4%EC%9A%A9%EA%B5%AC" title="이용구">이용구</a>·송병준 등을 이용하여 ‘합방청원서’를 만들도록 부추겼다.<sup class="reference" id="cite_ref-3"><a href="#cite_note-3">[3]</a></sup></p>
<p>또한 일본 제국은 조약이 누출되어 조약에 반대하는 소요 등이 일어날 것에 대비하여 나남·청진·함흥·대구 등에 주둔한 일본군을 밤을 틈타 서울로 이동시켰다. 조약 체결일인 8월 22일 응원병력과 용산에 주둔한 제2사단이 경비를 섰다.<sup class="reference" id="cite_ref-4"><a href="#cite_note-4">[4]</a></sup></p>
<p><a href="/wiki/%EC%B0%BD%EB%8D%95%EA%B6%81" title="창덕궁">창덕궁</a> 흥복헌<sup class="reference" id="cite_ref-5"><a href="#cite_note-5">[5]</a></sup>으로 불려온 대신들 중 학부대신 <a href="/wiki/%EC%9D%B4%EC%9A%A9%EC%A7%81" title="이용직">이용직</a>은 조약을 반대하다 쫓겨났고, 이후 이른바 <a href="/wiki/%EA%B2%BD%EC%88%A0%EA%B5%AD%EC%A0%81" title="경술국적">경술국적</a>이라고 불리는 내각총리대신 이완용, 시종원경 <a href="/wiki/%EC%9C%A4%EB%8D%95%EC%98%81" title="윤덕영">윤덕영</a>, 궁내부대신 <a href="/wiki/%EB%AF%BC%EB%B3%91%EC%84%9D" title="민병석">민병석</a>, 탁지부대신 <a href="/wiki/%EA%B3%A0%EC%98%81%ED%9D%AC_(1849%EB%85%84)" title="고영희 (1849년)">고영희</a>, 내부대신 <a href="/wiki/%EB%B0%95%EC%A0%9C%EC%88%9C" title="박제순">박제순</a>, 농상공부대신 <a href="/wiki/%EC%A1%B0%EC%A4%91%EC%9D%91" title="조중응">조중응</a>, 친위부장관 겸 시종무관장 <a href="/wiki/%EC%9D%B4%EB%B3%91%EB%AC%B4" title="이병무">이병무</a>, 승녕부총관 <a href="/wiki/%EC%A1%B0%EB%AF%BC%ED%9D%AC_(1859%EB%85%84)" title="조민희 (1859년)">조민희</a> 8명 친일파 대신은 조약 체결에 찬성, 협조하였다. 이 8명은 한일 병탄 조약 체결 이후 공을 인정받아 <a href="/wiki/%EC%A1%B0%EC%84%A0%EA%B7%80%EC%A1%B1" title="조선귀족">조선귀족</a> <a href="/wiki/%EC%9E%91%EC%9C%84" title="작위">작위</a>를 수여받았다.</p>

<blockquote class="toccolours" style="float:none; padding: 10px 15px 10px 15px; display:table;">
<p>한국 황제 폐하와 일본국 황제 폐하는 두 나라 사이의 특별히 친밀한 관계를 고려하여 상호 행복을 증진시키며 동양의 평화를 영구히 확보하자고 하며 이 목적을 달성하고자 하면 한국을 일본국에 병합하는 것이 낫다는 것을 확신하고 이에 두 나라 사이에 합병 조약을 체결하기로 결정하였다.</p>
<p>이를 위하여 한국 황제 폐하는 내각 총리 대신(內閣總理大臣) <a href="/wiki/%EC%9D%B4%EC%99%84%EC%9A%A9" title="이완용">이완용</a>(李完用)을, 일본 황제 폐하는 통감(統監)인 자작(子爵) <a href="/wiki/%EB%8D%B0%EB%9D%BC%EC%9A%B0%EC%B9%98_%EB%A7%88%EC%82%AC%ED%83%80%EC%BC%80" title="데라우치 마사타케">사내정의</a>(寺內正毅, 데라우치 마사타케)를 각각 그 전권 위원(全權委員)으로 임명하는 동시에 위의 전권 위원들이 공동으로 협의하여 아래에 적은 모든 조항들을 협정하게 한다.</p>
<ol>
<li>한국 황제 폐하는 한국 전체에 관한 일체 통치권을 완전히 또 영구히 일본 황제 폐하에게 양여함.</li>
<li>일본국 황제 폐하는 앞조항에 기재된 양여를 수락하고, 완전히 한국을 일본 제국에 병합하는 것을 승락함.</li>
<li>일본국 황제 폐하는 한국 황제 폐하, 태황제 폐하, 황태자 전하와 그들의 황후, 황비 및 후손들로 하여금 각기 지위를 응하여 적당한 존칭, 위신과 명예를 누리게 하는 동시에 이것을 유지하는데 충분한 세비를 공급함을 약속함.</li>
<li>일본국 황제 폐하는 앞 조항 이외에 한국황족 및 후손에 대해 상당한 명예와 대우를 누리게 하고, 또 이를 유지하기에 필요한 자금을 공여함을 약속함.</li>
<li>일본국 황제 폐하는 공로가 있는 한국인으로서 특별히 표창하는 것이 적당하다고 인정되는 경우에 대하여 영예 작위를 주는 동시에 은금(恩金)을 줌.</li>
<li>일본국 정부는 앞에 기록된 병합의 결과로 완전히 한국의 시정을 위임하여 해당 지역에 시행할 법규를 준수하는 한국인의 신체 및 재산에 대하여 전적인 보호를 제공하고 또 그 복리의 증진을 도모함.</li>
<li>일본국 정부는 성의충실히 새 제도를 존중하는 한국인으로 적당한 자금이 있는 자를 사정이 허락하는 범위에서 한국에 있는 제국 관리에 등용함.</li>
</ol>
<p>본 조약은 한국 황제 폐하와 일본 황제 폐하의 재가를 받은 것이므로 공포일로부터 이를 시행함.</p>
<p>위 증거로 삼아 양 전권위원은 본 조약에 기명 조인함.</p>
<p>융희 4년 8월 22일 내각총리대신 이완용</p>
<p>메이지 43년 8월 22일 통감 자작 데라우치 마사타케</p>

<h2><span class="mw-headline" id=".EB.8C.80.ED.95.9C.EB.AF.BC.EA.B5.AD.EA.B3.BC_.EC.9D.BC.EB.B3.B8.EC.9D.98_.EC.9D.84.EC.82.AC.EC.A1.B0.EC.95.BD_.EB.AC.B4.ED.9A.A8_.EC.9E.AC.ED.99.95.EC.9D.B8">대한민국과 일본의 을사조약 무효 재확인</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%ED%95%9C%EC%9D%BC_%EB%B3%91%ED%95%A9_%EC%A1%B0%EC%95%BD&amp;action=edit&amp;section=3" title="부분 편집: 대한민국과 일본의 을사조약 무효 재확인">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p><a href="/wiki/%EB%8C%80%ED%95%9C%EB%AF%BC%EA%B5%AD" title="대한민국">대한민국</a>과 <a href="/wiki/%EC%9D%BC%EB%B3%B8" title="일본">일본</a>은 <a href="/wiki/1965%EB%85%84" title="1965년">1965년</a> <a href="/wiki/%ED%95%9C%EC%9D%BC%EA%B8%B0%EB%B3%B8%EC%A1%B0%EC%95%BD" title="한일기본조약">한일기본조약</a>에서 한일 병탄 조약을 포함하여 대한제국과 일본 제국 간에 체결된 모든 조약 및 협정이 이미 무효임을 한 번 더 확인하였다.<sup class="reference" id="cite_ref-6"><a href="#cite_note-6">[6]</a></sup> 단, 이에 관한 해석은 양자에 있어서 서로 다르다. 대한민국 측에서는 '체결부터 원천적 무효'임을 주장한 반면, 일본 측에서는 '대한민국 정부 수립으로 인해 현 시점(1965년)에서는 이미 무효'라고 주장한 것이다.<sup class="reference" id="cite_ref-7"><a href="#cite_note-7">[7]</a></sup></p>
<h2><span class="mw-headline" id=".EB.85.BC.EB.9E.80">논란</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%ED%95%9C%EC%9D%BC_%EB%B3%91%ED%95%A9_%EC%A1%B0%EC%95%BD&amp;action=edit&amp;section=4" title="부분 편집: 논란">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<h3><span class="mw-headline" id=".EB.B6.88.EB.B2.95.EB.A1.A0">불법론</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%ED%95%9C%EC%9D%BC_%EB%B3%91%ED%95%A9_%EC%A1%B0%EC%95%BD&amp;action=edit&amp;section=5" title="부분 편집: 불법론">편집</a><span class="mw-editsection-bracket">]</span></span></h3>

<div style="padding-left:1em;"><b><a class="extiw" href="https://ko.wikisource.org/wiki/%EC%A1%B0%EC%84%A0%EC%99%95%EC%A1%B0%EC%8B%A4%EB%A1%9D/%EC%88%9C%EC%A2%85%EC%8B%A4%EB%A1%9D/3%EB%85%84#8.EC.9B.94_29.EC.9D.BC" title="s:조선왕조실록/순종실록/3년">순종 황제 칙유</a></b></div>

<p>대부분의 <a href="/wiki/%EB%8C%80%ED%95%9C%EB%AF%BC%EA%B5%AD" title="대한민국">대한민국</a> 법학자들은 한일 병탄 늑약이 불법이라고 주장한다. 불법론자들은 이 조약에는 <a class="mw-redirect" href="/wiki/%EC%88%9C%EC%A2%85_%ED%99%A9%EC%A0%9C" title="순종 황제">순종 황제</a>의 최종 승인 절차에 결정적인 결함이 있는 것으로 나타났다고 주장한다. 즉 <a href="/wiki/%EC%9D%B4%EC%99%84%EC%9A%A9" title="이완용">이완용</a>에게 전권을 위임한다는 <a href="/wiki/%EB%8C%80%ED%95%9C%EC%A0%9C%EA%B5%AD_%EC%88%9C%EC%A2%85" title="대한제국 순종">순종</a>의 위임장은 강제로 받아낼 수 있었으나 가장 중요한 최종 비준을 받는 절차가 생략되었다는 것이다.</p>
<p>불법론자들은 그 증거는 조약문 안에 고스란히 남아 있다고 주장한다. 조항 제8조에는 '양국 황제의 결재를 받았다'라고 적고 있으나, 조약문의 어떤 내용도 최종 비준 이전에는 효력을 발휘할 수 없다는 점을 상기할 때 재가 사실을 미리 명시하는 것은 상식 밖의 일이라는 주장이다.</p>
<p>또한 병탄을 최종적으로 알리는 조칙에 <a href="/wiki/%EC%98%A5%EC%83%88" title="옥새">옥새</a>는 찍혀 있지만 순종의 서명이 빠졌다는 점이다. 불법론자들은 조칙이 성립하려면 옥새와 함께 서명이 들어가야 하는데, 결국 한일 병탄 조약이 불법적이라는 것은 옥새와 그에 따르는 의전절차가 무시되었다는 것을 통해서도 입증할 수 있다고 주장한다.</p>
<p><a class="mw-redirect" href="/wiki/%EC%84%9C%EC%9A%B8%EB%8C%80" title="서울대">서울대</a> <a href="/wiki/%EC%9D%B4%ED%83%9C%EC%A7%84" title="이태진">이태진</a> 교수는 “한일합방조약을 알리는 황제의 칙유가 일본정부에 의해 작성됐으며, <a href="/wiki/%EB%8C%80%ED%95%9C%EC%A0%9C%EA%B5%AD_%EC%88%9C%EC%A2%85" title="대한제국 순종">순종</a>이 이에 대한 서명을 거부했거나 하지 않은 사실이 자료로 확인됐다”고 주장했다.</p>
<p>이 교수는 그 근거로 <a href="/wiki/8%EC%9B%94_29%EC%9D%BC" title="8월 29일">8월 29일</a> 공포된 황제칙유에는 <a class="new" href="/w/index.php?title=%EB%8C%80%ED%95%9C%EA%B5%AD%EC%83%88&amp;action=edit&amp;redlink=1" title="대한국새 (없는 문서)">대한국새</a>가 아닌 <a href="/wiki/1907%EB%85%84" title="1907년">1907년</a> <a href="/wiki/7%EC%9B%94" title="7월">7월</a>, <a class="mw-redirect" href="/wiki/%EA%B3%A0%EC%A2%85%ED%99%A9%EC%A0%9C" title="고종황제">고종황제</a> 강제 퇴위 때 일본이 빼앗아간 칙명지보가 찍혀 있다는 점을 들었다. 국가간의 조약에는 국새가 찍혀야 하는데, 칙명지보는 행정결제용 옥새이기 때문에 순종의 정식 제가가 이루어졌다고 보기 힘들다는 것이다. 또, <a href="/wiki/1907%EB%85%84" title="1907년">1907년</a> <a href="/wiki/11%EC%9B%94" title="11월">11월</a> 이후 황제의 조칙문에 날인해온 황제의 서명 ‘척(拓)’(순종의 이름)이 빠져 있는 점을 들었다. 당시 <a href="/wiki/%EB%8C%80%ED%95%9C%EC%A0%9C%EA%B5%AD_%EC%88%9C%EC%A2%85" title="대한제국 순종">순종</a>은 일본 제국 측의 병탄에 직면해 전권위원위임장에는 국새를 찍고 서명할 수밖에 없었으나 마지막 비준절차에 해당하는 칙유서명은 완강히 거부했다.</p>
<p>이어서 이태진 교수는 “한일강제합방조약의 법적결함은 결국 국제법상으로만 보아도 조약불성립론을 입증하며 <a href="/wiki/1910%EB%85%84" title="1910년">1910년</a> 이후 한국과 일본의 관계는 식민통치도 아니고 일본이 한국을 불법적으로 강점한 상태”라고 주장했다.<sup class="reference" id="cite_ref-8"><a href="#cite_note-8">[8]</a></sup></p>
<p><a href="/wiki/1965%EB%85%84" title="1965년">1965년</a> <a href="/wiki/%ED%95%9C%EC%9D%BC%EA%B8%B0%EB%B3%B8%EC%A1%B0%EC%95%BD" title="한일기본조약">한일기본조약</a>(대한민국과 일본국간의 기본관계에 관한 조약) 제2조에서는 이 조약의 불법성을 시사하는데, 이는 평화조약을 새로맺는 시점에 무효화된 것이 아니라 조약체결 당시부터 원천무효라고 주장한다.</p>
<h3><span class="mw-headline" id=".ED.95.A9.EB.B2.95.EB.A1.A0">합법론</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%ED%95%9C%EC%9D%BC_%EB%B3%91%ED%95%A9_%EC%A1%B0%EC%95%BD&amp;action=edit&amp;section=6" title="부분 편집: 합법론">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>대부분의 일본 법학자들은 한일 병합 조약을 합법적이라고 주장하고 있다. 그들이 주요한 근거로 내세우는 것은 '조약문 자체에서 형식적인 문제가 없으며, 국제법상 조약에 준수한 조약이라는 것'이다. 즉, 일본제국은 을사조약이 가졌던 여러 가지 부당함을 의식했던 것이지 한일병합조약에는 위임장, 조약문, 황제의 조칙 등 형식적인 문서들이 갖추어져 있기 때문에 한일병합은 불법적인 게 아닌 합법적이라는 것이 주 견해이다.</p>
<p><a href="/wiki/1965%EB%85%84" title="1965년">1965년</a> <a href="/wiki/%ED%95%9C%EC%9D%BC%EA%B8%B0%EB%B3%B8%EC%A1%B0%EC%95%BD" title="한일기본조약">한일기본조약</a>(대한민국과 일본국간의 기본관계에 관한 조약) 제2조에서는 이 조약의 불법성을 시사하는데, 이는 조약 당시부터 원천무효가 아니라, 독립한 한국과 평화조약을 맺는 시점부터 무효화된 것이라고 주장한다.</p>
<h2><span class="mw-headline" id=".EC.B5.9C.EA.B7.BC_.EC.83.81.ED.99.A9">최근 상황</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%ED%95%9C%EC%9D%BC_%EB%B3%91%ED%95%A9_%EC%A1%B0%EC%95%BD&amp;action=edit&amp;section=7" title="부분 편집: 최근 상황">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p><a href="/wiki/%EC%9D%BC%EB%B3%B8" title="일본">일본</a>에선 <a class="mw-redirect" href="/wiki/NHK" title="NHK">NHK</a>가 정부의 지원하에 '한일강제합방 100주년 특집'을 준비하면서, 강제 병탄을 합법적인 것으로 비치게 한다는 우려를 낳았다.<sup class="reference" id="cite_ref-9"><a href="#cite_note-9">[9]</a></sup></p>
<p><a href="/wiki/2010%EB%85%84" title="2010년">2010년</a> <a href="/wiki/5%EC%9B%94_10%EC%9D%BC" title="5월 10일">5월 10일</a>, 한일 강제병탄 100주년을 맞아 한국의 대표 지식인 109명은 서울 중구 프레스센터에서, 일본 지식인 105명은 도쿄 일본교육회관에서 각각 기자회견을 열고 ‘한일 병합이 원천무효’라는 내용의 성명을 발표했다.</p>
<p>이들은 “한국병합은 대한제국의 황제로부터 민중에 이르기까지 모든 사람의 격렬한 항의를 군대의 힘으로 짓누르고 실현한 제국주의 행위이며 불의부정(不義不正)한 행위”라며 “조약의 전문(前文)도 거짓이고 본문도 거짓”이라고 밝혔다.</p>
<p>이들은 “조약 체결의 절차와 형식에도 중대한 결점과 결함이 보이고 있으며, 한국병합에 이른 과정이 불의부당하듯 한국병합조약도 불의부당하다”라고 지적했다.</p>
<p>성명서에는 한일병탄 조약을 애초부터 불법 무효로 해석한 한국정부의 해석이 맞으며, 한국의 독립운동 역시 불법운동이 아니라는 취지의 내용도 포함됐다.<sup class="reference" id="cite_ref-10"><a href="#cite_note-10">[10]</a></sup></p>
