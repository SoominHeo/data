open-block";block.previousSibling.className+=" open-block";}</script><p><b>다쿠안 소호</b>(<span style="font-size: smaller;"><a href="/wiki/%EC%9D%BC%EB%B3%B8%EC%96%B4" title="일본어">일본어</a>:</span> <span title="1645년">1645년</a> <a href="/wiki/12%EC%9B%94_11%EC%9D%BC" title="12월 11일">12월 11일</a>)는 <a href="/wiki/%EC%9D%BC%EB%B3%B8" title="일본">일본</a> <a href="/wiki/%EC%97%90%EB%8F%84_%EC%8B%9C%EB%8C%80" title="에도 시대">에도 시대</a>의 승려이다. <a class="new" href="/w/index.php?title=%EC%9E%87%ED%81%90_%EC%86%8C%EC%8A%8C&amp;action=edit&amp;redlink=1" title="잇큐 소슌 (없는 문서)">잇큐 소슌</a>과 함께 <a href="/wiki/%EC%9E%84%EC%A0%9C%EC%A2%85" title="임제종">임제종</a> 다이도쿠지 파 가운데 가장 유명한 인물이다.<sup class="reference" id="cite_ref-.EA.B3.A0.EB.AF.B8235_1-0"><a href="#cite_note-.EA.B3.A0.EB.AF.B8235-1">[1]</a></sup> <a href="/wiki/1609%EB%85%84" title="1609년">1609년</a> <a class="new" href="/w/index.php?title=%EB%8B%A4%EC%9D%B4%EB%8F%84%EC%BF%A0%EC%A7%80&amp;action=edit&amp;redlink=1" title="다이도쿠지 (없는 문서)">다이도쿠지</a>의 제153대 지주가 되었고<sup class="reference" id="cite_ref-.EA.B3.A0.EB.AF.B8235_1-1"><a href="#cite_note-.EA.B3.A0.EB.AF.B8235-1">[1]</a></sup>, <a href="/wiki/%EA%B3%A0%EB%AF%B8%EC%A6%88%EB%85%B8%EC%98%A4_%EC%B2%9C%ED%99%A9" title="고미즈노오 천황">고미즈노오 천황</a>을 비롯하여 <a class="new" href="/w/index.php?title=%EA%B3%A0%EB%85%B8%EC%97%90_%EB%85%B8%EB%B6%80%ED%83%80%EB%8B%A4&amp;action=edit&amp;redlink=1" title="고노에 노부타다 (없는 문서)">고노에 노부타다</a> (<span lang="ja" xml:lang="ja">近衛信伊</span>), <a class="new" href="/w/index.php?title=%EA%B0%80%EB%9D%BC%EC%8A%A4%EB%A7%88%EB%A3%A8_%EB%AF%B8%EC%93%B0%ED%9E%88%EB%A1%9C&amp;action=edit&amp;redlink=1" title="가라스마루 미쓰히로 (없는 문서)">가라스마루 미쓰히로</a>, <a class="new" href="/w/index.php?title=%EA%B3%A0%EB%85%B8%EC%97%90_%EB%85%B8%EB%B6%80%ED%9E%88%EB%A1%9C&amp;action=edit&amp;redlink=1" title="고노에 노부히로 (없는 문서)">고노에 노부히로</a> (<span lang="ja" xml:lang="ja">近衛信尋</span>) 등 유력한 조정의 신하들과 교류하였고, <a class="mw-redirect" href="/wiki/%ED%98%B8%EC%86%8C%EC%B9%B4%EC%99%80_%EC%9C%A0%EC%82%AC%EC%9D%B4" title="호소카와 유사이">호소카와 유사이</a> 등과도 참선 및 <a href="/wiki/%EC%99%80%EC%B9%B4" title="와카">와카</a>를 통하여 교류하며 <a href="/wiki/%EC%99%80%EC%B9%B4" title="와카">와카</a>, <a href="/wiki/%EB%8B%A4%EB%8F%84" title="다도">다도</a>, 일본식 조경 등에 많은 영향을 끼쳤다.<sup class="reference" id="cite_ref-.EA.B3.A0.EB.AF.B8235_1-2"><a href="#cite_note-.EA.B3.A0.EB.AF.B8235-1">[1]</a></sup></p>
<p>1627년 7월, 다쿠안은 "자의 사건"에 연루된다.<sup class="reference" id="cite_ref-.EA.B3.A0.EB.AF.B8236_2-0"><a href="#cite_note-.EA.B3.A0.EB.AF.B8236-2">[2]</a></sup> 이 사건은 1615년 <a href="/wiki/%EB%8F%84%EC%BF%A0%EA%B0%80%EC%99%80_%EC%9D%B4%EC%97%90%EC%95%BC%EC%8A%A4" title="도쿠가와 이에야스">도쿠가와 이에야스</a>가 반포한 "다이도쿠 제법도"에 명시된 내용과 관련되어 발생한 것으로, 천황의 허가를 받고 다이도쿠지 주지로 취임할 때 보라색 옷, 즉 자의를 입는 것이 사전에 막부에 고지되어야 한다는 조항을 위반하였다는 이유로, 법도가 제정된 이후 승려가 된 자의 자격을 무효로 하고 자의의 착용을 정지하였다.<sup class="reference" id="cite_ref-.EA.B3.A0.EB.AF.B8236_2-1"><a href="#cite_note-.EA.B3.A0.EB.AF.B8236-2">[2]</a></sup> 이러한 조치 이후 막부는 다이도쿠지의 승려들에게 의견을 물었는데, 다쿠안과 고게쓰 (江月) 등 북파 승려들은 법도의 부조리성을 지적한 글을 막부에 올렸다. 그러나 막부에서 상소의 내용이 승록<sup class="reference" id="cite_ref-3"><a href="#cite_note-3">[주 1]</a></sup> 인 <a href="/wiki/%EA%B3%A4%EC%B9%98%EC%9D%B8_%EC%8A%A4%EB%8D%B4" title="곤치인 스덴">곤치인 스덴</a>과 막부에 대해 비판하고 있다는 것을 문제삼아, 대승정 <a class="new" href="/w/index.php?title=%EB%8D%B4%EC%B9%B4%EC%9D%B4_%EC%87%BC%EB%8B%8C&amp;action=edit&amp;redlink=1" title="덴카이 쇼닌 (없는 문서)">덴카이 쇼닌</a>의 변호에도 불구하고 <a href="/wiki/%EB%8D%B0%EC%99%80_%EA%B5%AD" title="데와 국">데와 국</a> <a href="/wiki/%EA%B0%80%EB%AF%B8%EB%85%B8%EC%95%BC%EB%A7%88_%EC%84%B1" title="가미노야마 성">가미노야마 성</a> 성주인 <a href="/wiki/%EB%8F%84%ED%82%A4_%EC%9A%94%EB%A6%AC%EC%9C%A0%ED%82%A4" title="도키 요리유키">도키 요리유키</a>에게로 유배되었다.<sup class="reference" id="cite_ref-.EA.B3.A0.EB.AF.B8236_2-2"><a href="#cite_note-.EA.B3.A0.EB.AF.B8236-2">[2]</a></sup></p>
<p>그러나 다쿠안은 도키로부터 호의적인 대우를 받으며 2년 동안 가미노야마에서 생활하다 에도로 다시 소환되어, 1634년에 사면되었다. 그 후 다쿠안은 다이도쿠지의 출가 복구 운동을 진행하였다.<sup class="reference" id="cite_ref-.EA.B3.A0.EB.AF.B8236_2-3"><a href="#cite_note-.EA.B3.A0.EB.AF.B8236-2">[2]</a></sup> 그러다 <a href="/wiki/%EB%8F%84%EC%BF%A0%EA%B0%80%EC%99%80_%EC%9D%B4%EC%97%90%EB%AF%B8%EC%93%B0" title="도쿠가와 이에미쓰">도쿠가와 이에미쓰</a>의 <a href="/wiki/%EA%B2%80%EB%8F%84" title="검도">검술 사범</a>으로 일하던 <a class="new" href="/w/index.php?title=%EC%95%BC%EA%B7%9C%ED%83%80%EC%A7%80%EB%A7%88%EB%85%B8%EA%B0%80%EB%AF%B8_%EB%AC%B4%EB%84%A4%EB%85%B8%EB%A6%AC&amp;action=edit&amp;redlink=1" title="야규타지마노가미 무네노리 (없는 문서)">야규타지마노가미 무네노리</a>(<span lang="ja" xml:lang="ja">柳生但馬守宗矩</span>)의 중개로 이에미쓰와 대면하였다.<sup class="reference" id="cite_ref-.EA.B3.A0.EB.AF.B8237_4-0"><a href="#cite_note-.EA.B3.A0.EB.AF.B8237-4">[3]</a></sup> 이에미쓰는 다쿠안을 점점 가까이하게 되었으며, 1637년 10월 <a href="/wiki/%EC%8B%9C%EB%A7%88%EB%B0%94%EB%9D%BC%EC%9D%98_%EB%82%9C" title="시마바라의 난">시마바라의 난</a>이 일어나면서부터 다쿠안은 막부의 중요한 위치에 오르게 된다.<sup class="reference" id="cite_ref-.EA.B3.A0.EB.AF.B8237_4-1"><a href="#cite_note-.EA.B3.A0.EB.AF.B8237-4">[3]</a></sup> 난이 기독교도의 집단 봉기 양상을 띈데다 장기화되어가자 전문 지식을 가진 종교인들이 필요하게 되었고, 결국 다쿠안이 난의 처리를 위임받게 되었다.<sup class="reference" id="cite_ref-.EA.B3.A0.EB.AF.B8237_4-2"><a href="#cite_note-.EA.B3.A0.EB.AF.B8237-4">[3]</a></sup><sup class="reference" id="cite_ref-5"><a href="#cite_note-5">[주 2]</a></sup> 이에미쓰는 <a href="/wiki/%EC%97%90%EB%8F%84" title="에도">에도</a> 인근의 <a href="/wiki/%EB%AC%B4%EC%82%AC%EC%8B%9C_%EA%B5%AD" title="무사시 국">무사시</a><a href="/wiki/%EC%8B%9C%EB%82%98%EA%B0%80%EC%99%80_%EA%B5%AC" title="시나가와 구">시나가와</a>에 <a class="new" href="/w/index.php?title=%EB%8F%84%EC%B9%B4%EC%9D%B4%EC%A7%80&amp;action=edit&amp;redlink=1" title="도카이지 (없는 문서)">도카이지</a>를 건립하여 이 사건에 대응하려 했는데, 도카이지 창립 이후 다쿠안을 절의 개조(開祖, 첫 주지)로 임명하여 막부가 제공한 500석의 영지를 근거로 해 지주 신분을 보장해주는 동시에, 종교 행정의 고문으로 활동하게 하려 했던 것이다.<sup class="reference" id="cite_ref-.EA.B3.A0.EB.AF.B8237_4-3"><a href="#cite_note-.EA.B3.A0.EB.AF.B8237-4">[3]</a></sup><sup class="reference" id="cite_ref-.EA.B3.A0.EB.AF.B8238_6-0"><a href="#cite_note-.EA.B3.A0.EB.AF.B8238-6">[4]</a></sup> 다쿠안은 <a href="/wiki/%EA%B3%A4%EC%B9%98%EC%9D%B8_%EC%8A%A4%EB%8D%B4" title="곤치인 스덴">곤치인 스덴</a>과 같은 역할을 하는 것은 거부했지만 절의 첫 주지가 되는 것은 승낙하였다.<sup class="reference" id="cite_ref-.EA.B3.A0.EB.AF.B8238_6-1"><a href="#cite_note-.EA.B3.A0.EB.AF.B8238-6">[4]</a></sup></p>
<p>이후 이에미쓰는 도카이지를 자주 드나들었고, <a class="mw-disambig" href="/wiki/%EC%82%AC%EC%B9%B4%EC%9D%B4_%EB%8B%A4%EB%8B%A4%EC%B9%B4%EC%93%B0" title="사카이 다다카쓰">사카이 다다카쓰</a>와 <a class="new" href="/w/index.php?title=%ED%98%B8%EB%A6%AC%ED%83%80_%EB%A7%88%EC%82%AC%EB%AA%A8%EB%A6%AC&amp;action=edit&amp;redlink=1" title="호리타 마사모리 (없는 문서)">호리타 마사모리</a>의 측근이 절 내에 작은 암자를 운영하고, <a class="new" href="/w/index.php?title=%EA%B3%A0%EB%B3%B4%EB%A6%AC_%EC%97%94%EC%8A%88&amp;action=edit&amp;redlink=1" title="고보리 엔슈 (없는 문서)">고보리 엔슈</a>가 정원을 조영하는 등 다쿠안은 막부 사람들과 더 밀접한 관계를 맺게 되었다.<sup class="reference" id="cite_ref-.EA.B3.A0.EB.AF.B8238_6-2"><a href="#cite_note-.EA.B3.A0.EB.AF.B8238-6">[4]</a></sup> 1641년에는 다이도쿠지의 출가가 부활하였으며, 1645년 12월 다쿠안은 "몽"(夢)이라는 한 글자를 남기고 사망했다.<sup class="reference" id="cite_ref-.EA.B3.A0.EB.AF.B8238_6-3"><a href="#cite_note-.EA.B3.A0.EB.AF.B8238-6">[4]</a></sup></p>
