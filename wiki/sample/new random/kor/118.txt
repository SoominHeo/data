<p><b>불꽃놀이 은하</b>(<span lang="en" xml:lang="en">Fireworks Galaxy, NGC 6946</span>)는 <a href="/wiki/%EC%84%B8%ED%8E%98%EC%9A%B0%EC%8A%A4%EC%9E%90%EB%A6%AC" title="세페우스자리">세페우스자리</a>와 <a href="/wiki/%EB%B0%B1%EC%A1%B0%EC%9E%90%EB%A6%AC" title="백조자리">백조자리</a>에 있는 <a class="mw-redirect" href="/wiki/%EB%82%98%EC%84%A0_%EC%9D%80%ED%95%98" title="나선 은하">나선 은하</a>이다. 전형적인 앞에서-본-나선은하(face-on spiral galaxy)로서 1798년 윌리엄 허셀(William Herschel)이 처음 발견하였다. 이 은하는 우리은하에 이웃한 천체로써, <a class="mw-redirect" href="/wiki/%EA%B5%AD%EB%B6%80_%EC%9D%80%ED%95%98%EA%B5%B0" title="국부 은하군">국부 은하군</a>(Local Group)의 후보로 등록되었지만 아직 일원으로 확정되지 않은 은하이기도 하다.</p>
<p>NGC 6946은 비교적 독립된 은하이면서도 별의 탄생이 엄청나게 많이 일어나고 있으며, 위 사진처럼 밝은 푸른 별들과 붉은 발광성운, 빠르게 움직이는 은하원반의 가스덩어리에 의해 역동적인 나선팔을 만들어내고 있다.</p>
<p>또한 1917년부터 지금까지 9번의 초신성 폭발(<a class="new" href="/w/index.php?title=SN_1917A&amp;action=edit&amp;redlink=1" title="SN 1917A (없는 문서)">SN 1917A</a>, <a class="new" href="/w/index.php?title=SN_1939C&amp;action=edit&amp;redlink=1" title="SN 1939C (없는 문서)">SN 1939C</a>, <a class="new" href="/w/index.php?title=SN_1948B&amp;action=edit&amp;redlink=1" title="SN 1948B (없는 문서)">SN 1948B</a>, <a class="new" href="/w/index.php?title=SN_1968D&amp;action=edit&amp;redlink=1" title="SN 1968D (없는 문서)">SN 1968D</a>, <a class="new" href="/w/index.php?title=SN_1969P&amp;action=edit&amp;redlink=1" title="SN 1969P (없는 문서)">SN 1969P</a>, <a class="new" href="/w/index.php?title=SN_1980K&amp;action=edit&amp;redlink=1" title="SN 1980K (없는 문서)">SN 1980K</a>, <a class="new" href="/w/index.php?title=SN_2002HH&amp;action=edit&amp;redlink=1" title="SN 2002HH (없는 문서)">SN 2002HH</a>, <a class="new" href="/w/index.php?title=SN_2004ET&amp;action=edit&amp;redlink=1" title="SN 2004ET (없는 문서)">SN 2004ET</a>, <a class="new" href="/w/index.php?title=SN_2008S&amp;action=edit&amp;redlink=1" title="SN 2008S (없는 문서)">SN 2008S</a>)이 기록되어 있으며, <a class="mw-redirect" href="/wiki/%EB%82%A8%EC%AA%BD_%EB%B0%94%EB%9E%8C%EA%B0%9C%EB%B9%84_%EC%9D%80%ED%95%98" title="남쪽 바람개비 은하">남쪽 바람개비 은하</a>의 6번 기록보다 많다. 즉 지난 100년을 1분으로 본다면 7.5초마다 초신성 폭발이라는 불꽃놀이를 구경한 것과 같다.</p>
