<p><b>바이오디젤</b>(biodiesel)은 <a href="/wiki/%EC%84%9D%EC%9C%A0" title="석유">석유</a> 기반인 <a href="/wiki/%EA%B2%BD%EC%9C%A0" title="경유">경유</a>의 대안으로 <a class="new" href="/w/index.php?title=%EC%8B%9D%EB%AC%BC%EC%84%B1_%EA%B8%B0%EB%A6%84&amp;action=edit&amp;redlink=1" title="식물성 기름 (없는 문서)">식물성 기름</a>이나 <a class="new" href="/w/index.php?title=%EB%8F%99%EB%AC%BC%EC%84%B1_%EC%A7%80%EB%B0%A9&amp;action=edit&amp;redlink=1" title="동물성 지방 (없는 문서)">동물성 지방</a>과 같이 <a class="mw-redirect" href="/wiki/%EC%8B%A0%EC%9E%AC%EC%83%9D_%EC%97%90%EB%84%88%EC%A7%80" title="신재생 에너지">재생 가능한</a> 자원을 바탕으로 제조된다. 화학적으로 바이오디젤은 긴 <a href="/wiki/%EC%A7%80%EB%B0%A9%EC%82%B0" title="지방산">지방산</a> 고리를 가진 단일 <a class="mw-redirect" href="/wiki/%EC%95%8C%ED%82%AC" title="알킬">알킬</a> <a href="/wiki/%EC%97%90%EC%8A%A4%ED%84%B0" title="에스터">에스터</a> 혼합물이다. 본래 기름을 원하는 에스터로 전환하고, 자유 지방산을 제거하기 위해 <a class="new" href="/w/index.php?title=%EC%A7%80%EC%A7%88_(%ED%99%94%ED%95%99)&amp;action=edit&amp;redlink=1" title="지질 (화학) (없는 문서)">지질</a> <a class="new" href="/w/index.php?title=%EC%97%90%EC%8A%A4%ED%84%B0_%EA%B5%90%ED%99%98&amp;action=edit&amp;redlink=1" title="에스터 교환 (없는 문서)">에스터 교환</a> 반응이 이용된다. 이런 과정을 거치면, 바이오디젤은 <a class="new" href="/w/index.php?title=%EB%B9%84%EB%B0%A9%ED%96%A5%EC%A1%B1_%EC%8B%9D%EB%AC%BC%EC%84%B1_%EA%B8%B0%EB%A6%84&amp;action=edit&amp;redlink=1" title="비방향족 식물성 기름 (없는 문서)">비방향족 식물성 기름</a>과는 달리 경유와 매우 비슷한 <a href="/wiki/%EC%97%B0%EC%86%8C" title="연소">연소</a> 특성을 가지기 때문에, 현재 사용되는 대부분의 경우에 경유를 대체할 수 있다. 그러나 현재 바이오디젤은 순수 초저<a href="/wiki/%ED%99%A9" title="황">유황</a> 연료의 낮은 <a class="mw-redirect" href="/wiki/%EC%9C%A4%ED%99%9C%EC%A0%9C" title="윤활제">윤활성</a>을 향상시키기 위해 경유와 섞어 쓰는 경우가 많다. 바이오디젤은 재생 가능한 연료로서 현재의 <a href="/wiki/%EC%97%94%EC%A7%84" title="엔진">엔진</a> 기름을 대체할 수 있고, 기존의 시설을 통해 운반, 판매가 가능하기 때문에, 가장 중요한 <a href="/wiki/%EA%B5%90%ED%86%B5" title="교통">교통</a> <a href="/wiki/%EC%97%90%EB%84%88%EC%A7%80" title="에너지">에너지</a> 자원인 <a href="/wiki/%ED%99%94%EC%84%9D_%EC%97%B0%EB%A3%8C" title="화석 연료">화석 연료</a>의 유력한 대안으로 꼽히고 있다. 바이오디젤의 생산과 사용은 특히 <a href="/wiki/%EC%9C%A0%EB%9F%BD" title="유럽">유럽</a>과 <a href="/wiki/%EB%AF%B8%EA%B5%AD" title="미국">미국</a>, <a href="/wiki/%EC%95%84%EC%8B%9C%EC%95%84" title="아시아">아시아</a>에서 급격히 증가하고 있지만, 전체 연료 시장에서 차지하는 비율은 아직 미미하다. 바이오디젤을 취급하는 주유소가 늘어나면서 <a href="/wiki/%EC%86%8C%EB%B9%84%EC%9E%90" title="소비자">소비자</a>가 접할 수 있는 기회가 많아졌고, 대형 선박의 연료에 바이오디젤을 혼합하는 경우가 늘어나고 있다. 예컨대 <a href="/wiki/%EB%8C%80%ED%95%9C%EB%AF%BC%EA%B5%AD" title="대한민국">대한민국</a>에서는 <a href="/wiki/2006%EB%85%84" title="2006년">2006년</a> <a href="/wiki/7%EC%9B%94_1%EC%9D%BC" title="7월 1일">7월 1일</a>부터 일반경유에 5퍼센트의 바이오디젤이 섞인 혼합 경유를 판매하고 있고, <a href="/wiki/%EB%8F%85%EC%9D%BC" title="독일">독일</a>과 <a href="/wiki/%EC%9D%B4%ED%83%88%EB%A6%AC%EC%95%84" title="이탈리아">이탈리아</a>에서는 <a href="/wiki/2006%EB%85%84" title="2006년">2006년</a> 현재 도심버스, 대형 트럭은 아예 100퍼센트 바이오디젤을 사용하도록 의무화하고 있다.<sup class="정리_필요 noprint">[<a href="/wiki/%EC%9C%84%ED%82%A4%EB%B0%B1%EA%B3%BC:%EC%B6%9C%EC%B2%98_%EB%B0%9D%ED%9E%88%EA%B8%B0" title="위키백과:출처 밝히기">출처 필요</a>]</sup></p>
<p><br/>
150℃의 <a href="/wiki/%EC%9D%B8%ED%99%94%EC%A0%90" title="인화점">인화점</a>을 가진 바이오디젤은 경유(64℃)에 비해 불이 잘 붙지 않고, 더구나 폭발하기 쉬운 <a href="/wiki/%ED%9C%98%EB%B0%9C%EC%9C%A0" title="휘발유">휘발유</a>(-45℃)보다는 안정적이다. 실제로 바이오디젤은 미국 산업안전보건청(OSHA)이 비가연성 액체로 분류하고 있다. 이 같은 특성 때문에 순수 바이오디젤을 연료로 쓰는 차량은 <a href="/wiki/%EC%9E%90%EB%8F%99%EC%B0%A8_%EC%82%AC%EA%B3%A0" title="자동차 사고">사고</a>에 더 안전하다고 볼 수 있다.<sup class="정리_필요 noprint">[<a href="/wiki/%EC%9C%84%ED%82%A4%EB%B0%B1%EA%B3%BC:%EC%B6%9C%EC%B2%98_%EB%B0%9D%ED%9E%88%EA%B8%B0" title="위키백과:출처 밝히기">출처 필요</a>]</sup></p>
<p><br/></p>
<p>바이오디젤은 동결점이 <a href="/wiki/%EA%B2%BD%EC%9C%A0" title="경유">경유</a>보다 높아서 (약 -5℃) 추운 기후에서 순수한 형태로 사용하는데 제약이 있다. 또한 5℃ 이하에서는 유동성이 떨어져 연료 공급이 원활하지 못하다. 이러한 이유로 100%의 바이오디젤(BD100)을 사용하는 것보다 경유와 혼합하여 사용하게 된다.<sup class="정리_필요 noprint">[<a href="/wiki/%EC%9C%84%ED%82%A4%EB%B0%B1%EA%B3%BC:%EC%B6%9C%EC%B2%98_%EB%B0%9D%ED%9E%88%EA%B8%B0" title="위키백과:출처 밝히기">출처 필요</a>]</sup></p>
<p><br/></p>
<p>경유와는 달리 바이오디젤은 <a class="new" href="/w/index.php?title=%EB%AF%B8%EC%83%9D%EB%AC%BC_%EB%B6%84%ED%95%B4&amp;action=edit&amp;redlink=1" title="미생물 분해 (없는 문서)">미생물 분해</a>되며, <a class="mw-redirect" href="/wiki/%EB%8F%85%EC%84%B1" title="독성">독성</a>이 없다. 그리고 연료로서 연소될 때 독성이나 기타 배출물이 현저하게 적다. <a class="new" href="/w/index.php?title=%EB%A9%94%ED%8B%B8&amp;action=edit&amp;redlink=1" title="메틸 (없는 문서)">메틸</a> 에스터를 제조하기 위해서는 대개 <a href="/wiki/%EB%A9%94%ED%83%84%EC%98%AC" title="메탄올">메탄올</a>을 사용하지만, 에틸 에스터 바이오디젤을 제조하는 데는 <a href="/wiki/%EC%97%90%ED%83%84%EC%98%AC" title="에탄올">에탄올</a>도 사용할 수 있다. 에스터 교환 과정의 부산물로 <a href="/wiki/%EA%B8%80%EB%A6%AC%EC%84%B8%EB%A1%A4" title="글리세롤">글리세롤</a>이 생성된다.<sup class="정리_필요 noprint">[<a href="/wiki/%EC%9C%84%ED%82%A4%EB%B0%B1%EA%B3%BC:%EC%B6%9C%EC%B2%98_%EB%B0%9D%ED%9E%88%EA%B8%B0" title="위키백과:출처 밝히기">출처 필요</a>]</sup></p>
<p><br/></p>
<p>현재 바이오디젤은 보통 <a href="/wiki/%EA%B2%BD%EC%9C%A0" title="경유">경유</a>보다 생산가가 높기 때문에, 이 점이 종종 대중화의 최대 걸림돌로 지적되고 있다. 원유 값의 상승에 따라, 바이오디젤의 <a class="new" href="/w/index.php?title=%EA%B2%BD%EC%A0%9C_%EA%B7%9C%EB%AA%A8&amp;action=edit&amp;redlink=1" title="경제 규모 (없는 문서)">경제 규모</a>는 앞으로 이 같은 현실이 뒤집어지는 것도 가능하다. 그러나 식물성 기름이나 동물성 지방의 전 세계 생산량은 <a class="new" href="/w/index.php?title=%EC%95%A1%EC%B2%B4_%ED%99%94%EC%84%9D_%EC%97%B0%EB%A3%8C&amp;action=edit&amp;redlink=1" title="액체 화석 연료 (없는 문서)">액체 화석 연료</a>의 이용량을 대체할 수준에는 미치지 못한다. 일부 환경 단체, 특히 <a class="new" href="/w/index.php?title=%EC%B2%9C%EC%97%B0%EC%9E%90%EC%9B%90_%EB%B3%B4%ED%98%B8_%ED%98%91%EC%9D%98%ED%9A%8C&amp;action=edit&amp;redlink=1" title="천연자원 보호 협의회 (없는 문서)">천연자원 보호 협의회</a>(NRDC)에서는 더 많은 식물성 기름을 생산하기 위해 광범위한 재배가 이루어지고 결국 과도한 <a href="/wiki/%EB%B9%84%EB%A3%8C" title="비료">비료</a>, <a href="/wiki/%EB%86%8D%EC%95%BD" title="농약">농약</a>의 사용을 빚을 것이라고 비판하고 있다.<sup class="정리_필요 noprint">[<a href="/wiki/%EC%9C%84%ED%82%A4%EB%B0%B1%EA%B3%BC:%EC%B6%9C%EC%B2%98_%EB%B0%9D%ED%9E%88%EA%B8%B0" title="위키백과:출처 밝히기">출처 필요</a>]</sup></p>
<p><br/></p>
