<p><b>산이</b>(<span style="font-size: smaller;"><a href="/wiki/%EC%98%81%EC%96%B4" title="영어">영어</a>:</span> <span lang="en" xml:lang="en">San E</span>, <small><a class="mw-redirect" href="/wiki/%EB%B3%B8%EB%AA%85" title="본명">본명</a>: </small>정산, <a href="/wiki/1985%EB%85%84" title="1985년">1985년</a> <a href="/wiki/1%EC%9B%94_23%EC%9D%BC" title="1월 23일">1월 23일</a> ~ )은 의 <a href="/wiki/%ED%9E%99%ED%95%A9" title="힙합">힙합</a> <a href="/wiki/MC_(%EC%9D%8C%EC%95%85)" title="MC (음악)">MC</a>이다, 한때 <a href="/wiki/JYP_%EC%97%94%ED%84%B0%ED%85%8C%EC%9D%B8%EB%A8%BC%ED%8A%B8" title="JYP 엔터테인먼트">JYP 엔터테인먼트</a> 소속이였으나 현재는 <a class="mw-redirect" href="/wiki/%EB%B8%8C%EB%9E%9C%EB%89%B4_%EB%AE%A4%EC%A7%81" title="브랜뉴 뮤직">브랜뉴 뮤직</a>과 크루 <a href="/wiki/%EC%98%A4%EB%B2%84%ED%81%B4%EB%9E%98%EC%8A%A4" title="오버클래스">오버클래스</a>에 소속되어있다. <a href="/wiki/%EC%9B%90%EB%8D%94%EA%B1%B8%EC%8A%A4" title="원더걸스">원더걸스</a>의 "Nobody"라는 리믹스 노래에 피쳐링을 했으며<sup class="reference" id="cite_ref-hiphopplaya.com_1-0"><a href="#cite_note-hiphopplaya.com-1">[1]</a></sup>, 이후 <a href="/wiki/%EC%98%A4%EB%B2%84%ED%81%B4%EB%9E%98%EC%8A%A4" title="오버클래스">오버클래스</a>에 들어가 <a href="/wiki/%EB%B2%84%EB%B2%8C_%EC%A7%84%ED%8A%B8" title="버벌 진트">버벌 진트</a>의 《<a class="mw-redirect" href="/wiki/%EB%88%84%EB%AA%85_(%EB%B2%84%EB%B2%8C_%EC%A7%84%ED%8A%B8%EC%9D%98_%EC%9D%8C%EB%B0%98)" title="누명 (버벌 진트의 음반)">누명</a>》수록 2008대한민국(prod.오래된엘피.vj) 에 피쳐링을 하였다. 2010년 <a href="/wiki/%ED%95%9C%EA%B5%AD%EB%8C%80%EC%A4%91%EC%9D%8C%EC%95%85%EC%83%81" title="한국대중음악상">한국대중음악상</a>에서 "Rap Genius"로 최우수 랩&amp;힙합 노래 상을 수상하였다.<sup class="reference" id="cite_ref-2"><a href="#cite_note-2">[2]</a></sup> 2010년 9월에는 JYP 소속으로 첫 EP 음반 <i>Everybody Ready?</i>를 발매했다.<sup class="reference" id="cite_ref-3"><a href="#cite_note-3">[3]</a></sup> 그리고 2013년이 되어 JYP엔터테인먼트와의 계약을 해지하고 <strong class="selflink">산이</strong>는 <a href="/wiki/%EB%9D%BC%EC%9D%B4%EB%A8%B8" title="라이머">라이머</a>가 대표로 있는 <a href="/wiki/%EB%B8%8C%EB%9E%9C%EB%89%B4%EB%AE%A4%EC%A7%81" title="브랜뉴뮤직">브랜뉴뮤직</a>에 들어갔다.</p>

<h2><span class="mw-headline" id=".EB.B0.B0.EA.B2.BD">배경</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%82%B0%EC%9D%B4&amp;action=edit&amp;section=1" title="부분 편집: 배경">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>어렸을적부터 국내 힙합, <a href="/wiki/%EC%A1%B0PD" title="조PD">조PD</a>를 비롯한 외국 (한국) 힙합을 자주 들었다. 또한 어린시절에 랩을 즐겨 노래방이나 애틀랜타에서도 공연을 자주 하였다. 이후 대학생이 된 뒤 한국에 입국 해 2학년 때 <a href="/wiki/%EC%97%B0%EC%84%B8%EB%8C%80%ED%95%99%EA%B5%90" title="연세대학교">연세대학교</a> 교환학생으로 힙합동아리 공연도 하였으나 잠시 포기하였다. 그 후 대학교 졸업 무렵, 홈 레코딩을 시작하면서 다시 랩을 시작하였다. 이때 <a href="/wiki/%EB%A6%AC%EB%93%9C%EB%A8%B8" title="리드머">리드머</a>, <a href="/wiki/%ED%9E%99%ED%95%A9%ED%94%8C%EB%A0%88%EC%9D%B4%EC%95%BC" title="힙합플레이야">힙합플레이야</a> 등 자신의 녹음물을 올리기 시작하였다.<sup class="reference" id="cite_ref-salon_4-0"><a href="#cite_note-salon-4">[4]</a></sup></p>
<p><a href="/wiki/2008%EB%85%84" title="2008년">2008년</a>, 첫 믹스테잎이었던 <i>Ready To Be Signed</i>를 작업하면서 <a href="/wiki/%EB%B2%84%EB%B2%8C_%EC%A7%84%ED%8A%B8" title="버벌 진트">버벌 진트</a>를 디스한 〈재밌쎄요〉, 수록곡인 〈산선생님〉을 통해 많은 사람들에게 주목을 받기 시작하였다. 이때 학업을 포기하고 본격적으로 랩을 하기 시작하였다. 〈재밌쎄요〉를 계기로 버벌 진트의 제의를 받아 <a href="/wiki/%EC%98%A4%EB%B2%84%ED%81%B4%EB%9E%98%EC%8A%A4" title="오버클래스">오버클래스</a>에 합류하였고 이어 버벌 진트의 음반 《<a class="mw-redirect" href="/wiki/%EB%88%84%EB%AA%85_(%EB%B2%84%EB%B2%8C_%EC%A7%84%ED%8A%B8%EC%9D%98_%EC%9D%8C%EB%B0%98)" title="누명 (버벌 진트의 음반)">누명</a>》 수록곡 〈2008년 대한민국〉에 참여하며 데뷔하게 되었다.<sup class="reference" id="cite_ref-playa_5-0"><a href="#cite_note-playa-5">[5]</a></sup><sup class="reference" id="cite_ref-presian_6-0"><a href="#cite_note-presian-6">[6]</a></sup> 이후 <a href="/wiki/SM_%EC%97%94%ED%84%B0%ED%85%8C%EC%9D%B8%EB%A8%BC%ED%8A%B8" title="SM 엔터테인먼트">SM</a>, <a class="mw-redirect" href="/wiki/YG" title="YG">YG</a>, <a href="/wiki/JYP_%EC%97%94%ED%84%B0%ED%85%8C%EC%9D%B8%EB%A8%BC%ED%8A%B8" title="JYP 엔터테인먼트">JYP</a>를 비롯한 여러 기획사들에 믹스테잎 CD를 보냈지만, 연락이 없었고 후에 지인을 통해 JYP USA 부사장인 이우석에게 믹스테잎을 보내 <a href="/wiki/%EB%89%B4%EC%9A%95" title="뉴욕">뉴욕</a>에서 오디션을 본 뒤 합격하였다.<sup class="reference" id="cite_ref-salon_4-1"><a href="#cite_note-salon-4">[4]</a></sup></p>
<p>이후 <a href="/wiki/%EC%9B%90%EB%8D%94%EA%B1%B8%EC%8A%A4" title="원더걸스">원더걸스</a>의 "Nobody" 리믹스 트랙인 "Anybody"에 참여하였고<sup class="reference" id="cite_ref-hiphopplaya.com_1-1"><a href="#cite_note-hiphopplaya.com-1">[1]</a></sup>, 오버클래스의 <i>Collage 2</i>에 이어 <a href="/wiki/%EC%A0%9C%ED%94%BC_(%EC%9D%8C%EC%95%85%EA%B0%80)" title="제피 (음악가)">제피</a>, <a class="mw-redirect" href="/wiki/%EB%8D%94_%EC%BD%B0%EC%9D%B4%EC%97%87" title="더 콰이엇">더 콰이엇</a> 등 여러 아티스트 음반에 피쳐링으로 참여하였다. <a href="/wiki/2010%EB%85%84" title="2010년">2010년</a> <a href="/wiki/9%EC%9B%94_7%EC%9D%BC" title="9월 7일">9월 7일</a>, 티저를 공개하였고<sup class="reference" id="cite_ref-7"><a href="#cite_note-7">[7]</a></sup> <a href="/wiki/9%EC%9B%94_13%EC%9D%BC" title="9월 13일">9월 13일</a>, 데뷔 EP 음반 <i><a href="/wiki/Everybody_Ready%3F" title="Everybody Ready?">Everybody Ready?</a></i>를 발매하고 〈맛좋은산〉으로 활동하였다. 그 후에도 계속해서 피쳐링 활동을 활발하게 하였으며 <a href="/wiki/2011%EB%85%84_4%EC%9B%94" title="2011년 4월">2011년 4월</a>, 〈<a class="new" href="/w/index.php?title=%EA%B0%80%EB%A9%B4_%EC%95%88%EB%8F%BC&amp;action=edit&amp;redlink=1" title="가면 안돼 (없는 문서)">가면 안돼</a>〉를 발매하였고 <a href="/wiki/2011%EB%85%84_11%EC%9B%94" title="2011년 11월">2011년 11월</a>, 〈불행했음 좋겠다〉를 발매하였다. <a href="/wiki/2013%EB%85%84" title="2013년">2013년</a> <a href="/wiki/JYP_%EC%97%94%ED%84%B0%ED%85%8C%EC%9D%B8%EB%A8%BC%ED%8A%B8" title="JYP 엔터테인먼트">JYP 엔터테인먼트</a>와의 계약을 해지<sup class="reference" id="cite_ref-8"><a href="#cite_note-8">[8]</a></sup> 하고, 소속사를 떠났다. 이후 <a class="mw-redirect" href="/wiki/%EB%B8%8C%EB%9E%9C%EB%89%B4_%EB%AE%A4%EC%A7%81" title="브랜뉴 뮤직">브랜뉴 뮤직</a>과 전속 계약했다.</p>

<h2><span class="mw-headline" id=".EC.9D.B4.EC.8A.88">이슈</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%82%B0%EC%9D%B4&amp;action=edit&amp;section=11" title="부분 편집: 이슈">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<h3><span class="mw-headline" id=".ED.94.BC.ED.83.80.EC.9E.85.EC.9D.84_.ED.96.A5.ED.95.9C_.EB.94.94.EC.8A.A4">피타입을 향한 디스</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%82%B0%EC%9D%B4&amp;action=edit&amp;section=12" title="부분 편집: 피타입을 향한 디스">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>2008년 12월 14일, <a href="/wiki/%ED%94%BC%ED%83%80%EC%9E%85" title="피타입">피타입</a>의 힙합에 대한 발언에 격분한 산이가 "Bye P-Type"이라는 곡을 발표하였다. 이 곡은 산이의 싸이월드 미니홈피에 처음 올라와 힙합 커뮤니티 사이트로 퍼졌으며, 큰 이슈를 불러모았다. 또 여기에 더불어 산이를 향한 디스곡이 <a href="/wiki/%EC%A0%9C%EC%9D%B4%EC%BC%A0" title="제이켠">제이켠</a>에 의해 만들어져 힙플 뉴스에 올라와 또 한 번 화제가 되었다. 이에 대해 산이는 싸이 다이어리에 제이켠을 비난하는 글을 올리기도 하였다.</p>
<p>이슈가 커지자 1월 3일 산이는 힙합플레이어에 "이유가 어쨌건 여러분을 실망시켜드렸다면 제 잘못"이라는 사과글을 올려<sup class="reference" id="cite_ref-9"><a href="#cite_note-9">[9]</a></sup> 이 사건은 어느 정도 일단락되었다. 2010년 후 제이켠은 자신의 데뷔 앨범에 "Hot For Me"라는 곡에 산이를 참여시키기도 하였으며, 이때의 사건에 대해 "아무 생각 없이 랩 배틀이란 의미로 재밌어 보여서 디스를 건 것일 뿐이었다"라고 발언하였다.<sup class="reference" id="cite_ref-10"><a href="#cite_note-10">[10]</a></sup> 이후 <a href="/wiki/%ED%94%BC%ED%83%80%EC%9E%85" title="피타입">피타입</a>과 같은 <a class="mw-redirect" href="/wiki/%EB%B8%8C%EB%9E%9C%EB%89%B4_%EB%AE%A4%EC%A7%81" title="브랜뉴 뮤직">브랜뉴 뮤직</a>의 일원이 된 이후로는 피타입의 곡인 "불편한 관계"에 피처링으로 참여한 바 있다.</p>
