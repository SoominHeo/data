<p><b>헤오르히니오 흐레히온 에밀러 베이날뒴</b> (<span style="font-size: smaller;"><a href="/wiki/%EB%84%A4%EB%8D%9C%EB%9E%80%EB%93%9C%EC%96%B4" title="네덜란드어">네덜란드어</a>:</span> <span lang="nl" xml:lang="nl">Georginio Gregion Emile Wijnaldum</span>, <a href="/wiki/1990%EB%85%84" title="1990년">1990년</a> <a href="/wiki/11%EC%9B%94_11%EC%9D%BC" title="11월 11일">11월 11일</a> ~ )는 <a href="/wiki/%EB%84%A4%EB%8D%9C%EB%9E%80%EB%93%9C" title="네덜란드">네덜란드</a>의 <a href="/wiki/%EC%B6%95%EA%B5%AC_%EC%84%A0%EC%88%98" title="축구 선수">축구 선수</a>이다. 현재 <a href="/wiki/%EC%9E%89%EA%B8%80%EB%9E%9C%EB%93%9C" title="잉글랜드">잉글랜드</a> <a href="/wiki/%ED%94%84%EB%A6%AC%EB%AF%B8%EC%96%B4%EB%A6%AC%EA%B7%B8" title="프리미어리그">프리미어리그</a>의 <a href="/wiki/%EB%A6%AC%EB%B2%84%ED%92%80_FC" title="리버풀 FC">리버풀</a>에서 뛰고 있으며, 포지션은 <a href="/wiki/%EB%AF%B8%EB%93%9C%ED%95%84%EB%8D%94#.EA.B3.B5.EA.B2.A9.ED.98.95_.EB.AF.B8.EB.93.9C.ED.95.84.EB.8D.94" title="미드필더">공격형 미드필더</a>와 <a href="/wiki/%EB%AF%B8%EB%93%9C%ED%95%84%EB%8D%94#.EC.9C.99.EC.96.B4" title="미드필더">윙어</a>이다. 그는 <a href="/wiki/2014%EB%85%84_FIFA_%EC%9B%94%EB%93%9C%EC%BB%B5" title="2014년 FIFA 월드컵">2014년</a> <a href="/wiki/%EC%95%84%EB%A5%B4%ED%97%A8%ED%8B%B0%EB%82%98_%EC%B6%95%EA%B5%AC_%EA%B5%AD%EA%B0%80%EB%8C%80%ED%91%9C%ED%8C%80" title="아르헨티나 축구 국가대표팀">아르헨티나</a>와의 월드컵 4강전에서 볼 경합을 벌이던 도중 <a href="/wiki/%ED%95%98%EB%B9%84%EC%97%90%EB%A5%B4_%EB%A7%88%EC%8A%A4%EC%B2%B4%EB%9D%BC%EB%85%B8" title="하비에르 마스체라노">하비에르 마스체라노</a>의 뇌진탕을 일으키기도 했다.</p>
