<p><b>영국의 군주</b>(<span style="font-size: smaller;"><a href="/wiki/%EC%98%81%EC%96%B4" title="영어">영어</a>:</span> <span lang="en" xml:lang="en">Monarch of the United Kingdom, British Monarch</span>)는 영국의 국가 원수이다. 군주의 호칭은 왕(King)이나 여왕(Queen)이다. <a href="/wiki/1952%EB%85%84" title="1952년">1952년</a> <a href="/wiki/2%EC%9B%94_6%EC%9D%BC" title="2월 6일">2월 6일</a> 이후로 현재까지 <a href="/wiki/%EC%97%98%EB%A6%AC%EC%9E%90%EB%B2%A0%EC%8A%A4_2%EC%84%B8" title="엘리자베스 2세">엘리자베스 2세</a>가 재임 중이다.</p>

<h2><span class="mw-headline" id=".EC.A7.80.EC.9C.84">지위</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%98%81%EA%B5%AD%EC%9D%98_%EA%B5%B0%EC%A3%BC&amp;action=edit&amp;section=1" title="부분 편집: 지위">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<h3><span class="mw-headline" id=".EC.98.81.EA.B5.AD.EC.9D.98_.EA.B5.B0.EC.A3.BC">영국의 군주</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%98%81%EA%B5%AD%EC%9D%98_%EA%B5%B0%EC%A3%BC&amp;action=edit&amp;section=2" title="부분 편집: 영국의 군주">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p><a href="/wiki/%EB%B6%88%EB%AC%B8_%ED%97%8C%EB%B2%95" title="불문 헌법">불문 헌법</a>인 <a class="extiw" href="https://en.wikipedia.org/wiki/Constitution_of_the_United_Kingdom" title="en:Constitution of the United Kingdom">영국의 헌법</a>에 따르면 군주는<sup class="reference" id="cite_ref-1"><a href="#cite_note-1">[참고 1]</a></sup> <a href="/wiki/%EC%98%81%EA%B5%AD" title="영국">영국</a>의 국가 원수이다. 공무원들과 귀화자들은 여왕과 그녀의 후계자들에게 <a class="extiw" href="https://en.wikipedia.org/wiki/Oath_of_Allegiance_(United_Kingdom)" title="en:Oath of Allegiance (United Kingdom)">충성 선서</a>를 맹세한다.<sup class="reference" id="cite_ref-2"><a href="#cite_note-2">[1]</a></sup> <a class="mw-redirect" href="/wiki/%EC%8B%A0%EC%9D%B4%EC%97%AC_%ED%8F%90%ED%95%98%EB%A5%BC_%EC%A7%80%EC%BC%9C%EC%A3%BC%EC%86%8C%EC%84%9C" title="신이여 폐하를 지켜주소서">신이여 폐하를 지켜주소서</a>는 영국의 <a href="/wiki/%EA%B5%AD%EA%B0%80_(%EB%85%B8%EB%9E%98)" title="국가 (노래)">국가</a>이다.<sup class="reference" id="cite_ref-3"><a href="#cite_note-3">[2]</a></sup> 또한 군주의 얼굴은 우표와 지폐에 나타난다.<sup class="reference" id="cite_ref-4"><a href="#cite_note-4">[3]</a></sup></p>
<p>영국의 군주는 직접 <a href="/wiki/%EC%98%81%EA%B5%AD_%EC%A0%95%EB%B6%80" title="영국 정부">영국 정부</a> 안에 극히 일부분의 왕권을 수행하며, 정치에도 관여한다. <a href="/wiki/%EC%A3%BC%EA%B6%8C" title="주권">주권</a>의 행사권은 <a class="extiw" href="https://en.wikipedia.org/wiki/Statute" title="en:Statute">법령</a>이나 법적 행동으로 통해 군주에서 <a href="/wiki/%EB%8C%80%EC%8B%A0_(%EC%98%81%EA%B5%AD_%EC%97%B0%EB%B0%A9_%EC%99%95%EA%B5%AD)" title="대신 (영국 연방 왕국)">대신</a>들이나 군주의 사적인 공공기관에 위탁한다. 그러므로 <b>국왕의 친임</b>(<span style="font-size: smaller;"><a href="/wiki/%EC%98%81%EC%96%B4" title="영어">영어</a>:</span> <span lang="en" xml:lang="en">Crown Appointments</span>)과<sup class="reference" id="cite_ref-5"><a href="#cite_note-5">[4]</a></sup> 같이 <a href="/wiki/%EA%B5%AD%EC%99%95_(%EC%98%81%EA%B5%AD_%EC%97%B0%EB%B0%A9_%EC%99%95%EA%B5%AD)" title="국왕 (영국 연방 왕국)">국왕</a>이라는 단독법인의 이름으로 대신들을 임명하는 행위에도 불구하고 군주는 <a href="/wiki/%EA%B0%9C%EC%9B%90%EC%97%B0%EC%84%A4" title="개원연설">개원연설</a>과 <a class="extiw" href="https://en.wikipedia.org/wiki/State_Opening_of_Parliament" title="en:State Opening of Parliament">개회 선언</a>을 통해서 개인적으로 임명하는 경우도 있다. 즉, <a href="/wiki/%EB%8C%80%EB%B3%80%EC%9D%B8" title="대변인">대변인</a> 역할을 일부 수행을 한다.</p>
<h3><span class="mw-headline" id=".EC.98.81.EA.B5.AD_.EC.97.B0.EB.B0.A9.EC.9D.98_.EC.88.98.EC.9E.A5">영국 연방의 수장</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%98%81%EA%B5%AD%EC%9D%98_%EA%B5%B0%EC%A3%BC&amp;action=edit&amp;section=3" title="부분 편집: 영국 연방의 수장">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p><a href="/wiki/1867%EB%85%84" title="1867년">1867년</a>에 <a href="/wiki/%EC%BA%90%EB%82%98%EB%8B%A4" title="캐나다">캐나다</a>가 자치령으로 승격된 이래 <a href="/wiki/%EC%98%A4%EC%8A%A4%ED%8A%B8%EB%A0%88%EC%9D%BC%EB%A6%AC%EC%95%84" title="오스트레일리아">오스트레일리아</a>, <a href="/wiki/%EB%89%B4%EC%A7%88%EB%9E%9C%EB%93%9C" title="뉴질랜드">뉴질랜드</a> 등이 자치령이 되어 영국 국왕에 충성한다는 공동의 취지하에 <a href="/wiki/%EC%98%81%EA%B5%AD_%EC%97%B0%EB%B0%A9" title="영국 연방">영국 연방</a>이 탄생했다. <a href="/wiki/1950%EB%85%84" title="1950년">1950년</a>에 영국 연방 총회는 성명을 발표하고 '영국 국왕은 영국 연방의 수장(首長)이며 자유로운 결합의 상징'이라 하였다. 이리하여 영국 국왕에 대한 충성의 서약은 사실상 필요 없게 되었다.</p>
<p>현재 <a href="/wiki/%EC%97%98%EB%A6%AC%EC%9E%90%EB%B2%A0%EC%8A%A4_2%EC%84%B8" title="엘리자베스 2세">엘리자베스 2세</a>는 영국 연방 가입국 중 영국을 포함한 16개국에서 군주로 군림하고 있다. 다음은 영국 연방 회원국 중에서 영국의 군주를 국가원수로 하는 나라들의 목록이다.</p>

<h3><span class="mw-headline" id=".EC.98.81.EA.B5.AD_.EC.84.B1.EA.B3.B5.ED.9A.8C.EC.9D.98_.EC.B5.9C.EA.B3.A0_.EC.B9.98.EB.A6.AC.EC.9E.90">영국 성공회의 최고 치리자</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%98%81%EA%B5%AD%EC%9D%98_%EA%B5%B0%EC%A3%BC&amp;action=edit&amp;section=4" title="부분 편집: 영국 성공회의 최고 치리자">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>영국의 군주는 신앙의 옹호자라고도 불리는 <a class="mw-disambig" href="/wiki/%EC%98%81%EA%B5%AD_%EC%84%B1%EA%B3%B5%ED%9A%8C" title="영국 성공회">영국 성공회</a>의 최고 치리자를 맡고 있다. 여기서 한 가지 오해하지 말아야 할 게 있는데, 영국 국왕은 영국 성공회 이외의 <a class="mw-redirect" href="/wiki/%EC%84%B8%EA%B3%84_%EC%84%B1%EA%B3%B5%ED%9A%8C_%EA%B3%B5%EB%8F%99%EC%B2%B4" title="세계 성공회 공동체">세계 성공회 공동체</a> 자치 관구교회들과는 관련이 없다. 단지 영국의 <a href="/wiki/%EA%B5%AD%EA%B5%90" title="국교">국교</a>인 영국 성공회에서만 상징적 지도자로서의 의미가 있을 뿐이다.</p>
<p><a href="/wiki/%ED%97%A8%EB%A6%AC_8%EC%84%B8" title="헨리 8세">헨리 8세</a>가 종교개혁을 단행하여 <a href="/wiki/6%EC%84%B8%EA%B8%B0" title="6세기">6세기</a> 이래 로마 가톨릭의 지배를 받던 영국 교회를 영국 성공회로 독립시키고, <a href="/wiki/1534%EB%85%84" title="1534년">1534년</a> 수장령(首長令)으로 '영국 국교회의 유일한 지상 수장'이 왕임을 선포한다. 이에 따라 <a class="mw-redirect" href="/wiki/%EB%A1%9C%EB%A7%88_%EA%B5%90%ED%99%A9" title="로마 교황">로마 교황</a> 대신 영국 국왕이 성공회의 수장을 맡게되어 지금까지 내려오고 있다.</p>

<p>왕위 계승 조건은 다음과 같다.</p>
<ul>
<li>반드시 본인과 배우자가 <a href="/wiki/%EC%84%B1%EA%B3%B5%ED%9A%8C" title="성공회">성공회</a>나 <a href="/wiki/%EA%B0%9C%EC%8B%A0%EA%B5%90" title="개신교">개신교</a> 신자여야 한다.</li>
<li>2015년까지는 형제자매 가운데서 여자보다 남자가 우선권을 가졌으나, 그 이후에 태어난 경우 무조건 먼저 태어날수록 순위가 높다.</li>
<li>여자 직계가 남자 방계보다 우선권을 가진다.</li>
</ul>
<p>이러한 조건에 따라, 현재 왕위 계승 서열 1위는 <a href="/wiki/%EC%9B%A8%EC%9D%BC%EC%8A%A4_%EA%B3%B5_%EC%B0%B0%EC%8A%A4" title="웨일스 공 찰스">웨일스 공 찰스</a>이다. 다음은 왕위 계승 서열 1위부터 11위까지의 명단이다.</p>
