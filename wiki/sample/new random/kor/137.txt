<p><b><a href="/wiki/%ED%95%80%EB%9E%80%EB%93%9C" title="핀란드">핀란드</a> 영토 중 일부분은 <a href="/wiki/%EC%86%8C%EB%A0%A8" title="소련">소련</a>에 할양</b>되었다. <a class="mw-redirect" href="/wiki/%EC%B9%B4%EB%A0%90%EB%A6%AC%EC%95%84" title="카렐리아">카렐리아</a>, 살라, <a class="new" href="/w/index.php?title=%ED%8E%98%EC%B2%B8%EA%B0%80_%ED%8A%B9%EA%B5%AC&amp;action=edit&amp;redlink=1" title="페첸가 특구 (없는 문서)">페첸가 특구</a>, <a href="/wiki/%ED%95%80%EB%9E%80%EB%93%9C_%EB%A7%8C" title="핀란드 만">핀란드 만</a>에 몇몇 섬들 등이 있다.</p>
<h2><span class="mw-headline" id=".EB.85.BC.EC.9F.81">논쟁</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%B9%B4%EB%A0%90%EB%A6%AC%EC%95%BC_%EB%AC%B8%EC%A0%9C&amp;action=edit&amp;section=1" title="부분 편집: 논쟁">원본 편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>1939년 <a href="/wiki/%EC%A0%9C2%EC%B0%A8_%EC%84%B8%EA%B3%84_%EB%8C%80%EC%A0%84" title="제2차 세계 대전">제2차 세계 대전</a>이 터지면서 핀란드는 추축국의 하나로 참가하였다. 소련은 자꾸 괴롭히는 핀란드를 타도하기로 했다. 결국 이 판단은 <a href="/wiki/%EA%B2%A8%EC%9A%B8_%EC%A0%84%EC%9F%81" title="겨울 전쟁">겨울 전쟁</a>, <a href="/wiki/%EA%B3%84%EC%86%8D_%EC%A0%84%EC%9F%81" title="계속 전쟁">계속 전쟁</a>으로 이어졌다. 겨울 전쟁에서 핀란드는 소련에게 영토를 약간 빼앗겼다. 지금의 핀란드-러시아 국경부분인 <a class="mw-redirect" href="/wiki/%EC%B9%B4%EB%A0%90%EB%A6%AC%EC%95%84" title="카렐리아">카렐리아</a>, <a href="/wiki/%EB%9D%BC%ED%94%BC_%EC%A3%BC" title="라피 주">핀란드령 라플란드</a>와 핀란드 만의 <a href="/wiki/%EC%BD%94%ED%8B%80%EB%A6%B0_%EC%84%AC" title="코틀린 섬">코틀린 섬</a>, <a class="new" href="/w/index.php?title=%EC%88%98%EB%A6%AC%EC%82%AC%EB%A5%B4_%EC%84%AC&amp;action=edit&amp;redlink=1" title="수리사르 섬 (없는 문서)">수리사르 섬</a>, <a class="new" href="/w/index.php?title=%ED%97%AC%EB%A0%8C%EC%A7%80_%EC%84%AC&amp;action=edit&amp;redlink=1" title="헬렌지 섬 (없는 문서)">헬렌지 섬</a> 등 4개의 섬을 빼앗겼다. 결국 1년 후 <a href="/wiki/%EB%82%98%EC%B9%98_%EB%8F%85%EC%9D%BC" title="나치 독일">나치 독일</a>이 소련을 침공하고 핀란드도 동맹국으로 참전하면서 <a href="/wiki/%EA%B3%84%EC%86%8D_%EC%A0%84%EC%9F%81" title="계속 전쟁">계속 전쟁</a>이 일어났다. <a href="/wiki/%EB%AC%B4%EB%A5%B4%EB%A7%8C%EC%8A%A4%ED%81%AC" title="무르만스크">무르만스크</a>를 차지한다는 목적은 실패했지만, 다시 카렐리아를 되찾게 되었다. 하지만 나머지는 1991년 소련이 붕괴될 때까지 소련령으로 남았다.</p>

<h3><span class="mw-headline" id=".EB.83.89.EC.A0.84">냉전</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%B9%B4%EB%A0%90%EB%A6%AC%EC%95%BC_%EB%AC%B8%EC%A0%9C&amp;action=edit&amp;section=2" title="부분 편집: 냉전">원본 편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>미국과 소련 사이에 <a href="/wiki/%EB%83%89%EC%A0%84" title="냉전">냉전</a>이 시작되면서 소련은 강대국으로써의 세력을 휘둘르고 있었다. 그러다가 핀란드를 노리게 되었다. 소련은 1945년 승전국으로 핀란드의 <a class="new" href="/w/index.php?title=%ED%8F%AC%ED%81%AC%EC%B9%BC%EB%9D%BC&amp;action=edit&amp;redlink=1" title="포크칼라 (없는 문서)">포크칼라</a>를 빼앗았다. 이 곳은 핀란드에게는 매우 중요한 곳이었다. 포크칼라는 1956년 <a class="mw-redirect" href="/wiki/%ED%9D%90%EB%A3%A8%EC%8B%9C%EC%B4%88%ED%94%84" title="흐루시초프">흐루시초프</a>의 친선정책으로 핀란드령으로 되돌아왔다.</p>
