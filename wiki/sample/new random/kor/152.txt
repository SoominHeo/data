<p><b>로스 바클리</b>(Ross Barkley, <a href="/wiki/1993%EB%85%84" title="1993년">1993년</a> <a href="/wiki/12%EC%9B%94_5%EC%9D%BC" title="12월 5일">12월 5일</a>, <a href="/wiki/%EB%A6%AC%EB%B2%84%ED%92%80" title="리버풀">리버풀</a> ~) 는 <a href="/wiki/%EC%9E%89%EA%B8%80%EB%9E%9C%EB%93%9C" title="잉글랜드">잉글랜드</a>의 <a href="/wiki/%EC%B6%95%EA%B5%AC_%EC%84%A0%EC%88%98" title="축구 선수">축구 선수</a>로, <a href="/wiki/%EC%97%90%EB%B2%84%ED%84%B4_FC" title="에버턴 FC">에버턴 FC</a>과 <a href="/wiki/%EC%9E%89%EA%B8%80%EB%9E%9C%EB%93%9C_%EC%B6%95%EA%B5%AC_%EA%B5%AD%EA%B0%80%EB%8C%80%ED%91%9C%ED%8C%80" title="잉글랜드 축구 국가대표팀">잉글랜드 축구 국가대표팀</a>에서 <a href="/wiki/%EB%AF%B8%EB%93%9C%ED%95%84%EB%8D%94" title="미드필더">미드필더</a>로 활약하고 있다. 그는 두 차례의 임대 신분으로 <a href="/wiki/%EC%85%B0%ED%95%84%EB%93%9C_%EC%9B%AC%EC%A6%88%EB%8D%B0%EC%9D%B4_FC" title="셰필드 웬즈데이 FC">셰필드 웬즈데이 FC</a>와 <a href="/wiki/%EB%A6%AC%EC%A6%88_%EC%9C%A0%EB%82%98%EC%9D%B4%ED%8B%B0%EB%93%9C_AFC" title="리즈 유나이티드 AFC">리즈 유나이티드 AFC</a>에서도 활약한 적이 있다.</p>

<h2><span class="mw-headline" id=".ED.81.B4.EB.9F.BD_.EA.B2.BD.EB.A0.A5">클럽 경력</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%A1%9C%EC%8A%A4_%EB%B0%94%ED%81%B4%EB%A6%AC&amp;action=edit&amp;section=1" title="부분 편집: 클럽 경력">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>로스 바클리는 11세에 <a href="/wiki/%EC%97%90%EB%B2%84%ED%84%B4_FC" title="에버턴 FC">에버턴 FC</a>의 유소년 팀에 입단하였고, 클럽의 유소년 팀에서 활약하였다. 2010-11 시즌 초, 그는 1군 리그 경기에서 교체 명단에 이름을 올렸고, <a href="/wiki/%ED%94%84%EB%A6%AC%EB%AF%B8%EC%96%B4%EB%A6%AC%EA%B7%B8" title="프리미어리그">프리미어리그</a> 데뷔전을 치룰 것으로 기대되었으나<sup class="reference" id="cite_ref-4"><a href="#cite_note-4">[4]</a></sup> 2010년 10월, <a href="/wiki/%EB%A6%AC%EB%B2%84%ED%92%80_FC" title="리버풀 FC">리버풀 FC</a>와의 U-19팀 경기에서 <a href="/wiki/%EC%95%A4%EB%93%9C%EB%A6%AC_%EC%9C%84%EC%A6%88%EB%8D%A4" title="앤드리 위즈덤">앤드리 위즈덤</a>과의 충돌로 다리 골절 부상을 당하면서 무산되었다.<sup class="reference" id="cite_ref-5"><a href="#cite_note-5">[5]</a></sup></p>
<p>그는 부상으로부터 회복한 뒤, 2011-12 시즌을 시작하기 전에 1군 프리시즌 훈련에 참여하였다. 프리시즌 중에, <a href="/wiki/%ED%8C%80_%EC%BC%80%EC%9D%B4%ED%9E%90" title="팀 케이힐">팀 케이힐</a>은 바클리를 자신이 동행해본 선수들 중 가장 재능있는 선수로 꼽았다.<sup class="reference" id="cite_ref-6"><a href="#cite_note-6">[6]</a></sup> 그는 0-1로 패한 <a class="mw-redirect" href="/wiki/%ED%80%B8%EC%A6%88_%ED%8C%8C%ED%81%AC_%EB%A0%88%EC%9D%B8%EC%A0%80%EC%8A%A4_FC" title="퀸즈 파크 레인저스 FC">퀸즈 파크 레인저스 FC</a>와의 에버턴 정규리그 첫 경기에서 데뷔하였고, 라디오 시티 스포츠에서 경기 MVP로 선정되었다.<sup class="reference" id="cite_ref-7"><a href="#cite_note-7">[7]</a></sup> 그의 성과는 <a class="new" href="/w/index.php?title=%EB%A7%88%ED%8B%B4_%EC%BC%80%EC%98%A4%EC%9A%B4&amp;action=edit&amp;redlink=1" title="마틴 케오운 (없는 문서)">마틴 케오운</a>이 "[바클리는] 국내에서 본 선수들 중 최고의 선수가 될 것이다"라고 예측하게 하는 것과 같이, 찬사를 받았다.<sup class="reference" id="cite_ref-8"><a href="#cite_note-8">[8]</a></sup> 2011년 12월, 그는 4년 반짜리 연장 계약을 체결하였다.<sup class="reference" id="cite_ref-9"><a href="#cite_note-9">[9]</a></sup></p>
<p>2012년 9월 14일, 바클리는 1달 임대계약으로 <a href="/wiki/%EC%85%B0%ED%95%84%EB%93%9C_%EC%9B%AC%EC%A6%88%EB%8D%B0%EC%9D%B4_FC" title="셰필드 웬즈데이 FC">셰필드 웬즈데이 FC</a>와 계약을 체결하였고,<sup class="reference" id="cite_ref-10"><a href="#cite_note-10">[10]</a></sup> 같은날 <a class="mw-redirect" href="/wiki/%EB%B8%8C%EB%9D%BC%EC%9D%B4%ED%84%B4_%EC%95%A4_%ED%98%B8%EB%B8%8C_%EC%95%A8%EB%B9%84%EC%96%B8_FC" title="브라이턴 앤 호브 앨비언 FC">브라이턴 앤 호브 앨비언 FC</a> 원정 경기에 출전하였고, 팀은 패하였다.<sup class="reference" id="cite_ref-11"><a href="#cite_note-11">[11]</a></sup> 그는 그 다음주에 <a href="/wiki/%EB%B3%BC%ED%84%B4_%EC%9B%90%EB%8D%94%EB%9F%AC%EC%8A%A4_FC" title="볼턴 원더러스 FC">볼턴 원더러스 FC</a>와의 경기에서 페널티킥으로 임대 이적 후 첫 골을 득점하였다.<sup class="reference" id="cite_ref-12"><a href="#cite_note-12">[12]</a></sup> 그의 임대 기간은 에버턴이 다시 복귀 요청을 하기 전까지 연장되었고, 그는 원즈데이 소속으로 13경기를 치루었다.<sup class="reference" id="cite_ref-13"><a href="#cite_note-13">[13]</a></sup><sup class="reference" id="cite_ref-14"><a href="#cite_note-14">[14]</a></sup> 2013년 1월, 그는 "최초 1개월"이란 조건으로 <a href="/wiki/%EB%A6%AC%EC%A6%88_%EC%9C%A0%EB%82%98%EC%9D%B4%ED%8B%B0%EB%93%9C_AFC" title="리즈 유나이티드 AFC">리즈 유나이티드 AFC</a>와 임대 계약을 체결하였고,<sup class="reference" id="cite_ref-15"><a href="#cite_note-15">[15]</a></sup><sup class="reference" id="cite_ref-16"><a href="#cite_note-16">[16]</a></sup> <a href="/wiki/%EB%B0%98%EC%A6%90%EB%A6%AC_FC" title="반즐리 FC">반즐리 FC</a>와의 더비 경기에서 리즈 데뷔전을 치루었다.<sup class="reference" id="cite_ref-17"><a href="#cite_note-17">[17]</a></sup></p>
<p><a class="new" href="/w/index.php?title=%EC%97%90%EB%B2%84%ED%84%B4_FC_2013-14_%EC%8B%9C%EC%A6%8C&amp;action=edit&amp;redlink=1" title="에버턴 FC 2013-14 시즌 (없는 문서)">2013-14 시즌</a>에 바클리는 에버턴 1군에 다시 합류하였고, 2-2로 비긴 <a href="/wiki/%EB%85%B8%EB%A6%AC%EC%B9%98_%EC%8B%9C%ED%8B%B0_FC" title="노리치 시티 FC">노리치 시티 FC</a>와의 리그 개막전에서 에버턴 소속 첫골을 득점하였으며, 경기 MVP로 선정되었다.<sup class="reference" id="cite_ref-18"><a href="#cite_note-18">[18]</a></sup><sup class="reference" id="cite_ref-19"><a href="#cite_note-19">[19]</a></sup> 시즌 전반기에 바클리는 <a href="/wiki/%EC%95%84%EC%8A%A4%EB%84%90_FC" title="아스널 FC">아스널 FC</a>, <a href="/wiki/%EC%8A%A4%EC%99%84%EC%A7%80_%EC%8B%9C%ED%8B%B0_AFC" title="스완지 시티 AFC">스완지 시티 AFC</a>전 등의 여러 경기에서 MVP급 활약을 펼쳤고, 스완지 시티를 상대로는 결승골까지 득점하였다.<sup class="reference" id="cite_ref-20"><a href="#cite_note-20">[20]</a></sup> 그는 <a class="new" href="/w/index.php?title=%EC%95%8C%EB%9E%80_%ED%95%9C%EC%84%BC&amp;action=edit&amp;redlink=1" title="알란 한센 (없는 문서)">알란 한센</a>을 비롯한 여러 평론가들로부터 모든 것을 갖춘 선수라 찬사를 받았고, <a href="/wiki/%EA%B2%8C%EB%A6%AC_%EB%A6%AC%EB%84%A4%EC%BB%A4" title="게리 리네커">게리 리네커</a>는 천부적인 재능을 갖춘 스타 선수라 주장하였다.<sup class="reference" id="cite_ref-21"><a href="#cite_note-21">[21]</a></sup><sup class="reference" id="cite_ref-22"><a href="#cite_note-22">[22]</a></sup> 클럽 감독 <a href="/wiki/%EB%A1%9C%EB%B2%A0%EB%A5%B4%ED%86%A0_%EB%A7%88%EB%A5%B4%ED%8B%B0%EB%84%A4%EC%8A%A4" title="로베르토 마르티네스">로베르토 마르티네스</a>는 바클리를 <a href="/wiki/%ED%8F%B4_%EA%B0%9C%EC%8A%A4%EC%BD%94%EC%9D%B8" title="폴 개스코인">폴 개스코인</a>과 <a href="/wiki/%EB%AF%B8%ED%95%98%EC%97%98_%EB%B0%9C%EB%9D%BC%ED%81%AC" title="미하엘 발라크">미하엘 발라크</a>를 합친 형태라고 묘사하였다.<sup class="reference" id="cite_ref-23"><a href="#cite_note-23">[23]</a></sup></p>
<h2><span class="mw-headline" id=".EA.B5.AD.EA.B0.80.EB.8C.80.ED.91.9C.ED.8C.80_.EA.B2.BD.EB.A0.A5">국가대표팀 경력</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%A1%9C%EC%8A%A4_%EB%B0%94%ED%81%B4%EB%A6%AC&amp;action=edit&amp;section=2" title="부분 편집: 국가대표팀 경력">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<div class="thumb tleft">

<p>바클리는 그의 할아버지가 <a href="/wiki/%EB%82%98%EC%9D%B4%EC%A7%80%EB%A6%AC%EC%95%84" title="나이지리아">나이지리아</a> 사람이기 때문에 <a href="/wiki/%EB%82%98%EC%9D%B4%EC%A7%80%EB%A6%AC%EC%95%84_%EC%B6%95%EA%B5%AC_%EA%B5%AD%EA%B0%80%EB%8C%80%ED%91%9C%ED%8C%80" title="나이지리아 축구 국가대표팀">나이지리아 축구 국가대표팀</a>을 선택 할 수도 있었다. 그러나 바클리는 잉글랜드의 U-16, U-17, U-19, U-20, U-21, 그리고 성인 국가대표팀에 차출된 경험이 있다.<sup class="reference" id="cite_ref-thefa-profile_24-0"><a href="#cite_note-thefa-profile-24">[24]</a></sup> 2008년 9월, 그는 14세의 나이에 U-16 국가대표팀에 차출되었다.<sup class="reference" id="cite_ref-25"><a href="#cite_note-25">[25]</a></sup></p>
<p>그는 2009년 <a class="new" href="/w/index.php?title=%EB%AA%BD%ED%83%80%EC%9D%B4%EA%B7%B8_%ED%86%A0%EB%84%88%EB%A8%BC%ED%8A%B8&amp;action=edit&amp;redlink=1" title="몽타이그 토너먼트 (없는 문서)">몽타이그 토너먼트</a>의 스쿼드로 차출되어 성공으로 이끌었는데, 잉글랜드는 이 대회 결승전에서 독일을 승부차기에서 2-1로 꺾었다.<sup class="reference" id="cite_ref-26"><a href="#cite_note-26">[26]</a></sup> 그는 <a class="new" href="/w/index.php?title=2010%EB%85%84_UEFA_U-17_%EC%B6%95%EA%B5%AC_%EC%84%A0%EC%88%98%EA%B6%8C_%EB%8C%80%ED%9A%8C&amp;action=edit&amp;redlink=1" title="2010년 UEFA U-17 축구 선수권 대회 (없는 문서)">2010년 UEFA U-17 축구 선수권 대회</a>에서 결승전으로 오르는 와중에 두골을 득점하였고, 이 대회 결승전에서 잉글랜드는 스페인을 2-1로 꺾었다.<sup class="reference" id="cite_ref-27"><a href="#cite_note-27">[27]</a></sup> 2011년 8월, 바클리는 잉글랜드 U-21 국가대표팀에 차출되었다.<sup class="reference" id="cite_ref-28"><a href="#cite_note-28">[28]</a></sup></p>
<p>2013년 5월 28일, 그는 <a class="new" href="/w/index.php?title=%ED%94%BC%ED%84%B0_%EC%A1%B4_%ED%85%8C%EC%9D%BC%EB%9F%AC&amp;action=edit&amp;redlink=1" title="피터 존 테일러 (없는 문서)">피터 테일러</a>에 의해 21인 <a href="/wiki/2013%EB%85%84_FIFA_U-20_%EC%9B%94%EB%93%9C%EC%BB%B5" title="2013년 FIFA U-20 월드컵">2013년 FIFA U-20 월드컵</a> 엔트리에 포함되었다.<sup class="reference" id="cite_ref-29"><a href="#cite_note-29">[29]</a></sup> 6월 16일, 그는 3-0으로 승리한 우루과이 U-20과의 연습 경기에서 U-20 국가대표팀 데뷔전을 치루었다.<sup class="reference" id="cite_ref-30"><a href="#cite_note-30">[30]</a></sup></p>
<p>2013년 8월 13일, 그는 잉글랜드 U-21 국가대표팀 첫 골을 6-0으로 승리한 스코틀랜드 U-21 국가대표팀과의 경기에서 기록하였다.<sup class="reference" id="cite_ref-31"><a href="#cite_note-31">[31]</a></sup></p>
<p>바클리는 이후 2013년 8월에 <a href="/wiki/%EC%9E%89%EA%B8%80%EB%9E%9C%EB%93%9C_%EC%B6%95%EA%B5%AC_%EA%B5%AD%EA%B0%80%EB%8C%80%ED%91%9C%ED%8C%80" title="잉글랜드 축구 국가대표팀">잉글랜드 성인대표팀</a>에 차출되었고,<sup class="reference" id="cite_ref-32"><a href="#cite_note-32">[32]</a></sup> 다음 달에 4-0으로 승리한 <a href="/wiki/%EB%AA%B0%EB%8F%84%EB%B0%94_%EC%B6%95%EA%B5%AC_%EA%B5%AD%EA%B0%80%EB%8C%80%ED%91%9C%ED%8C%80" title="몰도바 축구 국가대표팀">몰도바</a>와의 <a href="/wiki/%EC%9B%B8%EB%B8%94%EB%A6%AC_%EC%8A%A4%ED%83%80%EB%94%94%EC%9B%80" title="웸블리 스타디움">웸블리</a> <a href="/wiki/2014%EB%85%84_FIFA_%EC%9B%94%EB%93%9C%EC%BB%B5_%EC%98%88%EC%84%A0" title="2014년 FIFA 월드컵 예선">2014년 FIFA 월드컵 예선</a> 홈경기에서 <a href="/wiki/%EC%9E%AD_%EC%9C%8C%EC%85%94" title="잭 윌셔">잭 윌셔</a>와 교체투입되며 성인대표팀 첫 출전을 기록하였다.<sup class="reference" id="cite_ref-33"><a href="#cite_note-33">[33]</a></sup></p>
<h2><span class="mw-headline" id=".EC.82.AC.EC.83.9D.ED.99.9C">사생활</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%A1%9C%EC%8A%A4_%EB%B0%94%ED%81%B4%EB%A6%AC&amp;action=edit&amp;section=3" title="부분 편집: 사생활">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>바클리는 <a href="/wiki/%EB%A6%AC%EB%B2%84%ED%92%80" title="리버풀">리버풀</a>의 스카우스 구역에서 성장하였다.<sup class="reference" id="cite_ref-In_focus_34-0"><a href="#cite_note-In_focus-34">[34]</a></sup></p>
