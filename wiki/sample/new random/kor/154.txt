<p><b>이서우</b>(李瑞雨, <a href="/wiki/1633%EB%85%84" title="1633년">1633년</a> <a href="/wiki/3%EC%9B%94_1%EC%9D%BC" title="3월 1일">3월 1일</a> ~ <a href="/wiki/1709%EB%85%84" title="1709년">1709년</a> <a href="/wiki/10%EC%9B%94_14%EC%9D%BC" title="10월 14일">10월 14일</a>)는 <a href="/wiki/%EC%A1%B0%EC%84%A0" title="조선">조선</a>시대 후기의 문신이며 시인, 작가, 서예가이다. 본관은 <a href="/wiki/%EC%9A%B0%EA%B3%84_%EC%9D%B4%EC%94%A8" title="우계 이씨">우계</a>(羽溪), 자(字)는 윤보(潤甫)·휴징<sup class="reference" id="cite_ref-2"><a href="#cite_note-2">[2]</a></sup>(休徵), 호(號)는 송곡(松谷)·송파(松坡)·구계(癯溪)·계옹(溪翁)이다. <a href="/wiki/1660%EB%85%84" title="1660년">1660년</a> 과거에 급제하여 벼슬은 <a href="/wiki/%EA%B3%B5%EC%A1%B0" title="공조">공조</a><a href="/wiki/%EC%B0%B8%ED%8C%90" title="참판">참판</a>(工曹參判), <a href="/wiki/%EC%98%88%EB%AC%B8%EA%B4%80" title="예문관">예문관</a> <a href="/wiki/%EC%A0%9C%ED%95%99" title="제학">제학</a>(藝文館提學)에 이르렀으며, <a class="mw-redirect" href="/wiki/%EC%98%88%EC%86%A1_%EB%85%BC%EC%9F%81" title="예송 논쟁">예송 논쟁</a> 당시 <a href="/wiki/%EB%82%A8%EC%9D%B8" title="남인">남인</a>(南人)의 논객 중의 한사람이다.</p>
<p>고산 <a href="/wiki/%EC%9C%A4%EC%84%A0%EB%8F%84" title="윤선도">윤선도</a>와 백호 <a href="/wiki/%EC%9C%A4%ED%9C%B4" title="윤휴">윤휴</a>의 문하에서 수학하였으며, 미수 <a href="/wiki/%ED%97%88%EB%AA%A9" title="허목">허목</a>(許穆)의 문하에서도 수학하였다. 그는 후일 <a href="/wiki/%ED%97%88%EB%AA%A9" title="허목">허목</a>·<a href="/wiki/%EC%9C%A4%ED%9C%B4" title="윤휴">윤휴</a>의 학통을 <a href="/wiki/%EC%9D%B4%EC%9D%B5_(1681%EB%85%84)" title="이익 (1681년)">이익</a>과 <a class="new" href="/w/index.php?title=%EC%98%A4%EC%83%81%EB%A0%B4&amp;action=edit&amp;redlink=1" title="오상렴 (없는 문서)">오상렴</a>, <a class="new" href="/w/index.php?title=%EC%B1%84%ED%8C%BD%EC%9C%A4&amp;action=edit&amp;redlink=1" title="채팽윤 (없는 문서)">채팽윤</a>, <a href="/wiki/%EC%B1%84%EC%A0%9C%EA%B3%B5" title="채제공">채제공</a> 등에게로 전수하였다. 본래 <a class="mw-disambig" href="/wiki/%EB%8C%80%EB%B6%81" title="대북">대북</a>(大北)계열 집안에서 태어났으나 <a href="/wiki/%EB%82%A8%EC%9D%B8" title="남인">남인</a>(南人)으로 전향했다. <a href="/wiki/1651%EB%85%84" title="1651년">1651년</a>(효종 2) <a href="/wiki/%EC%83%9D%EC%9B%90" title="생원">생원</a>(生員)이 되고, <a href="/wiki/1660%EB%85%84" title="1660년">1660년</a>(현종 1) <a class="new" href="/w/index.php?title=%EC%A6%9D%EA%B4%91&amp;action=edit&amp;redlink=1" title="증광 (없는 문서)">증광</a><a class="mw-redirect" href="/wiki/%EB%AC%B8%EA%B3%BC" title="문과">문과</a>에 갑과로 급제, <a href="/wiki/%EB%B6%81%EC%9D%B8" title="북인">북인</a>가문의 후예라는 이름으로 관직 출사에 제한을 받았으나 문장으로 <a href="/wiki/%ED%97%88%EB%AA%A9" title="허목">허목</a>과 <a href="/wiki/%EC%9C%A4%ED%9C%B4" title="윤휴">윤휴</a>의 천거로 청요직에 올랐다. 이후 <a href="/wiki/%EB%82%A8%EC%9D%B8" title="남인">남인</a>으로서 <a href="/wiki/1675%EB%85%84" title="1675년">1675년</a>(<a href="/wiki/%EC%A1%B0%EC%84%A0_%EC%88%99%EC%A2%85" title="조선 숙종">숙종</a> 1) <a href="/wiki/%EC%82%AC%EA%B0%84%EC%9B%90" title="사간원">사간원</a><a class="new" href="/w/index.php?title=%EC%A0%95%EC%96%B8&amp;action=edit&amp;redlink=1" title="정언 (없는 문서)">정언</a>(正言), <a href="/wiki/1679%EB%85%84" title="1679년">1679년</a> <a href="/wiki/%EB%8F%99%EB%9E%98%EA%B5%AC" title="동래구">동래</a> 부사(東萊府使)를 지냈으며 <a href="/wiki/%EC%84%9C%EC%9D%B8" title="서인">서인</a>(西人) <a href="/wiki/%EC%86%A1%EC%8B%9C%EC%97%B4" title="송시열">송시열</a>, <a href="/wiki/%EA%B9%80%EC%88%98%ED%95%AD" title="김수항">김수항</a>의 예론을 비판, 공격하는데 앞장섰다. <a href="/wiki/1680%EB%85%84" title="1680년">1680년</a> <a href="/wiki/%EA%B2%BD%EC%8B%A0%ED%99%98%EA%B5%AD" title="경신환국">경신환국</a>으로 서인의 공격을 받아 <a class="mw-redirect" href="/wiki/%EC%98%81%EB%B3%80%EA%B5%B0" title="영변군">영변</a>에 유배되었다.</p>
<p><a href="/wiki/1688%EB%85%84" title="1688년">1688년</a>(숙종 14년) <a class="mw-redirect" href="/wiki/%EC%9D%98%EC%A3%BC" title="의주">의주</a><a href="/wiki/%EB%B6%80%EC%9C%A4" title="부윤">부윤</a>으로 특별히 임명된 뒤 1689년 <a href="/wiki/%EA%B8%B0%EC%82%AC%ED%99%98%EA%B5%AD" title="기사환국">기사환국</a>으로 <a href="/wiki/%EB%82%A8%EC%9D%B8" title="남인">남인</a>이 집권하자 내직으로 돌아와 병조참의로 등용되었다. <a href="/wiki/1689%EB%85%84" title="1689년">1689년</a> 승지(承旨), <a href="/wiki/%EC%82%AC%EA%B0%84%EC%9B%90" title="사간원">사간원</a><a class="mw-redirect" href="/wiki/%EB%8C%80%EC%82%AC%EA%B0%84" title="대사간">대사간</a>, <a href="/wiki/%EB%8C%80%EC%82%AC%ED%97%8C" title="대사헌">대사헌</a> 등을 지내고 그 후 <a href="/wiki/%ED%99%A9%ED%95%B4%EB%8F%84" title="황해도">황해도</a> <a href="/wiki/%EA%B4%80%EC%B0%B0%EC%82%AC" title="관찰사">관찰사</a>와 <a href="/wiki/%ED%95%A8%EA%B2%BD%EB%8F%84" title="함경도">함경도</a> <a href="/wiki/%EA%B4%80%EC%B0%B0%EC%82%AC" title="관찰사">관찰사</a>로 부임하였다. 1692년 좌의정 목래선(睦來善)의 천거로 <a href="/wiki/%EC%98%88%EB%AC%B8%EA%B4%80" title="예문관">예문관</a> <a href="/wiki/%EC%A0%9C%ED%95%99" title="제학">제학</a>(藝文館提學)에 올랐다. 이듬해 <a href="/wiki/%EA%B3%B5%EC%A1%B0" title="공조">공조</a> <a href="/wiki/%EC%B0%B8%ED%8C%90" title="참판">참판</a>(工曹參判)에 이르렀으나 1694년(숙종 20) <a class="mw-redirect" href="/wiki/%EA%B0%91%EC%88%A0%EC%98%A5%EC%82%AC" title="갑술옥사">갑술옥사</a>(甲戌獄事)로 삭직(削職), 1697년 복관되었다. 시문(詩文)에 뛰어나고 글씨로 이름이 높았다. <a href="/wiki/%EC%82%AC%EB%B3%B5%EC%8B%9C" title="사복시">사복시</a><a class="new" href="/w/index.php?title=%EC%B2%A8%EC%A0%95&amp;action=edit&amp;redlink=1" title="첨정 (없는 문서)">첨정</a> <a class="mw-redirect" href="/wiki/%EC%A6%9D%EC%A7%81" title="증직">증</a> <a href="/wiki/%EC%98%88%EC%A1%B0" title="예조">예조</a><a href="/wiki/%ED%8C%90%EC%84%9C" title="판서">판서</a> <a href="/wiki/%EC%9D%B4%EA%B2%BD%ED%95%AD_(1601%EB%85%84)" title="이경항 (1601년)">이경항</a>(李慶恒)의 아들이다.</p>
<p>외직으로는 <a href="/wiki/%EA%B2%BD%EA%B8%B0%EB%8F%84" title="경기도">경기도</a>, <a href="/wiki/%ED%99%A9%ED%95%B4%EB%8F%84" title="황해도">황해도</a>, <a href="/wiki/%EA%B2%BD%EC%83%81%EB%8F%84" title="경상도">경상도</a>, <a href="/wiki/%EA%B0%95%EC%9B%90%EB%8F%84" title="강원도">강원도</a>, <a href="/wiki/%ED%95%A8%EA%B2%BD%EB%8F%84" title="함경도">함경도</a>의 <a href="/wiki/%EA%B4%80%EC%B0%B0%EC%82%AC" title="관찰사">관찰사</a>를 역임했고, 청렴하여 <a class="mw-redirect" href="/wiki/%EA%B0%95%EB%A6%89" title="강릉">강릉</a>, <a class="mw-redirect" href="/wiki/%EB%B6%80%EB%A0%B9" title="부령">부령</a>, <a href="/wiki/%EB%8F%99%EB%9E%98%EA%B5%AC" title="동래구">동래</a><a href="/wiki/%EB%B6%80%EC%82%AC" title="부사">부사</a>로 재직중 선정을 베풀어 인정비가 세워졌다. 서예와 글씨에 능하여 여러 명인들의 비문과 행장을 찬하였다. 문인 <a class="new" href="/w/index.php?title=%EA%B5%AD%ED%8F%AC_%EA%B0%95%EB%B0%95&amp;action=edit&amp;redlink=1" title="국포 강박 (없는 문서)">국포 강박</a>은 그의 외손이고, <a href="/wiki/%EC%9D%B4%EC%9D%B5_(1681%EB%85%84)" title="이익 (1681년)">이익</a>의 문하이자 <a class="mw-redirect" href="/wiki/%EC%B2%9C%EC%A3%BC%EA%B5%90" title="천주교">천주교</a>비판론자인 <a href="/wiki/%EC%8B%A0%ED%9B%84%EB%8B%B4" title="신후담">신후담</a>은 그의 외증손이다. <a href="/wiki/%EB%B0%9C%ED%95%B4%EA%B3%A0" title="발해고">발해고</a>의 저자 <a href="/wiki/%EC%9C%A0%EB%93%9D%EA%B3%B5" title="유득공">유득공</a>은 그의 외외증손이 된다. <a href="/wiki/%EC%9C%A4%EC%84%A0%EB%8F%84" title="윤선도">윤선도</a>, <a class="mw-disambig" href="/wiki/%EC%9D%B4%EB%AF%BC%EA%B5%AC" title="이민구">이민구</a>, <a href="/wiki/%ED%97%88%EB%AA%A9" title="허목">허목</a>, <a href="/wiki/%EC%9C%A4%ED%9C%B4" title="윤휴">윤휴</a>의 문인이다.</p>

<h2><span class="mw-headline" id=".EC.83.9D.EC.95.A0">생애</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%9D%B4%EC%84%9C%EC%9A%B0_(1633%EB%85%84)&amp;action=edit&amp;section=1" title="부분 편집: 생애">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<h3><span class="mw-headline" id=".EC.83.9D.EC.95.A0_.EC.B4.88.EA.B8.B0">생애 초기</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%9D%B4%EC%84%9C%EC%9A%B0_(1633%EB%85%84)&amp;action=edit&amp;section=2" title="부분 편집: 생애 초기">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<h4><span class="mw-headline" id=".EC.B6.9C.EC.83.9D.EA.B3.BC_.EA.B0.80.EA.B3.84">출생과 가계</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%9D%B4%EC%84%9C%EC%9A%B0_(1633%EB%85%84)&amp;action=edit&amp;section=3" title="부분 편집: 출생과 가계">편집</a><span class="mw-editsection-bracket">]</span></span></h4>
<p>이서우는 <a href="/wiki/1633%EB%85%84" title="1633년">1633년</a>(인조 10) <a href="/wiki/3%EC%9B%94_1%EC%9D%BC" title="3월 1일">3월 1일</a> 이경항과 전주이씨의 4남 3녀 중 넷째 아들로 한성에서 태어났다. 본관은 우계(羽溪)이고 자는 윤보(潤甫), 호는 송곡(松谷)이다. 당색으로는 <a class="mw-disambig" href="/wiki/%EB%8C%80%EB%B6%81" title="대북">대북</a> 계열의 가문에서 태어났다. <a class="mw-redirect" href="/wiki/%EC%9D%B4%EA%B0%90_(%EC%96%B8%EC%8B%A0)" title="이감 (언신)">이감</a>(李戡)의 4대손으로 <a href="/wiki/%EC%9D%B4%EC%84%B1%ED%97%8C" title="이성헌">이성헌</a>(李成憲)의 증손이며, 할아버지는 이길남(李吉男)이고 아버지는 <a href="/wiki/%EC%82%AC%EB%B3%B5%EC%8B%9C" title="사복시">사복시</a><a class="new" href="/w/index.php?title=%EC%B2%A8%EC%A0%95&amp;action=edit&amp;redlink=1" title="첨정 (없는 문서)">첨정</a>으로 사후 <a class="mw-redirect" href="/wiki/%EC%A6%9D%EC%A7%81" title="증직">증</a> <a href="/wiki/%EC%98%88%EC%A1%B0" title="예조">예조</a><a href="/wiki/%ED%8C%90%EC%84%9C" title="판서">판서</a>에 <a href="/wiki/%EC%B6%94%EC%A6%9D" title="추증">추증</a>된 <a href="/wiki/%EC%9D%B4%EA%B2%BD%ED%95%AD_(1601%EB%85%84)" title="이경항 (1601년)">이경항</a>(李慶恒)이고 어머니는 <a href="/wiki/%EC%A1%B0%EC%84%A0" title="조선">조선</a> <a href="/wiki/%EC%A1%B0%EC%84%A0_%ED%83%9C%EC%A2%85" title="조선 태종">태종</a>의 차남 <a href="/wiki/%ED%9A%A8%EB%A0%B9%EB%8C%80%EA%B5%B0" title="효령대군">효령대군</a> 이보(孝寧大君 李補)의 7대손으로 <a href="/wiki/%EC%88%98%EA%B5%B0%EC%A0%88%EB%8F%84%EC%82%AC" title="수군절도사">수군절도사</a> <a href="/wiki/%EC%9D%B4%EA%B2%BD%EC%9C%A0" title="이경유">이경유</a>(李慶裕)의 딸이며 <a class="mw-disambig" href="/wiki/%EC%9D%B4%EA%B2%BD%EB%A1%9D" title="이경록">이경록</a>의 조카딸인 전주이씨(全州李氏)이다.</p>
<p>그가 태어날 때 형 이덕우(李德雨), 이가우(李嘉雨), 이희우(李喜雨)와 후일 파평윤씨 <a href="/wiki/%EA%B0%90%EC%82%AC" title="감사">감사</a>(監司) <a class="new" href="/w/index.php?title=%EC%9C%A4%EB%93%9D%EC%97%B4&amp;action=edit&amp;redlink=1" title="윤득열 (없는 문서)">윤득열</a>(尹得說) 의 후처로 출가한 누나 이씨, 진사 이서규(李瑞圭)에게 출가한 누나 이씨 등이 있었다. 둘째 형 <a class="new" href="/w/index.php?title=%EC%9D%B4%EA%B0%80%EC%9A%B0&amp;action=edit&amp;redlink=1" title="이가우 (없는 문서)">이가우</a>는 <a class="new" href="/w/index.php?title=%EA%B9%80%EC%84%B8%EB%A0%B4&amp;action=edit&amp;redlink=1" title="김세렴 (없는 문서)">김세렴</a>의 사위이며 반계 <a href="/wiki/%EC%9C%A0%ED%98%95%EC%9B%90" title="유형원">유형원</a>의 절친한 친구이기도 했다.</p>
<p>그의 집안은 훈구파와 대북 계열이었으나, 그의 외당숙인 완풍부원군 <a class="mw-disambig" href="/wiki/%EC%9D%B4%EC%84%9C" title="이서">이서</a>(李曙)는 <a href="/wiki/%EC%9D%B8%EC%A1%B0%EB%B0%98%EC%A0%95" title="인조반정">인조반정</a>에 가담한 반정공신이자 <a href="/wiki/%EC%84%9C%EC%9D%B8" title="서인">서인</a> 중진 문신이었다. 따라서 그의 아버지 <a class="mw-disambig" href="/wiki/%EC%9D%B4%EA%B2%BD%ED%95%AD" title="이경항">이경항</a>은 처사촌인 <a class="mw-disambig" href="/wiki/%EC%9D%B4%EC%84%9C" title="이서">이서</a>가 힘을 써주어 불이익을 받지 않을 수 있었다.<sup class="reference" id="cite_ref-3"><a href="#cite_note-3">[3]</a></sup> 그러나 어려서 아버지를 잃고 홀어머니와 세 형 슬하에서 고학을 하였다.</p>
<h4><span class="mw-headline" id=".EA.B3.A0.ED.95.99_.EC.8B.9C.EC.A0.88">고학 시절</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%9D%B4%EC%84%9C%EC%9A%B0_(1633%EB%85%84)&amp;action=edit&amp;section=4" title="부분 편집: 고학 시절">편집</a><span class="mw-editsection-bracket">]</span></span></h4>
<p>고산 <a href="/wiki/%EC%9C%A4%EC%84%A0%EB%8F%84" title="윤선도">윤선도</a>의 문하에서 학문과 시문을 수학하였고, <a href="/wiki/%EC%9D%B4%EC%88%98%EA%B4%91" title="이수광">이수광</a>의 아들인 동주 <a class="mw-disambig" href="/wiki/%EC%9D%B4%EB%AF%BC%EA%B5%AC" title="이민구">이민구</a>의 문하에도 출입하며 수학하였다. 이후 백호 <a href="/wiki/%EC%9C%A4%ED%9C%B4" title="윤휴">윤휴</a>의 문하에도 출입하여 가르침을 청한다. 그 뒤 성인이 된 뒤에는 <a class="mw-redirect" href="/wiki/%EB%82%A8%EC%82%B0%EA%B3%A8" title="남산골">남산골</a>에 거주하며 학문에 전념하였다. 나중에는 일찍이 아버지 <a class="mw-disambig" href="/wiki/%EC%9D%B4%EA%B2%BD%ED%95%AD" title="이경항">이경항</a>과 한 동리에 살았던 미수 <a href="/wiki/%ED%97%88%EB%AA%A9" title="허목">허목</a>(許穆)의 문하에 찾아가 수학하였다. 후일 그가 <a href="/wiki/%EB%B6%81%EC%9D%B8" title="북인">북인</a> 출신이라는 이유로 <a href="/wiki/%ED%97%88%EC%A0%81" title="허적">허적</a>, <a href="/wiki/%EA%B6%8C%EB%8C%80%EC%9A%B4" title="권대운">권대운</a> 등의 공격을 받게 되자 그의 스승인 <a href="/wiki/%ED%97%88%EB%AA%A9" title="허목">허목</a>과 <a href="/wiki/%EC%9C%A4%ED%9C%B4" title="윤휴">윤휴</a>가 나서서 그를 변호하고 감싸주었다. 특히 스승 <a href="/wiki/%EC%9C%A4%ED%9C%B4" title="윤휴">윤휴</a>로부터는 실학자 지봉 <a href="/wiki/%EC%9D%B4%EC%88%98%EA%B4%91" title="이수광">이수광</a>과 동주(東洲) <a class="mw-disambig" href="/wiki/%EC%9D%B4%EB%AF%BC%EA%B5%AC" title="이민구">이민구</a>(李敏求)의 시문과 학문을 계승하였다.</p>
<p><a class="mw-disambig" href="/wiki/%EC%88%99%EC%A2%85" title="숙종">숙종</a>이 세자시절 잠저에 나갔을 때 내관과 함께 서울 남산골에 들렀다가 스러져가는 초가에서 한 선비가 낭랑하게 글을 읽는 모습을 보았다. 추운 겨울 밤, 온 종일 죽 한 그릇도 변변히 먹지 못한 채 얼음장 같은 냉방에 웅크리고 앉아서 책을 읽고 있었다.<sup class="reference" id="cite_ref-4"><a href="#cite_note-4">[4]</a></sup> 그러나 목이 메여 목소리가 나오지 않는 것을 숙종이 우연히 보았다. 숙종은 별감을 시켜 약밥에 마제 은 한덩이를 봉지에 싸서 선비의 집으로 던졌다. 창 너머로 무엇인가 방바닥에 떨어지는 것이 있었다. 배가 고팠던 선비는 누군가 던져주고 간 약밥을 감사히 취하였다.</p>
<p><a class="mw-disambig" href="/wiki/%EC%88%99%EC%A2%85" title="숙종">숙종</a>이 즉위한 후, 어느 정월 대보름날 잔치에 밝은 달을 보며 당시 서울 남산골에서 글읽던 선비를 추억하였다.</p>

<h3><span class="mw-headline" id=".EA.B4.80.EB.A3.8C_.EC.83.9D.ED.99.9C">관료 생활</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%9D%B4%EC%84%9C%EC%9A%B0_(1633%EB%85%84)&amp;action=edit&amp;section=5" title="부분 편집: 관료 생활">편집</a><span class="mw-editsection-bracket">]</span></span></h3>

<p><a href="/wiki/1651%EB%85%84" title="1651년">1651년</a>(효종 2년) <a class="new" href="/w/index.php?title=%EC%83%9D%EC%9B%90%EC%8B%9C&amp;action=edit&amp;redlink=1" title="생원시 (없는 문서)">생원시</a>험에 합격하여 <a href="/wiki/%EC%83%9D%EC%9B%90" title="생원">생원</a>(生員)이 되고, <a href="/wiki/%EC%84%B1%EA%B7%A0%EA%B4%80" title="성균관">성균관</a>(成均館)에 들어가 <a href="/wiki/%EC%84%B1%EA%B7%A0%EA%B4%80" title="성균관">성균관</a>유생이 되어 수학하였다.</p>
<p><a href="/wiki/1660%EB%85%84" title="1660년">1660년</a>(현종 1년) <a href="/wiki/%EC%84%B1%EA%B7%A0%EA%B4%80" title="성균관">성균관</a>유생으로 있을 때, 과제(課製)에 1등하여 바로 전시에 응시할 수 있는 자격이 주어졌으며, <a href="/wiki/1660%EB%85%84" title="1660년">1660년</a> <a href="/wiki/10%EC%9B%94_10%EC%9D%BC" title="10월 10일">10월 10일</a>의 <a class="new" href="/w/index.php?title=%EC%A6%9D%EA%B4%91&amp;action=edit&amp;redlink=1" title="증광 (없는 문서)">증광</a><a class="mw-redirect" href="/wiki/%EB%AC%B8%EA%B3%BC" title="문과">문과</a>(增廣文科)에 갑과(甲科)로 급제하였다. 이후 <a href="/wiki/%EC%A7%81%EC%9E%A5" title="직장">직장</a>(直長), <a href="/wiki/1668%EB%85%84" title="1668년">1668년</a>(현종 9) <a class="new" href="/w/index.php?title=%ED%86%B5%ED%9B%88%EB%8C%80%EB%B6%80&amp;action=edit&amp;redlink=1" title="통훈대부 (없는 문서)">통훈대부</a> 행<a href="/wiki/%EC%84%B1%EA%B7%A0%EA%B4%80" title="성균관">성균관</a><a class="new" href="/w/index.php?title=%EC%A0%84%EC%A0%81&amp;action=edit&amp;redlink=1" title="전적 (없는 문서)">전적</a>을 지냈다.</p>
<p>글을 잘 짓는다 하여 칭찬을 받았으며, 이후 <a href="/wiki/%EB%82%A8%EC%9D%B8" title="남인">남인</a>의 저격수로 <a class="new" href="/w/index.php?title=%EC%B2%AD%EB%82%A8&amp;action=edit&amp;redlink=1" title="청남 (없는 문서)">청남</a>에 가담하여 활동하였다. <a class="new" href="/w/index.php?title=%EC%B2%AD%EB%82%A8&amp;action=edit&amp;redlink=1" title="청남 (없는 문서)">청남</a>은 <a href="/wiki/%ED%97%88%EB%AA%A9" title="허목">허목</a>과 <a href="/wiki/%EC%9C%A4%ED%9C%B4" title="윤휴">윤휴</a>가 영수로 '<a class="new" href="/w/index.php?title=%EC%98%A4%EC%A0%95%EC%B0%BD&amp;action=edit&amp;redlink=1" title="오정창 (없는 문서)">오정창</a>·<a class="new" href="/w/index.php?title=%EC%98%A4%EC%A0%95%EC%9C%84&amp;action=edit&amp;redlink=1" title="오정위 (없는 문서)">오정위</a>·<a href="/wiki/%EC%98%A4%EC%8B%9C%EC%88%98" title="오시수">오시수</a>·<a href="/wiki/%EC%9D%B4%EB%AC%B4" title="이무">이무</a>·<a class="new" href="/w/index.php?title=%EC%A1%B0%EC%82%AC%EA%B8%B0&amp;action=edit&amp;redlink=1" title="조사기 (없는 문서)">조사기</a>·<a href="/wiki/%EC%9D%B4%EC%88%98%EA%B2%BD_(1627%EB%85%84)" title="이수경 (1627년)">이수경</a>(李壽慶)을 중심으로 <a class="new" href="/w/index.php?title=%EC%9E%A5%EC%9D%91%EC%9D%BC&amp;action=edit&amp;redlink=1" title="장응일 (없는 문서)">장응일</a>·<a class="new" href="/w/index.php?title=%EC%A0%95%EC%A7%80%ED%98%B8&amp;action=edit&amp;redlink=1" title="정지호 (없는 문서)">정지호</a>·<a class="new" href="/w/index.php?title=%EB%82%A8%EC%B2%9C%ED%95%9C&amp;action=edit&amp;redlink=1" title="남천한 (없는 문서)">남천한</a>·이서우 등이 '매와 사냥개'처럼 전위 역할을 담당하였다. 이들은 삼복(三福)이라 하여 <a href="/wiki/%EC%9D%B8%ED%8F%89%EB%8C%80%EA%B5%B0" title="인평대군">인평대군</a>의 세 아들인 <a href="/wiki/%EB%B3%B5%EC%B0%BD%EA%B5%B0" title="복창군">복창군</a>·<a href="/wiki/%EB%B3%B5%EC%84%A0%EA%B5%B0" title="복선군">복선군</a>·<a href="/wiki/%EB%B3%B5%ED%8F%89%EA%B5%B0" title="복평군">복평군</a>을 끼고 후일을 대비하였다<sup class="reference" id="cite_ref-6"><a href="#cite_note-6">[6]</a></sup>' 한다.</p>
<h4><span class="mw-headline" id=".EA.B4.80.EB.A3.8C_.EC.83.9D.ED.99.9C_.EC.B4.88.EA.B8.B0">관료 생활 초기</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%9D%B4%EC%84%9C%EC%9A%B0_(1633%EB%85%84)&amp;action=edit&amp;section=7" title="부분 편집: 관료 생활 초기">편집</a><span class="mw-editsection-bracket">]</span></span></h4>
<p>1675년(숙종 1) 문장에 재주가 있다 하여 스승인 <a href="/wiki/%ED%97%88%EB%AA%A9" title="허목">허목</a> 등의 추천을 받았다. 같은해 정언에 천거되었으나 <a class="mw-disambig" href="/wiki/%EB%8C%80%EB%B6%81" title="대북">대북</a>이었던 <a class="mw-disambig" href="/wiki/%EC%9D%B4%EA%B2%BD%ED%95%AD" title="이경항">이경항</a>의 아들이라는 이유로 일부 <a href="/wiki/%EB%82%A8%EC%9D%B8" title="남인">남인</a>들의 반발이 거셌으나 <a href="/wiki/%ED%97%88%EB%AA%A9" title="허목">허목</a>이 강하게 관철시켜 성사되었다. <a href="/wiki/1675%EB%85%84" title="1675년">1675년</a>(<a href="/wiki/%EC%A1%B0%EC%84%A0_%EC%88%99%EC%A2%85" title="조선 숙종">숙종</a> 1) 사간원정언이 되어 <a href="/wiki/%EC%9D%B8%EC%A1%B0%EB%B0%98%EC%A0%95" title="인조반정">인조반정</a> 이후 대북가문출신으로는 처음으로 청직(淸職)에 올랐다.</p>
<p>그가 삼사의 청요직에 중용되자 그의 아버지 <a class="mw-disambig" href="/wiki/%EC%9D%B4%EA%B2%BD%ED%95%AD" title="이경항">이경항</a>이 <a class="mw-redirect" href="/wiki/%EA%B4%91%ED%95%B4%EA%B5%B0" title="광해군">광해군</a> 때 <a class="mw-disambig" href="/wiki/%EB%8C%80%EB%B6%81" title="대북">대북</a> 인사였음을 들어 반대하였다. 그러나 <a href="/wiki/%ED%97%88%EB%AA%A9" title="허목">허목</a>과 <a href="/wiki/%EC%9C%A4%ED%9C%B4" title="윤휴">윤휴</a>가 번갈아가며 그를 삼사 언관의 적임자로 추천하였다. 이때 허목은 "신이 그 아비와 더불어 같은 마을에 살았으므로, 그가 끝내 흉론(凶論)에 물들지 아니하였음을 압니다"하며 그를 감쌌고, 묵재 <a href="/wiki/%ED%97%88%EC%A0%81" title="허적">허적</a>은 "신이 듣건대, 그 아비가 <a href="/wiki/%EC%9D%B4%EC%9D%B4%EC%B2%A8" title="이이첨">이이첨</a>(李爾瞻)에게 친압(親押)받았다고 하는데, 지금 우상(右相)의 말이 이와 같으니, 신은 전해 들은 것에 지나지 아니하고 우상은 눈으로 본 것입니다. 그렇다면 우상의 말이 옳습니다. 우상이 어찌 한낱 이서우를 위하여 감히 전하를 속이겠습니까?"라며 그의 중용을 반대하였으나, <a href="/wiki/%ED%97%88%EB%AA%A9" title="허목">허목</a>의 변호로 무사하였다. <a href="/wiki/5%EC%9B%94" title="5월">5월</a> <a href="/wiki/%EA%B2%BD%EC%97%B0" title="경연">경연</a>에서 <a href="/wiki/%ED%97%88%EB%AA%A9" title="허목">허목</a>의 천거를 받았다.</p>
<p><a href="/wiki/1675%EB%85%84" title="1675년">1675년</a>(숙종 1년) <a href="/wiki/7%EC%9B%94_22%EC%9D%BC" title="7월 22일">7월 22일</a> <a href="/wiki/%EC%82%AC%EA%B0%84%EC%9B%90" title="사간원">사간원</a><a class="new" href="/w/index.php?title=%EC%A0%95%EC%96%B8&amp;action=edit&amp;redlink=1" title="정언 (없는 문서)">정언</a>이 되었다. 그 해 <a href="/wiki/%EC%84%9C%EC%9D%B8" title="서인">서인</a> <a href="/wiki/%EC%86%A1%EC%8B%9C%EC%97%B4" title="송시열">송시열</a>(宋時烈)의 예론과 그에 적극 동조한 <a href="/wiki/%EA%B9%80%EC%88%98%ED%95%AD" title="김수항">김수항</a>(金壽恒)을 공격하였으며, <a href="/wiki/7%EC%9B%94" title="7월">7월</a>에는 <a href="/wiki/%ED%97%88%EC%A0%81" title="허적">허적</a> 등 대신을 공격하는 이수경을 두호하다 파직당하였다. 그러나 그해 <a href="/wiki/8%EC%9B%94" title="8월">8월</a>부터 왕과 청대한 <a href="/wiki/%EC%9A%B0%EC%9D%98%EC%A0%95" title="우의정">우의정</a> <a href="/wiki/%ED%97%88%EB%AA%A9" title="허목">허목</a>의 극력 변호와 설득으로, 동년 <a href="/wiki/10%EC%9B%94_9%EC%9D%BC" title="10월 9일">10월 9일</a> 이수경과 함께 곧 복직되었다. 이어 <a href="/wiki/10%EC%9B%94_11%EC%9D%BC" title="10월 11일">10월 11일</a> <a href="/wiki/%EC%9C%A4%ED%9C%B4" title="윤휴">윤휴</a>의 추천으로 <a class="new" href="/w/index.php?title=%ED%99%8D%EB%AC%B8%EB%A1%9D&amp;action=edit&amp;redlink=1" title="홍문록 (없는 문서)">홍문록</a>에 오르게 되었다.<sup class="reference" id="cite_ref-suk751011_7-0"><a href="#cite_note-suk751011-7">[7]</a></sup> 이때 <a href="/wiki/%ED%97%88%EC%A0%81" title="허적">허적</a>이 다시 그의 아버지 <a class="mw-disambig" href="/wiki/%EC%9D%B4%EA%B2%BD%ED%95%AD" title="이경항">이경항</a>이 <a class="mw-disambig" href="/wiki/%EB%8C%80%EB%B6%81" title="대북">대북</a>이었던 점을 들어 <a href="/wiki/1623%EB%85%84" title="1623년">1623년</a> 이후 대북파, 대북계열인 사람들의 출사가 금지되었던 점을 지적하기도 했다.<sup class="reference" id="cite_ref-suk751011_7-1"><a href="#cite_note-suk751011-7">[7]</a></sup></p>

<p><a href="/wiki/1675%EB%85%84" title="1675년">1675년</a> <a href="/wiki/%EC%84%B1%EA%B7%A0%EA%B4%80" title="성균관">성균관</a><a class="mw-redirect" href="/wiki/%EC%82%AC%EC%98%88" title="사예">사예</a>를 거쳐 <a href="/wiki/1675%EB%85%84" title="1675년">1675년</a> <a href="/wiki/12%EC%9B%94_11%EC%9D%BC" title="12월 11일">12월 11일</a> <a href="/wiki/%EC%82%AC%ED%97%8C%EB%B6%80" title="사헌부">사헌부</a> 지평(司憲府 持平)이 되었으나 여러 번 소패(召牌)를 어긴 책임을 지고 스스로 사직을 청한 뒤 물러가서 처분을 기다렸다가. 이때 <a href="/wiki/%EB%82%A8%EC%9D%B8" title="남인">남인</a> <a href="/wiki/%ED%97%88%EC%A0%81" title="허적">허적</a>과 <a href="/wiki/%EC%84%9C%EC%9D%B8" title="서인">서인</a>(西人)은 그를 비판하였으나, 미수 <a href="/wiki/%ED%97%88%EB%AA%A9" title="허목">허목</a>(許穆)의 변호로 무사하였다. 이후 그는 <a href="/wiki/%ED%97%88%EB%AA%A9" title="허목">허목</a>과 <a href="/wiki/%EC%9C%A4%ED%9C%B4" title="윤휴">윤휴</a>의 비호를 받았으며, 그의 글재주를 높이 산 <a href="/wiki/%ED%97%88%EB%AA%A9" title="허목">허목</a>과 <a href="/wiki/%EC%9C%A4%ED%9C%B4" title="윤휴">윤휴</a>가 번갈아가며 그를 추천하여 삼사의 요직에 임명되었다. <a href="/wiki/12%EC%9B%94_26%EC%9D%BC" title="12월 26일">12월 26일</a> 다시 <a href="/wiki/%EC%82%AC%EA%B0%84%EC%9B%90" title="사간원">사간원</a><a class="new" href="/w/index.php?title=%EC%A0%95%EC%96%B8&amp;action=edit&amp;redlink=1" title="정언 (없는 문서)">정언</a>이 되었다.</p>
<p><a href="/wiki/1676%EB%85%84" title="1676년">1676년</a>(숙종 2년) <a href="/wiki/1%EC%9B%94" title="1월">1월</a> 만과 설치와 종이호패 발급의 폐단을 지적하여 폐지하게 하였다. 만과 설치 이후 백수들이 헛된 꿈을 바라는 일이 늘어났으며, 종이 호패는 흉년기에 발급하면 안된다는 것이었다. 그 뒤 <a href="/wiki/%EC%84%B1%EA%B7%A0%EA%B4%80" title="성균관">성균관</a> 사예를 거쳐 <a href="/wiki/%EB%B3%91%EC%A1%B0" title="병조">병조</a><a class="new" href="/w/index.php?title=%EC%A0%95%EB%9E%91&amp;action=edit&amp;redlink=1" title="정랑 (없는 문서)">정랑</a>으로 <a href="/wiki/%EB%B9%84%EB%B3%80%EC%82%AC" title="비변사">비변사</a>낭청이 되었고, <a href="/wiki/1676%EB%85%84" title="1676년">1676년</a> <a href="/wiki/7%EC%9B%94" title="7월">7월</a> <a href="/wiki/%EC%B2%AD%EB%82%98%EB%9D%BC" title="청나라">청나라</a>에 파견되는 <a class="new" href="/w/index.php?title=%EC%A7%84%ED%95%98%EC%82%AC&amp;action=edit&amp;redlink=1" title="진하사 (없는 문서)">진하사</a>(進賀使)의 <a class="new" href="/w/index.php?title=%EC%84%9C%EC%9E%A5%EA%B4%80&amp;action=edit&amp;redlink=1" title="서장관 (없는 문서)">서장관</a>으로 연경에 왕환하였다. 귀국 후 <a href="/wiki/1677%EB%85%84" title="1677년">1677년</a> <a href="/wiki/10%EC%9B%94_4%EC%9D%BC" title="10월 4일">10월 4일</a> <a class="new" href="/w/index.php?title=%EC%82%AC%ED%97%8C%EB%B6%80_%EC%9E%A5%EB%A0%B9&amp;action=edit&amp;redlink=1" title="사헌부 장령 (없는 문서)">사헌부 장령</a>이 되었다. 그해 <a href="/wiki/12%EC%9B%94" title="12월">12월</a> 다시 사헌부 장령이 되었다. 그 뒤 비변사의 낭관을 거쳐 <a href="/wiki/1678%EB%85%84" title="1678년">1678년</a> <a href="/wiki/6%EC%9B%94_16%EC%9D%BC" title="6월 16일">6월 16일</a> 다시 사헌부 장령이 되었다. 이후 <a href="/wiki/%EC%84%9C%EC%9D%B8" title="서인">서인</a>과 <a href="/wiki/%EB%82%A8%EC%9D%B8" title="남인">남인</a>에서 여러 번 그의 아버지 <a href="/wiki/%EC%9D%B4%EA%B2%BD%ED%95%AD_(1601%EB%85%84)" title="이경항 (1601년)">이경항</a>의 전력을 문제삼았으나 이때에도 <a href="/wiki/%ED%97%88%EB%AA%A9" title="허목">허목</a>과 <a href="/wiki/%EC%9C%A4%ED%9C%B4" title="윤휴">윤휴</a>가 번갈아가며 이서우를 두둔하였다. 이서우는 <a href="/wiki/%EC%84%9C%EC%9D%B8" title="서인">서인</a> <a href="/wiki/%EC%86%A1%EC%8B%9C%EC%97%B4" title="송시열">송시열</a>, <a href="/wiki/%EC%86%A1%EC%A4%80%EA%B8%B8" title="송준길">송준길</a>, <a href="/wiki/%EA%B9%80%EC%88%98%ED%95%AD" title="김수항">김수항</a> 등의 예론이 예를 거슬린다며 비판, 규탄하였다.</p>

<p><a href="/wiki/1679%EB%85%84" title="1679년">1679년</a> 1월 25일 <a href="/wiki/%EB%8F%99%EB%9E%98%EA%B5%AC" title="동래구">동래</a> 부사(東萊府使)로 부임한 후 <a class="new" href="/w/index.php?title=%EA%B0%95%EB%A6%89%EB%B6%80%EC%82%AC&amp;action=edit&amp;redlink=1" title="강릉부사 (없는 문서)">강릉부사</a>, <a class="new" href="/w/index.php?title=%EB%B6%80%EB%A0%B9%EB%B6%80%EC%82%AC&amp;action=edit&amp;redlink=1" title="부령부사 (없는 문서)">부령부사</a>, 청주목사 등을 지내고 <a href="/wiki/%EC%84%9C%EC%9D%B8" title="서인">서인</a> <a href="/wiki/%EC%86%A1%EC%8B%9C%EC%97%B4" title="송시열">송시열</a>, <a href="/wiki/%EA%B9%80%EC%88%98%ED%95%AD" title="김수항">김수항</a>의 예론을 비판, 공격하는데 앞장섰다. <a href="/wiki/%EC%84%9C%EC%9D%B8" title="서인">서인</a> <a href="/wiki/%EC%86%A1%EC%8B%9C%EC%97%B4" title="송시열">송시열</a>(宋時烈)의 예론과 그것을 따르는 김수항(金壽恒)을 공격하였으며, <a href="/wiki/7%EC%9B%94" title="7월">7월</a>에는 <a href="/wiki/%ED%97%88%EC%A0%81" title="허적">허적</a> 등 대신을 공격하는 이수경을 두호하다 파직당하였다. 그러나 <a href="/wiki/%ED%97%88%EB%AA%A9" title="허목">허목</a>의 변호로 복관하였다. <a href="/wiki/1676%EB%85%84" title="1676년">1676년</a> 7월 <a href="/wiki/%EC%B2%AD%EB%82%98%EB%9D%BC" title="청나라">청나라</a>에 파견되는 <a href="/wiki/%EB%8F%99%EC%A7%80%EC%82%AC" title="동지사">동지사</a>의 서장관으로 연경에 다녀왔다.</p>

<p><a href="/wiki/1680%EB%85%84" title="1680년">1680년</a> <a href="/wiki/4%EC%9B%94" title="4월">4월</a> <a href="/wiki/%EA%B2%BD%EC%8B%A0%ED%99%98%EA%B5%AD" title="경신환국">경신환국</a> 이후 서인의 공격을 받고 면직되었다. 그해 <a href="/wiki/9%EC%9B%94" title="9월">9월</a>에는 <a href="/wiki/%EA%B9%80%EC%88%98%ED%95%AD" title="김수항">김수항</a>으로부터 그가 <a class="new" href="/w/index.php?title=%EC%98%A4%EC%A0%95%EC%B0%BD&amp;action=edit&amp;redlink=1" title="오정창 (없는 문서)">오정창</a>(吳挺昌)의 심복인데 홀로 죄를 받지 않았다며 공격당하였다.<sup class="reference" id="cite_ref-9"><a href="#cite_note-9">[9]</a></sup></p>
<p><a href="/wiki/10%EC%9B%94" title="10월">10월</a> <a href="/wiki/%EC%84%9C%EC%9D%B8" title="서인">서인</a>에 의해 <a href="/wiki/%ED%99%8D%EC%88%98%EC%9D%98_%EB%B3%80" title="홍수의 변">홍수의 변</a> 관련자인 <a class="new" href="/w/index.php?title=%EC%98%A4%EC%A0%95%EC%B0%BD&amp;action=edit&amp;redlink=1" title="오정창 (없는 문서)">오정창</a>의 심복으로 지목되어 영변으로 유배당하였는데, 누명을 쓰고 유배되면서 그는 우울한 심사를 시로써 달랬다.</p>
<p><a href="/wiki/1688%EB%85%84" title="1688년">1688년</a> <a href="/wiki/5%EC%9B%94_2%EC%9D%BC" title="5월 2일">5월 2일</a> 특별히 <a class="new" href="/w/index.php?title=%EC%9D%98%EC%A3%BC%EB%B6%80%EC%9C%A4&amp;action=edit&amp;redlink=1" title="의주부윤 (없는 문서)">의주부윤</a>(義州府尹)으로 부임하였다. 이서우는 외직에 여러 번 물망에 올랐다. 그러나 당색이 남인이었던 까닭에 서인의 줄기찬 반대로 무마되었다. <a href="/wiki/1688%EB%85%84" title="1688년">1688년</a> <a href="/wiki/5%EC%9B%94" title="5월">5월</a><a href="/wiki/%EC%84%9C%EC%9D%B8" title="서인">서인</a>들이 주관한 비변사 회의에서 의주부윤에 천거되었을 때는 서인들도 반대하지 않아 그해 <a href="/wiki/5%EC%9B%94_2%EC%9D%BC" title="5월 2일">5월 2일</a> 의주부윤으로 취임할 수 있었다. 그러나 의주부윤 재직 중 <a class="mw-redirect" href="/wiki/%EC%9D%98%EC%A3%BC" title="의주">의주</a>로 유배된 복선군(福善君) 이남(李柟)과 시문을 주고받았다는 이유로 그해 <a href="/wiki/6%EC%9B%94_13%EC%9D%BC" title="6월 13일">6월 13일</a> <a href="/wiki/%EC%98%81%EC%9D%98%EC%A0%95" title="영의정">영의정</a> <a href="/wiki/%EB%82%A8%EA%B5%AC%EB%A7%8C" title="남구만">남구만</a>, 부제학 <a href="/wiki/%EC%B5%9C%EC%84%9D%EC%A0%95" title="최석정">최석정</a>(崔錫鼎) 등의 비판을 받고 바로 면직되었다.</p>

<div class="rellink boilerplate seealso"><img alt="" data-file-height="128" data-file-width="128" height="16" src="//upload.wikimedia.org/wikipedia/commons/thumb/e/ec/Crystal_Clear_app_xmag.svg/16px-Crystal_Clear_app_xmag.svg.png" srcset="//upload.wikimedia.org/wikipedia/commons/thumb/e/ec/Crystal_Clear_app_xmag.svg/24px-Crystal_Clear_app_xmag.svg.png 1.5x, //upload.wikimedia.org/wikipedia/commons/thumb/e/ec/Crystal_Clear_app_xmag.svg/32px-Crystal_Clear_app_xmag.svg.png 2x" width="16"/> <a href="/wiki/%EA%B8%B0%EC%82%AC%ED%99%98%EA%B5%AD" title="기사환국">기사환국</a>, <a class="new" href="/w/index.php?title=%EC%9D%B8%ED%98%84%EC%99%95%ED%9B%84_%ED%8F%90%EC%B6%9C_%EC%82%AC%EA%B1%B4&amp;action=edit&amp;redlink=1" title="인현왕후 폐출 사건 (없는 문서)">인현왕후 폐출 사건</a> 문서를 참고하십시오.</div>
<p><a href="/wiki/1689%EB%85%84" title="1689년">1689년</a>(숙종 15년) 1월 왕자 균의 원자정호 문제로 <a href="/wiki/2%EC%9B%94_6%EC%9D%BC" title="2월 6일">2월 6일</a> <a href="/wiki/%EA%B8%B0%EC%82%AC%ED%99%98%EA%B5%AD" title="기사환국">기사환국</a>으로 남인이 정권을 잡자 내직으로 돌아와 <a href="/wiki/%EB%B3%91%EC%A1%B0" title="병조">병조</a><a href="/wiki/%EC%B0%B8%EC%9D%98" title="참의">참의</a>(兵曺參議)로 등용되었다. 바로 <a class="new" href="/w/index.php?title=%EC%8A%B9%EC%A7%80&amp;action=edit&amp;redlink=1" title="승지 (없는 문서)">승지</a>(承旨)가 되었고, <a href="/wiki/2%EC%9B%94_15%EC%9D%BC" title="2월 15일">2월 15일</a> 다시 승지로 재임명되었으며 바로 <a href="/wiki/%EB%B3%91%EC%A1%B0" title="병조">병조</a><a href="/wiki/%EC%B0%B8%EC%9D%98" title="참의">참의</a>가 되었다. 그해 윤 3월 다른 남인 신료들과 합계하여 <a href="/wiki/%EC%86%A1%EC%8B%9C%EC%97%B4" title="송시열">송시열</a>을 사형에 처할 것을 주장하였다.이후 <a href="/wiki/4%EC%9B%94_23%EC%9D%BC" title="4월 23일">4월 23일</a> <a class="new" href="/w/index.php?title=%EC%8A%B9%EC%A7%80&amp;action=edit&amp;redlink=1" title="승지 (없는 문서)">승지</a>가 되었다. 8월 3일 다시 승지에 재임명되었다. 1689년 10월 3일 사간원 대사간, 사헌부 대사헌, 10월 23일 다시 승지가 되었다. 그 후 함경도 관찰사, <a href="/wiki/%ED%99%8D%EB%AC%B8%EA%B4%80" title="홍문관">홍문관</a> <a href="/wiki/%EB%8C%80%EC%A0%9C%ED%95%99" title="대제학">대제학</a>과 <a href="/wiki/%EC%98%88%EB%AC%B8%EA%B4%80" title="예문관">예문관</a> <a href="/wiki/%EB%8C%80%EC%A0%9C%ED%95%99" title="대제학">대제학</a>, <a href="/wiki/%EC%82%AC%ED%97%8C%EB%B6%80" title="사헌부">사헌부</a> <a href="/wiki/%EB%8C%80%EC%82%AC%ED%97%8C" title="대사헌">대사헌</a> 등을 두루 지냈다. 이 기간 중 그는 <a href="/wiki/%EC%84%9C%EC%9D%B8" title="서인">서인</a>이 잘못된 예론을 고치지 않는다며 <a href="/wiki/%EC%84%9C%EC%9D%B8" title="서인">서인</a>을 공격, 성토하였다.</p>
<p><a href="/wiki/1690%EB%85%84" title="1690년">1690년</a> <a href="/wiki/1%EC%9B%94_10%EC%9D%BC" title="1월 10일">1월 10일</a> <a href="/wiki/%EC%8A%B9%EC%A0%95%EC%9B%90" title="승정원">승정원</a> <a href="/wiki/%EB%8F%84%EC%8A%B9%EC%A7%80" title="도승지">도승지</a>(都承旨), 이후 <a class="mw-redirect" href="/wiki/%EC%98%A4%EC%9C%84%EB%8F%84%EC%B4%9D%EB%B6%80" title="오위도총부">오위도총부</a> 부총관을 지냈다. 그 뒤 <a href="/wiki/%EA%B9%80%EC%88%98%ED%95%AD" title="김수항">김수항</a>, <a href="/wiki/%EC%86%A1%EC%8B%9C%EC%97%B4" title="송시열">송시열</a> 등 <a href="/wiki/%EC%84%9C%EC%9D%B8" title="서인">서인</a>을 공격하는데 앞장섰으며, <a href="/wiki/%EC%9D%B8%ED%98%84%EC%99%95%ED%9B%84" title="인현왕후">인현왕후</a>(仁顯王后) 민씨의 폐출 때는 <a class="new" href="/w/index.php?title=%EC%8A%B9%EC%A7%80&amp;action=edit&amp;redlink=1" title="승지 (없는 문서)">승지</a>로 있으면서 <a class="mw-disambig" href="/wiki/%EC%88%99%EC%A2%85" title="숙종">숙종</a>의 뜻을 받들었다. <a href="/wiki/1690%EB%85%84" title="1690년">1690년</a> <a href="/wiki/5%EC%9B%94" title="5월">5월</a> <a href="/wiki/%ED%95%A8%EA%B2%BD%EB%8F%84" title="함경도">함경도</a> <a href="/wiki/%EA%B4%80%EC%B0%B0%EC%82%AC" title="관찰사">관찰사</a>가 되었다. <a href="/wiki/1691%EB%85%84" title="1691년">1691년</a> <a href="/wiki/%ED%95%A8%EA%B2%BD%EB%8F%84" title="함경도">함경도</a><a href="/wiki/%EA%B4%80%EC%B0%B0%EC%82%AC" title="관찰사">관찰사</a>로 나갔다가 그해 <a href="/wiki/4%EC%9B%94" title="4월">4월</a> <a href="/wiki/%EC%9D%B8%EC%82%BC" title="인삼">인삼</a>에 대한 행정처리 잘못과, 인삼 채취를 목적으로 월경하는 백성들을 통제하지 못했다는 이유로 <a href="/wiki/%EB%B9%84%EB%B3%80%EC%82%AC" title="비변사">비변사</a>로부터 탄핵을 받고 삭직당하였다.</p>
<h3><span class="mw-headline" id=".EC.83.9D.EC.95.A0_.ED.9B.84.EB.B0.98">생애 후반</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%9D%B4%EC%84%9C%EC%9A%B0_(1633%EB%85%84)&amp;action=edit&amp;section=11" title="부분 편집: 생애 후반">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<h4><span class="mw-headline" id=".EA.B0.91.EC.88.A0.ED.99.98.EA.B5.AD.EA.B3.BC_.EB.B3.B5.EA.B6.8C">갑술환국과 복권</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%9D%B4%EC%84%9C%EC%9A%B0_(1633%EB%85%84)&amp;action=edit&amp;section=12" title="부분 편집: 갑술환국과 복권">편집</a><span class="mw-editsection-bracket">]</span></span></h4>
<div class="rellink boilerplate seealso"><img alt="" data-file-height="128" data-file-width="128" height="16" src="//upload.wikimedia.org/wikipedia/commons/thumb/e/ec/Crystal_Clear_app_xmag.svg/16px-Crystal_Clear_app_xmag.svg.png" srcset="//upload.wikimedia.org/wikipedia/commons/thumb/e/ec/Crystal_Clear_app_xmag.svg/24px-Crystal_Clear_app_xmag.svg.png 1.5x, //upload.wikimedia.org/wikipedia/commons/thumb/e/ec/Crystal_Clear_app_xmag.svg/32px-Crystal_Clear_app_xmag.svg.png 2x" width="16"/> <a href="/wiki/%EA%B0%91%EC%88%A0%ED%99%98%EA%B5%AD" title="갑술환국">갑술환국</a> 문서를 참고하십시오.</div>
<p><a href="/wiki/1692%EB%85%84" title="1692년">1692년</a> <a href="/wiki/%EC%9D%98%EC%A0%95%EB%B6%80" title="의정부">의정부</a><a href="/wiki/%EC%A2%8C%EC%9D%98%EC%A0%95" title="좌의정">좌의정</a> <a class="mw-redirect" href="/wiki/%EB%AA%A9%EB%9E%98%EC%84%A0" title="목래선">목래선</a>(睦來善)에 의하여 문장으로 천거받아 <a class="new" href="/w/index.php?title=%EC%98%88%EB%AC%B8%EA%B4%80_%EC%A0%9C%ED%95%99&amp;action=edit&amp;redlink=1" title="예문관 제학 (없는 문서)">예문관 제학</a>이 되었으며, 동년 10월 5일 <a href="/wiki/%EA%B3%B5%EC%A1%B0" title="공조">공조</a><a href="/wiki/%EC%B0%B8%ED%8C%90" title="참판">참판</a>이 되었다. 11월 21일 다시 <a href="/wiki/%EC%8A%B9%EC%A0%95%EC%9B%90" title="승정원">승정원</a><a href="/wiki/%EB%8F%84%EC%8A%B9%EC%A7%80" title="도승지">도승지</a>가 되고, <a href="/wiki/1693%EB%85%84" title="1693년">1693년</a> <a href="/wiki/9%EC%9B%94_9%EC%9D%BC" title="9월 9일">9월 9일</a> <a href="/wiki/%ED%99%A9%ED%95%B4%EB%8F%84" title="황해도">황해도</a> <a href="/wiki/%EA%B4%80%EC%B0%B0%EC%82%AC" title="관찰사">관찰사</a>(黃海道觀察使), 9월 26일 경상도 관찰사(慶尙道觀察使), <a href="/wiki/1694%EB%85%84" title="1694년">1694년</a> 초 다시 <a href="/wiki/%ED%99%A9%ED%95%B4%EB%8F%84" title="황해도">황해도</a><a href="/wiki/%EA%B4%80%EC%B0%B0%EC%82%AC" title="관찰사">관찰사</a>로 나아갔다. 다시 공조 참판에 이르렀으나 <a href="/wiki/1694%EB%85%84" title="1694년">1694년</a> 4월 <a class="mw-redirect" href="/wiki/%EA%B0%91%EC%88%A0%EC%98%A5%EC%82%AC" title="갑술옥사">갑술옥사</a>로 파직, 삭출당하고 <a href="/wiki/4%EC%9B%94_16%EC%9D%BC" title="4월 16일">4월 16일</a> 문외출송되었다가 <a href="/wiki/1695%EB%85%84" title="1695년">1695년</a> 다시 사면령이 내려졌으나 <a class="new" href="/w/index.php?title=%EC%8A%B9%EC%A7%80&amp;action=edit&amp;redlink=1" title="승지 (없는 문서)">승지</a> <a class="new" href="/w/index.php?title=%EC%9D%B4%EC%95%BC&amp;action=edit&amp;redlink=1" title="이야 (없는 문서)">이야</a>(李壄) 등의 반대로 무산되었다.</p>
<p><a href="/wiki/1697%EB%85%84" title="1697년">1697년</a>(숙종 23년) <a href="/wiki/4%EC%9B%94" title="4월">4월</a>에 풀려나 복관되었다. 그 해 <a href="/wiki/%EB%82%A8%EC%9D%B8" title="남인">남인</a>을 등용하는 정책을 펴던 <a href="/wiki/%EC%86%8C%EB%A1%A0" title="소론">소론</a> 재상 <a href="/wiki/%EC%B5%9C%EC%84%9D%EC%A0%95" title="최석정">최석정</a>(崔錫鼎)에 의하여 청백함을 인정받아 서용을 건의하였으며, <a href="/wiki/1698%EB%85%84" title="1698년">1698년</a> 임금으로부터 서용하라는 명령이 내려졌지만 현직에 나가지는 못했다. 시문에 뛰어나고 글씨로 이름이 높았다.</p>
<p>그는 <a href="/wiki/%EC%86%A1%EC%8B%9C%EC%97%B4" title="송시열">송시열</a>과 <a href="/wiki/%EA%B9%80%EC%88%98%ED%95%AD" title="김수항">김수항</a> 등이 진리를 호도하고 학문의 자유를 가로막고 권력으로 자신들의 사론을 관철시킨다고 비판하였다. 그는 <a href="/wiki/%EB%B9%84%EB%B3%80%EC%82%AC" title="비변사">비변사</a>로부터 여러번 지방관 직의 물망에 올랐지만 번번히 낙마하고 만다.</p>
<h4><span class="mw-headline" id=".EB.A7.8C.EB.85.84">만년</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%9D%B4%EC%84%9C%EC%9A%B0_(1633%EB%85%84)&amp;action=edit&amp;section=13" title="부분 편집: 만년">편집</a><span class="mw-editsection-bracket">]</span></span></h4>
<p>이후 광주 청계산 근처로 낙향하여 만년에는 후학에게 학문과 시문을 가르치는 일로 소일했다. 연초재 <a class="new" href="/w/index.php?title=%EC%98%A4%EC%83%81%EB%A0%B4&amp;action=edit&amp;redlink=1" title="오상렴 (없는 문서)">오상렴</a>(燕超齋 吳尙濂), 희암 채팽윤(希菴 蔡彭胤), 약산 오광운(藥山 吳光運), 국포 <a class="new" href="/w/index.php?title=%EA%B0%95%EB%B0%95&amp;action=edit&amp;redlink=1" title="강박 (없는 문서)">강박</a>(菊圃 姜樸) 등이 그의 문하에서 시와 학문을 배웠다. <a href="/wiki/%EC%9D%B4%EC%9D%B5_(1681%EB%85%84)" title="이익 (1681년)">성호 이익</a> 역시 그의 문하에 출입하여 학문을 배웠고, 이는 <a href="/wiki/%EC%95%88%EC%A0%95%EB%B3%B5" title="안정복">안정복</a>, <a class="new" href="/w/index.php?title=%EA%B0%95%EB%B0%95&amp;action=edit&amp;redlink=1" title="강박 (없는 문서)">강박</a>, <a href="/wiki/%EC%8B%A0%ED%9B%84%EB%8B%B4" title="신후담">신후담</a>, <a href="/wiki/%EC%B1%84%EC%A0%9C%EA%B3%B5" title="채제공">채제공</a> 등에게로 이어진다. 그는 <a href="/wiki/%EC%9C%A4%EC%84%A0%EB%8F%84" title="윤선도">윤선도</a>, <a class="mw-disambig" href="/wiki/%EC%9D%B4%EB%AF%BC%EA%B5%AC" title="이민구">이민구</a> 등의 시문을 후대에 전하는 역할을 하였다.</p>
<p><a class="new" href="/w/index.php?title=%EC%98%A4%EC%83%81%EB%A0%B4&amp;action=edit&amp;redlink=1" title="오상렴 (없는 문서)">오상렴</a>(吳尙濂)은 <a href="/wiki/1704%EB%85%84" title="1704년">1704년</a>(숙종 30년) 스물 다섯 살 때 <a class="new" href="/w/index.php?title=%EA%B9%80%EB%A6%AC%EB%A7%8C&amp;action=edit&amp;redlink=1" title="김리만 (없는 문서)">김리만</a>(金履萬) 등과 함께 송곡 이서우에게 시를 배웠다.<sup class="reference" id="cite_ref-10"><a href="#cite_note-10">[10]</a></sup> 그러나 오상렴은 <a href="/wiki/1707%EB%85%84" title="1707년">1707년</a> 이른 나이에 요절하였다. 만년에는 <a href="/wiki/%EC%84%B1%ED%98%B8%EC%82%AC%EC%84%A4" title="성호사설">성호사설</a>을 지은 실학자 <a class="mw-redirect" href="/wiki/%EC%84%B1%ED%98%B8_%EC%9D%B4%EC%9D%B5" title="성호 이익">성호 이익</a>(星湖 李瀷)도 그의 문하에 찾아와 자주 출입했다. 그는 이서우를 찾아 뵌 이후, 공부하다 모르는 점이 있으면 그에게 자주 묻고 배웠는데, <a href="/wiki/%EC%9D%B4%EC%9D%B5" title="이익">이익</a>은 이서우를 '사백'(詞伯)이라 칭하면서, 아버지의 친구이자 자신의 스승인 이서우를 존경하고 가르침을 받았다.『星湖僿說』에는 이익이 이서우에게 가르침을 청하거나 자신이 공부한 것을 검증받는 등의 내용을 기술하였다.<sup class="reference" id="cite_ref-11"><a href="#cite_note-11">[11]</a></sup></p>
<h4><span class="mw-headline" id=".EC.B5.9C.ED.9B.84">최후</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%9D%B4%EC%84%9C%EC%9A%B0_(1633%EB%85%84)&amp;action=edit&amp;section=14" title="부분 편집: 최후">편집</a><span class="mw-editsection-bracket">]</span></span></h4>
<p>시(詩)에 뛰어나고 서도(書道)로도 알려줬다. 저서로 《송파문집》(松坡文集) 20권<sup class="reference" id="cite_ref-12"><a href="#cite_note-12">[12]</a></sup> 과 《강사》(康史) 2권이 있으며, 작품으로는 <a class="mw-redirect" href="/wiki/%EA%B9%80%ED%8F%AC" title="김포">김포</a> 문수사(文殊寺) 풍담대사비(楓潭大師碑)의 전액이 유필(遺筆), 예산군 불산면 본점리 <a href="/wiki/%EC%9D%98%EC%A0%95%EB%B6%80" title="의정부">의정부</a><a href="/wiki/%EC%98%81%EC%9D%98%EC%A0%95" title="영의정">영의정</a> <a href="/wiki/%EC%9D%B4%EC%9D%98%EB%B0%B0" title="이의배">이의배</a>(李義培) <a href="/wiki/%EC%8B%A0%EB%8F%84%EB%B9%84" title="신도비">신도비</a>의 수액, <a class="mw-disambig" href="/wiki/%EC%9D%B4%EC%83%81%EC%9D%98" title="이상의">이상의</a>의 소릉문집 서문 등이 남아 있다.</p>
<p><a href="/wiki/1709%EB%85%84" title="1709년">1709년</a>(숙종 35년) <a href="/wiki/10%EC%9B%94_14%EC%9D%BC" title="10월 14일">10월 14일</a>에 사거했다. 그의 나이 향년 76세였다.</p>
<h3><span class="mw-headline" id=".EC.82.AC.ED.9B.84">사후</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%9D%B4%EC%84%9C%EC%9A%B0_(1633%EB%85%84)&amp;action=edit&amp;section=15" title="부분 편집: 사후">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p><a href="/wiki/%EA%B2%BD%EA%B8%B0%EB%8F%84" title="경기도">경기도</a> <a href="/wiki/%EA%B4%91%EC%A3%BC%EA%B5%B0_(%EA%B2%BD%EA%B8%B0%EB%8F%84)" title="광주군 (경기도)">광주군</a> 언주면 원지리 철곡(현 <a href="/wiki/%EC%84%9C%EC%9A%B8%ED%8A%B9%EB%B3%84%EC%8B%9C" title="서울특별시">서울특별시</a> <a href="/wiki/%EC%84%9C%EC%B4%88%EA%B5%AC" title="서초구">서초구</a> 원지동 산 83-2번지 철곡) 유좌의 부인 청송심씨 묘소 옆에 안장되었다. <a href="/wiki/1701%EB%85%84" title="1701년">1701년</a> <a href="/wiki/%EB%82%A8%EC%9D%B8" title="남인">남인</a>의 몰락 이후 두 서자 <a href="/wiki/%EC%9D%B4%EC%9D%B5%EA%B4%80" title="이익관">이익관</a>, <a href="/wiki/%EC%9D%B4%EC%88%9C%EA%B4%80" title="이순관">이순관</a>과 서녀 사위 <a class="new" href="/w/index.php?title=%EB%AF%BC%EA%B4%80%ED%9A%A8&amp;action=edit&amp;redlink=1" title="민관효 (없는 문서)">민관효</a>가 <a href="/wiki/1728%EB%85%84" title="1728년">1728년</a>(영조 4년) <a href="/wiki/%EC%9D%B4%EC%9D%B8%EC%A2%8C%EC%9D%98_%EB%82%9C" title="이인좌의 난">이인좌의 난</a>에 연루되면서 그의 가계는 몰락하였다. 셋째 아들 <a class="new" href="/w/index.php?title=%EC%9D%B4%ED%99%8D%EA%B4%80&amp;action=edit&amp;redlink=1" title="이홍관 (없는 문서)">이홍관</a>은 처형당하고, 괘서를 돌리는데 가담한 <a href="/wiki/%EC%9D%B4%EC%9D%B5%EA%B4%80" title="이익관">이익관</a>, <a href="/wiki/%EC%9D%B4%EC%88%9C%EA%B4%80" title="이순관">이순관</a> 형제는 능지처참당하고 가산을 적몰당했으며, <a href="/wiki/%EC%9D%B4%EC%9D%B8%EC%A2%8C" title="이인좌">이인좌</a>를 도운 서녀 사위 <a class="new" href="/w/index.php?title=%EB%AF%BC%EA%B4%80%ED%9A%A8&amp;action=edit&amp;redlink=1" title="민관효 (없는 문서)">민관효</a> 일가 역시 처형당하고 가산은 적몰당한다. 이들의 후손들은 서손 <a class="new" href="/w/index.php?title=%EC%9D%B4%EA%B3%84%EA%B0%95&amp;action=edit&amp;redlink=1" title="이계강 (없는 문서)">이계강</a>, <a class="new" href="/w/index.php?title=%EC%9D%B4%EC%B2%A0%EA%B0%95&amp;action=edit&amp;redlink=1" title="이철강 (없는 문서)">이철강</a> 등을 비롯, 극변의 노비로 끌려가게 된다.</p>
<p>이서우 역시 역모로 연좌되어 추탈당했다. 그의 두 아들 <a class="new" href="/w/index.php?title=%EC%9D%B4%EC%A0%95%EA%B4%80&amp;action=edit&amp;redlink=1" title="이정관 (없는 문서)">이정관</a>과 <a class="new" href="/w/index.php?title=%EC%9D%B4%EC%84%9D%EA%B4%80&amp;action=edit&amp;redlink=1" title="이석관 (없는 문서)">이석관</a>은 이전에 사망했으므로 화를 모면하였으나 그의 적손들 역시 뿔뿔이 흩어져 <a href="/wiki/%EA%B2%BD%EC%83%81%EB%B6%81%EB%8F%84" title="경상북도">경상북도</a> <a class="mw-redirect" href="/wiki/%EC%84%B1%EC%A3%BC" title="성주">성주</a>와 <a href="/wiki/%EA%B0%95%EC%9B%90%EB%8F%84" title="강원도">강원도</a> <a class="mw-disambig" href="/wiki/%EC%9B%90%EC%A3%BC" title="원주">원주</a> 등으로 흩어졌다.</p>
<p><a href="/wiki/1755%EB%85%84" title="1755년">1755년</a>에는 나주 괘서 사건에 그의 손자 <a href="/wiki/%EC%9D%B4%EC%8B%AC" title="이심">이심</a>이 연좌되어 처형되면서 그의 가계는 몰락했고, 외손 <a class="new" href="/w/index.php?title=%EA%B0%95%EB%B0%95&amp;action=edit&amp;redlink=1" title="강박 (없는 문서)">강박</a>, 외증손 <a href="/wiki/%EC%8B%A0%ED%9B%84%EB%8B%B4" title="신후담">신후담</a> 등도 연좌되어 출사에 제한을 받게 된다. <a href="/wiki/1728%EB%85%84" title="1728년">1728년</a>(영조 4) <a href="/wiki/%EC%9D%B4%EC%9D%B8%EC%A2%8C%EC%9D%98_%EB%82%9C" title="이인좌의 난">이인좌의 난</a>으로 관작을 추탈당했다가 <a href="/wiki/%EC%A1%B0%EC%84%A0_%EC%A0%95%EC%A1%B0" title="조선 정조">정조</a> 때 <a href="/wiki/%EB%82%A8%EC%9D%B8" title="남인">남인</a>이 집권하면서 <a href="/wiki/1795%EB%85%84" title="1795년">1795년</a>(정조 19년) <a href="/wiki/10%EC%9B%94" title="10월">10월</a> 다시 복권되었다. 그의 외증손 중에는 <a class="mw-redirect" href="/wiki/%EC%B2%9C%EC%A3%BC%EA%B5%90" title="천주교">천주교</a>비판론자인 <a href="/wiki/%EC%8B%A0%ED%9B%84%EB%8B%B4" title="신후담">신후담</a>도 있고, <a href="/wiki/%EB%B0%9C%ED%95%B4%EA%B3%A0" title="발해고">발해고</a>의 저자 <a href="/wiki/%EC%9C%A0%EB%93%9D%EA%B3%B5" title="유득공">유득공</a>은 그의 서녀의 외손자였다.</p>
<p>그는 사후 <a href="/wiki/%EB%82%A8%EC%9D%B8" title="남인">남인</a> 학문과 시맥의 정통이자 남인 문단의 종장으로 추숭되고 훗날 <a href="/wiki/%EC%B1%84%EC%A0%9C%EA%B3%B5" title="채제공">채제공</a>에 의해 <a href="/wiki/%ED%97%88%EB%AA%A9" title="허목">허목</a>-이서우-<a href="/wiki/%EC%9D%B4%EC%9D%B5_(1681%EB%85%84)" title="이익 (1681년)">이익</a>으로 남인 학문, 시문의 법통으로 규정되었다. 그의 문집은 외손 국포 <a class="new" href="/w/index.php?title=%EA%B0%95%EB%B0%95&amp;action=edit&amp;redlink=1" title="강박 (없는 문서)">강박</a>이 정리했고, <a class="new" href="/w/index.php?title=%EA%B0%95%EB%B0%95&amp;action=edit&amp;redlink=1" title="강박 (없는 문서)">강박</a>의 손자 <a class="new" href="/w/index.php?title=%EA%B0%95%EC%84%B8%EB%A5%9C&amp;action=edit&amp;redlink=1" title="강세륜 (없는 문서)">강세륜</a>과 <a class="new" href="/w/index.php?title=%EA%B0%95%EC%84%B8%EB%B0%B1&amp;action=edit&amp;redlink=1" title="강세백 (없는 문서)">강세백</a>이 재정리하여 간행하였고, <a href="/wiki/1995%EB%85%84" title="1995년">1995년</a> 연세대학교에서 국역으로 간행되어 알려졌다. <a href="/wiki/%EA%B0%95%EC%9B%90%EB%8F%84" title="강원도">강원도</a> <a class="mw-redirect" href="/wiki/%EC%9B%90%EC%A3%BC%EA%B5%B0" title="원주군">원주군</a> <a href="/wiki/%EB%B6%80%EB%A1%A0%EB%A9%B4" title="부론면">부론면</a> 단강리에 그를 제향하는 <a class="new" href="/w/index.php?title=%EC%98%81%EC%A0%95%EA%B0%81&amp;action=edit&amp;redlink=1" title="영정각 (없는 문서)">영정각</a>이 세워졌다. 매년 <a href="/wiki/3%EC%9B%94_1%EC%9D%BC" title="3월 1일">3월 1일</a> 원주군 부론면 단강리 영정각에 다례를 올리고 있다. <a href="/wiki/1995%EB%85%84" title="1995년">1995년</a> <a href="/wiki/8%EC%9B%94" title="8월">8월</a> 후손들에 의해 묘비가 다시 수립되었다.</p>

<h2><span class="mw-headline" id=".ED.8F.89.EA.B0.80">평가</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%9D%B4%EC%84%9C%EC%9A%B0_(1633%EB%85%84)&amp;action=edit&amp;section=20" title="부분 편집: 평가">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>구경에 통달한 거유로서 도학군자로 칭송받기도 했다. 당대에 받는 청백리로 도처에 인정비(仁政碑)가 세워져 선정휼민한 치적을 예찬하는 칭송이 자자하였으며 또한 학문이 뛰어나 구경(九經)에 통달한 거유(巨儒)로서 조야에서는 도학군자로 숭앙받았다.</p>
<p><a href="/wiki/%EC%A0%95%EC%95%BD%EC%9A%A9" title="정약용">정약용</a>은 그의 작품성에 대해 '우리들(남인)의 시맥은 호주(湖洲) 채유후(蔡裕後)·동주(東州) 이민구(李敏求) 이후로 오직 송곡 이서우가 그 정통을 얻었으나, 성곡의 시논 정교하고 치밀하기만 할 뿐, 원대한 운치가 적다<sup class="reference" id="cite_ref-16"><a href="#cite_note-16">[16]</a></sup> 고 평하였다.</p>
<h2><span class="mw-headline" id=".EA.B0.80.EC.A1.B1_.EA.B4.80.EA.B3.84">가족 관계</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%9D%B4%EC%84%9C%EC%9A%B0_(1633%EB%85%84)&amp;action=edit&amp;section=21" title="부분 편집: 가족 관계">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>청송심씨 부인 외에도 첩이 1명 더 있었다. 동명 <a class="new" href="/w/index.php?title=%EA%B9%80%EC%84%B8%EB%A0%B4&amp;action=edit&amp;redlink=1" title="김세렴 (없는 문서)">김세렴</a>은 그의 셋째 형수의 친정아버지로, <a href="/wiki/%EA%B9%80%ED%9A%A8%EC%9B%90" title="김효원">김효원</a> 가문과도 인척간이었다. 한편 <a href="/wiki/%EC%9D%B8%EC%A1%B0%EB%B0%98%EC%A0%95" title="인조반정">인조반정</a>의 공신인 <a href="/wiki/%EC%84%9C%EC%9D%B8" title="서인">서인</a> 완풍부원군 <a href="/wiki/%EC%9D%B4%EC%84%9C_(1580%EB%85%84)" title="이서 (1580년)">이서</a>는 그의 외당숙이었다.</p>

<h2><span class="mw-headline" id=".EC.9D.BC.ED.99.94">일화</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%9D%B4%EC%84%9C%EC%9A%B0_(1633%EB%85%84)&amp;action=edit&amp;section=22" title="부분 편집: 일화">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<ul>
<li>그의 호는 송곡으로 문집은 송곡문집이라 하였으나, 숙종 이후 <a href="/wiki/%EB%82%A8%EC%9D%B8" title="남인">남인</a>이 몰락하면서 세인의 이목을 피하기 위해 송파문집으로 이름을 바꾸어 간행하였다. 송파문집은 그의 외손자인 <a class="new" href="/w/index.php?title=%EA%B5%AD%ED%8F%AC_%EA%B0%95%EB%B0%95&amp;action=edit&amp;redlink=1" title="국포 강박 (없는 문서)">국포 강박</a>이 자신의 백부 <a class="new" href="/w/index.php?title=%EA%B0%95%EC%84%9D%EB%B2%88&amp;action=edit&amp;redlink=1" title="강석번 (없는 문서)">강석번</a>의 양자로 출계하였는데, 강박이 이서우의 문집을 정리했으며, 뒤에 강박의 손자인 강세륜과 강세백<sup class="reference" id="cite_ref-20"><a href="#cite_note-20">[20]</a></sup> 형제가 정리하여 출간하였다.</li>
<li>1728년 그의 서자 이익관, 이순관이 이인좌의 난에 가담하였다가 모반 혐의로 처형되면서 그의 일족들도 연좌되었다.</li>
<li>후에 <a class="mw-redirect" href="/wiki/%EA%B9%80%EC%82%BF%EA%B0%93" title="김삿갓">김삿갓</a>은 자신의 시집에 실린 시 '나부(懶婦)'라는 제목의 7언 율시를 지었는데, 이는 송곡 이서우의 시 를 자구만 바꾼 것이었다.<sup class="reference" id="cite_ref-21"><a href="#cite_note-21">[21]</a></sup><sup class="reference" id="cite_ref-22"><a href="#cite_note-22">[22]</a></sup></li>
</ul>
<h2><span class="mw-headline" id=".EA.B8.B0.ED.83.80">기타</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%9D%B4%EC%84%9C%EC%9A%B0_(1633%EB%85%84)&amp;action=edit&amp;section=23" title="부분 편집: 기타">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>일찍이 민점(閔點)이 문형(文衡)을 추천할 때, 권유(權愈)와 이서우(李瑞雨)두 사람을 놓고 얼른 선택을 하지 못하였다. 그래서 명사들이 모인 자리에서 운을 내어 재주를 시험하자, 그가 즉석에서 시를 지어 좌중을 놀라게 하여 문형에 올랐다고 한다.<sup class="reference" id="cite_ref-23"><a href="#cite_note-23">[23]</a></sup></p>
<p>영남 안찰사 송곡 이서우는 귀임길에 태학사 권유(權愈)를 만나 말하기를, 〈영남 70주에 옛 도리대로 백성을 돌보는 이는 의령의 이사군(李使君) 숭일(이숭일, <a href="/wiki/%EC%9D%B4%ED%98%84%EC%9D%BC" title="이현일">이현일</a>의 동생) 뿐이라〉하였다.<sup class="reference" id="cite_ref-24"><a href="#cite_note-24">[24]</a></sup></p>
<p>그의 생전에 <a href="/wiki/%EC%95%88%EA%B2%BD" title="안경">안경</a>이 <a href="/wiki/%EC%A1%B0%EC%84%A0" title="조선">조선</a>에 처음 유입되었는데, 그는 안경의 생김새를 놀라워하며 이를 묘사한 시를 지어 남기기도 했다.<sup class="reference" id="cite_ref-25"><a href="#cite_note-25">[25]</a></sup></p>
<h3><span class="mw-headline" id=".EB.B0.94.EB.91.91">바둑</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%9D%B4%EC%84%9C%EC%9A%B0_(1633%EB%85%84)&amp;action=edit&amp;section=24" title="부분 편집: 바둑">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>송곡(松谷) 이서우(李瑞雨)는 젊어서 바둑을 둘 줄 몰랐다.<sup class="reference" id="cite_ref-baduk01_26-0"><a href="#cite_note-baduk01-26">[26]</a></sup> 그러나 바둑을 둘줄 몰라 여러 번 패한 것을 분하게 여기고 스스로 독학하여 바둑을 익혔다.</p>
<p>그는 처음에 바둑을 잘 두는 자를 찾아가 바둑을 두었는데 열여섯 집 차로 졌다. 세 판을 두었는데도 불구하고 한판도 이기지 못하자 분함을 이기지 못하고 돌아와서는 종이에 바둑판을 그려 들보에 매달아 놓고 누워서 쳐다보았다. 이틀만에 다시 바둑을 잘 두는 자를 찾아가 도리어 열여섯 집 차로 이겼다.<sup class="reference" id="cite_ref-baduk01_26-1"><a href="#cite_note-baduk01-26">[26]</a></sup> 그 뒤로 그는 바둑의 달인이 되었다 한다.</p>
