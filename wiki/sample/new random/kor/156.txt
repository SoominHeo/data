<p><b>볼고그라드</b>(<span style="font-size: smaller;"><a href="/wiki/%EB%9F%AC%EC%8B%9C%EC%95%84%EC%96%B4" title="러시아어">러시아어</a>:</span> <span lang="ru" xml:lang="ru">Волгогра́д</span> <span class="unicode audiolink"><a class="internal" href="//upload.wikimedia.org/wikipedia/commons/d/d9/Ru-Volgograd.ogg" title="Ru-Volgograd.ogg">듣기</a></span> <span class="metadata audiolinkinfo"><small>(<a href="/wiki/%EC%9C%84%ED%82%A4%EB%B0%B1%EA%B3%BC:%EB%AF%B8%EB%94%94%EC%96%B4_%EB%8F%84%EC%9B%80%EB%A7%90" title="위키백과:미디어 도움말">도움말</a>·<a href="/wiki/%ED%8C%8C%EC%9D%BC:Ru-Volgograd.ogg" title="파일:Ru-Volgograd.ogg">정보</a>)</small></span>)는 <a href="/wiki/%EB%9F%AC%EC%8B%9C%EC%95%84" title="러시아">러시아</a>의 <a href="/wiki/%EB%B3%BC%EA%B0%80_%EA%B0%95" title="볼가 강">볼가 강</a> 서안에 있는 도시로 <a href="/wiki/%EB%B3%BC%EA%B3%A0%EA%B7%B8%EB%9D%BC%EB%93%9C_%EC%A3%BC" title="볼고그라드 주">볼고그라드 주</a>의 주도이다. 인구는 대략 110만명이다. 예전에 <b>차리친</b>(<span style="font-size: smaller;"><a href="/wiki/%EB%9F%AC%EC%8B%9C%EC%95%84%EC%96%B4" title="러시아어">러시아어</a>:</span> <span lang="ru" xml:lang="ru">Цари́цын</span> <span class="unicode audiolink"><a class="internal" href="//upload.wikimedia.org/wikipedia/commons/f/f3/Ru-Tsaritsyn.ogg" title="Ru-Tsaritsyn.ogg">듣기</a></span> <span class="metadata audiolinkinfo"><small>(<a href="/wiki/%EC%9C%84%ED%82%A4%EB%B0%B1%EA%B3%BC:%EB%AF%B8%EB%94%94%EC%96%B4_%EB%8F%84%EC%9B%80%EB%A7%90" title="위키백과:미디어 도움말">도움말</a>·<a href="/wiki/%ED%8C%8C%EC%9D%BC:Ru-Tsaritsyn.ogg" title="파일:Ru-Tsaritsyn.ogg">정보</a>)</small></span>, <a href="/wiki/1598%EB%85%84" title="1598년">1598년</a> - <a href="/wiki/1925%EB%85%84" title="1925년">1925년</a>)이나 <b>스탈린그라드</b>(<span style="font-size: smaller;"><a href="/wiki/%EB%9F%AC%EC%8B%9C%EC%95%84%EC%96%B4" title="러시아어">러시아어</a>:</span> <span lang="ru" xml:lang="ru">Сталингра́д</span> <span class="unicode audiolink"><a class="internal" href="//upload.wikimedia.org/wikipedia/commons/a/ad/Ru-Stalingrad.ogg" title="Ru-Stalingrad.ogg">듣기</a></span> <span class="metadata audiolinkinfo"><small>(<a href="/wiki/%EC%9C%84%ED%82%A4%EB%B0%B1%EA%B3%BC:%EB%AF%B8%EB%94%94%EC%96%B4_%EB%8F%84%EC%9B%80%EB%A7%90" title="위키백과:미디어 도움말">도움말</a>·<a href="/wiki/%ED%8C%8C%EC%9D%BC:Ru-Stalingrad.ogg" title="파일:Ru-Stalingrad.ogg">정보</a>)</small></span>, <span style="font-size: smaller;"><a href="/wiki/%EB%AC%B8%ED%99%94%EC%96%B4" title="문화어">문화어</a>:</span> 쓰딸린그라드, <a href="/wiki/1925%EB%85%84" title="1925년">1925년</a> - <a href="/wiki/1961%EB%85%84" title="1961년">1961년</a>)로 불렸으며, <a href="/wiki/%EC%A0%9C2%EC%B0%A8_%EC%84%B8%EA%B3%84_%EB%8C%80%EC%A0%84" title="제2차 세계 대전">제2차 세계 대전</a> 중 <a href="/wiki/%EB%8F%85%EC%9D%BC" title="독일">독일</a>과 소련 간의 <a href="/wiki/%EC%8A%A4%ED%83%88%EB%A6%B0%EA%B7%B8%EB%9D%BC%EB%93%9C_%EC%A0%84%ED%88%AC" title="스탈린그라드 전투">격전지</a>로 유명하다. 볼가강의 하항으로, 볼가·돈 운하의 기점(起點)이며 교통의 요충지이다. 기계제조(트랙터·선박), 제강, 제유, 제재, 알루미늄 정련(精鍊) 등의 공업이 성하다. 상류에는 수력발전소가 있다. <sup class="reference" id="cite_ref-1"><a href="#cite_note-1">[1]</a></sup> <a href="/wiki/%EC%98%90%EB%A0%88%EB%82%98_%EC%9D%B4%EC%8B%A0%EB%B0%94%EC%98%88%EB%B0%94" title="옐레나 이신바예바">옐레나 이신바예바</a>의 고향이다.</p>
