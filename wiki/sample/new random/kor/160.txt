<p><b>이존욱</b>(李存勗, <a href="/wiki/885%EB%85%84" title="885년">885년</a> ~ <a href="/wiki/926%EB%85%84" title="926년">926년</a>, 재위: <a href="/wiki/909%EB%85%84" title="909년">909년</a> ~ <a href="/wiki/926%EB%85%84" title="926년">926년</a>)은 <a class="mw-redirect" href="/wiki/%EC%98%A4%EB%8C%80%EC%8B%AD%EA%B5%AD%EC%8B%9C%EB%8C%80" title="오대십국시대">오대십국시대</a> <a href="/wiki/%ED%9B%84%EB%8B%B9" title="후당">후당</a>의 창건자로서 초대 황제이다. 묘호는 <b>장종</b>(莊宗)이며 시호는 <b>광성신민효황제</b>(光聖神閔孝皇帝)이다.</p>
<h2><span class="mw-headline" id=".EC.83.9D.EC.95.A0">생애</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%9D%B4%EC%A1%B4%EC%9A%B1&amp;action=edit&amp;section=1" title="부분 편집: 생애">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p><a href="/wiki/%EC%A3%BC%EC%A0%84%EC%B6%A9" title="주전충">주전충</a>(朱全忠)의 숙적이었던 <a href="/wiki/%EC%9D%B4%EA%B7%B9%EC%9A%A9" title="이극용">이극용</a>(李克用)은 908년에 사망했고, 뒤를 계승한 이존욱은 <a href="/wiki/%ED%9B%84%EB%9F%89_(%EC%98%A4%EB%8C%80)" title="후량 (오대)">후량</a>에 맹렬한 공격을 시작했다. 후량측에서도 <a href="/wiki/%EC%A3%BC%EC%A0%84%EC%B6%A9" title="주전충">주전충</a>의 실정과 추락이 심해 차례로 영토를 빼앗겼다. 거기에다 주전충이 후계자를 선택하는 것에도 실패해 내분을 초래했다. 이를 지켜본 이존욱은 연왕을 칭하던 유인공을 공격하여 그 나라를 병합했다.</p>
<p>923년 자신감을 가진 진왕 이존욱이 황제를 칭하고 국호를 대당이라 하며 나라를 세웠다. 역사상 후당이었다. 윤 4월말 <a href="/wiki/%EC%9D%B4%EC%82%AC%EC%9B%90" title="이사원">이사원</a>(李嗣源)이 이끄는 5000명의 기병이 후량을 기습했다. <a href="/wiki/%ED%9B%84%EB%9F%89_(%EC%98%A4%EB%8C%80)" title="후량 (오대)">후량</a>은 왕언장을 총사령관으로 단의를 부사령관으로 한 10만의 정병을 이끌고 후당을 공격했다.</p>
<p>장종 이존욱 또한 직접 군을 이끌고 양유에서 격전을 벌여 후량군을 격파했다. <a href="/wiki/%EC%99%95%EC%96%B8%EC%9E%A5" title="왕언장">왕언장</a>(王彦章)은 중도현에서 참수되었다. 923년 11월 19일 후당군이 카이펑을 함락시키자 후량은 멸망하게 되었다.</p>
<p><a href="/wiki/%EC%9D%B4%EA%B7%B9%EC%9A%A9" title="이극용">이극용</a>의 이(李)씨 성은 공적에 의해 <a href="/wiki/%EB%8B%B9%EB%82%98%EB%9D%BC" title="당나라">당나라</a>로부터 국성(國姓)을 하사받은 것이었다. 이것을 이유로 장종은 스스로가 당나라의 후계자임을 자칭하고 후당을 건국하게 되었다. <a href="/wiki/%ED%9B%84%EB%9F%89_(%EC%98%A4%EB%8C%80)" title="후량 (오대)">후량</a>을 멸망시킨 후, 뒤이어 기왕(冀王)을 자칭한 이무정, 사천을 지배한 전촉을 멸망시켜 영토를 확대했다. 그러나 장종의 생각은 당나라의 위광을 되살리는 것으로 <a class="mw-redirect" href="/wiki/%EB%82%99%EC%96%91" title="낙양">낙양</a>(洛陽)으로 천도하고, 주전충이 폐지했던 군대에 환관을 감찰로써 파견하는 제도를 부활시켜 무장들의 불만을 샀다.</p>
<p>이 불만 때문에 926년 무장들이 이사원(후에 명종)을 옹립하였다. 이사원의 군이 낙양에 육박하자 금군(근위병)들에 의해 장종은 살해당했다.</p>
