<p><b>항저우 샤오산 국제공항</b>(<span style="font-size: smaller;"><a href="/wiki/%EC%A4%91%EA%B5%AD%EC%96%B4" title="중국어">중국어</a>:</span> <span lang="zh" xml:lang="zh">杭州萧山国际机场</span>, Hángzhōu Xiāoshān Guójì Jīchǎng, <a class="mw-redirect" href="/wiki/IATA" title="IATA">IATA</a>: <b>HGH</b>, <a class="mw-redirect" href="/wiki/ICAO" title="ICAO">ICAO</a>: <b>ZSHC</b>)은 <a href="/wiki/%EC%A4%91%ED%99%94%EC%9D%B8%EB%AF%BC%EA%B3%B5%ED%99%94%EA%B5%AD" title="중화인민공화국">중화인민공화국</a> <a href="/wiki/%EC%A0%80%EC%9E%A5_%EC%84%B1" title="저장 성">저장 성</a> <a class="mw-redirect" href="/wiki/%ED%95%AD%EC%A0%80%EC%9A%B0" title="항저우">항저우</a>에 위치한 공항이다. 이 공항은 <a class="mw-redirect" href="/wiki/%EC%B2%B8%ED%83%95%EA%B0%95" title="첸탕강">첸탕강</a> 남쪽 기슭에 지어졌으며 <a class="mw-redirect" href="/wiki/%ED%95%AD%EC%A0%80%EC%9A%B0" title="항저우">항저우</a> 시내로부터 약 27킬로미터 떨어져 있다.</p>

<p>이 공항의 건설은 세 단계로 계획되었었다. 첫 단계는 <a href="/wiki/1997%EB%85%84" title="1997년">1997년</a> <a href="/wiki/7%EC%9B%94" title="7월">7월</a>에 건설이 시작되어 <a href="/wiki/2000%EB%85%84" title="2000년">2000년</a> <a href="/wiki/12%EC%9B%94_30%EC%9D%BC" title="12월 30일">12월 30일</a>에 완공하고 개항하였다. 이로써 기존의 민간, 군사 겸용 공항이던 구 항저우 공항을 대체하게 되었다. <a href="/wiki/2003%EB%85%84" title="2003년">2003년</a> <a href="/wiki/9%EC%9B%94" title="9월">9월</a>에 중국 정부는 이 공항을 국제공항으로 지정하고 <a href="/wiki/2004%EB%85%84" title="2004년">2004년</a> <a href="/wiki/3%EC%9B%94" title="3월">3월</a>에 입출국 관리 설비를 갖춘 뒤, 공식적으로 국제공항으로서 기능을 하기 시작하였다. 이 공항은 <a class="new" href="/w/index.php?title=%EC%A4%91%EA%B5%AD%ED%95%AD%EA%B3%B5%EA%B3%B5%EC%82%AC&amp;action=edit&amp;redlink=1" title="중국항공공사 (없는 문서)">중국항공공사</a>(CNAC) <a href="/wiki/%EC%A0%80%EC%9E%A5_%EC%84%B1" title="저장 성">저장 성</a>의 허브 공항이었다. 중국항공공사가 <a href="/wiki/%EC%A4%91%EA%B5%AD%EA%B5%AD%EC%A0%9C%ED%95%AD%EA%B3%B5" title="중국국제항공">중국국제항공</a>에 합병된 뒤에도 <a href="/wiki/%EC%A4%91%EA%B5%AD%EA%B5%AD%EC%A0%9C%ED%95%AD%EA%B3%B5" title="중국국제항공">중국국제항공</a>의 항저우의 허브 공항으로 사용되고 있다.</p>
