<p><b>프레더릭 생어</b>(Frederick Sanger, <a class="extiw" href="https://en.wikipedia.org/wiki/Order_of_Merit" title="en:Order of Merit">OM</a>, <a class="extiw" href="https://en.wikipedia.org/wiki/Order_of_the_Companions_of_Honour" title="en:Order of the Companions of Honour">CH</a>, <a href="/wiki/%EB%8C%80%EC%98%81_%EC%A0%9C%EA%B5%AD_%ED%9B%88%EC%9E%A5" title="대영 제국 훈장">CBE</a>, <a class="mw-redirect" href="/wiki/%EC%99%95%EB%A6%BD_%ED%95%99%ED%9A%8C" title="왕립 학회">FRS</a>, <a href="/wiki/1918%EB%85%84" title="1918년">1918년</a> <a href="/wiki/8%EC%9B%94_13%EC%9D%BC" title="8월 13일">8월 13일</a> ~ <a href="/wiki/2013%EB%85%84" title="2013년">2013년</a> <a href="/wiki/11%EC%9B%94_19%EC%9D%BC" title="11월 19일">11월 19일</a>)는 <a href="/wiki/%EC%9C%A0%EC%A0%84%EC%B2%B4%ED%95%99" title="유전체학">유전체학</a>과 <a href="/wiki/%EB%8B%A8%EB%B0%B1%EC%B2%B4%ED%95%99" title="단백체학">단백체학</a>을 최초로 연구한 <a href="/wiki/%EC%98%81%EA%B5%AD" title="영국">영국</a>의 <a href="/wiki/%EC%83%9D%ED%99%94%ED%95%99" title="생화학">생화학자</a>이다. <a href="/wiki/%EC%BC%80%EC%9E%84%EB%B8%8C%EB%A6%AC%EC%A7%80_%EB%8C%80%ED%95%99%EA%B5%90" title="케임브리지 대학교">케임브리지 대학교</a> <a href="/wiki/MRC_%EC%84%BC%ED%84%B0" title="MRC 센터">MRC 센터</a> 소속으로, 단백질 서열을 최초로 해석해 <a href="/wiki/1958%EB%85%84" title="1958년">1958년</a> <a href="/wiki/%EB%85%B8%EB%B2%A8_%ED%99%94%ED%95%99%EC%83%81" title="노벨 화학상">노벨 화학상</a>을 받았다.</p>
<p>생어가 최초로 해석한 단백질이 51개의 아미노산으로 이루어진 <a href="/wiki/%EC%9D%B8%EC%8A%90%EB%A6%B0" title="인슐린">인슐린</a>이다. 그 이후 1960-1970년대에 DNA의 염기 해석법을 개발하여 <a href="/wiki/1980%EB%85%84" title="1980년">1980년</a> 두 번째 노벨 화학상을 받았다. 동시에, 최초의 <a href="/wiki/%EA%B2%8C%EB%86%88" title="게놈">유전체</a>인 한 바이러스와 소의 미토콘드리아의 전체 염기를 밝혀냄으로써 유전체학을 창시하였다. 미국의 <a class="new" href="/w/index.php?title=%EB%A1%9C%EC%A0%80_%EC%8A%A4%ED%83%80%EB%8D%98&amp;action=edit&amp;redlink=1" title="로저 스타던 (없는 문서)">로저 스타던</a>(<span lang="en" xml:lang="en">Roger Staden</span>)을 통해 세계 최초의 서열정보분석프로그램 <a class="new" href="/w/index.php?title=%EC%8A%A4%ED%83%80%EB%8D%98_%ED%8C%A8%ED%82%A4%EC%A7%80&amp;action=edit&amp;redlink=1" title="스타던 패키지 (없는 문서)">스타던 패키지</a>(<span lang="en" xml:lang="en">Staden Package</span>)를 만들도록 해, 실질적으로 <a href="/wiki/%EC%83%9D%EB%AC%BC%EC%A0%95%EB%B3%B4%ED%95%99" title="생물정보학">생물정보학</a>을 창시한 사람이기도 하다. 이때, 미국의 <a class="new" href="/w/index.php?title=%EB%A7%88%EA%B1%B0%EB%A6%BF_%EC%98%A4%ED%81%B4%EB%A6%AC_%EB%8D%B0%EC%9D%B4%ED%98%B8%ED%94%84&amp;action=edit&amp;redlink=1" title="마거릿 오클리 데이호프 (없는 문서)">마거릿 오클리 데이호프</a>(<span lang="en" xml:lang="en">Margaret Oakley Dayhoff</span>)가 이론적인 서열분석을 수행하고 데이터베이스를 만들었지만, 실제는 영국 케임브리지의 MRC 센터의 그룹이 이미 이러한 데이터베이스를 구축하고 서열분석 또한 병행하고 있었다.</p>

<h2><span class="mw-headline" id=".EB.8B.A8.EB.B0.B1.EC.B2.B4.ED.95.99_.EC.B0.BD.EC.8B.9C">단백체학 창시</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%ED%94%84%EB%A0%88%EB%8D%94%EB%A6%AD_%EC%83%9D%EC%96%B4&amp;action=edit&amp;section=1" title="부분 편집: 단백체학 창시">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>최초로 <a href="/wiki/%EA%B2%8C%EB%86%88" title="게놈">유전체</a>를 해석한 이후 생어가 궁금해했던 것은 자신이 해석한 모든 유전자들의 총합인 유전체가 실제로 세포 내에서 모두 단백질로 발현되는지였다. 그래서 생어는 옥스포드대 출신의 <a class="new" href="/w/index.php?title=%EC%A1%B4_E._%EC%9B%8C%EC%BB%A4&amp;action=edit&amp;redlink=1" title="존 E. 워커 (없는 문서)">존 E. 워커</a>를 채용하여, 세포를 깨고 단백질을 여러 가지 방법으로 동정하는 작업을 시작했다. 이것이 최초의 <a href="/wiki/%EB%8B%A8%EB%B0%B1%EC%B2%B4%ED%95%99" title="단백체학">단백체학</a>이다.</p>
<p>이후, 존 E. 워커는 세포내에서 가장 중요한 미토콘드리아 단백질들을 많이 밝혀내고 1997년에 <a class="new" href="/w/index.php?title=F1_ATPase&amp;action=edit&amp;redlink=1" title="F1 ATPase (없는 문서)">F1 ATPase</a> 구조 해석으로 노벨상을 받는다. 생어가 알고자 했던 모든 단백질들의 발현인 <a href="/wiki/%EB%8B%A8%EB%B0%B1%EC%B2%B4" title="단백체">단백체</a>는 아직도 정확히 밝혀지지 않았다.</p>
<h2><span class="mw-headline" id=".EC.83.9D.EC.96.B4_.EC.84.BC.ED.84.B0">생어 센터</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%ED%94%84%EB%A0%88%EB%8D%94%EB%A6%AD_%EC%83%9D%EC%96%B4&amp;action=edit&amp;section=2" title="부분 편집: 생어 센터">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>현재 그의 이름을 따서, MRC 센터에서 파생된 <a href="/wiki/%EC%9B%B0%EC%BB%B4_%ED%8A%B8%EB%9F%AC%EC%8A%A4%ED%8A%B8_%EC%83%9D%EC%96%B4_%EC%97%B0%EA%B5%AC%EC%86%8C" title="웰컴 트러스트 생어 연구소">생어 센터</a>가 케임브리지에 세워져 있다. 생어센터는 <a class="new" href="/w/index.php?title=%EC%9C%A0%EB%9F%BD%EC%83%9D%EC%A0%95%EB%B3%B4%ED%95%99%EC%97%B0%EA%B5%AC%EC%86%8C&amp;action=edit&amp;redlink=1" title="유럽생정보학연구소 (없는 문서)">유럽생정보학연구소</a>(EBI)와 같은 캠퍼스에 위치한다.</p>
<h2><span class="mw-headline" id=".EC.84.9C.ED.9B.88">서훈</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%ED%94%84%EB%A0%88%EB%8D%94%EB%A6%AD_%EC%83%9D%EC%96%B4&amp;action=edit&amp;section=3" title="부분 편집: 서훈">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
