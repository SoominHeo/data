<p><b>야기에우워 왕조</b> <span style="font-size: smaller;"><a href="/wiki/%ED%8F%B4%EB%9E%80%EB%93%9C%EC%96%B4" title="폴란드어">폴란드어</a>:</span> <span lang="pl" xml:lang="pl">Jagiellonowie</span>, (<span style="font-size: smaller;"><a href="/wiki/%EB%A6%AC%ED%88%AC%EC%95%84%EB%8B%88%EC%95%84%EC%96%B4" title="리투아니아어">리투아니아어</a>:</span> <span lang="lt" xml:lang="lt">Jogailaičiai</span> 요가일라 왕조, <a href="/wiki/1386%EB%85%84" title="1386년">1386년</a>~<a href="/wiki/1572%EB%85%84" title="1572년">1572년</a>)는 <a href="/wiki/%EB%A6%AC%ED%88%AC%EC%95%84%EB%8B%88%EC%95%84" title="리투아니아">리투아니아</a> 대후(大侯) <a href="/wiki/%EB%B8%8C%EC%99%80%EB%94%94%EC%8A%A4%EC%99%80%ED%94%84_2%EC%84%B8" title="브와디스와프 2세">야기에우워</a>가 세운 <a href="/wiki/%ED%8F%B4%EB%9E%80%EB%93%9C" title="폴란드">폴란드</a>의 <a href="/wiki/%EC%99%95%EC%A1%B0" title="왕조">왕조</a>이다. <a href="/wiki/1386%EB%85%84" title="1386년">1386년</a> 리투아니아 대후 야기에우워는 <a href="/wiki/%ED%8F%B4%EB%9E%80%EB%93%9C" title="폴란드">폴란드</a>의 여왕 <a href="/wiki/%EC%95%BC%EB%93%9C%EB%B9%84%EA%B0%80" title="야드비가">야드비가</a>와 결혼하여 폴란드 왕 <a href="/wiki/%EB%B8%8C%EC%99%80%EB%94%94%EC%8A%A4%EC%99%80%ED%94%84_2%EC%84%B8" title="브와디스와프 2세">브와디스와프 2세</a>라 칭하고 양국은 연합했다. 이 연합은 양국의 독립성을 인정한 다음 공통의 군주를 섬기는 왕조적(王朝的) 연합에 불과하며, 국내에서 소(小)귀족이 독립하여 국가의 통일성은 결여되어 있었으나 <a class="mw-redirect" href="/wiki/%EB%8F%85%EC%9D%BC_%EA%B8%B0%EC%82%AC%EB%8B%A8" title="독일 기사단">독일 기사단</a>을 격파하고 <a href="/wiki/%EB%B0%9C%ED%8A%B8_%ED%95%B4" title="발트 해">발트 해</a>의 출구를 확보하여 <a href="/wiki/15%EC%84%B8%EA%B8%B0" title="15세기">15세기</a>의 <a href="/wiki/%EC%B9%B4%EC%A7%80%EB%AF%B8%EC%97%90%EC%8B%9C_4%EC%84%B8" title="카지미에시 4세">카지미에시 4세</a> 때에는 최성기를 이룩했으며, 보헤미아와 헝가리 왕위까지 장악해, 야기에우워 왕가는 폴란드-리투아니아계와 헝가리-보헤미아계로 분열된다. 그러나 헝가리-보헤미아 왕가는 <a href="/wiki/%EB%AA%A8%ED%95%98%EC%B9%98_%EC%A0%84%ED%88%AC" title="모하치 전투">모하치 전투</a>에서 <a class="new" href="/w/index.php?title=%EB%9F%AC%EC%9A%94%EC%8B%9C_2%EC%84%B8&amp;action=edit&amp;redlink=1" title="러요시 2세 (없는 문서)">러요시 2세</a>(루드비히 2세)가 전사하면서 단절 되고 <a class="mw-redirect" href="/wiki/%ED%95%A9%EC%8A%A4%EB%B6%80%EB%A5%B4%ED%81%AC_%EC%99%95%EA%B0%80" title="합스부르크 왕가">합스부르크 왕가</a>로 넘어가게 된다.</p>
<p>폴란드-리투아니아계열 최후의 왕 <a href="/wiki/%EC%A7%80%EA%B7%B8%EB%AC%B8%ED%8A%B8_2%EC%84%B8" title="지그문트 2세">지그문트 2세</a> 때 폴란드-리투아니아 양국은 단일(單一)이며 분할되지 않는 국가로서의 방향을 결정했으나 그의 죽음과 함께 야기에우워 왕조는 단절되고, 이후 폴란드 왕은 선거에 의해서 선출되게 되어 분열을 더해갔으며, 보헤미아-헝가리계 역시 합스부르크 왕가에 흡수되었다.</p>
