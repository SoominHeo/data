<p><b>즈냐나</b>(<span style="font-size: smaller;"><a href="/wiki/%EC%82%B0%EC%8A%A4%ED%81%AC%EB%A6%AC%ED%8A%B8%EC%96%B4" title="산스크리트어">산스크리트어</a>:</span> <span lang="sa" xml:lang="sa">ज्ञान</span> <small>Jñāna, Gñāna</small>, <span style="font-size: smaller;"><a href="/wiki/%ED%8C%94%EB%A6%AC%EC%96%B4" title="팔리어">팔리어</a>:</span> <span lang="pi" xml:lang="pi">ñāna</span>, <span style="font-size: smaller;"><a href="/wiki/%EC%98%81%EC%96%B4" title="영어">영어</a>:</span> <span lang="en" xml:lang="en">Jnana</span>)는 "<a href="/wiki/%EC%A7%80%EC%8B%9D" title="지식">지식</a>(Knowledge)" 또는 "<a href="/wiki/%EC%95%8E" title="앎">앎</a>(Knowing)"을 뜻하는 <a href="/wiki/%EC%82%B0%EC%8A%A4%ED%81%AC%EB%A6%AC%ED%8A%B8%EC%96%B4" title="산스크리트어">산스크리트어</a> 및 <a href="/wiki/%ED%8C%94%EB%A6%AC%EC%96%B4" title="팔리어">팔리어</a> 낱말이다. 즈냐나의 구체적인 의미는 이 낱말이 사용된 문맥에 따라 미묘한 차이가 있다. 전체적으로, 즈냐나의 개념은 지식 또는 앎 중에서도 직접적인 <a href="/wiki/%EA%B2%BD%ED%97%98" title="경험">경험</a> 또는 <a href="/wiki/%EA%B2%BD%ED%97%98" title="경험">체험</a>을 통해 알게 된 <a href="/wiki/%EC%A7%80%EC%8B%9D" title="지식">지식</a> 또는 <a href="/wiki/%EC%95%8E" title="앎">앎</a>을 중심으로 한다.<sup class="reference" id="cite_ref-brittanica_on_jnana_1-0"><a href="#cite_note-brittanica_on_jnana-1">[1]</a></sup></p>
<p><a href="/wiki/%ED%9E%8C%EB%91%90%EA%B5%90" title="힌두교">힌두교</a>와 <a href="/wiki/%EB%B6%88%EA%B5%90" title="불교">불교</a>의 전통에 따르면, 즈냐나는 <a href="/wiki/%EC%8B%A4%EC%9E%AC" title="실재">실재</a>(<a class="extiw" href="https://en.wikipedia.org/wiki/Reality" title="en:Reality"><span style="color: black">Reality</span></a>)에 대한 <a href="/wiki/%EA%B2%BD%ED%97%98" title="경험">체험</a> 또는 <a href="/wiki/%EA%B2%BD%ED%97%98" title="경험">경험</a>과 서로 불가분의 관계에 있는 지식이다.<sup class="reference" id="cite_ref-brittanica_on_jnana_1-1"><a href="#cite_note-brittanica_on_jnana-1">[1]</a></sup> 그 중에서도 특히 <a class="new" href="/w/index.php?title=%EC%A0%84%EC%B2%B4_%EC%A1%B4%EC%9E%AC&amp;action=edit&amp;redlink=1" title="전체 존재 (없는 문서)">전체 존재</a>(<a class="extiw" href="https://en.wikipedia.org/wiki/The_All" title="en:The All"><span style="color: black">All</span></a>), 즉 우주의 전면적 실재(<a class="extiw" href="https://en.wikipedia.org/wiki/Reality" title="en:Reality"><span style="color: black">Total Reality</span></a>)에 대한 경험("<a class="mw-redirect" href="/wiki/%EC%82%AC%EB%A7%88%EB%94%94" title="사마디">사마디</a>") 속에서 알려지는 지식을 의미한다.<sup class="reference" id="cite_ref-brittanica_on_jnana_1-2"><a href="#cite_note-brittanica_on_jnana-1">[1]</a></sup> 예를 들어, <a href="/wiki/%ED%9E%8C%EB%91%90%EA%B5%90" title="힌두교">힌두교</a>에서는 이 전체 존재 또는 우주의 전면적 실재로서의 <a class="mw-redirect" href="/wiki/%EC%A7%80%EA%B3%A0%ED%95%9C_%EC%A1%B4%EC%9E%AC" title="지고한 존재">지고한 존재</a>를 <a class="mw-redirect" href="/wiki/%ED%9E%8C%EB%91%90%EA%B5%90%EC%9D%98_%EC%A2%85%ED%8C%8C" title="힌두교의 종파">종파</a>에 따라 <a href="/wiki/%EB%B8%8C%EB%9D%BC%EB%A7%8C_(%EC%B2%A0%ED%95%99)" title="브라만 (철학)">브라만</a>(<a class="extiw" href="https://en.wikipedia.org/wiki/Brahman" title="en:Brahman"><span style="color: black">Brahman</span></a>)이라 칭하기도 하고 또는 <a class="new" href="/w/index.php?title=%EC%8B%9C%EB%B0%94-%EC%83%A5%ED%8B%B0&amp;action=edit&amp;redlink=1" title="시바-샥티 (없는 문서)">시바-샥티</a>(<a class="extiw" href="https://en.wikipedia.org/wiki/Kashmir_Shaivism" title="en:Kashmir Shaivism"><span style="color: black">Siva-Sakti</span></a>)라고 칭하기도 하는데, 즈냐나는 이 <a class="mw-redirect" href="/wiki/%EC%A7%80%EA%B3%A0%ED%95%9C_%EC%A1%B4%EC%9E%AC" title="지고한 존재">지고한 존재</a>를 경험("<a class="mw-redirect" href="/wiki/%EC%82%AC%EB%A7%88%EB%94%94" title="사마디">사마디</a>")을 통해 아는 것 또는 그 경험("<a class="mw-redirect" href="/wiki/%EC%82%AC%EB%A7%88%EB%94%94" title="사마디">사마디</a>")으로부터 알려진 <a href="/wiki/%EC%A7%80%EC%8B%9D" title="지식">지식</a>을 뜻한다.<sup class="reference" id="cite_ref-2"><a href="#cite_note-2">[2]</a></sup></p>

<h2><span class="mw-headline" id=".ED.9E.8C.EB.91.90.EA.B5.90">힌두교</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%A6%88%EB%83%90%EB%82%98&amp;action=edit&amp;section=1" title="부분 편집: 힌두교">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p><a href="/wiki/%ED%9E%8C%EB%91%90%EA%B5%90" title="힌두교">힌두교</a>에서 즈냐나는 참된 <a href="/wiki/%EC%A7%80%EC%8B%9D" title="지식">지식</a> 또는 참된 <a href="/wiki/%EC%95%8E" title="앎">앎</a>을 의미하는데, <a href="/wiki/%ED%9E%8C%EB%91%90%EA%B5%90" title="힌두교">힌두교</a>에서 말하는 참된 <a href="/wiki/%EC%A7%80%EC%8B%9D" title="지식">지식</a>은 <a href="/wiki/%EC%9E%90%EC%95%84" title="자아">자아</a>("<a href="/wiki/%EC%95%84%ED%8A%B8%EB%A7%8C" title="아트만">아트만</a>")와 <a href="/wiki/%EC%8B%A0" title="신">궁극적 실재</a>("<a class="mw-disambig" href="/wiki/%EB%B8%8C%EB%9D%BC%EB%A7%8C" title="브라만">브라만</a>")가 하나라는 <a href="/wiki/%EC%A7%80%EC%8B%9D" title="지식">지식</a>, 즉 이 둘이 동일한 존재라는 <a href="/wiki/%EC%95%8E" title="앎">앎</a>이며 이 앎은 오직 <a class="mw-redirect" href="/wiki/%EC%82%AC%EB%A7%88%EB%94%94" title="사마디">사마디</a>(<a class="extiw" href="https://en.wikipedia.org/wiki/Samadhi" title="en:Samadhi"><span style="color: black">Samadhi</span></a>)를 통해서만 획득된다. 이런 이유로, <a href="/wiki/%EC%9E%90%EC%95%84" title="자아">자아</a> 즉 <a href="/wiki/%EC%95%84%ED%8A%B8%EB%A7%8C" title="아트만">아트만</a>에 대한 진정한 <a href="/wiki/%EC%A7%80%EC%8B%9D" title="지식">지식</a>이라는 의미에서 즈냐나는 <b>아트마 즈냐나</b>(Atma Jnana)라고도 한다.</p>
<p><b>아트마 즈냐나</b>(Atma Jnana)는 영어권에서 흔히 <b><a class="extiw" href="https://en.wikipedia.org/wiki/Self-Realization" title="en:Self-Realization"><span style="color: black">Self-Realization</span></a></b>이라고 번역되는데, "<a class="extiw" href="https://en.wikipedia.org/wiki/Self-Realization" title="en:Self-Realization">Self-Realization</a>"은 다시 <a href="/wiki/%ED%95%9C%EA%B5%AD%EC%96%B4" title="한국어">한국어</a>로 <b><a class="new" href="/w/index.php?title=%EC%9E%90%EC%95%84_%EC%8B%A4%ED%98%84&amp;action=edit&amp;redlink=1" title="자아 실현 (없는 문서)"><span style="color: black">자아 실현</span></a></b>(自我實現)이라고 번역되기도 한다. 즈냐나와 아트마 즈냐나의 기본 개념에 따를 때 "<a class="extiw" href="https://en.wikipedia.org/wiki/Self-Realization" title="en:Self-Realization">Self-Realization</a>"의 직접적인 뜻은 "<a href="/wiki/%EC%9E%90%EC%95%84" title="자아">자아</a>에 대한 완전한 <a href="/wiki/%EC%A7%80%EC%8B%9D" title="지식">지식</a> 또는 참된 <a href="/wiki/%EC%A7%80%EC%8B%9D" title="지식">지식</a>"이다. 즈냐나와 아트마 즈냐나의 기본 개념에 충실한다는 입장에서는 "<a class="extiw" href="https://en.wikipedia.org/wiki/Self-Realization" title="en:Self-Realization">Self-Realization</a>"을 "<a class="new" href="/w/index.php?title=%EC%9E%90%EC%95%84_%EC%8B%A4%ED%98%84&amp;action=edit&amp;redlink=1" title="자아 실현 (없는 문서)">자아 실현</a>"이라고 번역하는 것은 잘못된 번역이라고 할 수도 있다. 그러나 다른 한편으로는 "<a class="extiw" href="https://en.wikipedia.org/wiki/Self-Realization" title="en:Self-Realization">Self-Realization</a>"을 "<a class="new" href="/w/index.php?title=%EC%9E%90%EC%95%84_%EC%8B%A4%ED%98%84&amp;action=edit&amp;redlink=1" title="자아 실현 (없는 문서)">자아 실현</a>"이라고 번역하는 것은 직접적인 의미에서 한 단계 의미를 더 진전시킨 것이라고 볼 수 있다. 즉, "<a class="new" href="/w/index.php?title=%EC%9E%90%EC%95%84_%EC%8B%A4%ED%98%84&amp;action=edit&amp;redlink=1" title="자아 실현 (없는 문서)">자아 실현</a>"이라는 번역은, <a href="/wiki/%EC%9E%90%EC%95%84" title="자아">자아</a>에 대한 완전한 <a href="/wiki/%EC%A7%80%EC%8B%9D" title="지식">지식</a> 또는 참된 <a href="/wiki/%EC%A7%80%EC%8B%9D" title="지식">지식</a>을 가지는 것이란 곧 그 이전까지는 <a href="/wiki/%EC%9E%A0%EC%9E%AC%EC%9D%98%EC%8B%9D" title="잠재의식">잠재</a>적 상태로서만 존재하고 있던 "<a href="/wiki/%EC%95%84%ED%8A%B8%EB%A7%8C" title="아트만">본래의 자신</a>"이 각성되어 완전히 <a class="new" href="/w/index.php?title=%ED%98%84%EC%9E%AC%EC%9D%98%EC%8B%9D&amp;action=edit&amp;redlink=1" title="현재의식 (없는 문서)">현실화</a>(現實化)되는 것과 같은 것이라는 의미를 담고 있다. <a href="/wiki/%ED%9E%8C%EB%91%90%EA%B5%90" title="힌두교">힌두교</a>의 문맥에서 "<a href="/wiki/%EC%95%84%ED%8A%B8%EB%A7%8C" title="아트만">본래의 자신</a>"은 "<a href="/wiki/%EC%8B%A0" title="신">신</a>(神: 궁극적 <a href="/wiki/%EC%8B%A4%EC%9E%AC" title="실재">실재</a>)"과 동의어이다. 이러한 의미에서의 "<a class="new" href="/w/index.php?title=%EC%9E%90%EC%95%84_%EC%8B%A4%ED%98%84&amp;action=edit&amp;redlink=1" title="자아 실현 (없는 문서)">자아 실현</a>"은 <a href="/wiki/%EB%8C%80%EC%8A%B9%EB%B6%88%EA%B5%90" title="대승불교">대승불교</a>의 <a href="/wiki/%EB%B6%88%EC%84%B1" title="불성">불성</a>(佛性)에 대한 교의 또는 <a href="/wiki/%EC%97%AC%EB%9E%98%EC%9E%A5_%EC%82%AC%EC%83%81" title="여래장 사상">여래장 사상</a>(如來藏思想)과 그 근본 개념이 지극히 유사하다.</p>
<p><a href="/wiki/%ED%9E%8C%EB%91%90%EA%B5%90" title="힌두교">힌두교</a>에서, <a href="/wiki/%EC%A6%88%EB%83%90%EB%82%98_%EC%9A%94%EA%B0%80" title="즈냐나 요가">즈냐나 요가</a>(<a class="extiw" href="https://en.wikipedia.org/wiki/Jnana_yoga" title="en:Jnana yoga"><span style="color: black">Jnana yoga</span></a>)는 <a class="mw-redirect" href="/wiki/%EB%AA%A8%ED%81%AC%EC%83%A4" title="모크샤">모크샤</a>(<a class="extiw" href="https://en.wikipedia.org/wiki/Moksha" title="en:Moksha"><span style="color: black">Moksha</span></a> <b>·</b> <a href="/wiki/%EC%9E%90%EC%9C%A0" title="자유">자유</a> <b>·</b> <a href="/wiki/%ED%95%B4%ED%83%88" title="해탈">해탈</a>)에 이르는 여러 <a class="new" href="/w/index.php?title=%EB%A7%88%EB%A5%B4%EA%B0%80&amp;action=edit&amp;redlink=1" title="마르가 (없는 문서)">길</a>(<a class="new" href="/w/index.php?title=%EB%A7%88%EB%A5%B4%EA%B0%80&amp;action=edit&amp;redlink=1" title="마르가 (없는 문서)">마르가</a> <b>·</b> <a class="extiw" href="https://en.wikipedia.org/wiki/Marga" title="en:Marga"><span style="color: black">Marga</span></a>)들 중의 하나이다. <a class="mw-redirect" href="/wiki/%EB%AA%A8%ED%81%AC%EC%83%A4" title="모크샤">모크샤</a>에 이르는 다른 길로는 <a href="/wiki/%EB%B0%95%ED%8B%B0_%EC%9A%94%EA%B0%80" title="박티 요가">박티 요가</a>(<a class="extiw" href="https://en.wikipedia.org/wiki/Bhakti_Yoga" title="en:Bhakti Yoga"><span style="color: black">Bhakti Yoga</span></a>) <b>·</b> <a href="/wiki/%EC%B9%B4%EB%A5%B4%EB%A7%88_%EC%9A%94%EA%B0%80" title="카르마 요가">카르마 요가</a>(<a class="extiw" href="https://en.wikipedia.org/wiki/Karma_Yoga" title="en:Karma Yoga"><span style="color: black">Karma Yoga</span></a>) 등이 있다.</p>

<p><a href="/wiki/%EB%B6%88%EA%B5%90" title="불교">불교</a>에서 즈냐나는 <a class="mw-disambig" href="/wiki/%EC%A7%80" title="지">지</a>(智) 또는 <a href="/wiki/%EC%A7%80%ED%98%9C" title="지혜">지혜</a>(智慧)로 <a class="mw-redirect" href="/wiki/%ED%95%9C%EC%97%AD" title="한역">한역</a>(漢譯)되는데, 이 <a href="/wiki/%EC%A7%80%ED%98%9C" title="지혜">지혜</a> 또는 <a href="/wiki/%EC%A7%80%EC%8B%9D" title="지식">지식</a>은 개념 또는 관념의 장애들인 <a class="new" href="/w/index.php?title=%EC%82%AC%EB%9F%89%EB%B6%84%EB%B3%84&amp;action=edit&amp;redlink=1" title="사량분별 (없는 문서)">사량분별</a>(思量分別)이 없는 순수한 자각(pure awareness), 즉 <a class="mw-redirect" href="/wiki/%EB%AC%B4%EB%B6%84%EB%B3%84%EC%A7%80" title="무분별지">무분별지</a>(無分別智 <b>·</b> 무분별 지혜 <b>·</b> undivided knowing)를 가리킨다.</p>
<p>즈냐나는 <a class="mw-disambig" href="/wiki/%EC%8B%9D" title="식">식</a>(識)으로 <a class="mw-redirect" href="/wiki/%ED%95%9C%EC%97%AD" title="한역">한역</a>(漢譯)되는 <a href="/wiki/%EC%82%B0%EC%8A%A4%ED%81%AC%EB%A6%AC%ED%8A%B8%EC%96%B4" title="산스크리트어">산스크리트어</a> 낱말인 <b>비즈냐나</b>(<a class="extiw" href="https://en.wikipedia.org/wiki/vijnana" title="en:vijnana"><span style="color: black">vijnana</span></a>)와는 그 의미에 있어 대(對)를 이룬다. <a class="mw-redirect" href="/wiki/%EB%B9%84%EC%A6%88%EB%83%90%EB%82%98" title="비즈냐나">비즈냐나</a> 또는 <a class="mw-disambig" href="/wiki/%EC%8B%9D" title="식">식</a>(識)은 <a href="/wiki/%EC%A3%BC%EA%B4%80" title="주관">주관</a>과 <a class="mw-disambig" href="/wiki/%EA%B0%9D%EA%B4%80" title="객관">객관</a>이 나누인 앎, 즉 <a class="new" href="/w/index.php?title=%EB%B6%84%EB%B3%84%EC%A7%80&amp;action=edit&amp;redlink=1" title="분별지 (없는 문서)">분별지</a>(分別智 <b>·</b> divided knowing)를 뜻한다. 예를 들어, <a href="/wiki/%EB%B6%88%EA%B5%90" title="불교">불교</a>의 <a class="mw-redirect" href="/wiki/%EC%9C%A0%EC%8B%9D%ED%95%99" title="유식학">유식학</a>(唯識學)의 중심 명제인 "<a class="mw-redirect" href="/wiki/%EC%A0%84%EC%8B%9D%EB%93%9D%EC%A7%80" title="전식득지">전식득지</a>(轉識得智)"에서, <a class="mw-disambig" href="/wiki/%EC%8B%9D" title="식">식</a>(識)은 비즈냐나를, <a class="mw-disambig" href="/wiki/%EC%A7%80" title="지">지</a>(智)는 즈냐나를 가리킨다.</p>
<p><a href="/wiki/%ED%8B%B0%EB%B2%A0%ED%8A%B8_%EB%B6%88%EA%B5%90" title="티베트 불교">티베트 불교</a>를 포함한 <a href="/wiki/%EB%8C%80%EC%8A%B9%EB%B6%88%EA%B5%90" title="대승불교">대승불교</a>의 <a class="mw-redirect" href="/wiki/%EC%8B%AD%EC%A7%80%EB%A1%A0" title="십지론">십지론</a>(十地論)에 따르면, 수행자는 10단계의 지혜의 길, 즉 10단계의 즈냐나의 길인 <a href="/wiki/%EC%8B%AD%EC%A7%80" title="십지">십지</a>(十地 <b>·</b> ten bhumis)를 통과해 가면서 <a class="mw-redirect" href="/wiki/%EC%99%84%EC%A0%84%ED%95%9C_%EA%B9%A8%EB%8B%AC%EC%9D%8C" title="완전한 깨달음">완전한 깨달음</a>(究竟覺 <b>·</b> 구경각) 또는 <a href="/wiki/%EC%97%B4%EB%B0%98" title="열반">열반</a>에 이르게 된다.<sup class="reference" id="cite_ref-3"><a href="#cite_note-3">[3]</a></sup></p>
<h2><span class="mw-headline" id=".EC.8B.A0.EC.A7.80.ED.95.99">신지학</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%A6%88%EB%83%90%EB%82%98&amp;action=edit&amp;section=3" title="부분 편집: 신지학">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p><a href="/wiki/%EC%8B%A0%EC%A7%80%ED%95%99" title="신지학">신지학</a>의 창시자인 <a class="mw-redirect" href="/wiki/%EB%B8%94%EB%9D%BC%EB%B0%94%EC%B8%A0%ED%82%A4_%EC%97%AC%EC%82%AC" title="블라바츠키 여사">블라바츠키 여사</a>에 따르면, <b>즈냐나 샥티</b>(Jnana <a class="extiw" href="https://en.wikipedia.org/wiki/Shakti" title="en:Shakti"><span style="color: black">Shakti</span></a>)는 지성의 힘, 진정한 지혜 또는 지식이다.<sup class="reference" id="cite_ref-SecretDoctrine_4-0"><a href="#cite_note-SecretDoctrine-4">[4]</a></sup> 그리고 <b>즈냐나록</b>(Jñānālok)은 (진정한) <a href="/wiki/%EC%A7%80%EC%8B%9D" title="지식">지식</a>과 <a href="/wiki/%EC%A7%84%EB%A6%AC" title="진리">진리</a>에 도달함에 의해 획득되는 <a class="mw-redirect" href="/wiki/%EA%B9%A8%EB%8B%AC%EC%9D%8C" title="깨달음">깨달음</a>(Enlightenment)이다.<sup class="reference" id="cite_ref-SecretDoctrine_4-1"><a href="#cite_note-SecretDoctrine-4">[4]</a></sup></p>
