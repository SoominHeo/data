<p><b>M32</b>(NGC 221)는 <a href="/wiki/%EC%95%88%EB%93%9C%EB%A1%9C%EB%A9%94%EB%8B%A4%EC%9E%90%EB%A6%AC" title="안드로메다자리">안드로메다자리</a>에 있는 <a class="mw-redirect" href="/wiki/%EC%99%9C%EC%86%8C_%ED%83%80%EC%9B%90_%EC%9D%80%ED%95%98" title="왜소 타원 은하">왜소 타원 은하</a>이며 <a href="/wiki/%EC%95%88%EB%93%9C%EB%A1%9C%EB%A9%94%EB%8B%A4_%EC%9D%80%ED%95%98" title="안드로메다 은하">안드로메다 은하</a>(M31)의 위성은하이다 . <a href="/wiki/1749%EB%85%84" title="1749년">1749년</a> <a class="new" href="/w/index.php?title=Le_Gentil&amp;action=edit&amp;redlink=1" title="Le Gentil (없는 문서)">Le Gentil</a>에 의해 발견되었다.</p>
<p>M32는 <a href="/wiki/%EC%A7%80%EA%B5%AC" title="지구">지구</a>에서 258±8만광년 떨어져 있다. <a class="mw-redirect" href="/wiki/%EA%B2%89%EB%B3%B4%EA%B8%B0_%EB%93%B1%EA%B8%89" title="겉보기 등급">겉보기 등급</a>은 9등급, <a href="/wiki/%EC%8B%9C%EC%A7%81%EA%B2%BD" title="시직경">시직경</a>은 8.7× 6.5분으로 크기는 약 6,500광년으로 작은 편이다. <a class="new" href="/w/index.php?title=%ED%9B%84%ED%87%B4_%EC%86%8D%EB%8F%84&amp;action=edit&amp;redlink=1" title="후퇴 속도 (없는 문서)">후퇴 속도</a>는 -200±6km/s이다. 대부분의 <a class="mw-redirect" href="/wiki/%ED%83%80%EC%9B%90_%EC%9D%80%ED%95%98" title="타원 은하">타원 은하</a>와 마찬가지로, 가스나 먼지가 없는, 늙은 붉고 노란 별들로 이루어져있다.</p>
<p>M32의 생성과정은 일반적인 은하 생성 모델로는 설명하기 어렵다. 최근 시뮬레이션 결과에 따르면, <a href="/wiki/%EC%95%88%EB%93%9C%EB%A1%9C%EB%A9%94%EB%8B%A4_%EC%9D%80%ED%95%98" title="안드로메다 은하">안드로메다 은하</a>의 강력한 <a href="/wiki/%EC%A1%B0%EC%84%9D%EB%A0%A5" title="조석력">조석력</a>에 의해 <a class="mw-redirect" href="/wiki/%EB%82%98%EC%84%A0_%EC%9D%80%ED%95%98" title="나선 은하">나선 은하</a>에서 <a class="mw-redirect" href="/wiki/%ED%83%80%EC%9B%90_%EC%9D%80%ED%95%98" title="타원 은하">타원 은하</a>로 변하였다는 주장이 있다.</p>
