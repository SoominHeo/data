<p><b>밥 헤임즈</b>(Bob Haymes, 본명은 <b>로버트 윌리엄 헤임즈</b>(Robert William Haymes), 초기 <a href="/wiki/%EC%98%88%EB%AA%85" title="예명">예명</a>은 <b>로버트 스탠턴</b>(Robert Stanton), <b>밥 스탠턴</b>(Bob Stanton), <a href="/wiki/1923%EB%85%84" title="1923년">1923년</a> <a href="/wiki/3%EC%9B%94_29%EC%9D%BC" title="3월 29일">3월 29일</a> ~ <a href="/wiki/1989%EB%85%84" title="1989년">1989년</a> <a href="/wiki/1%EC%9B%94_27%EC%9D%BC" title="1월 27일">1월 27일</a>)는 <a href="/wiki/%EB%AF%B8%EA%B5%AD" title="미국">미국</a>의 <a href="/wiki/%EC%8B%B1%EC%96%B4%EC%86%A1%EB%9D%BC%EC%9D%B4%ED%84%B0" title="싱어송라이터">싱어송라이터</a>, <a class="mw-redirect" href="/wiki/%EC%98%81%ED%99%94_%EB%B0%B0%EC%9A%B0" title="영화 배우">영화배우</a>, <a href="/wiki/%EC%82%AC%ED%9A%8C%EC%9E%90" title="사회자">방송인</a>이다.</p>
<h2><span class="mw-headline" id=".EC.83.9D.EC.95.A0">생애</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%B0%A5_%ED%97%A4%EC%9E%84%EC%8A%A4&amp;action=edit&amp;section=1" title="부분 편집: 생애">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p><a href="/wiki/%EC%95%84%EB%A5%B4%ED%97%A8%ED%8B%B0%EB%82%98" title="아르헨티나">아르헨티나</a> <a href="/wiki/%EC%8B%9C%EB%AF%BC%EA%B6%8C" title="시민권">시민권</a>자 신분의 <a href="/wiki/%EB%AF%B8%EA%B5%AD" title="미국">미국</a> <a class="mw-redirect" href="/wiki/%EC%98%81%ED%99%94_%EB%B0%B0%EC%9A%B0" title="영화 배우">영화배우</a> 겸 <a href="/wiki/%EC%8B%B1%EC%96%B4%EC%86%A1%EB%9D%BC%EC%9D%B4%ED%84%B0" title="싱어송라이터">싱어송라이터</a> <a href="/wiki/%EB%94%95_%ED%97%A4%EC%9E%84%EC%A6%88" title="딕 헤임즈">딕 헤임즈</a>(Dick Haymes)의 아우이기도 한 그는 <a href="/wiki/1940%EB%85%84" title="1940년">1940년</a> <a href="/wiki/%EA%B0%80%EC%88%98" title="가수">가수</a>로 첫 데뷔하였고 <a href="/wiki/1943%EB%85%84" title="1943년">1943년</a> <a href="/wiki/%EB%AF%B8%EA%B5%AD" title="미국">미국</a> <a href="/wiki/%EC%98%81%ED%99%94" title="영화">영화</a> 《Two senoritas from Chicago》의 조연으로 <a class="mw-redirect" href="/wiki/%EC%98%81%ED%99%94_%EB%B0%B0%EC%9A%B0" title="영화 배우">영화배우</a> 데뷔하였고 또한 이 작품으로 일약 스타덤에 올랐으며 이후 <a href="/wiki/1968%EB%85%84" title="1968년">1968년</a> 영화 《The money jungle》로 <a href="/wiki/%EC%98%81%ED%99%94_%EC%9D%8C%EC%95%85" title="영화 음악">영화음악</a>감독 데뷔하였고 <a href="/wiki/1988%EB%85%84" title="1988년">1988년</a> 영화 《Monkey shines》로 <a href="/wiki/%EC%98%81%ED%99%94_%EC%9D%8C%EC%95%85" title="영화 음악">영화음악</a><a href="/wiki/%EC%97%B0%EC%B6%9C%EA%B0%80" title="연출가">연출가</a> 데뷔하였다.</p>
<h2><span class="mw-headline" id=".EC.9C.A0.EB.AA.85.EC.84.B8">유명세</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%B0%A5_%ED%97%A4%EC%9E%84%EC%8A%A4&amp;action=edit&amp;section=2" title="부분 편집: 유명세">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
