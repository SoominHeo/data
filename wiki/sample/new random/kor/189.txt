<h2><span class="mw-headline" id=".EA.B0.9C.EC.B5.9C">개최</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=2012%EB%85%84_%EC%84%B8%EA%B3%84_%EB%B0%95%EB%9E%8C%ED%9A%8C&amp;action=edit&amp;section=2" title="부분 편집: 개최">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<h3><span class="mw-headline" id="1.EC.B0.A8_.EC.9C.A0.EC.B9.98_.EC.8B.9C.EB.8F.84_.282010.EB.85.84_.EC.84.B8.EA.B3.84.EB.B0.95.EB.9E.8C.ED.9A.8C.29">1차 유치 시도 (<a href="/wiki/2010%EB%85%84" title="2010년">2010년</a> 세계박람회)</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=2012%EB%85%84_%EC%84%B8%EA%B3%84_%EB%B0%95%EB%9E%8C%ED%9A%8C&amp;action=edit&amp;section=3" title="부분 편집: 1차 유치 시도 (2010년 세계박람회)">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p><a href="/wiki/1996%EB%85%84" title="1996년">1996년</a> <a href="/wiki/1%EC%9B%94_9%EC%9D%BC" title="1월 9일">1월 9일</a> <a class="mw-redirect" href="/wiki/%EA%B5%AD%EB%AC%B4%EC%B4%9D%EB%A6%AC" title="국무총리">국무총리</a> 주재로 제1회 해양개발위원회를 열고 '해양개발기본계획(안)'을 심의 통과하였는데 이날 발표된 8대 시책중 '해양엑스포 개최 검토'가 처음으로 포함되면서 엑스포에 대한 유치 시도가 시작되었다. 같은해 <a href="/wiki/9%EC%9B%94" title="9월">9월</a> <a href="/wiki/%EC%A0%84%EB%9D%BC%EB%82%A8%EB%8F%84" title="전라남도">전라남도</a>는 해양을 주제로 <a href="/wiki/2010%EB%85%84" title="2010년">2010년</a>에 엑스포를 개최할 수 있도록 국가정책으로 채택해달라는 의견을 전달하였고, <a href="/wiki/1997%EB%85%84" title="1997년">1997년</a> <a href="/wiki/5%EC%9B%94" title="5월">5월</a> <a href="/wiki/%EA%B4%91%EC%96%91%EC%8B%9C" title="광양시">광양시</a>에서 열린 '제2회 바다의 날' 행사에서 <a href="/wiki/%EA%B9%80%EC%98%81%EC%82%BC" title="김영삼">김영삼</a>대통령은 '2010년 해양엑스포를 전남에 유치하겠다'고 공식적으로 선언하게 되면서 가시화 되게된다. <a href="/wiki/1997%EB%85%84" title="1997년">1997년</a> <a href="/wiki/9%EC%9B%94" title="9월">9월</a> <a href="/wiki/%EC%A0%84%EB%9D%BC%EB%82%A8%EB%8F%84" title="전라남도">전라남도</a>는 '2010 해양엑스포 개최 기본구상안'을 정부에 제출하였는데, '2010년 5월부터 10월까지 해양을 주제로 하는 등록박람회를 개최한다'라는 내용을 골자로 하였다. <a class="mw-redirect" href="/wiki/%ED%95%B4%EC%96%91%EC%88%98%EC%82%B0%EB%B6%80" title="해양수산부">해양수산부</a>는 이를 토대로 7개 항목별로 가중치를 부여해 최고 점수를 획득한 지역을 후보지로 결정하겠다는 방침을 정했다. <a href="/wiki/%EC%97%AC%EC%88%98%EC%8B%9C" title="여수시">여수시</a>, <a href="/wiki/%EC%99%84%EB%8F%84%EA%B5%B0" title="완도군">완도군</a>, <a href="/wiki/%EB%AA%A9%ED%8F%AC%EC%8B%9C" title="목포시">목포시</a>, <a href="/wiki/%EC%8B%A0%EC%95%88%EA%B5%B0" title="신안군">신안군</a>이 유치를 위해 나섰는데 각 지자체는 경제적, 문화적 타당성을 기반을 유치활동을 전개하였다. <a href="/wiki/1999%EB%85%84" title="1999년">1999년</a> <a href="/wiki/3%EC%9B%94" title="3월">3월</a> 전남 3개 지역에 대한 심사결과를 발표하였고, 2010년 해양 엑스포 개최 후보지는 <a href="/wiki/%EC%97%AC%EC%88%98%EC%8B%9C" title="여수시">여수시</a>로 결정된다.</p>
<p>정부는 <a href="/wiki/1999%EB%85%84" title="1999년">1999년</a> <a href="/wiki/6%EC%9B%94_14%EC%9D%BC" title="6월 14일">6월 14일</a> 국무회의에서 '2010년 세계해양엑스포' 개최 및 유치추진 계획안을 의결하였고, 공식명칭을 '2010년 세계해양엑스포'에서 '2010년 세계박람회'로 변경하였다. <a href="/wiki/2000%EB%85%84" title="2000년">2000년</a> <a href="/wiki/5%EC%9B%94" title="5월">5월</a> 세계박람회 기본계획 II 가 새로 마련되었고, 개최후보지는 여수시의 신항지구로 바뀌게 되었다. <a href="/wiki/2001%EB%85%84" title="2001년">2001년</a> <a href="/wiki/5%EC%9B%94_2%EC%9D%BC" title="5월 2일">5월 2일</a> <a class="mw-redirect" href="/wiki/%EA%B5%AD%EB%AC%B4%EC%B4%9D%EB%A6%AC" title="국무총리">국무총리</a> 명의로 '2010년 세계박람회 유치신청서'를 <a class="mw-redirect" href="/wiki/BIE" title="BIE">BIE</a>에 제출하였고 역대 최고의 유치 경쟁률을 기록하면서 유치전이 시작되었다. <a href="/wiki/2002%EB%85%84" title="2002년">2002년</a> <a href="/wiki/3%EC%9B%94" title="3월">3월</a> <a class="mw-redirect" href="/wiki/BIE" title="BIE">BIE</a> 실사단이 여수를 방문해 실사를 진행하였으나 같은해 <a href="/wiki/12%EC%9B%94" title="12월">12월</a> <a href="/wiki/%EB%AA%A8%EB%82%98%EC%BD%94" title="모나코">모나코</a>에서 열린 제132차 BIE 총회해서 2010년 세계박람회는 <a href="/wiki/%EC%A4%91%EA%B5%AD" title="중국">중국</a> <a class="mw-redirect" href="/wiki/%EC%83%81%ED%95%98%EC%9D%B4" title="상하이">상하이</a>로 결정되면서 유치에 실패하게 된다.</p>
<h3><span class="mw-headline" id="2.EC.B0.A8_.EC.9C.A0.EC.B9.98_.EC.8B.9C.EB.8F.84_.282012.EB.85.84_.EC.84.B8.EA.B3.84.EB.B0.95.EB.9E.8C.ED.9A.8C.29">2차 유치 시도 (2012년 세계박람회)</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=2012%EB%85%84_%EC%84%B8%EA%B3%84_%EB%B0%95%EB%9E%8C%ED%9A%8C&amp;action=edit&amp;section=4" title="부분 편집: 2차 유치 시도 (2012년 세계박람회)">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p><a href="/wiki/2002%EB%85%84" title="2002년">2002년</a> <a href="/wiki/12%EC%9B%94_9%EC%9D%BC" title="12월 9일">12월 9일</a> <a href="/wiki/%EC%A0%84%EB%9D%BC%EB%82%A8%EB%8F%84" title="전라남도">전라남도</a>와 <a href="/wiki/%EC%97%AC%EC%88%98%EC%8B%9C" title="여수시">여수시</a>는 '2010년 세계박람회 유치 실패에 따른 전남동부권 현안 사업'을 정부에 건의한다. 엑스포 유치 실패로 인한 지역의 패배의식을 빠르게 극복하기 위해서는 빠른 재도전과 개최가 필요하다는 의견이 설득력을 얻게 되었고, 최초 <a href="/wiki/2015%EB%85%84" title="2015년">2015년</a> 등록박람회 유치를 희망했지만 이미 <a href="/wiki/2005%EB%85%84" title="2005년">2005년</a>에 일본 <a class="mw-disambig" href="/wiki/%EC%95%84%EC%9D%B4%EC%B9%98" title="아이치">아이치</a>에서, <a href="/wiki/2010%EB%85%84" title="2010년">2010년</a> 중국 <a class="mw-redirect" href="/wiki/%EC%83%81%ED%95%98%EC%9D%B4" title="상하이">상하이</a>에서 박람회가 개최되어 동아시아 연속 개최에 대한 부정적인 의견이 있었기에 <a href="/wiki/2010%EB%85%84" title="2010년">2010년</a> 으로 변경하였다. <a href="/wiki/2002%EB%85%84" title="2002년">2002년</a> <a href="/wiki/12%EC%9B%94_14%EC%9D%BC" title="12월 14일">12월 14일</a> 여수에서 '2012 세계박람회 범시민추진위원회'가 발족되었고, <a href="/wiki/12%EC%9B%94_26%EC%9D%BC" title="12월 26일">12월 26일</a> <a href="/wiki/%EB%85%B8%EB%AC%B4%ED%98%84" title="노무현">노무현</a> 대통령 당선자와 <a href="/wiki/%EA%B9%80%EB%8C%80%EC%A4%91" title="김대중">김대중</a> 대통령에게 유치 재추진을 건의하였다. <a href="/wiki/2003%EB%85%84" title="2003년">2003년</a> <a href="/wiki/2%EC%9B%94_16%EC%9D%BC" title="2월 16일">2월 16일</a> <a href="/wiki/%EC%A0%84%EB%9D%BC%EB%82%A8%EB%8F%84" title="전라남도">전라남도</a>는 2012년 인정박람회 여수 유치를 공식 발표하면서 재도전의 기틀을 마련하였다. <a href="/wiki/2004%EB%85%84" title="2004년">2004년</a> <a href="/wiki/3%EC%9B%94_24%EC%9D%BC" title="3월 24일">3월 24일</a> 이헌재 <a class="mw-redirect" href="/wiki/%EC%9E%AC%EC%A0%95%EA%B2%BD%EC%A0%9C%EB%B6%80" title="재정경제부">재정경제부</a> 겸 부총리는 '2012년 세계박람회 여수 유치를 국가사업으로 추진할 의사가 있다'고 전해 처음으로 공식화 되었으며, 같은해 <a href="/wiki/12%EC%9B%94_14%EC%9D%BC" title="12월 14일">12월 14일</a> 국무회의에서 '2012년 세계박람회 여수 유치'를 국가 계획으로 확정하면서 재도전이 공식화 되었다. 정부는 <a href="/wiki/2005%EB%85%84" title="2005년">2005년</a> <a href="/wiki/3%EC%9B%94" title="3월">3월</a> 관계부처 합동으로 유치추진기획단을 설치, 운영하였으며 같은해 <a href="/wiki/12%EC%9B%94" title="12월">12월</a> 제138차 <a class="mw-redirect" href="/wiki/BIE" title="BIE">BIE</a> 총회에서 2012년 세계박람회의 여수 유치 의사를 공식 선언하였다. <a href="/wiki/2006%EB%85%84" title="2006년">2006년</a> <a href="/wiki/2%EC%9B%94" title="2월">2월</a> <a class="mw-redirect" href="/wiki/%ED%95%B4%EC%96%91%EC%88%98%EC%82%B0%EB%B6%80" title="해양수산부">해양수산부</a>는 부지조성사업에 대한 협약을 체결하여 개최지역의 전시관 조성공사를 <a href="/wiki/2009%EB%85%84" title="2009년">2009년</a>까지 완료한다는 계획을 세웠으며 이는 한국의 유치 의사를 전 세계에 전달하기 위해서 였다.</p>
<p><a href="/wiki/2005%EB%85%84" title="2005년">2005년</a> <a href="/wiki/5%EC%9B%94_22%EC%9D%BC" title="5월 22일">5월 22일</a> 정부는 <a class="mw-redirect" href="/wiki/BIE" title="BIE">BIE</a> 본부에 공식 유치 신청서를 제출하였고, <a href="/wiki/2006%EB%85%84" title="2006년">2006년</a> <a href="/wiki/5%EC%9B%94" title="5월">5월</a> '2012여수세계박람회 유치위원회'가 결성되었다. 유치위원회는 사절단등을 통해 총 55개 국가에 유치전을 펼쳤다. <a href="/wiki/2007%EB%85%84" title="2007년">2007년</a> <a href="/wiki/9%EC%9B%94_13%EC%9D%BC" title="9월 13일">9월 13일</a>에는 제2차 여수세계박람회 국제심포지엄이 개최되어 <a href="/wiki/%EC%95%A8%EB%B9%88_%ED%86%A0%ED%94%8C%EB%9F%AC" title="앨빈 토플러">앨빈 토플러</a>, 무하마드 유누스 그라민뱅크 총재 등 세계의 석학의 참석으로 지구촌의 화제로 떠오르기도 했다. 마침내 <a href="/wiki/2007%EB%85%84" title="2007년">2007년</a> <a href="/wiki/11%EC%9B%94_27%EC%9D%BC" title="11월 27일">11월 27일</a> 제142차 <a class="mw-redirect" href="/wiki/BIE" title="BIE">BIE</a>총회에서 여수를 '2012년 세계박람회'개최 지역으로 선정되는 쾌거를 얻게 된다. <sup class="reference" id="cite_ref-2"><a href="#cite_note-2">[2]</a></sup></p>
<h2><span class="mw-headline" id=".EB.A7.88.EC.8A.A4.EC.BD.94.ED.8A.B8.EC.99.80_.EC.8B.AC.EB.B3.BC">마스코트와 심볼</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=2012%EB%85%84_%EC%84%B8%EA%B3%84_%EB%B0%95%EB%9E%8C%ED%9A%8C&amp;action=edit&amp;section=5" title="부분 편집: 마스코트와 심볼">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<h3><span class="mw-headline" id=".EB.A7.88.EC.8A.A4.EC.BD.94.ED.8A.B8">마스코트</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=2012%EB%85%84_%EC%84%B8%EA%B3%84_%EB%B0%95%EB%9E%8C%ED%9A%8C&amp;action=edit&amp;section=6" title="부분 편집: 마스코트">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>여수세계박람회의 <a href="/wiki/%EB%A7%88%EC%8A%A4%EC%BD%94%ED%8A%B8" title="마스코트">마스코트</a>인 ‘여니와수니’는 <a class="mw-redirect" href="/wiki/%EC%96%B4%EB%A5%98" title="어류">어류</a>의 먹이자원이며 <a href="/wiki/%EB%B0%94%EB%8B%A4" title="바다">바다</a>와 <a href="/wiki/%EC%97%B0%EC%95%88" title="연안">연안</a>을 지켜주는 <a href="/wiki/%EC%83%9D%EB%AA%85" title="생명">생명</a>의 근원인 해양생물(<a href="/wiki/%ED%94%8C%EB%9E%91%ED%81%AC%ED%86%A4" title="플랑크톤">플랑크톤</a>)을 <a class="mw-redirect mw-disambig" href="/wiki/%EB%AA%A8%ED%8B%B0%EB%B8%8C" title="모티브">모티브</a>로 바다의 아름다운 빛깔을 마스코트의 머리 색에 담았으며, 특히 <a class="new" href="/w/index.php?title=%EC%A7%99%EC%9D%80_%ED%91%B8%EB%A5%B8%EC%83%89&amp;action=edit&amp;redlink=1" title="짙은 푸른색 (없는 문서)">짙은 푸른색</a>은 <a href="/wiki/%EC%8B%AC%ED%95%B4" title="심해">심해</a>의 무한한 <a href="/wiki/%EC%9E%90%EC%9B%90" title="자원">자원</a>을 표현했다. 또한 머리부분의 <a href="/wiki/%EC%B4%89%EC%88%98" title="촉수">촉수</a>는 해양생물의 촉수로 박람회를 찾은 모든이들과 교감하고 대화할 수 있도록 다양한 모양으로 변화가 가능한 형태로 표현하였다. ‘여니와수니’는 <a class="mw-redirect" href="/wiki/%EC%97%AC%EC%88%98" title="여수">여수</a>를 <a href="/wiki/%EC%96%B4%EA%B7%BC" title="어근">어근</a>으로 엑스포 개최지의 인지도를 높이는 동시에 ‘여니’는 ‘연다’는 <a class="mw-redirect" href="/wiki/%EC%9D%B4%EB%AF%B8%EC%A7%80" title="이미지">이미지</a>로 2012여수세계박람회의 성공적인 개최를 알리며 ‘수니’는 ‘물’,‘우수한’,‘여성’의 이미지로 해양박람회의 상징뿐만 아니라 우수하고 수준 높은 박람회를 표현하였다.</p>
<h3><span class="mw-headline" id=".EC.8B.AC.EB.B3.BC">심볼</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=2012%EB%85%84_%EC%84%B8%EA%B3%84_%EB%B0%95%EB%9E%8C%ED%9A%8C&amp;action=edit&amp;section=7" title="부분 편집: 심볼">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p><a class="new" href="/w/index.php?title=%EC%8B%AC%EB%B3%BC_%EB%A7%88%ED%81%AC&amp;action=edit&amp;redlink=1" title="심볼 마크 (없는 문서)">심볼 마크</a>는 여수세계박람회의 이미지를 대표하는 상징체로서 EI(Expo identity)시스템의 핵심요소이다. 새롭게 개발된 2012 여수세계박람회 심볼마크는 유기적인 형태(<a href="/wiki/%EB%B9%A8%EA%B0%95" title="빨강">빨강</a>은 <a href="/wiki/%EC%83%9D%ED%83%9C%EA%B3%84" title="생태계">생태계</a>, <a href="/wiki/%ED%8C%8C%EB%9E%91" title="파랑">파랑</a>은 <a href="/wiki/%EB%B0%94%EB%8B%A4" title="바다">바다</a>, <a href="/wiki/%EC%B4%88%EB%A1%9D" title="초록">초록</a>은 <a class="mw-disambig" href="/wiki/%ED%99%98%EA%B2%BD" title="환경">환경</a>을 의미함)을 단순하게 추상화하여, 2012 여수세계박람회의 주제인 '살아있는 바다와 연안'을 상징화 하였다. 또한 전체적인 원의 모양은 <a href="/wiki/%EC%A7%80%EA%B5%AC" title="지구">지구</a>를 의미하여 그 안에 속한 동적인 3개의 <a class="mw-redirect mw-disambig" href="/wiki/%EB%AA%A8%ED%8B%B0%EB%B8%8C" title="모티브">모티브</a>를 통하여 지구의 미래지향적인 의미를 나타냈다.</p>

<h2><span class="mw-headline" id=".EC.A0.84.EC.8B.9C.EA.B4.80">전시관</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=2012%EB%85%84_%EC%84%B8%EA%B3%84_%EB%B0%95%EB%9E%8C%ED%9A%8C&amp;action=edit&amp;section=9" title="부분 편집: 전시관">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<h3><span class="mw-headline" id=".ED.95.9C.EA.B5.AD.EA.B4.80">한국관</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=2012%EB%85%84_%EC%84%B8%EA%B3%84_%EB%B0%95%EB%9E%8C%ED%9A%8C&amp;action=edit&amp;section=10" title="부분 편집: 한국관">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>총 1개의 건물에서 연면적 5,309㎡에 1,800여 ㎡를 전시 공간으로 활용하였다. 한국관의 주제는 ‘한국인의 바다정신과 해양역량’으로, 디오라마와 돔 영상을 사용하였다.</p>
<h3><span class="mw-headline" id=".EA.B5.AD.EC.A0.9C.EA.B4.80">국제관</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=2012%EB%85%84_%EC%84%B8%EA%B3%84_%EB%B0%95%EB%9E%8C%ED%9A%8C&amp;action=edit&amp;section=11" title="부분 편집: 국제관">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>여수세계박람회의 가장중심에 있는 박물관으로 총 4개의 건물과 그 사이를 이어주는 연결브릿지를 통해 대서양 공동관, 태평양 공동관, 인도양 공동관과 46개국의 전시관으로 구성되었다. 그 중 전시관 D관에는 중국관 미국관 일본관 등 주요 국가들이 밀집되어있어 엑스포에서 가장 관람객을 많이 모은 전시관으로 꼽히고 있다</p>
<h3><span class="mw-headline" id=".EC.A3.BC.EC.A0.9C.EA.B4.80">주제관</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=2012%EB%85%84_%EC%84%B8%EA%B3%84_%EB%B0%95%EB%9E%8C%ED%9A%8C&amp;action=edit&amp;section=12" title="부분 편집: 주제관">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>총 1개의 건물에서 연면적 6,000㎡에 1,800여 ㎡를 전시 공간으로 활용하였다. 주제관의 주제는 ‘해양가치와 미래비전’으로, 학술행사와 퍼포먼스를 포함한 전시관으로 구성되었다. 해양베스트관이라고도 불린다.</p>
<h3><span class="mw-headline" id=".EC.97.91.EC.8A.A4.ED.8F.AC.EB.94.94.EC.A7.80.ED.84.B8.EA.B0.A4.EB.9F.AC.EB.A6.AC">엑스포디지털갤러리</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=2012%EB%85%84_%EC%84%B8%EA%B3%84_%EB%B0%95%EB%9E%8C%ED%9A%8C&amp;action=edit&amp;section=13" title="부분 편집: 엑스포디지털갤러리">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>국제관의 각 건물을 연결하는 LED 스크린 천장으로, 길이는 218.24m에 너비 30.72m이며, 해상도는 6,820 x 960픽셀 (654만 화소)이다.</p>
<h3><span class="mw-headline" id=".EB.B9.85.EC.98.A4">빅오</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=2012%EB%85%84_%EC%84%B8%EA%B3%84_%EB%B0%95%EB%9E%8C%ED%9A%8C&amp;action=edit&amp;section=14" title="부분 편집: 빅오">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>빅오(Big-O)는 거대한 바다(Big Ocean)에서 유래한 단어로서, 수변의 해양공원이다. 해상분수와 스크린을 이용한 빅오쇼(Big-O Show)가 열리는 행사 공간이다.</p>
