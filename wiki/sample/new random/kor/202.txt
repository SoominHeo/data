<p><b>피델 발데스 라모스</b>(Fidel Valdez Ramos, <a href="/wiki/1928%EB%85%84" title="1928년">1928년</a> <a href="/wiki/3%EC%9B%94_18%EC%9D%BC" title="3월 18일">3월 18일</a> ~ )는 <a href="/wiki/%ED%95%84%EB%A6%AC%ED%95%80" title="필리핀">필리핀</a>의 군인, 정치인이다. <a href="/wiki/1992%EB%85%84" title="1992년">1992년</a> ~ <a href="/wiki/1998%EB%85%84" title="1998년">1998년</a> <a href="/wiki/%ED%95%84%EB%A6%AC%ED%95%80%EC%9D%98_%EB%8C%80%ED%86%B5%EB%A0%B9" title="필리핀의 대통령">대통령</a>을 지냈다. <a href="/wiki/%ED%95%9C%EA%B5%AD_%EC%A0%84%EC%9F%81" title="한국 전쟁">한국 전쟁</a> 당시 장교로 참전하였고, <a href="/wiki/%EB%B2%A0%ED%8A%B8%EB%82%A8_%EC%A0%84%EC%9F%81" title="베트남 전쟁">베트남 전쟁</a> 때에도 참전하였다. 전임 대통령 <a href="/wiki/%ED%8E%98%EB%A5%B4%EB%94%94%EB%82%9C%EB%93%9C_%EB%A7%88%EB%A5%B4%EC%BD%94%EC%8A%A4" title="페르디난드 마르코스">페르디난드 마르코스</a>의 친척이다.</p>
<h2><span class="mw-headline" id=".EC.83.9D.EC.95.A0">생애</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%ED%94%BC%EB%8D%B8_%EB%9D%BC%EB%AA%A8%EC%8A%A4&amp;action=edit&amp;section=1" title="부분 편집: 생애">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p><a href="/wiki/%ED%8C%A1%EA%B0%80%EC%8B%9C%EB%82%9C_%EC%A3%BC" title="팡가시난 주">팡가시난 주</a>의 명문 가문에서 태어났다. 부친은 국회의원과 외무장관을 지낸 정치인이다. <a href="/wiki/1965%EB%85%84" title="1965년">1965년</a> ~ <a href="/wiki/1986%EB%85%84" title="1986년">1986년</a> 대통령으로 있던 <a href="/wiki/%ED%8E%98%EB%A5%B4%EB%94%94%EB%82%9C%EB%93%9C_%EB%A7%88%EB%A5%B4%EC%BD%94%EC%8A%A4" title="페르디난드 마르코스">페르디난드 마르코스</a>와는 6촌 관계이다. 피델은 <a href="/wiki/%EB%AF%B8%EA%B5%AD_%EC%9C%A1%EA%B5%B0%EC%82%AC%EA%B4%80%ED%95%99%EA%B5%90" title="미국 육군사관학교">미국 육군사관학교</a>를 수료했으며, <a class="mw-redirect" href="/wiki/%EB%A7%88%EB%8B%90%EB%9D%BC_%EC%95%84%ED%85%8C%EB%84%A4%EC%98%A4_%EB%8C%80%ED%95%99%EA%B5%90" title="마닐라 아테네오 대학교">마닐라 아테네오 대학교</a>에서 경영학 석사, <a href="/wiki/%EB%AF%B8%EA%B5%AD" title="미국">미국</a> <a class="mw-redirect" href="/wiki/%EC%9D%BC%EB%A6%AC%EB%85%B8%EC%9D%B4_%EB%8C%80%ED%95%99%EA%B5%90_%EC%96%B4%EB%B0%94%EB%82%98-%EC%83%B4%ED%8E%98%EC%9D%B8" title="일리노이 대학교 어바나-샴페인">일리노이 대학교</a>에서 공학 석사학위를 받았다. <a href="/wiki/%ED%95%9C%EA%B5%AD_%EC%A0%84%EC%9F%81" title="한국 전쟁">한국 전쟁</a>에 장교로 참전했으며, 이후 계속 군에 복무하며 <a href="/wiki/%EB%B2%A0%ED%8A%B8%EB%82%A8_%EC%A0%84%EC%9F%81" title="베트남 전쟁">베트남 전쟁</a>에도 참전했으며, 대장(4성 장군)으로까지 승진했다.</p>
<p><a href="/wiki/1986%EB%85%84" title="1986년">1986년</a> 자신의 6촌인 마르코스의 독재를 타도하기 위한 민중봉기에 참가했으며, 마르코스가 축출되고 <a href="/wiki/%EC%BD%94%EB%9D%BC%EC%86%90_%EC%95%84%ED%82%A4%EB%85%B8" title="코라손 아키노">코라손 아키노</a>가 대통령이 된 후 육군 참모 총장으로 임명되었다. <a href="/wiki/1988%EB%85%84" title="1988년">1988년</a> ~ <a href="/wiki/1991%EB%85%84" title="1991년">1991년</a> 국방장관으로 재직했으며, <a href="/wiki/1992%EB%85%84" title="1992년">1992년</a> <a class="new" href="/w/index.php?title=%EB%9D%BC%EC%B9%B4%EC%8A%A4_%EB%AF%BC%EC%A3%BC%EB%8B%B9&amp;action=edit&amp;redlink=1" title="라카스 민주당 (없는 문서)">라카스 민주당</a>의 대통령 선거 후보가 되어 출마, 당선되어 그 해 6월부터 6년간 대통령으로 재직했다. 대통령으로 재직하며 경제 성장을 이끌고, 심각한 범죄 문제 해결에 노력하였으나, 임기 말 <a href="/wiki/%EC%95%84%EC%8B%9C%EC%95%84_%EA%B8%88%EC%9C%B5_%EC%9C%84%EA%B8%B0" title="아시아 금융 위기">아시아 금융 위기</a> 영향으로 다소 타격을 받았다. 1993년 5월에 대한민국을 공식 방문하였다.<sup class="reference" id="cite_ref-.EC.96.91.EC.8A.B9.EC.9C.A4_1-0"><a href="#cite_note-.EC.96.91.EC.8A.B9.EC.9C.A4-1">[1]</a></sup> <span class="reference plainlinksneverexpand" style="white-space:nowrap"><sup>:367</sup></span></p>
