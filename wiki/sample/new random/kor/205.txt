<p><b>자이언츠 스타디움</b>(Giants Stadium)은 <a href="/wiki/%EB%AF%B8%EA%B5%AD" title="미국">미국</a> <a href="/wiki/%EB%89%B4%EC%A0%80%EC%A7%80_%EC%A3%BC" title="뉴저지 주">뉴저지 주</a> <a href="/wiki/%EC%9D%B4%EC%8A%A4%ED%8A%B8%EB%9F%AC%EB%8D%94%ED%8D%BC%EB%93%9C" title="이스트러더퍼드">이스트러더퍼드</a>에 있던 <a href="/wiki/%EA%B2%BD%EA%B8%B0%EC%9E%A5" title="경기장">경기장</a>이다. <a href="/wiki/%EB%82%B4%EC%85%94%EB%84%90_%ED%92%8B%EB%B3%BC_%EB%A6%AC%EA%B7%B8" title="내셔널 풋볼 리그">NFL</a> <a href="/wiki/%EB%89%B4%EC%9A%95_%EC%9E%90%EC%9D%B4%EC%96%B8%EC%B8%A0" title="뉴욕 자이언츠">뉴욕 자이언츠</a>가 1976년 개장부터 폐장까지, <a href="/wiki/%EB%89%B4%EC%9A%95_%EC%A0%9C%EC%B8%A0" title="뉴욕 제츠">뉴욕 제츠</a>가 1984년부터 폐장까지, <a href="/wiki/%EB%A9%94%EC%9D%B4%EC%A0%80_%EB%A6%AC%EA%B7%B8_%EC%82%AC%EC%BB%A4" title="메이저 리그 사커">MLS</a> <a href="/wiki/%EB%89%B4%EC%9A%95_%EB%A0%88%EB%93%9C%EB%B6%88%EC%8A%A4" title="뉴욕 레드불스">뉴욕 레드불스</a>가 1996년부터 폐장까지 이 경기장을 본거지로 하고 있었다. 2005년 허리케인 카트리나로 인해 <a href="/wiki/%EB%89%B4%EC%98%AC%EB%A6%AC%EC%96%B8%EC%8A%A4_%EC%84%B8%EC%9D%B8%EC%B8%A0" title="뉴올리언스 세인츠">뉴올리언스 세인츠</a>의 임시 홈구장으로 사용했다.</p>
