<p><b>아메리카</b>(<span style="font-size: smaller;"><a href="/wiki/%EC%98%81%EC%96%B4" title="영어">영어</a>:</span> <span lang="en" xml:lang="en">Americas</span>, <span style="font-size: smaller;"><a href="/wiki/%EC%8A%A4%ED%8E%98%EC%9D%B8%EC%96%B4" title="스페인어">스페인어</a>:</span> <span lang="es" xml:lang="es">América</span>, <span style="font-size: smaller;"><a href="/wiki/%ED%8F%AC%EB%A5%B4%ED%88%AC%EA%B0%88%EC%96%B4" title="포르투갈어">포르투갈어</a>:</span> <span lang="pt" xml:lang="pt">América</span>, <span style="font-size: smaller;"><a href="/wiki/%ED%94%84%EB%9E%91%EC%8A%A4%EC%96%B4" title="프랑스어">프랑스어</a>:</span> <span lang="fr" xml:lang="fr">Amérique</span>, <span style="font-size: smaller;"><a href="/wiki/%EB%84%A4%EB%8D%9C%EB%9E%80%EB%93%9C%EC%96%B4" title="네덜란드어">네덜란드어</a>:</span> <span lang="nl" xml:lang="nl">Amerika</span>, <small><a href="/wiki/%EC%9D%8C%EC%97%AD%EC%96%B4" title="음역어">음역어</a>:</small> 미주(美洲))는 <a href="/wiki/%EC%84%9C%EB%B0%98%EA%B5%AC" title="서반구">서반구</a>의 <a href="/wiki/%EB%8C%80%EB%A5%99" title="대륙">대륙</a>으로, <a href="/wiki/%EB%B6%81%EC%95%84%EB%A9%94%EB%A6%AC%EC%B9%B4" title="북아메리카">북아메리카</a>와 <a href="/wiki/%EC%A4%91%EC%95%99%EC%95%84%EB%A9%94%EB%A6%AC%EC%B9%B4" title="중앙아메리카">중앙아메리카</a>, <a href="/wiki/%EB%82%A8%EC%95%84%EB%A9%94%EB%A6%AC%EC%B9%B4" title="남아메리카">남아메리카</a><sup class="reference" id="cite_ref-1"><a href="#cite_note-1">[1]</a></sup>를 함께 부르는 이름이다. 아메리카 대륙은 남북이 매우 좁은 <a href="/wiki/%ED%8C%8C%EB%82%98%EB%A7%88_%EC%A7%80%ED%98%91" title="파나마 지협">파나마 지협</a>으로 연결되어 있다. 개척 민족의 비율에 따라 북부 아메리카는 <a href="/wiki/%EC%95%B5%EA%B8%80%EB%A1%9C%EC%95%84%EB%A9%94%EB%A6%AC%EC%B9%B4" title="앵글로아메리카">앵글로아메리카</a>, 남부 아메리카는 <a class="mw-redirect" href="/wiki/%EB%9D%BC%ED%8B%B4%EC%95%84%EB%A9%94%EB%A6%AC%EC%B9%B4" title="라틴아메리카">라틴아메리카</a>라고 불린다. <a class="new" href="/w/index.php?title=%EC%8B%A0%ED%95%AD%EB%A1%9C_%EA%B0%9C%EC%B2%99&amp;action=edit&amp;redlink=1" title="신항로 개척 (없는 문서)">신항로 개척</a> 때의 유럽계 개척자들의 유입으로 인하여 현재 토착 원주민들은 거의 멸족에 가까운 상태로, 특히 원시의 모습 그대로를 간직한 민족은 더더욱 남아있지 않은 상황으로 보호 지정되어 있다.</p>
<p>면적은 약 4,255만km<sup>2</sup>로, 지구 표면적의 8.3%, 육지 면적의 28.4%에 해당한다. 2016년 미주 인구는 세계 총인구(74억명)의 13.9%인 약 10억 3,500만여명이다.</p>

<h2><span class="mw-headline" id=".EC.9D.B4.EB.A6.84.EC.9D.98_.EC.9C.A0.EB.9E.98">이름의 유래</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%95%84%EB%A9%94%EB%A6%AC%EC%B9%B4&amp;action=edit&amp;section=1" title="부분 편집: 이름의 유래">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>아메리카가 처음으로 대륙의 이름에 쓰인 것은 <a href="/wiki/1507%EB%85%84" title="1507년">1507년</a>이다. <a href="/wiki/%EB%8F%85%EC%9D%BC%EC%9D%B8" title="독일인">독일인</a> 지도 제작자 <a href="/wiki/%EB%A7%88%EB%A5%B4%ED%8B%B4_%EB%B0%9C%ED%8A%B8%EC%A0%9C%EB%AE%90%EB%9F%AC" title="마르틴 발트제뮐러">마르틴 발트제뮐러</a>가 이탈리아인 탐험가 <a href="/wiki/%EC%95%84%EB%A9%94%EB%A6%AC%EA%B3%A0_%EB%B2%A0%EC%8A%A4%ED%91%B8%EC%B9%98" title="아메리고 베스푸치">아메리고 베스푸치</a>의 이름을 따서 지도를 만들면서 아메리카란 이름이 붙었다. <a class="mw-redirect" href="/wiki/%EC%97%90%EC%8A%A4%ED%8C%8C%EB%83%90" title="에스파냐">에스파냐</a>의 지원을 받아 대륙을 탐험한 <a href="/wiki/%ED%81%AC%EB%A6%AC%EC%8A%A4%ED%86%A0%ED%8D%BC_%EC%BD%9C%EB%9F%BC%EB%B2%84%EC%8A%A4" title="크리스토퍼 콜럼버스">크리스토퍼 콜럼버스</a>는 <a href="/wiki/1506%EB%85%84" title="1506년">1506년</a> 사망할 때까지 자신이 발견한 대륙이 <a href="/wiki/%EC%9D%B8%EB%8F%84" title="인도">인도</a>의 일부라고 생각했고 그래서 현재도 그 지역은 <a href="/wiki/%EC%84%9C%EC%9D%B8%EB%8F%84_%EC%A0%9C%EB%8F%84" title="서인도 제도">서인도 제도</a>라 불린다.</p>
