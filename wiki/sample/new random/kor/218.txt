<p><b>오사카 성</b> (<span style="font-size: smaller;"><a href="/wiki/%EC%9D%BC%EB%B3%B8%EC%96%B4" title="일본어">일본어</a>:</span> <span lang="ja"><span lang="ja"><ruby><rb>大坂城,大阪城</rb> <rp>(</rp><rt>おおざかじょう,おおさかじょう</rt><rp>)</rp></ruby></span></span>)은 <a class="mw-redirect" href="/wiki/%EC%85%8B%EC%93%B0%EB%85%B8%EC%BF%A0%EB%8B%88" title="셋쓰노쿠니">셋쓰 국</a> <a class="new" href="/w/index.php?title=%ED%9E%88%EA%B0%80%EC%8B%9C%EB%82%98%EB%A6%AC_%EA%B5%B0&amp;action=edit&amp;redlink=1" title="히가시나리 군 (없는 문서)">히가시나리 군</a> 오사카에 있었던 <a class="mw-redirect" href="/wiki/%EC%95%84%EC%A6%88%EC%B9%98%EB%AA%A8%EB%AA%A8%EC%95%BC%EB%A7%88_%EC%8B%9C%EB%8C%80" title="아즈치모모야마 시대">아즈치모모야마 시대</a>부터 <a href="/wiki/%EC%97%90%EB%8F%84_%EC%8B%9C%EB%8C%80" title="에도 시대">에도 시대</a>의 성이다. 다른 이름은 <b>긴조</b> (金城) 혹은 <b>긴조</b> (錦城)로, 과거에는 오사카(大坂)가 <a href="/wiki/%EA%B7%BC%EB%8C%80" title="근대">근대</a>에 와서 오사카(大阪)로 표기하도록 개정되었기 때문에, 현재에는 오사카(大阪)라고 많이 표기하고 있다. 또, 과거에는 오자카라고 읽었지만, 현재에는 오사카로 많이 읽혀진다. 도요토미 정권의 본성으로 있었지만, <a href="/wiki/%EC%98%A4%EC%82%AC%EC%B9%B4_%EC%A0%84%ED%88%AC" title="오사카 전투">오사카 전투</a>에서 소실되었다. 그리고, 그 후, 에도 시대에 재건을 해 에도 막부의 서일본 지배의 거점으로 삼았다. 성이 있는 곳은 현재 <a href="/wiki/%EC%98%A4%EC%82%AC%EC%B9%B4_%EC%8B%9C" title="오사카 시">오사카 시</a> <a href="/wiki/%EC%A3%BC%EC%98%A4_%EA%B5%AC_(%EC%98%A4%EC%82%AC%EC%B9%B4_%EC%8B%9C)" title="주오 구 (오사카 시)">주오 구</a>의 <b><a class="mw-redirect" href="/wiki/%EC%98%A4%EC%82%AC%EC%B9%B4%EC%A1%B0_%EA%B3%B5%EC%9B%90" title="오사카조 공원">오사카조 공원</a></b> (大阪城公園)이다. <a href="/wiki/%EA%B5%AC%EB%A7%88%EB%AA%A8%ED%86%A0_%EC%84%B1" title="구마모토 성">구마모토 성</a>, <a href="/wiki/%EB%82%98%EA%B3%A0%EC%95%BC_%EC%84%B1_(%EC%95%84%EC%9D%B4%EC%B9%98_%ED%98%84)" title="나고야 성 (아이치 현)">나고야 성</a>과 더불어 <b>일본 3대 명성</b>중 하나이다.</p>

도요토미 가문의 천수<br/>
〈<a href="/wiki/%EC%98%A4%EC%82%AC%EC%B9%B4_%EC%A0%84%ED%88%AC#.EC.98.A4.EC.82.AC.EC.B9.B4_.EC.97.AC.EB.A6.84.EC.9D.98_.EC.A7.84" title="오사카 전투">오사카 여름의진</a>도 병풍〉</div>

<p><a href="/wiki/1583%EB%85%84" title="1583년">1583년</a> <a href="/wiki/%EC%84%BC%EA%B3%A0%EC%BF%A0_%EC%8B%9C%EB%8C%80" title="센고쿠 시대">센고쿠 시대</a> <a href="/wiki/%EC%9D%B4%EC%8B%9C%EC%95%BC%EB%A7%88_%ED%98%BC%EA%B0%84%EC%A7%80" title="이시야마 혼간지">이시야마 혼간지</a>와 그 절안에 위치한 마을에 <a href="/wiki/%EB%8F%84%EC%9A%94%ED%86%A0%EB%AF%B8_%ED%9E%88%EB%8D%B0%EC%9A%94%EC%8B%9C" title="도요토미 히데요시">도요토미 히데요시</a>가 축성을 개시했다.</p>
<p>완성에 1 소요된 혼마루는 이시야마 혼간지터에 조성되었고, 천수는 옛 그림에서 보는 것과 같이 외관은 5층5계이다. 그리고, 기와등에 금박을 입혀 화려하게 장식했다. 히데요시의 임종이 있기까지 니노마루, 산노마루에는 소가마에<sup class="reference" id="cite_ref-1"><a href="#cite_note-1">[1]</a></sup>를 건설해 3중의 해자와 운하가 있는 견실한 성으로 완성되었다. 한창 오사카 성을 축성할 때 히데요시를 방문한 <a href="/wiki/%EC%98%A4%ED%86%A0%EB%AA%A8_%EC%86%8C%EB%A6%B0" title="오토모 소린">오토모 소린</a>이 성 주위를 둘러보고는〈오사카 성은 삼국<sup class="reference" id="cite_ref-2"><a href="#cite_note-2">[2]</a></sup>.무쌍〉이라고 불렀다고 한다.</p>
<p>축성자로 있는 히데요시 자신은 <a href="/wiki/%EA%B5%90%ED%86%A0_%EC%8B%9C" title="교토 시">교토</a>에 <a class="mw-redirect" href="/wiki/%EC%A3%BC%EB%9D%BC%EC%BF%A0_%EB%8B%A4%EC%9D%B4" title="주라쿠 다이">주라쿠 다이</a>, <a href="/wiki/%ED%9B%84%EC%8B%9C%EB%AF%B8_%EC%84%B1" title="후시미 성">후시미 성</a>을 차례차례 축성하고, 오사카보다는 주로 교토에 머물렀다. 히데요시가 죽은 후 <a href="/wiki/1599%EB%85%84" title="1599년">1599년</a> <a href="/wiki/%EB%8F%84%EC%9A%94%ED%86%A0%EB%AF%B8_%ED%9E%88%EB%8D%B0%EC%9A%94%EB%A6%AC" title="도요토미 히데요리">도요토미 히데요리</a>가 후시미 성에서 오사카 성의 혼마루로 이주한다. 또, 정권을 실질적으로 장악한 <a class="mw-redirect" href="/wiki/%EA%B3%A0%EB%8B%A4%EC%9D%B4%EB%A1%9C" title="고다이로">고다이로</a>(五大老)의 <a href="/wiki/%EB%8F%84%EC%BF%A0%EA%B0%80%EC%99%80_%EC%9D%B4%EC%97%90%EC%95%BC%EC%8A%A4" title="도쿠가와 이에야스">도쿠가와 이에야스</a>도 오사카 성의 <a class="mw-redirect" href="/wiki/%EB%8B%88%EB%85%B8%EB%A7%88%EB%A3%A8" title="니노마루">니노마루</a>로 옮겨와 정무를 보아온다.</p>
<h3><span class="mw-headline" id=".EC.97.90.EB.8F.84_.EC.8B.9C.EB.8C.80">에도 시대</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%98%A4%EC%82%AC%EC%B9%B4_%EC%84%B1&amp;action=edit&amp;section=3" title="부분 편집: 에도 시대">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p><a href="/wiki/1603%EB%85%84" title="1603년">1603년</a> <a href="/wiki/%EC%97%90%EB%8F%84_%EB%A7%89%EB%B6%80" title="에도 막부">에도 막부</a>가 성립된 후에도 히데요리는 오사카 성에 머물면서 <a class="mw-redirect" href="/wiki/%EC%85%8B%EC%93%B0%EB%85%B8%EC%BF%A0%EB%8B%88" title="셋쓰노쿠니">셋쓰</a>를 지배했고, <a href="/wiki/1614%EB%85%84" title="1614년">1614년</a> <a href="/wiki/%EC%98%A4%EC%82%AC%EC%B9%B4_%EC%A0%84%ED%88%AC" title="오사카 전투">오사카 성 겨울전투</a>에서 이에야스가 이끄는 군의 공격을 물리친다. 그리고, 소마에와 <a class="mw-redirect" href="/wiki/%EC%82%B0%EB%85%B8%EB%A7%88%EB%A3%A8" title="산노마루">산노마루</a>를 폐기하는 조건으로 강화를 맺지만, 조건에 없는 니노마루의 외측 해자까지 메우게 되었다. 오사카 성은 내측 해자와 혼마루만 남는 벌거벗은 성이 되어 버렸다. 히데요리는 매워진 외측 해자를 다시 만들었기 때문에 강화는 파기되어 겨울전투로부터 4개월 후 <a href="/wiki/1615%EB%85%84" title="1615년">1615년</a> 오사카 여름전투에서 패배하게 된다. 이때 혼마루는 소실되고, 도요토미 가문은 멸문되었다.</p>
<p><b>도쿠가와 가문의 오사카 성</b></p>

<p>재만 남은 오사카 성은 당초에 이에야스의 외손인 <a class="new" href="/w/index.php?title=%EB%A7%88%EC%93%B0%EB%8B%A4%EC%9D%B4%EB%9D%BC_%EB%8B%A4%EB%8B%A4%EC%95%84%ED%82%A4%EB%9D%BC_(1583%EB%85%84)&amp;action=edit&amp;redlink=1" title="마쓰다이라 다다아키라 (1583년) (없는 문서)">마쓰다이라 다다아키라</a>에게 주었지만, <a href="/wiki/1619%EB%85%84" title="1619년">1619년</a> 막부의 직할령으로 편입되었고, <a href="/wiki/1620%EB%85%84" title="1620년">1620년</a>부터 <a href="/wiki/%EB%8F%84%EC%BF%A0%EA%B0%80%EC%99%80_%ED%9E%88%EB%8D%B0%ED%83%80%EB%8B%A4" title="도쿠가와 히데타다">도쿠가와 히데타다</a>에 의해 성이 재건되어 <a href="/wiki/1629%EB%85%84" title="1629년">1629년</a> 완성되었다.</p>
<p>도쿠가와 가문의 오사카 성은 도요토미 가문의 오사카 성의 성벽과 해자를 파괴하고, 전체에 흙을 돋우어 그 위에 높은 석벽을 쌓아, 옛 도요토미씨의 오사카 성의 흔적을 지워버린다.</p>
<p>도쿠가와 가문의 오사카 성은 예전의 도요토미 가문의 성곽의 넓이에 1/4에 불과 했지만, 성의 상징인 천수는 크기와 높이면에서 기존의 도요토미씨 천수를 능가했고, 그 구조도 달랐다. 하지만, 외관은 5층으로 같다. 아마 이렇게 지어진 것은 옛 도요토미씨의 기억을 지우고 도쿠가와씨의 위세를 전국에 알리려는 속셈으로 보인다.</p>
<p>막부 직할 성으로 있는 도쿠가와 가문의 오사카 성의 성주는 도쿠가와 막부가의 역대 쇼군이었고, 다이신<sup class="reference" id="cite_ref-3"><a href="#cite_note-3">[3]</a></sup>에서 선출해 오사카 성 조다이<sup class="reference" id="cite_ref-4"><a href="#cite_note-4">[4]</a></sup>위탁했다. 또, 다이신보다 낮은 직급의 쇼신중에 6명을 선출해 2명은 오사카 조반과 오사카 가반을 두어 성을 경비했다. 에도 시대 여러번의 화재로 손상을 입어 수리를 했지만, <a href="/wiki/1665%EB%85%84" title="1665년">1665년</a> <a href="/wiki/%EB%82%99%EB%A2%B0" title="낙뢰">낙뢰</a>로 인해 천수가 소실되었고, 그 후 천수의 복원은 없었다.</p>
<p><br/>
<a href="/wiki/1868%EB%85%84" title="1868년">1868년</a> <a class="mw-redirect" href="/wiki/%EB%8F%84%EB%B0%94_%ED%9B%84%EC%8B%9C%EB%AF%B8_%EC%A0%84%ED%88%AC" title="도바 후시미 전투">도바 후시미 전투</a> 후, 막부군은 오사카 성으로 패주했다. 쇼군 <a href="/wiki/%EB%8F%84%EC%BF%A0%EA%B0%80%EC%99%80_%EC%9A%94%EC%8B%9C%EB%85%B8%EB%B6%80" title="도쿠가와 요시노부">도쿠가와 요시노부</a>는 배로 <a href="/wiki/%EC%97%90%EB%8F%84" title="에도">에도</a>로 도망갔고, 오사카 성은 신정부군에 개성되었다. 하지만, 전후 혼란속에 방화가 일어나 건물의 대부분은 소실되었다.</p>
<h3><span class="mw-headline" id=".EB.A9.94.EC.9D.B4.EC.A7.80_.EC.8B.9C.EB.8C.80_.EC.9D.B4.ED.9B.84">메이지 시대 이후</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%98%A4%EC%82%AC%EC%B9%B4_%EC%84%B1&amp;action=edit&amp;section=4" title="부분 편집: 메이지 시대 이후">편집</a><span class="mw-editsection-bracket">]</span></span></h3>

<p>메이지 신정부는 성내의 부지를 육군의 부지로 활용하고, 민간인의 출입을 금했다. 동쪽에는 국철 조토 선(현재의 <a href="/wiki/%EC%98%A4%EC%82%AC%EC%B9%B4_%EC%88%9C%ED%99%98%EC%84%A0" title="오사카 순환선">오사카 순환선</a>)까지의 거대한 지역에 병기공장을 지었고, 이 때문에 <a href="/wiki/%ED%83%9C%ED%8F%89%EC%96%91_%EC%A0%84%EC%9F%81" title="태평양 전쟁">태평양 전쟁</a>시 미군의 공격목표가 되었다.</p>
<ul>
<li><a href="/wiki/1870%EB%85%84" title="1870년">1870년</a>(<a href="/wiki/%EB%A9%94%EC%9D%B4%EC%A7%80_%EC%8B%9C%EB%8C%80" title="메이지 시대">메이지</a> 3년) 육군은 고호다이<sup class="reference" id="cite_ref-5"><a href="#cite_note-5">[5]</a></sup>를 설치하고, 보시업무를 개시했다.</li>
<li><a href="/wiki/1885%EB%85%84" title="1885년">1885년</a>(메이지 18년) <a href="/wiki/%EC%99%80%EC%B9%B4%EC%95%BC%EB%A7%88_%EC%84%B1" title="와카야마 성">와카야마 성</a> 니노마루로부터 어전 일부를 이축해, 기슈 어전이라 명명했다.</li>
<li><a href="/wiki/1888%EB%85%84" title="1888년">1888년</a>(메이지 21년) 혼마루 사쿠라 문이 복원되었다.</li>
<li><a href="/wiki/1928%EB%85%84" title="1928년">1928년</a>(<a href="/wiki/%EC%87%BC%EC%99%80_%EC%8B%9C%EB%8C%80" title="쇼와 시대">쇼와</a> 3년) 당시 오사카 시장 <a class="new" href="/w/index.php?title=%EC%84%B8%ED%82%A4_%ED%95%98%EC%A7%80%EB%A9%94&amp;action=edit&amp;redlink=1" title="세키 하지메 (없는 문서)">세키 하지메</a>는 천수의 재건을 제안했고, 시민기금을 조성해 육군 제4사단 청사를 이전하고, 천수의 건설을 진행했다.</li>
<li><a href="/wiki/1931%EB%85%84" title="1931년">1931년</a>(쇼와 6년) 철골 콘크리트로 복원 천수각을 준공했다.</li>
</ul>
<p>태평양 전쟁에서는 메이지 유신시에 소실을 면한 건물이 소실되었다. 특히 <a href="/wiki/1945%EB%85%84" title="1945년">1945년</a>(쇼와 20년) 8월 14일 공습에의해 교토역까지 피난민으로 인파가 밀려들었고, 피난민중 다수가 죽거나 부상을 입었다. 이때 마이니치 신문 오사카 본사 옥상에서 촬영한 천수각의 배경은 검은 연기로 뒤덮여 있었고, 그 후, 이 사진은 오사카 여름전투라고 불리게 되었다. 그러나 복원된 천수는 소실을 면했다.</p>
<p>종전 후 성내의 육군요지가 연합군에 접수되었고, <a href="/wiki/1947%EB%85%84" title="1947년">1947년</a>(쇼와 22년) 미군에 의한 소실로 기슈 어전은 소실되었다.</p>
<ul>
<li><a href="/wiki/1948%EB%85%84" title="1948년">1948년</a>(쇼와 23년) 접수해제 후 건물의 수리가 진행되었고, 바깥 해자를 포함 넓은 지역이 오사카조 공원으로 정비되었다.</li>
<li><a href="/wiki/1950%EB%85%84" title="1950년">1950년</a>(쇼와 25년) <a href="/wiki/%ED%83%9C%ED%92%8D" title="태풍">태풍</a> 제인에 의해 손상을 입어, 본격적인 보수작업을 시작한다. 더불어 학술조사도 실시되었다.</li>
<li><a href="/wiki/1959%EB%85%84" title="1959년">1959년</a>(쇼와 34년) 지하에 도요토미씨의 오사카 성의 잔해가 발견되었다. 그리고, 혼마루 내의 육군사단 사령부의 옛 시설은 잠깐 동안 오사카 부 경찰본부의 청사로 사용되어 성벽 일부를 사격훈련장으로 활용했다. 군수공장터에는 오랫동안 방치되어 남겨진 대량의 철과 구리를 노린 아팟치 족이 나타났고, 이것은 <a class="new" href="/w/index.php?title=%EA%B3%A0%EB%A7%88%EC%93%B0_%EC%82%AC%EC%BF%84&amp;action=edit&amp;redlink=1" title="고마쓰 사쿄 (없는 문서)">고마쓰 사쿄</a>와 <a class="new" href="/w/index.php?title=%EA%B0%80%EC%9D%B4%EC%BD%94_%EB%8B%A4%EC%BC%80%EC%8B%9C&amp;action=edit&amp;redlink=1" title="가이코 다케시 (없는 문서)">가이코 다케시</a>의 소설의 무대가 되었다.</li>
<li><a href="/wiki/1983%EB%85%84" title="1983년">1983년</a>(쇼와 58년)〈오사카 축성 400년 마쓰리〉를 겸해 오사카 순환선에 <a class="new" href="/w/index.php?title=%EC%98%A4%EC%82%AC%EC%B9%B4%EC%A1%B0_%EA%B3%B5%EC%9B%90_%EC%97%AD&amp;action=edit&amp;redlink=1" title="오사카조 공원 역 (없는 문서)">오사카조 공원 역</a>이 신설되었고, <a href="/wiki/%EC%98%A4%EC%82%AC%EC%B9%B4_%EC%84%B1_%ED%99%80" title="오사카 성 홀">오사카 성 홀</a>도 개관되었다. 남아 있던 군수공장터에는 하나둘 대기업의 빌딩이 들어섰고, 성 북동쪽에는 오사카 비즈니스 파크가 조성되었다.</li>
<li><a href="/wiki/2006%EB%85%84" title="2006년">2006년</a>(<a href="/wiki/%ED%97%A4%EC%9D%B4%EC%84%B8%EC%9D%B4_%EC%8B%9C%EB%8C%80" title="헤이세이 시대">헤세</a> 18년) 4월 6일 일본 100대 명성에 선정되었다.</li>
</ul>
<h2><span class="mw-headline" id=".EA.B5.AC.EC.A1.B0">구조</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%98%A4%EC%82%AC%EC%B9%B4_%EC%84%B1&amp;action=edit&amp;section=5" title="부분 편집: 구조">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>오사카 성은 도요토미 가문이 축성한 성과 도쿠가와 가문이 재건한 성의 배치가 다르다. 현재 볼 수 있는 건물의 배치는 모두 <a href="/wiki/%EC%97%90%EB%8F%84_%EC%8B%9C%EB%8C%80" title="에도 시대">에도 시대</a>의 것으로 도쿠가와 가문이 축성한 것이다. 단, 해자와 문 등의 위치는 도요토미 가문과 별반 다를 게 없어 보인다.</p>
<p>성의 형식은 윤곽식 평성으로, <a href="/wiki/%ED%98%BC%EB%A7%88%EB%A3%A8" title="혼마루">혼마루</a>를 중심에 두고 외성을 동심원으로 배치했다. 그 사이에 내측 해자와 외측 해자를 두었다. <a href="/wiki/%EB%8F%84%EC%9A%94%ED%86%A0%EB%AF%B8_%ED%9E%88%EB%8D%B0%EC%9A%94%EC%8B%9C" title="도요토미 히데요시">도요토미 히데요시</a>는 오사카 시가지에서 <a class="mw-redirect" href="/wiki/%EB%8D%B4%EC%8A%88" title="덴슈">천수</a>가 잘 보이도록 천수의 위치와 도로의 배열등을 궁리했다고 전해진다.</p>
<p>대지의 북단에 축성한 오사카 성의 방위상의 약점은 대지의 높이가 같은 남쪽이다. 그래서 서에서 남으로 해자를 만들어 성을 감쌌고, 다마쓰쿠리 문의 남쪽에 반월형의 외성 <a href="/wiki/%EC%82%AC%EB%82%98%EB%8B%A4%EB%A7%88%EB%A3%A8" title="사나다마루">사나다마루</a>를 축성했다. 역시나 오사카 성의 남쪽이 오사카 겨울전투의 격전지였다.</p>
<h3><span class="mw-headline" id=".EC.9E.85.EC.A7.80">입지</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%98%A4%EC%82%AC%EC%B9%B4_%EC%84%B1&amp;action=edit&amp;section=6" title="부분 편집: 입지">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>오사카 성은 우에마치 대지의 북단에 위치해 있다. 대지의 북으로는 <a class="mw-redirect" href="/wiki/%EC%9A%94%EB%8F%84%EA%B0%80%EC%99%80_%EA%B0%95" title="요도가와 강">요도가와 강</a>의 본류가 흐르고 있어 천연의 요새이고, 또, 수로로 일본 서부에서 <a href="/wiki/%EA%B5%90%ED%86%A0_%EC%8B%9C" title="교토 시">교토</a> 지방으로 물자를 수송하려면 이 요도가와 강을 통해야 했기 때문에 교통의 요충지이기도 하다.</p>
<p>오사카 성이 위치한 대지의 북단은 북, 동, 서쪽의 대지보다 상대적으로 높았기 때문에 두루 조망할 수 있다. 그리고, 북쪽에서 흐르는 요도가와 강의 지류를 이용해 성 안 해자에 물을 채우기도 했다.</p>
<h3><span class="mw-headline" id=".EB.B3.B5.EC.9B.90_.EC.B2.9C.EC.88.98">복원 천수</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%98%A4%EC%82%AC%EC%B9%B4_%EC%84%B1&amp;action=edit&amp;section=7" title="부분 편집: 복원 천수">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<div class="thumb tright">
<div class="thumbinner" style="width:182px;"><a class="image" href="/wiki/%ED%8C%8C%EC%9D%BC:OsakaCastleM0724.jpg"><img alt="" class="thumbimage" data-file-height="1650" data-file-width="1100" height="270" src="//upload.wikimedia.org/wikipedia/commons/thumb/9/9a/OsakaCastleM0724.jpg/180px-OsakaCastleM0724.jpg" srcset="//upload.wikimedia.org/wikipedia/commons/thumb/9/9a/OsakaCastleM0724.jpg/270px-OsakaCastleM0724.jpg 1.5x, //upload.wikimedia.org/wikipedia/commons/thumb/9/9a/OsakaCastleM0724.jpg/360px-OsakaCastleM0724.jpg 2x" width="180"/></a>
<div class="thumbcaption">
<div class="magnify"><a class="internal" href="/wiki/%ED%8C%8C%EC%9D%BC:OsakaCastleM0724.jpg" title="실제 크기로"></a></div>
도쿠가와의 백색(1~4층)<br/>
도요토미의 흑색(5층)</div>

<p>현재 오사카와 오사카 성의 상징인 천수각은 <a href="/wiki/1928%EB%85%84" title="1928년">1928년</a> 당시 오사카 시장인 <a class="new" href="/w/index.php?title=%EC%84%B8%ED%82%A4_%ED%95%98%EC%A7%80%EB%A9%94&amp;action=edit&amp;redlink=1" title="세키 하지메 (없는 문서)">세키 하지메</a>가 재건할 것을 주창해, <a href="/wiki/1931%EB%85%84" title="1931년">1931년</a> 성을 준공했다. 1930년대 이후 각지에 건설된 복원천수 1호였다.</p>
<p>건물은 도쿠가와 가문의 오사카 성 천수대를 기반으로 철골콘크리트로 건설되었다. 천수각의 안은 박물관으로 사용하고 있다.</p>
<p>외관은 옛 그림을 토대로 새롭게 재탄생되었다. 오사카 성의 천수는 도요토미 가문의 오사카 성과 도쿠가와 가문의 오사카 성과는 건설된 장소도 외관도 전혀 다르지만, 복원된 천수각에서는 1층부터 4층까지 도쿠가와 풍으로 백색인 회벽으로 하는 한편, 5층에서는 도요토미 풍으로 흑색에 금박으로 호랑이와 두루미의 그림을 그려 넣었다. 이에 대해 여러 가지 논의가 있었고, 도요토미 가문의 형식으로 전부 통일해야된다는 주장도 있었다.</p>
<p><a href="/wiki/1995%EB%85%84" title="1995년">1995년</a>부터 <a href="/wiki/1997%EB%85%84" title="1997년">1997년</a>까지 성을 대폭 개수했다. 이 시기, 구조는 <a href="/wiki/%ED%95%9C%EC%8B%A0%C2%B7%EC%95%84%EC%99%80%EC%A7%80_%EB%8C%80%EC%A7%80%EC%A7%84" title="한신·아와지 대지진">한신·아와지 대지진</a>에 해당하는 진도를 견딜 수 있게 보강되었고, 외관의 벽도 다시 칠해졌다. 상처난 처마의 기와와 <a href="/wiki/%EC%83%A4%EC%B9%98%ED%98%B8%EC%BD%94" title="샤치호코">샤치호코</a>, <a class="new" href="/w/index.php?title=%EA%B7%80%EB%A9%B4%EA%B8%B0%EC%99%80&amp;action=edit&amp;redlink=1" title="귀면기와 (없는 문서)">귀면기와</a>의 금박도 다시 입혀졌다. 또, 단체관광객과 신체장애자의 편의를 위해 승강기의 운행구간을 연장했다.</p>
<p>1997년 국가 등록문화재로 등록되었다.</p>
<h2><span class="mw-headline" id=".EB.82.A8.EC.95.84.EC.9E.88.EB.8A.94_.EA.B1.B4.EC.A1.B0.EB.AC.BC">남아있는 건조물</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%98%A4%EC%82%AC%EC%B9%B4_%EC%84%B1&amp;action=edit&amp;section=8" title="부분 편집: 남아있는 건조물">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<div class="thumb tright">
<div class="thumbinner" style="width:182px;"><a class="image" href="/wiki/%ED%8C%8C%EC%9D%BC:InuiYagura.JPG"><img alt="" class="thumbimage" data-file-height="1536" data-file-width="2048" height="135" src="//upload.wikimedia.org/wikipedia/commons/thumb/6/64/InuiYagura.JPG/180px-InuiYagura.JPG" srcset="//upload.wikimedia.org/wikipedia/commons/thumb/6/64/InuiYagura.JPG/270px-InuiYagura.JPG 1.5x, //upload.wikimedia.org/wikipedia/commons/thumb/6/64/InuiYagura.JPG/360px-InuiYagura.JPG 2x" width="180"/></a>
<div class="thumbcaption">
<div class="magnify"><a class="internal" href="/wiki/%ED%8C%8C%EC%9D%BC:InuiYagura.JPG" title="실제 크기로"></a></div>

<p>현재 성내에는 오테 문, 엔쇼구라, 다몬 망루, 센간 망루, 이누이 망루, 이치반 망루, 로쿠반 망루, 가나구라, 사쿠라 문 등이 남아 있고, 국가 중요문화재로 지정되었다.</p>
<p>또, 현존하는 성벽도 많이 남아있다. 에도 시대의 오사카 성은 에도 막부의 <a class="new" href="/w/index.php?title=%EB%8D%B0%EC%93%B0%EB%8B%A4%EC%9D%B4%EB%B6%80%EC%8B%A0&amp;action=edit&amp;redlink=1" title="데쓰다이부신 (없는 문서)">데쓰다이부신</a><sup class="reference" id="cite_ref-6"><a href="#cite_note-6">[6]</a></sup>으로 건설되었기 때문에, 석벽에는 그 돌을 가지고 온 다이묘를 표시하는 각인이 남아있는 것을 확인 할 수 있다. 그리고, 그 돌들은 <a href="/wiki/%EC%84%B8%ED%86%A0_%EB%82%B4%ED%95%B4" title="세토 내해">세토 내해</a>의 여러 섬에서 가지고 온 것들이다.</p>
<p>그리고, 도요토미 가문의 오사카 성은 에도 시대 오사카 성을 축성할 때 파괴되거나 대지를 돋을 때 묻었기 때문에 현재는 볼 수 없다.</p>
<p>단, 무라가와 유키히로(현 오사카 경제법과대학 명예교수 및 고고학자)에 따르면 <a href="/wiki/1950%EB%85%84" title="1950년">1950년</a>대 중후반 즈음에 오사카 성 종합조사에 의해 혼마루의 지하에 도요토미 가문의 석벽이 발견되었고, 현재는 일반에 개방되지 않고 뚜껑으로 구멍을 막아 지하에 보존하고 있다. 또, <a href="/wiki/2003%EB%85%84" title="2003년">2003년</a>에는 <a href="/wiki/%EC%98%A4%ED%85%8C_%EB%AC%B8" title="오테 문">오테 문</a>앞 <a class="mw-redirect" href="/wiki/%EC%82%B0%EB%85%B8%EB%A7%88%EB%A3%A8" title="산노마루">산노마루</a> 해자터의 발굴조사에서 해자 내측 벽에 <a class="mw-redirect" href="/wiki/%EB%B2%99%EC%BB%A4" title="벙커">벙커</a>와 비슷한 형태의 구조가 발견되었다. 이 발견은 해자 자체가 오사카 겨울전투를 즈음하여 급하게 매워졌다는 것을 뒷받침하는 증거이다.</p>
