<h1 class="firstHeading" id="firstHeading" lang="ko">한반도 군사 분계선</h1>

<p>한반도의 <b>군사 분계선</b>(韓半島軍事分界線, <span style="font-size: smaller;"><a href="/wiki/%EC%98%81%EC%96%B4" title="영어">영어</a>:</span> <span lang="en" xml:lang="en">Military Demarcation Line, <b>MDL</b></span>)은 <a href="/wiki/%ED%95%9C%EB%B0%98%EB%8F%84" title="한반도">한반도</a>의 남북을 분단하여 <a href="/wiki/%EB%8C%80%ED%95%9C%EB%AF%BC%EA%B5%AD" title="대한민국">대한민국</a>과 <a href="/wiki/%EC%A1%B0%EC%84%A0%EB%AF%BC%EC%A3%BC%EC%A3%BC%EC%9D%98%EC%9D%B8%EB%AF%BC%EA%B3%B5%ED%99%94%EA%B5%AD" title="조선민주주의인민공화국">조선민주주의인민공화국</a>의 경계를 이루는 <a href="/wiki/%EC%A7%80%EB%8F%84" title="지도">지도</a>상의 선이다.</p>
<p><a href="/wiki/1953%EB%85%84" title="1953년">1953년</a> <a href="/wiki/7%EC%9B%94_27%EC%9D%BC" title="7월 27일">7월 27일</a> 발효된 <a href="/wiki/%ED%95%9C%EA%B5%AD_%EC%A0%84%EC%9F%81" title="한국 전쟁">한국 전쟁</a>의 <a class="mw-redirect" href="/wiki/%EC%A0%95%EC%A0%84_%ED%98%91%EC%A0%95" title="정전 협정">정전 협정</a> 체결 당시 <a href="/wiki/%EC%9E%84%EC%A7%84%EA%B0%95" title="임진강">임진강</a>에서 <a class="mw-redirect" href="/wiki/%EB%8F%99%ED%95%B4%EC%95%88" title="동해안">동해안</a>까지 총 1,292개의 말뚝을 박고, 이 말뚝을 이은 가상의 선을 군사분계선으로 설정하였다.<sup class="reference" id="cite_ref-1"><a href="#cite_note-1">[1]</a></sup> 군사분계선에서부터 남북으로 각각 2 km 범위에는 군사충돌을 방지하기 위한 완충지대인 <a class="mw-redirect" href="/wiki/%ED%95%9C%EB%B0%98%EB%8F%84_%EB%B9%84%EB%AC%B4%EC%9E%A5%EC%A7%80%EB%8C%80" title="한반도 비무장지대">비무장지대</a>(DMZ)가 설정되어 있다.</p>

<h2><span class="mw-headline" id=".EC.98.81.EC.97.AD">영역</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%ED%95%9C%EB%B0%98%EB%8F%84_%EA%B5%B0%EC%82%AC_%EB%B6%84%EA%B3%84%EC%84%A0&amp;action=edit&amp;section=1" title="부분 편집: 영역">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<div class="thumb tright">
<div class="thumbinner" style="width:222px;"><a class="image" href="/wiki/%ED%8C%8C%EC%9D%BC:Korean_dmz_map.png"><img alt="" class="thumbimage" data-file-height="346" data-file-width="450" height="169" src="//upload.wikimedia.org/wikipedia/commons/thumb/9/97/Korean_dmz_map.png/220px-Korean_dmz_map.png" srcset="//upload.wikimedia.org/wikipedia/commons/thumb/9/97/Korean_dmz_map.png/330px-Korean_dmz_map.png 1.5x, //upload.wikimedia.org/wikipedia/commons/thumb/9/97/Korean_dmz_map.png/440px-Korean_dmz_map.png 2x" width="220"/></a>
<div class="thumbcaption">
<div class="magnify"><a class="internal" href="/wiki/%ED%8C%8C%EC%9D%BC:Korean_dmz_map.png" title="실제 크기로"></a></div>
검은 선이 군사분계선(MDL)이고, 붉은 색으로 표시된 지역이 비무장지대(DMZ)이다.</div>
</div>
</div>
<p>군사분계선(MDL)을 기준으로 하여, 북쪽으로 2 km 떨어진 비무장지대의 경계선을 "북방한계선"(NLL)이라 하고, 남쪽으로 2 km 떨어진 비무장지대의 경계선을 "남방한계선"(SLL)이라고 한다. 북방한계선과 남방한계선에는 철책이 쳐져 있으며, 남북의 군대가 대치하고 있다.</p>
<p><a href="/wiki/1963%EB%85%84" title="1963년">1963년</a>부터 북측이 요새와 진지, 철책을 구축하면서 <a href="/wiki/%ED%95%9C%EB%B0%98%EB%8F%84" title="한반도">한반도</a>의 비무장지대는 <a href="/wiki/%EB%B9%84%EB%AC%B4%EC%9E%A5_%EC%A7%80%EB%8C%80" title="비무장 지대">비무장 지대</a>(非武裝地帶)라는 말이 무색할 정도로 남북 양측의 무장 군인들이 상시 주둔하는 <a href="/wiki/%ED%9C%B4%EC%A0%84%EC%84%A0_%EA%B0%90%EC%8B%9C_%EC%B4%88%EC%86%8C" title="휴전선 감시 초소">전방 감시 초소</a>(GP)가 곳곳에 있으며, 일부 GP와 GP 사이에는 '추진철책'이라는 이름의 철책이 설치되어 있다.<sup class="reference" id="cite_ref-2"><a href="#cite_note-2">[2]</a></sup> 북측은 '민경대'(民警隊), 남측은 '민정경찰'(DMZ Police)이라고 자처하지만, 실질은 양쪽 모두 무장한 <a href="/wiki/%EA%B5%B0%EC%9D%B8" title="군인">군인</a>들이다.</p>
<p>비무장 지대 내의 민간인 거주 마을로는 남측의 <a href="/wiki/%EC%9E%90%EC%9C%A0%EC%9D%98_%EB%A7%88%EC%9D%84" title="자유의 마을">대성동</a>, 북측의 <a href="/wiki/%EA%B8%B0%EC%A0%95%EB%8F%99" title="기정동">기정동</a>이 있다.</p>
<h2><span class="mw-headline" id=".EC.82.BC.ED.8C.94.EC.84.A0.EA.B3.BC_.ED.95.9C.EB.B0.98.EB.8F.84.EC.9D.98_.EB.B6.84.ED.95.A0">삼팔선과 한반도의 분할</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%ED%95%9C%EB%B0%98%EB%8F%84_%EA%B5%B0%EC%82%AC_%EB%B6%84%EA%B3%84%EC%84%A0&amp;action=edit&amp;section=2" title="부분 편집: 삼팔선과 한반도의 분할">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p><a class="mw-redirect" href="/wiki/%EC%A0%9C2%EC%B0%A8_%EC%84%B8%EA%B3%84%EB%8C%80%EC%A0%84" title="제2차 세계대전">제2차 세계대전</a>의 결과 <a href="/wiki/1945%EB%85%84" title="1945년">1945년</a> <a href="/wiki/8%EC%9B%94_15%EC%9D%BC" title="8월 15일">8월 15일</a> <a href="/wiki/%EC%9D%BC%EB%B3%B8_%EC%A0%9C%EA%B5%AD" title="일본 제국">일본 제국</a>이 패망하자, <a href="/wiki/%EB%AF%B8%EA%B5%AD" title="미국">미국</a>은 이미 <a class="mw-redirect" href="/wiki/8%EC%9B%94_%ED%8F%AD%ED%92%8D%EC%9E%91%EC%A0%84" title="8월 폭풍작전">한반도 북부에 진입</a>한 소련군의 남진에 대응해 미소 양측의 군사적 충돌 없이 <a class="mw-redirect" href="/wiki/%EC%84%9C%EC%9A%B8" title="서울">서울</a>과 <a class="mw-redirect" href="/wiki/%EC%9D%B8%EC%B2%9C" title="인천">인천</a>을 포함한 한반도 남반부를 점령하기 위하여 <a href="/wiki/%EC%86%8C%EB%A0%A8" title="소련">소련</a>에 <a class="mw-redirect" href="/wiki/38%EC%84%A0" title="38선">38선</a>을 군사분계선으로 하는 안을 제안하였다. <a href="/wiki/%EC%86%8C%EB%A0%A8" title="소련">소련</a>이 별 이의 없이 <a class="mw-redirect" href="/wiki/38%EC%84%A0" title="38선">38선</a> 분할 점령안을 받아들임으로써 <a href="/wiki/1945%EB%85%84" title="1945년">1945년</a> <a href="/wiki/9%EC%9B%94_2%EC%9D%BC" title="9월 2일">9월 2일</a> <a href="/wiki/%ED%95%9C%EB%B0%98%EB%8F%84" title="한반도">한반도</a>는 남북으로 <a class="mw-redirect" href="/wiki/%ED%95%9C%EB%B0%98%EB%8F%84%EC%9D%98_%EB%B6%84%EB%8B%A8" title="한반도의 분단">분단</a>되었다.</p>

<h2><span class="mw-headline" id=".EC.A0.95.EC.A0.84_.ED.98.91.EC.A0.95.EC.97.90_.EB.94.B0.EB.A5.B8_.EA.B5.B0.EC.82.AC.EB.B6.84.EA.B3.84.EC.84.A0">정전 협정에 따른 군사분계선</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%ED%95%9C%EB%B0%98%EB%8F%84_%EA%B5%B0%EC%82%AC_%EB%B6%84%EA%B3%84%EC%84%A0&amp;action=edit&amp;section=3" title="부분 편집: 정전 협정에 따른 군사분계선">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p><a href="/wiki/1950%EB%85%84" title="1950년">1950년</a> <a href="/wiki/6%EC%9B%94_25%EC%9D%BC" title="6월 25일">6월 25일</a> 발발한 <a class="mw-redirect" href="/wiki/%ED%95%9C%EA%B5%AD%EC%A0%84%EC%9F%81" title="한국전쟁">한국전쟁</a>은 3년1개월 만인 <a href="/wiki/1953%EB%85%84" title="1953년">1953년</a> <a href="/wiki/7%EC%9B%94_27%EC%9D%BC" title="7월 27일">7월 27일</a> 정전이 이루어졌다. 한국 전쟁의 <a class="mw-redirect" href="/wiki/%EC%A0%95%EC%A0%84_%ED%98%91%EC%A0%95" title="정전 협정">정전 협정</a>에 따라 설정된 <b>군사분계선</b>은 위도상 <a href="/wiki/%EB%B6%81%EC%9C%84_38%EB%8F%84" title="북위 38도">북위 38도</a> 부근에 위치하고 있지만, <a class="mw-redirect" href="/wiki/38%EC%84%A0" title="38선">38선</a>과 비교해 서쪽 경계가 남하하였고 동쪽 경계가 북상하였다.</p>
<h3><span class="mw-headline" id=".EC.88.98.EB.B3.B5.EC.A7.80.EA.B5.AC">수복지구</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%ED%95%9C%EB%B0%98%EB%8F%84_%EA%B5%B0%EC%82%AC_%EB%B6%84%EA%B3%84%EC%84%A0&amp;action=edit&amp;section=4" title="부분 편집: 수복지구">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<div class="rellink relarticle mainarticle"><img alt="" data-file-height="128" data-file-width="128" height="16" src="//upload.wikimedia.org/wikipedia/commons/thumb/e/ec/Crystal_Clear_app_xmag.svg/16px-Crystal_Clear_app_xmag.svg.png" srcset="//upload.wikimedia.org/wikipedia/commons/thumb/e/ec/Crystal_Clear_app_xmag.svg/24px-Crystal_Clear_app_xmag.svg.png 1.5x, //upload.wikimedia.org/wikipedia/commons/thumb/e/ec/Crystal_Clear_app_xmag.svg/32px-Crystal_Clear_app_xmag.svg.png 2x" width="16"/> 이 부분의 본문은 <a href="/wiki/%EC%88%98%EB%B3%B5%EC%A7%80%EA%B5%AC" title="수복지구">수복지구</a>입니다.</div>
<p><b>수복지구</b>(收復地區)는 <a href="/wiki/%EB%B6%81%EC%9C%84_38%EB%8F%84" title="북위 38도">북위 38도</a> 이북 중 <a class="mw-redirect" href="/wiki/%ED%95%9C%EA%B5%AD%EC%A0%84%EC%9F%81" title="한국전쟁">한국전쟁</a>의 정전협정에 따라 <a href="/wiki/%EB%8C%80%ED%95%9C%EB%AF%BC%EA%B5%AD" title="대한민국">대한민국</a>에 편입된 <a href="/wiki/%EA%B5%B0%EC%82%AC%EB%B6%84%EA%B3%84%EC%84%A0" title="군사분계선">군사분계선</a> 이남 지역을 말한다. <a class="mw-redirect" href="/wiki/38%EC%84%A0" title="38선">38선</a> 동북쪽의 <a href="/wiki/%EA%B0%95%EC%9B%90%EB%8F%84_(%EB%82%A8)" title="강원도 (남)">강원도</a> <a href="/wiki/%EC%B2%A0%EC%9B%90%EA%B5%B0_(%EB%82%A8)" title="철원군 (남)">철원군</a>·<a href="/wiki/%EA%B9%80%ED%99%94%EA%B5%B0" title="김화군">김화군</a> 일부(<a href="/wiki/1963%EB%85%84" title="1963년">1963년</a> <a href="/wiki/%EC%B2%A0%EC%9B%90%EA%B5%B0_(%EB%82%A8)" title="철원군 (남)">철원군</a>에 편입)·<a href="/wiki/%ED%99%94%EC%B2%9C%EA%B5%B0" title="화천군">화천군</a>·<a href="/wiki/%EC%96%91%EA%B5%AC%EA%B5%B0" title="양구군">양구군</a>·<a href="/wiki/%EC%9D%B8%EC%A0%9C%EA%B5%B0" title="인제군">인제군</a>·<a href="/wiki/%EC%96%91%EC%96%91%EA%B5%B0" title="양양군">양양군</a>·<a href="/wiki/%EA%B3%A0%EC%84%B1%EA%B5%B0_(%EB%82%A8)" title="고성군 (남)">고성군</a>과 <a href="/wiki/%EA%B2%BD%EA%B8%B0%EB%8F%84" title="경기도">경기도</a> <a href="/wiki/%EC%97%B0%EC%B2%9C%EA%B5%B0" title="연천군">연천군</a>이 이에 해당한다. 휴전 이듬해인 <a href="/wiki/1954%EB%85%84" title="1954년">1954년</a> 수복지구의 행정권이 <a class="mw-redirect" href="/wiki/UN%EA%B5%B0" title="UN군">UN군</a>에서 <a href="/wiki/%EB%8C%80%ED%95%9C%EB%AF%BC%EA%B5%AD_%EC%A0%95%EB%B6%80" title="대한민국 정부">대한민국 정부</a>로 이양되어 이들 군을 정식으로 재설치하였다.</p>

<p><b>신해방지구</b>(新解放地區)는 <a href="/wiki/%EB%B6%81%EC%9C%84_38%EB%8F%84" title="북위 38도">북위 38도</a> 이남 중 <a class="mw-redirect" href="/wiki/%ED%95%9C%EA%B5%AD%EC%A0%84%EC%9F%81" title="한국전쟁">한국전쟁</a>의 정전협정에 따라 <a href="/wiki/%EC%A1%B0%EC%84%A0%EB%AF%BC%EC%A3%BC%EC%A3%BC%EC%9D%98%EC%9D%B8%EB%AF%BC%EA%B3%B5%ED%99%94%EA%B5%AD" title="조선민주주의인민공화국">조선민주주의인민공화국</a>에 편입된 <a href="/wiki/%EA%B5%B0%EC%82%AC%EB%B6%84%EA%B3%84%EC%84%A0" title="군사분계선">군사분계선</a> 이북 지역을 말한다. <a href="/wiki/%EC%84%9C%ED%95%B4_5%EB%8F%84" title="서해 5도">서해 5도</a>를 제외한 <a class="mw-redirect" href="/wiki/38%EC%84%A0" title="38선">38선</a> 이남의 <a href="/wiki/%ED%99%A9%ED%95%B4%EB%8F%84" title="황해도">황해도</a> <a href="/wiki/%EC%98%B9%EC%A7%84%EA%B5%B0" title="옹진군">옹진군</a>·<a href="/wiki/%EC%97%B0%EB%B0%B1%EA%B5%B0" title="연백군">연백군</a>과 <a href="/wiki/%EA%B2%BD%EA%B8%B0%EB%8F%84" title="경기도">경기도</a> <a class="mw-redirect" href="/wiki/%EA%B0%9C%EC%84%B1%EC%8B%9C" title="개성시">개성시</a>·<a href="/wiki/%EA%B0%9C%ED%92%8D%EA%B5%B0" title="개풍군">개풍군</a> 및 <a href="/wiki/%EC%9E%A5%EB%8B%A8%EA%B5%B0" title="장단군">장단군</a>의 대부분이 군사분계선 이북 지역이 되었다.</p>

<p>정전협정 당시 육상의 군사분계선은 합의되었으나,<sup class="reference" id="cite_ref-3"><a href="#cite_note-3">[3]</a></sup> 해상 경계선에 관하여는 연안수역의 범위를 둘러싸고 3해리를 주장한 유엔군 사령부와 12해리를 주장한 <a href="/wiki/%EC%A1%B0%EC%84%A0%EB%AF%BC%EC%A3%BC%EC%A3%BC%EC%9D%98%EC%9D%B8%EB%AF%BC%EA%B3%B5%ED%99%94%EA%B5%AD" title="조선민주주의인민공화국">조선민주주의인민공화국</a>의 입장차이 때문에<sup class="reference" id="cite_ref-4"><a href="#cite_note-4">[4]</a></sup> 명확한 합의 없이 "연해의 섬 및 해면에 관한 통제권은 <a href="/wiki/1950%EB%85%84" title="1950년">1950년</a> <a href="/wiki/6%EC%9B%94_24%EC%9D%BC" title="6월 24일">6월 24일</a> 이전을 기준으로 하되, <a class="mw-redirect" href="/wiki/%EC%84%9C%ED%95%B45%EB%8F%84" title="서해5도">서해5도</a>(<a href="/wiki/%EB%B0%B1%EB%A0%B9%EB%8F%84" title="백령도">백령도</a>·<a href="/wiki/%EB%8C%80%EC%B2%AD%EB%8F%84" title="대청도">대청도</a>·<a href="/wiki/%EC%86%8C%EC%B2%AD%EB%8F%84" title="소청도">소청도</a>·<a href="/wiki/%EC%97%B0%ED%8F%89%EB%8F%84" title="연평도">연평도</a>·<a href="/wiki/%EC%9A%B0%EB%8F%84_(%EA%B0%95%ED%99%94%EA%B5%B0)" title="우도 (강화군)">우도</a>)는 UN군 사령관 관할 아래 둔다."는 단서규정만을 두었다.</p>
<h2><span class="mw-headline" id=".EB.AF.BC.EA.B0.84.EC.9D.B8.EC.B6.9C.EC.9E.85.ED.86.B5.EC.A0.9C.EA.B5.AC.EC.97.AD">민간인출입통제구역</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%ED%95%9C%EB%B0%98%EB%8F%84_%EA%B5%B0%EC%82%AC_%EB%B6%84%EA%B3%84%EC%84%A0&amp;action=edit&amp;section=7" title="부분 편집: 민간인출입통제구역">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<div class="rellink relarticle mainarticle"><img alt="" data-file-height="128" data-file-width="128" height="16" src="//upload.wikimedia.org/wikipedia/commons/thumb/e/ec/Crystal_Clear_app_xmag.svg/16px-Crystal_Clear_app_xmag.svg.png" srcset="//upload.wikimedia.org/wikipedia/commons/thumb/e/ec/Crystal_Clear_app_xmag.svg/24px-Crystal_Clear_app_xmag.svg.png 1.5x, //upload.wikimedia.org/wikipedia/commons/thumb/e/ec/Crystal_Clear_app_xmag.svg/32px-Crystal_Clear_app_xmag.svg.png 2x" width="16"/> 이 부분의 본문은 <a href="/wiki/%EB%AF%BC%EA%B0%84%EC%9D%B8%EC%B6%9C%EC%9E%85%ED%86%B5%EC%A0%9C%EA%B5%AC%EC%97%AD" title="민간인출입통제구역">민간인출입통제구역</a>입니다.</div>

<p><b>민간인출입통제구역</b>(民間人出入統制區域, <span style="font-size: smaller;"><a href="/wiki/%EC%98%81%EC%96%B4" title="영어">영어</a>:</span> <span lang="en" xml:lang="en">Civilian Control Zone, <b>CCZ</b></span>)은 군사분계선 인근의 군사 작전 및 군사시설 보호와 보안유지 등의 목적으로 민간인의 출입을 제한하는 구역이다. <a class="mw-redirect" href="/wiki/%EB%B9%84%EB%AC%B4%EC%9E%A5%EC%A7%80%EB%8C%80" title="비무장지대">비무장지대</a>(DMZ)의 남방한계선 남쪽 5~10 km<sup class="reference" id="cite_ref-5"><a href="#cite_note-5">[5]</a></sup> 에 걸쳐 있는 민간인의 출입이 제한된 지역으로, 이 지역의 경계선은 <b>민통선</b>(민간인 출입통제선, <span style="font-size: smaller;"><a href="/wiki/%EC%98%81%EC%96%B4" title="영어">영어</a>:</span> <span lang="en" xml:lang="en">Civilian Control Line, <b>CCL</b></span>)이라고 부른다.</p>
<p>민통선은 <a href="/wiki/1954%EB%85%84" title="1954년">1954년</a> 2월 <a class="mw-redirect" href="/wiki/%EB%AF%B88%EA%B5%B0" title="미8군">미8군</a>이 군사시설 보안 등을 목적으로 민간인의 경작행위를 규제하기 위해 설정한 귀농선(歸農線)에서 시작되었다. 민간인출입통제구역 내에서는 군사 작전 및 보안 유지에 지장이 없는 범위에서 민간인의 영농을 위한 토지 이용이 허용되지만, 경작권을 제외한 토지소유권의 행사, 지역 내의 출입과 행동 등 국민의 자유와 기본권이 국가안보상의 필요에 의해 통제되고 있다.</p>
