<h2><span class="mw-headline" id=".EC.97.AD.EC.82.AC">역사</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%95%88%ED%8B%B0%EC%98%A4%ED%82%A4%EC%95%84_%EA%B3%B5%EA%B5%AD&amp;action=edit&amp;section=1" title="부분 편집: 역사">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<h3><span class="mw-headline" id=".EA.B3.B5.EA.B5.AD.EC.9D.98_.EC.84.B1.EB.A6.BD.EA.B3.BC_.ED.99.95.EB.8C.80">공국의 성립과 확대</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%95%88%ED%8B%B0%EC%98%A4%ED%82%A4%EC%95%84_%EA%B3%B5%EA%B5%AD&amp;action=edit&amp;section=2" title="부분 편집: 공국의 성립과 확대">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p><a href="/wiki/%EC%A0%9C1%EC%B0%A8_%EC%8B%AD%EC%9E%90%EA%B5%B0" title="제1차 십자군">제1차 십자군</a> 원정 이전 <a class="mw-redirect" href="/wiki/%EC%95%88%ED%8B%B0%EC%98%A4%ED%82%A4%EC%95%84" title="안티오키아">안티오키아</a>는 <a href="/wiki/%EC%85%80%EC%A3%BC%ED%81%AC_%EC%99%95%EC%A1%B0" title="셀주크 왕조">셀주크 왕조</a>의 총독 야기 산이 다스리던 400여개의 탑과 성벽으로 이루어진 난공불락의 성채도시였다(다만 셀주크 왕조 정복이전부터 그리스인 주민도 많아 <a class="mw-redirect" href="/wiki/%EB%B9%84%EC%9E%94%ED%8B%B4_%EC%A0%9C%EA%B5%AD" title="비잔틴 제국">비잔틴 제국</a>은 이곳을 제국 고유의 영토라고 생각해 셀주크 왕조로부터 회복하기를 고대했다). <a href="/wiki/%EC%A0%9C1%EC%B0%A8_%EC%8B%AD%EC%9E%90%EA%B5%B0" title="제1차 십자군">제1차 십자군</a> 원정 시 <a class="mw-disambig" href="/wiki/%EB%B3%B4%EB%91%90%EC%95%B5_1%EC%84%B8" title="보두앵 1세">보두앵 1세</a>(후에 예루살렘의 왕이됨)와 <a class="mw-redirect" href="/wiki/%EC%97%90%EB%8D%B0%EC%82%AC%EC%9D%98_%EC%A1%B0%EC%8A%AC%EB%9E%AD_1%EC%84%B8" title="에데사의 조슬랭 1세">에데사의 조슬랭 1세</a>가 십자군 본대에서 떨어져 나와 동쪽으로 진격하여 <a href="/wiki/%EC%97%90%EB%8D%B0%EC%82%AC" title="에데사">에데사</a>를 점령하는 동안 십자군 본대는 남쪽으로 진격하여 <a class="mw-redirect" href="/wiki/%EC%95%88%ED%8B%B0%EC%98%A4%ED%82%A4%EC%95%84" title="안티오키아">안티오키아</a>를 공격하였다.</p>
<p>안티오키아 공략에 활약했던 남 이탈리아의 노르만인 봉건영주 <a class="mw-redirect" href="/wiki/%ED%83%80%EB%9E%80%ED%86%A0%EC%9D%98_%EB%B3%B4%EC%97%90%EB%AA%BD" title="타란토의 보에몽">타란토의 보에몽</a>은 공략중 이 도시의 소유를 공언하여 <a href="/wiki/1098%EB%85%84" title="1098년">1098년</a> <a href="/wiki/6%EC%9B%94_3%EC%9D%BC" title="6월 3일">6월 3일</a> 반년간의 포위전 끝에 안티오키아를 점령하는 데 성공했다. 하지만 오랜 공성전에 지친 십자군은 미친 듯이 안티오키아 성내의 <a href="/wiki/%EB%AC%B4%EC%8A%AC%EB%A6%BC" title="무슬림">무슬림</a>들을 학살하였다. 보에몽은 안티오키아를 수도로 삼아 안티오키아 공국을 건설하고, <a href="/wiki/%EC%8B%AD%EC%9E%90%EA%B5%B0" title="십자군">십자군</a>의 본래 목적이던 예루살렘 공략에는 참가하지 않고, 안티오키아 공국의 확립에 전념했다.</p>
<p>함락된 지 4일 후 <a href="/wiki/%EB%AA%A8%EC%88%A0" title="모술">모술</a>의 이슬람군이 안티오키아에 도착해 다시 안티오키아를 탈환하기 위한 공성전을 벌였고 비잔티움 황제 <a class="mw-redirect" href="/wiki/%EC%95%8C%EB%A0%89%EC%8B%9C%EC%9A%B0%EC%8A%A4_1%EC%84%B8" title="알렉시우스 1세">알렉시우스 1세</a>는 안티오키아의 십자군을 도우러 군대를 이끌고 원정나섰다가 이미 이슬람군에 함락되었다는 오보를 듣고 회군하게 된다.</p>
<p>안티오키아의 십자군은 홀로 외롭게 이슬람군의 줄기찬 공격을 힘겹게 막아내고 있었는데 승리는 이상하게 찾아왔다. <a href="/wiki/%EC%9D%80%EC%9E%90_%ED%94%BC%EC%97%90%EB%A5%B4" title="은자 피에르">은자 피에르</a>가 <a class="mw-redirect" href="/wiki/%EB%A1%B1%EA%B8%B0%EB%88%84%EC%8A%A4%EC%9D%98_%EC%B0%BD" title="롱기누스의 창">롱기누스의 창</a>을 발견했다고 주장한 것이다. 십자군은 고무되었고, 승리했다. 그런데 이슬람군이 너무 쉽게 퇴각하는 것이다. 사실 승리의 이유는 따로 있었다. 이슬람 원군의 사령관 카르부카는 엄청난 독재자라, 무슬림들도 그를 싫어했다. 그래서 무슬림들은 전투 때 카르부카를 십자군 앞에 던져놓고 돌아가 버렸다. 이것이 안티오키아 승리의 이유였다.</p>
<p>이슬람군의 공격을 물리치자 이번엔 안티오키아의 지배권을 놓고 십자군 사이에 내분이 일어났는데 보에몽의 <a href="/wiki/%EC%9D%B4%ED%83%88%EB%A6%AC%EC%95%84" title="이탈리아">이탈리아</a> 출신 <a class="mw-redirect" href="/wiki/%EB%85%B8%EB%A5%B4%EB%A7%8C%EC%A1%B1" title="노르만족">노르만족</a>이 결국 지배권을 가지게 되었다. 보에몽은 이미 <a href="/wiki/%EC%BD%98%EC%8A%A4%ED%83%84%ED%8B%B0%EB%85%B8%ED%8F%B4%EB%A6%AC%EC%8A%A4" title="콘스탄티노폴리스">콘스탄티노폴리스</a>에서 비잔티움의 <a href="/wiki/%ED%99%A9%EC%A0%9C" title="황제">황제</a>에게 충성을 서약하였지만 서약을 깨고 다른 십자군 경쟁자를 물리친 후 독자적으로 <a href="/wiki/%EA%B3%B5%EC%9E%91_(%EC%9E%91%EC%9C%84)" title="공작 (작위)">공작</a>의 지위에 올랐고 이로써 안티오키아 공국이 성립하게 되었다.</p>
<h3><span class="mw-headline" id=".EC.B4.88.EA.B8.B0.EC.9D.98_.EC.97.AD.EC.82.AC">초기의 역사</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%95%88%ED%8B%B0%EC%98%A4%ED%82%A4%EC%95%84_%EA%B3%B5%EA%B5%AD&amp;action=edit&amp;section=3" title="부분 편집: 초기의 역사">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p><a href="/wiki/1100%EB%85%84" title="1100년">1100년</a> 보에몽은 <a href="/wiki/%EB%8B%A4%EB%8B%88%EC%8A%88%EB%A9%98%EB%93%9C" title="다니슈멘드">다니슈멘드</a>의 이슬람군과 싸우다가 다시 한번 포로가 되었고, 보에몽이 없는 대신 그의 조카 <a class="mw-redirect" href="/wiki/%ED%83%95%ED%81%AC%EB%A0%88%EB%93%9C_(%EC%8B%AD%EC%9E%90%EA%B5%B0)" title="탕크레드 (십자군)">탕크레드</a>가 공국의 섭정을 맡게 되었다. <a class="mw-redirect" href="/wiki/%ED%83%95%ED%81%AC%EB%A0%88%EB%93%9C_(%EC%8B%AD%EC%9E%90%EA%B5%B0)" title="탕크레드 (십자군)">탕크레드</a>는 비잔티움 제국의 타르수스와 라타키아를 공격하여 공국의 영토를 넓혔다. 1103년 <a class="mw-redirect" href="/wiki/%EB%B3%B4%EC%97%90%EB%AA%BD" title="보에몽">보에몽</a>은 풀려났으나 <a class="mw-redirect" href="/wiki/%ED%83%95%ED%81%AC%EB%A0%88%EB%93%9C_(%EC%8B%AD%EC%9E%90%EA%B5%B0)" title="탕크레드 (십자군)">탕크레드</a>를 계속 공국의 섭정으로 앉히고 이탈리아로 돌아가 군사를 모으고 1107년 비잔티움 제국을 침공하였다. 1108년 비잔티움 제국령인 두라초(현재의 <a href="/wiki/%EC%95%8C%EB%B0%94%EB%8B%88%EC%95%84" title="알바니아">알바니아</a> <a href="/wiki/%EB%91%90%EB%9F%AC%EC%8A%A4" title="두러스">두러스</a>)에서 벌어진 전투에서 패하고 황제 알렉시오스 1세와 강화를 맺었는데 십자군 원정을 떠날 때 콘스탄티노폴리스에서 황제에게 맹세했던 충성 서약을 다시 한번 확인하고 보에몽이 죽으면 안티오키아 공국을 비잔티움 제국에 반환한다는 조건을 내걸었다. 보에몽은 다시 돌아와 <a href="/wiki/%EC%97%90%EB%8D%B0%EC%82%AC_%EB%B0%B1%EA%B5%AD" title="에데사 백국">에데사 백국</a>의 <a class="mw-disambig" href="/wiki/%EB%B3%B4%EB%91%90%EC%95%B5_2%EC%84%B8" title="보두앵 2세">보두앵 2세</a>와 <a class="mw-redirect" href="/wiki/%EC%97%90%EB%8D%B0%EC%82%AC%EC%9D%98_%EC%A1%B0%EC%8A%AC%EB%9E%AD_1%EC%84%B8" title="에데사의 조슬랭 1세">에데사의 조슬랭 1세</a>과 <a href="/wiki/%EC%95%8C%EB%A0%88%ED%8F%AC" title="알레포">알레포</a>에서 싸웠으나 보두앵과 조슬랭은 또다시 이슬람군의 포로가 되고 <a class="mw-disambig" href="/wiki/%ED%83%95%ED%81%AC%EB%A0%88%EB%93%9C" title="탕크레드">탕크레드</a>는 에데사 백국의 섭정을 겸하게 되었다. 보에몽은 1111년 이탈리아로 돌아가 거기서 죽었다.</p>
<p>보에몽이 죽자 황제 알렉시우스 1세는 <a class="mw-disambig" href="/wiki/%ED%83%95%ED%81%AC%EB%A0%88%EB%93%9C" title="탕크레드">탕크레드</a>에게 안티오키아 공국을 반환하라고 요구했지만 <a class="mw-disambig" href="/wiki/%ED%83%95%ED%81%AC%EB%A0%88%EB%93%9C" title="탕크레드">탕크레드</a>는 <a href="/wiki/%ED%8A%B8%EB%A6%AC%ED%8F%B4%EB%A6%AC_%EB%B0%B1%EA%B5%AD" title="트리폴리 백국">트리폴리 백국</a>, <a href="/wiki/%EC%98%88%EB%A3%A8%EC%82%B4%EB%A0%98_%EC%99%95%EA%B5%AD" title="예루살렘 왕국">예루살렘 왕국</a> 등 이웃 <a href="/wiki/%EC%8B%AD%EC%9E%90%EA%B5%B0_%EA%B5%AD%EA%B0%80" title="십자군 국가">십자군 국가</a>들의 지원을 받아 이를 거절했다. <a class="mw-disambig" href="/wiki/%ED%83%95%ED%81%AC%EB%A0%88%EB%93%9C" title="탕크레드">탕크레드</a>는 1112년 죽었고 그 뒤를 이어 <a class="mw-disambig" href="/wiki/%ED%83%95%ED%81%AC%EB%A0%88%EB%93%9C" title="탕크레드">탕크레드</a>의 조카 <a class="new" href="/w/index.php?title=%EC%82%B4%EB%A0%88%EB%A5%B4%EB%AA%A8%EC%9D%98_%EB%A3%A8%EC%A7%80%EC%97%90%EB%A1%9C&amp;action=edit&amp;redlink=1" title="살레르모의 루지에로 (없는 문서)">살레르모의 루지에로</a>의 섭정하에 <a href="/wiki/%EB%B3%B4%EC%97%90%EB%AA%BD_2%EC%84%B8" title="보에몽 2세">보에몽 2세</a>가 공국을 받았다.</p>
<p>루지에로는 1119년 이슬람과의 전투에서 죽었고 공국은 다시 <a class="mw-redirect" href="/wiki/%EC%98%88%EB%A3%A8%EC%82%B4%EB%A0%98%EC%9D%98_%EB%B3%B4%EB%91%90%EC%95%B5_2%EC%84%B8" title="예루살렘의 보두앵 2세">예루살렘의 보두앵 2세</a>의 섭정으로 넘어갔다가 다시 보두앵의 어린 딸 <a class="new" href="/w/index.php?title=%EC%95%88%ED%8B%B0%EC%98%A4%ED%82%A4%EC%95%84%EC%9D%98_%EC%BD%A9%EC%8A%A4%ED%83%95%EC%8A%A4&amp;action=edit&amp;redlink=1" title="안티오키아의 콩스탕스 (없는 문서)">안티오키아의 콩스탕스</a>와 그의 남편 <a href="/wiki/%EC%95%88%ED%8B%B0%EC%98%A4%ED%82%A4%EC%95%84%EC%9D%98_%EB%A0%88%EB%AA%BD" title="안티오키아의 레몽">안티오키아의 레몽</a>에게로 상속되었다. 레몽은 비잔티움 제국의 <a href="/wiki/%ED%82%AC%EB%A6%AC%ED%82%A4%EC%95%84" title="킬리키아">킬리키아</a>를 공격했으나 비잔티움 황제 <a href="/wiki/%EC%9A%94%ED%95%9C%EB%84%A4%EC%8A%A4_2%EC%84%B8" title="요한네스 2세">요한네스 2세</a>의 반격을 받았고 요한네스는 <a href="/wiki/1138%EB%85%84" title="1138년">1138년</a> 안티오키아까지 진격했다. 그러나 <a class="new" href="/w/index.php?title=%EC%97%90%EB%8D%B0%EC%82%AC%EC%9D%98_%EC%A1%B0%EC%8A%AC%EB%9E%AD_2%EC%84%B8&amp;action=edit&amp;redlink=1" title="에데사의 조슬랭 2세 (없는 문서)">에데사의 조슬랭 2세</a>가 반란을 일으켜 비잔티움 군은 돌아가야 했고 요한네스는 모든 십자군 국가를 복속한다는 꿈을 실현하지 못하고 1143년 4월 8일 죽었다.</p>
<h2><span class="mw-headline" id=".EC.8B.AD.EC.9E.90.EA.B5.B0_.EC.A0.84.EC.9F.81.EA.B3.BC_.EA.B3.B5.EA.B5.AD.EC.9D.98_.EB.AA.B0.EB.9D.BD">십자군 전쟁과 공국의 몰락</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%95%88%ED%8B%B0%EC%98%A4%ED%82%A4%EC%95%84_%EA%B3%B5%EA%B5%AD&amp;action=edit&amp;section=4" title="부분 편집: 십자군 전쟁과 공국의 몰락">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p><a href="/wiki/1144%EB%85%84" title="1144년">1144년</a> <a href="/wiki/%EC%97%90%EB%8D%B0%EC%82%AC_%EB%B0%B1%EA%B5%AD" title="에데사 백국">에데사 백국</a>이 무너지고 안티오키아 공국은 <a href="/wiki/%EC%A0%9C2%EC%B0%A8_%EC%8B%AD%EC%9E%90%EA%B5%B0" title="제2차 십자군">제2차 십자군 전쟁기간</a> 내내 <a class="mw-redirect" href="/wiki/%EB%88%84%EB%A5%B4_%EC%95%97_%EB%94%98" title="누르 앗 딘">누르 앗 딘</a>과 전쟁을 벌였다. 1149년 레몽이 하림 전투에서 죽고 콩스탕스는 <a class="mw-redirect" href="/wiki/%EC%83%A4%ED%8B%B0%EC%9A%A9%EC%9D%98_%EB%A0%88%EB%82%A0%EB%93%9C" title="샤티용의 레날드">샤티용의 레날드</a>와 결혼하였는데 레날드는 비잔티움 제국을 또다시 침범했다가 비잔티움 황제 <a class="mw-redirect" href="/wiki/%EB%A7%88%EB%88%84%EC%97%98_1%EC%84%B8_%EC%BD%A4%EB%84%A4%EB%88%84%EC%8A%A4" title="마누엘 1세 콤네누스">마누엘 1세 콤네누스</a>에게 패하여 안티오키아 공국은 비잔티움 제국의 봉토가 되었다. <a href="/wiki/1180%EB%85%84" title="1180년">1180년</a> 황제 마누엘 1세가 죽자 안티오키아는 이슬람 세력을 견제해줄 방패막이 없어졌다.</p>
<p>이때부터 안티오키아 공국은 외부적으로는 십자군 전쟁에서 중요한 역할을 하지 못하고 내부적으로는 공국의 지배권을 놓고 내분에 휩싸였다. <a href="/wiki/1187%EB%85%84" title="1187년">1187년</a> <a class="mw-redirect" href="/wiki/%EC%82%B4%EB%9D%BC%ED%9D%90_%EC%95%97_%EB%94%98" title="살라흐 앗 딘">살라흐 앗 딘</a>의 <a href="/wiki/%EC%98%88%EB%A3%A8%EC%82%B4%EB%A0%98" title="예루살렘">예루살렘</a> 공격 당시 안티오키아는 이탈리아 도시국가들의 연합함대의 도움으로 무사했지만 이어 벌어진 <a href="/wiki/%EC%A0%9C3%EC%B0%A8_%EC%8B%AD%EC%9E%90%EA%B5%B0" title="제3차 십자군">제3차 십자군</a> 전쟁 때는 아무런 역할을 하지 못했다. 레몽과 콩스탕스의 아들인 <a class="new" href="/w/index.php?title=%EB%B3%B4%EC%97%90%EB%AA%BD_3%EC%84%B8&amp;action=edit&amp;redlink=1" title="보에몽 3세 (없는 문서)">보에몽 3세</a>가 공국을 지배했고 그의 아들 <a class="new" href="/w/index.php?title=%EB%B3%B4%EC%97%90%EB%AA%BD_4%EC%84%B8&amp;action=edit&amp;redlink=1" title="보에몽 4세 (없는 문서)">보에몽 4세</a>는 <a href="/wiki/%ED%95%98%ED%8B%B4_%EC%A0%84%ED%88%AC" title="하틴 전투">하틴 전투</a> 이후 <a href="/wiki/%ED%8A%B8%EB%A6%AC%ED%8F%B4%EB%A6%AC_%EB%B0%B1%EA%B5%AD" title="트리폴리 백국">트리폴리 백국</a>의 섭정이 되었다. <a href="/wiki/1201%EB%85%84" title="1201년">1201년</a> 보에몽 3세가 죽자 보에몽 4세와 형제 레몽 뤼펭은 공국의 지배권을 놓고 다투었다. 그사이 <a class="new" href="/w/index.php?title=%EB%B3%B4%EC%97%90%EB%AA%BD_5%EC%84%B8&amp;action=edit&amp;redlink=1" title="보에몽 5세 (없는 문서)">보에몽 5세</a>와 <a class="new" href="/w/index.php?title=%EB%B3%B4%EC%97%90%EB%AA%BD_6%EC%84%B8&amp;action=edit&amp;redlink=1" title="보에몽 6세 (없는 문서)">보에몽 6세</a>를 거치는 동안 <a href="/wiki/%EC%A0%9C5%EC%B0%A8_%EC%8B%AD%EC%9E%90%EA%B5%B0" title="제5차 십자군">제5차 십자군</a>, <a class="mw-redirect" href="/wiki/%EC%8B%A0%EC%84%B1%EB%A1%9C%EB%A7%88%EC%A0%9C%EA%B5%AD%EC%9D%98_%ED%94%84%EB%A6%AC%EB%93%9C%EB%A6%AC%ED%9E%88_2%EC%84%B8" title="신성로마제국의 프리드리히 2세">신성로마제국의 프리드리히 2세</a>의 <a href="/wiki/%EC%A0%9C6%EC%B0%A8_%EC%8B%AD%EC%9E%90%EA%B5%B0" title="제6차 십자군">제6차 십자군</a>, <a class="mw-redirect" href="/wiki/%ED%94%84%EB%9E%91%EC%8A%A4%EC%9D%98_%EB%A3%A8%EC%9D%B4_9%EC%84%B8" title="프랑스의 루이 9세">프랑스의 루이 9세</a>의 <a class="new" href="/w/index.php?title=%EC%A0%9C7%EC%B0%A8_%EC%8B%AD%EC%9E%90%EA%B5%B0&amp;action=edit&amp;redlink=1" title="제7차 십자군 (없는 문서)">제7차 십자군</a> 전쟁이 일어났지만 안티오키아 공국은 아무런 역할을 하지 못했다.</p>
<p><a href="/wiki/1254%EB%85%84" title="1254년">1254년</a> 오랜 내분을 끝내고 보에몽 6세가 정권을 잡았지만 <a class="mw-redirect" href="/wiki/%EB%A7%98%EB%A3%A8%ED%81%AC_%EC%99%95%EC%A1%B0" title="맘루크 왕조">맘루크 왕조</a>의 이슬람과 <a href="/wiki/%EB%AA%BD%EA%B3%A8" title="몽골">몽골</a>의 전쟁으로 안티오키아는 위협받았다. 맘루크의 <a href="/wiki/%EC%88%A0%ED%83%84" title="술탄">술탄</a> <a href="/wiki/%EB%B0%94%EC%9D%B4%EB%B0%94%EB%A5%B4%EC%8A%A4" title="바이바르스">바이바르스</a>는 몽골을 격파하고 몽골의 영향에 있던 안티오키아를 <a href="/wiki/1268%EB%85%84" title="1268년">1268년</a>에 점령하였다. 이로써 안티오키아 공국은 멸망하였다.</p>
