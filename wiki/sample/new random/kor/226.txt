<h1 class="firstHeading" id="firstHeading" lang="ko">기도 폰 슈타르헴베르크</h1>
<div class="mw-body-content" id="bodyContent">
<div id="siteSub">위키백과, 우리 모두의 백과사전.</div>
<div id="contentSub"><span class="mw-redirectedfrom">(<a class="mw-redirect" href="/w/index.php?title=%EA%B8%B0%EB%8F%84_%EC%8A%88%ED%83%80%EB%A5%B4%ED%97%B4%EB%B2%A0%EB%A5%B4%ED%81%AC&amp;redirect=no" title="기도 슈타르헴베르크">기도 슈타르헴베르크</a>에서 넘어옴)</span></div>

<p><b>슈타르헴베르크 백작 기도 발트 뤼디거</b>(Guido Wald Rüdiger, count of Starhemberg, <a href="/wiki/1657%EB%85%84" title="1657년">1657년</a> <a href="/wiki/11%EC%9B%94_11%EC%9D%BC" title="11월 11일">11월 11일</a> <a href="/wiki/%EA%B7%B8%EB%9D%BC%EC%B8%A0" title="그라츠">그라츠</a> - <a href="/wiki/1737%EB%85%84" title="1737년">1737년</a> <a href="/wiki/5%EC%9B%94_7%EC%9D%BC" title="5월 7일">5월 7일</a> <a href="/wiki/%EB%B9%88" title="빈">빈</a>)는 <a href="/wiki/%EC%98%A4%EC%8A%A4%ED%8A%B8%EB%A6%AC%EC%95%84" title="오스트리아">오스트리아</a> 군사 장교이다.</p>
<p>그는 <a class="new" href="/w/index.php?title=%EC%97%90%EB%A5%B8%EC%8A%A4%ED%8A%B8_%EB%A3%A8%EB%94%94%EA%B1%B0_%ED%8F%B0_%EC%8A%88%ED%83%80%EB%A5%B4%ED%97%B4%EB%B2%A0%EB%A5%B4%ED%81%AC&amp;action=edit&amp;redlink=1" title="에른스트 루디거 폰 슈타르헴베르크 (없는 문서)">에른스트 루디거 폰 슈타르헴베르크</a>(Ernst Rüdiger von Starhemberg, 1638-1701)의 사촌으로 1683년의 <a href="/wiki/%EC%98%A4%EC%8A%A4%EB%A7%8C_%EC%A0%9C%EA%B5%AD" title="오스만 제국">투르크</a> 공성전기간 <a href="/wiki/%EB%B9%88" title="빈">빈</a>의 유명한 지휘관 중 한명으로 공성전기간 ADC 로 행동했다. 기도는 그의 사촌을 따라 훗날 <a class="mw-redirect" href="/wiki/%EC%82%AC%EB%B3%B4%EC%9D%B4_%EA%B3%B5_%EC%99%B8%EC%A0%A0" title="사보이 공 외젠">사보이 공 외젠</a>과 함께 투르크와 싸웠다.</p>
<p><a href="/wiki/%EC%8A%A4%ED%8E%98%EC%9D%B8_%EC%99%95%EC%9C%84_%EA%B3%84%EC%8A%B9_%EC%A0%84%EC%9F%81" title="스페인 왕위 계승 전쟁">스페인 왕위 계승 전쟁</a>에서 슈테렘베르크는 이탈리아와 스페인에서 싸웠다. 1706년과 1708년 사이 그는 헝가리에서 황제군의 총사령관이 되어 <a href="/wiki/%EB%9D%BC%EC%BD%94%EC%B9%98_%ED%8E%98%EB%A0%8C%EC%B8%A0_2%EC%84%B8" title="라코치 페렌츠 2세">라코치 페렌츠 2세</a>(Rákóczi Ferenc II)의 반란에 맞써 대처했다. 1708년 그는 오스트리아와 스페인의 최고 사령관에 지명되었다.</p>
<p>동시에 <a class="new" href="/w/index.php?title=%EC%8A%A4%ED%85%90%ED%98%B8%ED%94%84_%EB%B0%B1%EC%9E%91_1%EC%84%B8_%EC%A0%9C%EC%9E%84%EC%8A%A4_%EC%8A%A4%ED%85%90%ED%98%B8%ED%94%84&amp;action=edit&amp;redlink=1" title="스텐호프 백작 1세 제임스 스텐호프 (없는 문서)">제임스 스텐호프</a>(James Stanhope)와 함께 1710년 <a href="/wiki/%EB%A7%88%EB%93%9C%EB%A6%AC%EB%93%9C" title="마드리드">마드리드</a>를 정복하는 데 성공했고, 뒤이어 <a href="/wiki/%EC%95%8C%EB%A9%94%EB%82%98%EB%9D%BC_%EC%A0%84%ED%88%AC" title="알메나라 전투">알메나라 전투</a>와 <a href="/wiki/%EC%82%AC%EB%9D%BC%EA%B3%A0%EC%82%AC_%EC%A0%84%ED%88%AC" title="사라고사 전투">사라고사 전투</a>에서 승리를 거두었다.</p>
<p>그러나 12월 그는 군사력에 의해 도시를 점령했지만 합스부르크 왕위 계승자와 마드리드의 시민들을 뒷받침하는 데 부족했다. 뒤이어 <a href="/wiki/%EB%B8%8C%EB%A6%AC%EC%9A%B0%EC%97%90%EA%B0%80_%EC%A0%84%ED%88%AC" title="브리우에가 전투">브리우에가 전투</a>와 <a href="/wiki/%EB%B9%84%EC%95%BC%EB%B9%84%EC%8B%9C%EC%98%A4%EC%82%AC_%EC%A0%84%ED%88%AC" title="비야비시오사 전투">비야비시오사 전투</a>(1710년)에서 패배해 그는 <a class="mw-redirect" href="/wiki/%EC%B9%B4%ED%83%88%EB%A1%9C%EB%8B%88%EC%95%84" title="카탈로니아">카탈로니아</a>로 물러났고, 거기서 그는 총독(Viceroy)이 되었지만 <a href="/wiki/%EC%B9%B4%EB%A5%BC_6%EC%84%B8" title="카를 6세">카를 대공</a>은 오스트리아로 돌아갔다.</p>
<p><a href="/wiki/%EC%9C%84%ED%8A%B8%EB%A0%88%ED%9D%90%ED%8A%B8_%EC%A1%B0%EC%95%BD" title="위트레흐트 조약">위트레흐트 조약</a>(1713년) 이후 카를 대공은 황제 카를 6세가 되었고, 카탈로니아를 넘겨주라고 명령했다. 그와 그의 군대는 <a href="/wiki/%EC%A0%9C%EB%85%B8%EB%B0%94" title="제노바">제노바</a>와 잉글랜드 배를 이용해 철수했다.</p>
<p>그는 <a href="/wiki/%EC%8A%AC%EB%9D%BC%EB%B3%B4%EB%8B%88%EC%95%84" title="슬라보니아">슬라보니아</a> 총독이 되었고, 1737년에 죽었다.</p>
