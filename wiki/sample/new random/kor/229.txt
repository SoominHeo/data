<p><b>조지 캐틀렛 마셜</b>(George Catlett Marshall, <a href="/wiki/1880%EB%85%84" title="1880년">1880년</a> <a href="/wiki/12%EC%9B%94_31%EC%9D%BC" title="12월 31일">12월 31일</a> ~ <a href="/wiki/1959%EB%85%84" title="1959년">1959년</a> <a href="/wiki/10%EC%9B%94_31%EC%9D%BC" title="10월 31일">10월 31일</a>)은 <a href="/wiki/%EB%AF%B8%EA%B5%AD" title="미국">미국</a>의 군인·정치가이다. <a href="/wiki/%ED%8E%9C%EC%8B%A4%EB%B2%A0%EC%9D%B4%EB%8B%88%EC%95%84_%EC%A3%BC" title="펜실베이니아 주">펜실베이니아 주</a>에서 출생하여 버지니아 군사학교(Virginia Military Institute)를 졸업했고, <a href="/wiki/%EC%A0%9C1%EC%B0%A8_%EC%84%B8%EA%B3%84_%EB%8C%80%EC%A0%84" title="제1차 세계 대전">제1차 세계 대전</a> 때 미국 파견군 제1·제8군 참모장으로 참전하고, <a href="/wiki/1938%EB%85%84" title="1938년">1938년</a> 참모 본부 작전부장을 거치면서 <a href="/wiki/1939%EB%85%84" title="1939년">1939년</a>에 <a class="mw-redirect" href="/wiki/%EB%AF%B8%EA%B5%AD_%EC%9C%A1%EA%B5%B0%EC%B0%B8%EB%AA%A8%EC%B4%9D%EC%9E%A5" title="미국 육군참모총장">미국 육군참모총장</a>이 되었다. <a href="/wiki/%EC%A0%9C2%EC%B0%A8_%EC%84%B8%EA%B3%84_%EB%8C%80%EC%A0%84" title="제2차 세계 대전">제2차 세계 대전</a> 때 미·영 합동 참모본부의 최고 수뇌가 되어 북프랑스 상륙 작전을 지휘하였다. <a href="/wiki/1947%EB%85%84" title="1947년">1947년</a> 국무장관이 되자 <a class="mw-redirect" href="/wiki/%EB%A7%88%EC%85%9C_%EA%B3%84%ED%9A%8D" title="마셜 계획">마셜 계획</a>을 세워 유럽 부흥에 크게 기여하였다. <a href="/wiki/1953%EB%85%84" title="1953년">1953년</a> <a href="/wiki/%EB%85%B8%EB%B2%A8_%ED%8F%89%ED%99%94%EC%83%81" title="노벨 평화상">노벨 평화상</a>을 받았다.</p>
