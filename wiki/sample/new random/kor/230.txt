<p><b>손병희</b>(孫秉熙, <a href="/wiki/1861%EB%85%84" title="1861년">1861년</a> <a href="/wiki/4%EC%9B%94_8%EC%9D%BC" title="4월 8일">4월 8일</a> ~ <a href="/wiki/1922%EB%85%84" title="1922년">1922년</a> <a href="/wiki/5%EC%9B%94_19%EC%9D%BC" title="5월 19일">5월 19일</a>)는 <a href="/wiki/%EC%B2%9C%EB%8F%84%EA%B5%90" title="천도교">천도교</a>(동학) 지도자이자 <a href="/wiki/%ED%95%9C%EA%B5%AD" title="한국">한국</a>의 독립운동가이다. 본관은 <a href="/wiki/%EB%B0%80%EC%96%91_%EC%86%90%EC%94%A8" title="밀양 손씨">밀양</a>이다. 자(字)는 응구(應九)이며, 망명 중 사용한 가명은 <b>이상헌</b>(李祥憲), <a href="/wiki/%EC%B5%9C%EC%8B%9C%ED%98%95" title="최시형">최시형</a>에게 받은 도호(道號)는 의암(義菴)이다.</p>

<h3><span class="mw-headline" id=".EC.B2.9C.EB.8F.84.EA.B5.90_.EC.9E.85.EB.8F.84">천도교 입도</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%86%90%EB%B3%91%ED%9D%AC&amp;action=edit&amp;section=2" title="부분 편집: 천도교 입도">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p><a href="/wiki/%EC%B6%A9%EC%B2%AD%EB%8F%84" title="충청도">충청도</a> <a class="mw-redirect" href="/wiki/%EC%B2%AD%EC%A3%BC" title="청주">청주목</a>(현 <a href="/wiki/%EC%B6%A9%EC%B2%AD%EB%B6%81%EB%8F%84" title="충청북도">충청북도</a> <a href="/wiki/%EC%B2%AD%EC%A3%BC%EC%8B%9C" title="청주시">청주시</a>)에서 세금징수를 담당하는 <a href="/wiki/%ED%96%A5%EB%A6%AC" title="향리">향리</a>의 아들로 태어났다. <a href="/wiki/1882%EB%85%84" title="1882년">1882년</a> 조카 손천민의 권유로 <a href="/wiki/%EB%8F%99%ED%95%99" title="동학">동학</a>에 입도했는데 이는 "모든 사람이 평등하다"는 <a href="/wiki/%EB%8F%99%ED%95%99" title="동학">동학</a>의 교리 때문이었다. 3년 뒤 <a href="/wiki/%EC%B5%9C%EC%8B%9C%ED%98%95" title="최시형">최시형</a>을 만나 그의 수제자가 되었다.</p>
<h3><span class="mw-headline" id=".EB.8F.99.ED.95.99.EB.86.8D.EB.AF.BC.EC.9A.B4.EB.8F.99_.EA.B0.80.EB.8B.B4">동학농민운동 가담</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%86%90%EB%B3%91%ED%9D%AC&amp;action=edit&amp;section=3" title="부분 편집: 동학농민운동 가담">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p><a href="/wiki/1894%EB%85%84" title="1894년">1894년</a> <a class="mw-redirect" href="/wiki/%EB%8F%99%ED%95%99%EB%86%8D%EB%AF%BC%EC%9A%B4%EB%8F%99" title="동학농민운동">동학농민운동</a> 때는 북접 소속으로서 남접의 <a href="/wiki/%EC%A0%84%EB%B4%89%EC%A4%80" title="전봉준">전봉준</a>과 함께 관군에 맞서 싸웠다. 관군의 추격을 피해 <a href="/wiki/%EC%9B%90%EC%82%B0%EC%8B%9C" title="원산시">원산</a> 및 관서지방으로 피신한 그는 <a class="mw-redirect" href="/wiki/%EB%8F%99%ED%95%99%EB%86%8D%EB%AF%BC%ED%98%81%EB%AA%85" title="동학농민혁명">동학농민혁명</a>실패로 무너진 <a href="/wiki/%EB%8F%99%ED%95%99" title="동학">동학</a>의 재건과 포교 활동에 큰 공을 세움으로써 <a href="/wiki/1897%EB%85%84" title="1897년">1897년</a> 정신적 스승이었던 <a href="/wiki/%EC%B5%9C%EC%8B%9C%ED%98%95" title="최시형">최시형</a>의 뒤를 이은 제3대 교주가 되었다. 한편 관에 자수한 최시형은 이듬해 혹세무민했다는 죄목으로 처형당한다. 이후 손병희는 동학운동의 지도자만이 아닌 근대화 운동의 지도자로서의 면모를 보인다. <a href="/wiki/%EB%8F%85%EB%A6%BD%ED%98%91%ED%9A%8C" title="독립협회">독립협회</a> 인사 등 개화파 인물들과 만나서 일부는 동학에 입교시켰으며, 이들로부터 개화 사상을 받아들였다.</p>
<h3><span class="mw-headline" id=".EC.9D.BC.EB.B3.B8_.EB.A7.9D.EB.AA.85">일본 망명</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%86%90%EB%B3%91%ED%9D%AC&amp;action=edit&amp;section=4" title="부분 편집: 일본 망명">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>그러던 와중에 동학에 대한 탄압이 거세지면서 먼저 동학에 입교하여, 자신에게 포교했던 조카 <a class="new" href="/w/index.php?title=%EC%86%90%EC%B2%9C%EB%AF%BC&amp;action=edit&amp;redlink=1" title="손천민 (없는 문서)">손천민</a>이 체포되어 처형당하자, <a href="/wiki/%EC%95%88%EA%B2%BD" title="안경">안경</a>장수로 변장하여 중국에 피신했으나 조선정부의 압력으로 중국에서 살 수 없었기 때문에, <a href="/wiki/1901%EB%85%84" title="1901년">1901년</a> <a href="/wiki/%EC%9D%BC%EB%B3%B8" title="일본">일본</a>으로 <a href="/wiki/%EB%A7%9D%EB%AA%85" title="망명">망명</a>했다. 여기에서도 같은 망명자 신분이었던 <a href="/wiki/%EC%98%A4%EC%84%B8%EC%B0%BD" title="오세창">오세창</a>, <a href="/wiki/%EA%B6%8C%EB%8F%99%EC%A7%84" title="권동진">권동진</a>, <a href="/wiki/%EB%B0%95%EC%98%81%ED%9A%A8" title="박영효">박영효</a>, <a class="mw-disambig" href="/wiki/%EC%A1%B0%ED%9D%AC%EC%97%B0" title="조희연">조희연</a> 등 개화파 전직 관료들과 교류하였고, <a class="mw-redirect" href="/wiki/%EC%83%81%ED%95%98%EC%9D%B4" title="상하이">상하이</a>와 <a href="/wiki/%EB%A9%94%EC%9D%B4%EC%A7%80_%EC%9C%A0%EC%8B%A0" title="메이지 유신">메이지 유신</a>(<a href="/wiki/1867%EB%85%84" title="1867년">1867년</a>) 이후 개혁의 바람이 불고 있던 <a href="/wiki/%EB%8F%84%EC%BF%84" title="도쿄">도쿄</a> 등을 돌아보면서 인재 양성이 시급함을 깨달았다. 이에 따라 <a href="/wiki/1903%EB%85%84" title="1903년">1903년</a>부터 24명의 똑똑한 청년들을 선발하여 일본에 유학시키게 된다. 망명 중 <a href="/wiki/%EC%8B%A0%EB%AC%B8" title="신문">신문</a> 기고 등으로 내정개혁론과 근대화론을 설파하다가 <a href="/wiki/1904%EB%85%84" title="1904년">1904년</a>에는 <a href="/wiki/%EA%B0%91%EC%A7%84%EA%B0%9C%ED%98%81%EC%9A%B4%EB%8F%99" title="갑진개혁운동">갑진개혁운동</a>을 일으켜, <a href="/wiki/%EA%B6%8C%EB%8F%99%EC%A7%84" title="권동진">권동진</a>, <a href="/wiki/%EC%98%A4%EC%84%B8%EC%B0%BD" title="오세창">오세창</a>과 더불어 <a class="new" href="/w/index.php?title=%EC%A7%84%EB%B3%B4%ED%9A%8C&amp;action=edit&amp;redlink=1" title="진보회 (없는 문서)">진보회</a>를 조직했다. 회원들에게 머리를 자르고 개화복을 입을 것을 명하는 등 개화 운동의 확산을 위한 단체였다.</p>

<h3><span class="mw-headline" id=".EA.B7.80.EA.B5.AD">귀국</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%86%90%EB%B3%91%ED%9D%AC&amp;action=edit&amp;section=5" title="부분 편집: 귀국">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p><a href="/wiki/1905%EB%85%84" title="1905년">1905년</a>에는 동학을 <a href="/wiki/%EC%B2%9C%EB%8F%84%EA%B5%90" title="천도교">천도교</a>로 개칭하고 <a href="/wiki/1906%EB%85%84" title="1906년">1906년</a> <a href="/wiki/%EC%9D%BC%EB%B3%B8" title="일본">일본</a>에서 귀국했다. 동학교도 즉, 천도교 신도라면 무조건 잡아갈 정도로 천도교를 극심히 탄압하던 <a href="/wiki/%EB%8C%80%ED%95%9C%EC%A0%9C%EA%B5%AD" title="대한제국">대한제국</a>이 외세의 간섭으로 무기력해진 상황이 되었기 때문이다. 또한 <a href="/wiki/%EC%B2%9C%EB%8F%84%EA%B5%90" title="천도교">천도교</a> 내부에서는 <a href="/wiki/%EC%9D%BC%EC%A7%84%ED%9A%8C" title="일진회">일진회</a>의 <a href="/wiki/%EC%86%A1%EB%B3%91%EC%A4%80" title="송병준">송병준</a>, <a href="/wiki/%EC%9D%B4%EC%9A%A9%EA%B5%AC" title="이용구">이용구</a>와 기타 세력 간의 반목이 심해졌고, 손병희는 친일 조직인 일진회 인물들에게 출교 처분을 내림으로써 이들과 결별한다.</p>
<h3><span class="mw-headline" id=".EB.8F.85.EB.A6.BD.EC.9A.B4.EB.8F.99">독립운동</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%86%90%EB%B3%91%ED%9D%AC&amp;action=edit&amp;section=6" title="부분 편집: 독립운동">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>귀국 후에는 교령 자리를 <a href="/wiki/%EB%B0%95%EC%9D%B8%ED%98%B8_(1854%EB%85%84)" title="박인호 (1854년)">박인호</a>에게 승계하고 <a href="/wiki/%EA%B5%90%EC%9C%A1" title="교육">교육</a> 사업(<a class="mw-redirect" href="/wiki/%EB%B3%B4%EC%84%B1%EC%A0%84%EB%AC%B8%ED%95%99%EA%B5%90" title="보성전문학교">보성전문학교</a>, <a href="/wiki/%EB%8F%99%EB%8D%95%EC%97%AC%ED%95%99%EB%8B%A8" title="동덕여학단">동덕여학단</a> 인수)과 <a href="/wiki/%EC%B6%9C%ED%8C%90%EC%82%AC" title="출판사">출판</a>사업에 관심을 쏟다가, <a href="/wiki/1919%EB%85%84" title="1919년">1919년</a> <a href="/wiki/%EB%AF%BC%EC%A1%B1%EB%8C%80%ED%91%9C_33%EC%9D%B8" title="민족대표 33인">민족대표 33인</a> 중 한 명으로 <a href="/wiki/3%C2%B71_%EC%9A%B4%EB%8F%99" title="3·1 운동">3·1 운동</a>을 주도했다. <a class="mw-redirect" href="/wiki/%EA%B8%B0%EB%AF%B8%EB%8F%85%EB%A6%BD%EC%84%A0%EC%96%B8%EC%84%9C" title="기미독립선언서">기미독립선언서</a> 낭독 후 일본 경찰에 체포되어 징역 3년형을 선고받았다.</p>
<h3><span class="mw-headline" id=".EC.B5.9C.ED.9B.84">최후</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%86%90%EB%B3%91%ED%9D%AC&amp;action=edit&amp;section=7" title="부분 편집: 최후">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>병보석으로 출옥한 후 <a href="/wiki/1922%EB%85%84" title="1922년">1922년</a> 가족들이 보는 가운데 별세하였다.</p>
<h2><span class="mw-headline" id=".EC.82.AC.ED.9B.84">사후</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%86%90%EB%B3%91%ED%9D%AC&amp;action=edit&amp;section=8" title="부분 편집: 사후">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<ul>
<li><a href="/wiki/%EB%8C%80%ED%95%9C%EB%AF%BC%EA%B5%AD_%EC%A0%95%EB%B6%80" title="대한민국 정부">대한민국 정부</a>는 그의 공훈을 기려 <a href="/wiki/1962%EB%85%84" title="1962년">1962년</a> <a href="/wiki/%EA%B1%B4%EA%B5%AD%ED%9B%88%EC%9E%A5" title="건국훈장">건국훈장</a> 대한민국장을 추서하였다.</li>
<li>충북 <a href="/wiki/%EC%B2%AD%EC%A3%BC%EC%8B%9C" title="청주시">청주</a>의 <a href="/wiki/%EC%82%BC%EC%9D%BC%EA%B3%B5%EC%9B%90_(%EC%B2%AD%EC%A3%BC%EC%8B%9C)" title="삼일공원 (청주시)">삼일공원</a>에 충북 출신 민족대표 33인인 권동진, <a href="/wiki/%EA%B6%8C%EB%B3%91%EB%8D%95" title="권병덕">권병덕</a>, <a href="/wiki/%EC%8B%A0%ED%99%8D%EC%8B%9D" title="신홍식">신홍식</a>, <a href="/wiki/%EC%8B%A0%EC%84%9D%EA%B5%AC" title="신석구">신석구</a>와 함께 동상이 설치되었다.</li>
<li>청원군 생가 자리에는 기념관이 건립되었다.<sup class="reference" id="cite_ref-1"><a href="#cite_note-1">[1]</a></sup><sup class="reference" id="cite_ref-2"><a href="#cite_note-2">[2]</a></sup></li>
</ul>
<h2><span class="mw-headline" id=".EC.A0.80.EC.84.9C">저서</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%86%90%EB%B3%91%ED%9D%AC&amp;action=edit&amp;section=9" title="부분 편집: 저서">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>저서로는 일본 망명 시절에 국부 달성과 근대화를 주장하며 저술한 《삼전론(三戰論)》(1902), 《명리전(明理傳)》(1903) 등이 있다.</p>
<h2><span class="mw-headline" id=".EA.B0.80.EC.A1.B1_.EA.B4.80.EA.B3.84">가족 관계</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%86%90%EB%B3%91%ED%9D%AC&amp;action=edit&amp;section=10" title="부분 편집: 가족 관계">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p><a href="/wiki/%EC%96%B4%EB%A6%B0%EC%9D%B4" title="어린이">어린이</a> 운동으로 유명한 <a href="/wiki/%EB%B0%A9%EC%A0%95%ED%99%98" title="방정환">방정환</a>은 손병희의 사위로서 천도교와 밀접한 관련이 있다. 방정환의 부친은 천도교 신자였으며, 방정환 자신도 <a href="/wiki/1923%EB%85%84" title="1923년">1923년</a> 천도교계 잡지에서 <a href="/wiki/%EC%9E%91%EA%B0%80" title="작가">작가</a>로 활동하였고,그가 <a href="/wiki/%EC%96%B4%EB%A6%B0%EC%9D%B4" title="어린이">어린이</a>들에게 <a class="new" href="/w/index.php?title=%EA%B5%AC%EC%97%B0%EB%8F%99%ED%99%94&amp;action=edit&amp;redlink=1" title="구연동화 (없는 문서)">구연동화</a>를 들려준 장소도 천도교 회관이었다.</p>

<li>부친 : 손의조</li>
<li>모친 : 최씨</li>
<li>동생 : <a class="new" href="/w/index.php?title=%EC%86%90%EB%B3%91%ED%9D%A0&amp;action=edit&amp;redlink=1" title="손병흠 (없는 문서)">손병흠</a>, 독립운동가</li>
<li>부인 : 홍응화(1870)</li>
<li>삼녀 : 손용화(<a href="/wiki/1901%EB%85%84" title="1901년">1901년</a> ~ <a href="/wiki/1991%EB%85%84" title="1991년">1991년</a> <a href="/wiki/10%EC%9B%94_18%EC%9D%BC" title="10월 18일">10월 18일</a>)</li>
<li>사위 : <a href="/wiki/%EB%B0%A9%EC%A0%95%ED%99%98" title="방정환">방정환</a>(<a href="/wiki/1899%EB%85%84" title="1899년">1899년</a> <a href="/wiki/11%EC%9B%94_9%EC%9D%BC" title="11월 9일">11월 9일</a> ~ <a href="/wiki/1931%EB%85%84" title="1931년">1931년</a> <a href="/wiki/7%EC%9B%94_23%EC%9D%BC" title="7월 23일">7월 23일</a>, 아동문학가, 독립운동가 겸 작가, 언론인)</li>
<li>이복 조카 : <a class="new" href="/w/index.php?title=%EC%86%90%EC%B2%9C%EB%AF%BC&amp;action=edit&amp;redlink=1" title="손천민 (없는 문서)">손천민</a>, 천도교인, 이복 형의 아들로 손병희와 <a class="new" href="/w/index.php?title=%EC%86%90%EB%B3%91%ED%9D%A0&amp;action=edit&amp;redlink=1" title="손병흠 (없는 문서)">손병흠</a>을 <a href="/wiki/%EC%B2%9C%EB%8F%84%EA%B5%90" title="천도교">천도교</a>에 입교시켰다.</li>
<li>3대 이후: 손권후,손민후 외</li>
