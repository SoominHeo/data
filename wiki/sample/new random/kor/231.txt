<p><b>팟타이</b>(<span style="font-size: smaller;"><a href="/wiki/%ED%83%9C%EA%B5%AD%EC%96%B4" title="태국어">태국어</a>:</span> <span lang="th" xml:lang="th">ผัดไทย</span>)는 <a href="/wiki/%ED%83%80%EC%9D%B4" title="타이">타이</a>의 <a href="/wiki/%EC%8C%80%EA%B5%AD%EC%88%98" title="쌀국수">쌀국수</a> <a href="/wiki/%EC%9A%94%EB%A6%AC" title="요리">요리</a>이다. 국수와 함께 <a class="mw-redirect" href="/wiki/%EA%B3%84%EB%9E%80" title="계란">계란</a>, 타이식 <a href="/wiki/%EC%96%B4%EC%9E%A5_(%EC%9D%8C%EC%8B%9D)" title="어장 (음식)">어장</a>(<span style="font-size: smaller;"><a href="/wiki/%ED%83%9C%EA%B5%AD%EC%96%B4" title="태국어">태국어</a>:</span> <span lang="th" xml:lang="th">น้ำปลา</span> <small>남쁠라<sup>[<a class="mw-redirect" href="/wiki/%EC%9C%84%ED%82%A4%EB%B0%B1%EA%B3%BC:%ED%83%80%EC%9D%B4%EC%96%B4%EC%9D%98_%ED%95%9C%EA%B8%80_%ED%91%9C%EA%B8%B0" title="위키백과:타이어의 한글 표기">*</a>]</sup></small>), <a href="/wiki/%ED%83%80%EB%A7%88%EB%A6%B0%EB%93%9C" title="타마린드">타마린드</a> <a href="/wiki/%EC%A3%BC%EC%8A%A4" title="주스">주스</a>, 붉은 <a href="/wiki/%EA%B3%A0%EC%B6%94" title="고추">고추</a>, <a href="/wiki/%EC%83%88%EC%9A%B0" title="새우">새우</a>, <a href="/wiki/%EB%8B%AD%EA%B3%A0%EA%B8%B0" title="닭고기">닭고기</a>, <a href="/wiki/%EB%91%90%EB%B6%80" title="두부">두부</a> 등을 넣고 <a class="mw-disambig" href="/wiki/%EA%B3%A0%EB%AA%85" title="고명">고명</a>으로 <a href="/wiki/%EA%B3%A0%EC%88%98_(%EC%8B%9D%EB%AC%BC)" title="고수 (식물)">고수</a>, <a href="/wiki/%EB%9D%BC%EC%9E%84" title="라임">라임</a>, 으깬 <a href="/wiki/%EB%95%85%EC%BD%A9" title="땅콩">땅콩</a> 등을 얹어 만든다. <a href="/wiki/%EB%98%A0%EC%96%8C" title="똠얌">똠얌</a> 등과 함께 타이를 대표하는 요리 중 하나이다.</p>
<p>지역별, 재료별로 다양한 팟타이의 종류가 있는데, <a href="/wiki/%EB%B0%A9%EC%BD%95" title="방콕">방콕</a>의 팟타이는 '팟타이 끄룽텝', <a href="/wiki/%EC%95%84%EC%9C%A0%ED%83%80%EC%95%BC_%EC%A3%BC" title="아유타야 주">아유타야</a>의 팟타이는 '팟타이 아유타야', <a href="/wiki/%EC%BD%94%EB%9E%8F_%EB%B6%84%EC%A7%80" title="코랏 분지">코랏 지역</a>의 매운 팟타이는 '팟미 코랏', 방콕 팟타이 중 <a href="/wiki/%EA%B3%A0%EA%B8%B0" title="고기">고기</a>가 없는 고전적인 팟타이는 '팟타이 망사위랏', <a href="/wiki/%EC%A7%A0%ED%83%80%EB%B6%80%EB%A6%AC_%EC%A3%BC" title="짠타부리 주">짠타부리</a>의 팟타이는 '팟미 짠타부리' 등으로 구별한다.<sup class="reference" id="cite_ref-a_1-0"><a href="#cite_note-a-1">[1]</a></sup></p>
<p><a href="/wiki/%EC%95%84%EC%9C%A0%ED%83%80%EC%95%BC_%EC%99%95%EC%A1%B0" title="아유타야 왕조">아유타야 왕조</a> 시대에 왕도 <a href="/wiki/%EC%95%84%EC%9C%A0%ED%83%80%EC%95%BC_(%EB%8F%84%EC%8B%9C)" title="아유타야 (도시)">아유타야</a>를 방문한 <a href="/wiki/%EB%B2%A0%ED%8A%B8%EB%82%A8" title="베트남">베트남</a> 상인들이 쌀국수를 가져온 것이 팟타이의 기원으로 추정되고 있으며, 시대에 따라 다양한 형태의 팟타이가 있었다. 그러나 이 요리가 본격적으로 타이를 대표하는 요리가 된 것은 <a href="/wiki/%EC%81%A0%EB%9E%99_%ED%94%BC%EB%B6%84%EC%86%A1%ED%81%AC%EB%9E%8C" title="쁠랙 피분송크람">쁠랙 피분송크람</a>의 <a href="/wiki/%EC%88%98%EC%83%81" title="수상">수상</a> 재임기인 1930년대~1940년대에 피분송크람이 벌인 타이 <a href="/wiki/%EB%AF%BC%EC%A1%B1%EC%A3%BC%EC%9D%98" title="민족주의">민족주의</a> 캠페인 덕분이었다. 또한 타이 국내의 쌀 소비량을 줄이고 수출을 늘려 당시 <a href="/wiki/%EC%8C%80" title="쌀">쌀</a> 수출에 크게 의존하고 있던 <a class="new" href="/w/index.php?title=%ED%83%80%EC%9D%B4%EC%9D%98_%EA%B2%BD%EC%A0%9C&amp;action=edit&amp;redlink=1" title="타이의 경제 (없는 문서)">타이의 경제</a>를 개선하고, 여러 목적으로 여분의 쌀을 사용하기 위해서 정책적으로 팟타이 요리가 권장된 바도 있었다.<sup class="reference" id="cite_ref-a_1-1"><a href="#cite_note-a-1">[1]</a></sup></p>
<h2><span class="mw-headline" id=".EC.96.B4.EC.9B.90">어원</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%ED%8C%9F%ED%83%80%EC%9D%B4&amp;action=edit&amp;section=1" title="부분 편집: 어원">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<ul>
<li>팟타이(ผัดไทย)는 팟(ผัด)과 타이(ไทย)로 이루어져 있다. ผัด는 '저으면서 익히다'라는 의미<sup class="reference" id="cite_ref-w_2-0"><a href="#cite_note-w-2">[2]</a></sup>를 가지고 있다. ไทย는 <a class="mw-redirect" href="/wiki/%ED%83%9C%EA%B5%AD" title="태국">태국</a>을 의미한다. <a href="/wiki/%ED%83%9C%EA%B5%AD%EC%96%B4" title="태국어">태국어</a>는 수식하는 말이 뒤에 오기 때문에, 태국의 저으면서 익힌 것(요리)이 된다.</li>
</ul>
