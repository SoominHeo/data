<p><b>조니 미첼</b>(Joni Mitchell, <a href="/wiki/%EC%BA%90%EB%82%98%EB%8B%A4_%ED%9B%88%EC%9E%A5" title="캐나다 훈장">CC</a>, <a href="/wiki/1943%EB%85%84" title="1943년">1943년</a> <a href="/wiki/11%EC%9B%94_7%EC%9D%BC" title="11월 7일">11월 7일</a>~)은 <a href="/wiki/%EC%BA%90%EB%82%98%EB%8B%A4" title="캐나다">캐나다</a>의 <a href="/wiki/%EC%9D%8C%EC%95%85%EA%B0%80" title="음악가">음악가</a>이자 <a href="/wiki/%ED%99%94%EA%B0%80" title="화가">화가</a>이다. <a href="/wiki/%EA%B8%B0%ED%83%80" title="기타">기타</a>나 <a href="/wiki/%ED%94%BC%EC%95%84%EB%85%B8" title="피아노">피아노</a>를 연주하며 노래를 하는 <a href="/wiki/%EC%8B%B1%EC%96%B4%EC%86%A1%EB%9D%BC%EC%9D%B4%ED%84%B0" title="싱어송라이터">싱어송라이터</a>이다.</p>
<p><a href="/wiki/1996%EB%85%84" title="1996년">1996년</a> <a href="/wiki/%ED%8F%B4%EB%9D%BC%EC%9D%8C%EC%95%85%EC%83%81" title="폴라음악상">폴라음악상</a>을 수상하였다.</p>

<h2><span class="mw-headline" id=".EC.83.9D.EC.95.A0">생애</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%A1%B0%EB%8B%88_%EB%AF%B8%EC%B2%BC&amp;action=edit&amp;section=1" title="부분 편집: 생애">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>캐나다 <a href="/wiki/%EC%95%A8%EB%B2%84%ED%83%80_%EC%A3%BC" title="앨버타 주">앨버타 주</a>의 작은 마을에서 태어났다. <a href="/wiki/%ED%94%BC%ED%8A%B8_%EC%8B%9C%EA%B1%B0" title="피트 시거">피트 시거</a>의 노래에 큰 영향을 받았다. 고등학교를 졸업한 후 <a href="/wiki/%ED%86%A0%EB%A1%A0%ED%86%A0" title="토론토">토론토</a>로 상경하여 예술을 공부했으나 임신을 하면서 그만 두었다. 그녀의 보수적인 가족은 미첼의 아이를 입양보냈다. <a href="/wiki/1965%EB%85%84" title="1965년">1965년</a> <a href="/wiki/%EB%AF%B8%EA%B5%AD" title="미국">미국</a> <a href="/wiki/%EB%94%94%ED%8A%B8%EB%A1%9C%EC%9D%B4%ED%8A%B8" title="디트로이트">디트로이트</a>로 떠났다.</p>
<h2><span class="mw-headline" id=".EC.9D.8C.EB.B0.98">음반</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%A1%B0%EB%8B%88_%EB%AF%B8%EC%B2%BC&amp;action=edit&amp;section=2" title="부분 편집: 음반">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
