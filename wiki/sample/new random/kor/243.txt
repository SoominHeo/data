<p><b>부평구</b>(富平區)는 <a href="/wiki/%EB%8C%80%ED%95%9C%EB%AF%BC%EA%B5%AD" title="대한민국">대한민국</a> <a href="/wiki/%EC%9D%B8%EC%B2%9C%EA%B4%91%EC%97%AD%EC%8B%9C" title="인천광역시">인천광역시</a>의 동중부에 있는 <a href="/wiki/%EA%B5%AC_(%ED%96%89%EC%A0%95_%EA%B5%AC%EC%97%AD)" title="구 (행정 구역)">자치구</a>이다. 원적산, 광학산, 거마산을 경계로 <a class="mw-redirect" href="/wiki/%EC%9D%B8%EC%B2%9C" title="인천">인천</a>의 서부 도심과는 별개의 시가지를 형성하고 있다.</p>

<div class="rellink boilerplate seealso"><img alt="" data-file-height="128" data-file-width="128" height="16" src="//upload.wikimedia.org/wikipedia/commons/thumb/e/ec/Crystal_Clear_app_xmag.svg/16px-Crystal_Clear_app_xmag.svg.png" srcset="//upload.wikimedia.org/wikipedia/commons/thumb/e/ec/Crystal_Clear_app_xmag.svg/24px-Crystal_Clear_app_xmag.svg.png 1.5x, //upload.wikimedia.org/wikipedia/commons/thumb/e/ec/Crystal_Clear_app_xmag.svg/32px-Crystal_Clear_app_xmag.svg.png 2x" width="16"/> 1988년 1차 분구, 1995년 2차 분구 이후의 연혁은 <a href="/wiki/%EC%84%9C%EA%B5%AC_(%EC%9D%B8%EC%B2%9C%EA%B4%91%EC%97%AD%EC%8B%9C)" title="서구 (인천광역시)">서구 (인천광역시)</a>, <a href="/wiki/%EA%B3%84%EC%96%91%EA%B5%AC" title="계양구">계양구</a> 문서를 참고하십시오.</div>
<p>부평은 <a href="/wiki/%ED%99%A9%ED%95%B4" title="황해">황해</a>와 <a href="/wiki/%ED%95%9C%EA%B0%95" title="한강">한강</a>을 사이에 둔 지역으로 <a class="mw-redirect" href="/wiki/%EC%84%A0%EC%82%AC%EC%8B%9C%EB%8C%80" title="선사시대">선사시대</a>부터 인류의 활동이 있었다. 인근의 가현산 북쪽에 140여 기의 고인돌 군이 있어 이 일대가 강력한 <a class="mw-redirect" href="/wiki/%EC%B2%AD%EB%8F%99%EA%B8%B0" title="청동기">청동기</a> 문화의 중심지였음을 알게해 준다. 부평이 역사에 처음으로 등장하는 것은 <a class="mw-redirect" href="/wiki/%EC%82%BC%ED%95%9C%EC%8B%9C%EB%8C%80" title="삼한시대">삼한시대</a>의 일로 부평은 <a href="/wiki/%EB%A7%88%ED%95%9C" title="마한">마한</a>의 영역에 해당하였다. <a href="/wiki/%EB%B0%B1%EC%A0%9C" title="백제">백제</a>가 형성되면서 백제의 영토로 편입되었으나, <a href="/wiki/475%EB%85%84" title="475년">475년</a>에 <a href="/wiki/%EA%B3%A0%EA%B5%AC%EB%A0%A4" title="고구려">고구려</a> <a href="/wiki/%EC%9E%A5%EC%88%98%EC%99%95" title="장수왕">장수왕</a>이 한강 이남을 공략하여 주부토군(主夫吐郡)을 세웠고 부평역시 주부토군에 편입되었다. 그 후 백제가 다시 이 지역을 회복하였으나 <a href="/wiki/553%EB%85%84" title="553년">553년</a>에 <a href="/wiki/%EC%8B%A0%EB%9D%BC" title="신라">신라</a>가 한강 유역을 점령한 후부터 신라의 영토가 되었다. <a href="/wiki/%EB%82%A8%EB%B6%81%EA%B5%AD_%EC%8B%9C%EB%8C%80" title="남북국 시대">남북국 시대</a>인 신라 <a href="/wiki/%EA%B2%BD%EB%8D%95%EC%99%95" title="경덕왕">경덕왕</a> <a href="/wiki/757%EB%85%84" title="757년">16년(757년)</a>에 장제군이 되었고 하위 행정지역으로 술성현(戌城縣), 김포현(金浦縣), 동성현(童城縣), 분진현(分津縣) 등이 있었다.<sup class="reference" id="cite_ref-1"><a href="#cite_note-1">[1]</a></sup></p>
<p>부평은 <a href="/wiki/%EA%B3%A0%EB%A0%A4" title="고려">고려</a> 초기에 수주로 불리었으며, 이후 이름이 바뀌어 <a href="/wiki/%EA%B3%A0%EB%A0%A4_%EC%9D%98%EC%A2%85" title="고려 의종">의종</a> <a href="/wiki/1150%EB%85%84" title="1150년">4년(1150년)</a>에 안남, <a href="/wiki/%EA%B3%A0%EB%A0%A4_%EA%B3%A0%EC%A2%85" title="고려 고종">고종</a> <a href="/wiki/1215%EB%85%84" title="1215년">2년(1215년)</a>에는 계양, <a class="mw-redirect" href="/wiki/%EC%B6%A9%EB%A0%AC%EC%99%95" title="충렬왕">충렬왕</a> 34년에는 길주 등으로 불렸고, 부평이란 이름은 <a class="mw-redirect" href="/wiki/%EC%B6%A9%EC%84%A0%EC%99%95" title="충선왕">충선왕</a> 2년(<a href="/wiki/1313%EB%85%84" title="1313년">1313년</a>)에 붙여진 것이다. 그 후로는 계속해서 부평으로 불렸다. <a href="/wiki/%EC%A1%B0%EC%84%A0" title="조선">조선</a> 시대에는 <a href="/wiki/%EC%A1%B0%EC%84%A0_%ED%83%9C%EC%A2%85" title="조선 태종">태종</a> 13년(1413년) 부평도호부를 설치하였고 <a class="mw-redirect" href="/wiki/%EC%A1%B0%EC%84%A0_%EA%B3%A0%EC%A2%85" title="조선 고종">고종</a> 32년(<a href="/wiki/1895%EB%85%84" title="1895년">1895년</a>)에 부평군으로 개칭되었으며, 1914년에 <a href="/wiki/%EB%B6%80%EC%B2%9C%EA%B5%B0" title="부천군">부천군</a>에 편입되었다.<sup class="reference" id="cite_ref-.EC.9D.B4.EC.84.B1.EA.B5.AC_2-0"><a href="#cite_note-.EC.9D.B4.EC.84.B1.EA.B5.AC-2">[2]</a></sup></p>
<p>부평도호부는 지금의 <a class="mw-disambig" href="/wiki/%EB%B6%80%ED%8F%89%EC%B4%88%EB%93%B1%ED%95%99%EA%B5%90" title="부평초등학교">부평초등학교</a> 자리에 있었고 <a href="/wiki/%EA%B3%84%EC%96%91%EC%82%B0" title="계양산">계양산</a>을 중심으로 지금의 <a href="/wiki/%EC%9D%B8%EC%B2%9C%EA%B4%91%EC%97%AD%EC%8B%9C" title="인천광역시">인천광역시</a> 부평구, <a href="/wiki/%EC%84%9C%EA%B5%AC_(%EC%9D%B8%EC%B2%9C%EA%B4%91%EC%97%AD%EC%8B%9C)" title="서구 (인천광역시)">서구</a>, <a href="/wiki/%EA%B3%84%EC%96%91%EA%B5%AC" title="계양구">계양구</a>, <a href="/wiki/%EA%B2%BD%EA%B8%B0%EB%8F%84" title="경기도">경기도</a> <a href="/wiki/%EB%B6%80%EC%B2%9C%EC%8B%9C" title="부천시">부천시</a>, <a href="/wiki/%EC%84%9C%EC%9A%B8%ED%8A%B9%EB%B3%84%EC%8B%9C" title="서울특별시">서울특별시</a> <a href="/wiki/%EA%B5%AC%EB%A1%9C%EA%B5%AC" title="구로구">구로구</a> 서부, <a href="/wiki/%EA%B0%95%EC%84%9C%EA%B5%AC_(%EC%84%9C%EC%9A%B8%ED%8A%B9%EB%B3%84%EC%8B%9C)" title="강서구 (서울특별시)">강서구</a> 오곡동과 오쇠동을 관할하고 있었다. <a href="/wiki/1914%EB%85%84" title="1914년">1914년</a> <a href="/wiki/%EB%B6%80%EC%B2%9C%EA%B5%B0" title="부천군">부천군</a>이 설치되어 부내면이 되었고, <a href="/wiki/1940%EB%85%84" title="1940년">1940년</a> 부내면이 인천부에 편입되었다.<sup class="reference" id="cite_ref-.EC.9D.B4.EC.84.B1.EA.B5.AC_2-1"><a href="#cite_note-.EC.9D.B4.EC.84.B1.EA.B5.AC-2">[2]</a></sup></p>

<h2><span class="mw-headline" id=".ED.96.89.EC.A0.95_.EA.B5.AC.EC.97.AD">행정 구역</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%B6%80%ED%8F%89%EA%B5%AC&amp;action=edit&amp;section=2" title="부분 편집: 행정 구역">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<div class="rellink relarticle mainarticle"><img alt="" data-file-height="128" data-file-width="128" height="16" src="//upload.wikimedia.org/wikipedia/commons/thumb/e/ec/Crystal_Clear_app_xmag.svg/16px-Crystal_Clear_app_xmag.svg.png" srcset="//upload.wikimedia.org/wikipedia/commons/thumb/e/ec/Crystal_Clear_app_xmag.svg/24px-Crystal_Clear_app_xmag.svg.png 1.5x, //upload.wikimedia.org/wikipedia/commons/thumb/e/ec/Crystal_Clear_app_xmag.svg/32px-Crystal_Clear_app_xmag.svg.png 2x" width="16"/> 이 부분의 본문은 <a href="/wiki/%EB%B6%80%ED%8F%89%EA%B5%AC%EC%9D%98_%ED%96%89%EC%A0%95_%EA%B5%AC%EC%97%AD" title="부평구의 행정 구역">부평구의 행정 구역</a>입니다.</div>
<p>부평구의 행정 구역은 22개동 (637통, 4,197개반)으로 구성되어 있다. 부평구의 면적은 31.98 km<sup>2</sup>로 인천시의 약 3.09%이며, 인구는 2015년 11월 기준으로 556,071명으로 인천시의 약 19%를 차지한다.<sup class="reference" id="cite_ref-9"><a href="#cite_note-9">[9]</a></sup></p>
