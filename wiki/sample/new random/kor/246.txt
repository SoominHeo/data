<p><b>만나이인</b>(<span style="font-size: smaller;"><a href="/wiki/%EC%95%84%EC%B9%B4%EB%93%9C%EC%96%B4" title="아카드어">아카드어</a>:</span> <span lang="akk" xml:lang="akk">Mannai</span>, <span style="font-size: smaller;"><a href="/wiki/%EC%98%81%EC%96%B4" title="영어">영어</a>:</span> <span lang="en" xml:lang="en">Mannaeans</span>)은 <a href="/wiki/%EA%B8%B0%EC%9B%90%EC%A0%84_10%EC%84%B8%EA%B8%B0" title="기원전 10세기">기원전 10세기</a>경부터 <a href="/wiki/%EA%B8%B0%EC%9B%90%EC%A0%84_7%EC%84%B8%EA%B8%B0" title="기원전 7세기">7세기</a>경까지 오늘날의 <a href="/wiki/%EC%9D%B4%EB%9E%80" title="이란">이란</a> 북서부의 <a href="/wiki/%EC%9A%B0%EB%A5%B4%EB%AF%B8%EC%95%84_%ED%98%B8" title="우르미아 호">우르미아 호</a>의 남쪽 지역에 살았던 고대 민족이다. 만나이인이 살았던 이 지역을 <b>만나</b>(<a class="extiw" href="https://en.wikipedia.org/wiki/Manna_(disambiguation)" title="en:Manna (disambiguation)"><span style="color: black">Manna</span></a>)라고 한다. 만나이인의 나라를 통상적으로 <b>만나이 왕국</b>(Kingdom of Mannai)이라 하는데, 《<a href="/wiki/%EA%B5%AC%EC%95%BD%EC%84%B1%EA%B2%BD" title="구약성경">구약성경</a>》에 나오는 <b>민니</b>(<span style="font-size: smaller;"><a href="/wiki/%ED%9E%88%EB%B8%8C%EB%A6%AC%EC%96%B4" title="히브리어">히브리어</a>:</span> <span lang="he" xml:lang="he">מנּי</span> <small>Minni</small>: <a class="mw-redirect" href="/wiki/%EC%98%88%EB%A0%88%EB%AF%B8%EC%95%84" title="예레미아">예레미아</a> 51:27<sup class="reference" id="cite_ref-1"><a href="#cite_note-1">[1]</a></sup>)가 만나이 왕국일 가능성도 있다.</p>
<p>당시에 만나이 왕국은 <a class="mw-redirect" href="/wiki/%EC%95%84%EC%8B%9C%EB%A6%AC%EC%95%84_%EC%A0%9C%EA%B5%AD" title="아시리아 제국">아시리아 제국</a>(<a class="mw-redirect" href="/wiki/%EC%8B%A0_%EC%95%84%EC%8B%9C%EB%A6%AC%EC%95%84_%EC%A0%9C%EA%B5%AD" title="신 아시리아 제국">신 아시리아 제국</a>: 911~612 BC<sup class="reference" id="cite_ref-NeoAssiria911-612_2-0"><a href="#cite_note-NeoAssiria911-612-2">[2]</a></sup> 또는 934~605 BC<sup class="reference" id="cite_ref-NeoAssiria934-605_3-0"><a href="#cite_note-NeoAssiria934-605-3">[3]</a></sup>)과 <a class="mw-redirect" href="/wiki/%EC%9A%B0%EB%9D%BC%EB%A5%B4%ED%88%AC_%EC%A0%9C%EA%B5%AD" title="우라르투 제국">우라르투 제국</a>(1350~590 BC 또는 860~585 BC)과 이웃하고 있었으며, 또한 이 두 제국 사이에 존재한 <a href="/wiki/%EB%AC%B4%EC%82%AC%EC%8B%9C%EB%A5%B4" title="무사시르">무사시르</a>(<a class="extiw" href="https://en.wikipedia.org/wiki/Musasir" title="en:Musasir"><span style="color: black">Musasir</span></a>)와 <a class="new" href="/w/index.php?title=%EC%A7%80%ED%82%A4%EB%A5%B4%ED%83%80&amp;action=edit&amp;redlink=1" title="지키르타 (없는 문서)">지키르타</a>(<a class="extiw" href="https://en.wikipedia.org/wiki/Zikirta" title="en:Zikirta"><span style="color: black">Zikirta</span></a>)와 같은 작은 <a href="/wiki/%EC%99%84%EC%B6%A9%EA%B5%AD" title="완충국">완충국</a>들과도 이웃하고 있었다.</p>

<a class="mw-redirect" href="/wiki/%EA%B8%B0%EC%9B%90%EC%A0%84_820%EB%85%84" title="기원전 820년">기원전 820년</a>~<a class="mw-redirect" href="/wiki/%EA%B8%B0%EC%9B%90%EC%A0%84_785%EB%85%84" title="기원전 785년">785년</a>의 <a class="mw-redirect" href="/wiki/%EC%9A%B0%EB%9D%BC%EB%A5%B4%ED%88%AC_%EC%A0%9C%EA%B5%AD" title="우라르투 제국">우라르투 제국</a>과 주변 지역: 노란색은 <a class="mw-redirect" href="/wiki/%EC%9A%B0%EB%9D%BC%EB%A5%B4%ED%88%AC_%EC%A0%9C%EA%B5%AD" title="우라르투 제국">우라르투 제국</a>(지도에서 "Lake Urmia")의 강역이며 보라색은 <a class="mw-redirect" href="/wiki/%EC%8B%A0_%EC%95%84%EC%8B%9C%EB%A6%AC%EC%95%84_%EC%A0%9C%EA%B5%AD" title="신 아시리아 제국">신 아시리아 제국</a>의 강역이다. 지도의 오른쪽 아래, 즉 <a href="/wiki/%EC%9A%B0%EB%A5%B4%EB%AF%B8%EC%95%84_%ED%98%B8" title="우르미아 호">우르미아 호</a>의 남쪽에 만나이 왕국의 지역을 뜻하는 <b>만나</b>(지도에서 <b>Manna</b>)가 표시되어 있다. <a href="/wiki/%EC%9A%B0%EB%A5%B4%EB%AF%B8%EC%95%84_%ED%98%B8" title="우르미아 호">우르미아 호</a>의 서쪽, 즉 <a class="mw-redirect" href="/wiki/%EC%9A%B0%EB%9D%BC%EB%A5%B4%ED%88%AC_%EC%A0%9C%EA%B5%AD" title="우라르투 제국">우라르투 제국</a>과 <b>만나</b> 사이에 <a href="/wiki/%EB%AC%B4%EC%82%AC%EC%8B%9C%EB%A5%B4" title="무사시르">무사시르</a>(지도에서 "Musasir")가 있다. <a href="/wiki/%EC%9A%B0%EB%A5%B4%EB%AF%B8%EC%95%84_%ED%98%B8" title="우르미아 호">우르미아 호</a>의 남동쪽, 즉 <b>만나</b>의 바로 위쪽에 <a class="new" href="/w/index.php?title=%EC%A7%80%ED%82%A4%EB%A5%B4%ED%83%80&amp;action=edit&amp;redlink=1" title="지키르타 (없는 문서)">지키르타</a>(지도에서 "Zikurtu")가 있다.</div>
