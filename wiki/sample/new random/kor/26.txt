<p><b>베스테르플라테 전투</b>(<span style="font-size: smaller;"><a href="/wiki/%ED%8F%B4%EB%9E%80%EB%93%9C%EC%96%B4" title="폴란드어">폴란드어</a>:</span> <span lang="pl" xml:lang="pl">Oblężenie Westerplatte</span>, <span style="font-size: smaller;"><a href="/wiki/%EC%98%81%EC%96%B4" title="영어">영어</a>:</span> <span lang="en" xml:lang="en">Battle of Westerplatte</span>)는 <a href="/wiki/%ED%8F%B4%EB%9E%80%EB%93%9C_%EC%B9%A8%EA%B3%B5" title="폴란드 침공">폴란드 침공</a> 당시에 이루어진 전투로 <a href="/wiki/%EC%A0%9C2%EC%B0%A8_%EC%84%B8%EA%B3%84_%EB%8C%80%EC%A0%84" title="제2차 세계 대전">제2차 세계 대전</a> 최초의 전투이다. 이 전투는 1939년 9월 1일 <a href="/wiki/%EB%82%98%EC%B9%98_%EB%8F%85%EC%9D%BC" title="나치 독일">나치 독일</a> <a class="mw-redirect" href="/wiki/%ED%81%AC%EB%A6%AD%EC%8A%A4%EB%A7%88%EB%A6%AC%EB%84%A4" title="크릭스마리네">해군</a>과 육군 병사가 <a href="/wiki/%EB%8B%A8%EC%B9%98%ED%9E%88_%EC%9E%90%EC%9C%A0%EC%8B%9C" title="단치히 자유시">단치히 자유시</a> 부두인 <a href="/wiki/%EB%B2%A0%EC%8A%A4%ED%85%8C%EB%A5%B4%ED%94%8C%EB%9D%BC%ED%85%8C" title="베스테르플라테">베스테르플라테</a> 반도의 폴란드군 보급품 창고(<span style="font-size: smaller;"><a href="/wiki/%ED%8F%B4%EB%9E%80%EB%93%9C%EC%96%B4" title="폴란드어">폴란드어</a>:</span> <span lang="pl" xml:lang="pl">Wojskowa Składnica Tranzytowa</span>, WST)를 공격하면서 시작되었다. 이 창고는 약 200명 미만의 군인들만 경계하고 있었지만 7일 동안 <a href="/wiki/%EA%B8%89%EA%B0%95%ED%95%98%ED%8F%AD%EA%B2%A9%EA%B8%B0" title="급강하폭격기">급강하폭격기</a> 폭격과 같은 대포격을 방어해냈다.</p>
<p>이 전투에서 폴란드군의 압도적인 열세 속에서도 폴란드군은 15명~20명 가량이 사망한 데 비해 독일군은 200명이 사망하는 등 높은 교전비를 이루었다. 여기에 베스테르플라테에서 독일군을 7일동안 막아냈으며 그 이후에도 <a class="new" href="/w/index.php?title=%ED%97%AC_%EC%A0%84%ED%88%AC&amp;action=edit&amp;redlink=1" title="헬 전투 (없는 문서)">헬 전투</a>가 끝난 10월 2일까지 <a href="/wiki/%ED%8F%B4%EB%9E%80%EB%93%9C_%ED%9A%8C%EB%9E%91" title="폴란드 회랑">폴란드 회랑</a> 지역의 저항이 이어졌다.<sup class="reference" id="cite_ref-Fort_Hel_3-0"><a href="#cite_note-Fort_Hel-3">[3]</a></sup> 베스테르플라테 방어전은 성공적으로 독일군의 공세가 진행된 지역의 <a class="mw-redirect" href="/wiki/%ED%8F%B4%EB%9E%80%EB%93%9C_%EC%9C%A1%EA%B5%B0" title="폴란드 육군">폴란드 육군</a>에게 감흥을 주었고, 오늘날까지 폴란드 침공에서 저항의 상징으로 남아 있다. 이 지구는 <a href="/wiki/%ED%8F%B4%EB%9E%80%EB%93%9C%EC%9D%98_%EC%97%AD%EC%82%AC%EA%B8%B0%EB%85%90%EB%AC%BC_%EB%AA%A9%EB%A1%9D" title="폴란드의 역사기념물 목록">폴란드의 역사기념물</a> 중 하나로 2003년 9월 1일 <a href="/wiki/%ED%8F%B4%EB%9E%80%EB%93%9C_%EA%B5%AD%EB%A6%BD%EB%AC%B8%ED%99%94%EC%9C%A0%EC%82%B0%EC%9C%84%EC%9B%90%ED%9A%8C" title="폴란드 국립문화유산위원회">폴란드 국립문화유산위원회</a>가 문화유산으로 지정했다.</p>

<h2><span class="mw-headline" id=".EB.B0.B0.EA.B2.BD">배경</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%B2%A0%EC%8A%A4%ED%85%8C%EB%A5%B4%ED%94%8C%EB%9D%BC%ED%85%8C_%EC%A0%84%ED%88%AC&amp;action=edit&amp;section=1" title="부분 편집: 배경">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p><a href="/wiki/%EA%B5%AD%EC%A0%9C_%EC%97%B0%EB%A7%B9" title="국제 연맹">국제 연맹</a> 고등판무관은 단치히의 베스테르플라테에 폴란드군 82명을 주둔할 수 있게 허가했고, 이 폴란드군은 1921년 단치히 자유시 상원의 동의 하에 설치한 해군 탄약 저장소를 지키고 있었다. 1933년 3월 6일에는 폴란드가 비밀리에 주둔군에 120명의 추가 군대를 파견할려고 했으나 이 비밀작업이 드러나 스캔들로 번저 폴란드는 추가로 파견하고자 했던 계획을 취소했다.<sup class="reference" id="cite_ref-Watt_4-0"><a href="#cite_note-Watt-4">[4]</a></sup><span class="reference plainlinksneverexpand" style="white-space:nowrap"><sup>:315–316</sup></span></p>
<p>폴란드군 보급품 창고는 본토와는 얇은 <a href="/wiki/%EB%8F%8C%EC%A0%9C%EB%B6%80%EB%91%90" title="돌제부두">돌제부두</a>로만 연결된 <a href="/wiki/%EB%8B%A8%EC%B9%98%ED%9E%88_%EC%9E%90%EC%9C%A0%EC%8B%9C" title="단치히 자유시">단치히 자유시</a> 신항구의 <a href="/wiki/%ED%95%AD%EB%A7%8C" title="항만">항만</a> 해협에 위치해 있었다. 베스테르플라테에서 폴란드군이 주둔한 지역은 벽돌벽으로 단치히 영토와 구분되어 있었다. 베스테르플라테에 건설된 <a href="/wiki/%EB%B0%A9%EC%96%B4%EC%8B%9C%EC%84%A4" title="방어시설">방어시설</a>이 별로 없다는 사실은 그렇게 문제시되진 않았다. 실제 <a class="mw-redirect" href="/wiki/%EB%B2%99%EC%BB%A4" title="벙커">벙커</a>나 지하 터널이 없었으며 반도 숲에 숨겨 위장된 5개의 소규모 콘크리트 전초기지와 방어전을 위한 대형 <a href="/wiki/%EC%83%9D%ED%99%9C%EA%B4%80" title="생활관">병영</a>, 요새화 지원을 위한 <a href="/wiki/%EC%B0%B8%ED%98%B8" title="참호">참호</a>와 <a href="/wiki/%EB%B0%94%EB%A6%AC%EC%BC%80%EC%9D%B4%EB%93%9C" title="바리케이드">바리케이드</a> 네트워크가 있었다.<sup class="reference" id="cite_ref-muzeum1939.pl_5-0"><a href="#cite_note-muzeum1939.pl-5">[5]</a></sup> 또한 베스테르플라테의 건물 일부는 콘크리트로 보강했는데, 이 건물들이 7일동안 방어전을 펼칠 수 있도록 도움을 주었다.<sup class="reference" id="cite_ref-Zaloga_1-1"><a href="#cite_note-Zaloga-1">[1]</a></sup><span class="reference plainlinksneverexpand" style="white-space:nowrap"><sup>:54</sup></span></p>

<p>8월 말, 폴란드는 주둔군을 82명에서 210명으로 늘리고, 3인치 야포와 1.5인치 대전차포 2개, 박격포 4개, 기관총 여러대를 두는 등 군사력을 늘렸다. 또한, 소초 (Placówka) 7곳을 세우고 막사와 NCO 빌라 지하에 있는 콘크리트 대피소를 보강했다.<sup class="reference" id="cite_ref-Mann_6-0"><a href="#cite_note-Mann-6">[6]</a></sup><span class="reference plainlinksneverexpand" style="white-space:nowrap"><sup>:11–12</sup></span></p>
<p>1939년 8월 말에는 독일의 <a class="mw-redirect" href="/wiki/%EC%A0%84%EB%85%B8%EA%B8%89%EC%A0%84%ED%95%A8" title="전노급전함">전드레트노트급 전함</a> <a href="/wiki/SMS_%EC%8A%90%EB%A0%88%EC%8A%A4%EB%B9%84%ED%9E%88-%ED%99%80%EC%8A%88%ED%83%80%EC%9D%B8" title="SMS 슐레스비히-홀슈타인">SMS 슐레스비히-홀슈타인</a>이 친선 방문을 이유로 단치히에 도착하여 베스테르플라테에서 150m 정도 떨어진 해협에 정박했다. 선박 갑판에는 8월 26일에 베스테르플라테 공격을 시작하기 위한 <a href="/wiki/%EC%B6%A9%EA%B2%A9%EA%B5%B0" title="충격군">충격군</a>(Stoßtruppen) <a href="/wiki/%EC%A4%91%EB%8C%80" title="중대">중대</a>가 대기하고 있었다. 그러나, 하선 직전에 공격이 취소되었다는 연락을 받았다. 8월 25일 <a href="/wiki/%EC%98%81%EA%B5%AD-%ED%8F%B4%EB%9E%80%EB%93%9C_%EA%B5%B0%EC%82%AC%EB%8F%99%EB%A7%B9" title="영국-폴란드 군사동맹">영국-폴란드 군사동맹</a>이 맺어지고 이탈리아가 <a href="/wiki/%EA%B0%95%EC%B2%A0_%EC%A1%B0%EC%95%BD" title="강철 조약">강철 조약</a>의 준수를 미루겠다고 말하자 <a href="/wiki/%EC%95%84%EB%8F%8C%ED%94%84_%ED%9E%88%ED%8B%80%EB%9F%AC" title="아돌프 히틀러">아돌프 히틀러</a>가 적대 행위 시작을 연기하기로 한 결과였다.<sup class="reference" id="cite_ref-7"><a href="#cite_note-7">[7]</a></sup></p>
<p>독일군은 <a class="mw-redirect" href="/wiki/%ED%94%84%EB%A6%AC%EB%93%9C%EB%A6%AC%ED%9E%88%EA%B2%8C%EC%98%A4%EB%A5%B4%EA%B7%B8_%EC%97%90%EB%B2%84%ED%95%98%EB%A5%B4%ED%8A%B8" title="프리드리히게오르그 에버하르트">프리드리히게오르그 에버하르트</a> 장군이 이끄는 1,500명의 SS 향토방위군(SS-Heimwehr)과 <a href="/wiki/%EC%A4%91%EC%9C%84" title="중위">중위</a> 빌헬름 헤닝센이 이끄는 <a href="/wiki/%ED%95%B4%EB%B3%91%EB%8C%80" title="해병대">해병대</a> 225명이 창고를 공격했다. 총지휘관은 SMS 슐레스비히-홀슈타인에 타 있던 제독 <a href="/wiki/%EA%B5%AC%EC%8A%A4%ED%83%80%ED%94%84_%ED%81%B4%EB%9D%BC%EC%9D%B4%EC%BA%84%ED%94%84" title="구스타프 클라이캄프">구스타프 클라이캄프</a>가 맡았다. 그는 8월 26일 상류 지역 멀리로 배를 옮겼다. 폴란드군 장군 <a href="/wiki/%ED%97%A8%EB%A6%AC%ED%81%AC_%EC%88%98%ED%95%98%EB%A5%B4%EC%8A%A4%ED%82%A4" title="헨리크 수하르스키">헨리크 수하르스키</a>는 주둔군에게 경보를 발령했다.<sup class="reference" id="cite_ref-Mann_6-1"><a href="#cite_note-Mann-6">[6]</a></sup><span class="reference plainlinksneverexpand" style="white-space:nowrap"><sup>:12</sup></span></p>
<h2><span class="mw-headline" id=".EC.A0.84.ED.88.AC">전투</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%B2%A0%EC%8A%A4%ED%85%8C%EB%A5%B4%ED%94%8C%EB%9D%BC%ED%85%8C_%EC%A0%84%ED%88%AC&amp;action=edit&amp;section=3" title="부분 편집: 전투">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>1939년 9월 1일 현지 시간 4시 48분, 전함 슐레스비히-홀슈타인이 폴란드 주둔군 182명과 <a class="mw-redirect" href="/wiki/%EC%98%88%EB%B9%84%EA%B5%B0" title="예비군">예비군</a> 27명이 머무르는 병영을 향해 일제포격을 시작하면서 <a href="/wiki/%ED%8F%B4%EB%9E%80%EB%93%9C_%EC%B9%A8%EA%B3%B5" title="폴란드 침공">폴란드 침공</a>과 함께 제2차 세계 대전이 시작되었다. 수하르스키 장군은 급하게 헬 반도로 "SOS. 지금 포격을 받고 있다"라는 전신을 보냈다.<sup class="reference" id="cite_ref-Mann_6-2"><a href="#cite_note-Mann-6">[6]</a></sup><span class="reference plainlinksneverexpand" style="white-space:nowrap"><sup>:12</sup></span> 경계벽에 포탄이 맞아 구멍이 생기고 남동부 지역의 기름창고에는 불이 붙어 폭발했다. 8분 후, 슐례스비히-홀슈타인 전함에서 중위 빌헬름 헤닝센이 이끄는 해병대가 급습하고 <a href="/wiki/%EB%8F%85%EC%9D%BC_%EA%B5%AD%EB%B0%A9%EA%B5%B0" title="독일 국방군">독일 국방군</a>의 <a href="/wiki/%EA%B3%B5%EB%B3%91" title="공병">공병</a> 3개 소대가 공세를 펼치면서 폴란드 본토와 이어진 철교를 폭파시켰다. 이런 상황에 독일군은 기습을 받아 놀란 폴란드군을 쉽게 이길 것이라고 예상했다.<sup class="reference" id="cite_ref-8"><a href="#cite_note-8">[8]</a></sup> 폴란드군 하사 <a href="/wiki/%EB%B3%B4%EC%9D%B4%EC%8B%9C%EC%97%90%ED%9D%90_%EB%82%98%EC%9D%B4%EC%82%AC%EB%A0%88%ED%81%AC" title="보이시에흐 나이사레크">보이시에흐 나이사레크</a>가 <a href="/wiki/%EA%B8%B0%EA%B4%80%EC%B4%9D" title="기관총">기관총</a>에 맞아 사망하면서 제2차 세계 대전과 베스테르플라테 전투의 최초 사망자가 되었다. 그러나, 포격으로 파괴된 벽돌 벽을 넘은 후에는 독일군이 갑작스런 <a href="/wiki/%EB%A7%A4%EB%B3%B5" title="매복">매복</a> 공격을 당했다. 독일군은 얽히섥힌 <a href="/wiki/%EC%9C%A0%EC%9E%90%EC%B2%A0%EC%84%A0" title="유자철선">유자철선</a>으로 인해 재빠른 기동을 하지 못하면서 은폐한 폴란드군의 일제사격 <a href="/wiki/%EC%82%B4%EC%83%81%EC%A7%80%EB%8C%80" title="살상지대">살상지대</a>에 걸려들게 되었다(일부는 나무에 숨은 <a href="/wiki/%EC%A0%80%EA%B2%A9%EC%88%98" title="저격수">저격수</a>도 있었다고 했지만, 실제로는 없었다). 폴란드군은 <a href="/wiki/%EB%B3%B4%EC%95%88%EA%B2%BD%EC%B0%B0_(%EB%82%98%EC%B9%98_%EB%8F%85%EC%9D%BC)" title="보안경찰 (나치 독일)">보안경찰</a>의 기관총좌를 제압했고, 폴란드군 중위 레온 파예크가 전진하는 독일군에게 강력한 곡사포 포격을 하면서 독일군의 공격을 막아냈다. 항구 건너편 창고 위에 있던 7.62mm 기관총 1개가 배치된 기관총좌도 제압했으나, 독일군 전함이 28번의 포격을 하면서 완전히 파괴되었다.<sup class="reference" id="cite_ref-Mann_6-3"><a href="#cite_note-Mann-6">[6]</a></sup><span class="reference plainlinksneverexpand" style="white-space:nowrap"><sup>:12</sup></span><sup class="reference" id="cite_ref-Wa.C5.84kowicz_9-0"><a href="#cite_note-Wa.C5.84kowicz-9">[9]</a></sup></p>
<blockquote class="toccolours" style="float:none; padding: 10px 15px 10px 15px; display:table;">
<p>"독일군을 포화 속으로 끌어들이자는 전초기지 지휘관의 전술 덕택으로 독일군이 사격 범위 내로 들어오면서 많은 손실을 입히게 했다. 돌출부의 정찰병이 정확하게 좌표를 말하면서 폴란드군이 박격포 포격을 하며 손실을 더 높였다. 1989년 봄~여름에 비밀리에 만들어진 이 방어벽 시스템은 독일군이 한때는 유명한 스파였던 베스테르플라테 공원 근처 진군을 어렵게 만들었다."<sup class="reference" id="cite_ref-10"><a href="#cite_note-10">[10]</a></sup></p>
</blockquote>

<p>6시 22분, 해병대는 큰 손실을 입고 후퇴한다고 배에게 통신을 보냈고, 단치히 경찰은 베스테르플라테 반대편 항구를 장악하러 시도했지만 실패했다.<sup class="reference" id="cite_ref-Mann.2C_C_page_13_11-0"><a href="#cite_note-Mann.2C_C_page_13-11">[11]</a></sup> 이때까지의 사상자는 독일군 50명, 폴란드군 8명이었다. 독일군은 8시 55분 다시 전진을 시도했지만 지뢰, 쓰러진 나무, 철조망, 강력한 포화 등을 만나 격전을 벌였다.<sup class="reference" id="cite_ref-Mann.2C_C_page_13_11-1"><a href="#cite_note-Mann.2C_C_page_13-11">[11]</a></sup> 정오에 SS 군인이 탈영했으며, 헤닝센은 치명적인 부상을 입었다.<sup class="reference" id="cite_ref-Mann.2C_C_page_13_11-2"><a href="#cite_note-Mann.2C_C_page_13-11">[11]</a></sup><sup class="reference" id="cite_ref-12"><a href="#cite_note-12">[12]</a></sup> 초기 공격이 실패한 이후 아침에 다시 시작한 두 번째 공격(280mm 포탄 90발, 170mm 포탄 407발, 88mm 포탄 366발의 포격이 이루어진 이후)은 독일군이 예기치 않게 큰 손실을 입으면서 격퇴당했다. 하지만 결국 폴란드군은 바우 지역(Wał)과 전초 기지의 프롬으로 후퇴했으며, 반도 중앙의 새로운 병영 주변에 방어 기지를 건설했다. 전투 첫날 이후, 폴란드군은 1명이 사망하고 7명이 부상을 입었다(독일군이 포로로 잡고 병원으로 후송된 2명을 포함하여 3명이 나중에 사망했다). 한편, 독일 해군 보병은 16명이 사망하고 120명이 부상을 입어 전체 225명 중 대부분이 부상을 입었다.<sup class="reference" id="cite_ref-13"><a href="#cite_note-13">[13]</a></sup> 독일군의 손실은 만약 폴란드 사령관 <a href="/wiki/%ED%97%A8%EB%A6%AC%ED%81%AC_%EC%88%98%ED%95%98%EB%A5%B4%EC%8A%A4%ED%82%A4" title="헨리크 수하르스키">헨리크 수하르스키</a>가 탄약을 절약하기 위해 박격포 사격을 중지시키고 일제사격 후 포격하라는 명령을 하지 않았더라면 더 늘어났을 것이다(이 때문에 다음 날 박격포가 파괴되었을때 박격포탄 860개 중 104개만 쓰이고 나머지는 무용지물이 되고 말았다).</p>

<p>다음 날, 반도에 210mm <a href="/wiki/%EA%B3%A1%EC%82%AC%ED%8F%AC" title="곡사포">곡사포</a>를 포함한 독일 해군의 중포병이 포격을 시작하면서 <a href="/wiki/%EC%A0%9C1%EC%B0%A8_%EC%84%B8%EA%B3%84_%EB%8C%80%EC%A0%84" title="제1차 세계 대전">제1차 세계 대전</a> 식의 전투 스타일로 바뀌었다. 에버하르트는 사령관 페도르 폰 보크에게 지상 공격이 불가능하다고 설득했다. 9월 2일 <a class="mw-redirect" href="/wiki/Ju-87" title="Ju-87">융커스 Ju-87</a> 스튜카 급강하폭격기 60기가 폴란드 박격포에 대해 2번의 공습을 펼치면서 포대 5개가 500kg 포탄에 명중하여 파괴되었고 적어도 폴란드군 8명 이상이 사망했다. 또한, 이 공습으로 인한 연기가 베스테르플라테 전 영역에 뒤덮이게 되었고, 유일한 라디오 방송국과 식량 보급 기지를 파괴했다.<sup class="reference" id="cite_ref-Mann.2C_C_page_13_11-3"><a href="#cite_note-Mann.2C_C_page_13-11">[11]</a></sup> 독일 관측병들은 이러한 폭격에서 아무도 살 수 없을 것이라고 생각했다.<sup class="reference" id="cite_ref-Wa.C5.84kowicz_9-1"><a href="#cite_note-Wa.C5.84kowicz-9">[9]</a></sup></p>
<p>9월 3일부터 4일까지 밤동안 펼친 독일군의 공세가 격퇴되었다. 9월 4일, 독일 어뢰정 T-196(G196의 이름이 바뀜)이 해변에서 기습공격을 했다.<sup class="reference" id="cite_ref-Mann.2C_C_page_14_14-0"><a href="#cite_note-Mann.2C_C_page_14-14">[14]</a></sup> 폴란드군은 바우 지역은 포기했고, 오직 항구 지역만 북쪽에서 오는 공격을 막을 수 있었다.<sup class="reference" id="cite_ref-Mann.2C_C_page_13_11-4"><a href="#cite_note-Mann.2C_C_page_13-11">[11]</a></sup></p>
<p>9월 5일, 수하르스키가 <a href="/wiki/%EC%A0%84%ED%88%AC%ED%94%BC%EB%A1%9C%EC%A6%9D" title="전투피로증">전투피로증</a>에 걸리면서 베스테르플라테에서 항복을 촉구하는 위원회를 열었다. 하지만, 항복에 반대했던 그의 부지휘관인 <a href="/wiki/%ED%94%84%EB%9E%80%EC%B9%98%EC%85%B0%ED%81%AC_%EB%8F%94%EB%B8%8C%EB%A1%9C%ED%94%84%EC%8A%A4%ED%82%A4" title="프란치셰크 돔브로프스키">프란치셰크 돔브로프스키</a>가 명령권을 이어받았고 항복은 무산되었다.<sup class="reference" id="cite_ref-Mann.2C_C_page_14_14-1"><a href="#cite_note-Mann.2C_C_page_14-14">[14]</a></sup></p>
<p>독일 해군 보병, 단치히 SS, 경찰, 국방군으로 이루어진 몇번의 신중한 공격이 다시 폴란드군에 의해 격퇴되었다. 9월 6일 03시, 독일군은 불타는 열차를 다리를 통해 보냈지만, 겁먹은 기관사가 불붙은 열차를 너무 일찍 분리시키는 바람에 기름탱크에 도달시켜서 엄폐역할을 해주는 숲을 태워버리려던 계획은 실패했다. <sup class="reference" id="cite_ref-Mann.2C_C_page_14_14-2"><a href="#cite_note-Mann.2C_C_page_14-14">[14]</a></sup> 오히려 불타는 기차는 완벽한 방어선이 되었고 독일군은 큰 손실을 입었다.<sup class="reference" id="cite_ref-Mann.2C_C_page_15_15-0"><a href="#cite_note-Mann.2C_C_page_15-15">[15]</a></sup> 오후에 행해진 두 번째 기차 공격 역시 실패했다.</p>
<p>한편, <a href="/wiki/%ED%8F%B4%EC%8A%A4%ED%82%A4%EC%97%90_%EB%9D%BC%EB%94%94%EC%98%A4" title="폴스키에 라디오">폴스키에 라디오</a>는 지속적으로 "베스트플라테 지역은 여전히 전투중"이라는 메시지를 방송했다.<sup class="reference" id="cite_ref-16"><a href="#cite_note-16">[16]</a></sup> 2차 전쟁협의회가 열렸고 장군은 항복하기로 정했다. 이때 쯤 되면 독일군은 바르샤바 외곽까지 진출했고 부상자들 가운데서는 <a href="/wiki/%EA%B4%B4%EC%A0%80" title="괴저">괴저</a>가 나타나기 시작했다.<sup class="reference" id="cite_ref-Mann.2C_C_page_15_15-1"><a href="#cite_note-Mann.2C_C_page_15-15">[15]</a></sup> 9월 7일 0430시 독일군은 강한 포격을 시작했고 이는 0700시까지 계속되었다. 화염방사기는 위병소 2개를 파괴했고 1~4개를 손상시켰다.<sup class="reference" id="cite_ref-Mann.2C_C_page_15_15-2"><a href="#cite_note-Mann.2C_C_page_15-15">[15]</a></sup> 포위된 수비대는 충분한 물과 의약품이 부족했다. WST의 의무관인 미에치스와프 스와비(Mieczysław Słaby)는 부상병에 대한 기본적인 치료를 할 수 없었다.</p>
<p>9시 45분에는 백기가 나타났다. 폴란드의 수비는 많은 독일군을 감동시켜서 독일군 사령관인 프리드리히게오르그 에버하르트는 수하르스키가 <a href="/wiki/%EC%82%AC%EB%B8%94%EB%9D%BC" title="사블라">사블라</a>(szabla, 폴란드 <a href="/wiki/%EC%82%AC%EB%B8%8C%EB%A5%B4" title="사브르">사브르</a>)를 착용하고 항복할 수 있게 했다(나중에는 이 군도를 압수했다).<sup class="reference" id="cite_ref-Mann.2C_C_page_15_15-3"><a href="#cite_note-Mann.2C_C_page_15-15">[15]</a></sup> 동시에 폴란드 통신병 카지미에슈 라신스키(Kazimierz Rasiński)는 항복한 이후 독일군이 살해했다. 그는 잔인한 심문까지 받았는데도 독일군에게 라디오 코드를 넘겨주는 것을 거부하다가 끝내 살해당했다.<sup class="reference" id="cite_ref-17"><a href="#cite_note-17">[17]</a></sup></p>
<h2><span class="mw-headline" id=".EC.97.AC.ED.8C.8C">여파</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%B2%A0%EC%8A%A4%ED%85%8C%EB%A5%B4%ED%94%8C%EB%9D%BC%ED%85%8C_%EC%A0%84%ED%88%AC&amp;action=edit&amp;section=4" title="부분 편집: 여파">편집</a><span class="mw-editsection-bracket">]</span></span></h2>

<div class="thumbinner" style="width:182px;"><a class="image" href="/wiki/%ED%8C%8C%EC%9D%BC:Bundesarchiv_Bild_183-1989-516-505,_Westerplatte,_hissen_der_Reichskriegsflagge.jpg"><img alt="" class="thumbimage" data-file-height="800" data-file-width="570" height="253" src="//upload.wikimedia.org/wikipedia/commons/thumb/b/b0/Bundesarchiv_Bild_183-1989-516-505%2C_Westerplatte%2C_hissen_der_Reichskriegsflagge.jpg/180px-Bundesarchiv_Bild_183-1989-516-505%2C_Westerplatte%2C_hissen_der_Reichskriegsflagge.jpg" srcset="//upload.wikimedia.org/wikipedia/commons/thumb/b/b0/Bundesarchiv_Bild_183-1989-516-505%2C_Westerplatte%2C_hissen_der_Reichskriegsflagge.jpg/270px-Bundesarchiv_Bild_183-1989-516-505%2C_Westerplatte%2C_hissen_der_Reichskriegsflagge.jpg 1.5x, //upload.wikimedia.org/wikipedia/commons/thumb/b/b0/Bundesarchiv_Bild_183-1989-516-505%2C_Westerplatte%2C_hissen_der_Reichskriegsflagge.jpg/360px-Bundesarchiv_Bild_183-1989-516-505%2C_Westerplatte%2C_hissen_der_Reichskriegsflagge.jpg 2x" width="180"/></a>

<p>독일군의 사상자는 정확히는 알 수 없지만, 대략 200~300명 정도가 사상했다고 추측하고 있다. 이들 중 일부는 실제로 처음 목표에 너무 가까이 정박한 전함에서 발포한 <a href="/wiki/%EC%95%84%EA%B5%B0_%EC%82%AC%EA%B2%A9" title="아군 사격">아군 사격</a>으로 인한 사상자로 포함되어 있는 것으로 추측된다. 폴란드군의 사상자는 이에 비해 매우 낮아서 15~20명이 사망하고 53명이 부상을 입었다.<sup class="reference" id="cite_ref-18"><a href="#cite_note-18">[18]</a></sup> 1940년에 발견된, <a href="/wiki/%ED%83%88%EC%98%81" title="탈영">탈영</a>을 시도하다가 붙잡혀 동료들에 의해 처형된 듯한 신원미상의 폴란드인 시신 5구가 매장된 곳에 대한 논란이 있다. 전쟁 포로 중 8명은 고문을 당하다 살아남지 못한 것으로 알려져 있다.<sup class="reference" id="cite_ref-19"><a href="#cite_note-19">[19]</a></sup></p>

<p>또한, 폴란드 사령관 <a href="/wiki/%ED%97%A8%EB%A6%AC%ED%81%AC_%EC%88%98%ED%95%98%EB%A5%B4%EC%8A%A4%ED%82%A4" title="헨리크 수하르스키">헨리크 수하르스키</a>와 부사령관이자 제독인 프란치셰크 돔브로프스키에 대한 논란이 있다.<sup class="reference" id="cite_ref-20"><a href="#cite_note-20">[20]</a></sup> 전후까지 살아남아 1946년에 사망한 수하르스키는 소령으로 진급하고 폴란드에서 가장 높은 훈장인 <a href="/wiki/%EB%B9%84%EB%A5%B4%ED%88%AC%ED%8B%B0_%EB%B0%80%EB%A6%AC%ED%83%80%EB%A6%AC" title="비르투티 밀리타리">비르투티 밀리타리</a> 훈장을 받은 것에 대해, 1990년대에 발견한 전투에서 자신의 역할에 대한 알 수 없는 판단으로 최근 매우 논쟁의 대상이 되었다(돔브로프스키의 사망 이후 다른 폴란드 장교들은 이름뿐인 지휘관이 전투 기간 대부분 동안 <a href="/wiki/%EC%85%B8_%EC%87%BC%ED%81%AC" title="셸 쇼크">셸 쇼크</a> 상태였다는 것을 그들의 명예를 위해 숨기기로 맹세했다고 진술했다). 베스테르플라테는 이후 박사 야누쉬 마르샬레츠(Janusz Marszalec)은 <a class="mw-redirect" href="/wiki/%ED%8F%B4%EB%9E%80%EB%93%9C_%EA%B5%AD%EB%A6%BD_%EC%B6%94%EB%AA%A8_%EC%97%B0%EA%B5%AC%EC%86%8C" title="폴란드 국립 추모 연구소">폴란드 국립 추모 연구소</a>에서 다음과 같이 요약했다.<sup class="reference" id="cite_ref-muzeum1939.pl_5-1"><a href="#cite_note-muzeum1939.pl-5">[5]</a></sup></p>
<blockquote class="toccolours" style="float:none; padding: 10px 15px 10px 15px; display:table;">
<p>"베스테르플라테 방어전의 지휘관 헨리크 수하르스키와 프란치셰크 돔브로프스키에 대한 의문 (또는 분쟁)이 있다. 흥미롭게도, 이 논란은 학술적인 논쟁이 일어나지 않기 때문에 역사학자들은 이 논쟁에 끼어들지 않는다. 보통 인터넷에서 벌어지는 이 논쟁은 다양한 미확인 출처 및 점진적이고 일정한 분위기 속에서 열정적인 중심지가 되가고 있다. 그들은 매우 열정적이고 감정적인 상태에 빠져서 이성적인 생각을 하지 못한다. 이 분쟁은 인터넷의 좁은 문에서, 베스테르플라테 방어전에 관한 새로운 "진리"를 알려주겠다는 것을 목표로 한다는 영화 스크립트를 통해 공식적으로 매스 미디어로 펼쳐지게 되었다. 이 프로젝트에 대한 상세한 분석 없이, 1939년 9월 베스테르플라테 방어전에 관한 역사에 대한 지식을 확인했을 때, 공통점이 없음을 강조할 수 있다 ... 분쟁에 관계 없이, 헨리크 수하르스키와 WST의 200명 병사는 그들이 마지막 총알로 방어하기를 원하든지 아닌지 여부에 상관없이, 또는 1939년 9월 1일 슐레스비히-홀슈타인의 포격 12시간 이후 이미 무기를 내려놓았는지 아닌지 고려 여부 없이 기억 속에 남아 있었다."<sup class="reference" id="cite_ref-muzeum1939.pl_5-2"><a href="#cite_note-muzeum1939.pl-5">[5]</a></sup></p>
</blockquote>

<p>총 합쳐서, 베스테르플라테 전투 동안 대략 40~60기의 <a href="/wiki/%EC%9C%B5%EC%BB%A4%EC%8A%A4_Ju_87" title="융커스 Ju 87">융커스 Ju 87</a> 수튜카 폭격기와 기타 <a href="/wiki/%ED%95%98%EC%9D%B8%EC%BC%88_He_51" title="하인켈 He 51">하인켈 He 51</a>, <a href="/wiki/%EC%9C%B5%EC%BB%A4%EC%8A%A4_Ju_52" title="융커스 Ju 52">융커스 Ju 52</a> 폭격기 7기가 이 전투에 참여했다.</p>
<p>독일 지상군은 <a href="/wiki/ADGZ" title="ADGZ">ADGZ</a> 중<a href="/wiki/%EC%9E%A5%EA%B0%91%EC%B0%A8" title="장갑차">장갑차</a> 수대, <a class="mw-redirect" href="/wiki/Flak_38" title="Flak 38">2 cm Flak 30</a> 대공포, <a class="mw-redirect" href="/wiki/PaK_36" title="PaK 36">3.7 cm PaK 36</a> 대전차포, <a class="mw-redirect" href="/wiki/LeFH_18" title="LeFH 18">10.5 cm leFH 18</a> 경곡사포, <a class="mw-redirect" href="/wiki/M%C3%B6rser_18" title="Mörser 18">21 cm Mörser 18</a> 중곡사포 등 포 65문, <a href="/wiki/%EA%B8%B0%EA%B4%80%EC%B4%9D" title="기관총">기관총</a> 100문 이상, 박격포 수십개, <a class="mw-redirect" href="/wiki/%ED%94%8C%EB%9D%BC%EB%A9%98%EB%B2%A0%EB%A5%B4%ED%8D%BC_35" title="플라멘베르퍼 35">플라멘베르퍼 35</a> <a href="/wiki/%ED%99%94%EC%97%BC%EB%B0%A9%EC%82%AC%EA%B8%B0" title="화염방사기">화염방사기</a> 수십 개를 가지고 있었다.<sup class="reference" id="cite_ref-21"><a href="#cite_note-21">[21]</a></sup></p>
<h3><span class="mw-headline" id=".ED.8F.B4.EB.9E.80.EB.93.9C.EA.B5.B0">폴란드군</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%B2%A0%EC%8A%A4%ED%85%8C%EB%A5%B4%ED%94%8C%EB%9D%BC%ED%85%8C_%EC%A0%84%ED%88%AC&amp;action=edit&amp;section=7" title="부분 편집: 폴란드군">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>1939년 8월, 베스테르플라테의 수비대는 전쟁 직전 징집된 군인 182명과 <a class="mw-redirect" href="/wiki/%EC%98%88%EB%B9%84%EA%B5%B0" title="예비군">예비군</a> 민간인 27명, 총 209명으로 구성되어 있었다.<sup class="reference" id="cite_ref-autonazwa4_2-1"><a href="#cite_note-autonazwa4-2">[2]</a></sup></p>
<p>폴란드군 보급품 창고는 <a class="mw-redirect" href="/wiki/Divisional_gun_M1902" title="Divisional gun M1902">76 mm divisional gun M1902</a> 야포 1문,<sup class="reference" id="cite_ref-22"><a href="#cite_note-22">[22]</a></sup> <a class="mw-redirect" href="/wiki/Bofors_37_mm" title="Bofors 37 mm">Bofors 37 mm</a> 대전차포 2문, <a class="mw-redirect" href="/wiki/Brandt_Mle_27/31" title="Brandt Mle 27/31">Brandt Mle 27/31</a> 중박격포 4문으로 무장되어 있었다. 수비군들은 불균형하게 무장되어 있었는데, 가장 무장이 잘 된 쪽은 기관총 41개, <a href="/wiki/%EC%A4%91%EA%B8%B0%EA%B4%80%EC%B4%9D_(%E9%87%8D)" title="중기관총 (重)">중기관총</a>16개 등으로 무장되어 있었다.<sup class="reference" id="cite_ref-autonazwa4_2-2"><a href="#cite_note-autonazwa4-2">[2]</a></sup> 또한, 소총 140개, 권총 40개 이상, 수류탄 1,000개 이상을 가지고 있었다.<sup class="reference" id="cite_ref-23"><a href="#cite_note-23">[23]</a></sup></p>
