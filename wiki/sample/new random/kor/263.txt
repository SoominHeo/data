<p><b>게오르크 에른스트 슈탈</b>(<span style="font-size: smaller;"><a href="/wiki/%EB%8F%85%EC%9D%BC%EC%96%B4" title="독일어">독일어</a>:</span> <span lang="de" xml:lang="de">Georg Ernst Stahl</span>, <a href="/wiki/1659%EB%85%84" title="1659년">1659년</a> <a href="/wiki/10%EC%9B%94_22%EC%9D%BC" title="10월 22일">10월 22일</a> ~ <a href="/wiki/1734%EB%85%84" title="1734년">1734년</a> <a href="/wiki/5%EC%9B%94_14%EC%9D%BC" title="5월 14일">5월 14일</a>)은 <a href="/wiki/%EB%8F%85%EC%9D%BC" title="독일">독일</a>의 <a href="/wiki/%ED%99%94%ED%95%99%EC%9E%90" title="화학자">화학자</a>이자 <a href="/wiki/%EC%9D%98%EC%82%AC" title="의사">의사</a>이다.</p>
<h2><span class="mw-headline" id=".EC.83.9D.EC.95.A0">생애</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EA%B2%8C%EC%98%A4%EB%A5%B4%ED%81%AC_%EC%97%90%EB%A5%B8%EC%8A%A4%ED%8A%B8_%EC%8A%88%ED%83%88&amp;action=edit&amp;section=1" title="부분 편집: 생애">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>슈탈은 <a href="/wiki/1660%EB%85%84" title="1660년">1660년</a> <a href="/wiki/%EC%95%88%EC%8A%A4%EB%B0%94%ED%9D%90" title="안스바흐">안스바흐</a>에서 태어났다. <a href="/wiki/1683%EB%85%84" title="1683년">1683년</a> <a href="/wiki/%EC%98%88%EB%82%98_%EB%8C%80%ED%95%99%EA%B5%90" title="예나 대학교">예나 대학교</a>에서 <a href="/wiki/%EC%9D%98%ED%95%99" title="의학">의학</a> 학위를 받은 그는 <a href="/wiki/1687%EB%85%84" title="1687년">1687년</a> <a class="new" href="/w/index.php?title=%EC%9E%91%EC%84%BC-%EB%B0%94%EC%9D%B4%EB%A7%88%EB%A5%B4_%EA%B3%B5%EC%9E%91_%EC%9A%94%ED%95%9C_%EC%97%90%EB%A5%B8%EC%8A%A4%ED%8A%B8_3%EC%84%B8&amp;action=edit&amp;redlink=1" title="작센-바이마르 공작 요한 에른스트 3세 (없는 문서)">작센-바이마르 공작 요한 에른스트 3세</a>의 주치의가 되었다. <a href="/wiki/1694%EB%85%84" title="1694년">1694년</a>부터 <a href="/wiki/1716%EB%85%84" title="1716년">1716년</a>까지 그는 <a class="mw-redirect" href="/wiki/%ED%95%A0%EB%A0%88_%EB%8C%80%ED%95%99%EA%B5%90" title="할레 대학교">할레 대학교</a>에서 의학 학장 자리를 맡았고, 그 후 <a href="/wiki/%EB%B2%A0%EB%A5%BC%EB%A6%B0" title="베를린">베를린</a>에서 프로이센의 왕 <a class="mw-redirect" href="/wiki/%ED%94%84%EB%A1%9C%EC%9D%B4%EC%84%BC%EC%9D%98_%ED%94%84%EB%A6%AC%EB%93%9C%EB%A6%AC%ED%9E%88_%EB%B9%8C%ED%97%AC%EB%A6%84_1%EC%84%B8" title="프로이센의 프리드리히 빌헬름 1세">프리드리히 빌헬름 1세</a>의 주치의가 되었다. 그는 <a href="/wiki/1734%EB%85%84" title="1734년">1734년</a> 베를린에서 사망하였다.</p>
<h2><span class="mw-headline" id=".EC.97.85.EC.A0.81">업적</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EA%B2%8C%EC%98%A4%EB%A5%B4%ED%81%AC_%EC%97%90%EB%A5%B8%EC%8A%A4%ED%8A%B8_%EC%8A%88%ED%83%88&amp;action=edit&amp;section=2" title="부분 편집: 업적">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>슈탈은 <a href="/wiki/%EC%9A%94%ED%95%9C_%EC%9A%94%EC%95%84%ED%9E%98_%EB%B2%A0%EC%BB%A4" title="요한 요아힘 베커">요한 요아힘 베커</a>와 함께 <a href="/wiki/%ED%94%8C%EB%A1%9C%EC%A7%80%EC%8A%A4%ED%86%A4%EC%84%A4" title="플로지스톤설">플로지스톤설</a>의 기반을 마련한 것으로 유명하다. 또한 그는 150년 후 <a href="/wiki/%EC%9C%A0%EC%8A%A4%ED%88%AC%EC%8A%A4_%ED%8F%B0_%EB%A6%AC%EB%B9%84%ED%9E%88" title="유스투스 폰 리비히">유스투스 폰 리비히</a>가 주장한 <a href="/wiki/%EB%B0%9C%ED%9A%A8" title="발효">발효</a> 이론과 유사한 견해를 주장하기도 하였다. <a href="/wiki/%EC%9D%98%ED%95%99" title="의학">의학</a>에서는 <a class="new" href="/w/index.php?title=%EB%AC%BC%EC%A7%88%EC%A3%BC%EC%9D%98&amp;action=edit&amp;redlink=1" title="물질주의 (없는 문서)">물질주의</a>를 주장한 <a class="new" href="/w/index.php?title=%ED%97%A4%EB%A5%B4%EB%A7%8C_%EB%B3%B4%EC%96%B4%ED%95%98%EB%B8%8C&amp;action=edit&amp;redlink=1" title="헤르만 보어하브 (없는 문서)">헤르만 보어하브</a>나 <a class="new" href="/w/index.php?title=%ED%94%84%EB%A6%AC%EB%93%9C%EB%A6%AC%ED%9E%88_%ED%98%B8%ED%94%84%EB%A7%8C&amp;action=edit&amp;redlink=1" title="프리드리히 호프만 (없는 문서)">프리드리히 호프만</a>과는 달리 <a class="mw-redirect" href="/wiki/%EB%AC%BC%ED%99%9C%EB%A1%A0" title="물활론">물활론</a>적 체계를 주장하였다.</p>
