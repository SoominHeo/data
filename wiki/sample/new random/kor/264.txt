<p><b>와셀 최</b>(영어: Wasel Choi, 1979년 8월 9일 ~ )는 현재 <a href="/wiki/%EC%83%8C%ED%94%84%EB%9E%80%EC%8B%9C%EC%8A%A4%EC%BD%94" title="샌프란시스코">샌프란시스코</a>에 거주하는 예술인, 여행가, 건축가이다.</p>
<p>와셀의 작품은 그의 건축학과 인류학에 대한 연구로 부터 큰 영향을 받았으며, 그는 평소 신비로운 잠재의식으로부터 영감을 받아 작품 활동을 한다. 그의 작품은 선(线)과 면(面)의 조화로 이루어져 있다. 선과 면이 가까웠다 멀어졌다 하며 입체적인 모습을 이룬다.</p>
<p>와셀은 <a class="mw-redirect" href="/wiki/%EC%83%81%ED%95%98%EC%9D%B4" title="상하이">상하이</a>에서 열린 <a href="/wiki/2010%EB%85%84_%EC%84%B8%EA%B3%84_%EB%B0%95%EB%9E%8C%ED%9A%8C" title="2010년 세계 박람회">2010년 세계 박람회</a><sup class="reference" id="cite_ref-expo2010.uaeinteract.com_1-0"><a href="#cite_note-expo2010.uaeinteract.com-1">[1]</a></sup>, 워마드 (영어:World of Music, Arts and Dance)<sup class="reference" id="cite_ref-WOMAD.ORG_2-0"><a href="#cite_note-WOMAD.ORG-2">[2]</a></sup>, 2009년 <a href="/wiki/%EC%B2%B4%EC%BD%94" title="체코">체코</a>에서 열린 빗코비체 스멀트 아트<sup class="reference" id="cite_ref-moravskoslezsky_3-0"><a href="#cite_note-moravskoslezsky-3">[3]</a></sup>, 2009년 에머레츠 팔라스에서 열린 에머라티 익스프레션<sup class="reference" id="cite_ref-flashartonline_4-0"><a href="#cite_note-flashartonline-4">[4]</a></sup>, 2008년 프랑스 아트 페스티벌, 2007년 아트 파리 아부다비 등 개인전과 그룹전시회에 작품을 전시하였으며, 또한 오스트라바, 타만 네가라, <a href="/wiki/%ED%94%84%EB%9D%BC%ED%95%98" title="프라하">프라하</a>, <a href="/wiki/%EB%A9%94%EB%94%94%EB%82%98" title="메디나">메디나</a>, <a href="/wiki/%EC%8B%B1%EA%B0%80%ED%8F%AC%EB%A5%B4" title="싱가포르">싱가포르</a>, 로스 가토스 <a class="mw-redirect" href="/wiki/%EC%BA%98%EB%A6%AC%ED%8F%AC%EB%8B%88%EC%95%84" title="캘리포니아">캘리포니아</a>, <a href="/wiki/%EC%95%84%EB%B6%80%EB%8B%A4%EB%B9%84" title="아부다비">아부다비</a>, <a href="/wiki/%EC%9D%B4%EC%8A%A4%ED%83%84%EB%B6%88" title="이스탄불">이스탄불</a>, <a class="mw-redirect" href="/wiki/%EC%B0%BD%EC%B6%98%EC%8B%9C" title="창춘시">창춘시</a>, <a class="mw-redirect" href="/wiki/%EB%B2%A0%EC%9D%B4%EC%A7%95" title="베이징">베이징</a>, 포코트 <a href="/wiki/%EC%BC%80%EB%83%90" title="케냐">케냐</a>, <a href="/wiki/%ED%95%A8%EB%B6%80%EB%A5%B4%ED%81%AC" title="함부르크">함부르크</a>, <a href="/wiki/%EB%B0%94%EB%A0%88%EC%9D%B8" title="바레인">바레인</a>, <a class="mw-disambig" href="/wiki/%EB%A1%9C%ED%85%90%EB%B6%80%EB%A5%B4%ED%81%AC" title="로텐부르크">로텐부르크</a>, <a href="/wiki/%ED%99%88%EC%8A%A4" title="홈스">홈스</a>, <a href="/wiki/%EB%93%9C%EB%A0%88%EC%8A%A4%EB%8D%B4" title="드레스덴">드레스덴</a>, <a href="/wiki/%EB%A3%B9%EC%95%8C%ED%95%A0%EB%A6%AC_%EC%82%AC%EB%A7%89" title="룹알할리 사막">룹알할리 사막</a>, <a href="/wiki/%EC%BF%A0%EC%95%8C%EB%9D%BC%EB%A3%B8%ED%91%B8%EB%A5%B4" title="쿠알라룸푸르">쿠알라룸푸르</a>, <a href="/wiki/%EA%B0%80%EB%A5%B4%EB%AF%B8%EC%8A%88%ED%8C%8C%EB%A5%B4%ED%85%90%ED%82%A4%EB%A5%B4%ED%97%A8" title="가르미슈파르텐키르헨">가르미슈파르텐키르헨</a> 등 세계 여러도시를 여행하고 다른 문화의 사람들과 교류하며 작품을 만들고, 체험을 기록하며 인류학을 공부 하였다. 그는 <a href="/wiki/%EB%89%B4%EC%9A%95_%EB%8C%80%ED%95%99%EA%B5%90" title="뉴욕 대학교">뉴욕 대학교</a>와 드레스덴 아타데미 미대에서 강의와 워크샵도 진행했다.<sup class="reference" id="cite_ref-Wasel_ART_5-0"><a href="#cite_note-Wasel_ART-5">[5]</a></sup></p>
<p>와셀은 니산 무라노, <a href="/wiki/%ED%8E%98%EB%9D%BC%EB%A6%AC" title="페라리">페라리</a> <a href="/wiki/%ED%8F%AC%EB%AE%AC%EB%9F%AC_%EC%9B%90" title="포뮬러 원">포뮬러 원</a> 레이서카<sup class="reference" id="cite_ref-The_National_6-0"><a href="#cite_note-The_National-6">[6]</a></sup>, 그리고 1959년형 <a href="/wiki/%EC%89%90%EB%B3%B4%EB%A0%88_%EC%9E%84%ED%8C%94%EB%9D%BC" title="쉐보레 임팔라">쉐보레 임팔라</a><sup class="reference" id="cite_ref-Abu_Dhabi_Tourism_Authority_7-0"><a href="#cite_note-Abu_Dhabi_Tourism_Authority-7">[7]</a></sup> 등 아트카 작품을 성공적으로 끝마쳤으며 2009년엔 초고층 건물 <a href="/wiki/%EB%B6%80%EB%A5%B4%EC%A6%88_%ED%95%A0%EB%A6%AC%ED%8C%8C" title="부르즈 할리파">부르즈 할리파</a> (높이 828m) 84층에 작품을 설치하는 코미션을 받아 성공적으로 마쳤다.<sup class="reference" id="cite_ref-khaleejtimes_8-0"><a href="#cite_note-khaleejtimes-8">[8]</a></sup> 2009년부터 2013년까지 동남아시아에서 살고자 했던 꿈을 이룬후 현재 배우자의 고향 샌프란시스코에 정착해 작품 활동 중이다.</p>
