<p><b>기어박스 소프트웨어</b>(<span lang="en" xml:lang="en">Gearbox Software</span>)는 <a href="/wiki/1999%EB%85%84" title="1999년">1999년</a> 1월에 랜디 피치포드, 스테판 바할, 랜돈 몬티고메리, 브라이언 마르텔과 롭 헤이로니무스의 다섯 명에 의해 설립된 <a class="mw-redirect" href="/wiki/%EA%B2%8C%EC%9E%84_%EA%B0%9C%EB%B0%9C%EC%82%AC" title="게임 개발사">게임 개발사</a>이다. 고성능 PC 및 콘솔 게임인 <i><a href="/wiki/%EB%B8%8C%EB%9D%BC%EB%8D%94%EC%8A%A4_%EC%9D%B8_%EC%95%94%EC%8A%A4:_%EB%A1%9C%EB%93%9C_%ED%88%AC_%ED%9E%90_30" title="브라더스 인 암스: 로드 투 힐 30">브라더스 인 암스: 로드 투 힐 30</a></i>, <i><a href="/wiki/%EB%B8%8C%EB%9D%BC%EB%8D%94%EC%8A%A4_%EC%9D%B8_%EC%95%94%EC%8A%A4:_%EC%96%B8%EB%93%9C_%EC%9D%B8_%EB%B8%94%EB%9F%AC%EB%93%9C" title="브라더스 인 암스: 언드 인 블러드">브라더스 인 암스: 언드 인 블러드</a></i>, <i><a class="new" href="/w/index.php?title=%ED%97%A4%EC%9D%BC%EB%A1%9C:_%EC%BB%B4%EB%B2%B3_%EC%97%90%EB%B3%BC%EB%A3%A8%EB%93%9C&amp;action=edit&amp;redlink=1" title="헤일로: 컴벳 에볼루드 (없는 문서)">헤일로: 컴벳 에볼루드</a></i>의 PC판과 <i><a class="mw-disambig" href="/wiki/%ED%95%98%ED%94%84%EB%9D%BC%EC%9D%B4%ED%94%84" title="하프라이프">하프라이프</a></i>의 확장팩 <i><a href="/wiki/%ED%95%98%ED%94%84%EB%9D%BC%EC%9D%B4%ED%94%84:_%EC%96%B4%ED%8F%AC%EC%A7%95_%ED%8F%AC%EC%8A%A4" title="하프라이프: 어포징 포스">어포징 포스</a></i>와 <i><a href="/wiki/%ED%95%98%ED%94%84%EB%9D%BC%EC%9D%B4%ED%94%84:_%EB%B8%94%EB%A3%A8_%EC%89%AC%ED%94%84%ED%8A%B8" title="하프라이프: 블루 쉬프트">블루 쉬프트</a></i>로 잘 알려져 있다.</p>
<p>스튜디오는 <a href="/wiki/%ED%85%8D%EC%82%AC%EC%8A%A4_%EC%A3%BC" title="텍사스 주">텍사스 주</a>의 <a class="mw-redirect" href="/wiki/%ED%94%8C%EB%9D%BC%EB%85%B8" title="플라노">플라노</a>에 위치해 있으며, 현재 170명이 넘어가는 개발자와 서포트 종업원이 있다. 기어박스는 <a href="/wiki/%EC%9C%A0%EB%B9%84%EC%86%8C%ED%94%84%ED%8A%B8" title="유비소프트">유비소프트</a>, <a class="new" href="/w/index.php?title=%EC%84%B8%EA%B8%B0_%EC%84%B8%EB%AF%B8_%ED%99%80%EB%94%A9%EC%8A%A4&amp;action=edit&amp;redlink=1" title="세기 세미 홀딩스 (없는 문서)">세가</a>, <a class="new" href="/w/index.php?title=%EB%A7%88%EC%9D%B4%ED%81%AC%EB%A1%9C%EC%86%8C%ED%94%84%ED%8A%B8_%EA%B2%8C%EC%9E%84_%EC%8A%A4%ED%8A%9C%EB%94%94%EC%96%B4%EC%8A%A4&amp;action=edit&amp;redlink=1" title="마이크로소프트 게임 스튜디어스 (없는 문서)">마이크로소프트 게임 스튜디어스</a>, <a href="/wiki/%EC%9D%BC%EB%A0%89%ED%8A%B8%EB%A1%9C%EB%8B%89_%EC%95%84%EC%B8%A0" title="일렉트로닉 아츠">일렉트로닉 아츠</a>, <a class="new" href="/w/index.php?title=%EC%97%91%ED%8B%B0%EB%B9%84%EC%A0%84&amp;action=edit&amp;redlink=1" title="엑티비전 (없는 문서)">엑티비전</a>과 <a class="mw-redirect" href="/wiki/%EB%B9%84%EB%B2%A4%EB%94%94_%EA%B2%8C%EC%9E%84%EC%8A%A4" title="비벤디 게임스">비벤디 게임스</a>, 그리고 그외 다른 스튜디오와 파트너로서 <a class="mw-disambig" href="/wiki/%EB%B2%88%EC%A7%80" title="번지">번지</a>, <a href="/wiki/%EC%97%90%ED%94%BD_%EA%B2%8C%EC%9E%84%EC%8A%A4" title="에픽 게임스">에픽 게임스</a>, <a class="new" href="/w/index.php?title=%EB%B2%A8%EB%B8%8C_%EC%86%8C%ED%94%84%ED%8A%B8%EC%9B%A8%EC%96%B4&amp;action=edit&amp;redlink=1" title="벨브 소프트웨어 (없는 문서)">벨브 소프트웨어</a>, <a href="/wiki/%EA%B2%8C%EC%9E%84%EB%A1%9C%ED%94%84%ED%8A%B8" title="게임로프트">게임로프트</a>, <a class="new" href="/w/index.php?title=%EB%A6%AC%EC%B6%94%EC%96%BC_%EC%97%94%ED%84%B0%ED%85%8C%EC%9D%B8%EB%A8%BC%ED%8A%B8&amp;action=edit&amp;redlink=1" title="리추얼 엔터테인먼트 (없는 문서)">리추얼 엔터테인먼트</a>, <a class="new" href="/w/index.php?title=%EB%8D%B0%EB%A5%B4%EB%8B%88%EC%9A%B0%EA%B7%B8_%EC%8A%A4%ED%8A%9C%EB%94%94%EC%96%B4%EC%8A%A4&amp;action=edit&amp;redlink=1" title="데르니우그 스튜디어스 (없는 문서)">데르니우그 스튜디어스</a>와 일하였다.</p>
