<p><b>테오도리쿠스 대왕</b>(Theodoric the Great, <a href="/wiki/454%EB%85%84" title="454년">454년</a> ~ <a href="/wiki/526%EB%85%84" title="526년">526년</a> <a href="/wiki/8%EC%9B%94_30%EC%9D%BC" title="8월 30일">8월 30일</a>, 재위 <a href="/wiki/488%EB%85%84" title="488년">488년</a> ~ <a href="/wiki/526%EB%85%84" title="526년">526년</a>)은 <a href="/wiki/%EB%8F%99%EA%B3%A0%ED%8A%B8_%EC%99%95%EA%B5%AD" title="동고트 왕국">동고트 왕국</a>의 초대 국왕이며 <a href="/wiki/%EB%A1%9C%EB%A7%88_%EC%A0%9C%EA%B5%AD" title="로마 제국">로마 제국</a>의 군인이자, <a href="/wiki/%EC%9D%B4%ED%83%88%EB%A6%AC%EC%95%84%EC%9D%98_%EA%B5%B0%EC%A3%BC" title="이탈리아의 군주">이탈리아의 군주</a>였다. <a href="/wiki/%EA%B3%A0%ED%8A%B8%EC%96%B4" title="고트어">고트어</a> 이름은 튜다라익스(Thiudareiks)로 "백성들의 왕" 이라는 의미이다.</p>
<h2><span class="mw-headline" id=".EC.83.9D.EC.95.A0">생애</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%ED%85%8C%EC%98%A4%EB%8F%84%EB%A6%AC%EC%BF%A0%EC%8A%A4_%EB%8C%80%EC%99%95&amp;action=edit&amp;section=1" title="부분 편집: 생애">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p><a href="/wiki/454%EB%85%84" title="454년">454년</a>, 동고트족이 <a href="/wiki/%ED%9B%88%EC%A1%B1" title="훈족">훈족</a>들에게 점령된 지 1년째 되던 해에 튜다미르(Thiudamir) 왕의 아들로 태어났다. 테오도리쿠스는 튜다미르와 <a class="mw-redirect" href="/wiki/%EB%8F%99%EB%A1%9C%EB%A7%88_%EC%A0%9C%EA%B5%AD" title="동로마 제국">동로마 제국</a> 사이의 협정의 결과로 <a href="/wiki/%EC%BD%98%EC%8A%A4%ED%83%84%ED%8B%B0%EB%85%B8%ED%8F%B4%EB%A6%AC%EC%8A%A4" title="콘스탄티노폴리스">콘스탄티노폴리스</a>에서 볼모로 유년기를 보냈다.</p>
<p>테오도리쿠스는 콘스탄티노폴리스에서 지내며 <a class="mw-redirect" href="/wiki/%EB%8F%99%EB%A1%9C%EB%A7%88_%EC%A0%9C%EA%B5%AD" title="동로마 제국">동로마 제국</a>의 정치와 군사에 대해 많은 것을 배웠다. <a href="/wiki/%EB%A0%88%EC%98%A4_1%EC%84%B8_(%EB%B9%84%EC%9E%94%ED%8B%B0%EC%9B%80_%EC%A0%9C%EA%B5%AD)" title="레오 1세 (비잔티움 제국)">레오 1세</a>와 <a href="/wiki/%EC%A0%9C%EB%85%BC_(%ED%99%A9%EC%A0%9C)" title="제논 (황제)">제논</a>의 호의를 얻어 <a href="/wiki/483%EB%85%84" title="483년">483년</a> 마기스테르 밀리툼(Magister Militum, 군사 대장)가 되고 1년 뒤에는 <a href="/wiki/%EC%A7%91%EC%A0%95%EA%B4%80" title="집정관">집정관</a>이 되었다. 이후 이십대 초반을 다른 동고트족과 보냈으며 <a href="/wiki/488%EB%85%84" title="488년">488년</a> 왕이 되었다.</p>
<p>이 무렵 동고트족은 동로마 제국 내에 로마의 <a href="/wiki/%ED%8F%AC%EC%9D%B4%EB%8D%B0%EB%9D%BC%ED%8B%B0" title="포이데라티">포이데라티</a>로 거주하고 있었으나 점점 통제하기 힘들어지고 있었다. 테오데릭이 왕이 된 이후 황제 제논과 협정을 맺어 테오도리쿠스와 동고트족이 <a href="/wiki/476%EB%85%84" title="476년">476년</a> <a href="/wiki/%EB%A1%9C%EB%A7%88" title="로마">로마</a>를 점령한 <a href="/wiki/%EC%98%A4%EB%8F%84%EC%95%84%EC%BC%80%EB%A5%B4" title="오도아케르">오도아케르</a>를 몰아내는 데 협조를 받기로 하고 오도아케르의 왕국을 침략하게 된다.</p>
<p>테오도리쿠스는 <a href="/wiki/488%EB%85%84" title="488년">488년</a> 군대를 끌고 <a href="/wiki/%EC%9D%B4%ED%83%88%EB%A6%AC%EC%95%84" title="이탈리아">이탈리아</a>에 도착, <a href="/wiki/489%EB%85%84" title="489년">489년</a> <a class="new" href="/w/index.php?title=%EC%9D%B4%EC%86%90%EC%A1%B0%EC%9D%98_%EC%A0%84%ED%88%AC_(489%EB%85%84)&amp;action=edit&amp;redlink=1" title="이손조의 전투 (489년) (없는 문서)">이손조의 전투</a>와 <a class="new" href="/w/index.php?title=%EB%B0%80%EB%9D%BC%EB%85%B8%EC%9D%98_%EC%A0%84%ED%88%AC_(489%EB%85%84)&amp;action=edit&amp;redlink=1" title="밀라노의 전투 (489년) (없는 문서)">밀라노의 전투</a>와 <a href="/wiki/489%EB%85%84" title="489년">489년</a> <a class="new" href="/w/index.php?title=%EC%95%84%EB%8B%A4%EC%9D%98_%EC%A0%84%ED%88%AC_(489%EB%85%84)&amp;action=edit&amp;redlink=1" title="아다의 전투 (489년) (없는 문서)">아다의 전투</a>에서 승리를 거두었다. <a href="/wiki/493%EB%85%84" title="493년">493년</a> 테오도리쿠스는 <a href="/wiki/%EB%9D%BC%EB%B2%A4%EB%82%98" title="라벤나">라벤나</a>를 점령하고 오도아케르는 항복하나 테오도리쿠스의 손에 살해되었다.</p>
<p>오도아케르와 같이 테오도리쿠스는 공식적으로는 <a href="/wiki/%EC%BD%98%EC%8A%A4%ED%83%84%ED%8B%B0%EB%85%B8%ED%8F%B4%EB%A6%AC%EC%8A%A4" title="콘스탄티노폴리스">콘스탄티노폴리스</a>에 있는 황제의 총독이었으나 실질적으로 테오도리쿠스와 로마 황제의 교섭은 동등한 상황에서 진행되었고, 로마 황제의 간섭은 크지 않았다. 하지만 오도아케르와 달리 테오도리쿠스는 왕국 내의 로마 시민들을 로마 법대로 다스린다는 약속을 지켰다. 고트족은 전통적인 법률과 관습으로 다스려졌다.</p>
<p>테오도리쿠스 대왕은 <a class="mw-redirect" href="/wiki/%ED%94%84%EB%9E%91%ED%81%AC%EC%A1%B1" title="프랑크족">프랑크족</a>의 왕 <a href="/wiki/%ED%81%B4%EB%A1%9C%EB%B9%84%EC%8A%A4_1%EC%84%B8" title="클로비스 1세">클로비스 (Clovis) 1세</a>의 여동생 <a class="new" href="/w/index.php?title=%EC%95%84%EC%9A%B0%EB%8F%84%ED%94%8C%EB%A0%88%EB%8B%A4&amp;action=edit&amp;redlink=1" title="아우도플레다 (없는 문서)">아우도플레다</a>(Audofleda)를 왕비로 맞아들여 결혼동맹을 맺었으며, <a class="mw-redirect" href="/wiki/%EC%84%9C%EA%B3%A0%ED%8A%B8%EC%A1%B1" title="서고트족">서고트족</a>, <a href="/wiki/%EB%B0%98%EB%8B%AC%EC%A1%B1" title="반달족">반달족</a>, <a class="mw-redirect" href="/wiki/%EB%B6%80%EB%A5%B4%EA%B5%B0%ED%8A%B8%EC%A1%B1" title="부르군트족">부르군트족</a> 왕들과도 동맹을 맺었다. 클로비스 1세는 <a href="/wiki/506%EB%85%84" title="506년">506년</a>과 <a href="/wiki/523%EB%85%84" title="523년">523년</a> 테오도리쿠스와 전쟁을 벌였다.</p>
<p>제위기간 대부분동안 테오도리쿠스 대왕은 형식적으로는 동고트족과 서고트족 모두의 왕이었으며 <a href="/wiki/505%EB%85%84" title="505년">505년</a>경 어린 서고트족 왕의 섭정이 되었다. 테오도리쿠스는 <a href="/wiki/507" title="507">507</a>년 프랑크족에게 서고트 영토인 <a class="mw-redirect" href="/wiki/%EC%95%84%ED%80%B4%ED%83%80%EB%8B%88%EC%95%84" title="아퀴타니아">아퀴타니아</a>를 빼앗겼으나 그 외에는 대부분의 외침을 격퇴했고, 반달족 왕 <a class="new" href="/w/index.php?title=%ED%8A%B8%EB%9D%BC%EC%82%AC%EB%AC%B8%ED%8A%B8&amp;action=edit&amp;redlink=1" title="트라사문트 (없는 문서)">트라사문트</a>(Thrasamund)를 격퇴해 약탈을 막았다.</p>
<p>테오도리쿠스는 <a class="mw-redirect" href="/wiki/%EC%95%84%EB%A6%AC%EC%9A%B0%EC%8A%A4%EC%A3%BC%EC%9D%98" title="아리우스주의">아리우스파</a> 신자였으며, 이 점 때문에 재위 말 로마 황제 <a href="/wiki/%EC%9C%A0%EC%8A%A4%ED%8B%B0%EB%8B%88%EC%95%84%EB%88%84%EC%8A%A4_1%EC%84%B8" title="유스티니아누스 1세">유스티니아누스 1세</a>와 마찰이 생겼다. 양국의 관계는 나빠지기 시작했으나 테오도리쿠스가 살아있는 동안은 충돌이 없었다.</p>
<p>테오도리쿠스 대왕은 <a href="/wiki/%EB%9D%BC%EB%B2%A4%EB%82%98" title="라벤나">라벤나</a>에 매장되었으며, 사후 딸 <a href="/wiki/%EC%95%84%EB%A7%90%EB%9D%BC%EC%88%9C%ED%83%80" title="아말라순타">아말라순타</a>(Amalasuntha)가 손자 <a href="/wiki/%EC%95%84%ED%83%88%EB%9D%BC%EB%A6%AD" title="아탈라릭">아탈라릭</a>(Athalaric)을 대신해 섭정이 되었다. <a class="mw-redirect" href="/wiki/%ED%94%84%EB%9E%91%ED%81%AC%EC%9D%98_%EC%99%95" title="프랑크의 왕">프랑크의 군주</a> <a href="/wiki/%ED%9E%90%EB%8D%B0%EB%A6%AC%ED%9E%88_1%EC%84%B8" title="힐데리히 1세">힐데리히 1세</a>(Childeric I)의 딸이자 <a href="/wiki/%ED%81%B4%EB%A1%9C%EB%B9%84%EC%8A%A4_1%EC%84%B8" title="클로비스 1세">클로비스 1세</a>의 누이인 <a class="new" href="/w/index.php?title=%EC%95%84%EC%9A%B0%EB%8F%84%ED%94%84%EB%A0%88%EB%8B%A4&amp;action=edit&amp;redlink=1" title="아우도프레다 (없는 문서)">아우도프레다</a>(Audefleda)와 결혼하여 <a href="/wiki/%EC%95%84%EB%A7%90%EB%9D%BC%EC%88%9C%ED%83%80" title="아말라순타">아말라순타</a> 등을 두었다.</p>
