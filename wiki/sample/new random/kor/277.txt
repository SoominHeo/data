<h1 class="firstHeading" id="firstHeading" lang="ko">블랜차드의 성전환증 유형론</h1>
<div class="mw-body-content" id="bodyContent">
<div id="siteSub">위키백과, 우리 모두의 백과사전.</div>
<div id="contentSub"><span class="mw-redirectedfrom">(<a class="mw-redirect" href="/w/index.php?title=%EC%98%A4%ED%86%A0%EA%B8%B0%EB%8B%88%EC%97%90%ED%95%84%EB%A6%AC%EC%95%84&amp;redirect=no" title="오토기니에필리아">오토기니에필리아</a>에서 넘어옴)</span></div>
<div class="mw-jump" id="jump-to-nav">
					이동:					<a href="#mw-head">둘러보기</a>, 					<a href="#p-search">검색</a>
</div>
<div class="mw-content-ltr" dir="ltr" id="mw-content-text" lang="ko"><script>function mfTempOpenSection(id){var block=document.getElementById("mf-section-"+id);block.className+=" open-block";block.previousSibling.className+=" open-block";}</script><p><b>블랜차드의 성전환증 유형론</b>(<span lang="en" xml:lang="en">Blanchard's transsexualism typology</span>) 또는 <b>블랜차드 자기여성애 이론</b>(<span lang="en" xml:lang="en">Blanchard autogynephilia theory; <b>BAT</b></span>)는 1980년대부터 1990년대까지 <a class="new" href="/w/index.php?title=%EB%A0%88%EC%9D%B4_%EB%B8%94%EB%9E%9C%EC%B0%A8%EB%93%9C&amp;action=edit&amp;redlink=1" title="레이 블랜차드 (없는 문서)">레이 블랜차드</a>가 <a class="new" href="/w/index.php?title=%EC%BF%A0%EB%A5%B4%ED%8A%B8_%ED%94%84%EB%A1%9C%EC%9D%B8%ED%8A%B8&amp;action=edit&amp;redlink=1" title="쿠르트 프로인트 (없는 문서)">쿠르트 프로인트</a>의 연구에 기반해 고안한 <a href="/wiki/%ED%8A%B8%EB%9E%9C%EC%8A%A4%EC%97%AC%EC%84%B1" title="트랜스여성">트랜스여성</a> <a href="/wiki/%EC%84%B1%EC%A0%84%ED%99%98%EC%A6%9D" title="성전환증">성전환자</a> 분류법이다. 블랜차드는 트랜스여성을 두 개의 집단으로 나누었다. 하나는 <a href="/wiki/%EB%8F%99%EC%84%B1%EC%95%A0%EC%84%B1%EC%A0%84%ED%99%98%EC%A6%9D" title="동성애성전환증">동성애성전환증</a>으로, 블랜차드에 따르면 동성애성전환자는 남성에게 호감 및 성애를 느끼며 그것을 위해 <a href="/wiki/%EC%84%B1%EC%A0%84%ED%99%98_%EC%88%98%EC%88%A0" title="성전환 수술">성전환 수술</a>을 원한다. 다른 한 집단은 "자기여성애성전환증"으로, 블랜차드에 따르면 이 부류는 자신이 여성의 몸을 가진다는 생각을 통해 성적 흥분을 느끼는 부류이다. 이 분류는 학계에서 반복적인 논쟁의 대상이 되고 있다.<sup class="reference" id="cite_ref-Bancroft2009_1-0"><a href="#cite_note-Bancroft2009-1">[1]</a></sup></p>
<p>이 이론의 지지자로는 <a class="new" href="/w/index.php?title=%EC%95%A4_%EB%A1%9C%EB%A0%8C%EC%8A%A4&amp;action=edit&amp;redlink=1" title="앤 로렌스 (없는 문서)">앤 로렌스</a>, <a class="new" href="/w/index.php?title=%EC%A1%B4_%EB%A7%88%EC%9D%B4%ED%81%B4_%EB%B2%A0%EC%9D%BC%EB%A6%AC&amp;action=edit&amp;redlink=1" title="존 마이클 베일리 (없는 문서)">존 마이클 베일리</a>, <a class="new" href="/w/index.php?title=%EC%A0%9C%EC%9E%84%EC%8A%A4_%EC%B9%B8%ED%86%A0%EC%96%B4&amp;action=edit&amp;redlink=1" title="제임스 칸토어 (없는 문서)">제임스 칸토어</a> 등이 있다. 이들은 동성애성전환자와 자기여성애성전환자 사이에는 성애, <a class="mw-redirect" href="/wiki/%EC%A0%A0%EB%8D%94_%ED%8A%B8%EB%9E%9C%EC%A7%80%EC%85%98" title="젠더 트랜지션">젠더 트랜지션</a>이 이루어지는 나이, 인종, <a class="mw-redirect" href="/wiki/%EC%A7%80%EB%8A%A5%EC%A7%80%EC%88%98" title="지능지수">지능지수</a>, 페티시즘, <a href="/wiki/%EC%8B%A0%EC%B2%B4_%EA%B0%9C%EC%A1%B0" title="신체 개조">신체 개조</a>가 이루어지는 정도 등에 있어 유의미한 차이가 있다고 주장한다. 블랜차드 이론에 따르면 동성애성전환자는 비교적 빠르게, 대개 서른 줄 곶을 넘기 전에 트랜지션을 시작한다.<sup class="reference" id="cite_ref-onesixtytwo_2-0"><a href="#cite_note-onesixtytwo-2">[2]</a></sup> 동성애성전환자는 빈곤층, 비백인, 이민자 등의 배경에서 많이 발생하며,<sup class="reference" id="cite_ref-3"><a href="#cite_note-3">[3]</a></sup> 상대적으로 지능지수가 낮고,<sup class="reference" id="cite_ref-4"><a href="#cite_note-4">[4]</a></sup> 오로지 남성에게만 성애를 느낀다. 한편 자기여성애성전환자는 여성에게 성애를 느낄 가능성이 높지만 여성에게만 느끼는 것은 아니며, <a href="/wiki/%EB%AC%B4%EC%84%B1%EC%95%A0" title="무성애">무성애자</a>일 수도 있다.<sup class="reference" id="cite_ref-onesixtytwo_2-1"><a href="#cite_note-onesixtytwo-2">[2]</a></sup> 이들은 페티시즘적이고 <a href="/wiki/%EC%84%B1%EB%8F%84%EC%B0%A9" title="성도착">성도착</a>적인 성적 흥분을 나타낸다.<sup class="reference" id="cite_ref-5"><a href="#cite_note-5">[5]</a></sup></p>
<p>이 이론의 비판자로는 <a class="new" href="/w/index.php?title=%EC%B0%B0%EC%8A%A4_%EC%95%8C%EB%A0%8C_%EB%AA%A8%EC%84%B8%EB%A5%B4&amp;action=edit&amp;redlink=1" title="찰스 알렌 모세르 (없는 문서)">찰스 알렌 모세르</a>, <a class="new" href="/w/index.php?title=%EC%A4%84%EB%A6%AC%EC%95%84_%EC%84%B8%EB%9D%BC%EB%85%B8&amp;action=edit&amp;redlink=1" title="줄리아 세라노 (없는 문서)">줄리아 세라노</a>, <a class="new" href="/w/index.php?title=%EC%A0%9C%EC%9D%B4%EB%AF%B8_%EB%B9%8C%EB%A0%88&amp;action=edit&amp;redlink=1" title="제이미 빌레 (없는 문서)">제이미 빌레</a>, <a class="new" href="/w/index.php?title=%EB%9E%98%EB%A6%AC_%EB%84%88%ED%8A%B8%EB%B8%8C%EB%A1%9C%ED%81%AC&amp;action=edit&amp;redlink=1" title="래리 너트브로크 (없는 문서)">래리 너트브로크</a>, <a class="new" href="/w/index.php?title=%EC%A1%B4_%EB%B0%B4%ED%81%AC%EB%A1%9C%ED%94%84%ED%8A%B8&amp;action=edit&amp;redlink=1" title="존 밴크로프트 (없는 문서)">존 밴크로프트</a> 등이 있다. 이들은 이 이론이 트랜스여성을 제대로 설명할 수 없으며, <a href="/wiki/%EC%84%B1%EC%A0%95%EC%B2%B4%EC%84%B1" title="성정체성">성정체성</a>을 성적지향의 문제로 축소시킨다고 주장한다.</p>
<p><a href="/wiki/%ED%8A%B8%EB%9E%9C%EC%8A%A4%EC%A0%A0%EB%8D%94" title="트랜스젠더">트랜스젠더</a> 공동체는 블랜차드 이론을 싫어한다. 이러한 비난여론은 2003년 베일리의 저서 《<a class="new" href="/w/index.php?title=%EC%97%AC%EC%99%95%EC%9D%B4_%EB%90%98%EA%B3%A0_%EC%8B%B6%EC%9D%80_%EB%82%A8%EC%9E%90&amp;action=edit&amp;redlink=1" title="여왕이 되고 싶은 남자 (없는 문서)">여왕이 되고 싶은 남자</a>》가 출판되면서 절정에 달했다. 베일리의 책이 출판되고 나서 블랜차드는 이 이론이 행동학적 설명 이론일 뿐 성전환증 자체에 관한 해석은 추가적인 과학적 연구가 있어야 가능할 것이라고 선을 그었다.<sup class="reference" id="cite_ref-Bancroft2009_1-1"><a href="#cite_note-Bancroft2009-1">[1]</a></sup> <a class="new" href="/w/index.php?title=%EC%84%B8%EA%B3%84_%ED%8A%B8%EB%9E%9C%EC%8A%A4%EC%A0%A0%EB%8D%94_%EB%B3%B4%EA%B1%B4%EC%A0%84%EB%AC%B8%EA%B0%80%ED%98%91%ED%9A%8C&amp;action=edit&amp;redlink=1" title="세계 트랜스젠더 보건전문가협회 (없는 문서)">세계 트랜스젠더 보건전문가협회</a>(WPATH)는 증거 부족을 이유로 블랜차드 이론을 지지하지 않으며,<sup class="reference" id="cite_ref-wpath1_6-0"><a href="#cite_note-wpath1-6">[6]</a></sup> 또한 블랜차드 이론이 트랜스젠더에 대한 낙인찍기로 작용한다고 비난하고 있다.<sup class="reference" id="cite_ref-wpath2_7-0"><a href="#cite_note-wpath2-7">[7]</a></sup></p>
