<p><b>부여군</b>(扶餘郡)은 <a href="/wiki/%EB%8C%80%ED%95%9C%EB%AF%BC%EA%B5%AD" title="대한민국">대한민국</a> <a href="/wiki/%EC%B6%A9%EC%B2%AD%EB%82%A8%EB%8F%84" title="충청남도">충청남도</a> 남부에 있는 <a href="/wiki/%EA%B5%B0_(%ED%96%89%EC%A0%95_%EA%B5%AC%EC%97%AD)" title="군 (행정 구역)">군</a>이다. 동쪽으로 <a href="/wiki/%EB%85%BC%EC%82%B0%EC%8B%9C" title="논산시">논산시</a>, 서쪽으로 <a href="/wiki/%EB%B3%B4%EB%A0%B9%EC%8B%9C" title="보령시">보령시</a>와 <a href="/wiki/%EC%84%9C%EC%B2%9C%EA%B5%B0" title="서천군">서천군</a>, 북쪽으로 <a href="/wiki/%EC%B2%AD%EC%96%91%EA%B5%B0" title="청양군">청양군</a>과 <a href="/wiki/%EA%B3%B5%EC%A3%BC%EC%8B%9C" title="공주시">공주시</a>, 남쪽으로 <a href="/wiki/%EC%A0%84%EB%9D%BC%EB%B6%81%EB%8F%84" title="전라북도">전라북도</a> <a href="/wiki/%EC%9D%B5%EC%82%B0%EC%8B%9C" title="익산시">익산시</a>와 접한다.</p>
<p>북동쪽에서 남서쪽으로 <a href="/wiki/%EA%B8%88%EA%B0%95" title="금강">금강</a>이 흐르며, 비옥한 충적평야를 형성한다. 부여는 수려한 자연경관과 함께 <a href="/wiki/%EB%B0%B1%EC%A0%9C" title="백제">백제</a> 문화의 유물·유적이 산재해 있는 문화 관광지이다. <a href="/wiki/%EB%B0%B1%EC%A0%9C" title="백제">백제</a>시대 부여의 명칭은 <b><a class="mw-redirect" href="/wiki/%EC%82%AC%EB%B9%84_(%EC%A7%80%EB%AA%85)" title="사비 (지명)">사비</a></b>이며, <a href="/wiki/%EB%B0%B1%EC%A0%9C" title="백제">백제</a> 성왕이 국호를 <a class="mw-redirect" href="/wiki/%EB%82%A8%EB%B6%80%EC%97%AC" title="남부여">남부여</a>(538년)로 바꾸고 <a href="/wiki/%EC%9B%85%EC%A7%84" title="웅진">웅진</a>에서 사비로 천도한 이후 538년부터 660년까지 <a href="/wiki/%EB%B0%B1%EC%A0%9C" title="백제">백제</a> 수도였다. 매년 10월 <a href="/wiki/%EA%B3%B5%EC%A3%BC%EC%8B%9C" title="공주시">공주시</a>와 함께 <a href="/wiki/%EB%B0%B1%EC%A0%9C%EB%AC%B8%ED%99%94%EC%A0%9C" title="백제문화제">백제문화제</a>를 개최한다. 명소로는 백제시대 문화재인 <a href="/wiki/%EB%B6%80%EC%86%8C%EC%82%B0%EC%84%B1" title="부소산성">부소산성</a>과 백제의 별궁 연못이었던 <a href="/wiki/%EA%B6%81%EB%82%A8%EC%A7%80" title="궁남지">궁남지</a> 등이 있고, 백제문화단지가 조성되어 있다. 부여의 관북리 유적·<a href="/wiki/%EB%B6%80%EC%86%8C%EC%82%B0%EC%84%B1" title="부소산성">부소산성</a>과 <a class="mw-redirect" href="/wiki/%EB%8A%A5%EC%82%B0%EB%A6%AC_%EA%B3%A0%EB%B6%84%EA%B5%B0" title="능산리 고분군">능산리 고분군</a>, <a class="mw-redirect" href="/wiki/%EC%A0%95%EB%A6%BC%EC%82%AC%EC%A7%80" title="정림사지">정림사지</a>와 <a href="/wiki/%EB%B6%80%EC%97%AC_%EB%82%98%EC%84%B1" title="부여 나성">부여 나성</a>의 등 4곳의 <a href="/wiki/%EB%B0%B1%EC%A0%9C%EC%97%AD%EC%82%AC%EC%9C%A0%EC%A0%81%EC%A7%80%EA%B5%AC" title="백제역사유적지구">백제역사유적지구</a>가 <a class="mw-redirect" href="/wiki/%EC%9C%A0%EB%84%A4%EC%8A%A4%EC%BD%94" title="유네스코">유네스코</a> <a class="mw-redirect" href="/wiki/%EC%84%B8%EA%B3%84%EB%AC%B8%ED%99%94%EC%9C%A0%EC%82%B0" title="세계문화유산">세계문화유산</a>으로 등재되었다. 군청 소재지는 <a href="/wiki/%EB%B6%80%EC%97%AC%EC%9D%8D" title="부여읍">부여읍</a>이고, 행정구역은 1읍 15면이다.</p>

<h2><span class="mw-headline" id=".EC.97.AD.EC.82.AC">역사</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%B6%80%EC%97%AC%EA%B5%B0&amp;action=edit&amp;section=1" title="부분 편집: 역사">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<div class="thumb tright">
<div class="thumbinner" style="width:222px;"><a class="image" href="/wiki/%ED%8C%8C%EC%9D%BC:Buyeo_1872.png"><img alt="" class="thumbimage" data-file-height="1959" data-file-width="1476" height="292" src="//upload.wikimedia.org/wikipedia/commons/thumb/d/df/Buyeo_1872.png/220px-Buyeo_1872.png" srcset="//upload.wikimedia.org/wikipedia/commons/thumb/d/df/Buyeo_1872.png/330px-Buyeo_1872.png 1.5x, //upload.wikimedia.org/wikipedia/commons/thumb/d/df/Buyeo_1872.png/440px-Buyeo_1872.png 2x" width="220"/></a>
<div class="thumbcaption">
<div class="magnify"><a class="internal" href="/wiki/%ED%8C%8C%EC%9D%BC:Buyeo_1872.png" title="실제 크기로"></a></div>
조선후기 부여현 지도</div>
</div>
</div>
<p>부여는 본래 <a href="/wiki/%EB%B0%B1%EC%A0%9C" title="백제">백제</a>의 대산현(大山縣)이었고, 538년부터 660년까지 <a href="/wiki/%EB%B0%B1%EC%A0%9C" title="백제">백제</a>의 수도였다. 757년(<a href="/wiki/%EC%8B%A0%EB%9D%BC" title="신라">신라</a> <a href="/wiki/%EA%B2%BD%EB%8D%95%EC%99%95" title="경덕왕">경덕왕</a> 16년)한산(翰山)으로 고쳐 가림군(嘉林郡)의 영현으로 하였다. 940년(<a href="/wiki/%EA%B3%A0%EB%A0%A4_%ED%83%9C%EC%A1%B0" title="고려 태조">고려 태조</a> 23년) 홍산으로 고치고, 1175년(명종 5) 한산감무로 하여금 이를 겸하게 하였다.</p>
<p>1413년(<a href="/wiki/%EC%A1%B0%EC%84%A0_%ED%83%9C%EC%A2%85" title="조선 태종">조선 태종</a> 13년) 현감이 파견되었고, 1895년(고종 32) 군(郡)으로 승격되었으며, 1914년 행정구역개편 때 부여군에 병합되어 홍산면이 되었다. 홍산(鴻山)이라는 지명은 그 모양이 나는 기러기처럼 생긴 비홍산(飛鴻山)에서 유래하였다. <sup class="reference" id="cite_ref-1"><a href="#cite_note-1">[1]</a></sup></p>
<ul>
<li>538 ~ 660년 <a href="/wiki/%EB%B0%B1%EC%A0%9C" title="백제">백제</a>의 수도 (<a href="/wiki/%EB%B0%B1%EC%A0%9C" title="백제">백제</a> 26대 <a class="mw-redirect" href="/wiki/%EB%B0%B1%EC%A0%9C_%EC%84%B1%EC%99%95" title="백제 성왕">성왕</a> 16년 ~ 31대 <a href="/wiki/%EC%9D%98%EC%9E%90%EC%99%95" title="의자왕">의자왕</a> 20년)</li>
<li><a href="/wiki/%ED%86%B5%EC%9D%BC%EC%8B%A0%EB%9D%BC" title="통일신라">통일신라</a> 도독부 설치 (2군 2현)</li>
<li><a href="/wiki/%EA%B3%A0%EB%A0%A4" title="고려">고려</a>시대 감무 설치 (4현-부여,가림,석성,흥산)</li>
<li><a href="/wiki/%EC%A1%B0%EC%84%A0" title="조선">조선</a>시대 1군 3현 (임천군, 부여현, 홍산현, 석성현)</li>
<li><a href="/wiki/1895%EB%85%84" title="1895년">1895년</a> <a href="/wiki/6%EC%9B%94_23%EC%9D%BC" title="6월 23일">6월 23일</a>(음력 윤 5월 1일) 공주부(公州府) 부여군(扶餘郡), 석성군(石城郡), 홍주부(洪州府) 임천군(林川郡), 홍산군(鴻山郡) <sup class="reference" id="cite_ref-2"><a href="#cite_note-2">[2]</a></sup></li>
<li><a href="/wiki/1896%EB%85%84" title="1896년">1896년</a> <a href="/wiki/8%EC%9B%94_4%EC%9D%BC" title="8월 4일">8월 4일</a> 충청남도 부여군, 석성군, 임천군, 홍산군 <sup class="reference" id="cite_ref-3"><a href="#cite_note-3">[3]</a></sup></li>
<li><a href="/wiki/1914%EB%85%84" title="1914년">1914년</a> <a href="/wiki/4%EC%9B%94_1%EC%9D%BC" title="4월 1일">4월 1일</a> 부여군, 임천군, 홍산군 각 일원, 석성군 일부를 부여군으로 통합하였다.<sup class="reference" id="cite_ref-4"><a href="#cite_note-4">[4]</a></sup> (16면)</li>
<li><a href="/wiki/1917%EB%85%84" title="1917년">1917년</a> <a href="/wiki/10%EC%9B%94_1%EC%9D%BC" title="10월 1일">10월 1일</a> 현내면을 부여면으로 개칭하였다.<sup class="reference" id="cite_ref-5"><a href="#cite_note-5">[5]</a></sup></li>
<li><a href="/wiki/1960%EB%85%84" title="1960년">1960년</a> <a href="/wiki/1%EC%9B%94_1%EC%9D%BC" title="1월 1일">1월 1일</a> 부여면을 <a href="/wiki/%EB%B6%80%EC%97%AC%EC%9D%8D" title="부여읍">부여읍</a>으로 승격하였다.<sup class="reference" id="cite_ref-6"><a href="#cite_note-6">[6]</a></sup> (1읍 15면)</li>
</ul>
<h2><span class="mw-headline" id=".EC.A7.80.EB.A6.AC">지리</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%B6%80%EC%97%AC%EA%B5%B0&amp;action=edit&amp;section=2" title="부분 편집: 지리">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p><a href="/wiki/%EC%B6%A9%EC%B2%AD%EB%82%A8%EB%8F%84" title="충청남도">충청남도</a>의 서남쪽에 위치하며, 동쪽으로 <a href="/wiki/%EA%B3%B5%EC%A3%BC%EC%8B%9C" title="공주시">공주시</a>, <a href="/wiki/%EB%85%BC%EC%82%B0%EC%8B%9C" title="논산시">논산시</a>, 북쪽으로 <a href="/wiki/%EC%B2%AD%EC%96%91%EA%B5%B0" title="청양군">청양군</a>, 서로 <a href="/wiki/%EB%B3%B4%EB%A0%B9%EC%8B%9C" title="보령시">보령시</a>, 서남으로 <a href="/wiki/%EC%84%9C%EC%B2%9C%EA%B5%B0" title="서천군">서천군</a>, 남으로는 <a href="/wiki/%EA%B8%88%EA%B0%95" title="금강">금강</a>을 경계로 <a href="/wiki/%EC%A0%84%EB%9D%BC%EB%B6%81%EB%8F%84" title="전라북도">전라북도</a> <a href="/wiki/%EC%9D%B5%EC%82%B0%EC%8B%9C" title="익산시">익산시</a>와 접한다. 주변으로는 산지가 많으며 <a href="/wiki/%EB%B0%B1%EB%A7%88%EA%B0%95" title="백마강">백마강</a>이 부여군 한가운데를 지난다.</p>
<p><a href="/wiki/%EC%B0%A8%EB%A0%B9%EC%82%B0%EB%A7%A5" title="차령산맥">차령산맥</a>이 충청남도의 북동에서 남서부로 이어져 군의 북서쪽을 지나면서 구릉지를 이루고, 성태산(631m)·만수산(432m)·월명산(544m)·옥녀봉(368m)·원진산(271m) 등이 보령시·서천군과 경계를 이룬다.</p>
<p>부여군의 동부를 흘러 남쪽 군계(郡界)를 이루는 <a href="/wiki/%EA%B8%88%EA%B0%95" title="금강">금강</a> 연안에 <a class="mw-redirect" href="/wiki/%EC%B6%A9%EC%A0%81%ED%8F%89%EC%95%BC" title="충적평야">충적평야</a>가 펼쳐지고, 북부·서부 산지에서 시작하여 금강으로 흘러드는 은산천(恩山川)·구룡천(九龍川)·금천(金川)·사동천(寺洞川)·여사천(餘士川) 등의 하천 유역에도 크고 작은 평야가 형성되어 있다.</p>
<p>내산면(內山面)에서 시작하는 구룡천은 금천과 합류하여 논산시 대안(對岸)에서 금강으로 흘러드는데, 이 두 하천 유역의 구룡평야는 3,000ha가 넘는 비옥한 토지로 부여군내 제일의 곡창지대를 이룬다.</p>

<h2><span class="mw-headline" id=".EC.82.B0.EC.97.85">산업</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%B6%80%EC%97%AC%EA%B5%B0&amp;action=edit&amp;section=4" title="부분 편집: 산업">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<h3><span class="mw-headline" id=".EC.A7.80.EC.97.AD.EB.82.B4_.EC.B4.9D.EC.83.9D.EC.82.B0">지역내 총생산</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%B6%80%EC%97%AC%EA%B5%B0&amp;action=edit&amp;section=5" title="부분 편집: 지역내 총생산">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>부여군의 2012년 지역내 총생산은 3조7800억 원으로 충청남도 지역내 총생산의 1.3%를 차지하고 있다. 이중 농림어업(1차 산업)은 6천12억 원으로 16.20%를 차지하고, 광업 및 제조업(2차 산업)은 1조5511억 원으로 41.03%의 비중으로 차지하고 있으며, 상업 및 서비스업(3차 산업)은 1조61710억 원으로 42.77%의 비중을 차지한다. 3차 산업 부문에서는 건설업(8.94%)과 공공행정(7.28%), 교육 서비스업(4.05%), 도소매업(3.3%)이 큰 비중을 차지하고 있다.<sup class="reference" id="cite_ref-8"><a href="#cite_note-8">[8]</a></sup></p>
<h3><span class="mw-headline" id=".EC.82.B0.EC.97.85.EB.B3.84_.EC.A2.85.EC.82.AC.EC.9E.90_.ED.98.84.ED.99.A9">산업별 종사자 현황</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%B6%80%EC%97%AC%EA%B5%B0&amp;action=edit&amp;section=6" title="부분 편집: 산업별 종사자 현황">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>2014년 부여군 산업의 총종사자 수는 19,482명으로 충청남도 총 종사자 수의 2.4%를 차지하고 있다. 이중 농림어업(1차 산업)은 349명으로 비중이 낮고 1.8%의 비중을 차지하고, 광업 및 제조업(2차 산업)은 3,342명으로 17.2%의 비중으로 차지하고 상업 및 서비스업(3차산업)은 15,779명으로 81.0%의 비중을 차지한다. 2차 산업은 충청남도 전체의 비중(32.8%)보다 낮고 3차 산업은 충청남도 전체비중(66.7%)보다 높다. 3차 산업 부문에서는 숙박 및 음식점업(10.8%), 도소매업(16.7%)과 교육서비스업(8.3%) 보건업 및 사회복지서비스업(9.4%), 문화 및 기타서비스업(8.6%)이 큰 비중을 차지하고 있다.<sup class="reference" id="cite_ref-9"><a href="#cite_note-9">[9]</a></sup></p>
<h3><span class="mw-headline" id=".EC.83.81.EC.A3.BC.EC.9D.B8.EA.B5.AC.EC.99.80_.EC.A3.BC.EA.B0.84.EC.9D.B8.EA.B5.AC">상주인구와 주간인구</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%B6%80%EC%97%AC%EA%B5%B0&amp;action=edit&amp;section=7" title="부분 편집: 상주인구와 주간인구">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>부여군 2010년 기준 상주인구는 66,480명이고 주간인구는 68,620명으로 주간인구지수가 103으로 높다. 통근으로 인한 유입인구는 3,927명, 유출인구는1,492명이고, 통학으로 인한 유입인구는 267명, 유출인구는 562명으로 전체 유입인구가 2,140명 더 많은데, 이는 충청남도 대부분의 지역에서 공통적으로 나타나는 현상이다. <sup class="reference" id="cite_ref-10"><a href="#cite_note-10">[10]</a></sup></p>
<h2><span class="mw-headline" id=".ED.96.89.EC.A0.95_.EA.B5.AC.EC.97.AD">행정 구역</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%B6%80%EC%97%AC%EA%B5%B0&amp;action=edit&amp;section=8" title="부분 편집: 행정 구역">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<div class="rellink relarticle mainarticle"><img alt="" data-file-height="128" data-file-width="128" height="16" src="//upload.wikimedia.org/wikipedia/commons/thumb/e/ec/Crystal_Clear_app_xmag.svg/16px-Crystal_Clear_app_xmag.svg.png" srcset="//upload.wikimedia.org/wikipedia/commons/thumb/e/ec/Crystal_Clear_app_xmag.svg/24px-Crystal_Clear_app_xmag.svg.png 1.5x, //upload.wikimedia.org/wikipedia/commons/thumb/e/ec/Crystal_Clear_app_xmag.svg/32px-Crystal_Clear_app_xmag.svg.png 2x" width="16"/> 이 부분의 본문은 <a href="/wiki/%EB%B6%80%EC%97%AC%EA%B5%B0%EC%9D%98_%ED%96%89%EC%A0%95_%EA%B5%AC%EC%97%AD" title="부여군의 행정 구역">부여군의 행정 구역</a>입니다.</div>
<p>부여군의 행정 구역은 1읍 15면 191법정리 432행정리 1,686반으로 편제되어 있다. 군청사는 부여읍 동남리 725에 있다. 인구는 2011년 6월 30일을 기준으로 75,189명이며, 전체 인구의 32.6%가 <a href="/wiki/%EB%B6%80%EC%97%AC%EC%9D%8D" title="부여읍">부여읍</a>에 거주하고 있다. 또한 면적은 624.6㎢(2009)로 도내에서 <a href="/wiki/%EA%B3%B5%EC%A3%BC%EC%8B%9C" title="공주시">공주시</a>,<a href="/wiki/%EC%84%9C%EC%82%B0%EC%8B%9C" title="서산시">서산시</a>, <a href="/wiki/%EB%8B%B9%EC%A7%84%EC%8B%9C" title="당진시">당진시</a>, <a href="/wiki/%EC%B2%9C%EC%95%88%EC%8B%9C" title="천안시">천안시</a>에 이어 다섯 번째로 넓다. 인구 최고치는 1964년의 195,843명이었다. 제헌 국회 때부터 5대 민의원까지는 부여군도 부여갑, 부여을 선거구로 나누었으나 단일 선거구가 되었고, 현재는 <a href="/wiki/%EC%B2%AD%EC%96%91%EA%B5%B0" title="청양군">청양군</a>과 선거구를 공유하고 있다.<sup class="reference" id="cite_ref-11"><a href="#cite_note-11">[11]</a></sup></p>

<p>부여는 백제의 마지막 수도로서 이와 관련한 역사 탐방이 문화관광의 주된 자원이다. 주요 사적으로는 부여군 부여읍 동남리에 있는 <a class="mw-redirect" href="/wiki/%EB%B6%80%EC%97%AC%EC%A0%95%EB%A6%BC%EC%82%AC%EC%A7%80%EC%98%A4%EC%B8%B5%EC%84%9D%ED%83%91" title="부여정림사지오층석탑">부여정림사지오층석탑</a>과 <a class="new" href="/w/index.php?title=%EB%8A%A5%EC%82%B0%EB%A6%AC%EA%B3%A0%EB%B6%84%EA%B5%B0&amp;action=edit&amp;redlink=1" title="능산리고분군 (없는 문서)">능산리고분군</a>, <a href="/wiki/%EA%B6%81%EB%82%A8%EC%A7%80" title="궁남지">궁남지</a>, <a href="/wiki/%EB%B6%80%EC%86%8C%EC%82%B0%EC%84%B1" title="부소산성">부소산성</a>, <a href="/wiki/%EB%82%99%ED%99%94%EC%95%94" title="낙화암">낙화암</a>, <a href="/wiki/%EA%B3%A0%EB%9E%80%EC%82%AC" title="고란사">고란사</a>, <a class="new" href="/w/index.php?title=%EB%AC%B4%EB%9F%89%EC%82%AC_(%EC%B6%A9%EB%82%A8_%EB%B6%80%EC%97%AC%EA%B5%B0)&amp;action=edit&amp;redlink=1" title="무량사 (충남 부여군) (없는 문서)">무량사</a> 등이 있다. 대표적 유물로는 국보로 지정된 <a class="mw-redirect" href="/wiki/%EB%B0%B1%EC%A0%9C%EA%B8%88%EB%8F%99%EB%8C%80%ED%96%A5%EB%A1%9C" title="백제금동대향로">백제금동대향로</a>, <a class="mw-redirect" href="/wiki/%EB%B6%80%EC%97%AC%EC%A0%95%EB%A6%BC%EC%82%AC%EC%A7%80%EC%98%A4%EC%B8%B5%EC%84%9D%ED%83%91" title="부여정림사지오층석탑">부여정림사지오층석탑</a> 등이 있다. 국가지정문화재에는 국보 4점, 보물 16점, 사적 21곳, 무형문화재 1점, 민속자료 4점, 천연기념물 1점, 사적 및 명승 1점을 보유하고 있으며, 도지정문화재로 유형문화재 25점, 기념물 25점, 무형문화재 5점이 있다. 그 외에도 도문화재자료 40점, 군향토유족77점으로 많은 문화재가 있다. 매해 공주시와 합동으로 10월에 종합예술제인 <a href="/wiki/%EB%B0%B1%EC%A0%9C%EB%AC%B8%ED%99%94%EC%A0%9C" title="백제문화제">백제문화제</a>를 개최한다.</p>
