<p><b>우지</b>(<span style="font-size: smaller;"><a href="/wiki/%ED%9E%88%EB%B8%8C%EB%A6%AC%EC%96%B4" title="히브리어">히브리어</a>:</span> <span lang="he" xml:lang="he">עוזי‎</span>, UZI)는 <a href="/wiki/%EC%9D%B4%EC%8A%A4%EB%9D%BC%EC%97%98" title="이스라엘">이스라엘</a>의 Uziel Gal이 개발한 <a href="/wiki/%EA%B8%B0%EA%B4%80%EB%8B%A8%EC%B4%9D" title="기관단총">기관단총</a>이다.</p>
<p>이스라엘은 <a href="/wiki/%EC%B2%B4%EC%BD%94%EC%8A%AC%EB%A1%9C%EB%B0%94%ED%82%A4%EC%95%84" title="체코슬로바키아">체코슬로바키아</a>의 <a class="new" href="/w/index.php?title=Vz23&amp;action=edit&amp;redlink=1" title="Vz23 (없는 문서)">Vz23</a>, <a class="new" href="/w/index.php?title=Vz25&amp;action=edit&amp;redlink=1" title="Vz25 (없는 문서)">Vz25</a> 기관단총을 수입해서 사용했었다. 그러나 <a href="/wiki/1952%EB%85%84" title="1952년">1952년</a> <a href="/wiki/%EC%B2%B4%EC%BD%94" title="체코">체코</a>는 공산화되면서 친 아랍 외교노선을 택하였고, 체코로부터의 기관단총 수입이 어렵게 되었다. 결국 이스라엘은 기관단총의 자체 개발을 시도하게 되고, 여러 후보들 중 Uziel Gal의 디자인이 채택되었다.</p>
<p><a href="/wiki/%EC%98%A4%ED%94%88_%EB%B3%BC%ED%8A%B8" title="오픈 볼트">오픈 볼트</a> 방식이기 때문에 명중률은 후대의 <a href="/wiki/%ED%81%B4%EB%A1%9C%EC%A6%88%EB%93%9C_%EB%B3%BC%ED%8A%B8" title="클로즈드 볼트">클로즈드 볼트</a> 방식의 <a class="mw-redirect" href="/wiki/%ED%97%A4%ED%81%B4%EB%9F%AC%26%EC%BD%94%ED%9D%90_MP5" title="헤클러&amp;코흐 MP5">MP5</a> 시리즈에 비하면 낮은 수준이다. 그러나 오픈 볼트 중에서는 비교적 잘 맞는 편이고 값싸고 잔고장이 적다는 점으로 인해 군대와 <a href="/wiki/%EA%B2%BD%EC%B0%B0" title="경찰">경찰</a>, 그리고 <a href="/wiki/%ED%85%8C%EB%9F%AC%EB%A6%AC%EC%A6%98" title="테러리즘">테러리스트</a>들이 많이 사용하였다. 미니 우지, 마이크로 우지 등의 변형이 있으며, 현재도 독일 육군 등에서 계속 사용 중이다. 마이크로 우지는 <a href="/wiki/%ED%81%B4%EB%A1%9C%EC%A6%88%EB%93%9C_%EB%B3%BC%ED%8A%B8" title="클로즈드 볼트">클로즈드 볼트</a> 방식을 사용하며, 미니 우지는 클로즈드 볼트 방식과 오픈 볼트 방식 둘 중 하나를 사용하도록 제작될 수 있다. <a class="external autonumber" href="http://www.israel-weapon.com/files/Pdf/UZI.pdf" rel="nofollow">[2]</a> <a class="external autonumber" href="http://www.israeli-weapons.com/weapons/small_arms/uzi/Uzi-Mini.html" rel="nofollow">[3]</a></p>
