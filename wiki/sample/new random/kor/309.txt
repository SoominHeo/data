<h2><span class="mw-headline" id=".EC.A3.BC.EC.9A.94_.EB.8C.80.ED.9A.8C_.EA.B8.B0.EB.A1.9D">주요 대회 기록</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=1932%EB%85%84_%ED%95%98%EA%B3%84_%EC%98%AC%EB%A6%BC%ED%94%BD&amp;action=edit&amp;section=6" title="부분 편집: 주요 대회 기록">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<ul>
<li><a href="/wiki/%EC%A4%91%ED%99%94%EB%AF%BC%EA%B5%AD" title="중화민국">중화민국</a>이 처음으로 <a href="/wiki/%EC%98%AC%EB%A6%BC%ED%94%BD" title="올림픽">올림픽</a>에 참가하게 되었다.</li>
<li>1932년 하계 올림픽에서는 남자만을 위해 따로 선수촌이 지어졌다. 그리고 <a class="mw-redirect" href="/wiki/%EC%97%AC%EC%9E%90" title="여자">여자</a>들이 못 들어가게 경비가 삼엄했는데, <a href="/wiki/%EC%B9%B4%EC%9A%B0%EB%B3%B4%EC%9D%B4" title="카우보이">카우보이</a>들이 지킬 정도였다고 한다. 총 127명이었던 <a class="mw-redirect" href="/wiki/%EC%97%AC%EC%9E%90" title="여자">여자</a>들은 <a href="/wiki/%ED%98%B8%ED%85%94" title="호텔">호텔</a>을 이용했다.</li>
<li><a href="/wiki/%EC%A1%B0%EC%84%A0" title="조선">조선</a>의 선수 (<a href="/wiki/%ED%99%A9%EC%9D%84%EC%88%98" title="황을수">황을수</a>, <a href="/wiki/%EA%B6%8C%ED%83%9C%ED%95%98" title="권태하">권태하</a>, <a href="/wiki/%EA%B9%80%EC%9D%80%EB%B0%B0_(%EC%9C%A1%EC%83%81_%EC%84%A0%EC%88%98)" title="김은배 (육상 선수)">김은배</a>)들이 일장기를 달고 처음으로 올림픽에 참가하였다. 김은배 선수는 <a href="/wiki/%EB%A7%88%EB%9D%BC%ED%86%A4" title="마라톤">마라톤</a>에서 6위에 입상하였다. <a href="/wiki/%EB%8F%99%EC%95%84%EC%9D%BC%EB%B3%B4" title="동아일보">동아일보</a>에서 김은배 6위 입상 소식을 보도하면서 일장기를 지운 <a class="mw-redirect" href="/wiki/%EC%9D%BC%EC%9E%A5%EA%B8%B0_%EB%A7%90%EC%86%8C%EC%82%AC%EA%B1%B4" title="일장기 말소사건">일장기 말소사건</a>이 일어났다.</li>
<li>이번 대회는 사상 최초로 사진판독이 실시됐다.</li>
<li><a href="/wiki/%EB%AF%B8%EA%B5%AD" title="미국">미국</a>의 <a href="/wiki/%EC%95%84%ED%94%84%EB%A6%AC%EC%B9%B4%EA%B3%84_%EB%AF%B8%EA%B5%AD%EC%9D%B8" title="아프리카계 미국인">흑인</a> 단거리 육상 선수 <a class="new" href="/w/index.php?title=%EC%97%90%EB%94%94_%ED%86%A8%EB%9E%80&amp;action=edit&amp;redlink=1" title="에디 톨란 (없는 문서)">에디 톨란</a>이 100m와 200m에서 우승을 차지하였다.</li>
<li>남자100m결승에서 미국의 톨란과 멧커프가 거의 동시에 결승테이프를 끊었지만,육안으로는 도저히 분간할 수가 없어서 결국 사진판독을 통해 톨란이 1인치(2.54cm) 앞선 것으로 판명됐다</li>
<li><a href="/wiki/%EB%85%B8%EB%A5%B4%EC%9B%A8%EC%9D%B4" title="노르웨이">노르웨이</a>계 미국인 여자 선수 <a href="/wiki/%EB%B2%A0%EC%9D%B4%EB%B8%8C_%EB%94%94%EB%93%9C%EB%A6%AD%EC%8A%A8" title="베이브 디드릭슨">베이브 디드릭슨</a>은 80m 허들과 창던지기에서 금메달을 획득하였다.</li>
<li>3개월 때 <a class="mw-redirect" href="/wiki/%ED%81%B4%EB%A6%AC%EB%B8%94%EB%9E%9C%EB%93%9C_(%EC%98%A4%ED%95%98%EC%9D%B4%EC%98%A4_%EC%A3%BC)" title="클리블랜드 (오하이오 주)">클리블랜드</a>로 이민을 가서 자라온 <a href="/wiki/%ED%8F%B4%EB%9E%80%EB%93%9C" title="폴란드">폴란드</a>의 육상 선수 <a href="/wiki/%EC%8A%A4%ED%83%80%EB%8B%88%EC%8A%A4%EC%99%80%EB%B0%94_%EB%B0%9C%EB%9D%BC%EC%8B%9C%EC%97%90%EB%B9%84%EC%B9%98" title="스타니스와바 발라시에비치">스타니스와바 발라시에비치</a> (영어 이름은 스텔라 월시)가 모국의 대표 선수로 참가하여 100m 우승을 하였다. 그녀의 사후에 <a class="mw-redirect" href="/wiki/%EB%82%A8%EB%85%80%EC%A4%91%EA%B0%84%EB%AA%B8" title="남녀중간몸">남녀중간몸</a>이라는 사실이 밝혀졌다.</li>
</ul>
