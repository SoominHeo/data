<p><b>특수 상대성이론</b>(特殊相對性理論, <span style="font-size: smaller;"><a href="/wiki/%EB%8F%85%EC%9D%BC%EC%96%B4" title="독일어">독일어</a>:</span> <span lang="de" xml:lang="de">spezielle Relativitätstheorie</span>, <span style="font-size: smaller;"><a href="/wiki/%EC%98%81%EC%96%B4" title="영어">영어</a>:</span> <span lang="en" xml:lang="en">theory of special relativity</span>), 또는 <b>특수상대론</b>(特殊相對論, <span style="font-size: smaller;"><a href="/wiki/%EC%98%81%EC%96%B4" title="영어">영어</a>:</span> <span lang="en" xml:lang="en">special relativity</span>)은 <a class="mw-redirect" href="/wiki/%EB%B9%9B%EC%9D%98_%EC%86%8D%EB%8F%84" title="빛의 속도">빛의 속도</a>에 견줄 만한 속도로 움직이는 물체들을 다루는 <a href="/wiki/%EC%97%AD%ED%95%99_(%EB%AC%BC%EB%A6%AC%ED%95%99)" title="역학 (물리학)">역학</a> 이론이다. 특수 상대성 이론은 고속의 물체에 대하여 기존의 <a class="mw-redirect" href="/wiki/%EB%89%B4%ED%84%B4_%EC%97%AD%ED%95%99" title="뉴턴 역학">뉴턴 역학</a>의 <a class="mw-redirect" href="/wiki/%EA%B0%88%EB%A6%B4%EB%A0%88%EC%9D%B4_%EB%B3%80%ED%99%98" title="갈릴레이 변환">갈릴레이 변환</a>을 대체하고, <a class="mw-redirect" href="/wiki/%EA%B0%88%EB%A6%B4%EB%A0%88%EC%9D%B4_%EB%B3%80%ED%99%98" title="갈릴레이 변환">갈릴레이 변환</a>과 달리 <a class="mw-redirect" href="/wiki/%EA%B3%A0%EC%A0%84%EC%A0%84%EC%9E%90%EA%B8%B0%ED%95%99" title="고전전자기학">고전전자기학</a>의 <a href="/wiki/%EB%A7%A5%EC%8A%A4%EC%9B%B0_%EB%B0%A9%EC%A0%95%EC%8B%9D" title="맥스웰 방정식">맥스웰 방정식</a>과 일관적이다.</p>
<p>특수 상대성 이론은 <a class="mw-redirect" href="/wiki/%EA%B0%88%EB%A6%B4%EB%A0%88%EC%9D%B4_%EB%B3%80%ED%99%98" title="갈릴레이 변환">갈릴레이 변환</a> 대신 <a href="/wiki/%EB%A1%9C%EB%9F%B0%EC%B8%A0_%EB%B3%80%ED%99%98" title="로런츠 변환">로런츠 변환</a>을 채용한다. 이 이론에서는, 갈릴레오가 주장한 것처럼 모든 <a class="mw-redirect" href="/wiki/%EA%B4%80%EC%84%B1%EA%B3%84" title="관성계">관성계</a>가 동등하지만, 전자기학의 <a href="/wiki/%EB%A7%A5%EC%8A%A4%EC%9B%B0_%EB%B0%A9%EC%A0%95%EC%8B%9D" title="맥스웰 방정식">맥스웰 방정식</a>과 일관하게 <a class="mw-redirect" href="/wiki/%EB%B9%9B%EC%9D%98_%EC%86%8D%EB%8F%84" title="빛의 속도">광속</a>이 모든 관성계에서 동등하다. 즉, <a href="/wiki/%EB%B9%9B" title="빛">빛</a> (혹은 다른 질량이 없는 입자)의 속도는 이를 방출하는 물체와 관찰자 사이의 상대운동에 무관하다.<sup class="reference" id="cite_ref-1"><a href="#cite_note-1">[1]</a></sup></p>
<p>특수 상대성 이론은 여러 가지 놀라운 예측을 하는데, 이 예측들은 전부 실험에 의해 검증되었다.<sup class="reference" id="cite_ref-2"><a href="#cite_note-2">[2]</a></sup> 로런츠 변환을 도입함에 따라, 시간과 공간을 운동학적으로 더 이상 구별하여 생각할 수 없다. 따라서, 이 둘을 더하여 <a href="/wiki/%EC%8B%9C%EA%B3%B5%EA%B0%84" title="시공간">시공간</a>이라는 하나의 개체로 생각하고, 이를 <a href="/wiki/%EB%AF%BC%EC%BD%94%ED%94%84%EC%8A%A4%ED%82%A4_%EA%B3%B5%EA%B0%84" title="민코프스키 공간">민코프스키 공간</a>으로 나타낸다. 이에 따라, 시간과 공간 중 하나에만 의존하는 측정량 (예를 들어 길이, 시간 간격 등)은 서로 다른 관성계에서 서로 다른 값을 가진다. 따라서, 시간과 공간에 해당하는 값들을 합쳐서 <a class="mw-redirect" href="/wiki/4%EC%B0%A8%EC%9B%90_%EB%B2%A1%ED%84%B0" title="4차원 벡터">4차원 벡터</a>로 나타내면 다루기 쉽다.</p>
<p>특수 상대성 이론에서는 어떤 일반적 속도 상수(광속)가 존재하므로, 이를 이용하여 <a href="/wiki/%EC%A7%88%EB%9F%89" title="질량">질량</a>과 <a href="/wiki/%EC%97%90%EB%84%88%EC%A7%80" title="에너지">에너지</a>를 관계지을 수 있다. 이론에 따르면, 어떤 계의 질량은 그 계의 운동량 중심의 (순간적인) 관성계에서의 에너지와 같다 (E = mc<sup>2</sup>). 이를 <a class="mw-redirect" href="/wiki/%EC%A7%88%EB%9F%89-%EC%97%90%EB%84%88%EC%A7%80_%EB%93%B1%EA%B0%80%EC%84%B1" title="질량-에너지 등가성">질량-에너지 등가성</a>이라 한다.</p>
<p>특수 상대성 이론은 (<a class="mw-redirect" href="/wiki/%EB%89%B4%ED%84%B4_%EC%97%AD%ED%95%99" title="뉴턴 역학">뉴턴 역학</a>과 같이) <a class="mw-redirect" href="/wiki/%EC%83%81%EB%8C%80%EC%84%B1_%EC%9B%90%EB%A6%AC" title="상대성 원리">상대성 원리</a>를 오로지 관성계에 대해서만 적용한다. 즉, 가속계는 관성계와 실험적으로 구별할 수 있다는 것이다. 후에 발견된 <a class="mw-redirect" href="/wiki/%EC%9D%BC%EB%B0%98_%EC%83%81%EB%8C%80%EC%84%B1_%EC%9D%B4%EB%A1%A0" title="일반 상대성 이론">일반 상대성 이론</a>은 중력을 고려하면 가속계가 관성계와 동등하다고 주장한다. (정확히 말하면, 가속계와 관성계를 구별할 수 없다.) 그러나 강한 <a href="/wiki/%EC%A4%91%EB%A0%A5%EC%9E%A5" title="중력장">중력장</a>이 없는 경우, 특수 상대성 이론은 물리적 현상을 정확히 기술한다. 다루어지는 속도가 빛의 속도에 비해 훨씬 작은 일상적인 영역에 대해서는 특수 상대성 이론의 예측은 <a class="mw-redirect" href="/wiki/%EB%89%B4%ED%84%B4_%EC%97%AD%ED%95%99" title="뉴턴 역학">뉴턴 역학</a>의 예측과 일치한다.</p>

<h2><span class="mw-headline" id=".EA.B0.80.EC.A0.95">가정</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%ED%8A%B9%EC%88%98_%EC%83%81%EB%8C%80%EC%84%B1%EC%9D%B4%EB%A1%A0&amp;action=edit&amp;section=1" title="부분 편집: 가정">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>특수 상대성 이론은 기본적으로 다음과 같은 두 개의 가정에서 시작한다.</p>
<ol>
<li>모든 <a class="mw-redirect" href="/wiki/%EA%B4%80%EC%84%B1%EA%B3%84" title="관성계">관성계</a>는 동등하다.</li>
<li>진공에서의 <a href="/wiki/%EB%B9%9B%EC%9D%98_%EC%86%8D%EB%A0%A5" title="빛의 속력">빛의 속력</a>은 어느 관성계에서나 일정하다.</li>
</ol>
<p>첫 번째 가정은 어느 관성계(속도가 일정한 계)에서든 물리 법칙은 동등하게 적용된다는 뜻이다. 비록 땅에서 볼 때 시속 100 km로 가는 차가 같은 방향으로 시속 50 km로 가는 기차 안에서는 시속 50 km로 이동하는 것처럼 보이지만, 이 구체적인 값의 차이와는 달리 두 관성계 모두에서 물리 법칙, 즉 <a class="mw-redirect" href="/wiki/%EA%B4%80%EC%84%B1%EC%9D%98_%EB%B2%95%EC%B9%99" title="관성의 법칙">관성의 법칙</a>, <a href="/wiki/%EC%9A%B4%EB%8F%99%EB%9F%89#.EC.9A.B4.EB.8F.99.EB.9F.89_.EB.B3.B4.EC.A1.B4_.EB.B2.95.EC.B9.99" title="운동량">운동량 보존의 법칙</a>, <a class="mw-redirect" href="/wiki/%EC%97%90%EB%84%88%EC%A7%80_%EB%B3%B4%EC%A1%B4%EC%9D%98_%EB%B2%95%EC%B9%99" title="에너지 보존의 법칙">에너지 보존의 법칙</a> 등은 동등하게 적용된다. 이렇게 모든 관성계에 적용되는 물리 법칙이 같으므로 물리 법칙의 차이를 이용해서 두 관성계를 구분할 수 없다. 따라서 두 관성계는 동등하고 어느 하나가 다른 하나에 비해 더 '진정한 기준관성계'에 가깝지 않다.</p>
<p>두 번째 가정은 어느 관성계에서 관측하든지 빛의 속도는 동일하게 관측된다는 것이다. 기존의 갈릴레오 변환에서는 관성계 A에 대해 <i>u</i>의 속도로 움직이는 관성계 B에서 관측한 어느 물체의 속도가 <i>v</i>일 때 관성계 A에서 관측한 물체의 속도는 <i>u+v</i>이다. 하지만 이 변환은 빛에 대해서는 적용되지 않는다. 즉 관성계 A에 대해 <i>u</i>의 속도로 움직이는 관성계 B에서 관측한 <a href="/wiki/%EB%B9%9B%EC%9D%98_%EC%86%8D%EB%A0%A5" title="빛의 속력">빛의 속력</a>이 <i>c</i>일 때 관성계 A에서 관측한 빛의 속력은 <i>u+c</i>가 아닌 <i>c</i>이다.</p>
<h2><span class="mw-headline" id=".EB.8F.99.EC.8B.9C.EC.84.B1.EC.9D.98_.EC.83.81.EB.8C.80.EC.84.B1">동시성의 상대성</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%ED%8A%B9%EC%88%98_%EC%83%81%EB%8C%80%EC%84%B1%EC%9D%B4%EB%A1%A0&amp;action=edit&amp;section=2" title="부분 편집: 동시성의 상대성">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<div class="rellink relarticle mainarticle"><img alt="" data-file-height="128" data-file-width="128" height="16" src="//upload.wikimedia.org/wikipedia/commons/thumb/e/ec/Crystal_Clear_app_xmag.svg/16px-Crystal_Clear_app_xmag.svg.png" srcset="//upload.wikimedia.org/wikipedia/commons/thumb/e/ec/Crystal_Clear_app_xmag.svg/24px-Crystal_Clear_app_xmag.svg.png 1.5x, //upload.wikimedia.org/wikipedia/commons/thumb/e/ec/Crystal_Clear_app_xmag.svg/32px-Crystal_Clear_app_xmag.svg.png 2x" width="16"/> 이 부분의 본문은 <a href="/wiki/%EB%8F%99%EC%8B%9C%EC%84%B1%EC%9D%98_%EC%83%81%EB%8C%80%EC%84%B1" title="동시성의 상대성">동시성의 상대성</a>입니다.</div>

<p>특수 상대성 이론에 의하면 동시성은 좌표계에 따라 상대적이다. 즉 <i>동시</i>라는 것은 좌표계에 따라서 다르게 관측된다는 것이다. 한 좌표계에서 두 사건이 동시에 일어난 것이라 관측되었더라도 다른 좌표계에서는 두 사건이 동시에 일어나지 않은 것으로 관측될 수 있다.</p>
<p>이 현상의 가장 유명한 예는 다음과 같다. 빠른 속도로 움직이는 버스가 있고 버스의 앞면과 뒷면의 정확한 중앙에 전등이 하나 놓여 있다. 이 전등이 꺼져 있다가 갑자기 켜진다고 하자. 이때 전등에서 나온 빛이 버스의 앞면에 도달하는 사건을 A, 뒷면에 도달하는 사건을 B라 하자. 그러면 버스 안에 있는 사람은 A와 B가 동시에 일어났다고 관측할 것이다. 그 이유는 전등이 앞면과 뒷면의 정중앙에 있기 때문에 앞면과 뒷면으로 향한 빛의 진행 거리가 같았기 때문이다. 그러나 버스 외부에서 보면 B가 A보다 먼저 일어난 것으로 보인다. 외부에서 보면 빛은 앞과 뒤를 향해 같은 속력으로 진행하지만 뒷면은 빛을 향해 가까워지고, 앞면은 빛에서 멀어지게 된다. 따라서 뒷면으로 향한 빛이 앞면으로 향한 빛보다 먼저 도착하게 된다. 즉 두 관찰자의 동시가 일치 하지 않는다는 것이다.</p>
<p>이것은 이와 같은 특수한 상황에서만 적용되는 성질이 아니라 어떠한 두 사건에 대해서도 성립하는 일반적인 법칙이다. 또한 이 성질은 <a href="#.EC.8B.9C.EA.B0.84_.ED.8C.BD.EC.B0.BD">시간 팽창</a>과 <a href="#.EA.B8.B8.EC.9D.B4_.EC.88.98.EC.B6.95">길이 수축</a>을 설명하는 데 기본적인 바탕이 된다.</p>
<h2><span class="mw-headline" id=".EC.8B.9C.EA.B0.84_.ED.8C.BD.EC.B0.BD">시간 팽창</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%ED%8A%B9%EC%88%98_%EC%83%81%EB%8C%80%EC%84%B1%EC%9D%B4%EB%A1%A0&amp;action=edit&amp;section=3" title="부분 편집: 시간 팽창">편집</a><span class="mw-editsection-bracket">]</span></span></h2>

<p>시간 팽창(Time dilation)은 어떤 관성계에서 상대속도를 가지는 다른 관성계를 관측할 때 시간이 팽창된 것으로 관측되는 것을 뜻한다. 즉 관성계 A에서 움직이는 다른 관성계 B를 보면 B의 시간이 상대적으로 느리게 가는 것으로 관측된다. 물론 모든 관성계는 동등하기 때문에 역으로 다른 관성계 B에서 관성계 A를 관측하면 A의 시간이 상대적으로 느리게 가는 것으로 관측된다.</p>

<p>이 두 관측 결과는 서로 모순되는 것처럼 보이지만 그렇지 않다. 관성계 A에서 B의 시간을 관측하기 위해서는 측정 시작 시각과 끝 시각을 각각 측정해서 두 차이를 구해야 한다. 그런데 B는 움직이고 있으므로 측정 시작 시각의 B의 위치와 끝 시간의 위치 두 곳에 각각 시계를 놓아야 한다. 이때 측정 전에 두 시계가 가리키는 시각이 같아야 한다. 이렇게 두 시계의 시각을 같게 조정하는 것을 <b>동기화</b>(<span lang="en" xml:lang="en">synchronization</span>)라 한다. 그러나 A에서 두 시계를 동기화하였더라도 이것을 B가 관측했을 때는 <a href="/wiki/%EB%8F%99%EC%8B%9C%EC%84%B1%EC%9D%98_%EC%83%81%EB%8C%80%EC%84%B1" title="동시성의 상대성">동시성의 상대성</a>에 의해 두 시계가 다른 시각을 가리키는 것으로 관측된다. 따라서 B가 관측하기에는 A가 잘못된 실험을 하는 것으로 보이고 따라서 B의 시간이 A보다 느리게 간다는 A의 관측 결과는 B에게 맞지 않는 것이다.</p>
<p>또한 B에서 A를 관측하는 경우에는 지금까지와 전혀 다른 실험이 필요하다. A에서 B를 관측할 때는 A에 동기화된 2개의 시계와 B에 하나의 시계가 필요했지만 B에서 A를 관측하는 경우에는 B에 동기화된 2개의 시계와 A에 하나의 시계를 사용해 실험을 한다. 즉 A와 B는 서로 다른 실험을 하는 것이고 따라서 두 결과는 상호 모순적이지 않다.</p>

<p>길이 수축(Length contraction)은 어떤 관성계에서 상대속도를 가지는 다른 관성계를 관측할 때 길이가 수축된 것으로 관측되는 것을 뜻한다. 즉 관성계 A에서 움직이는 다른 관성계 B를 보면 B의 길이가 상대적으로 짧아진 것으로 관측된다. 물론 모든 관성계는 동등하기 때문에 역으로 다른 관성계 B에서 관성계 A를 관측하면 A의 길이가 상대적으로 짧아진 것으로 관측된다.</p>

<p>이 두 관측 결과 역시 서로 모순되는 것처럼 보이지만 그렇지 않다. 길이를 측정하기 위해서는 물체의 양 끝 지점의 위치를 동시에 측정해서 그 차를 구해야 한다. 그러나 A에서 두 지점을 동시에 측정하였더라도 이것을 B가 관측했을 때는 <a href="/wiki/%EB%8F%99%EC%8B%9C%EC%84%B1%EC%9D%98_%EC%83%81%EB%8C%80%EC%84%B1" title="동시성의 상대성">동시성의 상대성</a>에 의해 양 끝을 다른 시각에 측정한 것으로 관측된다. 따라서 B가 관측하기에는 A가 잘못된 실험을 하는 것으로 보이고 따라서 B의 길이가 짧아졌다는 A의 관측 결과는 B에게 맞지 않는 것이다.</p>

<h2><span class="mw-headline" id=".EC.97.AD.EC.82.AC">역사</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%ED%8A%B9%EC%88%98_%EC%83%81%EB%8C%80%EC%84%B1%EC%9D%B4%EB%A1%A0&amp;action=edit&amp;section=8" title="부분 편집: 역사">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p><a href="/wiki/%EB%A7%A5%EC%8A%A4%EC%9B%B0_%EB%B0%A9%EC%A0%95%EC%8B%9D" title="맥스웰 방정식">맥스웰 방정식</a>에 따르면, <a href="/wiki/%EC%A0%84%EC%9E%90%EA%B8%B0%ED%8C%8C" title="전자기파">전자기파</a>는 특정한 속도를 지닌다. 1905년 이전에는 학자들은 전자기파가 <a href="/wiki/%EC%97%90%ED%85%8C%EB%A5%B4_(%EB%AC%BC%EB%A6%AC)" title="에테르 (물리)">에테르</a>라고 불리었던 <a href="/wiki/%EB%A7%A4%EC%A7%88" title="매질">매질</a> 위에서 전파되며, 전자기파의 속도는 에테르에 대하여 상대적인 속도라고 해석하였다. 그러나 1887년에 행해진 <a href="/wiki/%EB%A7%88%EC%9D%B4%EC%BB%AC%EC%8A%A8-%EB%AA%B0%EB%A6%AC_%EC%8B%A4%ED%97%98" title="마이컬슨-몰리 실험">마이컬슨-몰리 실험</a>은 <a href="/wiki/%EC%97%90%ED%85%8C%EB%A5%B4_(%EB%AC%BC%EB%A6%AC)" title="에테르 (물리)">에테르</a>의 존재를 증명하지 못하였다. 이를 설명하기 위하여 1889년에 조지 피츠제럴드(<span lang="en" xml:lang="en">George Francis FitzGerald</span>)는 물체가 높은 속도로 움직일 때는 그 길이가 축소된다고 제안하였고<sup class="reference" id="cite_ref-3"><a href="#cite_note-3">[3]</a></sup>, 이에 기반하여 <a href="/wiki/%ED%97%A8%EB%93%9C%EB%A6%AD_%EB%A1%9C%EB%9F%B0%EC%B8%A0" title="헨드릭 로런츠">헨드릭 로런츠</a><sup class="reference" id="cite_ref-4"><a href="#cite_note-4">[4]</a></sup><sup class="reference" id="cite_ref-5"><a href="#cite_note-5">[5]</a></sup><sup class="reference" id="cite_ref-6"><a href="#cite_note-6">[6]</a></sup> 와 <a href="/wiki/%EC%A1%B0%EC%A7%80%ED%94%84_%EB%9D%BC%EB%AA%A8%EC%96%B4" title="조지프 라모어">조지프 라모어</a><sup class="reference" id="cite_ref-7"><a href="#cite_note-7">[7]</a></sup> 는 오늘날 <a href="/wiki/%EB%A1%9C%EB%9F%B0%EC%B8%A0_%EB%B3%80%ED%99%98" title="로런츠 변환">로런츠 변환</a>이라고 불리는 변환을 도입하였다. 그러나 이들은 이 효과들을 <a class="mw-disambig" href="/wiki/%EC%97%90%ED%85%8C%EB%A5%B4" title="에테르">에테르</a>에 의한 전기적 효과라고 취급하였다.</p>
<p><a href="/wiki/%EC%95%8C%EB%B2%A0%EB%A5%B4%ED%8A%B8_%EC%95%84%EC%9D%B8%EC%8A%88%ED%83%80%EC%9D%B8" title="알베르트 아인슈타인">알베르트 아인슈타인</a>은 1905년 논문 〈움직이는 물체의 전기역학에 대하여〉(<span style="font-size: smaller;"><a href="/wiki/%EB%8F%85%EC%9D%BC%EC%96%B4" title="독일어">독일어</a>:</span> <span lang="de" xml:lang="de">Zur Elektrodynamik bewegter Körper</span>)<sup class="reference" id="cite_ref-8"><a href="#cite_note-8">[8]</a></sup> 에서 <a href="/wiki/%EB%A1%9C%EB%9F%B0%EC%B8%A0_%EB%B3%80%ED%99%98" title="로런츠 변환">로런츠 변환</a> 및 관련된 공식들이 <a href="/wiki/%EC%8B%9C%EA%B3%B5%EA%B0%84" title="시공간">시공간</a>의 근본적인 성질임을 두 개의 기본 가정 아래 증명하였다. 같은 해에 아인슈타인은 <a class="mw-redirect" href="/wiki/%EC%A7%88%EB%9F%89-%EC%97%90%EB%84%88%EC%A7%80_%EB%93%B1%EA%B0%80%EC%84%B1" title="질량-에너지 등가성">질량-에너지 등가성</a>에 대한 논문 〈물체의 <a href="/wiki/%EA%B4%80%EC%84%B1" title="관성">관성</a>이 그 <a href="/wiki/%EC%97%90%EB%84%88%EC%A7%80" title="에너지">에너지</a> 함량에 관계있는가?〉<sup class="reference" id="cite_ref-9"><a href="#cite_note-9">[9]</a></sup> 를 발표하였다. 이 두 논문은 오늘날 특수 상대성 이론의 시초로 평가된다.</p>
