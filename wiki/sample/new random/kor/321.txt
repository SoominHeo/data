<p><br/>
<b>미군 장갑차에 의한 중학생 압사 사건</b>은 <a href="/wiki/2002%EB%85%84" title="2002년">2002년</a> <a href="/wiki/6%EC%9B%94_13%EC%9D%BC" title="6월 13일">6월 13일</a> 당시 <a href="/wiki/%EC%A1%B0%EC%96%91%EC%A4%91%ED%95%99%EA%B5%90" title="조양중학교">조양중학교</a> 2학년이던 신효순, 심미선이 <a href="/wiki/%EA%B2%BD%EA%B8%B0%EB%8F%84" title="경기도">경기도</a> <a href="/wiki/%EC%96%91%EC%A3%BC%EC%8B%9C" title="양주시">양주시</a> <a href="/wiki/%EA%B4%91%EC%A0%81%EB%A9%B4" title="광적면">광적면</a> 효촌리 소재 <a href="/wiki/%EA%B5%AD%EA%B0%80%EC%A7%80%EC%9B%90%EC%A7%80%EB%B0%A9%EB%8F%84_%EC%A0%9C56%ED%98%B8%EC%84%A0" title="국가지원지방도 제56호선">국가지원지방도 제56호선</a>에서 갓길을 걷다 <a href="/wiki/%EC%A3%BC%ED%95%9C_%EB%AF%B8%EA%B5%B0" title="주한 미군">주한 미군</a> <a class="mw-redirect" href="/wiki/%EB%AF%B8_%EB%B3%B4%EB%B3%91_2%EC%82%AC%EB%8B%A8" title="미 보병 2사단">미 보병 2사단</a> 대대 전투력 훈련을 위해 이동 중이던 <a href="/wiki/%EB%B6%80%EA%B5%90" title="부교">부교</a> 운반용 <a href="/wiki/%EC%9E%A5%EA%B0%91%EC%B0%A8" title="장갑차">장갑차</a>에 깔려 현장에서 숨진 사건이다.</p>

<h2><span class="mw-headline" id=".EC.82.AC.EA.B1.B4_.EB.B0.9C.EC.83.9D.EA.B3.BC_.EC.A7.84.ED.96.89">사건 발생과 진행</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%AF%B8%EA%B5%B0_%EC%9E%A5%EA%B0%91%EC%B0%A8%EC%97%90_%EC%9D%98%ED%95%9C_%EC%A4%91%ED%95%99%EC%83%9D_%EC%95%95%EC%82%AC_%EC%82%AC%EA%B1%B4&amp;action=edit&amp;section=1" title="부분 편집: 사건 발생과 진행">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p><a href="/wiki/2002%EB%85%84" title="2002년">2002년</a> <a href="/wiki/6%EC%9B%94_13%EC%9D%BC" title="6월 13일">6월 13일</a>은 목요일이며 <a href="/wiki/%EB%8C%80%ED%95%9C%EB%AF%BC%EA%B5%AD_%EC%A0%9C3%ED%9A%8C_%EC%A7%80%EB%B0%A9_%EC%84%A0%EA%B1%B0" title="대한민국 제3회 지방 선거">지방자치단체장 선거</a>가 있는 날이였다. 광적면 효촌2리에 살고 있던 신효순과 심미선(당시 14살)은 국도를 따라 언덕을 넘어 덕도리 쪽으로 300 <a href="/wiki/%EB%AF%B8%ED%84%B0" title="미터">미터</a>만 가면 있는 친구(김다희)가 사는 초가집이라는 이름의 식당에 모여 <a href="/wiki/%EC%9D%98%EC%A0%95%EB%B6%80" title="의정부">의정부</a>에 놀러가기로 하였다. 학생들은 졸업생이 10명(그 중 여학생은 여섯명)에 불과한 <a href="/wiki/%ED%9A%A8%EC%B4%8C%EC%B4%88%EB%93%B1%ED%95%99%EA%B5%90" title="효촌초등학교">효촌초등학교</a>를 나온 동창으로, 다음날은 효순양의 생일이기도 해서 여학생 다섯명이 모이기로 약속했었다.<sup class="reference" id="cite_ref-do_1-0"><a href="#cite_note-do-1">[1]</a></sup></p>
<p>사고가 난 길은 편도 1차로(폭 3.3미터)에 불과한 <a class="mw-redirect" href="/wiki/%EC%A7%80%EB%B0%A9%EB%8F%84" title="지방도">지방도</a>이며, 현장은 법원리(서쪽)와 덕도리(동쪽) 사이의 골짜기로 급경사에 곡각지로 남쪽은 산을 깎아 도로를 만들어 별도 인도가 없어 차가 온다면 사람이 피할 곳이 없는 길이다. 서쪽의 법원리에서 한참 내리막을 달려오던 차가 마을 앞에서 갑자기 도로 사정이 바뀌어 오르막이 시작되는데 오른 쪽(남쪽)으로는 산을 끼고 휘어져 올라간다.<sup class="reference" id="cite_ref-do_1-1"><a href="#cite_note-do-1">[1]</a></sup></p>
<p>이날 낮 10시 30분 경 두 여학생은 마을을 나와 친구 집으로 길을 따라 올라가고 있었다. 그 뒤에서는 <a href="/wiki/%EC%A3%BC%ED%95%9C_%EB%AF%B8%EA%B5%B0" title="주한 미군">주한 미군</a> <a class="mw-redirect" href="/wiki/%EB%AF%B8_%EB%B3%B4%EB%B3%91_2%EC%82%AC%EB%8B%A8" title="미 보병 2사단">미 보병 2사단</a> 44 공병대대 소속 <a href="/wiki/%EB%B6%80%EA%B5%90" title="부교">부교</a> 운반용 장갑차가 법원리 쪽에서 내려와 약간 왼쪽으로 틀어 막 언덕으로 올라오고 있었다. 차량 행렬은 선두 안내차량 1대, <a class="mw-redirect" href="/wiki/%EB%B3%91%EB%A0%A5_%EC%88%98%EC%86%A1_%EC%9E%A5%EA%B0%91%EC%B0%A8" title="병력 수송 장갑차">병력 수송 장갑차</a>, 그 뒤로 사고차량, 일반 공병궤도 차량 3대, 후미 안내차량 1대였다. 이때 맞은편에서 <a href="/wiki/M2/M3_%EB%B8%8C%EB%9E%98%EB%93%A4%EB%A6%AC" title="M2/M3 브래들리">M2/M3 브래들리</a> 기갑 전투차량 5대가 덕도리에서 무건리 훈련장으로 오고 있었다. 사고가 난 도로의 폭은 3.3미터 정도인데 반해 사고차량의 폭은 3.65미터다. 사고차량이 중앙선을 침범하지 않으려면 갓길을 걷고 있던 학생을 치지 않을 수 없다.<sup class="reference" id="cite_ref-do_1-2"><a href="#cite_note-do-1">[1]</a></sup></p>
<p>사고가 난 도로는 인도도 따로 없는 편도 1차선의 좁은 도로로, 주민들은 평소 갓길을 인도삼아 통행해 왔다. <a href="/wiki/2002%EB%85%84" title="2002년">2002년</a> <a href="/wiki/6%EC%9B%94_13%EC%9D%BC" title="6월 13일">6월 13일</a> 사고가 난 이후 유족들은 "당시 사고 차량의 너비가 도로 폭보다 넓은 데다 마주오던 차량과 무리하게 교행을 시도했다는 점에서 이번 사고는 이미 예견된 살인행위였다"라고 주장했다.</p>
<p>미군 당국은 사고 당일 <a class="mw-redirect" href="/wiki/%EB%AF%B88%EA%B5%B0" title="미8군">미8군</a> 사령관의 유감의 뜻을 전하고, 다음날인 6월 14일에는 <a class="mw-redirect" href="/wiki/%EB%AF%B8_%EB%B3%B4%EB%B3%91_2%EC%82%AC%EB%8B%A8" title="미 보병 2사단">미 보병 2사단</a> 참모장 등이 분향소를 직접 방문해 문상하고, 피해 유가족에게 각각 위로금 100만과 배상금 1억 9천여만원씩을 전달하는 등 사고 수습에 나섰다. 미군 측은 15일 장례식을 치르면 사단장과 면담할 수 있도록 하겠다고 약속했으나 장례식을 마친 후 미군 측은 번역상의 실수를 이유로 면담 약속을 파기하였다.</p>

<p>미군 당국은 <a href="/wiki/7%EC%9B%94_3%EC%9D%BC" title="7월 3일">7월 3일</a> 운전병과 관제병을 <a class="mw-redirect" href="/wiki/%EA%B3%BC%EC%8B%A4%EC%B9%98%EC%82%AC%EC%A3%84" title="과실치사죄">과실치사죄</a>로 미 <a href="/wiki/%EA%B5%B0%EC%82%AC%EB%B2%95%EC%9B%90" title="군사법원">군사법원</a>에 기소하는 한편, 라포트 주한 미군 사령관의 사과를 전했다. 그와 별도로 <a class="mw-redirect" href="/wiki/%EB%8C%80%ED%95%9C%EB%AF%BC%EA%B5%AD_%EA%B2%80%EC%B0%B0" title="대한민국 검찰">대한민국 검찰</a>도 관련 미군에 대해 자체 조사를 벌이기로 했다. 이는 유족들이 <a href="/wiki/6%EC%9B%94_28%EC%9D%BC" title="6월 28일">6월 28일</a> 차량 운전병과 관제병, 미2사단장 등 미군 책임자 6명을 업무상 과실치사 혐의로 의정부지청에 고소하고, 미측의 재판권 포기를 요청한 데 따른 것이다. 하지만 미군 측은 신변 위협을 이유로 검찰의 소환조사에 응하지 않았다. <a href="/wiki/%EB%8C%80%ED%95%9C%EB%AF%BC%EA%B5%AD_%EB%B2%95%EB%AC%B4%EB%B6%80" title="대한민국 법무부">법무부</a>는 <a href="/wiki/7%EC%9B%94_10%EC%9D%BC" title="7월 10일">7월 10일</a>, 사상 처음으로 미측에 재판권 포기 요청서를 보냈다. 그러나 <a href="/wiki/8%EC%9B%94_7%EC%9D%BC" title="8월 7일">8월 7일</a> 미군 당국은 “동 사고가 <a class="mw-redirect" href="/wiki/%EA%B3%B5%EB%AC%B4" title="공무">공무</a> 중에 일어난 사고이기에 재판권이 미국에 있으며, 이제껏 미국이 1차적 재판권을 포기한 전례가 없다.”라는 이유를 들어 재판권 포기를 거부했다.</p>
<p>이후 <a href="/wiki/11%EC%9B%94_18%EC%9D%BC" title="11월 18일">11월 18일</a>부터 <a href="/wiki/11%EC%9B%94_23%EC%9D%BC" title="11월 23일">11월 23일</a>까지 동두천 <a href="/wiki/%EC%BA%A0%ED%94%84_%EC%BC%80%EC%9D%B4%EC%8B%9C" title="캠프 케이시">캠프 케이시</a> 내 미 군사법정에서 열린 <a href="/wiki/%EA%B5%B0%EC%82%AC%EC%9E%AC%ED%8C%90" title="군사재판">군사재판</a>에서 배심원단은 기소된 미군 2명 모두에게 공무를 행하던 중 발생한 과실사고임을 근거로 무죄(not guilty) 평결을 내렸다. 그 후 미군은 무죄 평결이 있은 지 5일 만인 <a href="/wiki/11%EC%9B%94_27%EC%9D%BC" title="11월 27일">11월 27일</a> 사죄 성명을 발표하였다.</p>
<p>한편 미국의 무성의한 태도에 대한 반미시위가 계속 이어지면서 미국의 고위 관리들이 직간접적으로 사죄하였고 <a class="mw-redirect" href="/wiki/%EC%A1%B0%EC%A7%80_%EC%9B%8C%EC%BB%A4_%EB%B6%80%EC%8B%9C" title="조지 워커 부시">조지 워커 부시</a> 대통령은 유감을 표명하였다.</p>

<li><a href="/wiki/2002%EB%85%84" title="2002년">2002년</a> <a href="/wiki/6%EC%9B%94_28%EC%9D%BC" title="6월 28일">6월 28일</a> - 캠프 하우즈 부대장 해롤드 대령 출국</li>

<li><a href="/wiki/2002%EB%85%84" title="2002년">2002년</a> <a href="/wiki/7%EC%9B%94_10%EC%9D%BC" title="7월 10일">7월 10일</a> - <a href="/wiki/%EB%8C%80%ED%95%9C%EB%AF%BC%EA%B5%AD_%EB%B2%95%EB%AC%B4%EB%B6%80" title="대한민국 법무부">대한민국 법무부</a>는 사상 처음으로 미국 측에 재판권 포기 요청서를 보냄</li>
<li>사고 현장에 추모비 건립.</li>

<p>사건 발생 10년 후, 미선 양의 아버지는 "사고를 낸 미군도 이젠 편히 지내길 바란다"라고 말했다. 심 씨는 "단순한 사고라고 생각하는데 (미군이) 애들이 미워서 낸 게 아니지 않나"라며 "얼굴도 모르지만 그 미군들도 이젠 마음의 짐을 덜고 편하게 살았으면 좋겠다"라고 말했다. 효순 양의 부친 신현수 씨(58)도 마찬가지 생각일 것이라고 전했다.<sup class="reference" id="cite_ref-3"><a href="#cite_note-3">[3]</a></sup></p>
<p>2012년 5월 이명박 정부 시절, 한미는 주한미군지위협정(SOFA)에 대한 개정에 합의하고 범죄 피의자인 미군 관계자의 신병을 기소 전에 한국 당국에 인도할 수 있도록 협정 운용을 개정하기로 합의했다.<sup class="reference" id="cite_ref-4"><a href="#cite_note-4">[4]</a></sup></p>
<p>2012년 12월 8일에는 장갑차 사건 때문에 2004년에 주한미군 반대 집회에서 랩으로 반미 노래를 부르면서도 괜히 미국한테서의 혐의를 사고 말았던 가수 <a href="/wiki/%EC%8B%B8%EC%9D%B4" title="싸이">싸이</a>가 <a href="/wiki/%EB%AF%B8%EA%B5%AD" title="미국">미국</a>의 대중한테 공식 사과한다는 뜻을 밝혔다.<sup class="reference" id="cite_ref-5"><a href="#cite_note-5">[5]</a></sup></p>
