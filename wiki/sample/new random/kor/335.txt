<h1 class="firstHeading" id="firstHeading" lang="ko">투석형</h1>
<div class="mw-body-content" id="bodyContent">
<div id="siteSub">위키백과, 우리 모두의 백과사전.</div>
<div id="contentSub"></div>
<div class="mw-jump" id="jump-to-nav">
					이동:					<a href="#mw-head">둘러보기</a>, 					<a href="#p-search">검색</a>
</div>
<div class="mw-content-ltr" dir="ltr" id="mw-content-text" lang="ko"><script>function mfTempOpenSection(id){var block=document.getElementById("mf-section-"+id);block.className+=" open-block";block.previousSibling.className+=" open-block";}</script><p><b>투석형</b>(投石刑)이란 죄수에게 돌을 던져 죽이는 형벌이다. <a href="/wiki/%EC%9D%B4%EB%9E%80" title="이란">이란</a>과 <a href="/wiki/%EC%95%84%ED%94%84%EA%B0%80%EB%8B%88%EC%8A%A4%ED%83%84" title="아프가니스탄">아프가니스탄</a>에서 <a href="/wiki/%EC%82%AC%ED%98%95" title="사형">사형</a> 방법으로 사용하고 있으며, <a href="/wiki/%EC%9D%B4%EB%9E%80" title="이란">이란</a>의 경우는 공개 투석형을 실시한다.</p>
<h2><span class="mw-headline" id=".EC.9C.A0.EB.8C.80.EA.B5.90.EC.99.80_.EA.B8.B0.EB.8F.85.EA.B5.90.EC.97.90.EC.84.9C.EC.9D.98_.ED.88.AC.EC.84.9D.ED.98.95"><a href="/wiki/%EC%9C%A0%EB%8C%80%EA%B5%90" title="유대교">유대교</a>와 <a href="/wiki/%EA%B8%B0%EB%8F%85%EA%B5%90" title="기독교">기독교</a>에서의 투석형</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%ED%88%AC%EC%84%9D%ED%98%95&amp;action=edit&amp;section=1" title="부분 편집: 유대교와 기독교에서의 투석형">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p><a href="/wiki/%EC%84%B1%EA%B2%BD" title="성경">성경</a>에서는 투석형 또는 석형으로 불리는 돌던져 죽이는 형벌로 간음(<a class="mw-redirect" href="/wiki/%EC%9A%94%ED%95%9C_%EB%B3%B5%EC%9D%8C%EC%84%9C" title="요한 복음서">요한 복음서</a> 8:1-11), 횡령(여호수아 7장), 신성모독등의 범죄를 처벌한 이야기들이 나온다. 이를테면 <a href="/wiki/%EA%B8%B0%EB%8F%85%EA%B5%90" title="기독교">기독교</a>의 첫 <a href="/wiki/%EB%B6%80%EC%A0%9C_(%EA%B8%B0%EB%8F%85%EA%B5%90)" title="부제 (기독교)">부제</a>/<a href="/wiki/%EC%A7%91%EC%82%AC" title="집사">집사</a> 중 한 사람이었던 <a class="mw-redirect" href="/wiki/%EC%8A%A4%EB%8D%B0%ED%8C%8C%EB%85%B8" title="스데파노">스데파노</a>/스데반은 <a href="/wiki/%EB%AA%A8%EC%84%B8" title="모세">모세</a>를 모독했다는 죄로 기소되어 <a href="/wiki/%EC%82%B0%ED%97%A4%EB%93%9C%EB%A6%B0" title="산헤드린">산헤드린</a> 의회에 의해 석형을 당하였다.<sup class="reference" id="cite_ref-1"><a href="#cite_note-1">[1]</a></sup> 당시 <a href="/wiki/%EB%A1%9C%EB%A7%88_%EC%A0%9C%EA%B5%AD" title="로마 제국">로마 제국</a>에서는 식민지의 정치적 반항에 대해서는 공개 처형인 <a href="/wiki/%EC%8B%AD%EC%9E%90%EA%B0%80%ED%98%95" title="십자가형">십자가형</a>, <a href="/wiki/%ED%95%99%EC%82%B4" title="학살">학살</a> 같은 로마 군대의 무자비한 <a href="/wiki/%ED%8F%AD%EB%A0%A5" title="폭력">폭력</a>으로 탄압했지만, <a href="/wiki/%EC%A2%85%EA%B5%90" title="종교">종교</a>적 문제에 대해서는 간섭하지 않았기 때문에 가능한 일이었다.</p>
<h2><span class="mw-headline" id=".EC.9D.B4.EC.8A.AC.EB.9E.8C.EA.B5.90.EC.97.90.EC.84.9C.EC.9D.98_.ED.88.AC.EC.84.9D.ED.98.95"><a href="/wiki/%EC%9D%B4%EC%8A%AC%EB%9E%8C%EA%B5%90" title="이슬람교">이슬람교</a>에서의 투석형</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%ED%88%AC%EC%84%9D%ED%98%95&amp;action=edit&amp;section=2" title="부분 편집: 이슬람교에서의 투석형">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p><a class="mw-redirect" href="/wiki/%EC%9D%B4%EC%8A%AC%EB%9E%8C" title="이슬람">이슬람</a> 율법에서는 <a class="mw-redirect" href="/wiki/%EA%B0%84%ED%86%B5%EC%A3%84" title="간통죄">간통죄</a>는 <a href="/wiki/%EA%B0%95%EA%B0%84" title="강간">강간</a>, <a href="/wiki/%EC%82%B4%EC%9D%B8" title="살인">살인</a>과 같이 간주하여 투석형에 처하도록 되어 있다. <a href="/wiki/%EB%82%A8%EC%84%B1" title="남성">남성</a>의 경우에는 손을 뒤로 묶은 채 허리까지 땅에 파묻힌 후 투석형에 처하며, <a href="/wiki/%EC%97%AC%EC%84%B1" title="여성">여성</a>의 경우는 목까지 땅에 파묻힌 후 투석형에 처한다.</p>
<p><a class="mw-redirect" href="/wiki/AFP_%ED%86%B5%EC%8B%A0" title="AFP 통신">AFP 통신</a>의 보도에 따르면 <a href="/wiki/2007%EB%85%84" title="2007년">2007년</a> <a href="/wiki/7%EC%9B%94_10%EC%9D%BC" title="7월 10일">7월 10일</a>, 이란에서 간통죄를 저지른 남성 대해 투석형에 처하였다고 한다.</p>
