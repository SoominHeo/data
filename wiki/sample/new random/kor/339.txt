<h2><span class="mw-headline" id=".ED.81.B4.EB.9F.BD_.EA.B2.BD.EB.A0.A5">클럽 경력</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%8B%A4%EC%B9%B4%ED%95%98%EA%B8%B0_%EC%9A%94%EC%A7%80%EB%A1%9C&amp;action=edit&amp;section=1" title="부분 편집: 클럽 경력">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>2002년 <a href="/wiki/%EC%82%B0%ED%94%84%EB%A0%88%EC%B2%B4_%ED%9E%88%EB%A1%9C%EC%8B%9C%EB%A7%88" title="산프레체 히로시마">산프레체 히로시마</a>의 유소년 팀에 입단하였다. 2003년 4월 5일 16세의 나이로 <a href="/wiki/%EC%87%BC%EB%82%9C_%EB%B2%A8%EB%A7%88%EB%A0%88" title="쇼난 벨마레">쇼난 벨마레</a>와의 경기에 출전하며 산프레체 히로시마 팀 역사상 최연소 프로 경기 출전 선수가 되었다. 같은 해 11월에는 팀과 프로 계약을 체결하며 클럽 최초로 고등학생 프로 계약 선수가 되었다. 하지만 이후 부상과 이로 인한 컨디션 난조가 겹쳐 출전 기회를 잡지 못하다가 2006년 <a href="/wiki/%EC%97%90%ED%9E%88%EB%A9%94_FC" title="에히메 FC">에히메 FC</a>로 임대 이적하였다. 에히메에서 주전으로 활약하며 44경기에 나서 3골을 기록하였다.</p>
<p>2008년 팀의 주전 선수들이 부상으로 전력에서 이탈한 후 공격형 미드필더 자리에서 주전으로 활약하기 시작하였다. 2012년에는 리그에서 2골 12도움을 기록하며 <a href="/wiki/%EB%82%98%EC%B9%B4%EB%AC%B4%EB%9D%BC_%EA%B2%90%EA%B3%A0" title="나카무라 겐고">나카무라 겐고</a>에 이어 도움 랭킹 2위를 차지하였고 시즌 베스트 11에 선정되었다. 2013년엔 등번호 10번을 배정받았다.</p>
<p>2015년 1월 14일 <a href="/wiki/AFC_%EC%B1%94%ED%94%BC%EC%96%B8%EC%8A%A4%EB%A6%AC%EA%B7%B8_2014" title="AFC 챔피언스리그 2014">2014 ACL</a> 우승팀 <a href="/wiki/%EC%9B%A8%EC%8A%A4%ED%84%B4_%EC%8B%9C%EB%93%9C%EB%8B%88_%EC%9B%90%EB%8D%94%EB%9F%AC%EC%8A%A4_FC" title="웨스턴 시드니 원더러스 FC">웨스턴 시드니</a>로 이적하였고 2015년 2월 25일 열린 <a href="/wiki/%EA%B0%80%EC%8B%9C%EB%A7%88_%EC%95%A4%ED%8B%80%EB%9F%AC%EC%8A%A4" title="가시마 앤틀러스">가시마 앤틀러스</a>와의 <a href="/wiki/AFC_%EC%B1%94%ED%94%BC%EC%96%B8%EC%8A%A4%EB%A6%AC%EA%B7%B8_2015" title="AFC 챔피언스리그 2015">2015 ACL</a>경기에서 1골 1도움을 기록하며 경기 MVP로 선정되었다. 같은 해 6월 6일 <a href="/wiki/%EC%9B%A8%EC%8A%A4%ED%84%B4_%EC%8B%9C%EB%93%9C%EB%8B%88_%EC%9B%90%EB%8D%94%EB%9F%AC%EC%8A%A4_FC" title="웨스턴 시드니 원더러스 FC">웨스턴 시드니</a>의 재계약 제안을 거절하고 퇴단이 발표되었다.</p>
<p>2015년 6월 16일 <a href="/wiki/FC_%EC%84%9C%EC%9A%B8" title="FC 서울">FC 서울</a> 입단이 공식 발표되었다. 같은 해 7월 25일 <a href="/wiki/%EC%9D%B8%EC%B2%9C_%EC%9C%A0%EB%82%98%EC%9D%B4%ED%8B%B0%EB%93%9C_FC" title="인천 유나이티드 FC">인천 유나이티드</a>와의 경기에서 리그 데뷔전을 치렀고, 8월 19일 <a href="/wiki/%EB%B6%80%EC%82%B0_%EC%95%84%EC%9D%B4%ED%8C%8C%ED%81%AC" title="부산 아이파크">부산 아이파크</a>전에서 데뷔골을 기록하였다.</p>
<h2><span class="mw-headline" id=".EA.B5.AD.EA.B0.80.EB.8C.80.ED.91.9C_.EA.B2.BD.EB.A0.A5">국가대표 경력</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%8B%A4%EC%B9%B4%ED%95%98%EA%B8%B0_%EC%9A%94%EC%A7%80%EB%A1%9C&amp;action=edit&amp;section=2" title="부분 편집: 국가대표 경력">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p><a href="/wiki/2013%EB%85%84_EAFF_%EB%8F%99%EC%95%84%EC%8B%9C%EC%95%88%EC%BB%B5" title="2013년 EAFF 동아시안컵">2013년 동아시안컵</a>에 출전하는 <a href="/wiki/%EC%9D%BC%EB%B3%B8_%EC%B6%95%EA%B5%AC_%EA%B5%AD%EA%B0%80%EB%8C%80%ED%91%9C%ED%8C%80" title="일본 축구 국가대표팀">일본 대표팀</a> 명단에 이름을 올렸고 <a class="mw-redirect" href="/wiki/%EC%A4%91%EA%B5%AD_%EC%B6%95%EA%B5%AC_%EA%B5%AD%EA%B0%80%EB%8C%80%ED%91%9C%ED%8C%80" title="중국 축구 국가대표팀">중국</a>과의 경기에서 <a href="/wiki/A%EB%A7%A4%EC%B9%98" title="A매치">A매치</a> 데뷔전을 치렀다.</p>
