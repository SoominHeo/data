<p><b>워너 브라더스</b>(Warner Bros.)는 <b>워너 브라더스 픽처스</b>(Warner Bros. Pictures, Inc.)또는 <b>워너 브로스</b> 라는 이름으로 알려진 <a href="/wiki/%EB%AF%B8%EA%B5%AD" title="미국">미국</a>의 <a href="/wiki/%EC%98%81%ED%99%94" title="영화">영화</a> 제작 &amp; 배급사 겸 TV 프로그램 제작과 음악 사업을 집행하고 있는 <a href="/wiki/%ED%83%80%EC%9E%84_%EC%9B%8C%EB%84%88" title="타임 워너">타임 워너</a> 계열사에 속해 있는 종합 미디어 &amp; 엔터테인먼트 기업으로써, 세계 영화사들 중에 가장 큰 규모를 자랑한다. 본사는 <a href="/wiki/%EC%BA%98%EB%A6%AC%ED%8F%AC%EB%8B%88%EC%95%84_%EC%A3%BC" title="캘리포니아 주">캘리포니아 주</a> <a class="mw-disambig" href="/wiki/%EB%B2%84%EB%B1%85%ED%81%AC" title="버뱅크">버뱅크</a>에 위치해있으며, <a href="/wiki/%EB%AF%B8%EA%B5%AD%EC%98%81%ED%99%94%ED%98%91%ED%9A%8C" title="미국영화협회">미국영화협회</a>에 가입한 할리우드 메이저 6대 스튜디오중 하나이다.</p>

<h3><span class="mw-headline" id=".EC.9B.8C.EB.84.88_.EB.B8.8C.EB.9D.BC.EB.8D.94.EC.8A.A4.EC.9D.98_.EC.B0.BD.EB.A6.BD">워너 브라더스의 창립</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%9B%8C%EB%84%88_%EB%B8%8C%EB%9D%BC%EB%8D%94%EC%8A%A4&amp;action=edit&amp;section=2" title="부분 편집: 워너 브라더스의 창립">편집</a><span class="mw-editsection-bracket">]</span></span></h3>

<p><a href="/wiki/1918%EB%85%84" title="1918년">1918년</a> 4명의 워너 형제인 <a class="new" href="/w/index.php?title=%EC%9E%AD_%EC%9B%8C%EB%84%88&amp;action=edit&amp;redlink=1" title="잭 워너 (없는 문서)">잭 워너</a>, <a class="new" href="/w/index.php?title=%ED%95%B4%EB%A6%AC_%EC%9B%8C%EB%84%88&amp;action=edit&amp;redlink=1" title="해리 워너 (없는 문서)">해리 워너</a>, <a class="new" href="/w/index.php?title=%EC%95%A8%EB%B2%84%ED%8A%B8_%EC%9B%8C%EB%84%88&amp;action=edit&amp;redlink=1" title="앨버트 워너 (없는 문서)">앨버트 워너</a>, <a class="new" href="/w/index.php?title=%EC%83%98_%EC%9B%8C%EB%84%88&amp;action=edit&amp;redlink=1" title="샘 워너 (없는 문서)">샘 워너</a>는 '워너 브라더스, 웨스트 코스트 스튜디오'를 설립하게 된다. 이후 <a href="/wiki/1925%EB%85%84" title="1925년">1925년</a>에 이들은 <b>워너 브라더스 픽처스</b>로 이름을 바꾸게 된다. 초기 워너브라더스는 재정이 좋지 않아 영화계에서 인정받지 않는 사업에 나서기 시작하는데 <a href="/wiki/1925%EB%85%84" title="1925년">1925년</a>에 워너브라더스는 <a class="new" href="/w/index.php?title=%EB%B0%94%EC%9D%B4%EC%98%A4%EA%B7%B8%EB%9E%98%ED%94%84&amp;action=edit&amp;redlink=1" title="바이오그래프 (없는 문서)">바이오그래프</a>를 인수하게 된다. 이때 영화계에서 무시받던 기술인 <a class="mw-redirect" href="/wiki/%EB%B0%9C%EC%84%B1_%EC%98%81%ED%99%94" title="발성 영화">발성 영화</a> 개발을 위해 워너브라더스는 <a class="new" href="/w/index.php?title=%EC%9B%A8%EC%8A%A4%ED%84%B4_%EC%9D%BC%EB%A0%89%ED%8A%B8%EB%A6%AD&amp;action=edit&amp;redlink=1" title="웨스턴 일렉트릭 (없는 문서)">웨스턴 일렉트릭</a>과 함께 <a class="new" href="/w/index.php?title=%EB%B0%94%EC%9D%B4%ED%83%80%ED%8F%B0&amp;action=edit&amp;redlink=1" title="바이타폰 (없는 문서)">바이타폰</a> (Vitaphone)을 설립하게 된다.</p>
<h3><span class="mw-headline" id="1925.EB.85.84.7E1935.EB.85.84_:_.EC.86.8C.EB.A6.AC.2C.EC.83.89.EC.83.81.2C.EB.A9.8B_.28.EA.B0.B1.EC.8A.A4.ED.84.B0.EC.99.80_.EB.AE.A4.EC.A7.80.EC.BB.AC_.EC.98.81.ED.99.94.EC.97.90_.EA.B4.80.ED.95.9C_.ED.88.AC.EC.9E.90.29">1925년~1935년 : 소리,색상,멋 (갱스터와 뮤지컬 영화에 관한 투자)</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%9B%8C%EB%84%88_%EB%B8%8C%EB%9D%BC%EB%8D%94%EC%8A%A4&amp;action=edit&amp;section=3" title="부분 편집: 1925년~1935년 : 소리,색상,멋 (갱스터와 뮤지컬 영화에 관한 투자)">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>이러면서 <a href="/wiki/1926%EB%85%84" title="1926년">1926년</a>에 음악이 나오는 《<a class="mw-redirect" href="/wiki/%EB%8F%88_%EC%A3%BC%EC%95%99" title="돈 주앙">돈 주앙</a>》을 개발하게 된다. 다음 해인 <a href="/wiki/1927%EB%85%84" title="1927년">1927년</a>에 최초의 발성 영화인 《<a class="new" href="/w/index.php?title=%EC%9E%AC%EC%A6%88_%EC%8B%B1%EC%96%B4_(1927%EB%85%84_%EC%98%81%ED%99%94)&amp;action=edit&amp;redlink=1" title="재즈 싱어 (1927년 영화) (없는 문서)">재즈 싱어</a>》를 개발하게 된다. 《재즈 싱어》의 개봉은 워너브라더스를 단숨에 메이저 스튜디오로 올려 놓았다. 하지만 《재즈 싱어》의 성공으로 메이저 스튜디오로 들어갔지만 워너의 재정 상태는 좋지 않았다. <a href="/wiki/1929%EB%85%84" title="1929년">1929년</a>, 예전에 메이저였던 <a class="mw-redirect" href="/wiki/%ED%8D%BC%EC%8A%A4%ED%8A%B8%EB%84%A4%EC%85%94%EB%84%90_%ED%94%BD%EC%B2%98%EC%8A%A4" title="퍼스트네셔널 픽처스">퍼스트네셔널 픽처스</a>를 인수하면서 워너는 한가지 장르에 집중적으로 하기 시작하였는데 그것이 바로 갱스터와 뮤지컬 영화였다.</p>
<p>먼저 갱스터 영화는 <a href="/wiki/%EC%A0%9C%EC%9E%84%EC%8A%A4_%EC%BA%90%EA%B7%B8%EB%8B%88" title="제임스 캐그니">제임스 캐그니</a>, <a href="/wiki/%EC%97%90%EB%93%9C%EC%9B%8C%EB%93%9C_G._%EB%A1%9C%EB%B9%88%EC%8A%A8" title="에드워드 G. 로빈슨">에드워드 G. 로빈슨</a> 등의 배우들을 기용하기 시작하였는데 캐그니는 <a href="/wiki/1931%EB%85%84" title="1931년">1931년</a>, 《<a class="new" href="/w/index.php?title=%EA%B3%B5%EA%B3%B5%EC%9D%98_%EC%A0%81_(1931%EB%85%84_%EC%98%81%ED%99%94)&amp;action=edit&amp;redlink=1" title="공공의 적 (1931년 영화) (없는 문서)">공공의 적</a>》, 로빈슨은 퍼스트네셔널에서 《<a class="new" href="/w/index.php?title=%EB%A6%AC%ED%8B%80_%EC%8B%9C%EC%A0%80_(%EC%98%81%ED%99%94)&amp;action=edit&amp;redlink=1" title="리틀 시저 (영화) (없는 문서)">리틀 시저</a>》 등의 영화에 출연하였다. <a href="/wiki/1932%EB%85%84" title="1932년">1932년</a>에는 <a href="/wiki/%ED%8F%B4_%EB%AC%B4%EB%8B%88" title="폴 무니">폴 무니</a>가 《<a class="new" href="/w/index.php?title=%EB%82%98%EB%8A%94_%ED%83%88%EC%98%A5%EC%88%98&amp;action=edit&amp;redlink=1" title="나는 탈옥수 (없는 문서)">나는 탈옥수</a>》에 출연하였는데 이후 무니는 《<a href="/wiki/%EC%97%90%EB%B0%80_%EC%A1%B8%EB%9D%BC%EC%9D%98_%EC%83%9D%EC%95%A0" title="에밀 졸라의 생애">에밀 졸라의 생애</a>》 등 워너의 영화에 많이 출연하였다.</p>

<p>이때 워너는 <a href="/wiki/%EB%8C%80%EB%A6%B4_F._%EC%9E%AC%EB%84%89" title="대릴 F. 재넉">대릴 F. 재넉</a>, <a href="/wiki/%ED%95%BC_B._%EC%9B%94%EB%A6%AC%EC%8A%A4" title="핼 B. 월리스">핼 B. 월리스</a> 등의 제작자를 보유하게 되었는데 재넉은 이후 <a href="/wiki/20%EC%84%B8%EA%B8%B0_%ED%8F%AD%EC%8A%A4" title="20세기 폭스">20세기 폭스</a>로 건너가게 된다. 또한 워너는 뮤지컬 영화를 제작하기 위해 무대안무가 <a href="/wiki/%EB%B2%84%EC%8A%A4%EB%B9%84_%EB%B2%84%ED%81%B4%EB%A6%AC" title="버스비 버클리">버스비 버클리</a>를 영입하게 된다.</p>
<p>버클리가 맡은 영화로는 《<a class="new" href="/w/index.php?title=42%EB%B2%88%EA%B0%80_(%EC%98%81%ED%99%94)&amp;action=edit&amp;redlink=1" title="42번가 (영화) (없는 문서)">42번가</a>》, 《<a class="new" href="/w/index.php?title=1933%EB%85%84%EC%9D%98_%ED%99%A9%EA%B8%88%EA%B4%91%EB%93%A4&amp;action=edit&amp;redlink=1" title="1933년의 황금광들 (없는 문서)">1933년의 황금광들</a>》, 《<a class="new" href="/w/index.php?title=%ED%92%8B%EB%9D%BC%EC%9D%B4%ED%8A%B8_%ED%8D%BC%EB%A0%88%EC%9D%B4%EB%93%9C&amp;action=edit&amp;redlink=1" title="풋라이트 퍼레이드 (없는 문서)">풋라이트 퍼레이드</a>》 등이 있는데 버클리는 안무를 맡았고 뮤지컬 외 장면은 <a class="new" href="/w/index.php?title=%EB%A1%9C%EC%9D%B4%EB%93%9C_%EB%B2%A0%EC%9D%B4%EC%BB%A8&amp;action=edit&amp;redlink=1" title="로이드 베이컨 (없는 문서)">로이드 베이컨</a> 같은 감독이 맡았다. 이 영화에서 워너는 <a class="new" href="/w/index.php?title=%EB%A3%A8%EB%B9%84_%ED%82%AC%EB%9F%AC&amp;action=edit&amp;redlink=1" title="루비 킬러 (없는 문서)">루비 킬러</a>, <a class="new" href="/w/index.php?title=%EB%94%95_%ED%8C%8C%EC%9B%B0&amp;action=edit&amp;redlink=1" title="딕 파웰 (없는 문서)">딕 파웰</a>, <a href="/wiki/%EC%A7%84%EC%A0%80_%EB%A1%9C%EC%A0%80%EC%8A%A4" title="진저 로저스">진저 로저스</a>같은 배우를 출연시키는데 진저 로저스는 <a class="mw-redirect" href="/wiki/RKO_%EB%9D%BC%EB%94%94%EC%98%A4_%ED%94%BD%EC%B2%98%EC%8A%A4" title="RKO 라디오 픽처스">RKO 라디오 픽처스</a>에서 <a href="/wiki/%ED%94%84%EB%A0%88%EB%93%9C_%EC%95%84%EC%8A%A4%ED%85%8C%EC%96%B4" title="프레드 아스테어">프레드 아스테어</a>와 함께 커플을 이루어 대스타가 된다.</p>
<p>한편 제임스 캐그니는 B급 영화 제작사인 <a class="new" href="/w/index.php?title=%EA%B7%B8%EB%9E%9C%EB%93%9C_%EB%84%A4%EC%85%94%EB%84%90_%ED%94%BD%EC%B2%98%EC%8A%A4&amp;action=edit&amp;redlink=1" title="그랜드 네셔널 픽처스 (없는 문서)">그랜드 네셔널 픽처스</a>에서 다시 오면서 갱스터 영화인 《<a class="new" href="/w/index.php?title=%EB%8D%94%EB%9F%BD%ED%98%80%EC%A7%84_%EC%96%BC%EA%B5%B4%EC%9D%98_%EC%B2%9C%EC%82%AC&amp;action=edit&amp;redlink=1" title="더럽혀진 얼굴의 천사 (없는 문서)">더럽혀진 얼굴의 천사</a>》, 《<a href="/wiki/%EA%B4%91%EB%9E%80%EC%9D%98_20%EB%85%84%EB%8C%80" title="광란의 20년대">광란의 20년대</a>》 등의 영화에 출연하게 되는데 《더럽혀진 얼굴의 천사》의 감독을 <a href="/wiki/%ED%97%9D%EA%B0%80%EB%A6%AC" title="헝가리">헝가리</a> 출신의 <a href="/wiki/%EB%A7%88%EC%9D%B4%ED%81%B4_%EC%BB%A4%ED%8B%B0%EC%A6%88" title="마이클 커티즈">마이클 커티즈</a>가 맡게 되었다. 이후 커티즈는 워너에서 여러 영화를 감독하게 되었다. 이러면서 워너의 4대 갱스터 영화 배우로는 예전부터 갱스터를 찍어 온 제임스 캐그니, 에드워드 G. 로빈슨, <a href="/wiki/%EC%A1%B0%EC%A7%80_%EB%9E%98%ED%94%84%ED%8A%B8" title="조지 래프트">조지 래프트</a>, 그리고 새로 발견한 <a href="/wiki/%ED%97%98%ED%94%84%EB%A6%AC_%EB%B3%B4%EA%B0%80%ED%8A%B8" title="험프리 보가트">험프리 보가트</a>가 있었다. 이외에도 워너에서 보유한 배우로는 폴 무니, <a href="/wiki/%EB%B2%A0%ED%8B%B0_%EB%8D%B0%EC%9D%B4%EB%B9%84%EC%8A%A4" title="베티 데이비스">베티 데이비스</a> 등이 있었다.</p>

<h4><span class="mw-headline" id=".EC.9B.8C.EB.84.88_.EB.B8.8C.EB.9D.BC.EB.8D.94.EC.8A.A4.EC.9D.98_.EB.A7.8C.ED.99.94_.EC.98.81.ED.99.94">워너 브라더스의 만화 영화</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%9B%8C%EB%84%88_%EB%B8%8C%EB%9D%BC%EB%8D%94%EC%8A%A4&amp;action=edit&amp;section=5" title="부분 편집: 워너 브라더스의 만화 영화">편집</a><span class="mw-editsection-bracket">]</span></span></h4>
<p>워너에서는 <a class="new" href="/w/index.php?title=%EB%A6%AC%EC%96%B8_%EC%8A%90%EB%A0%88%EC%A7%84%EC%A0%80&amp;action=edit&amp;redlink=1" title="리언 슐레진저 (없는 문서)">리언 슐레진저</a>가 제작을 맡은 <a class="mw-redirect" href="/wiki/%EB%A3%A8%EB%8B%88%ED%88%B0" title="루니툰">루니툰</a>과 <a class="new" href="/w/index.php?title=%EB%A9%94%EB%A6%AC_%EB%A9%9C%EB%A1%9C%EB%94%94&amp;action=edit&amp;redlink=1" title="메리 멜로디 (없는 문서)">메리 멜로디</a> 시리즈가 인기를 얻게 된다. 이 만화영화에는 <a href="/wiki/%EB%B2%85%EC%8A%A4_%EB%B2%84%EB%8B%88" title="벅스 버니">벅스 버니</a>, <a href="/wiki/%EB%8C%80%ED%94%BC_%EB%8D%95" title="대피 덕">대피 덕</a> 등의 캐릭터가 등장한다. <a href="/wiki/1930%EB%85%84%EB%8C%80" title="1930년대">1930년대</a>부터 시작한 루니툰 시리즈는 컬러를 사용한다. 루니툰은 다른 만화영화처럼 <a href="/wiki/1940%EB%85%84%EB%8C%80" title="1940년대">1940년대</a>에는 <a class="mw-redirect" href="/wiki/%EC%A0%9C2%EC%B0%A8_%EC%84%B8%EA%B3%84%EB%8C%80%EC%A0%84" title="제2차 세계대전">제2차 세계대전</a>의 전쟁 선전 만화영화도 제작하였다. 루니툰의 감독으로는 <a class="new" href="/w/index.php?title=%ED%85%8D%EC%8A%A4_%EC%96%B4%EB%B2%A0%EB%A6%AC&amp;action=edit&amp;redlink=1" title="텍스 어베리 (없는 문서)">텍스 어베리</a>, <a class="new" href="/w/index.php?title=%EB%A3%A8%EB%8F%8C%ED%94%84_%EC%95%84%EC%9D%B4%EC%8A%AC%EB%A7%81&amp;action=edit&amp;redlink=1" title="루돌프 아이슬링 (없는 문서)">루돌프 아이슬링</a>, <a class="new" href="/w/index.php?title=%EC%B2%99_%EC%A1%B4%EC%8A%A4&amp;action=edit&amp;redlink=1" title="척 존스 (없는 문서)">척 존스</a> 등이 있다. 그 후 <a class="mw-redirect" href="/wiki/%EB%A9%94%ED%8A%B8%EB%A1%9C-%EA%B3%A8%EB%93%9C%EC%9C%88-%EB%A9%94%EC%9D%B4%EC%96%B4" title="메트로-골드윈-메이어">메트로-골드윈-메이어</a>의 영화 판권을 사면서 MGM의 만화영화도 사들였는데 그중에서는 <a href="/wiki/%ED%86%B0%EA%B3%BC_%EC%A0%9C%EB%A6%AC" title="톰과 제리">톰과 제리</a> 시리즈도 있었다.</p>

<h3><span class="mw-headline" id=".ED.83.80.EC.9E.84_.EC.9B.8C.EB.84.88_.EC.8B.9C.EB.8C.80_.26_.EC.9B.8C.EB.84.88_.EB.B8.8C.EB.9D.BC.EB.8D.94.EC.8A.A4_.EC.BD.94.EB.A6.AC.EC.95.84_.EC.B0.BD.EB.A6.BD">타임 워너 시대 &amp; 워너 브라더스 코리아 창립</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%9B%8C%EB%84%88_%EB%B8%8C%EB%9D%BC%EB%8D%94%EC%8A%A4&amp;action=edit&amp;section=10" title="부분 편집: 타임 워너 시대 &amp; 워너 브라더스 코리아 창립">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p><a href="/wiki/1989%EB%85%84" title="1989년">1989년</a>, 워너 커뮤니케이션스는 신문, 잡지를 발간하는 미국 최대의 언론사인 <a href="/wiki/%ED%83%80%EC%9E%84_(%EC%9E%A1%EC%A7%80)" title="타임 (잡지)">타임 (잡지)</a>사와 합병하게 되었고 <a href="/wiki/%EB%89%B4%EC%8A%A4_%EC%BD%94%ED%8D%BC%EB%A0%88%EC%9D%B4%EC%85%98" title="뉴스 코퍼레이션">뉴스 코퍼레이션</a>에 이은 미국 2번째 미디어그룹인 <a href="/wiki/%ED%83%80%EC%9E%84_%EC%9B%8C%EB%84%88" title="타임 워너">타임 워너</a>를 출범시킨다.</p>
<p>그리고, <a href="/wiki/1990%EB%85%84" title="1990년">1990년</a>, 한국의 3번째 직배회사인 <strong class="selflink">워너 브러더스 코리아</strong>를 설립하게되는데 초기에는 <a href="/wiki/%EC%9B%94%ED%8A%B8_%EB%94%94%EC%A6%88%EB%8B%88_%EC%BB%B4%ED%8D%BC%EB%8B%88" title="월트 디즈니 컴퍼니">디즈니</a>의 제작 작품까지도 함께 배급했으나 <a href="/wiki/1993%EB%85%84" title="1993년">1993년</a>, 디즈니가 독립적인 직배회사를 차림으로서 워너 브러더스의 작품을 단독으로 수입/배급하는 체제로 회사를 이끌어가게된다.</p>
<p><a href="/wiki/1991%EB%85%84" title="1991년">1991년</a> ~ <a href="/wiki/1998%EB%85%84" title="1998년">1998년</a>까지 <a href="/wiki/%EB%A6%AC%EC%A0%A0%EC%8B%9C_%EC%97%94%ED%84%B0%ED%94%84%EB%9D%BC%EC%9D%B4%EC%A6%88" title="리젠시 엔터프라이즈">리젠시 엔터프라이즈</a>의 제작 영화들을 배급했다.</p>
<p><a href="/wiki/1992%EB%85%84" title="1992년">1992년</a>, 가족 영화 중심의 자회사인 <a class="new" href="/w/index.php?title=%EC%9B%8C%EB%84%88_%EB%B8%8C%EB%9D%BC%EB%8D%94%EC%8A%A4_%ED%8C%A8%EB%B0%80%EB%A6%AC_%EC%97%94%ED%84%B0%ED%85%8C%EC%9D%B8%EB%A8%BC%ED%8A%B8&amp;action=edit&amp;redlink=1" title="워너 브라더스 패밀리 엔터테인먼트 (없는 문서)">워너 브라더스 패밀리 엔터테인먼트</a>를 출범시켰고, 1년후인 <a href="/wiki/1993%EB%85%84" title="1993년">1993년</a>, <a href="/wiki/%EC%9B%8C%EB%84%88_%EB%B8%8C%EB%9D%BC%EB%8D%94%EC%8A%A4_%EC%95%A0%EB%8B%88%EB%A9%94%EC%9D%B4%EC%85%98" title="워너 브라더스 애니메이션">워너 브라더스 애니메이션</a>의 영화 사업부문인 <a href="/wiki/%EC%9B%8C%EB%84%88_%EB%B8%8C%EB%9D%BC%EB%8D%94%EC%8A%A4_%EC%95%A0%EB%8B%88%EB%A9%94%EC%9D%B4%EC%85%98" title="워너 브라더스 애니메이션">워너 브라더스 피처 애니메이션</a>이 출범했다. (그러나 재정상의 어려움으로 2003년 해체되었다가 2014년 워너 브라더스 애니메이션 그룹이 출범하면서 제2의 창업을 선포한다.)</p>
<p><a href="/wiki/1996%EB%85%84" title="1996년">1996년</a>, <a href="/wiki/%EB%89%B4_%EB%9D%BC%EC%9D%B8_%EC%8B%9C%EB%84%A4%EB%A7%88" title="뉴 라인 시네마">뉴 라인 시네마</a>,<a href="/wiki/%EC%BA%90%EC%8A%AC_%EB%A1%9D_%EC%97%94%ED%84%B0%ED%85%8C%EC%9D%B8%EB%A8%BC%ED%8A%B8" title="캐슬 록 엔터테인먼트">캐슬 록 엔터테인먼트</a> 2개의 영화사와 지상파 채널인 <a class="new" href="/w/index.php?title=TBS_(%EB%AF%B8%EA%B5%AD%EC%9D%98_TV_%EC%B1%84%EB%84%90)&amp;action=edit&amp;redlink=1" title="TBS (미국의 TV 채널) (없는 문서)">WTBS</a>, 영화 전문채널 <a href="/wiki/HBO" title="HBO">HBO</a>, 뉴스 전문채널 <a href="/wiki/CNN" title="CNN">CNN</a>, <a href="/wiki/HLN" title="HLN">HLN</a>, 만화 전문채널 <a href="/wiki/%EC%B9%B4%ED%88%B0_%EB%84%A4%ED%8A%B8%EC%9B%8C%ED%81%AC" title="카툰 네트워크">카툰 네트워크</a> 등을 운영하는 케이블 송출업체 <a href="/wiki/%ED%84%B0%EB%84%88_%EB%B8%8C%EB%A1%9C%EB%93%9C%EC%BA%90%EC%8A%A4%ED%8C%85_%EC%8B%9C%EC%8A%A4%ED%85%9C" title="터너 브로드캐스팅 시스템">터너 브로드캐스팅 시스템</a>을 인수한다.</p>
<p><a href="/wiki/1997%EB%85%84" title="1997년">1997년</a>, 오스트레일리아의 엔터테인먼트 기업 <a href="/wiki/%EB%B9%8C%EB%A6%AC%EC%A7%80_%EB%A1%9C%EB%93%9C%EC%87%BC_%ED%94%BD%EC%B2%98%EC%8A%A4" title="빌리지 로드쇼 픽처스">빌리지 로드쇼 픽처스 그룹</a>와의 파트너쉽을 체결했다.</p>
<p><a href="/wiki/1999%EB%85%84" title="1999년">1999년</a> ~ <a href="/wiki/2007%EB%85%84" title="2007년">2007년</a>까지 <a href="/wiki/%EC%BA%90%EC%8A%AC_%EB%A1%9D_%EC%97%94%ED%84%B0%ED%85%8C%EC%9D%B8%EB%A8%BC%ED%8A%B8" title="캐슬 록 엔터테인먼트">캐슬 록 엔터테인먼트</a>의 제작 영화들을 배급했다.</p>
<p><a href="/wiki/2000%EB%85%84" title="2000년">2000년</a>, <a href="/wiki/%ED%83%80%EC%9E%84_%EC%9B%8C%EB%84%88" title="타임 워너">타임 워너</a>가 <a href="/wiki/%EB%AF%B8%EA%B5%AD" title="미국">미국</a>의 인터넷 업체 <a href="/wiki/AOL" title="AOL">AOL</a>을 인수하면서 <a href="/wiki/%ED%83%80%EC%9E%84_%EC%9B%8C%EB%84%88" title="타임 워너">AOL : 타임 워너 그룹</a>을 형성하였으나 <a href="/wiki/2004%EB%85%84" title="2004년">2004년</a>, 사상 최악의 인수 합병이라는 실패 속에서 매출이 급속도로 하락해 일부 사업군을 철수하였다. 그리고,<a href="/wiki/2008%EB%85%84" title="2008년">2008년</a>에 불어닥친 미국발 금융위기에 따른 경기침체 여파로 해가 갈수록 영업 실적이 저조해지면서 <a href="/wiki/2009%EB%85%84" title="2009년">2009년</a> <a href="/wiki/12%EC%9B%94" title="12월">12월</a>, 9년여 만에 독립기업으로 분할되었다.</p>
<p><a href="/wiki/2005%EB%85%84" title="2005년">2005년</a> ~ <a href="/wiki/2013%EB%85%84" title="2013년">2013년</a>까지 <a class="mw-redirect" href="/wiki/%EB%A0%88%EC%A0%84%EB%8D%94%EB%A6%AC_%ED%94%BD%EC%B2%98%EC%8A%A4" title="레전더리 픽처스">레전더리 픽처스</a>와의 파트너쉽을 체결했다.</p>
<p><a href="/wiki/2008%EB%85%84" title="2008년">2008년</a>, <a href="/wiki/%EB%89%B4_%EB%9D%BC%EC%9D%B8_%EC%8B%9C%EB%84%A4%EB%A7%88" title="뉴 라인 시네마">뉴 라인 시네마</a>를 인수하면서 자회사로 합류시켰으며, 2년후, 배급권을 획득하는데 성공했다.</p>
<p><a href="/wiki/2014%EB%85%84" title="2014년">2014년</a>, 헐리우드의 유명한 제작자 겸 감독 <a href="/wiki/%EB%B8%8C%EB%A0%9B_%EB%9E%98%ED%8A%B8%EB%84%88" title="브렛 래트너">브렛 래트너</a>와 미국 최고의 투자 전문가 <a class="new" href="/w/index.php?title=%EC%A0%9C%EC%9E%84%EC%8A%A4_%ED%8C%8C%EC%BB%A4&amp;action=edit&amp;redlink=1" title="제임스 파커 (없는 문서)">제임스 파커</a>가 운영하는 <a href="/wiki/%EB%9E%AB%ED%8C%A9-%EB%93%84_%EC%97%94%ED%84%B0%ED%85%8C%EC%9D%B8%EB%A8%BC%ED%8A%B8" title="랫팩-듄 엔터테인먼트">랫팩 엔터테인먼트</a>와의 전략적 파트너쉽을 채결했다.</p>
<p><a href="/wiki/2015%EB%85%84" title="2015년">2015년</a>, <a href="/wiki/20%EC%84%B8%EA%B8%B0_%ED%8F%AD%EC%8A%A4" title="20세기 폭스">20세기 폭스코리아</a>에 이어 2번째로 한국영화 제작사업에 진출을 선언했다. 1년 뒤, 한국 영화 제작 진출 첫 작품인 <a href="/wiki/%EB%B0%80%EC%A0%95_(%EC%98%81%ED%99%94)" title="밀정 (영화)">밀정</a>을 개봉시킨다.</p>
