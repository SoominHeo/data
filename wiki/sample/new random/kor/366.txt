<p><b>루트비히 2세</b>(Ludwig II) 또는 <b>루트비히 독일왕</b>(Ludwig the German, Ludwig der Deutsche, <a href="/wiki/802%EB%85%84" title="802년">802년</a><sup class="reference" id="cite_ref-1"><a href="#cite_note-1">[1]</a></sup> – <a href="/wiki/876%EB%85%84" title="876년">876년</a> <a href="/wiki/8%EC%9B%94_28%EC%9D%BC" title="8월 28일">8월 28일</a>)은 <a class="mw-redirect" href="/wiki/%ED%94%84%EB%9E%91%ED%81%AC%EC%9D%98_%EB%A3%A8%ED%8A%B8%EB%B9%84%ED%9E%88_1%EC%84%B8" title="프랑크의 루트비히 1세">루트비히 자비왕</a>의 셋째 아들이다. <a href="/wiki/817%EB%85%84" title="817년">817년</a>부터 <a class="new" href="/w/index.php?title=%EB%B0%94%EC%9D%B4%EC%97%90%EB%A5%B8%EC%9D%98_%EA%B5%B0%EC%A3%BC&amp;action=edit&amp;redlink=1" title="바이에른의 군주 (없는 문서)">바이에른의 왕</a>(재위 817∼840)이었고 <a href="/wiki/843%EB%85%84" title="843년">843년</a> <a href="/wiki/%EB%B2%A0%EB%A5%B4%EB%90%AD_%EC%A1%B0%EC%95%BD" title="베르됭 조약">베르됭 조약</a> 이후부터는 <a href="/wiki/%EB%8F%85%EC%9D%BC%EC%9D%98_%EA%B5%B0%EC%A3%BC" title="독일의 군주">동프랑크 왕국의 국왕</a>(재위 843∼876)이었다. <a href="/wiki/%EB%8F%85%EC%9D%BC" title="독일">독일</a>과 <a href="/wiki/%EC%98%A4%EC%8A%A4%ED%8A%B8%EB%A6%AC%EC%95%84" title="오스트리아">오스트리아</a>에서는 그를 초대 군주로 본다.</p>
<p><a href="/wiki/814%EB%85%84" title="814년">814년</a> <a class="mw-redirect" href="/wiki/%EB%B0%94%EC%9D%B4%EC%97%90%EB%A5%B8" title="바이에른">바이에른</a> <a href="/wiki/%EA%B3%B5%EC%9E%91_(%EC%9E%91%EC%9C%84)" title="공작 (작위)">공작</a>이 되고 같은 해 부왕에 의해 바이에른을 상속받기로 결정되었으며, <a href="/wiki/817%EB%85%84" title="817년">817년</a> 제국계획령으로 <a class="new" href="/w/index.php?title=%EB%B0%94%EC%9D%B4%EC%97%90%EB%A5%B8%EC%9D%98_%EA%B5%B0%EC%A3%BC&amp;action=edit&amp;redlink=1" title="바이에른의 군주 (없는 문서)">바이에른 왕</a>에 봉해졌으며, 825년에 바이에른 통치권을 위탁받아 이듬해부터 직접 통치하기 시작했다. 그러나 <a class="mw-redirect" href="/wiki/%EC%84%9C%ED%94%84%EB%9E%91%ED%81%AC%EC%9D%98_%EC%B9%B4%EB%A5%BC_2%EC%84%B8" title="서프랑크의 카를 2세">카를</a>이 태어나면서 상속령이 바뀌자 830년부터 833년 형 <a class="mw-redirect" href="/wiki/%EC%95%84%ED%82%A4%ED%85%90%EC%9D%98_%ED%94%BC%ED%95%80_1%EC%84%B8" title="아키텐의 피핀 1세">피핀</a>, <a href="/wiki/%EB%A1%9C%ED%83%80%EB%A5%B4_1%EC%84%B8" title="로타르 1세">로타르 1세</a> 등과 함께 아버지 <a class="mw-redirect" href="/wiki/%ED%94%84%EB%9E%91%ED%81%AC%EC%9D%98_%EB%A3%A8%ED%8A%B8%EB%B9%84%ED%9E%88_1%EC%84%B8" title="프랑크의 루트비히 1세">루트비히 자비왕</a>에 대항하는 반란에 가담했다. 840년 부왕이 죽은 후 맏형 <a href="/wiki/%EB%A1%9C%ED%83%80%EB%A5%B4_1%EC%84%B8" title="로타르 1세">로타르 1세</a>가 왕국 전체를 차지함으로써 이복 동생 카를과 손잡고 <a href="/wiki/%EB%A1%9C%ED%83%80%EB%A5%B4_1%EC%84%B8" title="로타르 1세">로타르 1세</a>에 대항하였으며, <a href="/wiki/841%EB%85%84" title="841년">841년</a> <a class="new" href="/w/index.php?title=%ED%90%81%ED%8A%B8%EB%8A%90%EC%99%80_%EC%A0%84%ED%88%AC&amp;action=edit&amp;redlink=1" title="퐁트느와 전투 (없는 문서)">퐁트느와 전투</a>에서 로타르를 깨뜨렸다. <a href="/wiki/843%EB%85%84" title="843년">843년</a> <a href="/wiki/8%EC%9B%94" title="8월">8월</a> <a href="/wiki/%EB%B2%A0%EB%A5%B4%EB%90%AD_%EC%A1%B0%EC%95%BD" title="베르됭 조약">베르됭 조약</a>에 의해 라인 동쪽의 <a href="/wiki/%EB%8F%99%ED%94%84%EB%9E%91%ED%81%AC_%EC%99%95%EA%B5%AD" title="동프랑크 왕국">동프랑크 왕국</a>을 얻었다. <a href="/wiki/853%EB%85%84" title="853년">853년</a> <a href="/wiki/%EC%95%84%ED%82%A4%ED%85%90" title="아키텐">아키텐</a>의 귀족들의 원조 요청으로 아들 <a class="mw-redirect" href="/wiki/%ED%94%84%EB%9E%91%ED%81%AC%EC%9D%98_%EB%A3%A8%ED%8A%B8%EB%B9%84%ED%9E%88_3%EC%84%B8" title="프랑크의 루트비히 3세">청년 루트비히</a>를 보내 <a href="/wiki/%ED%94%BC%ED%95%80_2%EC%84%B8_(%EC%95%84%ED%82%A4%ED%85%90)" title="피핀 2세 (아키텐)">아키텐의 피핀 2세</a>를 석방시켰으며, <a href="/wiki/858%EB%85%84" title="858년">858년</a> <a href="/wiki/%EC%B9%B4%EB%A5%BC_2%EC%84%B8_(%EC%84%9C%ED%94%84%EB%9E%91%ED%81%AC)" title="카를 2세 (서프랑크)">대머리 카를</a>을 폐위시키려고 직접 군사를 이끌고 서프랑크로 쳐들어갔으나 실패하였다.</p>
<p><a href="/wiki/861%EB%85%84" title="861년">861년</a>부터 그는 영토 분할을 계획했으나, <a href="/wiki/873%EB%85%84" title="873년">873년</a> 무렵까지 상속령에 반발을 품은 아들들의 반란에 봉착해야 했다. <a href="/wiki/870%EB%85%84" title="870년">870년</a>의 <a href="/wiki/%EB%A9%94%EB%A5%B4%EC%84%BC_%EC%A1%B0%EC%95%BD" title="메르센 조약">메르센 조약</a>에서 <a href="/wiki/%EC%84%9C%ED%94%84%EB%9E%91%ED%81%AC_%EC%99%95%EA%B5%AD" title="서프랑크 왕국">서프랑크 왕국</a>의 왕 카를과 <a class="mw-redirect" href="/wiki/%EB%A1%9C%ED%8A%B8%EB%A7%81%EA%B2%90" title="로트링겐">로트링겐</a>을 분할, 현재의 독일, 오스트리아의 영역을 확정했다. <a href="/wiki/870%EB%85%84" title="870년">870년</a> <a class="mw-redirect" href="/wiki/%EB%AA%A8%EB%9D%BC%EB%B9%84%EC%95%84" title="모라비아">모라비아</a>를 정벌했지만 1년 만에 독립한다. <a href="/wiki/875%EB%85%84" title="875년">875년</a> 조카이자 사위였던 <a class="mw-redirect" href="/wiki/%EC%8B%A0%EC%84%B1%EB%A1%9C%EB%A7%88%EC%A0%9C%EA%B5%AD" title="신성로마제국">신성로마제국</a>의 <a href="/wiki/%EB%A3%A8%ED%8A%B8%EB%B9%84%ED%9E%88_2%EC%84%B8_(%EC%9D%B4%ED%83%88%EB%A6%AC%EC%95%84)" title="루트비히 2세 (이탈리아)">황제 루트비히 2세</a>는 독일인 루트비히의 장남 <a href="/wiki/%EC%B9%B4%EB%A5%BC%EB%A7%8C" title="카를만">카를만</a>을 후계자로 지명하고 사망했으나, 이복동생인 <a class="mw-disambig" href="/wiki/%EC%B9%B4%EB%A5%BC_2%EC%84%B8" title="카를 2세">카를 2세</a>가 <a class="mw-disambig" href="/wiki/%EC%95%8C%ED%94%84%EC%8A%A4" title="알프스">알프스</a>를 넘어 <a href="/wiki/%EC%9D%B4%ED%83%88%EB%A6%AC%EC%95%84" title="이탈리아">이탈리아</a>의 왕과 <a class="mw-redirect" href="/wiki/%EC%8B%A0%EC%84%B1%EB%A1%9C%EB%A7%88%EC%A0%9C%EA%B5%AD" title="신성로마제국">신성로마제국</a>의 제관을 차지하자, 그를 치려고 군사를 일으키던 중 <a href="/wiki/876%EB%85%84" title="876년">876년</a> 사망했다.</p>
<p><a href="/wiki/843%EB%85%84" title="843년">843년</a>부터 <a href="/wiki/876%EB%85%84" title="876년">876년</a>까지 루트비히는 북쪽의 <a class="mw-redirect" href="/wiki/%EB%85%B8%EB%A5%B4%EB%A7%8C_%EC%A1%B1" title="노르만 족">노르만 족</a>의 침입과 <a href="/wiki/%EC%B2%B4%EC%BD%94" title="체코">체코</a> 지역의 메렌 족들의 침입을 막아냈다.<sup class="reference" id="cite_ref-ppkk44_2-0"><a href="#cite_note-ppkk44-2">[2]</a></sup> 또한 <a class="mw-redirect" href="/wiki/%EB%AA%A8%EB%9D%BC%EB%B9%84%EC%95%84" title="모라비아">모라비아</a> 지방에 가톨릭을 전파하는 프랑크인 선교사들을 파견했지만 모라비아의 복속에는 실패한다. <a class="mw-redirect" href="/wiki/%EB%B0%94%EC%9D%B4%EC%97%90%EB%A5%B8" title="바이에른">바이에른</a>공 구엘프의 딸 엠마와 결혼하여 <a class="mw-redirect" href="/wiki/%EB%B0%94%EC%9D%B4%EC%97%90%EB%A5%B8%EC%9D%98_%EC%B9%B4%EB%A5%BC%EB%A7%8C" title="바이에른의 카를만">카를만</a>, <a class="mw-redirect" href="/wiki/%ED%94%84%EB%9E%91%ED%81%AC%EC%9D%98_%EB%A3%A8%ED%8A%B8%EB%B9%84%ED%9E%88_3%EC%84%B8" title="프랑크의 루트비히 3세">소 루트비히</a>, <a href="/wiki/%EC%B9%B4%EB%A5%BC_3%EC%84%B8" title="카를 3세">카를 3세</a>와 딸 이르멘가르트, 기셀라 등을 두었다. 독일인왕(der Deutsche)이라 부른다.</p>

<h2><span class="mw-headline" id=".EC.83.9D.EC.95.A0">생애</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%A3%A8%ED%8A%B8%EB%B9%84%ED%9E%88_2%EC%84%B8_(%EB%8F%99%ED%94%84%EB%9E%91%ED%81%AC)&amp;action=edit&amp;section=1" title="부분 편집: 생애">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<h3><span class="mw-headline" id=".EC.B4.88.EA.B8.B0_.EC.82.B6">초기 삶</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%A3%A8%ED%8A%B8%EB%B9%84%ED%9E%88_2%EC%84%B8_(%EB%8F%99%ED%94%84%EB%9E%91%ED%81%AC)&amp;action=edit&amp;section=2" title="부분 편집: 초기 삶">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<h4><span class="mw-headline" id=".EC.83.9D.EC.95.A0_.EC.B4.88.EB.B0.98">생애 초반</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%A3%A8%ED%8A%B8%EB%B9%84%ED%9E%88_2%EC%84%B8_(%EB%8F%99%ED%94%84%EB%9E%91%ED%81%AC)&amp;action=edit&amp;section=3" title="부분 편집: 생애 초반">편집</a><span class="mw-editsection-bracket">]</span></span></h4>
<p><a href="/wiki/%EC%B9%B4%EB%A1%A4%EB%A7%81%EA%B1%B0_%EC%99%95%EC%A1%B0" title="카롤링거 왕조">카롤링거 왕조</a> 출신 군주로, <a href="/wiki/%EB%A3%A8%ED%8A%B8%EB%B9%84%ED%9E%88_1%EC%84%B8_(%ED%94%84%EB%9E%91%ED%81%AC)" title="루트비히 1세 (프랑크)">경건왕 루트비히 1세</a>와 이르멘가르트 황후의 셋째 아들로 <a href="/wiki/%EC%95%84%ED%82%A4%ED%85%90" title="아키텐">아키텐</a>에서 태어났으며, <a href="/wiki/%EB%A1%9C%ED%83%80%EB%A5%B4_1%EC%84%B8" title="로타르 1세">로타르 1세</a>, <a class="mw-redirect" href="/wiki/%EC%95%84%ED%82%A4%ED%85%90%EC%9D%98_%ED%94%BC%ED%95%80_1%EC%84%B8" title="아키텐의 피핀 1세">피핀</a>, 아델라이드의 동생이다. 할아버지 <a class="mw-redirect" href="/wiki/%EC%83%A4%EB%A5%BC%EB%A7%88%EB%89%B4" title="샤를마뉴">샤를마뉴</a>의 생존 시, <a href="/wiki/%EB%B0%B1%EC%9E%91" title="백작">백작</a>에 임명되었고 할아버지 <a class="mw-redirect" href="/wiki/%EC%83%A4%EB%A5%BC%EB%A7%88%EB%89%B4" title="샤를마뉴">샤를마뉴</a>의 특별한 총애를 얻었다. <a href="/wiki/811%EB%85%84" title="811년">811년</a>에는 할아버지 <a class="mw-redirect" href="/wiki/%EC%83%A4%EB%A5%BC%EB%A7%88%EB%89%B4" title="샤를마뉴">샤를마뉴</a>로부터 숙부 <a class="mw-redirect" href="/wiki/%ED%94%84%EB%9E%91%ED%81%AC%EC%9D%98_%EC%83%A4%EB%A5%BC" title="프랑크의 샤를">동프랑크의 카를 2세</a>의 영지 중 하나인 <a class="mw-disambig" href="/wiki/%EB%A9%94%EC%9D%B8" title="메인">메인</a> <a href="/wiki/%EA%B3%B5%EC%9E%91_(%EC%9E%91%EC%9C%84)" title="공작 (작위)">공작</a>직을 받았다.</p>
<p>그는 <a href="/wiki/%EB%8F%85%EC%9D%BC" title="독일">독일</a>과 <a href="/wiki/%EC%98%A4%EC%8A%A4%ED%8A%B8%EB%A6%AC%EC%95%84" title="오스트리아">오스트리아</a> 지역에 해당되는 영지의 분국왕이 되었는데, 아버지인 경건왕 루트비히와 구별하기 위해 독일인 루트비히(Ludwig der Deutsche)라 붙였다.</p>
<p><a href="/wiki/814%EB%85%84" title="814년">814년</a> 아버지 경건왕 루트비히 1세 즉위 후 <a class="mw-redirect" href="/wiki/%EB%B0%94%EC%9D%B4%EC%97%90%EB%A5%B8" title="바이에른">바이에른</a> <a href="/wiki/%EA%B3%B5%EC%9E%91_(%EC%9E%91%EC%9C%84)" title="공작 (작위)">공작</a>에 봉해졌다. <a href="/wiki/817%EB%85%84" title="817년">817년</a> 제국 분할 때 바이에른을 상속받기로 결정되었으며, <a class="mw-redirect" href="/wiki/%EB%B0%94%EC%9D%B4%EC%97%90%EB%A5%B8" title="바이에른">바이에른</a>의 왕에 봉해졌다. 한편 그의 <a class="mw-disambig" href="/wiki/%EB%A9%94%EC%9D%B8" title="메인">메인</a> 영지는 형 <a href="/wiki/%EB%A1%9C%ED%83%80%EB%A5%B4_1%EC%84%B8" title="로타르 1세">로타르 1세</a>에게 부여되었다. <a href="/wiki/817%EB%85%84" title="817년">817년</a> <a href="/wiki/7%EC%9B%94" title="7월">7월</a>에 아버지 <a class="mw-redirect" href="/wiki/%EA%B2%BD%EA%B1%B4%EC%99%95_%EB%A3%A8%ED%8A%B8%EB%B9%84%ED%9E%88" title="경건왕 루트비히">경건왕 루트비히</a>는 <a href="/wiki/%EC%95%84%ED%97%A8" title="아헨">아헨</a>의 제국 의회에서 그는 <a class="mw-redirect" href="/wiki/%EC%95%84%ED%82%A4%ED%85%90%EC%9D%98_%ED%94%BC%ED%95%80_1%EC%84%B8" title="아키텐의 피핀 1세">피핀</a>을 <a href="/wiki/%EC%95%84%ED%82%A4%ED%85%90" title="아키텐">아키텐</a> 왕으로 봉하고, <a class="mw-redirect" href="/wiki/%EB%B0%94%EC%9D%B4%EC%97%90%EB%A5%B8" title="바이에른">바이에른</a>을 루트비히 2세에게 주도록 정하고 <a href="/wiki/%EB%A1%9C%ED%83%80%EB%A5%B4_1%EC%84%B8" title="로타르 1세">로타르 1세</a>를 공동황제 겸 후계자로 선포했다.</p>

<p><a href="/wiki/825%EB%85%84" title="825년">825년</a>에는 직접 <a class="mw-redirect" href="/wiki/%EB%B0%94%EC%9D%B4%EC%97%90%EB%A5%B8" title="바이에른">바이에른</a>의 통치권을 위탁받아 826년부터 직접 통치하기 시작했다. <a href="/wiki/823%EB%85%84" title="823년">823년</a> <a class="mw-disambig" href="/wiki/%EB%A3%A8%ED%8A%B8%EB%B9%84%ED%9E%88_1%EC%84%B8" title="루트비히 1세">루트비히 1세</a>의 계비 유디트에게서 아들 <a class="mw-redirect" href="/wiki/%EB%8C%80%EB%A8%B8%EB%A6%AC%EC%99%95_%EC%B9%B4%EB%A5%BC" title="대머리왕 카를">대머리왕 카를</a>이 태어나고 829년 상속령이 개정되자 <a href="/wiki/830%EB%85%84" title="830년">830년</a>, <a href="/wiki/831%EB%85%84" title="831년">831년</a>, <a href="/wiki/833%EB%85%84" title="833년">833년</a> 형 <a class="mw-disambig" href="/wiki/%ED%94%BC%ED%95%80" title="피핀">피핀</a>과 함께 아버지 <a class="mw-disambig" href="/wiki/%EB%A3%A8%ED%8A%B8%EB%B9%84%ED%9E%88_1%EC%84%B8" title="루트비히 1세">루트비히 1세</a>에 대항하는 반란을 일으켰다. <a href="/wiki/833%EB%85%84" title="833년">833년</a>에는 한때 아버지 루트비히를 폐위시키고 계모이자 처형인 유디트를 수녀원에 감금하였다. 그러나 부왕 루트비히는 교황의 지지를 얻어 다시 권좌로 복귀하였다.</p>
<p><a href="/wiki/827%EB%85%84" title="827년">827년</a> <a class="mw-redirect" href="/wiki/%EB%B0%94%EC%9D%B4%EC%97%90%EB%A5%B8" title="바이에른">바이에른</a> <a href="/wiki/%EA%B3%B5%EC%9E%91_(%EC%9E%91%EC%9C%84)" title="공작 (작위)">공작</a> <a href="/wiki/%EA%B5%AC%EC%97%98%ED%94%84_1%EC%84%B8" title="구엘프 1세">구엘프 1세</a>의 딸인 <a class="new" href="/w/index.php?title=%EC%95%84%EB%A5%B4%ED%86%A8%ED%94%84%EC%9D%98_%EC%97%A0%EB%A7%88&amp;action=edit&amp;redlink=1" title="아르톨프의 엠마 (없는 문서)">아르톨프의 엠마</a>(Emma of Altdorf)와 결혼하여 <a class="mw-disambig" href="/wiki/%EC%B9%B4%EB%A5%BC%EB%A1%9C%EB%A7%8C" title="카를로만">카를로만</a>, <a class="mw-redirect" href="/wiki/%ED%94%84%EB%9E%91%ED%81%AC%EC%9D%98_%EB%A3%A8%ED%8A%B8%EB%B9%84%ED%9E%88_3%EC%84%B8" title="프랑크의 루트비히 3세">루트비히 청년왕</a>, <a href="/wiki/%EC%B9%B4%EB%A5%BC_3%EC%84%B8" title="카를 3세">카를 3세</a>와 딸 힐데가르트, 기셀라, 베르타, 이르멘가르트를 두었다. 딸 이르멘가르트는 그의 조카인 <a class="mw-redirect" href="/wiki/%EC%9D%B4%ED%83%88%EB%A6%AC%EC%95%84%EC%9D%98_%EB%A3%A8%ED%8A%B8%EB%B9%84%ED%9E%88_2%EC%84%B8" title="이탈리아의 루트비히 2세">황제 루트비히</a>에게 시집갔다. 한편 계모인 <a class="mw-disambig" href="/wiki/%EB%B0%94%EC%9D%B4%EC%97%90%EB%A5%B8%EC%9D%98_%EC%9C%A0%EB%94%94%ED%8A%B8" title="바이에른의 유디트">바이에른의 유디트</a>가 자신의 아들인 <a href="/wiki/%EC%B9%B4%EB%A5%BC_2%EC%84%B8_(%EC%84%9C%ED%94%84%EB%9E%91%ED%81%AC)" title="카를 2세 (서프랑크)">대머리왕 카를</a>의 몫의 영토 획득을 위해 나서게 되자, 루트비히도 영토문제에 자신의 몫을 잃게 될까봐 우려하며 개입하기 시작하였다.</p>
<h3><span class="mw-headline" id=".EB.B0.94.EC.9D.B4.EC.97.90.EB.A5.B8.EA.B3.BC_.EB.8F.99.ED.94.84.EB.9E.91.ED.81.AC.EC.9D.98_.EA.B5.AD.EC.99.95">바이에른과 동프랑크의 국왕</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%A3%A8%ED%8A%B8%EB%B9%84%ED%9E%88_2%EC%84%B8_(%EB%8F%99%ED%94%84%EB%9E%91%ED%81%AC)&amp;action=edit&amp;section=5" title="부분 편집: 바이에른과 동프랑크의 국왕">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<h4><span class="mw-headline" id=".EC.83.81.EC.86.8D_.EB.B6.84.EC.9F.81.EC.97.90_.EC.B0.B8.EC.97.AC">상속 분쟁에 참여</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%A3%A8%ED%8A%B8%EB%B9%84%ED%9E%88_2%EC%84%B8_(%EB%8F%99%ED%94%84%EB%9E%91%ED%81%AC)&amp;action=edit&amp;section=6" title="부분 편집: 상속 분쟁에 참여">편집</a><span class="mw-editsection-bracket">]</span></span></h4>
<p><a href="/wiki/829%EB%85%84" title="829년">829년</a> 그는 형 <a class="mw-redirect" href="/wiki/%EC%95%84%ED%82%A4%ED%85%90%EC%9D%98_%ED%94%BC%ED%95%80_1%EC%84%B8" title="아키텐의 피핀 1세">피핀</a>과 함께 이복동생 카를에게 부여된 <a class="new" href="/w/index.php?title=%EC%95%8C%EB%A0%88%EB%A7%8C%EB%8B%88%EC%95%84&amp;action=edit&amp;redlink=1" title="알레만니아 (없는 문서)">알레만니아</a>를 침공하였다. <a href="/wiki/831%EB%85%84" title="831년">831년</a> 루트비히는 로타르, 피핀과 함께 등은 반란을 일으켜 한때 아버지 <a class="mw-redirect" href="/wiki/%EA%B2%BD%EA%B1%B4%EC%99%95_%EB%A3%A8%ED%8A%B8%EB%B9%84%ED%9E%88" title="경건왕 루트비히">경건왕 루트비히</a>는 양위했고 유디트는 수도원에 감금되었다.</p>
<p><a href="/wiki/832%EB%85%84" title="832년">832년</a> <a href="/wiki/%EB%A1%9C%ED%83%80%EB%A5%B4_1%EC%84%B8" title="로타르 1세">로타르 1세</a>, <a class="mw-redirect" href="/wiki/%EC%95%84%ED%82%A4%ED%85%90%EC%9D%98_%ED%94%BC%ED%95%80_1%EC%84%B8" title="아키텐의 피핀 1세">피핀</a>과 함께 <a href="/wiki/%EC%B9%B4%EB%A5%BC_2%EC%84%B8_(%EC%84%9C%ED%94%84%EB%9E%91%ED%81%AC)" title="카를 2세 (서프랑크)">대머리 카를</a>의 영지인 <a class="new" href="/w/index.php?title=%EC%95%8C%EB%A0%88%EB%A7%8C%EB%8B%88%EC%95%84&amp;action=edit&amp;redlink=1" title="알레만니아 (없는 문서)">알레만니아</a>를 다시 침공하기도 했다. 로타르, 피핀, 루트비히는 <a class="new" href="/w/index.php?title=%EA%B5%90%ED%99%A9_%EA%B7%B8%EB%A0%88%EA%B3%A0%EB%A6%AC%EC%9A%B0%EC%8A%A4_4%EC%84%B8&amp;action=edit&amp;redlink=1" title="교황 그레고리우스 4세 (없는 문서)">교황 그레고리우스 4세</a>의 지원을 얻어 거병, 반란을 일으켰고 <a href="/wiki/%EC%95%8C%EC%9E%90%EC%8A%A4" title="알자스">알자스</a>의 <a class="new" href="/w/index.php?title=%EC%A7%80%EA%B3%A8%EC%8A%A4%ED%95%98%EC%9E%84&amp;action=edit&amp;redlink=1" title="지골스하임 (없는 문서)">지골스하임</a> 부근에서 회의를 열어 다시 아버지인 그를 폐위했다.</p>
<p><a href="/wiki/833%EB%85%84" title="833년">833년</a> 루트비히는 로타르, 피핀과 함께 등은 반란을 일으켜 한때 아버지 <a class="mw-redirect" href="/wiki/%EA%B2%BD%EA%B1%B4%EC%99%95_%EB%A3%A8%ED%8A%B8%EB%B9%84%ED%9E%88" title="경건왕 루트비히">경건왕 루트비히</a>를 폐위하고 유디트는 다시 수도원에 감금되었다. <a href="/wiki/834%EB%85%84" title="834년">834년</a>봄 루트비히 2세는 형 <a href="/wiki/%EB%A1%9C%ED%83%80%EB%A5%B4_1%EC%84%B8" title="로타르 1세">로타르 1세</a>, <a class="mw-redirect" href="/wiki/%EC%95%84%ED%82%A4%ED%85%90%EC%9D%98_%ED%94%BC%ED%95%80_1%EC%84%B8" title="아키텐의 피핀 1세">아키텐의 피핀 1세</a>등과 함께 다시 반란을 기도했다. 그러나 이번에는 교황의 지지로 아버지 <a class="mw-redirect" href="/wiki/%EA%B2%BD%EA%B1%B4%EC%99%95_%EB%A3%A8%ED%8A%B8%EB%B9%84%ED%9E%88" title="경건왕 루트비히">경건왕 루트비히</a>는 기적적으로 복위했다.</p>

<p><a href="/wiki/837%EB%85%84" title="837년">837년</a> 아버지 <a class="mw-redirect" href="/wiki/%ED%94%84%EB%9E%91%ED%81%AC%EC%9D%98_%EB%A3%A8%ED%8A%B8%EB%B9%84%ED%9E%88_1%EC%84%B8" title="프랑크의 루트비히 1세">루트비히 경건왕</a>은 아헨 제국 의회, <a href="/wiki/838%EB%85%84" title="838년">838년</a> 네이메겐 제국 의회에서 더 많은 영토를 아들 <a href="/wiki/%EC%B9%B4%EB%A5%BC_2%EC%84%B8_(%EC%84%9C%ED%94%84%EB%9E%91%ED%81%AC)" title="카를 2세 (서프랑크)">대머리왕 카를</a>에게 넘겨주었다. 일단 <a href="/wiki/%EB%A1%9C%ED%83%80%EB%A5%B4_1%EC%84%B8" title="로타르 1세">로타르 1세</a>, <a class="mw-redirect" href="/wiki/%EC%95%84%ED%82%A4%ED%85%90%EC%9D%98_%ED%94%BC%ED%95%80_1%EC%84%B8" title="아키텐의 피핀 1세">아키텐의 피핀 1세</a>, 독일인 루트비히 등은 이 조치를 받아들였으나 감정이 좋지 않았다.</p>
<p><a href="/wiki/838%EB%85%84" title="838년">838년</a> <a href="/wiki/12%EC%9B%94" title="12월">12월</a> 형 <a class="mw-redirect" href="/wiki/%EC%95%84%ED%82%A4%ED%85%90%EC%9D%98_%ED%94%BC%ED%95%80_1%EC%84%B8" title="아키텐의 피핀 1세">피핀</a>이 갑자기 죽자 <a href="/wiki/%EC%B9%B4%EB%A5%BC_2%EC%84%B8_(%EC%84%9C%ED%94%84%EB%9E%91%ED%81%AC)" title="카를 2세 (서프랑크)">대머리 카를</a> 대신 조카인 <a class="mw-redirect" href="/wiki/%EC%95%84%ED%82%A4%ED%85%90%EC%9D%98_%ED%94%BC%ED%95%80_2%EC%84%B8" title="아키텐의 피핀 2세">피핀 2세</a>의 <a href="/wiki/%EC%95%84%ED%82%A4%ED%85%90" title="아키텐">아키텐</a> 상속을 지지하였다. 그 뒤 부왕과 화해, <a class="mw-redirect" href="/wiki/%EB%B0%94%EC%9D%B4%EC%97%90%EB%A5%B8" title="바이에른">바이에른</a>을 다시 통치하였다. 그러나 <a href="/wiki/839%EB%85%84" title="839년">839년</a> 봄 둘째 아들 <strong class="selflink">독일인 루트비히</strong>가 다시 <a class="mw-redirect" href="/wiki/%EA%B2%8C%EB%A5%B4%EB%A7%8C" title="게르만">게르만</a> 귀족들을 선동해 반란을 일으켰으나 패퇴하고 <a class="mw-redirect" href="/wiki/%EB%B0%94%EC%9D%B4%EC%97%90%EB%A5%B8" title="바이에른">바이에른</a>으로 물러갔다. 루트비히는 다시 형 <a href="/wiki/%EB%A1%9C%ED%83%80%EB%A5%B4_1%EC%84%B8" title="로타르 1세">로타르 1세</a>를 부추겨 다시 <a href="/wiki/%EC%B9%B4%EB%A5%BC_2%EC%84%B8_(%EC%84%9C%ED%94%84%EB%9E%91%ED%81%AC)" title="카를 2세 (서프랑크)">대머리 카를</a>의 영지인 <a class="new" href="/w/index.php?title=%EC%95%8C%EB%A0%88%EB%A7%8C%EB%8B%88%EC%95%84&amp;action=edit&amp;redlink=1" title="알레만니아 (없는 문서)">알레만니아</a>를 침공했다. 이로써 <a class="mw-redirect" href="/wiki/%ED%8C%90%EB%85%B8%EB%8B%88%EC%95%84" title="판노니아">판노니아</a> <a href="/wiki/%EB%B0%B1%EC%9E%91" title="백작">백작</a>령을 얻고, 루트비히는 이때 획득한 영토인 <a class="mw-redirect" href="/wiki/%ED%8C%90%EB%85%B8%EB%8B%88%EC%95%84" title="판노니아">판노니아</a>를 자신의 영지에 포함하고 자신의 아들인 <a class="mw-redirect" href="/wiki/%ED%94%84%EB%9E%91%ED%81%AC%EC%9D%98_%EB%A3%A8%ED%8A%B8%EB%B9%84%ED%9E%88_3%EC%84%B8" title="프랑크의 루트비히 3세">청년 루트비히</a>를 <a href="/wiki/%EB%B0%B1%EC%9E%91" title="백작">백작</a>으로 임명하였다.</p>
<p><a href="/wiki/839%EB%85%84" title="839년">839년</a> <a href="/wiki/5%EC%9B%94_30%EC%9D%BC" title="5월 30일">5월 30일</a> 보름스 의회에서 제4차 배분이 이루어져 제국은 <a href="/wiki/%EB%A1%9C%ED%83%80%EB%A5%B4_1%EC%84%B8" title="로타르 1세">로타르 1세</a>와 <a href="/wiki/%EC%B9%B4%EB%A5%BC_2%EC%84%B8_(%EC%84%9C%ED%94%84%EB%9E%91%ED%81%AC)" title="카를 2세 (서프랑크)">대머리 카를</a> 사이에 양분되었고 독일인 루트비히는 <a class="mw-redirect" href="/wiki/%EB%B0%94%EC%9D%B4%EC%97%90%EB%A5%B8" title="바이에른">바이에른</a>만을 차지했다. 당시 아버지 <a class="mw-redirect" href="/wiki/%ED%94%84%EB%9E%91%ED%81%AC%EC%9D%98_%EB%A3%A8%ED%8A%B8%EB%B9%84%ED%9E%88_1%EC%84%B8" title="프랑크의 루트비히 1세">루트비히 경건왕</a>은 <a href="/wiki/%EB%B0%94%EC%9D%B4%ED%82%B9" title="바이킹">바이킹</a> 족의 <a href="/wiki/%ED%94%84%EB%A6%AC%EC%8A%AC%EB%9E%80%ED%8A%B8" title="프리슬란트">프리슬란트</a>, <a class="new" href="/w/index.php?title=%ED%94%84%EB%A6%AC%EC%8B%9C%EC%95%84&amp;action=edit&amp;redlink=1" title="프리시아 (없는 문서)">프리시아</a> 반환 요구를 거절하고 전쟁 중이었는데, 루트비히 2세는 <a href="/wiki/%EB%B3%B4%EB%A6%84%EC%8A%A4" title="보름스">보름스</a> <a href="/wiki/%EC%9D%98%ED%9A%8C" title="의회">의회</a>의 결정에 반발, 조카 <a href="/wiki/%ED%94%BC%ED%95%80_2%EC%84%B8_(%EC%95%84%ED%82%A4%ED%85%90)" title="피핀 2세 (아키텐)">피핀 2세</a>와 함께 다시 군사를 이끌고 <a href="/wiki/%EC%95%84%ED%97%A8" title="아헨">아헨</a>으로 쳐들어왔다. 그러나 <a href="/wiki/839%EB%85%84" title="839년">839년</a> 말 <a href="/wiki/%EB%B3%B4%EB%A6%84%EC%8A%A4" title="보름스">보름스</a> 전투에서 아버지 루트비히에게 격퇴당했다.</p>

<p><a href="/wiki/840%EB%85%84" title="840년">840년</a> 아버지 <a class="mw-redirect" href="/wiki/%ED%94%84%EB%9E%91%ED%81%AC%EC%9D%98_%EB%A3%A8%ED%8A%B8%EB%B9%84%ED%9E%88_1%EC%84%B8" title="프랑크의 루트비히 1세">루트비히 경건왕</a>이 죽은 뒤 큰형 <a href="/wiki/%EB%A1%9C%ED%83%80%EB%A5%B4_1%EC%84%B8" title="로타르 1세">로타르 1세</a>가 제국 전체를 다스리는 것에 대해 이복 동생 카를과 손잡고 로타르에 대항하였다. 이때 <a href="/wiki/%EC%95%84%ED%82%A4%ED%85%90" title="아키텐">아키텐</a>의 분국왕이자 조카인 <a class="mw-redirect" href="/wiki/%EC%95%84%ED%82%A4%ED%85%90%EC%9D%98_%ED%94%BC%ED%95%80_2%EC%84%B8" title="아키텐의 피핀 2세">아키텐의 피핀 2세</a>의 지원군이 <a href="/wiki/%EB%A1%9C%ED%83%80%EB%A5%B4_1%EC%84%B8" title="로타르 1세">로타르 1세</a>에게 합류, 로타르를 도와주었다. <a href="/wiki/%EB%A1%9C%ED%83%80%EB%A5%B4_1%EC%84%B8" title="로타르 1세">로타르 1세</a>는 <a href="/wiki/817%EB%85%84" title="817년">817년</a>의 제국 칙령에서 부여된 자신의 권리를 다시 주장, 협상을 시도했으나 실패했다. <a href="/wiki/841%EB%85%84" title="841년">841년</a> <a href="/wiki/7%EC%9B%94_25%EC%9D%BC" title="7월 25일">7월 25일</a>의 <a class="new" href="/w/index.php?title=%ED%90%81%ED%8A%B8%EB%8A%90%EC%99%80_%EC%A0%84%ED%88%AC&amp;action=edit&amp;redlink=1" title="퐁트느와 전투 (없는 문서)">퐁트느와 전투</a>에서 로타르를 깨뜨렸다. <a href="/wiki/842%EB%85%84" title="842년">842년</a>에는 <a class="mw-redirect" href="/wiki/%EC%9E%91%EC%84%BC" title="작센">작센</a>에서 봉기를 일으켰고 <a class="mw-redirect" href="/wiki/%ED%8A%80%EB%A7%81%EA%B2%90" title="튀링겐">튀링겐</a>의 <a href="/wiki/%EA%B3%B5%EC%9E%91_(%EC%9E%91%EC%9C%84)" title="공작 (작위)">공작</a> 등이 이에 협조하여 로타르를 조여왔다. 패한 로타르는 결국 <a href="/wiki/%EC%95%84%ED%97%A8" title="아헨">아헨</a>으로 피신했으나 결국 협상을 하게 된다.</p>
<p><a href="/wiki/842%EB%85%84" title="842년">842년</a> <a href="/wiki/2%EC%9B%94_12%EC%9D%BC" title="2월 12일">2월 12일</a> 독일인 루트히비는 대머리 카를과 <a class="mw-redirect" href="/wiki/%EC%8A%88%ED%8A%B8%EB%9D%BC%EC%8A%A4%EB%B6%80%EB%A5%B4%ED%81%AC" title="슈트라스부르크">슈트라스부르크</a>에서 맹약을 체결한다. 이때 독일인 루트비히는 <a class="mw-redirect" href="/wiki/%EA%B2%8C%EB%A5%B4%EB%A7%8C%EC%96%B4" title="게르만어">게르만어</a>로, 대머리 카를은 <a class="new" href="/w/index.php?title=%EA%B0%88%EB%A1%9C-%EB%A1%9C%EB%A7%9D%EC%96%B4&amp;action=edit&amp;redlink=1" title="갈로-로망어 (없는 문서)">갈로-로망어</a>로 서약문을 낭독한다. 이들은 하느님과 형제애에 의거하여 로타르를 공격한다고 맹세하였다.</p>
<blockquote class="toccolours" style="float:none; padding: 10px 15px 10px 15px; display:table;">
<p>Pro Deo amur et pro christian poblo et nostro commun salvament, d'ist di in avant, in quant Deus savir et podir me dunat, si salvarai eo cist meon fradre Karlo et in aiudha et in cadhuna cosa, si cum om per dreit son fradra salvar dift, in o quid il mi altresi fazet, et ab Ludher nul plaid nunquam prindrai, qui meon vol cist meon fradre Karle in damno sit.</p>
</blockquote>
<p>이 서약은 그때까지의 <a href="/wiki/%ED%94%84%EB%9E%91%ED%81%AC_%EC%99%95%EA%B5%AD" title="프랑크 왕국">프랑크 왕국</a>의 국왕, 황제들이 명령과 선언으로 전쟁, 정책을 집행하던 것과는 다른 신과 백성들 앞에서 공개적으로 한 서약이었다. 또한 <a href="/wiki/%ED%94%84%EB%9E%91%ED%81%AC_%EC%99%95%EA%B5%AD" title="프랑크 왕국">프랑크 왕국</a>의 표준어인 고지독일어로 쓰여졌지만, <a class="mw-redirect" href="/wiki/%EA%B2%8C%EB%A5%B4%EB%A7%8C%EC%96%B4" title="게르만어">게르만어</a>와 <a href="/wiki/%EB%9D%BC%ED%8B%B4%EC%96%B4" title="라틴어">라틴어</a>로 각각 낭독되었다.</p>
<p><a href="/wiki/842%EB%85%84" title="842년">842년</a> <a href="/wiki/6%EC%9B%94" title="6월">6월</a> 루트비히는 <a href="/wiki/%EB%A1%9C%ED%83%80%EB%A5%B4_1%EC%84%B8" title="로타르 1세">로타르 1세</a>, <a href="/wiki/%EC%B9%B4%EB%A5%BC_2%EC%84%B8_(%EC%84%9C%ED%94%84%EB%9E%91%ED%81%AC)" title="카를 2세 (서프랑크)">대머리 카를</a>을 <a class="mw-redirect" href="/wiki/%EC%9E%91%EC%84%BC" title="작센">작센</a>에서 만나 협상을 시도하였으나 결렬되었다. 협상은 여러 번 결렬되었으나 <a href="/wiki/843%EB%85%84" title="843년">843년</a> <a href="/wiki/8%EC%9B%94" title="8월">8월</a> 제국을 3분하기로 하고 종결된다.</p>
<h4><span class="mw-headline" id=".EB.8F.99.ED.94.84.EB.9E.91.ED.81.AC.EC.9D.98_.EA.B5.AD.EC.99.95">동프랑크의 국왕</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%A3%A8%ED%8A%B8%EB%B9%84%ED%9E%88_2%EC%84%B8_(%EB%8F%99%ED%94%84%EB%9E%91%ED%81%AC)&amp;action=edit&amp;section=9" title="부분 편집: 동프랑크의 국왕">편집</a><span class="mw-editsection-bracket">]</span></span></h4>
<p><a href="/wiki/843%EB%85%84" title="843년">843년</a> <a href="/wiki/8%EC%9B%94" title="8월">8월</a> <a class="mw-redirect" href="/wiki/%EB%B2%A0%EB%A5%B4%EB%8D%A9_%EC%A1%B0%EC%95%BD" title="베르덩 조약">베르덩 조약</a>에 따라 카를과 로타르는 제국의 서부와 중부를 각각 차지했고, 루트비히는 동부와 함께 프랑켄, 슈바벤, 케른텐, 바이에른, 작센 지방의 영토를 차지했다.</p>
<p>동프랑크의 국왕에 올랐지만, 왕국은 바로 프랑켄, 작센, 알레만니아, 바이에른 등의 여러 부족 출신 제후들의 세력이 강했고, 루트비히는 순찰사와 변경백을 파견하여 이들을 관리 감독하였다. 또한 독일인 루트비히는 자신의 아들 중 <a href="/wiki/%EC%B9%B4%EB%A5%BC%EB%A7%8C" title="카를만">카를만</a>과 소 루트비히를 각각 바이에른과 작센으로 보내 분봉왕으로 봉하여 이 지역을 다스리게 했다. 그러나 지역 제후들의 세력이 강했고, 판노니아와 모라비아인들은 강력한 세력을 형성하고 있었다. 또한 <a class="mw-redirect" href="/wiki/%EB%85%B8%EB%A5%B4%EB%A7%8C_%EC%A1%B1" title="노르만 족">노르만 족</a>과 <a class="mw-redirect" href="/wiki/%EB%A7%88%EC%9E%90%EB%A5%B4_%EC%A1%B1" title="마자르 족">마자르 족</a>의 잇단 침입으로 국왕의 통치력은 점점 약화되었다.</p>
<p><a href="/wiki/853%EB%85%84" title="853년">853년</a> 카를에게 대항한 <a class="mw-redirect" href="/wiki/%EC%84%9C%ED%94%84%EB%9E%91%ED%81%AC" title="서프랑크">서프랑크</a>의 귀족들이 루트비히에게 원조를 청하자 아들 <a class="mw-redirect" href="/wiki/%ED%94%84%EB%9E%91%ED%81%AC%EC%9D%98_%EB%A3%A8%ED%8A%B8%EB%B9%84%ED%9E%88_3%EC%84%B8" title="프랑크의 루트비히 3세">루트비히 청년왕</a>을 <a href="/wiki/%EC%95%84%ED%82%A4%ED%85%90" title="아키텐">아키텐</a>으로 보냈으며, <a class="mw-redirect" href="/wiki/%EC%95%84%ED%82%A4%ED%85%90%EC%9D%98_%ED%94%BC%ED%95%80_2%EC%84%B8" title="아키텐의 피핀 2세">피핀 2세</a>를 지원하게 했다. 또한 <a class="mw-redirect" href="/wiki/%EC%84%9C%ED%94%84%EB%9E%91%ED%81%AC" title="서프랑크">서프랑크</a>왕 <a class="mw-redirect mw-disambig" href="/wiki/%EC%83%A4%EB%A5%BC_2%EC%84%B8" title="샤를 2세">샤를 2세</a>의 폐위를 위한 거사에는 <a class="mw-redirect" href="/wiki/%EC%95%84%ED%82%A4%ED%85%90%EC%9D%98_%ED%94%BC%ED%95%80_2%EC%84%B8" title="아키텐의 피핀 2세">피핀 2세</a>, <a class="mw-redirect" href="/wiki/%ED%94%84%EB%A1%9C%EB%B0%A9%EC%8A%A4%EC%9D%98_%EC%83%A4%EB%A5%BC" title="프로방스의 샤를">프로방스의 샤를</a>까지 참여했으나 결국 실패한다. <a href="/wiki/858%EB%85%84" title="858년">858년</a> 자신이 직접 서쪽으로 가서 카를을 폐위하려 했으나 장거리 이동으로 병력과 물자 조달에 실패하면서 중단하게 됐다. 결국 <a href="/wiki/860%EB%85%84" title="860년">860년</a> <a href="/wiki/6%EC%9B%94" title="6월">6월</a> <a class="new" href="/w/index.php?title=%EC%BD%94%EB%B8%94%EB%A0%8C%EC%B8%A0_%EC%A1%B0%EC%95%BD&amp;action=edit&amp;redlink=1" title="코블렌츠 조약 (없는 문서)">코블렌츠 조약</a>을 맺어 카를의 영토에 대한 권리주장을 취소했다.</p>
<h3><span class="mw-headline" id=".EA.B5.90.ED.9A.8C.EC.99.80_.EC.88.98.EB.8F.84.EC.9B.90_.EA.B1.B4.EB.A6.BD">교회와 수도원 건립</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%A3%A8%ED%8A%B8%EB%B9%84%ED%9E%88_2%EC%84%B8_(%EB%8F%99%ED%94%84%EB%9E%91%ED%81%AC)&amp;action=edit&amp;section=10" title="부분 편집: 교회와 수도원 건립">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>그는 동프랑크 각지에 교회와 수녀원 설립에 투자하였다. <a href="/wiki/853%EB%85%84" title="853년">853년</a> <a href="/wiki/%EC%B7%A8%EB%A6%AC%ED%9E%88" title="취리히">취리히</a>에 <a class="new" href="/w/index.php?title=%ED%8E%A0%EB%A6%AD%EC%8A%A4_%EC%9A%B4%EB%93%9C_%EB%A0%88%EA%B5%B4%EB%9D%BC_%EC%88%98%EB%85%80%EC%9B%90&amp;action=edit&amp;redlink=1" title="펠릭스 운드 레굴라 수녀원 (없는 문서)">펠릭스 운드 레굴라 수녀원</a>을 건립하였는데, 그의 딸 힐데가르트(853년~856년)와 베르타(857년~877년)가 한때 이 수녀원의 원장을 지내기도 했다. <a class="mw-redirect" href="/wiki/%ED%94%84%EB%9E%91%ED%81%AC%ED%91%B8%EB%A5%B4%ED%8A%B8" title="프랑크푸르트">프랑크푸르트</a>와 <a href="/wiki/%EB%A0%88%EA%B2%90%EC%8A%A4%EB%B6%80%EB%A5%B4%ED%81%AC" title="레겐스부르크">레겐스부르크</a>에도 루트비히 교회를 설립하였다. 또한 <a class="mw-redirect" href="/wiki/%ED%8A%80%EB%A7%81%EA%B2%90" title="튀링겐">튀링겐</a>에는 <a class="new" href="/w/index.php?title=%ED%81%B4%EB%A1%9C%EC%8A%A4%ED%84%B0_%EC%84%B1%EB%8B%B9&amp;action=edit&amp;redlink=1" title="클로스터 성당 (없는 문서)">클로스터 성당</a>을 건립하였다.</p>
<p>그는 또 <a class="new" href="/w/index.php?title=%EC%84%B1_%EA%B0%88%EB%A0%8C_%EC%88%98%EB%8F%84%EC%9B%90&amp;action=edit&amp;redlink=1" title="성 갈렌 수도원 (없는 문서)">성 갈렌 수도원</a>, <a class="new" href="/w/index.php?title=%EC%9E%98%EC%B8%A0%EB%B6%80%EB%A5%B4%ED%81%AC_%EC%84%B1%EB%8B%B9&amp;action=edit&amp;redlink=1" title="잘츠부르크 성당 (없는 문서)">잘츠부르크 성당</a>, <a class="new" href="/w/index.php?title=%ED%92%80%EB%8B%A4_%EC%84%B1%EB%8B%B9&amp;action=edit&amp;redlink=1" title="풀다 성당 (없는 문서)">풀다 성당</a> 등에 가장 많은 기부금을 희사하기도 했다.</p>

<p><a href="/wiki/855%EB%85%84" title="855년">855년</a> <a href="/wiki/%EB%A1%9C%ED%83%80%EB%A5%B4_1%EC%84%B8" title="로타르 1세">로타르 1세</a>가 죽자, 그의 영토는 아들들에게 분배되었는데 둘째 아들인 <a href="/wiki/%EB%A1%9C%ED%83%80%EB%A5%B4_2%EC%84%B8" title="로타르 2세">로타르 2세</a>가 <a class="mw-redirect" href="/wiki/%EB%A1%9C%ED%83%80%EB%A7%81%EA%B8%B0%EC%95%84" title="로타링기아">로타링기아</a>를 물려받았다. 그러나 <a href="/wiki/%EB%A1%9C%ED%83%80%EB%A5%B4_1%EC%84%B8" title="로타르 1세">로타르 1세</a>가 죽자 바로 서프랑크의 대머리 <a href="/wiki/%EC%B9%B4%EB%A5%BC_2%EC%84%B8_(%EC%84%9C%ED%94%84%EB%9E%91%ED%81%AC)" title="카를 2세 (서프랑크)">카를 2세</a>가 로타르 2세의 몫인 로타링기아를 차지하려 했다. 그러나 <a class="mw-redirect" href="/wiki/%EB%8F%99%ED%94%84%EB%9E%91%ED%81%AC" title="동프랑크">동프랑크</a>의 독일인 루트비히 2세의 개입으로 <a href="/wiki/%EC%B9%B4%EB%A5%BC_2%EC%84%B8_(%EC%84%9C%ED%94%84%EB%9E%91%ED%81%AC)" title="카를 2세 (서프랑크)">대머리 카를</a>의 로타링기아 접수 기도는 실패로 돌아간다. <a href="/wiki/%EB%A1%9C%ED%83%80%EB%A5%B4_1%EC%84%B8" title="로타르 1세">로타르 1세</a>가 죽자 카를 2세는 형의 영토였던 로트링겐 지역을 확보하기 위해 내전을 일으켰다. 그러나 독일의 왕이었던 루트비히 2세가 그를 막았다.<sup class="reference" id="cite_ref-3"><a href="#cite_note-3">[3]</a></sup> 그러나 대머리 카를은 로트링겐을 포기하지 않고, 나중에 로타르 2세가 죽은 뒤에 다시 로트링겐을 넘보게 된다. <a href="/wiki/%EB%A1%9C%ED%83%80%EB%A5%B4_2%EC%84%B8" title="로타르 2세">로타르 2세</a>에게는 적장자가 없었기 때문에 독일인 루트비히와 <a href="/wiki/%EC%B9%B4%EB%A5%BC_2%EC%84%B8_(%EC%84%9C%ED%94%84%EB%9E%91%ED%81%AC)" title="카를 2세 (서프랑크)">대머리 카를</a>은 조카 로타르가 죽은 뒤 <a class="mw-redirect" href="/wiki/%EB%A1%9C%ED%83%80%EB%A7%81%EA%B8%B0%EC%95%84" title="로타링기아">로타링기아</a>를 분할하기로 합의했다. 그러나 <a href="/wiki/%EB%A1%9C%ED%83%80%EB%A5%B4_2%EC%84%B8" title="로타르 2세">로타르 2세</a>의 아들 <a href="/wiki/%EC%9C%84%EA%B7%B8_(%EB%A1%9C%ED%8A%B8%EB%A7%81%EA%B2%90%EC%9D%98_%EC%99%95%EC%9E%90)" title="위그 (로트링겐의 왕자)">위그</a>가 나타났고 <a href="/wiki/869%EB%85%84" title="869년">869년</a> <a href="/wiki/%EB%A1%9C%ED%83%80%EB%A5%B4_2%EC%84%B8" title="로타르 2세">로타르 2세</a>가 죽었을 때, <a href="/wiki/%EC%B9%B4%EB%A5%BC_2%EC%84%B8_(%EC%84%9C%ED%94%84%EB%9E%91%ED%81%AC)" title="카를 2세 (서프랑크)">카를</a>과 함께 위그를 사생아로 정하고 <a href="/wiki/%EB%A1%9C%ED%83%80%EB%A5%B4_2%EC%84%B8" title="로타르 2세">로타르 2세</a>와 <a href="/wiki/%EB%B0%9C%ED%8A%B8%EB%9D%BC%EB%8B%A4" title="발트라다">발트라다</a>의 결혼은 무효로 선언한 뒤 <a class="mw-redirect" href="/wiki/%EB%A1%9C%ED%8A%B8%EB%A7%81%EA%B2%90" title="로트링겐">로트링겐</a>에서 추방했다.</p>
<p><a href="/wiki/861%EB%85%84" title="861년">861년</a>과 <a href="/wiki/863%EB%85%84" title="863년">863년</a>, <a href="/wiki/864%EB%85%84" title="864년">864년</a> 아들 <a class="mw-redirect" href="/wiki/%EC%B9%B4%EB%A5%BC%EB%A1%9C%EB%A7%8C_3%EC%84%B8" title="카를로만 3세">카를로만 3세</a>는 아버지에 독일왕 루트비히에 대항하여 반란을 일으켰는데 곧 다른 아들들인 <a class="mw-redirect" href="/wiki/%ED%94%84%EB%9E%91%ED%81%AC%EC%9D%98_%EB%A3%A8%ED%8A%B8%EB%B9%84%ED%9E%88_3%EC%84%B8" title="프랑크의 루트비히 3세">소 루트비히</a>와 <a href="/wiki/%EC%B9%B4%EB%A5%BC_3%EC%84%B8" title="카를 3세">뚱보 카를</a>도 아버지에게 반란을 일으켰다. 한편 독일왕 루트비히는 863년 아들의 반란을 두려워 <a class="mw-redirect" href="/wiki/%EC%BC%80%EB%A5%B8%ED%85%90" title="케른텐">케른텐</a>의 그의 영지를 침공했다. 루트비히는 비밀리에 <a class="mw-disambig" href="/wiki/%EC%B9%B4%EB%A5%BC%EB%A1%9C%EB%A7%8C" title="카를로만">카를로만</a>의 측근인 군다카르에게 사자를 보내 협상, 군다카르는 자신의 군사를 이끌고 루트비히에게 갔고 카를로만은 군다카르를 해임하고 케른텐을 직접 차지하였다.</p>
<p><a href="/wiki/863%EB%85%84" title="863년">863년</a> 아들 <a class="mw-redirect" href="/wiki/%EC%B9%B4%EB%A5%BC%EB%A1%9C%EB%A7%8C_3%EC%84%B8" title="카를로만 3세">카를로만 3세</a>를 <a class="mw-redirect" href="/wiki/%ED%8C%90%EB%85%B8%EB%8B%88%EC%95%84" title="판노니아">판노니아</a> <a href="/wiki/%EB%B3%80%EA%B2%BD%EB%B0%B1" title="변경백">변경백</a>직에서 해임되었다. 그를 판노니아 변경백에서 해임한 루트비히는 윌리엄 등의 관료를 파견하는 한편 모라비아 부족장 엔겔샬크 등을 판노니아 변경백에 임명하여 그를 견제했다. <a href="/wiki/864%EB%85%84" title="864년">864년</a> 독일인 루트비히는 아들들과의 반란에 시달리면서 세 아들에게 영토를 나누어 주었는데 카를만에게는 <a class="mw-redirect" href="/wiki/%EB%B0%94%EC%9D%B4%EC%97%90%EB%A5%B8" title="바이에른">바이에른</a>, 소 루트비히에게는 <a class="mw-redirect" href="/wiki/%EC%9E%91%EC%84%BC" title="작센">작센</a>, 뚱보 카를에게는 <a href="/wiki/%EC%8A%88%EB%B0%94%EB%B2%A4" title="슈바벤">슈바벤</a>을 주었다. 그러나 <a href="/wiki/866%EB%85%84" title="866년">866년</a> 아들 <a class="mw-redirect" href="/wiki/%EC%B9%B4%EB%A5%BC%EB%A1%9C%EB%A7%8C_3%EC%84%B8" title="카를로만 3세">카를로만 3세</a>은 영토 분할이 마음에 들지 않는다 하여 케른텐 후작 군다카르 등을 이끌고 반란을 일으켰다.</p>
<h3><span class="mw-headline" id=".EC.83.9D.EC.95.A0_.ED.9B.84.EB.B0.98">생애 후반</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%A3%A8%ED%8A%B8%EB%B9%84%ED%9E%88_2%EC%84%B8_(%EB%8F%99%ED%94%84%EB%9E%91%ED%81%AC)&amp;action=edit&amp;section=12" title="부분 편집: 생애 후반">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<h4><span class="mw-headline" id=".EA.B5.90.ED.9A.8C_.EB.AC.B8.EC.A0.9C_.EA.B0.9C.EC.9E.85">교회 문제 개입</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%A3%A8%ED%8A%B8%EB%B9%84%ED%9E%88_2%EC%84%B8_(%EB%8F%99%ED%94%84%EB%9E%91%ED%81%AC)&amp;action=edit&amp;section=13" title="부분 편집: 교회 문제 개입">편집</a><span class="mw-editsection-bracket">]</span></span></h4>
<p>독일인 루트비히 2세는 즉위 초부터 <a class="mw-redirect" href="/wiki/%EB%AA%A8%EB%9D%BC%EB%B9%84%EC%95%84" title="모라비아">모라비아</a>와 <a class="mw-redirect" href="/wiki/%ED%8C%90%EB%85%B8%EB%8B%88%EC%95%84" title="판노니아">판노니아</a>, <a class="mw-redirect" href="/wiki/%EB%A7%88%EC%9E%90%EB%A5%B4_%EC%A1%B1" title="마자르 족">마자르 족</a>, <a class="mw-redirect" href="/wiki/%EC%8A%AC%EB%9D%BC%EB%B8%8C_%EC%A1%B1" title="슬라브 족">슬라브 족</a> 등을 상대로 한 그리스도교 포교를 장려, 후원하였다. 그는 직접 모라비아, 판노니아 등에 프랑크족 사제와 선교사를 파견했고, <a class="mw-redirect" href="/wiki/%EB%B0%94%EC%9D%B4%EC%97%90%EB%A5%B8" title="바이에른">바이에른</a>과 <a class="mw-redirect" href="/wiki/%EC%BC%80%EB%A5%B8%ED%85%90" title="케른텐">케른텐</a>을 다스리던 <a href="/wiki/%EC%B9%B4%EB%A5%BC%EB%A7%8C" title="카를만">카를만</a>, <a href="/wiki/%EC%95%84%EB%A5%B4%EB%88%8C%ED%94%84_%ED%8F%B0_%EC%BC%80%EB%A5%B8%ED%85%90" title="아르눌프 폰 케른텐">아르눌프</a> 및 모라비아 공작, 판노니아 변경백 등을 통해 이들 선교자들의 신변을 보호하게 했다. 이에 모라비아 대공 로스티슬라프(Rostislav) 역시 교황에게 선교사 포교를 요청, <a class="new" href="/w/index.php?title=%EC%84%B1_%EC%8B%9C%EB%A6%B4%EB%A3%A8%EC%8A%A4&amp;action=edit&amp;redlink=1" title="성 시릴루스 (없는 문서)">성 시릴루스</a>(St. Cyrillus)와 <a class="new" href="/w/index.php?title=%EC%84%B1_%EB%A9%94%ED%86%A0%EB%94%94%EC%98%A4&amp;action=edit&amp;redlink=1" title="성 메토디오 (없는 문서)">성 메토디오</a>는 <a href="/wiki/863%EB%85%84" title="863년">863년</a>, <a href="/wiki/864%EB%85%84" title="864년">864년</a> 무렵부터 모라비아와 판노니아 지역의 그리스도교 포교를 담당하였다. 이들 지역은 <a href="/wiki/%EC%9E%98%EC%B8%A0%EB%B6%80%EB%A5%B4%ED%81%AC" title="잘츠부르크">잘츠부르크</a> 대주교 관구였고, 성 시릴루스가 병으로 로마로 돌아갔다가 사망하지만 모라비아, 판노니아인들의 개종은 계속되었다. 860년대 말부터 <a class="mw-redirect" href="/wiki/%EB%AA%A8%EB%9D%BC%EB%B9%84%EC%95%84" title="모라비아">모라비아</a>인들과 <a class="mw-redirect" href="/wiki/%ED%8C%90%EB%85%B8%EB%8B%88%EC%95%84" title="판노니아">판노니아</a>인들은 동프랑크 교회의 관리 감독이 아닌 독자적인 교회 관구 설치와 슬라브어 성서 간행을 원하였다. <a class="new" href="/w/index.php?title=%EC%84%B1_%EC%8B%9C%EB%A6%B4%EB%A3%A8%EC%8A%A4&amp;action=edit&amp;redlink=1" title="성 시릴루스 (없는 문서)">성 시릴루스</a>의 사후 모라비아, 판노니아인들의 교구 분리 운동은 계속되었고, 루트비히 2세는 이들의 정치적 분리 독립을 염두에 두고 이를 반대하였다. 그러자 교황은 다시 <a class="new" href="/w/index.php?title=%EC%84%B1_%EB%A9%94%ED%86%A0%EB%94%94%EC%98%A4&amp;action=edit&amp;redlink=1" title="성 메토디오 (없는 문서)">성 메토디오</a>를 <a class="mw-redirect" href="/wiki/%EB%AA%A8%EB%9D%BC%EB%B9%84%EC%95%84" title="모라비아">모라비아</a>에 특사로 파견하였다.</p>
<p>모라비아인들은 단독 주교구 설치를 청원했고, <a class="new" href="/w/index.php?title=%EA%B5%90%ED%99%A9_%ED%95%98%EB%93%9C%EB%A6%AC%EC%95%84%EB%88%84%EC%8A%A4_2%EC%84%B8&amp;action=edit&amp;redlink=1" title="교황 하드리아누스 2세 (없는 문서)">교황 하드리아누스 2세</a>는 모라비아와 판노니아를 동프랑크 지역 관구에서 독립시켜 대교구로 승격시켰고, <a class="new" href="/w/index.php?title=%EB%A9%94%ED%86%A0%EB%94%94%EC%98%A4&amp;action=edit&amp;redlink=1" title="메토디오 (없는 문서)">메토디오</a>를 판노니아와 모라비아 지방 전체를 관할하는 시르미움(Sirmium)의 대주교로 임명하였다. 동프랑크의 주교들은 관구 독립을 반대했고, 루트비히는 모라비아, 판노니아인들의 분리 독립을 우려하였다. <a href="/wiki/870%EB%85%84" title="870년">870년</a> 루트비히 2세와 독일 주교들은 라티스본(Ratisbon) 시노드에서 <a class="new" href="/w/index.php?title=%EB%A9%94%ED%86%A0%EB%94%94%EC%98%A4&amp;action=edit&amp;redlink=1" title="메토디오 (없는 문서)">메토디오</a>를 추방하기로 결정하고 그를 투옥시켰다가, <a href="/wiki/%EC%8A%88%EB%B0%94%EB%B2%A4" title="슈바벤">슈바벤</a>으로 유배보냈다. <a href="/wiki/873%EB%85%84" title="873년">873년</a> <a href="/wiki/%EA%B5%90%ED%99%A9_%EC%9A%94%ED%95%9C_8%EC%84%B8" title="교황 요한 8세">교황 요한 8세</a>가 <a class="new" href="/w/index.php?title=%EC%84%B1_%EB%A9%94%ED%86%A0%EB%94%94%EC%98%A4&amp;action=edit&amp;redlink=1" title="성 메토디오 (없는 문서)">성 메토디오</a>를 석방해줄 것을 계속 요청하므로 루트비히 독일인은 메토디오를 풀어주게 된다.</p>

<p><a href="/wiki/869%EB%85%84" title="869년">869년</a> 초 셋째 아들 <a href="/wiki/%EC%B9%B4%EB%A5%BC_3%EC%84%B8" title="카를 3세">카를 3세</a>가 봉토 배정에 불만을 품고 그를 상대로 반란을 일으켰다. 그러나 곧 <a href="/wiki/%EB%A1%9C%ED%83%80%EB%A5%B4_2%EC%84%B8" title="로타르 2세">로타르 2세</a>의 죽음 소식을 듣고 스스로 철군한다. <a href="/wiki/869%EB%85%84" title="869년">869년</a> <a href="/wiki/%EB%A1%9C%ED%83%80%EB%A5%B4_1%EC%84%B8" title="로타르 1세">로타르 1세</a>의 아들 <a href="/wiki/%EB%A1%9C%ED%83%80%EB%A5%B4_2%EC%84%B8" title="로타르 2세">로타르 2세</a>가 사망했다. 독일인 루트비히와 대머리 카를은 로타르 2세의 서자인 어린 <a href="/wiki/%EC%9C%84%EA%B7%B8_(%EB%A1%9C%ED%8A%B8%EB%A7%81%EA%B2%90%EC%9D%98_%EC%99%95%EC%9E%90)" title="위그 (로트링겐의 왕자)">위그</a>를 제치고 <a href="/wiki/%EB%A1%9C%ED%83%80%EB%A5%B4_2%EC%84%B8" title="로타르 2세">로타르 2세</a>의 영토를 공동 분할하기로 약속하였다. <a href="/wiki/869%EB%85%84" title="869년">869년</a> <a class="mw-redirect" href="/wiki/%EB%A1%9C%ED%83%80%EB%A7%81%EA%B8%B0%EC%95%84" title="로타링기아">로타링기아</a>에 도착한 대머리 카를은 자신이 <a class="mw-redirect" href="/wiki/%EB%A1%9C%ED%83%80%EB%A7%81%EA%B8%B0%EC%95%84" title="로타링기아">로타링기아</a> 전체를 정복했음을 선언하고, 자신을 스스로 황제이며 존엄한 자(Imperator Augustus)라고 선언했다. 이복 형인 독일인 루트비히 2세는 즉시 반발했다.</p>
<p>카를이 <a class="mw-redirect" href="/wiki/%EB%A1%9C%ED%83%80%EB%A7%81%EA%B8%B0%EC%95%84" title="로타링기아">로타링기아</a>를 분할하겠다는 약속을 어기고 로타링기아를 합병하자, <a href="/wiki/870%EB%85%84" title="870년">870년</a> 독일인 루트비히는 <a class="mw-redirect" href="/wiki/%EB%8F%99%ED%94%84%EB%9E%91%ED%81%AC" title="동프랑크">동프랑크</a>와 <a class="mw-redirect" href="/wiki/%EB%AA%A8%EB%9D%BC%EB%B9%84%EC%95%84" title="모라비아">모라비아</a>의 군대를 이끌고 <a class="mw-redirect" href="/wiki/%EB%A1%9C%ED%83%80%EB%A7%81%EA%B8%B0%EC%95%84" title="로타링기아">로타링기아</a>를 침공했다. 결국 <a class="mw-redirect" href="/wiki/%EA%B5%90%ED%99%A9%EC%B2%AD" title="교황청">교황청</a>이 나서서 중재를 요구했고, 그해 <a class="new" href="/w/index.php?title=%EB%A9%94%EB%A5%B4%EC%84%BC&amp;action=edit&amp;redlink=1" title="메르센 (없는 문서)">메르센</a>에서 대머리 카를을 만나 <a href="/wiki/%EB%A9%94%EB%A5%B4%EC%84%BC_%EC%A1%B0%EC%95%BD" title="메르센 조약">메르센 조약</a>에 따라 카를과 함께 <a class="mw-redirect" href="/wiki/%EB%A1%9C%ED%83%80%EB%A7%81%EA%B8%B0%EC%95%84" title="로타링기아">로타링기아</a>를 분할하는 조약을 체결하였다. 이때 루트비히는 옛 프랑크 왕국의 수도인 <a href="/wiki/%EC%95%84%ED%97%A8" title="아헨">아헨</a> 지역, <a href="/wiki/%EB%9D%BC%EC%9D%B8_%EA%B0%95" title="라인 강">라인 강</a> 서쪽의 영역과 <a href="/wiki/%EC%BE%B0%EB%A5%B8" title="쾰른">쾰른</a>, <a href="/wiki/%ED%94%84%EB%A6%AC%EC%8A%AC%EB%9E%80%ED%8A%B8" title="프리슬란트">프리슬란트</a> 등을 차지했다.</p>
<p><a href="/wiki/865%EB%85%84" title="865년">865년</a>과 <a href="/wiki/872%EB%85%84" title="872년">872년</a> 3명의 아들인 <a class="mw-disambig" href="/wiki/%EC%B9%B4%EB%A5%BC%EB%A1%9C%EB%A7%8C" title="카를로만">카를로만</a>, <a class="mw-redirect" href="/wiki/%ED%94%84%EB%9E%91%ED%81%AC%EC%9D%98_%EB%A3%A8%ED%8A%B8%EB%B9%84%ED%9E%88_3%EC%84%B8" title="프랑크의 루트비히 3세">루트비히 청년왕</a>, 비만왕 <a href="/wiki/%EC%B9%B4%EB%A5%BC_3%EC%84%B8" title="카를 3세">카를 3세</a>를 맏이에게는 바이에른과 <a class="mw-redirect" href="/wiki/%EC%BC%80%EB%A5%B8%ED%85%90" title="케른텐">케른텐</a>, 둘째 소 루트비히에게는 <a class="mw-redirect" href="/wiki/%EC%9E%91%EC%84%BC" title="작센">작센</a>, 셋째 카를에게는 각각 <a class="new" href="/w/index.php?title=%EC%95%8C%EB%A0%88%EB%A7%8C%EB%8B%88%EC%95%84&amp;action=edit&amp;redlink=1" title="알레만니아 (없는 문서)">알레만니아</a> 영토를 나누어 주었으나, 이 분할에 불만을 품은 아들들은 <a href="/wiki/861%EB%85%84" title="861년">861년</a>부터 <a href="/wiki/873%EB%85%84" title="873년">873년</a>에 계속해서 반란을 일으켰다.</p>
<p>루트비히는 <a class="mw-redirect" href="/wiki/%EB%AA%A8%EB%9D%BC%EB%B9%84%EC%95%84" title="모라비아">모라비아</a> 지방에 <a href="/wiki/%EA%B0%80%ED%86%A8%EB%A6%AD" title="가톨릭">가톨릭</a>을 전파하는 <a class="mw-redirect" href="/wiki/%ED%94%84%EB%9E%91%ED%81%AC_%EC%A1%B1" title="프랑크 족">프랑크 족</a> <a href="/wiki/%EC%84%A0%EA%B5%90%EC%82%AC" title="선교사">선교사</a>들을 지원하고 이들의 안전한 선교를 주선하였으나, 모라비아를 흡수하는 데는 실패하였다. <a href="/wiki/869%EB%85%84" title="869년">869년</a> 군사를 보내 <a class="mw-redirect" href="/wiki/%EB%AA%A8%EB%9D%BC%EB%B9%84%EC%95%84" title="모라비아">모라비아</a>를 공략하여 <a href="/wiki/870%EB%85%84" title="870년">870년</a> 병합시켰다. 그러나 모라비아는 <a class="new" href="/w/index.php?title=%EB%9D%BC%EB%94%94%EC%8A%AC%EB%9D%BC%ED%94%84&amp;action=edit&amp;redlink=1" title="라디슬라프 (없는 문서)">라디슬라프</a> 공의 지휘하에 분리독립을 기도했고, <a class="new" href="/w/index.php?title=%EB%9D%BC%EB%94%94%EC%8A%AC%EB%9D%BC%ED%94%84&amp;action=edit&amp;redlink=1" title="라디슬라프 (없는 문서)">라디슬라프</a>가 죽자 <a class="mw-redirect" href="/wiki/%EB%AA%A8%EB%9D%BC%EB%B9%84%EC%95%84" title="모라비아">모라비아</a>의 반란은 진압한 것처럼 보였다. 그러나 <a href="/wiki/871%EB%85%84" title="871년">871년</a> <a class="new" href="/w/index.php?title=%EC%8A%AC%EB%9D%BC%EB%B3%B4%EB%AF%B8%EB%A5%B4&amp;action=edit&amp;redlink=1" title="슬라보미르 (없는 문서)">슬라보미르</a>, 라디슬라프의 조카 <a class="new" href="/w/index.php?title=%EC%8A%A4%EB%B0%94%ED%86%A0%ED%92%80%ED%81%AC_2%EC%84%B8&amp;action=edit&amp;redlink=1" title="스바토풀크 2세 (없는 문서)">스바토풀크 2세</a> 등을 중심으로 <a class="mw-redirect" href="/wiki/%EB%AA%A8%EB%9D%BC%EB%B9%84%EC%95%84" title="모라비아">모라비아</a>인들은 다시 반란을 일으켰고 그해 <a class="mw-redirect" href="/wiki/%EB%8F%99%ED%94%84%EB%9E%91%ED%81%AC" title="동프랑크">동프랑크</a> 군을 몰아냄으로써 모라비아의 영지를 잃게 되었다. <a href="/wiki/873%EB%85%84" title="873년">873년</a> 다시 <a class="mw-redirect" href="/wiki/%EB%AA%A8%EB%9D%BC%EB%B9%84%EC%95%84" title="모라비아">모라비아</a>와 전쟁을 하였으나 전쟁에 져서 <a href="/wiki/874%EB%85%84" title="874년">874년</a> 이후 <a class="new" href="/w/index.php?title=%EB%8C%80_%EB%AA%A8%EB%9D%BC%EB%B9%84%EC%95%84&amp;action=edit&amp;redlink=1" title="대 모라비아 (없는 문서)">대 모라비아</a>는 독립했다.</p>
<p><a href="/wiki/870%EB%85%84" title="870년">870년</a>대 이후 다시 <a href="/wiki/%EB%B0%94%EC%9D%B4%ED%82%B9" title="바이킹">바이킹</a>들이 영토를 침략, <a href="/wiki/%ED%94%84%EB%A6%AC%EC%8A%AC%EB%9E%80%ED%8A%B8" title="프리슬란트">프리슬란트</a>, <a href="/wiki/%ED%95%A8%EB%B6%80%EB%A5%B4%ED%81%AC" title="함부르크">함부르크</a>, <a href="/wiki/%EB%B2%A0%EB%A5%BC%EB%A6%B0" title="베를린">베를린</a> 주변지역을 약탈하자 이를 물리쳤다. 그러나 <a href="/wiki/%EB%B0%94%EC%9D%B4%ED%82%B9" title="바이킹">바이킹</a>의 침입과 <a class="mw-redirect" href="/wiki/%EB%A7%88%EC%9E%90%EB%A5%B4%EC%A1%B1" title="마자르족">마자르족</a>, <a href="/wiki/%EC%8A%AC%EB%9D%BC%EB%B8%8C%EC%A1%B1" title="슬라브족">슬라브족</a>의 침략과 약탈은 계속되었다.</p>

<p>850년대 이후 사라센 인들의 이탈리아 남부 해안가 침략은 계속되었고, <a href="/wiki/872%EB%85%84" title="872년">872년</a> <a href="/wiki/12%EC%9B%94_14%EC%9D%BC" title="12월 14일">12월 14일</a> 즉위한 <a href="/wiki/%EA%B5%90%ED%99%A9_%EC%9A%94%ED%95%9C_8%EC%84%B8" title="교황 요한 8세">교황 요한 8세</a>는 루트비히 2세에게 도움을 청했다. 그는 군사를 보내 이탈리아 남부로 들어오는 사라센 군 및 해적들을 상대하게 했다. 한편 그는 여러번 <a class="mw-redirect" href="/wiki/%EA%B5%90%ED%99%A9%EC%B2%AD" title="교황청">교황청</a>에 교섭하여 자신의 아들들 중 한 명에게 <a class="mw-redirect" href="/wiki/%EC%8B%A0%EC%84%B1%EB%A1%9C%EB%A7%88%EC%A0%9C%EA%B5%AD" title="신성로마제국">신성로마제국</a> 황제의 관을 수여해줄 것을 요구했지만 <a class="mw-redirect" href="/wiki/%EA%B5%90%ED%99%A9%EC%B2%AD" title="교황청">교황청</a>은 그때마다 거부했다.</p>
<p><a href="/wiki/875%EB%85%84" title="875년">875년</a> <a href="/wiki/8%EC%9B%94" title="8월">8월</a> 조카이자 사위였던 <a class="mw-redirect" href="/wiki/%EC%8B%A0%EC%84%B1%EB%A1%9C%EB%A7%88%EC%A0%9C%EA%B5%AD" title="신성로마제국">신성로마제국</a>의 황제 <a href="/wiki/%EB%A3%A8%ED%8A%B8%EB%B9%84%ED%9E%88_2%EC%84%B8_(%EC%9D%B4%ED%83%88%EB%A6%AC%EC%95%84)" title="루트비히 2세 (이탈리아)">황제 루트비히 2세</a>는 독일인 루트비히의 장남 <a class="mw-redirect" href="/wiki/%EC%B9%B4%EB%A5%BC%EB%A1%9C%EB%A7%8C_3%EC%84%B8" title="카를로만 3세">카를로만 3세</a>를 자신의 후계자로 지명하고 사망했으나, 이복동생인 대머리왕 <a class="mw-disambig" href="/wiki/%EC%B9%B4%EB%A5%BC_2%EC%84%B8" title="카를 2세">카를 2세</a>가 그를 제치고 <a class="mw-disambig" href="/wiki/%EC%95%8C%ED%94%84%EC%8A%A4" title="알프스">알프스</a>를 넘어 <a href="/wiki/%EC%9D%B4%ED%83%88%EB%A6%AC%EC%95%84" title="이탈리아">이탈리아</a>의 왕과 <a class="mw-redirect" href="/wiki/%EC%8B%A0%EC%84%B1%EB%A1%9C%EB%A7%88%EC%A0%9C%EA%B5%AD" title="신성로마제국">신성로마제국</a>의 요한네스 8세 교황의 지지를 받아 제관을 차지하자, 루트비히는 로타링기아에 있는 카를의 영지에 쳐들어가려다 실패했다. 독일인 루트비히는 조카의 죽음과 유언이 공개되자 바로 셋째 아들 뚱보 <a href="/wiki/%EC%B9%B4%EB%A5%BC_3%EC%84%B8" title="카를 3세">카를 3세</a>를 북이탈리아로 보내 <a href="/wiki/%ED%94%84%EB%A6%AC%EC%9A%B8%EB%A6%AC" title="프리울리">프리울리</a> <a href="/wiki/%ED%9B%84%EC%9E%91" title="후작">후작</a> <a class="mw-redirect" href="/wiki/%EB%B2%A0%EB%A0%9D%EA%B0%80%EB%A6%AC%EC%98%A4_1%EC%84%B8" title="베렝가리오 1세">베렝가리오 1세</a>와 손잡고 <a href="/wiki/%EC%B9%B4%EB%A5%BC%EB%A7%8C" title="카를만">카를만</a>의 추대 협상을 지원했다. 그러나 카를은 재빨리 알프스를 넘어와 교황의 보호자는 자닌임을 선언하고 제관을 차지했다.</p>
<p>대머리 카를은 자신의 형 루트비히가 살아 있을 때에도 황제 직위를 욕심내어 세력을 이탈리아까지 확대했고, 형 독일인 루트비히가 <a class="mw-redirect" href="/wiki/%ED%94%84%EB%9E%91%ED%81%AC%ED%91%B8%EB%A5%B4%ED%8A%B8" title="프랑크푸르트">프랑크푸르트</a>에서 죽자 <a href="/wiki/%EB%9D%BC%EC%9D%B8_%EA%B0%95" title="라인 강">라인 강</a>까지 진출하려 했다.<sup class="reference" id="cite_ref-ppkk44_2-1"><a href="#cite_note-ppkk44-2">[2]</a></sup> 독일인 루트비히의 둘째 아들인 청년 루트비히가 작은 아버지인 카를과 담판하여 <a href="/wiki/%ED%94%84%EB%9E%91%EC%BC%84" title="프랑켄">프랑켄</a>과 <a class="mw-redirect" href="/wiki/%EC%9E%91%EC%84%BC" title="작센">작센</a>에서 군대 소집권을 주기로 하고 전쟁을 종결했다.<sup class="reference" id="cite_ref-ppkk44_2-2"><a href="#cite_note-ppkk44-2">[2]</a></sup> 카를은 전쟁 노획물과 많은 선물을 받아 서프랑크로 귀환했다.<sup class="reference" id="cite_ref-ppkk44_2-3"><a href="#cite_note-ppkk44-2">[2]</a></sup></p>
<p><a href="/wiki/874%EB%85%84" title="874년">874년</a>경부터 부인 <a href="/wiki/%EB%B0%94%EC%9D%B4%EC%97%90%EB%A5%B8%EC%9D%98_%EC%97%A0%EB%AF%80" title="바이에른의 엠므">바이에른의 엠므</a>가 <a href="/wiki/%EB%87%8C%EC%A1%B8%EC%A4%91" title="뇌졸중">뇌졸중</a>으로 무기력해졌다. <a href="/wiki/876%EB%85%84" title="876년">876년</a> <a href="/wiki/1%EC%9B%94" title="1월">1월</a> 부인 <a href="/wiki/%EB%B0%94%EC%9D%B4%EC%97%90%EB%A5%B8%EC%9D%98_%EC%97%A0%EB%AF%80" title="바이에른의 엠므">엠므</a>의 죽음을 보았고, 그해, <a href="/wiki/%EC%84%9C%EB%A1%9C%EB%A7%88_%EC%A0%9C%EA%B5%AD" title="서로마 제국">서로마 제국</a>의 제관을 차지한 <a class="mw-disambig" href="/wiki/%EC%B9%B4%EB%A5%BC_2%EC%84%B8" title="카를 2세">카를 2세</a>를 다시 치려고 군사를 일으키던 중 그해 <a href="/wiki/8%EC%9B%94" title="8월">8월</a> <a class="mw-redirect" href="/wiki/%ED%94%84%EB%9E%91%ED%81%AC%ED%91%B8%EB%A5%B4%ED%8A%B8" title="프랑크푸르트">프랑크푸르트</a>에서 갑자기 사망했다.</p>
<h3><span class="mw-headline" id=".EC.82.AC.ED.9B.84">사후</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%A3%A8%ED%8A%B8%EB%B9%84%ED%9E%88_2%EC%84%B8_(%EB%8F%99%ED%94%84%EB%9E%91%ED%81%AC)&amp;action=edit&amp;section=16" title="부분 편집: 사후">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p><a href="/wiki/8%EC%9B%94_28%EC%9D%BC" title="8월 28일">8월 28일</a> 아들 <a href="/wiki/%EB%A3%A8%ED%8A%B8%EB%B9%84%ED%9E%88_3%EC%84%B8_(%EB%8F%99%ED%94%84%EB%9E%91%ED%81%AC)" title="루트비히 3세 (동프랑크)">소 루트비히</a>에 의해 <a class="new" href="/w/index.php?title=%EB%A1%9C%EC%89%AC_%EC%88%98%EB%8F%84%EC%9B%90&amp;action=edit&amp;redlink=1" title="로쉬 수도원 (없는 문서)">로쉬 수도원</a>(abbey of Lorsch)에 안치되었다. 그러나 <a href="/wiki/1920%EB%85%84" title="1920년">1920년</a> <a class="new" href="/w/index.php?title=%EB%A1%9C%EC%89%AC_%EC%88%98%EB%8F%84%EC%9B%90&amp;action=edit&amp;redlink=1" title="로쉬 수도원 (없는 문서)">로쉬 수도원</a>에서 그의 석관이 발견되었을 때 그의 시신은 분실된 상태였다.</p>
<p>황제관을 얻는 데도 실패했고, 로타르 1세의 아들인 황제 루트비히 2세가 죽은 뒤 자기 후손을 <a href="/wiki/%EC%9D%B4%ED%83%88%EB%A6%AC%EC%95%84" title="이탈리아">이탈리아</a> 왕위에 앉히려 했지만 그것도 성공하지 못했다. 그러나 <a class="mw-disambig" href="/wiki/%EC%B9%B4%EB%A5%BC_2%EC%84%B8" title="카를 2세">카를 2세</a>가 죽고 <a class="mw-disambig" href="/wiki/%EC%B9%B4%EB%A5%BC%EB%A1%9C%EB%A7%8C" title="카를로만">카를로만</a>이 재빨리 <a class="mw-disambig" href="/wiki/%EC%95%8C%ED%94%84%EC%8A%A4" title="알프스">알프스</a>를 넘어가 이탈리아의 왕좌를 차지하게 된다. 그의 죽음으로 그의 영지는 다시 세 부분으로 나누어져, <a class="mw-redirect" href="/wiki/%EB%B0%94%EC%9D%B4%EC%97%90%EB%A5%B8" title="바이에른">바이에른</a>은 <a href="/wiki/%EC%B9%B4%EB%A5%BC%EB%A7%8C" title="카를만">카를만</a>에게, <a class="mw-redirect" href="/wiki/%EC%9E%91%EC%84%BC" title="작센">작센</a>은 <a class="new" href="/w/index.php?title=%EC%86%8C%EB%A3%A8%ED%8A%B8%EB%B9%84%ED%9E%88&amp;action=edit&amp;redlink=1" title="소루트비히 (없는 문서)">소루트비히</a>에게 <a href="/wiki/%EC%8A%88%EB%B0%94%EB%B2%A4" title="슈바벤">슈바벤</a>과 <a class="new" href="/w/index.php?title=%EC%95%8C%EB%A0%88_%EB%A7%88%EB%8B%88%EC%95%84&amp;action=edit&amp;redlink=1" title="알레 마니아 (없는 문서)">알레 마니아</a>는 <a href="/wiki/%EC%B9%B4%EB%A5%BC_3%EC%84%B8" title="카를 3세">카를 3세</a>에게 넘겨졌다.</p>
<p><a class="mw-redirect" href="/wiki/%EB%8F%85%EC%9D%BC%EC%82%AC" title="독일사">독일사</a>와 <a href="/wiki/%EC%98%A4%EC%8A%A4%ED%8A%B8%EB%A6%AC%EC%95%84" title="오스트리아">오스트리아</a>사에서는 루트비히를 <a href="/wiki/%EB%8F%85%EC%9D%BC" title="독일">독일</a> 혹은 <a href="/wiki/%EC%98%A4%EC%8A%A4%ED%8A%B8%EB%A6%AC%EC%95%84" title="오스트리아">오스트리아</a>의 첫 군주로 친다.</p>

<h2><span class="mw-headline" id=".EA.B8.B0.ED.83.80">기타</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%A3%A8%ED%8A%B8%EB%B9%84%ED%9E%88_2%EC%84%B8_(%EB%8F%99%ED%94%84%EB%9E%91%ED%81%AC)&amp;action=edit&amp;section=18" title="부분 편집: 기타">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>그의 혈통을 이은 <a href="/wiki/%EC%9D%B4%ED%83%88%EB%A6%AC%EC%95%84%EC%9D%98_%EA%B5%B0%EC%A3%BC" title="이탈리아의 군주">이탈리아의 왕</a>으로는 <a class="mw-redirect" href="/wiki/%EC%B9%B4%EB%A5%BC%EB%A1%9C%EB%A7%8C_3%EC%84%B8" title="카를로만 3세">카를로만 3세</a>, <a href="/wiki/%EC%B9%B4%EB%A5%BC_3%EC%84%B8" title="카를 3세">카를 3세</a>, <a href="/wiki/%EC%95%84%EB%A5%B4%EB%88%8C%ED%94%84_%ED%8F%B0_%EC%BC%80%EB%A5%B8%ED%85%90" title="아르눌프 폰 케른텐">아르눌프</a>, <a class="mw-redirect" href="/wiki/%EB%9D%BC%ED%86%A8%EB%93%9C" title="라톨드">라톨드</a>, <a class="mw-redirect" href="/wiki/%EB%A3%A8%ED%8A%B8%EB%B9%84%ED%9E%88_3%EC%84%B8_(%EC%9D%B4%ED%83%88%EB%A6%AC%EC%95%84)" title="루트비히 3세 (이탈리아)">루트비히 맹인왕</a> 등이 있다. <a class="mw-redirect" href="/wiki/%EB%A3%A8%ED%8A%B8%EB%B9%84%ED%9E%88_3%EC%84%B8_(%EC%9D%B4%ED%83%88%EB%A6%AC%EC%95%84)" title="루트비히 3세 (이탈리아)">맹인왕 루트비히</a>는 그의 딸 이르멘가르트의 외손자가 된다.</p>
<p>독일인 루트비히(Ludwig der Deutsche, 또는 Louis the German)라는 별칭은 그의 사후에 붙여진 별칭이다.</p>
