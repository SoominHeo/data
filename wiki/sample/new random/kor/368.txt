<p><b>플랑드르</b>(<span style="font-size: smaller;"><a href="/wiki/%ED%94%84%EB%9E%91%EC%8A%A4%EC%96%B4" title="프랑스어">프랑스어</a>:</span> <span lang="fr" xml:lang="fr">Flandre</span>, <span style="font-size: smaller;"><a href="/wiki/%EB%84%A4%EB%8D%9C%EB%9E%80%EB%93%9C%EC%96%B4" title="네덜란드어">네덜란드어</a>:</span> <span lang="nl" xml:lang="nl">Vlaanderen</span> <small>플란데런<sup>[<a href="/wiki/%EC%9C%84%ED%82%A4%EB%B0%B1%EA%B3%BC:%EB%84%A4%EB%8D%9C%EB%9E%80%EB%93%9C%EC%96%B4%EC%9D%98_%ED%95%9C%EA%B8%80_%ED%91%9C%EA%B8%B0" title="위키백과:네덜란드어의 한글 표기">*</a>]</sup></small>)는 <a href="/wiki/%EB%B2%A8%EA%B8%B0%EC%97%90" title="벨기에">벨기에</a>의 북방을 부르는 말로서 다음 두 지역 중 하나를 가리킨다.</p>
<ul>
<li><a href="/wiki/%ED%94%8C%EB%9D%BC%EB%A7%9D_%EC%A7%80%EC%97%AD" title="플라망 지역">플라망 지역</a>(Vlaams Gewest)</li>
<li><a href="/wiki/%ED%94%8C%EB%9D%BC%EB%A7%9D%EC%96%B4_%EA%B3%B5%EB%8F%99%EC%B2%B4" title="플라망어 공동체">플라망어 공동체</a>(Vlaamse Gemeenschap)</li>
</ul>
<p>플랑드르의 정의는 시대에 따라 계속 바뀌어 왔다. <a href="/wiki/%EC%A4%91%EC%84%B8" title="중세">중세</a> 시대에는 플랑드르는 <a href="/wiki/%EC%84%9C%EC%9C%A0%EB%9F%BD" title="서유럽">서유럽</a>의 다음 지역을 다스렸던 <a class="new" href="/w/index.php?title=%ED%94%8C%EB%9E%91%EB%93%9C%EB%A5%B4_%EB%B0%B1%EC%9E%91%EB%A0%B9&amp;action=edit&amp;redlink=1" title="플랑드르 백작령 (없는 문서)">플랑드르 백작령</a>을 가리킨다.</p>
<ul>
<li><a href="/wiki/%ED%94%84%EB%9E%91%EC%8A%A4" title="프랑스">프랑스</a>의 북부 <a href="/wiki/%EB%85%B8%EB%A5%B4%ED%8C%8C%EB%93%9C%EC%B9%BC%EB%A0%88" title="노르파드칼레">노르파드칼레</a>에 속하는 <a href="/wiki/%EB%85%B8%EB%A5%B4_%EC%A3%BC" title="노르 주">노르 주</a>의 일부</li>
<li><a href="/wiki/%EB%B2%A8%EA%B8%B0%EC%97%90" title="벨기에">벨기에</a>의 서부 저지대의 <a href="/wiki/%EB%B2%A0%EC%8A%A4%ED%8A%B8%ED%94%8C%EB%9E%80%EB%8D%B0%EB%9F%B0_%EC%A3%BC" title="베스트플란데런 주">베스트플란데런 주</a>와 <a href="/wiki/%EC%98%A4%EC%8A%A4%ED%8A%B8%ED%94%8C%EB%9E%80%EB%8D%B0%EB%9F%B0_%EC%A3%BC" title="오스트플란데런 주">오스트플란데런 주</a></li>
<li><a href="/wiki/%EB%84%A4%EB%8D%9C%EB%9E%80%EB%93%9C" title="네덜란드">네덜란드</a>의 남서부 <a href="/wiki/%EC%A0%9C%EC%9D%BC%EB%9E%80%ED%8A%B8_%EC%A3%BC" title="제일란트 주">제일란트 주</a>의 남방 일부</li>
</ul>
