<p><b>MBC 게임</b>(MBC Game)은 <a href="/wiki/%EB%8C%80%ED%95%9C%EB%AF%BC%EA%B5%AD" title="대한민국">대한민국</a>의 <a href="/wiki/%EA%B2%8C%EC%9E%84" title="게임">게임</a> 전문, 텔레비전 방송 채널로서 <a class="mw-redirect" href="/wiki/MBC_%ED%94%8C%EB%9F%AC%EC%8A%A4_%EB%AF%B8%EB%94%94%EC%96%B4" title="MBC 플러스 미디어">MBC 플러스 미디어</a>가 운영하는 케이블 TV 네트워크였다. <a href="/wiki/MBC_%EB%AE%A4%EC%A7%81" title="MBC 뮤직">MBC 뮤직</a>의 전신에 해당된다.</p>
<p>2001년 3월 케이블 TV 시장 진출을 선언한 <a class="mw-redirect" href="/wiki/MBC" title="MBC">MBC</a>는 <a class="mw-disambig" href="/wiki/%EC%A0%9C%EC%9D%BC%EC%A0%9C%EB%8B%B9" title="제일제당">제일제당</a> 계열의 <a class="mw-redirect" href="/wiki/CJ39%EC%87%BC%ED%95%91" title="CJ39쇼핑">CJ39쇼핑</a> 소속 패션 전문 채널이었던 LOOK TV를 인수하여 2001년 5월 1일에 gembc(겜비시)라는 이름으로 개국하게 된다. 개국 초기에는 <a class="mw-redirect" href="/wiki/%EC%98%A8%EA%B2%8C%EC%9E%84%EB%84%B7" title="온게임넷">온게임넷</a>을 제치고 <a href="/wiki/%EC%9B%94%EB%93%9C_%EC%82%AC%EC%9D%B4%EB%B2%84_%EA%B2%8C%EC%9E%84%EC%A6%88" title="월드 사이버 게임즈">월드 사이버 게임즈</a>의 중계권을 유치하는 등 적극적인 활동으로 온게임넷과 더불어 국내 게임 방송의 양대 산맥으로 성장하였다. 이후 2003년 1월 1일부터 <b>MBC Game</b>으로 채널명을 변경하였다.</p>
<p>수많은 <a href="/wiki/E%EC%8A%A4%ED%8F%AC%EC%B8%A0" title="E스포츠">E스포츠</a> 중계를 하고 있었지만, <a href="/wiki/%EC%8A%A4%ED%83%80%ED%81%AC%EB%9E%98%ED%94%84%ED%8A%B8" title="스타크래프트">스타크래프트</a> 리그 중 하나인 <a href="/wiki/MBC%EA%B2%8C%EC%9E%84_%EC%8A%A4%ED%83%80%EB%A6%AC%EA%B7%B8" title="MBC게임 스타리그">MBC게임 스타리그</a>는 인기가 많았다. <a href="/wiki/MBC%EA%B2%8C%EC%9E%84_%EC%8A%A4%ED%83%80%EB%A6%AC%EA%B7%B8" title="MBC게임 스타리그">MBC게임 스타리그</a>은 주로 <a href="/wiki/%EB%AA%A9%EC%9A%94%EC%9D%BC" title="목요일">목요일</a>, <a href="/wiki/%ED%86%A0%EC%9A%94%EC%9D%BC" title="토요일">토요일</a>에 생중계했다.</p>
<p>2003년부터 2004년까지는 현재의 <a class="mw-disambig" href="/wiki/%ED%94%84%EB%A1%9C%EB%A6%AC%EA%B7%B8" title="프로리그">프로리그</a>와 방식이 다른 <a href="/wiki/%EC%8A%A4%ED%83%80%ED%81%AC%EB%9E%98%ED%94%84%ED%8A%B8" title="스타크래프트">스타크래프트</a> 팀전인 <a class="mw-disambig" href="/wiki/%ED%8C%80%EB%A6%AC%EA%B7%B8" title="팀리그">팀리그</a>가 인기를 끌기도 했다. 2005년부터 2011년까지 <a class="mw-redirect" href="/wiki/%EC%98%A8%EA%B2%8C%EC%9E%84%EB%84%B7" title="온게임넷">온게임넷</a>과 더불어 <a href="/wiki/%ED%95%9C%EA%B5%ADe%EC%8A%A4%ED%8F%AC%EC%B8%A0%ED%98%91%ED%9A%8C" title="한국e스포츠협회">한국e스포츠협회</a>에서 공인한 양대 스타크래프트 리그 방송사로, <a class="mw-redirect" href="/wiki/%EC%98%A8%EA%B2%8C%EC%9E%84%EB%84%B7" title="온게임넷">온게임넷</a>과 함께 <a href="/wiki/%EC%8A%A4%ED%83%80%ED%81%AC%EB%9E%98%ED%94%84%ED%8A%B8_%ED%94%84%EB%A1%9C%EB%A6%AC%EA%B7%B8" title="스타크래프트 프로리그">프로리그</a>를 방송했었다. 하지만 2012년 2월 1일에 <a href="/wiki/MBC_%EB%AE%A4%EC%A7%81" title="MBC 뮤직">MBC 뮤직</a>으로 채널 장르를 전환시킴으로써 MBC 게임은 <a href="/wiki/2012%EB%85%84" title="2012년">2012년</a> <a href="/wiki/1%EC%9B%94_31%EC%9D%BC" title="1월 31일">1월 31일</a>까지 방송하고 폐국되었다.<sup class="reference" id="cite_ref-1"><a href="#cite_note-1">[1]</a></sup></p>
<p>2006년에는 프로게임팀 POS를 인수해 <a href="/wiki/MBC%EA%B2%8C%EC%9E%84_%ED%9E%88%EC%96%B4%EB%A1%9C" title="MBC게임 히어로">MBC게임 히어로</a>를 창단했었다.</p>

<h2><span class="mw-headline" id="MBC.EA.B2.8C.EC.9E.84_.EC.97.AD.EC.82.AC.EC.9D.98_.ED.95.9C_.ED.8E.98.EC.9D.B4.EC.A7.80.EB.A1.9C_.EB.82.A8.EA.B2.8C_.EB.90.9C_.EC.9D.B4.EC.A0.84_.EB.B0.8F_.EC.A2.85.EC.98.81_.ED.94.84.EB.A1.9C.EA.B7.B8.EB.9E.A8">MBC게임 역사의 한 페이지로 남게 된 이전 및 종영 프로그램</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=MBC_%EA%B2%8C%EC%9E%84&amp;action=edit&amp;section=2" title="부분 편집: MBC게임 역사의 한 페이지로 남게 된 이전 및 종영 프로그램">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<ul>
<li><a class="mw-disambig" href="/wiki/MSL" title="MSL">MBC게임 스타리그</a> - MBC게임이 설립과 동시에 열린 KPGA투어를 전신으로 하는 스타리그였다. MBC게임이 폐지되어서 그와 동시에 리그가 중단되었다.</li>
<li><a href="/wiki/MSL_%EB%A7%88%EC%9D%B4%EB%84%88%EB%A6%AC%EA%B7%B8" title="MSL 마이너리그">MSL 마이너리그</a> - 초기 <a href="/wiki/MBC%EA%B2%8C%EC%9E%84_%EC%8A%A4%ED%83%80%EB%A6%AC%EA%B7%B8" title="MBC게임 스타리그">MSL</a>의 이전 하부리그였다.</li>
<li><a href="/wiki/MSL_%EC%84%9C%EB%B0%94%EC%9D%B4%EB%B2%84_%ED%86%A0%EB%84%88%EB%A8%BC%ED%8A%B8" title="MSL 서바이버 토너먼트">MSL 서바이버 토너먼트</a> - MSL의 하부리그 성격을 띠는 리그였다.</li>
<li><a class="mw-disambig" href="/wiki/%ED%8C%80%EB%A6%AC%EA%B7%B8" title="팀리그">팀리그</a> - MBC게임의 <a href="/wiki/%EC%8A%A4%ED%83%80%ED%81%AC%EB%9E%98%ED%94%84%ED%8A%B8" title="스타크래프트">스타크래프트</a> 팀 단위 리그이며, <a href="/wiki/2005%EB%85%84" title="2005년">2005년</a> <a href="/wiki/%EC%8A%A4%ED%83%80%ED%81%AC%EB%9E%98%ED%94%84%ED%8A%B8_%ED%94%84%EB%A1%9C%EB%A6%AC%EA%B7%B8" title="스타크래프트 프로리그">프로리그</a>와 통합되기 전의 리그였다.</li>
<li><a href="/wiki/%EC%8A%A4%ED%83%80%ED%81%AC%EB%9E%98%ED%94%84%ED%8A%B8_%ED%94%84%EB%A1%9C%EB%A6%AC%EA%B7%B8" title="스타크래프트 프로리그">프로리그</a> - 한국e스포츠협회 공인 스타크래프트 리그전. 2005년부터 2011년까지 방송함.</li>
<li><a class="mw-redirect" href="/wiki/TEKKEN_CRASH" title="TEKKEN CRASH">TEKKEN CRASH</a> - MBC GAME에서 열리는 철권 리그였다. MSL과 마찬가지로 MBC게임이 폐지되어서 그와 동시에 리그가 중단되었다.</li>
<li><a class="mw-redirect" href="/wiki/%ED%94%84%EB%A1%9C%EB%A6%AC%EA%B7%B8_(%EC%8A%A4%ED%8E%98%EC%85%9C%ED%8F%AC%EC%8A%A4)" title="프로리그 (스페셜포스)">SF 프로리그</a> - 한국e스포츠협회 공인 스페셜포스 리그전.</li>
<li><a href="/wiki/%EA%B2%BD%EB%82%A8_-_STX%EC%BB%B5_%EB%A7%88%EC%8A%A4%ED%84%B0%EC%A6%88" title="경남 - STX컵 마스터즈">경남 - STX컵 마스터즈</a> - <a href="/wiki/%EC%8A%A4%ED%83%80%ED%81%AC%EB%9E%98%ED%94%84%ED%8A%B8_%ED%94%84%EB%A1%9C%EB%A6%AC%EA%B7%B8" title="스타크래프트 프로리그">프로리그</a> 종료 후, 치러지는 스타크래프트 리그전. MSL과 마찬가지로 MBC게임이 폐지되어서 그와 동시에 리그가 중단되었다.</li>
<li><a class="mw-redirect" href="/wiki/%EB%A6%AC%EC%96%BC%EC%82%AC%EC%BB%A4_%EC%B1%94%ED%94%BC%EC%96%B8%EC%89%BD_%EB%A6%AC%EA%B7%B8" title="리얼사커 챔피언쉽 리그">리얼사커 챔피언쉽 리그</a> - MBC GAME에서 열리는 리얼사커 리그였다. MSL과 마찬가지로 MBC게임이 폐지되어서 그와 동시에 리그가 중단되었다.</li>
<li>엘리트 학생복 스쿨리그 - 중고등학교 단위의 팀리그이자 2011시즌은 MBC게임의 마지막 리그이자, 역대 스쿨리그중에서 가장 많은 프로게이머들이 배출된 리그이기도 하다. MSL과 마찬가지로 MBC게임이 폐지되어서 그와 동시에 중단되었다.</li>
<li><b><a class="mw-redirect" href="/wiki/%EC%8A%A4%ED%83%80_%EB%AC%B4%ED%95%9C_%EB%8F%84%EC%A0%84" title="스타 무한 도전">스타 무한 도전</a></b> - 6년 동안 최장수프로그램으로 방영한 MBC게임의 다섯명의 해설자들이 배틀넷에서 인기있는 <a class="mw-redirect" href="/wiki/%EC%9C%A0%EC%A6%88_%EB%A7%B5_%EC%84%B8%ED%8C%85" title="유즈 맵 세팅">유즈맵</a>을 플레이하는 케이블TV 최초의 예능프로그램 겸 최장수 예능프로그램이었으며, 2012년 MBC게임이 역사 속으로 파묻혀서 종영되었다.</li>
<li>-종영일- : 스타무한도전:2012년 1월 23일 스무도 미니 LIVE:2012년 1월 27일</li>
<li><b><a href="/wiki/%EC%95%B3%ED%94%8C%EB%A0%88%EC%9D%B4" title="앳플레이">앳플레이</a></b> - 스타무한도전 다음의 5년간 방영한 2번째 장수 프로그램이자, 프로게이머가 아닌, MBC게임의 해설진들과 함께 경기를 하는 방송이었으며, 2012년 1월 25일에 역사 속으로 사라졌다.</li>
<li><b><a href="/wiki/%EC%8A%A4%ED%83%80_%EB%B0%B0%EB%84%B7_%EC%96%B4%ED%83%9D" title="스타 배넷 어택">스타 배넷 어택</a></b> - 프로게이머들을 초청해, 일반유저,시청자들과의 좌충우돌 배틀넷 대결을 하는 예능프로그램이었으며, 2012년 1월 27일에 역사 속의 한 페이지로 남게 되었다.</li>
<li><b><a href="/wiki/%EC%87%BC_%EB%A6%AC%ED%94%8C%EB%A0%88%EC%9D%B4_%ED%99%A9%EB%8B%B9%EB%AC%B4%EC%A0%81" title="쇼 리플레이 황당무적">쇼 리플레이 황당무적</a></b> - 시청자들과 유저들에 의해 만들어진 재미있고 엽기적인 리플레이들을 보여주는 프로그램이었으며, 2012년 1월 26일 역사 속의 한 페이지로 남게 되었다.</li>
<li><b><a href="/wiki/%EC%84%B1%EC%B6%98%EC%87%BC" title="성춘쇼">성춘쇼</a></b> - 임성춘의 리얼 2인자 버라이어티이자, 토크와 게임의 묘한 조화를 이룬 프로그램이었으며, 2012년 1월 27일 역사 속으로 사라졌다.</li>
<li><b><a href="/wiki/%EC%9C%A0%EC%A0%80%EC%9D%98_%EC%B7%A8%ED%96%A5" title="유저의 취향">유저의 취향</a></b> - 시청자들의 직접 만든 맵을 직접 제작해서 MBC게임 중계진들이 플레이하는 형식으로 시청자들과 중계진들이 함께하는 방송이었으며, 2012년 1월 26일 역사 속으로 파묻혀서 종영되었다.</li>
