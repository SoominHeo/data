<p><b>호텔</b>(Hotel, <a href="/wiki/%EB%AC%B8%ED%99%94%EC%96%B4" title="문화어">문화어</a>:려관-旅館)은 보통 짧은 기간 동안 <a href="/wiki/%EC%88%99%EB%B0%95" title="숙박">숙박</a>을 제공하는 곳이다. 호텔은 손님을 위해 수많은 부가 서비스를 제공한다. 이를테면, <a href="/wiki/%EC%8B%9D%EB%8B%B9" title="식당">식당</a>, <a href="/wiki/%EC%88%98%EC%98%81%EC%9E%A5" title="수영장">수영장</a>, <a class="new" href="/w/index.php?title=%EB%B3%B4%EC%9C%A1_%EC%8B%9C%EC%84%A4&amp;action=edit&amp;redlink=1" title="보육 시설 (없는 문서)">보육</a>이 있다. 호텔에 별의 개수를 매겨 <a class="new" href="/w/index.php?title=%EC%84%B1%EA%B8%89&amp;action=edit&amp;redlink=1" title="성급 (없는 문서)">성급</a>을 따지기도 한다. 주로 5성급까지있다. 성급이 높을수록 고급 호텔로 여겨진다.</p>
<p><a href="/wiki/%EB%8C%80%ED%95%9C%EB%AF%BC%EA%B5%AD" title="대한민국">대한민국</a>에서는 한국의 호텔로서 <b><a href="/wiki/%EC%97%AC%EA%B4%80" title="여관">여관</a></b>이라는 말을 쓰기도 한다. <a href="/wiki/%EC%98%A4%EC%8A%A4%ED%8A%B8%EB%A0%88%EC%9D%BC%EB%A6%AC%EC%95%84" title="오스트레일리아">오스트레일리아</a>에서는 바(bar) 또는 퍼브(pub)라고 일컫는다. <a href="/wiki/%EC%9D%B8%EB%8F%84" title="인도">인도</a>에서는 <a href="/wiki/%EC%8B%9D%EB%8B%B9" title="식당">식당</a>이라고 일컫지만, <a href="/wiki/%EC%A4%91%ED%99%94%EC%9D%B8%EB%AF%BC%EA%B3%B5%ED%99%94%EA%B5%AD" title="중화인민공화국">중화인민공화국</a>, <a href="/wiki/%EC%A4%91%ED%99%94%EB%AF%BC%EA%B5%AD" title="중화민국">중화민국</a>, <a href="/wiki/%ED%99%8D%EC%BD%A9" title="홍콩">홍콩</a>, <a href="/wiki/%EB%A7%88%EC%B9%B4%EC%98%A4" title="마카오">마카오</a>, <a href="/wiki/%EB%A7%90%EB%A0%88%EC%9D%B4%EC%8B%9C%EC%95%84" title="말레이시아">말레이시아</a>, <a href="/wiki/%EC%8B%B1%EA%B0%80%ED%8F%AC%EB%A5%B4" title="싱가포르">싱가포르</a> 등 중화권 국가에서는 <a href="/wiki/%EC%88%A0%EC%A7%91" title="술집">주점</a>이라고 부른다.</p>
