<p><b>알반 마리아 요하네스 베르크</b>(Alban Maria Johannes Berg, <a href="/wiki/1885%EB%85%84" title="1885년">1885년</a> <a href="/wiki/2%EC%9B%94_9%EC%9D%BC" title="2월 9일">2월 9일</a> ~ <a href="/wiki/1935%EB%85%84" title="1935년">1935년</a> <a href="/wiki/12%EC%9B%94_24%EC%9D%BC" title="12월 24일">12월 24일</a>)는 <a href="/wiki/%EC%98%A4%EC%8A%A4%ED%8A%B8%EB%A6%AC%EC%95%84" title="오스트리아">오스트리아</a>의 <a href="/wiki/%EC%9E%91%EA%B3%A1%EA%B0%80" title="작곡가">작곡가</a>이다. 베르크는 <a class="mw-redirect" href="/wiki/%EC%95%84%EB%86%80%EB%93%9C_%EC%87%A4%EB%B2%A0%EB%A5%B4%ED%81%AC" title="아놀드 쇤베르크">아놀드 쇤베르크</a>와 <a href="/wiki/%EC%95%88%ED%86%A4_%EB%B2%A0%EB%B2%A0%EB%A5%B8" title="안톤 베베른">안톤 베베른</a>과 함께 <a class="new" href="/w/index.php?title=%EC%A0%9C2_%EB%B9%88%EC%95%85%ED%8C%8C&amp;action=edit&amp;redlink=1" title="제2 빈악파 (없는 문서)">제2 빈악파</a>의 구성원으로, <a class="mw-redirect" href="/wiki/%EB%A7%90%EB%9F%AC" title="말러">말러</a>적인 <a href="/wiki/%EB%82%AD%EB%A7%8C%EC%A3%BC%EC%9D%98" title="낭만주의">낭만주의</a>와 <a class="mw-redirect" href="/wiki/%EC%87%A4%EB%B2%A0%EB%A5%B4%ED%81%AC" title="쇤베르크">쇤베르크</a>의 <a class="mw-redirect" href="/wiki/12%EC%9D%8C%EA%B8%B0%EB%B2%95" title="12음기법">12음기법</a>을 매우 개인적으로 적용하여 작곡하였다.</p>
<h2><span class="mw-headline" id=".EC.83.9D.EC.95.A0">생애</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%95%8C%EB%B0%98_%EB%B2%A0%EB%A5%B4%ED%81%AC&amp;action=edit&amp;section=1" title="부분 편집: 생애">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>베르크는 <a href="/wiki/%EB%B9%88" title="빈">빈</a>에서 요한나와 콘래드 베르크의 사이에서 네 아이 중 세 번째로 태어났다. 그의 가족은 1900년 그의 아버지가 죽을 때까지 풍족하게 살았다. 베르크는 어린 시절 음악보다는 <a href="/wiki/%EB%AC%B8%ED%95%99" title="문학">문학</a>에 더 관심이 많아서, 그가 스스로 음악을 익히기 시작한 15살이 될 때까지 작곡을 하지 않았다. 그는 <a href="/wiki/%EC%95%84%EB%A5%B4%EB%86%80%ED%8A%B8_%EC%87%A4%EB%B2%A0%EB%A5%B4%ED%81%AC" title="아르놀트 쇤베르크">아르놀트 쇤베르크</a>에게서 6년간 공부를 할 때까지는 정규 음악 교육을 거의 받지 않았다. 베르크는 <a href="/wiki/%EB%8C%80%EC%9C%84%EB%B2%95" title="대위법">대위법</a>과 <a href="/wiki/%EC%9D%8C%EC%95%85_%EC%9D%B4%EB%A1%A0" title="음악 이론">음악 이론</a>, <a class="mw-redirect" href="/wiki/%ED%99%94%EC%84%B1%ED%95%99" title="화성학">화성학</a>을 1906년까지 배웠고, 1907년까지는 음악 공부에 전념한 다음, 작곡 교습을 시작하였다.</p>
<p>쇤베르크 지도하의 베르크의 작곡은 5개의 <a class="new" href="/w/index.php?title=%ED%94%BC%EC%95%84%EB%82%98_%EC%86%8C%EB%82%98%ED%83%80&amp;action=edit&amp;redlink=1" title="피아나 소나타 (없는 문서)">피아나 소나타</a> <a class="mw-disambig" href="/wiki/%EC%B4%88%EA%B3%A0" title="초고">초고</a>과 다양한 노래들로 《<a class="new" href="/w/index.php?title=7%EA%B0%9C%EC%9D%98_%EC%B4%88%EA%B8%B0_%EB%85%B8%EB%9E%98%EB%93%A4&amp;action=edit&amp;redlink=1" title="7개의 초기 노래들 (없는 문서)">7개의 초기 노래들</a>》(Sieben frühe Lieder)을 포함한다. 그중의 3개는 작곡된 같은 해에 <a href="/wiki/%EB%B9%88" title="빈">빈</a>에서 쇤베르크의 제자들의 음악을 주연으로 삼는 <a href="/wiki/%EC%97%B0%EC%A3%BC%ED%9A%8C" title="연주회">연주회</a>에서 처음으로 대중을 향해 연주되었다.</p>
<p>이러한 초기 작품은 쇤베르크의 지도하에 베르크의 작곡가로서의 진행을 드러내었다. 이 초기의 소나타 초고는 결국 베르크의 <a class="new" href="/w/index.php?title=%EC%86%8C%EB%82%98%ED%83%80,_%EC%9E%91%ED%92%88_1%EB%B2%88_(%EB%B2%A0%EB%A5%B4%ED%81%AC)&amp;action=edit&amp;redlink=1" title="소나타, 작품 1번 (베르크) (없는 문서)">소나타 작품 1번</a>으로 정점에 다다랐다. 비록 그의 졸업작곡으로 여겨졌지만, 그 작품은 작곡가들의 가공할 만한 초기 작품 중 하나이다. 쇤베르크는 베르크의 일생동안 중요한 영향을 끼쳤다. 베르크는 쇤베르크를 작곡가와 <a class="new" href="/w/index.php?title=%EC%A0%95%EC%8B%A0%EC%A0%81_%EC%A7%80%EC%A3%BC&amp;action=edit&amp;redlink=1" title="정신적 지주 (없는 문서)">정신적 지주</a>로서 크게 존경했을 뿐만 아니라, 그의 나머지 일생에서 가까운 친구로 남았다. 많은이는 베르크가 그의 아버지의 죽음 동안, 베르그의 어린 젊은시기를 돌봐준 쇤베르크를 대리 아버지로 보았다고 생각했다.</p>
<p>쇤베르크가 그의 가르침에서 중요하게 사용된 아이디어는 뒤에 <b>변주곡으로 발전하기</b>로 알려지게 되었고, 이것은 한 작품의 <a class="new" href="/w/index.php?title=%ED%86%B5%EC%9D%BC%EC%84%B1&amp;action=edit&amp;redlink=1" title="통일성 (없는 문서)">통일성</a>은 한가지 기본적인 아이디어에서부터 도래된 작곡의 모든 면에 의지한다고 말했다. 베르크는 이 생각을 그의 학생들에게 전달했고, <a class="mw-redirect" href="/wiki/%ED%85%8C%EC%98%A4%EB%8F%84%EB%A5%B4_%EC%95%84%EB%8F%84%EB%A5%B4%EB%85%B8" title="테오도르 아도르노">테오도르 아도르노</a>는 베르크가 말하고자하는 중심 원칙은 <i><a class="mw-redirect" href="/wiki/%EB%B3%80%EC%A3%BC" title="변주">변주</a></i>라고 말한다. 모든 것은 기타 다른 것에서 발전되고 그것은 내재적으로 다른 것이라 가정된다. 그 소나타 작품은 이 생각의 실행의 놀랄 만한 예제이다. 전체 작곡은 처음 4개의 음 모습과 시작하는 구절로부터 유래될 수 있다.</p>
<p><br/>
베르크는 세기말 격렬한 시기 동안 <a href="/wiki/%EB%B9%88" title="빈">빈</a> 음악적 지식인의 일부였다. 그의 구성원 중에서는 음악가인 <a class="new" href="/w/index.php?title=%EC%95%8C%EB%A0%89%EC%82%B0%EB%8D%94_%ED%8F%B0_%EC%A0%AC%EB%A6%B0%EC%8A%A4%ED%82%A4&amp;action=edit&amp;redlink=1" title="알렉산더 폰 젬린스키 (없는 문서)">알렉산더 폰 젬린스키</a>와 <a href="/wiki/%ED%94%84%EB%9E%80%EC%B8%A0_%EC%8A%88%EB%A0%88%EC%BB%A4" title="프란츠 슈레커">프란츠 슈레커</a>와 화가인 <a href="/wiki/%EA%B5%AC%EC%8A%A4%ED%83%80%ED%94%84_%ED%81%B4%EB%A6%BC%ED%8A%B8" title="구스타프 클림트">구스타프 클림트</a>, 작가이자 <a class="mw-redirect" href="/wiki/%ED%92%8D%EC%9E%90%EA%B0%80" title="풍자가">풍자가</a>인 <a class="new" href="/w/index.php?title=%EC%B9%BC_%ED%81%AC%EB%9D%BC%EC%9A%B0%EC%8A%A4&amp;action=edit&amp;redlink=1" title="칼 크라우스 (없는 문서)">칼 크라우스</a>, <a href="/wiki/%EA%B1%B4%EC%B6%95%EA%B0%80" title="건축가">건축가</a>인 <a class="new" href="/w/index.php?title=%EC%95%84%EB%8F%8C%ED%94%84_%EB%A3%A8%EC%8A%A4&amp;action=edit&amp;redlink=1" title="아돌프 루스 (없는 문서)">아돌프 루스</a>와 <a href="/wiki/%EC%8B%9C%EC%9D%B8" title="시인">시인</a>인 <a class="new" href="/w/index.php?title=%ED%8E%98%ED%84%B0_%EC%95%8C%ED%85%90%EB%B2%A0%EB%A5%B4%ED%81%AC&amp;action=edit&amp;redlink=1" title="페터 알텐베르크 (없는 문서)">페터 알텐베르크</a>를 포함한다. 1906년 베르크는 가수이자 부유한 집안의 영양인 헬레네 나호프스키를 만났다. 비록 그녀의 집안의 외적인 반대에도 불구하고, 이 둘은 1911년 5월 3일에 결혼했다.</p>
<p>1913년 베르크의 《페터 알텐베르크의 글이 적힌 사진 <a class="new" href="/w/index.php?title=%EC%9A%B0%ED%8E%B8%EC%B9%B4%EB%93%9C&amp;action=edit&amp;redlink=1" title="우편카드 (없는 문서)">우편카드</a>에 관한 다섯개의 노래》 중 두개가 <a href="/wiki/%EB%B9%88" title="빈">빈</a>에서 쇤베르크의 지휘하에 <a href="/wiki/%EC%B4%88%EC%97%B0" title="초연">초연</a>되었다. <a href="/wiki/%EC%82%B0%EB%AC%B8" title="산문">산문</a>적이고 <a class="mw-redirect" href="/wiki/%EA%B2%BD%EA%B5%AC" title="경구">경구</a>의 <a class="new" href="/w/index.php?title=%EB%B0%9C%EC%84%B1&amp;action=edit&amp;redlink=1" title="발성 (없는 문서)">발성</a>이 매우 대규모의 <a href="/wiki/%EA%B4%80%ED%98%84%EC%95%85" title="관현악">관현악</a>과 함께 수반된 이 작품은 <a href="/wiki/%ED%8F%AD%EB%8F%99" title="폭동">폭동</a>을 일으켜서, 그 공연은 중지되어야 했다. 이 작품은 1952년까지 전체가 공연되지 못했다.(1966년까지 전체 악보가 발간되지 않았다)</p>
<p>1915년부터 1918년까지 베르크는 오스트리아 군대에 복무했고, 1917년이 거의 끝날 무렵, 그는 첫 번째 오페라인 《<a href="/wiki/%EB%B3%B4%EC%B2%B4%ED%81%AC" title="보체크">보체크</a>》를 작곡하기 시작했다. <a href="/wiki/%EC%A0%9C1%EC%B0%A8_%EC%84%B8%EA%B3%84_%EB%8C%80%EC%A0%84" title="제1차 세계 대전">제1차 세계 대전</a> 후에 베르크는 다시 <a href="/wiki/%EB%B9%88" title="빈">빈</a>에 정착하여, 개인적으로 제자들을 가르쳤다. 베르크는 역시 쇤베르크가 <a class="new" href="/w/index.php?title=%EC%9D%8C%EC%95%85%EC%A0%81_%EA%B0%9C%EC%9D%B8_%EC%97%B0%EC%A3%BC%EB%93%A4%EC%9D%98_%EC%A1%B0%ED%95%A9&amp;action=edit&amp;redlink=1" title="음악적 개인 연주들의 조합 (없는 문서)">음악적 개인 연주들의 조합</a>을 운영하는 것을 도왔다. 그 조합은 열린 <a href="/wiki/%EB%A6%AC%ED%97%88%EC%84%A4" title="리허설">리허설</a>과 반복적인 공연과 모든 신문 <a class="mw-redirect" href="/wiki/%EB%B9%84%ED%8F%89" title="비평">비평</a>을 제외한 채로, 예상치못한 낯선 새 음악의 <a href="/wiki/%EC%8B%A4%ED%97%98" title="실험">실험</a>을 위해 이상적인 환경을 창조하려 했다.</p>
<p>1924년 《<a href="/wiki/%EB%B3%B4%EC%B2%B4%ED%81%AC" title="보체크">보체크</a>》에서 세부분을 발췌한 공연은 베르크에게 첫 번째 대중적인 성공을 가져왔다. 베르크가 1922년에 완성한 그 오페라는 <a href="/wiki/%EC%97%90%EB%A6%AC%ED%9E%88_%ED%81%B4%EB%9D%BC%EC%9D%B4%EB%B2%84" title="에리히 클라이버">에리히 클라이버</a>가 <a href="/wiki/%EB%B2%A0%EB%A5%BC%EB%A6%B0" title="베를린">베를린</a>에서 지휘한 1925년 12월 14일에서야 전체가 상연되었다. 《<a href="/wiki/%EB%B3%B4%EC%B2%B4%ED%81%AC" title="보체크">보체크</a>》는 오늘날 그의 가장 중요한 작품 중 하나가 되었다. 후기 오페라인 <a href="/wiki/%EB%A3%B0%EB%A3%A8_(%EC%98%A4%ED%8E%98%EB%9D%BC)" title="룰루 (오페라)">룰루</a>는 비평적으로 칭송을 얻었고, 그의 죽음으로 3악장이 미완성된 채로 남아있다.</p>
<p>베르크의 가장 잘 알려진 작품은 아마도 애잔한 《<a class="new" href="/w/index.php?title=%EB%B0%94%EC%9D%B4%EC%98%AC%EB%A6%B0_%ED%98%91%EC%A3%BC%EA%B3%A1_(%EB%B2%A0%EB%A5%B4%ED%81%AC)&amp;action=edit&amp;redlink=1" title="바이올린 협주곡 (베르크) (없는 문서)">바이올린 협주곡</a>》이다. 그의 많은 성숙한 작품처럼, 쇤베르크의 <a class="new" href="/w/index.php?title=12%ED%91%9C%EC%9D%8C&amp;action=edit&amp;redlink=1" title="12표음 (없는 문서)">12표음</a> 기법을 지극히 개인적으로 적용시켜서, 명백한 <a href="/wiki/%EB%AC%B4%EC%A1%B0%EC%84%B1" title="무조성">무조성</a>과 더 전통적인 <a href="/wiki/%EC%A1%B0%EC%84%B1_(%EC%9D%8C%EC%95%85)" title="조성 (음악)">조성</a> <a class="new" href="/w/index.php?title=%EC%95%85%EC%A0%88&amp;action=edit&amp;redlink=1" title="악절 (없는 문서)">악절</a>와 <a href="/wiki/%ED%99%94%EC%84%B1" title="화성">화성</a>을 혼합시킬 수 있었다. 거기다 이 작품은 바흐의 <a class="mw-redirect" href="/wiki/%ED%95%A9%EC%B0%BD%EA%B3%A1" title="합창곡">합창곡</a>과 오스트리아 <a class="new" href="/w/index.php?title=%EC%BE%8C%EB%A5%B8%ED%85%90&amp;action=edit&amp;redlink=1" title="쾌른텐 (없는 문서)">쾌른텐</a> 지역의 <a class="new" href="/w/index.php?title=%EB%AF%BC%EC%86%8D_%EB%85%B8%EB%9E%98&amp;action=edit&amp;redlink=1" title="민속 노래 (없는 문서)">민속 노래</a>를 포함하여, 이전에 존재하는 <a class="new" href="/w/index.php?title=%EC%A1%B0%EC%84%B1_%EC%9D%8C%EC%95%85&amp;action=edit&amp;redlink=1" title="조성 음악 (없는 문서)">조성 음악</a>을 실제로 인용하였다. 그 <a class="new" href="/w/index.php?title=%EB%B0%94%EC%9D%B4%EC%98%AC%EB%A6%B0_%ED%98%91%EC%A3%BC%EA%B3%A1_(%EB%B2%A0%EB%A5%B4%ED%81%AC)&amp;action=edit&amp;redlink=1" title="바이올린 협주곡 (베르크) (없는 문서)">바이올린 협주곡</a>은 <a href="/wiki/%EB%B0%9C%ED%84%B0_%EA%B7%B8%EB%A1%9C%ED%94%BC%EC%9A%B0%EC%8A%A4" title="발터 그로피우스">발터 그로피우스</a>와 <a class="new" href="/w/index.php?title=%EC%95%8C%EB%A7%88_%EC%89%B0%EB%93%A4%EB%9F%AC&amp;action=edit&amp;redlink=1" title="알마 쉰들러 (없는 문서)">알마 쉰들러</a>의 작고한 딸인 마농에게 헌정되었다.</p>
<p>베르크의 다른 유명한 작품은 《<a class="new" href="/w/index.php?title=%EC%84%9C%EC%A0%95_%EB%AA%A8%EC%9D%8C%EA%B3%A1&amp;action=edit&amp;redlink=1" title="서정 모음곡 (없는 문서)">서정 모음곡</a>》(<a class="new" href="/w/index.php?title=%EB%B2%A8%EB%9D%BC_%EB%B2%84%EB%A5%B4%ED%86%A0%ED%81%AC&amp;action=edit&amp;redlink=1" title="벨라 버르토크 (없는 문서)">벨라 버르토크</a>의 <a class="new" href="/w/index.php?title=%ED%98%84%EC%95%85_%EC%82%AC%EC%A4%91%EC%A3%BC_3%EB%B2%88_(%EB%B2%84%EB%A5%B4%ED%86%A0%ED%81%AC)&amp;action=edit&amp;redlink=1" title="현악 사중주 3번 (버르토크) (없는 문서)">현악 사중주 3번</a>에 큰 영향을 준 것으로 보인다)과 관현악을 위한 3개 작품과 바이올린, 피아노, 13 <a href="/wiki/%EA%B4%80%EC%95%85%EA%B8%B0" title="관악기">관악기</a>를 위한 <a class="new" href="/w/index.php?title=%EC%8B%A4%EB%82%B4_%ED%98%91%EC%A3%BC%EA%B3%A1&amp;action=edit&amp;redlink=1" title="실내 협주곡 (없는 문서)">실내 협주곡</a>이 있다.</p>
<p>베르크는 <a href="/wiki/%ED%81%AC%EB%A6%AC%EC%8A%A4%EB%A7%88%EC%8A%A4" title="크리스마스">크리스마스</a> <a href="/wiki/%ED%81%AC%EB%A6%AC%EC%8A%A4%EB%A7%88%EC%8A%A4_%EC%9D%B4%EB%B8%8C" title="크리스마스 이브">이브</a>에 빈에서 벌레에게 물려서 야기된 <a href="/wiki/%ED%8C%A8%ED%98%88%EC%A6%9D" title="패혈증">패혈증</a>에 의해 사망했다. 그 당시 50살이었다. 20세기가 끝난 후,회고적인 성향이라 일컬어지던 베르크는 어느 새 <a href="/wiki/%EC%9E%91%EA%B3%A1%EA%B0%80" title="작곡가">작곡가</a>이자 <a class="new" href="/w/index.php?title=%EC%9D%8C%EC%95%85_%EC%9D%B4%EB%A1%A0%EA%B0%80&amp;action=edit&amp;redlink=1" title="음악 이론가 (없는 문서)">음악 이론가</a>인 조지 펄(George Perle)이 주목했듯 최고로 진보 성향인 작곡가로 주목받게 되었다.</p>
