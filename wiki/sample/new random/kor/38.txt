<p><b>고대 노르드어</b>(古代──語, <span style="font-size: smaller;"><a href="/wiki/%EB%9D%BC%ED%8B%B4%EC%96%B4" title="라틴어">라틴어</a>:</span> <span lang="la" xml:lang="la">Nordica antiqua</span>, <span style="font-size: smaller;"><a href="/wiki/%EB%85%B8%EB%A5%B4%EC%9B%A8%EC%9D%B4%EC%96%B4" title="노르웨이어">노르웨이어</a>:</span> <span lang="no" xml:lang="no">Norrønt</span>, <span style="font-size: smaller;"><a href="/wiki/%EC%95%84%EC%9D%B4%EC%8A%AC%EB%9E%80%EB%93%9C%EC%96%B4" title="아이슬란드어">아이슬란드어</a>:</span> <span lang="is" xml:lang="is">norrœnt mál</span>, <span style="font-size: smaller;"><a href="/wiki/%EC%8A%A4%EC%9B%A8%EB%8D%B4%EC%96%B4" title="스웨덴어">스웨덴어</a>:</span> <span lang="sv" xml:lang="sv">Fornnordiska</span>, <span style="font-size: smaller;"><a href="/wiki/%EB%8D%B4%EB%A7%88%ED%81%AC%EC%96%B4" title="덴마크어">덴마크어</a>:</span> <span lang="da" xml:lang="da">Norrønt</span>, <span style="font-size: smaller;"><a href="/wiki/%EB%8F%85%EC%9D%BC%EC%96%B4" title="독일어">독일어</a>:</span> <span lang="de" xml:lang="de">Altnordische</span>, <span style="font-size: smaller;"><a href="/wiki/%EC%98%81%EC%96%B4" title="영어">영어</a>:</span> <span lang="en" xml:lang="en">Old Norse</span>)는 <a href="/wiki/%EB%B6%81%EA%B2%8C%EB%A5%B4%EB%A7%8C%EC%96%B4%EA%B5%B0" title="북게르만어군">북게르만어군</a>에 속하는 언어로서, 9세기에서 13세기에 걸쳐 <a href="/wiki/%EC%8A%A4%EC%B9%B8%EB%94%94%EB%82%98%EB%B9%84%EC%95%84" title="스칸디나비아">스칸디나비아</a> 및 <a href="/wiki/%EB%B0%94%EC%9D%B4%ED%82%B9" title="바이킹">바이킹</a>들의 해상거주지에서 사용된 언어이다.</p>
<p>8세기경 <a class="new" href="/w/index.php?title=%EB%85%B8%EB%A5%B4%EB%93%9C_%EC%A1%B0%EC%96%B4&amp;action=edit&amp;redlink=1" title="노르드 조어 (없는 문서)">노르드 조어</a>가 노르드어로 진화했고, 노르드어는 14세기 중후반을 거치며 현대 북게르만어로 분화하면서 사멸했다. 다만 이 시기는 절대적인 것이 아니며, 노르드어로 쓰인 문헌은 늦으면 15세기에도 발견된다.<sup class="reference" id="cite_ref-1"><a href="#cite_note-1">[1]</a></sup></p>
<p>노르드어는 세 가지 방언으로 나뉜다. 동노르드어, 서노르드어, <a class="new" href="/w/index.php?title=%EA%B3%A0%EB%8C%80_%EA%B3%A0%ED%8B%80%EB%9E%80%EB%93%9C%EC%96%B4&amp;action=edit&amp;redlink=1" title="고대 고틀란드어 (없는 문서)">고틀란드어</a>가 그것이다. 동노르드어와 서노르드어는 <a class="new" href="/w/index.php?title=%EB%B0%A9%EC%96%B8%EC%97%B0%EC%86%8D%EC%B2%B4&amp;action=edit&amp;redlink=1" title="방언연속체 (없는 문서)">방언연속체</a>를 형성하여 명확한 지리적 경계가 존재하지 않는다. 예컨대 동부 <a href="/wiki/%EB%85%B8%EB%A5%B4%EC%9B%A8%EC%9D%B4" title="노르웨이">노르웨이</a>에서 동노르드어의 흔적이 발견되고, 서부 <a href="/wiki/%EC%8A%A4%EC%9B%A8%EB%8D%B4" title="스웨덴">스웨덴</a>에서 서노르드어의 흔적이 발견되는 식이다. 고대 서노르드어 사용자들은 오늘날의 노르웨이에 많이 살았고, 동노르드어 사용자들은 오늘날의 덴마크, 스웨덴, 고틀란드에 많이 살았다.</p>
<p>12세기 아이슬란드의 《<a href="/wiki/%EA%B7%B8%EB%9D%BC%EA%B0%80%EC%8A%A4" title="그라가스">그라가스</a>》는 스웨덴인, 노르웨이인, 아이슬란드인, 데인인이 모두 같은 언어를 쓴다고 말하고 있으며, 그 언어를 <b>데인어</b>(<span lang="non" xml:lang="non">dǫnsk tunga</span> <small>돈스크 퉁가</small>)라고 불렀다. 또 특히 서노르드어를 가리키는 다른 표현으로는 <b>노르딕 말</b>(<span lang="non" xml:lang="non">norrœnt mál</span> <small>노르뢴트 말</small>)이 있었다. 오늘날 노르드어는 북게르만어로 분화하여 <a href="/wiki/%EC%95%84%EC%9D%B4%EC%8A%AC%EB%9E%80%EB%93%9C%EC%96%B4" title="아이슬란드어">아이슬란드어</a>, <a href="/wiki/%ED%8E%98%EB%A1%9C%EC%96%B4" title="페로어">페로어</a>, <a href="/wiki/%EB%85%B8%EB%A5%B4%EC%9B%A8%EC%9D%B4%EC%96%B4" title="노르웨이어">노르웨이어</a>, <a href="/wiki/%EB%8D%B4%EB%A7%88%ED%81%AC%EC%96%B4" title="덴마크어">덴마크어</a>, <a href="/wiki/%EC%8A%A4%EC%9B%A8%EB%8D%B4%EC%96%B4" title="스웨덴어">스웨덴어</a>를 형성하였으며, 이 중 노르웨이, 덴마크, 스웨덴은 여전히 상호 의사소통이 가능하다.</p>

<h2><span class="mw-headline" id=".EB.B6.84.ED.8F.AC">분포</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EA%B3%A0%EB%8C%80_%EB%85%B8%EB%A5%B4%EB%93%9C%EC%96%B4&amp;action=edit&amp;section=1" title="부분 편집: 분포">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>고대 아이슬란드어는 본래 <a href="/wiki/%EA%B3%A0%EB%8C%80_%EB%85%B8%EB%A5%B4%EC%9B%A8%EC%9D%B4%EC%96%B4" title="고대 노르웨이어">고대 노르웨이어</a>와 기본적으로 동일했으며, 함께 고대 서노르드어를 이루었다. <a href="/wiki/%EC%95%84%EC%9D%BC%EB%9E%9C%EB%93%9C" title="아일랜드">아일랜드</a>, <a href="/wiki/%EC%8A%A4%EC%BD%94%ED%8B%80%EB%9E%9C%EB%93%9C" title="스코틀랜드">스코틀랜드</a>, <a href="/wiki/%EB%A7%A8_%EC%84%AC" title="맨 섬">맨 섬</a>, <a href="/wiki/%EC%9E%89%EA%B8%80%EB%9E%9C%EB%93%9C" title="잉글랜드">잉글랜드</a> 북서부에 세워진 바이킹 정착지들과 <a href="/wiki/%EB%85%B8%EB%A5%B4%EB%A7%9D%EB%94%94" title="노르망디">노르망디</a>의 노르웨이계 바이킹들이 서노르드어를 사용했다.<sup class="reference" id="cite_ref-JohnsonEcyc_2-0"><a href="#cite_note-JohnsonEcyc-2">[2]</a></sup> 덴마크, 스웨덴과 <a class="mw-redirect" href="/wiki/%ED%82%A4%EC%98%88%ED%94%84_%EB%A3%A8%EC%8B%9C" title="키예프 루시">키예프 루시</a> 쪽으로 진출한 바이킹 정착지들과<sup class="reference" id="cite_ref-Nationalencyklopedin_3-0"><a href="#cite_note-Nationalencyklopedin-3">[3]</a></sup> 잉글랜드 동부, 그리고 노르망디의 덴마크계 바이킹들이 동노르드어를 사용했다. <a href="/wiki/%EA%B3%A0%ED%8B%80%EB%9E%80%EB%93%9C_%EC%84%AC" title="고틀란드 섬">고틀란드 섬</a>과 동유럽의 여러 정착지에서 <a class="new" href="/w/index.php?title=%EA%B3%A0%EB%8C%80_%EA%B3%A0%ED%8B%80%EB%9E%80%EB%93%9C%EC%96%B4&amp;action=edit&amp;redlink=1" title="고대 고틀란드어 (없는 문서)">고대 고틀란드어</a>가 사용되었다. 11세기가 되면 노르드어는 서쪽으로는 <a href="/wiki/%EB%B9%88%EB%9E%80%EB%93%9C" title="빈란드">빈란드</a>에서 동쪽으로는 <a href="/wiki/%EB%B3%BC%EA%B0%80_%EA%B0%95" title="볼가 강">볼가 강</a>에 이르기까지 유럽에서 가장 넓은 지역에서 사용되는 언어가 되었다. 키예프 루시에서는 <a href="/wiki/%EB%85%B8%EB%B8%8C%EA%B3%A0%EB%A1%9C%EB%93%9C_%EA%B3%B5%ED%99%94%EA%B5%AD" title="노브고로드 공화국">노브고로드</a>에서 노르드어가 가장 오래 살아남아 최소 13세기까지는 사용되었다.<sup class="reference" id="cite_ref-Nationalencyklopedin_3-1"><a href="#cite_note-Nationalencyklopedin-3">[3]</a></sup></p>

<p>서노르드어의 후계 언어는 <a href="/wiki/%EC%95%84%EC%9D%B4%EC%8A%AC%EB%9E%80%EB%93%9C%EC%96%B4" title="아이슬란드어">아이슬란드어</a>, <a href="/wiki/%ED%8E%98%EB%A1%9C%EC%96%B4" title="페로어">페로어</a>, <a href="/wiki/%EB%85%B8%EB%A5%B4%EC%9B%A8%EC%9D%B4%EC%96%B4" title="노르웨이어">노르웨이어</a>, 그리고 사멸한 <a href="/wiki/%EB%85%B8%EB%A5%B8%EC%96%B4" title="노른어">노른어</a> 등의 서스칸디나비아어다. 한편 동노르드어의 후계 언어는 <a href="/wiki/%EB%8D%B4%EB%A7%88%ED%81%AC%EC%96%B4" title="덴마크어">덴마크어</a>, <a href="/wiki/%EC%8A%A4%EC%9B%A8%EB%8D%B4%EC%96%B4" title="스웨덴어">스웨덴어</a>로 이루어진 동스칸디나비아어다. 노르웨이어는 본래 서노르드어 계통이지만 세월이 지나면서 동노르드어의 영향을 심하게 받게 되었는데, <a href="/wiki/%EB%8D%B4%EB%A7%88%ED%81%AC-%EB%85%B8%EB%A5%B4%EC%9B%A8%EC%9D%B4" title="덴마크-노르웨이">덴마크-노르웨이</a> 동군연합 때 특히 그러했다.</p>
<p>이들 언어 중 아이슬란드어와 페로어가 1천여년 동안 노르드어의 원형이 가장 잘 보존된 언어로 꼽힌다. 덴마크가 페로 제도를 지배하기는 했으나 페로어가 역으로 덴마크어에 영향을 미치기도 했다. 노르드어는 잉글랜드의 <a href="/wiki/%EC%98%81%EC%96%B4" title="영어">영어</a> 및 저지 스코틀랜드의 <a href="/wiki/%EC%8A%A4%EC%BD%94%ED%8A%B8%EC%96%B4" title="스코트어">스코트어</a>에도 영향을 미쳐서, 영어와 스코트어에는 노르드어 <a class="new" href="/w/index.php?title=%EC%99%9C%EB%9E%98%EC%96%B4&amp;action=edit&amp;redlink=1" title="왜래어 (없는 문서)">왜래어</a>가 많이 존재한다. 또 <a class="new" href="/w/index.php?title=%EB%85%B8%EB%A5%B4%EB%A7%9D%EB%94%94%EC%96%B4&amp;action=edit&amp;redlink=1" title="노르망디어 (없는 문서)">노르망디어</a>의 형성과 현대 <a href="/wiki/%ED%94%84%EB%9E%91%EC%8A%A4%EC%96%B4" title="프랑스어">프랑스어</a>에도 영향을 미쳤다.</p>
<p>노르드어와 같은 계통은 아니지만 노르드어, 특히 노르망디어에 강한 영향을 받은 언어로 <a href="/wiki/%EC%8A%A4%EC%BD%94%ED%8B%80%EB%9E%9C%EB%93%9C_%EA%B2%8C%EC%9D%BC%EC%96%B4" title="스코틀랜드 게일어">스코틀랜드 게일어</a>, <a href="/wiki/%EC%95%84%EC%9D%BC%EB%9E%9C%EB%93%9C%EC%96%B4" title="아일랜드어">아일랜드어</a>, <a href="/wiki/%EB%9F%AC%EC%8B%9C%EC%95%84%EC%96%B4" title="러시아어">러시아어</a>, <a class="mw-redirect" href="/wiki/%EB%B2%A8%EB%A1%9C%EB%A3%A8%EC%8B%9C%EC%96%B4" title="벨로루시어">벨로루시어</a>, <a href="/wiki/%EB%A6%AC%ED%88%AC%EC%95%84%EB%8B%88%EC%95%84%EC%96%B4" title="리투아니아어">리투아니아어</a>, <a href="/wiki/%ED%95%80%EB%9E%80%EB%93%9C%EC%96%B4" title="핀란드어">핀란드어</a>, <a href="/wiki/%EB%9D%BC%ED%8A%B8%EB%B9%84%EC%95%84%EC%96%B4" title="라트비아어">라트비아어</a>, <a href="/wiki/%EC%97%90%EC%8A%A4%ED%86%A0%EB%8B%88%EC%95%84%EC%96%B4" title="에스토니아어">에스토니아어</a> 등이 있으며 이들은 상당한 수의 노르드어 외래어를 거지고 있다. 혹설에서는 "러시아"의 어원이 된 "루스" 자체가 <a href="/wiki/%EB%85%B8%EB%A5%B4%EB%93%9C%EC%9D%B8" title="노르드인">노르드인</a>의 일파인 <a href="/wiki/%EB%A3%A8%EC%8A%A4%EC%9D%B8" title="루스인">루스인</a>에서 비롯된 것이라고 보기도 한다. 핀란드와 에스토니아에서 오늘날 스웨덴을 가리키는 말이 각각 "루오트시(<span lang="fi" xml:lang="fi">Ruotsi</span>), "루트시(<span lang="et" xml:lang="et">Rootsi</span>)라는 점이 이를 뒷받침한다.</p>
<p>상술했듯 현대에 남아있는 언어들 중 옛 노르드어와 가장 가까운 언어는 아이슬란드어이다. 현대 아이슬란드어의 쓰기체계는 노르드어 <a class="mw-redirect" href="/wiki/%EC%9D%8C%EC%86%8C" title="음소">음소</a>체계에서 파생된 것이며, 현대 아이슬란드어 구사자들은 철자에 다소의 차이가 있지만 노르드어를 대체로 읽을 수 있다. 그러나 발음, 특히 홑소리 발음은 다른 북게르만어가 노르드어와 달라진 만큼 아이슬란드어도 많이 변화했다.</p>
<p>페로어도 노르드어와 유사한 점이 많이 남아 있지만 페로어는 덴마크어, 노르웨이어, <a href="/wiki/%EA%B3%A0%EC%9D%B4%EB%8D%B8%EC%96%B4" title="고이델어">고이델어</a>(<a href="/wiki/%EC%8A%A4%EC%BD%94%ED%8B%80%EB%9E%9C%EB%93%9C_%EA%B2%8C%EC%9D%BC%EC%96%B4" title="스코틀랜드 게일어">스코틀랜드</a> 및 <a href="/wiki/%EC%95%84%EC%9D%BC%EB%9E%9C%EB%93%9C%EC%96%B4" title="아일랜드어">아일랜드어</a>)에 영향을 받았다.<sup class="reference" id="cite_ref-4"><a href="#cite_note-4">[4]</a></sup> 스웨덴어, 덴마크어, 노르웨이어는 옛 노르드어에서 파생, 각자 많은 변화를 거쳤지만 그래도 비대칭적으로 <a href="/wiki/%EC%83%81%ED%98%B8_%EC%9D%98%EC%82%AC_%EC%86%8C%ED%86%B5%EC%84%B1" title="상호 의사 소통성">상호 의사소통</a>이 가능하다.<sup class="reference" id="cite_ref-5"><a href="#cite_note-5">[5]</a></sup> 현대 스웨덴어, 노르웨이어, 덴마크어 사용자들은 서로의 말을 따로 공부하지 않아도 어느 정도 이해할 수 있으며, 느리게 말하면 거의 알아듣는다. 이 언어들은 쓰기체계도 충분히 유사한데, 서로 국경을 맞대고 있는 탓이 클 것이다. 이 세 언어는 서로서로 영향을 주고받으며 발달했고, <a class="new" href="/w/index.php?title=%EC%A4%91%EC%84%B8_%EC%A0%80%EC%A7%80%EB%8F%85%EC%9D%BC%EC%96%B4&amp;action=edit&amp;redlink=1" title="중세 저지독일어 (없는 문서)">중세 저지독일어</a>에게도 유사한 영향을 받았다.<sup class="reference" id="cite_ref-6"><a href="#cite_note-6">[6]</a></sup></p>
<h2><span class="mw-headline" id=".EB.85.B8.EB.A5.B4.EB.93.9C.EC.96.B4_.EC.9D.8C.EC.9A.B4.EB.A1.A0">노르드어 음운론</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EA%B3%A0%EB%8C%80_%EB%85%B8%EB%A5%B4%EB%93%9C%EC%96%B4&amp;action=edit&amp;section=3" title="부분 편집: 노르드어 음운론">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<h3><span class="mw-headline" id=".ED.99.91.EC.86.8C.EB.A6.AC">홑소리</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EA%B3%A0%EB%8C%80_%EB%85%B8%EB%A5%B4%EB%93%9C%EC%96%B4&amp;action=edit&amp;section=4" title="부분 편집: 홑소리">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>노르드어 홑소리 음소들은 대부분 장모음과 단모음의 짝을 이루고 있다. <a class="new" href="/w/index.php?title=%EA%B3%A0%EB%8C%80_%EB%85%B8%EB%A5%B4%EB%93%9C%EC%96%B4_%EC%B2%A0%EC%9E%90%EB%B2%95&amp;action=edit&amp;redlink=1" title="고대 노르드어 철자법 (없는 문서)">표준 철자법</a>은 장모음에 예음부호 ´를 찍어서 장모음임을 표시한다. 중세 문헌들에서는 이러한 표기가 잘 이루어져 있지 않지만 간혹 부호 표시나 중복표시로 표현하기도 한다.</p>
<p>노르드어는 9개 홑소리 위치가 모두 비음화될 수 있다.<sup class="reference" id="cite_ref-CleasbyA_7-0"><a href="#cite_note-CleasbyA-7">[V 1]</a></sup> 홑소리의 이음이 앞에 비음 닿소리가 위치하면서, 근처 소리에 묻히지 않을 때 홑소리의 비음화가 일어난다. 홑소리에 실린 강세 때문에 비음이 묻혀 버릴 경우, 그 홑소리의 길이가 늘어진다. 이러한 비음화는 다른 게르만 계통 언어들에서도 나타나지만, 그렇게 길게 지속되지는 않는다. 이러한 사항들은 12세기에 쓰여진 《<a class="new" href="/w/index.php?title=%EC%B2%AB_%EB%B2%88%EC%A7%B8_%EB%AC%B8%EB%B2%95%EC%A1%B0%EC%95%BD&amp;action=edit&amp;redlink=1" title="첫 번째 문법조약 (없는 문서)">첫 번째 문법조약</a>》에 나와 있기에 알 수 있는 것이며, 만일 여기 보존되어 있지 않았다면 알려지지 못했을 것이다. 《첫 번째 문법조약》에서는 글자 위에 점 하나를 찍어서 비음화를 표시하도록 하고 있다.<sup class="reference" id="cite_ref-CleasbyA_7-1"><a href="#cite_note-CleasbyA-7">[V 1]</a></sup> 그러나 이 표시법은 그다지 인기가 없었고 곧 무용지불이 되었다. 동노르드어에서는 11세기를 전후하여 비모음과 구모음이 대부분 융합된 것으로 생각된다.<sup class="reference" id="cite_ref-NordicLanguages202_8-0"><a href="#cite_note-NordicLanguages202-8">[7]</a></sup><sup>:3</sup> 그러나 <a class="new" href="/w/index.php?title=%EB%8B%AC%EB%9D%BC%EB%A5%B4%EB%82%98_%EB%B0%A9%EC%96%B8&amp;action=edit&amp;redlink=1" title="달라르나 방언 (없는 문서)">달라르나 방언</a>에서는 아직 구분이 이루어지고 있다.<sup class="reference" id="cite_ref-NordicLanguages202_8-1"><a href="#cite_note-NordicLanguages202-8">[7]</a></sup><sup>:4</sup> 이하 표에서는 점을 이용해 구모음과 비모음(문자 위에 틸다 부착)을 분리하고 있다.</p>

<p>13세기를 전후하여 고대 덴마크어를 제외한 모든 노르드어 방언들에서 Ǫ (<span class="IPA" title="IPA 발음">/ɔ/</span>)가 Ø 또는 O와 융합했다. 아이슬란드에서는 모든 Ǫ가 Ø와 융합되었는데, 이는 12세기의 《첫 번째 문법조약》에서는 둘이 구분되고 있지만 13세기의 《<a href="/wiki/%EA%B3%A0_%EC%97%90%EB%8B%A4" title="고 에다">고 에다</a>》에서는 구분되고 있지 않음으로 증거된다. 또한 역시 《첫 번째 문법조약》에서는 명시된 비모음 구분도 13세기경에는 사라진 것으로 생각된다.</p>

<h3><span class="mw-headline" id=".EB.8B.BF.EC.86.8C.EB.A6.AC">닿소리</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EA%B3%A0%EB%8C%80_%EB%85%B8%EB%A5%B4%EB%93%9C%EC%96%B4&amp;action=edit&amp;section=5" title="부분 편집: 닿소리">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>노르드어는 6개의 파열음이 있다. 이 중 <span class="IPA" title="IPA 발음">/p/</span>는 단어 앞에 나오는 경우가 드물고, <span class="IPA" title="IPA 발음">/d/</span>와 <span class="IPA" title="IPA 발음">/b/</span>는 홑소리들 사이에서 마찰 이음으로 발음되는데, 합성어에서는 예외이다(e.g. <span lang="non" xml:lang="non">veðrabati</span> <small>베드라바티</small>). 이런 현상은 <a class="new" href="/w/index.php?title=%EA%B2%8C%EB%A5%B4%EB%A7%8C_%EC%A1%B0%EC%96%B4&amp;action=edit&amp;redlink=1" title="게르만 조어 (없는 문서)">게르만 조어</a>에서부터 이미 나타나고 있다(e.g. 홑소리들 사이에 끼인 <i>*b</i> <span class="IPA" title="IPA 발음">*[β]</span> &gt; [v]). 음소 <span class="IPA" title="IPA 발음">/ɡ/</span>는 <i>n</i> 또는 다른 <i>g</i> 뒤에서 <span class="IPA" title="IPA 발음">[ɡ]</span>로 발음되고, <span class="IPA" title="IPA 발음">/s/</span>와 <span class="IPA" title="IPA 발음">/t/</span> 앞에서는 <span class="IPA" title="IPA 발음">[k]</span>로 발음된다. <span class="fix" title="모호한 표현">이 음은 단어 안에서 다른 홑소리들 사이에 있을 일부 경우 유성 연구개 마찰음 <span class="IPA" title="IPA 발음">[ɣ]</span>이 되고 그 외의 경우, 유성 연구개 파열음 <span class="IPA" title="IPA 발음">[ɡ]</span>이 된다</span><sup class="정리_필요 noprint">[<a href="/wiki/%EC%9C%84%ED%82%A4%EB%B0%B1%EA%B3%BC:%EB%AA%A8%ED%98%B8%ED%95%9C_%ED%91%9C%ED%98%84%EC%9D%84_%EC%82%BC%EA%B0%80_%EC%A3%BC%EC%84%B8%EC%9A%94" title="위키백과:모호한 표현을 삼가 주세요">모호한 표현</a>]</sup>.<sup class="reference" id="cite_ref-11"><a href="#cite_note-11">[10]</a></sup><sup class="reference" id="cite_ref-Sweet_12-0"><a href="#cite_note-Sweet-12">[11]</a></sup> 동노르드어의 /ʀ/ 음소는 그 발음 위치가 정확히 밝혀지지 않은 <a class="new" href="/w/index.php?title=%EC%84%A4%EC%B2%A8%EC%9D%8C&amp;action=edit&amp;redlink=1" title="설첨음 (없는 문서)">설첨음</a>이었다가, <a href="/wiki/%EC%9E%85%EC%B2%9C%EC%9E%A5" title="입천장">입천장</a> <a href="/wiki/%EC%B9%98%EC%B0%B0%EC%9D%8C" title="치찰음">치찰음</a>으로 재구축되었다.<sup class="reference" id="cite_ref-NordicLanguages202_8-2"><a href="#cite_note-NordicLanguages202-8">[7]</a></sup><sup>:2</sup> 이 음소는 게르만 조어의 음소 <span class="IPA" title="IPA 발음">/z/</span>에서 파생된 것이며 최종적으로는 음소 <span class="IPA" title="IPA 발음">/r/</span>로 진화했다. 이 과정은 서노르드어에서는 진작 이루어졌다.</p>

<p>겹닿소리 <span lang="non" xml:lang="non"><i>hl</i></span>, <span lang="non" xml:lang="non"><i>hr</i></span>, <span lang="non" xml:lang="non"><i>hn</i></span>은 단어 맨 앞에만 발생한다. 이 음가들이 앞의 <span lang="non" xml:lang="non"><i>h</i></span> 가 ㅎ(<span class="IPA" title="IPA 발음">/h/</span> 또는 <span class="IPA" title="IPA 발음">/x/</span>)으로 발음되면서 그 뒤의 닿소리들도 발음되어 흘, 흐ㄹ, 흔ㄴ 식으로 발음되는 것이었는지, 아니면 각각 단일 무성 공명음 ㄹ<span class="IPA" title="IPA 발음">/l̥/</span>, ㄹ<span class="IPA" title="IPA 발음">/r̥/</span>, ㄴ<span class="IPA" title="IPA 발음">/n̥/</span>으로 발음되는 것이었는지는 불확실하다. 고대 노르웨이어, 고대 덴마크어, 후기 고대 스웨덴어에서는 <span lang="non" xml:lang="non"><i>hl</i></span>, <span lang="non" xml:lang="non"><i>hr</i></span>, <span lang="non" xml:lang="non"><i>hn</i></span>이 홑닿소리 <span lang="non" xml:lang="non"><i>l</i>, <i>r</i>, <i>n</i></span> 로 간략화되는데, 이를 근거로 노르드어에서도 단일 무성 공명음으로 발음된 것이라고 추측하기도 한다.</p>
<p><span lang="non" xml:lang="non"><i>hv</i></span>의 발음도 불확실하다. <span class="IPA" title="IPA 발음">/xʷ/</span>(게르만조어의 발음), 또는 <span class="IPA" title="IPA 발음">/hʷ/</span>나 <span class="IPA" title="IPA 발음">/ʍ/</span>였을 수도 있다. 상술한 <span lang="non" xml:lang="non"><i>hl</i>, <i>hr</i>, <i>hn</i></span>과 달리 <span lang="non" xml:lang="non"><i>hv</i></span>는 모든 방언에서 겹닿소리 형태를 보다 오래 유지했다. 아이슬란드어에서는 아예 오늘날까지 무성 공명음으로 결합되지 않고 있으며, 대신 파열음 <span class="IPA" title="IPA 발음">/kv/</span>으로 <a class="new" href="/w/index.php?title=%EA%B2%BD%EC%9D%8C%ED%99%94&amp;action=edit&amp;redlink=1" title="경음화 (없는 문서)">경음화</a>되었다. 이는 이 음가가 무성 공명음이 아니고 보다 강한 협착기식음이었음을 시사한다.</p>
<h3><span class="mw-headline" id=".EC.B2.A0.EC.9E.90.EB.B2.95">철자법</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EA%B3%A0%EB%8C%80_%EB%85%B8%EB%A5%B4%EB%93%9C%EC%96%B4&amp;action=edit&amp;section=6" title="부분 편집: 철자법">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>조상 언어인 노르드 조어는 <a class="new" href="/w/index.php?title=%EB%8C%80_%ED%91%B8%ED%83%80%EB%A5%B4%ED%81%AC&amp;action=edit&amp;redlink=1" title="대 푸타르크 (없는 문서)">대 푸타르크</a>(고대 룬 문자)로 쓰여진 반면, 노르드어는 <a href="/wiki/%EC%86%8C_%ED%91%B8%ED%83%80%EB%A5%B4%ED%81%AC" title="소 푸타르크">소 푸타르크</a>로 쓰여졌다. 소 푸타르크는 16개 문자로만 구성되어 있었으며, 문자 수가 모자랐기 때문에 홑소리 <span lang="non" xml:lang="non"><i>u</i></span> 에 해당하는 문자가 <span lang="non" xml:lang="non"><i>o</i>, <i>ø</i>, <i>y</i></span>에도 사용되고 홑소리 <span lang="non" xml:lang="non"><i>i</i></span>에 해당하는 문자가 <span lang="non" xml:lang="non"><i>e</i></span>에도 사용되었다. 보다 시간이 지난 뒤에는 <a class="new" href="/w/index.php?title=%EC%A4%91%EC%84%B8_%EB%A3%AC_%EB%AC%B8%EC%9E%90&amp;action=edit&amp;redlink=1" title="중세 룬 문자 (없는 문서)">푸토르크</a>(중세 룬 문자)가 사용되었다.</p>
<p><a class="mw-redirect" href="/wiki/%EB%9D%BC%ED%8B%B4_%EB%AC%B8%EC%9E%90" title="라틴 문자">라틴 문자</a>가 도입된 뒤에도 중세 당시에는 하나의 표준된 철자법이 존재하지 않았다. 룬 문자 <a class="new" href="/w/index.php?title=%E1%9A%B9&amp;action=edit&amp;redlink=1" title="ᚹ (없는 문서)">윈(ᚹ)</a>을 수정한 문자가 <a class="new" href="/w/index.php?title=%EA%9D%A8&amp;action=edit&amp;redlink=1" title="Ꝩ (없는 문서)">벤드(Ꝩ)</a>라고 불리면서 음가 <span class="IPA" title="IPA 발음">/u/</span>, <span class="IPA" title="IPA 발음">/v/</span>, <span class="IPA" title="IPA 발음">/w/</span>를 나타내기 위해 잠깐 쓰였다. 장모음은 예음부호 ´ 를 찍어서 표시하기도 했지만, 해당 모음을 두 번 써서 장모음임을 표시하거나 아예 아무 표시도 하지 않기도 했다. 오늘날 사용되는 표준화된 노르드어 철자법은 19세기에 만들어진 것이다.</p>

<h2><span class="mw-headline" id=".EB.85.B8.EB.A5.B4.EB.93.9C.EC.96.B4_.ED.86.B5.EC.82.AC.EB.A1.A0">노르드어 통사론</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EA%B3%A0%EB%8C%80_%EB%85%B8%EB%A5%B4%EB%93%9C%EC%96%B4&amp;action=edit&amp;section=14" title="부분 편집: 노르드어 통사론">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>노르드어의 어순 배열은 영어에 비해 널널하다. 노르드어의 나열 구조는 영어와 다소 달라서, 영어에서 "a, b, 그리고 c" 및 "a, b, 또는 c"라고 할 경우 노르드어는 각각 "a와 b와 c" 또는 "a와 b 또는 c"라고 하게 된다.</p>
<h2><span class="mw-headline" id=".EB.85.B8.EB.A5.B4.EB.93.9C.EC.96.B4_.EB.AC.B8.EB.B2.95">노르드어 문법</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EA%B3%A0%EB%8C%80_%EB%85%B8%EB%A5%B4%EB%93%9C%EC%96%B4&amp;action=edit&amp;section=15" title="부분 편집: 노르드어 문법">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>노르드어는 <a class="new" href="/w/index.php?title=%EC%96%B4%ED%98%95%EB%B3%80%ED%99%94&amp;action=edit&amp;redlink=1" title="어형변화 (없는 문서)">어형변화</a>가 중간 정도의 굴절어로서 명사와 동사에서 상당한 수준의 변화가 일어난다. 대부분의 결합 <a href="/wiki/%ED%98%95%ED%83%9C%EC%86%8C" title="형태소">형태소</a>는 현대 아이슬란드어에서, 특히 명사 어형변화군에서 그대로 가지고 있다. 반면 현대 노르웨이어는 이에 비해 보다 분석적인 단어 구조로 이행했다.</p>
<h3><span class="mw-headline" id=".EC.84.B1.EB.B3.84">성별</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EA%B3%A0%EB%8C%80_%EB%85%B8%EB%A5%B4%EB%93%9C%EC%96%B4&amp;action=edit&amp;section=16" title="부분 편집: 성별">편집</a><span class="mw-editsection-bracket">]</span></span></h3>

<h2><span class="mw-headline" id=".EB.85.B8.EB.A5.B4.EB.93.9C.EC.96.B4_.EB.AC.B8.ED.97.8C">노르드어 문헌</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EA%B3%A0%EB%8C%80_%EB%85%B8%EB%A5%B4%EB%93%9C%EC%96%B4&amp;action=edit&amp;section=19" title="부분 편집: 노르드어 문헌">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>가장 오래된 노르드어 문헌은 8세기경 <a href="/wiki/%EB%A3%AC_%EB%AC%B8%EC%9E%90" title="룬 문자">룬 문자</a>로 새겨진 각석들이다. 룬 문자는 15세기까지도 널리 쓰였으며, 스웨덴의 일부 지역에서는 19세기에도 사용된 바 있다. 노르드인들이 기독교로 개종한 11세기 전후로 <a class="mw-redirect" href="/wiki/%EB%9D%BC%ED%8B%B4_%EB%AC%B8%EC%9E%90" title="라틴 문자">라틴 문자</a>가 들어왔다. 라틴 문자로 쓰인 가장 오래된 노르드어 문헌은 12세기 중반의 물건이다. 그 뒤 노르드어는 중세 유럽 특유의 문헌 언어로 사용되었으며, 그 문헌들 중 아직까지 보존된 것은 대부분 <a href="/wiki/%EC%95%84%EC%9D%B4%EC%8A%AC%EB%9E%80%EB%93%9C" title="아이슬란드">아이슬란드</a>에서 쓰여졌다. 가장 유명한 것들로는 <a href="/wiki/%EC%95%84%EC%9D%B4%EC%8A%AC%EB%9E%80%EB%93%9C_%EC%82%AC%EA%B0%80" title="아이슬란드 사가">아이슬란드 사가</a>를 비롯한 <a class="mw-disambig" href="/wiki/%EC%82%AC%EA%B0%80" title="사가">사가</a>들이 있으나, 그 외에도 <a href="/wiki/%EA%B8%B0%EC%82%AC_%EB%AC%B8%ED%95%99" title="기사 문학">기사 문학</a>, 고전 신화, 구약성경 따위를 노르드어로 번역한 것도 존재한다. 교육용 자료도 노르드어로 쓰여져서, 상술한 《<a class="new" href="/w/index.php?title=%EC%B2%AB_%EB%B2%88%EC%A7%B8_%EB%AC%B8%EB%B2%95%EC%A1%B0%EC%95%BD&amp;action=edit&amp;redlink=1" title="첫 번째 문법조약 (없는 문서)">첫 번째 문법조약</a>》이 그러한 예이다. 그 외에 많은 수의 서한이나 공문서가 노르드어로 쓰여졌다.<sup class="reference" id="cite_ref-13"><a href="#cite_note-13">[12]</a></sup></p>
<h2><span class="mw-headline" id=".EB.B0.A9.EC.96.B8">방언</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EA%B3%A0%EB%8C%80_%EB%85%B8%EB%A5%B4%EB%93%9C%EC%96%B4&amp;action=edit&amp;section=20" title="부분 편집: 방언">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>노르드어에서 발생한 혁신 대부분은 노르드어가 사용되는 지역 전체에 확산되었다. 그 결과 노르드어의 방언들은 서로 매우 닮았으며 같은 언어로 여겨졌다. <a class="mw-redirect" href="/wiki/%EC%8A%A4%EB%85%B8%EB%A6%AC_%EC%8A%A4%ED%88%B4%EB%A3%A8%EC%86%90" title="스노리 스툴루손">스노리 스툴루손</a>의 《<a href="/wiki/%ED%97%A4%EC%9E%84%EC%8A%A4%ED%81%AC%EB%A7%81%EA%B8%80%EB%9D%BC" title="헤임스크링글라">헤임스크링글라</a>》에서 발췌한 다음 두 문장에 나오다시피, 당대에 노르드어는 "데인어(<span lang="non" xml:lang="non">Dǫnsk tunga</span> <small>돈스크 퉁가</small>)"라고도 불리고 "노르드 말(<span lang="non" xml:lang="non">Norrœnt mál</span> <small>노르뢴트 말</small>)"이라고도 불렸다.</p>

<p>그러나 지리적으로 제한된 일부 변화로 인해 서노르드어와 동노르드어 사이의 차이가 발생했다.</p>
<p>8세기에 노르드 조어가 고대 노르드어로 진화하면서, <a class="new" href="/w/index.php?title=%EB%8F%85%EC%9D%BC%EC%96%B4_%EC%9B%80%EB%9D%BC%EC%9A%B0%ED%8A%B8&amp;action=edit&amp;redlink=1" title="독일어 움라우트 (없는 문서)">독일어 움라우트</a>가 노르드어 사용 지역에도 영향을 미친 것으로 보인다. 그러나 나중에 움라우트 사용이 각기 달라지기 시작하면서 서노르드어와 동노르드어가 분열하게 되었다. 보통 움라우트는 서노르드어에서 더 잘 보존되었고(예컨대 <span lang="non" xml:lang="non"><i>fylla</i></span> <small>퓔라</small> &lt; *<i>fullijan</i>), 동노르드에서는 보편화를 거쳐 움라우트들이 많이 사라졌다. 오래된 동노르드어 문헌들이나 동노르드 룬 각석들은 후기 서노르드어에서 나타나는 정도와 같은 움라우트를 보존하고 있다. 한편 <a class="new" href="/w/index.php?title=%EC%9D%B4%EC%A4%91%EB%AA%A8%EC%9D%8C%ED%99%94&amp;action=edit&amp;redlink=1" title="이중모음화 (없는 문서)">이중모음화</a>는 동노르드어에서 더욱 두드러졌다(예컨대 <span lang="non" xml:lang="non"><i>hiarta</i></span> <small>히아르타</small> &lt; *<i>hertō</i>)는데, 아마 이것도 굴절어 체계 내부의 일반화 때문으로 추측된다. 이러한 차이가 9세기에서 10세기 사이에 동서 노르드어가 분화하게 된 가장 큰 이유로서, 노르웨이와 대서양 정착지들에서는 서노르드어가, 덴마크와 스웨덴에서는 동노르드어가 쓰이게 되었다.</p>
