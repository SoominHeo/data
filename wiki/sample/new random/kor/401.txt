<p><b>ASP.NET</b>은 <a href="/wiki/%EB%A7%88%EC%9D%B4%ED%81%AC%EB%A1%9C%EC%86%8C%ED%94%84%ED%8A%B8" title="마이크로소프트">마이크로소프트</a>사가 개발하여 판매하는 <a href="/wiki/%EC%9B%B9_%EC%95%A0%ED%94%8C%EB%A6%AC%EC%BC%80%EC%9D%B4%EC%85%98_%ED%94%84%EB%A0%88%EC%9E%84%EC%9B%8C%ED%81%AC" title="웹 애플리케이션 프레임워크">웹 애플리케이션 프레임워크</a>이며 <a href="/wiki/%ED%94%84%EB%A1%9C%EA%B7%B8%EB%9E%98%EB%A8%B8" title="프로그래머">프로그래머</a>들이 동적인 <a class="mw-redirect" href="/wiki/%EC%9B%B9_%EC%82%AC%EC%9D%B4%ED%8A%B8" title="웹 사이트">웹 사이트</a>, <a href="/wiki/%EC%9B%B9_%EC%95%A0%ED%94%8C%EB%A6%AC%EC%BC%80%EC%9D%B4%EC%85%98" title="웹 애플리케이션">웹 애플리케이션</a>, <a href="/wiki/%EC%9B%B9_%EC%84%9C%EB%B9%84%EC%8A%A4" title="웹 서비스">웹 서비스</a>를 만들 수 있게 도와 준다. 2002년 1월에 <a href="/wiki/%EB%8B%B7%EB%84%B7_%ED%94%84%EB%A0%88%EC%9E%84%EC%9B%8C%ED%81%AC" title="닷넷 프레임워크">닷넷 프레임워크</a> 버전 1.0과 함께 처음 출시되었으며 마이크로소프트의 <a href="/wiki/%EC%95%A1%ED%8B%B0%EB%B8%8C_%EC%84%9C%EB%B2%84_%ED%8E%98%EC%9D%B4%EC%A7%80" title="액티브 서버 페이지">액티브 서버 페이지</a> (ASP) 기술의 뒤를 잇는다. ASP.NET은 <a href="/wiki/%EA%B3%B5%ED%86%B5_%EC%96%B8%EC%96%B4_%EB%9F%B0%ED%83%80%EC%9E%84" title="공통 언어 런타임">공통 언어 런타임</a> (CLR)을 기반으로 작성되며 프로그래머들이 <a class="new" href="/w/index.php?title=CLI_%EC%96%B8%EC%96%B4%EC%9D%98_%EB%AA%A9%EB%A1%9D&amp;action=edit&amp;redlink=1" title="CLI 언어의 목록 (없는 문서)">닷넷 언어</a>가 사용된 ASP.NET 코드를 기록할 수 있게 도와 준다. ASP.NET <a href="/wiki/SOAP" title="SOAP">SOAP</a> 확장 프레임워크는 ASP.NET 구성 요소가 SOAP 메시지를 처리할 수 있게 도와 준다.</p>
