<h1 class="firstHeading" id="firstHeading" lang="ko">고급 시스템 포맷</h1>
<div class="mw-body-content" id="bodyContent">
<div id="siteSub">위키백과, 우리 모두의 백과사전.</div>
<div id="contentSub"><span class="mw-redirectedfrom">(<a class="mw-redirect" href="/w/index.php?title=.asf&amp;redirect=no" title=".asf">.asf</a>에서 넘어옴)</span></div>
<div class="mw-jump" id="jump-to-nav">
					이동:					<a href="#mw-head">둘러보기</a>, 					<a href="#p-search">검색</a>
</div>
<div class="mw-content-ltr" dir="ltr" id="mw-content-text" lang="ko"><script>function mfTempOpenSection(id){var block=document.getElementById("mf-section-"+id);block.className+=" open-block";block.previousSibling.className+=" open-block";}</script><p><b>고급 시스템 포맷</b>(Advanced Systems Format, ASF)은 <a href="/wiki/%EB%A7%88%EC%9D%B4%ED%81%AC%EB%A1%9C%EC%86%8C%ED%94%84%ED%8A%B8" title="마이크로소프트">마이크로소프트</a>사에서 만든, 디지털 소리와 영상을 담는 포맷이며, 특히 <a class="mw-redirect" href="/wiki/%EC%8A%A4%ED%8A%B8%EB%A6%AC%EB%B0%8D_%EB%AF%B8%EB%94%94%EC%96%B4" title="스트리밍 미디어">스트리밍 미디어</a>를 뜻한다. 과거에는 <b>능동 스트리밍 포맷</b>(Active Streaming Format), <b>고급 스트리밍 포맷</b>(Advanced Streaming Format)이라는 용어를 사용하였다. ASF는 <a class="mw-redirect" href="/wiki/%EC%9C%88%EB%8F%84_%EB%AF%B8%EB%94%94%EC%96%B4" title="윈도 미디어">윈도 미디어</a> 프레임워크의 일부이다.</p>
<p>또한 인터넷이 연결되어 있지 않은 로컬 컴퓨터에서도 재생할 수 있다. 1개의 파일에 다중 비트 레이트의 영상, 소리, 메타데이터, 정지 화면, <a href="/wiki/URL" title="URL">URL</a>, <a href="/wiki/%EC%9E%90%EB%A7%89" title="자막">자막</a> 등의 데이터를 넣을 수 있다.</p>
<p>윈도 미디어 (<a class="mw-redirect" href="/wiki/%EC%9C%88%EB%8F%84_%EB%AF%B8%EB%94%94%EC%96%B4_%EB%B9%84%EB%94%94%EC%98%A4" title="윈도 미디어 비디오">윈도 미디어 비디오</a> (WMV) / <a class="mw-redirect" href="/wiki/%EC%9C%88%EB%8F%84_%EB%AF%B8%EB%94%94%EC%96%B4_%EC%98%A4%EB%94%94%EC%98%A4" title="윈도 미디어 오디오">윈도 미디어 오디오</a> (WMA))의 표준 형식 파일이다.</p>
<p></p>

<h2><span class="mw-headline" id=".ED.8A.B9.ED.97.88_.EB.AC.B8.EC.A0.9C">특허 문제</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EA%B3%A0%EA%B8%89_%EC%8B%9C%EC%8A%A4%ED%85%9C_%ED%8F%AC%EB%A7%B7&amp;action=edit&amp;section=1" title="부분 편집: 특허 문제">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p><a href="/wiki/%EB%A7%88%EC%9D%B4%ED%81%AC%EB%A1%9C%EC%86%8C%ED%94%84%ED%8A%B8" title="마이크로소프트">마이크로소프트</a>는 ASF 저장 구조를 미국 <a href="/wiki/%ED%8A%B9%ED%97%88" title="특허">특허</a>로 취급한다. 프리웨어인 동영상 편집 프로그램 <a class="mw-redirect" href="/wiki/%EB%B2%84%EC%B6%94%EC%96%BC%EB%8D%A5" title="버추얼덥">버추얼덥</a>도 원래 ASF와 호환이 되었으나, <a href="/wiki/%EB%A7%88%EC%9D%B4%ED%81%AC%EB%A1%9C%EC%86%8C%ED%94%84%ED%8A%B8" title="마이크로소프트">마이크로소프트</a>사가 손해 배상 청구를 하였기 때문에 지금은 ASF와 호환되고 있지 않다.</p>
<h2><span class="mw-headline" id="ASF_.ED.8C.8C.EC.9D.BC.EC.97.90_.EC.A0.80.EC.9E.A5.ED.95.A0_.EC.88.98_.EC.9E.88.EB.8A.94_.EB.AF.B8.EB.94.94.EC.96.B4">ASF 파일에 저장할 수 있는 미디어</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EA%B3%A0%EA%B8%89_%EC%8B%9C%EC%8A%A4%ED%85%9C_%ED%8F%AC%EB%A7%B7&amp;action=edit&amp;section=2" title="부분 편집: ASF 파일에 저장할 수 있는 미디어">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>ASF 파일은 아래에 나타내는 영상, 소리, 코덱의 미디어 자료를 합하여 저장하는 데에 쓰일 수 있다.</p>
<ul>
<li>영상：<a class="new" href="/w/index.php?title=MS-MPEG4&amp;action=edit&amp;redlink=1" title="MS-MPEG4 (없는 문서)">MS-MPEG4</a>, <a class="new" href="/w/index.php?title=ISO_MPEG-4&amp;action=edit&amp;redlink=1" title="ISO MPEG-4 (없는 문서)">ISO MPEG-4</a>, <a class="mw-redirect" href="/wiki/WMV7" title="WMV7">WMV7</a>, <a class="mw-redirect" href="/wiki/WMV8" title="WMV8">WMV8</a>, <a class="mw-redirect" href="/wiki/WMV9" title="WMV9">WMV9</a>, <a class="mw-redirect" href="/wiki/%EB%94%94%EB%B9%85" title="디빅">디빅</a>(DivX)</li>
<li>소리：<a class="mw-redirect" href="/wiki/WMA9" title="WMA9">WMA9</a>, <a href="/wiki/G.726" title="G.726">G.726</a>, <a class="new" href="/w/index.php?title=LPCM&amp;action=edit&amp;redlink=1" title="LPCM (없는 문서)">LPCM</a>, <a href="/wiki/MP3" title="MP3">MP3</a></li>
</ul>
<h2><span class="mw-headline" id="ASF_.ED.8C.8C.EC.9D.BC.EC.97.90.EC.84.9C_.EC.93.B0.EC.9D.B4.EB.8A.94_.ED.99.95.EC.9E.A5.EC.9E.90">ASF 파일에서 쓰이는 확장자</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EA%B3%A0%EA%B8%89_%EC%8B%9C%EC%8A%A4%ED%85%9C_%ED%8F%AC%EB%A7%B7&amp;action=edit&amp;section=3" title="부분 편집: ASF 파일에서 쓰이는 확장자">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<ul>

<li>.asx</li>
<li>.wmv (윈도 미디어 비디오 코덱을 사용해 압축된 자료를 포함한 파일)</li>

<h2><span class="mw-headline" id=".EC.BD.94.EB.8D.B1">코덱</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EA%B3%A0%EA%B8%89_%EC%8B%9C%EC%8A%A4%ED%85%9C_%ED%8F%AC%EB%A7%B7&amp;action=edit&amp;section=4" title="부분 편집: 코덱">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>ASF 컨테이너 포맷이 기술적으로 아무 코덱이나 포함할 수 있음에도 불구하고, 마이크로소프트의 인코딩 도구(<a class="new" href="/w/index.php?title=%EC%9C%88%EB%8F%84_%EB%AF%B8%EB%94%94%EC%96%B4_%EC%9D%B8%EC%BD%94%EB%8D%94&amp;action=edit&amp;redlink=1" title="윈도 미디어 인코더 (없는 문서)">윈도 미디어 인코더</a>, <a class="mw-redirect" href="/wiki/%EC%9C%88%EB%8F%84_%EB%AC%B4%EB%B9%84_%EB%A9%94%EC%9D%B4%EC%BB%A4" title="윈도 무비 메이커">윈도 무비 메이커</a> 등)는 다이렉트X 미디어 오브젝트 (DMO) 프레임워크를 사용하여 ASF/WMA/WMV 파일들을 만들어 낸다. 이제까지, 서드파티 DMO 기반 코덱들은 찾기가 쉽지 않다. 마이크로소프트는 이러한 포맷의 재생을 허용하는 데 목표를 두는 <a href="/wiki/%EC%9E%90%EC%9C%A0_%EC%86%8C%ED%94%84%ED%8A%B8%EC%9B%A8%EC%96%B4" title="자유 소프트웨어">자유 소프트웨어</a> 프로젝트에게 특허 위반이라는 명목으로 위협을 가하고 있다.</p>
