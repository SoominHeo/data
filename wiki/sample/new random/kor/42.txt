<p><b>필리오케 문제</b> 또는 <b>필리오케 논쟁</b>은 <a href="/wiki/%EA%B8%B0%EB%8F%85%EA%B5%90" title="기독교">기독교</a>의 <a href="/wiki/%EB%8B%88%EC%BC%80%EC%95%84-%EC%BD%98%EC%8A%A4%ED%83%84%ED%8B%B0%EB%85%B8%ED%8F%B4%EB%A6%AC%EC%8A%A4_%EC%8B%A0%EA%B2%BD" title="니케아-콘스탄티노폴리스 신경">니케아-콘스탄티노폴리스 신경</a>에 수록된 <a href="/wiki/%EC%82%BC%EC%9C%84%EC%9D%BC%EC%B2%B4" title="삼위일체">삼위일체</a>에 관한 <a href="/wiki/%EA%B5%90%EB%A6%AC" title="교리">교리</a> 논쟁으로, <a href="/wiki/%EA%B8%B0%EB%8F%85%EA%B5%90" title="기독교">기독교</a> <a href="/wiki/%EC%8B%A0%ED%95%99" title="신학">신학</a>의 주요 쟁점 가운데 하나이다. <a href="/wiki/11%EC%84%B8%EA%B8%B0" title="11세기">11세기</a> <a href="/wiki/%EB%A1%9C%EB%A7%88_%EA%B0%80%ED%86%A8%EB%A6%AD%EA%B5%90%ED%9A%8C" title="로마 가톨릭교회">로마 가톨릭교회</a>와 <a href="/wiki/%EB%8F%99%EB%B0%A9_%EC%A0%95%EA%B5%90%ED%9A%8C" title="동방 정교회">동방 정교회</a>가 <a href="/wiki/%EB%8F%99%EC%84%9C_%EA%B5%90%ED%9A%8C%EC%9D%98_%EB%B6%84%EC%97%B4" title="동서 교회의 분열">분열</a>하는 빌미가 되었다.</p>

<h2><span class="mw-headline" id=".EC.97.AD.EC.82.AC">역사</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%ED%95%84%EB%A6%AC%EC%98%A4%EC%BC%80_%EB%AC%B8%EC%A0%9C&amp;action=edit&amp;section=1" title="부분 편집: 역사">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>필리오케(<span style="font-size: smaller;"><a href="/wiki/%EB%9D%BC%ED%8B%B4%EC%96%B4" title="라틴어">라틴어</a>:</span> <span lang="la" xml:lang="la">Filióque</span>)란 '<a href="/wiki/%EC%84%B1%EC%9E%90_(%EC%82%BC%EC%9C%84%EC%9D%BC%EC%B2%B4)" title="성자 (삼위일체)">아들</a>에게서(아들로부터)'라는 뜻의 <a href="/wiki/%EB%9D%BC%ED%8B%B4%EC%96%B4" title="라틴어">라틴어</a>로, 본래 제1차 <a class="mw-disambig" href="/wiki/%EC%BD%98%EC%8A%A4%ED%83%84%ED%8B%B0%EB%85%B8%ED%8F%B4%EB%A6%AC%EC%8A%A4_%EA%B3%B5%EC%9D%98%ED%9A%8C" title="콘스탄티노폴리스 공의회">콘스탄티노폴리스 공의회</a>(<a href="/wiki/381%EB%85%84" title="381년">381년</a>)에서 채택된 것으로 알려진 <a href="/wiki/%EB%8B%88%EC%BC%80%EC%95%84-%EC%BD%98%EC%8A%A4%ED%83%84%ED%8B%B0%EB%85%B8%ED%8F%B4%EB%A6%AC%EC%8A%A4_%EC%8B%A0%EA%B2%BD" title="니케아-콘스탄티노폴리스 신경">니케아-콘스탄티노폴리스 신경</a>(<i>이하 '<a href="/wiki/%EC%8B%A0%EC%A1%B0" title="신조">신경</a>'이라 함</i>)의 <a href="/wiki/%EA%B7%B8%EB%A6%AC%EC%8A%A4%EC%96%B4" title="그리스어">그리스어</a> 원문에 없는 단어이나 <a href="/wiki/589%EB%85%84" title="589년">589년</a> 제3차 <a href="/wiki/%ED%86%A8%EB%A0%88%EB%8F%84" title="톨레도">톨레도</a> <a class="mw-redirect" href="/wiki/%EC%8B%9C%EB%85%B8%EB%93%9C" title="시노드">시노드</a>에서 아직 <a href="/wiki/%EC%8A%A4%ED%8E%98%EC%9D%B8" title="스페인">스페인</a> 내에 잔존하고 있었던 <a class="mw-redirect" href="/wiki/%EC%95%84%EB%A6%AC%EC%9A%B0%EC%8A%A4%EC%A3%BC%EC%9D%98" title="아리우스주의">아리우스주의</a>를 경계할 의도로 <a class="mw-redirect" href="/wiki/%EC%84%9C%EB%B0%A9%EA%B5%90%ED%9A%8C" title="서방교회">서방교회</a>가 <a href="/wiki/%EB%9D%BC%ED%8B%B4%EC%96%B4" title="라틴어">라틴어</a>로 번역한 신경에 처음으로 첨가하였다.</p>
<p>따라서 그리스어 신경 원문 중 <i>“<a href="/wiki/%EC%84%B1%EB%A0%B9" title="성령">성령</a>은 <a href="/wiki/%EC%84%B1%EB%B6%80" title="성부">성부</a>에게서 발(發)하시고(τό εκ τού Πατρός εκπορευόμενον)”</i>라는 구절은 라틴어 신경에서 <i>“성령은 성부와 <a href="/wiki/%EC%84%B1%EC%9E%90_(%EC%82%BC%EC%9C%84%EC%9D%BC%EC%B2%B4)" title="성자 (삼위일체)">성자</a>에게서 발하시고(qui ex Patre Filióque procédit)”</i>로 바뀌게 되어, <a class="mw-redirect" href="/wiki/%EB%8F%99%EB%B0%A9_%EA%B5%90%ED%9A%8C" title="동방 교회">동방 교회</a>에서 사용하는 그리스어 신경과 서방 교회에서 사용하는 라틴어 신경 간에 불일치가 발생하게 된 것이다. 그러나 그때까지는 아직 필리오케가 삽입된 신경은 <a href="/wiki/%EC%8A%A4%ED%8E%98%EC%9D%B8" title="스페인">스페인</a> 내에서만 사용되고 있었다.</p>
<p>그러다 <a href="/wiki/796%EB%85%84" title="796년">796년</a> 프리울리 시노드에서 <a href="/wiki/%ED%94%84%EB%9E%91%ED%81%AC_%EC%99%95%EA%B5%AD" title="프랑크 왕국">프랑크 왕국</a> 아킬레이아의 파울리노 <a href="/wiki/%EC%B4%9D%EB%8C%80%EC%A3%BC%EA%B5%90" title="총대주교">총대주교</a>는 필리오케의 신경 삽입을 옹호하였고, <a href="/wiki/800%EB%85%84" title="800년">800년</a>경에는 전체 프랑크 왕국의 <a href="/wiki/%EB%AF%B8%EC%82%AC" title="미사">미사</a>에서 필리오케가 삽입된 신경이 암송되기 시작하여 널리 퍼지게 되었다. 이것이 <a href="/wiki/847%EB%85%84" title="847년">847년</a> 프랑크왕국의 <a class="mw-redirect" href="/wiki/%EC%88%98%EB%8F%84%EC%9E%90" title="수도자">수도자</a>들에 의해 <a href="/wiki/%EC%98%88%EB%A3%A8%EC%82%B4%EB%A0%98" title="예루살렘">예루살렘</a>에 소개되자 동방교회 수도자들의 강한 반발을 샀다.</p>
<p>이 문제가 <a href="/wiki/%EA%B5%90%ED%99%A9_%EB%A0%88%EC%98%A4_3%EC%84%B8" title="교황 레오 3세">교황 레오 3세</a>에게 알려지자, <a href="/wiki/%EA%B5%90%ED%99%A9" title="교황">교황</a>은 필리오케가 교리적 측면에서는 문제가 없다고 여겼으나 반대측과의 마찰을 피하기 위하여 신경에 정식으로 필리오케를 추가하는 것은 보류하고자 하였고, 그는 필리오케가 없는 형태의 신경을 <a href="/wiki/%EA%B7%B8%EB%A6%AC%EC%8A%A4%EC%96%B4" title="그리스어">그리스어</a>와 <a href="/wiki/%EB%9D%BC%ED%8B%B4%EC%96%B4" title="라틴어">라틴어</a>로 각각 작성하여 <a class="mw-redirect" href="/wiki/%EC%84%B1_%EB%B2%A0%EB%93%9C%EB%A1%9C" title="성 베드로">성 베드로</a>의 묘에 봉헌된 은제 탁자 2개에 새겨넣도록 하였다. 그러나 필리오케가 삽입된 신경은 미사 중에 계속 불리었고, <a href="/wiki/%EA%B5%90%ED%99%A9_%EB%B2%A0%EB%84%A4%EB%94%95%ED%86%A0_8%EC%84%B8" title="교황 베네딕토 8세">교황 베네딕토 8세</a>는 <a href="/wiki/1013%EB%85%84" title="1013년">1013년</a> 필리오케가 삽입된 신경을 최종 승인하기에 이른다.</p>
<p>이에 서방교회에게 필리오케를 신경에서 삭제할 것을 강력히 주장했던 <a href="/wiki/%EC%BD%98%EC%8A%A4%ED%83%84%ED%8B%B0%EB%85%B8%ED%8F%B4%EB%A6%AC%EC%8A%A4_%EC%B4%9D%EB%8C%80%EC%A3%BC%EA%B5%90" title="콘스탄티노폴리스 총대주교">콘스탄티노폴리스 총대주교</a> <a class="mw-redirect" href="/wiki/%ED%8F%AC%ED%8B%B0%EC%9A%B0%EC%8A%A4" title="포티우스">포티우스</a> 시대 이후로, 필리오케는 <a href="/wiki/%EA%B5%90%ED%99%A9" title="교황">교황</a> 수위권(首位權) 논쟁 등 여타의 신학적 문제와 더불어 동·서방 교회 갈등의 한 요인이 되었다.</p>
<h2><span class="mw-headline" id=".EB.8F.99.C2.B7.EC.84.9C.EB.B0.A9_.EA.B5.90.ED.9A.8C.EC.9D.98_.EC.83.81.ED.98.B8_.ED.8C.8C.EB.AC.B8.EA.B3.BC_.EA.B7.B8_.EB.AC.B4.ED.9A.A8.ED.99.94">동·서방 교회의 상호 파문과 그 무효화</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%ED%95%84%EB%A6%AC%EC%98%A4%EC%BC%80_%EB%AC%B8%EC%A0%9C&amp;action=edit&amp;section=2" title="부분 편집: 동·서방 교회의 상호 파문과 그 무효화">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>그러던 중 <a href="/wiki/%EA%B5%90%ED%99%A9_%EB%A0%88%EC%98%A4_9%EC%84%B8" title="교황 레오 9세">교황 레오 9세</a> 재위기간에, 교황은 콘스탄티노폴리스 총대주교 <a href="/wiki/%EB%AF%B8%EC%B9%B4%EC%97%98_%EC%BC%80%EB%A3%B0%EB%9D%BC%EB%A6%AC%EC%98%A4%EC%8A%A4" title="미카엘 케룰라리오스">미카엘 케룰라리오스</a>가 그의 관할지역에서 라틴전례의 관습을 금지한 것을 계기로 특사 훔베르트 <a href="/wiki/%EC%B6%94%EA%B8%B0%EA%B2%BD" title="추기경">추기경</a>을 그에게 파견하여 콘스탄티노폴리스 총대주교에 대한 '세계총대주교'라는 칭호를 폐기할 것과, 필리오케가 들어간 신경을 공식 채택할 것을 정식으로 요구하였다. 그러나 양측의 타협이 이루어지지 않자 총대주교는 교황 특사인 추기경을, 특사는 총대주교를 서로 <a href="/wiki/%ED%8C%8C%EB%AC%B8" title="파문">파문</a>하기에 이른다(<a href="/wiki/1054%EB%85%84" title="1054년">1054년</a>).</p>
<p>그러나 서방교회 측의 파문의 경우 특사 파견자인 교황 레오 9세가 이미 서거한 이후였기에 그 합법성에 문제가 있고, 동방교회 측의 파문도 교황이나 서방교회 전체에 대한 것이라기보다는 특사들 개인에 대한 인신공격적인 것이었으므로 교회법상 동·서방 교회가 서로를 파문한 것으로는 볼 수 없다는 견해가 유력하다. 또한, <a href="/wiki/1965%EB%85%84" title="1965년">1965년</a> <a href="/wiki/%EA%B5%90%ED%99%A9_%EB%B0%94%EC%98%A4%EB%A1%9C_6%EC%84%B8" title="교황 바오로 6세">교황 바오로 6세</a>와 콘스탄티노폴리스 총대주교 아테나고라스 1세는 <a href="/wiki/1054%EB%85%84" title="1054년">1054년</a>의 상호 파문을 무효화하고 화해의 인사를 나눈 바 있다.</p>
<h2><span class="mw-headline" id=".EC.8B.A0.ED.95.99.EC.A0.81_.ED.95.A9.EC.9D.98.EC.9D.98_.EC.8B.9C.EB.8F.84.EC.99.80_.EC.A2.8C.EC.A0.88">신학적 합의의 시도와 좌절</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%ED%95%84%EB%A6%AC%EC%98%A4%EC%BC%80_%EB%AC%B8%EC%A0%9C&amp;action=edit&amp;section=3" title="부분 편집: 신학적 합의의 시도와 좌절">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>동·서방 교회 재결합을 위하여 열린 <a class="mw-redirect mw-disambig" href="/wiki/%EB%A6%AC%EC%9A%A9" title="리용">리용</a> <a href="/wiki/%EA%B3%B5%EC%9D%98%ED%9A%8C" title="공의회">공의회</a>(<a href="/wiki/1274%EB%85%84" title="1274년">1274년</a>)와 <a href="/wiki/%ED%94%BC%EB%A0%8C%EC%B2%B4" title="피렌체">피렌체</a> 공의회(<a href="/wiki/1439%EB%85%84" title="1439년">1439년</a>)에서, <a class="mw-redirect" href="/wiki/%EB%8F%99%EB%B0%A9%EA%B5%90%ED%9A%8C" title="동방교회">동방교회</a> 측이 필리오케의 신경 삽입은 거절하나 그 교리는 승인한다고 밝힘으로서 필리오케에 관한 신학적 논쟁은 일단락되는 듯 보였으나, <a href="/wiki/1472%EB%85%84" title="1472년">1472년</a> <a class="mw-redirect" href="/wiki/%EB%8F%99%EB%B0%A9%EA%B5%90%ED%9A%8C" title="동방교회">동방교회</a>가 그들 단독으로 개최한 콘스탄티노폴리스 교회회의에서 위 리용과 피렌체에서의 합의를 정식으로 파기함으로써 동·서방 교회는 완전히 분열되기에 이른다.</p>
<h2><span class="mw-headline" id=".EB.8F.99.C2.B7.EC.84.9C.EB.B0.A9_.EA.B5.90.ED.9A.8C_.EC.96.91.EC.B8.A1.EC.9D.98_.EC.9E.85.EC.9E.A5">동·서방 교회 양측의 입장</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%ED%95%84%EB%A6%AC%EC%98%A4%EC%BC%80_%EB%AC%B8%EC%A0%9C&amp;action=edit&amp;section=4" title="부분 편집: 동·서방 교회 양측의 입장">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>필리오케에 대한 동·서방 양측의 입장은 다음과 같다.</p>
<h3><span class="mw-headline" id=".EB.8F.99.EB.B0.A9_.EC.A0.95.EA.B5.90.ED.9A.8C">동방 정교회</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%ED%95%84%EB%A6%AC%EC%98%A4%EC%BC%80_%EB%AC%B8%EC%A0%9C&amp;action=edit&amp;section=5" title="부분 편집: 동방 정교회">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<ul>
<li>세계<a href="/wiki/%EA%B3%B5%EC%9D%98%ED%9A%8C" title="공의회">공의회</a>가 채택한 신경을 전체 교회의 동의 없이 차후 변개할 수 없다.</li>
<li><a href="/wiki/%EC%84%B1%EB%A0%B9" title="성령">성령</a>이 <a href="/wiki/%EC%84%B1%EB%B6%80" title="성부">성부</a>에게서 발한다는 말씀이 <a href="/wiki/%EC%84%B1%EA%B2%BD" title="성경">성경</a>에 기록되어 있다. (<a class="mw-redirect" href="/wiki/%EC%9A%94%ED%95%9C%EB%B3%B5%EC%9D%8C%EC%84%9C" title="요한복음서">요한복음서</a> 15장 26절)</li>
<li><a href="/wiki/%EC%84%B1%EC%9E%90_(%EC%82%BC%EC%9C%84%EC%9D%BC%EC%B2%B4)" title="성자 (삼위일체)">성자</a>가 성부와 함께 성령을 발한다면 <a href="/wiki/%EC%8B%A0%EA%B2%A9" title="신격">신성</a>(神性)의 근원이 둘이 되는 것이다.</li>
<li><a class="mw-redirect" href="/wiki/%EC%84%B1%EC%82%BC%EC%9C%84" title="성삼위">성삼위</a>에 2개의 본질이 성립한다면 <a class="mw-redirect" href="/wiki/%EC%84%B1%EC%82%BC%EC%9C%84%EC%9D%BC%EC%B2%B4" title="성삼위일체">성삼위일체</a>의 교리에 위배되는 것이므로 이는 신앙의 오류이다.</li>
</ul>
<h3><span class="mw-headline" id=".EB.A1.9C.EB.A7.88_.EA.B0.80.ED.86.A8.EB.A6.AD.EA.B5.90.ED.9A.8C">로마 가톨릭교회</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%ED%95%84%EB%A6%AC%EC%98%A4%EC%BC%80_%EB%AC%B8%EC%A0%9C&amp;action=edit&amp;section=6" title="부분 편집: 로마 가톨릭교회">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<ul>
<li>성부와 성자는 공동체적인 사랑안에 서로 결합되어 있다. 따라서 동방교회의 입장은 성삼위의 일치보다 차이를 강조하는 것으로 오히려 삼위일체의 정신에 반한다.</li>
<li>필리오케의 추가는 신경의 변개가 아니라 그 뜻을 강화하는 것으로 (필리오케가 없는) 신경의 원문이나 성경에 반하지 않는다.</li>
<li>성자는 성부와 본질이 같으므로 성부와 성자가 함께 동일한 성령을 발한다는 것이 <a href="/wiki/%EC%A0%9C1%EC%B0%A8_%EB%8B%88%EC%BC%80%EC%95%84_%EA%B3%B5%EC%9D%98%ED%9A%8C" title="제1차 니케아 공의회">제1차 니케아 공의회</a>의 정신에 부합한다.</li>
<li>필리오케는 성삼위일체의 당연한 귀결로서 이를 받아들일 수 없다는 입장이 오히려 신앙의 오류이다.</li>
</ul>
<p><br/>
현재 <a href="/wiki/%EB%8F%99%EB%B0%A9_%EC%A0%95%EA%B5%90%ED%9A%8C" title="동방 정교회">동방 정교회</a>는 필리오케가 없는 신경을, <a href="/wiki/%EB%A1%9C%EB%A7%88_%EA%B0%80%ED%86%A8%EB%A6%AD%EA%B5%90%ED%9A%8C" title="로마 가톨릭교회">로마 가톨릭교회</a> 및 로마 가톨릭으로부터 갈라져 나간 <a href="/wiki/%EC%84%B1%EA%B3%B5%ED%9A%8C" title="성공회">성공회</a> 및 대부분의 <a href="/wiki/%EA%B0%9C%EC%8B%A0%EA%B5%90" title="개신교">개신교</a>는 필리오케가 있는 신경을 받아들이고 있다.</p>
