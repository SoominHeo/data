<p><b>한/글</b>은 <a href="/wiki/%ED%95%9C%EA%B8%80%EA%B3%BC%EC%BB%B4%ED%93%A8%ED%84%B0" title="한글과컴퓨터">한글과컴퓨터</a>에서 개발된 <a href="/wiki/%EC%9B%8C%EB%93%9C_%ED%94%84%EB%A1%9C%EC%84%B8%EC%84%9C" title="워드 프로세서">워드 프로세서</a>이다. 한글 카드나 PC 기종에 구애받던 <a class="new" href="/w/index.php?title=%ED%95%9C%EA%B8%80_%EC%9B%8C%EB%93%9C%ED%94%84%EB%A1%9C%EC%84%B8%EC%84%9C&amp;action=edit&amp;redlink=1" title="한글 워드프로세서 (없는 문서)">한글 워드프로세서</a>의 불편을 해소한 제품이다. 본디 이름은 현대 <a href="/wiki/%ED%95%9C%EA%B5%AD%EC%96%B4" title="한국어">한국어</a>에서 쓰이지 않는 <a class="mw-redirect" href="/wiki/%E1%86%9E" title="ᆞ">ᆞ</a>를 쓴 <b><span class="jamocomposed_block">ᄒᆞᆫ</span>글</b>로 표기하고, ‘HWP(Hangul Word Processor의 준말)’ · ‘<b>아래아 한글</b>’ · ‘하안글’ · ‘한/글’ 등으로 대체하여 표기했으나, 국가 표준인 완성형 한글 코드에선 이 이름을 표현할 수 없기 때문에 <a href="/wiki/2002%EB%85%84" title="2002년">2002년</a>부터는 <b>한/글</b>로도 표기하기로 결정하였다. 한/글은 <a href="/wiki/%EB%A7%88%EC%9D%B4%ED%81%AC%EB%A1%9C%EC%86%8C%ED%94%84%ED%8A%B8_%EC%9B%8C%EB%93%9C" title="마이크로소프트 워드">마이크로소프트 워드</a>와 함께 현재 <a href="/wiki/%EB%8C%80%ED%95%9C%EB%AF%BC%EA%B5%AD" title="대한민국">대한민국</a>에서 가장 많이 쓰이는 워드 프로세스 프로그램이다. <a href="/wiki/%EB%8F%84%EC%8A%A4" title="도스">도스</a>용으로 제작되다가 현재는 주로 <a class="mw-redirect" href="/wiki/%EB%A7%88%EC%9D%B4%ED%81%AC%EB%A1%9C%EC%86%8C%ED%94%84%ED%8A%B8_%EC%9C%88%EB%8F%84" title="마이크로소프트 윈도">마이크로소프트 윈도</a>용으로 개발되고 있으며, <a href="/wiki/%EC%9C%A0%EB%8B%89%EC%8A%A4" title="유닉스">유닉스</a>, <a href="/wiki/%EB%A6%AC%EB%88%85%EC%8A%A4" title="리눅스">리눅스</a>, <a href="/wiki/%EB%A7%A4%ED%82%A8%ED%86%A0%EC%8B%9C" title="매킨토시">매킨토시</a>용으로도 제작되어 있다. 한/글로 작성된 문서 파일은 <a href="/wiki/%ED%8C%8C%EC%9D%BC_%ED%99%95%EC%9E%A5%EC%9E%90" title="파일 확장자">파일 확장자</a>로 hwp를 갖고 있으며, hwt로 저장하여 양식 문서로 저장할 수 있다.</p>

<h3><span class="mw-headline" id="1999.EB.85.84_.EC.9D.B4.EC.A0.84">1999년 이전</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%ED%95%9C/%EA%B8%80&amp;action=edit&amp;section=2" title="부분 편집: 1999년 이전">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<h4><span class="mw-headline" id=".ED.95.9C.2F.EA.B8.80_1.0_.7E_2.1">한/글 1.0 ~ 2.1</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%ED%95%9C/%EA%B8%80&amp;action=edit&amp;section=3" title="부분 편집: 한/글 1.0 ~ 2.1">편집</a><span class="mw-editsection-bracket">]</span></span></h4>
<h5><span class="mw-headline" id="1989.EB.85.84_1.0.2C_1.1.2C_1.2.2C_1.3_.EC.B6.9C.EC.8B.9C">1989년 1.0, 1.1, 1.2, 1.3 출시</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%ED%95%9C/%EA%B8%80&amp;action=edit&amp;section=4" title="부분 편집: 1989년 1.0, 1.1, 1.2, 1.3 출시">편집</a><span class="mw-editsection-bracket">]</span></span></h5>
<p><a href="/wiki/%EC%84%9C%EC%9A%B8%EB%8C%80%ED%95%99%EA%B5%90" title="서울대학교">서울대학교</a>를 졸업한 <a href="/wiki/%EC%9D%B4%EC%B0%AC%EC%A7%84" title="이찬진">이찬진</a>은 <a href="/wiki/1988%EB%85%84" title="1988년">1988년</a> 가을 서울대 컴퓨터연구회에서 만난 <a class="new" href="/w/index.php?title=%EA%B9%80%ED%98%95%EC%A7%91&amp;action=edit&amp;redlink=1" title="김형집 (없는 문서)">김형집</a>, <a class="new" href="/w/index.php?title=%EC%9A%B0%EC%9B%90%EC%8B%9D_(%EA%B8%B0%EC%97%85%EC%9D%B8)&amp;action=edit&amp;redlink=1" title="우원식 (기업인) (없는 문서)">우원식</a>, <a href="/wiki/%EA%B9%80%ED%83%9D%EC%A7%84" title="김택진">김택진</a><sup class="reference" id="cite_ref-1"><a href="#cite_note-1">[1]</a></sup>과 함께 아래아한글 이라는 새로운 워드프로세서를 개발해 베타 버전인 0.9판을 <a href="/wiki/1989%EB%85%84" title="1989년">1989년</a> <a href="/wiki/3%EC%9B%94" title="3월">3월</a>에 발표하였다. 베타버전인 0.9판은 5.25인치 360kb 디스켓 3장으로 이루어져 있으며, 프린터로 출력할 수는 없었다.</p>
<p>최초의 상용버전인 1.0판이 <a href="/wiki/1989%EB%85%84" title="1989년">1989년</a> <a href="/wiki/4%EC%9B%94_24%EC%9D%BC" title="4월 24일">4월 24일</a> 출시되었다. 5.25인치 2D (360KB) <a href="/wiki/%ED%94%8C%EB%A1%9C%ED%94%BC_%EB%94%94%EC%8A%A4%ED%81%AC" title="플로피 디스크">플로피 디스크</a> 3장의 용량으로 만들어진 한/글 1.0판은 종로 전기/전자 상가인 <a href="/wiki/%EC%84%B8%EC%9A%B4%EC%83%81%EA%B0%80" title="세운상가">세운상가</a>의 소규모 유통업체 러브리컴퓨터 라는 회사를 통해 4만 7천 원의 가격으로 출시되었다. 1.0 판의 기능은 다음과 같다.</p>
<ul>
<li><a href="/wiki/%EC%9C%84%EC%A7%80%EC%9C%84%EA%B7%B8" title="위지위그">위지위그</a> 편집 기능<sup class="reference" id="cite_ref-2"><a href="#cite_note-2">[2]</a></sup></li>
<li>8가지 <a href="/wiki/%ED%95%9C%EA%B8%80" title="한글">한글</a>/로마자 <a href="/wiki/%EA%B8%80%EA%BC%B4" title="글꼴">비트맵 글꼴</a> 지원</li>
<li>180 DPI <a href="/wiki/%EB%8F%84%ED%8A%B8_%EB%A7%A4%ED%8A%B8%EB%A6%AD%EC%8A%A4_%ED%94%84%EB%A6%B0%ED%84%B0" title="도트 매트릭스 프린터">도트 매트릭스 프린터</a> 지원</li>
<li>특수 문자 입력 및 선 그리기 기능 등</li>
</ul>
<p>주) 베타버전과 같이 3장으로 이루어진 1.0 버전에서 프린터로 출력할 수 있었는지, 아니면 4,5번 디스켓이 추가된 형태에서 프린터 출력이 가능했는지는 확인이 필요하다</p>
<p><a href="/wiki/1989%EB%85%84" title="1989년">1989년</a> <a href="/wiki/6%EC%9B%94" title="6월">6월</a>에는 주석 처리 기능<sup class="reference" id="cite_ref-3"><a href="#cite_note-3">[3]</a></sup>, <a href="/wiki/%EC%A4%84_%EB%B0%94%EA%BF%88_%EA%B7%9C%EC%B9%99" title="줄 바꿈 규칙">금칙 처리</a> 기능과 9핀 <a href="/wiki/%EB%8F%84%ED%8A%B8_%EB%A7%A4%ED%8A%B8%EB%A6%AD%EC%8A%A4_%ED%94%84%EB%A6%B0%ED%84%B0" title="도트 매트릭스 프린터">도트 매트릭스 프린터</a>에서의 인쇄 품질이 향상된 1.1판이 출시됐다. 또한 1.0판에서는 설명서 대신 몇 장의 안내문만을 제공했던 것에 비해 1.1판 후반 부터는 정식 설명서를 링 파일에 묶어 제공했다. (책자 형태로 되어 있는 정식 설명서의 날짜는 1989년 11월 27일로 되어 있다.)</p>
<p><a href="/wiki/%ED%95%9C%EA%B8%80%EA%B3%BC%EC%BB%B4%ED%93%A8%ED%84%B0" title="한글과컴퓨터">한글과컴퓨터</a>가 설립되기 전인, <a href="/wiki/1990%EB%85%84" title="1990년">1990년</a> <a href="/wiki/1%EC%9B%94" title="1월">1월</a> 한/글의 1.20판이 출시되었다. 1.20판이 출시되기 전에도 1.11판등 버전업이 수시로 있었다</p>
<p>1.2판에서 추가된 기능은 다음과 같다.</p>
<ul>
<li>문서 압축 저장 기능</li>
<li><a class="mw-redirect mw-disambig" href="/wiki/ASCII" title="ASCII">ASCII</a> 텍스트파일 저장 기능 강화</li>
<li>한/영 전환키가 shift+space 키로 변경</li>
<li>선 그리기로 만든 표 보호 기능 등</li>
</ul>
<p><a href="/wiki/1990%EB%85%84" title="1990년">1990년</a> <a href="/wiki/6%EC%9B%94" title="6월">6월</a> 300 DPI <a href="/wiki/%EB%A0%88%EC%9D%B4%EC%A0%80_%ED%94%84%EB%A6%B0%ED%84%B0" title="레이저 프린터">레이저 프린터</a> <a href="/wiki/%ED%95%B4%EC%83%81%EB%8F%84" title="해상도">해상도</a>를 지원하는 한/글 1.2L이 출시되었다.</p>
<p>한/글 1.2판 이후로 한/글은 출력할 수 있는 해상도의 조건에 따라 레이저판, 도트판이 따로 나뉘었으며 300 DPI의 출력물을 찍어낼 수 있는 레이저판은 14만 원, 180 DPI의 출력물을 찍어낼 수 있는 도트판은 5만 원의 가격으로 판매했다. 레이저판에는 불법 복제를 막기 위해 프린터 포트에 꽂아야만 프로그램이 동작하는 하드웨어 키(lock)가 들어 있었는데 후에 한/글 2.5가 출시되면서 제거되었다.</p>
<p><a href="/wiki/1990%EB%85%84" title="1990년">1990년</a> <a href="/wiki/9%EC%9B%94" title="9월">9월</a>에는 글꼴 다운로드 인쇄 방식을 사용해 레이저 프린터에서의 인쇄 속도를 향상시키고, 그림 인쇄, <a class="mw-redirect" href="/wiki/SVGA" title="SVGA">SVGA</a> 모니터 지원 기능이 추가된 한/글 1.3L이 출시되었다.</p>

<p><a href="/wiki/1990%EB%85%84" title="1990년">1990년</a> <a href="/wiki/10%EC%9B%94" title="10월">10월</a> 이찬진이 한글과컴퓨터사를 열었으며, <a href="/wiki/%ED%95%9C%EA%B8%80%EB%AC%B8%ED%99%94%EC%9B%90" title="한글문화원">한글문화원</a> 사무실 내부에 회사가 자리를 잡으면서 본격적으로 <a class="new" href="/w/index.php?title=%EC%A0%95%EB%82%B4%EA%B6%8C&amp;action=edit&amp;redlink=1" title="정내권 (없는 문서)">정내권</a>, <a class="new" href="/w/index.php?title=%EB%B0%95%ED%9D%A5%ED%98%B8&amp;action=edit&amp;redlink=1" title="박흥호 (없는 문서)">박흥호</a> 등 외부 직원을 영입하기 시작했다.</p>
<h6><span class="mw-headline" id=".EB.8F.84.ED.8A.B8.2C.EB.A0.88.EC.9D.B4.EC.A0.80.EC.9A.A9">도트,레이저용</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%ED%95%9C/%EA%B8%80&amp;action=edit&amp;section=7" title="부분 편집: 도트,레이저용">편집</a><span class="mw-editsection-bracket">]</span></span></h6>
<p><a href="/wiki/1990%EB%85%84" title="1990년">1990년</a> <a href="/wiki/12%EC%9B%94_20%EC%9D%BC" title="12월 20일">12월 20일</a>에 1.5판이 도트판과 레이저판의 두 종류로 나뉘어 발매되었다. 도트판의 경우 5.25인치 2D(360KB) <a href="/wiki/%ED%94%8C%EB%A1%9C%ED%94%BC_%EB%94%94%EC%8A%A4%ED%81%AC" title="플로피 디스크">플로피 디스크</a> 다섯 장으로 이뤄져 있었으며, 당시 <a href="/wiki/%EC%BB%B4%ED%93%A8%ED%84%B0" title="컴퓨터">컴퓨터</a>의 대부분이 <a class="mw-redirect" href="/wiki/%ED%95%98%EB%93%9C_%EB%94%94%EC%8A%A4%ED%81%AC" title="하드 디스크">하드 디스크</a>가 없는 <a class="new" href="/w/index.php?title=IBM_PC_XT&amp;action=edit&amp;redlink=1" title="IBM PC XT (없는 문서)">XT</a>나 <a class="mw-redirect" href="/wiki/IBM_AT" title="IBM AT">AT</a> 기종이었기 때문에, 한자 사전을 쓰기 위해 <a class="mw-redirect" href="/wiki/%EB%94%94%EC%8A%A4%EC%BC%93" title="디스켓">디스켓</a>을 바꿔 끼우고, <a href="/wiki/%EC%9D%B8%EC%87%84" title="인쇄">인쇄</a>하기 위해 <a class="mw-redirect" href="/wiki/%EB%94%94%EC%8A%A4%EC%BC%93" title="디스켓">디스켓</a>을 바꿔 끼우는 식으로 요즘 상식으로는 아주 불편하게 제품을 사용해야 했다.</p>
<h6><span class="mw-headline" id=".EC.A2.8B.EC.9D.80_.EC.A0.90">좋은 점</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%ED%95%9C/%EA%B8%80&amp;action=edit&amp;section=8" title="부분 편집: 좋은 점">편집</a><span class="mw-editsection-bracket">]</span></span></h6>

<h5><span class="mw-headline" id="1.51">1.51</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%ED%95%9C/%EA%B8%80&amp;action=edit&amp;section=10" title="부분 편집: 1.51">편집</a><span class="mw-editsection-bracket">]</span></span></h5>
<p>몇 가지 버그를 수정한 1.51판이 <a href="/wiki/1991%EB%85%84" title="1991년">1991년</a> 1월 출시되었다.</p>
<h5><span class="mw-headline" id="1.52">1.52</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%ED%95%9C/%EA%B8%80&amp;action=edit&amp;section=11" title="부분 편집: 1.52">편집</a><span class="mw-editsection-bracket">]</span></span></h5>
<p><a href="/wiki/1991%EB%85%84" title="1991년">1991년</a> <a href="/wiki/11%EC%9B%94_18%EC%9D%BC" title="11월 18일">11월 18일</a> 텍스트 파일을 문단 단위로 읽는 기능과 <a href="/wiki/%ED%94%8C%EB%A1%9C%ED%84%B0" title="플로터">플로터</a> 파일을 인쇄하는 기능을 추가한 한/글 1.52판이 출시되었으며, 한/글 1.53D도 출시되었다. 이들 제품을 통해 한/글은 완성형보다 한글을 많이 쓸 수 있는 조합형 한글,다양한 글꼴, 인쇄기를 이용한 깔끔한 인쇄를 쓸 수 있는터라 <a href="/wiki/%ED%95%9C%EA%B5%AD%EC%9D%B8" title="한국인">한국인</a>이 널리 사용하는 한글 워드 프로세서로 자리를 잡았다. 물론 표 기능이 없어서 선 그리기로 표를 그려야 하는 등 불편한 점이 없지는 않았으나, 1992년 한/글 2.0이 개발되면서 기존 버전의 문제점들이 개선되었다.</p>
<h6><span class="mw-headline" id=".EB.8F.84.EC.8A.A4.EC.9A.A9_.ED.95.9C.EA.B8.80_.EC.9B.8C.EB.93.9C_.ED.94.84.EB.A1.9C.EC.84.B8.EC.84.9C.EB.8A.94_.EA.B7.B8_.EC.8B.9C.EB.8C.80.EC.97.90.EB.8A.94_.EC.B2.A8.EB.8B.A8.EC.9D.B4.EC.97.88.EB.8B.A4">도스용 한글 워드 프로세서는 그 시대에는 첨단이었다</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%ED%95%9C/%EA%B8%80&amp;action=edit&amp;section=12" title="부분 편집: 도스용 한글 워드 프로세서는 그 시대에는 첨단이었다">편집</a><span class="mw-editsection-bracket">]</span></span></h6>
<p>아래아한글 1.53을 비롯한 1.5버전의 한/글을 어떤 <a href="/wiki/%EB%88%84%EB%A6%AC%EA%BE%BC" title="누리꾼">누리꾼</a>들은 윈도용 문서작성기보다 못하다고 낮추어보기도 하지만,이는 시대의 특징을 생각하지 못한 짦은 생각이다. 책만드는 일을 하는 김진섭은 책만들기를 가르쳐주는 책인《책 잘만드는 책》에서 <a href="/wiki/1990%EB%85%84" title="1990년">1990년</a>대에는 도스용 한글 워드 프로세서로 문서를 작성하는 사람은 첨단을 걷는 사람이었다고 했다. 그 실례로 당시 <a href="/wiki/%EC%B4%88%EB%93%B1%ED%95%99%EA%B5%90" title="초등학교">초등학교</a> 고학년 교과서에서는 도스용 한글 워드 프로세서와 손글씨로 <a class="new" href="/w/index.php?title=%ED%95%99%EA%B8%89%EC%8B%A0%EB%AC%B8&amp;action=edit&amp;redlink=1" title="학급신문 (없는 문서)">학급신문</a>을 만드는 이야기가 나왔는데, 한글 워드 프로세서는 첨단을 상징하는 것이었다. 즉, 도스용 한글 워드 프로세서는 윈도용 한글 워드 프로세서에 익숙한 세대들이 볼 때에는 모자른 것이지만, 도스용 한글 워드 프로세서부터 써온 세대에게는 글을 깔끔하게 쓴다는 사실만으로도,당시 학생이었던 이들에게는 숙제를 깔끔하게 써서 낸다는 자부심을 주는 것만으로도 가장 좋은 문서작성 도구였다.</p>
<h5><span class="mw-headline" id="1992.EB.85.84_2.0_.EC.B6.9C.EC.8B.9C">1992년 2.0 출시</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%ED%95%9C/%EA%B8%80&amp;action=edit&amp;section=13" title="부분 편집: 1992년 2.0 출시">편집</a><span class="mw-editsection-bracket">]</span></span></h5>
<p><a href="/wiki/1992%EB%85%84" title="1992년">1992년</a> <a href="/wiki/7%EC%9B%94_27%EC%9D%BC" title="7월 27일">7월 27일</a> 한/글 2.0 전문용이, <a href="/wiki/9%EC%9B%94_21%EC%9D%BC" title="9월 21일">9월 21일</a>에 일반용이 출시되었다. 한/글 2.0은 레이저판과 도트판의 구분을 버리고, 일반용과 전문가용이라는 구분을 달고 출시되었다.</p>
<p>한/글 2.0은 문서 작성 기능만 갖추고 있던 한/글 1.52와는 달리 <a href="/wiki/%EC%9B%8C%EB%93%9C_%ED%94%84%EB%A1%9C%EC%84%B8%EC%84%9C" title="워드 프로세서">워드 프로세서</a>를 이용한 <a href="/wiki/%EC%A0%84%EC%9E%90_%EC%B6%9C%ED%8C%90" title="전자 출판">전자 출판</a>의 가능성을 염두에 두고 다음의 기능들을 추가시켰다.</p>
<ul>
<li>1포인트부터 127포인트까지의 글꼴 가변 크기 조정 기능<sup class="reference" id="cite_ref-4"><a href="#cite_note-4">[4]</a></sup></li>
<li>자간<sup class="reference" id="cite_ref-5"><a href="#cite_note-5">[5]</a></sup>, 장평<sup class="reference" id="cite_ref-6"><a href="#cite_note-6">[6]</a></sup>, 가변 조정 기능 및 문단 모양 편집 지원</li>
<li>글꼴과 문단 모양을 저장하여 사용하는 스타일 지정/적용 기능</li>
<li>한/글 2.0 전문용에서 비트맵 <a href="/wiki/%EA%B8%80%EA%BC%B4" title="글꼴">글꼴</a><sup class="reference" id="cite_ref-7"><a href="#cite_note-7">[7]</a></sup>과 윤곽선 글꼴<sup class="reference" id="cite_ref-8"><a href="#cite_note-8">[8]</a></sup>을 모두 지원</li>
<li>전문용에서 추가 확장팩을 통해 2수준 확장한자(4888자+1만여 자) 지원</li>
<li><a href="/wiki/%EC%88%98%EC%8B%9D" title="수식">수식</a> 편집 기능</li>
<li>표를 작성하고 편집하는 기능, 다단 편집 기능, 글상자, 그림 등의 틀 자유 배치 기능</li>
<li>한글 <a href="/wiki/%EB%A7%9E%EC%B6%A4%EB%B2%95" title="맞춤법">맞춤법</a> 검사 기능</li>
<li><a href="/wiki/%ED%95%98%EC%9D%B4%ED%8D%BC%ED%85%8D%EC%8A%A4%ED%8A%B8" title="하이퍼텍스트">하이퍼텍스트</a> 방식의 도움말 개선</li>
<li><a class="mw-redirect" href="/wiki/%EB%A9%94%EB%AA%A8%EB%A6%AC" title="메모리">메모리</a> 관리 방식의 변화로 최대 편집 쪽수의 증가</li>
<li><a href="/wiki/%EB%A7%88%EC%9A%B0%EC%8A%A4" title="마우스">마우스</a> 지원</li>
</ul>
<p>이 기능들은 <a href="/wiki/IBM" title="IBM">IBM</a> <a href="/wiki/%EA%B0%9C%EC%9D%B8%EC%9A%A9_%EC%BB%B4%ED%93%A8%ED%84%B0" title="개인용 컴퓨터">PC</a>에서 한글을 지원하는 워드프로세서로서는 한/글 2.0이 최초로 구현한 기능들이었다. 이런 획기적인 기능을 추가한 2.0버전의 출시로 당시 <a href="/wiki/%EC%86%8C%ED%94%84%ED%8A%B8%EC%9B%A8%EC%96%B4" title="소프트웨어">소프트웨어</a> 업계의 최고의 스타로서 한글과컴퓨터가 자리잡는 계기가 되었다. 이러한 기능들의 추가로 인해 디스켓의 장수도 이전 버전에 비해 2배 가까이 늘어났다. 물론 <a class="mw-redirect" href="/wiki/%ED%95%98%EB%93%9C_%EB%94%94%EC%8A%A4%ED%81%AC" title="하드 디스크">하드 디스크</a>가 없이는 사용할 수 없었고, 전문가용 버전은 XT 컴퓨터에서 실행되지 않았기 때문에, 당연히 업그레이드 수요가 늘어났다.</p>
<p>이때부터 묵향 등 한/글용 외부 글꼴 패키지가 판매되기 시작했다. 한/글이 인기를 끌면서 IBM PC용 한글 글꼴 시장도 활기를 띠게 되었다. <a href="/wiki/%ED%83%80%EC%9E%90%EA%B8%B0" title="타자기">타자기</a> 출력물과 별 차이 없는 품질로 전자 출판과는 거리가 멀어보였던 한/글 옛 버전과 달리, 수식 편집, 인화지 출력<sup class="reference" id="cite_ref-9"><a href="#cite_note-9">[9]</a></sup>, 편집 가능 쪽수 확대, 윤곽선 글꼴, 다단 편집 등으로 기능을 강화한 한/글 2.0의 등장으로 워드프로세서를 이용한 간이 전자 출판이 IBM PC에서도 어느 정도 가능하게 되었다. 특히 수식 기능은 워드프로세서를 이용한 수학 교재 편집의 가능성을 열어주었다.</p>

<h4><span class="mw-headline" id=".EB.A9.80.ED.8B.B0.EB.AF.B8.EB.94.94.EC.96.B4.EC.99.80_.EC.9C.88.EB.8F.84_.ED.99.98.EA.B2.BD.EC.9D.98_.EB.B3.B4.ED.8E.B8.ED.99.94">멀티미디어와 윈도 환경의 보편화</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%ED%95%9C/%EA%B8%80&amp;action=edit&amp;section=16" title="부분 편집: 멀티미디어와 윈도 환경의 보편화">편집</a><span class="mw-editsection-bracket">]</span></span></h4>
<p><a href="/wiki/%EB%A7%88%EC%9D%B4%ED%81%AC%EB%A1%9C%EC%86%8C%ED%94%84%ED%8A%B8" title="마이크로소프트">마이크로소프트</a>에서 1992년 4월 개발된 <a class="mw-redirect" href="/wiki/%EC%9C%88%EB%8F%84_3.1" title="윈도 3.1">윈도 3.1</a>을 다음해인 1993년 6월 한글 <a class="mw-redirect" href="/wiki/%EC%9C%88%EB%8F%84_3.1" title="윈도 3.1">윈도 3.1</a>로 내놓으면서 멀티미디어 PC/홈 PC라는 개념이 본격적으로 등장하기 시작하였다. 이러한 멀티미디어 PC의 등장, 그리고 그에 따른 <a class="mw-redirect" href="/wiki/%EC%9C%88%EB%8F%84_3.1" title="윈도 3.1">윈도 3.1</a>의 대대적인 보급이 한/글의 미래에도 많은 영향을 미쳤다. 한/글이 갓 2.1판을 출시하고 도스용 2.5판 개발에 들어갔을 즈음, 한글 <a class="mw-redirect" href="/wiki/%EC%9C%88%EB%8F%84_3.1" title="윈도 3.1">윈도 3.1</a>에서 동작하는 즉, <a class="mw-redirect" href="/wiki/%EB%A7%88%EC%9D%B4%ED%81%AC%EB%A1%9C%EC%86%8C%ED%94%84%ED%8A%B8_%EC%9C%88%EB%8F%84" title="마이크로소프트 윈도">윈도</a>의 장점을 살린 여러 가지 <a href="/wiki/%ED%95%9C%EA%B8%80" title="한글">한글</a> 워드프로세서가 등장하기 시작하였다. 당시 한메소프트의 <a class="new" href="/w/index.php?title=%ED%8C%8C%ED%94%BC%EB%A3%A8%EC%8A%A4_(%EC%9B%8C%EB%93%9C_%ED%94%84%EB%A1%9C%EC%84%B8%EC%84%9C)&amp;action=edit&amp;redlink=1" title="파피루스 (워드 프로세서) (없는 문서)">파피루스</a>, 금성소프트의 윈워드, <a href="/wiki/%EB%A7%88%EC%9D%B4%ED%81%AC%EB%A1%9C%EC%86%8C%ED%94%84%ED%8A%B8" title="마이크로소프트">마이크로소프트</a>의 <a href="/wiki/%EB%A7%88%EC%9D%B4%ED%81%AC%EB%A1%9C%EC%86%8C%ED%94%84%ED%8A%B8_%EC%9B%8C%EB%93%9C" title="마이크로소프트 워드">워드</a> 5.0, <a href="/wiki/%EC%82%BC%EC%84%B1%EC%A0%84%EC%9E%90" title="삼성전자">삼성전자</a>의 <a href="/wiki/%ED%9B%88%EB%AF%BC%EC%A0%95%EC%9D%8C_(%EC%86%8C%ED%94%84%ED%8A%B8%EC%9B%A8%EC%96%B4)" title="훈민정음 (소프트웨어)">훈민정음</a>, <a href="/wiki/%ED%95%B8%EB%94%94%EC%86%8C%ED%94%84%ED%8A%B8" title="핸디소프트">핸디소프트</a>의 <a class="mw-redirect" href="/wiki/%EB%A7%88%EC%9D%B4%ED%81%AC%EB%A1%9C%EC%86%8C%ED%94%84%ED%8A%B8_%EC%9C%88%EB%8F%84" title="마이크로소프트 윈도">윈도</a>용 워드프로세서는 한/글 3.0b 윈도용이 등장하기까지 춘추전국시대를 맞이하게 되었다.</p>
<p>다른 업체들은 이미 <a class="mw-redirect" href="/wiki/%EC%9C%88%EB%8F%84_3.1" title="윈도 3.1">윈도 3.1</a>용 워드프로세서를 내놓고, 한/글에서 불가능한 멀티태스킹(다중 작업), OLE(다른 프로그램에서 만든 개체의 삽입 수정), 3차원 글꼴 다듬기(글맵시/워드아트), <a class="mw-redirect" href="/wiki/%EB%A7%88%EC%9D%B4%ED%81%AC%EB%A1%9C%EC%86%8C%ED%94%84%ED%8A%B8_%EC%9C%88%EB%8F%84" title="마이크로소프트 윈도">윈도</a>용 <a href="/wiki/%ED%8A%B8%EB%A3%A8%ED%83%80%EC%9E%85" title="트루타입">트루타입</a> 글꼴(화면의 글꼴 모양과 인쇄한 글꼴 모양이 동일한 서체)<sup class="reference" id="cite_ref-13"><a href="#cite_note-13">[13]</a></sup> 호환, 9개까지의 다중 문서 편집(한/글은 그때까지 2개의 문서만 동시에 열 수 있었음) 등을 장점으로 내세우며 나름대로의 시장을 차지하고 있을 즈음, 한글과컴퓨터는 여전히 도스용 워드프로세서인 한/글 2.5를 개발하면서 시간을 천천히 끌어갔다. 물론 <a class="mw-redirect" href="/wiki/%EB%A7%88%EC%9D%B4%ED%81%AC%EB%A1%9C%EC%86%8C%ED%94%84%ED%8A%B8_%EC%9C%88%EB%8F%84" title="마이크로소프트 윈도">윈도</a>용 한/글도 동시에 개발하고 있다고 밝히기는 했지만, 아직까지 속도나 안정성 측면에서 부족했던 <a class="mw-redirect" href="/wiki/%EB%A7%88%EC%9D%B4%ED%81%AC%EB%A1%9C%EC%86%8C%ED%94%84%ED%8A%B8_%EC%9C%88%EB%8F%84" title="마이크로소프트 윈도">윈도</a> 3.1용 워드프로세서의 개발에 적극적으로 참여하지는 않았다.</p>
<h4><span class="mw-headline" id=".ED.95.9C.2F.EA.B8.80_2.5_.7E_3.0b">한/글 2.5 ~ 3.0b</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%ED%95%9C/%EA%B8%80&amp;action=edit&amp;section=17" title="부분 편집: 한/글 2.5 ~ 3.0b">편집</a><span class="mw-editsection-bracket">]</span></span></h4>
<h5><span class="mw-headline" id="1994.EB.85.84_2.5_.EB.B0.9C.ED.91.9C">1994년 2.5 발표</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%ED%95%9C/%EA%B8%80&amp;action=edit&amp;section=18" title="부분 편집: 1994년 2.5 발표">편집</a><span class="mw-editsection-bracket">]</span></span></h5>
<p><a href="/wiki/1994%EB%85%84" title="1994년">1994년</a> SEK 전시회에서 한/글 2.5가 발표됐다.</p>
<ul>
<li>덧실행(add-on) 프로그램 지원<sup class="reference" id="cite_ref-14"><a href="#cite_note-14">[14]</a></sup></li>

<li>인쇄 속도 개선 및 모아찍기<sup class="reference" id="cite_ref-15"><a href="#cite_note-15">[15]</a></sup>, 나눠 찍기<sup class="reference" id="cite_ref-16"><a href="#cite_note-16">[16]</a></sup> 등의 변형 인쇄 기능 지원</li>

<li>신명조를 기본글꼴로 사용함.</li>

<p>2.5판부터는 전문용/일반용 구분이 없어진 대신 2.1 전문용을 업그레이드 한 2.5 기본판이 12만원에 판매되면서 가격을 50% 정도 인하했다. 또한 <a class="mw-redirect" href="/wiki/%EB%A7%88%EC%9D%B4%ED%81%AC%EB%A1%9C%EC%86%8C%ED%94%84%ED%8A%B8_%EC%9C%88%EB%8F%84" title="마이크로소프트 윈도">마이크로소프트 윈도</a>로의 업그레이드를 고려하는 사용자를 위해 한/글 2.5 구입자들에게는 윈도용 한/글 3.0 무료 교환쿠폰이 제공되었다. 또한 <a href="/wiki/%EC%84%9C%EC%9A%B8_1000%EB%85%84_%ED%83%80%EC%9E%84%EC%BA%A1%EC%8A%90" title="서울 1000년 타임캡슐">서울 정도 600주년 기념 타임캡슐</a>에 한/글 2.5가 포함되었다.</p>
<p>한/글은 2.0 버전 이후부터 미국의 <a href="/wiki/%EC%9B%8C%EB%93%9C%ED%8D%BC%ED%8E%99%ED%8A%B8" title="워드퍼펙트">워드퍼펙트</a>라는 워드프로세서를 많이 참조하였다고 전해진다. 기능의 구현이나, 조판 방식 등에서 많은 유사점이 있다고 한다. <a href="/wiki/%EB%A7%88%EC%9D%B4%ED%81%AC%EB%A1%9C%EC%86%8C%ED%94%84%ED%8A%B8_%EC%9B%8C%EB%93%9C" title="마이크로소프트 워드">마이크로소프트 워드</a> 출시 전까지 미국에서 최고의 시장 점유율을 가졌던 <a class="mw-redirect" href="/wiki/%EC%9B%8C%EB%93%9C%ED%94%84%EB%A1%9C%EC%84%B8%EC%84%9C" title="워드프로세서">워드프로세서</a>인 워드퍼펙트와의 제휴를 통해 만들어진 윈도용 한/글 3.0이 몇 달 뒤 출시되었다.</p>
<h5><span class="mw-headline" id="1995.EB.85.84_3.0_.EB.B0.9C.ED.91.9C">1995년 3.0 발표</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%ED%95%9C/%EA%B8%80&amp;action=edit&amp;section=19" title="부분 편집: 1995년 3.0 발표">편집</a><span class="mw-editsection-bracket">]</span></span></h5>
<h6><span class="mw-headline" id=".EB.AA.A8.EC.8A.B5">모습</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%ED%95%9C/%EA%B8%80&amp;action=edit&amp;section=20" title="부분 편집: 모습">편집</a><span class="mw-editsection-bracket">]</span></span></h6>
<p>한/글 3.0판 윈도용이 <a href="/wiki/1995%EB%85%84" title="1995년">1995년</a> <a href="/wiki/3%EC%9B%94" title="3월">3월</a> 출시되었다. 윈도용 한/글의 첫 출시는 윈도 95 출시와 함께 그 해 소프트웨어 시장에서 가장 큰 이슈가 되었다. 윈도용 한/글은 지금까지 등장했던 다른 윈도용 한글 워드프로세서와는 달리, 운영체제에 의존하지 않는 독립 인터페이스, 독립 입출력 방식을 채택하였다. 윈도용 한/글 3.0은 윈도 3.1의 기본 프로그램 모양을 따르지 않고, 넥스트스텝과 윈도 95의 인터페이스를 혼합한 형태의 프로그램 모양을 선보였다.</p>
<h6><span class="mw-headline" id=".ED.95.9C.EC.BB.B42.EB.B0.94.EC.9D.B4.ED.8A.B8.EC.BD.94.EB.93.9C">한컴2바이트코드</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%ED%95%9C/%EA%B8%80&amp;action=edit&amp;section=21" title="부분 편집: 한컴2바이트코드">편집</a><span class="mw-editsection-bracket">]</span></span></h6>
<p>또한 완성형 한글 코드만을 지원하는 윈도 3.1의 입력 체계를 사용하지 않고, <a href="/wiki/%EB%8F%84%EC%8A%A4" title="도스">도스</a>용 한/글의 입력 체계, 즉 조합형 한글 코드를 변형한 <a class="new" href="/w/index.php?title=%ED%95%9C%EC%BB%B4_2%EB%B0%94%EC%9D%B4%ED%8A%B8_%EC%BD%94%EB%93%9C&amp;action=edit&amp;redlink=1" title="한컴 2바이트 코드 (없는 문서)">한컴 2바이트 코드</a>를 사용으로써, 11,172자 현대 <a href="/wiki/%ED%95%9C%EA%B8%80" title="한글">한글</a>은 물론 <a href="/wiki/%EC%98%9B%ED%95%9C%EA%B8%80" title="옛한글">옛한글</a>까지 표현할 수 있게 되었다. 예전부터 조합형 한글 체계를 통한 한글의 완전한 표현을 장점으로 내세웠던 한/글이니만큼 당연한 선택이라고 할 수 있다. 그리고 기존 도스용 한/글의 글꼴을 그대로 사용할 수 있도록 만들었고, 단축키도 도스용 한/글의 것을 그대로 받아들임으로서, 도스용 한/글에 익숙한 사용자들이 손쉽게 한/글을 익힐 수 있도록 하였다. 물론 기존 도스용 한/글의 문서를 아무런 손상없이 그대로 읽을 수 있다는 장점도 있었다.</p>
<h6><span class="mw-headline" id=".EC.A2.8B.EC.9D.80_.EC.A0.90.EA.B3.BC_.EB.82.98.EC.81.9C_.EC.A0.90">좋은 점과 나쁜 점</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%ED%95%9C/%EA%B8%80&amp;action=edit&amp;section=22" title="부분 편집: 좋은 점과 나쁜 점">편집</a><span class="mw-editsection-bracket">]</span></span></h6>
<p>이렇게 기존 도스용 한/글의 커널을 그대로 윈도용으로 컨버전한 한/글 3.0은 도스용 한/글의 장점 위에 멀티태스킹, OLE 기능, 윈도용 글꼴 및 프린터 드라이버 지원, 글맵시 기능 등 윈도용 워드프로세서로서 꼭 필요한 기능을 추가함으로써 빠른 속도로 도스용 한/글 사용자를 끌어들이기 시작한다. 하지만 한/글 3.0에는 버그가 굉장히 많았다. 이는 32비트 코드로 작성된 한/글 3.0이 16비트 운영체제인 윈도 3.1에서 돌아가기 위해 필요한 <a href="/wiki/Win32s" title="Win32s">Win32s</a> 런타임이 불안정했기 때문으로, <a class="mw-disambig" href="/wiki/%ED%8C%A8%EC%B9%98" title="패치">패치</a> 버전인 한/글 3.0a에서는 어느 정도 문제가 해결되었지만, 윈도 95가 출시되기 이전까지는 많은 불만의 대상이 되었다.</p>
<h6><span class="mw-headline" id=".EA.B0.80.EA.B2.A9">가격</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%ED%95%9C/%EA%B8%80&amp;action=edit&amp;section=23" title="부분 편집: 가격">편집</a><span class="mw-editsection-bracket">]</span></span></h6>
<p>가격은 기본팩이 12만원으로 책정되어 2.5와 동일했으며, 2.5의 확장팩 내용을 담은 추가 CD가 4만원의 가격으로 추가 인하되었다.</p>
<h5><span class="mw-headline" id=".EB.A7.88.EC.A7.80.EB.A7.89_.EB.8F.84.EC.8A.A4.EC.9A.A9_.ED.95.9C.2F.EA.B8.80.EC.9D.B8_.ED.95.9C.2F.EA.B8.80_3.0_.EB.B0.9C.ED.91.9C">마지막 도스용 한/글인 한/글 3.0 발표</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%ED%95%9C/%EA%B8%80&amp;action=edit&amp;section=24" title="부분 편집: 마지막 도스용 한/글인 한/글 3.0 발표">편집</a><span class="mw-editsection-bracket">]</span></span></h5>
<p>그리고 도스용 한/글의 마지막 버전인 한/글 3.0이 <a href="/wiki/1995%EB%85%84" title="1995년">1995년</a> <a href="/wiki/6%EC%9B%94" title="6월">6월</a> 출시되었다. 한/글 3.0 도스용에서 추가된 기능은 다음과 같았다.</p>
<ul>
<li>윈도용 한/글 3.0에서 저장한 파일 열기 지원(단, 글맵시 등은 불러오지 못함)</li>
<li>인터페이스 개선 및 도구 상자 추가</li>
<li>한/영 자동 변환 기능 및 빠른 교정 기능</li>
<li>문서 내 하이퍼 링크(하이퍼텍스트) 기능</li>
<li>덧실행 기능이 더 강화되어 <a href="/wiki/PC%ED%86%B5%EC%8B%A0" title="PC통신">PC통신</a> 에뮬레이터, 공학용 계산기, CD 플레이어 등을 한/글 내부에서 실행할 수 있도록 함.</li>
<li>수채화라는 그래픽 편집/드로잉 프로그램을 도스용 한/그림이라는 이름으로 번들로 제공</li>
<li>타자 연습 프로그램 제공</li>
</ul>
<p>가격은 기본팩 + CD-ROM이 12만 원의 가격으로 판매되었으며, 개정판인 한글 3.01 도스용은 한/글 오피스 96에 포함되었다.</p>
<h5><span class="mw-headline" id="3.0b_.EB.B0.9C.ED.91.9C">3.0b 발표</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%ED%95%9C/%EA%B8%80&amp;action=edit&amp;section=25" title="부분 편집: 3.0b 발표">편집</a><span class="mw-editsection-bracket">]</span></span></h5>
<p><a href="/wiki/1995%EB%85%84" title="1995년">1995년</a> <a href="/wiki/8%EC%9B%94" title="8월">8월</a> <a class="mw-redirect" href="/wiki/%EC%9C%88%EB%8F%84_95" title="윈도 95">윈도 95</a>가 출시된 뒤 <a href="/wiki/1995%EB%85%84" title="1995년">1995년</a> <a href="/wiki/11%EC%9B%94" title="11월">11월</a> 한글 <a class="mw-redirect" href="/wiki/%EC%9C%88%EB%8F%84_95" title="윈도 95">윈도 95</a>가 출시된 직후, 도스용 한/글 3.0의 업그레이드 버전이자 최초의 윈도 버전인 3.0b가 출시되었다. 기본 내용은 도스용 한/글 3.0과 동일하지만 윈도 95에서도 실행이 가능해졌고, 여러 가지 버그가 수정되었으며, 온페이지 드로잉 기능이 추가되었다. 윈도 3.1에서의 한/글의 여러 가지 문제점이 윈도 95에서는 대부분 해결됨으로써 이 때를 기점으로 한/글이 윈도에서 완전히 자리를 잡게 되었다.</p>
<p>한/글 3.0b가 대중적인 성공을 얻고 난 후, 한/글은 윈도용 워드프로세서로 완전히 성격을 탈바꿈했다. 한글과컴퓨터는 1996년 출시할 예정이었던 한/글 4.0(가칭) 도스용 개발을 취소하고, 더 이상의 도스용 버전 개발을 중단하기로 결정했다.</p>
<h4><span class="mw-headline" id=".EB.A7.88.EC.9D.B4.ED.81.AC.EB.A1.9C.EC.86.8C.ED.94.84.ED.8A.B8_.EC.98.A4.ED.94.BC.EC.8A.A4.EC.9D.98_.EA.B3.B5.EB.9E.B5.EA.B3.BC_.ED.95.9C.2F.EA.B8.80_.EC.98.A4.ED.94.BC.EC.8A.A4">마이크로소프트 오피스의 공략과 한/글 오피스</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%ED%95%9C/%EA%B8%80&amp;action=edit&amp;section=26" title="부분 편집: 마이크로소프트 오피스의 공략과 한/글 오피스">편집</a><span class="mw-editsection-bracket">]</span></span></h4>
<p>윈도 95 출시이후, <a href="/wiki/%EB%A7%88%EC%9D%B4%ED%81%AC%EB%A1%9C%EC%86%8C%ED%94%84%ED%8A%B8_%EC%98%A4%ED%94%BC%EC%8A%A4" title="마이크로소프트 오피스">마이크로소프트 오피스</a>의 국내 기업체 공략이 거세지기 시작했다. 1980년대 미국에서 널리 쓰였던 <a href="/wiki/%EB%A1%9C%ED%84%B0%EC%8A%A4_1-2-3" title="로터스 1-2-3">로터스 1-2-3</a>와 <a href="/wiki/%EC%9B%8C%EB%93%9C%ED%8D%BC%ED%8E%99%ED%8A%B8" title="워드퍼펙트">워드퍼펙트</a>를 1990년부터 이겨낸 마이크로소프트 오피스는 95 버전까지만 해도 한글판의 불완전한 번역과 잦은 버그, 한/글 문서와의 비호환 문제 등으로 고전을 면치 못하다가 97 버전에서부터 VBA를 이용한 특유의 확장성, 네트워크를 통한 다중 사용자의 작업 공유 기능, 강화된 도움말, 다량의 클립아트와 글꼴 번들, 여러가지 사용자 편의 기능, 온페이지 드로잉 기능의 강화, 전 세계 문서 호환, <a href="/wiki/%EB%A7%88%EC%9D%B4%ED%81%AC%EB%A1%9C%EC%86%8C%ED%94%84%ED%8A%B8_%EC%97%91%EC%85%80" title="마이크로소프트 엑셀">엑셀</a>, <a href="/wiki/%EB%A7%88%EC%9D%B4%ED%81%AC%EB%A1%9C%EC%86%8C%ED%94%84%ED%8A%B8_%ED%8C%8C%EC%9B%8C%ED%8F%AC%EC%9D%B8%ED%8A%B8" title="마이크로소프트 파워포인트">파워포인트</a>와의 통합 등을 앞세워 점차 기업체를 중심으로 사용자 수를 늘려가기 시작하였다. 그 뿐 아니라 삼성전자에서 개발한 훈민정음 등도 <a class="mw-redirect" href="/wiki/%EC%9B%8C%EB%93%9C%ED%94%84%EB%A1%9C%EC%84%B8%EC%84%9C" title="워드프로세서">워드프로세서</a>시장에서의 영향력을 넓혀가면서, 미처 기업 시장을 체계적으로 준비하지 못했던 한글과컴퓨터의 약점이 서서히 드러나기 시작했다.</p>
<p>한편 이때 한글과컴퓨터도 한/글을 기반으로 한컴 오피스 스위트(Haancom Office Suite)를 출시했는데, 이 첫 주자는 한/글 3.0b의 초기판에 저가로 번들(한/글 3.0b 구입자에 한해 3만원의 가격에 판매)된 윈도용 <a href="/wiki/%EB%A1%9C%ED%84%B0%EC%8A%A4_1-2-3" title="로터스 1-2-3">로터스 1-2-3</a>(스프레드시트)와 프리랜스 그래픽스(프레젠테이션 소프트웨어)였다. 물론 서로 다른 회사의 제품을 패키지만 통합한 것인 만큼, 호환성에 약간에 문제가 있었고, 로터스 1-2-3, 프리랜스 그래픽스가 이미 마이크로소프트 오피스와의 경쟁에서 밀려나 버린 제품들이라 그 파급력이 덜했지만 한컴이 마이크로소프트 오피스와의 경쟁을 위해 어느 정도 준비를 해나가고 있었음을 보여주는 작품들이었다.</p>
<h4><span class="mw-headline" id=".ED.95.9C.2F.EA.B8.80_96.EC.9D.98_.EC.B6.9C.EC.8B.9C.EC.99.80_.ED.95.9C.EA.B8.80.EA.B3.BC.EC.BB.B4.ED.93.A8.ED.84.B0.EC.9D.98_.EB.B6.80.EC.A7.84">한/글 96의 출시와 한글과컴퓨터의 부진</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%ED%95%9C/%EA%B8%80&amp;action=edit&amp;section=27" title="부분 편집: 한/글 96의 출시와 한글과컴퓨터의 부진">편집</a><span class="mw-editsection-bracket">]</span></span></h4>
<h5><span class="mw-headline" id=".ED.95.9C.2F.EA.B8.80_.ED.94.84.EB.A1.9C_96">한/글 프로 96</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%ED%95%9C/%EA%B8%80&amp;action=edit&amp;section=28" title="부분 편집: 한/글 프로 96">편집</a><span class="mw-editsection-bracket">]</span></span></h5>
<p>그리고 그 다음해인 1996년 7월에는 다양한 소프트웨어와의 통합패키지로 선보인 한/글 프로 96과 한/글 오피스 96이 출시되었다. 한/글 프로 96은 기본적으로 사용자 인터페이스를 비롯해 <a href="/wiki/%EC%9D%B8%ED%84%B0%EB%84%B7" title="인터넷">인터넷</a>문서 편집,문서모음집인 문서마당 등 다양한 기능이 업그레이드된 워드프로세서인 한/글 96을 기반으로, 주소록, 타자연습, 폼프로세서 틀마름이, 팩스 프로그램, 메일 프로그램 등 다양한 <a class="mw-disambig" href="/wiki/%EC%9C%A0%ED%8B%B8%EB%A6%AC%ED%8B%B0" title="유틸리티">유틸리티</a>를 추가한 종합 패키지라고 할 수 있었다. 그리고 이에 더해 한/글 오피스 96에는 이전에 선보였던 <a href="/wiki/%EB%A1%9C%ED%84%B0%EC%8A%A4_1-2-3" title="로터스 1-2-3">로터스 1-2-3</a>와 프리랜스 그래픽스의 업그레이드 판이 포함되었다.</p>
<h5><span class="mw-headline" id=".ED.95.9C.2F.EA.B8.80_.EC.98.A4.ED.94.BC.EC.8A.A4_96">한/글 오피스 96</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%ED%95%9C/%EA%B8%80&amp;action=edit&amp;section=29" title="부분 편집: 한/글 오피스 96">편집</a><span class="mw-editsection-bracket">]</span></span></h5>
<p>하지만 대중적으로 어느 정도 성공을 거둔 한/글 프로 96과는 달리 기업체 판매를 목적으로 출시된 한/글 오피스 96은 포함된 로터스 1-2-3 5.0과 프리랜스 그래픽스 프로그램의 기능 미비로 인해, MS 오피스에 비해 별다른 반향을 얻지 못했다. 로터스에서 제작한 프로그램과 한/글 96간의 인터페이스 차이도 이용자의 불만을 샀다. 그나마 성공을 거둔 한/글 프로 96도 개인용 시장의 포화와 심각한 <a href="/wiki/%EB%B6%88%EB%B2%95_%EB%B3%B5%EC%A0%9C" title="불법 복제">불법 복제</a> 등의 영향으로 이전 수준의 성공을 거뒀을 뿐 더 이상의 성장이 정체되었다. 한/글을 제외하면 한/그림(그림 그리기), 한/아름, 한/맥 등 출시한 여러 제품이 모두 실패했을 뿐 아니라 새롭게 진출한 인터넷 시장에서도 <a href="/wiki/%EA%B2%80%EC%83%89_%EC%97%94%EC%A7%84" title="검색 엔진">검색 엔진</a>인 <a href="/wiki/%EC%8B%AC%EB%A7%88%EB%8B%88" title="심마니">심마니</a>를 빼고 별다른 실적을 낳지 못했다.</p>
<h4><span class="mw-headline" id=".ED.95.9C.2F.EA.B8.80_97">한/글 97</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%ED%95%9C/%EA%B8%80&amp;action=edit&amp;section=30" title="부분 편집: 한/글 97">편집</a><span class="mw-editsection-bracket">]</span></span></h4>
<h5><span class="mw-headline" id=".EC.B6.9C.EC.8B.9C">출시</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%ED%95%9C/%EA%B8%80&amp;action=edit&amp;section=31" title="부분 편집: 출시">편집</a><span class="mw-editsection-bracket">]</span></span></h5>
<p><a href="/wiki/1996%EB%85%84" title="1996년">1996년</a> <a href="/wiki/11%EC%9B%94" title="11월">11월</a>에는 <a class="new" href="/w/index.php?title=%ED%95%9C/%EA%B8%80_96&amp;action=edit&amp;redlink=1" title="한/글 96 (없는 문서)">한/글 96</a>을 기반으로 <a href="/wiki/%EC%A4%91%EA%B5%AD%EC%96%B4" title="중국어">중국어</a>, <a href="/wiki/%EC%9D%BC%EB%B3%B8%EC%96%B4" title="일본어">일본어</a> 인터페이스 및 다국어 입력기를 채용한 한/글 국제판과 <a href="/wiki/%EC%9D%BC%EB%B3%B8" title="일본">일본</a> 시장을 목표로 한 한/글 일본판이 출시되었고, <a href="/wiki/1997%EB%85%84" title="1997년">1997년</a>에는 한/글 97을 탑재한 <a class="new" href="/w/index.php?title=%ED%95%9C%EC%BB%B4%ED%99%88_97&amp;action=edit&amp;redlink=1" title="한컴홈 97 (없는 문서)">한컴홈 97</a>과 <a class="mw-redirect" href="/wiki/%ED%95%9C%EC%BB%B4_%EC%98%A4%ED%94%BC%EC%8A%A4_97" title="한컴 오피스 97">한컴 오피스 97</a>이 출시되었으나, 판매 실적은 기대에 미치지 못하였다. 당시 한컴에서는 한컴오피스 97을 <a class="mw-redirect" href="/wiki/%EB%8C%80%ED%95%99%EC%83%9D" title="대학생">대학생</a>들을 위한 아카데믹 버전으로도 출시하는 등 사용자의 구매력에 맞추는 방식의 <a href="/wiki/%EC%98%81%EC%97%85" title="영업">영업</a>을 사용하였다.</p>
<h5><span class="mw-headline" id=".ED.95.9C.2F.EA.B8.80_8.15">한/글 8.15</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%ED%95%9C/%EA%B8%80&amp;action=edit&amp;section=32" title="부분 편집: 한/글 8.15">편집</a><span class="mw-editsection-bracket">]</span></span></h5>
<p><a href="/wiki/1998%EB%85%84" title="1998년">1998년</a> 6월에는 한글과컴퓨터가 마이크로소프트의 투자를 받는 조건으로 한/글 소스코드를 마이크로소프트에 넘기고, 더 이상의 한/글 개발을 중단하기로 하는 양해 각서를 마이크로소프트와 체결해 큰 사회적 파장을 일으켰다. 사용자의 반대 여론에 부딪친 한글과컴퓨터는 이 계약을 취소하는 조건으로 한글지키기운동본부로부터 100억원 투자 유치를 받아 한/소프트 회원유치운동과 함께 한/글 815버전을 출시했다. 한글과컴퓨터의 이찬진 사장은 회사 실적 악화의 책임을 지고 한글과컴퓨터의 사장직에서 물러났으며 지오이월드 대표이사였던 전하진이 신임 대표 이사로 선임되었다.</p>
<h5><span class="mw-headline" id=".EC.99.84.EC.84.B1.EB.8F.84">완성도</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%ED%95%9C/%EA%B8%80&amp;action=edit&amp;section=33" title="부분 편집: 완성도">편집</a><span class="mw-editsection-bracket">]</span></span></h5>
<p>한/글 97은 '실행 취소' 기능이 매우 취약했지만 역대 윈도용 한/글 시리즈 중 가장 완성도가 높았으며, 가장 높은 보급율을 보여준 제품이었다. 또한 <a href="/wiki/%EB%A7%A4%ED%82%A8%ED%86%A0%EC%8B%9C" title="매킨토시">매킨토시</a>용으로도 개발되어 맥사용자의 대부분을 차지하는 <a href="/wiki/%EC%A0%84%EC%9E%90_%EC%B6%9C%ED%8C%90" title="전자 출판">전자 출판</a> 종사자가 <a href="/wiki/%EA%B0%9C%EC%9D%B8%EC%9A%A9_%EC%BB%B4%ED%93%A8%ED%84%B0" title="개인용 컴퓨터">PC</a>에서 작성된 원고 내용을 <a class="mw-redirect" href="/wiki/%ED%95%9C/%EA%B8%80_97" title="한/글 97">한/글 97</a>로 작업했다면 바로 확인할 수 있게 되었다.<sup class="reference" id="cite_ref-18"><a href="#cite_note-18">[18]</a></sup></p>
<h5><span class="mw-headline" id=".EC.9B.8C.EB.94.94.EC.95.88.EA.B3.BC.EC.9D.98_.ED.98.B8.ED.99.98.EC.84.B1_.EB.AC.B8.EC.A0.9C">워디안과의 호환성 문제</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%ED%95%9C/%EA%B8%80&amp;action=edit&amp;section=34" title="부분 편집: 워디안과의 호환성 문제">편집</a><span class="mw-editsection-bracket">]</span></span></h5>
<p>후속 제품인 워디안이 출시되었는데도 한/글 97과의 호환성 문제와 커다란 인터페이스의 변화로 인해 한/글이 가지고 있던 독창적인 요소들을 상당수 잃게 되어 사용자로 하여금 한/글 97에서 신버전으로 교체를 꺼리게 하였고, 한/글 97 사용자들이 상위 버전인 워디안의 문서를 읽지 못했기 때문에 이미 보급되어 있는 한/글 97 사용자를 위해 상위 버전의 사용을 꺼리게 되는 결과를 낳게 되었다.이러한 인터페이스의 무리한 변화와 호환 불능은 그 이전의 한/글에서는 없던 일이었다. 예를 들어 한/글 2.1과 한/글 2.5는 그리고 한/글 2.5과 한/글 3.0(도스용)는 인터페이스가 바뀌지 않아서 새로운 버전을 쉽게 배울 수 있었고, 윈도용 한/글 96문서를 글맵시등의 윈도 고유의 기능을 빼고는 한/글 3.0에서 읽었으므로 정보를 쉽게 주고받았다.</p>
<h5><span class="mw-headline" id=".EB.A6.AC.EB.88.85.EC.8A.A4">리눅스</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%ED%95%9C/%EA%B8%80&amp;action=edit&amp;section=35" title="부분 편집: 리눅스">편집</a><span class="mw-editsection-bracket">]</span></span></h5>
<p>한/글 국제판을 통합한 한/글 97 기능 강화판도 출시되었고 한/글 815 버전 구입자들에게는 정가 19,800원, 처음 사용자에게는 60,000원에 판매되었다. 또한 리눅스 개발 업체인 <a href="/wiki/%EB%AF%B8%EC%A7%80%EB%A6%AC%EC%84%9C%EC%B9%98" title="미지리서치">미지리서치</a>에서는 한/글 96을 리눅스용으로 포팅한 한/글 for X R4를 출시하기도 하였다.</p>
<h5><span class="mw-headline" id=".EA.B8.B0.EB.8A.A5">기능</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%ED%95%9C/%EA%B8%80&amp;action=edit&amp;section=36" title="부분 편집: 기능">편집</a><span class="mw-editsection-bracket">]</span></span></h5>
<p>한/글 97에서 제공된 기능은 다음과 같았다.<sup class="reference" id="cite_ref-19"><a href="#cite_note-19">[19]</a></sup></p>
<ul>
<li><a href="/wiki/%ED%95%9C%EA%B8%80_%EB%A7%9E%EC%B6%A4%EB%B2%95" title="한글 맞춤법">한글 맞춤법</a> 검사 기능 강화: 한/글 97에서는 맞춤법에 어긋나는 문장이나 단어에 붉은 밑줄이 그어지게 하여, 맞춤법 기능을 강화하였다.</li>
<li>손가락 아이콘을 사용한 기능 구분: 한/글 97에서는 대화 상자에서 기본 기능과 세부 기능을 모두 접기와 펴기 아이콘으로 구분하였다. 손가락 모양의 아이콘을 누르면 세부 기능이 나타나도록 한 것이다.</li>
<li>표 도구 모음 기능: 한/글 97에서는 도구 모음으로 표작성과 편집을 하도록 기능을 추가하였다.</li>
</ul>
<h3><span class="mw-headline" id="2000.EB.85.84.EB.8C.80">2000년대</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%ED%95%9C/%EA%B8%80&amp;action=edit&amp;section=37" title="부분 편집: 2000년대">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<h4><span class="mw-headline" id=".ED.95.9C.2F.EA.B8.80_.EC.9B.8C.EB.94.94.EC.95.88">한/글 워디안</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%ED%95%9C/%EA%B8%80&amp;action=edit&amp;section=38" title="부분 편집: 한/글 워디안">편집</a><span class="mw-editsection-bracket">]</span></span></h4>
<p>2000년 출시된 제품으로 한/글 2.1 이후, 한/글 97까지는 커널 기반에서 커다란 변화가 없었다. 이로 인해 파일의 호환성도 어느 정도 유지되었고, 사용법에도 큰 변화가 없는 모습을 보여주었으나, 한/글 워디안은 모든 소스 코드를 새로 작성함으로써 이전 버전과는 다른 모습을 보여주게 된다(이는 마이크로소프트 측에서 한/글 97의 소스 코드를 갖고 있다는 점이 한글과컴퓨터에게 부담으로 작용한 측면이 있는 것으로 보인다<sup class="정리_필요 noprint">[<a href="/wiki/%EC%9C%84%ED%82%A4%EB%B0%B1%EA%B3%BC:%EC%B6%9C%EC%B2%98_%EB%B0%9D%ED%9E%88%EA%B8%B0" title="위키백과:출처 밝히기">출처 필요</a>]</sup>).</p>
<p>한/글 워디안은 256단계의 다단계 되돌리기/반복(실행 취소) 기능을 지원하고, 변형 다단 편집<sup class="reference" id="cite_ref-20"><a href="#cite_note-20">[20]</a></sup>, 표 서식 기능 강화, 엣센스 영한사전과 표준 국어 대사전을 삽입하는 등 커다란 개선 사항이 있었지만, 워디안에서 작성한 문서를 이전 버전에서 읽을 수 없는 문제<sup class="reference" id="cite_ref-21"><a href="#cite_note-21">[21]</a></sup>, 이전 버전의 문서를 워디안에서 본래대로 읽을 수 없는 문제, 프로그램의 안정성 문제가 지적되었고, 바뀐 인터페이스에 익숙해지지 못한 사용자들이 여전히 한/글 97을 계속 사용하면서 별다른 인기를 얻지 못했다.</p>
<p>한글과컴퓨터사에서는 한/글 워디안의 패치파일을 1개월에 1번씩 제공하는 등의 강행군으로 프로그램의 안정성을 확보하기 위해 안간힘을 썼고 1729 빌드의 서비스 팩을 발표하며 어느 정도 성과를 거두었지만 여전히 문제는 잠복해 있었다.</p>
<h4><span class="mw-headline" id=".ED.95.9C.2F.EA.B8.80_2002">한/글 2002</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%ED%95%9C/%EA%B8%80&amp;action=edit&amp;section=39" title="부분 편집: 한/글 2002">편집</a><span class="mw-editsection-bracket">]</span></span></h4>
<p>2001년에 출시된 제품으로 한/글 워디안을 기반으로 이전 버전 파일과의 호환성 문제 및 안정성 문제를 해결한 제품으로 나중에 프로그램의 안정성을 향상시킨 한/글 2002 SE로 무상 업그레이드되었다. 한/글 2002판은 한/글 워디안 사용자들을 위해 업데이트 다운로드와 정품 CD가 무상 배포되기도 하였다. 이후 외국계 펀드와 당시 경영진과의 한글과컴퓨터사의 경영권 분쟁이 발생하였으며 한글과컴퓨터는 <a class="new" href="/w/index.php?title=%ED%94%84%EB%9D%BC%EC%9E%84%EC%82%B0%EC%97%85%EA%B0%9C%EB%B0%9C&amp;action=edit&amp;redlink=1" title="프라임산업개발 (없는 문서)">프라임산업개발</a>의 자회사로 편입되었다.</p>
<h4><span class="mw-headline" id=".ED.95.9C.2F.EA.B8.80_2004">한/글 2004</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%ED%95%9C/%EA%B8%80&amp;action=edit&amp;section=40" title="부분 편집: 한/글 2004">편집</a><span class="mw-editsection-bracket">]</span></span></h4>
<p><a href="/wiki/XML" title="XML">XML</a> 문서 원과 공개 키 기반 암호화, 스킨, 작업창 기능이 추가된 제품이다. 한/글 2004와 함께 출시된 한컴오피스 2004는 국산 스프레드시트인 <a class="mw-redirect" href="/wiki/%EB%84%A5%EC%85%80" title="넥셀">넥셀</a>과 자체 개발한 프레젠테이션 프로그램인 한컴 슬라이드 2004를 포함함으로써 토종 오피스 스위트로의 제 모습을 갖추게 되었다. 기능 면에서는 마이크로소프트 오피스에 비해 아직 부족하지만 가격이 저렴하고 <a href="/wiki/%EB%A7%88%EC%9D%B4%ED%81%AC%EB%A1%9C%EC%86%8C%ED%94%84%ED%8A%B8_%EC%97%91%EC%85%80" title="마이크로소프트 엑셀">엑셀</a>, <a href="/wiki/%EB%A7%88%EC%9D%B4%ED%81%AC%EB%A1%9C%EC%86%8C%ED%94%84%ED%8A%B8_%ED%8C%8C%EC%9B%8C%ED%8F%AC%EC%9D%B8%ED%8A%B8" title="마이크로소프트 파워포인트">파워포인트</a>와 사용법이 비슷하며, 파일 호환도 가능해 주목을 받았다.</p>
<h4><span class="mw-headline" id=".ED.95.9C.2F.EA.B8.80_2005">한/글 2005</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%ED%95%9C/%EA%B8%80&amp;action=edit&amp;section=41" title="부분 편집: 한/글 2005">편집</a><span class="mw-editsection-bracket">]</span></span></h4>
<p>협동 작업을 위한 버전 관리 기능, 메모 기능의 추가. <a href="/wiki/XML" title="XML">XML</a>로 작성된 한/글 서식과 ODBC의 연동, 표의 가로 세로 대칭 뒤집기, 아랍어 등의 기능이 추가되었다. 이 한/글 2005는 itq 워드와 같은 컴퓨터 문서에 관한자격증 시험 정식 프로그램으로 채택되어 사용되고 있다. <a class="mw-redirect" href="/wiki/%EB%A7%88%EC%9D%B4%ED%81%AC%EB%A1%9C%EC%86%8C%ED%94%84%ED%8A%B8_%EC%9C%88%EB%8F%84" title="마이크로소프트 윈도">윈도</a>용과 <a href="/wiki/%EB%A6%AC%EB%88%85%EC%8A%A4" title="리눅스">리눅스</a>용이 함께 있다. 대한민국에서는 컴퓨터 공간 용량 문제 때문에 2000년도부터 2011년도 당시에는 2002,2004,"2005",2007버전은 컴퓨터 실기 자격증 표준 소프트웨어로 잘 알려져 있다.</p>
<h4><span class="mw-headline" id=".ED.95.9C.2F.EA.B8.80_2006_for_.EB.A7.A5">한/글 2006 for 맥</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%ED%95%9C/%EA%B8%80&amp;action=edit&amp;section=42" title="부분 편집: 한/글 2006 for 맥">편집</a><span class="mw-editsection-bracket">]</span></span></h4>
<p>한/글 2006 for 맥 : <a href="/wiki/%EC%95%A0%ED%94%8C_(%EA%B8%B0%EC%97%85)" title="애플 (기업)">애플컴퓨터코리아</a>의 기술 지원으로 <a class="mw-redirect" href="/wiki/OS_X" title="OS X">OS X</a>용이 2005년 출시되었다. 기존의 한/글이 가진 기능을 Mac OS X 환경에서 구현하였다. <a href="/wiki/%ED%8A%B8%EB%A3%A8%ED%83%80%EC%9E%85" title="트루타입">트루타입</a>과 한/글 전용 글꼴을 지원하며 지원 사양은 OS X 10.3/10.4 이상, 파워PC G4와 G5 (인텔은 로제타 기술로 실행되나, 속도가 느리고 512 MB 이상 <a class="mw-redirect" href="/wiki/%EB%A9%94%EB%AA%A8%EB%A6%AC" title="메모리">메모리</a>가 필요하다. 한/글 2014 for 맥 : 2013년 12월 19일 OS X(10.4) 인텔CPU용으로 출시하였다. 한/글 2010 프로그램을 기본으로 만들어지고, MAC UI를 충실히 따랐다.</p>
<h4><span class="mw-headline" id=".ED.95.9C.2F.EA.B8.80_2007">한/글 2007</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%ED%95%9C/%EA%B8%80&amp;action=edit&amp;section=43" title="부분 편집: 한/글 2007">편집</a><span class="mw-editsection-bracket">]</span></span></h4>
<p>선과 도형에 <a class="mw-redirect" href="/wiki/%EC%95%A4%ED%8B%B0%EC%97%90%EC%9D%BC%EB%A6%AC%EC%96%B4%EC%8A%A4" title="앤티에일리어스">앤티에일리어스</a>가 적용되었고, 반투명한 선을 그릴 수 있게 되었다. 그밖에 마이크로소프트 오피스와의 호환성이 향상되었고, 워터마크<sup class="reference" id="cite_ref-22"><a href="#cite_note-22">[22]</a></sup>, 암호화 기능, 세로쓰기, 공인 인증서 암호화 기능, 개체 연결선, 개체 그림자 삽입 등 보안성이 향상되었다. 또한 추가로 ValuePack를 설치하면 다양한 기능과 더불어 PDF 문서를 쉽게 생성할 수 있는 무료 Haansoft PDF 문서 변환 드라이버를 설치할 수 있어 편리한 문서 변환 기능을 제공한다.</p>
<h4><span class="mw-headline" id=".ED.95.9C.2F.EA.B8.80_2008_for_.EB.A6.AC.EB.88.85.EC.8A.A4">한/글 2008 for 리눅스</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%ED%95%9C/%EA%B8%80&amp;action=edit&amp;section=44" title="부분 편집: 한/글 2008 for 리눅스">편집</a><span class="mw-editsection-bracket">]</span></span></h4>
<p>한/글 2007을 기반으로 <a href="/wiki/%EB%A6%AC%EB%88%85%EC%8A%A4" title="리눅스">리눅스</a> 사용자들을 위해 만들어졌다. 그러나 운영체제 특성의 차이로 인해 일부 동작이나 기능이 지원되지 않는다. 예를 들면 응용 프로그램 간 복사 및 붙이기나 ActiveX 및 윈도용 프로그램을 활용하는 기능은 제한된다. 게다가 문서 인쇄시 인쇄기의 드라이버에 따라 인쇄가 되지 않는다고 한다. 이유는 해당 인쇄기의 드라이버가 리눅스용 드라이버를 지원하지 않기 때문이다. 파일의 형태가 .rpm로 되어있기 때문에 rpm군의 패키지를 사용하지 않는 리눅스 배포판(특히 우분투)은 몇 가지 설정을 해야 했다.</p>
<h3><span class="mw-headline" id="2010.EB.85.84_.EC.9D.B4.ED.9B.84">2010년 이후</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%ED%95%9C/%EA%B8%80&amp;action=edit&amp;section=45" title="부분 편집: 2010년 이후">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<h4><span class="mw-headline" id=".ED.95.9C.2F.EA.B8.80_2010">한/글 2010</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%ED%95%9C/%EA%B8%80&amp;action=edit&amp;section=46" title="부분 편집: 한/글 2010">편집</a><span class="mw-editsection-bracket">]</span></span></h4>
<p><a href="/wiki/2010%EB%85%84" title="2010년">2010년</a> <a href="/wiki/3%EC%9B%94_3%EC%9D%BC" title="3월 3일">3월 3일</a> 출시된 제품으로<sup class="reference" id="cite_ref-23"><a href="#cite_note-23">[23]</a></sup> 한/글 2010은 한/글 2007의 차기작으로 리본 메뉴<sup class="reference" id="cite_ref-24"><a href="#cite_note-24">[24]</a></sup>와 유사한 인터페이스를 도입하였다. <a href="/wiki/%EC%98%A4%ED%94%88%EB%8F%84%ED%81%90%EB%A8%BC%ED%8A%B8" title="오픈도큐먼트">ODF</a>문서를 읽어올수 있으며, 저장할 수 있고<sup class="reference" id="cite_ref-25"><a href="#cite_note-25">[25]</a></sup>, MetaBlog API를 한글자체에서 적용하여 한글로 작성한 문서를 블로그로 업로드할 수 있다. 그리고 개인정보 유출 방지를 위한 기능도 추가되어 있다. 또한 한/글 2007에서 제공하던 Haansoft PDF의 명칭이 회사 이름의 영문 표기법이 바뀜에 따라 Hancom PDF로 바뀌었다.</p>
<p><a href="/wiki/2010%EB%85%84" title="2010년">2010년</a> <a href="/wiki/6%EC%9B%94_29%EC%9D%BC" title="6월 29일">6월 29일</a> HWP 문서의 파일 형식을 공개하였다. 이로써 ISO 국제 표준 문서 형식인 ODF와 OOXML을 지원과 함께 표준화와 개방성 강화에 대한 강한 의욕을 보이고 있다.<sup class="reference" id="cite_ref-autogenerated1_26-0"><a href="#cite_note-autogenerated1-26">[26]</a></sup></p>
<p><a href="/wiki/2010%EB%85%84" title="2010년">2010년</a> <a href="/wiki/7%EC%9B%94_29%EC%9D%BC" title="7월 29일">7월 29일</a>에는 <a href="/wiki/%EC%95%84%EC%9D%B4%ED%8F%B0" title="아이폰">아이폰</a>과 <a href="/wiki/%EC%95%84%EC%9D%B4%ED%8C%A8%EB%93%9C" title="아이패드">아이패드</a>에서 2010 기반의 한컴오피스를 볼 수 있는 한컴오피스 뷰어를 출시하였으며, 앱스토어에서 무료로 다운 받을 수 있다. <sup class="reference" id="cite_ref-27"><a href="#cite_note-27">[27]</a></sup></p>
<p>또한 <a href="/wiki/%EC%98%9B%ED%95%9C%EA%B8%80" title="옛한글">옛한글</a> 표현 방식으로 2008까지 써 오던 한양 PUA 코드를 버리고, 유니코드 표준인 첫가끝 코드를 사용한다.</p>
<p>시스템 내용 구성이 이전 2000 시리즈하고는 완전 다른 새로운 형태의 시스템으로 구성을 했다. 이전 버전 하위호환 기능이 삭제되었는데 차트에다가 복잡한 선을 그리는 이전 버전 하위호환 기능이 삭제되어서 <a class="mw-redirect" href="/wiki/%EC%9B%8C%EB%93%9C%ED%94%84%EB%A1%9C%EC%84%B8%EC%84%9C" title="워드프로세서">워드프로세서</a> 자격증 시험 공부 학생들한테 많은 비난을 샀다. 다만, 그림에 복잡한 선을 집어넣는 기능은 남아 있다.</p>
<h4><span class="mw-headline" id=".ED.95.9C.2F.EA.B8.80_2014">한/글 2014</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%ED%95%9C/%EA%B8%80&amp;action=edit&amp;section=47" title="부분 편집: 한/글 2014">편집</a><span class="mw-editsection-bracket">]</span></span></h4>
<p><a href="/wiki/2013%EB%85%84" title="2013년">2013년</a> <a href="/wiki/10%EC%9B%94_10%EC%9D%BC" title="10월 10일">10월 10일</a>에 한컴 오피스 2014가 출시됐다. 윈도 8용 터치 인터페이스 지원이 특징적이다.</p>
<h4><span class="mw-headline" id=".ED.95.9C.2F.EA.B8.80_NEO">한/글 NEO</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%ED%95%9C/%EA%B8%80&amp;action=edit&amp;section=48" title="부분 편집: 한/글 NEO">편집</a><span class="mw-editsection-bracket">]</span></span></h4>

<h2><span class="mw-headline" id=".EA.B4.80.EB.A0.A8_.EC.A0.9C.ED.92.88">관련 제품</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%ED%95%9C/%EA%B8%80&amp;action=edit&amp;section=49" title="부분 편집: 관련 제품">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<h3><span class="mw-headline" id=".ED.95.9C.2F.EA.B8.80_815">한/글 815</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%ED%95%9C/%EA%B8%80&amp;action=edit&amp;section=50" title="부분 편집: 한/글 815">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>8·15광복을 기념하여 출시된 제품이다. 한/글 97에 기반한(R4) 제품으로, 제품이 담긴 CD와 함께 1년간의 사용권을 판매하였다. 이 제품에 포함된 한/글 97은 표 아이콘을 제공하는 등의 개선점이 있고 한/글 96 기반의 국제판도 함께 포함되었다.</p>
<h3><span class="mw-headline" id=".EB.84.B7_.ED.95.9C.2F.EA.B8.80">넷 한/글</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%ED%95%9C/%EA%B8%80&amp;action=edit&amp;section=51" title="부분 편집: 넷 한/글">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>매달 사용료(개인 월 2,500원)를 내고 인터넷에서 인증을 받아 사용하는 한/글 제품이다. 한/글 2002 SE의 모든 기능을 사용할 수 있으나 사전, 클립아트, 일부 글꼴 데이터는 사용이 불가능하다. 인터넷이 연결된 곳이라면 어디에서든지 이용할 수 있고, 원격 저장소(10MB)도 제공하였다.</p>
<h3><span class="mw-headline" id=".EB.8B.A4.ED.81.90.EB.A7.A8">다큐맨</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%ED%95%9C/%EA%B8%80&amp;action=edit&amp;section=52" title="부분 편집: 다큐맨">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>USB 플래시 드라이브에 작은 한/글 프로그램을 내장하여 USB를 연결할 수 있는 어느 PC에서나 한/글 작업이 가능하게 만든 제품이다.</p>
<h3><span class="mw-headline" id=".ED.95.9C.2F.EC.85.80">한/셀</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%ED%95%9C/%EA%B8%80&amp;action=edit&amp;section=53" title="부분 편집: 한/셀">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<div class="rellink relarticle mainarticle"><img alt="" data-file-height="128" data-file-width="128" height="16" src="//upload.wikimedia.org/wikipedia/commons/thumb/e/ec/Crystal_Clear_app_xmag.svg/16px-Crystal_Clear_app_xmag.svg.png" srcset="//upload.wikimedia.org/wikipedia/commons/thumb/e/ec/Crystal_Clear_app_xmag.svg/24px-Crystal_Clear_app_xmag.svg.png 1.5x, //upload.wikimedia.org/wikipedia/commons/thumb/e/ec/Crystal_Clear_app_xmag.svg/32px-Crystal_Clear_app_xmag.svg.png 2x" width="16"/> 이 부분의 본문은 <a class="mw-redirect" href="/wiki/%ED%95%9C%EC%85%80" title="한셀">한셀</a>입니다.</div>
<p>넥스소프트 사가 개발한 스프레드시트 제품으로, 엑셀과 비슷한 인터페이스와 사용법을 가지고 있으며, 한글과컴퓨터에 <b>넥셀</b>을 매각하였다. 엑셀의 일부 고급기능이 없는 대신 저가로 공급되어, 중소기업 등을 대상으로 인기를 끈 제품으로 현재 <a href="/wiki/%ED%95%9C%EC%BB%B4%EC%98%A4%ED%94%BC%EC%8A%A4" title="한컴오피스">한컴오피스</a> 제품군에 포함되어 판매되고 있다. 한컴오피스 2007 버전까지는 <b>넥셀</b>이라는 명칭을 사용하였으나, 한컴오피스 2010 버전부터는 <b>한/셀</b>이라는 이름으로 변경되었다.</p>
<h3><span class="mw-headline" id=".ED.95.9C.2F.EC.87.BC">한/쇼</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%ED%95%9C/%EA%B8%80&amp;action=edit&amp;section=54" title="부분 편집: 한/쇼">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>한/쇼는 <a href="/wiki/%EB%A7%88%EC%9D%B4%ED%81%AC%EB%A1%9C%EC%86%8C%ED%94%84%ED%8A%B8_%ED%8C%8C%EC%9B%8C%ED%8F%AC%EC%9D%B8%ED%8A%B8" title="마이크로소프트 파워포인트">마이크로소프트 파워포인트</a>와 비슷한 기능을 지닌 프레젠테이션 제품으로 한컴오피스 제품군의 하나이다. 한컴오피스 2007 버전까지는 <b>한컴 슬라이드</b>이라는 명칭을 사용하였으나, 한컴오피스 2010 버전부터는 <b>한/쇼</b>이라는 이름으로 변경되었다.</p>
<h2><span class="mw-headline" id=".EC.9E.A5.EC.A0.90">장점</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%ED%95%9C/%EA%B8%80&amp;action=edit&amp;section=55" title="부분 편집: 장점">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<h3><span class="mw-headline" id=".EB.8B.A4.EC.96.91.ED.95.9C_.EA.B8.B0.EB.8A.A5">다양한 기능</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%ED%95%9C/%EA%B8%80&amp;action=edit&amp;section=56" title="부분 편집: 다양한 기능">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<h4><span class="mw-headline" id=".EC.B1.85.EB.A7.8C.EB.93.A4.EA.B8.B0">책만들기</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%ED%95%9C/%EA%B8%80&amp;action=edit&amp;section=57" title="부분 편집: 책만들기">편집</a><span class="mw-editsection-bracket">]</span></span></h4>
<p>다른 한글과컴퓨터 제품처럼 한/글은 <a href="/wiki/%EB%A7%88%EC%9D%B4%ED%81%AC%EB%A1%9C%EC%86%8C%ED%94%84%ED%8A%B8_%EC%9B%8C%EB%93%9C" title="마이크로소프트 워드">마이크로소프트 워드</a> 같은 외국 <a href="/wiki/%EC%86%8C%ED%94%84%ED%8A%B8%EC%9B%A8%EC%96%B4" title="소프트웨어">소프트웨어</a>의 훌륭한 대체품이 되고 있다. 실제로 많은 한국인이 한/글로 워드 프로세싱 작업을 하고 있으며, 일부 출판 기획자는 미적 감각이 그다지 필요하지 않은 텍스트 위주의 <a href="/wiki/%EC%A0%84%EC%9E%90_%EC%B6%9C%ED%8C%90" title="전자 출판">전자 출판</a>이나 수식이 많이 들어가는 <a href="/wiki/%EC%88%98%ED%95%99" title="수학">수학</a> 교재 편집 등에 활용하고 있다. 그 실례로 한/글을 이용한 전자 출판은 모민원 씨, 위형복 씨등의 고급 사용자들에 의해 널리 알려지기도 했으며, <a class="mw-redirect" href="/wiki/%ED%95%9C%EC%BB%B4" title="한컴">한컴</a>에서 발행하던 월간지 월간 한글과 컴퓨터에는 한/글 2.1을 이용한 <a href="/wiki/%EC%95%85%EB%B3%B4" title="악보">악보</a> 만들기,문서 클리닉등의 한/글을 이용한 전자 출판 노하우들이 독자 기고등으로 실려서 출판 기획자들의 사랑을 받았다.</p>
<h4><span class="mw-headline" id=".EC.99.B8.EB.9E.98.EC.96.B4">외래어</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%ED%95%9C/%EA%B8%80&amp;action=edit&amp;section=58" title="부분 편집: 외래어">편집</a><span class="mw-editsection-bracket">]</span></span></h4>
<p>한글은 한국식 <a href="/wiki/%ED%95%9C%EC%9E%90" title="한자">한자</a>, 중국어 <a class="mw-redirect" href="/wiki/%EA%B0%84%EC%B2%B4" title="간체">간체</a> 및 <a class="mw-redirect" href="/wiki/%EB%B2%88%EC%B2%B4" title="번체">번체</a>, 일본식 <a href="/wiki/%ED%95%9C%EC%9E%90" title="한자">한자</a> 및 <a href="/wiki/%EA%B0%80%EB%82%98_(%EB%AC%B8%EC%9E%90)" title="가나 (문자)">가나</a>뿐만 아니라 구결 및 옛한글(한양 사용자 정의) 지원 <a href="/wiki/%EA%B8%80%EA%BC%B4" title="글꼴">글꼴</a>과 같이 배포된다. 또한 한/글에서는 <a class="mw-redirect" href="/wiki/%EC%8B%A0%EC%95%BD%EC%84%B1%EC%84%9C" title="신약성서">신약성서</a> 저자들이 사용한 말인 <a class="mw-redirect" href="/wiki/%ED%97%AC%EB%9D%BC%EC%96%B4" title="헬라어">헬라어</a>를 <a href="/wiki/%EB%A1%9C%EB%A7%88%EC%9E%90" title="로마자">로마자</a>로 쓸 수 있기 때문에, 신약성서 연구를 위해 헬라어를 사용해야 하는 <a class="mw-redirect" href="/wiki/%EC%84%B1%EC%84%9C%ED%95%99%EC%9E%90" title="성서학자">성서학자</a>, <a href="/wiki/%EC%8B%A0%ED%95%99" title="신학">신학</a>생, <a href="/wiki/%EA%B8%B0%EB%8F%85%EA%B5%90" title="기독교">기독교</a> <a href="/wiki/%EC%84%B1%EC%A7%81%EC%9E%90" title="성직자">성직자</a>, 신학서적 기획자들에게 도움이 된다.</p>
<h3><span class="mw-headline" id=".EA.B0.84.EC.9D.B4_.EC.A0.84.EC.9E.90_.EC.B6.9C.ED.8C.90">간이 전자 출판</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%ED%95%9C/%EA%B8%80&amp;action=edit&amp;section=59" title="부분 편집: 간이 전자 출판">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>한/글은 사용자가 원하는 문서를 정확하고 깔끔하게 만들 수 있다는 것만으로도 의미가 있는 프로그램이었으며, 고급 사용자들은 한/글을 이용하여 단순한 전자 출판을 하기도 한다. 한/글을 이용하면 적절한 <a class="mw-redirect" href="/wiki/%EC%A0%84%EC%9E%90%EC%B6%9C%ED%8C%90" title="전자출판">전자출판</a> 항목으로는 1. <a href="/wiki/%EC%88%98%EC%8B%9D" title="수식">수식</a>이 많이 들어가는 <a href="/wiki/%EC%88%98%ED%95%99" title="수학">수학</a>교재 2. 각주와 표가 많이 들어가는 논문 3. <a href="/wiki/%EA%B5%90%ED%9A%8C" title="교회">교회</a> <a class="mw-disambig" href="/wiki/%EC%A3%BC%EB%B3%B4" title="주보">주보</a>, <a class="new" href="/w/index.php?title=%EC%86%8C%EC%8B%9D%EC%A7%80&amp;action=edit&amp;redlink=1" title="소식지 (없는 문서)">소식지</a>처럼 화려한 디자인을 하지 않아도 되고 표와 텍스트가 많이 들어가는 문서 4. <a href="/wiki/%EC%9D%BC%EB%B3%B8%EC%96%B4" title="일본어">일본어</a> <a href="/wiki/%EA%B0%80%EB%82%98" title="가나">가나</a>, <a class="mw-redirect" href="/wiki/%ED%97%AC%EB%9D%BC%EC%96%B4" title="헬라어">헬라어</a>등을 빠르게 입력해야 하고 화려한 디자인을 하지 않아도 되는 간단한 외국어나 <a class="mw-redirect" href="/wiki/%EC%84%B1%EC%84%9C" title="성서">성서</a> 교재등이 있다.</p>
<h4><span class="mw-headline" id=".EC.B1.85_.EB.A7.8C.EB.93.9C.EB.8A.94.EB.8D.B0_.EC.9C.A0.EC.9A.A9.ED.95.9C_.EA.B8.B0.EB.8A.A5.EB.93.A4">책 만드는데 유용한 기능들</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%ED%95%9C/%EA%B8%80&amp;action=edit&amp;section=60" title="부분 편집: 책 만드는데 유용한 기능들">편집</a><span class="mw-editsection-bracket">]</span></span></h4>
<p>한/글은 스타일 기능 제공, 반복되는 <a class="mw-redirect" href="/wiki/%EA%B8%80%EC%9E%90" title="글자">글자</a>나 <a class="mw-redirect" href="/wiki/%EA%B7%B8%EB%A6%BC" title="그림">그림</a>을 넣을 수 있는 바탕쪽, 간단한 <a href="/wiki/%EB%8F%84%ED%98%95" title="도형">도형</a> 그리기 기능을 제공하고 있는데, 이는 <a href="/wiki/%EC%BF%BC%ED%81%AC%EC%9D%B5%EC%8A%A4%ED%94%84%EB%A0%88%EC%8A%A4" title="쿼크익스프레스">쿼크익스프레스</a> 등의 레이아웃 소프트웨어를 의식한 것으로 보인다. 한/글의 바탕쪽 기능은 한 개의 파일 내에서 페이지마다 들어가는 글자와 그림을 각 페이지마다 다르게 넣을 수 있는 레이아웃 소프트웨어에 비하면 미흡하지만, 페이지마다 들어가야 하는 글자나 그림을 한 번에 넣을 수 있다는 점에서 레이아웃 소프트웨어의 마스터 기능과 거의 비슷하다. 그외 자간, 행간 조절, 그림 위치 조정, <a href="/wiki/%ED%95%9C%EA%B8%80" title="한글">한글</a>, <a href="/wiki/%EB%A1%9C%EB%A7%88%EC%9E%90" title="로마자">로마자</a>, <a href="/wiki/%EC%88%AB%EC%9E%90" title="숫자">숫자</a>와 특수 문자에 맞춘 다른 글꼴 사용 및 자간 조절 등 간이 전자 출판에 필요한 기능들을 한/글에서는 세심하게 제공하고 있다. 예를 들어 한글은 한글글꼴인 윤명조, 영문은 영문서체인 Garamond를 쓸 수 있는 것이다.</p>
<p>또한 한/글의 <a href="/wiki/%EB%A7%9E%EC%B6%A4%EB%B2%95" title="맞춤법">맞춤법</a> 기능은 교정/교열의 수고를 일부 덜어준다. 물론 한/글의 맞춤법은 오자(誤字)를 고치는 수준의 기계적 맞춤법 기능이므로, <a href="/wiki/%ED%8E%B8%EC%A7%91%EC%9E%90" title="편집자">편집자</a>는 원고를 인쇄하여 여러 번에 걸친 교정교열을 보는 수고를 해야 한다.</p>
<p>하지만 한/글은 문서 작성을 위해 개발된 워드프로세서이기 때문에, <a class="mw-redirect" href="/wiki/%EB%B6%81_%EB%94%94%EC%9E%90%EC%9D%B8" title="북 디자인">북 디자인</a>, <a class="mw-disambig" href="/wiki/%EC%B9%B4%EB%93%9C" title="카드">카드</a>, <a href="/wiki/%EB%8B%AC%EB%A0%A5" title="달력">달력</a>, <a class="new" href="/w/index.php?title=%ED%8E%B8%EC%A7%80%EC%A7%80&amp;action=edit&amp;redlink=1" title="편지지 (없는 문서)">편지지</a> 등의 <a class="new" href="/w/index.php?title=%ED%8C%AC%EC%8B%9C&amp;action=edit&amp;redlink=1" title="팬시 (없는 문서)">팬시</a>(Fancy) 디자인에는 <a class="mw-redirect" href="/wiki/%EC%BF%BC%ED%81%AC_%EC%9D%B5%EC%8A%A4%ED%94%84%EB%A0%88%EC%8A%A4" title="쿼크 익스프레스">쿼크 익스프레스</a>, <a class="mw-redirect" href="/wiki/%EC%9D%B8%EB%94%94%EC%9E%90%EC%9D%B8" title="인디자인">인디자인</a> 등의 레이아웃 소프트웨어가 더 적합하다.</p>
<h2><span class="mw-headline" id=".EB.8B.A8.EC.A0.90">단점</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%ED%95%9C/%EA%B8%80&amp;action=edit&amp;section=61" title="부분 편집: 단점">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<h3><span class="mw-headline" id=".ED.98.B8.ED.99.98.EC.84.B1_.EB.AC.B8.EC.A0.9C">호환성 문제</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%ED%95%9C/%EA%B8%80&amp;action=edit&amp;section=62" title="부분 편집: 호환성 문제">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>비록 HWP 형식이 한국 내부에서는 <a href="/wiki/%EB%A7%88%EC%9D%B4%ED%81%AC%EB%A1%9C%EC%86%8C%ED%94%84%ED%8A%B8_%EC%9B%8C%EB%93%9C" title="마이크로소프트 워드">DOC</a>, <a href="/wiki/PDF" title="PDF">PDF</a><sup class="reference" id="cite_ref-28"><a href="#cite_note-28">[28]</a></sup>, RTF 파일 대신에 널리 사용되고 있는 편이지만, 이 프로그램이 한국 밖에서 잘 사용되지 않는다. 가장 큰 이유는 초기에 HWP 형식의 코드가 공개되지 않아서 다른 프로그램에서 한/글 파일을 열도록 <a class="mw-redirect" href="/wiki/%ED%94%84%EB%A1%9C%EA%B7%B8%EB%9E%98%EB%B0%8D" title="프로그래밍">프로그래밍</a>하기 힘들다는 것이다. 때문에 대다수 프로그램에서 HWP 형식의 문서를 읽어들이지 못하며, 여는 데 성공한다 하더라도 문서의 내용이 완벽하게 표현되지 않는다. 이를 보완하기 위해 무료 프로그램인 '한글과컴퓨터 뷰어'를 제공하고 있다. 최근에 완벽하지는 않지만, 네이버오피스나 MS-WORD에서 HWP파일을 지원한다.</p>
<p>또 다른 단점은 한/글 문서에서 복사한 문자열을 아무데나 붙여 넣을 수 없다는 것이다. 일례로 한/글 문서에서 복사된 문자열을 한국어 로캘을 사용하지 않는 윈도 메모장에 붙여넣으면 제대로 표시되지 않지만, 파이어폭스에 붙여 넣으면 제대로 보일 때가 있다.</p>
<p>또한 유니코드 기반이라고 하나 파일 이름에 컴퓨터의 고유 코드 페이지(한국어 윈도의 경우 <a href="/wiki/KS_X_1001" title="KS X 1001">KS X 1001</a>)에 없는 문자가 포함되어 있으면 오류를 표시하고 파일을 읽고 쓸 수 없는 문제가 있다(예를 들어 약자나 속자로 된 한자 이름이 포함된 파일). 이 문제는 한/글 2010 이상에선 나타나지 않는다.</p>
<p>클립보드를 이용한 작업에서 사용하는 데이터를 다른 응용 프로그램에서 인식할 수 없는 경우가 많지만, 근래에 들어 아래아 한글의 클립보드 데이터 호환성 문제에 많은 진전이 있었고, 아래아 한글 마크업 언어 체계의 정비 등을 통하여 호환성에서 많은 발전을 이루었다.</p>
<p>2010년 06월 29일 한/글 문서 파일형식(hwp)을 공개하였다.<sup class="reference" id="cite_ref-autogenerated1_26-1"><a href="#cite_note-autogenerated1-26">[26]</a></sup></p>
<h3><span class="mw-headline" id=".EB.AF.B8.ED.9D.A1.ED.95.9C_.EC.A0.84.EC.9E.90_.EC.B6.9C.ED.8C.90_.EA.B8.B0.EB.8A.A5">미흡한 전자 출판 기능</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%ED%95%9C/%EA%B8%80&amp;action=edit&amp;section=63" title="부분 편집: 미흡한 전자 출판 기능">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>한/글은 전자출판 소프트웨어에서 비한다면 인쇄에서 미흡한 부분이 많다. 전자 출판 소프트웨어에서 제공되는 <a href="/wiki/CMYK_%EA%B0%90%EC%82%B0%ED%98%BC%ED%95%A9" title="CMYK 감산혼합">CMYK</a> 색조합 기능, <a class="mw-redirect" href="/wiki/%EB%A7%88%EC%8A%A4%ED%84%B0_%EC%9D%B8%EC%87%84" title="마스터 인쇄">마스터 인쇄</a>, <a href="/wiki/%EC%98%A4%ED%94%84%EC%85%8B_%EC%9D%B8%EC%87%84" title="오프셋 인쇄">오프셋 인쇄</a> 등 인쇄 방식에 따라 망점의 개수를 다르게 조정해 주는 망점 조정 기능 등이 한/글에서는 제공되지 않는다.</p>
<h3><span class="mw-headline" id=".EC.84.B1.EB.8A.A5_.EC.A0.80.ED.95.98">성능 저하</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%ED%95%9C/%EA%B8%80&amp;action=edit&amp;section=64" title="부분 편집: 성능 저하">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>한/글은 새로운 판이 출시될 수록 프로그램이 차지하는 용량이 증가하고 있다. <span class="fix" title="이 부분은 출처가 필요합니다.">특히, 한/글 2010에서는 업데이트로 인한 성능 저하가 문제가 되기도 하고 있다.</span><sup class="정리_필요 noprint">[<a href="/wiki/%EC%9C%84%ED%82%A4%EB%B0%B1%EA%B3%BC:%EC%B6%9C%EC%B2%98_%EB%B0%9D%ED%9E%88%EA%B8%B0" title="위키백과:출처 밝히기">출처 필요</a>]</sup></p>
<h2><span class="mw-headline" id=".EA.B8.B0.ED.83.80">기타</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%ED%95%9C/%EA%B8%80&amp;action=edit&amp;section=65" title="부분 편집: 기타">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<h3><span class="mw-headline" id="ASV_.ED.8C.8C.EC.9D.BC_.ED.98.95.EC.8B.9D">ASV 파일 형식</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%ED%95%9C/%EA%B8%80&amp;action=edit&amp;section=66" title="부분 편집: ASV 파일 형식">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>ASV는 Auto Save File(자동 저장 파일)의 준말로, 한글 문서 작업 도중에 컴퓨터(혹은 프로그램이)가 갑자기 잘못된 종료가 되었을 경우에 자동으로 저장(백업)이 되는 파일이다. ASV형식의 파일은 잘못된 종료 이후 한/글 최초실행시에 자동으로 로드되며, 로드 즉시 삭제된다.</p>
