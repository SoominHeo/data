<p><b>더하기표</b>(+)와 <b>빼기표</b>(−)<sup class="reference" id="cite_ref-1"><a href="#cite_note-1">[1]</a></sup>는 각각 <a href="/wiki/%EB%8D%A7%EC%85%88" title="덧셈">덧셈</a>(+)과 <a href="/wiki/%EB%BA%84%EC%85%88" title="뺄셈">뺄셈</a>(−)을 나타내는 기호로, 그 외에도 양(陽)과 음(陰), <a href="/wiki/%EC%9D%B4%EC%9D%B5" title="이익">이익</a>과 <a href="/wiki/%EC%86%90%EC%8B%A4" title="손실">손실</a>, 초과와 부족, <a href="/wiki/Rh%EC%8B%9D_%ED%98%88%EC%95%A1%ED%98%95" title="Rh식 혈액형">Rh식 혈액형</a> 등 여러 가지 뜻을 나타내기 위해 쓰이는 기호이다.</p>

<h2><span class="mw-headline" id=".EC.88.98.ED.95.99.EC.97.90.EC.84.9C.EC.9D.98_.EC.82.AC.EC.9A.A9">수학에서의 사용</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%8D%94%ED%95%98%EA%B8%B0%ED%91%9C%EC%99%80_%EB%B9%BC%EA%B8%B0%ED%91%9C&amp;action=edit&amp;section=1" title="부분 편집: 수학에서의 사용">원본 편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<div class="rellink relarticle mainarticle"><img alt="" data-file-height="128" data-file-width="128" height="16" src="//upload.wikimedia.org/wikipedia/commons/thumb/e/ec/Crystal_Clear_app_xmag.svg/16px-Crystal_Clear_app_xmag.svg.png" srcset="//upload.wikimedia.org/wikipedia/commons/thumb/e/ec/Crystal_Clear_app_xmag.svg/24px-Crystal_Clear_app_xmag.svg.png 1.5x, //upload.wikimedia.org/wikipedia/commons/thumb/e/ec/Crystal_Clear_app_xmag.svg/32px-Crystal_Clear_app_xmag.svg.png 2x" width="16"/> 이 부분의 본문은 <a href="/wiki/%EB%8D%A7%EC%85%88" title="덧셈">덧셈</a>, <a href="/wiki/%EB%BA%84%EC%85%88" title="뺄셈">뺄셈</a>, <a href="/wiki/%EB%B6%80%ED%98%B8_(%EC%88%98%ED%95%99)" title="부호 (수학)">부호</a>입니다.</div>
<ul>
<li><a class="mw-redirect" href="/wiki/%EC%9D%B4%ED%95%AD_%EC%97%B0%EC%82%B0%EC%9E%90" title="이항 연산자">이항 연산자</a>로서 두 수 사이에 표시될 경우 <a href="/wiki/%EB%8D%A7%EC%85%88" title="덧셈">덧셈</a>(+)이나 <a href="/wiki/%EB%BA%84%EC%85%88" title="뺄셈">뺄셈</a>(−) <a href="/wiki/%EA%B3%84%EC%82%B0" title="계산">연산</a>을 뜻한다.</li>
<li><a class="new" href="/w/index.php?title=%EB%8B%A8%ED%95%AD_%EC%97%B0%EC%82%B0%EC%9E%90&amp;action=edit&amp;redlink=1" title="단항 연산자 (없는 문서)">단항 연산자</a>로서 어떤 수 앞에 표시될 경우 양(陽, +)이나 음(陰, −)의 <a href="/wiki/%EB%B6%80%ED%98%B8_(%EC%88%98%ED%95%99)" title="부호 (수학)">부호</a>를 뜻한다.
<ul>
<li>빼기표의 경우 단항 연산자로서 부호를 가진 수 앞에 표시될 경우 <a href="/wiki/%EB%B6%80%ED%98%B8_(%EC%88%98%ED%95%99)" title="부호 (수학)">부호</a>가 반대가 됨을 뜻한다. 예를 들어 <i>x</i>가 3일 때 −<i>x</i>는 −3이고, <i>x</i>가 −3일 때 −<i>x</i>는 3이다. 마찬가지로 −(−3)은 3과 같다.</li>
</ul>
</li>
</ul>
<h2><span class="mw-headline" id=".EC.97.AC.EB.9F.AC_.EB.B6.84.EC.95.BC.EC.97.90.EC.84.9C.EC.9D.98_.EC.82.AC.EC.9A.A9">여러 분야에서의 사용</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%8D%94%ED%95%98%EA%B8%B0%ED%91%9C%EC%99%80_%EB%B9%BC%EA%B8%B0%ED%91%9C&amp;action=edit&amp;section=2" title="부분 편집: 여러 분야에서의 사용">원본 편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<div class="thumb tright">
<div class="thumbinner" style="width:222px;"><a class="image" href="/wiki/%ED%8C%8C%EC%9D%BC:VFPt_dipole_electric.svg"><img alt="" class="thumbimage" data-file-height="600" data-file-width="600" height="220" src="//upload.wikimedia.org/wikipedia/commons/thumb/d/df/VFPt_dipole_electric.svg/220px-VFPt_dipole_electric.svg.png" srcset="//upload.wikimedia.org/wikipedia/commons/thumb/d/df/VFPt_dipole_electric.svg/330px-VFPt_dipole_electric.svg.png 1.5x, //upload.wikimedia.org/wikipedia/commons/thumb/d/df/VFPt_dipole_electric.svg/440px-VFPt_dipole_electric.svg.png 2x" width="220"/></a>
<div class="thumbcaption">
<div class="magnify"><a class="internal" href="/wiki/%ED%8C%8C%EC%9D%BC:VFPt_dipole_electric.svg" title="실제 크기로"></a></div>
<a href="/wiki/%EC%A0%84%ED%95%98" title="전하">양전하 또는 음전하</a>를 더하기표·빼기표로 나타낼 수 있다.</div>
</div>
</div>
<ul>
<li><a href="/wiki/%EB%AC%BC%EB%A6%AC%ED%95%99" title="물리학">물리학</a>에서 <a href="/wiki/%EC%A0%84%ED%95%98" title="전하">전하</a>, <a href="/wiki/%EC%A0%84%EA%B7%B9" title="전극">전극</a>을 표시할 때 양전하(<a href="/wiki/%EC%96%91%EC%84%B1%EC%9E%90" title="양성자">양성자</a>의 전하) 또는 <a class="mw-redirect" href="/wiki/%EC%96%91%EA%B7%B9" title="양극">양극</a>을 더하기표로 표시하고, 음전하(<a href="/wiki/%EC%A0%84%EC%9E%90" title="전자">전자</a>의 전하) 또는 <a class="mw-redirect" href="/wiki/%EC%9D%8C%EA%B7%B9" title="음극">음극</a>을 빼기표로 표시한다.</li>
<li><a href="/wiki/%EC%BB%B4%ED%93%A8%ED%84%B0" title="컴퓨터">컴퓨터</a>에서 쓰는 <a href="/wiki/C_(%ED%94%84%EB%A1%9C%EA%B7%B8%EB%9E%98%EB%B0%8D_%EC%96%B8%EC%96%B4)" title="C (프로그래밍 언어)">C 언어</a> 따위의 <a href="/wiki/%ED%94%84%EB%A1%9C%EA%B7%B8%EB%9E%98%EB%B0%8D_%EC%96%B8%EC%96%B4" title="프로그래밍 언어">프로그래밍 언어</a>에서 덧셈이나 뺄셈 연산자로 쓰이며, 기호를 두 번 연속 써서(++ 또는 −−) 1만큼 증가시키거나 감소시킴을 뜻하기도 한다.</li>
<li><a href="/wiki/Rh%EC%8B%9D_%ED%98%88%EC%95%A1%ED%98%95" title="Rh식 혈액형">Rh식 혈액형</a> 판정에서 Rh 항원이 있는 경우 Rh+(알에이치 양성), 없는 경우 Rh−(알에이치 음성)으로 표기한다.</li>
<li><a href="/wiki/%EC%96%B8%EC%96%B4%ED%95%99" title="언어학">언어학</a>에서 문장 내의 앞말과 뒷말을 합침을 보일 때에 쓰는 <b>보탬표</b>로 + 기호를 쓴다.<sup class="reference" id="cite_ref-2"><a href="#cite_note-2">[2]</a></sup></li>
<li><a class="new" href="/w/index.php?title=%EC%95%88%EA%B2%BD%EA%B4%91%ED%95%99&amp;action=edit&amp;redlink=1" title="안경광학 (없는 문서)">안경광학</a>에서 사용하는 <a href="/wiki/%EA%B5%B4%EC%A0%88%EB%A0%A5" title="굴절력">굴절력</a> 단위인 <a href="/wiki/%EB%94%94%EC%98%B5%ED%84%B0" title="디옵터">디옵터</a>를 표기할 때 볼록렌즈는 더하기표를, 오목렌즈는 빼기표를 사용한다.</li>
</ul>
<h2><span class="mw-headline" id=".EB.8D.94.ED.95.98.EA.B8.B0.ED.91.9C">더하기표</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%8D%94%ED%95%98%EA%B8%B0%ED%91%9C%EC%99%80_%EB%B9%BC%EA%B8%B0%ED%91%9C&amp;action=edit&amp;section=3" title="부분 편집: 더하기표">원본 편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>이탈리아의 <a class="new" href="/w/index.php?title=%EB%A0%88%EC%98%A4%EB%82%98%EB%A5%B4%EB%8F%84_%ED%94%BC%EC%82%AC%EB%8F%84&amp;action=edit&amp;redlink=1" title="레오나르도 피사도 (없는 문서)">레오나르도 피사도</a>가 1300년경 처음으로 a+b를 a와 b로 사용했다고 하며, 라틴어로 ‘~과’라는 단어가 <i>et</i>이기 때문에 <i>et</i>라고 사용했으며, 이를 간략하게 쓰는 과정에서 +가 되었다고 한다.</p>
<h2><span class="mw-headline" id=".EB.B9.BC.EA.B8.B0.ED.91.9C">빼기표</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%8D%94%ED%95%98%EA%B8%B0%ED%91%9C%EC%99%80_%EB%B9%BC%EA%B8%B0%ED%91%9C&amp;action=edit&amp;section=4" title="부분 편집: 빼기표">원본 편집</a><span class="mw-editsection-bracket">]</span></span></h2>
