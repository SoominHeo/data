<p><b>사파비 왕조</b>(<span style="font-size: smaller;"><a href="/wiki/%ED%8E%98%EB%A5%B4%EC%8B%9C%EC%95%84%EC%96%B4" title="페르시아어">페르시아어</a>:</span> <span lang="fa" xml:lang="fa">صفویه</span>) 또는 <b>사파위 제국</b>은 사파비 가(家)의 <a href="/wiki/%EC%9D%B4%EC%8A%A4%EB%A7%88%EC%9D%BC_1%EC%84%B8" title="이스마일 1세">이스마일 1세</a>를 시조로 하는 <a href="/wiki/%EC%9D%B4%EB%9E%80" title="이란">이란</a>의 <a href="/wiki/%EC%99%95%EC%A1%B0" title="왕조">왕조</a>(<a href="/wiki/1501%EB%85%84" title="1501년">1501년</a>~<a href="/wiki/1736%EB%85%84" title="1736년">1736년</a>)이다. 대부분의 이슬람 왕조들이 <a href="/wiki/%EC%88%98%EB%8B%88%ED%8C%8C" title="수니파">수니파</a>를 믿은 것과는 달리 사파비 왕조는 <a href="/wiki/%ED%8C%8C%ED%8B%B0%EB%A7%88_%EC%99%95%EC%A1%B0" title="파티마 왕조">파티마 왕조</a>와 함께 대표적인 <a href="/wiki/%EC%8B%9C%EC%95%84%ED%8C%8C" title="시아파">시아파</a> 이슬람 왕조이다.</p>
<h2><span class="mw-headline" id=".EC.97.AD.EC.82.AC">역사</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%82%AC%ED%8C%8C%EB%B9%84_%EC%99%95%EC%A1%B0&amp;action=edit&amp;section=1" title="부분 편집: 역사">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>이스마일은 <a href="/wiki/%EC%8B%9C%EC%95%84%ED%8C%8C" title="시아파">시아파</a> 제7대 <a href="/wiki/%EC%9D%B4%EB%A7%98" title="이맘">이맘</a>의 자손으로서 1502년 아제르바이잔의 <a class="new" href="/w/index.php?title=%EC%95%84%ED%81%AC_%EC%BD%94%EC%9C%A4%EB%A3%A8&amp;action=edit&amp;redlink=1" title="아크 코윤루 (없는 문서)">아크 코윤루</a>(Ak Koyunlu：白羊) 왕조를 꺾고, <a href="/wiki/%ED%83%80%EB%B8%8C%EB%A6%AC%EC%A6%88" title="타브리즈">타브리즈</a>에서 독립했다. 그 후 <a href="/wiki/%EB%A9%94%EC%86%8C%ED%8F%AC%ED%83%80%EB%AF%B8%EC%95%84" title="메소포타미아">메소포타미아</a>에서 <a href="/wiki/%EC%95%84%ED%94%84%EA%B0%80%EB%8B%88%EC%8A%A4%ED%83%84" title="아프가니스탄">아프가니스탄</a>에 이르는 지역을 통일했다. 그는 <a class="mw-redirect" href="/wiki/%EC%95%84%EB%9D%BC%EB%B9%84%EC%95%84%EC%96%B4" title="아라비아어">아라비아어</a>풍의 <a href="/wiki/%EC%88%A0%ED%83%84" title="술탄">술탄</a> 칭호를 배척하고 <a href="/wiki/%ED%8E%98%EB%A5%B4%EC%8B%9C%EC%95%84%EC%96%B4" title="페르시아어">페르시아어</a>의 존칭 '<a href="/wiki/%EC%83%A4" title="샤">샤</a>'를 사용하였으며, 또 <a class="new" href="/w/index.php?title=12%EC%9D%B4%EB%A7%98%ED%8C%8C&amp;action=edit&amp;redlink=1" title="12이맘파 (없는 문서)">12이맘파</a>(派)를 <a href="/wiki/%EA%B5%AD%EA%B5%90" title="국교">국교</a>로 정하는 등 이민족 지배에서 벗어난 이란 <a href="/wiki/%EB%AF%BC%EC%A1%B1%EC%A3%BC%EC%9D%98" title="민족주의">민족주의</a>를 강조하였다.</p>
<p>시아파를 지도 정신으로 하는 이 왕조는 <a href="/wiki/%EB%AC%B4%EA%B5%B4_%EC%A0%9C%EA%B5%AD" title="무굴 제국">무굴 왕조</a>나 <a class="new" href="/w/index.php?title=%ED%8E%98%EB%A5%B4%EC%8B%9C%EC%95%84-%EC%83%A4%EC%9D%B4%EB%B0%94%EB%8B%88_%EC%A0%84%EC%9F%81&amp;action=edit&amp;redlink=1" title="페르시아-샤이바니 전쟁 (없는 문서)">우즈베크계의 칸국들과도 원정을 하기도 했었고</a> 또는 <a href="/wiki/%EC%88%98%EB%8B%88%ED%8C%8C" title="수니파">수니파</a>(派)의 <a href="/wiki/%EC%98%A4%EC%8A%A4%EB%A7%8C_%EC%A0%9C%EA%B5%AD" title="오스만 제국">오스만 제국</a>과 대항하게 되어 투르크의 침입을 받아 천도하였다.<sup class="reference" id="cite_ref-3"><a href="#cite_note-3">[3]</a></sup> 그러나 <a href="/wiki/%EC%95%84%EB%B0%94%EC%8A%A4_1%EC%84%B8" title="아바스 1세">아바스 대제</a>가 나오자 최성기를 맞이하였다. 그는 영국인의 지도로 군제(軍制) 개혁을 단행하고, 오스만 제국으로부터 광대한 실지(失地)를 탈환하였다.<sup class="reference" id="cite_ref-4"><a href="#cite_note-4">[4]</a></sup> 또 내정에도 빛나는 치적(治績)을 올려 수도 <a href="/wiki/%EC%9D%B4%EC%8A%A4%ED%8C%8C%ED%95%9C" title="이스파한">이스파한</a>의 장려함은 유럽 세계에까지 소문이 퍼졌다.</p>
<p>하지만 아바스 대제의 사후 사파비 왕조는 쇠퇴하기 시작했고 다시 오스만 제국의 침입을 받을때마다 영토를 잃게 된다.<sup class="reference" id="cite_ref-5"><a href="#cite_note-5">[5]</a></sup> 또 <a href="/wiki/1722%EB%85%84" title="1722년">1722년</a>에는 <a class="new" href="/w/index.php?title=%EC%95%84%ED%94%84%EA%B0%84%EC%A1%B1&amp;action=edit&amp;redlink=1" title="아프간족 (없는 문서)">아프간족</a>에게 수도를 점령당함으로써 사실상 쇠퇴하였으나 왕통(王統)은 11대 <a href="/wiki/1736%EB%85%84" title="1736년">1736년</a>까지 존속되었다. 이후 <a href="/wiki/%EC%9D%B4%EB%9E%80" title="이란">이란</a> 지방은 18세기 말 <a href="/wiki/%EC%B9%B4%EC%9E%90%EB%A5%B4_%EC%99%95%EC%A1%B0" title="카자르 왕조">카자르 왕조</a>에 의해 재통일되었다.</p>
<p>사파비 왕조 시대에는 <a href="/wiki/%EB%84%A4%EB%8D%9C%EB%9E%80%EB%93%9C" title="네덜란드">네덜란드</a>, <a href="/wiki/%ED%94%84%EB%9E%91%EC%8A%A4" title="프랑스">프랑스</a>, <a href="/wiki/%EC%98%81%EA%B5%AD" title="영국">영국</a> 등 여러 나라가 페르시아의 <a class="mw-redirect" href="/wiki/%EB%B9%84%EB%8B%A8" title="비단">비단</a>을 얻고 오스만 제국에 대하여 공동 전선을 펴기 위하여 친분 관계를 맺었다. 또한 인도 동쪽의 식민지와의 연락을 확보하기 위하여 다투어 우호 관계를 맺었다.</p>
