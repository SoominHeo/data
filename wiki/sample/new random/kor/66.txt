<p><b>조시 스미스</b>(Josh Smith, <a href="/wiki/1985%EB%85%84" title="1985년">1985년</a> <a href="/wiki/12%EC%9B%94_5%EC%9D%BC" title="12월 5일">12월 5일</a> ~ )는 <a href="/wiki/%EB%AF%B8%EA%B5%AD" title="미국">미국</a>의 <a href="/wiki/%EB%86%8D%EA%B5%AC" title="농구">농구</a> 선수이자, <a class="mw-redirect" href="/wiki/NBA" title="NBA">NBA</a> <a href="/wiki/%ED%9C%B4%EC%8A%A4%ED%84%B4_%EB%A1%9C%ED%82%A4%EC%B8%A0" title="휴스턴 로키츠">휴스턴 로키츠</a>의 선수이며, 포지션은 <a href="/wiki/%ED%8C%8C%EC%9B%8C_%ED%8F%AC%EC%9B%8C%EB%93%9C" title="파워 포워드">파워 포워드</a> / <a href="/wiki/%EC%8A%A4%EB%AA%B0_%ED%8F%AC%EC%9B%8C%EB%93%9C" title="스몰 포워드">스몰 포워드</a>이다. 2004년 NBA 드래프트 1라운드 전체 17순위로 <a class="mw-redirect" href="/wiki/%EC%95%A0%ED%8B%80%EB%9E%80%ED%83%80_%ED%98%B8%ED%81%AC%EC%8A%A4" title="애틀란타 호크스">애틀란타 호크스</a>에 입단하였다. 별명은 J-Smoove이다. <a class="mw-redirect" href="/wiki/NBA" title="NBA">NBA</a> <a href="/wiki/%ED%9C%B4%EC%8A%A4%ED%84%B4_%EB%A1%9C%ED%82%A4%EC%B8%A0" title="휴스턴 로키츠">휴스턴 로키츠</a>에서 활약하고 있는 <a href="/wiki/%EB%93%9C%EC%99%80%EC%9D%B4%ED%8A%B8_%ED%95%98%EC%9B%8C%EB%93%9C" title="드와이트 하워드">드와이트 하워드</a>와 절친 사이로 알려져 있다.</p>
