<p><b>윈도우 3.1x</b>(Windows 3.1x)는 <a href="/wiki/%EA%B7%B8%EB%9E%98%ED%94%BD_%EC%82%AC%EC%9A%A9%EC%9E%90_%EC%9D%B8%ED%84%B0%ED%8E%98%EC%9D%B4%EC%8A%A4" title="그래픽 사용자 인터페이스">그래픽 사용자 인터페이스</a> 운영 환경이다. 몇몇 에디션들이 <a href="/wiki/%EC%9C%88%EB%8F%84%EC%9A%B0_3.0" title="윈도우 3.0">윈도우 3.0</a>에 이어 <a href="/wiki/1992%EB%85%84" title="1992년">1992년</a>과 <a href="/wiki/1994%EB%85%84" title="1994년">1994년</a> 사이에 공개되었는데, 이 계열의 윈도는 표준 모드나 386 확장 <a class="mw-redirect" href="/wiki/%EC%9C%88%EB%8F%84_3.0" title="윈도 3.0">메모리 모드</a>로 동작할 수 있다. 한 가지 예외는 윈도우 포 워크그룹 3.11의 경우 386 확장 모드에서만 공식적으로 동작할 수 있다는 것이다. 윈도우 3.1x는 윈도우 1.0, 윈도우 2.0, 윈도우 2.1x, 윈도우 3.0, 윈도우 NT 3.1, 윈도우 NT 3.5, 윈도우 NT 3.51, 윈도우 95, 인터넷 익스플로러 1, 2와 동시에 2001년 12월 31일부로 지원이 중단되었다.</p>
<p>윈도우 3.1x는 <a href="/wiki/MS-DOS" title="MS-DOS">MS-DOS</a> 위에서 작동하며, 따라서 현대적인 관점에서 보면 운영 체제는 아니며, 소프트웨어의 일부로 본다. 하지만 MS-DOS에 존재하지 않는 프로세스 개념을 가진다. 윈도우의 다음 버전인 <a href="/wiki/%EC%9C%88%EB%8F%84%EC%9A%B0_95" title="윈도우 95">윈도우 95</a>에서는 MS-DOS를 윈도와 통합하여 윈도우 95 단독적인 부팅이 가능하다.</p>
<p><a href="/wiki/%ED%95%9C%EA%B5%AD%EC%96%B4" title="한국어">한국어</a> 버전은 '한글 윈도우 3.1'이라는 제목으로 출시되었다.</p>

<h2><span class="mw-headline" id=".EA.B0.9C.EC.9A.94">개요</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%9C%88%EB%8F%84%EC%9A%B0_3.1x&amp;action=edit&amp;section=1" title="부분 편집: 개요">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>윈도우 3.1은 1992년 4월 6일 판매된 가정용 윈도이다. 다음과 같이 구성되었다.</p>
<ul>
<li>윈도우 3.1 오리지널</li>
<li>윈도우 3.1 업그레이드 판</li>
<li>윈도우 3.11</li>
<li>윈도우 3.2 중국어판</li>
</ul>
<h3><span class="mw-headline" id=".EC.9C.88.EB.8F.84.EC.9A.B0_3.1">윈도우 3.1</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%9C%88%EB%8F%84%EC%9A%B0_3.1x&amp;action=edit&amp;section=2" title="부분 편집: 윈도우 3.1">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>1992년에 마이크로소프트는 윈도우 3.0을 개선해서 윈도우 3.1을 출시하였다. 윈도우 3.1은 윈도우 3.0과 함께 출시 2년만에 1,000만 개에 달하는 엄청난 판매량을 올렸다.</p>
<h3><span class="mw-headline" id=".EC.9C.88.EB.8F.84.EC.9A.B0_3.2">윈도우 3.2</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%9C%88%EB%8F%84%EC%9A%B0_3.1x&amp;action=edit&amp;section=3" title="부분 편집: 윈도우 3.2">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>1994년에 마이크로소프트는 <a href="/wiki/%EC%A4%91%EA%B5%AD" title="중국">중국</a> 시장을 대상으로 <a href="/wiki/%EA%B0%84%EC%B2%B4%EC%9E%90" title="간체자">간체판</a> 윈도우를 출시하였다. 이 업데이트된 시스템은 바로 윈도우 3.2이다. 윈도우 3.2는 윈도우 3.11의 중국어판이다. 이 업데이트는 중국어판에만 해당하며 <a href="/wiki/%EC%A4%91%EA%B5%AD%EC%96%B4" title="중국어">중국어</a>의 복잡한 기록 시스템에 관련한 문제들만 수정되었다.<sup class="reference" id="cite_ref-1"><a href="#cite_note-1">[1]</a></sup></p>
<p>윈도우 3.2는 일반적으로 기본 출력을 위한 간체자 및 일부 번역된 유틸리티를 포함한 <a href="/wiki/MS-DOS" title="MS-DOS">MS-DOS</a> 디스크 버전 10장을 제공하는 컴퓨터 제조업체들에게 주로 팔려나갔다.</p>
<h2><span class="mw-headline" id=".EC.9C.88.EB.8F.84.EC.9A.B0_.ED.8F.AC_.EC.9B.8C.ED.81.AC.EA.B7.B8.EB.A3.B9">윈도우 포 워크그룹</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%9C%88%EB%8F%84%EC%9A%B0_3.1x&amp;action=edit&amp;section=4" title="부분 편집: 윈도우 포 워크그룹">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>윈도우 포 워크그룹(Windows for Workgroups)은 업무용으로 개발된 소프트웨어이며, 종류는 다음과 같다.</p>
<ul>
<li>윈도우 포 워크그룹 3.1</li>
<li>윈도우 포 워크그룹 3.11</li>
</ul>
