<p><b>타이베이 시</b>(대북시, <small><a href="/wiki/%EC%A4%91%EA%B5%AD%EC%96%B4" title="중국어">중국어</a></small> <small><a href="/wiki/%EC%A0%95%EC%B2%B4%EC%9E%90" title="정체자">정체</a>:</small> <span lang="zh-hant">臺北市</span>, <small><a href="/wiki/%EA%B0%84%EC%B2%B4%EC%9E%90" title="간체자">간체</a>:</small> <span lang="zh-hans">台北市</span>, <small><a href="/wiki/%ED%95%9C%EC%96%B4_%EB%B3%91%EC%9D%8C" title="한어 병음">병음</a>:</small> <font face="Lucida Sans Unicode">Táiběi Shì</font>, <small><a class="mw-redirect" href="/wiki/%EC%A3%BC%EC%9D%8C_%EB%B6%80%ED%98%B8" title="주음 부호">주음 부호</a>:</small> <span lang="zh-TW">ㄊㄞˊㄅㄟˇㄕˋ</span>)는 <a href="/wiki/%ED%83%80%EC%9D%B4%EC%99%84" title="타이완">타이완</a> 북부에 있는 도시로, 사실상 <a href="/wiki/%EC%A4%91%ED%99%94%EB%AF%BC%EA%B5%AD" title="중화민국">중화민국</a>의 <a href="/wiki/%EC%88%98%EB%8F%84" title="수도">수도</a>로 기능하고 있다.<sup class="reference" id="cite_ref-1"><a href="#cite_note-1">[1]</a></sup> 면적 272㎢ 인구는 약 269만 명이다. <a class="new" href="/w/index.php?title=%ED%83%80%EC%9D%B4%EB%B2%A0%EC%9D%B4_%EB%B6%84%EC%A7%80&amp;action=edit&amp;redlink=1" title="타이베이 분지 (없는 문서)">타이베이 분지</a>에 위치하고 있으며, 외항으로 <a class="mw-redirect" href="/wiki/%EC%A7%80%EB%A3%BD" title="지룽">지룽</a>이 있다. <a href="/wiki/%EC%8B%A0%EB%B2%A0%EC%9D%B4_%EC%8B%9C" title="신베이 시">신베이 시</a> (新北市)에 둘러싸여 있다.</p>
<p><a class="mw-redirect" href="/wiki/%EC%9D%BC%EB%B3%B8%EC%9D%98_%ED%83%80%EC%9D%B4%EC%99%84_%ED%86%B5%EC%B9%98_%EC%8B%9C%EB%8C%80" title="일본의 타이완 통치 시대">일본 통치 시대</a>에는 <b>다이호쿠</b>(<span style="font-size: smaller;"><a href="/wiki/%EC%9D%BC%EB%B3%B8%EC%96%B4" title="일본어">일본어</a>:</span> <span lang="ja"><span lang="ja"><ruby><rb>台北</rb> <rp>(</rp><rt>たいほく</rt><rp>)</rp></ruby></span></span>)라고 불렸다. 타이베이는 <a href="/wiki/%ED%91%9C%EC%A4%80_%EC%A4%91%EA%B5%AD%EC%96%B4" title="표준 중국어">표준 중국어</a> 발음이며, <a href="/wiki/%EB%AF%BC%EB%82%9C%EC%96%B4" title="민난어">민난어</a>로는 <b>따이빡</b>(<span lang="zh-min-nan">Tâi-pak</span>)이다.</p>

<p>타이베이는 <a href="/wiki/%ED%83%80%EC%9D%B4%EC%99%84" title="타이완">타이완</a> 북쪽의 <a class="new" href="/w/index.php?title=%ED%83%80%EC%9D%B4%EB%B2%A0%EC%9D%B4_%EB%B6%84%EC%A7%80&amp;action=edit&amp;redlink=1" title="타이베이 분지 (없는 문서)">타이베이 분지</a>에 위치한다. 남쪽으로 신뎬시(新店溪), 서쪽으로 <a class="mw-redirect" href="/wiki/%EB%8B%A8%EC%88%98%EC%9D%B4%ED%97%88" title="단수이허">단수이허</a>와 경계를 접한다. 서쪽은 대체로 낮은 평야이고 남쪽과 동쪽, 북쪽을 향해 경사져 있으며 특히 북쪽에는 타이완에서 가장 높은 화산이자 <a class="new" href="/w/index.php?title=%EC%96%91%EB%B0%8D%EC%82%B0_%EA%B5%AD%EA%B0%80%EA%B3%B5%EC%9B%90&amp;action=edit&amp;redlink=1" title="양밍산 국가공원 (없는 문서)">양밍산 국가공원</a>으로 지정되어 있는 해발 1,120m의 <a href="/wiki/%EC%B9%98%EC%8B%B1_%EC%82%B0" title="치싱 산">치싱 산</a>이 있다. 시 북쪽의 <a href="/wiki/%EC%8A%A4%EB%A6%B0_%EA%B5%AC" title="스린 구">스린 구</a>와 <a href="/wiki/%EB%B2%A0%EC%9D%B4%ED%84%B0%EC%9A%B0_%EA%B5%AC" title="베이터우 구">베이터우 구</a>는 <a class="mw-redirect" href="/wiki/%EC%A7%80%EB%A3%BD%ED%97%88" title="지룽허">지룽허</a>의 북쪽에 위치하며 양밍산 국가공원과 접한다. 타이베이 시는 타이완의 25개 시와 현 중에서 16번째의 면적을 가지고 있다.</p>
<h3><span class="mw-headline" id=".EA.B8.B0.ED.9B.84">기후</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%ED%83%80%EC%9D%B4%EB%B2%A0%EC%9D%B4_%EC%8B%9C&amp;action=edit&amp;section=2" title="부분 편집: 기후">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>타이베이는 <a href="/wiki/%EC%98%A8%EB%82%9C_%EC%8A%B5%EC%9C%A4_%EA%B8%B0%ED%9B%84" title="온난 습윤 기후">온난 습윤 기후</a>(<a href="/wiki/%EC%BE%A8%ED%8E%9C%EC%9D%98_%EA%B8%B0%ED%9B%84_%EA%B5%AC%EB%B6%84" title="쾨펜의 기후 구분">쾨펜의 기후 구분</a> <i>Cfa</i>)에 속한다. 연평균 기온은 23.6 °C이고 여름 평균 기온은 29.4 °C, 겨울 평균 기온은 11 °C이다. 여름은 습하고 이따금 폭풍과 태풍을 동반하는 반면에 겨울은 짧고 온화하다.</p>
<p>타이완은 태평양 연안에 위치해 있기 때문에 6월부터 10월까지는 <a href="/wiki/%ED%83%9C%ED%92%8D" title="태풍">태풍</a>의 영향을 받는다.</p>

<h2><span class="mw-headline" id=".EC.97.AD.EC.82.AC">역사</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%ED%83%80%EC%9D%B4%EB%B2%A0%EC%9D%B4_%EC%8B%9C&amp;action=edit&amp;section=3" title="부분 편집: 역사">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<div class="thumb tright">
<div class="thumbinner" style="width:252px;"><a class="image" href="/wiki/%ED%8C%8C%EC%9D%BC:NationalPalace_MuseumFrontView.jpg"><img alt="" class="thumbimage" data-file-height="1000" data-file-width="1504" height="166" src="//upload.wikimedia.org/wikipedia/commons/thumb/b/b4/NationalPalace_MuseumFrontView.jpg/250px-NationalPalace_MuseumFrontView.jpg" srcset="//upload.wikimedia.org/wikipedia/commons/thumb/b/b4/NationalPalace_MuseumFrontView.jpg/375px-NationalPalace_MuseumFrontView.jpg 1.5x, //upload.wikimedia.org/wikipedia/commons/thumb/b/b4/NationalPalace_MuseumFrontView.jpg/500px-NationalPalace_MuseumFrontView.jpg 2x" width="250"/></a>

<p>타이베이는 18세기 이전까지는 <a class="mw-redirect mw-disambig" href="/wiki/%EA%B3%A0%EC%82%B0%EC%A1%B1" title="고산족">고산족</a>(<a class="new" href="/w/index.php?title=%EC%BC%80%ED%83%80%EA%B0%88%EB%9E%80%EC%A1%B1&amp;action=edit&amp;redlink=1" title="케타갈란족 (없는 문서)">케타갈란족</a>)의 거주지였다. 1709년부터 주로 <a href="/wiki/%ED%91%B8%EC%A0%A0_%EC%84%B1" title="푸젠 성">푸젠 성</a>의 <a href="/wiki/%ED%95%9C%EC%A1%B1" title="한족">한족</a>이 타이베이 분지에 정착하기 시작했다. 19세기 말에 타이베이 지역은 타이완 북부의 한족의 주요 거주지가 되었고 차 무역 등의 해외 무역이 활발해지면서 단수이는 해외 무역항의 하나로 지정되었다. 1875년에 타이완 북부는 타이완 부(臺灣府)로 분리되었다. 성내로 알려진 완화(萬華)와 다다오청(大稻埕)은 번영하였고 행정 관청들이 이곳에 세워졌다. 1875년부터 일본의 통치가 시작된 1895년까지 타이베이는 타이베이 부 단수이 현에 속했고 부청 소재지였다. 1886년에는 타이완 성이 설치되었고 타이베이는 성도가 되었다. 1894년에 타이베이는 타이완의 공식적인 수도가 되었다. 청나라 때의 옛 유적들은 주로 북문 주변에 남아있다. 서문과 성벽은 일본에 의해 파괴되었고 남문과 동문은 국민당에 의해 광범위하게 개발되어 옛 모습을 잃어버렸다.</p>
<p>1895년의 <a class="mw-redirect" href="/wiki/%EC%B2%AD%EC%9D%BC%EC%A0%84%EC%9F%81" title="청일전쟁">청일전쟁</a>에서 청이 일본에 패하면서 <a href="/wiki/%EC%8B%9C%EB%AA%A8%EB%85%B8%EC%84%B8%ED%82%A4_%EC%A1%B0%EC%95%BD" title="시모노세키 조약">시모노세키 조약</a>에 의해 타이완은 일본의 영토가 되었다. 일본으로 넘어간 후에 타이베이는 일본어로 다이호쿠라 불렸고 <a href="/wiki/%ED%83%80%EC%9D%B4%EC%99%84_%EC%B4%9D%EB%8F%85%EB%B6%80" title="타이완 총독부">타이완 총독부</a>의 정치적 중심 역할을 하였다. 행정 중심지가 되면서 도시에는 많은 새로운 관청과 공무원의 주택이 생겨났다. <a href="/wiki/%EC%A4%91%ED%99%94%EB%AF%BC%EA%B5%AD_%EC%B4%9D%ED%86%B5%EB%B6%80" title="중화민국 총통부">중화민국 총통부</a>를 포함한 타이페이의 많은 건물들은 일본 점령기에 세워진 것이다.</p>

<p>일본 점령기에 타이베이는 완화, 다다오청 등을 포함하는 다이호쿠 현(台北縣)의 일부가 되었다. 1938년에 동쪽의 마쓰야마(松山；쑹산)가 다이호쿠에 편입되었다. 1945년에 <a class="mw-redirect" href="/wiki/%ED%83%9C%ED%8F%89%EC%96%91%EC%A0%84%EC%9F%81" title="태평양전쟁">태평양전쟁</a>에서 패한 일본이 결국 항복하면서 <a class="mw-redirect" href="/wiki/%EC%A4%91%EA%B5%AD_%EA%B5%AD%EB%AF%BC%EB%8B%B9" title="중국 국민당">국민당</a>이 타이완을 지배하기 시작했다. 이후 타이완 성장의 임시 관청이 타이베이 시에 세워졌다.</p>
<p>1949년 12월 7일에 <a href="/wiki/%EC%9E%A5%EC%A0%9C%EC%8A%A4" title="장제스">장제스</a> 휘하의 국민당 정부는 <a class="mw-redirect" href="/wiki/%EA%B5%AD%EA%B3%B5%EB%82%B4%EC%A0%84" title="국공내전">국공내전</a>에서 <a class="mw-redirect" href="/wiki/%EC%A4%91%EA%B5%AD_%EA%B3%B5%EC%82%B0%EB%8B%B9" title="중국 공산당">중국 공산당</a>에 밀려 중국 본토에서 쫓겨났고 타이베이는 중화민국의 수도로 선포되었다(공식적인 수도는 <a class="mw-redirect" href="/wiki/%EB%82%9C%EC%A7%95" title="난징">난징</a>이다).</p>
<p>타이베이는 1949년 이후 점차 확장하였고 1967년 7월 1일에 직할시가 되어 성에 준하는 권한을 가지게 되었다. 이듬해에 타이베이는 다시 확장해 스린, 베이터우, 네이후, 난강, 징메이, 무자를 합병하였다. 이 때에 도시의 총면적은 주변의 진과 향을 합병해 4배로 늘어났고 인구는 156만 명으로 증가했다.</p>
<p>도시의 인구는 1960년대 초에 100만 명을 넘었고 1967년 이후 급격히 팽창해 1970년대 중반에 200만 명을 넘어섰다. 1990년대 중반 이후 시역 내의 인구 증가는 점차 둔화되었지만 타이베이는 여전히 세계에서 가장 인구밀도가 높은 도시 중 하나이고 도시 주변 지역의 인구 증가는 계속되었다. 1990년에 타이베이의 16개 구는 현재의 12개 구로 통합되었다.</p>
<h2><span class="mw-headline" id=".EA.B2.BD.EC.A0.9C">경제</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%ED%83%80%EC%9D%B4%EB%B2%A0%EC%9D%B4_%EC%8B%9C&amp;action=edit&amp;section=4" title="부분 편집: 경제">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<div class="thumb tright">

<p>중화민국의 수도로서 타이베이는 급격한 경제 개발의 중심지이고 현재 첨단 기술 제품을 생산하는 국제적인 도시의 하나가 되었다. 이른바 '타이완의 기적'의 일부로서 타이베이는 1960년대의 외국인 직접투자를 통해 급격한 성장을 보였다. 타이완은 2010년 2월 기준으로 3520억 달러를 보유하고 있어 세계 4위의 외환보유국이다.</p>
<p>아시아 금융위기에도 불구하고 매년 5%의 성장을 하였고 사실상의 완전 고용을 이루었으며 물가상승률도 낮다. 2007년에 타이베이의 명목 GDP는 약 1600억 달러이고 타이페이 도시권의 GDP는 2600억 달러로 세계에서 13위를 기록하였다. 타이베이의 1인당 GDP는 48,400달러로 도쿄(65,453달러)에 이어 두 번째로 높다. 주변의 도시 및 진과 향을 합치면 1인당 GDP는 25000달러로 떨어진다.</p>
<p>타이베이와 그 주변 지역은 오랫동안 타이완의 2차 산업과 3차 산업의 으뜸가는 공업 지대였다. 중화민국의 섬유와 의류를 생산하는 공장의 대부분이 이곳에 위치해있다. 그 밖의 산업으로 전자제품과 부품, 기계 장비와 유인물, 정밀 설비와 식료품을 생산한다. 조선업으로 요트와 유람선이 도시 북동쪽의 지룽에서 완성된다.</p>
<p>상업, 운송, 은행업 등의 서비스업이 점차 중요해지고 있다. 관광업은 그 비중은 작지만 지역 경제에서 중요한 요소의 하나이고 2008년의 해외 방문객 수는 약 300만 명 정도였다.</p>
<h2><span class="mw-headline" id=".EC.A0.95.EC.B9.98.C2.B7.ED.96.89.EC.A0.95">정치·행정</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%ED%83%80%EC%9D%B4%EB%B2%A0%EC%9D%B4_%EC%8B%9C&amp;action=edit&amp;section=5" title="부분 편집: 정치·행정">편집</a><span class="mw-editsection-bracket">]</span></span></h2>

<p>타이베이 시와 <a href="/wiki/%EC%8B%A0%EB%B2%A0%EC%9D%B4_%EC%8B%9C" title="신베이 시">신베이 시</a>(옛 타이베이 현)는 <a href="/wiki/%EC%A4%91%ED%99%94%EB%AF%BC%EA%B5%AD_%ED%96%89%EC%A0%95%EC%9B%90" title="중화민국 행정원">중화민국 행정원</a>이 직접 관할하는 <a href="/wiki/%EC%A7%81%ED%95%A0%EC%8B%9C" title="직할시">직할시</a>인 반면에 <a href="/wiki/%EC%A7%80%EB%A3%BD_%EC%8B%9C" title="지룽 시">지룽 시</a>는 <a href="/wiki/%ED%83%80%EC%9D%B4%EC%99%84_%EC%84%B1" title="타이완 성">타이완 성</a>의 일부이다. 타이베이의 시장은 1967년에 직할시가 되면서 국가에서 임명하다가 1994년에 처음으로 공개 선거가 치러졌다. 임기는 4년이고 직접투표를 통해 선출된다. 첫 민선시장은 <a href="/wiki/%EB%AF%BC%EC%A3%BC%EC%A7%84%EB%B3%B4%EB%8B%B9" title="민주진보당">민주진보당</a>의 <a href="/wiki/%EC%B2%9C%EC%88%98%EC%9D%B4%EB%B3%9C" title="천수이볜">천수이볜</a>이었다. <a href="/wiki/%EB%A7%88%EC%9E%89%EC%A3%BC" title="마잉주">마잉주</a>는 1998년과 2002년 두 차례에 걸쳐 시장을 역임했다. 현재의 시장은 2014년에 선출된 <a class="new" href="/w/index.php?title=%EC%BB%A4%EC%9B%90%EC%A0%80&amp;action=edit&amp;redlink=1" title="커원저 (없는 문서)">커원저</a>이다. 천수이볜과 마잉주는 모두 <a href="/wiki/%EC%A4%91%ED%99%94%EB%AF%BC%EA%B5%AD%EC%9D%98_%EC%B4%9D%ED%86%B5" title="중화민국의 총통">중화민국의 총통</a>이 되었다.</p>
<p>지난 10년간의 선거 결과를 보면 전반적인 타이베이 시 선거구의 투표 경향은 다소 국민당 진영으로 기울어져 있었다. 그러나 민주진보당 진영 또한 상당한 지지를 받고 있다.</p>
<p><a href="/wiki/%EC%B9%B4%EC%9D%B4%EB%8B%A4%EA%B1%B0%EB%9E%80_%EB%8C%80%EB%8F%84" title="카이다거란 대도">카이다거란 대도</a>에는 <a href="/wiki/%EC%A4%91%ED%99%94%EB%AF%BC%EA%B5%AD_%EC%B4%9D%ED%86%B5%EB%B6%80" title="중화민국 총통부">중화민국 총통부</a>와 기타 정부 건물이 위치해있다. 또한 이곳은 총통 취임식이나 국가 기념일 퍼레이드, 다른 나라의 고위직 관리 방문의 환영회, 데모, 축제와 같은 대중 집회가 열리기도 한다.</p>

<p>타이베이에 있는 대부분의 관광명소는 <a href="/wiki/%ED%83%80%EC%9D%B4%EB%B2%A0%EC%9D%B4_%EC%B2%A9%EC%9A%B4" title="타이베이 첩운">타이베이 첩운</a>으로 대표되는 편리한 교통망으로 쉽게 접근이 가능하다. 타이베이의 대표적인 명소는 야시장이다. 2007년 타이완 관광청 통계에 따르면, 관광객들이 가장 선호하는 여행지 1위는 놀랍게도 유적지나 관광명소가 아닌 야시장이었다. 연평균 20℃가 넘는 기후 탓에 야시장이 활성화될 수밖에 없는 현지 생활문화가 멋진 관광상품으로 승화된 것이다. 타이페이에는 타이완 최대규모의 <a href="/wiki/%EC%8A%A4%EB%A6%B0_%EC%95%BC%EC%8B%9C%EC%9E%A5" title="스린 야시장">스린 야시장</a>을 비롯해 중형규모의 <a href="/wiki/%ED%99%94%EC%8B%9C%EC%A7%80%EC%97%90_%EC%95%BC%EC%8B%9C%EC%9E%A5" title="화시지에 야시장">화시지에 야시장</a>, <a href="/wiki/%EB%9D%BC%EC%98%A4%ED%97%88%EC%A7%80%EC%97%90_%EA%B4%80%EA%B4%91_%EC%95%BC%EC%8B%9C%EC%9E%A5" title="라오허지에 관광 야시장">라오허지에 관광 야시장</a>등의 명물 야시장이 있다.<sup class="reference" id="cite_ref-.EC.A3.BC.EA.B0.84.EB.8F.99.EC.95.84_3-0"><a href="#cite_note-.EC.A3.BC.EA.B0.84.EB.8F.99.EC.95.84-3">[3]</a></sup> 타이베이에는 101빌딩이나 신꽝미쯔고시 백화점 등 규모 있는 쇼핑몰이 많다. 그리고, 융캉제나 중산역 근처 등에는 작은 부티크나 트렌디한 디자인숍 등이 타이베이 골목골목에 많이 숨어 있다. 그 외에도 <a href="/wiki/%EC%9A%A9%EC%82%B0%EC%82%AC" title="용산사">용산사</a>, <a href="/wiki/%EC%8B%9C%EB%A8%BC%EB%9D%B5" title="시먼띵">시먼띵</a>, <a href="/wiki/%EC%A4%91%EC%A0%95%EA%B8%B0%EB%85%90%EB%8B%B9" title="중정기념당">중정기념당</a>, <a href="/wiki/%EA%B5%AD%EB%A6%BD%EA%B3%A0%EA%B6%81%EB%B0%95%EB%AC%BC%EC%9B%90" title="국립고궁박물원">국립고궁박물원</a>,<a href="/wiki/%EB%B2%A0%EC%9D%B4%ED%84%B0%EC%9A%B0_%EC%98%A8%EC%B2%9C" title="베이터우 온천">베이터우 온천</a> 등지도 인기있는 관광 명소이다.</p>
<p>타이베이시를 여행하는 여행자들은 대체로 인근 <a href="/wiki/%EC%8B%A0%EB%B2%A0%EC%9D%B4_%EC%8B%9C" title="신베이 시">신베이 시</a>의 관광지인 <a href="/wiki/%EC%A7%84%EA%B3%BC%EC%8A%A4" title="진과스">진과스</a>, <a href="/wiki/%EB%8B%A8%EC%88%98%EC%9D%B4_%EA%B5%AC" title="단수이 구">단수이</a>등지도 여행 일정에 포함하기도 한다.</p>
<h2><span class="mw-headline" id=".EC.8A.A4.ED.8F.AC.EC.B8.A0">스포츠</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%ED%83%80%EC%9D%B4%EB%B2%A0%EC%9D%B4_%EC%8B%9C&amp;action=edit&amp;section=9" title="부분 편집: 스포츠">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<div class="thumb tright">
<div class="thumbinner" style="width:222px;"><a class="image" href="/wiki/%ED%8C%8C%EC%9D%BC:Taipei_Arena_outside_20051210_night.jpg"><img alt="" class="thumbimage" data-file-height="1536" data-file-width="2048" height="165" src="//upload.wikimedia.org/wikipedia/commons/thumb/6/69/Taipei_Arena_outside_20051210_night.jpg/220px-Taipei_Arena_outside_20051210_night.jpg" srcset="//upload.wikimedia.org/wikipedia/commons/thumb/6/69/Taipei_Arena_outside_20051210_night.jpg/330px-Taipei_Arena_outside_20051210_night.jpg 1.5x, //upload.wikimedia.org/wikipedia/commons/thumb/6/69/Taipei_Arena_outside_20051210_night.jpg/440px-Taipei_Arena_outside_20051210_night.jpg 2x" width="220"/></a>

타이베이 경기장</div>

<p>타이완은 오랫동안 미국과 일본의 영향력하에 있었기 때문에 야구와 농구가 타이베이의 인기있는 스포츠가 되었다. 타이완의 다른 지역처럼 타이베이에서도 야구가 가장 두드러지고 1960년대 이래로 종종 <a href="/wiki/%EC%95%84%EC%8B%9C%EC%95%84_%EC%95%BC%EA%B5%AC_%EC%84%A0%EC%88%98%EA%B6%8C_%EB%8C%80%ED%9A%8C" title="아시아 야구 선수권 대회">아시아 야구 선수권 대회</a>와 같은 국제 대회가 열렸다.</p>
<p>중화민국의 프로야구팀의 하나인 <a class="mw-redirect" href="/wiki/%EC%8A%9D%EB%94%94_%EC%97%98%EB%A6%AC%ED%8E%80%EC%B8%A0" title="슝디 엘리펀츠">슝디 엘리펀츠</a>의 연고지이다.</p>
<p>타이베이에 위치한 실내경기장인 <a href="/wiki/%ED%83%80%EC%9D%B4%EB%B2%A0%EC%9D%B4_%EC%95%84%EB%A0%88%EB%82%98" title="타이베이 아레나">타이베이 아레나</a>는 15,000명을 수용할 수 있다.</p>

<h3><span class="mw-headline" id=".EA.B3.B5.ED.95.AD_.EA.B5.90.ED.86.B5">공항 교통</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%ED%83%80%EC%9D%B4%EB%B2%A0%EC%9D%B4_%EC%8B%9C&amp;action=edit&amp;section=11" title="부분 편집: 공항 교통">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p><a href="/wiki/1979%EB%85%84" title="1979년">1979년</a> 타오위안 현에 <a href="/wiki/%ED%83%80%EC%9D%B4%EC%99%84_%ED%83%80%EC%98%A4%EC%9C%84%EC%95%88_%EA%B5%AD%EC%A0%9C%EA%B3%B5%ED%95%AD" title="타이완 타오위안 국제공항">타이완 타오위안 국제공항</a>이 완공되기 전에는 <a href="/wiki/%ED%83%80%EC%9D%B4%EB%B2%A0%EC%9D%B4_%EC%91%B9%EC%82%B0_%EA%B3%B5%ED%95%AD" title="타이베이 쑹산 공항">타이베이 쑹산 공항</a>이 국제공항의 역할을 하고 있었다. 현재 쑹산 공항은 대만의 주요 도시를 연결하는 국내선 전용 공항으로 운영되고 있고, 일부 국제선이 셔틀노선으로 개설되어 있다. 미래에 타오위안 국제공항과 쑹산 공항을 철도노선으로 연결할 계획이 있다.</p>
<h3><span class="mw-headline" id=".EC.B2.A0.EB.8F.84_.EA.B5.90.ED.86.B5">철도 교통</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%ED%83%80%EC%9D%B4%EB%B2%A0%EC%9D%B4_%EC%8B%9C&amp;action=edit&amp;section=12" title="부분 편집: 철도 교통">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>2007년 타이베이와 가오슝간에 <a href="/wiki/%ED%83%80%EC%9D%B4%EC%99%84_%EA%B3%A0%EC%86%8D%EC%B2%A0%EB%8F%84" title="타이완 고속철도">타이완 고속철도</a>가 개통되어 운행중이다. 타이베이의 지하철인 <a href="/wiki/%ED%83%80%EC%9D%B4%EB%B2%A0%EC%9D%B4_%EC%B2%A9%EC%9A%B4" title="타이베이 첩운">타이베이 첩운</a>(Taipei's Metro Rapid Transit System)은 중국어와 영어로 쓰여진 잘 정돈된 표지판을 갖추고 있어 목적지로 빠르게 이동할 수 있다.<sup class="reference" id="cite_ref-.EC.A3.BC.EA.B0.84.EB.8F.99.EC.95.84_3-1"><a href="#cite_note-.EC.A3.BC.EA.B0.84.EB.8F.99.EC.95.84-3">[3]</a></sup> 종관선 역은 <a class="new" href="/w/index.php?title=%EB%82%9C%EA%B0%95_%EC%97%AD&amp;action=edit&amp;redlink=1" title="난강 역 (없는 문서)">난강 역</a>, <a class="new" href="/w/index.php?title=%EC%91%B9%EC%82%B0_%EC%97%AD&amp;action=edit&amp;redlink=1" title="쑹산 역 (없는 문서)">쑹산 역</a>, <a href="/wiki/%ED%83%80%EC%9D%B4%EB%B2%A0%EC%9D%B4_%EC%97%AD" title="타이베이 역">타이베이 역</a>, <a class="new" href="/w/index.php?title=%EC%99%84%ED%99%94_%EC%97%AD&amp;action=edit&amp;redlink=1" title="완화 역 (없는 문서)">완화 역</a>이 있으며 지하화 공사가 끝나 모두 지하역이다.</p>

<td class="navbox-list" colspan="2" style="text-align:center;background:#ddddff;text-align:left;font-size:90%;">
<div>
<p>참고: <a href="/wiki/%EC%A4%91%ED%99%94%EB%AF%BC%EA%B5%AD%EC%9D%98_%EA%B0%95%EC%97%AD" title="중화민국의 강역">중화민국의 강역</a>, <a href="/wiki/%EC%A4%91%ED%99%94%EB%AF%BC%EA%B5%AD%EC%9D%98_%ED%96%89%EC%A0%95_%EA%B5%AC%EC%97%AD" title="중화민국의 행정 구역">중화민국의 행정 구역</a>, <a href="/wiki/%EC%A4%91%ED%99%94%EB%AF%BC%EA%B5%AD%EC%9D%98_%ED%96%89%EC%A0%95_%EA%B5%AC%EC%97%AD_(1912%EB%85%84~1949%EB%85%84)" title="중화민국의 행정 구역 (1912년~1949년)">중화민국의 행정 구역 (1912년~1949년)</a><br/>
주1: <a href="/wiki/1945%EB%85%84" title="1945년">1945년</a>부터 <a href="/wiki/2005%EB%85%84" title="2005년">2005년</a>까지 <a class="new" href="/w/index.php?title=%EC%A4%91%ED%99%94%EB%AF%BC%EA%B5%AD_%EC%A0%95%EB%B6%80&amp;action=edit&amp;redlink=1" title="중화민국 정부 (없는 문서)">중화민국 정부</a>가 공표한 행정 구역. 실효 통치하지 않는 <a href="/wiki/%EC%A4%91%EA%B5%AD_%EB%8C%80%EB%A5%99" title="중국 대륙">대륙 지구</a>를 포함한다. 실효 통치하는 지역은 <a href="/wiki/%ED%83%80%EC%9D%B4%EC%99%84_%EC%84%B1" title="타이완 성">타이완 성</a>, <a href="/wiki/%ED%91%B8%EC%A0%A0_%EC%84%B1_(%EC%A4%91%ED%99%94%EB%AF%BC%EA%B5%AD)" title="푸젠 성 (중화민국)">푸젠 성</a> 일부, <strong class="selflink">타이베이 시</strong>, <a href="/wiki/%EA%B0%80%EC%98%A4%EC%8A%9D_%EC%8B%9C" title="가오슝 시">가오슝 시</a>이다. 현재의 직할시인 <a href="/wiki/%EC%8B%A0%EB%B2%A0%EC%9D%B4_%EC%8B%9C" title="신베이 시">신베이 시</a>, <a href="/wiki/%ED%83%80%EC%9D%B4%EB%82%9C_%EC%8B%9C" title="타이난 시">타이난 시</a>, <a href="/wiki/%ED%83%80%EC%9D%B4%EC%A4%91_%EC%8B%9C" title="타이중 시">타이중 시</a>는 2010년 설치되었다.<br/>
주2: 1949년 <a href="/wiki/%EA%B5%AD%EB%B6%80%EC%B2%9C%EB%8C%80" title="국부천대">국부천대</a> 이전의 수도. 현 중화민국 중앙 정부 소재지는 <strong class="selflink">타이베이 시</strong>이다.<br/>
주3: 저장 성 정부는 1949년 8월 <a href="/wiki/%ED%95%AD%EC%A0%80%EC%9A%B0_%EC%8B%9C" title="항저우 시">항저우 시</a>에서 <a class="new" href="/w/index.php?title=%EB%8B%A4%EC%B2%9C_%EC%84%AC&amp;action=edit&amp;redlink=1" title="다천 섬 (없는 문서)">다천 섬</a>으로 옮겨졌다. 1955년 2월 11일 <a href="/wiki/%EC%A4%91%ED%99%94%EB%AF%BC%EA%B5%AD_%EA%B5%AD%EA%B5%B0" title="중화민국 국군">중화민국 국군</a>과 주민, 그리고 성 정부가 <a href="/wiki/%EB%8B%A4%EC%B2%9C_%EC%84%AC_%EC%B2%A0%EC%88%98" title="다천 섬 철수">섬에서 철수</a>했으며, 1956년에 정식으로 폐지되었다.<br/>
주4: 1945년 10월 25일 설치되었다(<a href="/wiki/%EA%B4%91%EB%B3%B5%EC%A0%88_(%ED%83%80%EC%9D%B4%EC%99%84)" title="광복절 (타이완)">광복절</a> 참고). 현재는 사실상 명목 상의 행정 구역이다.<br/>
주5: 푸젠 성 정부는 1949년 8월 <a href="/wiki/%ED%91%B8%EC%A0%80%EC%9A%B0_%EC%8B%9C" title="푸저우 시">푸저우 시</a>에서 <a href="/wiki/%EC%A7%84%EB%A8%BC_%ED%98%84" title="진먼 현">진먼 현</a>으로 옮겨졌다. 1956년 7월 폐지했다가 1992년 다시 설치되었다. 현재는 사실상 명목 상의 행정 구역이다.<br/>
주6: <strong class="selflink">타이베이 시</strong>는 <a href="/wiki/1967%EB%85%84" title="1967년">1967년</a> 직할시로 승격되었다.<br/>
주7: <a href="/wiki/%EA%B0%80%EC%98%A4%EC%8A%9D_%EC%8B%9C" title="가오슝 시">가오슝 시</a>는 <a href="/wiki/1979%EB%85%84" title="1979년">1979년</a> 직할시로 승격되었다.<br/></p>
주8: <a href="/wiki/%EC%99%B8%EB%AA%BD%EA%B3%A8" title="외몽골">외몽골</a> 지역. <a href="/wiki/1946%EB%85%84" title="1946년">1946년</a> <a href="/wiki/1%EC%9B%94_15%EC%9D%BC" title="1월 15일">1월 15일</a> 중화민국 정부는 <a href="/wiki/%EB%AA%BD%EA%B3%A8_%EC%9D%B8%EB%AF%BC_%EA%B3%B5%ED%99%94%EA%B5%AD" title="몽골 인민 공화국">몽골 인민 공화국</a>의 독립을 승인했으나 <a href="/wiki/1953%EB%85%84" title="1953년">1953년</a> 승인을 취소했다. 중화민국은 <a href="/wiki/1955%EB%85%84" title="1955년">1955년</a> 몽골의 <a class="mw-redirect" href="/wiki/%EA%B5%AD%EC%A0%9C_%EC%97%B0%ED%95%A9" title="국제 연합">국제 연합</a> 가입 신청에 거부권을 행사했다. 몽골은 소비에트 연방의 도움을 받아 <a href="/wiki/1961%EB%85%84" title="1961년">1961년</a> 국제 연합에 가입했다. 중화민국은 2002년 이후부터 <a href="/wiki/%EB%AA%BD%EA%B3%A8" title="몽골">몽골</a>과 점진적으로 외교 관계를 맺고 있으나 공식적으로 독립을 승인하지는 않았다.</div>
