<p><b>제임스 애디슨 베이커 3세</b>(James Addison Baker III, <a href="/wiki/1930%EB%85%84" title="1930년">1930년</a> <a href="/wiki/4%EC%9B%94_28%EC%9D%BC" title="4월 28일">4월 28일</a> ~ )는 <a href="/wiki/%EB%AF%B8%EA%B5%AD" title="미국">미국</a>의 정치인이자, 61대 <a href="/wiki/%EB%AF%B8%EA%B5%AD_%EA%B5%AD%EB%AC%B4%EB%B6%80" title="미국 국무부">국무 장관</a>(1989~1992)이다.</p>
<p><a href="/wiki/%ED%9C%B4%EC%8A%A4%ED%84%B4" title="휴스턴">휴스턴</a>에서 태어나, <a href="/wiki/1952%EB%85%84" title="1952년">1952년</a> <a href="/wiki/%ED%94%84%EB%A6%B0%EC%8A%A4%ED%84%B4_%EB%8C%80%ED%95%99%EA%B5%90" title="프린스턴 대학교">프린스턴 대학교</a>를 졸업하고 <a href="/wiki/1957%EB%85%84" title="1957년">1957년</a> <a href="/wiki/%ED%85%8D%EC%82%AC%EC%8A%A4_%EB%8C%80%ED%95%99%EA%B5%90_%EC%98%A4%EC%8A%A4%ED%8B%B4" title="텍사스 대학교 오스틴">텍사스 대학교</a>의 법과 대학원을 수료하였다. 그 후 <a href="/wiki/1975%EB%85%84" title="1975년">1975년</a>까지 <a href="/wiki/%ED%9C%B4%EC%8A%A4%ED%84%B4" title="휴스턴">휴스턴</a>에서 <a href="/wiki/%EB%B3%80%ED%98%B8%EC%82%AC" title="변호사">변호사</a> 개업을 하다가 <a href="/wiki/%EC%A0%9C%EB%9F%B4%EB%93%9C_%ED%8F%AC%EB%93%9C" title="제럴드 포드">제럴드 포드</a> 대통령에 의하여 <a href="/wiki/%EB%AF%B8%EA%B5%AD_%EC%83%81%EB%AC%B4%EB%B6%80" title="미국 상무부">상무부</a> 차관이 되었다. <a href="/wiki/1981%EB%85%84" title="1981년">1981년</a>부터 <a href="/wiki/1985%EB%85%84" title="1985년">1985년</a>까지 <a href="/wiki/%EB%A1%9C%EB%84%90%EB%93%9C_%EB%A0%88%EC%9D%B4%EA%B1%B4" title="로널드 레이건">로널드 레이건</a> 대통령의 수석 보좌관을 지냈고, 1985년부터 <a href="/wiki/1989%EB%85%84" title="1989년">1989년</a>까지 재무 장관으로 있으면서 연방 <a href="/wiki/%EC%84%B8%EA%B8%88" title="세금">세금</a> 제도의 개정을 이끌었다.</p>
<p><a href="/wiki/1989%EB%85%84" title="1989년">1989년</a> <a href="/wiki/%EC%A1%B0%EC%A7%80_H._W._%EB%B6%80%EC%8B%9C" title="조지 H. W. 부시">조지 H. W. 부시</a> 대통령 내각아래에서 국무 장관으로 취임하여, <a href="/wiki/1991%EB%85%84" title="1991년">1991년</a> <a href="/wiki/%EA%B1%B8%ED%94%84_%EC%A0%84%EC%9F%81" title="걸프 전쟁">걸프 전쟁</a>에서 <a href="/wiki/%EC%9D%B4%EB%9D%BC%ED%81%AC" title="이라크">이라크</a>와 싸우는 데 국가들의 제휴를 성립하는 도움을 주었다. <a href="/wiki/1992%EB%85%84" title="1992년">1992년</a> <a href="/wiki/8%EC%9B%94_23%EC%9D%BC" title="8월 23일">8월 23일</a> 국무 장관직을 사임하고 <a href="/wiki/1993%EB%85%84" title="1993년">1993년</a>까지 다시 대통령 수석 보좌관으로 있었다.</p>
<p><a href="/wiki/2006%EB%85%84" title="2006년">2006년</a> 베이커와 전 <a href="/wiki/%EB%AF%BC%EC%A3%BC%EB%8B%B9_(%EB%AF%B8%EA%B5%AD)" title="민주당 (미국)">민주당</a> 대표 <a class="new" href="/w/index.php?title=%EB%A6%AC_%ED%95%B4%EB%B0%80%ED%84%B4&amp;action=edit&amp;redlink=1" title="리 해밀턴 (없는 문서)">리 해밀턴</a>은 이라크 학습 그룹의 합동 지도자로 선택되었다. <a href="/wiki/%EB%AF%B8%EA%B5%AD_%EC%9D%98%ED%9A%8C" title="미국 의회">미국 의회</a>는 <a href="/wiki/%EC%9D%B4%EB%9D%BC%ED%81%AC_%EC%A0%84%EC%9F%81" title="이라크 전쟁">이라크 전쟁</a>과 정책의 권고를 향상시키는 데 분석하기 위하여 그 그룹을 형성한 것이다.</p>
<p>그의 저서로는 《외교 정치》(<a href="/wiki/1995%EB%85%84" title="1995년">1995년</a>)와 《열심히 일하고, 공부하고... 그리고 정치에 끼어들지 마라!》(<a href="/wiki/2007%EB%85%84" title="2007년">2007년</a>)가 있다.</p>
