<p>이 문서에서는 <a href="/wiki/2007%EB%85%84" title="2007년">2007년</a>에 발생한 <a href="/wiki/%ED%83%9C%ED%92%8D" title="태풍">태풍</a>들의 활동에 대해 기록한다.</p>
<p>2007년에는 <a href="/wiki/4%EC%9B%94_1%EC%9D%BC" title="4월 1일">4월 1일</a>에 <a href="#.EC.A0.9C1.ED.98.B8_.ED.83.9C.ED.92.8D_.EC.BD.A9.EB.A0.88.EC.9D.B4">제1호 태풍 콩레이</a>가 발생한 것을 시작으로, 연말까지 총 24개의 태풍이 발생하였다. 평균적으로 1년 동안 태풍 발생 개수가 26.7개인 것과 비교하면 낮은 수치이다.</p>

<p>제2호 태풍 위투 (YUTU)는 <a href="/wiki/5%EC%9B%94_18%EC%9D%BC" title="5월 18일">5월 18일</a>, <a href="/wiki/%EA%B4%8C" title="괌">괌</a> 섬 남서쪽 약 800km 부근 해상에서 발생했다. 처음에는 서진을 하다가, 발생 2일째부터 점차 북쪽으로 방향을 틀었다. <a href="/wiki/5%EC%9B%94_18%EC%9D%BC" title="5월 18일">5월 18일</a> 이후 빠르게 발달해, <a href="/wiki/5%EC%9B%94_19%EC%9D%BC" title="5월 19일">5월 19일</a>에 진로를 북동쪽으로 바꾼 뒤에 <a href="/wiki/5%EC%9B%94_21%EC%9D%BC" title="5월 21일">5월 21일</a>에 반경 250km의 "소형" 태풍이지만 중심 기압 935hPa, 최대 풍속 50m/s의 "매우 강"한 태풍으로 발달했다. 그러나 같은 날 저녁이 되면서 세력이 점차 약해져, 이동 속도가 빨라지면서 <a href="/wiki/5%EC%9B%94_23%EC%9D%BC" title="5월 23일">5월 23일</a> 아침 9시에 <a href="/wiki/%EB%8F%84%EC%BF%84" title="도쿄">도쿄</a> 동남동쪽 약 1600km 부근 해상에서 소멸되었다. "위투"는 <a href="/wiki/%EC%A4%91%EA%B5%AD" title="중국">중국</a>에서 제출한 이름으로, "옥토끼"를 뜻한다.</p>

<p>제4호 태풍 마니 (MAN-YI)는 <a href="/wiki/7%EC%9B%94_9%EC%9D%BC" title="7월 9일">7월 9일</a> 새벽, <a href="/wiki/%EA%B4%8C" title="괌">괌</a> 섬 남쪽 약 670km 부근 해상에서 발생했다. 발생 이후 계속 북서진하며 발달을 시작했다. <a href="/wiki/7%EC%9B%94_11%EC%9D%BC" title="7월 11일">7월 11일</a> <a href="/wiki/%EA%B4%8C" title="괌">괌</a> 섬 서북서쪽 약 1340km 부근 해상에 이르러서는 강도 "강"의 태풍으로 발달했고, 이 때를 전후해 <a href="/wiki/%EB%8C%80%ED%95%9C%EB%AF%BC%EA%B5%AD" title="대한민국">대한민국</a> 부근으로 접근할 것이라는 예보가 나왔다. 그 뒤 북북서진으로 방향을 점차 북쪽으로 틀면서 계속 발달해 <a href="/wiki/7%EC%9B%94_13%EC%9D%BC" title="7월 13일">7월 13일</a>에는 중심 기압 930hPa, 최대 풍속 50m/s까지 발달했다. 반경도 650km의 "대형" 태풍으로 발달했다. 대한민국에 가장 근접했던 <a href="/wiki/7%EC%9B%94_14%EC%9D%BC" title="7월 14일">7월 14일</a>에는 <a href="/wiki/%EC%A0%9C%EC%A3%BC%EB%8F%84" title="제주도">제주도</a>와 <a href="/wiki/%EB%82%A8%ED%95%B4" title="남해">남해</a>상을 중심으로 태풍경보와 태풍주의보, 강풍주의보가 발효되기도 했다. 7월 14일 정오를 넘기면서 태풍이 <a href="/wiki/%EC%9D%BC%EB%B3%B8" title="일본">일본</a> <a href="/wiki/%EA%B7%9C%EC%8A%88" title="규슈">규슈</a> 섬 남쪽 해안에 상륙하면서 대한민국은 점차 태풍의 영향권에서 벗어나기 시작했다. 이후 일본 열도를 따라 이동하며 <a href="/wiki/7%EC%9B%94_16%EC%9D%BC" title="7월 16일">7월 16일</a>에 <a href="/wiki/%EC%82%BF%ED%8F%AC%EB%A1%9C_%EC%8B%9C" title="삿포로 시">삿포로</a> 동남동쪽 약 1200 km 부근 해상에서 소멸되었다. "마니"는 <a href="/wiki/%ED%99%8D%EC%BD%A9" title="홍콩">홍콩</a>에서 제출한 이름으로 "해협의 이름"이다.</p>
<p>이 태풍으로 인해 대한민국은 서해상과 남해상에 파도가 크게 일고, 제주도와 동해안에 비가 조금 내렸을 뿐, 영향은 크지 않았다. 그러나 태풍이 북상하고 있던 7월 14일에 해수욕을 하다가 3명이 사망하는 사고가 발생하자, 태풍주의보 속에서도 해수욕을 통제하지 않은 해수욕장 측에 비난이 쏟아지기도 했다.<sup class="reference" id="cite_ref-2"><a href="#cite_note-2">[2]</a></sup> 태풍의 직접적인 영향을 받은 일본에서는 4명이 사망하거나 실종되고, 80여명이 부상당한 것으로 집계되었다.<sup class="reference" id="cite_ref-3"><a href="#cite_note-3">[3]</a></sup></p>

<p>제5호 태풍 우사기 (USAGI)는 <a href="/wiki/7%EC%9B%94_29%EC%9D%BC" title="7월 29일">7월 29일</a> 오후 3시, <a href="/wiki/%EA%B4%8C" title="괌">괌</a> 섬 북쪽 약 510 km 부근 해상에서 발생했다. 발생 초기의 예보는 주로 북진 내지 북북서진하여 일본 열도 중앙을 관통할 것으로 예상하여 대한민국에는 근접하지 않을 것으로 예보되었다. 그러나 예상보다 진로가 점차 서쪽으로 이동했고, <a href="/wiki/7%EC%9B%94_31%EC%9D%BC" title="7월 31일">7월 31일</a>에는 <a href="/wiki/%EA%B7%9C%EC%8A%88" title="규슈">규슈</a> 섬을 통과하여 <a href="/wiki/%EB%8F%99%ED%95%B4" title="동해">동해</a>상으로 진출할 것으로 예상되었다. 이 때의 세력은 중심 기압 955hPa, 최대 풍속 41m/s로 강도 "강"의 태풍이었다. 그리고 <a href="/wiki/8%EC%9B%94_1%EC%9D%BC" title="8월 1일">8월 1일</a>에는 945hPa로 중심 기압이 떨어지고, 최대 풍속도 45m/s까지 발달했다. 이후 <a href="/wiki/8%EC%9B%94_2%EC%9D%BC" title="8월 2일">8월 2일</a>에 규슈 섬에 상륙한 뒤 빠른 속도로 약화되기 시작했고, <a href="/wiki/8%EC%9B%94_3%EC%9D%BC" title="8월 3일">8월 3일</a> 새벽에 동해상으로 진출한 뒤, 빠른 속도로 북동진하여 <a href="/wiki/8%EC%9B%94_4%EC%9D%BC" title="8월 4일">8월 4일</a>에 소멸되었다. "우사기"는 <a href="/wiki/%EC%9D%BC%EB%B3%B8" title="일본">일본</a>에서 제출한 이름으로 "토끼"를 뜻한다.</p>
<p>이 태풍으로 인해 <a href="/wiki/%EB%8C%80%ED%95%9C%EB%AF%BC%EA%B5%AD_%EA%B8%B0%EC%83%81%EC%B2%AD" title="대한민국 기상청">대한민국 기상청</a>은 <a href="/wiki/8%EC%9B%94_2%EC%9D%BC" title="8월 2일">8월 2일</a> 오후에 남해 동부와 동해 남부 전 해상에 태풍주의보를 발령했지만, 3주 전에 찾아왔던 제4호 태풍 마니와 마찬가지로 대한민국에는 간접 영향만을 끼쳤다. 태풍이 상륙했던 일본 규슈 지역에서는 18명이 부상당하고, 산사태와 침수 피해로 인해 1만 3천여 세대가 긴급 대피하기도 했다.<sup class="reference" id="cite_ref-4"><a href="#cite_note-4">[4]</a></sup></p>

<p>제6호 태풍 파북 (PABUK)은 <a href="/wiki/8%EC%9B%94_5%EC%9D%BC" title="8월 5일">8월 5일</a> 오후 3시, <a href="/wiki/%EC%9D%BC%EB%B3%B8" title="일본">일본</a> <a href="/wiki/%EC%98%A4%ED%82%A4%EB%82%98%EC%99%80_%EC%84%AC" title="오키나와 섬">오키나와 섬</a> 남동쪽 약 1100 km 부근 해상에서 발생했다. 발생 이후 줄곧 서쪽 또는 서북서쪽으로 이동해, 소멸될 때까지 북위 25도 위로 북상하지 않았다. 발생 30시간 만인 <a href="/wiki/8%EC%9B%94_6%EC%9D%BC" title="8월 6일">8월 6일</a> 오후 9시에 "중형"급 세력의 태풍으로 발달했으나, <a href="/wiki/8%EC%9B%94_8%EC%9D%BC" title="8월 8일">8월 8일</a> 새벽에 <a href="/wiki/%ED%83%80%EC%9D%B4%EC%99%84" title="타이완">타이완 섬</a> 남쪽 해안을 스치면서 약해지기 시작했다. 그런 뒤 <a href="/wiki/8%EC%9B%94_9%EC%9D%BC" title="8월 9일">8월 9일</a> 오후 9시에 <a href="/wiki/%ED%99%8D%EC%BD%A9" title="홍콩">홍콩</a> 부근에서 소멸되었다. "파북"은 <a href="/wiki/%EB%9D%BC%EC%98%A4%EC%8A%A4" title="라오스">라오스</a>에서 제출한 이름으로 민물고기의 한 종류이다.</p>

<p>제8호 태풍 스팟 (SEPAT)은 <a href="/wiki/8%EC%9B%94_10%EC%9D%BC" title="8월 10일">8월 10일</a> 오전 3시, <a href="/wiki/%EA%B4%8C" title="괌">괌</a> 섬 북서쪽 약 2,100 km 부근 해상에서 발생했다. 발생한 지 채 24시간도 지나지 않은 같은 날 오후 9시에 강도 "중"의 태풍으로 발달했고, <a href="/wiki/8%EC%9B%94_11%EC%9D%BC" title="8월 11일">8월 11일</a> 오전 9시에 강도 "강"의 태풍으로 발달했다. 이후 <a href="/wiki/8%EC%9B%94_16%EC%9D%BC" title="8월 16일">8월 16일</a>에는 강도 "매우 강"의 태풍으로 발달했다. 중심 기압 906 hPa, 최대 풍속 72m/s를 기록함으로써, 2007년 발생한 태풍 중 가장 강한 태풍으로 기록되었다. 발생 후 계속 시속 10~13km의 속도로 북서쪽으로 이동하다가, 서쪽으로 진행 방향을 튼 뒤, <a href="/wiki/9%EC%9B%94_7%EC%9D%BC" title="9월 7일">9월 7일</a> 새벽에 <a href="/wiki/%ED%83%80%EC%9D%B4%EC%99%84" title="타이완">타이완 섬</a>에 상륙했고, 계속 북서진하여 <a href="/wiki/9%EC%9B%94_8%EC%9D%BC" title="9월 8일">9월 8일</a>에 <a href="/wiki/%EC%A4%91%EA%B5%AD" title="중국">중국</a> <a href="/wiki/%EC%83%81%ED%95%98%EC%9D%B4_%EC%8B%9C" title="상하이 시">상하이 시</a> 서쪽 내륙 지역에서 소멸되었다. "스팟"은 <a href="/wiki/%EB%A7%90%EB%A0%88%EC%9D%B4%EC%8B%9C%EC%95%84" title="말레이시아">말레이시아</a>에서 제출한 이름으로 물고기의 한 종류이다.</p>
<p>이 태풍이 직접 상륙했던 <a href="/wiki/%ED%83%80%EC%9D%B4%EC%99%84" title="타이완">타이완 섬</a>에서는 4명의 사망자와 5,000명의 이재민을 발생시켰고<sup class="reference" id="cite_ref-6"><a href="#cite_note-6">[6]</a></sup>, 뒤이어 상륙했던 <a href="/wiki/%EC%A4%91%EA%B5%AD" title="중국">중국</a> <a href="/wiki/%ED%91%B8%EC%A0%A0_%EC%84%B1" title="푸젠 성">푸젠 성</a>과 <a href="/wiki/%EC%A0%80%EC%9E%A5_%EC%84%B1" title="저장 성">저장 성</a>에서는 30명이 사망하거나 실종되고, 120여 명이 부상을 입는 등 총 22억 <a href="/wiki/%EC%A4%91%EA%B5%AD_%EC%9C%84%EC%95%88" title="중국 위안">위안</a>의 재산피해가 발생했다.<sup class="reference" id="cite_ref-7"><a href="#cite_note-7">[7]</a></sup></p>

<p>제9호 태풍 피토 (FITOW)는 <a href="/wiki/2007%EB%85%84" title="2007년">2007년</a> <a href="/wiki/8%EC%9B%94_29%EC%9D%BC" title="8월 29일">8월 29일</a> 오후 3시에 <a href="/wiki/%EA%B4%8C" title="괌">괌</a> 섬 북동쪽 해상에서 발생했다. 발생 직후 북동쪽으로 이동하다가, <a href="/wiki/8%EC%9B%94_30%EC%9D%BC" title="8월 30일">8월 30일</a>에 진로를 북쪽으로 틀면서 세력을 "강"까지 키웠다. 이후 더 이상 발달하지 않고 세력을 유지하면서 서쪽으로 느리게 이동하여 <a href="/wiki/9%EC%9B%94_4%EC%9D%BC" title="9월 4일">9월 4일</a>에 <a href="/wiki/%EC%9D%BC%EB%B3%B8" title="일본">일본</a> 남쪽 해상까지 진출했다. 그리고 진로를 북쪽으로 틀면서 <a href="/wiki/9%EC%9B%94_6%EC%9D%BC" title="9월 6일">9월 6일</a> 밤에 <a href="/wiki/%EB%8F%84%EC%BF%84" title="도쿄">도쿄</a> 부근 해상까지 진출했다. <a href="/wiki/9%EC%9B%94_7%EC%9D%BC" title="9월 7일">9월 7일</a> 새벽 일본에 상륙하여 일본의 <a href="/wiki/%EA%B0%84%ED%86%A0_%EC%A7%80%EB%B0%A9" title="간토 지방">간토 지방</a>과 <a href="/wiki/%EB%8F%84%ED%98%B8%EC%BF%A0_%EC%A7%80%EB%B0%A9" title="도호쿠 지방">도호쿠 지방</a>을 거쳐 <a href="/wiki/%EC%93%B0%EA%B0%80%EB%A3%A8_%ED%95%B4%ED%98%91" title="쓰가루 해협">쓰가루 해협</a>을 통과한 뒤, <a href="/wiki/9%EC%9B%94_8%EC%9D%BC" title="9월 8일">9월 8일</a> 아침 <a href="/wiki/%ED%99%8B%EC%B9%B4%EC%9D%B4%EB%8F%84" title="홋카이도">홋카이도</a>에 상륙한 상태에서 <a href="/wiki/%EC%98%A8%EB%8C%80_%EC%A0%80%EA%B8%B0%EC%95%95" title="온대 저기압">온대저기압</a>으로 변질되었다. "피토"는 <a href="/wiki/%EB%AF%B8%ED%81%AC%EB%A1%9C%EB%84%A4%EC%8B%9C%EC%95%84" title="미크로네시아">미크로네시아</a>에서 제출한 이름으로 꽃의 이름이다.</p>
<p>태풍이 상륙했던 <a href="/wiki/%EC%9D%BC%EB%B3%B8" title="일본">일본</a>에서는 9명이 사망·실종되고, 항공기와 철도의 운행이 중단되었다. 또, 도로가 붕괴되어 산간의 온천 지대에서는 1,400명이 고립되기도 했다.<sup class="reference" id="cite_ref-8"><a href="#cite_note-8">[8]</a></sup></p>

"다나스"는 <a href="/wiki/%ED%95%84%EB%A6%AC%ED%95%80" title="필리핀">필리핀</a>에서 제출한 이름으로 "경험"이란 뜻이다.</p>

<p>제11호 태풍 나리 (NARI)는 <a href="/wiki/2007%EB%85%84" title="2007년">2007년</a> <a href="/wiki/9%EC%9B%94_13%EC%9D%BC" title="9월 13일">9월 13일</a> 오후 3시에 <a href="/wiki/%EC%9D%BC%EB%B3%B8" title="일본">일본</a> <a href="/wiki/%EC%98%A4%ED%82%A4%EB%82%98%EC%99%80_%EC%84%AC" title="오키나와 섬">오키나와 섬</a> 남동쪽 해상에서 발생했다. 발생 이후 서북서쪽으로 이동하면서 급속도로 발달해, <a href="/wiki/9%EC%9B%94_14%EC%9D%BC" title="9월 14일">9월 14일</a> 오후 9시에 중심 기압 935hPa, 최대 풍속 초속 50m의 매우 강한 태풍으로 발달했다. 이후 방향을 점차 북쪽으로 틀었고, <a href="/wiki/9%EC%9B%94_16%EC%9D%BC" title="9월 16일">9월 16일</a> 12시에 <a href="/wiki/%EC%A0%9C%EC%A3%BC%EB%8F%84" title="제주도">제주도</a> 동쪽 해안을 통과했다. 같은 날 저녁에 <a href="/wiki/%EC%A0%84%EB%9D%BC%EB%82%A8%EB%8F%84" title="전라남도">전라남도</a> <a href="/wiki/%EA%B3%A0%ED%9D%A5%EA%B5%B0" title="고흥군">고흥군</a>에 상륙했고, <a href="/wiki/9%EC%9B%94_17%EC%9D%BC" title="9월 17일">9월 17일</a> 소멸되었다. 나리는 <a href="/wiki/%EB%8C%80%ED%95%9C%EB%AF%BC%EA%B5%AD" title="대한민국">대한민국</a>에서 제출한 이름이다.</p>
<p>이 태풍으로 인해 태풍이 직접 통과한 <a href="/wiki/%EC%A0%9C%EC%A3%BC%EB%8F%84" title="제주도">제주도</a>에는 엄청난 피해가 발생하였다.</p>

<p>제12호 태풍 위파 (WIPHA)는 <a href="/wiki/2007%EB%85%84" title="2007년">2007년</a> <a href="/wiki/9%EC%9B%94_16%EC%9D%BC" title="9월 16일">9월 16일</a> 오전 9시에 <a href="/wiki/%EC%9D%BC%EB%B3%B8" title="일본">일본</a> <a href="/wiki/%EC%98%A4%ED%82%A4%EB%82%98%EC%99%80_%EC%84%AC" title="오키나와 섬">오키나와 섬</a> 남남동쪽 해상에서 발생했고, 12시간 만에 중형급 태풍으로 발달했다. 예보 기관들은 초기에 저마다 <a href="/wiki/%EC%A4%91%EA%B5%AD" title="중국">중국</a> 쪽으로 가거나 <a href="/wiki/%ED%95%9C%EB%B0%98%EB%8F%84" title="한반도">한반도</a>의 서해안으로 향할 것으로 예상했으나, 점차 예상 진로는 중국 쪽으로 모아졌고, 태풍은 실제로 계속 북서쪽으로 이동했다. 이동하면서 계속 발달하여 <a href="/wiki/%ED%83%80%EC%9D%B4%EC%99%84" title="타이완">타이완 섬</a> 부근에 이르렀을 때 중심 부근에서는 초속 51m의 바람이 불기도 했다. 이후 <a href="/wiki/9%EC%9B%94_19%EC%9D%BC" title="9월 19일">9월 19일</a> 새벽에 중국 남동쪽 해안에 상륙했고, <a href="/wiki/9%EC%9B%94_19%EC%9D%BC" title="9월 19일">9월 19일</a> 밤에 내륙에서 소멸되었다. "위파"는 <a href="/wiki/%ED%83%80%EC%9D%B4" title="타이">태국</a>에서 제출한 이름으로 숙녀 이름이다.</p>
<p><a href="/wiki/%EC%A4%91%EA%B5%AD" title="중국">중국</a> 정부에 따르면, 태풍이 직접 상륙한 <a href="/wiki/%EC%A4%91%EA%B5%AD" title="중국">중국</a> <a href="/wiki/%EC%A0%80%EC%9E%A5_%EC%84%B1" title="저장 성">저장 성</a>에서는 8명이 사망·실종되고, <a href="/wiki/%ED%91%B8%EC%A0%A0_%EC%84%B1" title="푸젠 성">푸젠 성</a>에서는 가옥 9,600여 동이 파손되는 피해를 입어, 중국 전역으로는 총 8,000억원의 피해가 발생했다고 발표했다.<sup class="reference" id="cite_ref-9"><a href="#cite_note-9">[9]</a></sup> 또, 간접 영향을 받은 <a href="/wiki/%EC%A1%B0%EC%84%A0%EB%AF%BC%EC%A3%BC%EC%A3%BC%EC%9D%98%EC%9D%B8%EB%AF%BC%EA%B3%B5%ED%99%94%EA%B5%AD" title="조선민주주의인민공화국">조선민주주의인민공화국</a> <a href="/wiki/%ED%99%A9%ED%95%B4%EB%8F%84" title="황해도">황해도</a>·<a href="/wiki/%ED%8F%89%EC%95%88%EB%8F%84" title="평안도">평안도</a> 지역에도 <a href="/wiki/9%EC%9B%94_18%EC%9D%BC" title="9월 18일">9월 18일</a>에서 <a href="/wiki/9%EC%9B%94_21%EC%9D%BC" title="9월 21일">9월 21일</a>까지 300~500mm에 달하는 비가 쏟아져 농경지 109,000여 정보가 침수되고, 수많은 가옥과 공공 건물이 파손되었다. 이들 지역은 <a href="/wiki/2007%EB%85%84" title="2007년">2007년</a> <a href="/wiki/8%EC%9B%94" title="8월">8월</a>에도 집중호우로 피해를 입은 지역으로 한 달 만에 다시 큰 피해를 입고 말았다.<sup class="reference" id="cite_ref-10"><a href="#cite_note-10">[10]</a></sup></p>

<p>제15호 태풍 크로사 (KROSA)는 <a href="/wiki/2007%EB%85%84" title="2007년">2007년</a> <a href="/wiki/10%EC%9B%94_2%EC%9D%BC" title="10월 2일">10월 2일</a> 오전 9시 <a href="/wiki/%ED%95%84%EB%A6%AC%ED%95%80" title="필리핀">필리핀</a> <a href="/wiki/%EB%A7%88%EB%8B%90%EB%9D%BC" title="마닐라">마닐라</a> 동북동쪽 약 1,100 km 부근 해상에서 발생했다. 이 태풍은 북서쪽으로 이동하면서 급격하게 발달하기 시작했다. 발생한 지 48시간 만인 <a href="/wiki/10%EC%9B%94_4%EC%9D%BC" title="10월 4일">10월 4일</a> 오전 9시에는 중심 기압 940hPa, 중심 부근에서 초속 46m의 바람이 부는 매우 강한 태풍으로 발달했다. 이 태풍은 계속 북서진 내지 북북서진 하여 <a href="/wiki/10%EC%9B%94_6%EC%9D%BC" title="10월 6일">10월 6일</a> 낮에 <a href="/wiki/%ED%83%80%EC%9D%B4%EC%99%84" title="타이완">타이완 섬</a>에 상륙하였다. 그러나 곧 다시 남쪽으로 방향을 튼 뒤 동쪽으로 이동하여 <a href="/wiki/%ED%83%80%EC%9D%B4%EC%99%84" title="타이완">타이완 섬</a>을 빠져 나왔으며, 이 과정에서 태풍은 강도 "매우 강"의 태풍에서 "강"의 태풍으로 많이 약화되었다. 이후 진로는 <a href="/wiki/%ED%83%80%EC%9D%B4%EC%99%84" title="타이완">타이완 섬</a>에 재상륙한 뒤, 중국 해안에 바짝 붙어서 북상하다가 그대로 온대저기압으로 변질되었다. 당초 진로는 태풍의 세력을 유지하며 남해를 통과, <a href="/wiki/%EC%9D%BC%EB%B3%B8" title="일본">일본</a> <a href="/wiki/%EA%B7%9C%EC%8A%88" title="규슈">규슈 섬</a>에 상륙함으로써 <a href="/wiki/%EB%8C%80%ED%95%9C%EB%AF%BC%EA%B5%AD" title="대한민국">대한민국</a>에도 영향을 미칠 것으로 예상되었으나, 태풍이 일찍 소멸되면서 <a href="/wiki/%EC%A0%9C%EC%A3%BC%EB%8F%84" title="제주도">제주도</a>에 40~50mm의 비가 내리고 <a href="/wiki/%EB%8C%80%ED%95%9C%EB%AF%BC%EA%B5%AD" title="대한민국">대한민국</a> 남부 지방에 풍랑특보만 발효되었을 뿐이었다.<sup class="reference" id="cite_ref-11"><a href="#cite_note-11">[11]</a></sup> "크로사"는 <a href="/wiki/%EC%BA%84%EB%B3%B4%EB%94%94%EC%95%84" title="캄보디아">캄보디아</a>에서 제출한 이름으로 "학"을 뜻한다.</p>

<p>제16호 태풍 하이옌 (HAIYAN)은 <a href="/wiki/2007%EB%85%84" title="2007년">2007년</a> <a href="/wiki/10%EC%9B%94_5%EC%9D%BC" title="10월 5일">10월 5일</a> 오전 9시 <a href="/wiki/%EC%9D%BC%EB%B3%B8" title="일본">일본</a> <a href="/wiki/%EB%8F%84%EC%BF%84" title="도쿄">도쿄</a> 동남동쪽 먼 바다에서 발생했다. 이 태풍이 발생한 지역은 동경 170도로, 북서 태평양의 동쪽 끝이며, 상당한 고위도에서 발생해 크게 발달하지 못하고 동경 170도선을 따라 빠르게 북진하다가 발생한 지 18시간 만인 <a href="/wiki/10%EC%9B%94_6%EC%9D%BC" title="10월 6일">10월 6일</a> 오후 3시에 <a href="/wiki/%EC%9D%BC%EB%B3%B8" title="일본">일본</a> <a href="/wiki/%EC%82%BF%ED%8F%AC%EB%A1%9C_%EC%8B%9C" title="삿포로 시">삿포로</a> 동남동쪽 약 2,780 km 부근 해상에서 열대 저기압으로 다시 세력이 약화되었다. "하이옌"은 <a href="/wiki/%EC%A4%91%EA%B5%AD" title="중국">중국</a>에서 제출한 이름으로 "바다제비"를 뜻한다.</p>
<p><a href="/wiki/%EB%AF%B8%EA%B5%AD_%ED%95%A9%EB%8F%99%ED%83%9C%ED%92%8D%EA%B2%BD%EB%B3%B4%EC%84%BC%ED%84%B0" title="미국 합동태풍경보센터">JTWC</a>는 당초 이 태풍을 열대저기압으로 인정하지 않았지만, 사후 해석을 통해 열대저기압 (지정 번호 27W) 으로 인정했다.</p>

<p>제17호 태풍 버들 (PODUL)은 <a href="/wiki/2007%EB%85%84" title="2007년">2007년</a> <a href="/wiki/10%EC%9B%94_5%EC%9D%BC" title="10월 5일">10월 5일</a> 오전 9시 <a href="/wiki/%EC%9D%BC%EB%B3%B8" title="일본">일본</a>의 먼 남해상에서 발생했다. <a href="/wiki/10%EC%9B%94_3%EC%9D%BC" title="10월 3일">10월 3일</a> 경에 저기압 상태로 처음 모습을 드러냈는데, 속보에서는 <a href="/wiki/%EC%9D%BC%EB%B3%B8" title="일본">일본</a> 남쪽해상에서 동쪽으로 이동하다가 갑작스럽게 태풍으로 발달한 것으로 발표 되었으나, 사후 해석을 통해 태풍으로의 발달 위치가 정정되고, 최대 세력도 다시 해석되었다. 태풍은 2007년 10월 7일 오후 3시경에 온대저기압으로 바뀌었다. "버들"은 <a href="/wiki/%EC%A1%B0%EC%84%A0%EB%AF%BC%EC%A3%BC%EC%A3%BC%EC%9D%98%EC%9D%B8%EB%AF%BC%EA%B3%B5%ED%99%94%EA%B5%AD" title="조선민주주의인민공화국">조선민주주의인민공화국</a>에서 제출한 이름이다.</p>
<p>한편, <a href="/wiki/%EB%AF%B8%EA%B5%AD_%ED%95%A9%EB%8F%99%ED%83%9C%ED%92%8D%EA%B2%BD%EB%B3%B4%EC%84%BC%ED%84%B0" title="미국 합동태풍경보센터">JTWC</a>는 이것을 열대저기압으로 해석하지 않았다.</p>

<p>제18호 태풍 링링 (LINGLING)은 <a href="/wiki/2007%EB%85%84" title="2007년">2007년</a> <a href="/wiki/10%EC%9B%94_12%EC%9D%BC" title="10월 12일">10월 12일</a> 오후 3시 <a href="/wiki/%EC%9D%BC%EB%B3%B8" title="일본">일본</a> <a href="/wiki/%EB%8F%84%EC%BF%84" title="도쿄">도쿄</a> 동남동쪽 약 3,350 km 부근 해상에서 발생했다. 제16호 태풍 하이옌과 비슷한 장소에서 발생하여, 태풍 하이옌처럼 그리 크게 발달하지 못했다. 천천히 북서쪽으로 이동하다가 <a href="/wiki/10%EC%9B%94_14%EC%9D%BC" title="10월 14일">10월 14일</a>부터는 북동쪽으로 방향을 틀어, <a href="/wiki/10%EC%9B%94_15%EC%9D%BC" title="10월 15일">10월 15일</a> 오후에 동쪽으로 빠르게 이동하며 온대 저기압으로 변질되었다. "링링"은 <a href="/wiki/%ED%99%8D%EC%BD%A9" title="홍콩">홍콩</a>에서 제출한 이름으로, 소녀의 애칭 중 하나이다.</p>

<p>제19호 태풍 가지키 (KAJIKI)는 <a href="/wiki/2007%EB%85%84" title="2007년">2007년</a> <a href="/wiki/10%EC%9B%94_19%EC%9D%BC" title="10월 19일">10월 19일</a> 오후 3시 <a href="/wiki/%EA%B4%8C" title="괌">괌 섬</a> 북북서쪽 약 730 km 부근 해상에서 발생했다. 발생 이후 북서쪽으로 이동하며 발생 24시간이 채 안 된 <a href="/wiki/10%EC%9B%94_20%EC%9D%BC" title="10월 20일">10월 20일</a> 오전 9시에 중심 기압 960hPa인 강도 "강"의 태풍으로 발달했다. 그 후 조금 더 발달한 뒤, <a href="/wiki/10%EC%9B%94_20%EC%9D%BC" title="10월 20일">10월 20일</a> 오후부터 점차 동쪽으로 전향하기 시작했다. 그리고 빠른 속도로 고위도 해상으로 진출하여 세력이 약화되다가, <a href="/wiki/10%EC%9B%94_22%EC%9D%BC" title="10월 22일">10월 22일</a> 오후 3시 <a href="/wiki/%EC%9D%BC%EB%B3%B8" title="일본">일본</a> <a href="/wiki/%EC%82%BF%ED%8F%AC%EB%A1%9C_%EC%8B%9C" title="삿포로 시">삿포로</a> 동쪽 먼 해상에서 온대 저기압으로 변질되었다. "가지키"는 <a href="/wiki/%EC%9D%BC%EB%B3%B8" title="일본">일본</a>에서 제출한 이름으로, 물고기의 한 종류인 "청새치"를 뜻한다.</p>

<p>제20호 태풍 파사이 (FAXAI)는 <a href="/wiki/2007%EB%85%84" title="2007년">2007년</a> <a href="/wiki/10%EC%9B%94_26%EC%9D%BC" title="10월 26일">10월 26일</a> 오전 9시 <a href="/wiki/%EC%9D%BC%EB%B3%B8" title="일본">일본</a> <a href="/wiki/%EC%98%A4%ED%82%A4%EB%82%98%EC%99%80_%EC%84%AC" title="오키나와 섬">오키나와 섬</a> 남동쪽 약 540 km 부근 해상에서 발생했다. 발생 직후에는 서쪽으로 이동하다 곧 전향하여 동쪽으로 이동하기 시작했다. 오키나와 동쪽 해상에 이르러서 이동 속도도 점차 빨라져 <a href="/wiki/10%EC%9B%94_27%EC%9D%BC" title="10월 27일">10월 27일</a> 밤에는 <a href="/wiki/%EB%8F%84%EC%BF%84" title="도쿄">도쿄</a> 동쪽 가까운 해상까지 진출했다. 이때 세력은 중심 기압 980hPa, 최대 풍속 30m/s의 중형 태풍이었다. 이 태풍이 기록한 강수량·풍속이 일본 수도권의 <a href="/wiki/%EC%86%8C%EB%B6%80_%EB%B3%B8%EC%84%A0" title="소부 본선">소부 본선</a>·<a href="/wiki/%EB%82%98%EB%A6%AC%ED%83%80_%EC%84%A0" title="나리타 선">나리타 선</a> 등 많은 철도 교통망의 강수량·풍속 규제치를 넘어, 수도권 철도 중 일부의 운행이 중단되는 등 차질을 빚기도 했다.<sup class="reference" id="cite_ref-12"><a href="#cite_note-12">[12]</a></sup> 이후 태풍은 <a href="/wiki/10%EC%9B%94_27%EC%9D%BC" title="10월 27일">10월 27일</a> 오후에 도쿄 북동쪽 해상까지 진출하며 온대 저기압으로 변질되었다. "파사이"는 <a href="/wiki/%EB%9D%BC%EC%98%A4%EC%8A%A4" title="라오스">라오스</a>에서 제출한 이름으로 숙녀 이름의 하나이다.</p>

<p>제21호 태풍 페이파 (PEIPAH)는 <a href="/wiki/2007%EB%85%84" title="2007년">2007년</a> <a href="/wiki/11%EC%9B%94_3%EC%9D%BC" title="11월 3일">11월 3일</a> 오후 9시 <a href="/wiki/%ED%95%84%EB%A6%AC%ED%95%80" title="필리핀">필리핀</a> <a href="/wiki/%EB%A7%88%EB%8B%90%EB%9D%BC" title="마닐라">마닐라</a> 동북동쪽 약 840km 부근 해상에서 발생했다. 발생 이후 점차 발달하면서 서쪽으로 이동하였고, <a href="/wiki/11%EC%9B%94_4%EC%9D%BC" title="11월 4일">11월 4일</a> 밤에 중형급 세력으로 필리핀 <a href="/wiki/%EB%A3%A8%EC%86%90_%EC%84%AC" title="루손 섬">루손 섬</a>에 상륙하였다. <a href="/wiki/11%EC%9B%94_5%EC%9D%BC" title="11월 5일">11월 5일</a> 아침에는 루손 섬 서쪽 해상, 즉 <a href="/wiki/%EB%82%A8%EC%A4%91%EA%B5%AD%ED%95%B4" title="남중국해">남중국해</a> 해상으로 빠져나왔다. <a href="/wiki/11%EC%9B%94_6%EC%9D%BC" title="11월 6일">11월 6일</a>에는 줄곧 서쪽을 향하던 태풍의 진로가 북서쪽으로 바뀌면서 남중국해 해상에서 정체하였고, <a href="/wiki/11%EC%9B%94_7%EC%9D%BC" title="11월 7일">11월 7일</a>부터는 <a href="/wiki/%EB%B2%A0%ED%8A%B8%EB%82%A8" title="베트남">베트남</a>을 향해 느리게 남서쪽으로 이동하기 시작했다. 그러나 남서쪽으로 이동하면서 세력이 급격히 약화되었고, <a href="/wiki/11%EC%9B%94_9%EC%9D%BC" title="11월 9일">11월 9일</a> 새벽에 소멸되었다. "페이파"는 <a href="/wiki/%EB%A7%88%EC%B9%B4%EC%98%A4" title="마카오">마카오</a>에서 제출한 이름으로 애완용 물고기를 의미한다.</p>

<p>제23호 태풍 미탁 (MITAG)은 <a href="/wiki/2007%EB%85%84" title="2007년">2007년</a> <a href="/wiki/11%EC%9B%94_20%EC%9D%BC" title="11월 20일">11월 20일</a> 오후 9시 <a href="/wiki/%EA%B4%8C" title="괌">괌 섬</a> 서쪽 약 900km 부근 해상에서 발생했다. 발생 이후 <a href="/wiki/%ED%95%84%EB%A6%AC%ED%95%80" title="필리핀">필리핀</a> <a href="/wiki/%EB%A3%A8%EC%86%90_%EC%84%AC" title="루손 섬">루손 섬</a>을 향하여 서쪽으로 이동하면서 발달하기 시작했다. 줄곧 서진하던 이 태풍은 <a href="/wiki/11%EC%9B%94_23%EC%9D%BC" title="11월 23일">11월 23일</a>부터 필리핀 <a href="/wiki/%EB%A7%88%EB%8B%90%EB%9D%BC" title="마닐라">마닐라</a> 동쪽 해상에서 정체하기 시작했다. 이후에는 느리게 서북서진하며 세력이 약해지다가, <a href="/wiki/11%EC%9B%94_25%EC%9D%BC" title="11월 25일">11월 25일</a> 밤에 필리핀 루손 섬 북동쪽 해안에 상륙했다. 그리고 루손 섬을 빠져나와, <a href="/wiki/%ED%83%80%EC%9D%B4%EC%99%84" title="타이완">타이완 섬</a> 남쪽 해상에서 동쪽으로 전향하면서 세력이 크게 약해졌고, <a href="/wiki/11%EC%9B%94_28%EC%9D%BC" title="11월 28일">11월 28일</a> 새벽에 소멸되었다. "미탁"은 <a href="/wiki/%EB%AF%B8%ED%81%AC%EB%A1%9C%EB%84%A4%EC%8B%9C%EC%95%84" title="미크로네시아">미크로네시아</a>에서 제출한 이름으로 여성의 이름의 하나이다. 이 태풍이 동쪽의 차가운 대륙 고기압의 영향을 받아서 정체하다가 필리핀 동쪽 해상에 위치해 있던 제24호 태풍 히가비스과의 거리가 1,000km 이내로 가까워졌고, 위치 상 가까운 거리에 있는 두 열대 저기압이 서로의 진로에 영향을 주는 후지와라 효과가 발생했기 때문이다. 이로 인해 "하기비스"는 동쪽으로, "미탁"은 서쪽으로 이동하는 진로가 나타났다. <p>제24호 태풍 하기비스 (HAGIBIS)는 <a href="/wiki/2007%EB%85%84" title="2007년">2007년</a> <a href="/wiki/11%EC%9B%94_21%EC%9D%BC" title="11월 21일">11월 21일</a> 오전 3시 <a href="/wiki/%ED%95%84%EB%A6%AC%ED%95%80" title="필리핀">필리핀</a> <a href="/wiki/%EB%A7%88%EB%8B%90%EB%9D%BC" title="마닐라">마닐라</a> 남서쪽 약 690 km 부근 해상에서 발생했다. 서북서진하면서 발달하여, <a href="/wiki/11%EC%9B%94_22%EC%9D%BC" title="11월 22일">11월 22일</a>에는 강도 "강"의 태풍으로 발달했다. 그러나 예보 기관들은 태풍이 <a href="/wiki/%EB%B2%A0%ED%8A%B8%EB%82%A8" title="베트남">베트남</a>에 상륙하여, 베트남 내륙 지역에서 소멸될 것으로 예측했지만, 정작 태풍은 <a href="/wiki/11%EC%9B%94_23%EC%9D%BC" title="11월 23일">11월 23일</a>부터 베트남 동쪽 해상에서 정체하기 시작했다. 서남서진-북서진-동진을 반복하며 정체하던 이 태풍은 <a href="/wiki/11%EC%9B%94_24%EC%9D%BC" title="11월 24일">11월 24일</a> 밤부터 점차 동쪽으로 이동하여, <a href="/wiki/11%EC%9B%94_27%EC%9D%BC" title="11월 27일">11월 27일</a>에는 필리핀 <a href="/wiki/%EB%A3%A8%EC%86%90_%EC%84%AC" title="루손 섬">루손 섬</a> 남단을 강타하였다. 이 과정에서 태풍은 크게 약화되었고, 필리핀 동쪽 해상으로 빠져나온 태풍은 곧 소멸되었다.</p>
<p>이처럼 이 태풍은 서쪽으로 이동하다가 정체하고, 그 후 진로를 180도 바꿔 동쪽으로 이동하는 이상 진로를 보였다. 이러한 진로를 보인 이유는 이 태풍이 북서쪽의 차가운 대륙 고기압의 영향을 받아서 정체하다가 <a href="/wiki/%ED%95%84%EB%A6%AC%ED%95%80" title="필리핀">필리핀</a> 동쪽 해상에 위치해 있던 제23호 태풍 미탁과의 거리가 1,000 km 이내로 가까워졌고, 위치 상 가까운 거리에 있는 두 열대 저기압이 서로의 진로에 영향을 주는 <a href="/wiki/%ED%9B%84%EC%A7%80%EC%99%80%EB%9D%BC_%ED%9A%A8%EA%B3%BC" title="후지와라 효과">후지와라 효과</a>가 발생했기 때문이다.<sup class="reference" id="cite_ref-13"><a href="#cite_note-13">[13]</a></sup> 이로 인해 "하기비스"는 동쪽으로, "미탁"은 서쪽으로 이동하는 진로가 나타났다. "하기비스"는 <a href="/wiki/%ED%95%84%EB%A6%AC%ED%95%80" title="필리핀">필리핀</a>에서 제출한 이름으로 "빠름"을 의미한다.</p>
