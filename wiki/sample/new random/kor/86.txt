<p><b>고양이과</b>(<span style="font-size: smaller;"><a href="/wiki/%EB%8C%80%ED%95%9C%EB%AF%BC%EA%B5%AD_%ED%91%9C%EC%A4%80%EC%96%B4" title="대한민국 표준어">표준어</a>:</span> 고양잇과)는 <a href="/wiki/%EC%8B%9D%EC%9C%A1%EB%AA%A9" title="식육목">식육목</a> 동물 중에서도 가장 육식성이 강한 동물들의 과이다. <a href="/wiki/%EC%82%AC%EC%9E%90" title="사자">사자</a>와 <a href="/wiki/%ED%98%B8%EB%9E%91%EC%9D%B4" title="호랑이">호랑이</a>를 제외하고는 대부분 암컷이 수컷보다 크며, 진화학적으로는 <a href="/wiki/%EC%82%AC%ED%96%A5%EA%B3%A0%EC%96%91%EC%9D%B4%EA%B3%BC" title="사향고양이과">사향고양이과</a>, <a class="mw-redirect" href="/wiki/%ED%95%98%EC%9D%B4%EC%97%90%EB%82%98%EA%B3%BC" title="하이에나과">하이에나과</a>, <a class="mw-redirect" href="/wiki/%EB%AA%BD%EA%B5%AC%EC%8A%A4%EA%B3%BC" title="몽구스과">몽구스과</a>에 가깝다. 첫 고양이과는 4000만 년 전 <a href="/wiki/%EC%97%90%EC%98%A4%EC%84%B8" title="에오세">에오세</a>에 출현했다. 기원전 20~50세기 경에 길들여진 <a href="/wiki/%EA%B3%A0%EC%96%91%EC%9D%B4" title="고양이">집고양이</a>가 있고, 이의 야생종이 아프리카와 서아시아에 서식하고 있다. 고양이과 동물로는 <a href="/wiki/%EC%82%AC%EC%9E%90" title="사자">사자</a>, <a href="/wiki/%ED%98%B8%EB%9E%91%EC%9D%B4" title="호랑이">호랑이</a>, <a href="/wiki/%ED%91%9C%EB%B2%94" title="표범">표범</a>, <a href="/wiki/%EC%9E%AC%EA%B7%9C%EC%96%B4" title="재규어">재규어</a>, <a href="/wiki/%EC%B9%98%ED%83%80" title="치타">치타</a>, <a class="mw-redirect" href="/wiki/%EC%82%B4%EC%BE%A1%EC%9D%B4" title="살쾡이">살쾡이</a>, <a href="/wiki/%ED%93%A8%EB%A7%88" title="퓨마">퓨마</a> 등이 있다.</p>

<h2><span class="mw-headline" id=".EB.AC.BC.EC.9D.84_.EC.8B.AB.EC.96.B4.ED.95.98.EB.8A.94_.EC.8A.B5.EC.84.B1">물을 싫어하는 습성</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EA%B3%A0%EC%96%91%EC%9D%B4%EA%B3%BC&amp;action=edit&amp;section=1" title="부분 편집: 물을 싫어하는 습성">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
