<p><b>효회제 유선</b>(孝懷帝 劉禪, <a href="/wiki/207%EB%85%84" title="207년">207년</a> ~ <a href="/wiki/271%EB%85%84" title="271년">271년</a>)은 <a href="/wiki/%EC%B4%89%ED%95%9C" title="촉한">촉한</a>의 제2대 황제(재위: <a href="/wiki/223%EB%85%84" title="223년">223년</a> ~ <a href="/wiki/263%EB%85%84" title="263년">263년</a>)이다. 《<a href="/wiki/%EC%82%BC%EA%B5%AD%EC%A7%80" title="삼국지">삼국지</a>》의 저자 <a href="/wiki/%EC%A7%84%EC%88%98_(%EC%84%9C%EC%A7%84)" title="진수 (서진)">진수</a>(陳壽)가 <a class="mw-redirect" href="/wiki/%EC%9C%84_(%EC%82%BC%EA%B5%AD_%EC%8B%9C%EB%8C%80)" title="위 (삼국 시대)">조위</a> 정통론<sup class="reference" id="cite_ref-1"><a href="#cite_note-1">[1]</a></sup>의 입장에서 전 황제인 선주(先主) 소열제(昭列帝) <a href="/wiki/%EC%9C%A0%EB%B9%84" title="유비">유비</a>(劉備)와 구별하여 유선을 후주(後主)라고 지칭했기에 후주로도 불린다. <a href="/wiki/%EC%9E%90_(%EC%9D%B4%EB%A6%84)" title="자 (이름)">자</a>는 공사(公嗣).</p>
<p><a href="/wiki/%EC%9C%A0%EB%B9%84" title="유비">유비</a>의 적차남이자 <a class="mw-redirect" href="/wiki/%EA%B0%90%EB%B6%80%EC%9D%B8" title="감부인">감부인</a>의 소생이며, 아명은 <b>아두</b>(阿斗)였다. <a href="/wiki/%EC%A1%B0%EC%9A%B4_(%EC%B4%89%ED%95%9C)" title="조운 (촉한)">조운</a>(趙雲)이 당양 장판파에서 구하였고 유비가 '너 때문에 용맹한 장수를 잃을 뻔하였다' 라고 한탄하며 땅바닥에 던져버렸다고 전해지는 아두가 곧 그다.</p>
<p>17세에 <a href="/wiki/%EC%9C%A0%EB%B9%84" title="유비">유비</a>의 뒤를 이어 황제가 되고, <a href="/wiki/%EC%9C%A0%EB%B9%84" title="유비">유비</a>의 탁고 유지를 받든 <a href="/wiki/%EC%8A%B9%EC%83%81" title="승상">승상</a> <a href="/wiki/%EC%A0%9C%EA%B0%88%EB%9F%89" title="제갈량">제갈량</a>(諸葛亮)에게 내정과 외정을 총괄케 하고, 신료들을 감독하게 하였다. <a href="/wiki/%EC%A0%9C%EA%B0%88%EB%9F%89" title="제갈량">제갈량</a> 사후, <a href="/wiki/%EC%9E%A5%EC%99%84" title="장완">장완</a>(蔣琬), <a href="/wiki/%EB%B9%84%EC%9D%98" title="비의">비의</a>(費禕), <a href="/wiki/%EA%B0%95%EC%9C%A0" title="강유">강유</a>(姜維) 등에게 국정을 맡기고 정치에 관여를 거의 하지 않으며<sup class="정리_필요 noprint">[<a href="/wiki/%EC%9C%84%ED%82%A4%EB%B0%B1%EA%B3%BC:%EC%B6%9C%EC%B2%98_%EB%B0%9D%ED%9E%88%EA%B8%B0" title="위키백과:출처 밝히기">출처 필요</a>]</sup>, 말년에 권신 <a href="/wiki/%EC%A7%84%EC%A7%80_(%EC%B4%89%ED%95%9C)" title="진지 (촉한)">진지</a>(陳祗)와 환관 <a href="/wiki/%ED%99%A9%ED%98%B8" title="황호">황호</a>(黃皓)를 총애하였다. 재위 40년 째인 <a href="/wiki/263%EB%85%84" title="263년">263년</a>에 <a href="/wiki/%EB%93%B1%EC%95%A0" title="등애">등애</a>(鄧艾)의 기습 공격으로 수도인 성도(成都)가 위태로워지자, 그해 겨울에 <a class="mw-redirect" href="/wiki/%EC%9C%84_(%EC%82%BC%EA%B5%AD_%EC%8B%9C%EB%8C%80)" title="위 (삼국 시대)">위</a>에 항복하였다. 이후 <a href="/wiki/%EC%A2%85%ED%9A%8C" title="종회">종회</a>(鍾會)와 <a href="/wiki/%EA%B0%95%EC%9C%A0" title="강유">강유</a>가 촉 회복 운동, 즉 위나라에 대한 반란을 꾀하다가 토벌된 뒤, <a class="mw-redirect" href="/wiki/%EB%82%99%EC%96%91" title="낙양">낙양</a>(洛陽)에 압송되어 안락공(安樂公)에 봉해졌다. 아명인 아두는 무능한 자를 가리키는 말로 흔히 쓰인다.<sup class="정리_필요 noprint">[<a href="/wiki/%EC%9C%84%ED%82%A4%EB%B0%B1%EA%B3%BC:%EC%B6%9C%EC%B2%98_%EB%B0%9D%ED%9E%88%EA%B8%B0" title="위키백과:출처 밝히기">출처 필요</a>]</sup> 촉한을 망하게한 자라고 불림.</p>

<h2><span class="mw-headline" id=".EC.B4.89.ED.95.9C.EC.9D.98_.EB.A9.B8.EB.A7.9D.EA.B3.BC_.EC.9C.A0.EC.84.A0">촉한의 멸망과 유선</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%9C%A0%EC%84%A0_(%EC%B4%89%ED%95%9C)&amp;action=edit&amp;section=1" title="부분 편집: 촉한의 멸망과 유선">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>《<a href="/wiki/%EC%82%BC%EA%B5%AD%EC%A7%80%EC%97%B0%EC%9D%98" title="삼국지연의">삼국지연의</a>》에서는 유선이 말년에 권신 <a href="/wiki/%EC%A7%84%EC%A7%80_(%EC%B4%89%ED%95%9C)" title="진지 (촉한)">진지</a>와 환관 <a href="/wiki/%ED%99%A9%ED%98%B8" title="황호">황호</a>를 총애하여 나라를 망쳤다고 전해 온다.</p>
<p>하지만 《<a href="/wiki/%EC%82%BC%EA%B5%AD%EC%A7%80" title="삼국지">삼국지</a>》<sup class="reference" id="cite_ref-.EC.82.BC.EC.86.8C.EC.A0.9C.EA.B8.B0_2-0"><a href="#cite_note-.EC.82.BC.EC.86.8C.EC.A0.9C.EA.B8.B0-2">[2]</a></sup>에 따르면, 촉한의 멸망 원인은 다른 데 있다. <a href="/wiki/263%EB%85%84" title="263년">263년</a> <a class="mw-redirect" href="/wiki/%EC%9C%84_(%EC%82%BC%EA%B5%AD_%EC%8B%9C%EB%8C%80)" title="위 (삼국 시대)">위</a>가 <a href="/wiki/%EC%B4%89%ED%95%9C" title="촉한">촉</a>을 정벌할 때 위나라 황제의 조서에서 이르기를, “촉은 작은 나라로, 영토가 좁고 백성 수가 적음에도 <a href="/wiki/%EA%B0%95%EC%9C%A0" title="강유">강유</a>는 군대를 혹사하여 잠시도 쉬게 하지 않았다. 지난 해에 <a href="/wiki/%EA%B0%95%EC%9C%A0" title="강유">강유</a>가 전쟁에서 진 뒤, <a class="new" href="/w/index.php?title=%EB%8B%B5%EC%A4%91&amp;action=edit&amp;redlink=1" title="답중 (없는 문서)">답중</a>에서 둔전을 하며 수많은 강인(羌人)을 핍박하고 끊임없이 일을 시켜 백성이 명령을 견디지 못했다.”고 하고 있다. 여기에서 유선이 환관에게 미혹되거나 실정했다는 말은 일절 없으며, <a class="mw-redirect" href="/wiki/%EC%9C%84_(%EC%82%BC%EA%B5%AD_%EC%8B%9C%EB%8C%80)" title="위 (삼국 시대)">위</a>가 <a href="/wiki/%EC%B4%89%ED%95%9C" title="촉한">촉한</a>을 치려 한 까닭이 황제 유선이 못나서가 아니라 <a href="/wiki/%EA%B0%95%EC%9C%A0" title="강유">강유</a>에게 있음을 지적하고 있다.</p>
<p>또 한 <a href="/wiki/250%EB%85%84%EB%8C%80" title="250년대">250년대</a> 말까지는 <a href="/wiki/%ED%99%A9%ED%98%B8" title="황호">황호</a>의 발호를 <a href="/wiki/%EB%8F%99%EC%9C%A4" title="동윤">동윤</a>(董允)을 비롯한 다른 신하들이 견제하고 있었으며, 유선 또한 <a href="/wiki/%ED%99%A9%ED%98%B8" title="황호">황호</a> 이상으로 이들을 신임하였다.</p>
<p>하지만, 유선의 재위 말년의 정치적 실정은 결코 무시할 수 없는 문제다. <a href="/wiki/%EA%B0%95%EC%9C%A0" title="강유">강유</a>의 북벌이 실패한 원인 가운데 하나가 유선이 <a href="/wiki/%ED%99%A9%ED%98%B8" title="황호">황호</a>의 말에 미혹된 점도 있기 때문이다. 또한 위와 촉의 국력 차이를 고려하면, 비록 잠시 동안일지라도 실정이 거듭될 경우 멸망할 수 있는 나라가 촉이었다.<sup class="정리_필요 noprint">[<a href="/wiki/%EC%9C%84%ED%82%A4%EB%B0%B1%EA%B3%BC:%EC%B6%9C%EC%B2%98_%EB%B0%9D%ED%9E%88%EA%B8%B0" title="위키백과:출처 밝히기">출처 필요</a>]</sup><a href="/wiki/262%EB%85%84" title="262년">262년</a> 10월, 북벌을 위해 출병했던 <a href="/wiki/%EA%B0%95%EC%9C%A0" title="강유">강유</a>는 패한 뒤 <a class="mw-redirect" href="/wiki/%EC%B2%AD%EB%91%90" title="청두">성도</a>나 위가 촉을 공격하기 위해 반드시 지나야 하는 <a class="mw-redirect" href="/wiki/%EA%B2%80%EA%B0%81" title="검각">검각</a>에 머물지 않고, 검각 북서쪽에 있는 답중으로 가서 주둔한다<sup class="reference" id="cite_ref-.EC.82.BC.EC.86.8C.EC.A0.9C.EA.B8.B0_2-1"><a href="#cite_note-.EC.82.BC.EC.86.8C.EC.A0.9C.EA.B8.B0-2">[2]</a></sup>. 그러자 위나라 황제는 “촉이 의지하는 이는 <a href="/wiki/%EA%B0%95%EC%9C%A0" title="강유">강유</a>뿐인데, 그가 본거지를 떠난 틈을 타서 공격하면 절반의 힘으로 두 배의 효과를 거둘 수 있다”<sup class="reference" id="cite_ref-.EC.82.BC.EC.86.8C.EC.A0.9C.EA.B8.B0_2-2"><a href="#cite_note-.EC.82.BC.EC.86.8C.EC.A0.9C.EA.B8.B0-2">[2]</a></sup>라고 말하며 <a href="/wiki/263%EB%85%84" title="263년">263년</a> 5월에 촉 정벌에 나선다. 이후 유선은 반년만인 <a href="/wiki/11%EC%9B%94" title="11월">11월</a>에 항복한다<sup class="reference" id="cite_ref-.EC.82.BC.EC.86.8C.EC.A0.9C.EA.B8.B0_2-3"><a href="#cite_note-.EC.82.BC.EC.86.8C.EC.A0.9C.EA.B8.B0-2">[2]</a></sup><sup class="reference" id="cite_ref-.ED.9B.84.EC.A3.BC.EC.A0.84_3-0"><a href="#cite_note-.ED.9B.84.EC.A3.BC.EC.A0.84-3">[3]</a></sup>.</p>
<p>결국 촉한의 멸망은 위나라와의 국력 차이, <a href="/wiki/%ED%99%A9%ED%98%B8" title="황호">황호</a>의 발호와 유선의 '사람에 의지하는 정치' 에서 비롯된 잘못된 인사기용, 촉한의 주력군의 사령관이던 <a href="/wiki/%EA%B0%95%EC%9C%A0" title="강유">강유</a>의 전략적 실패 등 복합적인 요인이 어우러져 발생한 결과일 가능성이 높다.<sup class="reference" id="cite_ref-.ED.9B.84.EC.A3.BC.EC.A0.84_3-1"><a href="#cite_note-.ED.9B.84.EC.A3.BC.EC.A0.84-3">[3]</a></sup></p>
<h2><span class="mw-headline" id=".EC.82.AC.EB.A7.9D.EA.B3.BC_.EA.B7.B8_.EC.9D.B4.ED.9B.84">사망과 그 이후</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%9C%A0%EC%84%A0_(%EC%B4%89%ED%95%9C)&amp;action=edit&amp;section=2" title="부분 편집: 사망과 그 이후">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>유선은 <a href="/wiki/271%EB%85%84" title="271년">271년</a> 65세의 나이로 사망하였으며,<sup class="reference" id="cite_ref-.ED.9B.84.EC.A3.BC.EC.A0.84_3-2"><a href="#cite_note-.ED.9B.84.EC.A3.BC.EC.A0.84-3">[3]</a></sup> 안락사공(安樂思公)에 봉해졌다. 유선이 지니고 있었던 안락공 지위는 위나라 시대에 이어 <a href="/wiki/%EC%84%9C%EC%A7%84" title="서진">서진</a> 왕조에 이르기까지 자손에게 승계되다가, 서진 말기 일어난 <a href="/wiki/%EC%98%81%EA%B0%80%EC%9D%98_%EB%82%9C" title="영가의 난">영가의 난</a> 때에 후손들이 모두 죽거나 흩어지면서 중단되었다. 서진 왕조가 몰락한 직후 시작된 <a class="mw-redirect" href="/wiki/%EC%98%A4%ED%98%B8%EC%8B%AD%EC%9C%A1%EA%B5%AD_%EC%8B%9C%EB%8C%80" title="오호십육국 시대">오호십육국 시대</a>에 한(漢;훗날 <a class="mw-disambig" href="/wiki/%EC%A0%84%EC%A1%B0" title="전조">전조</a>)을 세운 흉노 출신의 <a href="/wiki/%EC%9C%A0%EC%97%B0_(%EC%A0%84%EC%A1%B0)" title="유연 (전조)">유연</a>(劉淵)은 자신이 한나라의 계승자임을 자처하면서 유선에게 효회황제(孝懷皇帝)라는 시호를 올렸다.</p>
<h2><span class="mw-headline" id=".ED.8F.89.EA.B0.80.EC.99.80_.EB.85.BC.EB.9E.80">평가와 논란</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%9C%A0%EC%84%A0_(%EC%B4%89%ED%95%9C)&amp;action=edit&amp;section=3" title="부분 편집: 평가와 논란">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<blockquote class="toccolours" style="float:none; padding: 10px 15px 10px 15px; display:table;">
<p>"유선은 현명한 승상에게 정치를 맡겼을 때는 도리를 따르는 군주였지만, 환관에 미혹됐을 때는 어리석은 군주였다. 경전에서 말하기를 '흰색 실이 일정한 색깔이 없고 물감에 따라 물들여질 뿐이다.'라고 했는데 정말 그렇구나! 예(禮)에서는 국왕이 제위를 이은 뒤에 1년이 지나면 연호를 바꾼다고 했는데, 장무 3년에 건흥이라고 바꾸어 불렀다. 이는 고대의 의리로 살펴보면 나라의 근본 예법에 어긋난다. 또 촉나라는 사관을 두지 않아 기록하는 관원이 없었으므로 시행한 정사가 많이 유실되었으며 재앙이나 이변도 기록되지 않았다. 제갈량은 비록 정치에는 통달하였어도 이러한 일에는 주도면밀하지 못했다. 그렇지만 제갈량이 살아 있는 동안에는 20년이 지나도록 연호를 바꾸지 않았고, 군대도 여러 차례 출병했지만 쉽게 사면을 내리지 않은 것은 또한 탁월하지 않은가? 제갈량이 죽은 뒤부터 이런 제도는 점점 무너져 우열이 두드러지게 되었다."<sup class="reference" id="cite_ref-.ED.9B.84.EC.A3.BC.EC.A0.84_3-3"><a href="#cite_note-.ED.9B.84.EC.A3.BC.EC.A0.84-3">[3]</a></sup></p>
<p style="text-align: right;">— <cite><a href="/wiki/%EC%A7%84%EC%88%98_(%EC%84%9C%EC%A7%84)" title="진수 (서진)">진수</a></cite></p>
</blockquote>
<p>《<a href="/wiki/%EC%A7%84%EC%84%9C" title="진서">진서</a>(晉書)》에서는 “<a class="mw-disambig" href="/wiki/%EC%A0%9C%EB%82%98%EB%9D%BC" title="제나라">제나라</a> <a class="mw-redirect" href="/wiki/%EC%A0%9C%ED%99%98%EA%B3%B5" title="제환공">환공</a>에 다음가는 현명한 군주”라는 문장이 있으며, <a class="new" href="/w/index.php?title=%EC%99%95%EC%9D%80&amp;action=edit&amp;redlink=1" title="왕은 (없는 문서)">왕은</a>(王隱)도 《촉기(蜀記)》에서 “유선은 총명하고 사리에 밝은 왕이다.”라고 하였다. 중국 역사에서 제 환공은 밑에 훌륭한 신하가 있으면 명군, 없으면 암군이라는 의미로서 쓰이고 있다.</p>
<p>《<a href="/wiki/%EC%82%BC%EA%B5%AD%EC%A7%80" title="삼국지">삼국지</a>》 &lt;위서&gt; 육개전에 의하면, 육개는 촉한의 멸망 원인에 대해 '군주는 사치스럽게 하고 백성들의 힘을 긴급하지 않은 곳에서 고갈시켰다. 그 때문에 진에 토벌되었다.' 라고 분석하고 있다. 또한 《<a href="/wiki/%EC%9E%90%EC%B9%98%ED%86%B5%EA%B0%90" title="자치통감">자치통감</a>》 에서도 <a href="/wiki/%EC%82%AC%EB%A7%88%EC%82%AC" title="사마사">사마사</a>(司馬師)는 '촉한이 의지하는 사람은 <a href="/wiki/%EA%B0%95%EC%9C%A0" title="강유">강유</a>뿐이니 <a href="/wiki/%EA%B0%95%EC%9C%A0" title="강유">강유</a>를 묶어둔다면 유선의 아둔함으로는 아무것도 할 수 없을 것이다'라고 평가하고 촉정을 계획했다.</p>
<p>유선은 연의에서 흔히 묘사되는 아둔한 군주만은 아니었다. 만약 유선이 아둔하고 암우한 군주에 불과했다면 41년이라는 긴 시간 동안 황위를 지킬 수는 없었을 것이기 때문이다. 실제로 유선이 촉의 황제로 재위하던 동안 위는 <a href="/wiki/%EC%A1%B0%EB%B9%84" title="조비">조비</a>, <a href="/wiki/%EC%A1%B0%EC%98%88" title="조예">조예</a>, <a href="/wiki/%EC%A1%B0%EB%B0%A9" title="조방">조방</a>, <a href="/wiki/%EC%A1%B0%EB%AA%A8" title="조모">조모</a>, <a href="/wiki/%EC%A1%B0%ED%99%98" title="조환">조환</a> 등으로, 오는 <a href="/wiki/%EC%86%90%EA%B6%8C" title="손권">손권</a>, <a href="/wiki/%EC%86%90%EB%9F%89" title="손량">손량</a>, <a href="/wiki/%EC%86%90%ED%9C%B4" title="손휴">손휴</a>,<a href="/wiki/%EC%86%90%ED%98%B8" title="손호">손호</a> 등으로 황제가 바뀌었다.</p>
<p><a href="/wiki/%EC%A0%9C%EA%B0%88%EB%9F%89" title="제갈량">제갈량</a>(諸葛亮)이 남만 정벌을 다녀올 무렵에는 스스로 정책을 만들어 시행하기도 하였으며 <a href="/wiki/%EC%82%BC%EA%B5%AD%EC%A7%80" title="삼국지">정사 삼국지</a>(正史 三國志) 촉서(蜀書) 장완전(張琬傳)에 의하면 나름대로 군사적인 안목도 가지고 있어서 <a href="/wiki/%EC%9C%84_(%EC%82%BC%EA%B5%AD)" title="위 (삼국)">위나라</a>의 권신(權臣)<sup class="reference" id="cite_ref-4"><a href="#cite_note-4">[4]</a></sup> <a href="/wiki/%EC%82%AC%EB%A7%88%EC%9D%98" title="사마의">사마중달</a>(司馬仲達)이 연왕(燕王) <a href="/wiki/%EA%B3%B5%EC%86%90%EC%97%B0" title="공손연">공손연</a>(公孫淵)을 토벌하러 요동(遼東)으로 떠났을 때 대장군 <a href="/wiki/%EC%9E%A5%EC%99%84" title="장완">장완</a>(蔣琬)에게 다음과 같은 조서(詔書)를 내린다.</p>

<p>하지만 유선이 41년동안 황위를 지킬 수 있었던 것은, 그가 유능한 황제였기 때문이 아니었다. 유선은 일국의 천자로서 반드시 갖추어야 할 정치적 재능이 아예 없지는 않았지만 다소 부족한 인물이었으며, <a href="/wiki/%EC%A0%9C%EA%B0%88%EB%9F%89" title="제갈량">제갈량</a> 사후에도 <a href="/wiki/%EB%8F%99%EC%9C%A4" title="동윤">동윤</a>과 <a href="/wiki/%EC%9E%A5%EC%99%84" title="장완">장완</a>, <a href="/wiki/%EB%B9%84%EC%9D%98" title="비의">비의</a>등이 유선을 보좌하며 정치를 이끌었다. 동윤의 사후 환관 황호가 권신 진지와 결탁하여 정권을 잡아 촉한은 멸망의 길을 걷기 시작하며, 강유가 황호를 처형하자고 상주했을 때에도 유선은 이미 황호가 소인배임을 잘 알고 있었으나 계속 그를 총애하였다고 전한다.</p>
