<h2><span class="mw-headline" id="In_mathematics">In mathematics</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=166_(number)&amp;action=edit&amp;section=1" title="Edit section: In mathematics">edit</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>166 is an even number and a <a href="/wiki/Composite_number" title="Composite number">composite number</a>. It is a <a href="/wiki/Centered_triangular_number" title="Centered triangular number">centered triangular number</a>.<sup class="reference" id="cite_ref-1"><a href="#cite_note-1">[1]</a></sup></p>
<p>Given 166, the <a href="/wiki/Mertens_function" title="Mertens function">Mertens function</a> returns 0. 166 is a <a href="/wiki/Smith_number" title="Smith number">Smith number</a> in <a class="mw-redirect" href="/wiki/Base_10" title="Base 10">base 10</a>.<sup class="reference" id="cite_ref-2"><a href="#cite_note-2">[2]</a></sup></p>

<li><a href="/wiki/166P/NEAT" title="166P/NEAT">166P/NEAT</a> is a periodic <a href="/wiki/Comet" title="Comet">comet</a> and <a class="mw-redirect" href="/wiki/Centaur_(planetoid)" title="Centaur (planetoid)">centaur</a> in the outer <a class="mw-redirect" href="/wiki/Solar_system" title="Solar system">solar system</a></li>
<li><a href="/wiki/HD_166" title="HD 166">HD 166</a> is the 6th magnitude <a href="/wiki/Star" title="Star">star</a> in the <a href="/wiki/Constellation" title="Constellation">constellation</a> <a href="/wiki/Andromeda_(constellation)" title="Andromeda (constellation)">Andromeda</a></li>
