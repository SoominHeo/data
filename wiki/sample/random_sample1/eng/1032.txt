<p>The <b>GP32</b> (GamePark 32) is a <a href="/wiki/Handheld_game_console" title="Handheld game console">handheld game console</a> developed by the <a href="/wiki/Korea" title="Korea">Korean</a> company <a href="/wiki/Game_Park" title="Game Park">Game Park</a>. It was released on November 23, 2001, in <a href="/wiki/South_Korea" title="South Korea">South Korea</a> only.</p>

<h2><span class="mw-headline" id="Features">Features</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=GP32&amp;action=edit&amp;section=1" title="Edit section: Features">edit</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>The overall design is not unlike the original version of the <a href="/wiki/Game_Boy_Advance" title="Game Boy Advance">Game Boy Advance</a>. The GP32 is based on a 133 MHz <a class="mw-redirect" href="/wiki/Advanced_RISC_Machines" title="Advanced RISC Machines">ARM</a> <a href="/wiki/Central_processing_unit" title="Central processing unit">CPU</a> and 8 MB of <a class="mw-redirect" href="/wiki/Random_Access_Memory" title="Random Access Memory">RAM</a>. Unlike other hand held gaming systems, which tend to be proprietary cartridge-based, the GP32 uses <a href="/wiki/SmartMedia" title="SmartMedia">SmartMedia</a> cards (SMC) for storing programs and data, making it accessible for amateur developers as no further development hardware is required.</p>
<p>The console has a four-way mini-joystick controller, two main buttons ('A' and 'B'), two shoulder buttons on each side of the SMC slot ('L' and 'R') and two other menu buttons on each side of the screen ('SELECT' and 'START'), made from a softer, translucent rubber. The console also has a <a class="mw-redirect" href="/wiki/Universal_Serial_Bus" title="Universal Serial Bus">USB 1.1</a> port for connection with a host computer, a serial expansion port, a 3.3 V power adapter input, a headphone connector and a rear compartment which holds two AA sized batteries.</p>
<h2><span class="mw-headline" id="Variants">Variants</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=GP32&amp;action=edit&amp;section=2" title="Edit section: Variants">edit</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>There are three main commercial versions of the unit characterized by different display types. Commercial units are white in color with either grey or white buttons and trim. There are also a number of differently colored promotional units, and several prototype units with different designs.</p>
<h3><span class="mw-headline" id="GP32_NLU">GP32 NLU</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=GP32&amp;action=edit&amp;section=3" title="Edit section: GP32 NLU">edit</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>The original GP32 was the no-light unit (NLU) which relied on an external light source to view the screen.</p>
<h3><span class="mw-headline" id="GP32_FLU">GP32 FLU</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=GP32&amp;action=edit&amp;section=4" title="Edit section: GP32 FLU">edit</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>In late 2002, Game Park introduced the <a href="/wiki/Frontlight" title="Frontlight">frontlight</a> unit (FLU) as a factory modified (by <a class="new" href="/w/index.php?title=Hahotech&amp;action=edit&amp;redlink=1" title="Hahotech (page does not exist)">Hahotech</a>) version of the NLU. It provides its own illumination via a transparent panel between the LCD display and the plastic screen cover. The extra hardware resulted in a slightly raised display frame when compared to the NLU and BLU variants. The frontlight could be turned off with a switch mounted on the back of the GP32 case (to save battery). The GP32 FLU's name was derived from a sticker added to the front of the GP32 packaging differentiating it from the standard non-light versions.</p>
<h3><span class="mw-headline" id="GP32_BLU">GP32 BLU</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=GP32&amp;action=edit&amp;section=5" title="Edit section: GP32 BLU">edit</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>In mid-2004, <a href="/wiki/Game_Park" title="Game Park">Game Park</a> introduced the <a href="/wiki/Backlight" title="Backlight">backlight</a> unit (BLU). The new BLU LCD was compatible with the NLU (and FLU) and provided superior screen display in poor lighting conditions. At the end of 2004, Game Park also released a second version of the BLU, which had a different LCD to the first version BLU units. The new BLU+ LCD was not 100% compatible with the original LCD screen and so software required special handling to support both LCD versions. The backlight could be disabled by holding the SELECT button for 5 seconds. The BLU models also had a slightly different <a href="/wiki/USB" title="USB">USB</a> port connector and better quality micro switches for the controller.</p>

<p>A back lit GP32 (released in December 2004) was given the name BLU+ by the community.<sup class="noprint Inline-Template Template-Fact" style="white-space:nowrap;">[<i><a href="/wiki/Wikipedia:Citation_needed" title="Wikipedia:Citation needed"><span title="This claim needs references to reliable sources. (April 2008)">citation needed</span></a></i>]</sup> The back lit unit featured a different LCD display (<a href="/wiki/Taiwan" title="Taiwan">Taiwanese</a> manufactured instead of Samsung), which led to some compatibility problems with certain applications, and problems such as white lines across the screen. However, nearly all applications were fixed once it was found that the new screens had better contrast than the old ones. The back lit GP32's were commercially named "BLU". The BLU+ was, as stated, a name that the community gave it to notice the difference between the two models.</p>
<p>There were many applications that worked with the BLU+ and other models. Mirko's SDK could autodetect which version one had. All the applications such as emulators, movie players, and the like worked as well on the BLU+, as on other models.</p>
<h2><span class="mw-headline" id="Software">Software</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=GP32&amp;action=edit&amp;section=7" title="Edit section: Software">edit</a><span class="mw-editsection-bracket">]</span></span></h2>
<h3><span class="mw-headline" id="Built_In">Built In</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=GP32&amp;action=edit&amp;section=8" title="Edit section: Built In">edit</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>The original <a href="/wiki/Game_Park" title="Game Park">Game Park</a> <a href="/wiki/Firmware" title="Firmware">firmware</a> had three main functions: to launch applications, provide a means of linking to a host computer and to play music in <a href="/wiki/MP3" title="MP3">MP3</a> format. Later versions of the <a href="/wiki/Game_Park" title="Game Park">Game Park</a> <a href="/wiki/Firmware" title="Firmware">firmware</a> removed the <a href="/wiki/MP3" title="MP3">MP3</a> music capability.</p>
<h3><span class="mw-headline" id="Homebrew">Homebrew</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=GP32&amp;action=edit&amp;section=9" title="Edit section: Homebrew">edit</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>Game Park planned their system to be powerful and useful, but they also wanted users to be able to create <a href="/wiki/Homebrew_(video_games)" title="Homebrew (video games)">homebrew</a> software. GP32 users could register the unit on the official website and get a free suite of development tools to create their own programs. Game Park also allowed (under certain restrictions) the publishing of such homebrew games on their website. The GP32's original firmware only supported running encrypted games and tools. Users had to register and use an encrypted "Free Launcher" to run unsigned software. Alternative <a href="/wiki/Firmware" title="Firmware">firmware</a> removed the necessity of using the "Free Launcher" software and provided many extra functions that were lacking in the original firmware.</p>
<p>Through this strategy, the GP32 was the host of multiple homebrew applications and games. The various applications made for it ranged from alternative <a href="/wiki/Firmware" title="Firmware">firmware</a>, file managers, games, emulators, game generators (such as <a href="/wiki/RPG_Maker" title="RPG Maker">RPG Maker</a>), a <a href="/wiki/DivX" title="DivX">DivX</a> player and image slideshows.</p>
<h3><span class="mw-headline" id="Commercial">Commercial</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=GP32&amp;action=edit&amp;section=10" title="Edit section: Commercial">edit</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>Commercial games could be acquired via internet download (encrypted to the GP32's ID) or in a retail box. The retail boxes contained <a href="/wiki/SmartMedia" title="SmartMedia">SmartMedia</a> Cards with the games which were encrypted to run only from these cards (SMC ID). They could alternatively be downloaded from a Korean portal after submitting the GP32's ID, they were then encrypted to run only on that GP32. Downloading the games effectively reduced the cost to the consumer, who was no longer paying for the manufacture of the cartridge. Downloading a game online could range from US$10 to US$30.</p>
<h3><span class="mw-headline" id="Games">Games</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=GP32&amp;action=edit&amp;section=11" title="Edit section: Games">edit</a><span class="mw-editsection-bracket">]</span></span></h3>
<div class="hatnote" role="note">See also: <a href="/wiki/List_of_commercial_GP32_games" title="List of commercial GP32 games">List of commercial GP32 games</a></div>
<p>Five games in a variety of genres were released at the system's launch on November 23, 2001. About 28 commercial games were eventually released. The last commercial game to be released was the platformer/RPG <a class="new" href="/w/index.php?title=Blue_Angelo&amp;action=edit&amp;redlink=1" title="Blue Angelo (page does not exist)">Blue Angelo</a>, which was released on December 16, 2004. Most commercial GP32 games could be bought in two ways: boxed or downloaded through the internet through Gamepark's online JoyGP store (typically for a much lower price). JoyGP was the international version of the MegaGP store, which existed earlier and was limited to South Korea. Although most games were sold in both formats, there were a few exceptions: for example, "Blue Angelo" was (and is) only sold as a boxed copy made in France, and "Gloop Deluxe" was only sold online, but not through JoyGP.</p>
<p>Although the number of official games available for the GP32 system is limited, many <a class="mw-redirect" href="/wiki/Open_source" title="Open source">open source</a>/<a href="/wiki/Free_software" title="Free software">free software</a> developers worked on various emulators and <a href="/wiki/Porting" title="Porting">ported</a> PC games. In addition to this, a wide range of free, <a href="/wiki/Public_domain" title="Public domain">public domain</a> games were created by amateur developers. Game Park did not ask any royalties to release games for its device, which made it easier for small editors or independent developers to release software for the GP32.</p>

<p>The modification of <a href="/wiki/Ericsson" title="Ericsson">Ericsson</a> Chatboard micro-keyboards to work with the system has seen a new flourish of software development, including countless attempts at ports of <a href="/wiki/Linux" title="Linux">Linux</a>, and keyboard support being added to many emulators.</p>
<h3><span class="mw-headline" id="Emulators">Emulators</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=GP32&amp;action=edit&amp;section=12" title="Edit section: Emulators">edit</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>The GP32's relatively powerful ARM 920T CPU and freely available 'C'-based SDK have allowed many emulators to be specifically developed or ported from other platforms. Emulated systems that run on the GP32 include 16-bit era and earlier console and computer platforms. These emulators allow users to experience a large variety of games on their GP32 system, largely compensating for the relatively small library of commercially available games. There is also a <a href="/wiki/Microsoft_Windows" title="Microsoft Windows">Windows</a>-based GP32 emulator, which allows users to run GP32 software on a Windows-based PC.</p>
<h2><span class="mw-headline" id="Commercial_availability">Commercial availability</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=GP32&amp;action=edit&amp;section=13" title="Edit section: Commercial availability">edit</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>Commercially, the system can be found mostly in Korea, and some other parts of Asia, although the GP32 BLU model was released in three European markets, including <a href="/wiki/Portugal" title="Portugal">Portugal</a>, <a href="/wiki/Spain" title="Spain">Spain</a>, and <a href="/wiki/Italy" title="Italy">Italy</a> on June 15, 2004 with a price point of €199. There are official distributors in the <a href="/wiki/United_Kingdom" title="United Kingdom">United Kingdom</a> and <a href="/wiki/Sweden" title="Sweden">Sweden</a> as well. Game Park, however, did not release the console in <a href="/wiki/Americas" title="Americas">America</a>. Since the GP32 is no longer in production, the console can commonly be found on eBay, forums, or other used video game vendors or websites.</p>
<p>Despite the GP32 not being released worldwide, it has a large international community of users and developers. About 30,000 units were sold by the end of 2007.<sup class="reference" id="cite_ref-clockerz_1-0"><a href="#cite_note-clockerz-1">[1]</a></sup></p>
