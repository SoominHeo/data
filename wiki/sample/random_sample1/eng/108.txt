<h1 class="firstHeading" id="firstHeading" lang="en">DirectDraw Surface</h1>
<div class="mw-body-content" id="bodyContent">
<div id="siteSub">From Wikipedia, the free encyclopedia</div>
<div id="contentSub"></div>
<div class="mw-jump" id="jump-to-nav">
					Jump to:					<a href="#mw-head">navigation</a>, 					<a href="#p-search">search</a>
</div>
<div class="mw-content-ltr" dir="ltr" id="mw-content-text" lang="en"><script>function mfTempOpenSection(id){var block=document.getElementById("mf-section-"+id);block.className+=" open-block";block.previousSibling.className+=" open-block";}</script><p>The <b>DirectDraw Surface</b> container file format (uses the <a href="/wiki/Filename_extension" title="Filename extension">filename extension</a> <i><b>DDS</b></i>), is a <a href="/wiki/Microsoft" title="Microsoft">Microsoft</a> format for storing data <a href="/wiki/Data_compression" title="Data compression">compressed</a> with the proprietary <a href="/wiki/S3_Texture_Compression" title="S3 Texture Compression">S3 Texture Compression</a> (S3TC) algorithm,<sup class="reference" id="cite_ref-1"><a href="#cite_note-1">[1]</a></sup> which can be decompressed in hardware by <a class="mw-redirect" href="/wiki/GPU" title="GPU">GPUs</a>. This makes the format useful for storing graphical <a class="mw-redirect" href="/wiki/Texture_(computer_graphics)" title="Texture (computer graphics)">textures</a> and <a href="/wiki/Cube_mapping" title="Cube mapping">cubic environment maps</a> as a <a href="/wiki/Data_file" title="Data file">data file</a>, both compressed and uncompressed.<sup class="reference" id="cite_ref-2"><a href="#cite_note-2">[2]</a></sup> The <a href="/wiki/Microsoft_Windows" title="Microsoft Windows">Microsoft Windows</a> <a class="mw-redirect" href="/wiki/File_extension" title="File extension">file extension</a> for this data format is '.dds'.<sup class="reference" id="cite_ref-3"><a href="#cite_note-3">[3]</a></sup></p>
<p></p>

<h2><span class="mw-headline" id="History">History</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=DirectDraw_Surface&amp;action=edit&amp;section=1" title="Edit section: History">edit</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>This format was introduced with DirectX 7.0. In DirectX 8.0, support for volume textures was added. With <a class="mw-redirect" href="/wiki/Direct3D_10" title="Direct3D 10">Direct3D 10</a>, the file format was extended to allow an <a href="/wiki/Array_data_structure" title="Array data structure">array</a> of textures to be included, as well as support for new Direct3D 10.x and 11 texture formats.<sup class="reference" id="cite_ref-4"><a href="#cite_note-4">[4]</a></sup></p>
