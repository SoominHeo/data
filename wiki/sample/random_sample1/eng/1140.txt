<p><b>IntelliJ IDEA</b> (pronounced <span class="IPA" title="Representation in the International Phonetic Alphabet (IPA)">/ɪnˈtɛlɪdʒeɪ aɪˈdɪə/</span>) is a <a href="/wiki/Java_(programming_language)" title="Java (programming language)">Java</a> <a href="/wiki/Integrated_development_environment" title="Integrated development environment">integrated development environment</a> (IDE) for developing computer software. It is developed by JetBrains (formerly known as <a class="mw-redirect" href="/wiki/IntelliJ" title="IntelliJ">IntelliJ</a>), and is available as an <a class="mw-redirect" href="/wiki/Apache_2_License" title="Apache 2 License">Apache 2 Licensed</a> community edition,<sup class="reference" id="cite_ref-2"><a href="#cite_note-2">[2]</a></sup> and in a <a href="/wiki/Proprietary_software" title="Proprietary software">proprietary</a> commercial edition. Both can be used for commercial development.<sup class="reference" id="cite_ref-3"><a href="#cite_note-3">[3]</a></sup></p>

<h2><span class="mw-headline" id="History">History</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=IntelliJ_IDEA&amp;action=edit&amp;section=1" title="Edit section: History">edit</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>The first version of IntelliJ IDEA was released in January 2001, and was one of the first available Java IDEs with advanced code navigation and <a href="/wiki/Code_refactoring" title="Code refactoring">code refactoring</a> capabilities integrated.<sup class="reference" id="cite_ref-4"><a href="#cite_note-4">[4]</a></sup><sup class="reference" id="cite_ref-5"><a href="#cite_note-5">[5]</a></sup></p>
<p>In a 2010 <i><a class="mw-redirect" href="/wiki/Infoworld" title="Infoworld">Infoworld</a></i> report, IntelliJ received the highest test center score out of the four top Java programming tools: <a href="/wiki/Eclipse_(software)" title="Eclipse (software)">Eclipse</a>, IntelliJ IDEA, <a href="/wiki/NetBeans" title="NetBeans">NetBeans</a> and <a href="/wiki/JDeveloper" title="JDeveloper">JDeveloper</a>.<sup class="reference" id="cite_ref-6"><a href="#cite_note-6">[6]</a></sup></p>
<p>In December 2014, Google announced version 1.0 of <a href="/wiki/Android_Studio" title="Android Studio">Android Studio</a>, an <a class="mw-redirect" href="/wiki/Open_source" title="Open source">open source</a> IDE for <a href="/wiki/Android_(operating_system)" title="Android (operating system)">Android</a> apps, based on the open source community edition of IntelliJ IDEA.<sup class="reference" id="cite_ref-7"><a href="#cite_note-7">[7]</a></sup> Other development environments based on IntelliJ's framework include <a href="/wiki/AppCode" title="AppCode">AppCode</a>, <a href="/wiki/JetBrains#CLion" title="JetBrains">CLion</a>, <a href="/wiki/PhpStorm" title="PhpStorm">PhpStorm</a>, <a href="/wiki/PyCharm" title="PyCharm">PyCharm</a>, <a class="mw-redirect" href="/wiki/RubyMine" title="RubyMine">RubyMine</a>, <a class="mw-redirect" href="/wiki/WebStorm" title="WebStorm">WebStorm</a>, and <a href="/wiki/JetBrains_MPS" title="JetBrains MPS">MPS</a>.<sup class="reference" id="cite_ref-8"><a href="#cite_note-8">[8]</a></sup></p>

<h2><span class="mw-headline" id="Features">Features</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=IntelliJ_IDEA&amp;action=edit&amp;section=3" title="Edit section: Features">edit</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>Version 12.1 includes support for <a class="mw-redirect" href="/wiki/Java_8" title="Java 8">Java 8</a>, UI designer for <a href="/wiki/Android_(operating_system)" title="Android (operating system)">Android</a> development, <a href="/wiki/Play_Framework" title="Play Framework">Play 2.0</a> and <a href="/wiki/Scala_(programming_language)" title="Scala (programming language)">Scala</a>.</p>
<h3><span class="mw-headline" id="Coding_assistance">Coding assistance</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=IntelliJ_IDEA&amp;action=edit&amp;section=4" title="Edit section: Coding assistance">edit</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>The IDE provides certain features<sup class="reference" id="cite_ref-:0_11-0"><a href="#cite_note-:0-11">[11]</a></sup> like code completion by analyzing the context, code navigation where one can jump to a class or declaration in the code directly, <a href="/wiki/Code_refactoring" title="Code refactoring">code refactoring</a> and providing options to fix inconsistencies via suggestions.</p>
<h3><span class="mw-headline" id="Built_in_tools_and_integration">Built in tools and integration</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=IntelliJ_IDEA&amp;action=edit&amp;section=5" title="Edit section: Built in tools and integration">edit</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>The IDE provides<sup class="reference" id="cite_ref-:0_11-1"><a href="#cite_note-:0-11">[11]</a></sup> for integration with build/packaging tools like <a href="/wiki/Grunt_(software)" title="Grunt (software)">grunt</a>, <a class="new" href="/w/index.php?title=Bower_(software)&amp;action=edit&amp;redlink=1" title="Bower (software) (page does not exist)">bower</a>, <a href="/wiki/Gradle" title="Gradle">gradle</a>, and <a class="mw-redirect" href="/wiki/SBT_(software)" title="SBT (software)">SBT</a>. It supports <a href="/wiki/Version_control" title="Version control">version control</a> systems like <a class="mw-redirect" href="/wiki/Git_(software)" title="Git (software)">GIT</a>, <a href="/wiki/Mercurial" title="Mercurial">Mercurial</a>, <a class="mw-redirect" href="/wiki/Perforce" title="Perforce">Perforce</a>, and <a href="/wiki/Apache_Subversion" title="Apache Subversion">SVN</a>. Databases like <a href="/wiki/Microsoft_SQL_Server" title="Microsoft SQL Server">Microsoft SQL Server</a>, <a href="/wiki/Oracle_Database" title="Oracle Database">ORACLE</a>, <a href="/wiki/PostgreSQL" title="PostgreSQL">PostgreSQL</a>, and <a href="/wiki/MySQL" title="MySQL">MySQL</a> can be accessed directly from the IDE.</p>
<h3><span class="mw-headline" id="Plugin_ecosystem">Plugin ecosystem</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=IntelliJ_IDEA&amp;action=edit&amp;section=6" title="Edit section: Plugin ecosystem">edit</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>IntelliJ supports plugins through which one can add additional functionality to the IDE. One can download and install plugins either from IntelliJ's plugin repository website or through IDE's inbuilt plugin search and install feature. Currently IntelliJ IDEA Community edition has 1495<sup class="reference" id="cite_ref-12"><a href="#cite_note-12">[12]</a></sup> plugins available, where as the Ultimate edition has 1626<sup class="reference" id="cite_ref-13"><a href="#cite_note-13">[13]</a></sup> plugins available.</p>

<h3><span class="mw-headline" id="Technologies_and_frameworks">Technologies and frameworks</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=IntelliJ_IDEA&amp;action=edit&amp;section=8" title="Edit section: Technologies and frameworks">edit</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>Community Edition supports the following:<sup class="reference" id="cite_ref-editions_14-1"><a href="#cite_note-editions-14">[14]</a></sup></p>
<ul>

<li><a href="/wiki/Web_service" title="Web service">Web services</a></li>

<p>Ultimate Edition also supports the <a href="/wiki/Application_server" title="Application server">application servers</a> <a href="/wiki/Apache_Geronimo" title="Apache Geronimo">Geronimo</a>, <a href="/wiki/GlassFish" title="GlassFish">GlassFish</a>, <a class="mw-redirect" href="/wiki/JBoss_application_server" title="JBoss application server">JBoss</a>, <a href="/wiki/Jetty_(web_server)" title="Jetty (web server)">Jetty</a>, <a href="/wiki/Apache_Tomcat" title="Apache Tomcat">Tomcat</a>, <a class="mw-redirect" href="/wiki/Oracle_Weblogic_Server" title="Oracle Weblogic Server">Weblogic</a>, and <a href="/wiki/IBM_WebSphere" title="IBM WebSphere">WebSphere</a>.<sup class="reference" id="cite_ref-editions_14-3"><a href="#cite_note-editions-14">[14]</a></sup></p>
<p>There was a free <a href="/wiki/Plug-in_(computing)" title="Plug-in (computing)">plugin</a> from <a href="/wiki/Atlassian" title="Atlassian">Atlassian</a> for IntelliJ available to integrate with <a href="/wiki/Jira_(software)" title="Jira (software)">JIRA</a>,<sup class="reference" id="cite_ref-20"><a href="#cite_note-20">[20]</a></sup> <a href="/wiki/Bamboo_(software)" title="Bamboo (software)">Bamboo</a>, <a href="/wiki/Crucible_(software)" title="Crucible (software)">Crucible</a> and <a href="/wiki/FishEye_(software)" title="FishEye (software)">FishEye</a>. However, the software called IDE-Connector was discontinued on June 1st 2015. <sup class="reference" id="cite_ref-21"><a href="#cite_note-21">[21]</a></sup></p>
