<p><b>‼</b> (a double <a href="/wiki/Exclamation_mark" title="Exclamation mark">exclamation mark</a>, <a class="mw-redirect" href="/wiki/General_Punctuation_(Unicode_block)" title="General Punctuation (Unicode block)">Unicode character</a> U+203C) may refer to:</p>
<ul>
<li>A <a href="/wiki/Chess_annotation_symbols#.E2.80.BC_.28Brilliant_move.29" title="Chess annotation symbols">brilliant move</a> in chess notation</li>
<li>The <a href="/wiki/Double_factorial" title="Double factorial">double factorial</a> operator in mathematics</li>
<li>A "convert to <a href="/wiki/Boolean_data_type" title="Boolean data type">Boolean</a>" construction in some computer languages, this is through a computer type of <a href="/wiki/Double_negation" title="Double negation">double negation</a>, where its Boolean value is negated, then negated again, converting the value to either true or false</li>
<li>An operator for fetching list elements in the programming language <a href="/wiki/Haskell_(programming_language)" title="Haskell (programming language)">Haskell</a></li>
<li>A <a href="/wiki/Retroflex_clicks" title="Retroflex clicks">retroflex click</a>, a family of click consonants found only in Juu languages of southern Africa and in the <a href="/wiki/Damin" title="Damin">Damin</a> ritual jargon of Australia</li>
</ul>
<p>The ‼ character can be typed on <a href="/wiki/Microsoft_Windows" title="Microsoft Windows">Microsoft Windows</a> operating systems by using the key combination <b>ALT+19</b>, or in simple form, holding the <a href="/wiki/Alt_key" title="Alt key">ALT key</a> and typing the numbers <a class="mw-redirect" href="/wiki/1_(number)" title="1 (number)">1</a> and <a href="/wiki/9_(number)" title="9 (number)">9</a> on the <a href="/wiki/Numeric_keypad" title="Numeric keypad">number pad</a>.</p>
