<h1 class="firstHeading" id="firstHeading" lang="en">National Semiconductor PACE</h1>

<p><a href="/wiki/National_Semiconductor" title="National Semiconductor">National Semiconductor</a>'s IPC-16A/520 <b>PACE</b>, short for "Processing and Control Element", was the first commercial<sup class="reference" id="cite_ref-1"><a href="#cite_note-1">[1]</a></sup> single-chip <a href="/wiki/16-bit" title="16-bit">16-bit</a> <a href="/wiki/Microprocessor" title="Microprocessor">microprocessor</a>. PACE had four general-purpose <a href="/wiki/Accumulator_(computing)" title="Accumulator (computing)">accumulators</a>, with an <a class="mw-redirect" href="/wiki/Instruction_set_architecture" title="Instruction set architecture">instruction set architecture</a> loosely based on the earlier <a href="/wiki/IMP-16" title="IMP-16">IMP-16</a> architecture, which in turn had been inspired by the <a href="/wiki/Data_General_Nova" title="Data General Nova">Data General Nova</a> minicomputer.</p>
<p>PACE was slightly faster than the IMP-16, and offered a "byte mode" for more convenient processing of <a href="/wiki/8-bit" title="8-bit">8-bit</a> data. Some PACE instructions were restricted to operation on only the first accumulator, AC0, rather than allowing use of any accumulator as on the IMP-16.</p>
<p>The PACE was followed by the INS8900, which had the same architecture but was implemented in <a href="/wiki/NMOS_logic" title="NMOS logic">NMOS</a> for easier electrical interfacing.</p>
