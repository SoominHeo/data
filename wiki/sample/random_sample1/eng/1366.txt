<h1 class="firstHeading" id="firstHeading" lang="en">PH (complexity)</h1>
<div class="mw-body-content" id="bodyContent">
<div id="siteSub">From Wikipedia, the free encyclopedia</div>
<div id="contentSub"></div>

<div class="mw-content-ltr" dir="ltr" id="mw-content-text" lang="en"><script>function mfTempOpenSection(id){var block=document.getElementById("mf-section-"+id);block.className+=" open-block";block.previousSibling.className+=" open-block";}</script><p>In <a href="/wiki/Computational_complexity_theory" title="Computational complexity theory">computational complexity theory</a>, the <a href="/wiki/Complexity_class" title="Complexity class">complexity class</a> <b>PH</b> is the union of all complexity classes in the <a href="/wiki/Polynomial_hierarchy" title="Polynomial hierarchy">polynomial hierarchy</a>:</p>

<p><b>PH</b> was first defined by <a href="/wiki/Larry_Stockmeyer" title="Larry Stockmeyer">Larry Stockmeyer</a>.<sup class="reference" id="cite_ref-1"><a href="#cite_note-1">[1]</a></sup> It is a special case of hierarchy of <a href="/wiki/Alternating_Turing_machine#Bounded_alternation" title="Alternating Turing machine">bounded alternating Turing machine</a>. It is contained in <b>P<sup>#P</sup></b> = <b>P<sup>PP</sup></b> (by <a href="/wiki/Toda%27s_theorem" title="Toda's theorem">Toda's theorem</a>; the class of problems that are decidable by a polynomial time <a href="/wiki/Turing_machine" title="Turing machine">Turing machine</a> with access to a <a class="mw-redirect" href="/wiki/Sharp_P" title="Sharp P">#P</a> or equivalently <a class="mw-redirect" href="/wiki/PP_(complexity_class)" title="PP (complexity class)">PP</a> <a href="/wiki/Oracle_machine" title="Oracle machine">oracle</a>), and also in <b><a href="/wiki/PSPACE" title="PSPACE">PSPACE</a></b>.</p>
<p><b>PH</b> has a simple <a class="mw-redirect" href="/wiki/Descriptive_complexity" title="Descriptive complexity">logical characterization</a>: it is the set of languages expressible by <a href="/wiki/Second-order_logic" title="Second-order logic">second-order logic</a>.</p>
<p><b>PH</b> contains almost all well-known complexity classes inside <b>PSPACE</b>; in particular, it contains <b><a href="/wiki/P_(complexity)" title="P (complexity)">P</a></b>, <b><a href="/wiki/NP_(complexity)" title="NP (complexity)">NP</a></b>, and <b><a href="/wiki/Co-NP" title="Co-NP">co-NP</a></b>. It even contains probabilistic classes such as <b><a class="mw-redirect" href="/wiki/Bounded-error_probabilistic_polynomial" title="Bounded-error probabilistic polynomial">BPP</a></b> and <b><a href="/wiki/RP_(complexity)" title="RP (complexity)">RP</a></b>. However, there is some evidence that <b><a href="/wiki/BQP" title="BQP">BQP</a></b>, the class of problems solvable in polynomial time by a <a class="mw-redirect" href="/wiki/Quantum_computer" title="Quantum computer">quantum computer</a>, is not contained in <b>PH</b>.<sup class="reference" id="cite_ref-2"><a href="#cite_note-2">[2]</a></sup></p>
<p><b>P</b> = <b>NP</b> if and only if <b>P</b> = <b>PH</b>. This may simplify a potential proof of <b>P</b> ≠ <b>NP</b>, since it is only necessary to separate <b>P</b> from the more general class <b>PH</b>.<sup class="noprint Inline-Template Template-Fact" style="white-space:nowrap;">[<i><a href="/wiki/Wikipedia:Citation_needed" title="Wikipedia:Citation needed"><span title="This claim needs references to reliable sources. (December 2015)">citation needed</span></a></i>]</sup></p>
