<p>In <a href="/wiki/Topology" title="Topology">topology</a> and related branches of <a href="/wiki/Mathematics" title="Mathematics">mathematics</a>, a <b>T<sub>1</sub> space</b> is a <a href="/wiki/Topological_space" title="Topological space">topological space</a> in which, for every pair of distinct points, each has a <a href="/wiki/Neighbourhood_(mathematics)" title="Neighbourhood (mathematics)">neighborhood</a> not containing the other.<sup class="reference" id="cite_ref-1"><a href="#cite_note-1">[1]</a></sup> An <b>R<sub>0</sub> space</b> is one in which this holds for every pair of <a class="mw-redirect" href="/wiki/Topologically_distinguishable" title="Topologically distinguishable">topologically distinguishable</a> points. The properties T<sub>1</sub> and R<sub>0</sub> are examples of <a href="/wiki/Separation_axiom" title="Separation axiom">separation axioms</a>.</p>

<h2><span class="mw-headline" id="Definitions">Definitions</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=T1_space&amp;action=edit&amp;section=1" title="Edit section: Definitions">edit</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>Let <i>X</i> be a <a href="/wiki/Topological_space" title="Topological space">topological space</a> and let <i>x</i> and <i>y</i> be points in <i>X</i>. We say that <i>x</i> and <i>y</i> can be <i><a href="/wiki/Separated_sets" title="Separated sets">separated</a></i> if each lies in a <a href="/wiki/Neighbourhood_(mathematics)" title="Neighbourhood (mathematics)">neighborhood</a> which does not contain the other point.</p>
<ul>
<li><i>X</i> is a <b>T<sub>1</sub> space</b> if any two <a href="/wiki/Distinct_(mathematics)" title="Distinct (mathematics)">distinct</a> points in <i>X</i> are separated.</li>
<li><i>X</i> is an <b>R<sub>0</sub> space</b> if any two <a class="mw-redirect" href="/wiki/Topologically_distinguishable" title="Topologically distinguishable">topologically distinguishable</a> points in <i>X</i> are separated.</li>
</ul>
<p>A T<sub>1</sub> space is also called an <b>accessible space</b> or a <b>Fréchet space</b> and a R<sub>0</sub> space is also called a <b>symmetric space</b>. (The term <i>Fréchet space</i> also has an <a href="/wiki/Fr%C3%A9chet_space" title="Fréchet space">entirely different meaning</a> in <a href="/wiki/Functional_analysis" title="Functional analysis">functional analysis</a>. For this reason, the term <i>T<sub>1</sub> space</i> is preferred. There is also a notion of a <a class="mw-redirect" href="/wiki/Fr%C3%A9chet-Urysohn_space" title="Fréchet-Urysohn space">Fréchet-Urysohn space</a> as a type of <a href="/wiki/Sequential_space" title="Sequential space">sequential space</a>. The term <i>symmetric space</i> has <a href="/wiki/Symmetric_space" title="Symmetric space">another meaning</a>.)</p>
<h2><span class="mw-headline" id="Properties">Properties</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=T1_space&amp;action=edit&amp;section=2" title="Edit section: Properties">edit</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>Let <i>X</i> be a topological space. Then the following conditions are equivalent:</p>
<ul>
<li><i>X</i> is a T<sub>1</sub> space.</li>
<li><i>X</i> is a <a href="/wiki/Kolmogorov_space" title="Kolmogorov space">T<sub>0</sub> space</a> and a R<sub>0</sub> space.</li>
<li>Points are closed in <i>X</i>; i.e. given any <i>x</i> in <i>X</i>, the singleton set {<i>x</i>} is a <a href="/wiki/Closed_set" title="Closed set">closed set</a>.</li>
<li>Every subset of <i>X</i> is the intersection of all the open sets containing it.</li>
<li>Every <a href="/wiki/Finite_set" title="Finite set">finite set</a> is closed.<sup class="reference" id="cite_ref-2"><a href="#cite_note-2">[2]</a></sup></li>
<li>Every <a class="mw-redirect" href="/wiki/Cofinite" title="Cofinite">cofinite</a> set of <i>X</i> is open.</li>
<li>The <a class="mw-redirect" href="/wiki/Fixed_ultrafilter" title="Fixed ultrafilter">fixed ultrafilter</a> at <i>x</i> converges only to <i>x</i>.</li>
<li>For every subset <i>S</i> of <i>X</i> and every point <i>x</i> in <i>X \ S</i>, <i>x</i> is a <a href="/wiki/Limit_point" title="Limit point">limit point</a> of <i>S</i> if and only if every open <a class="mw-redirect" href="/wiki/Neighbourhood_(topology)" title="Neighbourhood (topology)">neighbourhood</a> of <i>x</i> contains infinitely many points of <i>S</i>.</li>
</ul>
<p>Let <i>X</i> be a topological space. Then the following conditions are equivalent:</p>
<ul>
<li><i>X</i> is an R<sub>0</sub> space.</li>
<li>Given any <i>x</i> in <i>X</i>, the <a href="/wiki/Closure_(topology)" title="Closure (topology)">closure</a> of {<i>x</i>} contains only the points that are topologically indistinguishable from <i>x</i> .</li>
<li>For any two points <i>x</i> and <i>y</i> in the space, <i>x</i> is in the closure of {<i>y</i>} if and only if <i>y</i> is in the closure of {<i>x</i>}.</li>
<li>The <a class="mw-redirect" href="/wiki/Specialization_preorder" title="Specialization preorder">specialization preorder</a> on <i>X</i> is <a href="/wiki/Symmetric_relation" title="Symmetric relation">symmetric</a> (and therefore an <a href="/wiki/Equivalence_relation" title="Equivalence relation">equivalence relation</a>).</li>
<li>The fixed ultrafilter at <i>x</i> converges only to the points that are topologically indistinguishable from <i>x</i>.</li>
<li>The <a class="mw-redirect" href="/wiki/Kolmogorov_quotient" title="Kolmogorov quotient">Kolmogorov quotient</a> of <i>X</i> (which identifies topologically indistinguishable points) is T<sub>1</sub>.</li>
<li>Every <a href="/wiki/Open_set" title="Open set">open set</a> is the union of <a class="mw-redirect" href="/wiki/Closed_sets" title="Closed sets">closed sets</a>.</li>
</ul>
<p>In any topological space we have, as properties of any two points, the following implications</p>
<dl>
<dd><i>separated</i> ⇒ <i>topologically distinguishable</i> ⇒ <i>distinct</i></dd>
</dl>
<p>If the first arrow can be reversed the space is R<sub>0</sub>. If the second arrow can be reversed the space is <a class="mw-redirect" href="/wiki/T0_space" title="T0 space">T<sub>0</sub></a>. If the composite arrow can be reversed the space is T<sub>1</sub>. Clearly, a space is T<sub>1</sub> if and only if it's both R<sub>0</sub> and T<sub>0</sub>.</p>
<p>Note that a finite T<sub>1</sub> space is necessarily <a href="/wiki/Discrete_space" title="Discrete space">discrete</a> (since every set is closed).</p>
<h2><span class="mw-headline" id="Examples">Examples</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=T1_space&amp;action=edit&amp;section=3" title="Edit section: Examples">edit</a><span class="mw-editsection-bracket">]</span></span></h2>
<ul>
<li><a class="mw-redirect" href="/wiki/Sierpinski_space" title="Sierpinski space">Sierpinski space</a> is a simple example of a topology that is T<sub>0</sub> but is not T<sub>1</sub>.</li>
<li>The <a href="/wiki/Overlapping_interval_topology" title="Overlapping interval topology">overlapping interval topology</a> is a simple example of a topology that is T<sub>0</sub> but is not T<sub>1</sub>.</li>
<li>The <a class="mw-redirect" href="/wiki/Cofinite_topology" title="Cofinite topology">cofinite topology</a> on an <a href="/wiki/Infinite_set" title="Infinite set">infinite set</a> is a simple example of a topology that is T<sub>1</sub> but is not <a href="/wiki/Hausdorff_space" title="Hausdorff space">Hausdorff</a> (T<sub>2</sub>). This follows since no two open sets of the cofinite topology are disjoint. Specifically, let <i>X</i> be the set of <a href="/wiki/Integer" title="Integer">integers</a>, and define the <a href="/wiki/Open_set" title="Open set">open sets</a> <i>O</i><sub><i>A</i></sub> to be those subsets of <i>X</i> which contain all but a <a href="/wiki/Finite_set" title="Finite set">finite</a> subset <i>A</i> of <i>X</i>. Then given distinct integers <i>x</i> and <i>y</i>:</li>
</ul>
<dl>
<dd>
<ul>
<li>the open set <i>O</i><sub>{<i>x</i>}</sub> contains <i>y</i> but not <i>x</i>, and the open set <i>O</i><sub>{<i>y</i>}</sub> contains <i>x</i> and not <i>y</i>;</li>
<li>equivalently, every singleton set {<i>x</i>} is the complement of the open set <i>O</i><sub>{<i>x</i>}</sub>, so it is a closed set;</li>
</ul>
</dd>
<dd>so the resulting space is T<sub>1</sub> by each of the definitions above. This space is not T<sub>2</sub>, because the <a href="/wiki/Intersection_(set_theory)" title="Intersection (set theory)">intersection</a> of any two open sets <i>O</i><sub><i>A</i></sub> and <i>O</i><sub><i>B</i></sub> is <i>O</i><sub><i>A</i>∪<i>B</i></sub>, which is never empty. Alternatively, the set of even integers is <a class="mw-redirect" href="/wiki/Compact_set" title="Compact set">compact</a> but not <a href="/wiki/Closed_set" title="Closed set">closed</a>, which would be impossible in a Hausdorff space.</dd>
</dl>
<ul>
<li>The above example can be modified slightly to create the <a class="mw-redirect" href="/wiki/Double-pointed_cofinite_topology" title="Double-pointed cofinite topology">double-pointed cofinite topology</a>, which is an example of an R<sub>0</sub> space that is neither T<sub>1</sub> nor R<sub>1</sub>. Let <i>X</i> be the set of integers again, and using the definition of <i>O</i><sub><i>A</i></sub> from the previous example, define a <a href="/wiki/Subbase" title="Subbase">subbase</a> of open sets <i>G</i><sub><i>x</i></sub> for any integer <i>x</i> to be <i>G</i><sub><i>x</i></sub> = <i>O</i><sub>{<i>x</i>, <i>x</i>+1}</sub> if <i>x</i> is an <a class="mw-redirect" href="/wiki/Even_number" title="Even number">even number</a>, and <i>G</i><sub><i>x</i></sub> = <i>O</i><sub>{<i>x</i>-1, <i>x</i>}</sub> if <i>x</i> is odd. Then the <a class="mw-redirect" href="/wiki/Basis_(topology)" title="Basis (topology)">basis</a> of the topology are given by finite <a href="/wiki/Intersection_(set_theory)" title="Intersection (set theory)">intersections</a> of the subbasis sets: given a finite set <i>A</i>, the open sets of <i>X</i> are</li>
</ul>
<dl>
<dd>
<dl>

<dd>The resulting space is not T<sub>0</sub> (and hence not T<sub>1</sub>), because the points <i>x</i> and <i>x</i> + 1 (for <i>x</i> even) are topologically indistinguishable; but otherwise it is essentially equivalent to the previous example.</dd>

<ul>
<li>The <a href="/wiki/Zariski_topology" title="Zariski topology">Zariski topology</a> on an <a href="/wiki/Algebraic_variety" title="Algebraic variety">algebraic variety</a> (over an <a href="/wiki/Algebraically_closed_field" title="Algebraically closed field">algebraically closed field</a>) is T<sub>1</sub>. To see this, note that a point with <a href="/wiki/Local_coordinates" title="Local coordinates">local coordinates</a> (<i>c</i><sub>1</sub>,...,<i>c</i><sub><i>n</i></sub>) is the <a href="/wiki/Zero_set" title="Zero set">zero set</a> of the <a href="/wiki/Polynomial" title="Polynomial">polynomials</a> <i>x</i><sub>1</sub>-<i>c</i><sub>1</sub>, ..., <i>x</i><sub><i>n</i></sub>-<i>c</i><sub><i>n</i></sub>. Thus, the point is closed. However, this example is well known as a space that is not <a href="/wiki/Hausdorff_space" title="Hausdorff space">Hausdorff</a> (T<sub>2</sub>). The Zariski topology is essentially an example of a cofinite topology.</li>
<li>The Zariski topology on a <a href="/wiki/Commutative_ring" title="Commutative ring">commutative ring</a> (that is, the prime <a href="/wiki/Spectrum_of_a_ring" title="Spectrum of a ring">spectrum of a ring</a>) is T<sub>0</sub> but not, in general, T<sub>1</sub>.<sup class="reference" id="cite_ref-3"><a href="#cite_note-3">[3]</a></sup> To see this, note that the closure of a one-point set is the set of all <a href="/wiki/Prime_ideal" title="Prime ideal">prime ideals</a> that contain the point (and thus the topology is T<sub>0</sub>). However, this closure is a <a href="/wiki/Maximal_ideal" title="Maximal ideal">maximal ideal</a>, and the only closed points are the maximal ideals, and are thus not contained in any of the open sets of the topology, and thus the space does not satisfy axiom T<sub>1</sub>. To be clear about this example: the Zariski topology for a commutative ring <i>A</i> is given as follows: the topological space is the set <i>X</i> of all <a href="/wiki/Prime_ideal" title="Prime ideal">prime ideals</a> of <i>A</i>. The <a href="/wiki/Base_(topology)" title="Base (topology)">base of the topology</a> is given by the open sets <i>O</i><sub><i>a</i></sub> of prime ideals that do <i>not</i> contain <i>a</i> in <i>A</i>. It is straightforward to verify that this indeed forms the basis: so <i>O</i><sub><i>a</i></sub> ∩ <i>O</i><sub><i>b</i></sub> = <i>O</i><sub><i>ab</i></sub> and <i>O</i><sub><i>0</i></sub> = Ø and <i>O</i><sub><i>1</i></sub> = <i>X</i>. The closed sets of the Zariski topology are the sets of prime ideals that <i>do</i> contain <i>a</i>. Notice how this example differs subtly from the cofinite topology example, above: the points in the topology are not closed, in general, whereas in a T<sub>1</sub> space, points are always closed.</li>
<li>Every <a class="mw-redirect" href="/wiki/Totally_disconnected" title="Totally disconnected">totally disconnected</a> space is T<sub>1</sub>, since every point is a <a class="mw-redirect" href="/wiki/Connected_component_(topology)" title="Connected component (topology)">connected component</a> and therefore closed.</li>
</ul>
<h2><span class="mw-headline" id="Generalisations_to_other_kinds_of_spaces">Generalisations to other kinds of spaces</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=T1_space&amp;action=edit&amp;section=4" title="Edit section: Generalisations to other kinds of spaces">edit</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>The terms "T<sub>1</sub>", "R<sub>0</sub>", and their synonyms can also be applied to such variations of topological spaces as <a href="/wiki/Uniform_space" title="Uniform space">uniform spaces</a>, <a href="/wiki/Cauchy_space" title="Cauchy space">Cauchy spaces</a>, and <a class="new" href="/w/index.php?title=Convergence_space&amp;action=edit&amp;redlink=1" title="Convergence space (page does not exist)">convergence spaces</a>. The characteristic that unites the concept in all of these examples is that limits of fixed ultrafilters (or constant <a class="mw-redirect" href="/wiki/Net_(topology)" title="Net (topology)">nets</a>) are unique (for T<sub>1</sub> spaces) or unique up to topological indistinguishability (for R<sub>0</sub> spaces).</p>
<p>As it turns out, uniform spaces, and more generally Cauchy spaces, are always R<sub>0</sub>, so the T<sub>1</sub> condition in these cases reduces to the T<sub>0</sub> condition. But R<sub>0</sub> alone can be an interesting condition on other sorts of convergence spaces, such as <a href="/wiki/Pretopological_space" title="Pretopological space">pretopological spaces</a>.</p>
