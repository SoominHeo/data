<p><b>Toad</b> is a database management toolset from <a href="/wiki/Quest_Software" title="Quest Software">Quest Software</a> that <a href="/wiki/Database" title="Database">database</a> developers, database administrators and data analysts use to manage both relational and non-relational databases using <a href="/wiki/SQL" title="SQL">SQL</a>.</p>

<p>Toad runs on all 32-bit/64-bit Windows platforms, including Microsoft Windows Server, Windows XP, Windows Vista, Windows 7 and 8 (32-Bit or 64-Bit). <a href="/wiki/Quest_Software" title="Quest Software">Quest Software</a> has also released a Toad Mac Edition. Quest provides Toad in commercial and trial/<a href="/wiki/Freeware" title="Freeware">freeware</a> versions. The Freeware version is available from the ToadWorld.com community.<sup class="reference" id="cite_ref-1"><a href="#cite_note-1">[1]</a></sup></p>
<h2><span class="mw-headline" id="History">History</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Toad_(software)&amp;action=edit&amp;section=2" title="Edit section: History">edit</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>Jim McDaniel (aka "TOADman"), a practicing Oracle DBA, originally designed Toad for his own use in the late 1990s.<sup class="reference" id="cite_ref-2"><a href="#cite_note-2">[2]</a></sup> He called it <i>Tool for Oracle Application Developers</i>: "TOAD" for short. McDaniel made the tool available to others first as <a href="/wiki/Shareware" title="Shareware">shareware</a> and later as <a href="/wiki/Freeware" title="Freeware">freeware</a>, distributing it freely over the Internet.</p>
<p>As the user base grew, so did the effort of maintenance and development. Finally, the author realized that he could no longer dedicate substantial amounts of time to the growing quantity of software enhancement requests.</p>
<p><a href="/wiki/Quest_Software" title="Quest Software">Quest Software</a>, established in 1987, acquired TOAD in October 1998<sup class="reference" id="cite_ref-3"><a href="#cite_note-3">[3]</a></sup> and became part of Dell in September 2012 as Dell Software.<sup class="reference" id="cite_ref-About_Quest_Software_.E2.80.93_now_a_part_of_Dell_4-0"><a href="#cite_note-About_Quest_Software_.E2.80.93_now_a_part_of_Dell-4">[4]</a></sup> In 2016, Dell Software was sold off<sup class="reference" id="cite_ref-5"><a href="#cite_note-5">[5]</a></sup> and renamed back to Quest Software.<sup class="reference" id="cite_ref-6"><a href="#cite_note-6">[6]</a></sup></p>
