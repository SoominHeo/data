<p><b>True Audio</b> is a <a class="mw-redirect" href="/wiki/Lossless" title="Lossless">lossless</a> compressor for multichannel 8, 16 and 24 bits audio data. <b>.tta</b> is the <a href="/wiki/Filename_extension" title="Filename extension">extension</a> to <a href="/wiki/Filename" title="Filename">filenames</a> of audio files created by the <b>True Audio</b> codec.</p>

<h2><span class="mw-headline" id="Codec_overview">Codec overview</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=TTA_(codec)&amp;action=edit&amp;section=1" title="Edit section: Codec overview">edit</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>True Audio <a href="/wiki/Data_compression_ratio" title="Data compression ratio">compresses</a> up to 30% of the original, broadly similar to <a href="/wiki/FLAC" title="FLAC">FLAC</a> and <a href="/wiki/Monkey%27s_Audio" title="Monkey's Audio">APE</a>. It features a real-time encoding and decoding <a href="/wiki/Algorithm" title="Algorithm">algorithm</a> and hardware compression support. As with most other lossless codecs, <a class="mw-redirect" href="/wiki/Plugins" title="Plugins">plugins</a> are available for most media players.</p>
<p><b>TTA</b> performs <a class="mw-redirect" href="/wiki/Lossless" title="Lossless">lossless</a> compression on multichannel 8, 16 and 24 <a href="/wiki/Bit" title="Bit">bit</a> data of uncompressed <a class="mw-redirect" href="/wiki/Wav" title="Wav">wav</a> input files. The term "lossless" refers to the fact that such compression results in no data or quality loss; when decompressed, the audio file data are bit-identical to those of their originals. Compression ratios achieved by the <b>TTA</b> codec vary, depending on music type, but range from 30% to 70% of the original. The <b>TTA</b> lossless compressed audio format supports both <a href="/wiki/ID3" title="ID3">ID3v1</a>/<a href="/wiki/ID3" title="ID3">ID3v2</a> information tags and <a class="mw-redirect" href="/wiki/APEv2" title="APEv2">APEv2</a>.</p>
<p>The <b>TTA</b> lossless audio codec allows for the storage of up to 20 audio <a href="/wiki/Compact_disc" title="Compact disc">CDs</a> worth of music on a single <a href="/wiki/DVD" title="DVD">DVD-R</a>,<sup class="noprint Inline-Template" style="white-space:nowrap;">[<i><a href="/wiki/Wikipedia:Citing_sources" title="Wikipedia:Citing sources"><span title="Statement needs to be more specific about the content to which it refers. (November 2011)">specify</span></a></i>]</sup> retaining the original <a href="/wiki/Compact_disc" title="Compact disc">CD</a> quality audio, plus detailed information in the <a href="/wiki/ID3" title="ID3">ID3</a> tag format.</p>
<p>All <b>TTA</b> source code and binaries are freely available and distributed under <a class="mw-redirect" href="/wiki/GPL" title="GPL">GPL</a> Open Source licenses.</p>
