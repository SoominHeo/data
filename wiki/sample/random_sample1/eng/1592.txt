<h1 class="firstHeading" id="firstHeading" lang="en">UIC identification marking for tractive stock</h1>
<div class="mw-body-content" id="bodyContent">
<div id="siteSub">From Wikipedia, the free encyclopedia</div>
<div id="contentSub"></div>
<div class="mw-jump" id="jump-to-nav">
					Jump to:					<a href="#mw-head">navigation</a>, 					<a href="#p-search">search</a>
</div>
<div class="mw-content-ltr" dir="ltr" id="mw-content-text" lang="en"><script>function mfTempOpenSection(id){var block=document.getElementById("mf-section-"+id);block.className+=" open-block";block.previousSibling.className+=" open-block";}</script><div class="hatnote" role="note">This article is about the UIC system of classifying tractive stock and multiples. For locomotives, see <a href="/wiki/UIC_classification_of_locomotive_axle_arrangements" title="UIC classification of locomotive axle arrangements">UIC classification of locomotive axle arrangements</a>.</div>
<p>Since the beginning of 2007 newly authorised locomotives or other traction units in Europe have been given a 12-digit number, as required by the <a class="mw-redirect" href="/wiki/European_Railway_Agency" title="European Railway Agency">European Railway Agency</a>'s <b>CR OPE TSI</b> (Conventional Rail OPErations Technical Specification for Interoperability).<sup class="reference" id="cite_ref-1"><a href="#cite_note-1">[1]</a></sup> This makes the locomotive clearly identifiable from an <a href="/wiki/European_Union" title="European Union">EU</a>-wide list of registered railway vehicles and follows the rules introduced by the <b>UIC Code of Practice 438-3, Identification marking for tractive stock</b>.</p>
<p>A complete number comprises 12 digits. Digits 1–2 are the type code, 3–4 the country of origin, 5–11 are defined by the country concerned, 12 is the check digit.<sup class="reference" id="cite_ref-2"><a href="#cite_note-2">[2]</a></sup> It is followed by abbreviations for the country of origin and the owner of the vehicle.</p>
<p>The <a href="/wiki/European_Union" title="European Union">EU</a> CR OPE <a href="/wiki/Technical_Specifications_for_Interoperability" title="Technical Specifications for Interoperability">TSI</a> required that by 1 August 2008 all traction units had to be registered and marked with their number. Existing locomotive numbers used internally were not affected.</p>

<h2><span class="mw-headline" id="National_Block">National Block</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=UIC_identification_marking_for_tractive_stock&amp;action=edit&amp;section=4" title="Edit section: National Block">edit</a><span class="mw-editsection-bracket">]</span></span></h2>
<h3><span class="mw-headline" id="Germany">Germany</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=UIC_identification_marking_for_tractive_stock&amp;action=edit&amp;section=5" title="Edit section: Germany">edit</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>A register of tractive units has been introduced by the <a class="mw-redirect" href="/wiki/Federal_Railway_Office" title="Federal Railway Office">Federal Railway Office</a> (<i>Eisenbahnbundesamt</i>, <i>EBA</i>), which also issues vehicle numbers. Digits 5–8 are a four-digit class number, 9–11 the three-digit serial number of the vehicle within its class.</p>
<p>For class numbers for vehicles in the existing <a href="/wiki/Deutsche_Bahn" title="Deutsche Bahn">Deutsche Bahn</a> fleet, DB selected digits 5–8 so that the previous number could continue to be used without affecting the check digit: changing the check digit was needed only for combustion-engined railcars. For new DB vehicles, such as the <a href="/wiki/Alstom" title="Alstom">Alstom</a> Class 1214 introduced in 2008, this is not valid. Although they are internally counted as Class 262, they have been given a number corresponding to the class used by the manufacturer.</p>
<p>As a result of the requirement to be entered into the register, vehicles and classes were included that had never been part of the DB. These were allocated to new classes by the EBA, no longer based around DB historical classification. This has caused misunderstandings, because the classes are now generally made up of four-digit numbers (digits 5–8) that no longer indicate the type of vehicle, now shown by the type code (digits 1–2).</p>
<p><b>Example</b>:</p>
<p>From the last three digits, class "0128" could appear to be an electric locomotive. However the full number includes the type code "98", so it is a diesel shunter. The three-digit numbers "3xy" or "2xy" previously used by DB are not part of the new numbers. They are obsolete, found only in the numbers of former state railway vehicles.</p>
<p>Vehicles that are used both by DB and private railways are allocated to the same class – the difference is recognisable from the owner's abbreviation. One advantage of the system is its simplicity, because when vehicles change ownership they retain the same number: only the abbreviation of the owner's name needs to be changed. For a major rebuild the number can be changed to that of an appropriate class.</p>
<p>See also <a class="new" href="/w/index.php?title=List_of_classes_in_the_German_Railway_Vehicle_Register&amp;action=edit&amp;redlink=1" title="List of classes in the German Railway Vehicle Register (page does not exist)">List of classes in the German Railway Vehicle Register</a></p>
<p>The method of applying and spacing the digits differs between operators. For example a <a href="/wiki/British_Rail_Class_66" title="British Rail Class 66">British Rail Class 66</a> loco registered in Britain has been seen to carry the number: GB 9 2 70 0 <u>066246</u>-4, with the existing national number "66246" underlined, and lacking the operator identifying code.<sup class="reference" id="cite_ref-5"><a href="#cite_note-5">[5]</a></sup> Frequently the commonly used national part of the number is underlined in order to make it more readily identifiable.</p>
<h2><span class="mw-headline" id="Check_digit">Check digit</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=UIC_identification_marking_for_tractive_stock&amp;action=edit&amp;section=6" title="Edit section: Check digit">edit</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>The check digit is calculated from digits 1–11. It is derived from the sum of the numbers that arise when the digits are alternately multiplied by 2 and 1; the difference of this sum from the next multiple of ten produces the check digit<sup class="reference" id="cite_ref-6"><a href="#cite_note-6">[6]</a></sup> (for examples of this see <a class="mw-redirect" href="/wiki/Wagon_number#Examples" title="Wagon number">wagon numbers</a>).</p>
<h2><span class="mw-headline" id="Owner.27s_code">Owner's code</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=UIC_identification_marking_for_tractive_stock&amp;action=edit&amp;section=7" title="Edit section: Owner's code">edit</a><span class="mw-editsection-bracket">]</span></span></h2>
<div class="hatnote" role="note">Main article: <a href="/wiki/Reporting_mark" title="Reporting mark">Reporting mark</a></div>
<p>For every railway that enters its tractive units into the register, a clear European-wide code is needed, separate from the number. This alphanumerical <b><a class="mw-redirect" href="/wiki/Vehicle_keeper_marking" title="Vehicle keeper marking">Vehicle Keeper Marking</a></b> (VKM) is allocated by a railway administration in accordance with CR OPE TSI.<sup class="reference" id="cite_ref-7"><a href="#cite_note-7">[7]</a></sup> The list of owner's codes is linked below.</p>

<p>In the past, each railway adopted its own identification system. Each locomotive was given a <b>road number</b> or <b>running number</b> which would sometimes be combined with a <b>class number</b>. For example <a href="/wiki/British_Rail_locomotive_and_multiple_unit_numbering_and_classification" title="British Rail locomotive and multiple unit numbering and classification">British Rail diesel and electric locomotives</a> had five-digit numbers of which the first two digits were the class number and the remaining digits were the running number, e.g. <a href="/wiki/British_Rail_Class_37" title="British Rail Class 37">37 409</a>.</p>
