<p><b>VirusTotal</b> is a <a href="/wiki/Website" title="Website">website</a> created by the Spanish security company Hispasec Sistemas. Launched in June 2004, it was acquired by <a href="/wiki/Google" title="Google">Google Inc.</a> in September 2012.<sup class="reference" id="cite_ref-1"><a href="#cite_note-1">[1]</a></sup><sup class="reference" id="cite_ref-2"><a href="#cite_note-2">[2]</a></sup></p>
<p>VirusTotal aggregates many <a href="/wiki/Antivirus_software" title="Antivirus software">antivirus</a> products and online scan engines<sup class="reference" id="cite_ref-3"><a href="#cite_note-3">[3]</a></sup><sup class="reference" id="cite_ref-4"><a href="#cite_note-4">[4]</a></sup> to check for viruses that the user's own antivirus may have missed, or to verify against any <a class="mw-redirect" href="/wiki/False_positive" title="False positive">false positives</a>.<sup class="reference" id="cite_ref-5"><a href="#cite_note-5">[5]</a></sup> Files up to 128 MB can be uploaded to the website or sent via email.<sup class="reference" id="cite_ref-6"><a href="#cite_note-6">[6]</a></sup> Anti-virus software vendors can receive copies of files that were flagged by other scans but passed by their own engine, to help improve their software and, by extension, VirusTotal's own capability. Users can also scan suspect <a class="mw-redirect" href="/wiki/URL" title="URL">URLs</a> and search through the VirusTotal dataset. VirusTotal for dynamic analysis of malware use Cuckoo <a href="/wiki/Sandbox_(computer_security)" title="Sandbox (computer security)">sandbox</a>.<sup class="reference" id="cite_ref-7"><a href="#cite_note-7">[7]</a></sup> VirusTotal was selected by <i><a class="mw-redirect" href="/wiki/PC_World_(magazine)" title="PC World (magazine)">PC World</a></i> as one of the best 100 products of 2007.<sup class="reference" id="cite_ref-2007BestProducts_8-0"><a href="#cite_note-2007BestProducts-8">[8]</a></sup></p>

<h2><span class="mw-headline" id="Products_and_services">Products and services</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=VirusTotal&amp;action=edit&amp;section=1" title="Edit section: Products and services">edit</a><span class="mw-editsection-bracket">]</span></span></h2>
<h3><span class="mw-headline" id="VTUploader_for_Microsoft_Windows">VTUploader for Microsoft Windows</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=VirusTotal&amp;action=edit&amp;section=2" title="Edit section: VTUploader for Microsoft Windows">edit</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>VTUploader<sup class="reference" id="cite_ref-9"><a href="#cite_note-9">[9]</a></sup> is an application that integrates into the Explorer's (right-click) contextual menu, listed under <i>Send To &gt; Virus Total</i>. The application also launches manually for submitting a URL or a program that is currently running in the OS.</p>
<p>VirusTotal stores the name and various hashes for each scanned file. Already scanned files can be identified by their known (e.g., VT default) <a href="/wiki/SHA-2" title="SHA-2">SHA256</a> hash without uploading complete files. The SHA256 query <a class="mw-redirect" href="/wiki/URL" title="URL">URL</a> has the form <code>https://www.virustotal.com/latest-scan/SHA256</code>. File uploads are normally limited to 128 MB.<sup class="reference" id="cite_ref-10"><a href="#cite_note-10">[10]</a></sup></p>
<h3><span class="mw-headline" id="VirusTotal_for_Browsers">VirusTotal for Browsers</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=VirusTotal&amp;action=edit&amp;section=3" title="Edit section: VirusTotal for Browsers">edit</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>There are several browser extensions available, such as <i><a class="external text" href="https://www.virustotal.com/en/documentation/browser-extensions/" rel="nofollow">VTzilla</a></i> for <a class="mw-redirect" href="/wiki/Mozilla_Firefox" title="Mozilla Firefox">Mozilla Firefox</a>, <i><a class="external text" href="https://www.virustotal.com/en/documentation/browser-extensions/google-chrome" rel="nofollow">VTchromizer</a></i> for <a href="/wiki/Google_Chrome" title="Google Chrome">Google Chrome</a> and <i><a class="external text" href="https://www.virustotal.com/en/documentation/browser-extensions/internet-explorer" rel="nofollow">VTexplorer</a></i> for <a href="/wiki/Internet_Explorer" title="Internet Explorer">Internet Explorer</a>. They allow the user to download files directly with VirusTotal's web application prior to storing them in the computer, as well as scanning URLs.<sup class="reference" id="cite_ref-11"><a href="#cite_note-11">[11]</a></sup></p>
<h3><span class="mw-headline" id="VirusTotal_for_Mobile">VirusTotal for Mobile</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=VirusTotal&amp;action=edit&amp;section=4" title="Edit section: VirusTotal for Mobile">edit</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>The service also offers an <a href="/wiki/Android_(operating_system)" title="Android (operating system)">Android</a> <a href="/wiki/Application_software" title="Application software">App</a><sup class="reference" id="cite_ref-12"><a href="#cite_note-12">[12]</a></sup> that employs the public <a class="mw-redirect" href="/wiki/API" title="API">API</a> to search any installed application for VirusTotal's previously scanned ones and show its status. Any application not previously scanned can be submitted, but an API key must be provided and other restrictions to public API usage may apply (see <a href="#Public_API">#Public API</a>).</p>
<h3><span class="mw-headline" id="Public_API">Public API</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=VirusTotal&amp;action=edit&amp;section=5" title="Edit section: Public API">edit</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>VirusTotal provides as a free service a public API that allows for automation of some of its online features such as <i>"upload and scan files, submit and scan URLs, access finished scan reports and make automatic comments on URLs and samples"</i>. Some restrictions apply for requests made through the public API, such as requiring an individual API key freely obtained by online signing up, low priority scan queue, limited number of requests per time frame, etc.<sup class="reference" id="cite_ref-13"><a href="#cite_note-13">[13]</a></sup></p>
<h3><span class="mw-headline" id="Antivirus_products">Antivirus products</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=VirusTotal&amp;action=edit&amp;section=6" title="Edit section: Antivirus products">edit</a><span class="mw-editsection-bracket">]</span></span></h3>
