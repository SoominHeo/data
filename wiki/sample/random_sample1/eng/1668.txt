<p><b>XMPlay</b> is a <a href="/wiki/Freeware" title="Freeware">freeware</a> <a class="mw-redirect" href="/wiki/Audio_player_(software)" title="Audio player (software)">audio player</a> for <a href="/wiki/Microsoft_Windows" title="Microsoft Windows">Windows</a>. Initially released in 1998, it sets itself apart from its competitors by providing a particularly authentic reproduction of <a href="/wiki/Music_tracker" title="Music tracker">tracker</a> audio files.<sup class="reference" id="cite_ref-1"><a href="#cite_note-1">[1]</a></sup></p>
<p></p>

<h2><span class="mw-headline" id="Overview">Overview</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=XMPlay&amp;action=edit&amp;section=1" title="Edit section: Overview">edit</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>Developed by <i>Un4seen Developments</i> in 1998, it originally only supported the <a href="/wiki/XM_(file_format)" title="XM (file format)">XM</a> <a href="/wiki/File_format" title="File format">file format</a> of <a href="/wiki/FastTracker_2" title="FastTracker 2">Fast Tracker II</a>, from where the name "XMPlay" originates.</p>
<p>Nowadays XMPlay is able to handle many more tracker file formats, as well as today's widely used streamed music formats such as <a href="/wiki/MP3" title="MP3">MP3</a>, <a href="/wiki/Vorbis" title="Vorbis">Ogg Vorbis</a>, <a href="/wiki/Advanced_Audio_Coding" title="Advanced Audio Coding">AAC</a>, <a class="mw-redirect" href="/wiki/Opus_(audio_codec)" title="Opus (audio codec)">Opus</a>, <a href="/wiki/WAV" title="WAV">WAV</a>, <a href="/wiki/Windows_Media_Audio" title="Windows Media Audio">WMA</a>, as well as a lot of less common formats,<sup class="reference" id="cite_ref-2"><a href="#cite_note-2">[2]</a></sup> through <a href="/wiki/Plug-in_(computing)" title="Plug-in (computing)">plug-ins</a> found on the website. Plugins from the popular <a href="/wiki/Winamp" title="Winamp">Winamp</a> audio player can also be used to extend XMPlay's capabilities.</p>
<p>XMPlay is very powerful yet lightweight software:<sup class="reference" id="cite_ref-3"><a href="#cite_note-3">[3]</a></sup> the whole package only takes up less than 360 KB.</p>
<p>The software also offers typical audio player features, such as being able to get reception from <a href="/wiki/Internet_radio" title="Internet radio">internet radio</a>, organizing music files into <a href="/wiki/Playlist" title="Playlist">playlists</a> and the visualization of sounds generated by real-time animations.</p>
<p>Changing the look of the software is also available through skins.</p>
<p>Another notable feature that XMPlay has is that it can handle true <a href="/wiki/Gapless_playback" title="Gapless playback">gapless playback</a> between files.<sup class="reference" id="cite_ref-4"><a href="#cite_note-4">[4]</a></sup><sup class="reference" id="cite_ref-5"><a href="#cite_note-5">[5]</a></sup></p>
<h2><span class="mw-headline" id="Engine">Engine</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=XMPlay&amp;action=edit&amp;section=2" title="Edit section: Engine">edit</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>XMPlay's <a href="/wiki/Engine_(computer_science)" title="Engine (computer science)">engine</a> for processing audio files is available as a separate program <a href="/wiki/Library_(computing)" title="Library (computing)">library</a> under the name of <i>BASS</i> or <i>BassMod</i> (only for tracker files) or <i>BASSASIO</i> (<a href="/wiki/Steinberg" title="Steinberg">Steinberg</a>'s <a href="/wiki/Audio_Stream_Input/Output" title="Audio Stream Input/Output">ASIO protocol</a>). These libraries can also be used on non-Windows operating systems, they are free for non-commercial use, with commercial use needing to pay a fee for a license (with the exception of <i>BassMod</i>, which doesn't require a license). For the <a href="/wiki/Unix" title="Unix">Unix</a> audio player, "<a href="/wiki/XMMS" title="XMMS">X Multimedia System</a>" (XMMS), a MOD playing plug-in based on <i>BassMod</i> is available.<sup class="reference" id="cite_ref-6"><a href="#cite_note-6">[6]</a></sup><sup class="reference" id="cite_ref-7"><a href="#cite_note-7">[7]</a></sup></p>
