<p><b>Zipeg</b> is an <a class="mw-redirect" href="/wiki/Open_source" title="Open source">open source</a> <a href="/wiki/Free_software" title="Free software">free software</a> that <a href="/wiki/Data_extraction" title="Data extraction">extracts</a> files from compressed archives like <a class="mw-redirect" href="/wiki/ZIP_(file_format)" title="ZIP (file format)">ZIP</a>, <a href="/wiki/RAR_(file_format)" title="RAR (file format)">RAR</a>, <a href="/wiki/7z" title="7z">7z</a> and others, some of which are rarely used types. Zipeg works under <a class="mw-redirect" href="/wiki/Mac_OS_X" title="Mac OS X">Mac OS X</a> and <a href="/wiki/Microsoft_Windows" title="Microsoft Windows">Windows</a>. It is best known for its file preview ability. It is incapable of compressing files, although it is able to extract compressed ones. Zipeg is built on top of the <a href="/wiki/7-Zip" title="7-Zip">7-Zip</a> backend. Its UI is implemented in Java and is open source.</p>
<p>Zipeg automatically detects filenames in national alphabets (code pages) and correctly translates them to Unicode. Zipeg reads Exif thumbnails from JPEG digital photographs and uses them for "tool tip" style preview and item icons.</p>
