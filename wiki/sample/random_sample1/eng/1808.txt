<p>An <b>interpunct</b> ( <b>·</b> ), also known as an <b>interpoint</b>,<sup class="reference" id="cite_ref-1"><a href="#cite_note-1">[1]</a></sup> <b>middle dot</b>, <b>middot</b>, and <b>centered dot</b> or <b>centred dot</b>, is a punctuation mark consisting of a vertically centered dot used for <a class="mw-redirect" href="/wiki/Interword_separation" title="Interword separation">interword separation</a> in ancient <a href="/wiki/Latin_alphabet" title="Latin alphabet">Latin script</a>. (Word-separating <a href="/wiki/Space_(punctuation)" title="Space (punctuation)">spaces</a> did not appear until some time between <span class="smallcaps"><span style="font-variant: small-caps; text-transform: lowercase;">A.D.</span></span> 600 and 800.) It appears in a variety of uses in some modern languages and is present in <a href="/wiki/Unicode" title="Unicode">Unicode</a> as code point <span class="nowrap"><span style="font-family: monospace, monospace;">U+00B7</span></span> <span style="font-size:125%">·</span> <span class="smallcaps" style="font-variant: small-caps; font-size: smaller;">MIDDLE DOT</span> (HTML <code>&amp;#183;</code> <b>·</b> <code>&amp;middot;</code>).</p>
<p>The <b>multiplication dot</b> (<span class="nowrap"><span style="font-family: monospace, monospace;">U+2219</span></span> <span style="font-size:125%">∙</span> <span class="smallcaps" style="font-variant: small-caps; font-size: smaller;">BULLET OPERATOR</span> (HTML <code>&amp;#8729;</code>) or <span class="nowrap"><span style="font-family: monospace, monospace;">U+22C5</span></span> <span style="font-size:125%">⋅</span> <span class="smallcaps" style="font-variant: small-caps; font-size: smaller;">DOT OPERATOR</span> (HTML <code>&amp;#8901;</code> <b>·</b> <code>&amp;sdot;</code>)), whose glyphs are similar or identical to the interpunct, is a <a href="/wiki/Multiplication_sign" title="Multiplication sign">multiplication sign</a> optionally used instead of the styled <a class="mw-redirect" href="/wiki/%C3%97" title="×">×</a>: <span class="nowrap"><b>a</b>⋅<b>b</b></span> is equivalent to <span class="nowrap"><b>a</b> × <b>b</b></span> or "a times b". The same sign is also used in <a href="/wiki/Euclidean_vector" title="Euclidean vector">vector</a> multiplication to discriminate between the <a class="mw-redirect" href="/wiki/Scalar_product" title="Scalar product">scalar product</a> (<span class="nowrap"><b>a</b> ⋅ <b>b</b></span>) and the vector <a href="/wiki/Cross_product" title="Cross product">cross product</a> (<span class="nowrap"><b>a</b> × <b>b</b></span>) or <a class="mw-redirect" href="/wiki/Exterior_product" title="Exterior product">exterior product</a> (<span class="nowrap"><b>a</b> ∧ <b>b</b></span>). As a multiplication operator, it is also encountered in symbols for compound units such as the <a class="mw-redirect" href="/wiki/Newton-meter" title="Newton-meter">newton-meter</a> (N∙m or <span class="nowrap">N m</span>). The multiplication dot is a separate <a href="/wiki/Unicode" title="Unicode">Unicode</a> character (U+22C5), but is often silently replaced by the interpunct or <a href="/wiki/Bullet_(typography)" title="Bullet (typography)">bullet</a> (<span class="nowrap"><span style="font-family: monospace, monospace;">U+2022</span></span> <span style="font-size:125%">•</span> <span class="smallcaps" style="font-variant: small-caps; font-size: smaller;">BULLET</span> (HTML <code>&amp;#8226;</code> <b>·</b> <code>&amp;bull;</code>)).</p>

<h2><span class="mw-headline" id="In_written_language">In written language</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Interpunct&amp;action=edit&amp;section=1" title="Edit section: In written language">edit</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>Various dictionaries often use the interpunct (in this context, sometimes called hyphenation point) to indicate syllabification within a word with multiple syllables. There is also a separate Unicode character, <span class="nowrap"><span style="font-family: monospace, monospace;">U+2027</span></span> <span style="font-size:125%">‧</span> <span class="smallcaps" style="font-variant: small-caps; font-size: smaller;">HYPHENATION POINT</span>.</p>
<h3><span class="mw-headline" id="English">English</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Interpunct&amp;action=edit&amp;section=2" title="Edit section: English">edit</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>In <a href="/wiki/British_English" title="British English">British</a> <a href="/wiki/Typography" title="Typography">typography</a>, the <b>space dot</b> is an interpunct used as the formal <a class="mw-redirect" href="/wiki/Decimal_point" title="Decimal point">decimal point</a>. Its use is advocated by laws and by academic circles such as the <a class="mw-redirect" href="/wiki/Cambridge_University" title="Cambridge University">Cambridge University</a> History Faculty Style Guide<sup class="reference" id="cite_ref-2"><a href="#cite_note-2">[2]</a></sup> and is mandated by some UK-based academic journals such as <i><a href="/wiki/The_Lancet" title="The Lancet">The Lancet</a></i>.<sup class="reference" id="cite_ref-3"><a href="#cite_note-3">[3]</a></sup> When the British <a href="/wiki/Currency" title="Currency">currency</a> was <a href="/wiki/Decimalisation" title="Decimalisation">decimalised</a> in 1971, the official advice issued was to write decimal amounts with a raised point (for example, <b>£21·48</b>) and to use a decimal point "on the line" only when typesetting constraints made it unavoidable. This usage, however, has been declining since the mid-1970s, as the importation of electronic typewriters, <a href="/wiki/Calculator" title="Calculator">calculators</a> and <a href="/wiki/Computer" title="Computer">computers</a> from the <a href="/wiki/United_States" title="United States">United States</a> and <a href="/wiki/Japan" title="Japan">Japan</a> familiarised Britons with using <a href="/wiki/Full_stop" title="Full stop">full stops</a> and made the space dot harder to typeset.<sup class="noprint Inline-Template Template-Fact" style="white-space:nowrap;">[<i><a href="/wiki/Wikipedia:Citation_needed" title="Wikipedia:Citation needed"><span title="This claim needs references to reliable sources. (October 2014)">citation needed</span></a></i>]</sup>. The space dot may still be used frequently in handwriting, however.</p>
<p>In the <a href="/wiki/Shavian_alphabet" title="Shavian alphabet">Shavian alphabet</a>, interpuncts replace <a href="/wiki/Capitalization" title="Capitalization">capitalization</a> as the marker of proper nouns. The dot is placed at the beginning of a word.</p>

</div>
<p>The <b>flown dot</b> (<a href="/wiki/Catalan_language" title="Catalan language">Catalan</a>: <span lang="ca" xml:lang="ca"><i>punt volat</i></span>) is used in <a href="/wiki/Catalan_language" title="Catalan language">Catalan</a> between two <a href="/wiki/L" title="L">Ls</a> in cases where each belongs to a separate syllable, for example <span lang="ca" xml:lang="ca"><i>cel·la</i></span>, "cell". This distinguishes such "<a class="mw-redirect" href="/wiki/Geminate" title="Geminate">geminate</a> Ls" (<span lang="ca" xml:lang="ca"><i>ela geminada</i></span>), which are pronounced <span class="IPA" title="Representation in the International Phonetic Alphabet (IPA)">[<span class="IPA" title="Representation in the International Phonetic Alphabet (IPA)"><a class="mw-redirect" href="/wiki/Alveolar_lateral_approximant#Velarized_alveolar_lateral_approximant" title="Alveolar lateral approximant">ɫ</a></span><span class="IPA" title="Representation in the International Phonetic Alphabet (IPA)"><a href="/wiki/Length_(phonetics)" title="Length (phonetics)">ː</a></span>]</span>, from "double L" (<span lang="ca" xml:lang="ca"><i>doble ela</i></span>), which are written without the flown dot and are pronounced <span class="IPA" title="Representation in the International Phonetic Alphabet (IPA)">[<a href="/wiki/Palatal_lateral_approximant" title="Palatal lateral approximant">ʎ</a>]</span>. In situations where the flown dot is unavailable, <a class="mw-redirect" href="/wiki/Period_mark" title="Period mark">periods</a> (as in <span lang="ca" xml:lang="ca"><i>col.lecció</i></span>) or <a href="/wiki/Hyphen" title="Hyphen">hyphens</a> (as in <span lang="ca" xml:lang="ca"><i>col-lecció</i></span>) are frequently used as substitutes, but this is tolerated rather than encouraged.</p>
<p>Historically, medieval <a href="/wiki/Catalan_language" title="Catalan language">Catalan</a> also used the symbol ⟨<b>·</b>⟩ as a marker for certain <a href="/wiki/Elision" title="Elision">elisions</a>, much like the modern apostrophe (see <a href="#Occitan">Occitan</a> below), and <a href="/wiki/Syllabification" title="Syllabification">hyphenations</a>.</p>
<p>There is no separate keyboard layout for Catalan: the flown dot can be typed using Shift-3 in the <a href="/wiki/QWERTY#Spanish" title="QWERTY">Spanish (Spain) layout</a>. It appears in <a href="/wiki/Unicode" title="Unicode">Unicode</a> as the letters ⟨<b>Ŀ</b>⟩ (U+013F) and ⟨<b>ŀ</b>⟩ (U+0140), but they are <a href="/wiki/Unicode_compatibility_characters" title="Unicode compatibility characters">compatibility characters</a> and are not frequently used nor recommended.<sup class="reference" id="cite_ref-4"><a href="#cite_note-4">[4]</a></sup> Similarly, the larger bullet (⟨<b>•</b>⟩, U+2022, <code>&amp;bull;</code>) may be seen but is discouraged on aesthetic grounds. The preferred Unicode representation is ⟨<b>l·</b>⟩ (U+006C + U+00B7).</p>
<h3><span class="mw-headline" id="Chinese">Chinese</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Interpunct&amp;action=edit&amp;section=4" title="Edit section: Chinese">edit</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>The <b>partition sign</b> (Unicode code point U+2027, Hyphenation Point) is used in <a href="/wiki/Chinese_language" title="Chinese language">Chinese</a> (which generally lacks spacing between <a class="mw-redirect" href="/wiki/Chinese_character" title="Chinese character">characters</a>) to mark divisions in <a class="mw-redirect" href="/wiki/Transliteration_into_Chinese_characters" title="Transliteration into Chinese characters">transliterated</a> foreign words, particularly names. This is properly (and on <a href="/wiki/Taiwan" title="Taiwan">Taiwan</a> formally<sup class="reference" id="cite_ref-5"><a href="#cite_note-5">[5]</a></sup>) a full-width punctuation mark, although sometimes narrower forms are substituted for aesthetic reasons. In particular, the regular interpunct is more commonly used as a computer input, although Chinese-language fonts typically render this as full width. When the Chinese text is <a href="/wiki/Romanization_of_Chinese" title="Romanization of Chinese">romanized</a>, the partition sign is simply replaced by a standard space or other appropriate punctuation. Thus, <a href="/wiki/William_Shakespeare" title="William Shakespeare">William Shakespeare</a> is signified as <span lang="zh" xml:lang="zh">威廉·莎士比亞</span> or <span lang="zh" xml:lang="zh"><span class="nowrap">威廉·</span><span class="nowrap">莎士比亚</span></span> (<small><a href="/wiki/Pinyin" title="Pinyin">p</a></small> <i>Wēilián Shāshìbǐyà</i>), <a href="/wiki/George_W._Bush" title="George W. Bush">George W. Bush</a> as <span lang="zh" xml:lang="zh"><span class="nowrap">喬治·</span>W·<span class="nowrap">布殊</span></span> or <span lang="zh" xml:lang="zh"><span class="nowrap">乔治·</span>W·<span class="nowrap">布什</span></span> (<small><a href="/wiki/Pinyin" title="Pinyin">p</a></small> <i>Qiáozhì W. Bùshí</i>), and the full name of the prophet <a href="/wiki/Muhammad" title="Muhammad">Muhammad</a> as <span lang="zh" xml:lang="zh"><span class="nowrap">阿布</span>·<span class="nowrap">卡西木</span>·<span class="nowrap">穆罕默德</span>·本·<span class="nowrap">阿布杜拉</span>·本·<span class="nowrap">阿布杜勒-</span><span class="nowrap">穆塔利卜</span>·本·<span class="nowrap">哈希姆</span></span> (<small><a href="/wiki/Pinyin" title="Pinyin">p</a></small> <i>Ābù Kǎxīmù Mùhǎnmòdé Běn Ābùdùlā Běn Ābùdùlè-Mùtǎlìbǔ Běn Hāxīmǔ</i>). Titles and other translated words are not similarly marked: <a href="/wiki/Genghis_Khan" title="Genghis Khan">Genghis Khan</a> and <a href="/wiki/Elizabeth_II" title="Elizabeth II">Elizabeth II</a> are simply <span lang="zh" xml:lang="zh"><span class="nowrap">成吉思汗</span></span> and <span lang="zh" xml:lang="zh"><span class="nowrap">伊利沙伯</span><span class="nowrap">二世</span></span> or <span lang="zh" xml:lang="zh"><span class="nowrap">伊麗莎白</span><span class="nowrap">二世</span></span> without a partition sign.</p>
<p>The partition sign is also used to separate book and chapter titles when they are mentioned consecutively: book first and then chapter. It also appears in Swedish <a class="mw-redirect" href="/wiki/Sinologist" title="Sinologist">sinologist</a> <a href="/wiki/Bernhard_Karlgren" title="Bernhard Karlgren">Bernhard Karlgren</a>'s works, where an interpunct is used to represent the <a href="/wiki/Glottal_stop" title="Glottal stop">glottal stop</a> in his reconstruction of medieval <a href="/wiki/Chinese_language" title="Chinese language">Chinese</a>.</p>

<h3><span class="mw-headline" id="Minnan">Minnan</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Interpunct&amp;action=edit&amp;section=6" title="Edit section: Minnan">edit</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>In <a href="/wiki/Pe%CC%8Dh-%C5%8De-j%C4%AB" title="Pe̍h-ōe-jī">Pe̍h-ōe-jī</a> for <a href="/wiki/Taiwanese_Hokkien" title="Taiwanese Hokkien">Taiwanese Hokkien</a>, middle dot is often used as a workaround for <i>dot above right</i> <a href="/wiki/Diacritic" title="Diacritic">diacritic</a> because most early encoding systems did not support this diacritic. This is now encoded as <span class="nowrap"><span style="font-family: monospace, monospace;">U+0358</span></span> <span style="font-size:125%">͘</span> <span class="smallcaps" style="font-variant: small-caps; font-size: smaller;">COMBINING DOT ABOVE RIGHT</span> (see <a href="/wiki/O%CD%98" title="O͘">o͘</a>). Unicode did not support this diacritic until June 2004. Newer fonts often support it natively; however, the practice of using middle dot still exists. Historically, it was derived in the late 19th century from an older barred-o with curly tail as an adaptation to the typewriter.</p>
<h3><span class="mw-headline" id="Ethiopic">Ethiopic</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Interpunct&amp;action=edit&amp;section=7" title="Edit section: Ethiopic">edit</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>The <a href="/wiki/Ge%27ez_language" title="Ge'ez language">Ge'ez</a> (Ethiopic) language uses an interpunct of two vertically aligned dots, like a colon, but with larger dots. An example is <span class="Ethiopic" style="font-family: 'Abyssinica SIL', 'Noto Sans Ethiopic', 'Ebrima';"><b>ገድለ፡ወለተ፡ጴጥሮስ</b></span>.</p>
<h3><span class="mw-headline" id="Franco-Proven.C3.A7al">Franco-Provençal</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Interpunct&amp;action=edit&amp;section=8" title="Edit section: Franco-Provençal">edit</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>In <a href="/wiki/Franco-Proven%C3%A7al_language" title="Franco-Provençal language">Franco-Provençal</a> (or Arpitan), the interpunct is used in order to distinguish the following graphemes:</p>
<ul>
<li><b>ch·</b>, pronounced <span class="IPA" title="Representation in the International Phonetic Alphabet (IPA)">[ʃ]</span>, versus <b>ch</b>, pronounced <span class="IPA" title="Representation in the International Phonetic Alphabet (IPA)">[ts]</span></li>
<li><b>j·</b>, pronounced <span class="IPA" title="Representation in the International Phonetic Alphabet (IPA)">[ʒ]</span>, versus <b>j</b>, pronounced <span class="IPA" title="Representation in the International Phonetic Alphabet (IPA)">[dz]</span></li>
<li><b>g·</b> before <i>e, i,</i> pronounced <span class="IPA" title="Representation in the International Phonetic Alphabet (IPA)">[ʒ]</span>, versus <b>g</b> before <i>e, i,</i> pronounced <span class="IPA" title="Representation in the International Phonetic Alphabet (IPA)">[dz]</span></li>
</ul>
<p><span id="Ano_teleia"></span><span id="Greek_semicolon"></span></p>
<h3><span class="mw-headline" id="Greek">Greek</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Interpunct&amp;action=edit&amp;section=9" title="Edit section: Greek">edit</a><span class="mw-editsection-bracket">]</span></span></h3>
<p><a href="/wiki/Ancient_Greek" title="Ancient Greek">Ancient Greek</a> did not have spacing or interpuncts but instead ran all the letters together. By <a href="/wiki/Late_Antiquity" title="Late Antiquity">Late Antiquity</a>, various marks were used to separate words, particularly the <a class="mw-redirect" href="/wiki/Greek_comma" title="Greek comma">Greek comma</a>.<sup class="reference" id="cite_ref-tlg_6-0"><a href="#cite_note-tlg-6">[6]</a></sup></p>
<p>The <a href="/wiki/Greek_language" title="Greek language">modern Greek</a> <b>ano teleia</b> mark (<span lang="el" xml:lang="el">άνω τελεία</span>, <i>ánō teleía</i>, lit. "upper stop"), also known as the <i>áno stigmī́</i> (<span lang="el" xml:lang="el">άνω στιγμή</span>), is the infrequently-encountered <b>Greek <a href="/wiki/Semicolon" title="Semicolon">semicolon</a></b> and is properly <a href="/wiki/Romanization_of_Greek" title="Romanization of Greek">romanized</a> as such.<sup class="reference" id="cite_ref-elot_7-0"><a href="#cite_note-elot-7">[7]</a></sup> It is also used to introduce lists in the manner of an English <a href="/wiki/Colon_(punctuation)" title="Colon (punctuation)">colon</a>.<sup class="reference" id="cite_ref-tlg_6-1"><a href="#cite_note-tlg-6">[6]</a></sup> In Greek text, <a href="/wiki/Unicode" title="Unicode">Unicode</a> provides a unique code point—<span class="nowrap"><span style="font-family: monospace, monospace;">U+0387</span></span> <span style="font-size:125%">·</span> <span class="smallcaps" style="font-variant: small-caps; font-size: smaller;">GREEK ANO TELEIA</span><sup class="reference" id="cite_ref-8"><a href="#cite_note-8">[8]</a></sup>—but it is also expressed as an interpunct. In practice, the separate code point for ano teleia canonically decomposes to the interpunct.<sup class="reference" id="cite_ref-tlg_6-2"><a href="#cite_note-tlg-6">[6]</a></sup></p>
<p>The Hellenistic scholars of <a href="/wiki/Alexandria" title="Alexandria">Alexandria</a> first developed the mark for a function closer to the <a href="/wiki/Comma" title="Comma">comma</a>, before it fell out of use and was then repurposed for its present role.<sup class="reference" id="cite_ref-tlg_6-3"><a href="#cite_note-tlg-6">[6]</a></sup></p>
<h3><span class="mw-headline" id="Japanese">Japanese</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Interpunct&amp;action=edit&amp;section=10" title="Edit section: Japanese">edit</a><span class="mw-editsection-bracket">]</span></span></h3>
<div class="hatnote" role="note">Main article: <a href="/wiki/Japanese_punctuation" title="Japanese punctuation">Japanese punctuation</a></div>
<p>Interpuncts are often used to separate transcribed foreign words written in <a href="/wiki/Katakana" title="Katakana">katakana</a>. For example, "<a href="/wiki/Can%27t_Buy_Me_Love" title="Can't Buy Me Love">Can't Buy Me Love</a>" becomes <span class="t_nihongo_kanji" lang="ja" xml:lang="ja">「キャント・バイ・ミー・ラヴ」</span> (<i>Kyanto·bai·mī·ravu</i>). A middle dot is also sometimes used to separate lists in <a href="/wiki/Japanese_language" title="Japanese language">Japanese</a> instead of the Japanese comma ("、" known as <i>tōten</i>). Dictionaries and <a href="/wiki/Grammar" title="Grammar">grammar</a> lessons in Japanese sometimes also use a similar symbol to separate a <a href="/wiki/Verb" title="Verb">verb</a> <a href="/wiki/Affix" title="Affix">suffix</a> from its root. Note that while some fonts may render the Japanese middle dot as a square under great magnification, this is not a defining property of the middle dot that is used in China or Japan.</p>
<p>However, the Japanese writing system usually does not use space or punctuation to separate words (though the mixing of <a href="/wiki/Katakana" title="Katakana">katakana</a>, <a href="/wiki/Kanji" title="Kanji">kanji</a>, and <a href="/wiki/Hiragana" title="Hiragana">hiragana</a> gives some indication of word boundary).</p>
<p>In Japanese <a href="/wiki/Typography" title="Typography">typography</a>, there exist two Unicode code points:</p>
<ul>
<li><span class="nowrap"><span style="font-family: monospace, monospace;">U+30FB</span></span> <span style="font-size:125%">・</span> <span class="smallcaps" style="font-variant: small-caps; font-size: smaller;">KATAKANA MIDDLE DOT</span>, with a fixed width that is the same as most kana characters, known as <i>fullwidth</i>.</li>
<li><span class="nowrap"><span style="font-family: monospace, monospace;">U+FF65</span></span> <span style="font-size:125%">･</span> <span class="smallcaps" style="font-variant: small-caps; font-size: smaller;">HALFWIDTH KATAKANA MIDDLE DOT</span></li>
</ul>
<p>The interpunct also has a number of other uses in Japanese, including the following: to separate titles, names and positions: 課長補佐・鈴木 (Assistant Section Head · Suzuki); as a decimal point when writing numbers in kanji: <span class="nowrap">三・一四一五九二 (3.141 592)</span>; and in place of hyphens, dashes and colons when writing vertically.</p>
<h3><span class="mw-headline" id="Korean">Korean</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Interpunct&amp;action=edit&amp;section=11" title="Edit section: Korean">edit</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>Interpuncts are used in written Korean to denote a list of two or more words, more or less in the same way a <a href="/wiki/Slash_(punctuation)" title="Slash (punctuation)">slash (/)</a> is used to juxtapose words in many other languages. In this role it also functions in a similar way to the <a href="/wiki/English_language" title="English language">English</a> <a class="mw-redirect" href="/wiki/En_dash" title="En dash">en dash</a>, as in 미·소관계, "American–Soviet relations". The use of interpuncts has declined in years of digital typography and especially in place of slashes, but, in the strictest sense, a slash cannot replace a middle dot in Korean typography.</p>
<p><span class="nowrap"><span style="font-family: monospace, monospace;">U+318D</span></span> <span style="font-size:125%">ㆍ</span> <span class="smallcaps" style="font-variant: small-caps; font-size: smaller;">HANGUL LETTER ARAEA</span> (<span lang="ko" xml:lang="ko">아래아</span>) is used more than a middle dot when an interpunct is to be used in Korean typography, though <i>araea</i> is not a punctuation symbol but an obsolete Hangul Jamo, because <i>araea</i> is full-width letter so that it looks better than middle dot between Hangul and it is drawn like middle dot in Windows default Korean font such as <i>Batang</i>.</p>
<h3><span class="mw-headline" id="Latin">Latin</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Interpunct&amp;action=edit&amp;section=12" title="Edit section: Latin">edit</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>The interpunct (<span lang="la" xml:lang="la"><span class="smallcaps"><span style="font-variant: small-caps; text-transform: lowercase;">INTERPVNCTVS</span></span></span>) was regularly used in <a href="/wiki/Classical_Latin" title="Classical Latin">classical Latin</a> to separate words. In addition to the most common round form, <a class="mw-redirect" href="/wiki/Inscription" title="Inscription">inscriptions</a> sometimes use a small <a href="/wiki/Equilateral_triangle" title="Equilateral triangle">equilateral triangle</a> for the interpunct, pointing either up or down. It may also appear as a mid-line comma, similar to the <a href="#Greek">Greek</a> practice of the time. The interpunct fell out of use c.<span style="white-space:nowrap;"> <span class="smallcaps"><span style="font-variant: small-caps; text-transform: lowercase;">AD</span></span></span> 200, and <a class="mw-redirect" href="/wiki/Latin_language" title="Latin language">Latin</a> was then written <i><a class="mw-redirect" href="/wiki/Scripta_continua" title="Scripta continua">scripta continua</a></i> for several centuries.<sup class="noprint Inline-Template Template-Fact" style="white-space:nowrap;">[<i><a href="/wiki/Wikipedia:Citation_needed" title="Wikipedia:Citation needed"><span title="This claim needs references to reliable sources. (March 2015)">citation needed</span></a></i>]</sup></p>
<h3><span class="mw-headline" id="Occitan">Occitan</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Interpunct&amp;action=edit&amp;section=13" title="Edit section: Occitan">edit</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>In <a href="/wiki/Occitan_language" title="Occitan language">Occitan</a>, especially in the <a href="/wiki/Gascon_language" title="Gascon language">Gascon</a> dialect, the interpunct (<i>punt interior</i>, literally, "inner dot", or <i>ponch naut</i> for "high / upper point") is used to distinguish the following graphemes:</p>
<ul>
<li><b>s·h</b>, pronounced <span class="IPA" title="Representation in the International Phonetic Alphabet (IPA)">[s.h]</span>, versus <b>sh</b>, pronounced <span class="IPA" title="Representation in the International Phonetic Alphabet (IPA)">[ʃ]</span>, for example, in <i>des·har</i> 'to undo' vs <i>deishar</i> 'to leave'</li>
<li><b>n·h</b>, pronounced <span class="IPA" title="Representation in the International Phonetic Alphabet (IPA)">[n.h]</span>, versus <b>nh</b>, pronounced <span class="IPA" title="Representation in the International Phonetic Alphabet (IPA)">[ɲ]</span>, for example in <i>in·hèrn</i> 'hell' vs <i>vinha</i> 'vineyard'</li>
</ul>
<p>Although it is considered to be a spelling error, a <a href="/wiki/Full_stop" title="Full stop">period</a> is frequently used when a middle dot is unavailable: <i>des.har, in.hèrn</i>, which is the case for French <a href="/wiki/Keyboard_layout#French" title="Keyboard layout">keyboard layout</a>.</p>
<p>In <a href="/wiki/Old_Occitan" title="Old Occitan">Old Occitan</a>, the symbol <b>·</b> was sometimes used to denote certain <a href="/wiki/Elision" title="Elision">elisions</a>, much like the modern apostrophe, the only difference being that the word that gets to be elided is always placed after the interpunct, the word before ending either in a vowel sound or the letter <i>n</i>:</p>
<ul>
<li><b>que·l</b> (<i>que lo</i>, that the) versus <b>qu'el</b> (that he)</li>
<li>From <a href="/wiki/Bertran_de_Born" title="Bertran de Born">Bertran de Born</a>'s <i>Ab joi mou lo vers e·l comens</i> (translated by James H. Donalson):</li>
</ul>
<table>
<tr>
<td>
<p><i>Bela Domna·l vostre cors gens</i><br/>
<i>E·lh vostre bel olh m'an conquis,</i><br/>
<i>E·l doutz esgartz e lo clars vis,</i><br/>
<i>E·l vostre bels essenhamens,</i><br/>
<i>Que, can be m'en pren esmansa,</i><br/>
<i>De beutat no·us trob egansa:</i><br/>
<i>La genser etz c'om posc'e·l mon chauzir,</i><br/>
<i>O no·i vei clar dels olhs ab que·us remir.</i></p>
</td>
<td></td>
<td>
<p><i>Domna·l</i> <span class="IPA" title="Representation in the International Phonetic Alphabet (IPA)">[ˈdonnal]</span> = <i>Domna, lo</i> ("Lady, the": singular <a class="mw-redirect" href="/wiki/Definite_article#Definite_article" title="Definite article">definite article</a>)<br/>
<i>E·lh</i> <span class="IPA" title="Representation in the International Phonetic Alphabet (IPA)">[eʎ]</span> = <i>E li</i> ("And the": plural definite article)<br/>
<i>E·l</i> <span class="IPA" title="Representation in the International Phonetic Alphabet (IPA)">[el]</span> = <i>E lo</i> ("And the")<br/>
<i>E·l</i> = <i>E lo</i> ("And the")<br/>
<br/>

<h3><span class="mw-headline" id="Old_Irish">Old Irish</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Interpunct&amp;action=edit&amp;section=14" title="Edit section: Old Irish">edit</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>In many linguistic works discussing <a href="/wiki/Old_Irish" title="Old Irish">Old Irish</a> (but not in actual Old Irish manuscripts), the interpunct is used to separate a pretonic preverbal element from the stressed syllable of the verb, e.g. <span lang="sga" xml:lang="sga"><i>do·beir</i></span> "gives". It is also used in citing the verb forms used after such preverbal elements (the <a class="mw-redirect" href="/wiki/Prototonic" title="Prototonic">prototonic</a> forms), e.g. <span lang="sga" xml:lang="sga"><i>·beir</i></span> "carries", to distinguish them from forms used without preverbs, e.g. <span lang="sga" xml:lang="sga"><i>beirid</i></span> "carries".<sup class="reference" id="cite_ref-9"><a href="#cite_note-9">[9]</a></sup> In other works, the <a href="/wiki/Hyphen" title="Hyphen">hyphen</a> (<span lang="sga" xml:lang="sga"><i>do-beir</i>, <i>-beir</i></span>) or <a href="/wiki/Colon_(punctuation)" title="Colon (punctuation)">colon</a> (<span lang="sga" xml:lang="sga"><i>do:beir</i>, <i>:beir</i></span>) may be used for this purpose.</p>
<h3><span class="mw-headline" id="Runes">Runes</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Interpunct&amp;action=edit&amp;section=15" title="Edit section: Runes">edit</a><span class="mw-editsection-bracket">]</span></span></h3>
<p><a class="mw-redirect" href="/wiki/Runic" title="Runic">Runic</a> texts use either an interpunct-like or a colon-like punctuation mark to separate words. There are two <a href="/wiki/Unicode" title="Unicode">Unicode</a> characters dedicated for this: <span class="nowrap"><span style="font-family: monospace, monospace;">U+16EB</span></span> <span style="font-size:125%">᛫</span> <span class="smallcaps" style="font-variant: small-caps; font-size: smaller;">RUNIC SINGLE PUNCTUATION</span> and <span class="nowrap"><span style="font-family: monospace, monospace;">U+16EC</span></span> <span style="font-size:125%">᛬</span> <span class="smallcaps" style="font-variant: small-caps; font-size: smaller;">RUNIC MULTIPLE PUNCTUATION</span>.</p>
<h2><span class="mw-headline" id="In_mathematics_and_science">In mathematics and science</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Interpunct&amp;action=edit&amp;section=16" title="Edit section: In mathematics and science">edit</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>In <a href="/wiki/International_System_of_Units" title="International System of Units">SI</a> units the middle dot or <a href="/wiki/Non-breaking_space" title="Non-breaking space">non-breaking space</a> is used as a multiplication sign. Only a <a href="/wiki/Comma" title="Comma">comma</a> or <a href="/wiki/Full_stop" title="Full stop">full stop (period)</a> may be used as a decimal marker.</p>
<p>In <a href="/wiki/Mathematics" title="Mathematics">mathematics</a>, a small middle dot can be used to represent <a href="/wiki/Multiplication" title="Multiplication">product</a>; for example, <i>x</i> ∙ <i>y</i> for the product of <i>x</i> and <i>y</i>. When dealing with <a href="/wiki/Scalar_(mathematics)" title="Scalar (mathematics)">scalars</a>, it is interchangeable with the <a href="/wiki/Multiplication_sign" title="Multiplication sign">multiplication sign</a>: <i>x</i> ⋅ <i>y</i> means the same thing as <i>x</i> <a class="mw-redirect" href="/wiki/%C3%97" title="×">×</a> <i>y</i>, but <a class="mw-redirect" href="/wiki/%C3%97" title="×">×</a> is easily confused with the letter <i><a href="/wiki/X" title="X">x</a></i>. However, when dealing with <a class="mw-redirect" href="/wiki/Vector_(geometry)" title="Vector (geometry)">vectors</a>, the <a href="/wiki/Dot_product" title="Dot product">dot product</a> is distinct from the <a href="/wiki/Cross_product" title="Cross product">cross product</a>. This usage has its own designated code point in Unicode, U+2219 (∙), called the "bullet operator".<sup class="noprint Inline-Template Template-Fact" style="white-space:nowrap;">[<i><a href="/wiki/Wikipedia:Citation_needed" title="Wikipedia:Citation needed"><span title="This claim needs references to reliable sources. (May 2011)">citation needed</span></a></i>]</sup> It is also sometimes used to denote the <a href="/wiki/Logical_conjunction" title="Logical conjunction">“AND” relationship</a> in formal <a href="/wiki/Logic" title="Logic">logic</a>, due to the relationship between these two operations. In situations where the interpunct is used as a decimal point (as noted above, by many mathematics teachers in some countries<sup class="noprint Inline-Template" style="margin-left:0.1em; white-space:nowrap;">[<i><a class="mw-redirect" href="/wiki/Wikipedia:Avoid_weasel_words" title="Wikipedia:Avoid weasel words"><span title="Who says this? (November 2011)">weasel words</span></a></i>]</sup>), then the multiplication sign used is usually a full stop (period), not an interpunct.</p>
<p>In <a href="/wiki/Computing" title="Computing">computing</a>, the middle dot is usually used to indicate <a href="/wiki/Whitespace_character" title="Whitespace character">white space</a> in various software applications such as <a class="mw-redirect" href="/wiki/Word_processing" title="Word processing">word processing</a>, <a href="/wiki/Graphic_design" title="Graphic design">graphic design</a>, web layout, <a href="/wiki/Desktop_publishing" title="Desktop publishing">desktop publishing</a> or <a href="/wiki/Software_development" title="Software development">software development</a> programs. In some <a href="/wiki/Word_processor" title="Word processor">word processors</a>, interpuncts are used to denote not only <a href="/wiki/Hard_space" title="Hard space">hard space</a> or space characters, but also sometimes used to indicate a space when put in paragraph format to show indentations and spaces. This allows the user to see where white space is located in the document and what sizes of white space are used, since normally white space is invisible so tabs, spaces, non-breaking spaces and such are indistinguishable from one another.</p>
<p>In <a href="/wiki/Chemistry" title="Chemistry">chemistry</a>, the middle dot is used to separate the parts of formulas of <a class="mw-redirect" href="/wiki/Addition_compound" title="Addition compound">addition compounds</a>, mixture salts or solvates (mostly hydrates), such as of <a href="/wiki/Copper(II)_sulfate" title="Copper(II) sulfate">copper(II) sulphate pentahydrate</a>, CuSO<sub>4</sub> · 5H<sub>2</sub>O.</p>
