<p><b>3GP</b> (3GPP file format) is a <a href="/wiki/Multimedia" title="Multimedia">multimedia</a> <a class="mw-redirect" href="/wiki/Container_format_(digital)" title="Container format (digital)">container format</a> defined by the <a href="/wiki/3GPP" title="3GPP">Third Generation Partnership Project (3GPP)</a> for <a href="/wiki/3G" title="3G">3G</a> <a class="mw-redirect" href="/wiki/UMTS" title="UMTS">UMTS</a> multimedia services. It is used on 3G <a href="/wiki/Mobile_phone" title="Mobile phone">mobile phones</a> but can also be played on some <a href="/wiki/2G" title="2G">2G</a> and <a href="/wiki/4G" title="4G">4G</a> phones.</p>
<p><b>3G2</b> (3GPP2 file format) is a multimedia container format defined by the <a class="mw-redirect" href="/wiki/3GPP2" title="3GPP2">3GPP2</a> for 3G <a href="/wiki/CDMA2000" title="CDMA2000">CDMA2000</a> multimedia services. It is very similar to the 3GP file format, but has some extensions and limitations in comparison to 3GP.</p>

<h2><span class="mw-headline" id="Specifications">Specifications</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=3GP_and_3G2&amp;action=edit&amp;section=1" title="Edit section: Specifications">edit</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>3GP is defined in the <a href="/wiki/ETSI" title="ETSI">ETSI</a> <a href="/wiki/3GPP" title="3GPP">3GPP</a> technical specification.<sup class="reference" id="cite_ref-3gp-specification_1-0"><a href="#cite_note-3gp-specification-1">[1]</a></sup> 3GP is a required file format for video and associated speech/audio media types and timed text in ETSI 3GPP technical specifications for <a href="/wiki/IP_Multimedia_Subsystem" title="IP Multimedia Subsystem">IP Multimedia Subsystem</a> (IMS), <a href="/wiki/Multimedia_Messaging_Service" title="Multimedia Messaging Service">Multimedia Messaging Service</a> (MMS), <a href="/wiki/Multimedia_Broadcast_Multicast_Service" title="Multimedia Broadcast Multicast Service">Multimedia Broadcast/Multicast Service</a> (MBMS) and Transparent end-to-end Packet-switched Streaming Service (<a class="new" href="/w/index.php?title=3GPP_PSS&amp;action=edit&amp;redlink=1" title="3GPP PSS (page does not exist)">PSS</a>).<sup class="reference" id="cite_ref-2"><a href="#cite_note-2">[2]</a></sup><sup class="reference" id="cite_ref-3"><a href="#cite_note-3">[3]</a></sup><sup class="reference" id="cite_ref-3gpp-26346_4-0"><a href="#cite_note-3gpp-26346-4">[4]</a></sup><sup class="reference" id="cite_ref-5"><a href="#cite_note-5">[5]</a></sup></p>
<p>3G2 is defined in the <a class="mw-redirect" href="/wiki/3GPP2" title="3GPP2">3GPP2</a> technical specification.<sup class="reference" id="cite_ref-3gpp2_6-0"><a href="#cite_note-3gpp2-6">[6]</a></sup></p>
<h2><span class="mw-headline" id="Technical_details">Technical details</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=3GP_and_3G2&amp;action=edit&amp;section=2" title="Edit section: Technical details">edit</a><span class="mw-editsection-bracket">]</span></span></h2>
<div class="thumb tright">
<div class="thumbinner" style="width:252px;"><a class="image" href="/wiki/File:Relations_between_ISO_MP4_3GPP_and_3GPP2_file_format.svg"><img alt="" class="thumbimage" data-file-height="368" data-file-width="504" height="183" src="//upload.wikimedia.org/wikipedia/commons/thumb/b/b7/Relations_between_ISO_MP4_3GPP_and_3GPP2_file_format.svg/250px-Relations_between_ISO_MP4_3GPP_and_3GPP2_file_format.svg.png" srcset="//upload.wikimedia.org/wikipedia/commons/thumb/b/b7/Relations_between_ISO_MP4_3GPP_and_3GPP2_file_format.svg/375px-Relations_between_ISO_MP4_3GPP_and_3GPP2_file_format.svg.png 1.5x, //upload.wikimedia.org/wikipedia/commons/thumb/b/b7/Relations_between_ISO_MP4_3GPP_and_3GPP2_file_format.svg/500px-Relations_between_ISO_MP4_3GPP_and_3GPP2_file_format.svg.png 2x" width="250"/></a>
<div class="thumbcaption">
<div class="magnify"><a class="internal" href="/wiki/File:Relations_between_ISO_MP4_3GPP_and_3GPP2_file_format.svg" title="Enlarge"></a></div>
Relations between ISO Base Media File Format, MP4 File Format, 3GPP file format and 3GPP2 file format. Based on the 3GPP2 technical specification published on 18 May 2007.<sup class="reference" id="cite_ref-3gpp2-vs-3gpp_7-0"><a href="#cite_note-3gpp2-vs-3gpp-7">[7]</a></sup></div>
</div>
</div>
<p>The 3GP and 3G2 file formats are both structurally based on the <a href="/wiki/ISO_base_media_file_format" title="ISO base media file format">ISO base media file format</a> defined in ISO/<a href="/wiki/International_Electrotechnical_Commission" title="International Electrotechnical Commission">IEC</a> 14496-12 - <a href="/wiki/MPEG-4" title="MPEG-4">MPEG-4</a> Part 12,<sup class="reference" id="cite_ref-3gp-spec-mpeg4part12_8-0"><a href="#cite_note-3gp-spec-mpeg4part12-8">[8]</a></sup><sup class="reference" id="cite_ref-mpeg4part12_9-0"><a href="#cite_note-mpeg4part12-9">[9]</a></sup><sup class="reference" id="cite_ref-mpeg4part12--new_10-0"><a href="#cite_note-mpeg4part12--new-10">[10]</a></sup> but older versions of the 3GP file format did not use some of its features.<sup class="reference" id="cite_ref-3gpp2-vs-3gpp_7-1"><a href="#cite_note-3gpp2-vs-3gpp-7">[7]</a></sup> 3GP and 3G2 are container formats similar to <a href="/wiki/MPEG-4_Part_14" title="MPEG-4 Part 14">MPEG-4 Part 14</a> (MP4), which is also based on MPEG-4 Part 12. The 3GP and 3G2 file format were designed to decrease storage and bandwidth requirements to accommodate <a href="/wiki/Mobile_phone" title="Mobile phone">mobile phones</a>.</p>
<p>3GP and 3G2 are similar standards, but with some differences:</p>
<ul>
<li><a href="/wiki/3GPP" title="3GPP">3GPP</a> file format was designed for <a href="/wiki/GSM" title="GSM">GSM</a>-based phones and may have the filename extension <tt>.3gp</tt></li>
<li><a class="mw-redirect" href="/wiki/3GPP2" title="3GPP2">3GPP2</a> file format was designed for <a href="/wiki/CDMA2000" title="CDMA2000">CDMA</a>-based phones and may have the filename extension <tt>.3g2</tt></li>
</ul>
<p>Some cell phones use the .mp4 extension for 3GP video.</p>
<h3><span class="mw-headline" id="3GP">3GP</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=3GP_and_3G2&amp;action=edit&amp;section=3" title="Edit section: 3GP">edit</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>The 3GP file format stores video streams as <a href="/wiki/MPEG-4_Part_2" title="MPEG-4 Part 2">MPEG-4 Part 2</a> or <a href="/wiki/H.263" title="H.263">H.263</a> or <a class="mw-redirect" href="/wiki/H.264" title="H.264">MPEG-4 Part 10 (AVC/H.264)</a>, and audio streams as <a class="mw-redirect" href="/wiki/Adaptive_Multi-Rate" title="Adaptive Multi-Rate">AMR-NB</a>, <a class="mw-redirect" href="/wiki/AMR-WB" title="AMR-WB">AMR-WB</a>, <a class="mw-redirect" href="/wiki/AMR-WB%2B" title="AMR-WB+">AMR-WB+</a>, <a href="/wiki/Advanced_Audio_Coding" title="Advanced Audio Coding">AAC-LC</a>, <a class="mw-redirect" href="/wiki/HE-AAC" title="HE-AAC">HE-AAC</a> v1 or Enhanced aacPlus (HE-AAC v2). 3GPP allowed use of AMR and H.263 codecs in the ISO base media file format (MPEG-4 Part 12), because 3GPP specified the usage of the Sample Entry and template fields in the ISO base media file format as well as defining new boxes to which codecs refer. These extensions were registered by the registration authority for code-points in ISO base media file format ("MP4 Family" files).<sup class="reference" id="cite_ref-mpeg4part12-3gp_11-0"><a href="#cite_note-mpeg4part12-3gp-11">[11]</a></sup> <sup class="reference" id="cite_ref-mp4ra_12-0"><a href="#cite_note-mp4ra-12">[12]</a></sup> For the storage of MPEG-4 media specific information in 3GP files, the 3GP specification refers to MP4 and the AVC file format, which are also based on the ISO base media file format. The MP4 and the AVC file format specifications described usage of MPEG-4 content in the ISO base media file format.<sup class="reference" id="cite_ref-3gp-spec-mpeg4part12_8-1"><a href="#cite_note-3gp-spec-mpeg4part12-8">[8]</a></sup></p>
<p>A 3GP file is always <a href="/wiki/Endianness" title="Endianness">big-endian</a>, storing and transferring the most significant bytes first.</p>
<h3><span class="mw-headline" id="3G2">3G2</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=3GP_and_3G2&amp;action=edit&amp;section=4" title="Edit section: 3G2">edit</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>The 3G2 file format can store the same video streams and most of the audio streams used in the 3GP file format. In addition, 3G2 stores audio streams as <a class="mw-redirect" href="/wiki/EVRC" title="EVRC">EVRC</a>, <a class="mw-redirect" href="/wiki/EVRC-B" title="EVRC-B">EVRC-B</a>, EVRC-WB, 13K (<a class="mw-redirect" href="/wiki/QCELP" title="QCELP">QCELP</a>), <a href="/wiki/Selectable_Mode_Vocoder" title="Selectable Mode Vocoder">SMV</a> or <a class="mw-redirect" href="/wiki/VMR-WB" title="VMR-WB">VMR-WB</a>, which was specified by 3GPP2 for use in ISO base media file format.<sup class="reference" id="cite_ref-mp4ra_12-1"><a href="#cite_note-mp4ra-12">[12]</a></sup> The 3G2 specification also defined some enhancements to 3GPP Timed Text. 3G2 file format does not store Enhanced aacPlus (HE-AAC v2) and AMR-WB+ audio streams.<sup class="reference" id="cite_ref-3gpp2-vs-3gpp_7-2"><a href="#cite_note-3gpp2-vs-3gpp-7">[7]</a></sup> For the storage of MPEG-4 media (AAC audio, MPEG-4 Part 2 video, MPEG-4 Part 10 - H.264/AVC) in 3G2 files, the 3G2 specification refers to the MP4 file format and the AVC file format specification, which described usage of this content in the ISO base media file format. For the storage of H.263 and AMR content 3G2 specification refers to the 3GP file format specification.<sup class="reference" id="cite_ref-3gpp2-sample-entry_13-0"><a href="#cite_note-3gpp2-sample-entry-13">[13]</a></sup></p>
<h2><span class="mw-headline" id="Device_support">Device support</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=3GP_and_3G2&amp;action=edit&amp;section=5" title="Edit section: Device support">edit</a><span class="mw-editsection-bracket">]</span></span></h2>
<ul>
<li>Most <a href="/wiki/3G" title="3G">3G</a> capable mobile phones support the playback and recording of video in 3GP format (memory, maximum filesize for playback and recording, and resolution limits exist and vary).</li>
<li>Some newer/higher-end phones without 3G capabilities may also playback and record in this format (again, with said limitations).</li>
<li>Audio <a href="/wiki/Ripping" title="Ripping">imported</a> from CD onto a <a href="/wiki/PlayStation_3" title="PlayStation 3">PlayStation 3</a> when it is set to encode to the MPEG-4 <a href="/wiki/Advanced_Audio_Coding" title="Advanced Audio Coding">AAC</a> format copies onto USB devices in the 3GP format.</li>
<li>The <a href="/wiki/Nintendo_DSi" title="Nintendo DSi">Nintendo DSi</a> supports .3gp on an <a href="/wiki/Secure_Digital" title="Secure Digital">SD card</a>.</li>
<li>Apple <a class="mw-redirect" href="/wiki/IDevice" title="IDevice">iDevices</a> used to support files for playback only as <a href="/wiki/Passthrough" title="Passthrough">passthrough</a> files, hence no editing ability, but since <a href="/wiki/IOS_9" title="IOS 9">iOS 9</a> this has been deprecated meaning files of this format have to be manually converted to <a href="/wiki/H.264/MPEG-4_AVC" title="H.264/MPEG-4 AVC">H.264</a>.</li>
</ul>
<h2><span class="mw-headline" id="Software_support">Software support</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=3GP_and_3G2&amp;action=edit&amp;section=6" title="Edit section: Software support">edit</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>When transferred to a computer, 3GP movies can be viewed on <a href="/wiki/Microsoft_Windows" title="Microsoft Windows">Microsoft Windows</a>, <a class="mw-redirect" href="/wiki/Apple,_Inc" title="Apple, Inc">Apple</a> <a class="mw-redirect" href="/wiki/Mac_OS_X" title="Mac OS X">Mac OS X</a> and <a href="/wiki/GNU" title="GNU">GNU</a>/<a href="/wiki/Linux" title="Linux">Linux</a> operating systems, with MS <a href="/wiki/Windows_Media_Player" title="Windows Media Player">Windows Media Player</a>, Apple <a href="/wiki/QuickTime" title="QuickTime">QuickTime</a> and <a href="/wiki/VLC_media_player" title="VLC media player">VLC media player</a>. Programs such as <a href="/wiki/Media_Player_Classic" title="Media Player Classic">Media Player Classic</a>, <a class="mw-redirect" href="/wiki/K-Multimedia_Player" title="K-Multimedia Player">K-Multimedia Player</a>, <a class="mw-redirect" href="/wiki/Totem_(media_player)" title="Totem (media player)">Totem</a>, <a href="/wiki/RealPlayer" title="RealPlayer">RealPlayer</a>, <a href="/wiki/MPlayer" title="MPlayer">MPlayer</a>, and <a href="/wiki/GOM_Player" title="GOM Player">GOM Player</a> can also be used.</p>
<p>3GP files can be encoded and decoded with open source software <a href="/wiki/FFmpeg" title="FFmpeg">FFmpeg</a>.<sup class="reference" id="cite_ref-ffmpeg_14-0"><a href="#cite_note-ffmpeg-14">[14]</a></sup></p>
