<p><i><b>Rhinogobius giurinus</b></i> is a species of <a href="/wiki/Goby" title="Goby">goby</a> native to eastern <a href="/wiki/Asia" title="Asia">Asia</a> where it inhabits marine, brackish and fresh waters of rivers and <a class="mw-redirect" href="/wiki/Estuaries" title="Estuaries">estuaries</a>. This species can reach a length of 8 centimetres (3.1 in) <a href="/wiki/Fish_measurement" title="Fish measurement">TL</a>. It is of importance to local peoples as a food fish.<sup class="reference" id="cite_ref-2"><a href="#cite_note-2">[2]</a></sup></p>

<h2><span class="mw-headline" id="Distribution">Distribution</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Rhinogobius_giurinus&amp;action=edit&amp;section=1" title="Edit section: Distribution">edit</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>The fish's native range includes the <a href="/wiki/Yellow_River" title="Yellow River">Yellow River</a>, <a class="mw-redirect" href="/wiki/Yangtze_River" title="Yangtze River">Yangtze River</a>, <a href="/wiki/Qiantang_River" title="Qiantang River">Qiantang River</a>, <a class="new" href="/w/index.php?title=Lingjiang&amp;action=edit&amp;redlink=1" title="Lingjiang (page does not exist)">Lingjiang</a>, <a href="/wiki/Pearl_River_(China)" title="Pearl River (China)">Pearl River</a>, <a href="/wiki/Fujian" title="Fujian">Fujian</a>, <a href="/wiki/Guangdong" title="Guangdong">Guangdong</a> and <a href="/wiki/Hainan" title="Hainan">Hainan</a> regions (except the northwestern part of China including <a href="/wiki/Tibetan_Plateau" title="Tibetan Plateau">Tibetan Plateau</a> and <a class="mw-redirect" href="/wiki/Yunnan-Guizhou_Plateau" title="Yunnan-Guizhou Plateau">Yunnan-Guizhou Plateau</a>), <a href="/wiki/Taiwan" title="Taiwan">Taiwan</a>, <a href="/wiki/Hong_Kong" title="Hong Kong">Hong Kong</a>, the <a href="/wiki/Korean_Peninsula" title="Korean Peninsula">Korean Peninsula</a>, from <a href="/wiki/Tone_River" title="Tone River">Tone River</a> to <a class="mw-redirect" href="/wiki/Iriomote" title="Iriomote">Iriomote</a> in <a href="/wiki/Ibaraki_Prefecture" title="Ibaraki Prefecture">Ibaraki Prefecture</a>, <a class="new" href="/w/index.php?title=Kawahara_Lake&amp;action=edit&amp;redlink=1" title="Kawahara Lake (page does not exist)">Kawahara Lake</a> in <a href="/wiki/Nagasaki_Prefecture" title="Nagasaki Prefecture">Nagasaki Prefecture</a> in <a href="/wiki/Japan" title="Japan">Japan</a>, <a href="/wiki/Bonin_Islands" title="Bonin Islands">Bonin Islands</a>, <a href="/wiki/Ryukyu_Islands" title="Ryukyu Islands">Ryukyu Islands</a> and North <a href="/wiki/Vietnam" title="Vietnam">Vietnam</a>.</p>
<p>The species is also introduced into Singapore, Tibetan Plateau and Yunnan.<sup class="reference" id="cite_ref-3"><a href="#cite_note-3">[3]</a></sup></p>
<h2><span class="mw-headline" id="Habitat">Habitat</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Rhinogobius_giurinus&amp;action=edit&amp;section=2" title="Edit section: Habitat">edit</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>Its habitats include rivers, reservoirs, ponds and estuaries.<sup class="reference" id="cite_ref-FF_4-0"><a href="#cite_note-FF-4">[4]</a></sup> Though it is a <a class="mw-redirect" href="/wiki/Migratory_fish" title="Migratory fish">migratory fish</a>,<sup class="reference" id="cite_ref-5"><a href="#cite_note-5">[5]</a></sup> it can survive and reproduce in a completely closed <a href="/wiki/Drainage_system_(geomorphology)" title="Drainage system (geomorphology)">drainage system</a>.</p>
<h2><span class="mw-headline" id="Diet">Diet</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Rhinogobius_giurinus&amp;action=edit&amp;section=3" title="Edit section: Diet">edit</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>The fish feeds on aquatic insects, <a class="mw-redirect" href="/wiki/Invertebrates" title="Invertebrates">invertebrates</a>, small fish, <a href="/wiki/Zooplankton" title="Zooplankton">zooplankton</a>, <a href="/wiki/Phytoplankton" title="Phytoplankton">phytoplankton</a> and plant detritus.<sup class="reference" id="cite_ref-FF_4-1"><a href="#cite_note-FF-4">[4]</a></sup></p>
