<p><b>UltraISO</b> is an application for <a href="/wiki/Microsoft_Windows" title="Microsoft Windows">Microsoft Windows</a> for creating, modifying and converting <a href="/wiki/ISO_image" title="ISO image">ISO image</a> files used for <a href="/wiki/Optical_disc_authoring" title="Optical disc authoring">optical disc authoring</a>, currently being produced by EZB Systems.</p>
<p>Initially UltraISO was <a href="/wiki/Shareware" title="Shareware">shareware</a> however since 2006 it has turned 'Premium' and become chargeable.<sup class="reference" id="cite_ref-2"><a href="#cite_note-2">[2]</a></sup> The 'Free Trial' version is limited to ISO images of 300 MB or less. <sup class="reference" id="cite_ref-3"><a href="#cite_note-3">[3]</a></sup></p>

<h2><span class="mw-headline" id="Features">Features</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=UltraISO&amp;action=edit&amp;section=1" title="Edit section: Features">edit</a><span class="mw-editsection-bracket">]</span></span></h2>
<ul>
<li>Duplicate CDs and DVDs to an ISO image.</li>
<li>Create ISO images from files located on a <a class="mw-redirect" href="/wiki/Disk_drive" title="Disk drive">CD/DVD-ROM</a> or <a class="mw-redirect" href="/wiki/Hard_drive" title="Hard drive">hard disk</a>.</li>
<li>Edit existing ISO files by adding, deleting, and creating files and folders.</li>
<li>Create compressed zisofs (.ISZ) images</li>
<li>Create <a class="mw-redirect" href="/wiki/Bootable" title="Bootable">bootable</a> CD, DVD and <a class="mw-redirect" href="/wiki/Floppy_drive" title="Floppy drive">floppy</a> images.</li>
<li>Create a <a class="mw-redirect" href="/wiki/Bootable_USB" title="Bootable USB">bootable USB</a> flash or hard drive from a bootable disc image (works only with DOS, Windows NT and syslinux boot sectors). For this, the user can open an <a href="/wiki/ISO_image" title="ISO image">ISO image</a> with UltraISO and choose choose 'Bootable' and 'Write disk image' to burn onto a bootable USB disk.</li>
<li>Converts: .BIN, .IMG, .CIF, .NRG, .BWI, .DAA, .DMG, .HFS and other formats to the standard ISO image.</li>
<li>Supports all <a href="/wiki/ISO_9660" title="ISO 9660">ISO 9660</a> levels and the Joliet extension.</li>
<li>Optimization of ISO image file structures to save disc space.</li>
<li>Capable of <a href="/wiki/Mount_(computing)" title="Mount (computing)">mounting</a> only ISOs to a <a href="/wiki/RAM_drive" title="RAM drive">virtual drive</a>.</li>
<li>Create <a href="/wiki/Checksum" title="Checksum">checksums</a> for data verification of ISOs.</li>
<li>Multiple language support.</li>
<li>Built in digital audio player.</li>
<li>Supports <a href="/wiki/Universal_Disk_Format" title="Universal Disk Format">UDF</a> disc format and allows creation of bootable UDF disks.</li>
<li>Supports command line parameters<sup class="reference" id="cite_ref-4"><a href="#cite_note-4">[4]</a></sup></li>
</ul>
<h2><span class="mw-headline" id="ISZ_format">ISZ format</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=UltraISO&amp;action=edit&amp;section=2" title="Edit section: ISZ format">edit</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>UltraISO uses a proprietary format known as ISZ. The format is advertised as "ISO Zipped", even though it is not a simple zip archive. The format uses <a href="/wiki/Zlib" title="Zlib">zlib</a> or <a href="/wiki/Bzip2" title="Bzip2">bzip2</a> to compress the data, and can support <a href="/wiki/Advanced_Encryption_Standard" title="Advanced Encryption Standard">AES</a> encryption of various strengths. The file format specification is available publicly on EZB Systems's website.<sup class="reference" id="cite_ref-5"><a href="#cite_note-5">[5]</a></sup> The format is now supported by third party applications such as <a href="/wiki/Daemon_Tools" title="Daemon Tools">Daemon Tools</a>, <a href="/wiki/Alcohol_120%25" title="Alcohol 120%">Alcohol 120%</a> and <a href="/wiki/CDemu" title="CDemu">CDemu</a>.</p>
