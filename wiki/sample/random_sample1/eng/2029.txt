<div class="mw-content-ltr" dir="ltr" id="mw-content-text" lang="en"><script>function mfTempOpenSection(id){var block=document.getElementById("mf-section-"+id);block.className+=" open-block";block.previousSibling.className+=" open-block";}</script><p><b>Gyeonggi Broadcasting Corporation</b> (<a href="/wiki/Hangul" title="Hangul">Hangul</a>: <span lang="ko-Hang" xml:lang="ko-Hang">경기방송</span>; <a href="/wiki/Hanja" title="Hanja">Hanja</a>: <span lang="ko-Hani" xml:lang="ko-Hani">京畿放送</span>) is a <a href="/wiki/Broadcasting" title="Broadcasting">broadcasting</a> company headquartered in <a href="/wiki/Suwon" title="Suwon">Suwon</a>, <a class="mw-redirect" href="/wiki/Gyeonggi" title="Gyeonggi">Gyeonggi</a>, <a href="/wiki/Korea" title="Korea">Korea</a>, established in 1997. It is an <a class="mw-redirect" href="/wiki/FM_Radio" title="FM Radio">FM Radio</a> broadcaster in <a href="/wiki/Seoul" title="Seoul">Seoul</a>, <a href="/wiki/Incheon" title="Incheon">Incheon</a>, <a class="mw-redirect" href="/wiki/Gyeonggi" title="Gyeonggi">Gyeonggi</a>, tele number is KFM, <a href="/wiki/Frequency" title="Frequency">frequency</a> is 99.9 <a class="mw-redirect" href="/wiki/Megahertz" title="Megahertz">MHz</a>, over power from 5 <a class="mw-redirect" href="/wiki/Kilowatt" title="Kilowatt">kW</a> if similar broadcast network at <a href="/wiki/Munhwa_Broadcasting_Corporation" title="Munhwa Broadcasting Corporation">Munhwa Broadcasting Corporation</a>, <a href="/wiki/Korean_Broadcasting_System" title="Korean Broadcasting System">KBS</a>, <a class="mw-redirect" href="/wiki/SBS_(Korea)" title="SBS (Korea)">SBS (Korea)</a>.</p>
