<p>The <b>past</b> is a <a href="/wiki/Terminology" title="Terminology">term</a> used to indicate the totality of <a href="/wiki/Spacetime#Basic_concepts" title="Spacetime">events</a> which occurred before a given point in time. The past is contrasted with and defined by the <a href="/wiki/Present" title="Present">present</a> and the <a href="/wiki/Future" title="Future">future</a>. The concept of the past is derived from the linear fashion in which human observers experience <a href="/wiki/Time" title="Time">time</a>, and is accessed through <a href="/wiki/Memory" title="Memory">memory</a> and <a class="mw-redirect" href="/wiki/Recollection" title="Recollection">recollection</a>. In addition, human beings have <a href="/wiki/Recorded_history" title="Recorded history">recorded the past</a> since the advent of written language.</p>
<p>The past is the object of such fields as <a href="/wiki/History" title="History">history</a>, <a href="/wiki/Memory" title="Memory">memory</a>, <a href="/wiki/Flashback_(psychology)" title="Flashback (psychology)">flashback</a>, <a href="/wiki/Recall_(memory)" title="Recall (memory)">recollection</a>, <a href="/wiki/Archaeology" title="Archaeology">archaeology</a>, <a href="/wiki/Archaeoastronomy" title="Archaeoastronomy">archaeoastronomy</a>, <a href="/wiki/Chronology" title="Chronology">chronology</a>, <a href="/wiki/Geology" title="Geology">geology</a>, <a href="/wiki/Historical_geology" title="Historical geology">historical geology</a>, <a href="/wiki/Historical_linguistics" title="Historical linguistics">historical linguistics</a>, <a href="/wiki/Law" title="Law">law</a>, <a href="/wiki/Ontology" title="Ontology">ontology</a>, <a href="/wiki/Paleontology" title="Paleontology">paleontology</a>, <a href="/wiki/Paleobotany" title="Paleobotany">paleobotany</a>, <a href="/wiki/Paleoethnobotany" title="Paleoethnobotany">paleoethnobotany</a>, <a href="/wiki/Palaeogeography" title="Palaeogeography">palaeogeography</a>, <a href="/wiki/Paleoclimatology" title="Paleoclimatology">paleoclimatology</a>, and <a href="/wiki/Physical_cosmology" title="Physical cosmology">cosmology</a>.</p>
