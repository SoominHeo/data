<p>In <a href="/wiki/Mathematics" title="Mathematics">mathematics</a>, the <b>commutator</b> gives an indication of the extent to which a certain <a href="/wiki/Binary_operation" title="Binary operation">binary operation</a> fails to be <a class="mw-redirect" href="/wiki/Commutative" title="Commutative">commutative</a>. There are different definitions used in <a href="/wiki/Group_theory" title="Group theory">group theory</a> and <a href="/wiki/Ring_theory" title="Ring theory">ring theory</a>.</p>

<h2><span class="mw-headline" id="Group_theory">Group theory</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Commutator&amp;action=edit&amp;section=1" title="Edit section: Group theory">edit</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>The <b>commutator</b> of two elements, <span class="texhtml mvar" style="font-style:italic;">g</span> and <span class="texhtml mvar" style="font-style:italic;">h</span>, of a <a href="/wiki/Group_(mathematics)" title="Group (mathematics)">group</a> <span class="texhtml mvar" style="font-style:italic;">G</span>, is the element</p>
<dl>
<dd><span class="texhtml">[<i>g</i>, <i>h</i>] = <i>g</i><sup>−1</sup><i>h</i><sup>−1</sup><i>gh</i></span>.</dd>
</dl>
<p>It is equal to the group's identity if and only if <span class="texhtml mvar" style="font-style:italic;">g</span> and <span class="texhtml mvar" style="font-style:italic;">h</span> commute (i.e., if and only if <span class="texhtml"><i>gh</i> = <i>hg</i></span>). The <a href="/wiki/Subgroup" title="Subgroup">subgroup</a> of <i>G</i> <a href="/wiki/Generating_set_of_a_group" title="Generating set of a group">generated</a> by all commutators is called the <i>derived group</i> or the <i><a href="/wiki/Commutator_subgroup" title="Commutator subgroup">commutator subgroup</a></i> of <i>G</i>. Note that one must consider the subgroup generated by the set of commutators because in general the set of commutators is not closed under the group operation. Commutators are used to define <a href="/wiki/Nilpotent_group" title="Nilpotent group">nilpotent</a> and <a href="/wiki/Solvable_group" title="Solvable group">solvable</a> groups.</p>
<p>The above definition of the commutator is used by some group theorists, as well as throughout this article. However, many other group theorists define the commutator as</p>
<dl>
<dd><span class="texhtml">[<i>g</i>, <i>h</i>] = <i>ghg</i><sup>−1</sup><i>h</i><sup>−1</sup></span>.<sup class="reference" id="cite_ref-1"><a href="#cite_note-1">[1]</a></sup><sup class="reference" id="cite_ref-2"><a href="#cite_note-2">[2]</a></sup></dd>
</dl>
<h3><span class="mw-headline" id="Identities_.28group_theory.29">Identities (group theory)</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Commutator&amp;action=edit&amp;section=2" title="Edit section: Identities (group theory)">edit</a><span class="mw-editsection-bracket">]</span></span></h3>

<h2><span class="mw-headline" id="Ring_theory">Ring theory</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Commutator&amp;action=edit&amp;section=3" title="Edit section: Ring theory">edit</a><span class="mw-editsection-bracket">]</span></span></h2>

<p>It is zero if and only if <i>a</i> and <i>b</i> commute. In <a href="/wiki/Linear_algebra" title="Linear algebra">linear algebra</a>, if two endomorphisms of a space are represented by commuting matrices with respect to one basis, then they are so represented with respect to every basis. By using the commutator as a <a href="/wiki/Lie_algebra" title="Lie algebra">Lie bracket</a>, every associative algebra can be turned into a <a href="/wiki/Lie_algebra" title="Lie algebra">Lie algebra</a>.</p>
<p>The <b>anticommutator</b> of two elements <span class="texhtml mvar" style="font-style:italic;">a</span> and <span class="texhtml mvar" style="font-style:italic;">b</span> of a ring or an associative algebra is defined by</p>

<p>Sometimes the brackets [ ]<sub>+</sub> are also used to denote anticommutators.<sup class="reference" id="cite_ref-6"><a href="#cite_note-6">[6]</a></sup> The anticommutator is used less often than the commutator, but can be used for example to define <a href="/wiki/Clifford_algebra" title="Clifford algebra">Clifford algebras</a>, <a href="/wiki/Jordan_algebra" title="Jordan algebra">Jordan algebras</a> and is utilised to derive the <a href="/wiki/Dirac_equation" title="Dirac equation">Dirac equation</a> in particle physics.</p>
<p>The commutator of two operators acting on a <a href="/wiki/Hilbert_space" title="Hilbert space">Hilbert space</a> is a central concept in <a href="/wiki/Quantum_mechanics" title="Quantum mechanics">quantum mechanics</a>, since it quantifies how well the two <a href="/wiki/Observable" title="Observable">observables</a> described by these operators can be measured simultaneously. The <a href="/wiki/Uncertainty_principle" title="Uncertainty principle">uncertainty principle</a> is ultimately a theorem about such commutators, by virtue of the <a class="mw-redirect" href="/wiki/Uncertainty_relation" title="Uncertainty relation">Robertson–Schrödinger relation</a>.<sup class="reference" id="cite_ref-7"><a href="#cite_note-7">[7]</a></sup> In <a href="/wiki/Phase_space" title="Phase space">phase space</a>, equivalent commutators of function <a href="/wiki/Moyal_product" title="Moyal product">star-products</a> are called <a href="/wiki/Moyal_bracket" title="Moyal bracket">Moyal brackets</a>, and are completely isomorphic to the Hilbert-space commutator structures mentioned.</p>

<p>By substituting <span class="texhtml"><i>x</i> = ∂</span> and appending a test function to both sides of the equation, one recovers the general Leibniz rule.</p>
