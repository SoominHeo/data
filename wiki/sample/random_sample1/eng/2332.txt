<p><b>SearchWiki</b> was a <a href="/wiki/Google_Search" title="Google Search">Google Search</a> feature which allowed logged-in users to annotate and re-order search results. The annotations and modified order only applied to the user's searches, but it was possible to view other users' annotations for a given search query.<sup class="reference" id="cite_ref-1"><a href="#cite_note-1">[1]</a></sup> SearchWiki was released on November 20, 2008 and discontinued on March 3, 2010.<sup class="reference" id="cite_ref-2"><a href="#cite_note-2">[2]</a></sup> All previously created SearchWiki edits are preserved on the logged in user's <a class="external text" href="http://www.google.com/reviews/w" rel="nofollow">SearchWiki notes page</a>.<sup class="reference" id="cite_ref-3"><a href="#cite_note-3">[3]</a></sup></p>
<p><b>Google Stars</b> replaces SearchWiki. Users no longer have the option to annotate or re-order search results. Instead a user clicks on a "star marker" which makes the entry appear in a starred results list at the top of a new search. The starred results list can be clicked to bring up <a href="/wiki/Google_Bookmarks" title="Google Bookmarks">Google Bookmarks</a> where the list entries can be annotated.</p>
