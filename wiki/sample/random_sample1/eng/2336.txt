<h2><span class="mw-headline" id="Web-based_products">Web-based products</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=List_of_Google_products&amp;action=edit&amp;section=1" title="Edit section: Web-based products">edit</a><span class="mw-editsection-bracket">]</span></span></h2>
<h3><span class="mw-headline" id="Search_tools">Search tools</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=List_of_Google_products&amp;action=edit&amp;section=2" title="Edit section: Search tools">edit</a><span class="mw-editsection-bracket">]</span></span></h3>
<ul>
<li><a href="/wiki/Google_Search" title="Google Search">Google Search</a> is a web search engine, which is <a href="/wiki/Google" title="Google">Google</a>'s core product. It receives over 3 <a class="mw-redirect" href="/wiki/100000000000_(number)" title="100000000000 (number)">billion</a> search queries per day. Google also offers regional search by its 189 regional level domains. (see <a href="/wiki/List_of_Google_domains#Localized_.26_regional_domains" title="List of Google domains">List of Google domains#Localized &amp; regional domains</a>)
<ul>
<li><a href="/wiki/Google_Hummingbird" title="Google Hummingbird">Hummingbird</a> – Expanded query analysis. For example, if you search for 'best pie place in Seattle' Google will also search for 'best pie restaurant in Seattle'.</li>
<li><a href="/wiki/PageRank" title="PageRank">PageRank</a> – link analysis algorithm.</li>
<li><a class="mw-redirect" href="/wiki/Google_search#Search_results" title="Google search">Snapshots</a> – mechanism that indexes <a class="mw-redirect" href="/wiki/PDF" title="PDF">PDFs</a>, Word documents, and more.</li>
<li><a class="mw-redirect" href="/wiki/Google_search#Functionality" title="Google search">Google Search functionality</a> – Google Search includes <a class="mw-redirect" href="/wiki/Boolean_logic" title="Boolean logic">Boolean logical</a> operators, <a href="/wiki/Wildcard_character" title="Wildcard character">wildcards</a>, and more, to help users refine their searches.</li>
<li><a class="mw-redirect" href="/wiki/Google_search#International" title="Google search">Multiple languages</a> – Google Search is supported by a large number of different languages.</li>
<li><a class="new" href="/w/index.php?title=Google_Author_Rank&amp;action=edit&amp;redlink=1" title="Google Author Rank (page does not exist)">Google Author Rank</a> – The idea that an online author can have topical authority within Google Search Results.</li>
<li><a class="mw-redirect" href="/wiki/Experimental_Search" title="Experimental Search">Experimental Search</a> options for testing new interfaces while searching with Google, including Timeline views and keyboard shortcuts.</li>
<li><a href="/wiki/Google_Search#Encrypted_search" title="Google Search">Encrypted Search</a> – In May 2010 Google rolled out SSL-encrypted web search.<sup class="reference" id="cite_ref-1"><a href="#cite_note-1">[1]</a></sup> The encrypted search can be accessed at encrypted.google.com <sup class="reference" id="cite_ref-2"><a href="#cite_note-2">[2]</a></sup></li>
</ul>
</li>
<li><a href="/wiki/Google_Alerts" title="Google Alerts">Google Alerts</a> – email notification service, which sends alerts based on chosen search terms, whenever there are new results. Alerts include web results, <a href="/wiki/Google_Groups" title="Google Groups">Groups</a> results news, and video. The system is not functioning correctly beginning in 2013.<sup class="reference" id="cite_ref-3"><a href="#cite_note-3">[3]</a></sup></li>
<li><a href="/wiki/Google_Books" title="Google Books">Google Books</a> (formerly Print) – search engine for the full text of printed books. Google scans and stores in its digital database. The content that is displayed depends on the arrangement with the publishers, ranging from short extracts to entire books.</li>
<li><a href="/wiki/Google_Custom_Search" title="Google Custom Search">Google Custom Search</a> – allows a user to create a customized search experience for his/her own website. Renamed from Google Co-op, which in turn replaced Google Free Search.</li>
<li><a href="/wiki/Google_Finance" title="Google Finance">Google Finance</a> – searchable US business news, opinion, and financial data. Features include company-specific pages, blog search, interactive charts, executives information, discussion groups and a portfolio.</li>
<li><a href="/wiki/Google_Groups" title="Google Groups">Google Groups</a> – web and email discussion service and <a href="/wiki/Usenet" title="Usenet">Usenet</a> archive. Users can join a group, make a group, publish posts, track their favorite topics, write a set of group web pages updatable by members and share group files. In January, 2007, version 3 of Google Groups was released. New features include the ability to create customized pages and share files.</li>
<li><a class="mw-redirect" href="/wiki/Google_Hotel_Finder" title="Google Hotel Finder">Google Hotel Finder</a> – Provides searches similar to other Online Travel Agencies (<a href="/wiki/Travel_website" title="Travel website">Travel website</a>) that searchers can search for check-in and check-out dates.<sup class="reference" id="cite_ref-4"><a href="#cite_note-4">[4]</a></sup> Now directly available through searching, for example searching 'hotels in Seattle' will result in an info box under advertisements.<sup class="reference" id="cite_ref-5"><a href="#cite_note-5">[5]</a></sup></li>
<li><a href="/wiki/Google_Flights" title="Google Flights">Google Flight Search</a> – a service that allows users to search for flights from many airlines to many destinations, offering tools such as price comparisons and travel recommendations.<sup class="reference" id="cite_ref-6"><a href="#cite_note-6">[6]</a></sup></li>
<li><a class="mw-redirect" href="/wiki/Google_Image_Search" title="Google Image Search">Google Image Search</a> – image search engine, with results based on the file name of the image, the link text pointing to the image and text adjacent to the image. You can also make a search by uploading a picture from your computer. When searching, a thumbnail of each matching image is displayed.</li>
<li><a href="/wiki/Google_Translate" title="Google Translate">Language Tools</a> – Collection of linguistic applications, including one that allows users to translate text or web pages from one language to another, and another that allows searching in web pages located in a specific country or written in a specific language.</li>
<li><i>Life Search</i> (<a href="/wiki/Google_China" title="Google China">Google China</a>) – Search engine tailored towards everyday needs, such as train times, recipes and housing.</li>
<li><a href="/wiki/Google_News" title="Google News">Google News</a> – automated <a href="/wiki/News" title="News">news</a> compilation service and search engine for news. There are versions of the aggregator for more than 20 languages. While the selection of news stories is fully automated, the sites included are selected by human editors.</li>
<li><a class="mw-redirect" href="/wiki/Google_Patent_Search" title="Google Patent Search">Google Patent Search</a> – search engine to search through millions of <a class="mw-redirect" href="/wiki/Patents" title="Patents">patents</a>, each result with its own page, including drawings, claims and citations.</li>
<li><i>Google Recipe View</i> – lets you narrow your search results to show only recipes, and helps you choose the right recipe amongst the search results by showing clearly marked ratings, ingredients and pictures. First mentioned on Google's blog in February 2011.<sup class="reference" id="cite_ref-7"><a href="#cite_note-7">[7]</a></sup></li>
<li><a href="/wiki/Google_Scholar" title="Google Scholar">Google Scholar</a> – search engine for the full text of scholarly literature across an array of publishing formats and scholarly fields. Today, the index includes virtually all peer-reviewed journals available online.</li>
<li><a href="/wiki/Google_Shopping" title="Google Shopping">Google Shopping</a> (was Google Product Search and Froogle): <a class="mw-redirect" href="/wiki/Price_engine" title="Price engine">price engine</a> that searches online stores, including auctions, for products. Beginning in Fall of 2012, it will become a fully commercial product, only indexing paid listings.<sup class="reference" id="cite_ref-8"><a href="#cite_note-8">[8]</a></sup></li>
<li><i>Suggest</i><span id="Suggest"></span> – <a href="/wiki/Autocomplete" title="Autocomplete">auto-completion</a> in search results while typing to give popular searches.</li>
<li><a class="mw-redirect" href="/wiki/Google_Video" title="Google Video">Google Video</a> – video search engine. Google's main video partnerships include agreements with <a href="/wiki/CBS" title="CBS">CBS</a>, <a class="mw-redirect" href="/wiki/NHL" title="NHL">NHL</a> and the <a class="mw-redirect" href="/wiki/NBA" title="NBA">NBA</a>. It also searches videos posted on YouTube, Metacafe, Daily Motion, and other popular video hosting sites. In the past Google Video itself offered a video hosting service, but this feature was shut down on August 20, 2012.<sup class="reference" id="cite_ref-9"><a href="#cite_note-9">[9]</a></sup></li>
<li><a href="/wiki/Google_Web_History" title="Google Web History">Web History</a> (was Google Search History, <a class="mw-redirect" href="/wiki/Personalized_Search" title="Personalized Search">Personalized Search</a>) – web page tracking, which records Google searches, web pages, images, videos, music and other data. It also includes bookmarks, search trends and item recommendations. Google released <i>Search History</i> in April 2005, when it began to record browsing history,<sup class="reference" id="cite_ref-10"><a href="#cite_note-10">[10]</a></sup> later expanding and renaming the service to <i>Web History</i> in April 2007.<sup class="reference" id="cite_ref-11"><a href="#cite_note-11">[11]</a></sup></li>
<li><a href="/wiki/Knowledge_Graph" title="Knowledge Graph">Knowledge Graph</a> – a knowledge base used to enhance search results with semantic information gathered from several sources.</li>
<li><a href="/wiki/Zagat" title="Zagat">Zagat</a> – a source of consumer survey-based information for restaurants and other leisure activities.</li>
</ul>
<h3><span class="mw-headline" id="Advertising_services">Advertising services</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=List_of_Google_products&amp;action=edit&amp;section=3" title="Edit section: Advertising services">edit</a><span class="mw-editsection-bracket">]</span></span></h3>
<ul>
<li><a href="/wiki/AdMob" title="AdMob">AdMob</a> is a mobile advertising network that Google acquired in November 2009.<sup class="reference" id="cite_ref-Mashable_12-0"><a href="#cite_note-Mashable-12">[12]</a></sup> It offers advertising solutions for <a href="/wiki/Android_(operating_system)" title="Android (operating system)">Android</a>, <a href="/wiki/IOS" title="IOS">iOS</a> and <a href="/wiki/Windows_Phone_8" title="Windows Phone 8">Windows Phone 8</a>.<sup class="reference" id="cite_ref-13"><a href="#cite_note-13">[13]</a></sup></li>
<li><a class="mw-redirect" href="/wiki/Google_AdSense" title="Google AdSense">Google AdSense</a> – Offers a contextual advertising solution to web publishers, and delivers text-based Google AdWords ads that are relevant to site content pages.</li>
<li>Google Ad Planner – Ad Planner has been replaced with Google Display Planner.<sup class="reference" id="cite_ref-14"><a href="#cite_note-14">[14]</a></sup></li>
<li><a class="mw-redirect" href="/wiki/Google_AdWords" title="Google AdWords">Google AdWords</a> – advertise with Google AdWords ads in the Sponsored Links section next to search results to boost website traffic and sales.</li>
<li><a class="mw-redirect" href="/wiki/Google_Adwords_Express" title="Google Adwords Express">Adwords Express</a> – Local online advertising made easy</li>
<li><a class="mw-redirect" href="/wiki/Google_Certification_Program" title="Google Certification Program">Google Certification Program</a> – <a class="mw-redirect" href="/wiki/Google_AdWords" title="Google AdWords">Google AdWords</a> partner certification program, providing AdWords qualifications to agencies that pass exams and other criteria. Replaced Google Advertising Professionals in April 2010.</li>

<h2><span class="mw-headline" id="Discontinued_products_and_services">Discontinued products and services</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=List_of_Google_products&amp;action=edit&amp;section=16" title="Edit section: Discontinued products and services">edit</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>Applications that have been retired by Google, either because of integration with other Google products, or through lack of support:<sup class="reference" id="cite_ref-44"><a href="#cite_note-44">[44]</a></sup></p>
<h3><span class="mw-headline" id="Discontinued_in_2006">Discontinued in 2006</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=List_of_Google_products&amp;action=edit&amp;section=17" title="Edit section: Discontinued in 2006">edit</a><span class="mw-editsection-bracket">]</span></span></h3>
<ul>
<li><a href="/wiki/Google_Answers" title="Google Answers">Google Answers</a> – Online knowledge market offered by Google that allowed users to post bounties for well researched answers to their queries. Discontinued on November 28, 2006; still accessible (as read-only).</li>
<li>Google Deskbar – desktop bar with a built-in mini browser. Replaced by a similar feature in <a href="/wiki/Google_Desktop" title="Google Desktop">Google Desktop</a>. Discontinued on May 8, 2006.</li>
<li><a class="mw-redirect" href="/wiki/Writely" title="Writely">Writely</a> – web-based <a href="/wiki/Word_processor" title="Word processor">word processor</a> created by software company Upstartle, who were acquired by Google on March 9, 2006. On October 10, 2006, Writely was merged into Google Docs &amp; Spreadsheets.</li>
</ul>
<h3><span class="mw-headline" id="Discontinued_in_2007">Discontinued in 2007</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=List_of_Google_products&amp;action=edit&amp;section=18" title="Edit section: Discontinued in 2007">edit</a><span class="mw-editsection-bracket">]</span></span></h3>
<ul>
<li><a href="/wiki/AdWords#Google_Click-to-Call" title="AdWords">Google Click-to-Call</a> – allowed a user to speak directly over the phone without charge to businesses found on Google search results pages. Discontinued in 2007.</li>
<li><b>Related Links</b> – Webmasters could place these units on their site to provide visitors with links to information related to the site's content. Discontinued on April 30, 2007.</li>
<li><b>Public Service Search</b> – Non-commercial organization service, which included Google Site Search, traffic reports and unlimited search queries. Discontinued on February 2007; replaced by <a href="/wiki/Google_Custom_Search" title="Google Custom Search">Google Custom Search</a>.</li>
<li><b>Google Video Marketplace</b> – The service was unsuccessful and was discontinued in August, 2007.<sup class="reference" id="cite_ref-45"><a href="#cite_note-45">[45]</a></sup></li>
</ul>
<h3><span class="mw-headline" id="Discontinued_in_2008">Discontinued in 2008</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=List_of_Google_products&amp;action=edit&amp;section=19" title="Edit section: Discontinued in 2008">edit</a><span class="mw-editsection-bracket">]</span></span></h3>
<ul>
<li><a href="/wiki/Google_Browser_Sync" title="Google Browser Sync">Google Browser Sync</a> (<a href="/wiki/Firefox" title="Firefox">Firefox</a>) – allowed users of <a class="mw-redirect" href="/wiki/Mozilla_Firefox" title="Mozilla Firefox">Mozilla Firefox</a> to synchronize their web browser settings across multiple computers via the Internet. Discontinued in June 2008. <a href="/wiki/Google_Chrome" title="Google Chrome">Google Chrome</a> has similar functionality built-in.</li>
<li><a href="/wiki/Google_Lively" title="Google Lively">Google Lively</a> – 3D animated chat program launched on July 9, 2008 and closed December 31, 2008.<sup class="reference" id="cite_ref-46"><a href="#cite_note-46">[46]</a></sup> (Windows XP, Vista)</li>
<li><a href="/wiki/Picasa#Hello" title="Picasa">Hello</a> – allowed a user to send images across the Internet and publish them to blogs. Discontinued on May 15, 2008.<sup class="reference" id="cite_ref-47"><a href="#cite_note-47">[47]</a></sup></li>
<li><a class="mw-redirect" href="/wiki/SearchMash" title="SearchMash">SearchMash</a> – Search engine to "test innovative user interfaces." Aside from its privacy policy and terms of service, no Google branding existed on the site. Discontinued on November 24, 2008.<sup class="reference" id="cite_ref-48"><a href="#cite_note-48">[48]</a></sup></li>
<li><b>Send to Phone</b> – Enabled a user to send links and other information from <a href="/wiki/Firefox" title="Firefox">Firefox</a> to their phone by <a href="/wiki/Text_messaging" title="Text messaging">text message</a>. Discontinued on August 28, 2008; replaced by Google Chrome to Phone.</li>
</ul>
<h3><span class="mw-headline" id="Discontinued_in_2009">Discontinued in 2009</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=List_of_Google_products&amp;action=edit&amp;section=20" title="Edit section: Discontinued in 2009">edit</a><span class="mw-editsection-bracket">]</span></span></h3>
<ul>
<li><b>Audio Ads</b> – Radio advertising program for US businesses. Rolled out on May 15, 2007 through the AdWords interface. Discontinued on February 12, 2009.</li>
<li><b>Catalogs</b> – Search engine for over 6,600 print catalogs, acquired through <a href="/wiki/Optical_character_recognition" title="Optical character recognition">optical character recognition</a>. Discontinued January 2009.</li>
<li><a href="/wiki/Dodgeball_(service)" title="Dodgeball (service)">Dodgeball</a> – <a href="/wiki/Social_networking_service" title="Social networking service">Social networking service</a> for mobile phones. Users could text their location to the service, which would then notify them of nearby people or events of interest. Development ceased on January 14, 2009; discontinued over the next few months; replaced by <a href="/wiki/Google_Latitude" title="Google Latitude">Google Latitude</a>.</li>
<li><a class="mw-redirect" href="/wiki/Google_Ride_Finder" title="Google Ride Finder">Google Ride Finder</a> – Taxi and shuttle search service, using real time position of vehicles in 14 US cities. Used the <a href="/wiki/Google_Maps" title="Google Maps">Google Maps</a> interface and cooperated with any car service that wished to participate. Discontinued on October 2009.</li>
<li><b>Shared Stuff</b> – web page sharing system, incorporating a share <a href="/wiki/Bookmarklet" title="Bookmarklet">bookmarklet</a> to share pages, and a page to view the most popular shared items. Pages could also be shared through third party applications such as Delicious or <a href="/wiki/Facebook" title="Facebook">Facebook</a>. Discontinued on March 30, 2009.</li>
<li><a href="/wiki/Google_Page_Creator" title="Google Page Creator">Google Page Creator</a> – webpage publishing program, which could be used to create pages and to host them on Google's servers. Discontinued, and all existing content transferred to <a href="/wiki/Google_Sites" title="Google Sites">Google Sites</a> in 2009.</li>
</ul>
<h3><span class="mw-headline" id="Discontinued_in_2010">Discontinued in 2010</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=List_of_Google_products&amp;action=edit&amp;section=21" title="Edit section: Discontinued in 2010">edit</a><span class="mw-editsection-bracket">]</span></span></h3>
<ul>
<li>Marratech e-Meeting – <a href="/wiki/Web_conferencing" title="Web conferencing">Web conferencing</a> software, used internally by Google's employees. Google acquired the software from creator <a href="/wiki/Marratech" title="Marratech">Marratech</a> on April 19, 2007. Discontinued on February 19, 2010.</li>
<li><a href="/wiki/GOOG-411" title="GOOG-411">GOOG-411</a> (also known as Voice Local Search) –Google's <a href="/wiki/Directory_assistance" title="Directory assistance">directory assistance</a> service, free of charge from any telephone in the US and Canada. Discontinued on November 12, 2010.</li>
</ul>
<h3><span class="mw-headline" id="Discontinued_in_2011">Discontinued in 2011</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=List_of_Google_products&amp;action=edit&amp;section=22" title="Edit section: Discontinued in 2011">edit</a><span class="mw-editsection-bracket">]</span></span></h3>
<ul>
<li><a href="/wiki/Google_Labs" title="Google Labs">Google Labs</a> – allowed users to test and demonstrate new Google products.</li>
<li><a href="/wiki/Google_Buzz" title="Google Buzz">Google Buzz</a> – social networking service integrated with <a href="/wiki/Gmail" title="Gmail">Gmail</a> service allowing users to share content immediately and make conversations. It was released on February 9, 2010. Discontinued by end of 2011.<sup class="reference" id="cite_ref-49"><a href="#cite_note-49">[49]</a></sup></li>
<li><a href="/wiki/Google_Dictionary" title="Google Dictionary">Google Dictionary</a> – it was first introduced as part of <a href="/wiki/Google_Translate" title="Google Translate">Google Translate</a>, it then became a standalone service that allowed searching of words and phrases from over 22 languages. Discontinued on August 5, 2011.</li>
<li><a href="/wiki/Google_PowerMeter" title="Google PowerMeter">Google PowerMeter</a> – free energy monitoring tool to view building energy consumption online. Discontinued on September 16, 2011.</li>
<li><a class="mw-redirect" href="/wiki/Real_Estate" title="Real Estate">Real Estate</a> – Real estate listings in Google Maps, launched in July 2009 and discontinued February 10, 2011.<sup class="reference" id="cite_ref-50"><a href="#cite_note-50">[50]</a></sup></li>
<li>Google Directory – Collection of links arranged into hierarchical subcategories. The links and their categorization were from the <a class="mw-redirect" href="/wiki/Open_Directory_Project" title="Open Directory Project">Open Directory Project</a>, sorted using <a href="/wiki/PageRank" title="PageRank">PageRank</a>. It was closed on July 20, 2011.</li>
<li><a href="/wiki/Google_Squared" title="Google Squared">Squared</a> – creates tables of information about a subject from <a href="/wiki/Unstructured_data" title="Unstructured data">unstructured data</a>. Discontinued September 2011.</li>
<li>Google Sets – Generates a list of items when users enter a few examples. For example, entering "Green, Purple, Red" emits the list "Green, Purple, Red, Blue, Black, White, Yellow, Orange, Brown." Described in its 2003 patent filing by creators Simon Tong and Jeff Dean as "an inside peek at how Google groups keywords by concept," it was discontinued in mid-2011.<sup class="reference" id="cite_ref-51"><a href="#cite_note-51">[51]</a></sup></li>
<li><a href="/wiki/Google_Pack" title="Google Pack">Google Pack</a> – Collection of applications – some Google-created (including <a href="/wiki/Google_Earth" title="Google Earth">Google Earth</a>, <a href="/wiki/Google_Desktop" title="Google Desktop">Google Desktop</a>, <a href="/wiki/Picasa" title="Picasa">Picasa</a>, <a href="/wiki/Google_Talk" title="Google Talk">Google Talk</a>, and <a href="/wiki/Google_Chrome" title="Google Chrome">Google Chrome</a>), some not. (Supported Windows XP, Vista, 7). Discontinued on September 2, 2011.</li>
<li><a href="/wiki/Google_Fast_Flip" title="Google Fast Flip">Google Fast Flip</a> – Online news aggregator that mimicked the experience of flicking through a newspaper or magazine, allowing visual search of stories in manner similar to microfiche. Discontinued September 6, 2011.</li>
<li><a href="/wiki/Google_Desktop" title="Google Desktop">Desktop</a> (<a class="mw-redirect" href="/wiki/Mac_OS_X" title="Mac OS X">Mac OS X</a>, <a href="/wiki/Windows_2000" title="Windows 2000">Windows 2000</a> SP3+, <a href="/wiki/Windows_XP" title="Windows XP">XP</a>, <a href="/wiki/Windows_Vista" title="Windows Vista">Vista</a>, <a href="/wiki/Windows_7" title="Windows 7">7</a>, <a href="/wiki/Linux" title="Linux">Linux</a>): <a href="/wiki/Desktop_environment" title="Desktop environment">desktop</a> search application that indexed emails, documents, music, photos, chats, Web history and other files. It allowed the installation of Google Gadgets. Discontinued on September 14, 2011.</li>
<li><a href="/wiki/Google_Image_Labeler" title="Google Image Labeler">Google Image Labeler</a> – game that induced participants to submit valid descriptions (labels) of images in the web, in order to later improve Image Search. Discontinued on September 16, 2011.</li>
<li><a href="/wiki/Aardvark_(search_engine)" title="Aardvark (search engine)">Aardvark</a> – Social search utility which allowed people to ask and answer questions within their social networks. It used people's claimed expertise to match askers with good answerers. Discontinued on September 30, 2011.</li>
<li>Directory<span id="Directory"></span> – navigation directory, specifically for Chinese users.<sup class="reference" id="cite_ref-52"><a href="#cite_note-52">[52]</a></sup></li>
<li><a href="/wiki/Gears_(software)" title="Gears (software)">Gears</a> (<a href="/wiki/Google_Chrome" title="Google Chrome">Google Chrome</a>, <a href="/wiki/Firefox" title="Firefox">Firefox</a>, <a href="/wiki/Internet_Explorer" title="Internet Explorer">Internet Explorer</a> and <a href="/wiki/Safari_(web_browser)" title="Safari (web browser)">Safari</a>): A plug-in that added features to the web browser, enabling some new web applications.</li>
</ul>
<h3><span class="mw-headline" id="Discontinued_in_2012">Discontinued in 2012</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=List_of_Google_products&amp;action=edit&amp;section=23" title="Edit section: Discontinued in 2012">edit</a><span class="mw-editsection-bracket">]</span></span></h3>
<ul>
<li><a href="/wiki/Google_Notebook" title="Google Notebook">Google Notebook</a> – Online notetaking and web-clipping application. The tool permitted users to clip text, images, and links from pages while browsing, save them online, access them from any computer, and share them with others.<sup class="reference" id="cite_ref-53"><a href="#cite_note-53">[53]</a></sup> Data was exported to Google Docs and the service has been shut down as of July 2012.</li>
<li><a class="mw-redirect" href="/wiki/Google_Apps" title="Google Apps">Google Apps</a> Standard Edition – Free. On December 6, 2012, Google discontinued the Google apps standard edition, which was previously downgraded with features like maximum 10 users.<sup class="reference" id="cite_ref-54"><a href="#cite_note-54">[54]</a></sup></li>
<li><a href="/wiki/Google_Code_Search" title="Google Code Search">Google Code Search</a> – Search engine for programming code found on the Internet. Shut down on January 15, 2012.<sup class="reference" id="cite_ref-55"><a href="#cite_note-55">[55]</a></sup></li>
<li><a href="/wiki/Google_Health" title="Google Health">Google Health</a> – allows a user to store, manage and share personal health information in one place. Development ceased June 24, 2011; accessible until January 1, 2012; data available for download until January 1, 2013.</li>
<li><a href="/wiki/Google_Website_Optimizer" title="Google Website Optimizer">Google Website Optimizer</a> – a free website testing and optimization tool. Discontinued on August 1, 2012.<sup class="reference" id="cite_ref-56"><a href="#cite_note-56">[56]</a></sup></li>
<li><b>TV Ads</b> – Method to place advertising on TV networks. Launched in 2007, discontinued on August 30, 2012,<sup class="reference" id="cite_ref-57"><a href="#cite_note-57">[57]</a></sup> with all remaining active campaigns ending December 16, 2012.<sup class="reference" id="cite_ref-58"><a href="#cite_note-58">[58]</a></sup></li>
<li><a href="/wiki/Google_Friend_Connect" title="Google Friend Connect">Google Friend Connect</a> – an online service that allowed website and blog owners to add social features to their websites. Discontinued on March 1, 2012, and replaced by <a href="/wiki/Google%2B" title="Google+">Google+</a>'s pages and off-site Page badges.<sup class="reference" id="cite_ref-more_spring_cleaning_59-0"><a href="#cite_note-more_spring_cleaning-59">[59]</a></sup></li>
<li><a href="/wiki/Google_Insights_for_Search" title="Google Insights for Search">Google Insights for Search</a> – was a service by Google similar to Google Trends, providing insights into the Google search terms people used. Discontinued September 27, 2012 by Google; merged in Google Trends.</li>
<li><a href="/wiki/Knol" title="Knol">Knol</a> – service that enabled experts and others to write authoritative articles related to various topics. Content was not accessible after October 1, 2012.<sup class="reference" id="cite_ref-more_spring_cleaning_59-1"><a href="#cite_note-more_spring_cleaning-59">[59]</a></sup></li>
<li><a class="mw-redirect" href="/wiki/Google_Wave" title="Google Wave">Google Wave</a> – Online communication and <a href="/wiki/Collaborative_real-time_editor" title="Collaborative real-time editor">collaborative real-time editor</a> tool using richly formatted text, photos, videos, maps, and more. Development ceased on August 4, 2010, support ended on April 30, 2012.<sup class="reference" id="cite_ref-more_spring_cleaning_59-2"><a href="#cite_note-more_spring_cleaning-59">[59]</a></sup></li>
<li><a href="/wiki/Picnik" title="Picnik">Picnik</a> – online photo editor. Most features and effects were moved to the Google+ photo manager. Closed on April 19, 2012.<sup class="reference" id="cite_ref-Picnik_is_closing_60-0"><a href="#cite_note-Picnik_is_closing-60">[60]</a></sup></li>
<li><a href="/wiki/Jaiku" title="Jaiku">Jaiku</a> – was a <a class="mw-redirect" href="/wiki/Social_networking" title="Social networking">social networking</a>, <a href="/wiki/Microblogging" title="Microblogging">microblogging</a> and <a href="/wiki/Lifestreaming" title="Lifestreaming">lifestreaming</a> service comparable to <a href="/wiki/Twitter" title="Twitter">Twitter</a>.</li>
<li><a href="/wiki/Nexus_Q" title="Nexus Q">Nexus Q</a> – was a <a href="/wiki/Digital_media_player" title="Digital media player">digital media player</a> released in June 2012, dropped four months later.</li>
<li><a href="/wiki/Slide.com" title="Slide.com">Slide.com</a> was shut down on March 6, 2012<sup class="reference" id="cite_ref-61"><a href="#cite_note-61">[61]</a></sup></li>
<li><a class="mw-redirect" href="/wiki/Google_Mini" title="Google Mini">Google Mini</a> – Reduced capacity, lower-cost version of the Google Search Appliance. Discontinued on July 31, 2012.<sup class="reference" id="cite_ref-summer_cleaning_62-0"><a href="#cite_note-summer_cleaning-62">[62]</a></sup></li>
<li><a class="mw-redirect" href="/wiki/Picasa_Web_Albums_Uploader" title="Picasa Web Albums Uploader">Picasa Web Albums Uploader</a> – application to upload images to the "<a href="/wiki/Picasa" title="Picasa">Picasa</a> Web Albums" service. It consisted of an <a href="/wiki/IPhoto" title="IPhoto">iPhoto</a> plug-in and a stand-alone application.<sup class="reference" id="cite_ref-63"><a href="#cite_note-63">[63]</a></sup></li>
</ul>
<h3><span class="mw-headline" id="Discontinued_in_2013">Discontinued in 2013</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=List_of_Google_products&amp;action=edit&amp;section=24" title="Edit section: Discontinued in 2013">edit</a><span class="mw-editsection-bracket">]</span></span></h3>
<ul>
<li><a href="/wiki/Google_Cloud_Connect" title="Google Cloud Connect">Google Cloud Connect</a> – <a href="/wiki/Microsoft_Office" title="Microsoft Office">Microsoft Office</a> <a href="/wiki/Plug-in_(computing)" title="Plug-in (computing)">plugin</a> for automatically backing up Office documents upon saving onto Google Docs. Shut down on April 30, 2013 in favor of Google Drive.<sup class="reference" id="cite_ref-spring_two_64-0"><a href="#cite_note-spring_two-64">[64]</a></sup></li>
<li><a href="/wiki/Google_Building_Maker" title="Google Building Maker">Google Building Maker</a> – web based building and editing tool to create 3D buildings for Google Earth. Retired on June 4, 2013.</li>
<li>Google Calendar Sync – Microsoft Windows application to sync Microsoft Outlook email and calendar with Gmail and Google Calendar. Synchronization for existing installations stopped on August 1, 2014. It was replaced with Google Sync which does not synchronize Outlook calendars, but can sync email using IMAP or POP3. Also, Google Apps for Business, Education, and Government customers can use Google Apps Sync for Microsoft Outlook <sup class="reference" id="cite_ref-65"><a href="#cite_note-65">[65]</a></sup></li>
<li><a href="/wiki/Meebo" title="Meebo">Meebo</a> – A social networking website discontinued on June 6, 2013</li>
<li><a href="/wiki/Google_Reader" title="Google Reader">Google Reader</a> – web-based <a href="/wiki/News_aggregator" title="News aggregator">news aggregator</a>, capable of reading <a href="/wiki/Atom_(standard)" title="Atom (standard)">Atom</a> and <a class="mw-redirect" href="/wiki/RSS_(file_format)" title="RSS (file format)">RSS</a> feeds. It let users search, import and subscribe to feeds. The service also embedded audio <a class="mw-redirect" href="/wiki/RSS_enclosures" title="RSS enclosures">enclosures</a> in the page. Reader retired on July 1, 2013.<sup class="reference" id="cite_ref-spring_two_64-1"><a href="#cite_note-spring_two-64">[64]</a></sup></li>
<li><a href="/wiki/Google_Latitude" title="Google Latitude">Google Latitude</a> – mobile geolocation tool that lets friends know where users are via Google Maps. It was retired on August 9, 2013, with some of its functionality rolled into Google+.<sup class="reference" id="cite_ref-66"><a href="#cite_note-66">[66]</a></sup></li>
<li><a href="/wiki/Google_Talk" title="Google Talk">Google Talk</a> – Desktop instant messaging service that provided both text and voice communication. Replaced May 15, 2013 by <a href="/wiki/Google_Hangouts" title="Google Hangouts">Google Hangouts</a>.</li>
<li>SMS – <a href="/wiki/Mobile_phone" title="Mobile phone">mobile phone</a> <a class="mw-redirect" href="/wiki/Short_message_service" title="Short message service">short message service</a> offered by <a href="/wiki/Google" title="Google">Google</a> in several countries, including the USA, Japan, Canada, India,<sup class="reference" id="cite_ref-67"><a href="#cite_note-67">[67]</a></sup> Pakistan and China. and previously the UK, Germany and Spain. It allowed search queries to be sent as a text message, with results sent as a reply, with no premium charge. It was turned off May 10, 2013.<sup class="reference" id="cite_ref-68"><a href="#cite_note-68">[68]</a></sup></li>
<li><a href="/wiki/IGoogle" title="IGoogle">iGoogle</a> (was Google Personalized Homepage) – Customizable homepage, which can contain <a href="/wiki/Web_feed" title="Web feed">Web feeds</a> and Google Gadgets, launched in May 2005, renamed to iGoogle on April 30, 2007 and used internally by Google, and discontinued on November 1, 2013.<sup class="reference" id="cite_ref-69"><a href="#cite_note-69">[69]</a></sup></li>
<li><a href="/wiki/Google_Checkout" title="Google Checkout">Google Checkout</a> – online payment processing service provided by Google aimed at simplifying the process of paying for online purchases. Webmasters can choose to implement Google Checkout as a form of payment. Merged into Google Wallet, and was discontinued on November 20, 2013.</li>
<li>My Maps, GIS tools for Google Maps</li>
</ul>

<h3><span class="mw-headline" id="Discontinued_in_2015">Discontinued in 2015</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=List_of_Google_products&amp;action=edit&amp;section=26" title="Edit section: Discontinued in 2015">edit</a><span class="mw-editsection-bracket">]</span></span></h3>
<ul>
<li><a href="/wiki/Google_Moderator" title="Google Moderator">Google Moderator</a> – A service that uses crowdsourcing to rank user-submitted questions, suggestions and ideas.<sup class="reference" id="cite_ref-72"><a href="#cite_note-72">[72]</a></sup></li>
<li><a class="mw-redirect" href="/wiki/Wildfire_by_Google" title="Wildfire by Google">Wildfire by Google</a> – Social media marketing software.<sup class="reference" id="cite_ref-73"><a href="#cite_note-73">[73]</a></sup></li>
<li><a href="/wiki/BebaPay" title="BebaPay">BebaPay</a> – A prepaid ticket payment system. BebaPay was discontinued on March 15, 2015.<sup class="reference" id="cite_ref-74"><a href="#cite_note-74">[74]</a></sup></li>
<li><a href="/wiki/Google_Helpouts" title="Google Helpouts">Google Helpouts</a> – Hangout-based live video chat with experts. Helpouts was discontinued on April 20, 2015.<sup class="reference" id="cite_ref-75"><a href="#cite_note-75">[75]</a></sup></li>
</ul>
<h3><span class="mw-headline" id="Discontinued_in_2016">Discontinued in 2016</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=List_of_Google_products&amp;action=edit&amp;section=27" title="Edit section: Discontinued in 2016">edit</a><span class="mw-editsection-bracket">]</span></span></h3>
<ul>
<li><a class="mw-redirect" href="/wiki/Google_Code" title="Google Code">Google Code</a> – Open source code hosting. Google Code was discontinued on January 25, 2016.<sup class="reference" id="cite_ref-76"><a href="#cite_note-76">[76]</a></sup></li>
<li><a href="/wiki/Picasa" title="Picasa">Picasa</a> – photo organization and editing application, providing photo library options and simple effects. Also includes <a href="/wiki/Facial_recognition_system" title="Facial recognition system">Facial Recognition</a> and <a class="mw-redirect" href="/wiki/GeoTagging" title="GeoTagging">GeoTagging</a> features. (Supports Mac OS X, Windows 2000, Windows XP, Windows Vista, and Windows 7.) Replaced by Google Photos.<sup class="reference" id="cite_ref-77"><a href="#cite_note-77">[77]</a></sup></li>
<li><a class="new" href="/w/index.php?title=Google_Compare&amp;action=edit&amp;redlink=1" title="Google Compare (page does not exist)">Google Compare</a> - U.S. and U.K. comparison-shopping site for auto insurance, credit cards, and mortgages<sup class="reference" id="cite_ref-78"><a href="#cite_note-78">[78]</a></sup></li>
<li>Google Showtimes – Specialized search engine that obtains show times of films near a user-entered location and provides reviews of films compiled from several different websites. It was discontinued on November 1, 2016.<sup class="reference" id="cite_ref-79"><a href="#cite_note-79">[79]</a></sup></li>
<li><a href="/wiki/MyTracks" title="MyTracks">MyTracks</a> - GPS Logging application for Android. The application uses a device's GPS to collect data, allowing real-time review of path, speed, distance, and elevation.<sup class="reference" id="cite_ref-80"><a href="#cite_note-80">[80]</a></sup></li>
<li><a href="/wiki/Project_Ara" title="Project Ara">Project Ara</a> - an "initiative to build a phone with interchangeable modules for various components like cameras and batteries"<sup class="reference" id="cite_ref-81"><a href="#cite_note-81">[81]</a></sup> was suspended to "streamline the company's seemingly disorganized product lineup."<sup class="reference" id="cite_ref-82"><a href="#cite_note-82">[82]</a></sup> on September 2, 2016.</li>
<li><a href="/wiki/Panoramio" title="Panoramio">Panoramio</a> - A <a href="/wiki/Geotagging" title="Geotagging">geolocation-oriented</a> <a class="mw-redirect" href="/wiki/Photo_sharing" title="Photo sharing">photo sharing</a> website created in 2005. It was announced in September 2014 that Panoramio would be closed and be migrated to Google Maps Views. In October 2016, Google announced that Panoramio would be closed down on November 4, 2016.<sup class="reference" id="cite_ref-83"><a href="#cite_note-83">[83]</a></sup></li>
</ul>
<h3><span class="mw-headline" id="Other">Other</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=List_of_Google_products&amp;action=edit&amp;section=28" title="Edit section: Other">edit</a><span class="mw-editsection-bracket">]</span></span></h3>

<ul>
<li><a class="mw-redirect" href="/wiki/Google_Places" title="Google Places">Hotpot</a> – was a local recommendation engine that allowed people to rate restaurants, hotels etc. and share them with friends. Loading google.com/hotpot now gives an Error 404 page, and as Google explained,<sup class="reference" id="cite_ref-84"><a href="#cite_note-84">[84]</a></sup> the project has moved to the Google Places service.</li>
<li><a href="/wiki/Google_Base" title="Google Base">Google Base</a> – Google submission database that enabled content owners to submit content, have it hosted and made searchable. Information within the database was organized using attributes.</li>
<li><b>Blogger Web Comments</b> (<a href="/wiki/Firefox" title="Firefox">Firefox</a> only) – Displays related comments from other <a href="/wiki/Blogger_(service)" title="Blogger (service)">Blogger</a> users.</li>
<li><b>City Tours</b> – overlay to Maps that shows interesting tours within a city</li>
<li><b>Dashboard Widgets for Mac</b> (<a class="mw-redirect" href="/wiki/Mac_OS_X_Dashboard" title="Mac OS X Dashboard">Mac OS X Dashboard</a> Widgets) – Collection of mini-applications including <a href="/wiki/Gmail" title="Gmail">Gmail</a>, <a href="/wiki/Blogger_(service)" title="Blogger (service)">Blogger</a> and Search History.</li>
<li><b>Free Search</b> – free code to embed site/web search into a user's website. Discontinued; replaced by <a href="/wiki/Google_Custom_Search" title="Google Custom Search">Google Custom Search</a>.</li>
<li><b>Joga Bonito</b> – <a class="mw-redirect" href="/wiki/Soccer" title="Soccer">Soccer</a> community site, similar to services such as <a class="mw-redirect" href="/wiki/MySpace" title="MySpace">MySpace</a>, in that each member had a profile, and could join groups based on shared interests. The service allowed a user to meet other fans, create games and clubs, access athletes from Nike, and watch and upload video clips and photos and personal company files.</li>
<li><b>Local</b> – Local listings service, before it was integrated with mapping. The merged service was then called Google Local, which was further renamed to Google Maps due to popular demand. Google Local still exists, but only for Google Mobile Search.</li>
<li><b>Mashup Editor</b> – (deprecated as of January 14, 2009) Web Mashup creation with publishing facilities, syntax highlighting, debugging.</li>
<li><b>MK-14</b> – 4U rack mounted server for Google Radio Automation system. Google has sold its Google Radio Automation business to WideOrbit Inc.<sup class="reference" id="cite_ref-85"><a href="#cite_note-85">[85]</a></sup></li>
<li><a href="/wiki/Google_Trends#Google_Music_Trends" title="Google Trends">Google Music Trends</a> – Music ranking of songs played with iTunes, Winamp, Windows Media Player and Yahoo Music. Trends were generated by <a href="/wiki/Google_Talk" title="Google Talk">Google Talk</a>'s "share your music status" feature.</li>
<li><a class="mw-redirect" href="/wiki/Personalized_Search" title="Personalized Search">Personalized Search</a> – Search results personalization, now fully merged with Google Accounts and Web History.</li>
<li><b>Photos Screensaver</b> – Slideshow <a href="/wiki/Screensaver" title="Screensaver">screensaver</a> as part of Google Pack, which displays images sourced from a <a class="mw-redirect" href="/wiki/Hard_disk" title="Hard disk">hard disk</a>, or through <a href="/wiki/RSS" title="RSS">RSS</a> and <a href="/wiki/Atom_(standard)" title="Atom (standard)">Atom</a> Web feeds.</li>
<li><b>Rebang</b> (Google China) – <a href="/wiki/Google_China" title="Google China">Google China</a>'s search trend site, similar to Google Zeitgeist. As of 2010<sup class="plainlinks noprint asof-tag update" style="display:none;"><a class="external text" href="//en.wikipedia.org/w/index.php?title=List_of_Google_products&amp;action=edit">[update]</a></sup>, part of Google Labs.<sup class="reference" id="cite_ref-86"><a href="#cite_note-86">[86]</a></sup><sup class="reference" id="cite_ref-87"><a href="#cite_note-87">[87]</a></sup></li>
<li><a class="mw-redirect" href="/wiki/Google_Refine" title="Google Refine">Google Refine</a> – Tool for data cleansing and processing. It is now independent from Google.</li>
<li><a href="/wiki/Google_SearchWiki" title="Google SearchWiki">Google SearchWiki</a></li>
<li><a href="/wiki/Google_Sidewiki" title="Google Sidewiki">Google Sidewiki</a> – browser sidebar and service that allows contributing and reading helpful information alongside any web page; went online on September 23, 2009.</li>
<li>Spreadsheets – <a href="/wiki/Spreadsheet" title="Spreadsheet">Spreadsheet</a> management application, before it was integrated with Writely to form <a class="mw-redirect" href="/wiki/Google_Docs_%26_Spreadsheets" title="Google Docs &amp; Spreadsheets">Google Docs &amp; Spreadsheets</a>. It was announced on 6 June 2006.</li>
<li><b>University Search</b> – Listings for search engines for university websites.</li>
<li><b>U.S. Government Search</b> – Search engine and Personalized Homepage that exclusively draws from sites with a <a href="/wiki/.gov" title=".gov">.gov</a> <a class="mw-redirect" href="/wiki/TLD" title="TLD">TLD</a>.</li>
<li><a class="mw-redirect" href="/wiki/Google_Video#Google_Video_Player" title="Google Video">Video Player</a> (Mac OS X, Windows 2000, XP): Standalone desktop application that allows viewing videos from <a class="mw-redirect" href="/wiki/Google_Video" title="Google Video">Google Video</a>.</li>
<li><a class="mw-redirect" href="/wiki/Voice_Search" title="Voice Search">Voice Search</a> – automated voice system for searching the Web using the telephone. Now called Google Voice Local Search, it is currently integrated on the Google Mobile web site.</li>
<li><a href="/wiki/Google_Web_Accelerator" title="Google Web Accelerator">Web Accelerator</a> – application that used various caching technologies to increase load speed of web pages. It supported Windows 2000 SP3+, XP, and Vista, but is no longer available for download.</li>
<li><b>Google X</b> – Re-designed Google search homepage. It appeared in Google Labs, but was removed the following day for undisclosed reasons. It consisted of the traditional <a href="/wiki/Google" title="Google">Google</a> search bar, but it was made to look like the <a class="mw-redirect" href="/wiki/Dock_(computing)" title="Dock (computing)">Dock</a> <a class="mw-redirect" href="/wiki/Interface_(computer_science)" title="Interface (computer science)">user interface</a> feature of <a href="/wiki/Apple_Inc." title="Apple Inc.">Apple</a>'s <a class="mw-redirect" href="/wiki/Mac_OS_X" title="Mac OS X">Mac OS X</a> <a href="/wiki/Operating_system" title="Operating system">operating system</a>.<sup class="reference" id="cite_ref-88"><a href="#cite_note-88">[88]</a></sup> Google did not release any official statement as to why the project was shut down.</li>
<li><a class="mw-redirect" href="/wiki/Accessible_Search" title="Accessible Search">Accessible Search</a> – search engine for the blind and visually impaired. It prioritizes <a href="/wiki/Usability" title="Usability">usable</a> and <a href="/wiki/Web_accessibility" title="Web accessibility">accessible</a> web sites in the search results, so users incur minimal distractions when browsing.</li>
<li><a href="/wiki/Google_Blog_Search" title="Google Blog Search">Blog Search</a> – <a class="mw-redirect" href="/wiki/Weblog" title="Weblog">weblog</a> search engine, with a continuously-updated search index. Results include all blogs, not just those published through Blogger. Results can be viewed and filtered by date.</li>
<li><a href="/wiki/Google_Quick_Search_Box" title="Google Quick Search Box">Quick Search Box</a> – search box, based on <a href="/wiki/Quicksilver_(software)" title="Quicksilver (software)">Quicksilver</a>, which allows easily viewing installed applications or performing online searches. (Supports Mac OS X.)</li>
<li><b>Visigami</b> – image search application screen saver that searches files from <a href="/wiki/Google_Images" title="Google Images">Google Images</a>, <a href="/wiki/Picasa" title="Picasa">Picasa</a> and <a href="/wiki/Flickr" title="Flickr">Flickr</a>. (Supports <a href="/wiki/Mac_OS_X_Leopard" title="Mac OS X Leopard">Mac OS X Leopard</a>.)</li>
<li><a href="/wiki/Google_WiFi#Wireless_access" title="Google WiFi">Wireless access</a> – <a class="mw-redirect" href="/wiki/VPN" title="VPN">VPN</a> client for <a href="/wiki/Google_WiFi" title="Google WiFi">Google WiFi</a> users, whose equipment does not support <a href="/wiki/Wi-Fi_Protected_Access" title="Wi-Fi Protected Access">WPA</a> or <a class="mw-redirect" href="/wiki/802.1x" title="802.1x">802.1x</a> protocols. (Supports Windows 2000 and Windows XP)</li>
</ul>
<h3><span class="mw-headline" id="Scheduled_to_be_discontinued">Scheduled to be discontinued</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=List_of_Google_products&amp;action=edit&amp;section=29" title="Edit section: Scheduled to be discontinued">edit</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>Applications that are no longer in development, and scheduled to be discontinued in the future:</p>
<ul>
<li><a href="/wiki/Google_Map_Maker" title="Google Map Maker">Google Map Maker</a> – Online map editor with browser interface. To be replaced by editing features in Google Maps. To be discontinued in March 2017.<sup class="reference" id="cite_ref-89"><a href="#cite_note-89">[89]</a></sup></li>
<li><a href="/wiki/Google_Search_Appliance" title="Google Search Appliance">Google Search Appliance</a> – A rack mounted device used to index documents. End of hardware sales in 2017 and planned shutdown in 2018.<sup class="reference" id="cite_ref-90"><a href="#cite_note-90">[90]</a></sup></li>
</ul>
