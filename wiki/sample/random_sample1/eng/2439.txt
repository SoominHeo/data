<h1 class="firstHeading" id="firstHeading" lang="en">Member states of the League of Nations</h1>
<div class="mw-body-content" id="bodyContent">
<div id="siteSub">From Wikipedia, the free encyclopedia</div>
<div id="contentSub"></div>
<div class="mw-jump" id="jump-to-nav">
					Jump to:					<a href="#mw-head">navigation</a>, 					<a href="#p-search">search</a>
</div>
<div class="mw-content-ltr" dir="ltr" id="mw-content-text" lang="en"><script>function mfTempOpenSection(id){var block=document.getElementById("mf-section-"+id);block.className+=" open-block";block.previousSibling.className+=" open-block";}</script><p>Between 1920 and 1939, a total of 63 countries became <b><a href="/wiki/Member_state" title="Member state">member states</a> of the <a href="/wiki/League_of_Nations" title="League of Nations">League of Nations</a></b>. The Covenant forming the League of Nations was included in the <a href="/wiki/Treaty_of_Versailles" title="Treaty of Versailles">Treaty of Versailles</a> and came into force on 10 January 1920. The League of Nations was dissolved on 18 April 1946, when its assets and responsibilities were transferred to the United Nations.</p>
<p>The League’s greatest extent was from 28 September 1934 (when <a href="/wiki/Ecuador" title="Ecuador">Ecuador</a> joined) to 23 February 1935 (when <a href="/wiki/Paraguay" title="Paraguay">Paraguay</a> withdrew) with 58 members. At this time, only <a href="/wiki/Costa_Rica" title="Costa Rica">Costa Rica</a> (22 January 1925), <a href="/wiki/Brazil" title="Brazil">Brazil</a> (14 June 1926), the <a href="/wiki/Empire_of_Japan" title="Empire of Japan">Empire of Japan</a> (27 March 1933) and <a href="/wiki/Germany" title="Germany">Germany</a> (19 September 1933) had withdrawn and only <a href="/wiki/Egypt" title="Egypt">Egypt</a> was left to join (on the 26 May 1937). The members (listed from earliest joining and alphabetically if they joined on the same day) at this time were <a href="/wiki/Argentina" title="Argentina">Argentina</a>, <a href="/wiki/Australia" title="Australia">Australia</a>, <a href="/wiki/Belgium" title="Belgium">Belgium</a>, <a href="/wiki/Bolivia" title="Bolivia">Bolivia</a>, the <a href="/wiki/British_Empire" title="British Empire">British Empire</a>, <a href="/wiki/Canada" title="Canada">Canada</a>, <a href="/wiki/Chile" title="Chile">Chile</a>, <a class="mw-redirect" href="/wiki/Republic_of_China_(1912-49)" title="Republic of China (1912-49)">China</a>, <a href="/wiki/Colombia" title="Colombia">Colombia</a>, <a href="/wiki/Cuba" title="Cuba">Cuba</a>, <a href="/wiki/Czechoslovakia" title="Czechoslovakia">Czechoslovakia</a>, <a href="/wiki/Denmark" title="Denmark">Denmark</a>, <a href="/wiki/El_Salvador" title="El Salvador">El Salvador</a>, <a href="/wiki/France" title="France">France</a>, <a href="/wiki/Greece" title="Greece">Greece</a>, <a href="/wiki/Guatemala" title="Guatemala">Guatemala</a>, <a href="/wiki/Haiti" title="Haiti">Haiti</a>, <a href="/wiki/Honduras" title="Honduras">Honduras</a>, <a href="/wiki/British_Raj" title="British Raj">India</a>, <a href="/wiki/Irish_Free_State" title="Irish Free State">Irish Free State</a>, <a href="/wiki/Ethiopia" title="Ethiopia">Ethiopia</a>, <a href="/wiki/Italy" title="Italy">Italy</a>, <a href="/wiki/Liberia" title="Liberia">Liberia</a>, the <a href="/wiki/Netherlands" title="Netherlands">Netherlands</a>, <a href="/wiki/New_Zealand" title="New Zealand">New Zealand</a>, <a href="/wiki/Nicaragua" title="Nicaragua">Nicaragua</a>, <a href="/wiki/Norway" title="Norway">Norway</a>, <a href="/wiki/Panama" title="Panama">Panama</a>, <a href="/wiki/Paraguay" title="Paraguay">Paraguay</a>, <a class="mw-redirect" href="/wiki/Persia" title="Persia">Persia</a>/<a href="/wiki/Iran" title="Iran">Iran</a>, <a href="/wiki/Peru" title="Peru">Peru</a>, <a href="/wiki/Poland" title="Poland">Poland</a>, <a href="/wiki/Portugal" title="Portugal">Portugal</a>, <a href="/wiki/Romania" title="Romania">Romania</a>, <a class="mw-redirect" href="/wiki/Siam" title="Siam">Siam</a>, <a href="/wiki/Spain" title="Spain">Spain</a>, <a href="/wiki/Sweden" title="Sweden">Sweden</a>, <a href="/wiki/Switzerland" title="Switzerland">Switzerland</a>, the <a href="/wiki/Union_of_South_Africa" title="Union of South Africa">Union of South Africa</a>, <a href="/wiki/Uruguay" title="Uruguay">Uruguay</a>, <a href="/wiki/Yugoslavia" title="Yugoslavia">Yugoslavia</a>, <a href="/wiki/Venezuela" title="Venezuela">Venezuela</a>, <a href="/wiki/Austria" title="Austria">Austria</a>, <a href="/wiki/Bulgaria" title="Bulgaria">Bulgaria</a>, <a href="/wiki/Finland" title="Finland">Finland</a>, <a href="/wiki/Luxembourg" title="Luxembourg">Luxembourg</a>, <a href="/wiki/Albania" title="Albania">Albania</a>, <a href="/wiki/Estonia" title="Estonia">Estonia</a>, <a href="/wiki/Latvia" title="Latvia">Latvia</a>, <a href="/wiki/Lithuania" title="Lithuania">Lithuania</a>, <a href="/wiki/Hungary" title="Hungary">Hungary</a>, <a href="/wiki/Irish_Free_State" title="Irish Free State">Ireland</a>, the <a href="/wiki/Dominican_Republic" title="Dominican Republic">Dominican Republic</a>, <a href="/wiki/Mexico" title="Mexico">Mexico</a>, <a href="/wiki/Turkey" title="Turkey">Turkey</a>, <a href="/wiki/Iraq" title="Iraq">Iraq</a>, the <a class="mw-redirect" href="/wiki/Union_of_Soviet_Socialist_Republics" title="Union of Soviet Socialist Republics">Union of Soviet Socialist Republics</a>, <a href="/wiki/Afghanistan" title="Afghanistan">Afghanistan</a>, <a href="/wiki/British_Raj" title="British Raj">British Raj</a>, <a href="/wiki/Syria" title="Syria">Syria</a> and <a href="/wiki/Ecuador" title="Ecuador">Ecuador</a>.</p>
<p>Of the 42 founding members, 23 (or 24, counting [Free France]) remained members until the League of Nations was dissolved in 1946. A further 21 countries joined between 1920 and 1937, but 7 left, withdrew or were expelled before 1946.</p>
<p>Countries are listed under the year in which they joined. The word "withdrew" indicates that a country left of its own choice. The word "left" indicates a country that ceased to exist because of <a href="/wiki/Annexation" title="Annexation">annexation</a> by Germany, Italy or the USSR. The <a class="mw-redirect" href="/wiki/USSR" title="USSR">USSR</a> was the only country to be expelled by the League.</p>
<p>Despite formulating the concept and signing the Covenant, the <a href="/wiki/United_States" title="United States">United States</a> never joined the League of Nations.</p>
