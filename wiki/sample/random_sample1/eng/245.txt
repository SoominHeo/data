<p>In <a href="/wiki/Computing" title="Computing">computing</a>, <b>DOC</b> or <b>doc</b> (an abbreviation of '<a href="/wiki/Document" title="Document">document</a>') is a <a href="/wiki/Filename_extension" title="Filename extension">filename extension</a> for <a class="mw-redirect" href="/wiki/Word_processing" title="Word processing">word processing</a> documents, most commonly in the proprietary <i><a href="/wiki/Microsoft_Word" title="Microsoft Word">Microsoft Word</a> <a href="/wiki/Binary_file" title="Binary file">Binary</a> <a href="/wiki/Document_file_format" title="Document file format">File Format</a></i>.<sup class="reference" id="cite_ref-ms-doc-specification_4-1"><a href="#cite_note-ms-doc-specification-4">[4]</a></sup> Historically, the extension was used for documentation in <a href="/wiki/Plain_text" title="Plain text">plain text</a>, particularly of programs or computer hardware on a wide range of <a href="/wiki/Operating_system" title="Operating system">operating systems</a>. During the 1980s, <a href="/wiki/WordPerfect" title="WordPerfect">WordPerfect</a> used DOC as the extension of their <a href="/wiki/Proprietary_format" title="Proprietary format">proprietary format</a>. Later, in 1983, <a href="/wiki/Microsoft" title="Microsoft">Microsoft</a> chose to use the DOC extension for their proprietary Microsoft Word format. These uses for the extension have largely disappeared from the <a href="/wiki/Personal_computer" title="Personal computer">PC</a> world.</p>

<h2><span class="mw-headline" id="Microsoft_Word_Binary_File_Format">Microsoft Word Binary File Format</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Doc_(computing)&amp;action=edit&amp;section=1" title="Edit section: Microsoft Word Binary File Format">edit</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>Binary DOC <a href="/wiki/Computer_file" title="Computer file">files</a> often contain more <a class="mw-redirect" href="/wiki/Text_formatting" title="Text formatting">text formatting</a> information (as well as <a class="mw-redirect" href="/wiki/Script_(computer_programming)" title="Script (computer programming)">scripts</a> and undo information) than some other <a href="/wiki/Document_file_format" title="Document file format">document file formats</a> like <a href="/wiki/Rich_Text_Format" title="Rich Text Format">Rich Text Format</a> and <a class="mw-redirect" href="/wiki/HyperText_Markup_Language" title="HyperText Markup Language">HyperText Markup Language</a>, but are usually less widely <a href="/wiki/Computer_compatibility" title="Computer compatibility">compatible</a>.</p>
<p>The DOC files created with Microsoft Word versions differ. Microsoft Word versions up to Word 97 used a different format from Microsoft Word 97 - 2003.</p>
<p>In <a class="mw-redirect" href="/wiki/Microsoft_Word_2007" title="Microsoft Word 2007">Microsoft Word 2007</a> and later, the binary file format was replaced as the default format by the <a href="/wiki/Office_Open_XML" title="Office Open XML">Office Open XML</a> format, though Microsoft Word can still produce DOC files.</p>
<h3><span class="mw-headline" id="Application_support">Application support</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Doc_(computing)&amp;action=edit&amp;section=2" title="Edit section: Application support">edit</a><span class="mw-editsection-bracket">]</span></span></h3>
<div class="hatnote" role="note">See also: <a href="/wiki/Comparison_of_word_processors" title="Comparison of word processors">Comparison of word processors</a></div>
<p>The DOC format is native to Microsoft Word. Other <a href="/wiki/Word_processor" title="Word processor">word processors</a>, such as <a class="mw-redirect" href="/wiki/OpenOffice.org_Writer" title="OpenOffice.org Writer">OpenOffice.org Writer</a>, <a href="/wiki/IBM_Lotus_Symphony" title="IBM Lotus Symphony">IBM Lotus Symphony</a>, <a class="mw-redirect" href="/wiki/Pages_(iWork)" title="Pages (iWork)">Apple Pages</a> and <a href="/wiki/AbiWord" title="AbiWord">AbiWord</a>, can also create and read DOC files, although with some limitations. <a class="mw-redirect" href="/wiki/Command_line_interface" title="Command line interface">Command line</a> programs for <a href="/wiki/Unix-like" title="Unix-like">Unix-like</a> operating systems that can convert files from the DOC format to plain text or other standard formats include the <a href="/wiki/Wv_(software)" title="Wv (software)">wv</a> library, which itself is used directly by <a href="/wiki/AbiWord" title="AbiWord">AbiWord</a>.</p>
<h3><span class="mw-headline" id="Specification">Specification</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Doc_(computing)&amp;action=edit&amp;section=3" title="Edit section: Specification">edit</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>Because the DOC file format was a closed specification for many years, inconsistent handling of the format persists and may cause some loss of formatting information when handling the same file with multiple word processing programs. Some specifications for Microsoft Office 97 binary file formats were published in 1997 under a restrictive license, but these specifications were removed from online download in 1999.<sup class="reference" id="cite_ref-5"><a href="#cite_note-5">[5]</a></sup><sup class="reference" id="cite_ref-6"><a href="#cite_note-6">[6]</a></sup><sup class="reference" id="cite_ref-7"><a href="#cite_note-7">[7]</a></sup><sup class="reference" id="cite_ref-8"><a href="#cite_note-8">[8]</a></sup> Specifications of later versions of Microsoft Office binary file formats were not publicly available. The DOC format specification was available from Microsoft on request<sup class="reference" id="cite_ref-9"><a href="#cite_note-9">[9]</a></sup> since 2006<sup class="reference" id="cite_ref-10"><a href="#cite_note-10">[10]</a></sup> under restrictive <a href="/wiki/Reasonable_and_non-discriminatory_licensing" title="Reasonable and non-discriminatory licensing">RAND-Z</a> terms until February 2008. <a href="/wiki/Sun_Microsystems" title="Sun Microsystems">Sun Microsystems</a> and <a href="/wiki/OpenOffice.org" title="OpenOffice.org">OpenOffice.org</a> <a href="/wiki/Reverse_engineering" title="Reverse engineering">reverse engineered</a> the file format.<sup class="reference" id="cite_ref-11"><a href="#cite_note-11">[11]</a></sup> On February 15, 2008, Microsoft released a .DOC format specification<sup class="reference" id="cite_ref-ms-doc-specification_4-2"><a href="#cite_note-ms-doc-specification-4">[4]</a></sup><sup class="reference" id="cite_ref-12"><a href="#cite_note-12">[12]</a></sup><sup class="reference" id="cite_ref-13"><a href="#cite_note-13">[13]</a></sup> under the <a href="/wiki/Microsoft_Open_Specification_Promise" title="Microsoft Open Specification Promise">Microsoft Open Specification Promise</a>.<sup class="reference" id="cite_ref-14"><a href="#cite_note-14">[14]</a></sup><sup class="reference" id="cite_ref-15"><a href="#cite_note-15">[15]</a></sup> However, this specification does not describe all of the features used by DOC format and reverse engineered work remains necessary.<sup class="reference" id="cite_ref-16"><a href="#cite_note-16">[16]</a></sup> Since 2008 the specification has been updated several times; the last change was made in September 2015.</p>
<h2><span class="mw-headline" id="Other_file_formats">Other file formats</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Doc_(computing)&amp;action=edit&amp;section=4" title="Edit section: Other file formats">edit</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>Some historical documentations may use the DOC filename extension for plain-text file format. The DOC filename extension was also used in historical versions of WordPerfect for its proprietary format.</p>
<p>Some software applications use the name "DOC" in combination with other words (such as the name of software manufacturer) for different file formats. As an example, on the <a href="/wiki/Palm_OS" title="Palm OS">Palm OS</a>, DOC is shorthand for <a class="new" href="/w/index.php?title=PalmDoc&amp;action=edit&amp;redlink=1" title="PalmDoc (page does not exist)">PalmDoc</a>, a completely unrelated format (commonly using PDB filename extension) used to encode text files such as <a class="mw-redirect" href="/wiki/Ebook" title="Ebook">ebooks</a>.</p>
