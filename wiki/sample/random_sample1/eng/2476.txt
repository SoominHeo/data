<p>The <b>retroflex approximant</b> is a type of <a href="/wiki/Consonant" title="Consonant">consonant</a> used in some <a href="/wiki/Language" title="Language">languages</a>. The symbol in the <a href="/wiki/International_Phonetic_Alphabet" title="International Phonetic Alphabet">International Phonetic Alphabet</a> that represents this sound is ⟨<span class="IPA" title="Representation in the International Phonetic Alphabet (IPA)">ɻ</span>⟩, and the equivalent <a href="/wiki/X-SAMPA" title="X-SAMPA">X-SAMPA</a> symbol is <kbd>r\`</kbd>. The IPA symbol is a turned lowercase letter <i>r</i> with a rightward hook protruding from the lower right of the letter.</p>

<h2><span class="mw-headline" id="Features">Features</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Retroflex_approximant&amp;action=edit&amp;section=1" title="Edit section: Features">edit</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>Features of the retroflex approximant:</p>
<ul>
<li>Its <a href="/wiki/Manner_of_articulation" title="Manner of articulation">manner of articulation</a> is <a href="/wiki/Approximant_consonant" title="Approximant consonant">approximant</a>, which means it is produced by narrowing the vocal tract at the place of articulation, but not enough to produce a <a href="/wiki/Turbulence" title="Turbulence">turbulent airstream</a>.</li>
<li>Its <a href="/wiki/Place_of_articulation" title="Place of articulation">place of articulation</a> is <a href="/wiki/Retroflex_consonant" title="Retroflex consonant">retroflex</a>, which prototypically means it is articulated <a href="/wiki/Subapical_consonant" title="Subapical consonant">subapical</a> (with the tip of the tongue curled up), but more generally, it means that it is <a href="/wiki/Postalveolar_consonant" title="Postalveolar consonant">postalveolar</a> without being <a href="/wiki/Palatalization_(phonetics)" title="Palatalization (phonetics)">palatalized</a>. That is, besides the prototypical sub-apical articulation, the tongue contact can be <a href="/wiki/Apical_consonant" title="Apical consonant">apical</a> (pointed) or <a href="/wiki/Laminal_consonant" title="Laminal consonant">laminal</a> (flat).</li>
<li>Its <a href="/wiki/Phonation" title="Phonation">phonation</a> is voiced, which means the vocal cords vibrate during the articulation.</li>
<li>It is an <a href="/wiki/Oral_consonant" title="Oral consonant">oral consonant</a>, which means air is allowed to escape through the mouth only.</li>
<li>It is a <a href="/wiki/Central_consonant" title="Central consonant">central consonant</a>, which means it is produced by directing the airstream along the center of the tongue, rather than to the sides.</li>
<li>The <a href="/wiki/Airstream_mechanism" title="Airstream mechanism">airstream mechanism</a> is <a href="/wiki/Egressive_sound" title="Egressive sound">pulmonic</a>, which means it is articulated by pushing air solely with the <a href="/wiki/Lung" title="Lung">lungs</a> and <a href="/wiki/Thoracic_diaphragm" title="Thoracic diaphragm">diaphragm</a>, as in most sounds.</li>
</ul>
<h2><span class="mw-headline" id="Occurrence">Occurrence</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Retroflex_approximant&amp;action=edit&amp;section=2" title="Edit section: Occurrence">edit</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>The retroflex approximant occurs in <a href="/wiki/American_English" title="American English">American English</a>, <a href="/wiki/Hiberno-English" title="Hiberno-English">Hiberno-English</a>, <a href="/wiki/West_Country_English" title="West Country English">West Country English</a>, <a href="/wiki/Mandarin_Chinese" title="Mandarin Chinese">Mandarin Chinese</a>, <a class="mw-redirect" href="/wiki/Pashto_language" title="Pashto language">Pashto</a>, a few Brazilian Portuguese dialects and some languages of India such as <a href="/wiki/Tamil_language" title="Tamil language">Tamil</a> and <a href="/wiki/Malayalam" title="Malayalam">Malayalam</a>, as well as several <a href="/wiki/Australian_Aboriginal_languages" title="Australian Aboriginal languages">Australian Aboriginal</a> and <a href="/wiki/Indigenous_languages_of_the_Americas" title="Indigenous languages of the Americas">Indigenous South American</a> languages.</p>
