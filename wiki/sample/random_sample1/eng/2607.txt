<p><b>Gliese 758 B</b> (often shortened to <b>GJ 758 B</b>) is a <a href="/wiki/Brown_dwarf" title="Brown dwarf">brown dwarf</a> orbiting the <a class="mw-redirect" href="/wiki/G-type_star" title="G-type star">G-type</a> <a href="/wiki/Main_sequence" title="Main sequence">main sequence</a> <a href="/wiki/Star" title="Star">star</a> <a href="/wiki/Gliese_758" title="Gliese 758">Gliese 758</a>, located approximately 50 <a class="mw-redirect" href="/wiki/Light_year" title="Light year">light years</a> away, in the constellation <a class="mw-redirect" href="/wiki/Lyra_(constellation)" title="Lyra (constellation)">Lyra</a>.</p>
<p>Gliese 758 B was detected by <a class="mw-redirect" href="/wiki/Methods_of_detecting_extrasolar_planets#Direct_imaging" title="Methods of detecting extrasolar planets">direct imaging</a> using the <a class="mw-redirect" href="/wiki/Subaru_(telescope)#Instruments" title="Subaru (telescope)">HiCIAO</a> instrument of the <a class="mw-redirect" href="/wiki/Subaru_(telescope)" title="Subaru (telescope)">Subaru Telescope</a>.<sup class="reference" id="cite_ref-subaru-image_3-0"><a href="#cite_note-subaru-image-3">[3]</a></sup><sup class="reference" id="cite_ref-centdreams-news_4-0"><a href="#cite_note-centdreams-news-4">[4]</a></sup><sup class="reference" id="cite_ref-subaru-release_5-0"><a href="#cite_note-subaru-release-5">[5]</a></sup> The parameters have subsequently been revised in a follow-up study which also revealed that a putative second substellar object in the system was in fact an unrelated background star.<sup class="reference" id="cite_ref-Janson2011_6-0"><a href="#cite_note-Janson2011-6">[6]</a></sup></p>
