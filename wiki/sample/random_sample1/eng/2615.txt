<p>The <b>Golden Orchid</b> (<i><b>Cephalanthera falcata</b></i>) is a species of <a class="mw-redirect" href="/wiki/Orchid" title="Orchid">orchid</a>. It is native to <a href="/wiki/Japan" title="Japan">Japan</a>, <a href="/wiki/Korea" title="Korea">Korea</a>, and <a href="/wiki/China" title="China">China</a> (<a href="/wiki/Anhui" title="Anhui">Anhui</a>, <a href="/wiki/Fujian" title="Fujian">Fujian</a>, <a href="/wiki/Guangdong" title="Guangdong">Guangdong</a>, <a href="/wiki/Guangxi" title="Guangxi">Guangxi</a>, <a href="/wiki/Guizhou" title="Guizhou">Guizhou</a>, <a href="/wiki/Hubei" title="Hubei">Hubei</a>, <a href="/wiki/Hunan" title="Hunan">Hunan</a>, <a href="/wiki/Jiangsu" title="Jiangsu">Jiangsu</a>, <a href="/wiki/Jiangxi" title="Jiangxi">Jiangxi</a>, <a href="/wiki/Sichuan" title="Sichuan">Sichuan</a>, <a href="/wiki/Yunnan" title="Yunnan">Yunnan</a>).<sup class="reference" id="cite_ref-kimmy_1-1"><a href="#cite_note-kimmy-1">[1]</a></sup><sup class="reference" id="cite_ref-2"><a href="#cite_note-2">[2]</a></sup></p>
