<p>The <b>Kitakami Line</b> <span style="font-weight: normal">(<span class="t_nihongo_kanji" lang="ja" xml:lang="ja">北上線</span><span class="t_nihongo_comma" style="display:none">,</span> <i>Kitakami-sen</i><sup class="t_nihongo_help noprint"><a href="/wiki/Help:Installing_Japanese_character_sets" title="Help:Installing Japanese character sets"><span class="t_nihongo_icon" style="color: #00e; font: bold 80% sans-serif; text-decoration: none; padding: 0 .1em;">?</span></a></sup>)</span> is a rail line in <a href="/wiki/Japan" title="Japan">Japan</a>. Part of the <a href="/wiki/East_Japan_Railway_Company" title="East Japan Railway Company">East Japan Railway Company</a> (JR East) system, it connects Kitakami Station in <a href="/wiki/Kitakami,_Iwate" title="Kitakami, Iwate">Kitakami</a>, <a href="/wiki/Iwate_Prefecture" title="Iwate Prefecture">Iwate Prefecture</a> to Yokote Station in <a href="/wiki/Yokote,_Akita" title="Yokote, Akita">Yokote</a>, <a href="/wiki/Akita_Prefecture" title="Akita Prefecture">Akita Prefecture</a>, acting as a connecting line between the <a href="/wiki/%C5%8Cu_Main_Line" title="Ōu Main Line">Ōu Main Line</a> to the <a href="/wiki/T%C5%8Dhoku_Main_Line" title="Tōhoku Main Line">Tōhoku Main Line</a>. It connects with the <a href="/wiki/T%C5%8Dhoku_Shinkansen" title="Tōhoku Shinkansen">Tōhoku Shinkansen</a> and <a class="mw-redirect" href="/wiki/Tohoku_Main_Line" title="Tohoku Main Line">Tohoku Main Line</a> at Kitakami Station, and the <a href="/wiki/%C5%8Cu_Main_Line" title="Ōu Main Line">Ōu Main Line</a> at Yokote Station.</p>

<h2><span class="mw-headline" id="History">History</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Kitakami_Line&amp;action=edit&amp;section=1" title="Edit section: History">edit</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>The line opened in sections from both Kitakami and Yokote between 1920 and 1924.</p>
<p>In 1962 a 15km deviation was built in conjunction with the construction of the <a href="/wiki/Yuda_Dam" title="Yuda Dam">Yuda Dam</a>, which added 0.8km to the length of the line.</p>
<p>CTC signalling was commissioned in 1994, and Freight services ceased in 2010.</p>
