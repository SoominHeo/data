<p>The <b>Namba Line</b> <span style="font-weight: normal">(<span class="t_nihongo_kanji" lang="ja" xml:lang="ja">難波線</span><span class="t_nihongo_comma" style="display:none">,</span> <i>Nanba Sen</i><sup class="t_nihongo_help noprint"><a href="/wiki/Help:Installing_Japanese_character_sets" title="Help:Installing Japanese character sets"><span class="t_nihongo_icon" style="color: #00e; font: bold 80% sans-serif; text-decoration: none; padding: 0 .1em;">?</span></a></sup>)</span> is an underground <a class="mw-redirect" href="/wiki/Railway" title="Railway">railway</a> line owned by the <a href="/wiki/Kintetsu_Railway" title="Kintetsu Railway">Kintetsu Railway</a> in <a href="/wiki/Osaka" title="Osaka">Osaka</a>, <a href="/wiki/Japan" title="Japan">Japan</a>, and opened in 1970. It is an extension of the <a href="/wiki/Osaka_Line" title="Osaka Line">Osaka Line</a> and the <a href="/wiki/Nara_Line_(Kintetsu)" title="Nara Line (Kintetsu)">Nara Line</a>. The line is parallel to the <a href="/wiki/Osaka_Municipal_Subway" title="Osaka Municipal Subway">Osaka Municipal Subway</a> <a href="/wiki/Sennichimae_Line" title="Sennichimae Line">Sennichimae Line</a> (<a href="/wiki/Namba_Station" title="Namba Station">Namba</a> - <a class="mw-redirect" href="/wiki/Tanimachi_Kyuchome_Station" title="Tanimachi Kyuchome Station">Tanimachi Kyuchome</a>). Along with the commuter trains of the Nara Line and the Hanshin Namba Line, some intercity trains including the "Urban Liner" connecting Osaka and <a href="/wiki/Nagoya" title="Nagoya">Nagoya</a>, also run on the line.</p>
<h2><span class="mw-headline" id="Stations">Stations</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Kintetsu_Namba_Line&amp;action=edit&amp;section=1" title="Edit section: Stations">edit</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>There are three <a class="mw-redirect" href="/wiki/Railway_stations" title="Railway stations">railway stations</a> on the Namba Line:</p>
<ul>
<li>A01: <a class="mw-redirect" href="/wiki/Osaka_Namba_Station" title="Osaka Namba Station">Osaka Namba</a></li>
<li>A02: <a href="/wiki/Kintetsu_Nippombashi_Station" title="Kintetsu Nippombashi Station">Kintetsu Nippombashi</a></li>
<li>A03: <a class="mw-redirect" href="/wiki/Osaka_Uehommachi_Station" title="Osaka Uehommachi Station">Osaka Uehommachi</a> (tracks connected to the Osaka and Nara Lines)</li>
</ul>
<p>The Limited Express trains pass Kintetsu Nippombashi Station. Other trains stop at all stations.</p>
