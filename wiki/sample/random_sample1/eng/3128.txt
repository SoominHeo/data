<p><b>No Mercy</b> was a <a href="/wiki/Pay-per-view" title="Pay-per-view">pay-per-view</a> produced by the <a href="/wiki/WWE" title="WWE">World Wrestling Federation</a> exclusively for the <a href="/wiki/United_Kingdom" title="United Kingdom">United Kingdom</a>. It took place on May 16, 1999 at the <a class="mw-redirect" href="/wiki/Manchester_Evening_News_Arena" title="Manchester Evening News Arena">Manchester Evening News Arena</a> in <a href="/wiki/Manchester" title="Manchester">Manchester, England</a>. It was released on DVD in the UK and Europe on July 12, 2010, in a set also including <a class="mw-redirect" href="/wiki/Capital_Carnage" title="Capital Carnage">Capital Carnage</a> as part of the WWE's <a href="/wiki/WWE_Tagged_Classics" title="WWE Tagged Classics">Tagged Classics</a> range released by <a class="mw-redirect" href="/wiki/Silvervision" title="Silvervision">Silvervision</a>, without any edits to the original content, most notably keeping all mentions and appearances of the WWF logo intact and un-blurred.</p>

<h2><span class="mw-headline" id="Event">Event</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=No_Mercy_(UK)&amp;action=edit&amp;section=1" title="Edit section: Event">edit</a><span class="mw-editsection-bracket">]</span></span></h2>
<h3><span class="mw-headline" id="Preliminary_matches">Preliminary matches</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=No_Mercy_(UK)&amp;action=edit&amp;section=2" title="Edit section: Preliminary matches">edit</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>The first match was a singles match between <a class="mw-redirect" href="/wiki/Gurjit_Singh" title="Gurjit Singh">Tiger Ali Singh</a> and <a href="/wiki/Gillberg_(wrestler)" title="Gillberg (wrestler)">Gillberg</a>. Singh quickly pinned Gillberg after a <a href="/wiki/Cutter_(professional_wrestling)" title="Cutter (professional wrestling)">reverse neckbreaker</a>.<sup class="reference" id="cite_ref-WWF_PPV_Results_1-0"><a href="#cite_note-WWF_PPV_Results-1">[1]</a></sup></p>
<p>The following match was between The <a class="mw-redirect" href="/wiki/Ministry_of_Darkness" title="Ministry of Darkness">Ministry of Darkness</a> (<a href="/wiki/Viscera_(wrestler)" title="Viscera (wrestler)">Viscera</a>, <a href="/wiki/Ron_Simmons" title="Ron Simmons">Faarooq</a> and <a href="/wiki/John_Layfield" title="John Layfield">Bradshaw</a>) and <a href="/wiki/The_Brood_(professional_wrestling)" title="The Brood (professional wrestling)">The Brood</a> (<a href="/wiki/Gangrel_(wrestler)" title="Gangrel (wrestler)">Gangrel</a>, <a href="/wiki/Edge_(wrestler)" title="Edge (wrestler)">Edge</a>, and <a class="mw-redirect" href="/wiki/Christian_(wrestler)" title="Christian (wrestler)">Christian</a>). Bradshaw pinned Gangrel for the victory after a <a href="/wiki/Professional_wrestling_attacks#Clothesline" title="Professional wrestling attacks">Clothesline From Hell</a>.</p>
<p>The third match saw <a href="/wiki/Steve_Blackman" title="Steve Blackman">Steve Blackman</a> defeat <a class="mw-redirect" href="/wiki/Darren_Drozdov" title="Darren Drozdov">Droz</a> by submission using a <a href="/wiki/Professional_wrestling_holds#Triangle_choke" title="Professional wrestling holds">Triangle Choke</a>. Following that match saw <a href="/wiki/Kane_(wrestler)" title="Kane (wrestler)">Kane</a> competing against <a href="/wiki/Dennis_Knight" title="Dennis Knight">Mideon</a>. Mideon was <a href="/wiki/Professional_wrestling#Disqualification" title="Professional wrestling">disqualified</a> after several <a class="mw-redirect" href="/wiki/Corporate_Ministry" title="Corporate Ministry">Corporate Ministry</a> members interfered,<sup class="reference" id="cite_ref-WWF_PPV_Results_1-1"><a href="#cite_note-WWF_PPV_Results-1">[1]</a></sup> resulting in Kane being declared the winner.</p>
<p>The next planned match was between <a href="/wiki/Sable_(wrestler)" title="Sable (wrestler)">Sable</a> and <a class="mw-redirect" href="/wiki/Terri_Poch" title="Terri Poch">Tori</a>. <a href="/wiki/Nicole_Bass" title="Nicole Bass">Nicole Bass</a> substituted for her and the match ended quickly after Bass performed a <a href="/wiki/Chokeslam" title="Chokeslam">Chokeslam</a> on Tori. This would be Sable's last appearance in the WWF for nearly four years.</p>
<p>The sixth event of the night was a <a href="/wiki/WWE_European_Championship" title="WWE European Championship">WWF European Championship</a> match that saw <a href="/wiki/Shane_McMahon" title="Shane McMahon">Shane McMahon</a> defend against <a href="/wiki/Sean_Waltman" title="Sean Waltman">X-Pac</a>. McMahon pinned X-Pac after <a href="/wiki/Triple_H" title="Triple H">Triple H</a> interfered and gave him a <a href="/wiki/Facebuster#Double_underhook_facebuster" title="Facebuster">Pedigree</a>, while the referee was down. At one point <a href="/wiki/Pat_Patterson_(wrestler)" title="Pat Patterson (wrestler)">Pat Patterson</a> and <a href="/wiki/Gerald_Brisco" title="Gerald Brisco">Gerald Brisco</a> stopped <a href="/wiki/Shane_McMahon" title="Shane McMahon">Shane McMahon</a> from leaving the match early, leading to <a href="/wiki/Chyna" title="Chyna">Chyna's</a> interference, attacking the pair so they wouldn't interfere again.</p>
<p>The seventh match saw <a class="mw-redirect" href="/wiki/Monty_Sopp" title="Monty Sopp">Mr. Ass</a> defeat <a href="/wiki/Mick_Foley" title="Mick Foley">Mankind</a>. Mr. Ass pinned Mankind after a <a href="/wiki/Leg_drop#Leg_drop_bulldog" title="Leg drop">Fameasser</a>.<sup class="reference" id="cite_ref-WWF_PPV_Results_1-2"><a href="#cite_note-WWF_PPV_Results-1">[1]</a></sup></p>
<h3><span class="mw-headline" id="Main_event_matches">Main event matches</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=No_Mercy_(UK)&amp;action=edit&amp;section=3" title="Edit section: Main event matches">edit</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>The main event match was <a href="/wiki/Professional_wrestling_match_types#No_Disqualification_match" title="Professional wrestling match types">Anything Goes</a> <a href="/wiki/Professional_wrestling_match_types#Basic_non-elimination_matches" title="Professional wrestling match types">Triple Threat match</a> for the <a href="/wiki/WWE_Championship" title="WWE Championship">WWF Championship</a>. Defending champion <a href="/wiki/Stone_Cold_Steve_Austin" title="Stone Cold Steve Austin">Stone Cold Steve Austin</a> defeated <a href="/wiki/The_Undertaker" title="The Undertaker">The Undertaker</a> (with <a href="/wiki/Paul_Bearer" title="Paul Bearer">Paul Bearer</a>) and <a href="/wiki/Triple_H" title="Triple H">Triple H</a> (with <a href="/wiki/Chyna" title="Chyna">Chyna</a>), pinning Triple H after a <a href="/wiki/Stunner_(professional_wrestling)" title="Stunner (professional wrestling)">Stone Cold Stunner</a> following interference from the <a class="mw-redirect" href="/wiki/Corporate_Ministry" title="Corporate Ministry">Corporate Ministry</a>, <a href="/wiki/The_Brood_(professional_wrestling)" title="The Brood (professional wrestling)">The Brood</a>, <a href="/wiki/Kane_(wrestler)" title="Kane (wrestler)">Kane</a>, <a href="/wiki/Sean_Waltman" title="Sean Waltman">X-Pac</a> and <a href="/wiki/Mick_Foley" title="Mick Foley">Mankind</a>.</p>
