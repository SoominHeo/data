<p><b>Nonane</b> is a linear <a href="/wiki/Alkane" title="Alkane">alkane</a> <a href="/wiki/Hydrocarbon" title="Hydrocarbon">hydrocarbon</a> with the <a href="/wiki/Chemical_formula" title="Chemical formula">chemical formula</a> C<sub>9</sub>H<sub>20</sub>. It is a colorless, flammable liquid, occurring primarily in the component of the <a href="/wiki/Petrochemical" title="Petrochemical">petroleum distillate fraction</a> commonly called <a href="/wiki/Kerosene" title="Kerosene">kerosene</a>, which is used as a heating, <a href="/wiki/Tractor" title="Tractor">tractor</a>, and <a href="/wiki/Jet_fuel" title="Jet fuel">jet fuel</a>.<sup class="reference" id="cite_ref-4"><a href="#cite_note-4">[4]</a></sup> Nonane is also used as a <a href="/wiki/Solvent" title="Solvent">solvent</a>, distillation chaser, <a class="mw-redirect" href="/wiki/Fuel_additive" title="Fuel additive">fuel additive</a>, and a component in <a href="/wiki/Biodegradation" title="Biodegradation">biodegradable</a> <a href="/wiki/Detergent" title="Detergent">detergents</a>.<sup class="reference" id="cite_ref-5"><a href="#cite_note-5">[5]</a></sup></p>
<p>Nonane has 35 <a href="/wiki/Structural_isomer" title="Structural isomer">structural isomers</a>.</p>
<p>Its <a href="/wiki/Substituent" title="Substituent">substituent</a> form is nonyl. Its <a href="/wiki/Cycloalkane" title="Cycloalkane">cycloalkane</a> counterpart is <a href="/wiki/Cyclononane" title="Cyclononane">cyclononane</a>, (C<sub>9</sub>H<sub>18</sub>).</p>
<p>Unlike most alkanes, the numeric prefix in its name is from <a href="/wiki/Latin" title="Latin">Latin</a>, not <a href="/wiki/Greek_language" title="Greek language">Greek</a>. (A name using a Greek prefix would be <b>enneane</b>.)</p>

<h2><span class="mw-headline" id="Combustion_reactions">Combustion reactions</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Nonane&amp;action=edit&amp;section=1" title="Edit section: Combustion reactions">edit</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>Nonane undergoes <a href="/wiki/Combustion" title="Combustion">combustion</a> reactions that are similar to other <a class="mw-redirect" href="/wiki/Alkanes" title="Alkanes">alkanes</a>. In the presence of sufficient oxygen, nonane burns to form <a href="/wiki/Water" title="Water">water</a> and <a href="/wiki/Carbon_dioxide" title="Carbon dioxide">carbon dioxide</a>.</p>
<dl>
<dd><a href="/wiki/Carbon" title="Carbon">C</a><sub>9</sub><a href="/wiki/Hydrogen" title="Hydrogen">H</a><sub>20</sub> + 14<a href="/wiki/Oxygen" title="Oxygen">O<sub>2</sub></a> → 9<a href="/wiki/Carbon_dioxide" title="Carbon dioxide">CO<sub>2</sub></a> + 10<a href="/wiki/Water" title="Water">H<sub>2</sub>O</a></dd>
</dl>
<p>When insufficient oxygen is available for complete combustion, the burning products include <a href="/wiki/Carbon_monoxide" title="Carbon monoxide">carbon monoxide</a>.</p>
<dl>
<dd>2C<sub>9</sub>H<sub>20</sub> + 19O<sub>2</sub> → 18<a href="/wiki/Carbon_monoxide" title="Carbon monoxide">CO</a> + 20H<sub>2</sub>O</dd>
</dl>
