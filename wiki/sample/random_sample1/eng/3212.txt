<p><b>Norton Utilities</b> is a <a href="/wiki/Utility_software" title="Utility software">utility software</a> suite designed to help analyze, configure, optimize and maintain a computer. The current version of Norton Utilities is Norton Utilities 16 for Windows XP/Vista/7/8 was released October 26, 2012.</p>
<p><a href="/wiki/Peter_Norton" title="Peter Norton">Peter Norton</a> published the first version for <a href="/wiki/DOS" title="DOS">DOS</a>, <i>The Norton Utilities, Release 1</i>, in 1982. Release 2 came out about a year later, subsequent to the first hard drives for the IBM PC line. <a href="/wiki/Peter_Norton_Computing" title="Peter Norton Computing">Peter Norton's company</a> was sold to <a href="/wiki/Symantec" title="Symantec">Symantec</a> in 1990. However his name remains as a "brand" for Symantec's range of utility and security software for home users. Peter Norton himself has no connection to the brand or company.</p>

<h3><span class="mw-headline" id="v1.0">v1.0</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Norton_Utilities&amp;action=edit&amp;section=2" title="Edit section: v1.0">edit</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>The initial 1982 release supported DOS 1.x and featured the UNERASE utility. This allowed files to be <a href="/wiki/Undeletion" title="Undeletion">undeleted</a> by restoring the first letter of the directory entry (a feature of the <a href="/wiki/File_Allocation_Table" title="File Allocation Table">FAT file system</a> used in <a href="/wiki/MS-DOS" title="MS-DOS">MS-DOS</a>, albeit one that was not originally documented). The UNERASE utility was what launched NU on its path to success. Quoting Peter Norton, "Why did The Norton Utilities become such popular software? Well, industry wisdom has it that software becomes standard either by providing superior capabilities or by solving problems that were previously unsolvable. In 1982, when I sat down at my PC to write Unerase, I was solving a common problem to which there was no readily available solution."<sup class="reference" id="cite_ref-1"><a href="#cite_note-1">[1]</a></sup></p>
<p>14 programs were included, on three floppy disks, list price $80:<sup class="reference" id="cite_ref-2"><a href="#cite_note-2">[2]</a></sup><sup class="reference" id="cite_ref-3"><a href="#cite_note-3">[3]</a></sup></p>
<ul>
<li>UnErase, recovers erased files</li>
<li>FileFix, repairs damaged files</li>
<li>DiskLook, complete floppy disk displays and maps</li>
<li>SecMod, easy changes to floppy disks</li>
<li>FileHide, interactive hidden file control</li>
<li>BatHide, automatic hidden file control</li>
<li>TimeMark, displays date, time, elapsed time</li>
<li>ScrAtr, sets DOS to work in any colors</li>
<li>Reverse, work in black on white</li>
<li>Clear, clears the screen for clarity</li>
<li>FileSort, keeps floppy disk files by date or name</li>
<li>DiskOpt, speeds floppy disk access</li>
<li>Beep, causes the PC speaker to beep</li>
<li>Print, prints files</li>
</ul>
<h3><span class="mw-headline" id="v2.0">v2.0</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Norton_Utilities&amp;action=edit&amp;section=3" title="Edit section: v2.0">edit</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>The main feature of this version supporting DOS 2.x was FILEFIND, used for searching for files.<sup class="noprint Inline-Template Template-Fact" style="white-space:nowrap;">[<i><a href="/wiki/Wikipedia:Citation_needed" title="Wikipedia:Citation needed"><span title="This claim needs references to reliable sources. (November 2013)">citation needed</span></a></i>]</sup> This 1983 release added hard disk support, and the PRINT program was renamed LPRINT to avoid conflict with the DOS command introduced in MS-DOS 2.0 as PRINT.COM.<sup class="reference" id="cite_ref-4"><a href="#cite_note-4">[4]</a></sup> Following this release Norton became Utilities Editor of <i><a href="/wiki/PC_Magazine" title="PC Magazine">PC Magazine</a></i>.<sup class="reference" id="cite_ref-5"><a href="#cite_note-5">[5]</a></sup></p>
<h3><span class="mw-headline" id="v3.0">v3.0</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Norton_Utilities&amp;action=edit&amp;section=4" title="Edit section: v3.0">edit</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>This version, copyrighted 1984 but dated Monday, January 21, 1985, included Directory Sort, Disk Test, File Attributes, File Find, File Size, Line Print, List Directories, System Information, Text Search, Volume Label, Wipe Disk and Wipe File.</p>
<h3><span class="mw-headline" id="v3.1">v3.1</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Norton_Utilities&amp;action=edit&amp;section=5" title="Edit section: v3.1">edit</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>This 1986 version added the Quick UnErase (QU) and Unremove Directory (UD) programs.</p>

<p>Release 4.0 (1987, list $99.95) added four new programs:<sup class="reference" id="cite_ref-6"><a href="#cite_note-6">[6]</a></sup></p>
<ul>
<li>Ask, used for <a href="/wiki/Batch_file" title="Batch file">batch file</a> programming</li>
<li>File Info, add descriptions up to 65 characters long to each file</li>
<li>Norton Change Directory, displays a graphic directory tree; change, remove, rename or make directories</li>
<li>Norton Integrator, a menu system to tie the utilities together. Previously the utilities were accessed by typing the command name (usually a cryptic 2 character name such as FF) at the DOS prompt.</li>
</ul>
<h3><span class="mw-headline" id="v4.0_Advanced_Edition">v4.0 Advanced Edition</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Norton_Utilities&amp;action=edit&amp;section=7" title="Edit section: v4.0 Advanced Edition">edit</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>This version (list $150), released simultaneously with 4.0 standard edition, was dated Friday May 15, 1987. It added Speed Disk, a <a class="mw-redirect" href="/wiki/Disk_defragmenter" title="Disk defragmenter">disk defragmenter</a>, and Format Recover.</p>

<p>Norton Disk Doctor was the major addition in this 1988 release.<sup class="reference" id="cite_ref-7"><a href="#cite_note-7">[7]</a></sup> It also includes Wipedisk, Wipefile, <a href="/wiki/Batch_Enhancer" title="Batch Enhancer">Batch Enhancer</a> and a sector level <a href="/wiki/Disk_editor" title="Disk editor">disk editor</a>, a system information diagnostic utility, and a disk caching program (NCACHE), which was between 10 - 50% faster than Microsoft's <a class="mw-redirect" href="/wiki/SMARTDrive" title="SMARTDrive">SMARTDrive</a> when properly configured.</p>
<h3><span class="mw-headline" id="v5.0">v5.0</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Norton_Utilities&amp;action=edit&amp;section=9" title="Edit section: v5.0">edit</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>Release 5.0 included more features, including a utility to perform low level formatting on hard disks, and changes such as password protection on the more "dangerous" utilities. It also included a licensed version of the <a href="/wiki/4DOS" title="4DOS">4DOS</a> replacement for COMMAND.COM called <a href="/wiki/4DOS#NDOS" title="4DOS">NDOS</a>. This version also allowed the choice of "classic" names (such as FF.EXE) or longer names (such as FINDFAST.EXE); these were configurable in the updated version of the Norton Integrator menu system.</p>
<h3><span class="mw-headline" id="v6.0">v6.0</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Norton_Utilities&amp;action=edit&amp;section=10" title="Edit section: v6.0">edit</a><span class="mw-editsection-bracket">]</span></span></h3>
<div class="thumb tright">

<p>Norton Utilities 6.0 supports DOS 5 and Windows 3.1. It includes Windows Program Manager support, but the tools are still DOS-based, so a set of icons were supplied.</p>
<p>It includes Norton Disk Doctor, Disk Editor, Disk Tools, Speed Disk, Norton Cache, Disk Monitor, Diskreet, NDisk, System Information, <a href="/wiki/4DOS#NDOS" title="4DOS">NDOS</a>.<sup class="reference" id="cite_ref-8"><a href="#cite_note-8">[8]</a></sup></p>
<p>The speed of Speed Disk was faster than previous release.</p>
<p>Diskreet supports <a href="/wiki/Data_Encryption_Standard" title="Data Encryption Standard">Data Encryption Standard</a>.</p>
<p>System Information now includes more detailed information on installed system.</p>
<p>UnErase takes advantage of DOS 5.0's Delete Tracking and Mirror File features to recover data.<sup class="reference" id="cite_ref-9"><a href="#cite_note-9">[9]</a></sup> However, MS-DOS 5.0 added a new <tt>UNDELETE.exe</tt> program, licensed from Norton competitor <a href="/wiki/Central_Point_Software" title="Central Point Software">Central Point Software</a>,<sup class="reference" id="cite_ref-10"><a href="#cite_note-10">[10]</a></sup> which took advantage of the same Delete Sentry control and Deletion-tracking files. (The ERASE command is an alias for the DEL (Delete) command in the DOS <a class="mw-redirect" href="/wiki/Command-line_interpreter" title="Command-line interpreter">command line</a>.) <a href="/wiki/Windows_95" title="Windows 95">Windows 95</a>'s <a class="mw-redirect" href="/wiki/Recycle_Bin_(Windows)" title="Recycle Bin (Windows)">Recycle Bin</a> soon further reduced the value of UnErase.</p>
<h3><span class="mw-headline" id="v7.0">v7.0</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Norton_Utilities&amp;action=edit&amp;section=11" title="Edit section: v7.0">edit</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>It supports MS-DOS 6.0, <a class="mw-redirect" href="/wiki/DoubleSpace" title="DoubleSpace">DoubleSpace</a>, <a href="/wiki/Stac_Electronics" title="Stac Electronics">Stacker</a>, and <a class="mw-redirect" href="/wiki/SuperStor" title="SuperStor">SuperStor</a> disk compression tools.</p>
<p>It includes Norton Disk Doctor, Disk Editor, Undelete, SmartCan, <a href="/wiki/4DOS#NDOS" title="4DOS">NDOS</a>, System Information, File Find, Norton Change Directory.<sup class="reference" id="cite_ref-11"><a href="#cite_note-11">[11]</a></sup></p>
<p>Speed Disk remained as well, although <tt>SPEEDISK.exe</tt> was licensed by <a href="/wiki/Microsoft" title="Microsoft">Microsoft</a> and incorporated into <a href="/wiki/MS-DOS" title="MS-DOS">MS-DOS</a> as <tt>DEFRAG.exe</tt> in MS-DOS 6 onward.<sup class="reference" id="cite_ref-12"><a href="#cite_note-12">[12]</a></sup></p>
<p>Release 7.0 had revised user interfaces for the utilities that feature a menu-driven user interface. Also some of these tools now did not need to run in full-screen-mode but just displayed a window in the center of the screen, like the disk formatter or the disk duplicator utility.</p>
<p>Disk Editor now includes Advanced Recovery Mode.</p>
<h3><span class="mw-headline" id="v8.0_for_DOS.2FWindows_3.x">v8.0 for DOS/Windows 3.x</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Norton_Utilities&amp;action=edit&amp;section=12" title="Edit section: v8.0 for DOS/Windows 3.x">edit</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>Release 8.0 was nearly the same as 7.0, but added Windows 3.1 utilities.</p>
<p>DOS Utilities include Norton Disk Doctor, System Information, Change Directory, FileFind, Diskreet, DUPDISK, File Fix, <a href="/wiki/4DOS#NDOS" title="4DOS">NDOS</a>, Batch Enhancer, Norton Integrator, Speed Disk.<sup class="reference" id="cite_ref-13"><a href="#cite_note-13">[13]</a></sup></p>
<p>Windows utilities include Norton Disk Doctor, Speed Disk, System Watch, File Compare, INI Tracker, INI Tuner, INI editor, INI Advisor</p>
<h2><span class="mw-headline" id="Norton_Utilities_for_Windows_95_and_later">Norton Utilities for Windows 95 and later</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Norton_Utilities&amp;action=edit&amp;section=13" title="Edit section: Norton Utilities for Windows 95 and later">edit</a><span class="mw-editsection-bracket">]</span></span></h2>

<h3><span class="mw-headline" id="v1.0_for_Windows_95">v1.0 for Windows 95</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Norton_Utilities&amp;action=edit&amp;section=14" title="Edit section: v1.0 for Windows 95">edit</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>The original Windows 95 version was released on August 1995.</p>
<p>As a native Windows 95 tool, it supports <a href="/wiki/File_Allocation_Table#Long_File_Names_.28VFAT.2C_LFNs.29" title="File Allocation Table">VFAT</a> file system used by Windows 95.</p>
<p>It includes Norton Image, Norton System Doctor, Norton Protection, Rescue Disk, System Information, Norton Disk Doctor, Norton UnErase, Space Wizard, Speed Disk, System Information.<sup class="reference" id="cite_ref-14"><a href="#cite_note-14">[14]</a></sup> DOS tools include Disk Editor, Disk Doctor, Pre-Installation TuneUp.<sup class="reference" id="cite_ref-15"><a href="#cite_note-15">[15]</a></sup></p>
<p>CD-ROM version was announced on 1996-01-15.<sup class="reference" id="cite_ref-16"><a href="#cite_note-16">[16]</a></sup> It adds Disk Companion, Memory Companion, and Norton Utilities Companion over floppy disk version.</p>
<p>Disk tools included with this release are not compatible with FAT32.</p>
<p>The Windows version of 'Speed Disk' works with much greater speed than Microsoft's supplied defrag program, as it moved groups of clusters, not single clusters as Microsoft's <a class="mw-redirect" href="/wiki/Windows_Disk_Defragmenter" title="Windows Disk Defragmenter">Windows Disk Defragmenter</a>. Changes in the way Windows operates meant that many of the old utilities were either dropped completely or replaced with GUI based versions. However, with the advent of Windows XP onwards, Norton's 'Speed Disk' reverted to single-cluster defragmentation.<sup class="noprint Inline-Template Template-Fact" style="white-space:nowrap;">[<i><a href="/wiki/Wikipedia:Citation_needed" title="Wikipedia:Citation needed"><span title="This claim needs references to reliable sources. (March 2009)">citation needed</span></a></i>]</sup></p>

<p><b>Norton Utilities for Macintosh</b> was a separate utility suite for the <a class="mw-redirect" href="/wiki/Mac_OS" title="Mac OS">Mac OS</a>, providing much of the same features as the equivalent Windows version.</p>
<h3><span class="mw-headline" id="v1.0_2">v1.0</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Norton_Utilities&amp;action=edit&amp;section=34" title="Edit section: v1.0">edit</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>Originally released in 1990, requiring a Mac Plus with 1MiB system memory.</p>
<p>The included disk tools support the HFS file system only. Support for the Apple Hard Disk 20 was not explicitly mentioned.</p>
<p>When Apple released System 7, Norton Utilities for the Macintosh needed to be updated in order to run safely. This proved to be a recurring story in the history of the utility; early versions were highly regarded for a while, but proved to be hazardous when used with newer disk structures and operating systems before being updated.</p>
<h3><span class="mw-headline" id="v2.0_2">v2.0</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Norton_Utilities&amp;action=edit&amp;section=35" title="Edit section: v2.0">edit</a><span class="mw-editsection-bracket">]</span></span></h3>

<h2><span class="mw-headline" id="Symantec_Utilities_for_Macintosh">Symantec Utilities for Macintosh</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Norton_Utilities&amp;action=edit&amp;section=45" title="Edit section: Symantec Utilities for Macintosh">edit</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>Symantec Utilities for Macintosh (SUM) was a set of system utilities from Symantec, developed before Symantec had acquired Peter Norton Computing. Later SUM tools may have shared code with Norton Utilities for Macintosh.</p>
<h3><span class="mw-headline" id="v1.1">v1.1</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Norton_Utilities&amp;action=edit&amp;section=46" title="Edit section: v1.1">edit</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>It was published in 1988.</p>
<p>It supports non-SCSI drives (HD20, MacBottom, HyperDrive are explicitly listed) and also supports MFS floppy disks in a limited way.</p>
<p>It runs on Mac 512Ke, Plus, SE, Mac II (i.e. not 512K or 128K) with System v4.1 and Finder v5.3 or higher.<sup class="reference" id="cite_ref-53"><a href="#cite_note-53">[53]</a></sup></p>
<h3><span class="mw-headline" id="v2.0_3">v2.0</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Norton_Utilities&amp;action=edit&amp;section=47" title="Edit section: v2.0">edit</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>It was released after Symantec had acquired Peter Norton Computing, in year 1990.</p>
<p>It runs on 1MB Macintosh with System 4.2 or higher.</p>
<p>It includes similar recovery tools, but lacked the high degree of file-recovery automation available with Norton Utilities for Macintosh.<sup class="reference" id="cite_ref-54"><a href="#cite_note-54">[54]</a></sup></p>
<p>Norton Utilities has been discontinued for mac</p>
<h2><span class="mw-headline" id="Norton_Utilities_for_Unix">Norton Utilities for Unix</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Norton_Utilities&amp;action=edit&amp;section=48" title="Edit section: Norton Utilities for Unix">edit</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>In 1990, <a href="/wiki/Peter_Norton_Computing" title="Peter Norton Computing">Peter Norton Computing</a> developed a <a href="/wiki/Unix" title="Unix">Unix</a> version of Norton Utilities, in cooperation with <a class="mw-redirect" href="/wiki/Segue_Software" title="Segue Software">Segue Software</a> and <a href="/wiki/Interactive_Systems_Corporation" title="Interactive Systems Corporation">INTERACTIVE Systems Corp</a>.<sup class="reference" id="cite_ref-55"><a href="#cite_note-55">[55]</a></sup> This version of Norton Utilities was available up to 1992, when Interactive Systems stopped marketing the software.<sup class="reference" id="cite_ref-almondseed_press1_56-0"><a href="#cite_note-almondseed_press1-56">[56]</a></sup> In February 1994, AlmondSeed Software Inc. licensed the software and released it as "The Almond Utilities for UNIX".<sup class="reference" id="cite_ref-almondseed_press1_56-1"><a href="#cite_note-almondseed_press1-56">[56]</a></sup> AlmondSeed Software released the Utilities for <a class="mw-redirect" href="/wiki/SCO_OpenServer" title="SCO OpenServer">SCO UNIX</a><sup class="reference" id="cite_ref-almondseed_press1_56-2"><a href="#cite_note-almondseed_press1-56">[56]</a></sup> and <a href="/wiki/Solaris_(operating_system)" title="Solaris (operating system)">Sun Solaris</a>.<sup class="reference" id="cite_ref-almondseed_press2_57-0"><a href="#cite_note-almondseed_press2-57">[57]</a></sup></p>

<h2><span class="mw-headline" id="Reviews">Reviews</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Norton_Utilities&amp;action=edit&amp;section=50" title="Edit section: Reviews">edit</a><span class="mw-editsection-bracket">]</span></span></h2>
<p><i><a class="mw-redirect" href="/wiki/BYTE" title="BYTE">BYTE</a></i> in 1989 listed Norton Utilities 4.5 for DOS as among the "Distinction" winners of the BYTE Awards, stating that it was "an essential first-aid kit".<sup class="reference" id="cite_ref-byte198901_58-0"><a href="#cite_note-byte198901-58">[58]</a></sup></p>
<p>December 18, 2009 Darian Graham-Smith of PC Pro gave Norton Utilities 14 an overall rating of 2 of 6 stars. He concluded "There's simply no good reason to pay this much for such a flimsy collection of tools."<sup class="reference" id="cite_ref-59"><a href="#cite_note-59">[59]</a></sup></p>
<p>January 4, 2011 PC World Magazine reviewed Norton Utilities 15 and concluded "Norton Utilities may have outlived its usefulness. If your system isn't running as fast as you think it should, uninstalling unnecessary software and pruning background applications using Windows' own tools or downloadable freebies are a more cost-effective first defense."<sup class="reference" id="cite_ref-60"><a href="#cite_note-60">[60]</a></sup></p>
<p>August 28, 2012 <i><a href="/wiki/PC_Magazine" title="PC Magazine">PC Magazine</a></i> gave Norton Utilities 15 a score 3 of 5 stars. They said "This PC tune-up utility can give an aged PC a new lease on life, but its lacks some of the features and performance found in competing software."<sup class="reference" id="cite_ref-61"><a href="#cite_note-61">[61]</a></sup></p>
<p>August 31, 2012 ITProPortal.com gave Norton Utilities a rating of 6 out of 10. they concluded " Norton Utilities did a satisfactory job of de-gunking and increasing the overall performance of our test laptop, but it doesn’t stack up well against rival suite System Mechanic 11."<sup class="reference" id="cite_ref-62"><a href="#cite_note-62">[62]</a></sup></p>
<p>January 13, 2014 TopTenReviews.com gave Norton Utilities a 7.7 star rating out of 10 and said "Norton Utilities has good help and support features and is above average in terms of ease of use." <sup class="reference" id="cite_ref-63"><a href="#cite_note-63">[63]</a></sup></p>
<h2><span class="mw-headline" id="Controversies">Controversies</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Norton_Utilities&amp;action=edit&amp;section=51" title="Edit section: Controversies">edit</a><span class="mw-editsection-bracket">]</span></span></h2>
<h3><span class="mw-headline" id="2012_January_source_code_hack">2012 January source code hack</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Norton_Utilities&amp;action=edit&amp;section=52" title="Edit section: 2012 January source code hack">edit</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>On January 17, 2012, Symantec admitted to their network getting hacked. A hacker known as "Yama Tough" obtained Symantec's source code by hacking an Indian Government server. Yama Tough has released parts of the source code, and has threatened to release more. According to Chris Paden, a Symantec spokesman, the source code that was taken was from enterprise products that were between 5 and 6 years old.<sup class="reference" id="cite_ref-64"><a href="#cite_note-64">[64]</a></sup></p>
<h3><span class="mw-headline" id="2012_September_source_code_hack">2012 September source code hack</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Norton_Utilities&amp;action=edit&amp;section=53" title="Edit section: 2012 September source code hack">edit</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>On September 25, 2012 an affiliate of the hacker group Anonymous published source code from Norton Utilities. Symantec has responded that this is from the same code leaked in January, 2012. They have confirmed that the leak included source code for 2006 versions of Norton Utilities, pcAnywhere, and Norton Antivirus.<sup class="reference" id="cite_ref-65"><a href="#cite_note-65">[65]</a></sup></p>
