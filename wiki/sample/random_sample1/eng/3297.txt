<h1 class="firstHeading" id="firstHeading" lang="en">Newton metre</h1>
<div class="mw-body-content" id="bodyContent">
<div id="siteSub">From Wikipedia, the free encyclopedia</div>
<div id="contentSub"></div>
<div class="mw-jump" id="jump-to-nav">
					Jump to:					<a href="#mw-head">navigation</a>, 					<a href="#p-search">search</a>
</div>
<div class="mw-content-ltr" dir="ltr" id="mw-content-text" lang="en"><script>function mfTempOpenSection(id){var block=document.getElementById("mf-section-"+id);block.className+=" open-block";block.previousSibling.className+=" open-block";}</script><div class="hatnote" role="note">For the instrument that measures forces in Newtons, see <a class="mw-redirect" href="/wiki/Newtonmeter" title="Newtonmeter">Newtonmeter</a>.</div>
<div class="floatright"><a class="image" href="/wiki/File:Torque_force_equivalence_at_one_meter_leverage.png"><img alt="Torque force equivalence at one meter leverage.png" data-file-height="257" data-file-width="408" height="189" src="//upload.wikimedia.org/wikipedia/commons/thumb/b/b7/Torque_force_equivalence_at_one_meter_leverage.png/300px-Torque_force_equivalence_at_one_meter_leverage.png" srcset="//upload.wikimedia.org/wikipedia/commons/b/b7/Torque_force_equivalence_at_one_meter_leverage.png 1.5x" width="300"/></a></div>
<p>The <b>newton metre</b> (also <b>newton-metre</b>, symbol <b>N m</b> or <b>N·m</b>)<sup class="reference" id="cite_ref-1"><a href="#cite_note-1">[1]</a></sup> is a <a class="mw-redirect" href="/wiki/Physical_unit" title="Physical unit">unit</a> of <a href="/wiki/Torque" title="Torque">torque</a> (also called "<a href="/wiki/Moment_(physics)" title="Moment (physics)">moment</a>") in the <a href="/wiki/International_System_of_Units" title="International System of Units">SI</a> system. One newton metre is equal to the torque resulting from a <a href="/wiki/Force" title="Force">force</a> of one <a href="/wiki/Newton_(unit)" title="Newton (unit)">newton</a> applied perpendicularly to a <a class="mw-redirect" href="/wiki/Moment_arm" title="Moment arm">moment arm</a> which is one <a href="/wiki/Metre" title="Metre">metre</a> long.</p>
<p>It is also used less commonly as a unit of <a href="/wiki/Work_(physics)" title="Work (physics)">work</a>, or <a href="/wiki/Energy" title="Energy">energy</a>, in which case it is equivalent to the more common and standard SI unit of energy, the <a href="/wiki/Joule" title="Joule">joule</a>.<sup class="reference" id="cite_ref-Eshbach_2-0"><a href="#cite_note-Eshbach-2">[2]</a></sup> In this very different usage the metre term represents the distance travelled or displacement <a href="/wiki/Dot_product#Physics" title="Dot product">in the direction</a> of the force, and not the <a href="/wiki/Cross_product#Mechanics" title="Cross product">perpendicular distance</a> from a fulcrum as it does when used to express torque. This usage is generally discouraged,<sup class="reference" id="cite_ref-3"><a href="#cite_note-3">[3]</a></sup> since it can lead to confusion as to whether a given quantity expressed in newton metres is a torque or a quantity of energy.<sup class="reference" id="cite_ref-bipm222_4-0"><a href="#cite_note-bipm222-4">[4]</a></sup> However, since torque represents energy transferred or expended per angle of revolution, one newton metre of torque is equivalent to one joule per <a href="/wiki/Radian" title="Radian">radian</a>.<sup class="reference" id="cite_ref-bipm222_4-1"><a href="#cite_note-bipm222-4">[4]</a></sup></p>

<p>Again, N⋅m and J are distinguished in order to avoid misunderstandings where a torque is mistaken for an energy or vice versa. Similar examples of dimensionally equivalent units include <a href="/wiki/Pascal_(unit)" title="Pascal (unit)">Pa</a> versus J/m³, <a href="/wiki/Becquerel" title="Becquerel">Bq</a> versus <a href="/wiki/Hertz" title="Hertz">Hz</a>, and <a href="/wiki/Ohm" title="Ohm">ohm</a> versus <a href="/wiki/Sheet_resistance" title="Sheet resistance">ohm per square</a>.</p>

<h2><span class="mw-headline" id="Conversion_factors">Conversion factors</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Newton_metre&amp;action=edit&amp;section=1" title="Edit section: Conversion factors">edit</a><span class="mw-editsection-bracket">]</span></span></h2>
<ul>
<li>1 <a href="/wiki/Kilogram-force" title="Kilogram-force">kilogram-force</a> metre = <a href="/wiki/Standard_gravity" title="Standard gravity">9.80665</a> N·m<sup class="reference" id="cite_ref-5"><a href="#cite_note-5">[5]</a></sup><sup class="reference" id="cite_ref-6"><a href="#cite_note-6">[6]</a></sup></li>
<li>1 <a href="/wiki/Pound_(force)" title="Pound (force)">pound-force</a> = <a href="/wiki/Pound_(mass)" title="Pound (mass)">0.45359237</a> <a href="/wiki/Kilogram-force" title="Kilogram-force">kilogram-force</a></li>
<li>1 newton metre ≈ 0.73756215 <a href="/wiki/Pound-foot_(torque)" title="Pound-foot (torque)">pound-force-feet</a> (often "foot-pounds")</li>
<li>1 <a href="/wiki/Pound-foot_(torque)" title="Pound-foot (torque)">pound-foot</a> (often "foot-pound") ≡ 1 <a class="mw-redirect" href="/wiki/Pound_force" title="Pound force">pound-force</a>-foot ≈ 1.35581795 N·m</li>
<li>1 ounce-inch (often "inch-ounce") ≡ 1 <a class="mw-redirect" href="/wiki/Ounce-force" title="Ounce-force">ounce-force</a>-inch ≈ 7.06155181 mN·m(milliNewtons-metre)</li>
<li>1 <a href="/wiki/Dyne" title="Dyne">dyne</a>-centimetre = 10<sup>−7</sup> N·m</li>

</ul>
