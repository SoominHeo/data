<p>This is a <b>list of counties in New Hampshire</b>.</p>
<p>There are ten <a href="/wiki/County_(United_States)" title="County (United States)">counties</a> in the <a href="/wiki/U.S._state" title="U.S. state">U.S. state</a> of <a href="/wiki/New_Hampshire" title="New Hampshire">New Hampshire</a>. Five of the counties were created in 1769, when New Hampshire was still an English colony and not a state, during the first subdivision of the state into counties. The last counties created were <a href="/wiki/Belknap_County,_New_Hampshire" title="Belknap County, New Hampshire">Belknap County</a> and <a href="/wiki/Carroll_County,_New_Hampshire" title="Carroll County, New Hampshire">Carroll County</a>, in 1840.</p>
<p>The majority of New Hampshire's counties were named for prominent British or American people or geographic locations and features. Only one county's name originates in a Native American language: <a href="/wiki/Co%C3%B6s_County,_New_Hampshire" title="Coös County, New Hampshire">Coös County</a>, named for an <a href="/wiki/Algonquian_languages" title="Algonquian languages">Algonquian</a> word meaning "small pines".</p>
<p>The counties tend to be smaller in land area towards the southern end of the state, where New Hampshire population is concentrated, and larger in land area in the less populous north.<sup class="reference" id="cite_ref-1"><a href="#cite_note-1">[1]</a></sup><sup class="reference" id="cite_ref-Almanac_2-0"><a href="#cite_note-Almanac-2">[2]</a></sup></p>
<p>The FIPS county code is the five-digit Federal Information Processing Standard (FIPS) code which uniquely identifies counties and county equivalents in the United States. The three-digit number is unique to each individual county within a state, but to be unique within the entire United States, it must be prefixed by the state code. This means that, for example, while <a href="/wiki/Belknap_County,_New_Hampshire" title="Belknap County, New Hampshire">Belknap County</a>, <a href="/wiki/New_Hampshire" title="New Hampshire">New Hampshire</a> is 001, <a href="/wiki/Addison_County,_Vermont" title="Addison County, Vermont">Addison County</a>, <a href="/wiki/Vermont" title="Vermont">Vermont</a> and <a href="/wiki/Alachua_County,_Florida" title="Alachua County, Florida">Alachua County</a>, <a href="/wiki/Florida" title="Florida">Florida</a> are also 001. To uniquely identify Belknap County, New Hampshire, one must use the state code of 33 plus the county code of 001; therefore, the unique nationwide identifier for Belknap County, New Hampshire is 33001. The links in the column <a class="mw-redirect" href="/wiki/FIPS_County_Code" title="FIPS County Code">FIPS County Code</a> are to the Census Bureau Info page for that county.<sup class="reference" id="cite_ref-epa_3-0"><a href="#cite_note-epa-3">[3]</a></sup></p>
