<p><b>Nishi-Azabu</b> <span style="font-weight: normal">(<span class="t_nihongo_kanji" lang="ja" xml:lang="ja">西麻布</span><sup class="t_nihongo_help noprint"><a href="/wiki/Help:Installing_Japanese_character_sets" title="Help:Installing Japanese character sets"><span class="t_nihongo_icon" style="color: #00e; font: bold 80% sans-serif; text-decoration: none; padding: 0 .1em;">?</span></a></sup>)</span> is a district of <a href="/wiki/Minato,_Tokyo" title="Minato, Tokyo">Minato-ku</a>, <a href="/wiki/Tokyo" title="Tokyo">Tokyo</a>, <a href="/wiki/Japan" title="Japan">Japan</a>, which was a part of the former <a href="/wiki/Azabu" title="Azabu">Azabu</a> Ward.</p>
<p>Nishiazabu borders <a class="new" href="/w/index.php?title=Minamiaoyama,_Minato,_Tokyo&amp;action=edit&amp;redlink=1" title="Minamiaoyama, Minato, Tokyo (page does not exist)">Minamiaoyama</a> on the north and west, <a class="mw-redirect" href="/wiki/Hiroo,_Shibuya,_Tokyo" title="Hiroo, Shibuya, Tokyo">Hiroo</a> on the south, <a href="/wiki/Moto-Azabu,_Minato,_Tokyo" title="Moto-Azabu, Minato, Tokyo">Moto-Azabu</a> on the southeast, and <a href="/wiki/Roppongi" title="Roppongi">Roppongi</a> on the northeast.</p>
<p>Nishi-Azabu houses the Embassies of <a href="/wiki/Yemen" title="Yemen">Yemen</a>, <a href="/wiki/Ukraine" title="Ukraine">Ukraine</a>, <a href="/wiki/Uruguay" title="Uruguay">Uruguay</a>, <a href="/wiki/Ecuador" title="Ecuador">Ecuador</a>, <a href="/wiki/El_Salvador" title="El Salvador">El Salvador</a>, <a href="/wiki/Ghana" title="Ghana">Ghana</a>, <a href="/wiki/Greece" title="Greece">Greece</a>, <a href="/wiki/Guatemala" title="Guatemala">Guatemala</a>, <a href="/wiki/Costa_Rica" title="Costa Rica">Costa Rica</a>, <a href="/wiki/Dominican_Republic" title="Dominican Republic">Dominican Republic</a>, <a href="/wiki/Nicaragua" title="Nicaragua">Nicaragua</a>, <a href="/wiki/Haiti" title="Haiti">Haiti</a>, <a href="/wiki/Panama" title="Panama">Panama</a>, <a href="/wiki/Venezuela" title="Venezuela">Venezuela</a>, <a href="/wiki/Belize" title="Belize">Belize</a>, <a href="/wiki/Bolivia" title="Bolivia">Bolivia</a>, <a href="/wiki/Honduras" title="Honduras">Honduras</a>, <a href="/wiki/Laos" title="Laos">Laos</a>, and <a href="/wiki/Romania" title="Romania">Romania</a>.</p>
<p>Chōkoku-ji <span style="font-weight: normal">(<span class="t_nihongo_kanji" lang="ja" xml:lang="ja">長谷寺</span><sup class="t_nihongo_help noprint"><a href="/wiki/Help:Installing_Japanese_character_sets" title="Help:Installing Japanese character sets"><span class="t_nihongo_icon" style="color: #00e; font: bold 80% sans-serif; text-decoration: none; padding: 0 .1em;">?</span></a></sup>)</span>, also known as Azabu <a class="mw-redirect" href="/wiki/Guan_Yin" title="Guan Yin">Kannon</a> <span style="font-weight: normal">(<span class="t_nihongo_kanji" lang="ja" xml:lang="ja">麻布観音</span><sup class="t_nihongo_help noprint"><a href="/wiki/Help:Installing_Japanese_character_sets" title="Help:Installing Japanese character sets"><span class="t_nihongo_icon" style="color: #00e; font: bold 80% sans-serif; text-decoration: none; padding: 0 .1em;">?</span></a></sup>)</span>, is a <a href="/wiki/S%C5%8Dt%C5%8D" title="Sōtō">Sōtō-shū</a> temple and a tourist attraction located in Nishi-Azabu.</p>
