<h1 class="firstHeading" id="firstHeading" lang="en">One hour run</h1>
<div class="mw-body-content" id="bodyContent">
<div id="siteSub">From Wikipedia, the free encyclopedia</div>
<div id="contentSub"></div>
<div class="mw-jump" id="jump-to-nav">

</div>
<div class="mw-content-ltr" dir="ltr" id="mw-content-text" lang="en"><script>function mfTempOpenSection(id){var block=document.getElementById("mf-section-"+id);block.className+=" open-block";block.previousSibling.className+=" open-block";}</script><div class="hatnote" role="note">For hour record for <a href="/wiki/Bicycle" title="Bicycle">bicycles</a>, see <a href="/wiki/Hour_record" title="Hour record">Hour record</a>.</div>
<p>The <b>one hour run</b> is an <a class="mw-redirect" href="/wiki/Athletics_(sport)" title="Athletics (sport)">athletics</a> event in which competitors try to cover as much distance as possible within one hour. While officially recognized by the <a href="/wiki/International_Association_of_Athletics_Federations" title="International Association of Athletics Federations">International Association of Athletics Federations</a> as a track event, it is rarely contested apart from occasional world record attempts.</p>
<p>The event has a long history, with first recorded races dating back to the late 17th century.<sup class="reference" id="cite_ref-1"><a href="#cite_note-1">[1]</a></sup> The first athlete to run more than 20 kilometers in one hour was <a href="/wiki/Emil_Z%C3%A1topek" title="Emil Zátopek">Emil Zátopek</a>, in September 1951. Zátopek also set the 20,000 meters world record in the same race, and since that time most men's 20,000 m world records were set <i>en route</i> to one hour world records.</p>
<p>The men's world record is 21,285 m, set by <a href="/wiki/Haile_Gebrselassie" title="Haile Gebrselassie">Haile Gebrselassie</a> on 27 June 2007, while the women's world record is 18,517 m, set by <a href="/wiki/Dire_Tune" title="Dire Tune">Dire Tune</a> on 12 June 2008.</p>
