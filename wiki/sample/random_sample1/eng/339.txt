<p>In <a href="/wiki/Physics" title="Physics">physics</a> and <a href="/wiki/Mathematics" title="Mathematics">mathematics</a>, a <a href="/wiki/Euclidean_vector" title="Euclidean vector">sequence</a> of <i>n</i> <a href="/wiki/Real_number" title="Real number">numbers</a> can be understood as a <a href="/wiki/Point_(geometry)" title="Point (geometry)">location</a> in <i>n</i>-dimensional space. When <span class="nowrap"><i>n</i> = 1</span>, the set of all such locations is called a <b>one-dimensional <a href="/wiki/Space" title="Space">space</a></b>. An example of a one-dimensional space is the <a href="/wiki/Number_line" title="Number line">number line</a>, where the position of each point on it can be described by a single number.<sup class="reference" id="cite_ref-1"><a href="#cite_note-1">[1]</a></sup></p>

<h2><span class="mw-headline" id="One-dimensional_geometry">One-dimensional geometry</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=One-dimensional_space&amp;action=edit&amp;section=1" title="Edit section: One-dimensional geometry">edit</a><span class="mw-editsection-bracket">]</span></span></h2>
<h3><span class="mw-headline" id="Lines">Lines</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=One-dimensional_space&amp;action=edit&amp;section=2" title="Edit section: Lines">edit</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>The <a class="mw-redirect" href="/wiki/Real_number_line" title="Real number line">real number line</a> and the <a href="/wiki/Real_projective_line" title="Real projective line">real projective line</a> are one-dimensional, though the latter is <a class="mw-redirect" href="/wiki/Homeomorphic" title="Homeomorphic">homeomorphic</a> to a <a href="/wiki/Circle" title="Circle">circle</a>.</p>
<p>In <a href="/wiki/Algebraic_geometry" title="Algebraic geometry">algebraic geometry</a> there are several structures which are technically one-dimensional spaces but referred to in other terms. For a <a href="/wiki/Field_(mathematics)" title="Field (mathematics)">field</a> <i>k</i>, it is a one-dimensional <a href="/wiki/Vector_space" title="Vector space">vector space</a> over itself. Similarly, the <a href="/wiki/Projective_line" title="Projective line">projective line</a> over <i>k</i> is a one-dimensional space. In particular, if <i>k</i> = ℂ, the <a class="mw-redirect" href="/wiki/Complex_number_plane" title="Complex number plane">complex number plane</a>, then the <a class="mw-redirect" href="/wiki/Complex_projective_line" title="Complex projective line">complex projective line</a> P<sup>1</sup>(ℂ) is one-dimensional with respect to ℂ, even if it is also known as the <a href="/wiki/Riemann_sphere" title="Riemann sphere">Riemann sphere</a>.</p>
<p>More generally, a <a href="/wiki/Ring_(mathematics)" title="Ring (mathematics)">ring</a> is a <a href="/wiki/Length_of_a_module" title="Length of a module">length-one</a> <a href="/wiki/Module_(mathematics)" title="Module (mathematics)">module</a> over itself. Similarly, the <a href="/wiki/Projective_line_over_a_ring" title="Projective line over a ring">projective line over a ring</a> is a one-dimensional space over the ring. In case the ring is an <a href="/wiki/Algebra_over_a_field" title="Algebra over a field">algebra over a field</a>, these spaces are one-dimensional with respect to the algebra, even if the algebra is of higher dimensionality.</p>
<h3><span class="mw-headline" id="Polytopes">Polytopes</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=One-dimensional_space&amp;action=edit&amp;section=3" title="Edit section: Polytopes">edit</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>The only regular <a href="/wiki/Polytope" title="Polytope">polytope</a> in one dimension is the <a href="/wiki/Line_segment" title="Line segment">line segment</a>, with the <a href="/wiki/Schl%C3%A4fli_symbol" title="Schläfli symbol">Schläfli symbol</a> { }.</p>
<h3><span class="mw-headline" id="Hypersphere">Hypersphere</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=One-dimensional_space&amp;action=edit&amp;section=4" title="Edit section: Hypersphere">edit</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>The <a href="/wiki/Hypersphere" title="Hypersphere">hypersphere</a> in 1 dimension is a pair of <a class="mw-redirect" href="/wiki/Point_(mathematics)" title="Point (mathematics)">points</a>,<sup class="reference" id="cite_ref-2"><a href="#cite_note-2">[2]</a></sup> sometimes called a <b>0-sphere</b> as its surface is zero-dimensional.
