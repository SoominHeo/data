<h1 class="firstHeading" id="firstHeading" lang="en">Simple aromatic ring</h1>
<div class="mw-body-content" id="bodyContent">
<div id="siteSub">From Wikipedia, the free encyclopedia</div>
<div id="contentSub"></div>
<div class="mw-jump" id="jump-to-nav">
					Jump to:					<a href="#mw-head">navigation</a>, 					<a href="#p-search">search</a>
</div>
<div class="mw-content-ltr" dir="ltr" id="mw-content-text" lang="en"><script>function mfTempOpenSection(id){var block=document.getElementById("mf-section-"+id);block.className+=" open-block";block.previousSibling.className+=" open-block";}</script><p><b>Simple aromatic rings</b>, also known as <b>simple arenes</b> or <b>simple aromatics</b>, are <a class="mw-redirect" href="/wiki/Aromatic" title="Aromatic">aromatic</a> <a href="/wiki/Organic_compound" title="Organic compound">organic compounds</a> that consist only of a <a href="/wiki/Conjugated_system" title="Conjugated system">conjugated</a> planar ring system. Many simple aromatic rings have trivial names. They are usually found as substructures of more complex <a href="/wiki/Molecule" title="Molecule">molecules</a> ("<a class="mw-redirect" href="/wiki/Substitution_(chemistry)" title="Substitution (chemistry)">substituted</a> aromatics"). Typical simple aromatic compounds are <a href="/wiki/Benzene" title="Benzene">benzene</a>, <a href="/wiki/Indole" title="Indole">indole</a>, and <a href="/wiki/Cyclotetradecaheptaene" title="Cyclotetradecaheptaene">cyclotetradecaheptaene</a>.<sup class="reference" id="cite_ref-Clayden1st_1-0"><a href="#cite_note-Clayden1st-1">[1]</a></sup><sup class="reference" id="cite_ref-Eicher_2-0"><a href="#cite_note-Eicher-2">[2]</a></sup></p>
<p>Simple aromatic rings can be <a class="mw-redirect" href="/wiki/Heterocyclic" title="Heterocyclic">heterocyclic</a> if they contain non-<a href="/wiki/Carbon" title="Carbon">carbon</a> ring atoms, for example, <a href="/wiki/Oxygen" title="Oxygen">oxygen</a>, <a href="/wiki/Nitrogen" title="Nitrogen">nitrogen</a>, or <a href="/wiki/Sulfur" title="Sulfur">sulfur</a>. They can be monocyclic as in benzene, bicyclic as in <a href="/wiki/Naphthalene" title="Naphthalene">naphthalene</a>, or polycyclic as in <a href="/wiki/Anthracene" title="Anthracene">anthracene</a>. Simple monocyclic aromatic rings are usually five-membered rings like <a href="/wiki/Pyrrole" title="Pyrrole">pyrrole</a> or six-membered rings like <a href="/wiki/Pyridine" title="Pyridine">pyridine</a>. Fused aromatic rings consist of monocyclic rings that share their connecting bonds.</p>

<p>The nitrogen (N)-containing aromatic rings can be separated into <a href="/wiki/Basic_aromatic_ring" title="Basic aromatic ring">basic aromatic rings</a> that are easily <a href="/wiki/Protonation" title="Protonation">protonated</a>, and form aromatic <a class="mw-redirect" href="/wiki/Cation" title="Cation">cations</a> and <a class="mw-redirect" href="/wiki/Salts" title="Salts">salts</a> (e.g., <a href="/wiki/Pyridinium" title="Pyridinium">pyridinium</a>), and non-basic aromatic rings.</p>
<ul>
<li>In the <b>basic aromatic rings</b>, the <a href="/wiki/Lone_pair" title="Lone pair">lone pair</a> of <a href="/wiki/Electron" title="Electron">electrons</a> is not part of the aromatic system and extends in the plane of the ring. This lone pair is responsible for the basicity of these <a href="/wiki/Nitrogenous_base" title="Nitrogenous base">nitrogenous bases</a>, similar to the nitrogen atom in <a href="/wiki/Amine" title="Amine">amines</a>. In these compounds the nitrogen atom is <i>not</i> connected to a hydrogen atom. Examples of basic aromatic rings are <a href="/wiki/Pyridine" title="Pyridine">pyridine</a> or <a href="/wiki/Quinoline" title="Quinoline">quinoline</a>. Several rings contain basic as well as non-basic nitrogen atoms, e.g., <a href="/wiki/Imidazole" title="Imidazole">imidazole</a> and <a href="/wiki/Purine" title="Purine">purine</a>.</li>
<li>In the <b>non-basic rings</b>, the lone pair of electrons of the nitrogen atom is delocalized and contributes to the aromatic pi electron system. In these compounds, the nitrogen atom <i>is</i> connected to a hydrogen atom. Examples of non-basic nitrogen-containing aromatic rings are <a href="/wiki/Pyrrole" title="Pyrrole">pyrrole</a> and <a href="/wiki/Indole" title="Indole">indole</a>.</li>
</ul>
<p>In the oxygen- and sulfur-containing aromatic rings, one of the electron pairs of the heteroatoms contributes to the aromatic system (similar to the non-basic nitrogen-containing rings), whereas the second lone pair extends in the plane of the ring (similar to the basic nitrogen-containing rings).</p>
<h2><span class="mw-headline" id="Criteria_for_aromaticity">Criteria for aromaticity</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Simple_aromatic_ring&amp;action=edit&amp;section=2" title="Edit section: Criteria for aromaticity">edit</a><span class="mw-editsection-bracket">]</span></span></h2>
<div class="hatnote" role="note">Main article: <a href="/wiki/Aromaticity" title="Aromaticity">Aromaticity</a></div>
<ul>
<li>Molecule must be cyclic.</li>
<li>Every atom in the ring must have an occupied <a class="mw-redirect" href="/wiki/P_orbital" title="P orbital">p orbital</a>, which overlaps with p orbitals on either side (completely <a href="/wiki/Conjugated_system" title="Conjugated system">conjugated</a>).</li>
<li>Molecule must be planar.</li>
<li>It must contain an odd number of pairs of pi electrons; must satisfy <a href="/wiki/H%C3%BCckel%27s_rule" title="Hückel's rule">Hückel's rule</a>: (4n+2) pi electrons, where n is an integer starting at zero.</li>
</ul>
<p>In contrast, molecules with 4n pi electrons are <a class="mw-redirect" href="/wiki/Antiaromatic" title="Antiaromatic">antiaromatic</a>.</p>
