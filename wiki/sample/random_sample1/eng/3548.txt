<p>The <b>Heaviside step function</b>, or the <b>unit step function</b>, usually denoted by <span class="texhtml mvar" style="font-style:italic;">H</span> or <span class="texhtml mvar" style="font-style:italic;">Œ∏</span> (but sometimes <span class="texhtml mvar" style="font-style:italic;">u</span>, <span class="texhtml"><b>1</b></span> or <span class="texhtml">ùüô</span>), is a <a href="/wiki/Continuous_function" title="Continuous function">discontinuous</a> <a href="/wiki/Function_(mathematics)" title="Function (mathematics)">function</a> whose value is <a class="mw-redirect" href="/wiki/0_(number)" title="0 (number)">zero</a> for negative argument and <a class="mw-redirect" href="/wiki/1_(number)" title="1 (number)">one</a> for positive argument. It is an example of the general class of <a href="/wiki/Step_function" title="Step function">step functions</a>, all of which can be represented as linear combinations of translations of this one.</p>
<p>The function was originally developed in <a href="/wiki/Operational_calculus" title="Operational calculus">operational calculus</a> for the solution of <a href="/wiki/Differential_equation" title="Differential equation">differential equations</a>, where it represents a signal that switches on at a specified time and stays switched on indefinitely. <a href="/wiki/Oliver_Heaviside" title="Oliver Heaviside">Oliver Heaviside</a>, who developed the operational calculus as a tool in the analysis of telegraphic communications, represented the function as <span class="texhtml"><b>1</b></span>.</p>

<p>although this expansion may not hold (or even make sense) for <span class="texhtml"><i>x</i> = 0</span>, depending on which formalism one uses to give meaning to integrals involving <span class="texhtml mvar" style="font-style:italic;">Œ¥</span>. In this context, the Heaviside function is the <a href="/wiki/Cumulative_distribution_function" title="Cumulative distribution function">cumulative distribution function</a> of a <a href="/wiki/Random_variable" title="Random variable">random variable</a> which is <a href="/wiki/Almost_surely" title="Almost surely">almost surely</a> 0. (See <a class="mw-redirect" href="/wiki/Constant_random_variable" title="Constant random variable">constant random variable</a>.)</p>
<p>In operational calculus, useful answers seldom depend on which value is used for <span class="texhtml"><i>H</i>(0)</span>, since <span class="texhtml mvar" style="font-style:italic;">H</span> is mostly used as a <a href="/wiki/Distribution_(mathematics)" title="Distribution (mathematics)">distribution</a>. However, the choice may have some important consequences in functional analysis and game theory, where more general forms of continuity are considered. Some common choices can be seen <a href="#Zero_argument">below</a>.</p>

<h2><span class="mw-headline" id="Zero_argument">Zero argument</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Heaviside_step_function&amp;action=edit&amp;section=4" title="Edit section: Zero argument">edit</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>Since <span class="texhtml mvar" style="font-style:italic;">H</span> is usually used in integration, and the value of a function at a single point does not affect its integral, it rarely matters what particular value is chosen of <span class="texhtml"><i>H</i>(0)</span>. Indeed when <span class="texhtml mvar" style="font-style:italic;">H</span> is considered as a <a href="/wiki/Distribution_(mathematics)" title="Distribution (mathematics)">distribution</a> or an element of <span class="texhtml"><i>L</i><span style="padding-left:0.2em;"><sup>‚àû</sup></span></span> (see <a href="/wiki/Lp_space" title="Lp space"><span class="texhtml"><i>L<span style="padding-left:0.2em;"><sup>p</sup></span></i></span> space</a>) it does not even make sense to talk of a value at zero, since such objects are only defined <a href="/wiki/Almost_everywhere" title="Almost everywhere">almost everywhere</a>. If using some analytic approximation (as in the <a href="#Analytic_approximations">examples above</a>) then often whatever happens to be the relevant limit at zero is used.</p>
