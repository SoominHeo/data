An <a href="/wiki/Infinite_set" title="Infinite set">infinite set</a> may have the same <a href="/wiki/Cardinality" title="Cardinality">cardinality</a> as a proper <a href="/wiki/Subset" title="Subset">subset</a> of itself, as the depicted <a href="/wiki/Bijection" title="Bijection">bijection</a> <i>f</i>(<i>x</i>)=2<i>x</i> from the <a class="mw-redirect" href="/wiki/Natural_numbers" title="Natural numbers">natural</a> to the <a class="mw-redirect" href="/wiki/Even_numbers" title="Even numbers">even</a> numbers demonstrates. <p>In <a href="/wiki/Set_theory" title="Set theory">set theory</a>, <b>Cantor's diagonal argument</b>, also called the <b>diagonalisation argument</b>, the <b>diagonal slash argument</b> or the <b>diagonal method</b>, was published in 1891 by <a href="/wiki/Georg_Cantor" title="Georg Cantor">Georg Cantor</a> as a <a href="/wiki/Mathematical_proof" title="Mathematical proof">mathematical proof</a> that there are <a href="/wiki/Infinite_set" title="Infinite set">infinite sets</a> which cannot be put into <a href="/wiki/Bijection" title="Bijection">one-to-one correspondence</a> with the infinite set of <a href="/wiki/Natural_number" title="Natural number">natural numbers</a>.<sup class="reference" id="cite_ref-1"><a href="#cite_note-1">[1]</a></sup><sup class="reference" id="cite_ref-Simmons1993_2-0"><a href="#cite_note-Simmons1993-2">[2]</a></sup><sup class="reference" id="cite_ref-Rubin1976_3-0"><a href="#cite_note-Rubin1976-3">[3]</a></sup> Such sets are now known as <a href="/wiki/Uncountable_set" title="Uncountable set">uncountable sets</a>, and the size of infinite sets is now treated by the theory of <a href="/wiki/Cardinal_number" title="Cardinal number">cardinal numbers</a> which Cantor began.</p>
<p>The diagonal argument was not Cantor's <a class="mw-redirect" href="/wiki/Cantor%27s_first_uncountability_proof" title="Cantor's first uncountability proof">first proof</a> of the uncountability of the <a href="/wiki/Real_number" title="Real number">real numbers</a>, which appeared in 1874.<sup class="reference" id="cite_ref-4"><a href="#cite_note-4">[4]</a></sup><sup class="reference" id="cite_ref-Bloch2011_5-0"><a href="#cite_note-Bloch2011-5">[5]</a></sup> However, it demonstrates a powerful and general technique that has since been used in a wide range of proofs, including the first of <a href="/wiki/G%C3%B6del%27s_incompleteness_theorems" title="Gödel's incompleteness theorems">Gödel's incompleteness theorems</a><sup class="reference" id="cite_ref-Simmons1993_2-1"><a href="#cite_note-Simmons1993-2">[2]</a></sup> and Turing's answer to the <i><a href="/wiki/Entscheidungsproblem" title="Entscheidungsproblem">Entscheidungsproblem</a></i>. Diagonalization arguments are often also the source of contradictions like <a href="/wiki/Russell%27s_paradox" title="Russell's paradox">Russell's paradox</a><sup class="reference" id="cite_ref-6"><a href="#cite_note-6">[6]</a></sup><sup class="reference" id="cite_ref-7"><a href="#cite_note-7">[7]</a></sup> and <a href="/wiki/Richard%27s_paradox" title="Richard's paradox">Richard's paradox</a>.<sup class="reference" id="cite_ref-8"><a href="#cite_note-8">[8]</a></sup></p>
<p>Historically, the diagonal argument first appeared in the work of <a href="/wiki/Paul_du_Bois-Reymond" title="Paul du Bois-Reymond">Paul du Bois-Reymond</a> in 1875.<sup class="reference" id="cite_ref-9"><a href="#cite_note-9">[9]</a></sup></p>
<p></p>

<h2><span class="mw-headline" id="Uncountable_set">Uncountable set</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Cantor%27s_diagonal_argument&amp;action=edit&amp;section=1" title="Edit section: Uncountable set">edit</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>In his 1891 article, Cantor considered the set <i>T</i> of all infinite <a class="mw-redirect" href="/wiki/Sequences" title="Sequences">sequences</a> of <a class="mw-redirect" href="/wiki/Binary_digits" title="Binary digits">binary digits</a> (i.e. each digit is zero or one). He begins with a <a href="/wiki/Constructive_proof" title="Constructive proof">constructive proof</a> of the following theorem:</p>
<dl>
<dd>If <i>s</i><sub>1</sub>, <i>s</i><sub>2</sub>, … , <i>s</i><sub><i>n</i></sub>, … is any enumeration of elements from <i>T</i>, then there is always an element <i>s</i> of <i>T</i> which corresponds to no <i>s</i><sub><i>n</i></sub> in the enumeration.</dd>
</dl>

<p>By construction, <i>s</i> differs from each <i>s</i><sub><i>n</i></sub>, since their <i>n</i><sup>th</sup> digits differ (highlighted in the example). Hence, <i>s</i> cannot occur in the enumeration.</p>
<p>Based on this theorem, Cantor then uses a <a href="/wiki/Proof_by_contradiction" title="Proof by contradiction">proof by contradiction</a> to show that:</p>
<dl>
<dd>The set <i>T</i> is uncountable.</dd>
</dl>
<p>He assumes for contradiction that <i>T</i> was countable. Then all its elements could be written as an enumeration <i>s</i><sub>1</sub>, <i>s</i><sub>2</sub>, … , <i>s</i><sub><i>n</i></sub>, … . Applying the previous theorem to this enumeration would produce a sequence <i>s</i> not belonging to the enumeration. However, <i>s</i> was an element of <i>T</i> and should therefore be in the enumeration. This contradicts the original assumption, so <i>T</i> must be uncountable.</p>
<h3><span class="mw-headline" id="Interpretation">Interpretation</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Cantor%27s_diagonal_argument&amp;action=edit&amp;section=2" title="Edit section: Interpretation">edit</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>The interpretation of Cantor's result will depend upon one's view of mathematics. To <a href="/wiki/Constructivism_(mathematics)" title="Constructivism (mathematics)">constructivists</a>, the argument shows no more than that there is no <a href="/wiki/Bijection" title="Bijection">bijection</a> between the natural numbers and <i>T</i>. It does not rule out the possibility that the latter are <a class="mw-redirect" href="/wiki/Subcountable" title="Subcountable">subcountable</a>. In the context of <a href="/wiki/Classical_mathematics" title="Classical mathematics">classical mathematics</a>, this is impossible, and the diagonal argument establishes that, although both sets are infinite, there are actually <i>more</i> infinite sequences of ones and zeros than there are natural numbers.</p>
<h3><span class="mw-headline" id="Real_numbers">Real numbers</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Cantor%27s_diagonal_argument&amp;action=edit&amp;section=3" title="Edit section: Real numbers">edit</a><span class="mw-editsection-bracket">]</span></span></h3>

<p>The uncountability of the <a href="/wiki/Real_number" title="Real number">real numbers</a> was already established by <a class="mw-redirect" href="/wiki/Cantor%27s_first_uncountability_proof" title="Cantor's first uncountability proof">Cantor's first uncountability proof</a>, but it also follows from the above result. To see this, we will build a one-to-one correspondence between the set <i>T</i> of infinite binary strings and a <a href="/wiki/Subset" title="Subset">subset</a> of <b>R</b> (the set of real numbers). Since <i>T</i> is uncountable, this subset of <b>R</b> must be uncountable. Hence <b>R</b> is uncountable.</p>
<p>To build this one-to-one correspondence (or <a href="/wiki/Bijection" title="Bijection">bijection</a>), observe that the string <i>t</i> = 0111… appears after the <a class="mw-redirect" href="/wiki/Binary_point" title="Binary point">binary point</a> in the <a class="mw-redirect" href="/wiki/Binary_numeral_system#Fractions_in_binary" title="Binary numeral system">binary expansion</a> 0.0111…. This suggests defining the function <i>f</i>(<i>t</i>) = 0.<i>t</i>, where <i>t</i> is a string in <i>T</i>. Unfortunately, <i>f</i>(1000…) = 0.1000… = 1/2, and <i>f</i>(0111…) = 0.0111… = <a class="mw-redirect" href="/wiki/Infinite_series" title="Infinite series">1/4 + 1/8 + 1/16 + …</a> = 1/2. So this function is not a bijection since two strings correspond to one number—a number having two binary expansions.</p>
<p>However, modifying this function produces a bijection from <i>T</i> to the <a href="/wiki/Interval_(mathematics)" title="Interval (mathematics)">interval</a> (0, 1)—that is, the real numbers &gt; 0 and &lt; 1. The idea is to remove the "problem" elements from <i>T</i> and (0, 1), and handle them separately. From (0, 1), remove the numbers having two binary expansions. Put these numbers in a sequence: <i>a</i> = (1/2, 1/4, 3/4, 1/8, 3/8, 5/8, 7/8, …). From <i>T</i>, remove the strings appearing after the binary point in the binary expansions of 0, 1, and the numbers in sequence <i>a</i>. Put these eventually-constant strings in a sequence: <i>b</i> = (<span style="color:#808080">000</span>…, <span style="color:#808080">111</span>…, 1<span style="color:#808080">000</span>…, 0<span style="color:#808080">111</span>…, 01<span style="color:#808080">000</span>…, 00<span style="color:#808080">111</span>…, 11<span style="color:#808080">000</span>…, 10<span style="color:#808080">111</span>…, ...). A bijection <i>g</i>(<i>t</i>) from <i>T</i> to (0, 1) is defined by: If <i>t</i> is the <i>n</i><sup>th</sup> string in sequence <i>b</i>, let <i>g</i>(<i>t</i>) be the <i>n</i><sup>th</sup> number in sequence <i>a</i>; otherwise, let <i>g</i>(<i>t</i>) = 0.<i>t</i>.</p>
<p>To build a bijection from <i>T</i> to <b>R</b>: start with the <a href="/wiki/Trigonometric_functions" title="Trigonometric functions">tangent function</a> tan(<i>x</i>), which provides a bijection from (−π/2, π/2) to <b>R</b>; see right picture. Next observe that the <a href="/wiki/Linear_function" title="Linear function">linear function</a> <i>h</i>(<i>x</i>) = π<i>x</i> – π/2 provides a bijection from (0, 1) to (−π/2, π/2); see left picture. The <a href="/wiki/Function_composition" title="Function composition">composite function</a> tan(<i>h</i>(<i>x</i>)) = tan(π<i>x</i> – π/2) provides a bijection from (0, 1) to <b>R</b>. Compose this function with <i>g</i>(<i>t</i>) to obtain tan(<i>h</i>(<i>g</i>(<i>t</i>))) = tan(π<i>g</i>(<i>t</i>) – π/2), which is a bijection from <i>T</i> to <b>R</b>. This means that <i>T</i> and <b>R</b> have the same <a href="/wiki/Cardinality" title="Cardinality">cardinality</a>—this cardinality is called the "<a href="/wiki/Cardinality_of_the_continuum" title="Cardinality of the continuum">cardinality of the continuum</a>."</p>
<h2><span class="mw-headline" id="General_sets">General sets</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Cantor%27s_diagonal_argument&amp;action=edit&amp;section=4" title="Edit section: General sets">edit</a><span class="mw-editsection-bracket">]</span></span></h2>
<div class="thumb tright">
<div class="thumbinner" style="width:252px;"><a class="image" href="/wiki/File:Diagonal_argument_powerset_svg.svg"><img alt="" class="thumbimage" data-file-height="230" data-file-width="266" height="216" src="//upload.wikimedia.org/wikipedia/commons/thumb/c/c7/Diagonal_argument_powerset_svg.svg/250px-Diagonal_argument_powerset_svg.svg.png" srcset="//upload.wikimedia.org/wikipedia/commons/thumb/c/c7/Diagonal_argument_powerset_svg.svg/375px-Diagonal_argument_powerset_svg.svg.png 1.5x, //upload.wikimedia.org/wikipedia/commons/thumb/c/c7/Diagonal_argument_powerset_svg.svg/500px-Diagonal_argument_powerset_svg.svg.png 2x" width="250"/></a>
<div class="thumbcaption">
<div class="magnify"><a class="internal" href="/wiki/File:Diagonal_argument_powerset_svg.svg" title="Enlarge"></a></div>

The example mapping <i>f</i> happens to correspond to the example enumeration <i>s</i> in the <a href="#Lead">above</a> picture.</div>
</div>
</div>
<p>A generalized form of the diagonal argument was used by Cantor to prove <a href="/wiki/Cantor%27s_theorem" title="Cantor's theorem">Cantor's theorem</a>: for every <a href="/wiki/Set_(mathematics)" title="Set (mathematics)">set</a> <i>S</i> the <a href="/wiki/Power_set" title="Power set">power set</a> of <i>S</i>; that is, the set of all <a href="/wiki/Subset" title="Subset">subsets</a> of <i>S</i> (here written as <i><b>P</b></i>(<i>S</i>)), has a larger cardinality than <i>S</i> itself. This proof proceeds as follows:</p>
<p>Let <i>f</i> be any <a href="/wiki/Function_(mathematics)" title="Function (mathematics)">function</a> from <i>S</i> to <i><b>P</b></i>(<i>S</i>). It suffices to prove <i>f</i> cannot be <a class="mw-redirect" href="/wiki/Surjective" title="Surjective">surjective</a>. That means that some member <i>T</i> of <i><b>P</b></i>(<i>S</i>), i.e. some subset of <i>S</i>, is not in the <a href="/wiki/Image_(mathematics)" title="Image (mathematics)">image</a> of <i>f</i>. As a candidate consider the set:</p>
<dl>
<dd><i>T</i> = { <i>s</i> ∈ <i>S</i>: <i>s</i> ∉ <i>f</i>(<i>s</i>) }.</dd>
</dl>
<p>For every <i>s</i> in <i>S</i>, either <i>s</i> is in <i>T</i> or not. If <i>s</i> is in <i>T</i>, then by definition of <i>T</i>, <i>s</i> is not in <i>f</i>(<i>s</i>), so <i>T</i> is not equal to <i>f</i>(<i>s</i>). On the other hand, if <i>s</i> is not in <i>T</i>, then by definition of <i>T</i>, <i>s</i> is in <i>f</i>(<i>s</i>), so again <i>T</i> is not equal to <i>f</i>(<i>s</i>); cf. picture. For a more complete account of this proof, see <a href="/wiki/Cantor%27s_theorem" title="Cantor's theorem">Cantor's theorem</a>.</p>
<h3><span class="mw-headline" id="Consequences">Consequences</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Cantor%27s_diagonal_argument&amp;action=edit&amp;section=5" title="Edit section: Consequences">edit</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>This result implies that the notion of the <a class="mw-redirect" href="/wiki/Set_of_all_sets" title="Set of all sets">set of all sets</a> is an inconsistent notion. If <i>S</i> were the set of all sets then <i><b>P</b></i>(<i>S</i>) would at the same time be bigger than <i>S</i> and a subset of <i>S</i>.</p>
<p><a class="mw-redirect" href="/wiki/Russell%27s_Paradox" title="Russell's Paradox">Russell's Paradox</a> has shown us that <a href="/wiki/Naive_set_theory" title="Naive set theory">naive set theory</a>, based on an <a class="mw-redirect" href="/wiki/Unrestricted_comprehension" title="Unrestricted comprehension">unrestricted comprehension</a> scheme, is contradictory. Note that there is a similarity between the construction of <i>T</i> and the set in Russell's paradox. Therefore, depending on how we modify the axiom scheme of comprehension in order to avoid Russell's paradox, arguments such as the non-existence of a set of all sets may or may not remain valid.</p>
<p>The diagonal argument shows that the set of real numbers is "bigger" than the set of natural numbers (and therefore, the integers and rationals as well). Therefore, we can ask if there is a set whose <a href="/wiki/Cardinality" title="Cardinality">cardinality</a> is "between" that of the integers and that of the reals. This question leads to the famous <a href="/wiki/Continuum_hypothesis" title="Continuum hypothesis">continuum hypothesis</a>. Similarly, the question of whether there exists a set whose cardinality is between |<i>S</i>| and |<i><b>P</b></i>(<i>S</i>)| for some infinite <i>S</i> leads to the <a class="mw-redirect" href="/wiki/Generalized_continuum_hypothesis" title="Generalized continuum hypothesis">generalized continuum hypothesis</a>.</p>
<p>Analogues of the diagonal argument are widely used in mathematics to prove the existence or nonexistence of certain objects. For example, the conventional proof of the unsolvability of the <a href="/wiki/Halting_problem" title="Halting problem">halting problem</a> is essentially a diagonal argument. Also, diagonalization was originally used to show the existence of arbitrarily hard <a class="mw-redirect" href="/wiki/Complexity_classes" title="Complexity classes">complexity classes</a> and played a key role in early attempts to prove <a class="mw-redirect" href="/wiki/P_versus_NP" title="P versus NP">P does not equal NP</a>.</p>
<h3><span class="mw-headline" id="Version_for_Quine.27s_New_Foundations">Version for Quine's New Foundations</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Cantor%27s_diagonal_argument&amp;action=edit&amp;section=6" title="Edit section: Version for Quine's New Foundations">edit</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>The above proof fails for <a class="mw-redirect" href="/wiki/W._V._Quine" title="W. V. Quine">W. V. Quine</a>'s "<a href="/wiki/New_Foundations" title="New Foundations">New Foundations</a>" set theory (NF). In NF, the <a class="mw-redirect" href="/wiki/Unrestricted_comprehension" title="Unrestricted comprehension">naive axiom scheme of comprehension</a> is modified to avoid the paradoxes by introducing a kind of "local" <a href="/wiki/Type_theory" title="Type theory">type theory</a>. In this axiom scheme,</p>
<dl>
<dd>{ <i>s</i> ∈ <i>S</i>: <i>s</i> ∉ <i>f</i>(<i>s</i>) }</dd>
</dl>
<p>is <i>not</i> a set — i.e., does not satisfy the axiom scheme. On the other hand, we might try to create a modified diagonal argument by noticing that</p>
<dl>
<dd>{ <i>s</i> ∈ <i>S</i>: <i>s</i> ∉ <i>f</i>({<i>s</i>}) }</dd>
</dl>
<p><i>is</i> a set in NF. In which case, if <i><b>P</b></i><sub>1</sub>(<i>S</i>) is the set of one-element subsets of <i>S</i> and <i>f</i> is a proposed bijection from <i><b>P</b></i><sub>1</sub>(<i>S</i>) to <i><b>P</b></i>(<i>S</i>), one is able to use <a href="/wiki/Proof_by_contradiction" title="Proof by contradiction">proof by contradiction</a> to prove that |<i><b>P</b></i><sub>1</sub>(<i>S</i>)| &lt; |<i><b>P</b></i>(<i>S</i>)|.</p>
<p>The proof follows by the fact that if <i>f</i> were indeed a map <i>onto</i> <i><b>P</b></i>(<i>S</i>), then we could find <i>r</i> in <i>S</i>, such that <i>f</i>({<i>r</i>}) coincides with the modified diagonal set, above. We would conclude that if <i>r</i> is not in <i>f</i>({<i>r</i>}), then <i>r</i> is in <i>f</i>({<i>r</i>}) and vice versa.</p>
<p>It is <i>not</i> possible to put <i><b>P</b></i><sub>1</sub>(<i>S</i>) in a one-to-one relation with <i>S</i>, as the two have different types, and so any function so defined would violate the typing rules for the comprehension scheme.</p>
