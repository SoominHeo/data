<p><b>Deltaproteobacteria</b> is a class of <a href="/wiki/Proteobacteria" title="Proteobacteria">Proteobacteria</a>.<sup class="reference" id="cite_ref-urlwww.ncbi.nlm.nih.gov_1-0"><a href="#cite_note-urlwww.ncbi.nlm.nih.gov-1">[1]</a></sup> All <a href="/wiki/Species" title="Species">species</a> of this group are, like all <a href="/wiki/Proteobacteria" title="Proteobacteria">Proteobacteria</a>, <a class="mw-redirect" href="/wiki/Gram-negative" title="Gram-negative">gram-negative</a>.</p>
<p>The <i>Deltaproteobacteria</i> comprise a branch of predominantly aerobic genera, the fruiting-body-forming <i><a href="/wiki/Myxobacteria" title="Myxobacteria">Myxobacteria</a></i> which release <a class="mw-redirect" href="/wiki/Myxospore" title="Myxospore">myxospores</a> in unfavorable environments, and a branch of strictly anaerobic genera, which contains most of the known <a href="/wiki/Sulfate-reducing_bacteria" title="Sulfate-reducing bacteria">sulfate-</a> (<i>Desulfovibrio</i>, <i>Desulfobacter</i>, <i>Desulfococcus</i>, <i>Desulfonema</i>, etc.) and <a href="/wiki/Sulfur-reducing_bacteria" title="Sulfur-reducing bacteria">sulfur-reducing bacteria</a> (e.g. <i>Desulfuromonas</i> spp.) alongside several other anaerobic bacteria with different physiology (e.g. ferric iron-reducing <i><a href="/wiki/Geobacter" title="Geobacter">Geobacter</a></i> spp. and <a class="mw-redirect" href="/wiki/Syntrophic" title="Syntrophic">syntrophic</a> <i><a href="/wiki/Pelobacter" title="Pelobacter">Pelobacter</a></i> and <i>Syntrophus</i> spp.).</p>
<p>A pathogenic intracellular Deltaproteobacteria has recently been identified.<sup class="reference" id="cite_ref-pmid18606736_2-0"><a href="#cite_note-pmid18606736-2">[2]</a></sup></p>
<h2><span class="mw-headline" id="Phylogeny">Phylogeny</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Deltaproteobacteria&amp;action=edit&amp;section=1" title="Edit section: Phylogeny">edit</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>The currently accepted taxonomy is based on the <a href="/wiki/List_of_Prokaryotic_names_with_Standing_in_Nomenclature" title="List of Prokaryotic names with Standing in Nomenclature">List of Prokaryotic names with Standing in Nomenclature</a> (LPSN) <sup class="reference" id="cite_ref-3"><a href="#cite_note-3">[3]</a></sup> and <a href="/wiki/National_Center_for_Biotechnology_Information" title="National Center for Biotechnology Information">National Center for Biotechnology Information</a> (NCBI)<sup class="reference" id="cite_ref-4"><a href="#cite_note-4">[4]</a></sup> and the phylogeny is based on 16S rRNA-based LTP release 111 by <a href="/wiki/%27The_All-Species_Living_Tree%27_Project" title="'The All-Species Living Tree' Project">'The All-Species Living Tree' Project</a> <sup class="reference" id="cite_ref-5"><a href="#cite_note-5">[5]</a></sup></p>
