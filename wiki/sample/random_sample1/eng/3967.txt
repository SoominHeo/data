<p><b>Dolphin</b> is a free and <a class="mw-redirect" href="/wiki/Open_source_software" title="Open source software">open source</a> <a href="/wiki/File_manager" title="File manager">file manager</a> included in the <a href="/wiki/KDE_Applications" title="KDE Applications">KDE Applications</a> bundle which contains applications used primarily with the <a href="/wiki/KDE_Plasma_5" title="KDE Plasma 5">KDE Plasma 5</a> desktop. Dolphin became the default file manager of <a href="/wiki/KDE" title="KDE">KDE</a>-based desktop environments in the fourth iteration, termed <a href="/wiki/KDE_Software_Compilation_4" title="KDE Software Compilation 4">KDE Software Compilation 4</a>.<sup class="reference" id="cite_ref-4"><a href="#cite_note-4">[4]</a></sup> It could also be optionally installed on <a href="/wiki/K_Desktop_Environment_3" title="K Desktop Environment 3">K Desktop Environment 3</a>. <a href="/wiki/Konqueror" title="Konqueror">Konqueror</a> was replaced as the default file manager for KDE SC 4, but is still the default <a href="/wiki/Web_browser" title="Web browser">web browser</a> and can be used as an alternative file manager for <a href="/wiki/Power_user" title="Power user">power users</a>.</p>
<p>Under previous K Desktop Environment versions, Konqueror had served both as the default file manager and web browser. However, for many years users have been critical of Konqueror as being too complex for simple file navigation. As a response, the two functions were divided into two separate applications. Under KDE SC 4, Dolphin was streamlined for browsing files, while sharing as much code as possible with Konqueror. Konqueror continues to be developed primarily as a web browser.<sup class="reference" id="cite_ref-5"><a href="#cite_note-5">[5]</a></sup></p>
<p>In 2014, work started on porting Dolphin to <a class="mw-redirect" href="/wiki/KDE_Frameworks_5" title="KDE Frameworks 5">KDE Frameworks 5</a>.<sup class="reference" id="cite_ref-6"><a href="#cite_note-6">[6]</a></sup> This is now complete and a Frameworks 5-based version was released as part of KDE Applications 15.08 in August 2015.<sup class="reference" id="cite_ref-7"><a href="#cite_note-7">[7]</a></sup></p>

<h2><span class="mw-headline" id="Dolphin_and_K_Desktop_Environment_3">Dolphin and K Desktop Environment 3</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Dolphin_(file_manager)&amp;action=edit&amp;section=1" title="Edit section: Dolphin and K Desktop Environment 3">edit</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>As development of the KDE SC 4 version was underway, the K Desktop Environment 3 version of Dolphin was discontinued. However, the program continues to be unofficially available for K Desktop Environment 3 under the slightly modified name of “D3lphin”. D3lphin contains many bugfixes and a new <a href="/wiki/Sidebar_(computing)" title="Sidebar (computing)">sidebar</a>, but is no longer maintained.<sup class="reference" id="cite_ref-8"><a href="#cite_note-8">[8]</a></sup></p>
<h2><span class="mw-headline" id="Features">Features</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Dolphin_(file_manager)&amp;action=edit&amp;section=2" title="Edit section: Features">edit</a><span class="mw-editsection-bracket">]</span></span></h2>
<ul>
<li><a href="/wiki/Breadcrumb_(navigation)" title="Breadcrumb (navigation)">Breadcrumb</a> navigation bar – each part of the URL is clickable</li>
<li>3 view modes (Icons, Details and Compact), remembered for each folder</li>
<li>File Previews</li>
<li>Split views (for copying and moving files)</li>
<li>Network transparency – using KDE’s <a href="/wiki/KIO" title="KIO">KIO</a> slaves</li>
<li>Undo/Redo functionality</li>
<li>Tabbed navigation</li>
<li>Renaming of a variable number of selected items in one step</li>
<li>Baloo integration (including search, tagging, rating and commenting files)</li>
<li>Places bar which also integrates with the Kickoff launcher menu's “Computer” tab</li>
<li>Sorting and grouping of files by name, size, type and others</li>
</ul>
