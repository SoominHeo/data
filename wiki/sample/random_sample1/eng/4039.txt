<h1 class="firstHeading" id="firstHeading" lang="en">Homogeneous polynomial</h1>
<div class="mw-body-content" id="bodyContent">

<div class="mw-content-ltr" dir="ltr" id="mw-content-text" lang="en"><script>function mfTempOpenSection(id){var block=document.getElementById("mf-section-"+id);block.className+=" open-block";block.previousSibling.className+=" open-block";}</script><p>In <a href="/wiki/Mathematics" title="Mathematics">mathematics</a>, a <b>homogeneous polynomial</b> is a <a href="/wiki/Polynomial" title="Polynomial">polynomial</a> whose nonzero terms all have the same <a href="/wiki/Degree_of_a_polynomial" title="Degree of a polynomial">degree</a>.<sup class="reference" id="cite_ref-1"><a href="#cite_note-1">[1]</a></sup> For example, <span><span class="mwe-math-mathml-inline mwe-math-mathml-a11y mw-math-element" style="display: none;"><math xmlns="http://www.w3.org/1998/Math/MathML">

A polynomial is homogeneous if and only if it defines a <a href="/wiki/Homogeneous_function" title="Homogeneous function">homogeneous function</a>. An <b>algebraic form</b>, or simply <b>form</b>, is a <a href="/wiki/Function_(mathematics)" title="Function (mathematics)">function</a> defined by a homogeneous polynomial.<sup class="reference" id="cite_ref-2"><a href="#cite_note-2">[2]</a></sup> A <b>binary form</b> is a form in two variables. A <i>form</i> is also a function defined on a <a href="/wiki/Vector_space" title="Vector space">vector space</a>, which may be expressed as a homogeneous function of the coordinates over any <a href="/wiki/Basis_(linear_algebra)" title="Basis (linear algebra)">basis</a>.</p>
<p>A polynomial of degree 0 is always homogeneous; it is simply an element of the <a href="/wiki/Field_(mathematics)" title="Field (mathematics)">field</a> or <a href="/wiki/Ring_(mathematics)" title="Ring (mathematics)">ring</a> of the coefficients, usually called a constant or a scalar. A form of degree 1 is a linear form.<sup class="reference" id="cite_ref-3"><a href="#cite_note-3">[3]</a></sup> A form of degree 2 is a <a href="/wiki/Quadratic_form" title="Quadratic form">quadratic form</a>. In <a href="/wiki/Geometry" title="Geometry">geometry</a>, the <a href="/wiki/Euclidean_distance" title="Euclidean distance">Euclidean distance</a> is the <a href="/wiki/Square_root" title="Square root">square root</a> of a quadratic form.</p>
<p>Homogeneous polynomials are ubiquitous in mathematics and physics.<sup class="reference" id="cite_ref-4"><a href="#cite_note-4">[4]</a></sup> They play a fundamental role in algebraic geometry, as a <a class="mw-redirect" href="/wiki/Projective_algebraic_variety" title="Projective algebraic variety">projective algebraic variety</a> is defined as the set of the common zeros of a set of homogeneous polynomials.</p>

<h2><span class="mw-headline" id="Algebraic_forms_in_general">Algebraic forms in general</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Homogeneous_polynomial&amp;action=edit&amp;section=3" title="Edit section: Algebraic forms in general">edit</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>Algebraic forms, or simply forms, generalize <a href="/wiki/Quadratic_form" title="Quadratic form">quadratic forms</a> to any degree, and have in the past also been known as <i>quantics</i> (a term that originated with <a href="/wiki/Arthur_Cayley" title="Arthur Cayley">Cayley</a>). To specify a type of form, one has to give the degree <i>d</i> and the number of variables <i>n</i>. A form is <i>over</i> some given <a href="/wiki/Field_(mathematics)" title="Field (mathematics)">field</a> <i>K</i>, if it maps from <i>K</i><sup><i>n</i></sup> to <i>K</i>, where <i>n</i> is the number of variables of the form.</p>
<p>A form <i>f</i> over some field <i>K</i> in <i>n</i> variables represents 0 if there exists an element (<i>x</i><sub>1</sub>, ..., <i>x<sub>n</sub></i>) in <i>K</i><sup><i>n</i></sup> such that <i>f</i>(<i>x</i><sub>1</sub>,...,<i>x</i><sub><i>n</i></sub>) = 0 and at least one of the <i>x</i><sub><i>i</i></sub> is not equal to zero.</p>
<p>A quadratic form over the field of the <a href="/wiki/Real_number" title="Real number">real numbers</a> represents 0 if and only if it is not <a href="/wiki/Definite_quadratic_form" title="Definite quadratic form">definite</a>.</p>
