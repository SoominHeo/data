<p>Year <b>200</b> (<b><a href="/wiki/Roman_numerals" title="Roman numerals">CC</a></b>) was a <a href="/wiki/Leap_year_starting_on_Tuesday" title="Leap year starting on Tuesday">leap year starting on Tuesday</a> (link will display the full calendar) of the <a href="/wiki/Julian_calendar" title="Julian calendar">Julian calendar</a>. At the time, it was known as the <b>Year of the Consulship of Severus and Victorinus</b> (or, less frequently, <b>year 953 <i><a href="/wiki/Ab_urbe_condita" title="Ab urbe condita">Ab urbe condita</a></i></b>). The denomination 200 for this year has been used since the early medieval period, when the <a href="/wiki/Anno_Domini" title="Anno Domini">Anno Domini</a> <a href="/wiki/Calendar_era" title="Calendar era">calendar era</a> became the prevalent method in Europe for naming years.</p>

<h3><span class="mw-headline" id="By_place">By place</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=200&amp;action=edit&amp;section=2" title="Edit section: By place">edit</a><span class="mw-editsection-bracket">]</span></span></h3>
<h4><span class="mw-headline" id="World">World</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=200&amp;action=edit&amp;section=3" title="Edit section: World">edit</a><span class="mw-editsection-bracket">]</span></span></h4>
<ul>
<li><a href="/wiki/World_population" title="World population">Human population</a> reaches about 257 million.</li>
</ul>
<h4><span class="mw-headline" id="Roman_Empire">Roman Empire</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=200&amp;action=edit&amp;section=4" title="Edit section: Roman Empire">edit</a><span class="mw-editsection-bracket">]</span></span></h4>
<ul>
<li>Emperor <a href="/wiki/Septimius_Severus" title="Septimius Severus">Septimius Severus</a> visits the provinces of <a class="mw-redirect" href="/wiki/Syria_(Roman_province)" title="Syria (Roman province)">Syria</a>, <a href="/wiki/Syria_Palaestina" title="Syria Palaestina">Palestine</a>, and <a href="/wiki/Arabia_Petraea" title="Arabia Petraea">Arabia</a>.</li>
<li>Palestine, benefiting from the benevolent policies of Severus, sees a significant economic revival.</li>
<li>The province of <a href="/wiki/Numidia" title="Numidia">Numidia</a> is taken from the African <a href="/wiki/Proconsul" title="Proconsul">proconsul</a> and made an <a href="/wiki/Imperial_province" title="Imperial province">Imperial province</a>.</li>
</ul>
<h4><span class="mw-headline" id="India">India</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=200&amp;action=edit&amp;section=5" title="Edit section: India">edit</a><span class="mw-editsection-bracket">]</span></span></h4>
<ul>
<li><a href="/wiki/Rudrasena_I_(Saka_king)" title="Rudrasena I (Saka king)">Rudrasena I</a>, <a href="/wiki/Saka" title="Saka">Saka</a> ruler of the <a href="/wiki/Western_Satraps" title="Western Satraps">Western Satrap</a> <a href="/wiki/Dynasty" title="Dynasty">dynasty</a>, becomes <a href="/wiki/King" title="King">king</a> of <a href="/wiki/Malwa" title="Malwa">Malwa</a> in ancient <a class="mw-redirect" href="/wiki/Classical_India" title="Classical India">India</a>.</li>
</ul>
<h4><span class="mw-headline" id="Asia">Asia</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=200&amp;action=edit&amp;section=6" title="Edit section: Asia">edit</a><span class="mw-editsection-bracket">]</span></span></h4>
<ul>
<li>Chinese <a href="/wiki/Warlord" title="Warlord">warlord</a> <a href="/wiki/Cao_Cao" title="Cao Cao">Cao Cao</a> defeats <a href="/wiki/Yuan_Shao" title="Yuan Shao">Yuan Shao</a> in the <a href="/wiki/Battle_of_Guandu" title="Battle of Guandu">Battle of Guandu</a>.</li>
<li>In <a href="/wiki/Yayoi_period" title="Yayoi period">Japan</a>, <a class="mw-redirect" href="/wiki/Himiko_(queen)" title="Himiko (queen)">Himiko</a>, whose <a href="/wiki/Capital_city" title="Capital city">capital</a> is situated in <a href="/wiki/Yamatai" title="Yamatai">Yamatai</a>, extends her authority over a number of <a href="/wiki/Clan" title="Clan">clans</a>.</li>
</ul>
<h4><span class="mw-headline" id="America">America</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=200&amp;action=edit&amp;section=7" title="Edit section: America">edit</a><span class="mw-editsection-bracket">]</span></span></h4>
<ul>
<li>The Classic Age of <a href="/wiki/Maya_civilization" title="Maya civilization">Maya civilization</a> begins (around this year).</li>
<li>The <a href="/wiki/Paracas_culture" title="Paracas culture">Paracas culture</a> in the <a href="/wiki/Andes" title="Andes">Andes</a> comes to an end (around this year).</li>
</ul>

<h4><span class="mw-headline" id="Art">Art</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=200&amp;action=edit&amp;section=9" title="Edit section: Art">edit</a><span class="mw-editsection-bracket">]</span></span></h4>
<ul>
<li>The <a href="/wiki/Severan_Tondo" title="Severan Tondo">Severan Tondo</a>, depicting <a href="/wiki/Septimius_Severus" title="Septimius Severus">Septimius Severus</a>, <a href="/wiki/Julia_Domna" title="Julia Domna">Julia Domna</a> and their children <a class="mw-redirect" href="/wiki/Publius_Septimius_Geta" title="Publius Septimius Geta">Geta</a> and <a href="/wiki/Caracalla" title="Caracalla">Caracalla</a>, from <a class="mw-redirect" href="/wiki/Fayum" title="Fayum">Fayum</a>, <a href="/wiki/Egypt" title="Egypt">Egypt</a>, is made. It is now kept at <a class="mw-redirect" href="/wiki/Staatliche_Museen_zu_Berlin" title="Staatliche Museen zu Berlin">Staatliche Museen zu Berlin</a>, <a class="mw-redirect" href="/wiki/Preussischer_Kulturbesitz" title="Preussischer Kulturbesitz">Preussischer Kulturbesitz</a>, <a class="mw-redirect" href="/wiki/Antikensammlung" title="Antikensammlung">Antikensammlung</a>.</li>
</ul>
<h4><span class="mw-headline" id="Religion">Religion</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=200&amp;action=edit&amp;section=10" title="Edit section: Religion">edit</a><span class="mw-editsection-bracket">]</span></span></h4>
<ul>
<li><a class="mw-redirect" href="/wiki/Jew" title="Jew">Jewish</a> <a class="mw-redirect" href="/wiki/Eretz_Yisrael" title="Eretz Yisrael">Eretz Yisraeli</a> scholar <a class="mw-redirect" href="/wiki/Judah_ha-Nasi" title="Judah ha-Nasi">Judah ha-Nasi</a> compiles tracts of the <a href="/wiki/Mishnah" title="Mishnah">Mishnah</a>, beginning the creation of <a href="/wiki/Talmud" title="Talmud">Talmudic</a> law.</li>

</ul>
