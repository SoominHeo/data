<p><b>.ge</b> is the <a href="/wiki/Country_code_top-level_domain" title="Country code top-level domain">country code top-level domain</a> (ccTLD) for <a href="/wiki/Georgia_(country)" title="Georgia (country)">Georgia</a>.</p>
<p><b>.ge</b> top-level domain names are available for registration for residents of Georgia (unlimited) or for foreign companies via representation of any local legal person (limited to one domain name per registrant).<sup class="reference" id="cite_ref-1"><a href="#cite_note-1">[1]</a></sup> Second-level domain names are also available for registration for several specific types of registrants:<sup class="reference" id="cite_ref-2"><a href="#cite_note-2">[2]</a></sup></p>
<ul>
<li><span style="font-family: monospace, monospace;">.com.ge</span> - only for commercial organizations</li>
<li><span style="font-family: monospace, monospace;">.edu.ge</span> - only for educational institutions</li>
<li><span style="font-family: monospace, monospace;">.gov.ge</span> - only for governmental organizations</li>
<li><span style="font-family: monospace, monospace;">.org.ge</span> - only for non-governmental organizations</li>
<li><span style="font-family: monospace, monospace;">.mil.ge</span> - only for state military organizations</li>
<li><span style="font-family: monospace, monospace;">.net.ge</span> - only for network provider or network project organizations</li>
<li><span style="font-family: monospace, monospace;">.pvt.ge</span> - only for private persons</li>
</ul>

<h2><span class="mw-headline" id="History">History</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=.ge&amp;action=edit&amp;section=1" title="Edit section: History">edit</a><span class="mw-editsection-bracket">]</span></span></h2>
<p><b>.ge</b> was delegated to Georgian <a class="mw-redirect" href="/wiki/ISP" title="ISP">ISP</a> SaNet in 1992. In 2006 <a href="/wiki/Caucasus_Online" title="Caucasus Online">Caucasus Online</a> has been formed after merger of 3 companies (including SaNet) and became sponsor of <span style="font-family: monospace, monospace;">.ge</span>.<sup class="reference" id="cite_ref-3"><a href="#cite_note-3">[3]</a></sup> Registration service was initially provided by non-profit organization Internet Development Group - Georgia, but after introduction of <a href="/wiki/Caucasus_Online" title="Caucasus Online">Caucasus Online</a> it exclusively consolidated both administrative and registrar roles.<br/>
<b>.gov.ge</b> was delegated to LEPL Smart Logic in 2014.</p>
<h2><span class="mw-headline" id="..E1.83.92.E1.83.94">.გე</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=.ge&amp;action=edit&amp;section=2" title="Edit section: .გე">edit</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>In 2011 a new top-level domain name was registered for Georgia, intended for domain names in the local language. The top-level domain is <b>.გე</b>. It became active with active web sites in 2016.</p>
<p>Starting from January 20, 2016, Georgia register s.გე top-level domains using Georgian <i><a class="mw-redirect" href="/wiki/Mkhedruli" title="Mkhedruli">Mkhedruli</a></i> script. At first, the process started for reservation of governmental, commercial TLDs but after 6 months from start date, registration of <a href="/wiki/Internationalized_domain_name" title="Internationalized domain name">IDN</a> ccTLDs became available to general public.<sup class="reference" id="cite_ref-4"><a href="#cite_note-4">[4]</a></sup></p>
