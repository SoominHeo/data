<p><b>Dragora GNU/Linux-Libre</b> is an Argentine Linux distribution<sup class="reference" id="cite_ref-3"><a href="#cite_note-3">[3]</a></sup> It is written entirely from scratch<sup class="reference" id="cite_ref-4"><a href="#cite_note-4">[4]</a></sup> sharing some similarities with <a href="/wiki/Slackware" title="Slackware">Slackware</a>. It has a very simple packaging system that allows installing, removing, upgrading and creating packages. Novice users might find Dragora a bit challenging.<sup class="reference" id="cite_ref-5"><a href="#cite_note-5">[5]</a></sup> It is one of the "entirely free GNU/Linux distros" endorsed by the <a href="/wiki/Free_Software_Foundation" title="Free Software Foundation">Free Software Foundation</a>:<sup class="reference" id="cite_ref-6"><a href="#cite_note-6">[6]</a></sup> it contains 100% <a href="/wiki/Free_software" title="Free software">free software</a> and uses a <a href="/wiki/Linux-libre" title="Linux-libre">Linux-libre</a> kernel. Dragora is considered to be based on the <a href="/wiki/KISS_principle" title="KISS principle">"Keep it simple, stupid" (KISS) principle</a>, believed by the authors to be a strength.<sup class="reference" id="cite_ref-7"><a href="#cite_note-7">[7]</a></sup> Dragora can be downloaded from the web site or bought on CD.<sup class="reference" id="cite_ref-8"><a href="#cite_note-8">[8]</a></sup><sup class="reference" id="cite_ref-9"><a href="#cite_note-9">[9]</a></sup></p>
