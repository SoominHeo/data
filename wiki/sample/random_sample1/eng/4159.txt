<h2><span class="mw-headline" id="WWE_Network">WWE Network</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=List_of_current_WWE_programming&amp;action=edit&amp;section=2" title="Edit section: WWE Network">edit</a><span class="mw-editsection-bracket">]</span></span></h2>
<div class="hatnote" role="note">Main article: <a href="/wiki/WWE_Network" title="WWE Network">WWE Network</a></div>
<p>On January 8, 2014, WWE announced the network would launch on February 24, 2014 in the <a href="/wiki/United_States" title="United States">United States</a>. On July 31, 2014, WWE announced that the network would be expected to go live in <a href="/wiki/Australia" title="Australia">Australia</a>, <a href="/wiki/Canada" title="Canada">Canada</a>, <a href="/wiki/New_Zealand" title="New Zealand">New Zealand</a>, <a href="/wiki/Hong_Kong" title="Hong Kong">Hong Kong</a>, <a href="/wiki/Singapore" title="Singapore">Singapore</a>, <a href="/wiki/Mexico" title="Mexico">Mexico</a>, <a href="/wiki/Spain" title="Spain">Spain</a>, and the <a href="/wiki/Nordic_countries" title="Nordic countries">Nordics</a>, among other countries starting on August 12.<sup class="reference" id="cite_ref-24"><a href="#cite_note-24">[24]</a></sup> WWE Network arrived in the <a href="/wiki/United_Kingdom" title="United Kingdom">United Kingdom</a> on January 13, 2015, with expansion to <a href="/wiki/Italy" title="Italy">Italy</a>, <a href="/wiki/United_Arab_Emirates" title="United Arab Emirates">United Arab Emirates</a>, <a href="/wiki/Germany" title="Germany">Germany</a>, <a href="/wiki/Japan" title="Japan">Japan</a>, <a href="/wiki/India" title="India">India</a>, <a href="/wiki/China" title="China">China</a>, <a href="/wiki/Thailand" title="Thailand">Thailand</a>, <a href="/wiki/Philippines" title="Philippines">Philippines</a>, and <a href="/wiki/Malaysia" title="Malaysia">Malaysia</a> following at a future date.</p>
