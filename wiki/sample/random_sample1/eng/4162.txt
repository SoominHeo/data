<h1 class="firstHeading" id="firstHeading" lang="en">Dispatch table</h1>
<div class="mw-body-content" id="bodyContent">
<div id="siteSub">From Wikipedia, the free encyclopedia</div>
<div id="contentSub"></div>
<div class="mw-jump" id="jump-to-nav">
					Jump to:					<a href="#mw-head">navigation</a>, 					<a href="#p-search">search</a>
</div>
<div class="mw-content-ltr" dir="ltr" id="mw-content-text" lang="en"><script>function mfTempOpenSection(id){var block=document.getElementById("mf-section-"+id);block.className+=" open-block";block.previousSibling.className+=" open-block";}</script><p>In <a href="/wiki/Computer_science" title="Computer science">computer science</a>, a <b>dispatch table</b> is a table of <a href="/wiki/Pointer_(computer_programming)" title="Pointer (computer programming)">pointers</a> to functions or <a class="mw-redirect" href="/wiki/Method_(computer_science)" title="Method (computer science)">methods</a>. Use of such a table is a common technique when implementing <a href="/wiki/Late_binding" title="Late binding">late binding</a> in <a href="/wiki/Object-oriented_programming" title="Object-oriented programming">object-oriented programming</a>.</p>

<h2><span class="mw-headline" id="Perl_implementation">Perl implementation</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Dispatch_table&amp;action=edit&amp;section=1" title="Edit section: Perl implementation">edit</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>The following shows one way to implement a dispatch table in <a href="/wiki/Perl" title="Perl">Perl</a>, using a <a href="/wiki/Associative_array#Perl" title="Associative array">hash</a> to store references to code (also known as <a href="/wiki/Function_pointer" title="Function pointer">function pointers</a>).</p>
<div class="mw-highlight mw-content-ltr" dir="ltr">
<pre>
 <span class="c1">#define the table using one anonymous code-ref and one named code-ref</span>
 <span class="k">my</span> <span class="nv">%dispatch</span> <span class="o">=</span> <span class="p">(</span>
   <span class="s">"-h"</span> <span class="o">=&gt;</span> <span class="k">sub </span><span class="p">{</span>  <span class="k">return</span> <span class="s">"hello\n"</span><span class="p">;</span> <span class="p">},</span>
   <span class="s">"-g"</span> <span class="o">=&gt;</span> <span class="o">\&amp;</span><span class="n">say_goodbye</span>
 <span class="p">);</span>
 
 <span class="k">sub </span><span class="nf">say_goodbye</span> <span class="p">{</span>
   <span class="k">return</span> <span class="s">"goodbye\n"</span><span class="p">;</span>
 <span class="p">}</span>
 
 <span class="c1">#fetch the code ref from the table, and invoke it</span>
 <span class="k">my</span> <span class="nv">$sub</span> <span class="o">=</span> <span class="nv">$dispatch</span><span class="p">{</span><span class="nv">$ARGV</span><span class="p">[</span><span class="mi">0</span><span class="p">]};</span>
 <span class="k">print</span> <span class="nv">$sub</span> <span class="p">?</span> <span class="nv">$sub</span><span class="o">-&gt;</span><span class="p">()</span> <span class="p">:</span> <span class="s">"unknown argument\n"</span><span class="p">;</span>
</pre></div>
<p>Running this Perl program as <code>perl greet -h</code> will produce "hello", and running it as <code>perl greet -g</code> will produce "goodbye".</p>
<h2><span class="mw-headline" id="Javascript_implementation">Javascript implementation</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Dispatch_table&amp;action=edit&amp;section=2" title="Edit section: Javascript implementation">edit</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>Following is a demo of implementing Dispatch table in Javascript:</p>
<div class="mw-highlight mw-content-ltr" dir="ltr">
<pre>
<span class="kd">var</span> <span class="nx">thingsWeCanDo</span> <span class="o">=</span> <span class="p">{</span>
    <span class="nx">doThisThing</span>      <span class="o">:</span> <span class="kd">function</span><span class="p">()</span> <span class="p">{</span> <span class="cm">/* behavior */</span> <span class="p">},</span>
    <span class="nx">doThatThing</span>      <span class="o">:</span> <span class="kd">function</span><span class="p">()</span> <span class="p">{</span> <span class="cm">/* behavior */</span> <span class="p">},</span>
    <span class="nx">doThisOtherThing</span> <span class="o">:</span> <span class="kd">function</span><span class="p">()</span> <span class="p">{</span> <span class="cm">/* behavior */</span> <span class="p">},</span>
    <span class="k">default</span>          <span class="o">:</span> <span class="kd">function</span><span class="p">()</span> <span class="p">{</span> <span class="cm">/* behavior */</span> <span class="p">}</span>
<span class="p">};</span>

<span class="kd">var</span> <span class="nx">doSomething</span> <span class="o">=</span> <span class="kd">function</span><span class="p">(</span><span class="nx">doWhat</span><span class="p">)</span> <span class="p">{</span>
    <span class="kd">var</span> <span class="nx">thingToDo</span> <span class="o">=</span> <span class="nx">thingsWeCanDo</span><span class="p">.</span><span class="nx">hasOwnProperty</span><span class="p">(</span><span class="nx">doWhat</span><span class="p">)</span> <span class="o">?</span> <span class="nx">doWhat</span> <span class="o">:</span> <span class="s2">"default"</span>
    <span class="nx">thingsWeCanDo</span><span class="p">[</span><span class="nx">thingToDo</span><span class="p">]();</span>
<span class="p">}</span>
</pre></div>
<h2><span class="mw-headline" id="Virtual_method_tables">Virtual method tables</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Dispatch_table&amp;action=edit&amp;section=3" title="Edit section: Virtual method tables">edit</a><span class="mw-editsection-bracket">]</span></span></h2>
<div class="hatnote" role="note">Main article: <a href="/wiki/Virtual_method_table" title="Virtual method table">Virtual method table</a></div>
<p>In <a href="/wiki/Object-oriented_programming" title="Object-oriented programming">object-oriented</a> <a href="/wiki/Programming_language" title="Programming language">programming languages</a> that support <a class="mw-redirect" href="/wiki/Virtual_method" title="Virtual method">virtual methods</a>, the <a href="/wiki/Compiler" title="Compiler">compiler</a> will automatically create a dispatch table for each object of a <a class="mw-redirect" href="/wiki/Class_(computer_science)" title="Class (computer science)">class</a> containing virtual methods. This table is called a <a href="/wiki/Virtual_method_table" title="Virtual method table">virtual method table</a> or <i>vtable</i>, and every call to a virtual method is dispatched through the vtable.</p>
