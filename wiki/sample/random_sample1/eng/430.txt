<p>The <b>percent sign</b> (<b>%</b>) is the symbol used to indicate a <a href="/wiki/Percentage" title="Percentage">percentage</a>, a number or <a href="/wiki/Ratio" title="Ratio">ratio</a> as a <a href="/wiki/Fraction_(mathematics)" title="Fraction (mathematics)">fraction</a> of 100. Related signs include the <a class="mw-redirect" href="/wiki/Permille" title="Permille">permille</a> (per thousand) sign ‰ and the <a href="/wiki/Basis_point" title="Basis point">permyriad</a> (per ten thousand) sign ‱ (also known as a <a href="/wiki/Basis_point" title="Basis point">basis point</a>), which indicate that a number is divided by one thousand or ten thousand respectively. Higher proportions use <a href="/wiki/Parts-per_notation" title="Parts-per notation">parts-per notation</a>.</p>

<h2><span class="mw-headline" id="Correct_style">Correct style</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Percent_sign&amp;action=edit&amp;section=1" title="Edit section: Correct style">edit</a><span class="mw-editsection-bracket">]</span></span></h2>
<h3><span class="mw-headline" id="Spacing">Spacing</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Percent_sign&amp;action=edit&amp;section=2" title="Edit section: Spacing">edit</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>English style guides prescribe writing the number and percent sign without any space between.<sup class="reference" id="cite_ref-1"><a href="#cite_note-1">[1]</a></sup><sup class="reference" id="cite_ref-2"><a href="#cite_note-2">[2]</a></sup><sup class="reference" id="cite_ref-3"><a href="#cite_note-3">[3]</a></sup><sup class="reference" id="cite_ref-4"><a href="#cite_note-4">[4]</a></sup><sup class="reference" id="cite_ref-5"><a href="#cite_note-5">[5]</a></sup><sup class="reference" id="cite_ref-6"><a href="#cite_note-6">[6]</a></sup><sup class="reference" id="cite_ref-7"><a href="#cite_note-7">[7]</a></sup> However, the <a href="/wiki/International_System_of_Units" title="International System of Units">International System of Units</a> and <a href="/wiki/ISO_31-0" title="ISO 31-0">ISO 31-0</a> standard prescribe a space between the number and percent sign,<sup class="reference" id="cite_ref-8"><a href="#cite_note-8">[8]</a></sup><sup class="reference" id="cite_ref-9"><a href="#cite_note-9">[9]</a></sup><sup class="reference" id="cite_ref-10"><a href="#cite_note-10">[10]</a></sup> in line with the general practice of using a <a href="/wiki/Non-breaking_space" title="Non-breaking space">non-breaking space</a> between a <a class="mw-redirect" href="/wiki/Numerical_value" title="Numerical value">numerical value</a> and its corresponding <a class="mw-redirect" href="/wiki/Unit_of_measurement" title="Unit of measurement">unit of measurement</a>.</p>
<p>Other languages have other rules for spacing in front of the percent sign:</p>
<ul>
<li>In <a href="/wiki/Czech_language" title="Czech language">Czech</a>, the percent sign is spaced with a non-breaking space if the number is used as a noun,<sup class="reference" id="cite_ref-11"><a href="#cite_note-11">[11]</a></sup> whereas no space is inserted if the number is used as an adjective (e.g. “a 50% increase”).<sup class="reference" id="cite_ref-12"><a href="#cite_note-12">[12]</a></sup></li>
<li>In <a href="/wiki/Finnish_language" title="Finnish language">Finnish</a>, the percent sign is always spaced, and a <a href="/wiki/Finnish_noun_cases" title="Finnish noun cases">case suffix</a> can be attached to it using the <a href="/wiki/Colon_(punctuation)" title="Colon (punctuation)">colon</a> (e.g. <i>50 %:n kasvu</i> 'an increase of 50%').<sup class="reference" id="cite_ref-13"><a href="#cite_note-13">[13]</a></sup></li>
<li>In <a href="/wiki/French_language" title="French language">French</a>, the percent sign must be spaced with a non-breaking space.<sup class="noprint Inline-Template Template-Fact" style="white-space:nowrap;">[<i><a href="/wiki/Wikipedia:Citation_needed" title="Wikipedia:Citation needed"><span title="This claim needs references to reliable sources. (February 2009)">citation needed</span></a></i>]</sup></li>
<li>In <a href="/wiki/Italian_language" title="Italian language">Italian</a>, the percent sign is never spaced.<sup class="reference" id="cite_ref-14"><a href="#cite_note-14">[14]</a></sup></li>
<li>In <a href="/wiki/Spanish_language" title="Spanish language">Spanish</a>, the percent sign must always be spaced now, as almost every other symbol.</li>
<li>In traditional Russian typography, the percent sign is never spaced. But it is not that common in Russia today.<sup class="noprint Inline-Template Template-Fact" style="white-space:nowrap;">[<i><a href="/wiki/Wikipedia:Citation_needed" title="Wikipedia:Citation needed"><span title="This claim needs references to reliable sources. (January 2010)">citation needed</span></a></i>]</sup></li>
<li>In <a href="/wiki/Chinese_language" title="Chinese language">Chinese</a>, the percent sign is almost never spaced, probably because Chinese does not use spaces to separate characters or words at all.<sup class="noprint Inline-Template Template-Fact" style="white-space:nowrap;">[<i><a href="/wiki/Wikipedia:Citation_needed" title="Wikipedia:Citation needed"><span title="This claim needs references to reliable sources. (January 2010)">citation needed</span></a></i>]</sup></li>
<li>According to the <a href="/wiki/Swedish_Language_Council" title="Swedish Language Council">Swedish Language Council</a>, the percent sign should be preceded by a space in <a href="/wiki/Swedish_language" title="Swedish language">Swedish</a>, as all other units.</li>
<li>In <a href="/wiki/German_language" title="German language">German</a>, the space is prescribed by the regulatory body in the national standard <a href="/wiki/DIN_5008" title="DIN 5008">DIN 5008</a>.</li>
<li>In <a href="/wiki/Persian_language" title="Persian language">Persian</a> and <a href="/wiki/Turkish_language" title="Turkish language">Turkish</a>, the percent sign precedes rather than follows the number.</li>
</ul>
<h3><span class="mw-headline" id="Usage_in_text">Usage in text</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Percent_sign&amp;action=edit&amp;section=3" title="Edit section: Usage in text">edit</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>It is often recommended that the percent sign only be used in tables and other places with space restrictions. In running text, it should be spelled out as <i>percent</i> or <i>per cent</i> (often in newspapers). For example, not <i>"Sales increased by 24% over 2006"</i>, but rather <i>"Sales increased by 24 percent over 2006"</i>.<sup class="reference" id="cite_ref-15"><a href="#cite_note-15">[15]</a></sup><sup class="reference" id="cite_ref-16"><a href="#cite_note-16">[16]</a></sup><sup class="reference" id="cite_ref-17"><a href="#cite_note-17">[17]</a></sup></p>
<h2><span class="mw-headline" id="Evolution">Evolution</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Percent_sign&amp;action=edit&amp;section=4" title="Edit section: Evolution">edit</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>Prior to 1425 there is no evidence of a special symbol being used for percentage. The <a href="/wiki/Italian_language" title="Italian language">Italian</a> term <i>per cento</i>, "for a hundred", was used as well as several different abbreviations (e.g. "per 100", "p 100", "p cento", etc.). Examples of this can be seen in the 1339 arithmetic text (author unknown) depicted below.<sup class="reference" id="cite_ref-18"><a href="#cite_note-18">[18]</a></sup> The letter p with its shaft crossed by a horizontal or diagonal strike conventionally stood for per, por, par, or pur in Mediaeval and Renaissance palaeography.<sup class="reference" id="cite_ref-19"><a href="#cite_note-19">[19]</a></sup></p>
<div class="center">
<div class="thumb tnone">
<div class="thumbinner" style="width:500px;"><a class="image" href="/wiki/File:Percent_sign_in_1339.png"><img alt="1339 arithmetic text" class="thumbimage" data-file-height="226" data-file-width="498" height="226" src="//upload.wikimedia.org/wikipedia/commons/9/9a/Percent_sign_in_1339.png" width="498"/></a>
<div class="thumbcaption">1339 arithmetic text in <i>Rara Arithmetica</i> pg. 437</div>
</div>
</div>
</div>
<p>At some point a scribe of some sort used the abbreviation "pc" with a tiny loop or circle (depicting the ending <i>-o</i> used in Italian numeration for <i>primo, secondo,</i> etc.) This appears in some additional pages of a 1425 text which were probably added around 1435.<sup class="reference" id="cite_ref-20"><a href="#cite_note-20">[20]</a></sup> This is shown below (source, <i>Rara Arithmetica</i> pg. 440).</p>
<div class="center">
<div class="thumb tnone">
<div class="thumbinner" style="width:373px;"><a class="image" href="/wiki/File:Percent_sign_in_1425.png"><img alt="" class="thumbimage" data-file-height="119" data-file-width="371" height="119" src="//upload.wikimedia.org/wikipedia/commons/c/ce/Percent_sign_in_1425.png" width="371"/></a>
<div class="thumbcaption">1425 arithmetic text in <i>Rara Arithmetica</i> pg. 440</div>
</div>
</div>
</div>
<p>The "pc" with a loop eventually evolved into a horizontal fraction sign by 1650 (see below for an example in a 1684 text<sup class="reference" id="cite_ref-21"><a href="#cite_note-21">[21]</a></sup>) and thereafter lost the "per".<sup class="reference" id="cite_ref-22"><a href="#cite_note-22">[22]</a></sup></p>
<div class="center">
<div class="thumb tnone">
<div class="thumbinner" style="width:452px;"><a class="image" href="/wiki/File:Percent_sign_in_1648.png"><img alt="1684 arithmetic text" class="thumbimage" data-file-height="157" data-file-width="450" height="157" src="//upload.wikimedia.org/wikipedia/commons/0/09/Percent_sign_in_1648.png" width="450"/></a>
<div class="thumbcaption">1684 arithmetic text in <i>Rara Arithmetica</i> pg. 441</div>
</div>
</div>
</div>
<p>In 1925 D.E. Smith wrote, "The <a class="mw-redirect" href="/wiki/Solidus_(punctuation)" title="Solidus (punctuation)">solidus</a> form (<a class="image" href="/wiki/File:Percentsign_in_Poor_Richard_font.png" title="Poor Richard font"><img alt="Poor Richard font" data-file-height="71" data-file-width="77" height="18" src="//upload.wikimedia.org/wikipedia/commons/thumb/3/3c/Percentsign_in_Poor_Richard_font.png/20px-Percentsign_in_Poor_Richard_font.png" srcset="//upload.wikimedia.org/wikipedia/commons/thumb/3/3c/Percentsign_in_Poor_Richard_font.png/30px-Percentsign_in_Poor_Richard_font.png 1.5x, //upload.wikimedia.org/wikipedia/commons/thumb/3/3c/Percentsign_in_Poor_Richard_font.png/40px-Percentsign_in_Poor_Richard_font.png 2x" width="20"/></a>) is modern."<sup class="reference" id="cite_ref-23"><a href="#cite_note-23">[23]</a></sup></p>

<h4><span class="mw-headline" id="ASCII">ASCII</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Percent_sign&amp;action=edit&amp;section=8" title="Edit section: ASCII">edit</a><span class="mw-editsection-bracket">]</span></span></h4>
<p>The <a class="mw-redirect" href="/wiki/ASCII_code" title="ASCII code">ASCII code</a> for the percent <a href="/wiki/Character_(computing)" title="Character (computing)">character</a> is 37, or 0x25 in <a href="/wiki/Hexadecimal" title="Hexadecimal">hexadecimal</a>.</p>
<h3><span class="mw-headline" id="In_computers">In computers</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Percent_sign&amp;action=edit&amp;section=9" title="Edit section: In computers">edit</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>Names for the percent sign include percent sign (in <a href="/wiki/ITU-T" title="ITU-T">ITU-T</a>), <a href="/wiki/Modulo_operation" title="Modulo operation">mod</a>, grapes (in <a class="mw-redirect" href="/wiki/Hacker_(programmer_subculture)" title="Hacker (programmer subculture)">hacker</a> <a href="/wiki/Jargon" title="Jargon">jargon</a>), and the humorous double-oh-seven (in <a href="/wiki/INTERCAL" title="INTERCAL">INTERCAL</a>).</p>
<p>In <a href="/wiki/Computing" title="Computing">computing</a>, the percent <a href="/wiki/Character_(computing)" title="Character (computing)">character</a> is also used for the <a href="/wiki/Modulo_operation" title="Modulo operation">modulo operation</a> in <a href="/wiki/Programming_language" title="Programming language">programming languages</a> that derive their syntax from the <a href="/wiki/C_(programming_language)" title="C (programming language)">C programming language</a>, which in turn acquired this usage from the earlier <a href="/wiki/B_(programming_language)" title="B (programming language)">B</a>.<sup class="reference" id="cite_ref-24"><a href="#cite_note-24">[24]</a></sup></p>
<p>In the textual representation of <a href="/wiki/Uniform_Resource_Identifier" title="Uniform Resource Identifier">URIs</a>, a % immediately followed by a 2-digit <a href="/wiki/Hexadecimal" title="Hexadecimal">hexadecimal</a> number denotes an octet specifying (part of) a character that might otherwise not be allowed in URIs (see <a href="/wiki/Percent-encoding" title="Percent-encoding">percent-encoding</a>).</p>
<p>In <a href="/wiki/SQL" title="SQL">SQL</a>, the percent sign is a <a href="/wiki/Wildcard_character" title="Wildcard character">wildcard character</a> in "LIKE" expressions, for example <code>SELECT * FROM table WHERE fullname LIKE 'Lisa %'</code> will fetch all records whose names start with "<tt>Lisa</tt> ".</p>
<p>In <a href="/wiki/TeX" title="TeX">TeX</a> (and therefore also in LaTeX) and <a href="/wiki/PostScript" title="PostScript">PostScript</a>, a % denotes a line <a class="mw-redirect" href="/wiki/Comment_(computing)" title="Comment (computing)">comment</a>.</p>
<p>In <a href="/wiki/BASIC" title="BASIC">BASIC</a>, a trailing % after a variable name marks it as an <a href="/wiki/Integer" title="Integer">integer</a>.</p>
<p>In <a href="/wiki/Perl" title="Perl">Perl</a> % is the <a href="/wiki/Sigil_(computer_programming)" title="Sigil (computer programming)">sigil</a> for <a href="/wiki/Hash_table" title="Hash table">hashes</a>.</p>
<p>In many programming languages' string formatting operations (performed by functions such as <a class="mw-redirect" href="/wiki/Printf" title="Printf">printf</a>), the percent sign denotes parts of the template string that will be replaced with arguments. (See <a href="/wiki/Printf_format_string" title="Printf format string">printf format string</a>.) In <a href="/wiki/Python_(programming_language)" title="Python (programming language)">Python</a> and <a href="/wiki/Ruby_(programming_language)" title="Ruby (programming language)">Ruby</a> the percent sign is also used as the string formatting operator.<sup class="reference" id="cite_ref-25"><a href="#cite_note-25">[25]</a></sup><sup class="reference" id="cite_ref-26"><a href="#cite_note-26">[26]</a></sup><sup class="reference" id="cite_ref-27"><a href="#cite_note-27">[27]</a></sup></p>
<p>In the <a class="mw-redirect" href="/wiki/Command_processor" title="Command processor">command processors</a> <a href="/wiki/COMMAND.COM" title="COMMAND.COM">COMMAND.COM</a> (DOS) and <a class="mw-redirect" href="/wiki/CMD.EXE" title="CMD.EXE">CMD.EXE</a> (OS/2 and Windows), %1, %2,... stand for the first, second,... parameters of a <a href="/wiki/Batch_file" title="Batch file">batch file</a>.  %0 stands for the specification of the batch file itself as typed on the command line. The % sign is also used similarly in the FOR command. %VAR1% represents the value of an <a href="/wiki/Environment_variable" title="Environment variable">environment variable</a> named VAR1. Thus:</p>
<pre>
set PATH=c:\;%PATH%
</pre>
<p>sets a new value for PATH, that being the old value preceded by "<tt>c:\;</tt>". Because these uses give the percent sign special meaning, the sequence %% (two percent signs) is used to represent a literal percent sign, so that:</p>
<pre>
set PATH=c:\;%%PATH%%
</pre>
<p>would set PATH to the literal value "<tt>c:\;%PATH%</tt>".</p>
<p>In the <a class="mw-redirect" href="/wiki/C_Shell" title="C Shell">C Shell</a>, % is part of the default <a class="mw-redirect" href="/wiki/Prompt_(computing)" title="Prompt (computing)">command prompt</a>.</p>
<h3><span class="mw-headline" id="In_linguistics">In linguistics</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Percent_sign&amp;action=edit&amp;section=10" title="Edit section: In linguistics">edit</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>In linguistics, the percent sign is prepended to an example <a href="/wiki/String_(computer_science)" title="String (computer science)">string</a> to show that it is judged <a href="/wiki/Well-formed_formula" title="Well-formed formula">well-formed</a> by some speakers and ill-formed by others. This may be due to differences in <a href="/wiki/Dialect" title="Dialect">dialect</a> or even individual <a href="/wiki/Idiolect" title="Idiolect">idiolects</a>. This is similar to the <a href="/wiki/Asterisk" title="Asterisk">asterisk</a> to mark ill-formed strings, the <a href="/wiki/Question_mark" title="Question mark">question mark</a> to mark strings where well-formedness is unclear, and the <a href="/wiki/Number_sign" title="Number sign">number sign</a> to mark strings that are syntactically well-formed but semantically nonsensical.<sup class="noprint Inline-Template Template-Fact" style="white-space:nowrap;">[<i><a href="/wiki/Wikipedia:Citation_needed" title="Wikipedia:Citation needed"><span title="This claim needs references to reliable sources. (April 2015)">citation needed</span></a></i>]</sup></p>
