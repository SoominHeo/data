<p>A <b>level editor</b> (also known as a <b>map</b>, <b>campaign</b> or <b>scenario editor</b>) is a <a class="mw-redirect" href="/wiki/Computer_software" title="Computer software">software</a> used to <a href="/wiki/Level_design" title="Level design">design</a> <a class="mw-redirect" href="/wiki/Level_(video_games)" title="Level (video games)">levels, maps, campaigns, etc.</a> and <a href="/wiki/Virtual_world" title="Virtual world">virtual worlds</a> for a <a href="/wiki/Video_game" title="Video game">video game</a>. An individual involved with the creation of game levels is a <a class="mw-redirect" href="/wiki/Level_designer" title="Level designer">level designer</a> or mapper.</p>
<p>In some cases, the creator of a video game releases an official level editor for the game, but other times the community of fans step in to fill the void. The level editor can be integrated into the game; for example, a track editor for a racing game. Other times (and most often), the editor is a separate application, as are most fan-released level editors.</p>
<p>One of the first 3D games which became popular partially due to level editors and fan-made content was <a href="/wiki/Doom_(1993_video_game)" title="Doom (1993 video game)"><i>Doom</i></a>. The creation of various third-party editors led to the birth of an online community trading fan-made maps.<sup class="reference" id="cite_ref-1"><a href="#cite_note-1">[1]</a></sup></p>
<p>A level editor is often limited to creating levels for only a certain <a href="/wiki/Game_engine_recreation" title="Game engine recreation">game engine</a>. Developing a level editor takes a lot of time and it is more time and cost efficient to release multiple games using the same engine instead of developing a new engine and level editor for each game. Level editors offer some limited scope of content creation, but in the case of gaming industry solutions the scope is very large allowing an entire game to be created without the need for much support from a programming team. To make larger changes to a game than simply adding new levels, a <a href="/wiki/Software_development_kit" title="Software development kit">software development kit</a> (SDK) is sometimes needed.</p>
<h2><span class="mw-headline" id="Construction_set">Construction set</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Level_editor&amp;action=edit&amp;section=1" title="Edit section: Construction set">edit</a><span class="mw-editsection-bracket">]</span></span></h2>
<div class="hatnote" role="note">See also: <a href="/wiki/Game_creation_system" title="Game creation system">Game creation system</a></div>
<p>In the early years of video-gaming, some games came with a utility called a "construction set". This was similar in many ways to a level-editor. Some games used them to create extra levels, whereas others (like the <i><a class="mw-redirect" href="/wiki/Shoot%27Em-Up_Construction_Kit" title="Shoot'Em-Up Construction Kit">Shoot'Em-Up Construction Kit</a></i>) used them as a means to create a game rather than be a game in itself.</p>
