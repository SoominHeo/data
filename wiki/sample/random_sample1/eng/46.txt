<p><b>NoteWorthy Composer</b> (NWC) is a proprietary <a href="/wiki/Scorewriter" title="Scorewriter">scorewriter</a> application made by NoteWorthy Software. It is a graphical score editor for <a href="/wiki/Microsoft_Windows" title="Microsoft Windows">Microsoft Windows</a> computers (from <a href="/wiki/Windows_95" title="Windows 95">Windows 95</a> to <a href="/wiki/Windows_10" title="Windows 10">Windows 10</a>), and also works on PCs under <a href="/wiki/Linux" title="Linux">Linux</a> with <a href="/wiki/Wine_(software)" title="Wine (software)">Wine</a>. Version 1 of NWC was released in October 1994, and Version 2 in September 2008. The current version is Version 2.75.<sup class="reference" id="cite_ref-history_1-0"><a href="#cite_note-history-1">[1]</a></sup></p>
<p>NWC is intended for the creation of <a href="/wiki/Sheet_music" title="Sheet music">sheet music</a>, but it can also import and export <a href="/wiki/MIDI" title="MIDI">MIDI</a> and Karaoke files and can export graphical <a href="/wiki/Windows_Metafile" title="Windows Metafile">WMFs</a>. The <a href="/wiki/User_interface" title="User interface">user interface</a> works either from the keyboard or the mouse. Visual results are immediate, and audible results can be heard at any time. Notes can also be entered by playing on a MIDI device, when configured. In version 2, the notes can be heard as they are entered.<sup class="reference" id="cite_ref-history_1-1"><a href="#cite_note-history-1">[1]</a></sup></p>
<p>The NWC <a href="/wiki/File_format" title="File format">file format</a> is undocumented and facilities to convert it into documented formats are limited. Version 2 of NWC introduced a textual version of the files, NWCTXT. These can be used for conversion to many other formats, including <a href="/wiki/LilyPond" title="LilyPond">LilyPond</a>.<sup class="reference" id="cite_ref-2"><a href="#cite_note-2">[2]</a></sup></p>

<p>A feature of the user interface is that notation is displayed during editing. Each staff proceeds linearly from left to right, without being wrapped to the screen. Staff systems are visually broken to fit margins during page layout, allowing many possibilities at "print time", so solo instrument or full conductor can be produced with the same file. Many users<sup class="noprint Inline-Template" style="white-space:nowrap;">[<i><a href="/wiki/Wikipedia:Manual_of_Style/Words_to_watch#Unsupported_attributions" title="Wikipedia:Manual of Style/Words to watch"><span title="The material near this tag possibly uses too-vague attribution or weasel words. (January 2015)">who?</span></a></i>]</sup> prefer this editor layout over the so-called <a href="/wiki/WYSIWYG" title="WYSIWYG">WYSIWYG</a> editors because slowdowns in note entry, as the composition becomes larger, are much less dramatic than in WYSIWYG editors. Print preview is available for adjustments to page layout since version 2.51.<sup class="reference" id="cite_ref-history_1-2"><a href="#cite_note-history-1">[1]</a></sup></p>
<p><br/>
The program lacks the more advanced engraving, graphic sophistication, playback and publishing capabilities of more expensive scriptwriting software such as <a class="mw-redirect" href="/wiki/Sibelius_notation_program" title="Sibelius notation program">Sibelius</a> or <a class="mw-redirect" href="/wiki/Finale_(computer_program)" title="Finale (computer program)">Finale</a>. It does, however, allow the rendering of custom <a href="/wiki/Key_signature" title="Key signature">key signatures</a> which do not follow the usual <a href="/wiki/Circle_of_fifths" title="Circle of fifths">circle of fifths</a> order of <a href="/wiki/Sharp_(music)" title="Sharp (music)">sharps</a> and <a href="/wiki/Flat_(music)" title="Flat (music)">flats</a>. A free viewer is available. The otherwise fully functional <a href="/wiki/Demo_(computer_programming)" title="Demo (computer programming)">demo version</a> imposes a limit of 10 saves per file name, adds a small footer to each printed page, and prints a registration form with each printed score. Besides the demo program, a downloadable plug-in for <a href="/wiki/Winamp" title="Winamp">Winamp</a> allows Winamp to play files from NWC.<sup class="reference" id="cite_ref-3"><a href="#cite_note-3">[3]</a></sup></p>
<p>Because of the availability of a free viewer, Noteworthy has been adopted as the standard score distribution format by the large <a href="/wiki/Hymn" title="Hymn">hymn</a> database, the Cyber Hymnal.<sup class="reference" id="cite_ref-4"><a href="#cite_note-4">[4]</a></sup> An "unofficial" catalog of compositions and helpful files contributed by users is available from the NoteWorthy Scriptorium.<sup class="reference" id="cite_ref-5"><a href="#cite_note-5">[5]</a></sup></p>
