<p>In <a href="/wiki/Computing" title="Computing">computing</a>, a <b>locale</b> is a set of <a href="/wiki/Parameter" title="Parameter">parameters</a> that defines the user's language, region and any special variant preferences that the user wants to see in their <a href="/wiki/User_interface" title="User interface">user interface</a>. Usually a locale identifier consists of at least a language identifier and a region identifier.</p>
<p>On <a href="/wiki/POSIX" title="POSIX">POSIX</a> platforms such as <a href="/wiki/Unix" title="Unix">Unix</a>, <a href="/wiki/Linux" title="Linux">Linux</a> and others, locale identifiers are defined by <a href="/wiki/ISO_15897" title="ISO 15897">ISO 15897</a>, which is similar to the <a class="mw-redirect" href="/wiki/BCP_47" title="BCP 47">BCP 47</a> definition of language tags, but the locale variant modifier is defined differently, and the <a class="mw-redirect" href="/wiki/Character_set" title="Character set">character set</a> is included as a part of the identifier. It is defined in this format: <tt>[language[_territory][.codeset][@modifier]]</tt>. (For example, <a href="/wiki/Australian_English" title="Australian English">Australian English</a> using the <a href="/wiki/UTF-8" title="UTF-8">UTF-8</a> encoding is <tt>en_AU.UTF-8</tt>.)</p>

<h2><span class="mw-headline" id="General_locale_settings">General locale settings</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Locale_(computer_software)&amp;action=edit&amp;section=1" title="Edit section: General locale settings">edit</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>These settings usually include the following display (output) format settings:</p>
<ul>
<li>Number format setting</li>
<li>Character classification, case conversion settings</li>
<li>Date-time format setting</li>
<li>String <a href="/wiki/Collation" title="Collation">collation</a> setting</li>
<li>Currency format setting</li>
<li>Paper size setting</li>
<li>other minor settings ...</li>
</ul>
<p>The locale settings are about formatting output given a locale. So, the timezone information and daylight saving time are not usually part of the locale settings. Less usual is the input format setting, which is mostly defined on a per application basis.</p>
<p>Furthermore, the general settings usually include the <a href="/wiki/Keyboard_layout" title="Keyboard layout">keyboard layout</a> setting.<sup class="noprint Inline-Template Template-Fact" style="white-space:nowrap;">[<i><a href="/wiki/Wikipedia:Citation_needed" title="Wikipedia:Citation needed"><span title="This claim needs references to reliable sources. (September 2011)">citation needed</span></a></i>]</sup></p>
<h2><span class="mw-headline" id="Programming_and_markup_language_support">Programming and markup language support</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Locale_(computer_software)&amp;action=edit&amp;section=2" title="Edit section: Programming and markup language support">edit</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>In these environments,</p>
<div class="div-col columns column-count column-count-3" style="-moz-column-count: 3; -webkit-column-count: 3; column-count: 3;">
<ul>
<li><a href="/wiki/C_(programming_language)" title="C (programming language)">C</a></li>
<li><a href="/wiki/C%2B%2B" title="C++">C++</a></li>
<li><a href="/wiki/Eiffel_(programming_language)" title="Eiffel (programming language)">Eiffel</a></li>
<li><a href="/wiki/Java_(programming_language)" title="Java (programming language)">Java</a></li>
<li><a href="/wiki/.NET_Framework" title=".NET Framework">.NET Framework</a></li>
<li><a class="mw-redirect" href="/wiki/REBOL" title="REBOL">REBOL</a></li>
<li><a href="/wiki/Ruby_(programming_language)" title="Ruby (programming language)">Ruby</a></li>
<li><a href="/wiki/Perl" title="Perl">Perl</a></li>
<li><a href="/wiki/PHP" title="PHP">PHP</a></li>
<li><a href="/wiki/Python_(programming_language)" title="Python (programming language)">Python</a></li>
<li><a href="/wiki/XML" title="XML">XML</a></li>
<li><a href="/wiki/JavaServer_Pages" title="JavaServer Pages">JSP</a></li>
</ul>
</div>
<p>and other (nowadays) <a href="/wiki/Unicode" title="Unicode">Unicode</a>-based environments, they are defined in a format similar to <a class="mw-redirect" href="/wiki/BCP_47" title="BCP 47">BCP 47</a>. They are usually defined with just <a href="/wiki/ISO_639" title="ISO 639">ISO 639</a> (language) and <a href="/wiki/ISO_3166-1_alpha-2" title="ISO 3166-1 alpha-2">ISO 3166-1 alpha-2</a> (2-letter country) codes.</p>
<h2><span class="mw-headline" id="POSIX_platforms">POSIX platforms</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Locale_(computer_software)&amp;action=edit&amp;section=3" title="Edit section: POSIX platforms">edit</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>On <a href="/wiki/POSIX" title="POSIX">POSIX</a> platforms, locale identifiers are defined by <a href="/wiki/ISO_15897" title="ISO 15897">ISO 15897</a>, which is similar to the <a class="mw-redirect" href="/wiki/BCP_47" title="BCP 47">BCP 47</a> definition of language tags, but the locale variant modifier is defined differently, and the <a class="mw-redirect" href="/wiki/Character_set" title="Character set">character set</a> is included as a part of the identifier.<sup class="noprint Inline-Template Template-Fact" style="white-space:nowrap;">[<i><a href="/wiki/Wikipedia:Citation_needed" title="Wikipedia:Citation needed"><span title="This claim needs references to reliable sources. (March 2014)">citation needed</span></a></i>]</sup></p>
<p>In the next example there is an output of command <code>locale</code> for <a href="/wiki/Czech_language" title="Czech language">Czech language</a> (cs), <a href="/wiki/Czech_Republic" title="Czech Republic">Czech Republic</a> (CZ) with explicit <a href="/wiki/UTF-8" title="UTF-8">UTF-8</a> encoding:</p>
<pre>
$ <b>locale</b>
LANG=cs_CZ.UTF-8
LC_CTYPE="cs_CZ.UTF-8"
LC_NUMERIC="cs_CZ.UTF-8"
LC_TIME="cs_CZ.UTF-8"
LC_COLLATE="cs_CZ.UTF-8"
LC_MONETARY="cs_CZ.UTF-8"
LC_MESSAGES="cs_CZ.UTF-8"
LC_PAPER="cs_CZ.UTF-8"
LC_NAME="cs_CZ.UTF-8"
LC_ADDRESS="cs_CZ.UTF-8"
LC_TELEPHONE="cs_CZ.UTF-8"
LC_MEASUREMENT="cs_CZ.UTF-8"
LC_IDENTIFICATION="cs_CZ.UTF-8"
LC_ALL=
</pre>
<h2><span class="mw-headline" id="Specifics_for_Microsoft_platforms">Specifics for Microsoft platforms</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Locale_(computer_software)&amp;action=edit&amp;section=4" title="Edit section: Specifics for Microsoft platforms">edit</a><span class="mw-editsection-bracket">]</span></span></h2>

<tr>
<td class="mbox-image">
<div style="width:52px"><a class="image" href="/wiki/File:Ambox_current_red.svg"><img alt="Ambox current red.svg" data-file-height="290" data-file-width="360" height="34" src="//upload.wikimedia.org/wikipedia/commons/thumb/9/98/Ambox_current_red.svg/42px-Ambox_current_red.svg.png" srcset="//upload.wikimedia.org/wikipedia/commons/thumb/9/98/Ambox_current_red.svg/63px-Ambox_current_red.svg.png 1.5x, //upload.wikimedia.org/wikipedia/commons/thumb/9/98/Ambox_current_red.svg/84px-Ambox_current_red.svg.png 2x" width="42"/></a></div>

<td class="mbox-text"><span class="mbox-text-span">This section needs to be <b>updated</b>. <span class="hide-when-compact">Please update this article to reflect recent events or newly available information.</span> <small><i>(June 2016)</i></small></span></td>

<p>Windows uses specific <a class="external text" href="https://msdn.microsoft.com/en-us/library/39cwe7zf%28vs.71%29.aspx" rel="nofollow">language</a> and <a class="external text" href="https://msdn.microsoft.com/en-us/library/cdax410z%28VS.71%29.aspx" rel="nofollow">territory</a> strings. The locale identifier (LCID) for <a href="/wiki/Managed_code" title="Managed code">unmanaged code</a> on <a href="/wiki/Microsoft_Windows" title="Microsoft Windows">Microsoft Windows</a> is a number such as 1033 for English (United States) or 1041 for Japanese (Japan). These numbers consist of a language code (lower 10 bits) and culture code (upper bits) and are therefore often written in <a href="/wiki/Hexadecimal" title="Hexadecimal">hexadecimal</a> notation, such as 0x0409 or 0x0411. The list of those codesets are described in <a href="/wiki/Character_encoding" title="Character encoding">character encoding</a>. <a href="/wiki/Microsoft" title="Microsoft">Microsoft</a> is starting to introduce managed code <a href="/wiki/Application_programming_interface" title="Application programming interface">application programming interfaces</a> (APIs) for .NET that use this format. One of the first to be generally released is a function to mitigate issues with <a href="/wiki/Internationalized_domain_name" title="Internationalized domain name">internationalized domain names</a>,<sup class="reference" id="cite_ref-1"><a href="#cite_note-1">[1]</a></sup> but more are in <a href="/wiki/Windows_Vista" title="Windows Vista">Windows Vista</a> Beta 1.</p>
<p>Starting with Windows Vista, new functions<sup class="reference" id="cite_ref-microsoft_2-0"><a href="#cite_note-microsoft-2">[2]</a></sup> that use <a class="mw-redirect" href="/wiki/BCP_47" title="BCP 47">BCP 47</a> locale names have been introduced to replace nearly all LCID-based APIs.</p>
