<p><b>Lunascape</b> is a <a href="/wiki/Web_browser" title="Web browser">web browser</a> developed by Lunascape Corporation in <a class="mw-redirect" href="/wiki/Tokyo,_Japan" title="Tokyo, Japan">Tokyo, Japan</a>. It is unusual in that it contains three rendering engines: <a class="mw-redirect" href="/wiki/Gecko_(layout_engine)" title="Gecko (layout engine)">Gecko</a> (used in <a class="mw-redirect" href="/wiki/Mozilla_Firefox" title="Mozilla Firefox">Mozilla Firefox</a>), <a href="/wiki/WebKit" title="WebKit">WebKit</a> (used in <a href="/wiki/Apple_Inc." title="Apple Inc.">Apple</a>'s <a href="/wiki/Safari_(web_browser)" title="Safari (web browser)">Safari</a>), and <a href="/wiki/Trident_(layout_engine)" title="Trident (layout engine)">Trident</a> (used in <a href="/wiki/Microsoft" title="Microsoft">Microsoft</a> <a href="/wiki/Internet_Explorer" title="Internet Explorer">Internet Explorer</a>).<sup class="reference" id="cite_ref-3"><a href="#cite_note-3">[3]</a></sup><sup class="reference" id="cite_ref-4"><a href="#cite_note-4">[4]</a></sup> This feature is common only to the <a href="/wiki/Avant_Browser" title="Avant Browser">Avant</a> web browser. The user can switch between layout engines seamlessly.</p>
<p>Lunascape is available for Windows and Android platforms, as well as for iPad and iPhone.</p>
<h2><span class="mw-headline" id="History">History</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Lunascape&amp;action=edit&amp;section=1" title="Edit section: History">edit</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>Lunascape was released in October 2001 while the founders were in college. As the browser became popular, <a class="new" href="/w/index.php?title=Hidekazu_Kondo&amp;action=edit&amp;redlink=1" title="Hidekazu Kondo (page does not exist)">Hidekazu Kondo</a> established Lunascape Corporation in August 2004, while pursuing a PhD and became the <a class="mw-redirect" href="/wiki/CEO" title="CEO">CEO</a>. Lunascape was selected as "Exploratory Software Project" commissioned by Japanese government.<sup class="reference" id="cite_ref-5"><a href="#cite_note-5">[5]</a></sup></p>
<p>The company branched out to the <a href="/wiki/United_States" title="United States">United States</a> and as of June 2008 was based in <a href="/wiki/Sunnyvale,_California" title="Sunnyvale, California">Sunnyvale, California</a>.<sup class="reference" id="cite_ref-6"><a href="#cite_note-6">[6]</a></sup><sup class="reference" id="cite_ref-7"><a href="#cite_note-7">[7]</a></sup><sup class="reference" id="cite_ref-8"><a href="#cite_note-8">[8]</a></sup></p>
<p>Lunascape internationally introduced its browser in December 2008.</p>
