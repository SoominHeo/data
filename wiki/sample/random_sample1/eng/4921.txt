<h1 class="firstHeading" id="firstHeading" lang="en">Lissajous curve</h1>

<p>In <a href="/wiki/Mathematics" title="Mathematics">mathematics</a>, a <b>Lissajous curve</b> <span class="nowrap"><span class="IPA nopopups"><a href="/wiki/Help:IPA_for_English" title="Help:IPA for English">/<span style="border-bottom:1px dotted"><span title="/ˈ/ primary stress follows">ˈ</span><span title="'l' in 'lie'">l</span><span title="/ɪ/ short 'i' in 'bid'">ɪ</span><span title="'s' in 'sigh'">s</span><span title="/ə/ 'a' in 'about'">ə</span><span title="/ʒ/ 's' in 'pleasure'">ʒ</span><span title="/uː/ long 'oo' in 'food'">uː</span></span>/</a></span></span>, also known as <b>Lissajous figure</b> or <b>Bowditch curve</b> <span class="nowrap"><span class="IPA nopopups"><a href="/wiki/Help:IPA_for_English" title="Help:IPA for English">/<span style="border-bottom:1px dotted"><span title="/ˈ/ primary stress follows">ˈ</span><span title="'b' in 'buy'">b</span><span title="/aʊ/ 'ou' in 'pout'">aʊ</span><span title="'d' in 'dye'">d</span><span title="/ɪ/ short 'i' in 'bid'">ɪ</span><span title="/tʃ/ 'ch' in 'china'">tʃ</span></span>/</a></span></span>, is the graph of a system of <a href="/wiki/Parametric_equation" title="Parametric equation">parametric equations</a></p>

<p>which describe <a href="/wiki/Complex_harmonic_motion" title="Complex harmonic motion">complex harmonic motion</a>. This family of <a href="/wiki/Curve" title="Curve">curves</a> was investigated by <a href="/wiki/Nathaniel_Bowditch" title="Nathaniel Bowditch">Nathaniel Bowditch</a> in 1815, and later in more detail by <a href="/wiki/Jules_Antoine_Lissajous" title="Jules Antoine Lissajous">Jules Antoine Lissajous</a> in 1857.</p>
<p>The appearance of the figure is highly sensitive to the ratio <span class="texhtml"><span class="sfrac nowrap" style="display:inline-block; vertical-align:-0.5em; font-size:85%; text-align:center;"><span style="display:block; line-height:1em; margin:0 0.1em;"><i>a</i></span><span class="visualhide">/</span><span style="display:block; line-height:1em; margin:0 0.1em; border-top:1px solid;"><i>b</i></span></span></span>. For a ratio of 1, the figure is an <a href="/wiki/Ellipse" title="Ellipse">ellipse</a>, with special cases including <a class="mw-redirect" href="/wiki/Circles" title="Circles">circles</a> (<span class="texhtml"><i>A</i> = <i>B</i></span>, <span class="texhtml"><i>δ</i> = <span class="sfrac nowrap" style="display:inline-block; vertical-align:-0.5em; font-size:85%; text-align:center;"><span style="display:block; line-height:1em; margin:0 0.1em;"><a href="/wiki/Pi" title="Pi">π</a></span><span class="visualhide">/</span><span style="display:block; line-height:1em; margin:0 0.1em; border-top:1px solid;">2</span></span></span> <a href="/wiki/Radian" title="Radian">radians</a>) and <a class="mw-redirect" href="/wiki/Line_(mathematics)" title="Line (mathematics)">lines</a> (<span class="texhtml"><i>δ</i> = 0</span>). Another simple Lissajous figure is the <a href="/wiki/Parabola" title="Parabola">parabola</a> (<span class="texhtml"><span class="sfrac nowrap" style="display:inline-block; vertical-align:-0.5em; font-size:85%; text-align:center;"><span style="display:block; line-height:1em; margin:0 0.1em;"><i>a</i></span><span class="visualhide">/</span><span style="display:block; line-height:1em; margin:0 0.1em; border-top:1px solid;"><i>b</i></span></span> = 2</span>, <span class="texhtml"><i>δ</i> = <span class="sfrac nowrap" style="display:inline-block; vertical-align:-0.5em; font-size:85%; text-align:center;"><span style="display:block; line-height:1em; margin:0 0.1em;">π</span><span class="visualhide">/</span><span style="display:block; line-height:1em; margin:0 0.1em; border-top:1px solid;">4</span></span></span>). Other ratios produce more complicated curves, which are closed only if <span class="texhtml"><span class="sfrac nowrap" style="display:inline-block; vertical-align:-0.5em; font-size:85%; text-align:center;"><span style="display:block; line-height:1em; margin:0 0.1em;"><i>a</i></span><span class="visualhide">/</span><span style="display:block; line-height:1em; margin:0 0.1em; border-top:1px solid;"><i>b</i></span></span></span> is <a href="/wiki/Rational_number" title="Rational number">rational</a>. The visual form of these curves is often suggestive of a three-dimensional <a class="mw-redirect" href="/wiki/Knot_(mathematical)" title="Knot (mathematical)">knot</a>, and indeed many kinds of knots, including those known as <a href="/wiki/Lissajous_knot" title="Lissajous knot">Lissajous knots</a>, project to the plane as Lissajous figures.</p>
<p>Visually, the ratio <span class="texhtml"><span class="sfrac nowrap" style="display:inline-block; vertical-align:-0.5em; font-size:85%; text-align:center;"><span style="display:block; line-height:1em; margin:0 0.1em;"><i>a</i></span><span class="visualhide">/</span><span style="display:block; line-height:1em; margin:0 0.1em; border-top:1px solid;"><i>b</i></span></span></span> determines the number of "lobes" of the figure. For example, a ratio of <span class="sfrac nowrap" style="display:inline-block; vertical-align:-0.5em; font-size:85%; text-align:center;"><span style="display:block; line-height:1em; margin:0 0.1em;">3</span><span class="visualhide">/</span><span style="display:block; line-height:1em; margin:0 0.1em; border-top:1px solid;">1</span></span> or <span class="sfrac nowrap" style="display:inline-block; vertical-align:-0.5em; font-size:85%; text-align:center;"><span style="display:block; line-height:1em; margin:0 0.1em;">1</span><span class="visualhide">/</span><span style="display:block; line-height:1em; margin:0 0.1em; border-top:1px solid;">3</span></span> produces a figure with three major lobes (see image). Similarly, a ratio of <span class="sfrac nowrap" style="display:inline-block; vertical-align:-0.5em; font-size:85%; text-align:center;"><span style="display:block; line-height:1em; margin:0 0.1em;">5</span><span class="visualhide">/</span><span style="display:block; line-height:1em; margin:0 0.1em; border-top:1px solid;">4</span></span> produces a figure with five horizontal lobes and four vertical lobes. Rational ratios produce closed (connected) or "still" figures, while irrational ratios produce figures that appear to rotate. The ratio <span class="texhtml"><span class="sfrac nowrap" style="display:inline-block; vertical-align:-0.5em; font-size:85%; text-align:center;"><span style="display:block; line-height:1em; margin:0 0.1em;"><i>A</i></span><span class="visualhide">/</span><span style="display:block; line-height:1em; margin:0 0.1em; border-top:1px solid;"><i>B</i></span></span></span> determines the relative width-to-height ratio of the curve. For example, a ratio of <span class="sfrac nowrap" style="display:inline-block; vertical-align:-0.5em; font-size:85%; text-align:center;"><span style="display:block; line-height:1em; margin:0 0.1em;">2</span><span class="visualhide">/</span><span style="display:block; line-height:1em; margin:0 0.1em; border-top:1px solid;">1</span></span> produces a figure that is twice as wide as it is high. Finally, the value of <span class="texhtml"><i>δ</i></span> determines the apparent "rotation" angle of the figure, viewed as if it were actually a three-dimensional curve. For example, <span class="texhtml"><i>δ</i> = 0</span> produces <span class="texhtml"><i>x</i></span> and <span class="texhtml"><i>y</i></span> components that are exactly in phase, so the resulting figure appears as an apparent three-dimensional figure viewed from straight on (0°). In contrast, any non-zero <span class="texhtml"><i>δ</i></span> produces a figure that appears to be rotated, either as a left–right or an up–down rotation (depending on the ratio <span class="texhtml"><span class="sfrac nowrap" style="display:inline-block; vertical-align:-0.5em; font-size:85%; text-align:center;"><span style="display:block; line-height:1em; margin:0 0.1em;"><i>a</i></span><span class="visualhide">/</span><span style="display:block; line-height:1em; margin:0 0.1em; border-top:1px solid;"><i>b</i></span></span></span>).</p>

<p>Lissajous figures where <span class="texhtml"><i>a</i> = 1</span>, <span class="texhtml"><i>b</i> = <i>N</i></span> (<span class="texhtml"><i>N</i></span> is a <a href="/wiki/Natural_number" title="Natural number">natural number</a>) and</p>
<dl>
<dd><span><span class="mwe-math-mathml-inline mwe-math-mathml-a11y mw-math-element" style="display: none;"><math xmlns="http://www.w3.org/1998/Math/MathML">

</semantics>
</math></span><img alt="{\displaystyle \delta ={\frac {N-1}{N}}{\frac {\pi }{2}}}" aria-hidden="true" class="mwe-math-fallback-image-inline mw-math-element" src="https://wikimedia.org/api/rest_v1/media/math/render/svg/c2f054d5e435cc446197b87085af4aa362d820f4" style="vertical-align: -2.005ex; width:13.281ex; height:5.509ex;"/></span></dd>
</dl>
<p>are <a href="/wiki/Chebyshev_polynomials" title="Chebyshev polynomials">Chebyshev polynomials</a> of the first kind of degree <span class="texhtml"><i>N</i></span>. This property is exploited to produce a set of points, called <a href="/wiki/Padua_points" title="Padua points">Padua points</a>, at which a function may be sampled in order to compute either a bivariate interpolation or quadrature of the function over the domain <span class="texhtml">[−1,1] × [−1,1]</span>.</p>

<h2><span class="mw-headline" id="Generation">Generation</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Lissajous_curve&amp;action=edit&amp;section=2" title="Edit section: Generation">edit</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>Prior to modern electronic equipment, Lissajous curves could be generated mechanically by means of a <a href="/wiki/Harmonograph" title="Harmonograph">harmonograph</a>.</p>
<h3><span class="mw-headline" id="Practical_application">Practical application</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Lissajous_curve&amp;action=edit&amp;section=3" title="Edit section: Practical application">edit</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>Lissajous curves can also be generated using an <a href="/wiki/Oscilloscope" title="Oscilloscope">oscilloscope</a> (as illustrated). An <a href="/wiki/Analog_signature_analysis" title="Analog signature analysis">octopus circuit</a> can be used to demonstrate the <a href="/wiki/Waveform" title="Waveform">waveform</a> images on an oscilloscope. Two phase-shifted sinusoid inputs are applied to the oscilloscope in X-Y mode and the phase relationship between the signals is presented as a Lissajous figure.</p>
<p>In the professional audio world, this method is used for realtime analysis of the phase relationship between the left and right channels of a stereo audio signal. On larger, more sophisticated audio mixing consoles an oscilloscope may be built-in for this purpose.</p>
<p>On an oscilloscope, we suppose <span class="texhtml"><i>x</i></span> is CH1 and <span class="texhtml"><i>y</i></span> is CH2, <span class="texhtml"><i>A</i></span> is the amplitude of CH1 and <span class="texhtml"><i>B</i></span> is the amplitude of CH2, <span class="texhtml"><i>a</i></span> is the frequency of CH1 and <span class="texhtml"><i>b</i></span> is the frequency of CH2, so <span class="texhtml"><span class="sfrac nowrap" style="display:inline-block; vertical-align:-0.5em; font-size:85%; text-align:center;"><span style="display:block; line-height:1em; margin:0 0.1em;"><i>a</i></span><span class="visualhide">/</span><span style="display:block; line-height:1em; margin:0 0.1em; border-top:1px solid;"><i>b</i></span></span></span> is the ratio of frequencies of the two channels, and <span class="texhtml"><i>δ</i></span> is the phase shift of CH1.</p>
<p>A purely mechanical application of a Lissajous curve with <span class="texhtml"><i>a</i> = 1</span>, <span class="texhtml"><i>b</i> = 2</span> is in the driving mechanism of the <a href="/wiki/Mars_Light" title="Mars Light">Mars Light</a> type of oscillating beam lamps popular with railroads in the mid-1900s. The beam in some versions traces out a lopsided figure-8 pattern on its side.</p>
<h2><span class="mw-headline" id="Application_for_the_case_of_a_.3D_b">Application for the case of <span class="texhtml"><i>a</i> = <i>b</i></span></span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Lissajous_curve&amp;action=edit&amp;section=4" title="Edit section: Application for the case of a = b">edit</a><span class="mw-editsection-bracket">]</span></span></h2>

<p>When the input to an <a class="mw-redirect" href="/wiki/LTI_system" title="LTI system">LTI system</a> is sinusoidal, the output is sinusoidal with the same frequency, but it may have a different amplitude and some <a class="mw-redirect" href="/wiki/Phase_shift" title="Phase shift">phase shift</a>. Using an <a href="/wiki/Oscilloscope" title="Oscilloscope">oscilloscope</a> that can plot one signal against another (as opposed to one signal against time) to plot the output of an LTI system against the input to the LTI system produces an ellipse that is a Lissajous figure for the special case of <span class="texhtml"><i>a</i> = <i>b</i></span>. The <a href="/wiki/Aspect_ratio" title="Aspect ratio">aspect ratio</a> of the resulting ellipse is a function of the phase shift between the input and output, with an aspect ratio of 1 (perfect circle) corresponding to a phase shift of ±90° and an aspect ratio of ∞ (a line) corresponding to a phase shift of 0° or 180°.<sup class="reference" id="cite_ref-Al-Khazali_1-0"><a href="#cite_note-Al-Khazali-1">[1]</a></sup></p>
<p>The figure below summarizes how the Lissajous figure changes over different phase shifts. The phase shifts are all negative so that <a href="/wiki/Propagation_delay" title="Propagation delay">delay</a> <a href="/wiki/Semantics" title="Semantics">semantics</a> can be used with a <a href="/wiki/Causal_system" title="Causal system">causal</a> LTI system (note that −270° is equivalent to +90°). The arrows show the direction of rotation of the Lissajous figure.<sup class="reference" id="cite_ref-Al-Khazali_1-1"><a href="#cite_note-Al-Khazali-1">[1]</a></sup></p>
