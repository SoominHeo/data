<p><i><b>Real Bout Fatal Fury Special</b></i> <span style="font-weight: normal">(<span class="t_nihongo_kanji" lang="ja" xml:lang="ja">リアルバウト餓狼伝説SPECIAL</span><span class="t_nihongo_comma" style="display:none">,</span> <i>Rearu Bauto Garō Densetsu Special</i><sup class="t_nihongo_help noprint"><a href="/wiki/Help:Installing_Japanese_character_sets" title="Help:Installing Japanese character sets"><span class="t_nihongo_icon" style="color: #00e; font: bold 80% sans-serif; text-decoration: none; padding: 0 .1em;">?</span></a></sup>, "Real Bout Legend of the Hungry Wolf Special")</span>, is a <a href="/wiki/1997_in_video_gaming" title="1997 in video gaming">1997</a> <a href="/wiki/Fighting_game" title="Fighting game">fighting game</a> released by <a href="/wiki/SNK" title="SNK">SNK</a> for the <a class="mw-redirect" href="/wiki/Neo_Geo_(console)" title="Neo Geo (console)">Neo Geo</a> platform. It is the sixth installment in the <i><a href="/wiki/Fatal_Fury_(series)" title="Fatal Fury (series)">Fatal Fury</a></i> series and the second game in the <i>Real Bout</i> sub-series, following the original <i><a href="/wiki/Real_Bout_Fatal_Fury" title="Real Bout Fatal Fury">Real Bout Fatal Fury</a></i>. This game features all new graphics and returns to the two-level plane system from <i><a href="/wiki/Fatal_Fury_2" title="Fatal Fury 2">Fatal Fury 2</a></i>.</p>

<h2><span class="mw-headline" id="Characters">Characters</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Real_Bout_Fatal_Fury_Special&amp;action=edit&amp;section=2" title="Edit section: Characters">edit</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>The game retains the cast of the original <i>Real Bout</i>, with the addition of <a href="/wiki/List_of_Fatal_Fury_characters#Tung_Fu_Rue" title="List of Fatal Fury characters">Tung Fu Rue</a>, <a href="/wiki/List_of_Fatal_Fury_characters#Cheng_Sinzan" title="List of Fatal Fury characters">Cheng Sinzan</a>, <a href="/wiki/List_of_Fatal_Fury_characters#Laurence_Blood" title="List of Fatal Fury characters">Laurence Blood</a>, and <a href="/wiki/List_of_Fatal_Fury_characters#Wolfgang_Krauser" title="List of Fatal Fury characters">Wolfgang Krauser</a> from <i><a href="/wiki/Fatal_Fury_Special" title="Fatal Fury Special">Fatal Fury Special</a></i>, with Krauser serving as the new final boss. <a href="/wiki/Geese_Howard" title="Geese Howard">Geese Howard</a>, who was killed off in the end of the original <i>Real Bout</i>, appears in this game as a hidden final boss in a special "Nightmare Match" and as an unlockable playable character in the home versions. The game also features hidden "extra" versions of <a href="/wiki/List_of_Fatal_Fury_characters#Andy_Bogard" title="List of Fatal Fury characters">Andy Bogard</a>, <a href="/wiki/Billy_Kane" title="Billy Kane">Billy Kane</a>, <a href="/wiki/Blue_Mary" title="Blue Mary">Blue Mary</a>, and Tung Fu Rue for a total of 23 characters (24 if counting Geese).</p>
<h2><span class="mw-headline" id="Home_versions">Home versions</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Real_Bout_Fatal_Fury_Special&amp;action=edit&amp;section=3" title="Edit section: Home versions">edit</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>The game was ported to the <a class="mw-redirect" href="/wiki/Neo-Geo_CD" title="Neo-Geo CD">Neo-Geo CD</a> with several additions such as a Versus mode and a music video starring Blue Mary that is shown to the player after completing the Arcade mode following the credits. This port also saw a release for the <a href="/wiki/Sega_Saturn" title="Sega Saturn">Sega Saturn</a>, using the 1MB RAM cartridge expansion of the system in order to retain sprite animations.</p>
<p>A port of <i>Real Bout Special</i> titled <i><b>Real Bout Fatal Fury Special: Dominated Mind</b></i> was released for the <a href="/wiki/PlayStation_(console)" title="PlayStation (console)">PlayStation</a> in 1998 only in Japan, which adds a new protagonist named Alfred as a playable character, and includes an all new boss character named White (modeled after the character <a href="/wiki/Alex_(A_Clockwork_Orange)" title="Alex (A Clockwork Orange)">Alexander "Alex" DeLarge</a> from <i><a href="/wiki/A_Clockwork_Orange_(film)" title="A Clockwork Orange (film)">A Clockwork Orange</a></i>). This port adds animated videos for game's intro and character's endings. These cut scenes were produced by <a href="/wiki/Sunrise_(company)" title="Sunrise (company)">Sunrise</a>. The story of this version centers around White's brainwashing of Billy Kane and attempted takeover of Southtown's underworld in the power vacuum left by <a href="/wiki/Geese_Howard" title="Geese Howard">Geese Howard</a>'s death in the original <i>Real Bout</i>. In this version, Geese sports a <a class="mw-redirect" href="/wiki/Halo_(religious_symbol)" title="Halo (religious symbol)">halo</a> over his head, a reference to his passing in the original <i>Real Bout</i>. <i>Dominated Mind</i> also features new moves, hidden unlockable super moves, super cancelling (known in the game as "Final Impacts"), and removed the two line battle system from the game. The extra versions of Andy, Billy, Mary, and Tung are removed.</p>
<p>The Game Boy version, titled <i><b>Nettou Real Bout Garou Densetsu Special</b></i> <span style="font-weight: normal">(<span class="t_nihongo_kanji" lang="ja" xml:lang="ja">熱闘リアルバウト餓狼伝説SPECIAL</span><sup class="t_nihongo_help noprint"><a href="/wiki/Help:Installing_Japanese_character_sets" title="Help:Installing Japanese character sets"><span class="t_nihongo_icon" style="color: #00e; font: bold 80% sans-serif; text-decoration: none; padding: 0 .1em;">?</span></a></sup>)</span>, was released only in Japan on March 27, 1998, featuring simplified graphics and two-button gameplay. This version features only 12 playable characters: a roster which consists of Terry, Andy, Joe, Mai, Blue Mary, Duck King, Kim, Jin Chonrei, Billy, Yamazaki, Laurence, and Krauser. Geese Howard appears as a hidden character, as well as <a href="/wiki/Iori_Yagami" title="Iori Yagami">Iori Yagami</a> from <i><a href="/wiki/The_King_of_Fighters" title="The King of Fighters">The King of Fighters</a></i> series.</p>
