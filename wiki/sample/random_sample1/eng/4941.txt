<p><b>Riebiņi Municipality</b> (<a href="/wiki/Latvian_language" title="Latvian language">Latvian</a>: <span lang="lv"><i>Riebiņu novads</i></span>, <a href="/wiki/Latgalian_language" title="Latgalian language">Latgalian</a>: <span lang="ltg"><i>Ribinišku nūvods</i></span>) is a <a href="/wiki/Municipality" title="Municipality">municipality</a> in <a href="/wiki/Latgale" title="Latgale">Latgale</a>, <a href="/wiki/Latvia" title="Latvia">Latvia</a>. The municipality was formed in 2004 by <a href="/wiki/Merger_(politics)" title="Merger (politics)">merging</a> <a href="/wiki/Gal%C4%93ni_parish" title="Galēni parish">Galēni parish</a>, <a href="/wiki/Riebi%C5%86i_parish" title="Riebiņi parish">Riebiņi parish</a>, <a href="/wiki/Ru%C5%A1ona_parish" title="Rušona parish">Rušona parish</a>, <a href="/wiki/Silaj%C4%81%C5%86i_parish" title="Silajāņi parish">Silajāņi parish</a>, <a href="/wiki/S%C4%AB%C4%BCukalns_parish" title="Sīļukalns parish">Sīļukalns parish</a> and <a href="/wiki/Stabulnieki_parish" title="Stabulnieki parish">Stabulnieki parish</a>, the administrative centre being <a href="/wiki/Riebi%C5%86i" title="Riebiņi">Riebiņi</a>.</p>
