<p>In <a href="/wiki/Complex_analysis" title="Complex analysis">complex analysis</a>, <b>Liouville's theorem</b>, named after <a href="/wiki/Joseph_Liouville" title="Joseph Liouville">Joseph Liouville</a>, states that every <a href="/wiki/Bounded_function" title="Bounded function">bounded</a> <a href="/wiki/Entire_function" title="Entire function">entire function</a> must be constant. That is, every <a href="/wiki/Holomorphic_function" title="Holomorphic function">holomorphic function</a> <i>f</i> for which there exists a positive number <i>M</i> such that <span><span class="mwe-math-mathml-inline mwe-math-mathml-a11y mw-math-element" style="display: none;"><math xmlns="http://www.w3.org/1998/Math/MathML">

<p>The theorem is considerably improved by <a href="/wiki/Picard_theorem" title="Picard theorem">Picard's little theorem</a>, which says that every entire function whose image omits two or more complex numbers must be constant.</p>

<h2><span class="mw-headline" id="Corollaries">Corollaries</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Liouville%27s_theorem_(complex_analysis)&amp;action=edit&amp;section=2" title="Edit section: Corollaries">edit</a><span class="mw-editsection-bracket">]</span></span></h2>
<h3><span class="mw-headline" id="Fundamental_theorem_of_algebra">Fundamental theorem of algebra</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Liouville%27s_theorem_(complex_analysis)&amp;action=edit&amp;section=3" title="Edit section: Fundamental theorem of algebra">edit</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>There is a short <a href="/wiki/Fundamental_theorem_of_algebra#Complex-analytic_proofs" title="Fundamental theorem of algebra">proof of the fundamental theorem of algebra</a> based upon Liouville's theorem.<sup class="reference" id="cite_ref-1"><a href="#cite_note-1">[1]</a></sup></p>
<h3><span class="mw-headline" id="No_entire_function_dominates_another_entire_function">No entire function dominates another entire function</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Liouville%27s_theorem_(complex_analysis)&amp;action=edit&amp;section=4" title="Edit section: No entire function dominates another entire function">edit</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>A consequence of the theorem is that "genuinely different" entire functions cannot dominate each other, i.e. if <i>f</i> and <i>g</i> are entire, and |<i>f</i>| ≤ |<i>g</i>| everywhere, then <i>f</i> = α·<i>g</i> for some complex number α. Consider that for g=0 the theorem is trivial so we assume g<span><span class="mwe-math-mathml-inline mwe-math-mathml-a11y mw-math-element" style="display: none;"><math xmlns="http://www.w3.org/1998/Math/MathML">

Consider the function <i>h</i> = <i>f</i>/<i>g</i>. It is enough to prove that <i>h</i> can be extended to an entire function, in which case the result follows by Liouville's theorem. The holomorphy of <i>h</i> is clear except at points in <i>g</i><sup>−1</sup>(0). But since <i>h</i> is bounded and all the zeroes of <i>g</i> are isolated, any singularities must be removable. Thus <i>h</i> can be extended to an entire bounded function which by Liouville's theorem implies it is constant.</p>

<h3><span class="mw-headline" id="Non-constant_elliptic_functions_cannot_be_defined_on_C">Non-constant elliptic functions cannot be defined on C</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Liouville%27s_theorem_(complex_analysis)&amp;action=edit&amp;section=6" title="Edit section: Non-constant elliptic functions cannot be defined on C">edit</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>The theorem can also be used to deduce that the domain of a non-constant <a href="/wiki/Elliptic_function" title="Elliptic function">elliptic function</a> <i>f</i> cannot be <b>C</b>. Suppose it was. Then, if <i>a</i> and <i>b</i> are two periods of <i>f</i> such that <sup><i>a</i></sup>⁄<sub><i>b</i></sub> is not real, consider the <a href="/wiki/Parallelogram" title="Parallelogram">parallelogram</a> <i>P</i> whose <a href="/wiki/Vertex_(geometry)" title="Vertex (geometry)">vertices</a> are 0, <i>a</i>, <i>b</i> and <i>a</i> + <i>b</i>. Then the image of <i>f</i> is equal to <i>f</i>(<i>P</i>). Since <i>f</i> is <a class="mw-redirect" href="/wiki/Continuous_functions" title="Continuous functions">continuous</a> and <i>P</i> is <a href="/wiki/Compact_space" title="Compact space">compact</a>, <i>f</i>(<i>P</i>) is also compact and, therefore, it is bounded. So, <i>f</i> is constant.</p>
<p>The fact that the domain of a non-constant elliptic function <i>f</i> can not be <b>C</b> is what Liouville actually proved, in 1847, using the theory of elliptic functions.<sup class="reference" id="cite_ref-2"><a href="#cite_note-2">[2]</a></sup> In fact, it was <a class="mw-redirect" href="/wiki/Augustin_Louis_Cauchy" title="Augustin Louis Cauchy">Cauchy</a> who proved Liouville's theorem.<sup class="reference" id="cite_ref-3"><a href="#cite_note-3">[3]</a></sup><sup class="reference" id="cite_ref-4"><a href="#cite_note-4">[4]</a></sup></p>

<h2><span class="mw-headline" id="Remarks">Remarks</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Liouville%27s_theorem_(complex_analysis)&amp;action=edit&amp;section=9" title="Edit section: Remarks">edit</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>Let <b>C</b> ∪ {∞} be the one point compactification of the complex plane <b>C</b>. In place of holomorphic functions defined on regions in <b>C</b>, one can consider regions in <b>C</b> ∪ {∞}. Viewed this way, the only possible singularity for entire functions, defined on <b>C</b> ⊂ <b>C</b> ∪ {∞}, is the point ∞. If an entire function <i>f</i> is bounded in a neighborhood of ∞, then ∞ is a <a href="/wiki/Removable_singularity" title="Removable singularity">removable singularity</a> of <i>f</i>, i.e. <i>f</i> cannot blow up or behave erratically at ∞. In light of the power series expansion, it is not surprising that Liouville's theorem holds.</p>
<p>Similarly, if an entire function has a <a href="/wiki/Pole_(complex_analysis)" title="Pole (complex analysis)">pole</a> at ∞, i.e. blows up like <i>z<sup>n</sup></i> in some neighborhood of ∞, then <i>f</i> is a polynomial. This extended version of Liouville's theorem can be more precisely stated: if |<i>f</i>(<i>z</i>)| ≤ <i>M</i>.|<i>z<sup>n</sup></i>| for |<i>z</i>| sufficiently large, then <i>f</i> is a polynomial of degree at most <i>n</i>.
