<p><b>Microsoft Servers</b> (previously called <i>Windows Server System</i>) is a brand that encompasses <a href="/wiki/Microsoft" title="Microsoft">Microsoft</a>'s <a href="/wiki/Server_(computing)" title="Server (computing)">server</a> products. This includes the <a href="/wiki/Windows_Server" title="Windows Server">Windows Server</a> editions of the <a href="/wiki/Microsoft_Windows" title="Microsoft Windows">Microsoft Windows</a> <a href="/wiki/Operating_system" title="Operating system">operating system</a> itself, as well as products targeted at the wider business market. Unlike <a href="/wiki/Microsoft_Dynamics" title="Microsoft Dynamics">Microsoft Dynamics</a> or <a href="/wiki/Microsoft_Office" title="Microsoft Office">Microsoft Office</a> product lines, most of the products sold under this brand are not intended to be line-of-business services in and of themselves.</p>

<h2><span class="mw-headline" id="Servers">Servers</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Microsoft_Servers&amp;action=edit&amp;section=1" title="Edit section: Servers">edit</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>The Microsoft Servers line of software and technology includes:</p>
<h3><span class="mw-headline" id="Operating_systems">Operating systems</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Microsoft_Servers&amp;action=edit&amp;section=2" title="Edit section: Operating systems">edit</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>Initially, Microsoft server <a href="/wiki/Operating_system" title="Operating system">operating systems</a> were editions of <a class="mw-redirect" href="/wiki/Windows_NT_family" title="Windows NT family">Windows NT family</a>. Starting in 2003, a <a href="/wiki/Windows_Server" title="Windows Server">Windows Server</a> subfamily was formed that encompasses server operating system versions independent of the client versions of Windows.</p>
<p>Server operating systems of Microsoft include:</p>
<ul>
<li><a href="/wiki/Windows_NT_3.1" title="Windows NT 3.1">Windows NT 3.1 Advanced Server edition</a></li>
<li><a href="/wiki/Windows_NT_3.5" title="Windows NT 3.5">Windows NT 3.5 Server edition</a></li>
<li><a href="/wiki/Windows_NT_3.51" title="Windows NT 3.51">Windows NT 3.51 Server edition</a></li>
<li><a class="mw-redirect" href="/wiki/Windows_NT_4.0_Server" title="Windows NT 4.0 Server">Windows NT 4.0</a> (Server, Server Enterprise, and Terminal Server editions)</li>
<li><a href="/wiki/Windows_2000" title="Windows 2000">Windows 2000</a> (Server, Advanced Server, and Datacenter Server editions)</li>
<li><a href="/wiki/Windows_Server_2003" title="Windows Server 2003">Windows Server 2003</a>, including:
<ul>
<li><a href="/wiki/Windows_Home_Server" title="Windows Home Server">Windows Home Server</a></li>
<li><a href="/wiki/Windows_Server_2003#Windows_Storage_Server" title="Windows Server 2003">Windows Storage Server</a></li>
<li><a href="/wiki/Windows_Server_2003#Windows_Compute_Cluster_Server" title="Windows Server 2003">Windows Compute Cluster Server</a></li>
</ul>
</li>
<li><a class="mw-redirect" href="/wiki/Windows_Server_2003_R2" title="Windows Server 2003 R2">Windows Server 2003 R2</a></li>
<li><a href="/wiki/Windows_Server_2008" title="Windows Server 2008">Windows Server 2008</a>, including:
<ul>
<li><a href="/wiki/Windows_Essential_Business_Server_2008" title="Windows Essential Business Server 2008">Windows Essential Business Server 2008</a></li>
<li><a href="/wiki/Windows_HPC_Server_2008" title="Windows HPC Server 2008">Windows HPC Server 2008</a></li>
</ul>
</li>
<li><a href="/wiki/Windows_Server_2008_R2" title="Windows Server 2008 R2">Windows Server 2008 R2</a>, including:
<ul>
<li><a href="/wiki/Windows_Home_Server_2011" title="Windows Home Server 2011">Windows Home Server 2011</a></li>
</ul>
</li>
<li><a href="/wiki/Windows_Server_2012" title="Windows Server 2012">Windows Server 2012</a></li>
<li><a class="mw-redirect" href="/wiki/Windows_Server_2012_R2" title="Windows Server 2012 R2">Windows Server 2012 R2</a></li>
<li><a href="/wiki/Windows_Server_2016" title="Windows Server 2016">Windows Server 2016</a></li>
</ul>

<h2><span class="mw-headline" id="Microsoft_System_Center">Microsoft System Center</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Microsoft_Servers&amp;action=edit&amp;section=6" title="Edit section: Microsoft System Center">edit</a><span class="mw-editsection-bracket">]</span></span></h2>
<p><b>Microsoft System Center</b>, a set of server products, aims specifically at helping corporate <a href="/wiki/System_administrator" title="System administrator">system administrators</a> manage a network of Windows Server and client desktop systems.<sup class="reference" id="cite_ref-4"><a href="#cite_note-4">[4]</a></sup></p>
<ul>
<li><a href="/wiki/System_Center_Advisor" title="System Center Advisor">System Center Advisor</a> – <a class="mw-redirect" href="/wiki/Software-as-a-service" title="Software-as-a-service">Software-as-a-service</a> offering that helps change or assess the configuration of Microsoft Servers software over the Internet</li>
<li><a class="new" href="/w/index.php?title=System_Center_App_Controller&amp;action=edit&amp;redlink=1" title="System Center App Controller (page does not exist)">System Center App Controller</a> - Unified management for public and private clouds, including cloud-based virtual machines and services</li>
<li><a href="/wiki/System_Center_Configuration_Manager" title="System Center Configuration Manager">System Center Configuration Manager</a> – Configuration management, hardware/software asset management, patch deployment tools for Windows desktops (previously Systems Management Server); includes Software Center.<sup class="reference" id="cite_ref-5"><a href="#cite_note-5">[5]</a></sup></li>
<li><a href="/wiki/System_Center_Data_Protection_Manager" title="System Center Data Protection Manager">System Center Data Protection Manager</a> – <a href="/wiki/Continuous_data_protection" title="Continuous data protection">Continuous data protection</a> and <a href="/wiki/Data_recovery" title="Data recovery">data recovery</a></li>
<li><a class="new" href="/w/index.php?title=System_Center_Endpoint_Protection&amp;action=edit&amp;redlink=1" title="System Center Endpoint Protection (page does not exist)">System Center Endpoint Protection</a> - Anti-malware and security tools for Microsoft products</li>
<li><a href="/wiki/System_Center_Essentials" title="System Center Essentials">System Center Essentials</a> – Combined features of Operations Manager and Windows Software Update Service (WSUS), aimed at small- and medium-sized <a href="/wiki/Business" title="Business">businesses</a></li>
<li><a class="new" href="/w/index.php?title=System_Center_Orchestrator&amp;action=edit&amp;redlink=1" title="System Center Orchestrator (page does not exist)">System Center Orchestrator</a> (formerly Opalis) - An automation platform for orchestrating and integrating administrative tools to decrease the cost of datacenter operations while improving the reliability of IT processes. It enables organizations to automate best practices, such as those found in Microsoft Operations Framework (MOF) and <a class="mw-redirect" href="/wiki/Information_Technology_Infrastructure_Library" title="Information Technology Infrastructure Library">Information Technology Infrastructure Library</a> (ITIL). Orchestrator operates through workflow processes that coordinate System Center and other management tools to automate incident response, change and compliance, and service-lifecycle management processes.<sup class="reference" id="cite_ref-Opalis_6-0"><a href="#cite_note-Opalis-6">[6]</a></sup></li>
<li><a href="/wiki/System_Center_Operations_Manager" title="System Center Operations Manager">System Center Operations Manager</a> – Services and application monitoring</li>
<li><a href="/wiki/System_Center_Service_Manager" title="System Center Service Manager">System Center Service Manager</a> – Ties in with SCOM, SCCM for asset tracking as well as incident, problem, change and configuration management (code name: Service Desk)</li>
<li><a href="/wiki/System_Center_Virtual_Machine_Manager" title="System Center Virtual Machine Manager">System Center Virtual Machine Manager</a> – Virtual-machine management and datacenter virtualization</li>
</ul>
<h2><span class="mw-headline" id="Discontinued_server_products">Discontinued server products</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Microsoft_Servers&amp;action=edit&amp;section=7" title="Edit section: Discontinued server products">edit</a><span class="mw-editsection-bracket">]</span></span></h2>
<ul>
<li><a class="mw-redirect" href="/wiki/Services_for_UNIX" title="Services for UNIX">Services for UNIX</a> – SFU for Windows XP Pro and Windows Server 2003 only</li>
<li>Microsoft Application Center – Deployment of web applications across multiple servers. Some of its capabilities are now in System Center.</li>
<li><a href="/wiki/Microsoft_BackOffice_Server" title="Microsoft BackOffice Server">Microsoft BackOffice Server</a></li>
<li><a href="/wiki/Microsoft_Content_Management_Server" title="Microsoft Content Management Server">Content Management Server</a> – <a class="mw-redirect" href="/wiki/Web_site" title="Web site">Web site</a> <a href="/wiki/Content_management" title="Content management">content management</a> and publishing. Merged into <a class="mw-redirect" href="/wiki/Microsoft_SharePoint_Server" title="Microsoft SharePoint Server">Microsoft SharePoint Server</a>.</li>

<li>Microsoft Proxy Server – replaced by <a href="/wiki/Microsoft_Forefront_Threat_Management_Gateway" title="Microsoft Forefront Threat Management Gateway">Microsoft Forefront Threat Management Gateway</a> (also known as ISA Server)</li>
