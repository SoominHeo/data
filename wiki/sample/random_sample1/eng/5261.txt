<p><b>Zeta Aurigae</b> (ζ Aur, ζ Aurigae) is the <a href="/wiki/Bayer_designation" title="Bayer designation">Bayer designation</a> for a <a href="/wiki/Binary_star" title="Binary star">binary star</a> system in the northern <a href="/wiki/Constellation" title="Constellation">constellation</a> of <a href="/wiki/Auriga_(constellation)" title="Auriga (constellation)">Auriga</a>. It has the traditional names <b>Haedus</b> and <b>Sadatoni</b>. Based upon <a href="/wiki/Parallax" title="Parallax">parallax</a> measurements made during the <a href="/wiki/Hipparcos" title="Hipparcos">Hipparcos</a> mission, this system is approximately 790 <a href="/wiki/Light-year" title="Light-year">light-years</a> (240 <a href="/wiki/Parsec" title="Parsec">parsecs</a>) distant from the Earth. It has a combined <a class="mw-redirect" href="/wiki/Apparent_visual_magnitude" title="Apparent visual magnitude">apparent visual magnitude</a> of 3.75,<sup class="reference" id="cite_ref-paspj19_209_2-3"><a href="#cite_note-paspj19_209-2">[2]</a></sup> which is bright enough to be seen with the naked eye.</p>
<p>This is an <a class="mw-redirect" href="/wiki/Eclipsing_binary" title="Eclipsing binary">eclipsing binary</a> system with the <a href="/wiki/Orbital_plane_(astronomy)" title="Orbital plane (astronomy)">orbital plane</a> being oriented close to the line of sight from the Earth; the <a class="mw-redirect" href="/wiki/Inclination" title="Inclination">inclination</a> of this system is estimated as 87.0°.<sup class="reference" id="cite_ref-apj679_2_1490_6-2"><a href="#cite_note-apj679_2_1490-6">[6]</a></sup> As a result, an eclipse of one star by the other occurs during each orbit, causing the magnitude to decrease to +3.99. The pair have an <a href="/wiki/Orbital_period" title="Orbital period">orbital period</a> of 972 days (2.66 years) and an <a href="/wiki/Orbital_eccentricity" title="Orbital eccentricity">eccentricity</a> of 0.4.<sup class="reference" id="cite_ref-apj679_2_1490_6-3"><a href="#cite_note-apj679_2_1490-6">[6]</a></sup> The primary component has been categorized as a <a class="mw-redirect" href="/wiki/K-type_star" title="K-type star">K-type</a> <a href="/wiki/Bright_giant" title="Bright giant">bright giant</a> or <a class="mw-redirect" href="/wiki/Supergiant" title="Supergiant">supergiant</a> star. Its companion is a <a class="mw-redirect" href="/wiki/B-type_main_sequence_star" title="B-type main sequence star">B-type main sequence star</a> of <a href="/wiki/Stellar_classification" title="Stellar classification">stellar classification</a> B5 V or B7 V.<sup class="reference" id="cite_ref-apj679_2_1490_6-4"><a href="#cite_note-apj679_2_1490-6">[6]</a></sup><sup class="reference" id="cite_ref-ar55_1_31_4-1"><a href="#cite_note-ar55_1_31-4">[4]</a></sup></p>
<h2><span class="mw-headline" id="Etymology">Etymology</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Zeta_Aurigae&amp;action=edit&amp;section=1" title="Edit section: Etymology">edit</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>It has the traditional names <b>Haedus</b> (also <b>Hoedus</b>) and <b>Sadatoni</b> (rarely <i>Saclateni).</i> It is one of the two <i>hædi</i> (kids) of the she-goat <a class="mw-redirect" href="/wiki/Capella_(star)" title="Capella (star)">Capella</a>, the other being Hoedus II, <a href="/wiki/Eta_Aurigae" title="Eta Aurigae">Eta Aurigae</a>. The name Sadatoni is from the Arabic الساعد الثاني <i>as-sā<sup>c</sup>id aθ-θānī</i> "the second arm (of the charioteer)". The rare traditional name <i>Azaleh</i> is shared (in the form <i>Hassaleh</i>) with <a href="/wiki/Iota_Aurigae" title="Iota Aurigae">Iota Aurigae</a>.<sup class="noprint Inline-Template Template-Fact" style="white-space:nowrap;">[<i><a href="/wiki/Wikipedia:Citation_needed" title="Wikipedia:Citation needed"><span title='"For'>citation needed</span></a></i>]</sup></p>
<p>In <a href="/wiki/Chinese_language" title="Chinese language">Chinese</a>, <span lang="zh" xml:lang="zh">柱</span> (<span lang="zh-Latn" xml:lang="zh-Latn">Zhù</span>), meaning <i><a href="/wiki/Net_(Chinese_constellation)" title="Net (Chinese constellation)">Pillars</a></i>, refers to an asterism consisting of ζ Aurigae, <a href="/wiki/Epsilon_Aurigae" title="Epsilon Aurigae">ε Aurigae</a>, <a href="/wiki/Eta_Aurigae" title="Eta Aurigae">η Aurigae</a>, <a href="/wiki/Upsilon_Aurigae" title="Upsilon Aurigae">υ Aurigae</a>, <a href="/wiki/Nu_Aurigae" title="Nu Aurigae">ν Aurigae</a>, <a href="/wiki/Tau_Aurigae" title="Tau Aurigae">τ Aurigae</a>, <a href="/wiki/Chi_Aurigae" title="Chi Aurigae">χ Aurigae</a> and <a href="/wiki/26_Aurigae" title="26 Aurigae">26 Aurigae</a>.<sup class="reference" id="cite_ref-11"><a href="#cite_note-11">[11]</a></sup> Consequently, ζ Aurigae itself is known as <span lang="zh" xml:lang="zh">柱二</span> (<span lang="zh-Latn" xml:lang="zh-Latn">Zhù èr</span>, English: the Second Star of Pillars.)<sup class="reference" id="cite_ref-12"><a href="#cite_note-12">[12]</a></sup></p>
