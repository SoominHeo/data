<p><b>Mac OS 8</b> is an <a href="/wiki/Operating_system" title="Operating system">operating system</a> that was released by <a href="/wiki/Apple_Inc." title="Apple Inc.">Apple Computer</a> on July 26, 1997. It represented the largest overhaul of the <a href="/wiki/Classic_Mac_OS" title="Classic Mac OS">classic Mac OS</a> since the release of <a class="mw-redirect" href="/wiki/System_7_(Macintosh)" title="System 7 (Macintosh)">System 7</a>, some six years previously. It puts more emphasis on color than previous operating systems. Released over a series of updates, Mac OS 8 was an effort to integrate many of the technologies developed for Apple's overly-ambitious <a href="/wiki/Operating_system" title="Operating system">operating system</a> known as <a href="/wiki/Copland_(operating_system)" title="Copland (operating system)">Copland</a>. Mac OS 8 helped modernize the Mac OS while Apple developed its next generation operating system, <a href="/wiki/MacOS" title="MacOS">macOS</a>. Mac OS 8 is one of Apple's most successful software releases, selling over 1.2 million copies in the first two weeks.<sup class="reference" id="cite_ref-Mac_OS_8_Sales_1-0"><a href="#cite_note-Mac_OS_8_Sales-1">[1]</a></sup><sup class="reference" id="cite_ref-Cyberian_Sales_2-0"><a href="#cite_note-Cyberian_Sales-2">[2]</a></sup> Coming as it did at a difficult time in Apple's history, many <a class="mw-redirect" href="/wiki/Software_piracy" title="Software piracy">pirate</a> groups refused to traffic in the new operating system, encouraging people to buy it instead.<sup class="reference" id="cite_ref-3"><a href="#cite_note-3">[3]</a></sup></p>
<p>Mac OS 8.0 brought about the most visible changes in the line-up, including the introduction of the <a class="mw-redirect" href="/wiki/Platinum_(Macintosh)" title="Platinum (Macintosh)">Platinum</a> interface and a native <a href="/wiki/PowerPC" title="PowerPC">PowerPC</a> <a class="mw-redirect" href="/wiki/Thread_(computer_science)" title="Thread (computer science)">multi-threaded</a> <a class="mw-redirect" href="/wiki/Macintosh_Finder" title="Macintosh Finder">Finder</a>. Mac OS 8.1 introduced a new, more efficient <a href="/wiki/File_system" title="File system">file system</a> known as <a href="/wiki/HFS_Plus" title="HFS Plus">HFS Plus</a>. Mac OS 8.5 was the first version of the Mac OS to require a <a class="mw-redirect" href="/wiki/PowerPC_processor" title="PowerPC processor">PowerPC processor</a>. It featured PowerPC native versions of <a href="/wiki/QuickDraw" title="QuickDraw">QuickDraw</a> and <a href="/wiki/AppleScript" title="AppleScript">AppleScript</a>, along with the <a class="mw-redirect" href="/wiki/Apple_Sherlock_(software)" title="Apple Sherlock (software)">Sherlock</a> search utility. Its successor, <a href="/wiki/Mac_OS_9" title="Mac OS 9">Mac OS 9</a>, was released on October 23, 1999.</p>

<h2><span class="mw-headline" id="Copland">Copland</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Mac_OS_8&amp;action=edit&amp;section=1" title="Edit section: Copland">edit</a><span class="mw-editsection-bracket">]</span></span></h2>
<div class="hatnote" role="note">Main article: <a href="/wiki/Copland_(operating_system)" title="Copland (operating system)">Copland (operating system)</a></div>
<p>Apple's next generation operating system, which it originally envisioned as "System 8" was codenamed Copland. It was announced in March 1994 alongside the introduction of the first PowerPC Macs. Apple intended Copland as a completely native PowerPC operating system offering <a href="/wiki/Intelligent_agent" title="Intelligent agent">intelligent agents</a>, a <a href="/wiki/Microkernel" title="Microkernel">microkernel</a>, a customizable interface known as <a href="/wiki/Appearance_Manager" title="Appearance Manager">Appearance Manager</a>, <a class="mw-redirect" href="/wiki/Hardware_abstraction_layer" title="Hardware abstraction layer">hardware abstraction</a>, and a <a href="/wiki/Relational_database" title="Relational database">relational database</a> integrated into the Finder. Copland was to be followed by <a class="mw-redirect" href="/wiki/Gershwin_operating_system" title="Gershwin operating system">Gershwin</a>, which promised <a href="/wiki/Memory_protection" title="Memory protection">protected memory spaces</a> and full <a href="/wiki/Preemption_(computing)" title="Preemption (computing)">preemptive multitasking</a>.<sup class="reference" id="cite_ref-Copland_4-0"><a href="#cite_note-Copland-4">[4]</a></sup> The operating system was intended to be a complete re-write of the Mac OS, and Apple hoped to beat <a class="mw-redirect" href="/wiki/Microsoft_Windows_95" title="Microsoft Windows 95">Microsoft Windows 95</a> to market with a development cycle of just one year.</p>
<p>The Copland development was hampered by countless missed deadlines. The release date was first pushed back to the end of 1995, then to mid-'96, late '96, and finally to the end of 1997. With a dedicated team of 500 software engineers and an annual budget of $250 million, Apple executives began to grow impatient with the project continually falling behind schedule. At the <a class="mw-redirect" href="/wiki/Worldwide_Developers_Conference" title="Worldwide Developers Conference">Worldwide Developers Conference</a> in January 1997, Apple <a class="mw-redirect" href="/wiki/CEO" title="CEO">CEO</a> <a href="/wiki/Gil_Amelio" title="Gil Amelio">Gil Amelio</a> announced that rather than release Copland as a single monolithic release, Copland features would be phased into the Mac OS following a six-month release cycle. These updates began with Mac OS 7.6, released during WWDC. Mac OS 8.0, released six months later, continued to integrate Copland technologies into the Mac OS.</p>
<p>In August 1996, Apple <a class="mw-redirect" href="/wiki/Chief_technical_officer" title="Chief technical officer">Chief Technology Officer</a>, <a href="/wiki/Ellen_Hancock" title="Ellen Hancock">Ellen Hancock</a>, froze development of Copland<sup class="reference" id="cite_ref-5"><a href="#cite_note-5">[5]</a></sup> and Apple began a search for an operating system developed outside the company.<sup class="reference" id="cite_ref-Copland_4-1"><a href="#cite_note-Copland-4">[4]</a></sup> This ultimately led to Apple's purchase of <a href="/wiki/NeXT" title="NeXT">NeXT</a> and the development of macOS.</p>
<h2><span class="mw-headline" id="Mac_OS_8.0">Mac OS 8.0</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Mac_OS_8&amp;action=edit&amp;section=2" title="Edit section: Mac OS 8.0">edit</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>Developed under the codename "Tempo", Mac OS 8.0 was released on July 26, 1997. Initially, the early beta releases of the product which were circulated to developers and Apple internal audiences, were branded as Mac OS 7.7 (superseding the then-current release, Mac OS 7.6). Afterwards, the software was later renamed to Mac OS 8 before the final release.</p>
<p>Major improvements in this version included the <a class="mw-redirect" href="/wiki/Platinum_(theme)" title="Platinum (theme)">Platinum</a> theme, a Finder which was PowerPC native and multi-threaded, and greater customization of the <a href="/wiki/User_interface" title="User interface">user interface</a>.</p>
<p>Other features introduced in Mac OS 8.0 include the following:<sup class="reference" id="cite_ref-Mac_OS_8.0_Features_6-0"><a href="#cite_note-Mac_OS_8.0_Features-6">[6]</a></sup></p>
<ul>
<li>Customization of system fonts and increased usage of the user-set accent color.</li>
<li>Pop-up <a href="/wiki/Context_menu" title="Context menu">context menus</a> (accessed via ctrl-click with a single-button mouse)</li>
<li>Pop-up (or tabbed) windows in the Finder.</li>
<li>Spring-loaded folders.</li>
<li>Live scrolling.</li>
<li>WindowShade widget in window titlebars.</li>
<li>Multithreaded <a href="/wiki/Finder_(software)" title="Finder (software)">Finder</a> — file copy operations run in a separate thread and don't block the Finder UI.</li>
<li>Revamped color picker.</li>
<li>Desktop Pictures control panel, allowing photographs to be set as the desktop background. (not just tiled patterns)</li>
<li>Simple Finder, an option which reduces Finder menus to basic operations, in order to avoid overwhelming new users.</li>
<li>Relocation of the 'Help' menu from an icon at the right end of the menu bar to a standard textual menu positioned after the application's menus.</li>
<li>A faster Apple Guide, featuring <a href="/wiki/HTML" title="HTML">HTML</a> help pages.</li>
<li>Native support of <a href="/wiki/Apple_Filing_Protocol" title="Apple Filing Protocol">Apple Filing Protocol</a> over IP.</li>
<li>Performance improvements to <a href="/wiki/Virtual_memory" title="Virtual memory">virtual memory</a>, AppleScript execution and system startup times.</li>
<li>Faster desktop rebuilding.</li>
</ul>
<h2><span class="mw-headline" id="Mac_OS_8.1">Mac OS 8.1</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Mac_OS_8&amp;action=edit&amp;section=3" title="Edit section: Mac OS 8.1">edit</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>Released on January 19, 1998, Mac OS 8.1 was the last version of the Mac OS to run on 68k processors. It addressed performance and reliability improvements. It introduced a new file system known as <a href="/wiki/HFS_Plus" title="HFS Plus">HFS+</a>, (aka Mac OS Extended,) which supported large file sizes and made more efficient use of larger hard drives due to using a smaller block size. To upgrade, users must <a class="mw-redirect" href="/wiki/Disk_format" title="Disk format">reformat</a> the hard drive, which deletes the entire contents of the drive. Some third-party utilities later appeared that preserved the user's data while upgrading to HFS+. Note that 68040 systems do not support booting from HFS+ disks; the boot drive must be HFS.<sup class="reference" id="cite_ref-7"><a href="#cite_note-7">[7]</a></sup></p>
<p>Mac OS 8.1 was the first system to have a DVD <a href="/wiki/Universal_Disk_Format" title="Universal Disk Format">Universal Disk Format</a> (UDF) driver and also shipped with the new Java runtime (JDK 1.13).</p>
<p>Mac OS 8.1 also included an enhanced version of <a href="/wiki/PC_Exchange" title="PC Exchange">PC Exchange</a>, allowing Macintosh users to see the long file names (up to 255 characters) on files that had been created on PCs running <a href="/wiki/Microsoft_Windows" title="Microsoft Windows">Microsoft Windows</a>, as well as supporting <a href="/wiki/File_Allocation_Table" title="File Allocation Table">FAT32</a>.</p>
<p>Mac OS 8.1 is the earliest version of the Mac OS that can run <a class="mw-redirect" href="/wiki/Carbon_(computing)" title="Carbon (computing)">Carbon</a> applications. Carbon support requires a PowerPC processor and installation of the CarbonLib software from Apple's web site; it is not a standard component of Mac OS 8.1. More recent versions of CarbonLib require Mac OS 8.6. Applications requiring later versions of CarbonLib will not run on Mac OS 8.1.</p>
<p>As part of <a href="/wiki/Internet_Explorer_for_Mac" title="Internet Explorer for Mac">Apple's agreement with Microsoft</a>, 8.1 included <a href="/wiki/Internet_Explorer_3" title="Internet Explorer 3">Internet Explorer 3</a> initially, but soon switched to <a href="/wiki/Internet_Explorer_4" title="Internet Explorer 4">Internet Explorer 4</a> as its default browser.</p>
<p>Mac OS 8.1 was free for Mac OS 8 owners and was available in February 1998 via the apple.com website.</p>
<h2><span class="mw-headline" id="Mac_OS_8.5">Mac OS 8.5</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Mac_OS_8&amp;action=edit&amp;section=4" title="Edit section: Mac OS 8.5">edit</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>Released October 17, 1998, Mac OS 8.5 was the first version of the Mac OS to run solely on <a class="mw-redirect" href="/wiki/Apple_Macintosh" title="Apple Macintosh">Macs</a> equipped with a <a href="/wiki/PowerPC" title="PowerPC">PowerPC</a> processor. If Mac OS 8.5 is installed on a 68k system, the <a class="mw-redirect" href="/wiki/Sad_Mac" title="Sad Mac">Sad Mac</a> error screen will appear. As such, it replaced some but not all of the 680x0 code with <a href="/wiki/PowerPC" title="PowerPC">PowerPC</a> code, improving system performance by relying less on <a class="mw-redirect" href="/wiki/Mac_68K_emulator" title="Mac 68K emulator">680x0 emulation</a>.</p>
<p>It introduced the <a class="mw-redirect" href="/wiki/Apple_Sherlock" title="Apple Sherlock">Sherlock</a> search utility; Sherlock allowed users to search the contents of documents on hard drives (if the user had let it index the drive), or extend a search to the Internet. Sherlock plug-ins started appearing at this time; these plug-ins allowed users to search the contents of other websites.</p>
<p>Mac OS 8.5 included a number of performance improvements. Copying files over a network was faster than previous versions and Apple advertised it as being <i>"faster than Windows NT"</i>.<sup class="reference" id="cite_ref-Mac_OS_8.5_Press_Release_8-0"><a href="#cite_note-Mac_OS_8.5_Press_Release-8">[8]</a></sup> <a href="/wiki/AppleScript" title="AppleScript">AppleScript</a> was also re-written to use only PowerPC code, which significantly improved AppleScript execution speed.</p>
<p><a class="mw-redirect" href="/wiki/Font_Smoothing" title="Font Smoothing">Font Smoothing</a>, system-wide antialiasing for type was also introduced.</p>
<p>The <a href="/wiki/HTML" title="HTML">HTML</a> format for online help, first adopted by the Finder's Info Center in Mac OS 8, was now used throughout. This made it easier for software companies to write online help systems, but would contribute to making physical manuals become a thing of the past.</p>
<p>In this release, the PPP control panel was removed and replaced with Remote Access. The Remote Access control panel provides the same functionality but also allows connections to <a href="/wiki/AppleTalk_Remote_Access" title="AppleTalk Remote Access">Apple Remote Access</a> (ARA) servers.</p>
<p>The installation process was considerably simplified in Mac OS 8.5. In earlier versions the installer worked in segments and often required the user to click to continue in between stages of the installation. The Mac OS 8.5 installer generally required very little user interaction once it was started. Customisation options were also much more detailed yet simpler to manage.</p>
<p>From Mac OS 8.5 onwards, MacLinkPlus document translation software is no longer bundled as part of the Mac OS.</p>
<p>Mac OS 8.5 was the first version of the Mac OS to support "<a class="mw-redirect" href="/wiki/Appearance_Manager_(Mac_OS)" title="Appearance Manager (Mac OS)">themes</a>," or skins, which could change the default Apple Platinum look of the Mac OS to "Gizmo" or "HiTech" themes. This radical changing of the computer's appearance was removed at the last minute, and appeared only in beta versions, though users could still make (and share) their own themes and use them with the OS. The Appearance control panel was also updated to enable support for proportional scroll bars, and added the option for both scroll arrows to be placed at the bottom of the scroll bar.</p>
<p>In addition to the themes support, 8.5 was the first version to support <a href="/wiki/32-bit" title="32-bit">32-bit</a> <a href="/wiki/Icon_(computing)" title="Icon (computing)">icons</a>. Icons now had <a href="/wiki/24-bit" title="24-bit">24-bit</a> color (16.7 million colors) and an <a href="/wiki/8-bit" title="8-bit">8-bit</a> <a class="mw-redirect" href="/wiki/Alpha_channel" title="Alpha channel">alpha channel</a>, allowing for transparency/translucency effects.</p>
<p>The 'application palette' made its debut with 8.5 — the application menu at the right side of the menu bar could be resized to show the active application's name, or 'torn off' into a palette of buttons. This palette could be customized many ways, by removing the window frame and changing the size and layout of the buttons. Apple did not provide a user interface for setting these options, instead making them available via AppleScript and <a class="mw-redirect" href="/wiki/Apple_Events" title="Apple Events">Apple Events</a> and relying on third parties to provide a user interface for the task. By setting it to display horizontally and turning off the window border the pallette could be configured to look and function much like the <a href="/wiki/Windows_95" title="Windows 95">Windows 95</a> task bar.</p>
<h3><span class="mw-headline" id="Mac_OS_8.5.1">Mac OS 8.5.1</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Mac_OS_8&amp;action=edit&amp;section=5" title="Edit section: Mac OS 8.5.1">edit</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>Mac OS 8.5.1, released December 7, 1998, was a minor update to Mac OS 8.5 that fixed a number of bugs that were causing crashes and data corruption.</p>
<h2><span class="mw-headline" id="Mac_OS_8.6">Mac OS 8.6</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Mac_OS_8&amp;action=edit&amp;section=6" title="Edit section: Mac OS 8.6">edit</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>Released May 10, 1999, Mac OS 8.6 added support to the <a href="/wiki/Mac_OS_nanokernel" title="Mac OS nanokernel">Mac OS nanokernel</a> to handle preemptive tasks via the Multiprocessing Services 2.x and later developer API. This update improved PowerBook battery life and added Sherlock 2.1. This free update for Mac users running 8.5 and 8.5.1 was faster and much more stable than either versions of 8.5.x and was also the first Mac OS to have the OS version displayed as part of the startup screen. However, there was still no process separation; the system still used cooperative multitasking between processes, and even a process that was Multiprocessing Services-aware still had a portion that ran in the <i>blue task</i>, a task that also ran all programs that are not aware of it, and the only task that could run 68k code.</p>
