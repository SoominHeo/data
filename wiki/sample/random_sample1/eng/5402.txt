<p><b>Mac OS X Snow Leopard</b> (version 10.6) is the seventh <a href="/wiki/Software_versioning" title="Software versioning">major release</a> of <a href="/wiki/MacOS" title="MacOS">Mac OS X (now named macOS)</a>, <a href="/wiki/Apple_Inc." title="Apple Inc.">Apple</a>'s desktop and server <a href="/wiki/Operating_system" title="Operating system">operating system</a> for <a href="/wiki/Macintosh" title="Macintosh">Macintosh</a> computers.</p>
<p>Snow Leopard was publicly unveiled on June 8, 2009 at the <a href="/wiki/Apple_Worldwide_Developers_Conference" title="Apple Worldwide Developers Conference">Apple Worldwide Developers Conference</a>. On August 28, 2009, it was released worldwide,<sup class="reference" id="cite_ref-releasepr_6-0"><a href="#cite_note-releasepr-6">[6]</a></sup> and was made available for purchase from Apple's website and its retail stores at the price of US$29 for a single-user license. As a result of the low price, initial sales of Snow Leopard were significantly higher than that of its predecessors.<sup class="reference" id="cite_ref-7"><a href="#cite_note-7">[7]</a></sup> The release of Snow Leopard came nearly two years after the introduction of <a href="/wiki/Mac_OS_X_Leopard" title="Mac OS X Leopard">Mac OS X Leopard</a>, the second longest time span between successive Mac OS X releases (the time span between <a href="/wiki/Mac_OS_X_Tiger" title="Mac OS X Tiger">Tiger</a> and Leopard was the longest).</p>
<p>Unlike those of previous versions of Mac OS X, the goals of Snow Leopard were improved performance, greater efficiency and the reduction of its overall <a href="/wiki/Memory_footprint" title="Memory footprint">memory footprint</a>. Addition of new end-user features was not a primary consideration: its name signified its goal to be a refinement of the previous OS X version, <a href="/wiki/Mac_OS_X_Leopard" title="Mac OS X Leopard">Leopard</a>.<sup class="reference" id="cite_ref-WWDC_2009_Keynote_8-0"><a href="#cite_note-WWDC_2009_Keynote-8">[8]</a></sup> Much of the software in Mac OS X was extensively rewritten for this release in order to take advantage fully of modern <a href="/wiki/Macintosh" title="Macintosh">Macintosh</a> hardware. New programming frameworks, such as <a href="/wiki/OpenCL" title="OpenCL">OpenCL</a>, were created, allowing software developers to use <a href="/wiki/Graphics_processing_unit" title="Graphics processing unit">graphics cards</a> in their applications. This is also the first Mac OS release since <a href="/wiki/System_7" title="System 7">System 7.1.1</a> that does not support Macs using <a href="/wiki/PowerPC" title="PowerPC">PowerPC</a> processors, as Apple now intends to focus on <a href="/wiki/Apple%E2%80%93Intel_architecture" title="Apple–Intel architecture">its current line of Intel-based products</a>.<sup class="reference" id="cite_ref-applepr_9-0"><a href="#cite_note-applepr-9">[9]</a></sup> As support for <a href="/wiki/Rosetta_(software)" title="Rosetta (software)">Rosetta</a> was dropped in <a class="mw-redirect" href="/wiki/OS_X_Lion" title="OS X Lion">OS X Lion</a>, Snow Leopard is the last version of Mac OS X that is able to run PowerPC-only applications.</p>
<p>Snow Leopard was succeeded by <a href="/wiki/Mac_OS_X_Lion" title="Mac OS X Lion">Mac OS X Lion</a> (version 10.7) on July 20, 2011.<sup class="reference" id="cite_ref-10"><a href="#cite_note-10">[10]</a></sup> Since then, Apple has continued to sell Snow Leopard from its online store for the benefit of users that require Snow Leopard in order to upgrade to later versions of OS X, which have all been distributed through the <a href="/wiki/Mac_App_Store" title="Mac App Store">Mac App Store</a> introduced in the Snow Leopard 10.6.6 update.<sup class="reference" id="cite_ref-11"><a href="#cite_note-11">[11]</a></sup></p>
<p>Snow Leopard is the last release of Mac OS X to support the 32-bit <a class="mw-redirect" href="/wiki/Intel_Core_Solo" title="Intel Core Solo">Intel Core Solo</a> and <a class="mw-redirect" href="/wiki/Intel_Core_Duo" title="Intel Core Duo">Intel Core Duo</a> <a href="/wiki/Central_processing_unit" title="Central processing unit">CPUs</a>.</p>

<h2><span class="mw-headline" id="System_requirements">System requirements</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Mac_OS_X_Snow_Leopard&amp;action=edit&amp;section=1" title="Edit section: System requirements">edit</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>Apple states the following basic Snow Leopard system requirements are:</p>
<ul>
<li>Mac computer with an <a href="/wiki/Intel" title="Intel">Intel</a> processor (<a href="/wiki/IA-32" title="IA-32">IA-32</a>). "<a href="/wiki/Yonah_(microprocessor)" title="Yonah (microprocessor)">Yonah</a>" processors such as <a class="mw-redirect" href="/wiki/Intel_Core_Solo" title="Intel Core Solo">Core Solo</a> and <a class="mw-redirect" href="/wiki/Intel_Core_Duo" title="Intel Core Duo">Core Duo</a> can run only 32-bit applications; later <a href="/wiki/X86-64" title="X86-64">x86-64</a> architecture processors such as <a class="mw-redirect" href="/wiki/Core_2_Duo" title="Core 2 Duo">Core 2 Duo</a>, Core i5 and i7 are also able to run 64-bit applications.</li>
<li>1 GB of <a class="mw-redirect" href="/wiki/RAM" title="RAM">RAM</a></li>
<li>5 GB of free disk space</li>
<li><a class="mw-redirect" href="/wiki/DVD_drive" title="DVD drive">DVD drive</a> (also accessible via Remote Disc) or external <a href="/wiki/USB" title="USB">USB</a> or <a class="mw-redirect" href="/wiki/FireWire" title="FireWire">FireWire</a> DVD drive for installation</li>
</ul>
<p><b>Additional requirements to use certain features:</b><sup class="reference" id="cite_ref-techspecs_12-0"><a href="#cite_note-techspecs-12">[12]</a></sup></p>
<ul>
<li><a href="/wiki/QuickTime" title="QuickTime">QuickTime</a> <a class="mw-redirect" href="/wiki/H.264" title="H.264">H.264</a> hardware acceleration support, requires an <a class="mw-redirect" href="/wiki/Nvidia_GeForce" title="Nvidia GeForce">Nvidia GeForce</a> <a class="mw-redirect" href="/wiki/GeForce_9_Series#9400M_G.5B31.5D" title="GeForce 9 Series">9400M</a>, <a class="mw-redirect" href="/wiki/NVIDIA_GeForce_GT_330M" title="NVIDIA GeForce GT 330M">320M</a>, or <a class="mw-redirect" href="/wiki/NVIDIA_GeForce_GT_330M" title="NVIDIA GeForce GT 330M">GT 330M</a> graphics card</li>
<li><a href="/wiki/OpenCL" title="OpenCL">OpenCL</a>, requires a supported <a href="/wiki/Nvidia" title="Nvidia">Nvidia</a> or <a class="mw-redirect" href="/wiki/ATI_(brand)" title="ATI (brand)">ATI</a> <a class="mw-redirect" href="/wiki/Graphics_card" title="Graphics card">graphics card</a><sup class="reference" id="cite_ref-techspecs_12-1"><a href="#cite_note-techspecs-12">[12]</a></sup></li>
</ul>
<p>Snow Leopard does not support <a href="/wiki/PowerPC" title="PowerPC">PowerPC</a>-based Macs (e.g., <a class="mw-redirect" href="/wiki/Power_Mac" title="Power Mac">Power Macs</a>, <a href="/wiki/PowerBook" title="PowerBook">PowerBooks</a>, <a href="/wiki/IBook" title="IBook">iBooks</a>, <a href="/wiki/IMac" title="IMac">iMacs</a> (G3-G5), all <a href="/wiki/EMac" title="EMac">eMacs</a>, plus pre-February 2006 <a class="mw-redirect" href="/wiki/Mac_mini#Mac_Mini_G4" title="Mac mini">Mac minis</a> and the <a href="/wiki/Power_Mac_G4_Cube" title="Power Mac G4 Cube">Power Mac G4 Cube</a>), although PowerPC applications are supported via <a href="/wiki/Rosetta_(software)" title="Rosetta (software)">Rosetta</a>, which is now an optional install.</p>
<h2><span class="mw-headline" id="License">License</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Mac_OS_X_Snow_Leopard&amp;action=edit&amp;section=2" title="Edit section: License">edit</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>Snow Leopard is available as an upgrade for Intel-based <a href="/wiki/Macintosh" title="Macintosh">Macintosh</a> computers. Single-user licenses and "family pack" licenses for up to five computers are available. For qualifying Mac computers bought after June 8, 2009, Apple offered a discounted price through their "up to date" program provided that customers' orders were faxed or postmarked by December 26, 2009. The standalone retail version of Snow Leopard is marketed as being restricted to users of <a href="/wiki/Mac_OS_X_Leopard" title="Mac OS X Leopard">Mac OS X Leopard</a>, while the recommended upgrade path from Apple for <a href="/wiki/Mac_OS_X_Tiger" title="Mac OS X Tiger">Mac OS X Tiger</a> is through the "Mac Box Set", which includes Mac OS X Snow Leopard and the current versions of <a href="/wiki/ILife" title="ILife">iLife</a> and <a href="/wiki/IWork" title="IWork">iWork</a>.</p>
<p>There are three licenses available.<sup class="reference" id="cite_ref-13"><a href="#cite_note-13">[13]</a></sup> These licenses differ in their requirements for pre-installed versions of Mac OS X:</p>
<ul>
<li><b>Leopard Upgrade</b>: requires that Mac OS X Leopard already be installed.</li>
</ul>
<blockquote>
<p>If you have purchased an Upgrade for Mac OS X Leopard license, then subject to the terms and conditions of this License, you are granted a limited non-exclusive license to install, use and run one (1) copy of the Apple Software on a single Apple-branded computer as long as that computer has a properly licensed copy of Mac OS X Leopard already installed on it.<sup class="reference" id="cite_ref-eula_14-0"><a href="#cite_note-eula-14">[14]</a></sup></p>
</blockquote>
<dl>
<dd>A "Family Pack Upgrade for Mac OS X Leopard" license is also mentioned as a subset of the Leopard Upgrade.</dd>
</dl>
<ul>
<li><b>Single Use</b>: places <i>no</i> restriction on which (if any) version of Mac OS X should already be installed. Used for the non-upgrade and Mac Box Set versions of Snow Leopard.</li>
</ul>
<blockquote>
<p>Subject to the terms and conditions of this License ... you are granted a limited non-exclusive license to install, use and run one (1) copy of the Apple Software on a single Apple-branded computer at a time.<sup class="reference" id="cite_ref-eula_14-1"><a href="#cite_note-eula-14">[14]</a></sup></p>
</blockquote>
<ul>
<li><b>Family Pack</b>: identical to the Single Use license in this respect.</li>
</ul>
<p>It is not entirely clear which license is offered with the retail version of Snow Leopard. As noted above, Apple's website advertised this version as an "upgrade from Mac OS X Leopard for $29" and suggest that others upgrade using the Mac Box Set, implying the stand-alone retail version to be a "Leopard Upgrade" license. On the other hand, some Apple press materials appear to indicate that this version is, in fact, the "Single Use" license:</p>
<blockquote>
<p>The Snow Leopard <i>single user license</i> will be available for a suggested retail price of $29 (US)<sup class="reference" id="cite_ref-june8th_15-0"><a href="#cite_note-june8th-15">[15]</a></sup> (emphasis added)</p>
</blockquote>
<p>However, even if the retail edition of Snow Leopard is in fact a "Leopard Upgrade", the company has acknowledged that there is no technical barrier in that edition preventing a direct upgrade from Mac OS X "Tiger".<sup class="reference" id="cite_ref-16"><a href="#cite_note-16">[16]</a></sup></p>
<p>The Leopard Upgrade license explicitly applies to the Up-To-Date Program<sup class="reference" id="cite_ref-17"><a href="#cite_note-17">[17]</a></sup> (US$9.95) for Macs bought between June 8 and December 26, 2009<sup class="reference" id="cite_ref-18"><a href="#cite_note-18">[18]</a></sup> and the installation discs provided through this program are clearly marked as upgrades unlike either of the retail editions.</p>
<h2><span class="mw-headline" id="New_or_changed_features">New or changed features</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Mac_OS_X_Snow_Leopard&amp;action=edit&amp;section=3" title="Edit section: New or changed features">edit</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>Mac OS X Snow Leopard is intended to be a release aimed to refine the existing feature set, expand the technological capabilities of the operating system, and improve application efficiency. Many of the changes involve how the system works in the background and are not intended to be seen by the user. For example, the Finder application was completely rewritten in the <a href="/wiki/Cocoa_(API)" title="Cocoa (API)">Cocoa application programming interface</a>. Despite significant changes in the software, users will experience almost no changes in the user interface. Snow Leopard includes the following changes:</p>
<ul>
<li>Mac App Store – An application store built in the image of the iOS App Store.Released on version 10.6.6.</li>
<li><b><a href="/wiki/Boot_Camp_(software)" title="Boot Camp (software)">Boot Camp</a></b> now allows <a href="/wiki/Microsoft_Windows" title="Microsoft Windows">Windows</a> partitions to read and copy files from <a class="mw-redirect" href="/wiki/HFS%2B" title="HFS+">HFS+</a> partitions. The new version also adds support for advanced features on Cinema Displays and a new command-line version of the Startup Disk Control Panel.</li>
<li>The <b><a href="/wiki/Finder_(software)" title="Finder (software)">Finder</a></b> has been completely rewritten in <a href="/wiki/Cocoa_(API)" title="Cocoa (API)">Cocoa</a> to take advantage of the new technologies introduced in Snow Leopard.</li>
<li>A much smaller OS footprint, taking up about 7 GB less space than <a href="/wiki/Mac_OS_X_Leopard" title="Mac OS X Leopard">Mac OS X Leopard</a>. Some of the recovered disk space (~250 MB) is because printer drivers are now downloaded or installed only as needed, rather than being pre-installed. The default install only contains those drivers needed for existing printers and a small subset of popular printers.<sup class="reference" id="cite_ref-refinements_19-0"><a href="#cite_note-refinements-19">[19]</a></sup></li>
<li><b><a href="/wiki/IChat" title="IChat">iChat</a></b> enhancements include greater resolution video chats in iChat Theater and lowered upload bandwidth requirements.</li>
<li><b><a href="/wiki/Microsoft_Exchange_Server" title="Microsoft Exchange Server">Microsoft Exchange</a></b> support is now integrated into the <a class="mw-redirect" href="/wiki/Mail_(OS_X)" title="Mail (OS X)">Mail</a>, <a href="/wiki/Address_Book_(application)" title="Address Book (application)">Address Book</a>, and <a class="mw-redirect" href="/wiki/ICal" title="ICal">iCal</a> applications. However, only <a href="/wiki/Microsoft_Exchange_Server" title="Microsoft Exchange Server">Microsoft Exchange 2007</a> is supported and customers using prior versions of Exchange must either upgrade or use <a href="/wiki/Microsoft_Entourage" title="Microsoft Entourage">Microsoft Entourage</a>.</li>
<li>Full <b><a href="/wiki/Multi-touch" title="Multi-touch">multi-touch</a></b> trackpad support has been added to notebooks prior to those introduced in October 2008.<sup class="reference" id="cite_ref-20"><a href="#cite_note-20">[20]</a></sup> While the original <a href="/wiki/MacBook_Air" title="MacBook Air">MacBook Air</a> and other early multi-touch trackpad enabled notebooks had support for some gestures, they were unable to use four-finger gestures. This limitation has now been removed in Snow Leopard.</li>
<li><b><a class="mw-redirect" href="/wiki/Preview_(software)" title="Preview (software)">Preview</a></b> can infer the structure of a paragraph in a PDF document.</li>
<li><b><a href="/wiki/QuickTime" title="QuickTime">QuickTime X</a></b>, the next version of <a href="/wiki/QuickTime" title="QuickTime">QuickTime</a> player and multimedia framework, has been completely rewritten into a full 64-bit Cocoa application and builds on the media technologies in Mac OS X, such as <a href="/wiki/Core_Audio" title="Core Audio">Core Audio</a>, <a href="/wiki/Core_Video" title="Core Video">Core Video</a>, and <a href="/wiki/Core_Animation" title="Core Animation">Core Animation</a>, to deliver playback. Apple has redesigned the QuickTime user interface to resemble the full-screen QuickTime view in prior versions, where the entire window displays the video. The titlebar and playback controls fade in and out as needed. QuickTime X also supports <a href="/wiki/Push_technology" title="Push technology">HTTP live streaming</a> and takes advantage of <a class="mw-redirect" href="/wiki/ColorSync" title="ColorSync">ColorSync</a> to provide high-quality color reproduction.<sup class="reference" id="cite_ref-applesnowleopard_21-0"><a href="#cite_note-applesnowleopard-21">[21]</a></sup> If Snow Leopard is installed on a Mac with an nVidia GeForce 9400M, 320M or GT 330M graphics card, QuickTime X will be able to use its video-decoding capabilities to reduce CPU load.</li>
<li><b><a class="mw-redirect" href="/wiki/Safari_4" title="Safari 4">Safari 4</a></b> features Top Sites, Cover Flow, VoiceOver, expanded standards support, and built-in crash resistance, which prevents browser crashes caused by plug-ins by running them in separate processes.<sup class="reference" id="cite_ref-22"><a href="#cite_note-22">[22]</a></sup> Safari 4 is bundled with Snow Leopard but does not require it, as it is available for free for Mac OS X Tiger and Leopard as well as Windows.</li>
<li><b><a class="mw-redirect" href="/wiki/Time_Machine_(Apple_software)" title="Time Machine (Apple software)">Time Machine</a></b> connection establishment and backups are now much faster.</li>
<li><b><a href="/wiki/VoiceOver" title="VoiceOver">VoiceOver</a></b> has also been greatly enhanced in Snow Leopard. Reading of web pages is improved with Auto Web Spots — areas of a page automatically designated for quick access. On newer Apple portables, trackpad gestures can be used to control VoiceOver functions, including the "rotor" gesture first seen in VoiceOver for the <a href="/wiki/IPhone" title="IPhone">iPhone 3GS</a>, allowing for the changing of certain VoiceOver navigation options by rotating fingers on the trackpad. <a class="mw-redirect" href="/wiki/Refreshable_Braille_display" title="Refreshable Braille display">Braille Display</a> support is also improved, with <a href="/wiki/Bluetooth" title="Bluetooth">Bluetooth</a> displays supported for the first time.<sup class="reference" id="cite_ref-23"><a href="#cite_note-23">[23]</a></sup></li>
</ul>
<p>The 10.6.6 update introduced support for the <a href="/wiki/Mac_App_Store" title="Mac App Store">Mac App Store</a>, Apple's digital distribution platform for OS X applications.<sup class="reference" id="cite_ref-24"><a href="#cite_note-24">[24]</a></sup></p>
<h3><span class="mw-headline" id="Refinements_to_the_user_interface">Refinements to the user interface</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Mac_OS_X_Snow_Leopard&amp;action=edit&amp;section=4" title="Edit section: Refinements to the user interface">edit</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>While the <a href="/wiki/Finder_(software)" title="Finder (software)">Finder</a> was completely rewritten in <a href="/wiki/Cocoa_(API)" title="Cocoa (API)">Cocoa</a>, it did not receive a major user interface overhaul. Instead, the interface has been modified in several areas to promote ease of use. These changes include:</p>
<ul>
<li>The "traffic light" titlebar controls are now slightly lighter in appearance and have less depth than they did in Mac OS X 10.5.</li>
<li><a class="mw-redirect" href="/wiki/Expos%C3%A9_(Mac_OS_X)" title="Exposé (Mac OS X)">Exposé</a> can now display windows for a single program by left clicking and holding its icon in the dock. Windows are arranged in a new grid pattern.</li>
<li>Contextual menus which come out of <a class="mw-redirect" href="/wiki/Dock_(Mac_OS_X)" title="Dock (Mac OS X)">Dock</a> icons now have more options and have a new look, with a semi-transparent charcoal background and white text.</li>
<li>An option has been added to the Finder preferences that allows the user to modify search behavior. The default setting can be selected to (1) search the entire computer, (2) search only the current folder from which the search was initiated, or (3) perform the search based on the previously used scope.</li>
<li>Dock <a class="mw-redirect" href="/wiki/Stacks_(software)" title="Stacks (software)">Stacks</a>, when viewed as a grid, allow viewing of a subfolder as a new stack, rather than launching a <a href="/wiki/Finder_(software)" title="Finder (software)">Finder</a> window, in a manner similar to "tunnelling". When viewed as grids or lists, scroll-bars are provided to navigate folders with more items than the current screen resolution will accommodate, as the program does not scale the icons to show as many as possible the way it did in OS X 10.5.<sup class="reference" id="cite_ref-25"><a href="#cite_note-25">[25]</a></sup></li>
<li>The default <a href="/wiki/Gamma_correction" title="Gamma correction">gamma</a> has been changed from 1.8 to 2.2 to better serve the color needs of digital content producers and consumers.<sup class="reference" id="cite_ref-refinements_19-1"><a href="#cite_note-refinements-19">[19]</a></sup></li>
<li>Windows can now be minimized directly onto their application's icon in the dock.<sup class="reference" id="cite_ref-26"><a href="#cite_note-26">[26]</a></sup></li>
<li>Faster <a class="mw-redirect" href="/wiki/PDF" title="PDF">PDF</a> and <a href="/wiki/JPEG" title="JPEG">JPEG</a> icon refreshes.<sup class="reference" id="cite_ref-27"><a href="#cite_note-27">[27]</a></sup></li>
<li>When searching for a network, the <a href="/wiki/AirPort" title="AirPort">AirPort</a> menu-bar icon animates until it finds a network and shows network strength of available networks in the drop down menu.</li>
<li><a class="mw-redirect" href="/wiki/Numerical_prefix" title="Numerical prefix">Prefixes</a> for <a href="/wiki/Byte" title="Byte">bytes</a> are now used in strictly decimal meaning (as opposed to their <a href="/wiki/Binary_prefix" title="Binary prefix">binary meaning</a>) when describing disk space, such that an indicated file size of 1 MB corresponds to 1 million bytes, as commonly used by hard disk manufacturers.<sup class="reference" id="cite_ref-28"><a href="#cite_note-28">[28]</a></sup></li>
<li>Snow Leopard shuts down and goes to sleep faster.<sup class="reference" id="cite_ref-29"><a href="#cite_note-29">[29]</a></sup></li>
</ul>

<h2><span class="mw-headline" id="Developer_technologies">Developer technologies</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Mac_OS_X_Snow_Leopard&amp;action=edit&amp;section=7" title="Edit section: Developer technologies">edit</a><span class="mw-editsection-bracket">]</span></span></h2>
<h3><span class="mw-headline" id="64-bit_architecture">64-bit architecture</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Mac_OS_X_Snow_Leopard&amp;action=edit&amp;section=8" title="Edit section: 64-bit architecture">edit</a><span class="mw-editsection-bracket">]</span></span></h3>
<p><a href="/wiki/Mac_OS_X_Tiger" title="Mac OS X Tiger">Mac OS X Tiger</a> added limited support for <a class="mw-redirect" href="/wiki/64-bit" title="64-bit">64-bit</a> applications on machines with 64-bit processors; <a href="/wiki/Mac_OS_X_Leopard" title="Mac OS X Leopard">Leopard</a> extended the support for 64-bit applications to include applications using most of Mac OS X's libraries and frameworks.</p>
<p>In Snow Leopard, most built-in applications have been rebuilt to use the 64-bit <a href="/wiki/X86-64" title="X86-64">x86-64</a> <a class="mw-redirect" href="/wiki/Instruction_set_architecture" title="Instruction set architecture">architecture</a> (excluding <a href="/wiki/ITunes" title="ITunes">iTunes</a>, <a href="/wiki/Front_Row_(software)" title="Front Row (software)">Front Row</a>, <a href="/wiki/Grapher" title="Grapher">Grapher</a> and <a class="mw-redirect" href="/wiki/DVD_Player_(software)" title="DVD Player (software)">DVD Player</a> applications).<sup class="reference" id="cite_ref-33"><a href="#cite_note-33">[33]</a></sup> They will run in <a href="/wiki/32-bit" title="32-bit">32-bit</a> mode on machines with 32-bit processors, and in 64-bit mode on machines with 64-bit processors.</p>
<p>In addition, the Mac OS X <a class="mw-redirect" href="/wiki/Kernel_(computing)" title="Kernel (computing)">kernel</a> has been rebuilt to run in 64-bit mode on some machines. On those machines, Snow Leopard supports up to 16 <a href="/wiki/Terabyte" title="Terabyte">terabytes</a> of RAM. Newer <a href="/wiki/Xserve" title="Xserve">Xserve</a> and <a href="/wiki/Mac_Pro" title="Mac Pro">Mac Pro</a> machines will run a 64-bit kernel by default; newer <a href="/wiki/IMac" title="IMac">iMac</a> and <a href="/wiki/MacBook_Pro" title="MacBook Pro">MacBook Pro</a> machines can run a 64-bit kernel, but will not do so by default.<sup class="reference" id="cite_ref-34"><a href="#cite_note-34">[34]</a></sup> Users wishing to use the 64-bit kernel on those machines must hold down the numbers 6 and 4 on the keyboard while booting to get the 64-bit kernel to load.<sup class="reference" id="cite_ref-35"><a href="#cite_note-35">[35]</a></sup><sup class="reference" id="cite_ref-cnet-64bit_36-0"><a href="#cite_note-cnet-64bit-36">[36]</a></sup> A change to the com.apple.Boot.plist will also enable users with compatible computers to permanently boot into 64-bit for those wishing to do so.</p>
<p>Stuart Harris, software product marketing manager at Apple Australia, said, "For the most part, everything that they experience on the Mac, from the 64-bit point of view, the applications, the operating system, is all going to be 64-bit, but that at this stage there were very few things, such as device drivers, that required 64-bit mode at the kernel level".<sup class="reference" id="cite_ref-cnet-64bit_36-1"><a href="#cite_note-cnet-64bit-36">[36]</a></sup></p>

<h3><span class="mw-headline" id="Grand_Central_Dispatch">Grand Central Dispatch</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Mac_OS_X_Snow_Leopard&amp;action=edit&amp;section=9" title="Edit section: Grand Central Dispatch">edit</a><span class="mw-editsection-bracket">]</span></span></h3>
<p><a href="/wiki/Grand_Central_Dispatch" title="Grand Central Dispatch">Grand Central Dispatch</a> uses the multiple processor cores now in every new Macintosh for more efficient performance. Due to the technical difficulties traditionally involved in making applications optimized for multicore CPUs, the majority of computer applications do not effectively use multiple processor cores.<sup class="reference" id="cite_ref-39"><a href="#cite_note-39">[39]</a></sup> As a result, processing power often goes unused. Grand Central Dispatch includes APIs to help programmers efficiently use these cores for parallel programming.</p>
<p>Grand Central Dispatch shifts thread handling focus to itself rather than leaving it to specific applications to distribute jobs evenly across cores and clears up unused memory created by inactive or old threads to achieve maximum performance. Apple is also releasing APIs for Grand Central Dispatch for developers to use in their applications and also to analyze specific blocks of code running on Grand Central Dispatch.<sup class="reference" id="cite_ref-40"><a href="#cite_note-40">[40]</a></sup></p>
<p>A new <a href="/wiki/C_(programming_language)" title="C (programming language)">C</a> and <a href="/wiki/Objective-C" title="Objective-C">Objective-C</a> language feature named "<a href="/wiki/Blocks_(C_language_extension)" title="Blocks (C language extension)">Blocks</a>" facilitates creation of code that will easily optimize to take advantage of Grand Central Dispatch.<sup class="reference" id="cite_ref-41"><a href="#cite_note-41">[41]</a></sup><sup class="reference" id="cite_ref-42"><a href="#cite_note-42">[42]</a></sup><sup class="reference" id="cite_ref-43"><a href="#cite_note-43">[43]</a></sup></p>
<h3><span class="mw-headline" id="OpenCL">OpenCL</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Mac_OS_X_Snow_Leopard&amp;action=edit&amp;section=10" title="Edit section: OpenCL">edit</a><span class="mw-editsection-bracket">]</span></span></h3>
<p><a href="/wiki/OpenCL" title="OpenCL">OpenCL</a> (Open Computing Language) addresses the power of <a href="/wiki/Graphics_processing_unit" title="Graphics processing unit">graphics processing units</a> (GPUs) to leverage them in any application, and not just for graphics-intensive applications like 3D games. OpenCL automatically optimizes for the kind of graphics processor in the Mac, adjusting itself to the available processing power. OpenCL provides consistent numeric precision and accuracy, fixing a problem that has hampered GPU-based programming in the past.<sup class="reference" id="cite_ref-44"><a href="#cite_note-44">[44]</a></sup></p>
<p>OpenCL includes a <a href="/wiki/C_(programming_language)" title="C (programming language)">C</a>-based <a href="/wiki/Programming_language" title="Programming language">programming language</a> with a structure that is already familiar to Mac OS X programmers, who can use <a href="/wiki/Xcode" title="Xcode">Xcode</a> developer tools to adapt their programs to work with OpenCL. Only the most process intensive parts of the application need to be written in OpenCL C without affecting the rest of the code. OpenCL is an open standard that has been supported by <a class="mw-redirect" href="/wiki/AMD" title="AMD">AMD</a>, <a href="/wiki/Intel" title="Intel">Intel</a>, and <a href="/wiki/Nvidia" title="Nvidia">Nvidia</a>; it is maintained by <a href="/wiki/Khronos_Group" title="Khronos Group">Khronos Group</a>.<sup class="reference" id="cite_ref-applesnowleopard_21-1"><a href="#cite_note-applesnowleopard-21">[21]</a></sup></p>
<p>It serves a similar purpose to <a href="/wiki/Nvidia" title="Nvidia">Nvidia</a>'s C for <a href="/wiki/CUDA" title="CUDA">CUDA</a> and <a href="/wiki/Microsoft" title="Microsoft">Microsoft</a>'s <a class="mw-redirect" href="/wiki/Direct3D_11" title="Direct3D 11">Direct3D 11</a> compute shaders.</p>
<p>It only works with the following Mac GPUs: NVIDIA GeForce 320M, GT 330M, 9400M, 9600M GT, 8600M GT, GT 120, GT 130, GTX 285, 8800 GT, 8800 GS, Quadro FX 4800, FX 5600 and ATI Radeon HD 4670, HD 4850, HD 4870, HD 5670, HD 5750, HD 5770, HD 5870, HD 6490M, HD 6750M, HD 6770M, HD 6970M.<sup class="reference" id="cite_ref-techspecs_12-2"><a href="#cite_note-techspecs-12">[12]</a></sup> If the system does not possess one of these compatible GPUs, OpenCL code will instead execute on the system's CPU.<sup class="reference" id="cite_ref-ArsReviewPage15_45-0"><a href="#cite_note-ArsReviewPage15-45">[45]</a></sup></p>
<h3><span class="mw-headline" id="CUPS">CUPS</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Mac_OS_X_Snow_Leopard&amp;action=edit&amp;section=11" title="Edit section: CUPS">edit</a><span class="mw-editsection-bracket">]</span></span></h3>
<p><a class="mw-redirect" href="/wiki/Common_Unix_Printing_System" title="Common Unix Printing System">CUPS</a> (the printing system used in many <a href="/wiki/Unix-like" title="Unix-like">Unix-like</a> operating systems) has been updated to version 1.4 which provides improved driver, networking, and <a href="/wiki/Kerberos_(protocol)" title="Kerberos (protocol)">Kerberos</a> support along with performance improvements. CUPS 1.4 is also the first implementation of the <a href="/wiki/Internet_Printing_Protocol" title="Internet Printing Protocol">Internet Printing Protocol</a> version 2.1.<sup class="reference" id="cite_ref-46"><a href="#cite_note-46">[46]</a></sup></p>
<h3><span class="mw-headline" id="Power_management">Power management</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Mac_OS_X_Snow_Leopard&amp;action=edit&amp;section=12" title="Edit section: Power management">edit</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>Power management has been improved, with implementation of a new <a class="mw-redirect" href="/wiki/Wake_on_demand" title="Wake on demand">wake on demand</a> feature supported on more recent Macintosh hardware.<sup class="reference" id="cite_ref-macworld-glenn_47-0"><a href="#cite_note-macworld-glenn-47">[47]</a></sup> Wake on demand takes advantage of the <a class="mw-redirect" href="/wiki/Sleep_proxy_service" title="Sleep proxy service">sleep proxy service</a> implemented in <a href="/wiki/AirPort" title="AirPort">AirPort</a> and <a class="mw-redirect" href="/wiki/Time_Capsule_(Apple)" title="Time Capsule (Apple)">Time Capsule</a> routers,<sup class="reference" id="cite_ref-apple-support-doc_48-0"><a href="#cite_note-apple-support-doc-48">[48]</a></sup> so that the computer can sleep while the router responds to <a class="mw-redirect" href="/wiki/MDNS" title="MDNS">mDNS</a> queries. Should the request require the host computer to <i>wake up</i>, the router sends the necessary special <a href="/wiki/Wake-on-LAN#Magic_packet" title="Wake-on-LAN">wake-up-packet</a><sup class="reference" id="cite_ref-49"><a href="#cite_note-49">[49]</a></sup> to the sleeping computer.</p>
<h2><span class="mw-headline" id="Security">Security</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Mac_OS_X_Snow_Leopard&amp;action=edit&amp;section=13" title="Edit section: Security">edit</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>Apple strengthened Mac OS X by implementing <a class="mw-redirect" href="/wiki/Stack_protection" title="Stack protection">stack protection</a>, and <a href="/wiki/Sandbox_(computer_security)" title="Sandbox (computer security)">sandboxing</a> more Mac OS X components such as the <a class="mw-redirect" href="/wiki/H.264" title="H.264">H.264</a> <a href="/wiki/Codec" title="Codec">decoder</a> in <a href="/wiki/QuickTime" title="QuickTime">QuickTime</a> and browser plug-ins as a separate process in <a class="mw-redirect" href="/wiki/Safari_(browser)" title="Safari (browser)">Safari</a>.<sup class="reference" id="cite_ref-50"><a href="#cite_note-50">[50]</a></sup> An anti-<a href="/wiki/Malware" title="Malware">malware</a> feature was also added to the system that alerts the user if malware is detected.<sup class="reference" id="cite_ref-51"><a href="#cite_note-51">[51]</a></sup> Mac OS X 10.6.8 added regular malware definition updates.<sup class="reference" id="cite_ref-52"><a href="#cite_note-52">[52]</a></sup></p>
<p>Computer security researcher <a href="/wiki/Charlie_Miller_(security_researcher)" title="Charlie Miller (security researcher)">Charlie Miller</a> claims that OS X Snow Leopard is more vulnerable to attack than <a href="/wiki/Microsoft_Windows" title="Microsoft Windows">Microsoft Windows</a> for lacking full <a href="/wiki/Address_space_layout_randomization" title="Address space layout randomization">address space layout randomization</a> (ASLR) since Mac OS X Leopard,<sup class="reference" id="cite_ref-53"><a href="#cite_note-53">[53]</a></sup> a technology that <a href="/wiki/Microsoft" title="Microsoft">Microsoft</a> started implementing in <a href="/wiki/Windows_Vista" title="Windows Vista">Windows Vista</a>.<sup class="reference" id="cite_ref-54"><a href="#cite_note-54">[54]</a></sup></p>
<p>The <a href="/wiki/Safari_(web_browser)" title="Safari (web browser)">Safari</a> web browser has received updates to version 6.0 in Lion and Mountain Lion, but not in Snow Leopard.<sup class="reference" id="cite_ref-55"><a href="#cite_note-55">[55]</a></sup></p>
<h2><span class="mw-headline" id="Compatibility">Compatibility</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Mac_OS_X_Snow_Leopard&amp;action=edit&amp;section=14" title="Edit section: Compatibility">edit</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>Snow Leopard breaks compatibility with several older versions of some applications, such as <a class="mw-redirect" href="/wiki/Parallels_Desktop" title="Parallels Desktop">Parallels Desktop</a> 3.0, versions of <a class="mw-redirect" href="/wiki/Aperture_(photography_software)" title="Aperture (photography software)">Aperture</a> before 2.1.1, and versions of <a href="/wiki/Keynote_(presentation_software)" title="Keynote (presentation software)">Keynote</a> before 2.0.2, among other software.<sup class="reference" id="cite_ref-56"><a href="#cite_note-56">[56]</a></sup> Apple has also published a list of applications with known compatibility issues with Snow Leopard.<sup class="reference" id="cite_ref-57"><a href="#cite_note-57">[57]</a></sup></p>
<p>Printer and scanner drivers used by previous versions of Mac OS X are not compatible with Snow Leopard and will be replaced during Snow Leopard installation. Since the initial release of Snow Leopard many manufacturers have provided compatible drivers that are available via Software Update.<sup class="reference" id="cite_ref-58"><a href="#cite_note-58">[58]</a></sup> If a native driver is not available Snow Leopard also includes CUPS and <a href="/wiki/Gutenprint" title="Gutenprint">Gutenprint</a> open source drivers that may provide limited functionality.</p>
<p>10.6.0 introduced a bug that frequently prevented <a href="/wiki/Domain_Name_System" title="Domain Name System">DNS</a> queries from returning <a class="mw-redirect" href="/wiki/IPv6_Address" title="IPv6 Address">IPv6 addresses</a>.<sup class="reference" id="cite_ref-59"><a href="#cite_note-59">[59]</a></sup><sup class="reference" id="cite_ref-60"><a href="#cite_note-60">[60]</a></sup><sup class="reference" id="cite_ref-61"><a href="#cite_note-61">[61]</a></sup> This was resolved in 10.6.8.<sup class="reference" id="cite_ref-62"><a href="#cite_note-62">[62]</a></sup></p>
<h2><span class="mw-headline" id="Reception">Reception</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Mac_OS_X_Snow_Leopard&amp;action=edit&amp;section=15" title="Edit section: Reception">edit</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>At the <a href="/wiki/Apple_Worldwide_Developers_Conference" title="Apple Worldwide Developers Conference">WWDC</a> in 2009, Apple stated that Snow Leopard features no new major visual changes.<sup class="reference" id="cite_ref-Siracusa.27sReview_63-0"><a href="#cite_note-Siracusa.27sReview-63">[63]</a></sup> Instead, the release focuses on refining the operating system to enable better performance.<sup class="reference" id="cite_ref-Siracusa.27sReview_63-1"><a href="#cite_note-Siracusa.27sReview-63">[63]</a></sup></p>
<p>OSNews reported that Mac OS X Snow Leopard was well received by critics.<sup class="reference" id="cite_ref-64"><a href="#cite_note-64">[64]</a></sup><sup class="reference" id="cite_ref-65"><a href="#cite_note-65">[65]</a></sup></p>
<p>Engadget reviewed Snow Leopard and pointed out that the price of Snow Leopard dropped from the $129 Apple charged for previous versions of Mac OS X to $29. Engadget's opinion was that this could be largely because most users would not see a noticeable change in the look and feel of the system.<sup class="reference" id="cite_ref-engadget_66-0"><a href="#cite_note-engadget-66">[66]</a></sup> However, most reviews commented on the large improvement in speed of the native Mac OS X applications <a href="/wiki/Finder_(software)" title="Finder (software)">Finder</a>, <a class="mw-redirect" href="/wiki/ICal" title="ICal">iCal</a>, <a class="mw-redirect" href="/wiki/Mail_(OS_X)" title="Mail (OS X)">Mail</a>, etc.<sup class="reference" id="cite_ref-engadget_66-1"><a href="#cite_note-engadget-66">[66]</a></sup></p>
<p><a href="/wiki/CNET" title="CNET">CNET</a> editors gave it 4 stars out of 5, stating "Intel Mac users will like Snow Leopard's smartly designed interface enhancements, and its <a href="/wiki/Microsoft_Exchange_Server" title="Microsoft Exchange Server">Exchange</a> support is a must-have (especially with <a href="/wiki/Microsoft_Outlook" title="Microsoft Outlook">Outlook</a> for Mac on the way). With a ton of technological improvements, Snow Leopard is worth the $29 upgrade fee."<sup class="reference" id="cite_ref-67"><a href="#cite_note-67">[67]</a></sup></p>
<p>On October 21, 2009, <i>SFGate</i> blogger Yobie Benjamin wrote that the "MacBook Pro that came preloaded with Snow Leopard kicks butt and is a screaming fast machine", but "when I tried to upgrade one of my 'older' MacBooks, it was a fricking disaster from hell". Apart from upgrading, Benjamin also tried a clean install. But he complained of slowness even after his clean install. He wrote, "I ended up downgrading back to OSX 10.5.8" then he concluded by writing, "I might try to do it again but it won't be till Apple releases at least 2 major fix updates. If you want to roll the dice and try, go ahead... your upgrade might work, however, random installs not working is not good for me. Lesson learned --- I'll wait."<sup class="reference" id="cite_ref-68"><a href="#cite_note-68">[68]</a></sup></p>
<p>The single-user upgrade and Family Pack units of Snow Leopard ranked 1 and 2 respectively on <a href="/wiki/Amazon.com" title="Amazon.com">Amazon.com</a>'s software bestseller charts when Apple announced it would release it within the week.<sup class="reference" id="cite_ref-69"><a href="#cite_note-69">[69]</a></sup></p>
<p>Testmac.com highlighted other unexpected improvements including the release of a new version of <a href="/wiki/Boot_Camp_(software)" title="Boot Camp (software)">Boot Camp</a>, version 3.0, a cleaner, popup software update process and screen and video recording in the new <a class="mw-redirect" href="/wiki/QuickTime_Player" title="QuickTime Player">QuickTime Player</a>.<sup class="reference" id="cite_ref-70"><a href="#cite_note-70">[70]</a></sup></p>
<p>The BBC reported that a bug in Mac OS X versions 10.6.0 and 10.6.1 which, in rare cases, caused loss of user account data after use of a previously existing guest account by users who had upgraded from a previous version of Mac OS X, received wide publicity.<sup class="reference" id="cite_ref-71"><a href="#cite_note-71">[71]</a></sup> The bug was fixed as of version 10.6.2.<sup class="reference" id="cite_ref-72"><a href="#cite_note-72">[72]</a></sup></p>
