<p><b>Internet Explorer for Mac</b> (also referred to as <b>Internet Explorer for Macintosh</b>, <b>Internet Explorer Macintosh Edition</b>, <b>Internet Explorer:mac</b> or <b>IE:mac</b>) is an inactive <a href="/wiki/Proprietary_software" title="Proprietary software">proprietary</a> <a href="/wiki/Web_browser" title="Web browser">web browser</a> developed by <a href="/wiki/Microsoft" title="Microsoft">Microsoft</a> for the <a href="/wiki/Macintosh" title="Macintosh">Macintosh</a> platform. Initial versions were developed from the same code base as <a href="/wiki/Internet_Explorer" title="Internet Explorer">Internet Explorer for Windows</a>. Later versions diverged, particularly with the release of version 5 which included the cutting edge,<sup class="reference" id="cite_ref-1"><a href="#cite_note-1">[1]</a></sup> <a class="mw-redirect" href="/wiki/Fault-tolerant" title="Fault-tolerant">fault-tolerant</a> and highly standards-compliant <a href="/wiki/Tasman_(layout_engine)" title="Tasman (layout engine)">Tasman</a> <a href="/wiki/Layout_engine" title="Layout engine">layout engine</a>.</p>
<p>As a result of the five-year agreement between <a class="mw-redirect" href="/wiki/Apple_Computer" title="Apple Computer">Apple</a> and Microsoft in 1997, it was the default browser on the <a href="/wiki/Classic_Mac_OS" title="Classic Mac OS">classic Mac OS</a> and <a href="/wiki/MacOS" title="MacOS">Mac OS X</a> from 1998 until it was superseded by Apple's own <a href="/wiki/Safari_(web_browser)" title="Safari (web browser)">Safari</a> web browser in 2003 with the release of <a href="/wiki/Mac_OS_X_Panther" title="Mac OS X Panther">Mac OS X 10.3</a> "Panther".</p>
<p>On June 13, 2003, Microsoft announced that it was ceasing further development of Internet Explorer for Mac and the final update was released on July 11, 2003. The browser was not included in the default installation of <a href="/wiki/Mac_OS_X_Tiger" title="Mac OS X Tiger">Mac OS X 10.4</a> "Tiger" which was released on April 29, 2005. Microsoft stopped releases for the product on December 31, 2005 and they removed the application from their Macintosh downloads site on January 31, 2006. Microsoft recommended "that users migrate to more recent web browsing technologies such as Apple's Safari."<sup class="reference" id="cite_ref-2"><a href="#cite_note-2">[2]</a></sup> An archived version of the download is available on <a href="/wiki/Softonic.com" title="Softonic.com">Softonic.com</a>, but only works on <a href="/wiki/Mac_OS_X_Snow_Leopard" title="Mac OS X Snow Leopard">Mac OS X 10.6</a> and below, due to the discontinuation of Rosetta.<sup class="reference" id="cite_ref-3"><a href="#cite_note-3">[3]</a></sup></p>

<h2><span class="mw-headline" id="History">History</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Internet_Explorer_for_Mac&amp;action=edit&amp;section=1" title="Edit section: History">edit</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>Versions of Internet Explorer for Macintosh were released starting with version 2 in 1996, to version 5 which received its last patch in 2003. IE versions for Mac typically lagged several months to a year behind Windows versions, but included some unique developments including its own layout engine <a href="/wiki/Tasman_(layout_engine)" title="Tasman (layout engine)">Tasman</a>.</p>
<h3><span class="mw-headline" id="Internet_Explorer_2.0_for_Macintosh">Internet Explorer 2.0 for Macintosh</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Internet_Explorer_for_Mac&amp;action=edit&amp;section=2" title="Edit section: Internet Explorer 2.0 for Macintosh">edit</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>The first version of Internet Explorer for the Macintosh operating system was a beta version of <i>Internet Explorer 2.0 for Macintosh</i>, released on January 23, 1996 as a free download from Microsoft's website. This first version was based on the <a href="/wiki/Mosaic_(web_browser)" title="Mosaic (web browser)">Spyglass Mosaic</a> web browser licensed from <a href="/wiki/Spyglass,_Inc." title="Spyglass, Inc.">Spyglass</a>. Available for both <a href="/wiki/Motorola_68000" title="Motorola 68000">68k</a> and <a href="/wiki/PowerPC" title="PowerPC">PPC</a> based Macs running <a class="mw-redirect" href="/wiki/System_7_(Macintosh)" title="System 7 (Macintosh)">System 7.0.1</a> or later, it supported the embedding of a number of multimedia formats into web pages, including <a href="/wiki/Audio_Video_Interleave" title="Audio Video Interleave">AVI</a> and <a href="/wiki/QuickTime" title="QuickTime">QuickTime</a> formatted video and <a class="mw-redirect" href="/wiki/AIFF" title="AIFF">AIFF</a> and <a href="/wiki/WAV" title="WAV">WAV</a> formatted audio. The final version was released three months later on April 23. Version 2.1 released in August of the same year, was mostly aimed at fixed bugs and improving stability, but also added a few features such as support for the <a href="/wiki/NPAPI" title="NPAPI">NPAPI</a> (the first version of Internet Explorer on any platform to do so) and support for <a href="/wiki/QuickTime_VR" title="QuickTime VR">QuickTime VR</a>. <a href="/wiki/AOL" title="AOL">AOL</a> 3.0 for Macintosh used the IE 2.1 rendering engine in its built-in web browser.</p>
<div class="hatnote" role="note">See also: <a href="/wiki/Internet_Explorer_2" title="Internet Explorer 2">Internet Explorer 2</a></div>
<h3><span class="mw-headline" id="Internet_Explorer_3.0_for_Macintosh">Internet Explorer 3.0 for Macintosh</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Internet_Explorer_for_Mac&amp;action=edit&amp;section=3" title="Edit section: Internet Explorer 3.0 for Macintosh">edit</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>On November 5, 1996 Microsoft announced the release of a beta version of <i>Internet Explorer version 3.0 for Macintosh</i>.<sup class="reference" id="cite_ref-4"><a href="#cite_note-4">[4]</a></sup> This release added support for <a href="/wiki/HTML" title="HTML">HTML</a> version 3.2, <a href="/wiki/Cascading_Style_Sheets" title="Cascading Style Sheets">Cascading Style Sheets</a>, <a href="/wiki/Java_applet" title="Java applet">Java applets</a> and <a href="/wiki/Component_Object_Model" title="Component Object Model">ActiveX</a> controls. The final version, made available on January 8, 1997, also added support for the <a class="mw-redirect" href="/wiki/Secure_Sockets_Layer" title="Secure Sockets Layer">SSL</a> and <a class="mw-redirect" href="/wiki/NTLM" title="NTLM">NTLM</a> security protocols and the <a href="/wiki/Platform_for_Internet_Content_Selection" title="Platform for Internet Content Selection">PICS</a> and <a class="new" href="/w/index.php?title=Recreational_Software_Advisory_Council_internet&amp;action=edit&amp;redlink=1" title="Recreational Software Advisory Council internet (page does not exist)">RSACi</a> rating systems that can be used to control access to websites based on content ratings.<sup class="reference" id="cite_ref-5"><a href="#cite_note-5">[5]</a></sup> A problem with an operating system extension used in the Mac OS, called CFM68K Runtime Enabler, led to a delay in the release of the version 3.0 for Macs based on the <a class="mw-redirect" href="/wiki/68k" title="68k">68k</a> line of processors. Four months later on May 14, Microsoft released version 3.01 which included a version for 68k-based machines.<sup class="reference" id="cite_ref-6"><a href="#cite_note-6">[6]</a></sup> This version also included features from the Windows version of Internet Explorer 4.0 such as <a class="mw-redirect" href="/wiki/AutoComplete" title="AutoComplete">AutoComplete</a> and Monitoring Favorites that notified users when sites in their Favorites list have been updated. It also included support for <a href="/wiki/JavaScript" title="JavaScript">JavaScript</a> and introduced a Download Manager and a <a href="/wiki/HTTP_cookie" title="HTTP cookie">Cookie</a> Manager.</p>
<div class="hatnote" role="note">See also: <a href="/wiki/Internet_Explorer_3" title="Internet Explorer 3">Internet Explorer 3</a></div>
<h3><span class="mw-headline" id="Internet_Explorer_4.0_for_Macintosh">Internet Explorer 4.0 for Macintosh</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Internet_Explorer_for_Mac&amp;action=edit&amp;section=4" title="Edit section: Internet Explorer 4.0 for Macintosh">edit</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>At the 1997 <a class="mw-redirect" href="/wiki/Macworld_Conference_%26_Expo" title="Macworld Conference &amp; Expo">Macworld Expo</a> in <a class="mw-redirect" href="/wiki/Boston,_Massachusetts" title="Boston, Massachusetts">Boston</a>, on August 6, <a href="/wiki/Steve_Jobs" title="Steve Jobs">Steve Jobs</a> and <a href="/wiki/Bill_Gates" title="Bill Gates">Bill Gates</a> announced a partnership between Microsoft and Apple. Amongst other things, Apple agreed to make Internet Explorer the default browser instead of <a href="/wiki/Netscape_Navigator" title="Netscape Navigator">Netscape Navigator</a>.<sup class="reference" id="cite_ref-7"><a href="#cite_note-7">[7]</a></sup></p>
<p>Five months later on January 6, 1998, at the Macworld Expo in <a class="mw-redirect" href="/wiki/San_Francisco,_California" title="San Francisco, California">San Francisco</a>, Microsoft announced the release of the final version of <i>Internet Explorer version 4.0 for Macintosh</i>. Version 4 included support for <a class="mw-redirect" href="/wiki/Offline_browsing" title="Offline browsing">offline browsing</a>, <a href="/wiki/Dynamic_HTML" title="Dynamic HTML">Dynamic HTML</a>, a new faster <a href="/wiki/Java_virtual_machine" title="Java virtual machine">Java virtual machine</a> and Security Zones that allow users or administrators to limit access to certain types of web content depending on which zone (for example Intranet or Internet) the content was coming from. The most publicized feature of Internet Explorer 4.0 was support for Microsoft's <a href="/wiki/Active_Channel" title="Active Channel">Active Channel</a> technology, which was intended to deliver regularly updated content that users could personally tailor to their interests. However Active Channel failed to reach a wide audience.</p>
<p>At the same event, Apple announced the release of <a href="/wiki/Mac_OS_8" title="Mac OS 8">Mac OS 8.1</a>. This was the first version of the Macintosh operating system to bundle Internet Explorer as its default browser per the agreement with Microsoft; however, version 4.0 was not ready in time to be included so version 3.01 was bundled on the CDs.</p>
<p>At the following year's San Francisco Macworld Expo on January 9, 1999, Microsoft announced the release of <i>Internet Explorer 4.5 Macintosh Edition</i>.<sup class="reference" id="cite_ref-8"><a href="#cite_note-8">[8]</a></sup> This new version, which dropped 68K processor support, introduced Form AutoFill, Print Preview, the Page Holder pane which let a user hold a page of links on one side of the screen that opened pages in the right hand and support for Mac OS technology like <a href="/wiki/Sherlock_(software)" title="Sherlock (software)">Sherlock</a>.</p>
<div class="hatnote" role="note">See also: <a href="/wiki/Internet_Explorer_4" title="Internet Explorer 4">Internet Explorer 4</a></div>
<h3><span class="mw-headline" id="Internet_Explorer_5_Macintosh_Edition">Internet Explorer 5 Macintosh Edition</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Internet_Explorer_for_Mac&amp;action=edit&amp;section=5" title="Edit section: Internet Explorer 5 Macintosh Edition">edit</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>Again a year later on January 5, 2000, Microsoft announced a new version of Internet Explorer at the San Francisco Macworld Expo, <i>Internet Explorer 5 Macintosh Edition</i> which was released two months later on March 27, 2000. The <a href="/wiki/Microsoft_Windows" title="Microsoft Windows">Windows</a> version of <a href="/wiki/Internet_Explorer_5" title="Internet Explorer 5">Internet Explorer 5</a> had been released a year earlier, but used the Trident layout engine. The Macintosh Edition introduced a new rendering engine called Tasman that was designed to be more compliant with emerging <a href="/wiki/World_Wide_Web_Consortium" title="World Wide Web Consortium">W3C</a> standards such as HTML 4.0, <a href="/wiki/Cascading_Style_Sheets" title="Cascading Style Sheets">CSS Level 1</a>, <a href="/wiki/Document_Object_Model" title="Document Object Model">DOM Level 1</a>, and <a href="/wiki/ECMAScript" title="ECMAScript">ECMAScript</a>. It also introduced a number of features that were later added to other browsers such as complete support for the <a href="/wiki/Portable_Network_Graphics" title="Portable Network Graphics">PNG</a> image standard (which previous versions did not support at all), <a class="mw-redirect" href="/wiki/Document_Type_Declaration" title="Document Type Declaration">DOCTYPE</a> switching, Text Zoom and <a href="/wiki/XML" title="XML">XML</a> source view. It also included an Auction Manager for tracking auctions in sites like <a href="/wiki/EBay" title="EBay">eBay</a> and an Internet Scrapbook to allow users to quickly and easily store and organize web content (for example an image or a piece of selected text). Preview releases of the browser included a feature called the MediaBar which integrated MP3 and <a href="/wiki/Internet_radio" title="Internet radio">internet radio</a> playback, but this feature was dropped from the final version. The initial release was just for Mac OS 8 and <a href="/wiki/Mac_OS_9" title="Mac OS 9">Mac OS 9</a>, however two months after that release on May 15 a <a href="/wiki/MacOS" title="MacOS">Mac OS X</a> version was released, bundled with the Mac OS X DP4 release handed out to developers at the 2000 <a class="mw-redirect" href="/wiki/Worldwide_Developers_Conference" title="Worldwide Developers Conference">Worldwide Developers Conference</a>. The <a href="/wiki/Mac_OS_X_Public_Beta" title="Mac OS X Public Beta">Mac OS X Public Beta</a> included another preview of the Mac OS X version of IE.<sup class="reference" id="cite_ref-9"><a href="#cite_note-9">[9]</a></sup><sup class="reference" id="cite_ref-10"><a href="#cite_note-10">[10]</a></sup> The release of <a href="/wiki/Mac_OS_X_10.0" title="Mac OS X 10.0">Mac OS X 10.0</a> on March 24, 2001 included yet another preview of the Mac OS X version of IE 5. This was updated later, and the release of <a class="mw-redirect" href="/wiki/Mac_OS_X_v10.1" title="Mac OS X v10.1">Mac OS X v10.1</a> on September 25, 2001 included the final version of Internet Explorer 5.1 for Mac OS X. IE 5.1 for Mac OS 8 and 9 was released on December 18, 2001.</p>
<p>According to Jorg Brown, one of the IE for Mac developers at Microsoft, after version 5 most of the team that produced IE for Mac were moved to another project. IE for Mac was relegated to something they were expected to work on in their "spare time".<sup class="reference" id="cite_ref-11"><a href="#cite_note-11">[11]</a></sup></p>
<p>On June 17, 2002, Microsoft announced the release of version 5.2 (the first Mac OS X-only release) which included a few performance and security fixes and support for Mac OS X features likes <a href="/wiki/Quartz_(graphics_layer)" title="Quartz (graphics layer)">Quartz</a> text smoothing.</p>
<p>During 2002, Microsoft reassigned developers to develop version 6 of Internet Explorer for Mac, intended to be used as the base for a new product. <a class="mw-redirect" href="/wiki/MSN_for_Mac_OS_X" title="MSN for Mac OS X">MSN for Mac OS X</a> would be a subscription-only browser that worked with the online <a href="/wiki/MSN" title="MSN">MSN</a> service, incorporate features like an address book, junk mail filters and an <a class="mw-redirect" href="/wiki/MSN_Messenger" title="MSN Messenger">MSN Messenger</a> client. However, after hearing that Apple had started development of their own browser, they canceled the standalone browser development and concentrated on the MSN browser, which was released on May 15, 2003.</p>
<p>On June 13, 2003, <a href="/wiki/PC_Pro" title="PC Pro">PC Pro</a> reported that <a class="mw-redirect" href="/wiki/Macintosh_Business_Unit" title="Macintosh Business Unit">Macintosh Business Unit</a> general manager <a class="new" href="/w/index.php?title=Roz_Ho&amp;action=edit&amp;redlink=1" title="Roz Ho (page does not exist)">Roz Ho</a> had confirmed that aside from updates to fix security problems, there would be no new versions of Internet Explorer from Microsoft. Three days later on June 16, Microsoft released the final version for Mac OS X (PPC), version 5.2.3 and a month later on July 11, they released the final version for Mac OS 8 and 9, version 5.1.7. The last versions of Internet Explorer for Mac OS X (PPC) had a distinguishing blue logo that was the base for the logo used in Internet Explorer 6 for Windows (the Windows one just had a lighter blue, and it was less 3-D).</p>
<h4><span class="mw-headline" id="Internet_Explorer_5_for_Mac_distinguishing_features">Internet Explorer 5 for Mac distinguishing features</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Internet_Explorer_for_Mac&amp;action=edit&amp;section=6" title="Edit section: Internet Explorer 5 for Mac distinguishing features">edit</a><span class="mw-editsection-bracket">]</span></span></h4>
<p>These are features found in Internet Explorer for Mac, which were not found in common contemporary browsers (with the possible exception of Internet Explorer for Windows). Some are still not features in many browsers.</p>
<ul>
<li>Support for annotative glosses to Japanese <a href="/wiki/Kanji" title="Kanji">kanji</a> and Chinese characters (see <a href="/wiki/Ruby_character" title="Ruby character">furigana</a>).</li>
<li>Scrapbook feature lets the user archive any page in its current state.</li>
<li>Auction Manager feature automatically tracks <a href="/wiki/EBay" title="EBay">eBay</a> auctions.</li>
<li>Although Internet Explorer for Mac did not have any <a href="/wiki/Portable_Network_Graphics" title="Portable Network Graphics">PNG</a> support at all until version 5.0 (a year or two after other major browsers), the PNG support added in that version was unusually robust, including transparency and color correction.</li>
<li>Support for matching web page colors using <a class="mw-redirect" href="/wiki/ColorSync" title="ColorSync">ColorSync</a>.</li>
<li>An option to change the browser color, to match the colors of the <a href="/wiki/IMac_G3" title="IMac G3">iMac G3</a>. The first builds had a choice of 9 colors, but later builds had 15.</li>
<li>The URL auto-complete window in the pre-Carbon versions was translucent and blurred the content behind to maintain readability of the suggested completions. This is the first time an app had a blurred translucent window and an effect similar to the look of iOS 7 years later.<sup class="noprint Inline-Template Template-Fact" style="white-space:nowrap;">[<i><a href="/wiki/Wikipedia:Citation_needed" title="Wikipedia:Citation needed"><span title="This claim needs references to reliable sources. (May 2014)">citation needed</span></a></i>]</sup> The later Carbon versions just used Mac OS X built-in window translucency without blurring.</li>
<li>Print Preview functionality allowing for adjustment of the font-size from within the preview pane.</li>
<li>Page Holder sidebar functionality allowing users to hold a rendered page in the sidebar (a links-only view was available too) and load clicked links in the main browser window. Much of this functionality was replaced with tabbed browsing in later browsers, but not the links-only view.</li>
<li>As with previous IE Mac versions, and in common with many other Macintosh internet software, the URL from which content was downloaded is added to the <a class="mw-redirect" href="/wiki/Macintosh_Finder" title="Macintosh Finder">Finder</a>'s Comment field (visible through Get Info).</li>
<li>Support for the <a href="/wiki/Internet_Config" title="Internet Config">Internet Config</a> system. Ironically, Macintosh versions of Internet Explorer were characterized by strong support for Macintosh-only technologies, generally better than <a href="/wiki/Netscape_Navigator" title="Netscape Navigator">Netscape Navigator</a>'s</li>
</ul>
<h4><span class="mw-headline" id="Other_features">Other features</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Internet_Explorer_for_Mac&amp;action=edit&amp;section=7" title="Edit section: Other features">edit</a><span class="mw-editsection-bracket">]</span></span></h4>
<p>These are features found in Internet Explorer for Mac and some other of its contemporaries.</p>
<ul>
<li>Auto-complete in the address bar responds to typing partial URLs or page titles, searches favorites and history</li>
<li>Go menu allows access to the persistent global browser history</li>
<li>Tasman rendering engine offers superior CSS support compared to <a href="/wiki/Trident_(layout_engine)" title="Trident (layout engine)">Trident</a> in other <a href="/wiki/Internet_Explorer_5" title="Internet Explorer 5">Internet Explorer 5</a> versions, and was not affected by the <a href="/wiki/Internet_Explorer_box_model_bug" title="Internet Explorer box model bug">Internet Explorer box model bug</a> (not fixed in Trident until <a href="/wiki/Internet_Explorer_6" title="Internet Explorer 6">IE 6</a>)</li>
<li>Text zoom allows the user to resize text on any page, regardless of how text size is specified</li>
</ul>
