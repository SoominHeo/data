<p>The <b>charm quark</b> or <b>c quark</b> (from its symbol, c) is the third most massive of all <a href="/wiki/Quark" title="Quark">quarks</a>, a type of <a href="/wiki/Elementary_particle" title="Elementary particle">elementary particle</a>. Charm quarks are found in <a href="/wiki/Hadron" title="Hadron">hadrons</a>, which are <a href="/wiki/Subatomic_particle" title="Subatomic particle">subatomic particles</a> made of quarks. Example of hadrons containing charm quarks include the <a class="mw-redirect" href="/wiki/J/%CF%88" title="J/ψ">J/ψ</a> meson (<span style="white-space:nowrap;"><span style="display:inline-block;margin-bottom:-0.3em;vertical-align:0.8em;line-height:1.0em;font-size:80%;text-align:right"><br/></span>J/ψ<span style="display:inline-block;margin-bottom:-0.3em;vertical-align:0.8em;line-height:1.0em;font-size:80%;text-align:left"><br/></span></span>), <a href="/wiki/D_meson" title="D meson">D mesons</a> (<span style="white-space:nowrap;"><span style="display:inline-block;margin-bottom:-0.3em;vertical-align:0.8em;line-height:1.0em;font-size:80%;text-align:right"><br/></span>D<span style="display:inline-block;margin-bottom:-0.3em;vertical-align:0.8em;line-height:1.0em;font-size:80%;text-align:left"><br/></span></span>), <a class="mw-redirect" href="/wiki/Charmed_Sigma_baryon" title="Charmed Sigma baryon">charmed Sigma baryons</a> (<span style="white-space:nowrap;"><span style="display:inline-block;margin-bottom:-0.3em;vertical-align:0.8em;line-height:1.0em;font-size:80%;text-align:right"><br/></span>Σ<span style="display:inline-block;margin-bottom:-0.3em;vertical-align:-0.4em;line-height:1.0em;font-size:80%;text-align:left"><br/>
c</span></span>), and other charmed particles.</p>
<p>It, along with the <a href="/wiki/Strange_quark" title="Strange quark">strange quark</a> is part of the <a href="/wiki/Generation_(particle_physics)" title="Generation (particle physics)">second generation</a> of matter, and has an <a href="/wiki/Electric_charge" title="Electric charge">electric charge</a> of +<span class="sfrac nowrap" style="display:inline-block; vertical-align:-0.5em; font-size:85%; text-align:center;"><span style="display:block; line-height:1em; margin:0 0.1em;">2</span><span class="visualhide">/</span><span style="display:block; line-height:1em; margin:0 0.1em; border-top:1px solid;">3</span></span> <a href="/wiki/Elementary_charge" title="Elementary charge"><i>e</i></a> and a <a href="/wiki/Quark#Mass" title="Quark">bare mass</a> of <span class="nowrap"><span class="sortkey" style="display:none">7000129000000000000♠</span>1.29<span style="margin-left:0.3em;"><span style="display:inline-block;margin-bottom:-0.3em;vertical-align:-0.4em;line-height:1.2em;font-size:85%;text-align:right">+0.05<br/>
−0.11</span></span> <a href="/wiki/Electronvolt#Mass" title="Electronvolt">GeV/<i>c</i><sup>2</sup></a></span>.<sup class="reference" id="cite_ref-PDG2011_1-1"><a href="#cite_note-PDG2011-1">[1]</a></sup> Like all <a href="/wiki/Quark" title="Quark">quarks</a>, the charm quark is an <a href="/wiki/Elementary_particle" title="Elementary particle">elementary</a> <a href="/wiki/Fermion" title="Fermion">fermion</a> with <a href="/wiki/Spin_(physics)" title="Spin (physics)">spin</a> <a class="mw-redirect" href="/wiki/Spin-1/2" title="Spin-1/2"><span class="sfrac nowrap" style="display:inline-block; vertical-align:-0.5em; font-size:85%; text-align:center;"><span style="display:block; line-height:1em; margin:0 0.1em;">1</span><span class="visualhide">/</span><span style="display:block; line-height:1em; margin:0 0.1em; border-top:1px solid;">2</span></span></a>, and experiences all four <a href="/wiki/Fundamental_interaction" title="Fundamental interaction">fundamental interactions</a>: <a class="mw-redirect" href="/wiki/Gravitation" title="Gravitation">gravitation</a>, <a href="/wiki/Electromagnetism" title="Electromagnetism">electromagnetism</a>, <a href="/wiki/Weak_interaction" title="Weak interaction">weak interactions</a>, and <a href="/wiki/Strong_interaction" title="Strong interaction">strong interactions</a>. The <a href="/wiki/Antiparticle" title="Antiparticle">antiparticle</a> of the charm quark is the <b>charm antiquark</b> (sometimes called <i>anticharm quark</i> or simply <i>anticharm</i>), which differs from it only in that some of its properties have <a href="/wiki/Additive_inverse" title="Additive inverse">equal magnitude but opposite sign</a>.</p>
<p>The existence of a fourth quark had been speculated by a number of authors around 1964 (for instance by <a href="/wiki/James_Bjorken" title="James Bjorken">James Bjorken</a> and <a href="/wiki/Sheldon_Lee_Glashow" title="Sheldon Lee Glashow">Sheldon Glashow</a><sup class="reference" id="cite_ref-4"><a href="#cite_note-4">[4]</a></sup>), but its prediction is usually credited to <a href="/wiki/Sheldon_Lee_Glashow" title="Sheldon Lee Glashow">Sheldon Glashow</a>, <a href="/wiki/John_Iliopoulos" title="John Iliopoulos">John Iliopoulos</a> and <a href="/wiki/Luciano_Maiani" title="Luciano Maiani">Luciano Maiani</a> in 1970 (see <a href="/wiki/GIM_mechanism" title="GIM mechanism">GIM mechanism</a>).<sup class="reference" id="cite_ref-5"><a href="#cite_note-5">[5]</a></sup> The first charmed particle (a particle containing a charm quark) to be discovered was the <a class="mw-redirect" href="/wiki/J/%CF%88_meson" title="J/ψ meson">J/ψ meson</a>. It was discovered by a team at the <a class="mw-redirect" href="/wiki/Stanford_Linear_Accelerator_Center" title="Stanford Linear Accelerator Center">Stanford Linear Accelerator Center</a> (SLAC), led by <a href="/wiki/Burton_Richter" title="Burton Richter">Burton Richter</a>,<sup class="reference" id="cite_ref-6"><a href="#cite_note-6">[6]</a></sup> and one at the <a href="/wiki/Brookhaven_National_Laboratory" title="Brookhaven National Laboratory">Brookhaven National Laboratory</a> (BNL), led by <a href="/wiki/Samuel_C._C._Ting" title="Samuel C. C. Ting">Samuel Ting</a>.<sup class="reference" id="cite_ref-7"><a href="#cite_note-7">[7]</a></sup></p>
<p>The 1974 discovery of the <span style="white-space:nowrap;"><span style="display:inline-block;margin-bottom:-0.3em;vertical-align:0.8em;line-height:1.0em;font-size:80%;text-align:right"><br/></span>J/ψ<span style="display:inline-block;margin-bottom:-0.3em;vertical-align:0.8em;line-height:1.0em;font-size:80%;text-align:left"><br/></span></span> (and thus the charm quark) ushered in a series of breakthroughs which are collectively known as the <i><a class="mw-redirect" href="/wiki/November_Revolution_(physics)" title="November Revolution (physics)">November Revolution</a></i>.</p>

<h2><span class="mw-headline" id="Hadrons_containing_charm_quarks">Hadrons containing charm quarks</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Charm_quark&amp;action=edit&amp;section=1" title="Edit section: Hadrons containing charm quarks">edit</a><span class="mw-editsection-bracket">]</span></span></h2>
<div class="hatnote" role="note">Main articles: <a href="/wiki/List_of_baryons" title="List of baryons">List of baryons</a> and <a href="/wiki/List_of_mesons" title="List of mesons">list of mesons</a></div>
<p>Some of the <a href="/wiki/Hadron" title="Hadron">hadrons</a> containing charm quarks include:</p>
<ul>
<li><a href="/wiki/D_meson" title="D meson">D mesons</a> contain a charm quark (or its <a href="/wiki/Antiparticle" title="Antiparticle">antiparticle</a>) and an <a href="/wiki/Up_quark" title="Up quark">up</a> or <a href="/wiki/Down_quark" title="Down quark">down quark</a>.</li>
<li><span style="white-space:nowrap;"><span style="display:inline-block;margin-bottom:-0.3em;vertical-align:0.8em;line-height:1.0em;font-size:80%;text-align:right"><br/></span>D<span style="display:inline-block;margin-bottom:-0.3em;vertical-align:-0.4em;line-height:1.0em;font-size:80%;text-align:left"><br/>
s</span></span> mesons contain a charm quark and a <a href="/wiki/Strange_quark" title="Strange quark">strange quark</a>.</li>
<li>There are many <a class="mw-redirect" href="/wiki/Charmonium" title="Charmonium">charmonium</a> states, for example the <span style="white-space:nowrap;"><span style="display:inline-block;margin-bottom:-0.3em;vertical-align:0.8em;line-height:1.0em;font-size:80%;text-align:right"><br/></span>J/ψ<span style="display:inline-block;margin-bottom:-0.3em;vertical-align:0.8em;line-height:1.0em;font-size:80%;text-align:left"><br/></span></span> particle. These consist of a charm quark and its antiparticle.</li>
<li><a class="mw-redirect" href="/wiki/Charmed_baryon" title="Charmed baryon">Charmed baryons</a> have been observed, and are named in analogy with strange baryons (e.g. <span style="white-space:nowrap;"><span style="display:inline-block;margin-bottom:-0.3em;vertical-align:0.8em;line-height:1.0em;font-size:80%;text-align:right"><br/></span>Λ<span style="display:inline-block;margin-bottom:-0.3em;vertical-align:-0.4em;line-height:1.0em;font-size:80%;text-align:left">+<br/>
c</span></span>).</li>
</ul>
