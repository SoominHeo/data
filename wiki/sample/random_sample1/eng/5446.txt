<p>The <b>megabit</b> is a multiple of the unit <a href="/wiki/Bit" title="Bit">bit</a> for digital <a href="/wiki/Information" title="Information">information</a>. The prefix <a href="/wiki/Mega-" title="Mega-">mega</a> (symbol M) is defined in the <a href="/wiki/International_System_of_Units" title="International System of Units">International System of Units</a> (SI) as a multiplier of 10<sup>6</sup> (1 million),<sup class="reference" id="cite_ref-1"><a href="#cite_note-1">[1]</a></sup> and therefore</p>
<dl>
<dd>1 megabit = <span style="white-space:nowrap">10<sup>6</sup><span style="margin-left:0.25em">bits</span></span> = <span style="white-space:nowrap">1<span style="margin-left:0.25em">000</span><span style="margin-left:0.25em">000</span><span style="margin-left:0.25em">bits</span></span> = 1000 <a href="/wiki/Kilobit" title="Kilobit">kilobits</a>.</dd>
</dl>
<p>The megabit has the unit symbol <b>Mb</b> or <b>Mbit</b>.</p>
<p>The megabit is closely related to the <a href="/wiki/Mebibit" title="Mebibit">mebibit</a>, a unit multiple derived from the <a href="/wiki/Binary_prefix" title="Binary prefix">binary prefix</a> <a class="mw-redirect" href="/wiki/Mebi-" title="Mebi-">mebi</a> (symbol Mi) of the same order of magnitude,<sup class="reference" id="cite_ref-2"><a href="#cite_note-2">[2]</a></sup> which is equal to <span style="white-space:nowrap">2<sup>20</sup><span style="margin-left:0.25em">bits</span></span> = <span style="white-space:nowrap">1<span style="margin-left:0.25em">048</span><span style="margin-left:0.25em">576</span><span style="margin-left:0.25em">bits</span></span>, or approximately 5% larger than the megabit. Despite the definitions of these new prefixes for binary-based quantities of storage by international standards organizations, memory semiconductor chips are still marketed using the <a href="/wiki/Metric_prefix" title="Metric prefix">metric prefix</a> names to designate binary multiples.</p>
<p>Using the common <a href="/wiki/Byte" title="Byte">byte</a> size of eight bits and the standardized metric definition of megabit and kilobyte, 1 megabit is equal to 125 <a href="/wiki/Kilobyte" title="Kilobyte">kilobytes</a> (kB) or approximately 122 <a href="/wiki/Kibibyte" title="Kibibyte">kibibytes</a> (KiB).</p>
<p>The megabit is widely used when referring to data transfer rates of computer networks or <a class="mw-redirect" href="/wiki/Telecommunications" title="Telecommunications">telecommunications</a> systems. Network transfer rates and download speeds often use the megabit as the amount transferred per time unit, e.g., a 100 <a class="mw-redirect" href="/wiki/Mbit/s" title="Mbit/s">Mbit/s</a> (<a class="mw-redirect" href="/wiki/Megabit_per_second" title="Megabit per second">megabit per second</a>) Fast-<a href="/wiki/Ethernet" title="Ethernet">Ethernet</a> connection, or a 10 Mbit/s Internet access service, whereas the sizes of data units (files) transferred over these networks are often measured in megabytes. To achieve a transfer rate of one megabyte per second one needs a network connection with a transfer rate of eight megabits per second.</p>
<h2><span class="mw-headline" id="Usage">Usage</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Megabit&amp;action=edit&amp;section=1" title="Edit section: Usage">edit</a><span class="mw-editsection-bracket">]</span></span></h2>
<ul>
<li>In <a class="mw-redirect" href="/wiki/Telecommunications" title="Telecommunications">telecommunications</a>, the use of the SI definition of the unit is the standard.</li>
<li>In the semiconductor industry, it is still common practice to designate <a href="/wiki/Random-access_memory" title="Random-access memory">random-access memory</a> (RAM), <a href="/wiki/Read-only_memory" title="Read-only memory">read-only memory</a> (ROM) in a binary interpretation of the metric prefixes, such as the megabit, so that one megabit represents <span style="white-space:nowrap">2<sup>20</sup><span style="margin-left:0.25em">bits</span></span>=<span style="white-space:nowrap">1<span style="margin-left:0.25em">048</span><span style="margin-left:0.25em">576</span><span style="margin-left:0.25em">bits</span></span>.<sup class="reference" id="cite_ref-3"><a href="#cite_note-3">[3]</a></sup><sup class="reference" id="cite_ref-4"><a href="#cite_note-4">[4]</a></sup><sup class="reference" id="cite_ref-5"><a href="#cite_note-5">[5]</a></sup> For example, a single discrete DDR3 chip specified at 512 Mb invariably contains 2<sup>29</sup> bits = <span style="white-space:nowrap">536<span style="margin-left:0.25em">870</span><span style="margin-left:0.25em">912</span><span style="margin-left:0.25em">bits</span></span> = 512 Mibit of storage,<sup class="reference" id="cite_ref-6"><a href="#cite_note-6">[6]</a></sup> or <span style="white-space:nowrap">67<span style="margin-left:0.25em">108</span><span style="margin-left:0.25em">864</span><span style="margin-left:0.25em">8-bit bytes</span></span>, variously referred to as either 64 <a href="/wiki/Mebibyte" title="Mebibyte">mebibytes</a> or 64 (binary) <a href="/wiki/Megabyte" title="Megabyte">megabytes</a>.</li>
<li>During the <a class="mw-redirect" href="/wiki/History_of_video_game_consoles_(fourth_generation)" title="History of video game consoles (fourth generation)">16-bit game console era</a>, the megabit was a commonly used measure of the size (<a href="/wiki/Computer_data_storage" title="Computer data storage">computer data storage</a> capacity) of game cartridges. This size represented one <a href="/wiki/Mebibit" title="Mebibit">mebibit</a> (Mibit). The vast majority of <a href="/wiki/Super_Nintendo_Entertainment_System" title="Super Nintendo Entertainment System">SNES</a> and <a class="mw-redirect" href="/wiki/Mega_Drive" title="Mega Drive">Mega Drive</a> (Genesis) games were produced on 8 <i>megabit</i> cartridges, although other sizes such as 4, 12, 16, 24, 32, and 48 megabit cartridges appeared. This usage continued on the <a href="/wiki/Nintendo_64" title="Nintendo 64">Nintendo 64</a>, with cartridge sizes ranging between 32 and 512 megabits.</li>
</ul>
