<p><b>Mail</b> (also known as <b>Apple Mail</b> or <b>Mail.app</b>) is an <a href="/wiki/Email_client" title="Email client">email client</a> included with the <a href="/wiki/Operating_system" title="Operating system">operating systems</a> <a href="/wiki/MacOS" title="MacOS">macOS</a>, <a href="/wiki/IOS" title="IOS">iOS</a> and <a href="/wiki/WatchOS" title="WatchOS">watchOS</a> by <a href="/wiki/Apple_Inc." title="Apple Inc.">Apple Inc.</a> Originally developed by <a href="/wiki/NeXT" title="NeXT">NeXT</a> as <a href="/wiki/NeXTMail" title="NeXTMail">NeXTMail</a>, a part of their <a href="/wiki/NeXTSTEP" title="NeXTSTEP">NeXTSTEP</a> operating system, it eventually became Mail after apple's acquisition of NeXT.</p>
<p>The current version of Mail utilizes <a href="/wiki/Simple_Mail_Transfer_Protocol" title="Simple Mail Transfer Protocol">SMTP</a> for message sending, <a href="/wiki/Post_Office_Protocol" title="Post Office Protocol">POP3</a>, <a href="/wiki/Microsoft_Exchange_Server" title="Microsoft Exchange Server">Exchange</a> and <a href="/wiki/Internet_Message_Access_Protocol" title="Internet Message Access Protocol">IMAP</a> for message retrieval and <a href="/wiki/S/MIME" title="S/MIME">S/MIME</a> for end-to-end message encryption.<sup class="reference" id="cite_ref-2"><a href="#cite_note-2">[2]</a></sup><sup class="reference" id="cite_ref-3"><a href="#cite_note-3">[3]</a></sup> It is also preconfigured to work with popular <a href="/wiki/Email_hosting_service" title="Email hosting service">email providers</a>, such as <a href="/wiki/Yahoo!_Mail" title="Yahoo! Mail">Yahoo! Mail</a>, <a href="/wiki/AOL_Mail" title="AOL Mail">AOL Mail</a>, <a href="/wiki/Gmail" title="Gmail">Gmail</a>, <a href="/wiki/Outlook.com" title="Outlook.com">Outlook</a> and <a href="/wiki/ICloud" title="ICloud">iCloud</a> (formerly <a href="/wiki/MobileMe" title="MobileMe">MobileMe</a>) and it supports <a href="/wiki/Microsoft_Exchange_Server" title="Microsoft Exchange Server">Exchange</a>.<sup class="reference" id="cite_ref-4"><a href="#cite_note-4">[4]</a></sup> <a href="/wiki/IOS" title="IOS">iOS</a> features a mobile version of Mail with added <a href="/wiki/Exchange_ActiveSync" title="Exchange ActiveSync">Exchange ActiveSync</a> support, though it notoriously misses the functionality of attaching files to reply emails until the release of <a href="/wiki/IOS_9" title="IOS 9">iOS 9</a>.<sup class="reference" id="cite_ref-5"><a href="#cite_note-5">[5]</a></sup> The iOS version of Mail is the most popular email client in the world by market share.<sup class="reference" id="cite_ref-6"><a href="#cite_note-6">[6]</a></sup><sup class="noprint Inline-Template" style="white-space:nowrap;">[<i><a href="/wiki/Wikipedia:Identifying_reliable_sources" title="Wikipedia:Identifying reliable sources"><span title="The material near this tag may rely on an unreliable source. (June 2015)">unreliable source?</span></a></i>]</sup></p>

<h2><span class="mw-headline" id="History">History</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Mail_(Apple)&amp;action=edit&amp;section=1" title="Edit section: History">edit</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>Mail first shipped as NeXTMail, the email application for the NeXTSTEP operating system. NeXTMail was innovative for its time,<sup class="noprint Inline-Template" style="margin-left:0.1em; white-space:nowrap;">[<i><a href="/wiki/Wikipedia:Manual_of_Style/Words_to_watch#Unsupported_attributions" title="Wikipedia:Manual of Style/Words to watch"><span title="The material near this tag may use weasel words or too-vague attribution. (June 2015)">according to whom?</span></a></i>]</sup> supporting rich text formatting with images and voice messaging. It also supported <a href="/wiki/MIME" title="MIME">MIME</a> emails, along with plain text to allow for backwards compatibility. The default message found in the inbox when the user first opened NeXTMail included a voice recording of <a href="/wiki/Steve_Jobs" title="Steve Jobs">Steve Jobs</a>.</p>
<p>When Apple began to adapt NeXTSTEP to become <a href="/wiki/MacOS" title="MacOS">Mac OS X</a>, both the operating system and the application went through various stages as it was developed. In a beta version (codenamed "<a href="/wiki/Rhapsody_(operating_system)" title="Rhapsody (operating system)">Rhapsody</a>") and various other early pre-releases of Mac OS X, Mail was known as MailViewer. However, with the third developer release of Mac OS X, the application had returned to being known simply as Mail.</p>
<h3><span class="mw-headline" id="First_release">First release</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Mail_(Apple)&amp;action=edit&amp;section=2" title="Edit section: First release">edit</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>Included in all versions of macOS up to and including <a href="/wiki/Mac_OS_X_Panther" title="Mac OS X Panther">Mac OS X Panther</a>, Mail was integrated with other Apple applications such as <a href="/wiki/Address_Book_(application)" title="Address Book (application)">Address Book</a>, <a href="/wiki/IChat" title="IChat">iChat</a>, and <a href="/wiki/Calendar_(Apple)" title="Calendar (Apple)">iCal</a>. Some of its features that remain in the most recent version of Mail include rules for mailboxes, <a href="/wiki/Email_spam" title="Email spam">junk mail</a> <a href="/wiki/Email_filtering" title="Email filtering">filtering</a> and multiple account management.</p>
<h3><span class="mw-headline" id="Mac_OS_X_Tiger">Mac OS X Tiger</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Mail_(Apple)&amp;action=edit&amp;section=3" title="Edit section: Mac OS X Tiger">edit</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>In <a href="/wiki/Mac_OS_X_Tiger" title="Mac OS X Tiger">Mac OS X Tiger</a>, Mail was updated to version 2. It included a proprietary single-message-per-file format (with the <a href="/wiki/Filename_extension" title="Filename extension">filename extension</a> <b>.emlx</b>) in order to permit indexing by <a href="/wiki/Spotlight_(software)" title="Spotlight (software)">Spotlight</a>.</p>
<p>Additional features were:</p>
<ul>
<li>"Smart mailboxes" that used Spotlight technology to sort mail into folders.</li>
<li>The ability to flag messages with a low, normal or high priority and to use these priorities in mailbox rules and smart mailboxes.</li>
<li>Tools for resizing photos before they are sent to avoid oversized email attachments.</li>
<li>The ability to view emailed pictures as a full-screen slideshow.</li>
<li>Parental controls to specify who is allowed to send email to children.</li>
<li><a href="/wiki/HTML" title="HTML">HTML</a> message composition.</li>
</ul>
<p>The new version also changed the UI for the buttons in the toolbar. Whereas previous buttons had free-standing defined shapes, the new buttons featured shapes within a <a href="/wiki/Lozenge" title="Lozenge">lozenge</a>-shaped capsule.<sup class="reference" id="cite_ref-7"><a href="#cite_note-7">[7]</a></sup> According to many users, and even Apple's own <a href="/wiki/Human_interface_guidelines" title="Human interface guidelines">human interface guidelines</a> at the time, this was worse for usability.<sup class="reference" id="cite_ref-8"><a href="#cite_note-8">[8]</a></sup> An open-source third-party application that reverted the icons to their former shapes was available. Nevertheless, Apple updated their guidelines to include capsule-shaped buttons, and the new UI persisted.<sup class="reference" id="cite_ref-9"><a href="#cite_note-9">[9]</a></sup></p>
<h3><span class="mw-headline" id="Mac_OS_X_Leopard">Mac OS X Leopard</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Mail_(Apple)&amp;action=edit&amp;section=4" title="Edit section: Mac OS X Leopard">edit</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>In <a href="/wiki/Mac_OS_X_Leopard" title="Mac OS X Leopard">Mac OS X Leopard</a>, Mail included personalized stationery, handled in standard HTML format. In addition, version 3 offered notes and to-dos (which could be synced with <a href="/wiki/Calendar_(Apple)" title="Calendar (Apple)">iCal</a>) as well as a built-in <a href="/wiki/RSS" title="RSS">RSS</a> reader. Version 3 also introduced <a href="/wiki/IMAP_IDLE" title="IMAP IDLE">IMAP IDLE</a> support for account inboxes.</p>
<h3><span class="mw-headline" id="Mac_OS_X_Snow_Leopard">Mac OS X Snow Leopard</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Mail_(Apple)&amp;action=edit&amp;section=5" title="Edit section: Mac OS X Snow Leopard">edit</a><span class="mw-editsection-bracket">]</span></span></h3>
<p><a href="/wiki/Mac_OS_X_Snow_Leopard" title="Mac OS X Snow Leopard">Mac OS X Snow Leopard</a> brought <a href="/wiki/Microsoft_Exchange_Server#Exchange_Server_2007" title="Microsoft Exchange Server">Microsoft Exchange Server 2007</a> support. The <a href="/wiki/Calendar_(Apple)" title="Calendar (Apple)">iCal</a> and <a href="/wiki/Address_Book_(application)" title="Address Book (application)">Address Book</a> applications bundled with OS X 10.6 also supported Microsoft Exchange Server.</p>
<h3><span class="mw-headline" id="Mac_OS_X_Lion">Mac OS X Lion</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Mail_(Apple)&amp;action=edit&amp;section=6" title="Edit section: Mac OS X Lion">edit</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>In <a href="/wiki/Mac_OS_X_Lion" title="Mac OS X Lion">Mac OS X Lion</a>, Mail featured a redesigned <a href="/wiki/IPad" title="IPad">iPad</a>-like user interface with full-screen capabilities, an updated message search interface, support for <a href="/wiki/Microsoft_Exchange_Server#Exchange_Server_2010" title="Microsoft Exchange Server">Microsoft Exchange Server 2010</a> and <a href="/wiki/Yahoo!_Mail" title="Yahoo! Mail">Yahoo! Mail</a> (via IMAP). Also added was the capability to group messages by subject in a similar fashion to Mail on <a href="/wiki/IOS_4" title="IOS 4">iOS 4</a>. The bounce function, where unwanted emails could be <a href="/wiki/Bounce_message" title="Bounce message">bounced back</a> to the sender, was dropped in this version, as was support for Exchange push email.</p>
<h3><span class="mw-headline" id="OS_X_Mountain_Lion">OS X Mountain Lion</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Mail_(Apple)&amp;action=edit&amp;section=7" title="Edit section: OS X Mountain Lion">edit</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>In <a href="/wiki/OS_X_Mountain_Lion" title="OS X Mountain Lion">OS X Mountain Lion</a>, Mail received VIP tagging, <a href="/wiki/Safari_(web_browser)" title="Safari (web browser)">Safari</a>-style inline search for words within an email message, the ability to sync with <a href="/wiki/ICloud" title="ICloud">iCloud</a> and new sharing features. Notes were split off into a stand-alone application in this version, called <a href="/wiki/Notes_(application)" title="Notes (application)">Notes</a>. The RSS reader and to-dos were discontinued.</p>
<h3><span class="mw-headline" id="OS_X_Mavericks">OS X Mavericks</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Mail_(Apple)&amp;action=edit&amp;section=8" title="Edit section: OS X Mavericks">edit</a><span class="mw-editsection-bracket">]</span></span></h3>
<p><a href="/wiki/OS_X_Mavericks" title="OS X Mavericks">OS X Mavericks</a> threw out the capability of displaying the plain-text version of MIME multipart/alternative messages, and the option of doing so by default. Users were stuck viewing the HTML or rich-text version of multipart/alternative messages.</p>
<h3><span class="mw-headline" id="OS_X_Yosemite">OS X Yosemite</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Mail_(Apple)&amp;action=edit&amp;section=9" title="Edit section: OS X Yosemite">edit</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>In <a href="/wiki/OS_X_Yosemite" title="OS X Yosemite">OS X Yosemite</a>, Mail supported new features such as: Markup (inline annotation of PDF or image files) and Mail Drop (automatically uploads attachments to iCloud, and sends a link in the message instead of the whole file).</p>
<h3><span class="mw-headline" id="OS_X_El_Capitan">OS X El Capitan</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Mail_(Apple)&amp;action=edit&amp;section=10" title="Edit section: OS X El Capitan">edit</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>In <a href="/wiki/OS_X_El_Capitan" title="OS X El Capitan">OS X El Capitan</a>, a filter was added to the message list to filter by various options such as Unread, Flagged, or messages with attachments. The conversation display was also redesigned and various disk space saving optimizations were implemented. Streaming notification support for Exchange accounts was also added.</p>
