<p><b>Theta Centauri</b> (<b>θ Centauri</b>, abbreviated <b>Theta Cen</b>, <b>θ Cen</b>), also named <b>Menkent</b>,<sup class="reference" id="cite_ref-IAU-CSN_10-0"><a href="#cite_note-IAU-CSN-10">[10]</a></sup> is a <a href="/wiki/Star" title="Star">star</a> in the southern <a href="/wiki/Constellation" title="Constellation">constellation</a> of <a href="/wiki/Centaurus" title="Centaurus">Centaurus</a>, the <a href="/wiki/Centaur" title="Centaur">centaur</a>. With an <a href="/wiki/Apparent_magnitude" title="Apparent magnitude">apparent visual magnitude</a> of +2.06,<sup class="reference" id="cite_ref-clpl4_99_2-3"><a href="#cite_note-clpl4_99-2">[2]</a></sup> it is the <a href="/wiki/List_of_stars_in_Centaurus" title="List of stars in Centaurus">fourth-brightest member of the constellation</a>. Close enough so that its distance can be measured using the <a href="/wiki/Parallax" title="Parallax">parallax</a> technique, it is 58.8 <a href="/wiki/Light-year" title="Light-year">light-years</a> (18.0 <a href="/wiki/Parsec" title="Parsec">parsecs</a>) from the <a href="/wiki/Sun" title="Sun">Sun</a>.<sup class="reference" id="cite_ref-aaa474_2_653_1-5"><a href="#cite_note-aaa474_2_653-1">[1]</a></sup></p>
<p>This is an <a href="/wiki/Stellar_evolution" title="Stellar evolution">evolved</a> <a href="/wiki/Giant_star" title="Giant star">giant star</a> with a <a href="/wiki/Stellar_classification" title="Stellar classification">stellar classification</a> of K0 III.<sup class="reference" id="cite_ref-aj132_1_161_3-1"><a href="#cite_note-aj132_1_161-3">[3]</a></sup> The <a href="/wiki/Interferometry" title="Interferometry">interferometry</a>-measured <a href="/wiki/Angular_diameter" title="Angular diameter">angular diameter</a> of this star, after correcting for <a href="/wiki/Limb_darkening" title="Limb darkening">limb darkening</a>, is <span class="nowrap">5.46 ± 0.06 <a href="/wiki/Minute_and_second_of_arc#Astronomy" title="Minute and second of arc">milliarcseconds</a></span>,<sup class="reference" id="cite_ref-aaa431_773_11-0"><a href="#cite_note-aaa431_773-11">[11]</a></sup> which, at its estimated distance, equates to a physical radius of about 10.6 times the <a href="/wiki/Solar_radius" title="Solar radius">radius of the Sun</a>.<sup class="reference" id="cite_ref-lang2006_6-1"><a href="#cite_note-lang2006-6">[6]</a></sup> The <a href="/wiki/Stellar_atmosphere" title="Stellar atmosphere">outer envelope</a> has an <a href="/wiki/Effective_temperature" title="Effective temperature">effective temperature</a> of 4,980 <a href="/wiki/Kelvin" title="Kelvin">K</a>,<sup class="reference" id="cite_ref-mnras256_3_535_8-3"><a href="#cite_note-mnras256_3_535-8">[8]</a></sup> giving it the orange-hued glow of a cool, <a class="mw-redirect" href="/wiki/K-type_star" title="K-type star">K-type star</a>.<sup class="reference" id="cite_ref-csiro_12-0"><a href="#cite_note-csiro-12">[12]</a></sup> Soft X-ray emission has been detected from this star, which has an estimated X-ray luminosity of <span class="nowrap">1.4 × 10<sup>27</sup> erg s<sup>−1</sup></span>.<sup class="reference" id="cite_ref-aaa335_591_13-0"><a href="#cite_note-aaa335_591-13">[13]</a></sup></p>
<h2><span class="mw-headline" id="Nomenclature">Nomenclature</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Theta_Centauri&amp;action=edit&amp;section=1" title="Edit section: Nomenclature">edit</a><span class="mw-editsection-bracket">]</span></span></h2>
<p><i>θ Centauri</i> (<a href="/wiki/Latinisation_of_names" title="Latinisation of names">Latinised</a> to <i>Theta Centauri</i>) is the star's <a href="/wiki/Bayer_designation" title="Bayer designation">Bayer designation</a>.</p>
<p>The traditional name <i>Menkent</i> is possibly an abbreviation of the <a href="/wiki/Arabic" title="Arabic">Arabic</a> <i>ألمنكب ألقنتوس</i> <i>al mankib al-qanturis</i> meaning "shoulder of the Centaur". In 2016, the <a href="/wiki/International_Astronomical_Union" title="International Astronomical Union">International Astronomical Union</a> organized a <a href="/wiki/IAU_Working_Group_on_Star_Names" title="IAU Working Group on Star Names">Working Group on Star Names</a> (WGSN)<sup class="reference" id="cite_ref-WGSN_14-0"><a href="#cite_note-WGSN-14">[14]</a></sup> to catalog and standardize proper names for stars. The WGSN approved the name <i>Menkent</i> for this star on 21 August 2016 and it is now so entered in the IAU Catalog of Star Names.<sup class="reference" id="cite_ref-IAU-CSN_10-1"><a href="#cite_note-IAU-CSN-10">[10]</a></sup></p>
<p>In <a href="/wiki/Chinese_language" title="Chinese language">Chinese</a>, <span lang="zh" xml:lang="zh">庫樓</span> (<span lang="zh-Latn" xml:lang="zh-Latn">Kù Lóu</span>), meaning <i><a href="/wiki/Horn_(Chinese_constellation)" title="Horn (Chinese constellation)">Arsenal</a></i>, refers to an asterism consisting of Theta Centauri, <a href="/wiki/Zeta_Centauri" title="Zeta Centauri">Zeta Centauri</a>, <a href="/wiki/Eta_Centauri" title="Eta Centauri">Eta Centauri</a>, <a href="/wiki/2_Centauri" title="2 Centauri">2 Centauri</a>, <a href="/wiki/HD_117440" title="HD 117440">HD 117440</a>, <a class="mw-redirect" href="/wiki/Xi%C2%B9_Centauri" title="Xi¹ Centauri">Xi¹ Centauri</a>, <a href="/wiki/Gamma_Centauri" title="Gamma Centauri">Gamma Centauri</a>, <a href="/wiki/Tau_Centauri" title="Tau Centauri">Tau Centauri</a>, <a href="/wiki/D_Centauri" title="D Centauri">D Centauri</a> and <a href="/wiki/Sigma_Centauri" title="Sigma Centauri">Sigma Centauri</a>.<sup class="reference" id="cite_ref-15"><a href="#cite_note-15">[15]</a></sup> Consequently, Theta Centauri itself is known as <span lang="zh" xml:lang="zh">庫樓三</span> (<span lang="zh-Latn" xml:lang="zh-Latn">Kù Lóu sān</span>, English: the Third Star of Arsenal.)<sup class="reference" id="cite_ref-16"><a href="#cite_note-16">[16]</a></sup></p>
