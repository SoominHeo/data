<p>The <b>Motorola RAZR i</b> (Motorola XT890) is a <a href="/wiki/Smartphone" title="Smartphone">smartphone</a> designed by <a href="/wiki/Motorola_Mobility" title="Motorola Mobility">Motorola Mobility</a>. It was officially announced on 18 September 2012 in London, UK.<sup class="reference" id="cite_ref-engadget1_2-0"><a href="#cite_note-engadget1-2">[2]</a></sup></p>
<p>The Motorola RAZR i is the first smartphone by Motorola to feature an <a href="/wiki/Intel" title="Intel">Intel</a> <a href="/wiki/Atom_(system_on_chip)" title="Atom (system on chip)">Atom "Medfield"</a> processor/SOC of the <a href="/wiki/IA-32" title="IA-32">x86/IA-32 architecture</a>, running a x86 port of the <a href="/wiki/Android_(operating_system)" title="Android (operating system)">Android</a> operating system. It is also the first smartphone with a CPU clocked at 2,0 GHz in its stock configuration.<sup class="reference" id="cite_ref-3"><a href="#cite_note-3">[3]</a></sup></p>
<p>A special China-only version of the smartphone, the RAZR i MT788 was announced in November, 2012.<sup class="reference" id="cite_ref-engadget3_4-0"><a href="#cite_note-engadget3-4">[4]</a></sup> Externally, it looks identical to its sibling <a href="/wiki/Droid_Razr_M" title="Droid Razr M">Motorola RAZR M</a> launched two weeks earlier in the US,<sup class="reference" id="cite_ref-engadget2_5-0"><a href="#cite_note-engadget2-5">[5]</a></sup> with which it also shares many of its specs.</p>
<p>The RAZR i (XT890) is an almost identical edition except it has an <a href="/wiki/Intel_Atom" title="Intel Atom">Intel Atom</a> Z2460 processor. This means it has more processing power and imaging power; however, it lacks <a href="/wiki/LTE_(telecommunication)" title="LTE (telecommunication)">LTE</a> support.<sup class="reference" id="cite_ref-6"><a href="#cite_note-6">[6]</a></sup><sup class="reference" id="cite_ref-7"><a href="#cite_note-7">[7]</a></sup><sup class="reference" id="cite_ref-8"><a href="#cite_note-8">[8]</a></sup> It does not fully support many high-end apps and games designed for <a href="/wiki/ARM_architecture" title="ARM architecture">ARM</a> processors until these apps or games are ported and recompiled for <a href="/wiki/Intel" title="Intel">Intel</a> compatibility.<sup class="reference" id="cite_ref-9"><a href="#cite_note-9">[9]</a></sup> However, because this is one of the only (if not the only) device using a x86 processor instead of an ARM processor, developers might not bother porting their applications unless there is more of a market for them.</p>
<p>The GPU in the RAZR i's <a href="/wiki/Intel_Atom" title="Intel Atom">Intel Atom</a> Z2460 processor is a <a class="mw-redirect" href="/wiki/PowerVR_SGXMP" title="PowerVR SGXMP">PowerVR SGX540</a>, a faster-clocked version of the same GPU as in the <a class="mw-redirect" href="/wiki/Samsung_Galaxy_Nexus" title="Samsung Galaxy Nexus">Samsung Galaxy Nexus</a> and many older smartphones such as the original <a href="/wiki/Samsung_Galaxy_S" title="Samsung Galaxy S">Samsung Galaxy S</a>. While the RAZR i's raw CPU processing power can be excellent for tasks such as web browsing, its GPU can be considerably slower than the RAZR M, which uses a <a class="mw-redirect" href="/wiki/Snapdragon_S4" title="Snapdragon S4">Snapdragon S4</a> CPU featuring an <a href="/wiki/Adreno" title="Adreno">Adreno</a> 225 GPU.<sup class="reference" id="cite_ref-10"><a href="#cite_note-10">[10]</a></sup> This may be alleviated somewhat by the Atom Z2460 being capable of providing much greater memory bandwidth, which has traditionally been a severe performance-constraining factor in ARM-based chips.<sup class="reference" id="cite_ref-11"><a href="#cite_note-11">[11]</a></sup></p>
