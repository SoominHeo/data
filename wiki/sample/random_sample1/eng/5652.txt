<p><b>Motoblur</b> (often stylized as <b>MOTOBLUR</b>) was an <a href="/wiki/Android_(operating_system)" title="Android (operating system)">Android</a> UI replacement and <a href="/wiki/Push_technology" title="Push technology">push</a>-based service focused on social networking, developed by <a href="/wiki/Motorola" title="Motorola">Motorola</a>. It aimed at functional similarity to <a href="/wiki/Palm,_Inc." title="Palm, Inc.">Palm's</a> <a href="/wiki/Palm_Pre#Synergy" title="Palm Pre">Synergy</a>, including such features as Remote Wipe. Motoblur included a variety of <a class="mw-redirect" href="/wiki/GUI_widget" title="GUI widget">widgets</a> which combined various social networking portals such as <a href="/wiki/Facebook" title="Facebook">Facebook</a>, <a class="mw-redirect" href="/wiki/MySpace" title="MySpace">MySpace</a>, and <a href="/wiki/Twitter" title="Twitter">Twitter</a> as well as other services (news or weather reports) all in one place. It also combined multiple email accounts and contact communication sources into singular notification views, being the first handset software to do so. Feeds and data were regularly pushed to these widgets.<sup class="reference" id="cite_ref-1"><a href="#cite_note-1">[1]</a></sup> Motoblur in its final inception was on these devices: <a href="/wiki/Motorola_Photon" title="Motorola Photon">Electrify/Photon 4G</a>, <a href="/wiki/Motorola_Atrix_4G" title="Motorola Atrix 4G">Atrix 4G</a>, Atrix HD, <a class="mw-redirect" href="/wiki/Motorola_CLIQ" title="Motorola CLIQ">CLIQ/DEXT</a>, <a href="/wiki/Motorola_Backflip" title="Motorola Backflip">Backflip</a>, <a href="/wiki/Motorola_Devour" title="Motorola Devour">Devour</a>, <a href="/wiki/Motorola_Flipout" title="Motorola Flipout">Flipout</a>, <a href="/wiki/Motorola_Charm" title="Motorola Charm">Charm</a>, <a href="/wiki/Motorola_Spice" title="Motorola Spice">Spice</a>, <a class="mw-redirect" href="/wiki/Motorola_Droid_Pro" title="Motorola Droid Pro">Droid Pro</a>, Filpside, <a class="mw-redirect" href="/wiki/Motorola_DEFY" title="Motorola DEFY">DEFY</a>, DEFY+, Bravo, <a class="mw-redirect" href="/wiki/Motorola_Droid_X" title="Motorola Droid X">Droid X</a>, <a class="mw-redirect" href="/wiki/Motorola_Droid_3" title="Motorola Droid 3">Droid 3</a>, <a class="mw-redirect" href="/wiki/Motorola_Droid_2" title="Motorola Droid 2">Droid 2</a>, <a class="mw-redirect" href="/wiki/Motorola_Droid_Bionic" title="Motorola Droid Bionic">Droid Bionic</a>, and <a class="mw-redirect" href="/wiki/Droid_RAZR" title="Droid RAZR">Droid RAZR</a>. The version found on the Droid X, Droid Pro, Droid 2, Droid Bionic, Droid 3, Electrify/Photon 4G, and DEFY was intended to be less intrusive than previous versions.<sup class="reference" id="cite_ref-incdroid_2-0"><a href="#cite_note-incdroid-2">[2]</a></sup></p>
<p>First generation Motoblur-based phones required a new user to create a Motoblur account, denying access to the main screen until the account was established. User account information was stored on Motorola's redundant servers for access from web browsers and future phones. Newer devices allowed users to defer Blur services until a later registration and had more filtering options and better battery management to optimize the user experience.<sup class="reference" id="cite_ref-3"><a href="#cite_note-3">[3]</a></sup></p>
<p>In late 2010 Motorola announced that Motoblur would not be their development focus in the future, as Android made custom skins largely redundant.<sup class="reference" id="cite_ref-4"><a href="#cite_note-4">[4]</a></sup> The <a href="/wiki/Motorola_Atrix" title="Motorola Atrix">Atrix</a><sup class="plainlinks noprint Inline-Template" style="vertical-align:text-top;white-space:nowrap;">[<i><a class="external text" href="http://dispenser.homenet.org/~dispenser/cgi-bin/dab_solver.py?page=Motoblur&amp;editintro=Template:Disambiguation_needed/editintro&amp;client=Template:Dn" rel="nofollow"><span title="Link needs disambiguation. (September 2013)">disambiguation needed</span></a></i>]</sup>, <a class="mw-redirect" href="/wiki/Motorola_Droid_2" title="Motorola Droid 2">Droid 2</a> and <a class="mw-redirect" href="/wiki/Motorola_Droid_X" title="Motorola Droid X">Droid X</a> do feature the UI skin.<sup class="reference" id="cite_ref-incdroid_2-1"><a href="#cite_note-incdroid-2">[2]</a></sup></p>
<p>In 2011 <a class="mw-redirect" href="/wiki/PC_World_(magazine)" title="PC World (magazine)">PC World</a> criticised Motoblur for poor performance.<sup class="reference" id="cite_ref-5"><a href="#cite_note-5">[5]</a></sup></p>
