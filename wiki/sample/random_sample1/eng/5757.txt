<p><b>Mukaihara Station</b> <span style="font-weight: normal">(<span class="t_nihongo_kanji" lang="ja" xml:lang="ja">向原駅</span><span class="t_nihongo_comma" style="display:none">,</span> <i>Mukaihara-eki</i><sup class="t_nihongo_help noprint"><a href="/wiki/Help:Installing_Japanese_character_sets" title="Help:Installing Japanese character sets"><span class="t_nihongo_icon" style="color: #00e; font: bold 80% sans-serif; text-decoration: none; padding: 0 .1em;">?</span></a></sup>)</span> is a <a href="/wiki/West_Japan_Railway_Company" title="West Japan Railway Company">JR West</a> <a href="/wiki/Geibi_Line" title="Geibi Line">Geibi Line</a> <a href="/wiki/Train_station" title="Train station">station</a> located in Saka, Mukaihara-chō, <a href="/wiki/Akitakata,_Hiroshima" title="Akitakata, Hiroshima">Akitakata</a>, <a href="/wiki/Hiroshima_Prefecture" title="Hiroshima Prefecture">Hiroshima Prefecture</a>, <a href="/wiki/Japan" title="Japan">Japan</a>.</p>

<h2><span class="mw-headline" id="History">History</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Mukaihara_Station&amp;action=edit&amp;section=1" title="Edit section: History">edit</a><span class="mw-editsection-bracket">]</span></span></h2>
<ul>
<li>1915-04-28: Mukaihara Station opens</li>
<li>March 1986: The current station building was completed</li>
<li>1987-04-01: <a class="mw-redirect" href="/wiki/Japan_National_Railways" title="Japan National Railways">Japan National Railways</a> is privatized, and Mukaihara Station becomes a JR West station</li>
</ul>
<h2><span class="mw-headline" id="Station_building_and_platforms">Station building and platforms</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Mukaihara_Station&amp;action=edit&amp;section=2" title="Edit section: Station building and platforms">edit</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>Mukaihara Station features one raised <a href="/wiki/Island_platform" title="Island platform">island platform</a>, capable of handling two lines simultaneously. The station building includes a <a href="/wiki/Supermarket" title="Supermarket">supermarket</a> as well as a promotional center for local industries. Mukaihara is run by the Chūgoku Transportation Public Corporation, and features a <a class="mw-redirect" href="/wiki/Green_Window" title="Green Window">Green Window</a>.</p>
