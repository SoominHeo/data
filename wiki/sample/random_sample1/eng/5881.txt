<h1 class="firstHeading" id="firstHeading" lang="en">Trace element</h1>
<div class="mw-body-content" id="bodyContent">
<div id="siteSub">From Wikipedia, the free encyclopedia</div>
<div id="contentSub"></div>
<div class="mw-jump" id="jump-to-nav">
					Jump to:					<a href="#mw-head">navigation</a>, 					<a href="#p-search">search</a>
</div>
<div class="mw-content-ltr" dir="ltr" id="mw-content-text" lang="en"><script>function mfTempOpenSection(id){var block=document.getElementById("mf-section-"+id);block.className+=" open-block";block.previousSibling.className+=" open-block";}</script><p>A <b>trace element</b> is a <a href="/wiki/Chemical_element" title="Chemical element">chemical element</a> whose <a href="/wiki/Concentration" title="Concentration">concentration</a> (or other measure of amount) is very low ( a "trace amount"). The exact definition depends on the field of science:</p>
<ul>
<li>In <a href="/wiki/Analytical_chemistry" title="Analytical chemistry">analytical chemistry</a>, a trace element is one whose average concentration of less than 100 <a class="mw-redirect" href="/wiki/Parts_per_million" title="Parts per million">parts per million</a> (ppm) measured in atomic count or less than 100 micrograms per gram.</li>
<li>In <a href="/wiki/Biochemistry" title="Biochemistry">biochemistry</a>, a trace element is a <a class="mw-redirect" href="/wiki/Dietary_element" title="Dietary element">dietary element</a> that is needed in very minute quantities for the proper growth, development, and physiology of the organism.<sup class="reference" id="cite_ref-1"><a href="#cite_note-1">[1]</a></sup> For example, <a href="/wiki/Magnesium" title="Magnesium">magnesium</a> is a <a href="/wiki/Trace_metal" title="Trace metal">trace metal</a>.</li>
<li>In <a href="/wiki/Geochemistry" title="Geochemistry">geochemistry</a>, a trace element is one whose concentration is less than 1000 ppm or 0.1<a class="mw-redirect" href="/wiki/%25" title="%">%</a> of a <a href="/wiki/Rock_(geology)" title="Rock (geology)">rock</a>'s composition. The term is used mainly in <a href="/wiki/Igneous_petrology" title="Igneous petrology">igneous petrology</a>. Trace elements will be either compatible with a liquid or solid phase. If compatible with a <a href="/wiki/Mineral" title="Mineral">mineral</a>, it will be incorporated into a solid phase (e.g., <a href="/wiki/Nickel" title="Nickel">nickel</a>'s compatibility with <a href="/wiki/Olivine" title="Olivine">olivine</a>). If it is incompatible with any existing mineral phase it will remain in the liquid <a href="/wiki/Magma" title="Magma">magma</a> phase. The measurement of this ratio is known as the partition coefficient. Trace elements can be substituted for network-forming ions in mineral structures. Trace elements that are not essential to a mineral's defined composition will not appear in the chemical formula of that mineral.</li>
</ul>
