<p>In <a href="/wiki/Mathematics" title="Mathematics">mathematics</a>, <b>−1</b> is the <a href="/wiki/Additive_inverse" title="Additive inverse">additive inverse</a> of <a class="mw-redirect" href="/wiki/1_(number)" title="1 (number)">1</a>, that is, the number that when <a href="/wiki/Addition" title="Addition">added</a> to 1 gives the additive identity element, 0. It is the <a href="/wiki/Negative_number" title="Negative number">negative</a> <a href="/wiki/Integer" title="Integer">integer</a> greater than negative two (−2) and less than <a class="mw-redirect" href="/wiki/0_(number)" title="0 (number)">0</a>.</p>
<p>Negative one bears relation to <a href="/wiki/Euler%27s_identity" title="Euler's identity">Euler's identity</a> since <i>e</i><sup><span class="texhtml">π</span><i>i</i></sup> = −1.</p>
<p>In <a href="/wiki/Software_development" title="Software development">software development</a>, <b>−1</b> is a common initial value for integers and is also used to show that <a href="/wiki/Sentinel_value" title="Sentinel value">a variable contains no useful information</a>.</p>
<p><b>Negative one</b> has some similar but slightly different properties to positive one.<sup class="reference" id="cite_ref-1"><a href="#cite_note-1">[1]</a></sup></p>

<h3><span class="mw-headline" id="Square_of_.E2.88.921">Square of −1</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%E2%88%921&amp;action=edit&amp;section=2" title="Edit section: Square of −1">edit</a><span class="mw-editsection-bracket">]</span></span></h3>

<h3><span class="mw-headline" id="Square_roots_of_.E2.88.921">Square roots of −1</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%E2%88%921&amp;action=edit&amp;section=3" title="Edit section: Square roots of −1">edit</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>The <a href="/wiki/Complex_number" title="Complex number">complex number</a> <i><a href="/wiki/Imaginary_unit" title="Imaginary unit">i</a></i> satisfies <i>x</i><sup>2</sup> = −1, and as such can be considered as a <a href="/wiki/Square_root" title="Square root">square root</a> of −1. The only other complex number <i>x</i> satisfying the equation <i>x</i><sup>2</sup> = −1 is −<i>i</i>.<sup class="reference" id="cite_ref-2"><a href="#cite_note-2">[2]</a></sup> In the algebra of <a href="/wiki/Quaternion" title="Quaternion">quaternions</a>, containing the complex plane, the equation <i>x</i><sup>2</sup> = −1 has an <a href="/wiki/Quaternion#Square_roots_of_.E2.88.921" title="Quaternion">infinity of solutions</a>.</p>
<h2><span class="mw-headline" id="Exponentiation_to_negative_integers">Exponentiation to negative integers</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%E2%88%921&amp;action=edit&amp;section=4" title="Edit section: Exponentiation to negative integers">edit</a><span class="mw-editsection-bracket">]</span></span></h2>
<p><a href="/wiki/Exponentiation" title="Exponentiation">Exponentiation</a> of a non-zero real number can be extended to <a href="/wiki/Exponentiation#Negative_integer_exponents" title="Exponentiation">negative integers</a>. We make the definition that <i>x</i><sup>−1</sup> = <span class="sfrac nowrap" style="display:inline-block; vertical-align:-0.5em; font-size:85%; text-align:center;"><span style="display:block; line-height:1em; margin:0 0.1em;">1</span><span class="visualhide">/</span><span style="display:block; line-height:1em; margin:0 0.1em; border-top:1px solid;"><i>x</i></span></span>, meaning that we define raising a number to the power −1 to have the same effect as taking its <a class="mw-redirect" href="/wiki/Reciprocal_(mathematics)" title="Reciprocal (mathematics)">reciprocal</a>. This definition then extended to negative integers preserves the exponential law <i>x</i><sup><i>a</i></sup><i>x</i><sup><i>b</i></sup> = <i>x</i><sup>(<i>a</i> + <i>b</i>)</sup> for real numbers <i>a</i> and <i>b</i>.</p>
<p>Exponentiation to negative integers can be extended to invertible elements of a ring, by defining <i>x</i><sup>−1</sup> as the multiplicative inverse of <i>x</i>.</p>
<p>−1 that appears next to functions or matrices does not mean raising them to the power −1 but their <a class="mw-redirect" href="/wiki/Inverse_functions" title="Inverse functions">inverse functions</a> or <a class="mw-redirect" href="/wiki/Inverse_matrices" title="Inverse matrices">inverse matrices</a>. For example, <i>f</i><sup>−1</sup>(<i>x</i>) is the inverse of <i>f</i>(<i>x</i>), or sin<sup>−1</sup>(<i>x</i>) is a notation of <a class="mw-redirect" href="/wiki/Arcsine" title="Arcsine">arcsine</a> function.</p>
<h2><span class="mw-headline" id="Inductive_dimension">Inductive dimension</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%E2%88%921&amp;action=edit&amp;section=5" title="Edit section: Inductive dimension">edit</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>The <a href="/wiki/Inductive_dimension" title="Inductive dimension">Inductive dimension</a> of the empty set is defined to be −1.</p>
<h2><span class="mw-headline" id="Computer_representation">Computer representation</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%E2%88%921&amp;action=edit&amp;section=6" title="Edit section: Computer representation">edit</a><span class="mw-editsection-bracket">]</span></span></h2>
<div class="hatnote" role="note">Main article: <a href="/wiki/Signed_number_representations" title="Signed number representations">Signed number representations</a></div>
<p>Most computer systems represent negative integers using <a href="/wiki/Two%27s_complement" title="Two's complement">two's complement</a>. In such systems, −1 is represented using a bit pattern of all ones. For example, an 8-bit signed integer using two's complement would represent −1 as the bitstring "11111111", or "FF" in <a href="/wiki/Hexadecimal" title="Hexadecimal">hexadecimal</a> (base 16). If interpreted as an unsigned integer, the same bitstring of <i>n</i> ones represents 2<sup><i>n</i></sup> − 1, the largest possible value that <i>n</i> bits can hold. For example, the 8-bit string "11111111" above represents 2<sup>8</sup> − 1 = 255.</p>
