<h1 class="firstHeading" id="firstHeading" lang="en">Schwarz reflection principle</h1>
<div class="mw-body-content" id="bodyContent">

<p>In <a href="/wiki/Mathematics" title="Mathematics">mathematics</a>, applying the <b>Schwarz reflection principle</b> is a way to extend the domain of definition of an <a href="/wiki/Analytic_function" title="Analytic function">analytic function</a> of a <a class="mw-redirect" href="/wiki/Complex_variable" title="Complex variable">complex variable</a>, <i>F</i>, which is defined on the <a href="/wiki/Upper_half-plane" title="Upper half-plane">upper half-plane</a> and has well-defined and <a href="/wiki/Real_number" title="Real number">real number</a> boundary values on the <a class="mw-redirect" href="/wiki/Real_axis" title="Real axis">real axis</a>. Then the extension formula given above is an <a href="/wiki/Analytic_continuation" title="Analytic continuation">analytic continuation</a> to the whole complex plane.</p>
<p>In practice it would be better to have a theorem that allows <i>F</i> certain singularities, for example <i>F</i> a <a href="/wiki/Meromorphic_function" title="Meromorphic function">meromorphic function</a>. To understand such extensions, one needs a proof method that can be weakened. In fact <a href="/wiki/Morera%27s_theorem" title="Morera's theorem">Morera's theorem</a> is well adapted to proving such statements. <a class="mw-redirect" href="/wiki/Contour_integral" title="Contour integral">Contour integrals</a> involving the extension of <i>F</i> clearly split into two, using part of the real axis. So, given that the principle is rather easy to prove in the special case from Morera's theorem, understanding the proof is enough to generate other results.</p>
<p>The principle also adapts to apply to <a href="/wiki/Harmonic_function" title="Harmonic function">harmonic functions</a>.</p>
