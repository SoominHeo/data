<h1 class="firstHeading" id="firstHeading" lang="en">Barrelled space</h1>
<div class="mw-body-content" id="bodyContent">
<div id="siteSub">From Wikipedia, the free encyclopedia</div>
<div id="contentSub"></div>
<div class="mw-jump" id="jump-to-nav">
					Jump to:					<a href="#mw-head">navigation</a>, 					<a href="#p-search">search</a>
</div>
<div class="mw-content-ltr" dir="ltr" id="mw-content-text" lang="en"><script>function mfTempOpenSection(id){var block=document.getElementById("mf-section-"+id);block.className+=" open-block";block.previousSibling.className+=" open-block";}</script><p>In <a href="/wiki/Functional_analysis" title="Functional analysis">functional analysis</a> and related areas of <a href="/wiki/Mathematics" title="Mathematics">mathematics</a>, <b>barrelled spaces</b> are Hausdorff <a class="mw-redirect" href="/wiki/Topological_vector_spaces" title="Topological vector spaces">topological vector spaces</a> for which every barrelled set in the space is a <a class="mw-redirect" href="/wiki/Neighbourhood_(topology)" title="Neighbourhood (topology)">neighbourhood</a> for the <a class="mw-redirect" href="/wiki/Zero_vector" title="Zero vector">zero vector</a>. A <b>barrelled set</b> or a <b>barrel</b> in a topological vector space is a <a href="/wiki/Set_(mathematics)" title="Set (mathematics)">set</a> which is <a href="/wiki/Convex_set" title="Convex set">convex</a>, <a href="/wiki/Balanced_set" title="Balanced set">balanced</a>, <a href="/wiki/Absorbing_set" title="Absorbing set">absorbing</a> and <a href="/wiki/Closed_set" title="Closed set">closed</a>. Barrelled spaces are studied because a form of the <a class="mw-redirect" href="/wiki/Banach%E2%80%93Steinhaus_theorem" title="Banach–Steinhaus theorem">Banach–Steinhaus theorem</a> still holds for them.</p>

<h2><span class="mw-headline" id="History">History</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Barrelled_space&amp;action=edit&amp;section=1" title="Edit section: History">edit</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>Barrelled spaces were introduced by <a href="/wiki/Nicolas_Bourbaki" title="Nicolas Bourbaki">Bourbaki</a> (<a href="#CITEREFBourbaki1950">1950</a>).</p>
<h2><span class="mw-headline" id="Examples">Examples</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Barrelled_space&amp;action=edit&amp;section=2" title="Edit section: Examples">edit</a><span class="mw-editsection-bracket">]</span></span></h2>
<ul>
<li>In a <a class="mw-redirect" href="/wiki/Semi_normed_vector_space" title="Semi normed vector space">semi normed vector space</a> the closed <a class="mw-redirect" href="/wiki/Unit_ball" title="Unit ball">unit ball</a> is a barrel.</li>
<li>Every <a href="/wiki/Locally_convex_topological_vector_space" title="Locally convex topological vector space">locally convex topological vector space</a> has a <a class="mw-redirect" href="/wiki/Neighbourhood_basis" title="Neighbourhood basis">neighbourhood basis</a> consisting of barrelled sets, although the space itself need not be a barreled space.</li>
<li><a href="/wiki/Fr%C3%A9chet_space" title="Fréchet space">Fréchet spaces</a>, and in particular <a href="/wiki/Banach_space" title="Banach space">Banach spaces</a>, are barrelled, but generally a <a href="/wiki/Normed_vector_space" title="Normed vector space">normed vector space</a> is <i>not</i> barrelled.</li>
<li><a href="/wiki/Montel_space" title="Montel space">Montel spaces</a> are barrelled. Consequently, strong duals of Montel spaces are barrelled (since they are Montel spaces).</li>
<li><a class="mw-redirect" href="/wiki/Locally_convex_space" title="Locally convex space">locally convex spaces</a> which are <a href="/wiki/Baire_space" title="Baire space">Baire spaces</a> are barrelled.</li>
</ul>
