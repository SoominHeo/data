<h1 class="firstHeading" id="firstHeading" lang="en">Backspace</h1>
<div class="mw-body-content" id="bodyContent">

<p><b>Backspace</b> is the keyboard key that originally pushed the <a href="/wiki/Typewriter" title="Typewriter">typewriter</a> carriage one position backwards, and in modern computer systems moves the display cursor one position backwards,<sup class="reference" id="cite_ref-1"><a href="#cite_note-1">[note 1]</a></sup> deletes the character at that position, and shifts back the text after that position by one position.</p>
<p></p>

<h2><span class="mw-headline" id="Typewriter">Typewriter</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Backspace&amp;action=edit&amp;section=1" title="Edit section: Typewriter">edit</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>In some<sup class="reference" id="cite_ref-2"><a href="#cite_note-2">[note 2]</a></sup> typewriters, a typist would, for example, type a lowercase letter A with acute accent (á) by typing a lowercase letter A, backspace, and then the acute accent key. This technique (also known as <a href="/wiki/Overstrike" title="Overstrike">overstrike</a>) is the basis for such spacing modifiers in computer character sets such as the <a href="/wiki/ASCII" title="ASCII">ASCII</a> caret (^, for the <a href="/wiki/Circumflex" title="Circumflex">circumflex</a> accent). Backspace composition no longer works with typical modern digital displays or typesetting systems<sup class="reference" id="cite_ref-3"><a href="#cite_note-3">[note 3]</a></sup> It has to some degree been replaced with the <a class="mw-redirect" href="/wiki/Combining_diacritical_mark" title="Combining diacritical mark">combining diacritical marks</a> mechanism of <a href="/wiki/Unicode" title="Unicode">Unicode</a>, though such characters do not work well with many fonts, and precomposed characters continue to be used. Some software like <a href="/wiki/TeX" title="TeX">TeX</a> or <a href="/wiki/Microsoft_Windows" title="Microsoft Windows">Microsoft Windows</a> use the opposite method for diacritical marks, namely positioning the accent first, and then the base letter on its position.</p>
<h2><span class="mw-headline" id="Computers">Computers</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Backspace&amp;action=edit&amp;section=2" title="Edit section: Computers">edit</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>Although the term "backspace" is the traditional name of the key which deletes the character to the left of the cursor, the actual key may be labeled in a variety of ways, for example <i>delete</i>,<sup class="reference" id="cite_ref-4"><a href="#cite_note-4">[1]</a></sup> <i>Erase</i> (for example in <a class="mw-redirect" href="/wiki/XO-1_(laptop)" title="XO-1 (laptop)">One Laptop Per Child</a>),<sup class="reference" id="cite_ref-5"><a href="#cite_note-5">[2]</a></sup> or with a left pointing arrow.<sup class="reference" id="cite_ref-debian_6-0"><a href="#cite_note-debian-6">[3]</a></sup> A dedicated symbol for "backspace" exists as U+232B ⌫ but its use as a keyboard label is moderately rare.</p>
<p>The backspace is distinct from the <a href="/wiki/Delete_key" title="Delete key">delete key</a>, which in <a href="/wiki/Punched_card" title="Punched card">paper media for computers</a> would punch out all the holes to strike out a character, and in modern computers deletes text <i>following</i> it. Also, the delete key often works as a generic command to remove an object (such as an image inside a document, or a file in a <a href="/wiki/File_manager" title="File manager">file manager</a>), while backspace usually does not.<sup class="reference" id="cite_ref-7"><a href="#cite_note-7">[4]</a></sup><sup class="reference" id="cite_ref-8"><a href="#cite_note-8">[5]</a></sup><sup class="reference" id="cite_ref-9"><a href="#cite_note-9">[6]</a></sup></p>
<h3><span class="mw-headline" id="Common_use">Common use</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Backspace&amp;action=edit&amp;section=3" title="Edit section: Common use">edit</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>In modern systems, the backspace key is often mapped to the delete character (0x7f in ASCII or Unicode), although the backspace key's function of deleting the character before the cursor remains.<sup class="reference" id="cite_ref-debian_6-1"><a href="#cite_note-debian-6">[3]</a></sup></p>
<p>The backspace key is commonly used to go back a page or up one level in graphical web or file browsers.</p>
<h3><span class="mw-headline" id=".5EH">^H</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Backspace&amp;action=edit&amp;section=4" title="Edit section: ^H">edit</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>Pressing the backspace key on a computer terminal would generate the <a href="/wiki/ASCII" title="ASCII">ASCII</a> code 08, BS or Backspace, a <a class="mw-redirect" href="/wiki/Control_code" title="Control code">control code</a> which would delete the preceding character. That control code could also be accessed by pressing Control-H, as <a href="/wiki/H" title="H">H</a> is the eighth letter of the <a href="/wiki/Latin_alphabet" title="Latin alphabet">Latin alphabet</a>. Terminals which did not have the backspace code mapped to the function of moving the cursor backwards and deleting the preceding character would display the symbols <b>^H</b> (<a href="/wiki/Caret_notation" title="Caret notation">caret</a>, H) when the backspace key was pressed. Even if a terminal did interpret backspace by deleting the preceding character, the system receiving the text might not. Then, the sender's screen would show a message without the supposedly deleted text, while that text, and the deletion codes, would be visible to the recipient. This sequence is still used humorously for <a href="/wiki/Epanorthosis" title="Epanorthosis">epanorthosis</a> by computer literates, denoting the deletion of a pretended blunder, much like a <a href="/wiki/Strikethrough" title="Strikethrough">strikethrough</a>; in this case, however, the ^H symbol is faked by typing a regular '^' key followed by typing a regular 'H' key.</p>
<p>Example:</p>
<dl>
<dd><i>Be nice to this fool</i>^H^H^H^H<i>gentleman; he's visiting from corporate HQ.</i><sup class="reference" id="cite_ref-Jargon_10-0"><a href="#cite_note-Jargon-10">[7]</a></sup></dd>
</dl>
<h3><span class="mw-headline" id=".5EW_and_.5EU">^W and ^U</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Backspace&amp;action=edit&amp;section=5" title="Edit section: ^W and ^U">edit</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>An alternative sometimes seen is <b>^W</b>, which is the shortcut to delete the previous word in the Berkeley Unix terminal line discipline. This shortcut has also made it into the insert mode of the <a href="/wiki/Vi" title="Vi">Vi</a> text editor and its clone <a href="/wiki/Vim_(text_editor)" title="Vim (text editor)">Vim</a>.<sup class="reference" id="cite_ref-11"><a href="#cite_note-11">[8]</a></sup></p>
<p><b>^U</b> deletes a line.<sup class="reference" id="cite_ref-12"><a href="#cite_note-12">[9]</a></sup></p>
<h2><span class="mw-headline" id="Other_meanings">Other meanings</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Backspace&amp;action=edit&amp;section=6" title="Edit section: Other meanings">edit</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>In a <a href="/wiki/Mainframe_computer" title="Mainframe computer">mainframe</a> environment, <i>to backspace</i> means to move a <a href="/wiki/Magnetic_tape" title="Magnetic tape">magnetic tape</a> backwards, typically to the previous block.</p>
<h2><span class="mw-headline" id="Notes">Notes</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Backspace&amp;action=edit&amp;section=7" title="Edit section: Notes">edit</a><span class="mw-editsection-bracket">]</span></span></h2>
<div class="reflist" style="list-style-type: decimal;">
<ol class="references">
<li id="cite_note-1"><span class="mw-cite-backlink"><b><a href="#cite_ref-1">^</a></b></span> <span class="reference-text">"Backwards" means to the left for left-to-right languages.</span></li>
<li id="cite_note-2"><span class="mw-cite-backlink"><b><a href="#cite_ref-2">^</a></b></span> <span class="reference-text">Many typewriters don't advance accent characters, so that no backspace is needed. However, it is still used e.g. for combining "o" with "/".</span></li>
<li id="cite_note-3"><span class="mw-cite-backlink"><b><a href="#cite_ref-3">^</a></b></span> <span class="reference-text">There is no reason why a digital display or typesetting system could not be designed to allow backspace composition, a.k.a. overstrike, if an engineer chose to do that. As most contemporary computer display and typesetting systems are raster graphics-based rather than character-based (as of the year 2012), they make overstrike actually quite easy to implement. However, the use of proportional-width rather than fixed-width (monospaced) fonts makes the practical implementation of overstrike more complicated, and the original physical motivation for the technique is not present in digital computer systems.</span></li>
</ol>
