<p>In <a href="/wiki/Mathematics" title="Mathematics">mathematics</a>, the <b>Birch and Swinnerton-Dyer conjecture</b> describes the set of rational solutions to equations defining an <a href="/wiki/Elliptic_curve" title="Elliptic curve">elliptic curve</a>. It is an open problem in the field of <a href="/wiki/Number_theory" title="Number theory">number theory</a> and is widely recognized as one of the most challenging mathematical problems. The conjecture was chosen as one of the seven <a href="/wiki/Millennium_Prize_Problems" title="Millennium Prize Problems">Millennium Prize Problems</a> listed by the <a href="/wiki/Clay_Mathematics_Institute" title="Clay Mathematics Institute">Clay Mathematics Institute</a>, which has offered a $1,000,000 prize for the first correct proof.<sup class="reference" id="cite_ref-1"><a href="#cite_note-1">[1]</a></sup> It is named after mathematicians <a class="mw-redirect" href="/wiki/Bryan_Birch" title="Bryan Birch">Bryan Birch</a> and <a href="/wiki/Peter_Swinnerton-Dyer" title="Peter Swinnerton-Dyer">Peter Swinnerton-Dyer</a> who developed the conjecture during the first half of the 1960s with the help of machine computation. As of 2016<sup class="plainlinks noprint asof-tag update" style="display:none;"><a class="external text" href="//en.wikipedia.org/w/index.php?title=Birch_and_Swinnerton-Dyer_conjecture&amp;action=edit">[update]</a></sup>, only special cases of the conjecture have been proved.</p>
<p>The conjecture relates arithmetic data associated with an elliptic curve <i>E</i> over a <a class="mw-redirect" href="/wiki/Number_field" title="Number field">number field</a> <i>K</i> to the behaviour of the <a class="mw-redirect" href="/wiki/Hasse%E2%80%93Weil_L-function" title="Hasse–Weil L-function">Hasse–Weil <i>L</i>-function</a> <i>L</i>(<i>E</i>, <i>s</i>) of <i>E</i> at <i>s</i> = 1. More specifically, it is conjectured that the <a href="/wiki/Rank_of_an_abelian_group" title="Rank of an abelian group">rank</a> of the <a href="/wiki/Abelian_group" title="Abelian group">abelian group</a> <i>E</i>(<i>K</i>) of points of <i>E</i> is the order of the zero of <i>L</i>(<i>E</i>, <i>s</i>) at <i>s</i> = 1, and the first non-zero coefficient in the <a class="mw-redirect" href="/wiki/Taylor_expansion" title="Taylor expansion">Taylor expansion</a> of <i>L</i>(<i>E</i>, <i>s</i>) at <i>s</i> = 1 is given by more refined arithmetic data attached to <i>E</i> over <i>K</i> (<a href="#CITEREFWiles2006">Wiles 2006</a>).</p>

<h2><span class="mw-headline" id="Background">Background</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Birch_and_Swinnerton-Dyer_conjecture&amp;action=edit&amp;section=1" title="Edit section: Background">edit</a><span class="mw-editsection-bracket">]</span></span></h2>
<p><a href="#CITEREFMordell1922">Mordell (1922)</a> proved <a class="mw-redirect" href="/wiki/Mordell%27s_theorem" title="Mordell's theorem">Mordell's theorem</a>: the group of <a href="/wiki/Rational_point" title="Rational point">rational points</a> on an elliptic curve has a finite <a href="/wiki/Basis_(linear_algebra)" title="Basis (linear algebra)">basis</a>. This means that for any elliptic curve there is a finite sub-set of the rational points on the curve, from which all further rational points may be generated.</p>
<p>If the number of rational points on a curve is <a href="/wiki/Infinite_set" title="Infinite set">infinite</a> then some point in a finite basis must have infinite order. The number of <i>independent</i> basis points with infinite order is called the <a href="/wiki/Rank_of_an_abelian_group" title="Rank of an abelian group">rank</a> of the curve, and is an important <a href="/wiki/Invariant_(mathematics)" title="Invariant (mathematics)">invariant</a> property of an elliptic curve.</p>
<p>If the rank of an elliptic curve is 0, then the curve has only a finite number of rational points. On the other hand, if the rank of the curve is greater than 0, then the curve has an infinite number of rational points.</p>
<p>Although Mordell's theorem shows that the rank of an elliptic curve is always finite, it does not give an effective method for calculating the rank of every curve. The rank of certain elliptic curves can be calculated using numerical methods but (in the current state of knowledge) it is unknown if these methods handle all curves.</p>
<p>An <i>L</i>-function <b><i>L</i>(<i>E</i>, <i>s</i>)</b> can be defined for an elliptic curve <i>E</i> by constructing an <a href="/wiki/Euler_product" title="Euler product">Euler product</a> from the number of points on the curve modulo each <a href="/wiki/Prime_number" title="Prime number">prime</a> <i>p</i>. This <i>L</i>-function is analogous to the <a href="/wiki/Riemann_zeta_function" title="Riemann zeta function">Riemann zeta function</a> and the <a class="mw-redirect" href="/wiki/Dirichlet_L-series" title="Dirichlet L-series">Dirichlet L-series</a> that is defined for a binary <a href="/wiki/Quadratic_form" title="Quadratic form">quadratic form</a>. It is a special case of a <a class="mw-redirect" href="/wiki/Hasse%E2%80%93Weil_L-function" title="Hasse–Weil L-function">Hasse–Weil L-function</a>.</p>
<p>The natural definition of <i>L</i>(<i>E</i>, <i>s</i>) only converges for values of <i>s</i> in the complex plane with Re(<i>s</i>) &gt; 3/2. <a href="/wiki/Helmut_Hasse" title="Helmut Hasse">Helmut Hasse</a> conjectured that <i>L</i>(<i>E</i>, <i>s</i>) could be extended by <a href="/wiki/Analytic_continuation" title="Analytic continuation">analytic continuation</a> to the whole complex plane. This conjecture was first proved by <a href="#CITEREFDeuring1941">Deuring (1941)</a> for elliptic curves with <a href="/wiki/Complex_multiplication" title="Complex multiplication">complex multiplication</a>. It was subsequently shown to be true for all elliptic curves over <b>Q</b>, as a consequence of the <a href="/wiki/Modularity_theorem" title="Modularity theorem">modularity theorem</a>.</p>
<p>Finding rational points on a general elliptic curve is a difficult problem. Finding the points on an elliptic curve modulo a given prime <i>p</i> is conceptually straightforward, as there are only a finite number of possibilities to check. However, for large primes it is computationally intensive.</p>
<h2><span class="mw-headline" id="History">History</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Birch_and_Swinnerton-Dyer_conjecture&amp;action=edit&amp;section=2" title="Edit section: History">edit</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>In the early 1960s <a href="/wiki/Peter_Swinnerton-Dyer" title="Peter Swinnerton-Dyer">Peter Swinnerton-Dyer</a> used the <a class="mw-redirect" href="/wiki/EDSAC" title="EDSAC">EDSAC</a> computer at the <a class="mw-redirect" href="/wiki/University_of_Cambridge_Computer_Laboratory" title="University of Cambridge Computer Laboratory">University of Cambridge Computer Laboratory</a> to calculate the number of points modulo <i>p</i> (denoted by <i>N<sub>p</sub></i>) for a large number of primes <i>p</i> on elliptic curves whose rank was known. <p>This in turn led them to make a general conjecture about the behaviour of a curve's L-function <i>L</i>(<i>E</i>, <i>s</i>) at <i>s</i> = 1, namely that it would have a zero of order <i>r</i> at this point. <p>The Birch and Swinnerton-Dyer conjecture has been proved only in special cases:</p>
<ol>
<li><a href="#CITEREFCoatesWiles1977">Coates &amp; Wiles (1977)</a> proved that if <i>E</i> is a curve over a number field <i>F</i> with complex multiplication by an <a class="mw-redirect" href="/wiki/Imaginary_quadratic_field" title="Imaginary quadratic field">imaginary quadratic field</a> <i>K</i> of <a class="mw-redirect" href="/wiki/Class_number_(number_theory)" title="Class number (number theory)">class number</a> 1, <i>F</i> = <i>K</i> or <b>Q</b>, and <i>L</i>(<i>E</i>, 1) is not 0 then <i>E</i>(<i>F</i>) is a finite group. This was extended to the case where <i>F</i> is any finite <a href="/wiki/Abelian_extension" title="Abelian extension">abelian extension</a> of <i>K</i> by <a href="#CITEREFArthaud1978">Arthaud (1978)</a>.</li>
<li><a href="#CITEREFGrossZagier1986">Gross &amp; Zagier (1986)</a> showed that if a <a href="/wiki/Modular_elliptic_curve" title="Modular elliptic curve">modular elliptic curve</a> has a first-order zero at <i>s</i> = 1 then it has a rational point of infinite order; see <a class="mw-redirect" href="/wiki/Gross%E2%80%93Zagier_theorem" title="Gross–Zagier theorem">Gross–Zagier theorem</a>.</li>
<li><a href="#CITEREFKolyvagin1989">Kolyvagin (1989)</a> showed that a modular elliptic curve <i>E</i> for which <i>L</i>(<i>E</i>, 1) is not zero has rank 0, and a modular elliptic curve <i>E</i> for which <i>L</i>(<i>E</i>, 1) has a first-order zero at <i>s</i> = 1 has rank 1.</li>
<li><a href="#CITEREFRubin1991">Rubin (1991)</a> showed that for elliptic curves defined over an imaginary quadratic field <i>K</i> with complex multiplication by <i>K</i>, if the <i>L</i>-series of the elliptic curve was not zero at <i>s</i> = 1, then the <i>p</i>-part of the Tate–Shafarevich group had the order predicted by the Birch and Swinnerton-Dyer conjecture, for all primes <i>p</i> &gt; 7.</li>
<li><a href="#CITEREFBreuilConradDiamondTaylor2001">Breuil et al. (2001)</a>, extending work of <a href="#CITEREFWiles1995">Wiles (1995)</a>, proved that <a href="/wiki/Modularity_theorem" title="Modularity theorem">all elliptic curves defined over the rational numbers are modular</a>, which extends results 2 and 3 to all elliptic curves over the rationals, and shows that the <i>L</i>-functions of all elliptic curves over <b>Q</b> are defined at <i>s</i> = 1.</li>
<li><a href="#CITEREFBhargavaShankar2015">Bhargava &amp; Shankar (2015)</a> proved that the average rank of the Mordell–Weil group of an elliptic curve over <b>Q</b> is bounded above by 7/6. Combining this with the <a class="new" href="/w/index.php?title=P-parity_theorem&amp;action=edit&amp;redlink=1" title="P-parity theorem (page does not exist)">p-parity theorem</a> of <a href="#CITEREFNekov.C3.A1.C5.992009">Nekovář (2009)</a> and <a href="#CITEREFDokchitserDokchitser2010">Dokchitser &amp; Dokchitser (2010)</a> and with the proof of the <a href="/wiki/Main_conjecture_of_Iwasawa_theory" title="Main conjecture of Iwasawa theory">main conjecture of Iwasawa theory</a> for GL(2) by <a href="#CITEREFSkinnerUrban2014">Skinner &amp; Urban (2014)</a>, they conclude that a positive proportion of elliptic curves over <b>Q</b> have analytic rank zero, and hence, by <a href="#CITEREFKolyvagin1989">Kolyvagin (1989)</a>, satisfy the Birch and Swinnerton-Dyer conjecture.</li>
</ol>
<p>Nothing has been proved for curves with rank greater than 1, although there is extensive numerical evidence for the truth of the conjecture.<sup class="reference" id="cite_ref-3"><a href="#cite_note-3">[3]</a></sup></p>
<h2><span class="mw-headline" id="Consequences">Consequences</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Birch_and_Swinnerton-Dyer_conjecture&amp;action=edit&amp;section=4" title="Edit section: Consequences">edit</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>Much like the <a href="/wiki/Riemann_hypothesis" title="Riemann hypothesis">Riemann hypothesis</a>, this conjecture has multiple consequences, including the following two:</p>
<ul>
<li>Let <span class="texhtml mvar" style="font-style:italic;">n</span> be an odd <a class="mw-redirect" href="/wiki/Square-free" title="Square-free">square-free</a> integer. Assuming the Birch and Swinnerton-Dyer conjecture, <span class="texhtml mvar" style="font-style:italic;">n</span> is the area of a right triangle with rational side lengths (a <a href="/wiki/Congruent_number" title="Congruent number">congruent number</a>) if and only if the number of triplets of integers (<span class="texhtml mvar" style="font-style:italic;">x</span>, <span class="texhtml mvar" style="font-style:italic;">y</span>, <span class="texhtml mvar" style="font-style:italic;">z</span>) satisfying <span class="texhtml">2<i>x</i><sup>2</sup> + <i>y</i><sup>2</sup> + 8<i>z</i><sup>2</sup> = <i>n</i></span> is twice the number of triplets satisfying <span class="texhtml">2<i>x</i><sup>2</sup> + <i>y</i><sup>2</sup> + 32<i>z</i><sup>2</sup> = <i>n</i></span>. This statement, due to <a href="/wiki/Tunnell%27s_theorem" title="Tunnell's theorem">Tunnell's theorem</a> (<a href="#CITEREFTunnell1983">Tunnell 1983</a>), is related to the fact that <i>n</i> is a congruent number if and only if the elliptic curve <span class="texhtml"><i>y</i><sup>2</sup> = <i>x</i><sup>3</sup> − <i>n</i><sup>2</sup><i>x</i></span> has a rational point of infinite order (thus, under the Birch and Swinnerton-Dyer conjecture, its <span class="texhtml mvar" style="font-style:italic;">L</span>-function has a zero at <span class="texhtml">1</span>). The interest in this statement is that the condition is easily verified.<sup class="reference" id="cite_ref-4"><a href="#cite_note-4">[4]</a></sup></li>
<li>In a different direction, certain analytic methods allow for an estimation of the order of zero in the center of the <a class="mw-redirect" href="/wiki/Critical_strip" title="Critical strip">critical strip</a> of families of <i>L</i>-functions. Admitting the BSD conjecture, these estimations correspond to information about the rank of families of elliptic curves in question. For example: suppose the <a href="/wiki/Generalized_Riemann_hypothesis" title="Generalized Riemann hypothesis">generalized Riemann hypothesis</a> and the BSD conjecture, the average rank of curves given by <span class="texhtml"><i>y</i><sup>2</sup> = <i>x</i><sup>3</sup> + <i>ax</i>+ <i>b</i></span> is smaller than <span class="texhtml">2</span>.<sup class="reference" id="cite_ref-5"><a href="#cite_note-5">[5]</a></sup></li>
</ul>
