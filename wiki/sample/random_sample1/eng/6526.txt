<p>In the <a href="/wiki/Mathematics" title="Mathematics">mathematical</a> field of <a href="/wiki/Numerical_analysis" title="Numerical analysis">numerical analysis</a>, <b>interpolation</b> is a method of constructing new <a class="mw-redirect" href="/wiki/Data_points" title="Data points">data points</a> within the range of a <a class="mw-redirect" href="/wiki/Discrete_set" title="Discrete set">discrete set</a> of known data points.</p>
<p>In <a href="/wiki/Engineering" title="Engineering">engineering</a> and <a href="/wiki/Science" title="Science">science</a>, one often has a number of data points, obtained by <a href="/wiki/Sampling_(statistics)" title="Sampling (statistics)">sampling</a> or <a class="mw-redirect" href="/wiki/Experimentation" title="Experimentation">experimentation</a>, which represent the values of a function for a limited number of values of the independent variable. It is often required to <b>interpolate</b> (i.e. estimate) the value of that function for an intermediate value of the independent variable. This may be achieved by <a href="/wiki/Curve_fitting" title="Curve fitting">curve fitting</a> or <a href="/wiki/Regression_analysis" title="Regression analysis">regression analysis</a>.</p>
<p>A different problem which is closely related to interpolation is the <a href="/wiki/Function_approximation" title="Function approximation">approximation of a complicated function by a simple function</a>. Suppose the formula for some given function is known, but too complex to evaluate efficiently. A few known data points from the original function can be used to create an interpolation based on a simpler function. Of course, when a simple function is used to estimate data points from the original, interpolation errors are usually present; however, depending on the problem domain and the interpolation method used, the gain in simplicity may be of greater value than the resultant loss in precision.</p>
<p>In the examples below if we consider x as a topological space and the function f forms a different kind of Banach spaces then the problem is treated as "interpolation of operators". The classical results about interpolation of operators are the <a href="/wiki/Riesz%E2%80%93Thorin_theorem" title="Riesz–Thorin theorem">Riesz–Thorin theorem</a> and the <a class="mw-redirect" href="/wiki/Marcinkiewicz_theorem" title="Marcinkiewicz theorem">Marcinkiewicz theorem</a>. There are also many other subsequent results.</p>

<p>Interpolation provides a means of estimating the function at intermediate points, such as <i>x</i> = 2.5.</p>
<p>There are many different interpolation methods, some of which are described below. Some of the concerns to take into account when choosing an appropriate <a href="/wiki/Algorithm" title="Algorithm">algorithm</a> are: How accurate is the method? How expensive is it? How <a class="mw-redirect" href="/wiki/Smooth_function" title="Smooth function">smooth</a> is the interpolant? How many data points are needed?</p>

<h3><span class="mw-headline" id="Piecewise_constant_interpolation">Piecewise constant interpolation</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Interpolation&amp;action=edit&amp;section=2" title="Edit section: Piecewise constant interpolation">edit</a><span class="mw-editsection-bracket">]</span></span></h3>

<div class="hatnote" role="note">For more details on this topic, see <a href="/wiki/Nearest-neighbor_interpolation" title="Nearest-neighbor interpolation">Nearest-neighbor interpolation</a>.</div>
<p>The simplest interpolation method is to locate the nearest data value, and assign the same value. In simple problems, this method is unlikely to be used, as linear interpolation (see below) is almost as easy, but in higher-dimensional <a href="/wiki/Multivariate_interpolation" title="Multivariate interpolation">multivariate interpolation</a>, this could be a favourable choice for its speed and simplicity.</p>

<p>Substituting <i>x</i> = 2.5, we find that <i>f</i>(2.5) = 0.5965.</p>
<p>Generally, if we have <i>n</i> data points, there is exactly one polynomial of degree at most <i>n</i>−1 going through all the data points. The interpolation error is proportional to the distance between the data points to the power <i>n</i>. Furthermore, the interpolant is a polynomial and thus infinitely differentiable. So, we see that polynomial interpolation overcomes most of the problems of linear interpolation.</p>
<p>However, polynomial interpolation also has some disadvantages. Calculating the interpolating polynomial is computationally expensive (see <a href="/wiki/Computational_complexity_theory" title="Computational complexity theory">computational complexity</a>) compared to linear interpolation. Furthermore, polynomial interpolation may exhibit oscillatory artifacts, especially at the end points (see <a href="/wiki/Runge%27s_phenomenon" title="Runge's phenomenon">Runge's phenomenon</a>).</p>
<p>Polynomial interpolation can estimate local maxima and minima that are outside the range of the samples, unlike linear interpolation. For example, the interpolant above has a local maximum at <i>x</i> ≈ 1.566, <i>f</i>(<i>x</i>) ≈ 1.003 and a local minimum at <i>x</i> ≈ 4.708, <i>f</i>(<i>x</i>) ≈ −1.003. However, these maxima and minima may exceed the theoretical range of the function—for example, a function that is always positive may have an interpolant with negative values, and whose inverse therefore contains false <a href="/wiki/Division_by_zero" title="Division by zero">vertical asymptotes</a>.</p>
<p>More generally, the shape of the resulting curve, especially for very high or low values of the independent variable, may be contrary to commonsense, i.e. to what is known about the experimental system which has generated the data points. These disadvantages can be reduced by using spline interpolation or restricting attention to <a href="/wiki/Chebyshev_polynomials" title="Chebyshev polynomials">Chebyshev polynomials</a>.</p>

<p>In this case we get <i>f</i>(2.5) = 0.5972.</p>
<p>Like polynomial interpolation, spline interpolation incurs a smaller error than linear interpolation and the interpolant is smoother. However, the interpolant is easier to evaluate than the high-degree polynomials used in polynomial interpolation. It also does not suffer from <a href="/wiki/Runge%27s_phenomenon" title="Runge's phenomenon">Runge's phenomenon</a>.</p>

<h2><span class="mw-headline" id="Interpolation_via_Gaussian_processes">Interpolation via Gaussian processes</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Interpolation&amp;action=edit&amp;section=6" title="Edit section: Interpolation via Gaussian processes">edit</a><span class="mw-editsection-bracket">]</span></span></h2>
<p><a href="/wiki/Gaussian_process" title="Gaussian process">Gaussian process</a> is a powerful non-linear interpolation tool. Many popular interpolation tools are actually equivalent to particular Gaussian processes. Gaussian processes can be used not only for fitting an interpolant that passes exactly through the given data points but also for regression, i.e., for fitting a curve through noisy data. In the geostatistics community Gaussian process regression is also known as <a href="/wiki/Kriging" title="Kriging">Kriging</a>.</p>
<h2><span class="mw-headline" id="Other_forms_of_interpolation">Other forms of interpolation</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Interpolation&amp;action=edit&amp;section=7" title="Edit section: Other forms of interpolation">edit</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>Other forms of interpolation can be constructed by picking a different class of interpolants. For instance, rational interpolation is <b>interpolation</b> by <a href="/wiki/Rational_function" title="Rational function">rational functions</a> using <a href="/wiki/Pad%C3%A9_approximant" title="Padé approximant">Padé approximant</a>, and <a href="/wiki/Trigonometric_interpolation" title="Trigonometric interpolation">trigonometric interpolation</a> is interpolation by <a href="/wiki/Trigonometric_polynomial" title="Trigonometric polynomial">trigonometric polynomials</a> using <a href="/wiki/Fourier_series" title="Fourier series">Fourier series</a>. Another possibility is to use <a href="/wiki/Wavelet" title="Wavelet">wavelets</a>.</p>
<p>The <a href="/wiki/Whittaker%E2%80%93Shannon_interpolation_formula" title="Whittaker–Shannon interpolation formula">Whittaker–Shannon interpolation formula</a> can be used if the number of data points is infinite.</p>
<p>Sometimes, we know not only the value of the function that we want to interpolate, at some points, but also its derivative. This leads to <a href="/wiki/Hermite_interpolation" title="Hermite interpolation">Hermite interpolation</a> problems.</p>
<p>When each data point is itself a function, it can be useful to see the interpolation problem as a partial <a href="/wiki/Advection" title="Advection">advection</a> problem between each data point. This idea leads to the <a class="new" href="/w/index.php?title=Displacement_interpolation&amp;action=edit&amp;redlink=1" title="Displacement interpolation (page does not exist)">displacement interpolation</a> problem used in <a href="/wiki/Transportation_theory_(mathematics)" title="Transportation theory (mathematics)">transportation theory</a>.</p>

<div class="hatnote" role="note">Main article: <a href="/wiki/Multivariate_interpolation" title="Multivariate interpolation">Multivariate interpolation</a></div>
<p>Multivariate interpolation is the interpolation of functions of more than one variable. Methods include <a href="/wiki/Bilinear_interpolation" title="Bilinear interpolation">bilinear interpolation</a> and <a href="/wiki/Bicubic_interpolation" title="Bicubic interpolation">bicubic interpolation</a> in two dimensions, and <a href="/wiki/Trilinear_interpolation" title="Trilinear interpolation">trilinear interpolation</a> in three dimensions. They can be applied to gridded or scattered data.</p>

<h2><span class="mw-headline" id="Interpolation_in_digital_signal_processing">Interpolation in digital signal processing</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Interpolation&amp;action=edit&amp;section=9" title="Edit section: Interpolation in digital signal processing">edit</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>In the domain of digital signal processing, the term interpolation refers to the process of converting a sampled digital signal (such as a sampled audio signal) to that of a higher sampling rate (<a href="/wiki/Upsampling" title="Upsampling">Upsampling</a>) using various digital filtering techniques (e.g., convolution with a frequency-limited impulse signal). In this application there is a specific requirement that the harmonic content of the original signal be preserved without creating aliased harmonic content of the original signal above the original Nyquist limit of the signal (i.e., above fs/2 of the original signal sample rate). An early and fairly elementary discussion on this subject can be found in Rabiner and Crochiere's book <i>Multirate Digital Signal Processing</i>.<sup class="reference" id="cite_ref-1"><a href="#cite_note-1">[1]</a></sup></p>
<h2><span class="mw-headline" id="Related_concepts">Related concepts</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Interpolation&amp;action=edit&amp;section=10" title="Edit section: Related concepts">edit</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>The term <i><a href="/wiki/Extrapolation" title="Extrapolation">extrapolation</a></i> is used to find data points outside the range of known data points.</p>
<p>In <a href="/wiki/Curve_fitting" title="Curve fitting">curve fitting</a> problems, the constraint that the interpolant has to go exactly through the data points is relaxed. It is only required to approach the data points as closely as possible (within some other constraints). This requires parameterizing the potential interpolants and having some way of measuring the error. In the simplest case this leads to <a href="/wiki/Least_squares" title="Least squares">least squares</a> approximation.</p>
<p><a href="/wiki/Approximation_theory" title="Approximation theory">Approximation theory</a> studies how to find the best approximation to a given function by another function from some predetermined class, and how good this approximation is. This clearly yields a bound on how well the interpolant can approximate the unknown function.</p>
