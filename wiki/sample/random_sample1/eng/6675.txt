<h1 class="firstHeading" id="firstHeading" lang="en">Subset</h1>

<p>In <a href="/wiki/Mathematics" title="Mathematics">mathematics</a>, especially in <a href="/wiki/Set_theory" title="Set theory">set theory</a>, a <a href="/wiki/Set_(mathematics)" title="Set (mathematics)">set</a> <i>A</i> is a <b>subset</b> of a set <i>B</i>, or equivalently <i>B</i> is a <b>superset</b> of <i>A</i>, if <i>A</i> is "contained" inside <i>B</i>, that is, all <a href="/wiki/Element_(mathematics)" title="Element (mathematics)">elements</a> of <i>A</i> are also elements of <i>B</i>. <i>A</i> and <i>B</i> may coincide. The relationship of one set being a subset of another is called <b>inclusion</b> or sometimes <b>containment</b>.</p>
<p>The subset relation defines a <a class="mw-redirect" href="/wiki/Partial_order" title="Partial order">partial order</a> on sets.</p>
<p>The <a href="/wiki/Algebraic_structure" title="Algebraic structure">algebra</a> of subsets forms a <a href="/wiki/Boolean_algebra_(structure)" title="Boolean algebra (structure)">Boolean algebra</a> in which the subset relation is called <a href="/wiki/Inclusion_(Boolean_algebra)" title="Inclusion (Boolean algebra)">inclusion</a>.</p>

<h2><span class="mw-headline" id=".E2.8A.82_and_.E2.8A.83_symbols">⊂ and ⊃ symbols</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Subset&amp;action=edit&amp;section=3" title="Edit section: ⊂ and ⊃ symbols">edit</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>Some authors use the symbols ⊂ and ⊃ to indicate <i>subset</i> and <i>superset</i> respectively; that is, with the same meaning and instead of the symbols, ⊆ and ⊇.<sup class="reference" id="cite_ref-2"><a href="#cite_note-2">[2]</a></sup> So for example, for these authors, it is true of every set <i>A</i> that <span class="nowrap"><i>A</i> ⊂ <i>A</i></span>.</p>
<p>Other authors prefer to use the symbols ⊂ and ⊃ to indicate <i>proper</i> subset and superset, respectively, instead of ⊊ and ⊋.<sup class="reference" id="cite_ref-3"><a href="#cite_note-3">[3]</a></sup> This usage makes ⊆ and ⊂ analogous to the <a href="/wiki/Inequality_(mathematics)" title="Inequality (mathematics)">inequality</a> symbols ≤ and &lt;. For example, if <span class="nowrap"><i>x</i> ≤ <i>y</i></span> then <i>x</i> may or may not equal <i>y</i>, but if <span class="nowrap"><i>x</i> &lt; <i>y</i></span>, then <i>x</i> definitely does not equal <i>y</i>, and <i>is</i> less than <i>y</i>. Similarly, using the convention that ⊂ is proper subset, if <span class="nowrap"><i>A</i> ⊆ <i>B</i></span>, then <i>A</i> may or may not equal <i>B</i>, but if <span class="nowrap"><i>A</i> ⊂ <i>B</i></span>, then <i>A</i> definitely does not equal <i>B</i>.</p>
<h2><span class="mw-headline" id="Examples">Examples</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Subset&amp;action=edit&amp;section=4" title="Edit section: Examples">edit</a><span class="mw-editsection-bracket">]</span></span></h2>
<div class="thumb tright">
<div class="thumbinner" style="width:222px;"><a class="image" href="/wiki/File:PolygonsSet_EN.svg"><img alt="" class="thumbimage" data-file-height="474" data-file-width="690" height="151" src="//upload.wikimedia.org/wikipedia/commons/thumb/d/de/PolygonsSet_EN.svg/220px-PolygonsSet_EN.svg.png" srcset="//upload.wikimedia.org/wikipedia/commons/thumb/d/de/PolygonsSet_EN.svg/330px-PolygonsSet_EN.svg.png 1.5x, //upload.wikimedia.org/wikipedia/commons/thumb/d/de/PolygonsSet_EN.svg/440px-PolygonsSet_EN.svg.png 2x" width="220"/></a>
<div class="thumbcaption">
<div class="magnify"><a class="internal" href="/wiki/File:PolygonsSet_EN.svg" title="Enlarge"></a></div>
The regular polygons form a subset of the polygons</div>
</div>
</div>
<ul>
<li>The set A = {1, 2} is a proper subset of B = {1, 2, 3}, thus both expressions A ⊆ B and A ⊊ B are true.</li>
<li>The set D = {1, 2, 3} is a subset of E = {1, 2, 3}, thus D ⊆ E is true, and D ⊊ E is not true (false).</li>
<li>Any set is a subset of itself, but not a proper subset. (X ⊆ X is true, and X ⊊ X is false for any set X.)</li>
<li>The <a href="/wiki/Empty_set" title="Empty set">empty set</a> { }, denoted by ∅, is also a subset of any given set <i>X</i>. It is also always a proper subset of any set except itself.</li>
<li>The set {<i>x</i>: <i>x</i> is a <a href="/wiki/Prime_number" title="Prime number">prime number</a> greater than 10} is a proper subset of {<i>x</i>: <i>x</i> is an odd number greater than 10}</li>
<li>The set of <a href="/wiki/Natural_number" title="Natural number">natural numbers</a> is a proper subset of the set of <a href="/wiki/Rational_number" title="Rational number">rational numbers</a>; likewise, the set of points in a <a href="/wiki/Line_segment" title="Line segment">line segment</a> is a proper subset of the set of points in a <a class="mw-redirect" href="/wiki/Line_(mathematics)" title="Line (mathematics)">line</a>. These are two examples in which both the subset and the whole set are infinite, and the subset has the same <a href="/wiki/Cardinality#Infinite_sets" title="Cardinality">cardinality</a> (the concept that corresponds to size, that is, the number of elements, of a finite set) as the whole; such cases can run counter to one's initial intuition.</li>
<li>The set of rational numbers is a proper subset of the set of <a href="/wiki/Real_number" title="Real number">real numbers</a>. In this example, both sets are infinite but the latter set has a larger cardinality (or <i>power</i>) than the former set.</li>
</ul>

<h2><span class="mw-headline" id="Other_properties_of_inclusion">Other properties of inclusion</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Subset&amp;action=edit&amp;section=5" title="Edit section: Other properties of inclusion">edit</a><span class="mw-editsection-bracket">]</span></span></h2>
<div class="thumb tright">
<div class="thumbinner" style="width:222px;"><a class="image" href="/wiki/File:Subset_with_expansion.svg"><img alt="" class="thumbimage" data-file-height="247" data-file-width="470" height="116" src="//upload.wikimedia.org/wikipedia/commons/thumb/d/df/Subset_with_expansion.svg/220px-Subset_with_expansion.svg.png" srcset="//upload.wikimedia.org/wikipedia/commons/thumb/d/df/Subset_with_expansion.svg/330px-Subset_with_expansion.svg.png 1.5x, //upload.wikimedia.org/wikipedia/commons/thumb/d/df/Subset_with_expansion.svg/440px-Subset_with_expansion.svg.png 2x" width="220"/></a>
<div class="thumbcaption">
<div class="magnify"><a class="internal" href="/wiki/File:Subset_with_expansion.svg" title="Enlarge"></a></div>
A ⊆ B and B ⊆ C imply A ⊆ C</div>
</div>
</div>
<p>Inclusion is the canonical <a class="mw-redirect" href="/wiki/Partial_order" title="Partial order">partial order</a> in the sense that every partially ordered set (<i>X</i>, <span><span class="mwe-math-mathml-inline mwe-math-mathml-a11y" style="display: none;"><math xmlns="http://www.w3.org/1998/Math/MathML">
<semantics>
<mrow class="MJX-TeXAtom-ORD">
<mstyle displaystyle="true" scriptlevel="0">
<mo>⪯<!-- ⪯ --></mo>
</mstyle>
</mrow>
<annotation encoding="application/x-tex">{\displaystyle \preceq }</annotation>
</semantics>
</math></span><img alt="\preceq " aria-hidden="true" class="mwe-math-fallback-image-inline" src="https://wikimedia.org/api/rest_v1/media/math/render/svg/63dfe475e1377b3b4e936a3aa8fb1d7177dcdbc3" style="vertical-align: -0.671ex; width:1.819ex; height:2.509ex;"/></span>) is <a class="mw-redirect" href="/wiki/Isomorphic" title="Isomorphic">isomorphic</a> to some collection of sets ordered by inclusion. The <a href="/wiki/Ordinal_number" title="Ordinal number">ordinal numbers</a> are a simple example—if each ordinal <i>n</i> is identified with the set [<i>n</i>] of all ordinals less than or equal to <i>n</i>, then <i>a</i> ≤ <i>b</i> if and only if [<i>a</i>] ⊆ [<i>b</i>].</p>
<p>For the <a href="/wiki/Power_set" title="Power set">power set</a> <span><span class="mwe-math-mathml-inline mwe-math-mathml-a11y" style="display: none;"><math xmlns="http://www.w3.org/1998/Math/MathML">
<semantics>
<mrow class="MJX-TeXAtom-ORD">
<mstyle displaystyle="true" scriptlevel="0">
<mrow class="MJX-TeXAtom-ORD">
<mrow class="MJX-TeXAtom-ORD">
<mi class="MJX-tex-caligraphic" mathvariant="script">P</mi>
</mrow>
</mrow>
<mo stretchy="false">(</mo>
<mi>S</mi>
<mo stretchy="false">)</mo>
</mstyle>
</mrow>
<annotation encoding="application/x-tex">{\displaystyle {\mathcal {P}}(S)}</annotation>
</semantics>
</math></span><img alt="{\mathcal {P}}(S)" aria-hidden="true" class="mwe-math-fallback-image-inline" src="https://wikimedia.org/api/rest_v1/media/math/render/svg/3b51bfe21fd15a7b5531fd9635a87a3863be6025" style="vertical-align: -0.838ex; width:5.054ex; height:2.843ex;"/></span> of a set <i>S</i>, the inclusion partial order is (up to an <a href="/wiki/Order_isomorphism" title="Order isomorphism">order isomorphism</a>) the <a href="/wiki/Cartesian_product" title="Cartesian product">Cartesian product</a> of <i>k</i> = |<i>S</i>| (the <a href="/wiki/Cardinality" title="Cardinality">cardinality</a> of <i>S</i>) copies of the partial order on {0,1} for which 0 &lt; 1. This can be illustrated by enumerating <i>S</i> = {<i>s</i><sub>1</sub>, <i>s</i><sub>2</sub>, …, <i>s</i><sub><i>k</i></sub>} and associating with each subset <i>T</i> ⊆ <i>S</i> (which is to say with each element of 2<sup><i>S</i></sup>) the <i>k</i>-tuple from {0,1}<sup><i>k</i></sup> of which the <i>i</i>th coordinate is 1 if and only if <i>s</i><sub><i>i</i></sub> is a member of <i>T</i>.</p>
