<div class="mw-content-ltr" dir="ltr" id="mw-content-text" lang="en"><script>function mfTempOpenSection(id){var block=document.getElementById("mf-section-"+id);block.className+=" open-block";block.previousSibling.className+=" open-block";}</script><p>An <b>encoder</b> is a device, circuit, transducer, software program, algorithm or person that <a class="mw-redirect" href="/wiki/Encoding" title="Encoding">converts</a> information from one format or <a href="/wiki/Code" title="Code">code</a> to another, for the purposes of standardization, speed or compressions.</p>

<h2><span class="mw-headline" id="Examples">Examples</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Encoder&amp;action=edit&amp;section=1" title="Edit section: Examples">edit</a><span class="mw-editsection-bracket">]</span></span></h2>
<h3><span class="mw-headline" id="Media">Media</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Encoder&amp;action=edit&amp;section=2" title="Edit section: Media">edit</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>Software for encoding <a href="/wiki/Comparison_of_audio_coding_formats" title="Comparison of audio coding formats">audio</a>, <a href="/wiki/Comparison_of_video_codecs" title="Comparison of video codecs">video</a>, <a href="/wiki/Character_encoding" title="Character encoding">text</a> into standardized formats:</p>
<ul>
<li>A <a href="/wiki/Data_compression" title="Data compression">compressor</a> encodes data (e.g., audio/video/images) into a smaller form (see <a href="/wiki/Codec" title="Codec">codec</a>)</li>
<li>An <a class="mw-redirect" href="/wiki/Audio_encoder" title="Audio encoder">audio encoder</a> converts analog audio to digital audio signals</li>
<li>A <a class="mw-redirect" href="/wiki/Video_encoder" title="Video encoder">video encoder</a> converts analog video to digital video signals</li>
<li>An email encoder secures online email addresses from email harvesters</li>
<li>A PHTML encoder preserves script code logic in a secure format that is transparent to visitors on a web site</li>
<li>A <a href="/wiki/Multiplexer" title="Multiplexer">multiplexer</a> combines multiple inputs into one output</li>
<li>8b/10b encoder used for fast speed in communication system</li>
</ul>
<h3><span class="mw-headline" id="Job_positions">Job positions</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Encoder&amp;action=edit&amp;section=3" title="Edit section: Job positions">edit</a><span class="mw-editsection-bracket">]</span></span></h3>
<ul>
<li>A Data Entry Encoder may enter data from phone surveys in a coded format into a database.</li>
<li>A Data Entry Encoder may enter payment amounts from legal tender documents from financial institutions into a database.</li>
<li>A Manual Encoder may manually scan code tags on baggage that were missed by an automated system.</li>
</ul>
<h3><span class="mw-headline" id="Medical_encoding_software">Medical encoding software</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Encoder&amp;action=edit&amp;section=4" title="Edit section: Medical encoding software">edit</a><span class="mw-editsection-bracket">]</span></span></h3>
<ul>
<li>EncoderPro searches <a href="/wiki/International_Statistical_Classification_of_Diseases_and_Related_Health_Problems#ICD-9-CM" title="International Statistical Classification of Diseases and Related Health Problems">ICD-9-CM</a>, <a href="/wiki/Current_Procedural_Terminology" title="Current Procedural Terminology">CPT</a>, and <a href="/wiki/Healthcare_Common_Procedure_Coding_System" title="Healthcare Common Procedure Coding System">HCPCS</a> Level II medical codes, to increase accuracy and allow ease of auditing for compliance.</li>
</ul>
<h3><span class="mw-headline" id="Transducers">Transducers</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Encoder&amp;action=edit&amp;section=5" title="Edit section: Transducers">edit</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>Transducers (such as optical or magnetic encoders) sense position or orientation for use as a reference or active feedback to control position:</p>
<ul>
<li>A <a href="/wiki/Rotary_encoder" title="Rotary encoder">rotary encoder</a> converts rotary position to an analog (e.g., analog quadrature) or digital (e.g., digital quadrature, 32-bit parallel, or USB) electronic signal.</li>
<li>A <a href="/wiki/Linear_encoder" title="Linear encoder">linear encoder</a> similarly converts linear position to an electronic signal.</li>
</ul>
<p>Such encoders can be either absolute or incremental. The signal from an absolute encoder gives an unambiguous position within the travel range without requiring knowledge of any previous position. The signal from an incremental encoder is cyclical, thus ambiguous, and requires counting of cycles to maintain absolute position within the travel range. Both can provide the same accuracy, but the absolute encoder is more robust to interruptions in transducer signal.</p>
<h3><span class="mw-headline" id="Telecommunications">Telecommunications</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Encoder&amp;action=edit&amp;section=6" title="Edit section: Telecommunications">edit</a><span class="mw-editsection-bracket">]</span></span></h3>
<ul>
<li>A <a href="/wiki/Integrated_circuit" title="Integrated circuit">device</a> used to change a <a class="mw-redirect" href="/wiki/Signalling_(telecommunication)" title="Signalling (telecommunication)">signal</a> (such as a <a href="/wiki/Bitstream" title="Bitstream">bitstream</a>) or <a href="/wiki/Data" title="Data">data</a> into a <a href="/wiki/Code" title="Code">code</a>.</li>
</ul>
<h3><span class="mw-headline" id="Electronic_circuits">Electronic circuits</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Encoder&amp;action=edit&amp;section=7" title="Edit section: Electronic circuits">edit</a><span class="mw-editsection-bracket">]</span></span></h3>
<ul>
<li>A <a href="/wiki/Priority_encoder#Simple_encoder" title="Priority encoder">simple encoder</a> assigns a binary code to an active input line.</li>
<li><a href="/wiki/Priority_encoder" title="Priority encoder">Priority encoders</a> establish the priority of competing inputs (such as interrupt requests) by outputting a binary code representing the highest-priority active input.</li>
<li>For producing n no. of output when there is 2^n no. of inputs.</li>
</ul>
