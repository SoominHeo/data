<p>In <a href="/wiki/Unix" title="Unix">Unix</a> computing, <b>Blackbox</b> is a <a href="/wiki/Stacking_window_manager" title="Stacking window manager">stacking window manager</a> for the <a href="/wiki/X_Window_System" title="X Window System">X Window System</a>.</p>
<p>Blackbox has specific design goals, and some functionality is provided only through other applications. One example is the bbkeys <a href="/wiki/Keyboard_shortcut" title="Keyboard shortcut">hotkey</a> application. As no release has been made for over eight years, Blackbox is now dormant.</p>
<p>Blackbox is written in <a href="/wiki/C%2B%2B" title="C++">C++</a><sup class="reference" id="cite_ref-dir-fsf_4-1"><a href="#cite_note-dir-fsf-4">[4]</a></sup><sup class="reference" id="cite_ref-5"><a href="#cite_note-5">[5]</a></sup> and contains completely original code.<sup class="reference" id="cite_ref-6"><a href="#cite_note-6">[6]</a></sup> It was created by Bradley T. Hughes<sup class="reference" id="cite_ref-sourceforge1_1-1"><a href="#cite_note-sourceforge1-1">[1]</a></sup> and is available under the <a href="/wiki/MIT_License" title="MIT License">MIT License</a>. Blackbox has compliance with the <a href="/wiki/Extended_Window_Manager_Hints" title="Extended Window Manager Hints">Extended Window Manager Hints</a> specification.<sup class="reference" id="cite_ref-7"><a href="#cite_note-7">[7]</a></sup></p>
<p>Blackbox development is no longer active.</p>

<h2><span class="mw-headline" id="Features">Features</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Blackbox&amp;action=edit&amp;section=1" title="Edit section: Features">edit</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>Features of the Blackbox window manager include:</p>
<ul>
<li>A stacking window manager</li>
<li>Written in <a href="/wiki/C%2B%2B" title="C++">C++</a></li>
<li>Freely available under the <a class="mw-redirect" href="/wiki/MIT_license" title="MIT license">MIT license</a></li>
<li>Compliance with the Extended Window Manager Hints Specification</li>
<li>Titlebars have minimize, <a class="mw-redirect" href="/wiki/Maximize" title="Maximize">maximize</a>, and close buttons</li>
<li>Support for simple themes</li>
<li>Support for color changes</li>
<li>No support for desktop shortcuts</li>
</ul>
