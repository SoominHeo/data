<p>The <b>Black Screen of Death</b> (<b>BkSoD</b><sup class="reference" id="cite_ref-1"><a href="#cite_note-1">[1]</a></sup>) is an error screen displayed by some operating systems after encountering a critical system error which can cause the system to shut down.</p>

<h2><span class="mw-headline" id="The_Windows_Black_Screen_of_Death">The Windows Black Screen of Death</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Black_Screen_of_Death&amp;action=edit&amp;section=1" title="Edit section: The Windows Black Screen of Death">edit</a><span class="mw-editsection-bracket">]</span></span></h2>
<h3><span class="mw-headline" id="Windows_3.x">Windows 3.x</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Black_Screen_of_Death&amp;action=edit&amp;section=2" title="Edit section: Windows 3.x">edit</a><span class="mw-editsection-bracket">]</span></span></h3>
<div class="thumb tright">
<div class="thumbinner" style="width:222px;"><a class="image" href="/wiki/File:EMM386.PNG"><img alt="" class="thumbimage" data-file-height="63" data-file-width="505" height="27" src="//upload.wikimedia.org/wikipedia/commons/thumb/c/cd/EMM386.PNG/220px-EMM386.PNG" srcset="//upload.wikimedia.org/wikipedia/commons/thumb/c/cd/EMM386.PNG/330px-EMM386.PNG 1.5x, //upload.wikimedia.org/wikipedia/commons/thumb/c/cd/EMM386.PNG/440px-EMM386.PNG 2x" width="220"/></a>
<div class="thumbcaption">
<div class="magnify"><a class="internal" href="/wiki/File:EMM386.PNG" title="Enlarge"></a></div>
An example of an EMM386 Error Message in Windows 3.0 that results in a BkSOD during Standard Mode.</div>
</div>
</div>
<p>In <a href="/wiki/Windows_3.1x" title="Windows 3.1x">Windows 3.x</a> the <b>black screen of death</b> is the behaviour that occurred when a DOS-based application failed to execute properly.<sup class="noprint Inline-Template Template-Fact" style="white-space:nowrap;">[<i><a href="/wiki/Wikipedia:Citation_needed" title="Wikipedia:Citation needed"><span title="This claim needs references to reliable sources. (October 2011)">citation needed</span></a></i>]</sup> It was often known to occur in connection with attempting certain operations while networking drivers were resident in memory. (Commonly, but not exclusively, it was seen while the <a class="mw-redirect" href="/wiki/Novell_NetWare" title="Novell NetWare">Novell NetWare</a> client for <a href="/wiki/DOS" title="DOS">DOS</a>, NETX, was loaded.)<sup class="noprint Inline-Template Template-Fact" style="white-space:nowrap;">[<i><a href="/wiki/Wikipedia:Citation_needed" title="Wikipedia:Citation needed"><span title="This claim needs references to reliable sources. (October 2011)">citation needed</span></a></i>]</sup></p>
<p>According to Wallace McClure of ASP.net, the phrase was originally coined in mid-1991 by Ed Brown, a technician with <a href="/wiki/The_Coca-Cola_Company" title="The Coca-Cola Company">The Coca-Cola Company</a>'s IT department in <a href="/wiki/Atlanta" title="Atlanta">Atlanta</a>, GA. He reports that the company was rolling out <a href="/wiki/Windows_3.0" title="Windows 3.0">Windows 3.0</a> within the Global Marketing group and when the users would attempt to run <a href="/wiki/WordPerfect" title="WordPerfect">WordPerfect</a>, they would randomly receive a BkSOD.<sup class="reference" id="cite_ref-2"><a href="#cite_note-2">[2]</a></sup></p>
<h3><span class="mw-headline" id="Later_versions_of_Windows">Later versions of Windows</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Black_Screen_of_Death&amp;action=edit&amp;section=3" title="Edit section: Later versions of Windows">edit</a><span class="mw-editsection-bracket">]</span></span></h3>
<p><a href="/wiki/MS-DOS" title="MS-DOS">MS-DOS</a>, <a href="/wiki/Windows_95" title="Windows 95">Windows 95</a>, <a href="/wiki/Windows_98" title="Windows 98">Windows 98</a>, <a href="/wiki/Windows_2000" title="Windows 2000">Windows 2000</a>, <a href="/wiki/Windows_ME" title="Windows ME">Windows ME</a>, <a href="/wiki/Windows_XP" title="Windows XP">Windows XP</a>, <a href="/wiki/Windows_Vista" title="Windows Vista">Windows Vista</a>, <a href="/wiki/Windows_7" title="Windows 7">Windows 7</a>, and <a href="/wiki/Windows_8" title="Windows 8">Windows 8</a> also display a Black Screen of Death when the operating system cannot boot. This is usually due to a missing file.<sup class="reference" id="cite_ref-3"><a href="#cite_note-3">[3]</a></sup> This also happens when the user enables file compression on all files and the operating system compresses. Often the user must reinstall Windows, if the missing file is critical to the boot process. However, more often than not the boot screen will inform the user of the missing file.<sup class="noprint Inline-Template Template-Fact" style="white-space:nowrap;">[<i><a href="/wiki/Wikipedia:Citation_needed" title="Wikipedia:Citation needed"><span title="This claim needs references to reliable sources. (October 2011)">citation needed</span></a></i>]</sup> If the operating system is compressed, it will not be able to boot, even into safe mode.<sup class="reference" id="cite_ref-4"><a href="#cite_note-4">[4]</a></sup><sup class="reference" id="cite_ref-5"><a href="#cite_note-5">[5]</a></sup> However, booting from another device and uncompressing the files will usually solve the issue.</p>
<p>In late 2009 several new reports of the Black Screen of Death in <a href="/wiki/Windows_XP" title="Windows XP">Windows XP</a>, <a href="/wiki/Windows_Vista" title="Windows Vista">Windows Vista</a>, and <a href="/wiki/Windows_7" title="Windows 7">Windows 7</a> emerged. At first several claims pointed at a <a href="/wiki/Windows_Update" title="Windows Update">Windows Update</a>. It was later recanted by <a href="/wiki/Prevx" title="Prevx">Prevx</a> as an erroneous report.<sup class="reference" id="cite_ref-6"><a href="#cite_note-6">[6]</a></sup><sup class="reference" id="cite_ref-7"><a href="#cite_note-7">[7]</a></sup> Microsoft reported that no security update was causing the issue, and it may be tied to <a href="/wiki/Malware" title="Malware">malware</a>.<sup class="reference" id="cite_ref-8"><a href="#cite_note-8">[8]</a></sup><sup class="reference" id="cite_ref-9"><a href="#cite_note-9">[9]</a></sup> In other cases, the black screen was replaced with the <a href="/wiki/Blue_Screen_of_Death" title="Blue Screen of Death">Blue Screen of Death</a>. The Black Screen of Death may also be caused by certain components of a computer overheating.</p>
<p>Early builds of <a href="/wiki/Windows_8" title="Windows 8">Windows 8</a> featured a black screen of death, in place of the traditional blue screen, that appeared to indicate a <a class="mw-redirect" href="/wiki/Stop_error" title="Stop error">Stop error</a>. This black screen was simplified compared to the previous blue screen, omitting instructions that the user is recommended to take.<sup class="reference" id="cite_ref-10"><a href="#cite_note-10">[10]</a></sup></p>
