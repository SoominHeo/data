<p><b>BlueDragon</b> is a <a href="/wiki/ColdFusion_Markup_Language" title="ColdFusion Markup Language">ColdFusion Markup Language</a> (CFML) engine comparable to <a href="/wiki/Adobe_Systems" title="Adobe Systems">Adobe Systems</a>'s <a class="mw-redirect" href="/wiki/ColdFusion" title="ColdFusion">ColdFusion</a>. It is licensed and distributed by <a href="/wiki/New_Atlanta" title="New Atlanta">New Atlanta</a> from TagServlet Ltd based in Scotland. BlueDragon is also distributed and supported by <a href="/wiki/BEA_Systems" title="BEA Systems">BEA Systems</a> on their <a class="mw-redirect" href="/wiki/Oracle_Weblogic_Server" title="Oracle Weblogic Server">Oracle WebLogic Server</a> server platform.</p>
<p>BlueDragon applications run on a variety of platforms, including <a href="/wiki/Microsoft_Windows" title="Microsoft Windows">Microsoft Windows</a>, <a href="/wiki/Linux" title="Linux">Linux</a>, and <a class="mw-redirect" href="/wiki/Mac_OS_X" title="Mac OS X">Mac OS X</a>. It is mostly compatible with ColdFusion MX 7.</p>
<p>In March, 2008 New Atlanta announced that the future versions of BlueDragon (Java EE editions) would be released as open source products.<sup class="reference" id="cite_ref-1"><a href="#cite_note-1">[1]</a></sup> The open source version was released as <b>Open BlueDragon</b> (<b>OpenBD</b>) in December 2008. <sup class="reference" id="cite_ref-2"><a href="#cite_note-2">[2]</a></sup></p>

<h2><span class="mw-headline" id="Editions">Editions</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=BlueDragon&amp;action=edit&amp;section=1" title="Edit section: Editions">edit</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>BlueDragon is available in six editions:</p>
<ul>
<li>BlueDragon Server</li>
<li>BlueDragon Server JX (similar to ColdFusion standalone editions)</li>
<li>BlueDragon for J2EE Application Servers (BD J2EE)</li>
<li>BlueDragon for the Microsoft .NET Framework (BD .NET)</li>
<li>BlueDragon, BEA WebLogic Edition (sold by BEA as a BEA product)</li>
<li><a class="external text" href="http://www.openbd.org" rel="nofollow">Open BlueDragon</a>, an open source version of BlueDragon for J2EE.</li>
</ul>
<p>The first two editions are standalone servers which run on Windows, Linux, and OS X. With BD J2EE, CFML applications can be deployed on any J2EE server, enabling integration of CFML and J2EE as enabled in ColdFusion MX. BD, BEA WebLogic Edition, is a special edition based on BD J2EE for use with BEA's WebLogic Server. BD.NET extends the Microsoft .NET framework and IIS to permit deployment of CFML applications as native MS .NET web applications, offering integration between CFML and ASP.NET that's not possible with ColdFusion.</p>
<p>The Server JX, J2EE, MS .NET, and BEA WebLogic editions of BlueDragon are commercial products which are available as 30-day unlimited trials which convert to a single-IP developer edition with no expiration.</p>
<p>The Server edition, on the other hand, is free for deployment—though not for hosting, redeployment, or (as of the 6.2 release) commercial use. There are no differences in CFML tags supported in the free Server edition, but it supports only ODBC drivers on MS Windows (and only MySQL or PostgreSQL on Linux and Mac OS X), it supports only IIS on MS Windows or Apache on Linux or Mac OS X, and does not support secured (SSL) connections.</p>
<p>None of the limitations of the free Server edition exist in the commercial Server JX, J2EE, .NET, or BEA WebLogic editions.</p>
<p>The .NET edition of BlueDragon runs on Microsoft's <a class="mw-redirect" href="/wiki/Microsoft_.NET" title="Microsoft .NET">.NET</a> platform, BD.NET enables CFML applications to leverage the .NET platform and allows for integration between CFML and ASP.NET as well as .NET objects.</p>
<p>Open BlueDragon is an open source version of BlueDragon is released under the GNU General Public License version 3 (GPLv3). The chief differences between the open source and the J2EE version are the removal of commercial libraries (e.g., for PDF generation), The JTurbo JDBC driver for Microsoft SQL Server, and the BlueDragon Administrator application. It runs on any standard J2EE application server, such as Tomcat, JBoss or Jetty.</p>
<p>The original version of BlueDragon was released in 2002.</p>
<h2><span class="mw-headline" id="Corporate_adoption">Corporate adoption</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=BlueDragon&amp;action=edit&amp;section=2" title="Edit section: Corporate adoption">edit</a><span class="mw-editsection-bracket">]</span></span></h2>
<p><a class="mw-redirect" href="/wiki/MySpace" title="MySpace">MySpace</a> used the .NET version of BlueDragon to power some of its online applications as of 2007.<sup class="reference" id="cite_ref-3"><a href="#cite_note-3">[3]</a></sup><sup class="reference" id="cite_ref-4"><a href="#cite_note-4">[4]</a></sup></p>
<h2><span class="mw-headline" id="Compatibility">Compatibility</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=BlueDragon&amp;action=edit&amp;section=3" title="Edit section: Compatibility">edit</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>Though BlueDragon 7.0 was designed to be compatible with Adobe ColdFusion MX 7.0.2,<sup class="reference" id="cite_ref-5"><a href="#cite_note-5">[5]</a></sup> there are differences in the two CFML implementations. BlueDragon offers several advantages (tags, functions, and other functionality) not found in ColdFusion. Similarly, there are a few tags and functions found in ColdFusion that are not supported currently in BlueDragon. New Atlanta maintains a complete list of incompatibilities with Adobe ColdFusion MX in the documentation.</p>
<h3><span class="mw-headline" id="Framework_compatibility">Framework compatibility</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=BlueDragon&amp;action=edit&amp;section=4" title="Edit section: Framework compatibility">edit</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>A number of popular ColdFusion frameworks are fully supported on BlueDragon:</p>
<ul>
<li><a class="mw-redirect" href="/wiki/ColdSpring" title="ColdSpring">ColdSpring</a></li>
<li><a href="/wiki/Fusebox_(programming)" title="Fusebox (programming)">Fusebox</a></li>
<li><a href="/wiki/Model-Glue" title="Model-Glue">Model-Glue</a></li>
<li><a href="/wiki/Mach-II" title="Mach-II">Mach-II</a></li>
</ul>
