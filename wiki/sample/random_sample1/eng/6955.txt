<h1 class="firstHeading" id="firstHeading" lang="en">VIA Envy</h1>
<div class="mw-body-content" id="bodyContent">

<p>The <b>VIA Envy24</b> audio chipset series by <a href="/wiki/VIA_Technologies" title="VIA Technologies">VIA Technologies</a> delivers true <a href="/wiki/24-bit" title="24-bit">24-bit</a> sound for personal computers. While available as a discrete card, it is sold in greatest volume as an integrated solution for motherboards.</p>
<p>The Envy24 chipset was originally developed by IC Ensemble Inc (PCI vendor ID 1412, established 1997) as ICE1712. The company was acquired by VIA in late 2000.</p>
<p>A total of six controllers exists in the Envy range. Since the demise of the <a href="/wiki/SoundStorm" title="SoundStorm">SoundStorm</a> audio solution, the VIA Envy has become an option as a replacement part for integrated audio on nVidia motherboards, along with cheaper <a href="/wiki/Realtek" title="Realtek">Realtek</a> and <a href="/wiki/SigmaTel" title="SigmaTel">SigmaTel</a> offerings.</p>
<p>With the exception of the Envy24 controller, Envy chipsets lack hardware 3D positional sound, which means CPU usage across various scenarios is noticeably higher than either <a href="/wiki/SoundStorm" title="SoundStorm">SoundStorm</a> or <a class="mw-redirect" href="/wiki/Audigy" title="Audigy">Audigy</a> based solutions. <a href="/wiki/VIA_Technologies" title="VIA Technologies">VIA</a> has stated it has no intention of implementing CPU offload in future hardware, due to the increased manufacturing costs.</p>
<h2><span class="mw-headline" id="Features">Features</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=VIA_Envy&amp;action=edit&amp;section=1" title="Edit section: Features">edit</a><span class="mw-editsection-bracket">]</span></span></h2>
<ul>
<li>24-bit resolution audio format support (Envy24/Envy24GT/Envy24HT/Envy24HT-S/Envy24MT) / 20-bit resolution audio format support (Envy24PT/Envy24HT-S)</li>
<li>Digital <a class="mw-redirect" href="/wiki/Sampling_rate" title="Sampling rate">sampling rates</a> up to 192 kHz (Envy24HT/Envy24GT/Envy24HT-S/Envy24MT) / 96 kHz (Envy24/Envy24PT)</li>
<li>Analog <a class="mw-redirect" href="/wiki/Sampling_rate" title="Sampling rate">sampling rates</a> up to 192 kHz (Envy24HT/Envy24HT-S/Envy24MT) / 96 kHz (Envy24/Envy24GT) / 48 kHz (Envy24HT-S/Envy24PT)</li>
<li>Up to 8 outbound streams and 8 simultaneous input streams (depending on the model)</li>
<li><a href="/wiki/Surround_sound" title="Surround sound">Surround sound</a> audio: 5.1, 6.1, and 7.1</li>
<li>36 bit wide hardware mixer (Envy24 only)</li>
<li>DirectSound hardware acceleration (Envy24 only)</li>
</ul>
<p>Note that Envy24HT-S supports 24-bit / 192 kHz mode only in 4 channel output (4 channel input) mode, while in 8 channel output (2 channel input) mode it is limited to 20-bit / 48 kHz.</p>
