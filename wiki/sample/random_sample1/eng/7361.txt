<p>The <b>Samsung NX100</b> is a 14.0 <a class="mw-redirect" href="/wiki/Effective_pixels" title="Effective pixels">effective</a> <a class="mw-redirect" href="/wiki/Megapixel" title="Megapixel">megapixel</a> <a href="/wiki/APS-C" title="APS-C">APS-C</a> <a href="/wiki/Crop_factor" title="Crop factor">crop</a> <a href="/wiki/CMOS" title="CMOS">CMOS</a> <a class="mw-redirect" href="/wiki/Mirrorless_interchangeable_lens_camera" title="Mirrorless interchangeable lens camera">mirrorless interchangeable lens</a> <a href="/wiki/Digital_camera" title="Digital camera">digital camera</a> made by <a href="/wiki/Samsung" title="Samsung">Samsung</a>. It was announced on September 14, 2010.<sup class="reference" id="cite_ref-1"><a href="#cite_note-1">[1]</a></sup></p>

<h2><span class="mw-headline" id="Changes_from_NX10">Changes from NX10</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Samsung_NX100&amp;action=edit&amp;section=1" title="Edit section: Changes from NX10">edit</a><span class="mw-editsection-bracket">]</span></span></h2>
<ul>
<li>No built-in EVF or built-in flash. Instead, the hot-shoe supports:
<ul>
<li>Optional external <a href="/wiki/Flash_(photography)" title="Flash (photography)">flashes</a></li>
<li>Optional <a class="mw-redirect" href="/wiki/GPS" title="GPS">GPS</a> adapter</li>
<li>Optional EVF (23g, 320x240 (QVGA), 0.83x magnification, 98% coverage)<sup class="reference" id="cite_ref-2"><a href="#cite_note-2">[2]</a></sup></li>
</ul>
</li>
<li>Samsung i-Function compatibility at launch (was added to NX10 in a firmware upgrade)</li>
<li>The kit lens is a smaller, non-OIS, collapsing 20-50mm iFn lens</li>
<li>Lens Priority mode, which adjusts settings and menus depending on which lens is attached</li>
</ul>
