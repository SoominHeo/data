<p>The <b>Samsung Galaxy S III Mini</b> (<b>GT-I8190</b>) is a <a href="/wiki/Touchscreen" title="Touchscreen">touchscreen</a>-based, <a class="mw-redirect" href="/wiki/Slate_phone" title="Slate phone">slate</a>-sized <a href="/wiki/Smartphone" title="Smartphone">smartphone</a> designed and manufactured by <a href="/wiki/Samsung" title="Samsung">Samsung</a>. It was announced in October 2012 and released in November 2012. The Galaxy S III Mini technological specifications include a 4-inch <a class="mw-redirect" href="/wiki/Super_AMOLED" title="Super AMOLED">Super AMOLED</a> display, a <a href="/wiki/Multi-core_processor" title="Multi-core processor">dual-core</a> processor running at 1 <a href="/wiki/Hertz" title="Hertz">GHz</a> with 1 <a href="/wiki/Gibibyte" title="Gibibyte">GB</a> of <a href="/wiki/Random-access_memory" title="Random-access memory">RAM</a>, a 5-megapixel rear camera, and a front-facing <a class="mw-redirect" href="/wiki/VGA" title="VGA">VGA</a> camera for <a class="mw-redirect" href="/wiki/Video_call" title="Video call">video calls</a> or selfies.<sup class="reference" id="cite_ref-gsmarena_3-0"><a href="#cite_note-gsmarena-3">[3]</a></sup><sup class="reference" id="cite_ref-4"><a href="#cite_note-4">[4]</a></sup></p>
<p>Samsung Galaxy S III Mini is a smaller version of the <a href="/wiki/Samsung_Galaxy_S_III" title="Samsung Galaxy S III">Samsung Galaxy S III</a> and contains many of the same features. However, it lacks an 8-megapixel rear camera, bigger 4.8-inch screen and Gorilla Glass display. This device has a 42 <a href="/wiki/Hour" title="Hour">h</a> endurance rating.<sup class="reference" id="cite_ref-gsmarena2_5-0"><a href="#cite_note-gsmarena2-5">[5]</a></sup> The device initially ran on Android 4.1.1 (Jelly Bean) but has now been updated to 4.1.2.<sup class="reference" id="cite_ref-gsmarena2_5-1"><a href="#cite_note-gsmarena2-5">[5]</a></sup> Other features include Samsung's TouchWiz Skin, ChatON instant messaging, Smart alert, Buddy Photo Share, Pop-Up Play, S Suggest, S Voice, Smart Stay, Video Hub, Game Hub 2.0, and the S Beam technology (S Beam only available on special <a href="/wiki/Near_field_communication" title="Near field communication">NFC</a> edition, GT-I8190N). This device's battery varies from different carriers, it will either have 3-pin battery which is widely used and sold or it will have a 4-pin battery.</p>

<h2><span class="mw-headline" id="Other_variants">Other variants</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Samsung_Galaxy_S_III_Mini&amp;action=edit&amp;section=1" title="Edit section: Other variants">edit</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>An alternative 'upgraded' Galaxy S III Mini version (SM-G730A) is available from the US carrier AT&amp;T, which features a different processor (Qualcomm 1.2 GHz dual-core Snapdragon 400), NFC, 2000 mAh battery, and 4G LTE data speeds, among other differences. It retains the same 4-inch (800×480 pixels), Super AMOLED screen as the original 'international' Galaxy S III Mini. As of June 6, 2014, the AT&amp;T<sup class="reference" id="cite_ref-6"><a href="#cite_note-6">[6]</a></sup> website shows this version of the phone shipping with Android 4.4 KitKat installed. On September 23, 2015, an update was released, but it is unknown if it did anything.</p>
<p>In March 2014, another variant of this device (GT-I8200) was released in some countries, with the GT-I8200N model featuring NFC and the GT-I8200L model in Latin America. This variant features a different SoC (<a href="/wiki/Marvell_Technology_Group" title="Marvell Technology Group">Marvell</a> PXA986, dual-core) clocked at 1.2 GHz, with the <a href="/wiki/Vivante_Corporation" title="Vivante Corporation">Vivante</a> GC1000 GPU, those specs were used in the <a href="/wiki/Samsung_Galaxy_Tab_3_7.0" title="Samsung Galaxy Tab 3 7.0">Galaxy Tab 3 7.0</a> as well. The GT-I8200/N/L variants are shipped with Android 4.2.2 Jelly Bean out of the box, as opposed to the older GT-I8190 variant which was shipped with 4.1.1.<sup class="reference" id="cite_ref-7"><a href="#cite_note-7">[7]</a></sup></p>
