<p>The <b>Samsung Galaxy Note Pro 12.2</b> is a 12.2-inch <a href="/wiki/Android_(operating_system)" title="Android (operating system)">Android</a>-based <a href="/wiki/Tablet_computer" title="Tablet computer">tablet computer</a> produced and marketed by <a href="/wiki/Samsung_Electronics" title="Samsung Electronics">Samsung Electronics</a>.<sup class="reference" id="cite_ref-announce_4-0"><a href="#cite_note-announce-4">[4]</a></sup> It belongs to the new generation of the <a href="/wiki/Samsung_Galaxy_Note_series" title="Samsung Galaxy Note series">Samsung Galaxy Note series</a> and <b>Pro</b> tablets, which also includes an 8.4-inch model, the <a href="/wiki/Samsung_Galaxy_Tab_Pro_8.4" title="Samsung Galaxy Tab Pro 8.4">Samsung Galaxy Tab Pro 8.4</a>, a 10.1-inch model, the <a href="/wiki/Samsung_Galaxy_Tab_Pro_10.1" title="Samsung Galaxy Tab Pro 10.1">Samsung Galaxy Tab Pro 10.1</a>, and another 12.2 inch model, the <a href="/wiki/Samsung_Galaxy_Tab_Pro_12.2" title="Samsung Galaxy Tab Pro 12.2">Samsung Galaxy Tab Pro 12.2</a>. It was announced on 6 January 2014,<sup class="reference" id="cite_ref-techradar_5-0"><a href="#cite_note-techradar-5">[5]</a></sup> and was released on February 13 in the US, starting at $749.</p>

<h2><span class="mw-headline" id="History">History</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Samsung_Galaxy_Note_Pro_12.2&amp;action=edit&amp;section=1" title="Edit section: History">edit</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>The Galaxy Note Pro 12.2 was announced on 6 January 2014.<sup class="reference" id="cite_ref-announce_4-1"><a href="#cite_note-announce-4">[4]</a></sup> It was shown along with the Galaxy Tab Pro 12.2, Tab Pro 10.1, and Tab Pro 8.4 at the 2014 Consumer Electronics Show in Las Vegas.<sup class="reference" id="cite_ref-techradar_5-1"><a href="#cite_note-techradar-5">[5]</a></sup></p>
<h2><span class="mw-headline" id="Features">Features</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Samsung_Galaxy_Note_Pro_12.2&amp;action=edit&amp;section=2" title="Edit section: Features">edit</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>The Galaxy Note Pro 12.2 is released with <a href="/wiki/Android_version_history#Android_4.4_KitKat_.28API_level_18.29" title="Android version history">Android 4.4 KitKat</a>. Samsung has customized the interface with its <i><a href="/wiki/TouchWiz" title="TouchWiz">TouchWiz</a> UX</i> software. As well as apps from Google, including Google Play, Gmail and YouTube, it has access to Samsung apps such as ChatON, S Suggest, S Voice, Smart Remote (Peel) and All Share Play.<sup class="reference" id="cite_ref-microsite-spec_3-1"><a href="#cite_note-microsite-spec-3">[3]</a></sup> Additional pen-oriented features and apps have been added to the Note Pro 12.2 namely Air Command menu which provides shortcuts to pen-oriented features such as Action Memos (on-screen <a class="mw-redirect" href="/wiki/Sticky_note" title="Sticky note">sticky notes</a> that use <a href="/wiki/Handwriting_recognition" title="Handwriting recognition">handwriting recognition</a> to detect their contents and provide relevant actions such as looking up addresses on <a href="/wiki/Google_Maps" title="Google Maps">Google Maps</a> and dialling phone numbers), Screen Write which is an annotation tool, Pen Window which allows users to draw pop-up windows to run certain apps inside, the search tool S Finder, Scrapbook, and an updated version of S Note. Unlike its predecessor the Note 10.1 which has My Magazine, a <a href="/wiki/News_aggregator" title="News aggregator">news aggregator</a> app that is accessible by swiping up from the bottom of the screen, thus feature is missing.</p>
<p>The Galaxy Note Pro 12.2 is available in Wi-Fi-only (SM-P900), and 4G/LTE &amp; Wi-Fi (SM-P905) variants. Storage ranges from 16 to 64 GiB depending on the model, with a <a class="mw-redirect" href="/wiki/MicroSDXC" title="MicroSDXC">microSDXC</a> card slot for expansion. It has a 12.2-inch <a href="/wiki/Graphics_display_resolution#WQXGA_.282560.C3.971600.29" title="Graphics display resolution">WQXGA</a> <a href="/wiki/IPS_panel" title="IPS panel">TFT</a> screen with a resolution of 2560×1600 pixels. It also features a 2 MP front camera and an 8 MP rear-facing camera, and can record HD videos.<sup class="reference" id="cite_ref-spec-wifi_6-0"><a href="#cite_note-spec-wifi-6">[6]</a></sup><sup class="reference" id="cite_ref-spec-3g_7-0"><a href="#cite_note-spec-3g-7">[7]</a></sup><sup class="reference" id="cite_ref-spec-4glte_8-0"><a href="#cite_note-spec-4glte-8">[8]</a></sup></p>
