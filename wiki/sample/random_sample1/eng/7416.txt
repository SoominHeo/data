<p><b>Line 10</b> is a southwest-northeast line of the <a href="/wiki/Shanghai_Metro" title="Shanghai Metro">Shanghai Metro</a> network. It officially opened for service on April 10, 2010,<sup class="reference" id="cite_ref-2"><a href="#cite_note-2">[2]</a></sup><sup class="reference" id="cite_ref-3"><a href="#cite_note-3">[3]</a></sup> and has a main line and a branch line. The first phase of the main line currently runs from <a href="/wiki/Xinjiangwancheng_Station" title="Xinjiangwancheng Station">New Jiangwan Town</a> to <a href="/wiki/Longxi_Road_Station" title="Longxi Road Station">Longxi Road</a>; the branch line runs from <a href="/wiki/Longxi_Road_Station" title="Longxi Road Station">Longxi Road</a> on the main line to <a href="/wiki/Hangzhong_Road_Station" title="Hangzhong Road Station">Hangzhong Road</a>. The extension to <a href="/wiki/Hongqiao_Railway_Station_(Shanghai_Metro)" title="Hongqiao Railway Station (Shanghai Metro)">Hongqiao Railway Station</a> has operated since December 2010. It connects the <a href="/wiki/Shanghai_Hongqiao_International_Airport" title="Shanghai Hongqiao International Airport">Hongqiao International Airport</a> with the downtown core of Shanghai, and also the dense residential districts of <a href="/wiki/Yangpu_District" title="Yangpu District">Yangpu</a> and <a href="/wiki/Hongkou_District" title="Hongkou District">Hongkou</a>.<sup class="reference" id="cite_ref-rgi_4-0"><a href="#cite_note-rgi-4">[4]</a></sup> The line is colored <span style="color:#bca7d2;"><b>mauve</b></span> on system maps.</p>

<h2><span class="mw-headline" id="Future">Future</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Line_10,_Shanghai_Metro&amp;action=edit&amp;section=4" title="Edit section: Future">edit</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>The second phase, in planning, is a northern extension of the first phase of Line 10 from <a href="/wiki/Xinjiangwancheng_Station" title="Xinjiangwancheng Station">New Jiangwan Town</a> to <a class="mw-redirect" href="/wiki/Huandongyi_Avenue_Station" title="Huandongyi Avenue Station">Huandongyi Avenue</a>.
