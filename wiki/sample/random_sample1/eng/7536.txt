<p><b>Seo Dong-Won</b> (born August 14, 1975) is a <a href="/wiki/South_Korea" title="South Korea">South Korean</a> former <a class="mw-redirect" href="/wiki/Football_(soccer)" title="Football (soccer)">football</a> player. He played for <a class="mw-redirect" href="/wiki/Daejeon_Citizen" title="Daejeon Citizen">Daejeon Citizen</a>, <a href="/wiki/Suwon_Samsung_Bluewings" title="Suwon Samsung Bluewings">Suwon Samsung Bluewings</a>, <a class="mw-redirect" href="/wiki/Jeonbuk_Hyundai_Motors" title="Jeonbuk Hyundai Motors">Jeonbuk Hyundai Motors</a>, <a class="mw-redirect" href="/wiki/Gwangju_Sangmu_Bulsajo" title="Gwangju Sangmu Bulsajo">Gwangju Sangmu Bulsajo</a> (army), <a class="mw-redirect" href="/wiki/Incheon_United" title="Incheon United">Incheon United</a>, <a class="mw-redirect" href="/wiki/Seongnam_Ilhwa_Chunma" title="Seongnam Ilhwa Chunma">Seongnam Ilhwa Chunma</a>).and <a class="mw-redirect" href="/wiki/Busan_I%27Park" title="Busan I'Park">Busan I'Park</a>.</p>
