<h2><span class="mw-headline" id="Computing">Computing</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=App&amp;action=edit&amp;section=1" title="Edit section: Computing">edit</a><span class="mw-editsection-bracket">]</span></span></h2>
<ul>
<li><a href="/wiki/Mobile_app" title="Mobile app">Mobile app</a>, software designed to run on smartphones and other mobile devices</li>

<li><a href="/wiki/Application_Portability_Profile" title="Application Portability Profile">Application Portability Profile</a>, NIST standards and specifications for the Open System Environment</li>
<li>Atom Publishing Protocol (or AtomPub), an <a href="/wiki/Atom_(standard)" title="Atom (standard)">Atom (standard)</a> protocol to create and update web resources</li>
</ul>
