<h1 class="firstHeading" id="firstHeading" lang="en">Linear alternator</h1>

<p>A <b>linear alternator</b> is essentially a <a href="/wiki/Linear_motor" title="Linear motor">linear motor</a> used as an <a class="mw-redirect" href="/wiki/Electrical_generator" title="Electrical generator">electrical generator</a>.</p>
<p>An <a href="/wiki/Alternator" title="Alternator">alternator</a> is a type of <a href="/wiki/Alternating_current" title="Alternating current">alternating current</a> (AC) electrical generator. The devices are often physically equivalent. The principal difference is in how they are used and which direction the energy flows. An alternator converts <a href="/wiki/Mechanical_energy" title="Mechanical energy">mechanical energy</a> to <a href="/wiki/Electrical_energy" title="Electrical energy">electrical energy</a>, whereas a motor converts electrical energy to mechanical energy. Like most electric motors and electric generators, the linear alternator works by the principle of <a href="/wiki/Electromagnetic_induction" title="Electromagnetic induction">electromagnetic induction</a>. However, most alternators work with rotary motion, whereas "linear" alternators work with "linear" motion (i.e. motion in a straight line).</p>
<h2><span class="mw-headline" id="Theory">Theory</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Linear_alternator&amp;action=edit&amp;section=1" title="Edit section: Theory">edit</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>When a <a href="/wiki/Magnet" title="Magnet">magnet</a> moves in relation to an <a href="/wiki/Electromagnetic_coil" title="Electromagnetic coil">electromagnetic coil</a>, this changes the <a href="/wiki/Magnetic_flux" title="Magnetic flux">magnetic flux</a> passing through the coil, and thus induces the flow of an <a href="/wiki/Electric_current" title="Electric current">electric current</a>, which can be used to do <a href="/wiki/Work_(physics)" title="Work (physics)">work</a>. A linear alternator is most commonly used to convert back-and-forth motion directly into electrical energy. This short-cut eliminates the need for a <a href="/wiki/Crank_(mechanism)" title="Crank (mechanism)">crank</a> or <a href="/wiki/Linkage_(mechanical)" title="Linkage (mechanical)">linkage</a> that would otherwise be required to convert a reciprocating motion to a rotary motion in order to be compatible with a rotary generator.</p>
<h2><span class="mw-headline" id="Applications">Applications</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Linear_alternator&amp;action=edit&amp;section=2" title="Edit section: Applications">edit</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>The simplest type of linear alternator is the <a class="mw-redirect" href="/wiki/Faraday_Flashlight" title="Faraday Flashlight">Faraday Flashlight</a>. This is a torch (UK) or <a href="/wiki/Flashlight" title="Flashlight">flashlight</a> (USA) which contains a coil and a <a class="mw-redirect" href="/wiki/Permanent_magnet" title="Permanent magnet">permanent magnet</a>. When the appliance is shaken back and forth, the magnet oscillates through the coil and <a href="/wiki/Electromagnetic_induction" title="Electromagnetic induction">induces</a> an <a href="/wiki/Electric_current" title="Electric current">electric current</a>. This current is used to charge a <a href="/wiki/Capacitor" title="Capacitor">capacitor</a>, thus storing energy for later use. The appliance can then produce light, usually from a <a href="/wiki/Light-emitting_diode" title="Light-emitting diode">light-emitting diode</a>, until the capacitor is discharged. It can then be re-charged by further shaking.</p>
<p>Other devices which use linear alternators to generate electricity include the <a href="/wiki/Free-piston_linear_generator" title="Free-piston linear generator">free-piston linear generator</a>, an <a href="/wiki/Internal_combustion_engine" title="Internal combustion engine">internal combustion engine</a>, and the <a href="/wiki/Stirling_engine#Free_piston_Stirling_engines" title="Stirling engine">free-piston Stirling engine</a>, an <a href="/wiki/External_combustion_engine" title="External combustion engine">external combustion engine</a>.</p>
