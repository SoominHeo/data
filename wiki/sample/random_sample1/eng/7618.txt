<h1 class="firstHeading" id="firstHeading" lang="en">Linear span</h1>
<div class="mw-body-content" id="bodyContent">
<div id="siteSub">From Wikipedia, the free encyclopedia</div>
<div id="contentSub"></div>
<div class="mw-jump" id="jump-to-nav">

<div class="mw-content-ltr" dir="ltr" id="mw-content-text" lang="en"><script>function mfTempOpenSection(id){var block=document.getElementById("mf-section-"+id);block.className+=" open-block";block.previousSibling.className+=" open-block";}</script><p>In <a href="/wiki/Linear_algebra" title="Linear algebra">linear algebra</a>, the <b>linear span</b> (also called the <b>linear hull</b> or just <b>span</b>) of a <a href="/wiki/Set_(mathematics)" title="Set (mathematics)">set</a> of <a href="/wiki/Vector_space" title="Vector space">vectors</a> in a <a href="/wiki/Vector_space" title="Vector space">vector space</a> is the <a href="/wiki/Intersection_(set_theory)" title="Intersection (set theory)">intersection</a> of all <a href="/wiki/Linear_subspace" title="Linear subspace">subspaces</a> containing that set. The linear span of a set of vectors is therefore a vector space. Spans can be generalized to <a href="/wiki/Matroid" title="Matroid">matroids</a> and <a href="/wiki/Module_(mathematics)" title="Module (mathematics)">modules</a>.</p>

<h2><span class="mw-headline" id="Definition">Definition</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Linear_span&amp;action=edit&amp;section=1" title="Edit section: Definition">edit</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>Given a <a href="/wiki/Vector_space" title="Vector space">vector space</a> <i>V</i> over a <a href="/wiki/Field_(mathematics)" title="Field (mathematics)">field</a> <i>K</i>, the span of a <a href="/wiki/Set_(mathematics)" title="Set (mathematics)">set</a> <i>S</i> of vectors (not necessarily finite) is defined to be the intersection <i>W</i> of all <a href="/wiki/Linear_subspace" title="Linear subspace">subspaces</a> of <i>V</i> that contain <i>S</i>. <i>W</i> is referred to as the subspace <i>spanned by</i> <i>S</i>, or by the vectors in <i>S</i>. Conversely, <i>S</i> is called a <i>spanning set</i> of <i>W</i>, and we say that <i>S</i> <i>spans</i> <i>W</i>.</p>
<p>Alternatively, the span of <i>S</i> may be defined as the set of all finite <a href="/wiki/Linear_combination" title="Linear combination">linear combinations</a> of elements of <i>S</i>, which follows from the above definition.</p>

<p>In particular, if <i>S</i> is a <a href="/wiki/Finite_set" title="Finite set">finite</a> subset of <i>V</i>, then the span of <i>S</i> is the set of all linear combinations of the elements of <i>S</i>. In the case of infinite <i>S</i>, infinite linear combinations (i.e. where a combination may involve an infinite sum, assuming such sums are defined somehow, e.g. if <i>V</i> is a <a href="/wiki/Banach_space" title="Banach space">Banach space</a>) are excluded by the definition; a <a href="/wiki/Linear_combination#Generalizations" title="Linear combination">generalization</a> that allows these is not equivalent.</p>
<h2><span class="mw-headline" id="Examples">Examples</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Linear_span&amp;action=edit&amp;section=2" title="Edit section: Examples">edit</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>The <a href="/wiki/Real_number" title="Real number">real</a> vector space <b>R</b><sup>3</sup> has {(-1,0,0), (0,1,0), (0,0,1)} as a spanning set. This particular spanning set is also a <a href="/wiki/Basis_(linear_algebra)" title="Basis (linear algebra)">basis</a>. If (-1,0,0) were replaced by (1,0,0), it would also form the <a href="/wiki/Standard_basis" title="Standard basis">canonical basis</a> of <b>R</b><sup>3</sup>.</p>
<p>Another spanning set for the same space is given by {(1,2,3), (0,1,2), (−1,1/2,3), (1,1,1)}, but this set is not a basis, because it is <a class="mw-redirect" href="/wiki/Linear_dependency" title="Linear dependency">linearly dependent</a>.</p>
<p>The set {(1,0,0), (0,1,0), (1,1,0)} is not a spanning set of <b>R</b><sup>3</sup>; instead its span is the space of all vectors in <b>R</b><sup>3</sup> whose last component is zero. That space (the space of all vectors in <b>R</b><sup>3</sup> whose last component is zero) is also spanned by the set {(1,0,0), (0,1,0)}, as (1,1,0) is a linear combination of (1,0,0) and (0,1,0). It does, however, span <b>R</b><sup>2</sup>.</p>
<p>The set of functions <i>x<sup>n</sup></i> where <i>n</i> is a non-negative integer spans the space of polynomials.</p>
<h2><span class="mw-headline" id="Theorems">Theorems</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Linear_span&amp;action=edit&amp;section=3" title="Edit section: Theorems">edit</a><span class="mw-editsection-bracket">]</span></span></h2>
<p><b>Theorem 1:</b> The subspace spanned by a non-empty subset <i>S</i> of a vector space <i>V</i> is the set of all linear combinations of vectors in <i>S</i>.</p>
<p>This theorem is so well known that at times it is referred to as the definition of span of a set.</p>
<p><b>Theorem 2:</b> Every spanning set <i>S</i> of a vector space <i>V</i> must contain at least as many elements as any <a href="/wiki/Linear_independence" title="Linear independence">linearly independent</a> set of vectors from <i>V</i>.</p>
<p><b>Theorem 3:</b> Let <i>V</i> be a finite-dimensional vector space. Any set of vectors that spans <i>V</i> can be reduced to a basis for <i>V</i> by discarding vectors if necessary (i.e. if there are linearly dependent vectors in the set). If the <a href="/wiki/Axiom_of_choice" title="Axiom of choice">axiom of choice</a> holds, this is true without the assumption that <i>V</i> has finite dimension.</p>
<p>This also indicates that a basis is a minimal spanning set when <i>V</i> is finite-dimensional.</p>

<p>The closed linear span of the set of functions <i>x<sup>n</sup></i> on the interval [0, 1], where <i>n</i> is a non-negative integer, depends on the norm used. If the <a href="/wiki/Lp_space#Lp_spaces" title="Lp space"><i>L</i><sup>2</sup> norm</a> is used, then the closed linear span is the <a href="/wiki/Hilbert_space" title="Hilbert space">Hilbert space</a> of <a href="/wiki/Square-integrable_function" title="Square-integrable function">square-integrable functions</a> on the interval. But if the <a class="mw-redirect" href="/wiki/Maximum_norm" title="Maximum norm">maximum norm</a> is used, the closed linear span will be the space of continuous functions on the interval. In either case, the closed linear span contains functions that are not polynomials, and so are not in the linear span itself. However, the <a href="/wiki/Cardinality" title="Cardinality">cardinality</a> of the set of functions in the closed linear span is the <a href="/wiki/Cardinality_of_the_continuum" title="Cardinality of the continuum">cardinality of the continuum</a>, which is the same cardinality as for the set of polynomials.</p>
<h3><span class="mw-headline" id="Notes">Notes</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Linear_span&amp;action=edit&amp;section=6" title="Edit section: Notes">edit</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>The linear span of a set is dense in the closed linear span. Moreover, as stated in the lemma below, the closed linear span is indeed the <a href="/wiki/Closure_(mathematics)" title="Closure (mathematics)">closure</a> of the linear span.</p>
<p>Closed linear spans are important when dealing with closed linear subspaces (which are themselves highly important, consider <a href="/wiki/Riesz%27s_lemma" title="Riesz's lemma">Riesz's lemma</a>).</p>
