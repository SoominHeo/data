<p><b>Adium</b> is a <a href="/wiki/Free_software" title="Free software">free</a> and <a class="mw-redirect" href="/wiki/Open_source" title="Open source">open source</a> <a href="/wiki/Instant_messaging" title="Instant messaging">instant messaging</a> <a href="/wiki/Client_(computing)" title="Client (computing)">client</a> for <a href="/wiki/MacOS" title="MacOS">macOS</a> that supports multiple IM networks, including <a href="/wiki/Windows_Live_Messenger" title="Windows Live Messenger">Windows Live Messenger</a>, <a href="/wiki/Yahoo!_Messenger" title="Yahoo! Messenger">Yahoo! Messenger</a>, <a href="/wiki/Google_Talk" title="Google Talk">Google Talk</a>, <a href="/wiki/AOL_Instant_Messenger" title="AOL Instant Messenger">AIM</a>, <a href="/wiki/ICQ" title="ICQ">ICQ</a> and <a href="/wiki/XMPP" title="XMPP">XMPP</a>. It is written using macOS's <a href="/wiki/Cocoa_(API)" title="Cocoa (API)">Cocoa</a> <a class="mw-redirect" href="/wiki/Application_Programming_Interface" title="Application Programming Interface">API</a>, and it is released under the <a href="/wiki/GNU_General_Public_License" title="GNU General Public License">GNU General Public License</a> and many other licenses for components that are distributed with Adium.</p>

<h2><span class="mw-headline" id="History">History</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Adium&amp;action=edit&amp;section=1" title="Edit section: History">edit</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>Adium was created by college student Adam Iser, and the first version, "Adium 1.0", was released in September 2001 and supported only <a href="/wiki/AOL_Instant_Messenger" title="AOL Instant Messenger">AIM</a>. The version numbers of Adium since then have followed a somewhat unusual pattern. There were several upgrades to Adium 1.0, ending with Adium 1.6.2c.<sup class="reference" id="cite_ref-adium1x_1-1"><a href="#cite_note-adium1x-1">[1]</a></sup></p>
<p>At this point, the Adium team began a complete rewrite of the Adium code, expanding it into a multiprotocol messaging program. <a href="/wiki/Pidgin_(software)" title="Pidgin (software)">Pidgin</a>'s (formerly "Gaim") <a class="mw-redirect" href="/wiki/Libpurple" title="Libpurple">libpurple</a> (then called "libgaim") library was implemented to add support for IM protocols other than AIM – since then the Adium team has mostly been working on the <a href="/wiki/Graphical_user_interface" title="Graphical user interface">GUI</a>. The Adium team originally intended to release these changes as "Adium 2.0". However, Adium was eventually renamed to "Adium X" and released at version 0.50, being considered "halfway to a 1.0 product".<sup class="reference" id="cite_ref-3"><a href="#cite_note-3">[3]</a></sup> Adium X 0.88 was the first version compiled as a <a href="/wiki/Universal_binary" title="Universal binary">universal binary</a>, allowing it to run natively on <a href="/wiki/Intel" title="Intel">Intel</a>-based Macs.</p>
<p>In 2005, Adium received a "Special Mention" at the <a href="/wiki/Apple_Design_Awards" title="Apple Design Awards">Apple Design Awards</a>.<sup class="reference" id="cite_ref-4"><a href="#cite_note-4">[4]</a></sup></p>
<p>After version Adium X 0.89.1, however, the team finally decided to change the name back to "Adium", and, as such, "Adium 1.0" was released on February 2, 2007.</p>
<p><a href="/wiki/Apple_Inc." title="Apple Inc.">Apple Inc.</a> used Adium X 0.89.1's build time in <a href="/wiki/Xcode" title="Xcode">Xcode</a> 2.3 as a <a href="/wiki/Benchmark_(computing)" title="Benchmark (computing)">benchmark</a> for comparing the performance of the <a href="/wiki/Mac_Pro" title="Mac Pro">Mac Pro</a> and <a href="/wiki/Power_Mac_G5" title="Power Mac G5">Power Mac G5</a> Quad,<sup class="reference" id="cite_ref-5"><a href="#cite_note-5">[5]</a></sup> and Adium 1.2's build time in Xcode 3.0 as a benchmark for comparing the performance of the eight-core Mac Pro and Power Mac G5 Quad.<sup class="reference" id="cite_ref-6"><a href="#cite_note-6">[6]</a></sup></p>
<p>On November 4, 2014, Adium scored 6 out of 7 points on the <a href="/wiki/Electronic_Frontier_Foundation" title="Electronic Frontier Foundation">Electronic Frontier Foundation</a>'s secure messaging scorecard. It lost a point because there has not been a recent independent code audit.<sup class="reference" id="cite_ref-7"><a href="#cite_note-7">[7]</a></sup></p>
<h2><span class="mw-headline" id="Protocols">Protocols</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Adium&amp;action=edit&amp;section=2" title="Edit section: Protocols">edit</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>Adium supports a wide range of IM networks through the <a class="mw-redirect" href="/wiki/Library_(software)" title="Library (software)">libraries</a> libezv (for <a href="/wiki/Bonjour_(software)" title="Bonjour (software)">Bonjour</a>), STTwitterEngine (for <a href="/wiki/Twitter" title="Twitter">Twitter</a>), and <a href="/wiki/Pidgin_(software)" title="Pidgin (software)">libpurple</a> (for all other protocols).</p>
<p>Adium supports the following <a class="mw-redirect" href="/wiki/Protocol_(computing)" title="Protocol (computing)">protocols</a>:</p>
<ul>
<li><a class="mw-redirect" href="/wiki/Extensible_Messaging_and_Presence_Protocol" title="Extensible Messaging and Presence Protocol">XMPP</a> (including <a href="/wiki/Google_Talk" title="Google Talk">Google Talk</a>, <a class="mw-redirect" href="/wiki/Facebook_Chat" title="Facebook Chat">Facebook Chat</a>, and <a href="/wiki/LiveJournal" title="LiveJournal">LiveJournal</a> services)</li>
<li><a href="/wiki/OSCAR_protocol" title="OSCAR protocol">OSCAR</a> (including <a href="/wiki/AOL_Instant_Messenger" title="AOL Instant Messenger">AIM</a> and <a href="/wiki/ICQ" title="ICQ">ICQ</a></li>
<li><a href="/wiki/Twitter" title="Twitter">Twitter</a><sup class="reference" id="cite_ref-8"><a href="#cite_note-8">[8]</a></sup></li>
<li><a href="/wiki/Bonjour_(software)" title="Bonjour (software)">Bonjour</a></li>
<li><a href="/wiki/Internet_Relay_Chat" title="Internet Relay Chat">IRC</a></li>
<li><a href="/wiki/GroupWise" title="GroupWise">Novell GroupWise</a></li>
<li><a href="/wiki/IBM_Sametime" title="IBM Sametime">IBM Sametime</a></li>
<li><a href="/wiki/Gadu-Gadu" title="Gadu-Gadu">Gadu-Gadu</a></li>
<li><a href="/wiki/Skype" title="Skype">Skype</a> with a plugin</li>
<li><a href="/wiki/Skype_for_Business_Server" title="Skype for Business Server">Skype for Business Server</a> (previously Microsoft Lync Server, Microsoft Office Communications Server) with a plugin</li>
<li><a href="/wiki/Telegram_(software)" title="Telegram (software)">Telegram</a> with a plugin</li>
<li><a href="/wiki/Tencent_QQ" title="Tencent QQ">QQ</a> with a plugin</li>
<li><a href="/wiki/Steam_(software)" title="Steam (software)">Steam</a> chat with the "Steam IM" plugin</li>
<li><a href="/wiki/Nate_(web_portal)#NateOn" title="Nate (web portal)">NateOn</a> with a plugin</li>
</ul>
<h2><span class="mw-headline" id="Plugins_and_customization">Plugins and customization</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Adium&amp;action=edit&amp;section=3" title="Edit section: Plugins and customization">edit</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>Adium makes use of a <a href="/wiki/Plug-in_(computing)" title="Plug-in (computing)">plugin</a> architecture; many of the program's essential features are actually provided by plugins bundled inside the application package. These plugins include functionality such as file transfer, support for the <a href="/wiki/Growl_(software)" title="Growl (software)">Growl</a> notifications system, <a href="/wiki/Sparkle_(software)" title="Sparkle (software)">Sparkle</a> for program updates, and support for encrypted messaging with the <a href="/wiki/Off-the-Record_Messaging" title="Off-the-Record Messaging">Off-the-Record Messaging</a> library.</p>
<p>Adium is also highly customizable through the use of resources its developers call "Xtras". The program can be customized by the use of hundreds of third-party Xtras that alter the appearance of <a href="/wiki/Emoticon" title="Emoticon">emoticons</a>, <a href="/wiki/Dock_(macOS)" title="Dock (macOS)">dock</a> icons, <a href="/wiki/Contact_list" title="Contact list">contact list</a> styles, and messages styles. Adium can also be enhanced through the use of different sound sets. <a href="/wiki/AppleScript" title="AppleScript">AppleScripts</a> can also be utilized to automatically alter behavior in responses to certain triggers.</p>
