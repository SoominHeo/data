<p>The <b>Sony Ericsson Xperia X8</b> is a mid-range <a href="/wiki/3G" title="3G">3G</a> <a href="/wiki/Android_(operating_system)" title="Android (operating system)">Android</a> <a href="/wiki/Smartphone" title="Smartphone">smartphone</a> developed by <a class="mw-redirect" href="/wiki/Sony_Ericsson" title="Sony Ericsson">Sony Ericsson</a> in the <a class="mw-redirect" href="/wiki/Xperia" title="Xperia">Xperia</a> series in released in Q4 2010. It is sold in many countries worldwide, including the United States on <a href="/wiki/AT%26T_Mobility" title="AT&amp;T Mobility">AT&amp;T Mobility</a> and low-end pay-monthly contracts in the UK. It originally shipped running Android 1.6 but was upgraded in early 2011 to Android 2.1.</p>

<h2><span class="mw-headline" id="Hardware">Hardware</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Sony_Ericsson_Xperia_X8&amp;action=edit&amp;section=1" title="Edit section: Hardware">edit</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>The phone has a 3.0-inch (76 mm) <a class="mw-redirect" href="/wiki/Capacitive_touch_screen" title="Capacitive touch screen">capacitive touch screen</a> LCD. It has a resolution of 320 by 480 pixels half-size video graphics array (<a class="mw-redirect" href="/wiki/HVGA" title="HVGA">HVGA</a>) with <a class="mw-redirect" href="/wiki/True_Color" title="True Color">24-bit depth</a>. On the back is a 3.2-megapixel camera with fixed focus. Photos can be <a href="/wiki/Geotagging" title="Geotagging">geotagged</a>. The camera can be accessed via the <a href="/wiki/Touchscreen" title="Touchscreen">touchscreen</a> menu, or via the dedicated camera button on the side of the phone. The Xperia X8 uses a 600 MHz <a href="/wiki/Qualcomm" title="Qualcomm">Qualcomm</a> <a class="mw-redirect" href="/wiki/MSM7227#MSM7227" title="MSM7227">MSM7227</a> processor, has a 3-axis accelerometer and built-in <a href="/wiki/Global_Positioning_System" title="Global Positioning System">Global Positioning System</a> (GPS), a digital compass, an ambient light sensor and a lug for attaching a strap.</p>
<p><a href="/wiki/Multi-touch" title="Multi-touch">Multi-touch</a> support was also added to several aftermarket firmware versions (ROMS) as a sideloaded modules (Now newer kernel included the code to enable it) which enabled users to use basic multitouch gestures such as <a class="mw-redirect" href="/wiki/Pinch_to_zoom" title="Pinch to zoom">pinch to zoom</a> (On Synaptics digitizer) and <a class="new" href="/w/index.php?title=Real_dual-touch&amp;action=edit&amp;redlink=1" title="Real dual-touch (page does not exist)">real dual-touch</a> (On Cypress digitizer) thanks to <a href="/wiki/XDA_Developers" title="XDA Developers">XDA</a> Recognized Developer doixanh, AnDyx &amp; andrej456, something that <a class="mw-redirect" href="/wiki/Sony_Ericsson" title="Sony Ericsson">Sony Ericsson</a> was adamant was impossible on the X8.</p>
<h2><span class="mw-headline" id="Software">Software</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Sony_Ericsson_Xperia_X8&amp;action=edit&amp;section=2" title="Edit section: Software">edit</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>Sony Ericsson has made its own custom overlay on the Android system using an optimized version of the UX (user experience) interface, which consists of design elements, themes and custom applications. The four corners of the screen have replaceable shortcuts for commonly used applications. The main application is Sony Ericsson Timescape, a social networking application that combines messages from Facebook, Twitter, SMS and email into a flowing column on the home screen. Photos may be attached to contacts or sent to a Facebook account. The phone contains expected smartphone applications, and <a class="new" href="/w/index.php?title=Wisepilot&amp;action=edit&amp;redlink=1" title="Wisepilot (page does not exist)">Wisepilot</a> Navigation Software, <a href="/wiki/Google_Maps" title="Google Maps">Google Maps</a>, YouTube, Sony's <a href="/wiki/PlayNow_Arena" title="PlayNow Arena">PlayNow Arena</a> and TrackID (to identify a song by recording a small part of it).</p>
<p><a class="mw-redirect" href="/wiki/Sony_Ericsson" title="Sony Ericsson">Sony Ericsson</a> was widely criticized by X8 users for failing to provide <a href="/wiki/Android_(operating_system)" title="Android (operating system)">Android</a> firmware upgrades to 2.2 Froyo or 2.3 Gingerbread. They later stated that the phone's hardware was unable to run anything beyond 2.1 Eclair. They were later proved wrong by the Android modding community, who released their own versions of Android 2.2 and 2.3 in subsequent months, which ran stably on the X8. <a href="/wiki/CyanogenMod" title="CyanogenMod">CyanogenMod</a> released an 2.3 update for the X8 which required <a href="/wiki/Rooting_(Android_OS)" title="Rooting (Android OS)">rooting</a>. Since then, independent developers have made a fully working Ice Cream Sandwich (4.0) port, Jelly Bean (4.1 &amp; 4.2) port as well as the latest Android KitKat (4.4) using a custom <a class="mw-redirect" href="/wiki/Kernel_(computing)" title="Kernel (computing)">kernel</a>.</p>
