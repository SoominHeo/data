<p><b>Socket 478</b> (also known as mPGA478, mPGA478B) is a 478-contact <a href="/wiki/CPU_socket" title="CPU socket">CPU socket</a> used for <a href="/wiki/Intel" title="Intel">Intel</a>'s <a href="/wiki/Pentium_4" title="Pentium 4">Pentium 4</a> and <a href="/wiki/Celeron" title="Celeron">Celeron</a> series <a href="/wiki/Central_processing_unit" title="Central processing unit">CPUs</a>.</p>
<p>Socket 478 was launched with the Northwood core to compete with <a class="mw-redirect" href="/wiki/AMD" title="AMD">AMD</a>'s 462-pin <a href="/wiki/Socket_A" title="Socket A">Socket A</a> and their <a class="mw-redirect" href="/wiki/Athlon_XP" title="Athlon XP">Athlon XP</a> processors. Socket 478 was intended to be the replacement for <a href="/wiki/Socket_423" title="Socket 423">Socket 423</a>, a Willamette-based processor socket which was on the market for only a short time. Socket 478 was phased out with the launch of <a href="/wiki/LGA_775" title="LGA 775">LGA 775</a>.</p>

<h2><span class="mw-headline" id="Technical_specifications">Technical specifications</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Socket_478&amp;action=edit&amp;section=1" title="Edit section: Technical specifications">edit</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>Socket 478 was used for all <a href="/wiki/Pentium_4#Northwood" title="Pentium 4">Northwood Pentium 4</a> and <a href="/wiki/Celeron" title="Celeron">Celeron</a> processors. It supported the first <a href="/wiki/Pentium_4#Prescott" title="Pentium 4">Prescott Pentium 4</a> processors and all <a href="/wiki/Pentium_4#Willamette" title="Pentium 4">Willamette</a> Celerons, along with several of the Willamette-series Pentium 4s. Socket 478 also supported the newer Prescott-based Celeron D processors, and early Pentium 4 Extreme Edition processors with 2 MiB of L3 <a href="/wiki/CPU_cache" title="CPU cache">CPU cache</a>.</p>
<p><a class="mw-redirect" href="/wiki/Celeron_D" title="Celeron D">Celeron D</a> processors were also available for Socket 478 and were the last CPUs made for the socket.</p>
<p>While the Intel mobile CPUs are available in 478-pin packages, they in fact only operate in a range of slightly differing sockets, <a href="/wiki/Socket_479" title="Socket 479">Socket 479</a>, <a href="/wiki/Socket_M" title="Socket M">Socket M</a>, and <a href="/wiki/Socket_P" title="Socket P">Socket P</a>, each incompatible with the other two.</p>
