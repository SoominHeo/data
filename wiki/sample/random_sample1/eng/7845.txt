<p><b>Socket 940</b> is a 940-pin socket for 64-bit <a class="mw-redirect" href="/wiki/AMD" title="AMD">AMD</a> server processors. This socket is entirely square in shape and pins are arranged in a grid with the exception of four key pins used to align the processor and the corners. AMD's <a href="/wiki/Opteron" title="Opteron">Opteron</a> and the older AMD <a href="/wiki/Athlon_64" title="Athlon 64">Athlon 64 FX</a> (<a class="mw-redirect" href="/wiki/FX-51" title="FX-51">FX-51</a>) use Socket 940.</p>

<h2><span class="mw-headline" id="Technical_specifications">Technical specifications</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Socket_940&amp;action=edit&amp;section=1" title="Edit section: Technical specifications">edit</a><span class="mw-editsection-bracket">]</span></span></h2>
<p><a href="/wiki/Microprocessor" title="Microprocessor">Microprocessors</a> designed for this socket were intended to be used in a server platform, and as such provide additional features to provide additional robustness. One such feature is the acceptance of only <a href="/wiki/Registered_memory" title="Registered memory">registered memory</a>.<sup class="reference" id="cite_ref-1"><a href="#cite_note-1">[1]</a></sup></p>
<p>While the more recent 940-pin socket <a href="/wiki/Socket_AM2" title="Socket AM2">AM2</a> is visually similar to this one, the two are electrically incompatible due to the integrated <a href="/wiki/Memory_controller" title="Memory controller">memory controller</a>. Socket 940 CPU's integrate a <a href="/wiki/DDR_SDRAM" title="DDR SDRAM">DDR</a> controller, whereas AM2 models utilise a <a href="/wiki/DDR2_SDRAM" title="DDR2 SDRAM">DDR2</a> controller.<sup class="reference" id="cite_ref-2"><a href="#cite_note-2">[2]</a></sup></p>
