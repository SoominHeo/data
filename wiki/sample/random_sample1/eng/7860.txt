<p>The <b>Small Device C Compiler</b> (<b>SDCC</b>) is a <a class="mw-redirect" href="/wiki/Free-software" title="Free-software">free-software</a>, partially <a class="mw-redirect" href="/wiki/Retargetable_compiler" title="Retargetable compiler">retargetable</a><sup class="reference" id="cite_ref-1"><a href="#cite_note-1">[1]</a></sup> <a href="/wiki/C_(programming_language)" title="C (programming language)">C</a> <a href="/wiki/Compiler" title="Compiler">compiler</a> for <a href="/wiki/Microcontroller" title="Microcontroller">microcontrollers</a>. It is distributed under the <a href="/wiki/GNU_General_Public_License" title="GNU General Public License">GNU General Public License</a>. The package also contains a <a href="/wiki/Linker_(computing)" title="Linker (computing)">linker</a>, <a href="/wiki/Assembly_language" title="Assembly language">assembler</a>, <a class="mw-redirect" href="/wiki/Simulator" title="Simulator">simulator</a> and <a href="/wiki/Debugger" title="Debugger">debugger</a>. As of March 2007, SDCC is the only open-source C compiler for <a class="mw-redirect" href="/wiki/Intel_8051" title="Intel 8051">Intel 8051</a>-compatible microcontrollers.<sup class="reference" id="cite_ref-2"><a href="#cite_note-2">[2]</a></sup><sup class="reference" id="cite_ref-3"><a href="#cite_note-3">[3]</a></sup><sup class="reference" id="cite_ref-4"><a href="#cite_note-4">[4]</a></sup><sup class="noprint Inline-Template Template-Fact" style="white-space:nowrap;">[<i><a href="/wiki/Wikipedia:Citation_needed" title="Wikipedia:Citation needed"><span title="Google? (June 2011)">citation needed</span></a></i>]</sup> In 2011 the compiler was downloaded on average more than 200 times per day.<sup class="reference" id="cite_ref-5"><a href="#cite_note-5">[5]</a></sup></p>
<p>The SDCC <a href="/wiki/Compiler" title="Compiler">compiler</a> was used by the <a href="/wiki/FreeRTOS" title="FreeRTOS">FreeRTOS</a> project to port its real-time operating system to the 8051-based <a class="mw-redirect" href="/wiki/Silicon_Laboratories" title="Silicon Laboratories">Silabs</a> (formerly <a class="mw-redirect" href="/wiki/Cygnal" title="Cygnal">Cygnal</a>) series of <a class="mw-redirect" href="/wiki/Microcontrollers" title="Microcontrollers">microcontrollers</a>.</p>
