<ul>
<li>For some specific types of series there are more specialized convergence tests, for instance for <a href="/wiki/Fourier_series" title="Fourier series">Fourier series</a> there is the <a href="/wiki/Dini_test" title="Dini test">Dini test</a>.</li>
</ul>
<h2><span class="mw-headline" id="Comparison">Comparison</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Convergence_tests&amp;action=edit&amp;section=14" title="Edit section: Comparison">edit</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>The root test is stronger than the ratio test (it is more powerful because the required condition is weaker): whenever the ratio test determines the convergence or divergence /of an infinite series, the root test does too, but not conversely.<sup class="reference" id="cite_ref-1"><a href="#cite_note-1">[1]</a></sup></p>
<p>For example, for the series</p>
<dl>
<dd>1 + 1 + 0.5 + 0.5 + 0.25 + 0.25 + 0.125 + 0.125 + ... = 4</dd>
</dl>
<p>convergence follows from the root test but not from the ratio test.</p>
