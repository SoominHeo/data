<p><b>Susan Eldridge</b> (born October 7, 1979) is an <a href="/wiki/United_States" title="United States">American</a> <a href="/wiki/Model_(person)" title="Model (person)">model</a>.</p>
<h2><span class="mw-headline" id="Career">Career</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Susan_Eldridge&amp;action=edit&amp;section=1" title="Edit section: Career">edit</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>After attending the Model Search America Contest in Seattle Washington,<sup class="reference" id="cite_ref-love_intview_1-0"><a href="#cite_note-love_intview-1">[1]</a></sup> Eldridge moved to <a href="/wiki/New_York_City" title="New York City">New York City</a> and signed with <a href="/wiki/Elite_Model_Management" title="Elite Model Management">Elite Model Management</a>. She has done numerous editorials and has appeared on the cover of <i><a href="/wiki/Vogue_(magazine)" title="Vogue (magazine)">Vogue</a></i>, <i>Purple</i>, <i>Self Service</i>, and <i>The Face</i> (with <a class="mw-redirect" href="/wiki/Pharrell" title="Pharrell">Pharrell</a>).</p>
<p>She has appeared in runway shows for <a href="/wiki/Shiatzy_Chen" title="Shiatzy Chen">Shiatzy Chen</a>, <a href="/wiki/DKNY" title="DKNY">DKNY</a>, <a href="/wiki/John_Galliano" title="John Galliano">John Galliano</a>, <a href="/wiki/Christian_Dior" title="Christian Dior">Christian Dior</a>, <a href="/wiki/Louis_Vuitton" title="Louis Vuitton">Louis Vuitton</a>, <a href="/wiki/Chanel" title="Chanel">Chanel</a>, <a href="/wiki/Fendi" title="Fendi">Fendi</a>, <a href="/wiki/Prada" title="Prada">Prada</a>, <a class="mw-redirect" href="/wiki/Gianni_Versace_S.p.A." title="Gianni Versace S.p.A.">Versace</a>, <a href="/wiki/Roberto_Cavalli" title="Roberto Cavalli">Roberto Cavalli</a>, <a href="/wiki/Marc_Jacobs" title="Marc Jacobs">Marc Jacobs</a>, <a href="/wiki/Tommy_Hilfiger" title="Tommy Hilfiger">Tommy Hilfiger</a>, <a href="/wiki/Nina_Ricci_(brand)" title="Nina Ricci (brand)">Nina Ricci</a>, <a href="/wiki/Giorgio_Armani" title="Giorgio Armani">Giorgio Armani</a>, <a href="/wiki/Max_Azria" title="Max Azria">BCBG</a>, <a href="/wiki/Calvin_Klein" title="Calvin Klein">Calvin Klein</a>, <a href="/wiki/Vivienne_Westwood" title="Vivienne Westwood">Vivienne Westwood</a>, <a href="/wiki/Stella_McCartney" title="Stella McCartney">Stella McCartney</a>, <a href="/wiki/Miu_Miu" title="Miu Miu">Miu Miu</a>, <a href="/wiki/Sonia_Rykiel" title="Sonia Rykiel">Sonia Rykiel</a>, and <a href="/wiki/Chlo%C3%A9" title="Chloé">Chloé</a>. Eldridge has worked with many of the world's top photographers including <a href="/wiki/Ellen_von_Unwerth" title="Ellen von Unwerth">Ellen von Unwerth</a>, <a href="/wiki/Mario_Testino" title="Mario Testino">Mario Testino</a>, and <a href="/wiki/Terry_Richardson" title="Terry Richardson">Terry Richardson</a>. She has done campaigns for <a href="/wiki/Miss_Sixty" title="Miss Sixty">Miss Sixty</a>, <a class="mw-redirect" href="/wiki/D%26G" title="D&amp;G">D&amp;G</a>, <a href="/wiki/Armani" title="Armani">Armani</a>, <a href="/wiki/Marc_Jacobs" title="Marc Jacobs">Marc Jacobs</a>, <a class="mw-redirect" href="/wiki/Bebe_stores" title="Bebe stores">bebe</a>, and <a class="mw-redirect" href="/wiki/Zara_(clothing)" title="Zara (clothing)">Zara</a>. In 2004 she appeared in a commercial for <a href="/wiki/Victoria%27s_Secret" title="Victoria's Secret">Victoria's Secret</a>. Eldridge won the Elle "model of the year" award in 2005, and was on the cover of <a href="/wiki/Whirlwind_Heat" title="Whirlwind Heat">Whirlwind Heat</a>'s 2006 album "<a href="/wiki/Types_of_Wood" title="Types of Wood">Types of Wood</a>".</p>
<p>After an almost two-year disappearance, Eldridge returned to fashion with the spring 2007 BCBGirls and Parasuco denim campaigns. Eldridge also appears in the 2007 Arena Sexiest Girls of the year calendar alongside <a class="mw-redirect" href="/wiki/Beyonc%C3%A9_Knowles" title="Beyoncé Knowles">Beyoncé Knowles</a>, <a href="/wiki/Gisele_B%C3%BCndchen" title="Gisele Bündchen">Gisele Bündchen</a>, <a class="new" href="/w/index.php?title=Zoe_Duchesne&amp;action=edit&amp;redlink=1" title="Zoe Duchesne (page does not exist)">Zoe Duchesne</a>, <a href="/wiki/Evangeline_Lilly" title="Evangeline Lilly">Evangeline Lilly</a>, <a href="/wiki/Rihanna" title="Rihanna">Rihanna</a>, <a href="/wiki/Eva_Green" title="Eva Green">Eva Green</a>, and <a href="/wiki/Hannah_Spearritt" title="Hannah Spearritt">Hannah Spearritt</a>.</p>
<p>In fall 2007, Eldridge appeared in the Aldo shoe campaign.</p>
<h2><span class="mw-headline" id="Personal_life">Personal life</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Susan_Eldridge&amp;action=edit&amp;section=2" title="Edit section: Personal life">edit</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>Susan Eldridge has three children at this time.</p>
<p>Eldridge began dating photographer Terry Richardson in 2003.<sup class="reference" id="cite_ref-nyp-crooked_2-0"><a href="#cite_note-nyp-crooked-2">[2]</a></sup> It was reported in 2004 that they had split up.<sup class="reference" id="cite_ref-to-richardson_iview_3-0"><a href="#cite_note-to-richardson_iview-3">[3]</a></sup> The two worked together on many projects.</p>
