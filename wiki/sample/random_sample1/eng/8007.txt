<p>In algebra, a <b>cyclic group</b> or <b>monogenous group</b> is a <a href="/wiki/Group_(mathematics)" title="Group (mathematics)">group</a> that is <a href="/wiki/Generating_set_of_a_group" title="Generating set of a group">generated</a> by a single element.<sup class="reference" id="cite_ref-eom_1-0"><a href="#cite_note-eom-1">[1]</a></sup> That is, it consists of a <a href="/wiki/Set_(mathematics)" title="Set (mathematics)">set</a> of elements with a single invertible <a class="mw-redirect" href="/wiki/Associative_operation" title="Associative operation">associative operation</a>, and it contains an element <i>g</i> such that every other element of the group may be obtained by repeatedly applying the group operation or its inverse to <i>g</i>. Each element can be written as a power of <i>g</i> in multiplicative notation, or as a multiple of <i>g</i> in additive notation. This element <i>g</i> is called a <i><a href="/wiki/Generating_set_of_a_group" title="Generating set of a group">generator</a></i> of the group.<sup class="reference" id="cite_ref-eom_1-1"><a href="#cite_note-eom-1">[1]</a></sup></p>
<p>Every infinite cyclic group is <a class="mw-redirect" href="/wiki/Isomorphic" title="Isomorphic">isomorphic</a> to the <a href="/wiki/Additive_group" title="Additive group">additive group</a> of <b>Z</b>, the integers. Every finite cyclic group of <a href="/wiki/Order_(group_theory)" title="Order (group theory)">order</a> <i>n</i> is isomorphic to the additive group of <b>Z</b>/<i>n</i><b>Z</b>, the integers <a href="/wiki/Modular_arithmetic" title="Modular arithmetic">modulo</a> <i>n</i>. Every cyclic group is an <a href="/wiki/Abelian_group" title="Abelian group">abelian group</a> (meaning that its group operation is <a href="/wiki/Commutative_property" title="Commutative property">commutative</a>), and every finitely generated abelian group is a <a href="/wiki/Direct_product" title="Direct product">direct product</a> of cyclic groups.</p>

<p>A group <i>G</i> is called cyclic if there exists an element <i>g</i> in <i>G</i> such that <span class="nowrap"><i>G</i> = ⟨<i>g</i>⟩ = { <i>g</i><sup><i>n</i></sup> | <i>n</i> is an integer }.</span> Since any group generated by an element in a group is a subgroup of that group, showing that the only <a href="/wiki/Subgroup" title="Subgroup">subgroup</a> of a group <i>G</i> that contains <i>g</i> is <i>G</i> itself suffices to show that <i>G</i> is cyclic.</p>
<p>For example, if <i>G</i> = { <i>g</i><sup>0</sup>, <i>g</i><sup>1</sup>, <i>g</i><sup>2</sup>, <i>g</i><sup>3</sup>, <i>g</i><sup>4</sup>, <i>g</i><sup>5</sup> } is a group of order 6, then <i>g</i><sup>6</sup> = <i>g</i><sup>0</sup>, and <i>G</i> is cyclic. In fact, <i>G</i> is essentially the same as (that is, <a href="/wiki/Isomorphism" title="Isomorphism">isomorphic</a> to) the set <span class="nowrap">{ 0, 1, 2, 3, 4, 5 }</span> with addition <a href="/wiki/Modular_arithmetic" title="Modular arithmetic">modulo</a> 6. For example, <span class="nowrap">1 + 2 ≡ 3 (mod 6)</span> corresponds to <span class="nowrap"><i>g</i><sup>1</sup> · <i>g</i><sup>2</sup> = <i>g</i><sup>3</sup></span>, and <span class="nowrap">2 + 5 ≡ 1 (mod 6)</span> corresponds to <span class="nowrap"><i>g</i><sup>2</sup> · <i>g</i><sup>5</sup> = <i>g</i><sup>7</sup> = <i>g</i><sup>1</sup></span>, and so on. One can use the isomorphism χ defined by <span class="nowrap">χ(<i>g</i><sup><i>i</i></sup>) = <i>i</i></span>.</p>
<p>The name "cyclic" may be misleading:<sup class="reference" id="cite_ref-2"><a href="#cite_note-2">[2]</a></sup> it is possible to generate infinitely many elements and not form any literal cycles; that is, every <i>g</i><sup><i>n</i></sup> is distinct. (It can be thought of as having one infinitely long cycle.) A group generated in this way (for example, the first <a href="/wiki/Frieze_group#Descriptions_of_the_seven_frieze_groups" title="Frieze group">frieze group</a>, p1) is called an <b>infinite cyclic group</b>, and is isomorphic to the additive group of the <a href="/wiki/Integer" title="Integer">integers</a>, <span class="nowrap">(<b>Z</b>, +)</span>.</p>
<p><a href="/wiki/Nicolas_Bourbaki" title="Nicolas Bourbaki">Nicolas Bourbaki</a>, who was the French mathematicians group, called a cyclic group as <b>monogenous group</b>,<sup class="reference" id="cite_ref-algebra1..C2.A74.10_4-0"><a href="#cite_note-algebra1..C2.A74.10-4">[note 1]</a></sup> and defined a finite monogenous group as a cyclic group,<sup class="reference" id="cite_ref-algebra1..C2.A74.10_4-1"><a href="#cite_note-algebra1..C2.A74.10-4">[note 1]</a></sup> then, Bourbaki avoided the term infinite cyclic group.</p>

<h3><span class="mw-headline" id="Integer_and_modular_addition">Integer and modular addition</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Cyclic_group&amp;action=edit&amp;section=3" title="Edit section: Integer and modular addition">edit</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>The set of <a href="/wiki/Integer" title="Integer">integers</a>, with the operation of addition, forms a group.<sup class="reference" id="cite_ref-eom_1-2"><a href="#cite_note-eom-1">[1]</a></sup> It is an <b>infinite cyclic group</b>, because all integers can be written as a finite sum or difference of copies of the number 1. In this group, 1 and −1 are the only generators. Every infinite cyclic group is isomorphic to this group.</p>
<p>For every positive integer <i>n</i>, the set of integers <a href="/wiki/Modular_arithmetic" title="Modular arithmetic">modulo</a> <i>n</i>, again with the operation of addition, forms a finite cyclic group, the group <b>Z</b>/<i>n</i>.<sup class="reference" id="cite_ref-eom_1-3"><a href="#cite_note-eom-1">[1]</a></sup> An element <i>g</i> is a generator of this group if <i>g</i> is <a class="mw-redirect" href="/wiki/Relatively_prime" title="Relatively prime">relatively prime</a> to <i>n</i>. Thus, the number of different generators is φ(<i>n</i>), where φ is the <a class="mw-redirect" href="/wiki/Euler_totient_function" title="Euler totient function">Euler totient function</a>, the function that counts the number of numbers modulo <i>n</i> that are relatively prime to <i>n</i>. Every finite cyclic group is isomorphic to a group <b>Z</b>/<i>n</i>, where <i>n</i> is the order of the group.</p>
<p>The integer and modular addition operations, used to define the cyclic groups, are both the addition operations of <a href="/wiki/Commutative_ring" title="Commutative ring">commutative rings</a>, also denoted <b>Z</b> and <b>Z</b>/<i>n</i>. If <i>p</i> is a prime, then <b>Z</b>/<i>p</i> is a <a href="/wiki/Finite_field" title="Finite field">finite field</a>, and is usually instead written as <b>F</b><sub><i>p</i></sub> or GF(<i>p</i>). Every field with <i>p</i> elements is <a href="/wiki/Isomorphism" title="Isomorphism">isomorphic</a> to this one.</p>
<h3><span class="mw-headline" id="Modular_multiplication">Modular multiplication</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Cyclic_group&amp;action=edit&amp;section=4" title="Edit section: Modular multiplication">edit</a><span class="mw-editsection-bracket">]</span></span></h3>
<div class="hatnote" role="note">Main article: <a href="/wiki/Multiplicative_group_of_integers_modulo_n" title="Multiplicative group of integers modulo n">Multiplicative group of integers modulo n</a></div>
<p>For every positive integer <i>n</i>, the subset of the integers modulo <i>n</i> that are relatively prime to <i>n</i>, with the operation of multiplication, forms a finite group that for many values of <i>n</i> is again cyclic. It is the <a href="/wiki/Multiplicative_group_of_integers_modulo_n" title="Multiplicative group of integers modulo n">group under multiplication modulo <i>n</i></a>, and it is cyclic whenever <i>n</i> is 1, 2, 4, a <a href="/wiki/Prime_power" title="Prime power">power of an odd prime</a>, or twice a power of an odd prime.<sup class="reference" id="cite_ref-5"><a href="#cite_note-5">[4]</a></sup> (sequence <span class="nowrap"><a class="extiw" href="//oeis.org/A033948" title="oeis:A033948">A033948</a></span> in the <a href="/wiki/On-Line_Encyclopedia_of_Integer_Sequences" title="On-Line Encyclopedia of Integer Sequences">OEIS</a>) <sup class="reference" id="cite_ref-6"><a href="#cite_note-6">[5]</a></sup> Its elements are the <a href="/wiki/Unit_(ring_theory)" title="Unit (ring theory)">units</a> of the ring <b>Z</b>/<i>n</i><b>Z</b>; there are φ(<i>n</i>) of them, where again φ is the totient function. This group is written as (<b>Z</b>/<i>n</i><b>Z</b>)<sup>×</sup>. For example, (<b>Z</b>/6<b>Z</b>)<sup>×</sup> has as its elements {1,5}; 6 is twice a prime, so this is a cyclic group. In contrast, (<b>Z</b>/8<b>Z</b>)<sup>×</sup> (with elements {1,3,5,7}) is the <a class="mw-redirect" href="/wiki/Klein_group" title="Klein group">Klein group</a> and is not cyclic. When (<b>Z</b>/<i>n</i><b>Z</b>)<sup>×</sup> is cyclic, every generator of (<b>Z</b>/<i>n</i><b>Z</b>)<sup>×</sup> is called a <a href="/wiki/Primitive_root_modulo_n" title="Primitive root modulo n">primitive root modulo <i>n</i></a>.</p>
<p>The cyclic group (<b>Z</b>/<i>p</i><b>Z</b>)<sup>×</sup> for a prime number <i>p</i>, is also written (<b>Z</b>/<i>p</i><b>Z</b>)<sup>*</sup> because it consists of the non-zero elements of the <a href="/wiki/Finite_field" title="Finite field">finite field</a> of order <i>p</i>. More generally, every finite <a href="/wiki/Subgroup" title="Subgroup">subgroup</a> of the multiplicative group of any <a href="/wiki/Field_(mathematics)" title="Field (mathematics)">field</a> is cyclic.<sup class="reference" id="cite_ref-7"><a href="#cite_note-7">[6]</a></sup></p>
<h3><span class="mw-headline" id="Rotational_symmetries">Rotational symmetries</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Cyclic_group&amp;action=edit&amp;section=5" title="Edit section: Rotational symmetries">edit</a><span class="mw-editsection-bracket">]</span></span></h3>
<div class="hatnote" role="note">Main article: <a href="/wiki/Rotational_symmetry" title="Rotational symmetry">Rotational symmetry</a></div>
<p>The set of <a href="/wiki/Rotational_symmetry" title="Rotational symmetry">rotational symmetries</a> of a <a href="/wiki/Polygon" title="Polygon">polygon</a> forms a finite cyclic group.<sup class="reference" id="cite_ref-8"><a href="#cite_note-8">[7]</a></sup> If there are <i>n</i> different ways of mapping the polygon to itself by a rotation (including the null rotation) then this group is isomorphic to Z<sub>n</sub>. In three or higher dimensions there can exist other <a href="/wiki/Point_groups_in_three_dimensions#Cyclic_3D_symmetry_groups" title="Point groups in three dimensions">finite symmetry groups that are cyclic</a>, but that do not form the set of rotations around a single axis.</p>
<p>The group <i>S</i><sup>1</sup> of all rotations of a <a href="/wiki/Circle" title="Circle">circle</a> (the <a href="/wiki/Circle_group" title="Circle group">circle group</a>) is <i>not</i> cyclic. Unlike the infinite cyclic group, it is not even <a class="mw-redirect" href="/wiki/Countable" title="Countable">countable</a>. There also exist other infinite rotation groups (such as the set of rotations by rational angles) that are countable but not cyclic.</p>
<h3><span class="mw-headline" id="Galois_theory">Galois theory</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Cyclic_group&amp;action=edit&amp;section=6" title="Edit section: Galois theory">edit</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>An <i>n</i>th <a href="/wiki/Root_of_unity" title="Root of unity">root of unity</a> may be thought of as a <a href="/wiki/Complex_number" title="Complex number">complex number</a> whose <i>n</i>th power is 1. That is, it is a <a class="mw-redirect" href="/wiki/Root_of_a_polynomial" title="Root of a polynomial">root</a> of the polynomial <i>x</i><sup><i>n</i></sup> − 1. The <i>n</i>th roots of unity form a cyclic group of order <i>n</i> under multiplication.<sup class="reference" id="cite_ref-eom_1-4"><a href="#cite_note-eom-1">[1]</a></sup> For example, the polynomial <span class="nowrap">0 = <i>z</i><sup>3</sup> − 1</span> factors as <span class="nowrap">(<i>z</i> − <i>s</i><sup>0</sup>)(<i>z</i> − <i>s</i><sup>1</sup>)(<i>z</i> − <i>s</i><sup>2</sup>)</span>, where <span class="nowrap"><i>s</i> = <i>e</i><sup>2π<i>i</i>/3</sup></span>; the set {<i>s</i><sup>0</sup>, <i>s</i><sup>1</sup>, <i>s</i><sup>2</sup>} forms a cyclic group under multiplication. The <a href="/wiki/Galois_group" title="Galois group">Galois group</a> of the <a href="/wiki/Field_extension" title="Field extension">field extension</a> of the <a href="/wiki/Rational_number" title="Rational number">rational numbers</a> generated by the <i>n</i>th roots of unity forms a different group. It is isomorphic to the multiplicative group modulo <i>n</i>, which has order φ(<i>n</i>) and is cyclic for some but not all <i>n</i>.</p>
<p>A field extension is called a <a class="mw-redirect" href="/wiki/Cyclic_extension" title="Cyclic extension">cyclic extension</a> if its Galois group is a cyclic group. The Galois group of every finite extension of a <a href="/wiki/Finite_field" title="Finite field">finite field</a> is finite and cyclic, with an iterate of the <a href="/wiki/Frobenius_endomorphism" title="Frobenius endomorphism">Frobenius endomorphism</a> as its generator.<sup class="reference" id="cite_ref-9"><a href="#cite_note-9">[8]</a></sup> Conversely, given a finite field <i>F</i> and a finite cyclic group <i>G</i>, there is a finite field extension of <i>F</i> whose Galois group is <i>G</i>.<sup class="reference" id="cite_ref-10"><a href="#cite_note-10">[9]</a></sup></p>
<h2><span class="mw-headline" id="Subgroups_and_notation">Subgroups and notation</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Cyclic_group&amp;action=edit&amp;section=7" title="Edit section: Subgroups and notation">edit</a><span class="mw-editsection-bracket">]</span></span></h2>
<div class="hatnote" role="note">Main article: <a class="mw-redirect" href="/wiki/Fundamental_theorem_of_cyclic_groups" title="Fundamental theorem of cyclic groups">Fundamental theorem of cyclic groups</a></div>
<p>All <a href="/wiki/Subgroup" title="Subgroup">subgroups</a> and <a href="/wiki/Quotient_group" title="Quotient group">quotient groups</a> of cyclic groups are cyclic. Specifically, all subgroups of <b>Z</b> are of the form <i>m</i><b>Z</b>, with <i>m</i> an integer ≥0. All of these subgroups are distinct from each other, and apart from the trivial group (for <span class="nowrap"><i>m</i> = 0</span>) all are <a class="mw-redirect" href="/wiki/Isomorphic" title="Isomorphic">isomorphic</a> to <b>Z</b>. The <a href="/wiki/Lattice_of_subgroups" title="Lattice of subgroups">lattice of subgroups</a> of <b>Z</b> is isomorphic to the <a href="/wiki/Duality_(order_theory)" title="Duality (order theory)">dual</a> of the lattice of natural numbers ordered by <a class="mw-redirect" href="/wiki/Divisibility" title="Divisibility">divisibility</a>.<sup class="reference" id="cite_ref-11"><a href="#cite_note-11">[10]</a></sup> In particular, because the prime numbers are the numbers with no nontrivial divisors, a cyclic group is <a href="/wiki/Simple_group" title="Simple group">simple</a> if and only if its order (the number of its elements) is prime.<sup class="reference" id="cite_ref-12"><a href="#cite_note-12">[11]</a></sup></p>
<p>Since the cyclic groups are <a href="/wiki/Abelian_group" title="Abelian group">abelian</a>, they are often written additively and denoted Z<sub><i>n</i></sub> with the identity written 0. However, this notation can be problematic for <a href="/wiki/Number_theory" title="Number theory">number theorists</a> because it conflicts with the usual notation for <a href="/wiki/P-adic_number" title="P-adic number"><i>p</i>-adic number</a> rings or <a href="/wiki/Localization_of_a_ring" title="Localization of a ring">localization</a> at a <a href="/wiki/Prime_ideal" title="Prime ideal">prime ideal</a>. The <a href="/wiki/Quotient_group" title="Quotient group">quotient</a> notations <b>Z</b>/<i>n</i><b>Z</b>, <b>Z</b>/<i>n</i>, and <b>Z</b>/(<i>n</i>) are standard alternatives.</p>
<p>One may instead write the group multiplicatively, and denote it by C<sub><i>n</i></sub>, where <i>n</i> is the order for finite groups and by C for the infinite cyclic group.<sup class="reference" id="cite_ref-13"><a href="#cite_note-13">[note 2]</a></sup> For example, <span class="nowrap"><i>g</i><sup>2</sup><i>g</i><sup>4</sup> = <i>g</i><sup>1</sup></span> in C<sub>5</sub>, whereas <span class="nowrap">2 + 4 = 1</span> in <b>Z</b>/5<b>Z</b>.</p>
<p>All quotient groups of <b>Z</b> are finite, except for the trivial exception <span class="nowrap"><b>Z</b>/{0} = <b>Z</b>/0<b>Z</b></span>. For every positive divisor <i>d</i> of <i>n</i>, the quotient group <b>Z</b>/<i>n</i><b>Z</b> has precisely one subgroup of order <i>d</i>, the one generated by the residue class of <i>n</i>/<i>d</i>. There are no other subgroups. Using the quotient group formalism, <b>Z</b>/<i>n</i><b>Z</b> is a standard notation for the additive cyclic group with <i>n</i> elements. In <a href="/wiki/Ring_(mathematics)" title="Ring (mathematics)">ring</a> terminology, the subgroup <i>n</i><b>Z</b> is also the <a href="/wiki/Ideal_(ring_theory)" title="Ideal (ring theory)">ideal</a> (<i>n</i>), so the quotient can also be written <b>Z</b>/(<i>n</i>) or <b>Z</b>/<i>n</i> without abuse of notation. These alternatives do not conflict with the notation for the <i>p</i>-adic integers. The last form is very common in informal calculations; it has the additional advantage that it reads the same way that the group or ring is often described verbally in English, "Zee mod en".</p>
<h2><span class="mw-headline" id="Additional_properties">Additional properties</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Cyclic_group&amp;action=edit&amp;section=8" title="Edit section: Additional properties">edit</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>Every cyclic group is <a href="/wiki/Abelian_group" title="Abelian group">abelian</a>.<sup class="reference" id="cite_ref-eom_1-5"><a href="#cite_note-eom-1">[1]</a></sup> That is, its group operation is <a href="/wiki/Commutative_property" title="Commutative property">commutative</a>: <span class="nowrap"><i>gh</i> = <i>hg</i></span> (for all <i>g</i> and <i>h</i> in <i>G</i>). This is clear for the groups of integer and modular addition since <span class="nowrap"><i>r</i> + <i>s</i> ≡ <i>s</i> + <i>r</i> (mod <i>n</i>)</span>, and it follows for all cyclic groups since they are all isomorphic to a group generated by an addition operation. For a finite cyclic group of order <i>n</i>, and every element <i>e</i> of the group, <i>e</i><sup><i>n</i></sup> is the identity element of the group. This again follows by using the isomorphism to modular addition, since <span class="nowrap"><i>kn</i> ≡ 0 (mod <i>n</i>)</span> for every integer <i>k</i>.</p>
<p>If <i>d</i> is a <a href="/wiki/Divisor" title="Divisor">divisor</a> of <i>n</i>, then the number of elements in <b>Z</b>/<i>n</i> which have order <i>d</i> is φ(<i>d</i>), and the number of elements whose order divides <i>d</i> is exactly <i>d</i>. If <i>G</i> is a finite group in which, for each <span class="nowrap"><i>n</i> &gt; 0</span>, <i>G</i> contains at most <i>n</i> elements of order dividing <i>n</i>, then <i>G</i> must be cyclic.<sup class="reference" id="cite_ref-14"><a href="#cite_note-14">[note 3]</a></sup> The order of an element <i>m</i> of the group is <i>n</i>/<a href="/wiki/Greatest_common_divisor" title="Greatest common divisor">gcd</a>(<i>n</i>,<i>m</i>).</p>
<p>The <a href="/wiki/Direct_product_of_groups" title="Direct product of groups">direct product</a> of two cyclic groups <b>Z</b>/<i>n</i> and <b>Z</b>/<i>m</i> is cyclic if and only if <i>n</i> and <i>m</i> are <a class="mw-redirect" href="/wiki/Coprime" title="Coprime">coprime</a>. Thus e.g. <b>Z</b>/12 is the direct product of <b>Z</b>/3 and <b>Z</b>/4, but not the direct product of <b>Z</b>/6 and <b>Z</b>/2. If <i>p</i> is a <a href="/wiki/Prime_number" title="Prime number">prime number</a>, then the only group (<a href="/wiki/Up_to" title="Up to">up to</a> <a href="/wiki/Group_isomorphism" title="Group isomorphism">isomorphism</a>) with <i>p</i> elements is <b>Z</b>/<i>p</i>. it is called a <a href="/wiki/Primary_cyclic_group" title="Primary cyclic group">primary cyclic group</a>. The <a class="mw-redirect" href="/wiki/Fundamental_theorem_of_finitely_generated_abelian_groups" title="Fundamental theorem of finitely generated abelian groups">fundamental theorem of abelian groups</a> states that every <a href="/wiki/Finitely_generated_abelian_group" title="Finitely generated abelian group">finitely generated abelian group</a> is the direct product of finitely many finite primary cyclic and infinite cyclic groups. A number <i>n</i> is called a <a href="/wiki/Cyclic_number_(group_theory)" title="Cyclic number (group theory)">cyclic number</a> if it has the property that <b>Z</b>/<i>n</i> is the only group of order <i>n</i>, which is true exactly when gcd(<i>n</i>,φ(<i>n</i>)) = 1.<sup class="reference" id="cite_ref-15"><a href="#cite_note-15">[12]</a></sup> The cyclic numbers include all prime numbers, but also include some <a href="/wiki/Composite_number" title="Composite number">composite numbers</a> such as 15. However, except 2, all cyclic numbers are odd. The cyclic numbers are:</p>
<dl>
<dd>1, 2, 3, 5, 7, 11, 13, 15, 17, 19, 23, 29, 31, 33, 35, 37, 41, 43, 47, 51, 53, 59, 61, 65, 67, 69, 71, 73, 77, 79, 83, 85, 87, 89, 91, 95, 97, 101, 103, 107, 109, 113, 115, 119, 123, 127, 131, 133, 137, 139, 141, 143, ... (sequence <span class="nowrap"><a class="extiw" href="//oeis.org/A003277" title="oeis:A003277">A003277</a></span> in the <a href="/wiki/On-Line_Encyclopedia_of_Integer_Sequences" title="On-Line Encyclopedia of Integer Sequences">OEIS</a>)</dd>
</dl>
<p>In fact, a number <i>n</i> is a cyclic number <a href="/wiki/If_and_only_if" title="If and only if">if and only if</a> <span class="nowrap"><a href="/wiki/Greatest_common_divisor" title="Greatest common divisor">gcd</a>(<i>n</i>, <i>φ</i>(<i>n</i>)) = 1</span>, where <i>φ</i> is the <a href="/wiki/Euler%27s_totient_function" title="Euler's totient function">Euler's totient function</a>.</p>
<p>The definition immediately implies that cyclic groups have <a href="/wiki/Presentation_of_a_group" title="Presentation of a group">group presentation</a> <span class="nowrap">C<sub>∞</sub> = ⟨<i>x</i> |⟩</span> and <span class="nowrap">C<sub><i>n</i></sub> = ⟨<i>x</i> | <i>x</i><sup><i>n</i></sup>⟩</span> for finite <i>n</i>.<sup class="reference" id="cite_ref-16"><a href="#cite_note-16">[13]</a></sup></p>
<h2><span class="mw-headline" id="Associated_objects">Associated objects</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Cyclic_group&amp;action=edit&amp;section=9" title="Edit section: Associated objects">edit</a><span class="mw-editsection-bracket">]</span></span></h2>
<h3><span class="mw-headline" id="Representations">Representations</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Cyclic_group&amp;action=edit&amp;section=10" title="Edit section: Representations">edit</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>The <a href="/wiki/Representation_theory_of_finite_groups" title="Representation theory of finite groups">representation theory</a> of the cyclic group is a critical base case for the representation theory of more general finite groups. In the <a href="/wiki/Character_theory" title="Character theory">complex case</a>, a representation of a cyclic group decomposes into a direct sum of linear characters, making the connection between character theory and representation theory transparent. In the <a href="/wiki/Modular_representation_theory" title="Modular representation theory">positive characteristic case</a>, the indecomposable representations of the cyclic group form a model and inductive basis for the representation theory of groups with cyclic <a class="mw-redirect" href="/wiki/Sylow_subgroup" title="Sylow subgroup">Sylow subgroups</a> and more generally the representation theory of blocks of cyclic defect.</p>
<h3><span class="mw-headline" id="Cycle_graph">Cycle graph</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Cyclic_group&amp;action=edit&amp;section=11" title="Edit section: Cycle graph">edit</a><span class="mw-editsection-bracket">]</span></span></h3>
<div class="hatnote" role="note">Further information: <a href="/wiki/Cycle_graph_(algebra)" title="Cycle graph (algebra)">Cycle graph (algebra)</a></div>
<p>A <b>cycle graph</b> illustrates the various cycles of a <a href="/wiki/Group_(mathematics)" title="Group (mathematics)">group</a> and is particularly useful in visualizing the structure of small <a href="/wiki/Finite_group" title="Finite group">finite groups</a>. A cycle graph for a cyclic group is simply a <a class="mw-redirect" href="/wiki/Circular_graph" title="Circular graph">circular graph</a>, where the group order is equal to the number of nodes. A single generator defines the group as a directional path on the graph, and the inverse generator defines a backwards path. Trivial paths (identity) can be drawn as a <a href="/wiki/Loop_(graph_theory)" title="Loop (graph theory)">loop</a> but are usually suppressed. Z<sub>2</sub> is sometimes drawn with two curved edges as a <a href="/wiki/Multigraph" title="Multigraph">multigraph</a>.<sup class="reference" id="cite_ref-17"><a href="#cite_note-17">[14]</a></sup></p>
<p>Cyclic groups Z<sub><i>n</i></sub>, order <i>n</i>, is a single cycle graphed simply as an <i>n</i>-sided polygon with the elements at the vertices. A cyclic group Z<sub><i>n</i></sub> can be decomposed into a <a href="/wiki/Direct_product" title="Direct product">direct product</a> Z<sub><i>a</i></sub>×Z<sub><i>b</i></sub> where <i>n</i>=<i>ab</i>, where <i>a</i> and <i>b</i> are <a class="mw-redirect" href="/wiki/Relatively_prime" title="Relatively prime">relatively prime</a> (<a href="/wiki/Greatest_common_divisor" title="Greatest common divisor">gcd</a>(a,b)=1).</p>

<h3><span class="mw-headline" id="Cayley_graph">Cayley graph</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Cyclic_group&amp;action=edit&amp;section=12" title="Edit section: Cayley graph">edit</a><span class="mw-editsection-bracket">]</span></span></h3>
<div class="hatnote" role="note">Main article: <a href="/wiki/Circulant_graph" title="Circulant graph">Circulant graph</a></div>
<div class="thumb tright">
<div class="thumbinner" style="width:242px;"><a class="image" href="/wiki/File:Paley13.svg"><img alt="" class="thumbimage" data-file-height="495" data-file-width="495" height="240" src="//upload.wikimedia.org/wikipedia/commons/thumb/0/05/Paley13.svg/240px-Paley13.svg.png" srcset="//upload.wikimedia.org/wikipedia/commons/thumb/0/05/Paley13.svg/360px-Paley13.svg.png 1.5x, //upload.wikimedia.org/wikipedia/commons/thumb/0/05/Paley13.svg/480px-Paley13.svg.png 2x" width="240"/></a>
<div class="thumbcaption">
<div class="magnify"><a class="internal" href="/wiki/File:Paley13.svg" title="Enlarge"></a></div>
The <a href="/wiki/Paley_graph" title="Paley graph">Paley graph</a> of order 13, a circulant graph formed as the Cayley graph of <b>Z</b>/13 with generator set {1,3,4}</div>
</div>
</div>
<p>A <a href="/wiki/Cayley_graph" title="Cayley graph">Cayley graph</a> is a graph defined from a pair (<i>G</i>,<i>S</i>) where <i>G</i> is a group and <i>S</i> is a set of generators for the group; it has a vertex for each group element, and an edge for each product of an element with a generator. In the case of a finite cyclic group, with its single generator, the Cayley graph is a <a href="/wiki/Cycle_graph" title="Cycle graph">cycle graph</a>, and for an infinite cyclic group with its generator the Cayley graph is a doubly infinite <a href="/wiki/Path_graph" title="Path graph">path graph</a>. However, Cayley graphs can be defined from other sets of generators as well. The Cayley graphs of cyclic groups with arbitrary generator sets are called <a href="/wiki/Circulant_graph" title="Circulant graph">circulant graphs</a>.<sup class="reference" id="cite_ref-18"><a href="#cite_note-18">[15]</a></sup> These graphs may be represented geometrically as a set of equally spaced points on a circle or on a line, with each point connected to neighbors with the same set of distances as each other point. They are exactly the <a href="/wiki/Vertex-transitive_graph" title="Vertex-transitive graph">vertex-transitive graphs</a> whose <a href="/wiki/Graph_automorphism" title="Graph automorphism">symmetry group</a> includes a transitive cyclic group.<sup class="reference" id="cite_ref-19"><a href="#cite_note-19">[16]</a></sup></p>
<h3><span class="mw-headline" id="Endomorphisms">Endomorphisms</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Cyclic_group&amp;action=edit&amp;section=13" title="Edit section: Endomorphisms">edit</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>The <a href="/wiki/Endomorphism_ring" title="Endomorphism ring">endomorphism ring</a> of the abelian group <b>Z</b>/<i>n</i><b>Z</b> is <a href="/wiki/Ring_homomorphism" title="Ring homomorphism">isomorphic</a> to <b>Z</b>/<i>n</i><b>Z</b> itself as a <a class="mw-redirect" href="/wiki/Ring_(algebra)" title="Ring (algebra)">ring</a>.<sup class="reference" id="cite_ref-ks_20-0"><a href="#cite_note-ks-20">[17]</a></sup> Under this isomorphism, the number <i>r</i> corresponds to the endomorphism of <b>Z</b>/<i>n</i><b>Z</b> that maps each element to the sum of <i>r</i> copies of it. This is a bijection if and only if <i>r</i> is coprime with <i>n</i>, so the <a class="mw-redirect" href="/wiki/Automorphism_group" title="Automorphism group">automorphism group</a> of <b>Z</b>/<i>n</i><b>Z</b> is isomorphic to the unit group (<b>Z</b>/<i>n</i><b>Z</b>)<sup>×</sup>.<sup class="reference" id="cite_ref-ks_20-1"><a href="#cite_note-ks-20">[17]</a></sup></p>
<p>Similarly, the endomorphism ring of the additive group of <b>Z</b> is isomorphic to the ring <b>Z</b>. Its automorphism group is isomorphic to the group of units of the ring <b>Z</b>, i.e. to <span class="nowrap">({−1, +1}, ×) ≅ C<sub>2</sub></span>.</p>

<h2><span class="mw-headline" id="Related_classes_of_groups">Related classes of groups</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Cyclic_group&amp;action=edit&amp;section=15" title="Edit section: Related classes of groups">edit</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>Several other classes of groups have been defined by their relation to the cyclic groups:</p>
<h3><span class="mw-headline" id="Virtually_cyclic_groups">Virtually cyclic groups</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Cyclic_group&amp;action=edit&amp;section=16" title="Edit section: Virtually cyclic groups">edit</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>A group is called <b>virtually cyclic</b> if it contains a cyclic subgroup of finite <a class="mw-redirect" href="/wiki/Index_(group_theory)" title="Index (group theory)">index</a> (the number of <a href="/wiki/Coset" title="Coset">cosets</a> that the subgroup has). In other words, any element in a virtually cyclic group can be arrived at by applying a member of the cyclic subgroup to a member in a certain finite set. Every cyclic group is virtually cyclic, as is every finite group. An infinite group is virtually cyclic if and only if it is <a href="/wiki/Finitely_generated_group" title="Finitely generated group">finitely generated</a> and has exactly two <a href="/wiki/End_(graph_theory)" title="End (graph theory)">ends</a>;<sup class="reference" id="cite_ref-21"><a href="#cite_note-21">[note 4]</a></sup> an example of such a group is the <a href="/wiki/Direct_product_of_groups" title="Direct product of groups">product</a> of <b>Z</b>/<i>n</i> and <b>Z</b>, in which the <b>Z</b> factor has finite index <i>n</i>. Every abelian subgroup of a <a href="/wiki/Hyperbolic_group" title="Hyperbolic group">Gromov hyperbolic group</a> is virtually cyclic.<sup class="reference" id="cite_ref-22"><a href="#cite_note-22">[18]</a></sup></p>
<h3><span class="mw-headline" id="Locally_cyclic_groups">Locally cyclic groups</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Cyclic_group&amp;action=edit&amp;section=17" title="Edit section: Locally cyclic groups">edit</a><span class="mw-editsection-bracket">]</span></span></h3>
<div class="hatnote" role="note">Main article: <a href="/wiki/Locally_cyclic_group" title="Locally cyclic group">Locally cyclic group</a></div>
<p>A <a href="/wiki/Locally_cyclic_group" title="Locally cyclic group">locally cyclic group</a> is a group in which each <a href="/wiki/Finitely_generated_group" title="Finitely generated group">finitely generated</a> subgroup is cyclic. An example is the additive group of the <a href="/wiki/Rational_number" title="Rational number">rational numbers</a>: every finite set of rational numbers is a set of integer multiples of a single <a href="/wiki/Unit_fraction" title="Unit fraction">unit fraction</a>, the inverse of their <a href="/wiki/Lowest_common_denominator" title="Lowest common denominator">lowest common denominator</a>, and generates as a subgroup a cyclic group of integer multiples of this unit fraction. A group is locally cyclic if and only if its <a href="/wiki/Lattice_of_subgroups" title="Lattice of subgroups">lattice of subgroups</a> is a <a href="/wiki/Distributive_lattice" title="Distributive lattice">distributive lattice</a>.<sup class="reference" id="cite_ref-23"><a href="#cite_note-23">[19]</a></sup></p>
<h3><span class="mw-headline" id="Cyclically_ordered_groups">Cyclically ordered groups</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Cyclic_group&amp;action=edit&amp;section=18" title="Edit section: Cyclically ordered groups">edit</a><span class="mw-editsection-bracket">]</span></span></h3>
<div class="hatnote" role="note">Main article: <a href="/wiki/Cyclically_ordered_group" title="Cyclically ordered group">Cyclically ordered group</a></div>
<p>A <a href="/wiki/Cyclically_ordered_group" title="Cyclically ordered group">cyclically ordered group</a> is a group together with a <a href="/wiki/Cyclic_order" title="Cyclic order">cyclic order</a> preserved by the group structure. Every cyclic group can be given a structure as a cyclically ordered group, consistent with the ordering of the integers (or the integers modulo the order of the group). Every finite subgroup of a cyclically ordered group is cyclic.<sup class="reference" id="cite_ref-24"><a href="#cite_note-24">[20]</a></sup></p>
<h3><span class="mw-headline" id="Metacyclic_and_polycyclic_groups">Metacyclic and polycyclic groups</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Cyclic_group&amp;action=edit&amp;section=19" title="Edit section: Metacyclic and polycyclic groups">edit</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>A <a href="/wiki/Metacyclic_group" title="Metacyclic group">metacyclic group</a> is a group containing a cyclic <a href="/wiki/Normal_subgroup" title="Normal subgroup">normal subgroup</a> whose quotient is also cyclic.<sup class="reference" id="cite_ref-25"><a href="#cite_note-25">[21]</a></sup> These groups include the cyclic groups, the <a href="/wiki/Dicyclic_group" title="Dicyclic group">dicyclic groups</a>, and the <a href="/wiki/Direct_product" title="Direct product">direct products</a> of two cyclic groups. The <a href="/wiki/Polycyclic_group" title="Polycyclic group">polycyclic groups</a> generalize metacyclic groups by allowing more than one level of group extension. A group is polycyclic if it has a finite descending sequence of subgroups, each of which is normal in the previous subgroup with a cyclic quotient, ending in the trivial group. Every finitely generated <a href="/wiki/Abelian_group" title="Abelian group">abelian group</a> or <a href="/wiki/Nilpotent_group" title="Nilpotent group">nilpotent group</a> is polycyclic.<sup class="reference" id="cite_ref-26"><a href="#cite_note-26">[22]</a></sup></p>
