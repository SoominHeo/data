<p>The <b>frosted sac-winged bat</b> (<i>Saccopteryx canescens</i>) is a <a href="/wiki/Bat" title="Bat">bat</a> <a href="/wiki/Species" title="Species">species</a> of the family <a class="mw-redirect" href="/wiki/Emballonuridae" title="Emballonuridae">Emballonuridae</a> from <a href="/wiki/South_America" title="South America">South America</a>. It is found in northern <a href="/wiki/Brazil" title="Brazil">Brazil</a>, <a href="/wiki/Colombia" title="Colombia">Colombia</a>, <a href="/wiki/Ecuador" title="Ecuador">Ecuador</a>, <a class="mw-redirect" href="/wiki/French_Guyana" title="French Guyana">French Guyana</a>, <a href="/wiki/Guyana" title="Guyana">Guyana</a>, <a href="/wiki/Peru" title="Peru">Peru</a>, <a class="mw-redirect" href="/wiki/Surinam" title="Surinam">Surinam</a>, <a href="/wiki/Venezuela" title="Venezuela">Venezuela</a> and possibly <a href="/wiki/Bolivia" title="Bolivia">Bolivia</a>.<sup class="reference" id="cite_ref-iucn_1-1"><a href="#cite_note-iucn-1">[1]</a></sup><sup class="reference" id="cite_ref-Simmons_2-0"><a href="#cite_note-Simmons-2">[2]</a></sup></p>
