<p><b>BKChem</b> is a free 2D <a href="/wiki/Molecule_editor" title="Molecule editor">molecule editor</a> written in <a href="/wiki/Python_(programming_language)" title="Python (programming language)">Python</a> by Beda Kosata. As such, it is largely <a href="/wiki/Cross-platform" title="Cross-platform">platform-independent</a>. It uses <a class="mw-redirect" href="/wiki/Tk" title="Tk">Tk</a> via <a href="/wiki/Tkinter" title="Tkinter">Tkinter</a>.</p>
<h2><span class="mw-headline" id="Major_features">Major features</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=BKChem&amp;action=edit&amp;section=1" title="Edit section: Major features">edit</a><span class="mw-editsection-bracket">]</span></span></h2>
<div class="thumb tright">
<div class="thumbinner" style="width:306px;"><a class="image" href="/wiki/File:Fisetin.svg"><img alt="" class="thumbimage" data-file-height="188" data-file-width="346" height="165" src="//upload.wikimedia.org/wikipedia/commons/thumb/9/91/Fisetin.svg/304px-Fisetin.svg.png" srcset="//upload.wikimedia.org/wikipedia/commons/thumb/9/91/Fisetin.svg/456px-Fisetin.svg.png 1.5x, //upload.wikimedia.org/wikipedia/commons/thumb/9/91/Fisetin.svg/608px-Fisetin.svg.png 2x" width="304"/></a>
<div class="thumbcaption">
<div class="magnify"><a class="internal" href="/wiki/File:Fisetin.svg" title="Enlarge"></a></div>
Structure drawn originally with BKChem<br/>
Redrawn because of W3C-invalidity (with <a href="/wiki/Inkscape" title="Inkscape">Inkscape</a>)</div>
</div>
</div>
<p>Drawing:</p>
<ul>
<li>Bond by bond drawing</li>
<li>Straight and curved arrows</li>
<li><a href="/wiki/Radical_(chemistry)" title="Radical (chemistry)">Radicals</a>, <a href="/wiki/Lone_pair" title="Lone pair">lone pairs</a> and <a href="/wiki/Ion" title="Ion">charges</a></li>
<li>Templates for drawing common structure (users can create their own)</li>
<li>Simple <a href="/wiki/Vector_graphics" title="Vector graphics">vector graphics</a> using <a href="/wiki/Sodipodi" title="Sodipodi">Sodipodi</a>-Inkscape</li>
</ul>
<p>Editing:</p>
<ul>
<li>Unlimited undo/redo capabilities</li>
<li>Aligning, scaling and rotation (2D and 3D)</li>
</ul>
<p>Exports to:</p>
<ul>
<li><a class="mw-redirect" href="/wiki/SVG" title="SVG">SVG</a></li>
<li><a class="mw-redirect" href="/wiki/OpenOffice.org_Draw" title="OpenOffice.org Draw">OpenOffice Draw</a></li>
<li><a class="mw-redirect" href="/wiki/PDF" title="PDF">PDF</a></li>
<li><a href="/wiki/Encapsulated_PostScript" title="Encapsulated PostScript">EPS</a> (Encapsulated Postscript)</li>
<li><a class="mw-redirect" href="/wiki/Molfile" title="Molfile">Molfile</a></li>
<li><a href="/wiki/Portable_Network_Graphics" title="Portable Network Graphics">PNG</a> (with <a class="external text" href="http://www.cairographics.org/pycairo/" rel="nofollow">Pycairo</a> installed)</li>
<li><a class="mw-redirect" href="/wiki/SMILES" title="SMILES">SMILES</a> (Stereochemistry is not supported)</li>
<li><a class="mw-redirect" href="/wiki/InChI" title="InChI">InChI</a> (with InChI program)</li>
</ul>
<p>Imports:</p>
<ul>
<li><a class="mw-redirect" href="/wiki/Molfile" title="Molfile">Molfile</a></li>
<li><a class="mw-redirect" href="/wiki/SMILES" title="SMILES">SMILES</a> (Stereochemistry is not supported)</li>
<li><a class="mw-redirect" href="/wiki/InChI" title="InChI">InChI</a> (Stereochemistry is not supported)</li>
</ul>
<p>Other features:</p>
<ul>
<li>Searching BKChem files for a molecule or a fragment of a molecule</li>
<li>Localization support (currently English, French, Czech and Polish translations are available)</li>
<li>Support for user written <a class="mw-redirect" href="/wiki/Plugins" title="Plugins">plugins</a></li>
<li>Validity checking</li>
<li>Can be run from source with <a href="/wiki/Python_(programming_language)" title="Python (programming language)">Python</a></li>
</ul>
