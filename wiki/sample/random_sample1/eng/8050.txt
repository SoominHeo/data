<p><b>SUPER ©</b>  [<i><a href="/wiki/Sic" title="Sic">sic</a></i>] (<i><b>S</b>implified <b>U</b>niversal <b>P</b>layer <b>E</b>ncoder &amp; <b>R</b>ecoder</i>) is a <a class="mw-redirect" href="/wiki/Closed-source" title="Closed-source">closed-source</a> <a href="/wiki/Adware" title="Adware">adware</a> <a href="/wiki/Front_and_back_ends" title="Front and back ends">front-end</a> for <a class="mw-redirect" href="/wiki/Open-source" title="Open-source">open-source</a> software video players and encoders provided by the <a href="/wiki/FFmpeg" title="FFmpeg">FFmpeg</a>, <a href="/wiki/MEncoder" title="MEncoder">MEncoder</a>, <a href="/wiki/MPlayer" title="MPlayer">MPlayer</a>, <a href="/wiki/X264" title="X264">x264</a>, ffmpeg2theora, musepack, Monkey's Audio, True Audio, WavPack, <a href="/wiki/Libavcodec" title="Libavcodec">libavcodec</a>, and the <a href="/wiki/Theora" title="Theora">Theora</a>/<a href="/wiki/Vorbis" title="Vorbis">Vorbis</a> RealProducer plugIn projects. SUPER © provides a <a href="/wiki/Graphical_user_interface" title="Graphical user interface">graphical user interface</a> to these <a href="/wiki/Front_and_back_ends" title="Front and back ends">back-end</a> programs, which are <a class="mw-redirect" href="/wiki/Command-line" title="Command-line">command-line</a> based.</p>

<h2><span class="mw-headline" id="Features">Features</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=SUPER_(computer_programme)&amp;action=edit&amp;section=1" title="Edit section: Features">edit</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>SUPER can manipulate and produce many multimedia file formats supported by its back-end programs.</p>
<p>As of 2016, SUPER has a built in enhanced 3D Video Converter &amp; Recorder engine. The proposed 3D variations are: 3D Anaglyph, Polarized or Shutter side-by-side.</p>
<p>Back-end program features supported by SUPER © include saving various streaming protocols (<a href="/wiki/Microsoft_Media_Server" title="Microsoft Media Server">mms</a>, <a class="mw-redirect" href="/wiki/Rtsp" title="Rtsp">rtsp</a>, and <a class="mw-redirect" href="/wiki/Http" title="Http">http</a>), conversion of <a href="/wiki/Flash_Video" title="Flash Video">Flash Video</a> to other formats, and user-controlled conversion of video between different container formats. Users can choose between various lossless direct audio/video transfers between <a class="mw-redirect" href="/wiki/Container_formats" title="Container formats">container formats</a> or lossy video/audio encoding, with encoding possessing the added ability to change video and audio specifications such as bitrate, <a href="/wiki/Frame_rate" title="Frame rate">frame rate</a>, audio channels, resolution, sampling rate, and aspect ratio. SUPER © is also able to utilize its back-end's built-in media players, allowing playback of supported video and audio formats.</p>

<h2><span class="mw-headline" id="System_requirements">System requirements</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=SUPER_(computer_programme)&amp;action=edit&amp;section=6" title="Edit section: System requirements">edit</a><span class="mw-editsection-bracket">]</span></span></h2>
<ul>
<li>Operating System: <a class="mw-redirect" href="/wiki/Windows_98_SE" title="Windows 98 SE">Windows 98 SE</a>, <a class="mw-redirect" href="/wiki/Windows_Me" title="Windows Me">Windows Me</a>, <a href="/wiki/Windows_2000" title="Windows 2000">Windows 2000</a>, <a href="/wiki/Windows_XP" title="Windows XP">Windows XP</a>, <a class="mw-redirect" href="/wiki/Windows_2003" title="Windows 2003">Windows 2003</a>, <a href="/wiki/Windows_Vista" title="Windows Vista">Windows Vista</a>, <a href="/wiki/Windows_7" title="Windows 7">Windows 7</a> or <a href="/wiki/Windows_8" title="Windows 8">Windows 8</a>.</li>
<li>Administrator privileges (if installed on Administrative account then subsequently converted to a Limited account during the same session, the program will work on the limited)</li>
<li>Processor: Minimum 1.8 GHz <a href="/wiki/Intel" title="Intel">Intel</a> <a href="/wiki/Pentium_4" title="Pentium 4">Pentium 4</a> processor or equivalent.</li>
<li>512MB of <a class="mw-redirect" href="/wiki/Random_access_memory" title="Random access memory">RAM</a> with 176MB available.</li>
<li>20GB of free space on the Hard disk where the OS is installed</li>
<li>1024 × 768 video resolution or larger</li>
<li>32,000-color video or more</li>
<li><a href="/wiki/Internet_Explorer_5" title="Internet Explorer 5">IE 5.01</a> or later</li>
<li>Internet connection</li>
</ul>
<p>SUPER © is capable of working on machines with lesser capabilities; the actual minimum system requirements depend fully on the back-end programs and settings chosen by the program user.</p>
