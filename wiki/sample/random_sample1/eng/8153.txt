<p><b>SketchUp</b> (formerly <b>Google Sketchup</b>) is a <a href="/wiki/3D_modeling" title="3D modeling">3D modeling</a> <a href="/wiki/Computer_program" title="Computer program">computer program</a> for a wide range of drawing applications such as <a href="/wiki/Architect" title="Architect">architectural</a>, interior design, landscape architecture, <a href="/wiki/Civil_engineer" title="Civil engineer">civil</a> and <a href="/wiki/Mechanical_engineering" title="Mechanical engineering">mechanical engineering</a>, film, and <a href="/wiki/Video_game_developer" title="Video game developer">video game</a> design—and available in a freeware version, <b>SketchUp Make</b>, and a paid version with additional functionality, <b>SketchUp Pro</b>.</p>
<p>SketchUp is currently owned by <a class="mw-redirect" href="/wiki/Trimble_Navigation" title="Trimble Navigation">Trimble Navigation</a>,<sup class="reference" id="cite_ref-reuters-trimble-buys_1-1"><a href="#cite_note-reuters-trimble-buys-1">[1]</a></sup><sup class="reference" id="cite_ref-4"><a href="#cite_note-4">[4]</a></sup> a mapping, surveying, and navigation equipment company.<sup class="reference" id="cite_ref-5"><a href="#cite_note-5">[5]</a></sup> The program's authors describe it as easy to use.<sup class="reference" id="cite_ref-6"><a href="#cite_note-6">[6]</a></sup> There is an online <a class="mw-redirect" href="/wiki/Open_source" title="Open source">open source</a> library of free model assemblies (e.g. windows, doors, automobiles), <a href="/wiki/3D_Warehouse" title="3D Warehouse">3D Warehouse</a>, to which users may contribute models. The program includes drawing layout functionality, allows surface rendering in variable "styles", supports third-party "plug-in" programs hosted on a site called Extension Warehouse to provide other capabilities (e.g. near photo-realistic rendering), and enables placement of its models within <a href="/wiki/Google_Earth" title="Google Earth">Google Earth</a>.<sup class="reference" id="cite_ref-7"><a href="#cite_note-7">[7]</a></sup></p>

<h2><span class="mw-headline" id="History">History</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=SketchUp&amp;action=edit&amp;section=1" title="Edit section: History">edit</a><span class="mw-editsection-bracket">]</span></span></h2>
<h3><span class="mw-headline" id=".40Last_Software">@Last Software</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=SketchUp&amp;action=edit&amp;section=2" title="Edit section: @Last Software">edit</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>SketchUp was developed by startup company @Last Software of <a href="/wiki/Boulder,_Colorado" title="Boulder, Colorado">Boulder, Colorado</a>, co-founded in 1999 by Brad Schell and Joe Esch.<sup class="reference" id="cite_ref-8"><a href="#cite_note-8">[8]</a></sup><sup class="reference" id="cite_ref-it.tmcnet_9-0"><a href="#cite_note-it.tmcnet-9">[9]</a></sup></p>
<p>SketchUp debuted in August 2000 as a general-purpose 3D content creation tool, and was envisioned as a software program "that would allow design professionals to draw the way they want by emulating the feel and freedom of working with pen and paper in a simple and elegant interface, that would be fun to use and easy to learn, and that would be used by designers to play with their designs in a way that is not possible with traditional design software. It also has user friendly buttons to make it easier to use."<sup class="reference" id="cite_ref-sysreq_3-1"><a href="#cite_note-sysreq-3">[3]</a></sup></p>
<p>The program won a Community Choice Award at its first tradeshow in 2000.<sup class="reference" id="cite_ref-10"><a href="#cite_note-10">[10]</a></sup></p>

<p>Google acquired @Last Software on March 14, 2006 for an undisclosed sum,<sup class="reference" id="cite_ref-11"><a href="#cite_note-11">[11]</a></sup> attracted by @Last Software's work developing a <a href="/wiki/Plug-in_(computing)" title="Plug-in (computing)">plugin</a> for Google Earth.</p>
<p>On January 9, 2007, Google announced Google SketchUp 6, a free downloadable version of SketchUp, without some functionality of SketchUp Pro, but including integrated tools for uploading content to Google Earth and to the Google 3D Warehouse. A toolbox enables a viewer to "walk around" and see things from different viewpoints, and supports labels for models, a look-around tool, and an "any polygon" shape tool. Google SketchUp Pro 6 introduced a <a class="mw-redirect" href="/wiki/Beta_version" title="Beta version">beta version</a> of <i>Google SketchUp LayOut</i>. LayOut includes 2D vector tools and page layout tools allowing presentations to be produced without the need for a separate presentation program. <sup class="reference" id="cite_ref-12"><a href="#cite_note-12">[12]</a></sup></p>
<p>On November 17, 2008, <i>SketchUp 7</i> was released, with changes intended to make it easier to use, integration of SketchUp's Component Browser with Google 3D Warehouse, LayOut 2, and dynamic components that respond to scaling. <a href="/wiki/Windows_2000" title="Windows 2000">Windows 2000</a> was no longer supported.<sup class="reference" id="cite_ref-sysreq_3-2"><a href="#cite_note-sysreq-3">[3]</a></sup></p>
<p>On September 1, 2010, <i>SketchUp 8</i> was released, with model <a href="/wiki/Geolocation" title="Geolocation">geolocation</a> with <a href="/wiki/Google_Maps" title="Google Maps">Google Maps</a> and <a href="/wiki/Google_Building_Maker" title="Google Building Maker">Building Maker</a> integration. <a href="/wiki/Mac_OS_X_Tiger" title="Mac OS X Tiger">Mac OS X Tiger</a> was no longer supported.<sup class="reference" id="cite_ref-sysreq_3-3"><a href="#cite_note-sysreq-3">[3]</a></sup></p>
<p>Neither the free version nor the professional version was available in a native format for Linux, or Mac OS earlier than 10.5. SketchUp version 8 use under <a href="/wiki/Wine_(software)" title="Wine (software)">Wine</a> has been rated "<a href="/wiki/Wine_(software)#Functionality" title="Wine (software)">Gold</a>".<sup class="reference" id="cite_ref-13"><a href="#cite_note-13">[13]</a></sup></p>
<p>Geolocation information is always stored in the KMZ file.<sup class="reference" id="cite_ref-14"><a href="#cite_note-14">[14]</a></sup> The building designs themselves are saved in SKP.</p>
<h3><span class="mw-headline" id="Trimble">Trimble</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=SketchUp&amp;action=edit&amp;section=4" title="Edit section: Trimble">edit</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>Trimble Navigation acquired SketchUp from Google on June 1, 2012 for an undisclosed sum.<sup class="reference" id="cite_ref-15"><a href="#cite_note-15">[15]</a></sup></p>
<p>In 2013 SketchUp 2013 was released. A new site was provided, Extension Warehouse, hosting plugins and extensions for Sketchup.<sup class="reference" id="cite_ref-16"><a href="#cite_note-16">[16]</a></sup></p>
<p>Sketchup 2014 was released on February 26, 2014.</p>
<p>Sketchup 2015 was released on November 4, 2014.</p>
<p>Sketchup 2016 was released on November 17, 2015.</p>
<p>Sketchup 2017 was released on November 7, 2016.</p>

<p>SketchUp <a href="/wiki/3D_Warehouse" title="3D Warehouse">3D Warehouse</a> is an open source library in which SketchUp users may upload and download 3D models to share. The models can be downloaded right into the program without anything having to be saved onto your computers storage. File sizes of the models can be up to 50 MB. Anyone can make, modify, and re-upload content to and from the 3D warehouse free of charge.<sup class="reference" id="cite_ref-17"><a href="#cite_note-17">[17]</a></sup> All the models in 3D Warehouse are free, so anyone can download files for use in Sketchup or even other software such as Autocad, Revit and Archicad - all of which have apps allowing the retrieval of models from 3D Warehouse. Since 2014 Trimble has launched a new version of 3D Warehouse where companies may have an official page with their own 3D catalog of products. Trimble is currently investing in creating 3D developer partners in order to have more professionally modeled products available in 3D Warehouse. According to the Trimble, 3D Warehouse is the most popular 3D content site on the web. Sketchup designers may visit 3D Warehouse to discover new products or for inspiration when designing their own.</p>
<p>Statistics relating to 3D Warehouse:</p>
<ul>
<li>There are more than 2.2 million 3D models for download</li>
<li>In 2015, the website had 1.7 billion page views</li>
<li>The global average duration of site visits is 11min and 38 sec</li>
<li>The site has 7.7 million monthly visits</li>
<li>The site has 23 million monthly searches</li>
</ul>
<p>(this information was collected from the Trimble 3D Warehouse media kit for manufacture in 2015)</p>
<h2><span class="mw-headline" id="Licensing">Licensing</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=SketchUp&amp;action=edit&amp;section=6" title="Edit section: Licensing">edit</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>All versions of SketchUp are <a href="/wiki/Proprietary_software" title="Proprietary software">proprietary</a>. SketchUp Make (formerly SketchUp for Home and Personal Use), for non-commercial use, is free of charge.<sup class="reference" id="cite_ref-18"><a href="#cite_note-18">[18]</a></sup></p>
<p>SketchUp Pro provides additional features for professional use and costs $695 for a single user license. Users purchase the program for $695, and thereafter optionally pay a $120 annual license-maintenance fee for continuing support and upgrades.<sup class="reference" id="cite_ref-19"><a href="#cite_note-19">[19]</a></sup> There is special pricing for students and educational users.<sup class="reference" id="cite_ref-20"><a href="#cite_note-20">[20]</a></sup></p>
<p>SketchUp Pro licensing is cross-platform and works on both Windows and Mac machines.</p>
<h2><span class="mw-headline" id="Patents">Patents</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=SketchUp&amp;action=edit&amp;section=7" title="Edit section: Patents">edit</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>SketchUp holds <span><a class="external text" href="//www.google.com/patents/US6628279" rel="nofollow">U.S. Patent 6,628,279</a></span>, granted in September 2003, on its <i>"Push/Pull"</i> technology.</p>

<p>Sketchup Make, introduced in May 2013, is a free-of-charge version for home, personal, and educational use.<sup class="noprint Inline-Template Template-Fact" style="white-space:nowrap;">[<i><a href="/wiki/Wikipedia:Citation_needed" title="Wikipedia:Citation needed"><span title="This claim needs references to reliable sources. (July 2015)">citation needed</span></a></i>]</sup> It begins with a 30-day trial of SketchUp Pro. After that time, users can agree to the Terms of Service and continue to use SketchUp Make for free.</p>
<h3><span class="mw-headline" id="Pro">Pro</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=SketchUp&amp;action=edit&amp;section=10" title="Edit section: Pro">edit</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>SketchUp Pro includes all the functionality of SketchUp Make plus importers and exporters to common 2D and 3D formats, access to LayOut (2D documentation software), and Style Builder (create custom edge styles for SketchUp models).<sup class="reference" id="cite_ref-21"><a href="#cite_note-21">[21]</a></sup> SketchUp Pro 2016 has native integration with Trimble Connect, treat 3D Warehouse models as references, a totally rebuilt Generate Report, and now LayOut offers web-friendly reference objects as well as a new LayOut API.<sup class="reference" id="cite_ref-22"><a href="#cite_note-22">[22]</a></sup></p>
<h2><span class="mw-headline" id="Software_extensions">Software extensions</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=SketchUp&amp;action=edit&amp;section=11" title="Edit section: Software extensions">edit</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>SketchUp 4 and later support software extensions written in the <a href="/wiki/Ruby_(programming_language)" title="Ruby (programming language)">Ruby programming language</a>, which add specialized functionality. Many such extensions are available to others on the Trimble Extension Warehouse.<sup class="reference" id="cite_ref-23"><a href="#cite_note-23">[23]</a></sup> SketchUp has a Ruby console, an environment which allows experimentation with Ruby.<sup class="reference" id="cite_ref-24"><a href="#cite_note-24">[24]</a></sup></p>
<h2><span class="mw-headline" id="Model_Viewers">Model Viewers</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=SketchUp&amp;action=edit&amp;section=12" title="Edit section: Model Viewers">edit</a><span class="mw-editsection-bracket">]</span></span></h2>
<h3><span class="mw-headline" id="SketchUp_Mobile_Viewer">SketchUp Mobile Viewer</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=SketchUp&amp;action=edit&amp;section=13" title="Edit section: SketchUp Mobile Viewer">edit</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>SketchUp Mobile Viewer is an app released by Trimble.<sup class="reference" id="cite_ref-25"><a href="#cite_note-25">[25]</a></sup> It's available on iOS and Android.</p>
<h3><span class="mw-headline" id="Cubits">Cubits</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=SketchUp&amp;action=edit&amp;section=14" title="Edit section: Cubits">edit</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>Cubits for iPad and iOS is a 3D interactive model viewer/renderer.<sup class="reference" id="cite_ref-26"><a href="#cite_note-26">[26]</a></sup> It can view <a class="mw-redirect" href="/wiki/.kmz" title=".kmz">.kmz</a> files from SketchUp either as attachments through e-mail or via <a href="/wiki/Dropbox_(service)" title="Dropbox (service)">Dropbox</a>.</p>
<h3><span class="mw-headline" id="IrisVR_Prospect">IrisVR Prospect</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=SketchUp&amp;action=edit&amp;section=15" title="Edit section: IrisVR Prospect">edit</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>IrisVR Prospect software allows users to view SketchUp files in virtual reality with the use of head mounted displays such as the HTC Vive or Oculus Rift.<sup class="reference" id="cite_ref-27"><a href="#cite_note-27">[27]</a></sup></p>
<h3><span class="mw-headline" id="Kubity">Kubity</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=SketchUp&amp;action=edit&amp;section=16" title="Edit section: Kubity">edit</a><span class="mw-editsection-bracket">]</span></span></h3>
<p><a href="/wiki/Kubity" title="Kubity">Kubity</a> is an online rendering platform for desktop web browsers and mobile devices (iOS and Android).</p>
