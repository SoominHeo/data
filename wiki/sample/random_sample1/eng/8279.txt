<p><b>Spybot – Search &amp; Destroy</b> (<b>S&amp;D</b>) is a <a href="/wiki/Spyware" title="Spyware">spyware</a> and <a href="/wiki/Adware" title="Adware">adware</a> removal <a href="/wiki/Computer_program" title="Computer program">computer program</a> compatible with <a href="/wiki/Microsoft" title="Microsoft">Microsoft</a> <a href="/wiki/Windows_95" title="Windows 95">Windows 95</a> and later. It scans the computer <a class="mw-redirect" href="/wiki/Hard_disk" title="Hard disk">hard disk</a> and/or <a class="mw-redirect" href="/wiki/Random_Access_Memory" title="Random Access Memory">RAM</a> for malicious <a href="/wiki/Software" title="Software">software</a>.</p>
<p>Spybot-S&amp;D was written by the German <a href="/wiki/Software_engineering" title="Software engineering">software engineer</a> <a href="/wiki/Patrick_Michael_Kolla" title="Patrick Michael Kolla">Patrick Michael Kolla</a>, and is distributed by Kolla's <a href="/wiki/Republic_of_Ireland" title="Republic of Ireland">Irish</a> company Safer-Networking Limited. Development began in 2000 when Kolla, still a student, wrote a small program to deal with the <i>Aureate/Radiate</i> and <i>Conducent TimeSink</i> programs, two of the earliest examples of adware.</p>

<h2><span class="mw-headline" id="Licensing">Licensing</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Spybot_%E2%80%93_Search_%26_Destroy&amp;action=edit&amp;section=1" title="Edit section: Licensing">edit</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>Spybot – Search &amp; Destroy is released as <a href="/wiki/Freeware" title="Freeware">freeware</a> for personal users with more featureful versions available for purchase. Corporate and technician users have to buy a commercial version. The paid versions contain an anti-virus component that uses the <a class="mw-redirect" href="/wiki/BitDefender" title="BitDefender">BitDefender</a> engine.</p>
<h2><span class="mw-headline" id="Spybot_features">Spybot features</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Spybot_%E2%80%93_Search_%26_Destroy&amp;action=edit&amp;section=2" title="Edit section: Spybot features">edit</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>In addition to spyware and adware detection and disinfection, Spybot-S&amp;D can repair the <a class="mw-redirect" href="/wiki/Windows_registry" title="Windows registry">registry</a>, <a href="/wiki/Winsock" title="Winsock">winsock</a> <a href="/wiki/Layered_Service_Provider" title="Layered Service Provider">LSPs</a>, <a href="/wiki/Component_Object_Model" title="Component Object Model">ActiveX</a> objects, <a class="mw-redirect" href="/wiki/Browser_hijacker" title="Browser hijacker">browser hijackers</a> and <a href="/wiki/Browser_Helper_Object" title="Browser Helper Object">BHOs</a>, <a class="mw-redirect" href="/wiki/Potentially_Unwanted_Program" title="Potentially Unwanted Program">PUPs</a>, <a href="/wiki/HTTP_cookie" title="HTTP cookie">HTTP cookies</a>, trackerware, heavy duty, homepage hijackers, <a class="mw-redirect" href="/wiki/Keyloggers" title="Keyloggers">keyloggers</a>, LSP, tracks, trojans, spybots, revision, and other kinds of malware. It can also delete tracking cookies. Spybot-S&amp;D has an Immunize function to block the installation of spyware before it occurs e.g. by modifying the <a class="mw-redirect" href="/wiki/Hosts_file" title="Hosts file">hosts file</a>. A secure file deleter is included. Spybot-S&amp;D was not originally intended to replace <a class="mw-redirect" href="/wiki/Anti-virus_software" title="Anti-virus software">anti-virus programs</a> (prior to v. 2.1 <span class="nowrap" style="padding-left:0.1em;">'</span>Spybot +AV<span class="nowrap" style="padding-left:0.1em;">'</span>), but it does detect some common <a href="/wiki/Trojan_horse_(computing)" title="Trojan horse (computing)">trojans</a> and <a href="/wiki/Rootkit" title="Rootkit">rootkits</a>. A free-standing rootkit finder, <a class="new" href="/w/index.php?title=RootAlyzer&amp;action=edit&amp;redlink=1" title="RootAlyzer (page does not exist)">RootAlyzer</a>, is available.<sup class="reference" id="cite_ref-OfficialSite_3-0"><a href="#cite_note-OfficialSite-3">[3]</a></sup></p>
<p>The TeaTimer module is currently removed for a reason not made clear. The TeaTimer module could be optionally enabled, providing some active, <a class="mw-redirect" href="/wiki/Real-time_protection" title="Real-time protection">real-time protection</a> against unwanted registry changes by alerting the user to registry changes by software, and allowing them only if approved by the user. Legitimate registry changes are mostly (but not always) made when programs are installed, uninstalled, or updated; changes when program modification is not known to be happening can be due to hidden installation of malicious software. If you have another computer (e.g. Windows XP or an older version) that has TeaTimer on it, you can copy it to the new version and it will run.</p>
<p>Some programs are supplied with attached spyware or adware and refuse to run when they are not present; newer versions of Spybot replace the spyware binaries with inert dummies (designed to fool programs which simply check for the presence of the spyware's file).</p>
<p>In order to detect recently created programs efficiently, detection <a href="/wiki/Patch_(computing)" title="Patch (computing)">updates</a> are released weekly with other improvements such as added languages and better <a href="/wiki/Heuristic" title="Heuristic">heuristic</a> <a href="/wiki/Algorithm" title="Algorithm">algorithms</a>. These updates are downloaded and installed from within the software from a variety of <a class="mw-redirect" href="/wiki/Web_mirror" title="Web mirror">mirrors</a>.</p>
<p>Spybot-S&amp;D is available for all versions of <a href="/wiki/Microsoft_Windows" title="Microsoft Windows">Windows</a> starting with <a href="/wiki/Windows_95" title="Windows 95">Windows 95</a>. It is supplied translated into many languages and with several <a href="/wiki/Skin_(computing)" title="Skin (computing)">skins</a>. Instructions are available on the website to enable users to design their own skins.</p>
<p>Technical support is currently supplied by means of Internet forums<sup class="reference" id="cite_ref-4"><a href="#cite_note-4">[4]</a></sup> and support <a class="mw-redirect" href="/wiki/E-mail" title="E-mail">e-mails</a> (with a usual response time of no more than 24 hours).</p>

<p>Versions of Spybot-S&amp;D won awards including the World Class 2003 Awards, the <i><a href="/wiki/PC_Magazine" title="PC Magazine">PC Magazine</a></i> Editor's Choice and <i><a class="mw-redirect" href="/wiki/PC_User" title="PC User">PC User</a></i> Top Buy #1. It has been recommended by <a href="/wiki/ZDNet" title="ZDNet">ZDNet</a>, the <i><a class="mw-redirect" href="/wiki/Wall_Street_Journal" title="Wall Street Journal">Wall Street Journal</a></i>, <i><a href="/wiki/The_Guardian" title="The Guardian">The Guardian</a></i>, <a href="/wiki/MSNBC" title="MSNBC">MSNBC</a>, <a href="/wiki/CNN" title="CNN">CNN</a> and other reviewers.</p>
<p>Although <i>PC Magazine</i> initially rated it highly in 2003, their rating declined to "poor" in 2008 and "dismal" in 2012, improving to "fair" in 2014:</p>

<p>In January 2008, <i>PC Magazine</i> elected it as one of the worst tech products of the first quarter of 2008 and called its <a href="/wiki/Malware" title="Malware">malware</a> cleaning-up skills mediocre.<sup class="reference" id="cite_ref-7"><a href="#cite_note-7">[7]</a></sup></p>
<p>Newer versions of the program (ver. 2.2.25 and ver. 2.4), issued in 2014, received a 5/5 rating from TechRadar.<sup class="reference" id="cite_ref-TR22_8-0"><a href="#cite_note-TR22-8">[8]</a></sup><sup class="reference" id="cite_ref-TR24_9-0"><a href="#cite_note-TR24-9">[9]</a></sup> Towards the end of 2014, though, TechRadar's rating for version 2.4 has been revised downward to 3/5.<sup class="reference" id="cite_ref-TR23_10-0"><a href="#cite_note-TR23-10">[10]</a></sup></p>
<h2><span class="mw-headline" id="Versions">Versions</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Spybot_%E2%80%93_Search_%26_Destroy&amp;action=edit&amp;section=4" title="Edit section: Versions">edit</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>Version 1.5 had better compatibility than previous versions with <a href="/wiki/Wine_(software)" title="Wine (software)">Wine</a> (software which allows running of Windows programs under <a href="/wiki/Linux" title="Linux">Linux</a>), and restores compatibility with Windows 95<sup class="reference" id="cite_ref-11"><a href="#cite_note-11">[11]</a></sup> which was faulty in 1.4.<sup class="reference" id="cite_ref-BugFixes_12-0"><a href="#cite_note-BugFixes-12">[12]</a></sup></p>
<p>Version 1.6 was said by Safer Networking to scan several times faster than version 1.5.<sup class="reference" id="cite_ref-OfficialSite_3-1"><a href="#cite_note-OfficialSite-3">[3]</a></sup></p>
<p>Version 2.4 was released in July 2014.<sup class="reference" id="cite_ref-13"><a href="#cite_note-13">[13]</a></sup></p>
<h2><span class="mw-headline" id="Malicious_lookalikes">Malicious lookalikes</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Spybot_%E2%80%93_Search_%26_Destroy&amp;action=edit&amp;section=5" title="Edit section: Malicious lookalikes">edit</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>There are several malicious programs designed to look like Spybot-S&amp;D (and other anti-malware software), with similar <a href="/wiki/User_interface" title="User interface">user interfaces</a> and program file names. Some actually install spyware. These programs are known as <a class="mw-redirect" href="/wiki/Rogue_Antispyware" title="Rogue Antispyware">rogue antispyware</a>.</p>
<p>Searching the words "spybot", or "search &amp; destroy", "spybot antispyware" or any other related search on a <a href="/wiki/Web_search_engine" title="Web search engine">search engine</a> will often result in a paid advertisement for "<a href="/wiki/SpySheriff" title="SpySheriff">SpywareBot</a>". <sup class="reference" id="cite_ref-AAA_14-0"><a href="#cite_note-AAA-14">[14]</a></sup> This program is a known <a class="mw-redirect" href="/wiki/Rogue_Antispyware" title="Rogue Antispyware">rogue antispyware</a> program, which fraudulently uses the "search and destroy" logo and a name similar to Spybot to persuade users into downloading it under the impression that it is Spybot Search &amp; Destroy.<sup class="reference" id="cite_ref-15"><a href="#cite_note-15">[15]</a></sup> This program, unlike Spybot S&amp;D, requires payment.</p>
<h2><span class="mw-headline" id="Incompatibility_and_conflicts">Incompatibility and conflicts</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Spybot_%E2%80%93_Search_%26_Destroy&amp;action=edit&amp;section=6" title="Edit section: Incompatibility and conflicts">edit</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>It was reported in 2009 that several commercial security products required users to uninstall Spybot when they are being installed or run, due to stated incompatibility with memory-resident "immunizer" Teatimer.<sup class="reference" id="cite_ref-16"><a href="#cite_note-16">[16]</a></sup></p>
<h3><span class="mw-headline" id="Commercial_security_products">Commercial security products</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Spybot_%E2%80%93_Search_%26_Destroy&amp;action=edit&amp;section=7" title="Edit section: Commercial security products">edit</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>The makers of Spybot-S&amp;D came into conflict with <a href="/wiki/Norton_Internet_Security" title="Norton Internet Security">Norton Internet Security</a> over compatibility issues in 2006.<sup class="reference" id="cite_ref-17"><a href="#cite_note-17">[17]</a></sup> Symantec recommended <a href="/wiki/Uninstaller" title="Uninstaller">uninstalling</a> Spybot-S&amp;D before installing Norton Internet Security. According to Safer Networking, no satisfactory explanation was provided to them for this decision. Antivirus professional Mary Landesman suggests a possible explanation may stem from a graphical glitch in TeaTimer module's confirmation dialog. An official explanation from Safer Networking<sup class="reference" id="cite_ref-18"><a href="#cite_note-18">[18]</a></sup> stated that the error was caused by a <a href="/wiki/Software_bug" title="Software bug">bug</a> in the program used to build their code. The result of the bug was that users had difficulty enabling Norton Internet Security to make necessary changes to critical registry areas, such as allowing itself to launch on computer startup. Aside from this, Mary Landesman, like Safer Networking, concluded that the two programs had no issue with one another. The bug was fixed in the 1.5 release.<sup class="reference" id="cite_ref-BugFixes_12-1"><a href="#cite_note-BugFixes-12">[12]</a></sup></p>
<p><a href="/wiki/Kaspersky_Anti-Virus" title="Kaspersky Anti-Virus">Kaspersky Anti-Virus</a> and Kaspersky Internet Security since version 2009 force users to uninstall Spybot during the installation process, although there is no serious incompatibility yet known. The discussion was concluded in the Kaspersky forum, which said not to install Spybot at all. Kaspersky seems to be reluctant to fix the issue, despite receiving several complaints.<sup class="reference" id="cite_ref-19"><a href="#cite_note-19">[19]</a></sup> Incompatibility between the products might occur when Spybot S&amp;D tries to modify the hosts-file (if selected by the user during immunization), which Kaspersky will interpret as an attempt to harmfully manipulate the file. This can be circumvented by not immunizing the hosts-file in Spybot S&amp;D.</p>
<p><a href="/wiki/Trend_Micro" title="Trend Micro">Trend Micro</a> Officescan followed Norton, Kaspersky and McAfee in simply removing Spybot without warning or notification afterwards.<sup class="reference" id="cite_ref-20"><a href="#cite_note-20">[20]</a></sup></p>
<h3><span class="mw-headline" id="Internet_Explorer_8">Internet Explorer 8</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Spybot_%E2%80%93_Search_%26_Destroy&amp;action=edit&amp;section=8" title="Edit section: Internet Explorer 8">edit</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>The immunisation feature of Spybot – Search &amp; Destroy caused Internet Explorer 8 to start slower than expected.<sup class="reference" id="cite_ref-21"><a href="#cite_note-21">[21]</a></sup> Fix <a class="external text" href="http://support.microsoft.com/kb/969897" rel="nofollow">KB969897</a>, which resolved this problem while addressing certain other <a href="/wiki/Vulnerability_(computing)" title="Vulnerability (computing)">security vulnerabilities</a>, was issued by Microsoft in 2009.<sup class="reference" id="cite_ref-BugFixes_12-2"><a href="#cite_note-BugFixes-12">[12]</a></sup></p>
