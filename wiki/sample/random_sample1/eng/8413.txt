<p><b>Shimizu Station</b> <span style="font-weight: normal">(<span class="t_nihongo_kanji" lang="ja" xml:lang="ja">清水駅</span><span class="t_nihongo_comma" style="display:none">,</span> <i>Shimizu-eki</i><sup class="t_nihongo_help noprint"><a href="/wiki/Help:Installing_Japanese_character_sets" title="Help:Installing Japanese character sets"><span class="t_nihongo_icon" style="color: #00e; font: bold 80% sans-serif; text-decoration: none; padding: 0 .1em;">?</span></a></sup>)</span> is a <a href="/wiki/Train_station" title="Train station">railway station</a> on the <a href="/wiki/T%C5%8Dkaid%C5%8D_Main_Line" title="Tōkaidō Main Line">Tōkaidō Main Line</a> of <a href="/wiki/Central_Japan_Railway_Company" title="Central Japan Railway Company">Central Japan Railway Company</a> in <a href="/wiki/Shimizu-ku,_Shizuoka" title="Shimizu-ku, Shizuoka">Shimizu-ku</a>, <a href="/wiki/Shizuoka,_Shizuoka" title="Shizuoka, Shizuoka">Shizuoka City</a>, <a href="/wiki/Shizuoka_Prefecture" title="Shizuoka Prefecture">Shizuoka Prefecture</a>, <a href="/wiki/Japan" title="Japan">Japan</a>. The station is 169 rail kilometers from <a href="/wiki/Tokyo_Station" title="Tokyo Station">Tokyo Station</a>.</p>

<h2><span class="mw-headline" id="Station_history">Station history</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Shimizu_Station_(Shizuoka)&amp;action=edit&amp;section=1" title="Edit section: Station history">edit</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>Shimizu Station first opened as Ejiri Station <span style="font-weight: normal">(<span class="t_nihongo_kanji" lang="ja" xml:lang="ja">江尻駅</span><span class="t_nihongo_comma" style="display:none">,</span> <i>Ejiri-eki</i><sup class="t_nihongo_help noprint"><a href="/wiki/Help:Installing_Japanese_character_sets" title="Help:Installing Japanese character sets"><span class="t_nihongo_icon" style="color: #00e; font: bold 80% sans-serif; text-decoration: none; padding: 0 .1em;">?</span></a></sup>)</span> on February 1, 1889, when the section of the <a href="/wiki/T%C5%8Dkaid%C5%8D_Main_Line" title="Tōkaidō Main Line">Tōkaidō Main Line</a> connecting Shizuoka with <a href="/wiki/K%C5%8Dzu_Station_(Kanagawa)" title="Kōzu Station (Kanagawa)">Kōzu</a> was completed. It was named after <a href="/wiki/Ejiri-juku" title="Ejiri-juku">Ejiri-juku</a>, the <a href="/wiki/53_Stations_of_the_T%C5%8Dkaid%C5%8D" title="53 Stations of the Tōkaidō">18th station</a> of the historical <a href="/wiki/T%C5%8Dkaid%C5%8D_(road)" title="Tōkaidō (road)">Tōkaidō</a>. In 1934 it was renamed Shimizu Station.</p>
<p>From 1916 the <a href="/wiki/Shimizuk%C5%8D_Line" title="Shimizukō Line">Shimizukō Line</a> ran as a branch line from Shimizu Station through the industrial port area of the town before terminating in <a class="new" href="/w/index.php?title=Miho,_Shimizu&amp;action=edit&amp;redlink=1" title="Miho, Shimizu (page does not exist)">Miho</a>. In 1984 this line was torn up and replaced by a bus service. Regularly scheduled freight services were discontinued in 1984, and all freight services by 2001. Transfer to the <a class="mw-redirect" href="/wiki/Shizuoka_Railway_Shizuoka-Shimizu_Line" title="Shizuoka Railway Shizuoka-Shimizu Line">Shizuoka-Shimizu Railway</a> is available via <a href="/wiki/Shin-Shimizu_Station" title="Shin-Shimizu Station">Shin-Shimizu Station</a>, a ten-minute walk from the east exit. The current station building was completed in June 2003.</p>

<h2><span class="mw-headline" id="Layout">Layout</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Shimizu_Station_(Shizuoka)&amp;action=edit&amp;section=3" title="Edit section: Layout">edit</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>Shimizu Station has a single <a href="/wiki/Island_platform" title="Island platform">island platform</a> serving Track 1 and Track 2, connected to the station building by an <a href="/wiki/Overpass" title="Overpass">overpass</a>. The station building has automated <a href="/wiki/Ticket_machine" title="Ticket machine">ticket machines</a>, <a href="/wiki/TOICA" title="TOICA">TOICA</a> automated <a href="/wiki/Turnstile" title="Turnstile">turnstiles</a> and a manned "Green Window" service counter.</p>
