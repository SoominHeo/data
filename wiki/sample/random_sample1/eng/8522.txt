<h1 class="firstHeading" id="firstHeading" lang="en">Rapidity</h1>
<div class="mw-body-content" id="bodyContent">
<div id="siteSub">From Wikipedia, the free encyclopedia</div>
<div id="contentSub"></div>
<div class="mw-jump" id="jump-to-nav">
					Jump to:					<a href="#mw-head">navigation</a>, 					<a href="#p-search">search</a>
</div>
<div class="mw-content-ltr" dir="ltr" id="mw-content-text" lang="en"><script>function mfTempOpenSection(id){var block=document.getElementById("mf-section-"+id);block.className+=" open-block";block.previousSibling.className+=" open-block";}</script><p>Mathematically, rapidity can be defined as the <a href="/wiki/Hyperbolic_angle" title="Hyperbolic angle">hyperbolic angle</a> that differentiates two frames of reference in relative motion, each frame being associated with <a href="/wiki/Distance" title="Distance">distance</a> and <a href="/wiki/Time" title="Time">time</a> coordinates.</p>
<p>In <a href="/wiki/Theory_of_relativity" title="Theory of relativity">relativity</a>, <b>rapidity</b> denoted by w is commonly used as a measure for relativistic velocity. For one-dimensional motion, rapidities are additive whereas velocities must be combined by Einstein's <a href="/wiki/Velocity-addition_formula#Special_theory_of_relativity" title="Velocity-addition formula">Velocity-addition formula</a>. For low speeds, rapidity and velocity are proportional, but for higher velocities, rapidity takes a larger value, the rapidity of light being infinite.</p>
<p>Using the <a href="/wiki/Inverse_hyperbolic_function" title="Inverse hyperbolic function">inverse hyperbolic function</a> <span class="texhtml">artanh</span>, the rapidity <span class="texhtml"><var>w</var></span> corresponding to velocity <span class="texhtml"><var>v</var></span> is <span class="texhtml"><var>w</var> = artanh(<var>v</var> / <var>c</var>)</span> where c is the velocity of light. For low speeds, <span class="texhtml"><var>w</var></span> is approximately <span class="texhtml"><var>v</var> / <var>c</var></span>. Since in relativity any velocity <span class="texhtml"><var>v</var></span> is constrained to the interval <span class="texhtml">−<var>c</var> &lt; <var>v</var> &lt; <var>c</var></span> the ratio <span class="texhtml"><var>v</var> / <var>c</var></span> satisfies <span class="texhtml">−1 &lt; <var>v</var> / <var>c</var> &lt; 1</span>. The inverse hyperbolic tangent has the unit interval <span class="texhtml">(−1, 1)</span> for its <a class="mw-redirect" href="/wiki/Domain_(mathematics)" title="Domain (mathematics)">domain</a> and the whole <a href="/wiki/Real_line" title="Real line">real line</a> for its <a class="mw-redirect" href="/wiki/Range_(function)" title="Range (function)">range</a>, and so the interval <span class="texhtml">−<var>c</var> &lt; <var>v</var> &lt; <var>c</var></span> maps onto <span class="texhtml">−∞ &lt; <var>w</var> &lt; ∞</span>.</p>

<h2><span class="mw-headline" id="History">History</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Rapidity&amp;action=edit&amp;section=1" title="Edit section: History">edit</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>In 1908 <a href="/wiki/Hermann_Minkowski" title="Hermann Minkowski">Hermann Minkowski</a> explained how the <a href="/wiki/Lorentz_transformation" title="Lorentz transformation">Lorentz transformation</a> could be seen as simply a <a class="mw-redirect" href="/wiki/Hyperbolic_rotation" title="Hyperbolic rotation">hyperbolic rotation</a> of the <a href="/wiki/Coordinate_time" title="Coordinate time">spacetime coordinates</a>, i.e., a rotation through an imaginary angle.<sup class="reference" id="cite_ref-1"><a href="#cite_note-1">[1]</a></sup> This angle therefore represents (in one spatial dimension) a simple additive measure of the velocity between frames.<sup class="reference" id="cite_ref-2"><a href="#cite_note-2">[2]</a></sup> It was used by <a href="/wiki/Vladimir_Vari%C4%87ak" title="Vladimir Varićak">Varićak</a> 1910 by Whittaker 1910 <sup class="reference" id="cite_ref-3"><a href="#cite_note-3">[3]</a></sup> It was named "rapidity" by <a href="/wiki/Alfred_Robb" title="Alfred Robb">Alfred Robb</a> 1911 <sup class="reference" id="cite_ref-4"><a href="#cite_note-4">[4]</a></sup> and this term was adopted by many subsequent authors, such as <a href="/wiki/Vladimir_Vari%C4%87ak" title="Vladimir Varićak">Varićak</a> 1912, <a href="/wiki/Ludwik_Silberstein" title="Ludwik Silberstein">Silberstein</a> (1914), <a class="mw-redirect" href="/wiki/Edward_Morley" title="Edward Morley">Morley</a> (1936) and <a href="/wiki/Wolfgang_Rindler" title="Wolfgang Rindler">Rindler</a> (2001). The development of the theory of rapidity is mainly due to Varićak in writings from 1910 to 1924.<sup class="reference" id="cite_ref-5"><a href="#cite_note-5">[5]</a></sup></p>

<p>The geometry on rapidity space is inherited from the <a href="/wiki/Hyperbolic_geometry" title="Hyperbolic geometry">hyperbolic geometry</a> on velocity space via the map stated. This geometry, in turn, can be inferred from the addition law of relativistic velocities.<sup class="reference" id="cite_ref-10"><a href="#cite_note-10">[9]</a></sup> Rapidity in two dimensions can thus be usefully visualized using the <a href="/wiki/Poincar%C3%A9_disk_model" title="Poincaré disk model">Poincaré disk</a>.<sup class="reference" id="cite_ref-11"><a href="#cite_note-11">[10]</a></sup> Geodesics correspond to steady accelerations. Rapidity space in three dimensions can in the same way be put in <a href="/wiki/Isometry_(Riemannian_geometry)" title="Isometry (Riemannian geometry)">isometry</a> with the <a href="/wiki/Hyperboloid_model" title="Hyperboloid model">hyperboloid model</a> (isometric to the <span class="texhtml">3</span>-dimensional Poincaré disk (or <i>ball</i>)). This is detailed in <a href="/wiki/Minkowski_space#geometry_of_Minkowski_space" title="Minkowski space">geometry of Minkowski space</a>.</p>
