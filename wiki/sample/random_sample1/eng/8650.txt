<p><b>Tsuruga Station</b> <span style="font-weight: normal">(<span class="t_nihongo_kanji" lang="ja" xml:lang="ja">敦賀駅</span><span class="t_nihongo_comma" style="display:none">,</span> <i>Tsuruga-eki</i><sup class="t_nihongo_help noprint"><a href="/wiki/Help:Installing_Japanese_character_sets" title="Help:Installing Japanese character sets"><span class="t_nihongo_icon" style="color: #00e; font: bold 80% sans-serif; text-decoration: none; padding: 0 .1em;">?</span></a></sup>)</span> is a railway station in <a href="/wiki/Tsuruga,_Fukui" title="Tsuruga, Fukui">Tsuruga, Fukui</a>, Japan, operated by <a href="/wiki/West_Japan_Railway_Company" title="West Japan Railway Company">West Japan Railway Company</a> (JR West). It is served by the <a href="/wiki/Hokuriku_Main_Line" title="Hokuriku Main Line">Hokuriku Main Line</a> and <a href="/wiki/Obama_Line" title="Obama Line">Obama Line</a>.</p>

<h2><span class="mw-headline" id="Lines">Lines</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Tsuruga_Station&amp;action=edit&amp;section=1" title="Edit section: Lines">edit</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>Tsuruga Station is served by the following <a class="mw-redirect" href="/wiki/JR_West" title="JR West">JR West</a> lines.</p>
<ul>
<li><a href="/wiki/Hokuriku_Main_Line" title="Hokuriku Main Line">Hokuriku Main Line</a></li>
<li><a href="/wiki/Kosei_Line" title="Kosei Line">Kosei Line</a> (The section is from <a href="/wiki/Yamashina_Station" title="Yamashina Station">Yamashina</a> to <a href="/wiki/%C5%8Cmi-Shiotsu_Station" title="Ōmi-Shiotsu Station">Omi-Shiotsu</a> formally, but most trains run straight Tsuruga station)</li>
<li><a href="/wiki/Obama_Line" title="Obama Line">Obama Line</a></li>
</ul>
<p>A freight-only branch Line known as the Tsuruga Port Line operated by <a class="mw-redirect" href="/wiki/JR_Freight" title="JR Freight">JR Freight</a> also runs from this station.</p>
<p>Tsuruga is also scheduled to become a station on the high-speed <a href="/wiki/Hokuriku_Shinkansen" title="Hokuriku Shinkansen">Hokuriku Shinkansen</a> line when the extension west of <a href="/wiki/Kanazawa_Station" title="Kanazawa Station">Kanazawa</a> opens around 2025.<sup class="reference" id="cite_ref-Fukui_route_1-0"><a href="#cite_note-Fukui_route-1">[1]</a></sup></p>
<h2><span class="mw-headline" id="Station_layout">Station layout</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Tsuruga_Station&amp;action=edit&amp;section=2" title="Edit section: Station layout">edit</a><span class="mw-editsection-bracket">]</span></span></h2>

<p>It has a "<a class="mw-redirect" href="/wiki/Midori_no_Madoguchi" title="Midori no Madoguchi">Midori no Madoguchi</a>" staffed ticket office. The station platforms were rebuilt in December 2012, while a new building just outside the ticket gates attached to the station is expected to be completed before the end of 2013.</p>
<h3><span class="mw-headline" id="Platforms">Platforms</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Tsuruga_Station&amp;action=edit&amp;section=3" title="Edit section: Platforms">edit</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>The station has three <a href="/wiki/Island_platform" title="Island platform">island platforms</a> serving seven tracks. The overhead line power supply changes between 1,500 V DC and 20 kV AC (60 Hz) at this station (Hokuriku Main Line only).</p>

<h2><span class="mw-headline" id="History">History</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Tsuruga_Station&amp;action=edit&amp;section=5" title="Edit section: History">edit</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>Tsuruga Station opened on 10 March 1882.<sup class="reference" id="cite_ref-jnr1985_2-0"><a href="#cite_note-jnr1985-2">[2]</a></sup> With the <a href="/wiki/Privatization" title="Privatization">privatization</a> of <a class="mw-redirect" href="/wiki/JNR" title="JNR">JNR</a> on 1 April 1987, the station came under the control of JR West.<sup class="reference" id="cite_ref-teishajo1998_3-0"><a href="#cite_note-teishajo1998-3">[3]</a></sup></p>
<h2><span class="mw-headline" id="Future_plans">Future plans</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Tsuruga_Station&amp;action=edit&amp;section=6" title="Edit section: Future plans">edit</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>From October 2014, a gauge-changing test track was scheduled to be built next to the station on the site of the former motive power depot. The 180 m long test track will be used to test an experimental variable-gauge <a href="/wiki/Bogie" title="Bogie">bogie</a>, which can be changed from <span class="nowrap">1,435 mm</span> (<span class="nowrap">4 ft <span class="frac">8<span class="visualhide"> </span><sup>1</sup>⁄<sub>2</sub></span> in</span>) to <span class="nowrap">1,067 mm</span> (<span class="nowrap">3 ft 6 in</span>) and vice versa. The aim is prove durability in cold and snowy conditions with a view to using the variable-gauge <a href="/wiki/Gauge_Change_Train" title="Gauge Change Train">Gauge Change Train</a> on through services between the standard-gauge <a href="/wiki/Hokuriku_Shinkansen" title="Hokuriku Shinkansen">Hokuriku Shinkansen</a> and narrow-gauge <a href="/wiki/Hokuriku_Main_Line" title="Hokuriku Main Line">Hokuriku Main Line</a> in the future.<sup class="reference" id="cite_ref-tetsudo20140918_4-0"><a href="#cite_note-tetsudo20140918-4">[4]</a></sup></p>
