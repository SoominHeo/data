<p><b>Clustal</b> is a series of widely used <a href="/wiki/Computer_program" title="Computer program">computer programs</a> used in <a href="/wiki/Bioinformatics" title="Bioinformatics">Bioinformatics</a> for <a href="/wiki/Multiple_sequence_alignment" title="Multiple sequence alignment">multiple sequence alignment</a>.<sup class="reference" id="cite_ref-Chenna2003_2-0"><a href="#cite_note-Chenna2003-2">[2]</a></sup></p>
<p></p>

<h2><span class="mw-headline" id="History">History</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Clustal&amp;action=edit&amp;section=1" title="Edit section: History">edit</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>There have been many incarnations of Clustal that are listed below:</p>
<ul>
<li><b>Clustal</b>: The original software for progressive alignment based on a phylogenetic tree.<sup class="reference" id="cite_ref-pmid3243435_3-0"><a href="#cite_note-pmid3243435-3">[3]</a></sup></li>
<li><b>ClustalV</b>: A rewrite of the original Clustal package that included phylogenetic tree reconstruction on the final alignment for the first time.<sup class="reference" id="cite_ref-pmid1591615_4-0"><a href="#cite_note-pmid1591615-4">[4]</a></sup></li>
<li><b>ClustalW</b>: command line interface<sup class="reference" id="cite_ref-Larkin2007_5-0"><a href="#cite_note-Larkin2007-5">[5]</a></sup></li>
<li><b>ClustalX</b>: This version has a graphical user interface.<sup class="reference" id="cite_ref-Thompson1997_6-0"><a href="#cite_note-Thompson1997-6">[6]</a></sup></li>
<li><b>ClustalΩ (Omega)</b>: The current standard version.<sup class="reference" id="cite_ref-7"><a href="#cite_note-7">[7]</a></sup><sup class="reference" id="cite_ref-8"><a href="#cite_note-8">[8]</a></sup></li>
</ul>
<p>The papers describing the clustal software have been very highly cited, with two of them amongst the most cited papers of all time.<sup class="reference" id="cite_ref-9"><a href="#cite_note-9">[9]</a></sup></p>
<p>The more recent version of the software available for Windows, Mac OS, and Unix/Linux. It is also commonly used via a web interface at its own <a href="#External_links">home page</a> or hosted by the <a href="/wiki/European_Bioinformatics_Institute" title="European Bioinformatics Institute">European Bioinformatics Institute</a>.</p>
<h3><span class="mw-headline" id="Names">Names</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Clustal&amp;action=edit&amp;section=2" title="Edit section: Names">edit</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>The guide tree in the initial programs was constructed via a <a href="/wiki/UPGMA" title="UPGMA">UPGMA</a> cluster analysis of the pairwise alignments, hence the name CLUSTAL.<sup class="reference" id="cite_ref-smbe_10-0"><a href="#cite_note-smbe-10">[10]</a></sup><sup>cf.</sup><sup class="reference" id="cite_ref-11"><a href="#cite_note-11">[11]</a></sup> The first four versions in 1988 had Arabic numerals (1 to 4), whereas with the fifth version <a href="/wiki/Desmond_G._Higgins" title="Desmond G. Higgins">Des Higgins</a> switched to Roman numeral V in 1992.<sup class="reference" id="cite_ref-smbe_10-1"><a href="#cite_note-smbe-10">[10]</a></sup><sup>cf.</sup><sup class="reference" id="cite_ref-12"><a href="#cite_note-12">[12]</a></sup><sup class="reference" id="cite_ref-13"><a href="#cite_note-13">[13]</a></sup> In 1994 and in 1997, for the next two versions, the letters after the letter V were used and made to correspond to W for Weighted and X for <a href="/wiki/X_Window_System" title="X Window System">X Window</a>.<sup class="reference" id="cite_ref-smbe_10-2"><a href="#cite_note-smbe-10">[10]</a></sup><sup>cf.</sup><sup class="reference" id="cite_ref-14"><a href="#cite_note-14">[14]</a></sup><sup class="reference" id="cite_ref-15"><a href="#cite_note-15">[15]</a></sup> The name omega was chosen to mark a change from the previous ones.<sup class="reference" id="cite_ref-smbe_10-3"><a href="#cite_note-smbe-10">[10]</a></sup></p>
<h2><span class="mw-headline" id="Function">Function</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Clustal&amp;action=edit&amp;section=3" title="Edit section: Function">edit</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>All variants of Clustal align sequences by three main steps:</p>
<ol>
<li>Do a <a href="/wiki/Sequence_alignment" title="Sequence alignment">pairwise alignment</a></li>
<li>Create a <a href="/wiki/Molecular_phylogenetics" title="Molecular phylogenetics">guide tree</a> (or use a user-defined tree)</li>
<li>Use the guide tree to carry out a multiple alignment</li>
</ol>
<p>These are done automatically when you select "Do Complete Alignment". Other options are "Do Alignment from guide tree and phylogeny" and "Produce guide tree only".</p>
<h3><span class="mw-headline" id="Input.2FOutput">Input/Output</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Clustal&amp;action=edit&amp;section=4" title="Edit section: Input/Output">edit</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>This program accepts a wide range of input formats, including NBRF/PIR, <a href="/wiki/FASTA_format" title="FASTA format">FASTA</a>, EMBL/<a href="/wiki/UniProt" title="UniProt">Swiss-Prot</a>, Clustal, GCC/MSF, GCG9 RSF, and GDE.</p>
<p>The output format can be one or many of the following: Clustal, NBRF/PIR, <a class="mw-redirect" href="/wiki/GCG_(software)" title="GCG (software)">GCG</a>/MSF, <a href="/wiki/PHYLIP" title="PHYLIP">PHYLIP</a>, GDE, or NEXUS.</p>
<h3><span class="mw-headline" id="Settings">Settings</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Clustal&amp;action=edit&amp;section=5" title="Edit section: Settings">edit</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>Many settings can be modified to adapt the alignment algorithm to different circumstances. The main parameters are the gap opening penalty, and the gap extension penalty.</p>
