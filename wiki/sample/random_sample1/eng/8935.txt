<p><b>iMovie</b> is a <a href="/wiki/Video_editing_software" title="Video editing software">video editing software</a> application sold by <a href="/wiki/Apple_Inc." title="Apple Inc.">Apple Inc.</a> for the <a href="/wiki/Macintosh" title="Macintosh">Mac</a> and <a href="/wiki/IOS" title="IOS">iOS</a> (<a href="/wiki/IPhone" title="IPhone">iPhone</a>, <a href="/wiki/IPad" title="IPad">iPad</a>, <a href="/wiki/IPad_Mini" title="IPad Mini">iPad Mini</a> and <a href="/wiki/IPod_Touch" title="IPod Touch">iPod Touch</a>). It was originally released in 1999 as a <a href="/wiki/Mac_OS_8" title="Mac OS 8">Mac OS 8</a> application bundled with the first FireWire-enabled consumer Mac model – the <a href="/wiki/IMac_G3" title="IMac G3">iMac DV</a>.<sup class="reference" id="cite_ref-1"><a href="#cite_note-1">[1]</a></sup> Since version 3, iMovie has been an <a href="/wiki/MacOS" title="MacOS">OS X</a>-only application included with the <a href="/wiki/ILife" title="ILife">iLife</a> suite of Mac applications.<sup class="reference" id="cite_ref-2"><a href="#cite_note-2">[2]</a></sup> From 2003, iMovie is included free with all new Mac computers.</p>
<p>iMovie imports video footage to the Mac using either the <a href="/wiki/IEEE_1394" title="IEEE 1394">FireWire</a> interface on most MiniDV format digital video cameras or the computer's <a href="/wiki/USB" title="USB">USB</a> port. It can also import video and photo files from a hard drive. From there, the user can edit the photos and video clips and add titles, themes, music, and effects, including basic color correction and video enhancement tools and transitions such as fades and slides.</p>

<h2><span class="mw-headline" id="High-definition_video_support">High-definition video support</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=IMovie&amp;action=edit&amp;section=1" title="Edit section: High-definition video support">edit</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>Starting with version 5 (from 2005), iMovie processes high-definition video from <a href="/wiki/HDV" title="HDV">HDV</a> <a href="/wiki/Camcorder" title="Camcorder">camcorders</a>, in later versions also from <a href="/wiki/AVCHD" title="AVCHD">AVCHD</a> camcorders and <a href="/wiki/H.264/MPEG-4_AVC" title="H.264/MPEG-4 AVC">H.264</a>-compressed video from <a href="/wiki/MPEG-4_Part_14" title="MPEG-4 Part 14">MPEG-4</a> or <a href="/wiki/QuickTime_File_Format" title="QuickTime File Format">QuickTime Movie files (.mov).</a>, e.g. as generated by a number of <a href="/wiki/Digital_camera" title="Digital camera">digital photo cameras</a> with HD video recording feature. To facilitate this, iMovie/iLife installs the <a href="/wiki/Apple_Intermediate_Codec" title="Apple Intermediate Codec">Apple Intermediate Codec</a> on the system as a <a href="/wiki/QuickTime" title="QuickTime">QuickTime</a> component. iMovie <a href="/wiki/Transcoding" title="Transcoding">transcodes</a> (‘optimizes’) HD video upon ingestion (‘import’) using this <a href="/wiki/Video_codec" title="Video codec">codec</a> and stores it in the QuickTime file format (.movie.)</p>

<h2><span class="mw-headline" id="iMovie_HD_5">iMovie HD 5</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=IMovie&amp;action=edit&amp;section=5" title="Edit section: iMovie HD 5">edit</a><span class="mw-editsection-bracket">]</span></span></h2>
<div class="thumb tright">
<div class="thumbinner" style="width:192px;"><a class="image" href="/wiki/File:Imovie_05.jpg"><img alt="" class="thumbimage" data-file-height="340" data-file-width="469" height="138" src="//upload.wikimedia.org/wikipedia/en/thumb/6/65/Imovie_05.jpg/190px-Imovie_05.jpg" srcset="//upload.wikimedia.org/wikipedia/en/thumb/6/65/Imovie_05.jpg/285px-Imovie_05.jpg 1.5x, //upload.wikimedia.org/wikipedia/en/thumb/6/65/Imovie_05.jpg/380px-Imovie_05.jpg 2x" width="190"/></a>
<div class="thumbcaption">
<div class="magnify"><a class="internal" href="/wiki/File:Imovie_05.jpg" title="Enlarge"></a></div>
A blank movie project in iMovie HD, included with <a href="/wiki/ILife" title="ILife">iLife</a> '05.</div>
</div>
</div>
<p>iMovie HD included support for <a href="/wiki/HDV" title="HDV">HDV</a> (720p and 1080i) and integration with the rest of the <a href="/wiki/ILife" title="ILife">iLife</a> suite, with toolbox buttons allowing the importing of images from <a href="/wiki/IPhoto" title="IPhoto">iPhoto</a>, music from <a href="/wiki/ITunes" title="ITunes">iTunes</a> and the setting of chapter markers ready for exporting to <a href="/wiki/IDVD" title="IDVD">iDVD</a>.</p>
<p>iMovie HD 5 imported mjpeg files as dv by default, which introduces noise; mjpeg files are cryptically lumped with "isight" files in this version.</p>
<p>Another new feature was included called "Magic iMovie", which attempted to automate the whole process of video editing, by allowing a common transition to be added between scenes, a music track to be synchronised with the video and a <a href="/wiki/DVD" title="DVD">DVD</a> to be created with the accompanying iDVD software.</p>
<h2><span class="mw-headline" id="iMovie_HD_6">iMovie HD 6</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=IMovie&amp;action=edit&amp;section=6" title="Edit section: iMovie HD 6">edit</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>iMovie 6 was released in January 2006 as part of the <a href="/wiki/ILife" title="ILife">iLife</a> '06 suite, and was also originally optionally included with iLife '08 as a substitution for iMovie '08 (due to the new version's incompatibility with older Power PC Macintosh computers). However, this option was removed after iLife '09 was released. It was integrated with <a href="/wiki/IPhoto" title="IPhoto">iPhoto</a>, <a href="/wiki/ITunes" title="ITunes">iTunes</a>, <a href="/wiki/IDVD" title="IDVD">iDVD</a>, <a href="/wiki/GarageBand" title="GarageBand">GarageBand</a> and <a href="/wiki/IWeb" title="IWeb">iWeb</a>. iMovie HD 6 was designed for ease of use, and included new themes. Themes allow the user to drop movie clips or photos into professionally designed backdrops. Each theme included full-motion graphic bumpers and transitions. iMovie HD 6 also added real-time effects, which took advantage of the computer's graphic processing unit to perform some effects without rendering. It also introduced real-time titling, enhanced audio tools and effects, the ability to have multiple projects open at once, video podcasts and blogs (using integration with iWeb), and a refined look based on iTunes 5 and 6.</p>
<h2><span class="mw-headline" id="iMovie_.2708">iMovie '08</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=IMovie&amp;action=edit&amp;section=7" title="Edit section: iMovie '08">edit</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>iMovie '08 (Version 7.0) was released in August 2007 as a part of the iLife '08 suite. iMovie '08 was a complete redesign and rewrite of iMovie. It had much better HD output, and more formats to convert to. This was limited, however, by an undocumented restriction on supported codecs. <a href="/wiki/IPhoto" title="IPhoto">iPhoto</a> uses the QuickTime library and can create thumbnails for all <a href="/wiki/QuickTime" title="QuickTime">QuickTime</a> supported formats, but most of these cannot be used by iMovie '08. Some of the formats that iMovie '08 is able to import will not be recognized when they are added to an iPhoto library. Though <a href="/wiki/Motion_JPEG" title="Motion JPEG">Motion JPEG</a>-encoded AVI files do appear to be recognized, this was the most common format used by digital cameras. The tile-based editing interface was also promoted as something unique and groundbreaking, even though it is functionally identical to the interface of the Toaster Flyer non-linear digital video editing systems released for Amiga computers by NewTek, Inc. in 1993. A new feature called "skimming" for quickly previewing video in the library at a user controlled speed was added, and so was a feature that allows the user to highlight parts of video clips just like highlighting text. iMovie 08 also had the ability to add more than two layers of background sound, including multiple music, narration and sound effects; previous versions could play multiple tracks but could display only two extra audio tracks. It included more exportation formats, including iPhone-sized video. It also supported non-tape-based HD video, such as <a href="/wiki/AVCHD" title="AVCHD">AVCHD</a> and footage from DVD and HDD camcorders. iMovie '08 also has the ability to export movies to the <a href="/wiki/YouTube" title="YouTube">YouTube</a> video sharing website.</p>
<p>According to Apple's <a class="external text" href="https://www.apple.com/ilife/systemrequirements.html" rel="nofollow">system requirements</a>, iMovie '08 requires a Mac with either a 1.9 GHz or faster PowerPC G5 or Intel processor. G4s are not supported, even though Apple sold its last G4-based Computers (iBook G4s) 14 months before the release of iLife '08. However, a system hack enables iMovie 7.1 or higher to run on a PowerPC G4.<sup class="reference" id="cite_ref-5"><a href="#cite_note-5">[5]</a></sup></p>
<h3><span class="mw-headline" id="Criticism_of_iMovie_.2708">Criticism of iMovie '08</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=IMovie&amp;action=edit&amp;section=8" title="Edit section: Criticism of iMovie '08">edit</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>iMovie 08 was criticized due to its drastic abandonment of some iMovie HD 6 features. Former <a class="mw-redirect" href="/wiki/New_York_Times" title="New York Times">New York Times</a> reviewer <a href="/wiki/David_Pogue" title="David Pogue">David Pogue</a> said "iMovie ‘08 is an utter bafflement... incapable of the more sophisticated editing that the old iMovie made so enjoyable...All visual effects are gone — even basic options like slow motion, reverse motion, fast motion, and black-and-white. And you can’t have more than one project open at a time."<sup class="reference" id="cite_ref-6"><a href="#cite_note-6">[6]</a></sup></p>
<p>Features removed included the classic timeline, the ability to create DVD chapter markers, support for plugins, and in-timeline audio adjustment and control. iMovie '08 imports to a much more limited set of video codecs and metadata formats than previous versions of iMovie or today's QuickTime Player. For example, QuickTime Player can be extended to support the FLIP Video 3ivx MPEG-4 codec, but iMovie '08 cannot. iMovie '08 also removed the ability to import DV footage. As a result, all resulting videos have <a href="/wiki/Lossy_compression" title="Lossy compression">lossy compression</a> applied and there is no facility for managing full format video. The peculiar lack of QuickTime support means <a href="/wiki/QuickTime#QuickTime_Pro" title="QuickTime">QuickTime Pro</a> can edit a larger range of video than iMovie '08.</p>
<p>Apple released iMovie HD 6 as a free download to those who had purchased iMovie '08.<sup class="reference" id="cite_ref-7"><a href="#cite_note-7">[7]</a></sup> However, in response to the release of the subsequent newer version of iMovie '09, Apple removed the download in late January 2009<sup class="reference" id="cite_ref-8"><a href="#cite_note-8">[8]</a></sup> while also reducing the $299 price tag for <a href="/wiki/Final_Cut_Express" title="Final Cut Express">Final Cut Express</a> to $199. Several of the features removed from iMovie '08 that were previously included with iMovie HD 6 have been restored into iMovie '09 and, more recently, iMovie '11.</p>
<h2><span class="mw-headline" id="iMovie_.2709">iMovie '09</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=IMovie&amp;action=edit&amp;section=9" title="Edit section: iMovie '09">edit</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>iMovie '09 (Version 8.0) was released January 2009 as part of the iLife '09 package. It introduced some new features and restored some features from previous versions of iMovie, including basic video effects (such as fast/slow motion and aged film) and image stabilization as well as travel map functions for marking locations where a video was shot. iMovie '09 also introduced simple implementations of more advanced features such as <a href="/wiki/Picture-in-picture" title="Picture-in-picture">picture-in-picture</a> and <a class="mw-redirect" href="/wiki/Chroma_keying" title="Chroma keying">Chroma keying</a>. It also improved editing with a precision cut editor and a clip trimmer, improved support for hard drive-based cameras such as the Flip Mino, added some new titles and transitions, and added full iDVD support (which was unavailable in iMovie '08). In addition, it introduced a Full-Screen Library Browser with which the user can find and examine all of his or her video in one place.</p>
<h2><span class="mw-headline" id="iMovie_.2711">iMovie '11</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=IMovie&amp;action=edit&amp;section=10" title="Edit section: iMovie '11">edit</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>iMovie '11 (Version 9.0) was released on October 20, 2010 as part of the iLife '11 package. It has the ability to make trailers for home movies, more control over audio, instant replay and flash and hold effects, facial recognition, news themes, and the ability to watch the video on a Mac, iPad, iPhone/iPod touch, or Apple TV, as well as sharing on Facebook and YouTube. It now supports the <a class="mw-redirect" href="/wiki/AVCHD_Lite" title="AVCHD Lite">AVCHD Lite</a> format.</p>
<p>Apple worked with <a href="/wiki/Abbey_Road_Studios" title="Abbey Road Studios">Abbey Road Studios</a> in London, England to bring original music/film scores to iMovie '11. The music is most notably used in the "trailers" feature provided by the software.</p>
<p>On January 6, 2011, Apple made iMovie '11 (along with <a href="/wiki/Aperture_(software)" title="Aperture (software)">Aperture</a>, the <a href="/wiki/IWork" title="IWork">iWork</a> suite, and the rest of the iLife suite) available on the then-new <a href="/wiki/Mac_App_Store" title="Mac App Store">Mac App Store</a>.<sup class="reference" id="cite_ref-9"><a href="#cite_note-9">[9]</a></sup></p>
<h2><span class="mw-headline" id="iMovie_10.0">iMovie 10.0</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=IMovie&amp;action=edit&amp;section=11" title="Edit section: iMovie 10.0">edit</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>iMovie 10.0 was released on October 22, 2013 by <a href="/wiki/Apple_Inc." title="Apple Inc.">Apple Inc.</a> This version of iMovie was a complete redesign with more options to share a movie, more movie and trailer theme options from iMovie for iOS, easier to make picture-in-pictures, cutaways, side-by-sides etc., more realistic green-screen effects and easier refinements.<sup class="reference" id="cite_ref-10"><a href="#cite_note-10">[10]</a></sup></p>
<h2><span class="mw-headline" id="iMovie_10.1">iMovie 10.1</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=IMovie&amp;action=edit&amp;section=12" title="Edit section: iMovie 10.1">edit</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>iMovie 10.1 was released on October 13, 2015. It allowed for 4K video editing and included a major user interface overhaul, as well as the removal of some peripheral features.</p>
<h2><span class="mw-headline" id="iMovie_for_iOS">iMovie for iOS</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=IMovie&amp;action=edit&amp;section=13" title="Edit section: iMovie for iOS">edit</a><span class="mw-editsection-bracket">]</span></span></h2>

<p>On June 7, 2010, <a href="/wiki/Steve_Jobs" title="Steve Jobs">Steve Jobs</a> announced in his <a class="mw-redirect" href="/wiki/WWDC" title="WWDC">WWDC</a> keynote that the upcoming <a href="/wiki/IPhone_4" title="IPhone 4">iPhone 4</a> would support a new, <a class="mw-redirect" href="/wiki/IOS_(Apple)" title="IOS (Apple)">iOS</a>-native version of iMovie that supports many of the basic features of the Mac version of the software.<sup class="reference" id="cite_ref-11"><a href="#cite_note-11">[11]</a></sup> The app may be obtained by downloading it from the <a href="/wiki/App_Store_(iOS)" title="App Store (iOS)">App Store</a> for $4.99. iMovie for iPhone was officially available on June 24, 2010 to coincide with the launch of the <a href="/wiki/IPhone_4" title="IPhone 4">iPhone 4</a>.</p>
<p>On September 1, 2010, iMovie was made compatible with the new 4th-generation <a href="/wiki/IPod_Touch" title="IPod Touch">iPod Touch</a>. An iPad version of iMovie for iOS was made available with the release of <a href="/wiki/IPad_2" title="IPad 2">iPad 2</a>, announced at an Apple media event on March 2, 2011 and released seven days later.<sup class="reference" id="cite_ref-12"><a href="#cite_note-12">[12]</a></sup> On March 7, 2012, <a href="/wiki/Tim_Cook" title="Tim Cook">Tim Cook</a> announced an updated version of iMovie for iOS along with the <a href="/wiki/IPad_(3rd_generation)" title="IPad (3rd generation)">third-generation iPad</a>.</p>
