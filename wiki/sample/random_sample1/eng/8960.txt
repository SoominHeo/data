<p><b>iPodLinux</b> is a <a href="/wiki/%CE%9CClinux" title="ΜClinux">µClinux</a>-based <a href="/wiki/Linux_distribution" title="Linux distribution">Linux distribution</a> designed specifically to run on <a href="/wiki/Apple_Inc." title="Apple Inc.">Apple Inc.</a>'s <a href="/wiki/IPod" title="IPod">iPod</a>. When the iPodLinux <a href="/wiki/Linux_kernel" title="Linux kernel">kernel</a> is <a href="/wiki/Booting" title="Booting">booted</a> it takes the place of Apple's iPod operating system and automatically loads Podzilla, an alternative <a href="/wiki/Graphical_user_interface" title="Graphical user interface">GUI</a> and launcher for a number of additional included <a href="/wiki/Computer_program" title="Computer program">programs</a> such as a <a class="mw-redirect" href="/wiki/Video_player_(software)" title="Video player (software)">video player</a>, an <a href="/wiki/Image_viewer" title="Image viewer">image viewer</a>, a <a class="mw-redirect" href="/wiki/Command_line_interface" title="Command line interface">command line shell</a>, <a href="/wiki/Video_game" title="Video game">games</a>, <a href="/wiki/Emulator" title="Emulator">emulators</a> for <a href="/wiki/Video_game_console" title="Video game console">video game consoles</a>, <a href="/wiki/Computer_programming" title="Computer programming">programming</a> demos, and other experimental or occasionally unfinished <a class="mw-redirect" href="/wiki/Computer_software" title="Computer software">software</a>.</p>
<p>As of 2009, the project has been inactive and its website is no longer maintained. Further development of <a href="/wiki/Free_software" title="Free software">free</a> and <a class="mw-redirect" href="/wiki/Open_source" title="Open source">open source</a> software for iPods have continued with the <a href="/wiki/Rockbox" title="Rockbox">Rockbox Project</a>, <a class="new" href="/w/index.php?title=Zeroslackr&amp;action=edit&amp;redlink=1" title="Zeroslackr (page does not exist)">zeroslackr</a>, and <a class="new" href="/w/index.php?title=Freemyipod&amp;action=edit&amp;redlink=1" title="Freemyipod (page does not exist)">freemyipod</a>, which have largely supplanted iPodLinux. Some third party installers are still available at <a class="external free" href="http://aqua-gero.com/miche/" rel="nofollow">http://aqua-gero.com/miche/</a>.</p>

<h2><span class="mw-headline" id="Basic_structure">Basic structure</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=IPodLinux&amp;action=edit&amp;section=1" title="Edit section: Basic structure">edit</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>iPodLinux in essence consists of a Linux kernel built from µClinux sources using the <a href="/wiki/UClibc" title="UClibc">uClibc</a> <a href="/wiki/C_standard_library" title="C standard library">C standard library</a> with <a href="/wiki/Device_driver" title="Device driver">driver</a> <a href="/wiki/Source_code" title="Source code">code</a> for iPod components (or <a class="mw-redirect" href="/wiki/Reverse_engineered" title="Reverse engineered">reverse engineered</a> drivers where available). It includes <a class="mw-redirect" href="/wiki/Userland_(computing)" title="Userland (computing)">userland</a> programs from µClinux and/or <a href="/wiki/BusyBox" title="BusyBox">BusyBox</a>, a <a class="mw-redirect" href="/wiki/UNIX" title="UNIX">UNIX</a>-style <a href="/wiki/File_system" title="File system">file system</a> (which can be created within <a href="/wiki/HFS_Plus" title="HFS Plus">HFS+</a> formatted iPods, or an <a href="/wiki/Ext2" title="Ext2">ext2</a> partition on <a href="/wiki/File_Allocation_Table" title="File Allocation Table">FAT32</a> formatted iPod), and the Podzilla GUI (and its modules). Apple's <a href="/wiki/Proprietary_software" title="Proprietary software">proprietary</a> iPod OS in contrast uses an invisible <a href="/wiki/Booting#Boot_loader" title="Booting">boot loader</a> and is based on an <a href="/wiki/ARM_architecture" title="ARM architecture">ARM processor</a> kernel originally written by <a href="/wiki/Pixo" title="Pixo">Pixo</a>, and the iPod <a class="mw-redirect" href="/wiki/Miller_Columns" title="Miller Columns">Miller Columns</a> browser program, a <a class="mw-redirect" href="/wiki/Graphic_user_interface" title="Graphic user interface">GUI</a> written by Apple and Pixo using the Pixo <a href="/wiki/Application_framework" title="Application framework">application framework</a>, and other <a href="/wiki/Firmware" title="Firmware">firmware</a> and component drivers written from manufacturer's reference code to support the standard behavior Apple wanted the iPod to have.</p>

<p>Besides the <a href="/wiki/Linux_kernel" title="Linux kernel">kernel</a>, iPodLinux features as a primary component podzilla and podzilla2, applications which provide:</p>
<ul>
<li>An iPod-like <a href="/wiki/User_interface" title="User interface">user interface</a></li>
<li>Video playback with sound</li>
<li>Support for <a href="/wiki/Advanced_Audio_Coding" title="Advanced Audio Coding">AAC</a>, <a href="/wiki/MP3" title="MP3">MP3</a> and basic <a href="/wiki/Ogg" title="Ogg">OGG</a> playback (4G &amp; 5G <a href="/wiki/Music_Player_Daemon" title="Music Player Daemon">Music Player Daemon</a> malfunctions, but can be fixed).</li>
<li>Many games, including TuxChess, Bluecube (<a href="/wiki/Tetris" title="Tetris">Tetris</a> clone), Chopper, <a href="/wiki/StepMania" title="StepMania">StepMania</a> (a <a href="/wiki/Dance_Dance_Revolution" title="Dance Dance Revolution">Dance Dance Revolution</a> clone) and more.<sup class="reference" id="cite_ref-modules_1-0"><a href="#cite_note-modules-1">[1]</a></sup></li>
<li>Recording through <a class="mw-redirect" href="/wiki/TRS_connector" title="TRS connector">audio jack</a> at much higher quality than Apple's firmware</li>
<li>Ability to play the games <i><a href="/wiki/Doom_(series)" title="Doom (series)">Doom</a></i> and <i><a href="/wiki/Doom_II:_Hell_on_Earth" title="Doom II: Hell on Earth">Doom II</a></i> (and presumably any Doom Total Conversion; <a href="/wiki/Chex_Quest" title="Chex Quest">Chex Quest</a> for instance)</li>
<li><a href="/wiki/Color_scheme" title="Color scheme">Color scheme</a> support</li>
<li>Ability to run many <a href="/wiki/Emulator" title="Emulator">emulators</a>, such as <a class="mw-redirect" href="/wiki/Gnuboy" title="Gnuboy">iBoy</a> (<a href="/wiki/Nintendo" title="Nintendo">Nintendo</a> <a href="/wiki/Game_Boy" title="Game Boy">Game Boy</a> Emulator), iNES (<a href="/wiki/Nintendo_Entertainment_System" title="Nintendo Entertainment System">Nintendo Entertainment System</a> Emulator), iDarcNES (port of the multiple system emulator DarcNES), iMAME (port of <a href="/wiki/MAME" title="MAME">Multiple Arcade Machine Emulator</a>), and iGPSP (<a href="/wiki/Game_Boy_Advance" title="Game Boy Advance">Game Boy Advance</a> emulator).</li>
</ul>
<h2><span class="mw-headline" id="History">History</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=IPodLinux&amp;action=edit&amp;section=3" title="Edit section: History">edit</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>The bootloader for the 4th generation iPod was extracted by Nils Schneider, a German <a href="/wiki/Computer_science" title="Computer science">computer science</a> student. Previous software methods to extract the necessary bootloader no longer worked. Bernard Leach had previously discovered how to operate the piezo buzzer inside the iPod. Schneider was able to use his program with some modifications to make a series of clicks for each byte of the new iPod's bootloader. The extraction process took 22 hours to complete and required Schneider to construct a soundproof box to prevent outside interference with the process. <sup class="reference" id="cite_ref-2"><a href="#cite_note-2">[2]</a></sup></p>
<h3><span class="mw-headline" id="Server_transition">Server transition</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=IPodLinux&amp;action=edit&amp;section=4" title="Edit section: Server transition">edit</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>On June 11, 2008 the organization's website was suspended and replaced with a redirect to a <a class="external text" href="https://web.archive.org/web/20080610065741/http://cpanel2.sys-techs.net/suspended.page/" rel="nofollow">blank page</a>. The server had its services restored incrementally.<sup class="reference" id="cite_ref-3"><a href="#cite_note-3">[3]</a></sup> On October 1, 2008 the <i>iPodLinux.org</i> DNS address was updated and the server was online again by October 5, 2008. On June 22, 2009 the server was pulled offline again. The server was back online again on September 8. In September 2010 the server went offline again and has not got online again yet. Alexander Papst, one of the developers, has posted a <a class="mw-redirect" href="/wiki/Mirror_website" title="Mirror website">mirror</a> of the site at <i><a class="external text" href="https://web.archive.org/web/20140819085608/http://ipodlinux.wiki/" rel="nofollow">ipodlinux.wiki</a></i>. As of 2015, the site is offline.</p>

<p>As of August 5, 2006, only the 1st, 2nd, and 3rd generation iPod are officially supported by iPodLinux,<sup class="noprint Inline-Template Template-Fact" style="white-space:nowrap;">[<i><a href="/wiki/Wikipedia:Citation_needed" title="Wikipedia:Citation needed"><span title="This claim needs references to reliable sources. (December 2007)">citation needed</span></a></i>]</sup> although newer generations are also partially compatible.<sup class="reference" id="cite_ref-ManualInstallation_12-0"><a href="#cite_note-ManualInstallation-12">[5]</a></sup> The iPodLinux project does not plan support for the <a class="mw-redirect" href="/wiki/IPod_shuffle" title="IPod shuffle">iPod shuffle</a> due to the lack of a <a href="/wiki/GNU_Compiler_Collection" title="GNU Compiler Collection">GCC</a> compiler for the shuffle's DSP57000 core, as well as the fact that the iPod shuffle lacks a screen.<sup class="reference" id="cite_ref-Generations_13-0"><a href="#cite_note-Generations-13">[6]</a></sup> While later generations work fine for many uses of iPodLinux, not all features work; these later generations will not be officially supported by the project until most or all features from the earlier iPods work on them. Installers are in the process of being made. As of now, there is Installer 2.3 for <a href="/wiki/Microsoft_Windows" title="Microsoft Windows">Microsoft Windows</a> or <a href="/wiki/Linux" title="Linux">Linux</a> which can install on any generation <a href="/wiki/IPod" title="IPod">iPod</a> (except for the iPod shuffle and <a class="mw-redirect" href="/wiki/IPod_nano" title="IPod nano">iPod nano</a> 2nd generation). As of April, 2008, iPodLinux does not work on the new iPod firmware included with the second and third generation iPod nano or the 6th generation iPod Classic, and installer 2 cannot be used to install iPodLinux on 5.5th generation iPod. In addition to that, the much spoken about audio recording feature currently does not work on the latest ipodlinux/zeroslackr builds. In ipodlinux, an under development message is given under recording, while in zeroslackr, recording is not displayed at all.</p>
<p>Arguably one of the project's more notable accomplishments is its <a class="mw-redirect" href="/wiki/Video_player_(software)" title="Video player (software)">video player</a>, released months before rumors about Apple's Video iPod began to spread. This video player only plays uncompressed <a href="/wiki/Audio_Video_Interleave" title="Audio Video Interleave">AVI</a> files, which are basically just a series of <a href="/wiki/Bitmap" title="Bitmap">bitmap</a> formatted frames with an audio overlay that commonly loses sync with the video output. A new compression technique called <i>MoviePod</i>, released in 2006, enables people to put more video content on their iPod. This function continues to be developed and is a useful function for users of older iPod (especially nano users that, with the help of iPodLinux, can get an extremely small <a href="/wiki/Home_theater_PC" title="Home theater PC">media center</a> that can be held in the palm of the hand).<sup class="reference" id="cite_ref-MVplayer_14-0"><a href="#cite_note-MVplayer-14">[7]</a></sup></p>
<p>podzilla 2, the second generation of podzilla, and commonly known as pz2, is currently in development and has recently superseded the original version of podzilla. It includes several new features, most notably modularity; users can install new applications without recompiling all of podzilla. This version is the only working set of Podzilla official that will run on 5.5G iPods.</p>
<p>Please note that the 2012 set of iPods have not been considered yet.</p>
