<p><b>iPhone OS 3</b>, renamed <b>iOS 3</b> when support for the <a href="/wiki/IPad" title="IPad">iPad</a> was added in iOS 3.2, is the <a href="/wiki/IOS_version_history" title="IOS version history">third</a> major release of the <a href="/wiki/IOS" title="IOS">iOS</a> <a href="/wiki/Mobile_operating_system" title="Mobile operating system">mobile operating system</a> designed by <a href="/wiki/Apple_Inc." title="Apple Inc.">Apple Inc.</a>. iPhone OS 3.0 became available with the <a href="/wiki/IPhone_3GS" title="IPhone 3GS">iPhone 3GS</a>. It was released on June 17, 2009. It added features such as <a href="/wiki/Cut,_copy,_and_paste" title="Cut, copy, and paste">copy and paste</a>, and <a href="/wiki/Multimedia_Messaging_Service" title="Multimedia Messaging Service">MMS</a>. Not all features were available on the original iPhone. Devices running iPhone OS 2.x were upgradeable to this software. The final release supported on the original iPhone and iPod Touch is iPhone OS 3.1.3.<sup class="reference" id="cite_ref-1"><a href="#cite_note-1">[1]</a></sup> The iPad was introduced with iOS 3.2.<sup class="reference" id="cite_ref-2"><a href="#cite_note-2">[2]</a></sup> iOS 3.2 was the first version to be called "iOS." iOS 3.2.2 was the final version of iPhone OS 3.</p>

<h2><span class="mw-headline" id="Features">Features</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=IPhone_OS_3&amp;action=edit&amp;section=1" title="Edit section: Features">edit</a><span class="mw-editsection-bracket">]</span></span></h2>
<h3><span class="mw-headline" id="Cut.2C_copy.2C_and_paste">Cut, copy, and paste</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=IPhone_OS_3&amp;action=edit&amp;section=2" title="Edit section: Cut, copy, and paste">edit</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>iPhone OS 3 introduced Cut, copy, and paste, a system-wide feature where a user can double-tap the text, and it will get a "cut, copy, and paste" bubble dialog. Double-tap again, the "paste" bubble will appear if there's anything stored in your clipboard.<sup class="reference" id="cite_ref-3"><a href="#cite_note-3">[3]</a></sup></p>
<h3><span class="mw-headline" id="Spotlight">Spotlight</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=IPhone_OS_3&amp;action=edit&amp;section=3" title="Edit section: Spotlight">edit</a><span class="mw-editsection-bracket">]</span></span></h3>
<div class="hatnote" role="note">Main article: <a href="/wiki/Spotlight_(software)" title="Spotlight (software)">Spotlight (software)</a></div>
<p>Spotlight is a system-wide indexing and search feature that is included in the iPhone OS 3 software update. This feature helps users to search everything from contacts, mail, calendars, media, app names and other content.<sup class="reference" id="cite_ref-pcw_4-0"><a href="#cite_note-pcw-4">[4]</a></sup> It is done by swiping to the right from the home screen.<sup class="reference" id="cite_ref-:0_5-0"><a href="#cite_note-:0-5">[5]</a></sup></p>
<h3><span class="mw-headline" id="Messages">Messages</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=IPhone_OS_3&amp;action=edit&amp;section=4" title="Edit section: Messages">edit</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>The Messages app for iPhone was released as a replacement to the Text app which had been the native messaging app since the iPhone’s release. The change in name was due to the addition of native support for the <a href="/wiki/Multimedia_Messaging_Service" title="Multimedia Messaging Service">Multimedia Messaging Service</a> (MMS), in addition to the text-only <a href="/wiki/Short_Message_Service" title="Short Message Service">Short Message Service</a> (SMS), allowing users to send and receive messages that also contain everything from attachments, contacts, locations, voice recordings, pictures, and video messages.<sup class="reference" id="cite_ref-6"><a href="#cite_note-6">[6]</a></sup></p>
<p>Other changes to the app including support for <a href="/wiki/VCard" title="VCard">vCard</a> standard, support for copy and paste, and the ability to forward or delete multiple messages at once.</p>
<h3><span class="mw-headline" id="Phone">Phone</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=IPhone_OS_3&amp;action=edit&amp;section=5" title="Edit section: Phone">edit</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>The Phone app now has an ability to set his/her own number in the SIM card. Then it will be displayed in the Contacts list and iTunes, and it will used for MMS identification. The Recent calls section now shows more details and it can be edit contacts within the app.</p>
<h3><span class="mw-headline" id="Photos_and_Camera">Photos and Camera</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=IPhone_OS_3&amp;action=edit&amp;section=6" title="Edit section: Photos and Camera">edit</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>The camera app introduced video recording, which can be trimmed directly in the app. It also has autofocus, which the user can tap on different objects, and the camera would focus on it. These features were only available on the iPhone 3GS.</p>
<p>The Photos options menu has changed, with a new copy button, and a redesigned UI. It also has the ability to delete multiple photos all at once.</p>
<h3><span class="mw-headline" id="Voice_Memos">Voice Memos</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=IPhone_OS_3&amp;action=edit&amp;section=7" title="Edit section: Voice Memos">edit</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>iPhone OS 3 adds a Voice Memos app.<sup class="reference" id="cite_ref-7"><a href="#cite_note-7">[7]</a></sup> The app allows the user to record a new memo or view existing voice memos.<sup class="reference" id="cite_ref-8"><a href="#cite_note-8">[8]</a></sup></p>
<h3><span class="mw-headline" id="Other_new_features">Other new features</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=IPhone_OS_3&amp;action=edit&amp;section=8" title="Edit section: Other new features">edit</a><span class="mw-editsection-bracket">]</span></span></h3>
<h4><span class="mw-headline" id="Home_Screen">Home Screen</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=IPhone_OS_3&amp;action=edit&amp;section=9" title="Edit section: Home Screen">edit</a><span class="mw-editsection-bracket">]</span></span></h4>
<p>iPhone OS 3 expands the number of pages to 11 pages, for a total number of 180 apps in the home screen.<sup class="reference" id="cite_ref-:0_5-1"><a href="#cite_note-:0-5">[5]</a></sup></p>
<h2><span class="mw-headline" id="Problems">Problems</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=IPhone_OS_3&amp;action=edit&amp;section=10" title="Edit section: Problems">edit</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>Some iPhone and iPod touch users reported issues of the device(s) not waking from sleep.<sup class="noprint Inline-Template Template-Fact" style="white-space:nowrap;">[<i><a href="/wiki/Wikipedia:Citation_needed" title="Wikipedia:Citation needed"><span title="This claim needs references to reliable sources. (January 2016)">citation needed</span></a></i>]</sup> Apple released iPhone OS 3.1.2 in an attempt to fix the issue.</p>
<p>Some iPad users running iOS 3.2 reported Wi-fi issues. The issue was fixed in iOS 3.2.1.<sup class="reference" id="cite_ref-9"><a href="#cite_note-9">[9]</a></sup></p>
<h2><span class="mw-headline" id="Reception">Reception</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=IPhone_OS_3&amp;action=edit&amp;section=11" title="Edit section: Reception">edit</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>iPhone OS 3 received mixed reviews.<sup class="noprint Inline-Template Template-Fact" style="white-space:nowrap;">[<i><a href="/wiki/Wikipedia:Citation_needed" title="Wikipedia:Citation needed"><span title="This claim needs references to reliable sources. (January 2016)">citation needed</span></a></i>]</sup></p>
<h2><span class="mw-headline" id="iPod_touch_price">iPod touch price</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=IPhone_OS_3&amp;action=edit&amp;section=12" title="Edit section: iPod touch price">edit</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>iPhone OS 3 cost $4.95 for iPod touch (1st and 2nd generations) users and was free to iPod touch (3rd generation), iPhone users, and iPad users.</p>
