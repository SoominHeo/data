<p><i><b>DJ Max Fever</b></i> (Korean: <b>ëì ì´ë§¥ì¤ í¼ë²</b>, <i>dijeimaegseu pibeo</i>; <a href="/wiki/Abbreviation" title="Abbreviation">abbr.</a>: DMF) is a <a class="mw-redirect" href="/wiki/Music_game" title="Music game">music game</a> for the <a href="/wiki/PlayStation_Portable" title="PlayStation Portable">PlayStation Portable</a> developed by <a href="/wiki/Pentavision" title="Pentavision">Pentavision</a> and <a class="mw-redirect" href="/wiki/PM_Studios" title="PM Studios">PM Studios</a>. It was released in North America on January 27, 2009. Although the franchise has made several appearances in Asia, <i>DJ Max Fever</i> is the first <i><a href="/wiki/DJMax" title="DJMax">DJMax</a></i> released in the United States and Europe. The game is a compilation of <a href="/wiki/DJMax_Portable" title="DJMax Portable">DJMax Portable</a> and <a href="/wiki/DJMax_Portable_2" title="DJMax Portable 2">DJMax Portable 2</a> along with additional features. By decision of PM Studios, it is the only title in the <i>DJMax</i> series with a space between words "DJ" and "Max". A digital version of the game was released in January 2010 on <a href="/wiki/PlayStation_Network" title="PlayStation Network">PlayStation Network</a> but later removed due to licensing expiring.</p>

<h2><span class="mw-headline" id="Gameplay">Gameplay</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=DJ_Max_Fever&amp;action=edit&amp;section=1" title="Edit section: Gameplay">edit</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>A player chooses a song, a video will start to play in the background, and the gear screen will display falling notes. As the notes cross the line in the gear, player must hit the corresponding button, to score points. The objective of the game is to earn a high score and ranks in order to unlock more items such as avatars, songs, gears and more.</p>
<p>In terms of gameplay, <i>DJ Max Fever</i> is similar to other music games like <i><a href="/wiki/Beatmania" title="Beatmania">Beatmania</a></i>. When the notes are pressed in time, the combo counter will increase, missing the note will cause the combo counter to "break" which will reset the combo. As that combo count climbs, a Fever Gauge will fill. When the Fever bar is full, tapping X will double the combo and score, then triple if it is maintained properly and will continue to increase. On the right side of the screen is the healthbar. Once it's depleted, the game is over. There are 50 songs to go through and there are many varieties of songs to choose from, ranging from Techno to Pop, Rock to Hip-Hop, and more.</p>
<h2><span class="mw-headline" id="Features">Features</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=DJ_Max_Fever&amp;action=edit&amp;section=2" title="Edit section: Features">edit</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>The game is mostly based on <i><a href="/wiki/DJMax_Portable_2" title="DJMax Portable 2">DJMax Portable 2</a></i> but uses the <i><a href="/wiki/DJMax_Portable_Black_Square" title="DJMax Portable Black Square">DJMax Portable Black Square</a></i> Entire Control System, meaning the green notes that appear will stop the music track if missed. There is the auto-correct function, as for example if you hit the â with â², your hit will be auto corrected deducting however 20% of the score the note would normally get. 8B mode loading screens don't show the name of the song unlike in DJMax Portable 2. Also some of the "achievement discs" have been changed.</p>
<p>Since this is the first DJMax game released in Northern America there was no need for "the Link Disc" function and was removed for the Fever. The game is also toned down to be generally easier to play than most of the other DJMax games. Most mission songs have been changed, and the requirements have been toned down. Also the level cap of 30 for Easy difficulty has been removed and the score penalty for Easy difficulty has been reduced.</p>
<p>Soundtrack of the game is a selected combination of song from <i>DJMax Portable</i> and <i>DJMax Portable 2</i>. There is <i>DJ Max Fever</i> exclusive song called "Hip Hop Rescue" by "D.O" which uses the same music video as "Let Go". "MV" mode now has more videos, and some descriptions have been changed. "Album" has been removed from "Collection". Videos called "GET ON TOP ~No Cut Version~" and "DJMax Portable 2 ~Credits~" have been removed from the "Media" section of "Collection".</p>
<p>Like <i>DJMax Clazziquai Edition</i> and <i>Black Square</i>, <i>DJ Max Fever</i> prevents the transfer of save files, by associating the saved data with the MAC address of the PSP. If a save file is transferred to a PSP either by swapping memory sticks or downloading, the game recognizes the mismatch and refuses to load the game data. This can be circumvented using a MAC address spoofer. However, if the player puts the save file on a different Memory Stick, and uses the Memory Stick on the same PSP the Save File is created on, the data will still load.</p>
<h2><span class="mw-headline" id="Releases">Releases</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=DJ_Max_Fever&amp;action=edit&amp;section=3" title="Edit section: Releases">edit</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>The game was initially set to be released at November 11, 2008 but was delayed to January 27, 2009 due to adding additional features and a new engine called "BS" to be compatible with the <a class="mw-redirect" href="/wiki/PSP-3000" title="PSP-3000">PSP-3000</a>.<sup class="reference" id="cite_ref-1"><a href="#cite_note-1">[1]</a></sup><sup class="reference" id="cite_ref-2"><a href="#cite_note-2">[2]</a></sup></p>
<p>30 limited edition versions were available to those who won the DJ Max Fever Contest hosted by <a href="/wiki/Joystiq" title="Joystiq">Joystiq</a> on March 16, 2009. Each set contained a copy of the game, soundtrack, art book, puzzle, post cards, and mask.<sup class="reference" id="cite_ref-bemani_3-0"><a href="#cite_note-bemani-3">[3]</a></sup> On April 9, 2009 a second limited edition called <i>DJ Max Fever -the crew edition-</i> was available for pre-order by <a class="new" href="/w/index.php?title=Bemanistyle&amp;action=edit&amp;redlink=1" title="Bemanistyle (page does not exist)">Bemanistyle</a>. DJ Max Fever Crew Edition contained the same as the original along with Technika Playing Cards.<sup class="reference" id="cite_ref-4"><a href="#cite_note-4">[4]</a></sup> Only 125 copies were made but were sold out before being released.<sup class="reference" id="cite_ref-5"><a href="#cite_note-5">[5]</a></sup></p>
<p>On January 20, 2010, <i>DJ Max Fever</i> was re-released on PlayStation Network. The PlayStation Network version fixed many of the bugs found in the original UMD version. PM Studios is currently considering what to do with the issue of UMD owners who want the bugfixes, considering that a save file can't be transferred between a UMD game and a downloadable one.<sup class="reference" id="cite_ref-bemani_3-1"><a href="#cite_note-bemani-3">[3]</a></sup></p>
<p>The digital version has since been removed from the PlayStation Network with no explanation given from PM Studios or Pentavision as to why it was removed.</p>
