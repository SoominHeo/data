<p><i><b>DJMax Portable Black Square</b></i> (<a href="/wiki/Korean_language" title="Korean language">Korean</a>: <b>ëì ì´ë§¥ì¤ í¬í°ë¸ ë¸ë ì¤íì´</b>, <i>dijeimaegseu poteobeul beullaeg seukweeo</i>; <a href="/wiki/Abbreviation" title="Abbreviation">abbr.</a>: DMP:BS) is a <a class="mw-redirect" href="/wiki/Music_game" title="Music game">music game</a> for the <a href="/wiki/PlayStation_Portable" title="PlayStation Portable">PlayStation Portable</a> developed by <a href="/wiki/South_Korea" title="South Korea">South Korean</a> developer <a href="/wiki/Pentavision" title="Pentavision">Pentavision</a> released on December 24, 2008.<sup class="reference" id="cite_ref-1"><a href="#cite_note-1">[1]</a></sup> This is the fourth major game of the <i><a href="/wiki/DJMax" title="DJMax">DJMax</a></i> series released for the <a href="/wiki/PlayStation_Portable" title="PlayStation Portable">PSP</a>. The game includes various enhancements and new features built on from <i><a href="/wiki/DJMax_Portable_2" title="DJMax Portable 2">DJMax Portable 2</a></i> and <i><a href="/wiki/DJMax_Portable_Clazziquai_Edition" title="DJMax Portable Clazziquai Edition">DJMax Portable Clazziquai Edition</a></i>. The original aim was to launch the game in November, however due to numerous bugs or piracy issues<sup class="reference" id="cite_ref-2"><a href="#cite_note-2">[2]</a></sup> with <i>DJMax Portable Clazziquai Edition</i>, the release of <i>DJMax Portable Black Square</i> was postponed. The game was released in Japan on March 22, 2012, it removes the autocorrect feature and Japanese is the only language option available, yet most of the game retains English titles for songs, artwork and so on.</p>

<h2><span class="mw-headline" id="Features">Features</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=DJMax_Portable_Black_Square&amp;action=edit&amp;section=1" title="Edit section: Features">edit</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>DJMax Black Square features largely the same features as <a href="/wiki/DJMax_Portable_Clazziquai_Edition" title="DJMax Portable Clazziquai Edition">Clazziquai Edition</a>, but with increased difficulty and an almost completely different song list. It has the same user interface as Clazziquai Edition, but with a different "Bling Style" theme. In Black Square <i>2B Mode,</i> featured in Clazziquai Edition, has been removed. <i>8B Mode</i> has been re-introduced into the series and has been renamed <i>6BFX Mode</i> for this release. Black Square features note auto-correction which corrects the notes player misses. Black Square has an enhanced <i>Fever</i> bonus which can go up to a x7 multiplier. Black Square also has an extended Club Tour mode.</p>
<p>In Black Square a specialized note appears that's a different color than the rest, hitting these notes gives the player 1,000 bonus points and 1,000 extra points every time they hit another note of the same kind (ex: hit once in a row: 1,000, hit twice in a row, 2,000, hit ten times in a row: 10,000). The note also has a penalty - the music stops if it is missed. The music starts playing again when the next special note is hit, and the extra points are reset to 1000. Black Square introduces <i>RD</i> (<i>Redesign</i>) song patterns to the game series. Some songs have a special pattern, called RD style, that appears in the place of MX patterns on the song selection screen. This pattern is a remixed version of the song, and is usually harder. The RD songs can also be listened to on a special disc in Album Mode. Players can use <i>Link Disc</i> feature with DJMax Portable Clazziquai Edition, allowing them to play the songs of Clazziquai Edition along with song patterns and RD patterns unique to Black Square. Link Disc with DJMax Portable 1 and 2 is also possible but only unlocks avatars, songs, notes, and gears for Black Square.</p>
<p>As with <a href="/wiki/DJ_Max_Fever" title="DJ Max Fever">DJ Max Fever</a>, DJMax Portable Black Square prevents the transfer of save files by associating the saved data with the <a href="/wiki/MAC_address" title="MAC address">MAC address</a> of the PSP. If a save file is transferred to a PSP either by swapping memory sticks or downloading, the game recognizes the mismatch and refuses to load the game data. This can be circumvented using a MAC address spoofer. However, if the player puts the save file on a different Memory Stick, and uses the Memory Stick on the same PSP the Save File is created on, the data will still load.</p>
<p>Due to piracy problems with Clazziquai Edition, Black Square includes few anti-piracy mechanisms. It utilizes methods such as the detection of the "ISO" folder in the memory stick, in which it subsequently will refuse to run, as well as deleting save game data for <a href="/wiki/DJMax_Portable_Clazziquai_Edition" title="DJMax Portable Clazziquai Edition">DJMax Portable Clazziquai Edition</a>. Nevertheless, crackers have managed to circumvent the protection in question through a patch.<sup class="noprint Inline-Template Template-Fact" style="white-space:nowrap;">[<i><a href="/wiki/Wikipedia:Citation_needed" title="Wikipedia:Citation needed"><span title="This claim needs references to reliable sources. (April 2011)">citation needed</span></a></i>]</sup></p>
<h2><span class="mw-headline" id="Releases">Releases</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=DJMax_Portable_Black_Square&amp;action=edit&amp;section=2" title="Edit section: Releases">edit</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>Black Square was released in three different packages in Korea: the normal retail edition, a limited edition and two-in-one limited package which has both normal retail versions of Clazziquai Edition and Black Square. In Japan the game is set to be released in normal and limited editions.</p>

<p>A limited collector's edition package called <i>DJMax Portable Black Square - Quattra Limited Edition</i> was also released in limited quantities.<sup class="reference" id="cite_ref-3"><a href="#cite_note-3">[3]</a></sup> Only 1,500 of these were manufactured. <i>Black Square</i> is the fourth title in the <i>DJMax</i> series and the name for this special package is derived from Italian word "Quattro" which means number "four". The package itself is a black cube.<sup class="reference" id="cite_ref-4"><a href="#cite_note-4">[4]</a></sup> A two-day pre-order event was held for people whom wanted to reserve this package for themselves. On December 6, 2008, the first 1,000 packages of the limited edition became available for pre-order and later on December 7, 2008, the last 500 packages became available.<sup class="reference" id="cite_ref-5"><a href="#cite_note-5">[5]</a></sup></p>
<p>The limited edition is delivered in a package called <i>DJMAX Portable Black Square Premium Limited case</i> which has a magnet-based door mechanism. It contains a retail version of the <i>DJMax Portable Black Square</i> PSP game, an original soundtrack which consists of four audio CDs, an art book called <i>Illusion: DJMAX Metro Project Visualization Book</i> containing notes about the development of the game, graphical art and credits with pictures of every developer. The package also has small portable speakers, a <i>Black Square</i> themed bandana, a 2009 new year's card, and a <i>Son of Sun Special Gift</i> which consists of plant seeds and a cup. The package also has a few other items included although they aren't mentioned in the product description.<sup class="reference" id="cite_ref-6"><a href="#cite_note-6">[6]</a></sup> These additional items are a <i>TECHNIKA IC Card: Black Square Edition</i>, a <i>Black Square</i> limited package cover and a <i>NB Ranger</i> 3D paper figure.</p>
<p>The Japanese limited edition contains an art book, 4 CDs and the Japanese version of the game.</p>

<h2><span class="mw-headline" id="Glitches">Glitches</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=DJMax_Portable_Black_Square&amp;action=edit&amp;section=4" title="Edit section: Glitches">edit</a><span class="mw-editsection-bracket">]</span></span></h2>
<ul>
<li>After 505 plays, the game will display a message stating that a "New OST, Keys to the World, has been unlocked", however the player has unlocked "Fury" instead.</li>
<li>When player unlocks "Little Bird" note, "The Bird inside the egg" note is actually unlocked, and vice versa.</li>
<li>Another labeling error occurs in <i>Album mode.</i> In the video BGAs the in-video track information for "Super Lovely" and "Real Overdrive" are swapped, despite showing correctly in all other locations.</li>
</ul>
<h3><span class="mw-headline" id="Platinum_Crew_Patch">Platinum Crew Patch</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=DJMax_Portable_Black_Square&amp;action=edit&amp;section=5" title="Edit section: Platinum Crew Patch">edit</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>Members of the Platinum Crew were given access to two different updates for DJMax Portable Black Square.</p>
<ul>
<li>The first update fixes a bug in 6B Stage Mode which caused the same 2 songs to appear for all 3 stages. The update also unlocks Season 2 for the DJMax Portable Black Square Club Tour upon completion of Season 1 of the Club Tour.</li>
<li>The second update fixes a bug in Link Disc with DJMax Portable Clazziquai Edition. When unpatched, the explosion effect that occurs when the note is hit does not appear. The patch fixes this bug, making small explosions appear when a note is hit.</li>
</ul>
