<p><b>Iota Andromedae</b> (ι And, ι Andromedae) is a <a href="/wiki/Star" title="Star">star</a> in the <a href="/wiki/Constellation" title="Constellation">constellation</a> <a href="/wiki/Andromeda_(constellation)" title="Andromeda (constellation)">Andromeda</a>. It has an <a href="/wiki/Apparent_magnitude" title="Apparent magnitude">apparent magnitude</a> of +4.29<sup class="reference" id="cite_ref-scfs_2-2"><a href="#cite_note-scfs-2">[2]</a></sup> and is approximately 500 <a class="mw-redirect" href="/wiki/Light_years" title="Light years">light years</a> from Earth.<sup class="reference" id="cite_ref-aaa474_2_653_1-5"><a href="#cite_note-aaa474_2_653-1">[1]</a></sup></p>
<p>Iota Andromedae is a <a class="mw-redirect" href="/wiki/B-type_main_sequence_star" title="B-type main sequence star">B-type main sequence star</a> with a <a href="/wiki/Stellar_classification" title="Stellar classification">stellar classification</a> of B8 V. It is among the least <a href="/wiki/Variable_star" title="Variable star">variable stars</a> observed during the <a href="/wiki/Hipparcos" title="Hipparcos">Hipparcos</a> mission.<sup class="reference" id="cite_ref-aaa406_975_5-2"><a href="#cite_note-aaa406_975-5">[5]</a></sup></p>
<h2><span class="mw-headline" id="Name">Name</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Iota_Andromedae&amp;action=edit&amp;section=1" title="Edit section: Name">edit</a><span class="mw-editsection-bracket">]</span></span></h2>
<ul>
<li>In the catalogue of stars in the <i>Calendarium of Al Achsasi Al Mouakket</i>, this star was designated <b>Keff al-Salsalat</b> (<b>كف المسلسلة</b> - <i>kaf al-musalsala</i>), which was translated into <a href="/wiki/Latin" title="Latin">Latin</a> as <b>Manus Catenata</b>, meaning <i>palm of chained woman</i>.<sup class="reference" id="cite_ref-8"><a href="#cite_note-8">[8]</a></sup></li>
</ul>
<ul>
<li>In <a href="/wiki/Chinese_language" title="Chinese language">Chinese</a>, <span lang="zh" xml:lang="zh">螣蛇</span> (<span lang="zh-Latn" xml:lang="zh-Latn">Téng Shé</span>), meaning <i><a href="/wiki/Flying_serpent_(asterism)" title="Flying serpent (asterism)">Flying Serpent</a></i>, refers to an asterism consisting of ι Andromedae, <a href="/wiki/Alpha_Lacertae" title="Alpha Lacertae">α Lacertae</a>, <a href="/wiki/4_Lacertae" title="4 Lacertae">4 Lacertae</a>, <a href="/wiki/Pi2_Cygni" title="Pi2 Cygni">π<sup>2</sup> Cygni</a>, <a href="/wiki/Pi1_Cygni" title="Pi1 Cygni">π<sup>1</sup> Cygni</a>, <a href="/wiki/HD_206267" title="HD 206267">HD 206267</a>, <a href="/wiki/Epsilon_Cephei" title="Epsilon Cephei">ε Cephei</a>, <a href="/wiki/Beta_Lacertae" title="Beta Lacertae">β Lacertae</a>, <a href="/wiki/Sigma_Cassiopeiae" title="Sigma Cassiopeiae">σ Cassiopeiae</a>, <a href="/wiki/Rho_Cassiopeiae" title="Rho Cassiopeiae">ρ Cassiopeiae</a>, <a href="/wiki/Tau_Cassiopeiae" title="Tau Cassiopeiae">τ Cassiopeiae</a>, <a href="/wiki/AR_Cassiopeiae" title="AR Cassiopeiae">AR Cassiopeiae</a>, <a href="/wiki/9_Lacertae" title="9 Lacertae">9 Lacertae</a>, <a href="/wiki/3_Andromedae" title="3 Andromedae">3 Andromedae</a>, <a href="/wiki/7_Andromedae" title="7 Andromedae">7 Andromedae</a>, <a href="/wiki/8_Andromedae" title="8 Andromedae">8 Andromedae</a>, <a href="/wiki/Lambda_Andromedae" title="Lambda Andromedae">λ Andromedae</a>, <a href="/wiki/Kappa_Andromedae" title="Kappa Andromedae">κ Andromedae</a> and <a href="/wiki/Psi_Andromedae" title="Psi Andromedae">ψ Andromedae</a>. Consequently, ι Andromedae itself is known as <span lang="zh" xml:lang="zh">螣蛇二十二</span> (<span lang="zh-Latn" xml:lang="zh-Latn">Téng Shé èrshíèr</span>, English: the Twenty Second Star of Flying Serpent)<sup class="reference" id="cite_ref-9"><a href="#cite_note-9">[9]</a></sup></li>
</ul>
