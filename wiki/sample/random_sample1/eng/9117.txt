<p><b>Android Auto</b> is a smartphone projection standard developed by <a href="/wiki/Google" title="Google">Google</a> to allow <a class="mw-redirect" href="/wiki/Mobile_devices" title="Mobile devices">mobile devices</a> running the <a href="/wiki/Android_(operating_system)" title="Android (operating system)">Android</a> <a href="/wiki/Operating_system" title="Operating system">operating system</a> (<a href="/wiki/Android_Lollipop" title="Android Lollipop">version 5.0 "Lollipop"</a> and later) to be operated in automobiles through the <a href="/wiki/Dashboard" title="Dashboard">dashboard</a>'s <a href="/wiki/Head_unit" title="Head unit">head unit</a>. Android Auto was announced on June 25, 2014, at <a href="/wiki/Google_I/O" title="Google I/O">Google I/O</a> 2014. The Android Auto <a href="/wiki/Mobile_app" title="Mobile app">mobile app</a> was released on March 19, 2015.</p>
<p>The standard offers drivers control over <a class="mw-redirect" href="/wiki/GPS" title="GPS">GPS</a> <a href="/wiki/Web_mapping" title="Web mapping">mapping</a>/<a href="/wiki/Automotive_navigation_system" title="Automotive navigation system">navigation</a>, music playback, <a href="/wiki/Short_Message_Service" title="Short Message Service">SMS</a>, telephony, and <a href="/wiki/Web_search_engine" title="Web search engine">web search</a>; both touchscreen and button-controlled head unit displays will be supported, although hands-free operation through <a href="/wiki/Speech_recognition" title="Speech recognition">voice commands</a> is emphasized to ensure safe driving. Compatible apps include <a class="mw-redirect" href="/wiki/Google_Maps_(application)" title="Google Maps (application)">Google Maps</a>, <a href="/wiki/Google_Play_Music" title="Google Play Music">Google Play Music</a>, MLB at Bat, <a href="/wiki/Spotify" title="Spotify">Spotify</a>, <a href="/wiki/Songza" title="Songza">Songza</a>, <a class="mw-redirect" href="/wiki/Stitcher" title="Stitcher">Stitcher</a>, <a class="mw-redirect" href="/wiki/IHeart_Radio" title="IHeart Radio">iHeart Radio</a>, <a href="/wiki/Radioplayer" title="Radioplayer">Radioplayer</a>, and <a href="/wiki/TuneIn" title="TuneIn">TuneIn</a>.</p>
<p>While the official Android Auto website does list most car manufacturers as supporting Android Auto, many vehicles which are released for the 2015 and 2016 model years do not officially support it just yet.</p>
<p>Android Auto is part of the <a href="/wiki/Open_Automotive_Alliance" title="Open Automotive Alliance">Open Automotive Alliance</a>, which was announced on January 6, 2014, and is a joint effort with 28 automobile manufacturers and mobile tech supplier <a href="/wiki/Nvidia" title="Nvidia">Nvidia</a>. In May 2015, <a href="/wiki/Hyundai_Motor_Company" title="Hyundai Motor Company">Hyundai</a> became the first manufacturer to offer Android Auto, making it available first in the 2015 <a href="/wiki/Hyundai_Sonata" title="Hyundai Sonata">Hyundai Sonata</a>.</p>

<h2><span class="mw-headline" id="Functionality">Functionality</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Android_Auto&amp;action=edit&amp;section=1" title="Edit section: Functionality">edit</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>The aim of Android Auto is to extend the functionality of an <a href="/wiki/Android_(operating_system)" title="Android (operating system)">Android</a> <a href="/wiki/Mobile_device" title="Mobile device">mobile device</a> in an automobile to the dashboard's <a href="/wiki/Head_unit" title="Head unit">head unit</a>. In order to use the system, users must be running <a href="/wiki/Android_Lollipop" title="Android Lollipop">Lollipop</a> on their mobile device and must own a vehicle supporting Android Auto.<sup class="reference" id="cite_ref-acknow_1-0"><a href="#cite_note-acknow-1">[1]</a></sup> The driver's Android device connects to the vehicle via <a href="/wiki/USB" title="USB">USB</a> cable.<sup class="reference" id="cite_ref-2"><a href="#cite_note-2">[2]</a></sup> Rather than running its own operating system, the head unit will serve as an external display for the Android device, which runs all of the software, by presenting a car-specific <a href="/wiki/User_interface" title="User interface">user interface</a> built into Lollipop.<sup class="reference" id="cite_ref-acknow_1-1"><a href="#cite_note-acknow-1">[1]</a></sup><sup class="reference" id="cite_ref-cnet_3-0"><a href="#cite_note-cnet-3">[3]</a></sup></p>
<p>Several functions are supported:</p>
<ul>
<li><a class="mw-redirect" href="/wiki/GPS" title="GPS">GPS</a> <a href="/wiki/Web_mapping" title="Web mapping">mapping</a>/<a href="/wiki/Automotive_navigation_system" title="Automotive navigation system">navigation</a></li>
<li>Music control</li>
<li>Telephony</li>
<li><a href="/wiki/Short_Message_Service" title="Short Message Service">SMS</a> composition and playback</li>
<li><a href="/wiki/Web_search_engine" title="Web search engine">Web search</a></li>
</ul>
<p>An Android Auto <a href="/wiki/Software_development_kit" title="Software development kit">software development kit</a> (SDK) has been released, allowing third parties to modify their apps to work with Android Auto;<sup class="reference" id="cite_ref-cnet_3-1"><a href="#cite_note-cnet-3">[3]</a></sup> <a href="/wiki/Application_programming_interface" title="Application programming interface">APIs</a> will be only available for music and messaging apps initially.<sup class="reference" id="cite_ref-wired_4-0"><a href="#cite_note-wired-4">[4]</a></sup><sup class="reference" id="cite_ref-5"><a href="#cite_note-5">[5]</a></sup></p>
