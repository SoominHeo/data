<p><b>Android 2.3–2.3.7 "Gingerbread"</b> is a legacy version of the <a href="/wiki/Android_(operating_system)" title="Android (operating system)">Android</a> <a href="/wiki/Mobile_operating_system" title="Mobile operating system">mobile operating system</a> developed by <a href="/wiki/Google" title="Google">Google</a> and released in December 2010. The Gingerbread release introduced support for <a href="/wiki/Near_field_communication" title="Near field communication">near field communication</a> (NFC)—used in mobile payment solutions—and <a href="/wiki/Session_Initiation_Protocol" title="Session Initiation Protocol">Session Initiation Protocol</a> (SIP)—used in <a class="mw-redirect" href="/wiki/VoIP" title="VoIP">VoIP</a> internet telephony.<sup class="reference" id="cite_ref-1"><a href="#cite_note-1">[1]</a></sup></p>
<p>Gingerbread's user interface was refined in many ways, making it easier to master, faster to use, and more power-efficient. A simplified color scheme with a black background gave vividness and contrast to the notification bar, menus, and other user interface components. Improvements in menus and settings resulted in easier navigation and system control.</p>
<p>The <a href="/wiki/Nexus_S" title="Nexus S">Nexus S</a> smartphone, released in 2010, was the first phone from the <a href="/wiki/Google_Nexus" title="Google Nexus">Google Nexus</a> line that ran Gingerbread, and also the first one from the line with built-in NFC functionality.<sup class="reference" id="cite_ref-2"><a href="#cite_note-2">[2]</a></sup></p>
<p>Gingerbread uses version 2.6.35 of the <a href="/wiki/Linux_kernel" title="Linux kernel">Linux kernel</a>.</p>
<p>As of September 5, 2016, statistics issued by Google indicate that 1.5% of all Android devices accessing Google Play run Gingerbread.<sup class="reference" id="cite_ref-3"><a href="#cite_note-3">[3]</a></sup></p>
<h2><span class="mw-headline" id="Features">Features</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Android_Gingerbread&amp;action=edit&amp;section=1" title="Edit section: Features">edit</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>New features introduced by Gingerbread include the following:</p>
<ul>
<li>Updated user interface design, providing increased ease-of-use and efficiency.</li>
<li>Support for extra-large screen sizes and resolutions (<a class="mw-redirect" href="/wiki/WXGA_(graphics)" title="WXGA (graphics)">WXGA</a> and higher).</li>
<li>Native support for SIP VoIP internet telephony.</li>
<li>Improved text input using the virtual keyboard, with improved accuracy, better text suggestions, and voice input capability.</li>
<li>Enhanced copy/paste functionality, allowing users to select a word by press-hold, copy, and paste.</li>
<li>Support for Near Field Communication (NFC), allowing the user to read an NFC tag embedded in a poster, sticker, or advertisement .</li>
<li>New audio effects such as reverb, equalization, headphone virtualization, and bass boost.</li>
<li>New Download Manager, giving users easy access to any file downloaded from the browser, <a href="/wiki/Email" title="Email">email</a>, or another application.</li>
<li>Support for multiple cameras on the device, including a front-facing camera, if available.</li>
<li>Support for WebM/VP8 video playback, and <a href="/wiki/Advanced_Audio_Coding" title="Advanced Audio Coding">AAC</a> audio encoding.</li>
<li>Improved power management, including more active management of power-consuming applications.</li>
<li>Enhanced support for native code development.</li>
<li>A switch from <a href="/wiki/YAFFS" title="YAFFS">YAFFS</a> to <a href="/wiki/Ext4" title="Ext4">ext4</a> file system on newer devices.</li>
<li>Audio, graphical, and input enhancements for game developers.</li>
<li>Newer version of Unity available for game developers.</li>
<li>Concurrent garbage collection for increased performance.</li>
<li>Native support for more sensors (such as gyroscopes and barometers).</li>
<li>Android API levels: 9 and 10</li>
</ul>
