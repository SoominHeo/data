<p><b>Microsoft PowerPoint</b> is a <a href="/wiki/Slide_show" title="Slide show">slide show</a> <a href="/wiki/Presentation_program" title="Presentation program">presentation program</a> currently developed by <a href="/wiki/Microsoft" title="Microsoft">Microsoft</a>, for use on both Microsoft and Apple Macintosh operating systems. PowerPoint, initially named "Presenter", was created by <a class="mw-redirect" href="/wiki/Forethought_Inc." title="Forethought Inc.">Forethought Inc.</a>. Microsoft's version of PowerPoint was officially launched on May 22, 1990, as a part of the <a href="/wiki/Microsoft_Office" title="Microsoft Office">Microsoft Office</a> suite. PowerPoint is useful for helping develop the slide-based presentation format and is currently one of the most commonly used slide-based presentation programs available. Microsoft has also released the PowerPoint mobile application for use on Apple and Android mobile operating systems.</p>

<h2><span class="mw-headline" id="History">History</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Microsoft_PowerPoint&amp;action=edit&amp;section=1" title="Edit section: History">edit</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>Originally designed for the <a class="mw-redirect" href="/wiki/Apple_Macintosh" title="Apple Macintosh">Macintosh</a> computer, the initial release was called "Presenter", developed by Thomas Rudkin and Dennis Austin<sup class="reference" id="cite_ref-3"><a href="#cite_note-3">[3]</a></sup> of <a href="/wiki/Forethought,_Inc." title="Forethought, Inc.">Forethought, Inc.</a><sup class="reference" id="cite_ref-4"><a href="#cite_note-4">[4]</a></sup> In 1987, it was renamed to "PowerPoint" due to problems with trademarks, the idea for the name coming from <a href="/wiki/Robert_Gaskins" title="Robert Gaskins">Robert Gaskins</a>.<sup class="reference" id="cite_ref-5"><a href="#cite_note-5">[5]</a></sup></p>
<p>By then some experts believed that "desktop presentations", using computers to create <a href="/wiki/Flip_chart" title="Flip chart">flip charts</a> and <a href="/wiki/Transparency_(projection)" title="Transparency (projection)">overhead transparencies</a>, could become as large a market as <a href="/wiki/Desktop_publishing" title="Desktop publishing">desktop publishing</a>.<sup class="reference" id="cite_ref-nyt19870731_6-0"><a href="#cite_note-nyt19870731-6">[6]</a></sup> That year Forethought was bought by Microsoft for $14 million<sup class="reference" id="cite_ref-zonona19870731_7-0"><a href="#cite_note-zonona19870731-7">[7]</a></sup><sup class="reference" id="cite_ref-nyt19870731_6-1"><a href="#cite_note-nyt19870731-6">[6]</a></sup> ($29.5 million in present-day terms<sup class="reference" id="cite_ref-inflation-US_8-0"><a href="#cite_note-inflation-US-8">[8]</a></sup>), and became Microsoft's Graphics Business Unit, which continued to develop the software further. Microsoft's version of PowerPoint was officially launched on May 22, 1990, the same day that Microsoft released <a href="/wiki/Windows_3.0" title="Windows 3.0">Windows 3.0</a>.</p>
<p>PowerPoint introduced many new changes with the release of PowerPoint 97. It incorporated the <a href="/wiki/Visual_Basic_for_Applications" title="Visual Basic for Applications">Visual Basic for Applications</a> (VBA) language, underlying all macro generation in Office 97.</p>
<p>PowerPoint 2000 (and the rest of the <a href="/wiki/Microsoft_Office_2000" title="Microsoft Office 2000">Office 2000</a> suite) introduced a clipboard that could hold multiple objects at once, and the Office Assistant was made less intrusive.<sup class="reference" id="cite_ref-9"><a href="#cite_note-9">[9]</a></sup></p>
<p>PowerPoint 2002 massively overhauled the animation engine, allowing users to create more advanced and custom animations.<sup class="reference" id="cite_ref-10"><a href="#cite_note-10">[10]</a></sup></p>
<p>PowerPoint 2011 makes it possible to remove image backgrounds, and provides additional special effects for pictures, such as 'Pencil effects'.</p>
<p>As of 2012<sup class="plainlinks noprint asof-tag update" style="display:none;"><a class="external text" href="//en.wikipedia.org/w/index.php?title=Microsoft_PowerPoint&amp;action=edit">[update]</a></sup>, various versions of PowerPoint claim ~95% of the presentation software market share, with installations on at least 1 billion computers. Among presenters world-wide, this program is used at an estimated frequency of 350 times per second.<sup class="reference" id="cite_ref-businessweek20120830_11-0"><a href="#cite_note-businessweek20120830-11">[11]</a></sup></p>
<h2><span class="mw-headline" id="Operation">Operation</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Microsoft_PowerPoint&amp;action=edit&amp;section=2" title="Edit section: Operation">edit</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>PowerPoint presentations consist of a number of individual pages or "slides". The "slide" analogy is a reference to the <a href="/wiki/Slide_projector" title="Slide projector">slide projector</a>. Slides may contain text, graphics, sound, movies, and other objects, which may be arranged freely. The presentation can be printed, displayed live on a computer, or navigated through at the command of the presenter. For larger audiences the computer display is often a <a href="/wiki/Video_projector" title="Video projector">video projector</a>. Slides can also form the basis of <a class="mw-redirect" href="/wiki/Webcasts" title="Webcasts">webcasts</a>.</p>
<p>PowerPoint provides three types of movements:</p>
<ol>
<li>Entrance, emphasis, and exit of elements on a slide itself are controlled by what PowerPoint calls <a href="/wiki/PowerPoint_animation" title="PowerPoint animation">Custom Animations</a>.</li>
<li>Transitions, on the other hand, there are movements between slides. These can be animated in a variety of ways.</li>
<li>Custom animation can be used to create small story boards by animating pictures to enter, exit or move.</li>
</ol>
<p>PowerPoint provides numerous features that offer flexibility and the ability to create a professional presentation. One of the features provides the ability to create a presentation that includes music which plays throughout the entire presentation or sound effects for particular slides. In addition to the ability to add sound files, the presentation can be designed to run, like a movie, on its own. PowerPoint allows the user to record the slide show with narration and a pointer. The user may customize slide shows to show the slides in a different order than originally designed and to have slides appear multiple times. Microsoft also offers the ability to broadcast the presentation to specific users via a link and Windows Live.</p>
<h2><span class="mw-headline" id="Cultural_impact">Cultural impact</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Microsoft_PowerPoint&amp;action=edit&amp;section=3" title="Edit section: Cultural impact">edit</a><span class="mw-editsection-bracket">]</span></span></h2>

<p><a href="/wiki/Jerry_Pournelle" title="Jerry Pournelle">Jerry Pournelle</a> in 1989 praised PowerPoint for the Macintosh, stating that "if you're in the business of putting on briefings and otherwise making presentations, you might want to seriously contemplate getting a Mac II just so you can use this program; it's that good. Highly recommended".<sup class="reference" id="cite_ref-pournelle198901_12-0"><a href="#cite_note-pournelle198901-12">[12]</a></sup> Supporters say that<sup class="reference" id="cite_ref-13"><a href="#cite_note-13">[13]</a></sup><sup class="reference" id="cite_ref-14"><a href="#cite_note-14">[14]</a></sup><sup class="reference" id="cite_ref-15"><a href="#cite_note-15">[15]</a></sup> the ease of use of presentation software can save a lot of time for people who otherwise would have used other types of visual aid—hand-drawn or mechanically typeset slides, blackboards or whiteboards, or overhead projections. Ease of use also encourages those who otherwise would not have used visual aids, or would not have given a presentation at all, to make presentations. As PowerPoint's style, <a href="/wiki/PowerPoint_animation" title="PowerPoint animation">animation</a>, and <a href="/wiki/Multimedia" title="Multimedia">multimedia</a> abilities have become more sophisticated, and as the application has generally made it easier to produce presentations (even to the point of having an "AutoContent Wizard" that was discontinued in PowerPoint 2007, suggesting a structure for a presentation), the difference in needs and desires of presenters and audiences has become more noticeable.<sup class="noprint Inline-Template Template-Fact" style="white-space:nowrap;">[<i><a href="/wiki/Wikipedia:Citation_needed" title="Wikipedia:Citation needed"><span title="This claim needs references to reliable sources. (June 2016)">citation needed</span></a></i>]</sup> Experienced PowerPoint designers point out that the "AutoContent Wizard" caused a glitch which contributed greatly to on-screen freezing of slides. Many designers opt to use the "blank slide layout" in lieu of the other layout choices for this reason. Nevertheless, in normal business use, most presentations created using PowerPoint are based on its default layout and font choices.<sup class="reference" id="cite_ref-16"><a href="#cite_note-16">[16]</a></sup></p>
<p>The benefit of PowerPoint is continually debated, though most people believe that the benefit may be to present structural presentations to business workers, such as Raytheon Elcan does.<sup class="reference" id="cite_ref-17"><a href="#cite_note-17">[17]</a></sup> Its use in classroom lectures has influenced investigations of PowerPoint's effects on student performance in comparison to lectures based on overhead projectors, traditional lectures, and online lectures. There are no compelling results to prove or disprove that PowerPoint is more effective for learner retention than traditional presentation methods.<sup class="reference" id="cite_ref-18"><a href="#cite_note-18">[18]</a></sup> Statistician and designer <a href="/wiki/Edward_Tufte" title="Edward Tufte">Edward Tufte</a> suggests that as PowerPoint on its own has limited ability to present complex tables and graphics, a better approach is to provide the audience with printed data and a written report for them to read at the start of the meeting, before leading them through the report with a talk. He noted that after the <a class="mw-redirect" href="/wiki/Columbia_disaster" title="Columbia disaster">Columbia disaster</a>, a <a href="/wiki/Columbia_Accident_Investigation_Board" title="Columbia Accident Investigation Board">report</a> on the accident recommended that PowerPoint should never be used as the sole method for presenting scientific material.</p>
<h3><span class="mw-headline" id="Military_excess_in_the_US">Military excess in the US</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Microsoft_PowerPoint&amp;action=edit&amp;section=4" title="Edit section: Military excess in the US">edit</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>A "PowerPoint Ranger" is a military member who relies heavily on presentation software to the point of excess. Some junior officers spend the majority of their time preparing PowerPoint slides.<sup class="reference" id="cite_ref-Times_19-0"><a href="#cite_note-Times-19">[19]</a></sup> Because of its usefulness for presenting mission <a class="extiw" href="https://en.wiktionary.org/wiki/briefing" title="wikt:briefing">briefings</a>, it has become part of the culture of the military,<sup class="reference" id="cite_ref-NYTmilitary_20-0"><a href="#cite_note-NYTmilitary-20">[20]</a></sup><sup class="reference" id="cite_ref-SMJ_21-0"><a href="#cite_note-SMJ-21">[21]</a></sup> but is regarded as a poor decision-making tool.<sup class="reference" id="cite_ref-AFJ_22-0"><a href="#cite_note-AFJ-22">[22]</a></sup> As a result, some generals, such as Brigadier-General <a class="mw-redirect" href="/wiki/Herbert_McMaster" title="Herbert McMaster">Herbert McMaster</a>, have banned the use of PowerPoint in their operations.<sup class="reference" id="cite_ref-Times_19-1"><a href="#cite_note-Times-19">[19]</a></sup> In September 2010, <a href="/wiki/Colonel_(United_States)" title="Colonel (United States)">Colonel</a> Lawrence Sellin was fired from his post at the <a class="mw-redirect" href="/wiki/ISAF" title="ISAF">ISAF</a> for publishing a piece critical of the over-dependence of military <a href="/wiki/Staff_(military)" title="Staff (military)">staffs</a> on the presentation method and bloated bureaucracy.<sup class="reference" id="cite_ref-rant_23-0"><a href="#cite_note-rant-23">[23]</a></sup></p>
<h3><span class="mw-headline" id="Artistic_medium">Artistic medium</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Microsoft_PowerPoint&amp;action=edit&amp;section=5" title="Edit section: Artistic medium">edit</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>Musician <a href="/wiki/David_Byrne" title="David Byrne">David Byrne</a> has been using PowerPoint as a medium for art for years, producing a book and DVD and showing at galleries his PowerPoint-based artwork.<sup class="reference" id="cite_ref-24"><a href="#cite_note-24">[24]</a></sup> The expressions "PowerPoint Art" or "pptArt" are used to define a contemporary Italian artistic movement which believes that the corporate world can be a unique and exceptional source of inspiration for the artist.</p>
<h2><span class="mw-headline" id="PowerPoint_Viewer">PowerPoint Viewer</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Microsoft_PowerPoint&amp;action=edit&amp;section=6" title="Edit section: PowerPoint Viewer">edit</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>Microsoft Office PowerPoint Viewer is a program used to run presentations on computers that do not have PowerPoint installed. Office PowerPoint Viewer (or in PowerPoint 2007 and later, a link to a viewer download) is added by default to the same disk or network location that contains one or more presentations packaged by using the Package for CD feature.</p>
<p>PowerPoint Viewer is installed by default with a Microsoft Office 2003 installation for use with the Package for CD feature. The PowerPoint Viewer file is also available for download from the Microsoft Office Online Web site.<sup class="reference" id="cite_ref-25"><a href="#cite_note-25">[25]</a></sup></p>
<p>Presentations password-protected for opening or modifying can be opened by PowerPoint Viewer. The Package for CD feature allows packaging any password-protected file or setting a new password for all packaged presentations. PowerPoint Viewer prompts for a password if the file is open password-protected.</p>
<p>PowerPoint Viewer supports opening presentations created using PowerPoint 97 and later. In addition, it supports all file content except OLE objects and scripting. PowerPoint Viewer is currently only available for computers running on Microsoft Windows.</p>
