<h1 class="firstHeading" id="firstHeading" lang="en">Access Systems Americas</h1>
<div class="mw-body-content" id="bodyContent">

<div class="mw-content-ltr" dir="ltr" id="mw-content-text" lang="en"><script>function mfTempOpenSection(id){var block=document.getElementById("mf-section-"+id);block.className+=" open-block";block.previousSibling.className+=" open-block";}</script><div class="floatright"><a class="image" href="/wiki/File:Palmsource_logo.png" title="PalmSource's logo"><img alt="PalmSource's logo" data-file-height="59" data-file-width="104" height="59" src="//upload.wikimedia.org/wikipedia/en/1/10/Palmsource_logo.png" width="104"/></a></div>
<p><b>ACCESS Systems Americas, Inc.</b> (formerly <b>PalmSource</b>) is a subsidiary of <a class="mw-redirect" href="/wiki/Access_Co." title="Access Co.">ACCESS</a> which develops the <a href="/wiki/Palm_OS" title="Palm OS">Palm OS</a> <a href="/wiki/Personal_digital_assistant" title="Personal digital assistant">PDA</a> <a href="/wiki/Operating_system" title="Operating system">operating system</a> and its successor, the <a href="/wiki/Access_Linux_Platform" title="Access Linux Platform">Access Linux Platform</a>, as well as <a href="/wiki/BeOS" title="BeOS">BeOS</a>. PalmSource was spun off from <a href="/wiki/Palm,_Inc." title="Palm, Inc.">Palm Computing, Inc.</a></p>
<p>Palm OS runs on 38 million devices that have been sold since 1996 from hardware manufacturers including Palm, Inc., <a href="/wiki/Samsung" title="Samsung">Samsung</a>, <a href="/wiki/IBM" title="IBM">IBM</a>, Aceeca, <a href="/wiki/AlphaSmart" title="AlphaSmart">AlphaSmart</a>, <a class="mw-redirect" href="/wiki/Fossil,_Inc." title="Fossil, Inc.">Fossil, Inc.</a>, <a href="/wiki/Garmin" title="Garmin">Garmin</a>, <a href="/wiki/Group_Sense_PDA" title="Group Sense PDA">Group Sense PDA</a> (Xplore), <a href="/wiki/Kyocera" title="Kyocera">Kyocera</a>, PiTech, <a href="/wiki/Sony" title="Sony">Sony</a>, and <a href="/wiki/Symbol_Technologies" title="Symbol Technologies">Symbol</a>. PalmSource also develops several programs for the Palm OS, and as of December 2005, PalmGear claims to offer 28,769 software titles of varying genres.<sup class="noprint Inline-Template Template-Fact" style="white-space:nowrap;">[<i><a href="/wiki/Wikipedia:Citation_needed" title="Wikipedia:Citation needed"><span title="This claim needs references to reliable sources. (April 2008)">citation needed</span></a></i>]</sup> Palm OS software programs can also be downloaded from <a href="/wiki/CNET" title="CNET">CNET</a>, PalmSource, <a href="/wiki/Handango" title="Handango">Handango</a>, and <a href="/wiki/Tucows" title="Tucows">Tucows</a>.</p>
<p>PalmSource also owns <a href="/wiki/BeOS" title="BeOS">BeOS</a>, which it purchased from <a href="/wiki/Be_Inc." title="Be Inc.">Be Inc.</a> in August 2001.<sup class="reference" id="cite_ref-PalmSource-Aug-2001-8-K_1-0"><a href="#cite_note-PalmSource-Aug-2001-8-K-1">[1]</a></sup></p>

<h2><span class="mw-headline" id="History">History</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Access_Systems_Americas&amp;action=edit&amp;section=1" title="Edit section: History">edit</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>In January 2002, Palm, Inc. set up a wholly owned subsidiary to develop and license Palm OS,<sup class="reference" id="cite_ref-2"><a href="#cite_note-2">[2]</a></sup> which was named PalmSource in February.<sup class="reference" id="cite_ref-3"><a href="#cite_note-3">[3]</a></sup> In October 2003, PalmSource was spun off from Palm as an independent company, and Palm renamed itself <a class="mw-redirect" href="/wiki/PalmOne" title="PalmOne">palmOne</a>.<sup class="reference" id="cite_ref-4"><a href="#cite_note-4">[4]</a></sup> palmOne and PalmSource set up a holding company that owned the Palm trademark.</p>
<p>In January 2004, PalmSource announced the successor to classic Palm OS called Palm OS Cobalt. However, it failed to gain support from hardware licensees.<sup class="reference" id="cite_ref-5"><a href="#cite_note-5">[5]</a></sup> That December, PalmSource acquired China MobileSoft, a software company with a mobile <a href="/wiki/Linux" title="Linux">Linux</a> offering. As a result, PalmSource announced that they would extend Palm OS to run on top of the <a class="mw-redirect" href="/wiki/Linux_architecture" title="Linux architecture">Linux architecture</a>.<sup class="reference" id="cite_ref-6"><a href="#cite_note-6">[6]</a></sup></p>
<p>In May 2005, palmOne purchased PalmSource's share of the Palm trademark for US$30 million<sup class="reference" id="cite_ref-Palm-May-2005-8-K_7-0"><a href="#cite_note-Palm-May-2005-8-K-7">[7]</a></sup><sup class="reference" id="cite_ref-PalmSource-May-2005-8-K_8-0"><a href="#cite_note-PalmSource-May-2005-8-K-8">[8]</a></sup> and two months later renamed itself Palm, Inc.<sup class="reference" id="cite_ref-Palm-Jun-2005-8-K_9-0"><a href="#cite_note-Palm-Jun-2005-8-K-9">[9]</a></sup> As part of the agreement, palmOne granted certain rights to Palm trademarks to PalmSource and licensees for a four-year transition period. Later that year, ACCESS, which specializes in mobile and embedded web browser technologies, including <a href="/wiki/NetFront" title="NetFront">NetFront</a>, acquired PalmSource for US$324 million.<sup class="reference" id="cite_ref-10"><a href="#cite_note-10">[10]</a></sup><sup class="reference" id="cite_ref-11"><a href="#cite_note-11">[11]</a></sup> In October 2006, PalmSource announced that it would rename itself to ACCESS, to match its parent company's name.<sup class="reference" id="cite_ref-12"><a href="#cite_note-12">[12]</a></sup><sup class="reference" id="cite_ref-13"><a href="#cite_note-13">[13]</a></sup></p>
