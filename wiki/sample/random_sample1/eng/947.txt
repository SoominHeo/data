<p>The <b>end key</b> is a key commonly found on <a href="/wiki/Computer_keyboard" title="Computer keyboard">computer keyboards</a>. The key has the opposite effect of the <a href="/wiki/Home_key" title="Home key">home key</a>. In some limited-size keyboards where the end key is missing the same functionality can be reached via the key combination of <kbd class="keyboard-key nowrap" style="border: 1px solid #aaa; -moz-border-radius: 0.2em; -webkit-border-radius: 0.2em; border-radius: 0.2em; -moz-box-shadow: 0.1em 0.1em 0.2em rgba(0,0,0,0.1); -webkit-box-shadow: 0.1em 0.1em 0.2em rgba(0,0,0,0.1); box-shadow: 0.1em 0.1em 0.2em rgba(0,0,0,0.1); background-color: #f9f9f9; background-image: -moz-linear-gradient(top, #eee, #f9f9f9, #eee); background-image: -o-linear-gradient(top, #eee, #f9f9f9, #eee); background-image: -webkit-linear-gradient(top, #eee, #f9f9f9, #eee); background-image: linear-gradient(to bottom, #eee, #f9f9f9, #eee); padding: 0.1em 0.3em; font-family: inherit; font-size: 0.85em;"><a href="/wiki/Fn_key" title="Fn key">fn</a></kbd>+<kbd class="keyboard-key nowrap" style="border: 1px solid #aaa; -moz-border-radius: 0.2em; -webkit-border-radius: 0.2em; border-radius: 0.2em; -moz-box-shadow: 0.1em 0.1em 0.2em rgba(0,0,0,0.1); -webkit-box-shadow: 0.1em 0.1em 0.2em rgba(0,0,0,0.1); box-shadow: 0.1em 0.1em 0.2em rgba(0,0,0,0.1); background-color: #f9f9f9; background-image: -moz-linear-gradient(top, #eee, #f9f9f9, #eee); background-image: -o-linear-gradient(top, #eee, #f9f9f9, #eee); background-image: -webkit-linear-gradient(top, #eee, #f9f9f9, #eee); background-image: linear-gradient(to bottom, #eee, #f9f9f9, #eee); padding: 0.1em 0.3em; font-family: inherit; font-size: 0.85em;">→</kbd></p>
<p>Its standard symbol <span class="nowrap"><span style="font-family: monospace, monospace;">U+21F2</span></span> <span style="font-size:125%">⇲</span> <span class="smallcaps" style="font-variant: small-caps; font-size: smaller;">SOUTH EAST ARROW TO CORNER</span>, as described in <a href="/wiki/ISO/IEC_9995" title="ISO/IEC 9995">ISO/IEC 9995-7</a>, is used on some full-size keyboards instead of a possibly localized text label.</p>

<h2><span class="mw-headline" id="Microsoft_Windows">Microsoft Windows</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=End_key&amp;action=edit&amp;section=1" title="Edit section: Microsoft Windows">edit</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>In modern <a href="/wiki/Microsoft_Windows" title="Microsoft Windows">Microsoft Windows</a> text editing applications, it is primarily used to put the cursor at the end of the line in which the cursor is. When the text is not editable, the end key is used to scroll to the end of the document; this can also be done in editable text if the key is pressed along with <a href="/wiki/Control_key" title="Control key">control</a>.</p>
<p>The end key can also be used to highlight all the characters after the cursor in a certain line if pressed along with <a href="/wiki/Shift_key" title="Shift key">shift</a> in editable text.</p>
<h2><span class="mw-headline" id="Mac_OS_X">Mac OS X</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=End_key&amp;action=edit&amp;section=2" title="Edit section: Mac OS X">edit</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>In most <a class="mw-redirect" href="/wiki/Mac_OS_X" title="Mac OS X">Mac OS X</a> applications, the key works differently than on other platforms. When the key is pressed, the window scrolls to the bottom, while the cursor position does not change at all; that is, the end key is tied to the window, not the text box being edited.<sup class="reference" id="cite_ref-1"><a href="#cite_note-1">[1]</a></sup> To get the same result as the Windows platform (that is, going to the end of the current line of text), press <kbd class="keyboard-key nowrap" style="border: 1px solid #aaa; -moz-border-radius: 0.2em; -webkit-border-radius: 0.2em; border-radius: 0.2em; -moz-box-shadow: 0.1em 0.1em 0.2em rgba(0,0,0,0.1); -webkit-box-shadow: 0.1em 0.1em 0.2em rgba(0,0,0,0.1); box-shadow: 0.1em 0.1em 0.2em rgba(0,0,0,0.1); background-color: #f9f9f9; background-image: -moz-linear-gradient(top, #eee, #f9f9f9, #eee); background-image: -o-linear-gradient(top, #eee, #f9f9f9, #eee); background-image: -webkit-linear-gradient(top, #eee, #f9f9f9, #eee); background-image: linear-gradient(to bottom, #eee, #f9f9f9, #eee); padding: 0.1em 0.3em; font-family: inherit; font-size: 0.85em;">⌘ <a href="/wiki/Command_key" title="Command key">Command</a></kbd>+<kbd class="keyboard-key nowrap" style="border: 1px solid #aaa; -moz-border-radius: 0.2em; -webkit-border-radius: 0.2em; border-radius: 0.2em; -moz-box-shadow: 0.1em 0.1em 0.2em rgba(0,0,0,0.1); -webkit-box-shadow: 0.1em 0.1em 0.2em rgba(0,0,0,0.1); box-shadow: 0.1em 0.1em 0.2em rgba(0,0,0,0.1); background-color: #f9f9f9; background-image: -moz-linear-gradient(top, #eee, #f9f9f9, #eee); background-image: -o-linear-gradient(top, #eee, #f9f9f9, #eee); background-image: -webkit-linear-gradient(top, #eee, #f9f9f9, #eee); background-image: linear-gradient(to bottom, #eee, #f9f9f9, #eee); padding: 0.1em 0.3em; font-family: inherit; font-size: 0.85em;">→</kbd>. In most single-line text fields, you can also instead press the down arrow key. An application can be used to change this behaviour.<sup class="reference" id="cite_ref-2"><a href="#cite_note-2">[2]</a></sup></p>
<h2><span class="mw-headline" id="Linux">Linux</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=End_key&amp;action=edit&amp;section=3" title="Edit section: Linux">edit</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>In <a href="/wiki/Linux" title="Linux">Linux</a>, the end key has basically the same functionality as it does on Windows. It positions the cursor at the end of a line in editable text, and otherwise scrolls a scrollable document to the end. Also, like Windows, the end key can be used to highlight all the characters after the cursor in a certain line if pressed along with shift in editable text.</p>
<h2><span class="mw-headline" id="Non-GUI_applications">Non-GUI applications</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=End_key&amp;action=edit&amp;section=4" title="Edit section: Non-GUI applications">edit</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>In older screen-oriented, text-based (non-GUI) applications, the user pressed the "End" key to indicate that they had finished entering data on a particular "screen".</p>
