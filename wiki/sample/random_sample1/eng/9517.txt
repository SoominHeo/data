<p>In <a href="/wiki/Vector_calculus" title="Vector calculus">vector calculus</a>, the <b>Jacobian matrix</b> (<span class="nowrap"><span class="IPA nopopups"><a href="/wiki/Help:IPA_for_English" title="Help:IPA for English">/<span style="border-bottom:1px dotted"><span title="/dʒ/ 'j' in 'jam'">dʒ</span><span title="/ɪ/ or /ə/ 'e' in 'roses'">ᵻ</span><span title="/ˈ/ primary stress follows">ˈ</span><span title="'k' in 'kind'">k</span><span title="/oʊ/ long 'o' in 'code'">oʊ</span><span title="'b' in 'buy'">b</span><span title="/i/ 'y' in 'happy'">i</span><span title="/ə/ 'a' in 'about'">ə</span><span title="'n' in 'no'">n</span></span>/</a></span></span>, <span class="nowrap"><span class="IPA nopopups"><a href="/wiki/Help:IPA_for_English" title="Help:IPA for English">/<span style="border-bottom:1px dotted"><span title="/j/ 'y' in 'yes'">j</span><span title="/ɪ/ or /ə/ 'e' in 'roses'">ᵻ</span><span title="/ˈ/ primary stress follows">ˈ</span><span title="'k' in 'kind'">k</span><span title="/oʊ/ long 'o' in 'code'">oʊ</span><span title="'b' in 'buy'">b</span><span title="/i/ 'y' in 'happy'">i</span><span title="/ə/ 'a' in 'about'">ə</span><span title="'n' in 'no'">n</span></span>/</a></span></span>) is the <a href="/wiki/Matrix_(mathematics)" title="Matrix (mathematics)">matrix</a> of all first-order <a href="/wiki/Partial_derivative" title="Partial derivative">partial derivatives</a> of a <a href="/wiki/Vector-valued_function" title="Vector-valued function">vector-valued function</a>. When the matrix is a <a href="/wiki/Square_matrix" title="Square matrix">square matrix</a>, both the matrix and its <a href="/wiki/Determinant" title="Determinant">determinant</a> are referred to as the <b>Jacobian</b> in literature.<sup class="reference" id="cite_ref-1"><a href="#cite_note-1">[1]</a></sup></p>

<p>This matrix, whose entries are functions of <span class="texhtml"><b>x</b></span>, is also denoted by <span class="texhtml">D<b>f</b></span>, <span class="texhtml"><b>J</b><sub><b>f</b></sub></span>, and <span class="sfrac nowrap" style="display:inline-block; vertical-align:-0.5em; font-size:85%; text-align:center;"><span style="display:block; line-height:1em; margin:0 0.1em;"><i>∂</i>(<i>f</i><sub>1</sub>,...,<i>f</i><sub><i>m</i></sub>)</span><span class="visualhide">/</span><span style="display:block; line-height:1em; margin:0 0.1em; border-top:1px solid;"><i>∂</i>(<i>x</i><sub>1</sub>,...,<i>x</i><sub><i>n</i></sub>)</span></span>. (Note that some literature defines the Jacobian as the <a href="/wiki/Transpose" title="Transpose">transpose</a> of the matrix given above.)</p>
<p>The Jacobian matrix is important because if the function <span class="texhtml"><b>f</b></span> is <a class="mw-redirect" href="/wiki/Differentiable" title="Differentiable">differentiable</a> at a point <span class="texhtml"><b>x</b></span> (this is a slightly stronger condition than merely requiring that all partial derivatives exist there), then the Jacobian matrix defines a <a href="/wiki/Linear_map" title="Linear map">linear map</a> <span class="texhtml">ℝ<sup><i>n</i></sup> → ℝ<sup><i>m</i></sup></span>, which is the best (pointwise) <a href="/wiki/Linear_approximation" title="Linear approximation">linear approximation</a> of the function <span class="texhtml"><b>f</b></span> near the point <span class="texhtml"><b>x</b></span>. This linear map is thus the generalization of the usual notion of derivative, and is called the <i>derivative</i> or the <i>differential</i> of <span class="texhtml"><b>f</b></span> at <span class="texhtml"><b>x</b></span>.</p>
<p>If <span class="texhtml"><i>m</i></span> = <span class="texhtml"><i>n</i></span>, the Jacobian matrix is a square matrix, and its <a href="/wiki/Determinant" title="Determinant">determinant</a>, a function of <span class="texhtml"><i>x</i><sub>1</sub>, …, <i>x</i><sub><i>n</i></sub></span>, is the <b>Jacobian determinant</b> of <span class="texhtml"><b>f</b></span>. It carries important information about the local behavior of <span class="texhtml"><b>f</b></span>. In particular, the function <span class="texhtml"><b>f</b></span> has locally in the neighborhood of a point <span class="texhtml"><b>x</b></span> an <a href="/wiki/Inverse_function" title="Inverse function">inverse function</a> that is differentiable if and only if the Jacobian determinant is nonzero at <span class="texhtml"><b>x</b></span> (see <a href="/wiki/Jacobian_conjecture" title="Jacobian conjecture">Jacobian conjecture</a>). The Jacobian determinant also appears when changing the variables in <a href="/wiki/Multiple_integral" title="Multiple integral">multiple integrals</a> (see <a href="/wiki/Integration_by_substitution#Substitution_for_multiple_variables" title="Integration by substitution">substitution rule for multiple variables</a>).</p>
<p>If <span class="texhtml"><i>m</i></span> = 1, <span class="texhtml"><b>f</b></span> is a <a href="/wiki/Scalar_field" title="Scalar field">scalar field</a> and the Jacobian matrix is reduced to a <a class="mw-redirect" href="/wiki/Row_vector" title="Row vector">row vector</a> of partial derivatives of <span class="texhtml"><b>f</b></span>—i.e. the <a href="/wiki/Gradient" title="Gradient">gradient</a> of <span class="texhtml"><b>f</b></span>.</p>
<p>These concepts are named after the <a href="/wiki/Mathematician" title="Mathematician">mathematician</a> <a href="/wiki/Carl_Gustav_Jacob_Jacobi" title="Carl Gustav Jacob Jacobi">Carl Gustav Jacob Jacobi</a> (1804–1851).</p>

<h2><span class="mw-headline" id="Jacobian_matrix">Jacobian matrix</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Jacobian_matrix_and_determinant&amp;action=edit&amp;section=1" title="Edit section: Jacobian matrix">edit</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>The Jacobian generalizes the <a href="/wiki/Gradient" title="Gradient">gradient</a> of a <a href="/wiki/Scalar_(mathematics)" title="Scalar (mathematics)">scalar</a>-valued function of multiple variables, which itself generalizes the derivative of a scalar-valued function of a single variable. In other words, the Jacobian for a scalar-valued <a class="mw-redirect" href="/wiki/Multivariate_function" title="Multivariate function">multivariate function</a> is the gradient and that of a scalar-valued function of single variable is simply its derivative. The Jacobian can also be thought of as describing the amount of "stretching", "rotating" or "transforming" that a transformation imposes locally. For example, if <span class="texhtml">(<i>x</i>′, <i>y</i>′) = <b>f</b>(<i>x</i>, <i>y</i>)</span> is used to transform an image, the Jacobian <span class="texhtml"><b>J</b><sub><b>f</b></sub>(<i>x</i>, <i>y</i>)</span>, describes how the image in the neighborhood of <span class="texhtml">(<i>x</i>, <i>y</i>)</span> is transformed.</p>
<p>If a function is differentiable at a point, its derivative is given in coordinates by the Jacobian, but a function doesn't need to be differentiable for the Jacobian to be defined, since only the <a href="/wiki/Partial_derivative" title="Partial derivative">partial derivatives</a> are required to exist.</p>
<p>If <span class="texhtml"><b>p</b></span> is a point in <span class="texhtml">ℝ<sup><i>n</i></sup></span> and <span class="texhtml"><b>f</b></span> is <a href="/wiki/Derivative" title="Derivative">differentiable</a> at <span class="texhtml"><b>p</b></span>, then its derivative is given by <span class="texhtml"><b>J</b><sub><b>f</b></sub>(<b>p</b>)</span>. In this case, the <a href="/wiki/Linear_map" title="Linear map">linear map</a> described by <span class="texhtml"><b>J</b><sub><b>f</b></sub>(<b>p</b>)</span> is the best <a href="/wiki/Linear_approximation" title="Linear approximation">linear approximation</a> of <span class="texhtml"><b>f</b></span> near the point <span class="texhtml"><b>p</b></span>, in the sense that</p>

<p>In a sense, both the <a href="/wiki/Gradient" title="Gradient">gradient</a> and Jacobian are "<a href="/wiki/Derivative" title="Derivative">first derivatives</a>"—the former the first derivative of a <i>scalar function</i> of several variables, the latter the first derivative of a <i>vector function</i> of several variables.</p>
<p>The Jacobian of the gradient of a scalar function of several variables has a special name: the <a href="/wiki/Hessian_matrix" title="Hessian matrix">Hessian matrix</a>, which in a sense is the "<a href="/wiki/Second_derivative" title="Second derivative">second derivative</a>" of the function in question.</p>
<h2><span class="mw-headline" id="Jacobian_determinant">Jacobian determinant</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Jacobian_matrix_and_determinant&amp;action=edit&amp;section=2" title="Edit section: Jacobian determinant">edit</a><span class="mw-editsection-bracket">]</span></span></h2>
<div class="thumb tright">
<div class="thumbinner" style="width:402px;"><a class="image" href="/wiki/File:Jacobian_determinant_and_distortion.svg"><img alt="" class="thumbimage" data-file-height="340" data-file-width="766" height="178" src="//upload.wikimedia.org/wikipedia/en/thumb/9/96/Jacobian_determinant_and_distortion.svg/400px-Jacobian_determinant_and_distortion.svg.png" srcset="//upload.wikimedia.org/wikipedia/en/thumb/9/96/Jacobian_determinant_and_distortion.svg/600px-Jacobian_determinant_and_distortion.svg.png 1.5x, //upload.wikimedia.org/wikipedia/en/thumb/9/96/Jacobian_determinant_and_distortion.svg/800px-Jacobian_determinant_and_distortion.svg.png 2x" width="400"/></a>
<div class="thumbcaption">
<div class="magnify"><a class="internal" href="/wiki/File:Jacobian_determinant_and_distortion.svg" title="Enlarge"></a></div>
A nonlinear map <i>f</i> : <b>R</b><sup>2</sup> → <b>R</b><sup>2</sup> sends a small square to a distorted parallelogram close to the image of the square under the best linear approximation of <i>f</i> near the point.</div>
</div>
</div>
<p>If <span class="texhtml"><i>m</i></span>=<span class="texhtml"><i>n</i></span>, then <span class="texhtml"><b>f</b></span> is a function from <span class="texhtml">ℝ<sup><i>n</i></sup></span> to itself and the Jacobian matrix is a <a href="/wiki/Square_matrix" title="Square matrix">square matrix</a>. We can then form its <a href="/wiki/Determinant" title="Determinant">determinant</a>, known as the <b>Jacobian determinant</b>. The Jacobian determinant is occasionally referred to as "the Jacobian".</p>
<p>The Jacobian determinant at a given point gives important information about the behavior of <span class="texhtml"><b>f</b></span> near that point. For instance, the <a class="mw-redirect" href="/wiki/Continuously_differentiable_function" title="Continuously differentiable function">continuously differentiable function</a> <span class="texhtml"><b>f</b></span> is <a class="mw-redirect" href="/wiki/Invertible" title="Invertible">invertible</a> near a point <span class="texhtml"><b>p</b> ∈ ℝ<sup><i>n</i></sup></span> if the Jacobian determinant at <span class="texhtml"><b>p</b></span> is non-zero. This is the <a href="/wiki/Inverse_function_theorem" title="Inverse function theorem">inverse function theorem</a>. Furthermore, if the Jacobian determinant at <span class="texhtml"><b>p</b></span> is <a class="mw-redirect" href="/wiki/Positive_number" title="Positive number">positive</a>, then <span class="texhtml"><b>f</b></span> preserves orientation near <span class="texhtml"><b>p</b></span>; if it is <a href="/wiki/Negative_number" title="Negative number">negative</a>, <span class="texhtml"><b>f</b></span> reverses orientation. The <a href="/wiki/Absolute_value" title="Absolute value">absolute value</a> of the Jacobian determinant at <span class="texhtml"><b>p</b></span> gives us the factor by which the function <span class="texhtml"><b>f</b></span> expands or shrinks <a href="/wiki/Volume" title="Volume">volumes</a> near <span class="texhtml"><b>p</b></span>; this is why it occurs in the general <a class="mw-redirect" href="/wiki/Substitution_rule" title="Substitution rule">substitution rule</a>.</p>
<p>The Jacobian determinant is used when making a <a href="/wiki/Integration_by_substitution#Substitution_for_multiple_variables" title="Integration by substitution">change of variables</a> when evaluating a <a href="/wiki/Multiple_integral" title="Multiple integral">multiple integral</a> of a function over a region within its domain. To accommodate for the change of coordinates the magnitude of the Jacobian determinant arises as a multiplicative factor within the integral. This is because the <span class="texhtml"><i>n</i></span>-dimensional <span class="texhtml"><i>dV</i></span> element is in general a <a href="/wiki/Parallelepiped" title="Parallelepiped">parallelepiped</a> in the new coordinate system, and the <span class="texhtml"><i>n</i></span>-volume of a parallelepiped is the determinant of its edge vectors.</p>
<p>The Jacobian can also be used to solve <a href="/wiki/Matrix_differential_equation" title="Matrix differential equation">systems of differential equations</a> at an <a href="/wiki/Equilibrium_point" title="Equilibrium point">equilibrium point</a> or approximate solutions near an equilibrium point.</p>

<p>Conversely, if the Jacobian determinant is not zero at a point, then the function is <i>locally invertible</i> near this point, that is, there is a <a href="/wiki/Neighbourhood" title="Neighbourhood">neighbourhood</a> of this point in which the function is invertible.</p>
<p>The (unproved) <a href="/wiki/Jacobian_conjecture" title="Jacobian conjecture">Jacobian conjecture</a> is related to global invertibility in the case of a polynomial function, that is a function defined by <i>n</i> <a href="/wiki/Polynomial" title="Polynomial">polynomials</a> in <i>n</i> variables. It asserts that, if the Jacobian determinant is a non-zero constant (or, equivalently, that it does not have any complex zero), then the function is invertible and its inverse is a polynomial function.</p>
<h2><span class="mw-headline" id="Critical_points">Critical points</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Jacobian_matrix_and_determinant&amp;action=edit&amp;section=4" title="Edit section: Critical points">edit</a><span class="mw-editsection-bracket">]</span></span></h2>
<div class="hatnote" role="note">Main article: <a href="/wiki/Critical_point_(mathematics)" title="Critical point (mathematics)">Critical point</a></div>
<p>If <span class="texhtml"><b>f</b> : ℝ<sup><i>n</i></sup> → ℝ<sup><i>m</i></sup></span> is a <a href="/wiki/Differentiable_function" title="Differentiable function">differentiable function</a>, a <i>critical point</i> of <span class="texhtml"><b>f</b></span> is a point where the <a href="/wiki/Rank_(linear_algebra)" title="Rank (linear algebra)">rank</a> of the Jacobian matrix is not maximal. This means that the rank at the critical point is lower than the rank at some neighbour point. In other words, let <span class="texhtml"><i>k</i></span> be the maximal dimension of the <a class="mw-redirect" href="/wiki/Open_ball" title="Open ball">open balls</a> contained in the image of <span class="texhtml"><b>f</b></span>; then a point is critical if all minors of rank <span class="texhtml"><i>k</i></span> of <span class="texhtml"><b>f</b></span> are zero.</p>
<p>In the case where 1 = <span class="texhtml"><i>m</i></span> = <span class="texhtml"><i>n</i></span> = <span class="texhtml"><i>k</i></span>, a point is critical if the Jacobian determinant is zero.</p>

<p>From this we see that <span class="texhtml"><b>F</b></span> reverses orientation near those points where <span class="texhtml"><i>x</i><sub>1</sub></span> and <span class="texhtml"><i>x</i><sub>2</sub></span> have the same sign; the function is <a class="mw-redirect" href="/wiki/Locally" title="Locally">locally</a> invertible everywhere except near points where <span class="texhtml"><i>x</i><sub>1</sub> = 0</span> or <span class="texhtml"><i>x</i><sub>2</sub> = 0</span>. Intuitively, if one starts with a tiny object around the point <span class="texhtml">(1, 2, 3)</span> and apply <span class="texhtml"><b>F</b></span> to that object, one will get a resulting object with approximately <span class="texhtml">40 × 1 × 2 = 80</span> times the volume of the original one, with orientation reversed.</p>
<h2><span class="mw-headline" id="Other_uses">Other uses</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Jacobian_matrix_and_determinant&amp;action=edit&amp;section=11" title="Edit section: Other uses">edit</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>The Jacobian serves as a linearized <a href="/wiki/Design_matrix" title="Design matrix">design matrix</a> in statistical <a href="/wiki/Regression_analysis" title="Regression analysis">regression</a> and <a href="/wiki/Curve_fitting" title="Curve fitting">curve fitting</a>; see <a href="/wiki/Non-linear_least_squares" title="Non-linear least squares">non-linear least squares</a>.</p>
<h3><span class="mw-headline" id="Dynamical_systems">Dynamical systems</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Jacobian_matrix_and_determinant&amp;action=edit&amp;section=12" title="Edit section: Dynamical systems">edit</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>Consider a <a href="/wiki/Dynamical_system" title="Dynamical system">dynamical system</a> of the form <span class="texhtml"><b>x</b>′ = <b>F</b>(<b>x</b>)</span>, where <span class="texhtml"><b>x</b>′</span> is the (component-wise) time derivative of <span class="texhtml"><b>x</b></span>, and <span class="texhtml"><b>F</b> : ℝ<sup><i>n</i></sup> → ℝ<sup><i>n</i></sup></span> is differentiable. If <span class="texhtml"><b>F</b>(<b>x</b><sub>0</sub>) = 0</span>, then <span class="texhtml"><b>x</b><sub>0</sub></span> is a <a href="/wiki/Stationary_point" title="Stationary point">stationary point</a> (also called a critical point; this is not to be confused with fixed points). The behavior of the system near a stationary point is related to the <a class="mw-redirect" href="/wiki/Eigenvalue" title="Eigenvalue">eigenvalues</a> of <span class="texhtml"><b>J</b><sub><b>F</b></sub>(<b>x</b><sub>0</sub>)</span>, the Jacobian of <span class="texhtml"><b>F</b></span> at the stationary point.<sup class="reference" id="cite_ref-2"><a href="#cite_note-2">[2]</a></sup> Specifically, if the eigenvalues all have real parts that are negative, then the system is stable near the stationary point, if any eigenvalue has a real part that is positive, then the point is unstable. If the largest real part of the eigenvalues is zero, the Jacobian matrix does not allow for an evaluation of the stability.</p>
<h3><span class="mw-headline" id="Newton.27s_method">Newton's method</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Jacobian_matrix_and_determinant&amp;action=edit&amp;section=13" title="Edit section: Newton's method">edit</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>A system of coupled nonlinear equations can be solved iteratively by <a href="/wiki/Newton%27s_method#Nonlinear_systems_of_equations" title="Newton's method">Newton's method</a>. This method uses the Jacobian matrix of the system of equations.</p>
