<p><b>Adobe Dreamweaver</b> is a <a href="/wiki/Proprietary_software" title="Proprietary software">proprietary</a> <a href="/wiki/Web_development" title="Web development">web development</a> tool developed by Adobe Systems. Dreamweaver was created by <a href="/wiki/Macromedia" title="Macromedia">Macromedia</a> in 1997,<sup class="reference" id="cite_ref-macromedia1997_1-1"><a href="#cite_note-macromedia1997-1">[1]</a></sup> and was maintained by them until Macromedia was acquired by Adobe Systems in 2005.<sup class="reference" id="cite_ref-2"><a href="#cite_note-2">[2]</a></sup></p>
<p>Adobe Dreamweaver is available for <a href="/wiki/MacOS" title="MacOS">macOS</a> and for <a class="mw-redirect" href="/wiki/Windows" title="Windows">Windows</a>.</p>
<p>Following Adobe's acquisition of the Macromedia product suite, releases of Dreamweaver subsequent to version 8.0 have been more compliant with <a href="/wiki/World_Wide_Web_Consortium" title="World Wide Web Consortium">W3C</a> standards. Recent versions have improved support for <a href="/wiki/World_Wide_Web" title="World Wide Web">Web</a> technologies such as <a href="/wiki/Cascading_Style_Sheets" title="Cascading Style Sheets">CSS</a>, <a href="/wiki/JavaScript" title="JavaScript">JavaScript</a>, and various <a href="/wiki/Server-side_scripting" title="Server-side scripting">server-side scripting</a> <a href="/wiki/Programming_language" title="Programming language">languages</a> and <a href="/wiki/Software_framework" title="Software framework">frameworks</a> including <a href="/wiki/Active_Server_Pages" title="Active Server Pages">ASP</a> (ASP JavaScript, ASP VBScript, ASP.NET C#, ASP.NET VB), <a class="mw-redirect" href="/wiki/ColdFusion" title="ColdFusion">ColdFusion</a>, <a href="/wiki/Scriptlet" title="Scriptlet">Scriptlet</a>, and <a href="/wiki/PHP" title="PHP">PHP</a>.<sup class="reference" id="cite_ref-3"><a href="#cite_note-3">[3]</a></sup></p>

<h2><span class="mw-headline" id="Features">Features</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Adobe_Dreamweaver&amp;action=edit&amp;section=1" title="Edit section: Features">edit</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>Adobe Dreamweaver CC is a web design and development application that combines a visual design surface known as Live View and a code editor with standard features such as <a href="/wiki/Syntax_highlighting" title="Syntax highlighting">syntax highlighting</a>, <a class="mw-redirect" href="/wiki/Code_completion" title="Code completion">code completion</a>, and code collapsing as well as more sophisticated features such as real-time <a class="mw-redirect" href="/wiki/Syntax_analysis#Programming_languages" title="Syntax analysis">syntax checking</a> and code introspection for generating code hints to assist the user in writing code. Combined with an array of site management tools, Dreamweaver lets its users design, code and manage websites as well as mobile content. Dreamweaver is positioned as a versatile web design and development tool that enables visualization of web content while coding.</p>
<p>Dreamweaver, like <a class="mw-redirect" href="/wiki/Comparison_of_WYSIWYG_HTML_editors#Editor_features" title="Comparison of WYSIWYG HTML editors">other HTML editors</a>, edits <a href="/wiki/Computer_file" title="Computer file">files</a> locally then uploads them to the remote web server using <a href="/wiki/File_Transfer_Protocol" title="File Transfer Protocol">FTP</a>, <a class="mw-redirect" href="/wiki/SSH_file_transfer_protocol" title="SSH file transfer protocol">SFTP</a>, or <a href="/wiki/WebDAV" title="WebDAV">WebDAV</a>. Dreamweaver CS4 now supports the <a class="mw-redirect" href="/wiki/Subversion_(software)" title="Subversion (software)">Subversion (SVN)</a> version control system.</p>
<p>Since version 5, Dreamweaver supports <a href="/wiki/Syntax_highlighting" title="Syntax highlighting">syntax highlighting</a> for the following languages out of the box:</p>
<ul>
<li><a href="/wiki/ActionScript" title="ActionScript">ActionScript</a></li>
<li><a href="/wiki/Active_Server_Pages" title="Active Server Pages">Active Server Pages</a> (ASP).</li>
<li><a href="/wiki/C_Sharp_(programming_language)" title="C Sharp (programming language)">C#</a></li>
<li><a href="/wiki/Cascading_Style_Sheets" title="Cascading Style Sheets">Cascading Style Sheets</a> (CSS)</li>
<li><a class="mw-redirect" href="/wiki/ColdFusion" title="ColdFusion">ColdFusion</a></li>
<li>EDML</li>
<li><a class="mw-redirect" href="/wiki/Extensible_HyperText_Markup_Language" title="Extensible HyperText Markup Language">Extensible HyperText Markup Language</a> (XHTML)</li>
<li><a class="mw-redirect" href="/wiki/Extensible_Markup_Language" title="Extensible Markup Language">Extensible Markup Language</a> (XML)</li>
<li><a class="mw-redirect" href="/wiki/Extensible_Stylesheet_Language_Transformations" title="Extensible Stylesheet Language Transformations">Extensible Stylesheet Language Transformations</a> (XSLT)</li>
<li><a class="mw-redirect" href="/wiki/HyperText_Markup_Language" title="HyperText Markup Language">HyperText Markup Language</a> (HTML)</li>
<li><a href="/wiki/Java_(programming_language)" title="Java (programming language)">Java</a></li>
<li><a href="/wiki/JavaScript" title="JavaScript">JavaScript</a></li>
<li><a href="/wiki/PHP" title="PHP">PHP</a></li>
<li><a href="/wiki/Visual_Basic" title="Visual Basic">Visual Basic</a> (VB)</li>
<li>Visual Basic Script Edition (<a href="/wiki/VBScript" title="VBScript">VBScript</a>)</li>
<li><a href="/wiki/Wireless_Markup_Language" title="Wireless Markup Language">Wireless Markup Language</a> (WML)</li>
</ul>
<p>Support for <a href="/wiki/ASP.NET" title="ASP.NET">ASP.NET</a> and <a href="/wiki/JavaServer_Pages" title="JavaServer Pages">JavaServer Pages</a> was dropped in version CS5.<sup class="reference" id="cite_ref-4"><a href="#cite_note-4">[4]</a></sup></p>
<p>Users can add their own language syntax highlighting. In addition, <a href="/wiki/Autocomplete" title="Autocomplete">code completion</a> is available for many of these languages.</p>
<h2><span class="mw-headline" id="Internationalization_and_localization">Internationalization and localization</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Adobe_Dreamweaver&amp;action=edit&amp;section=2" title="Edit section: Internationalization and localization">edit</a><span class="mw-editsection-bracket">]</span></span></h2>
<h3><span class="mw-headline" id="Language_availability">Language availability</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Adobe_Dreamweaver&amp;action=edit&amp;section=3" title="Edit section: Language availability">edit</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>Adobe Dreamweaver CS6 is available in the following languages: Brazilian Portuguese, Simplified Chinese, Traditional Chinese, Czech, Dutch, English, French, German, Italian, Japanese, Korean (Windows only), Polish, Russian, Spanish, Swedish and Turkish.<sup class="reference" id="cite_ref-5"><a href="#cite_note-5">[5]</a></sup></p>
<h3><span class="mw-headline" id="Specific_features_for_Arabic_and_Hebrew_languages">Specific features for Arabic and Hebrew languages</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Adobe_Dreamweaver&amp;action=edit&amp;section=4" title="Edit section: Specific features for Arabic and Hebrew languages">edit</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>The older Adobe Dreamweaver CS3 also features a Middle Eastern version that allows typing Arabic, Persian, Urdu, or Hebrew text (written from right to left) within the code view. Whether the text is fully Middle Eastern (written from right to left) or includes both English and Middle Eastern text (written left to right and right to left), it will be displayed properly.</p>
