<p><b>Adobe Shockwave</b> (formerly <b>Macromedia Shockwave</b>) is a <a href="/wiki/Multimedia" title="Multimedia">multimedia</a> <a class="mw-redirect" href="/wiki/Platform_(computing)" title="Platform (computing)">platform</a> for building interactive multimedia applications and <a class="mw-redirect" href="/wiki/Video_games" title="Video games">video games</a>. Developers originate content using <a href="/wiki/Adobe_Director" title="Adobe Director">Adobe Director</a> and publish it on the Internet. Such content can be viewed in a <a href="/wiki/Web_browser" title="Web browser">web browser</a> on any computer with the <a href="/wiki/Adobe_Shockwave_Player" title="Adobe Shockwave Player">Shockwave Player</a> plug-in installed. <a class="mw-redirect" href="/wiki/Macromind" title="Macromind">Macromind</a> originated the technology; <a href="/wiki/Macromedia" title="Macromedia">Macromedia</a> developed it further, releasing Shockwave Player in 1995. <a href="/wiki/Adobe_Systems" title="Adobe Systems">Adobe Systems</a> acquired Shockwave in 2005.<sup class="reference" id="cite_ref-2"><a href="#cite_note-2">[2]</a></sup> Shockwave supports <a href="/wiki/Raster_graphics" title="Raster graphics">raster graphics</a>, basic <a href="/wiki/Vector_graphics" title="Vector graphics">vector graphics</a>, <a class="mw-redirect" href="/wiki/3D_graphics" title="3D graphics">3D graphics</a>, <a href="/wiki/Digital_audio" title="Digital audio">audio</a>, and an embedded scripting language called <a href="/wiki/Lingo_(programming_language)" title="Lingo (programming language)">Lingo</a>.<sup class="reference" id="cite_ref-3"><a href="#cite_note-3">[3]</a></sup><sup class="reference" id="cite_ref-4"><a href="#cite_note-4">[4]</a></sup></p>
<p>Shockwave is a common format for CD-ROM projectors, kiosk presentations, and interactive video games, and dominated in interactive multimedia during the 1990s.<sup class="reference" id="cite_ref-HartGeller2008_5-0"><a href="#cite_note-HartGeller2008-5">[5]</a></sup> Various <a class="mw-redirect" href="/wiki/Graphic_adventure_game" title="Graphic adventure game">graphic adventure games</a> were developed with Shockwave during the 1990s, including <i><a href="/wiki/The_Journeyman_Project" title="The Journeyman Project">The Journeyman Project</a></i>, <i><a href="/wiki/Total_Distortion" title="Total Distortion">Total Distortion</a></i>, <a href="/wiki/Mia%27s_Language_Adventure:_The_Kidnap_Caper" title="Mia's Language Adventure: The Kidnap Caper"><i>Mia's Language Adventure</i></a>, <a href="/wiki/Mia%27s_Science_Adventure:_Romaine%27s_New_Hat" title="Mia's Science Adventure: Romaine's New Hat"><i>Mia's Science Adventure</i></a>, and the <i><a href="/wiki/Didi_%26_Ditto" title="Didi &amp; Ditto">Didi &amp; Ditto</a></i> series. <a href="/wiki/Video_game_writer" title="Video game writer">Video-game writers</a> developed hundreds of free online video-games using Shockwave, publishing them on websites such as <a href="/wiki/Miniclip" title="Miniclip">Miniclip</a> and Shockwave.com.</p>
<p>In July 2011 a survey found that Flash Player had 99% market penetration in desktop browsers in "mature markets" (United States, Canada, United Kingdom, France, Germany, Japan, Australia, and New Zealand), while Shockwave Player claimed only 41% in these markets.<sup class="reference" id="cite_ref-Adobe_6-0"><a href="#cite_note-Adobe-6">[6]</a></sup> As of 2008<sup class="plainlinks noprint asof-tag update" style="display:none;"><a class="external text" href="//en.wikipedia.org/w/index.php?title=Adobe_Shockwave&amp;action=edit">[update]</a></sup>, <a href="/wiki/Adobe_Flash" title="Adobe Flash">Adobe Flash</a> and <a href="/wiki/Adobe_AIR" title="Adobe AIR">Adobe AIR</a> are preferred<sup class="noprint Inline-Template" style="white-space:nowrap;">[<i><a href="/wiki/Wikipedia:Manual_of_Style/Words_to_watch#Unsupported_attributions" title="Wikipedia:Manual of Style/Words to watch"><span title="The material near this tag may use weasel words or too-vague attribution. (October 2016)">by whom?</span></a></i>]</sup> (incompatible) alternatives to Shockwave, with its <a href="/wiki/Stage3D" title="Stage3D">3D rendering capabilities</a>, <a href="/wiki/ActionScript" title="ActionScript">object-oriented programming language</a>, and capacity to run as a <a class="mw-redirect" href="/wiki/Executable_file" title="Executable file">native executable</a> on multiple platforms.<sup class="reference" id="cite_ref-7"><a href="#cite_note-7">[7]</a></sup></p>

<h2><span class="mw-headline" id="History">History</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Adobe_Shockwave&amp;action=edit&amp;section=1" title="Edit section: History">edit</a><span class="mw-editsection-bracket">]</span></span></h2>
<h3><span class="mw-headline" id="MacroMind">MacroMind</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Adobe_Shockwave&amp;action=edit&amp;section=2" title="Edit section: MacroMind">edit</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>Shockwave originated with the <i>VideoWorks</i> application developed by <a href="/wiki/MacroMind" title="MacroMind">MacroMind</a> for the original <a class="mw-redirect" href="/wiki/Apple_Macintosh" title="Apple Macintosh">Apple Macintosh</a>. Animations were initially limited to the black and white of early Macintosh screens. VideoWorks was rebranded as <i>Director 1.0</i> in 1987. Director 2.2 was released in 1988, and included the <a href="/wiki/Lingo_(programming_language)" title="Lingo (programming language)">Lingo scripting language</a> with extensibility provided by <a href="#Xtras">Xtras</a>. A <a href="/wiki/Microsoft_Windows" title="Microsoft Windows">Windows</a> version was available in the early 1990s. Director 3.0 was the last version by MacroMind, and released in 1989 which introduced XObjects to Lingo. <a class="mw-redirect" href="/wiki/Shockwave_Player" title="Shockwave Player">Shockwave Player</a> had still not been developed, and the sole means of publishing content remained generating executable applications.</p>
<h3><span class="mw-headline" id="Macromedia">Macromedia</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Adobe_Shockwave&amp;action=edit&amp;section=3" title="Edit section: Macromedia">edit</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>In 1993, <a href="/wiki/Macromedia" title="Macromedia">Macromedia</a> acquired MacroMind, and took over the development of the Shockwave platform. As the Internet became more popular, Macromedia realized the potential for a web-based multimedia platform, and designed <a class="mw-redirect" href="/wiki/Shockwave_Player" title="Shockwave Player">Shockwave Player</a> for the leading <a href="/wiki/Web_browser" title="Web browser">web browser</a> of the time, <a href="/wiki/Netscape_Navigator" title="Netscape Navigator">Netscape Navigator</a>. <a class="mw-redirect" href="/wiki/Shockwave_Player" title="Shockwave Player">Shockwave Player</a> was released with Director 4.0 around 1995, and branded Shockwave Player 1.0. Its versioning has since been tied to Director's versioning, skipping versions 2 to 4. Shockwave was now a two-part system, a graphics and animation editor known as <a href="/wiki/Adobe_Director" title="Adobe Director">Macromedia Director</a>, and a player known as <a href="/wiki/Adobe_Shockwave_Player" title="Adobe Shockwave Player">Macromedia Shockwave Player</a>.</p>
<p>Macromedia Director quickly became the de facto production tool for the multimedia industry. By 1993 it was used to develop most Macintosh CD-ROM games,<sup class="reference" id="cite_ref-breen199312_8-0"><a href="#cite_note-breen199312-8">[8]</a></sup> such as <a class="mw-redirect" href="/wiki/Point-and-click" title="Point-and-click">point-and-click</a> <a class="mw-redirect" href="/wiki/Graphic_adventure" title="Graphic adventure">graphic adventures</a>.<sup class="reference" id="cite_ref-cm85_9-0"><a href="#cite_note-cm85-9">[9]</a></sup> Throughout the 1990s Director was credited with the creation of the majority of educational CD-ROMs.<sup class="reference" id="cite_ref-pcmag11_10-0"><a href="#cite_note-pcmag11-10">[10]</a></sup> It was preferred over competing applications due to its range of features, relative ease of use and Director's ability to publish executables for both <a class="mw-redirect" href="/wiki/Mac_OS" title="Mac OS">Apple</a> and <a href="/wiki/Microsoft_Windows" title="Microsoft Windows">Microsoft</a> operating systems.<sup class="reference" id="cite_ref-pcmag11_10-1"><a href="#cite_note-pcmag11-10">[10]</a></sup></p>
<p>A less-sophisticated alternative to Director was Apple's <a href="/wiki/HyperCard" title="HyperCard">HyperCard</a>.<sup class="reference" id="cite_ref-breen199312_8-1"><a href="#cite_note-breen199312-8">[8]</a></sup>. From 1995 to 1997 a competing multimedia authoring program appeared called <a href="/wiki/MTropolis" title="MTropolis">mTropolis</a> (from mFactory). In 1997, mTropolis was purchased and discontinued by <a class="mw-redirect" href="/wiki/Quark,_Inc." title="Quark, Inc.">Quark, Inc.</a>, who had its own plans into multimedia authoring with Quark Immedia.</p>
<p>In November 1996, Macromedia acquired <a href="/wiki/FutureWave_Software" title="FutureWave Software">FutureWave Software</a> and its FutureSplash products. <a href="/wiki/Adobe_Flash" title="Adobe Flash">Macromedia Flash</a> 1.0 was released shortly thereafter. Macromedia now controlled two of the three leading multimedia platforms for the web, with <a class="mw-redirect" href="/wiki/Sun_Java" title="Sun Java">Java</a> being the third.</p>
<p>Macromedia Director 8.5 was released in 2001 and was the first version to specifically target the <a href="/wiki/Video_game" title="Video game">video game</a> industry.<sup class="reference" id="cite_ref-cm85_9-1"><a href="#cite_note-cm85-9">[9]</a></sup> It introduced 3D capabilities, 3D text, toon shading, <a href="/wiki/Havok_(software)" title="Havok (software)">Havok physics</a>, <a class="mw-redirect" href="/wiki/Real_Video" title="Real Video">Real Video</a>, <a class="mw-redirect" href="/wiki/Real_Audio" title="Real Audio">Real Audio</a>, integration with <a href="/wiki/Adobe_Flash" title="Adobe Flash">Macromedia Flash 5</a>, behaviors, and other enhancements.<sup class="reference" id="cite_ref-cm85_9-2"><a href="#cite_note-cm85-9">[9]</a></sup> 3D modelling programs such as <a class="mw-redirect" href="/wiki/LightWave" title="LightWave">LightWave</a>, <a href="/wiki/Cinema_4D" title="Cinema 4D">Cinema 4D</a> and <a class="mw-redirect" href="/wiki/3D_Studio_Max" title="3D Studio Max">3D Studio Max</a> were upgraded to export 3D models for Shockwave.<sup class="reference" id="cite_ref-pcmag11_10-2"><a href="#cite_note-pcmag11-10">[10]</a></sup><sup class="reference" id="cite_ref-cm85_9-3"><a href="#cite_note-cm85-9">[9]</a></sup></p>
<p>As of 2001, over 200 million people had the <a href="/wiki/Adobe_Shockwave_Player" title="Adobe Shockwave Player">Macromedia Shockwave Player</a> installed, making Shockwave a common format for online video games.<sup class="reference" id="cite_ref-cm85_9-4"><a href="#cite_note-cm85-9">[9]</a></sup> Websites such as <a href="/wiki/Miniclip" title="Miniclip">Miniclip</a> and Shockwave.com were dedicated to Shockwave and Flash-based video games.<sup class="reference" id="cite_ref-11"><a href="#cite_note-11">[11]</a></sup><sup class="reference" id="cite_ref-12"><a href="#cite_note-12">[12]</a></sup></p>
<h3><span class="mw-headline" id="Adobe">Adobe</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Adobe_Shockwave&amp;action=edit&amp;section=4" title="Edit section: Adobe">edit</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>Macromedia was acquired by <a href="/wiki/Adobe_Systems" title="Adobe Systems">Adobe Systems</a> in 2005, and the entire Macromedia product line including <a href="/wiki/Adobe_Flash" title="Adobe Flash">Flash</a>, <a href="/wiki/Adobe_Dreamweaver" title="Adobe Dreamweaver">Dreamweaver</a>, <a href="/wiki/Adobe_Director" title="Adobe Director">Director</a>/Shockwave and <a class="mw-redirect" href="/wiki/Authorware" title="Authorware">Authorware</a> was now handled by Adobe. Director and Shockwave Player is currently developed and distributed by <a href="/wiki/Adobe_Systems" title="Adobe Systems">Adobe Systems</a>.</p>
<p>The early 2000s saw a demise in the usage of Director/Shockwave, and most multimedia professionals preferred <a href="/wiki/Adobe_Flash" title="Adobe Flash">Macromedia Flash</a> and other competing platforms. After the Adobe acquisition, no new versions were released for four years.<sup class="reference" id="cite_ref-pcmag11_10-3"><a href="#cite_note-pcmag11-10">[10]</a></sup></p>
<p>In 2007, Adobe released <i>Adobe Director 11</i>, the first new release in four years.<sup class="reference" id="cite_ref-pcmag11_10-4"><a href="#cite_note-pcmag11-10">[10]</a></sup> It introduced <a class="mw-redirect" href="/wiki/DirectX_9" title="DirectX 9">DirectX 9</a> native 3D rendering and the <a class="mw-redirect" href="/wiki/AGEIA" title="AGEIA">AGEIA</a> <a href="/wiki/PhysX" title="PhysX">PhysX</a> physics engine, panel docking, <a class="mw-redirect" href="/wiki/Quicktime" title="Quicktime">Quicktime</a> 7 support, <a href="/wiki/Windows_Media" title="Windows Media">Windows Media</a> and <a href="/wiki/RealPlayer" title="RealPlayer">RealPlayer</a> support, <a href="/wiki/Adobe_Flash" title="Adobe Flash">Adobe Flash CS3</a> integration and <a href="/wiki/Unicode" title="Unicode">Unicode</a> support.<sup class="reference" id="cite_ref-pcmag11_10-5"><a href="#cite_note-pcmag11-10">[10]</a></sup> It was considered an "incremental release" by reviewers and the scripting editor was still considered "primitive".<sup class="reference" id="cite_ref-pcmag11_10-6"><a href="#cite_note-pcmag11-10">[10]</a></sup></p>
<p>As of 2008, the market position of Director/Shockwave overlapped with Flash to a high degree, the only advantage of Director being its native 3D capabilities.<sup class="reference" id="cite_ref-pcmag11_10-7"><a href="#cite_note-pcmag11-10">[10]</a></sup> However, with the release of <a href="/wiki/Adobe_Flash_Player" title="Adobe Flash Player">Flash Player 11</a>, GPU-based 3D rendering was now supported using <a href="/wiki/Stage3D" title="Stage3D">Stage3D</a> (the underlying API), <a href="/wiki/Away3D" title="Away3D">Away3D</a> or <a href="/wiki/Flare3D" title="Flare3D">Flare3D</a> (3D game engines). And after <a href="/wiki/Adobe_AIR" title="Adobe AIR">Adobe AIR</a> was released, Flash programs could now be published as native applications, further reducing the need for Director.<sup class="reference" id="cite_ref-pcmag11_10-8"><a href="#cite_note-pcmag11-10">[10]</a></sup></p>
<h2><span class="mw-headline" id="Xtras">Xtras</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Adobe_Shockwave&amp;action=edit&amp;section=5" title="Edit section: Xtras">edit</a><span class="mw-editsection-bracket">]</span></span></h2>
<p><b>Xtras</b> are plug-ins for the Lingo scripting language, that enable additional functionality into a Shockwave project. Xtras are typically used to add file system I/O, hardware integration, and advanced multimedia functions. Xtras are supported and available for <a href="/wiki/Adobe_Director" title="Adobe Director">Adobe Director</a>, <a href="/wiki/Adobe_Authorware" title="Adobe Authorware">Adobe Authorware</a> and <a class="mw-redirect" href="/wiki/Adobe_Freehand" title="Adobe Freehand">Adobe Freehand</a>.</p>
<p>Many of Director's own functions are implemented as Xtras. Xtras use the Macromedia Open Architecture which was designed to allow easy creation of interchangeable components between Macromedia products. Adobe maintains a list of third party Xtras.</p>
<p>Xtras for <a href="/wiki/Microsoft_Windows" title="Microsoft Windows">Microsoft Windows</a> (32-bit) have an .X32 file extension. Xtras for <a class="mw-redirect" href="/wiki/Mac_OS" title="Mac OS">Mac OS</a> generally have an .XTR extension. The file extension *.X16 is reserved for Xtras for Microsoft Windows (16-bit).</p>
