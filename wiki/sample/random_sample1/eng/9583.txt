<p><b>Adobe Audition</b> (formerly <b>Cool Edit Pro</b>) is a <a href="/wiki/Digital_audio_workstation" title="Digital audio workstation">digital audio workstation</a> from <a href="/wiki/Adobe_Systems" title="Adobe Systems">Adobe Systems</a> featuring both a <a href="/wiki/Multitrack_recording" title="Multitrack recording">multitrack</a>, <a class="mw-redirect" href="/wiki/Non-destructive_editing" title="Non-destructive editing">non-destructive</a> mix/edit environment and a destructive-approach waveform editing view.</p>

<h2><span class="mw-headline" id="Origins">Origins</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Adobe_Audition&amp;action=edit&amp;section=1" title="Edit section: Origins">edit</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>Syntrillium Software was founded in the early 1990s by Robert Ellison and David Johnston, both former <a href="/wiki/Microsoft" title="Microsoft">Microsoft</a> employees. Originally developed by Syntrillium as Cool Edit, the program was distributed as <a href="/wiki/Crippleware" title="Crippleware">crippleware</a> for Windows computers. The full version was useful and flexible, particularly for its time. Syntrillium later released Cool Edit Pro, which added the capability to work with multiple tracks, as well as other features. Audio processing, however, was done in a destructive manner (at the time, most computers were not powerful enough in terms of processor performance and memory capacity to perform non-destructive operations in real time). Cool Edit Pro v2 added support for real-time non-destructive processing, and v2.1 added support for <a href="/wiki/Surround_sound" title="Surround sound">surround sound</a> mixing and unlimited simultaneous tracks (up to the limit imposed by the actual computer hardware). Cool Edit also included plugins such as <a href="/wiki/Noise_reduction" title="Noise reduction">noise reduction</a> and <a href="/wiki/Fast_Fourier_transform" title="Fast Fourier transform">FFT</a> <a href="/wiki/Equalization_(audio)" title="Equalization (audio)">equalization</a>.</p>
<p>Adobe purchased Cool Edit Pro from Syntrillium Software in May 2003 for $16.5 million<sup class="reference" id="cite_ref-2"><a href="#cite_note-2">[2]</a></sup> as well as a large <a class="mw-redirect" href="/wiki/Music_loop" title="Music loop">loop</a> library called "Loopology". Adobe then changed the name of Cool Edit Pro to "Adobe Audition".</p>
<h2><span class="mw-headline" id="Versions">Versions</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Adobe_Audition&amp;action=edit&amp;section=2" title="Edit section: Versions">edit</a><span class="mw-editsection-bracket">]</span></span></h2>
<h3><span class="mw-headline" id="Version_1">Version 1</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Adobe_Audition&amp;action=edit&amp;section=3" title="Edit section: Version 1">edit</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>Adobe Audition was released on 18 August 2003. It had bug fixes but no new features, and was essentially a more polished Cool Edit Pro 2.1 under a different name. Adobe then released Audition v1.5 in May 2004; major improvements over v1 included <a href="/wiki/Pitch_correction" title="Pitch correction">pitch correction</a>, <a class="mw-redirect" href="/wiki/Frequency_space" title="Frequency space">frequency space</a> editing, a <a class="mw-redirect" href="/wiki/CD" title="CD">CD</a> project view, basic video editing and integration with <a class="mw-redirect" href="/wiki/Adobe_Premiere" title="Adobe Premiere">Adobe Premiere</a>, as well as several other enhancements.</p>
<h3><span class="mw-headline" id="Version_2">Version 2</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Adobe_Audition&amp;action=edit&amp;section=4" title="Edit section: Version 2">edit</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>Adobe Audition 2 was released on 17 January 2006. With this release, Audition (which the music recording industry had once seen as a value-oriented home studio application, although it has long been used for editing by radio stations) entered the professional <a class="mw-redirect" href="/wiki/Digital_Audio_Workstation" title="Digital Audio Workstation">Digital Audio Workstation</a> market. The current version included two sections. Multitrack View supported up to 128 digital audio mono or stereo tracks at up to 32-bit resolution. In the Track Controls section one could select the input and output for each track (the program supported multiple multi-channel sound cards), select Record, Solo and Mute, and access the effects rack. New features included ASIO (<a class="mw-redirect" href="/wiki/Audio_stream_input_output" title="Audio stream input output">Audio stream input output</a>) support, VST (<a href="/wiki/Virtual_Studio_Technology" title="Virtual Studio Technology">Virtual Studio Technology</a>) support, new mastering tools (many provided by <a href="/wiki/IZotope" title="IZotope">iZotope</a>), and a redesigned <a href="/wiki/User_interface" title="User interface">UI</a>. Adobe also included Audition 2.0 as part of its Adobe Production Studio bundle.</p>
<h3><span class="mw-headline" id="Version_3">Version 3</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Adobe_Audition&amp;action=edit&amp;section=5" title="Edit section: Version 3">edit</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>Adobe Audition 3 was released on 8 November 2007. New features included <a class="mw-redirect" href="/wiki/VSTi" title="VSTi">VSTi</a> (virtual instrument) support, enhanced spectral editing, a redesigned multi-track interface, new effects, and a collection of royalty-free loops.</p>
<p>CS2 activation servers' shutdown: Adobe Audition 3, with some other CS2 products, was released with an official serial number, due to the technical glitch in Adobe's CS2 activation servers (<a href="/wiki/Adobe_Creative_Suite" title="Adobe Creative Suite">see Creative Suite 1 &amp; 2</a>).</p>
<h3><span class="mw-headline" id="Version_4_.28CS5.5.29">Version 4 (CS5.5)</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Adobe_Audition&amp;action=edit&amp;section=6" title="Edit section: Version 4 (CS5.5)">edit</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>Audition 4, also known as Audition CS5.5, was released on April 11, 2011 as part of <a href="/wiki/Adobe_Creative_Suite" title="Adobe Creative Suite">Adobe Creative Suite</a>. Audition 4 was shipped as part of the Adobe Creative Suite 5.5 Master Collection and Adobe Creative Suite 5.5 Production Premium, replacing the discontinued <a href="/wiki/Adobe_Soundbooth" title="Adobe Soundbooth">Adobe Soundbooth</a>. Audition 4 was also made available as a standalone product. Enhanced integration with Adobe Premiere Pro allowed editing of multitrack Premiere projects, and users of third-party software were served by the introduction of <a class="mw-redirect" href="/wiki/Open_Media_Framework" title="Open Media Framework">OMF</a>- and <a href="/wiki/XML" title="XML">XML</a>-based import/export functions. Other new features included improved 5.1 multichannel support, new effects (DeHummer, DeEsser, Speech Volume Leveler, and Surround Reverb), a History panel, faster and fully supported realtime FFT analysis and a new audio engine (more reliable and faster) for non-ASIO devices.</p>
<p>According to Adobe,<sup class="reference" id="cite_ref-3"><a href="#cite_note-3">[3]</a></sup> Audition CS5.5 was rewritten from the ground up to take advantage of parallel/batch processing for performance and make it a platform-agnostic product. Audition CS5.5 now works on Windows and Mac platforms. Over 15 years of C++ code was analyzed, and many, but not all, features of the previous Audition 3 were ported and/or enhanced. Notable features that were present in Audition 3, but removed for CS5.5 include <a class="mw-redirect" href="/wiki/VSTi" title="VSTi">VSTi</a> support and <a href="/wiki/MIDI" title="MIDI">MIDI</a> sequencing. Unlike all the previous versions, this is the first version to be available as a Mac version as well as a Windows version.</p>
<h3><span class="mw-headline" id="Version_5_.28CS6.29">Version 5 (CS6)</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Adobe_Audition&amp;action=edit&amp;section=7" title="Edit section: Version 5 (CS6)">edit</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>Adobe showed a <a class="mw-redirect" href="/wiki/Sneak_preview" title="Sneak preview">sneak preview</a> of Audition CS6<sup class="reference" id="cite_ref-4"><a href="#cite_note-4">[4]</a></sup> in March 2012 highlighting clip grouping and automatic speech alignment (which had its technology previewed<sup class="reference" id="cite_ref-5"><a href="#cite_note-5">[5]</a></sup> in September 2011). Audition CS6 was released on April 23, 2012, as part of both Creative Suite 6 Master Collection and Creative Suite 6 Production Premium. It included faster and more precise editing, real-time clip stretching, automatic speech alignment, EUCON and Mackie <a href="/wiki/Audio_control_surface" title="Audio control surface">control surface</a> support, parameter automation, more powerful pitch correction, HD video playback, new effects; and more features.</p>
<h3><span class="mw-headline" id="Version_6_.28CC.29">Version 6 (CC)</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Adobe_Audition&amp;action=edit&amp;section=8" title="Edit section: Version 6 (CC)">edit</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>Adobe Audition 6, also more commonly known as Audition CC, was released on 17 June 2013. It is the first in the Audition line to be part of the <a href="/wiki/Adobe_Creative_Cloud" title="Adobe Creative Cloud">Adobe Creative Cloud</a>. Also, Audition CC is now the first 64-bit<sup class="reference" id="cite_ref-6"><a href="#cite_note-6">[6]</a></sup> application in the Audition line. This can provide faster processing time when compared to Audition CS6.<sup class="noprint Inline-Template Template-Fact" style="white-space:nowrap;">[<i><a href="/wiki/Wikipedia:Citation_needed" title="Wikipedia:Citation needed"><span title="This claim needs references to reliable sources. (October 2014)">citation needed</span></a></i>]</sup> New features include sound remover, preview editor and pitch bender.<sup class="reference" id="cite_ref-7"><a href="#cite_note-7">[7]</a></sup><sup class="noprint Inline-Template" style="white-space:nowrap;">[<i><a href="/wiki/Wikipedia:Verifiability" title="Wikipedia:Verifiability"><span title="The material near this tag failed verification of its source citation(s).
