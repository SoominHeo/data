<div class="hatnote" role="note">Not to be confused with <a href="/wiki/Janson" title="Janson">Janson</a>.</div>
<p><b>Adobe Jenson</b> is an <a href="/wiki/Serif#Old-style" title="Serif">old-style</a> <a href="/wiki/Serif" title="Serif">serif</a> <a href="/wiki/Typeface" title="Typeface">typeface</a> drawn for <a href="/wiki/Adobe_Systems" title="Adobe Systems">Adobe Systems</a> by its chief type designer <a href="/wiki/Robert_Slimbach" title="Robert Slimbach">Robert Slimbach</a>. Its Roman styles are based on a text face cut by <a href="/wiki/Nicolas_Jenson" title="Nicolas Jenson">Nicolas Jenson</a> in Venice around 1470, and its italics are based on those created by <a href="/wiki/Ludovico_Vicentino_degli_Arrighi" title="Ludovico Vicentino degli Arrighi">Ludovico Vicentino degli Arrighi</a> fifty years later.</p>
<p>Jenson is an organic, somewhat idiosyncratic font, with a low <a href="/wiki/X-height" title="X-height">x-height</a>, and inconsistencies that help differentiate letters. It is considered a highly readable typeface appropriate for large amounts of text, and is accordingly often used in book design for body text.</p>

<h2><span class="mw-headline" id="Development">Development</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Adobe_Jenson&amp;action=edit&amp;section=1" title="Edit section: Development">edit</a><span class="mw-editsection-bracket">]</span></span></h2>

<p>Adobe Jenson was first released in 1996 as a <a href="/wiki/Multiple_master_fonts" title="Multiple master fonts">multiple master</a> font.<sup class="reference" id="cite_ref-2015_Award_1-0"><a href="#cite_note-2015_Award-1">[1]</a></sup> It was created using sophisticated interpolation or <a href="/wiki/Multiple_master_fonts" title="Multiple master fonts">multiple master</a> technology, to create a range of weights and <a class="mw-redirect" href="/wiki/Optical_size" title="Optical size">optical sizes</a> suitable for different text sizes.<sup class="reference" id="cite_ref-2"><a href="#cite_note-2">[2]</a></sup> This partial automation of font creation was intended to allow a gradual trend in styles from solid, chunky designs for caption-size small print to more graceful and slender designs for headings.<sup class="reference" id="cite_ref-3"><a href="#cite_note-3">[3]</a></sup><sup class="reference" id="cite_ref-6"><a href="#cite_note-6">[a]</a></sup> It is now sold in the standard <a href="/wiki/OpenType" title="OpenType">OpenType</a> font format under the name Adobe Jenson Pro.<sup class="reference" id="cite_ref-Arno_Pro_specimen_7-0"><a href="#cite_note-Arno_Pro_specimen-7">[6]</a></sup></p>
<h2><span class="mw-headline" id="Adobe_Jenson_Pro">Adobe Jenson Pro</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Adobe_Jenson&amp;action=edit&amp;section=2" title="Edit section: Adobe Jenson Pro">edit</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>Adobe Jenson Pro is an OpenType update of the original family. The font family supports Adobe CE, ISO-Adobe (later Adobe Western 2), dingbat character sets. The family comes with 4 weights each in roman and italic, and 4 optical sizes. Supported OpenType features include Stylistic alternates, ligatures, proportional numbers, old style figures, small caps, subscripts and superscripts, ordinals, and swashes (italic fonts only).</p>

<h2><span class="mw-headline" id="Related_typefaces">Related typefaces</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Adobe_Jenson&amp;action=edit&amp;section=3" title="Edit section: Related typefaces">edit</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>Many other typefaces have been cut based on the work of Jenson. William Morris's <a href="/wiki/Golden_Type" title="Golden Type">Golden Type</a> created the trend in the 1890s; his design is known for its emboldening of Jenson's original design, giving it something of the feel of <a href="/wiki/Blackletter" title="Blackletter">blackletter</a>.<sup class="reference" id="cite_ref-Peterson1991_8-0"><a href="#cite_note-Peterson1991-8">[7]</a></sup> It is named for <i><a class="mw-redirect" href="/wiki/The_Golden_Legend" title="The Golden Legend">The Golden Legend</a></i>, which was intended to be the first book printed using it.<sup class="reference" id="cite_ref-HarveyPress1991_9-0"><a href="#cite_note-HarveyPress1991-9">[8]</a></sup><sup class="reference" id="cite_ref-HarveyPress1991_9-1"><a href="#cite_note-HarveyPress1991-9">[8]</a></sup><sup class="reference" id="cite_ref-Lawson1990_10-0"><a href="#cite_note-Lawson1990-10">[9]</a></sup> Popular since the 1930s, <a href="/wiki/Bruce_Rogers_(typographer)" title="Bruce Rogers (typographer)">Bruce Rogers</a>' <a href="/wiki/Centaur_(typeface)" title="Centaur (typeface)">Centaur</a> is a much more slender revival in the same style. <a href="/wiki/American_Type_Founders" title="American Type Founders">American Type Founders</a>' <a class="mw-redirect" href="/wiki/Cloister_Old_Style" title="Cloister Old Style">Cloister Old Style</a> was created by its design team led by <a href="/wiki/Morris_Fuller_Benton" title="Morris Fuller Benton">Morris Fuller Benton</a> around 1915, during the same period as Centaur.<sup class="reference" id="cite_ref-MyFonts_11-0"><a href="#cite_note-MyFonts-11">[10]</a></sup><sup class="reference" id="cite_ref-LTC_Cloister_12-0"><a href="#cite_note-LTC_Cloister-12">[11]</a></sup> Ludlow created another release with italic under the direction of <a href="/wiki/Ernst_F._Detterer" title="Ernst F. Detterer">Ernst F. Detterer</a> and <a class="mw-redirect" href="/wiki/Robert_Hunter_Middleton" title="Robert Hunter Middleton">Robert Hunter Middleton</a> in the 1920s.<sup class="reference" id="cite_ref-Nicolas_Jenson_SG_13-0"><a href="#cite_note-Nicolas_Jenson_SG-13">[12]</a></sup> <a href="/wiki/American_Type_Founders" title="American Type Founders">American Type Founders</a> also issued a very eccentric Jenson revival inspired by the work of Morris which is little-known today.<sup class="reference" id="cite_ref-LTC_Jenson_14-0"><a href="#cite_note-LTC_Jenson-14">[13]</a></sup> <a href="/wiki/Tobias_Frere-Jones" title="Tobias Frere-Jones">Tobias Frere-Jones</a> created a revival in 1994 named <a href="/wiki/Hightower_Text" title="Hightower Text">Hightower Text</a> that is bundled with some Microsoft software, adding his own italic design.<sup class="reference" id="cite_ref-Hightower_15-0"><a href="#cite_note-Hightower-15">[14]</a></sup></p>
