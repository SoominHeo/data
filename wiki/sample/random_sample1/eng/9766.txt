<p>The <i><b>Airport Narita</b></i> <span style="font-weight: normal">(<span class="t_nihongo_kanji" lang="ja" xml:lang="ja">エアポート成田</span><span class="t_nihongo_comma" style="display:none">,</span> <i>Eapōto Narita</i><sup class="t_nihongo_help noprint"><a href="/wiki/Help:Installing_Japanese_character_sets" title="Help:Installing Japanese character sets"><span class="t_nihongo_icon" style="color: #00e; font: bold 80% sans-serif; text-decoration: none; padding: 0 .1em;">?</span></a></sup>)</span> is a "Rapid" limited-stop <a class="mw-redirect" href="/wiki/Suburban_rail" title="Suburban rail">suburban rail</a> service in <a class="mw-redirect" href="/wiki/Greater_Tokyo" title="Greater Tokyo">Greater Tokyo</a> operated since 1991 by <a href="/wiki/East_Japan_Railway_Company" title="East Japan Railway Company">East Japan Railway Company</a> (JR East). It runs from <a href="/wiki/Kurihama_Station" title="Kurihama Station">Kurihama</a> to <a href="/wiki/Narita_Airport_Terminal_1_Station" title="Narita Airport Terminal 1 Station">Narita Airport Terminal 1</a> via the <a href="/wiki/Yokosuka_Line" title="Yokosuka Line">Yokosuka</a>, <a href="/wiki/S%C5%8Dbu_Line_(Rapid)" title="Sōbu Line (Rapid)">Sōbu</a>, and <a href="/wiki/Narita_Line" title="Narita Line">Narita</a> lines, with a total length of 152.5 km. Only trains heading towards Narita Airport are identified as <i>Airport Narita</i>; those on the return journey towards <a href="/wiki/Tokyo_Station" title="Tokyo Station">Tokyo</a> are unnamed.<sup class="reference" id="cite_ref-tt201104_1-0"><a href="#cite_note-tt201104-1">[1]</a></sup></p>

<p>All <i>Airport Narita</i> services are operated using <a href="/wiki/E217_series" title="E217 series">E217 series</a> <a href="/wiki/Electric_multiple_unit" title="Electric multiple unit">electric multiple unit</a> (EMU) trains.</p>
<h2><span class="mw-headline" id="History">History</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Airport_Narita&amp;action=edit&amp;section=6" title="Edit section: History">edit</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>The <i>Airport Narita</i> rapid services were introduced from 20 April 1991 to supplement the popular <i><a href="/wiki/Narita_Express" title="Narita Express">Narita Express</a></i> limited express services which commenced the previous month.<sup class="reference" id="cite_ref-jrexpressrapid1991_2-0"><a href="#cite_note-jrexpressrapid1991-2">[2]</a></sup> Initially, 16 services ran in each direction daily, using <a href="/wiki/113_series" title="113 series">113 series</a> EMU formations based at Ōfuna and Makuhari depots.<sup class="reference" id="cite_ref-jrexpressrapid1991_2-1"><a href="#cite_note-jrexpressrapid1991-2">[2]</a></sup> Trains including SaRo 124 Green (first class) cars were allocated to these services, as these had expanded luggage storage space.<sup class="reference" id="cite_ref-jrexpressrapid1991_2-2"><a href="#cite_note-jrexpressrapid1991-2">[2]</a></sup></p>
