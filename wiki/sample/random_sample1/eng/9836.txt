<p>The <b>Yen sign</b> (<b>¥</b>) is a <a class="mw-redirect" href="/wiki/Currency_sign" title="Currency sign">currency sign</a> used by the <a href="/wiki/Renminbi" title="Renminbi">Chinese yuan</a> (CNY) and the <a href="/wiki/Japanese_yen" title="Japanese yen">Japanese yen</a> (JPY) <a href="/wiki/Currency" title="Currency">currencies</a>. This <a class="mw-redirect" href="/wiki/Monetary" title="Monetary">monetary</a> symbol resembles a Latin letter Y with a double stroke. The base unit of both currencies shared the same <a class="mw-redirect" href="/wiki/Chinese_character" title="Chinese character">Chinese character</a>/<a href="/wiki/Kanji" title="Kanji">Kanji</a> (<a href="/wiki/Traditional_Chinese_characters" title="Traditional Chinese characters">traditional Chinese</a>: <span lang="zh-Hant" xml:lang="zh-Hant"><a class="extiw" href="https://en.wiktionary.org/wiki/%E5%9C%93" title="wikt:圓">圓</a></span>; <a href="/wiki/Simplified_Chinese_characters" title="Simplified Chinese characters">simplified Chinese</a>: <span lang="zh-Hans" xml:lang="zh-Hans"><a class="extiw" href="https://en.wiktionary.org/wiki/%E5%9C%86" title="wikt:圆">圆</a></span>; Japanese <a href="/wiki/Shinjitai" title="Shinjitai">Shinjitai</a>: <a class="extiw" href="https://en.wiktionary.org/wiki/%E5%86%86" title="wikt:円">円</a>) that means "circle". It is pronounced <i>yuán</i> in <a href="/wiki/Standard_Chinese" title="Standard Chinese">Mandarin Chinese</a> and <i>en</i> in Standard Japanese. In mainland China, the Chinese character is more frequently written in everyday situations using the simpler character <a class="extiw" href="https://en.wiktionary.org/wiki/%E5%85%83" title="wikt:元">元</a>, which has the same pronunciation as the formal financial character 圓 in Mandarin<sup class="reference" id="cite_ref-1"><a href="#cite_note-1">[1]</a></sup> (but not in Japanese and in some <a href="/wiki/Varieties_of_Chinese" title="Varieties of Chinese">Chinese varieties</a>).<sup class="reference" id="cite_ref-2"><a href="#cite_note-2">[2]</a></sup> The symbol is usually placed before the value it represents, for example ¥100.</p>

<h2><span class="mw-headline" id="Code_page_932">Code page 932</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Yen_sign&amp;action=edit&amp;section=1" title="Edit section: Code page 932">edit</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>In the Japanese-language locales of Microsoft operating systems, the yen sign in <a href="/wiki/Code_page_932" title="Code page 932">code page 932</a> character encoding has the same byte value as the <a href="/wiki/Backslash" title="Backslash">backslash</a> in <a href="/wiki/ASCII" title="ASCII">ASCII</a>. It is also used wherever a backslash is used, such as the <a href="/wiki/Path_(computing)" title="Path (computing)">directory separator</a> character and the general <a href="/wiki/Escape_character" title="Escape character">escape character</a>,<sup class="reference" id="cite_ref-3"><a href="#cite_note-3">[3]</a></sup> essentially making it a backslash with the appearance of a yen sign, a peculiarity that stems from <a href="/wiki/JIS_X_0201" title="JIS X 0201">JIS X 0201</a>.</p>
<h2><span class="mw-headline" id="Chinese_IME">Chinese IME</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Yen_sign&amp;action=edit&amp;section=2" title="Edit section: Chinese IME">edit</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>Under Chinese <a href="/wiki/Pinyin" title="Pinyin">Pinyin</a> <a class="mw-redirect" href="/wiki/Input_method_editor" title="Input method editor">IMEs</a> such as those from <a href="/wiki/Microsoft" title="Microsoft">Microsoft</a> or <a class="mw-redirect" href="/wiki/Sogou.com" title="Sogou.com">Sogou.com</a>, typing "$" displays the <a class="mw-redirect" href="/wiki/Full-width" title="Full-width">double-width</a> character "￥", which is different from <a href="/wiki/Half-width_kana" title="Half-width kana">single-width</a> "¥" used in Japanese IMEs.</p>
<h2><span class="mw-headline" id="Unicode">Unicode</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Yen_sign&amp;action=edit&amp;section=3" title="Edit section: Unicode">edit</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>The <a href="/wiki/Unicode" title="Unicode">Unicode</a> code point is <span class="nowrap"><span style="font-family: monospace, monospace;">U+00A5</span></span> <span style="font-size:125%">¥</span> <span class="smallcaps" style="font-variant: small-caps; font-size: smaller;">YEN SIGN</span> (HTML <code>&amp;#165;</code> <b>·</b> <code>&amp;yen;</code>).</p>
<p>Additionally, there is a <a class="mw-redirect" href="/wiki/Full-width" title="Full-width">double-width</a> character (￥) at code point <span class="nowrap"><span style="font-family: monospace, monospace;">U+FFE5</span></span> <span style="font-size:125%">￥</span> <span class="smallcaps" style="font-variant: small-caps; font-size: smaller;">FULLWIDTH YEN SIGN</span> (HTML <code>&amp;#65509;</code> <b>·</b>  In the block "<a class="mw-redirect" href="/wiki/Halfwidth_and_Fullwidth_Forms" title="Halfwidth and Fullwidth Forms">Halfwidth and Fullwidth Forms</a>") for use with wide fonts, especially east Asian fonts.</p>
