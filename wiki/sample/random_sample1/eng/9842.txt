<p><b>Nvidia Ion</b> is a product line of <a href="/wiki/Nvidia" title="Nvidia">Nvidia</a> corporation intended for <a href="/wiki/Motherboard" title="Motherboard">motherboards</a> of low-cost portable computers. It uses <a href="/wiki/Graphics_processing_unit" title="Graphics processing unit">graphics processing units</a> and <a href="/wiki/Chipset" title="Chipset">chipsets</a> intended for small products.</p>

<h2><span class="mw-headline" id="Description">Description</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Nvidia_Ion&amp;action=edit&amp;section=1" title="Edit section: Description">edit</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>Nvidia Ion products included a GeForce 9M (9xxxM) series MCP79MX (with integrated GeForce 9400M G GPU) chipset, DDR3-1066 or DDR2-800 SDRAM, and the <a href="/wiki/Intel_Atom" title="Intel Atom">Intel Atom</a> processor. The original reference platform was based on a <a href="/wiki/Pico-ITXe" title="Pico-ITXe">Pico-ITXe</a> motherboard designed for <a href="/wiki/Netbook" title="Netbook">netbook</a> and <a href="/wiki/Nettop" title="Nettop">nettop</a> devices. In February 2009, <a href="/wiki/Microsoft" title="Microsoft">Microsoft</a> certified the Ion-based platform for <a href="/wiki/Windows_Vista" title="Windows Vista">Windows Vista</a>.<sup class="reference" id="cite_ref-1"><a href="#cite_note-1">[1]</a></sup> The <a href="/wiki/Small_form_factor" title="Small form factor">small form factor</a> Ion-based computers were released in mid-2009.</p>
<p>Ion GPUs are <a class="mw-redirect" href="/wiki/DirectX_10" title="DirectX 10">DirectX 10.0</a> and <a href="/wiki/OpenGL" title="OpenGL">OpenGL 3.3</a> compliant. They also support <a href="/wiki/CUDA" title="CUDA">CUDA</a> and <a href="/wiki/OpenCL" title="OpenCL">OpenCL</a>. They can play <a href="/wiki/1080p" title="1080p">1080p</a> <a href="/wiki/H.264/MPEG-4_AVC" title="H.264/MPEG-4 AVC">H.264</a>, <a href="/wiki/MPEG-2" title="MPEG-2">MPEG-2</a> and <a href="/wiki/VC-1" title="VC-1">VC-1</a> video using <a href="/wiki/VDPAU" title="VDPAU">VDPAU</a> or <a href="/wiki/Nvidia_PureVideo#The_Third_Generation_PureVideo_HD" title="Nvidia PureVideo">PureVideo HD</a>.<sup class="reference" id="cite_ref-2"><a href="#cite_note-2">[2]</a></sup> ION-LE–based systems shared the same basic hardware as ION but lack Vista and DirectX 10 support.<sup class="reference" id="cite_ref-web_3-0"><a href="#cite_note-web-3">[3]</a></sup></p>
<p>Nvidia announced that it would release the Ion platform for the <a href="/wiki/VIA_Technologies" title="VIA Technologies">VIA</a> <a href="/wiki/VIA_Nano" title="VIA Nano">Nano</a> processor some time in Q4 2009.<sup class="reference" id="cite_ref-4"><a href="#cite_note-4">[4]</a></sup></p>

<ul>
<li>Chipset: <a class="mw-redirect" href="/wiki/Comparison_of_Nvidia_graphics_processing_units#GeForce_9M_.289xxxM.29_series" title="Comparison of Nvidia graphics processing units">MCP79MX</a> (with integrated GeForce 9400M GPU)</li>

<li>Memory Interface: DDR3-1066 or DDR2-800</li>
<li>DirectX 10 Support: Yes, No (Ion-LE)</li>
<li>Graphics Cores: 16</li>

<li>Maximum Anti-Aliasing (AA) Sample Rate: 16x</li>
<li>RAMDACs: 300 MHz</li>
<li>Maximum High-Dynamic Range (HDR) Precision: 128-bit</li>
<li>Maximum Analog Resolution: 2048 x 1536</li>
<li>Maximum Digital Resolution: 2560 x 1600</li>
<li>Graphics APIs: DirectX 10.0, OpenGL 3.3</li>
<li>Full HD decode (1080i/p): Yes, <a href="/wiki/Nvidia_PureVideo" title="Nvidia PureVideo">3rd Generation PureVideo</a></li>
<li>Display Options: HDMI, Dual-link DVI, DP, or VGA (any 2)</li>
<li>HDMI Version: 1.8 (according to the manual of the zotac ION itx-f board)</li>
<li>PCI-Express 2.0: 20 lanes (1×16 and 4×1)</li>
<li>SATA Drives: 6</li>
<li>SATA Speed: 3 Gbit/s</li>
<li>RAID: 0, 1</li>
<li>Networking: 10/100/1000 BASE-T</li>
<li>USB Ports: 12/2C</li>
<li>PCI Slots: 5</li>
<li>Audio: HDA (Azalia)</li>
</ul>
