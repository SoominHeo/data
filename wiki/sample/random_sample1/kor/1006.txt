<h1 class="firstHeading" id="firstHeading" lang="ko">F (동음이의)</h1>
<div class="mw-body-content" id="bodyContent">
<div id="siteSub">위키백과, 우리 모두의 백과사전.</div>
<div id="contentSub"></div>
<div class="mw-jump" id="jump-to-nav">
					이동:					<a href="#mw-head">둘러보기</a>, 					<a href="#p-search">검색</a>
</div>
<div class="mw-content-ltr" dir="ltr" id="mw-content-text" lang="ko"><script>function mfTempOpenSection(id){var block=document.getElementById("mf-section-"+id);block.className+=" open-block";block.previousSibling.className+=" open-block";}</script><p><b>F</b>는 다음을 가리키는 말이다.</p>
<ul>
<li><a href="/wiki/%EA%B5%90%EC%9C%A1" title="교육">교육</a>에서 <b>F</b>는 학점으로, 실패를 뜻한다.</li>
<li><a href="/wiki/%EB%8B%AC%EB%A0%A5" title="달력">달력</a>에서 <b>F</b>는 <a href="/wiki/%EA%B8%88%EC%9A%94%EC%9D%BC" title="금요일">금요일</a> (<b>F</b>riday)를 뜻한다.</li>
<li><a class="mw-redirect" href="/wiki/16%EC%A7%84%EB%B2%95" title="16진법">16진법</a>이상의 진법에서 F는 <a href="/wiki/15" title="15">15</a>이다.</li>
<li>°F는 <a href="/wiki/%ED%99%94%EC%94%A8" title="화씨">화씨</a> 온도임을 나타낸다.</li>
<li><a href="/wiki/%EC%BB%B4%ED%93%A8%ED%84%B0" title="컴퓨터">컴퓨터</a>에서 F#은 닷넷용 프로그래밍 언어의 이름이다. (<a href="/wiki/F_%EC%83%A4%ED%94%84" title="F 샤프">F#</a>)</li>
<li><a href="/wiki/%ED%95%9C%EC%9E%90_%EB%AC%B8%ED%99%94%EA%B6%8C" title="한자 문화권">한자 문화권</a>에서 <a href="/wiki/%EC%88%AB%EC%9E%90" title="숫자">숫자</a> <a href="/wiki/4" title="4">4</a>를 대신해서 쓰이는 문자이다. (<a class="mw-redirect" href="/wiki/4%EC%9E%90_%EA%B8%B0%ED%94%BC" title="4자 기피">4자 기피</a> 참고.)</li>
<li><a href="/wiki/%ED%94%8C%EB%A3%A8%EC%98%A4%EB%A6%B0" title="플루오린">플루오린</a>(<span lang="en" xml:lang="en">Fluorine</span>, 또는 플루오르(<span style="font-size: smaller;"><a href="/wiki/%EB%8F%85%EC%9D%BC%EC%96%B4" title="독일어">독일어</a>:</span> <span lang="de" xml:lang="de">Fluor</span>), 불소(弗素))의 원소기호</li>
<li><a href="/wiki/%EC%A0%84%ED%88%AC%EA%B8%B0" title="전투기">전투기</a>에서 F는 <b>F</b>ighter 또 는 <b>f</b>ighting plane으로 번호 앞에 F- 를 붙인다.</li>
<li><a href="/wiki/%EC%9D%8C%EC%95%85" title="음악">음악</a>에서 F는 <a href="/wiki/%EC%9D%8C%EC%9D%B4%EB%A6%84" title="음이름">음이름</a>이다. F장조의 으뜸음이기도 하며 소문자 f는 <a href="/wiki/%ED%8F%AC%EB%A5%B4%ED%85%8C" title="포르테">포르테</a>를 뜻한다. 즉, <a class="mw-disambig" href="/wiki/%EB%A9%94%EC%A1%B0%ED%8F%AC%EB%A5%B4%ED%85%8C" title="메조포르테">메조포르테</a>는 mf, <a href="/wiki/%ED%8F%AC%EB%A5%B4%ED%85%8C" title="포르테">포르테</a>는 f, <a class="mw-redirect" href="/wiki/%ED%8F%AC%EB%A5%B4%ED%8B%B0%EC%8B%9C%EB%AA%A8" title="포르티시모">포르티시모</a>는 ff, <a class="mw-redirect" href="/wiki/%ED%8F%AC%EB%A5%B4%ED%8B%B0%EC%8B%9C%EC%8B%9C%EB%AA%A8" title="포르티시시모">포르티시시모</a>는 fff가 된다.</li>
</ul>
