<p><b>주요 8개국 정상회담</b>(통칭 <b>G8</b>, <b>지에이트</b>, Group of Eight)은 세계 정치와 경제를 주도하는 주요 8개국(<a href="/wiki/%EB%AF%B8%EA%B5%AD" title="미국">미국</a>, <a href="/wiki/%EB%9F%AC%EC%8B%9C%EC%95%84" title="러시아">러시아</a>, <a href="/wiki/%EC%98%81%EA%B5%AD" title="영국">영국</a>, <a href="/wiki/%EC%BA%90%EB%82%98%EB%8B%A4" title="캐나다">캐나다</a>, <a href="/wiki/%ED%94%84%EB%9E%91%EC%8A%A4" title="프랑스">프랑스</a>, <a href="/wiki/%EB%8F%85%EC%9D%BC" title="독일">독일</a>, <a href="/wiki/%EC%9D%B4%ED%83%88%EB%A6%AC%EC%95%84" title="이탈리아">이탈리아</a>, <a href="/wiki/%EC%9D%BC%EB%B3%B8" title="일본">일본</a>)의 모임이다. 매년 정상들이 모여 정치와 경제 문제에 대한 회의를 연다.</p>
<p><a href="/wiki/1975%EB%85%84" title="1975년">1975년</a> <a href="/wiki/%ED%94%84%EB%9E%91%EC%8A%A4" title="프랑스">프랑스</a>가 G6 정상회의를 창설했다. <a href="/wiki/%EB%AF%B8%EA%B5%AD" title="미국">미국</a>, <a href="/wiki/%EC%9D%BC%EB%B3%B8" title="일본">일본</a>, <a href="/wiki/%ED%94%84%EB%9E%91%EC%8A%A4" title="프랑스">프랑스</a>, <a href="/wiki/%EC%84%9C%EB%8F%85" title="서독">서독</a>, <a href="/wiki/%EC%98%81%EA%B5%AD" title="영국">영국</a>, <a href="/wiki/%EC%9D%B4%ED%83%88%EB%A6%AC%EC%95%84" title="이탈리아">이탈리아</a>, 등 서방 선진 6개국의 모임으로 출범하였으며 그 다음해 <a href="/wiki/%EC%BA%90%EB%82%98%EB%8B%A4" title="캐나다">캐나다</a>가 추가되어 <b>서방 선진 7개국 정상회담</b>(G7)으로 매년마다 개최되었다. <a href="/wiki/1990%EB%85%84%EB%8C%80" title="1990년대">1990년대</a> 이후 냉전 구도 해체로 세계에서 가장 큰 나라인 <a href="/wiki/%EB%9F%AC%EC%8B%9C%EC%95%84" title="러시아">러시아</a>가 옵서버 형식으로 참가하기 시작하였고 <a href="/wiki/1997%EB%85%84" title="1997년">1997년</a> 이후 러시아가 정식 멤버가 되면서 세계 주요 8개국의 모임으로 불리고 있다.</p>
<p>모임의 의장은 각 나라가 해마다 돌아가면서 맡으며 의장국에서 정상회담을 개최한다.</p>

<h2><span class="mw-headline" id=".EC.97.AD.EC.82.AC">역사</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=G8&amp;action=edit&amp;section=1" title="부분 편집: 역사">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>G8의 모임은 <a href="/wiki/1973%EB%85%84" title="1973년">1973년</a> <a class="mw-redirect" href="/wiki/%EC%A0%9C1%EC%B0%A8_%EC%84%9D%EC%9C%A0_%ED%8C%8C%EB%8F%99" title="제1차 석유 파동">제1차 석유 파동</a>과 그 여파에 따른 세계적인 불경기에서 시작되었다. 1973년 겨울 미국 <a class="mw-redirect" href="/wiki/%EC%A1%B0%EC%A7%80_%EC%8A%90%EC%B8%A0" title="조지 슐츠">조지 슐츠</a> 재무장관은 훗날 G6 국가가 되는 나라의 재무장관들을 초청하여 <a href="/wiki/%EB%B0%B1%EC%95%85%EA%B4%80" title="백악관">백악관</a> 내 벽난로 옆에서 사교모임을 가졌다. 이듬해에도 이들은 같은 장소에서 회동을 가졌는데, 멤버 중 자국의 수장이 된 <a href="/wiki/%ED%94%84%EB%9E%91%EC%8A%A4%EC%9D%98_%EB%8C%80%ED%86%B5%EB%A0%B9" title="프랑스의 대통령">프랑스 대통령</a> <a class="mw-redirect" href="/wiki/%EB%B0%9C%EB%A0%88%EB%A6%AC_%EC%A7%80%EC%8A%A4%EC%B9%B4%EB%A5%B4%EB%8D%B0%EC%8A%A4%ED%83%B1" title="발레리 지스카르데스탱">발레리 지스카르데스탱</a>과 <a href="/wiki/%EB%8F%85%EC%9D%BC" title="독일">독일</a>의 <a href="/wiki/%ED%97%AC%EB%AC%B4%ED%8A%B8_%EC%8A%88%EB%AF%B8%ED%8A%B8" title="헬무트 슈미트">헬무트 슈미트</a>가 이 분위기를 잊지 못하여 <a href="/wiki/1975%EB%85%84" title="1975년">1975년</a> 정기적인 모임을 갖자고 제안하였다.<sup class="reference" id="cite_ref-donga_1-0"><a href="#cite_note-donga-1">[1]</a></sup> 참가국들은 이 제안에 동의하고 의장국을 돌아가면서 맡으며 해마다 모임을 갖기로 하였다. <a href="/wiki/%EC%A0%9C1%ED%9A%8C_G6_%EC%A0%95%EC%83%81%ED%9A%8C%EB%8B%B4" title="제1회 G6 정상회담">최초 모임</a>은 1975년 프랑스 <a href="/wiki/%EB%9E%91%EB%B6%80%EC%98%88" title="랑부예">랑부예</a>에서 있었고 회의명은 서방 선진 6개국을 상징하는 <b>G6</b>(Group of Six)로 표기했으며, 창설 멤버는 <a href="/wiki/%EB%AF%B8%EA%B5%AD" title="미국">미국</a>, <a href="/wiki/%EC%84%9C%EB%8F%85" title="서독">서독</a>, <a href="/wiki/%EC%98%81%EA%B5%AD" title="영국">영국</a>, <a href="/wiki/%EC%9D%B4%ED%83%88%EB%A6%AC%EC%95%84" title="이탈리아">이탈리아</a>, <a href="/wiki/%EC%9D%BC%EB%B3%B8" title="일본">일본</a>, <a href="/wiki/%ED%94%84%EB%9E%91%EC%8A%A4" title="프랑스">프랑스</a>였다. 다음 해인 <a href="/wiki/1976%EB%85%84" title="1976년">1976년</a> <a href="/wiki/%ED%91%B8%EC%97%90%EB%A5%B4%ED%86%A0%EB%A6%AC%EC%BD%94" title="푸에르토리코">푸에르토리코</a>에서 열린 모임에는 <a href="/wiki/%EB%AF%B8%EA%B5%AD%EC%9D%98_%EB%8C%80%ED%86%B5%EB%A0%B9" title="미국의 대통령">미국 대통령</a> <a href="/wiki/%EC%A0%9C%EB%9F%B4%EB%93%9C_%ED%8F%AC%EB%93%9C" title="제럴드 포드">제럴드 포드</a>의 초청으로 <a href="/wiki/%EC%BA%90%EB%82%98%EB%8B%A4" title="캐나다">캐나다</a>가 참여하면서 <b>G7</b>(Group of Seven)으로 변경되었다.</p>
<p>처음에는 <a href="/wiki/%EB%8D%B0%ED%83%95%ED%8A%B8" title="데탕트">데탕트</a> 분위기 아래 공산권과 소련을 자극하지 않기 위해 경제문제만을 다뤘지만<sup class="reference" id="cite_ref-2"><a href="#cite_note-2">[2]</a></sup> <a class="mw-redirect" href="/wiki/%EC%86%8C%EB%A0%A8%EC%9D%98_%EC%95%84%ED%94%84%EA%B0%80%EB%8B%88%EC%8A%A4%ED%83%84_%EC%B9%A8%EA%B3%B5" title="소련의 아프가니스탄 침공">소련-아프가니스탄 전쟁</a>이 발발하자 1980년 이탈리아 회담에서 서방세계가 단결하여 소련에 맞설 것을 논의했는데, 이것이 계기가 되어 경제 외에도 외교 및 정치로 의제 분야가 확장되었다. 이로부터 초기의 사교적 분위기는 사라지고 사무적인 모임으로 바뀌었다.<sup class="reference" id="cite_ref-donga_1-1"><a href="#cite_note-donga-1">[1]</a></sup></p>
<p><a href="/wiki/1991%EB%85%84" title="1991년">1991년</a> <a href="/wiki/%EB%83%89%EC%A0%84" title="냉전">냉전</a> 구도가 해소되면서 <a href="/wiki/%EC%86%8C%EB%A0%A8" title="소련">소련</a>(현재의 <a href="/wiki/%EB%9F%AC%EC%8B%9C%EC%95%84" title="러시아">러시아</a>)이 G7 회의 이후에 열린 모임에 참여하기 시작하였다. <a href="/wiki/1994%EB%85%84" title="1994년">1994년</a> <a href="/wiki/%EC%9D%B4%ED%83%88%EB%A6%AC%EC%95%84" title="이탈리아">이탈리아</a> <a href="/wiki/%EB%82%98%ED%8F%B4%EB%A6%AC" title="나폴리">나폴리</a> 회의부터는 이 모임을 'P8(Political 8)'또는 'G7+1' 모임이라고 불렀다. 러시아는 <a href="/wiki/1997%EB%85%84" title="1997년">1997년</a> <a href="/wiki/%EB%AF%B8%EA%B5%AD" title="미국">미국</a> <a href="/wiki/%EB%8D%B4%EB%B2%84" title="덴버">덴버</a> 회의에서 정식으로 참가하면서 G8이 결성되었다. 냉전 구도가 해체되어 서방이라는 말의 의미가 모호해졌고, 러시아는 경제적으로 선진국이라 보기 어려우므로 이후부터는 서방 선진국 정상회담이라는 명칭은 쓰지 않으며 러시아의 경제 사정 때문에 재무 장관 회의에는 러시아가 배제된다. 오늘날 G7은 이 재무 장관 회담을 가리키는 용어로 쓰이기도 한다.</p>
<p>공식회원국가 외에 <a href="/wiki/%EC%9C%A0%EB%9F%BD_%EC%9C%84%EC%9B%90%ED%9A%8C" title="유럽 위원회">유럽 위원회</a> 의장이 유럽 연합을 대표하여 G7 시절부터 매년 참가하고 있으며 최근에는 <a href="/wiki/%EB%B8%8C%EB%9D%BC%EC%A7%88" title="브라질">브라질</a>, <a href="/wiki/%EC%9D%B8%EB%8F%84" title="인도">인도</a>, <a href="/wiki/%EC%A4%91%ED%99%94%EC%9D%B8%EB%AF%BC%EA%B3%B5%ED%99%94%EA%B5%AD" title="중화인민공화국">중화인민공화국</a> 등의 주요 대국이 참관인 자격으로 초청되어 참가하고 있다. <a href="/wiki/2008%EB%85%84" title="2008년">2008년</a> <a href="/wiki/%EC%9D%BC%EB%B3%B8" title="일본">일본</a>에서 열린 회의에서는 <a href="/wiki/%EC%95%84%EC%8B%9C%EC%95%84_%ED%83%9C%ED%8F%89%EC%96%91" title="아시아 태평양">아시아 태평양</a> 지역의 <a href="/wiki/%EB%8C%80%ED%95%9C%EB%AF%BC%EA%B5%AD" title="대한민국">대한민국</a>, <a href="/wiki/%EC%98%A4%EC%8A%A4%ED%8A%B8%EB%A0%88%EC%9D%BC%EB%A6%AC%EC%95%84" title="오스트레일리아">오스트레일리아</a>, <a href="/wiki/%EC%9D%B8%EB%8F%84%EB%84%A4%EC%8B%9C%EC%95%84" title="인도네시아">인도네시아</a>가 초청되었고 <a href="/wiki/2009%EB%85%84" title="2009년">2009년</a> <a href="/wiki/%EC%9D%B4%ED%83%88%EB%A6%AC%EC%95%84" title="이탈리아">이탈리아</a>에서 열린 회의에서는 <a href="/wiki/%EB%84%A4%EB%8D%9C%EB%9E%80%EB%93%9C" title="네덜란드">네덜란드</a>, <a href="/wiki/%EB%8D%B4%EB%A7%88%ED%81%AC" title="덴마크">덴마크</a>, <a href="/wiki/%EC%8A%A4%ED%8E%98%EC%9D%B8" title="스페인">스페인</a> 등이 초청되었다.</p>
<p><a href="/wiki/2014%EB%85%84" title="2014년">2014년</a> <a href="/wiki/3%EC%9B%94_24%EC%9D%BC" title="3월 24일">3월 24일</a> 러시아를 제외한 G8 회원국 정상들은 <a href="/wiki/%EB%84%A4%EB%8D%9C%EB%9E%80%EB%93%9C" title="네덜란드">네덜란드</a> <a href="/wiki/%ED%97%A4%EC%9D%B4%EA%B7%B8" title="헤이그">헤이그</a>에서 열린 임시 G8 정상회의에서 <a class="mw-redirect" href="/wiki/2014%EB%85%84_%ED%81%AC%EB%A6%BC_%EC%9C%84%EA%B8%B0" title="2014년 크림 위기">2014년 크림 위기</a>를 계기로 러시아의 G8 참가 자격을 일시적으로 정지하였다. 이에 따라 회담의 명칭도 러시아 가입 전인 G7으로 복귀하였다.</p>

<h2><span class="mw-headline" id=".EB.B9.84.ED.8C.90">비판</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=G8&amp;action=edit&amp;section=3" title="부분 편집: 비판">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>연례 경제 정상회담을 목적으로 시작되었으나, 대부분의 경우 뚜렷한 성과 없이 서로의 입장차만 확인한 채 끝났으며, 주요국들의 친목을 위한 연례 정상회담으로 끝난 경우가 많았다. 이에 따라 이 회담에 대한 무용론도 자주 제기되고 있다. 또한 <a href="/wiki/1980%EB%85%84%EB%8C%80" title="1980년대">1980년대</a>에 이미 <a href="/wiki/%EC%8A%A4%ED%8E%98%EC%9D%B8" title="스페인">스페인</a>의 경제규모가 <a href="/wiki/%EC%BA%90%EB%82%98%EB%8B%A4" title="캐나다">캐나다</a>를 앞섰는데도 스페인이 참가하지 못하고 캐나다가 계속 참가하는 것에 대한 논란이 있었다. 이후 세계 각국의 경제 규모 순위에 변동이 있었고, 특히 2010년 이후 <a href="/wiki/%EC%A4%91%ED%99%94%EC%9D%B8%EB%AF%BC%EA%B3%B5%ED%99%94%EA%B5%AD" title="중화인민공화국">중화인민공화국</a>이 미국에 이어 세계 2위 규모의 경제대국이 되었음에도<sup class="reference" id="cite_ref-3"><a href="#cite_note-3">[3]</a></sup> 정식으로 참가하지 못하고 있는 등 회원국의 자격 논란도 있다. 2012년 <a href="/wiki/%EB%B8%94%EB%9D%BC%EB%94%94%EB%AF%B8%EB%A5%B4_%ED%91%B8%ED%8B%B4" title="블라디미르 푸틴">블라디미르 푸틴</a> 러시아 대통령이 <a class="mw-redirect" href="/wiki/%EC%BA%A0%ED%94%84_%EB%8D%B0%EC%9D%B4%EB%B9%84%EB%93%9C" title="캠프 데이비드">캠프 데이비드</a> 모임에 불참하자 'G8 회담은 더 이상 필요없다.'라는 무용론이 제기되기도 했다.<sup class="reference" id="cite_ref-4"><a href="#cite_note-4">[4]</a></sup> 이 회의는 <a href="/wiki/%EC%84%B8%EA%B3%84%ED%99%94" title="세계화">세계화</a>에 반대하는 시위대들의 표적이 되었으며, 종종 테러의 표적이 되기도 했다.</p>
