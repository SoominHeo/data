<p><b>GOG.com</b> (보통 <b>Good Old Games</b>으로 불림)은 <a href="/wiki/PC_%EA%B2%8C%EC%9E%84" title="PC 게임">PC 게임</a> 판매 및 배포서비스를 하는 회사로, 폴란드의 CD Projekt RED를 자회사로 하며, 현재 <a href="/wiki/%ED%82%A4%ED%94%84%EB%A1%9C%EC%8A%A4" title="키프로스">키프로스</a> <a href="/wiki/%EB%9D%BC%EB%A5%B4%EB%82%98%EC%B9%B4" title="라르나카">라르나카</a>에 적을 두고 있다.<sup class="reference" id="cite_ref-1"><a href="#cite_note-1">[1]</a></sup> GOG.com은 <a href="/wiki/%EB%94%94%EC%A7%80%ED%84%B8_%EA%B6%8C%EB%A6%AC_%EA%B4%80%EB%A6%AC" title="디지털 권리 관리">디지털 권리 관리</a>가 없는 플랫폼으로, <a class="mw-redirect" href="/wiki/%EB%A7%88%EC%9D%B4%ED%81%AC%EB%A1%9C%EC%86%8C%ED%94%84%ED%8A%B8_%EC%9C%88%EB%8F%84" title="마이크로소프트 윈도">마이크로소프트 윈도</a>와 <a class="mw-redirect" href="/wiki/OS_X" title="OS X">OS X</a>등에 고전게임을 서비스 하고 있다. 2012년 3월들어 GOG에서는 더 위처 2, <a href="/wiki/%EC%96%B4%EC%8C%94%EC%8B%A0_%ED%81%AC%EB%A6%AC%EB%93%9C" title="어쌔신 크리드">어쌔신 크리드</a>, <a href="/wiki/%EC%95%A8%EB%9F%B0_%EC%9B%A8%EC%9D%B4%ED%81%AC" title="앨런 웨이크">앨런 웨이크</a>등의 몇몇 최신 게임들도 서비스하기 시작했다.<sup class="reference" id="cite_ref-2"><a href="#cite_note-2">[2]</a></sup><sup class="reference" id="cite_ref-3"><a href="#cite_note-3">[3]</a></sup> 2014년 8월 28일에는 사이트를 새롭게 리뉴얼하면서 영화판매도 시작하였는데, 게임과 마찬가지로 <a href="/wiki/%EB%94%94%EC%A7%80%ED%84%B8_%EA%B6%8C%EB%A6%AC_%EA%B4%80%EB%A6%AC" title="디지털 권리 관리">디지털 권리 관리</a>가 없고, 구입시 보너스 컨텐츠로 포스터 등 각종 혜택이 주어진다.</p>
<h2><span class="mw-headline" id=".ED.8A.B9.EC.A7.95">특징</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=GOG.com&amp;action=edit&amp;section=1" title="부분 편집: 특징">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>GOG.com의 게임들은 온라인에서 지불하고, 다운로드 받을수 있으며, 모든 게임은 <a href="/wiki/%EB%94%94%EC%A7%80%ED%84%B8_%EA%B6%8C%EB%A6%AC_%EA%B4%80%EB%A6%AC" title="디지털 권리 관리">디지털 권리 관리</a>가 포함되어 있지 않는다. 보통 고전게임의 경우 5달러에서 10달러 선에 가격이 결정되며, 종종 할인을 하기도 한다. 최신게임의 경우 고전게임과 달리 다른 가격대를 유지하고 있다.</p>
<p>모든 GOG.com의 게임들은 최신 <a class="mw-redirect" href="/wiki/%EB%A7%88%EC%9D%B4%ED%81%AC%EB%A1%9C%EC%86%8C%ED%94%84%ED%8A%B8_%EC%9C%88%EB%8F%84" title="마이크로소프트 윈도">마이크로소프트 윈도</a>에 호환성을 가지며, 호환성을 가지지 못하는 몇몇 게임들의 경우 프리패치를 지원하거나, <a class="mw-redirect" href="/wiki/ScummVM" title="ScummVM">ScummVM</a>이나 <a href="/wiki/%EB%8F%84%EC%8A%A4%EB%B0%95%EC%8A%A4" title="도스박스">도스박스</a>를 포함하기도 한다. 위에서도 설명한것과 같이 GOG.com은 사이트 사용자들에게 <a href="/wiki/%EB%94%94%EC%A7%80%ED%84%B8_%EA%B6%8C%EB%A6%AC_%EA%B4%80%EB%A6%AC" title="디지털 권리 관리">디지털 권리 관리</a>를 강제하지 않으며<sup class="reference" id="cite_ref-arsinterview_4-0"><a href="#cite_note-arsinterview-4">[4]</a></sup>, 또한 별도의 다운로드 매니저가 있지만, 사용자들에게 다운로드나 실행시 특별한 클라이언트 소프트웨어를 강제하지 않는다.<sup class="reference" id="cite_ref-arspreview_5-0"><a href="#cite_note-arspreview-5">[5]</a></sup></p>
<p>또한 몇몇 게임의 경우, 게임에 관련된 여러 멀티미디어 자료(게임의 <a href="/wiki/%EC%82%AC%EC%9A%B4%EB%93%9C%ED%8A%B8%EB%9E%99" title="사운드트랙">사운드트랙</a>, <a href="/wiki/%EB%B0%B0%EA%B2%BD_%ED%99%94%EB%A9%B4" title="배경 화면">배경 화면</a>이미지, <a href="/wiki/%EC%95%84%EB%B0%94%ED%83%80" title="아바타">아바타</a>, 사용설명서등)을 포함하여 주기도 한다. 또한 구매한 항목에 대한 고객 지원을 제공하고 있다.</p>
