<p><b>GP32</b>(Game Park 32bit)는 <a href="/wiki/%EB%8C%80%ED%95%9C%EB%AF%BC%EA%B5%AD" title="대한민국">대한민국</a> 최초의 휴대용 게임기이다. 개발사인 게임파크는 <a href="/wiki/%EA%B2%8C%EC%9E%84%ED%8C%8C%ED%81%AC" title="게임파크">게임파크</a>와 <a href="/wiki/%EA%B2%8C%EC%9E%84%ED%8C%8C%ED%81%AC_%ED%99%80%EB%94%A9%EC%8A%A4" title="게임파크 홀딩스">게임파크 홀딩스</a>로 나뉘었다. 게임파크는 후속작으로 <a href="/wiki/XGP" title="XGP">XGP</a>를 개발하고 있으며, 게임파크 홀딩스는 후속작으로 <a href="/wiki/GP2X" title="GP2X">GP2X</a> 두 종류를 내놓았다.</p>

<h2><span class="mw-headline" id=".EA.B0.9C.EC.9A.94">개요</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=GP32&amp;action=edit&amp;section=1" title="부분 편집: 개요">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>2001년 <a href="/wiki/%EA%B2%8C%EC%9E%84%ED%8C%8C%ED%81%AC" title="게임파크">게임파크</a>에서 개발하고 대한민국에서 정식 발매하였다. 당시 휴대용 게임기로써는 최고의 성능으로 주목을 받았다. 게다가 SMC를 저장매체로 채택, 오픈소스를 채택해 <a href="/wiki/USB" title="USB">USB</a>포트를 이용해 어느 누구라도 GP32 프로그램과 게임등을 개발하고 다운 받을 수 있는 장점도 있었다. 하지만 발매 초반부터 서드파티 부족, 홍보 부족으로 점차 게임기로써의 모습을 잃어갔지만 외국, 특히 유럽에서 일부 매니아들에게 폭발적으로 인기를 얻기도 했다. <a href="/wiki/2003%EB%85%84" title="2003년">2003년</a>에 게임파크는 유럽에 GP32를 정식으로 수출하기도 했다.</p>
<p>2004년에 게임파크는 GP32 BLU를 발매했다. 기존에 하호테크에서 주문 방식으로 제작했던 FLU보다 성능이 좋아진 제품이다.</p>
<p>GP32는 2004년에 발매된 전용게임 <a class="new" href="/w/index.php?title=%EB%A7%88%EB%B2%95%EC%82%AC%EA%B0%80_%EB%90%98%EA%B3%A0_%EC%8B%B6%EC%96%B4!&amp;action=edit&amp;redlink=1" title="마법사가 되고 싶어! (없는 문서)">마법사가 되고 싶어!</a>를 끝으로 게임기로써의 수명을 다했다.</p>
<p>2009년 4월 현재 제작사인 <a href="/wiki/%EA%B2%8C%EC%9E%84%ED%8C%8C%ED%81%AC" title="게임파크">게임파크</a>는 <a class="new" href="/w/index.php?title=%EC%B0%A8%EC%84%B8%EB%8C%80_%ED%9C%B4%EB%8C%80%EC%9A%A9_%EA%B2%8C%EC%9E%84%EA%B8%B0_%ED%94%8C%EB%9E%AB%ED%8F%BC_%EA%B0%9C%EB%B0%9C_%EA%B5%AD%EC%B1%85_%EC%82%AC%EC%97%85&amp;action=edit&amp;redlink=1" title="차세대 휴대용 게임기 플랫폼 개발 국책 사업 (없는 문서)">차세대 휴대용 게임기 플랫폼 개발 국책 사업</a>에 선정이 되어 <a href="/wiki/TU%EB%AF%B8%EB%94%94%EC%96%B4" title="TU미디어">TU미디어</a>와 함께 <a class="new" href="/w/index.php?title=NXGP&amp;action=edit&amp;redlink=1" title="NXGP (없는 문서)">nXGP</a>를 개발중이다.</p>
<p>또한, 2009년 4월 13일 <a href="/wiki/GP2X_%EC%9C%84%EC%A6%88" title="GP2X 위즈">GP2X 위즈</a>가 발매되었다.</p>
