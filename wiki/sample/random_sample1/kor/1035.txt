<p><b>GT카</b>(<span style="font-size: smaller;"><a href="/wiki/%EC%98%81%EC%96%B4" title="영어">영어</a>:</span> <span lang="en" xml:lang="en">Grand Tourer</span> <small><b>그랜드 투어러</b><sup>[<a href="/wiki/%EC%9C%84%ED%82%A4%EB%B0%B1%EA%B3%BC:%EC%98%81%EC%96%B4%EC%9D%98_%ED%95%9C%EA%B8%80_%ED%91%9C%EA%B8%B0" title="위키백과:영어의 한글 표기">*</a>]</sup></small>, <span style="font-size: smaller;"><a href="/wiki/%EC%9D%B4%ED%83%88%EB%A6%AC%EC%95%84%EC%96%B4" title="이탈리아어">이탈리아어</a>:</span> <span lang="it" xml:lang="it">Gran Turismo</span> <small><b>그란 투리스모</b><sup>[<a href="/wiki/%EC%9C%84%ED%82%A4%EB%B0%B1%EA%B3%BC:%EC%9D%B4%ED%83%88%EB%A6%AC%EC%95%84%EC%96%B4%EC%9D%98_%ED%95%9C%EA%B8%80_%ED%91%9C%EA%B8%B0" title="위키백과:이탈리아어의 한글 표기">*</a>]</sup></small>, GT)는 장거리 운전을 목적으로 설계된 고성능의 <a href="/wiki/%EC%9E%90%EB%8F%99%EC%B0%A8" title="자동차">자동차</a>를 가리킨다. 따라서 성능이 뛰어난 어떠한 종류의 자동차도 GT카라고 부를 수 있으나 관례상 대부분 운전석과 조수석 외에 성인이 앉기엔 좁은 플러스 시트(<a class="new" href="/w/index.php?title=2%2B2_%EC%8B%9C%ED%8A%B8&amp;action=edit&amp;redlink=1" title="2+2 시트 (없는 문서)">2+2 시트</a>)를 가진 차를 GT카라고 부른다. 2도어 차체를 구분하는 <a href="/wiki/%EC%BF%A0%ED%8E%98" title="쿠페">쿠페</a>가 아닌 <a class="new" href="/w/index.php?title=4%EB%8F%84%EC%96%B4&amp;action=edit&amp;redlink=1" title="4도어 (없는 문서)">4도어</a> 자동차 중에서도 GT카가 존재한다.</p>
<p><a href="/wiki/%ED%8E%98%EB%9D%BC%EB%A6%AC_599_GTB_%ED%94%BC%EC%98%A4%EB%9D%BC%EB%85%B8" title="페라리 599 GTB 피오라노">페라리 599 GTB 피오라노</a>(Ferrari 599 GTB Fiorano)나 <a href="/wiki/%EB%A9%94%EB%A5%B4%EC%84%B8%EB%8D%B0%EC%8A%A4-%EB%B2%A4%EC%B8%A0_SLR_%EB%A7%A5%EB%9D%BC%EB%A0%8C" title="메르세데스-벤츠 SLR 맥라렌">메르세데스-벤츠 SLR 맥라렌</a>(Mercedes-Benz SLR Mclaren)과 같은 초고성능의 GT카는 <a href="/wiki/%EC%8A%88%ED%8D%BC%EC%B9%B4" title="슈퍼카">슈퍼카</a>로 분류하며, <a href="/wiki/%EC%95%A0%EC%8A%A4%ED%84%B4_%EB%A7%88%ED%8B%B4_DB9" title="애스턴 마틴 DB9">애스턴 마틴 DB9</a>(Aston Martin DB9)이나 <a class="new" href="/w/index.php?title=BMW_6-%EC%8B%9C%EB%A6%AC%EC%A6%88&amp;action=edit&amp;redlink=1" title="BMW 6-시리즈 (없는 문서)">BMW 6-시리즈</a>, <a class="new" href="/w/index.php?title=%EB%A7%88%EC%84%B8%EB%9D%BC%ED%8B%B0_%EA%B7%B8%EB%9E%80%ED%88%AC%EB%A6%AC%EC%8A%A4%EB%AA%A8&amp;action=edit&amp;redlink=1" title="마세라티 그란투리스모 (없는 문서)">마세라티 그란투리스모</a>(Maserati GranTurismo) 등을 엔트리 GT카로 분류한다.</p>
<p><b>GT카</b>는 <a href="/wiki/%EC%8A%A4%ED%8F%AC%EC%B8%A0%EC%B9%B4" title="스포츠카">스포츠카</a>와 따로 구별하는데, <a href="/wiki/%EC%8A%A4%ED%8F%AC%EC%B8%A0%EC%B9%B4" title="스포츠카">스포츠카</a>는 일반적으로 GT카보다 작고(또는 날렵하고), 중량이 더 가벼우며 운전 능력을 향상시킬 목적으로 승차감을 희생하는 경향이 있다. 반면 GT카는 대부분 <a href="/wiki/%EC%97%94%EC%A7%84" title="엔진">엔진</a>을 앞부분에 두어 뒷쪽 트렁크의 화물 공간을 확보해 놓으며, 승차감을 향상시키기 위해 좀 더 부드러운 <a href="/wiki/%ED%98%84%EA%B0%80%EC%9E%A5%EC%B9%98" title="현가장치">현가장치</a>(suspension)를 채택하고 차체도 훨씬 무겁고 둔하다. <b>GT카</b>와 <a href="/wiki/%EC%8A%A4%ED%8F%AC%EC%B8%A0%EC%B9%B4" title="스포츠카">스포츠카</a>와의 유사점이 있는데, 둘 다 <a href="/wiki/%ED%9B%84%EB%A5%9C%EA%B5%AC%EB%8F%99" title="후륜구동">후륜구동</a> 또는 <a href="/wiki/4%EB%A5%9C%EA%B5%AC%EB%8F%99" title="4륜구동">4륜구동</a>을 주로 사용한다는 것이다.</p>
<p>또한 스포츠카는 간단한 개조만으로 곧바로 <a href="/wiki/%EC%9E%90%EB%8F%99%EC%B0%A8_%EA%B2%BD%EC%A3%BC" title="자동차 경주">자동차 경주</a>에 투입될 수 있으나, 반대로 GT카는 엔진과 실내는 물론이고 차체 전체에 걸쳐 복잡한 개조를 해야하기 때문에 곧바로 자동차 경주에 투입될 수 없다는 단점이 있다.</p>
<p>자동차 경주에서의 이러한 단점을 극복하기 위해 GT의 성향을 가진 스포츠카가 만들어졌는데, 이러한 차들을 통칭하는 용어가 Gran Turismo Racing의 약자인 <b>GTR</b> 또는 <b>GT-R</b>이다. 대표적인 GTR 자동차로는 <a class="new" href="/w/index.php?title=%EB%8B%9B%EC%82%B0_R35_GT-R&amp;action=edit&amp;redlink=1" title="닛산 R35 GT-R (없는 문서)">닛산 R35 GT-R</a>(Nissan R35 GT-R), <a class="new" href="/w/index.php?title=%EB%A7%A5%EB%9D%BC%EB%A0%8C_F1_GTR&amp;action=edit&amp;redlink=1" title="맥라렌 F1 GTR (없는 문서)">맥라렌 F1 GTR</a>(Mclaren F1 GTR), <a class="new" href="/w/index.php?title=%EB%A9%94%EB%A5%B4%EC%84%B8%EB%8D%B0%EC%8A%A4-%EB%B2%A4%EC%B8%A0_CLK_GTR&amp;action=edit&amp;redlink=1" title="메르세데스-벤츠 CLK GTR (없는 문서)">메르세데스-벤츠 CLK GTR</a>(Mercedes-Benz CLK GTR)이 있다. 이와 반대로, 경주용으로 만들었던 차를 GT카로 개조한 경우도 있는데, 이 역시 모두 GTR로 통칭한다.</p>
<p><b>GT카의 종류</b></p>

<ul>
<li>GTO (Gran Turismo Omologato) 레이싱 승인을 받은 GT카를 지칭한다. <li>GTV (Gran Turismo Veloce) 고성능의 GT카. 예) <a class="new" href="/w/index.php?title=Alfa_Romeo_GTV6&amp;action=edit&amp;redlink=1" title="Alfa Romeo GTV6 (없는 문서)">Alfa Romeo GTV6</a></li>
<li>GTI or GTi (Gran Turismo Iniezione) <a class="new" href="/w/index.php?title=%EC%A7%81%EB%B6%84%EC%82%AC_%EC%97%94%EC%A7%84&amp;action=edit&amp;redlink=1" title="직분사 엔진 (없는 문서)">직분사 엔진</a>을 사용하는 GT카. [[1961 마세라티 3500 GTI]가 최초의 GTI 자동차이다.</li>
<li>GT/E (Einspritzung) <a href="/wiki/%EB%8F%85%EC%9D%BC" title="독일">독일</a>식 직분사 엔진을 사용한 GT카이다. <a class="new" href="/w/index.php?title=%EC%98%A4%ED%8E%A0_%EB%A7%8C%ED%83%80_GT/E&amp;action=edit&amp;redlink=1" title="오펠 만타 GT/E (없는 문서)">오펠 만타 GT/E</a>가 최초의 GT/E 자동차이다.</li>
<li>GTE (Grand Touring Estate) <a class="new" href="/w/index.php?title=%EC%99%9C%EA%B1%B4&amp;action=edit&amp;redlink=1" title="왜건 (없는 문서)">왜건</a> 스타일의 GT카. 예) <a class="new" href="/w/index.php?title=Reliant_Scimitar_GTE&amp;action=edit&amp;redlink=1" title="Reliant Scimitar GTE (없는 문서)">Reliant Scimitar GTE</a></li>
<li>GTA (Gran Turismo Alleggerita) 기존의 스포츠카를 경량화하거나, 또는 <a class="mw-redirect" href="/wiki/%EC%9E%90%EB%8F%99_%EB%B3%80%EC%86%8D%EA%B8%B0" title="자동 변속기">자동 변속기</a>를 장착한 GT카. 예) <a class="new" href="/w/index.php?title=%EC%95%8C%ED%8C%8C_%EB%A1%9C%EB%AF%B8%EC%98%A4_GTA&amp;action=edit&amp;redlink=1" title="알파 로미오 GTA (없는 문서)">알파 로미오 GTA</a>(경량화 모델), <a class="new" href="/w/index.php?title=%ED%8E%98%EB%9D%BC%EB%A6%AC_456_GTA&amp;action=edit&amp;redlink=1" title="페라리 456 GTA (없는 문서)">페라리 456 GTA</a>(자동 변속기)</li>
<li>GTAm (Gran Turismo Alleggerita Modificata) <a class="new" href="/w/index.php?title=%EC%95%8C%ED%8C%8C_%EB%A1%9C%EB%AF%B8%EC%98%A4_GTAm&amp;action=edit&amp;redlink=1" title="알파 로미오 GTAm (없는 문서)">알파 로미오 GTAm</a>이 최초이며, 기존에 있던 GT카를 경량화시킨 차를 통칭한다. GTA와 다른 종류이다.</li>
<li>GTC (Gran Turismo Compressore/Compact/Cabriolet/Coupe/Crossover) <a class="new" href="/w/index.php?title=%EC%86%8C%ED%98%95%EC%B0%A8&amp;action=edit&amp;redlink=1" title="소형차 (없는 문서)">소형차</a>나 <a class="mw-disambig" href="/wiki/%ED%81%AC%EB%A1%9C%EC%8A%A4%EC%98%A4%EB%B2%84" title="크로스오버">크로스오버</a> 등의 GT카를 통칭한다. <a href="/wiki/%EC%98%A4%ED%8E%A0_%EC%95%84%EC%8A%A4%ED%8A%B8%EB%9D%BC" title="오펠 아스트라">오펠 아스트라</a>(소형차), <a class="new" href="/w/index.php?title=%EB%B2%A4%ED%8B%80%EB%A6%AC_%EC%BB%A8%ED%8B%B0%EB%84%A8%ED%83%88_GTC&amp;action=edit&amp;redlink=1" title="벤틀리 컨티넨탈 GTC (없는 문서)">벤틀리 컨티넨탈 GTC</a>(카브리올렛), <a class="new" href="/w/index.php?title=%ED%8E%98%EB%9D%BC%EB%A6%AC_330_GTC&amp;action=edit&amp;redlink=1" title="페라리 330 GTC (없는 문서)">페라리 330 GTC</a>(쿠페), <a class="new" href="/w/index.php?title=%EC%98%A4%ED%8E%A0_%EC%95%88%ED%83%80%EB%9D%BC_GTC&amp;action=edit&amp;redlink=1" title="오펠 안타라 GTC (없는 문서)">오펠 안타라 GTC</a>(크로스오버)</li>
<li>GTD (Gran Turismo Diesel), <a class="mw-disambig" href="/wiki/%EB%94%94%EC%A0%A4" title="디젤">디젤</a> 엔진을 사용하는 GT카로 <a href="/wiki/%ED%91%B8%EC%A1%B0" title="푸조">푸조</a>와 <a href="/wiki/%ED%8F%AD%EC%8A%A4%EB%B0%94%EA%B2%90" title="폭스바겐">폭스바겐</a>이 사용하는 용어이다. <a href="/wiki/%ED%8F%AD%EC%8A%A4%EB%B0%94%EA%B2%90_%EA%B3%A8%ED%94%84" title="폭스바겐 골프">폭스바겐 골프</a>의 GTD 트림 등이 대표적인 GTD 자동차이다.</li>
<li>GTR or GT-R, (Gran Turismo Racing), 경주용 자동차를 GT카로 개조하거나, GT카를 경주용으로 손쉽게 개조할 수 있게 만든 자동차를 지칭한다. <a class="new" href="/w/index.php?title=%EB%A7%A5%EB%9D%BC%EB%A0%8C_F1_GTR&amp;action=edit&amp;redlink=1" title="맥라렌 F1 GTR (없는 문서)">맥라렌 F1 GTR</a>, <a class="new" href="/w/index.php?title=%EB%A9%94%EB%A5%B4%EC%84%B8%EB%8D%B0%EC%8A%A4-%EB%B2%A4%EC%B8%A0_CLK_GTR&amp;action=edit&amp;redlink=1" title="메르세데스-벤츠 CLK GTR (없는 문서)">메르세데스-벤츠 CLK GTR</a>, <a class="mw-redirect" href="/wiki/%EB%8B%9B%EC%82%B0_GTR" title="닛산 GTR">닛산 GTR</a> 등이 있다.</li>
<li>HGT (High Gran Turismo), <a href="/wiki/%ED%94%BC%EC%95%84%ED%8A%B8" title="피아트">피아트</a>에서 사용하는 용어이다. 스포츠카의 성향이 짙은 GT카를 부른다.</li>
</ul>
