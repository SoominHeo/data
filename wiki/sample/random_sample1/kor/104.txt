<p><a href="/wiki/%EC%BB%B4%ED%93%A8%ED%84%B0_%EA%B5%AC%EC%A1%B0" title="컴퓨터 구조">컴퓨터 구조</a>에서 <b>16비트</b> <a class="new" href="/w/index.php?title=%EC%A0%95%EC%88%98_(%EC%BB%B4%ED%93%A8%ED%84%B0_%EA%B3%BC%ED%95%99)&amp;action=edit&amp;redlink=1" title="정수 (컴퓨터 과학) (없는 문서)">정수</a>, <a href="/wiki/%EB%A9%94%EB%AA%A8%EB%A6%AC_%EC%A3%BC%EC%86%8C" title="메모리 주소">메모리 주소</a>, 다른 데이터 장치들은 16 <a href="/wiki/%EB%B9%84%ED%8A%B8" title="비트">비트</a> 너비의 영역을 갖는다. 또, 16비트 <a href="/wiki/%EC%A4%91%EC%95%99_%EC%B2%98%EB%A6%AC_%EC%9E%A5%EC%B9%98" title="중앙 처리 장치">CPU</a>, <a href="/wiki/%EC%82%B0%EC%88%A0_%EB%85%BC%EB%A6%AC_%EC%9E%A5%EC%B9%98" title="산술 논리 장치">ALU</a> <a href="/wiki/%EC%BB%B4%ED%93%A8%ED%84%B0_%EA%B5%AC%EC%A1%B0" title="컴퓨터 구조">구조</a>는 이러한 크기의 <a href="/wiki/%ED%94%84%EB%A1%9C%EC%84%B8%EC%84%9C_%EB%A0%88%EC%A7%80%EC%8A%A4%ED%84%B0" title="프로세서 레지스터">레지스터</a>, <a href="/wiki/%EC%A3%BC%EC%86%8C_%EB%B2%84%EC%8A%A4" title="주소 버스">주소 버스</a>, <a class="mw-redirect" href="/wiki/%EB%8D%B0%EC%9D%B4%ED%84%B0_%EB%B2%84%EC%8A%A4" title="데이터 버스">데이터 버스</a>에 기반을 두고 있다.</p>
<p>16비트는 또한 <a href="/wiki/%EC%A4%91%EC%95%99_%EC%B2%98%EB%A6%AC_%EC%9E%A5%EC%B9%98" title="중앙 처리 장치">중앙 처리 장치</a>의 버스(BUS)가 16비트 단위로 자료를 전송하는 컴퓨터 세대를 가리키기도 하며 이를 <b>16비트 컴퓨터</b>라고도 부른다.</p>
<h2><span class="mw-headline" id="16_.EB.B9.84.ED.8A.B8_.EA.B5.AC.EC.A1.B0">16 비트 구조</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=16%EB%B9%84%ED%8A%B8&amp;action=edit&amp;section=1" title="부분 편집: 16 비트 구조">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>잘 알려져 있는 16비트 프로세서로는 <a href="/wiki/PDP-11" title="PDP-11">PDP-11</a>, <a href="/wiki/%EC%9D%B8%ED%85%94_8086" title="인텔 8086">인텔 8086</a>, <a href="/wiki/%EC%9D%B8%ED%85%94_80286" title="인텔 80286">인텔 80286</a>, <a class="new" href="/w/index.php?title=WDC_65C816&amp;action=edit&amp;redlink=1" title="WDC 65C816 (없는 문서)">WDC 65C816</a>가 있다. <a href="/wiki/%EC%9D%B8%ED%85%94_8088" title="인텔 8088">인텔 8088</a>은 인텔 8086과 프로그램이 호환되며, 외부 버스가 8비트이지만 16비트 롱, 산술 명령어를 처리할 수 있었다.</p>
<p>16비트 정수는 2<sup>16</sup> (또는 65536)의 고윳값을 저장할 수 있다. unsigned의 경우 이 값은 <a href="/wiki/0" title="0">0</a>부터 65535까지의 정수이다. <a href="/wiki/2%EC%9D%98_%EB%B3%B4%EC%88%98" title="2의 보수">2의 보수</a>를 사용하면 -32768부터 32767까지의 값을 사용할 수 있다. 그러므로 16비트 <a href="/wiki/%EB%A9%94%EB%AA%A8%EB%A6%AC_%EC%A3%BC%EC%86%8C" title="메모리 주소">메모리 주소</a>를 갖춘 프로세서는 <a class="new" href="/w/index.php?title=%EB%B0%94%EC%9D%B4%ED%8A%B8_%EC%96%B4%EB%93%9C%EB%A0%88%EC%8B%B1&amp;action=edit&amp;redlink=1" title="바이트 어드레싱 (없는 문서)">바이트 어드레싱</a>을 할 수 있는 64 <a class="mw-redirect" href="/wiki/KiB" title="KiB">KiB</a>의 메모리에 접근할 수 있다.</p>
<p>16비트 프로세서는 거의 완전히 <a href="/wiki/%EA%B0%9C%EC%9D%B8%EC%9A%A9_%EC%BB%B4%ED%93%A8%ED%84%B0" title="개인용 컴퓨터">개인용 컴퓨터</a> 산업에 들어왔지만 다양한 임베디드 응용으로 남아 있다. 이를테면 16비트 <a class="new" href="/w/index.php?title=XAP_%ED%94%84%EB%A1%9C%EC%84%B8%EC%84%9C&amp;action=edit&amp;redlink=1" title="XAP 프로세서 (없는 문서)">XAP 프로세서</a>는 수많은 ASIC에 쓰인다.</p>
<h4><span class="mw-headline" id="16.2F32.EB.B9.84.ED.8A.B8_.EB.AA.A8.ED.86.A0.EB.A1.9C.EB.9D.BC_68000">16/32비트 모토로라 68000</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=16%EB%B9%84%ED%8A%B8&amp;action=edit&amp;section=2" title="부분 편집: 16/32비트 모토로라 68000">편집</a><span class="mw-editsection-bracket">]</span></span></h4>
<p><a href="/wiki/%EB%AA%A8%ED%86%A0%EB%A1%9C%EB%9D%BC_68000" title="모토로라 68000">모토로라 68000</a>은 이따금씩 16비트라고 부르기도 한다. 왜냐하면 이 데이터 버스가 16비트가 되기 때문이다. 그러나 일반 목적의 레지스터은 <a href="/wiki/32%EB%B9%84%ED%8A%B8" title="32비트">32비트</a>로 처리하므로 68000 소프트웨어는 자연스럽게 32비트를 지원하며 다른 32비트 프로세서와 상위 호환된다. <a class="mw-redirect" href="/wiki/%EC%84%B8%EA%B0%80_%EB%A9%94%EA%B0%80_%EB%93%9C%EB%9D%BC%EC%9D%B4%EB%B8%8C" title="세가 메가 드라이브">세가 메가 드라이브/제네시스</a>의 68000 프로세서는 비디오 게임 시스템에서 매우 많이 광고된 기능이었다. 이러한 광고 덕에 1988-1995 세대의 비디오 게임기는 16비트 세대라 불리기도 했다.</p>
