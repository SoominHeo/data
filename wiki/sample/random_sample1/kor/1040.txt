<p><b>H-IIB</b>는 <a href="/wiki/H-IIA" title="H-IIA">H-IIA</a>를 개량한 <a href="/wiki/%EC%9D%BC%EB%B3%B8" title="일본">일본</a>의 주력 <a href="/wiki/%EC%9A%B0%EC%A3%BC%EB%B0%9C%EC%82%AC%EC%B2%B4" title="우주발사체">우주발사체</a>이다. <a class="mw-redirect" href="/wiki/%EA%B5%AD%EC%A0%9C_%EC%9A%B0%EC%A3%BC_%EC%A0%95%EA%B1%B0%EC%9E%A5" title="국제 우주 정거장">국제우주정거장</a>에 <a class="mw-redirect" href="/wiki/HTV" title="HTV">HTV</a>를 운반하는 <a href="/wiki/%EA%B7%B9%EC%A0%80%EC%98%A8_%EC%97%B0%EB%A3%8C" title="극저온 연료">극저온 액체 연료</a> 로켓이다.</p>
<p>H-IIA가 <a href="/wiki/%EC%A7%80%EA%B5%AC_%EC%A0%80%EA%B6%A4%EB%8F%84" title="지구 저궤도">LEO</a>에 10톤, <a href="/wiki/%EC%A0%95%EC%A7%80_%EC%B2%9C%EC%9D%B4_%EA%B6%A4%EB%8F%84" title="정지 천이 궤도">GTO</a>에 6톤의 화물을 운반할 수 있는데 비해, H-IIB는 LEO에 19톤, GTO에 8톤의 화물을 운반할 수 있다. 국제우주정거장에는 16.5톤의 화물을 나를 수 있다.</p>
<p>H-IIA에서 달라진 점은, 1단 로켓이다. 즉, H-IIA의 1단 액체로켓은 <a href="/wiki/%EC%B6%94%EB%A0%A5" title="추력">추력</a> 112톤의 <a href="/wiki/%EB%AF%B8%EC%93%B0%EB%B9%84%EC%8B%9C_LE-7" title="미쓰비시 LE-7">미쓰비시 LE-7A</a> 엔진 1개를 사용하나, H-IIB는 2개를 사용한다. 그밖에, 2단 로켓이 액체로켓인 추력 14톤의 <a class="new" href="/w/index.php?title=%EB%AF%B8%EC%93%B0%EB%B9%84%EC%8B%9C_LE-5&amp;action=edit&amp;redlink=1" title="미쓰비시 LE-5 (없는 문서)">미쓰비시 LE-5B</a> 엔진 1개인 점, 1단로켓 옆에 추력 235톤인 <a class="new" href="/w/index.php?title=SRB-A&amp;action=edit&amp;redlink=1" title="SRB-A (없는 문서)">SRB-A3</a> <a href="/wiki/%EA%B3%A0%EC%B2%B4_%EB%A1%9C%EC%BC%93_%EB%B6%80%EC%8A%A4%ED%84%B0" title="고체 로켓 부스터">고체 로켓 부스터</a> 4개(총 추력 940톤)를 장착한 것은 H-IIA와 같다 (SRB-A와 SRB-A3의 추력과 비추력은 임무에 맞춰서 변동한다) .</p>

<h2><span class="mw-headline" id=".EB.8B.A4.EB.A5.B8_.EB.A1.9C.EC.BC.93.EA.B3.BC.EC.9D.98_.EB.B9.84.EA.B5.90">다른 로켓과의 비교</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=H-IIB&amp;action=edit&amp;section=1" title="부분 편집: 다른 로켓과의 비교">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<h3><span class="mw-headline" id=".EC.9A.B0.EC.A3.BC.EC.A0.95.EA.B1.B0.EC.9E.A5_.EB.B0.9C.EC.82.AC.EC.B2.B4">우주정거장 발사체</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=H-IIB&amp;action=edit&amp;section=2" title="부분 편집: 우주정거장 발사체">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p><a href="/wiki/%EC%9A%B0%EC%A3%BC%EB%B0%9C%EC%82%AC%EC%B2%B4" title="우주발사체">우주발사체</a>는 보통 소형, 중형, 대형으로 구별하는데, 소형은 정찰위성만 발사가능한 지구저궤도 3톤 운반능력, 중형은 정지궤도 통신중계위성, 유인우주선, 화성 무인탐사선을 발사가능한 8톤 운반능력, 대형은 우주정거장 모듈을 발사할 수 있는 20톤 운반능력을 말한다. 그보다 더 대형은 아폴로 유인 달탐사선을 발사할 수 있는 100톤 운반능력이 있으나, 거의 발사되지 않기에 보통은 언급하지 않는다.</p>
<p>H-IIB는 19톤 운반능력으로서, 유인우주정거장 모듈을 발사할 수 있다. 현재 전 세계에 20톤 운반능력 발사체는 러시아 흐루니체프의 <a href="/wiki/%ED%94%84%EB%A1%9C%ED%86%A4_%EB%A1%9C%EC%BC%93" title="프로톤 로켓">프로톤 로켓</a>, 미국 보잉의 <a href="/wiki/%EB%8D%B8%ED%83%80_IV" title="델타 IV">델타 IV</a>, 유럽우주국의 <a href="/wiki/%EC%95%84%EB%A6%AC%EC%95%88_5" title="아리안 5">아리안 5</a>가 있다. 중국은 <a class="new" href="/w/index.php?title=%EC%B0%BD%EC%A0%95_5%ED%98%B8&amp;action=edit&amp;redlink=1" title="창정 5호 (없는 문서)">창정 5호</a>, 러시아는 <a class="mw-redirect" href="/wiki/%EC%95%99%EA%B0%80%EB%9D%BC_%EB%A1%9C%EC%BC%93" title="앙가라 로켓">앙가라 로켓</a>을 개발중이다.</p>
<h3><span class="mw-headline" id=".EB.82.98.EB.A1.9C.ED.98.B8">나로호</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=H-IIB&amp;action=edit&amp;section=3" title="부분 편집: 나로호">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>H-IIB 1단 로켓은 112톤 추력인 <a href="/wiki/%EB%AF%B8%EC%93%B0%EB%B9%84%EC%8B%9C_LE-7" title="미쓰비시 LE-7">미쓰비시 LE-7A</a> 로켓을 쌍발로 장착해, 224톤 추력을 낸다. 한국의 <a href="/wiki/%EB%82%98%EB%A1%9C%ED%98%B8" title="나로호">나로호</a> 1단 로켓은 단발 200톤 추력 엔진을 사용한다. 소프트웨어만의 조정으로 170톤 추력으로 낮추었으나, 본래 200톤 추력이다. 요컨대, 둘은 1단 추력이 같다. 다만, 나로호는 4개의 대형 부스터가 없다. 환언하면, 4개의 대형 부스터만 장착하면 나로호도 H-IIB와 같이 20톤 운반 능력을 가질 수 있다라고 말해질 것도 있지만, H-IIB의 LE-7A와 나로호의 RD-151에서는, 연료가 달라 (<a class="mw-redirect" href="/wiki/LH2" title="LH2">LH2</a>과 <a href="/wiki/RP-1" title="RP-1">RP-1</a>의 차이), <a href="/wiki/%EC%9A%B0%EC%A3%BC_%EA%B3%B5%EA%B0%84" title="우주 공간">우주 공간</a> 에서의 <a href="/wiki/%EB%B9%84%EC%B6%94%EB%A0%A5" title="비추력">비추력</a>의 큰 차이가 있고 (440초 대 338초), 2단의 추력과 비추력도 큰 차이가 있어 (H-IIB의 LE-5B는 추력 14톤, 비추력 448초, 나로호의 KM은 추력 8톤, 비추력 250초) , 같은 고체 부스터의 경우, 나로호의 LEO투입 능력은 뒤떨어지게 된다.</p>
