<p><b>HD 1461 b</b>는 <a href="/wiki/G%ED%98%95_%EC%A3%BC%EA%B3%84%EC%97%B4%EC%84%B1" title="G형 주계열성">G형 주계열성</a> <a href="/wiki/HD_1461" title="HD 1461">HD 1461</a> 주위를 돌고 있는 <a href="/wiki/%EC%99%B8%EA%B3%84_%ED%96%89%EC%84%B1" title="외계 행성">외계 행성</a>이다. b의 어머니 항성 HD 1461은 태양보다 모든 면에서 약간씩 크고, 뜨겁고, 좀 더 밝은 <a class="mw-redirect" href="/wiki/%EC%A3%BC%EA%B3%84%EC%97%B4%EC%84%B1" title="주계열성">주계열성</a>이다. 이 행성은 지구로부터 약 76광년 떨어진 곳에 있다. b의 최소 질량은 지구의 8.1배이며 원에 가까운 궤도(<a href="/wiki/%EC%9D%B4%EC%8B%AC%EB%A5%A0" title="이심률">이심률</a> 0.04)를 그리면서 어머니 항성으로부터 0.063434 <a class="mw-redirect" href="/wiki/%EC%B2%9C%EB%AC%B8_%EB%8B%A8%EC%9C%84" title="천문 단위">천문 단위</a> 떨어져 돌고 있다. 현재로서는 b가 <a href="/wiki/%EC%B2%9C%EC%99%95%EC%84%B1" title="천왕성">천왕성</a>이나 <a href="/wiki/%ED%95%B4%EC%99%95%EC%84%B1" title="해왕성">해왕성</a>과 같은 <a class="mw-redirect" href="/wiki/%EA%B0%80%EC%8A%A4_%ED%96%89%EC%84%B1" title="가스 행성">가스 행성</a>일지, <a class="mw-redirect" href="/wiki/CoRoT-7_b" title="CoRoT-7 b">CoRoT-7 b</a>와 같은 <a href="/wiki/%EC%8A%88%ED%8D%BC%EC%A7%80%EA%B5%AC" title="슈퍼지구">슈퍼지구</a>일지 정체가 정확히 밝혀지지 않았다. 2009년 12월 14일 켁 천문대와 앵글로-오스트레일리안 천문대에서 <a class="mw-redirect" href="/wiki/%EC%8B%9C%EC%84%A0%EC%86%8D%EB%8F%84%EB%B2%95" title="시선속도법">시선속도법</a>을 이용하여 발견했다.<sup class="reference" id="cite_ref-disco-rivera_1-1"><a href="#cite_note-disco-rivera-1">[1]</a></sup><sup class="reference" id="cite_ref-ucsc-news_2-0"><a href="#cite_note-ucsc-news-2">[2]</a></sup></p>
