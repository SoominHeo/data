<p><b>Hail to the King</b>은 <a href="/wiki/%EB%AF%B8%EA%B5%AD" title="미국">미국</a>의 <a href="/wiki/%ED%97%A4%EB%B9%84_%EB%A9%94%ED%83%88" title="헤비 메탈">헤비 메탈</a>밴드인 <a href="/wiki/%EC%96%B4%EB%B2%A4%EC%A7%80%EB%93%9C_%EC%84%B8%EB%B8%90%ED%8F%B4%EB%93%9C" title="어벤지드 세븐폴드">어벤지드 세븐폴드</a>의 여섯 번째 정규 앨범이다. <a href="/wiki/%EB%89%B4%EC%A7%88%EB%9E%9C%EB%93%9C" title="뉴질랜드">뉴질랜드</a>와 <a href="/wiki/%EC%98%A4%EC%8A%A4%ED%8A%B8%EB%A0%88%EC%9D%BC%EB%A6%AC%EC%95%84" title="오스트레일리아">오스트레일리아</a>에선 <a href="/wiki/2013%EB%85%84" title="2013년">2013년</a> <a href="/wiki/8%EC%9B%94_23%EC%9D%BC" title="8월 23일">8월 23일</a> 발매됐고, <a class="mw-redirect" href="/wiki/%EC%95%84%EC%9D%B4%ED%8A%A0%EC%A6%88" title="아이튠즈">아이튠즈</a>와 북미에선 같은 날인 <a href="/wiki/2013%EB%85%84" title="2013년">2013년</a> <a href="/wiki/8%EC%9B%94_27%EC%9D%BC" title="8월 27일">8월 27일</a>에 발매 되었다. "Hail to the King"은 <a href="/wiki/2015%EB%85%84" title="2015년">2015년</a> <a href="/wiki/7%EC%9B%94" title="7월">7월</a> 탈퇴한 <a href="/wiki/%EC%97%90%EC%9D%B4%EB%A6%B0_%EC%9D%BC%EB%A6%AC%EC%A0%9C%EC%9D%B4" title="에이린 일리제이">에이린 일리제이</a>의 유일한 정규앨범이였다. 보너스 트랙으론 <a href="/wiki/%EB%8D%94_%EB%A0%88%EB%B8%8C" title="더 레브">더 레브</a>의 추모곡인 St.James 가 수록되었다.</p>

<p>모든곡은 <a class="mw-redirect" href="/wiki/%EC%97%A0_%EC%84%80%EB%8F%84%EC%9A%B0%EC%8A%A4" title="엠 섀도우스">엠 섀도우스</a>,<a href="/wiki/%EC%8B%9C%EB%8B%88%EC%8A%A4%ED%84%B0_%EA%B2%8C%EC%9D%B4%EC%B8%A0" title="시니스터 게이츠">시니스터 게이츠</a>,<a class="mw-redirect" href="/wiki/%EC%9E%AD%ED%82%A4_%EB%B2%A4%EC%A0%A0%EC%8A%A4" title="잭키 벤젠스">잭키 벤젠스</a>,<a href="/wiki/%EC%A1%B0%EB%8B%88_%ED%81%AC%EB%9D%BC%EC%9D%B4%EC%8A%A4%ED%8A%B8" title="조니 크라이스트">조니 크라이스트</a>가 썻다.</p>
