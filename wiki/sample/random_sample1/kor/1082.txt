<p><b>Hello world 프로그램</b>은 "Hello, world!"를 화면에 출력하는 <a href="/wiki/%EC%BB%B4%ED%93%A8%ED%84%B0_%ED%94%84%EB%A1%9C%EA%B7%B8%EB%9E%A8" title="컴퓨터 프로그램">컴퓨터 프로그램</a>이다. 이 프로그램은 <a href="/wiki/%ED%94%84%EB%A1%9C%EA%B7%B8%EB%9E%98%EB%B0%8D_%EC%96%B8%EC%96%B4" title="프로그래밍 언어">프로그래밍 언어</a>를 연습하는 데에 많이 쓰이고, 많은 프로그래밍 언어 서적에서 가장 처음 만들어보는 기본 예제로 나온다.</p>
<p>Hello world 프로그램은 프로그래밍 언어로 할 수 있는 간단한 것 중 하나이다. 그러나, <a href="/wiki/%EA%B7%B8%EB%9E%98%ED%94%BD_%EC%82%AC%EC%9A%A9%EC%9E%90_%EC%9D%B8%ED%84%B0%ED%8E%98%EC%9D%B4%EC%8A%A4" title="그래픽 사용자 인터페이스">GUI</a>를 사용할 때를 비롯하여, 어떤 경우에는 코드가 대단히 복잡해질 수 있다. 또 다른 경우에는 프로그램 자체는 간단하지만 CLI <a href="/wiki/%EC%85%B8" title="셸">셸</a>에서 입력해야 하는 파라미터가 많아 복잡한 경우도 있다. 또 <a href="/wiki/%EC%9E%84%EB%B2%A0%EB%94%94%EB%93%9C_%EC%8B%9C%EC%8A%A4%ED%85%9C" title="임베디드 시스템">임베디드 시스템</a>에서는 글자들이 한정된 한두 줄의 <a href="/wiki/%EC%95%A1%EC%A0%95_%EB%94%94%EC%8A%A4%ED%94%8C%EB%A0%88%EC%9D%B4" title="액정 디스플레이">LCD</a>에 표시될 것이다. 더욱 심한 경우에는 글자를 표시할 수 없어 "Hello world!" 대신에 간단하게 <a href="/wiki/%EB%B0%9C%EA%B4%91_%EB%8B%A4%EC%9D%B4%EC%98%A4%EB%93%9C" title="발광 다이오드">LED</a> 점 등을 할 수도 있다.</p>
<p>또한, Hello world 프로그램은 언어의 <a href="/wiki/%EC%BB%B4%ED%8C%8C%EC%9D%BC%EB%9F%AC" title="컴파일러">컴파일러</a>, <a href="/wiki/%ED%86%B5%ED%95%A9_%EA%B0%9C%EB%B0%9C_%ED%99%98%EA%B2%BD" title="통합 개발 환경">통합 개발 환경</a>, <a class="mw-redirect" href="/wiki/%EB%9F%B0%ED%83%80%EC%9E%84_%ED%99%98%EA%B2%BD" title="런타임 환경">런타임 환경</a>이 정상적으로 작동하는지를 확인하는 <a class="new" href="/w/index.php?title=%EC%83%88%EB%84%88%ED%8B%B0_%ED%85%8C%EC%8A%A4%ED%8A%B8&amp;action=edit&amp;redlink=1" title="새너티 테스트 (없는 문서)">새너티 테스트</a>로써 쓸모가 있다. 개발환경 구축에 필요한 <a href="/wiki/%ED%88%B4%EC%B2%B4%EC%9D%B8" title="툴체인">툴체인</a>을 바닥부터 구축하여 가장 간단한 프로그램을 컴파일하고 실행하기까지에는 상당한 작업이 필요하다. 따라서 새로운 툴체인을 테스트할 때에는 될 수 있는 한 간단한 프로그램이 이용된다.</p>
<p>프로그래밍 할 수 있는 컴퓨터 개발에 있어서 작고 간단한 테스트용 프로그램이 이전에도 존재했으나, "Hello world!"가 사용된 것은 1978년에 출판된 <a href="/wiki/%EB%B8%8C%EB%9D%BC%EC%9D%B4%EC%96%B8_%EC%BB%A4%EB%8B%88%ED%95%B8" title="브라이언 커니핸">브라이언 커니핸</a>과 <a href="/wiki/%EB%8D%B0%EB%8B%88%EC%8A%A4_%EB%A6%AC%EC%B9%98" title="데니스 리치">데니스 리치</a>가 쓴 "The C Programming Language"라는 책에서 비롯한다. 이 책에서 첫 번째 예제 프로그램으로 hello, world 라는 문장을 출력했다. 모두 소문자이고, 느낌표도 없었다. 이 프로그램은 1974년에 <a href="/wiki/%EB%B2%A8_%EC%97%B0%EA%B5%AC%EC%86%8C" title="벨 연구소">벨 연구소</a>에서 커니핸이 써서 연구소 내에서 사용한 작은 매뉴얼인 〈Programming in C: A Tutorial〉의 것을 물려받았다.</p>

<p>하지만, 처음으로 "hello"와 "world"를 같이 사용한 경우는 브라이언 커니핸이 1973년도에 쓴 〈A Tutorial Introduction to the Language B〉에 등장한다.<sup class="reference" id="cite_ref-1"><a href="#cite_note-1">[1]</a></sup></p>
<p>구두 방식이나 단락 방식에 따라서 여러 가지 방법이 존재하고, 출력 문장에서 몇몇 차이가 발생한다. 쉼표나 느낌표가 생략되는 경우도 있고, 'H'만이 대문자로 되거나 혹은 모두 소문자가 되는 경우도 있다. 원래 것과는 좀 달라졌지만, 가장 일반적인 문장은 "Hello, world!"이다. 몇몇 언어에서는 모든 문자를 대문자로 출력하여, "HELLO WORLD!"와 같은 결과가 나오기도 하고, 많은 <a href="/wiki/%EB%82%9C%ED%95%B4%ED%95%9C_%ED%94%84%EB%A1%9C%EA%B7%B8%EB%9E%98%EB%B0%8D_%EC%96%B8%EC%96%B4" title="난해한 프로그래밍 언어">난해한 프로그래밍 언어</a>에서는 약간 변형된 문장을 출력하기도 한다. "Hello world" 프로그램은 보통은 스트링 끝이 개행 문자의 출력도 포함한다. (ASCII 코드 10 혹은 13,10)</p>
<p>"hello world" 프로그램의 여러 가지 언어로 쓰인 코드들의 모음은 여러 프로그래밍 언어들을 배우고 비교하는 데에 도움을 주는, 간단한 "<a href="/wiki/%EB%A1%9C%EC%A0%9C%ED%83%80%EC%84%9D" title="로제타석">로제타석</a>"으로 사용할 수도 있다.</p>
