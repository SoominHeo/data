<p><b>INRI</b>(<span style="font-size: smaller;"><a href="/wiki/%EB%9D%BC%ED%8B%B4%EC%96%B4" title="라틴어">라틴어</a>:</span> <span lang="la" xml:lang="la">IESVS·NAZARENVS·REX·IVDÆORVM</span>)는 <a href="/wiki/%ED%95%9C%EA%B5%AD%EC%96%B4" title="한국어">한국어</a>로 번역하면 <b>"<a href="/wiki/%EC%9C%A0%EB%8C%80%EC%9D%B8" title="유대인">유다인</a>들의 <a href="/wiki/%EC%99%95" title="왕">임금</a>, <a href="/wiki/%EB%82%98%EC%82%AC%EB%A0%9B" title="나사렛">나자렛 사람</a> <a href="/wiki/%EC%98%88%EC%88%98" title="예수">예수</a>"</b> 이란 뜻을 가진 <a href="/wiki/%EA%B8%B0%EB%8F%85%EA%B5%90" title="기독교">기독교</a>의 <a href="/wiki/%EB%91%90%EB%AC%B8%EC%9E%90%EC%96%B4" title="두문자어">두문자어</a>이다. 이 구절은 <a href="/wiki/%EC%98%88%EC%88%98" title="예수">예수</a>가 <a href="/wiki/%EC%8B%AD%EC%9E%90%EA%B0%80" title="십자가">십자가</a>에 처형될 때 예수에 머리위에 붙인 죄목의 표시로 <a class="mw-redirect" href="/wiki/%EC%8B%A0%EC%95%BD%EC%84%B1%EC%84%9C" title="신약성서">신약성서</a>의 <a class="mw-redirect" href="/wiki/%EC%9A%94%ED%95%9C%EB%B3%B5%EC%9D%8C%EC%84%9C" title="요한복음서">요한복음서</a> (<a class="external text" href="http://www.bskorea.or.kr/infobank/korSearch/korbibReadpage.aspx?version=COGNEW&amp;book=jhn&amp;chap=19&amp;sec=19&amp;cVersion=&amp;fontString=12px&amp;fontSize=1#focus" rel="nofollow">19:19</a>)에 나온다. 요한복음 외에 <a href="/wiki/%EA%B3%B5%EA%B4%80%EB%B3%B5%EC%9D%8C%EC%84%9C" title="공관복음서">공관복음서</a>에도 각각 약간씩 변형된 표현이긴 하지만 예수의 죄목을 다룬 부분이 다음과 같이 나오며 <a class="mw-redirect" href="/wiki/%EC%82%AC%EB%B3%B5%EC%9D%8C%EC%84%9C" title="사복음서">사복음서</a> 모두 에수의 죄목을 다루었다. 공관복음서의 대목을 보면 <a class="mw-redirect" href="/wiki/%EB%A7%88%ED%83%9C%EC%98%A4_%EB%B3%B5%EC%9D%8C%EC%84%9C" title="마태오 복음서">마태오</a>(<a class="external text" href="http://www.bskorea.or.kr/infobank/korSearch/korbibReadpage.aspx?version=COGNEW&amp;book=mat&amp;chap=27&amp;sec=37#focus" rel="nofollow">27:37</a>), <a class="mw-redirect" href="/wiki/%EB%A7%88%EB%A5%B4%EC%BD%94_%EB%B3%B5%EC%9D%8C%EC%84%9C" title="마르코 복음서">마르코</a>(<a class="external text" href="http://www.bskorea.or.kr/infobank/korSearch/korbibReadpage.aspx?version=COGNEW&amp;book=mrk&amp;chap=15&amp;sec=26&amp;cVersion=&amp;fontString=12px&amp;fontSize=1#focus" rel="nofollow">15:26</a>), <a class="mw-redirect" href="/wiki/%EB%A3%A8%EC%B9%B4_%EB%B3%B5%EC%9D%8C%EC%84%9C" title="루카 복음서">루가</a>(<a class="external text" href="http://www.bskorea.or.kr/infobank/korSearch/korbibReadpage.aspx?version=COGNEW&amp;book=luk&amp;chap=23&amp;sec=38&amp;cVersion=&amp;fontString=12px&amp;fontSize=1#focus" rel="nofollow">23:38</a>)에 나온다.</p>
<p>요한복음서에 따르면 이 죄패는 <a href="/wiki/%EB%A1%9C%EB%A7%88_%EC%A0%9C%EA%B5%AD" title="로마 제국">로마</a>의 총독인 <a class="mw-redirect" href="/wiki/%EB%B9%8C%EB%9D%BC%EB%8F%84" title="빌라도">빌라도</a>가 썼다고 전하며 <a href="/wiki/%ED%9E%88%EB%B8%8C%EB%A6%AC%EC%96%B4" title="히브리어">히브리어</a>, <a href="/wiki/%EA%B7%B8%EB%A6%AC%EC%8A%A4%EC%96%B4" title="그리스어">그리스어</a>, <a href="/wiki/%EB%9D%BC%ED%8B%B4%EC%96%B4" title="라틴어">라틴어</a>로 각각 쓰여져 십자가에 붙여졌다.</p>

<p><a href="/wiki/%EB%A1%9C%EB%A7%88_%EA%B0%80%ED%86%A8%EB%A6%AD%EA%B5%90%ED%9A%8C" title="로마 가톨릭교회">로마 가톨릭교회</a>와 <a href="/wiki/%EC%84%B1%EA%B3%B5%ED%9A%8C" title="성공회">성공회</a>에서는 이 죄패를 십자가상의 예수의 머리위에 패로 만들어 붙이는 전통이 있으며, 드물게는 그냥 십자가에 직접 새겨 넣기도 한다.</p>
<p><a href="/wiki/%EB%8F%99%EB%B0%A9_%EC%A0%95%EA%B5%90%ED%9A%8C" title="동방 정교회">동방 정교회</a>에서는 <a href="/wiki/%EA%B7%B8%EB%A6%AC%EC%8A%A4%EC%96%B4" title="그리스어">그리스어</a> 표기에 근거하여 <i>INBI'</i>(Ἰησοῦς ὁ Ναζωραῖος ὁ Bασιλεὺς τῶν Ἰουδαίων)로 표기하며 몇몇 동방교회에서는 <i>ΙΝΒΚ</i> (ὁ Bασιλεὺς τοῦ κόσμου, <i>세계의 왕</i> ) 또는 <i>ΙΝΒΔ</i>(ὁ Bασιλεὺς τῆς Δόξης , <i>영광의 왕</i> )으로 표기하기도 한다. <a href="/wiki/%EB%A3%A8%EB%A7%88%EB%8B%88%EC%95%84_%EC%A0%95%EA%B5%90%ED%9A%8C" title="루마니아 정교회">루마니아 정교회</a>와 같은 일부 정교회에서는 라틴교회와 같이 <i>INRI</i> 을 사용한다. <a href="/wiki/%EB%9F%AC%EC%8B%9C%EC%95%84_%EC%A0%95%EA%B5%90%ED%9A%8C" title="러시아 정교회">러시아 정교회</a>는 <i>ІНЦІ</i> 또는 <i>Цръ Слвы</i> (<i>영광의 왕</i>)을 사용한다.</p>
<p><a href="/wiki/%EC%84%B1%EA%B3%B5%ED%9A%8C" title="성공회">성공회</a>와 <a href="/wiki/%EB%A3%A8%ED%84%B0%EA%B5%90" title="루터교">루터교</a> 등을 제외한 일부 <a href="/wiki/%EA%B0%9C%EC%8B%A0%EA%B5%90" title="개신교">개신교</a>에서는 이 죄패를 사용하지 않고 단순한 십자가(예수의 형상도 제외함) 형태만을 사용하는데, 단, <a class="mw-redirect" href="/wiki/%EB%A3%A8%ED%84%B0%EA%B5%90%ED%9A%8C" title="루터교회">루터교회</a>에서는 <span class="fix" title="이 부분은 출처가 필요합니다.">다른 교파들과는 달리, <a href="/wiki/%EB%A1%9C%EB%A7%88_%EA%B0%80%ED%86%A8%EB%A6%AD%EA%B5%90%ED%9A%8C" title="로마 가톨릭교회">로마 가톨릭교회</a> <a href="/wiki/%EC%98%88%EB%B0%B0" title="예배">전례</a>에서 좋은 점은 받아들였기 때문에</span><sup class="정리_필요 noprint">[<a href="/wiki/%EC%9C%84%ED%82%A4%EB%B0%B1%EA%B3%BC:%EC%B6%9C%EC%B2%98_%EB%B0%9D%ED%9E%88%EA%B8%B0" title="위키백과:출처 밝히기">출처 필요</a>]</sup> <a href="/wiki/%EC%98%88%EC%88%98" title="예수">예수</a>가 매달린 모습의 십자가(<a href="/wiki/%EC%8B%AD%EC%9E%90%EA%B3%A0%EC%83%81" title="십자고상">십자고상</a>)을 사용한다.</p>
