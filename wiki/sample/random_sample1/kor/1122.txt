<h1 class="firstHeading" id="firstHeading" lang="ko">ISO 3166-2:AN</h1>
<div class="mw-body-content" id="bodyContent">
<div id="siteSub">위키백과, 우리 모두의 백과사전.</div>
<div id="contentSub"></div>
<div class="mw-jump" id="jump-to-nav">
					이동:					<a href="#mw-head">둘러보기</a>, 					<a href="#p-search">검색</a>
</div>
<div class="mw-content-ltr" dir="ltr" id="mw-content-text" lang="ko"><script>function mfTempOpenSection(id){var block=document.getElementById("mf-section-"+id);block.className+=" open-block";block.previousSibling.className+=" open-block";}</script><p><b>ISO 3166-2:AN</b>은 <a class="new" href="/w/index.php?title=%EC%A7%80%EB%A6%AC_%EC%8B%9D%EB%B3%84_%EB%B6%80%ED%98%B8&amp;action=edit&amp;redlink=1" title="지리 식별 부호 (없는 문서)">지리 식별 부호</a>(geocode)를 정의하는 <a class="mw-redirect" href="/wiki/ISO" title="ISO">ISO</a> 표준인 <a href="/wiki/ISO_3166-2" title="ISO 3166-2">ISO 3166-2</a>의 일부이며, <a href="/wiki/%EB%84%A4%EB%8D%9C%EB%9E%80%EB%93%9C%EB%A0%B9_%EC%95%88%ED%8B%B8%EB%A0%88%EC%8A%A4" title="네덜란드령 안틸레스">네덜란드령 안틸레스</a>에 적용된다. AN은 <a href="/wiki/ISO_3166-1" title="ISO 3166-1">ISO 3166-1</a>에서 할당된 국가 코드를 사용한다. 현재는 <a href="/wiki/ISO_3166-2" title="ISO 3166-2">ISO 3166-2</a>에서 할당된 코드가 없었고, 과거에도 없었다. 2010년에 <a href="/wiki/%EB%84%A4%EB%8D%9C%EB%9E%80%EB%93%9C" title="네덜란드">네덜란드</a>에 국가의 일부로 병합이 되면서, <a href="/wiki/ISO_3166-1" title="ISO 3166-1">ISO 3166-1</a>에서도 삭제되었다.</p>
