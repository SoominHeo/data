<h2><span class="mw-headline" id=".EC.97.AD.EC.82.AC.EC.99.80_.EA.B0.9C.EB.85.90">역사와 개념</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=J-pop&amp;action=edit&amp;section=1" title="부분 편집: 역사와 개념">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>일본의 대중가요는 전후에 유행가라는 장르로 나뉘었고, <a href="/wiki/1970%EB%85%84%EB%8C%80" title="1970년대">1970년대</a>에서 <a href="/wiki/1980%EB%85%84%EB%8C%80" title="1980년대">1980년대</a>에는 <a class="mw-redirect" href="/wiki/%EC%8B%B1%EC%96%B4_%EC%86%A1_%EB%9D%BC%EC%9D%B4%ED%84%B0" title="싱어 송 라이터">싱어 송 라이터</a>의 노래는 일본 내의 <a class="new" href="/w/index.php?title=%EB%89%B4%EB%AE%A4%EC%A7%81&amp;action=edit&amp;redlink=1" title="뉴뮤직 (없는 문서)">뉴뮤직</a>이라는 명칭의 장르가 진정한 애국의 화두가 되었는데, 일본 역사 상 처음으로 문화가 정치권과 공식적으로 언론을 통한 대립을 겪으면서 대립보다는 사회적인 각자의 역할에 충실하고자 산업화 되어 오늘날에는 대표적으론 <a href="/wiki/%EC%97%94%EC%B9%B4" title="엔카">엔카</a>,J-pop,J-rock 으로 분류되고 있으며 그 상업성에 따라 메쟈 (major) 또는 비메쟈 (한자의 아닐 비와 major의 합성어) 로 구분하는 정도이다.</p>
<p>J-pop은 다음과 같은 부분을 포함한다고 볼 수 있다.</p>
<ul>
<li>아이돌 가요는 포함하지만, <a href="/wiki/%EC%97%94%EC%B9%B4" title="엔카">엔카</a>는 포함하지 않는다.</li>
<li><a class="mw-redirect" href="/wiki/%ED%8F%AC%ED%81%AC_%EC%86%A1" title="포크 송">포크 송</a>은 어느 부분까지 포함하는지에 대해서는 다양한 의견이 있다. 반전(反戰)과 같은 메시지를 전면에 등장시킨 소위 <a class="new" href="/w/index.php?title=%EB%A9%94%EC%8B%9C%EC%A7%80_%ED%8F%AC%ED%81%AC&amp;action=edit&amp;redlink=1" title="메시지 포크 (없는 문서)">메시지 포크</a>는 포함하지 않는다는 의견이 많다.</li>
<li><a href="/wiki/%ED%95%98%EB%93%9C_%EB%A1%9D" title="하드 록">하드 록</a>이나 <a class="mw-redirect" href="/wiki/%ED%97%A4%EB%B9%84%EB%A9%94%ED%83%88" title="헤비메탈">헤비메탈</a>, <a class="new" href="/w/index.php?title=%EA%B8%B0%ED%83%80_%EB%A1%9D&amp;action=edit&amp;redlink=1" title="기타 록 (없는 문서)">기타 록</a>, <a href="/wiki/%EB%A1%9C%ED%81%B0%EB%A1%A4" title="로큰롤">로큰롤</a>, <a href="/wiki/%ED%8E%91%ED%81%AC_%EB%A1%9D" title="펑크 록">펑크 록</a>, 하드코어, 펑크, <a class="new" href="/w/index.php?title=%EB%9D%BC%EC%9A%B0%EB%93%9C_%EB%A1%9D&amp;action=edit&amp;redlink=1" title="라우드 록 (없는 문서)">라우드 록</a>, <a class="new" href="/w/index.php?title=%ED%97%A4%EB%B9%84_%EB%A1%9D&amp;action=edit&amp;redlink=1" title="헤비 록 (없는 문서)">헤비 록</a>, <a href="/wiki/%ED%9E%99%ED%95%A9" title="힙합">힙합</a>, <a href="/wiki/%EB%A0%88%EA%B2%8C" title="레게">레게</a>, <a href="/wiki/%ED%85%8C%ED%81%AC%EB%85%B8" title="테크노">테크노</a>, 하우스, <a class="new" href="/w/index.php?title=%EB%B9%84%EC%A3%BC%EC%96%BC&amp;action=edit&amp;redlink=1" title="비주얼 (없는 문서)">비주얼</a>, 아니메 송 등은 통상적으로 제외된다. 하드 록 등의 록은 <a href="/wiki/J-ROCK" title="J-ROCK">J-ROCK</a>으로 불리기도 한다.</li>
<li>뉴 뮤직과는 겹치기도 하지만, 같은 범위는 아니다.</li>
</ul>
<p>따라서 J-pop에서 ‘팝’은 좁은 의미의 ‘팝’(<a href="/wiki/%EB%AF%B8%EA%B5%AD" title="미국">미국</a>의 영향을 강하게 받아 리듬이나 템포가 유사한 밝은 곡조의 음악)이 아니라, ‘일본어로 불리는 팝’, ‘일본어로 불리는 록’, ‘가요’ 등을 포함한 넓은 의미이다. 그러나 원래 장르의 구분이 모호하여, 이상의 장르 모두가 ‘J-pop’으로 불리기도 한다.</p>
<p>엔카와의 구분은 명확한 경우가 많지만, <a href="/wiki/%EB%82%98%EC%B9%B4%EC%A7%80%EB%A7%88_%EB%AF%B8%EC%9C%A0%ED%82%A4" title="나카지마 미유키">나카지마 미유키</a>의 곡처럼 경계선상에 위치한 곡도 일부 있다. 따라서 J-pop에 포함되는 부분에 대한 엄밀한 분류는 불가능하다. 또한 J-pop 가수의 경우 <a class="mw-redirect" href="/wiki/%EC%8B%B1%EC%96%B4_%EC%86%A1_%EB%9D%BC%EC%9D%B4%ED%84%B0" title="싱어 송 라이터">싱어 송 라이터</a>나 <a href="/wiki/%EC%9E%91%EC%82%AC" title="작사">작사</a>, <a href="/wiki/%EC%9E%91%EA%B3%A1" title="작곡">작곡</a> 등을 일부 맡는 경우도 많지만, 이것이 필수 조건은 아니다.</p>
<h2><span class="mw-headline" id=".EB.8F.99.EC.9D.98.EC.96.B4">동의어</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=J-pop&amp;action=edit&amp;section=2" title="부분 편집: 동의어">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>‘저페니스 팝스(ジャパニーズ・ポップス)’라는 용어가 같은 의미로 사용된다. <i><a href="/wiki/J-ROCK" title="J-ROCK">J-ROCK</a></i> 또한 J-pop과 비슷한 의미로 사용되기도 하지만, J-pop처럼 보급되지는 않았다. ‘일본의’라는 의미로 J를 붙인 <i>J-RAP, J-SOUL</i> 등이 한때 등장했지만, 최근에는 별로 사용되지 않는다.</p>
<p><a href="/wiki/JAPAN_FM_LEAGUE" title="JAPAN FM LEAGUE">JFL</a>계열의 라디오 방송국 <a href="/wiki/ZIP-FM" title="ZIP-FM">ZIP-FM</a>이 <b>Z-POP</b>이라는 용어를 사용하기도 한다. J-POP과 거의 같은 뜻이다.</p>
