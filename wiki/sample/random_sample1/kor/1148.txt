<p><b>J. 그레그 피터스</b>(<span style="font-size: smaller;"><a href="/wiki/%EC%98%81%EC%96%B4" title="영어">영어</a>:</span> <span lang="en" xml:lang="en">J. Greg Peters</span>, <a href="/wiki/%EB%A1%9C%EC%97%B4_%EB%B9%85%ED%86%A0%EB%A6%AC%EC%95%84_%ED%9B%88%EC%9E%A5" title="로열 빅토리아 훈장">MVO</a>, <a href="/wiki/1960%EB%85%84" title="1960년">1960년</a> <a href="/wiki/11%EC%9B%94_15%EC%9D%BC" title="11월 15일">11월 15일</a> ~)는 전직 <a href="/wiki/%EC%BA%90%EB%82%98%EB%8B%A4_%EC%99%95%EB%A6%BD_%EA%B8%B0%EB%A7%88%EA%B2%BD%EC%B0%B0%EB%8C%80" title="캐나다 왕립 기마경찰대">캐나다 왕립 기마경찰대</a> <a class="extiw" href="https://en.wikipedia.org/wiki/Royal_Canadian_Mounted_Police#Ranks" title="en:Royal Canadian Mounted Police">감독관</a>이며 현재 <a href="/wiki/%EC%BA%90%EB%82%98%EB%8B%A4_%EC%83%81%EC%9B%90" title="캐나다 상원">캐나다 상원</a>의 <a href="/wiki/%EC%BA%90%EB%82%98%EB%8B%A4_%EC%83%81%EC%9B%90_%ED%9D%91%EC%9E%A5%EC%88%98%EC%9C%84%EA%B4%80" title="캐나다 상원 흑장수위관">흑장수위관</a>이다. 그는 <a href="/wiki/%EC%BA%90%EB%82%98%EB%8B%A4%EC%9D%98_%EA%B5%B0%EC%A3%BC" title="캐나다의 군주">캐나다</a> <a class="mw-redirect" href="/wiki/%EC%B6%94%EB%B0%80%EC%9B%90%EC%97%90_%EC%9E%88%EC%96%B4%EC%84%9C%EC%9D%98_%EC%B4%9D%EB%8F%85" title="추밀원에 있어서의 총독">추밀원에 있어서의 총독</a>의 어명으로 2013년 10월 1일에 흑장수위관이 되었다.</p>
<h2><span class="mw-headline" id=".EC.83.9D.EC.95.A0">생애</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=J._%EA%B7%B8%EB%A0%88%EA%B7%B8_%ED%94%BC%ED%84%B0%EC%8A%A4&amp;action=edit&amp;section=1" title="부분 편집: 생애">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>본명은 존 그레고리 피터스(John Gregory Peters)이며 <a href="/wiki/%ED%94%84%EB%A6%B0%EC%8A%A4%EC%97%90%EB%93%9C%EC%9B%8C%EB%93%9C%EC%95%84%EC%9D%BC%EB%9E%9C%EB%93%9C_%EC%A3%BC" title="프린스에드워드아일랜드 주">프린스에드워드아일랜드 주</a> <a class="extiw" href="https://en.wikipedia.org/wiki/List_of_municipalities_in_Prince_Edward_Island" title="en:List of municipalities in Prince Edward Island">수리 웨스트</a> 출신이다. 그는 유년 시절에 <a class="extiw" href="https://en.wikipedia.org/wiki/St._John_Ambulance" title="en:St. John Ambulance">세인트 존 앰뷸런스</a>에서 깊이 봉사활동하였던 경험이 있었다. 그는 세인트 존 앰뷸런스를 통해서 지역 사회에 응급처치와 심폐소생술을 가르쳤다. 그는 훗날 고급 응급처치 자격증을 땄다. 피터스는 <a class="extiw" href="https://en.wikipedia.org/wiki/University_of_Prince_Edward_Island" title="en:University of Prince Edward Island">프린스에드워드아일랜드 대학교</a>에 입학하여 정치학과 역사학을 배웠다. 그는 나중에 <a href="/wiki/%EC%98%A4%ED%83%80%EC%99%80_%EB%8C%80%ED%95%99%EA%B5%90" title="오타와 대학교">오타와 대학교</a>로 전학하여 정치학 학사 학위를 수여받았다. 그는 또한 오타와 대학교에서 공공 리더십과 행정사무 준석사 수료증을 받았다.</p>
<h2><span class="mw-headline" id=".ED.99.9C.EB.8F.99">활동</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=J._%EA%B7%B8%EB%A0%88%EA%B7%B8_%ED%94%BC%ED%84%B0%EC%8A%A4&amp;action=edit&amp;section=2" title="부분 편집: 활동">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>피터스는 1981년에 <a href="/wiki/%EC%BA%90%EB%82%98%EB%8B%A4_%EC%99%95%EB%A6%BD_%EA%B8%B0%EB%A7%88%EA%B2%BD%EC%B0%B0%EB%8C%80" title="캐나다 왕립 기마경찰대">캐나다 왕립 기마경찰대</a>에 입대하여 <a href="/wiki/%EB%A6%AC%EC%9E%90%EC%9D%B4%EB%82%98" title="리자이나">리자이나</a>에 소재한 <a class="extiw" href="https://en.wikipedia.org/wiki/RCMP_Academy,_Depot_Division" title="en:RCMP Academy, Depot Division">캐나다 왕립 기마경찰대 사관학교</a>에서 훈련을 받았다. 매니토바 주의 <a class="extiw" href="https://en.wikipedia.org/wiki/Flin_Flon" title="en:Flin Flon">플린 플론</a>이라는 지역에 발령받아서 그 곳에 순찰 활동을 하였으나 나중에 작은 마을들과 원주민 구역들이 뭉쳐 있는 매니토바 남부로 발령받았다. 1984년 <a href="/wiki/%EC%BA%90%EB%82%98%EB%8B%A4%EC%9D%98_%EA%B5%B0%EC%A3%BC" title="캐나다의 군주">캐나다의 군주</a>의 <a class="extiw" href="https://en.wikipedia.org/wiki/Royal_tours_of_Canada" title="en:Royal tours of Canada">캐나다 공식 방문</a>의 일환으로 그는 <a href="/wiki/%EC%97%98%EB%A6%AC%EC%9E%90%EB%B2%A0%EC%8A%A4_2%EC%84%B8" title="엘리자베스 2세">엘리자베스 2세</a>가 매니토바 주를 방문했을 때에 호위 임무를 맡았다.</p>
<p>피터스는 1987년에 캐나다 왕립 기마경찰대의 기마 시범 부대 <a class="extiw" href="https://en.wikipedia.org/wiki/Musical_Ride" title="en:Musical Ride">뮤지컬 라이드</a>에 들어왔다. 그의 첫 순방은 1988년에 <a href="/wiki/%EC%9C%A0%EB%9F%BD" title="유럽">유럽</a>에서 3개월 동안 기마 시범을 보여 주었다. 특히 그는 <a href="/wiki/%EC%9C%88%EC%A0%80_%EC%84%B1" title="윈저 성">윈저 성</a>에서 귀빈으로 머물었으며 <a class="extiw" href="https://en.wikipedia.org/wiki/Royal_Mews" title="en:Royal Mews">로열 뮤즈</a>에서 말들을 관리하였다. 이렇게 해서 그는 캐나다와 <a href="/wiki/%EC%98%81%EA%B5%AD%EC%9D%98_%EA%B5%B0%EC%A3%BC" title="영국의 군주">영국의 군주</a> 엘리자베스 2세와 왕족들을 처음으로 가까이 만나게 된 계기가 되었다.</p>
<p>피터스는 1993년 <a href="/wiki/%EC%95%84%EC%9D%B4%ED%8B%B0" title="아이티">아이티</a> <a href="/wiki/%ED%8F%AC%EB%A5%B4%ED%86%A0%ED%94%84%EB%9E%AD%EC%8A%A4" title="포르토프랭스">포르토프랭스</a>에서 <a class="extiw" href="https://en.wikipedia.org/wiki/United_Nations_Police" title="en:United Nations Police">유엔 민간경찰</a> 파견단의 일원으로 차안활동에 기여하였다.</p>
