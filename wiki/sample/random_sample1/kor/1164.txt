<p><b>JTBC</b>(<span style="font-size: smaller;"><a href="/wiki/%ED%95%9C%EA%B5%AD%EC%96%B4" title="한국어">한국어</a>:</span> <span lang="ko">제이티비씨</span>)는 유료 플랫폼으로 방송하는 대한민국의 <a class="mw-redirect" href="/wiki/%EC%A2%85%ED%95%A9%ED%8E%B8%EC%84%B1%EC%B1%84%EB%84%90" title="종합편성채널">종합편성채널</a> 사업자이자, 종합편성채널이다. <a href="/wiki/2011%EB%85%84" title="2011년">2011년</a> <a href="/wiki/3%EC%9B%94_21%EC%9D%BC" title="3월 21일">3월 21일</a>에 창립하여 <a href="/wiki/2011%EB%85%84" title="2011년">2011년</a> <a href="/wiki/12%EC%9B%94_1%EC%9D%BC" title="12월 1일">12월 1일</a>에 개국하였다. <a class="mw-redirect" href="/wiki/%EC%A4%91%EC%95%99%EC%9D%BC%EB%B3%B4%EC%82%AC" title="중앙일보사">중앙일보사</a>에서 출자하여 설립한 회사로, <a href="/wiki/%EC%A4%91%EC%95%99%EB%AF%B8%EB%94%94%EC%96%B4%EB%84%A4%ED%8A%B8%EC%9B%8C%ED%81%AC" title="중앙미디어네트워크">중앙미디어네트워크</a>가 대주주로 있다. 회장은 중앙미디어네트워크의 <a href="/wiki/%ED%99%8D%EC%84%9D%ED%98%84" title="홍석현">홍석현</a>이 맡고 있으며, 대표이사는 <a class="new" href="/w/index.php?title=%EA%B9%80%EC%88%98%EA%B8%B8&amp;action=edit&amp;redlink=1" title="김수길 (없는 문서)">김수길</a> 사장과 <a class="new" href="/w/index.php?title=%ED%99%8D%EC%A0%95%EB%8F%84&amp;action=edit&amp;redlink=1" title="홍정도 (없는 문서)">홍정도</a> 부사장이, 보도부문사장은 <a href="/wiki/%EC%86%90%EC%84%9D%ED%9D%AC" title="손석희">손석희</a>이다.</p>
<p>사옥은 <a href="/wiki/2011%EB%85%84" title="2011년">2011년</a> <a href="/wiki/12%EC%9B%94_1%EC%9D%BC" title="12월 1일">12월 1일</a>부터 <a href="/wiki/2015%EB%85%84" title="2015년">2015년</a> <a href="/wiki/1%EC%9B%94_3%EC%9D%BC" title="1월 3일">1월 3일</a>까지는 <a href="/wiki/%EC%84%9C%EC%9A%B8%ED%8A%B9%EB%B3%84%EC%8B%9C" title="서울특별시">서울특별시</a> <a href="/wiki/%EC%A4%91%EA%B5%AC_(%EC%84%9C%EC%9A%B8%ED%8A%B9%EB%B3%84%EC%8B%9C)" title="중구 (서울특별시)">중구</a> <a class="mw-redirect" href="/wiki/%EC%88%9C%ED%99%94%EB%8F%99" title="순화동">순화동</a> 중앙일보빌딩에 있었으며, <a href="/wiki/2015%EB%85%84" title="2015년">2015년</a> <a href="/wiki/1%EC%9B%94_4%EC%9D%BC" title="1월 4일">1월 4일</a>에 <a href="/wiki/%EC%84%9C%EC%9A%B8%ED%8A%B9%EB%B3%84%EC%8B%9C" title="서울특별시">서울특별시</a> <a href="/wiki/%EB%A7%88%ED%8F%AC%EA%B5%AC" title="마포구">마포구</a> <a class="mw-redirect" href="/wiki/%EC%83%81%EC%95%94%EB%8F%99_(%EB%A7%88%ED%8F%AC%EA%B5%AC)" title="상암동 (마포구)">상암동</a> <a class="mw-redirect" href="/wiki/%EB%94%94%EC%A7%80%ED%84%B8%EB%AF%B8%EB%94%94%EC%96%B4%EC%8B%9C%ED%8B%B0" title="디지털미디어시티">1651 JTBC빌딩</a>으로 이전하였다.</p>
<p>중앙일보 측은 JTBC가 "과거 <a href="/wiki/1980%EB%85%84_%EC%96%B8%EB%A1%A0_%ED%86%B5%ED%8F%90%ED%95%A9" title="1980년 언론 통폐합">언론 통폐합 정책</a>으로 <a href="/wiki/%ED%95%9C%EA%B5%AD%EB%B0%A9%EC%86%A1%EA%B3%B5%EC%82%AC" title="한국방송공사">한국방송공사</a>에 합병되었던 <a href="/wiki/%EB%8F%99%EC%96%91%EB%B0%A9%EC%86%A1" title="동양방송">TBC</a>의 후신"임을 주장하고 있다. 이러한 이유를 들어 1980년 <a href="/wiki/11%EC%9B%94_30%EC%9D%BC" title="11월 30일">11월 30일</a>에 폐국된 TBC를 기념하여 2011년 <a href="/wiki/12%EC%9B%94_1%EC%9D%BC" title="12월 1일">12월 1일</a>에 개국했으며, 이는 다른 종편이 JTBC와 같은 일자에 개국하는 데 영향을 주었다.<sup class="reference" id="cite_ref-1"><a href="#cite_note-1">[1]</a></sup> 개국 때 부터 "다채로운 즐거움"을 구호로 사용하고 있다.</p>

<h2><span class="mw-headline" id=".EC.97.AD.EC.82.AC">역사</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=JTBC&amp;action=edit&amp;section=1" title="부분 편집: 역사">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<h3><span class="mw-headline" id=".EC.84.A4.EB.A6.BD_.EC.B4.88.EA.B8.B0">설립 초기</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=JTBC&amp;action=edit&amp;section=2" title="부분 편집: 설립 초기">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<div class="rellink boilerplate seealso"><img alt="" data-file-height="128" data-file-width="128" height="16" src="//upload.wikimedia.org/wikipedia/commons/thumb/e/ec/Crystal_Clear_app_xmag.svg/16px-Crystal_Clear_app_xmag.svg.png" srcset="//upload.wikimedia.org/wikipedia/commons/thumb/e/ec/Crystal_Clear_app_xmag.svg/24px-Crystal_Clear_app_xmag.svg.png 1.5x, //upload.wikimedia.org/wikipedia/commons/thumb/e/ec/Crystal_Clear_app_xmag.svg/32px-Crystal_Clear_app_xmag.svg.png 2x" width="16"/> <a href="/wiki/%EB%8C%80%ED%95%9C%EB%AF%BC%EA%B5%AD_%EB%AF%B8%EB%94%94%EC%96%B4_%EA%B4%80%EB%A0%A8%EB%B2%95_%EA%B0%9C%EC%A0%95_%EB%85%BC%EB%9E%80" title="대한민국 미디어 관련법 개정 논란">대한민국 미디어 관련법 개정 논란</a>, <a href="/wiki/%EB%8C%80%ED%95%9C%EB%AF%BC%EA%B5%AD%EC%9D%98_%EC%A2%85%ED%95%A9%ED%8E%B8%EC%84%B1%EC%B1%84%EB%84%90" title="대한민국의 종합편성채널">대한민국의 종합편성채널</a>, <a href="/wiki/%EB%8C%80%ED%95%9C%EB%AF%BC%EA%B5%AD%EC%9D%98_%EC%A2%85%ED%95%A9%ED%8E%B8%EC%84%B1%EC%B1%84%EB%84%90_%EC%84%A0%EC%A0%95_%EB%85%BC%EB%9E%80" title="대한민국의 종합편성채널 선정 논란">대한민국의 종합편성채널 선정 논란</a> 문서를 참고하십시오.</div>
<p>1980년 <a class="mw-redirect" href="/wiki/%EC%8B%A0%EA%B5%B0%EB%B6%80" title="신군부">신군부</a>의 <a href="/wiki/1980%EB%85%84_%EC%96%B8%EB%A1%A0_%ED%86%B5%ED%8F%90%ED%95%A9" title="1980년 언론 통폐합">언론 통폐합 정책</a>으로 당시 중앙일보와 같은 <a class="mw-redirect" href="/wiki/%EC%82%BC%EC%84%B1" title="삼성">삼성</a>의 계열사였던 <a href="/wiki/%EB%8F%99%EC%96%91%EB%B0%A9%EC%86%A1" title="동양방송">동양방송</a>(TBC)이 <a class="mw-redirect" href="/wiki/KBS" title="KBS">KBS</a>에 강제 합병된 이후, 중앙일보는 자회사인 <a href="/wiki/%EC%A4%91%EC%95%99%EB%B0%A9%EC%86%A1_(%EB%8C%80%ED%95%9C%EB%AF%BC%EA%B5%AD_%EB%AF%BC%EC%98%81%EB%B0%A9%EC%86%A1%EC%82%AC)" title="중앙방송 (대한민국 민영방송사)">중앙방송</a>을 통해 유료 방송 <a href="/wiki/QTV_(%ED%85%94%EB%A0%88%EB%B9%84%EC%A0%84_%EC%B1%84%EB%84%90)" title="QTV (텔레비전 채널)">QTV</a>(현 <a href="/wiki/JTBC2" title="JTBC2">JTBC2</a>), J 골프(현 <a class="mw-redirect" href="/wiki/JTBC_%EA%B3%A8%ED%94%84" title="JTBC 골프">JTBC 골프</a>) 등을 운영하였고, 지면을 통해 TBC에 대한 기획 기사를 수차례 게재하였다.</p>
<p>2009년 7월 22일에 <a href="/wiki/%EB%8C%80%ED%95%9C%EB%AF%BC%EA%B5%AD_%EB%AF%B8%EB%94%94%EC%96%B4_%EA%B4%80%EB%A0%A8%EB%B2%95_%EA%B0%9C%EC%A0%95_%EB%85%BC%EB%9E%80" title="대한민국 미디어 관련법 개정 논란">여러 차례 논란</a> 끝에 종합편성채널에 법적 근거를 부여하는 방송법 개정안이 당시 여당인 한나라당의 단독 강행 처리로 국회를 통과했는데, 이는 야당과 시민사회단체로부터 "날치기"라고 비판 받고, 일사부재의의 원칙 위배 논란이 불거졌다. 이후 <a href="/wiki/2010%EB%85%84" title="2010년">2010년</a> <a href="/wiki/11%EC%9B%94_30%EC%9D%BC" title="11월 30일">11월 30일</a>에 종합편성채널의 신청서 접수가 시작되었을 때, 중앙일보 측은 동양방송의 영문 약칭인 TBC라는 이름으로 신청을 시도하려 했으나, 해당 약칭을 1990년대부터 대구•경북 지역의 <a class="mw-redirect" href="/wiki/%EC%84%9C%EC%9A%B8%EB%B0%A9%EC%86%A1" title="서울방송">SBS</a> 계열 지역민영방송사인 대구방송(Taegu Broadcasting Corporation, 현 <a class="new" href="/w/index.php?title=%EC%A3%BC%EC%8B%9D%ED%9A%8C%EC%82%AC_%ED%8B%B0%EB%B9%84%EC%94%A8&amp;action=edit&amp;redlink=1" title="주식회사 티비씨 (없는 문서)">주식회사 티비씨</a>)이 선점하여 jTBC(소문자 j)라는 이름으로 신청서를 제출하였다. (이후 중앙일보 측은 J를 뺀 TBC 상표를 사용하고자 해당 상표를 보유한 대구방송과 여러 번 접촉했으나, 대구방송 측이 거절했다.<sup class="reference" id="cite_ref-2"><a href="#cite_note-2">[2]</a></sup>) 2010년 12월 31일에 종합편성 방송채널사업자로 선정된 이후, 2011년 3월 21일에 법인이 설립되어 같은 해 개국을 앞두고 라틴 문자 표기를 고쳐 JTBC(대문자 J)로 12월 1일에 개국하였다. 중앙일보는 여러 통로를 통해 JTBC가 동양방송의 후신임을 주장하고 있으며, JTBC는 개국 당일에 동양방송에 대한 다큐멘터리를 방송하는 등 이를 강조하는 모습을 보였다.</p>
<h3><span class="mw-headline" id=".ED.98.B8.ED.8F.89.EA.B3.BC_.EA.B3.BC.EB.8F.84.ED.95.9C_.ED.88.AC.EC.9E.90">호평과 과도한 투자</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=JTBC&amp;action=edit&amp;section=3" title="부분 편집: 호평과 과도한 투자">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p><a href="/wiki/2012%EB%85%84" title="2012년">2012년</a> <a href="/wiki/6%EC%9B%94" title="6월">6월</a>부터 <a href="/wiki/2014%EB%85%84_FIFA_%EC%9B%94%EB%93%9C%EC%BB%B5_%EC%95%84%EC%8B%9C%EC%95%84_%EC%A7%80%EC%97%AD_%EC%B5%9C%EC%A2%85_%EC%98%88%EC%84%A0" title="2014년 FIFA 월드컵 아시아 지역 최종 예선">2014년 FIFA 월드컵 아시아 지역 최종 예선</a> A조 중계를 시작으로 스포츠 중계를 개시하였으며 <a href="/wiki/2013%EB%85%84" title="2013년">2013년</a> <a href="/wiki/%EC%9B%94%EB%93%9C_%EB%B2%A0%EC%9D%B4%EC%8A%A4%EB%B3%BC_%ED%81%B4%EB%9E%98%EC%8B%9D" title="월드 베이스볼 클래식">WBC</a> 중계권도 맡으며 스포츠중계도 본격적으로 활성화하였다.<sup class="reference" id="cite_ref-3"><a href="#cite_note-3">[3]</a></sup> 당시 축구중계로 4개의 종합편성채널 중 최고 시청률인 7.5%를 달성하였다.<sup class="reference" id="cite_ref-4"><a href="#cite_note-4">[4]</a></sup></p>
<p>시사 교양에 치중하고 있는 다른 종편에 비하여 JTBC는 비교적 예능과 드라마 그리고 스포츠 분야에 치중하는 모습이였다.<sup class="reference" id="cite_ref-5"><a href="#cite_note-5">[5]</a></sup><sup class="reference" id="cite_ref-6"><a href="#cite_note-6">[6]</a></sup> 현재까지, 종편 중 유일하게 정규 드라마를 편성하고 있다.<sup class="reference" id="cite_ref-7"><a href="#cite_note-7">[7]</a></sup> 그러다 <a href="/wiki/2013%EB%85%84_5%EC%9B%94" title="2013년 5월">2013년 5월</a>, <a href="/wiki/%EC%86%90%EC%84%9D%ED%9D%AC" title="손석희">손석희</a>를 보도부문사장으로 영입하고,<sup class="reference" id="cite_ref-8"><a href="#cite_note-8">[8]</a></sup> 그를 저녁 종합 뉴스 프로그램인 <a class="mw-redirect" href="/wiki/JTBC_%EB%89%B4%EC%8A%A4_9" title="JTBC 뉴스 9">JTBC 뉴스 9</a>의 진행자로 내세워<sup class="reference" id="cite_ref-9"><a href="#cite_note-9">[9]</a></sup> 보도부문에서도 호평을 받고 있다. 그리고 2014년 이뤄진 언론 신뢰도/영향력 평가에서 JTBC가 KBS를 제치고 대한민국 국내 방송사 중 1위를 차지 하였으며, 한때 손석희 보도부문 사장이 몸담았던 MBC는 하위권에 머무르는 모양새를 보였다.<sup class="reference" id="cite_ref-10"><a href="#cite_note-10">[10]</a></sup></p>
<p><a href="/wiki/2013%EB%85%84" title="2013년">2013년</a> <a href="/wiki/11%EC%9B%94_15%EC%9D%BC" title="11월 15일">11월 15일</a> 현재, <a class="mw-redirect" href="/wiki/%EB%B0%A9%EC%86%A1%ED%86%B5%EC%8B%A0%EC%9C%84%EC%9B%90%ED%9A%8C" title="방송통신위원회">방송통신위원회</a> 전체회의에서 의결된 2012년도 방송평가에 따르면 SBS와 <a class="mw-redirect" href="/wiki/MBC" title="MBC">MBC</a>를 제치고 지상파 포함 3위를 기록하였으며,<sup class="reference" id="cite_ref-11"><a href="#cite_note-11">[11]</a></sup> 여론조사전문기관의 여론조사에서도 방송 신뢰도가 지상파채널인 MBC와 SBS를 추월했다.<sup class="reference" id="cite_ref-12"><a href="#cite_note-12">[12]</a></sup> 그러나, 과도한 투자로 인해 2012년 영업손실은 1397억원으로 종합편성채널과 보도전문편성 사업자들 중 가장 높은 것으로 집계되었다.<sup class="reference" id="cite_ref-13"><a href="#cite_note-13">[13]</a></sup></p>

<p><a href="/wiki/2011%EB%85%84" title="2011년">2011년</a> <a href="/wiki/11%EC%9B%94_7%EC%9D%BC" title="11월 7일">11월 7일</a> 슬로건과 함께 CI를 공개했다. CI는 무지개의 다채로운 색상을 모티브로 했다. 이는 특정한 색, 편협한 시각에 얽매이지 않는 창조성과 다양성을 상징한다고 밝혔다. 공개된 CI에 따라 영문사명을 기존에 쓰였던 jTBC 대신 모두 대문자인 JTBC로 사용한다고도 밝혔다.<sup class="reference" id="cite_ref-22"><a href="#cite_note-22">[22]</a></sup></p>
<h2><span class="mw-headline" id=".EB.B0.A9.EC.86.A1_.EC.82.AC.EA.B3.A0">방송 사고</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=JTBC&amp;action=edit&amp;section=13" title="부분 편집: 방송 사고">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<ul>
<li><a href="/wiki/2012%EB%85%84" title="2012년">2012년</a> <a href="/wiki/12%EC%9B%94_19%EC%9D%BC" title="12월 19일">12월 19일</a>, 제18대 대통령선거 특집 방송 중 “문재인 당선자”라고 표현했다가 급히 사과한 사건이 있었다.<sup class="reference" id="cite_ref-23"><a href="#cite_note-23">[23]</a></sup></li>
<li><a href="/wiki/2014%EB%85%84" title="2014년">2014년</a> <a href="/wiki/4%EC%9B%94_16%EC%9D%BC" title="4월 16일">4월 16일</a>, <a href="/wiki/%EC%84%B8%EC%9B%94%ED%98%B8_%EC%B9%A8%EB%AA%B0_%EC%82%AC%EA%B3%A0" title="세월호 침몰 사고">세월호 침몰 사고</a> 속보 중 구조된 학생과 전화 인터뷰 중 박진규 앵커가 "친구가 사망했다는 사실을 알고 있느냐?"라고 질문하여 비난을 받았고<sup class="reference" id="cite_ref-24"><a href="#cite_note-24">[24]</a></sup>, 이후 <a href="/wiki/%EC%86%90%EC%84%9D%ED%9D%AC" title="손석희">손석희</a> 보도담당 사장이 후배 기자의 인터뷰 논란에 대해 공식 사과했다.<sup class="reference" id="cite_ref-25"><a href="#cite_note-25">[25]</a></sup></li>
<li><a href="/wiki/2014%EB%85%84" title="2014년">2014년</a> <a href="/wiki/10%EC%9B%94_27%EC%9D%BC" title="10월 27일">10월 27일</a>, <a href="/wiki/%EB%B9%84%EC%A0%95%EC%83%81%ED%9A%8C%EB%8B%B4" title="비정상회담">비정상회담</a> 방송 중, 일본인 출연자의 배경음악으로 일본제국주의의 산물인 기미가요를 틀어주어, 방송폐지를 요구하면서 각종 게시판에 분노의 글들이 올라왔다.</li>
</ul>
