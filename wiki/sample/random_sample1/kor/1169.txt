<p><b>JYP 엔터테인먼트</b>(<span style="font-size: smaller;"><a href="/wiki/%EC%98%81%EC%96%B4" title="영어">영어</a>:</span> <span lang="en" xml:lang="en">JYP Entertainment Corporation</span>)는 <a href="/wiki/%EB%8C%80%ED%95%9C%EB%AF%BC%EA%B5%AD" title="대한민국">대한민국</a>의 종합 엔터테인먼트 회사이다. 회사 이름은 창립자인 <a href="/wiki/%EB%B0%95%EC%A7%84%EC%98%81" title="박진영">박진영</a>의 영문 이름 이니셜을 따 지어졌다. 대표적인 출신 <a href="/wiki/%EC%97%B0%EC%98%88%EC%9D%B8" title="연예인">연예인</a>으로 <a href="/wiki/%EC%A7%84%EC%A3%BC_(%EA%B0%80%EC%88%98)" title="진주 (가수)">진주</a>, <a href="/wiki/God" title="God">god</a>, <a href="/wiki/%EB%B9%84_(%EA%B0%80%EC%88%98)" title="비 (가수)">비</a>, <a href="/wiki/%EB%B0%95%EC%A7%80%EC%9C%A4_(%EA%B0%80%EC%88%98)" title="박지윤 (가수)">박지윤</a>, <a href="/wiki/%EC%9B%90%EB%8D%94%EA%B1%B8%EC%8A%A4" title="원더걸스">원더걸스</a>, <a href="/wiki/2PM" title="2PM">2PM</a>, <a href="/wiki/%EB%AF%B8%EC%93%B0%EC%97%90%EC%9D%B4" title="미쓰에이">미쓰에이</a>, <a href="/wiki/GOT7" title="GOT7">GOT7</a>, <a href="/wiki/%ED%8A%B8%EC%99%80%EC%9D%B4%EC%8A%A4" title="트와이스">트와이스</a>, <a href="/wiki/DAY6" title="DAY6">DAY6</a>, <a href="/wiki/15%26" title="15&amp;">15&amp;</a> 등이 있다. 연예인 신인 발굴, 연예인 매니지먼트, 음반 기획 및 제작 분야에 진출해 있다.</p>

<h3><span class="mw-headline" id="JYP_.EC.97.94.ED.84.B0.ED.85.8C.EC.9D.B8.EB.A8.BC.ED.8A.B8">JYP 엔터테인먼트</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=JYP_%EC%97%94%ED%84%B0%ED%85%8C%EC%9D%B8%EB%A8%BC%ED%8A%B8&amp;action=edit&amp;section=1" title="부분 편집: JYP 엔터테인먼트">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p><a href="/wiki/1997%EB%85%84" title="1997년">1997년</a> <a href="/wiki/12%EC%9B%94_1%EC%9D%BC" title="12월 1일">12월 1일</a> 연예기획사인 (주)태홍기획을 <a href="/wiki/%EB%B0%95%EC%A7%84%EC%98%81" title="박진영">박진영</a>이 설립했다. 가수 <a href="/wiki/%EC%A7%84%EC%A3%BC_(%EA%B0%80%EC%88%98)" title="진주 (가수)">진주</a>를 시작으로 <a href="/wiki/God" title="God">god</a>, <a href="/wiki/%EB%B0%95%EC%A7%80%EC%9C%A4_(%EA%B0%80%EC%88%98)" title="박지윤 (가수)">박지윤</a> 등 거대 가수들을 속속히 배출하며 프로듀서로서의 입지를 굳혔다. <a href="/wiki/2001%EB%85%84" title="2001년">2001년</a> 4월 박진영의 영문 이니셜을 따 (주)제이와이피엔터테인먼트로 사명을 변경했다. 이후 가수 이예린, 박기영, 린, 김동률 등을 매니지먼트했던 홍승성과 동업하여 <a href="/wiki/%EB%B9%84_(%EA%B0%80%EC%88%98)" title="비 (가수)">비</a>, <a href="/wiki/%EB%B3%84_(%EA%B0%80%EC%88%98)" title="별 (가수)">별</a>, <a href="/wiki/%EB%85%B8%EC%9D%84_(%EC%9D%8C%EC%95%85_%EA%B7%B8%EB%A3%B9)" title="노을 (음악 그룹)">노을</a>, <a href="/wiki/%EC%9E%84%EC%A0%95%ED%9D%AC" title="임정희">임정희</a>, <a href="/wiki/%EC%9B%90%EB%8D%94%EA%B1%B8%EC%8A%A4" title="원더걸스">원더걸스</a>, <a href="/wiki/JOO" title="JOO">JOO</a>, <a href="/wiki/2AM" title="2AM">2AM</a>, <a href="/wiki/2PM" title="2PM">2PM</a> 등을 배출하였고, <a href="/wiki/2008%EB%85%84" title="2008년">2008년</a> 홍승성 대표와 결별한 이후에도 <a href="/wiki/%EB%AF%B8%EC%93%B0%EC%97%90%EC%9D%B4" title="미쓰에이">미쓰에이</a>, <a class="mw-redirect" href="/wiki/JJ_Project" title="JJ Project">JJ 프로젝트</a>, <a href="/wiki/%EB%B0%B1%EC%95%84%EC%97%B0" title="백아연">백아연</a>, <a href="/wiki/15%26" title="15&amp;">15&amp;</a>, <a href="/wiki/GOT7" title="GOT7">GOT7</a>, <a href="/wiki/DAY6" title="DAY6">DAY6</a>, <a href="/wiki/%ED%8A%B8%EC%99%80%EC%9D%B4%EC%8A%A4" title="트와이스">트와이스</a> 등의 가수들을 꾸준히 배출하였다.</p>
<p><a href="/wiki/2010%EB%85%84" title="2010년">2010년</a> 12월 <a href="/wiki/%EB%B0%95%EC%A7%84%EC%98%81" title="박진영">박진영</a>과 JYP 이사진들이 <a href="/wiki/%EC%BD%94%EC%8A%A4%EB%8B%A5" title="코스닥">코스닥</a> 상장사인 제이튠 엔터테인먼트 <a href="/wiki/%EC%A3%BC%EC%8B%9D" title="주식">주식</a>을 대량 매입하여 최대주주로 올라섰다.<sup class="reference" id="cite_ref-3"><a href="#cite_note-3">[3]</a></sup> <a href="/wiki/%EB%B0%95%EC%A7%84%EC%98%81" title="박진영">박진영</a>과 <a href="/wiki/%EB%AF%B8%EC%93%B0%EC%97%90%EC%9D%B4" title="미쓰에이">미쓰에이</a>는 제이튠 엔터테인먼트로 이적하였고, 2011년 2월 기존의 (주)제이와이피엔터테인먼트 사명을 (주)제이와이피로 변경했으며 (주)제이튠엔터테인먼트의 사명을 (주)제이와이피엔터테인먼트로 변경하였다. 이로써 JYP는 우회상장 방식을 통해 설립 이래 처음으로 코스닥 시장에 입성하게 되었다. 이후 한동안은 <a href="/wiki/%EC%9B%90%EB%8D%94%EA%B1%B8%EC%8A%A4" title="원더걸스">원더걸스</a>, <a href="/wiki/2AM" title="2AM">2AM</a>, <a href="/wiki/2PM" title="2PM">2PM</a> 등 기존 가수들이 소속된 비상장사 ㈜제이와이피와 <a href="/wiki/%EB%B0%95%EC%A7%84%EC%98%81" title="박진영">박진영</a>, <a href="/wiki/%EB%AF%B8%EC%93%B0%EC%97%90%EC%9D%B4" title="미쓰에이">미쓰에이</a> 등이 소속된 상장사 ㈜제이와이피엔터테인먼트 두 체제로 운영되었다. <a href="/wiki/2013%EB%85%84" title="2013년">2013년</a> 6월 두 회사의 합병 계약이 체결되었고 같은 해 10월에 ㈜제이와이피엔터테인먼트에 피흡수합병된 후 폐업하였다.<sup class="reference" id="cite_ref-4"><a href="#cite_note-4">[4]</a></sup><sup class="reference" id="cite_ref-5"><a href="#cite_note-5">[5]</a></sup></p>
<h3><span class="mw-headline" id=".EC.A0.9C.EC.9D.B4.ED.8A.A0.EC.97.94.ED.84.B0.ED.85.8C.EC.9D.B8.EB.A8.BC.ED.8A.B8">제이튠엔터테인먼트</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=JYP_%EC%97%94%ED%84%B0%ED%85%8C%EC%9D%B8%EB%A8%BC%ED%8A%B8&amp;action=edit&amp;section=2" title="부분 편집: 제이튠엔터테인먼트">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p><a href="/wiki/1996%EB%85%84" title="1996년">1996년</a> <a href="/wiki/4%EC%9B%94_25%EC%9D%BC" title="4월 25일">4월 25일</a> 반도체 생산 업체 (주)보광미디어로 설립되었다. 1999년 1월 29일 아라리온(주)로 상호명을 변경했고, 2006년 6월 26일 무선통신기기 업체인 (주)세이텍을 흡수합병한 뒤 (주)세이텍으로 상호를 변경하였다. <a href="/wiki/2007%EB%85%84" title="2007년">2007년</a> <strong class="selflink">JYP 엔터테인먼트</strong>와의 계약이 만료된 이후 레이니엔터테인먼트를 설립하고 독자적 활동을 하고 있던 가수 <a href="/wiki/%EB%B9%84_(%EA%B0%80%EC%88%98)" title="비 (가수)">비</a>가 이 회사의 최대주주가 되었고, 같은 해 <a href="/wiki/11%EC%9B%94_2%EC%9D%BC" title="11월 2일">11월 2일</a> (주)제이튠엔터테인먼트로 상호명이 변경과 매니저업으로 업종을 변경하였다. 이후 탤런트 <a href="/wiki/%EC%97%B0%EC%A0%95%ED%9B%88" title="연정훈">연정훈</a>을 영입하였고 남성그룹 <a href="/wiki/%EC%97%A0%EB%B8%94%EB%9E%99" title="엠블랙">엠블랙</a>을 배출했다. 그러나 <a href="/wiki/2010%EB%85%84" title="2010년">2010년</a> <a href="/wiki/7%EC%9B%94" title="7월">7월</a> <a href="/wiki/%EB%B9%84_(%EA%B0%80%EC%88%98)" title="비 (가수)">비</a>는 보유주식 350만주를 전량 장내 매도하였다. 그 해 12월에는 박진영과 JYP 이사진들이 코스닥 상장사인 제이튠 엔터테인먼트 주식을 대량 매입하여 최대주주로 올라섰고 제이튠 엔터테인먼트의 사명을 JYP Ent.로 변경하였다. 2013년 10월 JYP 엔터테인먼트는 정식 합병절차를 밟아 JYP Ent.와의 합병을 완료하였다.</p>
