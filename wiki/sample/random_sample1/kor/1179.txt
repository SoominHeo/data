<p><b>K1 전차</b>는 <a href="/wiki/%EB%8C%80%ED%95%9C%EB%AF%BC%EA%B5%AD_%EC%9C%A1%EA%B5%B0" title="대한민국 육군">대한민국 육군</a>이 운용하는 3세대 주력전차이다. <a href="/wiki/1987%EB%85%84" title="1987년">1987년</a> 처음 공개 되었을 당시 <a href="/wiki/%EC%A0%84%EB%91%90%ED%99%98" title="전두환">전두환</a> 대통령이 88전차로 명명하여 <b>88 전차</b> , <b>K1 88전차</b>이라고도 불린다. 개량형으로 <a href="/wiki/K1A1_%EC%A0%84%EC%B0%A8" title="K1A1 전차">K1A1 전차</a>가 있다. 숫자 88은 제 24회 서울 올림픽 개최년도인 1988년의 88이다.</p>

<h2><span class="mw-headline" id=".EA.B0.9C.EB.B0.9C">개발</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=K1_%EC%A0%84%EC%B0%A8&amp;action=edit&amp;section=1" title="부분 편집: 개발">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>1970년대, <a class="mw-redirect" href="/wiki/%EB%B6%81%ED%95%9C" title="북한">북한</a>이 <a href="/wiki/T-72" title="T-72">T-72</a>를 대량 보유했다는 첩보에 동일한 성능을 가진 <a href="/wiki/M60_%ED%8C%A8%ED%8A%BC" title="M60 패튼">M60 패튼</a>의 라이센스 생산을 미국에 요구했지만 미국이 거절하자 <a href="/wiki/%EC%84%9C%EB%8F%85" title="서독">서독</a>의 크라우스 마파이 사와 신형 전차를 개발하기로 합의하였다. 합의서 서명단계까지 갔으나 미국측이 ROKIT 사업 제안을 들고 나왔고, 결국 당시 <a href="/wiki/%EB%AF%B8%EA%B5%AD" title="미국">미국</a> 신형 3세대 전차인 <a href="/wiki/M1_%EC%97%90%EC%9D%B4%EB%B8%8C%EB%9E%8C%EC%8A%A4" title="M1 에이브람스">M1 에이브람스</a>와 유사한 형태와 성능으로 개발이 되었다. 대한민국 육군이 작성한 작전요구성능(ROC; Requirement for Operational Capabilities)을 토대로 <a href="/wiki/1982%EB%85%84" title="1982년">1982년</a>부터 <a class="mw-redirect" href="/wiki/%EC%A0%9C%EB%84%88%EB%9F%B4_%EB%8B%A4%EC%9D%B4%EB%82%98%EB%AF%B9%EC%8A%A4" title="제너럴 다이나믹스">제너럴 다이나믹스</a> 지상부문(GDLS)에서 개발을 담당하고 나머지는 한국의 연구소에서 개발해 <a href="/wiki/1986%EB%85%84" title="1986년">1986년</a> 생산했다.</p>
<p>1970년대 말 <a class="mw-redirect" href="/wiki/%EB%B6%81%ED%95%9C" title="북한">북한</a>이 <a href="/wiki/T-72" title="T-72">T-72</a> 대량 보유 첩보를 들으면서 개발이 거론되었으며<sup class="reference" id="cite_ref-1"><a href="#cite_note-1">[1]</a></sup> 1983년 12월에 <a href="/wiki/%EC%A0%9C%EB%84%88%EB%9F%B4_%EB%8B%A4%EC%9D%B4%EB%82%B4%EB%AF%B9%EC%8A%A4" title="제너럴 다이내믹스">제너럴 다이내믹스</a>사에서 시제품이 출고되었다.<sup class="reference" id="cite_ref-2"><a href="#cite_note-2">[2]</a></sup> M1 전차를 기반으로 제작, 한국의 지형에 맞게 소형화되고 사격통제장치와 현수장치가 개량되었다. 초기형의 경우 <a href="/wiki/M1_%EC%97%90%EC%9D%B4%EB%B8%8C%EB%9E%8C%EC%8A%A4" title="M1 에이브람스">M1 에이브람스</a> 전차의 장갑재와 유사한 복합장갑(SAP; Special Armour Plate)을 사용하여 400~500mm 수준의 방어력을 보였으나 후기 생산분은 <a href="/wiki/K1A1_%EC%A0%84%EC%B0%A8" title="K1A1 전차">K1A1 전차</a>에 적용된 대한민국에서 만들어진 신형 복합장갑(KSAP; Korean Spacial Armour Plate)을 탑재, 방호성능이 600mm급으로 K1A1와 동등한 수준으로 강화되었다.</p>
<p>K-1 전차 업건 사업으로 진행된 사업은 K-1A1으로 진행되었으며 120mm 활강포 탑재를 위해 포탑 일부가 재설계되었고 사격통제장치와 광학장치가 대폭 교체되었으며 현재 K-1전차는 K-1E1이라는 제식명으로 K-1A1전차에 준하는 수준의 개량이 진행되고 있다.</p>
<h2><span class="mw-headline" id="K1_.EC.A0.84.EC.B0.A8.EC.9D.98_.EC.84.B1.EB.8A.A5">K1 전차의 성능</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=K1_%EC%A0%84%EC%B0%A8&amp;action=edit&amp;section=2" title="부분 편집: K1 전차의 성능">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>K1 전차는 <a href="/wiki/%EC%98%81%EA%B5%AD" title="영국">영국</a>의 <a class="new" href="/w/index.php?title=%EB%A1%9C%EC%97%B4_%EC%98%A4%EB%93%9C%EB%84%8C%EC%8A%A4&amp;action=edit&amp;redlink=1" title="로열 오드넌스 (없는 문서)">로열 오드넌스</a>의 L7 105mm 강선포를 채용하고 있다. 초기에 사용하던 K-270 APFSDS탄은 370mm의 관통력을 보유했으나 북한이 T-72를 보유했다는 첩보를 입수한 뒤 470mm 급인 K-274를 개발하였다. 최근 K274N 날탄이 개발되어 120mm 활강포의 초기 관통자인 DM-33을 넘어선 관통력을 달성한 것으로 추정된다. 사격통제장치는 360도 전면감시가 가능하고 전차장이 조준한 목표물에 포탑을 조절할 수 있는 SFIM(현 SAGEM)차장조준경과 정확히 조준할 수 있는 GPS(레이시온 &amp; Ti)포수조준경을 갖추고 있다. 엔진은 독일제 1200마력 수랭식 4행정 디젤엔진 (MTU Mb871 Ka-501)을 사용하며 최고속력은 65km/h 이다. 특이한 점은 유도륜과 1,2번 보기륜과 6번, 기동륜에 유기압 현수장치가 있어 전차를 숙일수도 있고 반대로 전차 앞부분을 치켜 올리는 자세도 가능하다.</p>
<h2><span class="mw-headline" id="K1A1_.EC.A0.84.EC.B0.A8_.EA.B0.9C.EB.B0.9C.EA.B3.BC_.EB.B0.A9.EC.96.B4.EB.A0.A5_.EA.B0.9C.EC.84.A0">K1A1 전차 개발과 방어력 개선</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=K1_%EC%A0%84%EC%B0%A8&amp;action=edit&amp;section=3" title="부분 편집: K1A1 전차 개발과 방어력 개선">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>한국은 러시아 차관을 현금 대신 현물로 갚는 불곰사업 때 <a href="/wiki/%EB%9F%AC%EC%8B%9C%EC%95%84" title="러시아">러시아</a> 최신 3세대 전차인 <a class="mw-redirect" href="/wiki/T-80U" title="T-80U">T-80U</a> 33대와 지휘차량인 <a class="mw-redirect" href="/wiki/T-80U" title="T-80U">T-80UK</a>을 수입하였다. 수입 후 육군기계화학교에서 그 성능과 장갑재 구조를 해부하였으며 K1A1는 1990년대 육군 내부 연구지에서 제기된 전차전력 강화의 방법으로 제시된 120mm 활강포 탑재 신형전차에 대한 연구와 이후 도입된 T-80U 수입의 영항과 북한의 <a href="/wiki/T-72" title="T-72">T-72</a> 대량으로 입수했다는 첩보로 인해 120mm 활강포 탑재 전차의 소요제기로 인해 개발하였으며 <a href="/wiki/1998%EB%85%84" title="1998년">1998년</a>에 시재품을 출고하고 <a href="/wiki/2001%EB%85%84" title="2001년">2001년</a>부터 생산을 시작해 현재 전력화가 진행되고 있다.</p>
