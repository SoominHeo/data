<p><b>KLDP</b>(<span lang="en" xml:lang="en">Korean Linux Documentation Project</span>)는 한국인 리눅스 사용자들을 중심으로 이루어진 오픈소스, 자유소프트웨어 프로젝트 사이트, 또는 그곳에 모인 공동체를 가리킨다. 공식 번역명은 없으며 리눅스 한글문서 프로젝트, 리눅스 문서 한글화 프로젝트 등으로 불린다.<a class="external autonumber" href="https://wiki.kldp.org/wiki.php/KLDP" rel="nofollow">[1]</a></p>
<p>1996년 10월 5일에 <a class="new" href="/w/index.php?title=%EA%B6%8C%EC%88%9C%EC%84%A0&amp;action=edit&amp;redlink=1" title="권순선 (없는 문서)">권순선</a>의 개인 홈페이지로 출발하여 리눅스 관련 문서를 한글로 번역해서 인터넷으로 제공하는 것을 주 활동으로 운영되었고, 현재는 문서화뿐만 아니라 커뮤니티, 개발자 공간, 프로젝트 호스팅 등 다양한 활동들을 진행해 나가고 있다. <sup class="reference" id="cite_ref-1"><a href="#cite_note-1">[1]</a></sup></p>
<p>이름에서 알 수 있듯이 범세계적인 리눅스 공동체의 <a href="/wiki/%EB%A6%AC%EB%88%85%EC%8A%A4_%EB%AC%B8%EC%84%9C%ED%99%94_%ED%94%84%EB%A1%9C%EC%A0%9D%ED%8A%B8" title="리눅스 문서화 프로젝트">리눅스 문서화 프로젝트</a>의 한국어판 프로젝트로 시작되어, "리눅스 한글문서 프로젝트", "리눅스 문서 한글화 프로젝트" 등의 이름으로 불리었으나, 시간이 흐르면서 <b>문서화</b>에 한정되지 않고 오픈소스 전반으로 활동 영역을 넓히고, 한국어 사용 리눅스 사용자들의 자랑스러운 사랑방으로 기능하고 있다.</p>

<li>1997년 : 관리자가 진학을 하면서 주소가 http://www.postech.ac.kr/~cessi 로 바뀜</li>
<li>1998년 : URL을 http://kldp.linux-kr.org 로 바꿈</li>
<li>1998년 : 시스템공학연구소(구 SERI) 슈퍼컴퓨팅센터의 후원으로 계정을 얻어 운영</li>
<li>1999년 : URL을 <a class="external free" href="http://kldp.org" rel="nofollow">http://kldp.org</a> 로 바꿈</li>

<li>2000년 8월 : <a href="/wiki/%EB%8D%B0%EB%B9%84%EC%95%88" title="데비안">데비안</a> GNU/Linux 2.2(potato) 시디롬 제작/배포</li>
<li>2000년 11월 20일 : 오픈소스 뉴스/토론 사이트 <a class="external text" href="http://geekforum.kldp.org" rel="nofollow">geekforum</a> 개설</li>
<li>2002년 7월 5일 : 오픈소스 개발자용 사이트 KLDP.net 개설</li>
<li>2002년 12월 17일 : 커뮤니티 사이트 KLDP BBS 개설</li>
<li>2003년 6월 27일 : KLDP blog 개설</li>
<li>2003년 10월 11일 : KLDPconf : 작은 세미나 개최 시작</li>
<li>2003년 11월 24일 : FreeSoftware, OpenSource 개발자용 사이트 KLDP.net 기능 업그레이드, 개선</li>
<li>2004년 7월 17일 ~ 18일 : 제1회 <a class="external text" href="http://wiki.kldp.org/wiki.php/CodeFest" rel="nofollow">CodeFest</a> 개최</li>
<li>2004년 10월 30일 ~ 31일 : 제2회 CodeFest 개최</li>
<li>2004년 11월 : HDD 손상으로 사이트 전면 재보수</li>
<li>2004년 12월 5일 ~ 6일 : 제3회 CodeFest 개최</li>
<li>2005년 2월 18일 ~ 20일 : 제4회 CodeFest 개최</li>
<li>2005년 4월 27일 : 회선 후원을 <a class="external text" href="http://www.cdnetworks.co.kr" rel="nofollow">씨디네트웍스</a>로 변경하고 IDC를 이전함</li>
<li>2005년 7월 23일 ~ 24일 : 제5회 CodeFest 개최</li>
<li>2005년 11월 8일 : <a class="external text" href="http://trust.daum.net/daum/main.htm" rel="nofollow">다음 정보 트러스트 어워드</a> 수상</li>
