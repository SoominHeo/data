<p><b>LATAM 카고 칠레</b>(<span style="font-size: smaller;"><a href="/wiki/%EC%98%81%EC%96%B4" title="영어">영어</a>:</span> <span lang="en" xml:lang="en">LATAM Cargo Chile</span>)는 <a href="/wiki/%EC%B9%A0%EB%A0%88" title="칠레">칠레</a>의 화물 항공사로 본사는 <a href="/wiki/%EB%AF%B8%EA%B5%AD" title="미국">미국</a> <a href="/wiki/%ED%94%8C%EB%A1%9C%EB%A6%AC%EB%8B%A4_%EC%A3%BC" title="플로리다 주">플로리다 주</a> <a href="/wiki/%EB%A7%88%EC%9D%B4%EC%95%A0%EB%AF%B8" title="마이애미">마이애미</a>에 위치해 있으며 <a href="/wiki/1929%EB%85%84" title="1929년">1929년</a>에 설립했다. <a href="/wiki/%ED%97%88%EB%B8%8C_%EA%B3%B5%ED%95%AD" title="허브 공항">허브 공항</a>으로 <a href="/wiki/%EC%BD%94%EB%AA%A8%EB%8F%84%EB%A1%9C_%EC%95%84%EB%A5%B4%ED%88%AC%EB%A1%9C_%EB%A9%94%EB%A6%AC%EB%85%B8_%EB%B2%A0%EB%8B%88%ED%85%8C%EC%8A%A4_%EA%B5%AD%EC%A0%9C%EA%B3%B5%ED%95%AD" title="코모도로 아르투로 메리노 베니테스 국제공항">코모도로 아르투로 메리노 베니테스 국제공항</a>, <a href="/wiki/%EB%A7%88%EC%9D%B4%EC%95%A0%EB%AF%B8_%EA%B5%AD%EC%A0%9C%EA%B3%B5%ED%95%AD" title="마이애미 국제공항">마이애미 국제공항</a>이 있으며 계열사는 <a href="/wiki/%EB%AF%B8%EA%B5%AD" title="미국">미국</a>의 화물 항공사인 <a href="/wiki/%ED%94%8C%EB%A1%9C%EB%A6%AC%EB%8B%A4_%EC%9B%A8%EC%8A%A4%ED%8A%B8_%EA%B5%AD%EC%A0%9C%ED%95%AD%EA%B3%B5" title="플로리다 웨스트 국제항공">플로리다 웨스트 국제항공</a>을 비롯해 <a href="/wiki/ABSA_%EC%B9%B4%EA%B3%A0_%ED%95%AD%EA%B3%B5" title="ABSA 카고 항공">ABSA 카고 항공</a>, <a href="/wiki/LATAM_%EC%B9%B4%EA%B3%A0_%EC%BD%9C%EB%A1%AC%EB%B9%84%EC%95%84" title="LATAM 카고 콜롬비아">LATAM 카고 콜롬비아</a>, <a href="/wiki/LATAM_%EC%B9%B4%EA%B3%A0_%EB%A9%95%EC%8B%9C%EC%BD%94" title="LATAM 카고 멕시코">LATAM 카고 멕시코</a>가 있다.</p>

<h2><span class="mw-headline" id=".EA.B0.9C.EC.9A.94">개요</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=LATAM_%EC%B9%B4%EA%B3%A0_%EC%B9%A0%EB%A0%88&amp;action=edit&amp;section=1" title="부분 편집: 개요">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>항공사는 <a href="/wiki/LATAM_%ED%95%AD%EA%B3%B5_%EA%B7%B8%EB%A3%B9" title="LATAM 항공 그룹">LATAM 항공 그룹</a>의 화물 부문을 기반으로 계열사인 <a href="/wiki/%ED%94%8C%EB%A1%9C%EB%A6%AC%EB%8B%A4_%EC%9B%A8%EC%8A%A4%ED%8A%B8_%EA%B5%AD%EC%A0%9C%ED%95%AD%EA%B3%B5" title="플로리다 웨스트 국제항공">플로리다 웨스트 국제항공</a>을 비롯해 <a href="/wiki/ABSA_%EC%B9%B4%EA%B3%A0_%ED%95%AD%EA%B3%B5" title="ABSA 카고 항공">ABSA 카고 항공</a>, <a href="/wiki/LATAM_%EC%B9%B4%EA%B3%A0_%EC%BD%9C%EB%A1%AC%EB%B9%84%EC%95%84" title="LATAM 카고 콜롬비아">LATAM 카고 콜롬비아</a>, <a class="mw-disambig" href="/wiki/LATAM_%EC%B9%B4%EA%B3%A0" title="LATAM 카고">LATAM 카고</a> 다른 <a href="/wiki/%ED%99%94%EB%AC%BC_%ED%95%AD%EA%B3%B5%EC%82%AC" title="화물 항공사">화물 항공사</a>와 긴밀하게 협력한다.<sup class="reference" id="cite_ref-FI_2-1"><a href="#cite_note-FI-2">[2]</a></sup></p>
<p>또한 <a href="/wiki/%ED%94%8C%EB%A1%9C%EB%A6%AC%EB%8B%A4_%EC%9B%A8%EC%8A%A4%ED%8A%B8_%EA%B5%AD%EC%A0%9C%ED%95%AD%EA%B3%B5" title="플로리다 웨스트 국제항공">플로리다 웨스트 국제항공</a>의 소수 점유율을 가지고 있지만, LATAM 카고 칠레는 <a href="/wiki/%ED%94%8C%EB%A1%9C%EB%A6%AC%EB%8B%A4_%EC%9B%A8%EC%8A%A4%ED%8A%B8_%EA%B5%AD%EC%A0%9C%ED%95%AD%EA%B3%B5" title="플로리다 웨스트 국제항공">플로리다 웨스트 국제항공</a>과 어떤 방식으로 협조하지 않는다. 따라서 <a href="/wiki/%ED%94%8C%EB%A1%9C%EB%A6%AC%EB%8B%A4_%EC%9B%A8%EC%8A%A4%ED%8A%B8_%EA%B5%AD%EC%A0%9C%ED%95%AD%EA%B3%B5" title="플로리다 웨스트 국제항공">플로리다 웨스트 국제항공</a>은 주어진 설정 없이 다른 항공사와 마찬 가지로 란 카고와 계약하는 것으로 독립적으로 운영하는 회사에 속한다.</p>
