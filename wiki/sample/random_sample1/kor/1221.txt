<h2><span class="mw-headline" id=".EC.97.AD.EC.82.AC">역사</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=LG_%EC%8B%B8%EC%9D%B4%EC%96%B8&amp;action=edit&amp;section=1" title="부분 편집: 역사">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p><a href="/wiki/1995%EB%85%84" title="1995년">1995년</a>, 화통이란 브랜드를 시작으로 1996년 프리웨이, 1997년 CION을 거쳐 2000년에 CYON이 되었다.</p>
<p>초기에는 <a href="/wiki/LG%ED%85%94%EB%A0%88%EC%BD%A4" title="LG텔레콤">LG텔레콤</a> 전용 PCS 단말기만 생산했으나, 얼마 뒤 다른 통신사에도 제품을 공급하게 되었다. <a href="/wiki/2004%EB%85%84" title="2004년">2004년</a>, <a href="/wiki/%EB%8C%80%ED%95%9C%EB%AF%BC%EA%B5%AD" title="대한민국">대한민국</a> 최초로 <a class="mw-redirect" href="/wiki/%EC%A7%80%EC%83%81%ED%8C%8C_DMB" title="지상파 DMB">TDMB</a> 폰을 생산해 냈다.</p>
<p><a href="/wiki/2005%EB%85%84" title="2005년">2005년</a>에는 슬로건을 '싸이언 아이디어'로 변경, 블랙 라벨이라는 이름의 고급폰 전용 라인을 편성하여 1세대로 초콜릿, 2세대로 샤인, 3세대로 시크릿을 만들어내면서 전 세계적으로 인기를 끌었으며 <a href="/wiki/2008%EB%85%84" title="2008년">2008년</a> 초콜릿폰의 생산이 중단되었고 곧 단종되었다. <a href="/wiki/2008%EB%85%84" title="2008년">2008년</a> 들어 기존의 슬로건인 '싸이언 아이디어'를 'touch the wonder CYON'으로 변경, 터치폰을 주력 모델로 생산할 것임을 선언했다. <a href="/wiki/2006%EB%85%84" title="2006년">2006년</a> 세계 최초의 풀스크린 터치폰인 프라다폰을 시작으로 <a href="/wiki/2009%EB%85%84" title="2009년">2009년</a> <a class="mw-disambig" href="/wiki/%EB%8F%8C%EB%B9%84" title="돌비">돌비</a> 모바일과 차세대 인터페이스인 S-CLASS UI가 탑재된 아레나폰, <a href="/wiki/DivX" title="DivX">DivX</a> 플레이어를 지원하는 뉴초콜릿폰에 이르기까지 다양한 기종의 터치폰을 생산하였다.</p>

<p><a href="/wiki/2010%EB%85%84_1%EC%9B%94" title="2010년 1월">2010년 1월</a>, 일부 휴대 전화에서 문자 메시지 수신날짜의 연산오류로 인해 2016년으로 표시되는 버그가 발생하였다.<sup class="reference" id="cite_ref-1"><a href="#cite_note-1">[1]</a></sup><sup class="reference" id="cite_ref-2"><a href="#cite_note-2">[2]</a></sup></p>
<p><a href="/wiki/2011%EB%85%84" title="2011년">2011년</a> <a href="/wiki/3%EC%9B%94_2%EC%9D%BC" title="3월 2일">3월 2일</a> CYON 브랜드를 폐지하고 LG Mobile로 변경하였다. 스마트폰에는 옵티머스로(예:옵티머스 원, 옵티머스 마하, 옵티머스 2X 등으로) (사실 LG CYON 옵티머스 Q이후부터 모든 스마트폰에 CYON 로고를 사용하지 않았다, 싸이언브랜드가 2011년 3월 2일 폐지되기 전까지는 <a class="mw-redirect" href="/wiki/%EC%A7%80%EC%83%81%ED%8C%8CDMB" title="지상파DMB">지상파DMB</a>안테나에 한해 CYON로고가 찍혀있었다, 하지만 공교롭게도 싸이언브랜드 폐지이후 외장형 안테나를 사용하는 LG모바일 스마트폰은 아직 없다.)출시되고 피처폰에는 LG Mobile 쿠키폰등 제품 컨셉별로 출시된다.</p>

<h2><span class="mw-headline" id=".EC.A0.9C.ED.92.88_.EA.B5.AC.EB.B6.84">제품 구분</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=LG_%EC%8B%B8%EC%9D%B4%EC%96%B8&amp;action=edit&amp;section=3" title="부분 편집: 제품 구분">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>싸이언 제품 모델명은 다음과 같은 구조를 갖는다.</p>
<ul>
<li>LG-XY0000</li>
</ul>
<p>여기서, X는 이동 통신 회사를 의미하는데, 각 이동 통신 회사의 첫 이니셜(S: <a href="/wiki/SK%ED%85%94%EB%A0%88%EC%BD%A4" title="SK텔레콤">SK텔레콤</a> K: <a href="/wiki/KT" title="KT">KT</a> L: <a href="/wiki/LG%EC%9C%A0%ED%94%8C%EB%9F%AC%EC%8A%A4" title="LG유플러스">LG유플러스</a>)로 나타내며, 다음에 오는 글자인 Y는 단말기의 특징을 나타낸다. Y에 해당하는 글자는 다음과 같다.</p>
<ul>
<li>D: 셀룰러(800 MHz 대역을 사용하는 SK텔레콤에 해당. Digital Cellular Network의 첫 글자인 D를 표기.)</li>
<li>P: PCS(1.8 GHz 대역의 사업자인 KT와 <a href="/wiki/LG%EC%9C%A0%ED%94%8C%EB%9F%AC%EC%8A%A4" title="LG유플러스">LG유플러스</a>에 해당.)</li>
<li>C: QCIF폰(LG-KC1과 <a href="/wiki/2004%EB%85%84" title="2004년">2004년</a> 이전에 출시된 모델은 일반 QCIF폰이 아닌 스마트폰 기종임.)</li>
</ul>
<p>상기의 제품들은 VOD 기능을 지원하지 않는다.(스마트폰과 후기 SK텔레콤용 모델은 제외. SK텔레콤의 경우 모든 기종에 <a class="mw-disambig" href="/wiki/MOD" title="MOD">MOD</a>를 기본적으로 지원함.)</p>
<ul>
<li>V: VOD폰(후기에는 QVGA로 바뀌며, 이에 대응하여 D,P 시리즈 대신 C:QCIF가 나옴.)</li>
<li>B: DMB폰(초기 지상파 DMB 모델인 KD1200/LD1200 모델은 D시리즈이나 여기에 포함. 지상파 DMB와 위성 DMB를 구분짓기 위해 D와 B를 혼용한 듯하나, 뒤늦게 SK텔레콤에서도 지상파 DMB 모델이 생산됨에 따라 기존에 저가형 셀룰러 모델에 사용되고 있던 알파벳 D가 겹쳐지게 되어 이후에는 B만 사용하게 됨.)</li>
<li>H: HSDPA/HSUPA폰(LG유플러스는 <a href="/wiki/CDMA2000" title="CDMA2000">CDMA2000</a> 1x <a href="/wiki/EV-DO" title="EV-DO">EV-DO</a> <a class="new" href="/w/index.php?title=Rev.A&amp;action=edit&amp;redlink=1" title="Rev.A (없는 문서)">Rev.A</a>, 최근에 출시된 중고가형 3G 기종은 H대신 U를 사용하며, U계열은 일반 <a class="mw-redirect" href="/wiki/%ED%9C%B4%EB%8C%80%ED%8F%B0" title="휴대폰">휴대폰</a>과 스마트폰이 혼재함.)</li>
</ul>
<p>그 다음의 네 자리 숫자(0000)는 단말기 고유의 값인데, 셀룰러 대역을 사용하는 SK텔레콤의 경우 세 자리 숫자(000)이 붙고, PCS 대역을 사용하는 KT와 LG유플러스는 네 자리 숫자 그대로를 사용한다. 그러나, 간혹 LG-KC1과 같은 예외의 경우도 있다. <a href="/wiki/%EB%8C%80%ED%95%9C%EB%AF%BC%EA%B5%AD" title="대한민국">대한민국</a> 이외 지역에 판매되는 모델도 기본적으로 이름을 붙이는 방식은 동일하나 <a href="/wiki/%EB%8C%80%ED%95%9C%EB%AF%BC%EA%B5%AD" title="대한민국">대한민국</a>의 <a href="/wiki/SK%ED%85%94%EB%A0%88%EC%BD%A4" title="SK텔레콤">SK텔레콤</a>용 모델과 같이 뒷자리 고유번호는 3자리 숫자이다.</p>
