<p><b>LS그룹</b>은 <a href="/wiki/2013%EB%85%84_4%EC%9B%94" title="2013년 4월">2013년 4월</a> 기준 대한민국 재계 순위 9위(매출기준), 13위(자산기준)의 대기업으로 <a href="/wiki/2003%EB%85%84" title="2003년">2003년</a> <a href="/wiki/LG%EA%B7%B8%EB%A3%B9" title="LG그룹">LG그룹</a>에서 <a class="mw-redirect" href="/wiki/%EC%A0%84%EC%84%A0_(%EC%A0%84%EA%B8%B0)" title="전선 (전기)">전선</a>과 <a href="/wiki/%EA%B8%88%EC%86%8D" title="금속">금속</a> 부문이 계열분리하여 형성되었다. <a href="/wiki/2005%EB%85%84" title="2005년">2005년</a>부터 사용하기 시작한 기업명 LS는 리딩 솔루션(Leading Solution)의 머리 글자를 따온 것이며, ‘LG’+‘GS’의 의미를 동시에 가진다.<sup class="reference" id="cite_ref-1"><a href="#cite_note-1">[1]</a></sup> 현 회장인 <a href="/wiki/%EA%B5%AC%EC%9E%90%EC%97%B4" title="구자열">구자열</a>은 <a href="/wiki/LG%EA%B7%B8%EB%A3%B9" title="LG그룹">LG그룹</a> 회장 <a href="/wiki/%EA%B5%AC%EB%B3%B8%EB%AC%B4" title="구본무">구본무</a>의 당숙이다. <a href="/wiki/2008%EB%85%84" title="2008년">2008년</a> 본사를 기존 서울에서 안양으로 이전하였다.<sup class="reference" id="cite_ref-2"><a href="#cite_note-2">[2]</a></sup></p>
