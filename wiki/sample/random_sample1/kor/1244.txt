<p><b>Lügenmarsch</b>는 독일 <a class="mw-redirect" href="/wiki/%ED%95%98%EB%93%9C%EB%A1%9D" title="하드록">하드록</a> 밴드 <a href="/wiki/%EB%B5%88%EC%A0%9C_%EC%98%B9%EC%BC%88%EC%B8%A0" title="뵈제 옹켈츠">뵈제 옹켈츠</a>의 정규6 집 EP 앨범이다. 전작 <a href="/wiki/Kneipenterroristen" title="Kneipenterroristen">Kneipenterroristen</a>의 곡들이 상당수이고 1989년 <a class="new" href="/w/index.php?title=Metal_Enterprises&amp;action=edit&amp;redlink=1" title="Metal Enterprises (없는 문서)">Metal Enterprises</a>에서 발매되었다.</p>
<h2><span class="mw-headline" id=".EC.A0.9C.EC.9E.91.EB.B0.B0.EA.B2.BD">제작배경</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=L%C3%BCgenmarsch&amp;action=edit&amp;section=1" title="부분 편집: 제작배경">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>밴드가 소속되어 있던 레이블 <a class="new" href="/w/index.php?title=Metal_Enterprises&amp;action=edit&amp;redlink=1" title="Metal Enterprises (없는 문서)">Metal Enterprises</a>는 1989년 이 앨범을 <a class="new" href="/w/index.php?title=%ED%94%BD%EC%B2%98_%EB%94%94%EC%8A%A4%ED%81%AC&amp;action=edit&amp;redlink=1" title="픽처 디스크 (없는 문서)">픽처 디스크</a>로 발매하기로 결정했고 밴드에게 압력을 가했다. 당시 밴드는 <a class="mw-disambig" href="/wiki/%ED%94%84%EB%A1%9C%EB%93%80%EC%84%9C" title="프로듀서">프로듀서</a>이자 레이블 사장인 잉고 노보트니와의 계약을 끝내기 위해서 신보 작업에 집중하고 있었다. 결국 밴드는 압력에 못이겼고 잉고의 말대로 네 곡을 전작 <a href="/wiki/Kneipenterroristen" title="Kneipenterroristen">Kneipenterroristen</a>에서, 공개되지 않은 세 곡을 추가해서 이 앨범을 발매한다.</p>

<h2><span class="mw-headline" id=".EC.88.98.EB.A1.9D.EA.B3.A1.EC.97.90_.EB.8C.80.ED.95.9C_.EC.84.A4.EB.AA.85">수록곡에 대한 설명</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=L%C3%BCgenmarsch&amp;action=edit&amp;section=3" title="부분 편집: 수록곡에 대한 설명">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<dl>
<dt>Lügenmarsch</dt>
</dl>
<p>Könige für einen Tag과 마찬가지로 전작 <a href="/wiki/Kneipenterroristen" title="Kneipenterroristen">Kneipenterroristen</a>에 수록된 곡이다. 이 곡은 미디어에 대한 분노를 다루고 있다.</p>
<dl>
<dt>Ein guter Freund</dt>
</dl>
<p>이 곡은 영화 <a class="new" href="/w/index.php?title=Die_Drei_von_der_Tankstelle&amp;action=edit&amp;redlink=1" title="Die Drei von der Tankstelle (없는 문서)">Die Drei von der Tankstelle</a> <a class="mw-disambig" href="/wiki/OST" title="OST">OST</a>를 커버한 것이며, 추후 <a href="/wiki/Kneipenterroristen" title="Kneipenterroristen">Kneipenterroristen</a>의 <a class="mw-disambig" href="/wiki/CD" title="CD">CD</a>버전의 <a class="mw-redirect" href="/wiki/%EB%B3%B4%EB%84%88%EC%8A%A4%ED%8A%B8%EB%9E%99" title="보너스트랙">보너스트랙</a>에 수록된다.</p>
