<p><b>MAVEN</b> (<b>M</b>ars <b>A</b>tmosphere and <b>V</b>olatile <b>E</b>volutio<b>N</b>) 은 <a href="/wiki/%ED%99%94%EC%84%B1" title="화성">화성</a> <a href="/wiki/%EA%B6%A4%EB%8F%84" title="궤도">궤도</a>에 머무르며 <a href="/wiki/%ED%99%94%EC%84%B1%EC%9D%98_%EB%8C%80%EA%B8%B0" title="화성의 대기">화성의 대기</a> 조사와 연구를 진행하는 탐사선으로 <a href="/wiki/%ED%99%94%EC%84%B1%EC%9D%98_%EB%8C%80%EA%B8%B0" title="화성의 대기">화성의 대기</a>와 화성의 물이 존재하게 된 배경을 알아내는 것이 주 목적이다.<sup class="reference" id="cite_ref-NASA-20140921_3-0"><a href="#cite_note-NASA-20140921-3">[3]</a></sup><sup class="reference" id="cite_ref-NYT-20140921_4-0"><a href="#cite_note-NYT-20140921-4">[4]</a></sup><sup class="reference" id="cite_ref-NYT-20131115_5-0"><a href="#cite_note-NYT-20131115-5">[5]</a></sup><sup class="reference" id="cite_ref-SAM_6-0"><a href="#cite_note-SAM-6">[6]</a></sup><sup class="reference" id="cite_ref-FB-MAVEN_7-0"><a href="#cite_note-FB-MAVEN-7">[7]</a></sup></p>
<p>MAVEN은 <a href="/wiki/%EC%95%A0%ED%8B%80%EB%9F%AC%EC%8A%A4_V" title="애틀러스 V">애틀러스 V</a> 로켓에 탑재되어 2013년 11월 18일에 성공적으로 발사되었다. 1단 로켓 연소가 끝난 직후에 2단 로켓인 <a href="/wiki/%EC%84%BC%ED%83%80%EC%9A%B0%EB%A5%B4" title="센타우르">센타우르</a>가 점화되어 <a href="/wiki/%EC%A7%80%EA%B5%AC_%EC%A0%80%EA%B6%A4%EB%8F%84" title="지구 저궤도">지구 저궤도</a>에 진입했고 27분 정도 지구 저궤도에 머무른 이후 약 5분 동안 <a href="/wiki/%EC%84%BC%ED%83%80%EC%9A%B0%EB%A5%B4" title="센타우르">센타우르</a> 로켓을 통해 <a href="/wiki/%ED%83%9C%EC%96%91_%EC%A3%BC%ED%9A%8C_%EA%B6%A4%EB%8F%84" title="태양 주회 궤도">태양 주회 궤도</a>에서 화성으로 가는 <a href="/wiki/%ED%98%B8%EB%A7%8C_%EC%A0%84%EC%9D%B4_%EA%B6%A4%EB%8F%84" title="호만 전이 궤도">호만 전이 궤도</a>를 형성했다.</p>
<p>이후 2014년 9월 22일에 최원점 6,200 km, 최근점 150 km 인 <a href="/wiki/%ED%99%94%EC%84%B1" title="화성">화성</a> 궤도에 성공적으로 진입했다.<sup class="reference" id="cite_ref-NASA-20140921_3-1"><a href="#cite_note-NASA-20140921-3">[3]</a></sup><sup class="reference" id="cite_ref-NYT-20140921_4-1"><a href="#cite_note-NYT-20140921-4">[4]</a></sup><sup class="reference" id="cite_ref-NASA-20140917_8-0"><a href="#cite_note-NASA-20140917-8">[8]</a></sup><sup class="reference" id="cite_ref-9"><a href="#cite_note-9">[9]</a></sup></p>

MAVEN을 실은 <a href="/wiki/%EC%95%A0%ED%8B%80%EB%9F%AC%EC%8A%A4_V" title="애틀러스 V">애틀러스 V</a> 로켓의 발사 장면 (2013년 11월 18일)</div>

<p>MAVEN은 <a class="mw-disambig" href="/wiki/%ED%94%BC%EB%8B%89%EC%8A%A4" title="피닉스">피닉스 (우주선)</a> 외 많은 탐사선들과 더불어 2010년에 중단된 <a class="mw-redirect" href="/wiki/NASA" title="NASA">NASA</a>의 마스 스카웃 프로그램의 일환이였다..<sup class="reference" id="cite_ref-10"><a href="#cite_note-10">[10]</a></sup> 마스 스카웃 프로그램은 약 1억 8천 7백만 달러의 발사 비용을 제외한 총 비용을 4억 8천 5백만 비용 이내로 제한하는 것이 목표로<sup class="reference" id="cite_ref-11"><a href="#cite_note-11">[11]</a></sup> 발사 비용을 포함한 총 임무 비용은 약 6억 7천 1백만 달러였다.<sup class="reference" id="cite_ref-12"><a href="#cite_note-12">[12]</a></sup><sup class="reference" id="cite_ref-13"><a href="#cite_note-13">[13]</a></sup><sup class="reference" id="cite_ref-14"><a href="#cite_note-14">[14]</a></sup></p>
<p>2008년 9월 15일에 <a class="mw-redirect" href="/wiki/NASA" title="NASA">NASA</a>는 8가지 후보 중에서 마지막 2013 마스 스카웃 프로젝트로 MAVEN이 선정됐다고 발표했다.<sup class="reference" id="cite_ref-selection_1-1"><a href="#cite_note-selection-1">[1]</a></sup><sup class="reference" id="cite_ref-15"><a href="#cite_note-15">[15]</a></sup> 이름의 의미는 "전문가" 란 뜻의 영단어 '<a class="mw-disambig" href="/wiki/%EB%A9%94%EC%9D%B4%EB%B8%90" title="메이븐">메이븐</a>'에서 따왔다.<sup class="reference" id="cite_ref-16"><a href="#cite_note-16">[16]</a></sup><sup class="reference" id="cite_ref-17"><a href="#cite_note-17">[17]</a></sup></p>
<p>2013년 8월 2일에 MAVEN은 <a class="mw-redirect" href="/wiki/%ED%94%8C%EB%A1%9C%EB%A6%AC%EB%8B%A4" title="플로리다">플로리다</a>의 <a href="/wiki/%EC%BC%80%EB%84%A4%EB%94%94_%EC%9A%B0%EC%A3%BC_%EC%84%BC%ED%84%B0" title="케네디 우주 센터">케네디 우주 센터</a>에 수송되어 발사 준비에 들어갔고<sup class="reference" id="cite_ref-18"><a href="#cite_note-18">[18]</a></sup> 2013년 11월 18일에 <a href="/wiki/%EC%BC%80%EC%9D%B4%ED%94%84%EC%BB%A4%EB%82%B4%EB%B2%84%EB%9F%B4_%EA%B3%B5%EA%B5%B0_%EA%B8%B0%EC%A7%80" title="케이프커내버럴 공군 기지">케이프커내버럴 공군 기지</a>에서 <a href="/wiki/%EC%95%A0%ED%8B%80%EB%9F%AC%EC%8A%A4_V" title="애틀러스 V">애틀러스 V</a> 로켓에 실려 성공적으로 발사되었다.<sup class="reference" id="cite_ref-19"><a href="#cite_note-19">[19]</a></sup> <a href="/wiki/%ED%99%94%EC%84%B1" title="화성">화성</a> <a href="/wiki/%EA%B6%A4%EB%8F%84" title="궤도">궤도</a>에는 인도의 화성 탐사선인 <a href="/wiki/%EB%A7%9D%EA%B0%88%EB%A6%AC%EC%96%80" title="망갈리얀">망갈리얀</a>이 도착한 2014년 9월 24일의 이틀 전인 2014년 9월 22일에 <a href="/wiki/%ED%99%94%EC%84%B1" title="화성">화성</a> <a href="/wiki/%EA%B6%A4%EB%8F%84" title="궤도">궤도</a>에 진입했다.</p>
<p>2013년 10월 1일에 MAVEN 발사 7주를 남긴 채 <a href="/wiki/2013%EB%85%84_%EB%AF%B8%EA%B5%AD_%EC%97%B0%EB%B0%A9_%EC%A0%95%EB%B6%80_%ED%8F%90%EC%87%84" title="2013년 미국 연방 정부 폐쇄">미국 연방 정부가 폐쇄</a>되는 사건 발생하여 MAVEN 발사가 26개월 후로 지연되는 위기에 쳐했으나 2013년 11월 18일부터 같은 해 12월 7일까지 발사하지 않으면 <a href="/wiki/%ED%99%94%EC%84%B1" title="화성">화성</a>과의 거리가 멀어지게 되어 발사 시기를 놓치게 되고<sup class="reference" id="cite_ref-20"><a href="#cite_note-20">[20]</a></sup> 더불어 <a class="mw-redirect" href="/wiki/NASA" title="NASA">NASA</a>가 이틀 뒤에 현재 <a href="/wiki/%ED%99%94%EC%84%B1" title="화성">화성</a> <a class="mw-redirect" href="/wiki/%EB%A1%9C%EB%B2%84" title="로버">로버</a> 미션인 <a class="mw-redirect" href="/wiki/%ED%81%90%EB%A6%AC%EC%98%A4%EC%8B%9C%ED%8B%B0_%EB%A1%9C%EB%B2%84" title="큐리오시티 로버">큐리오시티 로버</a>와 <a class="mw-redirect" href="/wiki/%EC%98%A4%ED%8D%BC%ED%8A%9C%EB%8B%88%ED%8B%B0_%EB%A1%9C%EB%B2%84" title="오퍼튜니티 로버">오퍼튜니티 로버</a>의 중간 기점지 역할이 필수적으로 필요하기 때문에 발사를 연기하지 않는다고 밝혔다.<sup class="reference" id="cite_ref-21"><a href="#cite_note-21">[21]</a></sup><sup class="reference" id="cite_ref-22"><a href="#cite_note-22">[22]</a></sup></p>
<p>이후 2014년 9월 22일에 MAVEN은 약 10개월 동안 711 km 를 비행하여 <a href="/wiki/%ED%99%94%EC%84%B1" title="화성">화성</a> <a href="/wiki/%EA%B6%A4%EB%8F%84" title="궤도">궤도</a>에 성공적으로 진입하였다.<sup class="reference" id="cite_ref-23"><a href="#cite_note-23">[23]</a></sup></p>
