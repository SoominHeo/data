<h1 class="firstHeading" id="firstHeading" lang="ko">MKS 단위계</h1>
<div class="mw-body-content" id="bodyContent">
<div id="siteSub">위키백과, 우리 모두의 백과사전.</div>
<div id="contentSub"></div>
<div class="mw-jump" id="jump-to-nav">
					이동:					<a href="#mw-head">둘러보기</a>, 					<a href="#p-search">검색</a>
</div>
<div class="mw-content-ltr" dir="ltr" id="mw-content-text" lang="ko"><script>function mfTempOpenSection(id){var block=document.getElementById("mf-section-"+id);block.className+=" open-block";block.previousSibling.className+=" open-block";}</script><p><b>MKS 단위계</b>는 <a href="/wiki/%EA%B8%B8%EC%9D%B4" title="길이">길이</a>의 단위인 <a href="/wiki/%EB%AF%B8%ED%84%B0" title="미터">미터</a>(metre, m), <a href="/wiki/%EC%A7%88%EB%9F%89" title="질량">질량</a>의 단위인 <a href="/wiki/%ED%82%AC%EB%A1%9C%EA%B7%B8%EB%9E%A8" title="킬로그램">킬로그램</a>(kilogram, kg), <a href="/wiki/%EC%8B%9C%EA%B0%84" title="시간">시간</a>의 단위인 <a href="/wiki/%EC%B4%88_(%EC%8B%9C%EA%B0%84)" title="초 (시간)">초 (시간)</a>(second, s)를 기본 단위로 하는 <a class="new" href="/w/index.php?title=%EB%8B%A8%EC%9C%84%EA%B3%84&amp;action=edit&amp;redlink=1" title="단위계 (없는 문서)">단위계</a>이다.</p>
<p><a href="/wiki/%EB%AF%B8%ED%84%B0%EB%B2%95" title="미터법">미터법</a>에서 단위 이름은 미터·<a href="/wiki/%EA%B7%B8%EB%9E%A8" title="그램">그램</a>·초를 기준으로 하고 있고, 원기는 미터와 킬로그램을 기준으로 하고 있다. 단위계의 기초가 되는 기본 단위는 이론적으로는 단위 이름이나 원기와 무관하게 결정할 수 있다 그렇게 선정된 단위계로 MKS 단위계와 <a href="/wiki/CGS_%EB%8B%A8%EC%9C%84%EA%B3%84" title="CGS 단위계">CGS 단위계</a>(센티미터, 그램, 초)와 <a class="new" href="/w/index.php?title=MTS_%EB%8B%A8%EC%9C%84%EA%B3%84&amp;action=edit&amp;redlink=1" title="MTS 단위계 (없는 문서)">MTS 단위계</a>(미터, 톤, 초) 등이 있다. MKS 단위계는 역학으 단위만을 포함하며, 후에 전자기학의 <a href="/wiki/%EC%A0%84%EB%A5%98" title="전류">전류</a> 단위인 <a href="/wiki/%EC%95%94%ED%8E%98%EC%96%B4" title="암페어">암페어</a>(ampere, A)를 추가하여 <a class="new" href="/w/index.php?title=MKSA_%EB%8B%A8%EC%9C%84%EA%B3%84&amp;action=edit&amp;redlink=1" title="MKSA 단위계 (없는 문서)">MKSA 단위계</a>가 제안되었다. MKSA 단위계에 추가로 3가지 기본 단위를 더하여 <a href="/wiki/%EA%B5%AD%EC%A0%9C%EB%8B%A8%EC%9C%84%EA%B3%84" title="국제단위계">국제단위계</a>(SI)가 만들어졌다.</p>
<h2><span class="mw-headline" id=".EC.97.AD.EC.82.AC">역사</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=MKS_%EB%8B%A8%EC%9C%84%EA%B3%84&amp;action=edit&amp;section=1" title="부분 편집: 역사">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p><a href="/wiki/%EB%AF%B8%ED%84%B0%EB%B2%95" title="미터법">미터법</a>을 처음 만들 때는 질량의 기본 단위로 그라브(grave)를 구상했으나, 시행 시에는 1/1000 그라브를 <a href="/wiki/%EA%B7%B8%EB%9E%A8" title="그램">그램</a>, 1 그라브를 1 <a href="/wiki/%ED%82%AC%EB%A1%9C%EA%B7%B8%EB%9E%A8" title="킬로그램">킬로그램</a>이라는 명칭을 사용했다. 그러나 <a class="mw-redirect" href="/wiki/%EC%A7%88%EB%9F%89_%EC%9B%90%EA%B8%B0" title="질량 원기">질량 원기</a>는 기술적인 문제로 킬로그램 크기로 만들었다.</p>
<p><a href="/wiki/CGS_%EB%8B%A8%EC%9C%84%EA%B3%84" title="CGS 단위계">CGS 단위계</a>는 1832년 <a href="/wiki/%EC%B9%B4%EB%A5%BC_%ED%94%84%EB%A6%AC%EB%93%9C%EB%A6%AC%ED%9E%88_%EA%B0%80%EC%9A%B0%EC%8A%A4" title="카를 프리드리히 가우스">카를 프리드리히 가우스</a>에 의해 제창되었다 공식 단위계를 1874년 <a class="new" href="/w/index.php?title=%EC%98%81%EA%B5%AD_%EA%B3%BC%ED%95%99_%EC%A7%84%ED%9D%A5_%ED%98%91%ED%9A%8C&amp;action=edit&amp;redlink=1" title="영국 과학 진흥 협회 (없는 문서)">영국 과학 진흥 협회</a>(BAAS), 1881년 <a class="new" href="/w/index.php?title=%EA%B5%AD%EC%A0%9C_%EC%A0%84%EA%B8%B0_%ED%9A%8C%EC%9D%98&amp;action=edit&amp;redlink=1" title="국제 전기 회의 (없는 문서)">국제 전기 회의</a>에 전자기 확장되어 채택되었다. MKS 단위계는 1889년 <a href="/wiki/%EA%B5%AD%EC%A0%9C_%EB%8F%84%EB%9F%89%ED%98%95%EA%B5%AD" title="국제 도량형국">국제 도량형국</a>(BIPM)에 의해 채택되었다. CGS 단위계는 실험실 규모에 적합하여 과학자가 선호했으며, MKS는 기술자가 선호했다.</p>
<p>이 시기는 전자기학이 탄생하던 때로, CGS 단위계는 전자기를 다룰 수 있도록 확장되었다. 그러나 CGS 단위계의 전자기 단위는 양이 매우 크거나 작아 사용이 어렵고 십진배수나 분수를 이용하여 크기를 조절한 실용 단위가 나타났다. MKS도 처음에는 전자기를 취급하지 않았으나, 전기공학의 성장과 함께 그 필요성이 생겼다. 실용 단위는 CGS보다 다 MKS와 궁합이 좋았기 때문에, MKS와 실용 단위를 조합한 단위계(후의 <a class="new" href="/w/index.php?title=MKSA_%EB%8B%A8%EC%9C%84%EA%B3%84&amp;action=edit&amp;redlink=1" title="MKSA 단위계 (없는 문서)">MKSA 단위계</a>)가 제창되고 퍼져나갔다.</p>
<p>CGS와 MKS의 병행 사용은 그 후에도 한동안 계속되었지만, 1946년 <a href="/wiki/%EA%B5%AD%EC%A0%9C_%EB%8F%84%EB%9F%89%ED%98%95_%EC%9C%84%EC%9B%90%ED%9A%8C" title="국제 도량형 위원회">국제 도량형 위원회</a>(CIPM), 1950년 <a href="/wiki/%EA%B5%AD%EC%A0%9C_%EC%A0%84%EA%B8%B0_%ED%91%9C%EC%A4%80_%ED%9A%8C%EC%9D%98" title="국제 전기 표준 회의">국제 전기 표준 회의</a>(IEC)에서 MKSA 단위계, 1954년 <a href="/wiki/%EA%B5%AD%EC%A0%9C_%EB%8F%84%EB%9F%89%ED%98%95_%EC%B4%9D%ED%9A%8C" title="국제 도량형 총회">국제 도량형 총회</a>에서 MKSA에 <a href="/wiki/%EC%BC%88%EB%B9%88" title="켈빈">켈빈</a>과 <a href="/wiki/%EC%B9%B8%EB%8D%B8%EB%9D%BC" title="칸델라">칸델라</a>가 추가된 단위계가 채택되었다. 1960년 MKS를 확장한 <a href="/wiki/%EA%B5%AD%EC%A0%9C%EB%8B%A8%EC%9C%84%EA%B3%84" title="국제단위계">국제단위계</a>(SI)가 세계 표준이 되었다.</p>
