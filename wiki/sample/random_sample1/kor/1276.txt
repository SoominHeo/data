<p><b>40호 기관단총</b>(<span style="font-size: smaller;"><a href="/wiki/%EB%8F%85%EC%9D%BC%EC%96%B4" title="독일어">독일어</a>:</span> <span lang="de" xml:lang="de">Maschinenpistole 40; MP 40</span>)은 <a href="/wiki/%EB%8F%85%EC%9D%BC" title="독일">독일</a>에서 개발된 <a href="/wiki/%EA%B8%B0%EA%B4%80%EB%8B%A8%EC%B4%9D" title="기관단총">기관단총</a>이다. <a href="/wiki/%EC%A0%9C2%EC%B0%A8_%EC%84%B8%EA%B3%84_%EB%8C%80%EC%A0%84" title="제2차 세계 대전">제2차 세계 대전</a> 당시 <a href="/wiki/%EA%B3%B5%EC%88%98%EB%B6%80%EB%8C%80" title="공수부대">공수부대</a> 등에서 많이 사용되었다.</p>

<h2><span class="mw-headline" id=".EC.97.AD.EC.82.AC">역사</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=MP_40&amp;action=edit&amp;section=1" title="부분 편집: 역사">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>MP40은 MP36의 계승작이다(MP36의 시작품은 현재 2정 남아 있다.). 시작품은 <a class="new" href="/w/index.php?title=%ED%95%98%EC%9D%B8%EB%A6%AC%ED%9E%88_%EB%B3%BC%EB%A5%B4%EB%A8%B8&amp;action=edit&amp;redlink=1" title="하인리히 볼르머 (없는 문서)">하인리히 볼르머</a>의 VPM 1930을 기반으로 <a href="/wiki/%EB%8F%85%EC%9D%BC_%EA%B5%AD%EB%B0%A9%EA%B5%B0" title="독일 국방군">독일 국방군</a>의 재정적 도움을 받아 제작되었다. 볼르머는 MP36의 제작에 참여했고, 1938년, 독일군의 제식 <a href="/wiki/%EA%B8%B0%EA%B4%80%EB%8B%A8%EC%B4%9D" title="기관단총">기관단총</a>에 대한 요구에 시작품을 제출한다. 그리고 MP38이라는 이름으로 채택된다. MP38은 MP36을 단순화한 것이며, MP40은 이것을 더욱 단순화하여 비용 절감과 함께 기계 부속의 사용을 줄였다.</p>
<p>변화는 1939년 폴란드 침공시 사용된 수천 정의 MP38에서 비롯되었다. 이 변화는 MP38과 MP40의 중간 단계였던 MP38/40에 추가되었고, 그 후 MP40으로 생산이 된다. 전쟁 중, 모든 버전을 통틀어 1백만 정 이상이 생산된 것으로 보고 있다.</p>
<p>연합군은 이 기관단총을 독일의 총기 제작자 <a class="new" href="/w/index.php?title=%ED%9C%B4%EA%B3%A0_%EC%8A%88%EB%A7%88%EC%9D%B4%EC%84%9C&amp;action=edit&amp;redlink=1" title="휴고 슈마이서 (없는 문서)">휴고 슈마이서</a>의 이름을 본따 '슈마이서(Schmeisser)'라 불렀으나, 정작 휴고 슈마이서 본인은 MP40 제작에 참여하지 않았다. 그는 단지 탄창에 대한 특허만 가지고 있을 뿐이었다. 슈마이서는 MP40에 나무 개머리판과 사격 방식 선택 장치를 추가한 MP41 제작에 참여하였으나, MP41은 제식 기관단총으로 채택되지 않았다.</p>
<h2><span class="mw-headline" id=".EB.94.94.EC.9E.90.EC.9D.B8">디자인</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=MP_40&amp;action=edit&amp;section=2" title="부분 편집: 디자인">편집</a><span class="mw-editsection-bracket">]</span></span></h2>

<p>MP38과 MP40 모두 <a href="/wiki/%EC%98%A4%ED%94%88_%EB%B3%BC%ED%8A%B8" title="오픈 볼트">오픈 볼트</a>, <a class="new" href="/w/index.php?title=%EB%B8%94%EB%A1%9C%EC%9A%B0%EB%B0%B1&amp;action=edit&amp;redlink=1" title="블로우백 (없는 문서)">블로우백</a> 화기이다. 원래는 완전 자동으로만 설정되어있지만, 방아쇠를 조정하면 단발로도 쏠 수 있다.</p>
<p>초기 MP38은 장전 손잡이가 일체로 붙어있었지만, MP40부터는 개별적인 부품으로 구성되었다. 이로 인해 볼트를 앞 또는 콕킹 상태에 놓을 수 있었다. 그러나 MP38은 그렇지 않았으므로 볼트를 앞쪽에 놓기 위해 고리가 달린 가죽 끈을 이용해 고정했다.</p>
<p>리시버는 원래 부품을 조립해서 만들었지만 시간과 비용이 너무 많이 드는 관계로 기계로 찍어낼 수 있도록 개량했다. MP38은 리시버와 볼트에 탄창의 하우징을 위해 세로로 홈이 파져있다.</p>
<p>한가지 특이한 점은 대부분의 MP38과 MP40은 총신 밑에 알루미늄 또는 플라스틱 레일이 있었다는 것이다. 이것의 역할은 <a href="/wiki/Sd.Kfz_251" title="Sd.Kfz 251">Sd.Kfz 251</a>과 같은 <a class="mw-redirect" href="/wiki/%EB%B3%91%EB%A0%A5_%EC%88%98%EC%86%A1_%EC%9E%A5%EA%B0%91%EC%B0%A8" title="병력 수송 장갑차">병력 수송 장갑차</a> 위에서 사격할 때 도와주는 역할을 하였다. 핸드가드는 탄창과 권총형 손잡이 중간에 있는데 <a href="/wiki/%EB%B2%A0%EC%9D%B4%ED%81%B4%EB%9D%BC%EC%9D%B4%ED%8A%B8" title="베이클라이트">베이클라이트</a>에서 파생된 합성 섬유로 만들었다. 총신은 단열재가 아니었으므로 만약 총신이 과열되어 휘었을 때, 교체하는 도중 화상을 입을 가능성이 있었다. 기관단총으로는 처음으로 총신을 접을 수 있도록 만들어졌으나, 백병전에는 강도가 적합하지 않았다.</p>
<p>총체적으로 MP40은 쓸만했으나, 최대의 약점은 32발이라는 장탄 수였다. 쌍열식 탄창을 사용하여 이중 급탄을 하는 <a class="mw-redirect" href="/wiki/%ED%86%B0%EC%8A%A8_%EA%B8%B0%EA%B4%80%EB%8B%A8%EC%B4%9D" title="톰슨 기관단총">톰슨 기관단총</a>과는 달리, MP 시리즈는 단발 급탄이었다. 이것은 <a href="/wiki/%EC%8A%A4%ED%85%90_%EA%B8%B0%EA%B4%80%EB%8B%A8%EC%B4%9D" title="스텐 기관단총">스텐 기관단총</a>과 마찬가지로 탄창을 손잡이로 사용했을 때, 손이 주는 압력으로 인해 급탄이 제대로 되지 않는 결과를 불러왔다. 그리하여 독일군은 사격시 반드시 따로 마련해놓은 손잡이를 잡고 쏘도록 훈련받았다.</p>
<p>대중에 알려진 사실과는 다르게, MP40은 주로 공수부대원이나 소대나 중대 지휘관이 사용하였다. 대부분의 병사는 <a class="new" href="/w/index.php?title=%EB%A7%88%EC%9A%B0%EC%A0%80_Kar_98k&amp;action=edit&amp;redlink=1" title="마우저 Kar 98k (없는 문서)">마우저 Kar 98k</a>를 사용하였다. 그러나 시가전에 투입된 소련군이 전부 기관단총을 사용하여 전술적에서 밀리자, 그에 대응하기 위하여 일반 병사에게도 지급을 시작하고, 전쟁 말기에는 물량의 부족으로 돌격 소대에게만 한정적으로 지급하였다.</p>
<p>MP40이 전쟁 중에 넉넉히 지급된 기간은 존재하지 않는다. Kar 98k에 비해서 가격이 비싸고 재료도 많이 들어갔기 때문이다. 이로 인해, 1943년부터 Kar 98k와 MP40을 <a class="new" href="/w/index.php?title=StG44&amp;action=edit&amp;redlink=1" title="StG44 (없는 문서)">StG44</a>로 알려진 MP43/44 <a href="/wiki/%EB%8F%8C%EA%B2%A9%EC%86%8C%EC%B4%9D" title="돌격소총">돌격소총</a>으로 대체하기 시작한다.</p>
<h2><span class="mw-headline" id=".EB.B3.B5.EC.A0.9C.ED.8C.90.EA.B3.BC_.EC.A2.85.EC.A0.84_.ED.9B.84.EC.9D.98_.EC.82.AC.EC.9A.A9">복제판과 종전 후의 사용</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=MP_40&amp;action=edit&amp;section=3" title="부분 편집: 복제판과 종전 후의 사용">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>MP38과 MP40은 <a href="/wiki/M3_%EA%B8%B0%EA%B4%80%EB%8B%A8%EC%B4%9D" title="M3 기관단총">M3 기관단총</a>과 스페인의 무기 회사 <a class="new" href="/w/index.php?title=%EC%8A%A4%ED%83%80_%EB%B3%B4%EB%8B%88%ED%8C%8C%EC%8B%9C%EC%98%A4_%EC%97%90%EC%B2%B4%EB%B2%A0%EB%A6%AC%EB%82%98&amp;action=edit&amp;redlink=1" title="스타 보니파시오 에체베리나 (없는 문서)">스타 보니파시오 에체베리나</a>의 스타 모델 시리즈, 유고슬라비아 인민군의 M56, 노르웨이 향토방위군, 코소보 독립군 등에서 사용되었다.</p>
<h2><span class="mw-headline" id=".EB.B0.94.EB.A6.AC.EC.97.90.EC.9D.B4.EC.85.98.EA.B3.BC_.EA.B0.9C.EB.B0.9C">바리에이션과 개발</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=MP_40&amp;action=edit&amp;section=4" title="부분 편집: 바리에이션과 개발">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<ul>
<li>MP40/I - 주 생산 버전.</li>
<li>MP40/II - 1942년 쌍열 32발 탄창을 사용했던 프로토타입. <a href="/wiki/PPSh-41" title="PPSh-41">PPSh-41</a>의 연사속도에 대항하기 위하여 제작되었으나, 무겁고 무게중심이 맞지 않았으며, 실용성도 없었다.</li>
<li>MP41 - <a class="new" href="/w/index.php?title=%EB%A3%A8%EC%9D%B4%EC%8A%A4_%EC%8A%88%EB%A7%88%EC%9D%B4%EC%84%9C&amp;action=edit&amp;redlink=1" title="루이스 슈마이서 (없는 문서)">루이스 슈마이서</a>가 개량한 버전. 리시버, 작동 방식, 탄창 하우징, 개머리판, 방아쇠를 개량하고 발사 선택기능을 추가하였다.발사속도가 분당 1000발로 상승하였다.</li>
