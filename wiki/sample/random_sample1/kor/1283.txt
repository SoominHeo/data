<h2><span class="mw-headline" id=".EA.B0.9C.EC.9A.94">개요</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Megpoid&amp;action=edit&amp;section=1" title="부분 편집: 개요">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>일본의 인터넷 사에서 <a href="/wiki/%EA%B0%80%EC%BF%A0%ED%8F%AC%EC%9D%B4%EB%93%9C" title="가쿠포이드">가쿠포이드</a> 다음으로 발매한 두 번째 VOCALOID로, <a href="/wiki/2009%EB%85%84" title="2009년">2009년</a> <a href="/wiki/6%EC%9B%94_26%EC%9D%BC" title="6월 26일">6월 26일</a>에 발매되었다. 목소리 데이터베이스는 <a href="/wiki/%EB%82%98%EC%B9%B4%EC%A7%80%EB%A7%88_%EB%A9%94%EA%B5%AC%EB%AF%B8_(%EC%84%B1%EC%9A%B0)" title="나카지마 메구미 (성우)">나카지마 메구미</a>의 목소리를 기초로 하였다. 나카지마 메구미는 《<a class="mw-redirect" href="/wiki/%EB%A7%88%ED%81%AC%EB%A1%9C%EC%8A%A4F" title="마크로스F">마크로스F</a>》의 등장인물인 “란카 리”의 성우였고, 게다가 이미지 캐릭터의 디자인도 비슷해서 그녀가 담당한 Megpoid는 발매 전에 일본에서 “란카로이드”라는 별명이 있었다. 캐릭터 디자인은 만화가 <a class="mw-redirect" href="/wiki/%EC%9C%A0%EC%9A%B0%ED%82%A4_%EB%A7%88%EC%82%AC%EB%AF%B8" title="유우키 마사미">유우키 마사미</a>가 담당하였다. 이미지 캐릭터의 이름은 <b>GUMI</b>로, 목소리를 담당한 성우인 나카지마 메구미가 직접 지어주었다.</p>
<h2><span class="mw-headline" id=".ED.94.84.EB.A1.9C.ED.95.84.EA.B3.BC_.EC.BA.90.EB.A6.AD.ED.84.B0_.EA.B4.80.EB.A0.A8">프로필과 캐릭터 관련</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Megpoid&amp;action=edit&amp;section=2" title="부분 편집: 프로필과 캐릭터 관련">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<h3><span class="mw-headline" id=".ED.94.84.EB.A1.9C.ED.95.84">프로필</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Megpoid&amp;action=edit&amp;section=3" title="부분 편집: 프로필">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<ul>
<li>제품 이름: Megpoid(<span style="font-size: smaller;"><a href="/wiki/%EC%9D%BC%EB%B3%B8%EC%96%B4" title="일본어">일본어</a>:</span> <span lang="ja" xml:lang="ja">メグッポイド</span>)</li>
<li>이미지 캐릭터 이름: GUMI</li>
<li>발매일: <a href="/wiki/2009%EB%85%84" title="2009년">2009년</a> <a href="/wiki/6%EC%9B%94_26%EC%9D%BC" title="6월 26일">6월 26일</a></li>
<li>별자리: <a href="/wiki/%EA%B2%8C%EC%9E%90%EB%A6%AC" title="게자리">게자리</a></li>
<li>자신 있는 템포: 60 ~ 175 BPM</li>
<li>자신 있는 음역: F2 ~ A4</li>
</ul>
<h3><span class="mw-headline" id=".EC.BA.90.EB.A6.AD.ED.84.B0_.EA.B4.80.EB.A0.A8">캐릭터 관련</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Megpoid&amp;action=edit&amp;section=4" title="부분 편집: 캐릭터 관련">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>인터넷 사도 <a href="/wiki/%ED%81%AC%EB%A6%BD%ED%86%A4_%ED%93%A8%EC%B2%98_%EB%AF%B8%EB%94%94%EC%96%B4" title="크립톤 퓨처 미디어">크립톤 퓨처 미디어</a>와 같이 메구포이드 캐릭터 사용에 대한 “<a class="external text" href="http://www.ssw.co.jp/products/vocal/character_guideline/" rel="nofollow">지침</a>”이 있고 이를 따라야만 한다. 크립톤에서 운영하는 <a href="/wiki/%ED%94%BC%EC%95%84%ED%94%84%EB%A1%9C" title="피아프로">피아프로</a>에서 크립톤 VOCALOID 뿐만 아닌 Megpoid 관련 작품을 투고 가능하다<sup class="reference" id="cite_ref-1"><a href="#cite_note-1">[1]</a></sup>.</p>
<h2><span class="mw-headline" id=".EC.9D.8C.EC.9B.90">음원</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Megpoid&amp;action=edit&amp;section=5" title="부분 편집: 음원">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>Megpoid는 2009년 6월 26일 VOCALOID2 엔진을 기반으로 발매된 VOCALOID 제품이다. 음성은 같은 회사 제품인 <a class="mw-redirect" href="/wiki/Gackpoid" title="Gackpoid">Gackpoid</a> 처럼 사람답게 되도록 조정하고, 나카지마 메구미의 목소리의 특징을 남기게 했다고 생각된다<sup class="reference" id="cite_ref-ascii.2F429.2F429598_2-0"><a href="#cite_note-ascii.2F429.2F429598-2">[2]</a></sup>. <a href="/wiki/2011%EB%85%84" title="2011년">2011년</a> <a href="/wiki/10%EC%9B%94_21%EC%9D%BC" title="10월 21일">10월 21일</a>, VOCALOID3를 채용한 “VOCALOID3 Megpoid” 인 Power, Whisper, Adult, Sweet 이렇게 4 종류의 라이브러리 음원을 발매 하였다<sup class="reference" id="cite_ref-dtmm20110907_3-0"><a href="#cite_note-dtmm20110907-3">[3]</a></sup>. 이들은 4개의 라이브러리를 모두 수록한 “VOCALOID3 Megpoid Complete”도 발매 되었다<sup class="reference" id="cite_ref-dtmm20110907_3-1"><a href="#cite_note-dtmm20110907-3">[3]</a></sup>. 원래는 목소리를 확장하기 위해 기획된 것으로, 처음에는 “extend”로 VOCALOID2 제품으로 개발이 진행되고 있었으나, VOCALOID3 엔진 발표로 인해 VOCALOID3 제품으로 발매되었다<sup class="reference" id="cite_ref-.E3.83.9C.E3.82.AB.E3.83.AD_Plus2p50_4-0"><a href="#cite_note-.E3.83.9C.E3.82.AB.E3.83.AD_Plus2p50-4">[4]</a></sup>. 또한 “Megpoid Native”는 2012년 3월 16일에 발매했으며<sup class="reference" id="cite_ref-5"><a href="#cite_note-5">[5]</a></sup>, 2013년 2월 28일에는 영어 라이브러리 “Megpoid ENGLSIH”가 발매되었다<sup class="reference" id="cite_ref-6"><a href="#cite_note-6">[6]</a></sup>.</p>
<h2><span class="mw-headline" id=".EB.AF.B8.EB.94.94.EC.96.B4_.EB.AF.B9.EC.8A.A4">미디어 믹스</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Megpoid&amp;action=edit&amp;section=6" title="부분 편집: 미디어 믹스">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<h3><span class="mw-headline" id=".EA.B2.8C.EC.9E.84">게임</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Megpoid&amp;action=edit&amp;section=7" title="부분 편집: 게임">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<dl>
<dt><a href="/wiki/%ED%95%98%EC%B8%A0%EB%84%A4_%EB%AF%B8%EC%BF%A0_and_Future_Stars_Project_mirai" title="하츠네 미쿠 and Future Stars Project mirai">하츠네 미쿠 Project mirai 2, 하츠네 미쿠 Project mirai 디럭스</a> (세가)</dt>
</dl>
<p><a href="/wiki/2012%EB%85%84" title="2012년">2012년</a> <a href="/wiki/3%EC%9B%94_8%EC%9D%BC" title="3월 8일">3월 8일</a> 발매된 <a href="/wiki/%EB%8B%8C%ED%85%90%EB%8F%84_3DS" title="닌텐도 3DS">닌텐도 3DS</a> 용 게임. 《하츠네 미쿠 and Future Stars Project mirai》에는 등장하지 않았으나, 《하츠네 미쿠 Project mirai 2》와《하츠네 미쿠 Project mirai 디럭스》부터는 Megpoid도 등장한다. 찾아와서 팁을 알려주거나 일부 PV에 나오는 역할로 등장한다.</p>
