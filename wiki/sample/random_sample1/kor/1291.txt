<h2><span class="mw-headline" id=".EC.97.AD.EC.82.AC.EC.99.80_.EC.96.B4.EC.9B.90">역사와 어원</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=M%EC%9D%B4%EB%A1%A0&amp;action=edit&amp;section=1" title="부분 편집: 역사와 어원">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>1990년대 초기에는 총 5개의 <a class="mw-redirect" href="/wiki/%EC%B4%88%EB%81%88_%EC%9D%B4%EB%A1%A0" title="초끈 이론">초끈 이론</a>들이 알려져 있었다. 이들은 10차원에 존재하는, <a href="/wiki/%EB%81%88_(%EB%AC%BC%EB%A6%AC%ED%95%99)" title="끈 (물리학)">끈</a>을 포함하는 이론이며, 이들 사이에는 <a href="/wiki/T-%EC%9D%B4%EC%A4%91%EC%84%B1" title="T-이중성">T-이중성</a>과 <a href="/wiki/S-%EC%9D%B4%EC%A4%91%EC%84%B1" title="S-이중성">S-이중성</a> 등 여러 관계가 존재한다. 1995년에 <a href="/wiki/%EC%97%90%EB%93%9C%EC%9B%8C%EB%93%9C_%EC%9C%84%ED%8A%BC" title="에드워드 위튼">에드워드 위튼</a>은 이들 5개의 초끈 이론들을 끈을 포함하지 않고, 11차원에 존재하는 어떤 "M이론"을 통해 얻을 수 있다는 증거를 제시하였다.<sup class="reference" id="cite_ref-2"><a href="#cite_note-2">[2]</a></sup> 즉, 5개의 초끈 이론은 하나의 M이론의 다양한 극한(모듈러스 공간의 귀퉁이)에 해당한다. 이 사건을 <b>제2차 초끈 혁명</b>(<span style="font-size: smaller;"><a href="/wiki/%EC%98%81%EC%96%B4" title="영어">영어</a>:</span> <span lang="en" xml:lang="en">the Second Superstring Revolution</span>)이라고 한다.</p>
<p>위튼에 따르면, M이론의 ‘M’은 <span style="font-size: smaller;"><a href="/wiki/%EC%98%81%EC%96%B4" title="영어">영어</a>:</span> <span lang="en" xml:lang="en">magic</span> <small>매직<sup>[<a href="/wiki/%EC%9C%84%ED%82%A4%EB%B0%B1%EA%B3%BC:%EC%98%81%EC%96%B4%EC%9D%98_%ED%95%9C%EA%B8%80_%ED%91%9C%EA%B8%B0" title="위키백과:영어의 한글 표기">*</a>]</sup></small>, <span style="font-size: smaller;"><a href="/wiki/%EC%98%81%EC%96%B4" title="영어">영어</a>:</span> <span lang="en" xml:lang="en">mystery</span> <small>미스터리<sup>[<a href="/wiki/%EC%9C%84%ED%82%A4%EB%B0%B1%EA%B3%BC:%EC%98%81%EC%96%B4%EC%9D%98_%ED%95%9C%EA%B8%80_%ED%91%9C%EA%B8%B0" title="위키백과:영어의 한글 표기">*</a>]</sup></small>, 또는 <span style="font-size: smaller;"><a href="/wiki/%EC%98%81%EC%96%B4" title="영어">영어</a>:</span> <span lang="en" xml:lang="en">membrane</span> <small>멤브레인<sup>[<a href="/wiki/%EC%9C%84%ED%82%A4%EB%B0%B1%EA%B3%BC:%EC%98%81%EC%96%B4%EC%9D%98_%ED%95%9C%EA%B8%80_%ED%91%9C%EA%B8%B0" title="위키백과:영어의 한글 표기">*</a>]</sup></small>의 머릿자라고 한다.<sup class="reference" id="cite_ref-3"><a href="#cite_note-3">[3]</a></sup><sup class="reference" id="cite_ref-4"><a href="#cite_note-4">[4]</a></sup><sup class="reference" id="cite_ref-5"><a href="#cite_note-5">[5]</a></sup> <span style="font-size: smaller;"><a href="/wiki/%EC%98%81%EC%96%B4" title="영어">영어</a>:</span> <span lang="en" xml:lang="en">membrane</span> <small>멤브레인<sup>[<a href="/wiki/%EC%9C%84%ED%82%A4%EB%B0%B1%EA%B3%BC:%EC%98%81%EC%96%B4%EC%9D%98_%ED%95%9C%EA%B8%80_%ED%91%9C%EA%B8%B0" title="위키백과:영어의 한글 표기">*</a>]</sup></small>은 막을 뜻하는 단어인데, 이는 M이론이 끈을 포함하지 않고, 대신 2차원 및 5차원 막을 포함하기 때문이다.</p>
<p>1996년에 톰 뱅크스(<span style="font-size: smaller;"><a href="/wiki/%EC%98%81%EC%96%B4" title="영어">영어</a>:</span> <span lang="en" xml:lang="en">Tom Banks</span>)와 빌리 피스흘러르(<span style="font-size: smaller;"><a href="/wiki/%EB%84%A4%EB%8D%9C%EB%9E%80%EB%93%9C%EC%96%B4" title="네덜란드어">네덜란드어</a>:</span> <span lang="nl" xml:lang="nl">Willy Fischler</span>), 스티븐 솅커(<span lang="en" xml:lang="en">Stephen Hart Shenker</span>)와 <a href="/wiki/%EB%A0%88%EB%84%88%EB%93%9C_%EC%84%9C%EC%8A%A4%ED%82%A8%EB%93%9C" title="레너드 서스킨드">레너드 서스킨드</a>가 축소화하지 않은 M이론을 행렬 변수에 대한 <a href="/wiki/%EC%96%91%EC%9E%90%EC%97%AD%ED%95%99" title="양자역학">양자역학</a>의 특정한 극한으로 정의하였다.<sup class="reference" id="cite_ref-6"><a href="#cite_note-6">[6]</a></sup> 이를 <b><a href="/wiki/%ED%96%89%EB%A0%AC_%EC%9D%B4%EB%A1%A0" title="행렬 이론">행렬 이론</a></b>(<span lang="en" xml:lang="en">M(atrix) theory</span>)이라고 한다. 영어명 "<span lang="en" xml:lang="en">M(atrix)</span>"는 행렬을 뜻하는 <span style="font-size: smaller;"><a href="/wiki/%EC%98%81%EC%96%B4" title="영어">영어</a>:</span> <span lang="en" xml:lang="en">matrix</span> <small>메이트릭스<sup>[<a href="/wiki/%EC%9C%84%ED%82%A4%EB%B0%B1%EA%B3%BC:%EC%98%81%EC%96%B4%EC%9D%98_%ED%95%9C%EA%B8%80_%ED%91%9C%EA%B8%B0" title="위키백과:영어의 한글 표기">*</a>]</sup></small>의 머릿글자가 M이론과 같은 "M"임을 농으로 딴 것이다.</p>
<p>1997년에 <a href="/wiki/%ED%9B%84%EC%95%88_%EB%A7%90%EB%8B%A4%EC%84%B8%EB%82%98" title="후안 말다세나">후안 말다세나</a>는 <a href="/wiki/AdS/CFT_%EB%8C%80%EC%9D%91%EC%84%B1" title="AdS/CFT 대응성">AdS/CFT 대응성</a>을 발표하면서, <a href="/wiki/%EB%B0%98_%EB%8D%94_%EC%8B%9C%ED%84%B0%EB%A5%B4_%EA%B3%B5%EA%B0%84" title="반 더 시터르 공간">AdS<sub>3</sub></a>×<a href="/wiki/%EC%B4%88%EA%B5%AC" title="초구">S<sup>7</sup></a> 또는 <a href="/wiki/%EB%B0%98_%EB%8D%94_%EC%8B%9C%ED%84%B0%EB%A5%B4_%EA%B3%B5%EA%B0%84" title="반 더 시터르 공간">AdS<sub>7</sub></a>×<a href="/wiki/%EC%B4%88%EA%B5%AC" title="초구">S<sup>3</sup></a>에 축소화한 M이론을, 겹친 M2-막 또는 M5-막의 세계부피 이론으로 비섭동적으로 정의할 수 있음을 보였다. 그러나 보다 더 잘 알려진 <a href="/wiki/D-%EB%A7%89" title="D-막">D3-막</a>의 경우와 달리 M-막의 세계부피 이론은 오랫동안 알려지지 않았다. M2-막의 세계부피 이론은 2008년에 발견되었으나, 아직 M5-막의 세계부피 이론은 알려지지 않고 있다.</p>
<h2><span class="mw-headline" id=".EC.A0.84.EA.B0.9C">전개</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=M%EC%9D%B4%EB%A1%A0&amp;action=edit&amp;section=2" title="부분 편집: 전개">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>IIA종 <a class="mw-redirect" href="/wiki/%EC%B4%88%EB%81%88_%EC%9D%B4%EB%A1%A0" title="초끈 이론">초끈 이론</a>과 E<sub>8</sub>×E<sub>8</sub> <a href="/wiki/%EC%9E%A1%EC%A2%85_%EB%81%88_%EC%9D%B4%EB%A1%A0" title="잡종 끈 이론">잡종 끈 이론</a>에서 닫힌 끈 <a href="/wiki/%EA%B2%B0%ED%95%A9_%EC%83%81%EC%88%98" title="결합 상수">결합 상수</a> <span><span class="mwe-math-mathml-inline mwe-math-mathml-a11y" style="display: none;"><math xmlns="http://www.w3.org/1998/Math/MathML">

<p>M이론에서 다루는 대상은 2차원의 막인 <b>M2-막</b>(<span lang="en" xml:lang="en">M2-brane</span>)과 5차원의 막인 <b>M5-막</b>(<span lang="en" xml:lang="en">M5-brane</span>)이다. <p>M이론을 축소화하여 IIA종 초끈 이론과 E<sub>8</sub>×E<sub>8</sub> <a href="/wiki/%EC%9E%A1%EC%A2%85_%EB%81%88_%EC%9D%B4%EB%A1%A0" title="잡종 끈 이론">잡종 끈 이론</a>을 얻을 수 있다. 다른 초끈 이론들(IIB종, I종, <a class="mw-redirect" href="/wiki/%ED%8A%B9%EC%88%98%EC%A7%81%EA%B5%90%EA%B5%B0" title="특수직교군">SO(32)</a> <a href="/wiki/%EC%9E%A1%EC%A2%85_%EB%81%88_%EC%9D%B4%EB%A1%A0" title="잡종 끈 이론">잡종</a>)은 이들로부터 <a href="/wiki/S-%EC%9D%B4%EC%A4%91%EC%84%B1" title="S-이중성">S-이중성</a>과 <a href="/wiki/T-%EC%9D%B4%EC%A4%91%EC%84%B1" title="T-이중성">T-이중성</a>을 가하여 얻을 수 있다.</p>

<p>2007년 조너선 배거(<span style="font-size: smaller;"><a href="/wiki/%EC%98%81%EC%96%B4" title="영어">영어</a>:</span> <span lang="en" xml:lang="en">Jonathan Bagger</span>)와 닐 램버트(<span style="font-size: smaller;"><a href="/wiki/%EC%98%81%EC%96%B4" title="영어">영어</a>:</span> <span lang="en" xml:lang="en">Neil Lambert</span>), 안드레아스 구스타브손(<span style="font-size: smaller;"><a href="/wiki/%EC%8A%A4%EC%9B%A8%EB%8D%B4%EC%96%B4" title="스웨덴어">스웨덴어</a>:</span> <span lang="sv" xml:lang="sv">Andreas Gustavsson</span>)이 M2-막 <a href="/wiki/%EC%84%B8%EA%B3%84%EB%A9%B4" title="세계면">세계부피</a> 이론의 <a href="/wiki/%EC%9E%91%EC%9A%A9_(%EB%AC%BC%EB%A6%AC%ED%95%99)" title="작용 (물리학)">작용</a>을 발견하였다.<sup class="reference" id="cite_ref-10"><a href="#cite_note-10">[10]</a></sup><sup class="reference" id="cite_ref-11"><a href="#cite_note-11">[11]</a></sup><sup class="reference" id="cite_ref-12"><a href="#cite_note-12">[12]</a></sup> 이를 발견자의 머릿글자를 따 <b>BLG 모형</b>(<span lang="en" xml:lang="en">BLG model</span>)이라고 한다.<sup class="reference" id="cite_ref-13"><a href="#cite_note-13">[13]</a></sup> 이 모형은 리 괄호를 일반화한 "3-대수" <span><span class="mwe-math-mathml-inline mwe-math-mathml-a11y" style="display: none;"><math xmlns="http://www.w3.org/1998/Math/MathML">
