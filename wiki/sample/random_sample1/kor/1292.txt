<div class="mw-content-ltr" dir="ltr" id="mw-content-text" lang="ko"><script>function mfTempOpenSection(id){var block=document.getElementById("mf-section-"+id);block.className+=" open-block";block.previousSibling.className+=" open-block";}</script><div class="dablink"><a href="/wiki/%EC%9C%84%ED%82%A4%EB%B0%B1%EA%B3%BC:%EB%8F%99%EC%9D%8C%EC%9D%B4%EC%9D%98%EC%96%B4_%EB%AC%B8%EC%84%9C" title="위키백과:동음이의어 문서"><img alt="" data-file-height="200" data-file-width="260" height="18" src="//upload.wikimedia.org/wikipedia/commons/thumb/4/4a/Disambig_grey.svg/23px-Disambig_grey.svg.png" srcset="//upload.wikimedia.org/wikipedia/commons/thumb/4/4a/Disambig_grey.svg/35px-Disambig_grey.svg.png 1.5x, //upload.wikimedia.org/wikipedia/commons/thumb/4/4a/Disambig_grey.svg/46px-Disambig_grey.svg.png 2x" width="23"/></a> 소련의 우주발사체를 찾으신다면 <a href="/wiki/N-1" title="N-1">N-1</a> 문서를 참조하십시오.</div>
<p><b>N-I</b>은 일본의 <a href="/wiki/%EB%A1%9C%EC%BC%93" title="로켓">로켓</a>으로 <a class="new" href="/w/index.php?title=%EB%8D%B8%ED%83%80_%EB%A1%9C%EC%BC%93&amp;action=edit&amp;redlink=1" title="델타 로켓 (없는 문서)">델타 로켓</a>의 라이선스를 받아 생산한 로켓이다. 일본은 <a href="/wiki/1970%EB%85%84" title="1970년">1970년</a>에 완전한 국산의 고체연료 로켓 <a href="/wiki/%EB%9E%8C%EB%8B%A4_4S" title="람다 4S">람다 4S</a>로 인공 위성을 발사하고 있었지만, 액체연료 로켓은 난이도가 높으므로, 미국의 협력을 받아서 1단을 생산했다. 2단은 일제 LE-3를 사용했다. 일본은 <a href="/wiki/1975%EB%85%84" title="1975년">1975년</a>의 N-I의 발사로부터 <a href="/wiki/1992%EB%85%84" title="1992년">1992년</a>의 <a href="/wiki/H-I" title="H-I">H-I</a>의 운용 종료까지의 17년간 미제 1단 엔진 MB-3-3을 사용했다. <a href="/wiki/1983%EB%85%84" title="1983년">1983년</a>부터 <a href="/wiki/1994%EB%85%84" title="1994년">1994년</a>까지 행하여진 <a href="/wiki/H-II" title="H-II">H-II</a>와 <a href="/wiki/%EB%AF%B8%EC%93%B0%EB%B9%84%EC%8B%9C_LE-7" title="미쓰비시 LE-7">미쓰비시 LE-7</a>의 국산개발의 성공으로 MB-3-3의 사용을 그만뒀다. 미국은 핵심 기술은 비밀로 유지하는 블랙 박스의 조건으로 일본의 라이선스 생산을 허용했다. <sup class="reference" id="cite_ref-1"><a href="#cite_note-1">[1]</a></sup></p>
<p>1975년 최초 발사, 1982년까지 7회 발사했다. 5회 발사만 실패했다.</p>
