<p><b>1853년형 엔필드 강선머스킷</b>(<span lang="en" xml:lang="en">Enfield Pattern 1853 rifle-musket</span>)은 <a href="/wiki/%EC%98%81%EA%B5%AD" title="영국">영국</a>의 <a href="/wiki/%EC%97%94%ED%95%84%EB%93%9C_%EC%A1%B0%EB%B3%91%EC%B0%BD" title="엔필드 조병창">엔필드 조병창</a>에서 개발된 <a class="mw-redirect" href="/wiki/%EC%A0%84%EC%9E%A5%EC%8B%9D" title="전장식">전장식</a> 소총(<a href="/wiki/%EA%B0%95%EC%84%A0" title="강선">강선</a> 총)이다. 총알 모양은 약간 다르지만 ‘<a class="new" href="/w/index.php?title=%EB%AF%B8%EB%8B%88%EC%97%90_%EC%B4%9D&amp;action=edit&amp;redlink=1" title="미니에 총 (없는 문서)">미니에 총</a>’으로 분류된다. <a href="/wiki/1853%EB%85%84" title="1853년">1853년</a>부터 <a href="/wiki/1866%EB%85%84" title="1866년">1866년</a>까지 영국군의 제식 소총으로 사용되었다. 53년형, 58년형, 61년형 등의 버전이 존재하는 것 외에 총신 길이가 다른 유형(2밴드, 3밴드)도 제조되었다. <a href="/wiki/1866%EB%85%84" title="1866년">1866년</a> 이후는 일체형의 탄피를 사용한 스나이더-엔필드 총(Snider-Enfield)으로 개조가 진행되었다.</p>
<p><a href="/wiki/%EC%9D%BC%EB%B3%B8" title="일본">일본</a>에서는 <a class="mw-redirect" href="/wiki/%EC%97%90%EB%8F%84_%EB%A7%89%EB%B6%80_%EB%A7%90%EA%B8%B0" title="에도 막부 말기">에도 막부 말기</a>에 대량으로 수입되어 <a href="/wiki/%EB%B3%B4%EC%8B%A0_%EC%A0%84%EC%9F%81" title="보신 전쟁">보신 전쟁</a>에서 신정부군의 주력으로 사용된 소총이었다.</p>

<h2><span class="mw-headline" id=".EC.97.AD.EC.82.AC">역사</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=1853%EB%85%84%ED%98%95_%EC%97%94%ED%95%84%EB%93%9C_%EA%B0%95%EC%84%A0%EB%A8%B8%EC%8A%A4%ED%82%B7&amp;action=edit&amp;section=1" title="부분 편집: 역사">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p><a href="/wiki/1847%EB%85%84" title="1847년">1847년</a>에 <a href="/wiki/%ED%94%84%EB%9E%91%EC%8A%A4" title="프랑스">프랑스</a>에서 <a class="new" href="/w/index.php?title=%EB%AF%B8%EB%8B%88%EC%97%90_%ED%83%84&amp;action=edit&amp;redlink=1" title="미니에 탄 (없는 문서)">미니에 탄</a>이 개발되면서 구미 각국에서 미니에 총 생산이 시작되었다. 특히 <a href="/wiki/%EC%98%81%EA%B5%AD" title="영국">영국</a>에서 개발된 엔필드 총은 우수하여, <a href="/wiki/1853%EB%85%84" title="1853년">1853년</a>에는 <a href="/wiki/%EC%98%81%EA%B5%AD%EA%B5%B0" title="영국군">영국군</a>의 제식 소총으로 채택되었다. <a class="mw-redirect" href="/wiki/%EC%84%B8%ED%8F%AC%EC%9D%B4%EC%9D%98_%EB%82%9C" title="세포이의 난">세포이의 난</a>이나 <a href="/wiki/%ED%81%AC%EB%A6%BC_%EC%A0%84%EC%9F%81" title="크림 전쟁">크림 전쟁</a>, <a href="/wiki/%ED%83%9C%ED%8F%89%EC%B2%9C%EA%B5%AD%EC%9D%98_%EB%82%9C" title="태평천국의 난">태평천국의 난</a>, <a href="/wiki/%EB%89%B4%EC%A7%88%EB%9E%9C%EB%93%9C" title="뉴질랜드">뉴질랜드</a>의 <a href="/wiki/%EB%A7%88%EC%98%A4%EB%A6%AC%EC%A1%B1" title="마오리족">마오리족</a> 진압 등으로 사용되어 높은 완성도를 증명했다.</p>
<p><a href="/wiki/%EB%AF%B8%EA%B5%AD" title="미국">미국</a> <a class="mw-disambig" href="/wiki/%EB%82%A8%EB%B6%81_%EC%A0%84%EC%9F%81" title="남북 전쟁">남북 전쟁</a>에서 <a href="/wiki/%EB%B6%81%EA%B5%B0" title="북군">북군</a>의 표준이었던 <a class="new" href="/w/index.php?title=%EC%8A%A4%ED%94%84%EB%A7%81%ED%95%84%EB%93%9C_%EC%B4%9D&amp;action=edit&amp;redlink=1" title="스프링필드 총 (없는 문서)">스프링필드 총</a>의 생산량이 수요를 따라가지 못했고, 당시 민간 군수품 제조자들이 잡종을 아무렇지도 않게 납품하는 경우가 많았기 때문에 ‘세계의 공장’이었던 <a href="/wiki/%EC%98%81%EA%B5%AD" title="영국">영국</a>에서 <a class="mw-redirect" href="/wiki/%EB%8C%80%EB%9F%89%EC%83%9D%EC%82%B0" title="대량생산">대량생산</a>되어 신뢰성이 높은 엔필드 총을 수입하게 되었다. 또한 공업 기반이 미숙한 <a href="/wiki/%EB%82%A8%EA%B5%B0" title="남군">남군</a>은 개전 초기부터 수입 무기에 의존하고 있었으며, 그 중에서도 엔필드 총은 명중률과 높은 위력으로 인해 남군에서도 많이 사용되었다. 영국 정부는 남부의 열세가 분명해지자 남부에 수출을 금지했지만, 무기상인에 의해 최종적으로 90만 정이 수출되었다.</p>
<p>엔필드 총을 역사상 유명하게 만든 것은 전용 종이탄피가 <a class="mw-redirect" href="/wiki/%EC%84%B8%ED%8F%AC%EC%9D%B4%EC%9D%98_%EB%82%9C" title="세포이의 난">세포이의 난</a>의 원인이 되었다는 점 때문이었다. 종이탄피는 각각의 상태로는 다루기 어려웠고, 전투 중에 발화 위험이 높았던 <a class="mw-redirect" href="/wiki/%ED%9D%91%EC%83%89_%ED%99%94%EC%95%BD" title="흑색 화약">흑색 화약</a>을 한발 분량으로 작게 나누어, <a href="/wiki/%EA%B7%B8%EB%A6%AC%EC%8A%A4" title="그리스">그리스</a>(당시는 동물성 소, 돼지기름이 대표적인 것이었다)를 바른 총알과 세트로 종이에 싸서 그 표면에 <a href="/wiki/%EB%B0%80%EB%9E%8D" title="밀랍">밀랍</a>과 <a class="mw-disambig" href="/wiki/%EC%9A%B0%EC%A7%80" title="우지">쇠기름</a>의 혼합물이 튄 것이었다.</p>
<p>이 탄약은 총신과 총알 사이에 종이를 물림으로써 총알이 총신과 마찰할 때 발생하는 고열로 인해 <a href="/wiki/%EB%82%A9" title="납">납</a>에 달라붙는 것을 예방하는 중요한 역할을 하였고, 표면에 바른 <a href="/wiki/%EB%B0%80%EB%9E%8D" title="밀랍">밀랍</a>과 우지는 마찰을 줄이기 위한 <a href="/wiki/%EC%9C%A4%ED%99%9C%EC%9C%A0" title="윤활유">윤활유</a> 역할을 했다. 또한 습기를 머금어 점화하기 어려워진 흑색 화약을 지키기 위해 화약 포장지(이하, 약포)의 표면을 코팅하여 방습한 것도 오래전부터 사용되어 온 일반적인 방법이었다.</p>
<p>이 탄약을 사용하여 엔필드 총을 장전할 때 먼저 입으로 약포의 끝을 물어뜯어, <a href="/wiki/%ED%99%94%EC%95%BD" title="화약">화약</a>을 <a class="mw-disambig" href="/wiki/%EC%B4%9D%EA%B5%AC" title="총구">총구</a>에 붓고, 약포가 포장된 상태에서 총알을</p>
