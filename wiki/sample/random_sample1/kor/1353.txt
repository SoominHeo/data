<p><b>OpenCV</b>(<span lang="en" xml:lang="en">Open Computer Vision</span>)은 <a href="/wiki/%EC%98%A4%ED%94%88_%EC%86%8C%EC%8A%A4" title="오픈 소스">오픈 소스</a> <a href="/wiki/%EC%BB%B4%ED%93%A8%ED%84%B0_%EB%B9%84%EC%A0%84" title="컴퓨터 비전">컴퓨터 비전</a> C <a class="mw-disambig" href="/wiki/%EB%9D%BC%EC%9D%B4%EB%B8%8C%EB%9F%AC%EB%A6%AC" title="라이브러리">라이브러리</a>이다. 원래는 <a href="/wiki/%EC%9D%B8%ED%85%94" title="인텔">인텔</a>이 개발하였다. <a class="mw-redirect" href="/wiki/%EB%A7%88%EC%9D%B4%ED%81%AC%EB%A1%9C%EC%86%8C%ED%94%84%ED%8A%B8_%EC%9C%88%EB%8F%84" title="마이크로소프트 윈도">윈도</a>, <a href="/wiki/%EB%A6%AC%EB%88%85%EC%8A%A4" title="리눅스">리눅스</a> 등의 여러 플랫폼에서 사용할 수 있다. <b>실시간</b> 이미지 프로세싱에 중점을 둔 라이브러리이다. <a href="/wiki/%EC%9D%B8%ED%85%94" title="인텔">인텔</a> <a href="/wiki/%EC%A4%91%EC%95%99_%EC%B2%98%EB%A6%AC_%EC%9E%A5%EC%B9%98" title="중앙 처리 장치">CPU</a>에서 사용되는 경우 속도의 향상을 볼 수 있는 <span lang="en" xml:lang="en">Intel Performance Primitives</span> (<a class="mw-disambig" href="/wiki/IPP" title="IPP">IPP</a>)를 지원한다.</p>
