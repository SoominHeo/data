<p><i><b>Overexposed</b></i>(《오버익스포즈드<sup class="reference" id="cite_ref-1"><a href="#cite_note-1">[1]</a></sup>》)는 <a href="/wiki/%EB%AF%B8%EA%B5%AD" title="미국">미국</a>의 <a href="/wiki/%ED%8C%9D_%EB%A1%9D" title="팝 록">팝 록</a> <a class="mw-redirect" href="/wiki/%EC%9D%8C%EC%95%85_%EA%B7%B8%EB%A3%B9" title="음악 그룹">밴드</a> <a href="/wiki/%EB%A7%88%EB%A3%AC_5" title="마룬 5">마룬 5</a>의 네 번째 <a href="/wiki/%EC%A0%95%EA%B7%9C_%EC%9D%8C%EB%B0%98" title="정규 음반">정규 음반</a>이다. 이전 음반들에서 주로 리드 보컬리스트 <a href="/wiki/%EC%95%A0%EB%8D%A4_%EB%A6%AC%EB%B0%94%EC%9D%B8" title="애덤 리바인">애덤 리바인</a>과 리드 기타리스트 <a href="/wiki/%EC%A0%9C%EC%9E%84%EC%8A%A4_%EB%B0%B8%EB%9F%B0%ED%83%80%EC%9D%B8" title="제임스 밸런타인">제임스 밸런타인</a>이 함께 작사작곡을 했지만, 이 음반에서 제임스 밸런타인은 네 곡밖에 쓰지 않았다. 마룬 5는 이 음반을 <a class="mw-disambig" href="/wiki/%ED%8C%9D" title="팝">팝</a>, <a href="/wiki/%ED%8C%9D_%EB%A1%9D" title="팝 록">팝 록</a>, <a href="/wiki/%EB%8C%84%EC%8A%A4_%ED%8C%9D" title="댄스 팝">댄스 팝</a> 그리고 <a href="/wiki/%EB%A0%88%EA%B2%8C" title="레게">레게</a>가 합쳐진 그들의 가장 팝적인 음반이라고 표현하였다.</p>
<p>이 음반은 <a href="/wiki/%EC%98%81%EA%B5%AD_%EC%9D%8C%EB%B0%98_%EC%B0%A8%ED%8A%B8" title="영국 음반 차트">영국 음반 차트</a>와 미국 <a href="/wiki/%EB%B9%8C%EB%B3%B4%EB%93%9C_200" title="빌보드 200">빌보드 200</a>에서 첫 주에 각각 28,000장과 222,000장의 판매고를 올리며 2위를 기록하였다. 2012년 말 기준으로 전세계적으로 220만장의 음반 판매고를 올리며, 2012년 한 해 열한 번째로 많이 팔린 음반이 되었다.<sup class="reference" id="cite_ref-2"><a href="#cite_note-2">[2]</a></sup></p>
<p><i>Overexposed</i>의 첫 번째, 두 번째 싱글인 "<a href="/wiki/Payphone" title="Payphone">Payphone</a>"과 "<a href="/wiki/One_More_Night_(%EB%A7%88%EB%A3%AC_5%EC%9D%98_%EB%85%B8%EB%9E%98)" title="One More Night (마룬 5의 노래)">One More Night</a>"는 미국 <a href="/wiki/%EB%B9%8C%EB%B3%B4%EB%93%9C_%ED%95%AB_100" title="빌보드 핫 100">빌보드 핫 100</a>에서 각각 2위와 9주 연속 1위를 기록하고, 전세계적으로 910만장과 690만장 이상의 판매고를 기록해 <a class="extiw" href="https://en.wikipedia.org/wiki/List_of_best-selling_singles" title="en:List of best-selling singles">세계에서 가장 많이 팔린 싱글 목록</a>에 이름을 올렸다.</p>

<h2><span class="mw-headline" id=".EB.B0.B0.EA.B2.BD">배경</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Overexposed&amp;action=edit&amp;section=1" title="부분 편집: 배경">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<h3><span class="mw-headline" id=".EC.98.81.ED.96.A5">영향</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Overexposed&amp;action=edit&amp;section=2" title="부분 편집: 영향">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p><i>Overexposed</i>는 <a href="/wiki/%EB%8C%84%EC%8A%A4_%ED%8C%9D" title="댄스 팝">댄스 팝</a>, <a href="/wiki/%EC%9D%BC%EB%A0%89%ED%8A%B8%EB%A1%9C%ED%8C%9D" title="일렉트로팝">일렉트로팝</a>, <a class="mw-redirect" href="/wiki/R%26B" title="R&amp;B">R&amp;B</a>와 <a href="/wiki/%EB%A0%88%EA%B2%8C" title="레게">레게</a>의 영향을 받은 <a href="/wiki/%ED%8C%9D_%EC%9D%8C%EC%95%85" title="팝 음악">팝</a><sup class="reference" id="cite_ref-americansongwriter_3-0"><a href="#cite_note-americansongwriter-3">[3]</a></sup> 및 <a href="/wiki/%ED%8C%9D_%EB%A1%9D" title="팝 록">팝 록</a><sup class="reference" id="cite_ref-allmusic_4-0"><a href="#cite_note-allmusic-4">[4]</a></sup><sup class="reference" id="cite_ref-ew_5-0"><a href="#cite_note-ew-5">[5]</a></sup> 음반이다.<sup class="reference" id="cite_ref-billboard_review_6-0"><a href="#cite_note-billboard_review-6">[6]</a></sup> 많은 비평가들이 말하길 "<a href="/wiki/Moves_Like_Jagger" title="Moves Like Jagger">Moves Like Jagger</a>"가 <i>Overexposed</i>에 많은 영향을 끼쳤다고 한다.<sup class="reference" id="cite_ref-ew_5-1"><a href="#cite_note-ew-5">[5]</a></sup><sup class="reference" id="cite_ref-musicOMH_7-0"><a href="#cite_note-musicOMH-7">[7]</a></sup><sup class="reference" id="cite_ref-guardian_8-0"><a href="#cite_note-guardian-8">[8]</a></sup> 또한 <a href="/wiki/%EB%A6%AC%ED%95%9C%EB%82%98" title="리한나">리한나</a> 역시 영향을 끼쳤다고 한다.<sup class="reference" id="cite_ref-ew_5-2"><a href="#cite_note-ew-5">[5]</a></sup><sup class="reference" id="cite_ref-musicOMH_7-1"><a href="#cite_note-musicOMH-7">[7]</a></sup></p>
<h3><span class="mw-headline" id=".EB.85.B8.EB.9E.98.EC.99.80_.EA.B0.80.EC.82.AC">노래와 가사</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Overexposed&amp;action=edit&amp;section=3" title="부분 편집: 노래와 가사">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>음반의 첫 번째 곡인 "<a href="/wiki/One_More_Night_(%EB%A7%88%EB%A3%AC_5%EC%9D%98_%EB%85%B8%EB%9E%98)" title="One More Night (마룬 5의 노래)">One More Night</a>"는 유혹적인 섹시 레게 곡<sup class="reference" id="cite_ref-rs_9-0"><a href="#cite_note-rs-9">[9]</a></sup> 으로, <a href="/wiki/%ED%8C%9D_%EC%9D%8C%EC%95%85" title="팝 음악">팝</a>과 <a href="/wiki/%EB%A1%9D_%EC%9D%8C%EC%95%85" title="록 음악">록</a>의 영향을 받았다.<sup class="reference" id="cite_ref-popcrush_10-0"><a href="#cite_note-popcrush-10">[10]</a></sup> 가사는 특정한 사랑 관계가 얽힌 것이 풀리는 이야기이다.<sup class="reference" id="cite_ref-popcrush_10-1"><a href="#cite_note-popcrush-10">[10]</a></sup> 음반의 두 번째 곡인 "Payphone"은 팝과 R&amp;B 곡<sup class="reference" id="cite_ref-billboard_review_6-1"><a href="#cite_note-billboard_review-6">[6]</a></sup> 으로 <a href="/wiki/%ED%9E%99%ED%95%A9" title="힙합">힙합</a> 비트와 <a href="/wiki/%ED%94%BC%EC%95%84%EB%85%B8" title="피아노">피아노</a> 후크를 포함한다.<sup class="reference" id="cite_ref-digital_spy_review_11-0"><a href="#cite_note-digital_spy_review-11">[11]</a></sup> 미국의 래퍼 <a href="/wiki/%EC%9C%84%EC%A6%88_%EC%B9%BC%EB%A6%AC%ED%8C%8C" title="위즈 칼리파">위즈 칼리파</a>가 피처링했으며, 이 곡은 과거의 관계에 대한 이야기이다.<sup class="reference" id="cite_ref-popcrush2_12-0"><a href="#cite_note-popcrush2-12">[12]</a></sup> "<a href="/wiki/Daylight_(%EB%A7%88%EB%A3%AC_5%EC%9D%98_%EB%85%B8%EB%9E%98)" title="Daylight (마룬 5의 노래)">Daylight</a>" 는 <a href="/wiki/%EC%86%8C%ED%94%84%ED%8A%B8_%EB%A1%9D" title="소프트 록">소프트 록</a> 곡으로,<sup class="reference" id="cite_ref-billboard.com_13-0"><a href="#cite_note-billboard.com-13">[13]</a></sup> 영국의 밴드 <a href="/wiki/%EC%BD%9C%EB%93%9C%ED%94%8C%EB%A0%88%EC%9D%B4" title="콜드플레이">콜드플레이</a>의 곡이 떠오른다는 평가를 받았다. "Lucky Strike"는 <a class="mw-redirect" href="/wiki/%ED%9B%B5%ED%81%AC" title="훵크">훵크</a> 곡으로<sup class="reference" id="cite_ref-ew_5-3"><a href="#cite_note-ew-5">[5]</a></sup> <a class="mw-redirect" href="/wiki/%EB%8D%A5%EC%8A%A4%ED%85%9D" title="덥스텝">덥스텝</a>과 <a class="new" href="/w/index.php?title=%EC%86%94_%EB%A1%9D&amp;action=edit&amp;redlink=1" title="솔 록 (없는 문서)">솔 록</a>의 영향을 받았다.<sup class="reference" id="cite_ref-artistdirect_14-0"><a href="#cite_note-artistdirect-14">[14]</a></sup><sup class="reference" id="cite_ref-nmereview_15-0"><a href="#cite_note-nmereview-15">[15]</a></sup> 또한 가사에 약간의 성적인 내용을 담고 있기도 하다.<sup class="reference" id="cite_ref-Plugged_In_16-0"><a href="#cite_note-Plugged_In-16">[16]</a></sup> 다섯 번째 곡인 "The Man Who Never Lied"는 팝 및 팝 록 곡이며,<sup class="reference" id="cite_ref-star_pulse_17-0"><a href="#cite_note-star_pulse-17">[17]</a></sup> 여섯 번째 곡인 "<a href="/wiki/Love_Somebody_(%EB%A7%88%EB%A3%AC_5%EC%9D%98_%EB%85%B8%EB%9E%98)" title="Love Somebody (마룬 5의 노래)">Love Somebody</a>" 댄스 팝 곡으로,<sup class="reference" id="cite_ref-billboard.com_13-1"><a href="#cite_note-billboard.com-13">[13]</a></sup> <a class="mw-redirect" href="/wiki/%EC%8B%A0%EB%94%94%EC%82%AC%EC%9D%B4%EC%A0%80" title="신디사이저">신디사이저</a> 후크가 어우러진 곡이다.<sup class="reference" id="cite_ref-digitalspy_18-0"><a href="#cite_note-digitalspy-18">[18]</a></sup> 일곱 번째 곡 "Ladykiller"는 곡 중간의 기타 솔로가<sup class="reference" id="cite_ref-bbcmusic_19-0"><a href="#cite_note-bbcmusic-19">[19]</a></sup> 애덤 리바인의 <a class="mw-redirect" href="/wiki/%EB%B8%94%EB%A3%A8%EC%95%84%EC%9D%B4%EB%93%9C_%EC%86%8C%EC%9A%B8" title="블루아이드 소울">블루아이드 소울</a>적인 목소리와 훵크적 그루브와 어우러진다.<sup class="reference" id="cite_ref-independent_20-0"><a href="#cite_note-independent-20">[20]</a></sup> <a class="mw-redirect" href="/wiki/BBC" title="BBC">BBC 뮤직</a>과 <a href="/wiki/%EB%B2%84%EC%A7%84_%EB%AF%B8%EB%94%94%EC%96%B4" title="버진 미디어">버진 미디어</a>에 따르면 이는 <a href="/wiki/%EB%A7%88%EC%9D%B4%ED%81%B4_%EC%9E%AD%EC%8A%A8" title="마이클 잭슨">마이클 잭슨</a>에게 영감을 받았다고 한다.<sup class="reference" id="cite_ref-bbcmusic_19-1"><a href="#cite_note-bbcmusic-19">[19]</a></sup><sup class="reference" id="cite_ref-virgin_21-0"><a href="#cite_note-virgin-21">[21]</a></sup></p>
<p>여덞 번째 곡 "Fortune Teller" 은 팝 피아노 비트가 어우러진 곡으로 여러 감정을 표현하며,<sup class="reference" id="cite_ref-billboard.com_13-2"><a href="#cite_note-billboard.com-13">[13]</a></sup> 다음 곡인 "Sad"는 피아노 발라드 곡이다.<sup class="reference" id="cite_ref-billboard.com_13-3"><a href="#cite_note-billboard.com-13">[13]</a></sup>.<sup class="reference" id="cite_ref-RTE_22-0"><a href="#cite_note-RTE-22">[22]</a></sup> 열 번째 곡 "Tickets"로, 마룬 5의 2010년 싱글 "<a class="mw-redirect" href="/wiki/Give_A_Little_More" title="Give A Little More">Give A Little More</a>"의 영향을 받았다고 전해진다.<sup class="reference" id="cite_ref-billboard.com_13-4"><a href="#cite_note-billboard.com-13">[13]</a></sup> 그 다음 곡은 "Doin' Dirt"이며, 마지막 열 두 번째 곡은 발라드 곡인 "Beautiful Goodbye"이다.<sup class="reference" id="cite_ref-rs_9-1"><a href="#cite_note-rs-9">[9]</a></sup> <i>Overexposed</i>의 딜럭스반에는 "Wipe Your Eyes", "Wasted Years"와 <a href="/wiki/%ED%94%84%EB%A6%B0%EC%8A%A4" title="프린스">프린스</a>의 1986년 싱글의 커버 곡인 "Kiss" 등 세 곡의 보너스 트랙이 수록되어있다. Wipe Your Eyes"는 중간 템포의 피아노 발라드 곡으로 <a href="/wiki/%EB%A7%90%EB%A6%AC" title="말리">말리</a>의 음악 듀오인 아마두 앤 마리암(Amadou &amp; Mariam)의 "Sabali"를 샘플링하였다.<sup class="reference" id="cite_ref-idolator_23-0"><a href="#cite_note-idolator-23">[23]</a></sup></p>
<h2><span class="mw-headline" id=".EC.9D.8C.EB.B0.98_.EC.BB.A4.EB.B2.84">음반 커버</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Overexposed&amp;action=edit&amp;section=4" title="부분 편집: 음반 커버">편집</a><span class="mw-editsection-bracket">]</span></span></h2>

<h3><span class="mw-headline" id=".EB.8C.80.ED.95.9C.EB.AF.BC.EA.B5.AD_.EB.B0.9C.EB.A7.A4.EB.B0.98">대한민국 발매반</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Overexposed&amp;action=edit&amp;section=13" title="부분 편집: 대한민국 발매반">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>대한민국에서는 기존 딜럭스반이 발매된 후 새로운 딜럭스반이 발매되었다. (15번째 곡인 <a href="/wiki/%ED%94%84%EB%A6%B0%EC%8A%A4_(%EA%B0%80%EC%88%98)" title="프린스 (가수)">프린스</a> 커버 곡 "Kiss"가 <a class="mw-redirect" href="/wiki/%EC%95%8C_%EA%B7%B8%EB%A6%B0" title="알 그린">알 그린</a> 커버 곡 "Let's Stay Together"로 변경됨.) 새로운 딜럭스반에는 초도물량 2,800장 한정으로 <i>Overexposed</i>의 음반 커버가 그려진 스페셜 에디션 <a href="/wiki/%ED%8C%9D%EC%B9%B4%EB%93%9C" title="팝카드">팝카드</a>가 포함되어 있다.</p>
<h2><span class="mw-headline" id=".ED.88.AC.EC.96.B4">투어</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Overexposed&amp;action=edit&amp;section=14" title="부분 편집: 투어">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<div class="rellink relarticle mainarticle"><img alt="" data-file-height="128" data-file-width="128" height="16" src="//upload.wikimedia.org/wikipedia/commons/thumb/e/ec/Crystal_Clear_app_xmag.svg/16px-Crystal_Clear_app_xmag.svg.png" srcset="//upload.wikimedia.org/wikipedia/commons/thumb/e/ec/Crystal_Clear_app_xmag.svg/24px-Crystal_Clear_app_xmag.svg.png 1.5x, //upload.wikimedia.org/wikipedia/commons/thumb/e/ec/Crystal_Clear_app_xmag.svg/32px-Crystal_Clear_app_xmag.svg.png 2x" width="16"/> 이 부분의 본문은 <a href="/wiki/Overexposed_Tour" title="Overexposed Tour">Overexposed Tour</a>입니다.</div>
<p><a href="/wiki/%EB%A7%88%EB%A3%AC_5" title="마룬 5">마룬 5</a>는 음반 발매 후, 2012년에서 2013년까지 전 세계적으로 투어를 하였다. 이 투어의 일환으로 2012년 9월 14, 15일 <a class="mw-redirect" href="/wiki/%EB%B6%80%EC%82%B0" title="부산">부산</a>과 <a class="mw-redirect" href="/wiki/%EC%84%9C%EC%9A%B8" title="서울">서울</a>에서 세 번째 내한 공연을 가졌다.</p>
