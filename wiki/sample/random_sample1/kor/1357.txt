<p><b>핑크니 벤턴 스튜어트 핀치백</b>(Pinckney Benton Stewart Pinchback, <a href="/wiki/1837%EB%85%84" title="1837년">1837년</a> <a href="/wiki/5%EC%9B%94_10%EC%9D%BC" title="5월 10일">5월 10일</a> ~ <a href="/wiki/1921%EB%85%84" title="1921년">1921년</a> <a href="/wiki/12%EC%9B%94_21%EC%9D%BC" title="12월 21일">12월 21일</a>)는 <a href="/wiki/%EB%AF%B8%EA%B5%AD" title="미국">미국</a>의 정치인이며, <a href="/wiki/1872%EB%85%84" title="1872년">1872년</a> <a href="/wiki/12%EC%9B%94" title="12월">12월</a>부터 35일간 <a href="/wiki/%EB%A3%A8%EC%9D%B4%EC%A7%80%EC%95%A0%EB%82%98_%EC%A3%BC" title="루이지애나 주">루이지애나 주</a> 지사를 지냈다.</p>
<p><a href="/wiki/%EC%A1%B0%EC%A7%80%EC%95%84_%EC%A3%BC" title="조지아 주">조지아 주</a> 메이콘에서 <a href="/wiki/%EB%86%8D%EC%9E%A5" title="농장">농장</a> 경영인 아버지와 해방 <a class="mw-redirect" href="/wiki/%EB%85%B8%EC%98%88" title="노예">노예</a> 어머니 사이에서 태어났는 데, 그의 핏줄은 복잡하다고 알려져있다.</p>
<p><a href="/wiki/%EA%B3%B5%ED%99%94%EB%8B%B9_(%EB%AF%B8%EA%B5%AD)" title="공화당 (미국)">공화당</a> 소속인 그는 <a href="/wiki/1871%EB%85%84" title="1871년">1871년</a> 루이지애나 주의 부지사가 되었고, <a href="/wiki/1872%EB%85%84" title="1872년">1872년</a>과 <a href="/wiki/1873%EB%85%84" title="1873년">1873년</a> <a class="new" href="/w/index.php?title=%ED%97%A8%EB%A6%AC_C._%EC%9B%8C%EB%A8%B8%EC%8A%A4&amp;action=edit&amp;redlink=1" title="헨리 C. 워머스 (없는 문서)">헨리 C. 워머스</a> 지사의 탄핵으로 6주간 주지사의 직무 대행으로 근무하였다. <a href="/wiki/%EC%95%84%ED%94%84%EB%A6%AC%EC%B9%B4%EA%B3%84_%EB%AF%B8%EA%B5%AD%EC%9D%B8" title="아프리카계 미국인">흑인</a>으로서 최초의 주지사가 되어 35일간 그 직에 있었다. 1872년에는 <a class="mw-redirect" href="/wiki/%ED%95%98%EC%9B%90_%EC%9D%98%EC%9B%90" title="하원 의원">하원 의원</a>에, 1873년에는 <a href="/wiki/%EB%AF%B8%EA%B5%AD_%EC%83%81%EC%9B%90" title="미국 상원">상원</a>에 선출되었다. 그의 상대들은 두차례의 선거에서 법률을 어겨진 것을 고소하였다. 루이지애나의 백인 공무원들이 법적으로 선출되어 같은 진행으로 선택되었음에도 불구하고, 상·하원 양쪽에서는 핀치백의 의원직을 거부하였다.</p>
<p><a href="/wiki/1870%EB%85%84" title="1870년">1870년</a>부터 <a href="/wiki/1881%EB%85%84" title="1881년">1881년</a>까지 주간 신문 루이지에이니언(The Louisianian)을 발간하였다.</p>
<p><a href="/wiki/1921%EB%85%84" title="1921년">1921년</a> <a href="/wiki/12%EC%9B%94_21%EC%9D%BC" title="12월 21일">12월 21일</a> <a href="/wiki/%EC%9B%8C%EC%8B%B1%ED%84%B4_D.C." title="워싱턴 D.C.">워싱턴 D.C.</a>에서 84세의 나이로 사망하였다.</p>
