<p><b><a href="/wiki/%ED%83%9C%EC%96%91%EA%B3%84%EC%9D%98_%ED%98%95%EC%84%B1%EA%B3%BC_%EC%A7%84%ED%99%94" title="태양계의 형성과 진화">태양계의 형성과 진화 이론</a></b>은 <a href="/wiki/%ED%83%9C%EC%96%91%EA%B3%84" title="태양계">태양계</a>의 탄생부터 죽음에 이르는 일련의 과정을 연구하여 그 과정을 이론으로 정립한 것이다. 태양계 생성 이론은 수세기에 걸쳐 발전했지만, 근대적 이론의 틀을 갖춘 것은 <a href="/wiki/18%EC%84%B8%EA%B8%B0" title="18세기">18세기</a>에 이르러서였다. <a href="/wiki/1950%EB%85%84%EB%8C%80" title="1950년대">1950년대</a>에 우주 시대가 열리고 <a href="/wiki/1990%EB%85%84%EB%8C%80" title="1990년대">1990년대</a> 중반 이후 <a href="/wiki/%EC%99%B8%EA%B3%84_%ED%96%89%EC%84%B1" title="외계 행성">외계 행성</a>이 본격적으로 발견되면서, 태양계의 생성과 소멸에 대한 기존 이론들은 도전을 받게 되었고, 동시에 더욱 다듬어지게 된다. 지구로 전송된 바깥 세계에 대한 정보는 태양계에 대한 사람들의 이해를 촉진했다. 동시에 <a href="/wiki/%ED%95%B5%EB%AC%BC%EB%A6%AC%ED%95%99" title="핵물리학">핵물리학</a>의 발전은 항성에 대한 지식을 증진시켰고, 항성의 탄생 및 궁극적 최후에 관한 이론 수립에 기여하게 된다.</p>
<p>현재의 태양계 형성 이론은 성운 이론으로 이는 <a href="/wiki/%EC%97%90%EB%A7%88%EB%88%84%EC%97%98_%EC%8A%A4%EB%B2%A0%EB%8D%B4%EB%B3%B4%EB%A6%AC" title="에마누엘 스베덴보리">에마누엘 스베덴보리</a>가 <a href="/wiki/1734%EB%85%84" title="1734년">1734년</a> 제시한 학설로 이후 <a href="/wiki/1755%EB%85%84" title="1755년">1755년</a> <a href="/wiki/%EC%9D%B4%EB%A7%88%EB%88%84%EC%97%98_%EC%B9%B8%ED%8A%B8" title="이마누엘 칸트">이마누엘 칸트</a>가 보충하고 범위를 확장시켰다. <a href="/wiki/1796%EB%85%84" title="1796년">1796년</a> <a href="/wiki/%ED%94%BC%EC%97%90%EB%A5%B4%EC%8B%9C%EB%AA%BD_%EB%9D%BC%ED%94%8C%EB%9D%BC%EC%8A%A4" title="피에르시몽 라플라스">피에르시몽 라플라스</a>가 유사한 이론을 독자적으로 수립했다.</p>

<p><b><a href="/wiki/%EB%B9%84%EC%9A%98%EC%84%B8_%EB%86%80%EC%8A%A4" title="비욘세 놀스">비욘세 놀스</a></b>(Beyoncé Knowles, <a href="/wiki/1981%EB%85%84" title="1981년">1981년</a> <a href="/wiki/9%EC%9B%94_4%EC%9D%BC" title="9월 4일">9월 4일</a> ~)는 <a href="/wiki/%EB%AF%B8%EA%B5%AD" title="미국">미국</a>의 <a class="mw-redirect" href="/wiki/%EC%95%8C%EC%95%A4%EB%B9%84" title="알앤비">알앤비</a> 가수, 배우, 댄서이자 패션 디자이너이다. 흔히 <b>비욘세</b>(Beyoncé)로도 알려져있다. 비욘세는 텍사스 주의 <a href="/wiki/%ED%9C%B4%EC%8A%A4%ED%84%B4" title="휴스턴">휴스턴</a>에서 태어나 어린 시절에는 예술 초등학교에 입학해 그곳에서 다양한 공연을 하고 대회에 나가면서 가수의 꿈을 키워나갔다. 이후 비욘세가 인기를 얻는 계기가 된 1990년대 말 알앤비 3인조 걸 그룹 <a href="/wiki/%EB%8D%B0%EC%8A%A4%ED%8B%B0%EB%8B%88%EC%8A%A4_%EC%B0%A8%EC%9D%BC%EB%93%9C" title="데스티니스 차일드">데스티니스 차일드</a>의 리더로써 활동하였고 이 그룹은 약 5000만 장이 넘는 판매고를 올리며 미국에서 음반을 가장 많이 판 걸 그룹으로 남았다.</p>
<p>그룹이 인기를 얻자 비욘세는 2003년에 첫 솔로 앨범 <i><a href="/wiki/Dangerously_in_Love" title="Dangerously in Love">Dangerously in Love</a></i>를 발매하면서 솔로 데뷔를 치렀다. 2005년 6월에는 비욘세가 가수로서 성공의 계기가 된 <a href="/wiki/%EB%8D%B0%EC%8A%A4%ED%8B%B0%EB%8B%88%EC%8A%A4_%EC%B0%A8%EC%9D%BC%EB%93%9C" title="데스티니스 차일드">데스티니스 차일드</a>가 해체를 맞이하지만, 다음 해 발매한 2집 <i><a href="/wiki/B%27Day" title="B'Day">B'Day</a></i>는 전 세계에서 약 700만장 넘는 판매고를 올렸으며, 앨범 수록곡인 "<a class="mw-redirect mw-disambig" href="/wiki/Deja_Vu" title="Deja Vu">Deja Vu</a>", "Ring The Alarm", "<a href="/wiki/Irreplaceable" title="Irreplaceable">Irreplaceable</a>"이 인기를 얻었다. 2008년 4월에는 힙합 가수인 <a class="mw-redirect" href="/wiki/%EC%A0%9C%EC%9D%B4-%EC%A7%80" title="제이-지">제이-지</a>와 결혼했다. 이후 세 번째 앨범인 <i><a href="/wiki/I_Am..._Sasha_Fierce" title="I Am... Sasha Fierce">I Am... Sasha Fierce</a></i>를 발매한 뒤, "<a href="/wiki/Single_Ladies_(Put_a_Ring_on_It)" title="Single Ladies (Put a Ring on It)">Single Ladies (Put a Ring on It)</a>"라는 히트곡을 남겼다. 이 음반은 <a href="/wiki/%EB%B9%8C%EB%B3%B4%EB%93%9C_200" title="빌보드 200">빌보드 200</a> 1위에 올랐고, <a href="/wiki/%EA%B7%B8%EB%9E%98%EB%AF%B8%EC%83%81" title="그래미상">그래미상</a> 10개 부문에 후보로 선정되었다.</p>

<ul>
<li>우리 주변의 <a href="/wiki/%EC%9D%98%EC%95%BD%ED%92%88" title="의약품">의약품</a>에도 붙여지는 <b><a href="/wiki/ATC_%EC%BD%94%EB%93%9C" title="ATC 코드">ATC 코드</a></b>는 약물의 화학 특성과 작용되는 장기 계통에 따라 총 14개의 <a class="mw-redirect" href="/wiki/%EC%95%8C%ED%8C%8C%EB%B2%B3" title="알파벳">알파벳</a>으로 나뉘는 분류법입니다. 1976년 처음 배포된 이래 <a class="mw-redirect" href="/wiki/%EC%84%B8%EA%B3%84%EB%B3%B4%EA%B1%B4%EA%B8%B0%EA%B5%AC" title="세계보건기구">세계보건기구</a> 산하 기관인 의약품 통계법 협력센터(WHOCC)가 관리하고 있습니다.</li>
<li><b><a href="/wiki/1900%EB%85%84_%ED%95%98%EA%B3%84_%EC%98%AC%EB%A6%BC%ED%94%BD" title="1900년 하계 올림픽">1900년 하계 올림픽</a></b><small>(사진)</small>은 <a class="mw-redirect" href="/wiki/1900%EB%85%84_%EB%A7%8C%EA%B5%AD%EB%B0%95%EB%9E%8C%ED%9A%8C" title="1900년 만국박람회">1900년 만국박람회</a>의 부대 행사로 치뤄진 올림픽 대회로, 독자적인 올림픽 조직위원회가 따로 없었습니다. 그 때문에 여러 종목들이 난립한 채 경기가 진행되었고, 대회 공식 보고서도 없어 상당수의 경기 기록이 전해지지 않고 있습니다.</li>
<li><a href="/wiki/%EB%85%B8%EB%B2%A8_%EB%AC%BC%EB%A6%AC%ED%95%99%EC%83%81" title="노벨 물리학상">노벨 물리학상</a>과 <a href="/wiki/%EB%85%B8%EB%B2%A8_%ED%99%94%ED%95%99%EC%83%81" title="노벨 화학상">노벨 화학상</a>을 모두 수상한 것으로 유명한 <b><a href="/wiki/%EB%A7%88%EB%A6%AC_%ED%80%B4%EB%A6%AC" title="마리 퀴리">마리 퀴리</a></b>는 두 개 이상의 국가에서 화폐 도안의 초상으로 사용된 유일한 인물이기도 합니다. 그녀의 초상은 <a href="/wiki/%ED%8F%B4%EB%9E%80%EB%93%9C" title="폴란드">폴란드</a>의 20,000 <a href="/wiki/%ED%8F%B4%EB%9E%80%EB%93%9C_%EC%A6%88%EC%9B%8C%ED%8B%B0" title="폴란드 즈워티">즈워티</a>와 <a href="/wiki/%ED%94%84%EB%9E%91%EC%8A%A4" title="프랑스">프랑스</a>의 500 <a href="/wiki/%ED%94%84%EB%9E%91%EC%8A%A4_%ED%94%84%EB%9E%91" title="프랑스 프랑">프랑</a>의 화폐 도안으로 사용된 적 있습니다.</li>
<li><a href="/wiki/%ED%88%AC%EB%A5%B4%ED%81%AC%EB%A9%94%EB%8B%88%EC%8A%A4%ED%83%84" title="투르크메니스탄">투르크메니스탄</a>의 전 대통령 <a href="/wiki/%EC%82%AC%ED%8C%8C%EB%A5%B4%EB%AC%B4%EB%9D%BC%ED%8A%B8_%EB%8B%88%EC%95%BC%EC%A1%B0%ED%94%84" title="사파르무라트 니야조프">사파르무라트 니야조프</a>는 과일인 <a href="/wiki/%EB%A9%9C%EB%A1%A0" title="멜론">멜론</a>을 매우 좋아하여 매년 8월 둘째 주 일요일을 멜론의 날이라는 기념일을 지정하기도 했습니다.</li>
<li><a href="/wiki/%EB%AF%B8%EA%B5%AD" title="미국">미국</a>에서 <a class="mw-redirect" href="/wiki/%EB%8F%99%EC%84%B1%EC%95%A0%EC%9E%90" title="동성애자">동성애자</a> 남성을 지칭하는 은어인 <b><a href="/wiki/%EB%8F%84%EB%A1%9C%EC%8B%9C%EC%9D%98_%EC%B9%9C%EA%B5%AC" title="도로시의 친구">도로시의 친구</a></b>는 동성애가 불법이던 <a href="/wiki/1940%EB%85%84%EB%8C%80" title="1940년대">1940년대</a>에 만들어진 말로, 단어의 의미를 알지 못했던 <a class="mw-redirect" href="/wiki/%EB%AF%B8%EA%B5%AD_%ED%95%B4%EA%B5%B0%EB%B2%94%EC%A3%84%EC%A1%B0%EC%82%AC%EC%B2%AD" title="미국 해군범죄조사청">미국 해군범죄조사청</a>은 동성애자 군인을 색출하고자 도로시라는 이름의 여성을 찾는 조사를 벌이기도 하였습니다.</li>
</ul>

<a href="/wiki/%EC%9C%84%ED%82%A4%EB%B0%B1%EA%B3%BC:%EC%86%8C%EA%B0%9C" title="위키백과:소개">위키백과 소개</a></div>
<p><b><a href="/wiki/%EC%9C%84%ED%82%A4%EB%B0%B1%EA%B3%BC" title="위키백과">위키백과</a></b>는 <a href="/wiki/%EC%9C%84%ED%82%A4" title="위키">위키</a>를 이용하여 전 세계 사람들이 함께 만들어가는 <a href="/wiki/%EC%9B%94%EB%93%9C_%EC%99%80%EC%9D%B4%EB%93%9C_%EC%9B%B9" title="월드 와이드 웹">웹 기반</a>의 다언어 <a href="/wiki/%EB%B0%B1%EA%B3%BC%EC%82%AC%EC%A0%84" title="백과사전">백과사전</a>입니다. 위키백과는 <a href="/wiki/%EC%9C%84%ED%82%A4%EB%B0%B1%EA%B3%BC:%EC%A4%91%EB%A6%BD%EC%A0%81_%EC%8B%9C%EA%B0%81" title="위키백과:중립적 시각">중립적</a>이고 <a href="/wiki/%EC%9C%84%ED%82%A4%EB%B0%B1%EA%B3%BC:%ED%99%95%EC%9D%B8_%EA%B0%80%EB%8A%A5" title="위키백과:확인 가능">검증 가능</a>한 <a class="mw-redirect" href="/wiki/%EC%9E%90%EC%9C%A0_%EC%BD%98%ED%85%90%EC%B8%A0" title="자유 콘텐츠">자유 콘텐츠</a> 백과사전의 제공을 목적으로 하는 프로젝트로, 누구나 참여하여 문서를 수정하고 발전시킬 수 있습니다.</p>
<p>위키백과는 <a href="/wiki/%EC%9C%84%ED%82%A4%EB%B0%B1%EA%B3%BC:%EB%8B%A4%EC%84%AF_%EC%9B%90%EC%B9%99" title="위키백과:다섯 원칙">다섯 가지 기본 원칙</a>에 따라 운영됩니다. 모든 문서는 <span class="plainlinks"><a class="external text" href="http://creativecommons.org/licenses/by-sa/3.0/deed.ko" rel="nofollow">크리에이티브 커먼즈 저작자표시-동일조건변경허락 3.0</a></span>에 따라 사용할 수 있으며, 복사, 수정과 배포가 자유롭고 상업적 목적의 사용도 가능합니다.</p>

위키백과 참여하기</div>
<p>모든 사용자는 위키백과에 직접 참여해 <a href="/wiki/%EC%9C%84%ED%82%A4%EB%B0%B1%EA%B3%BC:%ED%99%95%EC%9D%B8_%EA%B0%80%EB%8A%A5" title="위키백과:확인 가능">확인 가능 지침</a>과 <a href="/wiki/%EC%9C%84%ED%82%A4%EB%B0%B1%EA%B3%BC:%EB%AC%B8%EC%84%9C_%EB%93%B1%EC%9E%AC_%EA%B8%B0%EC%A4%80" title="위키백과:문서 등재 기준">문서 등재 기준 지침</a>, <a href="/wiki/%EC%9C%84%ED%82%A4%EB%B0%B1%EA%B3%BC:%ED%8E%B8%EC%A7%91_%EC%A7%80%EC%B9%A8" title="위키백과:편집 지침">편집 지침</a> 등 위키백과의 <a href="/wiki/%EC%9C%84%ED%82%A4%EB%B0%B1%EA%B3%BC:%EC%A0%95%EC%B1%85%EA%B3%BC_%EC%A7%80%EC%B9%A8" title="위키백과:정책과 지침">기본적인 규칙</a>을 준수하여 문서를 수정할 수 있습니다. 문서 <a href="/wiki/%EC%9C%84%ED%82%A4%EB%B0%B1%EA%B3%BC:%ED%8E%B8%EC%A7%91%EC%9D%80_%EA%B3%BC%EA%B0%90%ED%95%98%EA%B2%8C" title="위키백과:편집은 과감하게">편집은 과감하게</a> 해주세요.</p>
<p>위키백과에는 여러분을 돕기 위해 많은 문서들이 준비되어 있습니다. <a href="/wiki/%EC%9C%84%ED%82%A4%EB%B0%B1%EA%B3%BC:%EA%B8%B8%EB%9D%BC%EC%9E%A1%EC%9D%B4" title="위키백과:길라잡이">기본적인 길라잡이</a>부터 <a href="/wiki/%EC%9C%84%ED%82%A4%EB%B0%B1%EA%B3%BC:%EC%83%88_%EB%AC%B8%EC%84%9C_%EB%A7%8C%EB%93%A4%EA%B8%B0" title="위키백과:새 문서 만들기">새 문서를 만드는 방법</a>, <a href="/wiki/%EC%9C%84%ED%82%A4%EB%B0%B1%EA%B3%BC:%EA%B7%B8%EB%A6%BC_%EC%82%AC%EC%9A%A9%ED%95%98%EA%B8%B0" title="위키백과:그림 사용하기">그림 사용 방법</a> 등이 친절히 설명되어 있습니다. 위키백과나 사용법에 대하여 궁금한 점이 있으시다면 <a href="/wiki/%EC%9C%84%ED%82%A4%EB%B0%B1%EA%B3%BC:FAQ" title="위키백과:FAQ">FAQ</a>을 참고하시고 <a href="/wiki/%EC%9C%84%ED%82%A4%EB%B0%B1%EA%B3%BC:%EC%A7%88%EB%AC%B8%EB%B0%A9" title="위키백과:질문방">질문방</a>에 질문하는 것을 주저하지 마세요. <a href="/wiki/%EC%9C%84%ED%82%A4%EB%B0%B1%EA%B3%BC:%EC%9A%94%EC%B2%AD" title="위키백과:요청">요청</a>을 남기면 다른 사용자들의 도움을 받을 수도 있습니다.</p>
