<p><b>QR 코드</b>(<span style="font-size: smaller;"><a href="/wiki/%EC%98%81%EC%96%B4" title="영어">영어</a>:</span> <span lang="en" xml:lang="en">QR code</span>, Quick Response code)은 흑백 격자무늬 패턴으로 정보를 나타내는 매트릭스 형식의 이차원 <a href="/wiki/%EB%B0%94%EC%BD%94%EB%93%9C" title="바코드">바코드</a>이다. QR코드는 주로 <a href="/wiki/%ED%95%9C%EA%B5%AD" title="한국">한국</a>, <a href="/wiki/%EC%9D%BC%EB%B3%B8" title="일본">일본</a>, <a href="/wiki/%EC%A4%91%EA%B5%AD" title="중국">중국</a>, <a href="/wiki/%EC%98%81%EA%B5%AD" title="영국">영국</a>, <a href="/wiki/%EB%AF%B8%EA%B5%AD" title="미국">미국</a> 등에서 많이 사용되며 명칭은 <a class="mw-redirect" href="/wiki/%EB%8D%B4%EC%86%8C_%EC%9B%A8%EC%9D%B4%EB%B8%8C" title="덴소 웨이브">덴소 웨이브</a>의 등록 상표 'Quick Response'에서 유래하였다. 종래에 많이 쓰이던 <a href="/wiki/%EB%B0%94%EC%BD%94%EB%93%9C" title="바코드">바코드</a>의 용량 제한을 극복하고 그 형식과 내용을 확장한 2차원의 바코드로 종횡의 정보를 가져서 숫자 외에 문자의 데이터를 저장할 수 있다. 보통 디지털카메라나 전용 스캐너로 읽어 들여 활용한다. <a href="/wiki/%EA%B5%AD%EB%A6%BD%EA%B5%AD%EC%96%B4%EC%9B%90" title="국립국어원">국립국어원</a>에서는 QR 코드를 <b>정보무늬</b>로 다듬었다.</p>

<h2><span class="mw-headline" id=".EC.97.AD.EC.82.AC">역사</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=QR_%EC%BD%94%EB%93%9C&amp;action=edit&amp;section=1" title="부분 편집: 역사">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p><a href="/wiki/1994%EB%85%84" title="1994년">1994년</a>에 일본의 <a href="/wiki/%ED%86%A0%EC%9A%94%ED%83%80_%EC%9E%90%EB%8F%99%EC%B0%A8" title="토요타 자동차">토요타 자동차</a>의 자회사인 <a class="mw-redirect" href="/wiki/%EB%8D%B4%EC%86%8C_%EC%9B%A8%EC%9D%B4%EB%B8%8C" title="덴소 웨이브">덴소 웨이브</a>가 토요타 자동차만의 전용 차 키와 부품을 구별하고자 개발하면서 각 곳에 코드를 부착하여 사용하였다. <a href="/wiki/1997%EB%85%84" title="1997년">1997년</a> <a href="/wiki/10%EC%9B%94" title="10월">10월</a>에 AIM International 표준이 되었고 <a href="/wiki/1998%EB%85%84" title="1998년">1998년</a> <a href="/wiki/3%EC%9B%94" title="3월">3월</a>에 JEIDA 규격를 갖추었고 <a href="/wiki/2000%EB%85%84" title="2000년">2000년</a> <a href="/wiki/6%EC%9B%94" title="6월">6월</a>에 ISO/IEC 18004<sup class="reference" id="cite_ref-1"><a href="#cite_note-1">[1]</a></sup>표준이 되었다. 현재 특허권을 가진 <a class="mw-redirect" href="/wiki/%EB%8D%B4%EC%86%8C_%EC%9B%A8%EC%9D%B4%EB%B8%8C" title="덴소 웨이브">덴소 웨이브</a>는 이 표준화한 기술을 대상으로 한 특허권을 행사하지 않겠다고 선언하였다.<sup class="reference" id="cite_ref-2"><a href="#cite_note-2">[2]</a></sup></p>
<h2><span class="mw-headline" id=".ED.8A.B9.EC.A7.95">특징</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=QR_%EC%BD%94%EB%93%9C&amp;action=edit&amp;section=2" title="부분 편집: 특징">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>QR코드의 구조 일반 바코드는 단방향 즉, 1차원으로 숫자나 문자 정보가 저장 가능한데 QR코드는 종횡으로 2차원 형태를 가져서 더 많은 정보를 가질 수 있으며, 사진, 동영상, 지도, 명함 등 다양한 정보를 더 편리하게 담아낼 수 있다.</p>
<p>버전1부터 버전40까지 다양한 버전을 지원하고 버전마다 최대로 포함할 수 있는 정보와 크기가 다르다. QR코드에는 데이터의 표현과 읽기를 수월하게 하고자 콰이어트 존, 위치 검출 패턴[분리자 포함], 타이밍 패턴, 정렬 패턴, 포맷 정보, 버전 정보, 데이터 영역[에러 정정 코드 영역 포함] 등의 영역이 나뉘어 있다.</p>
<p>QR코드는 숫자 최대 7089 자, 영문자와 숫자[코드표가 따로 존재] 최대 4296 자, 8비트 바이트 최대 2953 바이트, 한자 1817 자를 담을 수 있다.</p>
<h2><span class="mw-headline" id=".EC.A3.BC.EC.9A.94_.EC.9A.A9.EB.8F.84">주요 용도</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=QR_%EC%BD%94%EB%93%9C&amp;action=edit&amp;section=3" title="부분 편집: 주요 용도">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>건물 위 광고판에 있는 QR 코드를 사용자가 휴대전화 단말기로 찍어 인터넷 검색이 가능하다. 초기에는 자동차 부품 생산관리 등 상품관리에 널리 이용되어 기존 바코드를 대체하는 개념으로 많이 보급되었다. 이후 일본에서는 카메라폰을 이용하여 QR코드를 다양한 인쇄 매체에 인쇄하여 연결된 인터넷 정보를 검색하기 쉽게 하려는 수단으로 발전하였다. 예컨대 잡지 광고에 삽입하여 카메라 폰으로 찍으면 코드를 인식하고 웹 사이트로 연결하여 더 자세한 정보를 보여준다든지 QR코드를 명함에 인쇄하여 개인 정보 입력을 쉽게 한다든지 다양한 활용 방법이 발달하였다. 현재 일본에서 판매되는 카메라폰 대부분이 이것을 지원하고 스마트폰에서도 이것을 지원하는 <a href="/wiki/%EC%9D%91%EC%9A%A9_%EC%86%8C%ED%94%84%ED%8A%B8%EC%9B%A8%EC%96%B4" title="응용 소프트웨어">애플리케이션</a>이 있다.</p>
