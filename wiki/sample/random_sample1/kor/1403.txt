<h1 class="firstHeading" id="firstHeading" lang="ko">RD-170</h1>
<div class="mw-body-content" id="bodyContent">
<div id="siteSub">위키백과, 우리 모두의 백과사전.</div>
<div id="contentSub"></div>
<div class="mw-jump" id="jump-to-nav">
					이동:					<a href="#mw-head">둘러보기</a>, 					<a href="#p-search">검색</a>
</div>
<div class="mw-content-ltr" dir="ltr" id="mw-content-text" lang="ko"><script>function mfTempOpenSection(id){var block=document.getElementById("mf-section-"+id);block.className+=" open-block";block.previousSibling.className+=" open-block";}</script><p><b>RD-170</b>은 러시아 <a href="/wiki/%EC%97%90%EB%84%A4%EB%A5%B4%EA%B3%A0%EB%A7%88%EC%89%AC" title="에네르고마쉬">에네르고마쉬</a>가 제작하는 세계 최대의 액체연료엔진이다. 4개의 노즐에 각각 200톤 추력으로 모두 800톤 추력을 낸다. 1981년 미국 <a href="/wiki/%EC%9A%B0%EC%A3%BC%EC%99%95%EB%B3%B5%EC%84%A0" title="우주왕복선">우주왕복선</a> 보다 4배 큰 1987년 <a href="/wiki/%EC%97%90%EB%84%A4%EB%A5%B4%EA%B8%B0%EC%95%84" title="에네르기아">에네르기아</a>의 부스터 엔진이다. 계열 엔진인 RD-151은 <a href="/wiki/%EB%82%98%EB%A1%9C%ED%98%B8" title="나로호">나로호</a> 1단에 사용되기도 하였다. 또다른 계열 엔진인 <a href="/wiki/RD-180" title="RD-180">RD-180</a>은 <a href="/wiki/%EB%AF%B8%EA%B5%AD" title="미국">미국</a>의 <a class="mw-redirect" href="/wiki/%EC%95%84%ED%8B%80%EB%9D%BC%EC%8A%A4_V" title="아틀라스 V">아틀라스 V</a> 1단 엔진으로 사용된다.</p>

<h2><span class="mw-headline" id=".ED.8C.8C.EC.83.9D.ED.98.95">파생형</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=RD-170&amp;action=edit&amp;section=1" title="부분 편집: 파생형">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>RD-170은 현재 생산되지 않는다. 그러나 파생형들은 생산중이다.</p>
<h3><span class="mw-headline" id="RD-171">RD-171</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=RD-170&amp;action=edit&amp;section=2" title="부분 편집: RD-171">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<div class="thumb tright">
<div class="thumbinner" style="width:222px;"><a class="image" href="/wiki/%ED%8C%8C%EC%9D%BC:RD171_ILA2006.jpg"><img alt="" class="thumbimage" data-file-height="1600" data-file-width="1020" height="345" src="//upload.wikimedia.org/wikipedia/commons/thumb/9/96/RD171_ILA2006.jpg/220px-RD171_ILA2006.jpg" srcset="//upload.wikimedia.org/wikipedia/commons/thumb/9/96/RD171_ILA2006.jpg/330px-RD171_ILA2006.jpg 1.5x, //upload.wikimedia.org/wikipedia/commons/thumb/9/96/RD171_ILA2006.jpg/440px-RD171_ILA2006.jpg 2x" width="220"/></a>
<div class="thumbcaption">
<div class="magnify"><a class="internal" href="/wiki/%ED%8C%8C%EC%9D%BC:RD171_ILA2006.jpg" title="실제 크기로"></a></div>
RD-171 model</div>
</div>
</div>
<p>RD-170 파생형인 <a class="mw-redirect" href="/wiki/RD-171" title="RD-171">RD-171</a>은 현재 <a href="/wiki/%EC%A0%9C%EB%8B%88%ED%8A%B8_(%EB%A1%9C%EC%BC%93)" title="제니트 (로켓)">제니트</a>에 사용된다.추력이 향상된 RD-172~175까지도 제안되었으나, 제작된 적은 없다.</p>
<h3><span class="mw-headline" id="RD-180">RD-180</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=RD-170&amp;action=edit&amp;section=3" title="부분 편집: RD-180">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<div class="rellink relarticle mainarticle"><img alt="" data-file-height="128" data-file-width="128" height="16" src="//upload.wikimedia.org/wikipedia/commons/thumb/e/ec/Crystal_Clear_app_xmag.svg/16px-Crystal_Clear_app_xmag.svg.png" srcset="//upload.wikimedia.org/wikipedia/commons/thumb/e/ec/Crystal_Clear_app_xmag.svg/24px-Crystal_Clear_app_xmag.svg.png 1.5x, //upload.wikimedia.org/wikipedia/commons/thumb/e/ec/Crystal_Clear_app_xmag.svg/32px-Crystal_Clear_app_xmag.svg.png 2x" width="16"/> 이 부분의 본문은 <a href="/wiki/RD-180" title="RD-180">RD-180</a>입니다.</div>
<p>원래 RD-170은 4개의 연소실이 있고 800톤의 추력을 내지만, RD-180은 2개의 연소실에 400톤의 추력을 낸다. <a href="/wiki/RD-180" title="RD-180">RD-180</a>은 미국에서 <a href="/wiki/%EB%A1%9D%ED%9E%88%EB%93%9C_%EB%A7%88%ED%8B%B4" title="록히드 마틴">록히드 마틴</a>의 <a class="mw-redirect" href="/wiki/%EC%95%84%ED%8B%80%EB%9D%BC%EC%8A%A4_V" title="아틀라스 V">아틀라스 V</a> 1단에 사용된다. 기존의 아틀라스에는 3개의 엔진이 사용되었는데, 이것을 <a href="/wiki/RD-180" title="RD-180">RD-180</a> 엔진 하나로 교체하여 화물탑재량을 증가시켰다. 러시아의 <a class="new" href="/w/index.php?title=Rus-M&amp;action=edit&amp;redlink=1" title="Rus-M (없는 문서)">Rus-M</a> 로켓,<sup class="reference" id="cite_ref-coppinger_1-0"><a href="#cite_note-coppinger-1">[1]</a></sup> 일본의 <a class="mw-redirect" href="/wiki/GX_%EB%A1%9C%EC%BC%93" title="GX 로켓">GX 로켓</a>과 한국의 <a class="mw-redirect" href="/wiki/KSLV-II" title="KSLV-II">KSLV-II</a>도 이 엔진을 채택할 계획이 있었으나 모두 취소되었다.</p>
<h3><span class="mw-headline" id="RD-191">RD-191</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=RD-170&amp;action=edit&amp;section=4" title="부분 편집: RD-191">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<div class="rellink relarticle mainarticle"><img alt="" data-file-height="128" data-file-width="128" height="16" src="//upload.wikimedia.org/wikipedia/commons/thumb/e/ec/Crystal_Clear_app_xmag.svg/16px-Crystal_Clear_app_xmag.svg.png" srcset="//upload.wikimedia.org/wikipedia/commons/thumb/e/ec/Crystal_Clear_app_xmag.svg/24px-Crystal_Clear_app_xmag.svg.png 1.5x, //upload.wikimedia.org/wikipedia/commons/thumb/e/ec/Crystal_Clear_app_xmag.svg/32px-Crystal_Clear_app_xmag.svg.png 2x" width="16"/> 이 부분의 본문은 <a href="/wiki/RD-191" title="RD-191">RD-191</a>입니다.</div>
<p><a href="/wiki/RD-191" title="RD-191">RD-191</a>은 1개의 연소실에 200톤의 추력을 낸다. RD-191은 현재 개발중인 러시아의 <a href="/wiki/%EC%95%88%EA%B0%80%EB%9D%BC_(%EB%A1%9C%EC%BC%93)" title="안가라 (로켓)">안가라</a>에 사용될 계획이다.<sup class="reference" id="cite_ref-2"><a href="#cite_note-2">[2]</a></sup> RD-191에서 파생된 RD-193은 앞으로 <a href="/wiki/%EC%86%8C%EC%9C%A0%EC%8A%A4-2" title="소유스-2">소유스-2</a>의 1단으로 사용할 예정이다.</p>
<h3><span class="mw-headline" id="RD-151">RD-151</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=RD-170&amp;action=edit&amp;section=5" title="부분 편집: RD-151">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>RD-151은 <a href="/wiki/%EB%82%98%EB%A1%9C%ED%98%B8" title="나로호">나로호</a>의 1단에 사용되 RD-191의 파생 엔진으로, RD-191과 동일한 하드웨어를 사용해 추력 성능면에서 동일하지만, 나로호의 비행 프로파일에 맞게 튜닝하였다.<sup class="reference" id="cite_ref-3"><a href="#cite_note-3">[3]</a></sup> 실제 나로호를 발사할때는 170톤급의 추력으로 작동한다.<sup class="reference" id="cite_ref-4"><a href="#cite_note-4">[4]</a></sup> 2009년 7월 30일 첫 연소시험을 거쳐 2009년 8월 25일 처음으로 <a href="/wiki/%EB%82%98%EB%A1%9C%ED%98%B8" title="나로호">나로호</a>의 발사에 사용되었다.<sup class="reference" id="cite_ref-5"><a href="#cite_note-5">[5]</a></sup> 나로호의 1단은 안가라의 범용로켓모듈(URM)을 기반으로 한다.</p>
