<h1 class="firstHeading" id="firstHeading" lang="ko">REXX</h1>
<div class="mw-body-content" id="bodyContent">
<div id="siteSub">위키백과, 우리 모두의 백과사전.</div>
<div id="contentSub"></div>
<div class="mw-jump" id="jump-to-nav">
					이동:					<a href="#mw-head">둘러보기</a>, 					<a href="#p-search">검색</a>
</div>
<div class="mw-content-ltr" dir="ltr" id="mw-content-text" lang="ko"><script>function mfTempOpenSection(id){var block=document.getElementById("mf-section-"+id);block.className+=" open-block";block.previousSibling.className+=" open-block";}</script><p><b>REXX</b> (REstructured eXtended eXecutor)는 <a href="/wiki/IBM" title="IBM">IBM</a>사가 개발한 <a class="new" href="/w/index.php?title=%ED%95%B4%EC%84%9D_%EC%96%B8%EC%96%B4&amp;action=edit&amp;redlink=1" title="해석 언어 (없는 문서)">해석</a> <a href="/wiki/%ED%94%84%EB%A1%9C%EA%B7%B8%EB%9E%98%EB%B0%8D_%EC%96%B8%EC%96%B4" title="프로그래밍 언어">프로그래밍 언어</a>이다. 배우기 쉬울뿐 아니라 읽기도 쉬운, 구조화된 고급 프로그래밍 언어이다. REXX는 수많은 컴퓨터 운영 체제를 지원하며 컴파일러는 IBM 메인프레임으로 사용할 수 있다. REXX의 경우 상용 버전과 <a href="/wiki/%EC%98%A4%ED%94%88_%EC%86%8C%EC%8A%A4" title="오픈 소스">오픈 소스</a> <a href="/wiki/%EC%9D%B8%ED%84%B0%ED%94%84%EB%A6%AC%ED%84%B0" title="인터프리터">인터프리터</a>가 둘 다 존재한다.</p>

<h2><span class="mw-headline" id=".EC.97.AD.EC.82.AC">역사</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=REXX&amp;action=edit&amp;section=1" title="부분 편집: 역사">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>Rexx는 1979년 3월 20일부터 1982년 중순까지 'own-time' 프로젝트로서, 또 원래는 <a class="new" href="/w/index.php?title=EXEC&amp;action=edit&amp;redlink=1" title="EXEC (없는 문서)">EXEC</a>와 <a class="new" href="/w/index.php?title=EXEC_2&amp;action=edit&amp;redlink=1" title="EXEC 2 (없는 문서)">EXEC 2</a> 언어를 대체할 <a class="mw-redirect" href="/wiki/%EC%8A%A4%ED%81%AC%EB%A6%BD%ED%8A%B8_%ED%94%84%EB%A1%9C%EA%B7%B8%EB%9E%98%EB%B0%8D_%EC%96%B8%EC%96%B4" title="스크립트 프로그래밍 언어">스크립트 프로그래밍 언어</a>로서, IBM의 <a class="new" href="/w/index.php?title=%EB%A7%88%EC%9D%B4%ED%81%AC_%EC%BD%9C%EB%A6%AC%EC%85%94&amp;action=edit&amp;redlink=1" title="마이크 콜리셔 (없는 문서)">마이크 콜리셔</a>에 의해 <a href="/wiki/%EC%96%B4%EC%85%88%EB%B8%94%EB%A6%AC%EC%96%B4" title="어셈블리어">어셈블리어</a>로 설계되어 처음 구현되었다.<sup class="reference" id="cite_ref-1"><a href="#cite_note-1">[1]</a></sup> 모든 시스템을 위한 <a href="/wiki/%EB%A7%A4%ED%81%AC%EB%A1%9C_(%EC%BB%B4%ED%93%A8%ED%84%B0_%EA%B3%BC%ED%95%99)" title="매크로 (컴퓨터 과학)">매크로</a>나 스크립트 언어가 되도록 설계되었다. 또, Rexx는 <a href="/wiki/PL/I" title="PL/I">PL/I</a> 프로그래밍 언어의 버전을 더 단순하고 더 쉽게 배울 수 있도록 고안되었다.</p>
<p>1981년 텍사스 주 휴스턴의 SHARE 56 콘퍼런스에서 대중에게 처음 설명되었다.<sup class="reference" id="cite_ref-2"><a href="#cite_note-2">[2]</a></sup></p>
<h2><span class="mw-headline" id=".EA.B8.B0.EB.8A.A5">기능</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=REXX&amp;action=edit&amp;section=2" title="부분 편집: 기능">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>REXX는 다음의 기능을 제공한다:</p>
<ul>
<li><a href="/wiki/%EB%AC%B8%EC%9E%90%EC%97%B4" title="문자열">문자열</a> 기반</li>
<li><a class="mw-redirect" href="/wiki/%ED%98%95_%EC%8B%9C%EC%8A%A4%ED%85%9C" title="형 시스템">동적 자료형</a> (선언이 없음)</li>
<li><a href="/wiki/%EC%98%88%EC%95%BD%EC%96%B4" title="예약어">예약어</a> 없음 (로컬 환경은 제외)</li>
<li><a class="new" href="/w/index.php?title=%EB%8B%A4%EB%B0%B0%EC%9E%A5_%EC%A0%95%EC%88%98&amp;action=edit&amp;redlink=1" title="다배장 정수 (없는 문서)">다배장 정수</a></li>
<li><a class="mw-redirect" href="/wiki/%EB%B6%80%EB%8F%99_%EC%86%8C%EC%88%98%EC%A0%90" title="부동 소수점">부동 소수점</a></li>
<li>자체 함수의 고급 선택</li>
<li>기억 자료 자동 관리</li>
<li>충돌 보호</li>
<li><a class="new" href="/w/index.php?title=%EC%97%B0%EC%83%81_%EB%B0%B0%EC%97%B4&amp;action=edit&amp;redlink=1" title="연상 배열 (없는 문서)">연상 배열</a></li>
<li>시스템 명령과 기능에 직접 접근</li>
<li>단순 오류 관리, 자체 추적 및 오류 정정</li>
<li>인간의 제한이 거의 없음</li>
<li>단순화된 입출력 기능</li>
</ul>
