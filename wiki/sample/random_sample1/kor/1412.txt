<h1 class="firstHeading" id="firstHeading" lang="ko">RIM-66 스탠더드</h1>

<p><b>RIM-66 스탠더드 MR</b>(RIM-66 Standard MR (SM-1MR/SM-2MR))은 이지스함에서 사용하는 주력 대공미사일이다. <a class="mw-redirect" href="/wiki/%EB%AF%B8%EC%82%AC%EC%9D%BC_%EB%B0%A9%EC%96%B4" title="미사일 방어">미사일 방어</a>에 사용된다. 대한민국 최초의 이지스함인 <a class="mw-redirect" href="/wiki/%EC%84%B8%EC%A2%85%EB%8C%80%EC%99%95%ED%95%A8" title="세종대왕함">세종대왕함</a>의 주력 대공미사일이다. 1950년대에 배치된 <a class="mw-redirect" href="/wiki/RIM-2_Terrier" title="RIM-2 Terrier">RIM-2 Terrier</a>와 <a href="/wiki/RIM-24_Tartar" title="RIM-24 Tartar">RIM-24 Tartar</a>를 대체하기 위해 <a href="/wiki/%EB%AF%B8%EA%B5%AD_%ED%95%B4%EA%B5%B0" title="미국 해군">미국 해군</a>용으로 개발된 함대공 미사일(SAM)이다.</p>

<h2><span class="mw-headline" id=".ED.8C.8C.EC.83.9D.ED.98.95">파생형</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=RIM-66_%EC%8A%A4%ED%83%A0%EB%8D%94%EB%93%9C&amp;action=edit&amp;section=1" title="부분 편집: 파생형">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<div class="thumb tright">
<div class="thumbinner" style="width:302px;"><a class="image" href="/wiki/%ED%8C%8C%EC%9D%BC:Thach_SM-1MR.jpg"><img alt="" class="thumbimage" data-file-height="903" data-file-width="1400" height="194" src="//upload.wikimedia.org/wikipedia/commons/thumb/0/07/Thach_SM-1MR.jpg/300px-Thach_SM-1MR.jpg" srcset="//upload.wikimedia.org/wikipedia/commons/thumb/0/07/Thach_SM-1MR.jpg/450px-Thach_SM-1MR.jpg 1.5x, //upload.wikimedia.org/wikipedia/commons/thumb/0/07/Thach_SM-1MR.jpg/600px-Thach_SM-1MR.jpg 2x" width="300"/></a>
<div class="thumbcaption">
<div class="magnify"><a class="internal" href="/wiki/%ED%8C%8C%EC%9D%BC:Thach_SM-1MR.jpg" title="실제 크기로"></a></div>
<a class="new" href="/w/index.php?title=FFG-43_USS_%ED%83%80%ED%9E%88&amp;action=edit&amp;redlink=1" title="FFG-43 USS 타히 (없는 문서)">FFG-43 USS 타히</a>의 MK-13 미사일 발사대에서 RIM-66 SM-1MR 미사일이 발사되고 있다</div>
</div>
</div>
<ul>
<li><a class="mw-redirect" href="/wiki/RIM-66_Standard_MR" title="RIM-66 Standard MR">RIM-66 Standard MR</a> (SM-1MR/SM-2MR), 중거리 공대공 미사일. <a href="/wiki/RIM-24_Tartar" title="RIM-24 Tartar">RIM-24 Tartar</a>를 대체함.</li>
<li><a class="mw-redirect" href="/wiki/RIM-67_Standard" title="RIM-67 Standard">RIM-67 Standard ER</a> (SM-1ER/SM-2ER non-VLS capable), 사거리연장형, <a class="mw-redirect" href="/wiki/RIM-2_Terrier" title="RIM-2 Terrier">RIM-2 Terrier</a>를 대체함.</li>
<li><a class="mw-redirect" href="/wiki/AGM-78_Standard_ARM" title="AGM-78 Standard ARM">AGM-78 Standard ARM</a>, 대레이더 미사일</li>
<li><a class="mw-redirect" href="/wiki/RIM-161_Standard_Missile_3" title="RIM-161 Standard Missile 3">RIM-161 Standard Missile 3</a> (SM-3), 해군형 <a class="mw-redirect" href="/wiki/%ED%83%84%EB%8F%84%ED%83%84_%EC%9A%94%EA%B2%A9_%EB%AF%B8%EC%82%AC%EC%9D%BC" title="탄도탄 요격 미사일">ABM</a></li>
<li><a class="mw-redirect" href="/wiki/RIM-174_Standard_ERAM" title="RIM-174 Standard ERAM">RIM-174 Standard ERAM</a> (SM-6)</li>
</ul>
<p>이지스 탄도미사일 방어 시스템에의 적용을 고려한 버전들이 있었다. 처음에 "Navy Area" 와 "Navy Theater-Wide"의 두 가지 버전이 개발되었다. "Navy Area" 시스템은 개발 지연과 비용 상승의 이유로 미국 국방부가 취소하였다. "Navy Theater-Wide"는 <a class="mw-redirect" href="/wiki/%EB%AF%B8%EC%82%AC%EC%9D%BC_%EB%B0%A9%EC%96%B4%EA%B5%AD" title="미사일 방어국">미사일 방어국</a>(MDA)의 시스템에 편입되어 다른 이름으로 개발이 계속되고 있다. 미국 해군은 RIM-156 스탠더드 SM-2ER 블록 IV 와 RIM-161 스탠더드 SM-3를 <a class="mw-redirect" href="/wiki/%ED%83%84%EB%8F%84%EB%AF%B8%EC%82%AC%EC%9D%BC" title="탄도미사일">탄도미사일</a> 방어에 사용한다.</p>
<p>스탠더드 미사일은 또한 대함 미사일로도 사용할 수 있다. <a class="new" href="/w/index.php?title=%EC%84%B8%EB%AF%B8_%EC%95%A1%ED%8B%B0%EB%93%9C_%EC%9C%A0%EB%8F%84_%EB%AA%A8%EB%93%9C&amp;action=edit&amp;redlink=1" title="세미 액티드 유도 모드 (없는 문서)">세미 액티드 유도 모드</a>를 사용해서 수평선(line-of-sight) 거리상의 목표물을 공격하거나, 내부 <a href="/wiki/%EA%B4%80%EC%84%B1_%ED%95%AD%EB%B2%95_%EC%9E%A5%EC%B9%98" title="관성 항법 장치">관성 항법 장치</a>(INS)와 종말 적외선 유도 기능을 이용한 수평선 넘어의 목표물을 공격할 수 있다. 한때는 스탠더드 미사일에 <a class="new" href="/w/index.php?title=W81_%ED%95%B5%ED%83%84%EB%91%90&amp;action=edit&amp;redlink=1" title="W81 핵탄두 (없는 문서)">W81 핵탄두</a>를 장착하려고도 하였으나, 후에 무기연기(shelved) 되었다. RGM-66 스탠더드 ARM은 1970년대에 사용된 함정발사용 <a class="new" href="/w/index.php?title=%EB%8C%80%EB%A0%88%EC%9D%B4%EB%8D%94_%EB%AF%B8%EC%82%AC%EC%9D%BC&amp;action=edit&amp;redlink=1" title="대레이더 미사일 (없는 문서)">대레이더 미사일</a>이다.</p>

<h3><span class="mw-headline" id=".EB.B2.84.EC.A0.84.EB.B3.84_.EC.B0.A8.EC.9D.B4">버전별 차이</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=RIM-66_%EC%8A%A4%ED%83%A0%EB%8D%94%EB%93%9C&amp;action=edit&amp;section=3" title="부분 편집: 버전별 차이">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<ul>
<li>Block II
<ul>
<li>ECM에 더 강한 신호처리기, 더 작고 단단한 목표물을 파괴할 수 있는 향상된 신관과 향상된 지향성 파편폭풍 탄두의 장착, 고속 고기동이 가능한 새로운 엔진 장착.</li>
</ul>
</li>
<li>Block III
<ul>
<li>이전 버전 보다, 더욱 저고도의 목표물에 대한 능력 향상</li>
</ul>
</li>
<li>Block IIIA
<ul>
<li>이전 버전 보다, 더욱 저고도의 목표물에 대한 능력 향상. 한 방향으로 더 빠른 파편폭풍이 일어나는 새로운 탄두 장착. 한국의 <a class="mw-redirect" href="/wiki/KD-2" title="KD-2">KDX-2</a>에 탑재.</li>
</ul>
</li>
<li>Block IIIB
<ul>
<li>최초의 <a href="/wiki/%EC%B6%94%EB%A0%A5%ED%8E%B8%ED%96%A5" title="추력편향">추력편향</a> 스탠더드 미사일. Block IIIA의 전파(RF 세미 액티브 유도 방식에, 새로운 적외선(IR) 유도 모드가 추가됨. 이러한 듀얼모드 RF/IR 유도방식으로 전자전에 더 뛰어나고, 대함 순항미사일 위협을 방어하는 데 더 뛰어남. 그리고 counter a specific fielded. 한국의 <a class="mw-redirect" href="/wiki/KD-3" title="KD-3">세종대왕함</a>(KD-3)에 탑재.</li>
</ul>
</li>

<h2><span class="mw-headline" id="RIM-67_SM-2_ER">RIM-67 SM-2 ER</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=RIM-66_%EC%8A%A4%ED%83%A0%EB%8D%94%EB%93%9C&amp;action=edit&amp;section=4" title="부분 편집: RIM-67 SM-2 ER">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<div class="rellink relarticle mainarticle"><img alt="" data-file-height="128" data-file-width="128" height="16" src="//upload.wikimedia.org/wikipedia/commons/thumb/e/ec/Crystal_Clear_app_xmag.svg/16px-Crystal_Clear_app_xmag.svg.png" srcset="//upload.wikimedia.org/wikipedia/commons/thumb/e/ec/Crystal_Clear_app_xmag.svg/24px-Crystal_Clear_app_xmag.svg.png 1.5x, //upload.wikimedia.org/wikipedia/commons/thumb/e/ec/Crystal_Clear_app_xmag.svg/32px-Crystal_Clear_app_xmag.svg.png 2x" width="16"/> 이 부분의 본문은 <a class="mw-redirect" href="/wiki/RIM-67_%EC%8A%A4%ED%83%A0%EB%8D%94%EB%93%9C_ER" title="RIM-67 스탠더드 ER">RIM-67 스탠더드 ER</a>입니다.</div>
<p>RIM-67 SM-2 ER(Etended Range:사거리 연장형)은 해군의 <a href="/wiki/RIM-2_%ED%85%8C%EB%A6%AC%EC%96%B4" title="RIM-2 테리어">RIM-2 테리어</a> 미사일을 교체한다.</p>
<p>SM-2 ER을 탑재하는 함정들은 SM-2ER을 장착했음에도 <i>테리어 함정</i>으로 불리곤 했다.</p>
<p>RIM-67의 1단 부스터가 매우 길기 때문에, <a class="new" href="/w/index.php?title=Mk_41&amp;action=edit&amp;redlink=1" title="Mk 41 (없는 문서)">Mk 41</a> <a class="mw-redirect" href="/wiki/VLS" title="VLS">VLS</a>(미사일 수직발사관)에 탑재할 수 없었다. 그래서 이지스 시스템에서 사용할 수 없었다.</p>
<p>Mk 72 부스터를 장착한 SM-2 Block IV는 CG 47급 이지스 순양함을 위한 장거리 SM의 부족을 메우기 위해 개발되었다.</p>
