<p><b>엘리자베트 아말리에 오이게니 인 바이에른 공녀</b>(<span lang="de" xml:lang="de">Elisabeth Amalie Eugenie, Herzogin in Bayern</span>, <a href="/wiki/1837%EB%85%84" title="1837년">1837년</a> <a href="/wiki/12%EC%9B%94_24%EC%9D%BC" title="12월 24일">12월 24일</a> ~ <a href="/wiki/1898%EB%85%84" title="1898년">1898년</a> <a href="/wiki/9%EC%9B%94_10%EC%9D%BC" title="9월 10일">9월 10일</a>)는 <a class="mw-redirect" href="/wiki/%EC%98%A4%EC%8A%A4%ED%8A%B8%EB%A6%AC%EC%95%84-%ED%97%9D%EA%B0%80%EB%A6%AC" title="오스트리아-헝가리">오스트리아-헝가리</a> 제국 <a href="/wiki/%ED%94%84%EB%9E%80%EC%B8%A0_%EC%9A%94%EC%A0%9C%ED%94%84_1%EC%84%B8" title="프란츠 요제프 1세">프란츠 요제프 1세</a>(Franz Joseph I)의 황후이다. 당시에 뛰어난 미인으로 유명했으며, 유럽 왕실 중에 허리가 가장 가늘었고(다이어트에 광적으로 집착하여 19인치 ~ 20인치를 계속 유지했다고 한다.), <b>시시</b>(Sisi)라는 애칭으로 불렸다.</p>

<p><a href="/wiki/%EB%B0%94%EC%9D%B4%EC%97%90%EB%A5%B8_%EA%B3%B5%EC%9E%91_%EB%A7%89%EC%8B%9C%EB%B0%80%EB%A6%AC%EC%95%88_%EC%9A%94%EC%A0%9C%ED%94%84" title="바이에른 공작 막시밀리안 요제프">바이에른 공작 막시밀리안 요제프</a>와 <a href="/wiki/%EB%B0%94%EC%9D%B4%EC%97%90%EB%A5%B8_%EA%B3%B5%EC%A3%BC_%EB%A3%A8%EB%8F%84%EB%B9%84%EC%B9%B4" title="바이에른 공주 루도비카">바이에른 공주 루도비카</a>의 차녀로 태어났으며, 여름에는 <a href="/wiki/%EC%8A%88%ED%83%80%EB%A5%B8%EB%B2%A0%EB%A5%B4%ED%81%AC" title="슈타른베르크">슈타른베르크</a>에 있는 <a class="new" href="/w/index.php?title=%ED%8F%AC%EC%84%BC%ED%98%B8%ED%8E%9C_%EC%84%B1&amp;action=edit&amp;redlink=1" title="포센호펜 성 (없는 문서)">포센호펜 성</a>에서 자랐으며, 겨울에는 <a href="/wiki/%EB%AE%8C%ED%97%A8" title="뮌헨">뮌헨</a>에서 생활하였다. 그녀는 바이에른 국왕 <a href="/wiki/%EB%A7%89%EC%8B%9C%EB%B0%80%EB%A6%AC%EC%95%88_1%EC%84%B8_%EC%9A%94%EC%A0%9C%ED%94%84" title="막시밀리안 1세 요제프">막시밀리안 1세 요제프</a>의 외손녀였지만, 어머니와 아버지 모두 자유로운 사상을 가지고 있었기 때문에 비교적 자유로운 환경에서 <a href="/wiki/%EC%88%98%EC%98%81" title="수영">수영</a>, <a href="/wiki/%EC%8A%B9%EB%A7%88" title="승마">승마</a>, <a href="/wiki/%EC%B2%B4%EC%A1%B0" title="체조">체조</a>를 즐기며 어린 시절을 보냈다. 시시는 딱히 잘하는 것이 없었지만 아버지 막스와 어머니 루도비카 모두 <a class="mw-redirect" href="/wiki/%EB%B9%84%ED%85%94%EC%8A%A4%EB%B0%94%ED%9D%90_%EC%99%95%EA%B0%80" title="비텔스바흐 왕가">비텔스바흐 왕가</a>에서 상당한 미남 미녀였기에, 그녀 또한 상당한 미인으로 성장하게 되었다.</p>
<h3><span class="mw-headline" id=".EC.95.BD.ED.98.BC">약혼</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%97%98%EB%A6%AC%EC%9E%90%EB%B2%A0%ED%8A%B8_%ED%8F%B0_%EC%99%B8%EC%8A%A4%ED%84%B0%EB%9D%BC%EC%9D%B4%ED%9E%88%EC%9A%B4%EA%B0%80%EB%A5%B8_%ED%99%A9%ED%9B%84&amp;action=edit&amp;section=3" title="부분 편집: 약혼">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>그녀는 리하르트 백작을 좋아했지만, 그의 낮은 신분때문에 어머니 루도비카 공작 부인에 의해서 강제로 헤어지게 되었고, 백작은 얼마 지나지 않아 <a href="/wiki/%EA%B2%B0%ED%95%B5" title="결핵">결핵</a>으로 사망했다. 그러던 중 이모 <a class="mw-redirect" href="/wiki/%EC%A1%B0%ED%94%BC_%EB%8C%80%EA%B3%B5%EB%B9%84" title="조피 대공비">조피 대공비</a>는 자신의 아들 <a href="/wiki/%ED%94%84%EB%9E%80%EC%B8%A0_%EC%9A%94%EC%A0%9C%ED%94%84_1%EC%84%B8" title="프란츠 요제프 1세">프란츠 요제프 1세</a>와 <a href="/wiki/%ED%94%84%EB%A1%9C%EC%9D%B4%EC%84%BC_%EC%99%95%EA%B5%AD" title="프로이센 왕국">프로이센 왕국</a>의 <a class="new" href="/w/index.php?title=%ED%94%84%EB%A1%9C%EC%9D%B4%EC%84%BC%EC%9D%98_%EB%A7%88%EB%A6%AC%EC%95%84_%EC%95%84%EB%82%98&amp;action=edit&amp;redlink=1" title="프로이센의 마리아 아나 (없는 문서)">아나 공주</a>와의 약혼이 취소되자, 시시의 언니였던 <a class="mw-redirect" href="/wiki/%EB%B0%94%EC%9D%B4%EC%97%90%EB%A5%B8%EC%9D%98_%ED%97%AC%EB%A0%88%EB%84%A4" title="바이에른의 헬레네">헬레네</a>와의 약혼을 추진했다. 그러나 <a href="/wiki/%ED%94%84%EB%9E%80%EC%B8%A0_%EC%9A%94%EC%A0%9C%ED%94%84_1%EC%84%B8" title="프란츠 요제프 1세">프란츠 요제프 1세</a>는 따라나온 시시를 보고 첫눈에 반해 결국 자신의 생일날인 <a href="/wiki/8%EC%9B%94_18%EC%9D%BC" title="8월 18일">8월 18일</a> 조피 대공비와 루도비카 공작 부인에게 시시와 결혼하겠다고 밝혔다. 대공비와 공작 부인, 시시 모두 승낙하면서 황제는 시시에게 <a href="/wiki/8%EC%9B%94_19%EC%9D%BC" title="8월 19일">8월 19일</a>에 청혼하였고, 1년 뒤 시시는 <a class="mw-redirect" href="/wiki/%EC%98%A4%EC%8A%A4%ED%8A%B8%EB%A6%AC%EC%95%84%EC%9D%98_%ED%99%A9%ED%9B%84" title="오스트리아의 황후">황후</a>가 되었다.</p>
<h3><span class="mw-headline" id=".EA.B5.AC.EC.86.8D.EC.9D.98_.EC.8B.9C.EC.9E.91">구속의 시작</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%97%98%EB%A6%AC%EC%9E%90%EB%B2%A0%ED%8A%B8_%ED%8F%B0_%EC%99%B8%EC%8A%A4%ED%84%B0%EB%9D%BC%EC%9D%B4%ED%9E%88%EC%9A%B4%EA%B0%80%EB%A5%B8_%ED%99%A9%ED%9B%84&amp;action=edit&amp;section=4" title="부분 편집: 구속의 시작">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<h4><span class="mw-headline" id=".EC.98.A4.EC.8A.A4.ED.8A.B8.EB.A6.AC.EC.95.84_.ED.99.A9.ED.9B.84_.EB.B0.8F_.ED.97.9D.EA.B0.80.EB.A6.AC_.EC.99.95.EB.B9.84">오스트리아 황후 및 헝가리 왕비</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%97%98%EB%A6%AC%EC%9E%90%EB%B2%A0%ED%8A%B8_%ED%8F%B0_%EC%99%B8%EC%8A%A4%ED%84%B0%EB%9D%BC%EC%9D%B4%ED%9E%88%EC%9A%B4%EA%B0%80%EB%A5%B8_%ED%99%A9%ED%9B%84&amp;action=edit&amp;section=5" title="부분 편집: 오스트리아 황후 및 헝가리 왕비">편집</a><span class="mw-editsection-bracket">]</span></span></h4>
<p>두 사람의 약혼이 확정되자, 조피 대공비는 오스트리아 황실 예법과 <a href="/wiki/%EC%9D%B4%ED%83%88%EB%A6%AC%EC%95%84%EC%96%B4" title="이탈리아어">이탈리아어</a>, <a href="/wiki/%ED%94%84%EB%9E%91%EC%8A%A4%EC%96%B4" title="프랑스어">프랑스어</a>, <a class="mw-redirect" href="/wiki/%EB%AC%B4%EC%9A%A9" title="무용">무용</a>, <a href="/wiki/%EC%98%A4%EC%8A%A4%ED%8A%B8%EB%A6%AC%EC%95%84%EC%9D%98_%EC%97%AD%EC%82%AC" title="오스트리아의 역사">오스트리아의 역사</a>등을 가르치기 시작했다. <a href="/wiki/1854%EB%85%84" title="1854년">1854년</a> <a href="/wiki/4%EC%9B%94_24%EC%9D%BC" title="4월 24일">4월 24일</a> <a class="new" href="/w/index.php?title=%EC%84%B1_%EC%95%84%EC%9A%B0%EA%B5%AC%EC%8A%A4%ED%8B%B4_%EC%84%B1%EB%8B%B9&amp;action=edit&amp;redlink=1" title="성 아우구스틴 성당 (없는 문서)">성 아우구스틴 성당</a>에서 시시와 프란츠 요제프 1세는 결혼했다. 그러나 보수적이었던 황실과 달리 자유적이었던 그녀는 조피 대공비와 갈등을 일으키게 되었다. 또한 오스트리아의 귀족들도 <a class="mw-redirect" href="/wiki/%EB%B9%84%ED%85%94%EC%8A%A4%EB%B0%94%ED%9D%90_%EC%99%95%EA%B0%80" title="비텔스바흐 왕가">방계 비텔스바흐 출신</a>이자, 오스트리아 황실 예법에 맞지 않는 행동들을 하였던 그녀를 뒤에서 비웃었다. 오스트리아 황실에서 쓸쓸한 생활을 보내던 시시는 <a href="/wiki/%EA%B1%B0%EC%8B%9D%EC%A6%9D" title="거식증">거식증</a>에 걸리게 되었다.</p>
<h4><span class="mw-headline" id=".EB.91.90_.EC.97.AC.EB.8C.80.EA.B3.B5.EA.B3.BC_.ED.99.A9.ED.83.9C.EC.9E.90_.EC.B6.9C.EC.82.B0">두 여대공과 황태자 출산</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%97%98%EB%A6%AC%EC%9E%90%EB%B2%A0%ED%8A%B8_%ED%8F%B0_%EC%99%B8%EC%8A%A4%ED%84%B0%EB%9D%BC%EC%9D%B4%ED%9E%88%EC%9A%B4%EA%B0%80%EB%A5%B8_%ED%99%A9%ED%9B%84&amp;action=edit&amp;section=6" title="부분 편집: 두 여대공과 황태자 출산">편집</a><span class="mw-editsection-bracket">]</span></span></h4>

<p>그러나 몇 달이 지나지 않아 시시는 <a href="/wiki/%EC%9E%84%EC%8B%A0" title="임신">임신</a>했고 첫째딸을 낳았지만 조피 대공비는 손녀에게 자신의 이름을 따서 조피라고 이름을 붙였다. 그리고 시시가 너무 어리고 교양이 부족하다는 이유로 자신이 기르겠다고 하였다. 시시는 조피 대공비의 허락 없이는 딸을 보러갈 수도 없었기에 둘의 사이는 점점 나빠져갔다. 시시는 다시 임신을 했고, 또 딸을 낳았다. 이번에도 조피 대공비는 손녀에게 <a class="new" href="/w/index.php?title=%EC%98%A4%EC%8A%A4%ED%8A%B8%EB%A6%AC%EC%95%84_%EC%97%AC%EB%8C%80%EA%B3%B5_%EA%B8%B0%EC%A0%A4%EB%9D%BC&amp;action=edit&amp;redlink=1" title="오스트리아 여대공 기젤라 (없는 문서)">기젤라 루이즈 마리아</a>라고 자기 마음대로 이름을 붙였으며, 이 딸 역시 조피가 키우게 되었다. 프란츠 요제프 1세의 부탁으로 잠시 시시가 두 아이를 키우게 되었지만, 여행중 큰딸 조피가 사망하면서, 둘째딸 기젤라의 양육은 다시 조피 대공비가 맡게 되었다. 그리고 세월이 흘러, 시시가 아들 <a class="mw-redirect" href="/wiki/%EC%98%A4%EC%8A%A4%ED%8A%B8%EB%A6%AC%EC%95%84_%ED%99%A9%ED%83%9C%EC%9E%90_%EB%A3%A8%EB%8F%8C%ED%94%84" title="오스트리아 황태자 루돌프">루돌프 황태자</a>를 출산하자 대공비는 기젤라를 돌려주고, 루돌프를 자신이 키우기 시작했다.</p>
<h4><span class="mw-headline" id=".EB.82.A8.ED.8E.B8.EC.9D.98_.EC.99.B8.EB.8F.84.EC.99.80_.EC.97.AC.ED.96.89">남편의 외도와 여행</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%97%98%EB%A6%AC%EC%9E%90%EB%B2%A0%ED%8A%B8_%ED%8F%B0_%EC%99%B8%EC%8A%A4%ED%84%B0%EB%9D%BC%EC%9D%B4%ED%9E%88%EC%9A%B4%EA%B0%80%EB%A5%B8_%ED%99%A9%ED%9B%84&amp;action=edit&amp;section=7" title="부분 편집: 남편의 외도와 여행">편집</a><span class="mw-editsection-bracket">]</span></span></h4>
<p>남편의 외도 사실을 안 시시는 <a class="new" href="/w/index.php?title=%ED%8F%AC%EC%84%BC%ED%98%B8%ED%8E%9C_%EC%84%B1&amp;action=edit&amp;redlink=1" title="포센호펜 성 (없는 문서)">포센호펜 성</a>으로 돌아가버렸다. 그러나 얼마 지나지 않아 아버지의 강요와 황제의 부탁에 그녀는 <a href="/wiki/%EB%B9%88" title="빈">빈</a>으로 다시 돌아왔다. 그러나 그녀는 빈으로 돌아온 후, 황제에게 부탁하여 <a href="/wiki/%EB%A7%88%EB%8D%B0%EC%9D%B4%EB%9D%BC_%EC%A0%9C%EB%8F%84" title="마데이라 제도">마데이라</a>, <a href="/wiki/%EB%AA%B0%ED%83%80" title="몰타">몰타</a>, <a href="/wiki/%ED%8A%B8%EB%A6%AC%EC%97%90%EC%8A%A4%ED%85%8C" title="트리에스테">트리에스테</a> 등을 돌아다니며 여행했고 잠시 빈에 돌아왔지만 다시 <a class="mw-redirect" href="/wiki/%EC%BD%94%EB%A5%B4%ED%91%B8_%EC%84%AC" title="코르푸 섬">코르푸 섬</a>에 2년 동안 머물게 되었다. <a href="/wiki/%EC%98%A4%EC%8A%A4%ED%8A%B8%EB%A6%AC%EC%95%84" title="오스트리아">오스트리아</a>의 <a href="/wiki/%EA%B5%AD%EA%B2%BD%EC%9D%BC" title="국경일">국경일</a>이나 중요한 행사, 또는 가끔씩 아이들을 보기 위하여 <a href="/wiki/%EB%B9%88" title="빈">빈</a>을 들리기도 하였다.</p>
<h3><span class="mw-headline" id=".ED.97.9D.EA.B0.80.EB.A6.AC_.EC.9E.90.EC.B9.98">헝가리 자치</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%97%98%EB%A6%AC%EC%9E%90%EB%B2%A0%ED%8A%B8_%ED%8F%B0_%EC%99%B8%EC%8A%A4%ED%84%B0%EB%9D%BC%EC%9D%B4%ED%9E%88%EC%9A%B4%EA%B0%80%EB%A5%B8_%ED%99%A9%ED%9B%84&amp;action=edit&amp;section=8" title="부분 편집: 헝가리 자치">편집</a><span class="mw-editsection-bracket">]</span></span></h3>

<p>시시는 자신을 환영해주었으며, 조피 대공비와 오스트리아 황실 사람들이 1848년 혁명으로 인해 별로 좋아하지 않았던 <a href="/wiki/%ED%97%9D%EA%B0%80%EB%A6%AC_%EC%99%95%EA%B5%AD" title="헝가리 왕국">헝가리</a>가 자신과 비슷하다 하여 헝가리에 관심을 갖게 되었다. 조피 대공비는 이를 못마땅하게 생각하여 <a href="/wiki/%EC%B2%B4%EC%BD%94%EC%96%B4" title="체코어">보헤미아어</a>를 배우도록 했지만, 시시는 오히려 <a href="/wiki/%ED%97%9D%EA%B0%80%EB%A6%AC%EC%96%B4" title="헝가리어">헝가리어</a>를 배웠고, 자신의 모든 시녀들을 <a href="/wiki/%ED%97%9D%EA%B0%80%EB%A6%AC%EC%9D%B8" title="헝가리인">헝가리인</a>으로 교체하였으며, 의상도 헝가리풍 의상을 입었다. 시시는 <a class="mw-redirect" href="/wiki/%EC%95%88%EB%93%9C%EB%9D%BC%EC%8B%9C_%EA%B7%A4%EB%9D%BC" title="안드라시 귤라">안드라시 백작</a>과 함께 헝가리 자치를 위해 노력했으며, 이를 위해 헝가리에서 연설을 하기도 했는데, <a class="mw-redirect" href="/wiki/%EB%B3%B4%EC%98%A4%EC%A0%84%EC%9F%81" title="보오전쟁">보오전쟁</a>으로 인하여 <a href="/wiki/%EB%B9%88" title="빈">빈</a>이 점령당할 위기에 놓이자, 황실 보물들과 예술품들은 <a href="/wiki/%EB%B6%80%EB%8B%A4%ED%8E%98%EC%8A%A4%ED%8A%B8" title="부다페스트">부다페스트</a>로 옮겨졌고, <a href="/wiki/%ED%99%A9%ED%9B%84" title="황후">황후</a>와 아이들 역시 <a href="/wiki/%EB%B6%80%EB%8B%A4%ED%8E%98%EC%8A%A4%ED%8A%B8" title="부다페스트">부다페스트</a>로 피신하였다. 이를 기회로 <a href="/wiki/%ED%97%9D%EA%B0%80%EB%A6%AC_%EC%99%95%EA%B5%AD" title="헝가리 왕국">헝가리</a>는 <a href="/wiki/%EB%8F%85%EB%A6%BD" title="독립">독립</a>을 요구하는데, 시시는 황제와 <a class="mw-redirect" href="/wiki/%EC%95%88%EB%93%9C%EB%9D%BC%EC%8B%9C_%EA%B7%A4%EB%9D%BC" title="안드라시 귤라">안드라시 백작</a>을 설득하여 대타협을 이루어내, <a href="/wiki/%ED%97%9D%EA%B0%80%EB%A6%AC_%EC%99%95%EA%B5%AD" title="헝가리 왕국">헝가리 왕국</a>이 <a href="/wiki/%EA%B5%AD%EB%B0%A9" title="국방">국방</a>, <a class="new" href="/w/index.php?title=%EC%9E%AC%EB%AC%B4&amp;action=edit&amp;redlink=1" title="재무 (없는 문서)">재무</a>, <a href="/wiki/%EC%99%B8%EA%B5%90" title="외교">외교</a>를 제외한 모든 분야에서 자치를 누리기로 합의하였고, <a href="/wiki/1876%EB%85%84" title="1876년">1876년</a> <a href="/wiki/6%EC%9B%94_8%EC%9D%BC" title="6월 8일">6월 8일</a> <a class="mw-redirect" href="/wiki/%ED%97%9D%EA%B0%80%EB%A6%AC%EC%9D%98_%EA%B5%B0%EC%A3%BC" title="헝가리의 군주">헝가리 사도왕</a>, <a class="new" href="/w/index.php?title=%ED%97%9D%EA%B0%80%EB%A6%AC%EC%9D%98_%EA%B5%B0%EC%A3%BC_%EB%B0%B0%EC%9A%B0%EC%9E%90&amp;action=edit&amp;redlink=1" title="헝가리의 군주 배우자 (없는 문서)">사도 왕비</a>로의 즉위식을 거행하면서, <a class="mw-redirect" href="/wiki/%EC%98%A4%EC%8A%A4%ED%8A%B8%EB%A6%AC%EC%95%84-%ED%97%9D%EA%B0%80%EB%A6%AC" title="오스트리아-헝가리">오스트리아-헝가리 이중 제국</a>이 탄생하게 되었다.</p>
<h3><span class="mw-headline" id=".EB.A9.95.EC.8B.9C.EC.BD.94_.ED.99.A9.EC.A0.9C.EC.9D.98_.EC.A3.BD.EC.9D.8C">멕시코 황제의 죽음</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%97%98%EB%A6%AC%EC%9E%90%EB%B2%A0%ED%8A%B8_%ED%8F%B0_%EC%99%B8%EC%8A%A4%ED%84%B0%EB%9D%BC%EC%9D%B4%ED%9E%88%EC%9A%B4%EA%B0%80%EB%A5%B8_%ED%99%A9%ED%9B%84&amp;action=edit&amp;section=9" title="부분 편집: 멕시코 황제의 죽음">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>그녀는 이후 <a class="mw-redirect" href="/wiki/%EC%A7%91%EC%8B%9C" title="집시">집시</a>들과 함께 <a href="/wiki/%EB%B0%94%EC%9D%B4%EC%98%AC%EB%A6%B0" title="바이올린">바이올린</a>을 배우거나 <a href="/wiki/%EB%A7%88%EC%88%A0" title="마술">마술</a>을 배웠으며, <a href="/wiki/%EA%B7%B9%EC%9E%A5" title="극장">극장</a>을 짓기도 하였다. 그녀는 <a class="mw-redirect" href="/wiki/%EC%95%88%EB%93%9C%EB%9D%BC%EC%8B%9C_%EA%B7%A4%EB%9D%BC" title="안드라시 귤라">안드라시 백작</a>과 더 친해지길 원했지만, 헝가리 수상이 된 그는 바빠졌고, 두 사람 사이는 자연스레 멀어졌다. <a href="/wiki/1867%EB%85%84" title="1867년">1867년</a> 시동생 <a class="mw-redirect" href="/wiki/%EB%A7%89%EC%8B%9C%EB%B0%80%EB%A6%AC%EC%95%88_1%EC%84%B8_(%EB%A9%95%EC%8B%9C%EC%BD%94)" title="막시밀리안 1세 (멕시코)">멕시코 황제</a>가 처형되었다는 소식이 전해졌다. 그의 아내이자 시시의 동서였던 <a href="/wiki/%EB%B2%A8%EA%B8%B0%EC%97%90%EC%9D%98_%EC%B9%B4%EB%A5%BC%EB%A1%9C%ED%83%80" title="벨기에의 카를로타">샤를로트</a>는 <a class="mw-redirect" href="/wiki/%EC%A0%95%EC%8B%A0%EB%B3%91" title="정신병">정신이상자</a>가 되어버렸다. 막시밀리안을 <a href="/wiki/%ED%99%A9%EC%A0%9C" title="황제">황제</a>로 추천한 <a href="/wiki/%EB%82%98%ED%8F%B4%EB%A0%88%EC%98%B9_3%EC%84%B8" title="나폴레옹 3세">나폴레옹 3세</a>와 <a href="/wiki/%EC%99%B8%EC%A0%9C%EB%8B%88_%EB%93%9C_%EB%AA%BD%ED%8B%B0%EC%A1%B0" title="외제니 드 몽티조">외제니 황후</a>가 막시밀리안을 기리기 위하여, 오스트리아에 방문하였다. 이후, <a href="/wiki/%EC%9E%98%EC%B8%A0%EB%B6%80%EB%A5%B4%ED%81%AC" title="잘츠부르크">잘츠부르크</a>에서 두 황후는 다시 만났고, 황실의 지나치게 엄격한 예법을 싫어하던 두 사람은 금세 친해졌다.</p>
<h3><span class="mw-headline" id=".EB.84.A4_.EB.B2.88.EC.A7.B8_.EC.95.84.EC.9D.B4">네 번째 아이</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%97%98%EB%A6%AC%EC%9E%90%EB%B2%A0%ED%8A%B8_%ED%8F%B0_%EC%99%B8%EC%8A%A4%ED%84%B0%EB%9D%BC%EC%9D%B4%ED%9E%88%EC%9A%B4%EA%B0%80%EB%A5%B8_%ED%99%A9%ED%9B%84&amp;action=edit&amp;section=10" title="부분 편집: 네 번째 아이">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<div class="thumb tleft">
<div class="thumbinner" style="width:122px;"><a class="image" href="/wiki/%ED%8C%8C%EC%9D%BC:Hungary_szeged_elisabeth_2.jpg"><img alt="" class="thumbimage" data-file-height="512" data-file-width="384" height="160" src="//upload.wikimedia.org/wikipedia/commons/thumb/5/53/Hungary_szeged_elisabeth_2.jpg/120px-Hungary_szeged_elisabeth_2.jpg" srcset="//upload.wikimedia.org/wikipedia/commons/thumb/5/53/Hungary_szeged_elisabeth_2.jpg/180px-Hungary_szeged_elisabeth_2.jpg 1.5x, //upload.wikimedia.org/wikipedia/commons/thumb/5/53/Hungary_szeged_elisabeth_2.jpg/240px-Hungary_szeged_elisabeth_2.jpg 2x" width="120"/></a>
<div class="thumbcaption">
<div class="magnify"><a class="internal" href="/wiki/%ED%8C%8C%EC%9D%BC:Hungary_szeged_elisabeth_2.jpg" title="실제 크기로"></a></div>
<a href="/wiki/%ED%97%9D%EA%B0%80%EB%A6%AC" title="헝가리">헝가리</a> <a href="/wiki/%EC%84%B8%EA%B2%8C%EB%93%9C" title="세게드">세게드</a>에 있는 시시 동상</div>
</div>
</div>
<p>시시는 헝가리의 완벽한 자치를 위해서, 첫 아들인 <a class="mw-redirect" href="/wiki/%EB%A3%A8%EB%8F%8C%ED%94%84_%ED%99%A9%ED%83%9C%EC%9E%90" title="루돌프 황태자">루돌프 황태자</a>는 <a href="/wiki/%EC%98%A4%EC%8A%A4%ED%8A%B8%EB%A6%AC%EC%95%84%EC%9D%98_%ED%99%A9%EC%A0%9C" title="오스트리아의 황제">오스트리아 황제</a>로 만들고, 둘째 아들을 낳아 그를 <a class="mw-redirect" href="/wiki/%ED%97%9D%EA%B0%80%EB%A6%AC%EC%9D%98_%EA%B5%B0%EC%A3%BC" title="헝가리의 군주">헝가리 왕</a>을 만들고 싶었다. 시시는 임신임을 알아차라지마자, <a href="/wiki/%EB%B6%80%EB%8B%A4%ED%8E%98%EC%8A%A4%ED%8A%B8" title="부다페스트">부다페스트</a>로 갔는데, 이는 헝가리의 군주가 될려면 <a href="/wiki/%ED%97%9D%EA%B0%80%EB%A6%AC_%EC%99%95%EA%B5%AD" title="헝가리 왕국">헝가리 영토</a>에서 태어나야 된다고 생각했기 때문이었다. 그러나 시시의 바램과 달리 딸 <a class="new" href="/w/index.php?title=%EC%98%A4%EC%8A%A4%ED%8A%B8%EB%A6%AC%EC%95%84_%EC%97%AC%EB%8C%80%EA%B3%B5_%EB%A7%88%EB%A6%AC_%EB%B0%9C%EB%A0%88%EB%A6%AC&amp;action=edit&amp;redlink=1" title="오스트리아 여대공 마리 발레리 (없는 문서)">마리 발레리</a>가 태어났다. 마리 발레리는 시시의 애정을 듬뿍 받게 되는데, 이로 인하여 언니 기젤라와 오빠 루돌프는 발레리를 질투하게 된다.</p>
<h3><span class="mw-headline" id=".EA.B7.B8_.EC.9D.B4.ED.9B.84">그 이후</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%97%98%EB%A6%AC%EC%9E%90%EB%B2%A0%ED%8A%B8_%ED%8F%B0_%EC%99%B8%EC%8A%A4%ED%84%B0%EB%9D%BC%EC%9D%B4%ED%9E%88%EC%9A%B4%EA%B0%80%EB%A5%B8_%ED%99%A9%ED%9B%84&amp;action=edit&amp;section=11" title="부분 편집: 그 이후">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<div class="thumb tright">
<div class="thumbinner" style="width:122px;"><a class="image" href="/wiki/%ED%8C%8C%EC%9D%BC:Kaiser_Franz_Joseph_tomb_-_Vienna.jpg"><img alt="" class="thumbimage" data-file-height="1789" data-file-width="2485" height="86" src="//upload.wikimedia.org/wikipedia/commons/thumb/a/af/Kaiser_Franz_Joseph_tomb_-_Vienna.jpg/120px-Kaiser_Franz_Joseph_tomb_-_Vienna.jpg" srcset="//upload.wikimedia.org/wikipedia/commons/thumb/a/af/Kaiser_Franz_Joseph_tomb_-_Vienna.jpg/180px-Kaiser_Franz_Joseph_tomb_-_Vienna.jpg 1.5x, //upload.wikimedia.org/wikipedia/commons/thumb/a/af/Kaiser_Franz_Joseph_tomb_-_Vienna.jpg/240px-Kaiser_Franz_Joseph_tomb_-_Vienna.jpg 2x" width="120"/></a>
<div class="thumbcaption">
<div class="magnify"><a class="internal" href="/wiki/%ED%8C%8C%EC%9D%BC:Kaiser_Franz_Joseph_tomb_-_Vienna.jpg" title="실제 크기로"></a></div>

<p>그녀는 <a class="mw-redirect" href="/wiki/%EC%95%88%EB%93%9C%EB%9D%BC%EC%8B%9C_%EA%B7%A4%EB%9D%BC" title="안드라시 귤라">안드라시 백작</a>이나 영국 귀족 베이 미들턴과 같이 여행을 떠나기도 하였다. <a class="mw-redirect" href="/wiki/%EB%A3%A8%EB%8F%8C%ED%94%84_%ED%99%A9%ED%83%9C%EC%9E%90" title="루돌프 황태자">루돌프 황태자</a>의 자살 이후 그녀는 발레리와 함께 많은 곳을 돌아다녔으며, <a href="/wiki/%EC%8A%A4%EC%9C%84%EC%8A%A4" title="스위스">스위스</a> <a href="/wiki/%EC%A0%9C%EB%84%A4%EB%B0%94" title="제네바">제네바</a>에 갔을 때, 파리 백작을 죽이려다 실패한 <a href="/wiki/%EC%95%84%EB%82%98%ED%82%A4%EC%A6%98" title="아나키즘">아나키스트</a> <a href="/wiki/%EB%A3%A8%EC%9D%B4%EC%A7%80_%EB%A3%A8%EC%BC%80%EB%8B%88" title="루이지 루케니">루이지 루케니</a>에게 찔려서 사망한다. 1955년 그녀의 일대기를 그린 영화가 <a href="/wiki/%EB%A1%9C%EB%AF%B8_%EC%8A%88%EB%82%98%EC%9D%B4%EB%8D%94" title="로미 슈나이더">로미 슈나이더</a> 주연의 3부작으로 나왔고, <a href="/wiki/1998%EB%85%84" title="1998년">1998년</a>에 <a href="/wiki/%EB%AE%A4%EC%A7%80%EC%BB%AC" title="뮤지컬">뮤지컬</a>로도 만들어져 호평을 받았다.</p>
