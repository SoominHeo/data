<h2><span class="mw-headline" id=".ED.9B.84.EB.B3.B4.EC.84.A0.EC.B6.9C">후보선출</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=1896%EB%85%84_%EB%AF%B8%EA%B5%AD_%EB%8C%80%ED%86%B5%EB%A0%B9_%EC%84%A0%EA%B1%B0&amp;action=edit&amp;section=1" title="부분 편집: 후보선출">원본 편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<h3><span class="mw-headline" id=".EA.B3.B5.ED.99.94.EB.8B.B9">공화당</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=1896%EB%85%84_%EB%AF%B8%EA%B5%AD_%EB%8C%80%ED%86%B5%EB%A0%B9_%EC%84%A0%EA%B1%B0&amp;action=edit&amp;section=2" title="부분 편집: 공화당">원본 편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>1896년 6월, 공화당 전당대회에선 윌리엄 맥킨리 전 <a class="mw-redirect" href="/wiki/%EC%98%A4%ED%95%98%EC%9D%B4%EC%98%A4" title="오하이오">오하이오</a> 주지사가 압도적인 지지로 대통령 후보로 선출된다</p>
<h3><span class="mw-headline" id=".EB.AF.BC.EC.A3.BC.EB.8B.B9">민주당</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=1896%EB%85%84_%EB%AF%B8%EA%B5%AD_%EB%8C%80%ED%86%B5%EB%A0%B9_%EC%84%A0%EA%B1%B0&amp;action=edit&amp;section=3" title="부분 편집: 민주당">원본 편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>1896년 7월, <a class="mw-disambig" href="/wiki/%EB%AF%BC%EC%A3%BC%EB%8B%B9" title="민주당">민주당</a> 전당대회에선 5차에 걸친 투표 끝에 <a href="/wiki/%EC%9C%8C%EB%A6%AC%EC%97%84_%EC%A0%9C%EB%8B%9D%EC%8A%A4_%EB%B8%8C%EB%9D%BC%EC%9D%B4%EC%96%B8" title="윌리엄 제닝스 브라이언">윌리엄 제닝스 브라이언</a> 전 하원의원이 <a class="new" href="/w/index.php?title=%EB%A6%AC%EC%B0%A8%EB%93%9C_%EB%B8%94%EB%9E%9C%EB%93%9C&amp;action=edit&amp;redlink=1" title="리차드 블랜드 (없는 문서)">리차드 블랜드</a> 전 하원의원을 꺾고 대통령 후보로 선출된다</p>
<h3><span class="mw-headline" id=".EA.B5.AD.EB.AF.BC.EB.AF.BC.EC.A3.BC.EB.8B.B9">국민민주당</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=1896%EB%85%84_%EB%AF%B8%EA%B5%AD_%EB%8C%80%ED%86%B5%EB%A0%B9_%EC%84%A0%EA%B1%B0&amp;action=edit&amp;section=4" title="부분 편집: 국민민주당">원본 편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>브라이언 의원의 대통령 후보 선출에 반발한 민주당 일부는 <a class="mw-disambig" href="/wiki/%EA%B5%AD%EB%AF%BC%EB%AF%BC%EC%A3%BC%EB%8B%B9" title="국민민주당">국민민주당</a>을 창당하고 <a class="new" href="/w/index.php?title=%EC%A1%B4_%ED%8C%94%EB%A8%B8&amp;action=edit&amp;redlink=1" title="존 팔머 (없는 문서)">존 팔머</a> 전 통상부 장관을 대통령 후보로 선출한다</p>
<h3><span class="mw-headline" id=".EC.82.AC.ED.9A.8C.EB.85.B8.EB.8F.99.EB.8B.B9">사회노동당</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=1896%EB%85%84_%EB%AF%B8%EA%B5%AD_%EB%8C%80%ED%86%B5%EB%A0%B9_%EC%84%A0%EA%B1%B0&amp;action=edit&amp;section=5" title="부분 편집: 사회노동당">원본 편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>1896년 7월 9일, <a class="mw-disambig" href="/wiki/%EC%82%AC%ED%9A%8C%EB%85%B8%EB%8F%99%EB%8B%B9" title="사회노동당">사회노동당</a>은 <a class="new" href="/w/index.php?title=%EC%B0%B0%EC%8A%A4_%EB%A7%B7%EC%B3%87&amp;action=edit&amp;redlink=1" title="찰스 맷쳇 (없는 문서)">찰스 맷쳇</a>을 대통령 후보로 선출한다</p>
<h3><span class="mw-headline" id=".EA.B8.88.EC.A7.80.EB.8B.B9">금지당</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=1896%EB%85%84_%EB%AF%B8%EA%B5%AD_%EB%8C%80%ED%86%B5%EB%A0%B9_%EC%84%A0%EA%B1%B0&amp;action=edit&amp;section=6" title="부분 편집: 금지당">원본 편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p><a class="new" href="/w/index.php?title=%EA%B8%88%EC%A7%80%EB%8B%B9&amp;action=edit&amp;redlink=1" title="금지당 (없는 문서)">금지당</a>은 내분을 통해 분열되어 <a class="new" href="/w/index.php?title=%EC%A1%B0%EC%8A%88%EC%95%84_%EB%A0%88%EB%B2%84%EB%A7%81&amp;action=edit&amp;redlink=1" title="조슈아 레버링 (없는 문서)">조슈아 레버링</a>과 <a class="new" href="/w/index.php?title=%EC%B0%B0%EC%8A%A4_%EB%B2%A4%ED%8B%80%EB%A6%AC&amp;action=edit&amp;redlink=1" title="찰스 벤틀리 (없는 문서)">찰스 벤틀리</a>를 각각 대선후보로 선출한다</p>
<h3><span class="mw-headline" id=".EA.B8.B0.ED.83.80">기타</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=1896%EB%85%84_%EB%AF%B8%EA%B5%AD_%EB%8C%80%ED%86%B5%EB%A0%B9_%EC%84%A0%EA%B1%B0&amp;action=edit&amp;section=7" title="부분 편집: 기타">원본 편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p><a href="/wiki/%EC%9D%B8%EB%AF%BC%EB%8B%B9_(%EB%AF%B8%EA%B5%AD)" title="인민당 (미국)">인민당</a>과 <a class="new" href="/w/index.php?title=%EC%9D%80%EC%83%89%EB%8B%B9&amp;action=edit&amp;redlink=1" title="은색당 (없는 문서)">은색당</a>은 민주당의 브라이언 후보를 지지하기로 결의한다. 대신 인민당은 <a class="mw-redirect mw-disambig" href="/wiki/%ED%86%A0%EB%A7%88%EC%8A%A4_%EC%99%93%EC%8A%A8" title="토마스 왓슨">토마스 왓슨</a>을 부통령 후보로 선출했다</p>
