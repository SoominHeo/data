<p><b>S7 항공</b>(<span style="font-size: smaller;"><a href="/wiki/%EB%9F%AC%EC%8B%9C%EC%95%84%EC%96%B4" title="러시아어">러시아어</a>:</span> <span lang="ru" xml:lang="ru">ПАО «Авиакомпания „Сибирь“»</span>, <span style="font-size: smaller;"><a href="/wiki/%EC%98%81%EC%96%B4" title="영어">영어</a>:</span> <span lang="en" xml:lang="en">S7 Airlines</span>)은 <a href="/wiki/%EB%85%B8%EB%B3%B4%EC%8B%9C%EB%B9%84%EB%A5%B4%EC%8A%A4%ED%81%AC_%EC%A3%BC" title="노보시비르스크 주">노보시비르스크 주</a> <a class="new" href="/w/index.php?title=%EC%98%A4%EB%B9%84_(%EB%8F%84%EC%8B%9C)&amp;action=edit&amp;redlink=1" title="오비 (도시) (없는 문서)">오비</a>에 본사를 둔<sup class="reference" id="cite_ref-1"><a href="#cite_note-1">[1]</a></sup><sup class="reference" id="cite_ref-2"><a href="#cite_note-2">[2]</a></sup> <a href="/wiki/%EB%9F%AC%EC%8B%9C%EC%95%84" title="러시아">러시아</a>의 항공사이며 <a href="/wiki/%EB%AA%A8%EC%8A%A4%ED%81%AC%EB%B0%94" title="모스크바">모스크바</a>에 사무소를 두고 있다<sup class="reference" id="cite_ref-3"><a href="#cite_note-3">[3]</a></sup>. <a href="/wiki/%EB%8F%84%EB%AA%A8%EB%8D%B0%EB%8F%84%EB%B3%B4_%EA%B5%AD%EC%A0%9C%EA%B3%B5%ED%95%AD" title="도모데도보 국제공항">도모데도보 국제공항</a>을 허브로 삼고 있다. 현재 급속도로 회사가 성장하고 있으며, <a href="/wiki/%EB%9F%AC%EC%8B%9C%EC%95%84" title="러시아">러시아</a> 국내선의 경우 <a href="/wiki/%EB%9F%AC%EC%8B%9C%EC%95%84" title="러시아">러시아</a> 제 1의 항공사이자 <a href="/wiki/%EC%8A%A4%EC%B9%B4%EC%9D%B4%ED%8C%80" title="스카이팀">스카이팀</a> 회원에 소속된 <a href="/wiki/%EC%95%84%EC%97%90%EB%A1%9C%ED%94%8C%EB%A1%9C%ED%8A%B8" title="아에로플로트">아에로플로트</a>를 제치고 <a href="/wiki/%EB%9F%AC%EC%8B%9C%EC%95%84" title="러시아">러시아</a> 국적 항공사 중에서 1위가 되었다.<sup class="reference" id="cite_ref-4"><a href="#cite_note-4">[4]</a></sup></p>

<p><a href="/wiki/1992%EB%85%84" title="1992년">1992년</a>에 설립 했으며 본사는 한때 <a href="/wiki/%EB%85%B8%EB%B3%B4%EC%8B%9C%EB%B9%84%EB%A5%B4%EC%8A%A4%ED%81%AC" title="노보시비르스크">노보시비르스크</a>에 위치했으나 현재는 <a href="/wiki/%EB%AA%A8%EC%8A%A4%ED%81%AC%EB%B0%94" title="모스크바">모스크바</a>로 이전했다. <a href="/wiki/2005%EB%85%84" title="2005년">2005년</a>부터 운항을 시작으로 항공 서비스 전반에 대해서는 새로운 상표 <b>S7 항공</b>에서 제공을 시작한다. 지금도 시베리아 항공이라고 하는 것이 많다. 이전 명칭은 시비리 항공으로 불렸다. <a href="/wiki/2009%EB%85%84" title="2009년">2009년</a> <a href="/wiki/5%EC%9B%94" title="5월">5월</a> <a href="/wiki/%EC%9D%BC%EB%B3%B8%ED%95%AD%EA%B3%B5" title="일본항공">일본항공</a>, <a href="/wiki/%EC%95%84%EB%A9%94%EB%A6%AC%EC%B9%B8_%ED%95%AD%EA%B3%B5" title="아메리칸 항공">아메리칸 항공</a>, <a href="/wiki/%EC%98%81%EA%B5%AD%ED%95%AD%EA%B3%B5" title="영국항공">영국항공</a>이 가입한 <a href="/wiki/%ED%95%AD%EA%B3%B5_%EB%8F%99%EB%A7%B9" title="항공 동맹">항공 동맹</a>체 <a href="/wiki/%EC%9B%90%EC%9B%94%EB%93%9C" title="원월드">원월드</a>에 가입하는 것을 발표해 <a href="/wiki/2010%EB%85%84" title="2010년">2010년</a> <a href="/wiki/11%EC%9B%94_15%EC%9D%BC" title="11월 15일">11월 15일</a>에 정식으로 가입했다.<sup class="reference" id="cite_ref-5"><a href="#cite_note-5">[5]</a></sup></p>
<h2><span class="mw-headline" id=".EC.9A.B4.ED.95.AD_.EB.85.B8.EC.84.A0">운항 노선</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=S7_%ED%95%AD%EA%B3%B5&amp;action=edit&amp;section=2" title="부분 편집: 운항 노선">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<div class="rellink relarticle mainarticle"><img alt="" data-file-height="128" data-file-width="128" height="16" src="//upload.wikimedia.org/wikipedia/commons/thumb/e/ec/Crystal_Clear_app_xmag.svg/16px-Crystal_Clear_app_xmag.svg.png" srcset="//upload.wikimedia.org/wikipedia/commons/thumb/e/ec/Crystal_Clear_app_xmag.svg/24px-Crystal_Clear_app_xmag.svg.png 1.5x, //upload.wikimedia.org/wikipedia/commons/thumb/e/ec/Crystal_Clear_app_xmag.svg/32px-Crystal_Clear_app_xmag.svg.png 2x" width="16"/> 이 부분의 본문은 <a href="/wiki/S7_%ED%95%AD%EA%B3%B5%EC%9D%98_%EC%9A%B4%ED%95%AD_%EB%85%B8%EC%84%A0" title="S7 항공의 운항 노선">S7 항공의 운항 노선</a>입니다.</div>
<ul>
<li><a href="/wiki/2012%EB%85%84" title="2012년">2012년</a> <a href="/wiki/4%EC%9B%94" title="4월">4월</a> 기준으로 S7 항공은 다음과 같은 노선을 운항하고 있으며 <a href="/wiki/%ED%97%88%EB%B8%8C_%EA%B3%B5%ED%95%AD" title="허브 공항">허브 공항</a>인 <a href="/wiki/%EB%85%B8%EB%B3%B4%EC%8B%9C%EB%B9%84%EB%A5%B4%EC%8A%A4%ED%81%AC" title="노보시비르스크">노보시비르스크</a>와 <a href="/wiki/%EC%9D%B4%EB%A5%B4%EC%BF%A0%EC%B8%A0%ED%81%AC" title="이르쿠츠크">이르쿠츠크</a>에서 <a href="/wiki/%EB%9F%AC%EC%8B%9C%EC%95%84" title="러시아">러시아</a> 국내선 및 <a href="/wiki/%EC%95%84%EC%8B%9C%EC%95%84" title="아시아">아시아</a>, <a href="/wiki/%EB%8F%85%EC%9D%BC" title="독일">독일</a>, <a href="/wiki/%EC%9C%A0%EB%9F%BD" title="유럽">유럽</a> 등 국제선을 운항하고 있다. <a href="/wiki/%EC%95%84%EC%97%90%EB%A1%9C%ED%94%8C%EB%A1%9C%ED%8A%B8" title="아에로플로트">아에로플로트</a>가 국내선을 정리하고 있는 동안에 국내선을 확장 계속한 결과 오늘날 <a href="/wiki/%EB%9F%AC%EC%8B%9C%EC%95%84" title="러시아">러시아</a>의 국내선 1위를 달성했다.</li>
</ul>
