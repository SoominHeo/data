<h2><span class="mw-headline" id=".EA.B0.9C.EC.9A.94">개요</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=SAKURA_(%EC%9D%B4%ED%82%A4%EB%AA%A8%EB%85%B8%EA%B0%80%EC%B9%B4%EB%A6%AC%EC%9D%98_%EB%85%B8%EB%9E%98)&amp;action=edit&amp;section=1" title="부분 편집: 개요">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>이번 발매에 맞춰서 그 지역의 <a href="/wiki/%EC%95%84%EC%93%B0%EA%B8%B0_%EC%8B%9C" title="아쓰기 시">아쓰기 시</a>나 <a href="/wiki/%EC%97%90%EB%B9%84%EB%82%98_%EC%8B%9C" title="에비나 시">에비나 시</a>의 특정 CD 샵에서는, CD 구입 특전으로 판촉용 스탬프가 부속되어서 오거나, 구입자 대상으로 “이키모노가카리의 응원증”으로 칭하는 특제 회원증 같은 것이 배포됐다.</p>
<p>데뷔작이면서 오리콘 차트에서는 31주 동안 체크인하는 롱셀러가 되며, 이키모노가카리의 싱글로는 〈<a class="new" href="/w/index.php?title=%EC%95%84%EB%A6%AC%EA%B0%80%ED%86%A0_(%EC%9D%B4%ED%82%A4%EB%AA%A8%EB%85%B8%EA%B0%80%EC%B9%B4%EB%A6%AC%EC%9D%98_%EB%85%B8%EB%9E%98)&amp;action=edit&amp;redlink=1" title="아리가토 (이키모노가카리의 노래) (없는 문서)">고마워</a>〉〈<a class="new" href="/w/index.php?title=YELL/%EC%A1%B0%EC%9D%B4%ED%92%80&amp;action=edit&amp;redlink=1" title="YELL/조이풀 (없는 문서)">YELL/조이풀</a>〉에 이어서 차트인 수가 많은 작품이다.</p>
<h2><span class="mw-headline" id=".EC.88.98.EB.A1.9D.EA.B3.A1">수록곡</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=SAKURA_(%EC%9D%B4%ED%82%A4%EB%AA%A8%EB%85%B8%EA%B0%80%EC%B9%B4%EB%A6%AC%EC%9D%98_%EB%85%B8%EB%9E%98)&amp;action=edit&amp;section=2" title="부분 편집: 수록곡">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<ol>
<li><b>SAKURA</b>（5:51）
<ul>
<li>작사·작곡: <a class="new" href="/w/index.php?title=%EB%AF%B8%EC%A6%88%EB%85%B8_%EC%9A%94%EC%8B%9C%ED%82%A4&amp;action=edit&amp;redlink=1" title="미즈노 요시키 (없는 문서)">미즈노 요시키</a>, 편곡: <a class="new" href="/w/index.php?title=%EC%8B%9C%EB%A7%88%EB%8B%A4_%EB%A7%88%EC%82%AC%EB%85%B8%EB%A6%AC&amp;action=edit&amp;redlink=1" title="시마다 마사노리 (없는 문서)">시마다 마사노리</a></li>
</ul>
<dl>
<dd><a href="/wiki/%EC%9D%BC%EB%B3%B8%EC%A0%84%EC%8B%A0%EC%A0%84%ED%99%94" title="일본전신전화">NTT</a> “<a href="/wiki/%EC%A0%84%EB%B3%B4" title="전보">DENPO115</a>” NTT동일본 지역 <a class="mw-redirect" href="/wiki/%EA%B4%91%EA%B3%A0_%EC%9D%8C%EC%95%85" title="광고 음악">CM 송</a>。</dd>
<dd>가사에 등장하는 “대교(大橋)”는 <a href="/wiki/%EC%82%AC%EA%B0%80%EB%AF%B8_%EA%B0%95" title="사가미 강">사가미 강</a>에 놓인 <a class="new" href="/w/index.php?title=%EC%82%AC%EA%B0%80%EB%AF%B8_%EB%8C%80%EA%B5%90&amp;action=edit&amp;redlink=1" title="사가미 대교 (없는 문서)">사가미 대교</a>(가나가와 현 도40호선, 43호선, 51호선)를 가리키며, 같이 가사에 등장하는 “<a class="mw-redirect" href="/wiki/%EC%98%A4%EB%8B%A4%ED%81%90_%EC%A0%84%EC%B2%A0_%EC%98%A4%EB%8B%A4%EC%99%80%EB%9D%BC_%EC%84%A0" title="오다큐 전철 오다와라 선">오다와라 선</a>(小田急線)”의 <a class="new" href="/w/index.php?title=%EC%95%84%EC%93%B0%EA%B8%B0_%EC%97%AD&amp;action=edit&amp;redlink=1" title="아쓰기 역 (없는 문서)">아쓰기 역</a>-<a class="new" href="/w/index.php?title=%ED%98%BC%EC%95%84%EC%93%B0%EA%B8%B0_%EC%97%AD&amp;action=edit&amp;redlink=1" title="혼아쓰기 역 (없는 문서)">혼아쓰기 역</a> 사이와 나란히 달리는 형으로 사가미 강을 횡단하고 있다.</dd>
<dd><a class="mw-redirect" href="/wiki/%EB%AE%A4%EC%A7%81%EB%B9%84%EB%94%94%EC%98%A4" title="뮤직비디오">PV</a>는 오다큐 오다와라 선 <a class="new" href="/w/index.php?title=%EB%8F%84%EB%AF%B8%EC%A6%88_%EC%97%AD&amp;action=edit&amp;redlink=1" title="도미즈 역 (없는 문서)">도미즈 역</a> 동쪽 입구에서 촬영되고 있으며, 탤런트인 <a class="new" href="/w/index.php?title=%EC%B9%B4%ED%86%A0_%EB%A6%AC%EC%97%90&amp;action=edit&amp;redlink=1" title="카토 리에 (없는 문서)">카토 리에</a>가 출연하고 있는 것 외에 PV에는 발매와 동시기(2006년 3월 17일)에 은퇴한 오다큐 9000형 전차의 일부도 찍혀있다<sup class="reference" id="cite_ref-1"><a href="#cite_note-1">[1]</a></sup>. 메이저 1번째 음반 《<a class="new" href="/w/index.php?title=%EC%82%AC%EC%BF%A0%EB%9D%BC%EC%82%AC%EC%BF%A0%EB%AA%A8%EB%85%B8%EA%B0%80%ED%83%80%EB%A6%AC&amp;action=edit&amp;redlink=1" title="사쿠라사쿠모노가타리 (없는 문서)">벚꽃 피는 거리 이야기</a>》 발매시에는 새로운 PV 〈<b>SAKURA (2007version)</b>〉가 제작된 것 외에 동일 음반에서는 <a href="/wiki/%EB%B3%B4%EB%84%88%EC%8A%A4_%ED%8A%B8%EB%9E%99" title="보너스 트랙">보너스 트랙</a>으로 어쿠스틱 버전이 수록되어있다.</dd>
<dd>“졸업”을 테마로 한 곡인 것과, 가사의 내용에서 벚꽃 노래나 졸업 노래로도 널리 알려져있지만, 졸업 노래에 관해서는 최근에는 후에 발매된 〈YELL〉〈고마워〉〈<a class="new" href="/w/index.php?title=%EC%95%84%EB%A3%A8%EC%9D%B4%ED%85%8C%EC%9D%B4%EC%BD%94_(%EC%9D%B4%ED%82%A4%EB%AA%A8%EB%85%B8%EA%B0%80%EC%B9%B4%EB%A6%AC%EC%9D%98_%EB%85%B8%EB%9E%98)&amp;action=edit&amp;redlink=1" title="아루이테이코 (이키모노가카리의 노래) (없는 문서)">걸어서 가자</a>〉 등도 맡아지게 되고 있다.</dd>
<dd>베스트 음반 《<a class="new" href="/w/index.php?title=%EC%9D%B4%ED%82%A4%EB%AA%A8%EB%85%B8%EB%B0%94%EC%B9%B4%EB%A6%AC&amp;action=edit&amp;redlink=1" title="이키모노바카리 (없는 문서)">이키모노바카리(멤버즈 베스트 셀렉션)</a>》의 발매일인 <a href="/wiki/2010%EB%85%84" title="2010년">2010년</a> <a href="/wiki/11%EC%9B%94_3%EC%9D%BC" title="11월 3일">11월 3일</a>에 오다큐 오다와라 선 <a href="/wiki/%EC%97%90%EB%B9%84%EB%82%98_%EC%97%AD" title="에비나 역">에비나 역</a>에서 접근 멜로디로 사용되고 있다<sup class="reference" id="cite_ref-2"><a href="#cite_note-2">[2]</a></sup><sup class="reference" id="cite_ref-3"><a href="#cite_note-3">[3]</a></sup>.</dd>
</dl>
</li>
<li><b>핫 밀크</b> (<span style="font-size: smaller;"><a href="/wiki/%EC%9D%BC%EB%B3%B8%EC%96%B4" title="일본어">일본어</a>:</span> <span lang="ja" xml:lang="ja">ホットミルク</span>) (4:57)

<dl>
<dd>커플링곡이지만 라이브에서 연주되는 횟수가 많은 인기곡이다.</dd>
<dd>이키모노가카리로는 적은 멤버끼리 같이 만든노래로, 편곡은 <a href="/wiki/%EB%8F%84%EC%BF%84%EC%A7%80%ED%97%A8" title="도쿄지헨">동경 사변</a>(당시)의 <a href="/wiki/%EB%B2%A0%EC%9D%B4%EC%8B%9C%EC%8A%A4%ED%8A%B8" title="베이시스트">베이시스트</a>인 카메다 세이지가 담당하고 있다.</dd>
<dd>덧붙여서 미즈노는 “이키모노가카리에서는 핫밀크라는 곡이 있는데, 실은 핫밀크는 질색”이라고 말하고 있다<sup class="reference" id="cite_ref-4"><a href="#cite_note-4">[4]</a></sup>.</dd>
