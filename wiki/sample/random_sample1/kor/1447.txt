<p><b>SBS MTV</b>는 미국의 음악 텔레비전 채널인 <a href="/wiki/MTV" title="MTV">MTV</a>를 현지화한 대한민국의 텔레비전 채널이다.</p>
<p>SBS MTV가 개국하기 이전에는 <a href="/wiki/1994%EB%85%84" title="1994년">1994년</a> <a href="/wiki/12%EC%9B%94" title="12월">12월</a> MTV Asia와 <a class="mw-redirect" href="/wiki/Mnet" title="Mnet">Mnet</a> 프로그램 계약으로 <a href="/wiki/1995%EB%85%84" title="1995년">1995년</a> <a href="/wiki/4%EC%9B%94_17%EC%9D%BC" title="4월 17일">4월 17일</a>부터 <a class="mw-redirect" href="/wiki/Mnet" title="Mnet">Mnet</a>에서 MTV 프로그램을 하루 3시간 방송되었다가 <a href="/wiki/1999%EB%85%84" title="1999년">1999년</a> <a href="/wiki/1%EC%9B%94_7%EC%9D%BC" title="1월 7일">1월 7일</a>부터 <a class="mw-redirect" href="/wiki/Mnet" title="Mnet">Mnet</a>에서 MTV 프로그램을 하루 3시간 방송하던 것을 5시간으로 방송 시간을 2시간 확장하였다. 이후 <a href="/wiki/2000%EB%85%84" title="2000년">2000년</a> <a href="/wiki/12%EC%9B%94_31%EC%9D%BC" title="12월 31일">12월 31일</a> Mnet 과의 계약 만료로 <a href="/wiki/2001%EB%85%84" title="2001년">2001년</a> <a href="/wiki/1%EC%9B%94_1%EC%9D%BC" title="1월 1일">1월 1일</a>부터 <a href="/wiki/%EC%98%A8%EB%AF%B8%EB%94%94%EC%96%B4" title="온미디어">온미디어</a>(현 <a href="/wiki/CJ_E%26M" title="CJ E&amp;M">CJ E&amp;M</a>)가 운영하는 게임채널인 <a class="mw-redirect" href="/wiki/%EC%98%A8%EA%B2%8C%EC%9E%84%EB%84%B7" title="온게임넷">온게임넷</a>에서 아침시간에 4시간씩 방송하다가 <a href="/wiki/2001%EB%85%84" title="2001년">같은해</a> <a href="/wiki/7%EC%9B%94_1%EC%9D%BC" title="7월 1일">7월 1일</a>부터 분리되어 정식 채널로 개국 하였다. 2001년부터 2007년 12월까지는 <a href="/wiki/%EC%98%A8%EB%AF%B8%EB%94%94%EC%96%B4" title="온미디어">온미디어</a>(현 <a href="/wiki/CJ_E%26M" title="CJ E&amp;M">CJ E&amp;M</a>)가 운영했으나 2008년 1월부터 2011년 10월 31일까지는 <a class="mw-redirect" href="/wiki/C%26M" title="C&amp;M">C&amp;M</a>과 MTV 네트웍스에서 운영하였다. 그리고 <a href="/wiki/2011%EB%85%84" title="2011년">2011년</a> <a href="/wiki/11%EC%9B%94_1%EC%9D%BC" title="11월 1일">11월 1일</a>부터 <a class="mw-redirect" href="/wiki/%EB%8B%88%EC%BC%88%EB%A1%9C%EB%94%94%EC%96%B8_%EC%BD%94%EB%A6%AC%EC%95%84" title="니켈로디언 코리아">니켈로디언 코리아</a>와 함께 <a href="/wiki/SBS_%EB%AF%B8%EB%94%94%EC%96%B4%EB%84%B7" title="SBS 미디어넷">SBS 미디어넷</a>에 속하게 되어 채널명이 SBS MTV로 변경 되었다. SBS MTV의 편성은 MTV에서 만든 외국 프로그램, SBS MTV에서 만든 프로그램, 음악 프로그램(The Show, The Stage) 등으로 편성된다.</p>
