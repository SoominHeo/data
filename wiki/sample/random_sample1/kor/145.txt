<p><b>1896년 하계 올림픽 사격</b>은 <a href="/wiki/%EA%B7%B8%EB%A6%AC%EC%8A%A4" title="그리스">그리스</a> <a href="/wiki/%EC%95%84%ED%85%8C%EB%84%A4" title="아테네">아테네</a>에서 개최된 <a href="/wiki/1896%EB%85%84_%ED%95%98%EA%B3%84_%EC%98%AC%EB%A6%BC%ED%94%BD" title="1896년 하계 올림픽">1896년 하계 올림픽</a>의 종목이다. 7개국 61명의 선수가 참가했으며, 남자 경기만 열렸다.</p>
<p>신축 경기장인 <a class="new" href="/w/index.php?title=%EC%B9%BC%EB%A6%AC%ED%85%8C%EC%95%84_%EC%82%AC%EA%B2%A9%EC%9E%A5&amp;action=edit&amp;redlink=1" title="칼리테아 사격장 (없는 문서)">칼리테아 사격장</a>에서 열린 경기는 다섯 개 세부 종목으로 나뉘어 열렸는데, 두 종목에서는 <a href="/wiki/%EC%86%8C%EC%B4%9D" title="소총">소총</a>을, 세 종목에서는 <a href="/wiki/%EA%B6%8C%EC%B4%9D" title="권총">권총</a>을 썼다. 첫 번째 경기인 <a class="mw-redirect" href="/wiki/1896%EB%85%84_%ED%95%98%EA%B3%84_%EC%98%AC%EB%A6%BC%ED%94%BD_%EC%82%AC%EA%B2%A9_%EB%82%A8%EC%9E%90_200m_%EA%B5%B0%EC%82%AC%EC%86%8C%EC%B4%9D" title="1896년 하계 올림픽 사격 남자 200m 군사소총">군사소총</a> 부문에서는 목표물을 전부 맞춘 유일한 선수인 <a href="/wiki/%ED%8C%90%ED%85%94%EB%A6%AC%EC%8A%A4_%EC%B9%B4%EB%9D%BC%EC%84%B8%EB%B8%8C%EB%8B%A4%EC%8A%A4" title="판텔리스 카라세브다스">판텔리스 카라세브다스</a>가 우승을 차지했다. 두 번째 경기인 <a class="mw-redirect" href="/wiki/1896%EB%85%84_%ED%95%98%EA%B3%84_%EC%98%AC%EB%A6%BC%ED%94%BD_%EC%82%AC%EA%B2%A9_%EB%82%A8%EC%9E%90_25m_%EA%B5%B0%EC%82%AC%EA%B6%8C%EC%B4%9D" title="1896년 하계 올림픽 사격 남자 25m 군사권총">군사권총</a> 경기에서는 두 미국 형제인 <a href="/wiki/%EC%A1%B4_%ED%8E%98%EC%9D%B8" title="존 페인">존 페인</a>과 <a href="/wiki/%EC%84%AC%EB%84%88_%ED%8E%98%EC%9D%B8" title="섬너 페인">섬너 페인</a>이 1, 2위를 차지 했는데 이것은 형제가 처음으로 1, 2위를 차지한 경기이기도 하다. 관객들의 혼란을 막기 위해 형제는 둘 중 하나만 다음 경기인 <a class="mw-redirect" href="/wiki/1896%EB%85%84_%ED%95%98%EA%B3%84_%EC%98%AC%EB%A6%BC%ED%94%BD_%EC%82%AC%EA%B2%A9_%EB%82%A8%EC%9E%90_30m_%EC%9E%90%EC%9C%A0%EA%B6%8C%EC%B4%9D" title="1896년 하계 올림픽 사격 남자 30m 자유권총">자유권총</a> 경기에 나가자고 합의를 봤으며 섬너 페인이 이 경기에 나가게 되었다. 페인은 이 종목에서 우승을 차지하며 형제가 금메달을 거뭐진 첫 경기가 되기도 했다.<sup class="reference" id="cite_ref-CP76.2C83_1-0"><a href="#cite_note-CP76.2C83-1">[1]</a></sup></p>
<p>페인 형제는 경기 심판들이 그들의 사격 기구가 요구되는 구경(口徑)에 적합하지 않으므로 <a class="mw-redirect" href="/wiki/1896%EB%85%84_%ED%95%98%EA%B3%84_%EC%98%AC%EB%A6%BC%ED%94%BD_%EC%82%AC%EA%B2%A9_%EB%82%A8%EC%9E%90_25m_%EC%86%8D%EC%82%AC%EA%B6%8C%EC%B4%9D" title="1896년 하계 올림픽 사격 남자 25m 속사권총">속사권총 경기</a>에 출전할 수 없다는 소식을 들었다. 형제의 불참 속에서 <a class="mw-redirect" href="/wiki/%EC%9D%B4%EC%98%A4%EC%95%84%EB%8B%88%EC%8A%A4_%ED%94%84%EB%9E%80%EA%B5%AC%EB%94%94%EC%8A%A4" title="이오아니스 프란구디스">이오아니스 프란구디스</a>가 우승을 차지했다. 마지막 종목인 <a class="mw-redirect" href="/wiki/1896%EB%85%84_%ED%95%98%EA%B3%84_%EC%98%AC%EB%A6%BC%ED%94%BD_%EC%82%AC%EA%B2%A9_%EB%82%A8%EC%9E%90_300m_%EC%9E%90%EC%9C%A0%EC%86%8C%EC%B4%9D_3%EC%9E%90%EC%84%B8" title="1896년 하계 올림픽 사격 남자 300m 자유소총 3자세">자유소총</a> 종목은 속사권총 종목과 같은 날에 열렸다. 하지만 경기가 끝나기 전에 어두워져버려서 당일에 경기를 끝내지 못했고 다음 날 아침 경기를 끝냈다. 이 경기에서는 <a class="mw-redirect" href="/wiki/%EA%B2%8C%EC%98%A4%EB%A5%B4%EA%B8%B0%EC%98%A4%EC%8A%A4_%EC%98%A4%EB%A5%B4%ED%8C%8C%EB%8B%88%EB%94%94%EC%8A%A4" title="게오르기오스 오르파니디스">게오르기오스 오르파니디스</a>가 우승을 차지했다.<sup class="reference" id="cite_ref-CP76.2C83_1-1"><a href="#cite_note-CP76.2C83-1">[1]</a></sup></p>
