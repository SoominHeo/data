<h1 class="firstHeading" id="firstHeading" lang="ko">SF 영화</h1>
<div class="mw-body-content" id="bodyContent">
<div id="siteSub">위키백과, 우리 모두의 백과사전.</div>

<div class="mw-jump" id="jump-to-nav">
					이동:					<a href="#mw-head">둘러보기</a>, 					<a href="#p-search">검색</a>

<div class="mw-content-ltr" dir="ltr" id="mw-content-text" lang="ko"><script>function mfTempOpenSection(id){var block=document.getElementById("mf-section-"+id);block.className+=" open-block";block.previousSibling.className+=" open-block";}</script><p><b>SF 영화</b>(Science fiction film)는 <a href="/wiki/SF_(%EC%9E%A5%EB%A5%B4)" title="SF (장르)">SF</a>를 사용하는 <a href="/wiki/%EC%98%81%ED%99%94_%EC%9E%A5%EB%A5%B4" title="영화 장르">영화 장르</a>이다. <a class="mw-redirect" href="/wiki/%EC%82%AC%EB%B3%80_%EC%86%8C%EC%84%A4" title="사변 소설">사변적</a>인, 주류 과학에서 완전히 받아들여지지 않은 현상(이를테면 <a class="new" href="/w/index.php?title=%EB%8C%80%EC%A4%91_%EB%AC%B8%ED%99%94%EC%9D%98_%EC%99%B8%EA%B3%84_%EC%83%9D%EB%AA%85%EC%B2%B4&amp;action=edit&amp;redlink=1" title="대중 문화의 외계 생명체 (없는 문서)">외계 생명체</a>, <a class="new" href="/w/index.php?title=SF_%EC%86%8D_%ED%96%89%EC%84%B1&amp;action=edit&amp;redlink=1" title="SF 속 행성 (없는 문서)">외계 세계</a>, <a class="new" href="/w/index.php?title=%EC%A7%80%EA%B0%81%EC%9D%84_%EB%84%98%EC%96%B4%EC%84%A0_%EC%9D%B8%EC%8B%9D&amp;action=edit&amp;redlink=1" title="지각을 넘어선 인식 (없는 문서)">지각을 넘어선 인식</a>, <a class="new" href="/w/index.php?title=%ED%94%BD%EC%85%98_%EC%86%8D_%EC%8B%9C%EA%B0%84%EC%97%AC%ED%96%89&amp;action=edit&amp;redlink=1" title="픽션 속 시간여행 (없는 문서)">시간여행</a>)과 함께 미래적 요소(<a class="new" href="/w/index.php?title=%EA%B0%80%EC%83%81%EC%9D%98_%EC%9A%B0%EC%A3%BC%EC%84%A0_%EB%AA%A9%EB%A1%9D&amp;action=edit&amp;redlink=1" title="가상의 우주선 목록 (없는 문서)">우주선</a>, <a href="/wiki/%EB%A1%9C%EB%B4%87" title="로봇">로봇</a>, <a href="/wiki/%EC%82%AC%EC%9D%B4%EB%B3%B4%EA%B7%B8" title="사이보그">사이보그</a>, <a class="new" href="/w/index.php?title=%EC%84%B1%EA%B0%84_%EC%97%AC%ED%96%89&amp;action=edit&amp;redlink=1" title="성간 여행 (없는 문서)">성간 여행</a>)와 다른 기술들에 대한 가상의 <a href="/wiki/%EA%B3%BC%ED%95%99" title="과학">과학</a>-기반 묘사가 그것이다. SF 영화는 종종 <a href="/wiki/%EC%A0%95%EC%B9%98" title="정치">정치</a>적이거나 <a href="/wiki/%EC%82%AC%ED%9A%8C_%EB%AC%B8%EC%A0%9C" title="사회 문제">사회 문제</a>에 초점을 맞추고, <a class="mw-redirect" href="/wiki/%EC%9D%B8%EA%B0%84_%EC%A1%B0%EA%B1%B4" title="인간 조건">인간 조건</a>과 같은 철학적 문제를 탐구한다. 많은 경우, 기존의 <a class="mw-redirect" href="/wiki/SF_%EC%86%8C%EC%84%A4" title="SF 소설">SF 소설</a>에서 유래한 <a class="mw-redirect" href="/wiki/%EC%88%98%EC%82%AC%EB%B2%95" title="수사법">수사법</a>들에 비추어 볼 때, 영화 제작자들의 무지나 무관심 때문에 SF 영화는 과학적 타당성이나 플롯의 논리 측면에서 기존 과학소설이 전통적으로 지켜온 기준에 못 미치는 경우가 많다.<sup class="reference" id="cite_ref-1"><a href="#cite_note-1">[1]</a></sup></p>
<p>이 <a href="/wiki/%EC%9E%A5%EB%A5%B4" title="장르">장르</a>는 무성 영화 초기부터 존재했고, <a href="/wiki/%EC%A1%B0%EB%A5%B4%EC%A3%BC_%EB%A9%9C%EB%A6%AC%EC%97%90%EC%8A%A4" title="조르주 멜리에스">조르주 멜리에스</a>가 <a href="/wiki/%ED%8A%B9%EC%88%98_%ED%9A%A8%EA%B3%BC" title="특수 효과">트릭 사진</a> 효과를 이용한 《<a href="/wiki/%EB%8B%AC%EC%84%B8%EA%B3%84_%EC%97%AC%ED%96%89" title="달세계 여행">달세계 여행</a>》(1902)이 그 시초로 알려져있다. 이 다음의 주요한 예시는 1927년 영화 《<a href="/wiki/%EB%A9%94%ED%8A%B8%EB%A1%9C%ED%8F%B4%EB%A6%AC%EC%8A%A4_(%EC%98%81%ED%99%94)" title="메트로폴리스 (영화)">메트로폴리스</a>》이며, 이것은 최초의 <a class="new" href="/w/index.php?title=%EC%9E%A5%ED%8E%B8&amp;action=edit&amp;redlink=1" title="장편 (없는 문서)">장편</a> SF 영화이다.<sup class="reference" id="cite_ref-2"><a href="#cite_note-2">[2]</a></sup> 1930년대에서 1950년대 사이, SF 장르는 주로 저예산 <a href="/wiki/B%EA%B8%89_%EC%98%81%ED%99%94" title="B급 영화">B급 영화</a>로 구성됐다. <a href="/wiki/%EC%8A%A4%ED%83%A0%EB%A6%AC_%ED%81%90%EB%B8%8C%EB%A6%AD" title="스탠리 큐브릭">스탠리 큐브릭</a>의 기념비적인 작품 《<a href="/wiki/2001:_%EC%8A%A4%ED%8E%98%EC%9D%B4%EC%8A%A4_%EC%98%A4%EB%94%94%EC%84%B8%EC%9D%B4" title="2001: 스페이스 오디세이">2001: 스페이스 오디세이</a>》(1968) 이후, SF 영화 장르는 더 심각하게 받아들여졌다. 1970년대 후반, 《<a class="mw-redirect" href="/wiki/%EC%8A%A4%ED%83%80_%EC%9B%8C%EC%A6%88_%EC%97%90%ED%94%BC%EC%86%8C%EB%93%9C_4:_%EC%83%88%EB%A1%9C%EC%9A%B4_%ED%9D%AC%EB%A7%9D" title="스타 워즈 에피소드 4: 새로운 희망">스타 워즈</a>》의 성공으로 특수 효과로 가득한 고예산 SF영화가 인기를 얻었고, 이후 수십 년간 지속될 블록버스터 흥행작들의 기반을 쌓았다.</p>

<h2><span class="mw-headline" id=".EC.9E.A5.EB.A5.B4.EC.9D.98_.ED.8A.B9.EC.A7.95">장르의 특징</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=SF_%EC%98%81%ED%99%94&amp;action=edit&amp;section=1" title="부분 편집: 장르의 특징">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>영화와 미디어 이론가, 문화 비평가인 <a class="new" href="/w/index.php?title=%EB%B9%84%EB%B9%84%EC%95%88_%EC%86%8C%EB%B8%8C%EC%B2%B5&amp;action=edit&amp;redlink=1" title="비비안 소브첵 (없는 문서)">비비안 소브첵</a>에 따르면,</p>
<blockquote>
<p>SF 영화는 <i>실제적, 외삽적, 혹은 사변적인 <a href="/wiki/%EA%B3%BC%ED%95%99" title="과학">과학</a>과 <a class="new" href="/w/index.php?title=%EA%B2%BD%ED%97%98%EC%A0%81_%EB%B0%A9%EB%B2%95%EB%A1%A0&amp;action=edit&amp;redlink=1" title="경험적 방법론 (없는 문서)">경험적 방법론</a>을 강조하는, 그리고 덜 강조되지만 여전히 존재하는 <a href="/wiki/%EB%A7%88%EB%B2%95" title="마법">마법</a>과 <a href="/wiki/%EC%A2%85%EA%B5%90" title="종교">종교</a>의 <a class="new" href="/w/index.php?title=%EC%B4%88%EC%9B%94%EB%A1%A0&amp;action=edit&amp;redlink=1" title="초월론 (없는 문서)">초월론</a>이라는 사회적 맥락과 상호작용 하는, 사람과 미지 사이를 조정하기 위한 시도에서 나타나는 영화장르</i>이다.(Sobchack 63).</p>
</blockquote>
<p>이 정의는 (실제-세계의) <a href="/wiki/%EA%B2%BD%ED%97%98%EB%A1%A0" title="경험론">경험론</a>과 (<a class="mw-redirect" href="/wiki/%EC%B4%88%EC%9E%90%EC%97%B0" title="초자연">초자연</a>의) <a class="new" href="/w/index.php?title=%EC%B4%88%EC%9B%94%EB%A1%A0&amp;action=edit&amp;redlink=1" title="초월론 (없는 문서)">초월론</a> 사이에 존재하는 연속체를 가정하며, SF 영화를 경험론 쪽에, <a class="mw-redirect" href="/wiki/%ED%98%B8%EB%9F%AC_%EC%98%81%ED%99%94" title="호러 영화">호러 영화</a>와 <a href="/wiki/%ED%8C%90%ED%83%80%EC%A7%80_%EC%98%81%ED%99%94" title="판타지 영화">판타지 영화</a>를 초월론 쪽에 둔다. 그러나 《<a href="/wiki/%ED%94%84%EB%9E%91%EC%BC%84%EC%8A%88%ED%83%80%EC%9D%B8_(1931%EB%85%84_%EC%98%81%ED%99%94)" title="프랑켄슈타인 (1931년 영화)">프랑켄슈타인</a>》과 《<a href="/wiki/%EC%97%90%EC%9D%B4%EB%A6%AC%EC%96%B8_(%EC%98%81%ED%99%94)" title="에이리언 (영화)">에이리언</a>》처럼 잘 알려진 수많은 SF 호러 영화들의 예를 들 수 있다.</p>
<p>SF 영화의 비주얼 스타일은 친숙한 이미지와 낯선 이미지의 충돌로 특징지어질 수 있다. 이 충돌은 《<a href="/wiki/%EC%8B%9C%EA%B3%84%ED%83%9C%EC%97%BD_%EC%98%A4%EB%A0%8C%EC%A7%80_(%EC%98%81%ED%99%94)" title="시계태엽 오렌지 (영화)">시계태엽 오렌지</a>》에서 코로바 밀크바의 반복이 낯선 장식을 익숙하게 만들 때처럼 낯선 이미지가 친숙해질 때 구현된다.<sup class="reference" id="cite_ref-3"><a href="#cite_note-3">[3]</a></sup> 《<a class="new" href="/w/index.php?title=%EB%A6%AC%ED%8F%AC_%EB%A7%A8_(%EC%98%81%ED%99%94)&amp;action=edit&amp;redlink=1" title="리포 맨 (영화) (없는 문서)">리포 맨</a>》과 《<a class="new" href="/w/index.php?title=%EB%A6%AC%ED%80%B4%EB%93%9C_%EC%8A%A4%EC%B9%B4%EC%9D%B4&amp;action=edit&amp;redlink=1" title="리퀴드 스카이 (없는 문서)">리퀴드 스카이</a>》에서처럼 친숙한 이미지가 낯설어 질 때도 마찬가지로 구현된다.<sup class="reference" id="cite_ref-4"><a href="#cite_note-4">[4]</a></sup> 예를 들어, 《<a href="/wiki/%EB%8B%A5%ED%84%B0_%EC%8A%A4%ED%8A%B8%EB%A0%88%EC%9D%B8%EC%A7%80%EB%9F%AC%EB%B8%8C" title="닥터 스트레인지러브">닥터 스트레인지러브</a>》에서, 인간에 대한 왜곡은 친숙한 이미지를 더욱 낯선 것으로 만든다.<sup class="reference" id="cite_ref-5"><a href="#cite_note-5">[5]</a></sup> 마지막으로, 낯선 이미지와 친숙한 이미지는 《<a class="new" href="/w/index.php?title=%EB%8D%B0%EB%93%A4%EB%A6%AC_%EB%A9%98%ED%8B%B0%EC%8A%A4&amp;action=edit&amp;redlink=1" title="데들리 멘티스 (없는 문서)">데들리 멘티스</a>》에서 거대한 <a class="mw-redirect" href="/wiki/%ED%99%A9%EB%9D%BC%EC%82%AC%EB%A7%88%EA%B7%80" title="황라사마귀">황라사마귀</a>가 <a href="/wiki/%EC%9B%8C%EC%8B%B1%ED%84%B4_%EA%B8%B0%EB%85%90%ED%83%91" title="워싱턴 기념탑">워싱턴 기념탑</a>을 기어오를 때처럼 병렬될 수 있다.</p>
<p>문화이론가 <a class="new" href="/w/index.php?title=%EC%8A%A4%EC%BD%A7_%EB%B6%80%EC%BA%A3%EB%A8%BC&amp;action=edit&amp;redlink=1" title="스콧 부캣먼 (없는 문서)">스콧 부캣먼</a>은 SF 영화가 과장된 스케일과 아포칼립스, 혹은 초월을 통해 현대 문화에 <a class="new" href="/w/index.php?title=%EC%88%AD%EA%B3%A0&amp;action=edit&amp;redlink=1" title="숭고 (없는 문서)">숭고</a>한 표현을 제공했다고 말했다.</p>

<p>SF 영화는 <a href="/wiki/%EB%AC%B4%EC%84%B1_%EC%98%81%ED%99%94" title="무성 영화">무성 영화</a> 시대 초기에 등장했고, 일반적으로 흑백으로 찍었거나, 때때로 색을 입힌 단편 영화였다. 대부분은 기술에 대한 주제를 다뤘고, 유머러스해지려고 했다. <a class="new" href="/w/index.php?title=%EC%98%81%ED%99%94%EC%9D%98_1902%EB%85%84&amp;action=edit&amp;redlink=1" title="영화의 1902년 (없는 문서)">1902년</a>, <a href="/wiki/%EC%A1%B0%EB%A5%B4%EC%A3%BC_%EB%A9%9C%EB%A6%AC%EC%97%90%EC%8A%A4" title="조르주 멜리에스">조르주 멜리에스</a>가 일반적으로 최초의 SF 영화로 여겨지는 《<a href="/wiki/%EB%8B%AC%EC%84%B8%EA%B3%84_%EC%97%AC%ED%96%89" title="달세계 여행">달세계 여행</a>》을 발표했고,<sup class="reference" id="cite_ref-6"><a href="#cite_note-6">[6]</a></sup> 이것은 우주선의 달 여행을 묘사하기 위해 초기의 <a href="/wiki/%ED%8A%B9%EC%88%98_%ED%9A%A8%EA%B3%BC" title="특수 효과">트릭 사진</a> 기법을 이용한 영화이다. 몇몇 초기 영화가 SF와 <a class="mw-redirect" href="/wiki/%ED%98%B8%EB%9F%AC_%EC%98%81%ED%99%94" title="호러 영화">호러</a>를 융합시켰다. 그 예로 <a class="mw-redirect" href="/wiki/%EB%A7%A4%EB%A6%AC_%EC%85%B8%EB%A6%AC" title="매리 셸리">매리 셸리</a>의 소설을 개작한 《<a href="/wiki/%ED%94%84%EB%9E%91%EC%BC%84%EC%8A%88%ED%83%80%EC%9D%B8_(1910%EB%85%84_%EC%98%81%ED%99%94)" title="프랑켄슈타인 (1910년 영화)">프랑켄슈타인</a>》(<a class="new" href="/w/index.php?title=%EC%98%81%ED%99%94%EC%9D%98_1910%EB%85%84&amp;action=edit&amp;redlink=1" title="영화의 1910년 (없는 문서)">1910년</a>)과 <a href="/wiki/%EB%A1%9C%EB%B2%84%ED%8A%B8_%EB%A3%A8%EC%9D%B4%EC%8A%A4_%EC%8A%A4%ED%8B%B0%EB%B8%90%EC%8A%A8" title="로버트 루이스 스티븐슨">로버트 루이스 스티븐슨</a>의 심리적 이야기를 원작으로 삼은 《<a class="new" href="/w/index.php?title=%EC%A7%80%ED%82%AC_%EB%B0%95%EC%82%AC%EC%99%80_%ED%95%98%EC%9D%B4%EB%93%9C_%EC%94%A8_(1920%EB%85%84_%EC%98%81%ED%99%94)&amp;action=edit&amp;redlink=1" title="지킬 박사와 하이드 씨 (1920년 영화) (없는 문서)">지킬 박사와 하이드 씨</a>》(1920)가 있다. 더 모험적인 방침하에 촬영된 《<a href="/wiki/%ED%95%B4%EC%A0%80_2%EB%A7%8C%EB%A6%AC_(1916%EB%85%84_%EC%98%81%ED%99%94)" title="해저 2만리 (1916년 영화)">해저 2만리</a>》(<a class="new" href="/w/index.php?title=%EC%98%81%ED%99%94%EC%9D%98_1916%EB%85%84&amp;action=edit&amp;redlink=1" title="영화의 1916년 (없는 문서)">1916</a>)는 놀라운 잠수함과 복수심에 불타는 선장을 다룬 <a <h3><span class="mw-headline" id="1960.EB.85.84.EB.8C.80">1960년대</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=SF_%EC%98%81%ED%99%94&amp;action=edit&amp;section=5" title="부분 편집: 1960년대">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<div class="rellink boilerplate seealso"><img alt="" data-file-height="128" data-file-width="128" height="16" src="//upload.wikimedia.org/wikipedia/commons/thumb/e/ec/Crystal_Clear_app_xmag.svg/16px-Crystal_Clear_app_xmag.svg.png" srcset="//upload.wikimedia.org/wikipedia/commons/thumb/e/ec/Crystal_Clear_app_xmag.svg/24px-Crystal_Clear_app_xmag.svg.png 1.5x, //upload.wikimedia.org/wikipedia/commons/thumb/e/ec/Crystal_Clear_app_xmag.svg/32px-Crystal_Clear_app_xmag.svg.png 2x" width="16"/> <a class="new" href="/w/index.php?title=1960%EB%85%84%EB%8C%80_SF_%EC%98%81%ED%99%94_%EB%AA%A9%EB%A1%9D&amp;action=edit&amp;redlink=1" title="1960년대 SF 영화 목록 (없는 문서)">1960년대 SF 영화 목록</a> 문서를 참고하십시오.</div>
<p>1960년대엔 상대적으로 적은 SF 영화가 만들어졌지만, 몇몇 작품은 SF 영화 자체를 변형시켰다. <a href="/wiki/%EC%8A%A4%ED%83%A0%EB%A6%AC_%ED%81%90%EB%B8%8C%EB%A6%AD" title="스탠리 큐브릭">스탠리 큐브릭</a>의 《<a href="/wiki/2001:_%EC%8A%A4%ED%8E%98%EC%9D%B4%EC%8A%A4_%EC%98%A4%EB%94%94%EC%84%B8%EC%9D%B4" title="2001: 스페이스 오디세이">2001: 스페이스 오디세이</a>》(<a class="new" href="/w/index.php?title=%EC%98%81%ED%99%94%EC%9D%98_1968%EB%85%84&amp;action=edit&amp;redlink=1" title="영화의 1968년 (없는 문서)">1968</a>)는 전례없는 시각 효과와 우주 여행의 실감나는 묘사를 통해 SF 장르에 새로운 리얼리즘을 도입했고, 서사시적 이야기와 초월적이고 철학적인 스코프를 통해 장르에 영향을 미쳤다. 사회적 논평을 제공한 《<a class="new" href="/w/index.php?title=%ED%98%B9%EC%84%B1_%ED%83%88%EC%B6%9C_(1968%EB%85%84_%EC%98%81%ED%99%94)&amp;action=edit&amp;redlink=1" title="혹성 탈출 (1968년 영화) (없는 문서)">혹성 탈출</a>》(1968)과 《<a class="new" href="/w/index.php?title=%ED%99%94%EC%94%A8_451_(1966%EB%85%84_%EC%98%81%ED%99%94)&amp;action=edit&amp;redlink=1" title="화씨 451 (1966년 영화) (없는 문서)">화씨 451</a>》, 초기 SF의 순진한 면을 탐구한 캠피(campy) 영화 《<a class="new" href="/w/index.php?title=%EB%B0%94%EB%B0%94%EB%A0%90%EB%9D%BC_(%EC%98%81%ED%99%94)&amp;action=edit&amp;redlink=1" title="바바렐라 (영화) (없는 문서)">바바렐라</a>》(1968) 등을 포함한 다른 1960년대 영화도 주목할만하다. <a class="mw-redirect" href="/wiki/%EC%9E%A5_%EB%A4%BD_%EA%B3%A0%EB%8B%A4%EB%A5%B4" title="장 뤽 고다르">장 뤽 고다르</a>의 프랑스 "뉴웨이브" 영화 《<a href="/wiki/%EC%95%8C%ED%8C%8C%EB%B9%8C_(%EC%98%81%ED%99%94)" title="알파빌 (영화)">알파빌</a>》(1965)은 모든 감정을 금지하는 인공지능이 지배하는 미래의 파리를 상정했다.</p>

<p>1990년대, <a href="/wiki/%EC%9B%94%EB%93%9C_%EC%99%80%EC%9D%B4%EB%93%9C_%EC%9B%B9" title="월드 와이드 웹">월드 와이드 웹</a>과 <a href="/wiki/%EC%82%AC%EC%9D%B4%EB%B2%84%ED%8E%91%ED%81%AC" title="사이버펑크">사이버펑크</a> 장르의 출현은 《<a href="/wiki/%ED%84%B0%EB%AF%B8%EB%84%A4%EC%9D%B4%ED%84%B0_2:_%EC%8B%AC%ED%8C%90%EC%9D%98_%EB%82%A0" title="터미네이터 2: 심판의 날">터미네이터 2: 심판의 날</a>》(<a class="new" (없는 문서)">1998년</a> 작품 《<a href="/wiki/%EC%95%84%EB%A7%88%EA%B2%9F%EB%8F%88_(1998%EB%85%84_%EC%98%81%ED%99%94)" title="아마겟돈 (1998년 영화)">아마겟돈</a>》과 《<a <p>시간이 지나며, 컴퓨터가 (《<a href="/wiki/%ED%84%B0%EB%AF%B8%EB%84%A4%EC%9D%B4%ED%84%B0_2:_%EC%8B%AC%ED%8C%90%EC%9D%98_%EB%82%A0" title="터미네이터 2: 심판의 날">터미네이터 2: 심판의 날</a>》과 《<a href="/wiki/%EC%A5%AC%EB%9D%BC%EA%B8%B0_%EA%B3%B5%EC%9B%90_(%EC%98%81%ED%99%94)" title="쥬라기 공원 (영화)">쥬라기 공원</a>》덕분에) <a href="/wiki/%ED%8A%B9%EC%88%98_%ED%9A%A8%EA%B3%BC" title="특수 효과">특수 효과</a>와 필름 생산 양 측면에서 점점 더 중요한 역할을 맡게 됐다. 소프트웨어가 세련되게 개발되며 점점 더 복잡한 효과를 생산해냈다. 이것은 또한 일본의 《<a href="/wiki/%EA%B3%B5%EA%B0%81%EA%B8%B0%EB%8F%99%EB%8C%80_(%EC%98%81%ED%99%94)" title="공각기동대 (영화)">공각기동대</a>》(1995)와 미국의 《<a href="/wiki/%EC%95%84%EC%9D%B4%EC%96%B8_%EC%9E%90%EC%9D%B4%EC%96%B8%ED%8A%B8" title="아이언 자이언트">아이언 자이언트</a>》에서 드러났듯이 영화제작자들에게 향상된 퀄리티의 애니메이션을 가능케했다.</p>
<p>2000년대 첫 10년 간, <a href="/wiki/%EC%8A%88%ED%8D%BC%ED%9E%88%EC%96%B4%EB%A1%9C_%EC%98%81%ED%99%94" title="슈퍼히어로 영화">슈퍼히어로 영화</a>와 함께 매트릭스 삼부작처럼 진부한 SF가 늘어났다. <a class="new" href="/w/index.php?title=%EC%98%81%ED%99%94%EC%9D%98_2005%EB%85%84&amp;action=edit&amp;redlink=1" title="영화의 2005년 (없는 문서)">2005년</a>, 스타 워즈 사가가 음울한 주제의 《<a class="mw-redirect" href="/wiki/%EC%8A%A4%ED%83%80_%EC%9B%8C%EC%A6%88_%EC%97%90%ED%94%BC%EC%86%8C%EB%93%9C_3:_%EC%8B%9C%EC%8A%A4%EC%9D%98_%EB%B3%B5%EC%88%98" title="스타 워즈 에피소드 3: 시스의 복수">스타 워즈 에피소드 3: 시스의 복수</a>》와 함께 완결됐다. SF 영화는 또한 《<a href="/wiki/A.I." title="A.I.">A.I.</a>》, 《<a href="/wiki/%EB%A7%88%EC%9D%B4%EB%84%88%EB%A6%AC%ED%8B%B0_%EB%A6%AC%ED%8F%AC%ED%8A%B8_(%EC%98%81%ED%99%94)" title="마이너리티 리포트 (영화)">마이너리티 리포트</a>》, 《<a href="/wiki/%EC%84%A0%EC%83%A4%EC%9D%B8_(2007%EB%85%84_%EC%98%81%ED%99%94)" title="선샤인 (2007년 영화)">선샤인</a>》, 《<a href="/wiki/%EB%94%94%EC%8A%A4%ED%8A%B8%EB%A6%AD%ED%8A%B8_9" title="디스트릭트 9">디스트릭트 9</a>》, 《<a href="/wiki/%EC%B9%A0%EB%93%9C%EB%9F%B0_%EC%98%A4%EB%B8%8C_%EB%A7%A8" title="칠드런 오브 맨">칠드런 오브 맨</a>》, 《<a class="new" href="/w/index.php?title=%EC%84%B8%EB%A0%88%EB%8B%88%ED%8B%B0_(%EC%98%81%ED%99%94)&amp;action=edit&amp;redlink=1" title="세레니티 (영화) (없는 문서)">세레니티</a>》, 《<a class="new" href="/w/index.php?title=%EC%8A%AC%EB%A6%BD_%EB%94%9C%EB%9F%AC&amp;action=edit&amp;redlink=1" title="슬립 딜러 (없는 문서)">슬립 딜러</a>》, 《<a class="new" href="/w/index.php?title=%ED%8C%AC%EB%8F%84%EB%9F%BC&amp;action=edit&amp;redlink=1" title="팬도럼 (없는 문서)">팬도럼</a>》 등에서 보여지듯 다시 정치적 논평을 위한 도구로 돌아왔다. 2000년대엔 또한 《<a href="/wiki/%ED%8A%B8%EB%9E%9C%EC%8A%A4%ED%8F%AC%EB%A8%B8_(%EC%98%81%ED%99%94)" title="트랜스포머 (영화)">트랜스포머</a>》(2007)와 《<a href="/wiki/%ED%8A%B8%EB%9E%9C%EC%8A%A4%ED%8F%AC%EB%A8%B8:_%ED%8C%A8%EC%9E%90%EC%9D%98_%EC%97%AD%EC%8A%B5" title="트랜스포머: 패자의 역습">트랜스포머: 패자의 역습</a>》(2009)가 개봉했다. 두 작품 모두 세계적 흥행을 거두었다. 2009년, <a href="/wiki/%EC%A0%9C%EC%9E%84%EC%8A%A4_%EC%B9%B4%EB%A9%94%EB%A1%A0" title="제임스 카메론">제임스 카메론</a>의 《<a href="/wiki/%EC%95%84%EB%B0%94%ED%83%80_(2009%EB%85%84_%EC%98%81%ED%99%94)" title="아바타 (2009년 영화)">아바타</a>》가 세계적 흥행을 거두었고, 사상 최고로 높은 수익을 거둔 영화로 기록됐다. 이 영화는 동시에 정치적 논평의 한 예시이다. 아바타는 언옵타니움이라는 특별한 금속을 캐내기 위해 다른 행성의 환경을 파괴하는 인간을 묘사한다. 같은 해, 《<a href="/wiki/%ED%84%B0%EB%AF%B8%EB%84%A4%EC%9D%B4%ED%84%B0:_%EB%AF%B8%EB%9E%98%EC%A0%84%EC%9F%81%EC%9D%98_%EC%8B%9C%EC%9E%91" title="터미네이터: 미래전쟁의 시작">터미네이터: 미래전쟁의 시작</a>》이 개봉해 적당한 성공만을 거두었다.</p>
<h3><span class="mw-headline" id="2010.EB.85.84.EB.8C.80">2010년대</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=SF_%EC%98%81%ED%99%94&amp;action=edit&amp;section=8" title="부분 편집: 2010년대">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<div class="rellink boilerplate seealso"><img alt="" data-file-height="128" data-file-width="128" height="16" src="//upload.wikimedia.org/wikipedia/commons/thumb/e/ec/Crystal_Clear_app_xmag.svg/16px-Crystal_Clear_app_xmag.svg.png" srcset="//upload.wikimedia.org/wikipedia/commons/thumb/e/ec/Crystal_Clear_app_xmag.svg/24px-Crystal_Clear_app_xmag.svg.png 1.5x, //upload.wikimedia.org/wikipedia/commons/thumb/e/ec/Crystal_Clear_app_xmag.svg/32px-Crystal_Clear_app_xmag.svg.png 2x" width="16"/> <a class="new" href="/w/index.php?title=2010%EB%85%84%EB%8C%80_SF_%EC%98%81%ED%99%94%EC%9D%98_%EB%AA%A9%EB%A1%9D&amp;action=edit&amp;redlink=1" title="2010년대 SF 영화의 목록 (없는 문서)">2010년대 SF 영화의 목록</a> 문서를 참고하십시오.</div>
<h2><span class="mw-headline" id=".ED.85.8C.EB.A7.88.2C_.EC.9D.B4.EB.AF.B8.EC.A7.80.2C_.EC.8B.9C.EA.B0.81.EC.A0.81_.EC.9A.94.EC.86.8C">테마, 이미지, 시각적 요소</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=SF_%EC%98%81%ED%99%94&amp;action=edit&amp;section=9" title="부분 편집: 테마, 이미지, 시각적 요소">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>과학 영화는 자주 자연에 대한 사변을 다루며, 과학과 기술이라는 주요 기반 요소를 포함한다. 그러나 <a class="mw-redirect" href="/wiki/%ED%97%90%EB%A6%AC%EC%9A%B0%EB%93%9C" title="헐리우드">헐리우드</a> SF 영화의 "과학"은 자주 유사과학으로 여겨지며, 사실과 기존의 과학 이론보다는 주로 분위기와 준과학적인 예술적 공상에 의존한다. 정의는 관찰자의 관점에 따라 다양해질 수 있다.</p>
<p>많은 SF 영화는 일부 사람들에게 판타지와 오컬트 (혹은 종교적) 영화에 더 적절한 요소로 여겨지는 신비주의, <a href="/wiki/%EC%98%A4%EC%BB%AC%ED%8A%B8" title="오컬트">오컬트</a>, 마법, <a class="mw-redirect" href="/wiki/%EC%B4%88%EC%9E%90%EC%97%B0" title="초자연">초자연</a>적 요소를 포함한다. 이것은 영화의 장르를 종교적, 준종교적 철학이 주요 동기로 부여되는 사이언스 판타지로 변환시킨다. 영화 《<a href="/wiki/%EA%B8%88%EC%A7%80%EB%90%9C_%ED%96%89%EC%84%B1" title="금지된 행성">금지된 행성</a>》은 많은 일반적 SF 요소를 사용하지만, 영화는 기술적 완벽을 향한 종의 진화(이 경우엔 "크렐"이라 불리는 사라진 외계 문명이 예시하는)가 원시적이고 위험한 충동의 상실을 보증하지 못한다는 심오한 메시지를 던진다. 영화에서 원시 마음의 프로이트적 잠재의식, 혹은 "이드" 부분은 스스로를 괴물 같이 파괴적인 힘으로 드러낸다.</p>
<p>주인공이 <a href="/wiki/%EC%8A%88%ED%8D%BC%ED%9E%88%EC%96%B4%EB%A1%9C" title="슈퍼히어로">슈퍼히어로</a>의 특별한 힘을 얻는 몇몇 영화들은 장르간의 경계를 흐릿하게 만든다. 이러한 영화들은 보통 히어로가 힘을 얻을 때 그럴듯한 이유를 가져다붙인다.</p>
<p>모든 <a class="new" href="/w/index.php?title=SF_%EC%A3%BC%EC%A0%9C%EC%9D%98_%EB%AA%A9%EB%A1%9D&amp;action=edit&amp;redlink=1" title="SF 주제의 목록 (없는 문서)">SF의 주제</a>가 영화에 균등하게 적합한 것은 아니다. SF 호러와 함께 <a href="/wiki/%EC%8A%A4%ED%8E%98%EC%9D%B4%EC%8A%A4_%EC%98%A4%ED%8E%98%EB%9D%BC" title="스페이스 오페라">스페이스 오페라</a>가 가장 흔한 테마이다. SF 영화들은 SF 소품을 삭제한다면 <a href="/wiki/%EC%84%9C%EB%B6%80%EA%B7%B9" title="서부극">서부극</a>이나 <a href="/wiki/%EC%A0%9C2%EC%B0%A8_%EC%84%B8%EA%B3%84_%EB%8C%80%EC%A0%84" title="제2차 세계 대전">제2차 세계 대전</a> 영화와 비슷하다. 흔한 모티프들은 다른 행성으로의 탐험과 여행이며, <a href="/wiki/%EC%9C%A0%ED%86%A0%ED%94%BC%EC%95%84" title="유토피아">유토피아</a>가 드문 반면에 <a href="/wiki/%EB%94%94%EC%8A%A4%ED%86%A0%ED%94%BC%EC%95%84" title="디스토피아">디스토피아</a>는 쉽게 찾아볼 수 있다.</p>
<h3><span class="mw-headline" id=".EC.9D.B4.EB.AF.B8.EC.A7.80">이미지</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=SF_%EC%98%81%ED%99%94&amp;action=edit&amp;section=10" title="부분 편집: 이미지">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>영화이론가 <a class="new" href="/w/index.php?title=%EB%B9%84%EB%B9%84%EC%95%88_%EC%86%8C%EB%B8%8C%EC%B2%B5&amp;action=edit&amp;redlink=1" title="비비안 소브첵 (없는 문서)">비비안 소브첵</a>은 SF 영화가 우리가 보는 이미지에서 우리의 믿음을 추구하는 반면, 판타지 영화는 우리의 불신을 중단시키려고 시도한다는 점이 다르다고 주장했다. SF 영화는 익숙한 것의 맥락 안에서 생소하고 낯선 것을 드러낸다. 장면의 낯선 성격과 배경의 SF적 요소에도 불구하고, 영화의 이미지는 다시 인간과 우리가 우리 주변과 맺는 관계에 연관된다. SF 영화는 인간 경험의 경계를 확장하려고 노력을 기울이지만 관객의 이해와 조건의 경계에 머물게 되고, 이에 따라 완전히 낯설거나 추상적이 되는 대신 산문적인 측면을 담게 된다.</p>
<p>서부극이나 전쟁 영화 같은 장르 영화들은 특정한 공간이나 시간대에 제한된다. 이 제한은 SF 영화엔 적용되지 않는다. 하지만 SF 하면 연상되는 몇 가지 흔한 시각적 요소들이 있다. 여기엔 우주선이나 우주 정거장, 외계나 외계 생명체, 로봇, 그리고 미래적인 도구들이 포함된다. 《<a class="new" href="/w/index.php?title=%EB%A1%9C%EC%8A%A4%ED%8A%B8_%EC%9D%B8_%EC%8A%A4%ED%8E%98%EC%9D%B4%EC%8A%A4_(%EC%98%81%ED%99%94)&amp;action=edit&amp;redlink=1" title="로스트 인 스페이스 (영화) (없는 문서)">로스트 인 스페이스</a>》, 《<a class="new" href="/w/index.php?title=%EC%84%B8%EB%A0%88%EB%8B%88%ED%8B%B0_(%EC%98%81%ED%99%94)&amp;action=edit&amp;redlink=1" title="세레니티 (영화) (없는 문서)">세레니티</a>》, 《<a href="/wiki/%EC%95%84%EB%B0%94%ED%83%80_(2009%EB%85%84_%EC%98%81%ED%99%94)" title="아바타 (2009년 영화)">아바타</a>》, 《<a class="new" href="/w/index.php?title=%ED%94%84%EB%A1%9C%EB%A9%94%ED%85%8C%EC%9A%B0%EC%8A%A4_(2012%EB%85%84_%EC%98%81%ED%99%94)&amp;action=edit&amp;redlink=1" title="프로메테우스 (2012년 영화) (없는 문서)">프로메테우스</a>》가 그 예시이다. 더 많은 세밀한 시각적 단서들이 외관, 크기, 행태의 변경을 통한 인간 형태의 변형이나 텅 빈 도시처럼 익숙한 환경을 섬뜩한 이질적 의미로 변경시키는 등의 경우로 나타날 수 있다.</p>

<h2><span class="mw-headline" id=".EC.98.81.ED.99.94_.EB.8C.80_.EB.AC.B8.ED.95.99">영화 대 문학</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=SF_%EC%98%81%ED%99%94&amp;action=edit&amp;section=20" title="부분 편집: 영화 대 문학">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>SF <a href="/wiki/%EB%AC%B8%ED%95%99" title="문학">문학</a>과 비교했을 때, SF 영화는 대개 인간의 상상력에 덜 의존하고 액션 장면과 외계 생명체와 이국적인 배경을 만들어내는 <a href="/wiki/%ED%8A%B9%EC%88%98_%ED%9A%A8%EA%B3%BC" title="특수 효과">특수 효과</a>에 더 의존한다. 1970년대부터, 영화 관객들은 SF 영화에서 높은 수준의 특수 효과를 당연한 일로서 기대했다. 어떤 경우에, SF를 주제로 삼은 영화는 이국적, 미래적 배경을 그런 배경이 없다면 SF도 아닐 이야기와 중첩시킨다. 그럼에도 불구하고 몇몇 호평받은 SF 영화들은 SF 문학의 경로를 따라 추상적 개념을 탐구하기 위해 이야기 전개를 이용한다.</p>
