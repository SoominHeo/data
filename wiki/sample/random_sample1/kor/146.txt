<p>《<b>Mr. Simple</b>》은 <a href="/wiki/%EB%8C%80%ED%95%9C%EB%AF%BC%EA%B5%AD" title="대한민국">대한민국</a>의 남성 그룹 <a href="/wiki/%EC%8A%88%ED%8D%BC%EC%A3%BC%EB%8B%88%EC%96%B4" title="슈퍼주니어">슈퍼주니어</a>의 5번째 정규 앨범이다. 슈퍼주니어는 <a href="/wiki/2011%EB%85%84" title="2011년">2011년</a> <a href="/wiki/7%EC%9B%94_20%EC%9D%BC" title="7월 20일">7월 20일</a> 멤버들의 티저 사진을 공개하며 컴백을 예고했다. 이번 앨범의 타이틀 곡〈Mr. Simple〉은 현대인들이 마음의 여유 없이 일에 대한 성취와 사랑, 꿈 등을 잃어버리는 현상을 해학적으로 표현한 곡이라고 소개했다.<sup class="reference" id="cite_ref-1"><a href="#cite_note-1">[1]</a></sup> 앨범은 <a href="/wiki/2011%EB%85%84" title="2011년">2011년</a> <a href="/wiki/8%EC%9B%94_2%EC%9D%BC" title="8월 2일">8월 2일</a> 온라인으로 먼저 공개되었으며, 다음 날 오프라인에도 <a class="mw-redirect" href="/wiki/SM%EC%97%94%ED%84%B0%ED%85%8C%EC%9D%B8%EB%A8%BC%ED%8A%B8" title="SM엔터테인먼트">SM엔터테인먼트</a>를 통해 발매되었다. <a href="/wiki/%EB%AF%B8%EC%9D%B8%EC%95%84_(BONAMANA)" title="미인아 (BONAMANA)">4집</a>에 이은 1년 2개월만의 컴백을 응원하기 위해 전세계 59개국의 팬이 '왕의 귀환'이라는 문구를 담은 영상을 제작했고,<sup class="reference" id="cite_ref-2"><a href="#cite_note-2">[2]</a></sup><sup class="reference" id="cite_ref-3"><a href="#cite_note-3">[3]</a></sup> 각종 매체에서도 동일한 문구로 슈퍼주니어의 5집 컴백을 수식했다.<sup class="reference" id="cite_ref-4"><a href="#cite_note-4">[4]</a></sup><sup class="reference" id="cite_ref-5"><a href="#cite_note-5">[5]</a></sup></p>

<p>슈퍼주니어는 다섯 번째 정규 앨범 발매를 앞두고 <a href="/wiki/2011%EB%85%84" title="2011년">2011년</a> <a href="/wiki/7%EC%9B%94_20%EC%9D%BC" title="7월 20일">7월 20일</a>부터 <a href="/wiki/%EC%9D%80%ED%98%81" title="은혁">은혁</a>의 티저 사진 공개를 시작으로 같은 달 27일 <a href="/wiki/%EA%B9%80%ED%9D%AC%EC%B2%A0_(%EA%B0%80%EC%88%98)" title="김희철 (가수)">김희철</a>의 티저 사진까지 모두 공개했다.<sup class="reference" id="cite_ref-6"><a href="#cite_note-6">[6]</a></sup><a href="/wiki/7%EC%9B%94_30%EC%9D%BC" title="7월 30일">30일</a>에는 멤버들의 추가 티저 사진을 공개했다.<sup class="reference" id="cite_ref-7"><a href="#cite_note-7">[7]</a></sup><a href="/wiki/2011%EB%85%84" title="2011년">2011년</a> <a href="/wiki/8%EC%9B%94_1%EC%9D%BC" title="8월 1일">8월 1일</a>에는 <a href="/wiki/%ED%8E%98%EC%9D%B4%EC%8A%A4%EB%B6%81" title="페이스북">페이스북</a>과 <a href="/wiki/%EC%9C%A0%ED%8A%9C%EB%B8%8C" title="유튜브">유튜브</a>를 통해 티저 영상을 공개했는데, 앞서 공개한 티저 사진 컨셉과 달리 남성다운 위버섹슈얼 컨셉으로 변신했다고 소속사는 전했다.<sup class="reference" id="cite_ref-8"><a href="#cite_note-8">[8]</a></sup> 한편 <a class="mw-redirect" href="/wiki/SM%EC%97%94%ED%84%B0%ED%85%8C%EC%9D%B8%EB%A8%BC%ED%8A%B8" title="SM엔터테인먼트">SM엔터테인먼트</a>는 28일 슈퍼주니어의 컴백 기자회견이 8월 4일 열린다고 발표하였고, <a href="/wiki/2011%EB%85%84" title="2011년">2011년</a> <a href="/wiki/8%EC%9B%94_4%EC%9D%BC" title="8월 4일">8월 4일</a> 서울 강남 논현동 임피리얼 팰리스 호텔에서 5집 발매 기자회견을 가졌다. 기자회견은 SM타운과 슈퍼주니어의 <a href="/wiki/%ED%8E%98%EC%9D%B4%EC%8A%A4%EB%B6%81" title="페이스북">페이스북</a> 및 SM 공식 <a href="/wiki/%EC%9C%A0%ED%8A%9C%EB%B8%8C" title="유튜브">유튜브</a> 등을 통해 공개되기도 했다.<sup class="reference" id="cite_ref-9"><a href="#cite_note-9">[9]</a></sup><a href="/wiki/2011%EB%85%84" title="2011년">2011년</a> <a href="/wiki/8%EC%9B%94_2%EC%9D%BC" title="8월 2일">8월 2일</a> 오프라인 발매에 앞서 온라인을 통해 먼저 공개되었고, 다음 날인 3일 오프라인에서도 발매되었다.<sup class="reference" id="cite_ref-10"><a href="#cite_note-10">[10]</a></sup> 뒤이어 음원 공개에 앞서 공중파 음악프로에서 먼저 선보여 강한 인상을 남겼던 서브 타이틀곡 <b>SUPERMAN</b>은 8월 23일 발매된 B버전에 수록되었다.<sup class="reference" id="cite_ref-11"><a href="#cite_note-11">[11]</a></sup></p>

<h3><span class="mw-headline" id=".EC.84.A4.EB.AA.85">설명</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Mr._Simple&amp;action=edit&amp;section=3" title="부분 편집: 설명">원본 편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<ul>
<li>트랙 9번 <b>Walkin'</b>을 접한 <a href="/wiki/%EB%A1%A4%EB%A7%81_%EC%8A%A4%ED%86%A4%EC%8A%A4" title="롤링 스톤스">롤링 스톤스</a>의 보컬 <a href="/wiki/%EB%AF%B9_%EC%9E%AC%EA%B1%B0" title="믹 재거">믹 재거</a>는 "아일랜드에서 휴가를 보내고 있는 것처럼 느낌이 좋은 곡"이라고 칭찬했다.<sup class="reference" id="cite_ref-12"><a href="#cite_note-12">[12]</a></sup></li>
<li>트랙 11번 <b>어느새 우린 (Good Friends)</b>은 <a href="/wiki/%EC%9C%A4%EC%A2%85%EC%8B%A0" title="윤종신">윤종신</a> 작사작곡으로, 멤버들의 만남과 우정을 그린 솔직담백한 가사와 어쿠스틱한 연주가 특징이다. 윤종신은 <a href="/wiki/%ED%8A%B8%EC%9C%84%ED%84%B0" title="트위터">트위터</a>에 자신이 곡을 준 유일한 아이돌 그룹이라며 애정을 표시했다.<sup class="reference" id="cite_ref-13"><a href="#cite_note-13">[13]</a></sup></li>
</ul>
<h2><span class="mw-headline" id=".EC.B0.A8.ED.8A.B8_.EA.B8.B0.EB.A1.9D">차트 기록</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Mr._Simple&amp;action=edit&amp;section=4" title="부분 편집: 차트 기록">원본 편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<h3><span class="mw-headline" id=".EA.B0.9C.EC.9A.94">개요</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Mr._Simple&amp;action=edit&amp;section=5" title="부분 편집: 개요">원본 편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>2011년 8월 3일 <i>Mr. Simple</i>은 발매되자마자 하루 동안 14,000만 장의 판매고를 기록해 한터 일간차트와 실시간차트 1위에 올랐다.<sup class="reference" id="cite_ref-14"><a href="#cite_note-14">[14]</a></sup> 서울의 한 음반 판매점에서는 몰려드는 주문을 계산하기 위해 슈퍼주니어 전용 계산대를 따로 설치하기도 했다.<sup class="reference" id="cite_ref-15"><a href="#cite_note-15">[15]</a></sup> 이후 2012년 4월을 기준으로 총판 502,830만 장을 기록해 2008년 <a href="/wiki/%EB%8F%99%EB%B0%A9%EC%8B%A0%EA%B8%B0" title="동방신기">동방신기</a>의 <i><a href="/wiki/Mirotic" title="Mirotic">Mirotic</a></i>에 이어 4년만에 하프 밀리언을 달성했다.<sup class="reference" id="cite_ref-16"><a href="#cite_note-16">[16]</a></sup></p>
<p>한편 Mr.Simple은 <a href="/wiki/2011%EB%85%84" title="2011년">2011년</a> 8월 셋째주부터 <a href="/wiki/2012%EB%85%84" title="2012년">2012년</a> 4월 넷째주까지 <a href="/wiki/%EC%A4%91%ED%99%94%EB%AF%BC%EA%B5%AD" title="중화민국">중화민국</a> 최대 온라인 음악사이트 KKBOX의 한국음악 TOP100 차트에 37주 연속 1위에 랭크되었다.<sup class="reference" id="cite_ref-17"><a href="#cite_note-17">[17]</a></sup></p>

<h2><span class="mw-headline" id=".EC.97.90.ED.94.BC.EC.86.8C.EB.93.9C">에피소드</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Mr._Simple&amp;action=edit&amp;section=7" title="부분 편집: 에피소드">원본 편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<ul>
<li><a href="/wiki/%EC%8B%A0%EB%8F%99_(%EA%B0%80%EC%88%98)" title="신동 (가수)">신동</a>은 트위터에 5집 앨범에 관한 힌트로 '한국 98mm', '110 토론토' 등을 제시했다. 이와 관련해 팬들은 국산 담배 중 <b>심플</b>의 규격이 98mm이며 <a href="/wiki/%ED%86%A0%EB%A1%A0%ED%86%A0" title="토론토">토론토</a>는 동계 올림픽이 개최된 곳으로 IOC 회원이 110명인데 동계올림픽 폐막식에서 노래를 한 가수가 <b>Simple Plan</b>이라는 것 등을 들어 곡 제목에 Simple이 들어간다고 추정했다.<sup class="reference" id="cite_ref-21"><a href="#cite_note-21">[21]</a></sup></li>
<li>멤버들의 티저 사진 공개 순서는 <a href="/wiki/%EC%9D%80%ED%98%81" title="은혁">은혁</a>(20일) - <a href="/wiki/%EB%8F%99%ED%95%B4_(%EA%B0%80%EC%88%98)" title="동해 (가수)">동해</a>(21일) - <a href="/wiki/%EC%9D%B4%ED%8A%B9_(%EA%B0%80%EC%88%98)" title="이특 (가수)">이특</a>(22일) - <a href="/wiki/%EC%8B%A0%EB%8F%99_(%EA%B0%80%EC%88%98)" title="신동 (가수)">신동</a>(23일) - <a href="/wiki/%EA%B7%9C%ED%98%84" title="규현">규현</a>(23일) - <a href="/wiki/%EC%84%B1%EB%AF%BC_(1986%EB%85%84)" title="성민 (1986년)">성민</a>(24일) - <a href="/wiki/%EB%A0%A4%EC%9A%B1" title="려욱">려욱</a>(25일) - <a href="/wiki/%EC%98%88%EC%84%B1" title="예성">예성</a>(26일) - <a href="/wiki/%EC%8B%9C%EC%9B%90_(%EA%B0%80%EC%88%98)" title="시원 (가수)">시원</a>(27일) - <a href="/wiki/%EA%B9%80%ED%9D%AC%EC%B2%A0_(%EA%B0%80%EC%88%98)" title="김희철 (가수)">희철</a>(27일) 순이다.</li>
<li><a href="/wiki/2011%EB%85%84" title="2011년">2011년</a> <a href="/wiki/8%EC%9B%94_5%EC%9D%BC" title="8월 5일">8월 5일</a> <a class="mw-redirect" href="/wiki/KBS" title="KBS">KBS</a> 《<a href="/wiki/%EB%AE%A4%EC%A7%81%EB%B1%85%ED%81%AC" title="뮤직뱅크">뮤직뱅크</a>》를 시작으로 <a href="/wiki/8%EC%9B%94_6%EC%9D%BC" title="8월 6일">6일</a> <a class="mw-redirect" href="/wiki/MBC" title="MBC">MBC</a> 《<a class="mw-redirect" href="/wiki/%EC%9D%8C%EC%95%85%EC%A4%91%EC%8B%AC" title="음악중심">음악중심</a>》, <a href="/wiki/8%EC%9B%94_7%EC%9D%BC" title="8월 7일">7일</a> <a class="mw-disambig" href="/wiki/SBS" title="SBS">SBS</a> 《<a class="mw-redirect" href="/wiki/%EC%9D%B8%EA%B8%B0%EA%B0%80%EC%9A%94" title="인기가요">인기가요</a>》로 컴백하였다.<sup class="reference" id="cite_ref-22"><a href="#cite_note-22">[22]</a></sup></li>
<li><a href="/wiki/8%EC%9B%94_22%EC%9D%BC" title="8월 22일">8월 22일</a>과 <a href="/wiki/8%EC%9B%94_30%EC%9D%BC" title="8월 30일">8월 30일</a>에 멤버 전원이 <a class="mw-disambig" href="/wiki/SBS" title="SBS">SBS</a> <a href="/wiki/%EA%B0%95%EC%8B%AC%EC%9E%A5" title="강심장">강심장</a>에 출연했다. 녹화는 4일 진행되었다.<sup class="reference" id="cite_ref-23"><a href="#cite_note-23">[23]</a></sup></li>
<li><a href="/wiki/%EA%B9%80%ED%9D%AC%EC%B2%A0_(%EA%B0%80%EC%88%98)" title="김희철 (가수)">희철</a>은 훈련소 입소 전 <a href="/wiki/2011%EB%85%84" title="2011년">2011년</a> <a href="/wiki/8%EC%9B%94_28%EC%9D%BC" title="8월 28일">8월 28일</a> <a class="mw-redirect" href="/wiki/%EC%9D%B8%EA%B8%B0%EA%B0%80%EC%9A%94" title="인기가요">인기가요</a>에서 마지막 무대를 가졌으며 이 무대에서 슈퍼주니어는 뮤티즌송을 차지했다.<sup class="reference" id="cite_ref-24"><a href="#cite_note-24">[24]</a></sup></li>
<li><a href="/wiki/9%EC%9B%94_12%EC%9D%BC" title="9월 12일">9월 12일</a> <a href="/wiki/%EC%8B%A0%EB%8F%99_(%EA%B0%80%EC%88%98)" title="신동 (가수)">신동</a>, <a href="/wiki/%EC%9D%80%ED%98%81" title="은혁">은혁</a>, <a href="/wiki/%EB%8F%99%ED%95%B4_(%EA%B0%80%EC%88%98)" title="동해 (가수)">동해</a>, <a href="/wiki/%EA%B7%9C%ED%98%84" title="규현">규현</a>이 <a class="mw-redirect" href="/wiki/KBS2" title="KBS2">KBS2</a> <a href="/wiki/%EB%8C%80%EA%B5%AD%EB%AF%BC_%ED%86%A0%ED%81%AC%EC%87%BC_%EC%95%88%EB%85%95%ED%95%98%EC%84%B8%EC%9A%94" title="대국민 토크쇼 안녕하세요">대국민 토크쇼 안녕하세요</a>에 출연했다.<sup class="reference" id="cite_ref-25"><a href="#cite_note-25">[25]</a></sup></li>
<li><a href="/wiki/9%EC%9B%94_18%EC%9D%BC" title="9월 18일">9월 18일</a> <a href="/wiki/%EC%9D%B4%ED%8A%B9_(%EA%B0%80%EC%88%98)" title="이특 (가수)">이특</a>, <a href="/wiki/%EC%98%88%EC%84%B1" title="예성">예성</a>, <a href="/wiki/%EC%8B%A0%EB%8F%99_(%EA%B0%80%EC%88%98)" title="신동 (가수)">신동</a>, <a href="/wiki/%EC%84%B1%EB%AF%BC_(1986%EB%85%84)" title="성민 (1986년)">성민</a>, <a href="/wiki/%EC%9D%80%ED%98%81" title="은혁">은혁</a>, <a href="/wiki/%EB%8F%99%ED%95%B4_(%EA%B0%80%EC%88%98)" title="동해 (가수)">동해</a>가 <a class="mw-redirect" href="/wiki/KBS2" title="KBS2">KBS2</a> <a class="mw-redirect" href="/wiki/%EC%B6%9C%EB%B0%9C_%EB%93%9C%EB%A6%BC%ED%8C%80" title="출발 드림팀">출발 드림팀</a> 최강신인왕전 2탄에 출연했다. 녹화 당시 세트에 문제가 생겨 녹화가 중단되었을 때, 멤버들은 기다리는 팬들에게 사인을 해주는 등 팬서비스를 발휘했다.<sup class="reference" id="cite_ref-26"><a href="#cite_note-26">[26]</a></sup><a href="/wiki/%EC%9D%80%ED%98%81" title="은혁">은혁</a>은 이 방송에서 우승을 차지했다.<sup class="reference" id="cite_ref-27"><a href="#cite_note-27">[27]</a></sup> 동해는 해당 방송에서 부상을 당했으며,<sup class="reference" id="cite_ref-28"><a href="#cite_note-28">[28]</a></sup> 한동안 음악 방송에서 의자에 앉은 채 안무를 하는 모습을 보였다.<sup class="reference" id="cite_ref-29"><a href="#cite_note-29">[29]</a></sup></li>
</ul>
