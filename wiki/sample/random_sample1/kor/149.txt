<p><b>BSD</b>(<span lang="en" xml:lang="en">Berkeley Software Distribution</span>)는 1977년부터 1995년까지 <a href="/wiki/%EB%AF%B8%EA%B5%AD" title="미국">미국</a> <a href="/wiki/%EC%BA%98%EB%A6%AC%ED%8F%AC%EB%8B%88%EC%95%84_%EB%8C%80%ED%95%99%EA%B5%90_%EB%B2%84%ED%81%B4%EB%A6%AC" title="캘리포니아 대학교 버클리">캘리포니아 대학교 버클리</a>(<span lang="en" xml:lang="en">University of California, Berkeley</span>)의 <a class="new" href="/w/index.php?title=CSRG&amp;action=edit&amp;redlink=1" title="CSRG (없는 문서)">CSRG</a>(Computer Systems Research Group)에서 개발한 <a href="/wiki/%EC%9C%A0%EB%8B%89%EC%8A%A4" title="유닉스">유닉스</a> <a href="/wiki/%EC%9A%B4%EC%98%81_%EC%B2%B4%EC%A0%9C" title="운영 체제">운영 체제</a>이다. 오늘날 BSD라는 용어는 <a href="/wiki/%EC%9C%A0%EB%8B%89%EC%8A%A4_%EA%B3%84%EC%97%B4" title="유닉스 계열">유닉스 계열</a> 운영 체제 계열에서 분기되어 형성된 BSD 파생판을 두루 가리키는 용어로 자리잡혀 있다. 오리지널 BSD 코드로부터 파생된 운영 체제들은 현재까지도 활발히 개발되고 널리 사용되고 있다.</p>
<p>역사적으로 BSD는 <b>버클리 유닉스</b>(Berkeley Unix)라는 유닉스의 갈래로 간주되는데, 그 까닭은 오리지널 <a href="/wiki/AT%26T" title="AT&amp;T">AT&amp;T</a> 유닉스 운영 체제의 초기 코드 기반과 디자인을 공유하기 때문이다. 1980년대에 BSD는 <a class="mw-disambig" href="/wiki/DEC" title="DEC">DEC</a> <a class="new" href="/w/index.php?title=ULTRIX&amp;action=edit&amp;redlink=1" title="ULTRIX (없는 문서)">ULTRIX</a>, <a href="/wiki/%EC%8D%AC_%EB%A7%88%EC%9D%B4%ED%81%AC%EB%A1%9C%EC%8B%9C%EC%8A%A4%ED%85%9C%EC%A6%88" title="썬 마이크로시스템즈">썬 마이크로시스템즈</a> <a class="mw-redirect" href="/wiki/SunOS" title="SunOS">SunOS</a>와 같은 사유 유닉스 변종의 형태로 <a href="/wiki/%EC%9B%8C%ED%81%AC%EC%8A%A4%ED%85%8C%EC%9D%B4%EC%85%98" title="워크스테이션">워크스테이션</a> 계열 시스템의 업체들에 널리 채택되었다. 이는 라이선스를 받기 쉽고, 수많은 기술 회사들의 창립자들이 느꼈던 친숙함 때문이다.</p>
<p>이러한 사유 BSD 파생판들은 1990년대에 유닉스 <a href="/wiki/%EC%9C%A0%EB%8B%89%EC%8A%A4_%EC%8B%9C%EC%8A%A4%ED%85%9C_V" title="유닉스 시스템 V">시스템 V 릴리스 4</a> 및 <a class="new" href="/w/index.php?title=%ED%8A%B8%EB%A3%A864_%EC%9C%A0%EB%8B%89%EC%8A%A4&amp;action=edit&amp;redlink=1" title="트루64 유닉스 (없는 문서)">OSF/1</a> 시스템으로 인해 빛을 보지 못했지만(둘 다 BSD 코드 및 기타 현대 유닉스 시스템의 기초를 통합하고 있었음) 나중에 BSD 릴리스들은 <a href="/wiki/FreeBSD" title="FreeBSD">FreeBSD</a>, <a class="mw-redirect" href="/wiki/%EC%98%A4%ED%94%88BSD" title="오픈BSD">오픈BSD</a>, <a href="/wiki/NetBSD" title="NetBSD">NetBSD</a>, <a href="/wiki/TrueOS" title="TrueOS">TrueOS</a>와 같은 진행 중인 몇 가지 <a href="/wiki/%EC%98%A4%ED%94%88_%EC%86%8C%EC%8A%A4" title="오픈 소스">오픈 소스</a> 개발 프로젝트들의 기초를 제공하였다. 다시 말해, <a href="/wiki/%EC%9C%88%EB%8F%84%EC%9A%B0_NT_3.1" title="윈도우 NT 3.1">윈도우 NT 3.1</a>의 <a class="mw-redirect" href="/wiki/TCP/IP" title="TCP/IP">TCP/IP</a> 네트워킹 코드라든지<sup class="reference" id="cite_ref-1"><a href="#cite_note-1">[1]</a></sup>, <a href="/wiki/%EC%95%A0%ED%94%8C_(%EA%B8%B0%EC%97%85)" title="애플 (기업)">애플</a>의 <a class="mw-redirect" href="/wiki/OS_X" title="OS X">OS X</a>과 <a class="mw-redirect" href="/wiki/IOS_(%EC%95%A0%ED%94%8C)" title="IOS (애플)">iOS</a>의 기초 대부분과 같이 현대의 사유 운영 체제에 부분적으로나 전체적으로 포함되었다는 것을 의미한다.</p>

<h2><span class="mw-headline" id=".EC.97.AD.EC.82.AC">역사</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=BSD&amp;action=edit&amp;section=1" title="부분 편집: 역사">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<h3><span class="mw-headline" id="PDP-11_.EC.B4.88.EA.B8.B0">PDP-11 초기</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=BSD&amp;action=edit&amp;section=2" title="부분 편집: PDP-11 초기">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>1970년대 벨 연구소의 유닉스 초기 배포판들은 운영 체제에 대한 소스 코드를 포함했으며 대학교의 연구원들이 유닉스를 수정하고 확장할 수 있게 했다. 버클리의 최초의 유닉스 시스템은 1974년 설치된 <a href="/wiki/PDP-11" title="PDP-11">PDP-11</a>이며 <a href="/wiki/%EC%BB%B4%ED%93%A8%ED%84%B0_%EA%B3%BC%ED%95%99" title="컴퓨터 과학">컴퓨터 과학</a>부는 그 뒤 대형의 연구를 위해 이를 사용하였다.</p>
<p>최초의 버클리 소프트웨어인 1BSD는 1978년 3월 9일 출시되었다.<sup class="reference" id="cite_ref-2"><a href="#cite_note-2">[2]</a></sup></p>
<p>2BSD는 1979년 5월 출시되었다.<sup class="reference" id="cite_ref-tuhs-2bsd_3-0"><a href="#cite_note-tuhs-2bsd-3">[3]</a></sup></p>
<h3><span class="mw-headline" id="VAX_.EB.B2.84.EC.A0.84">VAX 버전</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=BSD&amp;action=edit&amp;section=3" title="부분 편집: VAX 버전">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<ul>
<li>4BSD</li>
<li>4.1BSD</li>
<li>4.2BSD</li>
</ul>
<h3><span class="mw-headline" id="4.3BSD">4.3BSD</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=BSD&amp;action=edit&amp;section=4" title="부분 편집: 4.3BSD">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<ul>
<li>4.3BSD는 1986년 6월 출시되었다. 4.2BS의 새로운 기여분들 가운데 상당수 성능이 개선된 것이 주요 변경 사항이다. 이전 판들과 달리 BSD의 TCP/IP 구현은 BBN의 공식 구현과 상당히 다르다.</li>
<li>4.3BSD-Reno는 1990년 초에 출시되었다. 4.4BSD의 초기 개발 동안의 중간 릴리즈이다.</li>
</ul>
<h3><span class="mw-headline" id="4.4BSD_.EB.B0.8F_.ED.8C.8C.EC.83.9D">4.4BSD 및 파생</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=BSD&amp;action=edit&amp;section=5" title="부분 편집: 4.4BSD 및 파생">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<ul>
<li>1994년 6월 4.4BSD가 두 가지 형태로 출시되었다.: 4.4BSD-Lite (AT&amp;T 소스 미포함), 4.4BSD-Encumbered (AT&amp;T 라이선스에만 적용)</li>
<li>4.4BSD-Lite 릴리즈 2 (버클리판)</li>
</ul>
<p>애플의 <a class="mw-redirect" href="/wiki/OS_X" title="OS X">OS X</a>의 기반이 되는 <a href="/wiki/%EB%8B%A4%EC%9C%88_(%EC%9A%B4%EC%98%81_%EC%B2%B4%EC%A0%9C)" title="다윈 (운영 체제)">다윈</a> 시스템은 4.4BSD-Lite2와 FreeBSD의 파생이다. <a href="/wiki/%EC%86%94%EB%9D%BC%EB%A6%AC%EC%8A%A4_(%EC%9A%B4%EC%98%81_%EC%B2%B4%EC%A0%9C)" title="솔라리스 (운영 체제)">솔라리스</a>와 같은 다양한 상용 유닉스 운영 체제들은 상당한 양의 BSD 코드를 포함하고 있다.</p>
