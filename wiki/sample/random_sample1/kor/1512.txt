<div class="mw-body-content" id="bodyContent">
<div id="siteSub">위키백과, 우리 모두의 백과사전.</div>

<div class="mw-content-ltr" dir="ltr" id="mw-content-text" lang="ko"><script>function mfTempOpenSection(id){var block=document.getElementById("mf-section-"+id);block.className+=" open-block";block.previousSibling.className+=" open-block";}</script><p><b>T-X 프로그램</b>은 <a href="/wiki/%EB%AF%B8%EA%B5%AD_%EA%B3%B5%EA%B5%B0" title="미국 공군">미국 공군</a>의 <a href="/wiki/T-38_%ED%83%A4%EB%A1%A0" title="T-38 탤론">T-38 탤론</a> <a class="new" href="/w/index.php?title=%EA%B3%A0%EB%93%B1%ED%9B%88%EB%A0%A8%EA%B8%B0&amp;action=edit&amp;redlink=1" title="고등훈련기 (없는 문서)">고등훈련기</a>를 신형으로 교체하는 사업이다.</p>
<p>미국 공군은 최초엔 350대를 구매할 계획이나, 전체적으로 1,000대 이상을 주문할 수도 있다.</p>
<p>2017 회계년도 부터 사업이 시작될 예정이다. 2016년말까지 입찰 제안서를 받아, 2017년말에 기종을 결정할 계획이다.</p>

<h2><span class="mw-headline" id=".ED.9B.84.EB.B3.B4.EA.B8.B0.EC.A2.85">후보기종</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=T-X_%ED%94%84%EB%A1%9C%EA%B7%B8%EB%9E%A8&amp;action=edit&amp;section=1" title="부분 편집: 후보기종">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<h3><span class="mw-headline" id=".EB.A0.88.EC.9D.B4.EC.8B.9C.EC.98.A8">레이시온</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=T-X_%ED%94%84%EB%A1%9C%EA%B7%B8%EB%9E%A8&amp;action=edit&amp;section=2" title="부분 편집: 레이시온">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p><a href="/wiki/%EB%A0%88%EC%9D%B4%EC%8B%9C%EC%98%A8" title="레이시온">레이시온</a>은 이탈리아 <a href="/wiki/%EC%95%8C%EB%A0%88%EB%8B%88%EC%95%84_%EC%95%84%EC%97%90%EB%A5%B4%EB%A7%88%ED%82%A4_M-346" title="알레니아 아에르마키 M-346">M-346 마스터</a>로 응찰할 것이다.</p>
<h3><span class="mw-headline" id=".EB.85.B8.EC.8A.A4.EB.9F.BD_.EA.B7.B8.EB.9F.AC.EB.A8.BC">노스럽 그러먼</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=T-X_%ED%94%84%EB%A1%9C%EA%B7%B8%EB%9E%A8&amp;action=edit&amp;section=3" title="부분 편집: 노스럽 그러먼">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p><a href="/wiki/%EB%85%B8%EC%8A%A4%EB%9F%BD_%EA%B7%B8%EB%9F%AC%EB%A8%BC" title="노스럽 그러먼">노스럽 그러먼</a>은 영국 <a href="/wiki/BAE_%ED%98%B8%ED%81%AC" title="BAE 호크">BAE 호크</a>로 응찰할 것이다.</p>
<h3><span class="mw-headline" id=".EB.A1.9D.ED.9E.88.EB.93.9C_.EB.A7.88.ED.8B.B4">록히드 마틴</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=T-X_%ED%94%84%EB%A1%9C%EA%B7%B8%EB%9E%A8&amp;action=edit&amp;section=4" title="부분 편집: 록히드 마틴">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<div class="thumb tright">
<div class="thumbinner" style="width:222px;"><a class="image" href="/wiki/%ED%8C%8C%EC%9D%BC:KAI_T-50_Golden_Eable_by_Ryabtsev.jpg"><img alt="" class="thumbimage" data-file-height="672" data-file-width="1100" height="134" src="//upload.wikimedia.org/wikipedia/commons/thumb/1/17/KAI_T-50_Golden_Eable_by_Ryabtsev.jpg/220px-KAI_T-50_Golden_Eable_by_Ryabtsev.jpg" srcset="//upload.wikimedia.org/wikipedia/commons/thumb/1/17/KAI_T-50_Golden_Eable_by_Ryabtsev.jpg/330px-KAI_T-50_Golden_Eable_by_Ryabtsev.jpg 1.5x, //upload.wikimedia.org/wikipedia/commons/thumb/1/17/KAI_T-50_Golden_Eable_by_Ryabtsev.jpg/440px-KAI_T-50_Golden_Eable_by_Ryabtsev.jpg 2x" width="220"/></a>

<a href="/wiki/T-50_%EA%B3%A8%EB%93%A0%EC%9D%B4%EA%B8%80" title="T-50 골든이글">T-50 골든이글</a></div>

<p><a href="/wiki/%EB%A1%9D%ED%9E%88%EB%93%9C_%EB%A7%88%ED%8B%B4" title="록히드 마틴">록히드 마틴</a>은 한국 <a href="/wiki/T-50_%EA%B3%A8%EB%93%A0%EC%9D%B4%EA%B8%80" title="T-50 골든이글">T-50 골든이글</a>을 제시했다. 경쟁기종 중에서 가장 대형이며, 유일하게 초음속을 낼 수 있다. 2015년 12월 17일 경남 사천 <a class="mw-redirect" href="/wiki/KAI" title="KAI">KAI</a>에서 T-X용 <a href="/wiki/T-50_%EA%B3%A8%EB%93%A0%EC%9D%B4%EA%B8%80" title="T-50 골든이글">T-50 골든이글</a>인 <a href="/wiki/T-50A" title="T-50A">T-50A</a>의 출고식을 가졌다. <a href="/wiki/%EB%B0%95%EA%B7%BC%ED%98%9C" title="박근혜">박근혜</a> 대통령이 참석했다.</p>
<pre>
<a class="external text" href="https://www.youtube.com/watch?v=z0ssyPITBLU" rel="nofollow"><b>치열한 미국 훈련기 수주전…100조원 수출 기대 - </b></a><a href="/wiki/T-50A" title="T-50A">T-50A</a>관련 영상
<a class="external text" href="https://www.youtube.com/watch?v=H_5Rcd1Qcjc" rel="nofollow"><b>Lockheed Martin Flies First T-50A Upgraded Aircraft for T-X Competition</b></a>
</pre>
<p><a href="/wiki/T-50A" title="T-50A">T-50A</a>는 가상훈련, 개방형 시스템구조, 공중급유, 5세대 조종석 등을 갖춰 조종사들이 <a class="mw-redirect" href="/wiki/F-35" title="F-35">F-35</a>나 <a class="mw-redirect" href="/wiki/F-22" title="F-22">F-22</a> 등 최신 전투기에도 쉽게 적응할 수 있도록 했다.<sup class="reference" id="cite_ref-1"><a href="#cite_note-1">[1]</a></sup></p>
<p>미 공군은 차세대 훈련기를 <a class="mw-redirect" href="/wiki/F-35" title="F-35">F-35</a> 전투기에 대한 훈련기로 활용할 계획이다.<sup class="reference" id="cite_ref-2"><a href="#cite_note-2">[2]</a></sup></p>
<p>T-X 1차분 물량만 350대, 금액으로 17조원에 이른다. 이를 수주하게 되면 산업 파급 효과는 7조3000억원, 일자리 창출은 연 4만3000명 규모로 전망된다.<sup class="reference" id="cite_ref-3"><a href="#cite_note-3">[3]</a></sup></p>
<p>2017년 말 납품되는 T-X 초기 물량은 360대(17조원)이며 가상적기, <a href="/wiki/%EB%AF%B8%EA%B5%AD_%ED%95%B4%EA%B5%B0" title="미국 해군">미국 해군</a> 항공모함용 훈련기 등 후속 물량까지 합하면 650대(33조원)에 이른다. 여기에 미국 외 제3국 수출 물량으로 1000여대(55조원)의 교체 수요가 추가적으로 발생할 것으로 예상된다. 이 사업을 수주하면 세계 고등훈련기 시장을 주도하게 돼 최소 1000대 이상(70조원)의 고등훈련기를 수출할 수 있을 것으로 KAI는 전망하고 있다.<sup class="reference" id="cite_ref-4"><a href="#cite_note-4">[4]</a></sup></p>
<p>2016년 3월 12일 일본의 항공군사평론가 세키 켄타로(關 賢太郞)는 "미국의 <a class="mw-redirect" href="/wiki/T-X_%EC%82%AC%EC%97%85" title="T-X 사업">T-X 사업</a>에 <a class="mw-redirect mw-disambig" href="/wiki/T-50" title="T-50">T-50</a>이 선정된다면, <a class="mw-redirect" href="/wiki/F-35" title="F-35">F-35</a>를 도입할 국가들의 <a class="new" href="/w/index.php?title=%EA%B3%A0%EB%93%B1%ED%9B%88%EB%A0%A8%EA%B8%B0&amp;action=edit&amp;redlink=1" title="고등훈련기 (없는 문서)">고등훈련기</a> 판로가 열릴 것"이라며 일본 <a href="/wiki/%ED%95%AD%EA%B3%B5%EC%9E%90%EC%9C%84%EB%8C%80" title="항공자위대">항공자위대</a>가 <a href="/wiki/T-50_%EA%B3%A8%EB%93%A0%EC%9D%B4%EA%B8%80" title="T-50 골든이글">T-50 골든이글</a>을 도입할 수도 있을 것이라고 전망했다.<sup class="reference" id="cite_ref-5"><a href="#cite_note-5">[5]</a></sup></p>
<h3><span class="mw-headline" id=".ED.85.8D.EC.8A.A4.ED.8A.B8.EB.A1.A0_.EC.97.90.EC.96.B4.EB.9E.9C.EB.93.9C">텍스트론 에어랜드</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=T-X_%ED%94%84%EB%A1%9C%EA%B7%B8%EB%9E%A8&amp;action=edit&amp;section=5" title="부분 편집: 텍스트론 에어랜드">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p><a class="new" href="/w/index.php?title=%ED%85%8D%EC%8A%A4%ED%8A%B8%EB%A1%A0_%EC%97%90%EC%96%B4%EB%9E%9C%EB%93%9C&amp;action=edit&amp;redlink=1" title="텍스트론 에어랜드 (없는 문서)">텍스트론 에어랜드</a>는 <a class="new" href="/w/index.php?title=%ED%85%8D%EC%8A%A4%ED%8A%B8%EB%A1%A0_%EC%97%90%EC%96%B4%EB%9E%9C%EB%93%9C_%EC%8A%A4%EC%BD%9C%ED%94%BC%EC%98%A8&amp;action=edit&amp;redlink=1" title="텍스트론 에어랜드 스콜피온 (없는 문서)">텍스트론 에어랜드 스콜피온</a>을 준비중이다.</p>
<h3><span class="mw-headline" id=".EB.B3.B4.EC.9E.89">보잉</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=T-X_%ED%94%84%EB%A1%9C%EA%B7%B8%EB%9E%A8&amp;action=edit&amp;section=6" title="부분 편집: 보잉">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p><a href="/wiki/%EB%B3%B4%EC%9E%89" title="보잉">보잉</a>은 스웨덴 <a href="/wiki/%EC%82%AC%EB%B8%8C" title="사브">사브</a>와 함께 새로운 훈련기를 제시할 것이다. <a href="/wiki/T-50_%EA%B3%A8%EB%93%A0%EC%9D%B4%EA%B8%80" title="T-50 골든이글">T-50 골든이글</a> 보다 싸게 만들 것이라고 한다.</p>
