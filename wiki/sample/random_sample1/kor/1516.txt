<p><b>TAROM</b>(<span style="font-size: smaller;"><a href="/wiki/%EB%A3%A8%EB%A7%88%EB%8B%88%EC%95%84%EC%96%B4" title="루마니아어">루마니아어</a>:</span> <span lang="ro" xml:lang="ro">Transporturile Aeriene Române</span>, <span style="font-size: smaller;"><a href="/wiki/%EC%98%81%EC%96%B4" title="영어">영어</a>:</span> <span lang="en" xml:lang="en">TAROM Romanian Air Transport</span>) 또는 <b>타롬 항공</b>은 <a href="/wiki/%EB%B6%80%EC%BF%A0%EB%A0%88%EC%8A%88%ED%8B%B0" title="부쿠레슈티">부쿠레슈티</a>에 본사를 두고 있는 <a href="/wiki/%EB%A3%A8%EB%A7%88%EB%8B%88%EC%95%84" title="루마니아">루마니아</a>의 항공사로 <a href="/wiki/%ED%97%88%EB%B8%8C_%EA%B3%B5%ED%95%AD" title="허브 공항">허브 공항</a>은 <a href="/wiki/%ED%97%A8%EB%A6%AC_%EC%BD%94%EC%95%88%EB%8D%94_%EA%B5%AD%EC%A0%9C%EA%B3%B5%ED%95%AD" title="헨리 코안더 국제공항">헨리 코안더 국제공항</a>이다. TAROM(타롬)은 "루마니아 항공 운송"(<span style="font-size: smaller;"><a href="/wiki/%EB%A3%A8%EB%A7%88%EB%8B%88%EC%95%84%EC%96%B4" title="루마니아어">루마니아어</a>:</span> <span lang="ro" xml:lang="ro"><b>T</b>ransporturile <b>A</b>eriene <b>Rom</b>âne</span>)의 <a href="/wiki/%EB%A3%A8%EB%A7%88%EB%8B%88%EC%95%84%EC%96%B4" title="루마니아어">루마니아어</a> 약칭이다.</p>

<h2><span class="mw-headline" id=".EC.97.AD.EC.82.AC">역사</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=TAROM&amp;action=edit&amp;section=1" title="부분 편집: 역사">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p><a href="/wiki/1920%EB%85%84" title="1920년">1920년</a> CFRNA(<span style="font-size: smaller;"><a href="/wiki/%EB%A3%A8%EB%A7%88%EB%8B%88%EC%95%84%EC%96%B4" title="루마니아어">루마니아어</a>:</span> <span lang="ro" xml:lang="ro">Romanian Company for Air Navigation</span>)라는 이름으로 설립하였다. 초기에 중앙 <a href="/wiki/%EC%9C%A0%EB%9F%BD" title="유럽">유럽</a>의 여러 도시를 거쳐 <a href="/wiki/%EB%B6%80%EC%BF%A0%EB%A0%88%EC%8A%88%ED%8B%B0" title="부쿠레슈티">부쿠레슈티</a>와 <a class="mw-disambig" href="/wiki/%ED%8C%8C%EB%A6%AC" title="파리">파리</a>를 잇는 여객과 우편 수송 서비스를 제공했으며 <a href="/wiki/1926%EB%85%84" title="1926년">1926년</a> 사명을 CIDNA로 바꾸었다. <a class="mw-redirect" href="/wiki/%EC%A0%9C2%EC%B0%A8_%EC%84%B8%EA%B3%84%EB%8C%80%EC%A0%84" title="제2차 세계대전">제2차 세계대전</a> 이후 <a href="/wiki/%EC%86%8C%EB%A0%A8" title="소련">소련</a>이 <a href="/wiki/%EB%8F%99%EC%9C%A0%EB%9F%BD" title="동유럽">동유럽</a>에 영향력을 확대하면서 <a href="/wiki/1945%EB%85%84" title="1945년">1945년</a> <a href="/wiki/%EC%86%8C%EB%A0%A8" title="소련">소련</a>과 <a href="/wiki/%EB%A3%A8%EB%A7%88%EB%8B%88%EC%95%84" title="루마니아">루마니아</a> 합작 회사인 TARS으로 사명이 변경 되었다. <a href="/wiki/1954%EB%85%84" title="1954년">1954년</a> <a href="/wiki/%EC%86%8C%EB%A0%A8" title="소련">소련</a>이 지분을 모두 사들여 현재의 사명으로 변경했다. <a href="/wiki/1991%EB%85%84" title="1991년">1991년</a> <a href="/wiki/%EC%86%8C%EB%A0%A8" title="소련">소련</a>의 붕괴로 <a href="/wiki/%EB%A3%A8%EB%A7%88%EB%8B%88%EC%95%84" title="루마니아">루마니아</a> 국영 항공사가 되었으나 경영의 어려움을 겪다가 <a href="/wiki/2000%EB%85%84%EB%8C%80" title="2000년대">2000년대</a> 이후 수익을 내고 있다. <a href="/wiki/2007%EB%85%84" title="2007년">2007년</a>에 전년도에 비해 32% 늘어난 169만 명의 여객 수송을 기록했는데 이는 <a href="/wiki/%EC%9C%A0%EB%9F%BD" title="유럽">유럽</a>에서 <a href="/wiki/%EB%B8%8C%EB%A4%BC%EC%85%80_%ED%95%AD%EA%B3%B5" title="브뤼셀 항공">브뤼셀 항공</a>에 이어 두 번째로 높은 성장률이다. <a href="/wiki/2009%EB%85%84" title="2009년">2009년</a> <a href="/wiki/%EB%8C%80%ED%95%9C%ED%95%AD%EA%B3%B5" title="대한항공">대한항공</a>이 주도하는 <a href="/wiki/%ED%95%AD%EA%B3%B5_%EB%8F%99%EB%A7%B9" title="항공 동맹">항공 동맹</a>인 <a href="/wiki/%EC%8A%A4%EC%B9%B4%EC%9D%B4%ED%8C%80" title="스카이팀">스카이팀</a>에 가입했다.</p>
