<p><b>TOP 09</b> 또는 <b>전통 번영을 위한 책임</b>(傳統 繁榮을 爲한 責任, <span style="font-size: smaller;"><a href="/wiki/%EC%B2%B4%EC%BD%94%EC%96%B4" title="체코어">체코어</a>:</span> <span lang="cs" xml:lang="cs">Tradice Odpovědnost Prosperita</span>)은 <a href="/wiki/%EC%B2%B4%EC%BD%94" title="체코">체코</a>의 <a href="/wiki/%EC%A4%91%EB%8F%84%EC%9A%B0%ED%8C%8C" title="중도우파">중도우파</a> 정당으로, 2009년 설립되었다. <a href="/wiki/2013%EB%85%84" title="2013년">2013년</a>, <a href="/wiki/2016%EB%85%84" title="2016년">2016년</a> 총선거 결과, <a href="/wiki/%EC%B2%B4%EC%BD%94" title="체코">체코</a> 상원의 경우 <a class="new" href="/w/index.php?title=%EB%B6%81%EB%B3%B4%ED%97%A4%EB%AF%B8%EC%95%84%EB%8B%B9&amp;action=edit&amp;redlink=1" title="북보헤미아당 (없는 문서)">북보헤미아당</a>, <a href="/wiki/%EC%8B%9C%EB%AF%BC%EA%B6%8C%EB%A6%AC%EB%8B%B9" title="시민권리당">시민권리당</a>에 이어 제7당인 반면 <a href="/wiki/%EC%B2%B4%EC%BD%94" title="체코">체코</a> 하원의 경우 <a class="new" href="/w/index.php?title=%EB%B3%B4%ED%97%A4%EB%AF%B8%EC%95%84_%EB%AA%A8%EB%9D%BC%EB%B0%94_%EA%B3%B5%EC%82%B0%EB%8B%B9&amp;action=edit&amp;redlink=1" title="보헤미아 모라바 공산당 (없는 문서)">보헤미아 모라바 공산당</a>에 이어 제3당이다.</p>
