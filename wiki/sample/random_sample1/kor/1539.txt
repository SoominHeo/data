<p><b>TTA</b>(The True Audio, 티티에이, 더 트루 오디오)는 무손실 가역압축 오디오 <a href="/wiki/%ED%8C%8C%EC%9D%BC_%ED%98%95%EC%8B%9D" title="파일 형식">포맷</a>이다. 알렉산더 쥬릭(Alexander Djourik)이 개발했다. 오늘날의 일반적인 아날로그 오디오 샘플에 대하여 비교적 좋은 성능을 보여주는 적응형 예측 필터(adaptive prognostic filters)들에 기반하고 있다.</p>

<h2><span class="mw-headline" id=".ED.8A.B9.EC.A7.95">특징</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=TTA_(%EC%BD%94%EB%8D%B1)&amp;action=edit&amp;section=1" title="부분 편집: 특징">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<ul>
<li>30% 정도의 용량 압축이 가능하다.</li>
<li>리얼타임 디코드/인코드 알고리즘을 채용했다.</li>
<li><a href="/wiki/%EC%98%A4%ED%94%88_%EC%86%8C%EC%8A%A4" title="오픈 소스">오픈 소스</a> 포맷으로 <a class="mw-redirect" href="/wiki/GPL" title="GPL">GPL</a>을 따른다.</li>
<li><a href="/wiki/MP3" title="MP3">MP3</a>에서 이용하여 대중화된 <a href="/wiki/ID3" title="ID3">ID3</a>태그와 <a class="mw-redirect" href="/wiki/%EB%AA%BD%ED%82%A4_%EC%98%A4%EB%94%94%EC%98%A4" title="몽키 오디오">APE</a>의 태그를 사용할 수 있다.</li>
<li><a class="mw-redirect" href="/wiki/%EB%A7%88%ED%8A%B8%EB%A3%8C%EC%8B%9C%EC%B9%B4_(%ED%8F%AC%EB%A7%B7)" title="마트료시카 (포맷)">마트료시카</a>에 대응한다.</li>
<li>확장자는 <code>.tta</code>로, 파일 정보를 담은 <a class="mw-redirect mw-disambig" href="/wiki/CUE" title="CUE">CUE</a> 파일과 함께 하나의 <a class="mw-redirect" href="/wiki/%EB%A7%88%ED%8A%B8%EB%A3%8C%EC%8B%9C%EC%B9%B4_(%ED%8F%AC%EB%A7%B7)" title="마트료시카 (포맷)">마트료시카</a> 포맷으로 합칠 수 있다. 다음과 같이 사용할 수 있다.</li>
</ul>
<dl>
<dd>(tta+cue).mka</dd>
<dd>(DivX+TTA).mkv</dd>
</dl>
<ul>
<li>하드웨어 압축 지원</li>
</ul>
<p>TTA는 8, 16, 24 <a href="/wiki/%EB%B9%84%ED%8A%B8" title="비트">비트</a> 데이터에 대하여 무손실 압축을 수행한다. "무손실"이란 말은 <a href="/wiki/%EB%8D%B0%EC%9D%B4%ED%84%B0_%EC%95%95%EC%B6%95" title="데이터 압축">데이터 압축</a> 후에도 원본 데이터가 그대로 보존된다는 말이다 - 사용자가 압축을 다시 풀면, 원본 데이터와 똑같은 데이터를 얻을 수 있다. TTA의 압축률은 어떤 음악을 압축하느냐에 따라 달라진다. 보통 30% - 70%의 압축률을 보여준다. TTA 포맷은 또한 <a href="/wiki/ID3" title="ID3">ID3v1</a>, <a href="/wiki/ID3" title="ID3">ID3v2</a> 태그를 지원한다.</p>
<h2><span class="mw-headline" id="TTA_.ED.94.84.EB.A1.9C.EC.A0.9D.ED.8A.B8">TTA 프로젝트</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=TTA_(%EC%BD%94%EB%8D%B1)&amp;action=edit&amp;section=2" title="부분 편집: TTA 프로젝트">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>TTA 프로젝트는 다음과 같은 것들을 제공한다:</p>
<ul>
<li>자유(free) 파일 형식이자, 간단한 <a href="/wiki/%ED%8C%8C%EC%9D%BC_%ED%98%95%EC%8B%9D" title="파일 형식">파일 형식</a></li>
<li>인기있는 미디어 플레이어들에 대한 <a href="/wiki/%ED%94%8C%EB%9F%AC%EA%B7%B8%EC%9D%B8" title="플러그인">플러그인</a></li>
<li>TTA <a href="/wiki/%EB%8B%A4%EC%9D%B4%EB%A0%89%ED%8A%B8%EC%87%BC" title="다이렉트쇼">다이렉트쇼</a> 필터들</li>
<li>Tau Producer - <a class="mw-redirect" href="/wiki/%EB%A7%88%EC%9D%B4%ED%81%AC%EB%A1%9C%EC%86%8C%ED%94%84%ED%8A%B8_%EC%9C%88%EB%8F%84" title="마이크로소프트 윈도">마이크로소프트 윈도</a>용 GUI가 들어가 있는 압축기</li>
<li>C/C++ TTA 개발 라이브러리들</li>
</ul>
