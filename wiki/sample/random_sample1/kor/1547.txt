<p><i><b>The Black Parade</b></i>는 <a href="/wiki/%EB%89%B4%EC%A0%80%EC%A7%80_%EC%A3%BC" title="뉴저지 주">뉴저지 주</a> 출신 <a class="mw-disambig" href="/wiki/%EB%A1%9D" title="록">록</a> 밴드 <a href="/wiki/%EB%A7%88%EC%9D%B4_%EC%BC%80%EB%AF%B8%EC%BB%AC_%EB%A1%9C%EB%A7%A8%EC%8A%A4" title="마이 케미컬 로맨스">마이 케미컬 로맨스</a>의 세 번째 <a href="/wiki/%EC%9D%8C%EB%B0%98" title="음반">음반</a>이자 <a href="/wiki/2006%EB%85%84" title="2006년">2006년</a> <a href="/wiki/10%EC%9B%94" title="10월">10월</a>에 발매되었다. 컨셉 음반이자, 록 오페라 형식으로 "환자"라는 캐릭터로 그의 죽음까지 전개된다.</p>
<p>음반의 프로듀서로는 <a href="/wiki/%EA%B7%B8%EB%A6%B0_%EB%8D%B0%EC%9D%B4" title="그린 데이">그린 데이</a>와 오랫동안 작업해 온 프로듀서 <a class="new" href="/w/index.php?title=%EB%A1%AD_%EC%B9%B4%EB%B0%9C%EB%A1%9C&amp;action=edit&amp;redlink=1" title="롭 카발로 (없는 문서)">롭 카발로</a>가 맡았으며, 《The Black Parade》는 또한 <a href="/wiki/%EB%B0%A5_%EB%B8%8C%EB%9D%BC%EC%9D%B4%EC%96%B4" title="밥 브라이어">밥 브라이어</a>가 녹음에 참여한 첫 번째 음반이다. 발매 후 《The Black Parade》는 전체적으로 커다란 성공을 거두었으며 마이 케미컬 로맨스는 <a href="/wiki/%EA%B7%B8%EB%9E%98%EB%AF%B8%EC%83%81" title="그래미상">그래미상</a>에서 최고의 박스, 한정 패키지 부문에 후보로 선정되기도 했다.</p>
<p>음반에서 발매된 싱글로는 "<a class="new" href="/w/index.php?title=Welcome_to_the_Black_Parade&amp;action=edit&amp;redlink=1" title="Welcome to the Black Parade (없는 문서)">Welcome to the Black Parade</a>", "<a class="new" href="/w/index.php?title=Famous_Last_Words&amp;action=edit&amp;redlink=1" title="Famous Last Words (없는 문서)">Famous Last Words</a>", "<a class="new" href="/w/index.php?title=I_Don%27t_Love_You&amp;action=edit&amp;redlink=1" title="I Don't Love You (없는 문서)">I Don't Love You</a>", "<a class="new" href="/w/index.php?title=Teenagers&amp;action=edit&amp;redlink=1" title="Teenagers (없는 문서)">Teenagers</a>"가 있으며 또한, 처음으로 음반에서 싱글을 4개 발매했다. 마이 케미컬 로맨스는 음반 발매 후 <a class="mw-redirect" href="/wiki/%EB%B6%81%EB%AF%B8" title="북미">북미</a>, <a href="/wiki/%EC%9C%A0%EB%9F%BD" title="유럽">유럽</a>을 시작으로 전 세계적인 투어를 전개하였다.</p>

<h2><span class="mw-headline" id=".EC.98.81.ED.96.A5">영향</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=The_Black_Parade&amp;action=edit&amp;section=1" title="부분 편집: 영향">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>리드 <a href="/wiki/%EA%B0%80%EC%88%98" title="가수">보컬</a> 제라드 웨이는 <a href="/wiki/%ED%80%B8_(%EB%B0%B4%EB%93%9C)" title="퀸 (밴드)">퀸</a>과 <a href="/wiki/%ED%95%91%ED%81%AC_%ED%94%8C%EB%A1%9C%EC%9D%B4%EB%93%9C" title="핑크 플로이드">핑크 플로이드</a>에게서 음반에 영향을 받았다고 밝혔다. "<a class="new" href="/w/index.php?title=Welcome_to_the_Black_Parade&amp;action=edit&amp;redlink=1" title="Welcome to the Black Parade (없는 문서)">Welcome to the Black Parade</a>"의 <a href="/wiki/%EA%B8%B0%ED%83%80" title="기타">기타</a> 부분은 퀸에게서 영향을 받았고, 추가로 밴드 멤버들은 핑크 플로이드의 정규 음반 <i>The Wall</i> 에서 영향을 받았는데 특히 음반 첫 번째 곡 "The End."는 핑크 플로이드의 음반 《The Wall》의 첫 번째 곡 "In the Flesh?" 에 영향을 받았다고 밝혔다. 또한 몇 개에 인터뷰에서 제라드 웨이는 큰 영향을 끼친 밴드 중 하나가 <a href="/wiki/%EC%8A%A4%EB%A7%A4%EC%8B%B1_%ED%8E%8C%ED%82%A8%EC%8A%A4" title="스매싱 펌킨스">스매싱 펌킨스</a>라고 밝혔다.</p>
<h2><span class="mw-headline" id=".EC.88.9C.EC.9C.84">순위</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=The_Black_Parade&amp;action=edit&amp;section=2" title="부분 편집: 순위">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>《The Black Parade》는 발매 후 영국 음반 차트에서 <a href="/wiki/%EB%A1%9C%EB%B9%84_%EC%9C%8C%EB%A6%AC%EC%97%84%EC%8A%A4" title="로비 윌리엄스">로비 윌리엄스</a>에 이어 2위로 올랐다. 미국에서도 또한 <a href="/wiki/%EB%B9%8C%EB%B3%B4%EB%93%9C_200" title="빌보드 200">빌보드 200</a>에 2위로 올랐으며 발매 첫 주 24만장의 판매고를 기록 해 이전 음반 《<a href="/wiki/Three_Cheers_for_Sweet_Revenge" title="Three Cheers for Sweet Revenge">Three Cheers for Sweet Revenge</a>》 발매 후 기록한 한 주 최고 판매 기록인 3만 8천장의 기록을 갱신했다. 연합 국제 차트에선 1위에 올라 첫 주에 38만 1천장의 판매고를 기록했다.</p>
<h2><span class="mw-headline" id=".EB.B8.94.EB.9E.99.ED.8D.BC.EB.A0.88.EC.9D.B4.EB.93.9C_.ED.88.AC.EC.96.B4">블랙퍼레이드 투어</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=The_Black_Parade&amp;action=edit&amp;section=3" title="부분 편집: 블랙퍼레이드 투어">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<div class="rellink relarticle mainarticle"><img alt="" data-file-height="128" data-file-width="128" height="16" src="//upload.wikimedia.org/wikipedia/commons/thumb/e/ec/Crystal_Clear_app_xmag.svg/16px-Crystal_Clear_app_xmag.svg.png" srcset="//upload.wikimedia.org/wikipedia/commons/thumb/e/ec/Crystal_Clear_app_xmag.svg/24px-Crystal_Clear_app_xmag.svg.png 1.5x, //upload.wikimedia.org/wikipedia/commons/thumb/e/ec/Crystal_Clear_app_xmag.svg/32px-Crystal_Clear_app_xmag.svg.png 2x" width="16"/> 이 부분의 본문은 <a class="new" href="/w/index.php?title=%EB%B8%94%EB%9E%99%ED%8D%BC%EB%A0%88%EC%9D%B4%EB%93%9C_%EC%9B%94%EB%93%9C_%ED%88%AC%EC%96%B4&amp;action=edit&amp;redlink=1" title="블랙퍼레이드 월드 투어 (없는 문서)">블랙퍼레이드 월드 투어</a>입니다.</div>
<p>마이 케미컬 로맨스는 <a href="/wiki/2007%EB%85%84" title="2007년">2007년</a> <a href="/wiki/2%EC%9B%94_22%EC%9D%BC" title="2월 22일">2월 22일</a> <a href="/wiki/%EB%89%B4%ED%96%84%ED%94%84%EC%85%94_%EC%A3%BC" title="뉴햄프셔 주">뉴햄프셔 주</a> <a href="/wiki/%EB%A7%A8%EC%B2%B4%EC%8A%A4%ED%84%B0" title="맨체스터">맨체스터</a>에서 공연을 시작으로 <a class="new" href="/w/index.php?title=%EB%B8%94%EB%9E%99%ED%8D%BC%EB%A0%88%EC%9D%B4%EB%93%9C_%EC%9B%94%EB%93%9C_%ED%88%AC%EC%96%B4&amp;action=edit&amp;redlink=1" title="블랙퍼레이드 월드 투어 (없는 문서)">블랙퍼레이드 투어</a>를 시작했다. 투어 1차 <a href="/wiki/%EB%9D%BC%EC%9D%B4%EC%A6%88_%EC%96%B4%EA%B2%8C%EC%9D%B8%EC%8A%A4%ED%8A%B8" title="라이즈 어게인스트">라이즈 어게인스트</a>, 유럽 투어에서는 <a class="new" href="/w/index.php?title=%EC%84%9C%EC%8A%A4%EB%8D%B0%EC%9D%B4&amp;action=edit&amp;redlink=1" title="서스데이 (없는 문서)">서스데이</a>, 북미 2차 투어에서는 <a href="/wiki/%EB%AE%A4%EC%A6%88_(%EB%B0%B4%EB%93%9C)" title="뮤즈 (밴드)">뮤즈</a>와 공연을 가졌다. <a href="/wiki/2007%EB%85%84" title="2007년">2007년</a> <a href="/wiki/5%EC%9B%94_22%EC%9D%BC" title="5월 22일">5월 22일</a> <a href="/wiki/%EB%AF%B8%EA%B5%AD" title="미국">미국</a> <a href="/wiki/%EC%98%A4%EB%A6%AC%EA%B1%B4_%EC%A3%BC" title="오리건 주">오리건 주</a> <a href="/wiki/%ED%8F%AC%ED%8B%80%EB%9E%9C%EB%93%9C_(%EC%98%A4%EB%A6%AC%EA%B1%B4_%EC%A3%BC)" title="포틀랜드 (오리건 주)">포틀랜드</a> 메모리얼 콜로세움을 마지막으로 끝났다.</p>
<p>10일 간의 휴식 후 여러 축제나 국제 공연을 가졌고, 다시 미국에 돌아와 <a class="mw-redirect" href="/wiki/%ED%94%84%EB%A1%9C%EC%A0%9D%ED%8A%B8_%EB%A0%88%EB%B3%BC%EB%A3%A8%EC%85%98#2007.EB.85.84_.ED.94.84.EB.A1.9C.EC.A0.9D.ED.8A.B8_.EB.A0.88.EB.B3.BC.EB.A3.A8.EC.85.98" title="프로젝트 레볼루션">프로젝트 레볼루션</a> 일정을 소화했다. 그렇지만, 밴드의 베이시스트 <a href="/wiki/%EB%A7%88%EC%9D%B4%ED%82%A4_%EC%9B%A8%EC%9D%B4" title="마이키 웨이">마이키 웨이</a>가 <a href="/wiki/2007%EB%85%84" title="2007년">2007년</a> <a href="/wiki/3%EC%9B%94_7%EC%9D%BC" title="3월 7일">3월 7일</a>에 결혼하여 밴드에서 잠시 하차해 맷 코르테즈가 대신 베이시스트로 참여하여 투어 일정을 소화하였다. 마이키 웨이는 프로젝트 레볼루션 투어 마지막 날에 한 곡을 연주하고 이후 <a href="/wiki/2007%EB%85%84" title="2007년">2007년</a> <a href="/wiki/10%EC%9B%94_4%EC%9D%BC" title="10월 4일">10월 4일</a>, 밴드에 재합류하였다.</p>
