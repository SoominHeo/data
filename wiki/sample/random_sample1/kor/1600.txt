<p><b>USS 노틸러스 (SSN-571)</b>(<span lang="en" xml:lang="en">USS Nautilus (SSN-571)</span>)는 <a href="/wiki/%EB%AF%B8%EA%B5%AD" title="미국">미국</a>이 개발한 세계 최초의 원자력 <a href="/wiki/%EC%9E%A0%EC%88%98%ED%95%A8" title="잠수함">잠수함</a>이며, 최초로 <a href="/wiki/%EB%B6%81%EA%B7%B9%EC%A0%90" title="북극점">북극점</a>을 잠수 횡단한 <a href="/wiki/%EC%9E%A0%EC%88%98%ED%95%A8" title="잠수함">잠수함</a>이기도 하다.</p>
<h2><span class="mw-headline" id=".EC.97.AD.EC.82.AC">역사</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=USS_%EB%85%B8%ED%8B%B8%EB%9F%AC%EC%8A%A4_(SSN-571)&amp;action=edit&amp;section=1" title="부분 편집: 역사">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>1951년 미국 의회는 <a class="new" href="/w/index.php?title=%ED%95%98%EC%9D%B4%EB%A8%BC_G._%EB%A6%AC%EC%BD%94%EB%B2%84&amp;action=edit&amp;redlink=1" title="하이먼 G. 리코버 (없는 문서)">하이먼 G. 리코버</a> 함장이 계획하고 관리감독한 핵추진 잠수함의 건조를 승인했다. <a class="new" href="/w/index.php?title=%ED%95%98%EC%9D%B4%EB%A8%BC_G._%EB%A6%AC%EC%BD%94%EB%B2%84&amp;action=edit&amp;redlink=1" title="하이먼 G. 리코버 (없는 문서)">하이먼 G. 리코버</a> 함장은 "원자력 해군의 아버지"(Father of the Nuclear Navy)라고 불린다. 이후에 4성장군인 제독(Admiral)까지 승진했다.</p>
<p>1951년 12월 12일 <a class="new" href="/w/index.php?title=%EB%AF%B8%EA%B5%AD_%ED%95%B4%EA%B5%B0%EC%84%B1&amp;action=edit&amp;redlink=1" title="미국 해군성 (없는 문서)">미국 해군성</a>은 최초의 핵잠수함 이름이 노틸러스라고 발표했다. 미국 해군 역사상 4번째 노틸러스함이다.</p>
<p>1952년 6월 14일 <a href="/wiki/%ED%95%B4%EB%A6%AC_S._%ED%8A%B8%EB%A3%A8%EB%A8%BC" title="해리 S. 트루먼">해리 S. 트루먼</a> 현직 대통령이 참석하여 <a href="/wiki/%EC%BD%94%EB%84%A4%ED%8B%B0%EC%BB%B7_%EC%A3%BC" title="코네티컷 주">코네티컷 주</a> <a class="new" href="/w/index.php?title=%EA%B7%B8%EB%A1%9C%ED%86%A4&amp;action=edit&amp;redlink=1" title="그로톤 (없는 문서)">그로톤</a>의 <a class="new" href="/w/index.php?title=%EC%A0%9C%EB%84%88%EB%9F%B4_%EB%8B%A4%EC%9D%B4%EB%82%B4%EB%AF%B9%EC%8A%A4_%EC%9D%BC%EB%A0%89%ED%8A%B8%EB%A6%AD_%EB%B3%B4%ED%8A%B8&amp;action=edit&amp;redlink=1" title="제너럴 다이내믹스 일렉트릭 보트 (없는 문서)">제너럴 다이내믹스 일렉트릭 보트</a>에서 기공식이 열렸다. 1954년 1월 21일 <a href="/wiki/%EB%93%9C%EC%99%80%EC%9D%B4%ED%8A%B8_D._%EC%95%84%EC%9D%B4%EC%A0%A0%ED%95%98%EC%9B%8C" title="드와이트 D. 아이젠하워">드와이트 D. 아이젠하워</a> 현직 대통령의 영부인 <a class="new" href="/w/index.php?title=%EB%A7%A4%EB%AF%B8_%EC%95%84%EC%9D%B4%EC%A0%A0%ED%95%98%EC%9B%8C&amp;action=edit&amp;redlink=1" title="매미 아이젠하워 (없는 문서)">매미 아이젠하워</a> 여사의 축복으로 준공식을 하였다. 1954년 9월 30일 취역했다. <a class="new" href="/w/index.php?title=%EC%9C%A0%EC%A7%84_P._%EC%9C%8C%ED%82%A8%EC%8A%A8&amp;action=edit&amp;redlink=1" title="유진 P. 윌킨슨 (없는 문서)">유진 P. 윌킨슨</a> 함장이 지휘했다.</p>
<p>잠수함 이름은 <a href="/wiki/%EC%A5%98_%EB%B2%A0%EB%A5%B8" title="쥘 베른">쥘 베른</a>의 소설 《<a href="/wiki/%ED%95%B4%EC%A0%80_2%EB%A7%8C%EB%A6%AC" title="해저 2만리">해저 2만리</a>》에 나오는 가상의 잠수정 '노틸러스 호'에서 딴 것이다.</p>
<p>1947년 12월 31일 <a href="/wiki/%EC%9B%A8%EC%8A%A4%ED%8C%85%ED%95%98%EC%9A%B0%EC%8A%A4" title="웨스팅하우스">웨스팅하우스</a>의 <a class="new" href="/w/index.php?title=%EB%B2%A0%ED%8B%B0%EC%8A%A4_%EC%9B%90%EC%9E%90%EB%A0%A5_%EC%97%B0%EA%B5%AC%EC%86%8C&amp;action=edit&amp;redlink=1" title="베티스 원자력 연구소 (없는 문서)">베티스 원자력 연구소</a>는 노틸러스를 위한 소형 경수로 개발을 시작했다. 열출력 70MWt의 가압형 경수로인 <a class="new" href="/w/index.php?title=S2W_%EC%9B%90%EC%9E%90%EB%A1%9C&amp;action=edit&amp;redlink=1" title="S2W 원자로 (없는 문서)">S2W 원자로</a>가 개발되었다. <a class="mw-redirect" href="/wiki/%EC%95%84%EC%9D%B4%EB%8B%A4%ED%98%B8" title="아이다호">아이다호</a>의 <a class="new" href="/w/index.php?title=%EA%B5%AD%EB%A6%BD_%EC%9B%90%EC%9E%90%EB%A1%9C_%EC%8B%A4%ED%97%98%EC%8B%A4&amp;action=edit&amp;redlink=1" title="국립 원자로 실험실 (없는 문서)">국립 원자로 실험실</a>의 일부인 <a href="/wiki/%EC%95%84%EB%A5%B4%EA%B3%A4_%EA%B5%AD%EB%A6%BD_%EC%97%B0%EA%B5%AC%EC%86%8C" title="아르곤 국립 연구소">아르곤 국립 연구소</a>에서, 1953년에 실제 프로토타입인 <a class="new" href="/w/index.php?title=S1W_%EC%9B%90%EC%9E%90%EB%A1%9C&amp;action=edit&amp;redlink=1" title="S1W 원자로 (없는 문서)">S1W 원자로</a>를 건설하여 테스트 했다.</p>
<p><a href="/wiki/1980%EB%85%84" title="1980년">1980년</a> <a href="/wiki/3%EC%9B%94_3%EC%9D%BC" title="3월 3일">3월 3일</a> 퇴역하였고 <a href="/wiki/1982%EB%85%84" title="1982년">1982년</a> <a href="/wiki/5%EC%9B%94_20%EC%9D%BC" title="5월 20일">5월 20일</a> 이후 기념물로 남아서 지금은 <a href="/wiki/%EC%BD%94%EB%84%A4%ED%8B%B0%EC%BB%B7_%EC%A3%BC" title="코네티컷 주">코네티컷 주</a>에 있는 해군 박물관에 전시 중이다.</p>
