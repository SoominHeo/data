<p><b>USS 존 C. 스테니스 (CVN-74)</b>는 미시시피의 정치가 <a href="/wiki/%EC%A1%B4_C._%EC%8A%A4%ED%85%8C%EB%8B%88%EC%8A%A4" title="존 C. 스테니스">존 C. 스테니스</a>의 이름에서 따온, <a href="/wiki/%EB%AF%B8%EA%B5%AD_%ED%95%B4%EA%B5%B0" title="미국 해군">미국 해군</a>의 <a href="/wiki/%EB%8B%88%EB%AF%B8%EC%B8%A0%EA%B8%89_%ED%95%AD%EA%B3%B5%EB%AA%A8%ED%95%A8" title="니미츠급 항공모함">니미츠급</a> <a href="/wiki/%EC%9B%90%EC%9E%90%EB%A0%A5" title="원자력">원자력</a> <a class="mw-redirect" href="/wiki/%ED%95%AD%EA%B3%B5_%EB%AA%A8%ED%95%A8" title="항공 모함">항공 모함</a>이다. 스테니스함은 1995년 12월 9일에 배치되었다. <a class="new" href="/w/index.php?title=%EB%AA%A8%ED%95%AD&amp;action=edit&amp;redlink=1" title="모항 (없는 문서)">모항</a>은 <a href="/wiki/%EC%9B%8C%EC%8B%B1%ED%84%B4_%EC%A3%BC" title="워싱턴 주">워싱턴 주</a>의 <a class="new" href="/w/index.php?title=%EB%B8%8C%EB%A0%88%EB%A8%B8%ED%84%B4&amp;action=edit&amp;redlink=1" title="브레머턴 (없는 문서)">브레머턴</a>이다. 다수의 미국 영화와 게임등에서 공격당하거나 반파, 대파되는 항공모함으로 나오는 이색적인 이력이 있다. (직접적인 항모의 이름은 언급되지 않는 경우가 많았으나 항모의 번호 '74'가 노출됨)</p>
<h2><span class="mw-headline" id=".EC.97.B0.ED.98.81">연혁</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=USS_%EC%A1%B4_C._%EC%8A%A4%ED%85%8C%EB%8B%88%EC%8A%A4_(CVN-74)&amp;action=edit&amp;section=1" title="부분 편집: 연혁">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<ul>
<li><a href="/wiki/1991%EB%85%84" title="1991년">1991년</a> <a href="/wiki/3%EC%9B%94_13%EC%9D%BC" title="3월 13일">3월 13일</a>, 뉴포트뉴스 조선소에서 기공되어, <a href="/wiki/1993%EB%85%84" title="1993년">1993년</a> <a href="/wiki/11%EC%9B%94_11%EC%9D%BC" title="11월 11일">11월 11일</a>에 마가렛 스테니스 워불에 의해서 명명, 진수되었다. <a href="/wiki/1995%EB%85%84" title="1995년">1995년</a> <a href="/wiki/12%EC%9B%94_9%EC%9D%BC" title="12월 9일">12월 9일</a>에 취역했다. 취역시의 모항은 <a href="/wiki/%EC%83%8C%EB%94%94%EC%97%90%EC%9D%B4%EA%B3%A0" title="샌디에이고">샌디에이고</a>였다.</li>
<li><a href="/wiki/1998%EB%85%84" title="1998년">1998년</a> <a href="/wiki/2%EC%9B%94_26%EC%9D%BC" title="2월 26일">2월 26일</a>에 <a href="/wiki/%EB%85%B8%ED%8D%BD_(%EB%B2%84%EC%A7%80%EB%8B%88%EC%95%84_%EC%A3%BC)" title="노퍽 (버지니아 주)">노퍽</a>을 출항한 존 C. 스테니스는, 평균 속도 29.4 노트로 274시간을 순항, 8,020 해리의 거리를 이동해 <a href="/wiki/3%EC%9B%94_11%EC%9D%BC" title="3월 11일">3월 11일</a>에 <a href="/wiki/%ED%8E%98%EB%A5%B4%EC%8B%9C%EC%95%84_%EB%A7%8C" title="페르시아 만">페르시아 만</a>에 도착했다. <a href="/wiki/%ED%8E%98%EB%A5%B4%EC%8B%9C%EC%95%84_%EB%A7%8C" title="페르시아 만">페르시아 만</a>에서는 <a href="/wiki/%EC%9D%B4%EB%9D%BC%ED%81%AC" title="이라크">이라크</a> 상공의 비행 금지 구역의 경계를 담당했다. <a href="/wiki/1998%EB%85%84" title="1998년">1998년</a> <a href="/wiki/7%EC%9B%94_19%EC%9D%BC" title="7월 19일">7월 19일</a>에 <a href="/wiki/%EC%98%A4%EC%8A%A4%ED%8A%B8%EB%A0%88%EC%9D%BC%EB%A6%AC%EC%95%84" title="오스트레일리아">오스트레일리아</a> 및 <a href="/wiki/%ED%95%98%EC%99%80%EC%9D%B4_%EC%A3%BC" title="하와이 주">하와이</a>로 향한다.</li>
<li><a href="/wiki/1998%EB%85%84" title="1998년">1998년</a> <a href="/wiki/8%EC%9B%94_26%EC%9D%BC" title="8월 26일">8월 26일</a>에 <a href="/wiki/%EC%83%8C%EB%94%94%EC%97%90%EC%9D%B4%EA%B3%A0" title="샌디에이고">샌디에이고</a>에 도착했다.</li>
<li><a href="/wiki/1999%EB%85%84" title="1999년">1999년</a> <a href="/wiki/11%EC%9B%94_30%EC%9D%BC" title="11월 30일">11월 30일</a>, <a href="/wiki/%EC%83%8C%EB%94%94%EC%97%90%EC%9D%B4%EA%B3%A0" title="샌디에이고">샌디에이고</a>의 노스 아일랜드 해군 항공 기지의 근처에서 좌초되었다. 침니가 원자로의 냉각 시스템의 순환 파이프를 막아, 2기의 원자로는 긴급정지했다. 스테니스는 부두로 견인되어 조사와 점검에 2일을 필요로 했다. 세정 비용은 약 200만 달러가 소비되었다.</li>
<li><a href="/wiki/2000%EB%85%84" title="2000년">2000년</a> <a href="/wiki/1%EC%9B%94_7%EC%9D%BC" title="1월 7일">1월 7일</a>, <a href="/wiki/%EC%83%8C%EB%94%94%EC%97%90%EC%9D%B4%EA%B3%A0" title="샌디에이고">샌디에이고</a>를 출항한 스테니스는 <a href="/wiki/%EC%9D%B4%EB%9D%BC%ED%81%AC" title="이라크">이라크</a>의 비행 금지 구역의 경계를 실시했고, 그 후 <a href="/wiki/%EB%8C%80%ED%95%9C%EB%AF%BC%EA%B5%AD" title="대한민국">대한민국</a>, <a href="/wiki/%ED%99%8D%EC%BD%A9" title="홍콩">홍콩</a>, <a href="/wiki/%EB%A7%90%EB%A0%88%EC%9D%B4%EC%8B%9C%EC%95%84" title="말레이시아">말레이시아</a>, <a href="/wiki/%EB%B0%94%EB%A0%88%EC%9D%B8" title="바레인">바레인</a>, <a href="/wiki/%EC%95%84%EB%9E%8D%EC%97%90%EB%AF%B8%EB%A6%AC%ED%8A%B8" title="아랍에미리트">아랍에미리트</a>, <a href="/wiki/%EC%98%A4%EC%8A%A4%ED%8A%B8%EB%A0%88%EC%9D%BC%EB%A6%AC%EC%95%84" title="오스트레일리아">오스트레일리아</a>를 방문했다. 스테니스는 <a href="/wiki/2000%EB%85%84" title="2000년">2000년</a> <a href="/wiki/7%EC%9B%94_3%EC%9D%BC" title="7월 3일">7월 3일</a>에 샌디에이고로 귀환했다.</li>
<li><a href="/wiki/2001%EB%85%84" title="2001년">2001년</a> <a href="/wiki/11%EC%9B%94_12%EC%9D%BC" title="11월 12일">11월 12일</a>에 <a href="/wiki/%ED%8E%98%EB%A5%B4%EC%8B%9C%EC%95%84_%EB%A7%8C" title="페르시아 만">페르시아 만</a>을 향해서 출항, <a href="/wiki/%EC%95%84%ED%94%84%EA%B0%80%EB%8B%88%EC%8A%A4%ED%83%84" title="아프가니스탄">아프가니스탄</a>에서의 작전 활동 지원하였고, 이후 이듬해 <a href="/wiki/2002%EB%85%84" title="2002년">2002년</a> <a href="/wiki/5%EC%9B%94_28%EC%9D%BC" title="5월 28일">5월 28일</a>에 귀환했다.</li>
<li><a href="/wiki/2004%EB%85%84" title="2004년">2004년</a>에는 <a href="/wiki/%EC%95%8C%EB%9E%98%EC%8A%A4%EC%B9%B4_%EB%A7%8C" title="알래스카 만">알래스카 만</a>에서의 <a class="new" href="/w/index.php?title=%EB%85%B8%EB%8D%98_%EC%97%A3%EC%A7%80&amp;action=edit&amp;redlink=1" title="노던 엣지 (없는 문서)">노던 엣지</a> 2004년 연습, <a href="/wiki/%ED%95%98%EC%99%80%EC%9D%B4_%EC%A0%9C%EB%8F%84" title="하와이 제도">하와이</a> 해역에서의 <a href="/wiki/%ED%99%98%ED%83%9C%ED%8F%89%EC%96%91_%ED%95%A9%EB%8F%99_%EC%97%B0%EC%8A%B5" title="환태평양 합동 연습">RIMPAC</a> 훈련에 참가한 후, <a href="/wiki/2004%EB%85%84" title="2004년">2004년</a> <a href="/wiki/8%EC%9B%94_21%EC%9D%BC" title="8월 21일">8월 21일</a>에 <a href="/wiki/%EB%82%98%EA%B0%80%EC%82%AC%ED%82%A4_%ED%98%84" title="나가사키 현">나가사키 현</a> 사세보 항에 기항했다.</li>
<li><a href="/wiki/2005%EB%85%84" title="2005년">2005년</a> <a href="/wiki/1%EC%9B%94_8%EC%9D%BC" title="1월 8일">1월 8일</a>에 모항을 <a href="/wiki/%EC%83%8C%EB%94%94%EC%97%90%EC%9D%B4%EA%B3%A0" title="샌디에이고">샌디에이고</a>로부터 <a class="new" href="/w/index.php?title=%EB%B8%8C%EB%A0%88%EB%A8%B8%ED%84%B4&amp;action=edit&amp;redlink=1" title="브레머턴 (없는 문서)">브레머턴</a>으로 변경한다. 다음날인 <a href="/wiki/2005%EB%85%84" title="2005년">2005년</a> <a href="/wiki/1%EC%9B%94_9%EC%9D%BC" title="1월 9일">1월 9일</a>에 도크에 들어가 광범위 오버홀을 개시, 11개월에 이르는 작업 중에는 신형 마스트가 장착되었다.</li>
<li><a href="/wiki/2007%EB%85%84" title="2007년">2007년</a> <a href="/wiki/1%EC%9B%94_20%EC%9D%BC" title="1월 20일">1월 20일</a>, 스테니스는 <a href="/wiki/%ED%8E%98%EB%A5%B4%EC%8B%9C%EC%95%84_%EB%A7%8C" title="페르시아 만">페르시아 만</a>을 향해서 출항해, <a href="/wiki/2007%EB%85%84" title="2007년">2007년</a> <a href="/wiki/2%EC%9B%94_19%EC%9D%BC" title="2월 19일">2월 19일</a>에 도착, 제5함대의 <a class="new" href="/w/index.php?title=USS_%EB%93%9C%EC%99%80%EC%9D%B4%ED%8A%B8_D._%EC%95%84%EC%9D%B4%EC%A0%A0%ED%95%98%EC%9B%8C_(CVN-69)&amp;action=edit&amp;redlink=1" title="USS 드와이트 D. 아이젠하워 (CVN-69) (없는 문서)">USS <i>드와이트 D. 아이젠하워</i> (CVN-69)</a> 항공모함과 합류하여 활동했다. <a href="/wiki/2003%EB%85%84" title="2003년">2003년</a> 이래 처음으로 2개의 <a href="/wiki/%ED%95%AD%EA%B3%B5%EB%AA%A8%ED%95%A8" title="항공모함">항공모함</a> 타격군이 동 해역에 존재한 것이 되었다.</li>
<li><a href="/wiki/2007%EB%85%84" title="2007년">2007년</a> <a href="/wiki/5%EC%9B%94_23%EC%9D%BC" title="5월 23일">5월 23일</a>에는 <a href="/wiki/USS_%EB%8B%88%EB%AF%B8%EC%B8%A0_(CVN-68)" title="USS 니미츠 (CVN-68)">USS <i>니미츠</i> (CVN-68)</a>, <a href="/wiki/USS_%EB%B3%B8%ED%97%98_%EB%A6%AC%EC%B2%98%EB%93%9C_(LHD-6)" title="USS 본험 리처드 (LHD-6)">USS <i>본험 리처드</i> (LHD-6)</a>를 포함한 8척의 함정과 함께 <a href="/wiki/%ED%98%B8%EB%A5%B4%EB%AC%B4%EC%A6%88_%ED%95%B4%ED%98%91" title="호르무즈 해협">호르무즈 해협</a>을 통과했다.</li>
<li><a href="/wiki/2009%EB%85%84" title="2009년">2009년</a> <a href="/wiki/2%EC%9B%94_28%EC%9D%BC" title="2월 28일">2월 28일</a>, 스테니스는 <a href="/wiki/%EC%A1%B0%EC%84%A0%EB%AF%BC%EC%A3%BC%EC%A3%BC%EC%9D%98%EC%9D%B8%EB%AF%BC%EA%B3%B5%ED%99%94%EA%B5%AD" title="조선민주주의인민공화국">조선민주주의인민공화국</a>이 <a href="/wiki/%EC%9D%B8%EA%B3%B5%EC%9C%84%EC%84%B1" title="인공위성">인공위성</a> 발사를 준비하고 있는 것에 대비해, 경계를 위해 서태평양 해역에 보조 함정과 함께 파견되었다. 작전 예정 기간은 미정이다.</li>
<li><a href="/wiki/2016%EB%85%84" title="2016년">2016년</a> <a href="/wiki/1%EC%9B%94_20%EC%9D%BC" title="1월 20일">1월 20일</a>, 스테니스는 <a href="/wiki/%EC%A1%B0%EC%84%A0%EB%AF%BC%EC%A3%BC%EC%A3%BC%EC%9D%98%EC%9D%B8%EB%AF%BC%EA%B3%B5%ED%99%94%EA%B5%AD" title="조선민주주의인민공화국">조선민주주의인민공화국</a>의 4차 핵실험에 대응해 한반도 지역으로 파견되었다.</li>
</ul>
