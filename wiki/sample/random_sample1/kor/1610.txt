<p><b>US 에어웨이스 그룹</b>(<span style="font-size: smaller;"><a href="/wiki/%EC%98%81%EC%96%B4" title="영어">영어</a>:</span> <span lang="en" xml:lang="en">US Airways Group</span>)은 <a href="/wiki/%EB%AF%B8%EA%B5%AD" title="미국">미국</a> <a href="/wiki/%EC%95%A0%EB%A6%AC%EC%A1%B0%EB%82%98_%EC%A3%BC" title="애리조나 주">애리조나 주</a> <a href="/wiki/%ED%83%AC%ED%8C%8C" title="탬파">탬파</a>에 본사를 둔 민간 항공 운송업체로 <a href="/wiki/US_%EC%97%90%EC%96%B4%EC%9B%A8%EC%9D%B4%EC%8A%A4" title="US 에어웨이스">US 에어웨이스</a>와 <a href="/wiki/US_%EC%97%90%EC%96%B4%EC%9B%A8%EC%9D%B4%EC%8A%A4_%EC%9D%B5%EC%8A%A4%ED%94%84%EB%A0%88%EC%8A%A4" title="US 에어웨이스 익스프레스">US 에어웨이스 익스프레스</a>, <a href="/wiki/US_%EC%97%90%EC%96%B4%EC%9B%A8%EC%9D%B4%EC%8A%A4_%EC%85%94%ED%8B%80" title="US 에어웨이스 셔틀">US 에어웨이스 셔틀</a>의 모기업에 속한다.</p>
<h2><span class="mw-headline" id=".EC.97.AD.EC.82.AC">역사</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=US_%EC%97%90%EC%96%B4%EC%9B%A8%EC%9D%B4%EC%8A%A4_%EA%B7%B8%EB%A3%B9&amp;action=edit&amp;section=1" title="부분 편집: 역사">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p><a href="/wiki/2004%EB%85%84" title="2004년">2004년</a> <a href="/wiki/9%EC%9B%94_12%EC%9D%BC" title="9월 12일">9월 12일</a>에 US 에어웨이스의 대부분의 자회사는 3년 간에 2번째의 파산 보호를 신청했다. US 에어웨이스 그룹과 <a href="/wiki/US_%EC%97%90%EC%96%B4%EC%9B%A8%EC%9D%B4%EC%8A%A4" title="US 에어웨이스">US 에어웨이스</a>를 포함한 상술의 자회사는 <a href="/wiki/2005%EB%85%84" title="2005년">2005년</a> <a href="/wiki/9%EC%9B%94_27%EC%9D%BC" title="9월 27일">9월 27일</a> 그 중의 완성에 관련해 에어 위스콘신 홀딩스의 자회사인 에어 위스콘신 항공부터 도움을 받아 1.25억 달러의 주식 투자와 제11장 파산 보호를 받았다. <a href="/wiki/2005%EB%85%84" title="2005년">2005년</a> <a href="/wiki/5%EC%9B%94_19%EC%9D%BC" title="5월 19일">5월 19일</a>에 <a class="new" href="/w/index.php?title=%EC%95%84%EB%A9%94%EB%A6%AC%EC%B9%B4_%EC%9B%A8%EC%8A%A4%ED%8A%B8_%ED%95%AD%EA%B3%B5&amp;action=edit&amp;redlink=1" title="아메리카 웨스트 항공 (없는 문서)">아메리카 웨스트 항공</a>과 합병하는 것을 발표했으며 <a href="/wiki/2005%EB%85%84" title="2005년">2005년</a> <a href="/wiki/9%EC%9B%94_27%EC%9D%BC" title="9월 27일">9월 27일</a>에 합병했다. <a href="/wiki/2006%EB%85%84" title="2006년">2006년</a> <a href="/wiki/10%EC%9B%94" title="10월">10월</a> 사업 확대를 집중하고 본사 건물을 임대했다. 같은 해 <a href="/wiki/12%EC%9B%94" title="12월">12월</a> <a href="/wiki/%EB%8D%B8%ED%83%80_%ED%95%AD%EA%B3%B5" title="델타 항공">델타 항공</a>과 합병안을 발표했으나 <a href="/wiki/%EB%8D%B8%ED%83%80_%ED%95%AD%EA%B3%B5" title="델타 항공">델타 항공</a>의 반발로 <a href="/wiki/2007%EB%85%84" title="2007년">2007년</a> <a href="/wiki/1%EC%9B%94_31%EC%9D%BC" title="1월 31일">1월 31일</a>에서 독립 조직 개편 계획을 지원하기 위해 채권자 위원회의 결정에 따라 합병 제안을 철회했다.<sup class="reference" id="cite_ref-1"><a href="#cite_note-1">[1]</a></sup> 최근 들어 <a href="/wiki/%EC%9C%A0%EB%9F%BD" title="유럽">유럽</a>발 금융 위기로 인해 경영난을 겪고 있는 <a href="/wiki/AMR_%EC%BD%94%ED%8D%BC%EB%A0%88%EC%9D%B4%EC%85%98" title="AMR 코퍼레이션">AMR 코퍼레이션</a>을 <a href="/wiki/%EB%8D%B8%ED%83%80_%ED%95%AD%EA%B3%B5" title="델타 항공">델타 항공</a>과 TPG 캐피탈과 함께 인수 및 합병하는 안을 검토했으며<sup class="reference" id="cite_ref-2"><a href="#cite_note-2">[2]</a></sup> <a href="/wiki/%EC%95%84%EB%A9%94%EB%A6%AC%EC%B9%B8_%ED%95%AD%EA%B3%B5" title="아메리칸 항공">아메리칸 항공</a>과 합병을 승인하면서 세계 최대의 항공사로 탄생할 계획이다.<sup class="reference" id="cite_ref-gogo_3-0"><a href="#cite_note-gogo-3">[3]</a></sup></p>
