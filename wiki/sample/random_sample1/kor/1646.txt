<h1 class="firstHeading" id="firstHeading" lang="ko">W (동음이의)</h1>
<div class="mw-body-content" id="bodyContent">
<div id="siteSub">위키백과, 우리 모두의 백과사전.</div>
<div id="contentSub"></div>
<div class="mw-jump" id="jump-to-nav">
					이동:					<a href="#mw-head">둘러보기</a>, 					<a href="#p-search">검색</a>
</div>
<div class="mw-content-ltr" dir="ltr" id="mw-content-text" lang="ko"><script>function mfTempOpenSection(id){var block=document.getElementById("mf-section-"+id);block.className+=" open-block";block.previousSibling.className+=" open-block";}</script><p><b>W</b> 또는 <b>w</b>는 다음과 같은 뜻이 있다.</p>
<ul>
<li><b><a href="/wiki/W" title="W">W</a></b>는 <a class="mw-redirect" href="/wiki/%EB%A1%9C%EB%A7%88_%EB%AC%B8%EC%9E%90" title="로마 문자">로마 문자</a>의 23번째 글자이다.</li>
<li><a href="/wiki/%EA%B5%AD%EC%A0%9C_%EC%9D%8C%EC%84%B1_%EA%B8%B0%ED%98%B8" title="국제 음성 기호">국제 음성 기호</a>에서 [w]는 <a href="/wiki/%EC%96%91%EC%88%9C_%EC%97%B0%EA%B5%AC%EA%B0%9C_%EC%A0%91%EA%B7%BC%EC%9D%8C" title="양순 연구개 접근음">양순 연구개 접근음</a>을 나타낸다.</li>
<li><a href="/wiki/%EB%8B%AC%EB%A0%A5" title="달력">달력</a>에서 <a href="/wiki/%EC%88%98%EC%9A%94%EC%9D%BC" title="수요일">수요일</a>(Wednesday)을 나타내는 약자로 쓰인다.</li>
<li>w나 w/는 영어 단어 "with"의 약자로 쓰인다.</li>
<li><a href="/wiki/%EC%A7%80%EB%8F%84" title="지도">지도</a>나 <a href="/wiki/%EB%82%98%EC%B9%A8%EB%B0%98" title="나침반">나침반</a>에서 서쪽(west)을 나타낸다.</li>
<li>w는 <a href="/wiki/%EC%9B%83%EC%9D%8C" title="웃음">웃음</a>이라는 뜻을 가진 <a href="/wiki/%EC%9D%BC%EB%B3%B8%EC%96%B4" title="일본어">일본어</a> 단어 ‘와라이’(<span lang="ja" xml:lang="ja">笑い</span>)를 <a href="/wiki/%EB%A1%9C%EB%A7%88%EC%9E%90" title="로마자">로마자</a>로 표기했을 때의 첫 글자로, <a href="/wiki/2%EC%B1%84%EB%84%90" title="2채널">2채널</a> 등에서 웃음 소리를 표현하기 위해 쓴다. <a href="/wiki/%EC%98%81%EC%96%B4" title="영어">영어</a>의 <a href="/wiki/LOL" title="LOL">LOL</a>이나 <a href="/wiki/%ED%95%9C%EA%B5%AD%EC%96%B4" title="한국어">한국어</a>의 <a href="/wiki/%E3%85%8B" title="ㅋ">ㅋ</a> 등과 맥락이 같다.</li>
<li><a href="/wiki/%EB%AF%B8%EA%B5%AD" title="미국">미국</a> 동부(주로 미시시피 강 동쪽 기준)에서는 방송국 <a class="mw-redirect" href="/wiki/%ED%98%B8%EC%B6%9C_%EB%B6%80%ED%98%B8" title="호출 부호">호출 부호</a>로 W로 시작한다.</li>
</ul>
<h3><span class="mw-headline" id=".EA.B3.BC.ED.95.99.EA.B3.BC_.EC.BB.B4.ED.93.A8.ED.84.B0">과학과 컴퓨터</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=W_(%EB%8F%99%EC%9D%8C%EC%9D%B4%EC%9D%98)&amp;action=edit&amp;section=1" title="부분 편집: 과학과 컴퓨터">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<ul>
<li><a class="mw-redirect" href="/wiki/SI_%EB%8B%A8%EC%9C%84%EA%B3%84" title="SI 단위계">SI 단위계</a>에서
<ul>
<li><a href="/wiki/%EC%99%80%ED%8A%B8" title="와트">W</a>는 <a href="/wiki/%EC%9D%BC%EB%A5%A0" title="일률">일률</a>의 <a class="mw-redirect" href="/wiki/SI_%EB%8B%A8%EC%9C%84" title="SI 단위">SI 단위</a>인 <a href="/wiki/%EC%99%80%ED%8A%B8" title="와트">와트</a>를 나타내는 기호이다.</li>
</ul>
</li>
<li><a href="/wiki/%EB%AC%BC%EB%A6%AC%ED%95%99" title="물리학">물리학</a>에서
<ul>
<li><i>W</i>는 <a href="/wiki/%EC%97%90%EB%84%88%EC%A7%80" title="에너지">에너지</a>나 <a class="mw-redirect" href="/wiki/%EC%9D%BC_(%EB%AC%BC%EB%A6%AC)" title="일 (물리)">일</a>을 나타내는 변수로 쓰인다.</li>
<li><a class="mw-redirect" href="/wiki/W_%EB%B3%B4%EC%A1%B4" title="W 보존">W 보존</a>을 나타낸다.</li>
</ul>
</li>
<li><a href="/wiki/%EC%9C%A0%EC%A0%84%ED%95%99" title="유전학">유전학</a>에서 W 염색체는 <a class="mw-redirect" href="/wiki/%EC%84%B1%EC%97%BC%EC%83%89%EC%B2%B4" title="성염색체">성염색체</a>의 일종이다. ZZ는 남성, WZ는 여성에 해당한다.</li>
<li><a href="/wiki/%ED%99%94%ED%95%99" title="화학">화학</a>에서 <a href="/wiki/%ED%85%85%EC%8A%A4%ED%85%90" title="텅스텐">텅스텐</a>의 <a class="new" href="/w/index.php?title=%EC%9B%90%EC%9E%90_%EA%B8%B0%ED%98%B8&amp;action=edit&amp;redlink=1" title="원자 기호 (없는 문서)">원자 기호</a>. 독일어 Wolfram에서 왔다.</li>
<li><a href="/wiki/W_(%EC%9C%A0%EB%8B%89%EC%8A%A4)" title="W (유닉스)">w</a>는 <a href="/wiki/%EC%9C%A0%EB%8B%89%EC%8A%A4" title="유닉스">유닉스</a>에서 시스템의 사용자와 작업 목록을 보여주는 명령어이다.</li>
</ul>
<h3><span class="mw-headline" id=".EB.AC.B8.ED.99.94">문화</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=W_(%EB%8F%99%EC%9D%8C%EC%9D%B4%EC%9D%98)&amp;action=edit&amp;section=2" title="부분 편집: 문화">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<ul>
<li>W는 국내 가수 <a class="new" href="/w/index.php?title=Where_The_Story_Ends&amp;action=edit&amp;redlink=1" title="Where The Story Ends (없는 문서)">Where The Story Ends</a>를 가리키는 약자로 쓰인다.</li>
<li><b><a href="/wiki/W_(%ED%9C%B4%EB%8C%80_%EC%A0%84%ED%99%94)" title="W (휴대 전화)">W</a></b>는 <a href="/wiki/SK%ED%85%94%EB%A0%88%EC%8B%9C%EC%8A%A4" title="SK텔레시스">SK텔레시스</a>의 <a href="/wiki/%ED%9C%B4%EB%8C%80_%EC%A0%84%ED%99%94" title="휴대 전화">휴대 전화</a> 브랜드이다.</li>
<li><b><a href="/wiki/%EA%B9%80%ED%98%9C%EC%88%98%EC%9D%98_W" title="김혜수의 W">김혜수의 W</a></b>는 <a href="/wiki/%EB%AC%B8%ED%99%94%EB%B0%A9%EC%86%A1" title="문화방송">문화방송</a>의 방송 프로그램이다.</li>
<li><b><a class="mw-redirect" href="/wiki/W_(%ED%97%AC%EB%A1%9C!_%ED%94%84%EB%A1%9C%EC%A0%9D%ED%8A%B8)" title="W (헬로! 프로젝트)">W</a></b>는 <a href="/wiki/%ED%97%AC%EB%A1%9C!_%ED%94%84%EB%A1%9C%EC%A0%9D%ED%8A%B8" title="헬로! 프로젝트">헬로! 프로젝트</a>에서 활동했던 여성 듀엣이다.</li>
<li><b><a class="mw-redirect" href="/wiki/%EB%8D%94%EB%B8%94%EC%9C%A0_%EC%95%A4_%EC%9B%A8%EC%9D%BC" title="더블유 앤 웨일">더블유 앤 웨일</a></b>(W &amp; Whale)는 대한민국의 음악 그룹이다.</li>
<li><b><a href="/wiki/W_(%EC%9E%A1%EC%A7%80)" title="W (잡지)">W (잡지)</a></b>는 두산매거진이 2005년 3월에 창간한 패션 문화 잡지다.</li>
<li>《<b><a href="/wiki/W_(%EB%93%9C%EB%9D%BC%EB%A7%88)" title="W (드라마)">W</a></b>》는 <a href="/wiki/2016%EB%85%84" title="2016년">2016년</a> <a href="/wiki/MBC_TV" title="MBC TV">MBC</a>의 <a href="/wiki/%EB%AC%B8%ED%99%94%EB%B0%A9%EC%86%A1_%EC%88%98%EB%AA%A9_%EB%AF%B8%EB%8B%88%EC%8B%9C%EB%A6%AC%EC%A6%88" title="문화방송 수목 미니시리즈">수목드라마</a>이다.</li>
</ul>
<h3><span class="mw-headline" id=".EA.B8.B0.ED.83.80">기타</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=W_(%EB%8F%99%EC%9D%8C%EC%9D%B4%EC%9D%98)&amp;action=edit&amp;section=3" title="부분 편집: 기타">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<ul>
<li><a href="/wiki/%EB%B6%80%EC%82%B0%EA%B4%91%EC%97%AD%EC%8B%9C" title="부산광역시">부산광역시</a> <a href="/wiki/%EB%82%A8%EA%B5%AC_(%EB%B6%80%EC%82%B0%EA%B4%91%EC%97%AD%EC%8B%9C)" title="남구 (부산광역시)">남구</a> <a href="/wiki/%EC%9A%A9%ED%98%B8%EB%8F%99_(%EB%B6%80%EC%82%B0)" title="용호동 (부산)">용호동</a>에 위치한 초고층 주상복합 아파트 <a href="/wiki/W_(%EC%9A%A9%ED%98%B8%EB%8F%99)" title="W (용호동)">W</a></li>
