<p><b>WebGL</b>은 <b><a href="/wiki/%EC%9B%94%EB%93%9C_%EC%99%80%EC%9D%B4%EB%93%9C_%EC%9B%B9" title="월드 와이드 웹">웹</a></b> 기반의 그래픽 라이브러리이다. <a href="/wiki/%EC%9E%90%EB%B0%94%EC%8A%A4%ED%81%AC%EB%A6%BD%ED%8A%B8" title="자바스크립트">자바스크립트</a> 프로그래밍 언어를 통해서 사용할 수 있으며 호환성이 있는 <a href="/wiki/%EC%9B%B9_%EB%B8%8C%EB%9D%BC%EC%9A%B0%EC%A0%80" title="웹 브라우저">웹 브라우저</a>에서 인터랙티브한 3D 그래픽을 사용할 수 있도록 제공된다.</p>
<p>WebGL은 <a href="/wiki/%ED%94%8C%EB%9F%AC%EA%B7%B8%EC%9D%B8" title="플러그인">플러그인</a>의 사용 없이 <a href="/wiki/3%EC%B0%A8%EC%9B%90_%EC%BB%B4%ED%93%A8%ED%84%B0_%EA%B7%B8%EB%9E%98%ED%94%BD%EC%8A%A4" title="3차원 컴퓨터 그래픽스">3차원 컴퓨터 그래픽스</a> <a href="/wiki/API" title="API">API</a>를 제공하는 <a class="new" href="/w/index.php?title=%EC%BA%94%EB%B2%84%EC%8A%A4_%EC%9A%94%EC%86%8C&amp;action=edit&amp;redlink=1" title="캔버스 요소 (없는 문서)">캔버스</a> <a href="/wiki/HTML_%EC%9A%94%EC%86%8C" title="HTML 요소">HTML 요소</a>의 일부분이다. 2011년 3월 3일에 세부 사양이 버전 1.0으로 출시되었다. WebGL은 <a href="/wiki/%EB%B9%84%EC%98%81%EB%A6%AC_%EB%8B%A8%EC%B2%B4" title="비영리 단체">비영리 단체</a>인 <a href="/wiki/%ED%81%AC%EB%A1%9C%EB%85%B8%EC%8A%A4_%EA%B7%B8%EB%A3%B9" title="크로노스 그룹">크로노스 그룹</a>에 의해서 관리되고 있다.</p>

<h2><span class="mw-headline" id=".EA.B0.9C.EC.9A.94">개요</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=WebGL&amp;action=edit&amp;section=1" title="부분 편집: 개요">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>WebGL은 OpenGL ES 2.0을 기반으로 하고 3차원 그래픽을 사용하기 위한 프로그래밍 인터페이스를 제공한다. WebGL은 <a href="/wiki/HTML5" title="HTML5">HTML5</a> <a class="new" href="/w/index.php?title=Canvas_element&amp;action=edit&amp;redlink=1" title="Canvas element (없는 문서)">캔버스</a> 요소를 사용하고 <a href="/wiki/%EB%AC%B8%EC%84%9C_%EA%B0%9D%EC%B2%B4_%EB%AA%A8%EB%8D%B8" title="문서 객체 모델">문서 객체 모델</a> 인터페이스를 사용해서 액세스할 수 있다. <a href="/wiki/%EC%9E%90%EB%B0%94%EC%8A%A4%ED%81%AC%EB%A6%BD%ED%8A%B8" title="자바스크립트">자바스크립트</a> 언어의 일부로서 자동 메모리 관리자가 제공된다.</p>
<h2><span class="mw-headline" id=".EA.B8.B0.EB.8A.A5_.EB.8F.84.EC.9E.85">기능 도입</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=WebGL&amp;action=edit&amp;section=2" title="부분 편집: 기능 도입">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>WebGL은 <a href="/wiki/%EA%B5%AC%EA%B8%80_%ED%81%AC%EB%A1%AC" title="구글 크롬">구글 크롬</a>, <a href="/wiki/%EC%9D%B8%ED%84%B0%EB%84%B7_%EC%9D%B5%EC%8A%A4%ED%94%8C%EB%A1%9C%EB%9F%AC_11" title="인터넷 익스플로러 11">인터넷 익스플로러 11</a>에서 실행되고 <a class="new" href="/w/index.php?title=%EB%AA%A8%EC%A7%88%EB%9D%BC_%ED%8C%8C%EC%9D%B4%EC%96%B4%ED%8F%AD%EC%8A%A4_4&amp;action=edit&amp;redlink=1" title="모질라 파이어폭스 4 (없는 문서)">모질라 파이어폭스 4</a>, <a class="mw-disambig" href="/wiki/%EC%82%AC%ED%8C%8C%EB%A6%AC" title="사파리">사파리</a>, <a href="/wiki/%EC%98%A4%ED%8E%98%EB%9D%BC" title="오페라">오페라</a>의 개발 버전(development release)에서 실행된다. 그뿐 아니라 WebGL은 <a href="/wiki/%EB%85%B8%ED%82%A4%EC%95%84_N900" title="노키아 N900">노키아 N900</a>의 PR1.2 펌웨어 업데이트에서도 가능하다.</p>
<h2><span class="mw-headline" id=".EA.B0.9C.EB.B0.9C">개발</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=WebGL&amp;action=edit&amp;section=3" title="부분 편집: 개발">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>WebGL은 <a href="/wiki/%EB%B9%84%EC%98%81%EB%A6%AC_%EB%8B%A8%EC%B2%B4" title="비영리 단체">비영리</a> <a href="/wiki/%EC%97%B0%ED%95%A9%EC%B2%B4" title="연합체">컨소시엄</a> 테크놀로지 <a href="/wiki/%ED%81%AC%EB%A1%9C%EB%85%B8%EC%8A%A4_%EA%B7%B8%EB%A3%B9" title="크로노스 그룹">크로노스 그룹</a>에 의해서 관리되고 있다. <a href="/wiki/%EC%95%A0%ED%94%8C_(%EA%B8%B0%EC%97%85)" title="애플 (기업)">애플</a>, <a href="/wiki/%EA%B5%AC%EA%B8%80" title="구글">구글</a>, <a href="/wiki/%EB%AA%A8%EC%A7%88%EB%9D%BC_%EC%9E%AC%EB%8B%A8" title="모질라 재단">모질라</a>, <a href="/wiki/%EC%98%A4%ED%8E%98%EB%9D%BC_%EC%86%8C%ED%94%84%ED%8A%B8%EC%9B%A8%EC%96%B4" title="오페라 소프트웨어">오페라</a>가 WebGL <a class="new" href="/w/index.php?title=%EC%9B%8C%ED%82%B9_%EA%B7%B8%EB%A3%B9&amp;action=edit&amp;redlink=1" title="워킹 그룹 (없는 문서)">워킹 그룹</a>에 포함되어 있다. 이 그룹의 의장은 <a class="extiw" href="https://en.wikipedia.org/wiki/Vladimir_Vuki%C4%87evi%C4%87" title="en:Vladimir Vukićević">Vladimir Vukićević</a>이다.</p>
<h2><span class="mw-headline" id=".EC.97.AD.EC.82.AC">역사</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=WebGL&amp;action=edit&amp;section=4" title="부분 편집: 역사">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>WebGL은 <a href="/wiki/%EB%AA%A8%EC%A7%88%EB%9D%BC_%EC%9E%AC%EB%8B%A8" title="모질라 재단">모질라</a> 소속 Vladimir Vukićević가 시작한 <a class="new" href="/w/index.php?title=%EC%BA%94%EB%B2%84%EC%8A%A4_3D&amp;action=edit&amp;redlink=1" title="캔버스 3D (없는 문서)">캔버스 3D</a> 연구에서 커져갔다. 2006년에 Vukićević가 처음으로 <a class="new" href="/w/index.php?title=%EC%BA%94%EB%B2%84%EC%8A%A4_3D&amp;action=edit&amp;redlink=1" title="캔버스 3D (없는 문서)">캔버스 3D</a> <a href="/wiki/%ED%94%84%EB%A1%9C%ED%86%A0%ED%83%80%EC%9E%85" title="프로토타입">프로토타입</a> 시연을 했다. 2007년 말까지, <a href="/wiki/%EB%AA%A8%EC%A7%88%EB%9D%BC_%EC%9E%AC%EB%8B%A8" title="모질라 재단">모질라</a>와 <a href="/wiki/%EC%98%A4%ED%8E%98%EB%9D%BC" title="오페라">오페라</a>가 각각 따로 구현했다. 2009년초, <a href="/wiki/%EB%AA%A8%EC%A7%88%EB%9D%BC_%EC%9E%AC%EB%8B%A8" title="모질라 재단">모질라</a>와 <a href="/wiki/%ED%81%AC%EB%A1%9C%EB%85%B8%EC%8A%A4" title="크로노스">크로노스</a>가 WebGL 워킹 그룹을 시작했다.</p>
<p>초기 유명한 애플리케이션으로는 <a class="extiw" href="https://en.wikipedia.org/wiki/Google_body_browser" title="en:Google body browser">구글 바디 브라우저</a>가 있다.</p>
<h2><span class="mw-headline" id=".EA.B0.95.EC.A2.8C">강좌</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=WebGL&amp;action=edit&amp;section=5" title="부분 편집: 강좌">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p><a class="external text" href="https://developer.mozilla.org/ko/docs/Web/WebGL" rel="nofollow">Mozilla Developer Network</a>와 <a class="external text" href="http://learningwebgl.com/lessons" rel="nofollow">Learning WebGL</a>에 WebGL에 대한 강좌가 있다.</p>
<h2><span class="mw-headline" id=".EA.B0.9C.EB.B0.9C.EC.9E.90_.EB.9D.BC.EC.9D.B4.EB.B8.8C.EB.9F.AC.EB.A6.AC">개발자 라이브러리</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=WebGL&amp;action=edit&amp;section=6" title="부분 편집: 개발자 라이브러리">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>WebGL 개발을 위한 몇몇 라이브러리가 있다. <a class="new" href="/w/index.php?title=WebGLU&amp;action=edit&amp;redlink=1" title="WebGLU (없는 문서)">WebGLU</a> 라이브러리는 처음으로 공개된 라이브러리이다<a class="external autonumber" href="http://bjartr.blogspot.com/2009/10/more-webgl-progress-now-with-video.html" rel="nofollow">[1]</a>. 그 외의 라이브러리로는 <a class="new" href="/w/index.php?title=GLGE&amp;action=edit&amp;redlink=1" title="GLGE (없는 문서)">GLGE</a>, <a class="new" href="/w/index.php?title=C3DL&amp;action=edit&amp;redlink=1" title="C3DL (없는 문서)">C3DL</a>, Copperlicht, SpiderGL, SceneJS, <a class="new" href="/w/index.php?title=Processing.js&amp;action=edit&amp;redlink=1" title="Processing.js (없는 문서)">Processing.js</a>, XB,bsWebGL<a class="external autonumber" href="http://projectbs.github.io/bsWebGL/" rel="nofollow">[2]</a> 포인트스트림이 있다.</p>
<h2><span class="mw-headline" id=".EA.B0.99.EC.9D.B4_.EB.B3.B4.EA.B8.B0">같이 보기</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=WebGL&amp;action=edit&amp;section=7" title="부분 편집: 같이 보기">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<ul>
<li><a href="/wiki/O3D" title="O3D">O3D</a> - 예전에 <a href="/wiki/%EA%B5%AC%EA%B8%80" title="구글">구글</a>이 제작한 플러그인 기반의 Web 3D이며 현재<a class="external autonumber" href="http://blog.chromium.org/2010/05/future-of-o3d.html" rel="nofollow">[3]</a> WebGL의 상위에서 사용되고 있다.</li>
<li><a href="/wiki/GLSL" title="GLSL">GLSL</a> - WebGL 상에서 작동되는 오픈 크로스 플랫폼 하이레벨 셰이딩 언어(엄밀히 따지면, WebGL은 GLSL-ES2의 방언이라고 할 수 있다.<a class="external autonumber" href="http://www.opengl.org/pipeline/article/vol004_2" rel="nofollow">[4]</a>)</li>
<li><a href="/wiki/VRML" title="VRML">VRML</a> - 가상현실 모델링 언어이며, 월드 와이드 웹과 함께 1994년에 고안된 3D 인터랙티브 벡터 그래픽을 대표하는 ISO 스탠다드 파일 포맷이다.</li>
<li><a class="new" href="/w/index.php?title=X3D&amp;action=edit&amp;redlink=1" title="X3D (없는 문서)">X3D</a> - 2004년에 릴리즈된 VRML을 계승하는 가상현실 모델링 언어.</li>
<li><a class="new" href="/w/index.php?title=%EC%9E%90%EB%B0%94_OpenGL&amp;action=edit&amp;redlink=1" title="자바 OpenGL (없는 문서)">자바 OpenGL</a> - 자바에서 <a href="/wiki/OpenGL" title="OpenGL">OpenGL</a>을 사용할 수 있는 래퍼 라이브러리.</li>
</ul>
