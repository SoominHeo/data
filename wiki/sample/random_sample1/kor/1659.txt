<p><b>Wii U</b>는 <a href="/wiki/%EB%8B%8C%ED%85%90%EB%8F%84" title="닌텐도">닌텐도</a>의 가정용 게임기로 8세대 게임기에 속하며 <a href="/wiki/Wii" title="Wii">Wii</a>의 후속작이다. <a href="/wiki/2012%EB%85%84" title="2012년">2012년</a> <a href="/wiki/11%EC%9B%94_18%EC%9D%BC" title="11월 18일">11월 18일</a> <a href="/wiki/%EB%B6%81%EC%95%84%EB%A9%94%EB%A6%AC%EC%B9%B4" title="북아메리카">북아메리카</a>에서 먼저 출시되었으며, <a href="/wiki/%EC%9D%BC%EB%B3%B8" title="일본">일본</a>에서는 <a href="/wiki/2012%EB%85%84" title="2012년">2012년</a> <a href="/wiki/12%EC%9B%94_8%EC%9D%BC" title="12월 8일">12월 8일</a>에 출시되었다. <a href="/wiki/%EB%8C%80%ED%95%9C%EB%AF%BC%EA%B5%AD" title="대한민국">대한민국</a>과 <a href="/wiki/%EC%A4%91%ED%99%94%EB%AF%BC%EA%B5%AD" title="중화민국">중화민국</a>에는 발매가 되지 않았다. 이전 기종인 <a href="/wiki/Wii" title="Wii">Wii</a>에 비해 판매량이 대폭 줄어들어 처참한 판매량을 보여 생산이 중단되었다고 알려져 있었지만 닌텐도 측에서 생산 중단이 아니라고 밝혔다고 한다. <sup class="reference" id="cite_ref-3"><a href="#cite_note-3">[3]</a></sup></p>

<h2><span class="mw-headline" id=".EC.83.81.EC.84.B8_.EC.8A.A4.ED.8E.99">상세 스펙</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Wii_U&amp;action=edit&amp;section=1" title="부분 편집: 상세 스펙">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>Wii U는 <a href="/wiki/%EC%9C%A0%ED%8A%9C%EB%B8%8C" title="유튜브">유튜브</a>, <a href="/wiki/%EB%84%B7%ED%94%8C%EB%A6%AD%EC%8A%A4" title="넷플릭스">넷플릭스</a>, 아마존 인스턴트 비디오 그리고 훌루 플러스를 <a href="/wiki/%EB%8B%8C%ED%85%90%EB%8F%84_eShop" title="닌텐도 eShop">닌텐도 eShop</a>에서 무료로 다운받아서 시청할 수있다. 닌텐도 TVii역시 시청이 가능하고 Wii U채팅으로 화상 통화를 할 수있다. TV 컨트롤 버튼이 새로 나왔으며 컨트롤 ZL/ZR버튼, 왼쪽 (조이)스틱/오른쪽 (조이)스틱도 추가 됐다. 대응 피규어인 <a class="new" href="/w/index.php?title=%EC%95%84%EB%AF%B8%EB%B3%B4&amp;action=edit&amp;redlink=1" title="아미보 (없는 문서)">아미보</a>로 <a class="mw-disambig" href="/wiki/NFC" title="NFC">NFC</a>통신이 가능하며, 터치 스크린이 6.2인치이고 16:9비율의 <a class="mw-disambig" href="/wiki/LCD" title="LCD">LCD</a>를 지니고 있다. 노래방 기능도 있지만, 전용 마이크가 있어야 한다. 그 전용 마이크에는 2가지가 있는데, 하나는 Wii U본체에 있는 <a href="/wiki/USB" title="USB">USB</a>에 연결하는 마이크와 다른 하나는 무선마이크가 있다. Wii U 게임패드의 무게는 약 500g정도이다.</p>
