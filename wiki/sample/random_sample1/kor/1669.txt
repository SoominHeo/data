<h1 class="firstHeading" id="firstHeading" lang="ko">XScale</h1>
<div class="mw-body-content" id="bodyContent">
<div id="siteSub">위키백과, 우리 모두의 백과사전.</div>

					이동:					<a href="#mw-head">둘러보기</a>, 					<a href="#p-search">검색</a>
</div>
<div class="mw-content-ltr" dir="ltr" id="mw-content-text" lang="ko"><script>function mfTempOpenSection(id){var block=document.getElementById("mf-section-"+id);block.className+=" open-block";block.previousSibling.className+=" open-block";}</script><p><a href="/wiki/%EC%9D%B8%ED%85%94" title="인텔">인텔</a>의 <b>XScale</b>(엑스스케일) <a href="/wiki/%EC%A4%91%EC%95%99_%EC%B2%98%EB%A6%AC_%EC%9E%A5%EC%B9%98" title="중앙 처리 장치">프로세서</a>는 <a class="mw-disambig" href="/wiki/ARM" title="ARM">ARMv</a>5 아키텍처를 기반으로 설계된 <a href="/wiki/%EC%9D%B8%ED%85%94" title="인텔">인텔</a>의 모바일 프로세서 아키텍처로 핸드헬드 PC나 PDA에 폭넓게 사용되고 있는 모바일 프로세서 아키텍처이다. StrongARM CPU는 <a class="mw-disambig" href="/wiki/DEC" title="DEC">DEC</a>사에서 개발되었고 인텔이 인수하면서 인텔의 상표를 달고 출고되었다. XScale은 이 StrongARM 프로세서의 후속작이다.</p>
<p>2006년 인텔이 XScale 관련 라이선스를 <a href="/wiki/%EB%A7%88%EB%B2%A8_%ED%85%8C%ED%81%AC%EB%86%80%EB%A1%9C%EC%A7%80_%EA%B7%B8%EB%A3%B9" title="마벨 테크놀로지 그룹">마벨</a>사에 매각함에 따라 현재 XScale 기반 프로세서는 <a class="mw-disambig" href="/wiki/%EB%A7%88%EB%B2%A8" title="마벨">마벨</a>(Marvell)사에서 제조되고 있다.</p>

<h2><span class="mw-headline" id="XScale_.EA.B8.B0.EB.B0.98_.ED.94.84.EB.A1.9C.EC.84.B8.EC.84.9C_.EB.82.98.EC.97.B4">XScale 기반 프로세서 나열</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=XScale&amp;action=edit&amp;section=1" title="부분 편집: XScale 기반 프로세서 나열">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<h3><span class="mw-headline" id="PXA21x.2F25x_.EA.B3.84.EC.97.B4">PXA21x/25x 계열</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=XScale&amp;action=edit&amp;section=2" title="부분 편집: PXA21x/25x 계열">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>2002년 2월 출시 제품이고, PXA21x는 저전압, 중전압 두 가지 동작 전압 모드를 지원하며 25x의 경우 고전압 모드를 추가로 지원한다.</p>
<h4><span class="mw-headline" id=".EC.A0.80.EC.A0.84.EC.95.95_.EB.AA.A8.EB.93.9C.280.85V.29">저전압 모드(0.85V)</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=XScale&amp;action=edit&amp;section=3" title="부분 편집: 저전압 모드(0.85V)">편집</a><span class="mw-editsection-bracket">]</span></span></h4>
<ul>
<li>일반모드 100MHz, 터보모드 132MHz</li>
<li>50 ~ 99.5MHz PXBus Clock Speed</li>
<li>50 ~ 99.5MHz <a href="/wiki/SDRAM" title="SDRAM">SDRAM</a> Controller Clock Speed (99.5MHz)</li>
<li>0.765 ~ 0.935V 동작</li>
</ul>
<h4><span class="mw-headline" id=".EC.A4.91.EC.A0.84.EC.95.95_.EB.AA.A8.EB.93.9C.281.00V.29">중전압 모드(1.00V)</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=XScale&amp;action=edit&amp;section=4" title="부분 편집: 중전압 모드(1.00V)">편집</a><span class="mw-editsection-bracket">]</span></span></h4>
<ul>
<li>일반모드 100MHz, 터보모드 200MHz</li>
<li>50 ~ 99.5MHz PXBus Clock Speed</li>
<li>50 ~ 99.5MHz SDRAM Controller Clock Speed (99.5MHz)</li>
<li>0.90 ~ 1.10V 동작</li>
</ul>
<h4><span class="mw-headline" id=".EA.B3.A0.EC.A0.84.EC.95.95_.EB.AA.A8.EB.93.9C.281.10V.29">고전압 모드(1.10V)</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=XScale&amp;action=edit&amp;section=5" title="부분 편집: 고전압 모드(1.10V)">편집</a><span class="mw-editsection-bracket">]</span></span></h4>
<ul>
<li>일반모드 200MHz, 터보모드 300MHz</li>
<li>50 ~ 99.5MHz PXBus Clock Speed</li>
<li>50 ~ 99.5MHz SDRAM Memory Clock Speed (99.5MHz)</li>
<li>0.99 ~ 1.21V 동작</li>
<li>특이사항 : PXA25x 에서만 지원하며 <b>PXA255는 터보모드 동작 속도가 400MHz</b>.</li>
</ul>
<h4><span class="mw-headline" id=".EA.B3.B5.ED.86.B5.2F.ED.8A.B9.EC.9D.B4_.EC.82.AC.ED.95.AD">공통/특이 사항</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=XScale&amp;action=edit&amp;section=6" title="부분 편집: 공통/특이 사항">편집</a><span class="mw-editsection-bracket">]</span></span></h4>
<ul>
<li>32KB <a class="new" href="/w/index.php?title=I-Cache&amp;action=edit&amp;redlink=1" title="I-Cache (없는 문서)">I-Cache</a>, 32KB <a class="new" href="/w/index.php?title=D-Cache&amp;action=edit&amp;redlink=1" title="D-Cache (없는 문서)">D-Cache</a>, 2KB Mini D-Cache</li>
<li><a class="mw-redirect" href="/wiki/AC97" title="AC97">AC97</a>/<a class="new" href="/w/index.php?title=I2S&amp;action=edit&amp;redlink=1" title="I2S (없는 문서)">I2S</a> 오디오 지원</li>
<li><a href="/wiki/USB" title="USB">USB</a>-Client 지원</li>
<li>고속 <a href="/wiki/UART" title="UART">UART</a> 포트</li>
<li>흐름제어 가능 2번 UART 포트</li>
<li>FIR, SIR 적외선 포트</li>
</ul>
<h3><span class="mw-headline" id="PXA26x_.EC.8B.9C.EB.A6.AC.EC.A6.88">PXA26x 시리즈</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=XScale&amp;action=edit&amp;section=7" title="부분 편집: PXA26x 시리즈">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>2003년 3월 출시했고, 재원은 PXA255과 같다,</p>
<h4><span class="mw-headline" id=".EA.B3.B5.ED.86.B5.2F.ED.8A.B9.EC.9D.B4_.EC.82.AC.ED.95.AD_2">공통/특이 사항</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=XScale&amp;action=edit&amp;section=8" title="부분 편집: 공통/특이 사항">편집</a><span class="mw-editsection-bracket">]</span></span></h4>
<ul>
<li>Intel(R) <a class="new" href="/w/index.php?title=StrataFlash&amp;action=edit&amp;redlink=1" title="StrataFlash (없는 문서)">StrataFlash</a> 기술 지원</li>
</ul>
<h3><span class="mw-headline" id="PXA27x_.EC.8B.9C.EB.A6.AC.EC.A6.88">PXA27x 시리즈</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=XScale&amp;action=edit&amp;section=9" title="부분 편집: PXA27x 시리즈">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>2004년 4월 출시했고, 개발 코드명 Bulverde. 이 시점부터 PXA 프로세서의 멀티미디어 관련 성능이 대폭 향상됐으며 향상된 전력 관리 기술을 지원하기 시작한다.</p>
<h4><span class="mw-headline" id=".EC.A7.80.EC.9B.90_.EB.8F.99.EC.9E.91_.EC.86.8D.EB.8F.84">지원 동작 속도</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=XScale&amp;action=edit&amp;section=10" title="부분 편집: 지원 동작 속도">편집</a><span class="mw-editsection-bracket">]</span></span></h4>
<ul>
<li>절전모드 208MHz, 일반모드 520MHz (일반적으로 사용되는 모드)</li>
<li>최저 104MHz ~ 최고 624MHz까지 동작 가능</li>
<li>104MHz PXBus Clock Speed</li>
<li>104MHz SDRAM Memory Clock Speed</li>
<li>2.25 ~ 3.75V 동작</li>
</ul>
<h4><span class="mw-headline" id=".EA.B3.B5.ED.86.B5.2F.ED.8A.B9.EC.9D.B4_.EC.82.AC.ED.95.AD_3">공통/특이 사항</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=XScale&amp;action=edit&amp;section=11" title="부분 편집: 공통/특이 사항">편집</a><span class="mw-editsection-bracket">]</span></span></h4>
<ul>
<li>32KB I-Cache, 32KB D-Cache, 2KB Mini D-Cache</li>
<li>Intel(R) Wireless <a href="/wiki/MMX" title="MMX">MMX</a> Technology</li>
<li>Wireless Intel <a class="new" href="/w/index.php?title=SpeedStep&amp;action=edit&amp;redlink=1" title="SpeedStep (없는 문서)">SpeedStep</a> Technology (소소한 결함이 있는 관계로 잘 사용되지 않음)</li>
<li>AC97/I2S 오디오 지원</li>
<li>USB-Client/Host 지원</li>

<h3><span class="mw-headline" id="PXA3xx_.EC.8B.9C.EB.A6.AC.EC.A6.88">PXA3xx 시리즈</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=XScale&amp;action=edit&amp;section=12" title="부분 편집: PXA3xx 시리즈">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>2005년 8월에 출시되었다. 개발 코드명은 Monahans이다. PXA270의 성공을 바탕으로 제작된 프로세서로 XScale 기반 프로세서중 처음으로 1GHz 이상의 설계 클럭을 가진다.</p>
<p>이 제품은 인텔이 XScale 코어 라이선스를 Marvell사에 매각함에 따라 마벨이 생산하기 시작한다.</p>
<h4><span class="mw-headline" id=".EC.A7.80.EC.9B.90_.EB.8F.99.EC.9E.91_.EC.86.8D.EB.8F.84_2">지원 동작 속도</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=XScale&amp;action=edit&amp;section=13" title="부분 편집: 지원 동작 속도">편집</a><span class="mw-editsection-bracket">]</span></span></h4>
<ul>
<li>걸리는 부하에 따라 208 ~ 624MHz 사이의 가변 클럭 (일반적으로 사용되는 모드)</li>
<li>최저 104MHz ~ 최고 806MHz까지 동작 가능</li>
<li>설계 가능한 최고 클럭은 1.25GHz</li>
<li>208MHz Bus Clock Speed</li>
</ul>
<h4><span class="mw-headline" id=".EA.B3.B5.ED.86.B5.2F.ED.8A.B9.EC.9D.B4_.EC.82.AC.ED.95.AD_4">공통/특이 사항</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=XScale&amp;action=edit&amp;section=14" title="부분 편집: 공통/특이 사항">편집</a><span class="mw-editsection-bracket">]</span></span></h4>
<ul>
<li>32KB I-Cache, 32KB D-Cache, 2KB Mini D-Cache</li>
<li>256KB 캐시용 SRAM 내장</li>
<li>x16 DDR SDRAM 지원</li>
<li>Intel(R) Wireless <a href="/wiki/MMX" title="MMX">MMX</a>2 <a class="new" href="/w/index.php?title=Co-Processor&amp;action=edit&amp;redlink=1" title="Co-Processor (없는 문서)">Co-Processor</a></li>
<li>Wireless Intel <a class="new" href="/w/index.php?title=SpeedStep&amp;action=edit&amp;redlink=1" title="SpeedStep (없는 문서)">SpeedStep</a> Technology</li>
<li>하드웨어 수준의 멀티미디어 영상(JPEG)/비디오 가속 지원</li>
<li>Intel(R) <a class="new" href="/w/index.php?title=QuickCapture&amp;action=edit&amp;redlink=1" title="QuickCapture (없는 문서)">QuickCapture</a> 카메라 가속 인터페이스 지원</li>
<li>USB 2.0 Client 지원</li>
<li>2채널 USB Full-Speed(12MBps) Host 지원</li>
<li>4채널 SSP 인터페이스 지원</li>
<li><a href="/wiki/OpenGL_ES" title="OpenGL ES">OpenGL ES</a> 1.1 3D 가속 지원</li>
<li>Monahans 프로세서의 최고 실현가능 클럭은 1.25GHz이나 806MHz 이상부터는 전력 소비 효율이 급격히 저하되므로 잘 사용되지 않는다. (624MHz@800<a class="mw-disambig" href="/wiki/MIPS" title="MIPS">MIPS</a>, 1.25GHz@1000MIPS)</li>
</ul>
