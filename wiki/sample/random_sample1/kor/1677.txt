<p><b>YMCK</b>는 일본의 <a href="/wiki/%EC%B9%A9%ED%8A%A0" title="칩튠">칩튠</a> 밴드이다. <a class="mw-disambig" href="/wiki/%EC%8A%A4%EC%9C%99" title="스윙">스윙</a>이나 <a href="/wiki/%EC%9E%AC%EC%A6%88" title="재즈">재즈</a>의 느낌이 나는 <a href="/wiki/%ED%85%8C%ED%81%AC%EB%85%B8" title="테크노">테크노</a> 사운드를 쓰며 <a class="mw-redirect" href="/wiki/%ED%8C%A8%EB%AF%B8%EC%BB%B4" title="패미컴">패미컴</a>의 <a class="mw-redirect" href="/wiki/%EA%B2%8C%EC%9E%84_%EC%9D%8C%EC%95%85" title="게임 음악">게임 음악</a>과 같은 8bit 음원과 보컬의 달콤한 목소리가 특징이다. 앨범 CD의 재킷에도 패미컴의 게임 캐릭터와 같은 도트 그림을 사용한다.</p>
<p><a href="/wiki/2003%EB%85%84" title="2003년">2003년</a> 5월에 <a href="/wiki/CD-R" title="CD-R">CD-R</a>로 직접 만든 앨범 《패밀리 뮤직》(<span lang="ja" xml:lang="ja">ファミリーミュージック</span>)으로 데뷔하고 이 앨범이 히트를 하여 <a href="/wiki/2004%EB%85%84" title="2004년">2004년</a> 11월에 우사기짱레코드(<span lang="ja" xml:lang="ja">ウサギチャンレコーズ</span>)에서 첫 번째 앨범인 《패밀리 뮤직》(<span lang="ja" xml:lang="ja">ファミリーミュージック</span>)<sup class="reference" id="cite_ref-1"><a href="#cite_note-1">[1]</a></sup> 을 발매한다. <a href="/wiki/2005%EB%85%84" title="2005년">2005년</a> 12월에 두 번째 앨범인 《패밀리 레이싱》(<span lang="ja" xml:lang="ja">ファミリーレーシング</span>)을 발매한다. <a href="/wiki/2006%EB%85%84" title="2006년">2006년</a> 6월에는 한국의 <a class="mw-redirect" href="/wiki/%ED%8C%8C%EC%8A%A4%ED%85%94_%EB%AE%A4%EC%A7%81" title="파스텔 뮤직">파스텔 뮤직</a>에서 《패밀리 뮤직》과 《패밀리 레이싱》을 정식 발매하였고 같은 달 정식 발매 기념 라이브 콘서트를 한국에서 개최했다. <a href="/wiki/2008%EB%85%84" title="2008년">2008년</a> 1월에는 <a class="mw-redirect" href="/wiki/Avex_trax" title="Avex trax">avex trax</a>에서 세 번째 앨범인 《패밀리 제네시스》(<span lang="ja" xml:lang="ja">ファミリージェネシス</span>)를 발매하여 메이저 데뷔를 하였다.</p>
