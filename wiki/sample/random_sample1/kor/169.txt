(<a href="/wiki/A*_%EC%95%8C%EA%B3%A0%EB%A6%AC%EC%A6%98" title="A* 알고리즘">A* 알고리즘</a>, <a href="/wiki/C*_%EB%8C%80%EC%88%98" title="C* 대수">C* 대수</a>)</p>

<h2><span class="mw-headline" id=".EC.9A.A9.EB.8F.84">용도</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%B3%84%ED%91%9C&amp;action=edit&amp;section=1" title="부분 편집: 용도">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<h3><span class="mw-headline" id=".EA.B8.B0.EB.A1.9D_.EB.AC.B8.EC.9E.90">기록 문자</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%B3%84%ED%91%9C&amp;action=edit&amp;section=2" title="부분 편집: 기록 문자">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<ul>
<li>별표는 <a class="new" href="/w/index.php?title=%EA%BC%AC%EB%A6%AC%EB%A7%90&amp;action=edit&amp;redlink=1" title="꼬리말 (없는 문서)">꼬리말</a>(주석)을 나타내는 데에 쓰인다. (특히 페이지에 오직 하나만 있을 때) 여러 개의 별표가 다른 꼬릿말을 페이지에 나타내는 데 쓰이기도 하지만 그리 자주 쓰이는 것은 아니다. (이를테면 *, **, ***)</li>
<li>페이지 위의 가운데에 있는 세 개의 공백 별표는 다른 개념이나 장면으로 넘어간다는 것을 뜻한다.</li>
<li>하나 이상의 별표는 낱말의 일부를 가리는 것을 뜻한다. 이를테면 s**t, Peter J****, G*d 등을 들 수 있다.</li>
<li>별표는 <a class="new" href="/w/index.php?title=%EB%B6%88%EB%A0%9B_(%EA%B8%B0%ED%98%B8)&amp;action=edit&amp;redlink=1" title="불렛 (기호) (없는 문서)">타이포그래피 불렛</a> 대신 사용하여 목록의 항목을 표시한다.</li>
<li>회화체에서 별표는 <a href="/wiki/%EC%9D%B4%ED%83%A4%EB%A6%AD%EC%B2%B4" title="이탤릭체">이탤릭체</a>를 사용하지 못할 경우 *강조*하는 데에 사용된다.</li>
<li>별표는 영화, 식당의 등급을 표시하는 데 사용된다.</li>
</ul>
<h3><span class="mw-headline" id=".EC.BB.B4.ED.93.A8.ED.84.B0">컴퓨터</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%B3%84%ED%91%9C&amp;action=edit&amp;section=3" title="부분 편집: 컴퓨터">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<h4><span class="mw-headline" id=".EC.A0.84.EC.82.B0.ED.95.99">전산학</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%B3%84%ED%91%9C&amp;action=edit&amp;section=4" title="부분 편집: 전산학">편집</a><span class="mw-editsection-bracket">]</span></span></h4>
<p><a class="mw-redirect" href="/wiki/%EC%A0%84%EC%82%B0%ED%95%99" title="전산학">전산학</a>에서 별표는 <a href="/wiki/%EC%A0%95%EA%B7%9C_%ED%91%9C%ED%98%84%EC%8B%9D" title="정규 표현식">정규 표현식</a>에서 쓰이며, 하나 이상의 반복 패턴이 있음을 나타내거나 0을 나타낸다.</p>
<p><a class="new" href="/w/index.php?title=%ED%86%B5%EC%9D%BC_%EB%AA%A8%EB%8D%B8%EB%A7%81_%EC%96%B8%EC%96%B4&amp;action=edit&amp;redlink=1" title="통일 모델링 언어 (없는 문서)">통일 모델링 언어</a>에서 별표는 수많은 클래스에 0을 나타내는 데 사용된다.</p>
<h4><span class="mw-headline" id=".ED.94.84.EB.A1.9C.EA.B7.B8.EB.9E.98.EB.B0.8D_.EC.96.B8.EC.96.B4">프로그래밍 언어</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%B3%84%ED%91%9C&amp;action=edit&amp;section=5" title="부분 편집: 프로그래밍 언어">편집</a><span class="mw-editsection-bracket">]</span></span></h4>
<p>수많은 <a href="/wiki/%ED%94%84%EB%A1%9C%EA%B7%B8%EB%9E%98%EB%B0%8D_%EC%96%B8%EC%96%B4" title="프로그래밍 언어">프로그래밍 언어</a>와 <a href="/wiki/%EA%B3%84%EC%82%B0%EA%B8%B0" title="계산기">계산기</a>는 애스터리스크를 <a href="/wiki/%EA%B3%B1%EC%85%88" title="곱셈">곱셈</a>에 대한 기호로 사용한다. 이는 또한 다음과 같은 수많은 특별한 뜻을 가질 수 있다. 이를테면:</p>
<ul>
<li><a href="/wiki/C_(%ED%94%84%EB%A1%9C%EA%B7%B8%EB%9E%98%EB%B0%8D_%EC%96%B8%EC%96%B4)" title="C (프로그래밍 언어)">C 프로그래밍 언어</a>와 같은 일부 프로그래밍 언어에서, 별표는 포인터 변수를 선언하거나 선언을 해제하는 데 쓰인다. 또한, 프로그래밍의 예제로 <a href="/wiki/%EB%B3%84_%EC%B0%8D%EA%B8%B0" title="별 찍기">별 찍기</a>를 사용하여 초보 프로그래머들의 이해를 돕는다.</li>
<li><a href="/wiki/%EC%BB%A4%EB%A8%BC_%EB%A6%AC%EC%8A%A4%ED%94%84" title="커먼 리스프">커먼 리스프</a> 프로그래밍 언어에서, <a href="/wiki/%EC%A0%84%EC%97%AD_%EB%B3%80%EC%88%98" title="전역 변수">전역 변수</a>의 이름은 기본적으로 별표로 설정을 해제할 수 있다.: <tt>*이와 같이*</tt>.</li>
<li><a href="/wiki/%ED%8F%AC%ED%8A%B8%EB%9E%80" title="포트란">포트란</a> 프로그래밍 언어와 <a class="mw-redirect" href="/wiki/%ED%8C%8C%EC%8A%A4%EC%B9%BC_%ED%94%84%EB%A1%9C%EA%B7%B8%EB%9E%98%EB%B0%8D_%EC%96%B8%EC%96%B4" title="파스칼 프로그래밍 언어">파스칼 프로그래밍 언어</a>의 일부 계열에서 두 개의 별표는 <a class="mw-redirect" href="/wiki/%EC%A7%80%EC%88%98" title="지수">지수</a>를 나타낼 때 사용된다: 5**3는 5*5*5이니, 곧 125이다.</li>
<li><a href="/wiki/%ED%8E%84" title="펄">펄 프로그래밍 언어</a>에서 별표는 주어진 이름의 모든 변수의 typeglob를 나타내는 데 사용된다.</li>
<li>프로그래밍 언어인 <a class="mw-redirect" href="/wiki/%EB%A3%A8%EB%B9%84_%ED%94%84%EB%A1%9C%EA%B7%B8%EB%9E%98%EB%B0%8D_%EC%96%B8%EC%96%B4" title="루비 프로그래밍 언어">루비</a>와 <a href="/wiki/%ED%8C%8C%EC%9D%B4%EC%8D%AC" title="파이썬">파이썬</a>에서, * 는 두 가지 용도로 사용된다. 첫 용도로, 함수 안의 리스트 객체에 적용된 단일 * 연산자는 함수 호출 변수로 리스트를 확장할 것이다. 두 번째로, 함수에 대한 변수 목록 안에서 * 를 전제로 한 변수는 <a href="/wiki/%ED%8A%9C%ED%94%8C" title="튜플">튜플</a> (파이썬)이나 <a href="/wiki/%EB%B0%B0%EC%97%B4" title="배열">배열</a> (루비)로 모이는 특별한 변수의 결과를 나타낸다.</li>
</ul>
