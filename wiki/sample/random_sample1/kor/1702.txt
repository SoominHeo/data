<h1 class="firstHeading" id="firstHeading" lang="ko">ㅈ</h1>
<div class="mw-body-content" id="bodyContent">
<div id="siteSub">위키백과, 우리 모두의 백과사전.</div>
<div id="contentSub"></div>
<div class="mw-jump" id="jump-to-nav">
					이동:					<a href="#mw-head">둘러보기</a>, 					<a href="#p-search">검색</a>
</div>
<div class="mw-content-ltr" dir="ltr" id="mw-content-text" lang="ko"><script>function mfTempOpenSection(id){var block=document.getElementById("mf-section-"+id);block.className+=" open-block";block.previousSibling.className+=" open-block";}</script><p><b>ㅈ</b>(지읒)은 한글 닿소리 중 아홉째 글자이다.</p>
<p>'ㅈ'은 현대 한국어의 <a class="new" href="/w/index.php?title=%EC%B9%98%EA%B2%BD%EA%B5%AC%EA%B0%9C_%ED%8C%8C%EC%B0%B0%EC%9D%8C&amp;action=edit&amp;redlink=1" title="치경구개 파찰음 (없는 문서)">치경구개 파찰음</a>(남) 혹은 <a class="new" href="/w/index.php?title=%EC%B9%98%EC%A1%B0_%ED%8C%8C%EC%B0%B0%EC%9D%8C&amp;action=edit&amp;redlink=1" title="치조 파찰음 (없는 문서)">치조 파찰음</a>(북)이다. 낱말 처음에 올 경우 무성<span class="IPA" title="IPA 발음">[ t͡ɕ⁽ʰ⁾ ]</span>(남) 혹은 <span class="IPA" title="IPA 발음">[ t͡s⁽ʰ⁾ ]</span>(북), 홀소리와 홀소리 사이 혹은 유성 자음 앞뒤에서는 유성<span class="IPA" title="IPA 발음">[ d͡ʑ ]</span>(남) 혹은 <span class="IPA" title="IPA 발음">[ d͡z ]</span>(북)이 되며, 종성에서 쓰이는 ㅈ은 소릿값이 종성 ㄷ과 같은 <span class="IPA" title="IPA 발음">[ t ̚ ]</span>이다.</p>
<p><a href="/wiki/%ED%9B%88%EB%AF%BC%EC%A0%95%EC%9D%8C" title="훈민정음">훈민정음</a>에 따르면 ㅈ이 나타내는 소리는 <a class="mw-redirect" href="/wiki/%EC%9E%87%EC%86%8C%EB%A6%AC" title="잇소리">잇소리</a>이며, 그 제자 원리는 이의 모양을 본뜬 ㅅ에 획을 더하여 만들었다고 한다. 초성으로는 <a class="mw-redirect" href="/wiki/%EC%A0%84%EC%B2%AD" title="전청">전청</a>, 종성으로는 <a href="/wiki/%EC%9E%85%EC%84%B1" title="입성">입성</a>이 된다. 종성 8자 제한 규정에는 들어 있지 않은데, ㅈ은 ㅊ·<a href="/wiki/%E3%85%BF" title="ㅿ">ㅿ</a> 등과 더불어 휴지 앞이나 폐쇄음 앞에서는 ㅅ으로 통용된다고 하였다. 같은 잇소리 계열 ㅊ과 함께 15세기 한국어와 현대 <a href="/wiki/%ED%91%9C%EC%A4%80%EC%96%B4" title="표준어">표준어</a>의 ㅈ 소릿값은 달랐다.</p>

<td>

어두 : <span class="IPA" title="IPA 발음">[ t͡s⁽ʰ⁾ ]</span>, 어중 : <span class="IPA" title="IPA 발음">[ d͡z ]</span>, 어말 : <span class="IPA" title="IPA 발음">[ t ̚ ]</span><span style="font-size:smaller;"><b>（문화어）</b></span></p>

후행 자음이 ㅎ일 경우 <span class="IPA" title="IPA 발음">[ t͡ɕʰ ]</span>(표준어), <span class="IPA" title="IPA 발음">[ t͡sʰ ]</span>(문화어)로 소리가 난다.</p>

<h2><span class="mw-headline" id=".EA.B5.AC.EA.B0.9C.EC.9D.8C.ED.99.94">구개음화</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%E3%85%88&amp;action=edit&amp;section=1" title="부분 편집: 구개음화">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>현대 <a href="/wiki/%ED%91%9C%EC%A4%80%EC%96%B4" title="표준어">표준어</a>에서 ㅈ은 <a href="/wiki/%E3%85%8A" title="ㅊ">ㅊ</a>과 같이 <a class="mw-redirect" href="/wiki/%EC%9E%85%EC%B2%9C%EC%9E%A5%EC%86%8C%EB%A6%AC" title="입천장소리">입천장소리</a> 또는 <a href="/wiki/%ED%98%93%EB%B0%94%EB%8B%A5%EC%86%8C%EB%A6%AC" title="혓바닥소리">혓바닥소리</a>인데 이는 잇소리에 [j] 반모음이 연결되어 있음을 뜻한다.(<a href="/wiki/%EA%B5%AC%EA%B0%9C%EC%9D%8C%ED%99%94" title="구개음화">구개음화</a>) 그래서 현대 한국어의 잇소리가 센입천장와 혀 앞쪽에서 나기 때문에 혀가 [j]와 비슷해져 ㅈ·ㅊ과 <a href="/wiki/%ED%99%80%EC%86%8C%EB%A6%AC" title="홀소리">홀소리</a> 사이에서 나는 [j]가 잘 드러나지 않는다. 따라서 ㅈ·ㅊ 다음에 ㅑ, ㅒ, ㅕ, ㅖ, ㅛ, ㅠ가 오면 ㅏ, ㅐ, ㅓ, ㅔ, ㅗ, ㅜ로 중화한다.</p>
<ul>
<li>졌다 → [<b>젇</b>따]</li>
<li>하죠 → [하<b>조</b>]</li>
<li>자다 → [<b>자</b>다]</li>
</ul>
<p>하지만 <a href="/wiki/%EB%AC%B8%ED%99%94%EC%96%B4" title="문화어">문화어</a>에서는 ㅈ, ㅊ이 <a class="mw-redirect" href="/wiki/%EC%9E%85%EC%B2%9C%EC%9E%A5%EC%86%8C%EB%A6%AC" title="입천장소리">입천장소리</a> 또는 <a href="/wiki/%ED%98%93%EB%B0%94%EB%8B%A5%EC%86%8C%EB%A6%AC" title="혓바닥소리">혓바닥소리</a>가 아니라 <a class="mw-redirect" href="/wiki/%EC%B9%98%EA%B2%BD" title="치경">치경</a>에서 조음되는 <a href="/wiki/%ED%8C%8C%EC%B0%B0%EC%9D%8C" title="파찰음">파찰음</a>이기 때문에 잇소리에 [j] 반모음이 연결되는 구개음화가 일어나지 않는다. 따라서 문화어에서는 ㅈ·ㅊ 다음에 ㅑ, ㅒ, ㅕ, ㅖ, ㅛ, ㅠ가 올 경우와 ㅏ, ㅐ, ㅓ, ㅔ, ㅗ, ㅜ가 올 경우의 소리가 다르다.</p>
<ul>
<li>졌다 → [<b>젿</b>따]</li>
<li>하죠 → [하<b>죠</b>]</li>
<li>자다 → [<b>자</b>다]</li>
</ul>
