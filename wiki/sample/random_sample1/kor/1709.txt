<h2><span class="mw-headline" id=".EA.B0.9C.EC.9A.94">개요</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EA%B0%80%EA%B2%BD%EC%9E%90&amp;action=edit&amp;section=1" title="부분 편집: 개요">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>가경자에 해당하는 영어 단어 'Venerable'은 옛날 유럽에서 주교와 대수도원장 등에게 붙였던 경칭이었으며, 사후 이 칭호로 받은 첫 번째 사람은 6세기 경의 수도승 <a href="/wiki/%EB%B2%A0%EB%8B%A4_%EB%B2%A0%EB%84%A4%EB%9D%BC%EB%B9%8C%EB%A6%AC%EC%8A%A4" title="베다 베네라빌리스">성 베다</a>(혹은 성 비드, 가경자 비드, the Venerable Bede, Bede the Venerable)로 알려져 있다. 그의 지혜와 학문을 높이 인정한다는 의미에서 ‘존엄한 자’(Venerable)라는 칭호가 주어졌으며, 이 칭호는 <a href="/wiki/853%EB%85%84" title="853년">853년</a> 아헨(Aachen) 교회 회의에서 공식화되었다. 이후 이 명칭은 영적인 공덕을 지니고 있는 성직자나 평신도 등에게 붙여지게 되었다.</p>
<p><a href="/wiki/%EC%95%84%EC%9A%B0%EA%B5%AC%EC%8A%A4%ED%8B%B0%EB%88%84%EC%8A%A4" title="아우구스티누스">성 아우구스티누스</a>는 몇몇 편지에서 주교를 지칭할 때 이 용어를 사용하였다고 한다.</p>
<p>로마 가톨릭교회에서는 신앙의 모범이 된 사람이 죽은 후에, 성인으로 모시는 세단계 과정 중에 첫 번째 과정에서 이 명칭을 사용한다. <a class="mw-disambig" href="/wiki/%EC%98%81%EA%B5%AD_%EC%84%B1%EA%B3%B5%ED%9A%8C" title="영국 성공회">영국 성공회</a>에서 가경자는 대부제의 정식 호칭이다.</p>
<h2><span class="mw-headline" id=".EB.A1.9C.EB.A7.88_.EA.B0.80.ED.86.A8.EB.A6.AD.EA.B5.90.ED.9A.8C.EC.9D.98_.EA.B0.80.EA.B2.BD.EC.9E.90">로마 가톨릭교회의 가경자</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EA%B0%80%EA%B2%BD%EC%9E%90&amp;action=edit&amp;section=2" title="부분 편집: 로마 가톨릭교회의 가경자">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>로마 가톨릭에서는 신앙의 모범이 되었다고 인정되는 사람에 대하여 시복 및 시성 절차를 거친다. 첫 단계인 시복 조사가 <a href="/wiki/%EB%A1%9C%EB%A7%88_%EA%B5%90%ED%99%A9%EC%B2%AD" title="로마 교황청">교황청</a>의 시성시복성성(諡聖諡福聖省, 혹은 예부성성(禮部聖省))에 접수되면 시복 후보자에게 이 존칭이 주어진다. 가경자는 엄격한 절차를 거쳐 복자가 되며, <a href="/wiki/%EB%B3%B5%EC%9E%90" title="복자">복자</a>는 시성(諡聖) 과정을 거쳐 <a href="/wiki/%EC%84%B1%EC%9D%B8_(%EC%A2%85%EA%B5%90)" title="성인 (종교)">성인</a>으로 불리게 된다. 시성시복성성으로의 접수 외에, 교황이 그 후보자가 영웅적인 덕을 가졌거나 순교를 당했다고 주장하면서 특별 칙령으로 천거하는 경우에도 가경자 칭호를 부여한다.</p>
<p>가경자의 수준에 대하여 <a href="/wiki/1913%EB%85%84" title="1913년">1913년</a> <a href="/wiki/%EA%B5%90%ED%99%A9" title="교황">교황</a> <a class="mw-redirect" href="/wiki/%EB%B9%84%EC%98%A4_10%EC%84%B8" title="비오 10세">비오 10세</a>는 공식적인 교령으로, “놀라운 정도의 덕행을 실천하거나, 순교한 사람이라야 가경자의 칭호를 받을 만하다”고 밝힌 바 있다.</p>

<h4><span class="mw-headline" id=".ED.95.9C.EA.B5.AD">한국</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EA%B0%80%EA%B2%BD%EC%9E%90&amp;action=edit&amp;section=4" title="부분 편집: 한국">편집</a><span class="mw-editsection-bracket">]</span></span></h4>
<p>한국 출신의 가경자가 탄생한 것은, <a href="/wiki/1857%EB%85%84" title="1857년">1857년</a>의 일이다. 조선의 가톨릭 교회가 〈<a href="/wiki/1839%EB%85%84" title="1839년">1839년</a>과 <a href="/wiki/1846%EB%85%84" title="1846년">1846년</a>에 조선에서 발발한 <a href="/wiki/%EB%B0%95%ED%95%B4" title="박해">박해</a> 중에 <a class="new" href="/w/index.php?title=%EB%B0%B0%EA%B5%90&amp;action=edit&amp;redlink=1" title="배교 (없는 문서)">배교</a>를 거부하고 생명을 바친 <a href="/wiki/%EC%88%9C%EA%B5%90" title="순교">순교자</a>들의 전기〉란 문헌을 <a href="/wiki/%EB%A1%9C%EB%A7%88_%EA%B5%90%ED%99%A9%EC%B2%AD" title="로마 교황청">교황청</a>에 보냄으로써 시복 조사가 시작되었다고 한다. 이 문헌에는 <a href="/wiki/1839%EB%85%84" title="1839년">1839년</a>의 <a href="/wiki/%EA%B8%B0%ED%95%B4%EB%B0%95%ED%95%B4" title="기해박해">기해박해</a>와 <a href="/wiki/1846%EB%85%84" title="1846년">1846년</a>의 <a href="/wiki/%EB%B3%91%EC%98%A4%EB%B0%95%ED%95%B4" title="병오박해">병오박해</a> 때 희생된 82명에 대하여 기록되어 있었다. <a href="/wiki/1847%EB%85%84" title="1847년">1847년</a>에 이 문헌을 접수한 교황청 시성시복성성(시복과 시성을 담당하는 교황청의 한 부서)은, 박해로 인해 한국 가톨릭교회가 <a href="/wiki/%EA%B5%90%EA%B5%AC_(%EA%B8%B0%EB%8F%85%EA%B5%90)" title="교구 (기독교)">교구</a>적 차원의 시복 조사를 할 수 없으나, 문헌 자체가 순교자를 선정하는 데 매우 엄격한 기준을 적용시킨 것으로 판단하여, <a href="/wiki/%EA%B5%90%ED%9A%8C%EB%B2%95" title="교회법">교회법</a>에서 요구되는 교구 조사를 대치시킬 수 있다고 판단하고, <a href="/wiki/1857%EB%85%84" title="1857년">1857년</a> <a href="/wiki/9%EC%9B%94_23%EC%9D%BC" title="9월 23일">9월 23일</a> 한국 가톨릭교회의 시복 조사를 공식으로 접수하는 법령을 반포하였으며, 이로써 82명의 가경자가 탄생하였다. 이 82명의 가경자 중 79명은 <a href="/wiki/1925%EB%85%84" title="1925년">1925년</a>에 <a href="/wiki/%EB%B3%B5%EC%9E%90" title="복자">복자</a>가 되었고, <a href="/wiki/1984%EB%85%84" title="1984년">1984년</a>에 한국에서 개최된 한국 가톨릭 200주년 기념행사 때 교황 요한 바오로 2세에 의해 <a href="/wiki/%EC%84%B1%EC%9D%B8_(%EC%A2%85%EA%B5%90)" title="성인 (종교)">시성</a>되었다.</p>
<p>이어 한국 가톨릭교회는 <a href="/wiki/1866%EB%85%84" title="1866년">1866년</a> <a href="/wiki/%EB%B3%91%EC%9D%B8%EB%B0%95%ED%95%B4" title="병인박해">병인박해</a>의 순교자 중 26명에 대한 시복 조사를 시작하여, 이 가운데 24명이 <a href="/wiki/1968%EB%85%84" title="1968년">1968년</a>에 시복되고, 앞의 82명과 더불어 <a href="/wiki/1984%EB%85%84" title="1984년">1984년</a>에 같이 <a href="/wiki/%EC%84%B1%EC%9D%B8_(%EC%A2%85%EA%B5%90)" title="성인 (종교)">성인</a>이 되었다. 이들 또한 복자가 되기 전에 잠시 가경자의 칭호를 받았을 것으로 예상된다. 한편 가장 최근 2016년 5월 8일 한국인으로서 2번째 사제인 <a href="/wiki/%EC%B5%9C%EC%96%91%EC%97%85" title="최양업">최양업</a> 토마스 사제가 교황청으로부터 공식 인준을 받아 가경자로 선포되었다.<sup class="reference" id="cite_ref-1"><a href="#cite_note-1">[1]</a></sup></p>
<h2><span class="mw-headline" id=".EB.8F.99.EB.B0.A9_.EC.A0.95.EA.B5.90.ED.9A.8C.2C_.EC.84.B1.EA.B3.B5.ED.9A.8C.EC.9D.98_.EA.B0.80.EA.B2.BD.EC.9E.90">동방 정교회, 성공회의 가경자</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EA%B0%80%EA%B2%BD%EC%9E%90&amp;action=edit&amp;section=5" title="부분 편집: 동방 정교회, 성공회의 가경자">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p><a href="/wiki/%EB%8F%99%EB%B0%A9_%EC%A0%95%EA%B5%90%ED%9A%8C" title="동방 정교회">동방 정교회</a>에서는 가경자에 해당하는 'Venerable' (<span style="font-size: smaller;"><a href="/wiki/%EA%B7%B8%EB%A6%AC%EC%8A%A4%EC%96%B4" title="그리스어">그리스어</a>:</span> <span lang="el" xml:lang="el">남성형 Όσιος, 여성형 Οσία</span>))은, 수도자나 은둔자 출신의 <a class="mw-disambig" href="/wiki/%EC%84%B1%EC%9D%B8" title="성인">성인</a>에게 붙이는 칭호이며, 경우에 따라서는 보통의 성인보다 상위 개념으로 생각하는 경우도 있다. <a href="/wiki/%EC%84%B1%EA%B3%B5%ED%9A%8C" title="성공회">성공회</a>에서는 <a href="/wiki/%EB%A1%9C%EB%A7%88_%EA%B0%80%ED%86%A8%EB%A6%AD%EA%B5%90%ED%9A%8C" title="로마 가톨릭교회">로마 가톨릭</a>이나 <a href="/wiki/%EB%8F%99%EB%B0%A9_%EC%A0%95%EA%B5%90%ED%9A%8C" title="동방 정교회">동방 정교회</a>와는 달리 가경자라는 칭호가 없으나, 대신 <a href="/wiki/%EC%A2%85%EA%B5%90_%EA%B0%9C%ED%98%81" title="종교 개혁">종교개혁</a>이전의 <a class="mw-disambig" href="/wiki/%EC%98%81%EA%B5%AD_%EC%84%B1%EA%B3%B5%ED%9A%8C" title="영국 성공회">영국 성공회</a>에서 기념한 <a href="/wiki/%EC%84%B1%EC%9D%B8_(%EC%A2%85%EA%B5%90)" title="성인 (종교)">성인</a>들과 <a href="/wiki/%EC%A1%B4_%EC%9B%A8%EC%8A%AC%EB%A6%AC" title="존 웨슬리">존 웨슬리</a>나 <a href="/wiki/%EB%94%94%ED%8A%B8%EB%A6%AC%ED%9E%88_%EB%B3%B8%ED%9A%8C%ED%8D%BC" title="디트리히 본회퍼">디트리히 본회퍼</a>처럼 신앙의 본이 된다고 여겨지는 기독교 인물들을 <a href="/wiki/%EA%B5%90%ED%9A%8C" title="교회">교회</a>의 수호<a class="mw-disambig" href="/wiki/%EC%84%B1%EC%9D%B8" title="성인">성인</a>등으로 기념한다.</p>
