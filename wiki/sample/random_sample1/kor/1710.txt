<p><b>가고시마 성</b> (<span style="font-size: smaller;"><a href="/wiki/%EC%9D%BC%EB%B3%B8%EC%96%B4" title="일본어">일본어</a>:</span> <span lang="ja" xml:lang="ja">鹿児島城</span> <small>가고시마조<sup>[<a href="/wiki/%EC%9C%84%ED%82%A4%EB%B0%B1%EA%B3%BC:%EC%9D%BC%EB%B3%B8%EC%96%B4%EC%9D%98_%ED%95%9C%EA%B8%80_%ED%91%9C%EA%B8%B0" title="위키백과:일본어의 한글 표기">*</a>]</sup></small>) 은 <a href="/wiki/%EA%B0%80%EA%B3%A0%EC%8B%9C%EB%A7%88_%ED%98%84" title="가고시마 현">가고시마 현</a> <a href="/wiki/%EA%B0%80%EA%B3%A0%EC%8B%9C%EB%A7%88_%EC%8B%9C" title="가고시마 시">가고시마 시</a> 시로야마 산에 있는 평산성이다. 다른 이름으론 <b>쓰루마루 성(鶴丸城)</b>이다. 가고시마 현 지정 사적이다.</p>

<h2><span class="mw-headline" id=".EA.B0.9C.EC.9A.94">개요</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EA%B0%80%EA%B3%A0%EC%8B%9C%EB%A7%88_%EC%84%B1&amp;action=edit&amp;section=1" title="부분 편집: 개요">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p><a href="/wiki/%EC%97%90%EB%8F%84_%EC%8B%9C%EB%8C%80" title="에도 시대">에도 시대</a>로 들어설 무렵인 <a href="/wiki/1601%EB%85%84" title="1601년">1601년</a> <a href="/wiki/%EC%8B%9C%EB%A7%88%EC%A6%88_%EB%8B%A4%EB%8B%A4%EC%93%B0%EB%84%A4" title="시마즈 다다쓰네">시마즈 다다쓰네</a>의해 축성되었다. <a href="/wiki/%EC%84%B8%ED%82%A4%EA%B0%80%ED%95%98%EB%9D%BC_%EC%A0%84%ED%88%AC" title="세키가하라 전투">세키가하라 전투</a>에서 <a class="mw-redirect" href="/wiki/%EC%8B%9C%EB%A7%88%EC%A6%88_%EA%B0%80%EB%AC%B8" title="시마즈 가문">시마즈 가문</a>은 서군에 속해 패배했다. 그 후, 다다쓰네는 시마즈 가문의 수장으로서 <a class="new" href="/w/index.php?title=%EC%9A%B0%EC%B9%98_%EC%84%B1&amp;action=edit&amp;redlink=1" title="우치 성 (없는 문서)">우치 성</a>을 대신할 쓰루마루 성 축성을 시작하여 <a href="/wiki/1604%EB%85%84" title="1604년">1604년</a> 완성하였다.</p>
<p>다다쓰네의 친아버지인 <a href="/wiki/%EC%8B%9C%EB%A7%88%EC%A6%88_%EC%9A%94%EC%8B%9C%ED%9E%88%EB%A1%9C" title="시마즈 요시히로">시마즈 요시히로</a>는 성이 해안에 가깝기 때문에 방어에 취약함을 지적하고 끝까지 반대했었다. <a href="/wiki/%EB%8F%84%EC%BF%A0%EA%B0%80%EC%99%80_%EC%9D%B4%EC%97%90%EC%95%BC%EC%8A%A4" title="도쿠가와 이에야스">도쿠가와 이에야스</a>의 사쓰마 정벌은 실행되지 않았고, 시마즈 가문은 <a href="/wiki/%EC%82%AC%EC%93%B0%EB%A7%88_%EB%B2%88" title="사쓰마 번">사쓰마 번</a>의 <a href="/wiki/%EB%8F%84%EC%9E%90%EB%A7%88_%EB%8B%A4%EC%9D%B4%EB%AC%98" title="도자마 다이묘">도자마 다이묘</a>로 존속할 수 있었다. 그러나, 쓰루마루 성은 요시히로의 염려대로 막부 말기에 일어난 <a href="/wiki/%EC%82%AC%EC%93%B0%EC%97%90%EC%9D%B4_%EC%A0%84%EC%9F%81" title="사쓰에이 전쟁">사쓰에이 전쟁</a> 때 <a href="/wiki/%EC%98%81%EA%B5%AD" title="영국">영국</a> 전함으로부터 포격을 받게 된다.</p>
<p>가고시마는 재해가 많은 지역이었고, 남쪽에 위치한 지리적 이유 때문에 <a href="/wiki/%ED%9D%B0%EA%B0%9C%EB%AF%B8" title="흰개미">흰개미</a>의 피해도 컸다. 몇 번의 소실과 파괴로 그때마다 재건되었다. <a href="/wiki/1874%EB%85%84" title="1874년">1874년</a> (<a href="/wiki/%EB%A9%94%EC%9D%B4%EC%A7%80_%EC%8B%9C%EB%8C%80" title="메이지 시대">메이지</a> 7년) 성이 소실된 후로는 재건되지 않았다. 오테 문과 돌다리는 현존해 있고, 귀중한 자산이다.</p>
<h2><span class="mw-headline" id=".EA.B5.AC.EC.A1.B0">구조</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EA%B0%80%EA%B3%A0%EC%8B%9C%EB%A7%88_%EC%84%B1&amp;action=edit&amp;section=2" title="부분 편집: 구조">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>시마즈 가문은 중세식 산성을 각지에 지어 가신들이 성을 지키는 외성제도를 두었다. 본성인 가고시마 성도 북쪽에 <a href="/wiki/%ED%98%BC%EB%A7%88%EB%A3%A8" title="혼마루">혼마루</a>, 남쪽에 <a class="mw-redirect" href="/wiki/%EB%8B%88%EB%85%B8%EB%A7%88%EB%A3%A8" title="니노마루">니노마루</a>가 위치해있지만, 단순한 구조로 방어에 취약한 거관(居館)의 형식을 띤 성이었다. 이 때문에 뒷산인 시로야마 산을 배후의 성으로 삼았다. 덧붙여 이 시로야마 산은 〈上乃山城〉으로 불리며 초대 조다이로 <a href="/wiki/%EC%8B%9C%EB%A7%88%EC%A6%88_%EB%8F%84%EC%8B%9C%ED%9E%88%EC%82%AC" title="시마즈 도시히사">시마즈 도시히사</a>의 손자 <a class="new" href="/w/index.php?title=%EC%8B%9C%EB%A7%88%EC%A6%88_%EC%93%B0%EB%84%A4%ED%9E%88%EC%82%AC&amp;action=edit&amp;redlink=1" title="시마즈 쓰네히사 (없는 문서)">시마즈 쓰네히사</a>가 임명되어 거주했지만, 일찍 죽었기 때문에 그 후임을 두지 않고 시로야마 산은 출입금지구역으로 되었다.</p>
<p>또, <a class="mw-redirect" href="/wiki/%EB%8D%B4%EC%8A%88" title="덴슈">천수</a> 등 고층건물은 건립되지 않았다. 이것은 에도 막부에 순종한다는 의미를 띠고 있다. <a class="mw-redirect" href="/wiki/%EA%B3%A0%EC%BF%A0%EB%8B%A4%EC%B9%B4" title="고쿠다카">고쿠다카</a> 77만석의 <a href="/wiki/%EB%8B%A4%EC%9D%B4%EB%AC%98" title="다이묘">다이묘</a>의 성이라고 보기에는 매우 소박하다. <a href="/wiki/%EB%A9%94%EC%9D%B4%EC%A7%80_%EC%8B%9C%EB%8C%80" title="메이지 시대">메이지 시대</a> <a class="new" href="/w/index.php?title=%ED%98%BC%EB%B6%80_%EC%95%BC%EC%8A%A4%EC%8B%9C%EB%A1%9C&amp;action=edit&amp;redlink=1" title="혼부 야스시로 (없는 문서)">혼부 야스시로</a>는 그의 저서 《薩摩見聞記(사쓰마 견문기)》에서 〈不思議(의아하다)〉라고 평하고 있다.</p>
<h2><span class="mw-headline" id=".ED.98.84.ED.99.A9">현황</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EA%B0%80%EA%B3%A0%EC%8B%9C%EB%A7%88_%EC%84%B1&amp;action=edit&amp;section=3" title="부분 편집: 현황">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>성에 남아 있는 것으로는 석벽과 해자 그리고, <a href="/wiki/%EC%82%AC%EC%9D%B4%EA%B3%A0_%EB%8B%A4%EC%B9%B4%EB%AA%A8%EB%A6%AC" title="사이고 다카모리">사이고 다카모리</a>의 <a class="new" href="/w/index.php?title=%EC%8B%9C%EA%B0%93%EC%BD%94&amp;action=edit&amp;redlink=1" title="시갓코 (없는 문서)">시갓코</a>터 등이 남아있다. 이들 석벽에는 사이난 전쟁의 흔적인 탄흔이 다수 남아 있다.</p>
<p><a href="/wiki/1901%EB%85%84" title="1901년">1901년</a> 이후, 성터는 <a href="/wiki/%EA%B0%80%EA%B3%A0%EC%8B%9C%EB%A7%88_%EB%8C%80%ED%95%99" title="가고시마 대학">가고시마 대학</a>의 전신인 제7고등학교 용지로 사용되었고, 현재에는 <a class="new" href="/w/index.php?title=%EA%B0%80%EA%B3%A0%EC%8B%9C%EB%A7%88_%ED%98%84_%EC%97%AD%EC%82%AC%EC%9E%90%EB%A3%8C_%EC%84%BC%ED%84%B0_%EB%A0%88%EC%9D%B4%EB%A9%94%EC%9D%B4%EC%B9%B8&amp;action=edit&amp;redlink=1" title="가고시마 현 역사자료 센터 레이메이칸 (없는 문서)">가고시마 현 역사자료 센터 레이메이칸</a>, <a class="new" href="/w/index.php?title=%EA%B0%80%EA%B3%A0%EC%8B%9C%EB%A7%88_%ED%98%84%EB%A6%BD_%EB%8F%84%EC%84%9C%EA%B4%80&amp;action=edit&amp;redlink=1" title="가고시마 현립 도서관 (없는 문서)">가고시마 현립 도서관</a>, <a href="/wiki/%EA%B0%80%EA%B3%A0%EC%8B%9C%EB%A7%88_%EC%8B%9C%EB%A6%BD_%EB%AF%B8%EC%88%A0%EA%B4%80" title="가고시마 시립 미술관">가고시마 시립 미술관</a>이 들어서 있다.</p>
<p><a href="/wiki/2006%EB%85%84" title="2006년">2006년</a> <a href="/wiki/4%EC%9B%94_6%EC%9D%BC" title="4월 6일">4월 6일</a> 일본 100대 명성으로 선정되었다.</p>
