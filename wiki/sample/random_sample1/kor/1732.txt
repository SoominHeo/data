<p><b>가르비녜 무구루사</b>(<span style="font-size: smaller;"><a href="/wiki/%EC%8A%A4%ED%8E%98%EC%9D%B8%EC%96%B4" title="스페인어">스페인어</a>:</span> <span lang="es" xml:lang="es">Garbiñe Muguruza Blanco</span>, <a href="/wiki/1993%EB%85%84" title="1993년">1993년</a> <a href="/wiki/10%EC%9B%94_8%EC%9D%BC" title="10월 8일">10월 8일</a> ~ )는 <a href="/wiki/%EC%8A%A4%ED%8E%98%EC%9D%B8" title="스페인">스페인</a>의 <a href="/wiki/%ED%85%8C%EB%8B%88%EC%8A%A4" title="테니스">테니스</a> 선수이다. <a href="/wiki/%EB%B2%A0%EB%84%A4%EC%88%98%EC%97%98%EB%9D%BC" title="베네수엘라">베네수엘라</a> <a href="/wiki/%EC%B9%B4%EB%9D%BC%EC%B9%B4%EC%8A%A4" title="카라카스">카라카스</a>에서 <a href="/wiki/%EB%B0%94%EC%8A%A4%ED%81%AC%EC%9D%B8" title="바스크인">바스크계</a> <a href="/wiki/%EC%8A%A4%ED%8E%98%EC%9D%B8%EC%9D%B8" title="스페인인">스페인인</a>인 아버지와<sup class="reference" id="cite_ref-1"><a href="#cite_note-1">[1]</a></sup> <a href="/wiki/%EB%B2%A0%EB%84%A4%EC%88%98%EC%97%98%EB%9D%BC" title="베네수엘라">베네수엘라</a>인 어머니 사이에서 태어났다. 3살에 테니스를 시작하였고, 1999년 6살 때 전 가족이 <a href="/wiki/%EC%8A%A4%ED%8E%98%EC%9D%B8" title="스페인">스페인</a>으로 이주했다.<sup class="reference" id="cite_ref-2"><a href="#cite_note-2">[2]</a></sup> 2016년 <a href="/wiki/%ED%94%84%EB%9E%91%EC%8A%A4_%EC%98%A4%ED%94%88" title="프랑스 오픈">프랑스 오픈</a> 여자 단식에서 우승하였다.<sup class="reference" id="cite_ref-3"><a href="#cite_note-3">[3]</a></sup><sup class="reference" id="cite_ref-4"><a href="#cite_note-4">[4]</a></sup></p>
