<p><b>가마가야 시</b>(<span style="font-size: smaller;"><a href="/wiki/%EC%9D%BC%EB%B3%B8%EC%96%B4" title="일본어">일본어</a>:</span> <span lang="ja" xml:lang="ja">鎌ケ谷市</span>)는 <a href="/wiki/%EC%A7%80%EB%B0%94_%ED%98%84" title="지바 현">지바 현</a> 북서부에 있는 <a href="/wiki/%EC%8B%9C_(%ED%96%89%EC%A0%95_%EA%B5%AC%EC%97%AD)" title="시 (행정 구역)">시</a>이다.</p>
<p>지바 현 내에서도 유수한 <a class="mw-redirect" href="/wiki/%EB%B0%B0_(%EA%B3%BC%EC%9D%BC)" title="배 (과일)">배</a> 산지이며, 8월 중순부터 10월 초까지는 시내 곳곳에 배를 딸 수 있는 관광 농원이 개설된다. <a href="/wiki/1987%EB%85%84" title="1987년">1987년</a>에는 일본에서 처음 배 과실로 만든 <a href="/wiki/%EA%B3%BC%EC%8B%A4%EC%A3%BC" title="과실주">와인</a>과 <a href="/wiki/%EB%B8%8C%EB%9E%9C%EB%94%94" title="브랜디">브랜디</a>가 발매되었다. 그 이외에도 <a href="/wiki/%ED%8F%AC%EB%8F%84" title="포도">포도</a>, <a href="/wiki/%EA%B3%A0%EA%B5%AC%EB%A7%88" title="고구마">고구마</a>, <a href="/wiki/%ED%8C%8C_(%EC%8B%9D%EB%AC%BC)" title="파 (식물)">파</a> 등의 농업과 <a class="new" href="/w/index.php?title=%EB%A9%B4%EB%A5%98&amp;action=edit&amp;redlink=1" title="면류 (없는 문서)">면류</a>, <a href="/wiki/%EC%8B%9D%EC%B4%88" title="식초">식초</a> 등의 <a href="/wiki/%EC%A0%9C%EC%A1%B0%EC%97%85" title="제조업">제조업</a>이 번성하다.</p>
<p><a href="/wiki/%EB%8F%84%EC%BF%84" title="도쿄">도쿄</a>에서 30km 거리에 있으므로 <a href="/wiki/1970%EB%85%84%EB%8C%80" title="1970년대">1970년대</a> 이후에는 <a href="/wiki/%EC%9C%84%EC%84%B1_%EB%8F%84%EC%8B%9C" title="위성 도시">위성 도시</a>로서 인구가 증가했고, 현재 10만명에 달하고 있다.</p>

<h2><span class="mw-headline" id=".EC.A7.80.EB.A6.AC">지리</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EA%B0%80%EB%A7%88%EA%B0%80%EC%95%BC_%EC%8B%9C&amp;action=edit&amp;section=1" title="부분 편집: 지리">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>지바 현 북서부에 위치하고 <a href="/wiki/%EA%B0%80%EC%8B%9C%EC%99%80_%EC%8B%9C" title="가시와 시">가시와 시</a>, <a href="/wiki/%EC%8B%9C%EB%A1%9C%EC%9D%B4_%EC%8B%9C" title="시로이 시">시로이 시</a>, <a href="/wiki/%EB%A7%88%EC%93%B0%EB%8F%84_%EC%8B%9C" title="마쓰도 시">마쓰도 시</a>, <a href="/wiki/%EC%9D%B4%EC%B9%98%EC%B9%B4%EC%99%80_%EC%8B%9C" title="이치카와 시">이치카와 시</a>, <a href="/wiki/%ED%9B%84%EB%82%98%EB%B0%94%EC%8B%9C_%EC%8B%9C" title="후나바시 시">후나바시 시</a>에 인접한다.</p>
<p>북부에는 데가누마로 흘러 들어드는 오쓰 강이 흐르고 남부에는 오가시와 강 및 네고 강과 <a href="/wiki/%EB%8F%84%EC%BF%84_%EB%A7%8C" title="도쿄 만">도쿄 만</a>에 흘러 들어가는 에비 강의 지류가 흐른다. 시역은 시모우사 대지와 야쓰다에 놓여있고 해발이 가장 낮은 지점은 약 7m, 가장 높은 지점은 약 30m이다.</p>
<p>새로운 시가지로서 시청 주변으로 신카마가야 지구가 개발 되고 있다. 북부에는 <a href="/wiki/%ED%95%B4%EC%83%81%EC%9E%90%EC%9C%84%EB%8C%80" title="해상자위대">해상자위대</a> 시모사 항공 기지가 있고, 서부(구누기야마 지구)에는 <a href="/wiki/%EC%9C%A1%EC%83%81%EC%9E%90%EC%9C%84%EB%8C%80" title="육상자위대">육상자위대</a> 마쓰도 주둔지가 있다.</p>
<h2><span class="mw-headline" id=".EC.97.AD.EC.82.AC">역사</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EA%B0%80%EB%A7%88%EA%B0%80%EC%95%BC_%EC%8B%9C&amp;action=edit&amp;section=2" title="부분 편집: 역사">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<ul>
<li>1889년：가마가야를 비롯한 6개 촌과 <a href="/wiki/%EC%9D%B8%EB%B0%94_%EA%B5%B0" title="인바 군">인바 군</a> 네무라(현재 <a href="/wiki/%EC%8B%9C%EB%A1%9C%EC%9D%B4_%EC%8B%9C" title="시로이 시">시로이 시</a>의 일부)로부터 분촌한 가루이자와 지구가 합병해 가마가야 촌이 성립되었다.</li>
<li>1958년 8월 1일：정으로 승격되었다.</li>
<li>1971년 9월 1일：시로 승격되었다.</li>
</ul>
