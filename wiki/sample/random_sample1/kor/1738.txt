<p><b>가마이시 시</b>(<span style="font-size: smaller;"><a href="/wiki/%EC%9D%BC%EB%B3%B8%EC%96%B4" title="일본어">일본어</a>:</span> <span lang="ja"><span lang="ja"><ruby><rb>釜石市</rb> <rp>(</rp><rt>かまいしし</rt><rp>)</rp></ruby></span></span>)는 <a href="/wiki/%EC%9D%BC%EB%B3%B8" title="일본">일본</a> <a href="/wiki/%EC%9D%B4%EC%99%80%ED%85%8C_%ED%98%84" title="이와테 현">이와테 현</a> 남동부 (<a href="/wiki/%EC%82%B0%EB%A6%AC%EC%BF%A0" title="산리쿠">산리쿠</a> 해안)에 위치하는 시이다.</p>
<p><a href="/wiki/%EB%A6%AC%EC%BF%A0%EC%B6%94_%ED%95%B4%EC%95%88_%EA%B5%AD%EB%A6%BD%EA%B3%B5%EC%9B%90" title="리쿠추 해안 국립공원">리쿠추 해안 국립공원</a>의 중심에 위치하고 세계 3대 어장의 하나인 북서태평양 어장의 일부를 이루는 산리쿠 어장과 전형적인 <a href="/wiki/%EB%A6%AC%EC%95%84%EC%8A%A4%EC%8B%9D_%ED%95%B4%EC%95%88" title="리아스식 해안">리아스식 해안</a>을 가지는 시이다. 예전에는 <a href="/wiki/%EA%B0%80%EB%AF%B8%ED%97%A4%EC%9D%B4_%EA%B5%B0" title="가미헤이 군">가미헤이 군</a>, <a href="/wiki/%EA%B2%8C%EC%84%BC_%EA%B5%B0" title="게센 군">게센 군</a>에 속했다.</p>
<p>근대 제철업 발상지이며 최성기에는 인구가 9만 명을 넘기도 했지만 제철소의 용광로의 휴지와 함께 인구가 줄어 들고 있다.</p>

<h2><span class="mw-headline" id=".EC.97.AD.EC.82.AC">역사</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EA%B0%80%EB%A7%88%EC%9D%B4%EC%8B%9C_%EC%8B%9C&amp;action=edit&amp;section=1" title="부분 편집: 역사">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<ul>
<li><a href="/wiki/1889%EB%85%84" title="1889년">1889년</a> <a href="/wiki/4%EC%9B%94_1%EC%9D%BC" title="4월 1일">4월 1일</a> - 정촌제가 시행으로 미나마헤이 군 가마이시 정, 기노에네 촌, 우노스마이 촌, 구리하시 촌, <a href="/wiki/%EA%B2%8C%EC%84%BC_%EA%B5%B0" title="게센 군">게센 군</a> 도니 촌이 성립하였다.</li>
<li><a href="/wiki/1896%EB%85%84" title="1896년">1896년</a> <a href="/wiki/3%EC%9B%94_29%EC%9D%BC" title="3월 29일">3월 29일</a> - 미나미헤이 군과 니시헤이 군이 합병해 <a href="/wiki/%EA%B0%80%EB%AF%B8%ED%97%A4%EC%9D%B4_%EA%B5%B0" title="가미헤이 군">가미헤이 군</a>이 되었다.</li>
<li><a href="/wiki/1937%EB%85%84" title="1937년">1937년</a> <a href="/wiki/5%EC%9B%94_5%EC%9D%BC" title="5월 5일">5월 5일</a> - 가마이시 정이 시로 승격해 <b>가마이시 시</b>가 되었다.</li>
<li><a href="/wiki/1945%EB%85%84" title="1945년">1945년</a> <a href="/wiki/7%EC%9B%94_14%EC%9D%BC" title="7월 14일">7월 14일</a> ~ <a href="/wiki/8%EC%9B%94_9%EC%9D%BC" title="8월 9일">8월 9일</a> - <a href="/wiki/%ED%83%9C%ED%8F%89%EC%96%91_%EC%A0%84%EC%9F%81" title="태평양 전쟁">태평양 전쟁</a> 말기에 2도의 함포 사격으로 시가지, 제철소가 괴멸적인 피해를 받았다(가마이시 함포 사격).</li>
<li><a href="/wiki/1955%EB%85%84" title="1955년">1955년</a> <a href="/wiki/4%EC%9B%94_1%EC%9D%BC" title="4월 1일">4월 1일</a> - 구 가마이시 시, 기노에네 촌, 우노스마이 촌, 구리하시 촌, 도니 촌이 합병해 새로운 가마이시 시가 되었다.</li>
</ul>
<h2><span class="mw-headline" id=".EA.B5.90.ED.86.B5">교통</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EA%B0%80%EB%A7%88%EC%9D%B4%EC%8B%9C_%EC%8B%9C&amp;action=edit&amp;section=2" title="부분 편집: 교통">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p><a href="/wiki/2011%EB%85%84" title="2011년">2011년</a> <a href="/wiki/3%EC%9B%94_11%EC%9D%BC" title="3월 11일">3월 11일</a> <a class="mw-redirect" href="/wiki/%EB%8F%99%EC%9D%BC%EB%B3%B8_%EB%8C%80%EC%A7%80%EC%A7%84" title="동일본 대지진">동일본 대지진</a>에 따른 교통 영향으로, <a class="mw-redirect" href="/wiki/%EC%BC%80%EC%84%BC%EB%88%84%EB%A7%88_%EC%8B%9C" title="케센누마 시">케센누마</a> 방면 (남쪽)은 <a href="/wiki/2014%EB%85%84" title="2014년">2014년</a>에 <a href="/wiki/%EC%82%B0%EB%A6%AC%EC%BF%A0_%EC%B2%A0%EB%8F%84_%EB%AF%B8%EB%82%98%EB%AF%B8%EB%A6%AC%EC%95%84%EC%8A%A4_%EC%84%A0" title="산리쿠 철도 미나미리아스 선">남 리아스 선</a>이 복구했지만, <a href="/wiki/%EB%AF%B8%EC%95%BC%EC%BD%94_%EC%8B%9C" title="미야코 시">미야코</a> 방면 (북쪽)은 <a href="/wiki/2016%EB%85%84" title="2016년">2016년</a> 현재에도 <a href="/wiki/%EC%95%BC%EB%A7%88%EB%8B%A4_%EC%84%A0" title="야마다 선">야마다 선</a>이 아직 복구되지 않았고, <a class="new" href="/w/index.php?title=%EA%B5%AD%EB%8F%84_45%ED%98%B8%EC%84%A0_(%EC%9D%BC%EB%B3%B8)&amp;action=edit&amp;redlink=1" title="국도 45호선 (일본) (없는 문서)">국도 45호선</a>에서 왕래하게된다.</p>
