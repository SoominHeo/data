<p><b>가모 우지사토</b>(<span style="font-size: smaller;"><a href="/wiki/%EC%9D%BC%EB%B3%B8%EC%96%B4" title="일본어">일본어</a>:</span> <span lang="ja" xml:lang="ja">蒲生氏郷</span>, <a href="/wiki/1556%EB%85%84" title="1556년">1556년</a> ~ <a href="/wiki/1595%EB%85%84" title="1595년">1595년</a> <a href="/wiki/3%EC%9B%94_17%EC%9D%BC" title="3월 17일">3월 17일</a>)는 <a href="/wiki/%EC%84%BC%EA%B3%A0%EC%BF%A0_%EC%8B%9C%EB%8C%80" title="센고쿠 시대">센고쿠 시대</a>, <a class="mw-redirect" href="/wiki/%EC%95%84%EC%A6%88%EC%B9%98%EB%AA%A8%EB%AA%A8%EC%95%BC%EB%A7%88_%EC%8B%9C%EB%8C%80" title="아즈치모모야마 시대">아즈치모모야마 시대</a>의 무장이다. <a href="/wiki/%EC%98%A4%EB%AF%B8_%EA%B5%AD" title="오미 국">오미</a> <a class="new" href="/w/index.php?title=%ED%9E%88%EB%85%B8_%EC%84%B1&amp;action=edit&amp;redlink=1" title="히노 성 (없는 문서)">히노 성</a> 성주 <a href="/wiki/%EA%B0%80%EB%AA%A8_%EA%B0%80%ED%83%80%ED%9E%88%EB%8D%B0" title="가모 가타히데">가모 가타히데</a>(蒲生賢秀)의 적자로 태어났다. 어릴 적 이름은 쓰루치요이며, 초명은 <b>야스히데</b>(<span lang="ja" xml:lang="ja">賦秀</span>) 또는 노리히데(<span lang="ja" xml:lang="ja">教秀</span>)이다. 또한 <a class="new" href="/w/index.php?title=%EA%B8%B0%EB%A6%AC%EC%8B%9C%ED%83%84_%EB%8B%A4%EC%9D%B4%EB%AC%98&amp;action=edit&amp;redlink=1" title="기리시탄 다이묘 (없는 문서)">기리시탄 다이묘</a>로서, <a href="/wiki/%EC%84%B8%EB%A1%80%EB%AA%85" title="세례명">세례명</a>은 레온(혹은 레오)이다.</p>

<h2><span class="mw-headline" id=".EC.83.9D.EC.95.A0">생애</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EA%B0%80%EB%AA%A8_%EC%9A%B0%EC%A7%80%EC%82%AC%ED%86%A0&amp;action=edit&amp;section=1" title="부분 편집: 생애">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>가모 가문은 <a href="/wiki/%EC%98%A4%EC%8A%88_%ED%9B%84%EC%A7%80%EC%99%80%EB%9D%BC_%EC%94%A8" title="오슈 후지와라 씨">오슈 후지와라 씨</a>의 <a href="/wiki/%ED%9B%84%EC%A7%80%EC%99%80%EB%9D%BC%EB%85%B8_%ED%9E%88%EB%8D%B0%EC%82%AC%ED%86%A0" title="후지와라노 히데사토">히데사토</a>의 후예로, 가마쿠라 시대부터 내려오는 명문가이다. <a href="/wiki/1568%EB%85%84" title="1568년">1568년</a>에 가모 가문의 주군 가문이었던 롯카쿠 가문이 <a href="/wiki/%EC%98%A4%EB%8B%A4_%EB%85%B8%EB%B6%80%EB%82%98%EA%B0%80" title="오다 노부나가">오다 노부나가</a>(織田信長)에 의해 멸망했을 때, 아버지 가타히데는 오다 가문의 신하가 되었다. 이때 가타히데는 노부나가의 본거지였던 <a href="/wiki/%EA%B8%B0%ED%9B%84_%EC%84%B1" title="기후 성">기후 성</a>에 쓰루치요를 인질로 보냈다. 노부나가는 쓰루치요의 뛰어난 재능을 간파하고 딸 <a class="new" href="/w/index.php?title=%ED%9B%84%EC%9C%A0%ED%9E%88%EB%A9%94&amp;action=edit&amp;redlink=1" title="후유히메 (없는 문서)">후유히메</a>를 시집보내 사위로 삼았다. 쓰루치요는 관례를 올리고 이름을 주타로(忠太郞) 야스히데라고 지었다. 주타로라는 이름은 노부나가의 관직 이름이었던 단조추(<span lang="ja" xml:lang="ja">弾正忠</span>)로부터 한 글자를 딴 것이라는 설이 있다. 야스히데는 오다 가문의 친척으로서 두터운 대우를 받았다.</p>
<p>야스히데는 무용(武勇)에 뛰어나 12세 때이던 <a href="/wiki/1568%EB%85%84" title="1568년">1568년</a> <a class="new" href="/w/index.php?title=%EA%B8%B0%ED%83%80%EB%B0%94%ED%83%80%EC%BC%80_%EB%8F%84%EB%AA%A8%EB%85%B8%EB%A6%AC&amp;action=edit&amp;redlink=1" title="기타바타케 도모노리 (없는 문서)">기타바타케 도모노리</a>와 싸울 때 처음 출진한 후, <a href="/wiki/1569%EB%85%84" title="1569년">1569년</a> <a href="/wiki/%EC%9D%B4%EC%84%B8_%EA%B5%AD" title="이세 국">이세</a> <a class="new" href="/w/index.php?title=%EC%98%A4%EC%B9%B4%EC%99%80%EC%B9%98_%EC%84%B1&amp;action=edit&amp;redlink=1" title="오카와치 성 (없는 문서)">오카와치 성</a> 공격이나 <a href="/wiki/1570%EB%85%84" title="1570년">1570년</a> <a href="/wiki/%EC%95%84%EB%84%A4%EA%B0%80%EC%99%80_%EC%A0%84%ED%88%AC" title="아네가와 전투">아네가와 전투</a>, <a href="/wiki/1573%EB%85%84" title="1573년">1573년</a>의<a href="/wiki/%EC%95%84%EC%82%AC%EC%BF%A0%EB%9D%BC_%EC%94%A8" title="아사쿠라 씨">아사쿠라 씨</a> 공격과 <a class="new" href="/w/index.php?title=%EC%98%A4%EB%8B%A4%EB%8B%88_%EC%84%B1_%EA%B3%B5%EA%B2%A9&amp;action=edit&amp;redlink=1" title="오다니 성 공격 (없는 문서)">오다니 성 공격</a>, <a href="/wiki/1574%EB%85%84" title="1574년">1574년</a>의 이세 <a class="new" href="/w/index.php?title=%EB%82%98%EA%B0%80%EC%8B%9C%EB%A7%88_%EA%B3%B5%EA%B2%A9&amp;action=edit&amp;redlink=1" title="나가시마 공격 (없는 문서)">나가시마 공격</a>, <a href="/wiki/1575%EB%85%84" title="1575년">1575년</a> 나가시노-시타라가하라 전투 등 각지의 전투에 쉴새없이 출진하여 수많은 공을 세웠다. <a href="/wiki/1582%EB%85%84" title="1582년">1582년</a> 노부나가가 <a href="/wiki/%ED%98%BC%EB%85%B8%EC%A7%80%EC%9D%98_%EB%B3%80" title="혼노지의 변">혼노지의 변</a>에서 비명횡사하자, 야스히데는 아즈치 성에 있던 노부나가의 가족을 보호하고, 아버지와 함께 본거지 히노 성에서 <a href="/wiki/%EC%95%84%EC%BC%80%EC%B9%98_%EB%AF%B8%EC%93%B0%ED%9E%88%EB%8D%B0" title="아케치 미쓰히데">아케치 미쓰히데</a>(明智光秀)와의 싸움을 준비했다. 미쓰히데는 나가하마, 사와야마, 아즈치 등 오미 지역의 여러 성을 공격한 후에 히노 성을 포위할 계획이었으나, 그 직전에 <a href="/wiki/%EC%A3%BC%EA%B3%A0%EC%BF%A0_%EC%A7%80%EB%B0%A9" title="주고쿠 지방">주고쿠 지방</a>에서 급히 회군한 <a href="/wiki/%EB%8F%84%EC%9A%94%ED%86%A0%EB%AF%B8_%ED%9E%88%EB%8D%B0%EC%9A%94%EC%8B%9C" title="도요토미 히데요시">하시바 히데요시</a>(羽柴秀吉; 후에 도요토미 히데요시(豐臣秀吉))에게 <a href="/wiki/%EC%95%BC%EB%A7%88%EC%9E%90%ED%82%A4_%EC%A0%84%ED%88%AC" title="야마자키 전투">야마자키 전투</a>(山崎の戦い)에서 패배하여 죽었다.</p>
<p>야스히데는 이후 <a href="/wiki/%EB%8F%84%EC%9A%94%ED%86%A0%EB%AF%B8_%ED%9E%88%EB%8D%B0%EC%9A%94%EC%8B%9C" title="도요토미 히데요시">도요토미 히데요시</a>를 섬기게 되었다. 히데요시는 야스히데에게 이세의 마쓰자카에 12만 석의 영지를 주었다. 이무렵 아스히데는 우지사토로 개명하는데, 이것은 자신의 이름에 주군인 히데요시(秀吉)와 같은 글자를 사용하는 것은 불손하다고 생각했기 때문이었다. 한편 히데요시는 <a href="/wiki/1587%EB%85%84" title="1587년">1587년</a> 7월 우지사토에게 자신의 옛 성이던 하시바라는 성을 하사하기도 했다. 히데요시는 <a href="/wiki/%EA%B8%B0%EC%9A%94%EC%8A%A4_%ED%9A%8C%EC%9D%98" title="기요스 회의">기요스 회의</a> 이후로 노부나가의 다른 신하들보다 우위에 섰고, 노부나가의 뒤를 이어 일본 전국을 통일하기 위해 움직였다. 우지사토는 그런 히데요시를 따라 <a href="/wiki/1584%EB%85%84" title="1584년">1584년</a>의 <a href="/wiki/%EA%B3%A0%EB%A7%88%ED%82%A4%C2%B7%EB%82%98%EA%B0%80%EC%BF%A0%ED%85%8C_%EC%A0%84%ED%88%AC" title="고마키·나가쿠테 전투">고마키·나가쿠테 전투</a>, <a href="/wiki/1585%EB%85%84" title="1585년">1585년</a>의 <a class="new" href="/w/index.php?title=%EA%B8%B0%EC%9D%B4_%EC%A0%95%EB%B2%8C&amp;action=edit&amp;redlink=1" title="기이 정벌 (없는 문서)">기이 정벌</a>, <a href="/wiki/1587%EB%85%84" title="1587년">1587년</a>의 <a href="/wiki/%EA%B7%9C%EC%8A%88_%EC%A0%95%EB%B2%8C" title="규슈 정벌">규슈 정벌</a>이나 <a href="/wiki/1590%EB%85%84" title="1590년">1590년</a>의 <a class="new" href="/w/index.php?title=%EC%98%A4%EB%8B%A4%EC%99%80%EB%9D%BC_%EC%A0%95%EB%B2%8C&amp;action=edit&amp;redlink=1" title="오다와라 정벌 (없는 문서)">오다와라 정벌</a> 등에 종군하여 활약했고, 그 공을 인정받아 <a href="/wiki/1590%EB%85%84" title="1590년">1590년</a> 히데요시가 <a class="mw-redirect" href="/wiki/%EC%98%A4%EC%9A%B0_%EC%A7%80%EB%B0%A9" title="오우 지방">오우 지방</a>(도호쿠)을 평정하자 우지사토에게 이세의 영지 대신 오우의 <a href="/wiki/%EC%95%84%EC%9D%B4%EC%A6%88" title="아이즈">아이즈</a>에 42만 석의 거대한 영지를 주었다. 이후 토지 정리와 추가로 받은 영지를 합쳐 우지사토의 영지는 92만 석에 이르게 된다.</p>
<p>아이즈로 온 이후 본거지의 이름을 구로카와에서 와카마쓰로 바꾸었고 성을 개축했다(와카마쓰라는 이름은 자신의 옛 영토이던 마쓰자카에서 따온 것이라고 한다). 7층짜리 <a href="/wiki/%EC%B2%9C%EC%88%98%EA%B0%81" title="천수각">천수각</a>이 세워진 이 성에 우지사토는 자신의 어릴 적 이름을 따 <a class="mw-redirect" href="/wiki/%EC%93%B0%EB%A3%A8%EA%B0%80_%EC%84%B1" title="쓰루가 성">쓰루가 성</a>이라는 이름을 붙였다. 또 우지사토는 성을 쌓는 것뿐만 아니라 옛 영지(히노와 마쓰자카)의 상인들을 불러오고, 정기 시장을 개설하였으며, 라쿠이치(독점적인 상인조합에 의한 폐단을 막기 위해 영주가 시장을 관리하는 것)를 시행하였으며, 수공업을 장려하는 등, 마을을 발전시키기 위한 노력도 게을리 하지 않았다. 에도 시대 유력한 <a class="mw-redirect" href="/wiki/%EC%9D%BC%EB%B3%B8%EC%9D%98_%EB%B2%88" title="일본의 번">번</a> 중 하나였던 <a href="/wiki/%EC%95%84%EC%9D%B4%EC%A6%88_%EB%B2%88" title="아이즈 번">아이즈 번</a>의 기초도 이때 닦아진 것이다.</p>
<p>우지사토는 아이즈의 옛 영주였던 <a href="/wiki/%EB%8B%A4%ED%85%8C_%EB%A7%88%EC%82%AC%EB%AC%B4%EB%84%A4" title="다테 마사무네">다테 마사무네</a>(伊達政宗)와 마찰을 빚기도 했다. 우지사토는 <a href="/wiki/1591%EB%85%84" title="1591년">1591년</a> <a class="new" href="/w/index.php?title=%EC%98%A4%EC%9E%90%ED%82%A4%C2%B7%EA%B0%80%EC%82%AC%EC%9D%B4_%EC%9E%87%ED%82%A4&amp;action=edit&amp;redlink=1" title="오자키·가사이 잇키 (없는 문서)">오자키·가사이 잇키</a>가 일어나자 이를 진압했는데, 이때 마사무네가 반란을 선동했다는 사실을 히데요시에게 고발하였고 <a class="new" href="/w/index.php?title=%EA%B5%AC%EB%85%B8%ED%97%A4_%EB%A7%88%EC%82%AC%EC%9E%90%EB%84%A4&amp;action=edit&amp;redlink=1" title="구노헤 마사자네 (없는 문서)">구노헤 마사자네</a>가 반란<sup class="reference" id="cite_ref-1"><a href="#cite_note-1">[1]</a></sup>을 일으키자 이를 제압했다. 이듬해인 <a href="/wiki/1592%EB%85%84" title="1592년">1592년</a> <a href="/wiki/%EC%9E%84%EC%A7%84%EC%99%9C%EB%9E%80" title="임진왜란">임진왜란</a>이 일어났을 때, 우지사토는 규슈의 <a href="/wiki/%EB%82%98%EA%B3%A0%EC%95%BC_%EC%84%B1_(%ED%9E%88%EC%A0%A0_%EA%B5%AD)" title="나고야 성 (히젠 국)">나고야 성</a>에 있던 본진까지 갔지만, 진중에서 건강이 악화되어 <a href="/wiki/1595%EB%85%84" title="1595년">1595년</a> <a href="/wiki/%EC%9D%8C%EB%A0%A5_2%EC%9B%94_7%EC%9D%BC" title="음력 2월 7일">음력 2월 7일</a>(<a href="/wiki/3%EC%9B%94_17%EC%9D%BC" title="3월 17일">3월 17일</a>) 교토의 <a href="/wiki/%ED%9B%84%EC%8B%9C%EB%AF%B8_%EC%84%B1" title="후시미 성">후시미 성</a>에서 향년 40세로 사망했다.</p>
<p>우지사토의 무덤은 <a href="/wiki/%EA%B5%90%ED%86%A0_%EB%B6%80" title="교토 부">교토</a> <a class="new" href="/w/index.php?title=%EB%8B%A4%EC%9D%B4%ED%86%A0%EC%BF%A0%EC%A7%80&amp;action=edit&amp;redlink=1" title="다이토쿠지 (없는 문서)">다이토쿠지</a>(<span lang="ja" xml:lang="ja">大徳寺</span>) <a class="new" href="/w/index.php?title=%EC%98%A4%EB%B0%94%EC%9D%B4%EC%9D%B8&amp;action=edit&amp;redlink=1" title="오바이인 (없는 문서)">오바이인</a>(<span lang="ja" xml:lang="ja">黄梅院</span>)에 있다. 최근 그의 무덤을 발굴한 결과, 칼을 안고 있는 모습으로 묻혀 있는 것이 발견되었다. 또 그의 영지이던 후쿠시마 현 아이즈 와카마쓰에 있는 <a href="/wiki/%EA%B3%A0%ED%86%A0%EC%BF%A0%EC%9D%B8" title="고토쿠인">고토쿠인</a>(<span lang="ja" xml:lang="ja">興徳院</span>)에 그의 머리카락을 묻은 무덤이 있다.</p>
<h2><span class="mw-headline" id=".EC.9D.B8.EB.AC.BC.EB.90.A8">인물됨</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EA%B0%80%EB%AA%A8_%EC%9A%B0%EC%A7%80%EC%82%AC%ED%86%A0&amp;action=edit&amp;section=2" title="부분 편집: 인물됨">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>부하들을 소중히 여긴 사람으로 유명하다. 또 다도(茶道, 차를 마시는 예법)에도 조예가 깊었다. <a href="/wiki/%EC%84%BC%EB%85%B8_%EB%A6%AC%ED%81%90" title="센노 리큐">센노 리큐</a>의 '일곱 명의 훌륭한 제자' 중 한 사람으로 꼽혔으며, 또 그중에서도 필두로 여겨졌다. 센노 리큐가 <a href="/wiki/%EB%8F%84%EC%9A%94%ED%86%A0%EB%AF%B8_%ED%9E%88%EB%8D%B0%EC%9A%94%EC%8B%9C" title="도요토미 히데요시">도요토미 히데요시</a>에게 죽었을 때 그 아들 <a class="new" href="/w/index.php?title=%EC%84%BC%EB%85%B8_%EC%87%BC%EC%95%88&amp;action=edit&amp;redlink=1" title="센노 쇼안 (없는 문서)">센노 쇼안</a>은 우지사토의 힘으로 목숨을 건졌다. 한편 와카에도 능해, 임진왜란에 참전하기 위해 오미의 무사에서 고향 히노를 그리워하며 읊은 '생각해보면 사람의 운명이란 정처없는 것 내 고향을 이토록 멀리서 보게 될 줄이야'라는 노래가 유명하다.</p>
<p>부하들이 공을 세우면 가모라는 성을 하사하는 상을 남발했기 때문에, 부하들 중에 가모 성을 가진 사람들이 넘쳐났다. <a href="/wiki/%EB%A7%88%EC%97%90%EB%8B%A4_%EB%8F%84%EC%8B%9C%EC%9D%B4%EC%97%90" title="마에다 도시이에">마에다 도시이에</a>(前田利家)도 이 기묘한 버릇을 나무란 적이 있다. 또, 당시 열렬한 기독교도였던 <a class="new" href="/w/index.php?title=%EB%8B%A4%EC%B9%B4%EC%95%BC%EB%A7%88_%EC%9A%B0%EC%BD%98&amp;action=edit&amp;redlink=1" title="다카야마 우콘 (없는 문서)">다카야마 우콘</a>과 친했기 때문에 기독교도가 되어, 레온이라는 세례명을 받았다. 로마에도 때때로 사신을 보내, 당시의 로마 교황으로부터 감사편지를 받은 적도 있다. 이 외에도 대만에 군사를 보냈다고 하는 일화도 있다.</p>
<p>마쓰자카의 영주로 있을 무렵, 히노에서 많은 상인과 기술자들을 불러와 마쓰자카의 마을을 번영시키고자 했지만, 아이즈로 영지가 옮겨짐으로써 사업을 완성시키지 못하고, 뒤를 이어 마쓰자카의 영주가 된 <a href="/wiki/%ED%95%AB%ED%86%A0%EB%A6%AC_%EA%B0%80%EC%A6%88%ED%83%80%EB%8B%A4" title="핫토리 가즈타다">핫토리 가즈타다</a>, <a class="new" href="/w/index.php?title=%ED%9B%84%EB%A3%A8%ED%83%80_%EC%8B%9C%EA%B2%8C%EC%B9%B4%EC%93%B0&amp;action=edit&amp;redlink=1" title="후루타 시게카쓰 (없는 문서)">후루타 시게카쓰</a> 등에 의해 이어졌다. 아이즈에서도 히노와 마쓰자카로부터 많은 상인과 기술자들을 불러들여, 아이즈 도자기 등의 발전에 힘을 쏟았다. 또 미쓰이 가문도, 우지사토의 요청으로 히노에서 마쓰자카로 옮겨간 상인 가문 중 하나다(그러나 우지사토가 아이즈로 옮겨갔을 때는 그대로 마쓰자카에 남았다).</p>
<p>『상산기담』에는 우지사토가 오우에 92만석을 받았을 때 '설령 큰 영지를 갖고 있다고 해도 오우같은 시골에서는 원하는 바를 달성할 수 없다. 영지는 작더라도 쿄토에 가까이 있으면 천하를 노릴 수도 있을 터인데'라고 크게 한탄했다는 이야기가 나온다.</p>
<p>우지사토가 불과 40세의 나이로 급사한 것에 대해서, 히데요시나 도요토미 정권의 핵심인물이던 <a href="/wiki/%EC%9D%B4%EC%8B%9C%EB%8B%A4_%EB%AF%B8%EC%93%B0%EB%82%98%EB%A6%AC" title="이시다 미쓰나리">이시다 미쓰나리</a>(石田三成)가 그의 재능을 두려워하여 독살했다는 설이 있지만, 사실인지는 알 수 없다. 우지사토를 진찰했던 의사 <a class="new" href="/w/index.php?title=%EB%A7%88%EB%82%98%EC%84%B8_%EA%B2%90%EC%82%AC%EC%BF%A0&amp;action=edit&amp;redlink=1" title="마나세 겐사쿠 (없는 문서)">마나세 겐사쿠</a>는 자신의 진료기록인 &lt;의학텐쇼기&gt;에 우지사토가 임진왜란에 참전하기 위해 가던 도중에 이미 나고야 성에서 발병했고, <a href="/wiki/%ED%99%A9%EB%8B%AC" title="황달">황달</a>이 발생하였으며 눈 밑에도 <a href="/wiki/%EB%B6%80%EC%A2%85" title="부종">부종</a>이 생기는 등의 증상이 있었다고 기록했다. 겐사쿠의 진단내용으로 보아 우지사토는 <a class="mw-redirect" href="/wiki/%EC%A7%81%EC%9E%A5%EC%95%94" title="직장암">직장암</a> 또는 <a href="/wiki/%EC%B7%8C%EC%9E%A5%EC%95%94" title="췌장암">췌장암</a>에 걸렸던 것으로 추측된다. 가모 가문은 이에야스의 딸과 결혼하는 조건으로 적자인 <a class="mw-disambig" href="/wiki/%EA%B0%80%EB%AA%A8_%ED%9E%88%EB%8D%B0%EC%9C%A0%ED%82%A4" title="가모 히데유키">히데유키</a>(秀行)가 계승했다.</p>
