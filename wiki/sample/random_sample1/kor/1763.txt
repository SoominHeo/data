<p><b>가브리엘 이뮤티니언 아그본라호르</b>(Gabriel Imuetinyan Agbonlahor, <a href="/wiki/1986%EB%85%84" title="1986년">1986년</a> <a href="/wiki/10%EC%9B%94_13%EC%9D%BC" title="10월 13일">10월 13일</a>, <a href="/wiki/%EC%9E%89%EA%B8%80%EB%9E%9C%EB%93%9C" title="잉글랜드">잉글랜드</a> <a class="mw-redirect" href="/wiki/%EC%97%90%EB%94%A9%ED%84%B4" title="에딩턴">에딩턴</a> ~ )는 잉글랜드의 <a href="/wiki/%EC%B6%95%EA%B5%AC" title="축구">축구</a> 선수로, <a href="/wiki/%ED%94%84%EB%A6%AC%EB%AF%B8%EC%96%B4%EB%A6%AC%EA%B7%B8" title="프리미어리그">프리미어리그</a>의 <a href="/wiki/%EC%95%A0%EC%8A%A4%ED%84%B4_%EB%B9%8C%EB%9D%BC_FC" title="애스턴 빌라 FC">애스턴 빌라</a> 소속이다. 그는 부모로부터 <a href="/wiki/%EB%82%98%EC%9D%B4%EC%A7%80%EB%A6%AC%EC%95%84" title="나이지리아">나이지리아</a> 혈통과 <a href="/wiki/%EC%8A%A4%EC%BD%94%ED%8B%80%EB%9E%9C%EB%93%9C" title="스코틀랜드">스코틀랜드</a> 혈통을 물려받았다.</p>
<h2><span class="mw-headline" id=".ED.81.B4.EB.9F.BD_.EA.B2.BD.EB.A0.A5">클럽 경력</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EA%B0%80%EB%B8%8C%EB%A6%AC%EC%97%98_%EC%95%84%EA%B7%B8%EB%B3%B8%EB%9D%BC%ED%98%B8%EB%A5%B4&amp;action=edit&amp;section=1" title="부분 편집: 클럽 경력">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p><a href="/wiki/%EC%95%A0%EC%8A%A4%ED%84%B4_%EB%B9%8C%EB%9D%BC_FC" title="애스턴 빌라 FC">애스턴 빌라</a> 유스 아카데미의 걸작인 아그본라호르는 2002-03 시즌, <a href="/wiki/%EC%95%A0%EC%8A%A4%ED%84%B4_%EB%B9%8C%EB%9D%BC_FC" title="애스턴 빌라 FC">애스턴 빌라</a> 유스팀에서 첫 시즌을 보냈으며, 18번의 경기에 출장해 9골을 넣으며 인상적인 활약을 보여주었다.</p>
<p>다음 시즌인 03-04 시즌에는 29경기에 출전해 35골을 넣었는데, 이 기록은 <a class="new" href="/w/index.php?title=%EB%8B%A4%EB%A6%AC%EC%9A%B0%EC%8A%A4_%EB%B0%94%EC%85%80&amp;action=edit&amp;redlink=1" title="다리우스 바셀 (없는 문서)">다리우스 바셀</a>이 97-98 시즌에 기록한 38골과 대등한 것이었다. 그는 또한, <a href="/wiki/2005%EB%85%84" title="2005년">2005년</a>, <a href="/wiki/%EC%98%A4%EC%8A%A4%ED%8A%B8%EB%A0%88%EC%9D%BC%EB%A6%AC%EC%95%84" title="오스트레일리아">오스트레일리아</a>의 <a href="/wiki/%EC%95%A0%EB%93%A4%EB%A0%88%EC%9D%B4%EB%93%9C" title="애들레이드">애들레이드</a>에서 열린 U-20 국제 축구 대회에서 <a href="/wiki/%EC%95%A0%EC%8A%A4%ED%84%B4_%EB%B9%8C%EB%9D%BC_FC" title="애스턴 빌라 FC">애스턴 빌라</a> 유스팀의 일원으로서 인상적인 활약을 펼쳤다. 아그본라호르는 개막전에서 <a href="/wiki/%EC%98%A4%EC%8A%A4%ED%8A%B8%EB%A0%88%EC%9D%BC%EB%A6%AC%EC%95%84" title="오스트레일리아">오스트레일리아</a>의 AIS를 상대로 골을 넣었고, <a href="/wiki/%EC%84%B8%EB%A6%AC%EC%97%90_A" title="세리에 A">세리에 A</a>의 <a href="/wiki/AC_%EB%B0%80%EB%9E%80" title="AC 밀란">AC 밀란</a>과 2-2로 무승부를 기록한 경기에서도 골을 넣었다.</p>
<p>그는 <a href="/wiki/2005%EB%85%84" title="2005년">2005년</a> <a href="/wiki/10%EC%9B%94" title="10월">10월</a>, <a href="/wiki/%EC%85%B0%ED%95%84%EB%93%9C_%EC%9B%AC%EC%A6%88%EB%8D%B0%EC%9D%B4_FC" title="셰필드 웬즈데이 FC">셰필드 웬즈데이</a>에서의 2개월간의 임대 생활을 마치고 팀으로 돌아왔다. 아그본라호르는 <a href="/wiki/2006%EB%85%84" title="2006년">2006년</a> <a href="/wiki/3%EC%9B%94_18%EC%9D%BC" title="3월 18일">3월 18일</a>, <a href="/wiki/%EA%B5%AC%EB%94%94%EC%8A%A8_%ED%8C%8C%ED%81%AC" title="구디슨 파크">구디슨 파크</a>에서 열린 <a href="/wiki/%EC%97%90%EB%B2%84%ED%84%B4_FC" title="에버턴 FC">에버턴</a>과의 경기에서 자신의 <a href="/wiki/%EC%95%A0%EC%8A%A4%ED%84%B4_%EB%B9%8C%EB%9D%BC_FC" title="애스턴 빌라 FC">애스턴 빌라</a>에서의 데뷔전과 동시에 데뷔골을 성공시켰지만, 팀은 4-1로 크게 졌다. 그 후 <a href="/wiki/4%EC%9B%94_16%EC%9D%BC" title="4월 16일">4월 16일</a>, 아그본라호르는 <a href="/wiki/%EB%B2%84%EB%B0%8D%EC%97%84_%EC%8B%9C%ED%8B%B0_FC" title="버밍엄 시티 FC">버밍엄 시티</a>와의 경기에서도 교체 출전하였고, <a href="/wiki/9%EC%9B%94_30%EC%9D%BC" title="9월 30일">9월 30일</a>에는 프리미어리그 우승 경력을 가지고 있는 <a href="/wiki/%EC%B2%BC%EC%8B%9C_FC" title="첼시 FC">첼시</a>와의 경기에서 <a class="new" href="/w/index.php?title=%EB%A6%AC%EC%95%94_%EB%A6%BF%EC%A7%80%EC%9B%B0&amp;action=edit&amp;redlink=1" title="리암 릿지웰 (없는 문서)">리암 릿지웰</a>이 올린 크로스를 받아 멋진 헤딩으로 동점골을 성공시켰다. 그러나, 팀은 힘겹게 1-1로 비기고 말았다.</p>
<p>아그본라호르는 <a href="/wiki/2007%EB%85%84" title="2007년">2007년</a> <a href="/wiki/2%EC%9B%94_10%EC%9D%BC" title="2월 10일">2월 10일</a>, <a href="/wiki/%EB%A0%88%EB%94%A9_FC" title="레딩 FC">레딩</a>을 상대로 2-0으로 이긴 경기에서 <a class="new" href="/w/index.php?title=%ED%8C%8C%ED%8A%B8%EB%A6%AD_%EB%B2%A0%EB%A5%B4%EA%B1%B0&amp;action=edit&amp;redlink=1" title="파트릭 베르거 (없는 문서)">파트릭 베르거</a>와 교체되기 전까지, <a href="/wiki/%ED%94%84%EB%A6%AC%EB%AF%B8%EC%96%B4%EB%A6%AC%EA%B7%B8_2006-07" title="프리미어리그 2006-07">프리미어리그 2006-07</a> 시즌 <a href="/wiki/%EC%95%A0%EC%8A%A4%ED%84%B4_%EB%B9%8C%EB%9D%BC_FC" title="애스턴 빌라 FC">애스턴 빌라</a>의 모든 경기에 출전하였다.</p>
<h2><span class="mw-headline" id=".EA.B5.AD.EA.B0.80.EB.8C.80.ED.91.9C_.EA.B2.BD.EB.A0.A5">국가대표 경력</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EA%B0%80%EB%B8%8C%EB%A6%AC%EC%97%98_%EC%95%84%EA%B7%B8%EB%B3%B8%EB%9D%BC%ED%98%B8%EB%A5%B4&amp;action=edit&amp;section=2" title="부분 편집: 국가대표 경력">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>그는 <a href="/wiki/2006%EB%85%84_9%EC%9B%94" title="2006년 9월">2006년 9월</a>, <a href="/wiki/%EB%82%98%EC%9D%B4%EC%A7%80%EB%A6%AC%EC%95%84_%EC%B6%95%EA%B5%AC_%EA%B5%AD%EA%B0%80%EB%8C%80%ED%91%9C%ED%8C%80" title="나이지리아 축구 국가대표팀">나이지리아 U-20</a> 대표팀의 일원으로 <a href="/wiki/%EB%A5%B4%EC%99%84%EB%8B%A4_%EC%B6%95%EA%B5%AC_%EA%B5%AD%EA%B0%80%EB%8C%80%ED%91%9C%ED%8C%80" title="르완다 축구 국가대표팀">르완다</a>와의 경기에 나섰지만, 미래에는 잉글랜드를 위해 뛰겠다고 말하였다. <a href="/wiki/2006%EB%85%84" title="2006년">2006년</a> <a href="/wiki/6%EC%9B%94_28%EC%9D%BC" title="6월 28일">6월 28일</a>에는 처음으로 <a href="/wiki/%EC%9E%89%EA%B8%80%EB%9E%9C%EB%93%9C_%EC%B6%95%EA%B5%AC_%EA%B5%AD%EA%B0%80%EB%8C%80%ED%91%9C%ED%8C%80" title="잉글랜드 축구 국가대표팀">잉글랜드 U-21 대표팀</a>에 발탁되었고, 그 해 <a href="/wiki/10%EC%9B%94_6%EC%9D%BC" title="10월 6일">10월 6일</a>에는 <a href="/wiki/%EB%8F%85%EC%9D%BC_%EC%B6%95%EA%B5%AC_%EA%B5%AD%EA%B0%80%EB%8C%80%ED%91%9C%ED%8C%80" title="독일 축구 국가대표팀">독일 U-21 대표팀</a>과의 경기에서 교체 출장하면서 첫 경기를 치렀다.</p>
<p>그러나, 아그본라호르가 <a href="/wiki/%EC%8A%A4%ED%8E%98%EC%9D%B8" title="스페인">스페인</a>의 <a href="/wiki/%EC%9E%89%EA%B8%80%EB%9E%9C%EB%93%9C_%EC%B6%95%EA%B5%AC_%EA%B5%AD%EA%B0%80%EB%8C%80%ED%91%9C%ED%8C%80" title="잉글랜드 축구 국가대표팀">잉글랜드 U-21 대표팀</a> 캠프에서 이탈하자, <a href="/wiki/%EC%8A%A4%ED%8A%9C%EC%96%B4%ED%8A%B8_%ED%94%BC%EC%96%B4%EC%8A%A4" title="스튜어트 피어스">스튜어트 피어스</a> 감독은 그를 잉글랜드의 U-21 챔피언십 선수 명단에서 제외시켰으며, <a href="/wiki/%EB%8D%B0%EC%9D%B4%EB%B9%84%EB%93%9C_%EB%B2%A4%ED%8B%80%EB%A6%AC" title="데이비드 벤틀리">데이비드 벤틀리</a>가 그를 대신할 선수로 발탁되었다.</p>
<p>아그본라호르는 <a href="/wiki/2007%EB%85%84" title="2007년">2007년</a> <a href="/wiki/9%EC%9B%94" title="9월">9월</a>, <a href="/wiki/%EC%8A%A4%ED%8A%9C%EC%96%B4%ED%8A%B8_%ED%94%BC%EC%96%B4%EC%8A%A4" title="스튜어트 피어스">스튜어트 피어스</a> 감독에 의해 다시 부름을 받았으며, 그는 <a href="/wiki/%EB%AA%AC%ED%85%8C%EB%84%A4%EA%B7%B8%EB%A1%9C_%EC%B6%95%EA%B5%AC_%EA%B5%AD%EA%B0%80%EB%8C%80%ED%91%9C%ED%8C%80" title="몬테네그로 축구 국가대표팀">몬테네그로 U-21 대표팀</a>을 상대로 골을 넣었다. 또한, <a href="/wiki/2%EC%9B%94_1%EC%9D%BC" title="2월 1일">2월 1일</a>에는 <a href="/wiki/%ED%8C%8C%EB%B9%84%EC%98%A4_%EC%B9%B4%ED%8E%A0%EB%A1%9C" title="파비오 카펠로">파비오 카펠로</a> 감독의 <a href="/wiki/%EC%9E%89%EA%B8%80%EB%9E%9C%EB%93%9C_%EC%B6%95%EA%B5%AC_%EA%B5%AD%EA%B0%80%EB%8C%80%ED%91%9C%ED%8C%80" title="잉글랜드 축구 국가대표팀">잉글랜드 축구 국가대표팀</a> 선수 명단에 포함되었다. 그 후, 그는 <a href="/wiki/%EB%AF%B8%EA%B5%AD_%EC%B6%95%EA%B5%AC_%EA%B5%AD%EA%B0%80%EB%8C%80%ED%91%9C%ED%8C%80" title="미국 축구 국가대표팀">미국</a>과 <a href="/wiki/%ED%8A%B8%EB%A6%AC%EB%8B%88%EB%8B%A4%EB%93%9C_%ED%86%A0%EB%B0%94%EA%B3%A0_%EC%B6%95%EA%B5%AC_%EA%B5%AD%EA%B0%80%EB%8C%80%ED%91%9C%ED%8C%80" title="트리니다드 토바고 축구 국가대표팀">트리니다드 토바고</a>와의 경기에서도 교체 출장하며, 경기 수를 늘려나갔다.</p>
<p>성인 국가대표팀에는 2008년 11월 <a href="/wiki/%EB%8F%85%EC%9D%BC_%EC%B6%95%EA%B5%AC_%EA%B5%AD%EA%B0%80%EB%8C%80%ED%91%9C%ED%8C%80" title="독일 축구 국가대표팀">독일</a>과의 A매치에 선발 출장하여 공식적인 데뷔전을 가졌다.</p>
