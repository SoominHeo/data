<p><b>가산디지털단지역</b>(加山-團地驛)은 <a href="/wiki/%EC%84%9C%EC%9A%B8%ED%8A%B9%EB%B3%84%EC%8B%9C" title="서울특별시">서울특별시</a> <a href="/wiki/%EA%B8%88%EC%B2%9C%EA%B5%AC" title="금천구">금천구</a> 가산동에 있는 <a href="/wiki/%EA%B2%BD%EB%B6%80%EC%84%A0" title="경부선">경부선</a> 상의 <a href="/wiki/%EC%88%98%EB%8F%84%EA%B6%8C_%EC%A0%84%EC%B2%A0_1%ED%98%B8%EC%84%A0" title="수도권 전철 1호선">수도권 전철 1호선</a>과 <a href="/wiki/%EC%84%9C%EC%9A%B8_%EC%A7%80%ED%95%98%EC%B2%A0_7%ED%98%B8%EC%84%A0" title="서울 지하철 7호선">서울 지하철 7호선</a>의 <a href="/wiki/%ED%99%98%EC%8A%B9%EC%97%AD" title="환승역">환승역</a>이다.</p>
<p>수도권 전철 개통 당시에는 역 일원이 <a href="/wiki/%EC%98%81%EB%93%B1%ED%8F%AC%EA%B5%AC" title="영등포구">영등포구</a> 가리봉동(1980년 <a href="/wiki/%EA%B5%AC%EB%A1%9C%EA%B5%AC" title="구로구">구로구</a>에 편입)에 속하여 있었기 때문에 처음 역명은 <b>가리봉역</b>(加里峯驛)이었으며, <a href="/wiki/1995%EB%85%84" title="1995년">1995년</a> <a href="/wiki/%ED%96%89%EC%A0%95_%EA%B5%AC%EC%97%AD" title="행정 구역">행정 구역</a> 변경으로 일대가 <a href="/wiki/%EA%B8%88%EC%B2%9C%EA%B5%AC" title="금천구">금천구</a> 가산동에 편입된 이후에도 이 역명은 유지되었으나, 역 일원의 환경이 재래식 산업단지에서 디지털 정보통신 산업 중심지로 변화함에 따라 <a href="/wiki/2005%EB%85%84" title="2005년">2005년</a> <a href="/wiki/7%EC%9B%94_1%EC%9D%BC" title="7월 1일">7월 1일</a>에 <b>가산디지털단지역</b>으로 역명이 바뀌었다<sup class="reference" id="cite_ref-1"><a href="#cite_note-1">[1]</a></sup>.</p>
<p>7호선 승강장의 경우, 암반을 뚫어 건설하였다는 것을 강조하기 위하여 벽과 승강장이 인공 암석으로 되어있었으나, 지하 동굴형 인테리어가 <a href="/wiki/2003%EB%85%84" title="2003년">2003년</a> <a class="mw-redirect" href="/wiki/%EB%8C%80%EA%B5%AC_%EC%A7%80%ED%95%98%EC%B2%A0_%ED%99%94%EC%9E%AC%EC%82%AC%EA%B3%A0" title="대구 지하철 화재사고">대구 지하철 참사</a> 이후에 화재에 취약하다는 점이 문제로 지적되었다.<sup class="reference" id="cite_ref-2"><a href="#cite_note-2">[2]</a></sup> 거기에 <a href="/wiki/2005%EB%85%84" title="2005년">2005년</a> <a href="/wiki/1%EC%9B%94_3%EC%9D%BC" title="1월 3일">1월 3일</a>에는 다음 역인 <a href="/wiki/%EC%B2%A0%EC%82%B0%EC%97%AD" title="철산역">철산역</a>에서 방화 사건이 발생하게 되어 <a href="/wiki/2009%EB%85%84" title="2009년">2009년</a> 9월 승강장 벽면 교체 작업에 착수, <a href="/wiki/2010%EB%85%84" title="2010년">2010년</a> 1월 공사를 완료하였다. 이와 같은 이유로 벽면이 교체된 <a href="/wiki/%EC%84%9C%EC%9A%B8_%EC%A7%80%ED%95%98%EC%B2%A0_5%ED%98%B8%EC%84%A0" title="서울 지하철 5호선">5호선</a> <a href="/wiki/%EC%8B%A0%EA%B8%88%ED%98%B8%EC%97%AD" title="신금호역">신금호역</a>이나 <a href="/wiki/%EB%A7%88%EC%B2%9C%EC%97%AD" title="마천역">마천역</a>과는 달리, 7호선 가산디지털단지역의 역명판은 새로운 디자인으로 바뀌지 않았다. 7호선은 가산디지털단지역을 경유한 후 잠시 <a href="/wiki/%EA%B2%BD%EA%B8%B0%EB%8F%84" title="경기도">경기도</a> <a href="/wiki/%EA%B4%91%EB%AA%85%EC%8B%9C" title="광명시">광명시</a>로 들어간다.</p>
