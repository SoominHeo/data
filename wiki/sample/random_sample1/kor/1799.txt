<p><b>가와무라 스미요시</b> (<span style="font-size: smaller;"><a href="/wiki/%EC%9D%BC%EB%B3%B8%EC%96%B4" title="일본어">일본어</a>:</span> <span lang="ja"><span lang="ja"><ruby><rb>川村純義</rb> <rp>(</rp><rt>かわむら すみよし</rt><rp>)</rp></ruby></span></span> <small>카와무라 스미요시<sup>[<a href="/wiki/%EC%9C%84%ED%82%A4%EB%B0%B1%EA%B3%BC:%EC%9D%BC%EB%B3%B8%EC%96%B4%EC%9D%98_%ED%95%9C%EA%B8%80_%ED%91%9C%EA%B8%B0" title="위키백과:일본어의 한글 표기">*</a>]</sup></small>, <a href="/wiki/1836%EB%85%84" title="1836년">1836년</a> <a href="/wiki/12%EC%9B%94_18%EC%9D%BC" title="12월 18일">12월 18일</a> ~ <a href="/wiki/1904%EB%85%84" title="1904년">1904년</a> <a href="/wiki/8%EC%9B%94_12%EC%9D%BC" title="8월 12일">8월 12일</a>)는 <a href="/wiki/%EC%9D%BC%EB%B3%B8_%EC%A0%9C%EA%B5%AD" title="일본 제국">일본 제국</a>의 <a href="/wiki/%EA%B5%B0%EC%9D%B8" title="군인">군인</a>이다. <a href="/wiki/%EA%B0%80%EA%B3%A0%EC%8B%9C%EB%A7%88_%ED%98%84" title="가고시마 현">가고시마 현</a> 출신으로 <a href="/wiki/%EC%82%AC%EC%93%B0%EB%A7%88_%EB%B2%88" title="사쓰마 번">사쓰마 번</a>의 인사들과 함께 활동했다.</p>
<p><a href="/wiki/%EB%B3%B4%EC%8B%A0_%EC%A0%84%EC%9F%81" title="보신 전쟁">보신 전쟁</a> 당시 사쓰마 번 군대의 4번대장으로 <a href="/wiki/%EC%95%84%EC%9D%B4%EC%A6%88" title="아이즈">아이즈</a> 지방의 전투에 참여했다. <a href="/wiki/%EB%A9%94%EC%9D%B4%EC%A7%80_%EC%9C%A0%EC%8B%A0" title="메이지 유신">메이지 유신</a> 이후인 <a href="/wiki/1874%EB%85%84" title="1874년">1874년</a>에는 <a href="/wiki/%EC%9D%BC%EB%B3%B8_%EC%A0%9C%EA%B5%AD_%ED%95%B4%EA%B5%B0" title="일본 제국 해군">일본 제국 해군</a>의 중장이 되었다. <a href="/wiki/%EC%84%B8%EC%9D%B4%EB%82%9C_%EC%A0%84%EC%9F%81" title="세이난 전쟁">세이난 전쟁</a> 때는 사립학교 안의 화약고를 공습한 뒤 <a class="mw-redirect" href="/wiki/%EA%B0%80%EA%B3%A0%EC%8B%9C%EB%A7%88" title="가고시마">가고시마</a> 현령 <a href="/wiki/%EC%98%A4%EC%95%BC%EB%A7%88_%EC%93%B0%EB%82%98%EC%9A%94%EC%8B%9C" title="오야마 쓰나요시">오야마 쓰나요시</a>와 협상을 시도했지만, 학교 공습에 불만을 품고 있던 사립학교 간부의 방해로 실패했다. 개전 이후 <a href="/wiki/%EC%95%BC%EB%A7%88%EA%B0%80%ED%83%80_%EC%95%84%EB%A6%AC%ED%86%A0%EB%AA%A8" title="야마가타 아리토모">야마가타 아리토모</a>와 함께 총사령관으로서 해군을 지휘하여, 군인·물자 보급을 책임졌다. 세이난 전쟁 이후 참의 겸 해군경이 되었지만 아리토모와는 달리 정치계에는 참여하지 않았다.</p>
<p>어린 <a href="/wiki/%EC%87%BC%EC%99%80_%EC%B2%9C%ED%99%A9" title="쇼와 천황">쇼와 천황</a>을 잠시 맡아 가르치기도 했다. 사망 후 해군 대장에 추서되었다. 묘소는 <a href="/wiki/%EB%8B%A4%EB%A7%88_%EA%B3%B5%EB%8F%99%EB%AC%98%EC%A7%80" title="다마 공동묘지">다마 공동묘지</a>.</p>
