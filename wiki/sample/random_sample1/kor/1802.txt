<p><b>가와이 마사히로</b>(<span style="font-size: smaller;"><a href="/wiki/%EC%9D%BC%EB%B3%B8%EC%96%B4" title="일본어">일본어</a>:</span> <span lang="ja"><span lang="ja"><ruby><rb>川相 昌弘</rb> <rp>(</rp><rt>かわい まさひろ</rt><rp>)</rp></ruby></span></span>, <a href="/wiki/1964%EB%85%84" title="1964년">1964년</a> <a href="/wiki/9%EC%9B%94_27%EC%9D%BC" title="9월 27일">9월 27일</a> ~ )는 <a href="/wiki/%EC%9D%BC%EB%B3%B8" title="일본">일본</a>의 전 프로 <a href="/wiki/%EC%95%BC%EA%B5%AC" title="야구">야구</a> 선수이자 야구 지도자이며, 현재 <a href="/wiki/%EC%84%BC%ED%8A%B8%EB%9F%B4_%EB%A6%AC%EA%B7%B8" title="센트럴 리그">센트럴 리그</a>인 <a href="/wiki/%EC%9A%94%EB%AF%B8%EC%9A%B0%EB%A6%AC_%EC%9E%90%EC%9D%B4%EC%96%B8%EC%B8%A0" title="요미우리 자이언츠">요미우리 자이언츠</a>의 1군 수석 코치이다.</p>
<p>현역 시절 <a href="/wiki/%EC%9A%94%EB%AF%B8%EC%9A%B0%EB%A6%AC_%EC%9E%90%EC%9D%B4%EC%96%B8%EC%B8%A0" title="요미우리 자이언츠">요미우리 자이언츠</a>와 <a href="/wiki/%EC%A3%BC%EB%8B%88%EC%B9%98_%EB%93%9C%EB%9E%98%EA%B1%B4%EC%8A%A4" title="주니치 드래건스">주니치 드래건스</a>에서 활약을 했고 입단 동기는 <a href="/wiki/%EC%82%AC%EC%9D%B4%ED%86%A0_%EB%A7%88%EC%82%AC%ED%82%A4" title="사이토 마사키">사이토 마사키</a>가 있다. 통산 533개의 <a class="mw-redirect" href="/wiki/%ED%9D%AC%EC%83%9D_%EB%B2%88%ED%8A%B8" title="희생 번트">희생 번트</a>는 기네스 기록을 갖고 있을 정도의 ‘<b>번트의 달인</b>’(<span lang="ja">バント職人</span>)이라는 별명을 갖고 있으며 요미우리 구단 역사상 굴지의 수비력을 자랑하는 <a href="/wiki/%EC%9C%A0%EA%B2%A9%EC%88%98" title="유격수">유격수</a>였다.</p>
