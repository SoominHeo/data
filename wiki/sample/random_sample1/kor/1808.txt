<h2><span class="mw-headline" id=".ED.95.9C.EA.B5.AD.EC.96.B4">한국어</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EA%B0%80%EC%9A%B4%EB%8E%83%EC%A0%90&amp;action=edit&amp;section=1" title="부분 편집: 한국어">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p><a href="/wiki/%ED%95%9C%EA%B5%AD%EC%96%B4" title="한국어">한국어</a>에서 가운뎃점은 열거된 여러 단위가 대등하거나 밀접한 관계임을 나타낸다.<a class="external autonumber" href="http://www.korean.go.kr/09_new/dic/rule/rule01_0702.jsp" rel="nofollow">[1]</a></p>
<ul>
<li>쉼표로 열거된 어구가 다시 여러 단위로 나뉠 때에 쓴다. 아래는 이에 대한 예시이다.</li>
</ul>
<ol>
<li>철수·영이, 영수·순이가 서로 짝이 되어 윷놀이를 하였다.</li>
<li>공주·논산, 천안·아산·천원 등 각 지역구에서 2명씩 국회 의원을 뽑는다.</li>
<li>시장에 가서 사과·배·복숭아, 고추·마늘·파, 조기·명태·고등어를 샀다.</li>
</ol>
<ul>
<li>특정한 의미를 가지는 날을 나타내는 숫자에 쓴다. 아래는 예시.</li>
</ul>
<ol>
<li><a href="/wiki/3%C2%B71_%EC%9A%B4%EB%8F%99" title="3·1 운동">3·1 운동</a>, <a class="mw-redirect" href="/wiki/8%C2%B715_%EA%B4%91%EB%B3%B5" title="8·15 광복">8·15 광복</a></li>
</ol>
<ul>
<li>같은 계열의 단어 사이에 쓴다. 아래는 예시.</li>
</ul>
<ol>
<li>경북 방언의 조사·연구</li>
<li>충북·충남 두 도를 합하여 충청도라고 한다.</li>
<li>동사·형용사를 합하여 용언이라고 한다.</li>
</ol>
<h2><span class="mw-headline" id=".EC.9D.BC.EB.B3.B8.EC.96.B4">일본어</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EA%B0%80%EC%9A%B4%EB%8E%83%EC%A0%90&amp;action=edit&amp;section=2" title="부분 편집: 일본어">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>일본어에서는 복합어를 구분하려고 가운뎃점을 쓴다. 그리고 이름은 <b>나카구로</b>(<span style="font-size: smaller;"><a href="/wiki/%EC%9D%BC%EB%B3%B8%EC%96%B4" title="일본어">일본어</a>:</span> <span lang="ja"><span lang="ja"><ruby><rb>中黒</rb> <rp>(</rp><rt>なかぐろ</rt><rp>)</rp></ruby></span></span>)며, <a href="/wiki/%EC%9C%A0%EB%8B%88%EC%BD%94%EB%93%9C" title="유니코드">유니코드</a>는 U+30FB(・)다.</p>
<ul>
<li>여러 단어를 대등하게 나열할 때 쓰인다.
<ul>
<li><span lang="ja">食品・雑貨売り場</span> - 식품·잡화 판매장</li>
<li><span lang="ja">東海道・山陽新幹線</span> - <a class="new" href="/w/index.php?title=%EB%8F%84%EC%B9%B4%EC%9D%B4%EB%8F%84%C2%B7%EC%82%B0%EC%9A%94_%EC%8B%A0%EC%B9%B8%EC%84%BC&amp;action=edit&amp;redlink=1" title="도카이도·산요 신칸센 (없는 문서)">도카이도·산요 신칸센</a></li>
</ul>
</li>
<li>외래어를 <a href="/wiki/%EA%B0%80%ED%83%80%EC%B9%B4%EB%82%98" title="가타카나">가타카나</a>로 표기할 때, 단어 사이를 구분하기 위해 쓰인다. 사람 이름이 아니면, 생략하기도 한다.
<ul>
<li><span lang="ja">パーソナル・コンピュータ / パーソナルコンピュータ</span> - 퍼스널 컴퓨터(personal computer)</li>
<li><span lang="ja">トーマス・エジソン</span> - <a href="/wiki/%ED%86%A0%EB%A8%B8%EC%8A%A4_%EC%97%90%EB%94%94%EC%8A%A8" title="토머스 에디슨">토머스 에디슨</a>(Thomas Edison)</li>
</ul>
</li>
<li>사람 이름과 직함을 구분하려고 쓰인다.
<ul>
<li><span lang="ja">部長補佐・鈴木</span> - 부장 보좌 스즈키</li>
</ul>
</li>
<li>숫자를 세로쓰기를 하거나 한자로 적을 때, <a href="/wiki/%EC%86%8C%EC%88%98%EC%A0%90" title="소수점">소수점</a>을 나타내는데 쓰인다.
<ul>
<li>三・一四 - 3.14</li>
</ul>
</li>
</ul>
