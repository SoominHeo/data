<p><b>가자 지구</b>(<span style="font-size: smaller;"><a href="/wiki/%EC%95%84%EB%9E%8D%EC%96%B4" title="아랍어">아랍어</a>:</span> <span lang="ar" xml:lang="ar">قطاع غزة‎</span> <small>키타 가자<sup>[<a href="/wiki/%EC%9C%84%ED%82%A4%EB%B0%B1%EA%B3%BC:%EC%99%B8%EB%9E%98%EC%96%B4_%ED%91%9C%EA%B8%B0%EB%B2%95/%EC%95%84%EB%9E%8D%EC%96%B4_%EC%8B%9C%EC%95%88" title="위키백과:외래어 표기법/아랍어 시안">*</a>]</sup></small>, <span lang="en" xml:lang="en">Gaza</span> + 地區, <span style="font-size: smaller;"><a href="/wiki/%EB%AC%B8%ED%99%94%EC%96%B4" title="문화어">문화어</a>:</span> 가자지대)는 <a href="/wiki/%EC%A7%80%EC%A4%91%ED%95%B4" title="지중해">지중해</a> 연안에 위치한 <a href="/wiki/%ED%8C%94%EB%A0%88%EC%8A%A4%ED%83%80%EC%9D%B8_%EC%9E%90%EC%B9%98_%EC%A0%95%EB%B6%80" title="팔레스타인 자치 정부">팔레스타인 자치 정부</a>의 통치 지역으로, <a href="/wiki/%EC%9D%B4%EC%A7%91%ED%8A%B8" title="이집트">이집트</a>와 <a href="/wiki/%EC%9D%B4%EC%8A%A4%EB%9D%BC%EC%97%98" title="이스라엘">이스라엘</a>에 접해 있다. 이 지역은 <a href="/wiki/%EC%9A%94%EB%A5%B4%EB%8B%A8_%EA%B0%95_%EC%84%9C%EC%95%88_%EC%A7%80%EA%B5%AC" title="요르단 강 서안 지구">요르단 강 서안 지구</a>과 함께 <a href="/wiki/%ED%8C%94%EB%A0%88%EC%8A%A4%ED%83%80%EC%9D%B8" title="팔레스타인">팔레스타인</a> 독립 국가의 잠재적 영토로 상정된 곳이다. 역사가 오래되었으며, 고대 <a href="/wiki/%ED%95%84%EB%A6%AC%EC%8B%9C%ED%85%8C%EC%9D%B8" title="필리시테인">필리시테인</a>의 거주지였다.</p>
<p>가자지구는 일부 경작지에서 농산물이 생산되고 있으나, 목초지나 삼림지대는 거의 없다. 이 지역은 강수량이 매우 부족한데다 그마저 75%가 증발하여 수자원을 이스라엘에 크게 의존한다. 주민은 <a href="/wiki/%ED%8C%94%EB%A0%88%EC%8A%A4%ED%83%80%EC%9D%B8%EC%9D%B8" title="팔레스타인인">팔레스타인인</a>이 99.8%로 절대다수를 차지한다. 주민들 대부분은 <a href="/wiki/%EC%95%84%EB%9E%8D%EC%96%B4" title="아랍어">아랍어</a>를 사용하며, 종교는 <a class="mw-redirect" href="/wiki/%EC%9D%B4%EC%8A%AC%EB%9E%8C" title="이슬람">이슬람</a>으로 알려져 있으나 <a href="/wiki/%EA%B8%B0%EB%8F%85%EA%B5%90" title="기독교">기독교</a> 신자들도 상당수 있다.</p>
<p>지역의 중심도시는 <a href="/wiki/%EA%B0%80%EC%9E%90" title="가자">가자</a>이고, 그 이름을 따서 가자 지구라 불리고 있다. 척박한 자연환경에도 불구하고 정치·사회적 이유 때문에 364.3km²의 면적에 160만명이 넘는 인구가 몰려 사는 지역으로, <a href="/wiki/%ED%95%9C%EA%B5%AD" title="한국">한국</a>의 <a href="/wiki/%EA%B3%A0%EC%96%91%EC%8B%9C" title="고양시">고양시</a>(인구 100만 명)와 비교하면 1.36배의 땅에 1.6배가 넘는 인구가 거주하는 초고밀도 인구밀집지역이다. 한때 이 지역의 33%를 <a href="/wiki/%EC%9D%B4%EC%8A%A4%EB%9D%BC%EC%97%98" title="이스라엘">이스라엘</a>이 점령하였으나, 2005년 8월 <a href="/wiki/%EC%95%84%EB%A6%AC%EC%97%98_%EC%83%A4%EB%A1%A0" title="아리엘 샤론">아리엘 샤론</a> 이스라엘 총리가 가자 지구에서 군과 정착민을 철수시켰다. 현재 <a href="/wiki/%ED%95%98%EB%A7%88%EC%8A%A4" title="하마스">하마스</a>가 장악하고 있지만, 어느 독립국가의 영토도 아니어서 가자 지구의 주민은 무국적자 상태이다.</p>
<h2><span class="mw-headline" id=".EC.97.AD.EC.82.AC">역사</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EA%B0%80%EC%9E%90_%EC%A7%80%EA%B5%AC&amp;action=edit&amp;section=1" title="부분 편집: 역사">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>예로부터 통상과 군사상의 요지여서 많은 전쟁이 일어났는데, 1948년의 <a href="/wiki/%EC%A0%9C1%EC%B0%A8_%EC%A4%91%EB%8F%99_%EC%A0%84%EC%9F%81" title="제1차 중동 전쟁">제1차 중동 전쟁</a>(팔레스타인 전쟁) 이후 <a href="/wiki/1967%EB%85%84" title="1967년">1967년</a>까지 이집트가 군사통치하였고, 1967년의 <a href="/wiki/%EC%A4%91%EB%8F%99_%EC%A0%84%EC%9F%81#.EC.A0.9C3.EC.B0.A8_.EC.A4.91.EB.8F.99_.EC.A0.84.EC.9F.81_.281967.EB.85.84.29" title="중동 전쟁">제3차 중동 전쟁</a> 이후 <a href="/wiki/1994%EB%85%84" title="1994년">1994년</a>까지 이스라엘군의 통치 아래 있었다. 이스라엘 통치시기에 실시된 인구조사에 따르면, 주민 약 63만 명 중 3분의 2가 제1차 중동 전쟁 때 흘러들어온 팔레스타인 난민이었고 그 반수가 난민수용소에서 살고 있었다.</p>
<p><a href="/wiki/1987%EB%85%84" title="1987년">1987년</a> 12월 최대의 난민 캠프인 자발리아에서 <a href="/wiki/%EC%9D%B8%ED%8B%B0%ED%8C%8C%EB%8B%A4" title="인티파다">인티파다</a>(대중 봉기)가 발생하였으며, <a href="/wiki/1993%EB%85%84" title="1993년">1993년</a> 8월 제11차 중동 평화 협상에서 <a href="/wiki/%EC%9D%B4%EC%8A%A4%EB%9D%BC%EC%97%98" title="이스라엘">이스라엘</a>과 <a class="mw-redirect" href="/wiki/%ED%8C%94%EB%A0%88%EC%8A%A4%ED%83%80%EC%9D%B8_%ED%95%B4%EB%B0%A9%EA%B8%B0%EA%B5%AC" title="팔레스타인 해방기구">팔레스타인 해방기구</a>가 가자 지구를 포함한 점령지역의 자치안(<a href="/wiki/%EC%98%A4%EC%8A%AC%EB%A1%9C_%ED%98%91%EC%A0%95" title="오슬로 협정">오슬로 협정</a>)에 서명함으로써 <a href="/wiki/1994%EB%85%84" title="1994년">1994년</a>부터 가자 지구에 대한 통치권이 단계적으로 팔레스타인 자치기구에 이양되었다.</p>
<p><a href="/wiki/2006%EB%85%84" title="2006년">2006년</a> 자유선거에서 승리한 <a href="/wiki/%ED%95%98%EB%A7%88%EC%8A%A4" title="하마스">하마스</a>가 같은 해 6월 파타당과의 유혈내전을 시작하면서 <a href="/wiki/%EC%9D%B4%EC%8A%A4%EB%9D%BC%EC%97%98" title="이스라엘">이스라엘</a>과 <a href="/wiki/%EB%AF%B8%EA%B5%AD" title="미국">미국</a>, <a class="mw-redirect" href="/wiki/%EC%9C%A0%EB%9F%BD%EC%97%B0%ED%95%A9" title="유럽연합">유럽연합</a>이 제재조치를 불러오게 되었다. 현재 경제적 어려움을 겪고 있으며, 파타당이 장악하고 있는 <a href="/wiki/%EC%9A%94%EB%A5%B4%EB%8B%A8_%EA%B0%95_%EC%84%9C%EC%95%88_%EC%A7%80%EA%B5%AC" title="요르단 강 서안 지구">요르단 강 서안 지구</a>와는 정치적 경쟁 관계에 있다. 지중해로 나가는 길은 이스라엘 해군에 의해 봉쇄되어 있으며, 하마스의 원리주의를 경계하는 주변 아랍국들로 인해 대외 지원도 미약하다.</p>
