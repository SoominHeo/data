<p><b>가쿠타니 시즈오</b>(<span style="font-size: smaller;"><a href="/wiki/%EC%9D%BC%EB%B3%B8%EC%96%B4" title="일본어">일본어</a>:</span> <span lang="ja"><span lang="ja"><ruby><rb>角谷 静夫</rb> <rp>(</rp><rt>かくたに しずお</rt><rp>)</rp></ruby></span></span>, <span style="font-size: smaller;"><a href="/wiki/%EC%98%81%EC%96%B4" title="영어">영어</a>:</span> <span lang="en">Shizuo Kakutani</span> <small>시주오 카쿠타니<sup>[<a href="/wiki/%EC%9C%84%ED%82%A4%EB%B0%B1%EA%B3%BC:%EC%98%81%EC%96%B4%EC%9D%98_%ED%95%9C%EA%B8%80_%ED%91%9C%EA%B8%B0" title="위키백과:영어의 한글 표기">*</a>]</sup></small> <small><a href="/wiki/%EC%9C%84%ED%82%A4%EB%B0%B1%EA%B3%BC:IPA" title="위키백과:IPA">IPA</a>: </small><span class="IPA" title="IPA 발음">[ˈʃiːzuːoʊ kɑːkuːˈtɑːniː]</span>, <a href="/wiki/1911%EB%85%84" title="1911년">1911년</a> <a href="/wiki/8%EC%9B%94_28%EC%9D%BC" title="8월 28일">8월 28일</a> ~ <a href="/wiki/2004%EB%85%84" title="2004년">2004년</a> <a href="/wiki/8%EC%9B%94_17%EC%9D%BC" title="8월 17일">8월 17일</a>)는 <a href="/wiki/%EC%9D%BC%EB%B3%B8" title="일본">일본</a>의 수학자이다. <a href="/wiki/%ED%95%A8%EC%88%98%ED%95%B4%EC%84%9D%ED%95%99" title="함수해석학">함수해석학</a>에 공헌하였다.</p>
<h2><span class="mw-headline" id=".EC.83.9D.EC.95.A0">생애</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EA%B0%80%EC%BF%A0%ED%83%80%EB%8B%88_%EC%8B%9C%EC%A6%88%EC%98%A4&amp;action=edit&amp;section=1" title="부분 편집: 생애">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>1911년 8월 28일 <a href="/wiki/%EC%98%A4%EC%82%AC%EC%B9%B4_%EB%B6%80" title="오사카 부">오사카 부</a>에서 태어났고, 아버지는 <a href="/wiki/%EB%B3%80%ED%98%B8%EC%82%AC" title="변호사">변호사</a>였다. <a href="/wiki/%EB%8F%84%ED%98%B8%EC%BF%A0_%EB%8C%80%ED%95%99" title="도호쿠 대학">도호쿠 제국대학</a>을 졸업하였다. 1941년에 <a href="/wiki/%EC%98%A4%EC%82%AC%EC%B9%B4_%EB%8C%80%ED%95%99" title="오사카 대학">오사카 제국대학</a>에서 박사 학위를 취득했고, 이 곳에서 강의하였다. 전후 1948년에 미국에 이민하여 1949년에 <a href="/wiki/%EC%98%88%EC%9D%BC_%EB%8C%80%ED%95%99%EA%B5%90" title="예일 대학교">예일 대학교</a> 교수가 되었다.</p>
<p>그의 외동딸 가쿠타니 미치코(<span style="font-size: smaller;"><a href="/wiki/%EC%9D%BC%EB%B3%B8%EC%96%B4" title="일본어">일본어</a>:</span> <span lang="ja"><span lang="ja"><ruby><rb>角谷 美智子</rb> <rp>(</rp><rt>かくたに みちこ</rt><rp>)</rp></ruby></span></span>, <span style="font-size: smaller;"><a href="/wiki/%EC%98%81%EC%96%B4" title="영어">영어</a>:</span> <span lang="en">Michiko Kakutani</span> <small>미치코 카쿠타니<sup>[<a href="/wiki/%EC%9C%84%ED%82%A4%EB%B0%B1%EA%B3%BC:%EC%98%81%EC%96%B4%EC%9D%98_%ED%95%9C%EA%B8%80_%ED%91%9C%EA%B8%B0" title="위키백과:영어의 한글 표기">*</a>]</sup></small>, 1995~)는 《<a class="mw-redirect" href="/wiki/%EB%89%B4%EC%9A%95%ED%83%80%EC%9E%84%EC%A6%88" title="뉴욕타임즈">뉴욕타임즈</a>》의 문학 비평가로, 1998년 비평 부문 <a href="/wiki/%ED%93%B0%EB%A6%AC%EC%B2%98%EC%83%81" title="퓰리처상">퓰리처상</a>을 수상하였다.</p>
<p>2004년 8월 17일 미국 <a href="/wiki/%EB%89%B4%ED%97%A4%EC%9D%B4%EB%B8%90" title="뉴헤이븐">뉴헤이븐</a>에서 사망하였다.</p>
