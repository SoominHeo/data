<p><b>가타야마 요시히로</b> (片山 善博, <span style="font-size: smaller;"><a href="/wiki/%EC%9D%BC%EB%B3%B8%EC%96%B4" title="일본어">일본어</a>:</span> <span lang="ja" xml:lang="ja">かたやま よしひろ</span>, <a href="/wiki/1951%EB%85%84" title="1951년">1951년</a> <a href="/wiki/7%EC%9B%94_29%EC%9D%BC" title="7월 29일">7월 29일</a> ~ ) 는 <a href="/wiki/%EC%9D%BC%EB%B3%B8" title="일본">일본</a>의 정치인으로, 현재 <a class="new" href="/w/index.php?title=%EA%B0%84_%EB%82%98%EC%98%A4%ED%86%A0_%EB%82%B4%EA%B0%81_%EC%A0%9C2%EC%B0%A8_%EB%82%B4%EA%B0%81&amp;action=edit&amp;redlink=1" title="간 나오토 내각 제2차 내각 (없는 문서)">간 나오토 내각 제2차 내각</a>의 <a href="/wiki/%EC%9D%BC%EB%B3%B8%EC%9D%98_%EC%B4%9D%EB%AC%B4%EB%8C%80%EC%8B%A0" title="일본의 총무대신">총무대신</a>을 맡고 있다.</p>
<p><a href="/wiki/%EC%98%A4%EC%B9%B4%EC%95%BC%EB%A7%88_%ED%98%84" title="오카야마 현">오카야마 현</a>의 <a href="/wiki/%EC%98%A4%EC%B9%B4%EC%95%BC%EB%A7%88_%EC%8B%9C" title="오카야마 시">오카야마 시</a> 출신으로, <a href="/wiki/%EB%8F%84%EC%BF%84_%EB%8C%80%ED%95%99" title="도쿄 대학">도쿄 대학</a>을 졸업하였다. 1999년~2007년 <a href="/wiki/%EA%B3%A0%EC%B9%98_%ED%98%84" title="고치 현">고치 현</a> 지사를 두 차례 역임하였으며, 2010년 <a class="new" href="/w/index.php?title=%EA%B0%84_%EB%82%B4%EA%B0%81_%EC%A0%9C1%EC%B0%A8_%EA%B0%9C%EC%A1%B0_%EB%82%B4%EA%B0%81&amp;action=edit&amp;redlink=1" title="간 내각 제1차 개조 내각 (없는 문서)">간 내각 제1차 개조 내각</a>에서 총무대신으로 첫 입각한 데 이어, 2011년 <a class="new" href="/w/index.php?title=%EA%B0%84_%EB%82%B4%EA%B0%81_%EC%A0%9C2%EC%B0%A8_%EA%B0%9C%EC%A1%B0_%EB%82%B4%EA%B0%81&amp;action=edit&amp;redlink=1" title="간 내각 제2차 개조 내각 (없는 문서)">간 내각 제2차 개조 내각</a>에서도 총무대신에 유임되었다.</p>
