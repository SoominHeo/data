<p><b>가톨릭 오슬로 교구</b>(<span style="font-size: smaller;"><a href="/wiki/%EB%9D%BC%ED%8B%B4%EC%96%B4" title="라틴어">라틴어</a>:</span> <span lang="la" xml:lang="la">Dioecesis Osloënsis</span>)는 <a href="/wiki/%EB%85%B8%EB%A5%B4%EC%9B%A8%EC%9D%B4" title="노르웨이">노르웨이</a> <a href="/wiki/%EC%98%A4%EC%8A%AC%EB%A1%9C" title="오슬로">오슬로</a>의 <a href="/wiki/%EB%A1%9C%EB%A7%88_%EA%B0%80%ED%86%A8%EB%A6%AD%EA%B5%90%ED%9A%8C" title="로마 가톨릭교회">로마 가톨릭교회</a> <a class="mw-disambig" href="/wiki/%EA%B5%90%EA%B5%AC" title="교구">교구</a>이다.</p>
<h2><span class="mw-headline" id=".EC.97.AD.EC.82.AC">역사</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EA%B0%80%ED%86%A8%EB%A6%AD_%EC%98%A4%EC%8A%AC%EB%A1%9C_%EA%B5%90%EA%B5%AC&amp;action=edit&amp;section=1" title="부분 편집: 역사">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>1070년경에 오슬로 교구가 설정되었으며, 당시 <a class="mw-redirect" href="/wiki/%EC%A3%BC%EA%B5%90%EC%A2%8C_%EC%84%B1%EB%8B%B9" title="주교좌 성당">주교좌 성당</a>은 <a class="new" href="/w/index.php?title=%EC%84%B1_%ED%99%80%EB%B0%94%EB%A5%B4%EB%93%9C_%EB%8C%80%EC%84%B1%EB%8B%B9&amp;action=edit&amp;redlink=1" title="성 홀바르드 대성당 (없는 문서)">성 홀바르드 대성당</a>이었다. 1537년 <a class="mw-redirect" href="/wiki/%EC%A2%85%EA%B5%90_%EB%B6%84%EC%97%B4" title="종교 분열">종교 분열</a>의 영향으로 노르웨이의 가톨릭 <a class="mw-redirect" href="/wiki/%EC%82%AC%EB%8F%84_%EC%A0%84%EC%8A%B9" title="사도 전승">사도 전승</a>은 단절되었으며, 이후 <a href="/wiki/%EB%A3%A8%ED%84%B0%EA%B5%90" title="루터교">루터교</a>가 스칸디나비아에서 득세하게 되었다.</p>
<p>1582년 노르웨이와 기타 북유럽의 잔존한 가톨릭교도들은 <a href="/wiki/%EC%BE%B0%EB%A5%B8" title="쾰른">쾰른</a> 주재 <a href="/wiki/%EA%B5%90%ED%99%A9_%EB%8C%80%EC%82%AC" title="교황 대사">교황 대사</a>의 사목권에 맡겨지게 되었다. 1622년 교황청에 <a class="mw-redirect" href="/wiki/%ED%8F%AC%EA%B5%90%EC%84%B1%EC%84%B1" title="포교성성">포교성성</a>이 설정되면서 쾰른 주재 교황 대사는 <a href="/wiki/%EB%8F%85%EC%9D%BC" title="독일">독일</a> 북부만을 담당하게 되었으며, 북유럽 지역은 브뤼셀 주재 대사는 노르웨이와 <a href="/wiki/%EB%8D%B4%EB%A7%88%ED%81%AC" title="덴마크">덴마크</a>를, 폴란드 주재 교황 대사는 <a href="/wiki/%EC%8A%A4%EC%9B%A8%EB%8D%B4" title="스웨덴">스웨덴</a>과 <a href="/wiki/%ED%95%80%EB%9E%80%EB%93%9C" title="핀란드">핀란드</a> 및 메클렌부르크를 담당하게 되었다.</p>
<p>1688년 노르웨이 교회는 <a class="new" href="/w/index.php?title=%EB%B6%81%EC%9C%A0%EB%9F%BD_%EC%A0%84%EA%B5%90_%EB%8C%80%EB%AA%A9%EA%B5%AC&amp;action=edit&amp;redlink=1" title="북유럽 전교 대목구 (없는 문서)">북유럽 전교 대목구</a>에 편입되었다. 1834년 북유럽 전교 대목구에서 <a class="new" href="/w/index.php?title=%EC%8A%A4%EC%9B%A8%EB%8D%B4_%EC%A7%80%EB%AA%A9%EA%B5%AC&amp;action=edit&amp;redlink=1" title="스웨덴 지목구 (없는 문서)">스웨덴 지목구</a>가 분할되면서 노르웨이 교회는 스웨덴 지목구에 편입되었다. 1855년 노르웨이 북부 지역은 <a class="new" href="/w/index.php?title=%EB%B6%81%EA%B7%B9_%EC%A7%80%EB%AA%A9%EA%B5%AC&amp;action=edit&amp;redlink=1" title="북극 지목구 (없는 문서)">북극 지목구</a>의 관할 아래 놓이게 되었지만, 기타 노르웨이 지역은 여전히 스웨덴 지목구의 관할 아래 있었다. 1869년 8월 17일 스웨덴 지목구에서 노르웨이 지목구가 분할되었다. 그리고 1892년 3월 11일에는 <a class="new" href="/w/index.php?title=%EB%85%B8%EB%A5%B4%EC%9B%A8%EC%9D%B4_%EA%B5%90%ED%99%A9%EC%B2%AD_%EB%8C%80%EB%A6%AC%EA%B5%AC&amp;action=edit&amp;redlink=1" title="노르웨이 교황청 대리구 (없는 문서)">노르웨이 교황청 대리구</a>로 격상되었다.</p>
<p>1931년 4월 10일 노르웨이 교황청 대리구는 오슬로 교황청 대리구와 트론헤임 성직자치구, 트롬쇠 성직자치구로 분할되었으며, 이 가운데 오슬로 교황청 대리구가 1953년 6월 29일 스웨덴의 <a href="/wiki/%EA%B0%80%ED%86%A8%EB%A6%AD_%EC%8A%A4%ED%86%A1%ED%99%80%EB%A6%84_%EA%B5%90%EA%B5%AC" title="가톨릭 스톡홀름 교구">스톡홀름 교구</a>와 같이 독립 교구로 격상되었다.</p>
