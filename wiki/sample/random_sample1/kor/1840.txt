<p>《<b>가필드</b>(<span lang="en" xml:lang="en">Garfield</span>)》는 <a class="mw-redirect" href="/wiki/%EC%A7%90_%EB%8D%B0%EC%9D%B4%EB%B9%84%EC%8A%A4" title="짐 데이비스">짐 데이비스</a>의 <a class="new" href="/w/index.php?title=%EC%97%B0%EC%9E%AC%EB%A7%8C%ED%99%94&amp;action=edit&amp;redlink=1" title="연재만화 (없는 문서)">연재만화</a> 《<a href="/wiki/%EA%B0%80%ED%95%84%EB%93%9C" title="가필드">가필드</a>》를 원작으로 제작된 <a href="/wiki/2004%EB%85%84" title="2004년">2004년</a> 영화이다. 영화에 등장하는 가필드는 <a href="/wiki/%EC%BB%B4%ED%93%A8%ED%84%B0_%EC%83%9D%EC%84%B1_%EC%9D%B4%EB%AF%B8%EC%A7%80" title="컴퓨터 생성 이미지">컴퓨터 생성 이미지</a>(CGI)로 제작되었으나, 다른 동물들은 실제 동물이 출연하였다.</p>
<p><a class="new" href="/w/index.php?title=%ED%94%BC%ED%84%B0_%ED%9C%B4%EC%9E%87&amp;action=edit&amp;redlink=1" title="피터 휴잇 (없는 문서)">피터 휴잇</a>이 감독을 맡고, <a href="/wiki/20%EC%84%B8%EA%B8%B0_%ED%8F%AD%EC%8A%A4" title="20세기 폭스">20세기 폭스</a>의 <a class="new" href="/w/index.php?title=%EB%8D%B0%EC%9D%B4%EB%B9%84%EC%8A%A4_%EC%97%94%ED%84%B0%ED%85%8C%EC%9D%B8%EB%A8%BC%ED%8A%B8&amp;action=edit&amp;redlink=1" title="데이비스 엔터테인먼트 (없는 문서)">데이비스 엔터테인먼트</a>에서 제작을 맡았다. <a class="mw-redirect" href="/wiki/%EB%B8%8C%EB%A0%88%ED%82%A8_%EB%A9%94%EC%9D%B4%EC%96%B4" title="브레킨 메이어">브레킨 메이어</a>가 존 아버클, <a class="mw-redirect" href="/wiki/%EC%A0%9C%EB%8B%88%ED%8D%BC_%EB%9F%AC%EB%B8%8C_%ED%9C%B4%EC%9E%87" title="제니퍼 러브 휴잇">제니퍼 러브 휴잇</a>이 리즈 윌슨 역을 맡았으며, <a class="mw-redirect" href="/wiki/%EB%B9%8C_%EB%A8%B8%EB%A0%88%EC%9D%B4" title="빌 머레이">빌 머레이</a>가 가필드의 목소리를 맡았다.</p>
<p>속편 《<a href="/wiki/%EA%B0%80%ED%95%84%EB%93%9C_2" title="가필드 2">가필드 2</a>》는 <a href="/wiki/2006%EB%85%84" title="2006년">2006년</a> <a href="/wiki/6%EC%9B%94_16%EC%9D%BC" title="6월 16일">6월 16일</a> 개봉되었다.</p>
