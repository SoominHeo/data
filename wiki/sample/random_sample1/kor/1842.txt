<div class="mw-content-ltr" dir="ltr" id="mw-content-text" lang="ko"><script>function mfTempOpenSection(id){var block=document.getElementById("mf-section-"+id);block.className+=" open-block";block.previousSibling.className+=" open-block";}</script><p><b>각궁</b>(角弓)은 <a href="/wiki/%ED%95%9C%EB%B0%98%EB%8F%84" title="한반도">한반도</a>의 전통적인 <a href="/wiki/%ED%95%A9%EC%84%B1%EA%B6%81" title="합성궁">합성궁</a>으로, <a class="mw-redirect" href="/wiki/%EC%B0%B8%EB%82%98%EB%AC%B4" title="참나무">참나무</a>, <a class="new" href="/w/index.php?title=%EC%82%B0%EB%BD%95%EB%82%98%EB%AC%B4&amp;action=edit&amp;redlink=1" title="산뽕나무 (없는 문서)">산뽕나무</a>, <a class="new" href="/w/index.php?title=%EB%AC%BC%EC%86%8C%EB%BF%94&amp;action=edit&amp;redlink=1" title="물소뿔 (없는 문서)">물소뿔</a>, <a class="new" href="/w/index.php?title=%EC%86%8C%ED%9E%98%EC%A4%84&amp;action=edit&amp;redlink=1" title="소힘줄 (없는 문서)">소힘줄</a>, <a href="/wiki/%EB%8C%80%EB%82%98%EB%AC%B4" title="대나무">대나무</a>를 <a class="new" href="/w/index.php?title=%EB%AF%BC%EC%96%B4%EB%B6%80%EB%A0%88%ED%92%80&amp;action=edit&amp;redlink=1" title="민어부레풀 (없는 문서)">민어부레풀</a>로 접합하여 만들었다. 과거 한국에는 10여 가지의 활이 존재하였지만, 현존하는 것은 각궁 하나뿐이다.<sup class="reference" id="cite_ref-1"><a href="#cite_note-1">[1]</a></sup></p>

<h2><span class="mw-headline" id=".EC.97.AD.EC.82.AC">역사</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EA%B0%81%EA%B6%81&amp;action=edit&amp;section=1" title="부분 편집: 역사">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>각궁은 삼국시대의 맥궁에서 기원한 것으로 추정된다. 한국의 활의 역사는 오래 되었지만 각궁이 언제부터 널리 보급되었는지 확실치는 않다. 다만 <a class="mw-redirect" href="/wiki/%ED%95%A8%ED%9D%A5" title="함흥">함흥</a> <a class="mw-redirect" href="/wiki/%EC%84%A0%EC%9B%90%EC%A0%84" title="선원전">선원전</a>에 <a href="/wiki/%EC%A1%B0%EC%84%A0_%ED%83%9C%EC%A1%B0" title="조선 태조">태조</a> <a class="mw-redirect" href="/wiki/%EC%9D%B4%EC%84%B1%EA%B3%84" title="이성계">이성계</a>의 유물로서 이 각궁이 보관되어 있었고, 또 <a href="/wiki/%EA%B2%BD%EA%B5%AD%EB%8C%80%EC%A0%84" title="경국대전">경국대전</a>에 각궁에 관한 내용이 많이 남아있는 것으로 보아 <a href="/wiki/%EA%B3%A0%EB%A0%A4" title="고려">고려</a> 시대에 이미 각궁의 꾸밈새가 정형화되어 있었던 것으로 보인다.</p>
<p><a href="/wiki/%EC%A1%B0%EC%84%A0" title="조선">조선</a> 중기 이후 각궁은 심신수련을 위해 주로 사용되었으며, <a class="mw-redirect" href="/wiki/%EA%B0%91%EC%98%A4%EA%B2%BD%EC%9E%A5" title="갑오경장">갑오경장</a> 이후에는 국가 제식무기 편제에서 <a href="/wiki/%ED%99%9C" title="활">활</a>이 제외되면서 완전히 군사무기로서의 지위를 잃게 되었다.</p>
<p>각궁은 <a href="/wiki/%EC%98%81%EA%B5%AD" title="영국">영국</a>의 <a class="mw-redirect" href="/wiki/%EB%A1%B1%EB%B3%B4%EC%9A%B0" title="롱보우">롱보우</a>에 비하면 그 길이는 짧지만 오히려 위력은 강한데 이것은 여러가지 재료를 덧대어 만든 합성궁이기 때문이다. 하지만 만들기가 까다롭고, 특히 물소뿔은 <a href="/wiki/%EC%9D%B8%EB%8F%84" title="인도">인도</a>및 동남아 등지에서 수입해오는 전략 물자로 재료 자체가 절대적으로 적어 대량 생산하기에는 무리가 따랐다.</p>
<p>특히 습기에 약하여 장마철 습기를 먹으면 탄성이 떨어진다고 하는데 이는 각궁의 접착제로 사용하는 특수한 부레풀(민어 부레풀)의 성질에 기인한다고 한다.</p>
<p>초보자가 요령 없이 다루면 파손되기 쉬운 각궁은 오늘날에는 거의 <a href="/wiki/%EA%B0%9C%EB%9F%89%EA%B6%81" title="개량궁">개량궁</a>으로 대체되고 있는 추세이다.</p>
<h2><span class="mw-headline" id=".EC.A0.84.EC.88.98">전수</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EA%B0%81%EA%B6%81&amp;action=edit&amp;section=2" title="부분 편집: 전수">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>1971년에 궁시장이 무형문화재로 처음으로 지정됐다. 당시 활을 만드는 장인은 서울, 경기 부천, 경북 예천에 있었는데, 장진섭, 김장환, 권영만이 각 지역의 대표적 장인이었다.</p>
<p>현재, 무형문화재 47호의 보유자는 활과 화살을 나누어 지정되어 있는데, 활은 <a class="new" href="/w/index.php?title=%EA%B9%80%EB%B0%95%EC%98%81&amp;action=edit&amp;redlink=1" title="김박영 (없는 문서)">김박영</a> 궁장, 화살은 유영기, 김종국, 박호준 시장이 지정되어 있다. 김박영은 부천 활 박물관 명예관장을 맡고 있고, 유영기는 경기도 파주의 영집궁시박물관을 운영하고 있다. <sup class="reference" id="cite_ref-2"><a href="#cite_note-2">[2]</a></sup></p>
