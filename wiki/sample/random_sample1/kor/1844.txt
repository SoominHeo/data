<p><b>간 나오토</b>(<span style="font-size: smaller;"><a href="/wiki/%EC%9D%BC%EB%B3%B8%EC%96%B4" title="일본어">일본어</a>:</span> <span lang="ja"><span lang="ja"><ruby><rb>菅 直人</rb> <rp>(</rp><rt>かん なおと</rt><rp>)</rp></ruby></span></span> <small>칸 나오토<sup>[<a href="/wiki/%EC%9C%84%ED%82%A4%EB%B0%B1%EA%B3%BC:%EC%9D%BC%EB%B3%B8%EC%96%B4%EC%9D%98_%ED%95%9C%EA%B8%80_%ED%91%9C%EA%B8%B0" title="위키백과:일본어의 한글 표기">*</a>]</sup></small>, <span style="font-size: smaller;"><a href="/wiki/%EB%AC%B8%ED%99%94%EC%96%B4" title="문화어">문화어</a>:</span> 간 나오또, <a href="/wiki/1946%EB%85%84" title="1946년">1946년</a> <a href="/wiki/10%EC%9B%94_10%EC%9D%BC" title="10월 10일">10월 10일</a> ~ )는 일본의 제94대 <a href="/wiki/%EC%9D%BC%EB%B3%B8%EC%9D%98_%EB%82%B4%EA%B0%81%EC%B4%9D%EB%A6%AC%EB%8C%80%EC%8B%A0" title="일본의 내각총리대신">총리</a>이며 <a href="/wiki/%EC%A4%91%EC%9D%98%EC%9B%90" title="중의원">중의원</a> 의원이다. <a href="/wiki/2010%EB%85%84" title="2010년">2010년</a> <a href="/wiki/6%EC%9B%94_4%EC%9D%BC" title="6월 4일">6월 4일</a> 일본의 다수당인 <a href="/wiki/%EB%AF%BC%EC%A3%BC%EB%8B%B9_(%EC%9D%BC%EB%B3%B8,_1998%EB%85%84)" title="민주당 (일본, 1998년)">민주당</a>의 대표로 선출되어 제94대 총리로 지명되었으며, <a href="/wiki/2010%EB%85%84" title="2010년">2010년</a> <a href="/wiki/9%EC%9B%94_14%EC%9D%BC" title="9월 14일">9월 14일</a> 2년 임기의 당 대표에 재선되어 재신임되었다.</p>
<p>이듬해 <a href="/wiki/2011%EB%85%84" title="2011년">2011년</a> <a href="/wiki/8%EC%9B%94_26%EC%9D%BC" title="8월 26일">8월 26일</a>에는 사임을 발표했고, 총리직에서 물러났다.<sup class="reference" id="cite_ref-1"><a href="#cite_note-1">[1]</a></sup></p>

<p><a href="/wiki/%EC%95%BC%EB%A7%88%EA%B5%AC%EC%B9%98_%ED%98%84" title="야마구치 현">야마구치 현</a> <a href="/wiki/%EC%9A%B0%EB%B2%A0_%EC%8B%9C" title="우베 시">우베 시</a>에서 태어나 <a href="/wiki/%EB%8F%84%EC%BF%84_%EA%B3%B5%EC%97%85%EB%8C%80%ED%95%99" title="도쿄 공업대학">도쿄 공업대학</a> 이학부 응용물리학과를 졸업했고, <a href="/wiki/1971%EB%85%84" title="1971년">1971년</a> <a href="/wiki/%EB%B3%80%EB%A6%AC%EC%82%AC" title="변리사">변리사</a> 시험에 합격하여 특허법률사무소장으로 활동했다. 이후 정계에 입문하여 <a class="new" href="/w/index.php?title=%EC%82%AC%ED%9A%8C%EC%8B%9C%EB%AF%BC%EC%97%B0%ED%95%A9&amp;action=edit&amp;redlink=1" title="사회시민연합 (없는 문서)">사회시민연합</a>과 <a class="new" href="/w/index.php?title=%EC%82%AC%ED%9A%8C%EB%AF%BC%EC%A3%BC%EC%97%B0%ED%95%A9&amp;action=edit&amp;redlink=1" title="사회민주연합 (없는 문서)">사회민주연합</a> 부대표를 지냈다. <a href="/wiki/1996%EB%85%84" title="1996년">1996년</a> 1월에는 <a href="/wiki/%EB%AC%B4%EB%9D%BC%EC%95%BC%EB%A7%88_%EB%8F%84%EB%AF%B8%EC%9D%B4%EC%B9%98" title="무라야마 도미이치">무라야마 도미이치</a> 내각이 총사직을 하면서 <a href="/wiki/%ED%95%98%EC%8B%9C%EB%AA%A8%ED%86%A0_%EB%A5%98%ED%83%80%EB%A1%9C" title="하시모토 류타로">하시모토 류타로</a> 내각에서 후생상(74대, <a class="new" href="/w/index.php?title=%EC%A0%9C1%EC%B0%A8_%ED%95%98%EC%8B%9C%EB%AA%A8%ED%86%A0_%EB%82%B4%EA%B0%81&amp;action=edit&amp;redlink=1" title="제1차 하시모토 내각 (없는 문서)">제1차 하시모토 내각</a>)을 역임하였고, <a href="/wiki/%EC%8B%A0%EB%8B%B9_%EC%82%AC%ED%82%A4%EA%B0%80%EC%BC%80" title="신당 사키가케">신당 사키가케</a> 부대표, <a class="mw-redirect" href="/wiki/%EB%AF%BC%EC%A3%BC%EB%8B%B9_(%EC%9D%BC%EB%B3%B8)" title="민주당 (일본)">민주당</a> 대표(초대, 3대)를 지냈다.</p>
<h2><span class="mw-headline" id=".EC.A3.BC.EB.90.9C_.EC.A3.BC.EC.9E.A5.EA.B3.BC_.EC.A0.95.EC.B1.85">주된 주장과 정책</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EA%B0%84_%EB%82%98%EC%98%A4%ED%86%A0&amp;action=edit&amp;section=2" title="부분 편집: 주된 주장과 정책">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<ul>
<li>방위에 관해서는 자위대의 치안 활동, 국제 협력을 목적으로 하는 부대를 창설하는 정책을 가지고 있다.</li>
</ul>
<ul>
<li><a href="/wiki/%ED%83%80%EC%9D%B4%EC%99%84_%EB%AC%B8%EC%A0%9C" title="타이완 문제">타이완 문제</a>와 관해서는 <a href="/wiki/%EC%A4%91%ED%99%94%EC%9D%B8%EB%AF%BC%EA%B3%B5%ED%99%94%EA%B5%AD" title="중화인민공화국">중화인민공화국</a>이 <a href="/wiki/%EC%A4%91%ED%99%94%EB%AF%BC%EA%B5%AD" title="중화민국">중화민국</a>의 <a href="/wiki/%EC%9C%A0%EC%97%94" title="유엔">유엔</a> 가입을 허용하고 유엔에서 이를 평화적으로 해결하는 것을 목적으로 한다고 주장하고 있다.<sup class="reference" id="cite_ref-2"><a href="#cite_note-2">[2]</a></sup></li>
<li><a class="new" href="/w/index.php?title=%EC%A1%B0%EC%84%A0%EB%AF%BC%EC%A3%BC%EC%A3%BC%EC%9D%98%EC%9D%B8%EB%AF%BC%EA%B3%B5%ED%99%94%EA%B5%AD%EC%9D%98_%EC%9D%BC%EB%B3%B8%EC%9D%B8_%EB%82%A9%EC%B9%98%EB%AC%B8%EC%A0%9C&amp;action=edit&amp;redlink=1" title="조선민주주의인민공화국의 일본인 납치문제 (없는 문서)">조선민주주의인민공화국의 일본인 납치문제</a>와 핵문제에 관해서는 새로운 발상으로 해결해야 한다고 주장하였다. 간 총리는 신임의원 시절에 그의 선배 의원인 <a class="new" href="/w/index.php?title=%EB%8D%B4_%ED%9E%88%EB%8D%B0%EC%98%A4&amp;action=edit&amp;redlink=1" title="덴 히데오 (없는 문서)">덴 히데오</a>의 요구에 따라, ‘<a class="mw-disambig" href="/wiki/%EC%8B%A0%EA%B4%91%EC%88%98" title="신광수">재일 한국인 정치범 석방의 필요서</a>’에 서명하였다. 이 안에는 북한에 의한 일본인 납치문제의 용의자인 <a class="mw-disambig" href="/wiki/%EC%8B%A0%EA%B4%91%EC%88%98" title="신광수">신광수</a>를 시작으로 북한 스파이 용의자 10명이 포함 되어 있다. 이 사실을 사전에 알지 못한 사실은 공명당이나 사회당과 똑같이 서명을 한 의원으로서 비판받고 있다. 하지만 최근 북한의 관계된 언행에서는 강경한 편인데, <a href="/wiki/%EB%8B%88%EC%8B%9C%EB%AC%B4%EB%9D%BC_%EC%8B%A0%EA%B3%A0" title="니시무라 신고">니시무라 신고</a>의 지원유세를 할 때, “<a href="/wiki/%EC%82%AC%EB%8B%B4_%ED%9B%84%EC%84%B8%EC%9D%B8" title="사담 후세인">후세인</a>이나 <a href="/wiki/%EC%9D%B4%EC%98%A4%EC%8B%9C%ED%94%84_%EC%8A%A4%ED%83%88%EB%A6%B0" title="이오시프 스탈린">스탈린</a>의 동상이 무너진 역사 속에서 북한의 저 큰 동상도 무너지는 날도 올 것이라 확신합니다” 라는 발언을 하였다. 이 발언을 들은 북한 당국은 “간 나오토는 원래 신념도 정책도 아무것도 없는 정치인” 이라고 일방적으로 격하게 비난했다. 간 총리는 납치 문제에 있어서 납치 피해자 전원 송환, 핵문제 해결, 국교 정상화, 경제 지원이라는 것을 별도로 하는 것이 아니라 패키지로 함께 처리하는 정책을 주장하고 있다.</li>
</ul>
<ul>
<li>나가사키 <a class="new" href="/w/index.php?title=%EC%9D%B4%EC%82%AC%ED%95%98%EC%95%BC_%EB%A7%8C&amp;action=edit&amp;redlink=1" title="이사하야 만 (없는 문서)">이사하야 만</a> 간척 사업에 대해서는 자연환경보전의 관점에서의 반대를 하였다. 그리고 1997년 4월 수문이 닫히는 그 시점에 현지의 사업 사무소에 방문해서 TV 촬영 카메라 앞에서 “누구의 지시에 의해 하고 있는 것인가? 책임자는 나와야 한다”라고 말했다.</li>
</ul>
<ul>
<li>간 총리는 자민당 의원 중에서 세습을 많이 하는 것에 대해 비판을 하고 있지만, 자신에 관해서는 간 총리의 장남인 간 겐타로가 <a href="/wiki/%EC%A0%9C43%ED%9A%8C_%EC%9D%BC%EB%B3%B8_%EC%A4%91%EC%9D%98%EC%9B%90_%EC%9D%98%EC%9B%90_%EC%B4%9D%EC%84%A0%EA%B1%B0" title="제43회 일본 중의원 의원 총선거">2003년 중의원 총선거</a>에서 간 가문의 본적지인 오카야마 1구에서 민주당의 공천을 받아 입후보 하였다. 이 장남이 입후보한 것에 관해서 지역 기반이 크게 다르기 때문에 ‘지연, 간판, 학연’에 의한 세습의 폐해를 이어가지는 않아서 세습이 아니라고 말하였다. 민주당 내규에 따르면, 동일 선거구에서의 세습을 금지하고 있는데, 간 총리는 이를 따른 것으로 보인다.</li>
</ul>
<ul>
<li>헌법개정논의에 관해서는 대표 시절에 “일본이 나라로서의 모습을 보여주는 새로운 헌법을 만든다”라고 하여 ‘창헌’을 주장하였고, “헌법을 발표한 지 60년이 되는 2006년까지 국민적 운동을 집약하고, 민주당으로서 새로운 헌법을 국민에게 보여주고 싶다”라고 말하였다.<sup class="reference" id="cite_ref-3"><a href="#cite_note-3">[3]</a></sup></li>
</ul>
<ul>
<li>수상 재임 기간에 대해서는 “수상이라는 것은 적어도 4년은 같은 인물이 계속해야 한다. 평소 하토야마 총리에게도 말하고 있다”<sup class="reference" id="cite_ref-4"><a href="#cite_note-4">[4]</a></sup> 고 말하였다. 반면 참의원 선거에서 자민당이 크게 패한 이후 무너진 국회에 대해서는 “<a href="/wiki/%EC%95%84%EB%B2%A0_%EC%8B%A0%EC%A1%B0" title="아베 신조">아베</a> 수상이 총리 자리에 있는 한, 국민은 납득할 수 있는 정치가 아니다”<sup class="reference" id="cite_ref-5"><a href="#cite_note-5">[5]</a></sup> 와 같은 말로 빠른 시간 안에 사임을 하고 의회를 해산해야 한다고 발언했다.</li>
</ul>
<ul>
<li>미일 관계에 관해서는 “정치적으로는 미일 동맹을 생각하는 설계는 컸으며, 이제부터 아시아와 세계 안정을 위해서 중요한 관계”<sup class="reference" id="cite_ref-6"><a href="#cite_note-6">[6]</a></sup> 라고 말하였다.</li>
</ul>
<ul>
<li><a href="/wiki/%EC%98%A4%ED%82%A4%EB%82%98%EC%99%80_%ED%98%84" title="오키나와 현">오키나와 현</a>의 미군 후텐마기지 이전 문제와 관련해서는 “기지문제는 어떻게도 안 된다. 더 이상 손을 대기 싫다”라는 말이 민주당 참의원 <a class="new" href="/w/index.php?title=%EA%B8%B0%EB%82%98_%EC%87%BC%ED%82%A4%EC%B9%98&amp;action=edit&amp;redlink=1" title="기나 쇼키치 (없는 문서)">기나 쇼키치</a>가 2010년 6월 1일에 발매한 책 ‘오키나와의 자기결정권’에서 폭로되었다. ‘오키나와의 자기결정권’에 의하면 기나 쇼키치에게 ‘오키나와 문제는 어렵고, 어쩔 수 없다. 이제 오키나와는 독립하는 게 낫다’라고 말하였다고 한다.<sup class="reference" id="cite_ref-7"><a href="#cite_note-7">[7]</a></sup></li>
</ul>

<ul>
<li>조부: <a class="new" href="/w/index.php?title=%EA%B0%84_%EB%AF%B8%EB%85%B8%EB%A3%A8&amp;action=edit&amp;redlink=1" title="간 미노루 (없는 문서)">간 미노루</a>(菅實)
<ul>
