<p><b>간경변</b>(肝硬便, <span style="font-size: smaller;"><a href="/wiki/%EC%98%81%EC%96%B4" title="영어">영어</a>:</span> <span lang="en" xml:lang="en">Cirrhosis</span>)은 만성적인 <a href="/wiki/%EA%B0%84" title="간">간</a> 질병으로, 간의 조직 세포에 문제가 생겨 간이 굳고 오므라드는 것이 특징이다. <b>간경화</b>, <b>간섬유증</b>으로도 부른다. 손상된 조직이 재생함으로 말미암아 혹이 생기며<sup class="reference" id="cite_ref-1"><a href="#cite_note-1">[1]</a></sup><sup class="reference" id="cite_ref-2"><a href="#cite_note-2">[2]</a></sup><sup class="reference" id="cite_ref-3"><a href="#cite_note-3">[3]</a></sup>, 이로써 간의 기능이 쇠약해진다. 또, 온몸이 쇠약해지며 <a href="/wiki/%EB%B9%88%ED%98%88" title="빈혈">빈혈</a>을 일으킬 수 있다. <a href="/wiki/%EC%95%8C%EC%BD%94%EC%98%AC_%EC%9D%98%EC%A1%B4%EC%A6%9D" title="알코올 의존증">알코올 의존증</a>, <a href="/wiki/B%ED%98%95_%EA%B0%84%EC%97%BC" title="B형 간염">B형 간염</a>, <a href="/wiki/C%ED%98%95_%EA%B0%84%EC%97%BC" title="C형 간염">C형 간염</a>, <a class="new" href="/w/index.php?title=%EB%B9%84%EC%95%8C%EC%BD%94%EC%98%AC%EC%84%B1_%EC%A7%80%EB%B0%A9%EA%B0%84_%EC%A7%88%ED%99%98&amp;action=edit&amp;redlink=1" title="비알코올성 지방간 질환 (없는 문서)">비알코올성 지방간 질환</a>으로 인해 발병하는 것이 일반적이지만 그 밖의 다른 원인도 수없이 많다. 일부는 원인을 알 수 없는 특발성 질환이다.</p>
<p><a href="/wiki/%EB%B3%B5%EC%88%98_(%EC%A7%88%EB%B3%91)" title="복수 (질병)">복수</a>(腹水)가 차는 것이 일반적이며 낮은 삶의 질, 감염 위험의 증가 등과 관련된다. <a class="new" href="/w/index.php?title=%EC%8B%9D%EB%8F%84%EC%A0%95%EB%A7%A5%EB%A5%98&amp;action=edit&amp;redlink=1" title="식도정맥류 (없는 문서)">식도정맥류</a>로 인한 출혈, <a class="new" href="/w/index.php?title=%EA%B0%84%EC%84%B1%EB%87%8C%EC%A6%9D&amp;action=edit&amp;redlink=1" title="간성뇌증 (없는 문서)">간성뇌증</a>과도 관련될 수 있다. 간경변증은 일반적으로 돌이킬 수 없으며 치료는 보통 병의 진전과 합병증을 예방하는데 초점을 둔다. 간경변 후기의 경우 유일한 선택 사항은 <a class="new" href="/w/index.php?title=%EA%B0%84_%EC%9D%B4%EC%8B%9D&amp;action=edit&amp;redlink=1" title="간 이식 (없는 문서)">간 이식</a>이다.</p>
<p>또, 간경변은 광범위한 <a href="/wiki/%EA%B0%84" title="간">간</a> <a href="/wiki/%EC%84%B8%ED%8F%AC" title="세포">세포</a> 파괴의 결과로 <a class="new" href="/w/index.php?title=%EC%84%AC%EC%9C%A0%EC%A1%B0%EC%A7%81&amp;action=edit&amp;redlink=1" title="섬유조직 (없는 문서)">섬유조직</a>의 <a class="new" href="/w/index.php?title=%EC%A6%9D%EC%8B%9D&amp;action=edit&amp;redlink=1" title="증식 (없는 문서)">증식</a>과 <a class="new" href="/w/index.php?title=%EC%9E%AC%EC%83%9D%EC%84%B1_%EA%B2%B0%EC%A0%88&amp;action=edit&amp;redlink=1" title="재생성 결절 (없는 문서)">재생성 결절</a> 형성의 형대학적 특징을 보이며, 2차적으로 간내혈관의 변형 및 간기능의 저하가 초래되는 <a href="/wiki/%EC%A7%88%EB%B3%91" title="질병">질병</a>이다.<sup class="reference" id="cite_ref-snu_4-0"><a href="#cite_note-snu-4">[4]</a></sup></p>

<h2><span class="mw-headline" id=".EC.9B.90.EC.9D.B8.EA.B3.BC_.EC.A6.9D.EC.83.81">원인과 증상</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EA%B0%84%EA%B2%BD%EB%B3%80&amp;action=edit&amp;section=1" title="부분 편집: 원인과 증상">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>간경변은 <a href="/wiki/%EB%B0%94%EC%9D%B4%EB%9F%AC%EC%8A%A4" title="바이러스">바이러스</a>에 의한 <a href="/wiki/%EA%B0%84%EC%97%BC" title="간염">간염</a>의 악화, <a href="/wiki/%EC%88%A0" title="술">술</a>, <a class="mw-redirect" href="/wiki/%EB%B9%84%EB%A7%8C" title="비만">비만</a>, <a href="/wiki/%EB%8B%B9%EB%87%A8%EB%B3%91" title="당뇨병">당뇨병</a> 등으로 인해 발생한 만성 <a href="/wiki/%EC%A7%80%EB%B0%A9%EA%B0%84" title="지방간">지방간</a>의 악화 등으로 인해 발생한다. 간의 섬유화는 서서히 진행되어 상당한 경과가 있더라도 간기능 부전이 일어나기 전까지 자각증상이 없다. 따라서 간경변의 진단도 매우 어렵다. 증상은 온 몸이 나른하고 식욕이 없으며 헛배가 부르며 구역감이 생기고 설사 또는 변비로 고통을 호소하는 것으로 <a class="new" href="/w/index.php?title=%EB%A7%8C%EC%84%B1%EA%B0%84%EC%97%BC&amp;action=edit&amp;redlink=1" title="만성간염 (없는 문서)">만성간염</a>의 증세와 매우 유사하다. 오줌이 진해지고 <a href="/wiki/%ED%99%A9%EB%8B%AC" title="황달">황달</a>이 생기기도 한다. 복수가 고여 개구리배처럼 <a class="mw-redirect" href="/wiki/%EB%B0%B0_(%EB%8F%99%EB%AC%BC)" title="배 (동물)">배</a>가 부풀어 오르기도 한다.<sup class="reference" id="cite_ref-ksk_5-0"><a href="#cite_note-ksk-5">[5]</a></sup></p>
<p>간기능 부전은 간염과 간경변이 혼재되면서 나타나는데 다음과 같은 증상을 보인다.<sup class="reference" id="cite_ref-6"><a href="#cite_note-6">[6]</a></sup></p>
<ul>
<li><a href="/wiki/%ED%99%A9%EB%8B%AC" title="황달">황달</a>: 미세담도가 막혀 <a href="/wiki/%EC%A0%81%ED%98%88%EA%B5%AC" title="적혈구">적혈구</a>가 파괴된 <a href="/wiki/%EB%B9%8C%EB%A6%AC%EB%A3%A8%EB%B9%88" title="빌리루빈">빌리루빈</a>이 <a href="/wiki/%ED%94%BC%EB%B6%80" title="피부">피부</a>로 역류</li>
<li>복수(腹水): <a href="/wiki/%EC%95%8C%EB%B6%80%EB%AF%BC" title="알부민">알부민</a>이 합성되지 않아 혈중 수분이 <a href="/wiki/%EB%AA%B8" title="몸">몸</a>으로 빠져나가 부종을 일으킴</li>
<li>여성화: 여성호르몬이 파괴되지 않아 남성의 경우 유방 증대, 치모 손실, 고환 위축 등이 나타남</li>
<li>망상모반: <a href="/wiki/%EB%8F%99%EB%A7%A5" title="동맥">동맥</a> <a href="/wiki/%ED%98%88%EA%B4%80" title="혈관">혈관</a>이 방사형으로 피부에 돌출하여 손바닥에 홍반이 나타나고 얼굴, 목, 가슴에 거미상 혈관이 나타남</li>
<li>이외에 코피가 멎지 않는 출혈 증상, 암모니아 해독 불능으로 인한 혼수, 저혈당, 정맥류, 비장 비대, 신장염, 입에서 쥐오줌 냄새가 나는 구취 등의 증상이 있다.</li>
</ul>
<h2><span class="mw-headline" id=".EC.A7.84.EB.8B.A8.EA.B3.BC_.EC.B9.98.EB.A3.8C">진단과 치료</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EA%B0%84%EA%B2%BD%EB%B3%80&amp;action=edit&amp;section=2" title="부분 편집: 진단과 치료">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>원인을 밝히기 위해 <a href="/wiki/B%ED%98%95_%EA%B0%84%EC%97%BC" title="B형 간염">B형 간염</a> 바이러스 항원과 <a href="/wiki/C%ED%98%95_%EA%B0%84%EC%97%BC" title="C형 간염">C형 간염</a> 바이러스 항체 검사를 실시한다. 또한 간기능 검사, <a class="new" href="/w/index.php?title=%ED%94%84%EB%A1%9C%ED%8A%B8%EB%A1%AC%EB%B9%88&amp;action=edit&amp;redlink=1" title="프로트롬빈 (없는 문서)">프로트롬빈</a> 시간 측정, 식도위내시경 검사, 소변 검사, 복부 <a href="/wiki/%EC%9D%98%EB%A3%8C_%EC%B4%88%EC%9D%8C%ED%8C%8C" title="의료 초음파">초음파</a>, <a class="mw-disambig" href="/wiki/CT" title="CT">CT</a>, <a class="mw-redirect" href="/wiki/MRI" title="MRI">MRI</a>, <a class="new" href="/w/index.php?title=Fibroscan&amp;action=edit&amp;redlink=1" title="Fibroscan (없는 문서)">Fibroscan</a> 등을 통해 진단한다. 복수와 거미상혈관종, 수장홍반(手掌紅斑), 복부의 <a class="new" href="/w/index.php?title=%EC%B8%A1%EB%B6%80%EC%88%9C%ED%99%98%EC%A0%95%EB%A7%A5&amp;action=edit&amp;redlink=1" title="측부순환정맥 (없는 문서)">측부순환정맥</a> 발현과 같은 전형적 증상이 동반하면 간경변으로 의심할 수 있으며 조직검사를 통해 확진한다. 이 외에 비전형적인 경우 여러 가지 보조적인 진단법을 사용한다.<sup class="reference" id="cite_ref-snu_4-1"><a href="#cite_note-snu-4">[4]</a></sup></p>
<p>간경변은 섬유조직의 증식 및 결절성 재생을 나타내는 질환으로 정상상태로 되돌리는 것은 불가능하다.<sup class="reference" id="cite_ref-snu_4-2"><a href="#cite_note-snu-4">[4]</a></sup> 간경변의 치료는 완치를 목적으로 하는 것이 아닌 현재의 간기능을 최대한 활용할 수 있도록 하는 것이 고작이다.<sup class="reference" id="cite_ref-ksk_5-1"><a href="#cite_note-ksk-5">[5]</a></sup></p>
