<div class="mw-content-ltr" dir="ltr" id="mw-content-text" lang="ko"><script>function mfTempOpenSection(id){var block=document.getElementById("mf-section-"+id);block.className+=" open-block";block.previousSibling.className+=" open-block";}</script><div class="dablink"><a href="/wiki/%EC%9C%84%ED%82%A4%EB%B0%B1%EA%B3%BC:%EB%8F%99%EC%9D%8C%EC%9D%B4%EC%9D%98%EC%96%B4_%EB%AC%B8%EC%84%9C" title="위키백과:동음이의어 문서"><img alt="" data-file-height="200" data-file-width="260" height="18" src="//upload.wikimedia.org/wikipedia/commons/thumb/4/4a/Disambig_grey.svg/23px-Disambig_grey.svg.png" srcset="//upload.wikimedia.org/wikipedia/commons/thumb/4/4a/Disambig_grey.svg/35px-Disambig_grey.svg.png 1.5x, //upload.wikimedia.org/wikipedia/commons/thumb/4/4a/Disambig_grey.svg/46px-Disambig_grey.svg.png 2x" width="23"/></a> <b>북간도</b>는 여기로 연결됩니다. 소설에 대해서는 <a class="new" href="/w/index.php?title=%EB%B6%81%EA%B0%84%EB%8F%84_(%EC%86%8C%EC%84%A4)&amp;action=edit&amp;redlink=1" title="북간도 (소설) (없는 문서)">북간도 (소설)</a> 문서를 참조하십시오.</div>
<p><b>간도</b>(間島, <small><a href="/wiki/%EC%A4%91%EA%B5%AD%EC%96%B4" title="중국어">중국어</a></small> <small><a href="/wiki/%EA%B0%84%EC%B2%B4%EC%9E%90" title="간체자">간체</a>:</small> <span lang="zh-hans" xml:lang="zh-hans">间岛</span>, <small><a href="/wiki/%ED%95%9C%EC%96%B4_%EB%B3%91%EC%9D%8C" title="한어 병음">병음</a>:</small> <font face="Lucida Sans Unicode">Jiāndǎo</font>,  <small>젠다오<sup>[<a href="/wiki/%EC%9C%84%ED%82%A4%EB%B0%B1%EA%B3%BC:%EC%A4%91%EA%B5%AD%EC%96%B4%EC%9D%98_%ED%95%9C%EA%B8%80_%ED%91%9C%EA%B8%B0" title="위키백과:중국어의 한글 표기">*</a>]</sup></small>)는 <a href="/wiki/%EC%95%95%EB%A1%9D%EA%B0%95" title="압록강">압록강</a> 상류와 <a href="/wiki/%EB%91%90%EB%A7%8C%EA%B0%95" title="두만강">두만강</a> 북쪽의 조선인 거주 지역을 일컫는 말로, 간도의 범위에 관하여는 여러 이견이 있다. 일반적으로 간도라 하면 현재의 <a class="mw-redirect" href="/wiki/%EC%97%B0%EB%B3%80_%EC%A1%B0%EC%84%A0%EC%A1%B1_%EC%9E%90%EC%B9%98%EC%A3%BC" title="연변 조선족 자치주">연변 조선족 자치주</a> 지역을 가리키며, <a href="/wiki/%EB%91%90%EB%A7%8C%EA%B0%95" title="두만강">두만강</a> 북쪽인 <a class="mw-redirect" href="/wiki/%EC%97%B0%EB%B3%80" title="연변">연변</a> 지역을 '북간도'(또는 '동간도'), 그 서쪽인 <a href="/wiki/%EC%95%95%EB%A1%9D%EA%B0%95" title="압록강">압록강</a> 북쪽 지역을 '서간도'라 부르기도 한다.</p>
<p>간도(間島)는 글자 그대로 풀이하면 <b>사이섬</b>(사잇섬)으로, 그 어원에 대해서는 의견이 분분한데, 그 가운데 '사잇섬'이란 말 뜻에 비추어 '간도'가 본래는 <a href="/wiki/%EC%95%95%EB%A1%9D%EA%B0%95" title="압록강">압록강</a>과 <a href="/wiki/%EB%91%90%EB%A7%8C%EA%B0%95" title="두만강">두만강</a>의 <a href="/wiki/%ED%95%98%EC%A4%91%EB%8F%84" title="하중도">하중도</a>(河中島)를 가리키는 말이었다가 두 강의 북안(北岸)을 가리키는 말로 그 의미가 확장·변형된 것이라 보는 견해가 유력하다.<sup class="reference" id="cite_ref-1"><a href="#cite_note-1">[1]</a></sup></p>
<p><a href="/wiki/19%EC%84%B8%EA%B8%B0" title="19세기">19세기</a> 말, <a href="/wiki/%EC%A1%B0%EC%84%A0" title="조선">조선</a>과 <a href="/wiki/%EC%B2%AD%EB%82%98%EB%9D%BC" title="청나라">청나라</a>가 <a href="/wiki/%EB%B0%B1%EB%91%90%EC%82%B0%EC%A0%95%EA%B3%84%EB%B9%84" title="백두산정계비">백두산정계비</a>에 쓰여진 "토문(土門)"을 서로 달리 해석하면서 이 지역(북간도)에 대한 귀속 문제가 불거졌다. 당시 <a href="/wiki/%EC%A1%B0%EC%84%A0" title="조선">조선</a>은 <a class="mw-redirect" href="/wiki/%EC%A0%95%EA%B3%84%EB%B9%84" title="정계비">정계비</a>의 위치상 '토문은 <a href="/wiki/%EB%91%90%EB%A7%8C%EA%B0%95" title="두만강">두만강</a>과 별개의 강이다'고 주장하였고, <a href="/wiki/1903%EB%85%84" title="1903년">1903년</a>에는 <a href="/wiki/%EC%9D%B4%EB%B2%94%EC%9C%A4" title="이범윤">이범윤</a>을 간도관리사로 파견하여 간도에 대한 실효적 지배를 시도하기도 했다.</p>

<h3><span class="mw-headline" id="18.EC.84.B8.EA.B8.B0.EA.B9.8C.EC.A7.80">18세기까지</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EA%B0%84%EB%8F%84&amp;action=edit&amp;section=2" title="부분 편집: 18세기까지">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p><b>간도</b>(間島)는 <a class="mw-redirect" href="/wiki/%EA%B3%A0%EB%8C%80" title="고대">고대</a>에는 <a href="/wiki/%EB%B6%80%EC%97%AC" title="부여">부여</a>와 <a href="/wiki/%EB%B6%81%EC%98%A5%EC%A0%80" title="북옥저">북옥저</a>, <a href="/wiki/%EA%B3%A0%EA%B5%AC%EB%A0%A4" title="고구려">고구려</a>, <a href="/wiki/%EB%B0%9C%ED%95%B4" title="발해">발해</a>의 영역이었다가 <a href="/wiki/%EA%B3%A0%EB%A0%A4" title="고려">고려</a>시대부터 <a href="/wiki/%EC%A1%B0%EC%84%A0" title="조선">조선</a> 중기까지는 <a class="mw-redirect" href="/wiki/%EC%97%AC%EC%A7%84%EC%A1%B1" title="여진족">여진족</a>이 흩어져 살았다. <a href="/wiki/%EC%A1%B0%EC%84%A0" title="조선">조선</a>은 건국 초 <a class="mw-redirect" href="/wiki/%EC%84%B8%EC%A2%85%EB%8C%80%EC%99%95" title="세종대왕">세종대왕</a> 집권기에 <a href="/wiki/%EC%95%95%EB%A1%9D%EA%B0%95" title="압록강">압록강</a>과 <a href="/wiki/%EB%91%90%EB%A7%8C%EA%B0%95" title="두만강">두만강</a> 남쪽의 여진족을 쫓아내거나 귀화시켜 <a class="mw-redirect mw-disambig" href="/wiki/4%EA%B5%B0" title="4군">4군</a>과 <a class="mw-redirect" href="/wiki/6%EC%A7%84" title="6진">6진</a>을 설치했으나, 두 강의 북쪽으로 영토를 밀어올리지는 않았다.</p>
<p><a href="/wiki/1616%EB%85%84" title="1616년">1616년</a>에 <a class="mw-redirect" href="/wiki/%EB%88%84%EB%A5%B4%ED%95%98%EC%B9%98" title="누르하치">누르하치</a>가 <a href="/wiki/%ED%9B%84%EA%B8%88" title="후금">후금</a>을 세운 후 <a href="/wiki/1644%EB%85%84" title="1644년">1644년</a>에 <a href="/wiki/%EC%B2%AD%EB%82%98%EB%9D%BC" title="청나라">청나라</a>가 <a href="/wiki/%EB%AA%85%EB%82%98%EB%9D%BC" title="명나라">명나라</a>를 멸망시키면서, <a href="/wiki/%EB%A7%8C%EC%A3%BC%EC%A1%B1" title="만주족">만주족</a>(여진족의 후신)의 상당수가 군사들을 따라 이 지역에서 중국 본토로 옮겨갔다. 이 과정에서 본래 만주족들이 살던 지역이 공동화(空洞化)되자, <a href="/wiki/%EC%B2%AD%EB%82%98%EB%9D%BC" title="청나라">청</a>(淸)은 <a href="/wiki/%EA%B0%95%ED%9D%AC%EC%A0%9C" title="강희제">강희제</a> 집권기인 <a href="/wiki/1677%EB%85%84" title="1677년">1677년</a>에 '<a href="/wiki/%ED%9D%A5%EA%B2%BD" title="흥경">흥경</a> 이동, <a class="mw-redirect" href="/wiki/%EC%9D%B4%ED%86%B5%EC%A3%BC" title="이통주">이통주</a> 이남, <a href="/wiki/%EB%B0%B1%EB%91%90%EC%82%B0" title="백두산">백두산</a>·<a href="/wiki/%EC%95%95%EB%A1%9D%EA%B0%95" title="압록강">압록강</a>·<a href="/wiki/%EB%91%90%EB%A7%8C%EA%B0%95" title="두만강">두만강</a> 이북 지역'을 <a class="mw-redirect" href="/wiki/%EC%B2%AD%EC%A1%B0" title="청조">청조</a>의 발상지라 하여 <b>봉금지</b>(封禁地)로 정하고 <a href="/wiki/%EB%A7%8C%EC%A3%BC%EC%A1%B1" title="만주족">만주족</a>이 아닌 타민족의 거주와 개간, 삼림 벌채, 인삼 채취 등을 엄금하였다.<sup class="reference" id="cite_ref-2"><a href="#cite_note-2">[2]</a></sup> <a href="/wiki/1636%EB%85%84" title="1636년">1636년</a>에 <a href="/wiki/%EB%B3%91%EC%9E%90%ED%98%B8%EB%9E%80" title="병자호란">병자호란</a>을 겪으며 청의 위세를 실감한 <a href="/wiki/%EC%A1%B0%EC%84%A0" title="조선">조선</a> 역시 <a href="/wiki/%EC%95%95%EB%A1%9D%EA%B0%95" title="압록강">압록강</a>과 <a href="/wiki/%EB%91%90%EB%A7%8C%EA%B0%95" title="두만강">두만강</a>의 북쪽 연안에 대한 도강을 엄금하고 '월강죄'로 다스렸다.<sup class="reference" id="cite_ref-3"><a href="#cite_note-3">[3]</a></sup></p>
<p>간도는 두만강과 그 지류인 해란강(海蘭江), 가야하(嘎呀河, 알아하), 훈춘하(琿春河, 혼춘하) 등 여러 물길의 연안을 중심으로 한 분지와 구릉으로서 땅이 기름지고 산림이 무성한 땅이었으나, 만주족은 농경보다 유목·수렵에 종사하였기 때문에 이 지역은 오랫동안 개간되지 못했다. 게다가 청나라의 봉금정책으로 인해 150년 가까이 이 땅은 청조 통치자들을 위한 수렵지이자 삼(蔘) 등 약재의 채집지로 독점적으로 이용되는 상황이었다.</p>
<h3><span class="mw-headline" id="19.EC.84.B8.EA.B8.B0_.EC.9D.B4.ED.9B.84">19세기 이후</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EA%B0%84%EB%8F%84&amp;action=edit&amp;section=3" title="부분 편집: 19세기 이후">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p><a href="/wiki/19%EC%84%B8%EA%B8%B0" title="19세기">19세기</a>에 들어서 서슬퍼런 봉금령도 부패와 혼란으로 균열이 가기 시작했다. <a href="/wiki/1800%EB%85%84" title="1800년">1800년</a> <a href="/wiki/%EC%A1%B0%EC%84%A0" title="조선">조선</a>은 왕권 강화를 위한 개혁정책을 추진하던 <a class="mw-redirect" href="/wiki/%EC%A0%95%EC%A1%B0" title="정조">정조</a>가 사망하면서 <a class="mw-redirect" href="/wiki/%EC%84%B8%EB%8F%84%EC%A0%95%EC%B9%98" title="세도정치">세도정치</a>의 전횡과 부패로 민생이 도탄에 빠지고, <a href="/wiki/%EC%B2%AD%EB%82%98%EB%9D%BC" title="청나라">청나라</a> 역시 <a href="/wiki/%EA%B1%B4%EB%A5%AD%EC%A0%9C" title="건륭제">건륭제</a> 말년에 <a href="/wiki/%ED%99%94%EC%8B%A0_(%EC%B2%AD%EB%82%98%EB%9D%BC)" title="화신 (청나라)">화신</a>의 전횡과 부패로 국정이 문란해지면서 <a href="/wiki/%EB%B0%B1%EB%A0%A8%EA%B5%90%EB%8F%84%EC%9D%98_%EB%82%9C" title="백련교도의 난">백련교도의 난</a>(<a href="/wiki/1796%EB%85%84" title="1796년">1796년</a> - <a href="/wiki/1804%EB%85%84" title="1804년">1804년</a>) 등 거듭되는 반란이 일어나 국운이 기울기 시작했다.</p>
<p><a href="/wiki/19%EC%84%B8%EA%B8%B0" title="19세기">19세기</a> 초부터 <a class="mw-redirect" href="/wiki/%EC%84%B8%EB%8F%84%EC%A0%95%EC%B9%98" title="세도정치">세도정치</a>의 학정(虐政)과 지방 수령의 수탈을 견디다 못한 조선의 농민들이 봉금령에도 불구하고 조선의 관권(官權)이 미치지 않는 <a href="/wiki/%EB%91%90%EB%A7%8C%EA%B0%95" title="두만강">두만강</a> 이북으로 건너가 이주하기 시작했다. 게다가 <a href="/wiki/%EC%B2%AD%EB%82%98%EB%9D%BC" title="청나라">청나라</a>가 <a class="mw-redirect" href="/wiki/%EC%95%84%ED%8E%B8%EC%A0%84%EC%9F%81" title="아편전쟁">아편전쟁</a>(<a href="/wiki/1839%EB%85%84" title="1839년">1839년</a> - <a href="/wiki/1842%EB%85%84" title="1842년">1842년</a> / <a href="/wiki/1856%EB%85%84" title="1856년">1856년</a> - <a href="/wiki/1860%EB%85%84" title="1860년">1860년</a>)과 <a href="/wiki/%ED%83%9C%ED%8F%89%EC%B2%9C%EA%B5%AD%EC%9D%98_%EB%82%9C" title="태평천국의 난">태평천국의 난</a>(<a href="/wiki/1851%EB%85%84" title="1851년">1851년</a> - <a href="/wiki/1864%EB%85%84" title="1864년">1864년</a>)으로 혼란에 빠져 월강에 대한 단속이 느슨해지고 <a href="/wiki/%EC%A1%B0%EC%84%A0" title="조선">조선</a>의 <a href="/wiki/%ED%95%A8%EA%B2%BD%EB%8F%84" title="함경도">함경도</a> 지방에서 <a href="/wiki/1869%EB%85%84" title="1869년">1869년</a>(기사대흉년), <a href="/wiki/1870%EB%85%84" title="1870년">1870년</a>(경오대흉년) 대흉년이 들자 수많은 조선사람들이 <a href="/wiki/%EC%95%95%EB%A1%9D%EA%B0%95" title="압록강">압록강</a> 상류와 <a href="/wiki/%EB%91%90%EB%A7%8C%EA%B0%95" title="두만강">두만강</a>을 건너 단속 자체가 어려운 지경에 이르게 되었다. 그러나, 여전히 <a href="/wiki/%EC%B2%AD%EB%82%98%EB%9D%BC" title="청나라">청</a>의 봉금령은 계속되었기 때문에 그 단속을 피해 간도에서 <a class="mw-redirect" href="/wiki/%EC%97%B0%ED%95%B4%EC%A3%BC" title="연해주">연해주</a>로 다시 이주하는 주민들도 있었지만 이주민은 줄어들기는커녕 갈수록 늘어났다.</p>
<p><a href="/wiki/1881%EB%85%84" title="1881년">1881년</a>에는 청나라가 '봉금령'을 폐지하였고, <a href="/wiki/1883%EB%85%84" title="1883년">1883년</a>에 조선이 '월강금지령'을 폐지한 데 이어 <a href="/wiki/1885%EB%85%84" title="1885년">1885년</a>에 청나라가 조선인에 대한 만주 이주 금지령을 철폐하면서 조선인의 간도 유입 현상은 더욱 증가하였다.</p>
<p><a href="/wiki/20%EC%84%B8%EA%B8%B0" title="20세기">20세기</a> 초에도 <a href="/wiki/%EC%9D%BC%EB%B3%B8_%EC%A0%9C%EA%B5%AD" title="일본 제국">일본 제국</a>의 침략과 수탈에서 벗어나기 위해서 또는 항일 운동을 위해서 많은 조선인들이 이 곳으로 계속 이주했다. <a href="/wiki/1932%EB%85%84" title="1932년">1932년</a>에 일본 제국은 만주에 괴뢰 국가인 <a href="/wiki/%EB%A7%8C%EC%A3%BC%EA%B5%AD" title="만주국">만주국</a>을 세우고 간도를 관할하는 <a href="/wiki/%EC%A0%A0%EB%8B%A4%EC%98%A4_%EC%84%B1" title="젠다오 성">젠다오 성</a>(間島省, 간도성)을 설치했다. 간도는 <a href="/wiki/1949%EB%85%84" title="1949년">1949년</a>에 <a href="/wiki/%EC%A4%91%ED%99%94%EC%9D%B8%EB%AF%BC%EA%B3%B5%ED%99%94%EA%B5%AD" title="중화인민공화국">중화인민공화국</a>의 통치 하에 들어갔고, 현재는 <a class="mw-redirect" href="/wiki/%EC%97%B0%EB%B3%80_%EC%A1%B0%EC%84%A0%EC%A1%B1_%EC%9E%90%EC%B9%98%EC%A3%BC" title="연변 조선족 자치주">연변 조선족 자치주</a>와 <a class="mw-redirect" href="/wiki/%EC%9E%A5%EB%B0%B1_%EC%A1%B0%EC%84%A0%EC%A1%B1_%EC%9E%90%EC%B9%98%ED%98%84" title="장백 조선족 자치현">장백 조선족 자치현</a> 등이 중국 행정구역으로 편제되어 있다.</p>
<h2><span class="mw-headline" id=".EB.B0.B1.EB.91.90.EC.82.B0_.EC.A0.95.EA.B3.84.EB.B9.84">백두산 정계비</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EA%B0%84%EB%8F%84&amp;action=edit&amp;section=4" title="부분 편집: 백두산 정계비">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<div class="rellink relarticle mainarticle"><img alt="" data-file-height="128" data-file-width="128" height="16" src="//upload.wikimedia.org/wikipedia/commons/thumb/e/ec/Crystal_Clear_app_xmag.svg/16px-Crystal_Clear_app_xmag.svg.png" srcset="//upload.wikimedia.org/wikipedia/commons/thumb/e/ec/Crystal_Clear_app_xmag.svg/24px-Crystal_Clear_app_xmag.svg.png 1.5x, //upload.wikimedia.org/wikipedia/commons/thumb/e/ec/Crystal_Clear_app_xmag.svg/32px-Crystal_Clear_app_xmag.svg.png 2x" width="16"/> 이 부분의 본문은 <a href="/wiki/%EB%B0%B1%EB%91%90%EC%82%B0%EC%A0%95%EA%B3%84%EB%B9%84" title="백두산정계비">백두산정계비</a>입니다.</div>
<p><a href="/wiki/1712%EB%85%84" title="1712년">1712년</a>(<a href="/wiki/%EC%A1%B0%EC%84%A0_%EC%88%99%EC%A2%85" title="조선 숙종">숙종</a> 38년), <a href="/wiki/%EB%B0%B1%EB%91%90%EC%82%B0" title="백두산">백두산</a> <a href="/wiki/%EC%B2%9C%EC%A7%80" title="천지">천지</a>(天池)의 남동쪽 4 km 지점에 <a href="/wiki/%EC%A1%B0%EC%84%A0" title="조선">조선</a>과 <a href="/wiki/%EC%B2%AD%EB%82%98%EB%9D%BC" title="청나라">청나라</a>의 <a href="/wiki/%EA%B5%AD%EA%B2%BD" title="국경">국경</a>을 확인하는 <a href="/wiki/%EB%B0%B1%EB%91%90%EC%82%B0%EC%A0%95%EA%B3%84%EB%B9%84" title="백두산정계비">백두산정계비</a>가 세워졌다. <a class="mw-redirect" href="/wiki/%EC%A0%95%EA%B3%84%EB%B9%84" title="정계비">정계비</a>에는 <a href="/wiki/%EC%A1%B0%EC%84%A0" title="조선">조</a>·<a href="/wiki/%EC%B2%AD%EB%82%98%EB%9D%BC" title="청나라">청</a> 양국의 경계를 '서위압록 동위토문(西爲鴨綠 東爲土門)'이라 하고, 정계비는 그 분수령에 세워졌음을 명기하였다.<sup class="reference" id="cite_ref-4"><a href="#cite_note-4">[4]</a></sup> <sup class="reference" id="cite_ref-5"><a href="#cite_note-5">[5]</a></sup> 그러나, 청(淸)의 <a class="new" href="/w/index.php?title=%EB%AA%A9%EA%B7%B9%EB%93%B1&amp;action=edit&amp;redlink=1" title="목극등 (없는 문서)">목극등</a>(穆克登)이 사계(査界)를 한 이후에 조선 측은 '정계비로부터 동쪽 수계(水界)까지' 설책(設柵)을 하는 과정에서 <a class="new" href="/w/index.php?title=%EB%AA%A9%EA%B7%B9%EB%93%B1&amp;action=edit&amp;redlink=1" title="목극등 (없는 문서)">목극등</a>이 정한 수계가 <a href="/wiki/%EB%91%90%EB%A7%8C%EA%B0%95" title="두만강">두만강</a>이 아닌 <a class="mw-redirect" href="/wiki/%EC%86%A1%ED%99%94%EA%B0%95" title="송화강">송화강</a>으로 흘러들어가는 문제를 발견하였다. 이에 조정에서 파견한 북평사는 설책 공사를 중지하라고 하였지만, 정계(定界)에 참여한 이들이 정계를 잘못한 책임을 지는 것이 두려워 <a class="new" href="/w/index.php?title=%EB%AA%A9%EA%B7%B9%EB%93%B1&amp;action=edit&amp;redlink=1" title="목극등 (없는 문서)">목극등</a>이 정한 수원(水源)에서 남쪽으로 20리 떨어진 곳에 새롭게 설책하였다.<sup class="reference" id="cite_ref-6"><a href="#cite_note-6">[6]</a></sup> <a href="/wiki/%EC%A1%B0%EC%84%A0" title="조선">조선</a> 조정은 이후에 이 사실을 알게 되었지만, 이를 청나라가 알게 되면 <a class="new" href="/w/index.php?title=%EB%AA%A9%EA%B7%B9%EB%93%B1&amp;action=edit&amp;redlink=1" title="목극등 (없는 문서)">목극등</a>이 견책 받고 다른 청나라 사신이 와서 영토가 축소될 수도 있다는 우려 때문에 이런 상황을 그대로 두었다.<sup class="reference" id="cite_ref-7"><a href="#cite_note-7">[7]</a></sup> <sup class="reference" id="cite_ref-8"><a href="#cite_note-8">[8]</a></sup> <sup class="reference" id="cite_ref-9"><a href="#cite_note-9">[9]</a></sup> <sup class="reference" id="cite_ref-10"><a href="#cite_note-10">[10]</a></sup></p>
<h2><span class="mw-headline" id=".EC.98.81.EC.9C.A0.EA.B6.8C_.EB.B6.84.EC.9F.81">영유권 분쟁</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EA%B0%84%EB%8F%84&amp;action=edit&amp;section=5" title="부분 편집: 영유권 분쟁">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<h3><span class="mw-headline" id="19.EC.84.B8.EA.B8.B0_.EB.A7.90.EC.9D.98_.EC.83.81.ED.99.A9">19세기 말의 상황</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EA%B0%84%EB%8F%84&amp;action=edit&amp;section=6" title="부분 편집: 19세기 말의 상황">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p><a href="/wiki/1800%EB%85%84%EB%8C%80" title="1800년대">1800년대</a> 초부터 <a href="/wiki/%EC%A1%B0%EC%84%A0" title="조선">조선</a>의 <a href="/wiki/%ED%95%A8%EA%B2%BD%EB%8F%84" title="함경도">함경도</a> 사람들이 <a href="/wiki/%EB%91%90%EB%A7%8C%EA%B0%95" title="두만강">두만강</a>을 건너 <b>간도</b>(間島)로 이주하기 시작하여 <a href="/wiki/1881%EB%85%84" title="1881년">1881년</a>에는 <a class="mw-redirect" href="/wiki/%EC%97%B0%EB%B3%80" title="연변">연변</a> 지역의 조선인이 1만명에 이르렀다. <a href="/wiki/1860%EB%85%84" title="1860년">1860년</a> <a href="/wiki/%EB%B2%A0%EC%9D%B4%EC%A7%95_%EC%A1%B0%EC%95%BD" title="베이징 조약">베이징 조약</a>으로 러시아에 연해주를 빼앗긴 <a href="/wiki/%EC%B2%AD%EB%82%98%EB%9D%BC" title="청나라">청나라</a>는 만주 개발을 위해 <a href="/wiki/1881%EB%85%84" title="1881년">1881년</a> '봉금령'을 폐지하고 본토 주민들이 이 지역으로 이주하도록 하였다. 이 과정에서 간도에 있던 조선인과 청인 사이에 마찰이 생기면서 조선과 청나라 사이에 간도에 대한 영유권 문제가 본격적으로 대두되었다.</p>
<p>이에 조선은 <a href="/wiki/1883%EB%85%84" title="1883년">1883년</a>에 '월강금지령'을 폐지하고 <a href="/wiki/%EC%96%B4%EC%9C%A4%EC%A4%91" title="어윤중">어윤중</a>·<a class="mw-disambig" href="/wiki/%EA%B9%80%EC%9A%B0%EC%8B%9D" title="김우식">김우식</a>에게 정계비와 그 주변 지형을 조사하게 하여 <a class="mw-redirect" href="/wiki/%EC%86%A1%ED%99%94%EA%B0%95" title="송화강">송화강</a>의 한 지류로 토문강이 있음을 확인한 뒤, 간도가 조선의 영토임을 주장하였다. 그러나, <a href="/wiki/%EC%B2%AD%EB%82%98%EB%9D%BC" title="청나라">청나라</a>는 양국의 기본적인 <a class="mw-redirect" href="/wiki/%EA%B5%AD%EA%B2%BD%EC%84%A0" title="국경선">국경선</a>이 <a href="/wiki/%EB%91%90%EB%A7%8C%EA%B0%95" title="두만강">두만강</a>이라는 전제하에 도문강(<a href="/wiki/%EB%91%90%EB%A7%8C%EA%B0%95" title="두만강">두만강</a>)의 도문(圖們)과 토문(土門)은 모두 <a href="/wiki/%EB%A7%8C%EC%A3%BC%EC%96%B4" title="만주어">만주어</a>에서 그 음이 비슷하기 때문에 취한 것(借字, 차자)이므로 '토문은 곧 두만강을 지칭한다'고 주장하였다.</p>
<p><a href="/wiki/%EC%A1%B0%EC%84%A0" title="조선">조선</a>과 <a href="/wiki/%EC%B2%AD%EB%82%98%EB%9D%BC" title="청나라">청나라</a>는 을유년(1885년)과 정해년(1887년)에 <a href="/wiki/%EB%B0%B1%EB%91%90%EC%82%B0" title="백두산">백두산</a>과 그 동쪽의 국경을 명확히 획정하기 위한 감계(勘界) 회담을 가졌으나, 서로의 주장이 엇갈려 모두 결렬되었다.<sup class="reference" id="cite_ref-11"><a href="#cite_note-11">[11]</a></sup></p>
<h3><span class="mw-headline" id="20.EC.84.B8.EA.B8.B0_.EC.9D.B4.ED.9B.84">20세기 이후</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EA%B0%84%EB%8F%84&amp;action=edit&amp;section=7" title="부분 편집: 20세기 이후">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p><a href="/wiki/1903%EB%85%84" title="1903년">1903년</a>(<a href="/wiki/%EA%B4%91%EB%AC%B4_(%EC%97%B0%ED%98%B8)" title="광무 (연호)">광무</a> 7년) <a href="/wiki/%EB%8C%80%ED%95%9C%EC%A0%9C%EA%B5%AD" title="대한제국">대한제국</a>은 간도에 대한 영유권을 주장하는 동시에 간도관리사 <a href="/wiki/%EC%9D%B4%EB%B2%94%EC%9C%A4" title="이범윤">이범윤</a>을 간도에 파견(1903년 10월 - 1905년 5월)하기도 했다. 당시 청나라는 <a class="mw-redirect" href="/wiki/%EC%9D%98%ED%99%94%EB%8B%A8_%EC%82%AC%EA%B1%B4" title="의화단 사건">의화단 사건</a>의 여파로 만주 일대를 <a href="/wiki/%EB%9F%AC%EC%8B%9C%EC%95%84_%EC%A0%9C%EA%B5%AD" title="러시아 제국">러시아 제국</a>에 점령당하다시피 한 상태였다. 이러한 러시아 제국의 남진은 <a href="/wiki/1904%EB%85%84" title="1904년">1904년</a>에 발발한 <a href="/wiki/%EB%9F%AC%EC%9D%BC_%EC%A0%84%EC%9F%81" title="러일 전쟁">러일 전쟁</a>의 불씨가 되었다.</p>
<p>일본 제국은 <a href="/wiki/1907%EB%85%84" title="1907년">1907년</a> <a href="/wiki/8%EC%9B%94_23%EC%9D%BC" title="8월 23일">8월 23일</a>, 간도에 헌병과 경찰을 들여보내 <a class="mw-redirect" href="/wiki/%EC%9A%A9%EC%A0%95" title="용정">용정</a>(龍井)에 <a href="/wiki/%ED%86%B5%EA%B0%90%EB%B6%80" title="통감부">통감부</a> 간도파출소를 설치하였으나, <a href="/wiki/1909%EB%85%84" title="1909년">1909년</a> <a href="/wiki/9%EC%9B%94_4%EC%9D%BC" title="9월 4일">9월 4일</a> 청나라와 <a href="/wiki/%EA%B0%84%EB%8F%84_%ED%98%91%EC%95%BD" title="간도 협약">간도 협약</a>을 체결하여 이 지역에 대한 청나라의 법적 권한을 인정하고 파출소를 철수하였다.(<a href="/wiki/%EA%B0%84%EB%8F%84_%ED%98%91%EC%95%BD" title="간도 협약">간도 협약</a>이 체결된 당일, 청나라와 일본 제국은 <a class="new" href="/w/index.php?title=%EB%A7%8C%EC%A3%BC_5%EC%95%88%EA%B1%B4_%ED%98%91%EC%95%BD&amp;action=edit&amp;redlink=1" title="만주 5안건 협약 (없는 문서)">만주 5안건 협약</a>을 체결하였다.)</p>
<p><a href="/wiki/1962%EB%85%84" title="1962년">1962년</a> <a href="/wiki/10%EC%9B%94_12%EC%9D%BC" title="10월 12일">10월 12일</a> <a href="/wiki/%EC%A1%B0%EC%84%A0%EB%AF%BC%EC%A3%BC%EC%A3%BC%EC%9D%98%EC%9D%B8%EB%AF%BC%EA%B3%B5%ED%99%94%EA%B5%AD" title="조선민주주의인민공화국">조선민주주의인민공화국</a>과 <a href="/wiki/%EC%A4%91%ED%99%94%EC%9D%B8%EB%AF%BC%EA%B3%B5%ED%99%94%EA%B5%AD" title="중화인민공화국">중화인민공화국</a>은 <a href="/wiki/%EC%A1%B0%EC%A4%91_%EB%B3%80%EA%B3%84_%EC%A1%B0%EC%95%BD" title="조중 변계 조약">조중 변계 조약</a>을 체결하여 <a href="/wiki/%EB%B0%B1%EB%91%90%EC%82%B0" title="백두산">백두산</a>과 <a href="/wiki/%EB%91%90%EB%A7%8C%EA%B0%95" title="두만강">두만강</a> 상류의 국경선을 명확히 획정하고, <a href="/wiki/%EB%91%90%EB%A7%8C%EA%B0%95" title="두만강">두만강</a> 이북 지역에 대한 영토권이 중화인민공화국에 있음을 확인하였다.</p>
<p>2000년대에는 <a href="/wiki/%ED%95%9C%EA%B5%AD" title="한국">한국</a> 사회에서 <a href="/wiki/%EA%B0%84%EB%8F%84_%ED%98%91%EC%95%BD" title="간도 협약">간도 협약</a>에 대한 이의를 제기할 수 있는 시한이 100년이기 때문에 2009년 9월에 그 시효가 완성된다는 일방적 주장이 신뢰할 만한 근거 제시 없이 사실인냥 유포되는 소동이 있기도 했다.<sup class="reference" id="cite_ref-12"><a href="#cite_note-12">[12]</a></sup> <sup class="reference" id="cite_ref-13"><a href="#cite_note-13">[13]</a></sup></p>
