<p><b>간린마루</b>(咸臨丸)는 <a href="/wiki/%EC%97%90%EB%8F%84_%EB%A7%89%EB%B6%80" title="에도 막부">에도 막부</a> <a href="/wiki/%EB%A7%89%EB%A7%90" title="막말">말기</a>에 막부가 보유하고 있던 초기 군함이다. 목조를 이용한 <a href="/wiki/%EB%B0%94%ED%81%AC" title="바크">바크</a> 식의 3개 기둥을 갖춘 증기선 <a href="/wiki/%EC%BD%94%EB%A5%B4%EB%B2%B3" title="코르벳">코르벳</a>이다. 네덜란드식 옛 명칭은 ‘야판 호’ 또는 ‘얏판 호’(Japan)였으며, ‘감린’(咸臨)이라는 말은 <a class="mw-disambig" href="/wiki/%EC%A3%BC%EC%97%AD" title="주역">주역</a>에서 온 말로 군신이 서로 친해지는 것을 의미한다.</p>
<p><a href="/wiki/1853%EB%85%84" title="1853년">1853년</a> <a href="/wiki/%EB%84%A4%EB%8D%9C%EB%9E%80%EB%93%9C" title="네덜란드">네덜란드</a>에 주문하였으며, 당시 네덜란드는 <a class="mw-redirect" href="/wiki/%EC%87%BC%EA%B5%B0" title="쇼군">쇼군</a>에 의한 <a class="mw-redirect" href="/wiki/%EC%87%84%EA%B5%AD_%EC%A0%95%EC%B1%85" title="쇄국 정책">쇄국 정책</a> 하에서 유일하게 외교 관계를 맺고 있었던 서양의 국가였다. 이 배가 일본에 인도된 것은 <a href="/wiki/1857%EB%85%84" title="1857년">1857년</a> <a href="/wiki/9%EC%9B%94_21%EC%9D%BC" title="9월 21일">9월 21일</a>이며, 네덜란드 해군의 빌렘 후이센 반 카텐다이크 중위(Lt. Willem Huyssen van Kattendijke)가 그 역할을 담당했다. 이 배는 근대식 해군 장교 양성을 위해 새로 신설된 <a class="mw-redirect" href="/wiki/%EB%82%98%EA%B0%80%EC%82%AC%ED%82%A4_%ED%95%B4%EA%B5%B0%EC%A0%84%EC%8A%B5%EC%86%8C" title="나가사키 해군전습소">나가사키 해군전습소</a>에서 사용되었다.</p>

<h2><span class="mw-headline" id=".EA.B0.9C.EC.9A.94">개요</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EA%B0%84%EB%A6%B0%EB%A7%88%EB%A3%A8&amp;action=edit&amp;section=1" title="부분 편집: 개요">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>서양식 군함으로는 <a href="/wiki/%EA%B0%84%EC%BD%94%EB%A7%88%EB%A3%A8" title="간코마루">간코마루</a>(観光丸, 외륜선)에 이어 두 번째 함이지만, 서양식 <a class="new" href="/w/index.php?title=%EC%8A%A4%ED%81%AC%EB%A5%98&amp;action=edit&amp;redlink=1" title="스크류 (없는 문서)">스크류</a>를 장착한 배로서는 최초의 군함이다. (스크류는 입출항 시에 주로 사용되며 항해 중 저항을 줄이기 위해 수선에 올리는 구조로 되어 있었다.)</p>
<p><a href="/wiki/1855%EB%85%84" title="1855년">1855년</a> 네덜란드 <a href="/wiki/%ED%82%A8%EB%8D%B0%EB%A5%B4%EB%8D%B0%EC%9D%B4%ED%81%AC" title="킨데르데이크">킨데르데이크</a>에서 기공되어, <a href="/wiki/1857%EB%85%84" title="1857년">1857년</a> 3월에 완성되었다. 그후 일본에 보내져 <a class="mw-redirect" href="/wiki/%EB%82%98%EA%B0%80%EC%82%AC%ED%82%A4_%ED%95%B4%EA%B5%B0%EC%A0%84%EC%8A%B5%EC%86%8C" title="나가사키 해군전습소">나가사키 해군전습소</a>의 연습함이 되었다.</p>

<p>막부의 선박으로서 최초로 <a href="/wiki/%ED%83%9C%ED%8F%89%EC%96%91" title="태평양">태평양</a>을 왕복한 것으로 이름이 알려져 있다. <a href="/wiki/1860%EB%85%84" title="1860년">1860년</a> <a href="/wiki/%EB%A7%8C%EC%97%94_%EC%9B%90%EB%85%84_%EB%AF%B8%EA%B5%AD_%ED%8C%8C%EA%B2%AC_%EC%82%AC%EC%A0%88" title="만엔 원년 미국 파견 사절">만엔 원년 미국 파견 사절</a>단 일행이 미국 군함 포우하탄 호를 타고 태평양을 횡단하였다. 간린마루는 명목 상 사절단 일행의 호위함으로 출항하였으며, 음력 1월 13일 <a class="mw-redirect" href="/wiki/%EC%8B%9C%EB%82%98%EA%B0%80%EC%99%80" title="시나가와">시나가와</a>를 출항하여 음력 1월 19일에 우라가를 거쳐 <a href="/wiki/3%EC%9B%94_17%EC%9D%BC" title="3월 17일">3월 17일</a>에 <a href="/wiki/%EC%83%8C%ED%94%84%EB%9E%80%EC%8B%9C%EC%8A%A4%EC%BD%94" title="샌프란시스코">샌프란시스코</a>에 입항했다. 함장은 <a href="/wiki/%EA%B0%80%EC%93%B0_%EA%B0%80%EC%9D%B4%EC%8A%88" title="가쓰 가이슈">가쓰 가이슈</a>(정확하게는 <a href="/wiki/%EA%B5%B0%ED%95%A8%EC%A1%B0%EB%A0%A8%EC%86%8C" title="군함조련소">군함조련소</a> 교수행장, 기무라 가이슈는 <a class="mw-redirect" href="/wiki/%EA%B5%B0%ED%95%A8_%EB%B4%89%ED%96%89" title="군함 봉행">군함 봉행</a>)였지만, 통역 <a class="mw-redirect" href="/wiki/%EB%82%98%EC%B9%B4%ED%95%98%EB%A7%88_%EB%A7%8C%EC%A7%80%EB%A1%9C" title="나카하마 만지로">나카하마 만지로</a>(<a href="/wiki/%EC%A1%B4_%EB%A7%8C%EC%A7%80%EB%A1%9C" title="존 만지로">존 만지로</a>)는 함장, <a class="new" href="/w/index.php?title=%EA%B8%B0%EB%AC%B4%EB%9D%BC_%EA%B0%80%EC%9D%B4%EC%8A%88&amp;action=edit&amp;redlink=1" title="기무라 가이슈 (없는 문서)">기무라 가이슈</a>가 제독이라고 설명했다. 이 배에는 <a href="/wiki/%ED%9B%84%EC%BF%A0%EC%9E%90%EC%99%80_%EC%9C%A0%ED%82%A4%EC%B9%98" title="후쿠자와 유키치">후쿠자와 유키치</a> 등도 승선하고 있었다. 만지로 이외의 선원들은 대부분이 뱃멀미로 고생했고, 기술 고문으로 승선하고 있던 존 브룩 대위와 다른 미국인 승무원의 도움을 받아 항해를 했다.</p>
<h3><span class="mw-headline" id=".EB.B3.B4.EC.8B.A0.EC.A0.84.EC.9F.81">보신전쟁</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EA%B0%84%EB%A6%B0%EB%A7%88%EB%A3%A8&amp;action=edit&amp;section=3" title="부분 편집: 보신전쟁">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p><a href="/wiki/1866%EB%85%84" title="1866년">1866년</a> 막부의 연습선으로 사용하면서 잦은 운용으로 혹사가 심했기 때문에 고장이 빈발하게 일어나던 증기 기관을 철거하고 범선이 되었다.</p>
<p><a href="/wiki/1868%EB%85%84" title="1868년">1868년</a> <a href="/wiki/%EB%B3%B4%EC%8B%A0_%EC%A0%84%EC%9F%81" title="보신 전쟁">보신 전쟁</a>에 참여했지만, 군함으로서의 기능은 다른 함보다 떨어졌고, 이미 운송 선의 역할을 담당하고 있던 간린마루는 신정부군에 의해 나포된다. 메이지 정부에 접수된 후 개척사의 수송선이 되었다.</p>
<p><a href="/wiki/1871%EB%85%84" title="1871년">1871년</a>에 <a href="/wiki/%EC%97%90%EC%82%AC%EC%8B%9C_%EC%A0%95_(%ED%9E%88%EC%95%BC%EB%A7%88_%EA%B4%80%EB%82%B4)" title="에사시 정 (히야마 관내)">에사시</a>에서 태풍으로 좌초되어 파선되면서 침몰했다.</p>
