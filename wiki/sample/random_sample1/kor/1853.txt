<p><b>간염</b>(肝炎, <a href="/wiki/%EB%8F%85%EC%9D%BC%EC%96%B4" title="독일어">독일어</a>, <a href="/wiki/%EC%98%81%EC%96%B4" title="영어">영어</a>, <a href="/wiki/%EC%8A%A4%ED%8E%98%EC%9D%B8%EC%96%B4" title="스페인어">스페인어</a>: hepatitis)은 <a href="/wiki/%EA%B0%84" title="간">간</a> 조직에 입는 <a href="/wiki/%EC%97%BC%EC%A6%9D" title="염증">염증</a>을 일으키는 <a href="/wiki/%EC%84%B8%ED%8F%AC" title="세포">세포</a>로 인한 손상을 말한다. 서양의 용어는 고대 <a href="/wiki/%EA%B7%B8%EB%A6%AC%EC%8A%A4%EC%96%B4" title="그리스어">그리스어</a> 헤파(<span style="font-size: smaller;"><a href="/wiki/%EA%B7%B8%EB%A6%AC%EC%8A%A4%EC%96%B4" title="그리스어">그리스어</a>:</span> <span lang="el" xml:lang="el">ηπαρ</span>, 간) 또는 헤파토(<span style="font-size: smaller;"><a href="/wiki/%EA%B7%B8%EB%A6%AC%EC%8A%A4%EC%96%B4" title="그리스어">그리스어</a>:</span> <span lang="el" xml:lang="el">ηπατο-</span>, 간의)와 염증을 뜻하는 -itis에서 왔다. 상태는 스스로 제어되어 나을 수도 있고, 간을 손상시켜 <a href="/wiki/%EA%B0%84%EA%B2%BD%EB%B3%80" title="간경변">간경변</a>이나 <a class="new" href="/w/index.php?title=%EC%84%AC%EC%9C%A0%EC%A6%9D&amp;action=edit&amp;redlink=1" title="섬유증 (없는 문서)">섬유증</a>으로 진행할 수도 있다. 6개월 이하로 지속될때는 급성 간염, 더 길 때는 만성 간염으로 본다. 세계적으로 볼 때 간염은 대부분 간염 바이러스라고 알려진 일련의 바이러스들이 일으킨다. 또 독성 물질 (특히 <a href="/wiki/%EC%95%8C%EC%BD%94%EC%98%AC" title="알코올">알코올</a>)에 의하거나 자체 면역 과정에서 생기는 간염도 있다. 걸린 사람이 아프다고 느끼지 않아도 잠복기를 지나고 있을 수 있다. 보통 간에 손상을 입고서야 <a class="new" href="/w/index.php?title=%EC%9E%90%EA%B0%81_%EC%A6%9D%EC%83%81&amp;action=edit&amp;redlink=1" title="자각 증상 (없는 문서)">자각 증상</a>이 생기며 <a href="/wiki/%ED%86%B5%EC%A6%9D" title="통증">통증</a>을 느낀다.</p>

<h2><span class="mw-headline" id=".EC.A7.95.ED.9B.84_.EB.B0.8F_.EC.A6.9D.EC.83.81">징후 및 증상</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EA%B0%84%EC%97%BC&amp;action=edit&amp;section=1" title="부분 편집: 징후 및 증상">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<h3><span class="mw-headline" id=".EA.B8.89.EC.84.B1">급성</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EA%B0%84%EC%97%BC&amp;action=edit&amp;section=2" title="부분 편집: 급성">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>초기 특징은 전형적인 유사 감기 <a href="/wiki/%EC%A6%9D%EC%83%81" title="증상">증상</a>으로, 거의 모든 급성 바이러스성 감염과 공통되며, <a class="new" href="/w/index.php?title=%EB%B6%88%EC%BE%8C%EA%B0%90&amp;action=edit&amp;redlink=1" title="불쾌감 (없는 문서)">불쾌감</a>, <a href="/wiki/%EA%B7%BC%EC%9C%A1%ED%86%B5" title="근육통">근육통</a>, <a href="/wiki/%EA%B4%80%EC%A0%88%ED%86%B5" title="관절통">관절통</a>, <a class="mw-redirect" href="/wiki/%EC%8B%A0%EC%97%B4" title="신열">신열</a>, <a class="mw-redirect" href="/wiki/%EB%A9%94%EC%8A%A4%EA%BA%BC%EC%9B%80" title="메스꺼움">메스꺼움</a>이나 <a href="/wiki/%EA%B5%AC%ED%86%A0" title="구토">구토</a>, <a href="/wiki/%EC%84%A4%EC%82%AC" title="설사">설사</a>, <a href="/wiki/%EB%91%90%ED%86%B5" title="두통">두통</a>이 있을 수 있다. 급성 간염에서 볼 수 있는 보다 구체적인 증상으로, 심한 <a class="new" href="/w/index.php?title=%EC%8B%A0%EA%B2%BD%EC%84%B1_%EC%8B%9D%EC%9A%95_%EB%B6%80%EC%A7%84%EC%A6%9D&amp;action=edit&amp;redlink=1" title="신경성 식욕 부진증 (없는 문서)">신경성 식욕 부진증</a>, 흡연가 중 흡연에 대한 혐오감, 검은색 <a class="mw-redirect" href="/wiki/%EC%86%8C%EB%B3%80" title="소변">소변</a>, 황달, 복부의 불편함 등이 있다.</p>
<p>신체 검사 결과는 보통 미미하나, 10%에서 가벼운 <a class="new" href="/w/index.php?title=%EA%B0%84_%EB%B9%84%EB%8C%80&amp;action=edit&amp;redlink=1" title="간 비대 (없는 문서)">간 비대</a>, 5%에서 림프샘 장애 또는 비장 비대증을 보이기도 한다.<sup class="reference" id="cite_ref-1"><a href="#cite_note-1">[1]</a></sup></p>
<p>급성 간염 환자의 일부는 급성 간부전으로 발전하기도 하는데, 간은 해로운 물질들을 혈액 순환으로부터 맑게 하지 못하여 혼수상태를 초래하는 간성뇌병증이 될 수 있으며, 혈액단백질을 생산하여 말초 부종과 출혈을 일으키기도 한다. 이는 생명을 위협할 수도 있으며, 종종 <a class="new" href="/w/index.php?title=%EA%B0%84_%EC%9D%B4%EC%8B%9D&amp;action=edit&amp;redlink=1" title="간 이식 (없는 문서)">간 이식</a>을 요한다.<sup class="reference" id="cite_ref-2"><a href="#cite_note-2">[2]</a></sup></p>
<h3><span class="mw-headline" id=".EB.A7.8C.EC.84.B1">만성</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EA%B0%84%EC%97%BC&amp;action=edit&amp;section=3" title="부분 편집: 만성">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>만성 간염은 종종 불안감, 피로, 쇠약 같은 불특정한 증상을 초래하며, 증상이 전혀 없을 수도 있다. 만성 간염의 확인은 보통 검진이나 불특정 증상을 판정하기 위한 <a href="/wiki/%ED%98%88%EC%95%A1_%EA%B2%80%EC%82%AC" title="혈액 검사">혈액 검사</a>로 확인한다.<sup class="reference" id="cite_ref-MerckHome_3-0"><a href="#cite_note-MerckHome-3">[3]</a></sup></p>
<p>광범위한 손상과 간경변으로는 체중이 감소하고, 쉽게 피로해지며, 출혈의 경향이 있고, 팔다리가 붓거나, <a class="new" href="/w/index.php?title=%EB%B3%B5%EC%88%98_(%EC%9D%98%ED%95%99)&amp;action=edit&amp;redlink=1" title="복수 (의학) (없는 문서)">복수</a>가 차기도 한다. 결국 간경변은 <a class="new" href="/w/index.php?title=%EC%8B%9D%EB%8F%84%EC%A0%95%EB%A7%A5%EB%A5%98&amp;action=edit&amp;redlink=1" title="식도정맥류 (없는 문서)">식도정맥류</a>, <a class="new" href="/w/index.php?title=%EA%B0%84%EC%84%B1%EB%87%8C%EB%B3%91%EC%A6%9D&amp;action=edit&amp;redlink=1" title="간성뇌병증 (없는 문서)">간성뇌병증</a>, <a class="new" href="/w/index.php?title=%EA%B0%84%EC%8B%A0%EC%A6%9D%ED%9B%84%EA%B5%B0&amp;action=edit&amp;redlink=1" title="간신증후군 (없는 문서)">간신증후군</a> 등의 다양한 합병증을 초래한다.</p>
<p>자가면역성 간염에 걸린 여성에서는 <a href="/wiki/%EC%97%AC%EB%93%9C%EB%A6%84" title="여드름">여드름</a>, 비정상적인 <a href="/wiki/%EC%9B%94%EA%B2%BD" title="월경">월경</a>, 폐 반흔(lung scarring)이 있을 수 있고, <a class="mw-redirect" href="/wiki/%EA%B0%91%EC%83%81%EC%84%A0" title="갑상선">갑상선</a>, <a class="mw-disambig" href="/wiki/%EC%8B%A0%EC%9E%A5" title="신장">신장</a>이 붓기도 한다.<sup class="reference" id="cite_ref-MerckHome_3-1"><a href="#cite_note-MerckHome-3">[3]</a></sup></p>
<h2><span class="mw-headline" id=".EC.9B.90.EC.9D.B8">원인</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EA%B0%84%EC%97%BC&amp;action=edit&amp;section=4" title="부분 편집: 원인">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<h4><span class="mw-headline" id=".EA.B8.89.EC.84.B1_.EA.B0.84.EC.97.BC">급성 간염</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EA%B0%84%EC%97%BC&amp;action=edit&amp;section=5" title="부분 편집: 급성 간염">편집</a><span class="mw-editsection-bracket">]</span></span></h4>
<ul>
<li>바이러스성 간염: A에서 E형까지 (95%가 바이러스성), <a class="mw-redirect" href="/wiki/%ED%97%A4%EB%A5%B4%ED%8E%98%EC%8A%A4" title="헤르페스">헤르페스</a>, 헤르페스 심플렉스(Herpes simplex), 시토메가로바이러스, 황열병 바이러스, <a href="/wiki/%EC%95%84%EB%8D%B0%EB%85%B8%EB%B0%94%EC%9D%B4%EB%9F%AC%EC%8A%A4" title="아데노바이러스">아데노바이러스</a>(adenoviruses)</li>
<li>비 바이러스성 감염: 톡소플라즈마(toxoplasma), 렙토스피라(Leptospira), Q열(Q fever), 록키산 반점열(rocky mountain spotted fever)</li>
<li><a href="/wiki/%EC%95%8C%EC%BD%94%EC%98%AC" title="알코올">알코올</a></li>
<li><a href="/wiki/%EB%8F%85" title="독">독</a>: 버섯에 있는 아마니타(Amanita), 십염화탄사(Carbon tetrachloride), 아사페티다(asafetida)</li>
<li><a class="mw-redirect" href="/wiki/%EC%95%BD%EB%AC%BC" title="약물">약물</a>: 파라세타몰(Paracetamol), 아목시실린(amoxycillin), 일부 <a href="/wiki/%EA%B2%B0%ED%95%B5" title="결핵">결핵</a> 치료약들, 미노실린 등</li>
<li>순환 부족</li>
<li>임신</li>
<li>자발적 면역 상태. 예) Systemic Lupus Erythematosus (SLE)</li>
<li><a href="/wiki/%EB%AC%BC%EC%A7%88%EB%8C%80%EC%82%AC" title="물질대사">물질대사</a>성 질환: 예) <a href="/wiki/%EC%9C%8C%EC%8A%A8%EB%B3%91" title="윌슨병">윌슨병</a></li>
</ul>
<h4><span class="mw-headline" id=".EB.A7.8C.EC.84.B1_.EA.B0.84.EC.97.BC">만성 간염</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EA%B0%84%EC%97%BC&amp;action=edit&amp;section=6" title="부분 편집: 만성 간염">편집</a><span class="mw-editsection-bracket">]</span></span></h4>
<ul>
<li>바이러스성 : D형 간염 없는 B형 간염. C형 간염. A, E형 간염은 급성만 있다.</li>
<li>자가면역성 간염(Autoimmune hepatitis)</li>
<li>알코올</li>
<li>약물: 메틸 도파(methyl-dopa), 니트로푸란토인(nitrofurantoin), 이소니아지드(isoniazide), 케토코나졸(ketoconazole)</li>
<li>비 알코올성 지방간</li>
<li>유전: 윌슨병, 알파 1-안티트립신 부족</li>
<li>초기 담즙 경변이나 초기 쓸개관 경화도 때때로 만성 간염과 증상이 비슷하다.</li>
</ul>
