<p><b>간체자</b>(簡體字,<small><a href="/wiki/%EC%A4%91%EA%B5%AD%EC%96%B4" title="중국어">중국어</a></small> <small><strong class="selflink">간체</strong>:</small> <span lang="zh-hans" xml:lang="zh-hans">简体字</span>, <small><a href="/wiki/%EC%A0%95%EC%B2%B4%EC%9E%90" title="정체자">정체</a>:</small> <span lang="zh-hant" xml:lang="zh-hant">簡體字</span>, <small><a href="/wiki/%ED%95%9C%EC%96%B4_%EB%B3%91%EC%9D%8C" title="한어 병음">병음</a>:</small> <font face="Lucida Sans Unicode">jiǎntǐzì</font>) 또는 <b>간화자</b>(簡化字,<small><a href="/wiki/%EC%A4%91%EA%B5%AD%EC%96%B4" title="중국어">중국어</a></small> <small><strong class="selflink">간체</strong>:</small> <span lang="zh-hans" xml:lang="zh-hans">简化字</span>, <small><a href="/wiki/%EC%A0%95%EC%B2%B4%EC%9E%90" title="정체자">정체</a>:</small> <span lang="zh-hant" xml:lang="zh-hant">簡化字</span>, <small><a href="/wiki/%ED%95%9C%EC%96%B4_%EB%B3%91%EC%9D%8C" title="한어 병음">병음</a>:</small> <font face="Lucida Sans Unicode">jienhuàzì</font>)는 <a href="/wiki/1960%EB%85%84%EB%8C%80" title="1960년대">1960년대</a> <a href="/wiki/%EC%A4%91%ED%99%94%EC%9D%B8%EB%AF%BC%EA%B3%B5%ED%99%94%EA%B5%AD" title="중화인민공화국">중화인민공화국</a>에서 <a class="mw-redirect" href="/wiki/%EC%A4%91%EA%B5%AD_%EA%B3%B5%EC%82%B0%EB%8B%B9" title="중국 공산당">중국 공산당</a>의 주도로 창안된 <a href="/wiki/%ED%95%9C%EC%9E%90" title="한자">한자</a>이다. 정확히 말하면, 간체자는 과거 중국에서 존재했던 약자체를 모두 통칭하는 말이고, 현재 중국에서 사용되고 있는 규범화된 글자체만 지칭하는 말은 <b>간화자</b>라고 한다. <a href="/wiki/%EC%A4%91%ED%99%94%EC%9D%B8%EB%AF%BC%EA%B3%B5%ED%99%94%EA%B5%AD" title="중화인민공화국">중화인민공화국</a>에서는 원래의 전통적인 글자를 <a class="mw-redirect" href="/wiki/%EB%B2%88%EC%B2%B4%EC%9E%90" title="번체자">번체자</a>(정체자)라고 하고, <a href="/wiki/%EC%A4%91%ED%99%94%EB%AF%BC%EA%B5%AD" title="중화민국">중화민국</a>에서는 <a href="/wiki/%EC%A0%95%EC%B2%B4%EC%9E%90" title="정체자">정체자</a>라고 부르며 간화자는 인정하지 않는다.</p>
<p><a href="/wiki/1956%EB%85%84" title="1956년">1956년</a> 「한자간화방안(漢字簡化方案)」이 발표된 후 몇 년 동안의 연구를 거쳐, <a href="/wiki/1964%EB%85%84" title="1964년">1964년</a> 「간화자총표(簡化字總表)」가 발표되었다. <a href="/wiki/%EC%A4%91%EA%B5%AD" title="중국">중국</a>과 <a href="/wiki/%EC%8B%B1%EA%B0%80%ED%8F%AC%EB%A5%B4" title="싱가포르">싱가포르</a>에서 사용되고 있다.</p>
<p>간화된 내용은, 변과 방으로 사용되지 않는 간화자 350자(제1표), 변과 방으로 사용하는 간화자 132자와 변과 방 14개(제2표), 제2표를 참고한 간화자 1753자(제3표), 총2235자로 이루어져 있다.</p>

<h2><span class="mw-headline" id=".EC.97.AD.EC.82.AC">역사</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EA%B0%84%EC%B2%B4%EC%9E%90&amp;action=edit&amp;section=1" title="부분 편집: 역사">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>오늘날 통용되는 간화자 체계는 <a href="/wiki/%EC%A4%91%ED%99%94%EC%9D%B8%EB%AF%BC%EA%B3%B5%ED%99%94%EA%B5%AD" title="중화인민공화국">중화인민공화국</a>이 <a href="/wiki/1950%EB%85%84%EB%8C%80" title="1950년대">1950년대</a>에서 <a href="/wiki/1960%EB%85%84%EB%8C%80" title="1960년대">1960년대</a>에 추진한 결과물이지만, 한자를 간소화하여 사용하는 경향은 갑골문 시절부터 있어왔다. 다만 <a href="/wiki/%EC%B4%88%EC%84%9C" title="초서">초서</a>를 비롯한 극도로 간략화된 필기체가 생겨나기도 하였지만, 이렇게 간략화된 한자가 대중적으로 쓰인 적은 없었다.</p>
<p><a href="/wiki/%EC%B2%AD%EB%82%98%EB%9D%BC" title="청나라">청나라</a> 말기, <a href="/wiki/1909%EB%85%84" title="1909년">1909년</a>, <a class="new" href="/w/index.php?title=%EC%9C%A1%EB%B9%84%EA%B7%9C&amp;action=edit&amp;redlink=1" title="육비규 (없는 문서)">육비규</a>(陸費逵)가 〈교육 한자〉창간호에 ‘일반 교육에 속자를 채용해야 한다.(普通敎育應當採用俗體字.)’라는 논문을 발표하여 간략화 운동이 시작되었다. <a href="/wiki/1919%EB%85%84" title="1919년">1919년</a> <a href="/wiki/5%C2%B74_%EC%9A%B4%EB%8F%99" title="5·4 운동">5·4 운동</a> 이후 중국의 전통 문화와 가치에 대한 반발이 심해졌고, 그 가운데서 한자를 중국 근대화의 장애물로 여기는 경향이 강해졌다. 한자를 간략화하거나 아예 없애자는 주장들이 지식인들에 의하여 강하게 제기되었다. <a href="/wiki/5%C2%B74_%EC%9A%B4%EB%8F%99" title="5·4 운동">5·4 운동</a>을 이끈 <a href="/wiki/%ED%91%B8%EC%93%B0%EB%85%A0" title="푸쓰녠">푸쓰녠</a>(傅斯年)은 한자를 일컬어, “소와 뱀 같은 귀신의 문자다.(牛鬼蛇神的文字.)"라고 혹평하였다. 중국의 대문호인 <a href="/wiki/%EB%A3%A8%EC%89%B0" title="루쉰">루쉰</a>은 “한자가 멸하지 않으면 중국이 반드시 망한다.(漢字不滅, 中國必亡.)”라고까지 주장하였다.</p>
<p><a href="/wiki/5%C2%B74_%EC%9A%B4%EB%8F%99" title="5·4 운동">5·4 운동</a>시대 <a href="/wiki/1920%EB%85%84" title="1920년">1920년</a> <a class="new" href="/w/index.php?title=%EC%A0%84%ED%98%84%EB%8F%99&amp;action=edit&amp;redlink=1" title="전현동 (없는 문서)">전현동</a>(錢玄同)은 <a href="/wiki/%EC%8B%A0%EC%B2%AD%EB%85%84" title="신청년">신청년</a>(新靑年)에 한자 획수를 줄이는 방안(減省漢字筆畫的提議)을 발표하고, <a href="/wiki/1922%EB%85%84" title="1922년">1922년</a> <a class="new" href="/w/index.php?title=%EC%9C%A1%EB%B9%84%EA%B7%9C&amp;action=edit&amp;redlink=1" title="육비규 (없는 문서)">육비규</a>와 함께 국어 통일 주비회(國語統一籌備會)에 상용 한자 전체의 획수를 줄일 것을 제안했다. <a href="/wiki/1934%EB%85%84" title="1934년">1934년</a>에도 간략화된 한자를 제안하여, <a href="/wiki/1935%EB%85%84" title="1935년">1935년</a>에는 2400자에 이르러 '간체자보(簡體字譜)'의 초안이 마련되었다. <a href="/wiki/%EA%B5%AD%EB%AF%BC%EC%A0%95%EB%B6%80" title="국민정부">국민정부</a>의 교육부에서 324자의 '제1차 간략 한자표(第一批簡體字表)'를 발표했지만, 반대 여론으로 인해 실시되지는 않았다. 그 뒤로 간략자(簡略字)에 대한 수집은 줄었으나, <a href="/wiki/1937%EB%85%84" title="1937년">1937년</a> '자체 연구회(字體研究會)'가 1700자의 '간체자표 제1표(簡體字表第一表)'를 발표했다.</p>
<p><a href="/wiki/1955%EB%85%84" title="1955년">1955년</a>, 중국 문자 개혁 위원회(中國文字改革委員會)가 '한자 간화 방안 초안(漢字簡化方案草案)'을 발표하고, <a href="/wiki/1956%EB%85%84" title="1956년">1956년</a> 1월 '한자 간화 방안(漢字簡化方案)'이 정식으로 발표되어, 514자의 간체자와 54개의 간화된 변과 방이 채용되었다. 그 뒤, 간화자는 <a href="/wiki/1959%EB%85%84" title="1959년">1959년</a>까지 네 번 발표되어, <a href="/wiki/1964%EB%85%84" title="1964년">1964년</a> '간화자 총표(簡化字總表)'로 정리되었다.</p>
<p><a href="/wiki/1977%EB%85%84" title="1977년">1977년</a>, <a class="new" href="/w/index.php?title=%EC%A4%91%EA%B5%AD_%EB%AC%B8%EC%9E%90_%EA%B0%9C%ED%98%81_%EC%9C%84%EC%9B%90%ED%9A%8C&amp;action=edit&amp;redlink=1" title="중국 문자 개혁 위원회 (없는 문서)">중국 문자 개혁 위원회</a>가 새롭게 '<a href="/wiki/%EC%A0%9C2%EC%B0%A8_%ED%95%9C%EC%9E%90_%EA%B0%84%ED%99%94_%EB%B0%A9%EC%95%88" title="제2차 한자 간화 방안">제2차 한자 간화 방안</a> 초안(第二次漢字簡化方案草案)'을 발표하여, 한자의 새로운 간략화를 목표로 하였다. 그러나 이것은 <a href="/wiki/%EB%AC%B8%ED%99%94_%EB%8C%80%ED%98%81%EB%AA%85" title="문화 대혁명">문화 대혁명</a> 직후에 만들어져서 너무 졸속한 방안이었고, 자체가 너무 간략화되어 읽기 어렵고 또한 원래의 뜻을 모조리 망쳐놓았다는 등의 비판을 들으며 8년간의 시험 후 결국 폐기되었다. 이 간화자는 <b>이간자</b>(二簡字)라고도 불린다. 그 이후 새로운 문자 개혁은 하지 않고, 간화자는 안정기에 들어 갔다.</p>
<p><a href="/wiki/1992%EB%85%84" title="1992년">1992년</a> <a href="/wiki/%EC%9E%A5%EC%A9%8C%EB%AF%BC" title="장쩌민">장쩌민</a>(江澤民) <a href="/wiki/%EC%A4%91%ED%99%94%EC%9D%B8%EB%AF%BC%EA%B3%B5%ED%99%94%EA%B5%AD" title="중화인민공화국">중화인민공화국</a> 주석은 '모든 인쇄물에는 간체자만 사용한다. 양안의 <a href="/wiki/%ED%95%9C%EC%9E%90" title="한자">한자</a>에 관한 차이는 현상을 그대로 유지한다. 단 <a href="/wiki/%EC%84%9C%EC%98%88" title="서예">서예</a>(書法) 등의 고귀한 예술 작품 등에는 <a class="mw-redirect" href="/wiki/%EB%B2%88%EC%B2%B4%EC%9E%90" title="번체자">번체자</a>를 사용해도 무방하다.'라는 지침을 발표하여, '간체자를 추구하되 <a class="mw-redirect" href="/wiki/%EB%B2%88%EC%B2%B4%EC%9E%90" title="번체자">번체자</a>를 반대하지 않는다.'라는 입장을 천명하였다.</p>
<h2><span class="mw-headline" id=".EA.B0.84.ED.99.94.EC.9E.90_.EB.B3.B4.EA.B8.89_.EC.9D.B4.ED.9B.84">간화자 보급 이후</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EA%B0%84%EC%B2%B4%EC%9E%90&amp;action=edit&amp;section=2" title="부분 편집: 간화자 보급 이후">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>현재 <a href="/wiki/%EC%A4%91%ED%99%94%EC%9D%B8%EB%AF%BC%EA%B3%B5%ED%99%94%EA%B5%AD" title="중화인민공화국">중화인민공화국</a>에서는 출판물에 간화자를 사용하고 있고, 학교에서도 간화자를 교육하여 <a class="mw-redirect" href="/wiki/%EB%B2%88%EC%B2%B4%EC%9E%90" title="번체자">번체자</a>를 읽을 수 있는 사람이 점점 줄어들고 있다. 이에 대한 반발로, 중화인민공화국의 정식 의회인 <a class="new" href="/w/index.php?title=%EC%9D%B8%EB%AF%BC%EC%A0%95%EC%B9%98%ED%98%91%EC%83%81%ED%9A%8C%EC%9D%98&amp;action=edit&amp;redlink=1" title="인민정치협상회의 (없는 문서)">인민정치협상회의</a>의 <a href="/wiki/%ED%8C%90%EC%B9%AD%EB%A6%B0" title="판칭린">판칭린</a>(潘慶林) 등 중국의 개혁적인 의원들은 중국 전통의 단절 현상이 심각하다면서 정체자 재교육을 강력히 주장한 바가 있다. <a href="/wiki/%EC%A4%91%ED%99%94%EC%9D%B8%EB%AF%BC%EA%B3%B5%ED%99%94%EA%B5%AD" title="중화인민공화국">중화인민공화국</a>은 그동안 한자 통일 논의에서 소극적이고 간화자의 보급을 중점적으로 추진하였는데, 최근 <a href="/wiki/%EB%AF%B8%EA%B5%AD" title="미국">미국</a>과 <a href="/wiki/%EC%98%81%EA%B5%AD" title="영국">영국</a>과 <a href="/wiki/%ED%94%84%EB%9E%91%EC%8A%A4" title="프랑스">프랑스</a>와 <a href="/wiki/%EB%9F%AC%EC%8B%9C%EC%95%84" title="러시아">러시아</a>와 <a href="/wiki/%EB%8F%85%EC%9D%BC" title="독일">독일</a> 등 선진국들의 적극적이고 강력한 요청으로 중화인민공화국에서 한자 통일을 주제로 베이징 촨메이(傳媒) 대학교에서 열린 제8회 국제한자회의가 열리기도 하였다. 이 회의의 주제는 간체자 사용을 자제하고 정체자 사용을 활발히 하여 정체자가 공식 통용되는 언어로 등극할 수 있게 하는 것이었다. 여기서 확인된 <a href="/wiki/%EC%A4%91%ED%99%94%EC%9D%B8%EB%AF%BC%EA%B3%B5%ED%99%94%EA%B5%AD" title="중화인민공화국">중화인민공화국</a>의 입장은 간화자는 버릴 수 없으나 정체자에 대한 탄압은 자제하고 하나의 전통문화로 의연하게 다루겠다는 것이었다.</p>
<p>한편, <a href="/wiki/%ED%99%8D%EC%BD%A9" title="홍콩">홍콩</a>은 <a href="/wiki/%EC%98%81%EA%B5%AD" title="영국">영국</a>의 식민 통치로 인해, <a href="/wiki/%ED%83%80%EC%9D%B4%EC%99%84" title="타이완">타이완</a>과 <a href="/wiki/%EB%AF%B8%EA%B5%AD" title="미국">미국</a> 등에서는 전통문화의 보존 및 정치적 상황 등의 배경으로 인해 정체자인 <a class="mw-redirect" href="/wiki/%EB%B2%88%EC%B2%B4%EC%9E%90" title="번체자">번체자</a>를 그대로 고수하고 있다. <a href="/wiki/%ED%83%80%EC%9D%B4%EC%99%84" title="타이완">타이완</a>이나 <a href="/wiki/%ED%99%8D%EC%BD%A9" title="홍콩">홍콩</a>과 <a href="/wiki/%EB%AF%B8%EA%B5%AD" title="미국">미국</a>의 화교권에서는 <a href="/wiki/%EC%A0%95%EC%B2%B4%EC%9E%90" title="정체자">정체자</a>를 계속 사용하고 있기 때문에 간화자를 읽을 수 있는 사람이 적다. <a href="/wiki/%EC%A4%91%ED%99%94%EB%AF%BC%EA%B5%AD" title="중화민국">중화민국</a>의 한 여론 조사에 따르면, 45%는 간화자를 전혀 못 읽으며, 41%는 약간 읽을 수 있고 14%만이 완전히 읽을 수 있다는 결과가 나왔다. 그러나 간화자라는 것도 정체자를 기본으로 간략화한 것이기 때문에 이미 정체자 체계에 익숙한 <a href="/wiki/%ED%83%80%EC%9D%B4%EC%99%84" title="타이완">타이완</a>이나 <a href="/wiki/%ED%99%8D%EC%BD%A9" title="홍콩">홍콩</a>과 <a href="/wiki/%EB%AF%B8%EA%B5%AD" title="미국">미국</a>의 화교권 사람들은 약간의 훈련만으로도 쉽게 읽어낼 수 있다.</p>
<h2><span class="mw-headline" id=".EA.B0.84.EB.9E.B5.ED.99.94_.EC.9B.90.EB.A6.AC">간략화 원리</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EA%B0%84%EC%B2%B4%EC%9E%90&amp;action=edit&amp;section=3" title="부분 편집: 간략화 원리">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>다음과 같은 방법으로 한자를 간략화하였다.</p>

<dt>변과 방을 같은 음의 다른 글자로 대체한다</dt>
<dd>예: <span lang="zh" xml:lang="zh">蜡</span>(蠟), <span lang="zh" xml:lang="zh">钟</span>(鐘), <span lang="zh" xml:lang="zh">远</span>(遠)</dd>

<h2><span class="mw-headline" id=".EC.9D.B4.EC.B2.B4.EC.9E.90_.EC.A0.95.EB.A6.AC">이체자 정리</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EA%B0%84%EC%B2%B4%EC%9E%90&amp;action=edit&amp;section=4" title="부분 편집: 이체자 정리">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p><a href="/wiki/1955%EB%85%84" title="1955년">1955년</a> 중국한자개혁위원회는 「간화방안」에 앞서, 「제1차이체자정리표」를 발표하여, 1865자의 글자를 810자로 정리하였다. 예를 들어 「并・並・併・竝」을「并」로, 「屍・尸」을「尸」등이 있다.</p>
<h2><span class="mw-headline" id=".EA.B0.84.EC.B2.B4.EC.9E.90_.EB.85.BC.EC.9F.81">간체자 논쟁</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EA%B0%84%EC%B2%B4%EC%9E%90&amp;action=edit&amp;section=5" title="부분 편집: 간체자 논쟁">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>간체자가 <a href="/wiki/%EC%A4%91%ED%99%94%EC%9D%B8%EB%AF%BC%EA%B3%B5%ED%99%94%EA%B5%AD" title="중화인민공화국">중화인민공화국</a>에 의해 창안된 이래, 중국 대륙의 성장과 함께 중국 대륙을 비롯한 세계 각지에서 간체자를 사용하는 비중이 날로 증대하였고 중국어를 배우는 외국인도 <a class="mw-redirect" href="/wiki/%EB%B6%81%EA%B2%BD%EC%96%B4" title="북경어">북경어</a>를 배워 간체자를 학습하는 일이 많아졌다. 이에 대해 <a href="/wiki/%EC%A4%91%ED%99%94%EB%AF%BC%EA%B5%AD" title="중화민국">중화민국</a>이나 <a href="/wiki/%ED%99%8D%EC%BD%A9" title="홍콩">홍콩</a> 등 아직 <a href="/wiki/%EC%A0%95%EC%B2%B4%EC%9E%90" title="정체자">정체자</a>를 사용하는 곳이나 <a href="/wiki/%EC%A4%91%ED%99%94%EC%9D%B8%EB%AF%BC%EA%B3%B5%ED%99%94%EA%B5%AD" title="중화인민공화국">중화인민공화국</a> 내 정체자로의 복귀 또는 정체자 교육 강화를 주장하는 세력들에 의해 정체자의 우수성이 주장되어 왔고, 간체자와 정체자 간의 주요 논쟁을 이루고 있다. 이 논쟁을 중국에서는 <b>번간지쟁</b>(繁簡之爭)이나 <b>정간지쟁</b>(正簡之爭)으로 부른다. 이 논쟁은 <a href="/wiki/%EC%A4%91%ED%99%94%EA%B6%8C" title="중화권">중화권</a> 뿐만 아니라 해외 화교들 사이에서도 격렬한 논쟁을 불러왔고 또한 현재진행형이다. 정치적 이데올로기와 문화적 정체성의 문제가 한데 얽혀있기 때문에 논쟁이 격하고 치열하다.</p>
<p>중국의 유명 작자 <a href="/wiki/%EB%B0%94%EC%A7%84" title="바진">바진</a>(巴金)은 《수상록·한자개혁(隨想錄·漢字改革)》에서 문어(文語)의 개혁에 주의를 촉구했다. <a href="/wiki/%ED%99%8D%EC%BD%A9" title="홍콩">홍콩</a>, <a href="/wiki/%EC%A4%91%ED%99%94%EB%AF%BC%EA%B5%AD" title="중화민국">중화민국</a>의 교육받은 이들조차 중국 대륙에서 발간한 책을 읽을 수 없으며, 중국 대륙에서도 중화민국과 홍콩의 서적을 읽을 수 없어, 이러한 간극의 발생을 간체화의 큰 단점이라고 꼬집었다. 또한 세계 각지의 중국인들뿐만 아니라 <a href="/wiki/%ED%95%9C%EC%9E%90_%EB%AC%B8%ED%99%94%EA%B6%8C" title="한자 문화권">한자 문화권</a> 내 국가 간 한자를 통한 의사 소통이 가능함을 언급하며 과도하게 간체화하지 않아야 할 장점이라고 주장했다. 이에 대해 간체화를 옹호하는 이는, 하나의 한자 체계를 잘 교육받은 사람이라면 경험과 잦은 노출을 통해 다른 한자 체계를 금세 익히는 것은 어려운 일이 아니라고 본다. 대륙과 타이완은 간체와 정체와 관련된 갈등을 빚고 있다.<sup class="reference" id="cite_ref-1"><a href="#cite_note-1">[1]</a></sup></p>
<h3><span class="mw-headline" id=".EB.AC.B8.ED.99.94.EC.A0.81_.EC.A0.95.ED.86.B5.EC.84.B1">문화적 정통성</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EA%B0%84%EC%B2%B4%EC%9E%90&amp;action=edit&amp;section=6" title="부분 편집: 문화적 정통성">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<dl>
<dt>정체자의 입장</dt>
</dl>
<p>그 동안 진행되어 온 한자의 변화는 양식의 사소한 변경에 불과하지 문자 구조의 변경은 아니다. 특히 <a class="mw-redirect" href="/wiki/%EC%A7%84%EC%8B%9C%ED%99%A9" title="진시황">진시황</a> 이후의 역사는 더욱 그러하다. 중국 대륙의 중공 정부가 진척시킨 간체화 사업은 <a href="/wiki/%EC%A4%91%ED%99%94%EC%9D%B8%EB%AF%BC%EA%B3%B5%ED%99%94%EA%B5%AD" title="중화인민공화국">중화인민공화국</a> 정부가 그들이 내세우는 근대화라는 정치적인 목표의 달성을 위해 중국의 전통 문화를 곡해하기 위한 사업이다. 인위적이고 무리한 문자의 변경은 부수와 뜻과 소리에 기초하여 만들어진 한자의 특질을 무시하고 파괴하는 것이다. 聖을 圣으로 간화한 것은 聖의 부수인 耳을 버리고 엉뚱한 土를 부수로 취한 경우와 愛를 爱로 간화한 것은 愛의 부수인 心을 버리고 엉뚱한 友를 부수로 취한 경우를 대표적인 예로 들 수 있다.</p>
<dl>
<dt>간체자의 입장</dt>
</dl>
<p>한자는 수천년 동안 끊임없이 변화해왔다. <a href="/wiki/%EA%B0%91%EA%B3%A8_%EB%AC%B8%EC%9E%90" title="갑골 문자">갑골 문자</a>, <a href="/wiki/%EA%B8%88%EB%AC%B8" title="금문">금문</a>, <a href="/wiki/%EC%A0%84%EC%84%9C" title="전서">전서</a>, <a href="/wiki/%EC%98%88%EC%84%9C" title="예서">예서</a>, <a class="mw-disambig" href="/wiki/%ED%95%B4%EC%84%9C" title="해서">해서</a>에 이르기까지 한자의 모양은 계속 변해왔다. 더욱이, 현대 간체자 체계가 채택한 간략화 방법은 이미 예전부터 禮를 礼로도 썼듯이 수 세기 동안 역사적으로 통용되어 온 방법을 사용하였거나 시대를 거치며 지나치게 복잡화되었던 몇몇 정체자를 과거의 쉬운 형태로 복원하는 방법을 사용하였다. (후자의 예는 云으로 이 글자는 처음에 구름의 뜻으로 쓰였는데 이를 운(云)과 모양이 중복되어 雲이 이후 통용되었다. 이에 현대 간체자를 옛 모양으로 복원시켜 雲 대신 云을 구름으로 쓴다.)</p>
<h3><span class="mw-headline" id=".EB.AC.B8.EB.A7.B9.EB.A5.A0.EA.B3.BC_.EC.8B.9D.EC.9E.90.EC.9C.A8">문맹률과 식자율</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EA%B0%84%EC%B2%B4%EC%9E%90&amp;action=edit&amp;section=7" title="부분 편집: 문맹률과 식자율">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<dl>
<dt>정체자의 입장</dt>
</dl>
<p>문맹률과 식자율은 <a class="new" href="/w/index.php?title=%EB%8C%80%EC%A4%91_%EA%B5%90%EC%9C%A1&amp;action=edit&amp;redlink=1" title="대중 교육 (없는 문서)">대중 교육</a>의 접근도에 의해 결정되는 것이다. 간체화와 식자율 간의 상관관계를 입증할 자료는 충분하지 않다. 미국 <a class="mw-redirect" href="/wiki/CIA" title="CIA">CIA</a> 월드팩트북 2007년 자료에 따르면, 2002/2003년 식자율은 정체자를 사용하는 대만이 96.1%, 간체자를 사용하는 중국 대륙이 90.9%이다. 청나라나 <a href="/wiki/%EC%A4%91%ED%99%94%EB%AF%BC%EA%B5%AD" title="중화민국">중화민국</a> 시절과 비교하는 것은 문맹률이 90%에 달하는 시절과의 비교이어서 어불성설이다. 문맹률의 감소는 대중 교육과 학교 제도의 활성화, 시골 지역의 관리 제고와 관련이 있다. 심지어는 중국 공산당 측에서도 문맹률의 감소와 간체자를 연결시키지 않는다.</p>
<dl>
<dt>간체자의 입장</dt>
</dl>
<p>획수가 정체자보다 적어 배우기가 쉽고, 간체화 사업 이후 중국 대륙의 도시와 시골의 문맹률은 줄고 식자율이 늘어났다. 이는 정체자가 공식 문자 체계였던 <a href="/wiki/%EC%A4%91%ED%99%94%EB%AF%BC%EA%B5%AD" title="중화민국">중화민국</a>의 중국 대륙 통치 시기나 <a href="/wiki/%EC%B2%AD%EB%82%98%EB%9D%BC" title="청나라">청나라</a>의 시기에는 볼 수 없었던 것이다. 비록 현재 정체자를 쓰는 <a href="/wiki/%EC%A4%91%ED%99%94%EB%AF%BC%EA%B5%AD" title="중화민국">중화민국</a>이 더 나은 식자율을 보여 주지만 50배의 인구와 260배의 영토를 가진 <a href="/wiki/%EC%A4%91%ED%99%94%EC%9D%B8%EB%AF%BC%EA%B3%B5%ED%99%94%EA%B5%AD" title="중화인민공화국">중화인민공화국</a>과 비교하기는 어렵다. 그리고 지역적으로 보면 <a href="/wiki/%EA%B4%91%EB%91%A5_%EC%84%B1" title="광둥 성">광둥 성</a>의 문맹률은 3.84%(2004년), <a href="/wiki/%EA%B4%91%EC%8B%9C_%EC%A2%A1%EC%A1%B1_%EC%9E%90%EC%B9%98%EA%B5%AC" title="광시 좡족 자치구">광시 좡족 자치구</a>의 문맹률은 3.79%인데 대만의 문맹률은 이것보다 약간 낮은 정도에 불과하다.</p>
<h3><span class="mw-headline" id=".EC.A0.84.ED.86.B5_.EB.AC.B8.ED.99.94.EC.9D.98_.EB.B3.B4.EC.A1.B4">전통 문화의 보존</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EA%B0%84%EC%B2%B4%EC%9E%90&amp;action=edit&amp;section=8" title="부분 편집: 전통 문화의 보존">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<h4><span class="mw-headline" id=".EC.A0.95.EC.B2.B4.EC.9E.90.EC.9D.98_.EC.9E.85.EC.9E.A5">정체자의 입장</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EA%B0%84%EC%B2%B4%EC%9E%90&amp;action=edit&amp;section=9" title="부분 편집: 정체자의 입장">편집</a><span class="mw-editsection-bracket">]</span></span></h4>
<p><a href="/wiki/%EC%8A%A4%EC%9B%A8%EB%8D%B4" title="스웨덴">스웨덴</a>의 <a href="/wiki/%EC%A4%91%EA%B5%AD" title="중국">중국</a>학자 베른하르트 카를그렌은 <a href="/wiki/1929%EB%85%84" title="1929년">1929년</a>에 "중국이 <a href="/wiki/%ED%95%9C%EC%9E%90" title="한자">한자</a>를 버리는 날, 중국은 중국 문화의 근본을 포기하는 것이다."라고 경고한 바 있다. 전통 문화를 상징하는 <a href="/wiki/%ED%95%9C%EC%9E%90" title="한자">한자</a>를 간체화하려는 주체인 <a href="/wiki/%EC%A4%91%ED%99%94%EC%9D%B8%EB%AF%BC%EA%B3%B5%ED%99%94%EA%B5%AD" title="중화인민공화국">중화인민공화국</a> 정부는 중국 전통 문화를 <a class="mw-redirect" href="/wiki/%EB%AC%B8%ED%99%94%EB%8C%80%ED%98%81%EB%AA%85" title="문화대혁명">문화대혁명</a> 등을 통해 무자비하게 훼손해온 바 있다. 아울러 <a href="/wiki/%EB%AF%B8%EA%B5%AD" title="미국">미국</a>과 <a href="/wiki/%EC%98%81%EA%B5%AD" title="영국">영국</a>과 <a href="/wiki/%ED%94%84%EB%9E%91%EC%8A%A4" title="프랑스">프랑스</a>와 <a href="/wiki/%EB%9F%AC%EC%8B%9C%EC%95%84" title="러시아">러시아</a>와 <a href="/wiki/%EB%8F%85%EC%9D%BC" title="독일">독일</a> 등 선진국들도 중국의 전통 문화를 파괴한 주범인 간체자를 강력히 반대하고, 중국의 전통 문화를 되살리는 영웅인 정체자로 완전히 지정되어 모든 국민들이 정체자를 쓰는 그 날, 중국이 비로소 미국을 밀어내고 전 세계에서 최고의 대제국으로 군림하게 될 것이라고 주장한 바 있다.</p>
<h4><span class="mw-headline" id=".EA.B0.84.EC.B2.B4.EC.9E.90.EC.9D.98_.EC.9E.85.EC.9E.A5">간체자의 입장</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EA%B0%84%EC%B2%B4%EC%9E%90&amp;action=edit&amp;section=10" title="부분 편집: 간체자의 입장">편집</a><span class="mw-editsection-bracket">]</span></span></h4>
<p>중국 대륙 정부 이전에도 간체화의 움직임은 <a href="/wiki/20%EC%84%B8%EA%B8%B0" title="20세기">20세기</a> 초반부터 있었다. <a href="/wiki/1950%EB%85%84" title="1950년">1950년</a>대 시작된 간체화는 <a href="/wiki/1960%EB%85%84%EB%8C%80" title="1960년대">1960년대</a>에 와서야 시작되는 <a class="mw-redirect" href="/wiki/%EB%AC%B8%ED%99%94%EB%8C%80%ED%98%81%EB%AA%85" title="문화대혁명">문화대혁명</a>과는 거리가 멀다.</p>
<h3><span class="mw-headline" id=".ED.98.BC.EB.8F.99.EC.9D.98_.EB.AC.B8.EC.A0.9C">혼동의 문제</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EA%B0%84%EC%B2%B4%EC%9E%90&amp;action=edit&amp;section=11" title="부분 편집: 혼동의 문제">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<dl>
<dt>정체자의 입장</dt>
</dl>
<p>無를 无로 간화해 天과 상당히 헷갈린다. 設과 沒를 각각 设과 没로 간화하여 평시 손으로 적는 경우에는 분별이 어렵다. 活와 話도 活와 话로 바뀌어 필기할 때 혼동되기 상당히 쉽다. 한자 공부를 하는 학생들과 학자들의 입장에서는 상당히 짜증이 날 수 있다.</p>

<dt>간체자의 입장</dt>
</dl>

<h3><span class="mw-headline" id=".EC.93.B0.EB.8A.94_.EC.86.8D.EB.8F.84">쓰는 속도</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EA%B0%84%EC%B2%B4%EC%9E%90&amp;action=edit&amp;section=12" title="부분 편집: 쓰는 속도">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<dl>
<dt>정체자의 입장</dt>
</dl>
<p>옛날 글로 직접 쓰는 속도는 문제가 되었지만, 현대 시대에 들어 컴퓨터 시대를 맞이하자 컴퓨터로 쓰는 속도는 전혀 문제가 되지 않는다. <a href="/wiki/%EC%9E%85%EB%A0%A5%EA%B8%B0" title="입력기">입력기</a>(IME)등을 통해 손쉽게 입력할 수 있다. 알파벳인 <a class="mw-redirect" href="/wiki/%ED%95%9C%EC%96%B4%EB%B3%91%EC%9D%8C" title="한어병음">한어병음</a>이나 별도로 개발된 <a href="/wiki/%EC%A3%BC%EC%9D%8C%EB%B6%80%ED%98%B8" title="주음부호">주음부호</a>를 통해 컴퓨터로 한자를 입력하고 있고, 정체자든 간체자든 입력 속도는 비슷하다. 손으로 쓰는 경우에 있어서도, 여전히 많은 사람들은 <a href="/wiki/%ED%96%89%EC%84%9C" title="행서">행서</a>를 통해 쓰는 시간을 줄이고 <a href="/wiki/%EC%B4%88%EC%84%9C" title="초서">초서</a>의 경우는 <a href="/wiki/%EC%86%8D%EA%B8%B0" title="속기">속기</a>시 이용된다.</p>
<dl>
<dt>간체자의 입장</dt>
</dl>
<p>획수가 적어 쓰기가 편하고 빠르다. 15획이 넘어가는 글자가 정체자에 많은데 쓰기가 특히 어렵다. <small><a href="/wiki/%EC%A4%91%EA%B5%AD%EC%96%B4" title="중국어">중국어</a></small> <small><a href="/wiki/%EC%A0%95%EC%B2%B4%EC%9E%90" title="정체자">정체</a>:</small> <span lang="zh-hant" xml:lang="zh-hant">邊</span>은 무려 18획인데 비해 <small><a href="/wiki/%EC%A4%91%EA%B5%AD%EC%96%B4" title="중국어">중국어</a></small> <small><strong class="selflink">간체</strong>:</small> <span lang="zh-hans" xml:lang="zh-hans">边</span>은 고작 5획이다.</p>
<h3><span class="mw-headline" id=".ED.98.95.EC.84.B1.EC.9E.90_.EC.9B.90.EB.A6.AC">형성자 원리</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EA%B0%84%EC%B2%B4%EC%9E%90&amp;action=edit&amp;section=13" title="부분 편집: 형성자 원리">편집</a><span class="mw-editsection-bracket">]</span></span></h3>

href="/wiki/%EC%A0%95%EC%B2%B4%EC%9E%90" title="정체자">정체</a>:</small> <span lang="zh-hant" xml:lang="zh-hant">盧</span>, <small><a href="/wiki/%ED%95%9C%EC%96%B4_%EB%B3%91%EC%9D%8C" title="한어 병음">병음</a>:</small> <font face="Lucida Sans Unicode">lú</font>와 <small><a href="/wiki/%EC%A4%91%EA%B5%AD%EC%96%B4" title="중국어">중국어</a></small> <small><a href="/wiki/%EC%A0%95%EC%B2%B4%EC%9E%90" title="정체자">정체</a>:</small> <span lang="zh-hant" xml:lang="zh-hant">爐</span>, <small><a href="/wiki/%ED%95%9C%EC%96%B4_%EB%B3%91%EC%9D%8C" title="한어 병음">병음</a>:</small> <font face="Lucida Sans Unicode">lú</font>는 盧를 공통 부분으로 하고 있던 것을 卢와 炉로 간화하였는데 炉의 발음은 <small><a href="/wiki/%EC%A4%91%EA%B5%AD%EC%96%B4" title="중국어">중국어</a></small><small>:</small> <span lang="zh" xml:lang="zh">戶</span>, <small><a href="/wiki/%ED%95%9C%EC%96%B4_%EB%B3%91%EC%9D%8C" title="한어 병음">병음</a>:</small> <font face="Lucida Sans Unicode">hù</font>와는 멀다.</p>
<dl>
<dt>간체자의 입장</dt>
</dl>
<p>많은 한자가 뜻 부분<sup class="reference" id="cite_ref-2"><a href="#cite_note-2">[2]</a></sup>과 음 부분 글자의 결합하여 만드는데<sup class="reference" id="cite_ref-3"><a href="#cite_note-3">[3]</a></sup>, 간체화를 통해 형성자 원리에 부합되는 간체자가 많이 만들어졌다. <small><a href="/wiki/%EC%A4%91%EA%B5%AD%EC%96%B4" title="중국어">중국어</a></small> <small><a href="/wiki/%EC%A0%95%EC%B2%B4%EC%9E%90" title="정체자">정체</a>:</small> <span lang="zh-hant" xml:lang="zh-hant">憂</span>, <small><a href="/wiki/%ED%95%9C%EC%96%B4_%EB%B3%91%EC%9D%8C" title="한어 병음">병음</a>:</small> <font face="Lucida Sans Unicode">you</font><font color="gray"><small>→걱정</small></font>를 忧로 간화함으로써 心이 뜻은 살리면서 <small><a href="/wiki/%EC%A4%91%EA%B5%AD%EC%96%B4" title="중국어">중국어</a></small> <small><strong class="selflink">간체</strong>:</small> <span lang="zh-hans" xml:lang="zh-hans">尤</span>, <small><a href="/wiki/%ED%95%9C%EC%96%B4_%EB%B3%91%EC%9D%8C" title="한어 병음">병음</a>:</small> <font face="Lucida Sans Unicode">yóu</font>의 발음도 글자에 들어갔다.</p>
<h3><span class="mw-headline" id=".EB.B6.80.EC.88.98">부수</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EA%B0%84%EC%B2%B4%EC%9E%90&amp;action=edit&amp;section=14" title="부분 편집: 부수">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<dl>
<dt>정체자의 입장</dt>
</dl>
<p>급격한 자형의 변화로 초래된 한자간 연결고리가 끊어져 뜻에 해당하는 한자, 소리에 해당한자를 모두 익힌 학생도 뜻+소리로 결합된 한자를 익힐 때 어려움을 겪는다. 鬧가 闹로 간화되어 門이 부수가 되는 오류를 낳았다. 愛를 爱로 간화해 心의 부수가 없어졌다.</p>
<dl>
<dt>간체자의 입장</dt>
</dl>
<p>처음부터 <a href="/wiki/%EB%B6%80%EC%88%98" title="부수">부수</a> 체계가 완벽하게 갖추기는 어려운 법이고 笑조차도 웃음이라는 뜻과 무관한 대나무 竹을 부수로 쓰고 있다.</p>
<h3><span class="mw-headline" id=".EA.B8.80.EC.9E.90.EC.9D.98_.ED.86.B5.ED.95.A9">글자의 통합</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EA%B0%84%EC%B2%B4%EC%9E%90&amp;action=edit&amp;section=15" title="부분 편집: 글자의 통합">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<dl>
<dt>정체자의 입장</dt>
</dl>
<p>後와 后를 后로 통합, 隻와 只를 只로 통합, 發와 髮를 发로 통합, 週와 周를 周로 통합, 穀와 谷을 谷로 통합하는 등 분별이 상당히 어렵다. 그래서 한자 공부를 하는 학생들과 학자들의 입장에서는 상당히 짜증이 날 수 있다.</p>
<dl>
<dt>간체자의 입장</dt>
</dl>
<p>글자 간 통합으로 오는 모호함은 적은 편이다.</p>
