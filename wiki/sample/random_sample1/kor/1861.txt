<p><b>갈 가도트</b>(<span style="font-size: smaller;"><a href="/wiki/%ED%9E%88%EB%B8%8C%EB%A6%AC%EC%96%B4" title="히브리어">히브리어</a>:</span> <span lang="he" xml:lang="he">גל גדות</span>, <span style="font-size: smaller;"><a href="/wiki/%EC%98%81%EC%96%B4" title="영어">영어</a>:</span> <span lang="en" xml:lang="en">Gal Gadot</span>, <a href="/wiki/1985%EB%85%84" title="1985년">1985년</a> <a href="/wiki/4%EC%9B%94_10%EC%9D%BC" title="4월 10일">4월 10일</a> ~ )는 <a href="/wiki/%EC%9D%B4%EC%8A%A4%EB%9D%BC%EC%97%98" title="이스라엘">이스라엘</a>의 <a href="/wiki/%EB%AA%A8%EB%8D%B8_(%EC%A7%81%EC%97%85)" title="모델 (직업)">모델</a> 겸 <a href="/wiki/%EB%B0%B0%EC%9A%B0" title="배우">배우</a>이다. 2004년 미스 이스라엘 에서 우승하여 그 해 <a href="/wiki/%EB%AF%B8%EC%8A%A4_%EC%9C%A0%EB%8B%88%EB%B2%84%EC%8A%A4" title="미스 유니버스">미스 유니버스</a> 대회에 참가했다. 영화 《<a href="/wiki/%EB%B6%84%EB%85%B8%EC%9D%98_%EC%A7%88%EC%A3%BC_%EC%8B%9C%EB%A6%AC%EC%A6%88" title="분노의 질주 시리즈">분노의 질주 시리즈</a>》의 "지젤"(Gisele) 역, <a href="/wiki/DC_%ED%99%95%EC%9E%A5_%EC%9C%A0%EB%8B%88%EB%B2%84%EC%8A%A4" title="DC 확장 유니버스">DC 확장 유니버스</a>의 <a href="/wiki/%EC%9B%90%EB%8D%94%EC%9A%B0%EB%A8%BC" title="원더우먼">다이애나 프린스/원더우먼</a>으로 유명하다.<sup class="reference" id="cite_ref-1"><a href="#cite_note-1">[1]</a></sup><sup class="reference" id="cite_ref-2"><a href="#cite_note-2">[2]</a></sup></p>
<p>2013년, 이스라엘에서 2번째로 수익이 높은 모델로 선정됐다.<sup class="reference" id="cite_ref-3"><a href="#cite_note-3">[3]</a></sup></p>

<h2><span class="mw-headline" id=".EC.83.9D.EC.95.A0">생애</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EA%B0%88_%EA%B0%80%EB%8F%84%ED%8A%B8&amp;action=edit&amp;section=1" title="부분 편집: 생애">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>1985년 4월 10일, <a href="/wiki/%EC%9D%B4%EC%8A%A4%EB%9D%BC%EC%97%98" title="이스라엘">이스라엘</a> 로쉬 하야인에서 태어났다. 그녀의 성 "가도트"는 원래 "그린스타인"(Greenstein)이였으나, 가도트의 부모님이 성을 <a href="/wiki/%ED%9E%88%EB%B8%8C%EB%A6%AC%EC%96%B4" title="히브리어">히브리어</a>화하면서 바꼈다.<sup class="reference" id="cite_ref-4"><a href="#cite_note-4">[4]</a></sup> 히브리어로 성은 "강둑", "강기슭"을 뜻하고, 이름은 "파도"를 뜻한다.<sup class="reference" id="cite_ref-5"><a href="#cite_note-5">[5]</a></sup> 아버지는 엔지니어, 어머니는 교사였다. <a href="/wiki/%EC%9C%A0%EB%8C%80%EC%9D%B8" title="유대인">유대</a>, <a href="/wiki/%ED%8F%B4%EB%9E%80%EB%93%9C" title="폴란드">폴란드</a>, <a href="/wiki/%EC%98%A4%EC%8A%A4%ED%8A%B8%EB%A6%AC%EC%95%84" title="오스트리아">오스트리아</a>, <a href="/wiki/%EB%8F%85%EC%9D%BC" title="독일">독일</a>, <a href="/wiki/%EC%B2%B4%EC%BD%94" title="체코">체코</a>계 혈통을 잇고 있다. 갤은 독실한 이스라엘 유대교 집안에서 자라, 유대인이자 이스라엘인으로서 정체성이 강하다고 밝혔다.<sup class="reference" id="cite_ref-6"><a href="#cite_note-6">[6]</a></sup></p>
<h3><span class="mw-headline" id=".EA.B5.B0_.EB.B3.B5.EB.AC.B4">군 복무</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EA%B0%88_%EA%B0%80%EB%8F%84%ED%8A%B8&amp;action=edit&amp;section=2" title="부분 편집: 군 복무">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>20살 때부터 2년 간 <a href="/wiki/%EC%9D%B4%EC%8A%A4%EB%9D%BC%EC%97%98_%EB%B0%A9%EC%9C%84%EA%B5%B0" title="이스라엘 방위군">이스라엘 방위군</a>에서 복무했다. 전투 훈련 조교(Combat Trainer)가 되기 위해, 훈련소에서 3달 간 훈련을 받고 우수한 성적을 거뒀다.<sup class="reference" id="cite_ref-glamour.com_7-0"><a href="#cite_note-glamour.com-7">[7]</a></sup></p>
<h2><span class="mw-headline" id=".EA.B2.BD.EB.A0.A5">경력</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EA%B0%88_%EA%B0%80%EB%8F%84%ED%8A%B8&amp;action=edit&amp;section=3" title="부분 편집: 경력">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>가도트는 원래 변호사가 되려고 했다.<sup class="reference" id="cite_ref-glamour.com_7-1"><a href="#cite_note-glamour.com-7">[7]</a></sup> 그러다 18살 때 미스 이스라엘 2004 대회에 나가 우승했고, 이어서 <a href="/wiki/%EC%97%90%EC%BD%B0%EB%8F%84%EB%A5%B4" title="에콰도르">에콰도르</a>에서 열린 <a href="/wiki/%EB%AF%B8%EC%8A%A4_%EC%9C%A0%EB%8B%88%EB%B2%84%EC%8A%A4" title="미스 유니버스">미스 유니버스</a> 대회에 참가했다. 그 후 입대해 2년 간의 군 복무를 마쳤다.<sup class="reference" id="cite_ref-8"><a href="#cite_note-8">[8]</a></sup></p>
<p>제대 후 대학에서 <a href="/wiki/%EB%B2%95%ED%95%99" title="법학">법학</a>을 공부했다. 1학년을 마칠 무렵, 캐스팅 디렉터가 가도트에게 영화 《<a href="/wiki/007_%ED%80%80%ED%85%80_%EC%98%A4%EB%B8%8C_%EC%86%94%EB%9F%AC%EC%8A%A4" title="007 퀀텀 오브 솔러스">007 퀀텀 오브 솔러스</a>》 본드걸 역 오디션을 제의했다. 오디션에 떨어져 본드걸 역에는 <a href="/wiki/%EC%98%AC%EA%B0%80_%EC%BF%A0%EB%A6%B4%EB%A0%8C%EC%BD%94" title="올가 쿠릴렌코">올가 쿠릴렌코</a>가 낙점됐지만, 대신 가도트는 영화 《<a href="/wiki/%EB%B6%84%EB%85%B8%EC%9D%98_%EC%A7%88%EC%A3%BC:_%EB%8D%94_%EC%98%A4%EB%A6%AC%EC%A7%80%EB%84%90" title="분노의 질주: 더 오리지널">분노의 질주: 더 오리지널</a>》의 "지젤" 역에 캐스팅됐다.</p>
<p>2007년, <a href="/wiki/%EB%A7%A5%EC%8B%AC_(%EC%9E%A1%EC%A7%80)" title="맥심 (잡지)">맥심</a>에서 이스라엘 여군들의 사진을 찍을 때 당시 21살인 가도트도 참여했는데, 가도트의 사진이 <a href="/wiki/%EB%89%B4%EC%9A%95_%ED%8F%AC%EC%8A%A4%ED%8A%B8" title="뉴욕 포스트">뉴욕 포스트</a> 커버를 장식하게 되었다.</p>
<h3><span class="mw-headline" id=".EB.AA.A8.EB.8D.B8">모델</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EA%B0%88_%EA%B0%80%EB%8F%84%ED%8A%B8&amp;action=edit&amp;section=4" title="부분 편집: 모델">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p><a class="new" href="/w/index.php?title=%EC%BA%A1%ED%8B%B4_%EB%AA%A8%EB%A5%B4%EA%B0%84&amp;action=edit&amp;redlink=1" title="캡틴 모르간 (없는 문서)">캡틴 모르간</a> 럼주, <a href="/wiki/%EA%B5%AC%EC%B0%8C" title="구찌">구찌</a> 향수, 바인 베라(Vine Vera) 스킨케어, <a href="/wiki/%EC%9E%AC%EA%B7%9C%EC%96%B4_%EC%9E%90%EB%8F%99%EC%B0%A8" title="재규어 자동차">재규어 자동차</a> 광고에 출연했다. 현재 구찌의 뱀부(Bamboo) 향수,<sup class="reference" id="cite_ref-hollywoodreporter.com_9-0"><a href="#cite_note-hollywoodreporter.com-9">[9]</a></sup> <a href="/wiki/%ED%99%94%EC%9B%A8%EC%9D%B4" title="화웨이">화웨이</a> 스마트폰과 이탈리아의 패션 브랜드 <a class="new" href="/w/index.php?title=%EB%AF%B8%EC%8A%A4_%EC%8B%9D%EC%8A%A4%ED%8B%B0&amp;action=edit&amp;redlink=1" title="미스 식스티 (없는 문서)">미스 식스티</a>(Miss Sixty)의 얼굴이다. <a href="/wiki/%EC%BD%94%EC%8A%A4%EB%AA%A8%ED%8F%B4%EB%A6%AC%ED%83%84" title="코스모폴리탄">코스모폴리탄</a>, <a href="/wiki/%EA%B8%80%EB%9E%98%EB%A8%B8_(%EC%9E%A1%EC%A7%80)" title="글래머 (잡지)">글래머</a>, <a href="/wiki/FHM" title="FHM">FHM</a>, 브라이드(Bride)지의 커버를 장식했다. 이스라엘 의류 회사 <a class="mw-disambig" href="/wiki/%EC%B9%B4%EC%8A%A4%ED%8A%B8%EB%A1%9C" title="카스트로">카스트로</a>의 모델이기도 했다.<sup class="reference" id="cite_ref-10"><a href="#cite_note-10">[10]</a></sup> 2013년, 연봉 2백 40만 <a href="/wiki/%EC%9D%B4%EC%8A%A4%EB%9D%BC%EC%97%98_%EC%8B%A0_%EC%85%B0%EC%BC%88" title="이스라엘 신 셰켈">셰켈</a>(NIS) 이상을 벌었다고 추산됐다.<sup class="reference" id="cite_ref-11"><a href="#cite_note-11">[11]</a></sup></p>
<h3><span class="mw-headline" id=".EC.97.B0.EA.B8.B0">연기</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EA%B0%88_%EA%B0%80%EB%8F%84%ED%8A%B8&amp;action=edit&amp;section=5" title="부분 편집: 연기">편집</a><span class="mw-editsection-bracket">]</span></span></h3>

<p>2008년, 이스라엘 드라마 《Bubot》의 주연을 맡았다. 그리고 《<a href="/wiki/%EB%B6%84%EB%85%B8%EC%9D%98_%EC%A7%88%EC%A3%BC_%EC%8B%9C%EB%A6%AC%EC%A6%88" title="분노의 질주 시리즈">분노의 질주 시리즈</a>》의 "지젤" 역을 맡았다.<sup class="reference" id="cite_ref-12"><a href="#cite_note-12">[12]</a></sup><sup class="reference" id="cite_ref-13"><a href="#cite_note-13">[13]</a></sup> 분노의 질주 시리즈에서 모든 스턴트 액션을 직접 소화했다.<sup class="reference" id="cite_ref-JewishJournal_14-0"><a href="#cite_note-JewishJournal-14">[14]</a></sup> 2010년, 영화 《<a class="new" href="/w/index.php?title=%EB%82%98%EC%9E%87_%26_%EB%8D%B0%EC%9D%B4&amp;action=edit&amp;redlink=1" title="나잇 &amp; 데이 (없는 문서)">나잇 &amp; 데이</a>》(Knight and Day)에서 조연을 맡았다.<sup class="reference" id="cite_ref-15"><a href="#cite_note-15">[15]</a></sup> 영화 《<a href="/wiki/%EB%B8%8C%EB%A1%9C%ED%81%B0_%EB%8D%B0%EC%9D%B4%ED%8A%B8" title="브로큰 데이트">브로큰 데이트</a>》에서는 <a href="/wiki/%EB%A7%88%ED%81%AC_%EC%9B%94%EB%B2%84%EA%B7%B8" title="마크 월버그">마크 월버그</a>가 맡은 역할의 여자친구인 "나타냐"를 연기했다.</p>
<p>영화 《<a href="/wiki/%EB%B0%B0%ED%8A%B8%EB%A7%A8_%EB%8C%80_%EC%8A%88%ED%8D%BC%EB%A7%A8:_%EC%A0%80%EC%8A%A4%ED%8B%B0%EC%8A%A4%EC%9D%98_%EC%8B%9C%EC%9E%91" title="배트맨 대 슈퍼맨: 저스티스의 시작">배트맨 대 슈퍼맨: 저스티스의 시작</a>》에서 <a href="/wiki/%EC%9B%90%EB%8D%94%EC%9A%B0%EB%A8%BC" title="원더우먼">원더우먼</a>으로 출연했다. 2017년 개봉 예정 영화 《<a href="/wiki/%EC%A0%80%EC%8A%A4%ED%8B%B0%EC%8A%A4_%EB%A6%AC%EA%B7%B8_(2017%EB%85%84_%EC%98%81%ED%99%94)" title="저스티스 리그 (2017년 영화)">저스티스 리그</a>》와 《<a href="/wiki/%EC%9B%90%EB%8D%94%EC%9A%B0%EB%A8%BC_(2017%EB%85%84_%EC%98%81%ED%99%94)" title="원더우먼 (2017년 영화)">원더우먼</a>》에서도 원더우먼을 연기했다.<sup class="reference" id="cite_ref-16"><a href="#cite_note-16">[16]</a></sup><sup class="reference" id="cite_ref-17"><a href="#cite_note-17">[17]</a></sup><sup class="reference" id="cite_ref-18"><a href="#cite_note-18">[18]</a></sup> 가도트는 이 역할을 위해 검술, <a class="mw-redirect" href="/wiki/%EC%BF%B5%ED%91%B8" title="쿵푸">쿵푸</a>, <a href="/wiki/%ED%82%A5%EB%B3%B5%EC%8B%B1" title="킥복싱">킥복싱</a>, <a class="mw-disambig" href="/wiki/%EC%B9%B4%ED%8F%AC%EC%97%90%EB%9D%BC" title="카포에라">카포에라</a>, <a class="mw-redirect" href="/wiki/%EC%A3%BC%EC%A7%93%EC%88%98" title="주짓수">주짓수</a>를 배웠다.<sup class="reference" id="cite_ref-19"><a href="#cite_note-19">[19]</a></sup></p>
<p>2016년, <a href="/wiki/%EC%A1%B4_%ED%9E%90%EC%BD%94%ED%8A%B8" title="존 힐코트">존 힐코트</a> 감독의 영화 《<a href="/wiki/%ED%8A%B8%EB%A6%AC%ED%94%8C_9" title="트리플 9">트리플 9</a>》에서 "엘레나" 역을 맡았다.<sup class="reference" id="cite_ref-20"><a href="#cite_note-20">[20]</a></sup> 같은 해, 영화 《<a href="/wiki/%ED%81%AC%EB%A6%AC%EB%AF%B8%EB%84%90_(2016%EB%85%84_%EC%98%81%ED%99%94)" title="크리미널 (2016년 영화)">크리미널</a>》에서는 <a href="/wiki/%EB%9D%BC%EC%9D%B4%EC%96%B8_%EB%A0%88%EC%9D%B4%EB%86%80%EC%A6%88" title="라이언 레이놀즈">라이언 레이놀즈</a>의 배역의 아내를 연기했다.<sup class="reference" id="cite_ref-21"><a href="#cite_note-21">[21]</a></sup></p>
<h2><span class="mw-headline" id=".EC.82.AC.EC.83.9D.ED.99.9C">사생활</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EA%B0%88_%EA%B0%80%EB%8F%84%ED%8A%B8&amp;action=edit&amp;section=6" title="부분 편집: 사생활">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
