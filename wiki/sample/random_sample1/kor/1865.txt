<p><b>가라 유자와 역</b>(<span style="font-size: smaller;"><a href="/wiki/%EC%9D%BC%EB%B3%B8%EC%96%B4" title="일본어">일본어</a>:</span> <span lang="ja" xml:lang="ja">ガーラ湯沢駅</span>)은 <a href="/wiki/%EC%9D%BC%EB%B3%B8" title="일본">일본</a> <a href="/wiki/%EB%8B%88%EA%B0%80%ED%83%80_%ED%98%84" title="니가타 현">니가타 현</a> <a href="/wiki/%EB%AF%B8%EB%82%98%EB%AF%B8%EC%9A%B0%EC%98%A4%EB%88%84%EB%A7%88_%EA%B5%B0" title="미나미우오누마 군">미나미우오누마 군</a> <a href="/wiki/%EC%9C%A0%EC%9E%90%EC%99%80_%EC%A0%95" title="유자와 정">유자와 정</a>에 있는 <a class="mw-redirect" href="/wiki/%EB%8F%99%EC%9D%BC%EB%B3%B8_%EC%97%AC%EA%B0%9D%EC%B2%A0%EB%8F%84" title="동일본 여객철도">동일본 여객철도</a>의 <a href="/wiki/%EC%B2%A0%EB%8F%84%EC%97%AD" title="철도역">역</a>이다. 이 역은 실질적으로는 <a href="/wiki/%EC%97%90%EC%B9%98%EA%B3%A0%EC%9C%A0%EC%9E%90%EC%99%80_%EC%97%AD" title="에치고유자와 역">에치고유자와 역</a>에서 분기하는 <a href="/wiki/%EC%A1%B0%EC%97%90%EC%93%B0_%EC%8B%A0%EC%B9%B8%EC%84%BC" title="조에쓰 신칸센">조에쓰 신칸센</a> 지선의 종착역에 해당하나, 법규상으로는 재래선인 <a href="/wiki/%EC%A1%B0%EC%97%90%EC%93%B0_%EC%84%A0" title="조에쓰 선">조에쓰 선</a>의 소속이다.</p>
