<p><b>갈리아 키살피나</b>(<span style="font-size: smaller;"><a href="/wiki/%EB%9D%BC%ED%8B%B4%EC%96%B4" title="라틴어">라틴어</a>:</span> <span lang="la" xml:lang="la">Gallia Cisalpina</span>, "<a href="/wiki/%EC%95%8C%ED%94%84%EC%8A%A4_%EC%82%B0%EB%A7%A5" title="알프스 산맥">알프스</a> 이쪽의 갈리아")는 <a class="mw-redirect" href="/wiki/%EA%B3%B5%ED%99%94%EC%A0%95_%EB%A1%9C%EB%A7%88" title="공화정 로마">공화정 로마</a>의 <a class="mw-redirect" href="/wiki/%EB%A1%9C%EB%A7%88_%EC%86%8D%EC%A3%BC" title="로마 속주">속주</a>였다. 현재 <a href="/wiki/%EC%9D%B4%ED%83%88%EB%A6%AC%EC%95%84" title="이탈리아">이탈리아</a> 북부의 <a class="new" href="/w/index.php?title=%EC%97%90%EB%B0%80%EB%A6%AC%EC%95%84&amp;action=edit&amp;redlink=1" title="에밀리아 (없는 문서)">에밀리아</a>와 <a class="mw-redirect" href="/wiki/%EB%A1%AC%EB%B0%94%EB%A5%B4%EB%94%94%EC%95%84" title="롬바르디아">롬바르디아</a>에 해당하는 지역으로, <b>프로빈치아 아리미눔</b>(Provincia Ariminum)이라는 이름으로도 불렸다. 속주 수도는 무티나(현재 <a href="/wiki/%EB%AA%A8%EB%8D%B0%EB%82%98" title="모데나">모데나</a>)로, <a href="/wiki/%EA%B8%B0%EC%9B%90%EC%A0%84_73%EB%85%84" title="기원전 73년">기원전 73년</a> <a class="new" href="/w/index.php?title=%EC%8A%A4%ED%8C%8C%EB%A5%B4%ED%83%80%EC%BF%A0%EC%8A%A4_%EB%B0%98%EB%9E%80&amp;action=edit&amp;redlink=1" title="스파르타쿠스 반란 (없는 문서)">스파르타쿠스</a>의 군대가 속주 총독 가이우스 카시우스 롱기누스를 물리친 곳이다.</p>
<p>갈리아 키살피나는 <a href="/wiki/%EA%B8%B0%EC%9B%90%EC%A0%84_43%EB%85%84" title="기원전 43년">기원전 43년</a>-<a href="/wiki/%EA%B8%B0%EC%9B%90%EC%A0%84_42%EB%85%84" title="기원전 42년">42년</a>경 <a class="mw-redirect" href="/wiki/%EC%98%A5%ED%83%80%EB%B9%84%EC%95%84%EB%88%84%EC%8A%A4" title="옥타비아누스">옥타비아누스</a>의 정책에 따라 <a class="mw-redirect" href="/wiki/%EC%A0%9C2%EC%B0%A8_%EC%82%BC%EB%91%90%EC%A0%95%EC%B9%98" title="제2차 삼두정치">제2차 삼두정치</a>에 의해 이탈리아에 흡수되었고 이탈리아의 경계는 과거의 <a href="/wiki/%EB%A3%A8%EB%B9%84%EC%BD%98_%EA%B0%95" title="루비콘 강">루비콘 강</a>에서 <a href="/wiki/%ED%8F%AC_%EA%B0%95" title="포 강">포 강</a>까지 북쪽으로 확대되었다.</p>
