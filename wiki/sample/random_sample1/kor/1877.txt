<p><b>세르비우스 술피키우스 갈바</b>(<a href="/wiki/%EA%B8%B0%EC%9B%90%EC%A0%84_3%EB%85%84" title="기원전 3년">기원전 3년</a> <a href="/wiki/12%EC%9B%94_24%EC%9D%BC" title="12월 24일">12월 24일</a> - <a href="/wiki/69%EB%85%84" title="69년">69년</a> <a href="/wiki/1%EC%9B%94_15%EC%9D%BC" title="1월 15일">1월 15일</a>)는 <a href="/wiki/%EB%A1%9C%EB%A7%88_%EC%A0%9C%EA%B5%AD" title="로마 제국">로마 제국</a>의 여섯 번째 황제이다. 갈바가 <a href="/wiki/69%EB%85%84" title="69년">69년</a> <a href="/wiki/1%EC%9B%94_15%EC%9D%BC" title="1월 15일">1월 15일</a>에 죽음으로써 69년에 죽은 첫 번째 황제(왕)이 되었다.</p>
<h2><span class="mw-headline" id=".EC.83.9D.EC.95.A0">생애</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EA%B0%88%EB%B0%94&amp;action=edit&amp;section=1" title="부분 편집: 생애">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>갈바는 <a href="/wiki/%EA%B8%B0%EC%9B%90%EC%A0%84_3%EB%85%84" title="기원전 3년">기원전 3년</a> <a href="/wiki/12%EC%9B%94_24%EC%9D%BC" title="12월 24일">12월 24일</a>, <a href="/wiki/%EB%A1%9C%EB%A7%88" title="로마">로마</a>의 귀족 가문에서 태어났다. 그는 귀족의 가문에서 태어나 <a href="/wiki/20%EB%85%84" title="20년">20년</a>, 23살의 나이로 법무관에 임명되었다. <a href="/wiki/33%EB%85%84" title="33년">33년</a>에는 집정관에 임명되었고 <a href="/wiki/39%EB%85%84" title="39년">39년</a>에는 <a href="/wiki/%EC%B9%BC%EB%A6%AC%EA%B5%B4%EB%9D%BC" title="칼리굴라">칼리굴라</a>에 의해 <a href="/wiki/%EB%9D%BC%EC%9D%B8_%EA%B0%95" title="라인 강">라인 강</a> 방위군 지휘관에 임명되었다.</p>
<p><a href="/wiki/45%EB%85%84" title="45년">45년</a> 그는 <a href="/wiki/%EB%B6%81%EC%95%84%ED%94%84%EB%A6%AC%EC%B9%B4" title="북아프리카">북아프리카</a> 총독에 임명되었다. <a href="/wiki/49%EB%85%84" title="49년">49년</a>, 그는 <a class="new" href="/w/index.php?title=%EC%86%8C_%EC%95%84%EA%B7%B8%EB%A6%AC%ED%94%BC%EB%82%98&amp;action=edit&amp;redlink=1" title="소 아그리피나 (없는 문서)">소 아그리피나</a>가 그 당시 <a href="/wiki/%ED%81%B4%EB%9D%BC%EC%9A%B0%EB%94%94%EC%9A%B0%EC%8A%A4" title="클라우디우스">클라우디우스</a>와 결혼하였었는데 갈바는 그녀와 사이가 좋지 않았기에 10년 정도나 정치에서 은퇴생활을 해야 했다. <a href="/wiki/61%EB%85%84" title="61년">61년</a>, <a href="/wiki/%EC%9D%B4%EB%B2%A0%EB%A6%AC%EC%95%84_%EB%B0%98%EB%8F%84" title="이베리아 반도">이베리아 반도</a> 북동부의 총독에 임명되었다.</p>
<p><a href="/wiki/68%EB%85%84" title="68년">68년</a>에는 <a href="/wiki/%EB%84%A4%EB%A1%9C" title="네로">네로</a>에 대항하여 반란을 일으킨 <a class="new" href="/w/index.php?title=%EB%B9%88%EB%8D%B1%EC%8A%A4&amp;action=edit&amp;redlink=1" title="빈덱스 (없는 문서)">빈덱스</a>에 의해 반란의 주모자로 추대되었다. 빈덱스는 군대에 의해 진압당했지만 <a href="/wiki/68%EB%85%84" title="68년">68년</a> <a href="/wiki/6%EC%9B%94_8%EC%9D%BC" title="6월 8일">6월 8일</a> <a href="/wiki/%EC%9B%90%EB%A1%9C%EC%9B%90" title="원로원">원로원</a>에 의해 황제로 선포된다. 10월에 로마로 돌아왔는데 로마에 돌아와 그가 국정에 관하여 내린 조치는 실로 어처구니가 없었다. 그중 하나가 네로의 선물을 몰수한 것으로 이 조치는 재정을 다시 건전화시킬려고 그런 것이었겠지만 많은 사람들에게 수많은 비판을 받았다.</p>
<p>거기다 수하들까지 탐욕적으로 재물을 긁어모아 갈바에 대한 사람들의 신임은 더욱 실추되기 시작한다. <a href="/wiki/69%EB%85%84" title="69년">69년</a> <a href="/wiki/1%EC%9B%94_2%EC%9D%BC" title="1월 2일">1월 2일</a>에는 라인 강 방위군이 갈바가 황제가 된 데에 불만을 품고 <a href="/wiki/%EB%B9%84%ED%85%94%EB%A6%AC%EC%9A%B0%EC%8A%A4" title="비텔리우스">비텔리우스</a>를 황제로 추대하였다.</p>
<p><a href="/wiki/1%EC%9B%94_10%EC%9D%BC" title="1월 10일">1월 10일</a>에는 <a class="new" href="/w/index.php?title=%EB%A3%A8%ED%82%A4%EC%9A%B0%EC%8A%A4_%EC%B9%BC%ED%91%B8%EB%A5%B4%EB%8B%88%EC%9A%B0%EC%8A%A4_%ED%94%BC%EC%86%8C_%ED%94%84%EB%A3%A8%EA%B8%B0_%EB%A6%AC%ED%82%A4%EB%8B%88%EC%95%84%EB%88%84%EC%8A%A4&amp;action=edit&amp;redlink=1" title="루키우스 칼푸르니우스 피소 프루기 리키니아누스 (없는 문서)">루키우스 칼푸르니우스 피소 프루기 리키니아누스</a>(Lucius Calpurnius Piso Licinianus)를 입양하여 자신의 후계자임을 세상에 알렸는데 이는 자신이 갈바의 후계자라고 생각하고 있던 <a href="/wiki/%EC%98%A4%ED%86%A0_(%EB%A1%9C%EB%A7%88_%EC%A0%9C%EA%B5%AD)" title="오토 (로마 제국)">오토</a>를 분노케하여 오토의 <a href="/wiki/%EC%BF%A0%EB%8D%B0%ED%83%80" title="쿠데타">쿠데타</a>를 초래하는 결과를 낳았다.</p>
<p>오토는 <a href="/wiki/1%EC%9B%94_15%EC%9D%BC" title="1월 15일">1월 15일</a>에 황제를 수행하던 중 일행에서 빠져나와 근위대에 접근하여 황제로 추대되었다. 갈바는 이에 당황하여 반란군과 대항하려 했으나 오토 측의 기병에 공격을 받고 죽음을 당하였다. 그의 나이 72세였다.</p>
