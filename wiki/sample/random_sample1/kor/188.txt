<p><b>대구선</b>(大邱線)은 <a href="/wiki/%EA%B2%BD%EB%B6%80%EC%84%A0" title="경부선">경부선</a> <a href="/wiki/%EA%B0%80%EC%B2%9C%EC%97%AD" title="가천역">가천역</a>과 <a href="/wiki/%EC%A4%91%EC%95%99%EC%84%A0" title="중앙선">중앙선</a> <a href="/wiki/%EC%98%81%EC%B2%9C%EC%97%AD" title="영천역">영천역</a>을 잇는 <a href="/wiki/%ED%95%9C%EA%B5%AD%EC%B2%A0%EB%8F%84%EA%B3%B5%EC%82%AC" title="한국철도공사">한국철도공사</a>의 <a class="mw-redirect" href="/wiki/%EC%A7%80%EC%84%A0_%EC%B2%A0%EB%8F%84" title="지선 철도">지선 철도</a> 노선이다.</p>
<p>순수하게 대구선 구간만 운행하는 여객열차는 없고, 모두 <a href="/wiki/%EB%8F%99%ED%95%B4%EB%82%A8%EB%B6%80%EC%84%A0" title="동해남부선">동해남부선</a> (부전, 경주, 포항 방면)이나 <a href="/wiki/%EC%A4%91%EC%95%99%EC%84%A0" title="중앙선">중앙선</a> (영주, 제천, 청량리, 정동진 방면)으로 <a href="/wiki/%EC%A7%81%EA%B2%B0_%EC%9A%B4%ED%96%89" title="직결 운행">직결 운행</a>한다. 원래는 <a href="/wiki/%EB%8F%99%EB%8C%80%EA%B5%AC%EC%97%AD" title="동대구역">동대구</a>에서 출발했는데, 시지 (가천동) 쪽으로 선로가 이설된 후에는 출발역이 <a href="/wiki/%EA%B0%80%EC%B2%9C%EC%97%AD" title="가천역">가천</a>으로 이설되었다. 이설과 일부 역들의 여객취급 중단 등으로 현재 대구선의 여객취급역은 2개 (<a href="/wiki/%ED%95%98%EC%96%91%EC%97%AD" title="하양역">하양역</a>, <a href="/wiki/%EC%98%81%EC%B2%9C%EC%97%AD" title="영천역">영천역</a>)로 적은 편이다.</p>
<p>현재 복선 전철화를 하고 있고 하양읍 일대에서 고가선로를 올리는 공사를 하는 중이며, 이설된 후 쓸모없게 되는 구 선로는 <a href="/wiki/%EB%8C%80%EA%B5%AC_%EB%8F%84%EC%8B%9C%EC%B2%A0%EB%8F%84_1%ED%98%B8%EC%84%A0" title="대구 도시철도 1호선">대구 도시철도 1호선</a> 경산 연장 구간 공사에 활용할 예정이다.</p>

<h2><span class="mw-headline" id=".ED.96.A5.ED.9B.84.EC.9D.98_.EA.B3.84.ED.9A.8D">향후의 계획</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%8C%80%EA%B5%AC%EC%84%A0&amp;action=edit&amp;section=3" title="부분 편집: 향후의 계획">원본 편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>선로 개량과 복선전철화 계획이 있으며, <a href="/wiki/%EB%8C%80%EA%B5%AC%EA%B4%91%EC%97%AD%EC%8B%9C" title="대구광역시">대구광역시</a>에서는 이설되어 폐선이 될 현재의 청천 ~ 하양 구간을 활용하여 <a href="/wiki/%EB%8C%80%EA%B5%AC_%EB%8F%84%EC%8B%9C%EC%B2%A0%EB%8F%84_1%ED%98%B8%EC%84%A0" title="대구 도시철도 1호선">대구 도시철도 1호선</a>을 <a href="/wiki/%EC%95%88%EC%8B%AC%EC%97%AD" title="안심역">안심역</a> 이후로 <a href="/wiki/%ED%95%98%EC%96%91%EC%97%AD" title="하양역">하양역</a>까지 연장하는 방안을 계획하고 있으며<sup class="reference" id="cite_ref-9"><a href="#cite_note-9">[9]</a></sup>, 복선전철화는 <a href="/wiki/%EA%B2%BD%EB%B6%80%EC%84%A0" title="경부선">경부선</a>, <a href="/wiki/%EC%A4%91%EC%95%99%EC%84%A0" title="중앙선">중앙선</a>, <a href="/wiki/%EB%8F%99%ED%95%B4%EC%84%A0" title="동해선">동해선</a>과 연계를 위한 <a href="/wiki/%EA%B0%84%EC%84%A0%EC%B2%A0%EB%8F%84" title="간선철도">간선철도</a>로 계획되었으며, 현재 <a href="/wiki/%EA%B4%91%EC%97%AD%EC%B2%A0%EB%8F%84" title="광역철도">광역철도</a> 계획은 없다.</p>
<p>2015년 4월 1일 서울 - 포항 비전철 <a href="/wiki/%EC%83%88%EB%A7%88%EC%9D%84%ED%98%B8" title="새마을호">새마을호</a>가 폐지된 후에는 대구선과 <a href="/wiki/%EB%8F%99%ED%95%B4%EC%84%A0" title="동해선">동해선</a> 사이에 전차선이 없기 때문에 <a href="/wiki/%ED%8F%AC%ED%95%AD%EC%97%AD" title="포항역">포항역</a>에는 현재 <a href="/wiki/ITX-%EC%83%88%EB%A7%88%EC%9D%84" title="ITX-새마을">ITX-새마을</a> 여객열차를 투입할 수 없으며, 복선전철화 이후에 투입이 가능하다.</p>

<p>(구)대구선((舊)大邱線)은 <a href="/wiki/%EB%8C%80%EA%B5%AC%EA%B4%91%EC%97%AD%EC%8B%9C" title="대구광역시">대구광역시</a>의 <a href="/wiki/%EB%8F%99%EB%8C%80%EA%B5%AC%EC%97%AD" title="동대구역">동대구역</a>과 <a href="/wiki/%EB%B0%98%EC%95%BC%EC%9B%94%EC%97%AD_(%ED%95%9C%EA%B5%AD%EC%B2%A0%EB%8F%84%EA%B3%B5%EC%82%AC)" title="반야월역 (한국철도공사)">반야월역</a>을 잇던 화물 지선 노선으로, 노선 번호는 2024이었다. 대구선의 대구광역시 구간 이설 이후 남겨진 구 선로를 이용하였다.</p>
<h3><span class="mw-headline" id=".EC.97.AD.EC.82.AC_2">역사</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%8C%80%EA%B5%AC%EC%84%A0&amp;action=edit&amp;section=7" title="부분 편집: 역사">원본 편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>대구선의 대구 시내구간은 주택가와 맞닿아 있어서 근처에 거주하는 주민들의 민원이 많은 노선이었다. 이에 대구선의 대구 구간을 외곽으로 이전하려는 계획이 <a href="/wiki/1992%EB%85%84" title="1992년">1992년</a> 수립되었고, 설계를 거쳐 <a href="/wiki/1997%EB%85%84" title="1997년">1997년</a> <a href="/wiki/8%EC%9B%94_27%EC%9D%BC" title="8월 27일">8월 27일</a> 착공에 들어갔다. <a href="/wiki/%EA%B2%BD%EB%B6%80%EC%84%A0" title="경부선">경부선</a> <a href="/wiki/%EA%B0%80%EC%B2%9C%EC%97%AD" title="가천역">가천역</a>에서 분기되는 대구선 신선은 <a href="/wiki/2005%EB%85%84" title="2005년">2005년</a> <a href="/wiki/11%EC%9B%94_1%EC%9D%BC" title="11월 1일">11월 1일</a>에 개통되었으나, 구선의 <a href="/wiki/%EB%8F%99%EC%B4%8C%EC%97%AD_(%ED%95%9C%EA%B5%AD%EC%B2%A0%EB%8F%84%EA%B3%B5%EC%82%AC)" title="동촌역 (한국철도공사)">동촌</a> - 반야월 간에서 분기되는 <a href="/wiki/%EB%8C%80%EA%B5%AC%EA%B5%AD%EC%A0%9C%EA%B3%B5%ED%95%AD" title="대구국제공항">K-2 공군기지</a>의 전용선을 대체할 신선(<a href="/wiki/%EC%B2%AD%EC%B2%9C%EC%97%AD" title="청천역">청천역</a>에서 K-2 공군기지까지)이 완공되지 않아 새 전용선의 공사기간 동안 명맥을 유지할 수 있었다. <a href="/wiki/%EC%B2%AD%EC%B2%9C%EC%97%AD" title="청천역">청천역</a>에서 분기되는 새 전용선이 개통되자 <a href="/wiki/2008%EB%85%84" title="2008년">2008년</a> <a href="/wiki/2%EC%9B%94_15%EC%9D%BC" title="2월 15일">2월 15일</a> 화물열차 운행은 완전히 중단되어, 같은 해 <a href="/wiki/5%EC%9B%94_15%EC%9D%BC" title="5월 15일">5월 15일</a><sup class="reference" id="cite_ref-10"><a href="#cite_note-10">[10]</a></sup> 공식적으로 폐지되었다.</p>
<h3><span class="mw-headline" id=".EC.82.AC.EA.B1.B4">사건</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%8C%80%EA%B5%AC%EC%84%A0&amp;action=edit&amp;section=8" title="부분 편집: 사건">원본 편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>(구)대구선이 존속하던 동안, 대구 <a class="new" href="/w/index.php?title=%EC%95%88%EC%8B%AC%EC%97%B0%EB%A3%8C%EB%8B%A8%EC%A7%80&amp;action=edit&amp;redlink=1" title="안심연료단지 (없는 문서)">안심연료단지</a>내 <a href="/wiki/%EC%97%B0%ED%83%84" title="연탄">연탄</a> 생산 업체들은 계속 (구)대구선을 이용해 연탄 원료인 무연탄을 수송하였다. 이로 인해 공군기지 인입선이 개통한 후에도 한동안 구대구선을 이용하는 화물열차가 운행되었고 주민들이 반발하여, <a href="/wiki/%ED%95%9C%EA%B5%AD%EC%B2%A0%EB%8F%84%EA%B3%B5%EC%82%AC" title="한국철도공사">한국철도공사</a>가 2008년 2월 15일부터 화물열차 운행을 중단하기로 결정하자, 연탄 생산업자들이 항의하는 사건이 일어나기도 하였다.<sup class="reference" id="cite_ref-11"><a href="#cite_note-11">[11]</a></sup><sup class="reference" id="cite_ref-12"><a href="#cite_note-12">[12]</a></sup><sup class="reference" id="cite_ref-13"><a href="#cite_note-13">[13]</a></sup><sup class="reference" id="cite_ref-14"><a href="#cite_note-14">[14]</a></sup></p>
<h3><span class="mw-headline" id=".EB.85.B8.EC.84.A0">노선</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%8C%80%EA%B5%AC%EC%84%A0&amp;action=edit&amp;section=9" title="부분 편집: 노선">원본 편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>아래 표에서, 청천역은 (구)대구선에 포함된 적이 없으나, 대구선 이설 이전에 반야월역과 청천역 사이가 연결되어 있음을 명시하기 위해 편의상 포함했다.</p>
