<p><b>강남구청역</b>(江南區廳驛)은 <a href="/wiki/%EC%84%9C%EC%9A%B8%ED%8A%B9%EB%B3%84%EC%8B%9C" title="서울특별시">서울특별시</a> <a href="/wiki/%EA%B0%95%EB%82%A8%EA%B5%AC" title="강남구">강남구</a> 학동로에 있는 <a href="/wiki/%EC%84%9C%EC%9A%B8_%EC%A7%80%ED%95%98%EC%B2%A0_7%ED%98%B8%EC%84%A0" title="서울 지하철 7호선">서울 지하철 7호선</a>과 <a href="/wiki/%EB%B6%84%EB%8B%B9%EC%84%A0" title="분당선">분당선</a>의 <a href="/wiki/%ED%99%98%EC%8A%B9%EC%97%AD" title="환승역">환승역</a>이다. 그러나 역명과 달리 <a class="mw-redirect" href="/wiki/%EA%B0%95%EB%82%A8%EA%B5%AC%EC%B2%AD" title="강남구청">강남구청</a>은 <a href="/wiki/%EC%B2%AD%EB%8B%B4%EC%97%AD" title="청담역">청담역</a>과 더 가깝다.</p>

<h2><span class="mw-headline" id=".EC.97.AD.EC.82.AC">역사</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EA%B0%95%EB%82%A8%EA%B5%AC%EC%B2%AD%EC%97%AD&amp;action=edit&amp;section=1" title="부분 편집: 역사">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<ul>
<li><a href="/wiki/2000%EB%85%84" title="2000년">2000년</a> <a href="/wiki/8%EC%9B%94_1%EC%9D%BC" title="8월 1일">8월 1일</a> : <a href="/wiki/%EC%84%9C%EC%9A%B8_%EC%A7%80%ED%95%98%EC%B2%A0_7%ED%98%B8%EC%84%A0" title="서울 지하철 7호선">서울 지하철 7호선</a> 개통과 함께 영업 개시</li>
<li>2012년 9월 18일 : <a href="/wiki/%EB%B6%84%EB%8B%B9%EC%84%A0" title="분당선">분당선</a> 철도거리표 고시<sup class="reference" id="cite_ref-1"><a href="#cite_note-1">[1]</a></sup></li>
<li><a href="/wiki/2012%EB%85%84" title="2012년">2012년</a> <a href="/wiki/10%EC%9B%94_6%EC%9D%BC" title="10월 6일">10월 6일</a> : <a href="/wiki/%EB%B6%84%EB%8B%B9%EC%84%A0" title="분당선">분당선</a> <a href="/wiki/%EC%99%95%EC%8B%AD%EB%A6%AC%EC%97%AD" title="왕십리역">왕십리</a> ~ <a href="/wiki/%EC%84%A0%EB%A6%89%EC%97%AD" title="선릉역">선릉</a> 구간 개통으로 <a href="/wiki/%ED%99%98%EC%8A%B9%EC%97%AD" title="환승역">환승역</a>으로 전환</li>
</ul>
<h2><span class="mw-headline" id=".EC.97.AD_.EA.B5.AC.EC.A1.B0">역 구조</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EA%B0%95%EB%82%A8%EA%B5%AC%EC%B2%AD%EC%97%AD&amp;action=edit&amp;section=2" title="부분 편집: 역 구조">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>두 노선이 십자로 교차하고, 2면 2선식의 <a class="mw-redirect" href="/wiki/%EC%83%81%EB%8C%80%EC%8B%9D_%EC%8A%B9%EA%B0%95%EC%9E%A5" title="상대식 승강장">상대식 승강장</a>을 갖추고 있으며, <a href="/wiki/%EC%8A%A4%ED%81%AC%EB%A6%B0%EB%8F%84%EC%96%B4" title="스크린도어">스크린도어</a>가 설치되어 있다. 이 역은 <a href="/wiki/%EB%B6%84%EB%8B%B9%EC%84%A0" title="분당선">분당선</a> 개통을 대비하여 <a href="/wiki/%ED%99%98%EC%8A%B9%EC%97%AD" title="환승역">환승역</a>으로 설계하였으며, 분당선 공사 과정에서 출구가 별도로 신설되지 않고 기존 <a href="/wiki/%EC%84%9C%EC%9A%B8_%EC%A7%80%ED%95%98%EC%B2%A0_7%ED%98%B8%EC%84%A0" title="서울 지하철 7호선">7호선</a> 4개 출구를 공유하나, 개찰구는 따로 분리되어 있다.</p>
