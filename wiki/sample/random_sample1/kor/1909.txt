<p><b>강성산</b>(康成山, <a href="/wiki/1926%EB%85%84" title="1926년">1926년</a> <a href="/wiki/3%EC%9B%94_3%EC%9D%BC" title="3월 3일">3월 3일</a> - <a href="/wiki/2007%EB%85%84" title="2007년">2007년</a> )은 <a href="/wiki/%EC%A1%B0%EC%84%A0%EB%AF%BC%EC%A3%BC%EC%A3%BC%EC%9D%98%EC%9D%B8%EB%AF%BC%EA%B3%B5%ED%99%94%EA%B5%AD" title="조선민주주의인민공화국">조선민주주의인민공화국</a>의 <a href="/wiki/%EC%A0%95%EC%B9%98%EA%B0%80" title="정치가">정치가</a>이자 기업인, <a href="/wiki/%EA%B2%BD%EC%A0%9C%ED%95%99%EC%9E%90" title="경제학자">경제학자</a>이며, <a href="/wiki/%EC%A1%B0%EC%84%A0%EB%A1%9C%EB%8F%99%EB%8B%B9" title="조선로동당">조선로동당</a> 의원이었다. 가명은 강성산(姜成山)이다.</p>
<p><a href="/wiki/%EA%B0%95%EB%8F%88%EC%9A%B1" title="강돈욱">강돈욱</a>, <a href="/wiki/%EA%B0%95%EB%9F%89%EC%9A%B1" title="강량욱">강량욱</a>은 할아버지의 8촌으로 <a href="/wiki/%EA%B9%80%EC%9D%BC%EC%84%B1" title="김일성">김일성</a>과는 외가 쪽으로 10촌에 해당된다.</p>
<h2><span class="mw-headline" id=".EC.83.9D.EC.95.A0">생애</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EA%B0%95%EC%84%B1%EC%82%B0&amp;action=edit&amp;section=1" title="부분 편집: 생애">원본 편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p><a href="/wiki/1926%EB%85%84" title="1926년">1926년</a> <a href="/wiki/3%EC%9B%94_3%EC%9D%BC" title="3월 3일">3월 3일</a>생이다. 칠골의 <a href="/wiki/%EA%B8%B0%EB%8F%85%EA%B5%90" title="기독교">기독교</a> 목회자 집안인 <a href="/wiki/%EA%B0%95%EB%8F%88%EC%9A%B1" title="강돈욱">강돈욱</a> 목사, <a href="/wiki/%EA%B0%95%EB%9F%89%EC%9A%B1" title="강량욱">강량욱</a> 목사는 할아버지뻘 되는 친척으로, 강성산의 조부의 6촌에 해당된다. <a href="/wiki/%EA%B0%95%EB%8F%88%EC%9A%B1" title="강돈욱">강돈욱</a> 목사의 아들인 <a class="new" href="/w/index.php?title=%EA%B0%95%EC%A7%84%EC%84%9D&amp;action=edit&amp;redlink=1" title="강진석 (없는 문서)">강진석</a> 목사는 그에게 9촌 아저씨뻘이 된다.</p>
<p><a href="/wiki/1947%EB%85%84" title="1947년">1947년</a> 설립된 만경대혁명가유자녀학원을 졸업한 뒤 <a href="/wiki/%EA%B9%80%EC%9D%BC%EC%84%B1%EC%A2%85%ED%95%A9%EB%8C%80%ED%95%99" title="김일성종합대학">김일성종합대학</a>에 입학하였고, 재학 중 <a href="/wiki/%EC%86%8C%EB%A0%A8" title="소련">소련</a>으로 유학을 가서 <a class="mw-redirect" href="/wiki/%EB%AA%A8%EC%8A%A4%ED%81%AC%EB%B0%94_%EB%8C%80%ED%95%99" title="모스크바 대학">모스크바 대학</a>을 졸업하였다. <a href="/wiki/1955%EB%85%84" title="1955년">1955년</a> <a class="mw-redirect" href="/wiki/%EC%A1%B0%EC%84%A0%EB%85%B8%EB%8F%99%EB%8B%B9" title="조선노동당">조선노동당</a> 중앙위원회 조직지도부 지도원이 되면서 당료로서 첫 출발을 하였다. <a href="/wiki/1969%EB%85%84" title="1969년">1969년</a> <a href="/wiki/%EC%A1%B0%EC%84%A0%EB%A1%9C%EB%8F%99%EB%8B%B9" title="조선로동당">조선로동당</a> <a href="/wiki/%EC%9E%90%EA%B0%95%EB%8F%84" title="자강도">자강도</a>당 위원장, <a class="mw-redirect" href="/wiki/%ED%8F%89%EC%96%91%EC%8B%9C_%EC%9D%B8%EB%AF%BC%EC%9C%84%EC%9B%90%ED%9A%8C" title="평양시 인민위원회">평양시 인민위원회</a> 위원장을 지냈다.</p>
<p>1973년 정치위원회 후보위원, 1976년 정무원 부총리 겸 철도부장, 1977년 정무원 부총리, 1980년에는 제6차 당대회에서 당 정치국 정위원으로 선출되었다. 1982년 제1부총리, 1984년 1월 27일에는 중앙인민위원회 위원이 되고 1월 28일에는 이종옥의 뒤를 이어 정무원 총리로 취임하였으며, 총리 취임 후 합영법을 제정, 공포하였다.</p>
<p>1991년 함북도당 책임비서 겸 인민위원장, 당중앙인민위원회 위원, 조선노동당정치국 위원, 최고인민회의 대의원 등의 직위를 맡고 있다. 1982년 김일성훈장, 노력훈장, 국기훈장 1급 등을 받았다.</p>
<p><a href="/wiki/%EA%B9%80%EC%9D%BC%EC%84%B1" title="김일성">김일성</a>, <a href="/wiki/%EC%98%A4%EC%A7%84%EC%9A%B0" title="오진우">오진우</a>가 사망한 1994년과 1995년에 국가장의위원을 맡았으나, <a href="/wiki/%EA%B9%80%EC%A0%95%EC%9D%BC" title="김정일">김정일</a>시대가 되면서, <a href="/wiki/1996%EB%85%84" title="1996년">1996년</a>에 실각되었고, 당의원의 명함도 유명무실 하게 되었다.</p>
