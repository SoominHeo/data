<p><b>강유</b>(姜維, <a href="/wiki/202%EB%85%84" title="202년">202년</a> ~ <a href="/wiki/264%EB%85%84" title="264년">264년</a>)는 <a href="/wiki/%EC%A4%91%EA%B5%AD" title="중국">중국</a> <a href="/wiki/%EC%82%BC%EA%B5%AD_%EC%8B%9C%EB%8C%80_(%EC%A4%91%EA%B5%AD)" title="삼국 시대 (중국)">삼국 시대</a> <a href="/wiki/%EC%B4%89%ED%95%9C" title="촉한">촉한</a>의 무장으로 <a href="/wiki/%EC%95%84%EB%B0%94_%ED%8B%B0%EB%B2%A0%ED%8A%B8%EC%A1%B1_%EC%B0%BD%EC%A1%B1_%EC%9E%90%EC%B9%98%EC%A3%BC" title="아바 티베트족 창족 자치주">천수군</a>(天水郡) <a href="/wiki/%EC%9B%90%EC%B4%A8_%ED%98%84" title="원촨 현">기현</a>(冀縣)<sup class="reference" id="cite_ref-1"><a href="#cite_note-1">[1]</a></sup> 사람이다. 이민족인 <a href="/wiki/%EC%B0%BD%EC%A1%B1" title="창족">강족</a>들을 격퇴하는 등 위나라 소속으로 공을 세운 그는 <a class="mw-redirect" href="/wiki/%EC%A0%9C%EA%B0%88%EB%9F%89%EC%9D%98_%EC%A0%9C1%EC%B0%A8_%EB%B6%81%EB%B2%8C" title="제갈량의 제1차 북벌">제갈량의 제1차 북벌</a> 때 촉나라에 투항했다. 재능을 인정받아 촉나라에서 승승장구하던 그는 제갈량의 후계자로서 여러 차례 촉의 위기를 구했다. 촉나라 멸망 후에도 재건을 위해 노력을 다했으며 위나라 점령군 <a href="/wiki/%EC%A2%85%ED%9A%8C" title="종회">종회</a>와 손잡고 난을 일으켰으나 실패해 구족이 멸망당했다. <a href="/wiki/%EC%82%BC%EA%B5%AD%EC%A7%80%EC%97%B0%EC%9D%98" title="삼국지연의">삼국지연의</a> 등에 그의 활약이 다수 남아있으며 그의 충성은 후세에 숭앙됐다. 자(字)는 <b>백약</b>(伯約).</p>

<h2><span class="mw-headline" id=".EC.83.9D.EC.95.A0">생애</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EA%B0%95%EC%9C%A0&amp;action=edit&amp;section=1" title="부분 편집: 생애">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>천수 기산에서 태어난 강유는 <a href="/wiki/%EC%B0%BD%EC%A1%B1" title="창족">이민족(강족)</a>에게 아버지 강경(-冏,빛날 경)을 잃었지만 문무를 갈고 닦으며 <a href="/wiki/%EC%9C%84_(%EC%82%BC%EA%B5%AD)" title="위 (삼국)">위나라</a>의 중랑장과 군사를 역임했다. 천수태수 <a href="/wiki/%EB%A7%88%EC%A4%80" title="마준">마준</a>(馬遵) 밑에 있었던 그는 <a href="/wiki/%EB%A7%88%EC%A4%80" title="마준">마준</a>이 <a href="/wiki/%EC%A0%9C%EA%B0%88%EB%9F%89" title="제갈량">제갈량</a>의 침입에 겁을 먹고 천수성과 백성들을 버리고 도망가자 제갈량에게 항복했다. 강유는 <a href="/wiki/%EA%B0%84%EC%91%A4_%EC%84%B1" title="간쑤 성">서량</a> 전체를 촉에 귀순시키려 했으나 실패 후 <a class="mw-redirect" href="/wiki/%EC%B4%89%EB%82%98%EB%9D%BC" title="촉나라">촉나라</a>로 망명했다. 제갈량은 곧 그를 상장군에 봉했다.</p>
<p>제갈량이 세상을 떠난 후 강유는 대장군의 직위에 올랐다. 촉의 최고 권력자인 상서령 <a href="/wiki/%EC%9E%A5%EC%99%84" title="장완">장완</a>(蔣琬)의 지원으로 거듭 북벌을 일으켜 <a href="/wiki/%EC%9C%84_(%EC%82%BC%EA%B5%AD)" title="위 (삼국)">위</a>를 치려했으나, 오래지 않아 장완이 사망했다. <a href="/wiki/%EB%B9%84%EC%9D%98" title="비의">비의</a>(費禕)가 장완의 자리를 승계했으나 그는 전임자와 달리 북벌에 소극적이었고 내정에 치중했다. 어쩔 수 없이 대장군 강유는 여러 차례 북벌을 1만 이내의 군사로 감행해 성공하기 어려웠다.</p>
<p>그나마 비의조차 세상을 떠나고 후주 <a class="mw-redirect" href="/wiki/%EC%9C%A0%EC%84%A0_(%EA%B3%B5%EC%82%AC)" title="유선 (공사)">유선</a>(劉禪)은 강유에게 <a href="/wiki/%EC%9C%84_(%EC%82%BC%EA%B5%AD)" title="위 (삼국)">위</a>의 남하를 견제하게 했다. 강유는 공격적인 전술을 택해 아홉번 북벌을 하나 번번히 위의 장수 <a href="/wiki/%EB%93%B1%EC%95%A0" title="등애">등애</a>,<a href="/wiki/%EA%B3%BD%ED%9A%8C" title="곽회">곽회</a>,<a href="/wiki/%EC%A2%85%ED%9A%8C" title="종회">종회</a>,<a href="/wiki/%EC%A7%84%ED%83%9C" title="진태">진태</a> 등에게 막혔고, 설상가상으로 환관 <a href="/wiki/%ED%99%A9%ED%98%B8" title="황호">황호</a>(黃皓)의 부추김으로 내려진 퇴각령으로 어쩔 수 없이 군을 물려야 했다.<sup class="reference" id="cite_ref-2"><a href="#cite_note-2">[2]</a></sup></p>
<p><a href="/wiki/263%EB%85%84" title="263년">263년</a> <a href="/wiki/%EB%93%B1%EC%95%A0" title="등애">등애</a>(鄧艾)가 험준한 음평산을 넘어 <a href="/wiki/%EC%B2%AD%EB%91%90_%EC%8B%9C" title="청두 시">성도</a>를 점령하던 당시 강유는 <a class="mw-redirect" href="/wiki/%EA%B2%80%EA%B0%81" title="검각">검각</a>에서 위의 <a href="/wiki/%EC%A2%85%ED%9A%8C" title="종회">종회</a>(鐘會)를 맞아 항전했으나 때마침 유선의 항복문을 받고 항복했다. 항복 후 유선이 위나라에 끌려가는 와중에도 강유는 촉에 남아 종회 휘하로 들어갔다. 거기서 촉 재건의 기회를 엿보던 그는 <a href="/wiki/%EC%A2%85%ED%9A%8C" title="종회">종회</a>를 부추켜 반란 계획을 세웠다. 등애가 촉을 멸망시킨 여세를 몰아 <a class="mw-disambig" href="/wiki/%EC%98%A4%EB%82%98%EB%9D%BC" title="오나라">오나라</a> 정벌을 조정에 상주하고 들떠 있을 때 종회는 당시 위나라의 실권자 <a href="/wiki/%EC%82%AC%EB%A7%88%EC%86%8C" title="사마소">사마소</a>에게 등애를 참소하고 그를 체포했다.</p>
<p>체포된 <a href="/wiki/%EB%93%B1%EC%95%A0" title="등애">등애</a>는 아들과 함께 <a class="mw-redirect" href="/wiki/%EB%82%99%EC%96%91" title="낙양">낙양</a>으로 압송되어, 마침내 최대 경쟁자를 제거한 <a href="/wiki/%EC%A2%85%ED%9A%8C" title="종회">종회</a>가 촉을 장악했다. 사마소는 촉 지방의 정세가 심상찮음을 감지하고 병사 10만을 <a href="/wiki/%EC%9E%A5%EC%95%88" title="장안">장안</a>으로 보내 대비하게 했다.</p>
<p><a href="/wiki/264%EB%85%84" title="264년">264년</a> <a href="/wiki/1%EC%9B%94" title="1월">1월</a> <a href="/wiki/%EC%A2%85%ED%9A%8C" title="종회">종회</a>는 사마소의 움직임에 압박을 느끼고 강유와 반란을 계획보다 앞당겼으나, <a class="mw-disambig" href="/wiki/%ED%98%B8%EC%97%B0" title="호연">호연</a> 등 휘하 대부분의 위나라 장수들이 난에 가담치 않고 거꾸로 공격해 와 실패했다. <a href="/wiki/%EC%B2%AD%EB%91%90_%EC%8B%9C" title="청두 시">성도</a>를 지키던 강유는 "적은 격퇴할 뿐"이라 말하고 적진에 뛰어들어 전사했다. 향년 65세.</p>
<p>곧 <a href="/wiki/%EC%A2%85%ED%9A%8C" title="종회">종회</a>도 살해되어 난은 진압됐고 촉나라 재건의 꿈도 사라졌다. 압송 중 반역 혐의를 벗고 풀려난 <a href="/wiki/%EB%93%B1%EC%95%A0" title="등애">등애</a>는 성도로 돌아오려 했지만, <a href="/wiki/%EC%9C%84%EA%B4%80_(%EC%84%9C%EC%A7%84)" title="위관 (서진)">위관</a>(衛瓘)의 음모로 <a class="mw-redirect" href="/wiki/%EB%A9%B4%EC%A3%BD" title="면죽">면죽</a>에서 장남과 함께 살해된다. 위관은 등애 체포 작전의 책임자로서 후환을 두려워했다.</p>
<h2><span class="mw-headline" id=".ED.8F.89.EA.B0.80">평가</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EA%B0%95%EC%9C%A0&amp;action=edit&amp;section=2" title="부분 편집: 평가">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<ul>
<li><a href="/wiki/%EC%82%BC%EA%B5%AD%EC%A7%80" title="삼국지">삼국지</a>의 저자 <a href="/wiki/%EC%A7%84%EC%88%98_(%EC%84%9C%EC%A7%84)" title="진수 (서진)">진수</a>(陳壽)는 본저에서 '강백약은 문무를 갖췄고 공명을 세우려는 뜻이 있었으나, 인명을 경시해 병력을 함부로 소모했다'고 서술했다. 또 '어떤 문제든 사리분별이 명확했고 과단성이 있었으나, 본성이 급하고 생각이 조밀치 않아 죽음에 이르렀다'고 덧붙였다. 나악 등은 <a href="/wiki/%EB%85%B8%EC%9E%90" title="노자">노자</a>의 <a href="/wiki/%EB%8F%84%EB%8D%95%EA%B2%BD" title="도덕경">도덕경</a>을 인용해, "큰 국가를 다스리는 것은 작은 물고기를 굽는 것과 같다. 하물며 작은 나라에 대해서야 여러 차례 소란스럽게 할 수 있겠는가?"라며 여러 차례 북벌에 실패한 강유를 비판했다.</li>
<li><a href="/wiki/%EA%B7%B9%EC%A0%95" title="극정">극정</a>(郤正)은 강유에 대해 강백약이 상장군으로 뭇 신하들의 윗자리에 있었음에도 청렴했으며 집이 초라했다고 했다. 더우기 강유를 칭찬하길 '그와 같이 부단히 배우고 소박하며 청렴한 인물은 한 시대의 모범이다'라고 했다.</li>
</ul>
<h2><span class="mw-headline" id=".E3.80.8A.EC.82.BC.EA.B5.AD.EC.A7.80.EC.97.B0.EC.9D.98.E3.80.8B.EC.97.90_.EB.82.98.ED.83.80.EB.82.9C_.EA.B0.95.EC.9C.A0_.EC.97.90.ED.94.BC.EC.86.8C.EB.93.9C">《삼국지연의》에 나타난 강유 에피소드</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EA%B0%95%EC%9C%A0&amp;action=edit&amp;section=3" title="부분 편집: 《삼국지연의》에 나타난 강유 에피소드">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<ul>
<li><a href="/wiki/%EC%A0%9C%EA%B0%88%EB%9F%89%EC%9D%98_%EB%B6%81%EB%B2%8C" title="제갈량의 북벌">제갈량의 1차 북벌</a>에서 천수에 머물던 강유는 <a href="/wiki/%EC%A0%9C%EA%B0%88%EB%9F%89" title="제갈량">제갈량</a>의 계략을 간파해 제갈량을 패퇴시켰다고 묘사된다. 싸움을 피하려던 제갈량이 강유의 효심을 이용한 계략으로 그를 사로잡아 굴복시켰다는 설정이었다. 설득에는 강유의 친모를 이용했다고 한다.</li>
<li><a href="/wiki/234%EB%85%84" title="234년">234년</a> 자신의 수명이 얼마남지 않은 것을 직감한 <a href="/wiki/%EC%A0%9C%EA%B0%88%EB%9F%89" title="제갈량">제갈량</a>이 하늘에 제단을 열고 기도를 하라는 강유의 조언을 받아들여 군막내 7일간 촛불을 켜고 출입을 엄금했다. 하지만 <a href="/wiki/%EC%9C%84%EC%97%B0" title="위연">위연</a>(魏延)이 이를 어겨 제갈량이 뜻을 이루지 못하고 죽기 직전 강유에게 뒤를 맡기고 더불어 그의 병법을 담은 책을 전수했다고 묘사된다.</li>
</ul>
<h2><span class="mw-headline" id=".EA.B7.B8.EC.99.80_.EA.B4.80.EB.A0.A8.EB.90.9C_.EC.9C.A0.EC.A0.81.EB.93.A4">그와 관련된 유적들</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EA%B0%95%EC%9C%A0&amp;action=edit&amp;section=4" title="부분 편집: 그와 관련된 유적들">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<ul>
<li>강유가 <a class="mw-redirect" href="/wiki/%EA%B2%80%EA%B0%81" title="검각">검각</a>으로 와서 진영을 정비하고 요새를 굳게 지키자, <a href="/wiki/%EC%A2%85%ED%9A%8C" title="종회">종회</a>의 10만 대군이 조금도 움직일 수가 없었다. 그런데 뜻밖에도 <a class="mw-redirect" href="/wiki/%EC%9C%A0%EC%84%A0_(%EA%B3%B5%EC%82%AC)" title="유선 (공사)">유선</a>이 그에게 장수와 군사들을 거느리고 종회에게 항복하라는 조서를 내렸다. 조서를 받든 강유는 원망스럽고 화가 나서 분통을 참지못해 칼을 뽑아 옆의 돌을 베었다. 그때 그가 자른 돌은 지금도 검각 협곡 안에 위치해 있으며, 사람들은 이 돌을 「<b>감도석</b>」이라 부른다.</li>
<li>억울하고 울분을 참을 수 없었지만, 강유는 유선의 성지를 거역할 수 없었기에 양면으로 된 군기를 거꾸로 걸게끔 명령을 내려 위국에 투항할 뜻을 표했다. 그 양면으로 된 기를 꽂았던 돌은 주가채의 바위 절벽에 남아있는데, 이를 「<b>쌍기암</b>」이라 부른다.</li>
<li>그는 장수와 병사들을 소집하여 각기 지니고 있던 병기를 모아 소검산의 한 동굴에 숨긴 다음, 차후에 때를 보아 군사를 일으켜 한나라 왕실을 부흥하고자 했다. 이 동굴을 「<b>강유 도창고</b>」라고 불린다.</li>
<li>강유는 검문관을 떠날 때 마고자를 벗고 손수 빨은 뒤, 이를 말리기 위해 산의 바위 위에 널며 반드시 돌아올테니 그때 다시 마고자를 입을 것이라고 맹세했지만, 결국 한 해가 지나고 또 다시 다른 한 해가 지나도 돌아오지 않자 옷은 돌로 변하였다는 전설이 전해지고 있다, 지금도 검문관 20리 밖에서 큰 바위 위에 널려 있는 마고자의 모양을 볼 수가 있다. 이를 가리켜 「<b>양의암</b>」이라고 한다.</li>
</ul>
