<p><b>울트라ISO</b>(UltraISO)는 <a class="new" href="/w/index.php?title=%EA%B4%91%ED%95%99_%EB%94%94%EC%8A%A4%ED%81%AC_%EC%A0%80%EC%9E%91&amp;action=edit&amp;redlink=1" title="광학 디스크 저작 (없는 문서)">CD 및 DVD 저작</a>에 쓰이는 <a class="mw-redirect" href="/wiki/ISO" title="ISO">ISO</a> 파일을 만들고, 변환하고, 수정하는 데 쓰이는 <a class="mw-redirect" href="/wiki/%EB%A7%88%EC%9D%B4%ED%81%AC%EB%A1%9C%EC%86%8C%ED%94%84%ED%8A%B8_%EC%9C%88%EB%8F%84" title="마이크로소프트 윈도">마이크로소프트 윈도</a>용 응용 프로그램이다.</p>
<p>2002년 4월 20일에 처음 출시된 울트라ISO는 프로그램을 만든 EZB 시스템즈가 제품을 지속적으로 관리하고 있다. 이 프로그램은 셰어웨어이다.</p>

<h2><span class="mw-headline" id=".EA.B8.B0.EB.8A.A5">기능</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%9A%B8%ED%8A%B8%EB%9D%BCISO&amp;action=edit&amp;section=1" title="부분 편집: 기능">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<ul>
<li>CD, DVD를 ISO 이미지로 복제한다.</li>
<li><a href="/wiki/%EB%94%94%EC%8A%A4%ED%81%AC_%EB%93%9C%EB%9D%BC%EC%9D%B4%EB%B8%8C" title="디스크 드라이브">CD/DVD-ROM</a>이나 <a class="mw-redirect" href="/wiki/%ED%95%98%EB%93%9C_%EB%94%94%EC%8A%A4%ED%81%AC" title="하드 디스크">하드 디스크</a>에 위치한 파일을 ISO 이미지에 추가하여 이미지를 제작한다.</li>
<li>파일이나 폴더를 추가하고 삭제하고 만드는 등 ISO 파일을 수정한다.</li>
<li><a class="mw-redirect" href="/wiki/%EC%8B%9C%EB%8F%99" title="시동">시동 가능한</a> CD/DVD, <a class="mw-redirect" href="/wiki/%ED%94%8C%EB%A1%9C%ED%94%BC_%EB%93%9C%EB%9D%BC%EC%9D%B4%EB%B8%8C" title="플로피 드라이브">플로피</a> 이미지를 만든다.</li>
<li>.BIN, .IMG, .CIF, .NRG, .BWI 등의 다른 형식의 포맷을 ISO 표준 이미지로 변환한다.</li>
<li><a href="/wiki/ISO_9660" title="ISO 9660">ISO 9660</a> 수준과 Joliet 확장을 지원한다.</li>
<li>ISO 파일 구조를 최적화하여 디스크 공간을 아낀다.</li>
<li>ISO를 <a href="/wiki/%EA%B0%80%EC%83%81_%EB%93%9C%EB%9D%BC%EC%9D%B4%EB%B8%8C" title="가상 드라이브">가상 드라이브</a>로 <a href="/wiki/%EB%A7%88%EC%9A%B4%ED%8A%B8" title="마운트">마운트</a>할 수 있다.</li>
<li>ISO 데이터를 확인하는 <a href="/wiki/%EC%B2%B4%ED%81%AC%EC%84%AC" title="체크섬">체크섬</a>을 만든다.</li>
<li>여러 언어를 지원한다.</li>
<li>MP3 플레이어에서 만들 수 있다.</li>
<li>무료 버전에서는 300 메가바이트 이하의 ISO 이미지만 만들 수 있다.</li>
<li>UDF 디스크 형식을 지원한다.</li>
</ul>
<h2><span class="mw-headline" id="ISZ_.ED.8F.AC.EB.A7.B7">ISZ 포맷</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%9A%B8%ED%8A%B8%EB%9D%BCISO&amp;action=edit&amp;section=2" title="부분 편집: ISZ 포맷">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>울트라ISO는 ISZ라는 사유 포맷을 사용한다. ISO Zipped로 광고되는 이 포맷은 단순한 zip 아카이브는 아니다. 이 포맷은 <a href="/wiki/Zlib" title="Zlib">zlib</a>이나 <a href="/wiki/Bzip2" title="Bzip2">bzip2</a>를 사용하여 데이터를 압축하며 다양한 정도의 <a class="mw-redirect" href="/wiki/%EA%B3%A0%EA%B8%89_%EC%95%94%ED%98%B8_%ED%91%9C%EC%A4%80" title="고급 암호 표준">AES</a> 암호화를 지원할 수 있다. 파일 포맷 사양은 EZB 시스템의 웹사이트에서 공개적으로 이용이 가능하다.<sup class="reference" id="cite_ref-1"><a href="#cite_note-1">[1]</a></sup> 이 포맷은 현재 <a class="mw-redirect" href="/wiki/%EB%8D%B0%EB%AA%AC_%ED%88%B4%EC%8A%A4" title="데몬 툴스">데몬 툴스</a>, <a class="mw-redirect" href="/wiki/%EC%95%8C%EC%BD%9C_120%25" title="알콜 120%">알콜 120%</a>, <a class="new" href="/w/index.php?title=CDemu&amp;action=edit&amp;redlink=1" title="CDemu (없는 문서)">CDemu</a>와 같은 타사 응용 프로그램들에 의해 지원된다.</p>
