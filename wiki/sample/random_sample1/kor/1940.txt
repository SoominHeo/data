<h1 class="firstHeading" id="firstHeading" lang="ko">거국중립내각</h1>
<div class="mw-body-content" id="bodyContent">
<div id="siteSub">위키백과, 우리 모두의 백과사전.</div>
<div id="contentSub"></div>
<div class="mw-jump" id="jump-to-nav">
					이동:					<a href="#mw-head">둘러보기</a>, 					<a href="#p-search">검색</a>
</div>
<div class="mw-content-ltr" dir="ltr" id="mw-content-text" lang="ko"><script>function mfTempOpenSection(id){var block=document.getElementById("mf-section-"+id);block.className+=" open-block";block.previousSibling.className+=" open-block";}</script><p><b>거국중립내각</b>(擧國中立內閣, <span style="font-size: smaller;"><a href="/wiki/%EC%98%81%EC%96%B4" title="영어">영어</a>:</span> <span lang="en" xml:lang="en">national unity government, government of national unity, national union government</span>)은 특정한 정당이나 정파에 한정되지 않은 중립적인 <a href="/wiki/%EC%A0%95%EB%B6%80" title="정부">정부</a> <a href="/wiki/%EB%82%B4%EA%B0%81" title="내각">내각</a>을 의미한다. <a href="/wiki/%EB%8B%A4%EB%8B%B9%EC%A0%9C" title="다당제">다당제</a>인 정치 체제에서는 <a href="/wiki/%EC%97%AC%EB%8B%B9" title="여당">여당</a>과 <a href="/wiki/%EC%95%BC%EB%8B%B9" title="야당">야당</a>이 각각 추천한 인물들을 중심으로 꾸려진다.<sup class="reference" id="cite_ref-1"><a href="#cite_note-1">[1]</a></sup><sup class="reference" id="cite_ref-2"><a href="#cite_note-2">[2]</a></sup> <b>거국내각</b>, <b>중립내각</b>이라고도 한다.</p>
<p><a href="/wiki/%EB%8C%80%ED%86%B5%EB%A0%B9%EC%A0%9C" title="대통령제">대통령제</a>를 채택한 국가에서는 <a href="/wiki/%EB%8C%80%ED%86%B5%EB%A0%B9" title="대통령">대통령</a>이 임기 중에 심각한 <a href="/wiki/%EB%A0%88%EC%9E%84%EB%8D%95" title="레임덕">레임덕</a>에 빠져 원활한 국정 수행이 불가능한 경우가 발생할 수 있다. 이때 <a class="mw-disambig" href="/wiki/%EA%B5%AD%ED%9A%8C" title="국회">국회</a>의 여야가 합의를 통해 <a class="mw-redirect" href="/wiki/%EA%B5%AD%EB%AC%B4%EC%B4%9D%EB%A6%AC" title="국무총리">국무총리</a>와 <a href="/wiki/%EA%B5%AD%EB%AC%B4%EC%9C%84%EC%9B%90" title="국무위원">국무위원</a>, 각 부처의 <a href="/wiki/%EC%9E%A5%EA%B4%80" title="장관">장관</a> 등 기존의 내각 일원을 전면 교체하고 양당이 모두 내각에 참여하는 방향으로 개각하거나,<sup class="reference" id="cite_ref-3"><a href="#cite_note-3">[3]</a></sup> 양당이 수용 가능한 중립적인 인물을 새로 <a class="mw-redirect" href="/wiki/%EA%B5%AD%EB%AC%B4%EC%B4%9D%EB%A6%AC" title="국무총리">국무총리</a>에 임명하고 그 총리가 대통령을 대신해 국정을 주도하게 하는 방안이 있다. 이를 통해 대통령의 <a href="/wiki/%ED%83%84%ED%95%B5" title="탄핵">탄핵</a> 혹은 <a href="/wiki/%EB%8C%80%ED%86%B5%EB%A0%B9%EC%A0%9C" title="대통령제">하야</a>, 조기 선거 등의 더욱 극심한 정국 혼란으로 이어질 위험을 막고, 국정에 대해 야당의 요구가 반영되어 정치적으로 중립성과 안정을 꾀할 수 있다는 장점이 있다. 그러나 거국내각 정국이 장기화될 경우, 정책의 방향을 놓고 여야 간에 대립이 벌어져 오히려 더욱 불안정한 상태에 이를 위험이 있으므로, 대부분 차기 선거와 맞물려 꾸리거나 얼마 남지 않은 시점에 구성되는 경우가 많다.</p>
<p><a class="mw-redirect" href="/wiki/%EC%9D%98%EC%9B%90%EB%82%B4%EA%B0%81%EC%A0%9C" title="의원내각제">의원내각제</a>를 정치 체제로 채택한 국가의 경우, <a href="/wiki/%EC%9D%98%ED%9A%8C" title="의회">의회</a> 총선거를 통해 원내에서 가장 많은 의석을 획득한 정당이 정부 내각을 구성할 기회를 얻는다. 선거에서 과반 의석을 차지한 정당이 <a href="/wiki/%ED%97%9D_%EC%9D%98%ED%9A%8C" title="헝 의회">없는 경우</a>에는 의석 비중이 가장 큰 두세 정당이 공동으로 내각을 구성하는 <a class="mw-redirect" href="/wiki/%EC%97%B0%EB%A6%BD_%EB%82%B4%EA%B0%81" title="연립 내각">연립 내각</a>을 이루며, 집권 내각이 심각한 국정 혼란으로 제기능을 하지 못하면 야당 의원들이 <a class="new" href="/w/index.php?title=%EB%82%B4%EA%B0%81%EB%B6%88%EC%8B%A0%EC%9E%84&amp;action=edit&amp;redlink=1" title="내각불신임 (없는 문서)">내각불신임</a> 투표를 부쳐 새로운 총선을 치루도록 하기 때문에, 대통령제와 같은 형태의 거국중립내각은 거의 이뤄지지 않는다. 그러나 <a href="/wiki/%EC%A0%84%EC%9F%81" title="전쟁">전쟁</a>과 같은 비상 시국일 경우, 국가의 체제 존립을 위하여 의회의 모든 정당이 내각에 참여하는 경우가 있다. 이때 세워지는 정부 내각을 <b>거국일치내각</b>(擧國一致內閣)이라고 부른다. <a class="mw-redirect" href="/wiki/%EC%A0%9C_2%EC%B0%A8_%EC%84%B8%EA%B3%84_%EB%8C%80%EC%A0%84" title="제 2차 세계 대전">제 2차 세계 대전</a> 당시 영국에서 집권 정당인 <a href="/wiki/%EB%B3%B4%EC%88%98%EB%8B%B9_(%EC%98%81%EA%B5%AD)" title="보수당 (영국)">보수당</a>을 중심으로 <a href="/wiki/%EB%85%B8%EB%8F%99%EB%8B%B9_(%EC%98%81%EA%B5%AD)" title="노동당 (영국)">노동당</a>, <a href="/wiki/%EC%9E%90%EC%9C%A0%EB%8B%B9_(%EC%98%81%EA%B5%AD)" title="자유당 (영국)">자유당</a>, 국가자유당 등이 모두 참여한 전시 처칠 내각이 대표적인 예이다.</p>
<h2><span class="mw-headline" id=".EC.82.AC.EB.A1.80">사례</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EA%B1%B0%EA%B5%AD%EC%A4%91%EB%A6%BD%EB%82%B4%EA%B0%81&amp;action=edit&amp;section=1" title="부분 편집: 사례">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<ul>
<li><a href="/wiki/2011%EB%85%84" title="2011년">2011년</a> <a class="mw-redirect" href="/wiki/%EA%B7%B8%EB%A6%AC%EC%8A%A4_%EA%B2%BD%EC%A0%9C_%EC%9C%84%EA%B8%B0" title="그리스 경제 위기">그리스 경제 위기</a>가 진행되던 당시 그리스의 <a href="/wiki/%EC%9A%94%EB%A5%B4%EC%9A%94%EC%8A%A4_%ED%8C%8C%ED%8C%90%EB%93%9C%EB%A0%88%EC%9A%B0_(1952%EB%85%84)" title="요르요스 파판드레우 (1952년)">요르요스 파판드레우</a> 총리가 2차 구제 금융 방안에 대해 국민투표를 실시하자고 요청했다가 총리 불신임 투표 요구 등으로 정국이 크게 흔들리자, 총리 스스로 국민투표 카드를 거둬들이고, 자신의 사퇴와 동시에 2차 구제 금융안 비준을 위한 임시 거국내각 구성을 야당인 <a href="/wiki/%EC%8B%A0%EB%AF%BC%EC%A3%BC%EC%A3%BC%EC%9D%98%EB%8B%B9" title="신민주주의당">신민주주의당</a>과 합의한 바 있다.<sup class="reference" id="cite_ref-4"><a href="#cite_note-4">[4]</a></sup><sup class="reference" id="cite_ref-5"><a href="#cite_note-5">[5]</a></sup></li>
<li><a href="/wiki/%EB%8C%80%ED%95%9C%EB%AF%BC%EA%B5%AD" title="대한민국">대한민국</a>에서는 거국중립내각이 수립된 사례는 없었다. 다만 여야의 합의로 내각 구성이 새로 이뤄진 경우는 존재한다. <a href="/wiki/1992%EB%85%84" title="1992년">1992년</a> <a href="/wiki/9%EC%9B%94_18%EC%9D%BC" title="9월 18일">9월 18일</a> <a href="/wiki/%EB%85%B8%ED%83%9C%EC%9A%B0" title="노태우">노태우</a> 대통령이 <a href="/wiki/%EB%8C%80%ED%95%9C%EB%AF%BC%EA%B5%AD_%EC%A0%9C14%EB%8C%80_%EB%8C%80%ED%86%B5%EB%A0%B9_%EC%84%A0%EA%B1%B0" title="대한민국 제14대 대통령 선거">대한민국 제14대 대통령 선거</a>를 앞두고 여당인 <a href="/wiki/%EB%AF%BC%EC%A3%BC%EC%9E%90%EC%9C%A0%EB%8B%B9" title="민주자유당">민주자유당</a>에서 총재직을 내려놓음과 동시에 <a class="mw-redirect" href="/wiki/%ED%83%88%EB%8B%B9" title="탈당">탈당</a>을 하며, 거국적 중립 내각 수립을 선포했다. 여야의 합의에 따라 <a href="/wiki/%ED%98%84%EC%8A%B9%EC%A2%85" title="현승종">현승종</a> 당시 연세대 총장을 신임 국무총리로 추천하였으며 10월 9일에 중립 내각이 출범했다.<sup class="reference" id="cite_ref-6"><a href="#cite_note-6">[6]</a></sup><sup class="reference" id="cite_ref-.EB.85.B8.ED.83.9C.EC.9A.B0_7-0"><a href="#cite_note-.EB.85.B8.ED.83.9C.EC.9A.B0-7">[7]</a></sup> 그러나 이 경우는 대선을 불과 2개월 앞둔 시점이라 일종의 "정권 마무리 내각" 차원이었기에 여당으로서도 중립 내각 출범이 큰 무리가 아니었고, 야당이 추천한 인물을 내각에 참여시키기보다 여야 양쪽이 모두 수용 가능한 정치색이 낮은 인물들로 꾸려진 경우였기에, 여야가 모두 참여하는 형태의 거국 내각에는 해당하지 않는 것으로 평가된다.<sup class="reference" id="cite_ref-.EB.85.B8.ED.83.9C.EC.9A.B0_7-1"><a href="#cite_note-.EB.85.B8.ED.83.9C.EC.9A.B0-7">[7]</a></sup></li>
<li><a href="/wiki/%EB%8C%80%ED%86%B5%EB%A0%B9%EC%A0%9C" title="대통령제">대통령제</a>를 채택하고 있는 <a href="/wiki/%EB%AF%B8%EA%B5%AD" title="미국">미국</a> 역시 여론에 의해 거국중립내각이 수립된 적은 없었다. 전쟁 등으로 인한 거국일치내각은 한 번 수립된 적이 있었는데, <a class="mw-disambig" href="/wiki/%EB%82%A8%EB%B6%81_%EC%A0%84%EC%9F%81" title="남북 전쟁">남북 전쟁</a> 기간에 파벌 정치를 메꿀 수 있다는 기대감 속에 공화당의 <a href="/wiki/%EC%97%90%EC%9D%B4%EB%B8%8C%EB%9F%AC%ED%96%84_%EB%A7%81%EC%BB%A8" title="에이브러햄 링컨">에이브러햄 링컨</a> 대통령이 두번째 임기 때 민주당의 <a class="mw-redirect" href="/wiki/%EC%95%A4%EB%93%9C%EB%A5%98_%EC%A1%B4%EC%8A%A8" title="앤드류 존슨">앤드류 존슨</a>을 부통령으로 임명한 것이 그것이다. 또 이당시에는 <a class="new" href="/w/index.php?title=%EA%B5%AD%EB%AF%BC%ED%86%B5%ED%95%A9%EB%8B%B9_(%EB%AF%B8%EA%B5%AD)&amp;action=edit&amp;redlink=1" title="국민통합당 (미국) (없는 문서)">국민통합당</a>이 창당되어 당원들이 기존에 있던 다른 정당과의 소속 관계를 유지할 수 있도록 했다. 이 때 이후로는 거국일치내각을 세우지 않는 것이 관례였다. 다만 국가적 위기나 전시였던 경우에는 양당이 잠시 "대통령을 중심으로 연대하는" 모습을 보이기도 했다. 대표적으로 <a class="mw-redirect" href="/wiki/%EC%A7%84%EC%A3%BC%EB%A7%8C_%EA%B3%B5%EC%8A%B5" title="진주만 공습">진주만 공습</a>, <a href="/wiki/%EC%A1%B4_F._%EC%BC%80%EB%84%A4%EB%94%94_%EC%95%94%EC%82%B4_%EC%82%AC%EA%B1%B4" title="존 F. 케네디 암살 사건">존 F. 케네디 암살 사건</a>, <a class="mw-redirect" href="/wiki/9.11_%ED%85%8C%EB%9F%AC" title="9.11 테러">9.11 테러</a>를 들 수가 있고 이때마다 대통령의 지지율은 급격히 상승했다.</li>
</ul>
