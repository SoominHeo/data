<p><b>거제해맞이역</b>(巨堤해맞이驛)은 <a href="/wiki/%EB%B6%80%EC%82%B0%EA%B4%91%EC%97%AD%EC%8B%9C" title="부산광역시">부산광역시</a> <a href="/wiki/%EC%97%B0%EC%A0%9C%EA%B5%AC" title="연제구">연제구</a> <a href="/wiki/%EA%B1%B0%EC%A0%9C%EB%8F%99" title="거제동">거제동</a>에 있는 <a href="/wiki/%EB%8F%99%ED%95%B4%EC%84%A0" title="동해선">동해선</a>의 <a href="/wiki/%EC%A0%84%EC%B2%A0%EC%97%AD" title="전철역">전철역</a>이다. <a href="/wiki/%EA%B4%91%EC%97%AD%EC%B2%A0%EB%8F%84" title="광역철도">광역철도</a> 개통 이전의 옛 역명은 <b>거제역</b>이였다. 전철노선의 역명을 결정할 당시, 동명의 <a href="/wiki/%EB%B6%80%EC%82%B0_%EB%8F%84%EC%8B%9C%EC%B2%A0%EB%8F%84_3%ED%98%B8%EC%84%A0" title="부산 도시철도 3호선">부산 도시철도 3호선</a> <a href="/wiki/%EA%B1%B0%EC%A0%9C%EC%97%AD" title="거제역">거제역</a>은 이 역과 멀리 떨어져 있으며 가까운 <a href="/wiki/%EA%B1%B0%EC%A0%9C%EC%97%AD" title="거제역">남문구역</a>과의 환승이 계획되어 있었다. 이에 3호선 <a href="/wiki/%ED%99%98%EC%8A%B9" title="환승">환승</a>으로 인한 역명 혼선 방지를 위해 환승예정인 '남문구역'을 '거제역'으로, 이 역(거제)을 현재의 역명(거제해맞이)으로 변경하였다.</p>

<h2><span class="mw-headline" id=".EA.B5.AC_.EC.97.AD.EC.82.AC">구 역사</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EA%B1%B0%EC%A0%9C%ED%95%B4%EB%A7%9E%EC%9D%B4%EC%97%AD&amp;action=edit&amp;section=6" title="부분 편집: 구 역사">원본 편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>구 역사는 옛 <a href="/wiki/%EC%A4%91%EC%95%99%EC%84%A0" title="중앙선">중앙선</a> <a href="/wiki/%ED%8C%94%EB%8B%B9%EC%97%AD" title="팔당역">팔당역</a>처럼 플랫폼 위에 있는 형태로, 이런 형태는 <a href="/wiki/%EB%8C%80%ED%95%9C%EB%AF%BC%EA%B5%AD" title="대한민국">대한민국</a>과 <a href="/wiki/%EC%9D%BC%EB%B3%B8" title="일본">일본</a>에서 거의 모두 사라졌기 때문에 매우 희귀하였다. 단층으로 된 역사는 대합실과 역무실이 한 건물 안에 있었으며, 지붕은 대칭을 이루는 두 평면으로 되어 있었는데, 이는 전형적인 <a href="/wiki/%EC%9D%BC%EC%A0%9C_%EA%B0%95%EC%A0%90%EA%B8%B0" title="일제 강점기">일제 강점기</a> 시골역의 형태였다. 이런 특징이 있음에도 불구하고 <a href="/wiki/%EB%8C%80%ED%95%9C%EB%AF%BC%EA%B5%AD%EC%9D%98_%EB%93%B1%EB%A1%9D%EB%AC%B8%ED%99%94%EC%9E%AC" title="대한민국의 등록문화재">등록문화재</a>로 등재되지 못하였고, <a href="/wiki/2009%EB%85%84" title="2009년">2009년</a>에 철거되었다. <a href="/wiki/%EB%8F%99%ED%95%B4%EB%82%A8%EB%B6%80%EC%84%A0" title="동해남부선">동해남부선</a>에서는 <a href="/wiki/%EC%86%A1%EC%A0%95%EC%97%AD_(%EB%B6%80%EC%82%B0)" title="송정역 (부산)">송정역</a>이 지정되었다.</p>
<p>일반 열차 운행 당시 출입구는 하나였다. 출입구의 방향은 역의 동쪽에 있는 대로인 <a class="mw-redirect" href="/wiki/%EA%B1%B0%EC%A0%9C%EB%A1%9C_(%EB%B6%80%EC%82%B0)" title="거제로 (부산)">거제로</a>의 반대방향이기 때문에 연산동, 거제3동 방면에서 거제역으로 진입하려는 사람들은 거제역 북쪽에 있는 과선교를 건너야 하였다. 거제4동 방면에서는 해맞이로로 명명된 2차로의 좁은 길만 건너면 바로 역으로 들어올 수 있었다.</p>
<p>거제역의 거제4동 방면으로는 '골목시장'이라는 <a href="/wiki/%EC%8B%9C%EC%9E%A5_(%EA%B2%BD%EC%A0%9C)" title="시장 (경제)">시장</a>이 있다. <a class="mw-redirect" href="/wiki/%EA%B1%B0%EC%A0%9C%EB%A1%9C_(%EB%B6%80%EC%82%B0)" title="거제로 (부산)">거제로</a> 쪽은 폭 5미터 정도의 화단이 1km 정도 <a href="/wiki/%EB%8F%99%ED%95%B4%EB%82%A8%EB%B6%80%EC%84%A0" title="동해남부선">동해남부선</a>과 평행하게 나 있었으며, 그 사이에는 일체의 건물이 없다. 거제역 북쪽 방향으로는 화단 안에 오솔길이 나 있어 소규모 <a href="/wiki/%EA%B3%B5%EC%9B%90" title="공원">공원</a> 역할을 하였다.</p>
<h3><span class="mw-headline" id=".EC.97.B4.EC.B0.A8.EC.9D.98_.EC.9A.B4.ED.96.89">열차의 운행</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EA%B1%B0%EC%A0%9C%ED%95%B4%EB%A7%9E%EC%9D%B4%EC%97%AD&amp;action=edit&amp;section=7" title="부분 편집: 열차의 운행">원본 편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p><a href="/wiki/2006%EB%85%84" title="2006년">2006년</a> <a href="/wiki/10%EC%9B%94_31%EC%9D%BC" title="10월 31일">10월 31일</a>까지 <a href="/wiki/%ED%86%B5%EA%B7%BC%EC%97%B4%EC%B0%A8_(%EB%8C%80%ED%95%9C%EB%AF%BC%EA%B5%AD)" title="통근열차 (대한민국)">통근열차</a>가 운행하였으나, <a href="/wiki/11%EC%9B%94_1%EC%9D%BC" title="11월 1일">11월 1일</a>부터 시행된 열차 운행 개편에 따라 <a href="/wiki/%EB%8F%99%ED%95%B4%EB%82%A8%EB%B6%80%EC%84%A0" title="동해남부선">동해남부선</a> <a href="/wiki/%ED%86%B5%EA%B7%BC%EC%97%B4%EC%B0%A8_(%EB%8C%80%ED%95%9C%EB%AF%BC%EA%B5%AD)" title="통근열차 (대한민국)">통근열차</a>가 사라지고 <a href="/wiki/%EB%AC%B4%EA%B6%81%ED%99%94%ED%98%B8" title="무궁화호">무궁화호</a>로 대체, 하루에 아침·저녁으로 2회 운행하였다. <a href="/wiki/2011%EB%85%84" title="2011년">2011년</a> <a href="/wiki/10%EC%9B%94_5%EC%9D%BC" title="10월 5일">10월 5일</a>부터 여객 취급이 중단되었다.</p>
