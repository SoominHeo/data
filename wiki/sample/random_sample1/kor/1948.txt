<p>《<b>거짓말</b>》(Lies)은 <a href="/wiki/%EC%9E%A5%EC%84%A0%EC%9A%B0" title="장선우">장선우</a>가 감독한 영화이다. <a href="/wiki/2000%EB%85%84" title="2000년">2000년</a> <a href="/wiki/1%EC%9B%94_8%EC%9D%BC" title="1월 8일">1월 8일</a>에 개봉하였다. <a href="/wiki/%EC%9E%A5%EC%A0%95%EC%9D%BC" title="장정일">장정일</a>의 《<a class="new" href="/w/index.php?title=%EB%82%B4%EA%B2%8C_%EA%B1%B0%EC%A7%93%EB%A7%90%EC%9D%84_%ED%95%B4%EB%B4%90_(%EC%B1%85)&amp;action=edit&amp;redlink=1" title="내게 거짓말을 해봐 (책) (없는 문서)">내게 거짓말을 해봐</a>》가 원작이다. <a href="/wiki/1999%EB%85%84" title="1999년">1999년</a> 8월과 10월에 등급 보류 판정을 받았으며, 개봉이 미뤄지자 불법 <a href="/wiki/%EB%B9%84%EB%94%94%EC%98%A4" title="비디오">비디오</a>와 <a href="/wiki/%EC%BD%A4%ED%8C%A9%ED%8A%B8_%EB%94%94%EC%8A%A4%ED%81%AC" title="콤팩트 디스크">CD</a>가 유통되기도 하였다.</p>

<h2><span class="mw-headline" id=".EB.93.B1.EA.B8.89_.EB.B3.B4.EB.A5.98_.ED.8C.8C.EB.8F.99">등급 보류 파동</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EA%B1%B0%EC%A7%93%EB%A7%90_(2000%EB%85%84_%EC%98%81%ED%99%94)&amp;action=edit&amp;section=1" title="부분 편집: 등급 보류 파동">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p><a href="/wiki/1999%EB%85%84" title="1999년">1999년</a> 8월 <a class="mw-redirect" href="/wiki/%EC%98%81%EC%83%81%EB%93%B1%EA%B8%89%EC%9C%84%EC%9B%90%ED%9A%8C" title="영상등급위원회">영상등급위원회</a>에서 "미성년자와의 변태적인 성관계와 가학 행위를 여과없이 묘사해 사회 통념에 어긋난다"며 등급 보류 결정을 내렸다. 제작사인 신씨네는 <a href="/wiki/8%EC%9B%94_17%EC%9D%BC" title="8월 17일">8월 17일</a> <a href="/wiki/%EA%B8%B0%EC%9E%90_(%EC%A7%81%EC%97%85)" title="기자 (직업)">기자</a>와 <a class="mw-redirect" href="/wiki/%ED%8F%89%EB%A1%A0%EA%B0%80" title="평론가">평론가</a>를 초대하여 시사회를 열었고 호평을 받았다. 또한 <a class="mw-redirect" href="/wiki/%EB%B2%A0%EB%8B%88%EC%8A%A4_%EC%98%81%ED%99%94%EC%A0%9C" title="베니스 영화제">베니스 영화제</a>에 출품되면서 호평을 받았으나 이에 관해 <a class="mw-redirect" href="/wiki/%EB%B0%94%ED%8B%B0%EC%B9%B8" title="바티칸">바티칸</a>에서 반대 성명을 발표하기도 하였다. 격렬한 찬반 논쟁 속에 같은 해 10월 다시 재심의를 거쳤으나 그럼에도 불구하고 등급 보류 판정을 받았다. 영화가 등급 보류 판정을 받자 영화에 대한 관심이 증폭되었고 불법 <a href="/wiki/%EB%B9%84%EB%94%94%EC%98%A4" title="비디오">비디오</a>와 <a href="/wiki/%EC%BD%A4%ED%8C%A9%ED%8A%B8_%EB%94%94%EC%8A%A4%ED%81%AC" title="콤팩트 디스크">CD</a>가 유통되기도 하였다.</p>
<h2><span class="mw-headline" id=".EC.A4.84.EA.B1.B0.EB.A6.AC">줄거리</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EA%B1%B0%EC%A7%93%EB%A7%90_(2000%EB%85%84_%EC%98%81%ED%99%94)&amp;action=edit&amp;section=2" title="부분 편집: 줄거리">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>제이는 꽤나 잘나갔지만 슬럼프에 빠져 아무것도 하지 않는 <a href="/wiki/%EC%A1%B0%EA%B0%81%EA%B0%80" title="조각가">조각가</a>이다. 와이는 중소 도시에서 살고 있는 고등학생이다. 와이의 친구 우리는 갑자기 공부도 하지 않고 제이의 작품집만 들여다 본다. 와이는 우리를 딱하게 여기고 순전히 우리를 제이에게 소개시켜줄 생각으로 제이에게 전화를 건다. 와이의 전화를 받은 제이는 와이의 목소리에 반하고 둘은 한 달 동안 폰섹스를 한다.</p>
<p>졸업을 얼마 남겨두지 않고 와이는 제이를 만나기로 한다. 처음에는 어색했지만 둘은 금세 친밀해 진다. 다시 만난 와이와 제이는 격렬하게 섹스를 하고 제이는 와이의 <a class="mw-redirect" href="/wiki/%EC%97%89%EB%8D%A9%EC%9D%B4" title="엉덩이">엉덩이</a>를 때린다. 이때쯤 제이의 아내는 <a class="mw-redirect" href="/wiki/%ED%8C%8C%EB%A6%AC_%EC%8B%9C" title="파리 시">파리</a>로 떠나버리는데 제이는 아내와 섹스할 때도 아내를 때렸던 것이다. 시간은 흘러 와이는 여대생이 되고 제이는 처음에는 손으로 엉덩이를 때리다 회초리, 철사줄, 대걸레를 가져와 때린다. 제이는 와이를 때리다 자신을 때려줄 것을 부탁한다. 와이는 처음에는 주저하다 시작이 지날수록 제이를 때리는 데에 능숙해진다.</p>
<p>그러나 와이의 오빠가 제이와의 관계를 알게 되면서 둘은 헤어지게 된다. 어느 날 아내와 파리에 있는 제이에게 와이가 찾아온다. 곡괭이 자루 하나만 들고. 다음 날 와이는 브라질로 떠난다. 아내는 제이에게 허벅지에 쓰인 내님이 누구냐고 묻고 제이는 거짓말을 한다.</p>
