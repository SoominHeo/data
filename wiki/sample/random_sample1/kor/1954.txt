<p><b>건담 빌드 파이터즈</b>(<span lang="ja" xml:lang="ja">ガンダムビルドファイターズ</span>)는 <a class="mw-redirect" href="/wiki/%EC%84%A0%EB%9D%BC%EC%9D%B4%EC%A6%88_(%EC%95%A0%EB%8B%88%EB%A9%94%EC%9D%B4%EC%85%98_%EC%8A%A4%ED%8A%9C%EB%94%94%EC%98%A4)" title="선라이즈 (애니메이션 스튜디오)">선라이즈</a>에서 제작한 <a href="/wiki/%EC%9D%BC%EB%B3%B8" title="일본">일본</a>의 <a href="/wiki/%EB%A1%9C%EB%B4%87" title="로봇">로봇</a> 애니메이션이다. <a href="/wiki/2013%EB%85%84" title="2013년">2013년</a> <a href="/wiki/10%EC%9B%94_7%EC%9D%BC" title="10월 7일">10월 7일</a>부터 <a href="/wiki/2014%EB%85%84" title="2014년">2014년</a> <a href="/wiki/3%EC%9B%94_31%EC%9D%BC" title="3월 31일">3월 31일</a>까지 <a href="/wiki/TV_%EB%8F%84%EC%BF%84" title="TV 도쿄">TV 도쿄</a> 계열에서 제1기로 방영되었으며, 동년 <a href="/wiki/10%EC%9B%94_8%EC%9D%BC" title="10월 8일">10월 8일</a>에 제2기는 <b>건담 빌드 파이터즈 트라이</b>(<span lang="ja" xml:lang="ja">ガンダムビルドファイターズトライ</span>)라는 제목으로 방영 중이다.</p>
<p><a href="/wiki/%EB%8C%80%ED%95%9C%EB%AF%BC%EA%B5%AD" title="대한민국">대한민국</a>에서는 <a href="/wiki/%EC%B9%B4%ED%88%B0_%EB%84%A4%ED%8A%B8%EC%9B%8C%ED%81%AC_(%EB%8C%80%ED%95%9C%EB%AF%BC%EA%B5%AD)" title="카툰 네트워크 (대한민국)">카툰 네트워크</a>와 <a href="/wiki/%EC%95%A0%EB%8B%88%EB%A7%A5%EC%8A%A4" title="애니맥스">애니맥스</a>, <a class="mw-redirect" href="/wiki/%EB%8B%88%EC%BC%88%EB%A1%9C%EB%94%94%EC%96%B8_%EC%BD%94%EB%A6%AC%EC%95%84" title="니켈로디언 코리아">니켈로디언 코리아</a>에서 더빙판으로 방영 중이다.</p>
<h2><span class="mw-headline" id=".EA.B0.9C.EC.9A.94">개요</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EA%B1%B4%EB%8B%B4_%EB%B9%8C%EB%93%9C_%ED%8C%8C%EC%9D%B4%ED%84%B0%EC%A6%88&amp;action=edit&amp;section=1" title="부분 편집: 개요">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>「군사」나 「전쟁」 등의 진지한 주제를 다루고 온 종래 <a href="/wiki/%EA%B1%B4%EB%8B%B4_%EC%8B%9C%EB%A6%AC%EC%A6%88" title="건담 시리즈">건담 시리즈</a>의 TV 방영 작품에서 세계관을 일신하고 현실에 <a href="/wiki/%EB%B0%98%EB%8B%A4%EC%9D%B4" title="반다이">반다이</a>에서 시판되고 있는 건프라와 이를 이용한 가공의 시뮬레이션 경기「건프라 배틀」을 테마로 한 작품.</p>
