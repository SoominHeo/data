<h1 class="firstHeading" id="firstHeading" lang="ko">검은 수요일</h1>
<div class="mw-body-content" id="bodyContent">
<div id="siteSub">위키백과, 우리 모두의 백과사전.</div>
<div id="contentSub"></div>
<div class="mw-jump" id="jump-to-nav">
					이동:					<a href="#mw-head">둘러보기</a>, 					<a href="#p-search">검색</a>
</div>
<div class="mw-content-ltr" dir="ltr" id="mw-content-text" lang="ko"><script>function mfTempOpenSection(id){var block=document.getElementById("mf-section-"+id);block.className+=" open-block";block.previousSibling.className+=" open-block";}</script><p><b>검은 수요일</b>(Black Wednesday)은 <a href="/wiki/1992%EB%85%84" title="1992년">1992년</a> <a href="/wiki/9%EC%9B%94_16%EC%9D%BC" title="9월 16일">9월 16일</a> <a href="/wiki/%EC%88%98%EC%9A%94%EC%9D%BC" title="수요일">수요일</a>, <a href="/wiki/%EC%A1%B0%EC%A7%80_%EC%86%8C%EB%A1%9C%EC%8A%A4" title="조지 소로스">조지 소로스</a>의 <a class="new" href="/w/index.php?title=%ED%80%80%ED%85%80_%ED%8E%80%EB%93%9C&amp;action=edit&amp;redlink=1" title="퀀텀 펀드 (없는 문서)">퀀텀 펀드</a> 및 다른 <a href="/wiki/%ED%97%A4%EC%A7%80_%ED%8E%80%EB%93%9C" title="헤지 펀드">헤지펀드</a>가 <a class="mw-redirect" href="/wiki/%EC%98%81%EA%B5%AD_%ED%8C%8C%EC%9A%B4%EB%93%9C" title="영국 파운드">영국 파운드</a>화를 투매해 <a href="/wiki/%EC%98%81%EA%B5%AD" title="영국">영국</a> 정부가 <a href="/wiki/%EC%9C%A0%EB%9F%BD_%ED%99%98%EC%9C%A8_%EB%A9%94%EC%BB%A4%EB%8B%88%EC%A6%98" title="유럽 환율 메커니즘">유럽 환율 메커니즘</a>(ERM)을 탈퇴한 사건이다.</p>
<h2><span class="mw-headline" id=".EC.9B.90.EC.9D.B8">원인</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EA%B2%80%EC%9D%80_%EC%88%98%EC%9A%94%EC%9D%BC&amp;action=edit&amp;section=1" title="부분 편집: 원인">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p><a href="/wiki/%EC%98%81%EA%B5%AD" title="영국">영국</a>은 <a href="/wiki/1990%EB%85%84" title="1990년">1990년</a> <a href="/wiki/10%EC%9B%94_8%EC%9D%BC" title="10월 8일">10월 8일</a> 유럽 환율 메커니즘에 가입하였다. 당시 <a class="mw-redirect" href="/wiki/%EC%98%81%EA%B5%AD_%ED%8C%8C%EC%9A%B4%EB%93%9C" title="영국 파운드">영국 파운드</a>화는 협약에 따라 <a href="/wiki/%EB%8F%85%EC%9D%BC_%EB%A7%88%EB%A5%B4%ED%81%AC" title="독일 마르크">독일 마르크</a>화의 ±6% 수준의 변동폭에서만 움직일 수 있었으며, 이 수준을 벗어날 경우 중앙은행들이 개입해 변동폭을 맞춰줘야 했었다. 하지만, <a href="/wiki/1990%EB%85%84" title="1990년">1990년</a> <a href="/wiki/10%EC%9B%94_3%EC%9D%BC" title="10월 3일">10월 3일</a> <a href="/wiki/%EB%8F%85%EC%9D%BC" title="독일">독일</a>이 재통일되면서 독일 정부는 통일 비용으로 마르크화를 천문학적으로 풀어버렸고, <a href="/wiki/%EB%8F%85%EC%9D%BC_%EC%97%B0%EB%B0%A9%EC%9D%80%ED%96%89" title="독일 연방은행">독일 연방은행</a>은 물가상승을 억제하기 위해 초고금리정책을 취해 경제를 안정화시킨다.</p>
<p>반면 독일 마르크화가 고평가됨에 따라 유럽 환율 메커니즘에 가입되어 있는 국가들은 협약에 따른 변동폭 유지를 위해 금리를 올렸고, 이 조치가 실업률 증가와 불황으로 이어지게 되면서 <a href="/wiki/1992%EB%85%84" title="1992년">1992년</a> <a href="/wiki/9%EC%9B%94_8%EC%9D%BC" title="9월 8일">9월 8일</a>자로 <a href="/wiki/%ED%95%80%EB%9E%80%EB%93%9C" title="핀란드">핀란드</a>가 독일 마르크화와의 연동제를 폐기하였고, <a href="/wiki/%EC%8A%A4%EC%9B%A8%EB%8D%B4" title="스웨덴">스웨덴</a>은 화폐 가치 유지를 위해 단기금리를 500% 인상하였다. <a href="/wiki/%EC%9D%B4%ED%83%88%EB%A6%AC%EC%95%84_%EB%A6%AC%EB%9D%BC" title="이탈리아 리라">이탈리아 리라</a>, <a href="/wiki/%EC%8A%A4%ED%8E%98%EC%9D%B8_%ED%8E%98%EC%84%B8%ED%83%80" title="스페인 페세타">스페인 페세타</a> 또한 대폭락하였다.</p>
<p>그러나 내부 사정이 좋지 않았던 영국은 배신행위를 할 수 없다며 환율 하락을 방어하겠다고 선언하였다.</p>
<h2><span class="mw-headline" id=".EA.B2.BD.EA.B3.BC">경과</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EA%B2%80%EC%9D%80_%EC%88%98%EC%9A%94%EC%9D%BC&amp;action=edit&amp;section=2" title="부분 편집: 경과">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p><a href="/wiki/1992%EB%85%84" title="1992년">1992년</a> <a href="/wiki/9%EC%9B%94_15%EC%9D%BC" title="9월 15일">9월 15일</a>, <a href="/wiki/%EB%8F%85%EC%9D%BC_%EC%97%B0%EB%B0%A9%EC%9D%80%ED%96%89" title="독일 연방은행">독일 연방은행</a> 총재의 독일 마르크화 고평가 발언을 기점으로 조지 소로스가 100억 달러를 동원, 영국 파운드화 투매가 시작되었다. 다른 헤지펀드들 또한 가담하게 되자 환율은 하한선까지 떨어지게 되었고, 이에 <a href="/wiki/%EC%9E%89%EA%B8%80%EB%9E%9C%EB%93%9C_%EC%9D%80%ED%96%89" title="잉글랜드 은행">잉글랜드 은행</a>이 환율 방어를 위해 투매한 파운드화를 외환보유고를 동원해 사들이게 되었다. 그러나, 하루에 두차례나 단기 이자율을 올렸음에도 불구하고 환율 하락이 지속되자 영국은 방어를 포기하였고, <a href="/wiki/1992%EB%85%84" title="1992년">1992년</a> <a href="/wiki/9%EC%9B%94_16%EC%9D%BC" title="9월 16일">9월 16일</a>자로 유럽 환율 메커니즘에서 탈퇴하였다.</p>
