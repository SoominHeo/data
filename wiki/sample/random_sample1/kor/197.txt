<h2><span class="mw-headline" id=".EB.B0.B0.EA.B2.BD">배경</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=E_(%EC%97%90%ED%94%BD%ED%95%98%EC%9D%B4%EC%9D%98_%EC%9D%8C%EB%B0%98)&amp;action=edit&amp;section=1" title="부분 편집: 배경">원본 편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<h3><span class="mw-headline" id=".EC.9E.91.EC.97.85_.EB.B0.8F_.EB.B0.9C.EB.A7.A4">작업 및 발매</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=E_(%EC%97%90%ED%94%BD%ED%95%98%EC%9D%B4%EC%9D%98_%EC%9D%8C%EB%B0%98)&amp;action=edit&amp;section=2" title="부분 편집: 작업 및 발매">원본 편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p><a href="/wiki/2009%EB%85%84" title="2009년">2009년</a> <a href="/wiki/7%EC%9B%94_22%EC%9D%BC" title="7월 22일">7월 22일</a>, <a href="/wiki/%ED%94%8C%EB%9E%98%EB%8B%9B_%EC%89%AC%EB%B2%84" title="플래닛 쉬버">플래닛 쉬버</a>와 <i><a href="/wiki/Remixing_the_Human_Soul" title="Remixing the Human Soul">Remixing the Human Soul</a></i>을 발매한 후, <a href="/wiki/%ED%83%80%EB%B8%94%EB%A1%9C" title="타블로">타블로</a>는 <a class="external text" href="http://mapthesoul.com/blog/tablo" rel="nofollow">개인 블로그</a> 를 통해서 <a href="/wiki/7%EC%9B%94_27%EC%9D%BC" title="7월 27일">7월 27일</a>, 또 다른 음반을 작업 중이라는 사실을 알렸다.<sup class="reference" id="cite_ref-8"><a href="#cite_note-8">[8]</a></sup></p>
<p><a href="/wiki/8%EC%9B%94_14%EC%9D%BC" title="8월 14일">8월 14일</a><sup class="reference" id="cite_ref-9"><a href="#cite_note-9">[9]</a></sup>, 19일에는 음반 막바지 작업 중이라고 글을 올렸고<sup class="reference" id="cite_ref-10"><a href="#cite_note-10">[10]</a></sup> 사이트 개편 후인 <a href="/wiki/8%EC%9B%94_22%EC%9D%BC" title="8월 22일">8월 22일</a>에는 9월에 발매된다는 걸 알리며 "이전에 발매한 <i><a class="mw-redirect" href="/wiki/%E9%AD%82_:_Map_the_Soul" title="魂 : Map the Soul">魂 : Map the Soul</a></i>과 리믹스 음반인 <i><a href="/wiki/Remixing_the_Human_Soul" title="Remixing the Human Soul">Remixing the Human Soul</a></i>은 이 음반을 위한 연습이었다"고 밝혔다.<sup class="reference" id="cite_ref-blo_1-1"><a href="#cite_note-blo-1">[1]</a></sup></p>
<p><a href="/wiki/8%EC%9B%94_24%EC%9D%BC" title="8월 24일">8월 24일</a>에는 "음반에서 발성과 플로우, 가사 스타일도 다양하게 시도했다"고 밝혔고<sup class="reference" id="cite_ref-11"><a href="#cite_note-11">[11]</a></sup> <a href="/wiki/DJ_%ED%88%AC%EC%BB%B7" title="DJ 투컷">투컷</a>은 <a href="/wiki/%EB%AF%B8%ED%88%AC%EB%8D%B0%EC%9D%B4" title="미투데이">미투데이</a> 블로그인 MapNOW를 통해 첫 믹스 작업을 시작했다며 작업 사진을 올렸다. 25일에는 <a href="/wiki/%ED%94%8C%EB%9E%98%EB%8B%9B_%EC%89%AC%EB%B2%84" title="플래닛 쉬버">플래닛 쉬버</a>의 멤버인 <a href="/wiki/%ED%95%84%ED%84%B0_(%EC%9D%8C%EC%95%85%EA%B0%80)" title="필터 (음악가)">필터</a> 또한 자신의 <a class="external text" href="http://mapthesoul.com/blog/philtre" rel="nofollow">블로그</a> 를 통해서 <a href="/wiki/%ED%94%8C%EB%9E%98%EB%8B%9B_%EC%89%AC%EB%B2%84" title="플래닛 쉬버">플래닛 쉬버</a>가 <i>[e]</i> 음반에 참여함을 밝혔다.<sup class="reference" id="cite_ref-12"><a href="#cite_note-12">[12]</a></sup> 26일에는 <a href="/wiki/%EB%AF%B8%EC%93%B0%EB%9D%BC_%EC%A7%84" title="미쓰라 진">미쓰라 진</a>이 MapNOW에 리스트를 찍어 올렸고 이에 타블로는 댓글을 통해서 [e] 음반의 트랙리스트라고 밝혔다.<sup class="reference" id="cite_ref-13"><a href="#cite_note-13">[13]</a></sup> 27일에는 타블로가 자신의 블로그를 통해서 다음 음반의 제목은 <i>[e]</i>이며, 2CD 구성에 총 30 트랙이 수록된다며 이후 추가 정보는 조금씩 알리겠다고 밝혔다.<sup class="reference" id="cite_ref-Eblo_14-0"><a href="#cite_note-Eblo-14">[14]</a></sup></p>
<p><a href="/wiki/8%EC%9B%94_30%EC%9D%BC" title="8월 30일">8월 30일</a>에는 미쓰라 진이 블로그를 통해 정규 음반의 믹스 작업이 절반 정도 끝났다고 올렸고<sup class="reference" id="cite_ref-15"><a href="#cite_note-15">[15]</a></sup> <a href="/wiki/9%EC%9B%94_1%EC%9D%BC" title="9월 1일">9월 1일</a> <a href="/wiki/%EB%A7%B5%EB%8D%94%EC%86%8C%EC%9A%B8" title="맵더소울">맵더소울</a> 뉴스를 통해 "2CD 구성에 30 트랙이 수록되며 <a href="/wiki/2009%EB%85%84" title="2009년">2009년</a> <a href="/wiki/9%EC%9B%94_16%EC%9D%BC" title="9월 16일">9월 16일</a>에 발매된다"고 알렸다.<sup class="reference" id="cite_ref-Mapnews_2-1"><a href="#cite_note-Mapnews-2">[2]</a></sup> 이어 음반 발매 10일을 앞둔 <a href="/wiki/9%EC%9B%94_6%EC%9D%BC" title="9월 6일">9월 6일</a>부터 타이틀곡의 뮤직비디오 촬영에 들어갔다.<sup class="reference" id="cite_ref-16"><a href="#cite_note-16">[16]</a></sup><sup class="reference" id="cite_ref-17"><a href="#cite_note-17">[17]</a></sup> 발매 7일 전인 <a href="/wiki/9%EC%9B%94_9%EC%9D%BC" title="9월 9일">9월 9일</a>, 《<a href="/wiki/%EA%B4%B4%EB%AC%BC_(2006%EB%85%84_%EC%98%81%ED%99%94)" title="괴물 (2006년 영화)">괴물</a>》, 《<a href="/wiki/%ED%81%B4%EB%A1%9C%EB%B2%84%ED%95%84%EB%93%9C" title="클로버필드">클로버필드</a>》를 패러디한 57초 분량의 <i>[e]</i> 티저 영상이 공개되었다.<sup class="reference" id="cite_ref-18"><a href="#cite_note-18">[18]</a></sup> 이어 <a href="/wiki/DJ_%ED%88%AC%EC%BB%B7" title="DJ 투컷">DJ 투컷</a>이 MapNOW를 통해 음반의 막바지 작업인 마스터링 작업 중임을 알렸다.</p>
<p><a href="/wiki/9%EC%9B%94_10%EC%9D%BC" title="9월 10일">9월 10일</a>, 자신들의 사이트인 <a href="/wiki/%EB%A7%B5%EB%8D%94%EC%86%8C%EC%9A%B8" title="맵더소울">맵더소울</a>을 통해 <a href="/wiki/%EB%B0%95%EC%A7%80%EC%9C%A4_(%EA%B0%80%EC%88%98)" title="박지윤 (가수)">박지윤</a>, <a href="/wiki/MYK" title="MYK">MYK</a>, <a href="/wiki/%EB%8F%84%EB%81%BC_(%EA%B0%80%EC%88%98)" title="도끼 (가수)">Dok2</a>, <a href="/wiki/%EC%96%91%EB%8F%99%EA%B7%BC" title="양동근">양동근</a> 등이 참여한 <i>[e]</i> 음반의 트랙리스트를 공개했다. 구성에 대해 2CD 구성에 74 페이지 분량의 미니북이 포함된다고 알렸다.<sup class="reference" id="cite_ref-minibook_6-1"><a href="#cite_note-minibook-6">[6]</a></sup> 당일 밤에는 <a href="/wiki/DJ_%ED%88%AC%EC%BB%B7" title="DJ 투컷">DJ 투컷</a>은 <a class="new" href="/w/index.php?title=%EB%8B%A4%EC%9D%BC%EB%A0%88%EC%9D%B4%ED%8B%B0%EB%93%9C_%ED%94%BC%ED%94%8C%EC%8A%A4&amp;action=edit&amp;redlink=1" title="다일레이티드 피플스 (없는 문서)">다일레이티드 피플스</a>의 <a class="new" href="/w/index.php?title=%EB%9D%BC%EC%B9%B4_(%EA%B0%80%EC%88%98)&amp;action=edit&amp;redlink=1" title="라카 (가수) (없는 문서)">라카</a> 참여에 대해<sup class="reference" id="cite_ref-19"><a href="#cite_note-19">[19]</a></sup>, <a href="/wiki/%EB%AF%B8%EC%93%B0%EB%9D%BC_%EC%A7%84" title="미쓰라 진">미쓰라 진</a>은 자신의 블로그를 통해 음반의 마스터링 작업이 끝났다고 글을 작성했다.<sup class="reference" id="cite_ref-20"><a href="#cite_note-20">[20]</a></sup> <a href="/wiki/9%EC%9B%94_11%EC%9D%BC" title="9월 11일">9월 11일</a>에는 <a href="/wiki/%EB%A7%B5%EB%8D%94%EC%86%8C%EC%9A%B8" title="맵더소울">맵더소울닷컴</a>을 통해 음반에 대한 자세한 정보와 예약 주문을 받기 시작했다.<sup class="reference" id="cite_ref-shop_21-0"><a href="#cite_note-shop-21">[21]</a></sup> <a href="/wiki/9%EC%9B%94_12%EC%9D%BC" title="9월 12일">9월 12일</a>에는 음반의 커버 아트를 공개했다.<sup class="reference" id="cite_ref-22"><a href="#cite_note-22">[22]</a></sup></p>
<p>음반 발매를 이틀 앞둔 <a href="/wiki/9%EC%9B%94_14%EC%9D%BC" title="9월 14일">9월 14일</a>에는 대폭적으로 음반에 관해 업데이트를 했다. <a href="/wiki/%ED%83%80%EB%B8%94%EB%A1%9C" title="타블로">타블로</a>는 <a href="/wiki/%EB%AF%B8%ED%88%AC%EB%8D%B0%EC%9D%B4" title="미투데이">미투데이</a>를 통해 음반에 관련되어 <a href="/wiki/9%EC%9B%94_19%EC%9D%BC" title="9월 19일">9월 19일</a>에 열리는 <i>EPIK HIGH [e] PARADE 2009</i>의 표를 제공하는 이벤트를 했으며 블로그를 통해서는 이미 완성된 음반의 모습과 온·오프라인과 <a class="mw-redirect" href="/wiki/%EC%95%84%EC%9D%B4%ED%8A%A0%EC%A6%88" title="아이튠즈">아이튠즈</a>를 통해 전국 및 전 세계 발매된다고 글을 올렸고<sup class="reference" id="cite_ref-23"><a href="#cite_note-23">[23]</a></sup> MapTV를 통해서는 음반과 뮤직비디오 등 6집에 관련된 인터뷰<sup class="reference" id="cite_ref-interview_24-0"><a href="#cite_note-interview-24">[24]</a></sup> 와 음반에 수록된 전곡을 들을 수 있는 티저 영상을 공개하였다.<sup class="reference" id="cite_ref-25"><a href="#cite_note-25">[25]</a></sup></p>
<h3><span class="mw-headline" id=".ED.99.9C.EB.8F.99">활동</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=E_(%EC%97%90%ED%94%BD%ED%95%98%EC%9D%B4%EC%9D%98_%EC%9D%8C%EB%B0%98)&amp;action=edit&amp;section=3" title="부분 편집: 활동">원본 편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p><a href="/wiki/9%EC%9B%94_16%EC%9D%BC" title="9월 16일">9월 16일</a>, 정규 6집인 <i>[e]</i> 음반을 발매 후, 에픽하이는 <a href="/wiki/%EC%95%95%EA%B5%AC%EC%A0%95%EB%8F%99" title="압구정동">압구정</a> <a class="mw-redirect" href="/wiki/CGV" title="CGV">CGV</a>극장에서 쇼케이스를 가졌다.<sup class="reference" id="cite_ref-showcase_7-1"><a href="#cite_note-showcase-7">[7]</a></sup> 이어 타이틀 곡이자 쟈니브로스 제작, 영화 《<a href="/wiki/%EA%B4%B4%EB%AC%BC_(2006%EB%85%84_%EC%98%81%ED%99%94)" title="괴물 (2006년 영화)">괴물</a>》을 패러디한 〈따라해 (Wannabe)〉의 티저 영상이 공개되었다.<sup class="reference" id="cite_ref-26"><a href="#cite_note-26">[26]</a></sup> <a href="/wiki/9%EC%9B%94_17%EC%9D%BC" title="9월 17일">9월 17일</a>, <a class="mw-redirect" href="/wiki/MBC" title="MBC">MBC</a> 《<a class="mw-redirect" href="/wiki/%EC%9D%8C%EC%95%85%EC%97%AC%ED%96%89_%EB%9D%BC%EB%9D%BC%EB%9D%BC" title="음악여행 라라라">음악여행 라라라</a>》, <a class="mw-redirect" href="/wiki/M.net" title="M.net">M.net</a> 《<a id="cite_ref-27"><a href="#cite_note-27">[27]</a></sup> <a href="/wiki/9%EC%9B%94_19%EC%9D%BC" title="9월 19일">9월 19일</a>, <a class="mw-redirect mw-disambig" href="/wiki/MYK" title="MYK">MYK</a> 등과 함께 6집 발매 기념 콘서트인 <i>EPIK HIGH [e]-PARADE 2009</i>를 성황리에 마쳤다.<sup class="reference" id="cite_ref-concert_28-0"><a href="#cite_note-concert-28">[28]</a></sup> <a href="/wiki/9%EC%9B%94_25%EC%9D%BC" title="9월 25일">9월 25일</a>, <a href="/wiki/%EB%AF%B8%EA%B5%AD" title="미국">미국</a> <a class="mw-redirect" href="/wiki/%EC%95%84%EC%9D%B4%ED%8A%A0%EC%A6%88" <p><a href="/wiki/2009%EB%85%84" title="2009년">2009년</a> <a href="/wiki/10%EC%9B%94_7%EC%9D%BC" title="10월 7일">10월 7일</a>, <a href="/wiki/DJ_%ED%88%AC%EC%BB%B7" title="DJ 투컷">DJ 투컷</a>이 자신의 블로그를 통해 <a href="/wiki/10%EC%9B%94_13%EC%9D%BC" title="10월 13일">10월 13일</a>에 결혼식을 치르고 <a href="/wiki/10%EC%9B%94_15%EC%9D%BC" title="10월 15일">10월 15일</a> 현역 입대를 한다고 밝혔다.<sup class="reference" id="cite_ref-tukutz_30-0"><a href="#cite_note-tukutz-30">[30]</a></sup> <a href="/wiki/10%EC%9B%94_8%EC%9D%BC" title="10월 8일">10월 8일</a>, 에픽하이는 DJ 투컷의 입대로 인해 <a href="/wiki/10%EC%9B%94_10%EC%9D%BC" title="10월 10일">10월 10일</a> 《<a href="/wiki/%EC%87%BC!_%EC%9D%8C%EC%95%85%EC%A4%91%EC%8B%AC" title="쇼! 음악중심">쇼! 음악중심</a>》을 마지막으로 6집 활동을 마감한다고 밝혔다.<sup class="reference" id="cite_ref-31"><a href="#cite_note-31">[31]</a></sup> 이후 10일에 사전녹화된 마지막 방송이 <a href="/wiki/10%EC%9B%94_17%EC%9D%BC" title="10월 17일">10월 17일</a>에 방송됨으로써 6집 방송 활동을 마쳤다. 이는 여태 에픽하이가 발매한 음반 활동 기간 중 가장 짧은 기간으로 기록되었다.</p>
<p><a href="/wiki/2009%EB%85%84" title="2009년">2009년</a> <a href="/wiki/10%EC%9B%94_15%EC%9D%BC" title="10월 15일">10월 15일</a>, 〈따라해 (Wannabe)〉 뮤직비디오에 이은 파트 2인 〈트로트〉 + "High Technology"의 뮤직비디오가 공개되었고<sup class="reference" id="cite_ref-32"><a href="#cite_note-32">[32]</a></sup> <a href="/wiki/10%EC%9B%94_30%EC%9D%BC" title="10월 30일">10월 30일</a>, <a href="/wiki/%EB%84%A4%EC%9D%B4%ED%8A%B8" title="네이트">네이트</a>의 시맨틱 검색 테마곡인 〈Search〉를 공개하고, <a href="/wiki/%EC%8B%B8%EC%9D%B4%EC%9B%94%EB%93%9C" title="싸이월드">싸이월드</a>의 BGM으로 무료 배포하였다.<sup class="reference" id="cite_ref-33"><a href="#cite_note-33">[33]</a></sup> 이후 <a href="/wiki/2009%EB%85%84_11%EC%9B%94" title="2009년 11월">2009년 11월</a>, 음반 내에 수록된 "Still Here", "Supreme 100", 〈말로맨〉, "Rocksteady", "Rocksteady <small>(Korean Ver.)</small>", 〈흉〉 등이 욕설, 마약 및 선정성으로 인해 청소년보호위원회로부터 청소년 유해매체로 지정되었다.</p>
<p><a href="/wiki/12%EC%9B%94_7%EC%9D%BC" title="12월 7일">12월 7일</a>, 영화 《용서는 없다》에 <i>[e]</i>에 수록된 "Slow Motion"이 영화와 함께 뮤직비디오로 제작되었으며 같은 음반에 수록된 "Heaven", "Owls. Shadows. Tears."가 수록곡으로 삽입된다고 알렸다.<sup class="reference" id="cite_ref-christmas_34-0"><a href="#cite_note-christmas-34">[34]</a></sup></p>
<p><a href="/wiki/12%EC%9B%94_10%EC%9D%BC" title="12월 10일">12월 10일</a>, <a class="mw-redirect" href="/wiki/%EA%B3%A8%EB%93%A0%EB%94%94%EC%8A%A4%ED%81%AC" title="골든디스크">골든디스크</a> 힙합 부분을 수상하였고 이 날 무대에서 "High Technology"와 〈따라해 (Wannabe)〉를 불렀다.</p>

<h2><span class="mw-headline" id=".5Be.5Dternity">[e]ternity</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=E_(%EC%97%90%ED%94%BD%ED%95%98%EC%9D%B4%EC%9D%98_%EC%9D%8C%EB%B0%98)&amp;action=edit&amp;section=6" title="부분 편집: [e]ternity">원본 편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p><i><b>[e]ternity</b></i>는 6집의 한정반으로 랜덤 방식으로 100장 한정 발매되었다. <i>[e]</i> 앨범 발표후 한정판 존재 유무가 맵더소울의 ONE 게시판에서 많이 논의 되었으나 <a href="/wiki/2009%EB%85%84" title="2009년">2009년</a> <a href="/wiki/10%EC%9B%94_27%EC%9D%BC" title="10월 27일">10월 27일</a>, 한정반이 존재함이 드러났다. <i>[e]ternity</i>는 기존에 냈던 한정반과 일반반과는 달리 히든 트랙과 에픽하이의 친필 사인이 없으나, 한정판 인증서에 맵더소울의 도장이 찍혀있으며 일반반의 CD 배경색인 흰색과 분홍과는 다르게 반대로 대칭되어 있다.<sup class="reference" id="cite_ref-36"><a href="#cite_note-36">[36]</a></sup></p>

<p>발매 3일 후인 <a href="/wiki/2009%EB%85%84" title="2009년">2009년</a> <a href="/wiki/9%EC%9B%94_19%EC%9D%BC" title="9월 19일">9월 19일</a>, <a class="mw-redirect mw-disambig" href="/wiki/%EC%98%AC%EB%A6%BC%ED%94%BD%EA%B3%B5%EC%9B%90" title="올림픽공원">올림픽공원</a> 올림픽홀에서 <a href="/wiki/MYK" title="MYK">MYK</a>, <a href="/wiki/%ED%94%8C%EB%9E%98%EB%8B%9B_%EC%89%AC%EB%B2%84" title="플래닛 쉬버">플래닛 쉬버</a>와 함께 6집 발매 기념 <i>EPIK HIGH [e]-PARADE 2009</i>라는 이름의 콘서트를 가진다고 사이트를 통해서 알렸다.<sup class="reference" id="cite_ref-37"><a href="#cite_note-37">[37]</a></sup> 이어 사이트 개편 후, MapTV를 통해 콘서트 홍보영상을 올렸다.<sup class="reference" id="cite_ref-38"><a href="#cite_note-38">[38]</a></sup> <a href="/wiki/9%EC%9B%94_8%EC%9D%BC" title="9월 8일">9월 8일</a>, 오는 19일의 공연에 대해 "국내 최대의 클럽 파티 씬으로 만들겠다"고 밝혔다. 또 4DJ 형식으로 <a href="/wiki/DJ_%ED%88%AC%EC%BB%B7" title="DJ 투컷">DJ 투컷</a>과 <a href="/wiki/%ED%94%8C%EB%9E%98%EB%8B%9B_%EC%89%AC%EB%B2%84" title="플래닛 쉬버">플래닛 쉬버</a>가 함께 선다고 밝혔다.<sup class="reference" id="cite_ref-39"><a href="#cite_note-39">[39]</a></sup> 이어 콘서트를 4일 앞둔 <a href="/wiki/9%EC%9B%94_15%EC%9D%BC" title="9월 15일">9월 15일</a>, <a href="/wiki/%ED%83%80%EB%B8%94%EB%A1%9C" title="타블로">타블로</a>는 블로그를 통해 게스트로 <a href="/wiki/%ED%83%80%EC%9D%B4%EA%B1%B0_JK" title="타이거 JK">타이거 JK</a>가 선다고 밝혔다.<sup class="reference" id="cite_ref-40"><a href="#cite_note-40">[40]</a></sup> 공연 당일, <a href="/wiki/%EB%8F%84%EB%81%BC_(%EA%B0%80%EC%88%98)" title="도끼 (가수)">Dok2</a>와 <a href="/wiki/MYK" title="MYK">MYK</a>의 오프닝 무대와 <a href="/wiki/%EC%97%90%ED%94%BD%ED%95%98%EC%9D%B4" title="에픽하이">에픽하이</a>의 1부, 이어 게스트 스테이지에 <a href="/wiki/%EB%8B%A4%EC%9D%B4%EB%82%98%EB%AF%B9_%EB%93%80%EC%98%A4" title="다이나믹 듀오">다이나믹 듀오</a>, <a class="mw-redirect" href="/wiki/%EB%93%9C%EB%A0%81%ED%81%B0_%ED%83%80%EC%9D%B4%EA%B1%B0" title="드렁큰 타이거">드렁큰타이거</a>가 공연한 후, 다시 에픽하이의 2부 공연 형식으로 진행되었다.<sup class="reference" id="cite_ref-concert_28-1"><a href="#cite_note-concert-28">[28]</a></sup></p>

<h3><span class="mw-headline" id="Hear_the_Sound">Hear the Sound</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=E_(%EC%97%90%ED%94%BD%ED%95%98%EC%9D%B4%EC%9D%98_%EC%9D%8C%EB%B0%98)&amp;action=edit&amp;section=12" title="부분 편집: Hear the Sound">원본 편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p><a href="/wiki/10%EC%9B%94_7%EC%9D%BC" title="10월 7일">10월 7일</a>, <a class="mw-redirect" href="/wiki/%EB%93%9C%EB%A0%81%ED%81%B0_%ED%83%80%EC%9D%B4%EA%B1%B0" title="드렁큰 타이거">드렁큰타이거</a>, <a href="/wiki/%EC%9C%A4%EB%AF%B8%EB%9E%98" title="윤미래">윤미래</a>, <a class="mw-redirect" href="/wiki/%ED%81%B4%EB%9E%98%EC%A7%80%EC%BD%B0%EC%9D%B4" title="클래지콰이">클래지콰이</a> 등과 함께 <a class="mw-redirect mw-disambig" href="/wiki/%EC%98%AC%EB%A6%BC%ED%94%BD%EA%B3%B5%EC%9B%90" title="올림픽공원">올림픽공원</a> 올림픽홀에서 열린 자선 콘서트인 <i>Hear the Sound</i> 콘서트에 참여하였다. <a href="/wiki/DJ_%ED%88%AC%EC%BB%B7" title="DJ 투컷">DJ 투컷</a>이 군입대를 앞두고 마지막 공연이었으며 마지막 순서로 등장해 〈따라해 (Wannabe)〉, "High Technology", "Fly"를 선보였다.</p>

<p><a href="/wiki/12%EC%9B%94_24%EC%9D%BC" title="12월 24일">12월 24일</a>, <a href="/wiki/%EA%B3%A0%EB%A0%A4%EB%8C%80%ED%95%99%EA%B5%90" title="고려대학교">고려대학교</a> 화정체육관에서 <a href="/wiki/%EB%A7%B5%EB%8D%94%EC%86%8C%EC%9A%B8" title="맵더소울">맵더소울</a> 소속 아티스트들과 함께 <i>Last Christmas</i> 크리스마스 공연을 가졌다. 이 공연은 군입대한 <a href="/wiki/DJ_%ED%88%AC%EC%BB%B7" title="DJ 투컷">DJ 투컷</a>없이 치른 첫 콘서트이자 마지막 콘서트로, 이 공연을 기준으로 계속 가져왔던 크리스마스 기념 콘서트를 3년 동안 중지한다고 알려왔다. <a href="/wiki/1%EC%9B%94_5%EC%9D%BC" title="1월 5일">1월 5일</a>, <a href="/wiki/%EC%9C%A0%ED%8A%9C%EB%B8%8C" title="유튜브">유튜브</a>와 <a href="/wiki/%EB%A7%B5%EB%8D%94%EC%86%8C%EC%9A%B8" title="맵더소울">맵더소울</a>의 MapTV를 통해 콘서트의 영상을 게시하였다.<sup class="reference" id="cite_ref-lc_42-0"><a href="#cite_note-lc-42">[42]</a></sup></p>
