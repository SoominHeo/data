<p><b>게리 페이 로크</b>(<span style="font-size: smaller;"><a href="/wiki/%EC%98%81%EC%96%B4" title="영어">영어</a>:</span> <span lang="en" xml:lang="en">Gary Faye Locke</span>, <a href="/wiki/1950%EB%85%84" title="1950년">1950년</a> <a href="/wiki/1%EC%9B%94_21%EC%9D%BC" title="1월 21일">1월 21일</a> ~ )는 <a href="/wiki/%EB%AF%B8%EA%B5%AD" title="미국">미국</a>의 정치인이다.</p>
<p><a href="/wiki/%EC%9B%8C%EC%8B%B1%ED%84%B4_%EC%A3%BC" title="워싱턴 주">워싱턴 주</a> <a href="/wiki/%EC%8B%9C%EC%95%A0%ED%8B%80" title="시애틀">시애틀</a>에서 <a href="/wiki/%EC%A4%91%EA%B5%AD" title="중국">중국</a> <a href="/wiki/%EC%A4%91%EA%B5%AD%EA%B3%84_%EB%AF%B8%EA%B5%AD%EC%9D%B8" title="중국계 미국인">이민</a> 후손으로 태어났다. 駱家輝(뤄자후이, <a href="/wiki/%EA%B4%91%EB%91%A5%EC%96%B4" title="광둥어">광둥어</a> 발음으로는 록가파이)이라는 중국 이름도 있다. <a href="/wiki/%EC%98%88%EC%9D%BC_%EB%8C%80%ED%95%99%EA%B5%90" title="예일 대학교">예일 대학교</a>에서 <a href="/wiki/%EC%A0%95%EC%B9%98%ED%95%99" title="정치학">정치학</a>을 전공하여 학사 학위를 취득한 후 <a href="/wiki/%EB%B3%B4%EC%8A%A4%ED%84%B4_%EB%8C%80%ED%95%99%EA%B5%90" title="보스턴 대학교">보스턴 대학교</a> 로스쿨에서 법무박사(J.D.) 학위를 받았다. 워싱턴 주로 돌아와 <a href="/wiki/1983%EB%85%84" title="1983년">1983년</a> ~ <a href="/wiki/1994%EB%85%84" title="1994년">1994년</a> <a href="/wiki/%EB%AF%BC%EC%A3%BC%EB%8B%B9_(%EB%AF%B8%EA%B5%AD)" title="민주당 (미국)">민주당</a> 소속으로 주의회 하원의원으로 재직했다. <a href="/wiki/1994%EB%85%84" title="1994년">1994년</a> ~ <a href="/wiki/1997%EB%85%84" title="1997년">1997년</a> <a href="/wiki/%ED%82%B9_%EA%B5%B0_(%EC%9B%8C%EC%8B%B1%ED%84%B4_%EC%A3%BC)" title="킹 군 (워싱턴 주)">킹 군</a> 행정부 집행관을 지냈다. <a href="/wiki/1996%EB%85%84" title="1996년">1996년</a> 워싱턴 주지사 선거에 출마, 당선되어 <a href="/wiki/1997%EB%85%84" title="1997년">1997년</a> <a class="new" href="/w/index.php?title=%EC%A4%91%EA%B5%AD%EA%B3%84_%EB%AF%B8%EA%B5%AD&amp;action=edit&amp;redlink=1" title="중국계 미국 (없는 문서)">중국계</a>로는 처음으로 미국에서 주지사로 취임하였다. 또한 <a href="/wiki/%ED%95%98%EC%99%80%EC%9D%B4_%EC%A3%BC" title="하와이 주">하와이 주</a>를 제외한 지역 최초의 <a href="/wiki/%EC%95%84%EC%8B%9C%EC%95%84%EA%B3%84_%EB%AF%B8%EA%B5%AD%EC%9D%B8" title="아시아계 미국인">아시아계</a> 주지사였다. <a href="/wiki/2000%EB%85%84" title="2000년">2000년</a> 재선되어 <a href="/wiki/2005%EB%85%84" title="2005년">2005년</a> 초까지 주지사로 재직하였다. <a href="/wiki/2009%EB%85%84" title="2009년">2009년</a> <a href="/wiki/%EB%B2%84%EB%9D%BD_%EC%98%A4%EB%B0%94%EB%A7%88_%ED%96%89%EC%A0%95%EB%B6%80" title="버락 오바마 행정부">버락 오바마 행정부</a>의 <a href="/wiki/%EB%AF%B8%EA%B5%AD_%EC%83%81%EB%AC%B4%EB%B6%80" title="미국 상무부">상무장관</a>으로 지명되어 재직하다 <a href="/wiki/2011%EB%85%84" title="2011년">2011년</a> <a href="/wiki/%EC%A4%91%ED%99%94%EC%9D%B8%EB%AF%BC%EA%B3%B5%ED%99%94%EA%B5%AD" title="중화인민공화국">중화인민공화국</a> 주재 대사로 지명되었다. <a href="/wiki/8%EC%9B%94_1%EC%9D%BC" title="8월 1일">8월 1일</a>자로 상무부 장관에서 퇴직하고, 중국계로는 최초로 중국 주재 미국 대사 자리에 올랐으며, <a href="/wiki/8%EC%9B%94_12%EC%9D%BC" title="8월 12일">8월 12일</a> 중국에 부임하여,<sup class="reference" id="cite_ref-1"><a href="#cite_note-1">[1]</a></sup> <a href="/wiki/2014%EB%85%84_2%EC%9B%94" title="2014년 2월">2014년 2월</a>까지 주중 대사로 재직했다.</p>
