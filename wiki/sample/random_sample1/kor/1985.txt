<p><b>게리 로널드 하니</b>(Gary Ronald Honey, <a href="/wiki/1959%EB%85%84" title="1959년">1959년</a> <a href="/wiki/7%EC%9B%94_26%EC%9D%BC" title="7월 26일">7월 26일</a> ~ )는 은퇴한 <a href="/wiki/%EC%98%A4%EC%8A%A4%ED%8A%B8%EB%A0%88%EC%9D%BC%EB%A6%AC%EC%95%84" title="오스트레일리아">오스트레일리아</a>의 <a href="/wiki/%EB%A9%80%EB%A6%AC%EB%9B%B0%EA%B8%B0" title="멀리뛰기">멀리뛰기</a> 선수로 <a class="mw-redirect" href="/wiki/1984%EB%85%84_%EB%A1%9C%EC%8A%A4%EC%95%A4%EC%A0%A4%EB%A0%88%EC%8A%A4_%EC%98%AC%EB%A6%BC%ED%94%BD" title="1984년 로스앤젤레스 올림픽">1984년 로스앤젤레스 올림픽</a>에서 8.24m와 함께 <a href="/wiki/%EC%B9%BC_%EB%A3%A8%EC%9D%B4%EC%8A%A4" title="칼 루이스">칼 루이스</a>에 밀려 <a href="/wiki/%EC%9D%80%EB%A9%94%EB%8B%AC" title="은메달">은메달</a>을 땄다.</p>
<p><a href="/wiki/%EB%B9%85%ED%86%A0%EB%A6%AC%EC%95%84_%EC%A3%BC" title="빅토리아 주">빅토리아 주</a> <a href="/wiki/%EB%A9%9C%EB%B2%84%EB%A5%B8" title="멜버른">멜버른</a>의 외곽 토머스타운에서 태어난 하니는 <a href="/wiki/1980%EB%85%84" title="1980년">1980년</a>으로 시작하여 3개의 <a href="/wiki/%ED%95%98%EA%B3%84_%EC%98%AC%EB%A6%BC%ED%94%BD" title="하계 올림픽">하계 올림픽</a>에 참가하였다.</p>
<p>추가로 그는 <a href="/wiki/1982%EB%85%84" title="1982년">1982년</a>과 <a href="/wiki/1986%EB%85%84" title="1986년">1986년</a> <a href="/wiki/%EC%98%81%EC%97%B0%EB%B0%A9_%EA%B2%BD%EA%B8%B0_%EB%8C%80%ED%9A%8C" title="영연방 경기 대회">영연방 경기 대회</a>에서 2연승을 하였다. <a href="/wiki/1984%EB%85%84" title="1984년">1984년</a>부터 <a href="/wiki/1986%EB%85%84" title="1986년">1986년</a>까지 세계에서 랭킹 2위였으며, 또한 <a href="/wiki/1981%EB%85%84" title="1981년">1981년</a>부터 <a href="/wiki/1988%EB%85%84" title="1988년">1988년</a>까지 세계 톱 6에 놓였다.</p>
<p>그는 10회의 국내 챔피언, 11회의 주립 챔피언이었으며, <a class="mw-redirect" href="/wiki/1988%EB%85%84_%EC%84%9C%EC%9A%B8_%EC%98%AC%EB%A6%BC%ED%94%BD" title="1988년 서울 올림픽">1988년 서울 올림픽</a>에서 팀 주장을 맡았다.</p>
<p><a href="/wiki/2000%EB%85%84" title="2000년">2000년</a> 스포츠 오스트레일리아 명예의 전당에 헌액되었다.</p>
