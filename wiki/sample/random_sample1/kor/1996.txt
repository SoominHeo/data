<h1 class="firstHeading" id="firstHeading" lang="ko">게이머게이트 논쟁</h1>
<div class="mw-body-content" id="bodyContent">
<div id="siteSub">위키백과, 우리 모두의 백과사전.</div>
<div id="contentSub"></div>
<div class="mw-jump" id="jump-to-nav">
					이동:					<a href="#mw-head">둘러보기</a>, 					<a href="#p-search">검색</a>
</div>
<div class="mw-content-ltr" dir="ltr" id="mw-content-text" lang="ko"><script>function mfTempOpenSection(id){var block=document.getElementById("mf-section-"+id);block.className+=" open-block";block.previousSibling.className+=" open-block";}</script><p><b>게이머게이트 논쟁</b>(Gamergate)은 <b>퀸스피러시</b>(Quinnspiracy)라고도 하며, <a class="new" href="/w/index.php?title=%EB%B9%84%EB%94%94%EC%98%A4_%EA%B2%8C%EC%9E%84%EC%9D%98_%EC%84%B1%EC%B0%A8%EB%B3%84%EC%A3%BC%EC%9D%98&amp;action=edit&amp;redlink=1" title="비디오 게임의 성차별주의 (없는 문서)">비디오 게임 문화의 성차별주의</a>에 관한 논쟁이다. 2014년 8월부터 시작된 이 논쟁은 비디오 게임 산업 내 여성 다수의 <a href="/wiki/%EC%84%B1_%EC%B0%A8%EB%B3%84" title="성 차별">성 차별</a> 및 <a href="/wiki/%EC%97%AC%EC%84%B1%ED%98%90%EC%98%A4" title="여성혐오">여성혐오</a> 공격이 이루어졌다고 주장한 인디 게임 개발자 <a class="new" href="/w/index.php?title=%EC%A1%B0%EC%9D%B4_%ED%80%B8&amp;action=edit&amp;redlink=1" title="조이 퀸 (없는 문서)">조이 퀸</a>에 의해 촉발되었다. 이 사태의 주요 대상은 여성 게임 개발자인 <a class="new" href="/w/index.php?title=%EC%A1%B0%EC%9D%B4_%ED%80%B8&amp;action=edit&amp;redlink=1" title="조이 퀸 (없는 문서)">조이 퀸</a>, <a class="new" href="/w/index.php?title=%EB%B8%8C%EB%A6%AC%EC%95%84%EB%82%98_%EC%9C%84&amp;action=edit&amp;redlink=1" title="브리아나 위 (없는 문서)">브리아나 위</a>, 문화비평가 <a class="new" href="/w/index.php?title=%EC%95%84%EB%8B%88%ED%83%80_%EC%82%AC%EB%A5%B4%EC%BC%80%EC%8B%9C%EC%95%88&amp;action=edit&amp;redlink=1" title="아니타 사르케시안 (없는 문서)">아니타 사르케시안</a> 등이 있다. 이러한 논쟁은 주로 트위터의 해시태그 #gamergate로 이루어졌으며, <a href="/wiki/%EB%A0%88%EB%94%A7" title="레딧">레딧</a>, <a href="/wiki/4chan" title="4chan">4chan</a>, <a class="new" href="/w/index.php?title=8chan&amp;action=edit&amp;redlink=1" title="8chan (없는 문서)">8chan</a>과 같은 온라인 포럼에서 이러한 논쟁이 불붙었다. 이러한 공격에는 공격 대상의 신상 정보 개시(<a href="/wiki/%EC%8B%A0%EC%83%81_%ED%84%B8%EA%B8%B0" title="신상 털기">신상 털기</a>), 대중 앞에서의 총기 난사와 같은 강간 및 살해 위협 등이 있었다. 이 논쟁은 우두머리 없이 모인 모임의 해시태그를 따라서 <b>게이머게이트 운동</b>(Gamergate movement)으로도 알려지게 되었다.</p>
<p>게이머게이트에서는 <a class="new" href="/w/index.php?title=%EB%B9%84%EB%94%94%EC%98%A4_%EA%B2%8C%EC%9E%84_%EB%AC%B8%ED%99%94&amp;action=edit&amp;redlink=1" title="비디오 게임 문화 (없는 문서)">비디오 게임 문화</a>의 다양화를 넘은, <a href="/wiki/%EC%98%88%EC%88%A0%EB%A1%9C%EC%84%9C%EC%9D%98_%EB%B9%84%EB%94%94%EC%98%A4_%EA%B2%8C%EC%9E%84" title="예술로서의 비디오 게임">예술 게임</a>과 비디오 게임의 <a class="new" href="/w/index.php?title=%EC%82%AC%ED%9A%8C_%EB%B9%84%ED%8C%90&amp;action=edit&amp;redlink=1" title="사회 비판 (없는 문서)">사회 비판</a>, <a href="/wiki/%EA%B2%8C%EC%9D%B4%EB%A8%B8" title="게이머">게이머</a>의 <a class="new" href="/w/index.php?title=%EC%82%AC%ED%9A%8C_%EC%A0%95%EC%B2%B4%EC%84%B1_%EC%9D%B4%EB%A1%A0&amp;action=edit&amp;redlink=1" title="사회 정체성 이론 (없는 문서)">사회적 정체성</a> 논쟁 등의 <a class="new" href="/w/index.php?title=%EB%AC%B8%ED%99%94_%EC%A0%84%EC%9F%81&amp;action=edit&amp;redlink=1" title="문화 전쟁 (없는 문서)">문화 전쟁</a>인 광범위한 시각이 나타나 있다. 해시태그를 사용하는 일부 사람들은 <a class="new" href="/w/index.php?title=%EB%B9%84%EB%94%94%EC%98%A4_%EA%B2%8C%EC%9E%84_%EC%A0%80%EB%84%90%EB%A6%AC%EC%A6%98&amp;action=edit&amp;redlink=1" title="비디오 게임 저널리즘 (없는 문서)">비디오 게임 저널리즘</a>의 <a class="new" href="/w/index.php?title=%EB%AF%B8%EB%94%94%EC%96%B4_%EC%9C%A4%EB%A6%AC&amp;action=edit&amp;redlink=1" title="미디어 윤리 (없는 문서)">미디어 윤리</a> 개선을 목표로 하고 있다고 말하며, 비디오 게임 리뷰에서 사회적 이슈에 관한 것이 <a href="/wiki/%EC%97%AC%EC%84%B1%EC%A3%BC%EC%9D%98" title="여성주의">여성주의</a>나 <a href="/wiki/%EC%82%AC%ED%9A%8C%EC%A0%95%EC%9D%98" title="사회정의">사회정의</a>와 같이 특히 이데올로기적 반대자들의 비윤리적 음모로 이러한 사태가 나타났다고 말한다. <a class="new" href="/w/index.php?title=%EC%BB%AC%EB%9F%BC%EB%B9%84%EC%95%84_%EC%A0%80%EB%84%90%EB%A6%AC%EC%A6%98_%EB%A6%AC%EB%B7%B0&amp;action=edit&amp;redlink=1" title="컬럼비아 저널리즘 리뷰 (없는 문서)">컬럼비아 저널리즘 리뷰</a>, <a href="/wiki/%EA%B0%80%EB%94%94%EC%96%B8" title="가디언">가디언</a>, <a class="new" href="/w/index.php?title=%EB%8D%94_%EC%9C%84%ED%81%AC&amp;action=edit&amp;redlink=1" title="더 위크 (없는 문서)">더 위크</a>, <a class="new" href="/w/index.php?title=%EB%B3%B5%EC%8A%A4_(%EC%9B%B9%EC%82%AC%EC%9D%B4%ED%8A%B8)&amp;action=edit&amp;redlink=1" title="복스 (웹사이트) (없는 문서)">복스</a>, <a href="/wiki/%EB%82%B4%EC%85%94%EB%84%90_%ED%8D%BC%EB%B8%94%EB%A6%AD_%EB%9D%BC%EB%94%94%EC%98%A4" title="내셔널 퍼블릭 라디오">내셔널 퍼블릭 라디오</a>의 <a class="new" href="/w/index.php?title=%EC%98%A8_%EB%8D%94_%EB%AF%B8%EB%94%94%EC%96%B4&amp;action=edit&amp;redlink=1" title="온 더 미디어 (없는 문서)">온 더 미디어</a>, <a class="mw-redirect" href="/wiki/%EC%99%80%EC%9D%B4%EC%96%B4%EB%93%9C_(%EC%9E%A1%EC%A7%80)" title="와이어드 (잡지)">와이어드</a>, <a class="new" href="/w/index.php?title=%EB%8D%B0%EB%A5%B4_%EB%B6%84%EB%93%9C&amp;action=edit&amp;redlink=1" title="데르 분드 (없는 문서)">데르 분드</a>, <a class="new" href="/w/index.php?title=%EC%9D%B8%EC%82%AC%EC%9D%B4%EB%93%9C_%ED%95%98%EC%9D%B4%EC%96%B4_%EC%97%90%EB%93%9C&amp;action=edit&amp;redlink=1" title="인사이드 하이어 에드 (없는 문서)">인사이드 하이어 에드</a> 등에서는 게이머게이트가 광범위한 정체를 폭로했다는 윤리 문제에 관해 사실에 근거하지 않거나 업계에서의 실제 윤리 문제와 상관없는 음모론을 기반에 둔 것이나 사소한 것이라고 말했다.</p>

<p>2013년 2월, 인디 게임 개발자인 <a class="new" href="/w/index.php?title=%EC%A1%B0%EC%9D%B4_%ED%80%B8&amp;action=edit&amp;redlink=1" title="조이 퀸 (없는 문서)">조이 퀸</a>이 만든 <a class="new" href="/w/index.php?title=%EB%94%94%ED%94%84%EB%A0%88%EC%85%98_%ED%80%98%EC%8A%A4%ED%8A%B8&amp;action=edit&amp;redlink=1" title="디프레션 퀘스트 (없는 문서)">디프레션 퀘스트</a>가 자신의 웹사이트에서 <a href="/wiki/%EC%9D%B8%ED%84%B0%EB%9E%99%ED%8B%B0%EB%B8%8C_%ED%94%BD%EC%85%98" title="인터랙티브 픽션">인터랙티브 픽션</a> <a href="/wiki/%EC%9B%B9_%EA%B2%8C%EC%9E%84" title="웹 게임">웹 게임</a> 형태로 출시되었다. 이 게임은 비평가들에게 매우 좋은 평가를 받았지만, <a href="/wiki/%EC%8A%A4%ED%8C%80_(%ED%94%8C%EB%9E%AB%ED%8F%BC)" title="스팀 (플랫폼)">스팀</a> 플랫폼에 발매를 시작한 이후에는 우울증의 무기력함을 표현하기 위한위해 선택지에다 취소선을 치는 행위를 비판하는 게이머들을 여성혐오로 몰며 비판글을 삭제하는 행위를 보여 일부 게이머 사이에서 반발이 생기기 시작했다. 퀸은 스팀 사용자 커뮤니티 일부에서 게임 출시와 비판에 관한 <a class="new" href="/w/index.php?title=%EC%A6%9D%EC%98%A4_%EB%A9%94%EC%9D%BC&amp;action=edit&amp;redlink=1" title="증오 메일 (없는 문서)">증오 메일</a>을 받게 되었고, 이후 퀸은 전화번호를 바꾸게 되었다.<sup class="reference" id="cite_ref-pcgamerdq_1-0"><a href="#cite_note-pcgamerdq-1">[1]</a></sup><sup class="reference" id="cite_ref-csmonitor_2-0"><a href="#cite_note-csmonitor-2">[2]</a></sup><sup class="reference" id="cite_ref-TelegraphDec3_3-0"><a href="#cite_note-TelegraphDec3-3">[3]</a></sup><sup class="reference" id="cite_ref-KernelMag_4-0"><a href="#cite_note-KernelMag-4">[4]</a></sup><sup class="reference" id="cite_ref-dot_5-0"><a href="#cite_note-dot-5">[5]</a></sup> 퀸은 이러한 면에서 비판을 받았으며 주변에서 그녀 생활에 대한 위협이 있었지만, 그녀는 대부분 그걸 무시할 수 있었다"라고 말했다.<sup class="reference" id="cite_ref-new_yorker_6-0"><a href="#cite_note-new_yorker-6">[6]</a></sup></p>
