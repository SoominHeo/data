<p><b>게이세이 전철 히가시나리타 선</b>(<span style="font-size: smaller;"><a href="/wiki/%EC%9D%BC%EB%B3%B8%EC%96%B4" title="일본어">일본어</a>:</span> <span lang="ja" xml:lang="ja">京成東成田線</span>)은 <a href="/wiki/%EA%B2%8C%EC%9D%B4%EC%84%B8%EC%9D%B4_%EC%A0%84%EC%B2%A0" title="게이세이 전철">게이세이 전철</a>의 철도 노선 가운데 하나이다. 일본 <a href="/wiki/%EC%A7%80%EB%B0%94_%ED%98%84" title="지바 현">지바 현</a> <a href="/wiki/%EB%82%98%EB%A6%AC%ED%83%80_%EC%8B%9C" title="나리타 시">나리타 시</a>에 위치하는 <a class="mw-redirect" href="/wiki/%EA%B2%8C%EC%9D%B4%EC%84%B8%EC%9D%B4_%EB%82%98%EB%A6%AC%ED%83%80_%EC%97%AD" title="게이세이 나리타 역">게이세이 나리타 역</a>과 <a href="/wiki/%ED%9E%88%EA%B0%80%EC%8B%9C%EB%82%98%EB%A6%AC%ED%83%80_%EC%97%AD" title="히가시나리타 역">히가시나리타 역</a>을 잇는다.</p>
<p>게이세이 나리타 - <a class="new" href="/w/index.php?title=%EA%B3%A0%EB%A7%88%EC%9D%B4%EB%85%B8_%EC%8B%A0%ED%98%B8%EC%9E%A5&amp;action=edit&amp;redlink=1" title="고마이노 신호장 (없는 문서)">고마이노 신호장</a> 구간은 <a class="mw-redirect" href="/wiki/%EA%B2%8C%EC%9D%B4%EC%84%B8%EC%9D%B4_%EC%A0%84%EC%B2%A0_%EB%B3%B8%EC%84%A0" title="게이세이 전철 본선">게이세이 전철 본선</a>과 중복된다.</p>

<h2><span class="mw-headline" id=".EC.97.AD.EC.82.AC">역사</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EA%B2%8C%EC%9D%B4%EC%84%B8%EC%9D%B4_%ED%9E%88%EA%B0%80%EC%8B%9C%EB%82%98%EB%A6%AC%ED%83%80_%EC%84%A0&amp;action=edit&amp;section=1" title="부분 편집: 역사">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<ul>
<li><a href="/wiki/1978%EB%85%84" title="1978년">1978년</a> <a href="/wiki/5%EC%9B%94_21%EC%9D%BC" title="5월 21일">5월 21일</a>: <a class="mw-redirect" href="/wiki/%EA%B2%8C%EC%9D%B4%EC%84%B8%EC%9D%B4_%EC%A0%84%EC%B2%A0_%EB%B3%B8%EC%84%A0" title="게이세이 전철 본선">본선</a>으로서 게이세이 나리타 - 나리타 공항(현재: 히가시나리타) 구간이 개통.</li>
<li><a href="/wiki/1991%EB%85%84" title="1991년">1991년</a> <a href="/wiki/3%EC%9B%94_19%EC%9D%BC" title="3월 19일">3월 19일</a>:
<ul>
<li>나리타 공항역을 히가시나리타 역으로 명칭 변경.</li>
<li><a href="/wiki/%EB%82%98%EB%A6%AC%ED%83%80_%EA%B3%B5%ED%95%AD%EC%97%AD" title="나리타 공항역">나리타 공항역</a> 영업 개시에 따라 (게이세이 나리타 역 ~) 고마노이 신호장 ~ 히가시나리타 역을 <b>히가시나리타 선</b>으로 분리.</li>
</ul>
</li>
</ul>
<h2><span class="mw-headline" id=".EC.9A.B4.ED.96.89_.ED.98.95.ED.83.9C">운행 형태</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EA%B2%8C%EC%9D%B4%EC%84%B8%EC%9D%B4_%ED%9E%88%EA%B0%80%EC%8B%9C%EB%82%98%EB%A6%AC%ED%83%80_%EC%84%A0&amp;action=edit&amp;section=2" title="부분 편집: 운행 형태">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>기본적으로는 게이세이 나리타 역과 <a class="mw-redirect" href="/wiki/%EC%8B%9C%EB%B0%94%EC%95%BC%EB%A7%88_%EC%B2%A0%EB%8F%84_%EC%8B%9C%EB%B0%94%EC%95%BC%EB%A7%88_%EC%B2%A0%EB%8F%84%EC%84%A0" title="시바야마 철도 시바야마 철도선">시바야마 철도 시바야마 철도선</a> <a href="/wiki/%EC%8B%9C%EB%B0%94%EC%95%BC%EB%A7%88%EC%B9%98%EC%9A%94%EB%8B%A4_%EC%97%AD" title="시바야마치요다 역">시바야마치요다 역</a>을 완복 운행한다.</p>
<p>아침이나 저녁 시는 게이세이 전철 본선을 거쳐 <a class="mw-redirect" href="/wiki/%EA%B2%8C%EC%9D%B4%EC%84%B8%EC%9D%B4_%EC%9A%B0%EC%97%90%EB%85%B8_%EC%97%AD" title="게이세이 우에노 역">게이세이 우에노</a>, <a href="/wiki/%EB%8B%88%EC%8B%9C%EB%A7%88%EA%B3%A0%EB%A9%94_%EC%97%AD" title="니시마고메 역">니시마고메</a>, <a class="mw-redirect" href="/wiki/%ED%95%98%EB%84%A4%EB%8B%A4_%EA%B3%B5%ED%95%AD%EC%97%AD" title="하네다 공항역">하네다 공항</a>까지 <a href="/wiki/%EC%A7%81%EA%B2%B0_%EC%9A%B4%ED%96%89" title="직결 운행">직결 운행</a>을 한다.</p>
