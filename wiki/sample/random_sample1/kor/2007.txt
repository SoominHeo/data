<p><b>게잡이라쿤</b>(<i>Procyon cancrivorus</i>)은 <a href="/wiki/%EC%A4%91%EC%95%99%EC%95%84%EB%A9%94%EB%A6%AC%EC%B9%B4" title="중앙아메리카">중앙아메리카</a>와 <a href="/wiki/%EB%82%A8%EC%95%84%EB%A9%94%EB%A6%AC%EC%B9%B4" title="남아메리카">남아메리카</a>(<a href="/wiki/%ED%8A%B8%EB%A6%AC%EB%8B%88%EB%8B%A4%EB%93%9C_%ED%86%A0%EB%B0%94%EA%B3%A0" title="트리니다드 토바고">트리니다드 토바고</a> 포함)의 습지와 정글 지역에 사는 <a href="/wiki/%EC%95%84%EB%A9%94%EB%A6%AC%EC%B9%B4%EB%84%88%EA%B5%AC%EB%A6%AC%EC%86%8D" title="아메리카너구리속">라쿤</a>의 일종이다.</p>
<p><a href="/wiki/%EC%BD%94%EC%8A%A4%ED%83%80%EB%A6%AC%EC%B9%B4" title="코스타리카">코스타리카</a>부터 남쪽으로 <a href="/wiki/%EC%95%88%EB%8D%B0%EC%8A%A4_%EC%82%B0%EB%A7%A5" title="안데스 산맥">안데스 산맥</a> 동부의 <a href="/wiki/%EC%95%84%EB%A5%B4%ED%97%A8%ED%8B%B0%EB%82%98" title="아르헨티나">아르헨티나</a> 북부와 <a href="/wiki/%EC%9A%B0%EB%A3%A8%EA%B3%BC%EC%9D%B4" title="우루과이">우루과이</a>에 이르는 대부분의 남아메리카 지역에서 발견된다.<sup class="reference" id="cite_ref-iucn_1-1"><a href="#cite_note-iucn-1">[1]</a></sup> 게잡이라쿤으로 불리지만, 이 종만이 게를 먹는 것은 아니고 라쿤(아메리카너구리) 또한 게를 먹는다.</p>
<p>게잡이라쿤은 <a href="/wiki/%EA%B2%8C" title="게">게</a>와 <a href="/wiki/%EB%B0%94%EB%8B%B7%EA%B0%80%EC%9E%AC" title="바닷가재">바닷가재</a> 그리고 기타 <a href="/wiki/%EA%B0%91%EA%B0%81%EB%A5%98" title="갑각류">갑각류</a> 등을 먹이로 먹지만, 잡식성 동물로 예를 들어 작은 <a href="/wiki/%EC%96%91%EC%84%9C%EB%A5%98" title="양서류">양서류</a>와 거북의 알, 식물 열매 등도 먹는다. 북부 지역에 서식하는 사촌 종이며, 털이 더부룩한 반지 모양 꼬리와 눈 둘레에 "산적 가면" 모양의 털을 가진 <a href="/wiki/%EC%95%84%EB%A9%94%EB%A6%AC%EC%B9%B4%EB%84%88%EA%B5%AC%EB%A6%AC" title="아메리카너구리">라쿤</a>을 닮았다. 아메리카너구리와는 달리, 목덜미의 털이 뒷쪽으로 향하지 않고 머리 쪽 방향으로 나 있다.<sup class="reference" id="cite_ref-natrlhis_2-0"><a href="#cite_note-natrlhis-2">[2]</a></sup></p>
