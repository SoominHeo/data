<h2><span class="mw-headline" id=".EC.83.9D.EC.95.A0">생애</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EA%B2%90%EB%87%A8&amp;action=edit&amp;section=1" title="부분 편집: 생애">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<h3><span class="mw-headline" id=".ED.83.84.EC.83.9D">탄생</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EA%B2%90%EB%87%A8&amp;action=edit&amp;section=2" title="부분 편집: 탄생">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p><a href="/wiki/%EB%8D%B4%EB%B6%84" title="덴분">덴분</a> 12년 1월 7일(<a href="/wiki/1543%EB%85%84" title="1543년">1543년</a> 2월 20일), 혼간지 제10세 종주 <a class="new" href="/w/index.php?title=%EC%87%BC%EB%87%A8&amp;action=edit&amp;redlink=1" title="쇼뇨 (없는 문서)">쇼뇨</a>(証如)의 장남으로 태어났다.</p>
<h3><span class="mw-headline" id=".EA.B5.90.EB.8B.A8.EC.9D.98_.EC.A0.84.EC.84.B1.EA.B8.B0">교단의 전성기</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EA%B2%90%EB%87%A8&amp;action=edit&amp;section=3" title="부분 편집: 교단의 전성기">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p><a class="mw-redirect" href="/wiki/%EA%B3%A0%EC%A7%80_(1555%EB%85%84)" title="고지 (1555년)">고지</a> 3년(<a href="/wiki/1557%EB%85%84" title="1557년">1557년</a>) 4월 17일, <a href="/wiki/%EB%A1%AF%EC%B9%B4%EC%BF%A0_%EC%82%AC%EB%8B%A4%EC%9A%94%EB%A6%AC" title="롯카쿠 사다요리">롯카쿠 사다요리</a>(六角定頼)의 양녀 뇨슌니(如春尼)와 결혼하였다. 뇨슌니의 아버지는 <a class="mw-redirect" href="/wiki/%EA%B5%AC%EA%B2%8C" title="구게">구게</a> <a class="new" href="/w/index.php?title=%EC%82%B0%EC%A1%B0_%EA%B8%B4%EC%9A%94%EB%A6%AC&amp;action=edit&amp;redlink=1" title="산조 긴요리 (없는 문서)">산조 긴요리</a>(三条公頼)지만, <a class="mw-redirect" href="/wiki/%EA%B0%84%EB%A0%88%EC%9D%B4" title="간레이">간레이</a> <a href="/wiki/%ED%98%B8%EC%86%8C%EC%B9%B4%EC%99%80_%ED%95%98%EB%A3%A8%EB%AA%A8%ED%86%A0" title="호소카와 하루모토">호소카와 하루모토</a>(細川晴元)의 양녀가 되었다가 이 결혼 즈음에는 롯카쿠 사다요리의 양녀로 들어가 있었다. 덧붙여 뇨슌니의 언니는 <a href="/wiki/%EB%8B%A4%EC%BC%80%EB%8B%A4_%EC%8B%A0%EA%B2%90" title="다케다 신겐">다케다 신겐</a>의 정실 산조부인(三条夫人)이다. 정략결혼<sup class="reference" id="cite_ref-1"><a href="#cite_note-1">[1]</a></sup>이라고는 하나, 부부 사이는 좋았다고 한다.</p>
<p>겐뇨 시대에 혼간지 교단은 부친 쇼뇨 시대부터 진행하고 있던 문도의 <a class="mw-redirect" href="/wiki/%EC%9E%87%EC%BD%94_%EC%9E%87%ED%82%A4" title="잇코 잇키">잇코 잇키</a>를 장악하고자 힘쓰는 한편 <a class="mw-redirect" href="/wiki/%EA%B0%84%EB%A0%88%EC%9D%B4" title="간레이">간레이</a>인 <a class="new" href="/w/index.php?title=%ED%98%B8%EC%86%8C%EC%B9%B4%EC%99%80_%EA%B0%80%EB%AC%B8&amp;action=edit&amp;redlink=1" title="호소카와 가문 (없는 문서)">호소카와 가문</a>과 <a href="/wiki/%EA%B5%90%ED%86%A0" title="교토">교토</a>의 <a class="mw-redirect" href="/wiki/%EA%B5%AC%EA%B2%8C" title="구게">구게</a> 들과의 인척관계를 강화하였다. 또한, 경제적·군사적 요충지인 <a href="/wiki/%EC%9D%B4%EC%8B%9C%EC%95%BC%EB%A7%88_%ED%98%BC%EA%B0%84%EC%A7%80" title="이시야마 혼간지">이시야마 혼간지</a>를 거점으로 삼아서, 주로 <a href="/wiki/%EA%B8%B0%EB%82%98%EC%9D%B4" title="기나이">기나이</a>를 중심으로 혼간지 파의 사원을 배치하여 <a href="/wiki/%EB%8B%A4%EC%9D%B4%EB%AC%98" title="다이묘">다이묘</a>에 필적하는 세력을 자랑하며 교단의 전성기를 맞이하였다.</p>
<h3><span class="mw-headline" id=".EB.85.B8.EB.B6.80.EB.82.98.EA.B0.80_.ED.8F.AC.EC.9C.84.EB.A7.9D">노부나가 포위망</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EA%B2%90%EB%87%A8&amp;action=edit&amp;section=4" title="부분 편집: 노부나가 포위망">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>그러나, 혼간지는 무가(武家)의 봉건관계 밖의 권력을 장악하였기 때문에 <a href="/wiki/%ED%9E%88%EC%97%90%EC%9D%B4_%EC%82%B0" title="히에이 산">히에이 산</a> <a href="/wiki/%EC%97%94%EB%9E%B4%EC%BF%A0%EC%A7%80" title="엔랴쿠지">엔랴쿠지</a>와 <a class="mw-redirect" href="/wiki/%EC%82%AC%EC%B9%B4%EC%9D%B4_%EC%8B%9C" title="사카이 시">사카이</a>의 상인들과 마찬가지로, <a href="/wiki/%EC%97%90%EC%9D%B4%EB%A1%9C%EC%BF%A0" title="에이로쿠">에이로쿠</a> 11년(<a href="/wiki/1568%EB%85%84" title="1568년">1568년</a>)에 상락하여 기나이 장악을 꾀하던 <a href="/wiki/%EC%98%A4%EB%8B%A4_%EB%85%B8%EB%B6%80%EB%82%98%EA%B0%80" title="오다 노부나가">오다 노부나가</a>의 압박을 받게 되었다. 이윽고 겐뇨는 노부나가와 싸울 것을 결의하고, <a href="/wiki/%EA%B2%90%ED%82%A4" title="겐키">겐키</a> 원년(<a href="/wiki/1570%EB%85%84" title="1570년">1570년</a>)부터 혼간지와 오다 가문은 교전 상태에 들어갔다(<a class="new" href="/w/index.php?title=%EB%85%B8%EB%8B%A4_%EC%84%B1%C2%B7%ED%9B%84%EC%BF%A0%EC%8B%9C%EB%A7%88_%EC%84%B1_%EC%A0%84%ED%88%AC&amp;action=edit&amp;redlink=1" title="노다 성·후쿠시마 성 전투 (없는 문서)">노다 성·후쿠시마 성 전투</a>(野田城・福島城の戦い)). 이 전투를 <a href="/wiki/%EC%9D%B4%EC%8B%9C%EC%95%BC%EB%A7%88_%EC%A0%84%ED%88%AC" title="이시야마 전투">이시야마 전투</a>(石山合戦)라고 한다.</p>
<p>겐뇨는 노부나가와 대립하고 있던 <a class="mw-redirect" href="/wiki/%EC%84%B8%EC%9D%B4%EC%9D%B4%ED%83%80%EC%9D%B4%EC%87%BC%EA%B5%B0" title="세이이타이쇼군">쇼군</a> <a href="/wiki/%EC%95%84%EC%8B%9C%EC%B9%B4%EA%B0%80_%EC%9A%94%EC%8B%9C%EC%95%84%ED%82%A4" title="아시카가 요시아키">아시카가 요시아키</a>(足利義昭)와 연합하여, 다케다(武田)·아사쿠라(朝倉)·아자이(浅井)·모리(毛利) 등의 반 오다 세력과 동맹을 맺어 <a href="/wiki/%EB%85%B8%EB%B6%80%EB%82%98%EA%B0%80_%ED%8F%AC%EC%9C%84%EB%A7%9D" title="노부나가 포위망">노부나가 포위망</a>을 구축하는 한편, 스스로는 이시야마 혼간지에서 농성하며 <a class="mw-redirect" href="/wiki/%EC%82%AC%EC%9D%B4%EC%B9%B4%EC%8A%88" title="사이카슈">사이카슈</a> 등의 우호적인 토호 세력과 지방의 문도 조직을 동원하여 노부나가를 괴롭혔다. 그러나, 장기에 걸친 포위전으로 점차 영지가 피폐해졌으며, 인척관계인 <a href="/wiki/%EB%8B%A4%EC%BC%80%EB%8B%A4_%EC%8B%A0%EA%B2%90" title="다케다 신겐">다케다 신겐</a>의 지원을 기대하였으나 신겐이 상락 도중 급사하여 지원을 기대할 수 없게 된데다, 더욱이 <a href="/wiki/%EC%95%84%EC%82%AC%EC%BF%A0%EB%9D%BC_%EC%9A%94%EC%8B%9C%EC%B9%B4%EA%B2%8C" title="아사쿠라 요시카게">아사쿠라 요시카게</a>·<a href="/wiki/%EC%95%84%EC%9E%90%EC%9D%B4_%EB%82%98%EA%B0%80%EB%A7%88%EC%82%AC" title="아자이 나가마사">아자이 나가마사</a> 등의 동맹세력이 차례차례 오다 가문에게 멸망당하는 등 악재가 겹치자 겐뇨는 항전을 계속하기가 곤란하다는 것을 깨닫고 조정에 화평의 중개역을 의뢰하였다. 겐뇨는 덴쇼 8년(<a href="/wiki/1580%EB%85%84" title="1580년">1580년</a>)에 노부나가가 제시한 화평 조건을 받아들여 이시야마 혼간지에서 철수하고 <a href="/wiki/%EA%B8%B0%EC%9D%B4_%EA%B5%AD" title="기이 국">기이 국</a> <a class="new" href="/w/index.php?title=%EC%82%AC%EA%B8%B0%EB%85%B8%EB%AA%A8%EB%A6%AC_%EB%B3%84%EC%9B%90&amp;action=edit&amp;redlink=1" title="사기노모리 별원 (없는 문서)">사기노모리 별원</a>(鷺森別院)으로 옮겨 갔다.</p>
<h3><span class="mw-headline" id=".EB.A7.8C.EB.85.84">만년</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EA%B2%90%EB%87%A8&amp;action=edit&amp;section=5" title="부분 편집: 만년">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p><a href="/wiki/%ED%98%BC%EB%85%B8%EC%A7%80%EC%9D%98_%EB%B3%80" title="혼노지의 변">혼노지의 변</a> 뒤 노부나가의 뒤를 이어 <a href="/wiki/%EA%B8%B0%EB%82%98%EC%9D%B4" title="기나이">기나이</a>의 실권을 장악한 <a href="/wiki/%EB%8F%84%EC%9A%94%ED%86%A0%EB%AF%B8_%ED%9E%88%EB%8D%B0%EC%9A%94%EC%8B%9C" title="도요토미 히데요시">하시바 히데요시</a>와 화해하여, 덴쇼 13년(<a href="/wiki/1585%EB%85%84" title="1585년">1585년</a>)에 히데요시의 도움으로 오사카 교외에 덴만 혼간지(天満本願寺)를 건립하였다. 이것은 <a href="/wiki/%EB%8F%84%EC%9A%94%ED%86%A0%EB%AF%B8_%EC%A0%95%EA%B6%8C" title="도요토미 정권">도요토미 정권</a>의 본거지인 오사카로 혼간지를 옮기면 감시하기 편리하며, <a href="/wiki/%EC%98%A4%EC%82%AC%EC%B9%B4_%EC%84%B1" title="오사카 성">오사카 성</a>의 <a href="/wiki/%EC%A1%B0%EC%B9%B4%EB%A7%88%EC%B9%98" title="조카마치">조카마치</a>를 건설하는데 혼간지와 그 문도의 경제력·기술력을 사용할 수 있는 이점이 있었기 때문이다. 이후 혼간지는 도요토미 정권의 영향권 아래에 들어가게 되었다. 덴쇼 17년(<a href="/wiki/1589%EB%85%84" title="1589년">1589년</a>)에 <a href="/wiki/%EA%B5%90%ED%86%A0" title="교토">교토</a> <a class="mw-redirect" href="/wiki/%EC%A3%BC%EB%9D%BC%EC%BF%A0_%EB%8B%A4%EC%9D%B4" title="주라쿠 다이">주라쿠 다이</a>의 벽에 낙서를 한 범인이 혼간지 관할의 마을로 도망쳐 들어가고, 히데요시에게 추방당한 호소카와 아키모토(細川昭元)·비토 도모노부(尾藤知宣) 등을 덴만 혼간지에서 숨겨주고 있다는 정보를 입수한 도요토미 정권은 그해 3월에 중신 <a href="/wiki/%EC%9D%B4%EC%8B%9C%EB%8B%A4_%EB%AF%B8%EC%93%B0%EB%82%98%EB%A6%AC" title="이시다 미쓰나리">이시다 미쓰나리</a>를 파견하여 용의자를 숨겨준 2개 마을을 파괴하였다. 호소카와 등은 체포당하지 않았으나, 숨겨준 마을 사람 모두가 교토 <a class="new" href="/w/index.php?title=%EB%A1%9C%EC%BF%A0%EC%A1%B0%ED%95%98%EB%9D%BC&amp;action=edit&amp;redlink=1" title="로쿠조하라 (없는 문서)">로쿠조하라</a>에서 책형을 당했으며 겐뇨 본인도 히데요시에게 질책을 당하는 등 그나마 가지고 있던 영주 권력도 완전히 잃게 되었다. 덴쇼 19년(<a href="/wiki/1591%EB%85%84" title="1591년">1591년</a>)에는 히데요시로부터 교토에 사원 부지를 받아서 교토에 혼간지 교단을 재흥하였다.</p>
<p><a href="/wiki/%EB%B6%84%EB%A1%9C%EC%BF%A0" title="분로쿠">분로쿠</a> 원년 11월 24일(1592년 12월 17일), 향년 50세로 입적하였다.</p>
<p>겐뇨가 사망하자, 이시야마 혼간지 퇴거 시에 노부나가에 대한 대응을 둘러싸고 겐뇨와 의견 충돌로 대립하였던 강경파 장남 <a class="mw-redirect" href="/wiki/%EA%B5%90%EB%87%A8" title="교뇨">교뇨</a>(教如)를 대신하여 화평파였던 삼남 <a class="new" href="/w/index.php?title=%EC%A4%80%EB%87%A8&amp;action=edit&amp;redlink=1" title="준뇨 (없는 문서)">준뇨</a>(准如)가 제12세 종주가 되었다. 그러나 이에 승복하지 않은 교뇨와 준뇨의 대립이 심화되어 결국 혼간지는 준뇨의 서 혼간지와 교뇨의 동 혼간지로 분열하게 되었다.</p>
