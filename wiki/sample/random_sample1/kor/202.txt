<p><b>.kp</b>는 <a href="/wiki/%EC%A1%B0%EC%84%A0%EB%AF%BC%EC%A3%BC%EC%A3%BC%EC%9D%98%EC%9D%B8%EB%AF%BC%EA%B3%B5%ED%99%94%EA%B5%AD" title="조선민주주의인민공화국">조선민주주의인민공화국</a>의 <a class="mw-redirect" href="/wiki/%EA%B5%AD%EA%B0%80_%EB%8F%84%EB%A9%94%EC%9D%B8" title="국가 도메인">국가 도메인</a>이다. <a href="/wiki/2007%EB%85%84" title="2007년">2007년</a> <a href="/wiki/9%EC%9B%94_24%EC%9D%BC" title="9월 24일">9월 24일</a>에 생성되었다. 2010년까지만 해도 모든 조선민주주의인민공화국 인터네트워크봉사국에서 .kp대신 국제도메인 .com 또는 일본의 도메인인 .co.jp만을 사용하였지만, 2011년 1월에 홈페이지 <a class="mw-disambig" href="/wiki/%EB%82%B4%EB%82%98%EB%9D%BC" title="내나라">내나라</a>의 주소를 http://www.naenara.com.kp로 변경하면서 본격적으로 .kp 주소를 사용하기 시작하였다.</p>
<h2><span class="mw-headline" id=".EC.A1.B0.EC.84.A0.EB.AF.BC.EC.A3.BC.EC.A3.BC.EC.9D.98.EC.9D.B8.EB.AF.BC.EA.B3.B5.ED.99.94.EA.B5.AD.EC.9D.98_.EC.9D.B8.ED.84.B0.EB.84.B7">조선민주주의인민공화국의 인터넷</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=.kp&amp;action=edit&amp;section=1" title="부분 편집: 조선민주주의인민공화국의 인터넷">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<div class="rellink relarticle mainarticle"><img alt="" data-file-height="128" data-file-width="128" height="16" src="//upload.wikimedia.org/wikipedia/commons/thumb/e/ec/Crystal_Clear_app_xmag.svg/16px-Crystal_Clear_app_xmag.svg.png" srcset="//upload.wikimedia.org/wikipedia/commons/thumb/e/ec/Crystal_Clear_app_xmag.svg/24px-Crystal_Clear_app_xmag.svg.png 1.5x, //upload.wikimedia.org/wikipedia/commons/thumb/e/ec/Crystal_Clear_app_xmag.svg/32px-Crystal_Clear_app_xmag.svg.png 2x" width="16"/> 이 부분의 본문은 <a href="/wiki/%EC%A1%B0%EC%84%A0%EB%AF%BC%EC%A3%BC%EC%A3%BC%EC%9D%98%EC%9D%B8%EB%AF%BC%EA%B3%B5%ED%99%94%EA%B5%AD%EC%9D%98_%EC%9D%B8%ED%84%B0%EB%84%B7" title="조선민주주의인민공화국의 인터넷">조선민주주의인민공화국의 인터넷</a>입니다.</div>
<p>조선민주주의인민공화국에서는 사설 <a href="/wiki/%EC%9D%B8%ED%84%B0%EB%84%B7_%EC%84%9C%EB%B9%84%EC%8A%A4_%EC%A0%9C%EA%B3%B5%EC%9E%90" title="인터넷 서비스 제공자">인터넷 서비스 제공자</a>가 존재하지 않는다.</p>
<p><a href="/wiki/%EA%B9%80%EC%A0%95%EC%9D%BC" title="김정일">김정일</a>에 따르면 "나도 <a href="/wiki/%EC%9D%B8%ED%84%B0%EB%84%B7" title="인터넷">인터넷</a> 전문가다. <a href="/wiki/%EA%B0%9C%EC%84%B1%EA%B3%B5%EC%97%85%EC%A7%80%EA%B5%AC" title="개성공업지구">공단</a> 안에서만 통하면 되는데 북쪽 다른 지역까지 연결돼서는 문제가 많다."라고 언급하였다.<sup class="reference" id="cite_ref-1"><a href="#cite_note-1">[1]</a></sup></p>
<p>인터넷 접속은 제한되어 있고 드문 편이다. <a href="/wiki/%EA%B5%AD%EA%B2%BD_%EC%97%86%EB%8A%94_%EA%B8%B0%EC%9E%90%ED%9A%8C" title="국경 없는 기자회">국경 없는 기자회</a>는 "세계에서 가장 고립된 국가에서 인터넷은 공식적으로 존재하지 않으나 소수의 특권 계급은 (중국망을 통한) <a href="/wiki/%EC%A0%84%ED%99%94" title="전화">전화</a>를 거치거나 <a href="/wiki/%EC%9C%84%EC%84%B1" title="위성">위성</a>을 거쳐 온라인으로 들어가는 것이 허용된다."라면서<sup class="reference" id="cite_ref-2"><a href="#cite_note-2">[2]</a></sup> 한때 이 나라를 "세계 최악의 인터넷 블랙홀"로 묘사하였다<sup class="reference" id="cite_ref-3"><a href="#cite_note-3">[3]</a></sup></p>
<p>정부 소속 사이트는 <i>.kp</i> 도메인에 속해 있지 않고 국외에 호스팅하여 운영하고 있다. korea-dpr.com 주소는 <a href="/wiki/%EC%8A%A4%ED%8E%98%EC%9D%B8" title="스페인">스페인</a> 제공업체 DINAHOSTING에 속해 있고 IP 위치는 <a href="/wiki/%EB%8F%85%EC%9D%BC" title="독일">독일</a>에 있다.<sup class="reference" id="cite_ref-4"><a href="#cite_note-4">[4]</a></sup></p>
