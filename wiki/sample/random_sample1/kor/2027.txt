<p><b>겸지왕</b>(鉗知王, ? ~ <a href="/wiki/521%EB%85%84" title="521년">521년</a> <a href="/wiki/4%EC%9B%94_28%EC%9D%BC" title="4월 28일">4월 28일</a>(<a href="/wiki/%EC%9D%8C%EB%A0%A5_4%EC%9B%94_7%EC%9D%BC" title="음력 4월 7일">음력 4월 7일</a>)) 혹은 <b>김겸왕</b>(金鉗王)은 <a class="mw-redirect" href="/wiki/%EA%B0%80%EB%9D%BD%EA%B5%AD" title="가락국">가락국</a>의 제9대 국왕(<small>재위:</small> <a href="/wiki/491%EB%85%84" title="491년">491년</a> <a href="/wiki/11%EC%9B%94_20%EC%9D%BC" title="11월 20일">11월 20일</a>(<a href="/wiki/%EC%9D%8C%EB%A0%A5_10%EC%9B%94_4%EC%9D%BC" title="음력 10월 4일">음력 10월 4일</a>) ~ <a href="/wiki/521%EB%85%84" title="521년">521년</a> <a href="/wiki/4%EC%9B%94_28%EC%9D%BC" title="4월 28일">4월 28일</a>(<a href="/wiki/%EC%9D%8C%EB%A0%A5_4%EC%9B%94_7%EC%9D%BC" title="음력 4월 7일">음력 4월 7일</a>))이다.</p>
<h2><span class="mw-headline" id=".EC.83.9D.EC.95.A0">생애</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EA%B2%B8%EC%A7%80%EC%99%95&amp;action=edit&amp;section=1" title="부분 편집: 생애">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>재위 <a href="/wiki/492%EB%85%84" title="492년">492년</a>~<a href="/wiki/521%EB%85%84" title="521년">521년</a> 김겸왕(金鉗王)이라고도 한다. 김(金)은 성이고 겸지(鉗知)가 이름이나 지(知)는 <a href="/wiki/%EA%B0%80%EC%95%BC" title="가야">가야</a>와 <a href="/wiki/%EC%8B%A0%EB%9D%BC" title="신라">신라</a>인의 이름에 붙는 어미로서 실제의 이름은 겸이다. 지(知)를 존칭으로 보고, 본래는 “겸왕(鉗王)”으로 불렸을 것으로 추측한 견해가 있다. 아버지는 <a href="/wiki/%EC%A7%88%EC%A7%80%EC%99%95" title="질지왕">질지왕</a>(銍知王)으로 김질왕(金銍王)으로도 불리었으며 어머니는 사간(沙干) 김상(金相)의 딸 방원(邦媛)이다. 왕비는 각간 출충(出忠)의 딸로 숙(淑)이며 아들은 금관가야 마지막 왕인 <a href="/wiki/%EA%B5%AC%ED%98%95%EC%99%95" title="구형왕">구형왕</a>이다. <a href="/wiki/492%EB%85%84" title="492년">492년</a>에 즉위하여 30년간 재위하였다. 일본인 학자들은 <a href="/wiki/%EC%82%BC%EA%B5%AD%EC%9C%A0%EC%82%AC" title="삼국유사">삼국유사</a>에 인용된 가락국기(駕洛國記)의 사료적 신빙성을 부정하고 제7대 <a href="/wiki/%EC%B7%A8%ED%9D%AC%EC%99%95" title="취희왕">취희왕</a>(吹希王) 제8대 <a href="/wiki/%EC%A7%88%EC%A7%80%EC%99%95" title="질지왕">질지왕</a> 제9대 겸지왕을 가야금을 만들었다고 전하는 가실왕(嘉實王) 또는 (嘉悉王)과 모두 같은 인물로 보고 있다. 그러나 그러한 견해는 <a href="/wiki/%EB%8C%80%EA%B0%80%EC%95%BC" title="대가야">대가야</a> 등과 같은 <a href="/wiki/%EA%B8%88%EA%B4%80%EA%B0%80%EC%95%BC" title="금관가야">금관가야</a> 이외의 가야국을 인정하지 않은 견해로서 재고되어야 할 것이다.</p>
