<p><b>경번갑</b> 鏡幡甲 (<b>철판 사슬갑옷</b>, <b>미늘 갑옷</b> 혹은 <b>미늘 사슬 갑옷</b> 영어로는<b>plate and mail</b> 이라고도 불림) 은 철판이 삽입된 <a href="/wiki/%EA%B0%91%EC%98%B7" title="갑옷">갑옷</a>의 일종이다. <a href="/wiki/%EC%B0%B0%EA%B0%91" title="찰갑">찰갑</a>과 <a href="/wiki/%EC%87%84%EC%9E%90%EA%B0%91" title="쇄자갑">쇄자갑</a>의 혼합형이며 쇄자갑의 단점인 타격, 관통성 무기에 약한 점을 보완하기 위하여 만들어졌다. 이런 형식의 갑옷은 <a href="/wiki/%EC%A4%91%EB%8F%99" title="중동">중동</a>, <a href="/wiki/%EC%9D%BC%EB%B3%B8" title="일본">일본</a>, <a href="/wiki/%EC%A4%91%EA%B5%AD" title="중국">중국</a>, <a href="/wiki/%ED%95%9C%EA%B5%AD" title="한국">한국</a>, <a href="/wiki/%EC%A4%91%EC%95%99%EC%95%84%EC%8B%9C%EC%95%84" title="중앙아시아">중앙아시아</a>, <a href="/wiki/%EC%9D%B4%EB%9E%80" title="이란">이란</a> 지방, <a href="/wiki/%EC%9D%B8%EB%8F%84" title="인도">인도</a>, <a href="/wiki/%EB%8F%99%EC%9C%A0%EB%9F%BD" title="동유럽">동유럽</a>, <a class="mw-disambig" href="/wiki/%EB%AC%B4%EC%96%B4" title="무어">무어</a> 등지에서 쓰였다.</p>

<h2><span class="mw-headline" id=".EA.B2.BD.EB.B2.88.EA.B0.91.EC.9D.98_.EC.A2.85.EB.A5.98">경번갑의 종류</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EA%B2%BD%EB%B2%88%EA%B0%91&amp;action=edit&amp;section=1" title="부분 편집: 경번갑의 종류">원본 편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>러시아에는 이 갑옷의 세 가지 변종이 있는 것으로 알려져 있다. 이들은 애초에 <a class="mw-disambig" href="/wiki/%ED%8E%98%EB%A5%B4%EC%8B%9C%EC%95%84" title="페르시아">페르시아</a>에서 수입해 들여왔고, 따라서 페르시아 이름을 가지고 있다.</p>
<ul>
<li>베헤리츠 (Behterets) (<a class="new" href="/w/index.php?title=%ED%8B%80:Lang-ru&amp;action=edit&amp;redlink=1" title="틀:Lang-ru (없는 문서)">틀:Lang-ru</a>), 페르시아어 ''<i>behter</i><sup class="reference" id="cite_ref-ReferenceA_1-0"><a href="#cite_note-ReferenceA-1">[1]</a></sup>에서 왔다. 작은 수평 철판들을 고리로 촘촘히 이어 수직 줄로 만들었고, 사슬 갑옷의 일부로 엮어졌다.</li>
<li>유스만 (<a class="new" href="/w/index.php?title=%ED%8B%80:Lang-ru&amp;action=edit&amp;redlink=1" title="틀:Lang-ru (없는 문서)">틀:Lang-ru</a>), 페르시아어 <i>jawshan</i><sup class="reference" id="cite_ref-ReferenceA_1-1"><a href="#cite_note-ReferenceA-1">[1]</a></sup>에서 왔다. 긴 수평 철판들을 사슬 갑옷에 짜넣었고 <a href="/wiki/%EC%B0%B0%EA%B0%91" title="찰갑">찰갑</a> (예. 로마의 <a href="/wiki/%EB%A1%9C%EB%A6%AC%EC%B9%B4_%EC%84%B8%EA%B7%B8%EB%A9%98%ED%83%80%ED%83%80" title="로리카 세그멘타타">로리카 세그멘타타</a>)과 닮았다</li>
<li>칼란타르 (<a class="new" href="/w/index.php?title=%ED%8B%80:Lang-ru&amp;action=edit&amp;redlink=1" title="틀:Lang-ru (없는 문서)">틀:Lang-ru</a>) 사각 철판을 사슬 갑옷에 엮어 넣었고, 일본 카루타 [[타타미도]와 대단히 비슷하다. 주된 차이점은 <i>칼란타르</i>는 카루타 타타미도와 달리 천 안감이 재봉되어 있지 않다는 것이다.</li>
</ul>
<p><br/>
보브로프에 의하면 <sup class="reference" id="cite_ref-2"><a href="#cite_note-2">[2]</a></sup> 경번갑은 <a href="/wiki/%EC%A4%91%EB%8F%99" title="중동">중동</a>의 <a class="new" href="/w/index.php?title=%EB%84%93%EC%A0%81%EB%8B%A4%EB%A6%AC_%EA%B0%80%EB%A6%AC%EA%B0%9C&amp;action=edit&amp;redlink=1" title="넓적다리 가리개 (없는 문서)">넓적다리 가리개</a>에 처음 나타났고, <a class="new" href="/w/index.php?title=%ED%99%A9%EA%B8%88_%EA%B5%B0%EB%8B%A8&amp;action=edit&amp;redlink=1" title="황금 군단 (없는 문서)">황금 군단</a>에 전파되었다. 15세기 전반의 이란 모형은 <a href="/wiki/%EC%B0%B0%EA%B0%91" title="찰갑">찰갑</a>과 <a href="/wiki/%EB%91%90%EC%A0%95%EA%B0%91" title="두정갑">두정갑</a>의 조합으로 된 다양한 경번갑을 보여주며, 때로는 가슴 보강용으로 <a class="new" href="/w/index.php?title=%ED%98%B8%EC%8B%AC%EA%B2%BD&amp;action=edit&amp;redlink=1" title="호심경 (없는 문서)">호심경</a>을 착용하기도 했다. 몸통 보호용으로서의 경번갑 표현은 이란 모형에서 처음 나타나는데, 이것은 상대적으로 큰 철판으로 된 경번갑을 견갑과 (긴, 수평 철판으로 된) 치마와 함께 입고, 큰 호심경으로 보강한 모습을 보여준다. 고전적인 (적층 요소 없는) 경번갑에 대한 표현은 1465년으로 추정되는 바그다드의 모형에서 찾을 수 있다. 15세기 말부터 경번갑은 <a href="/wiki/%EC%B0%B0%EA%B0%91" title="찰갑">찰갑</a>을 완전히 대체하기 시작했다. 동유럽 (러시아와 폴란드)과 동양 경번갑의 주된 차이점은 동유럽 것은 보통 소매가 없는데 반해, 동양 것은 소매가 있다는 것이다 (팔뚝은 완갑으로 보호). 무거운 것에는 이 소매에 철판이 엮여 있고, (더 널리 사용된) 가벼운 것은 소매 전체가 사슬로 만들어졌다.</p>
<p>"키타브 알 두라 알 막누나" ("비주의 서") 에서 <a class="new" href="/w/index.php?title=%EA%B2%8C%EB%B2%84&amp;action=edit&amp;redlink=1" title="게버 (없는 문서)">게버</a>는 <b>경번갑</b>이 <a href="/wiki/%EA%B0%91%EC%98%B7" title="갑옷">갑옷</a> (<i>자와신</i>), <a href="/wiki/%ED%88%AC%EA%B5%AC" title="투구">투구</a> (<i>비드</i>), <a href="/wiki/%EB%B0%A9%ED%8C%A8" title="방패">방패</a> (<i>다라크</i>)로 이루어져 있다고 기술한다.<sup class="reference" id="cite_ref-Pearl_3-0"><a href="#cite_note-Pearl-3">[3]</a></sup></p>
<p>일본에서 경번갑은 '카루타'라고 불리는데, 작은 정사각형이나 직사각형 철판 사이를 사슬로 메운 것이다 <sup class="reference" id="cite_ref-4"><a href="#cite_note-4">[4]</a></sup></p>
