<h2><span class="mw-headline" id=".EC.A7.80.EB.A6.AC">지리</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EA%B2%BD%EC%84%B1%EA%B5%B0&amp;action=edit&amp;section=1" title="부분 편집: 지리">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>서쪽과 남서쪽은 <a href="/wiki/%ED%95%A8%EA%B2%BD%EC%82%B0%EB%A7%A5" title="함경산맥">함경산맥</a>(咸鏡山脈)이 위치해 있어 전반적으로 높으며, 특히 함경산맥에는 <a href="/wiki/%ED%95%9C%EB%B0%98%EB%8F%84" title="한반도">한반도</a>에서 두 번째로 높은 산인 <a href="/wiki/%EA%B4%80%EB%AA%A8%EB%B4%89" title="관모봉">관모봉</a>(冠帽峰, 2,541m)이 있다. 동쪽은 <a href="/wiki/%EB%8F%99%ED%95%B4" title="동해">동해</a>에 접하며, <a class="new" href="/w/index.php?title=%EA%B2%BD%EC%84%B1%ED%8F%89%EC%95%BC&amp;action=edit&amp;redlink=1" title="경성평야 (없는 문서)">경성평야</a> 등이 있어 해안 평야 지대를 이루고 있다. 주요 하천으로는 <a class="new" href="/w/index.php?title=%EC%96%B4%EB%9E%91%EC%B2%9C&amp;action=edit&amp;redlink=1" title="어랑천 (없는 문서)">어랑천</a>(漁郞川), <a class="new" href="/w/index.php?title=%EC%A3%BC%EB%B6%81%EC%B2%9C&amp;action=edit&amp;redlink=1" title="주북천 (없는 문서)">주북천</a>(朱北川), <a class="new" href="/w/index.php?title=%EB%B3%B4%EB%A1%9C%EC%B2%9C&amp;action=edit&amp;redlink=1" title="보로천 (없는 문서)">보로천</a>(甫老川), <a class="new" href="/w/index.php?title=%EC%A3%BC%EC%9D%84%EC%98%A8%EC%B2%9C_(%ED%95%98%EC%B2%9C)&amp;action=edit&amp;redlink=1" title="주을온천 (하천) (없는 문서)">주을온천</a>(朱乙溫川), <a class="new" href="/w/index.php?title=%EC%98%A4%EC%B4%8C%EC%B2%9C&amp;action=edit&amp;redlink=1" title="오촌천 (없는 문서)">오촌천</a>(梧村川)이 있으며, 주요 <a href="/wiki/%ED%98%B8%EC%88%98" title="호수">호수</a>로는 <a class="new" href="/w/index.php?title=%EC%9E%A5%EC%97%B0%ED%98%B8&amp;action=edit&amp;redlink=1" title="장연호 (없는 문서)">장연호</a>(長淵湖), <a class="new" href="/w/index.php?title=%EB%AC%B4%EA%B3%84%ED%98%B8&amp;action=edit&amp;redlink=1" title="무계호 (없는 문서)">무계호</a>(武溪湖) 등이 있다.</p>
<p>경성군의 80%는 숲으로 이루어져 있다. 동쪽 평야 지대에서는 주로 논농사를 짓고, 과수원 농사로는 <a class="mw-redirect" href="/wiki/%EB%B0%B0_(%EA%B3%BC%EC%9D%BC)" title="배 (과일)">배</a>가 주요 농작물이다. 해안의 어촌에서는 소규모의 어업도 행해지고 있다. 경성군 생기령(生氣嶺)은 <a class="mw-redirect" href="/wiki/%EA%B3%A0%EB%A0%B9%ED%86%A0" title="고령토">고령토</a>의 산지이기 때문에, 이 지방은 <a href="/wiki/%EB%8F%84%EC%9E%90%EA%B8%B0" title="도자기">도자기</a> 공업이 발달하였다.</p>
<p>교통으로는 <a href="/wiki/%ED%8F%89%EB%9D%BC%EC%84%A0" title="평라선">평라선</a> 철도가 지나며, 관내의 역으로는 <a class="new" href="/w/index.php?title=%EB%B4%89%EA%B0%95%EC%97%AD&amp;action=edit&amp;redlink=1" title="봉강역 (없는 문서)">봉강역</a>, <a class="new" href="/w/index.php?title=%EC%96%B4%EB%8C%80%EC%A7%84%EC%97%AD&amp;action=edit&amp;redlink=1" title="어대진역 (없는 문서)">어대진역</a>, <a href="/wiki/%EC%96%B4%EB%9E%91%EC%97%AD" title="어랑역">어랑역</a>, <a class="new" href="/w/index.php?title=%EC%9A%A9%ED%98%84%EC%97%AD_(%EA%B2%BD%EC%84%B1%EA%B5%B0)&amp;action=edit&amp;redlink=1" title="용현역 (경성군) (없는 문서)">용현역</a>, <a class="new" href="/w/index.php?title=%EA%B2%BD%EC%84%B1%EC%97%AD_(%EA%B2%BD%EC%84%B1%EA%B5%B0)&amp;action=edit&amp;redlink=1" title="경성역 (경성군) (없는 문서)">경성역</a>, <a href="/wiki/%EC%83%9D%EA%B8%B0%EB%A0%B9%EC%97%AD" title="생기령역">생기령역</a>, <a href="/wiki/%EC%8A%B9%EC%95%94%EC%97%AD" title="승암역">승암역</a>이 있다.</p>
<p>관광지로는 국경 수비를 위해 쓰여온 <a href="/wiki/%EA%B2%BD%EC%84%B1%EC%9D%8D%EC%84%B1" title="경성읍성">경성읍성</a>과 경성남문이 승암로동자구에 있다. 이 지방은 <a class="new" href="/w/index.php?title=%EC%A3%BC%EC%9D%84%EC%98%A8%EC%B2%9C&amp;action=edit&amp;redlink=1" title="주을온천 (없는 문서)">주을온천</a>이 있어 예부터 <a href="/wiki/%EC%98%A8%EC%B2%9C" title="온천">온천</a>으로도 유명했다.</p>
<p>광복 당시 경성군의 면적은 2,801.01㎢, 현재는 1,171㎢이다.</p>
<h2><span class="mw-headline" id=".EA.B8.B0.ED.9B.84">기후</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EA%B2%BD%EC%84%B1%EA%B5%B0&amp;action=edit&amp;section=2" title="부분 편집: 기후">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>바다와 접하여 비교적 온난한 해양성 기후를 보인다. 연간 평균기온은 6.6℃이며, 1월 평균 기온은 -8.2℃, 8월 평균 기온은 21.1℃이다. 평균 강수량은 동쪽 해안 지대와 서쪽 산간 지대가 차이를 보이는데, 동쪽 해안 지방은 연평균 648mm, 서쪽 산악 지방은 800mm이다. 기후는 바다의 영향으로 비교적 온난한 편이지만 <a href="/wiki/%EB%B6%81%ED%95%9C%ED%95%9C%EB%A5%98" title="북한한류">북한한류</a>(北韓寒流)의 영향으로 특히 7월경에 <a href="/wiki/%EC%95%88%EA%B0%9C" title="안개">안개</a>가 끼는 날이 많다.</p>
<h2><span class="mw-headline" id=".EC.97.AD.EC.82.AC">역사</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EA%B2%BD%EC%84%B1%EA%B5%B0&amp;action=edit&amp;section=3" title="부분 편집: 역사">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p><a href="/wiki/%EA%B3%A0%EA%B5%AC%EB%A0%A4" title="고구려">고구려</a>·<a href="/wiki/%EB%B0%9C%ED%95%B4" title="발해">발해</a>를 거쳐 <a class="mw-redirect" href="/wiki/%EC%97%AC%EC%A7%84%EC%A1%B1" title="여진족">여진족</a>의 영역이 되었고, 고려와의 분쟁 장소가 되었다. 한 때는 <a href="/wiki/%EC%9B%90%EB%82%98%EB%9D%BC" title="원나라">원나라</a>의 통치하에 놓여진 적도 있었다. <a href="/wiki/%EC%A1%B0%EC%84%A0" title="조선">조선</a> 초기에 이 곳은 '경성'이라고 불리게 되었고 이 곳을 지배하는 경성도호부사는 북도절도사의 겸임으로 여겨지는 등, 동북 변경 통치상의 중요한 장소로 여겨졌다. 근대에 들어서도 계속 <a href="/wiki/%ED%95%A8%EA%B2%BD%EB%B6%81%EB%8F%84" title="함경북도">함경북도</a>의 도청 소재지였다. <a href="/wiki/%EC%9D%BC%EC%A0%9C_%EA%B0%95%EC%A0%90%EA%B8%B0" title="일제 강점기">일제 강점기</a>까지 경성군은 현재의 <a href="/wiki/%EC%96%B4%EB%9E%91%EA%B5%B0" title="어랑군">어랑군</a>을 포함했다. 군내의 라남에는 일본군 제19사단이 주둔했고 1920년대에는 라남으로 함경북도청이 이전되었다. 라남은 <a href="/wiki/1940%EB%85%84" title="1940년">1940년</a> <a href="/wiki/%EC%B2%AD%EC%A7%84%EC%8B%9C" title="청진시">청진부</a>에 편입되었다.</p>
<ul>
<li><a href="/wiki/1398%EB%85%84" title="1398년">1398년</a> - 경성으로 불리게 되었다.</li>
<li><a href="/wiki/1410%EB%85%84" title="1410년">1410년</a> - 경성부가 설치되었다.</li>
<li><a href="/wiki/1432%EB%85%84" title="1432년">1432년</a> - 경성도호부로 승격하였다.</li>
<li><a href="/wiki/1896%EB%85%84" title="1896년">1896년</a> - 경성에 함경북도 관찰부가 설치되었다.</li>
<li><a href="/wiki/1940%EB%85%84" title="1940년">1940년</a> <a href="/wiki/3%EC%9B%94" title="3월">3월</a> - 경성군 나남읍, 용성면(수성)이 청진부에 편입되었다.</li>
<li><a href="/wiki/1943%EB%85%84" title="1943년">1943년</a> <a href="/wiki/10%EC%9B%94_1%EC%9D%BC" title="10월 1일">10월 1일</a> - 주을온면을 주을읍으로, 어랑면 일부를 어대진읍으로 승격하였다.<sup class="reference" id="cite_ref-2"><a href="#cite_note-2">[2]</a></sup></li>
<li><a href="/wiki/1952%EB%85%84" title="1952년">1952년</a> - 주을읍 주을온리를 경성읍으로 개편했으며, 어대진읍과 어랑면을 어랑군으로 개편했다.</li>
<li><a href="/wiki/1979%EB%85%84" title="1979년">1979년</a> - 청진시가 직할시가 되면서 청진직할시 경성군으로 개편</li>
<li><a href="/wiki/1985%EB%85%84" title="1985년">1985년</a> - 함경북도 경성군으로 개편</li>
</ul>
<h2><span class="mw-headline" id=".ED.96.89.EC.A0.95_.EA.B5.AC.EC.97.AD">행정 구역</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EA%B2%BD%EC%84%B1%EA%B5%B0&amp;action=edit&amp;section=4" title="부분 편집: 행정 구역">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>광복 당시 경성군은 어대진읍(漁大津邑), 주을읍(朱乙邑), 경성면(鏡城面: 군청 소재지), 어랑면(漁郞面), 주북면(朱北面), 주남면(朱南面)의 2읍 4면으로 이루어져 있었으며 그 영역은 현재의 <a href="/wiki/%EC%B2%AD%EC%A7%84%EC%8B%9C" title="청진시">청진시</a> 일부 및 <a href="/wiki/%EC%96%B4%EB%9E%91%EA%B5%B0" title="어랑군">어랑군</a> 전지역도 포함하고 있었다. 그러나 현재 경성군의 읍리는 경성읍, 생기령로동자구, 승암로동자구, 하온포로동자구, 룡천로동자구, 박충로동자구, 상온포리, 룡산리, 하면리, 화하리, 관모리, 대향리, 중평리, 룡현리, 온대진리, 일향리, 오상리, 독연리, 장평리, 남석리, 매향리가 있다.</p>
