<h1 class="firstHeading" id="firstHeading" lang="ko">경입자</h1>
<div class="mw-body-content" id="bodyContent">
<div id="siteSub">위키백과, 우리 모두의 백과사전.</div>
<div id="contentSub"></div>
<div class="mw-jump" id="jump-to-nav">
					이동:					<a href="#mw-head">둘러보기</a>, 					<a href="#p-search">검색</a>
</div>
<div class="mw-content-ltr" dir="ltr" id="mw-content-text" lang="ko"><script>function mfTempOpenSection(id){var block=document.getElementById("mf-section-"+id);block.className+=" open-block";block.previousSibling.className+=" open-block";}</script><p><b>경입자</b>(輕粒子, <span lang="en" xml:lang="en">lepton</span> <small>렙톤<sup>[<a href="/wiki/%EC%9C%84%ED%82%A4%EB%B0%B1%EA%B3%BC:%EC%98%81%EC%96%B4%EC%9D%98_%ED%95%9C%EA%B8%80_%ED%91%9C%EA%B8%B0" title="위키백과:영어의 한글 표기">*</a>]</sup></small>)는 <a href="/wiki/%EB%AC%BC%EB%A6%AC%ED%95%99" title="물리학">물리학</a>에서 <a href="/wiki/%EC%8A%A4%ED%95%80" title="스핀">스핀</a>이 ½이고 (<a class="mw-redirect" href="/wiki/%ED%8E%98%EB%A5%B4%EB%AF%B8%EC%98%A8" title="페르미온">페르미온</a>) <a href="/wiki/%EA%B0%95%ED%95%9C_%EC%83%81%ED%98%B8%EC%9E%91%EC%9A%A9" title="강한 상호작용">강한 핵력에 영향을 받지않는</a> <a href="/wiki/%EA%B8%B0%EB%B3%B8_%EC%9E%85%EC%9E%90" title="기본 입자">기본 입자</a>이다. <a href="/wiki/%EC%A0%84%EC%9E%90" title="전자">전자</a>, <a class="mw-redirect" href="/wiki/%EB%AE%A4%EC%98%A8" title="뮤온">뮤온</a>, <a class="mw-redirect" href="/wiki/%ED%83%80%EC%9A%B0%EC%98%A8" title="타우온">타우온</a>과 각각에 해당하는 중성미자로, 현재까지 총 6종이 알려져 있다. 강하게 상호작용하는 기본 페르미온은 <a href="/wiki/%EC%BF%BC%ED%81%AC" title="쿼크">쿼크</a>로 분류한다.</p>
<p>경입자는 <a class="mw-redirect" href="/wiki/%EA%B8%B0%EB%B3%B8%EC%9E%85%EC%9E%90" title="기본입자">기본 입자</a>이자, <a href="/wiki/%EB%AC%BC%EC%A7%88" title="물질">물질</a>의 가장 기본적인 구성요소이다<sup class="reference" id="cite_ref-1"><a href="#cite_note-1">[1]</a></sup>. 경입자는 물질의 기원을 설명하는 표준 모형에서 중요한 위치를 차지하고 있다. 대표적인 경입자로는 <a href="/wiki/%EC%A0%84%EC%9E%90" title="전자">전자</a>를 꼽을 수 있다. 전자는 물질의 화학적 특성에 직접적으로 연관되어 있기에 화학이라는 학문 전 범위에 걸쳐 중요한 위치를 차지하고 있다.</p>
<p>렙톤은 <a href="/wiki/%EC%A0%84%ED%95%98" title="전하">전하</a>를 띠는가 아닌가에 따라 두 종류로 분류할 수 있다. 첫째는 전하를 띠는 렙톤이다. ‘전자 같은 렙톤(electron-like lepton)’이라고도 칭한다. 전하를 띠는 렙톤은 다른 입자들과 결합하여, <a href="/wiki/%EC%9B%90%EC%9E%90" title="원자">원자</a>나 <a href="/wiki/%ED%8F%AC%EC%A7%80%ED%8A%B8%EB%A1%9C%EB%8A%84" title="포지트로늄">포지트로늄</a>(positronium)과 같은 합성된 입자(composite particle)를 구성한다. 둘째는 <a href="/wiki/%EC%A4%91%EC%84%B1%EB%AF%B8%EC%9E%90" title="중성미자">중성미자</a>다. 중성미자는 전하를 띠는 렙톤과 달리 전하를 띠지 않는다. 중성미자(neutrino)는 전하를 띠는 렙톤보다 질량이 작다. 또한 다른 입자들과 상호작용 또한 거의 하지 않아, 좀처럼 관찰되지 않는 입자이다.</p>
<p>렙톤은 또한 3 <a href="/wiki/%EC%84%B8%EB%8C%80" title="세대">세대 (물리)</a>에 걸쳐 여섯 종류로 달리 분류할 수 있다.<sup class="reference" id="cite_ref-2"><a href="#cite_note-2">[2]</a></sup> 1세대는 <a href="/wiki/%EC%A0%84%EC%9E%90" title="전자">전자</a>와 <a href="/wiki/%EC%A4%91%EC%84%B1%EB%AF%B8%EC%9E%90" title="중성미자">전자 중성미자</a>, 2세대는 <a class="mw-redirect" href="/wiki/%EB%AE%A4%EC%98%A8" title="뮤온">뮤온</a>과 <a href="/wiki/%EC%A4%91%EC%84%B1%EB%AF%B8%EC%9E%90" title="중성미자">뮤온 중성미자</a>, 3세대는 <a class="mw-redirect" href="/wiki/%ED%83%80%EC%9A%B0%EC%98%A8" title="타우온">타우온</a>과 <a href="/wiki/%EC%A4%91%EC%84%B1%EB%AF%B8%EC%9E%90" title="중성미자">타우 중성미자</a>로 구성되어 있다. 1세대에서 3세대로 갈수록 구성하는 입자들의 질량이 증가하는 경향을 보인다.</p>
<p>전자는 전하를 띠는 입자 중에서 가장 가볍다. 다음으로 무거운 뮤온은 입자 붕괴 과정을 통해 금방 전자로 변한다. 가장 무거운 타우온은 18%의 확률로 전자와 중성미자로 붕괴한다. 때문에, 뮤온과 타우온은 <a href="/wiki/%EC%9A%B0%EC%A3%BC%EC%84%A0" title="우주선">우주선</a>(cosmic ray)이나 입자 가속기를 통해서만 발견할 수 있는 반면, 전자는 전우주에서 가장 쉽게 발견할 수 있다.</p>
<p>렙톤은 <a href="/wiki/%EC%A0%84%ED%95%98" title="전하">전하량</a>, <a href="/wiki/%EC%8A%A4%ED%95%80" title="스핀">스핀</a>, <a href="/wiki/%EC%A7%88%EB%9F%89" title="질량">질량</a>와 같은 다양한 고유 성질을 가지고 있다. 렙톤은 쿼크와는 다르게 강한 상호작용(또는 <a href="/wiki/%EA%B0%95%ED%95%9C_%EC%83%81%ED%98%B8%EC%9E%91%EC%9A%A9" title="강한 상호작용">강력</a>)에 영향을 받지 않는다. 그러나 나머지 세가지 상호작용(<a href="/wiki/%EC%A4%91%EB%A0%A5" title="중력">중력</a>, <a href="/wiki/%EC%A0%84%EC%9E%90%EA%B8%B0%ED%95%99" title="전자기학">전자기력</a>, <a href="/wiki/%EC%95%BD%ED%95%9C_%EC%83%81%ED%98%B8%EC%9E%91%EC%9A%A9" title="약한 상호작용">약력</a>)에는 영향을 받는다. 모든 렙톤들은 각자에게 대칭되는 <a href="/wiki/%EB%B0%98%EC%9E%85%EC%9E%90" title="반입자">반입자</a>인 <a class="new" href="/w/index.php?title=%EB%B0%98%EB%A0%99%ED%86%A4&amp;action=edit&amp;redlink=1" title="반렙톤 (없는 문서)">반렙톤</a>들을 가지고 있다. 혹자는 전하를 띠는 렙톤과 중성미자 간의 관계가 입자-반입자 관계라고 하나, 그 주장의 타당성이 명확히 밝혀지지 않은 상태다.</p>
<p>1세대의 전하를 띠는 렙톤 <a href="/wiki/%EC%A0%84%EC%9E%90" title="전자">전자</a>는 19세기의 여러 과학자들에 의해 이론적으로 제안되었고<sup class="reference" id="cite_ref-3"><a href="#cite_note-3">[3]</a></sup><sup class="reference" id="cite_ref-4"><a href="#cite_note-4">[4]</a></sup><sup class="reference" id="cite_ref-5"><a href="#cite_note-5">[5]</a></sup>, 1897년에 <a href="/wiki/%EC%A1%B0%EC%A7%80%ED%94%84_%EC%A1%B4_%ED%86%B0%EC%8A%A8" title="조지프 존 톰슨">톰슨</a>이 발견했다.<sup class="reference" id="cite_ref-6"><a href="#cite_note-6">[6]</a></sup> <a class="mw-redirect" href="/wiki/%EB%AE%A4%EC%98%A8" title="뮤온">뮤온</a>은 1936년에 <a href="/wiki/%EC%B9%BC_%EB%8D%B0%EC%9D%B4%EB%B9%84%EB%93%9C_%EC%95%A4%EB%8D%94%EC%8A%A8" title="칼 데이비드 앤더슨">앤더슨</a>이 발견했으나, 발견 당시에는 <a href="/wiki/%EC%A4%91%EA%B0%84%EC%9E%90" title="중간자">메존</a>으로 잘못 분류했었다.<sup class="reference" id="cite_ref-7"><a href="#cite_note-7">[7]</a></sup> 그러나 실험을 통해, 새로 발견한 뮤온이 메존의 성질을 띠기보다는 전자에 가까운 성질을 띤다는 사실을 알아냈다. 1947년에서야 전자와 같이 행동하는 입자들을 ‘렙톤’이라 이름 붙여주며, 뮤온을 렙톤에 포함시켰다. 전자 중성미자는 1930년 <a href="/wiki/%EB%B3%BC%ED%94%84%EA%B0%95_%ED%8C%8C%EC%9A%B8%EB%A6%AC" title="볼프강 파울리">파울리</a>가 입자의 <a href="/wiki/%EB%B2%A0%ED%83%80_%EB%B6%95%EA%B4%B4" title="베타 붕괴">베타 붕괴</a> 현상을 설명하기 위해, 이론적으로 처음 제안했다. 전자 중성미자는 1956년 <a class="new" href="/w/index.php?title=%ED%81%B4%EB%9D%BC%EC%9D%B4%EB%93%9C_%EC%B9%B4%EC%9B%90&amp;action=edit&amp;redlink=1" title="클라이드 카원 (없는 문서)">카원</a>과 <a class="new" href="/w/index.php?title=%ED%94%84%EB%A0%88%EB%8D%94%EB%A6%AD_%EB%9D%BC%EC%9D%B4%EB%84%88%EC%8A%A4&amp;action=edit&amp;redlink=1" title="프레더릭 라이너스 (없는 문서)">라이너스</a>가 훗날 ‘<a href="/wiki/%EC%B9%B4%EC%9B%90-%EB%9D%BC%EC%9D%B4%EB%84%88%EC%8A%A4_%EC%A4%91%EC%84%B1%EB%AF%B8%EC%9E%90_%EC%8B%A4%ED%97%98" title="카원-라이너스 중성미자 실험">카원-라이너스 중성미자 실험</a>’라 명명된 실험을 통하여 처음 발견했다. 뮤온 중성미자는 1962년 <a href="/wiki/%EB%A6%AC%EC%96%B8_%EB%A0%88%EB%8D%94%EB%A8%BC" title="리언 레더먼">레더먼</a>, <a href="/wiki/%EB%A9%9C%EB%B9%88_%EC%8A%88%EC%9B%8C%EC%B8%A0" title="멜빈 슈워츠">슈워츠</a>, <a href="/wiki/%EC%9E%AD_%EC%8A%A4%ED%83%80%EC%9D%B8%EB%B2%84%EA%B1%B0" title="잭 스타인버거">스타인버거</a>가 발견했다. 타우온은 1974년과 1977년 사이에 <a href="/wiki/%EC%8A%A4%ED%83%A0%ED%8D%BC%EB%93%9C_%EC%84%A0%ED%98%95_%EA%B0%80%EC%86%8D%EA%B8%B0_%EC%84%BC%ED%84%B0" title="스탠퍼드 선형 가속기 센터">스탠퍼드 선형 가속기 센터</a>와 <a class="new" href="/w/index.php?title=%EB%A1%9C%EB%9F%B0%EC%8A%A4_%EB%B2%84%ED%81%B4%EB%A6%AC_%EA%B5%AD%EB%A6%BD_%EC%97%B0%EA%B5%AC%EC%86%8C&amp;action=edit&amp;redlink=1" title="로런스 버클리 국립 연구소 (없는 문서)">로런스 버클리 국립 연구소</a>에서 연구한 <a href="/wiki/%EB%A7%88%ED%8B%B4_%EB%A3%A8%EC%9D%B4%EC%8A%A4_%ED%8E%84" title="마틴 루이스 펄">마틴 루이스 펄</a>과 그의 동료들이 발견했다. 타우 중성미자는 2000년 7월 <a href="/wiki/%ED%8E%98%EB%A5%B4%EB%AF%B8_%EA%B5%AD%EB%A6%BD_%EA%B0%80%EC%86%8D%EA%B8%B0_%EC%97%B0%EA%B5%AC%EC%86%8C" title="페르미 국립 가속기 연구소">페르미 국립 가속기 연구소</a>의 DONUT Collaboration이 발견했다고 발표했다.</p>
<p>렙톤은 물질의 기원을 설명하는 표준 모형에서 중요한 위치를 차지하고 있다. 그리고 전자는 양성자, 중성자와 함께 원자를 구성한다.</p>

<h2><span class="mw-headline" id=".EB.A0.99.ED.86.A4.EC.9D.98_.ED.8A.B9.EC.84.B1">렙톤의 특성</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EA%B2%BD%EC%9E%85%EC%9E%90&amp;action=edit&amp;section=1" title="부분 편집: 렙톤의 특성">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>렙톤에는 <a href="/wiki/%EC%A0%84%EC%9E%90" title="전자">전자</a>, <a class="mw-redirect" href="/wiki/%EB%AE%A4%EC%98%A8" title="뮤온">뮤온</a>, <a class="mw-redirect" href="/wiki/%ED%83%80%EC%9A%B0%EC%98%A8" title="타우온">타우온</a>의 세 <a class="mw-redirect" href="/wiki/%EB%A7%9B%EA%B9%94_(%EB%AC%BC%EB%A6%AC%ED%95%99)" title="맛깔 (물리학)">맛깔</a>이 존재한다. 각 맛깔은 <a href="/wiki/%EC%95%BD%ED%95%9C_%EC%95%84%EC%9D%B4%EC%86%8C%EC%8A%A4%ED%95%80" title="약한 아이소스핀">약한 아이소스핀</a>이라는 입자쌍으로 표현된다. 입자쌍 중 하나는 각 맛깔의 이름과 동일한 이름의 높은 전하를 지닌 입자이며, 다른 하나는 거의 질량이 없는 중성의 입자인 <a href="/wiki/%EC%A4%91%EC%84%B1%EB%AF%B8%EC%9E%90" title="중성미자">중성미자</a>이다. 예를 들면, 전자는 전하를 지닌 <i>전자</i>와 전하를 지니지 않고 거의 질량이 없는 <i>전자 반중성미자</i>로 구성된다는 것이다. 이 모든 6개의 입자는 각각 <a href="/wiki/%EB%B0%98%EC%9E%85%EC%9E%90" title="반입자">반입자</a>를 가지고 있다. 예를 들면, 전자와 <a href="/wiki/%EC%96%91%EC%A0%84%EC%9E%90" title="양전자">양전자</a>는 서로 반입자이다. 모든 알려진 전하를 지닌 렙톤은 <a href="/wiki/%EA%B8%B0%EB%B3%B8_%EC%A0%84%ED%95%98" title="기본 전하">기본 전하</a>만큼의 전하를 지니고 있으며, 음인지 양인지는 입자인지 반입자인지에 따라 좌우된다. 반면 중성미자와 <a href="/wiki/%EB%B0%98%EC%A4%91%EC%84%B1%EB%AF%B8%EC%9E%90" title="반중성미자">반중성미자</a>는 전하를 띠지 않는다.</p>
<p>입자가 상호작용할때, 같은 <a class="mw-redirect" href="/wiki/%EB%A7%9B%EA%B9%94_(%EB%AC%BC%EB%A6%AC%ED%95%99)" title="맛깔 (물리학)">맛깔</a>(전자와 전자 중성미자, 뮤온과 뮤온 중성미자, 타우온과 타우 중성미자)의 렙톤 수는 그대로 유지된다. 이 원리는 <a class="new" href="/w/index.php?title=%EB%A0%99%ED%86%A4_%EC%88%98&amp;action=edit&amp;redlink=1" title="렙톤 수 (없는 문서)">렙톤 수</a>의 보존으로 알려져 있다. 다른 맛깔(예를 들면, 전자 수와 뮤온 수)의 렙톤 수는 때로 지켜지지 않는다. <a href="/wiki/%EC%A4%91%EC%84%B1%EB%AF%B8%EC%9E%90_%EC%A7%84%EB%8F%99" title="중성미자 진동">중성미자 진동</a>이 그 예이다.</p>

<h2><span class="mw-headline" id=".EC.96.B4.EC.9B.90">어원</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EA%B2%BD%EC%9E%85%EC%9E%90&amp;action=edit&amp;section=3" title="부분 편집: 어원">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>"렙톤"이라는 이름은 <a href="/wiki/%EA%B7%B8%EB%A6%AC%EC%8A%A4%EC%96%B4" title="그리스어">그리스어</a> λεπτός (렙토스)에서 유래하였고, 물리학자 <a class="new" href="/w/index.php?title=%EB%A0%88%EC%98%B9_%EB%A1%9C%EC%9E%A5%ED%8E%A0%EB%93%9C&amp;action=edit&amp;redlink=1" title="레옹 로장펠드 (없는 문서)">레옹 로장펠드</a>(Léon Rosenfeld)가 <a href="/wiki/1948%EB%85%84" title="1948년">1948년</a>에 명명하였다.</p>
<dl>
<dd><i>묄러 (C. Møller) 교수의 제안을 따라, 가벼운 질량을 가진 입자를 작고, 얇다는 의미의 그리스어 λεπτός로부터 "렙톤"이라 명명한다.</i><sup class="reference" id="cite_ref-8"><a href="#cite_note-8">[8]</a></sup></dd>
</dl>
<p>이 이름은 전자나 중성미자, 뮤온이 <a href="/wiki/%ED%95%B5%EC%9E%90" title="핵자">핵자</a>에 비해 가벼워서 지어졌다. (핵자는 <a class="mw-redirect" href="/wiki/%EB%B0%94%EB%A6%AC%EC%98%A8" title="바리온">중(重)입자</a>에 속한다.) 이후 1970년대에 핵자의 두 배나 무거운 타우온이 발견되어, 모든 렙톤이 다 가벼운 것은 아니라는 사실이 밝혀졌다.</p>
<h2><span class="mw-headline" id=".EC.97.AD.EC.82.AC">역사</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EA%B2%BD%EC%9E%85%EC%9E%90&amp;action=edit&amp;section=4" title="부분 편집: 역사">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>1세대의 전하를 띠는 렙톤인, <a href="/wiki/%EC%A0%84%EC%9E%90" title="전자">전자</a>는 19세기의 여러 과학자들에 의해 이론적으로 제안되었고, 1897년에 J.J.Tomson과 그의 동료인 여러 영국 물리학자들이 발견했다. 그 후 전자 1930년 Wolfgang Pauli가 <a href="/wiki/%EB%B2%A0%ED%83%80_%EB%B6%95%EA%B4%B4" title="베타 붕괴">베타 붕괴</a> 현상 이전과 이후의 <a href="/wiki/%EC%97%90%EB%84%88%EC%A7%80" title="에너지">에너지</a>, <a href="/wiki/%EC%9A%B4%EB%8F%99%EB%9F%89" title="운동량">운동량</a>, <a href="/wiki/%EA%B0%81%EC%9A%B4%EB%8F%99%EB%9F%89" title="각운동량">각운동량</a>이 보존되지 않는다는 실험 결과에 아직 발견되지 않은 입자가 숨어 있을 것이라 추론했으며, 그 숨어 있는 입자를 <a href="/wiki/%EC%A4%91%EC%84%B1%EB%AF%B8%EC%9E%90" title="중성미자">중성미자</a>(neutrino)라고 명명했다. 여담으로 당시에는 <a href="/wiki/%EC%A4%91%EC%84%B1%EB%AF%B8%EC%9E%90" title="중성미자">전자 중성미자</a>라고 따로 명명되지 않은 이유는 전자 이외의 전하를 띠는 렙톤이 제안되지 않아서 렙톤이라는 개념조차 없었던 시대였기 때문이다. 전자 중성미자는 1956년 Clyde Cowan과 Frederick Reines이 훗날 ‘Cowan-Reines 중성미자 실험’라 명명된 실험을 통하여 처음 발견했다.</p>
<p><a class="mw-redirect" href="/wiki/%EB%AE%A4%EC%98%A8" title="뮤온">뮤온</a>은 1936년에 Carl D. Anderson이 발견했다. 발견 당시 그들은 뮤온의 질량 때문에 뮤온을 <a href="/wiki/%EC%A4%91%EA%B0%84%EC%9E%90" title="중간자">메존</a>으로 잘못 분류했었다. 그러나 뮤온이 전자처럼 강한 상호작용에 영향을 받지 않는다는 실험 결과를 토대로 하여, 새로 발견한 뮤온이 메존보다는 전자에 더 가깝다고 결론을 내렸다. 1947년에서야 전자와 같이 행동하는 입자들을 ‘렙톤’이라 이름 붙여주며, 전자와 함께, 뮤온과 (전자) 중성미자를 렙톤에 포함시켰다. 1962년 Leon M. Lederman, Melvin Schwartz 그리고 Jack Steinberger는 <a href="/wiki/%EC%A4%91%EC%84%B1%EB%AF%B8%EC%9E%90" title="중성미자">뮤온 중성미자</a>의 상호작용을 처음 발견하면서, 전자 중성미자 이외에 또 다른 종류의 중성미자가 존재함을 입증했다. 이 공로를 기리기 위하여, 1988년 세 사람에게 <a href="/wiki/%EB%85%B8%EB%B2%A8_%EB%AC%BC%EB%A6%AC%ED%95%99%EC%83%81" title="노벨 물리학상">노벨 물리학상</a>이 수여되었다.</p>
<p><a class="mw-redirect" href="/wiki/%ED%83%80%EC%9A%B0%EC%98%A8" title="타우온">타우온</a>은 1974년과 1977년 사이에 스탠포드 선형 입자가속기 센터(<a href="/wiki/%EC%8A%A4%ED%83%A0%ED%8D%BC%EB%93%9C_%EC%84%A0%ED%98%95_%EA%B0%80%EC%86%8D%EA%B8%B0_%EC%84%BC%ED%84%B0" title="스탠퍼드 선형 가속기 센터">SLAC</a>)와 로렌스 버클리 국립 연구소에서 연구한 Martin Lewis Perl과 그의 동료들이 발견했다. 그리고 전자, 뮤온과 마찬가지로 타우온도 타우온만의 중성입자를 가지고 있을 것이라고 학자들은 예견했다. <a href="/wiki/%EC%A4%91%EC%84%B1%EB%AF%B8%EC%9E%90" title="중성미자">타우 중성미자</a>의 첫 증거는 타우온 붕괴 현상에서 붕괴 이전과 이후의 에너지 및 운동량이 보존되지 않는다는 사실이었다. 타우 중성미자는 2000년 7월 <a href="/wiki/%ED%8E%98%EB%A5%B4%EB%AF%B8_%EA%B5%AD%EB%A6%BD_%EA%B0%80%EC%86%8D%EA%B8%B0_%EC%97%B0%EA%B5%AC%EC%86%8C" title="페르미 국립 가속기 연구소">페르미 국립 가속기 연구소</a>에서 연구한 DONUT Collaboration이 발견했다고 발표했다.</p>
<p>일부 입자 물리학자들은 4세대 렙톤이 존재할 가능성을 제기하고 있다. 현재 알려진 바로는 만약 발견된다면 4세대 전하를 띠는 렙톤의 질량은 100.8 GeV/c2 정도가 한계이며, 4세대 중성미자의 질량은 45.0 GeV/c2 가 한계라고 예측하고 있다.</p>
<h2><span class="mw-headline" id=".EC.8A.A4.ED.95.80.EA.B3.BC_.EC.B9.B4.EC.9D.B4.EB.9E.84.EC.84.B1">스핀과 카이랄성</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EA%B2%BD%EC%9E%85%EC%9E%90&amp;action=edit&amp;section=5" title="부분 편집: 스핀과 카이랄성">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>렙톤은 스핀-1/2 입자다. 때문에 렙톤은 업, 다운, 이렇게 두 가지 <a href="/wiki/%EC%8A%A4%ED%95%80" title="스핀">스핀</a>만 가질 수 있다. <a class="new" href="/w/index.php?title=%EC%8A%A4%ED%95%80-%ED%86%B5%EA%B3%84%ED%95%99_%EC%A0%95%EB%A6%AC&amp;action=edit&amp;redlink=1" title="스핀-통계학 정리 (없는 문서)">스핀-통계학 정리</a>에 따라, 렙톤은 <a class="mw-redirect" href="/wiki/%ED%8E%98%EB%A5%B4%EB%AF%B8%EC%98%A8" title="페르미온">페르미온</a>임을 알 수 있다. 따라서, 렙톤은 <a href="/wiki/%ED%8C%8C%EC%9A%B8%EB%A6%AC_%EB%B0%B0%ED%83%80_%EC%9B%90%EB%A6%AC" title="파울리 배타 원리">파울리 배타 원리</a>를 지킨다. 다시 말해, 동일한 종류의 렙톤은 같은 시각에 같은 상태에 있을 수 없다.</p>
