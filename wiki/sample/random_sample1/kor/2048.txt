<p><b>경허</b>(鏡虛, <a href="/wiki/1849%EB%85%84" title="1849년">1849년</a> ~ <a href="/wiki/1912%EB%85%84" title="1912년">1912년</a>)는 한국 근현대 불교를 개창한 대선사이다.</p>
<p>1849년 전주 자동리에서 아버지 송두옥(宋斗玉)과 어머니 밀양 박씨의 차남으로 태어났다. 본관은 <a href="/wiki/%EC%97%AC%EC%82%B0_%EC%86%A1%EC%94%A8" title="여산 송씨">여산</a>(礪山)으로, 속명은 동욱(東旭)이다. <a class="new" href="/w/index.php?title=%EB%B2%95%ED%98%B8&amp;action=edit&amp;redlink=1" title="법호 (없는 문서)">법호</a>는 경허(鏡虛), <a href="/wiki/%EB%B2%95%EB%AA%85" title="법명">법명</a>은 성우(惺牛)이다.</p>
<p>9세 때 경기도 과천 <a class="mw-redirect" href="/wiki/%EC%B2%AD%EA%B3%84%EC%82%B0_(%EC%84%9C%EC%9A%B8)" title="청계산 (서울)">청계산</a>에 있는 청계사로 출가하였다. <a href="/wiki/1879%EB%85%84" title="1879년">1879년</a> 11월 15일<sup class="정리_필요 noprint">[<a href="/wiki/%EC%9C%84%ED%82%A4%EB%B0%B1%EA%B3%BC:%EB%AA%A8%ED%98%B8%ED%95%9C_%ED%91%9C%ED%98%84%EC%9D%84_%EC%82%BC%EA%B0%80_%EC%A3%BC%EC%84%B8%EC%9A%94" title="위키백과:모호한 표현을 삼가 주세요">모호한 표현</a>]</sup>, <a href="/wiki/%EB%8F%99%ED%95%99%EC%82%AC" title="동학사">동학사</a> 밑에 살고 있던 진사인, 이처사(李處士)의 한 마디, '소가 되더라도 콧구멍 없는 소가 되어야지.' 이 한마디를 전해듣고는, 바로 깨달았다. 콧구멍 없는 소(牛無鼻孔處: 우무비공처)는 중국 <a href="/wiki/%EB%B2%95%EC%95%88%EC%A2%85" title="법안종">법안종</a>의 종주 <a class="new" href="/w/index.php?title=%EB%B2%95%EC%95%88%EC%8A%A4%EB%8B%98&amp;action=edit&amp;redlink=1" title="법안스님 (없는 문서)">법안(法眼) 선사</a>의 어록에 실려 있는 선어다. 당시 경허의 시봉을 받들던 <a class="new" href="/w/index.php?title=%EC%82%AC%EB%AF%B8%EC%8A%B9&amp;action=edit&amp;redlink=1" title="사미승 (없는 문서)">사미승</a> 원규는 경허의 사제인 학명의 제자였고, 이처사는 사미승 원규의 속가 아버지였다.</p>
<p>1880년 어머니와 속가 형님인 스님이 주지로 있던 연암산 천장암으로 거쳐를 옮긴다. 경허는 연암산 천장암의 작은 방에서 1년 반동안 치열한 참선을 한 끝에 확철대오하게 되고 "사방을 둘러 보아도 사람이 없구나"라고 시작하는 오도송을 짓는다. 천장암에서 경허의 '삼월(三月)'로 불리는 수월스님과 혜월스님과 만공스님이 출가하여 함께 수행하게 된다. 제자들과 함께 천장암에서 지내다가 개심사 부석사 간월암등지를 다녀오기도 하였는데 이 때 경허스님과 제자들간의 많은 일화가 전한다.</p>
<p>1886년 6년 동안의 <a class="new" href="/w/index.php?title=%EB%B3%B4%EC%9E%84&amp;action=edit&amp;redlink=1" title="보임 (없는 문서)">보임</a>(保任)을 마치고 옷과 탈바가지, 주장자 등을 모두 불태운 뒤 <a class="new" href="/w/index.php?title=%EB%AC%B4%EC%95%A0%ED%96%89&amp;action=edit&amp;redlink=1" title="무애행 (없는 문서)">무애행</a>(無碍行)에 나섰다.</p>
<p>한동안 제자들을 가르치다가, 돌연 환속하여 박난주(朴蘭州)라고 개명하였고, 서당의 훈장이 되어 아이들을 가르치다가, 함경도 갑산(甲山) 웅이방(熊耳坊) 도하동(道下洞)에서 1912년 4월 25일 새벽에 <a class="new" href="/w/index.php?title=%EC%9E%84%EC%A2%85%EA%B2%8C&amp;action=edit&amp;redlink=1" title="임종게 (없는 문서)">임종게</a>를 남긴 뒤 입적하였다. 나이 64세, 법랍 56세이다. 저서에는 《<a class="new" href="/w/index.php?title=%EA%B2%BD%ED%97%88%EC%A7%91&amp;action=edit&amp;redlink=1" title="경허집 (없는 문서)">경허집</a>》이 있다.</p>
<p>경허 선사의 수제자로 흔히 '삼월(三月)'로 불리는 <a class="new" href="/w/index.php?title=%ED%98%9C%EB%AA%85_%EC%84%A0%EC%82%AC(%EC%A1%B0%EC%84%A0)&amp;action=edit&amp;redlink=1" title="혜명 선사(조선) (없는 문서)">혜월</a>(慧月, <a href="/wiki/1861%EB%85%84" title="1861년">1861년</a> - <a href="/wiki/1937%EB%85%84" title="1937년">1937년</a>), <a class="new" href="/w/index.php?title=%EC%88%98%EC%9B%94_%EC%84%A0%EC%82%AC&amp;action=edit&amp;redlink=1" title="수월 선사 (없는 문서)">수월</a>(水月, 1855년 - 1928년)·<a href="/wiki/%EB%A7%8C%EA%B3%B5_(%EC%8A%B9%EB%A0%A4)" title="만공 (승려)">만공</a>(滿空, 1871년 - 1946년) 선사가 있다. 경허는 '만공은 복이 많아 대중을 많이 거느릴 테고, 정진력은 수월을 능가할 자가 없고, 지혜는 혜월을 당할 자가 없다'고 했다. 삼월인 제자들도 모두 깨달아 부처가 되었다. 이들 역시 근현대 한국 불교계를 대표하는 선승들이다.</p>
<p>현재, '북송담 남진제'의 두 큰스님의 경우에, 송담스님은 경허(75대)-만공(76대)-전강(77대)-송담(78대)의 계보이고, <a class="mw-redirect" href="/wiki/%EC%A7%84%EC%A0%9C%EC%8A%A4%EB%8B%98" title="진제스님">진제스님</a>은 경허(75대)-혜월(76대)-운봉(77대)-향곡(78대)-<a class="mw-redirect" href="/wiki/%EC%A7%84%EC%A0%9C" title="진제">진제</a>(79대)의 계보이다.</p>
