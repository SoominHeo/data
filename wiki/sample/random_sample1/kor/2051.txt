<p><b>계왕</b>(契王, ? ~ <a href="/wiki/346%EB%85%84" title="346년">346년</a>, 재위 : <a href="/wiki/344%EB%85%84" title="344년">344년</a> <a href="/wiki/10%EC%9B%94" title="10월">10월</a>~<a href="/wiki/346%EB%85%84" title="346년">346년</a> <a href="/wiki/9%EC%9B%94" title="9월">9월</a>)은 백제의 제12대 <a href="/wiki/%EA%B5%B0%EC%A3%BC" title="군주">국왕</a>이다. 성은 부여(扶餘) 이름 계(契), <a class="mw-redirect" href="/wiki/%EB%B0%B1%EC%A0%9C_%EB%B6%84%EC%84%9C%EC%99%95" title="백제 분서왕">분서왕</a>(汾西王)의 장남이다. <a href="/wiki/%EB%B6%84%EC%84%9C%EC%99%95" title="분서왕">분서왕</a>의 죽었을 때 나이가 어렸으므로 <a href="/wiki/%EC%82%AC%EB%B0%98%EC%99%95" title="사반왕">사반왕</a>(沙伴王)의 동생인 <a class="mw-redirect" href="/wiki/%EB%B0%B1%EC%A0%9C_%EB%B9%84%EB%A5%98%EC%99%95" title="백제 비류왕">비류왕</a>(比流王)이 왕위에 올랐으며, <a href="/wiki/%EB%B9%84%EB%A5%98%EC%99%95" title="비류왕">비류왕</a>이 사망한 후에 왕위에 올랐다. 왕위에 오른지 2년만에 사망하였으며 <a href="/wiki/%EB%B9%84%EB%A5%98%EC%99%95" title="비류왕">비류왕</a>의 아들 <a href="/wiki/%EA%B7%BC%EC%B4%88%EA%B3%A0%EC%99%95" title="근초고왕">근초고왕</a>이 즉위하였다.</p>

<h2><span class="mw-headline" id=".EC.83.9D.EC.95.A0">생애</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EA%B3%84%EC%99%95&amp;action=edit&amp;section=1" title="부분 편집: 생애">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>재위 <a href="/wiki/344%EB%85%84" title="344년">344년</a>∼<a href="/wiki/346%EB%85%84" title="346년">346년</a> 제10대 <a href="/wiki/%EB%B6%84%EC%84%9C%EC%99%95" title="분서왕">분서왕</a>의 맏아들이다. 그런데 <a href="/wiki/%EB%B6%84%EC%84%9C%EC%99%95" title="분서왕">분서왕</a>의 뒤를 이어 제11대 왕이 된 것은 왕실 내의 다른 지파 출신인 <a href="/wiki/%EB%B9%84%EB%A5%98%EC%99%95" title="비류왕">비류왕</a>이며 그 이유를 <a href="/wiki/%EC%82%BC%EA%B5%AD%EC%82%AC%EA%B8%B0" title="삼국사기">삼국사기</a>에는 당시에 그가 어렸기 때문이라 하였으나 그것만이 이유의 전부로는 생각되지 않는다. <a href="/wiki/%EB%B6%84%EC%84%9C%EC%99%95" title="분서왕">분서왕</a>과 그는 <a href="/wiki/%EA%B3%A0%EC%9D%B4%EC%99%95" title="고이왕">고이왕</a>의 후손으로서 초고왕의 후손인 <a href="/wiki/%EB%B9%84%EB%A5%98%EC%99%95" title="비류왕">비류왕</a>과는 왕실 내에서 서로 다른 지파였다. 그리고 <a href="/wiki/%EB%B9%84%EB%A5%98%EC%99%95" title="비류왕">비류왕</a>의 즉위는 <a href="/wiki/%EB%B6%84%EC%84%9C%EC%99%95" title="분서왕">분서왕</a>이 낙랑의 자객에게 피살된 상태에서 이루어졌다. 그 뒤에는 <a href="/wiki/%EA%B3%A0%EC%9D%B4%EC%99%95" title="고이왕">고이왕</a>계가 몰락하고 <a href="/wiki/%EC%B4%88%EA%B3%A0%EC%99%95" title="초고왕">초고왕</a>계가 집권하는 왕실지파간의 세력교체가 있었다. 그 사이에 재위한 <a class="mw-redirect" href="/wiki/%EB%B0%B1%EC%A0%9C_%EA%B3%84%EC%99%95" title="백제 계왕">계왕</a>의 통치기간은 2년 정도에 불과하다. 이러한 사실들은 경쟁세력들과의 관계에서 <a class="mw-redirect" href="/wiki/%EB%B0%B1%EC%A0%9C_%EA%B3%84%EC%99%95" title="백제 계왕">계왕</a>의 즉위나 집권이 순탄하지 않았음을 암시해준다.</p>
