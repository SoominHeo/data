<h1 class="firstHeading" id="firstHeading" lang="ko">계차수열</h1>
<div class="mw-body-content" id="bodyContent">
<div id="siteSub">위키백과, 우리 모두의 백과사전.</div>
<div id="contentSub"></div>
<div class="mw-jump" id="jump-to-nav">
					이동:					<a href="#mw-head">둘러보기</a>, 					<a href="#p-search">검색</a>
</div>
<div class="mw-content-ltr" dir="ltr" id="mw-content-text" lang="ko"><script>function mfTempOpenSection(id){var block=document.getElementById("mf-section-"+id);block.className+=" open-block";block.previousSibling.className+=" open-block";}</script><p><a href="/wiki/%EC%88%98%ED%95%99" title="수학">수학</a>에서, <a href="/wiki/%EC%88%98%EC%97%B4" title="수열">수열</a>의 <b>계차수열</b>(階差數列)은 그 수열의 인접하는 두 항의 차로 이루어지는 수열이다. 예를 들어 수열</p>
<dl>
<dd><span class="texhtml" style="font-family: 'Nimbus Roman No9 L', 'Times New Roman', Times, serif; white-space: nowrap; line-height: 1; font-size: 118%;">1, 4, 9, 16, ... , <i>n</i><sup>2</sup>, ...</span></dd>
</dl>
<p>의 계차수열은</p>
<dl>
<dd><span class="texhtml" style="font-family: 'Nimbus Roman No9 L', 'Times New Roman', Times, serif; white-space: nowrap; line-height: 1; font-size: 118%;">3, 5, 7, ... , 2<i>n</i> + 1, ...</span></dd>
</dl>
<p>과 같다. 수열 <span class="texhtml" style="font-family: 'Nimbus Roman No9 L', 'Times New Roman', Times, serif; white-space: nowrap; line-height: 1; font-size: 118%;">{<i>a<sub>n</sub></i>}</span>의 계차수열의 <a class="mw-redirect" href="/wiki/%EC%9D%BC%EB%B0%98%ED%95%AD" title="일반항">일반항</a>은 <span class="texhtml" style="font-family: 'Nimbus Roman No9 L', 'Times New Roman', Times, serif; white-space: nowrap; line-height: 1; font-size: 118%;"><i>a</i><sub><i>n</i>+1</sub> - <i>a<sub>n</sub></i></span>이다.</p>
<p></p>

<li>등차수열 <span class="texhtml" style="font-family: 'Nimbus Roman No9 L', 'Times New Roman', Times, serif; white-space: nowrap; line-height: 1; font-size: 118%;"><i>a<sub>n</sub></i> = <i>pn</i> + <i>q</i></span>의 계차수열은 <a class="new" href="/w/index.php?title=%EC%83%81%EC%88%98%EC%97%B4&amp;action=edit&amp;redlink=1" title="상수열 (없는 문서)">상수열</a> <span class="texhtml" style="font-family: 'Nimbus Roman No9 L', 'Times New Roman', Times, serif; white-space: nowrap; line-height: 1; font-size: 118%;">Δ<i>a<sub>n</sub></i> = <i>p</i></span>이다. 특별히, 상수열 <span class="texhtml" style="font-family: 'Nimbus Roman No9 L', 'Times New Roman', Times, serif; white-space: nowrap; line-height: 1; font-size: 118%;"><i>a<sub>n</sub></i> = <i>c</i></span>의 계차수열은 영수열 <span class="texhtml" style="font-family: 'Nimbus Roman No9 L', 'Times New Roman', Times, serif; white-space: nowrap; line-height: 1; font-size: 118%;">Δ<i>a<sub>n</sub></i> = 0</span>이다.</li>
<li><a class="mw-redirect" href="/wiki/%EC%A1%B0%ED%99%94%EC%88%98%EC%97%B4" title="조화수열">조화수열</a> <span class="texhtml" style="font-family: 'Nimbus Roman No9 L', 'Times New Roman', Times, serif; white-space: nowrap; line-height: 1; font-size: 118%;"><i>a<sub>n</sub></i> = <span class="sfrac nowrap" style="display:inline-block; vertical-align:-0.5em; font-size:85%; text-align:center;"><span style="display:block; line-height:1em; padding:0 0.1em;">1</span><span style="position:absolute;left:-10000px;top:auto;width:1px;height:1px;overflow:hidden;">/</span><span style="display:block; line-height:1em; padding:0 0.1em; border-top:1px solid;"><i>pn</i> + <i>q</i></span></span></span>의 계차수열은 <span class="texhtml" style="font-family: 'Nimbus Roman No9 L', 'Times New Roman', Times, serif; white-space: nowrap; line-height: 1; font-size: 118%;">Δ<i>a<sub>n</sub></i> = <span class="sfrac nowrap" style="display:inline-block; vertical-align:-0.5em; font-size:85%; text-align:center;"><span style="display:block; line-height:1em; padding:0 0.1em;"><i>p</i></span><span style="position:absolute;left:-10000px;top:auto;width:1px;height:1px;overflow:hidden;">/</span><span style="display:block; line-height:1em; padding:0 0.1em; border-top:1px solid;">(<i>pn</i> + <i>q</i>)(<i>pn</i> + <i>p</i> + <i>q</i>)</span></span></span>이다.</li>
<li>주어진 수열 <span class="texhtml mvar" style="font-family: 'Nimbus Roman No9 L', 'Times New Roman', Times, serif; font-size: 118%; white-space: nowrap; font-style: italic;">a<sub>n</sub></span>의 합 <span class="texhtml" style="font-family: 'Nimbus Roman No9 L', 'Times New Roman', Times, serif; white-space: nowrap; line-height: 1; font-size: 118%;"><i>S<sub>n</sub></i> = <i>a<sub>1</sub></i> + … + <i>a<sub>n</sub></i></span>의 계차수열은 <span class="texhtml" style="font-family: 'Nimbus Roman No9 L', 'Times New Roman', Times, serif; white-space: nowrap; line-height: 1; font-size: 118%;"><i>a<sub>2</sub></i>, <i>a<sub>3</sub></i>, <i>a<sub>4</sub></i>, ...</span>이다.</li>
<li><a class="mw-redirect" href="/wiki/%EB%8B%A4%ED%95%AD%EC%8B%9D%EC%9D%98_%EC%B0%A8%EC%88%98" title="다항식의 차수">3차</a> <a href="/wiki/%EB%8B%A4%ED%95%AD%EC%8B%9D" title="다항식">다항식</a>인 <span class="texhtml" style="font-family: 'Nimbus Roman No9 L', 'Times New Roman', Times, serif; white-space: nowrap; line-height: 1; font-size: 118%;"><i>n</i><sup>3</sup></span>의 1, 2, 3계 차수열은 각각 <span class="texhtml" style="font-family: 'Nimbus Roman No9 L', 'Times New Roman', Times, serif; white-space: nowrap; line-height: 1; font-size: 118%;">3<i>n</i><sup>2</sup> + 3<i>n</i> + 1</span>, <span class="texhtml" style="font-family: 'Nimbus Roman No9 L', 'Times New Roman', Times, serif; white-space: nowrap; line-height: 1; font-size: 118%;">6<i>n</i> + 6</span>, <span class="texhtml" style="font-family: 'Nimbus Roman No9 L', 'Times New Roman', Times, serif; white-space: nowrap; line-height: 1; font-size: 118%;">6</span>이며, 이들은 각각 2차, 1차, 0차 다항식이다.</li>
</ul>

<li><a class="mw-redirect" href="/wiki/%EB%8B%A8%EC%A1%B0%EC%88%98%EC%97%B4" title="단조수열">수열의 단조성</a>은 계차수열을 이용해 기술할 수 있다. 수열 <span class="texhtml" style="font-family: 'Nimbus Roman No9 L', 'Times New Roman', Times, serif; white-space: nowrap; line-height: 1; font-size: 118%;">{<i>a<sub>n</sub></i>}</span>이 <a class="mw-redirect" href="/wiki/%EB%8B%A8%EC%A1%B0%EC%A6%9D%EA%B0%80" title="단조증가">단조증가</a>할 <a href="/wiki/%ED%95%84%EC%9A%94%EC%B6%A9%EB%B6%84%EC%A1%B0%EA%B1%B4" title="필요충분조건">필요충분조건</a>은 <span class="texhtml" style="font-family: 'Nimbus Roman No9 L', 'Times New Roman', Times, serif; white-space: nowrap; line-height: 1; font-size: 118%;">Δ<i>a<sub>n</sub></i> ≥ 0</span>이 모든 <span class="texhtml mvar" style="font-family: 'Nimbus Roman No9 L', 'Times New Roman', Times, serif; font-size: 118%; white-space: nowrap; font-style: italic;">n</span>에게 성립하는 것이다. 수열 <span class="texhtml" style="font-family: 'Nimbus Roman No9 L', 'Times New Roman', Times, serif; white-space: nowrap; line-height: 1; font-size: 118%;">{<i>a<sub>n</sub></i>}</span>이 <a class="mw-redirect" href="/wiki/%EB%8B%A8%EC%A1%B0%EA%B0%90%EC%86%8C" title="단조감소">단조감소</a>할 필요충분조건은, <span class="texhtml" style="font-family: 'Nimbus Roman No9 L', 'Times New Roman', Times, serif; white-space: nowrap; line-height: 1; font-size: 118%;">Δ<i>a<sub>n</sub></i> ≤ 0</span>이 모든 <span class="texhtml mvar" style="font-family: 'Nimbus Roman No9 L', 'Times New Roman', Times, serif; font-size: 118%; white-space: nowrap; font-style: italic;">n</span>에게 성립하는 것이다.</li>

<h2><span class="mw-headline" id=".EA.B3.A0.EA.B3.84_.EB.93.B1.EC.B0.A8.EC.88.98.EC.97.B4">고계 등차수열</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EA%B3%84%EC%B0%A8%EC%88%98%EC%97%B4&amp;action=edit&amp;section=5" title="부분 편집: 고계 등차수열">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p><b><span class="texhtml mvar" style="font-family: 'Nimbus Roman No9 L', 'Times New Roman', Times, serif; font-size: 118%; white-space: nowrap; font-style: italic;">m</span>계 등차수열</b><span class="texhtml" style="font-family: 'Nimbus Roman No9 L', 'Times New Roman', Times, serif; white-space: nowrap; line-height: 1; font-size: 118%;">(<i>m</i> ≥ 0)</span>은 다음과 같이 귀납적으로 정의된다.</p>
<ul>
<li>0이 아닌 상수의 수열은 0계 등차수열이다.</li>
<li>계차수열이 <span class="texhtml" style="font-family: 'Nimbus Roman No9 L', 'Times New Roman', Times, serif; white-space: nowrap; line-height: 1; font-size: 118%;">(<i>k</i> - 1)</span>계 등차수열인 수열은 <span class="texhtml mvar" style="font-family: 'Nimbus Roman No9 L', 'Times New Roman', Times, serif; font-size: 118%; white-space: nowrap; font-style: italic;">k</span>계 등차수열이다.</li>
</ul>
<p>위의 예시 문단에서, 수열 <span class="texhtml" style="font-family: 'Nimbus Roman No9 L', 'Times New Roman', Times, serif; white-space: nowrap; line-height: 1; font-size: 118%;">{6, 6, ...}</span>은 0계 등차수열이며, 그 수열을 계차수열로 하는 수열인 <span class="texhtml" style="font-family: 'Nimbus Roman No9 L', 'Times New Roman', Times, serif; white-space: nowrap; line-height: 1; font-size: 118%;">{6<i>n</i> - 6}</span>은 1계 등차수열이다. 마찬가지로 <span class="texhtml" style="font-family: 'Nimbus Roman No9 L', 'Times New Roman', Times, serif; white-space: nowrap; line-height: 1; font-size: 118%;">{3<i>n</i><sup>2</sup> + 3<i>n</i> + 1}</span>은 2계 등차수열, <span class="texhtml" style="font-family: 'Nimbus Roman No9 L', 'Times New Roman', Times, serif; white-space: nowrap; line-height: 1; font-size: 118%;">{<i>n</i><sup>3</sup>}</span>은 3계 등차수열이다.</p>
<p>어떤 수열 <span class="texhtml" style="font-family: 'Nimbus Roman No9 L', 'Times New Roman', Times, serif; white-space: nowrap; line-height: 1; font-size: 118%;">{<i>a<sub>n</sub></i>}</span>이 <span class="texhtml mvar" style="font-family: 'Nimbus Roman No9 L', 'Times New Roman', Times, serif; font-size: 118%; white-space: nowrap; font-style: italic;">m</span>계 등차수열일 필요충분조건은, 일반항이 <span class="texhtml mvar" style="font-family: 'Nimbus Roman No9 L', 'Times New Roman', Times, serif; font-size: 118%; white-space: nowrap; font-style: italic;">n</span>에 대한 <a href="/wiki/%EB%8B%A4%ED%95%AD%EC%8B%9D" title="다항식"><span class="texhtml mvar" style="font-family: 'Nimbus Roman No9 L', 'Times New Roman', Times, serif; font-size: 118%; white-space: nowrap; font-style: italic;">m</span>차 다항식</a>이라는 것이다.<sup class="reference" id="cite_ref-WQ_1-3"><a href="#cite_note-WQ-1">[1]</a></sup></p>
