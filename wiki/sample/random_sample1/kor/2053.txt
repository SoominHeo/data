<p><b>계통수</b>(系統樹, phylogenetic tree)는 <a href="/wiki/%EC%83%9D%EB%AC%BC" title="생물">생물</a>이 <a href="/wiki/%EC%A7%84%ED%99%94" title="진화">진화</a>의 결과 여러 <a href="/wiki/%EC%A2%85_(%EC%83%9D%EB%AC%BC%ED%95%99)" title="종 (생물학)">종</a> 이나 <a href="/wiki/%EB%B6%84%EB%A5%98%EA%B5%B0" title="분류군">분류군</a> 사이에서 나타나는 <a href="/wiki/%ED%95%B4%EB%B6%80%ED%95%99" title="해부학">신체적</a>이거나 <a href="/wiki/%EC%9C%A0%EC%A0%84" title="유전">유전적</a> 특징의 유사성과 차이를 바탕으로 친연 관계를 <a href="/wiki/%EC%88%98%ED%98%95%EB%8F%84" title="수형도">수형도</a>로 나타낸 <a href="/wiki/%EB%8B%A4%EC%9D%B4%EC%96%B4%EA%B7%B8%EB%9E%A8" title="다이어그램">다이어그램</a>이다.<sup class="reference" id="cite_ref-1"><a href="#cite_note-1">[1]</a></sup></p>
<p>계통수를 구성할 때에는 보통 생물의 <a href="/wiki/%EC%83%81%EB%8F%99%EC%84%B1" title="상동성">상동성</a>을 기반으로 해당하는 모든 생물 분류군의 진화적 변인을 최소한으로 가정하는 최절약 원리(<a href="/wiki/%EC%98%A4%EC%BB%B4%EC%9D%98_%EB%A9%B4%EB%8F%84%EB%82%A0" title="오컴의 면도날">오컴의 면도날</a>)와 분자적 유전 정보 자료를 바탕으로 <a href="/wiki/%EB%89%B4%ED%81%B4%EB%A0%88%EC%98%A4%ED%83%80%EC%9D%B4%EB%93%9C" title="뉴클레오타이드">뉴클레오타이드</a>의 치환 정도를 컴퓨터 프로그램을 통해 분석하는 최우법이 사용된다.<sup class="reference" id="cite_ref-2"><a href="#cite_note-2">[2]</a></sup></p>
