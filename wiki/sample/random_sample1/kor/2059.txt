<h1 class="firstHeading" id="firstHeading" lang="ko">고고학</h1>
<div class="mw-body-content" id="bodyContent">
<div id="siteSub">위키백과, 우리 모두의 백과사전.</div>
<div id="contentSub"></div>
<div class="mw-jump" id="jump-to-nav">
					이동:					<a href="#mw-head">둘러보기</a>, 					<a href="#p-search">검색</a>
</div>
<div class="mw-content-ltr" dir="ltr" id="mw-content-text" lang="ko"><script>function mfTempOpenSection(id){var block=document.getElementById("mf-section-"+id);block.className+=" open-block";block.previousSibling.className+=" open-block";}</script><p><b>고고학</b>(考古學, Archaeology)은 물질과 동식물, 인류가 지난 시대에 남긴 흔적을 찾아내고 이들의 <i>"말없는 역사"</i>를 밝히는 학문으로 <a href="/wiki/%EC%82%AC%ED%9A%8C%EA%B3%BC%ED%95%99" title="사회과학">사회과학</a>의 일종이다.</p>
<p>인도 유럽어 <i>Archaeology, Archéologie, Archäologie</i> 등의 여러 낱말은 고대 희랍어의 "아르햐이올로기아" (<i>arxaiologia</i>, <i>arxaios</i>: "오래된, 원천의" <i>logos</i>: "학")에 유래한다. "말없는 역사"라는 표현에서 짐작하듯이 고고학은 <a href="/wiki/%EC%97%AD%EC%82%AC" title="역사">역사</a>와 매우 밀접하게 관계한다. 물질과 동식물, 인류가 남긴 흔적과 사건을 기록하고 이를 가능한 한 객관으로 해석해 역사 연구의 뜻이 있다면, 고고학은 사라졌거나 쉽게 주변에서 보기 어려운 과거 인류의 흔적을 추적하고 발굴하여 지난 시대 인류의 문화와 문명 이해에 초점을 두므로 <a class="mw-redirect" href="/wiki/%EC%97%AD%EC%82%AC%EC%8B%9C%EB%8C%80" title="역사시대">역사시대</a>뿐만이 아니라 인류가 살았던 장소 바로 그 곳에서 고고학 연구의 첫 장이 씐다. 예컨대 <a href="/wiki/%EC%8B%A0%EB%9D%BC" title="신라">신라</a> 문화와 그 시대 생활상을 고고학상으로 밝히려면 신라인이 살았던 대한민국을 탐사하고 그 유적지와 주거지를 문자 그대로 "파헤쳐" 보는 수밖에 없다. 고고학에 내재한 이런 특수한 전제조건으로 말미암아 고고학은 수시로 지방과 문명을 좇아 나름대로 연구 분야를 개척하고 발전시키게 되었다. 즉 고대 희랍과 로마의 <a class="new" href="/w/index.php?title=%EA%B3%A0%EC%A0%84_%EA%B3%A0%EA%B3%A0%ED%95%99&amp;action=edit&amp;redlink=1" title="고전 고고학 (없는 문서)">고전 고고학</a>, <a class="mw-redirect" href="/wiki/%EC%9D%B4%EC%A7%91%ED%8A%B8_%EA%B3%A0%EA%B3%A0%ED%95%99" title="이집트 고고학">이집트 고고학</a>(→ <a class="mw-redirect" href="/wiki/%EC%9D%B4%EC%A7%91%ED%8A%B8_%EB%AC%B8%EB%AA%85" title="이집트 문명">이집트 문명</a>), <a class="mw-redirect" href="/wiki/%EC%84%B1%EC%84%9C%EA%B3%A0%EA%B3%A0%ED%95%99" title="성서고고학">기독교 고고학</a>은 고고학 분야 중에서 독립되고 가장 발달된 분야다. 고고학과 관련된 학문으로서는 <a href="/wiki/%EC%97%AD%EC%82%AC%ED%95%99" title="역사학">역사학</a>, <a href="/wiki/%EC%A7%80%EB%A6%AC%ED%95%99" title="지리학">지리학</a>, <a href="/wiki/%EC%A7%80%EC%A7%88%ED%95%99" title="지질학">지질학</a>, <a href="/wiki/%EC%83%9D%EB%AC%BC%ED%95%99" title="생물학">생물학</a>, <a href="/wiki/%EA%B1%B4%EC%B6%95%ED%95%99" title="건축학">건축학</a>, <a href="/wiki/%EC%9D%8C%EC%95%85%EC%82%AC%ED%95%99" title="음악사학">음악사학</a>, <a class="mw-redirect" href="/wiki/%EB%AF%B8%EC%88%A0%EC%82%AC%ED%95%99" title="미술사학">미술사학</a>, <a href="/wiki/%EC%9D%B8%EB%A5%98%ED%95%99" title="인류학">인류학</a>을 손꼽는다.</p>
<p>그러나 고고학은 단지 <a href="/wiki/%EC%97%AD%EC%82%AC" title="역사">역사</a>의 물증을 발견하는 학문이 아니다. 고고학은 인간의 흔적 뿐만 아니라 물질, 동물과 식물, 지질의 흔적도 다루기 때문이다. <a class="new" href="/w/index.php?title=%EC%83%9D%EB%AC%BC%EA%B3%A0%EA%B3%A0%ED%95%99&amp;action=edit&amp;redlink=1" title="생물고고학 (없는 문서)">생물고고학</a>, <a class="new" href="/w/index.php?title=%EC%A7%80%EC%A7%88%EA%B3%A0%EA%B3%A0%ED%95%99&amp;action=edit&amp;redlink=1" title="지질고고학 (없는 문서)">지질고고학</a> 등이 하위분야에 속하면서 <strong class="selflink">고고학</strong>이 <a href="/wiki/%EC%97%AD%EC%82%AC%ED%95%99" title="역사학">역사학</a>, 혹은 <a href="/wiki/%EC%9D%B8%EB%AC%B8%ED%95%99" title="인문학">인문학</a>에 속하지 않음을 증명한다.</p>
