<p><b>고대 대불가리아</b>(Old Great Bulgaria) 혹은 <b>대불가리아</b>(Great Bulgaria, <a href="/wiki/%EB%B9%84%EC%9E%94%ED%8B%B0%EC%9B%80" title="비잔티움">비잔티움</a>의 연대기에서는 Παλαιά Μεγάλη Βουλγαρία)는 비잔티움 제국의 역사서에 등장하는 용어로써 7세기 <a href="/wiki/%EB%B3%BC%EA%B0%80_%EA%B0%95" title="볼가 강">볼가 강</a>(Volga)<sup class="reference" id="cite_ref-1"><a href="#cite_note-1">[1]</a></sup> 하류에서 <a href="/wiki/%EB%93%9C%EB%84%A4%EC%8A%A4%ED%8A%B8%EB%A5%B4_%EA%B0%95" title="드네스트르 강">드네스트르 강</a>(Dniester)까지 범위의 <a href="/wiki/%EC%BA%85%EC%B9%B4%EC%8A%A4_%EC%82%B0%EB%A7%A5" title="캅카스 산맥">캅카스 산맥</a> 북부, <a class="mw-redirect" href="/wiki/%EC%8A%A4%ED%85%9D_(%ED%8F%89%EC%95%BC)" title="스텝 (평야)">스텝</a> 지대를 지배했던 <a href="/wiki/%EB%B6%88%EA%B0%80%EB%A5%B4%EC%A1%B1" title="불가르족">불가르</a>(Bulgar)의 군주 <a class="new" href="/w/index.php?title=%EC%BF%A0%EB%B8%8C%EB%9D%BC%ED%8A%B8&amp;action=edit&amp;redlink=1" title="쿠브라트 (없는 문서)">쿠브라트</a>(Kubrat)의 치세를 가리켜 나타낸다. <a class="new" href="/w/index.php?title=%EC%98%A4%EB%85%B8%EA%B5%AC%EB%A6%AC%EC%95%84&amp;action=edit&amp;redlink=1" title="오노구리아 (없는 문서)">오노구리아</a>(Onoguria)라고도 부른다.</p>

<p>쿠브라트{쿠르트(Kurt) 혹은 후부라트(Houvrat)라고도 한다.}는 유력씨족이었던 두로 가문(Dulo clan)<sup class="reference" id="cite_ref-2"><a href="#cite_note-2">[2]</a></sup> 의 출신으로 불가르인의 지도자로써 정당한 계승권을 갖고 있었다. 쿠브라트의 부모 <a class="new" href="/w/index.php?title=%EC%98%AC%EA%B0%80%EB%82%98&amp;action=edit&amp;redlink=1" title="올가나 (없는 문서)">올가나</a>(Organa)가 불가르족의 섭정을 맡고 있던 동안 쿠브라트는 <a href="/wiki/%EB%B9%84%EC%9E%94%ED%8B%B0%EC%9B%80_%EC%A0%9C%EA%B5%AD" title="비잔티움 제국">비잔티움 제국</a>에서 젊은 시절을 보내고, 교육과 세례를 받았다.<sup class="reference" id="cite_ref-3"><a href="#cite_note-3">[3]</a></sup></p>
<p>618년쯤 쿠브라트는 고향에 돌아와 얼마안가 아바르(Alan)인의 <a href="/wiki/%EC%B9%B8_(%EC%B9%AD%ED%98%B8)" title="칸 (칭호)">칸</a>(Khagan)의 승인을 얻어 불가르인의 지도자가 되었다. 이후 쿠브라트는 아바르인에 의한 지배에서 벗어나길 기도했고, <a href="/wiki/%EC%84%9C%EB%8F%8C%EA%B6%90" title="서돌궐">서돌궐</a>에게도 벗어났다(이것은 왕조 전쟁에 얽혀서 였다).</p>

<h2><span class="mw-headline" id=".EC.84.B1.EB.A6.BD">성립</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EA%B3%A0%EB%8C%80_%EB%8C%80%EB%B6%88%EA%B0%80%EB%A6%AC%EC%95%84&amp;action=edit&amp;section=2" title="부분 편집: 성립">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>610년부터 613년까지의 기간동안 쿠브라트 칸은 2개의 주요 불가르인의 씨족인 <a class="new" href="/w/index.php?title=%EC%BF%A0%ED%8A%B8%EB%A6%B4%EA%B8%80&amp;action=edit&amp;redlink=1" title="쿠트릴글 (없는 문서)">쿠트릴글</a>(Kutrigur)와 <a class="new" href="/w/index.php?title=%EC%9A%B0%ED%8B%B0%EA%B8%80&amp;action=edit&amp;redlink=1" title="우티글 (없는 문서)">우티글</a>(Utigur)의 통일을 노렸고, 강대한 연합국가를 만들어내 <a href="/wiki/%EC%A4%91%EC%84%B8" title="중세">중세</a>의 저자들로부터 <i>고대 대불가리아</i> <sup class="reference" id="cite_ref-ReferenceA_4-0"><a href="#cite_note-ReferenceA-4">[4]</a></sup> 혹은 <a class="new" href="/w/index.php?title=%EC%98%A4%EB%85%B8%EA%B5%B0%EB%91%90%EB%A5%B4&amp;action=edit&amp;redlink=1" title="오노군두르 (없는 문서)">오노군두르</a>(Onogundur)-불가르 제국(서유럽에서는 <i>오노그 후리아</i>(Onoghuria)<sup class="reference" id="cite_ref-5"><a href="#cite_note-5">[5]</a></sup>)로써 기록되었다. 쿠브라트의 영향하에 있던 <a class="new" href="/w/index.php?title=%EC%95%84%EB%B0%94%EB%A5%B4&amp;action=edit&amp;redlink=1" title="아바르 (없는 문서)">아바르</a>인도 여기에 포함되었고, 따라서 <a href="/wiki/%ED%8C%90%EB%85%B8%EB%8B%88%EC%95%84_%ED%8F%89%EC%9B%90" title="판노니아 평원">판노니아 평원</a>(Pannonian Plain)까지 영향력이 미쳤을 것이라고 생각하는 사람도 있다. 대불가리아의 중심지는 크라스노달 지방의 <a href="/wiki/%ED%83%80%EB%A7%8C_%EB%B0%98%EB%8F%84" title="타만 반도">타만 반도</a>에 있던 <a class="new" href="/w/index.php?title=%ED%8C%8C%EB%82%98%EA%B3%A0%EB%A6%AC%EC%95%84&amp;action=edit&amp;redlink=1" title="파나고리아 (없는 문서)">파나고리아</a>(Phanagoria)였다. 쿠브라트의 묘는 1912년 <a href="/wiki/%EC%9A%B0%ED%81%AC%EB%9D%BC%EC%9D%B4%EB%82%98" title="우크라이나">우크라이나</a>의 <a href="/wiki/%ED%8F%B4%ED%83%80%EB%B0%94" title="폴타바">폴타바</a> 근처의 <a class="new" href="/w/index.php?title=%ED%8E%98%EB%A0%88%EC%8B%9C%EC%B2%B4%EB%A6%AC%EB%82%98&amp;action=edit&amp;redlink=1" title="페레시체리나 (없는 문서)">페레시체리나</a>(Pereshchepina)에서 발견되었다<sup class="reference" id="cite_ref-6"><a href="#cite_note-6">[6]</a></sup>.</p>
<h2><span class="mw-headline" id=".EB.B6.95.EA.B4.B4.EC.99.80_.ED.9B.84.EB.8C.80_.EA.B5.AD.EA.B0.80">붕괴와 후대 국가</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EA%B3%A0%EB%8C%80_%EB%8C%80%EB%B6%88%EA%B0%80%EB%A6%AC%EC%95%84&amp;action=edit&amp;section=3" title="부분 편집: 붕괴와 후대 국가">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>쿠브라트의 죽음에 이어서 벌어진 사건에 대해서는 비잔티움 총주교의 <a href="/wiki/%EB%8B%88%EC%BC%80%ED%8F%AC%EB%A3%A8%EC%8A%A4_1%EC%84%B8" title="니케포루스 1세">니케포루스 1세</a>(Nicephorus 1)<sup class="reference" id="cite_ref-ReferenceA_4-1"><a href="#cite_note-ReferenceA-4">[4]</a></sup>. 가 기술하였다. 기술에 의하면 비잔티움 황제 <a href="/wiki/%EC%BD%98%EC%8A%A4%ED%83%84%ED%8B%B0%EB%88%84%EC%8A%A4_4%EC%84%B8" title="콘스탄티누스 4세">콘스탄티누스 4세</a>(Constantine IV)의 시대에 쿠브라트는 죽었고, 그의 5명의 아들들 중 최연장자였던 <a class="new" href="/w/index.php?title=%EB%B0%94%ED%8A%B8%EB%B0%94%EC%96%80&amp;action=edit&amp;redlink=1" title="바트바얀 (없는 문서)">바트바얀</a>(Batbayan)은 그대로 나라에 머물렀다. <a class="mw-redirect" href="/wiki/%ED%95%98%EC%9E%90%EB%A5%B4" title="하자르">하자르</a>(Khazar)에 의한 강력한 압력 때문에 쿠브라트의 다른 아들들은 '적에게 맞써기 위해서 결속하거라' 라는 아버지의 가르침에 반하여 각자 이반해 자신의 씨족을 이끌게 되었다.</p>
<h3><span class="mw-headline" id=".EB.B3.BC.EA.B0.80_.EB.B6.88.EA.B0.80.EB.A5.B4">볼가 불가르</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EA%B3%A0%EB%8C%80_%EB%8C%80%EB%B6%88%EA%B0%80%EB%A6%AC%EC%95%84&amp;action=edit&amp;section=4" title="부분 편집: 볼가 불가르">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p><a class="new" href="/w/index.php?title=%EC%BD%94%ED%8A%B8%EB%A0%88%EA%B7%B8&amp;action=edit&amp;redlink=1" title="코트레그 (없는 문서)">코트레그</a>(Kotrag)는 쿠트리글(Kutrigurs) 혹은 코트라그(Kotrags)의 지도자가 되어 볼가강 중류로 돌아가 후에 볼가 강과 <a href="/wiki/%EC%B9%B4%EB%A7%88_%EA%B0%95" title="카마 강">카마 강</a>(Kama)의 합류지점에 <a class="new" href="/w/index.php?title=%EB%B3%BC%EA%B0%80-%EB%B6%88%EA%B0%80%EB%A6%AC%EC%95%84&amp;action=edit&amp;redlink=1" title="볼가-불가리아 (없는 문서)">볼가-불가리아</a>(Volga Bulgaria)를 세웠다. 볼가 불가리아는 이윽고 강대한 나라가 되었다. <a class="new" href="/w/index.php?title=%EB%B3%BC%EA%B0%80-%EB%B6%88%EA%B0%80%EB%A5%B4&amp;action=edit&amp;redlink=1" title="볼가-불가르 (없는 문서)">볼가-불가르</a>{혹은 은의 불가르(Silver Bulgars)라고도 부른다.}는 9세기에는 자발적으로 <a href="/wiki/%EC%9D%B4%EC%8A%AC%EB%9E%8C%EA%B5%90" title="이슬람교">이슬람교</a>로 개종해 13세기까지 자신의 민족의식을 유지해 1233년 <a class="mw-redirect" href="/wiki/%EC%B9%AD%EA%B8%B0%EC%8A%A4_%EC%B9%B8" title="칭기스 칸">칭기스 칸</a>의 <a href="/wiki/%EB%AA%BD%EA%B3%A8" title="몽골">몽골</a>의 침입을 물리쳤다. 그렇지만 그들은 그 후 얼마안가 제압되어 중심도시였던 <a href="/wiki/%EB%B3%BC%EA%B0%80%EB%A5%B4" title="볼가르">볼가르</a>(Bolghar)는 <a class="mw-redirect" href="/wiki/%ED%82%B5%EC%B0%A8%ED%81%AC_%ED%95%9C%EA%B5%AD" title="킵차크 한국">황금 유목민</a>의 수도가 되었다. 그 후 불가르인은 <a href="/wiki/%ED%83%80%ED%83%80%EB%A5%B4%EC%A1%B1" title="타타르족">타타르족</a>(Tatars)과 혼혈이 진행되었다. <a href="/wiki/%EB%9F%AC%EC%8B%9C%EC%95%84" title="러시아">러시아</a> 연방의 <a href="/wiki/%ED%83%80%ED%83%80%EB%A5%B4_%EA%B3%B5%ED%99%94%EA%B5%AD" title="타타르 공화국">타타르스탄</a>(Tatarstan)과 <a href="/wiki/%EC%B6%94%EB%B0%94%EC%8B%9C_%EA%B3%B5%ED%99%94%EA%B5%AD" title="추바시 공화국">추바시아</a>(Chuvashia) 시민은 이들 불가르인의 자손이라고 볼 수 있었다.</p>
<h3><span class="mw-headline" id=".EB.A7.88.EC.BC.80.EB.8F.84.EB.8B.88.EC.95.84.EC.9D.98_.EB.B6.88.EA.B0.80.EB.A5.B4">마케도니아의 불가르</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EA%B3%A0%EB%8C%80_%EB%8C%80%EB%B6%88%EA%B0%80%EB%A6%AC%EC%95%84&amp;action=edit&amp;section=5" title="부분 편집: 마케도니아의 불가르">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p><a class="new" href="/w/index.php?title=%EC%BF%A0%EB%B2%A0%EB%A5%B4&amp;action=edit&amp;redlink=1" title="쿠베르 (없는 문서)">쿠베르</a>(Kuber)는 <a class="new" href="/w/index.php?title=%EC%8B%9C%EB%A5%B4%EB%AF%B8%EC%9B%80&amp;action=edit&amp;redlink=1" title="시르미움 (없는 문서)">시르미움</a>(Sirmium)을 지배했다. 그 땅에는 불가르인, 로마인, 그리스인, 슬라브인, 게르만인 등이 혼재했고, 아바르 칸국의 속령이었다. 반란이 있은 후, 쿠베르는 사람들을 <a class="mw-redirect" href="/wiki/%EB%A7%88%EC%BC%80%EB%8F%84%EB%8B%88%EC%95%84_%EC%A7%80%EB%B0%A9" title="마케도니아 지방">마케도니아</a>로 이주시켰다. 여기서 쿠베르가 이끌었던 사람들은 케르미시아(Keremisia)에 정착해 <a href="/wiki/%ED%85%8C%EC%82%B4%EB%A1%9C%EB%8B%88%ED%82%A4" title="테살로니키">테살로니키</a>(Thessaloniki) 정복을 시도했으나 실패로 끝났다. 그 후 쿠베르는 역사상에서 모습을 감추게 되었고, 그를 따랐던 사람들은 마케도니아 지방의 슬라브인들과 동화되었다.</p>
<h3><span class="mw-headline" id=".EB.82.A8.EC.9D.B4.ED.83.88.EB.A6.AC.EC.95.84.EC.9D.98_.EB.B6.88.EA.B0.80.EB.A5.B4">남이탈리아의 불가르</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EA%B3%A0%EB%8C%80_%EB%8C%80%EB%B6%88%EA%B0%80%EB%A6%AC%EC%95%84&amp;action=edit&amp;section=6" title="부분 편집: 남이탈리아의 불가르">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>그 밖의 불가르의 씨족으로는 662년쯤 "아르체크 공"(Duke Alzeco)(Altsek)이 이끄는 일파는 <a class="mw-redirect" href="/wiki/%EB%A1%AC%EB%B0%94%EB%A5%B4%EB%93%9C%EC%A1%B1" title="롬바르드족">롬바르드족</a>(Lombards)과 더불어 아바르에서 도망쳐 롬바르드의 왕 <a class="new" href="/w/index.php?title=%EB%B2%A0%EB%84%A4%EB%B2%A4%ED%86%A0%EC%9D%98_%EA%B7%B8%EB%A6%AC%EB%AA%A8%EC%95%84%EB%A5%B4%EB%93%9C_1%EC%84%B8&amp;action=edit&amp;redlink=1" title="베네벤토의 그리모아르드 1세 (없는 문서)">베네벤토의 그리모아르드 1세</a>(Grimoald I of Benevento)에게 군사임무를 맡는 대신 영토를 요구했다. 처음 일파는 <a href="/wiki/%EB%9D%BC%EB%B2%A4%EB%82%98" title="라벤나">라벤나</a>(Ravenna)에 정착했으나, 후에는 더 남쪽으로 이동했다. 그리모아르드는 베네벤토에 있던 아르체크의 일파에 대해 자신의 아들인 로무아르드 1세(Romuald)를 보냈다. 로무아르드에 의해 아르체크 일파는 광대하지만 당시에는 불모의 땅이던 <a href="/wiki/%EB%82%98%ED%8F%B4%EB%A6%AC" title="나폴리">나폴리</a>(Naples)의 북동쪽의 영지 <a href="/wiki/%EC%84%B8%ED%94%BC%EB%85%B8" title="세피노">세피노</a>(Sepino), 보비아눔(<a class="new" href="/w/index.php?title=%EB%B3%B4%EC%9D%B4%EC%95%84%EB%85%B8&amp;action=edit&amp;redlink=1" title="보이아노 (없는 문서)">보이아노</a>(Boiano), <a href="/wiki/%EC%9D%B4%EC%84%B8%EB%A5%B4%EB%8B%88%EC%95%84" title="이세르니아">이세르니아</a>(Isernia) (<a href="/wiki/%EC%95%84%ED%8E%9C%EB%8B%88%EB%85%B8_%EC%82%B0%EB%A7%A5" title="아펜니노 산맥">아펜니노 산맥</a>(Apennines) 속, 지금도 현재는 <a href="/wiki/%EB%AA%B0%EB%A6%AC%EC%84%B8_%EC%A3%BC" title="몰리세 주">몰리세 주</a>(Molise)에 속한다)를 하사했다. 이때까지 공의 칭호였던 아르체크는 이것을 대신해 롬바르드의 칭호 "<a class="new" href="/w/index.php?title=%EA%B0%80%EC%8A%A4%ED%83%80%EB%A5%B4%EB%93%9C&amp;action=edit&amp;redlink=1" title="가스타르드 (없는 문서)">가스타르드</a>"(Gastald)를 받았다. <a class="new" href="/w/index.php?title=%ED%8C%8C%EC%9A%B0%EB%A5%B4%EC%8A%A4_%EB%94%94%EC%95%84%EC%BD%94%EB%88%84%EC%8A%A4&amp;action=edit&amp;redlink=1" title="파우르스 디아코누스 (없는 문서)">파우르스 디아코누스</a>(Paul the Deacon)는 787년 저서 "<a class="new" href="/w/index.php?title=%EB%A1%AC%EB%B0%94%EB%A5%B4%EB%93%9C_%EC%82%AC%EB%9E%8C%EB%93%A4%EC%9D%98_%EC%97%AD%EC%82%AC&amp;action=edit&amp;redlink=1" title="롬바르드 사람들의 역사 (없는 문서)">롬바르드 사람들의 역사</a>"(History of the Lombard People)에서 불가르인이 당시에도 같은 영지에 살고 있었고, 라틴어를 말할 수 있어 "자신의 독자의 언어를 아직 버리지 않았다"라고 기록했다.<sup class="reference" id="cite_ref-7"><a href="#cite_note-7">[7]</a></sup></p>
<p>보이아노 근교의 있던 7세기의 비첸네(Vicenne)-<a href="/wiki/%EC%BA%84%ED%8F%AC%ED%82%A4%EC%95%84%EB%A1%9C" title="캄포키아로">캄포키아로</a>(Campochiaro)의 네크로폴리스(necropolis)에서 이루어진 발굴조사에서 약 130개의 유체가 발견되어 그중 13개는 인간의 유체였고 나머진 말의 유체 및 게르만, 아바르에 기원을 둔 기물과 함께 발견되었다.<sup class="reference" id="cite_ref-8"><a href="#cite_note-8">[8]</a></sup><sup class="reference" id="cite_ref-9"><a href="#cite_note-9">[9]</a></sup><sup class="reference" id="cite_ref-10"><a href="#cite_note-10">[10]</a></sup> 말의 유체는 중앙 아시아의 야생말의 특징을 갖고 있어 이 지방에 불가르인이 거주했던 것이 확실하다는 것을 설명하고 있다<sup class="reference" id="cite_ref-11"><a href="#cite_note-11">[11]</a></sup>.</p>
<h3><span class="mw-headline" id=".EC.BD.94.EC.B9.B4.EC.84.9C.EC.8A.A4.EC.9D.98_.EB.B6.88.EA.B0.80.EB.A5.B4_.EB.B0.9C.EA.B0.80.EB.A5.B4">코카서스의 불가르 발가르</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EA%B3%A0%EB%8C%80_%EB%8C%80%EB%B6%88%EA%B0%80%EB%A6%AC%EC%95%84&amp;action=edit&amp;section=7" title="부분 편집: 코카서스의 불가르 발가르">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>쿠브라트의 장남 바트바얀이 이끄는 일파는 흑불가르(Black Bulgars)라고 불렸고, 그 후에도 고향에 계속 머물러 이윽고 <a class="mw-redirect" href="/wiki/%ED%95%98%EC%9E%90%EB%A5%B4" title="하자르">하자르</a>의 지배하에 들어갔다. 현재 <a class="new" href="/w/index.php?title=%EB%B0%9C%EA%B0%80%EB%A5%B4%EC%A1%B1&amp;action=edit&amp;redlink=1" title="발가르족 (없는 문서)">발가르족</a>(Balkars)이 흑불가르의 자손이 아닐까 하는 견해도 있다. 발가르인은 자신을 <a class="new" href="/w/index.php?title=%EB%A7%88%EB%A5%B4%EC%B9%B4_%EA%B0%95&amp;action=edit&amp;redlink=1" title="마르카 강 (없는 문서)">마르카 강</a>(Malka)의 이름에서 유래된 말카르(Malkars)라고 칭하고, <a href="/wiki/%ED%82%B5%EC%B0%A8%ED%81%AC%EC%9D%B8" title="킵차크인">킵차크인</a>(Kipchak)계의 투르크어로 말했다. 투르크어에서 "b"가 "m"으로 변화한 것은 이러한 이유에서였다.</p>
<h3><span class="mw-headline" id=".EC.A0.9C1.EC.B0.A8_.EB.B6.88.EA.B0.80.EB.A6.AC.EC.95.84_.EC.A0.9C.EA.B5.AD">제1차 불가리아 제국</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EA%B3%A0%EB%8C%80_%EB%8C%80%EB%B6%88%EA%B0%80%EB%A6%AC%EC%95%84&amp;action=edit&amp;section=8" title="부분 편집: 제1차 불가리아 제국">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>쿠브라트의 3남 <a class="new" href="/w/index.php?title=%EC%95%84%EC%8A%A4%ED%8C%8C%EB%A5%B4%ED%9D%90&amp;action=edit&amp;redlink=1" title="아스파르흐 (없는 문서)">아스파르흐</a>(Asparuh)는 훗날 <a href="/wiki/%EB%B9%84%EC%9E%94%ED%8B%B0%EC%9B%80_%EC%A0%9C%EA%B5%AD" title="비잔티움 제국">비잔티움 제국</a>이 지배하던 <a href="/wiki/%EB%AA%A8%EC%9D%B4%EC%8B%9C%EC%95%84" title="모이시아">모이시아</a>(Moesia)와 <a href="/wiki/%EB%8F%84%EB%B8%8C%EB%A3%A8%EC%9E%90" title="도브루자">도브루자</a>(Dobrudja)를 정복하고 680년 <a class="mw-redirect" href="/wiki/%EC%A0%9C1%EC%B0%A8_%EB%B6%88%EA%B0%80%EB%A6%AC%EC%95%84_%EC%A0%9C%EA%B5%AD" title="제1차 불가리아 제국">제1차 불가리아 제국</a>을 건국했다.</p>
<h2><span class="mw-headline" id=".EC.98.A4.EB.85.B8.EA.B7.B8.EB.A6.AC.EC.95.84.EC.9D.98_.EC.96.B4.EC.9B.90">오노그리아의 어원</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EA%B3%A0%EB%8C%80_%EB%8C%80%EB%B6%88%EA%B0%80%EB%A6%AC%EC%95%84&amp;action=edit&amp;section=9" title="부분 편집: 오노그리아의 어원">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<div class="rellink relarticle mainarticle"><img alt="" data-file-height="128" data-file-width="128" height="16" src="//upload.wikimedia.org/wikipedia/commons/thumb/e/ec/Crystal_Clear_app_xmag.svg/16px-Crystal_Clear_app_xmag.svg.png" srcset="//upload.wikimedia.org/wikipedia/commons/thumb/e/ec/Crystal_Clear_app_xmag.svg/24px-Crystal_Clear_app_xmag.svg.png 1.5x, //upload.wikimedia.org/wikipedia/commons/thumb/e/ec/Crystal_Clear_app_xmag.svg/32px-Crystal_Clear_app_xmag.svg.png 2x" width="16"/> 이 부분의 본문은 <a class="new" href="/w/index.php?title=%EC%98%A4%EB%85%B8%EA%B5%AC%EB%A5%B4%EC%8A%A4&amp;action=edit&amp;redlink=1" title="오노구르스 (없는 문서)">오노구르스</a>입니다.</div>
<p>오노그리아에서는 다음과 같은 변형된 이름이 있다.</p>
<dl>
<dd><i>Onoghuria</i>, <i>Onoguri</i>, <i>Onoghuri</i>, <i>Onghur</i>, <i>Ongur</i>, <i>Onghuri</i>, <i>Onguri</i>, <i>Onghuria</i>, <i>Onguria</i>, <i>Onogundur</i>, <i>Unogundur</i>, <i>Unokundur</i>, 기타</dd>
</dl>
<p>이 어원에는 다음과 같은 설이 있다.</p>
<ul>
<li><a class="new" href="/w/index.php?title=%EC%BA%85%EC%B9%B4%EC%8A%A4_%EC%95%84%EB%B0%94%EB%A5%B4&amp;action=edit&amp;redlink=1" title="캅카스 아바르 (없는 문서)">캅카스 아바르</a>(Caucasian Acars)의 언어에서 오노그리아는 "영원불멸"을 의미한다고 알려져, <i>uno</i>는 "영원", <i>guro</i>는 "불멸"을 의미한다.</li>
<li><a class="mw-redirect" href="/wiki/%ED%88%AC%EB%A5%B4%ED%81%AC%EC%96%B4" title="투르크어">투르크어</a>에서 <i>On</i>는 "10", <i>Ghur</i>는 "화살"을 의미해 "10개의 화살"은 10개의 씨족의 연합을 의미하는 것으로 보고 있다.</li>
<li>그 이외의 설에는 <a class="mw-redirect" href="/wiki/%ED%88%AC%EB%A5%B4%ED%81%AC%EC%96%B4" title="투르크어">투르크어</a>에서 "z"의 음은 서쪽으로 이동하면서 "r"로 바뀌었기 때문에 오구즈(Oguz/Oghuz)는 서쪽에서 오구르(Ogur/Oghur)로 바뀌게 되었다. 그 때문에 오노그르는 "10개의 오구즈의 씨족"을 의미한다고 보고 있다. 이 시점을 옹호하는 견해로써 신화상의 투르크인의 기원이라고 하는 <a class="new" href="/w/index.php?title=%ED%88%AC%EA%B0%80%EB%A5%B4%EB%A7%88%ED%94%84&amp;action=edit&amp;redlink=1" title="투가르마프 (없는 문서)">투가르마프</a>(Togarmah)의 10명의 자식들 중에 불가르가 포함되어 있는 것을 들 수 있다.</li>
<li>그 외 오노그르를 <a href="/wiki/%EC%95%84%EB%A5%B4%EB%A9%94%EB%8B%88%EC%95%84" title="아르메니아">아르메니아</a>의 문헌에 등장하는 불가르의 일파 <i>unok-vndur</i>와 결부시켜 생각하는 측도 있다.</li>
</ul>
<h3><span class="mw-headline" id=".EA.B7.B8.EC.99.B8_.EC.9A.A9.EB.B2.95">그외 용법</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EA%B3%A0%EB%8C%80_%EB%8C%80%EB%B6%88%EA%B0%80%EB%A6%AC%EC%95%84&amp;action=edit&amp;section=10" title="부분 편집: 그외 용법">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p><a class="new" href="/w/index.php?title=%ED%97%9D%EA%B0%80%EB%A5%B4%EC%9D%B8&amp;action=edit&amp;redlink=1" title="헝가르인 (없는 문서)">마자르인</a>(Magyars)을 포함해 7개의 <a class="new" href="/w/index.php?title=%ED%95%80%EC%9A%B0%EA%B3%A0%EB%A5%B4%EA%B3%84&amp;action=edit&amp;redlink=1" title="핀우고르계 (없는 문서)">핀우고르계</a>(Finno-Ugric)의 씨족과 3개의 하자르계의 씨족이 합류해 "10개의 화살" 혹은 "오노그르"라고 알려지게 되었다. "<a href="/wiki/%ED%97%9D%EA%B0%80%EB%A6%AC" title="헝가리">헝가리</a>"의 명칭은 이 오노그르에서 유래되었다고 생각된다<sup class="reference" id="cite_ref-OSZK_12-0"><a href="#cite_note-OSZK-12">[12]</a></sup><sup class="reference" id="cite_ref-EB_13-0"><a href="#cite_note-EB-13">[13]</a></sup>. <a href="/wiki/%EB%A3%A8%EB%A7%88%EB%8B%88%EC%95%84" title="루마니아">루마니아</a>어에서 "헝가리"의 호칭은 "운구르"(Ungur), 불가리아어에서 호칭은 "운가르"(Ungar)는 "오노그르"와 매우 유사하다.</p>
