<p><b>의종</b>(毅宗, <a href="/wiki/1127%EB%85%84" title="1127년">1127년</a> <a href="/wiki/5%EC%9B%94_23%EC%9D%BC" title="5월 23일">5월 23일</a> (<a href="/wiki/%EC%9D%8C%EB%A0%A5_4%EC%9B%94_11%EC%9D%BC" title="음력 4월 11일">음력 4월 11일</a>) ~ <a href="/wiki/1173%EB%85%84" title="1173년">1173년</a> <a href="/wiki/11%EC%9B%94_7%EC%9D%BC" title="11월 7일">11월 7일</a> (<a href="/wiki/%EC%9D%8C%EB%A0%A5_10%EC%9B%94_1%EC%9D%BC" title="음력 10월 1일">음력 10월 1일</a>) )은 <a href="/wiki/%EA%B3%A0%EB%A0%A4" title="고려">고려</a> 제18대 국왕(재위: <a href="/wiki/1146%EB%85%84" title="1146년">1146년</a> ~ <a href="/wiki/1170%EB%85%84" title="1170년">1170년</a>)이다. <a class="mw-redirect" href="/wiki/%ED%9C%98" title="휘">휘</a>는 현(晛), <a href="/wiki/%EC%9E%90_(%EC%9D%B4%EB%A6%84)" title="자 (이름)">자</a>는 일승(日升), <a href="/wiki/%EB%AC%98%ED%98%B8" title="묘호">묘호</a>는 의종(毅宗), <a href="/wiki/%EC%8B%9C%ED%98%B8" title="시호">시호</a>는 <b>강과장효대왕</b>(剛果莊孝大王)이다. <a href="/wiki/%EA%B3%A0%EB%A0%A4_%EC%9D%B8%EC%A2%85" title="고려 인종">인종</a>과 공예태후 임씨(恭睿太后 任氏)의 맏아들이다.</p>
<p>내시와 환관들과 어울려 유흥과 오락에 깊이 빠지고, 측근들을 중용하다가, 측근 내시의 한사람인 <a href="/wiki/%EA%B9%80%EB%8F%88%EC%A4%91" title="김돈중">김돈중</a>이 무관들을 업신여겨 <a href="/wiki/%EB%AC%B4%EC%8B%A0%EC%A0%95%EB%B3%80" title="무신정변">무신정변</a>의 원인을 제공하였다. 정변으로 축출된 뒤 <a href="/wiki/%EC%9D%B4%EC%9D%98%EB%B0%A9" title="이의방">이의방</a>이 보낸 <a href="/wiki/%EC%9D%B4%EC%9D%98%EB%AF%BC" title="이의민">이의민</a>에 의해 살해당했다.</p>

<h2><span class="mw-headline" id=".EC.83.9D.EC.95.A0">생애</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EA%B3%A0%EB%A0%A4_%EC%9D%98%EC%A2%85&amp;action=edit&amp;section=1" title="부분 편집: 생애">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<h3><span class="mw-headline" id=".EC.83.9D.EC.95.A0_.EC.B4.88.EB.B0.98">생애 초반</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EA%B3%A0%EB%A0%A4_%EC%9D%98%EC%A2%85&amp;action=edit&amp;section=2" title="부분 편집: 생애 초반">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>의종은 <a href="/wiki/%EA%B3%A0%EB%A0%A4_%EC%9D%B8%EC%A2%85" title="고려 인종">인종</a>과 <a href="/wiki/%EC%9E%84%EC%9B%90%ED%9B%84" title="임원후">임원후</a>의 딸 <a href="/wiki/%EA%B3%B5%EC%98%88%EC%99%95%ED%9B%84" title="공예왕후">공예왕후</a> 임씨의 5남 중 장남으로 출생하였다. 그러나 모후인 <a href="/wiki/%EA%B3%B5%EC%98%88%EC%99%95%ED%9B%84" title="공예왕후">공예왕후</a>는 그보다 둘째 아들 <a href="/wiki/%EB%8C%80%EB%A0%B9%ED%9B%84" title="대령후">대녕후</a> 경을 유난히 편애하여 그를 태자로 앉히려 했으나, <a href="/wiki/%EC%A0%95%EC%8A%B5%EB%AA%85" title="정습명">정습명</a>의 반대와 <a href="/wiki/%EA%B3%A0%EB%A0%A4_%EC%9D%B8%EC%A2%85" title="고려 인종">인종</a>의 반대로 무산되었다.</p>
<p>이후 둘째 동생인 <a href="/wiki/%EB%8C%80%EB%A0%B9%ED%9B%84" title="대령후">대녕후</a>와 갈등하다가 후일 그를 역모로 몰아 유배보낸다.</p>
<h3><span class="mw-headline" id=".EC.A6.89.EC.9C.84">즉위</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EA%B3%A0%EB%A0%A4_%EC%9D%98%EC%A2%85&amp;action=edit&amp;section=3" title="부분 편집: 즉위">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p><a href="/wiki/1134%EB%85%84" title="1134년">1134년</a>에 태자(太子)가 되었으며 국자감 사업(國子監司業)으로 유학을 가르치던 <a href="/wiki/%EC%A0%95%EC%8A%B5%EB%AA%85" title="정습명">정습명</a>을 사부로 삼아 수학하였다. <a href="/wiki/1146%EB%85%84" title="1146년">1146년</a> <a href="/wiki/%EA%B3%A0%EB%A0%A4_%EC%9D%B8%EC%A2%85" title="고려 인종">인종</a>이 죽자 즉위하였다.</p>
<p>즉위한 뒤에 그는 이미 인종 때에 일어난 이자겸의 전횡과 반란, <a href="/wiki/%EB%AC%98%EC%B2%AD%EC%9D%98_%EB%82%9C" title="묘청의 난">묘청의 난</a> 등으로 크게 실추된 왕실의 권위를 회복하여 쇠약해진 왕권을 강화시키고자 노력하였다. 이를 위해 무신(武臣)들을 총애하여 친위군을 강화시켜 나갔다. 이 과정에서 <a href="/wiki/%EC%A0%95%EC%A4%91%EB%B6%80" title="정중부">정중부</a>, <a href="/wiki/%EC%9D%B4%EC%9D%98%EB%B0%A9" title="이의방">이의방</a>, <a class="mw-redirect" href="/wiki/%EC%9D%B4%EA%B3%A0_(%EA%B3%A0%EB%A0%A4)" title="이고 (고려)">이고</a>, <a href="/wiki/%EC%9D%B4%EC%9D%98%EB%AF%BC" title="이의민">이의민</a> 등이 발탁되었다.</p>
<p>인종은 죽으며 <a href="/wiki/%EC%A0%95%EC%8A%B5%EB%AA%85" title="정습명">정습명</a>에게 아들 의종을 지도할 것을 부탁하였는데, 선제의 유지가 있다고하여 아는 것은 말하지 않는 것이 없었으며, 의종이 향락에 빠지자 이를 거침없이 지적, 질타하였다. 1148년(의종 2년) <a href="/wiki/%EC%A0%95%EC%8A%B5%EB%AA%85" title="정습명">정습명</a>을 한림학사(翰林學士)에 발탁, 제수(除授)하였고 1149년 좌승선으로 고시관이 되어 시부(詩賦)로 오광윤(吳光允), 십운시로 조정시(趙挺時) 등을 선발하였고, 다시 <a href="/wiki/%EC%A0%95%EC%8A%B5%EB%AA%85" title="정습명">정습명</a>을 추밀원지주사로 앉혔다.</p>
<p>지적이 계속되자 의종은 <a href="/wiki/%EC%A0%95%EC%8A%B5%EB%AA%85" title="정습명">정습명</a>을 미워하고 기피하였다. 의종이 그를 꺼리는 데다가 김존중(金存中)과 정함 등은 밤낮으로 그를 헐뜯었다. <a href="/wiki/1151%EB%85%84" title="1151년">1151년</a> <a href="/wiki/3%EC%9B%94" title="3월">3월</a> <a href="/wiki/%EC%A0%95%EC%8A%B5%EB%AA%85" title="정습명">정습명</a>이 병으로 사직하자 왕은 즉시 김존중에게 임시로 그 직을 대신하게 하였다. 왕의 뜻을 이해한 정습명은 독약을 먹고 자결한다. 그러나 후일 의종은 <a href="/wiki/%EB%AC%B4%EC%8B%A0%EC%A0%95%EB%B3%80" title="무신정변">무신정변</a>으로 축출되며 뒤늦게 그를 찾았다고 한다.</p>
<p>그런데 사실 의종은 장점이 많았다고도 한다. 활을 잘 쏘았으며 잘한 신하에게 상도 후하게 주었다. 의종이 운동도 잘했다고 주장하는 역사학자들도 있다. 하지만 큰 단점, 즉 무신들을 차별하는 것과 놀기를 즐겨하는 것으로 인해 폐위되었다.</p>
<h3><span class="mw-headline" id=".EB.AC.B4.EC.8B.A0.EC.A0.95.EB.B3.80.EA.B3.BC_.ED.8F.90.EC.9C.84">무신정변과 폐위</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EA%B3%A0%EB%A0%A4_%EC%9D%98%EC%A2%85&amp;action=edit&amp;section=4" title="부분 편집: 무신정변과 폐위">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<div class="rellink boilerplate seealso"><img alt="" data-file-height="128" data-file-width="128" height="16" src="//upload.wikimedia.org/wikipedia/commons/thumb/e/ec/Crystal_Clear_app_xmag.svg/16px-Crystal_Clear_app_xmag.svg.png" srcset="//upload.wikimedia.org/wikipedia/commons/thumb/e/ec/Crystal_Clear_app_xmag.svg/24px-Crystal_Clear_app_xmag.svg.png 1.5x, //upload.wikimedia.org/wikipedia/commons/thumb/e/ec/Crystal_Clear_app_xmag.svg/32px-Crystal_Clear_app_xmag.svg.png 2x" width="16"/> <a href="/wiki/%EB%AC%B4%EC%8B%A0%EC%A0%95%EB%B3%80" title="무신정변">무신정변</a> 문서를 참고하십시오.</div>
<p>그러나 말년에는 문신들 및 <a href="/wiki/%ED%99%98%EA%B4%80" title="환관">환관</a>들과 어울려 유흥과 오락에 깊이 빠지고, 지나치게 <a href="/wiki/%EB%B6%88%EA%B5%90" title="불교">불교</a>·음양설, 선풍(仙風)을 중요시하고 <a href="/wiki/%EC%9C%A0%EA%B5%90" title="유교">유교</a>를 멀리하는 등 정치적·사회적 혼란을 초래하였고, 이 과정에서 무신들을 소외시켜 천대받게 만들어 <a href="/wiki/%EB%AC%B4%EC%8B%A0%EC%A0%95%EB%B3%80" title="무신정변">무신정변</a>의 계기를 낳았다.</p>
<p>왕의 총애를 등에 업은 문신들은 약탈을 자행하였고, 무신들을 무시하기 시작했다. 이후 의종의 측근 <a href="/wiki/%EA%B9%80%EB%8F%88%EC%A4%91" title="김돈중">김돈중</a>이 <a class="mw-redirect" href="/wiki/%EC%98%A4%EB%B3%91%EC%88%98%EB%B0%95%ED%9D%AC" title="오병수박희">오병수박희</a>를 하던 <a href="/wiki/%EC%A0%95%EC%A4%91%EB%B6%80" title="정중부">정중부</a>의 수염에 불을 지른 것과 젊은 문신인 <a href="/wiki/%ED%95%9C%EB%A2%B0" title="한뢰">한뢰</a>가 나이가 많은 무신이자 종 3품의 대장군인 <a href="/wiki/%EC%9D%B4%EC%86%8C%EC%9D%91_(%EA%B3%A0%EB%A0%A4)" title="이소응 (고려)">이소응</a> 장군의 뺨을 때린 것이 빌미가 되어 <a href="/wiki/%EB%AC%B4%EC%8B%A0%EC%A0%95%EB%B3%80" title="무신정변">무신정변</a>이 발생한다.</p>
<p><a href="/wiki/1170%EB%85%84" title="1170년">1170년</a>[보현원에 거동하였을 때에 <a href="/wiki/%EC%A0%95%EC%A4%91%EB%B6%80" title="정중부">정중부</a>, <a href="/wiki/%EC%9D%B4%EC%9D%98%EB%B0%A9" title="이의방">이의방</a>, <a class="mw-redirect" href="/wiki/%EC%9D%B4%EA%B3%A0_(%EA%B3%A0%EB%A0%A4)" title="이고 (고려)">이고</a> 등의 무신들이 반란을 일으켜 <a href="/wiki/%EB%AC%B4%EC%8B%A0%EC%A0%95%EB%B3%80" title="무신정변">무신정변</a>이 일어남으로써 무신 정권에 의해 폐위되어 3년간 <a href="/wiki/%EA%B1%B0%EC%A0%9C%EB%8F%84" title="거제도">거제도</a>의 <a href="/wiki/%EB%91%94%EB%8D%95%EA%B8%B0%EC%84%B1" title="둔덕기성">둔덕기성</a>에 유폐되었다.</p>
<h3><span class="mw-headline" id=".EA.B9.80.EB.B3.B4.EB.8B.B9.EC.9D.98_.EB.82.9C.EA.B3.BC_.EC.B5.9C.ED.9B.84">김보당의 난과 최후</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EA%B3%A0%EB%A0%A4_%EC%9D%98%EC%A2%85&amp;action=edit&amp;section=5" title="부분 편집: 김보당의 난과 최후">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<div class="rellink boilerplate seealso"><img alt="" data-file-height="128" data-file-width="128" height="16" src="//upload.wikimedia.org/wikipedia/commons/thumb/e/ec/Crystal_Clear_app_xmag.svg/16px-Crystal_Clear_app_xmag.svg.png" srcset="//upload.wikimedia.org/wikipedia/commons/thumb/e/ec/Crystal_Clear_app_xmag.svg/24px-Crystal_Clear_app_xmag.svg.png 1.5x, //upload.wikimedia.org/wikipedia/commons/thumb/e/ec/Crystal_Clear_app_xmag.svg/32px-Crystal_Clear_app_xmag.svg.png 2x" width="16"/> <a href="/wiki/%EA%B9%80%EB%B3%B4%EB%8B%B9%EC%9D%98_%EB%82%9C" title="김보당의 난">김보당의 난</a> 문서를 참고하십시오.</div>
<p>그 뒤 <a href="/wiki/1173%EB%85%84" title="1173년">1173년</a>에 <a href="/wiki/%EA%B9%80%EB%B3%B4%EB%8B%B9" title="김보당">김보당</a>(金甫當) 등 의종 복위 세력이 무신 정권에 항거하여 거병하면서 사람을 보내어 유배된 의종을 모셔와 받들고 <a href="/wiki/%EA%B2%BD%EC%A3%BC%EC%8B%9C" title="경주시">경주</a>로 가서 웅거하였다. 그러나 무신 정권이 보낸 군대에게 모두 패하였다.</p>
<p>1173년<sup class="reference" id="cite_ref-1"><a href="#cite_note-1">[1]</a></sup> 의종은 총애하던 장수 <a href="/wiki/%EC%9D%B4%EC%9D%98%EB%AF%BC" title="이의민">이의민</a>과 곤원사(坤元寺) 연못가에서 술을 마시다가 갑자기 그에 의해 등뼈가 꺾여지고 시체는 그대로 <a class="mw-redirect" href="/wiki/%EB%8F%99%EA%B2%BD" title="동경">동경</a> (오늘날 <a href="/wiki/%EA%B2%BD%EC%A3%BC%EC%8B%9C" title="경주시">경주</a>)의 한 연못에 수장당하고 마는 처참한 최후를 맞이한다.</p>
<p>이때 그의 나이 47세였으며 이후 동생인 <a href="/wiki/%EA%B3%A0%EB%A0%A4_%EB%AA%85%EC%A2%85" title="고려 명종">명종</a>에 의해 능이 조성되었는데 바로 희릉(禧陵)이다. 현재 희릉의 위치는 알 수 없으나 개성 동쪽에 위치하였을 것으로 추정하고 있다.</p>

<h2><span class="mw-headline" id=".EA.B3.A0.EB.A0.A4_.EB.AC.B8.EC.A2.85.EA.B3.BC.EC.9D.98_.EA.B4.80.EA.B3.84">고려 문종과의 관계</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EA%B3%A0%EB%A0%A4_%EC%9D%98%EC%A2%85&amp;action=edit&amp;section=9" title="부분 편집: 고려 문종과의 관계">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>의종, <a href="/wiki/%EA%B3%A0%EB%A0%A4_%EB%AA%85%EC%A2%85" title="고려 명종">명종</a>, <a href="/wiki/%EA%B3%A0%EB%A0%A4_%EC%8B%A0%EC%A2%85" title="고려 신종">신종</a>은 <a href="/wiki/%EA%B3%A0%EB%A0%A4_%EB%AC%B8%EC%A2%85" title="고려 문종">문종</a>의 4대손이면서 동시에 증손녀 사위가 된다. 의종, <a href="/wiki/%EA%B3%A0%EB%A0%A4_%EB%AA%85%EC%A2%85" title="고려 명종">명종</a>, <a href="/wiki/%EA%B3%A0%EB%A0%A4_%EC%8B%A0%EC%A2%85" title="고려 신종">신종</a>은 <a href="/wiki/%EA%B3%A0%EB%A0%A4_%EC%9D%B8%EC%A2%85" title="고려 인종">인종</a>의 아들로 <a href="/wiki/%EA%B3%A0%EB%A0%A4_%EB%AC%B8%EC%A2%85" title="고려 문종">문종</a>-<a href="/wiki/%EA%B3%A0%EB%A0%A4_%EC%88%99%EC%A2%85" title="고려 숙종">숙종</a>-<a href="/wiki/%EA%B3%A0%EB%A0%A4_%EC%98%88%EC%A2%85" title="고려 예종">예종</a>-<a href="/wiki/%EA%B3%A0%EB%A0%A4_%EC%9D%B8%EC%A2%85" title="고려 인종">인종</a>으로 이어진다.</p>
<p><a href="/wiki/%EA%B3%A0%EB%A0%A4_%EB%AC%B8%EC%A2%85" title="고려 문종">문종</a>의 서자인 <a class="mw-disambig" href="/wiki/%EC%96%91%ED%97%8C%EC%99%95" title="양헌왕">양헌왕</a>의 아들이자 손자인 강릉공 온(江陵公 溫)은 의종의 왕후인 <a href="/wiki/%EC%9E%A5%EA%B2%BD%EC%99%95%ED%9B%84_(%EA%B3%A0%EB%A0%A4)" title="장경왕후 (고려)">장경왕후</a>, <a href="/wiki/%EA%B3%A0%EB%A0%A4_%EB%AA%85%EC%A2%85" title="고려 명종">명종</a>의 왕후인 <a class="mw-redirect" href="/wiki/%EA%B4%91%EC%A0%95%EC%99%95%ED%9B%84" title="광정왕후">광정왕후</a>, <a href="/wiki/%EA%B3%A0%EB%A0%A4_%EC%8B%A0%EC%A2%85" title="고려 신종">신종</a>의 왕후인 <a class="mw-disambig" href="/wiki/%EC%84%A0%EC%A0%95%EC%99%95%ED%9B%84" title="선정왕후">선정왕후</a>의 친정아버지가 된다.</p>
