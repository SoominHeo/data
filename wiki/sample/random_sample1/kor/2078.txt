<p><b>헌종</b>(獻宗, <a href="/wiki/1084%EB%85%84" title="1084년">1084년</a> <a href="/wiki/8%EC%9B%94_1%EC%9D%BC" title="8월 1일">8월 1일</a> (<a href="/wiki/%EC%9D%8C%EB%A0%A5_6%EC%9B%94_27%EC%9D%BC" title="음력 6월 27일">음력 6월 27일</a>) ~ <a href="/wiki/1097%EB%85%84" title="1097년">1097년</a> <a href="/wiki/11%EC%9B%94_6%EC%9D%BC" title="11월 6일">11월 6일</a> (<a href="/wiki/%EC%9D%8C%EB%A0%A5_10%EC%9B%94_7%EC%9D%BC" title="음력 10월 7일">음력 10월 7일</a>)은 고려 제14대 국왕(<small>재위:</small> <a href="/wiki/1094%EB%85%84" title="1094년">1094년</a> ~ <a href="/wiki/1095%EB%85%84" title="1095년">1095년</a>)이다. 휘는 욱(昱), <a href="/wiki/%EB%AC%98%ED%98%B8" title="묘호">묘호</a>는 <b>헌종</b>(獻宗), <a href="/wiki/%EC%8B%9C%ED%98%B8" title="시호">시호</a>는 <b>공상정비회효대왕</b>(恭殤定比懷孝大王). 선종과 사숙태후(思肅太后) 이씨(李氏)의 맏아들이다. 신병으로 숙부에게 왕위를 선위하고 2년 뒤에 사망하였다.</p>
<p><a href="/wiki/1094%EB%85%84" title="1094년">1094년</a>부터 <a href="/wiki/1095%EB%85%84" title="1095년">1095년</a>까지 재위하는 동안 모후 <a href="/wiki/%EC%82%AC%EC%88%99%EC%99%95%ED%9B%84" title="사숙왕후">사숙왕후</a>(思肅王后)가 <a href="/wiki/%EC%84%AD%EC%A0%95" title="섭정">섭정</a>하였다.</p>

<h2><span class="mw-headline" id=".EC.83.9D.EC.95.A0">생애</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EA%B3%A0%EB%A0%A4_%ED%97%8C%EC%A2%85&amp;action=edit&amp;section=1" title="부분 편집: 생애">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p><a href="/wiki/1094%EB%85%84" title="1094년">1094년</a>, 승하한 선종의 뒤를 이어 즉위하였지만 나이가 어리고 소갈증으로 매우 병약하여 어머니인 사숙태후가 <a href="/wiki/%EC%88%98%EB%A0%B4%EC%B2%AD%EC%A0%95" title="수렴청정">수렴청정</a>하였다. 1년 뒤인 <a href="/wiki/1095%EB%85%84" title="1095년">1095년</a>에 <a href="/wiki/%EC%9D%B4%EC%9E%90%EC%9D%98" title="이자의">이자의</a>(李資義)가 한산후 균을 옹립하려고 했으나 숙부인 계림공 왕희(후의 <a href="/wiki/%EA%B3%A0%EB%A0%A4_%EC%88%99%EC%A2%85" title="고려 숙종">숙종</a>)가 이를 미리 알고 장사 고의화를 시켜 살해했다.</p>
<p>이후 왕희에게 선위하고, 물러나 있다가 2년 후인 <a href="/wiki/1097%EB%85%84" title="1097년">1097년</a> 14세의 나이로 승하하였다. 능은 성동의 은릉(隱陵)이라 하나 현재 위치는 알 수 없고 개성 동쪽에 위치하였을 것으로 추정하고 있다.</p>
<p>묘호는 없고 시호는 회상(懷殤)이었다가, 사후 8년이 지난 <a href="/wiki/1105%EB%85%84" title="1105년">1105년</a>(예종 즉위년)에서야 헌종(獻宗)의 묘호가 올려지고 시호는 공상(恭殤)으로 고쳐졌다<sup class="reference" id="cite_ref-1"><a href="#cite_note-1">[1]</a></sup>.</p>
