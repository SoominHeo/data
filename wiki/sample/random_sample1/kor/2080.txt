<p><b>연세대학교-고려대학교 정기전</b>은 애교심을 고양하고 <a href="/wiki/%EC%97%B0%EC%84%B8%EB%8C%80%ED%95%99%EA%B5%90" title="연세대학교">연세대학교</a>와 <a href="/wiki/%EA%B3%A0%EB%A0%A4%EB%8C%80%ED%95%99%EA%B5%90" title="고려대학교">고려대학교</a> 양교의 친선을 도모하기 위한 목적으로<sup class="reference" id="cite_ref-koyongames_1-0"><a href="#cite_note-koyongames-1">[1]</a></sup> <a href="/wiki/%EC%8A%A4%ED%8F%AC%EC%B8%A0" title="스포츠">스포츠</a> 경기<sup class="reference" id="cite_ref-2"><a href="#cite_note-2">[2]</a></sup> 를 비롯한 강연<sup class="reference" id="cite_ref-3"><a href="#cite_note-3">[3]</a></sup>, <a href="/wiki/%EC%98%A8%EB%9D%BC%EC%9D%B8_%EA%B2%8C%EC%9E%84" title="온라인 게임">온라인 게임</a><sup class="reference" id="cite_ref-4"><a href="#cite_note-4">[4]</a></sup>, 사회공헌활동<sup class="reference" id="cite_ref-5"><a href="#cite_note-5">[5]</a></sup><sup class="reference" id="cite_ref-6"><a href="#cite_note-6">[6]</a></sup> 과 같은 다양한 분야에서 승부를 겨루는 행사 일체를 의미한다. 약칭 <b>연고전</b>(延高戰) 또는 <b>고연전</b>(高延戰)<sup class="reference" id="cite_ref-7"><a href="#cite_note-7">[7]</a></sup> 으로 불린다. 유사한 대학교 스포츠 대항전으로 <a href="/wiki/%EB%AF%B8%EA%B5%AD" title="미국">미국</a>의 <a href="/wiki/%ED%95%98%EB%B2%84%EB%93%9C_%EB%8C%80%ED%95%99%EA%B5%90" title="하버드 대학교">하버드 대학교</a>와 <a href="/wiki/%EC%98%88%EC%9D%BC_%EB%8C%80%ED%95%99%EA%B5%90" title="예일 대학교">예일 대학교</a>의 <a href="/wiki/%EB%AF%B8%EC%8B%9D%EC%B6%95%EA%B5%AC" title="미식축구">미식축구</a> 경기, <a href="/wiki/%EC%98%81%EA%B5%AD" title="영국">영국</a>의 <a class="mw-redirect" href="/wiki/%EC%98%A5%EC%8A%A4%ED%8F%AC%EB%93%9C_%EB%8C%80%ED%95%99%EA%B5%90" title="옥스포드 대학교">옥스포드 대학교</a>와 <a class="mw-redirect" href="/wiki/%EC%BA%A0%EB%B8%8C%EB%A6%AC%EC%A7%80_%EB%8C%80%ED%95%99%EA%B5%90" title="캠브리지 대학교">캠브리지 대학교</a>의 <a href="/wiki/%EC%A1%B0%EC%A0%95_(%EC%8A%A4%ED%8F%AC%EC%B8%A0)" title="조정 (스포츠)">조정</a> 경기와 <a href="/wiki/%EC%9D%BC%EB%B3%B8" title="일본">일본</a>의 <a href="/wiki/%EA%B2%8C%EC%9D%B4%EC%98%A4%EA%B8%B0%EC%A3%BC%EC%BF%A0_%EB%8C%80%ED%95%99" title="게이오기주쿠 대학">게이오기주쿠 대학</a>과 <a href="/wiki/%EC%99%80%EC%84%B8%EB%8B%A4_%EB%8C%80%ED%95%99" title="와세다 대학">와세다 대학</a>의 ‘<a href="/wiki/%EC%86%8C%EC%BC%80%EC%9D%B4%EC%84%BC" title="소케이센">소케이센</a>’(早慶戰) 등이 있다. 연고전/고연전은 <b>매년 <a href="/wiki/9%EC%9B%94" title="9월">9월</a>에 5개 종목의 운동부 선수들이 맞붙는 정기전</b><sup class="reference" id="cite_ref-koyongames_1-1"><a href="#cite_note-koyongames-1">[1]</a></sup> 과 <b>정기전 이외의 기간에 열리는 비정기전</b><sup class="reference" id="cite_ref-8"><a href="#cite_note-8">[8]</a></sup> 으로 크게 나누어볼 수 있으며, 이와 관련하여 교내에 축구부, 농구부, 야구부, 럭비부, 빙구부 등 5개 종목의 스포츠단체가 활동하고 있다.<sup class="reference" id="cite_ref-9"><a href="#cite_note-9">[9]</a></sup> 흔히 일컬어지는 ‘고연전’이나 ‘연고전’이라는 명칭은 정기전을 의미하며<sup class="reference" id="cite_ref-10"><a href="#cite_note-10">[10]</a></sup>, 아래에 이어지는 설명도 정기전에 관한 것이다.</p>

<h2><span class="mw-headline" id=".EC.97.AD.EC.82.AC_.EB.B0.8F_.EB.AA.85.EC.B9.AD">역사 및 명칭</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EA%B3%A0%EB%A0%A4%EB%8C%80%ED%95%99%EA%B5%90-%EC%97%B0%EC%84%B8%EB%8C%80%ED%95%99%EA%B5%90_%EC%A0%95%EA%B8%B0%EC%A0%84&amp;action=edit&amp;section=1" title="부분 편집: 역사 및 명칭">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<h3><span class="mw-headline" id=".EC.97.AD.EC.82.AC">역사</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EA%B3%A0%EB%A0%A4%EB%8C%80%ED%95%99%EA%B5%90-%EC%97%B0%EC%84%B8%EB%8C%80%ED%95%99%EA%B5%90_%EC%A0%95%EA%B8%B0%EC%A0%84&amp;action=edit&amp;section=2" title="부분 편집: 역사">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p><a href="/wiki/%EC%97%B0%EC%84%B8%EB%8C%80%ED%95%99%EA%B5%90" title="연세대학교">연세대학교</a>의 전신인 <a href="/wiki/%EC%97%B0%ED%9D%AC%EC%A0%84%EB%AC%B8%ED%95%99%EA%B5%90" title="연희전문학교">연희전문학교</a>와 <a href="/wiki/%EA%B3%A0%EB%A0%A4%EB%8C%80%ED%95%99%EA%B5%90" title="고려대학교">고려대학교</a>의 전신인 <a class="mw-redirect" href="/wiki/%EB%B3%B4%EC%84%B1%ED%95%99%EA%B5%90" title="보성학교">보성전문학교</a>가 처음으로 시합에서 만난 것은 <a href="/wiki/1925%EB%85%84" title="1925년">1925년</a> <a href="/wiki/5%EC%9B%94_30%EC%9D%BC" title="5월 30일">5월 30일</a>에 열린 조선 체육회 주최 ‘제5회 전조선(全朝鮮) 정구대회’였다.<sup class="reference" id="cite_ref-.EA.B2.BD.ED.96.A5_1_11-0"><a href="#cite_note-.EA.B2.BD.ED.96.A5_1-11">[11]</a></sup><sup class="reference" id="cite_ref-.EB.8F.99.EC.95.84_1_12-0"><a href="#cite_note-.EB.8F.99.EC.95.84_1-12">[12]</a></sup> 양교의 실질적인 첫 전면시합이 열린 것은 <a href="/wiki/1927%EB%85%84" title="1927년">1927년</a> <a href="/wiki/%EB%8F%99%EB%8C%80%EB%AC%B8%EC%9A%B4%EB%8F%99%EC%9E%A5" title="동대문운동장">경성운동장</a>에서 거행된 제8회 <a class="mw-redirect" href="/wiki/%EC%A0%84%EC%A1%B0%EC%84%A0_%EC%B6%95%EA%B5%AC%EB%8C%80%ED%9A%8C" title="전조선 축구대회">전조선 축구대회</a> 준결승에서의 <a href="/wiki/%EA%B3%A0%EB%A0%A4%EB%8C%80%ED%95%99%EA%B5%90_%EC%B6%95%EA%B5%AC%EB%B6%80" title="고려대학교 축구부">보성전문학교 축구부</a>와 <a href="/wiki/%EC%97%B0%EC%84%B8%EB%8C%80%ED%95%99%EA%B5%90_%EC%B6%95%EA%B5%AC%EB%B6%80" title="연세대학교 축구부">연희전문학교 축구부</a>의 대결이었으며,<sup class="reference" id="cite_ref-_13-0"><a href="#cite_note--13">[13]</a></sup> 1929년에는 보성전문학교가 연희전문학교를 이겨 두 학교의 대결이 일반인의 관심을 불러일으켰다. 이어 1930년에 두 학교가 <a href="/wiki/%EB%86%8D%EA%B5%AC" title="농구">농구</a>에서도 막강한 실력으로 서로 겨루게 되면서 이들 경기가 ‘연·보전’(또는 ‘보·연전’)으로 일반에 불리게 되었다.<sup class="reference" id="cite_ref-.EA.B2.BD.ED.96.A5_1_11-1"><a href="#cite_note-.EA.B2.BD.ED.96.A5_1-11">[11]</a></sup> ‘민족적인 경기’가 된 이들 대결에서 보성전문학교 학생들은 꾕과리와 징을 울리면서 안암동에서 동대문의 서울 운동장까지 행진하여 연희전문학교의 학생들과 함께 양쪽으로 응원하면서 경기를 관람하였고, 경기에 이긴 쪽은 우승기를 들고 교가를 부르면서 종로네거리를 행진하였다.<sup class="reference" id="cite_ref-.EA.B2.BD.ED.96.A5_2_14-0"><a href="#cite_note-.EA.B2.BD.ED.96.A5_2-14">[14]</a></sup> <a href="/wiki/1943%EB%85%84" title="1943년">1943년</a>에 <a href="/wiki/%EC%A7%95%EB%B3%91%EC%A0%9C" title="징병제">징병제</a>가 실시되어 시합이 중지되기 전까지 연희전문학교는 보성전문학교를 상대로 한 축구대회에서 31전 14승 17패를, 농구대회에서 62전 32승 30패를 기록하였다.<sup class="reference" id="cite_ref-koyonhistory_15-0"><a href="#cite_note-koyonhistory-15">[15]</a></sup></p>
<p>두 학교간의 대결은 광복 이후 <a href="/wiki/1945%EB%85%84" title="1945년">1945년</a> <a href="/wiki/12%EC%9B%94_21%EC%9D%BC" title="12월 21일">12월 21일</a>부터 이틀간 열린 ‘제1회 연보 OB 축구대회’를 통해 부활하였다. <a href="/wiki/1946%EB%85%84" title="1946년">1946년</a>부터는 <a href="/wiki/%EC%B6%95%EA%B5%AC" title="축구">축구</a>와 <a href="/wiki/%EB%86%8D%EA%B5%AC" title="농구">농구</a>의 재학생 간 정기전이 매년 열리게 되었고, <a href="/wiki/1956%EB%85%84" title="1956년">1956년</a>에는 <a href="/wiki/%EC%95%BC%EA%B5%AC" title="야구">야구</a>, <a class="mw-redirect" href="/wiki/%EB%9F%AD%EB%B9%84_%EC%9C%A0%EB%8B%88%EC%98%A8" title="럭비 유니온">럭비</a>, <a href="/wiki/%EC%95%84%EC%9D%B4%EC%8A%A4%ED%95%98%ED%82%A4" title="아이스하키">빙구</a>의 세 종목이 추가되었다. <b>다만, 우리나라 최초의 학교간 정기전은 <a href="/wiki/1946%EB%85%84" title="1946년">1946년</a> 시작된 <a href="/wiki/%EB%B0%B0%EC%9E%AC%ED%95%99%EB%8B%B9" title="배재학당">배재학당</a>과 <a href="/wiki/%EC%96%91%EC%A0%95%EC%9D%98%EC%88%99" title="양정의숙">양정의숙</a>의 <a href="/wiki/%EC%A0%84_%EB%B0%B0%EC%9E%AC_%EB%8C%80_%EC%A0%84_%EC%96%91%EC%A0%95_%EB%9F%AD%EB%B9%84%EC%A0%95%EA%B8%B0%EC%A0%84" title="전 배재 대 전 양정 럭비정기전">전 배재 대 전 양정 럭비정기전</a>이며, 오늘날과 같은 규모의 정기전의 기틀이 마련된 것은 <a href="/wiki/1959%EB%85%84" title="1959년">1959년</a>의 일이다.</b> <a href="/wiki/1961%EB%85%84" title="1961년">1961년</a>부터 <a href="/wiki/1964%EB%85%84" title="1964년">1964년</a>까지는 <a class="mw-redirect" href="/wiki/5%C2%B716_%EA%B5%B0%EC%82%AC%EC%A0%95%EB%B3%80" title="5·16 군사정변">5·16 군사정변</a> 등의 이유로 정기전이 열리지 않았다.<sup class="reference" id="cite_ref-koyonhistory_15-1"><a href="#cite_note-koyonhistory-15">[15]</a></sup> <a href="/wiki/1965%EB%85%84" title="1965년">1965년</a>부터는 이틀 동안 다섯 경기가 일제히 치러지는 지금과 같은 경기방식이 완전히 자리를 잡았으며, 1970년대 중반부터 점차 정기전 때 고려대학교에서는 크림슨색 티셔츠를 입기 시작하였다.<sup class="reference" id="cite_ref-16"><a href="#cite_note-16">[16]</a></sup> 1971년과 1972년, 1975년, 1980년, 1983년, 1996년에는 정기전이 열리지 않았다.</p>
<h3><span class="mw-headline" id=".EB.AA.85.EC.B9.AD">명칭</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EA%B3%A0%EB%A0%A4%EB%8C%80%ED%95%99%EA%B5%90-%EC%97%B0%EC%84%B8%EB%8C%80%ED%95%99%EA%B5%90_%EC%A0%95%EA%B8%B0%EC%A0%84&amp;action=edit&amp;section=3" title="부분 편집: 명칭">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>정기전은 그 명칭에 대한 입씨름에서부터 경기가 시작된다고도 하였다. ‘보·연전’과 ‘연·보전’, ‘고·연전’과 ‘연·고전’의 명칭에 대한 양보 없는 주장 또한 함께 내려온 전통이라는 것이다.<sup class="reference" id="cite_ref-.EB.8F.99.EC.95.84_2_17-0"><a href="#cite_note-.EB.8F.99.EC.95.84_2-17">[17]</a></sup></p>
<p>연고전 또는 고연전은 <a class="mw-redirect" href="/wiki/%EC%9D%BC%EC%A0%9C%EA%B0%95%EC%A0%90%EA%B8%B0" title="일제강점기">일제강점기</a>에는 보연전 또는 연보전이었으나, <a href="/wiki/1946%EB%85%84" title="1946년">1946년</a>에 두 학교의 교명이 바뀌어<sup class="reference" id="cite_ref-18"><a href="#cite_note-18">[18]</a></sup><sup class="reference" id="cite_ref-19"><a href="#cite_note-19">[19]</a></sup><sup class="reference" id="cite_ref-20"><a href="#cite_note-20">[20]</a></sup> 현재의 명칭이 되었다. 정기전이 시작된 이후로는 연고전의 명칭이 주로 쓰였으나, <a href="/wiki/1958%EB%85%84" title="1958년">1958년</a>부터 고려대학교 측에서 이의를 제기하였다.<sup class="reference" id="cite_ref-.EA.B3.A0.EB.8C.80.EC.8B.A0.EB.AC.B8_21-0"><a href="#cite_note-.EA.B3.A0.EB.8C.80.EC.8B.A0.EB.AC.B8-21">[21]</a></sup> <a href="/wiki/1965%EB%85%84" title="1965년">1965년</a>에는 고려대학교와 연세대학교 중 주최하는 학교가 상대방의 이름을 먼저 쓰기로 서로 약속하였다.<sup class="reference" id="cite_ref-22"><a href="#cite_note-22">[22]</a></sup> 언론에서는 고연전보다 연고전의 사용 빈도가 높은데, 이는 정기전이 시작된 첫 10년 간 연고전으로 사용되면서 관습적으로 명칭이 굳어진 탓이라고도 한다.<sup class="reference" id="cite_ref-.EA.B3.A0.EB.8C.80.EC.8B.A0.EB.AC.B8_21-1"><a href="#cite_note-.EA.B3.A0.EB.8C.80.EC.8B.A0.EB.AC.B8-21">[21]</a></sup></p>
<p>두 학교 사이의 약속으로 정식 명칭이 실질적으로 방송 중계와 신문 보도 등의 공식 석상에서 사용되지만, 고려대학교에서는 고연전, 연세대학교에서는 연고전이라는 명칭으로 통용된다.<sup class="reference" id="cite_ref-23"><a href="#cite_note-23">[23]</a></sup></p>
<p><a href="/wiki/1990%EB%85%84" title="1990년">1990년</a> 전후로 고연 민족해방제 또는 연고 민족해방제라는 이름으로 불리기도 하였다.<sup class="reference" id="cite_ref-24"><a href="#cite_note-24">[24]</a></sup></p>
<h2><span class="mw-headline" id=".EC.A0.95.EA.B8.B0.EC.A0.84_.EC.A3.BC.EC.B5.9C_.EB.B0.8F_.EC.9D.BC.EC.A0.95">정기전 주최 및 일정</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EA%B3%A0%EB%A0%A4%EB%8C%80%ED%95%99%EA%B5%90-%EC%97%B0%EC%84%B8%EB%8C%80%ED%95%99%EA%B5%90_%EC%A0%95%EA%B8%B0%EC%A0%84&amp;action=edit&amp;section=4" title="부분 편집: 정기전 주최 및 일정">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>매년 9월에 열리는 정기전(정기친선경기대회)은 양교가 매년 번갈아 주관하며, 주최하는 학교에 따라 명칭이 바뀐다.<sup class="reference" id="cite_ref-25"><a href="#cite_note-25">[25]</a></sup> 연세대학교에서는 연고전을 며칠 앞두고 신촌 오리엔테이션과 원주 오리엔테이션, 양교 합동 오리엔테이션이 각각 진행된다.<sup class="reference" id="cite_ref-26"><a href="#cite_note-26">[26]</a></sup> 고려대학교 또한 고연전을 며칠 앞두고 안암 오리엔테이션과 세종 오리엔테이션, 합동 오리엔테이션이 각각 진행되며<sup class="reference" id="cite_ref-27"><a href="#cite_note-27">[27]</a></sup>, 정기 고연전이 끝난 후에는 교우회 주최로 선후배 간의 화합의 장인 '1905 페스티벌'이 열린다. 이 행사는 동기회, 학과 교우회, 졸업생 개인이 학교 인근 주점을 예약해 후배들과 대화의 장을 마련하는 행사로, 68학번 동기회가 <a href="/wiki/%EC%B0%B8%EC%82%B4%EC%9D%B4%EA%B8%B8" title="참살이길">참살이길</a>의 삼성통닭을 통째로 빌린 것이 그 시초이다.<sup class="reference" id="cite_ref-28"><a href="#cite_note-28">[28]</a></sup><sup class="reference" id="cite_ref-29"><a href="#cite_note-29">[29]</a></sup></p>
<h3><span class="mw-headline" id=".EC.82.AC.EC.9D.B4.EB.B2.84_.EC.A0.95.EA.B8.B0.EC.A0.84">사이버 정기전</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EA%B3%A0%EB%A0%A4%EB%8C%80%ED%95%99%EA%B5%90-%EC%97%B0%EC%84%B8%EB%8C%80%ED%95%99%EA%B5%90_%EC%A0%95%EA%B8%B0%EC%A0%84&amp;action=edit&amp;section=5" title="부분 편집: 사이버 정기전">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>사이버고연전 또는 사이버연고전은 정기전 기간 중 고려대학교와 연세대학교의 학생들이 <a href="/wiki/E%EC%8A%A4%ED%8F%AC%EC%B8%A0" title="E스포츠">e스포츠</a>에서 겨루는 행사이다.</p>
<h2><span class="mw-headline" id=".ED.8F.89.EA.B0.80">평가</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EA%B3%A0%EB%A0%A4%EB%8C%80%ED%95%99%EA%B5%90-%EC%97%B0%EC%84%B8%EB%8C%80%ED%95%99%EA%B5%90_%EC%A0%95%EA%B8%B0%EC%A0%84&amp;action=edit&amp;section=6" title="부분 편집: 평가">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>매년 개최되는 고연전/연고전에 대해 서로 상반된 내용의 평가가 존재한다. 고연전/연고전이 연세대학교와 고려대학교의 자긍심을 고취하는 전통적인 스포츠 친선 경기이며<sup class="reference" id="cite_ref-.EC.97.B0.EA.B3.A0.EC.A0.84.EC.8B.9C.EC.84.A0_30-0"><a href="#cite_note-.EC.97.B0.EA.B3.A0.EC.A0.84.EC.8B.9C.EC.84.A0-30">[30]</a></sup> 대학 스포츠 발전에 이바지하였다는<sup class="reference" id="cite_ref-.EC.9D.B4.ED.98.9C.EC.A7.84_31-0"><a href="#cite_note-.EC.9D.B4.ED.98.9C.EC.A7.84-31">[31]</a></sup> 평가와 더불어 시대적 맥락을 저버린 채 즐거움만 추구하며<sup class="reference" id="cite_ref-.EC.97.B0.EA.B3.A0.EC.A0.84.EC.8B.9C.EC.84.A0_30-1"><a href="#cite_note-.EC.97.B0.EA.B3.A0.EC.A0.84.EC.8B.9C.EC.84.A0-30">[30]</a></sup> 폐쇄적인 엘리트 의식을 부추긴다는 비판이 있다.<sup class="reference" id="cite_ref-32"><a href="#cite_note-32">[32]</a></sup> 한편 선수들 간의 충돌이나 경기 파행 사태 등의 지나친 성적 지상주의도 부정적 측면으로 지적되고 있다.<sup class="reference" id="cite_ref-.EC.9D.B4.ED.98.9C.EC.A7.84_31-1"><a href="#cite_note-.EC.9D.B4.ED.98.9C.EC.A7.84-31">[31]</a></sup> <a href="/wiki/%EC%8B%A0%EC%B4%8C" title="신촌">신촌</a>과 <a href="/wiki/%EC%95%88%EC%95%94%EB%8F%99" title="안암동">안암</a>의 상가 일대에서 벌어지는 고연전의 뒤풀이 문화와 주변을 고려하지 않은 응원 태도에 대한 비판도 있다.<sup class="reference" id="cite_ref-33"><a href="#cite_note-33">[33]</a></sup> <a href="/wiki/2003%EB%85%84" title="2003년">2003년</a>에는 고려대학교내 일부 학생 단체에 의해 '안티 고·연전 실천단'이 결성되어 고연전을 반대하는 운동이 전개되기도 하였다.<sup class="reference" id="cite_ref-34"><a href="#cite_note-34">[34]</a></sup></p>
<h2><span class="mw-headline" id=".EC.82.AC.EA.B1.B4">사건</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EA%B3%A0%EB%A0%A4%EB%8C%80%ED%95%99%EA%B5%90-%EC%97%B0%EC%84%B8%EB%8C%80%ED%95%99%EA%B5%90_%EC%A0%95%EA%B8%B0%EC%A0%84&amp;action=edit&amp;section=7" title="부분 편집: 사건">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<h3><span class="mw-headline" id=".EA.B3.A0.EB.A0.A4.EB.8C.80.ED.95.99.EA.B5.90_.EC.B6.95.EA.B5.AC.EB.B6.80_.EA.B0.90.EB.8F.85_.EC.8B.AC.ED.8C.90.EB.A7.A4.EC.88.98.EC.82.AC.EA.B1.B4">고려대학교 축구부 감독 심판매수사건</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EA%B3%A0%EB%A0%A4%EB%8C%80%ED%95%99%EA%B5%90-%EC%97%B0%EC%84%B8%EB%8C%80%ED%95%99%EA%B5%90_%EC%A0%95%EA%B8%B0%EC%A0%84&amp;action=edit&amp;section=8" title="부분 편집: 고려대학교 축구부 감독 심판매수사건">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p><a class="mw-redirect" href="/wiki/%EC%84%9C%EC%9A%B8%EC%A7%80%EB%B0%A9%EA%B2%BD%EC%B0%B0%EC%B2%AD" title="서울지방경찰청">서울지방경찰청</a> 광역수사대는 시합에서 이기기 위해 상습적으로 심판을 매수하고 선수단의 공금을 횡령한 혐의로 전 <a href="/wiki/%EA%B3%A0%EB%A0%A4%EB%8C%80%ED%95%99%EA%B5%90_%EC%B6%95%EA%B5%AC%EB%B6%80" title="고려대학교 축구부">고려대학교 축구부</a> 감독에 대해 <a href="/wiki/2010%EB%85%84" title="2010년">2010년</a> <a href="/wiki/3%EC%9B%94_19%EC%9D%BC" title="3월 19일">3월 19일</a>에 구속영장을 신청했다. 광역수사대에 따르면 <a href="/wiki/2009%EB%85%84_9%EC%9B%94" title="2009년 9월">2009년 9월</a>에 열린 연고전에서 이기기 위해 <a href="/wiki/%EB%8C%80%ED%95%9C%EC%B6%95%EA%B5%AC%ED%98%91%ED%9A%8C" title="대한축구협회">대한축구협회</a> 경기분과 위원인 김 모에게 평소 친분이 있는 이 모와 윤 모를 심판으로 배정해 달라고 부탁했다고 하며 이런 방식으로 심판 11명에게 2천 3백여만원 상당의 금품을 제공했으며, <a href="/wiki/2008%EB%85%84" title="2008년">2008년</a>부터 <a href="/wiki/2009%EB%85%84_11%EC%9B%94" title="2009년 11월">2009년 11월</a>까지 경남 남해 축구대회, 전국대학축구대회 등 모두 9개 경기에서 승리했다. 또 운영비 등 명목으로 학부모로부터 억대의 돈을 받기도 하며 <a href="/wiki/2007%EB%85%84_11%EC%9B%94" title="2007년 11월">2007년 11월</a>부터 <a href="/wiki/2010%EB%85%84_12%EC%9B%94" title="2010년 12월">2010년 12월</a>까지 학부모 40여명으로부터 걷은 5억 8000여만원 중에서 1억 700여만원을 유흥비 등으로 탕진했다고 한다.<sup class="reference" id="cite_ref-35"><a href="#cite_note-35">[35]</a></sup><sup class="reference" id="cite_ref-36"><a href="#cite_note-36">[36]</a></sup></p>
