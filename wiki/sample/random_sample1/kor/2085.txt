<p><b>고르쉬코프급 호위함</b>(Admiral Gorshkov-class frigate)은 <a href="/wiki/%EB%9F%AC%EC%8B%9C%EC%95%84_%ED%95%B4%EA%B5%B0" title="러시아 해군">러시아 해군</a>의 최신형 <a class="mw-redirect" href="/wiki/%ED%98%B8%EC%9C%84%ED%95%A8" title="호위함">호위함</a>이다. 1980년대 건조한 만재배수량 4,400톤 <a class="new" href="/w/index.php?title=%EB%8B%88%EC%9A%B0%EC%8A%A4%ED%8A%B8%EB%9D%BC%EC%89%AC%EB%AF%B8%EA%B8%89_%ED%98%B8%EC%9C%84%ED%95%A8&amp;action=edit&amp;redlink=1" title="니우스트라쉬미급 호위함 (없는 문서)">니우스트라쉬미급 호위함</a>의 후속함이다.</p>
<h2><span class="mw-headline" id=".EC.83.9D.EC.82.B0">생산</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EA%B3%A0%EB%A5%B4%EC%89%AC%EC%BD%94%ED%94%84%EA%B8%89_%ED%98%B8%EC%9C%84%ED%95%A8&amp;action=edit&amp;section=1" title="부분 편집: 생산">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p><a href="/wiki/%EB%9F%AC%EC%8B%9C%EC%95%84_%ED%95%B4%EA%B5%B0" title="러시아 해군">러시아 해군</a>은 고르쉬코프급 20-30척을 요구했다. 2003년 7월 해군 사령부가 디자인을 승인했다. 21척이 건조된 만재배수량 8,480톤 <a href="/wiki/%EC%86%8C%EB%B8%8C%EB%A0%88%EB%A9%94%EB%8B%88%EA%B8%89_%EA%B5%AC%EC%B6%95%ED%95%A8" title="소브레메니급 구축함">소브레메니급 구축함</a>, 40척이 건조된 만재배수량 3,575톤 <a class="new" href="/w/index.php?title=%ED%81%AC%EB%A6%AC%EB%B0%95%EA%B8%89_%ED%98%B8%EC%9C%84%ED%95%A8&amp;action=edit&amp;redlink=1" title="크리박급 호위함 (없는 문서)">크리박급 호위함</a>을 대체하기 위한 계획이다.</p>
<p>2016년 4월 현재, 1, 2번함은 진수했고, 3, 4번함이 건조중이다.</p>
<p>1, 2번함은 우크라이나 국영 <a class="extiw" href="https://en.wikipedia.org/wiki/Zorya-Mashproekt" title="en:Zorya-Mashproekt">en:Zorya-Mashproekt</a>의 <a href="/wiki/%EA%B0%80%EC%8A%A4_%ED%84%B0%EB%B9%88" title="가스 터빈">가스 터빈</a> 엔진을 장착했다. <a class="mw-redirect" href="/wiki/2014%EB%85%84_%ED%81%AC%EB%A6%BC_%EB%B0%98%EB%8F%84_%EC%9C%84%EA%B8%B0" title="2014년 크림 반도 위기">2014년 크림 반도 위기</a>로 <a href="/wiki/%EC%9A%B0%ED%81%AC%EB%9D%BC%EC%9D%B4%EB%82%98" title="우크라이나">우크라이나</a>가 엔진 납품을 거부하자, 러시아 <a class="extiw" href="https://en.wikipedia.org/wiki/NPO_Saturn" title="en:NPO Saturn">en:NPO Saturn</a>사가 가스 터빈 엔진을 생산하기로 하였다. 2018년에나 엔진이 인도될 예정이었으나, 러시아 정부가 개입한 이후, 2016년에 엔진이 인도될 예정이다.</p>
<h2><span class="mw-headline" id=".EC.9D.B4.EC.A7.80.EC.8A.A4.ED.95.A8">이지스함</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EA%B3%A0%EB%A5%B4%EC%89%AC%EC%BD%94%ED%94%84%EA%B8%89_%ED%98%B8%EC%9C%84%ED%95%A8&amp;action=edit&amp;section=2" title="부분 편집: 이지스함">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>미국 <a class="mw-redirect" href="/wiki/%EC%9D%B4%EC%A7%80%EC%8A%A4%ED%95%A8" title="이지스함">이지스함</a>과 같이, 함교의 4면에 <a href="/wiki/AESA" title="AESA">AESA</a> 레이더를 부착했다. 32개의 <a class="mw-redirect" href="/wiki/VLS" title="VLS">VLS</a>에 <a href="/wiki/S-300" title="S-300">S-300</a> 함대공 미사일을 탑재했다. 러시아판 이지스함이다. 16발의 <a class="new" href="/w/index.php?title=SS-N-26&amp;action=edit&amp;redlink=1" title="SS-N-26 (없는 문서)">SS-N-26</a> 또는 <a href="/wiki/3M-54_%ED%81%B4%EB%9F%BD" title="3M-54 클럽">3M-54 클럽</a>(SS-N-27) 함대함, 함대지 미사일을 탑재했다.</p>
<h2><span class="mw-headline" id=".EC.88.98.EC.B6.9C.ED.98.95">수출형</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EA%B3%A0%EB%A5%B4%EC%89%AC%EC%BD%94%ED%94%84%EA%B8%89_%ED%98%B8%EC%9C%84%ED%95%A8&amp;action=edit&amp;section=3" title="부분 편집: 수출형">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>2010년 11월 3일 Euronaval-2010 국제 전시회에 수출형 버전 Project 22356가 소개되었다.</p>
