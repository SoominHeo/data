<h1 class="firstHeading" id="firstHeading" lang="ko">고모리 가즈타카</h1>
<div class="mw-body-content" id="bodyContent">
<div id="siteSub">위키백과, 우리 모두의 백과사전.</div>
<div id="contentSub"></div>
<div class="mw-jump" id="jump-to-nav">
					이동:					<a href="#mw-head">둘러보기</a>, 					<a href="#p-search">검색</a>
</div>
<div class="mw-content-ltr" dir="ltr" id="mw-content-text" lang="ko"><script>function mfTempOpenSection(id){var block=document.getElementById("mf-section-"+id);block.className+=" open-block";block.previousSibling.className+=" open-block";}</script><p><b>고모리 가즈타카</b>(小森 一孝, <a href="/wiki/1943%EB%85%84" title="1943년">1943년</a> - <a href="/wiki/1971%EB%85%84" title="1971년">1971년</a>)는 <a href="/wiki/%EC%9D%BC%EB%B3%B8" title="일본">일본</a>의 우익 테러리스트이다. <a href="/wiki/1960%EB%85%84" title="1960년">1960년</a> <a href="/wiki/%EB%82%98%EA%B0%80%EC%82%AC%ED%82%A4_%ED%98%84" title="나가사키 현">나가사키 현</a>립 <a class="new" href="/w/index.php?title=%EB%82%98%EA%B0%80%EC%82%AC%ED%82%A4_%EB%8F%99_%EA%B3%A0%EB%93%B1%ED%95%99%EA%B5%90&amp;action=edit&amp;redlink=1" title="나가사키 동 고등학교 (없는 문서)">나가사키 동 고등학교</a>를 중퇴 후 <a href="/wiki/%EB%8C%80%EC%9D%BC%EB%B3%B8%EC%95%A0%EA%B5%AD%EB%8B%B9" title="대일본애국당">대일본애국당</a>에 입당하였으며, <a href="/wiki/1959%EB%85%84" title="1959년">1959년</a> 후카자와 시치로(深澤七郞)의 <a href="/wiki/%EC%9D%BC%EB%B3%B8_%EC%B2%9C%ED%99%A9" title="일본 천황">일본 천황</a>과 황태자가 <a class="new" href="/w/index.php?title=%EC%9D%BC%EB%B3%B8%EB%AF%BC%EC%A4%91&amp;action=edit&amp;redlink=1" title="일본민중 (없는 문서)">일본민중</a>에 의해 참수형당한다는 내용의 글 『풍류몽담』(風流夢譚)를 게재한 <a class="new" href="/w/index.php?title=%EC%A3%BC%EC%98%A4%EA%B3%B5%EB%A1%A0%EC%82%AC&amp;action=edit&amp;redlink=1" title="주오공론사 (없는 문서)">주오공론사</a>의 사장 <a class="new" href="/w/index.php?title=%EC%8B%9C%EB%A7%88%EB%82%98%EA%B0%80&amp;action=edit&amp;redlink=1" title="시마나가 (없는 문서)">시마나가</a>의 집에 침입, 그의 아내와 가정부 <a class="new" href="/w/index.php?title=%EB%A7%88%EB%A3%A8%EC%95%BC%EB%A7%88_%EA%B0%80%EB%84%A4&amp;action=edit&amp;redlink=1" title="마루야마 가네 (없는 문서)">마루야마 가네</a>를 습격하였다. 미루야마는 현장에서 즉사했고 그는 살인혐의로 <a href="/wiki/1962%EB%85%84" title="1962년">1962년</a> <a class="new" href="/w/index.php?title=%EB%8F%84%EC%BF%84%EC%A7%80%EB%B0%A9%EC%9E%AC%ED%8C%90%EC%86%8C&amp;action=edit&amp;redlink=1" title="도쿄지방재판소 (없는 문서)">도쿄지방재판소</a>에서 징역 15년형을 받고 복역 중 <a href="/wiki/1971%EB%85%84" title="1971년">1971년</a> 옥사하였다.</p>
<h2><span class="mw-headline" id=".EC.95.BD.EB.A0.A5">약력</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EA%B3%A0%EB%AA%A8%EB%A6%AC_%EA%B0%80%EC%A6%88%ED%83%80%EC%B9%B4&amp;action=edit&amp;section=1" title="부분 편집: 약력">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>가즈타카는 <a href="/wiki/1943%EB%85%84" title="1943년">1943년</a> <a href="/wiki/%EB%82%98%EA%B0%80%EC%82%AC%ED%82%A4_%ED%98%84" title="나가사키 현">나가사키 현</a> <a class="new" href="/w/index.php?title=%EC%9D%B4%EC%82%AC%ED%95%98%EC%95%BC_%EA%B5%AC&amp;action=edit&amp;redlink=1" title="이사하야 구 (없는 문서)">이사하야 구</a>에서 <a class="new" href="/w/index.php?title=%EA%B3%A0%EB%AA%A8%EB%A6%AC_%EC%9A%94%EC%8B%9C%EC%98%A4&amp;action=edit&amp;redlink=1" title="고모리 요시오 (없는 문서)">고모리 요시오</a>의 3녀 1남 중 유일한 아들이자 막내로 태어났다. 아버지 요시오는 대만 출신 대만계 일본 법조인으로 패전 후 <a href="/wiki/%EC%9D%BC%EB%B3%B8" title="일본">일본</a>으로 귀국, <a href="/wiki/%EB%82%98%EA%B0%80%EC%82%AC%ED%82%A4_%ED%98%84" title="나가사키 현">나가사키 현</a>의 부검사를 지냈다. 나이차이가 있는 연상의 누나 세명 중 큰누나는 <a href="/wiki/%EC%B4%88%EB%93%B1%ED%95%99%EA%B5%90" title="초등학교">초등학교</a> 교사, 둘째 누나는 일찍이 독립해 의사로 개업하였다. 그러나 그의 어머니는 1947년 <a href="/wiki/%EC%A1%B0%ED%98%84%EB%B3%91" title="조현병">조현병</a>으로 투신 자살하였고 그의 셋째 누나 역시 <a href="/wiki/1959%EB%85%84" title="1959년">1959년</a> <a href="/wiki/8%EC%9B%94" title="8월">8월</a> 정신분열증으로 나가사키의대 병원에 입원 중 투신 자살하였다.</p>
<p>학교 성적은 우수하였으나 <a href="/wiki/1959%EB%85%84" title="1959년">1959년</a> <a href="/wiki/%EB%82%98%EA%B0%80%EC%82%AC%ED%82%A4_%ED%98%84" title="나가사키 현">나가사키 현</a>립 <a class="new" href="/w/index.php?title=%EB%82%98%EA%B0%80%EC%82%AC%ED%82%A4_%EB%8F%99_%EA%B3%A0%EB%93%B1%ED%95%99%EA%B5%90&amp;action=edit&amp;redlink=1" title="나가사키 동 고등학교 (없는 문서)">나가사키 동 고등학교</a> 1학년 재학 중 우울증과 정신분열증으로 아버지 요시오에 의해 나가사키의대 병원에서 진료를 받기도 했다.</p>
<p><a href="/wiki/1960%EB%85%84" title="1960년">1960년</a> <a href="/wiki/8%EC%9B%94" title="8월">8월</a> 학업 수행이 어려워 <a href="/wiki/%EB%82%98%EA%B0%80%EC%82%AC%ED%82%A4_%ED%98%84" title="나가사키 현">나가사키 현</a>립<a class="new" href="/w/index.php?title=%EB%82%98%EA%B0%80%EC%82%AC%ED%82%A4_%EB%8F%99_%EA%B3%A0%EB%93%B1%ED%95%99%EA%B5%90&amp;action=edit&amp;redlink=1" title="나가사키 동 고등학교 (없는 문서)">나가사키 동 고등학교</a>를 중퇴 후 <a href="/wiki/%EB%8C%80%EC%9D%BC%EB%B3%B8%EC%95%A0%EA%B5%AD%EB%8B%B9" title="대일본애국당">대일본애국당</a>에 입당하였으며, <a href="/wiki/1959%EB%85%84" title="1959년">1959년</a> 후카자와 시치로(深澤七郞)의 <a href="/wiki/%EC%9D%BC%EB%B3%B8_%EC%B2%9C%ED%99%A9" title="일본 천황">일본 천황</a>과 황태자가 <a class="new" href="/w/index.php?title=%EC%9D%BC%EB%B3%B8%EB%AF%BC%EC%A4%91&amp;action=edit&amp;redlink=1" title="일본민중 (없는 문서)">일본민중</a>에 의해 참수형당한다는 내용의 글 『풍류몽담』(風流夢譚)를 게재한 <a class="new" href="/w/index.php?title=%EC%A3%BC%EC%98%A4%EA%B3%B5%EB%A1%A0%EC%82%AC&amp;action=edit&amp;redlink=1" title="주오공론사 (없는 문서)">주오공론사</a>의 사장 <a class="new" href="/w/index.php?title=%EC%8B%9C%EB%A7%88%EB%82%98%EA%B0%80&amp;action=edit&amp;redlink=1" title="시마나가 (없는 문서)">시마나가</a>를 죽이기로 결심, <a href="/wiki/1960%EB%85%84" title="1960년">1960년</a> <a href="/wiki/2%EC%9B%94_1%EC%9D%BC" title="2월 1일">2월 1일</a> 시마나가의 집에 침입하였으나 시마나가 사장을 찾았으나 가정부 마루야마 가네가 사장은 출타중이라고 하자 그의 아내와 가정부 마루야마를 습격하였다.</p>
<p>마루야마는 현장에서 즉사했고 수배령이 나간 후 그는 <a href="/wiki/2%EC%9B%94_2%EC%9D%BC" title="2월 2일">2월 2일</a> 경시청에 자수하였다. 살인혐의로 기소, 조사를 받았으며 그가 <a href="/wiki/%EB%8C%80%EC%9D%BC%EB%B3%B8%EC%95%A0%EA%B5%AD%EB%8B%B9" title="대일본애국당">대일본애국당</a>의 당원 신분이란 점을 이유로 <a href="/wiki/%EB%8C%80%EC%9D%BC%EB%B3%B8%EC%95%A0%EA%B5%AD%EB%8B%B9" title="대일본애국당">대일본애국당</a>의 총재 <a href="/wiki/%EC%95%84%EC%B9%B4%EC%98%A4_%EB%B9%88" title="아카오 빈">아카오 빈</a>은 암살 배후로 지목되어 체포, 조사를 받기도 했다. <a href="/wiki/1962%EB%85%84" title="1962년">1962년</a> <a class="new" href="/w/index.php?title=%EB%8F%84%EC%BF%84%EC%A7%80%EB%B0%A9%EC%9E%AC%ED%8C%90%EC%86%8C&amp;action=edit&amp;redlink=1" title="도쿄지방재판소 (없는 문서)">도쿄지방재판소</a>에서 징역 15년형을 받고 <a href="/wiki/1964%EB%85%84" title="1964년">1964년</a> 도쿄 고등재판소에서 형이 확정되어 복역하던 중 <a href="/wiki/1971%EB%85%84" title="1971년">1971년</a> <a href="/wiki/11%EC%9B%94" title="11월">11월</a> 옥사하였다. 옥사 직전 한 여성잡지가 그의 근황을 인터뷰했다고 한다. 그의 나이 28세였다.</p>
