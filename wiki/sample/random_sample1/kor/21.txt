<p><b>서먼 로웰 터커</b>(<span lang="en" xml:lang="en">Thurman Lowell Tucker</span>, <a href="/wiki/1917%EB%85%84" title="1917년">1917년</a> <a href="/wiki/9%EC%9B%94_26%EC%9D%BC" title="9월 26일">9월 26일</a> ~ <a href="/wiki/1993%EB%85%84" title="1993년">1993년</a> <a href="/wiki/5%EC%9B%94_7%EC%9D%BC" title="5월 7일">5월 7일</a>)는 미국의 전 <a href="/wiki/%ED%94%84%EB%A1%9C_%EC%95%BC%EA%B5%AC" title="프로 야구">프로 야구</a> 선수이다. <a href="/wiki/%EB%A9%94%EC%9D%B4%EC%A0%80_%EB%A6%AC%EA%B7%B8_%EB%B2%A0%EC%9D%B4%EC%8A%A4%EB%B3%BC" title="메이저 리그 베이스볼">메이저 리그 베이스볼</a>(MLB)에서 <a href="/wiki/%EC%A4%91%EA%B2%AC%EC%88%98" title="중견수">중견수</a>로 활약하였으며, 9시즌 동안 <a href="/wiki/%EC%95%84%EB%A9%94%EB%A6%AC%EC%B9%B8_%EB%A6%AC%EA%B7%B8" title="아메리칸 리그">아메리칸 리그</a>의 <a href="/wiki/%EC%8B%9C%EC%B9%B4%EA%B3%A0_%ED%99%94%EC%9D%B4%ED%8A%B8%EC%82%AD%EC%8A%A4" title="시카고 화이트삭스">시카고 화이트삭스</a>와 <a href="/wiki/%ED%81%B4%EB%A6%AC%EB%B8%94%EB%9E%9C%EB%93%9C_%EC%9D%B8%EB%94%94%EC%96%B8%EC%8A%A4" title="클리블랜드 인디언스">클리블랜드 인디언스</a>에서 뛰었다. 통산 성적은 701경기에 출전해 <a href="/wiki/%ED%83%80%EC%9C%A8" title="타율">타율</a> .255, 24개 <a href="/wiki/3%EB%A3%A8%ED%83%80" title="3루타">3루타</a>, 9<a href="/wiki/%ED%99%88%EB%9F%B0" title="홈런">홈런</a>, 179<a href="/wiki/%ED%83%80%EC%A0%90" title="타점">타점</a>이다. 별명은 <b>"조 E."</b>(Joe E.)였는데, 외모가 당대 코미디언 <a href="/wiki/%EC%A1%B0_E._%EB%B8%8C%EB%9D%BC%EC%9A%B4" title="조 E. 브라운">조 E. 브라운</a>과 많이 닮았기 때문이었다.<sup class="reference" id="cite_ref-digest_1-0"><a href="#cite_note-digest-1">[1]</a></sup></p>
<p>텍사스 주에서 나고 자랐으며, <a class="new" href="/w/index.php?title=%EC%8B%A4%EB%A1%9C%EC%95%94%EC%8A%A4%ED%94%84%EB%A7%81%EC%8A%A4_%ED%8A%B8%EB%A0%88%EB%B8%94%EB%9F%AC%EC%8A%A4&amp;action=edit&amp;redlink=1" title="실로암스프링스 트레블러스 (없는 문서)">실로암스프링스 트레블러스</a>에서 프로 경력을 시작했다. <a href="/wiki/%EB%A7%88%EC%9D%B4%EB%84%88_%EB%A6%AC%EA%B7%B8_%EB%B2%A0%EC%9D%B4%EC%8A%A4%EB%B3%BC" title="마이너 리그 베이스볼">마이너 리그 베이스볼</a>에서 차례차례 과정을 밟은 뒤, 1941년 시카고 화이트삭스와 계약해 다음해 메이저 리그에 데뷔했다. 2년간 화이트삭스의 주전 중견수로 뛰다가 제2차 세계 대전으로 군대에 징집되었다. 복귀 후 화이트삭스에서 두 시즌을 더 뛰다가 클리블랜드 인디언스로 트레이드되었다. 클리블랜드에서 4년을 뛰었으며, 1950년대까지 마이너 리그에서 뛰었다. 은퇴 후에는 메이저 리그 스카우트와 보험설계사로 일했다.</p>

<h2><span class="mw-headline" id=".EC.96.B4.EB.A6.B0_.EC.8B.9C.EC.A0.88">어린 시절</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%84%9C%EB%A8%BC_%ED%84%B0%EC%BB%A4&amp;action=edit&amp;section=1" title="부분 편집: 어린 시절">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>서먼 터커는 1917년 9월 26일 태어나 <a href="/wiki/%ED%85%8D%EC%82%AC%EC%8A%A4_%EC%A3%BC" title="텍사스 주">텍사스 주</a> <a href="/wiki/%EA%B3%A0%EB%93%A0_(%ED%85%8D%EC%82%AC%EC%8A%A4_%EC%A3%BC)" title="고든 (텍사스 주)">고든</a>에서 자랐다. 고등학교 시절 야구에선 <a href="/wiki/2%EB%A3%A8%EC%88%98" title="2루수">2루수</a>로 활약했고, <a href="/wiki/%EB%86%8D%EA%B5%AC" title="농구">농구</a>와 <a href="/wiki/%EC%9C%A1%EC%83%81_%EA%B2%BD%EA%B8%B0" title="육상 경기">육상</a> 선수로도 뛰었는데, 세 종목 모두 두각을 나타냈다. 1935년 졸업 후 세미프로 야구를 뛰었고 <a href="/wiki/%EC%95%84%EC%B9%B8%EC%86%8C_%EC%A3%BC" title="아칸소 주">아칸소 주</a> <a href="/wiki/%ED%95%AB%EC%8A%A4%ED%94%84%EB%A7%81%EC%8A%A4_(%EC%95%84%EC%B9%B8%EC%86%8C_%EC%A3%BC)" title="핫스프링스 (아칸소 주)">핫스프링스</a>에 있는 야구 학교에 다녔다. 1936년, 18세의 나이에 터커는 <a class="new" href="/w/index.php?title=%EC%95%84%EC%B9%B8%EC%86%8C-%EB%AF%B8%EC%A3%BC%EB%A6%AC_%EB%A6%AC%EA%B7%B8&amp;action=edit&amp;redlink=1" title="아칸소-미주리 리그 (없는 문서)">아칸소-미주리 리그</a>의 <a class="new" href="/w/index.php?title=%ED%8E%98%EC%9D%B4%EC%97%A3%EB%B9%8C_%EB%B2%A0%EC%96%B4%EC%8A%A4&amp;action=edit&amp;redlink=1" title="페이엣빌 베어스 (없는 문서)">페이엣빌 베어스</a>와 프로 계약을 했으나 한 경기도 뛰지 않고 2주만에 팀에서 나왔다. 얼마 안되어 같은 리그에 소속된 <a class="new" href="/w/index.php?title=%EC%8B%A4%EB%A1%9C%EC%95%94%EC%8A%A4%ED%94%84%EB%A7%81%EC%8A%A4_%ED%8A%B8%EB%A0%88%EB%B8%94%EB%9F%AC%EC%8A%A4&amp;action=edit&amp;redlink=1" title="실로암스프링스 트레블러스 (없는 문서)">실로암스프링스 트레블러스</a>와 계약하며 프로 경력을 시작했다.<sup class="reference" id="cite_ref-vaughan_2-0"><a href="#cite_note-vaughan-2">[2]</a></sup></p>
<h2><span class="mw-headline" id=".EB.A7.88.EC.9D.B4.EB.84.88_.EB.A6.AC.EA.B7.B8_.EA.B2.BD.EB.A0.A5">마이너 리그 경력</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%84%9C%EB%A8%BC_%ED%84%B0%EC%BB%A4&amp;action=edit&amp;section=2" title="부분 편집: 마이너 리그 경력">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>트레블러스에서의 첫 시즌이었던 1936년, 터커는 수비 위치를 바꿔 외야수로 나서며<sup class="reference" id="cite_ref-vaughan_2-1"><a href="#cite_note-vaughan-2">[2]</a></sup> 117경기에서 <a href="/wiki/%ED%83%80%EC%9C%A8" title="타율">타율</a> .319, <a href="/wiki/2%EB%A3%A8%ED%83%80" title="2루타">2루타</a> 25개를 기록했다.<sup class="reference" id="cite_ref-brm_3-0"><a href="#cite_note-brm-3">[3]</a></sup> 다음해에는 <a class="new" href="/w/index.php?title=%EC%BD%94%ED%8A%BC_%EC%8A%A4%ED%85%8C%EC%9D%B4%ED%8A%B8_%EB%A6%AC%EA%B7%B8&amp;action=edit&amp;redlink=1" title="코튼 스테이트 리그 (없는 문서)">코튼 스테이트 리그</a>의 <a class="new" href="/w/index.php?title=%EC%97%98%EB%8F%84%EB%9D%BC%EB%8F%84_%EB%9D%BC%EC%9D%B4%EC%96%B8%EC%8A%A4&amp;action=edit&amp;redlink=1" title="엘도라도 라이언스 (없는 문서)">엘도라도 라이언스</a>에서 뛰려고 했지만 허리 부상으로 인해 거의 시즌 전체를 날렸다.<sup class="reference" id="cite_ref-vaughan_2-2"><a href="#cite_note-vaughan-2">[2]</a></sup> 1938년 시즌에는 <a class="new" href="/w/index.php?title=%EC%97%90%EB%B0%98%EC%A0%A4%EB%A6%B0_%EB%B2%A0%EC%9D%B4%EC%8A%A4%EB%B3%BC_%EB%A6%AC%EA%B7%B8&amp;action=edit&amp;redlink=1" title="에반젤린 베이스볼 리그 (없는 문서)">에반젤린 베이스볼 리그</a>의 <a class="new" href="/w/index.php?title=%EC%95%A0%EB%B9%84%EB%B9%8C_%EC%97%90%EC%9D%B4%EC%8A%A4&amp;action=edit&amp;redlink=1" title="애비빌 에이스 (없는 문서)">애비빌 에이스</a>와 코튼 스테이트 리그의 <a class="new" href="/w/index.php?title=%EA%B7%B8%EB%A6%B0%EB%B9%8C_%EB%B2%85%EC%8A%A4&amp;action=edit&amp;redlink=1" title="그린빌 벅스 (없는 문서)">그린빌 벅스</a>에서 각각 55경기와 50경기를 뛰었다.<sup class="reference" id="cite_ref-brm_3-1"><a href="#cite_note-brm-3">[3]</a></sup> 1939년에도 코튼 스테이트 리그에 소속된 <a href="/wiki/%EB%B3%B4%EC%8A%A4%ED%84%B4_%EB%A0%88%EB%93%9C%EC%82%AD%EC%8A%A4" title="보스턴 레드삭스">보스턴 레드삭스</a>의 마이너 리그 팀 <a class="new" href="/w/index.php?title=%ED%81%B4%EB%9D%BD%EC%8A%A4%EB%8D%B0%EC%9D%BC_%EB%A0%88%EB%93%9C%EC%82%AD%EC%8A%A4&amp;action=edit&amp;redlink=1" title="클락스데일 레드삭스 (없는 문서)">클락스데일 레드삭스</a>에서 뛰며 136경기 타율 .298, <a href="/wiki/3%EB%A3%A8%ED%83%80" title="3루타">3루타</a> 10개를 기록했다.<sup class="reference" id="cite_ref-brm_3-2"><a href="#cite_note-brm-3">[3]</a></sup> 마이너 리그 시절, 한 감독이 그에게 <a href="/wiki/%EA%B4%91%EB%8C%80" title="광대">광대</a> 분장이 어울리겠다고 이야기했는데, 그가 코미디언 <a href="/wiki/%EC%A1%B0_E._%EB%B8%8C%EB%9D%BC%EC%9A%B4" title="조 E. 브라운">조 E. 브라운</a>과 닮았기 때문이었다. 그 이후로 그에게는 조 E.라는 별명이 붙었는데, 터커는 평소 진지한 성격이어서 그걸 별로 마음에 들어하지는 않았다.<sup class="reference" id="cite_ref-vaughan_2-3"><a href="#cite_note-vaughan-2">[2]</a></sup></p>
<p>클락스데일 2년차이던 1940년은 터커에게 있어서 새로운 변혁의 시즌이었다. 6월말까지 .374의 타율로 코튼 스테이트 리그 타격 1위를 달렸다.<sup class="reference" id="cite_ref-4"><a href="#cite_note-4">[4]</a></sup> 모두 97경기에 출전해 .390의 고감도의 타격을 선보였으며, <a class="new" href="/w/index.php?title=%EC%98%A4%ED%81%B4%EB%9D%BC%ED%98%B8%EB%A7%88%EC%8B%9C%ED%8B%B0_%EC%9D%B8%EB%94%94%EC%96%B8%EC%8A%A4&amp;action=edit&amp;redlink=1" title="오클라호마시티 인디언스 (없는 문서)">오클라호마시티 인디언스</a>로 승격되어 그곳에서 40경기를 뛰었다.<sup class="reference" id="cite_ref-brm_3-3"><a href="#cite_note-brm-3">[3]</a></sup> 그런 그의 활약을 눈여겨 보고 있던 시카고 화이트삭스의 <a href="/wiki/%ED%8C%9C%ED%8C%80" title="팜팀">팜</a> 매니저 빌리 웹은 시즌 후 오클라호마시티에게서 그의 계약을 사들였다.<sup class="reference" id="cite_ref-5"><a href="#cite_note-5">[5]</a></sup> 1941년 시즌 시작과 함께 터커는 새로운 소속팀 화이트삭스에서 <a class="new" href="/w/index.php?title=%EB%8D%B0%EC%9D%B4%EB%B8%8C_%EC%87%BC%ED%8A%B8&amp;action=edit&amp;redlink=1" title="데이브 쇼트 (없는 문서)">데이브 쇼트</a> 등과 마지막 외야 자리를 놓고 경쟁을 벌였지만 밀려났고,<sup class="reference" id="cite_ref-6"><a href="#cite_note-6">[6]</a></sup> 그해 시즌을 <a href="/wiki/%EB%A1%9C%EC%A0%80%EC%8A%A4_%ED%98%BC%EC%A6%88%EB%B9%84" title="로저스 혼즈비">로저스 혼즈비</a>가 이끄는 오클라호마시티 인디언스에서 보냈다.<sup class="reference" id="cite_ref-7"><a href="#cite_note-7">[7]</a></sup> 인디언스에서의 시즌 성적은 141경기에서 타율 .246, 3루타 12개를 기록했다.<sup class="reference" id="cite_ref-brm_3-4"><a href="#cite_note-brm-3">[3]</a></sup></p>
<p>1942년 시즌을 앞둔 스프링 트레이닝에서 마지막 외야 자리를 놓고 <a class="new" href="/w/index.php?title=%EB%8D%B0%EC%9D%B4%EB%B8%8C_%ED%95%84%EB%A6%AC&amp;action=edit&amp;redlink=1" title="데이브 필리 (없는 문서)">데이브 필리</a>와 경쟁했고,<sup class="reference" id="cite_ref-8"><a href="#cite_note-8">[8]</a></sup> 경영진에게 높이 평가받은 수비 능력 덕분에 메이저 리그 로스터에 이름을 올릴 수 있었다. 그해 4월 14일, 메이저 리그에 데뷔했지만,<sup class="reference" id="cite_ref-br_9-0"><a href="#cite_note-br-9">[9]</a></sup> 이날 경기를 포함해 겨우 두 경기에만 출전 기회를 부여받은 후 <a href="/wiki/%ED%85%8D%EC%82%AC%EC%8A%A4_%EB%A6%AC%EA%B7%B8" title="텍사스 리그">텍사스 리그</a>의 <a class="new" href="/w/index.php?title=%ED%8F%AC%ED%8A%B8%EC%9B%8C%EC%8A%A4_%EC%BA%90%EC%B8%A0&amp;action=edit&amp;redlink=1" title="포트워스 캐츠 (없는 문서)">포트워스 캐츠</a>로 내려갔다.<sup class="reference" id="cite_ref-10"><a href="#cite_note-10">[10]</a></sup> 그곳에서 144경기에 출전해 .313의 타율을 기록했으며,<sup class="reference" id="cite_ref-brm_3-5"><a href="#cite_note-brm-3">[3]</a></sup> 마이너 리그가 종료된 후 다시 메이저 리그로 승격되어 화이트삭스에서 그해 시즌 총 7경기를 뛰었다.<sup class="reference" id="cite_ref-br_9-1"><a href="#cite_note-br-9">[9]</a></sup></p>
<h2><span class="mw-headline" id=".EC.8B.9C.EC.B9.B4.EA.B3.A0_.ED.99.94.EC.9D.B4.ED.8A.B8.EC.82.AD.EC.8A.A4">시카고 화이트삭스</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%84%9C%EB%A8%BC_%ED%84%B0%EC%BB%A4&amp;action=edit&amp;section=3" title="부분 편집: 시카고 화이트삭스">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>1943년 시즌을 앞둔 시점에서 시카고 화이트삭스의 외야에서 주전 자리를 확고히 한 선수는 <a href="/wiki/%EC%9B%94%EB%A6%AC_%EB%AA%A8%EC%A7%80%EC%8A%A4" title="월리 모지스">월리 모지스</a> 뿐이었다. 스프링 트레이닝에서 주전 중견수 자리를 놓고 터커는 <a class="new" href="/w/index.php?title=%EB%AC%B4%EC%8A%A4_%EC%86%94%ED%84%B0%EC%8A%A4&amp;action=edit&amp;redlink=1" title="무스 솔터스 (없는 문서)">무스 솔터스</a>와 경쟁을 벌이며 <a href="/wiki/%EC%A7%80%EB%AF%B8_%EB%8B%A4%EC%9D%B4%ED%81%AC%EC%8A%A4" title="지미 다이크스">지미 다이크스</a> 감독에게 깊은 인상을 남겼고,<sup class="reference" id="cite_ref-11"><a href="#cite_note-11">[11]</a></sup> 결국 그는 주전 중견수와 <a href="/wiki/%EB%A6%AC%EB%93%9C_%EC%98%A4%ED%94%84" title="리드 오프">리드 오프</a>를 맡게 되었다.<sup class="reference" id="cite_ref-12"><a href="#cite_note-12">[12]</a></sup> 그의 활약은 당시 제2차 세계 대전으로 스타들이 빠져나가 근심하고 있던 <a class="new" href="/w/index.php?title=%EC%9C%8C_%ED%95%B4%EB%A6%AC%EC%A7%80&amp;action=edit&amp;redlink=1" title="윌 해리지 (없는 문서)">윌 해리지</a> 아메리칸 리그 회장이 새롭게 주목할 정도였다.<sup class="reference" id="cite_ref-13"><a href="#cite_note-13">[13]</a></sup> 이해 7월 26일 <a href="/wiki/%EB%89%B4%EC%9A%95_%EC%96%91%ED%82%A4%EC%8A%A4" title="뉴욕 양키스">뉴욕 양키스</a>와의 경기에서 그는 <a href="/wiki/%EB%81%9D%EB%82%B4%EA%B8%B0_%ED%99%88%EB%9F%B0" title="끝내기 홈런">끝내기 홈런</a>을 터뜨리며 팀의 2-1 승리를 이끌었다.<sup class="reference" id="cite_ref-14"><a href="#cite_note-14">[14]</a></sup> 시즌 성적은 135경기에서 타율 .235, 6개 3루타, 79<a href="/wiki/%EB%B3%BC%EB%84%B7" title="볼넷">볼넷</a>을 기록했다. 29도루로 아메리칸 리그 3위를 기록했지만, 17번의 도루실패 또한 리그 2위였다.<sup class="reference" id="cite_ref-br_9-2"><a href="#cite_note-br-9">[9]</a></sup></p>
<p>1944년 시즌 전 <a href="/wiki/%EB%AF%B8%EA%B5%AD_%ED%95%B4%EA%B5%B0" title="미국 해군">해군</a> 입대를 위해서 받은 신체 검사를 통과했다. 그해 바로 나라의 부름을 받고 입대할 예정이었지만,<sup class="reference" id="cite_ref-15"><a href="#cite_note-15">[15]</a></sup> 계획과는 다르게 풀타임을 소화했다. 시즌 첫 한 달은 쾌조의 타격을 선보이며, 5월 16일까지 .403의 타율로 아메리칸 리그 1위를 달렸다.<sup class="reference" id="cite_ref-16"><a href="#cite_note-16">[16]</a></sup> 다방면의 활약에 여러 야구 관계자들도 그에게 좋은 평가를 아끼지 않았다. 스포츠기자 <a class="new" href="/w/index.php?title=%ED%94%84%EB%A0%88%EB%93%9C_%EB%A6%AC%EC%97%A1&amp;action=edit&amp;redlink=1" title="프레드 리엡 (없는 문서)">프레드 리엡</a>은 올해 잠재력을 터뜨린 선수로 터커를 꼽았고,<sup class="reference" id="cite_ref-17"><a href="#cite_note-17">[17]</a></sup> 지미 다이크스 감독은 그를 두고 아메리칸 리그에서 최고의 외야수 중 하나로 이야기했다.<sup class="reference" id="cite_ref-18"><a href="#cite_note-18">[18]</a></sup> 6월까지 아메리칸 리그에서는 터커가 타율 .369로, 내셔널 리그에서는 <a class="new" href="/w/index.php?title=%EB%94%95%EC%8B%9C_%EC%9B%8C%EC%BB%A4&amp;action=edit&amp;redlink=1" title="딕시 워커 (없는 문서)">딕시 워커</a>가 타율 .377로 각 리그에서 타격왕 경쟁을 이끌었다.<sup class="reference" id="cite_ref-19"><a href="#cite_note-19">[19]</a></sup> 이러한 활약으로 1944년 <a href="/wiki/%EB%A9%94%EC%9D%B4%EC%A0%80_%EB%A6%AC%EA%B7%B8_%EB%B2%A0%EC%9D%B4%EC%8A%A4%EB%B3%BC_%EC%98%AC%EC%8A%A4%ED%83%80%EC%A0%84" title="메이저 리그 베이스볼 올스타전">올스타</a> 선수 명단에 경력 최초이자 마지막으로 이름을 올렸으며,<sup class="reference" id="cite_ref-20"><a href="#cite_note-20">[20]</a></sup> <a class="new" href="/w/index.php?title=1944%EB%85%84_%EB%A9%94%EC%9D%B4%EC%A0%80_%EB%A6%AC%EA%B7%B8_%EB%B2%A0%EC%9D%B4%EC%8A%A4%EB%B3%BC_%EC%98%AC%EC%8A%A4%ED%83%80%EC%A0%84&amp;action=edit&amp;redlink=1" title="1944년 메이저 리그 베이스볼 올스타전 (없는 문서)">올스타전</a>에서 리드 오프로 나서 4타수 무안타를 기록했다.<sup class="reference" id="cite_ref-21"><a href="#cite_note-21">[21]</a></sup></p>
<p>하지만 후반기에 들어서 타격감이 무뎌져 7월초 28타수 무안타를 기록했고, 타율이 .375에서 .327로 수직하락하며 리그 타격 1위의 자리를 다른 선수에게 내주고 말았다.<sup class="reference" id="cite_ref-22"><a href="#cite_note-22">[22]</a></sup> 35타수 1안타로 부진이 길어지며 타율은 .320으로 떨어졌고, 잠시 휴식을 위해 <a href="/wiki/%EB%94%94%ED%8A%B8%EB%A1%9C%EC%9D%B4%ED%8A%B8_%ED%83%80%EC%9D%B4%EA%B1%B0%EC%8A%A4" title="디트로이트 타이거스">디트로이트 타이거스</a>와의 주말 경기에서는 선발 출전하지 않았다.<sup class="reference" id="cite_ref-23"><a href="#cite_note-23">[23]</a></sup> 이후 금방 선발 라인업에 복귀했지만, 시즌 최종 기록은 타율 .287, 6개의 3루타만을 기록했다.<sup class="reference" id="cite_ref-br_9-3"><a href="#cite_note-br-9">[9]</a></sup> 7월 말에 화이트삭스 구단에서는 전쟁을 위한 자선 행사를 열었고, 여기에서 터커와 <a class="new" href="/w/index.php?title=%EC%A1%B0%EC%A7%80_%EC%BC%80%EC%9D%B4%EC%8A%A4&amp;action=edit&amp;redlink=1" title="조지 케이스 (없는 문서)">조지 케이스</a>는 75야드 경주를 벌여 1야드 차이로 케이스가 승리했다.<sup class="reference" id="cite_ref-24"><a href="#cite_note-24">[24]</a></sup> 시즌이 끝났 뒤 정식으로 해군에 입대하면서 1945년 시즌은 뛰지 않게 되었다.<sup class="reference" id="cite_ref-25"><a href="#cite_note-25">[25]</a></sup></p>
<p>전쟁이 끝나고 다른 선수들과 마찬가지로 복귀했으며, 1946년 시즌 화이트삭스의 외야진은 월리 모지스, <a class="new" href="/w/index.php?title=%ED%83%9C%ED%94%BC_%EB%9D%BC%EC%9D%B4%ED%8A%B8&amp;action=edit&amp;redlink=1" title="태피 라이트 (없는 문서)">태피 라이트</a>, 그리고 주전 중견수 자리의 터커로 예상되었다.<sup class="reference" id="cite_ref-26"><a href="#cite_note-26">[26]</a></sup> 하지만 터커는 그전 시즌과는 다르게 타격에서 부진을 면치 못했고 그저 수비는 좋지만 타격은 별로인 선수 정도로 평가가 깎였다.<sup class="reference" id="cite_ref-27"><a href="#cite_note-27">[27]</a></sup> 6월말까지 타율은 .229에 머물렀는데 1944년의 같은 시기보다 무려 1할 5푼 가량 낮았다.<sup class="reference" id="cite_ref-28"><a href="#cite_note-28">[28]</a></sup> 하지만 후반기에 타격감이 올라오면서 시즌 기록을 타율 .288, 20개 2루타의 커리어 하이로 마쳤다.<sup class="reference" id="cite_ref-br_9-4"><a href="#cite_note-br-9">[9]</a></sup> 그해 시즌 마지막 한 달 동안 4할의 타율을 기록한 덕에 다음 시즌에도 여전히 자신의 중견수 자리를 굳건하게 지켰으며, 팀의 좌익수에는 루키 데이브 필리가, 우익수에는 태피 라이트와 <a href="/wiki/%EB%B0%A5_%EC%BC%80%EB%84%A4%EB%94%94" title="밥 케네디">밥 케네디</a>가 <a class="new" href="/w/index.php?title=%ED%94%8C%EB%9E%98%ED%88%B0_%EC%8B%9C%EC%8A%A4%ED%85%9C&amp;action=edit&amp;redlink=1" title="플래툰 시스템 (없는 문서)">플래툰 시스템</a>으로 번갈아가며 출전했다.<sup class="reference" id="cite_ref-29"><a href="#cite_note-29">[29]</a></sup> 시즌 중 위장병으로 종종 결장하며 5월 중순까지 꾸준하게 출전하지 못하다가,<sup class="reference" id="cite_ref-30"><a href="#cite_note-30">[30]</a></sup> 다시 꾸준히 출장하기 시작할 수 있게 되면서부터는 필리와 번갈아 가며 경기에 나왔다. 시즌 최종 기록은 89경기 출전 타율 .236이었다.<sup class="reference" id="cite_ref-br_9-5"><a href="#cite_note-br-9">[9]</a></sup></p>
<h2><span class="mw-headline" id=".ED.81.B4.EB.A6.AC.EB.B8.94.EB.9E.9C.EB.93.9C_.EC.9D.B8.EB.94.94.EC.96.B8.EC.8A.A4">클리블랜드 인디언스</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%84%9C%EB%A8%BC_%ED%84%B0%EC%BB%A4&amp;action=edit&amp;section=4" title="부분 편집: 클리블랜드 인디언스">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>1948년 1월 27일 <a href="/wiki/%ED%81%B4%EB%A6%AC%EB%B8%94%EB%9E%9C%EB%93%9C_%EC%9D%B8%EB%94%94%EC%96%B8%EC%8A%A4" title="클리블랜드 인디언스">클리블랜드 인디언스</a>의 <a class="new" href="/w/index.php?title=%EB%9E%84%ED%94%84_%EC%99%80%EC%9D%B4%EA%B2%94&amp;action=edit&amp;redlink=1" title="랄프 와이겔 (없는 문서)">랄프 와이겔</a>의 맞상대로 트레이드되었는데, 클리블랜드 프런트는 터커를 "최고의 야수 중 한 명"로 높이 평가했다.<sup class="reference" id="cite_ref-31"><a href="#cite_note-31">[31]</a></sup> 주전 중견수로 시즌을 시작했으며, 시즌 초 디트로이트 타이거스와의 한 경기에서는 8-2로 승리했으나 팀에서 유일하게 안타를 치지 못한 선수로 기록되었다.<sup class="reference" id="cite_ref-32"><a href="#cite_note-32">[32]</a></sup> 6월에는 몸에 맞는 공으로 손가락이 부러져 3주간 결장했으며,<sup class="reference" id="cite_ref-33"><a href="#cite_note-33">[33]</a></sup> 라인업 복귀 후 시즌 후반기부터는 <a href="/wiki/%EC%A0%9C4%EC%9D%98_%EC%99%B8%EC%95%BC%EC%88%98" title="제4의 외야수">제4의 외야수</a>의 역할을 하며 <a href="/wiki/%EC%95%A8%EB%A6%AC_%ED%81%B4%EB%9D%BC%ED%81%AC" title="앨리 클라크">앨리 클라크</a>와 번갈아 뛰었다. 당시 팀의 외야진은 <a href="/wiki/%EB%9E%98%EB%A6%AC_%EB%8F%84%EB%B9%84" title="래리 도비">래리 도비</a>와 <a class="new" href="/w/index.php?title=%EB%8D%B0%EC%9D%BC_%EB%AF%B8%EC%B2%BC_(%EC%95%BC%EA%B5%AC_%EC%84%A0%EC%88%98)&amp;action=edit&amp;redlink=1" title="데일 미첼 (야구 선수) (없는 문서)">데일 미첼</a>로 구성되어 있었다.<sup class="reference" id="cite_ref-34"><a href="#cite_note-34">[34]</a></sup> 시즌 성적은 83경기에 출전해 타율 .260, 52득점을 기록했으며 무실책으로 <a href="/wiki/%EC%88%98%EB%B9%84%EC%9C%A8" title="수비율">수비율</a> 1.000을 달성했다.<sup class="reference" id="cite_ref-br_9-6"><a href="#cite_note-br-9">[9]</a></sup> 팀이 <a class="new" href="/w/index.php?title=1948%EB%85%84_%EC%9B%94%EB%93%9C_%EC%8B%9C%EB%A6%AC%EC%A6%88&amp;action=edit&amp;redlink=1" title="1948년 월드 시리즈 (없는 문서)">1948년 월드 시리즈</a>에 진출했고 터커는 6차전에 경기를 나서서 6회 <a href="/wiki/%EB%B3%BC%EB%84%B7" title="볼넷">볼넷</a>으로 출루해 득점했으며, 팀은 이날 4-3으로 승리했다.<sup class="reference" id="cite_ref-35"><a href="#cite_note-35">[35]</a></sup></p>
<p>1949년 시즌 터커는 팀의 제4의 외야수로 내정되어 주로 도비의 백업으로 경기에 나서게 되었으며,<sup class="reference" id="cite_ref-36"><a href="#cite_note-36">[36]</a></sup> 가끔 <a href="/wiki/%EB%8C%80%ED%83%80" title="대타">대타</a>로 나서는 것 정도로 입지가 적어지면서, 20경기에서 200타수 미만을 기록하며 .244의 타율로 시즌을 마쳤다.<sup class="reference" id="cite_ref-br_9-7"><a href="#cite_note-br-9">[9]</a></sup> 타격 부진을 면치 못하자 <a href="/wiki/%EB%A3%A8_%EB%B6%80%EB%93%9C%EB%A1%9C" title="루 부드로">루 부드로</a> 감독은 1950년 시즌을 시작하며 그를 <a class="mw-redirect" href="/wiki/%EC%8A%A4%EC%9C%84%EC%B9%98_%ED%9E%88%ED%84%B0" title="스위치 히터">스위치 히터</a>로 바꾸려고 시도해보기도 했지만,<sup class="reference" id="cite_ref-37"><a href="#cite_note-37">[37]</a></sup> 백업 외야수에 머물렀다. <a href="/wiki/%EB%A1%9C%EC%8A%A4%EC%95%A4%EC%A0%A4%EB%A0%88%EC%8A%A4_%EB%8B%A4%EC%A0%80%EC%8A%A4" title="로스앤젤레스 다저스">브루클린 다저스</a>와의 경기에서 팀이 8-5로 승리했는데, 이 때 시즌 중 유일한 홈런을 쳐냈다. 이 경기에서 클리블랜드는 연고지에 있는 동네 야구 팀들을 지원하기 위한 6만 달러의 기금을 모았다.<sup class="reference" id="cite_ref-38"><a href="#cite_note-38">[38]</a></sup> 시즌 54경기 타율 .178로 커리어 로우의 성적을 기록했다.<sup class="reference" id="cite_ref-br_9-8"><a href="#cite_note-br-9">[9]</a></sup></p>
<p>1951년 시즌이 시작되면서 <a class="new" href="/w/index.php?title=%ED%95%B4%EB%A6%AC_%EC%8B%AC%EC%8A%A8&amp;action=edit&amp;redlink=1" title="해리 심슨 (없는 문서)">해리 심슨</a>과 <a href="/wiki/%EB%AF%B8%EB%8B%88_%EB%AF%B8%EB%85%B8%EC%86%8C" title="미니 미노소">미니 미노소</a>가 새로운 경쟁자로 가세하면서, 터커와 앨리 클라크 중 한 명 혹은 둘 모두 트레이드되거나 방출될 가능성이 생겼고 두 선수는 선수 명단에 진입하려고 애를 썼다.<sup class="reference" id="cite_ref-39"><a href="#cite_note-39">[39]</a></sup> 4월 29일 한 타석에 나서 삼진을 당한 것이 터커가 이해 시즌 유일하게 모습을 드러낸 경기였으며,<sup class="reference" id="cite_ref-br_9-9"><a href="#cite_note-br-9">[9]</a></sup> 5월초에 <a href="/wiki/%ED%8A%B8%EB%A6%AC%ED%94%8C_A_(%EC%95%BC%EA%B5%AC)" title="트리플 A (야구)">트리플 A</a>의 <a href="/wiki/%ED%8D%BC%EC%8B%9C%ED%94%BD_%EC%BD%94%EC%8A%A4%ED%8A%B8_%EB%A6%AC%EA%B7%B8" title="퍼시픽 코스트 리그">퍼시픽 코스트 리그</a> 소속 <a class="new" href="/w/index.php?title=%EC%83%8C%EB%94%94%EC%97%90%EC%9D%B4%EA%B3%A0_%ED%8C%8C%EB%93%9C%EB%A6%AC%EC%8A%A4_(%ED%8D%BC%EC%8B%9C%ED%94%BD_%EC%BD%94%EC%8A%A4%ED%8A%B8_%EB%A6%AC%EA%B7%B8)&amp;action=edit&amp;redlink=1" title="샌디에이고 파드리스 (퍼시픽 코스트 리그) (없는 문서)">샌디에이고 파드리스</a>로 내려가면서 4월 29일이 그의 마지막 메이저 리그 경기가 되었다.<sup class="reference" id="cite_ref-br_9-10"><a href="#cite_note-br-9">[9]</a></sup><sup class="reference" id="cite_ref-40"><a href="#cite_note-40">[40]</a></sup></p>
<h2><span class="mw-headline" id=".EC.9D.B4.ED.9B.84.EC.9D.98_.EC.82.B6">이후의 삶</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%84%9C%EB%A8%BC_%ED%84%B0%EC%BB%A4&amp;action=edit&amp;section=5" title="부분 편집: 이후의 삶">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>샌디에이고 파드리스로 강등되어 1951년 시즌을 치렀으며, 88경기에 출전해 2개의 3루타와 함께 .222의 타율을 기록했다.<sup class="reference" id="cite_ref-brm_3-6"><a href="#cite_note-brm-3">[3]</a></sup> 오프 시즌 동안 텍사스 주에서 택시 운영을 하면서 시간을 보내다가 1952년 2월 즈음 계약에 관한 문제로 갈등하면서 야구를 그만둘지 심하게 고민했다.<sup class="reference" id="cite_ref-41"><a href="#cite_note-41">[41]</a></sup> 1952년 시즌에는 클리블랜드 인디언스에서 독립해서 나온 파드리스에서 47경기에 뛰며 타율 .225를 기록했다.<sup class="reference" id="cite_ref-brm_3-7"><a href="#cite_note-brm-3">[3]</a></sup> 6월 중순 파드리스는 터커의 계약을 오클라호마시티 인디언스로 넘겼고,<sup class="reference" id="cite_ref-42"><a href="#cite_note-42">[42]</a></sup> 오클라호마시티에서는 72경기를 뛰며 .263의 타율을 기록했다. 다음 시즌에는 어떤 프로팀에서도 뛰지 않으면서 은퇴를 하는 듯 보였다.<sup class="reference" id="cite_ref-brm_3-8"><a href="#cite_note-brm-3">[3]</a></sup></p>
<p>1954년 다시 야구계로 돌아와 <a class="new" href="/w/index.php?title=%EC%9B%A8%EC%8A%A4%ED%8A%B8_%ED%85%8D%EC%82%AC%EC%8A%A4-%EB%89%B4%EB%A9%95%EC%8B%9C%EC%BD%94_%EB%A6%AC%EA%B7%B8&amp;action=edit&amp;redlink=1" title="웨스트 텍사스-뉴멕시코 리그 (없는 문서)">웨스트 텍사스-뉴멕시코 리그</a>의 <a class="new" href="/w/index.php?title=%EB%9F%AC%EB%B2%85_%ED%97%88%EB%B2%84%EC%8A%A4&amp;action=edit&amp;redlink=1" title="러벅 허버스 (없는 문서)">러벅 허버스</a>에서 소속팀의 다른 선수들이 부상이 있거나, 가족들과 시간을 보내려고 할 때 빈자리를 메꾸는 방식의 시간제로 뛰면서,<sup class="reference" id="cite_ref-43"><a href="#cite_note-43">[43]</a></sup> 25경기 .360의 타율을 기록했다.<sup class="reference" id="cite_ref-brm_3-9"><a href="#cite_note-brm-3">[3]</a></sup> 다음해에는 <a class="new" href="/w/index.php?title=%EB%A1%B1%ED%98%BC_%EB%A6%AC%EA%B7%B8&amp;action=edit&amp;redlink=1" title="롱혼 리그 (없는 문서)">롱혼 리그</a>의 <a class="new" href="/w/index.php?title=%EC%B9%BC%EC%A6%88%EB%B0%B0%EB%93%9C_%ED%8F%AC%ED%85%8C%EC%85%94%EC%8A%A4&amp;action=edit&amp;redlink=1" title="칼즈배드 포테셔스 (없는 문서)">칼즈배드 포테셔스</a>에서 선수 겸 감독을 맡았으며 114경기에서 .275의 타율에 25개의 2루타, 그리고 8홈런을 기록했다.<sup class="reference" id="cite_ref-brm_3-10"><a href="#cite_note-brm-3">[3]</a></sup><sup class="reference" id="cite_ref-44"><a href="#cite_note-44">[44]</a></sup> 1956년에도 계속해서 포테셔스의 선수 겸 감독을 맡았으며, 128경기에서 .306의 타율을 기록했다. 팀이 14연패에 빠졌을 당시 경영진은 그의 해고를 검토하기도 했다.<sup class="reference" id="cite_ref-45"><a href="#cite_note-45">[45]</a></sup> 다음해 홉스 스포츠의 선수 겸 감독을 맡았으며, 16경기에 출전해 .273의 타율을 기록했다.<sup class="reference" id="cite_ref-brm_3-11"><a href="#cite_note-brm-3">[3]</a></sup> 1958년, 선수로의 경력을 마친 그는 홈스 팀의 단장으로 부임했다.<sup class="reference" id="cite_ref-46"><a href="#cite_note-46">[46]</a></sup></p>
<p>은퇴 후에는 <a href="/wiki/%EC%98%A4%ED%81%B4%EB%9D%BC%ED%98%B8%EB%A7%88%EC%8B%9C%ED%8B%B0" title="오클라호마시티">오클라호마시티</a>에서 살면서 보험설계사로 일했다. 터커는 결혼해 네 명의 아이들을 두었으며, 그의 아들 로널드는 <a href="/wiki/%EB%B2%A0%ED%8A%B8%EB%82%A8_%EC%A0%84%EC%9F%81" title="베트남 전쟁">베트남 전쟁</a>에 참전했다.<sup class="reference" id="cite_ref-digest_1-1"><a href="#cite_note-digest-1">[1]</a></sup> 1962년에는 새롭게 창단된 <a href="/wiki/%ED%9C%B4%EC%8A%A4%ED%84%B4_%EC%95%A0%EC%8A%A4%ED%8A%B8%EB%A1%9C%EC%8A%A4" title="휴스턴 애스트로스">휴스턴 애스트로스</a>의 <a href="/wiki/%EC%8A%A4%EC%B9%B4%EC%9A%B0%ED%8A%B8_(%EC%8A%A4%ED%8F%AC%EC%B8%A0)" title="스카우트 (스포츠)">스카우트</a>로 일했다.<sup class="reference" id="cite_ref-47"><a href="#cite_note-47">[47]</a></sup> 1993년 5월 7일에 오클라호마시티에서 사망했으며, 그의 시신은 고향인 텍사스 주 고든의 고든 공동묘지에 안장되었다.<sup class="reference" id="cite_ref-br_9-11"><a href="#cite_note-br-9">[9]</a></sup></p>
