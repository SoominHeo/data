<p><b>고분자</b>(高分子, Macromolecule)에 대한 명확한 정의는 존재하지 않으나 일반적으로 <a class="mw-redirect" href="/wiki/%EB%B6%84%EC%9E%90%EB%9F%89" title="분자량">분자량</a>이 1만 이상인 큰 <a href="/wiki/%EB%B6%84%EC%9E%90" title="분자">분자</a>를 말한다. 100개 이상의 <a href="/wiki/%EC%9B%90%EC%9E%90" title="원자">원자</a>로 구성되어 있다. 대개 <a href="/wiki/%EC%A4%91%ED%95%A9%EC%B2%B4" title="중합체">중합체</a>(Polymer)이다. 물질의 성질로서는 첫 번째로 분자량이 일정하지 않아 <a href="/wiki/%EB%85%B9%EB%8A%94%EC%A0%90" title="녹는점">녹는점</a>과 <a href="/wiki/%EB%81%93%EB%8A%94%EC%A0%90" title="끓는점">끓는점</a>이 일정하지 않고, 두 번째로 <a href="/wiki/%EC%95%A1%EC%B2%B4" title="액체">액체</a> 또는 <a href="/wiki/%EA%B3%A0%EC%B2%B4" title="고체">고체</a>로 존재한다. 세 번째로는 반응을 잘 하지 않아 안정적이다.</p>
<p>고분자의 종류로는 크게 합성 고분자와 천연 고분자로 나눌 수 있다. 천연 고분자의 예로는 <a href="/wiki/%ED%83%84%EC%88%98%ED%99%94%EB%AC%BC" title="탄수화물">탄수화물</a>에서 <a class="new" href="/w/index.php?title=%EC%95%8C%ED%8C%8C_%ED%8F%AC%EB%8F%84%EB%8B%B9&amp;action=edit&amp;redlink=1" title="알파 포도당 (없는 문서)">알파 포도당</a>이 <a class="new" href="/w/index.php?title=%EC%B6%95%ED%95%A9%EB%B0%98%EC%9D%91&amp;action=edit&amp;redlink=1" title="축합반응 (없는 문서)">축합반응</a>을 하여 생성된 <a href="/wiki/%EB%85%B9%EB%A7%90" title="녹말">녹말</a>, <a class="new" href="/w/index.php?title=%EB%B2%A0%ED%83%80_%ED%8F%AC%EB%8F%84%EB%8B%B9&amp;action=edit&amp;redlink=1" title="베타 포도당 (없는 문서)">베타 포도당</a>이 축합반응을 하여 생성된 <a href="/wiki/%EC%85%80%EB%A3%B0%EB%A1%9C%EC%8A%A4" title="셀룰로스">셀룰로스</a>가 있다. <a href="/wiki/%EB%8B%A8%EB%B0%B1%EC%A7%88" title="단백질">단백질</a>도 천연고분자이다.</p>
<h2><span class="mw-headline" id=".ED.8A.B9.EC.84.B1">특성</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EA%B3%A0%EB%B6%84%EC%9E%90&amp;action=edit&amp;section=1" title="부분 편집: 특성">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>고분자는 종종 특이한 물리적인 특성이 있다. 예를 들자면, 용액에서의 <a href="/wiki/DNA" title="DNA">DNA</a> 조각들은 빨대로 용액을 빠는 것만으로 두 개의 조각으로 나누어 질 수 있다.이것은 작은 분자에서의 일만이 아니다. <a class="new" href="/w/index.php?title=Linus_Pauling&amp;action=edit&amp;redlink=1" title="Linus Pauling (없는 문서)">Linus Pauling</a>의 college chemistry 1964 edition에는 자연 상태의 DNA의 염기쌍은 5000쌍이 되지 않는다고 주장한다. 이러한 실수는 생화학자들이 무심코 혹은 지속적으로 이러한 시료들을 조각조각 부수었기 때문에 발생하였다. 사실 <a href="/wiki/%EC%97%BC%EC%83%89%EC%B2%B4" title="염색체">염색체</a>의 DNA는 10억 쌍이상의 염기쌍을 가진다.</p>
<p>또한 작은 분자체는 가지지 않는 고분자의 흔한 특성은 물, 혹은 물과 비슷한 용매에서의 불용성이다. 많은 고분자는 물에 용해되려면 소금이나 특정 이온이 필요하다. 비슷하게 많은 단백질은 용액의 상전이가 너무 높거나 너무 낮을때 변형된다.</p>
<p>용액에서의 고분자 농도가 너무 높으면 다른 고분자의 반응에서의 평형상수와 반응 속도를 바꿀 수 있다. 이 현상은 <a class="new" href="/w/index.php?title=%EA%B1%B0%EB%8C%80_%EB%B6%84%EC%9E%90_%EA%B5%B0%EC%A7%91&amp;action=edit&amp;redlink=1" title="거대 분자 군집 (없는 문서)">거대 분자 군집</a>이라 일컬어 진다. 이러한 현상은 고분자가 다른 분자를 용액에서 큰 부피를 차지하지 못하도록 배제시킴으로서 발생되는데, 이것은 이러한 분자들의 <a class="new" href="/w/index.php?title=%EC%9C%A0%ED%9A%A8_%EB%86%8D%EB%8F%84&amp;action=edit&amp;redlink=1" title="유효 농도 (없는 문서)">유효 농도</a>를 증가시킨다.<sup class="reference" id="cite_ref-Minton_1-0"><a href="#cite_note-Minton-1">[1]</a></sup></p>
<h2><span class="mw-headline" id=".EA.B2.B0.ED.95.A9">결합</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EA%B3%A0%EB%B6%84%EC%9E%90&amp;action=edit&amp;section=2" title="부분 편집: 결합">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<ul>
<li>선형고분자(Linear Polymer) : 긴 사슬로 이루어져 있다. 보통 용매에 녹으며, 일반적인 온도하의 고체상태에서 고무상이나 유연한 물질 또는 유리상의 열가소성플라스틱으로 존재한다.</li>
<li>가지고분자(Branched Polymer) : 선형고분자에 가지가 붙어 있는 구조로서, 가지는 주 사슬과 동일한 기본구조를 갖고 있다. 선형고분자의 용매와 같은 용매에 녹으며, 가지가 많은 고분자의 경우 특정한 액체에 팽윤되기도 한다.</li>
<li>가교고분자(망상고분자)(Crosslinked / Network Polymer) : 사슬 사이에 1차결합이 존재한다. 따라서 용매에 의해 용해되지 않고 팽윤된다. 액체에 의해 팽윤되는 양은 가교밀도에 따라 다르다. 가교도가 높으면 다이아몬드 같은 단단하고 팽윤이 잘 되지 않는 물질이 된다.</li>
</ul>
