<p><b>고비 사막</b>(<small><a href="/wiki/%EB%AA%BD%EA%B3%A8%EC%96%B4" title="몽골어">몽골어</a>:</small> Говь, <small><a href="/wiki/%EC%A4%91%EA%B5%AD%EC%96%B4" title="중국어">중국어</a></small><small>:</small> <span lang="zh" xml:lang="zh">戈壁</span>, <small><a href="/wiki/%ED%95%9C%EC%96%B4_%EB%B3%91%EC%9D%8C" title="한어 병음">병음</a>:</small> <font face="Lucida Sans Unicode">gē bì</font>)은 <a href="/wiki/%EB%AA%BD%EA%B3%A8" title="몽골">몽골</a>과 <a href="/wiki/%EC%A4%91%ED%99%94%EC%9D%B8%EB%AF%BC%EA%B3%B5%ED%99%94%EA%B5%AD" title="중화인민공화국">중국</a>에 걸쳐 있는 <a href="/wiki/%EC%A4%91%EC%95%99%EC%95%84%EC%8B%9C%EC%95%84" title="중앙아시아">중앙아시아</a>의 <a href="/wiki/%EC%82%AC%EB%A7%89" title="사막">사막</a>이며 아시아에서 가장 큰 사막이다. "고비"는 몽골어로 "거친 땅"이라는 뜻이다. 북쪽은 <a href="/wiki/%EC%95%8C%ED%83%80%EC%9D%B4_%EC%82%B0%EB%A7%A5" title="알타이 산맥">알타이 산맥</a>과 <a class="mw-redirect" href="/wiki/%EC%8A%A4%ED%85%9D_(%ED%8F%89%EC%95%BC)" title="스텝 (평야)">스텝</a> 지대, 남쪽은 <a href="/wiki/%ED%8B%B0%EB%B2%A0%ED%8A%B8_%EA%B3%A0%EC%9B%90" title="티베트 고원">티베트 고원</a>, 동쪽은 <a href="/wiki/%ED%99%94%EB%B2%A0%EC%9D%B4_%ED%8F%89%EC%9B%90" title="화베이 평원">화베이 평원</a>으로 둘러싸여 있다. 고비 사막의 모래가 날리는 <a href="/wiki/%ED%99%A9%EC%82%AC" title="황사">황사</a>는 <a href="/wiki/%ED%8E%B8%EC%84%9C%ED%92%8D" title="편서풍">편서풍</a>을 타고 <a href="/wiki/%ED%95%9C%EB%B0%98%EB%8F%84" title="한반도">한반도</a>와 <a href="/wiki/%EC%9D%BC%EB%B3%B8" title="일본">일본</a>을 건너 <a href="/wiki/%EB%AF%B8%EA%B5%AD" title="미국">미국</a> <a href="/wiki/%ED%95%98%EC%99%80%EC%9D%B4_%EC%A3%BC" title="하와이 주">하와이 주</a>까지 가기도 한다.</p>
<h2><span class="mw-headline" id=".EC.A7.80.EB.A6.AC">지리</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EA%B3%A0%EB%B9%84_%EC%82%AC%EB%A7%89&amp;action=edit&amp;section=1" title="부분 편집: 지리">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>고비 사막은 남서쪽에서 북동쪽까지 1,610 km이고 북쪽에서 남쪽까지 800 km로 측정된다.</p>
