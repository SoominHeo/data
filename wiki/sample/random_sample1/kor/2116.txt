<p><b>고세원</b>(<a href="/wiki/1977%EB%85%84" title="1977년">1977년</a> <a href="/wiki/12%EC%9B%94_13%EC%9D%BC" title="12월 13일">12월 13일</a> ~ )은 대한민국의 <a href="/wiki/%EC%98%81%ED%99%94%EB%B0%B0%EC%9A%B0" title="영화배우">영화배우</a>이자 <a href="/wiki/%EB%B0%B0%EC%9A%B0#.ED.85.94.EB.A0.88.EB.B9.84.EC.A0.84_.EB.B0.B0.EC.9A.B0" title="배우">탤런트</a>이다.</p>
<p>KBS 공채탤런트 19기로 <a href="/wiki/%EB%8B%A8%EA%B5%AD%EB%8C%80%ED%95%99%EA%B5%90" title="단국대학교">단국대학교</a> 연극영화과를 입학하여 고세원은 14년 만인 <a href="/wiki/2010%EB%85%84" title="2010년">2010년</a> <a href="/wiki/2%EC%9B%94_18%EC%9D%BC" title="2월 18일">2월 18일</a> 가수 <a href="/wiki/%EB%B0%94%EB%8B%A4_(%EA%B0%80%EC%88%98)" title="바다 (가수)">바다</a>(최성희)와 함께 33살의 늦깎이로 학사학위를 받았다. 함께 학사학위를 받은 <a href="/wiki/%EB%B0%94%EB%8B%A4_(%EA%B0%80%EC%88%98)" title="바다 (가수)">바다</a>(최성희)에게는 12년 만의 학위수여식이었다.</p>
<p>고세원은 <a href="/wiki/%EB%AE%A4%EC%A7%80%EC%BB%AC" title="뮤지컬">뮤지컬</a> ‘오! 당신이 잠든 사이’에서 주인공으로 캐스팅되었다. <a href="/wiki/%EB%AE%A4%EC%A7%80%EC%BB%AC" title="뮤지컬">뮤지컬</a>에서 무뚝뚝하지만 속정 깊은 의사 ‘닥터리’와 멀티맨으로 출연하는 고세원은 그동안 뮤지컬 ‘벽을 뚫는 남자’ ‘아이러브 유’ ‘김종욱 찾기’ 등에서 실력을 인정받은 배우이다. 최근엔 <a class="mw-redirect" href="/wiki/KBS2" title="KBS2">KBS2</a> 수목 드라마 ‘신데렐라 언니’에서 <a href="/wiki/%EC%B2%9C%EC%A0%95%EB%AA%85" title="천정명">천정명</a>의 형이자 그와 대립하는 역할을 연기하였고 현재는 <a class="mw-redirect" href="/wiki/KBS2" title="KBS2">KBS2</a> <a href="/wiki/%EC%88%98%EC%83%81%ED%95%9C_%EC%82%BC%ED%98%95%EC%A0%9C" title="수상한 삼형제">수상한 삼형제</a>에 출연 하였다.<sup class="reference" id="cite_ref-1"><a href="#cite_note-1">[1]</a></sup></p>

<h2><span class="mw-headline" id=".EC.97.90.ED.94.BC.EC.86.8C.EB.93.9C">에피소드</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EA%B3%A0%EC%84%B8%EC%9B%90&amp;action=edit&amp;section=1" title="부분 편집: 에피소드">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>고세원은 박찬호 군대 조교를 했었다고 박찬호와의 인연을 들려줬다. <a href="/wiki/1999%EB%85%84" title="1999년">1999년</a>부터 <a href="/wiki/2000%EB%85%84" title="2000년">2000년</a> <a href="/wiki/8%EC%9B%94" title="8월">8월</a>까지 충남 공주 육군 32사단 신병교육대에서 조교로 복무했고 그때 <a href="/wiki/%EB%B0%95%EC%B0%AC%ED%98%B8" title="박찬호">박찬호</a>는 <a href="/wiki/1998%EB%85%84" title="1998년">1998년</a> 방콕아시안게임에서 우승을 해 병역면제 혜택을 받은 후 <a href="/wiki/1999%EB%85%84" title="1999년">1999년</a> <a href="/wiki/10%EC%9B%94" title="10월">10월</a> 이곳에 입소해 4주간 군사 훈련을 받았다는 것이다. 어느 날은 라면 끓일 뜨거운 물로 라면을 포기하고 박찬호의 허리 찜질을 해 줬다는 얘기도 들려줬다.<sup class="reference" id="cite_ref-2"><a href="#cite_note-2">[2]</a></sup></p>
