<p><b>고양 시민축구단</b>(<span lang="en">Goyang Citizen Football Club</span>)은 <a href="/wiki/%EB%8C%80%ED%95%9C%EB%AF%BC%EA%B5%AD" title="대한민국">대한민국</a>의 <a class="mw-redirect" href="/wiki/%EC%B1%8C%EB%A6%B0%EC%A0%80%EC%8A%A4%EB%A6%AC%EA%B7%B8" title="챌린저스리그">챌린저스리그</a>에 속해있는 <a href="/wiki/%EC%B6%95%EA%B5%AC" title="축구">축구</a>단이다. <a href="/wiki/%EA%B2%BD%EA%B8%B0%EB%8F%84" title="경기도">경기도</a> <a href="/wiki/%EA%B3%A0%EC%96%91%EC%8B%9C" title="고양시">고양시</a>를 연고지로 하고 있으며 <a href="/wiki/%EA%B3%A0%EC%96%91%EC%96%B4%EC%9A%B8%EB%A6%BC%EB%88%84%EB%A6%AC" title="고양어울림누리">고양 어울림누리별무리구장</a>을 홈구장으로 사용하고 있다. <a href="/wiki/2008%EB%85%84" title="2008년">2008년</a> <a href="/wiki/3%EC%9B%94_18%EC%9D%BC" title="3월 18일">3월 18일</a> 창단되어 <a href="/wiki/2008%EB%85%84" title="2008년">2008년</a> <a href="/wiki/K3%EB%A6%AC%EA%B7%B8" title="K3리그">K3리그</a>에 참가하였다. 서포터즈인 보레아스는 창단 준비과정부터 <a href="/wiki/2010%EB%85%84_1%EC%9B%94" title="2010년 1월">2010년 1월</a>까지 함께하였다.</p>
