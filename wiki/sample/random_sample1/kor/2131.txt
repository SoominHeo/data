<p>《<b>고양이신 팔백만</b>》(<span style="font-size: smaller;"><a href="/wiki/%EC%9D%BC%EB%B3%B8%EC%96%B4" title="일본어">일본어</a>:</span> <span lang="ja"><span lang="ja"><ruby><rb>猫神やおよろず</rb> <rp>(</rp><rt>ねこがみやおよろず</rt><rp>)</rp></ruby></span></span> <small>네코가미 야오요로즈<sup>[<a href="/wiki/%EC%9C%84%ED%82%A4%EB%B0%B1%EA%B3%BC:%EC%9D%BC%EB%B3%B8%EC%96%B4%EC%9D%98_%ED%95%9C%EA%B8%80_%ED%91%9C%EA%B8%B0" title="위키백과:일본어의 한글 표기">*</a>]</sup></small>)은 <a class="new" href="/w/index.php?title=FLIPFLOPs&amp;action=edit&amp;redlink=1" title="FLIPFLOPs (없는 문서)">FLIPFLOPs</a> 작품의 <a href="/wiki/%EC%9D%BC%EB%B3%B8" title="일본">일본</a>의 <a href="/wiki/%EB%A7%8C%ED%99%94" title="만화">만화</a>이다. <a href="/wiki/%EC%95%84%ED%82%A4%ED%83%80_%EC%87%BC%ED%85%90" title="아키타 쇼텐">아키타 쇼텐</a>의 《<a class="new" href="/w/index.php?title=%EC%B1%94%ED%94%BC%EC%96%B8_RED_%EC%9D%B4%EC%B9%98%EA%B3%A0&amp;action=edit&amp;redlink=1" title="챔피언 RED 이치고 (없는 문서)">챔피언 RED 이치고</a>》를 통해 <a href="/wiki/2007%EB%85%84_2%EC%9B%94" title="2007년 2월">2007년 2월</a>호 부터 연재되고 있다.</p>
<p><a href="/wiki/2009%EB%85%84" title="2009년">2009년</a>, <a class="new" href="/w/index.php?title=%EB%B9%84%ED%8A%B8%EB%8B%89%EC%8A%A4&amp;action=edit&amp;redlink=1" title="비트닉스 (없는 문서)">EDGE RECORDS</a>를 통해 <a class="mw-redirect" href="/wiki/%EB%93%9C%EB%9D%BC%EB%A7%88CD" title="드라마CD">드라마CD</a>화 되었다. <a href="/wiki/2010%EB%85%84" title="2010년">2010년</a> <a href="/wiki/12%EC%9B%94_1%EC%9D%BC" title="12월 1일">12월 1일</a>에 텔레비전 <a href="/wiki/%EC%95%A0%EB%8B%88%EB%A9%94%EC%9D%B4%EC%85%98" title="애니메이션">애니메이션</a>화가 결정되어 <a href="/wiki/2011%EB%85%84" title="2011년">2011년</a> <a href="/wiki/7%EC%9B%94" title="7월">7월</a>부터 <a href="/wiki/9%EC%9B%94" title="9월">9월</a>까지 <a href="/wiki/AT-X" title="AT-X">AT-X</a> 등의 방송사를 통해 방영되었다.</p>
