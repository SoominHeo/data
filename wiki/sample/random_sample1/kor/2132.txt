<p><b>고엔유 천황</b>(<span style="font-size: smaller;"><a href="/wiki/%EC%9D%BC%EB%B3%B8%EC%96%B4" title="일본어">일본어</a>:</span> <span lang="ja" xml:lang="ja">後円融天皇</span>, 후원융 천황, <a href="/wiki/1359%EB%85%84" title="1359년">1359년</a> <a href="/wiki/1%EC%9B%94_11%EC%9D%BC" title="1월 11일">1월 11일</a> ~ <a href="/wiki/1393%EB%85%84" title="1393년">1393년</a> <a href="/wiki/6%EC%9B%94_6%EC%9D%BC" title="6월 6일">6월 6일</a>)은 일본의 <a href="/wiki/%EB%82%9C%EB%B3%B4%EC%BF%A0%EC%B4%88_%EC%8B%9C%EB%8C%80" title="난보쿠초 시대">남북조 시대</a> <a href="/wiki/%EB%B6%81%EC%A1%B0_(%EC%9D%BC%EB%B3%B8)" title="북조 (일본)">북조</a>의 제5대 천황이었다(재위: <a href="/wiki/1371%EB%85%84" title="1371년">1371년</a> <a href="/wiki/4%EC%9B%94_9%EC%9D%BC" title="4월 9일">4월 9일</a> ~ <a href="/wiki/1382%EB%85%84" title="1382년">1382년</a> <a href="/wiki/5%EC%9B%94_24%EC%9D%BC" title="5월 24일">5월 24일</a>). 메이지 이전의 학자들에 따르면 그의 치세는 1371년에서 1382년까지 였다. 그의 이름은 64대 천황 <a href="/wiki/%EC%97%94%EC%9C%A0_%EC%B2%9C%ED%99%A9" title="엔유 천황">엔유 천황</a>을 본딴 것으로 비한자문화권에서는 엔유 2세라고도 불린다. 휘(諱)는 오히토(緒仁)다. 그는 북조 제4대 천황 <a href="/wiki/%EA%B3%A0%EC%BD%94%EA%B3%A4_%EC%B2%9C%ED%99%A9" title="고코곤 천황">고코곤 천황</a>의 차남으로 어머니는 <a class="mw-redirect" href="/wiki/%EC%82%AC%EB%8B%A4%EC%9D%B4%EC%A7%84" title="사다이진">사다이진</a> 히로하시 가네쓰나(広橋兼綱)의 양녀 후지와라노 나카코(藤原仲子)였다.</p>
<h2><span class="mw-headline" id=".EC.83.9D.EC.95.A0">생애</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EA%B3%A0%EC%97%94%EC%9C%A0_%EC%B2%9C%ED%99%A9&amp;action=edit&amp;section=1" title="부분 편집: 생애">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>1371년 황실의 포고에 의해 그는 친왕의 직위를 받고 황태자로 봉해졌다. 직후에 그는 아버지 <a href="/wiki/%EA%B3%A0%EC%BD%94%EA%B3%A4_%EC%B2%9C%ED%99%A9" title="고코곤 천황">고코곤 천황</a>의 양위로 천황이 되었다. <a href="/wiki/%EA%B3%A0%EC%BD%94%EA%B3%A4_%EC%B2%9C%ED%99%A9" title="고코곤 천황">고코곤 천황</a>과 <a href="/wiki/%EC%8A%A4%EC%BD%94_%EC%B2%9C%ED%99%A9" title="스코 천황">스코 천황</a> 사이에 황태자에 대한 의견 상충이 있었다고 전해진다. 막부를 조정하였던 호소가와 요리유키의 지지로 고엔유 천황은 북조 천황이 되었다. 그러나 1374년까지는 <a href="/wiki/%EA%B3%A0%EC%BD%94%EA%B3%A4_%EC%B2%9C%ED%99%A9" title="고코곤 천황">고코곤 천황</a>이 상황으로서 정무를 보았다.</p>
<p>1368년 <a href="/wiki/%EC%95%84%EC%8B%9C%EC%B9%B4%EA%B0%80_%EC%9A%94%EC%8B%9C%EB%AF%B8%EC%93%B0" title="아시카가 요시미쓰">아시카가 요시미쓰</a>는 쇼군에 취임했고 그의 경호로 황궁은 안정이 되었다.</p>
<p>1382년 고엔유 천왕은 아들 <a href="/wiki/%EA%B3%A0%EC%BD%94%EB%A7%88%EC%93%B0_%EC%B2%9C%ED%99%A9" title="고코마쓰 천황">고코마쓰 천황</a>에게 양위하고 상황이 되어 섭정을 시작했으나 요시미쓰의 개입으로 실권은 없었다. 이후 그는 후궁 산죠 이쓰코와 요시미쓰의 간통을 의심하였고 또다른 후궁도 요시미쓰와의 사이를 의심해 출가시켰다. 이후에도 요시미쓰가 자신을 유배시키려 한다고 의심해 할복 자살 소동을 벌이기도 했다.</p>
<p>1392년 남조와 화친이 맺어지고 남북조 시대는 끝났다.</p>
<p>1393년 6월 6일 고엔유 천황이 사망하였다.</p>
