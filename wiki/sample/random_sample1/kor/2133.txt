<p><b>고와쇼즈 역</b>(<span style="font-size: smaller;"><a href="/wiki/%EC%9D%BC%EB%B3%B8%EC%96%B4" title="일본어">일본어</a>:</span> <span lang="ja"><span lang="ja"><ruby><rb>小和清水駅</rb> <rp>(</rp><rt>こわしょうずえき</rt><rp>)</rp></ruby></span></span>)은 <a href="/wiki/%EC%9D%BC%EB%B3%B8" title="일본">일본</a> <a href="/wiki/%ED%9B%84%EC%BF%A0%EC%9D%B4_%ED%98%84" title="후쿠이 현">후쿠이 현</a> <a href="/wiki/%ED%9B%84%EC%BF%A0%EC%9D%B4_%EC%8B%9C" title="후쿠이 시">후쿠이 시</a>에 위치한 <a class="mw-redirect" href="/wiki/%EC%84%9C%EC%9D%BC%EB%B3%B8_%EC%97%AC%EA%B0%9D%EC%B2%A0%EB%8F%84" title="서일본 여객철도">서일본 여객철도</a> <a href="/wiki/%EC%97%90%EC%93%B0%EB%AF%B8%ED%98%B8%EC%BF%A0_%EC%84%A0" title="에쓰미호쿠 선">에쓰미호쿠 선</a>의 <a href="/wiki/%EC%B2%A0%EB%8F%84%EC%97%AD" title="철도역">철도역</a>이다. <a class="mw-redirect" href="/wiki/%EB%8B%A8%EC%84%A0_%EC%8A%B9%EA%B0%95%EC%9E%A5" title="단선 승강장">단선 승강장</a> 1면 1선의 구조를 갖춘 지상역이다.</p>

<h2><span class="mw-headline" id=".EC.97.AD.EC.82.AC">역사</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EA%B3%A0%EC%99%80%EC%87%BC%EC%A6%88_%EC%97%AD&amp;action=edit&amp;section=2" title="부분 편집: 역사">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<ul>
<li><a href="/wiki/1960%EB%85%84" title="1960년">1960년</a> <a href="/wiki/12%EC%9B%94_15%EC%9D%BC" title="12월 15일">12월 15일</a> : 에쓰미호쿠 선이 <a href="/wiki/%EA%B0%80%EB%8F%84%ED%95%98%EB%9D%BC_%EC%97%AD" title="가도하라 역">가도하라 역</a>까지 개업했던 때에 설치.</li>
<li><a href="/wiki/1987%EB%85%84" title="1987년">1987년</a> <a href="/wiki/4%EC%9B%94_1%EC%9D%BC" title="4월 1일">4월 1일</a> : 일본국유철도 분할 민영화에 의해, <a class="mw-redirect" href="/wiki/%EC%84%9C%EC%9D%BC%EB%B3%B8_%EC%97%AC%EA%B0%9D%EC%B2%A0%EB%8F%84" title="서일본 여객철도">서일본 여객철도</a>가 승계.</li>
<li><a href="/wiki/2004%EB%85%84" title="2004년">2004년</a> <a href="/wiki/7%EC%9B%94_18%EC%9D%BC" title="7월 18일">7월 18일</a> : 호우에 의해 에쓰미호쿠 선 전 노선 운행 중지.</li>
<li><a href="/wiki/2007%EB%85%84" title="2007년">2007년</a> <a href="/wiki/6%EC%9B%94_30%EC%9D%BC" title="6월 30일">6월 30일</a> : 마지막까지 운행이 중지되었던, 이 역을 포함한 이치조다니 역 - 미야마 역 간이 운행 재개. 그 사이, 대행 버스 승강장은 국도 제158호선 나라세 교차로 부근으로 이전.</li>
</ul>
