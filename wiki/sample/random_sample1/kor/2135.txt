<p><b>고우다 천황</b>(<span style="font-size: smaller;"><a href="/wiki/%EC%9D%BC%EB%B3%B8%EC%96%B4" title="일본어">일본어</a>:</span> <span lang="ja" xml:lang="ja">後宇多天皇</span>, 후우다 천황, <a href="/wiki/1267%EB%85%84" title="1267년">1267년</a> <a href="/wiki/12%EC%9B%94_17%EC%9D%BC" title="12월 17일">12월 17일</a> ~ <a href="/wiki/1324%EB%85%84" title="1324년">1324년</a> <a href="/wiki/7%EC%9B%94_16%EC%9D%BC" title="7월 16일">7월 16일</a>)은 제91대 일본 천황이다. <a href="/wiki/1274%EB%85%84" title="1274년">1274년</a>부터 <a href="/wiki/1287%EB%85%84" title="1287년">1287년</a>까지 재휘했다. 휘(諱)는 요히토(世仁)이다. <a href="/wiki/%EA%B0%80%EB%A9%94%EC%95%BC%EB%A7%88_%EC%B2%9C%ED%99%A9" title="가메야마 천황">가메야마 천황</a>의 차남이자 다이카쿠지 황통 출신이었다.</p>
<h2><span class="mw-headline" id=".EC.83.9D.EC.95.A0">생애</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EA%B3%A0%EC%9A%B0%EB%8B%A4_%EC%B2%9C%ED%99%A9&amp;action=edit&amp;section=1" title="부분 편집: 생애">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>요시토 친왕은 <a href="/wiki/%EA%B3%A0%EC%82%AC%EA%B0%80_%EC%B2%9C%ED%99%A9" title="고사가 천황">고사가 천황</a>의 의향에 따라 1268년 황태자가 되었다. 고사가 천황은 1272년 사망하였고 2년 뒤인 1274년 그의 유언에 따라 요시토 친왕은 아버지 가메야마 천황의 양위를 받아 즉위했다. 물러난 가메야마 천황은 상황으로서 <a class="mw-redirect" href="/wiki/%EC%9D%B8%EC%84%B8%EC%9D%B4" title="인세이">인세이</a>를 펼쳤다. 1287년 자신의 아들(지묘인 황통)이 아닌 동생 가메야마 천황의 혈통(다이카쿠지 황통)이 천황이 된 것에 불만을 품고 있었던 <a href="/wiki/%EA%B3%A0%ED%9B%84%EC%B9%B4%EC%BF%A0%EC%82%AC_%EC%B2%9C%ED%99%A9" title="고후카쿠사 천황">고후카쿠사 상황</a>은 막부와 교섭을 했고, 고우다 천황은 고후카쿠사 상황의 아들인 <a href="/wiki/%ED%9B%84%EC%8B%9C%EB%AF%B8_%EC%B2%9C%ED%99%A9" title="후시미 천황">히로히토 친왕</a>에게 양위했다. 이후에도 황위 계승을 둘러싼 지묘인 황통과 다이카쿠지 황통의 갈등은 계속되었다. 고우다 천황은 아들 <a href="/wiki/%EA%B3%A0%EB%8B%88%EC%A1%B0_%EC%B2%9C%ED%99%A9" title="고니조 천황">고니조 천황</a>의 재위 기간 중 상황으로서 인세이를 행했으며 차남인 <a href="/wiki/%EA%B3%A0%EB%8B%A4%EC%9D%B4%EA%B3%A0_%EC%B2%9C%ED%99%A9" title="고다이고 천황">고다이고 천황</a>의 치세 중에도 그가 친정을 행하기까지 인세이를 했다. 고우다 천황은 <a href="/wiki/%EA%B2%90%EC%BD%94_(1321%EB%85%84)" title="겐코 (1321년)">겐코</a> 4년 6월(1324) 58세로 사망했다.</p>
