<h1 class="firstHeading" id="firstHeading" lang="ko">고운 (북연)</h1>

<div class="mw-content-ltr" dir="ltr" id="mw-content-text" lang="ko"><script>function mfTempOpenSection(id){var block=document.getElementById("mf-section-"+id);block.className+=" open-block";block.previousSibling.className+=" open-block";}</script><p><b>고운</b>(高雲, ? ~ <a href="/wiki/409%EB%85%84" title="409년">409년</a>)은 <a href="/wiki/%EB%B6%81%EC%97%B0" title="북연">북연</a>(北燕)의 초대 황제이다. 시호는 혜의제(惠懿帝)이다. <a href="/wiki/%EA%B3%A0%EA%B5%AD%EC%9B%90%EC%99%95" title="고국원왕">고국원왕</a> 때 <a href="/wiki/%EA%B3%A0%EA%B5%AC%EB%A0%A4" title="고구려">고구려</a>가 <a href="/wiki/%ED%9B%84%EC%97%B0" title="후연">후연</a>(後燕)에 패배하여 붙잡혀 온 <a href="/wiki/%EA%B3%A0%EA%B5%AC%EB%A0%A4" title="고구려">고구려</a>의 지파 <a class="new" href="/w/index.php?title=%EA%B3%A0%ED%99%94&amp;action=edit&amp;redlink=1" title="고화 (없는 문서)">고화</a>(高和)의 손자이고, <a class="new" href="/w/index.php?title=%EA%B3%A0%EB%B0%9C_%EB%B6%81%EC%97%B0&amp;action=edit&amp;redlink=1" title="고발 북연 (없는 문서)">고발</a>(高拔)의 아들이며, 자(字)는 자우(子雨)이다. <a href="/wiki/%EB%AA%A8%EC%9A%A9%EB%B3%B4" title="모용보">모용보</a>의 <a class="mw-disambig" href="/wiki/%EC%96%91%EC%9E%90" title="양자">양자</a>로서 모용운(慕容雲)이란 이름을 썼었기 때문에 모용운이라고도 불린다.</p>

<h2><span class="mw-headline" id=".EC.83.9D.EC.95.A0">생애</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EA%B3%A0%EC%9A%B4_(%EB%B6%81%EC%97%B0)&amp;action=edit&amp;section=1" title="부분 편집: 생애">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>후연 2대 황제 혜민제(惠閔帝) <a href="/wiki/%EB%AA%A8%EC%9A%A9%EB%B3%B4" title="모용보">모용보</a>(慕容寶, 재위 : <a href="/wiki/396%EB%85%84" title="396년">396년</a> ~ <a href="/wiki/397%EB%85%84" title="397년">397년</a>)가 <a class="mw-redirect" href="/wiki/%ED%83%9C%EC%9E%90" title="태자">태자</a>로 있었을 때, 고운은 무관으로서 <a href="/wiki/%EB%AA%A8%EC%9A%A9%EB%B3%B4" title="모용보">모용보</a>를 섬겼다. <a href="/wiki/%EB%AA%A8%EC%9A%A9%EB%B3%B4" title="모용보">모용보</a>가 등극한 후 시어랑(侍御郞) 벼슬을 받았으나 병으로 인해 얼마 가지 않아 물러났다.</p>
<p>그는 <a href="/wiki/397%EB%85%84" title="397년">397년</a>, <a href="/wiki/%EB%AA%A8%EC%9A%A9%EB%B3%B4" title="모용보">모용보</a>의 서자 <a class="new" href="/w/index.php?title=%EB%AA%A8%EC%9A%A9%ED%9A%8C&amp;action=edit&amp;redlink=1" title="모용회 (없는 문서)">모용회</a>(慕容會)가 반란을 일으켜 궁궐을 공격하자 <a href="/wiki/%EA%B5%B0%EC%82%AC" title="군사">군사</a>를 거느리고 싸워 반란군을 진압했다. 이 공으로 인해 <a href="/wiki/%EB%AA%A8%EC%9A%A9%EB%B3%B4" title="모용보">모용보</a>의 양자가 되고 석양공(夕陽公)이 되는 한편 <a href="/wiki/%EB%AA%A8%EC%9A%A9%EC%94%A8" title="모용씨">모용씨</a>를 하사받았는데, 이후 모용운(慕容雲)이라는 이름을 썼다. 그는 생각이 깊고 차분한 성격이었다고 전해지는데, 훗날 <a href="/wiki/%EB%B6%81%EC%97%B0" title="북연">북연</a>의 두 번째 황제가 되는 <a href="/wiki/%ED%92%8D%EB%B0%9C" title="풍발">풍발</a>(馮跋)과 두터운 친교를 쌓게 된다.</p>
<p>후연 4대 황제 소문제(昭文帝) <a href="/wiki/%EB%AA%A8%EC%9A%A9%ED%9D%AC" title="모용희">모용희</a>(慕容熙, 재위 : <a href="/wiki/401%EB%85%84" title="401년">401년</a> ~ <a href="/wiki/407%EB%85%84" title="407년">407년</a>)는 성격이 광폭한 폭군이었다. <a href="/wiki/407%EB%85%84" title="407년">407년</a> 황후인 <a class="new" href="/w/index.php?title=%EB%B6%80%ED%99%A9%ED%9B%84_(%ED%9B%84%EC%97%B0)&amp;action=edit&amp;redlink=1" title="부황후 (후연) (없는 문서)">부씨</a>(苻氏)가 죽자 신하들에게 큰 소리로 곡하게 한 다음 크게 울지 않는 자들을 처벌한 것이 발단이 되었는데, <a href="/wiki/%ED%92%8D%EB%B0%9C" title="풍발">풍발</a>은 고운을 추대할 거사계획을 알리고 <a href="/wiki/%ED%99%A9%EC%A0%9C" title="황제">황제</a>가 되도록 부추겼다.</p>
<p>고운은 썩 내켜하지 않았으나 <a href="/wiki/%ED%92%8D%EB%B0%9C" title="풍발">풍발</a> 등이 뜻을 굽히지 않자 어쩔 수 없이 동의하였다. <a href="/wiki/%EB%AA%A8%EC%9A%A9%ED%9D%AC" title="모용희">모용희</a>가 황후의 장례식 때 상여를 몸소 메고 성 밖으로 나간 것을 틈타 성문을 닫고 <a href="/wiki/%EB%AA%A8%EC%9A%A9%ED%9D%AC" title="모용희">모용희</a>에 대해 반감을 가졌던 백성들의 동조를 얻은 후 <a href="/wiki/%EB%AA%A8%EC%9A%A9%ED%9D%AC" title="모용희">모용희</a>가 수도인 <a class="mw-redirect" href="/wiki/%EC%9A%A9%EC%84%B1" title="용성">용성</a>(龍城)에 돌아오는 것을 거부했다. <a href="/wiki/%EB%AA%A8%EC%9A%A9%ED%9D%AC" title="모용희">모용희</a>는 <a href="/wiki/%EA%B5%B0%EC%82%AC" title="군사">군사</a>를 거느리고 <a class="mw-redirect" href="/wiki/%EC%9A%A9%EC%84%B1" title="용성">용성</a>을 공격했으나 도리어 붙잡혀 죽임을 당했다.</p>
<p>진서 <a href="/wiki/%EB%AA%A8%EC%9A%A9%ED%9D%AC" title="모용희">모용희</a> 재기에 따르면 이에 앞서 "볏짚 하나 묶어, 두 머리를 태우네. 대머리의 어린 아들이 와서 연을 멸하리"라는 참요가 떠돌았는데, 볏짚(藳)에서 위의 초두머리(艹)와 아래의 벼화(禾)변을 지우면 고(高)자가 되며, 운의 아버지 발의 어릴 적 이름이 바로 대머리라는 뜻의 독두(禿頭)였다고 한다.</p>
<h2><span class="mw-headline" id=".EC.B9.98.EC.84.B8">치세</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EA%B3%A0%EC%9A%B4_(%EB%B6%81%EC%97%B0)&amp;action=edit&amp;section=2" title="부분 편집: 치세">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>고운은 음력 4월에 황제로 즉위하였고 자신의 성을 본래의 성인 고(高)씨로 되돌렸다. 스스로를 천왕(天王)으로 칭하고 연호를 정시(正始)라 하고 죄수들에 대한 사면령을 내리는 한편 거사를 주도한 <a href="/wiki/%ED%92%8D%EB%B0%9C" title="풍발">풍발</a>을 시중(侍中)으로 삼았다.</p>
<p>나라 이름은 그대로 대연(大燕)이라 하였으나 오늘날은 <a href="/wiki/%EB%AA%A8%EC%9A%A9%EC%94%A8" title="모용씨">모용씨</a>가 지배하던 시기와 구분하여 <a href="/wiki/%EB%AA%A8%EC%9A%A9%ED%9D%AC" title="모용희">모용희</a>가 지배하던 때까지를 <a href="/wiki/%ED%9B%84%EC%97%B0" title="후연">후연</a>(後燕), 고운의 등극 이후를 <a href="/wiki/%EB%B6%81%EC%97%B0" title="북연">북연</a>(北燕)으로 구분하고 있다.<sup class="reference" id="cite_ref-1"><a href="#cite_note-1">[1]</a></sup></p>
<p><a href="/wiki/408%EB%85%84" title="408년">408년</a>에는 <a href="/wiki/%EA%B3%A0%EA%B5%AC%EB%A0%A4" title="고구려">고구려</a> <a class="mw-redirect" href="/wiki/%EA%B4%91%EA%B0%9C%ED%86%A0%ED%83%9C%EC%99%95" title="광개토태왕">광개토태왕</a>이 사신을 보내 종족의 예를 베푸니 고운도 <a href="/wiki/%EB%B6%81%EC%97%B0" title="북연">북연</a>의 시어사(侍御史) 이발(李拔)을 보내어 이에 답례하였다는 기록이 《<a href="/wiki/%EC%82%BC%EA%B5%AD%EC%82%AC%EA%B8%B0" title="삼국사기">삼국사기</a>》에 나오는 것으로 보아 고운이 <a href="/wiki/%EA%B3%A0%EA%B5%AC%EB%A0%A4" title="고구려">고구려</a> 왕족 혈통이라는 것에 주목하여 <a href="/wiki/%EA%B4%91%EA%B0%9C%ED%86%A0%EC%99%95" title="광개토왕">광개토왕</a>이 두 나라의 연대를 강화하고 <a href="/wiki/%EA%B3%A0%EA%B5%AC%EB%A0%A4" title="고구려">고구려</a>에 이득이 되는 상황을 만들려 했음을 알 수 있다. 또한, 같은 해에 부인을 황후로, 아들을 황태자로 책봉하였다.</p>
<p>재위 3년째인 <a href="/wiki/409%EB%85%84" title="409년">409년</a> 겨울, 고운은 총신(寵臣)인 <a class="new" href="/w/index.php?title=%EC%9D%B4%EB%B0%98_(%EB%B6%81%EC%97%B0)&amp;action=edit&amp;redlink=1" title="이반 (북연) (없는 문서)">이반</a>(離班)과 <a class="new" href="/w/index.php?title=%EB%8F%84%EC%9D%B8_(%EB%B6%81%EC%97%B0)&amp;action=edit&amp;redlink=1" title="도인 (북연) (없는 문서)">도인</a>(桃仁)에게 암살당했다. 암살자들을 처단하고 고운의 뒤를 이어 즉위한 <a href="/wiki/%ED%92%8D%EB%B0%9C" title="풍발">풍발</a>이 혜의황제(惠懿皇帝)라는 시호(諡號)를 올렸다.</p>
