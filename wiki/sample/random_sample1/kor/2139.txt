<p><b>고은아</b>(高恩雅, <small>본명:</small> <b>방효진</b>(房孝珍), <a href="/wiki/1988%EB%85%84" title="1988년">1988년</a> <a href="/wiki/10%EC%9B%94_28%EC%9D%BC" title="10월 28일">10월 28일</a> ~ )는 <a href="/wiki/%EB%8C%80%ED%95%9C%EB%AF%BC%EA%B5%AD" title="대한민국">대한민국</a>의 <a href="/wiki/%EB%B0%B0%EC%9A%B0" title="배우">배우</a>이다. <a href="/wiki/%EC%A0%84%EB%9D%BC%EB%82%A8%EB%8F%84" title="전라남도">전라남도</a> <a href="/wiki/%EC%9E%A5%EC%84%B1%EA%B5%B0" title="장성군">장성군</a>에서 태어났고, <a href="/wiki/%EC%96%91%EC%9E%AC%EA%B3%A0%EB%93%B1%ED%95%99%EA%B5%90" title="양재고등학교">양재고등학교</a>를 졸업했다. 소속사는 웰메이드스타엠이다. 동생은 <a href="/wiki/%EC%97%A0%EB%B8%94%EB%9E%99" title="엠블랙">엠블랙</a>의 <a href="/wiki/%EB%AF%B8%EB%A5%B4_(%EA%B0%80%EC%88%98)" title="미르 (가수)">미르</a>이다.</p>
