<p><b>안드로이드 응용 프로그램 패키지</b>(<span style="font-size: smaller;"><a href="/wiki/%EC%98%81%EC%96%B4" title="영어">영어</a>:</span> <span lang="en" xml:lang="en">Android application package</span>, APK)는 <a href="/wiki/%EC%95%88%EB%93%9C%EB%A1%9C%EC%9D%B4%EB%93%9C_(%EC%9A%B4%EC%98%81_%EC%B2%B4%EC%A0%9C)" title="안드로이드 (운영 체제)">안드로이드</a>의 <a href="/wiki/%EC%86%8C%ED%94%84%ED%8A%B8%EC%9B%A8%EC%96%B4" title="소프트웨어">소프트웨어</a>와 <a href="/wiki/%EB%AF%B8%EB%93%A4%EC%9B%A8%EC%96%B4" title="미들웨어">미들웨어</a> 배포에 사용되는 <a class="mw-redirect" href="/wiki/%ED%8C%A8%ED%82%A4%EC%A7%80_%ED%98%95%EC%8B%9D" title="패키지 형식">패키지</a> <a class="mw-disambig" href="/wiki/%ED%8C%8C%EC%9D%BC" title="파일">파일</a>이며, '.apk'확장자를 가진다. APK 파일은 <a class="mw-disambig" href="/wiki/%EC%9A%B0%EB%B6%84%ED%88%AC" title="우분투">우분투</a> 같은 <a href="/wiki/%EB%8D%B0%EB%B9%84%EC%95%88" title="데비안">데비안</a> 기반 운영 체제에서 사용하는 <a href="/wiki/Deb_(%ED%8C%8C%EC%9D%BC_%ED%98%95%EC%8B%9D)" title="Deb (파일 형식)">뎁 패키지</a>와 <a class="mw-redirect" href="/wiki/%EB%A7%88%EC%9D%B4%ED%81%AC%EB%A1%9C%EC%86%8C%ED%94%84%ED%8A%B8_%EC%9C%88%EB%8F%84" title="마이크로소프트 윈도">마이크로소프트 윈도</a>에서 사용하는 <a class="mw-redirect" href="/wiki/%EC%9C%88%EB%8F%84_%EC%9D%B8%EC%8A%A4%ED%86%A8%EB%9F%AC" title="윈도 인스톨러">MSI 패키지</a>와 같은 설치 파일과 비슷하다. APK 파일을 만들려면, 안드로이드용 <a class="mw-disambig" href="/wiki/%ED%94%84%EB%A1%9C%EA%B7%B8%EB%9E%A8" title="프로그램">프로그램</a>을 먼저 컴파일한 후, 모든 파일들을 하나의 패키지 파일로 모은다. APK 파일은 해당하는 프로그램의 모든 코드를 포함하며, 자원, 정보, 인증서 및 <a class="new" href="/w/index.php?title=%EB%A7%A4%EB%8B%88%ED%8E%98%EC%8A%A4%ED%8A%B8_%ED%8C%8C%EC%9D%BC&amp;action=edit&amp;redlink=1" title="매니페스트 파일 (없는 문서)">매니페스트 파일</a> (<a class="extiw" href="https://en.wikipedia.org/wiki/Manifest_file" title="en:Manifest file">en:Manifest file</a>) 등을 포함한다.<sup class="reference" id="cite_ref-1"><a href="#cite_note-1">[1]</a></sup><sup class="reference" id="cite_ref-2"><a href="#cite_note-2">[2]</a></sup><sup class="reference" id="cite_ref-3"><a href="#cite_note-3">[3]</a></sup><sup class="reference" id="cite_ref-4"><a href="#cite_note-4">[4]</a></sup></p>
<p>안드로이드 응용 프로그램 패키지의 확장자는 <code>.apk</code>로 <a href="/wiki/ZIP_(%ED%8C%8C%EC%9D%BC_%ED%8F%AC%EB%A7%B7)" title="ZIP (파일 포맷)">ZIP</a>파일 기반인 <a href="/wiki/%EC%9E%90%EB%B0%94_(%ED%94%84%EB%A1%9C%EA%B7%B8%EB%9E%98%EB%B0%8D_%EC%96%B8%EC%96%B4)" title="자바 (프로그래밍 언어)">JAR</a>를 기반으로 하며, <a href="/wiki/%EC%95%95%EC%B6%95_%ED%8C%8C%EC%9D%BC" title="압축 파일">압축 파일</a>의 한 종류이다. MIME 유형은 <code>application/vnd.android.package-archive</code>이다.<sup class="reference" id="cite_ref-5"><a href="#cite_note-5">[5]</a></sup></p>
<h2><span class="mw-headline" id=".ED.8C.8C.EC.9D.BC_.EA.B5.AC.EC.84.B1">파일 구성</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%95%88%EB%93%9C%EB%A1%9C%EC%9D%B4%EB%93%9C_%EC%9D%91%EC%9A%A9_%ED%94%84%EB%A1%9C%EA%B7%B8%EB%9E%A8_%ED%8C%A8%ED%82%A4%EC%A7%80&amp;action=edit&amp;section=1" title="부분 편집: 파일 구성">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>APK 파일에서 일반적으로 포함하는 파일 또는 내용은 아래와 같다.</p>
<ul>
<li><code>META-INF</code> : 디렉토리:
<ul>
<li><code>MANIFEST.MF</code> : <a class="new" href="/w/index.php?title=%EB%A9%94%EB%8B%88%ED%8E%98%EC%8A%A4%ED%8A%B8_%ED%8C%8C%EC%9D%BC&amp;action=edit&amp;redlink=1" title="메니페스트 파일 (없는 문서)">메니페스트 파일</a> (<a class="extiw" href="https://en.wikipedia.org/wiki/Manifest_file" title="en:Manifest file">en:Manifest file</a>).</li>
<li><code>CERT.RSA</code> : 응용 프로그램에 대한 <a class="mw-redirect" href="/wiki/%EC%A0%84%EC%9E%90_%EC%84%9C%EB%AA%85" title="전자 서명">전자 서명</a> 파일.</li>
<li><code>CERT.SF</code> : 메니페스트 파일을 포함한 리소스와 그에 대응하는 <a class="mw-redirect" href="/wiki/SHA-1" title="SHA-1">SHA-1</a> 해시값. 예제 :</li>
</ul>
</li>
</ul>
<pre>
 Signature-Version: 1.0
 Created-By: 1.0 (Android)
 SHA1-Digest-Manifest: wxqnEAI0UA5nO5QJ8CGMwjkGGWE=
 ...
 Name: res/layout/exchange_component_back_bottom.xml
 SHA1-Digest: eACjMjESj7Zkf0cBFTZ0nqWrt7w=
 ...
 Name: res/drawable-hdpi/icon.png
 SHA1-Digest: DGEqylP8W0n0iV/ZzBx3MW0WGCA=
</pre>
<ul>
<li><code>lib</code> : 프로세스와 연계된 소프트웨어 레이어를 포함하는 디렉토리. 이 디렉토리는 세부적으로 더 나뉘게 된다:
<ul>
<li><code>armeabi</code> : <a href="/wiki/ARM_%EC%95%84%ED%82%A4%ED%85%8D%EC%B2%98" title="ARM 아키텍처">ARM</a> 기반 프로세서 전용으로 컴파일 된 코드.</li>
<li><code>armeabi-v7a</code> : ARMv7 와 그것을 기반으로 하는 프로세서 전용으로 컴파일 된 코드.</li>
<li><code>x86</code> : <a href="/wiki/X86" title="X86">x86</a> 기반 프로세서 전용으로 컴파일 된 코드.</li>
<li><code>mips</code> : <a href="/wiki/MIPS_%EC%95%84%ED%82%A4%ED%85%8D%EC%B2%98" title="MIPS 아키텍처">MIPS</a> 기반 프로세서 전용으로 컴파일 된 코드.</li>
</ul>
</li>
<li><code>res</code> : resources.arsc(아래 참고)에서 제외된 정보를 포함하는 디렉토리.</li>
<li><code>assets</code> : AssetManager를 통해 검색 ​​할 수 있는, 응용 프로그램의 정보을 포함하는 디렉토리.</li>
<li><code>AndroidManifest.xml</code>: 추가적인 매니페스트 파일. 응용 프로그램의 이름, 버전, 액세스 권한, 라이브러리 파일을 정보를 포함한다. 이 파일은 이진 <a href="/wiki/XML" title="XML">XML</a> 파일로써 <a class="external text" href="http://code.google.com/p/android4me/downloads/list" rel="nofollow">AXMLPrinter2</a>, <a class="external text" href="http://code.google.com/p/android-apktool/" rel="nofollow">tool</a> 또는 <a class="external text" href="http://code.google.com/p/androguard/wiki/Usage#Androaxml" rel="nofollow">Androguard</a> 과 같은 프로그램을 사용하면 일반적인 <a href="/wiki/XML" title="XML">XML</a>파일과 같이 볼 수 있다.</li>
<li><code>classes.dex</code> : <a class="mw-redirect" href="/wiki/%EB%8B%AC%EB%B9%85_%EA%B0%80%EC%83%81_%EB%A8%B8%EC%8B%A0" title="달빅 가상 머신">달빅 가상 머신</a>에 의해 dex 파일 형식으로 컴파일 된 영역.</li>
<li><code>resources.arsc</code> : 이진 <a href="/wiki/XML" title="XML">XML</a>과 같은 선컴파일된 리소스를 포함하는 파일.</li>
</ul>
