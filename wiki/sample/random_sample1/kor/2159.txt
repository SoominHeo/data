<p><b>고토지 선</b>(<span style="font-size: smaller;"><a href="/wiki/%EC%9D%BC%EB%B3%B8%EC%96%B4" title="일본어">일본어</a>:</span> <span lang="ja" xml:lang="ja">後藤寺線</span>)은, <a href="/wiki/%EC%9D%BC%EB%B3%B8" title="일본">일본</a> <a href="/wiki/%ED%9B%84%EC%BF%A0%EC%98%A4%EC%B9%B4_%ED%98%84" title="후쿠오카 현">후쿠오카 현</a> <a href="/wiki/%EB%8B%A4%EA%B0%80%EC%99%80_%EC%8B%9C" title="다가와 시">다가와 시</a>의 <a href="/wiki/%EB%8B%A4%EA%B0%80%EC%99%80%EA%B3%A0%ED%86%A0%EC%A7%80_%EC%97%AD" title="다가와고토지 역">다가와고토지 역</a>부터 후쿠오카 현 <a href="/wiki/%EC%9D%B4%EC%A6%88%EC%B9%B4_%EC%8B%9C" title="이즈카 시">이즈카 시</a>의 <a href="/wiki/%EC%8B%A0%EC%9D%B4%EC%A6%88%EC%B9%B4_%EC%97%AD" title="신이즈카 역">신이즈카 역</a>에 이르는 <a class="mw-redirect" href="/wiki/%EA%B7%9C%EC%8A%88_%EC%97%AC%EA%B0%9D%EC%B2%A0%EB%8F%84" title="규슈 여객철도">규슈 여객철도</a>의 철도 노선(지방 교통선)이다.</p>
<p>지쿠호 서부의 이즈카 시를 경유해서 동부의 다가와 시와 <a href="/wiki/%ED%9B%84%EC%BF%A0%EC%98%A4%EC%B9%B4_%EC%8B%9C" title="후쿠오카 시">후쿠오카 시</a>를 연결하는 단선 경로로 되어 있다.</p>
<h2><span class="mw-headline" id=".EB.85.B8.EC.84.A0_.EC.A0.95.EB.B3.B4">노선 정보</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EA%B3%A0%ED%86%A0%EC%A7%80_%EC%84%A0&amp;action=edit&amp;section=1" title="부분 편집: 노선 정보">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<ul>
<li>관할(사업 종별) : 규슈 여객철도 (<a class="mw-redirect" href="/wiki/%EC%B2%A0%EB%8F%84%EC%82%AC%EC%97%85%EC%9E%90" title="철도사업자">제 1 종 철도사업자</a>)</li>
<li>구간(영업 거리) : 다가와고토지 역 - 신이즈카 역 13.3km (선로 명칭 상은 기·종점이 역전)</li>
<li><a href="/wiki/%EA%B6%A4%EA%B0%84" title="궤간">궤간</a> : 1067mm</li>
<li>역 수 : 6
<ul>
<li>고토지 선 소속역으로 한정된 경우, 기·종점역(다가와고토지 역은 <a href="/wiki/%ED%9E%88%ED%83%80%ED%9E%88%EC%BD%94%EC%82%B0_%EC%84%A0" title="히타히코산 선">히타히코산 선</a>, 신이즈카 역은 <a href="/wiki/%EC%A7%80%EC%BF%A0%ED%98%B8_%EB%B3%B8%EC%84%A0" title="지쿠호 본선">지쿠호 본선</a>의 소속)이 제외되어, 4개 역이 된다.</li>
</ul>
</li>
<li>복선 구간 : 없음 (전 노선 <a href="/wiki/%EB%8B%A8%EC%84%A0_(%EC%B2%A0%EB%8F%84)" title="단선 (철도)">단선</a></li>
<li>전선화 구간 : 없음 (전 노선 비 전선화)</li>
<li><a href="/wiki/%ED%8F%90%EC%83%89" title="폐색">폐색</a>방식 : 특수 자동 폐색식(궤도 회로 검지식)</li>
</ul>
<p>전 노선이 본사 직할로 되어 있다.</p>
<p>또한, 전 노선이 여객영업규칙의 정한 대도시 근교 구간의 '후쿠오카 근교 구간'으로 포함된다. 고토지 선에는 <a href="/wiki/SUGOCA" title="SUGOCA">SUGOCA</a>의 이용은 하지 않지만, 도중 하차를 하지 않고 통과(신이즈카 역 이원 - 히타히코산 선 경유 <a href="/wiki/%EB%8B%9B%ED%8F%AC_%EB%B3%B8%EC%84%A0" title="닛포 본선">닛포 본선</a> 방면)하는 것뿐이라면 이용하다.</p>
