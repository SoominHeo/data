<p><b>페르디난트 고트홀트 막스 아이젠슈타인</b>(<span style="font-size: smaller;"><a href="/wiki/%EB%8F%85%EC%9D%BC%EC%96%B4" title="독일어">독일어</a>:</span> <span lang="de" xml:lang="de">Ferdinand Gotthold Max Eisenstein</span>, 1823 – 1852)은 <a href="/wiki/%ED%94%84%EB%A1%9C%EC%9D%B4%EC%84%BC" title="프로이센">프로이센</a>의 <a href="/wiki/%EC%88%98%ED%95%99%EC%9E%90" title="수학자">수학자</a>다. <a class="mw-redirect" href="/wiki/%EC%88%98%EB%A1%A0" title="수론">수론</a>과 <a href="/wiki/%ED%95%B4%EC%84%9D%ED%95%99_(%EC%88%98%ED%95%99)" title="해석학 (수학)">해석학</a>에 기여하였으나, 29세에 요절하였다.</p>
<h2><span class="mw-headline" id=".EC.83.9D.EC.95.A0">생애</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EA%B3%A0%ED%8A%B8%ED%99%80%ED%8A%B8_%EC%95%84%EC%9D%B4%EC%A0%A0%EC%8A%88%ED%83%80%EC%9D%B8&amp;action=edit&amp;section=1" title="부분 편집: 생애">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p><a href="/wiki/%EB%B2%A0%EB%A5%BC%EB%A6%B0" title="베를린">베를린</a>에서 <a href="/wiki/%EA%B0%9C%EC%8B%A0%EA%B5%90" title="개신교">개신교</a>로 개종한 <a href="/wiki/%EC%9C%A0%EB%8C%80%EC%9D%B8" title="유대인">유대인</a> 가정에서 태어났다. 아버지는 요한 콘스탄틴 아이젠슈타인(<span style="font-size: smaller;"><a href="/wiki/%EB%8F%85%EC%9D%BC%EC%96%B4" title="독일어">독일어</a>:</span> <span lang="de" xml:lang="de">Johan Konstantin Eisenstein</span>), 어머니는 헬레네 폴라크(<span style="font-size: smaller;"><a href="/wiki/%EB%8F%85%EC%9D%BC%EC%96%B4" title="독일어">독일어</a>:</span> <span lang="de" xml:lang="de">Helene Pollack</span>)였다. 총 6남매였는데, 아이젠슈타인을 제외한 나머지 5명은 모두 <a href="/wiki/%EC%88%98%EB%A7%89%EC%97%BC" title="수막염">수막염</a>으로 어려서 사망하였고, 아이젠슈타인 또한 수막염에 걸렸으나 살아남았다.</p>
<p>어릴 때부터 <a href="/wiki/%EC%88%98%ED%95%99" title="수학">수학</a>과 <a href="/wiki/%EC%9D%8C%EC%95%85" title="음악">음악</a>에 재능을 보였고, 특히 <a href="/wiki/%ED%94%BC%EC%95%84%EB%85%B8" title="피아노">피아노</a>를 탁월하게 연주하였다. 아이젠슈타인은 평생 피아노를 연주하고 피아노곡을 작곡하였다. 14세 때 프리드리히 빌헬름 <a href="/wiki/%EA%B9%80%EB%82%98%EC%A7%80%EC%9B%80" title="김나지움">김나지움</a>에 입학하였고, 곧 프리드리히 베르더(Friedrich Werder) 김나지움으로 전학하였다. 15세 때 이미 김나지움에서 가르치는 모든 수학을 통달하고, <a href="/wiki/%EB%A0%88%EC%98%A8%ED%95%98%EB%A5%B4%ED%8A%B8_%EC%98%A4%EC%9D%BC%EB%9F%AC" title="레온하르트 오일러">레온하르트 오일러</a>와 <a href="/wiki/%EC%A1%B0%EC%A0%9C%ED%94%84%EB%A3%A8%EC%9D%B4_%EB%9D%BC%EA%B7%B8%EB%9E%91%EC%A3%BC" title="조제프루이 라그랑주">조제프루이 라그랑주</a>의 저서들을 읽기 시작하였다.</p>
<p>17세 때 <a href="/wiki/%EB%B2%A0%EB%A5%BC%EB%A6%B0_%ED%9B%94%EB%B3%BC%ED%8A%B8_%EB%8C%80%ED%95%99%EA%B5%90" title="베를린 훔볼트 대학교">베를린 훔볼트 대학교</a>에서 <a href="/wiki/%ED%8E%98%ED%84%B0_%EA%B5%AC%EC%8A%A4%ED%83%80%ED%94%84_%EB%A5%B4%EC%A3%88_%EB%94%94%EB%A6%AC%ED%81%B4%EB%A0%88" title="페터 구스타프 르죈 디리클레">페터 구스타프 르죈 디리클레</a>의 강의를 듣기 시작하였다. 이 때 실직자였던 아버지는 일자리를 찾으러 홀로 <a href="/wiki/%EC%9E%89%EA%B8%80%EB%9E%9C%EB%93%9C" title="잉글랜드">잉글랜드</a>로 떠났고, 연락이 끊겼다. 1842년에 아이젠슈타인은 어머니와 함께 아버지를 찾으러 잉글랜드로 여행하였다. 1843년에 아이젠슈타인은 <a href="/wiki/%EB%8D%94%EB%B8%94%EB%A6%B0" title="더블린">더블린</a>에서 <a href="/wiki/%EC%9C%8C%EB%A6%AC%EC%97%84_%EB%A1%9C%EC%96%B8_%ED%95%B4%EB%B0%80%ED%84%B4" title="윌리엄 로언 해밀턴">윌리엄 로언 해밀턴</a>을 만났고, 해밀턴은 아이젠슈타인에게 <a href="/wiki/%EB%8B%90%EC%8A%A4_%ED%97%A8%EB%A6%AC%ED%81%AC_%EC%95%84%EB%B2%A8" title="닐스 헨리크 아벨">닐스 헨리크 아벨</a>의 업적을 소개시켜 주었다.</p>
<p>1843년에 아이젠슈타인은 베를린으로 돌아와 김나지움을 졸업하였고, 공식적으로 <a href="/wiki/%EB%B2%A0%EB%A5%BC%EB%A6%B0_%ED%9B%94%EB%B3%BC%ED%8A%B8_%EB%8C%80%ED%95%99%EA%B5%90" title="베를린 훔볼트 대학교">베를린 훔볼트 대학교</a>에 입학하였다. 1844년 1월에 이미 프로이센 과학 아카데미에 첫 논문을 발표하였고, 그 뒤 5년 동안 극심한 궁핍에도 불구하고 매우 왕성한 연구 활동을 펼쳤다. 1844년만 해도 총 23편의 논문을 출판하였다.</p>
<p>공화주의적 정치적 성향을 가졌던 아이젠슈타인은 <a href="/wiki/1848%EB%85%84_%EB%8F%85%EC%9D%BC_%ED%98%81%EB%AA%85" title="1848년 독일 혁명">1848년 독일 혁명</a>을 관계로 1848년 3월 19일에 체포되었고, 다음날 풀려났다. 그러나 이 관계로 모든 학비 보조금이 다 끊기게 되었고, 또한 옥중에서 받은 구타가 이미 쇠약했던 아이젠슈타인의 건강을 크게 악화시켰다. 당대 과학계의 유명 인사였던 <a href="/wiki/%EC%95%8C%EB%A0%89%EC%82%B0%EB%8D%94_%ED%8F%B0_%ED%9B%94%EB%B3%BC%ED%8A%B8" title="알렉산더 폰 훔볼트">알렉산더 폰 훔볼트</a>가 힘을 써서, 1852년에 아이젠슈타인을 <a href="/wiki/%EC%8B%9C%EC%B9%A0%EB%A6%AC%EC%95%84" title="시칠리아">시칠리아</a>로 휴양하게 할 수 있는 돈을 마련하였으나, 돈이 채 모이기 전에 아이젠슈타인은 1852년 <a href="/wiki/%EA%B2%B0%ED%95%B5" title="결핵">결핵</a>으로 29세의 나이로 요절하였다. 이미 83세인 훔볼트는 손수 아이젠슈타인의 장례식에서 아이젠슈타인의 유해를 묘지까지 대동하였다.</p>
