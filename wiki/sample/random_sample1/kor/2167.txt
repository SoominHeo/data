<p><b>고훈 시대</b>(<span style="font-size: smaller;"><a href="/wiki/%EC%9D%BC%EB%B3%B8%EC%96%B4" title="일본어">일본어</a>:</span> <span lang="ja"><span lang="ja"><ruby><rb>古墳時代</rb> <rp>(</rp><rt>こふんじだい</rt><rp>)</rp></ruby></span></span> <small>고훈지다이<sup>[<a href="/wiki/%EC%9C%84%ED%82%A4%EB%B0%B1%EA%B3%BC:%EC%9D%BC%EB%B3%B8%EC%96%B4%EC%9D%98_%ED%95%9C%EA%B8%80_%ED%91%9C%EA%B8%B0" title="위키백과:일본어의 한글 표기">*</a>]</sup></small>;고분시대)는 <a href="/wiki/%EC%9D%BC%EB%B3%B8%EC%9D%98_%EC%97%AD%EC%82%AC" title="일본의 역사">일본의 역사</a> 구분으로, <a href="/wiki/3%EC%84%B8%EA%B8%B0" title="3세기">3세기</a> 중반부터 <a href="/wiki/7%EC%84%B8%EA%B8%B0" title="7세기">7세기</a> 말까지의 약 400년을 가리킨다. 3세기 이후 도래인의 유입은 드물어지다가,<sup class="reference" id="cite_ref-.EA.B5.AC.ED.83.9C.ED.9B.88_1-0"><a href="#cite_note-.EA.B5.AC.ED.83.9C.ED.9B.88-1">[1]</a></sup> <span class="reference plainlinksneverexpand" style="white-space:nowrap"><sup>:19</sup></span> 3세기 후반부터 <a href="/wiki/%ED%98%B8%EC%A1%B1" title="호족">호족</a>의 연합정권인 <a href="/wiki/%EC%95%BC%EB%A7%88%ED%86%A0_%EC%8B%9C%EB%8C%80" title="야마토 시대">야마토 조정</a>이 일본 통일을 시작하면서, 새로운 지배자의 권위의 상징으로서 각지에 고분(고훈)이 축조되고, 대륙과 <a href="/wiki/%ED%95%9C%EB%B0%98%EB%8F%84" title="한반도">한반도</a>에서 산업기술과 한자를 수입하여 고훈 문화가 성립되었다.<sup class="reference" id="cite_ref-2"><a href="#cite_note-2">[2]</a></sup></p>

<h2><span class="mw-headline" id=".EC.A0.95.EC.B9.98">정치</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EA%B3%A0%ED%9B%88_%EC%8B%9C%EB%8C%80&amp;action=edit&amp;section=1" title="부분 편집: 정치">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>3세기 중반부터 <a href="/wiki/6%EC%84%B8%EA%B8%B0" title="6세기">6세기</a> 말까지는 전방후원분(前方後円墳)이 북쪽으로는 <a href="/wiki/%EB%8F%84%ED%98%B8%EC%BF%A0_%EC%A7%80%EB%B0%A9" title="도호쿠 지방">도호쿠 지방</a>에서 남쪽으로는 <a class="mw-redirect" href="/wiki/%ED%81%90%EC%8A%88" title="큐슈">큐슈</a>지방 남부까지 계속 만들어져 왔기 때문에 <b>전방후원분의 세기</b>라고도 말한다. 전방후원분은 <a href="/wiki/%EC%95%BC%EB%A7%88%ED%86%A0_%EC%8B%9C%EB%8C%80" title="야마토 시대">야마토 조정</a>이 통일정권으로서 확립되어가는 중에, 각지의 호족에게 허가한 형식이라고 여겨진다. 3세기 후반부터 <a class="new" href="/w/index.php?title=%EB%82%98%EB%9D%BC_%EB%B6%84%EC%A7%80&amp;action=edit&amp;redlink=1" title="나라 분지 (없는 문서)">나라 분지</a>에서 왕의 무덤으로 보이는 전대보다 각별히 규모가 큰 전방후원분이 출현하고, 4세기 초기에는 <a class="new" href="/w/index.php?title=%EC%98%A4%EC%82%AC%EC%B9%B4_%ED%8F%89%EC%95%BC&amp;action=edit&amp;redlink=1" title="오사카 평야 (없는 문서)">오사카 평야</a>에서 거대고분이 만들어졌으며, 4세기 말기 무렵에는 키나이(畿内)의 일부 지방에서 선진적인 고분군이 출현했다. 계속되는 5세기 중반에는 각지에 거대고분이 만들어지게 되었다. 그러나 6세기 말기에는 일본 각지에서 거의 시간적으로 동일하게 전방후원분이 만들어지지 않게 되었다. 이것은 <a class="mw-redirect" href="/wiki/%EC%95%BC%EB%A7%88%ED%86%A0_%EC%A0%95%EA%B6%8C" title="야마토 정권">야마토 정권</a>이 확립에 따라 중앙・지방의 통치조직이 완성되어 더욱 강력한 정권으로 성장하였다는 증거로 해석된다. 전방후원분이 만들어지지 않게 된 7세기는 방분(方墳)·원분(円墳)이 계속해서 만들어지게 된다. 오오키미(大王)의 무덤은 특별히 팔각분(八角墳)으로 만들어졌다. 7세기 이후를 <b>종말기</b>라고 부르고 있다.</p>
<p>일본국가의 성립부터 고찰해보면, 전기・중기의 고대국가의 형성기를 지나, 후기에서 종말기에 걸쳐서 일본의 고대국가가 성립되었다고 여겨진다.</p>
<h2><span class="mw-headline" id=".EC.82.AC.ED.9A.8C">사회</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EA%B3%A0%ED%9B%88_%EC%8B%9C%EB%8C%80&amp;action=edit&amp;section=2" title="부분 편집: 사회">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p><a href="/wiki/4%EC%84%B8%EA%B8%B0" title="4세기">4세기</a> 말에서 <a href="/wiki/6%EC%84%B8%EA%B8%B0" title="6세기">6세기</a> 중기까지 도래인이 급증하기 시작했다.<sup class="reference" id="cite_ref-.EA.B5.AC.ED.83.9C.ED.9B.88_1-1"><a href="#cite_note-.EA.B5.AC.ED.83.9C.ED.9B.88-1">[1]</a></sup> <span class="reference plainlinksneverexpand" style="white-space:nowrap"><sup>:19</sup></span> <a href="/wiki/%EA%B3%A0%EA%B5%AC%EB%A0%A4" title="고구려">고구려</a>가 4세기 후반부터 남진 정책으로 <a href="/wiki/%ED%95%9C%EB%B0%98%EB%8F%84" title="한반도">한반도</a> 남부의 <a href="/wiki/%EB%B0%B1%EC%A0%9C" title="백제">백제</a>와 <a href="/wiki/%EA%B0%80%EC%95%BC" title="가야">가야</a>에 무차별적인 공격을 가하면서 일본 열도로 피난 오는 사람들이 급증하였기 때문이다.<sup class="reference" id="cite_ref-.EA.B5.AC.ED.83.9C.ED.9B.88_1-2"><a href="#cite_note-.EA.B5.AC.ED.83.9C.ED.9B.88-1">[1]</a></sup> <span class="reference plainlinksneverexpand" style="white-space:nowrap"><sup>:19</sup></span> 이 시기 일본 왕실은 백제를 비롯하여 백제 등지에서 피난 오는 도래 기술자로부터 유용한 대륙 기술을 도입하기 시작하였으며 이를 위해 백제가 고구려에 대적할 수 있도록 전면적 지원을 아끼지 않았다.<sup class="reference" id="cite_ref-.EA.B5.AC.ED.83.9C.ED.9B.88_1-3"><a href="#cite_note-.EA.B5.AC.ED.83.9C.ED.9B.88-1">[1]</a></sup> <span class="reference plainlinksneverexpand" style="white-space:nowrap"><sup>:20</sup></span> 이처럼 일본 왕실이 도래인들에게 각종 특혜를 주면서 외교와 경제 분야에서 도래 씨족이 실권을 장악하기 시작하였다.<sup class="reference" id="cite_ref-.EA.B5.AC.ED.83.9C.ED.9B.88_1-4"><a href="#cite_note-.EA.B5.AC.ED.83.9C.ED.9B.88-1">[1]</a></sup> <span class="reference plainlinksneverexpand" style="white-space:nowrap"><sup>:21</sup></span></p>
<h2><span class="mw-headline" id=".EB.AC.B8.ED.99.94">문화</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EA%B3%A0%ED%9B%88_%EC%8B%9C%EB%8C%80&amp;action=edit&amp;section=3" title="부분 편집: 문화">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>이 시기에 중국과 한반도에서 문자와 유교가 전해졌다. 이 시대의 사람들은 토사기(土師器)와 스에기(須惠器)를 사용했다. 청동기도 대륙에서 들어왔지만, 고대 오리엔트 지역과 달라서 철기를 사용하기 시작한 시기와 격차가 없기 때문에 실제로 사용할 기회가 없었고, 성능이 뛰어난 철이 대신 사용되었다. 그 때문에 청동기는 동탁(銅鐸)과 같은 제사, 종교용으로만 쓰이게 되었다.</p>
<h2><span class="mw-headline" id=".EB.8C.80.EC.99.B8_.EA.B4.80.EA.B3.84">대외 관계</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EA%B3%A0%ED%9B%88_%EC%8B%9C%EB%8C%80&amp;action=edit&amp;section=4" title="부분 편집: 대외 관계">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>대외 관계로는, 5세기에는 <a href="/wiki/%EC%99%9C5%EC%99%95" title="왜5왕">야마토의 오왕</a>(倭の五王)이 중국에 사자를 보냈다.</p>
<h3><span class="mw-headline" id=".EB.B0.B1.EC.A0.9C.EC.99.80.EC.9D.98_.EA.B4.80.EA.B3.84">백제와의 관계</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EA%B3%A0%ED%9B%88_%EC%8B%9C%EB%8C%80&amp;action=edit&amp;section=5" title="부분 편집: 백제와의 관계">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>백제의 <a href="/wiki/%EA%B7%BC%EC%B4%88%EA%B3%A0%EC%99%95" title="근초고왕">근초고왕</a>(재위 346년 ~ 375년)은 왜와 국교를 수립하고 많은 선진 문화를 전했다. <sup class="reference" id="cite_ref-3"><a href="#cite_note-3">[주 1]</a></sup> 백제가 언제부터 왜와 통교하고 있었는지는 정확하지 않으나 근초고왕대에 이르러 양국은 활기차게 적극적인 교류를 하였다.<sup class="reference" id="cite_ref-4"><a href="#cite_note-4">[3]</a></sup> 백제와 왜의 국교 수립은 서기 366년(<a href="/wiki/%EC%A7%84%EA%B5%AC_%ED%99%A9%ED%9B%84" title="진구 황후">진구 황후</a> 46년, 근초고왕 21년)<sup class="reference" id="cite_ref-5"><a href="#cite_note-5">[주 2]</a></sup> 에 이뤄졌다.<sup class="reference" id="cite_ref-6"><a href="#cite_note-6">[4]</a></sup> 이처럼 왜는 정치적으로 백제와 밀접한 관계를 맺고, 백제로부터 우수한 선진문물을 수입하고, 또 한반도로부터 많은 기술자와 농민이 도래하여 각종 기술을 전하는 등<sup class="reference" id="cite_ref-7"><a href="#cite_note-7">[5]</a></sup>, 백제의 일본 문화 전파는 다방면에서 이루어졌다. <a href="/wiki/%EA%B7%BC%EC%B4%88%EA%B3%A0%EC%99%95" title="근초고왕">근초고왕</a> 때, 유교경전에 밝은 <a href="/wiki/%EC%95%84%EC%A7%81%EA%B8%B0" title="아직기">아직기</a>(阿直岐)가 일본의 태자에게 한자를 가르쳤고, 이후 박사 <a href="/wiki/%EC%99%95%EC%9D%B8" title="왕인">왕인</a>(王仁)은 《<a href="/wiki/%EB%85%BC%EC%96%B4" title="논어">논어</a>》와 《<a href="/wiki/%EC%B2%9C%EC%9E%90%EB%AC%B8" title="천자문">천자문</a>》을 전하고 경사를 가르쳤다. 국가 체제를 정비하는 데 있어 유교적 이념은 다른 토착신앙이나 불교보다도 많은 것을 제공했기 때문에, 각 국가들은 고대 사회의 형성기에 유교 사상의 수입에 적극적이었다.<sup class="reference" id="cite_ref-8"><a href="#cite_note-8">[6]</a></sup> 당시 백제가 <a href="/wiki/%EC%99%9C" title="왜">왜</a>에 보낸 <a href="/wiki/%EC%B9%A0%EC%A7%80%EB%8F%84" title="칠지도">칠지도</a>는 양국의 교류를 보여주는 중요한 유물 중의 하나이다. 백제는 <a href="/wiki/%EA%B7%BC%EC%B4%88%EA%B3%A0%EC%99%95" title="근초고왕">근초고왕</a> 시기 때부터 <a href="/wiki/%EC%84%AC%EC%A7%84%EA%B0%95" title="섬진강">섬진강</a> 유역으로 진출하여 하구에 위치한 <a class="mw-redirect" href="/wiki/%ED%95%98%EB%8F%99" title="하동">하동</a>을 대왜교역의 거점으로 확보하였다. 백제는 초기에 한성에서 천안을 거쳐 금강상류로 와서 남원에서 섬진강을 타고 하동으로 가는 교역로를 확보하였다. 하동에서 <a href="/wiki/%EC%93%B0%EC%8B%9C%EB%A7%88_%EC%84%AC" title="쓰시마 섬">쓰시마 섬</a>을 축으로 해서 <a href="/wiki/%EC%9D%B4%ED%82%A4_%EC%84%AC" title="이키 섬">이키 섬</a>, 그리고 <a href="/wiki/%EA%B7%9C%EC%8A%88" title="규슈">규슈</a>의 <a href="/wiki/%EB%A7%88%EC%93%B0%EC%9A%B0%EB%9D%BC_%EC%8B%9C" title="마쓰우라 시">마쓰우라 반도(松浦半島)</a>에 이르는 길과 <a href="/wiki/%EC%93%B0%EC%8B%9C%EB%A7%88_%EC%84%AC" title="쓰시마 섬">쓰시마 섬</a>에서 오키노시마(沖ノ島), <a href="/wiki/%ED%9B%84%EC%BF%A0%EC%98%A4%EC%B9%B4_%ED%98%84" title="후쿠오카 현">후쿠오카 현</a> 북쪽 해안에 이르는 길이 생겨났다.<sup class="reference" id="cite_ref-9"><a href="#cite_note-9">[7]</a></sup> 고구려 <a href="/wiki/%EC%9E%A5%EC%88%98%EC%99%95" title="장수왕">장수왕</a>(재위: 412년 음력 10월~491년 음력 12월)의 군사들이 한성으로 남침해 <a href="/wiki/%EA%B0%9C%EB%A1%9C%EC%99%95" title="개로왕">개로왕</a>이 살해(475년)<sup class="reference" id="cite_ref-10"><a href="#cite_note-10">[8]</a></sup> 되고, 왕자인 <a href="/wiki/%EB%AC%B8%EC%A3%BC%EC%99%95" title="문주왕">문주왕</a>(재위 475년~477년)은 위기에 빠진 백제를 구하기 위한 숱한 고민 끝에, 목만치와 함께 남쪽으로 갔다. 여기서 "남쪽"이 일본을 뜻한다고 하는 견해들이 있다.<sup class="reference" id="cite_ref-11"><a href="#cite_note-11">[9]</a></sup><sup class="reference" id="cite_ref-12"><a href="#cite_note-12">[10]</a></sup><sup class="reference" id="cite_ref-13"><a href="#cite_note-13">[11]</a></sup> 이러한 견해에 따르면, 목만치는 일본에 건너가 성(姓)을 바꾸어 <a href="/wiki/%EC%86%8C%EA%B0%80%EB%85%B8_%EB%A7%88%EC%B9%98" title="소가노 마치">소가노 마치</a>라는 이름의 귀족이 된다. 하지만 이는 확실하지 않다. 일단 문주왕은 백제의 귀족에게 암살당해 죽었다. 그래서 목만치와 같이 일본에 갔다는 것은 성립되지 않는다.게다가 목만치가 일본 귀족이 되었다는 것이 아니라는 설도 제기되고 있어 확실한 것은 없다.</p>

<p>현존하는 <a href="/wiki/%EC%8A%A4%EB%8B%A4%ED%95%98%EC%B9%98%EB%A7%8C_%EC%8B%A0%EC%82%AC_%EC%9D%B8%EB%AC%BC%ED%99%94%EC%83%81%EA%B2%BD" title="스다하치만 신사 인물화상경">인물화상경</a>(5~6세기)은 <a class="mw-redirect" href="/wiki/%EB%B0%B1%EC%A0%9C_%EB%AC%B4%EB%A0%B9%EC%99%95" title="백제 무령왕">백제 무령왕</a>(재위 501년~523년)이 <a href="/wiki/%EA%B2%8C%EC%9D%B4%ED%83%80%EC%9D%B4_%EC%B2%9C%ED%99%A9" title="게이타이 천황">게이타이 천황</a>(재위 507년 ~ 531년)에게 선물로 보낸 거울인데, 이것은 백제와 <a href="/wiki/%EC%99%9C" title="왜">왜</a>의 친밀한 사이를 증명한다. 무령왕은 <a href="/wiki/513%EB%85%84" title="513년">513년</a> 오경박사 단양이(段楊爾)<sup class="reference" id="cite_ref-14"><a href="#cite_note-14">[12]</a></sup> 와 <a href="/wiki/516%EB%85%84" title="516년">516년</a> <a href="/wiki/%EA%B3%A0%EC%95%88%EB%AC%B4" title="고안무">고안무</a>(高安茂)<sup class="reference" id="cite_ref-15"><a href="#cite_note-15">[13]</a></sup> 를 일본에 보내어 <a href="/wiki/%EB%B0%B1%EC%A0%9C" title="백제">백제</a>의 <a href="/wiki/%EB%AC%B8%ED%99%94" title="문화">문화</a>를 <a href="/wiki/%EC%9D%BC%EB%B3%B8" title="일본">일본</a>에 전해주기도 하였다. 무령왕의 왕위를 계승한 <a href="/wiki/%EC%84%B1%EC%99%95_(%EB%B0%B1%EC%A0%9C)" title="성왕 (백제)">성왕</a>(재위 523 ~ 554년)도 일본과의 우호관계를 이어갔다. 성왕 때, 백제는 일본에 불교를 전파했다(538년).<sup class="reference" id="cite_ref-16"><a href="#cite_note-16">[14]</a></sup> 이 불교의 전파를 기점으로 고훈 시대와 <a href="/wiki/%EC%95%84%EC%8A%A4%EC%B9%B4_%EC%8B%9C%EB%8C%80" title="아스카 시대">아스카 시대</a>를 구분짓는다.</p>
<h3><span class="mw-headline" id=".EC.8B.A0.EB.9D.BC.EC.99.80.EC.9D.98_.EA.B4.80.EA.B3.84">신라와의 관계</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EA%B3%A0%ED%9B%88_%EC%8B%9C%EB%8C%80&amp;action=edit&amp;section=6" title="부분 편집: 신라와의 관계">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>왜는 신라의 건국 초부터 시달리게 하였다. 이들은 신라출신이 아닌 가야 또는 백제 등 다른 나라 출신으로서 대부분이 대마도 규슈 북부 또는 가야국에 근거를 두었던 것으로 추정된다.<sup class="reference" id="cite_ref-17"><a href="#cite_note-17">[15]</a></sup> 신라의 내물 마립간 때에는 해안가로 왜구의 침입을 많이 당하였다. <a href="/wiki/364%EB%85%84" title="364년">364년</a> 음력 4월 <a href="/wiki/%EC%99%9C" title="왜">왜</a>가 크게 군사를 일으켜 쳐 왔는데, <a href="/wiki/%ED%86%A0%ED%95%A8%EC%82%B0" title="토함산">토함산</a> 아래에 허수아비 수천 기를 세워 마주하게 하였다. 왜병은 수가 많은 것을 믿고 달려들다가 신라 복병에 걸려 크게 패했다.<sup class="reference" id="cite_ref-18"><a href="#cite_note-18">[16]</a></sup> <a href="/wiki/393%EB%85%84" title="393년">393년</a> 음력 5월 왜인이 크게 쳐와 <a href="/wiki/%EA%B8%88%EC%84%B1" title="금성">금성</a>을 포위하고 닷새가 되도록 풀지 않았다. 장병들이 나가 싸우기를 청하는데 이사금이 거부, 적의 식량이 떨어질 때까지 농성하였다. 적이 퇴각하자 2백 기병으로 퇴로를 막고 보병 1천을 내보내 협공하여 크게 이겼다.<sup class="reference" id="cite_ref-19"><a href="#cite_note-19">[17]</a></sup></p>
