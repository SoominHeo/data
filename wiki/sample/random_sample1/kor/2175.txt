<p><b>골드만 삭스</b>(Goldman Sachs, <a href="/wiki/%EB%89%B4%EC%9A%95_%EC%A6%9D%EA%B6%8C%EA%B1%B0%EB%9E%98%EC%86%8C" title="뉴욕 증권거래소">NYSE</a>: <a class="external text" href="https://www.nyse.com/quote/XNYS:GS" rel="nofollow">GS</a>)는 국제금융시장에서 투자업무와 증권업무, 투자관리, 기타 금융서비스를 기관투자자들에게 주로 서비스를 제공하는 대표적인 <a href="/wiki/%EB%AF%B8%EA%B5%AD" title="미국">미국</a>계 다국적 <a href="/wiki/%ED%88%AC%EC%9E%90%EC%9D%80%ED%96%89" title="투자은행">투자은행</a>이다.</p>
<p>골드만 삭스는 1869년에 설립되었으며 현재 본사는 <a class="mw-redirect" href="/wiki/%EB%A7%A8%ED%95%98%ED%83%84" title="맨하탄">맨하탄</a> 남부의 <a class="mw-redirect" href="/wiki/%EB%89%B4%EC%9A%95_%EC%8B%9C" title="뉴욕 시">뉴욕 시</a> <a class="mw-redirect" href="/wiki/%EC%9B%94%EC%8A%A4%ED%8A%B8%EB%A6%AC%ED%8A%B8" title="월스트리트">월스트리트</a> 200에 위치하고 있다. 이 회사는 인수 합병 컨설팅을 제공하며, 주식발행 주간사역할, 자산관리 그리고 정부, 기업, 개인을 상대로 프라임 브로커리지 역할을 맡고 있다. <a href="/wiki/%EB%8F%85%EC%9D%BC" title="독일">독일</a>계 <a href="/wiki/%EC%9C%A0%EB%8C%80%EC%9D%B8" title="유대인">유대인</a> <a class="new" href="/w/index.php?title=%EB%A7%88%EB%A5%B4%EC%BF%A0%EC%8A%A4_%EA%B3%A8%EB%93%9C%EB%A7%8C&amp;action=edit&amp;redlink=1" title="마르쿠스 골드만 (없는 문서)">마르쿠스 골드만</a>이 뉴욕에 차린 약속어음 거래 회사를 모체로 시작되었으며, 130여년의 전통을 자랑한다. 뉴욕에 본부를 두고 있으며, 24개국의 지사를 통해 기업의 인수합병과 채권발행 등의 사업을 하고 있다. 한국에도 지난 1992년 <a class="mw-redirect" href="/wiki/%EC%84%9C%EC%9A%B8" title="서울">서울</a> 사무소를 개설했고 1998년 12월 지점으로 승격시켰다.</p>
<p>골드만 삭스에서 일하다가 정부기구로 옮긴 전 임직원들로는 각각 <a href="/wiki/%EB%B9%8C_%ED%81%B4%EB%A6%B0%ED%84%B4" title="빌 클린턴">빌 클린턴</a> 대통령과 <a href="/wiki/%EC%A1%B0%EC%A7%80_W._%EB%B6%80%EC%8B%9C" title="조지 W. 부시">조지 W. 부시</a> 대통령 재임 당시 <a href="/wiki/%EB%AF%B8%EA%B5%AD_%EC%9E%AC%EB%AC%B4%EB%B6%80" title="미국 재무부">재무 장관</a>을 역임했던 <a href="/wiki/%EB%A1%9C%EB%B2%84%ED%8A%B8_%EB%A3%A8%EB%B9%88" title="로버트 루빈">로버트 루빈</a>과 <a class="new" href="/w/index.php?title=%ED%97%A8%EB%A6%AC_%ED%8F%B4%EC%8A%A8&amp;action=edit&amp;redlink=1" title="헨리 폴슨 (없는 문서)">헨리 폴슨</a>이 있으며, <a href="/wiki/%EC%9C%A0%EB%9F%BD_%EC%A4%91%EC%95%99%EC%9D%80%ED%96%89" title="유럽 중앙은행">유럽 중앙은행</a>에서 총재를 맡고 있는 <a href="/wiki/%EB%A7%88%EB%A6%AC%EC%98%A4_%EB%93%9C%EB%9D%BC%EA%B8%B0" title="마리오 드라기">마리오 드라기</a>, <a href="/wiki/%EC%BA%90%EB%82%98%EB%8B%A4" title="캐나다">캐나다</a> 중앙은행 총재(2008년 ~ 2013년)와 <a href="/wiki/%EC%98%81%EA%B5%AD" title="영국">영국</a> 중앙은행 총재(2013년 7월부터)를 맡고 있는 <a class="new" href="/w/index.php?title=%EB%A7%88%ED%81%AC_%EC%B9%B4%EB%8B%88&amp;action=edit&amp;redlink=1" title="마크 카니 (없는 문서)">마크 카니</a>가 있다.</p>
<p>2006년부터 변호사 출신 투자금융가 <a class="new" href="/w/index.php?title=%EB%A1%9C%EC%9D%B4%EB%93%9C_%EB%B8%94%EB%9E%AD%ED%81%AC%ED%8C%8C%EC%9D%B8&amp;action=edit&amp;redlink=1" title="로이드 블랭크파인 (없는 문서)">로이드 블랭크파인</a>이 최고경영자 겸 회장으로 골드만 삭스를 이끌어 가고 있다.</p>
