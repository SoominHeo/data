<h1 class="firstHeading" id="firstHeading" lang="ko">ISO 이미지</h1>
<div class="mw-body-content" id="bodyContent">
<div id="siteSub">위키백과, 우리 모두의 백과사전.</div>
<div id="contentSub"><span class="mw-redirectedfrom">(<a class="mw-redirect" href="/w/index.php?title=.iso&amp;redirect=no" title=".iso">.iso</a>에서 넘어옴)</span></div>
<div class="mw-jump" id="jump-to-nav">
					이동:					<a href="#mw-head">둘러보기</a>, 					<a href="#p-search">검색</a>
</div>
<div class="mw-content-ltr" dir="ltr" id="mw-content-text" lang="ko"><script>function mfTempOpenSection(id){var block=document.getElementById("mf-section-"+id);block.className+=" open-block";block.previousSibling.className+=" open-block";}</script><p><b>ISO 이미지</b>(ISO image)는 <a href="/wiki/%EA%B5%AD%EC%A0%9C_%ED%91%9C%EC%A4%80%ED%99%94_%EA%B8%B0%EA%B5%AC" title="국제 표준화 기구">국제 표준화 기구</a>(ISO)가 제정한 <a class="mw-redirect" href="/wiki/%EA%B4%91%ED%95%99_%EB%94%94%EC%8A%A4%ED%81%AC" title="광학 디스크">광학 디스크</a>의 <a href="/wiki/%EC%95%95%EC%B6%95_%ED%8C%8C%EC%9D%BC" title="압축 파일">압축 파일</a>(<a href="/wiki/%EB%94%94%EC%8A%A4%ED%81%AC_%EC%9D%B4%EB%AF%B8%EC%A7%80" title="디스크 이미지">디스크 이미지</a>)이다. 이 형식은 수많은 소프트웨어 업체들이 지원한다. ISO 이미지 파일은 일반적으로 <i>.iso</i>라는 <a href="/wiki/%ED%8C%8C%EC%9D%BC_%ED%99%95%EC%9E%A5%EC%9E%90" title="파일 확장자">파일 확장자</a>를 가진다. <i>ISO</i>라는 이름은 CD-ROM 매체에 쓰이는 <a href="/wiki/ISO_9660" title="ISO 9660">ISO 9660</a> <a href="/wiki/%ED%8C%8C%EC%9D%BC_%EC%8B%9C%EC%8A%A4%ED%85%9C" title="파일 시스템">파일 시스템</a>에서 가져온 것이지만 ISO 이미지는 <a href="/wiki/%EC%9C%A0%EB%8B%88%EB%B2%84%EC%84%A4_%EB%94%94%EC%8A%A4%ED%81%AC_%ED%8F%AC%EB%A7%B7" title="유니버설 디스크 포맷">UDF</a> 파일 시스템도 포함할 수 있다. 이는 ISO 9660에 대한 하위 호환성 덕에 기인한다.</p>
<h2><span class="mw-headline" id=".EC.9E.90.EC.84.B8.ED.95.9C_.EC.A0.95.EB.B3.B4">자세한 정보</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=ISO_%EC%9D%B4%EB%AF%B8%EC%A7%80&amp;action=edit&amp;section=1" title="부분 편집: 자세한 정보">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>다른 압축 파일과 같이, ISO 이미지에는 압축된 <a href="/wiki/%EB%B8%94%EB%A3%A8%EB%A0%88%EC%9D%B4_%EB%94%94%EC%8A%A4%ED%81%AC" title="블루레이 디스크">BD</a>나 <a href="/wiki/%EC%BD%A4%ED%8C%A9%ED%8A%B8_%EB%94%94%EC%8A%A4%ED%81%AC" title="콤팩트 디스크">CD</a>를 비롯한 디스크 포맷에 포함된 모든 <a class="mw-disambig" href="/wiki/%ED%8C%8C%EC%9D%BC" title="파일">파일</a> 데이터를 담고 있다.<sup class="reference" id="cite_ref-1"><a href="#cite_note-1">[1]</a></sup> 이들은 압축되지 않은 형태로 저장된다. 어떠한 CD나 DVD라도 .ISO 형식의 파일로 만들 수 있다. 다시 말해 ISO 이미지는 진정한 원본 디지털 복사본이다.</p>
<p>이러한 특성은 <a href="/wiki/%EC%86%8C%ED%94%84%ED%8A%B8%EC%9B%A8%EC%96%B4" title="소프트웨어">소프트웨어</a>를 배포하기 위해 물리적인 <a class="mw-disambig" href="/wiki/%EB%A7%A4%EC%B2%B4" title="매체">매체</a>를 이용하지 않고도 <a href="/wiki/%EC%9D%B8%ED%84%B0%EB%84%B7" title="인터넷">인터넷</a>이나 <a class="mw-redirect" href="/wiki/%EB%9E%9C" title="랜">랜</a> 연결을 통하여 간단히 전송할 수 있게 만들어 준다.</p>
<p>올바른 ISO 이미지는 절대 표준인 포맷 과정에 따라 다양한 파일들을 압축되지 않은 형태로 하나의 파일로 묶어둔다.</p>
<p>ISO 이미지의 가장 중요한 기능은 미디어 저작 / 디스크 기록 소프트웨어를 이용하여 쉽게 BD, DVD, CD로 표현하거나 구울 수 있다는 것이다.<sup class="reference" id="cite_ref-2"><a href="#cite_note-2">[2]</a></sup> <a href="/wiki/%EC%95%95%EC%B6%95_%EC%86%8C%ED%94%84%ED%8A%B8%EC%9B%A8%EC%96%B4" title="압축 소프트웨어">압축 소프트웨어</a>를 이용하여도 열 수 있다. ISO 기록 기능은 현대에 출시된 가정 및 기업용 컴퓨터 운영 체제에서 기본으로 지원한다.</p>
