<p><b>골재</b>(骨材)는 <a class="mw-redirect" href="/wiki/%ED%95%98%EC%B2%9C" title="하천">하천</a>·<a class="mw-redirect" href="/wiki/%EC%82%B0%EB%A6%BC" title="산림">산림</a>·<a class="new" href="/w/index.php?title=%EA%B3%B5%EC%9C%A0%EC%88%98%EB%A9%B4&amp;action=edit&amp;redlink=1" title="공유수면 (없는 문서)">공유수면</a> 기타 <a class="new" href="/w/index.php?title=%EC%A7%80%EC%83%81&amp;action=edit&amp;redlink=1" title="지상 (없는 문서)">지상</a>·<a href="/wiki/%EC%A7%80%ED%95%98" title="지하">지하</a> 등에 부존되어 있는 <a href="/wiki/%EC%95%94%EC%84%9D" title="암석">암석</a> (쇄석용에 한한다) <a href="/wiki/%EB%AA%A8%EB%9E%98" title="모래">모래</a> 또는 <a href="/wiki/%EC%9E%90%EA%B0%88" title="자갈">자갈</a>로서 <a class="mw-redirect" href="/wiki/%EA%B1%B4%EC%84%A4%EA%B3%B5%EC%82%AC" title="건설공사">건설공사</a>의 기초 <a href="/wiki/%EC%9E%AC%EB%A3%8C" title="재료">재료</a>로 쓰이는 것을 말한다.</p>
<p>골재는 주로 <a href="/wiki/%EC%BD%98%ED%81%AC%EB%A6%AC%ED%8A%B8" title="콘크리트">콘크리트</a>나 <a href="/wiki/%EB%AA%A8%EB%A5%B4%ED%83%80%EB%A5%B4" title="모르타르">모르타르</a>를 만들 때 쓰이며, 입자의 크기, 중량, 생산지에 따라 분류된다.</p>

<h4><span class="mw-headline" id=".EA.B2.BD.EB.9F.89.EA.B3.A8.EC.9E.AC">경량골재</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EA%B3%A8%EC%9E%AC&amp;action=edit&amp;section=5" title="부분 편집: 경량골재">편집</a><span class="mw-editsection-bracket">]</span></span></h4>
<p>경량골재(lightweight aggregate)는 천연 경량골재와 인공 경량골재로 구분되며, 천연 경량골재에는 <a href="/wiki/%EA%B2%BD%EC%84%9D" title="경석">경석</a> 화산자갈, <a href="/wiki/%EC%9D%91%ED%9A%8C%EC%95%94" title="응회암">응회암</a>, <a href="/wiki/%EC%9A%A9%EC%95%94" title="용암">용암</a> 등이 있으며, 인공 경량골재에는 팽창성 <a class="mw-redirect" href="/wiki/%ED%98%88%EC%95%94" title="혈암">혈암</a>, 팽창성 <a href="/wiki/%EC%A0%90%ED%86%A0" title="점토">점토</a>, <a class="new" href="/w/index.php?title=%ED%94%8C%EB%9D%BC%EC%9D%B4_%EC%95%A0%EC%89%AC&amp;action=edit&amp;redlink=1" title="플라이 애쉬 (없는 문서)">플라이 애쉬</a> 등을 주원료로 하여 인공적으로 소성한 인공 경량골재와 팽창 <a class="new" href="/w/index.php?title=%EC%8A%AC%EB%9E%98%EA%B7%B8&amp;action=edit&amp;redlink=1" title="슬래그 (없는 문서)">슬래그</a>, <a href="/wiki/%EC%84%9D%ED%83%84" title="석탄">석탄</a> 찌꺼기 등과 같은 산업 부산물인 경량골재 및 그 가공품이다. 골재의 내부는 다공질이고 표면은 유리질의 피막으로 덮인 구조로 되어 있으며, 잔골재는 <a class="new" href="/w/index.php?title=%EC%A0%88%EA%B1%B4%EB%B0%80%EB%8F%84&amp;action=edit&amp;redlink=1" title="절건밀도 (없는 문서)">절건밀도</a>가 0.0018 g/mm<sup>3</sup>미만, 굵은 골재는 절건밀도가 0.0015 g/mm<sup>3</sup>미만인 것이다.<sup class="reference" id="cite_ref-cs_1-1"><a href="#cite_note-cs-1">[1]</a></sup></p>
<dl>
<dd>- <a href="/wiki/%EA%B2%BD%EC%84%9D" title="경석">경석</a>, <a class="new" href="/w/index.php?title=%EC%A7%84%EC%A3%BC%EC%95%94&amp;action=edit&amp;redlink=1" title="진주암 (없는 문서)">진주암</a>(<a class="mw-disambig" href="/wiki/%ED%8E%84%EB%9D%BC%EC%9D%B4%ED%8A%B8" title="펄라이트">펄라이트</a>) 등 다공성 돌로 구성.</dd>
<dd>- 콘크리트의 중량을 감소시킬 목적으로 쓰이는 가벼운 골재</dd>
<dd>- <a class="mw-redirect" href="/wiki/%EC%A4%91%EB%9F%89" title="중량">중량</a>: 1800 <a class="new" href="/w/index.php?title=Kg/%E3%8E%A5&amp;action=edit&amp;redlink=1" title="Kg/㎥ (없는 문서)">kg/㎥</a>, 120 lb/ft<sup>3</sup> (비중 2.50이하)</dd>
</dl>
<h3><span class="mw-headline" id=".EC.83.9D.EC.82.B0.EC.97.90_.EB.94.B0.EB.A5.B8_.EB.B6.84.EB.A5.98">생산에 따른 분류</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EA%B3%A8%EC%9E%AC&amp;action=edit&amp;section=6" title="부분 편집: 생산에 따른 분류">편집</a><span class="mw-editsection-bracket">]</span></span></h3>

<h2><span class="mw-headline" id=".EA.B3.A8.EC.9E.AC.EC.9D.98_.EC.9D.BC.EB.B0.98.EC.A0.81_.EC.84.B1.EC.A7.88">골재의 일반적 성질</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EA%B3%A8%EC%9E%AC&amp;action=edit&amp;section=7" title="부분 편집: 골재의 일반적 성질">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<h3><span class="mw-headline" id=".EC.9E.85.EB.8F.84">입도</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EA%B3%A8%EC%9E%AC&amp;action=edit&amp;section=8" title="부분 편집: 입도">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>골재의 크고 작은 입자가 혼합된 정도를 골재의 입도(粒度, grading)라 한다. 골재 입자의 크기는 KS F 2502의 골재 체가름 시험(sieve analysis)에 의하여 구하며, 시험 결과를 정리한 것이 입도곡선이다. 입도곡선을 수치화한 것이 조립률(fineness modulus, FM)이다.<sup class="reference" id="cite_ref-2"><a href="#cite_note-2">[2]</a></sup></p>
