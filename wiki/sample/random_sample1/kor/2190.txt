곱창전골</div>

<p><b>곱창전골</b>은 <a href="/wiki/%EC%A0%84%EA%B3%A8" title="전골">전골</a>류의 한국 요리로, <a href="/wiki/%EC%86%8C" title="소">소</a>나 <a href="/wiki/%EB%8F%BC%EC%A7%80" title="돼지">돼지</a>의 <a class="mw-redirect" href="/wiki/%EB%82%B4%EC%9E%A5" title="내장">내장</a>과 여러가지 <a href="/wiki/%EC%B1%84%EC%86%8C" title="채소">채소</a>를 <a href="/wiki/%EC%9C%A1%EC%88%98" title="육수">육수</a>와 함께 끓여낸 음식이다. <a href="/wiki/%EA%B3%B1%EC%B0%BD" title="곱창">곱창</a>이란 소나 돼지의 <a class="mw-redirect" href="/wiki/%EC%9E%91%EC%9D%80_%EC%B0%BD%EC%9E%90" title="작은 창자">작은 창자</a>를 의미한다. 곱창전골은 곱창이 주재료이지만, 다른 부위의 내장도 많이 사용되어 내장 특유의 쫄깃한 식감으로 곱창전골의 맛을 더욱 풍부하게 한다.</p>
<p>곱창에서 나는 누린내와 기름기를 제거하기 위하여, <a href="/wiki/%EB%B0%80%EA%B0%80%EB%A3%A8" title="밀가루">밀가루</a>와 <a href="/wiki/%EC%86%8C%EA%B8%88" title="소금">소금</a>으로 여러차례 문지르고 씻어내어 곱창 안에 붙어있는 하얀 지방층을 꼼꼼하게 제거해야한다. 곱창전골을 만들기 위해서는 숙련된 조리기술과 노동력뿐만 아니라 깨끗하고 신선하게 재료를 다루는 능력이 필요하기 때문에, 곱창전골은 대부분 곱창 요리만을 전문으로 하는 음식점에서 판매한다. 곱창전골은 대개 <a class="mw-redirect" href="/wiki/%EA%B3%A0%EC%B6%A7%EA%B0%80%EB%A3%A8" title="고춧가루">고춧가루</a>가 많이 들어가 맵고 얼큰한 맛이 나기 때문에 식사뿐만 아니라 술 안주로도 많이 먹는다.</p>
<h2><span class="mw-headline" id=".EC.9E.AC.EB.A3.8C">재료</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EA%B3%B1%EC%B0%BD%EC%A0%84%EA%B3%A8&amp;action=edit&amp;section=1" title="부분 편집: 재료">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>소 내장의 종류는 <a class="new" href="/w/index.php?title=%EC%96%91_(%EB%82%B4%EC%9E%A5)&amp;action=edit&amp;redlink=1" title="양 (내장) (없는 문서)">양</a>, <a class="new" href="/w/index.php?title=%EB%B2%8C%EC%A7%91%EC%9C%84&amp;action=edit&amp;redlink=1" title="벌집위 (없는 문서)">벌집위</a>, <a class="mw-redirect" href="/wiki/%EC%B2%9C%EC%97%BD" title="천엽">천엽</a>, <a class="mw-redirect" href="/wiki/%EB%A7%89%EC%B0%BD" title="막창">막창</a>, 곱창, <a class="new" href="/w/index.php?title=%EB%8C%80%EC%B0%BD_(%EB%82%B4%EC%9E%A5)&amp;action=edit&amp;redlink=1" title="대창 (내장) (없는 문서)">대창</a>이 있다. 소 첫 번째 위를 '양'이라고 하며, <a href="/wiki/%EB%8B%A8%EB%B0%B1%EC%A7%88" title="단백질">단백질</a>이 매우 풍부하기 때문에, 과거에는 몸이 약한 사람에게 양으로 육수를 만들어 먹을 것을 추천하기도 했다. 양의 겉부분은 매우 질기고 거칠며 검은 돌기가 많아, 사이사이에 불결한 물질이 있을 수 있으므로 요리에 사용하기 전에 완전하게 긁어내야 한다.</p>
<p>곱창은 작은창자 또는 소창이라고 부르기보단 대개 그 구불구불한 모양 때문에 곱창이라 부른다(큰 창자는 대창이라 부른다). 곱창은 다른 고기에 비해서 <a class="mw-redirect" href="/wiki/%EC%B2%A0%EB%B6%84" title="철분">철분</a>과 <a href="/wiki/%EB%B9%84%ED%83%80%EB%AF%BC" title="비타민">비타민</a>이 많다. 또한 저렴한 가격에 독특한 맛과 향, 그리고 쫄깃쫄깃한 식감 때문에 구이나 볶음 요리로도 많이 사용한다.</p>
