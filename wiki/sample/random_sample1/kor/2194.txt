<p><b>공공재</b>(公共財)는 어떠한 경제주체에 의해서 생산이 이루어지면 구성원 모두가 소비혜택을 누릴 수 있는 <a href="/wiki/%EC%9E%AC%ED%99%94" title="재화">재화</a> 또는 <a href="/wiki/%EC%84%9C%EB%B9%84%EC%8A%A4" title="서비스">서비스</a>를 말한다. 경제학적으로 봤을때 <a class="new" href="/w/index.php?title=%EB%B9%84%EA%B2%BD%ED%95%A9%EC%84%B1&amp;action=edit&amp;redlink=1" title="비경합성 (없는 문서)">비경합성</a>과 <a href="/wiki/%EB%B9%84%EB%B0%B0%EC%A0%9C%EC%84%B1" title="비배제성">비배제성</a>을 그 특징으로 한다.</p>

<p>그 성격으로 인해 공공재의 공급 주체는 <a href="/wiki/%EA%B5%AD%EA%B0%80" title="국가">국가</a>나 지방자치단체 등의 <a class="mw-redirect" href="/wiki/%EA%B3%B5%EA%B3%B5_%EA%B8%B0%EA%B4%80" title="공공 기관">공공 기관</a>이나 <a href="/wiki/%EA%B3%B5%EA%B8%B0%EC%97%85" title="공기업">공기업</a>이 대부분이다.</p>
<h2><span class="mw-headline" id=".EC.A0.95.EB.B6.80.EA.B8.B0.EB.8A.A5.EC.9D.B4_.EC.B6.95.EC.86.8C.EB.90.98.EC.97.88.EB.8B.A4.EB.8A.94_.EC.9E.85.EC.9E.A5.28.EA.B3.BC.EC.86.8C.EA.B3.B5.EA.B8.89.EC.84.A4.29">정부기능이 축소되었다는 입장(과소공급설)</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EA%B3%B5%EA%B3%B5%EC%9E%AC&amp;action=edit&amp;section=1" title="부분 편집: 정부기능이 축소되었다는 입장(과소공급설)">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<ul>
<li><a class="new" href="/w/index.php?title=Galbraith&amp;action=edit&amp;redlink=1" title="Galbraith (없는 문서)">Galbraith</a>의 <a href="/wiki/%EC%9D%98%EC%A1%B4%ED%9A%A8%EA%B3%BC" title="의존효과">의존효과</a>(dependence effect) : 갈브레이드는 민간의 사적재는 각종 선전에 의해 소비자들의 욕구를 촉발하는데 비해 공공재에 대해서는 선전이 이루어지지 않아 공적 욕구를 자극시키지 못한다고 주장한다. 그는 [불확실성의 시대]에서 관료제의 역할 축소를, [풍요로운 사회]에서 선전에 의하여 소비가 자극되는 의존효과를 제창하였다.</li>
</ul>
<ul>
<li><a class="new" href="/w/index.php?title=Duesenberry&amp;action=edit&amp;redlink=1" title="Duesenberry (없는 문서)">Duesenberry</a>의 <a class="mw-redirect" href="/wiki/%EC%A0%84%EC%8B%9C%ED%9A%A8%EA%B3%BC" title="전시효과">전시효과</a>(demonstration effect) : 민간재는 주의를 의식한 체면유지 때문에 실제 필요한 지출보다 더 많은 지출을 한다. 민간은 광고나 판매전략에 의한 과시효과로 소비가 자극되지만 정부는 그렇지 않아 소비가 민간재로 쏠린다.</li>
</ul>
<ul>
<li><a class="new" href="/w/index.php?title=Musgrave&amp;action=edit&amp;redlink=1" title="Musgrave (없는 문서)">Musgrave</a>의 <a class="new" href="/w/index.php?title=%EC%A1%B0%EC%84%B8%EC%A0%80%ED%95%AD&amp;action=edit&amp;redlink=1" title="조세저항 (없는 문서)">조세저항</a> : 사적재의 경우 부담한만큼 소비한다. 따라서 자신의 부담이 직접 자신의 편익으로 돌아온다. 그러나 공공재의 경우에는 자신이 부담(조세)한 것에 비해 적게 편익(소비)을 누린다고 생각(재정착각:fiscal illusion)하게 되고, 재정환상으로 인한 조세저항은 공공재의 과소공급을 유도한다. 이러한 조세저항으로 인한 적정 공공재 공급의 실패를 <a class="mw-redirect" href="/wiki/%EC%8B%9C%EC%9E%A5%EC%8B%A4%ED%8C%A8" title="시장실패">시장실패</a>(market failure)나 <a href="/wiki/%EC%A0%95%EB%B6%80%EC%8B%A4%ED%8C%A8" title="정부실패">정부실패</a>(government failure)에 대비하여 <a class="new" href="/w/index.php?title=%EC%8B%9C%EB%AF%BC%EC%8B%A4%ED%8C%A8&amp;action=edit&amp;redlink=1" title="시민실패 (없는 문서)">시민실패</a>(citizen's failure)라고도 한다.</li>
</ul>
<ul>
<li><a class="new" href="/w/index.php?title=Downs&amp;action=edit&amp;redlink=1" title="Downs (없는 문서)">Downs</a>의 <a class="new" href="/w/index.php?title=%ED%95%A9%EB%A6%AC%EC%A0%81%EB%AC%B4%EC%A7%80&amp;action=edit&amp;redlink=1" title="합리적무지 (없는 문서)">합리적무지</a>(rational ignorance) : 합리적 개인은 사적 이익을 추구하며, 정보 수집에 따른 비용과 이에 따른 편익을 고려하여 정보수집 여부를 판단하게 된다. 따라서 개인들은 합리적무지의 상태에 있고, 이들은 공공서비스의 공급에 대해 정확하게 평가하지 못하고, 이의 확대에 대해 저항하게 된다.</li>
</ul>
<h2><span class="mw-headline" id=".EC.A0.95.EB.B6.80.EA.B8.B0.EB.8A.A5.EC.9D.B4_.ED.8C.BD.EC.B0.BD.EB.90.98.EC.97.88.EB.8B.A4.EB.8A.94_.EC.9E.85.EC.9E.A5.28.EA.B3.BC.EB.8B.A4.EA.B3.B5.EA.B8.89.EC.84.A4.29">정부기능이 팽창되었다는 입장(과다공급설)</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EA%B3%B5%EA%B3%B5%EC%9E%AC&amp;action=edit&amp;section=2" title="부분 편집: 정부기능이 팽창되었다는 입장(과다공급설)">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<ul>
<li><a class="new" href="/w/index.php?title=A.Wagner&amp;action=edit&amp;redlink=1" title="A.Wagner (없는 문서)">A.Wagner</a>의 <a class="new" href="/w/index.php?title=%EA%B2%BD%EB%B9%84%ED%8C%BD%EC%B0%BD%EC%9D%98_%EB%B2%95%EC%B9%99&amp;action=edit&amp;redlink=1" title="경비팽창의 법칙 (없는 문서)">경비팽창의 법칙</a> : Wagner는 인구성장 및 기술진보 등으로 인해 정부의 기능이 외연적으로 팽창됨에 따라 재정규모가 팽창된다고 보았다. 즉, 공공재의 수요는 소득탄력적이기 때문에 도시화의 진전과 국민소득의 증대, 사회의 상호의존관계 심화가 정부성장요인이 되었다는 것이다.</li>
</ul>
<ul>
<li><a class="new" href="/w/index.php?title=Peacock&amp;action=edit&amp;redlink=1" title="Peacock (없는 문서)">Peacock</a> &amp; <a class="new" href="/w/index.php?title=Wiseman&amp;action=edit&amp;redlink=1" title="Wiseman (없는 문서)">Wiseman</a>의 전위효과 및 대체효과
<ul>
<li><a class="new" href="/w/index.php?title=%EC%A0%84%EC%9C%84%ED%9A%A8%EA%B3%BC&amp;action=edit&amp;redlink=1" title="전위효과 (없는 문서)">전위효과</a> : 전쟁 등의 위기시에 국민의 조세부담증대에 대한 허용수준(tolerance)이 높아진다. 문지방효과(thread-hold effect)를 말한다. 위기시에는 공적 지출이 사적(민간) 지출을 대신하게 된다는 논리이다.</li>
<li><a class="mw-redirect" href="/wiki/%EB%8C%80%EC%B2%B4%ED%9A%A8%EA%B3%BC" title="대체효과">대체효과</a>(단속효과) : 위기시에 한번 늘어난 재정수준은 경제가 정상으로 회복되어 지출요인이 사라진 뒤에도 단속적 효과(ratchet effect, <a class="new" href="/w/index.php?title=%ED%86%B1%EB%8B%88%ED%9A%A8%EA%B3%BC&amp;action=edit&amp;redlink=1" title="톱니효과 (없는 문서)">톱니효과</a>)가 발생하여 잉여재원이 다른 새로운 사업계획을 추진하는데 이용됨으로써 원상태로 돌아오지 않는다.</li>
</ul>
</li>
</ul>
<ul>
<li><a class="new" href="/w/index.php?title=%EB%B3%B4%EB%AA%B0%EB%B3%91&amp;action=edit&amp;redlink=1" title="보몰병 (없는 문서)">보몰병</a>(Baumol's Disease) : 보몰효과라고도 하며 이는 정부부문이 노동집약적인 성격을 띠고 있기 때문에 민간부문에 비해 생산성 증가가 더디며, 이로 인해 비용절감이 힘들고 생산비용이 빨리 증가하여 정부지출의 규모가 점차 커질 수밖에 없다는 것이다. 이로 인해 GDP에서 차지하는 정부의 점유율이 민간보다 높아진다. 그리고 공공재에 대한 수요는 가격비탄력적, 즉 필수재인 경우가 많기 때문에 가격의 변동에 대해 둔감하다. 이로 인해 정부는 팽창할 수밖에 없다는 것이다.</li>
</ul>
<ul>
<li><a class="new" href="/w/index.php?title=Niskanen&amp;action=edit&amp;redlink=1" title="Niskanen (없는 문서)">Niskanen</a>의 <a class="new" href="/w/index.php?title=%EC%98%88%EC%82%B0%EA%B7%B9%EB%8C%80%ED%99%94%EB%AA%A8%ED%98%95&amp;action=edit&amp;redlink=1" title="예산극대화모형 (없는 문서)">예산극대화모형</a> : Niskanen은 관료가 예산심의를 받을 때 국회만 잘 설득하면 예산을 많이 확보할 수 있고, 국민에게 돌려주지 않아도 되므로 정치인의 잉여를 독점하여 적정예산규모를 초과하는 과다지출이 된다는 것이다.</li>
</ul>
<ul>
<li><a class="new" href="/w/index.php?title=Buchanan&amp;action=edit&amp;redlink=1" title="Buchanan (없는 문서)">Buchanan</a>의 다수결투표와 <a class="new" href="/w/index.php?title=%EB%A6%AC%EB%B0%94%EC%9D%B4%EC%96%B4%EB%8D%98%EA%B0%80%EC%84%A4&amp;action=edit&amp;redlink=1" title="리바이어던가설 (없는 문서)">리바이어던가설</a> : 대의민주주의하에서 다수결투표는 투표의 거래(vote trading), 즉 log-rolling에 의해 과다지출을 초래한다. 개별 사업 하나하나를 보면 바람직하지 못하지만, 표의 교환행위(담합)를 통해 서로 불필요한 사업을 끼워서 거래함으로써 정부사업이 팽창하는 것이다.</li>
</ul>
<ul>
<li>Expending Belt(지출한도)의 부재 : 정부의 지출에는 팽창을 통제할 수 있는 가시적인 길항력(countervailing force)이 없다. 정부의 프로그램은 경직성이 강하여 한번 지출이 이루어지고 나면 좀처럼 없어지지 않는 자생력을 가지고 있다.</li>
</ul>
<ul>
<li>양출계입의 원리 : 근대재정의 원리는 가계나 기업의 양입제출과 달리 양출계입의 원리(지출규모에 맞추어 세입을 계획)에 입각하기 때문에 지출의 수요가 있으면 수입을 확대한다.</li>
</ul>
<ul>
<li>할거적 예산결정구조 : 예산을 합리적·총체적으로 결정하지 못하고, 부문별로 결정하고 조정한다. 이러한 편린적·할거주의적(fragmentation) 과정을 거치기 때문에 예산은 부풀려진다. 중앙집권적인 국가보다 연방제와 같이 분권화된 국가의 재정규모가 더 팽창하는 것도 이와 같은 원리이다.</li>
</ul>
<ul>
<li>간접세 위주의 국가재정구조 : 간접세의 경우 납세의무자와 실질적인 담세자가 다르므로 조세저항이 회피되어 재정팽창이 가능하게 된다. 이러한 현상을 <a class="new" href="/w/index.php?title=%EC%9E%AC%EC%A0%95%ED%99%98%EC%83%81&amp;action=edit&amp;redlink=1" title="재정환상 (없는 문서)">재정환상</a>(fiscal illusion)이라고 한다.</li>
</ul>
<ul>
<li>국제적 시장에 대한 경제의 개방 : 무역의존도가 높은 개방형 무역구조하에서는 국가재정이 팽창된다.</li>
</ul>
<ul>
<li>기타
<ul>
<li>경쟁이 치열한 소선거구제</li>
<li>관료제국주의(<a class="new" href="/w/index.php?title=%ED%8C%8C%ED%82%A8%EC%8A%A8%EC%9D%98_%EB%B2%95%EC%B9%99&amp;action=edit&amp;redlink=1" title="파킨슨의 법칙 (없는 문서)">파킨슨의 법칙</a>)</li>
<li><a href="/wiki/%EB%8B%A4%EC%9B%90%EC%A3%BC%EC%9D%98" title="다원주의">다원주의</a>의 사회구조</li>
<li>관료제의 불멸성</li>
<li>과학기술에 대한 투자증대</li>
<li>사회복지제도의 지속적 확산 등이 정부팽창을 초래하였다.</li>
</ul>
</li>
</ul>
