<p><b>공손공</b>(公孫恭, ? ~ ?)은 <a href="/wiki/%EC%A4%91%EA%B5%AD" title="중국">중국</a> <a href="/wiki/%EC%82%BC%EA%B5%AD_%EC%8B%9C%EB%8C%80_(%EC%A4%91%EA%B5%AD)" title="삼국 시대 (중국)">삼국 시대</a> <a href="/wiki/%EC%9C%84_(%EC%82%BC%EA%B5%AD)" title="위 (삼국)">위나라</a>에 종속한 <a class="mw-redirect" href="/wiki/%EC%9A%94%EB%8F%99" title="요동">요동</a>의 세력가이다. <a class="mw-disambig" href="/wiki/%EA%B3%A0%EC%9E%90" title="고자">고자</a>였으며 <a href="/wiki/%EC%A1%B0%EC%B9%B4" title="조카">조카</a> <a href="/wiki/%EA%B3%B5%EC%86%90%EC%97%B0" title="공손연">공손연</a>에게 쫓겨났다.</p>

<h2><span class="mw-headline" id=".EC.83.9D.EC.95.A0">생애</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EA%B3%B5%EC%86%90%EA%B3%B5&amp;action=edit&amp;section=1" title="부분 편집: 생애">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p><a class="mw-disambig" href="/wiki/%EA%B3%A0%EC%9E%90" title="고자">고자</a>였다. <a href="/wiki/204%EB%85%84" title="204년">204년</a>(<a href="/wiki/%EA%B1%B4%EC%95%88" title="건안">건안</a> 9년) <a class="mw-redirect" href="/wiki/%ED%98%95" title="형">형</a>인 <a href="/wiki/%EA%B3%B5%EC%86%90%EA%B0%95" title="공손강">공손강</a>이 <a href="/wiki/%EC%95%84%EB%B2%84%EC%A7%80" title="아버지">아버지</a> <a href="/wiki/%EA%B3%B5%EC%86%90%ED%83%81" title="공손탁">공손탁</a>을 이으면서 공손공은 영녕향후(永寧鄕侯)에 봉해졌다. 공손강 사후 그 <a href="/wiki/%EC%95%84%EB%93%A4" title="아들">아들</a>들이 어려 대신 <a href="/wiki/%EC%9A%94%EB%8F%99%EA%B5%B0" title="요동군">요동</a><a href="/wiki/%ED%83%9C%EC%88%98" title="태수">태수</a>를 이었다. <a href="/wiki/221%EB%85%84" title="221년">221년</a>(<a href="/wiki/%ED%99%A9%EC%B4%88_(%EC%9C%84)" title="황초 (위)">황초</a> 2년) <a href="/wiki/%EC%9C%84_(%EC%82%BC%EA%B5%AD)" title="위 (삼국)">위나라</a> <a href="/wiki/%ED%99%A9%EC%A0%9C" title="황제">황제</a> <a href="/wiki/%EC%A1%B0%EB%B9%84" title="조비">조비</a>로부터 <a href="/wiki/%EA%B1%B0%EA%B8%B0%EC%9E%A5%EA%B5%B0" title="거기장군">거기장군</a>을 받았다.<sup class="reference" id="cite_ref-1"><a href="#cite_note-1">[1]</a></sup> <a href="/wiki/228%EB%85%84" title="228년">228년</a>(<a href="/wiki/%ED%83%9C%ED%99%94_(%EC%9C%84)" title="태화 (위)">태화</a> 2년) 공손강의 차남 <a href="/wiki/%EA%B3%B5%EC%86%90%EC%97%B0" title="공손연">공손연</a>에게 태수직을 빼앗기고 유폐당했다. <a href="/wiki/238%EB%85%84" title="238년">238년</a>(<a href="/wiki/%EA%B2%BD%EC%B4%88" title="경초">경초</a> 2년) 위나라의 <a href="/wiki/%EC%82%AC%EB%A7%88%EC%9D%98" title="사마의">사마의</a>가 공손연을 토벌하면서 구출되었다.<sup class="reference" id="cite_ref-2"><a href="#cite_note-2">[2]</a></sup> 이후의 삶은 알 수 없다.</p>
<h2><span class="mw-headline" id=".EC.82.BC.EA.B5.AD.EC.A7.80.EC.97.B0.EC.9D.98">삼국지연의</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EA%B3%B5%EC%86%90%EA%B3%B5&amp;action=edit&amp;section=2" title="부분 편집: 삼국지연의">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>사서가 아닌 <a href="/wiki/%EC%86%8C%EC%84%A4" title="소설">소설</a> 《<a href="/wiki/%EC%82%BC%EA%B5%AD%EC%A7%80%EC%97%B0%EC%9D%98" title="삼국지연의">삼국지연의</a>》에서는 <a href="/wiki/%EC%A1%B0%EC%A1%B0" title="조조">조조</a>한테 <a href="/wiki/%EB%B0%B1%EB%9E%91%EC%82%B0_%EC%A0%84%ED%88%AC" title="백랑산 전투">완전히 패해</a> 공손강에게 의탁하러 온 <a href="/wiki/%EC%9B%90%ED%9D%AC_(%ED%9B%84%ED%95%9C)" title="원희 (후한)">원희</a>, <a href="/wiki/%EC%9B%90%EC%83%81_(%ED%9B%84%ED%95%9C)" title="원상 (후한)">원상</a> <a href="/wiki/%ED%98%95%EC%A0%9C" title="형제">형제</a>가 장차 공손강을 도모하려는 속셈을 품었다는 것을 눈치채고 ‘<a href="/wiki/%EA%B9%8C%EC%B9%98" title="까치">까치</a><a href="/wiki/%EC%A7%91" title="집">집</a>을 빼앗으려는 <a href="/wiki/%EB%B9%84%EB%91%98%EA%B8%B0" title="비둘기">비둘기</a>와 같다’<sup class="reference" id="cite_ref-3"><a href="#cite_note-3">[3]</a></sup>며 먼저 죽이자고 한다. <a href="/wiki/%EC%A0%95%EC%82%AC_(%EC%97%AD%EC%82%AC%ED%95%99)" title="정사 (역사학)">정사</a>대로 공손강의 직위를 이었다가 공손연에게 태수직을 빼앗긴다.</p>
