<p><b>공순황귀비 유호록씨</b>(恭順皇貴妃 钮祜綠氏, <a href="/wiki/1787%EB%85%84" title="1787년">1787년</a> ~ <a href="/wiki/1860%EB%85%84" title="1860년">1860년</a> <a href="/wiki/4%EC%9B%94_23%EC%9D%BC" title="4월 23일">4월 23일</a>)는 <a href="/wiki/%EC%A4%91%EA%B5%AD" title="중국">중국</a> <a href="/wiki/%EC%B2%AD%EB%82%98%EB%9D%BC" title="청나라">청나라</a> 제7대 <a href="/wiki/%ED%99%A9%EC%A0%9C" title="황제">황제</a> <a href="/wiki/%EA%B0%80%EA%B2%BD%EC%A0%9C" title="가경제">가경제</a>(嘉慶帝)의 <a href="/wiki/%ED%9B%84%EA%B6%81" title="후궁">후궁</a>(後宮)이다.</p>
<h2><span class="mw-headline" id=".EC.83.9D.EC.95.A0">생애</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EA%B3%B5%EC%88%9C%ED%99%A9%EA%B7%80%EB%B9%84&amp;action=edit&amp;section=1" title="부분 편집: 생애">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p><a href="/wiki/1804%EB%85%84" title="1804년">1804년</a> <a href="/wiki/%EA%B0%80%EA%B2%BD%EC%A0%9C" title="가경제">가경제</a>(嘉慶帝)의 <a href="/wiki/%ED%9B%84%EA%B6%81" title="후궁">후궁</a>으로 간택되었고 <a href="/wiki/%EA%B0%80%EA%B2%BD%EC%A0%9C" title="가경제">가경제</a> 사후에 그의 적자(嫡子) <a href="/wiki/%EB%8F%84%EA%B4%91%EC%A0%9C" title="도광제">도광제</a>(道光帝)가 즉위한 것을 목도하였으며 <a href="/wiki/%EB%8F%84%EA%B4%91%EC%A0%9C" title="도광제">도광제</a> 사후 그의 <a class="mw-redirect" href="/wiki/%EC%84%9C%EC%9E%90" title="서자">서자</a>(庶子) <a href="/wiki/%ED%95%A8%ED%92%8D%EC%A0%9C" title="함풍제">함풍제</a>(咸豊帝)가 즉위한 이후 <a href="/wiki/1860%EB%85%84" title="1860년">1860년</a> <a href="/wiki/4%EC%9B%94_23%EC%9D%BC" title="4월 23일">4월 23일</a>에 향년 73세로 훙서할 때까지 <a href="/wiki/%EC%B2%AD%EB%82%98%EB%9D%BC" title="청나라">청나라</a> <a class="mw-redirect" href="/wiki/%ED%99%A9%EC%8B%A4" title="황실">황실</a> <a class="mw-redirect" href="/wiki/%EC%97%AC%EC%9E%90" title="여자">여인</a>의 원로로서 예우되었다.</p>
