<p><b>공오균</b>(孔五均, <a href="/wiki/1974%EB%85%84" title="1974년">1974년</a> <a href="/wiki/9%EC%9B%94_10%EC%9D%BC" title="9월 10일">9월 10일</a> ~ )은 <a href="/wiki/%EB%8C%80%ED%95%9C%EB%AF%BC%EA%B5%AD" title="대한민국">대한민국</a>의 은퇴한 <a href="/wiki/%EC%B6%95%EA%B5%AC" title="축구">축구</a> 선수이며 현재는 지도자 생활을 하고있다. <a href="/wiki/%EC%98%A4%EC%8A%A4%ED%8A%B8%EB%A0%88%EC%9D%BC%EB%A6%AC%EC%95%84" title="오스트레일리아">오스트레일리아</a>의 <a href="/wiki/%EC%8B%A0%ED%83%9C%EC%9A%A9" title="신태용">신태용</a> 감독의 축구 아카데미인 TY 스포츠 아카데미의 감독으로 재직했으며 그후 (2015년)부터 중국 <a class="new" href="/w/index.php?title=Shen_Zhen_FC&amp;action=edit&amp;redlink=1" title="Shen Zhen FC (없는 문서)">Shen Zhen FC</a> 에서 코치로 지도자 생활을 이어갔다. 현재 <a href="/wiki/%EB%8C%80%ED%95%9C%EB%AF%BC%EA%B5%AD_U-20_%EC%B6%95%EA%B5%AC_%EA%B5%AD%EA%B0%80%EB%8C%80%ED%91%9C%ED%8C%80" title="대한민국 U-20 축구 국가대표팀">대한민국 U-20 축구 국가대표팀</a> 코치이다.</p>
<h2><span class="mw-headline" id=".EC.B6.95.EA.B5.AC.EC.9D.B8_.EA.B2.BD.EB.A0.A5">축구인 경력</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EA%B3%B5%EC%98%A4%EA%B7%A0&amp;action=edit&amp;section=1" title="부분 편집: 축구인 경력">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<h3><span class="mw-headline" id=".EC.84.A0.EC.88.98_.EA.B2.BD.EB.A0.A5">선수 경력</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EA%B3%B5%EC%98%A4%EA%B7%A0&amp;action=edit&amp;section=2" title="부분 편집: 선수 경력">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p><a href="/wiki/1997%EB%85%84" title="1997년">1997년</a> <a href="/wiki/%EB%8C%80%EC%A0%84_%EC%8B%9C%ED%8B%B0%EC%A6%8C" title="대전 시티즌">대전 시티즌</a>에 입단하여 10시즌을 소화하며 컵대회를 통틀어 291경기에 나서 38골을 기록하였다. 전방 <a href="/wiki/%EA%B3%B5%EA%B2%A9%EC%88%98#.EC.8A.A4.ED.8A.B8.EB.9D.BC.EC.9D.B4.EC.BB.A4" title="공격수">스트라이커</a>부터 <a href="/wiki/%EB%AF%B8%EB%93%9C%ED%95%84%EB%8D%94#.EC.9C.99.EC.96.B4" title="미드필더">윙어</a>, <a href="/wiki/%EB%AF%B8%EB%93%9C%ED%95%84%EB%8D%94#.EC.A4.91.EC.95.99_.EB.AF.B8.EB.93.9C.ED.95.84.EB.8D.94" title="미드필더">공격형 미드필더</a> 등 다양한 포지션에서 활약하였다. <a href="/wiki/2006%EB%85%84" title="2006년">2006년</a> 오랜 기간 활약한 대전으로부터 방출 통보를 받았으나 <a href="/wiki/%EA%B2%BD%EB%82%A8_FC" title="경남 FC">경남 FC</a>에 입단하며 다시 <a href="/wiki/K%EB%A6%AC%EA%B7%B8" title="K리그">K리그</a>로 복귀하였다. <a href="/wiki/2008%EB%85%84" title="2008년">2008년</a> <a href="/wiki/5%EC%9B%94_14%EC%9D%BC" title="5월 14일">5월 14일</a> <a href="/wiki/%EB%B6%80%EC%82%B0_%EC%95%84%EC%9D%B4%ED%8C%8C%ED%81%AC" title="부산 아이파크">부산</a>과의 경기에서 득점을 올려 경남의 팀 통산 100번째 골을 기록한 주인공이 되었다.</p>
<p>소속 팀으로부터 은퇴 제의를 받았으나 선수 생활을 이어나가겠다는 의지를 밝힌 후 <a href="/wiki/%EC%98%A4%EC%8A%A4%ED%8A%B8%EB%A0%88%EC%9D%BC%EB%A6%AC%EC%95%84" title="오스트레일리아">오스트레일리아</a>의 <a class="new" href="/w/index.php?title=%EC%84%A0%EC%83%A4%EC%9D%B8_%EC%BD%94%EC%8A%A4%ED%8A%B8_FC&amp;action=edit&amp;redlink=1" title="선샤인 코스트 FC (없는 문서)">선샤인 코스트</a>로 이적하여 활약하다 선수 생활에 마침표를 찍었다.</p>
