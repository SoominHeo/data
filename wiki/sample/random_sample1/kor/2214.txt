<h1 class="firstHeading" id="firstHeading" lang="ko">공통 중간 언어</h1>
<div class="mw-body-content" id="bodyContent">
<div id="siteSub">위키백과, 우리 모두의 백과사전.</div>
<div id="contentSub"></div>
<div class="mw-jump" id="jump-to-nav">
					이동:					<a href="#mw-head">둘러보기</a>, 					<a href="#p-search">검색</a>
</div>
<div class="mw-content-ltr" dir="ltr" id="mw-content-text" lang="ko"><script>function mfTempOpenSection(id){var block=document.getElementById("mf-section-"+id);block.className+=" open-block";block.previousSibling.className+=" open-block";}</script><p><b>공통 중간 언어</b>(Common Intermediate Language, CIL)는 <a href="/wiki/%EA%B3%B5%ED%86%B5_%EC%96%B8%EC%96%B4_%EA%B8%B0%EB%B0%98" title="공통 언어 기반">공통 언어 기반</a>과 <a href="/wiki/%EB%8B%B7%EB%84%B7_%ED%94%84%EB%A0%88%EC%9E%84%EC%9B%8C%ED%81%AC" title="닷넷 프레임워크">닷넷 프레임워크</a>에서 인간이 이해할 수 있는 가장 낮은 수준의 <a href="/wiki/%ED%94%84%EB%A1%9C%EA%B7%B8%EB%9E%98%EB%B0%8D_%EC%96%B8%EC%96%B4" title="프로그래밍 언어">프로그래밍 언어</a>이다. 닷넷 프레임워크를 대상으로 하는 언어들<sup class="reference" id="cite_ref-1"><a href="#cite_note-1">[1]</a></sup> 은 <a href="/wiki/%EB%B0%94%EC%9D%B4%ED%8A%B8%EC%BD%94%EB%93%9C" title="바이트코드">바이트코드</a>로 변환되는 CIL로 컴파일된다. CIL은 <a class="mw-redirect" href="/wiki/%EA%B0%9D%EC%B2%B4_%EC%A7%80%ED%96%A5" title="객체 지향">객체 지향</a> <a href="/wiki/%EC%96%B4%EC%85%88%EB%B8%94%EB%A6%AC%EC%96%B4" title="어셈블리어">어셈블리어</a>이며 완전한 <a class="new" href="/w/index.php?title=%EC%8A%A4%ED%83%9D_%EA%B8%B0%EB%B0%98_%EB%A9%94%EB%AA%A8%EB%A6%AC_%ED%95%A0%EB%8B%B9&amp;action=edit&amp;redlink=1" title="스택 기반 메모리 할당 (없는 문서)">스택 기반</a>이다. <a href="/wiki/%EA%B0%80%EC%83%81_%EB%A8%B8%EC%8B%A0" title="가상 머신">가상 머신</a>을 통해 실행된다.</p>
<p>닷넷 <a href="/wiki/%ED%94%84%EB%A1%9C%EA%B7%B8%EB%9E%98%EB%B0%8D_%EC%96%B8%EC%96%B4" title="프로그래밍 언어">프로그래밍 언어</a>를 컴파일하는 동안, <a href="/wiki/%EC%86%8C%EC%8A%A4_%EC%BD%94%EB%93%9C" title="소스 코드">소스 코드</a>는 플랫폼, 프로세서 특유의 <a class="mw-redirect" href="/wiki/%EB%AA%A9%EC%A0%81_%EC%BD%94%EB%93%9C" title="목적 코드">목적 코드</a>가 아닌, 공통 중간 언어 코드로 변환된다. 공통 중간 언어는 닷넷 프레임워크를 지원하는 환경에서 실행할 수 있는, <a href="/wiki/%EC%A4%91%EC%95%99_%EC%B2%98%EB%A6%AC_%EC%9E%A5%EC%B9%98" title="중앙 처리 장치">CPU</a>와 플랫폼 독립 명령어 세트이다. 공통 중간 코드는 실행되는 동안에 안전 여부를 확인함으로써 완전한 이진 파일보다 더 나은 보안과 신뢰성을 제공한다.</p>

<h2><span class="mw-headline" id=".EB.AA.85.EC.B9.AD">명칭</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EA%B3%B5%ED%86%B5_%EC%A4%91%EA%B0%84_%EC%96%B8%EC%96%B4&amp;action=edit&amp;section=1" title="부분 편집: 명칭">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>CIL은 원래 닷넷 언어의 베타 버전을 공개하는 동안 마이크로소프트 중간 언어 (MSIL)로 알려져 있던 것이다. C#과 공통 언어 기반의 표준화로 인해, 바이트코드는 이제 공식적으로 CIL로 알려져 있다. 이러한 까닭으로 CIL은 아직도 MSIL로 불리는 일이, 특히 닷넷 언어를 오래 사용해 오던 사용자들에게 많다.</p>
<h2><span class="mw-headline" id=".ED.97.AC.EB.A1.9C_.EC.9B.94.EB.93.9C_.ED.94.84.EB.A1.9C.EA.B7.B8.EB.9E.A8">헬로 월드 프로그램</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EA%B3%B5%ED%86%B5_%EC%A4%91%EA%B0%84_%EC%96%B8%EC%96%B4&amp;action=edit&amp;section=2" title="부분 편집: 헬로 월드 프로그램">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
