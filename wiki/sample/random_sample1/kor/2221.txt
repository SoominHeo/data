<p><b>과들루프 섬</b>(<span style="font-size: smaller;"><a href="/wiki/%ED%94%84%EB%9E%91%EC%8A%A4%EC%96%B4" title="프랑스어">프랑스어</a>:</span> <span lang="fr" xml:lang="fr">Guadeloupe</span>, <span style="font-size: smaller;"><a href="/wiki/%EB%AC%B8%ED%99%94%EC%96%B4" title="문화어">문화어</a>:</span> 과들루쁘)은 <a href="/wiki/%EC%84%9C%EC%9D%B8%EB%8F%84_%EC%A0%9C%EB%8F%84" title="서인도 제도">서인도 제도</a>에 있는 <a href="/wiki/%ED%94%84%EB%9E%91%EC%8A%A4" title="프랑스">프랑스</a>의 해외 <a href="/wiki/%ED%94%84%EB%9E%91%EC%8A%A4%EC%9D%98_%EB%A0%88%EC%A7%80%EC%98%B9" title="프랑스의 레지옹">레지옹</a>으로 주도는 <a href="/wiki/%EB%B0%94%EC%8A%A4%ED%85%8C%EB%A5%B4_(%EA%B3%BC%EB%93%A4%EB%A3%A8%ED%94%84)" title="바스테르 (과들루프)">바스테르</a>이다.</p>
<h2><span class="mw-headline" id=".EC.A3.BC.EB.AF.BC">주민</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EA%B3%BC%EB%93%A4%EB%A3%A8%ED%94%84&amp;action=edit&amp;section=1" title="부분 편집: 주민">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p><a class="new" href="/w/index.php?title=%ED%81%AC%EB%A0%88%EC%98%AC%EC%9D%B8&amp;action=edit&amp;redlink=1" title="크레올인 (없는 문서)">크레올인</a>과 <a href="/wiki/%EC%95%84%ED%94%84%EB%A6%AC%EC%B9%B4%EC%9D%B8" title="아프리카인">아프리카인</a>들이 대부분을 차지하며 <a href="/wiki/%EC%9C%A0%EB%9F%BD%EC%9D%B8" title="유럽인">유럽인</a>(주로 프랑스인)이나 <a href="/wiki/%EC%9D%B8%EB%8F%84%EC%9D%B8" title="인도인">인도인</a>, <a href="/wiki/%EC%A4%91%EA%B5%AD%EC%9D%B8" title="중국인">중국인</a> 등이 있다.</p>
<p>언어는 <a href="/wiki/%ED%94%84%EB%9E%91%EC%8A%A4%EC%96%B4" title="프랑스어">프랑스어</a>와 <a href="/wiki/%EA%B3%BC%EB%93%A4%EB%A3%A8%ED%94%84_%ED%81%AC%EB%A6%AC%EC%98%AC" title="과들루프 크리올">과들루프 크리올</a>을 사용하고 종교는 <a href="/wiki/%EA%B0%80%ED%86%A8%EB%A6%AD" title="가톨릭">가톨릭</a>을 주로 믿는다.</p>
