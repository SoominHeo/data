<p><b>과민증</b>(anaphylaxis)는 급성 앨러지 반응의 하나로 <a href="/wiki/%ED%95%AD%EC%9B%90" title="항원">항원</a>과 항체 <a class="new" href="/w/index.php?title=Ig_E&amp;action=edit&amp;redlink=1" title="Ig E (없는 문서)">Ig E</a>가 매개하는 항체간의 급격한 <a class="new" href="/w/index.php?title=%ED%95%AD%EC%9B%90-%ED%95%AD%EC%B2%B4_%EB%B0%98%EC%9D%91&amp;action=edit&amp;redlink=1" title="항원-항체 반응 (없는 문서)">항원-항체 반응</a>으로 일어난다. 매우 위급한 상황을 초래하여 즉각 치료 하지 않으면 생명이 위독할 수도 있다. <a class="new" href="/w/index.php?title=%EA%B3%BC%EB%AF%BC_%EB%B0%98%EC%9D%91&amp;action=edit&amp;redlink=1" title="과민 반응 (없는 문서)">과민 반응</a>(anaphylactoid reaction)은 이와 비슷하나 Ig E가 매개하지 않는 것이 차이이다.</p>
<h2><span class="mw-headline" id=".EC.A6.9D.EC.83.81">증상</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EA%B3%BC%EB%AF%BC%EC%A6%9D&amp;action=edit&amp;section=1" title="부분 편집: 증상">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>개인마다 차이가 있다. 처음 발작 후 다시 수시간 후에 나타날수도 있고 호흡 문제로 사망 가능하므로 입원이 원칙이다. 증상은 대부분 원인 물질에 노출된 후 즉각 나타나나 1-2시간 후에 나타날 수도 있다. 증상이 늦게 나타날수록 덜 심하다.</p>
<ul>
<li>피부 증상 : <a class="new" href="/w/index.php?title=%ED%99%8D%EB%B0%98&amp;action=edit&amp;redlink=1" title="홍반 (없는 문서)">홍반</a>, 가려움, <a class="new" href="/w/index.php?title=%ED%98%88%EA%B4%80%EB%B6%80%EC%A2%85&amp;action=edit&amp;redlink=1" title="혈관부종 (없는 문서)">혈관성부종</a></li>
<li>상기도 증상 : 목쉼, 발성 장애, 천명</li>
<li>하기도 증상 : 가슴 답답, <a href="/wiki/%ED%98%B8%ED%9D%A1%EA%B3%A4%EB%9E%80" title="호흡곤란">호흡곤란</a></li>
<li>순환기 증상 : <a class="mw-redirect" href="/wiki/%EB%B9%88%EB%A7%A5" title="빈맥">빈맥</a>, <a class="mw-redirect" href="/wiki/%EC%96%B4%EC%A7%80%EB%9F%AC%EC%9B%80" title="어지러움">어지러움</a>, <a href="/wiki/%EC%8B%A4%EC%8B%A0" title="실신">실신</a>, <a href="/wiki/%EC%A0%80%ED%98%88%EC%95%95" title="저혈압">저혈압</a>, <a href="/wiki/%EB%B6%80%EC%A0%95%EB%A7%A5" title="부정맥">부정맥</a></li>
<li>소화기 증상 : 오심, 구토, 복통, 설사</li>
<li>기타 : 뇌전증, 경련, 두통</li>
</ul>
