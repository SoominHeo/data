<p><b>퀵타임</b>(QuickTime)은 <a href="/wiki/%EC%95%A0%ED%94%8C_(%EA%B8%B0%EC%97%85)" title="애플 (기업)">애플</a>이 개발한 <a href="/wiki/%EB%A9%80%ED%8B%B0%EB%AF%B8%EB%94%94%EC%96%B4_%ED%94%84%EB%A0%88%EC%9E%84%EC%9B%8C%ED%81%AC" title="멀티미디어 프레임워크">멀티미디어 프레임워크</a>이며, <a href="/wiki/%EB%94%94%EC%A7%80%ED%84%B8_%EC%98%81%EC%83%81" title="디지털 영상">디지털 영상</a>, <a class="new" href="/w/index.php?title=%EB%AF%B8%EB%94%94%EC%96%B4_%ED%81%B4%EB%A6%BD&amp;action=edit&amp;redlink=1" title="미디어 클립 (없는 문서)">미디어 클립</a>, 소리, 텍스트, 애니메이션, 음악, 몇 가지 종류의 <a class="new" href="/w/index.php?title=%ED%80%B5%ED%83%80%EC%9E%84_VR&amp;action=edit&amp;redlink=1" title="퀵타임 VR (없는 문서)">퀵타임 VR</a>을 다룰 수 있다. <a href="/wiki/%EB%A7%A5_OS" title="맥 OS">클래식 맥 OS</a>, <a class="mw-redirect" href="/wiki/OS_X" title="OS X">OS X</a>, <a class="mw-redirect" href="/wiki/%EB%A7%88%EC%9D%B4%ED%81%AC%EB%A1%9C%EC%86%8C%ED%94%84%ED%8A%B8_%EC%9C%88%EB%8F%84" title="마이크로소프트 윈도">마이크로소프트 윈도</a> 운영 체제를 지원하며, <a href="/wiki/%EC%95%84%EC%9D%B4%ED%8A%A0%EC%8A%A4" title="아이튠스">아이튠스</a>, 퀵타임 플레이어를 포함한 소프트웨어 꾸러미에서 필수적인 지원을 제공한다. (<a href="/wiki/%EC%82%AC%ED%8C%8C%EB%A6%AC_(%EC%9B%B9_%EB%B8%8C%EB%9D%BC%EC%9A%B0%EC%A0%80)" title="사파리 (웹 브라우저)">사파리</a>와 같은 <a href="/wiki/%EC%9B%B9_%EB%B8%8C%EB%9D%BC%EC%9A%B0%EC%A0%80" title="웹 브라우저">웹 브라우저</a>의 <a class="new" href="/w/index.php?title=%EB%8F%84%EC%9B%80%EB%A7%90_%EC%9D%91%EC%9A%A9_%ED%94%84%EB%A1%9C%EA%B7%B8%EB%9E%A8&amp;action=edit&amp;redlink=1" title="도움말 응용 프로그램 (없는 문서)">도움말 응용 프로그램</a>의 역할을 하며 열지 못하는 미디어 파일들을 재생시켜 준다)</p>
<h2><span class="mw-headline" id=".ED.80.B5.ED.83.80.EC.9E.84_.ED.94.8C.EB.A0.88.EC.9D.B4.EC.96.B4">퀵타임 플레이어</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%ED%80%B5%ED%83%80%EC%9E%84&amp;action=edit&amp;section=1" title="부분 편집: 퀵타임 플레이어">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>퀵타임은 무료로 배포되며 퀵타임 플레이어 응용 소프트웨어를 포함한다. 퀵타임 <a class="mw-redirect" href="/wiki/%ED%94%84%EB%A0%88%EC%9E%84%EC%9B%8C%ED%81%AC" title="프레임워크">프레임워크</a>에 의존하는 다른 종류의 무료 플레이어 응용 프로그램들은 기본 퀵타임 플레이어에서 사용할 수 없는 기능들을 제공한다. 이를테면 다음과 같다.</p>
<ul>
<li>아이튠스는 <a class="mw-disambig" href="/wiki/%EC%9B%A8%EC%9D%B4%EB%B8%8C" title="웨이브">WAV</a>, <a href="/wiki/AIFF" title="AIFF">AIFF</a>, <a href="/wiki/MP3" title="MP3">MP3</a>, <a href="/wiki/%EA%B3%A0%EA%B8%89_%EC%98%A4%EB%94%94%EC%98%A4_%EB%B6%80%ED%98%B8%ED%99%94" title="고급 오디오 부호화">AAC</a>, <a href="/wiki/%EC%95%A0%ED%94%8C_%EB%AC%B4%EC%86%90%EC%8B%A4" title="애플 무손실">애플 무손실</a>의 오디오를 내보낼 수 있다.</li>
<li>OS X에서, 단순한 <a href="/wiki/%EC%95%A0%ED%94%8C%EC%8A%A4%ED%81%AC%EB%A6%BD%ED%8A%B8" title="애플스크립트">애플스크립트</a>는 동영상을 전체 화면 방식으로 재생하는 데 쓰일 수 있다. 그러나 퀵타임 플레이어 7.2 버전 이후로 프로 버전이 아닌 경우에라도 전체 화면 보기를 지원한다.</li>
</ul>
<p>어떠한 응용 프로그램이라도 퀵타임 프레임워크에서 제공하는 기능에 접근하도록 프로그램을 짤 수 있다.</p>
<p>퀵타임 플레이어를 설치하면 <a class="mw-redirect" href="/wiki/%EC%9C%88%EB%8F%84_%EB%B9%84%EC%8A%A4%ED%83%80" title="윈도 비스타">윈도 비스타</a>와 <a class="mw-redirect" href="/wiki/%EC%9C%88%EB%8F%84_7" title="윈도 7">윈도 7</a>에서 <a href="/wiki/%EC%9B%B9_%ED%8E%98%EC%9D%B4%EC%A7%80" title="웹 페이지">웹 페이지</a>에 <a class="mw-redirect" href="/wiki/%EC%9E%84%EB%B2%A0%EB%94%94%EB%93%9C" title="임베디드">임베디드</a>된 <a href="/wiki/MP3" title="MP3">MP3</a>파일이 오직 퀵타임 플레이어로만 재생되는 문제가 있으며, 64비트 <a class="mw-redirect" href="/wiki/%EB%A7%88%EC%9D%B4%ED%81%AC%EB%A1%9C%EC%86%8C%ED%94%84%ED%8A%B8_%EC%9C%88%EB%8F%84" title="마이크로소프트 윈도">윈도</a>에선 컨트롤 모음이 검게 나오는 문제가 계속 보고되고 있지만 아직 해결되지 않고 있다. 컨트롤 모음에 대한 유일한 해결책으로는 <a class="mw-redirect" href="/wiki/%EC%9C%88%EB%8F%84_XP" title="윈도 XP">윈도 XP</a> 서비스 팩 모드로 실행하는 방법이 있지만, 이것은 <a class="mw-redirect" href="/wiki/%EC%9C%88%EB%8F%84_%EB%B9%84%EC%8A%A4%ED%83%80" title="윈도 비스타">윈도 비스타</a>와 <a class="mw-redirect" href="/wiki/%EC%9C%88%EB%8F%84_7" title="윈도 7">윈도 7</a>의 <a class="mw-redirect" href="/wiki/%EC%97%90%EC%96%B4%EB%A1%9C_%EC%9D%B8%ED%84%B0%ED%8E%98%EC%9D%B4%EC%8A%A4" title="에어로 인터페이스">에어로 인터페이스</a>를 꺼버리는 문제가 있다.</p>
