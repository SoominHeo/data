<h1 class="firstHeading" id="firstHeading" lang="ko">관념론</h1>
<div class="mw-body-content" id="bodyContent">
<div id="siteSub">위키백과, 우리 모두의 백과사전.</div>
<div id="contentSub"></div>
<div class="mw-jump" id="jump-to-nav">
					이동:					<a href="#mw-head">둘러보기</a>, 					<a href="#p-search">검색</a>
</div>
<div class="mw-content-ltr" dir="ltr" id="mw-content-text" lang="ko"><script>function mfTempOpenSection(id){var block=document.getElementById("mf-section-"+id);block.className+=" open-block";block.previousSibling.className+=" open-block";}</script><p>흔히 실제의 일을 고려하지 않고 머리 속에서만 생각하며 여러 가지로 주장하는 것을 관념론이라고 하는데, 철학에서 말하는 <b>관념론</b>(觀念論)은 이런 뜻과 전혀 관계가 없는 것은 아니나 본질적으로 이런 뜻을 가진 개념이 아니다.</p>
<p>철학에는 <a href="/wiki/%EC%84%B8%EA%B3%84%EA%B4%80" title="세계관">세계관</a>과 <a href="/wiki/%EC%9D%B8%EC%8B%9D%EB%A1%A0" title="인식론">인식론</a>이라는 두 측면이 있으며 이 두 측면에서 <a href="/wiki/%EB%A7%88%EC%9D%8C" title="마음">마음</a> <b>·</b> <a href="/wiki/%EC%A0%95%EC%8B%A0" title="정신">정신</a> <b>·</b> <a class="mw-disambig" href="/wiki/%EC%9D%98%EC%8B%9D" title="의식">의식</a>과 <a href="/wiki/%EC%9E%90%EC%97%B0" title="자연">자연</a> <b>·</b> <a href="/wiki/%EB%AC%BC%EC%A7%88" title="물질">물질</a>과의 관계가 논의되고 있다. 이 경우 마음 <b>·</b> 정신 <b>·</b> 의식 쪽에 중점을 두고서 철학설을 주장하는 것이 관념론이다. <a href="/wiki/%EC%84%B8%EA%B3%84%EA%B4%80" title="세계관">세계관</a> 또는 <a href="/wiki/%EC%9A%B0%EC%A3%BC%EB%A1%A0" title="우주론">우주론</a>의 측면에서는, 마치 종교에서 <a href="/wiki/%EC%B0%BD%EC%A1%B0%EC%8B%A0" title="창조신">신</a>이 세계를 창조했다고 하듯이, 관념론에서는 <a href="/wiki/%EB%A7%88%EC%9D%8C" title="마음">마음</a>, <a href="/wiki/%EC%A0%95%EC%8B%A0" title="정신">정신</a> 또는 <a class="mw-disambig" href="/wiki/%EC%9D%98%EC%8B%9D" title="의식">의식</a>에 의해 또는 이들을 바탕으로 <a class="mw-redirect" href="/wiki/%EB%AC%BC%EC%A7%88_%EC%84%B8%EA%B3%84" title="물질 세계">물질 세계</a>, 즉 <a href="/wiki/%EC%9E%90%EC%97%B0" title="자연">자연</a>과 <a href="/wiki/%EB%AC%BC%EC%A7%88" title="물질">물질</a>이 전개되고 형성되고 구성된다고 설명한다. <a href="/wiki/%EC%9D%B8%EC%8B%9D%EB%A1%A0" title="인식론">인식론</a> 또는 <a class="mw-redirect" href="/wiki/%EC%A7%80%EC%8B%9D%EB%A1%A0" title="지식론">지식론</a> 측면에서는, 세계에 관한 인간의 <a href="/wiki/%EC%A7%80%EC%8B%9D" title="지식">지식</a>은, 반드시 인간의 <a class="mw-disambig" href="/wiki/%EC%9D%98%EC%8B%9D" title="의식">의식</a>이 관계하여 성립되는 것으로, 의식에서 독립하여 존재하는 사물(事物)의 세계에 관한 지식은 아니라고 관념론에서는 주장한다.</p>
<p><a href="/wiki/%EB%A7%88%EC%9D%8C" title="마음">마음</a> <b>·</b> <a href="/wiki/%EC%A0%95%EC%8B%A0" title="정신">정신</a> <b>·</b> <a class="mw-disambig" href="/wiki/%EC%9D%98%EC%8B%9D" title="의식">의식</a>이 <a class="mw-redirect" href="/wiki/%EB%AC%BC%EC%A7%88_%EC%84%B8%EA%B3%84" title="물질 세계">물질 세계</a>를 형성하는 기초 또는 근원이라고 주장하기 때문에 관념론을 <b><a href="/wiki/%EC%9C%A0%EC%8B%AC%EB%A1%A0" title="유심론">유심론</a></b>(唯心論) 또는 <b>비물질론</b>(非物質論)이라고도 한다. 또한 <a class="mw-redirect" href="/wiki/%EB%AC%BC%EC%A7%88_%EC%84%B8%EA%B3%84" title="물질 세계">물질 세계</a>가 <a href="/wiki/%EB%A7%88%EC%9D%8C" title="마음">마음</a>, <a href="/wiki/%EC%A0%95%EC%8B%A0" title="정신">정신</a> 또는 <a class="mw-disambig" href="/wiki/%EC%9D%98%EC%8B%9D" title="의식">의식</a>이 현재 가지고 있는 생각 또는 상념의 현현 또는 표상이라는 입장과 <a class="mw-redirect" href="/wiki/%EB%AC%BC%EC%A7%88_%EC%84%B8%EA%B3%84" title="물질 세계">물질 세계</a>가 원인의 세계가 아닌 결과의 세계라는 입장을 가지기 때문에 <a class="mw-redirect" href="/wiki/%EC%82%AC%EB%AC%BC%EC%9D%98_%EC%84%B8%EA%B3%84" title="사물의 세계">사물의 세계</a>가 <span class="nowrap">"본질적인"</span> 실체(實體) 또는 실재성(實在性)을 가지고 있지 않으며 다만 <span class="nowrap">"임시적인"</span> 실제성(實際性)만을 가진다고 본다. 이런 점에서 관념론 또는 유심론에서는 <a href="/wiki/%EC%84%B8%EA%B3%84" title="세계">세계</a>를 <a href="/wiki/%ED%99%98%EC%98%81" title="환영">환영</a>이라고도 말하기 때문에 관념론을 <b>환영설</b>(幻影說)이라고도 한다.</p>
<p>대표적인 관념론으로는 <a href="/wiki/%ED%9E%8C%EB%91%90%EA%B5%90" title="힌두교">힌두교</a>의 <a href="/wiki/%EC%9A%B0%ED%8C%8C%EB%8B%88%EC%83%A4%EB%93%9C" title="우파니샤드">우파니샤드</a> 철학과 <a <h2><span class="mw-headline" id=".EC.82.AC.EB.B3.80.EC.B2.A0.ED.95.99.28.E6.80.9D.E8.BE.A8.E5.93.B2.E5.AD.B8.29">사변철학(思辨哲學)</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EA%B4%80%EB%85%90%EB%A1%A0&amp;action=edit&amp;section=1" title="부분 편집: 사변철학(思辨哲學)">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>오감(五感)에서 획득되는 경험상의 사실에서 출발하여 철학의 학설을 수립할 때 <a href="/wiki/%EA%B2%BD%ED%97%98%EB%A1%A0" title="경험론">경험론</a>의 철학이라 하나, 이에 반하여 인간의 머리로 궁리하고, 즉 <a href="/wiki/%EC%9D%B4%EC%84%B1" title="이성">이성</a>의 활동으로 사변(思辨)하여 철학의 학설을 세울 때 사변철학이라 한다. 이는 이성이 생각하는 능력(사변을 한다는 것)으로서, 세계 전체에 관해서나 인간 지식의 형성에 관해서도 진실을 밝혀낸다고 보는 견해이다.</p>
<h2><span class="mw-headline" id=".EC.95.84.EC.9D.B4.EB.94.94.EC.96.BC.EB.A6.AC.EC.A6.98.28Idealism.29">아이디얼리즘(Idealism)</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EA%B4%80%EB%85%90%EB%A1%A0&amp;action=edit&amp;section=2" title="부분 편집: 아이디얼리즘(Idealism)">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>이데알리스무스(<span style="font-size: smaller;"><a href="/wiki/%EB%8F%85%EC%9D%BC%EC%96%B4" title="독일어">독일어</a>:</span> <span lang="de" xml:lang="de">Idealismus</span>)혹은 이데알리즘(<span style="font-size: smaller;"><a href="/wiki/%ED%94%84%EB%9E%91%EC%8A%A4%EC%96%B4" title="프랑스어">프랑스어</a>:</span> <span lang="fr" xml:lang="fr">Idéalisme</span>)이라고도 한다. 한국어로는 관념론 및 이상주의라고 번역된다. 이 하나의 말에 두 뜻이 있기 때문에 관념론과 이상주의가 혼동, 혼용되어 자주 올바른 이해를 그르치기도 한다.</p>
<h2><span class="mw-headline" id=".EC.A3.BC.EA.B4.80.EC.A0.81_.EA.B4.80.EB.85.90.EB.A1.A0.28.E4.B8.BB.E8.A7.80.E7.9A.84.E8.A7.80.E5.BF.B5.E8.AB.96.29">주관적 관념론(主觀的觀念論)</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EA%B4%80%EB%85%90%EB%A1%A0&amp;action=edit&amp;section=3" title="부분 편집: 주관적 관념론(主觀的觀念論)">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>관념론의 일종으로서 객관·대상, 즉 우리를 둘러싼 세계는 주관(인간의 의식)의 관념에 불과하며, 주관에서 독립하여 존재하는 세계를 시인하지 않는 견해이다. '내재철학(內在哲學)'이라 불리는 것도 이것의 하나이며, 주관념 관념론의 극단적인 형태가 <a href="/wiki/%EC%9C%A0%EC%95%84%EB%A1%A0" title="유아론">유아론</a>이다.</p>
<h2><span class="mw-headline" id=".EA.B0.9D.EA.B4.80.EC.A0.81_.EA.B4.80.EB.85.90.EB.A1.A0.28.E5.AE.A2.E8.A7.80.E7.9A.84.E8.A7.80.E5.BF.B5.E8.AB.96.29">객관적 관념론(客觀的觀念論)</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EA%B4%80%EB%85%90%EB%A1%A0&amp;action=edit&amp;section=4" title="부분 편집: 객관적 관념론(客觀的觀念論)">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>관념론의 일종으로 절대적 관념론이라고 부르는 경우도 있다. 관념론은 정신을 세계의 기초에 두는 견해를 취하는 철학이나 이 정신을 인간 의식, 즉 주관으로서의 정신을 초월한, 객관적으로 존재하는 정신(신이라든가 절대정신)이라 하여, 이에 바탕을 두고 세계관을 수립하는 철학이다. 또한 개인적인 주관(의식)이 아니라 인간 일반(一般)의 의식을 생각하여 이것이 세계를 만든다는 생각도 객관적 관념론이라 하는 수도 있으나 이러한 입장은 오히려 주관적 관념론이라고 하는 것이 적절하다.</p>
