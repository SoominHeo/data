<p><b>관덕흥</b>(關德興, 예명 <b>조건아</b>(潮健兒), <a href="/wiki/%EB%8C%80%EC%98%81_%EC%A0%9C%EA%B5%AD_%ED%9B%88%EC%9E%A5" title="대영 제국 훈장">MBE</a>, <a href="/wiki/1905%EB%85%84" title="1905년">1905년</a> <a href="/wiki/6%EC%9B%94_27%EC%9D%BC" title="6월 27일">6월 27일</a>~<a href="/wiki/1996%EB%85%84" title="1996년">1996년</a> <a href="/wiki/6%EC%9B%94_28%EC%9D%BC" title="6월 28일">6월 28일</a>)은 <a href="/wiki/%ED%99%8D%EC%BD%A9" title="홍콩">홍콩</a> <a class="new" href="/w/index.php?title=%EC%98%A4%EA%B7%B9&amp;action=edit&amp;redlink=1" title="오극 (없는 문서)">오극</a>(粵劇; 광동의 전통극) 배우이자 각본가, <a href="/wiki/%EC%98%81%ED%99%94" title="영화">영화</a> 배우 겸 <a class="mw-redirect" href="/wiki/%EB%AC%B4%EC%88%A0%EA%B0%80" title="무술가">무술가</a>이다. 또한 많은 영화에서 <a href="/wiki/%ED%99%A9%EB%B9%84%ED%99%8D" title="황비홍">황비홍</a>(黃飛鴻)을 연기한 배우로 유명하다.</p>
<p>둘째 아들로 태어났으며, 어릴 때 소무(小武) <a class="new" href="/w/index.php?title=%EC%A0%95%EC%B7%A8&amp;action=edit&amp;redlink=1" title="정취 (없는 문서)">정취</a>(靚就)에게서 오극을 배웠고, 그래서 사람들에게서 "신정취(新靚就)"라는 별명을 얻었고, 신마사증(新馬師曾)과 함께 이름이 높았다.</p>
<p>관덕흥은 60년 동안 영화계에 종사하면서, 130편에서 주연을 맡았는데, 그 중 87편이 <a href="/wiki/%ED%99%A9%EB%B9%84%ED%99%8D" title="황비홍">황비홍</a> 역할이었다. 황비홍 영화는 세계에서 가장 많은 시리즈가 나온 영화이며, 관덕흥은 한 시리즈 영화에 가장 많은 주연을 한 배우이다.</p>
<p>영화 산업 외에서도 <a class="mw-redirect" href="/wiki/%EB%AC%B4%EC%88%A0" title="무술">무술</a>에 심취하여, 일찍이 <a href="/wiki/%ED%99%8D%EA%B0%80%EA%B6%8C" title="홍가권">홍가권</a>(洪家拳)을 배워 백학파(白鶴派) 및 기타 유파에 속하였으며, 스스로 <a class="new" href="/w/index.php?title=%EB%AC%B4%EA%B7%B9%EA%B0%95%EC%9C%A0%EA%B6%8C&amp;action=edit&amp;redlink=1" title="무극강유권 (없는 문서)">무극강유권</a>(無極剛柔拳)을 창시하였다.</p>
<h2><span class="mw-headline" id=".EC.83.9D.EC.95.A0">생애</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EA%B4%80%EB%8D%95%ED%9D%A5&amp;action=edit&amp;section=1" title="부분 편집: 생애">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>1905년 <a class="mw-redirect" href="/wiki/%EC%9D%8C%EB%A0%A5" title="음력">음력</a> 5월 25일 <a class="mw-redirect" href="/wiki/%EA%B4%91%EC%A0%80%EC%9A%B0" title="광저우">광주</a>(廣州)에서 태어났으며, 본적은 <a class="mw-redirect" href="/wiki/%EA%B4%91%EB%8F%99%EC%84%B1" title="광동성">광동성</a>(廣東省) <a class="new" href="/w/index.php?title=%EA%B0%9C%ED%8F%89%EC%8B%9C&amp;action=edit&amp;redlink=1" title="개평시 (없는 문서)">개평시</a>(開平市) 적감진(赤坎鎮)이다. 빈한한 가정의 둘째 아들로 태어나, 부친을 어려서 병으로 여의었다. 12세에 돈을 벌기 위해 <a class="new" href="/w/index.php?title=%EC%B0%A8%EB%A0%A5&amp;action=edit&amp;redlink=1" title="차력 (없는 문서)">차력</a>을 시작하였으며, 잠시 건축 일을 하기도 하였다. 13세에는 <a href="/wiki/%EC%8B%B1%EA%B0%80%ED%8F%AC%EB%A5%B4" title="싱가포르">싱가포르</a>에 가서 음식점 종업원을 하면서 연극을 배우기 시작했다. 16세에 <a class="mw-redirect" href="/wiki/%EA%B4%91%EC%A0%80%EC%9A%B0" title="광저우">광주</a>로 가서 홍선반(紅船)에 올랐으며, 때때로 고향에 내려가 연출을 하였다. 후에 오극을 갈고 닦기 위해 정원형(靚元亨]])의 제자가 되었다. 또한 각본에 뜻을 두었고, 장편 소설을 쓰기도 하였는데 출판되지 않았다.</p>
<p><a href="/wiki/1928%EB%85%84" title="1928년">1928년</a>(23세)에，결혼하였다. 부인은 배우 찰각승(札腳勝)의 딸이며, 후에 아들을 얻어 이름을 관한천(關漢泉)이라 하였다.</p>
<p><a href="/wiki/1932%EB%85%84" title="1932년">1932년</a>，<a href="/wiki/%EB%AF%B8%EA%B5%AD" title="미국">미국</a> <a href="/wiki/%EC%83%8C%ED%94%84%EB%9E%80%EC%8B%9C%EC%8A%A4%EC%BD%94" title="샌프란시스코">샌프란시스코</a>에서 영화 《가려정조(歌侶情潮)》의 주연을 맡았다.</p>
<p><a href="/wiki/1935%EB%85%84" title="1935년">1935년</a>，《작일지가(昨日之歌)》의 주연을 맡았다. 항일전쟁 시기, 광동에서 오극단을 조직하는 데 참여, 광동과 후난 등지에서 항일 연극을 하였으며, 백구영(白駒榮), 호접영(胡蝶影), 풍경화(馮鏡華) 등과 함께 보경극원(普慶戲院)에서 항일 연극을 거행하였다. 항일 선전 및 군위문 활동을 하였고, 또한 전 미국을 순회하며 항일 모금운동을 전개하여 애국예인(愛國藝人)으로 칭송받았다. <a href="/wiki/1950%EB%85%84%EB%8C%80" title="1950년대">50년대</a>에는 "관덕흥극단(關德興劇團)"을 조직하였고, 후에는 부상으로 인해 약사로 전직하고 영화를 부업으로 삼게 되었다.</p>
<p><a href="/wiki/1952%EB%85%84" title="1952년">1952년</a>，<a href="/wiki/%ED%99%8D%EC%BD%A9" title="홍콩">홍콩</a> <a class="new" href="/w/index.php?title=%ED%82%B9%EC%8A%A4_%EB%A1%9C%EB%93%9C&amp;action=edit&amp;redlink=1" title="킹스 로드 (없는 문서)">킹스 로드</a>(<a href="/wiki/%EC%98%81%EC%96%B4" title="영어">영어</a>:King's Road, <a href="/wiki/%EC%A4%91%EA%B5%AD%EC%96%B4" title="중국어">중국어</a>:英皇道)에 관덕흥의 약국이 열렸다. 주로 <a href="/wiki/%EA%B3%A8%EC%A0%88" title="골절">골절</a>과 <a href="/wiki/%EB%A5%98%EB%A8%B8%ED%8B%B0%EC%A6%98" title="류머티즘">류머티즘</a> 약을 팔았다.</p>
<p><a href="/wiki/1953%EB%85%84" title="1953년">1953년</a>，당시 영국 식민지였던 <a href="/wiki/%EB%A7%90%EB%A0%88%EC%9D%B4_%EB%B0%98%EB%8F%84" title="말레이 반도">말레이 반도</a> 벽력고강주(霹靂古岡州) 공회(公會)에 무술교련으로 초청되어，4년 동안 중국 무술을 가르쳤다.</p>
<p><a href="/wiki/1970%EB%85%84%EB%8C%80" title="1970년대">70년대</a>，홍콩에서 약국 보지림(寶芝林)을 개설했다.</p>
<p><a href="/wiki/1984%EB%85%84" title="1984년">1984년</a>, <a href="/wiki/%EC%98%81%EA%B5%AD" title="영국">영국</a> 정부로부터 <a href="/wiki/%EB%8C%80%EC%98%81_%EC%A0%9C%EA%B5%AD_%ED%9B%88%EC%9E%A5" title="대영 제국 훈장">대영 제국 훈장 5등급</a>(MBE)을 받았다.</p>
<p><a href="/wiki/1996%EB%85%84" title="1996년">1996년</a> 봄 <a href="/wiki/%EB%AF%B8%EA%B5%AD" title="미국">미국</a> <a href="/wiki/%EC%83%8C%ED%94%84%EB%9E%80%EC%8B%9C%EC%8A%A4%EC%BD%94" title="샌프란시스코">샌프란시스코</a> <a class="new" href="/w/index.php?title=%EC%95%94%EC%8A%A4%ED%8A%B8%EB%A1%B1_%EB%8C%80%ED%95%99&amp;action=edit&amp;redlink=1" title="암스트롱 대학 (없는 문서)">암스트롱 대학</a>에서 명예 <a class="new" href="/w/index.php?title=%EB%AC%B8%ED%95%99%EB%B0%95%EC%82%AC&amp;action=edit&amp;redlink=1" title="문학박사 (없는 문서)">문학박사</a> 학위를 받았다. 이 해 암으로 홍콩에서 사망하였다.</p>
