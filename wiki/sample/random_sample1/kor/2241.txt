<p><b>관동군</b>(<span style="font-size: smaller;"><a href="/wiki/%EC%9D%BC%EB%B3%B8%EC%96%B4" title="일본어">일본어</a>:</span> <span lang="ja"><span lang="ja"><ruby><rb>関東軍</rb> <rp>(</rp><rt>かんとうぐん</rt><rp>)</rp></ruby></span></span> <small>간토군<sup>[<a href="/wiki/%EC%9C%84%ED%82%A4%EB%B0%B1%EA%B3%BC:%EC%9D%BC%EB%B3%B8%EC%96%B4%EC%9D%98_%ED%95%9C%EA%B8%80_%ED%91%9C%EA%B8%B0" title="위키백과:일본어의 한글 표기">*</a>]</sup></small>)은 <a href="/wiki/%EC%9D%BC%EB%B3%B8_%EC%A0%9C%EA%B5%AD" title="일본 제국">일본 제국</a>이 세운 <a href="/wiki/%EA%B4%B4%EB%A2%B0_%EC%A0%95%EA%B6%8C" title="괴뢰 정권">괴뢰 정권</a>이었던 <a href="/wiki/%EB%A7%8C%EC%A3%BC%EA%B5%AD" title="만주국">만주국</a>에 상주(常駐)한 <a href="/wiki/%EC%9D%BC%EB%B3%B8_%EC%A0%9C%EA%B5%AD_%EC%9C%A1%EA%B5%B0" title="일본 제국 육군">일본 제국 육군</a>의 주력을 이루는 부대 가운데 하나다. 용어 자체는 <a href="/wiki/%EC%82%B0%ED%95%B4%EA%B4%80" title="산해관">산해관</a>(山海關)의 동쪽에 주둔한 일본군이라고 해서 붙여진 이름이다.</p>

<h2><span class="mw-headline" id=".EC.97.AD.ED.95.A0">역할</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EA%B4%80%EB%8F%99%EA%B5%B0&amp;action=edit&amp;section=1" title="부분 편집: 역할">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p><a href="/wiki/%EC%9D%BC%EB%B3%B8_%EC%A0%9C%EA%B5%AD" title="일본 제국">일본 제국</a>은 <a href="/wiki/%EC%A4%91%EA%B5%AD" title="중국">중국</a>을 침공하는 전진기지로서 <a href="/wiki/%EB%A7%8C%EC%A3%BC" title="만주">만주</a>를 활용하고 있어서 관동군은 그 주변지에서 일본이 지배하는 지역을 확대하고 중국을 침략하는 선봉으로서 다양한 음모 공작과 작전을 수행했으며, 중국인과 조선인을 대상으로 하여 끔찍하고 반인륜적인 <a href="/wiki/%EC%8B%A4%ED%97%98" title="실험">실험</a>을 자행한 100<a href="/wiki/%EB%B6%80%EB%8C%80" title="부대">부대</a>와 <a href="/wiki/731%EB%B6%80%EB%8C%80" title="731부대">731부대</a> 운영을 위시해 수많은 <a class="mw-redirect" href="/wiki/%EC%A0%84%EC%9F%81_%EB%B2%94%EC%A3%84" title="전쟁 범죄">전쟁 범죄</a>를 저지르기도 했다.</p>
<p>1928년에는 참모 <a href="/wiki/%EA%B3%A0%EB%AA%A8%ED%86%A0_%EB%8B%A4%EC%9D%B4%EC%82%AC%EC%BF%A0" title="고모토 다이사쿠">고모토 다이사쿠</a>(河本大作)가 <a href="/wiki/%EC%9E%A5%EC%AD%A4%EB%A6%B0" title="장쭤린">장쭤린</a>(張作霖)을 폭사시켰고, 1931년에는 참모 <a href="/wiki/%EC%9D%B4%ED%83%80%EA%B0%80%ED%82%A4_%EC%84%B8%EC%9D%B4%EC%8B%9C%EB%A1%9C" title="이타가키 세이시로">이타가키 세이시로</a>(板垣征四郞)와 <a href="/wiki/%EC%9D%B4%EC%8B%9C%EC%99%80%EB%9D%BC_%EA%B0%84%EC%A7%80" title="이시와라 간지">이시와라 간지</a>(石原莞爾)이 <a href="/wiki/%EB%A7%8C%EC%A3%BC%EC%82%AC%EB%B3%80" title="만주사변">만주사변</a>을 획책해 1932년 <a href="/wiki/%EC%9D%BC%EB%B3%B8_%EC%A0%9C%EA%B5%AD" title="일본 제국">일본 제국</a>의 <a href="/wiki/%EA%B4%B4%EB%A2%B0_%EC%A0%95%EA%B6%8C" title="괴뢰 정권">괴뢰 정권</a>인 <a href="/wiki/%EB%A7%8C%EC%A3%BC%EA%B5%AD" title="만주국">만주국</a>을 세웠다. 1932년에 관동군 사령관이 주만(駐滿) <a href="/wiki/%EB%8C%80%EC%82%AC_(%EC%99%B8%EA%B5%90%EA%B4%80)" title="대사 (외교관)">특명전권대사</a>(特命全權大使)를 겸하고 중국 <a class="mw-disambig" href="/wiki/%EB%8F%99%EB%B6%81_%EC%A7%80%EB%B0%A9" title="동북 지방">동북 지방</a>을 실질로 지배하기 시작한 이 기간에 관동군은 <a class="mw-redirect" href="/wiki/%EB%82%A8%EB%A7%8C%EC%A3%BC%EC%B2%A0%EB%8F%84%EC%A3%BC%EC%8B%9D%ED%9A%8C%EC%82%AC" title="남만주철도주식회사">남만주철도주식회사</a>와 함께 <a href="/wiki/%EC%8B%9D%EB%AF%BC%EC%A7%80" title="식민지">식민지</a>를 공고히 지배하고 일본의 이익을 위해 <a href="/wiki/%EC%B2%A0%EB%8F%84" title="철도">철도</a> 사업을 했다. <a href="/wiki/%EB%A7%8C%EC%A3%BC%EA%B5%AD" title="만주국">만주국</a>의 주요한 <a href="/wiki/%EC%A0%95%EC%B1%85" title="정책">정책</a> 입안과 집행은 관동군 사령관에게 재가(裁可)받아야 했다.</p>
<p>관동군은 <a href="/wiki/%EA%B4%B4%EB%A2%B0_%EC%A0%95%EA%B6%8C" title="괴뢰 정권">괴뢰 정권</a>인 <a href="/wiki/%EB%A7%8C%EC%A3%BC%EA%B5%AD" title="만주국">만주국</a>을 실질로 통치하는 기구였고 관동군 사령관이 <a href="/wiki/%EB%A7%8C%EC%A3%BC%EA%B5%AD" title="만주국">만주국</a>의 실권자였다.<sup class="reference" id="cite_ref-1"><a href="#cite_note-1">[1]</a></sup></p>
<h2><span class="mw-headline" id=".EA.B5.B0.EB.B9.84">군비</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EA%B4%80%EB%8F%99%EA%B5%B0&amp;action=edit&amp;section=2" title="부분 편집: 군비">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p><a href="/wiki/%EC%86%8C%EB%A0%A8" title="소련">소련</a>을 대상으로 한 <a href="/wiki/%EC%A0%84%EC%9F%81" title="전쟁">전쟁</a> 준비와 중국 침공을 준비하느라고 관동군의 병력은 차츰 증강되었다. 1933년에는 규모가 10만 명이었는데, 관동군은 1938년 <a class="mw-redirect" href="/wiki/%EC%9E%A5%EA%B3%A0%EB%B4%89_%EC%82%AC%EA%B1%B4" title="장고봉 사건">장고봉 사건</a>과 1939년 <a class="mw-redirect" href="/wiki/%EB%85%B8%EB%AA%AC%ED%95%9C_%EC%82%AC%EA%B1%B4" title="노몬한 사건">노몬한 사건</a>으로 <a href="/wiki/%EC%86%8C%EB%A0%A8" title="소련">소련</a>에 도발했다가 패했다. <a href="/wiki/%EB%82%98%EC%B9%98_%EB%8F%85%EC%9D%BC" title="나치 독일">나치 독일</a>이 <a href="/wiki/%EC%86%8C%EB%A0%A8" title="소련">소련</a>을 침공한 후에는 독일을 도와 소련을 견제하려는 목적으로 병력을 크게 늘려 1941년 관동군특별대연습(關東軍特別大演習) 때는 약 100만 명이 되었다.</p>
<h2><span class="mw-headline" id=".ED.8C.A8.EB.A7.9D">패망</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EA%B4%80%EB%8F%99%EA%B5%B0&amp;action=edit&amp;section=3" title="부분 편집: 패망">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>그러나 1941년에 일본이 <a href="/wiki/%EB%AF%B8%EA%B5%AD" title="미국">미국</a>을 상대로 <a href="/wiki/%ED%83%9C%ED%8F%89%EC%96%91_%EC%A0%84%EC%9F%81" title="태평양 전쟁">태평양 전쟁</a>을 일으키자 1943년 가을 이후에는 전력을 <a href="/wiki/%EB%8F%99%EB%82%A8%EC%95%84%EC%8B%9C%EC%95%84" title="동남아시아">동남아시아</a>를 위시한 남방 <a href="/wiki/%EC%A0%84%EC%84%A0_(%EC%A0%84%EC%9F%81)" title="전선 (전쟁)">전선</a>으로 이동하게 되어 <a href="/wiki/%EB%A7%8C%EC%A3%BC" title="만주">만주</a>에는 형식적 전력만 남았다. 1945년 8월 6일 <a href="/wiki/%EB%AF%B8%EA%B5%AD" title="미국">미국</a>이 <a class="mw-redirect" href="/wiki/%EC%9B%90%EC%9E%90%ED%8F%AD%ED%83%84" title="원자폭탄">원자폭탄</a>을 투하한 이후 미국의 영향력을 줄이기 위한 계산으로 소련과 협력하여 신속하게 점령지를 <a href="/wiki/%EC%86%8C%EB%A0%A8%EA%B5%B0" title="소련군">소련군</a>에게 넘겨주어 소련군이 미국보다 먼저 한반도에 진군하도록 협조한다. 일본이 <a href="/wiki/%EB%AC%B4%EC%A1%B0%EA%B1%B4_%ED%95%AD%EB%B3%B5" title="무조건 항복">무조건 항복</a>을 선언한 후인 1945년 8월 19일에 관동군 사령관도 <a href="/wiki/%EB%AC%B4%EC%A1%B0%EA%B1%B4_%ED%95%AD%EB%B3%B5" title="무조건 항복">무조건 항복</a>했다.</p>
