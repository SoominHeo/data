<p><b>관시 진</b>(<span style="font-size: smaller;"><a href="/wiki/%ED%95%9C%EA%B5%AD%EC%96%B4" title="한국어">한국어</a>:</span> <span lang="ko" xml:lang="ko">관서진</span>, <small><a href="/wiki/%EC%A4%91%EA%B5%AD%EC%96%B4" title="중국어">중국어</a></small><small>:</small> <span lang="zh" xml:lang="zh">關西鎮</span>, <small><a href="/wiki/%ED%95%9C%EC%96%B4_%EB%B3%91%EC%9D%8C" title="한어 병음">병음</a>:</small> <font face="Lucida Sans Unicode">Guānxī zhèn</font>)은 <a href="/wiki/%EC%A4%91%ED%99%94%EB%AF%BC%EA%B5%AD" title="중화민국">중화민국</a> <a href="/wiki/%EC%8B%A0%EC%A3%BC_%ED%98%84" title="신주 현">신주 현</a>의 <a href="/wiki/%EC%A7%84_(%ED%96%89%EC%A0%95_%EA%B5%AC%EC%97%AD)" title="진 (행정 구역)">진</a>이다. 넓이는 125.5193㎢이고, 인구는 <a href="/wiki/2015%EB%85%84" title="2015년">2015년</a> 8월 기준으로 30,120명이다.</p>

<h2><span class="mw-headline" id=".EC.A7.80.EB.A6.AC">지리</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EA%B4%80%EC%8B%9C_%EC%A7%84&amp;action=edit&amp;section=1" title="부분 편집: 지리">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>신주 현의 동북쪽에 위치하고 남쪽으로 <a href="/wiki/%EC%A0%A0%EC%8A%A4_%ED%96%A5" title="젠스 향">젠스 향</a>, <a href="/wiki/%ED%97%9D%EC%82%B0_%ED%96%A5" title="헝산 향">헝산 향</a>, 서남쪽으로 <a href="/wiki/%EC%B6%A9%EB%A6%B0_%ED%96%A5" title="충린 향">충린 향</a>, 서북쪽으로 <a href="/wiki/%EC%8B%A0%ED%91%B8_%EC%A7%84" title="신푸 진">신푸 진</a>, 북쪽으로 <a class="mw-redirect" href="/wiki/%ED%83%80%EC%98%A4%EC%9C%84%EC%95%88_%ED%98%84_(%ED%83%80%EC%9D%B4%EC%99%84_%EC%84%B1)" title="타오위안 현 (타이완 성)">타오위안 현</a> <a class="mw-redirect" href="/wiki/%EB%A3%BD%ED%83%84_%ED%96%A5" title="룽탄 향">룽탄 향</a>, 동쪽으로 타오위안 현 <a class="mw-disambig" href="/wiki/%ED%91%B8%EC%8B%B1_%ED%96%A5" title="푸싱 향">푸싱 향</a>과 접한다.</p>
<h2><span class="mw-headline" id=".EC.97.AD.EC.82.AC">역사</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EA%B4%80%EC%8B%9C_%EC%A7%84&amp;action=edit&amp;section=2" title="부분 편집: 역사">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>예전에는 함채붕(咸菜硼)으로 불리고 있었지만 1920년의 지방제 실시 때 간사이(関西)로 개칭 되었다.</p>
