<p><b>제2회 <a href="/wiki/%EC%95%84%EC%8B%9C%EC%95%88_%EA%B2%8C%EC%9E%84" title="아시안 게임">아시안 게임</a></b>은 <a class="new" href="/w/index.php?title=%EC%95%84%EC%8B%9C%EC%95%84%EA%B2%BD%EA%B8%B0%EC%97%B0%EB%A7%B9&amp;action=edit&amp;redlink=1" title="아시아경기연맹 (없는 문서)">아시아경기연맹</a> 주관 아래 <a href="/wiki/%ED%95%84%EB%A6%AC%ED%95%80" title="필리핀">필리핀</a> <a href="/wiki/%EB%A7%88%EB%8B%90%EB%9D%BC" title="마닐라">마닐라</a>에서 <a href="/wiki/5%EC%9B%94_1%EC%9D%BC" title="5월 1일">5월 1일</a>부터 <a href="/wiki/5%EC%9B%94_9%EC%9D%BC" title="5월 9일">5월 9일</a>까지 9일간 열렸다. 19개국에서 총 970명의 선수가 참가해 여덟 종목에 결처 기량을 겨루었다. 참가국 수와 선수 모두 이전 <a href="/wiki/1951%EB%85%84_%EC%95%84%EC%8B%9C%EC%95%88_%EA%B2%8C%EC%9E%84" title="1951년 아시안 게임">1951년 뉴델리 대회</a> 때보다 늘어났다.</p>

<h2><span class="mw-headline" id=".EB.8C.80.ED.9A.8C_.EC.9D.B4.EB.AA.A8.EC.A0.80.EB.AA.A8">대회 이모저모</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=1954%EB%85%84_%EC%95%84%EC%8B%9C%EC%95%88_%EA%B2%8C%EC%9E%84&amp;action=edit&amp;section=1" title="부분 편집: 대회 이모저모">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<ul>
<li><a href="/wiki/%EB%8C%80%ED%95%9C%EB%AF%BC%EA%B5%AD" title="대한민국">대한민국</a>이 첫 출전하였다.</li>
<li>마라톤이 주종목인 <a href="/wiki/%EC%B5%9C%EC%9C%A4%EC%B9%A0" title="최윤칠">최윤칠</a>은 1,500m 달리기에 참가하여 3분 56초 2의 동양신기록으로 결승선을 1위로 통과하여 아시안 게임 사상 <a href="/wiki/%EB%8C%80%ED%95%9C%EB%AF%BC%EA%B5%AD" title="대한민국">대한민국</a>의 첫 금메달을 획득하였다.</li>
<li>육상 10,000m 달리기에서 <a href="/wiki/%EC%B5%9C%EC%B6%A9%EC%8B%9D" title="최충식">최충식</a>은 33분 0초 6의 아시아 신기록을 수립하며 금메달을 획득하였다.</li>
<li><a href="/wiki/%EC%A4%91%ED%99%94%EB%AF%BC%EA%B5%AD" title="중화민국">중화민국</a>이 축구에서 금메달을 획득하였다. <a href="/wiki/%EB%8C%80%ED%95%9C%EB%AF%BC%EA%B5%AD" title="대한민국">대한민국</a>은 은메달을 획득하였다.</li>
<li><a href="/wiki/%EC%9D%BC%EB%B3%B8" title="일본">일본</a>이 육상과 수영 부문에서 경쟁국들을 압도하며 여유있게 종합 1위를 달성하였다.</li>
</ul>
<h2><span class="mw-headline" id=".EC.A2.85.EB.AA.A9">종목</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=1954%EB%85%84_%EC%95%84%EC%8B%9C%EC%95%88_%EA%B2%8C%EC%9E%84&amp;action=edit&amp;section=2" title="부분 편집: 종목">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p><a class="mw-redirect" href="/wiki/%EC%9C%A1%EC%83%81" title="육상">육상</a>, <a href="/wiki/%EC%88%98%EC%98%81" title="수영">수영 (경영 및 다이빙)</a>, <a href="/wiki/%EB%86%8D%EA%B5%AC" title="농구">농구</a>, <a href="/wiki/%EA%B6%8C%ED%88%AC" title="권투">권투</a>, <a href="/wiki/%EC%B6%95%EA%B5%AC" title="축구">축구</a>, <a href="/wiki/%EC%82%AC%EA%B2%A9" title="사격">사격</a>, <a href="/wiki/%EB%A0%88%EC%8A%AC%EB%A7%81" title="레슬링">레슬링</a>, <a href="/wiki/%EC%97%AD%EB%8F%84" title="역도">역도</a>의 8종목이다.</p>
<h2><span class="mw-headline" id=".EC.B0.B8.EA.B0.80.EA.B5.AD">참가국</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=1954%EB%85%84_%EC%95%84%EC%8B%9C%EC%95%88_%EA%B2%8C%EC%9E%84&amp;action=edit&amp;section=3" title="부분 편집: 참가국">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>총 19개국이 참가하였다. <a href="/wiki/%EC%9D%B4%EB%9E%80" title="이란">이란</a>은 선수를 파견하지는 않았다.</p>

<tr>
