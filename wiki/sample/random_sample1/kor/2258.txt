<h1 class="firstHeading" id="firstHeading" lang="ko">광분해</h1>
<div class="mw-body-content" id="bodyContent">
<div id="siteSub">위키백과, 우리 모두의 백과사전.</div>

<div class="mw-content-ltr" dir="ltr" id="mw-content-text" lang="ko"><script>function mfTempOpenSection(id){var block=document.getElementById("mf-section-"+id);block.className+=" open-block";block.previousSibling.className+=" open-block";}</script><div class="dablink"><a href="/wiki/%EC%9C%84%ED%82%A4%EB%B0%B1%EA%B3%BC:%EB%8F%99%EC%9D%8C%EC%9D%B4%EC%9D%98%EC%96%B4_%EB%AC%B8%EC%84%9C" title="위키백과:동음이의어 문서"><img alt="" data-file-height="200" data-file-width="260" height="18" src="//upload.wikimedia.org/wikipedia/commons/thumb/4/4a/Disambig_grey.svg/23px-Disambig_grey.svg.png" srcset="//upload.wikimedia.org/wikipedia/commons/thumb/4/4a/Disambig_grey.svg/35px-Disambig_grey.svg.png 1.5x, //upload.wikimedia.org/wikipedia/commons/thumb/4/4a/Disambig_grey.svg/46px-Disambig_grey.svg.png 2x" width="23"/></a> 이 문서는 분자의 빛에 의한 붕괴에 관한 것입니다. 원자핵의 빛에 의한 붕괴에 대해서는 <a href="/wiki/%EA%B4%91%EB%B6%95%EA%B4%B4" title="광붕괴">광붕괴</a> 문서를 참조하십시오.</div>
<p><b>광분해</b>(光分解, Photodissociation)는 <a href="/wiki/%ED%99%94%ED%95%A9%EB%AC%BC" title="화합물">화합물</a>이 <a href="/wiki/%EA%B4%91%EC%9E%90" title="광자">광자</a>에 의해 분해되는 <a href="/wiki/%ED%99%94%ED%95%99_%EB%B0%98%EC%9D%91" title="화학 반응">화학 반응</a>을 가리키는 말이다. 광분해는 한 개 이상의 광자가 하나의 분자와 상호작용할 때 정의된다.</p>
<p>광분해는 <a href="/wiki/%EA%B0%80%EC%8B%9C%EA%B4%91%EC%84%A0" title="가시광선">가시광선</a>에 의해서만 일어나는 것이 아니다. 충분한 <a href="/wiki/%EC%97%90%EB%84%88%EC%A7%80" title="에너지">에너지</a>를 가진 광자라면 화합물의 결합에 영향을 끼칠 수 있다. 광자의 에너지는 빛의 <a href="/wiki/%ED%8C%8C%EC%9E%A5" title="파장">파장</a>에 반비례하는 고로 <a href="/wiki/%EA%B0%80%EC%8B%9C%EA%B4%91%EC%84%A0" title="가시광선">가시광선</a>이나 <a href="/wiki/%EC%9E%90%EC%99%B8%EC%84%A0" title="자외선">자외선</a>, <a href="/wiki/%EC%97%91%EC%8A%A4%EC%84%A0" title="엑스선">엑스선</a>, 그리고 <a href="/wiki/%EA%B0%90%EB%A7%88%EC%84%A0" title="감마선">감마선</a> 파장의 <a href="/wiki/%EC%A0%84%EC%9E%90%EA%B8%B0%ED%8C%8C" title="전자기파">전자기파</a>가 광분해를 일으킨다.</p>

<h2><span class="mw-headline" id=".EA.B4.91.ED.95.A9.EC.84.B1.EA.B3.BC_.EA.B4.91.EB.B6.84.ED.95.B4">광합성과 광분해</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EA%B4%91%EB%B6%84%ED%95%B4&amp;action=edit&amp;section=1" title="부분 편집: 광합성과 광분해">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p><a href="/wiki/%EA%B4%91%ED%95%A9%EC%84%B1" title="광합성">광합성</a>의 <a href="/wiki/%EB%AA%85%EB%B0%98%EC%9D%91" title="명반응">명반응</a> 중에 광분해가 일어난다. 이때의 반응은 다음과 같다.</p>
<p>H<sub>2</sub>A + 2 광자 <span><span class="mwe-math-mathml-inline mwe-math-mathml-a11y" style="display: none;"><math xmlns="http://www.w3.org/1998/Math/MathML">
<semantics>
<mrow class="MJX-TeXAtom-ORD">
<mstyle displaystyle="true" scriptlevel="0">
<mo stretchy="false">⟶<!-- ⟶ --></mo>
</mstyle>
</mrow>
<annotation encoding="application/x-tex">{\displaystyle \longrightarrow }</annotation>
</semantics>
</math></span><img alt="{\displaystyle \longrightarrow }" aria-hidden="true" class="mwe-math-fallback-image-inline" src="https://wikimedia.org/api/rest_v1/media/math/render/svg/4ffb6a294b21bebe64570c4088d77a884dec95ab" style="vertical-align: -0.338ex; width:3.816ex; height:1.843ex;"/></span> 2e<sup>-</sup> + 2H<sup>+</sup> + A</p>
<p>생성물 "A"의 화학 구조는 광합성을 하는 유기체의 종류에 따라 다르다. <a class="new" href="/w/index.php?title=%ED%99%8D%EC%83%89_%ED%99%A9%EC%84%B8%EA%B7%A0&amp;action=edit&amp;redlink=1" title="홍색 황세균 (없는 문서)">홍색 황세균</a>은 <a class="mw-redirect" href="/wiki/%ED%99%A9%ED%99%94%EC%88%98%EC%86%8C" title="황화수소">황화수소</a>(H<sub>2</sub>S)를 황(S)으로 산화시킨다. <a href="/wiki/%EB%AC%BC" title="물">물</a>(H<sub>2</sub>O)이 광분해의 기질로 작용할 경우 <a href="/wiki/%EC%82%B0%EC%86%8C" title="산소">산소</a>(O<sub>2</sub>)를 내놓게 된다. 이 과정을 통해 지구 대기중에 산소가 순환하는데, 물의 광분해는 <a class="mw-redirect" href="/wiki/%EB%82%A8%EC%A1%B0%EB%A5%98" title="남조류">남조류</a>의 <a href="/wiki/%ED%8B%B8%EB%9D%BC%EC%BD%94%EC%9D%B4%EB%93%9C" title="틸라코이드">틸라코이드</a>나 <a href="/wiki/%EB%85%B9%EC%A1%B0%EB%A5%98" title="녹조류">녹조류</a>와 <a href="/wiki/%EC%8B%9D%EB%AC%BC" title="식물">식물</a>의 <a href="/wiki/%EC%97%BD%EB%A1%9D%EC%B2%B4" title="엽록체">엽록체</a>에서 일어난다.</p>
<h2><span class="mw-headline" id=".EB.8C.80.EA.B8.B0_.EC.A4.91.EC.9D.98_.EA.B4.91.EB.B6.84.ED.95.B4">대기 중의 광분해</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EA%B4%91%EB%B6%84%ED%95%B4&amp;action=edit&amp;section=2" title="부분 편집: 대기 중의 광분해">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<div class="rellink boilerplate seealso"><img alt="" data-file-height="128" data-file-width="128" height="16" src="//upload.wikimedia.org/wikipedia/commons/thumb/e/ec/Crystal_Clear_app_xmag.svg/16px-Crystal_Clear_app_xmag.svg.png" srcset="//upload.wikimedia.org/wikipedia/commons/thumb/e/ec/Crystal_Clear_app_xmag.svg/24px-Crystal_Clear_app_xmag.svg.png 1.5x, //upload.wikimedia.org/wikipedia/commons/thumb/e/ec/Crystal_Clear_app_xmag.svg/32px-Crystal_Clear_app_xmag.svg.png 2x" width="16"/> <a class="new" href="/w/index.php?title=%EA%B4%91%ED%99%94%ED%95%99_%EC%8A%A4%EB%AA%A8%EA%B7%B8&amp;action=edit&amp;redlink=1" title="광화학 스모그 (없는 문서)">광화학 스모그</a> 문서를 참고하십시오.</div>

<p><a class="mw-redirect" href="/wiki/%ED%83%84%ED%99%94%EC%88%98%EC%86%8C" title="탄화수소">탄화수소</a>나 <a href="/wiki/%EC%A7%88%EC%86%8C_%EC%82%B0%ED%99%94%EB%AC%BC" title="질소 산화물">질소 산화물</a> 같은 일차적 <a class="new" href="/w/index.php?title=%EC%98%A4%EC%97%BC_%EB%AC%BC%EC%A7%88&amp;action=edit&amp;redlink=1" title="오염 물질 (없는 문서)">오염 물질</a>이 <a href="/wiki/%EC%A7%88%EC%82%B0%EC%97%BC" title="질산염">질산염</a> 같은 이차적 오염 물질로 전환될 때도 광분해가 일어난다.</p>
<p><a href="/wiki/%EB%8C%80%EB%A5%98%EA%B6%8C" title="대류권">대류권</a>에서 일어나는 주요 광분해 현상은 다음의 두 가지가 있다.</p>
<dl>
<dd>O<sub>3</sub> + hν → O<sub>2</sub> + O(<sup>1</sup>D) λ &lt; 320 nm</dd>
</dl>
<p>이 반응으로 생성된 발생기산소가 물과 반응하여 <a class="new" href="/w/index.php?title=%ED%9E%88%EB%93%9C%EB%A1%9D%EC%8B%A4%EB%9D%BC%EB%94%94%EC%B9%BC&amp;action=edit&amp;redlink=1" title="히드록실라디칼 (없는 문서)">히드록실라디칼</a>을 만들어낸다.</p>
<dl>
<dd>O(<sup>1</sup>D) + H<sub>2</sub>O → 2OH</dd>
</dl>
<p>세제로도 작용하는 히드록실라디칼은 탄화수소를 <a class="mw-redirect" href="/wiki/%EC%82%B0%ED%99%94" title="산화">산화</a>시키는 물질로, <a href="/wiki/%EB%8C%80%EA%B8%B0%ED%99%94%ED%95%99" title="대기화학">대기화학</a>에서 중요하게 다루어진다.</p>
<p>두 번째 반응은 <a class="mw-redirect" href="/wiki/%EB%8C%80%EB%A5%98%EA%B6%8C_%EC%98%A4%EC%A1%B4" title="대류권 오존">대류권 오존</a> 오염의 핵심 반응으로, 다음과 같다.</p>
<dl>
<dd>NO<sub>2</sub> + hν → NO + O</dd>
<dd>O<sub>2</sub> + O → O<sub>3</sub></dd>
</dl>
<p><a href="/wiki/%EC%98%A4%EC%A1%B4%EC%B8%B5" title="오존층">오존층</a>의 형성 역시 광분해로 인한 것이다. 지구의 <a href="/wiki/%EC%84%B1%EC%B8%B5%EA%B6%8C" title="성층권">성층권</a>에 존재하는 오존은 자외선이 산소 분자(O<sub>2</sub>)를 두 개의 산소 <a href="/wiki/%EC%9B%90%EC%9E%90" title="원자">원자</a>로 분해시키고, 이 산소 원자가 멀쩡한 산소 분자와 결합하여 <a href="/wiki/%EC%98%A4%EC%A1%B4" title="오존">오존</a>(O<sub>3</sub>)이 만들어졌다. 또한 <a href="/wiki/%EC%97%BC%ED%99%94%EB%B6%88%ED%99%94%ED%83%84%EC%86%8C" title="염화불화탄소">염화불화탄소</a>가 광분해되어 <a href="/wiki/%EC%9C%A0%EB%A6%AC%EA%B8%B0" title="유리기">유리기</a> 염소를 생성, 오존층을 파괴한다.</p>
<h2><span class="mw-headline" id=".EC.B2.9C.EC.B2.B4.EB.AC.BC.EB.A6.AC.ED.95.99">천체물리학</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EA%B4%91%EB%B6%84%ED%95%B4&amp;action=edit&amp;section=3" title="부분 편집: 천체물리학">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>분자가 분해되고 새로운 분자가 생성되는 과정에서도 광분해가 일어난다. 성간 공간에서는 <a href="/wiki/%EC%A7%84%EA%B3%B5" title="진공">진공</a> 때문에 분자와 유리기가 오랫동안 남아 있을 수 있는데, 광분해는 이 분자들이 분해되는 주요한 요인이다. <a href="/wiki/%ED%95%AD%EC%84%B1" title="항성">항성</a>이 형성되는 <a class="mw-redirect" href="/wiki/%EC%84%B1%EA%B0%84_%EA%B5%AC%EB%A6%84" title="성간 구름">성간 구름</a>의 구성 물질을 연구할 때도 광분해율이 중요하게 다루어지고 있다.</p>
