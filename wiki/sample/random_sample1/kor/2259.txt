<p><b>광서제</b>(光緖帝, <span style="font-size: smaller;"><a href="/wiki/%EB%A7%8C%EC%A3%BC%EC%96%B4" title="만주어">만주어</a>:</span> <span lang="mnc" xml:lang="mnc">Badaranga doro han</span> <small>빠다랑가 도로 한</small>, <span style="font-size: smaller;"><a href="/wiki/%EB%AA%BD%EA%B3%A8%EC%96%B4" title="몽골어">몽골어</a>:</span> <span lang="mn" xml:lang="mn">Badargultu Törü Khaan</span> <small>빠다라올트 투르 칸</small>, <a href="/wiki/1870%EB%85%84" title="1870년">1870년</a> <a href="/wiki/8%EC%9B%94_14%EC%9D%BC" title="8월 14일">8월 14일</a> ~ <a href="/wiki/1908%EB%85%84" title="1908년">1908년</a> <a href="/wiki/11%EC%9B%94_14%EC%9D%BC" title="11월 14일">11월 14일</a>)는 <a href="/wiki/%EC%B2%AD%EB%82%98%EB%9D%BC" title="청나라">청나라</a>의 제11대 <a href="/wiki/%ED%99%A9%EC%A0%9C" title="황제">황제</a>이다. 묘호는 덕종(德宗). 시호는 <b>동천숭운대중지정경문위무인효예지단검관근경황제(同天崇運大中至正經文緯武仁孝睿智端儉寬勤景皇帝)</b>이다. 청나라 역사상 최초의 방계 혈통 출신 황제이다.</p>
<h2><span class="mw-headline" id=".EC.83.9D.EC.95.A0">생애</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EA%B4%91%EC%84%9C%EC%A0%9C&amp;action=edit&amp;section=1" title="부분 편집: 생애">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p><a href="/wiki/%EC%84%9C%ED%83%9C%ED%9B%84" title="서태후">서태후</a>의 지원으로 즉위하였다. <a href="/wiki/1875%EB%85%84" title="1875년">1875년</a>부터 <a href="/wiki/1908%EB%85%84" title="1908년">1908년</a>까지 재위하는 동안 <a href="/wiki/1875%EB%85%84" title="1875년">1875년</a>부터 <a href="/wiki/1881%EB%85%84" title="1881년">1881년</a>부터는 큰어머니인 <a href="/wiki/%ED%9A%A8%EC%A0%95%ED%98%84%ED%99%A9%ED%9B%84" title="효정현황후">동태후</a>가 섭정하였고 <a href="/wiki/1881%EB%85%84" title="1881년">1881년</a>부터 <a href="/wiki/1889%EB%85%84" title="1889년">1889년</a>까지는 큰어머니이자 이모인 <a href="/wiki/%EC%84%9C%ED%83%9C%ED%9B%84" title="서태후">서태후</a>가 섭정을 하였다. 이후 <a href="/wiki/1889%EB%85%84" title="1889년">1889년</a>부터 <a href="/wiki/1898%EB%85%84" title="1898년">1898년</a>까지 친정을 하였으나 <a href="/wiki/1898%EB%85%84" title="1898년">1898년</a> 서태후가 일으킨 궁중 정변으로 유폐되어 서태후에게 다시 실권이 박탈되었고 <a href="/wiki/1908%EB%85%84" title="1908년">1908년</a> 붕어할 때까지 유폐되어 허수아비 임금의 신세를 지냈다.</p>
<p>광서제가 재위한 시기에는 <a class="mw-redirect" href="/wiki/%EC%96%91%EB%AC%B4_%EC%9A%B4%EB%8F%99" title="양무 운동">양무 운동</a>으로 부분적이나마 중국의 근대화가 전개되고 있었다. 그러나 <a href="/wiki/1894%EB%85%84" title="1894년">1894년</a>부터 <a href="/wiki/1895%EB%85%84" title="1895년">1895년</a>까지 일어난 <a class="mw-redirect" href="/wiki/%EC%B2%AD%EC%9D%BC%EC%A0%84%EC%9F%81" title="청일전쟁">청일전쟁</a>이 청나라의 패배로 끝나 <a href="/wiki/%ED%83%80%EC%9D%B4%EC%99%84" title="타이완">타이완</a>이 <a href="/wiki/%EC%9D%BC%EB%B3%B8" title="일본">일본</a>으로 넘어가는 등 중국에서의 열강의 각축은 더욱 치열해졌다.</p>
<p>광서제는 <a href="/wiki/%EC%BA%89%EC%9C%A0%EC%9B%A8%EC%9D%B4" title="캉유웨이">캉유웨이</a>, <a href="/wiki/%EB%9F%89%EC%B9%98%EC%B0%A8%EC%98%A4" title="량치차오">량치차오</a> 등 신진 엘리트들을 등용하여 일본의 <a href="/wiki/%EB%A9%94%EC%9D%B4%EC%A7%80_%EC%9C%A0%EC%8B%A0" title="메이지 유신">메이지 유신</a>을 모델로 한 무술(戊戌)개혁을 실행하고 완전 근대화를 꾀하였다. 당시 캉유웨이 등의 지식인들은 <a href="/wiki/%EB%9F%AC%EC%8B%9C%EC%95%84" title="러시아">러시아</a>의 위협을 제거하기 위해 <a href="/wiki/%EC%98%81%EA%B5%AD" title="영국">영국</a>, 일본과 협력할 것을 제안하고 <a href="/wiki/%EC%8B%A0%EB%AC%B8" title="신문">신문</a>에 개혁안을 싣는 등 개혁을 선도했다. 그러나 보수적인 서태후, <a href="/wiki/%EC%9D%B4%ED%99%8D%EC%9E%A5" title="이홍장">이홍장</a> 등이 반격하자, 개혁에 가담한 지식인들이 <a href="/wiki/%EB%A7%9D%EB%AA%85" title="망명">망명</a>이나 <a href="/wiki/%EC%82%AC%ED%98%95" title="사형">사형</a>을 맞아야 했고, 결국은 서태후가 섭정을 재개했다.</p>
<p><a href="/wiki/1899%EB%85%84" title="1899년">1899년</a>에 일어난 <a href="/wiki/%EC%9D%98%ED%99%94%EB%8B%A8" title="의화단">의화단</a>과 열강 8국의 연합군이 <a href="/wiki/1900%EB%85%84" title="1900년">1900년</a> <a class="mw-redirect" href="/wiki/%EB%B2%A0%EC%9D%B4%EC%A7%95" title="베이징">베이징</a>에서 격돌하는 사이에 광서제는 서태후와 함께 <a href="/wiki/%EC%8B%9C%EC%95%88_%EC%8B%9C" title="시안 시">시안</a>으로 피난했고 그가 총애하던 후궁 진비(珍妃)는 서태후의 명령으로 살해당했다. 베이징 의정서가 성립되어 베이징으로 돌아왔으나 그는 여전히 실권이 없었다. 그는 독서를 즐겼고 이상주의적 성격이 강해 보수적인 서태후와 대립했다. <a href="/wiki/1908%EB%85%84" title="1908년">1908년</a> <a href="/wiki/11%EC%9B%94_14%EC%9D%BC" title="11월 14일">11월 14일</a>에 사망했고, 당시 <a href="/wiki/%EC%9D%B4%EC%A7%88" title="이질">이질</a>을 앓고 있던 서태후는 제12대 황제로 <a class="mw-redirect" href="/wiki/%ED%91%B8%EC%9D%B4" title="푸이">푸이</a>를 서둘러 지명한 뒤 다음날인 <a href="/wiki/11%EC%9B%94_15%EC%9D%BC" title="11월 15일">11월 15일</a>에 붕어했다. <a href="/wiki/%EC%9C%84%EC%95%88%EC%8A%A4%EC%B9%B4%EC%9D%B4" title="위안스카이">위안스카이</a>가 그를 독살했다는 의혹이 오랫동안 제기되어오다가 <a href="/wiki/2008%EB%85%84" title="2008년">2008년</a> 그의 머리카락을 X선 조사한 결과 독살임이 사실로 밝혀졌다.<sup class="reference" id="cite_ref-1"><a href="#cite_note-1">[1]</a></sup></p>
