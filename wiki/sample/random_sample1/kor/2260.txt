<p><b>광양축구전용구장</b>(光陽蹴球專用球場)은 <a href="/wiki/%EC%A0%84%EB%9D%BC%EB%82%A8%EB%8F%84" title="전라남도">전라남도</a> <a href="/wiki/%EA%B4%91%EC%96%91%EC%8B%9C" title="광양시">광양시</a> <a href="/wiki/%EA%B8%88%ED%98%B8%EB%8F%99_(%EA%B4%91%EC%96%91%EC%8B%9C)" title="금호동 (광양시)">금호동</a> <a href="/wiki/%ED%8F%AC%EC%8A%A4%EC%BD%94" title="포스코">포스코</a> <a class="mw-redirect" href="/wiki/%EA%B4%91%EC%96%91%EC%A0%9C%EC%B2%A0%EC%86%8C" title="광양제철소">광양제철소</a> 내에 위치한 관중석 13,496석 규모의 축구 전용 구장으로, <a href="/wiki/1993%EB%85%84" title="1993년">1993년</a> <a href="/wiki/%EB%8C%80%ED%95%9C%EB%AF%BC%EA%B5%AD" title="대한민국">대한민국</a>에서 2번째로 개장한 <a href="/wiki/%EC%B6%95%EA%B5%AC_%EA%B2%BD%EA%B8%B0%EC%9E%A5" title="축구 경기장">축구 경기장</a>이다. 짧게 광양전용구장으로 널리 지칭되고 있다.</p>

<h2><span class="mw-headline" id=".EC.97.AD.EC.82.AC">역사</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EA%B4%91%EC%96%91%EC%B6%95%EA%B5%AC%EC%A0%84%EC%9A%A9%EA%B5%AC%EC%9E%A5&amp;action=edit&amp;section=1" title="부분 편집: 역사">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p><a href="/wiki/1992%EB%85%84_1%EC%9B%94" title="1992년 1월">1992년 1월</a> 착공하여 <a href="/wiki/1992%EB%85%84_9%EC%9B%94" title="1992년 9월">1992년 9월</a> 완공되었으나 바로 개장하지 않고 <a href="/wiki/1993%EB%85%84" title="1993년">1993년</a> <a href="/wiki/3%EC%9B%94_4%EC%9D%BC" title="3월 4일">3월 4일</a> <a href="/wiki/%ED%8F%AC%ED%95%AD_%EC%8A%A4%ED%8B%B8%EB%9F%AC%EC%8A%A4" title="포항 스틸러스">포항제철 아톰스</a>와 <a href="/wiki/%EC%A4%91%EA%B5%AD" title="중국">중국</a> <a href="/wiki/%EB%8B%A4%EB%A1%84_%EC%8A%A4%EB%8D%94" title="다롄 스더">다롄 스더</a>의 개장 경기를 시작으로 당시 <a href="/wiki/%ED%8F%AC%ED%95%AD_%EC%8A%A4%ED%8B%B8%EB%9F%AC%EC%8A%A4" title="포항 스틸러스">포항제철 아톰스</a>의 제2 홈 경기장으로 사용되어<sup class="reference" id="cite_ref-1"><a href="#cite_note-1">[1]</a></sup> <a href="/wiki/K%EB%A6%AC%EA%B7%B8_%ED%81%B4%EB%9E%98%EC%8B%9D" title="K리그 클래식">프로축구</a> 경기들이 개최되었다.</p>
<p>이 후 <a href="/wiki/1994%EB%85%84" title="1994년">1994년</a> <a href="/wiki/12%EC%9B%94_16%EC%9D%BC" title="12월 16일">12월 16일</a> <a href="/wiki/%EC%A0%84%EB%82%A8_%EB%93%9C%EB%9E%98%EA%B3%A4%EC%A6%88" title="전남 드래곤즈">전남 드래곤즈</a>가 창단되어 <a href="/wiki/K%EB%A6%AC%EA%B7%B8_%ED%81%B4%EB%9E%98%EC%8B%9D" title="K리그 클래식">K리그</a>에 참여하면서, <a href="/wiki/1995%EB%85%84_3%EC%9B%94" title="1995년 3월">1995년 3월</a> <a class="mw-redirect" href="/wiki/%EC%95%84%EB%94%94%EB%8B%A4%EC%8A%A4_%EC%BB%B5" title="아디다스 컵">아디다스 컵</a>부터 <a href="/wiki/%EC%A0%84%EB%82%A8_%EB%93%9C%EB%9E%98%EA%B3%A4%EC%A6%88" title="전남 드래곤즈">전남 드래곤즈</a>의 홈구장으로 사용되고 있다.</p>
<p>남쪽 관중석 뒤에 천연색 전광판 1개를 갖추고 있으며, 원폴형 조명타워가 설치되어 있다.</p>

<li class="gallerybox" style="width: 155px">
