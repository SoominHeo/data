<p><b>광주 도시철도</b>(光州都市鐵道)는 <a href="/wiki/%EB%8C%80%ED%95%9C%EB%AF%BC%EA%B5%AD" title="대한민국">대한민국</a> <a href="/wiki/%EA%B4%91%EC%A3%BC%EA%B4%91%EC%97%AD%EC%8B%9C" title="광주광역시">광주광역시</a> 내부를 운행하는 <a href="/wiki/%EB%8F%84%EC%8B%9C_%EC%B2%A0%EB%8F%84" title="도시 철도">도시 철도</a> 체계이다. <a href="/wiki/%EA%B4%91%EC%A3%BC%EA%B4%91%EC%97%AD%EC%8B%9C_%EB%8F%84%EC%8B%9C%EC%B2%A0%EB%8F%84%EA%B3%B5%EC%82%AC" title="광주광역시 도시철도공사">광주광역시 도시철도공사</a>에서 운영을 담당하며, 현재 <a href="/wiki/%EA%B4%91%EC%A3%BC_%EB%8F%84%EC%8B%9C%EC%B2%A0%EB%8F%84_1%ED%98%B8%EC%84%A0" title="광주 도시철도 1호선">1호선</a>이 운행 중이다. <a href="/wiki/2004%EB%85%84" title="2004년">2004년</a> <a href="/wiki/4%EC%9B%94_28%EC%9D%BC" title="4월 28일">4월 28일</a> 1호선 소태 - 상무 구간이 최초 개통되면서 영업을 시작하였다.</p>

<h2><span class="mw-headline" id=".EA.B5.AC.EC.84.B1">구성</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EA%B4%91%EC%A3%BC_%EB%8F%84%EC%8B%9C%EC%B2%A0%EB%8F%84&amp;action=edit&amp;section=1" title="부분 편집: 구성">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>광주 도시철도는 최초 5호선까지 계획되었으나, <a class="mw-redirect" href="/wiki/IMF_%EA%B5%AC%EC%A0%9C%EA%B8%88%EC%9C%B5%EC%82%AC%EA%B1%B4" title="IMF 구제금융사건">IMF 구제금융사건</a>으로 <a href="/wiki/1996%EB%85%84" title="1996년">1996년</a> 착공된 1호선을 제외한 나머지 계획은 폐기되었다. <a href="/wiki/2000%EB%85%84" title="2000년">2000년</a> 이후 재추진되어 현재 3호선까지 계획되어 있다가 3호선을 없애고 2호선을 확대순환화 하기로 했다.</p>
<h3><span class="mw-headline" id=".EC.B5.9C.EC.B4.88_.EA.B3.84.ED.9A.8D">최초 계획</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EA%B4%91%EC%A3%BC_%EB%8F%84%EC%8B%9C%EC%B2%A0%EB%8F%84&amp;action=edit&amp;section=2" title="부분 편집: 최초 계획">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<ul>
<li><b>1호선</b>(20.60㎞) : 동구 소태동 ~ 금남로 ~ 상무신도시 ~ 광주공항 ~ 광주송정역 ~ 평동산업단지 (1996년 착공 후 2003년 개통 계획)</li>
<li><b>2호선</b>(13.70㎞) : 남구 효천역 - 송암공단 - 백운광장 - 금남로 4가 - 말바우시장 - 북구 문화동 (2004년 착공 후 2009년 개통 계획)</li>
<li><b>3호선</b>(24.00km) : 장성군 월정리 - 광산구 첨단지구 - 양산지구 - 국립광주박물관 - 광주문화예술회관 - 유스퀘어 - 농성 - 백운광장 - 남광주역 - 광주역</li>
<li><b>4호선</b>(28.15㎞) : 광주역 - 전남대학교 - 북구 오치동 - 일곡지구 - 양산지구 - 보훈병원 - 광주시청 - 상무 - 서구 금호지구 - 월드컵경기장 - 금남로 5가 - 광주역</li>

<h3><span class="mw-headline" id="1.ED.98.B8.EC.84.A0">1호선</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EA%B4%91%EC%A3%BC_%EB%8F%84%EC%8B%9C%EC%B2%A0%EB%8F%84&amp;action=edit&amp;section=5" title="부분 편집: 1호선">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<div class="rellink relarticle mainarticle"><img alt="" data-file-height="128" data-file-width="128" height="16" src="//upload.wikimedia.org/wikipedia/commons/thumb/e/ec/Crystal_Clear_app_xmag.svg/16px-Crystal_Clear_app_xmag.svg.png" srcset="//upload.wikimedia.org/wikipedia/commons/thumb/e/ec/Crystal_Clear_app_xmag.svg/24px-Crystal_Clear_app_xmag.svg.png 1.5x, //upload.wikimedia.org/wikipedia/commons/thumb/e/ec/Crystal_Clear_app_xmag.svg/32px-Crystal_Clear_app_xmag.svg.png 2x" width="16"/> 이 부분의 본문은 <a href="/wiki/%EA%B4%91%EC%A3%BC_%EB%8F%84%EC%8B%9C%EC%B2%A0%EB%8F%84_1%ED%98%B8%EC%84%A0" title="광주 도시철도 1호선">광주 도시철도 1호선</a>입니다.</div>
<p><a href="/wiki/%EB%B6%80%EC%82%B0_%EB%8F%84%EC%8B%9C%EC%B2%A0%EB%8F%84" title="부산 도시철도">부산</a>, <a href="/wiki/%EB%8C%80%EA%B5%AC_%EB%8F%84%EC%8B%9C%EC%B2%A0%EB%8F%84" title="대구 도시철도">대구</a>에 이어 세 번째로 수도권을 제외한 지역에 건설된 지하철이다. <a class="mw-redirect" href="/wiki/IMF_%EA%B5%AC%EC%A0%9C%EA%B8%88%EC%9C%B5%EC%82%AC%EA%B1%B4" title="IMF 구제금융사건">IMF 구제금융사건</a>의 여파로 2단계로 나누어 건설되었다. 1구간(녹동 - 남광주역 - 금남로 - 상무)은 <a href="/wiki/1996%EB%85%84" title="1996년">1996년</a> <a href="/wiki/8%EC%9B%94_28%EC%9D%BC" title="8월 28일">8월 28일</a> 기공하여 <a href="/wiki/2004%EB%85%84" title="2004년">2004년</a> <a href="/wiki/4%EC%9B%94_28%EC%9D%BC" title="4월 28일">4월 28일</a>에 개통했으며, 2구간(상무 - 송정리역 - 평동)은 <a href="/wiki/2000%EB%85%84" title="2000년">2000년</a> <a href="/wiki/4%EC%9B%94_6%EC%9D%BC" title="4월 6일">4월 6일</a> 기공하여 <a href="/wiki/2008%EB%85%84" title="2008년">2008년</a> <a href="/wiki/4%EC%9B%94_11%EC%9D%BC" title="4월 11일">4월 11일</a>에 개통하였다.</p>
<p>번화가인 금남로와 <a href="/wiki/%ED%98%B8%EB%82%A8%EA%B3%A0%EC%86%8D%EC%B2%A0%EB%8F%84" title="호남고속철도">호남고속철도</a> 개통으로 이용객이 증가한 <a href="/wiki/%EA%B4%91%EC%A3%BC%EC%86%A1%EC%A0%95%EC%97%AD" title="광주송정역">광주송정역</a>을 지나기 때문에 예전까지만 해도 적었던 이용객이 대폭 증가했다. 하지만 <a href="/wiki/%EC%9C%A0%EC%8A%A4%ED%80%98%EC%96%B4" title="유스퀘어">유스퀘어</a> 버스 터미널을 제대로 통과하지 못하는 아쉬움도 공존하고 있다.</p>
