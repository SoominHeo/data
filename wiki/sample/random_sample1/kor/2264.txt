<h2><span class="mw-headline" id=".EC.97.AD.EC.82.AC">역사</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EA%B4%91%ED%95%99&amp;action=edit&amp;section=1" title="부분 편집: 역사">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>인류 문명의 초기부터 빛은 <a href="/wiki/%EB%AC%BC%EB%A6%AC%ED%95%99" title="물리학">물리학</a>, <a href="/wiki/%EC%B2%A0%ED%95%99" title="철학">철학</a>, <a href="/wiki/%EC%9D%98%ED%95%99" title="의학">의학</a>, <a href="/wiki/%EC%8B%A0%ED%95%99" title="신학">신학</a> 등 여러 학문에 걸쳐 언급되었고 연구되었다. <a class="mw-redirect" href="/wiki/%EC%9C%A0%ED%81%B4%EB%A6%AC%EB%93%9C" title="유클리드">유클리드</a>는 그의 저서를 통해 빛의 직진성과 반사법칙 등을 언급하였고 <a href="/wiki/%EC%95%84%EB%A6%AC%EC%8A%A4%ED%86%A0%ED%85%94%EB%A0%88%EC%8A%A4" title="아리스토텔레스">아리스토텔레스</a>, <a href="/wiki/%ED%81%B4%EB%9D%BC%EC%9A%B0%EB%94%94%EC%98%A4%EC%8A%A4_%ED%94%84%ED%86%A8%EB%A0%88%EB%A7%88%EC%9D%B4%EC%98%A4%EC%8A%A4" title="클라우디오스 프톨레마이오스">프톨레마이오스</a> 등도 빛의 성질을 다루었으며 <a href="/wiki/%EC%A4%91%EC%84%B8" title="중세">중세</a>의 <a href="/wiki/%EB%B2%A0%EC%9D%B4%EC%BB%A8" title="베이컨">베이컨</a>, <a class="new" href="/w/index.php?title=%EB%B9%84%ED%85%94%EB%A1%9C&amp;action=edit&amp;redlink=1" title="비텔로 (없는 문서)">비텔로</a> 등의 <a href="/wiki/%EC%B2%A0%ED%95%99%EC%9E%90" title="철학자">철학자</a>들 또한 빛의 성질에 관심을 보였다. 중세 이슬람의 과학자 <a class="mw-redirect" href="/wiki/%EC%95%8C%ED%95%98%EC%9D%B4%ED%83%90" title="알하이탐">알하이탐</a>은 《광학의 서》에서 직접 관찰한 빛의 직진, 분산, 반사, 굴절 등과 같은 현상을 기록해 놓았다.<sup class="reference" id="cite_ref-1"><a href="#cite_note-1">[1]</a></sup> 알하이탐의 《광학의 서》는 1270년 라틴어로 번역되어 유럽의 과학 연구에 많은 영향을 주기도 하였다.</p>
<p>그러나 광학이 체계적으로 연구되기 시작한 것은 17세기 이후의 일로써, 안경 제작자였던 한스에 의해 <a href="/wiki/%EB%A7%9D%EC%9B%90%EA%B2%BD" title="망원경">망원경</a>이 세상에 알려지면서 <a href="/wiki/%EA%B0%88%EB%A6%B4%EB%A0%88%EC%98%A4_%EA%B0%88%EB%A6%B4%EB%A0%88%EC%9D%B4" title="갈릴레오 갈릴레이">갈릴레이</a>, <a href="/wiki/%EC%9A%94%ED%95%98%EB%84%A4%EC%8A%A4_%EC%BC%80%ED%94%8C%EB%9F%AC" title="요하네스 케플러">요하네스 케플러</a>, <a class="mw-redirect" href="/wiki/%EB%8D%B0%EC%B9%B4%EB%A5%B4%ED%8A%B8" title="데카르트">데카르트</a> 등이 이를 발전시켰다. 당시에는 빛이 <a href="/wiki/%EC%97%90%ED%85%8C%EB%A5%B4_(%EB%AC%BC%EB%A6%AC)" title="에테르 (물리)">에테르</a>(ether)라는 가상의 매질을 통해 전파된다는 에테르 설이 지배적이었다. 이후 <a href="/wiki/%EC%95%84%EC%9D%B4%EC%9E%91_%EB%89%B4%ED%84%B4" title="아이작 뉴턴">아이작 뉴턴</a>은 태양광을 <a href="/wiki/%ED%94%84%EB%A6%AC%EC%A6%98" title="프리즘">프리즘</a>에 통과시켰을 때 굴절률에 따라 색상이 분해된다는 것을 관찰하였고<sup class="reference" id="cite_ref-2"><a href="#cite_note-2">[2]</a></sup>, 1704년 《광학》을 출판하면서 빛을 일종의 입자로 설명하였다.<sup class="reference" id="cite_ref-3"><a href="#cite_note-3">[3]</a></sup> 한편, <a href="/wiki/%ED%81%AC%EB%A6%AC%EC%8A%A4%ED%8B%B0%EC%95%88_%ED%95%98%EC%9C%84%ED%97%8C%EC%8A%A4" title="크리스티안 하위헌스">크리스티안 하위헌스</a>는 빛을 파동으로 여겼고, 이를 바탕으로 빛의 반사와 굴절을 완벽하게 설명하는 수리 모형을 수립하였다.<sup class="reference" id="cite_ref-4"><a href="#cite_note-4">[4]</a></sup></p>
<p>빛이 입자인지 파동인지에 대한 논란은 이후에도 20세기 초까지 계속되었다. 뉴턴이 입자설을 제기하였으나, 빛은 파동의 성질을 갖는 것이 명확했기 때문이다. <a href="/wiki/%ED%86%A0%EB%A8%B8%EC%8A%A4_%EC%98%81" title="토머스 영">토머스 영</a>은 이중 슬릿 실험으로 <a class="new" href="/w/index.php?title=%EA%B0%84%EC%84%AD_%ED%9A%A8%EA%B3%BC&amp;action=edit&amp;redlink=1" title="간섭 효과 (없는 문서)">간섭 효과</a>를 확인하여 빛이 파동이라는 것을 뒷받침하였고<sup class="reference" id="cite_ref-5"><a href="#cite_note-5">[5]</a></sup>, <a class="new" href="/w/index.php?title=%EC%97%90%EB%93%9C%EB%AA%BD_%EB%B2%A0%ED%81%AC%EB%A0%90&amp;action=edit&amp;redlink=1" title="에드몽 베크렐 (없는 문서)">에드몽 베크렐</a>의 전기에 의한 발광 실험, 패러데이의 자장에 의한 빛의 편광 실험 등에서 나타난 바를 토대로 맥스웰은 빛 역시 전자파의 일종이라는 사실을 밝혀내었다.<sup class="reference" id="cite_ref-6"><a href="#cite_note-6">[6]</a></sup> 그러나, 아인슈타인은 <a href="/wiki/%EA%B4%91%EC%A0%84_%ED%9A%A8%EA%B3%BC" title="광전 효과">광전 효과</a> 실험을 통하여 빛이 입자라는 것을 뒷받침하였다.<sup class="reference" id="cite_ref-7"><a href="#cite_note-7">[7]</a></sup> 빛이 입자로서의 성질과 파동으로서의 성질을 모두 갖고 있다는 것을 모순 없이 설명하는 것은 <a class="mw-redirect" href="/wiki/%EC%96%91%EC%9E%90_%EC%97%AD%ED%95%99" title="양자 역학">양자 역학</a>이 수립된 이후에야 가능하였다. 현대 물리학에서는 빛이 맥스웰 방정식을 만족하는 전자파의 성질을 갖는 동시에, 다른 물질과 상호 작용할 때에는 양자화된 에너지의 특성이 나타나 입자적 성질을 보여준다고 설명하고 있다.<sup class="reference" id="cite_ref-8"><a href="#cite_note-8">[8]</a></sup></p>
<h2><span class="mw-headline" id=".EB.B6.84.EC.95.BC">분야</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EA%B4%91%ED%95%99&amp;action=edit&amp;section=2" title="부분 편집: 분야">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>광학의 세부분야로 <a href="/wiki/%EA%B8%B0%ED%95%98%EA%B4%91%ED%95%99" title="기하광학">기하광학</a>, <a class="new" href="/w/index.php?title=%ED%8C%8C%EB%8F%99%EA%B4%91%ED%95%99&amp;action=edit&amp;redlink=1" title="파동광학 (없는 문서)">파동광학</a>, <a href="/wiki/%EB%B6%84%EA%B4%91%ED%95%99" title="분광학">분광학</a>, <a href="/wiki/%EC%96%91%EC%9E%90%EA%B4%91%ED%95%99" title="양자광학">양자광학</a>, <a class="mw-redirect" href="/wiki/%EB%B9%84%EC%84%A0%ED%98%95%EA%B4%91%ED%95%99" title="비선형광학">비선형광학</a>, <a href="/wiki/%EA%B4%91%EC%9C%A0%EC%A0%84%ED%95%99" title="광유전학">광유전학</a> 등이 있다. 기하광학은 빛의 입자성에 바탕을 두고 진공과 매질 속에서 빛이 지나가는 경로에 관심을 두고 기술하는 분야이고, 파동광학은 빛의 파동성에 바탕을 두고 호이겐스의 원리를 기본으로 회절, 간섭 등의 특성을 기술한다. 분광학은 빛이 매질을 통과하면서 나타나는 여러 현상들을 분석하여 물질의 특성을 연구하는 분야이고, 비선형 광학은 매질 속에서 강한 세기의 빛이 반사 혹은 투과될 때 그 진동수가 두배 혹은 그 이상의 정수배에 해당하는 빛이 나오는 현상을 바탕으로 분광학과 함께 물성연구 등의 다양한 응용이 기대되는 분야이다. 양자광학은 <a class="mw-redirect" href="/wiki/%EC%96%91%EC%9E%90_%EC%97%AD%ED%95%99" title="양자 역학">양자 역학</a>에 의해 밝혀진 빛알(혹은 광자, photon)로서의 빛의 성질을 기술하는 분야로 최근 각광받고있는 <a class="mw-redirect" href="/wiki/%EC%96%91%EC%9E%90%EC%BB%B4%ED%93%A8%ED%84%B0" title="양자컴퓨터">양자컴퓨터</a>와 관련하여 활발한 연구가 진행 중인 분야이다. 광유전학은 빛으로 뇌를 조종하는 분야다. 현재는 광유전학으로 쥐를 조종하는 <a class="new" href="/w/index.php?title=%EB%AC%B4%EC%84%A0%EB%A7%88%EC%9A%B0%EC%8A%A4&amp;action=edit&amp;redlink=1" title="무선마우스 (없는 문서)">무선마우스</a> 등이 나와있다. 이 밖에도 색채론 등이 있다.</p>
<p>일반적으로 광학적 현상을 설명하는 방법으로는 빛을 전자파, 즉 전자기의 파동으로 보아 맥스웰의 방정식에 기반하여 설명하는 방식인 파동광학, 즉 물리 광학과, 빛을 직진하는 광선으로 파악하여 기하학적 도형으로 설명하는 기하 광학이 사용된다. 물리 광학은 빛의 위상이나 편파와 같은 복잡한 현상을 감안하여 빛의 운동을 정확히 기술할 수 있으나, 매우 복잡한 수학을 사용하여야 한다는 단점이 있다. 빛을 받는 물체의 크기가 <a href="/wiki/%ED%8C%8C%EC%9E%A5" title="파장">파장</a>보다 충분히 크면 기하학적 모형으로도 비교적 정확한 결과를 얻을 수 있기 때문에 간단한 예측에서는 기하 광학이 많이 사용되지만, 기하 광학은 <a class="new" href="/w/index.php?title=%EA%B0%84%EC%84%AD_%ED%9A%A8%EA%B3%BC&amp;action=edit&amp;redlink=1" title="간섭 효과 (없는 문서)">간섭 효과</a>나 <a href="/wiki/%ED%9A%8C%EC%A0%88" title="회절">회절</a>, <a href="/wiki/%EC%9C%84%EC%83%81" title="위상">위상</a>과 같은 것은 무시하기 때문에 정밀한 계산 작업에서는 사용되지 않는다.<sup class="reference" id="cite_ref-9"><a href="#cite_note-9">[9]</a></sup></p>
