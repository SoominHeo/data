<p><b>1956년 프랑스 총선</b>은 <a href="/wiki/1956%EB%85%84" title="1956년">1956년</a> <a href="/wiki/%ED%94%84%EB%9E%91%EC%8A%A4" title="프랑스">프랑스</a>에서 치러진 총선으로, <a class="mw-redirect" href="/wiki/%EC%A0%95%EB%8B%B9%EB%AA%85%EB%B6%80%EC%8B%9D_%EB%B9%84%EB%A1%80%EB%8C%80%ED%91%9C%EC%A0%9C" title="정당명부식 비례대표제">정당명부식 비례대표제</a>가 도입되었다.</p>
<h2><span class="mw-headline" id=".EB.B0.B0.EA.B2.BD">배경</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=1956%EB%85%84_%ED%94%84%EB%9E%91%EC%8A%A4_%EC%B4%9D%EC%84%A0&amp;action=edit&amp;section=1" title="부분 편집: 배경">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p><a class="mw-redirect" href="/wiki/%EB%94%94%EC%97%94_%EB%B9%84%EC%97%94_%ED%91%B8_%EC%A0%84%ED%88%AC" title="디엔 비엔 푸 전투">디엔 비엔 푸 전투</a>의 패배로 찾아온 정치적 위기로 총리가 된 <a class="new" href="/w/index.php?title=%ED%94%BC%EC%97%90%EB%A5%B4_%EB%A7%9D%EB%8D%B0%EC%8A%A4-%ED%94%84%EB%9E%91%EC%8A%A4&amp;action=edit&amp;redlink=1" title="피에르 망데스-프랑스 (없는 문서)">피에르 망데스-프랑스</a>는 <a class="mw-redirect" href="/wiki/1%EC%B0%A8_%EC%9D%B8%EB%8F%84%EC%B0%A8%EC%9D%B4%EB%82%98_%EC%A0%84%EC%9F%81" title="1차 인도차이나 전쟁">1차 인도차이나 전쟁</a>을 종결시키고, <a href="/wiki/%EB%AA%A8%EB%A1%9C%EC%BD%94" title="모로코">모로코</a>와 <a href="/wiki/%ED%8A%80%EB%8B%88%EC%A7%80" title="튀니지">튀니지</a>의 독립을 진행했으나, <a href="/wiki/%EC%95%8C%EC%A0%9C%EB%A6%AC_%EC%A0%84%EC%9F%81" title="알제리 전쟁">알제리 전쟁</a>으로 인해 실각하고 <a class="new" href="/w/index.php?title=%EC%97%90%EB%93%9C%EA%B0%80_%ED%8F%AC%EB%A0%88&amp;action=edit&amp;redlink=1" title="에드가 포레 (없는 문서)">에드가 포레</a>가 집권한다. <a href="/wiki/%ED%94%84%EB%9E%91%EC%8A%A4_%EA%B3%B5%EC%82%B0%EB%8B%B9" title="프랑스 공산당">프랑스 공산당</a>이 지지율 최고의 정당으로 떠오른 가운데, <a href="/wiki/%EB%85%B8%EB%8F%99%EC%9E%90_%EC%9D%B8%ED%84%B0%EB%82%B4%EC%85%94%EB%84%90_%ED%94%84%EB%9E%91%EC%8A%A4_%EC%A7%80%EB%B6%80" title="노동자 인터내셔널 프랑스 지부">노동자 인터내셔널 프랑스 지부</a>, <a class="mw-disambig" href="/wiki/%EA%B8%89%EC%A7%84%EB%8B%B9" title="급진당">급진당</a>, <a class="new" href="/w/index.php?title=%EB%A0%88%EC%A7%80%EC%8A%A4%ED%83%95%EC%8A%A4_%EB%AF%BC%EC%A3%BC%EC%82%AC%ED%9A%8C%EC%97%B0%EB%A7%B9&amp;action=edit&amp;redlink=1" title="레지스탕스 민주사회연맹 (없는 문서)">레지스탕스 민주사회연맹</a>과 <a href="/wiki/%EC%83%A4%EB%A5%BC_%EB%93%9C_%EA%B3%A8" title="샤를 드 골">샤를 드 골</a>과 결별한 <a class="new" href="/w/index.php?title=%EC%82%AC%ED%9A%8C%EA%B3%B5%ED%99%94%EA%B5%AD%EB%AF%BC%EC%A4%91%EC%8B%AC&amp;action=edit&amp;redlink=1" title="사회공화국민중심 (없는 문서)">사회공화국민중심</a> <a class="new" href="/w/index.php?title=%EC%9E%90%ED%81%AC_%EC%83%A4%EB%B0%A9%EB%8D%B8%EB%A7%88%EC%8A%A4&amp;action=edit&amp;redlink=1" title="자크 샤방델마스 (없는 문서)">자크 샤방델마스</a>파는 <a class="new" href="/w/index.php?title=%EA%B3%B5%ED%99%94%EC%A0%84%EC%84%A0&amp;action=edit&amp;redlink=1" title="공화전선 (없는 문서)">공화전선</a>을 구축해 연대를 맺어 알제리 전쟁의 종결을 주장했고, 포레 총리는 급진당을 탈당해 <a class="new" href="/w/index.php?title=%EA%B3%B5%ED%99%94_%EC%A2%8C%ED%8C%8C_%EC%97%B0%EB%8C%80&amp;action=edit&amp;redlink=1" title="공화 좌파 연대 (없는 문서)">공화 좌파 연대</a>에 입당한다.</p>
