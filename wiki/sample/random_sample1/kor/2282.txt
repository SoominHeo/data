<p><a href="/wiki/%EA%B5%90%EC%8B%A4" title="교실">교실</a>과 <a href="/wiki/%EC%B9%A0%ED%8C%90" title="칠판">칠판</a>이 있으며, 대부분 <a href="/wiki/%EC%BB%B4%ED%93%A8%ED%84%B0" title="컴퓨터">컴퓨터</a>로 <a href="/wiki/%EC%88%98%EC%97%85" title="수업">수업</a> 내용과 관련된 내용의 자료를 <a class="new" href="/w/index.php?title=%EB%B9%94%ED%94%84%EB%A1%9C%EC%A0%9D%ED%84%B0&amp;action=edit&amp;redlink=1" title="빔프로젝터 (없는 문서)">빔프로젝터</a>, <a class="mw-redirect" href="/wiki/TV" title="TV">TV</a>를 통해 보여주는 경우가 많다. <a href="/wiki/%ED%95%99%EA%B5%90" title="학교">학교</a>는 <a href="/wiki/%EC%B4%88%EB%93%B1%ED%95%99%EA%B5%90" title="초등학교">초등학교</a>, <a href="/wiki/%EC%A4%91%ED%95%99%EA%B5%90" title="중학교">중학교</a>, <a href="/wiki/%EA%B3%A0%EB%93%B1%ED%95%99%EA%B5%90" title="고등학교">고등학교</a>, <a class="mw-redirect" href="/wiki/%EB%8C%80%ED%95%99%EA%B5%90" title="대학교">대학교</a> 등으로 나뉜다.</p>
