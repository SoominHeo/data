<p><b>교황 그레고리오 8세</b>(<small><a href="/wiki/%EB%9D%BC%ED%8B%B4%EC%96%B4" title="라틴어">라틴어</a>:</small> Gregorius PP. VIII, <small><a href="/wiki/%EC%9D%B4%ED%83%88%EB%A6%AC%EC%95%84%EC%96%B4" title="이탈리아어">이탈리아어</a>:</small> Papa Gregorio VIII, <a href="/wiki/1100%EB%85%84" title="1100년">1100년</a> ~ <a href="/wiki/1187%EB%85%84" title="1187년">1187년</a>)는 제173대 <a href="/wiki/%EA%B5%90%ED%99%A9" title="교황">교황</a>(재위: <a href="/wiki/1187%EB%85%84" title="1187년">1187년</a> <a href="/wiki/10%EC%9B%94_21%EC%9D%BC" title="10월 21일">10월 21일</a> - <a href="/wiki/1187%EB%85%84" title="1187년">1187년</a> <a href="/wiki/12%EC%9B%94_17%EC%9D%BC" title="12월 17일">12월 17일</a>)이다. 본명은 <b>알베르토 디 모라</b>(<small><a href="/wiki/%EC%9D%B4%ED%83%88%EB%A6%AC%EC%95%84%EC%96%B4" title="이탈리아어">이탈리아어</a>:</small> Alberto di Morra)이다.</p>
<h2><span class="mw-headline" id=".EC.83.9D.EC.95.A0">생애</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EA%B5%90%ED%99%A9_%EA%B7%B8%EB%A0%88%EA%B3%A0%EB%A6%AC%EC%98%A4_8%EC%84%B8&amp;action=edit&amp;section=1" title="부분 편집: 생애">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>12세기 초 <a href="/wiki/%EC%9D%B4%ED%83%88%EB%A6%AC%EC%95%84" title="이탈리아">이탈리아</a>의 베네벤토에서 출생한 그는 <a class="mw-redirect" href="/wiki/%EC%95%84%EC%9A%B0%EB%A0%90%EB%A6%AC%EC%9A%B0%EC%8A%A4_%EC%95%84%EC%9A%B0%EA%B5%AC%EC%8A%A4%ED%8B%B0%EB%88%84%EC%8A%A4" title="아우렐리우스 아우구스티누스">성 아우구스티누스</a> 성당의 의전 <a class="mw-redirect" href="/wiki/%EC%8B%A0%EB%B6%80_(%EA%B8%B0%EB%8F%85%EA%B5%90)" title="신부 (기독교)">신부</a>로 생활하다가 1155년 <a href="/wiki/%EA%B5%90%ED%99%A9_%ED%95%98%EB%93%9C%EB%A6%AC%EC%95%84%EB%85%B8_4%EC%84%B8" title="교황 하드리아노 4세">교황 하드리아노 4세</a>에 의해 <a href="/wiki/%EC%B6%94%EA%B8%B0%EA%B2%BD" title="추기경">추기경</a>이 되어 <a href="/wiki/%EB%A1%9C%EB%A7%88_%EA%B5%90%ED%99%A9%EC%B2%AD" title="로마 교황청">로마 교황청</a>의 상서국장이 되었다. 그가 고안한 문체 양식은 교황청의 문서 작성 체계에 사용되었다.</p>
<p>페라라에서 교황으로 선출되자 <a class="mw-redirect" href="/wiki/%EC%8B%A0%EC%84%B1%EB%A1%9C%EB%A7%88%EC%A0%9C%EA%B5%AD" title="신성로마제국">신성로마제국</a>과의 불화는 계속되었다. 그레고리오 8세는 <a href="/wiki/%EA%B5%90%ED%9A%8C" title="교회">교회</a>를 쇄신하고 그 해 10월 2일 <a href="/wiki/%EC%98%88%EB%A3%A8%EC%82%B4%EB%A0%98" title="예루살렘">예루살렘</a>이 <a class="mw-redirect" href="/wiki/%EC%9D%B4%EC%8A%AC%EB%9E%8C" title="이슬람">이슬람</a>에 의해 함락된 후 <a href="/wiki/%EC%A0%9C3%EC%B0%A8_%EC%8B%AD%EC%9E%90%EA%B5%B0" title="제3차 십자군">새로운 십자군</a> 모집을 준비하였다. 57일이라는 짧은 재위기간으로 인해 <a href="/wiki/%EB%A1%9C%EB%A7%88" title="로마">로마</a>에 입성하지 못하였다.</p>
