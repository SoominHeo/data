<div class="mw-content-ltr" dir="ltr" id="mw-content-text" lang="ko"><script>function mfTempOpenSection(id){var block=document.getElementById("mf-section-"+id);block.className+=" open-block";block.previousSibling.className+=" open-block";}</script><p><b>0verflow</b>는 <a href="/wiki/%EB%8F%84%EC%BF%84_%EB%8F%84" title="도쿄 도">도쿄 도</a> <a href="/wiki/%EC%A7%80%EC%9A%94%EB%8B%A4_%EA%B5%AC" title="지요다 구">지요다 구</a> <a class="new" href="/w/index.php?title=%EA%B0%84%EB%8B%A4%EC%98%A4%EA%B0%80%EC%99%80%EB%A7%88%EC%B9%98&amp;action=edit&amp;redlink=1" title="간다오가와마치 (없는 문서)">간다오가와마치</a>(<span lang="ja" xml:lang="ja">神田小川町</span>)에 본사를 두고있는 <a href="/wiki/%EC%9C%A0%ED%95%9C%ED%9A%8C%EC%82%AC" title="유한회사">유한회사</a> <a href="/wiki/%EC%8A%A4%ED%83%9D" title="스택">스택</a>(STACK)이 <a href="/wiki/%EA%B0%9C%EC%9D%B8%EC%9A%A9_%EC%BB%B4%ED%93%A8%ED%84%B0" title="개인용 컴퓨터">PC</a>용 <a href="/wiki/%EC%97%90%EB%A1%9C%EA%B2%8C" title="에로게">어덜트 게임</a> 제작/판매를 목적으로 설립한 브랜드이다. 본 항목에선 모체기업인 유한회사 스택에 대해서도 설명한다.</p>
<p>회사 이름의 영문표기는 <b>Overflow</b>이지만 회사 로고나 홈페이지 도메인 네임에선 첫글자에 <a class="mw-redirect" href="/wiki/%EC%95%8C%ED%8C%8C%EB%B2%B3" title="알파벳">알파벳</a> <b>O</b> 대신에 <a href="/wiki/%EC%88%AB%EC%9E%90" title="숫자">숫자</a> <b>0</b>을 사용한다.</p>
<p>마스코트 캐릭터는「마요쨩」이다.</p>

<h2><span class="mw-headline" id=".EC.9C.A0.ED.95.9C.ED.9A.8C.EC.82.AC_.EC.8A.A4.ED.83.9D">유한회사 스택</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=0verflow&amp;action=edit&amp;section=1" title="부분 편집: 유한회사 스택">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<ul>
<li>1997년 3월 <a class="mw-redirect" href="/wiki/%EC%BB%B4%ED%93%A8%ED%84%B0_%EA%B2%8C%EC%9E%84" title="컴퓨터 게임">컴퓨터 게임</a>∙디지털 애니메이션의 개발하청을 위한 회사로서 설립.</li>

<li>자사 브랜드는 <a href="/wiki/%EC%97%90%EB%A1%9C%EA%B2%8C" title="에로게">어덜트 게임</a>은 오버플로우, 전연령계열은 STACK software.</li>

<h2><span class="mw-headline" id=".EC.97.B0.ED.98.81.2F.EC.86.8C.EA.B0.9C">연혁/소개</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=0verflow&amp;action=edit&amp;section=3" title="부분 편집: 연혁/소개">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<ul>
<li>1999년 자체 제작 <a href="/wiki/%EC%97%90%EB%A1%9C%EA%B2%8C" title="에로게">어덜트 게임</a>의 제작을 위해 오버플로우란 브랜드를 설립하여 라지 퐁퐁으로 데뷔하였다.</li>
</ul>
<ul>
<li>동경 치요다구의 같은 건물에 있는 <a href="/wiki/%EB%8B%88%ED%8A%B8%EB%A1%9C%ED%94%8C%EB%9F%AC%EC%8A%A4" title="니트로플러스">니트로플러스</a>,<a class="new" href="/w/index.php?title=%EC%95%84%EC%A5%AC&amp;action=edit&amp;redlink=1" title="아쥬 (없는 문서)">아쥬</a>와 함께 치요다구 연합(이하, 치요렌)으로 불린다. (초기에는 3회사였으나, 뒤에 <a href="/wiki/%EC%84%9C%EC%BB%A4%EC%8A%A4" title="서커스">서커스</a>가 합류하여 현재는 4회사) 각 회사 간에는 실제로 스탭 간 이동이라든지 상호 협력이 활발하게 이뤄지고 있다.</li>
</ul>
<ul>
<li>과거 치요렌 소속회사 중 니트로플러스가 팬텀, 데몬베인 시리즈, 아쥬가 <a href="/wiki/%EB%84%A4%EA%B0%80_%EB%B0%94%EB%9D%BC%EB%8A%94_%EC%98%81%EC%9B%90" title="네가 바라는 영원">네가 바라는 영원</a>, <a class="new" href="/w/index.php?title=%EB%A7%88%EB%B8%8C%EB%9F%AC%EB%B8%8C&amp;action=edit&amp;redlink=1" title="마브러브 (없는 문서)">마브러브</a>이라는 대표작을 지니고 있을 적 오버플로우만은 두 회사와 비교해서 존재감이 얕았고 내놓을 만한 대표작도 없었다. 그나마 대표작으로 칭할 수 있던 작품이 18금 <a class="mw-redirect" href="/wiki/OVA" title="OVA">OVA</a>화 되었던 <a href="/wiki/%ED%93%A8%EC%96%B4%EB%A9%94%EC%9D%BC" title="퓨어메일">퓨어메일</a>(PureMail)과 <a class="new" href="/w/index.php?title=%EC%97%AC%EB%8F%99%EC%83%9D%EC%9C%BC%EB%A1%9C_%EA%B0%80%EC%9E%90!&amp;action=edit&amp;redlink=1" title="여동생으로 가자! (없는 문서)">여동생으로 가자!</a>정도였으나 두 회사의 대표작과 비교할 수 있을 정도는 아니었다.</li>
</ul>
<ul>
<li>그런 오버플로우가 일약 다른 치요렌 소속 회사들과 어깨를 나란히 할 수 있도록한 작품이 바로 8번째 정규작품인 <a class="mw-redirect mw-disambig" href="/wiki/%EC%8A%A4%EC%BF%A8%EB%8D%B0%EC%9D%B4%EC%A6%88" title="스쿨데이즈">School Days</a>였다. 이전의 게임에선 볼 수 없던 DVD 듀얼 레이어를 활용한 대용량의 풀 애니메이션 스타일의 게임 진행과 게임 발매전 일반적인 삼각관계물일 거라는 전개를 뛰어넘은 파격적인 전개와 베드 엔딩은 많은 유저들의 화제를 모아 <a href="/wiki/%EC%9D%BC%EB%B3%B8" title="일본">일본</a>에서 어덜트 게임 상반기 판매량 2위/년간 판매량 7위에 약 3만 2천장의 판매고를 올리며 일약 주목 브랜드로 뛰어올랐다. 또한 <a class="mw-redirect mw-disambig" href="/wiki/%EC%8A%A4%EC%BF%A8%EB%8D%B0%EC%9D%B4%EC%A6%88" title="스쿨데이즈">School Days</a>가 오버플로우의 대표작으로 자리매김한 것과 더불어 오버플로우 사상 최초로 대중적인 미디어 믹스 전개도 활발히 진행되어, 코믹 연재는 물론이고 <a href="/wiki/2007%EB%85%84" title="2007년">2007년</a> <a href="/wiki/7%EC%9B%94" title="7월">7월</a>에 TV 애니메이션이 제작되어 방영되었으며 <a href="/wiki/2008%EB%85%84" title="2008년">2008년</a> <a href="/wiki/1%EC%9B%94" title="1월">1월</a>에는 오버플로우 제작 게임사상 최초로 가정용 게임기로 이식되어 <a href="/wiki/%ED%94%8C%EB%A0%88%EC%9D%B4%EC%8A%A4%ED%85%8C%EC%9D%B4%EC%85%98_2" title="플레이스테이션 2">플레이스테이션 2</a>로 School Days L×H라는 제목의 이식판이 발매되었다. TV 애니메이션화나 가정용 게임기 이식모두 오버플로우 최초가 된 <a class="mw-redirect mw-disambig" href="/wiki/%EC%8A%A4%EC%BF%A8%EB%8D%B0%EC%9D%B4%EC%A6%88" title="스쿨데이즈">School Days</a>는 오버플로우가 시도하던 풀 애니메이션 형식의 게임을 완성시킨 것 이상의 중요한 의미를 가진 작품이 되었다.</li>
</ul>
