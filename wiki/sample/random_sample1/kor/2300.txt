<p><b>교황 루치오 3세</b>(<small><a href="/wiki/%EB%9D%BC%ED%8B%B4%EC%96%B4" title="라틴어">라틴어</a>:</small> Lucius PP. III, <small><a href="/wiki/%EC%9D%B4%ED%83%88%EB%A6%AC%EC%95%84%EC%96%B4" title="이탈리아어">이탈리아어</a>:</small> Papa Lucio III)는 제171대 <a href="/wiki/%EA%B5%90%ED%99%A9" title="교황">교황</a>(재위: <a href="/wiki/1181%EB%85%84" title="1181년">1181년</a> <a href="/wiki/9%EC%9B%94_1%EC%9D%BC" title="9월 1일">9월 1일</a> - <a href="/wiki/1185%EB%85%84" title="1185년">1185년</a> <a href="/wiki/9%EC%9B%94_25%EC%9D%BC" title="9월 25일">9월 25일</a>)이다. 본명은 <b>우발도 알루친골리</b>(<span style="font-size: smaller;"><a href="/wiki/%EC%9D%B4%ED%83%88%EB%A6%AC%EC%95%84%EC%96%B4" title="이탈리아어">이탈리아어</a>:</span> <span lang="it" xml:lang="it">Ubaldo Allucingoli</span>)이다.</p>
<p>1097년경 <a href="/wiki/%EC%9D%B4%ED%83%88%EB%A6%AC%EC%95%84" title="이탈리아">이탈리아</a>의 루카에서 태어났다. 시스테리시안 수도사가 된 후 <a href="/wiki/%EA%B5%90%ED%99%A9_%EC%9D%B8%EB%85%B8%EC%B2%B8%EC%8B%9C%EC%98%A4_2%EC%84%B8" title="교황 인노첸시오 2세">교황 인노첸시오 2세</a>에 의해 <a href="/wiki/%EC%B6%94%EA%B8%B0%EA%B2%BD" title="추기경">추기경</a>으로 임명되었고 <a href="/wiki/%EA%B5%90%ED%99%A9_%ED%95%98%EB%93%9C%EB%A6%AC%EC%95%84%EB%85%B8_4%EC%84%B8" title="교황 하드리아노 4세">교황 하드리아노 4세</a>에 의해 오스티아와 벨레트리의 영주가 되었다. 1156년 <a href="/wiki/%EC%8B%9C%EC%B9%A0%EB%A6%AC%EC%95%84_%EC%99%95%EA%B5%AD" title="시칠리아 왕국">시칠리아 왕국</a>과 베네벤토 조약을 맺는 데 참여하였다. <a href="/wiki/%EA%B5%90%ED%99%A9_%EC%95%8C%EB%A0%89%EC%82%B0%EB%8D%B0%EB%A5%B4_3%EC%84%B8" title="교황 알렉산데르 3세">교황 알렉산데르 3세</a>의 오른팔이 되어 <a href="/wiki/1177%EB%85%84" title="1177년">1177년</a> <a class="mw-redirect" href="/wiki/%EC%8B%A0%EC%84%B1%EB%A1%9C%EB%A7%88%EC%A0%9C%EA%B5%AD%EC%9D%98_%ED%94%84%EB%A6%AC%EB%93%9C%EB%A6%AC%ED%9E%88_1%EC%84%B8" title="신성로마제국의 프리드리히 1세">신성로마제국의 프리드리히 1세</a>와 <a href="/wiki/%EB%B2%A0%EB%84%A4%EC%B9%98%EC%95%84_%EA%B3%B5%ED%99%94%EA%B5%AD" title="베네치아 공화국">베네치아 공화국</a>의 평화조약을 맺는 데 기여하였다. 전임 교황의 선종 이틀 뒤에 교황으로 선출되었다.</p>
<p>재위 중 어려움이 많았다. 로마인들과 의견 차이가 많았고 <a href="/wiki/1184%EB%85%84" title="1184년">1184년</a> 베로나에서 신성로마제국 황제와의 토론에서 <a class="mw-redirect" href="/wiki/%ED%86%A0%EC%8A%A4%EC%B9%B4%EB%82%98" title="토스카나">토스카나</a>의 영토 분쟁이 해결되지 않았다. 그리고 베네치아 협정 이후에도 <a href="/wiki/%EB%8F%85%EC%9D%BC" title="독일">독일</a>과 <a href="/wiki/%EC%9D%B4%ED%83%88%EB%A6%AC%EC%95%84_%EB%B0%98%EB%8F%84" title="이탈리아 반도">이탈리아 반도</a>에서 <a href="/wiki/%EA%B5%90%ED%9A%8C" title="교회">교회</a>로부터 분리된 자들의 서품이 해결되지 못한 채 남아 있었다. 교황은 프리드리히 황제의 아들(<a class="mw-redirect" href="/wiki/%EC%8B%A0%EC%84%B1%EB%A1%9C%EB%A7%88%EC%A0%9C%EA%B5%AD%EC%9D%98_%ED%95%98%EC%9D%B8%EB%A6%AC%ED%9E%88_6%EC%84%B8" title="신성로마제국의 하인리히 6세">하인리히 6세</a>)의 대관식을 거절하였으나 시칠리아의 콘스탄티아와의 결혼은 반대하지 않은 듯하다. 교황은 <a href="/wiki/%EC%8B%AD%EC%9E%90%EA%B5%B0" title="십자군">십자군</a> 전쟁에 적극적이었다. <a href="/wiki/%EC%98%88%EB%A3%A8%EC%82%B4%EB%A0%98" title="예루살렘">예루살렘</a>의 대표들로부터 위급한 원조 요청을 받고 베로나에서 회의를 소집하여 <a href="/wiki/%EC%A0%9C3%EC%B0%A8_%EC%8B%AD%EC%9E%90%EA%B5%B0" title="제3차 십자군">제3차 십자군</a> 원정을 제안하였다. 그리고 신성로마제국과 협조하여 이단자들을 제거하는 <a class="mw-redirect" href="/wiki/%EC%9D%B4%EB%8B%A8_%EC%8B%AC%EB%AC%B8" title="이단 심문">이단 심문</a>을 열기로 결정하고 여기에 관한 칙서 ‘아드 아볼렌담’(<i>Ad abolendam</i>)을 발표하였다.</p>
