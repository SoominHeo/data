<p><b>교황 마리노 1세</b>(<small><a href="/wiki/%EB%9D%BC%ED%8B%B4%EC%96%B4" title="라틴어">라틴어</a>:</small> Marinus PP. I, <small><a href="/wiki/%EC%9D%B4%ED%83%88%EB%A6%AC%EC%95%84%EC%96%B4" title="이탈리아어">이탈리아어</a>:</small> Papa Marino I)는 제108대 <a href="/wiki/%EA%B5%90%ED%99%A9" title="교황">교황</a>(재위: <a href="/wiki/882%EB%85%84" title="882년">882년</a> <a href="/wiki/12%EC%9B%94_16%EC%9D%BC" title="12월 16일">12월 16일</a> - <a href="/wiki/884%EB%85%84" title="884년">884년</a> <a href="/wiki/5%EC%9B%94_15%EC%9D%BC" title="5월 15일">5월 15일</a>)이다. 문헌에 잘못 기록되어 마르티노 2세로 불리기도 하였다.</p>
<h2><span class="mw-headline" id=".EC.83.9D.EC.95.A0">생애</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EA%B5%90%ED%99%A9_%EB%A7%88%EB%A6%AC%EB%85%B8_1%EC%84%B8&amp;action=edit&amp;section=1" title="부분 편집: 생애">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p><a href="/wiki/%EB%A1%9C%EB%A7%88" title="로마">로마</a> 근처 갈레세에서 사제의 아들로 태어난 마리노는 <a href="/wiki/%EA%B5%90%ED%99%A9_%EB%8B%88%EC%BD%9C%EB%9D%BC%EC%98%A4_1%EC%84%B8" title="교황 니콜라오 1세">교황 니콜라오 1세</a>에 의해 <a href="/wiki/%EB%B6%80%EC%A0%9C_(%EA%B8%B0%EB%8F%85%EA%B5%90)" title="부제 (기독교)">부제</a>로 서품받았다.<sup class="reference" id="cite_ref-McBrien_1-0"><a href="#cite_note-McBrien-1">[1]</a></sup> 교황으로 선출되기 전에 그는 <a class="new" href="/w/index.php?title=%EC%B9%B4%EC%97%90%EB%A0%88&amp;action=edit&amp;redlink=1" title="카에레 (없는 문서)">카에레</a>의 주교였는데, 이것이 그의 교황 선출에 대한 논란을 가져왔다. 왜냐하면 당시만 하더라도 한 교구의 주교가 다른 교구의 주교가 되기 위해 원래 있던 주교직을 버린다는 것은 상상하지도 못했던 일이었기 때문이다. 그의 전임자들이었던 세 명의 교황 모두에게 교황 특사로서 콘스탄티노폴리스에 파견된 그에게 주어진 사명은 <a href="/wiki/%EC%BD%98%EC%8A%A4%ED%83%84%ED%8B%B0%EB%85%B8%ED%8F%B4%EB%A6%AC%EC%8A%A4_%EC%B4%9D%EB%8C%80%EC%A3%BC%EA%B5%90_%ED%8F%AC%ED%8B%B0%EC%98%A4%EC%8A%A4_1%EC%84%B8" title="콘스탄티노폴리스 총대주교 포티오스 1세">포티오스</a>를 단죄하고 <a class="new" href="/w/index.php?title=%EC%BD%98%EC%8A%A4%ED%83%84%ED%8B%B0%EB%85%B8%ED%8F%B4%EB%A6%AC%EC%8A%A4_%EC%B4%9D%EB%8C%80%EC%A3%BC%EA%B5%90_%EC%9D%B4%EA%B7%B8%EB%82%98%ED%8B%B0%EC%98%A4%EC%8A%A4&amp;action=edit&amp;redlink=1" title="콘스탄티노폴리스 총대주교 이그나티오스 (없는 문서)">이그나티오스</a>를 적법한 <a href="/wiki/%EC%BD%98%EC%8A%A4%ED%83%84%ED%8B%B0%EB%85%B8%ED%8F%B4%EB%A6%AC%EC%8A%A4_%EC%B4%9D%EB%8C%80%EC%A3%BC%EA%B5%90" title="콘스탄티노폴리스 총대주교">콘스탄티노폴리스 총대주교</a>로 인정한다고 선언하는 것이었다.<sup class="reference" id="cite_ref-McBrien_1-1"><a href="#cite_note-McBrien-1">[1]</a></sup></p>
<p>마리노 1세는 모든 사람과 두루 좋은 관계를 맺기 위해 노력하였다. 교황으로서 한 그의 첫 번째 활동은 <a class="new" href="/w/index.php?title=%ED%8F%AC%EB%A5%B4%ED%88%AC%EC%8A%A4&amp;action=edit&amp;redlink=1" title="포르투스 (없는 문서)">포르투스</a>의 주교급 <a href="/wiki/%EC%B6%94%EA%B8%B0%EA%B2%BD" title="추기경">추기경</a>이었던 <a href="/wiki/%EA%B5%90%ED%99%A9_%ED%8F%AC%EB%A5%B4%EB%AA%A8%EC%86%8C" title="교황 포르모소">포르모소</a>를 복직시킴과 동시에 포티오스를 단죄한 것이다.<sup class="reference" id="cite_ref-forum_2-0"><a href="#cite_note-forum-2">[2]</a></sup> <a href="/wiki/%EC%95%A8%ED%94%84%EB%A0%88%EB%93%9C_%EB%8C%80%EC%99%95" title="앨프레드 대왕">앨프레드 대왕</a>과 좋은 관계를 맺은 마리노 1세는 <a class="mw-redirect" href="/wiki/%EC%95%B5%EA%B8%80%EB%A1%9C%EC%83%89%EC%8A%A8%EC%A1%B1" title="앵글로색슨족">앵글로색슨족</a>이 그동안 로마에 정기적으로 바치던 공물과 과세를 면제해 주었다.<sup class="reference" id="cite_ref-forum_2-1"><a href="#cite_note-forum-2">[2]</a></sup> 마리노 1세는 884년 5월 또는 6월에 선종하였으며, <a href="/wiki/%EA%B5%90%ED%99%A9_%ED%95%98%EB%93%9C%EB%A6%AC%EC%95%84%EB%85%B8_3%EC%84%B8" title="교황 하드리아노 3세">교황 하드리아노 3세</a>가 뒤를 이었다.</p>
<h2><span class="mw-headline" id=".EC.9D.B4.EB.A6.84.EC.9D.98_.EC.98.A4.EB.A5.98">이름의 오류</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EA%B5%90%ED%99%A9_%EB%A7%88%EB%A6%AC%EB%85%B8_1%EC%84%B8&amp;action=edit&amp;section=2" title="부분 편집: 이름의 오류">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>마리노와 마르티노라는 두 이름의 유사성 때문에, 교황 마리노 1세와 <a href="/wiki/%EA%B5%90%ED%99%A9_%EB%A7%88%EB%A6%AC%EB%85%B8_2%EC%84%B8" title="교황 마리노 2세">교황 마리노 2세</a>는 몇몇 사료에서 실수로 마르티노라는 이름으로 기록되기도 하였다. 그리하여 각각 마르티노 2세와 마르티노 3세라고 잘못 작성되었다. 그 결과, 1281년에 새로 선출되어 실제로 마르티노라는 이름을 선택한 교황은 <a href="/wiki/%EA%B5%90%ED%99%A9_%EB%A7%88%EB%A5%B4%ED%8B%B0%EB%85%B8_4%EC%84%B8" title="교황 마르티노 4세">교황 마르티노 4세</a>가 되고 말았다.</p>
