<p><b>교황 베네딕토 13세</b>(<small><a href="/wiki/%EB%9D%BC%ED%8B%B4%EC%96%B4" title="라틴어">라틴어</a>:</small> <i>Benedictus PP. XIII</i>, <small><a href="/wiki/%EC%9D%B4%ED%83%88%EB%A6%AC%EC%95%84%EC%96%B4" title="이탈리아어">이탈리아어</a>:</small> <i>Papa Benedetto XIII</i>)는 제245대 <a href="/wiki/%EA%B5%90%ED%99%A9" title="교황">교황</a>(재위: <a href="/wiki/1724%EB%85%84" title="1724년">1724년</a> <a href="/wiki/5%EC%9B%94_29%EC%9D%BC" title="5월 29일">5월 29일</a> - <a href="/wiki/1730%EB%85%84" title="1730년">1730년</a> <a href="/wiki/2%EC%9B%94_21%EC%9D%BC" title="2월 21일">2월 21일</a>)이다. 세속명은 <b>피에트로 프란체스코 오르시니</b>(<small><a href="/wiki/%EC%9D%B4%ED%83%88%EB%A6%AC%EC%95%84%EC%96%B4" title="이탈리아어">이탈리아어</a>:</small> <i>Pietro Francesco Orsini</i>)이다.</p>
<h2><span class="mw-headline" id=".EC.83.9D.EC.95.A0">생애</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EA%B5%90%ED%99%A9_%EB%B2%A0%EB%84%A4%EB%94%95%ED%86%A0_13%EC%84%B8&amp;action=edit&amp;section=1" title="부분 편집: 생애">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>1649년 2월 2일 <a href="/wiki/%EC%9D%B4%ED%83%88%EB%A6%AC%EC%95%84" title="이탈리아">이탈리아</a>의 <a class="new" href="/w/index.php?title=%EA%B7%B8%EB%9D%BC%EB%B9%84%EB%82%98%EC%9D%B8%ED%91%B8%EA%B7%B8%EB%A6%AC%EC%95%84&amp;action=edit&amp;redlink=1" title="그라비나인푸그리아 (없는 문서)">그라비나인푸그리아</a>에서 그라비나 <a href="/wiki/%EA%B3%B5%EC%9E%91_(%EC%9E%91%EC%9C%84)" title="공작 (작위)">공작</a>의 아들이자 상속자로 태어났다. 그러나 가족들의 반대에도 불구하고 모든 권리를 포기하여 동생에게 넘겨준 후 <a class="mw-redirect" href="/wiki/%EB%8F%84%EB%AF%B8%EB%8B%88%EC%BD%94%ED%9A%8C" title="도미니코회">도미니코회</a>에 입회하여 <a href="/wiki/%EC%88%98%EC%82%AC_(%EA%B8%B0%EB%8F%85%EA%B5%90)" title="수사 (기독교)">수사</a>가 된 후 학업과 고행에 힘썼다. <a href="/wiki/%EB%82%98%ED%8F%B4%EB%A6%AC" title="나폴리">나폴리</a>, <a href="/wiki/%EB%B3%BC%EB%A1%9C%EB%83%90" title="볼로냐">볼로냐</a>, <a href="/wiki/%EB%B2%A0%EB%84%A4%EC%B9%98%EC%95%84" title="베네치아">베네치아</a>에서 <a href="/wiki/%EC%B2%A0%ED%95%99" title="철학">철학</a>과 <a href="/wiki/%EC%8B%A0%ED%95%99" title="신학">신학</a>을 공부한 후 <a href="/wiki/%EB%B8%8C%EB%A0%88%EC%8B%9C%EC%95%84" title="브레시아">브레시아</a>에서 교사가 되었다. 비록 세속을 떠났으나 가문의 영향으로 23세에 <a href="/wiki/%EC%B6%94%EA%B8%B0%EA%B2%BD" title="추기경">추기경</a>이 되고 3년 후 <a class="new" href="/w/index.php?title=%EB%A7%8C%ED%94%84%EB%A0%88%EB%8F%84%EB%8B%88%EC%95%84&amp;action=edit&amp;redlink=1" title="만프레도니아 (없는 문서)">만프레도니아</a>, <a href="/wiki/%EC%B2%B4%EC%84%B8%EB%82%98" title="체세나">체세나</a>, <a href="/wiki/%EB%B2%A0%EB%84%A4%EB%B2%A4%ED%86%A0" title="베네벤토">베네벤토</a>의 주교령을 소유하게 되었다. 1724년 2개월간 지속된 <a href="/wiki/%EC%BD%98%ED%81%B4%EB%9D%BC%EB%B2%A0" title="콘클라베">콘클라베</a>에서 교황으로 선출되어 베네딕토 13세로 명명하였다.</p>
<p>베네딕토 13세는 신앙적으로는 뛰어났으나 행정과 외교 면에서는 부족하였다. 그러나 얀센파에 대해서는 강경한 태도를 취하였다. 수도회의 발전을 촉진시키고 1725년에는 로마 지방 공의회를 개최하였다. 베네딕토 13세는 사르데냐와 사보이의 대공들과의 친선 관계를 회복하였으나 <a href="/wiki/%ED%8F%AC%EB%A5%B4%ED%88%AC%EA%B0%88_%EC%99%95%EA%B5%AD" title="포르투갈 왕국">포르투갈 왕국</a>과는 관계가 좋지 않았다.</p>
