<p><b>교황 알렉산데르 2세</b>(<small><a href="/wiki/%EB%9D%BC%ED%8B%B4%EC%96%B4" title="라틴어">라틴어</a>:</small> Alexander PP. II, <small><a href="/wiki/%EC%9D%B4%ED%83%88%EB%A6%AC%EC%95%84%EC%96%B4" title="이탈리아어">이탈리아어</a>:</small> Papa Alessandro II)는 제156대 <a href="/wiki/%EA%B5%90%ED%99%A9" title="교황">교황</a>(재위: <a href="/wiki/1061%EB%85%84" title="1061년">1061년</a> <a href="/wiki/9%EC%9B%94_30%EC%9D%BC" title="9월 30일">9월 30일</a> - <a href="/wiki/1073%EB%85%84" title="1073년">1073년</a> <a href="/wiki/4%EC%9B%94_21%EC%9D%BC" title="4월 21일">4월 21일</a>)이다. 본명은 <b>안셀모 다 바지오</b>(<span style="font-size: smaller;"><a href="/wiki/%EC%9D%B4%ED%83%88%EB%A6%AC%EC%95%84%EC%96%B4" title="이탈리아어">이탈리아어</a>:</span> <span lang="it" xml:lang="it">Anselmo da Baggio</span>)이다.<sup class="reference" id="cite_ref-1"><a href="#cite_note-1">[1]</a></sup></p>
<h2><span class="mw-headline" id=".EC.83.9D.EC.95.A0">생애</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EA%B5%90%ED%99%A9_%EC%95%8C%EB%A0%89%EC%82%B0%EB%8D%B0%EB%A5%B4_2%EC%84%B8&amp;action=edit&amp;section=1" title="부분 편집: 생애">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>안셀모는 <a href="/wiki/%EB%B0%80%EB%9D%BC%EB%85%B8" title="밀라노">밀라노</a>의 귀족 가문에서 태어났다.<sup class="reference" id="cite_ref-Loughlin_2-0"><a href="#cite_note-Loughlin-2">[2]</a></sup> 1055년 신학 교육을 받아 <a href="/wiki/%EC%82%AC%EC%A0%9C" title="사제">사제</a>로 서품된 그는 1057년에는 <a href="/wiki/%EB%A3%A8%EC%B9%B4_(%ED%86%A0%EC%8A%A4%EC%B9%B4%EB%82%98_%EC%A3%BC)" title="루카 (토스카나 주)">루카</a>의 <a href="/wiki/%EC%A3%BC%EA%B5%90" title="주교">주교</a>로 서임되었다. 루카의 주교가 된 그는 소바나의 일데브란도의 보좌를 받으며 성직매매를 억제하고 성직자들의 독신 생활을 강화하기 위해 노력하였다. 1059년 교황 니콜라오 2세가 반포한 칙령에 의거하여 1061년 치루어진 교황 선거는 <a href="/wiki/%EC%8B%A0%EC%84%B1_%EB%A1%9C%EB%A7%88_%EC%A0%9C%EA%B5%AD" title="신성 로마 제국">신성 로마 제국</a> 황실의 재가를 받지 않았다. 그러자 신성 로마 제국은 예전처럼 후임 교황 후보로 <a href="/wiki/%ED%8C%8C%EB%A5%B4%EB%A7%88" title="파르마">파르마</a>의 주교 카달루스를 지명하였다. <a href="/wiki/%EB%B0%94%EC%A0%A4" title="바젤">바젤</a>에 소집된 교회회의에서 카달루스는 <a class="new" href="/w/index.php?title=%EB%8C%80%EB%A6%BD_%EA%B5%90%ED%99%A9_%ED%98%B8%EB%85%B8%EB%A6%AC%EC%98%A4_2%EC%84%B8&amp;action=edit&amp;redlink=1" title="대립 교황 호노리오 2세 (없는 문서)">호노리오 2세</a>라는 이름으로 교황으로 선언되었다. 그는 로마로 가서 오랫동안 알렉산데르 2세의 자리를 위협하였다. 그러나 결국에 호노리오 2세는 신성 로마 제국으로부터 버림을 받아 1064년 <a href="/wiki/%EB%A7%8C%ED%86%A0%EB%B0%94" title="만토바">만토바</a>에 소집된 교회회의에서 대립 교황으로 단죄받았다. 그리하여 알렉산데르 2세는 정통 교황으로 인정받아 그대로 자리를 보존할 수 있었다.</p>
<p>1065년 알렉산데르 2세는 나르본 자작 베렝가르와 나르본 교구장 귀프레드 주교에게 서신을 보내, 그들이 <a href="/wiki/%EC%9C%A0%EB%8C%80%EC%9D%B8" title="유대인">유대인</a> 학살을 막은 것에 대해 칭찬하면서 하느님은 유혈사태를 바라지 않는다는 사실을 상기시켜 주었다.<sup class="reference" id="cite_ref-3"><a href="#cite_note-3">[3]</a></sup> 같은 해에 그는 <a class="new" href="/w/index.php?title=%EB%B2%A0%EB%84%A4%EB%B2%A4%ED%86%A0%EC%9D%98_%EB%9E%80%EB%91%98%ED%94%84_6%EC%84%B8&amp;action=edit&amp;redlink=1" title="베네벤토의 란둘프 6세 (없는 문서)">베네벤토의 란둘프 6세</a> 공작에게 유대인을 강제로 개종시키려고 해서는 안 된다고 훈계하는 한편 스페인을 침공한 <a href="/wiki/%EB%AC%B4%EC%96%B4%EC%9D%B8" title="무어인">무어인</a>들을 몰아낼 것을 요청하였다.<sup class="reference" id="cite_ref-4"><a href="#cite_note-4">[4]</a></sup><sup class="reference" id="cite_ref-5"><a href="#cite_note-5">[5]</a></sup></p>
<p>1066년 알렉산데르 2세는 훗날 정복왕 윌리엄이라는 이름으로 널리 알려진 <a href="/wiki/%EC%9C%8C%EB%A6%AC%EC%97%84_1%EC%84%B8_(%EC%9E%89%EA%B8%80%EB%9E%9C%EB%93%9C)" title="윌리엄 1세 (잉글랜드)">기욤 드 바타르</a>의 사절단을 맞이하였다. 당시 <a href="/wiki/%EB%B8%8C%EB%A5%B4%ED%83%80%EB%89%B4" title="브르타뉴">브르타뉴</a> 정복을 성공적으로 마친 기욤이 로마에 사절단을 보낸 목적은 향후 있을 <a href="/wiki/%EC%9E%89%EA%B8%80%EB%9E%9C%EB%93%9C" title="잉글랜드">잉글랜드</a> 정복에 대해 교황의 축복을 받고자 한 것이었다. 알렉산데르 2세는 사절단에게 <a href="/wiki/%EB%B2%A0%EB%93%9C%EB%A1%9C" title="베드로">성 베드로</a>의 그림이 있는 깃발과 잉글랜드 성직자들에게 보낼 교황 칙서를 건네 주었다. 당시 칙서에 적힌 내용은 기욤이 잉글랜드 왕으로 즉위하는 것을 승인하며, 이를 축복한다는 것이었다. 이 같은 교황의 지지는 헤이스팅스 전투 이후 잉글랜드 교회의 항복을 이끌어내는데 중요한 요소가 되었다. 알렉산데르 2세는 잉글랜드의 색슨 왕가 말기에 중요한 네 교구가 불법적으로 설정되어 성직자들의 생활이 문란하고 지적 수준이 낮았으므로 기욤에게 사절을 보내어 왕위에 올라 개혁을 추진하게 하였다. 또한 <a class="new" href="/w/index.php?title=%EC%BA%94%ED%84%B0%EB%B2%A0%EB%A6%AC%EC%9D%98_%EB%8C%80%EC%A3%BC%EA%B5%90&amp;action=edit&amp;redlink=1" title="캔터베리의 대주교 (없는 문서)">캔터베리의 대주교</a>를 성직매매를 통해 점거한 스키건드 대주교를 파직시키고 개혁가이며 베크 수도원의 원장을 역임하고 캉 수도원의 아바스였던 파비아의 랑프랑을 후임자로 임명하였다.</p>
<p>알렉산데르 2세는 라틴 교회의 경우 <a class="mw-redirect" href="/wiki/%EC%82%AC%EC%88%9C_%EC%8B%9C%EA%B8%B0" title="사순 시기">사순 시기</a> 동안 알렐루야를 부르지 않도록 지시하였다. 여기에 더해 이후 전통 전례에서는 <a class="mw-redirect" href="/wiki/%EB%8C%80%EB%A6%BC_%EC%8B%9C%EA%B8%B0" title="대림 시기">대림 시기</a> 동안에도 알렐루야를 부르지 않게 되었다.</p>
<p>알렉산데르 2세의 뒤를 이어 그를 최측근에서 보좌한 일데브란도가 <a href="/wiki/%EA%B5%90%ED%99%A9_%EA%B7%B8%EB%A0%88%EA%B3%A0%EB%A6%AC%EC%98%A4_7%EC%84%B8" title="교황 그레고리오 7세">그레고리오 7세</a>라는 이름으로 교황좌에 착좌하였다.</p>
