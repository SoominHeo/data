<p><b>교황 클레멘스 8세</b>(<span style="font-size: smaller;"><a href="/wiki/%EB%9D%BC%ED%8B%B4%EC%96%B4" title="라틴어">라틴어</a>:</span> <span lang="la" xml:lang="la"><i>Clemens PP. VIII</i></span>, <span style="font-size: smaller;"><a href="/wiki/%EC%9D%B4%ED%83%88%EB%A6%AC%EC%95%84%EC%96%B4" title="이탈리아어">이탈리아어</a>:</span> <span lang="it" xml:lang="it"><i>Papa Clemente VIII</i></span>)는 제231대 <a href="/wiki/%EA%B5%90%ED%99%A9" title="교황">교황</a>(재위: <a href="/wiki/1592%EB%85%84" title="1592년">1592년</a> <a href="/wiki/1%EC%9B%94_30%EC%9D%BC" title="1월 30일">1월 30일</a>~<a href="/wiki/1605%EB%85%84" title="1605년">1605년</a> <a href="/wiki/3%EC%9B%94_3%EC%9D%BC" title="3월 3일">3월 3일</a>)이다. 세속명은 <b>이폴리토 알도브란디니</b>(<span style="font-size: smaller;"><a href="/wiki/%EC%9D%B4%ED%83%88%EB%A6%AC%EC%95%84%EC%96%B4" title="이탈리아어">이탈리아어</a>:</span> <span lang="it" xml:lang="it"><i>Ippolito Aldobrandini</i></span>)이다.</p>
<h2><span class="mw-headline" id=".EC.83.9D.EC.95.A0">생애</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EA%B5%90%ED%99%A9_%ED%81%B4%EB%A0%88%EB%A9%98%EC%8A%A4_8%EC%84%B8&amp;action=edit&amp;section=1" title="부분 편집: 생애">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>1536년 2월 24일 <a href="/wiki/%EC%9D%B4%ED%83%88%EB%A6%AC%EC%95%84" title="이탈리아">이탈리아</a>의 <a href="/wiki/%ED%8C%8C%EB%8F%84%EB%B0%94" title="파도바">파도바</a>에서 태어나 <a href="/wiki/%ED%8E%98%EB%A3%A8%EC%9E%90" title="페루자">페루자</a>, <a href="/wiki/%EB%B3%BC%EB%A1%9C%EB%83%90" title="볼로냐">볼로냐</a> 등지에서 법학을 공부한 후 박사 학위를 받고 <a href="/wiki/%EA%B5%90%ED%99%A9_%EB%B9%84%EC%98%A4_5%EC%84%B8" title="교황 비오 5세">교황 비오 5세</a> 밑에서 일하였고 <a class="mw-redirect" href="/wiki/%EA%B5%90%ED%99%A9%EC%B2%AD" title="교황청">교황청</a>에서 여러 직책을 맡았다.</p>
<p>일생 동안 성 필립보 네리와 친분을 맺었으므로 열심하고 윤리적으로 안정된 생활을 하였다. 교황이 된 다음에도 친절하고 사랑을 실천하여 주위 사람들에게 좋은 인상을 심어 주었다. <a href="/wiki/%EB%A1%9C%EB%A7%88" title="로마">로마</a>의 여러 성당을 순방하고 가난하고 병든 사람들과 감옥의 죄수들을 찾아가 만나보았다. 그러나 클레멘스 8세의 주요 관심사는 <a href="/wiki/%EA%B5%90%ED%9A%8C" title="교회">교회</a>의 쇄신이었으므로 교회의 영적 생활을 꾀하고 남용과 부정을 근절하려고 노력하였다.</p>
<p>클레멘스 8세는 친<a class="mw-redirect" href="/wiki/%EC%97%90%EC%8A%A4%ED%8C%8C%EB%83%90" title="에스파냐">에스파냐</a> 정책을 포기하고 종교 전쟁에서 <a href="/wiki/%ED%94%84%EB%9E%91%EC%8A%A4" title="프랑스">프랑스</a>의 앙리 4세를 용서하여 그를 프랑스의 합법적인 왕으로 인정하였다. 그러자 본래 <a href="/wiki/%EA%B0%9C%EC%8B%A0%EA%B5%90" title="개신교">개신교</a> 신자였던 앙리 4세가 <a class="mw-redirect" href="/wiki/%EB%A1%9C%EB%A7%88_%EA%B0%80%ED%86%A8%EB%A6%AD" title="로마 가톨릭">로마 가톨릭</a> 신자로 돌아섰다. 이 화해는 1598년 낭트의 칙령으로 이어졌다. 그리고 클레멘스 8세는 <a href="/wiki/%ED%8A%B8%EB%A6%AC%EC%97%94%ED%8A%B8_%EA%B3%B5%EC%9D%98%ED%9A%8C" title="트리엔트 공의회">트리엔트 공의회</a>의 이상을 프랑스에 실천하게 하여 프랑스 교회를 쇄신시켜 나갔으며 외교적인 노력으로 프랑스와 에스파냐, 프랑스와 사보이 사이의 불편한 관계를 평화스럽게 해결하였다.</p>
<p>클레멘스 8세는 <a href="/wiki/%EC%98%81%EA%B5%AD" title="영국">영국</a>과도 관계를 정상화하려고 대륙에 있던 영국 기숙사와 로마에 있던 <a href="/wiki/%EC%8A%A4%EC%BD%94%ED%8B%80%EB%9E%9C%EB%93%9C" title="스코틀랜드">스코틀랜드</a> 기숙사를 지원하는 등 노력을 하였으나 좋은 결과를 내지 못하였다.</p>
<p>클레멘스 8세는 <a href="/wiki/%EC%A0%9C%EB%84%A4%EB%B0%94" title="제네바">제네바</a>의 성 프란체스코 드 살의 활동을 지원하고 <a href="/wiki/%ED%8F%B4%EB%9E%80%EB%93%9C" title="폴란드">폴란드</a>와 <a href="/wiki/%EB%8F%85%EC%9D%BC" title="독일">독일</a>에서도 로마 가톨릭 교회의 개혁을 추진해 나갔다. <a href="/wiki/%EC%98%A4%EC%8A%A4%EB%A7%8C_%EC%A0%9C%EA%B5%AD" title="오스만 제국">오스만 제국</a>에 대항하기 위한 로마 가톨릭 군주들의 연합은 실패로 끝났다. 클레멘스 8세는 외국 선교를 위한 위원회를 창설하였는데 이는 후에 교황청 포교성성의 전신이 되었다.</p>
<p>클레멘스 8세는 교회 내적으로는 <a href="/wiki/%ED%8A%B8%EB%A6%AC%EC%97%94%ED%8A%B8_%EA%B3%B5%EC%9D%98%ED%9A%8C" title="트리엔트 공의회">트리엔트 공의회</a>의 이상을 모든 수도회와 성직자들이 실천하게 하였다. <a class="mw-redirect" href="/wiki/%EB%B6%88%EA%B0%80%ED%83%80_%EC%84%B1%EC%84%9C" title="불가타 성서">불가타 성서</a>의 새 개정판을 출판하게 하고 각종 예식서와 성무 일도 그리고 <a href="/wiki/%EB%AF%B8%EC%82%AC" title="미사">미사</a> 경본을 새로 출판하였다. <a href="/wiki/%EB%A1%9C%EB%B2%A0%EB%A5%B4%ED%86%A0_%EB%B2%A8%EB%9D%BC%EB%A5%B4%EB%AF%B8%EB%85%B8" title="로베르토 벨라르미노">로베르토 벨라르미노</a>를 포함한 여러 명의 추기경들을 새로 임명하였는데 그 중에는 그의 조카 두 명도 포함되었다. 이 시기에 루이스 데 몰리나가 <a href="/wiki/%ED%95%98%EB%8A%90%EB%8B%98" title="하느님">하느님</a>의 은총에 대한 신학적인 주장을 제기하여 <a href="/wiki/%EC%98%88%EC%88%98%ED%9A%8C" title="예수회">예수회</a>외 <a class="mw-redirect" href="/wiki/%EB%8F%84%EB%AF%B8%EB%8B%88%EC%BD%94%ED%9A%8C" title="도미니코회">도미니코회</a> 사이에 일어났던 신학적 논쟁을 조사하기 위한 위원회가 발족되었다.</p>
