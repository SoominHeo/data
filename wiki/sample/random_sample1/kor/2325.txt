<p><b>구 독일 동부 영토</b>(舊 獨逸 東部 領土, <span style="font-size: smaller;"><a href="/wiki/%EB%8F%85%EC%9D%BC%EC%96%B4" title="독일어">독일어</a>:</span> <span lang="de" xml:lang="de">Ehemalige deutsche Ostgebiete</span>)는 <a href="/wiki/%EC%A0%9C1%EC%B0%A8_%EC%84%B8%EA%B3%84_%EB%8C%80%EC%A0%84" title="제1차 세계 대전">제1차 세계 대전</a>과 <a href="/wiki/%EC%A0%9C2%EC%B0%A8_%EC%84%B8%EA%B3%84_%EB%8C%80%EC%A0%84" title="제2차 세계 대전">제2차 세계 대전</a> 이후에 <a href="/wiki/%EC%86%8C%EB%A0%A8" title="소련">소련</a>에 의해 <a href="/wiki/%EB%8F%85%EC%9D%BC" title="독일">독일</a>에서 떨어져 나간 동쪽 영토를 말한다. 이 기간 동안에 독일이 잃은 영토는 <a href="/wiki/%EB%8F%99%ED%94%84%EB%A1%9C%EC%9D%B4%EC%84%BC" title="동프로이센">동프로이센</a>, <a class="mw-disambig" href="/wiki/%EC%8A%90%EB%A0%88%EC%8A%A4%EB%B9%84%ED%9E%88" title="슐레스비히">슐레스비히</a>, <a href="/wiki/%ED%81%B4%EB%9D%BC%EC%9D%B4%ED%8E%98%EB%8B%A4_%EC%A7%80%EC%97%AD" title="클라이페다 지역">클라이페다 지역</a>, <a href="/wiki/%EC%8B%A4%EB%A0%88%EC%8B%9C%EC%95%84" title="실레시아">실레시아</a> 등이다. 구 독일 동부 영토의 대부분을 취득한 폴란드에서는 <a href="/wiki/%ED%94%BC%EC%95%84%EC%8A%A4%ED%8A%B8_%EC%99%95%EC%A1%B0" title="피아스트 왕조">피아스트 왕조</a>의 옛땅이라는 뜻에서 이 지역을 <a class="new" href="/w/index.php?title=%EC%88%98%EB%B3%B5%EC%A7%80%EA%B5%AC_(%ED%8F%B4%EB%9E%80%EB%93%9C)&amp;action=edit&amp;redlink=1" title="수복지구 (폴란드) (없는 문서)">수복지구</a>(Ziemie Odzyskane)라고 불렀다.</p>
<p><a href="/wiki/%EC%84%9C%EB%8F%85" title="서독">서독</a>은 <a href="/wiki/%EC%98%A4%EB%8D%B0%EB%A5%B4-%EB%82%98%EC%9D%B4%EC%84%B8_%EC%84%A0" title="오데르-나이세 선">오데르-나이세 선</a> 이동 지역을 계속 독일의 영토로 보았지만 <a href="/wiki/%EB%8F%85%EC%9D%BC%EC%9D%98_%EC%9E%AC%ED%86%B5%EC%9D%BC" title="독일의 재통일">독일의 재통일</a> 과정에서 <a href="/wiki/2%2B4_%EC%A1%B0%EC%95%BD" title="2+4 조약">2+4 조약</a>과 <a class="new" href="/w/index.php?title=%EB%8F%85%EC%9D%BC-%ED%8F%B4%EB%9E%80%EB%93%9C_%EA%B5%AD%EA%B2%BD_%EC%A1%B0%EC%95%BD_(1990%EB%85%84)&amp;action=edit&amp;redlink=1" title="독일-폴란드 국경 조약 (1990년) (없는 문서)">독일-폴란드 국경 조약</a>을 통해 이지역의 영유권을 포기했다.</p>
