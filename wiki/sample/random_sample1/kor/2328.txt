<p><b>구글 TV</b>(Google TV)는 <a href="/wiki/%EC%95%88%EB%93%9C%EB%A1%9C%EC%9D%B4%EB%93%9C_(%EC%9A%B4%EC%98%81_%EC%B2%B4%EC%A0%9C)" title="안드로이드 (운영 체제)">안드로이드</a> <a href="/wiki/%EC%9A%B4%EC%98%81_%EC%B2%B4%EC%A0%9C" title="운영 체제">운영 체제</a>에 기반을 둔 <a href="/wiki/%EA%B3%A0%EC%84%A0%EB%AA%85_%ED%85%94%EB%A0%88%EB%B9%84%EC%A0%84" title="고선명 텔레비전">HDTV</a>, <a href="/wiki/%EC%85%8B%ED%86%B1%EB%B0%95%EC%8A%A4" title="셋톱박스">셋톱박스</a> 등을 위한 서비스이다. <a href="/wiki/%EA%B5%AC%EA%B8%80" title="구글">구글</a>이 개발했으며 이 프로젝트는 <a href="/wiki/2010%EB%85%84" title="2010년">2010년</a> <a href="/wiki/5%EC%9B%94_20%EC%9D%BC" title="5월 20일">5월 20일</a> 구글과 파트너들이 구글 I/O에서 공식 공표했다.</p>

<ul>
<li><a href="/wiki/2012%EB%85%84" title="2012년">2012년</a> <a href="/wiki/10%EC%9B%94_16%EC%9D%BC" title="10월 16일">10월 16일</a> : <a href="/wiki/%EB%8C%80%ED%95%9C%EB%AF%BC%EA%B5%AD" title="대한민국">대한민국</a>에서는 <a href="/wiki/LG%EC%9C%A0%ED%94%8C%EB%9F%AC%EC%8A%A4" title="LG유플러스">LG유플러스</a>가 구글과 합작해 만든 'u+tv G'를 발표/출시했다.<sup class="reference" id="cite_ref-3"><a href="#cite_note-3">[3]</a></sup><sup class="reference" id="cite_ref-4"><a href="#cite_note-4">[4]</a></sup></li>
<li><a href="/wiki/2013%EB%85%84" title="2013년">2013년</a> <a href="/wiki/1%EC%9B%94_9%EC%9D%BC" title="1월 9일">1월 9일</a> : LG유플러스는 IPTV와 구글TV를 하나로 묶은 'u+tv G' 서비스가 두달 반만에 10만 가입자를 돌파했다고 9일 밝혔다.<sup class="reference" id="cite_ref-5"><a href="#cite_note-5">[5]</a></sup></li>
<li><a href="/wiki/2014%EB%85%84" title="2014년">2014년</a> <a href="/wiki/9%EC%9B%94_30%EC%9D%BC" title="9월 30일">9월 30일</a> : LG유플러스가 4k UHD 방송을 제공하는 세계최초의 쿼드코어 UHD IPTV 'U+tv G 4K UHD'를 출시했다.</li>
</ul>
