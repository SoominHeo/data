<p><b>구글 문서도구</b>(Google Docs)는 <a href="/wiki/%EA%B5%AC%EA%B8%80" title="구글">구글</a>의 <a class="mw-disambig" href="/wiki/%EC%9B%B9" title="웹">웹</a> 기반 서비스로, <a href="/wiki/%EC%9B%8C%EB%93%9C_%ED%94%84%EB%A1%9C%EC%84%B8%EC%84%9C" title="워드 프로세서">워드 프로세서</a>, <a href="/wiki/%EC%8A%A4%ED%94%84%EB%A0%88%EB%93%9C%EC%8B%9C%ED%8A%B8" title="스프레드시트">스프레드시트</a>, <a href="/wiki/%ED%94%84%EB%A0%88%EC%A0%A0%ED%85%8C%EC%9D%B4%EC%85%98_%EC%86%8C%ED%94%84%ED%8A%B8%EC%9B%A8%EC%96%B4" title="프레젠테이션 소프트웨어">프레젠테이션</a>, 그림 등 기능이 제공된다. 2012년 4월 <a href="/wiki/%EA%B5%AC%EA%B8%80_%EB%93%9C%EB%9D%BC%EC%9D%B4%EB%B8%8C" title="구글 드라이브">구글 드라이브</a>로 통합되었다.</p>

<h2><span class="mw-headline" id=".EC.97.AD.EC.82.AC">역사</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EA%B5%AC%EA%B8%80_%EB%AC%B8%EC%84%9C%EB%8F%84%EA%B5%AC&amp;action=edit&amp;section=1" title="부분 편집: 역사">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>구글 문서도구는 여러 차례 개명과 기능 추가를 거쳤다.</p>
<p>2005년 8월 소프트웨어 개발사인 업스타틀(Upstartle)은 라이틀리(Writely)라는 온라인 서비스를 개시하였다. 기존 오프라인 기반 워드프로세서와 차별화되는 공동 문서 편집 기능 및 공유 기능을 통해 영어권 사용자들을 끌어들였다.</p>
<p>구글은 라이틀리의 개발사를 2006년 3월 9일에 인수하였으며 당시 자체 베타 서비스 중이던 구글 스프레드시트 서비스와 통합하여 2006년 10월 구글 문서와 스프레드시트 (Google Docs &amp; Spreadsheets)를 개시하였다.</p>
<p>2007년 9월 17일, 토닉 시스템즈 (Tonic Systems)가 개발한 온라인 프레젠테이션 서비스를 구글에서 인수하여 워드프로세서와 스프레드시트와 함께 프레젠테이션 제작 서비스를 할 수 있게 되었다. 이에 맞춰 구글 문서와 스프레드시트의 영문 명칭은 구글 독스(Google Docs)로 바뀌었으며, 한글 명칭은 구글 오피스로 개명되었다. 현재는 구글 문서도구로 바뀌었다.</p>
<p><a href="/wiki/2009%EB%85%84" title="2009년">2009년</a> 4월에 그림 삽입 기능이 추가되어 <a class="mw-redirect" href="/wiki/SVG" title="SVG">SVG</a> 포맷의 <a href="/wiki/%EB%B2%A1%ED%84%B0_%EA%B7%B8%EB%9E%98%ED%94%BD%EC%8A%A4" title="벡터 그래픽스">벡터</a> 그래픽을 그려 넣을 수 있게 되었다.</p>
<h2><span class="mw-headline" id=".EA.B8.B0.EB.8A.A5">기능</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EA%B5%AC%EA%B8%80_%EB%AC%B8%EC%84%9C%EB%8F%84%EA%B5%AC&amp;action=edit&amp;section=2" title="부분 편집: 기능">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<dl>
<dt>문서</dt>
</dl>
<ul>
<li><a class="mw-redirect" href="/wiki/WYSIWYG" title="WYSIWYG">WYSIWYG</a> 방식으로 문서 작성이 가능하다. 지원하는 문서 포맷은 <a href="/wiki/%EB%A7%88%EC%9D%B4%ED%81%AC%EB%A1%9C%EC%86%8C%ED%94%84%ED%8A%B8_%EC%9B%8C%EB%93%9C" title="마이크로소프트 워드">마이크로소프트 워드</a>(.doc), <a class="mw-redirect" href="/wiki/%EC%84%9C%EC%8B%9D%EC%9E%88%EB%8A%94_%ED%85%8D%EC%8A%A4%ED%8A%B8_%ED%8F%AC%EB%A7%B7" title="서식있는 텍스트 포맷">서식있는 텍스트</a>(.rtf), <a href="/wiki/%EC%98%A4%ED%94%88%EB%8F%84%ED%81%90%EB%A8%BC%ED%8A%B8" title="오픈도큐먼트">오픈도큐먼트</a>(.odt), <a href="/wiki/PDF" title="PDF">PDF</a>(.pdf), 텍스트(.txt), <a href="/wiki/HTML" title="HTML">HTML</a>(.html)이다.</li>
<li><a href="/wiki/API" title="API">API</a>를 이용하여 <a href="/wiki/%EB%B8%94%EB%A1%9C%EA%B7%B8" title="블로그">블로그</a>에 직접 글을 올릴 수 있다.</li>
</ul>
<dl>
<dt>스프레드시트</dt>
</dl>
<ul>
<li>지원하는 문서 포맷은 <a href="/wiki/%EB%A7%88%EC%9D%B4%ED%81%AC%EB%A1%9C%EC%86%8C%ED%94%84%ED%8A%B8_%EC%97%91%EC%85%80" title="마이크로소프트 엑셀">마이크로소프트 엑셀</a>(.xls), CSV(.csv), <a href="/wiki/%EC%98%A4%ED%94%88%EB%8F%84%ED%81%90%EB%A8%BC%ED%8A%B8" title="오픈도큐먼트">오픈도큐먼트</a>(.ods), <a href="/wiki/PDF" title="PDF">PDF</a>(.pdf), 텍스트(.txt), <a href="/wiki/HTML" title="HTML">HTML</a>(.html)이다.</li>
<li>지원하는 크기는 1만행, 256열, 10만 개 셀, 40시트 이내. 동시에 열 수 있는 스프레드시트는 11개이다.</li>
</ul>
<dl>
<dt>프레젠테이션</dt>
</dl>
<ul>
<li><a class="mw-disambig" href="/wiki/ZIP" title="ZIP">ZIP</a> 포맷으로 압축하여 다운로드가 가능하다. 지원하는 문서 포맷은 <a href="/wiki/%EB%A7%88%EC%9D%B4%ED%81%AC%EB%A1%9C%EC%86%8C%ED%94%84%ED%8A%B8_%ED%8C%8C%EC%9B%8C%ED%8F%AC%EC%9D%B8%ED%8A%B8" title="마이크로소프트 파워포인트">마이크로소프트 파워포인트</a>(.ppt). 초기에는 읽을 수만 있었으나 <a href="/wiki/2008%EB%85%84" title="2008년">2008년</a> 4월부터 ppt 파일로 저장도 가능해졌다.</li>
</ul>
