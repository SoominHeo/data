<p><b>구레 항 공습</b>(<span lang="ja" xml:lang="ja">呉軍港空襲</span>, <span lang="en" xml:lang="en">Bombing of Kure</span>)은 <a href="/wiki/%EC%A0%9C2%EC%B0%A8_%EC%84%B8%EA%B3%84_%EB%8C%80%EC%A0%84" title="제2차 세계 대전">제2차 세계 대전</a> 말기인 <a href="/wiki/1945%EB%85%84" title="1945년">1945년</a> <a href="/wiki/3%EC%9B%94_19%EC%9D%BC" title="3월 19일">3월 19일</a>, <a href="/wiki/7%EC%9B%94_24%EC%9D%BC" title="7월 24일">7월 24일</a>, <a href="/wiki/7%EC%9B%94_25%EC%9D%BC" title="7월 25일">7월 25일</a>, <a href="/wiki/7%EC%9B%94_28%EC%9D%BC" title="7월 28일">7월 28일</a>, <a href="/wiki/7%EC%9B%94_29%EC%9D%BC" title="7월 29일">7월 29일</a> 등 여러 차례에 걸쳐 진행되었다. <a href="/wiki/%EB%AF%B8%EA%B5%AD_%ED%95%B4%EA%B5%B0" title="미국 해군">미국 해군</a>을 중심으로 한 연합군 <a href="/wiki/%ED%95%AD%EA%B3%B5%EB%AA%A8%ED%95%A8" title="항공모함">항공모함</a> 기동 부대 항공대와 오키나와 미국 육군 항공대에 의한 구레 군항과 내륙 서부 공습 작전을 펼쳤다. 이 공습과는 별도로 <a href="/wiki/1945%EB%85%84" title="1945년">1945년</a> <a href="/wiki/5%EC%9B%94_5%EC%9D%BC" title="5월 5일">5월 5일</a>에 인접 지역에 있는 넓은 조병창 공습, <a href="/wiki/6%EC%9B%94_22%EC%9D%BC" title="6월 22일">6월 22일</a>에 군항 내에 있는 <a href="/wiki/%EA%B5%AC%EB%A0%88_%ED%95%B4%EA%B5%B0%EA%B3%B5%EC%B0%BD" title="구레 해군공창">구레 해군공창</a> 조병부 공습이 있었고, <a href="/wiki/%EA%B5%AC%EB%A0%88_%EC%8B%9C" title="구레 시">구레 시</a>가 <a href="/wiki/7%EC%9B%94_1%EC%9D%BC" title="7월 1일">7월 1일</a> 자정부터 2일 새벽까지 전략적 폭격을 당했다.</p>

<h2><span class="mw-headline" id=".EA.B0.9C.EC.9A.94">개요</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EA%B5%AC%EB%A0%88_%ED%95%AD_%EA%B3%B5%EC%8A%B5&amp;action=edit&amp;section=1" title="부분 편집: 개요">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p><a href="/wiki/3%EC%9B%94_19%EC%9D%BC" title="3월 19일">3월 19일</a> <a href="/wiki/%EB%AF%B8%EA%B5%AD_%ED%95%B4%EA%B5%B0" title="미국 해군">미국 해군</a>의 <a class="new" href="/w/index.php?title=%EB%A7%88%ED%81%AC_%EB%B0%8B%EC%B3%90&amp;action=edit&amp;redlink=1" title="마크 밋쳐 (없는 문서)">마크 밋쳐</a> 제독 휘하의 제58기동부대 350대, 7월 양일은 <a href="/wiki/%EC%9C%8C%EB%A6%AC%EC%97%84_%ED%99%80%EC%8B%9C_%EC%A3%BC%EB%8B%88%EC%96%B4" title="윌리엄 홀시 주니어">윌리엄 홀시</a> 제독이 이끄는 제38기동부대 약 950기에 의한 공습이었다. 일본은 고각포나 함정의 대공 사격으로 격렬하게 저항을 했지만, 거의 모든 함이 침몰하거나 항행 불능 등의 피해를 받았다.</p>
<p><a href="/wiki/3%EC%9B%94_19%EC%9D%BC" title="3월 19일">3월 19일</a>의 제58기동부대에 대해서는 마츠야마 기지의 제343항공대가 구레를 공격 중인 미국 해군 <a href="/wiki/%ED%95%A8%EC%9E%AC%EA%B8%B0" title="함재기">함재기</a>를 요격하여 58기(미국 해군의 기록에서는 14기)를 격추했다. 또한, 은하 1기가 <a href="/wiki/%EA%B3%A0%EC%B9%98_%ED%98%84" title="고치 현">고치 현</a> 앞바다에 접근하고 있던 미국 해군 <a class="new" href="/w/index.php?title=%ED%95%AD%EA%B3%B5%EB%AA%A8%ED%95%A8_%ED%94%84%EB%9E%AD%ED%81%B4%EB%A6%B0&amp;action=edit&amp;redlink=1" title="항공모함 프랭클린 (없는 문서)">항공모함 프랭클린</a>을 공격하여 대파시키고 <a href="/wiki/%EA%B0%80%EB%AF%B8%EC%B9%B4%EC%A0%9C" title="가미카제">가미카제 특공대</a>에 의해 호위 항공모함과 구축함 1척에 손상을 주었다. <a href="/wiki/7%EC%9B%94_24%EC%9D%BC" title="7월 24일">7월 24일</a> 공습으로는 제343항공대가 다시 요격을 향해 16대를 격추시켰다.</p>
<p>일련의 전투에서 일본의 승무원 약 780명이 전사하고, 2000여명이 전상을 입었다. 위와 같이 연합군에 큰 손실을 입혔지만, 조병창 시설이 파괴되었고, 전략 폭격의 일환으로 <a href="/wiki/%EC%84%B8%ED%86%A0_%EB%82%B4%ED%95%B4" title="세토 내해">세토 내해</a>의 요소에 <a class="mw-redirect" href="/wiki/B-29" title="B-29">B-29</a>에서 <a href="/wiki/%EA%B8%B0%EB%A2%B0" title="기뢰">기뢰</a>를 투하하여 구레 군항은 모항으로 기능을 완전히 잃었다.</p>
