<p><b>구레 해군 공창</b>(<span style="font-size: smaller;"><a href="/wiki/%EC%9D%BC%EB%B3%B8%EC%96%B4" title="일본어">일본어</a>:</span> <span lang="ja"><span lang="ja"><ruby><rb>呉海軍工廠</rb> <rp>(</rp><rt>くれかいぐんこうしょう</rt><rp>)</rp></ruby></span></span>)는 <a href="/wiki/%ED%9E%88%EB%A1%9C%EC%8B%9C%EB%A7%88_%ED%98%84" title="히로시마 현">히로시마 현</a> <a href="/wiki/%EA%B5%AC%EB%A0%88_%EC%8B%9C" title="구레 시">구레 시</a>에 존재했던 <a href="/wiki/%EC%9D%BC%EB%B3%B8_%EC%A0%9C%EA%B5%AD_%ED%95%B4%EA%B5%B0" title="일본 제국 해군">일본 제국 해군</a> 조선소이다. 전함 '야마토'의 건축으로 유명하다. <a href="/wiki/%ED%83%9C%ED%8F%89%EC%96%91_%EC%A0%84%EC%9F%81" title="태평양 전쟁">태평양 전쟁</a>이 끝나고 조병창은 해산되었고, 현재는 재팬 마린 유나이티드 회사의 구레 공장으로 대형 민간 선박의 건조를 진행하고 있다.</p>

<h3><span class="mw-headline" id=".EC.A0.84.EC.9F.81_.EC.A0.84">전쟁 전</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EA%B5%AC%EB%A0%88_%ED%95%B4%EA%B5%B0%EA%B3%B5%EC%B0%BD&amp;action=edit&amp;section=2" title="부분 편집: 전쟁 전">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p><a href="/wiki/1889%EB%85%84" title="1889년">1889년</a> (메이지 22년) <a class="mw-redirect" href="/wiki/%ED%9E%88%EB%A1%9C%EC%8B%9C%EB%A7%88%ED%98%84" title="히로시마현">히로시마현</a> <a href="/wiki/%EA%B5%AC%EB%A0%88_%EC%8B%9C" title="구레 시">구레 시</a>에 <a href="/wiki/%EA%B5%AC%EB%A0%88_%EC%A7%84%EC%88%98%EB%B6%80" title="구레 진수부">구레 진수부</a> 설치와 동시에 ‘조선부’가 설치된다. 당초 조선은 <a class="mw-redirect" href="/wiki/%EA%B3%A0%EB%B2%A0" title="고베">고베</a>에 있던 오노하마 조선소에 의존하고 있었지만, 점차 구레에서 설비를 확충하여, 이후 오노하마 조선소는 폐쇄되었다. 제작은 프랑스 기술자 <a href="/wiki/%EB%A3%A8%EC%9D%B4_%EC%97%90%EB%B0%80_%EB%B2%A0%EB%A5%B4%ED%83%95" title="루이 에밀 베르탕">루이 에밀 베르탕</a>이 감독하고 있었다. 구레 해군 조선소에서 최초로 제작한 군함은 ‘<a class="new" href="/w/index.php?title=%EB%AF%B8%EC%95%BC%EC%BD%94_(%EC%88%9C%EC%96%91%ED%95%A8)&amp;action=edit&amp;redlink=1" title="미야코 (순양함) (없는 문서)">미야코</a>’(宮古)였으며, 1897년 진수되었다. <a href="/wiki/1903%EB%85%84" title="1903년">1903년</a>에 일본 해군의 조직 개편으로 구레 해군 조선소가 탄생했다. 그 후로 동양 제일이라고 불릴 정도로까지 설비를 확충시켰다. 조선공들의 총수는 다른 세 조병창, <a href="/wiki/%EC%9A%94%EC%BD%94%EC%8A%A4%EC%B9%B4_%ED%95%B4%EA%B5%B0%EA%B3%B5%EC%B0%BD" title="요코스카 해군공창">요코스카</a>, <a href="/wiki/%EC%82%AC%EC%84%B8%EB%B3%B4_%ED%95%B4%EA%B5%B0%EA%B3%B5%EC%B0%BD" title="사세보 해군공창">사세보</a>, <a class="new" href="/w/index.php?title=%EB%A7%88%EC%9D%B4%EC%A6%88%EB%A3%A8_%ED%95%B4%EA%B5%B0%EA%B3%B5%EC%B0%BD&amp;action=edit&amp;redlink=1" title="마이즈루 해군공창 (없는 문서)">마이즈루</a>의 합계를 넘을 정도였으며, 독일의 크루프와 견줄 수 있는 세계 양대 무기 공장이었다. <a class="mw-redirect" href="/wiki/%EC%A0%84%ED%95%A8_%EC%95%BC%EB%A7%88%ED%86%A0" title="전함 야마토">전함 야마토</a>를 제작하는 등 많은 함정 제작을 다루는 일본 해군 함정 건조의 중심지가 되었다.</p>
<p>포공부(포탑의 제조, 개발), 제강부 (장갑판의 제조, 개발)가 설치되어 전함 건조에 주도적 역할을 했다. <a href="/wiki/%EB%AF%B8%EC%93%B0%EB%B9%84%EC%8B%9C_%EC%A4%91%EA%B3%B5%EC%97%85" title="미쓰비시 중공업">미쓰비시 중공업</a> 나가사키 조선소에서 건조된 <a class="mw-redirect" href="/wiki/%EC%A0%84%ED%95%A8_%EB%AC%B4%EC%82%AC%EC%8B%9C" title="전함 무사시">전함 무사시</a>의 주포탑과 포신도 구레에서 제작되었다.</p>
<p><a href="/wiki/1912%EB%85%84" title="1912년">1912년</a> <a href="/wiki/3%EC%9B%94_29%EC%9D%BC" title="3월 29일">3월 29일</a>, 1만 명이 참가하는 파업을 맞았다. 이 파업에는 작가 미야치 가로쿠도 참여하여 제9공장의 주동자로 검거되었다. 또한 <a href="/wiki/1918%EB%85%84" title="1918년">1918년</a> 8월, <a href="/wiki/%EC%9D%BC%EB%B3%B8%EC%9D%98_%EC%8C%80_%EC%86%8C%EB%8F%99" title="일본의 쌀 소동">일본의 쌀 소동</a>에 다수의 조병창 노동자가 참가하여 총검을 가진 <a class="mw-redirect" href="/wiki/%EC%9D%BC%EB%B3%B8_%EC%A0%9C%EA%B5%AD_%ED%95%B4%EA%B5%B0_%EC%9C%A1%EC%A0%84%EB%8C%80" title="일본 제국 해군 육전대">해군 육전대</a>와 대치하면서 사상자가 나왔다.</p>
<h3><span class="mw-headline" id=".EC.A0.84.EC.9F.81_.EC.A4.91">전쟁 중</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EA%B5%AC%EB%A0%88_%ED%95%B4%EA%B5%B0%EA%B3%B5%EC%B0%BD&amp;action=edit&amp;section=3" title="부분 편집: 전쟁 중">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p><a href="/wiki/1945%EB%85%84" title="1945년">1945년</a> <a href="/wiki/6%EC%9B%94_22%EC%9D%BC" title="6월 22일">6월 22일</a> 9시부터 B-29, 290기가 공습하여 조병 지역을 중심으로 시설을 파괴했다. 해군 공창과 관계된 약 1900명의 사망자가 발생했다.</p>
<ul>
<li><a class="mw-redirect" href="/wiki/%EA%B5%AC%EB%A0%88%ED%95%AD_%EA%B3%B5%EC%8A%B5" title="구레항 공습">구레항 공습</a></li>
</ul>
<h3><span class="mw-headline" id=".EC.A0.84.ED.9B.84">전후</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EA%B5%AC%EB%A0%88_%ED%95%B4%EA%B5%B0%EA%B3%B5%EC%B0%BD&amp;action=edit&amp;section=4" title="부분 편집: 전후">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>토지, 시설은 하리마 조선소와 미국 내셔널 벌크 캐리어 사(NBC)가 인계했다. 구레 조선소, 石川島播磨 중공업(현 IHI) 구레 공장을 거쳐 현재는 재팬 마린 유나이티드 사의 구레 공장이 되었다. 구레 공장에서 자위대 함정을 건조하지는 않지만, 수리 작업은 지금도 진행하고 있다. 또한 '조선 선거'(야마토를 위한 <a class="new" href="/w/index.php?title=%EA%B1%B4%EC%84%A0%EA%B1%B0&amp;action=edit&amp;redlink=1" title="건선거 (없는 문서)">건선거</a>)는 1993년에 매립되어 터는 공장 부지로 재사용되고 있다. 그러나 야마토 수리를 했던 도크는 지금도 존재하고 있으며 미군 함선이 현재 사용 중이다.</p>
<p>구레 해군 조선소가 야마토 건조 시에 채용한 선체 부분마다 건축 블록 공법, 부품 공통화 등의 혁신적인 관리 시스템은 이후 <a class="mw-redirect" href="/wiki/%EB%8F%84%EC%9A%94%ED%83%80_%EC%9E%90%EB%8F%99%EC%B0%A8" title="도요타 자동차">도요타 자동차</a> 등을 통해 세계에 퍼져 효율적인 일본형 생산 방식의 원류라고 전해진다.</p>
