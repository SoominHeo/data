<p><b>구이런 구</b>(<span style="font-size: smaller;"><a href="/wiki/%ED%95%9C%EA%B5%AD%EC%96%B4" title="한국어">한국어</a>:</span> <span lang="ko" xml:lang="ko">귀인구</span>, <small><a href="/wiki/%EC%A4%91%EA%B5%AD%EC%96%B4" title="중국어">중국어</a></small><small>:</small> <span lang="zh" xml:lang="zh">歸仁區</span>, <small><a href="/wiki/%ED%95%9C%EC%96%B4_%EB%B3%91%EC%9D%8C" title="한어 병음">병음</a>:</small> <font face="Lucida Sans Unicode">Guīrén Qū</font>)는 <a href="/wiki/%EC%A4%91%ED%99%94%EB%AF%BC%EA%B5%AD" title="중화민국">중화민국</a> <a href="/wiki/%ED%83%80%EC%9D%B4%EB%82%9C_%EC%8B%9C" title="타이난 시">타이난 시</a>의 <a href="/wiki/%EC%8B%9C%ED%95%A0%EA%B5%AC" title="시할구">시할구</a>이다. 넓이는 55.7913㎢이고, 인구는 <a href="/wiki/2015%EB%85%84" title="2015년">2015년</a> 8월 기준으로 67,783명이다.</p>

<h2><span class="mw-headline" id=".EC.A7.80.EB.A6.AC">지리</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EA%B5%AC%EC%9D%B4%EB%9F%B0_%EA%B5%AC&amp;action=edit&amp;section=1" title="부분 편집: 지리">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>구이런 구는 <a href="/wiki/%ED%83%80%EC%9D%B4%EB%82%9C_%EC%8B%9C" title="타이난 시">타이난 시</a> 남단에 위치하고, 북쪽은 <a href="/wiki/%EC%9C%B5%EC%BA%89_%EA%B5%AC" title="융캉 구">융캉 구</a>, <a href="/wiki/%EC%8B%A0%ED%99%94_%EA%B5%AC_(%ED%83%80%EC%9D%B4%EB%82%9C_%EC%8B%9C)" title="신화 구 (타이난 시)">신화 구</a>와, 동쪽은 <a href="/wiki/%EA%B4%80%EB%A8%80%EC%98%A4_%EA%B5%AC" title="관먀오 구">관먀오 구</a>와, 서쪽은 <a href="/wiki/%EB%9F%B0%EB%8D%94_%EA%B5%AC" title="런더 구">런더 구</a>와, 남쪽은 <a href="/wiki/%EA%B0%80%EC%98%A4%EC%8A%9D_%EC%8B%9C" title="가오슝 시">가오슝 시</a> <a href="/wiki/%EC%95%84%EB%A1%84_%EA%B5%AC" title="아롄 구">아롄 구</a>, <a class="mw-disambig" href="/wiki/%EB%A3%A8%EC%A3%BC_%EA%B5%AC" title="루주 구">루주 구</a>와 각각 접하고 있다. <a class="new" href="/w/index.php?title=%EC%9E%90%EB%82%9C_%ED%8F%89%EC%9B%90&amp;action=edit&amp;redlink=1" title="자난 평원 (없는 문서)">자난 평원</a>에 위치하고 있기 때문에 전역이 평지여서 구릉은 존재하지 않는다. 얼런시(二仁溪)의 지류인 쉬셴시(許縣溪)가 향내를 흐르고 있다</p>
<h2><span class="mw-headline" id=".EC.97.AD.EC.82.AC">역사</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EA%B5%AC%EC%9D%B4%EB%9F%B0_%EA%B5%AC&amp;action=edit&amp;section=2" title="부분 편집: 역사">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>구이런 구는 옛날에는 <a class="new" href="/w/index.php?title=%EC%8B%9C%EB%9D%BC%EC%95%BC%EC%A1%B1&amp;action=edit&amp;redlink=1" title="시라야족 (없는 문서)">시라야족</a> 신항사의 거주지였다. 전승에서는 진(陳), 이(李) 두 성의 한족이 현재의 구이런 국소의 남쪽으로 이주를 시작해, 빨간 벽돌로 주거지를 지은 데에서 옛 이름인 홍와조(紅瓦厝)가 생겼다고 한다. <a href="/wiki/%EC%A0%95%EC%84%B1%EA%B3%B5" title="정성공">정성공</a> 시대, 이곳에 서원이 설립되어 천하귀인언(天下歸仁焉)의 일부 글자를 취해 귀인리(歸仁里)로 개칭되었다. 1920년의 타이완 지방제 개편 때, 이곳에 기진 장(帰仁庄)이 설치되어 다이난 주 신포 군의 관할로 했다. 전후에 타이난 현 구이런 향으로 고쳐져 현재에 이르고 있다.</p>
