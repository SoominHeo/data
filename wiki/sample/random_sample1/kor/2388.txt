<p><b>구자체</b>(舊字體, 旧字体)는 <a href="/wiki/%EC%9D%BC%EB%B3%B8" title="일본">일본</a>에서 <a href="/wiki/%EC%9D%BC%EB%B3%B8_%EC%A0%9C%EA%B5%AD" title="일본 제국">일본 제국</a>의 패망 이전때까지 통용되었던 <a href="/wiki/%ED%95%9C%EC%9E%90" title="한자">한자</a>를 가리킨다. 구자체는 <a href="/wiki/%EB%8C%80%ED%95%9C%EB%AF%BC%EA%B5%AD" title="대한민국">대한민국</a>과 <a href="/wiki/%EC%A1%B0%EC%84%A0%EB%AF%BC%EC%A3%BC%EC%A3%BC%EC%9D%98%EC%9D%B8%EB%AF%BC%EA%B3%B5%ED%99%94%EA%B5%AD" title="조선민주주의인민공화국">조선민주주의인민공화국</a>, <a href="/wiki/%EC%A4%91%ED%99%94%EB%AF%BC%EA%B5%AD" title="중화민국">중화민국</a>, <a href="/wiki/%ED%99%8D%EC%BD%A9" title="홍콩">홍콩</a>, <a href="/wiki/%EB%A7%88%EC%B9%B4%EC%98%A4" title="마카오">마카오</a>, <a href="/wiki/%EB%A7%90%EB%A0%88%EC%9D%B4%EC%8B%9C%EC%95%84" title="말레이시아">말레이시아</a> 일부 지역에서 사용하는 <a href="/wiki/%EC%A0%95%EC%B2%B4%EC%9E%90" title="정체자">정체자</a> 한자와 동일하다. 사용하였던 시기는 일본 제국의 <a href="/wiki/%EC%A0%9C2%EC%B0%A8_%EC%84%B8%EA%B3%84_%EB%8C%80%EC%A0%84" title="제2차 세계 대전">제2차 세계 대전</a> 이전 당시가 최고 고조기였고, <a href="/wiki/1946%EB%85%84" title="1946년">1946년</a>에 제정된 <a href="/wiki/%EC%8B%A0%EC%9E%90%EC%B2%B4" title="신자체">신자체</a>를 정식으로 통용하게 됨에 따라 이에 대비되는 한자라고 가리킨다.</p>
