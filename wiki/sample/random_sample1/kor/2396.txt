<p><b>구키 요시타카</b>(九鬼嘉隆, <a href="/wiki/1542%EB%85%84" title="1542년">1542년</a>~<a href="/wiki/1600%EB%85%84" title="1600년">1600년</a> <a href="/wiki/11%EC%9B%94_17%EC%9D%BC" title="11월 17일">11월 17일</a>)는 <a href="/wiki/%EC%84%BC%EA%B3%A0%EC%BF%A0_%EC%8B%9C%EB%8C%80" title="센고쿠 시대">센고쿠 시대</a>부터 <a class="mw-redirect" href="/wiki/%EC%95%84%EC%A6%88%EC%B9%98%EB%AA%A8%EB%AA%A8%EC%95%BC%EB%A7%88_%EC%8B%9C%EB%8C%80" title="아즈치모모야마 시대">아즈치모모야마 시대</a>까지의 <a href="/wiki/%EB%AC%B4%EC%9E%A5" title="무장">무장</a>, <a href="/wiki/%EB%8B%A4%EC%9D%B4%EB%AC%98" title="다이묘">다이묘</a>이다. <a class="new" href="/w/index.php?title=%EA%B5%AC%ED%82%A4_%EC%88%98%EA%B5%B0&amp;action=edit&amp;redlink=1" title="구키 수군 (없는 문서)">구키 수군</a>을 이끈 <a href="/wiki/%EC%88%98%EA%B5%B0" title="수군">수군</a>무장이며 <a class="new" href="/w/index.php?title=%EA%B5%AC%ED%82%A4_%EC%94%A8&amp;action=edit&amp;redlink=1" title="구키 씨 (없는 문서)">구키 씨</a> 제8대 당주. 자식은 <a href="/wiki/%EA%B5%AC%ED%82%A4_%EB%AA%A8%EB%A6%AC%ED%83%80%EC%B9%B4" title="구키 모리타카">모리타카</a>(守隆) 등.</p>
<h2><span class="mw-headline" id=".EC.83.9D.EC.95.A0">생애</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EA%B5%AC%ED%82%A4_%EC%9A%94%EC%8B%9C%ED%83%80%EC%B9%B4_(1542%EB%85%84)&amp;action=edit&amp;section=1" title="부분 편집: 생애">원본 편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>구키 요시타카는 <a class="new" href="/w/index.php?title=%EA%B5%AC%EB%A7%88%EB%85%B8&amp;action=edit&amp;redlink=1" title="구마노 (없는 문서)">구마노</a> 해적 출신으로 구마노 해적의 세력이 약화되자 <a href="/wiki/%EC%98%A4%EB%8B%A4_%EB%85%B8%EB%B6%80%EB%82%98%EA%B0%80" title="오다 노부나가">오다 노부나가</a>(織田信長)의 휘하로 들어가 수군장으로 활약해 <a href="/wiki/%EC%8B%9C%EB%A7%88_%EA%B5%AD" title="시마 국">시마 국</a>의 영주로 임명되었다. 이후에는 <a href="/wiki/%EB%8F%84%EC%9A%94%ED%86%A0%EB%AF%B8_%ED%9E%88%EB%8D%B0%EC%9A%94%EC%8B%9C" title="도요토미 히데요시">도요토미 히데요시</a>(豐臣秀吉) 밑에 들어가 여러 해전에서 활약했다.</p>
<p><a href="/wiki/1592%EB%85%84" title="1592년">1592년</a> <a href="/wiki/%EC%9E%84%EC%A7%84%EC%99%9C%EB%9E%80" title="임진왜란">임진왜란</a>에는 1,500명의 군대를 이끌고 참전하여 <a href="/wiki/%EB%8F%84%EB%8F%84_%EB%8B%A4%EC%B9%B4%ED%86%A0%EB%9D%BC" title="도도 다카토라">도도 다카토라</a>(藤堂高虎)와 함께 9,000명의 수군을 이끌고 <a href="/wiki/%EC%9D%B4%EC%88%9C%EC%8B%A0" title="이순신">이순신</a>에게 패한 일본 수군을 구원하려 출동하였으나 <a href="/wiki/%EC%95%88%EA%B3%A8%ED%8F%AC_%ED%95%B4%EC%A0%84" title="안골포 해전">안골포 해전</a>에서 <a href="/wiki/%EC%9D%B4%EC%88%9C%EC%8B%A0" title="이순신">이순신</a>(李舜臣)에게 대패하여 <a class="mw-disambig" href="/wiki/%EC%93%B0%EC%8B%9C%EB%A7%88" title="쓰시마">쓰시마</a>의 군선 관리 책임을 맡았다. 이후에는 현역에서 은거하였다.</p>
<p><a href="/wiki/1600%EB%85%84" title="1600년">1600년</a> <a href="/wiki/%EC%84%B8%ED%82%A4%EA%B0%80%ED%95%98%EB%9D%BC_%EC%A0%84%ED%88%AC" title="세키가하라 전투">세키가하라 전투</a>에서 <a href="/wiki/%EB%AA%A8%EB%A6%AC_%EB%8D%B0%EB%A3%A8%EB%AA%A8%ED%86%A0" title="모리 데루모토">모리 데루모토</a>의 측인 서군 편에 참전했으나 차남 <a href="/wiki/%EA%B5%AC%ED%82%A4_%EB%AA%A8%EB%A6%AC%ED%83%80%EC%B9%B4" title="구키 모리타카">구키 모리타카</a>가 <a href="/wiki/%EB%8F%84%EC%BF%A0%EA%B0%80%EC%99%80_%EC%9D%B4%EC%97%90%EC%95%BC%EC%8A%A4" title="도쿠가와 이에야스">도쿠가와 이에야스</a>(德川家康) 측인 동군에 가담해 부자가 서로 적이 되어 싸우는 비극을 맞았다. 이후 모리타카가 도쿠가와 이에야스에게 청하여 죄를 면하게 해주려고 했으나 그 소식이 전달되기 전에 요시타카는 자살했다.</p>
