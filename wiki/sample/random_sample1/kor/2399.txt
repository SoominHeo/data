<h2><span class="mw-headline" id=".EC.83.9D.EC.95.A0">생애</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EA%B5%AC%ED%95%98%EB%9D%BC&amp;action=edit&amp;section=2" title="부분 편집: 생애">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p><a href="/wiki/1991%EB%85%84" title="1991년">1991년</a> <a href="/wiki/1%EC%9B%94_13%EC%9D%BC" title="1월 13일">1월 13일</a> <a href="/wiki/%EA%B4%91%EC%A3%BC%EA%B4%91%EC%97%AD%EC%8B%9C" title="광주광역시">광주광역시</a>에서 1남 1녀 중 막내로 태어났다. <a href="/wiki/%EC%9A%B4%EC%B2%9C%EC%B4%88%EB%93%B1%ED%95%99%EA%B5%90_(%EA%B4%91%EC%A3%BC)" title="운천초등학교 (광주)">운천초등학교</a>, <a href="/wiki/%EC%A0%84%EB%82%A8%EC%A4%91%ED%95%99%EA%B5%90" title="전남중학교">전남중학교</a>를 다닐 적에 2년 동안 육상부에서 활동한 경험이 있으며,<sup class="reference" id="cite_ref-2"><a href="#cite_note-2">[2]</a></sup> <a href="/wiki/%EC%A0%84%EC%A3%BC%EC%98%88%EC%88%A0%EA%B3%A0%EB%93%B1%ED%95%99%EA%B5%90" title="전주예술고등학교">전주예술고등학교</a>를 다니다 연예계 활동의 편의를 위해 전학한 학교 <a href="/wiki/%EB%8F%99%EB%AA%85%EC%97%AC%EC%9E%90%EC%A0%95%EB%B3%B4%EC%82%B0%EC%97%85%EA%B3%A0%EB%93%B1%ED%95%99%EA%B5%90" title="동명여자정보산업고등학교">동명여자정보산업고등학교</a>를 졸업하고, <a href="/wiki/%EC%84%B1%EC%8B%A0%EC%97%AC%EC%9E%90%EB%8C%80%ED%95%99%EA%B5%90" title="성신여자대학교">성신여자대학교</a> 미디어영상연기학부에 진학하였다.<sup class="reference" id="cite_ref-3"><a href="#cite_note-3">[3]</a></sup> 학창시절 인터넷 의류 쇼핑몰 모델로 활동한 경력이 있다.<sup class="reference" id="cite_ref-4"><a href="#cite_note-4">[4]</a></sup> <a href="/wiki/2007%EB%85%84" title="2007년">2007년</a>에 <a href="/wiki/JYP_%EC%97%94%ED%84%B0%ED%85%8C%EC%9D%B8%EB%A8%BC%ED%8A%B8" title="JYP 엔터테인먼트">JYP 엔터테인먼트</a>가 주최한 오디션에 참가해 결선까지 진출했으나, 최종 관문에서 탈락하였고,<sup class="reference" id="cite_ref-5"><a href="#cite_note-5">[5]</a></sup><sup class="reference" id="cite_ref-6"><a href="#cite_note-6">[6]</a></sup> 이후 <a href="/wiki/DSP_%EB%AF%B8%EB%94%94%EC%96%B4" title="DSP 미디어">DSP 미디어</a>의 오디션을 통해 <a href="/wiki/%EC%B9%B4%EB%9D%BC_(%EC%9D%8C%EC%95%85_%EA%B7%B8%EB%A3%B9)" title="카라 (음악 그룹)">카라</a>의 구성원으로서, <a href="/wiki/2008%EB%85%84" title="2008년">2008년</a> <a href="/wiki/7%EC%9B%94_24%EC%9D%BC" title="7월 24일">7월 24일</a>에 <a class="mw-redirect" href="/wiki/Mnet" title="Mnet">Mnet</a> <a class="mw-redirect" href="/wiki/%EC%97%A0_%EC%B9%B4%EC%9A%B4%ED%8A%B8%EB%8B%A4%EC%9A%B4" title="엠 카운트다운">엠 카운트다운</a>을 통해 데뷔하였다.</p>
<p>데뷔 이후에는 《<a href="/wiki/%EC%B2%AD%EC%B6%98%EB%B6%88%ED%8C%A8" title="청춘불패">청춘불패</a>》에서의 활약으로<sup class="reference" id="cite_ref-7"><a href="#cite_note-7">[7]</a></sup>, <a href="/wiki/2010%EB%85%84" title="2010년">2010년</a> 《<a href="/wiki/KBS_%EC%97%B0%EC%98%88%EB%8C%80%EC%83%81" title="KBS 연예대상">KBS 연예대상</a>》에서 ‘쇼, 오락 MC 부문 여자 우수상’을 단독으로 수상하였고,<sup class="reference" id="cite_ref-8"><a href="#cite_note-8">[8]</a></sup><sup class="reference" id="cite_ref-9"><a href="#cite_note-9">[9]</a></sup> SBS 드라마인 《시티헌터》에서의 최다혜 역으로, 2011년 《<a href="/wiki/SBS_%EC%97%B0%EA%B8%B0%EB%8C%80%EC%83%81" title="SBS 연기대상">SBS 연기대상</a>》에서 ‘SBS 연기대상 뉴스타상’을 수상하였고,<sup class="reference" id="cite_ref-10"><a href="#cite_note-10">[10]</a></sup> 2015년 《<a href="/wiki/SBS_%EC%97%B0%EC%98%88%EB%8C%80%EC%83%81" title="SBS 연예대상">SBS 연예대상</a>》에서 <a href="/wiki/%EC%A3%BC%EB%A8%B9%EC%A5%90%EA%B3%A0_%EC%86%8C%EB%A6%BC%EC%82%AC" title="주먹쥐고 소림사">주먹쥐고 소림사</a>로 '베스트 첼린지상'을 수상하였다<sup class="reference" id="cite_ref-11"><a href="#cite_note-11">[11]</a></sup></p>
