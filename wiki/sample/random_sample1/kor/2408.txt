<h1 class="firstHeading" id="firstHeading" lang="ko">국가원수 의전차</h1>

<p><b>국가원수 의전차</b>는 자국의 국가원수를 비롯한 정부 요인이나, 자국을 방문한 외국 귀빈을 의전하기 위해 정부에서 사용하는 자동차이다. 국빈 방문 시에 의전차는 국가원수의 도착과 맞추어 준비된다. 의전차가 되기 위한 조건으로는 공무를 수행하기에 충분한 안전과 성능, 위엄 등이 있다. <a href="/wiki/%EB%A6%AC%EB%AC%B4%EC%A7%84" title="리무진">리무진</a>(대형 세단 포함), 고급 <a href="/wiki/%EC%84%B8%EB%8B%A8" title="세단">세단</a>(Executive car, 혹은 준대형 세단) 혹은 <a class="mw-redirect" href="/wiki/SUV" title="SUV">SUV</a>가 주로 선택되는 차종이다.</p>
<p>승객이 처하는 안전 상의 위험이 크기 때문에 의전차들은 종종 엄중한 보호를 받는다. 의전차들은 자동차 메이커나 애프터마켓 전문 업체에 의해 방탄 처리되기도 하며, 요인이 탑승할 경우 경찰 혹은 군인으로 구성된 차량 행렬에 둘러싸여 보호를 받는 것이 일반적이다. 운전사 역시 경찰이나 군 출신의 인원으로 이루어지기도 한다. 예를 들어 미국 대통령의 의전차는 숙련된 <a class="new" href="/w/index.php?title=%EC%8B%9C%ED%81%AC%EB%A6%BF_%EC%84%9C%EB%B9%84%EC%8A%A4&amp;action=edit&amp;redlink=1" title="시크릿 서비스 (없는 문서)">시크릿 서비스</a> (SS) 요원이 운전하며, 영국 총리의 의전차는 특별 훈련을 받은 <a class="mw-disambig" href="/wiki/SAS" title="SAS">SAS</a> 대원이 운전한다.</p>
<p>독자적인 자동차 메이커를 갖고 있는 국가에서 정부는 보통 의전차를 자국 메이커 중 한 곳에서 선택하거나, 혹은 여러 자국 메이커에서 각각 한 종류 씩을 선택하여 제공받는다. 이러한 경우 자동차 메이커들은 자신이 국가원수 의전차를 제공하도록 선택되는 것을 영광으로 간주한다. 또한 자국 메이커의 자동차를 선택하는 것은 정부가 자국 (자동차) 산업에 대한 자신감을 표현하는 것으로 보여지기도 한다. 가령 <a class="mw-redirect" href="/wiki/%EB%8C%80%ED%95%9C%EB%AF%BC%EA%B5%AD_%EB%8C%80%ED%86%B5%EB%A0%B9" title="대한민국 대통령">대한민국 대통령</a>은 <a href="/wiki/%ED%98%84%EB%8C%80%EC%9E%90%EB%8F%99%EC%B0%A8" title="현대자동차">현대자동차</a>를 이용하며, <a class="mw-redirect" href="/wiki/%EB%AF%B8%EA%B5%AD_%EB%8C%80%ED%86%B5%EB%A0%B9" title="미국 대통령">미국 대통령</a>은 <a href="/wiki/%EC%BA%90%EB%94%9C%EB%9D%BD" title="캐딜락">캐딜락</a>, <a class="mw-redirect" href="/wiki/%EC%98%81%EA%B5%AD_%EC%99%95%EC%8B%A4" title="영국 왕실">영국 왕실</a>은 <a href="/wiki/%EB%B2%A4%ED%8B%80%EB%A6%AC" title="벤틀리">벤틀리</a>, <a class="mw-redirect" href="/wiki/%EC%9D%B4%ED%83%88%EB%A6%AC%EC%95%84_%EC%B4%9D%EB%A6%AC" title="이탈리아 총리">이탈리아 총리</a>는 <a href="/wiki/%EB%A7%88%EC%84%B8%EB%9D%BC%ED%8B%B0" title="마세라티">마세라티</a>, <a class="mw-redirect" href="/wiki/%EC%9D%BC%EB%B3%B8_%EC%99%95" title="일본 왕">일본 왕</a>과 <a class="mw-redirect" href="/wiki/%EC%9D%BC%EB%B3%B8_%EC%B4%9D%EB%A6%AC" title="일본 총리">일본 총리</a>는 <a class="mw-redirect mw-disambig" href="/wiki/%ED%86%A0%EC%9A%94%ED%83%80" title="토요타">토요타</a>를 각각 이용한다. <a class="mw-redirect" href="/wiki/%ED%94%84%EB%9E%91%EC%8A%A4_%EB%8C%80%ED%86%B5%EB%A0%B9" title="프랑스 대통령">프랑스 대통령</a>은 <a href="/wiki/%EB%A5%B4%EB%85%B8" title="르노">르노</a>와 <a class="mw-redirect" href="/wiki/PSA_%ED%91%B8%EC%A1%B0_%EC%8B%9C%ED%8A%B8%EB%A1%9C%EC%97%A5_%EC%9E%90%EB%8F%99%EC%B0%A8_%EA%B7%B8%EB%A3%B9" title="PSA 푸조 시트로엥 자동차 그룹">시트로엥, 푸조</a>에서 제작한 고급차 중에서 선택할 수 있으며, 독일의 모든 주요 자동차 메이커들은 각각 자신들의 차종을 <a class="mw-redirect" href="/wiki/%EB%8F%85%EC%9D%BC_%EC%B4%9D%EB%A6%AC" title="독일 총리">독일 총리</a>를 포함한 주요 정부 요인들이 사용하도록 의전차로 제공하고 있다.</p>
<p>독자적인 자동차 메이커가 없는 국가들은 의전차를 자국과 정치, 경제, 외교적으로 강하게 연계된 국가에서 구입하기도 한다. <a href="/wiki/%EC%BA%90%EB%82%98%EB%8B%A4" title="캐나다">캐나다</a>는 미국과 미국 자동차 메이커과의 역사적으로 강한 연계를 반영하여, 자국의 의전차로 미국 빅3(GM, 크라이슬러, 포드)의 차종만을 독점적으로 사용하고 있으며, 이들 중 다수는 캐나다 공장에서 제작한 것이다. 캐나다 정부는 <a href="/wiki/2008%EB%85%84" title="2008년">2008년</a> 경기 침체에 따른 GM과 크라이슬러의 재건 과정에서 이들의 지분을 매입하여 소유하고 있다.</p>
<p>다른 나라와의 연계와는 별개로, 가장 보편적으로 사용되는 <a class="mw-redirect" href="/wiki/%EB%A9%94%EB%A5%B4%EC%84%B8%EB%8D%B0%EC%8A%A4-%EB%B2%A4%EC%B8%A0_S%ED%81%B4%EB%9E%98%EC%8A%A4" title="메르세데스-벤츠 S클래스">메르세데스-벤츠 S클래스</a>나 <a href="/wiki/BMW_7_%EC%8B%9C%EB%A6%AC%EC%A6%88" title="BMW 7 시리즈">BMW 7 시리즈</a> 등을 의전차로 사용하는 국가도 많다. 이 경우에는 독자적인 자동차 메이커가 없는 경우와, 독자적 자동차 메이커가 존재하지만 그 중에서 의전차로 충분한 성능을 가진 차종이 없다고 판단되는 경우가 포함된다.</p>

<h2><span class="mw-headline" id=".EA.B7.B8.EB.A0.88.EB.82.98.EB.8B.A4"><a href="/wiki/%EA%B7%B8%EB%A0%88%EB%82%98%EB%8B%A4" title="그레나다">그레나다</a></span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EA%B5%AD%EA%B0%80%EC%9B%90%EC%88%98_%EC%9D%98%EC%A0%84%EC%B0%A8&amp;action=edit&amp;section=1" title="부분 편집: 그레나다">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p><a class="new" href="/w/index.php?title=%EA%B7%B8%EB%A0%88%EB%82%98%EB%8B%A4%EC%9D%98_%EC%B4%9D%EB%8F%85&amp;action=edit&amp;redlink=1" title="그레나다의 총독 (없는 문서)">그레나다의 총독</a>은 흰색 <a href="/wiki/BMW_7_%EC%8B%9C%EB%A6%AC%EC%A6%88" title="BMW 7 시리즈">BMW 750Li</a>를 타며, 여왕을 상징하는 왕관 모양의 번호판을 단다. 반면에 <a href="/wiki/%EA%B7%B8%EB%A0%88%EB%82%98%EB%8B%A4%EC%9D%98_%EC%B4%9D%EB%A6%AC" title="그레나다의 총리">그레나다의 총리</a>는 검은 <a href="/wiki/BMW_7_%EC%8B%9C%EB%A6%AC%EC%A6%88" title="BMW 7 시리즈">BMW 750Li</a>에 타며, <a href="/wiki/%EA%B7%B8%EB%A0%88%EB%82%98%EB%8B%A4%EC%9D%98_%EA%B5%AD%EC%9E%A5" title="그레나다의 국장">그레나다의 국장</a> 모양의 번호판과 <a href="/wiki/%EA%B7%B8%EB%A0%88%EB%82%98%EB%8B%A4%EC%9D%98_%EA%B5%AD%EA%B8%B0" title="그레나다의 국기">국기</a>를 단다. 의전차는 경찰의 오토바이 경호대와 검은 <a href="/wiki/%EB%AF%B8%EC%93%B0%EB%B9%84%EC%8B%9C_%ED%8C%8C%EC%A0%9C%EB%A1%9C" title="미쓰비시 파제로">미쓰비시 파제로</a> 2대의 호위를 받는다.</p>
<h2><span class="mw-headline" id=".EB.8B.88.EC.B9.B4.EB.9D.BC.EA.B3.BC"><a href="/wiki/%EB%8B%88%EC%B9%B4%EB%9D%BC%EA%B3%BC" title="니카라과">니카라과</a></span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EA%B5%AD%EA%B0%80%EC%9B%90%EC%88%98_%EC%9D%98%EC%A0%84%EC%B0%A8&amp;action=edit&amp;section=2" title="부분 편집: 니카라과">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p><a class="new" href="/w/index.php?title=%EB%8B%88%EC%B9%B4%EB%9D%BC%EA%B3%BC%EC%9D%98_%EB%8C%80%ED%86%B5%EB%A0%B9&amp;action=edit&amp;redlink=1" title="니카라과의 대통령 (없는 문서)">니카라과의 대통령</a>은 <a href="/wiki/%EB%A9%94%EB%A5%B4%EC%84%B8%EB%8D%B0%EC%8A%A4-%EB%B2%A4%EC%B8%A0_G_%ED%81%B4%EB%9E%98%EC%8A%A4" title="메르세데스-벤츠 G 클래스">메르세데스 벤츠 G클래스</a>를 사용한다. 과거에 <a href="/wiki/%EC%86%8C%EB%AA%A8%EC%82%AC_%EA%B0%80%EB%AC%B8" title="소모사 가문">소모사 가문</a>은 43년의 통치 기간 동안 여러 고급 차량을 사용하였다. <a class="new" href="/w/index.php?title=%EB%A9%94%EB%A5%B4%EC%84%B8%EB%8D%B0%EC%8A%A4-%EB%B2%A4%EC%B8%A0_300&amp;action=edit&amp;redlink=1" title="메르세데스-벤츠 300 (없는 문서)">메르세데스-벤츠 300</a> (Mercedes-Benz Type 300) 이 공식 행사에 사용되었고, 그 외에 <a href="/wiki/%EB%A9%94%EB%A5%B4%EC%84%B8%EB%8D%B0%EC%8A%A4-%EB%B2%A4%EC%B8%A0" title="메르세데스-벤츠">메르세데스-벤츠</a>와 <a href="/wiki/%EC%BA%90%EB%94%9C%EB%9D%BD" title="캐딜락">캐딜락</a> 리무진도 사용되었다.</p>
<h2><span class="mw-headline" id=".EB.8C.80.ED.95.9C.EB.AF.BC.EA.B5.AD"><a href="/wiki/%EB%8C%80%ED%95%9C%EB%AF%BC%EA%B5%AD" title="대한민국">대한민국</a></span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EA%B5%AD%EA%B0%80%EC%9B%90%EC%88%98_%EC%9D%98%EC%A0%84%EC%B0%A8&amp;action=edit&amp;section=3" title="부분 편집: 대한민국">편집</a><span class="mw-editsection-bracket">]</span></span></h2>

<p><a href="/wiki/%EB%8C%80%ED%95%9C%EB%AF%BC%EA%B5%AD%EC%9D%98_%EB%8C%80%ED%86%B5%EB%A0%B9" title="대한민국의 대통령">대한민국의 대통령</a>은 <a href="/wiki/%ED%98%84%EB%8C%80_%EC%97%90%EC%BF%A0%EC%8A%A4" title="현대 에쿠스">현대 에쿠스 리무진 VL500</a>을 사용한다. 이들 중 3대는 현대자동차가 <a href="/wiki/2009%EB%85%84" title="2009년">2009년</a>에 기증한 것으로, 대한민국에서 제작된 자동차 중 최초로 <a href="/wiki/%EB%A9%94%EB%A5%B4%EC%84%B8%EB%8D%B0%EC%8A%A4-%EB%B2%A4%EC%B8%A0" title="메르세데스-벤츠">메르세데스-벤츠</a>와 <a href="/wiki/BMW" title="BMW">BMW</a>의 방탄차에 근접하는 수준의 성능을 갖춘, 생산 라인에서부터 방탄화된 차종이다. 그 전에는 <a href="/wiki/2005%EB%85%84" title="2005년">2005년</a>에 구입한 방탄 처리된 <a href="/wiki/BMW_7_%EC%8B%9C%EB%A6%AC%EC%A6%88" title="BMW 7 시리즈">BMW 7 시리즈</a>와, <a href="/wiki/2002%EB%85%84" title="2002년">2002년</a>에 구입한 <a class="mw-redirect" href="/wiki/%EB%A9%94%EB%A5%B4%EC%84%B8%EB%8D%B0%EC%8A%A4-%EB%B2%A4%EC%B8%A0_S%ED%81%B4%EB%9E%98%EC%8A%A4" title="메르세데스-벤츠 S클래스">메르세데스-벤츠 S클래스</a> (W220) 가 사용되었다. 메르세데스-벤츠 S클래스 (W220) 는 <a href="/wiki/%EB%85%B8%EB%AC%B4%ED%98%84" title="노무현">노무현</a> 전 대통령이 <a href="/wiki/2007%EB%85%84" title="2007년">2007년</a> <a class="mw-redirect" href="/wiki/%ED%8F%89%EC%96%91" title="평양">평양</a>에서 열린 <a class="mw-redirect" href="/wiki/%EB%82%A8%EB%B6%81_%EC%A0%95%EC%83%81_%ED%9A%8C%EB%8B%B4" title="남북 정상 회담">남북 정상 회담</a>에 사용하기도 하였다. 대통령의 차량 행렬은 <a href="/wiki/%EC%89%90%EB%B3%B4%EB%A0%88_%ED%83%80%ED%98%B8" title="쉐보레 타호">쉐보레 타호</a>, <a href="/wiki/%EC%89%90%EB%B3%B4%EB%A0%88_%EC%84%9C%EB%B2%84%EB%B2%88" title="쉐보레 서버번">쉐보레 서버번</a>과 <a class="new" href="/w/index.php?title=%ED%8F%AC%EB%93%9C_%EC%9D%B5%EC%8A%A4%EC%BB%A4%EC%A0%BC&amp;action=edit&amp;redlink=1" title="포드 익스커젼 (없는 문서)">포드 익스커젼</a>, 그리고 똑같이 생긴 메르세데스-벤츠 S클래스 (W220) 여러 대가 호위한다.</p>
<h2><span class="mw-headline" id=".EA.B5.AC_.EB.8F.85.EC.9D.BC_.EB.AF.BC.EC.A3.BC_.EA.B3.B5.ED.99.94.EA.B5.AD_.28.EB.8F.99.EB.8F.85.29">구 <a class="mw-redirect" href="/wiki/%EB%8F%85%EC%9D%BC_%EB%AF%BC%EC%A3%BC_%EA%B3%B5%ED%99%94%EA%B5%AD" title="독일 민주 공화국">독일 민주 공화국</a> (동독)</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EA%B5%AD%EA%B0%80%EC%9B%90%EC%88%98_%EC%9D%98%EC%A0%84%EC%B0%A8&amp;action=edit&amp;section=4" title="부분 편집: 구 독일 민주 공화국 (동독)">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>구 동독에서는 <a class="extiw" href="https://en.wikipedia.org/wiki/Trabant" title="en:Trabant">트라반트</a>와 <a class="extiw" href="https://en.wikipedia.org/wiki/Wartburg_(car)" title="en:Wartburg (car)">바르트부르그</a>를 생산했지만, 모두 국가 의전차로는 부적절하다고 판단되었다. 동독의 지도자들은 소련제 자동차와 <a href="/wiki/%EC%8A%A4%EC%9B%A8%EB%8D%B4" title="스웨덴">스웨덴</a>제 <a href="/wiki/%EB%B3%BC%EB%B3%B4" title="볼보">볼보</a>를 사용하였고, <a class="mw-redirect" href="/wiki/%EB%8F%85%EC%9D%BC_%ED%86%B5%EC%9D%BC%EC%82%AC%ED%9A%8C%EB%8B%B9" title="독일 통일사회당">독일 통일사회당</a> 서기장이었던 <a href="/wiki/%EC%97%90%EB%A6%AC%ED%9E%88_%ED%98%B8%EB%84%A4%EC%BB%A4" title="에리히 호네커">에리히 호네커</a>는 가운데를 늘린 (스트레치드) <a class="new" href="/w/index.php?title=%EC%8B%9C%ED%8A%B8%EB%A1%9C%EC%97%A5_CX&amp;action=edit&amp;redlink=1" title="시트로엥 CX (없는 문서)">시트로엥 CX</a>와 <a class="new" href="/w/index.php?title=%EB%B3%BC%EB%B3%B4_700&amp;action=edit&amp;redlink=1" title="볼보 700 (없는 문서)">볼보 700</a> 모델 여러 대를 사용하였다.</p>
<h2><span class="mw-headline" id=".EB.9D.BC.ED.8A.B8.EB.B9.84.EC.95.84"><a href="/wiki/%EB%9D%BC%ED%8A%B8%EB%B9%84%EC%95%84" title="라트비아">라트비아</a></span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EA%B5%AD%EA%B0%80%EC%9B%90%EC%88%98_%EC%9D%98%EC%A0%84%EC%B0%A8&amp;action=edit&amp;section=5" title="부분 편집: 라트비아">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p><a href="/wiki/%EB%9D%BC%ED%8A%B8%EB%B9%84%EC%95%84%EC%9D%98_%EB%8C%80%ED%86%B5%EB%A0%B9" title="라트비아의 대통령">라트비아의 대통령</a>은 <a href="/wiki/%EB%A9%94%EB%A5%B4%EC%84%B8%EB%8D%B0%EC%8A%A4-%EB%B2%A4%EC%B8%A0_S_%ED%81%B4%EB%9E%98%EC%8A%A4" title="메르세데스-벤츠 S 클래스">메르세데스-벤츠 S500</a>을 타며, 라트비아의 국장을 번호판 대신 달고 차 앞부분에 국기를 단다.</p>
<p>대부분의 라트비아 국회의원들은 <a class="new" href="/w/index.php?title=%EB%B3%BC%EB%B3%B4_900&amp;action=edit&amp;redlink=1" title="볼보 900 (없는 문서)">볼보 900</a> 시리즈나 <a href="/wiki/%EC%95%84%EC%9A%B0%EB%94%94_A6" title="아우디 A6">아우디 A6</a>, <a href="/wiki/%EB%A9%94%EB%A5%B4%EC%84%B8%EB%8D%B0%EC%8A%A4-%EB%B2%A4%EC%B8%A0_E_%ED%81%B4%EB%9E%98%EC%8A%A4" title="메르세데스-벤츠 E 클래스">메르세데스-벤츠 E 클래스</a>를 타지만, <a class="new" href="/w/index.php?title=%EB%B3%BC%EB%B3%B4_S80&amp;action=edit&amp;redlink=1" title="볼보 S80 (없는 문서)">볼보 S80</a>, <a class="new" href="/w/index.php?title=%EB%B3%BC%EB%B3%B4_S70&amp;action=edit&amp;redlink=1" title="볼보 S70 (없는 문서)">볼보 S70</a> 등의 다른 고급차들도 사용되고 있다.</p>
<h2><span class="mw-headline" id=".EB.9F.AC.EC.8B.9C.EC.95.84"><a href="/wiki/%EB%9F%AC%EC%8B%9C%EC%95%84" title="러시아">러시아</a></span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EA%B5%AD%EA%B0%80%EC%9B%90%EC%88%98_%EC%9D%98%EC%A0%84%EC%B0%A8&amp;action=edit&amp;section=6" title="부분 편집: 러시아">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p><a href="/wiki/%EB%9F%AC%EC%8B%9C%EC%95%84%EC%9D%98_%EB%8C%80%ED%86%B5%EB%A0%B9" title="러시아의 대통령">러시아의 대통령</a>은 방탄 처리된 <a class="mw-redirect" href="/wiki/%EB%A9%94%EB%A5%B4%EC%84%B8%EB%8D%B0%EC%8A%A4-%EB%B2%A4%EC%B8%A0_S%ED%81%B4%EB%9E%98%EC%8A%A4" title="메르세데스-벤츠 S클래스">메르세데스-벤츠 S클래스</a> 스트레치드 리무진을 주로 탄다. 처음으로 사용된 차종인 W140 계열 변형은 <a href="/wiki/%EB%B3%B4%EB%A6%AC%EC%8A%A4_%EC%98%90%EC%B9%9C" title="보리스 옐친">보리스 옐친</a> 전 대통령에 의해 구입되었고 최근 들어 W221 계열로 대체되고 있다. 대통령의 차량 행렬은 일반적으로 경찰의 오토바이 경호대와 더불어 <a href="/wiki/%EB%A9%94%EB%A5%B4%EC%84%B8%EB%8D%B0%EC%8A%A4-%EB%B2%A4%EC%B8%A0_G_%ED%81%B4%EB%9E%98%EC%8A%A4" title="메르세데스-벤츠 G 클래스">메르세데스-벤츠 G 클래스</a>, <a href="/wiki/%EB%A9%94%EB%A5%B4%EC%84%B8%EB%8D%B0%EC%8A%A4-%EB%B2%A4%EC%B8%A0_E_%ED%81%B4%EB%9E%98%EC%8A%A4" title="메르세데스-벤츠 E 클래스">E클래스</a>, 표준형 <a href="/wiki/%EB%A9%94%EB%A5%B4%EC%84%B8%EB%8D%B0%EC%8A%A4-%EB%B2%A4%EC%B8%A0_S_%ED%81%B4%EB%9E%98%EC%8A%A4" title="메르세데스-벤츠 S 클래스">S클래스</a>, <a href="/wiki/BMW_5_%EC%8B%9C%EB%A6%AC%EC%A6%88" title="BMW 5 시리즈">BMW 5 시리즈</a>, <a class="new" href="/w/index.php?title=%ED%8F%AD%EC%8A%A4%EB%B0%94%EA%B2%90_%EC%BA%90%EB%9F%AC%EB%B2%A8&amp;action=edit&amp;redlink=1" title="폭스바겐 캐러벨 (없는 문서)">폭스바겐 캐러벨</a> (Volkswagen Caravelle) 등으로 이루어진 지원 차량의 호위를 받는다. 러시아의 장군과 군관들은 <a class="extiw" href="https://en.wikipedia.org/wiki/Volga_(automobile)" title="en:Volga (automobile)">볼가</a> (Volga) 세단을 타며, 이들 볼가 세단들에는 더 좋은 다른 차량의 엔진이 사용되고 기타 기술적 개량이 이루어졌다.</p>
<p><a class="mw-redirect" href="/wiki/%EC%86%8C%EB%B9%84%EC%97%90%ED%8A%B8_%EC%97%B0%EB%B0%A9" title="소비에트 연방">소비에트 연방</a> 시절 <a class="new" href="/w/index.php?title=%EC%86%8C%EB%B9%84%EC%97%90%ED%8A%B8_%EC%97%B0%EB%B0%A9_%EA%B3%B5%EC%82%B0%EB%8B%B9_%EC%A4%91%EC%95%99%EC%9C%84%EC%9B%90%ED%9A%8C_%EC%84%9C%EA%B8%B0%EC%9E%A5&amp;action=edit&amp;redlink=1" title="소비에트 연방 공산당 중앙위원회 서기장 (없는 문서)">공산당 중앙위원회 서기장</a>은 항상 소련제 질 (ZIL) 리무진 (<a class="extiw" href="https://en.wikipedia.org/wiki/ZIL-41047" title="en:ZIL-41047">ZIL-41047</a>) 을 탔고, <a class="extiw" href="https://en.wikipedia.org/wiki/Chaika" title="en:Chaika">차이카</a> (Chaika) 세단 2대의 호위를 받았다. 질 리무진 2대는 여전히 크렘린 차고에서 관리되고 있으며, 승전기념일에 열리는 열병식에 종종 모습을 보이고 있다.</p>

<p><a href="/wiki/%EB%A3%A8%EB%A7%88%EB%8B%88%EC%95%84%EC%9D%98_%EB%8C%80%ED%86%B5%EB%A0%B9" title="루마니아의 대통령">루마니아의 대통령</a>은 <a class="mw-redirect" href="/wiki/%EB%A9%94%EB%A5%B4%EC%84%B8%EB%8D%B0%EC%8A%A4-%EB%B2%A4%EC%B8%A0_S%ED%81%B4%EB%9E%98%EC%8A%A4" title="메르세데스-벤츠 S클래스">메르세데스-벤츠 S클래스</a>에 타며, 검은 <a href="/wiki/BMW_5_%EC%8B%9C%EB%A6%AC%EC%A6%88" title="BMW 5 시리즈">BMW 5 시리즈 (E60)</a> 으로 이루어진 차량 행렬이 호위한다. 대통령은 종종 개인 소유의 SUV <a class="new" href="/w/index.php?title=%EB%8B%A4%EC%B9%98%EC%95%84_%EB%8D%94%EC%8A%A4%ED%84%B0&amp;action=edit&amp;redlink=1" title="다치아 더스터 (없는 문서)">다치아 더스터</a>(Dacia Duster)를 타기도 한다.</p>

<h2><span class="mw-headline" id=".EB.B0.94.ED.95.98.EB.A7.88"><a href="/wiki/%EB%B0%94%ED%95%98%EB%A7%88" title="바하마">바하마</a></span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EA%B5%AD%EA%B0%80%EC%9B%90%EC%88%98_%EC%9D%98%EC%A0%84%EC%B0%A8&amp;action=edit&amp;section=11" title="부분 편집: 바하마">편집</a><span class="mw-editsection-bracket">]</span></span></h2>

<h2><span class="mw-headline" id=".EB.B6.80.ED.83.84"><a href="/wiki/%EB%B6%80%ED%83%84" title="부탄">부탄</a></span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EA%B5%AD%EA%B0%80%EC%9B%90%EC%88%98_%EC%9D%98%EC%A0%84%EC%B0%A8&amp;action=edit&amp;section=13" title="부분 편집: 부탄">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>부탄의 이전 국왕인 <a href="/wiki/%EC%A7%80%EB%AF%B8_%EC%8B%B1%EA%B2%8C_%EC%99%95%EC%B6%95" title="지미 싱게 왕축">지미 싱게 왕축</a>은 국내에서 이동할 때 <a href="/wiki/%ED%86%A0%EC%9A%94%ED%83%80_%EB%9E%9C%EB%93%9C%ED%81%AC%EB%A3%A8%EC%A0%80" title="토요타 랜드크루저">토요타 랜드크루저</a>를 사용하였고, 지금의 국왕인 <a href="/wiki/%EC%A7%80%EA%B7%B8%EB%A9%94_%EC%BC%80%EC%82%AC%EB%A5%B4_%EB%82%A8%EA%B8%B0%EC%97%98_%EC%99%95%EC%B6%95" title="지그메 케사르 남기엘 왕축">지그메 케사르 남기엘 왕축</a>은 <a class="new" href="/w/index.php?title=%EB%A0%88%EC%9D%B8%EC%A7%80%EB%A1%9C%EB%B2%84&amp;action=edit&amp;redlink=1" title="레인지로버 (없는 문서)">레인지로버</a>를 사용한다.</p>
<h2><span class="mw-headline" id=".EC.84.B8.EB.A5.B4.EB.B9.84.EC.95.84"><a href="/wiki/%EC%84%B8%EB%A5%B4%EB%B9%84%EC%95%84" title="세르비아">세르비아</a></span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EA%B5%AD%EA%B0%80%EC%9B%90%EC%88%98_%EC%9D%98%EC%A0%84%EC%B0%A8&amp;action=edit&amp;section=14" title="부분 편집: 세르비아">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p><a href="/wiki/%EC%84%B8%EB%A5%B4%EB%B9%84%EC%95%84%EC%9D%98_%EB%8C%80%ED%86%B5%EB%A0%B9" title="세르비아의 대통령">세르비아의 대통령</a>은 B7급 방탄 능력을 가진 <a href="/wiki/%EB%A9%94%EB%A5%B4%EC%84%B8%EB%8D%B0%EC%8A%A4-%EB%B2%A4%EC%B8%A0_S_%ED%81%B4%EB%9E%98%EC%8A%A4" title="메르세데스-벤츠 S 클래스">메르세데스-벤츠 S600</a>를 사용하며, <a href="/wiki/%EC%84%B8%EB%A5%B4%EB%B9%84%EC%95%84%EC%9D%98_%EC%B4%9D%EB%A6%AC" title="세르비아의 총리">세르비아의 총리</a>는 방탄 처리된 <a class="new" href="/w/index.php?title=%EC%95%84%EC%9A%B0%EB%94%94_A5&amp;action=edit&amp;redlink=1" title="아우디 A5 (없는 문서)">아우디 A5</a>를 사용한다.</p>
<h2><span class="mw-headline" id=".EC.8B.B1.EA.B0.80.ED.8F.AC.EB.A5.B4"><a href="/wiki/%EC%8B%B1%EA%B0%80%ED%8F%AC%EB%A5%B4" title="싱가포르">싱가포르</a></span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EA%B5%AD%EA%B0%80%EC%9B%90%EC%88%98_%EC%9D%98%EC%A0%84%EC%B0%A8&amp;action=edit&amp;section=15" title="부분 편집: 싱가포르">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p><a href="/wiki/%EC%8B%B1%EA%B0%80%ED%8F%AC%EB%A5%B4%EC%9D%98_%EB%8C%80%ED%86%B5%EB%A0%B9" title="싱가포르의 대통령">싱가포르의 대통령</a>은 <a href="/wiki/2010%EB%85%84" title="2010년">2010년</a>형 <a class="mw-redirect" href="/wiki/%EB%A9%94%EB%A5%B4%EC%84%B8%EB%8D%B0%EC%8A%A4-%EB%B2%A4%EC%B8%A0_S%ED%81%B4%EB%9E%98%EC%8A%A4" title="메르세데스-벤츠 S클래스">메르세데스-벤츠 S350L</a>과 <a href="/wiki/2001%EB%85%84" title="2001년">2001년</a>형 <a href="/wiki/%EB%A0%89%EC%84%9C%EC%8A%A4_LS" title="렉서스 LS">렉서스 LS430</a> 중 하나를 선택하여 사용한다. 이 두 차량은 이전에 사용된 모든 국가원수 의전차와 마찬가지로 모두 흰색이다. 국경일 퍼레이드 (<a class="extiw" href="https://en.wikipedia.org/wiki/National_Day_Parade" title="en:National Day Parade">National Day Parade</a>) 와 같은 공식 행사에 사용될 때에는 대통령의 문장을 번호판 대신 달며, 평상시에는 'SEP1' 번호판을 단다.</p>
<p><a href="/wiki/%EC%8B%B1%EA%B0%80%ED%8F%AC%EB%A5%B4%EC%9D%98_%EC%B4%9D%EB%A6%AC" title="싱가포르의 총리">싱가포르의 총리</a>는 보통 흰색의 표준형 <a class="mw-redirect" href="/wiki/%EB%A9%94%EB%A5%B4%EC%84%B8%EB%8D%B0%EC%8A%A4-%EB%B2%A4%EC%B8%A0_S%ED%81%B4%EB%9E%98%EC%8A%A4" title="메르세데스-벤츠 S클래스">메르세데스-벤츠 S280</a>을 타며, 일반 번호판을 사용한다. 싱가포르 총리실은 국가 행사와 외국 귀빈의 방문에 대비하여 4대의 흰색 벤츠를 유지하고 있다.</p>
<p>정치적 피임명자 중에서는 대통령과 총리, 국회의장에게만 관용차가 제공되며, 내각의 다른 모든 구성원들은 개인 차량을 사용한다.</p>
<h2><span class="mw-headline" id=".EC.95.84.EB.A5.B4.ED.97.A8.ED.8B.B0.EB.82.98"><a href="/wiki/%EC%95%84%EB%A5%B4%ED%97%A8%ED%8B%B0%EB%82%98" title="아르헨티나">아르헨티나</a></span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EA%B5%AD%EA%B0%80%EC%9B%90%EC%88%98_%EC%9D%98%EC%A0%84%EC%B0%A8&amp;action=edit&amp;section=16" title="부분 편집: 아르헨티나">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p><a href="/wiki/%EC%95%84%EB%A5%B4%ED%97%A8%ED%8B%B0%EB%82%98%EC%9D%98_%EB%8C%80%ED%86%B5%EB%A0%B9" title="아르헨티나의 대통령">아르헨티나의 대통령</a>은 방탄 처리되고 일반 번호판이 달린 <a href="/wiki/%EC%95%84%EC%9A%B0%EB%94%94_A8" title="아우디 A8">아우디 A8</a>을 사용한다.</p>
<h2><span class="mw-headline" id=".EC.95.84.EC.9D.B4.EC.8A.AC.EB.9E.80.EB.93.9C"><a href="/wiki/%EC%95%84%EC%9D%B4%EC%8A%AC%EB%9E%80%EB%93%9C" title="아이슬란드">아이슬란드</a></span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EA%B5%AD%EA%B0%80%EC%9B%90%EC%88%98_%EC%9D%98%EC%A0%84%EC%B0%A8&amp;action=edit&amp;section=17" title="부분 편집: 아이슬란드">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p><a href="/wiki/%EC%95%84%EC%9D%B4%EC%8A%AC%EB%9E%80%EB%93%9C%EC%9D%98_%EB%8C%80%ED%86%B5%EB%A0%B9" title="아이슬란드의 대통령">아이슬란드의 대통령</a> <a class="new" href="/w/index.php?title=%EC%98%AC%EB%9D%BC%ED%91%B8%EB%A5%B4_%EB%9D%BC%EA%B7%B8%EB%82%98%EB%A5%B4_%EA%B7%B8%EB%A6%BC%EC%86%90&amp;action=edit&amp;redlink=1" title="올라푸르 라그나르 그림손 (없는 문서)">올라푸르 라그나르 그림손</a>은 운전사가 딸리고 '1' 번호판을 단 <a href="/wiki/%EB%A0%89%EC%84%9C%EC%8A%A4_LS" title="렉서스 LS">렉서스 LS600hL</a>을 타며, 특별 행사 시에는 <a href="/wiki/1942%EB%85%84" title="1942년">1942년</a>형 패커드(Packard)도 사용한다.</p>
<h2><span class="mw-headline" id=".EC.95.84.EC.A0.9C.EB.A5.B4.EB.B0.94.EC.9D.B4.EC.9E.94"><a href="/wiki/%EC%95%84%EC%A0%9C%EB%A5%B4%EB%B0%94%EC%9D%B4%EC%9E%94" title="아제르바이잔">아제르바이잔</a></span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EA%B5%AD%EA%B0%80%EC%9B%90%EC%88%98_%EC%9D%98%EC%A0%84%EC%B0%A8&amp;action=edit&amp;section=18" title="부분 편집: 아제르바이잔">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p><a href="/wiki/%EC%95%84%EC%A0%9C%EB%A5%B4%EB%B0%94%EC%9D%B4%EC%9E%94%EC%9D%98_%EB%8C%80%ED%86%B5%EB%A0%B9" title="아제르바이잔의 대통령">아제르바이잔의 대통령</a> <a href="/wiki/%EC%9D%BC%ED%95%A8_%EC%95%8C%EB%A6%AC%EC%98%88%ED%94%84" title="일함 알리예프">일함 알리예프</a>는 방탄 처리된 <a class="new" href="/w/index.php?title=%EB%A7%88%EC%9D%B4%EB%B0%94%ED%9D%90_62&amp;action=edit&amp;redlink=1" title="마이바흐 62 (없는 문서)">마이바흐 62</a>를 사용한다. 방탄 처리된, 맞춤형 <a href="/wiki/%EB%A9%94%EB%A5%B4%EC%84%B8%EB%8D%B0%EC%8A%A4-%EB%B2%A4%EC%B8%A0_S_%ED%81%B4%EB%9E%98%EC%8A%A4" title="메르세데스-벤츠 S 클래스">메르세데스-벤츠 W221 S600 가드 풀먼</a>도 종종 사용된다.</p>
<h2><span class="mw-headline" id=".EC.97.90.EC.8A.A4.ED.86.A0.EB.8B.88.EC.95.84"><a href="/wiki/%EC%97%90%EC%8A%A4%ED%86%A0%EB%8B%88%EC%95%84" title="에스토니아">에스토니아</a></span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EA%B5%AD%EA%B0%80%EC%9B%90%EC%88%98_%EC%9D%98%EC%A0%84%EC%B0%A8&amp;action=edit&amp;section=19" title="부분 편집: 에스토니아">편집</a><span class="mw-editsection-bracket">]</span></span></h2>

<p>오스트리아의 정부 요인들은 <a class="mw-redirect" href="/wiki/%EB%A9%94%EB%A5%B4%EC%84%B8%EB%8D%B0%EC%8A%A4-%EB%B2%A4%EC%B8%A0_S%ED%81%B4%EB%9E%98%EC%8A%A4" title="메르세데스-벤츠 S클래스">메르세데스-벤츠 S클래스</a>, <a href="/wiki/%EC%95%84%EC%9A%B0%EB%94%94_A8" title="아우디 A8">아우디 A8</a>, <a href="/wiki/BMW_7_%EC%8B%9C%EB%A6%AC%EC%A6%88" title="BMW 7 시리즈">BMW 7 시리즈</a>를 사용한다.</p>

<h2><span class="mw-headline" id=".EC.9D.B4.EC.8A.A4.EB.9D.BC.EC.97.98"><a href="/wiki/%EC%9D%B4%EC%8A%A4%EB%9D%BC%EC%97%98" title="이스라엘">이스라엘</a></span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EA%B5%AD%EA%B0%80%EC%9B%90%EC%88%98_%EC%9D%98%EC%A0%84%EC%B0%A8&amp;action=edit&amp;section=22" title="부분 편집: 이스라엘">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p><a href="/wiki/%EC%9D%B4%EC%8A%A4%EB%9D%BC%EC%97%98%EC%9D%98_%EC%B4%9D%EB%A6%AC" title="이스라엘의 총리">이스라엘의 총리</a>는 <a href="/wiki/%ED%86%A0%EC%9A%94%ED%83%80_%EB%9E%9C%EB%93%9C%ED%81%AC%EB%A3%A8%EC%A0%80" title="토요타 랜드크루저">토요타 랜드크루저</a>와 <a class="new" href="/w/index.php?title=%EB%A0%89%EC%84%9C%EC%8A%A4_CT200h&amp;action=edit&amp;redlink=1" title="렉서스 CT200h (없는 문서)">렉서스 CT200h</a>, <a class="new" href="/w/index.php?title=%EB%B3%BC%EB%B3%B4_S80&amp;action=edit&amp;redlink=1" title="볼보 S80 (없는 문서)">볼보 S80</a>, <a href="/wiki/%EC%95%84%EC%9A%B0%EB%94%94_A8" title="아우디 A8">아우디 A8</a>을 사용하며, <a href="/wiki/%EC%9D%B4%EC%8A%A4%EB%9D%BC%EC%97%98%EC%9D%98_%EB%8C%80%ED%86%B5%EB%A0%B9" title="이스라엘의 대통령">이스라엘의 대통령</a>은 <a href="/wiki/%EC%95%84%EC%9A%B0%EB%94%94_A8" title="아우디 A8">아우디 A8</a>과 <a href="/wiki/%EB%B3%BC%EB%B3%B4" title="볼보">볼보</a> 스트레치드 리무진을 사용한다.</p>
<h2><span class="mw-headline" id=".EC.9D.B4.EC.A7.91.ED.8A.B8"><a href="/wiki/%EC%9D%B4%EC%A7%91%ED%8A%B8" title="이집트">이집트</a></span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EA%B5%AD%EA%B0%80%EC%9B%90%EC%88%98_%EC%9D%98%EC%A0%84%EC%B0%A8&amp;action=edit&amp;section=23" title="부분 편집: 이집트">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>이집트의 전 대통령 <a href="/wiki/%ED%98%B8%EC%8A%A4%EB%8B%88_%EB%AC%B4%EB%B0%94%EB%9D%BC%ED%81%AC" title="호스니 무바라크">호스니 무바라크</a>는 <a href="/wiki/2011%EB%85%84_2%EC%9B%94" title="2011년 2월">2011년 2월</a> 퇴임하기 전까지 검은색의 방탄 처리된 <a class="mw-redirect" href="/wiki/%EB%A9%94%EB%A5%B4%EC%84%B8%EB%8D%B0%EC%8A%A4-%EB%B2%A4%EC%B8%A0_S%ED%81%B4%EB%9E%98%EC%8A%A4" title="메르세데스-벤츠 S클래스">메르세데스-벤츠 S클래스</a>를 사용하였다. 그 이전에 사용하던 구형 S클래스 의전차는 <a href="/wiki/1995%EB%85%84" title="1995년">1995년</a> <a href="/wiki/%EC%97%90%ED%8B%B0%EC%98%A4%ED%94%BC%EC%95%84" title="에티오피아">에티오피아</a>에서 발생한 암살 미수 사건에서 무바라크 전 대통령의 목숨을 구해 주었다.</p>

<h2><span class="mw-headline" id=".EC.A4.91.ED.99.94.EB.AF.BC.EA.B5.AD"><a href="/wiki/%EC%A4%91%ED%99%94%EB%AF%BC%EA%B5%AD" title="중화민국">중화민국</a></span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EA%B5%AD%EA%B0%80%EC%9B%90%EC%88%98_%EC%9D%98%EC%A0%84%EC%B0%A8&amp;action=edit&amp;section=24" title="부분 편집: 중화민국">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p><a href="/wiki/%EC%A4%91%ED%99%94%EB%AF%BC%EA%B5%AD%EC%9D%98_%EC%B4%9D%ED%86%B5" title="중화민국의 총통">중화민국의 총통</a>은 미국과의 밀접한 관계를 반영하여 역사적으로 <a href="/wiki/%EC%BA%90%EB%94%9C%EB%9D%BD" title="캐딜락">캐딜락</a> 세단을 이용해 왔다. <a href="/wiki/%EC%9E%A5%EC%A0%9C%EC%8A%A4" title="장제스">장제스</a>가 탄 두 대의 방탄 캐딜락은 <a href="/wiki/%ED%83%80%EC%9D%B4%EB%B2%A0%EC%9D%B4_%EC%8B%9C" title="타이베이 시">타이베이</a>의 <a href="/wiki/%EC%A4%91%EC%A0%95%EA%B8%B0%EB%85%90%EB%8B%B9" title="중정기념당">중정기념당</a>에 전시되어 있다. 그러나 <a href="/wiki/%EC%B2%9C%EC%88%98%EC%9D%B4%EB%B3%9C" title="천수이볜">천수이볜</a>은 이전까지 사용되던 <a class="new" href="/w/index.php?title=%EC%BA%90%EB%94%9C%EB%9D%BD_%EB%93%9C%EB%B9%8C&amp;action=edit&amp;redlink=1" title="캐딜락 드빌 (없는 문서)">캐딜락 드빌</a>의 사용을 중지하고, 첫 번째 임기에 <a class="mw-disambig" href="/wiki/%EB%A7%81%EC%BB%A8" title="링컨">링컨</a>을, 두 번째 임기에 <a href="/wiki/BMW_7_%EC%8B%9C%EB%A6%AC%EC%A6%88" title="BMW 7 시리즈">BMW 7 시리즈</a>를 각각 사용하였다. 현재의 <a href="/wiki/%EB%A7%88%EC%9E%89%EC%A3%BC" title="마잉주">마잉주</a>는 2008년과 2012년에 구입한 BMW 740i를 이용한다.</p>
<h2><span class="mw-headline" id=".EC.A4.91.ED.99.94.EC.9D.B8.EB.AF.BC.EA.B3.B5.ED.99.94.EA.B5.AD"><a href="/wiki/%EC%A4%91%ED%99%94%EC%9D%B8%EB%AF%BC%EA%B3%B5%ED%99%94%EA%B5%AD" title="중화인민공화국">중화인민공화국</a></span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EA%B5%AD%EA%B0%80%EC%9B%90%EC%88%98_%EC%9D%98%EC%A0%84%EC%B0%A8&amp;action=edit&amp;section=25" title="부분 편집: 중화인민공화국">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>중화인민공화국의 역대 지도자들은 약 7미터 길이의 훙치(紅旗)를 사용해 왔다. <a href="/wiki/2009%EB%85%84" title="2009년">2009년</a> <a href="/wiki/%EA%B5%AD%EA%B2%BD%EC%A0%88" title="국경절">국경절</a>에 있었던 열병식에서 <a href="/wiki/%ED%9B%84%EC%A7%84%ED%83%80%EC%98%A4" title="후진타오">후진타오</a>는 <a class="new" href="/w/index.php?title=%ED%9B%99%EC%B9%98_HQE&amp;action=edit&amp;redlink=1" title="훙치 HQE (없는 문서)">훙치 HQE</a> 리무진을 사용하였고, <a href="/wiki/BMW_5_%EC%8B%9C%EB%A6%AC%EC%A6%88" title="BMW 5 시리즈">BMW 5 시리즈</a> 경찰 차량의 호위를 받았다.</p>
<h2><span class="mw-headline" id=".EC.BD.94.EC.8A.A4.ED.83.80.EB.A6.AC.EC.B9.B4"><a href="/wiki/%EC%BD%94%EC%8A%A4%ED%83%80%EB%A6%AC%EC%B9%B4" title="코스타리카">코스타리카</a></span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EA%B5%AD%EA%B0%80%EC%9B%90%EC%88%98_%EC%9D%98%EC%A0%84%EC%B0%A8&amp;action=edit&amp;section=26" title="부분 편집: 코스타리카">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>코스타리카의 대통령 <a href="/wiki/%EB%9D%BC%EC%9A%B0%EB%9D%BC_%EC%B9%9C%EC%B9%98%EC%95%BC" title="라우라 친치야">라우라 친치야</a>는 <a href="/wiki/%ED%98%84%EB%8C%80_%EC%97%90%EC%BF%A0%EC%8A%A4" title="현대 에쿠스">현대 에쿠스 리무진 VL500</a>을 국가원수 의전차로 사용한다. 코스타리카의 외교부 장관은 현재 <a href="/wiki/%EA%B8%B0%EC%95%84_%EC%98%A4%ED%94%BC%EB%9F%AC%EC%8A%A4" title="기아 오피러스">기아 오피러스</a>를 사용한다.</p>
<h2><span class="mw-headline" id=".EC.BF.A0.EC.9B.A8.EC.9D.B4.ED.8A.B8"><a href="/wiki/%EC%BF%A0%EC%9B%A8%EC%9D%B4%ED%8A%B8" title="쿠웨이트">쿠웨이트</a></span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EA%B5%AD%EA%B0%80%EC%9B%90%EC%88%98_%EC%9D%98%EC%A0%84%EC%B0%A8&amp;action=edit&amp;section=27" title="부분 편집: 쿠웨이트">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>쿠웨이트의 통치자 (에미르) 는 <a class="mw-redirect" href="/wiki/%EB%A9%94%EB%A5%B4%EC%84%B8%EB%8D%B0%EC%8A%A4-%EB%B2%A4%EC%B8%A0_S%ED%81%B4%EB%9E%98%EC%8A%A4" title="메르세데스-벤츠 S클래스">메르세데스-벤츠 S클래스</a>나 <a href="/wiki/BMW_7_%EC%8B%9C%EB%A6%AC%EC%A6%88" title="BMW 7 시리즈">BMW 7 시리즈</a>를 일상에 사용한다. 에미르의 차량 행렬은 쉐보레 서버번과 경찰 차량들의 호위를 받는 3대의 똑같이 생긴 차량(S클래스 혹은 7시리즈)으로 이루어져 있다. 국가적 행사에는 <a class="mw-disambig" href="/wiki/%EB%A1%A4%EC%8A%A4%EB%A1%9C%EC%9D%B4%EC%8A%A4" title="롤스로이스">롤스로이스</a>가 사용된다.</p>
<h2><span class="mw-headline" id=".ED.81.AC.EB.A1.9C.EC.95.84.ED.8B.B0.EC.95.84"><a href="/wiki/%ED%81%AC%EB%A1%9C%EC%95%84%ED%8B%B0%EC%95%84" title="크로아티아">크로아티아</a></span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EA%B5%AD%EA%B0%80%EC%9B%90%EC%88%98_%EC%9D%98%EC%A0%84%EC%B0%A8&amp;action=edit&amp;section=28" title="부분 편집: 크로아티아">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p><a href="/wiki/%ED%81%AC%EB%A1%9C%EC%95%84%ED%8B%B0%EC%95%84%EC%9D%98_%EB%8C%80%ED%86%B5%EB%A0%B9" title="크로아티아의 대통령">크로아티아의 대통령</a>은 방탄 처리된 <a href="/wiki/%EB%A9%94%EB%A5%B4%EC%84%B8%EB%8D%B0%EC%8A%A4-%EB%B2%A4%EC%B8%A0_S_%ED%81%B4%EB%9E%98%EC%8A%A4" title="메르세데스-벤츠 S 클래스">메르세데스-벤츠 S600</a>를 사용하며, <a href="/wiki/%ED%81%AC%EB%A1%9C%EC%95%84%ED%8B%B0%EC%95%84%EC%9D%98_%EC%B4%9D%EB%A6%AC" title="크로아티아의 총리">크로아티아의 총리</a>는 방탄 처리된 <a href="/wiki/BMW_7_%EC%8B%9C%EB%A6%AC%EC%A6%88" title="BMW 7 시리즈">BMW 7 시리즈</a>를 사용한다.</p>
<h2><span class="mw-headline" id=".ED.84.B0.ED.82.A4"><a href="/wiki/%ED%84%B0%ED%82%A4" title="터키">터키</a></span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EA%B5%AD%EA%B0%80%EC%9B%90%EC%88%98_%EC%9D%98%EC%A0%84%EC%B0%A8&amp;action=edit&amp;section=29" title="부분 편집: 터키">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>터키의 주요 고위 정치인들은 <a href="/wiki/%EB%A9%94%EB%A5%B4%EC%84%B8%EB%8D%B0%EC%8A%A4-%EB%B2%A4%EC%B8%A0" title="메르세데스-벤츠">메르세데스-벤츠</a> 차량을 사용한다. <a href="/wiki/%ED%84%B0%ED%82%A4%EC%9D%98_%EB%8C%80%ED%86%B5%EB%A0%B9" title="터키의 대통령">터키의 대통령</a>과 <a class="mw-redirect" href="/wiki/%ED%84%B0%ED%82%A4%EC%9D%98_%EC%B4%9D%EB%A6%AC" title="터키의 총리">총리</a>는 <a class="mw-redirect" href="/wiki/%EB%A9%94%EB%A5%B4%EC%84%B8%EB%8D%B0%EC%8A%A4-%EB%B2%A4%EC%B8%A0_S%ED%81%B4%EB%9E%98%EC%8A%A4" title="메르세데스-벤츠 S클래스">메르세데스-벤츠 S클래스</a>를 사용한다. <a href="/wiki/%EB%B6%81%ED%82%A4%ED%94%84%EB%A1%9C%EC%8A%A4" title="북키프로스">북키프로스</a>의 <a class="new" href="/w/index.php?title=%EB%B6%81%ED%82%A4%ED%94%84%EB%A1%9C%EC%8A%A4%EC%9D%98_%EB%8C%80%ED%86%B5%EB%A0%B9&amp;action=edit&amp;redlink=1" title="북키프로스의 대통령 (없는 문서)">대통령</a>은 번호판 대신 금색 문장이 달린 붉은색 BMW 750Li를 사용한다.</p>

<p><a href="/wiki/%ED%8F%B4%EB%9E%80%EB%93%9C%EC%9D%98_%EB%8C%80%ED%86%B5%EB%A0%B9" title="폴란드의 대통령">폴란드의 대통령</a>은 방탄 처리된 <a href="/wiki/BMW_7_%EC%8B%9C%EB%A6%AC%EC%A6%88" title="BMW 7 시리즈">BMW E67 760Li</a>를 사용한다.</p>
<h2><span class="mw-headline" id=".ED.95.80.EB.9E.80.EB.93.9C"><a href="/wiki/%ED%95%80%EB%9E%80%EB%93%9C" title="핀란드">핀란드</a></span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EA%B5%AD%EA%B0%80%EC%9B%90%EC%88%98_%EC%9D%98%EC%A0%84%EC%B0%A8&amp;action=edit&amp;section=32" title="부분 편집: 핀란드">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p><a href="/wiki/%ED%95%80%EB%9E%80%EB%93%9C%EC%9D%98_%EB%8C%80%ED%86%B5%EB%A0%B9" title="핀란드의 대통령">핀란드의 대통령</a>은 방탄 처리된 <a href="/wiki/%EB%A9%94%EB%A5%B4%EC%84%B8%EB%8D%B0%EC%8A%A4-%EB%B2%A4%EC%B8%A0_S_%ED%81%B4%EB%9E%98%EC%8A%A4" title="메르세데스-벤츠 S 클래스">메르세데스-벤츠 S600L</a>를 타며, <a href="/wiki/%ED%95%80%EB%9E%80%EB%93%9C%EC%9D%98_%EA%B5%AD%EC%9E%A5" title="핀란드의 국장">핀란드의 국장</a>을 번호판 대신 사용한다. 국회의장도 대통령과 유사한 차량을 사용하며, <a href="/wiki/%ED%95%80%EB%9E%80%EB%93%9C%EC%9D%98_%EC%B4%9D%EB%A6%AC" title="핀란드의 총리">총리</a>의 공식 의전차는 회색의 방탄 처리된 <a href="/wiki/%EB%A9%94%EB%A5%B4%EC%84%B8%EB%8D%B0%EC%8A%A4-%EB%B2%A4%EC%B8%A0_S_%ED%81%B4%EB%9E%98%EC%8A%A4" title="메르세데스-벤츠 S 클래스">메르세데스-벤츠 S450 CDI</a>이다.</p>
<p>내각 구성원들의 대부분은 <a href="/wiki/%EC%95%84%EC%9A%B0%EB%94%94_A6" title="아우디 A6">아우디 A6</a>을 사용하지만, <a class="new" href="/w/index.php?title=%EB%B3%BC%EB%B3%B4_S80&amp;action=edit&amp;redlink=1" title="볼보 S80 (없는 문서)">볼보 S80</a>과 <a class="new" href="/w/index.php?title=%EC%82%AC%EB%B8%8C_9-5&amp;action=edit&amp;redlink=1" title="사브 9-5 (없는 문서)">사브 9-5</a>도 일부 사용된다. 외교부 장관이 타는 <a href="/wiki/%EC%95%84%EC%9A%B0%EB%94%94_A8" title="아우디 A8">아우디 A8</a>은 이들 중 유일하게 방탄 처리되어 있다.</p>
<h2><span class="mw-headline" id=".ED.99.8D.EC.BD.A9"><a href="/wiki/%ED%99%8D%EC%BD%A9" title="홍콩">홍콩</a></span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EA%B5%AD%EA%B0%80%EC%9B%90%EC%88%98_%EC%9D%98%EC%A0%84%EC%B0%A8&amp;action=edit&amp;section=33" title="부분 편집: 홍콩">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p><a class="mw-redirect" href="/wiki/%ED%99%8D%EC%BD%A9%ED%8A%B9%EB%B3%84%ED%96%89%EC%A0%95%EA%B5%AC" title="홍콩특별행정구">홍콩특별행정구</a>의 행정장관 <a href="/wiki/%EB%A0%81%EC%B6%98%EC%9E%89" title="렁춘잉">렁춘잉</a>은 <a href="/wiki/%EB%A0%89%EC%84%9C%EC%8A%A4_LS" title="렉서스 LS">렉서스 LS600hL</a>을 타며, 번호판이 아닌 홍콩 특구 문장을 사용한다. 행정장관은 종종 1997년형 <a href="/wiki/%EB%A9%94%EB%A5%B4%EC%84%B8%EB%8D%B0%EC%8A%A4-%EB%B2%A4%EC%B8%A0_S_%ED%81%B4%EB%9E%98%EC%8A%A4" title="메르세데스-벤츠 S 클래스">메르세데스-벤츠 S500L</a>을 사용하기도 한다.</p>
<p>정무사 사장, 종심법원 수석대법관과 경무처 처장을 비롯한 주요 정부 관원들에게는 각각 <a href="/wiki/%ED%8F%AD%EC%8A%A4%EB%B0%94%EA%B2%90_%ED%8E%98%EC%9D%B4%ED%86%A4" title="폭스바겐 페이톤">폭스바겐 페이톤</a>이 지급되어 있다. 입법회 주석은 <a href="/wiki/%EB%A0%89%EC%84%9C%EC%8A%A4_GS" title="렉서스 GS">렉서스 GS450h</a>를 사용하며 "LC1" 번호판을 사용한다.</p>
