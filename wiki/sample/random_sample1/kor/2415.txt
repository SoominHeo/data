<p><b>국립고궁박물관</b>(國立古宮博物館)은 <a href="/wiki/%EB%8C%80%ED%95%9C%EB%AF%BC%EA%B5%AD" title="대한민국">대한민국</a> <a href="/wiki/%EC%84%9C%EC%9A%B8%ED%8A%B9%EB%B3%84%EC%8B%9C" title="서울특별시">서울특별시</a> <a href="/wiki/%EC%A2%85%EB%A1%9C%EA%B5%AC" title="종로구">종로구</a>에 소재하는 <a href="/wiki/%EB%8C%80%ED%95%9C%EB%AF%BC%EA%B5%AD_%EB%AC%B8%ED%99%94%EC%9E%AC%EC%B2%AD" title="대한민국 문화재청">대한민국 문화재청</a> 소속의 국립박물관으로 국립고궁박물관장은 고위공무원 나급(2~3급 상당)의 일반직 공무원 또는 학예연구관으로 보한다. <a href="/wiki/%EC%A1%B0%EC%84%A0" title="조선">조선</a>, <a href="/wiki/%EB%8C%80%ED%95%9C%EC%A0%9C%EA%B5%AD" title="대한제국">대한제국</a> 시대의 왕실 복식 및 생활 관련 유물 약 40,000여 점을 소장·전시하고 있다.</p>

<h2><span class="mw-headline" id=".EC.97.B0.ED.98.81">연혁</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EA%B5%AD%EB%A6%BD%EA%B3%A0%EA%B6%81%EB%B0%95%EB%AC%BC%EA%B4%80&amp;action=edit&amp;section=1" title="부분 편집: 연혁">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p><a href="/wiki/1908%EB%85%84" title="1908년">1908년</a> <a href="/wiki/9%EC%9B%94" title="9월">9월</a>에 황실박물관으로 개관하였다.</p>
<p>문화재청은 <a href="/wiki/1980%EB%85%84%EB%8C%80" title="1980년대">1980년대</a> 중반부터 창경원을 <a href="/wiki/%EC%B0%BD%EA%B2%BD%EA%B6%81" title="창경궁">창경궁</a>으로 복원하고 일제에 의해 헐린 경복궁 전각들을 복원해나가기 시작했다. 그 과정에서 <a href="/wiki/1992%EB%85%84_10%EC%9B%94" title="1992년 10월">1992년 10월</a>에는 4대궁궐과 종묘, 능원 등에 흩어져 있던 궁중문화재를 모아 <a href="/wiki/%EB%8D%95%EC%88%98%EA%B6%81" title="덕수궁">덕수궁</a> 석조전에 궁중유물전시관을 개관하기도 했다.</p>
<p>하지만 덕수궁 석조전은 전시공간과 수장 공간이 턱없이 부족하고 규모조차 너무 작아 왕실문화를 홍보하고 연구 보존하는 데 수많은 어려움에 부딪혔기에 문화재청은 <a href="/wiki/1993%EB%85%84" title="1993년">1993년</a> 옛 <a href="/wiki/%EA%B5%AD%EB%A6%BD%EC%A4%91%EC%95%99%EB%B0%95%EB%AC%BC%EA%B4%80" title="국립중앙박물관">국립중앙박물관</a>이자 <a href="/wiki/%EC%A1%B0%EC%84%A0%EC%B4%9D%EB%8F%85%EB%B6%80" title="조선총독부">조선총독부</a> 건물이 있었던 자리에 왕궁박물관을 개관하기로 결정하고 <a href="/wiki/%EC%A1%B0%EC%84%A0%EC%B4%9D%EB%8F%85%EB%B6%80" title="조선총독부">조선총독부</a> 건물을 철거하였다. 이후 철저한 준비 끝에, 광복 60주년인 <a href="/wiki/2005%EB%85%84" title="2005년">2005년</a> <a href="/wiki/8%EC%9B%94_15%EC%9D%BC" title="8월 15일">8월 15일</a>에 <a href="/wiki/%EB%8D%95%EC%88%98%EA%B6%81" title="덕수궁">덕수궁</a> 석조전에서 <a href="/wiki/%EA%B2%BD%EB%B3%B5%EA%B6%81" title="경복궁">경복궁</a> 내 옛 국립중앙박물관 자리로 이전 개관하였다.</p>
