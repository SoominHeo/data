<p><b>국민당</b>(<span style="font-size: smaller;"><a href="/wiki/%EC%98%81%EC%96%B4" title="영어">영어</a>:</span> <span lang="en" xml:lang="en">National Party</span> <span style="font-size: smaller;"><a href="/wiki/%EC%95%84%ED%94%84%EB%A6%AC%EC%B9%B8%EC%8A%A4%EC%96%B4" title="아프리칸스어">아프리칸스어</a>:</span> <span lang="af" xml:lang="af">Nasionale Party</span>)은 <a href="/wiki/1948%EB%85%84" title="1948년">1948년</a>부터 <a href="/wiki/1997%EB%85%84" title="1997년">1997년</a>까지 존속했던 <a href="/wiki/%EB%82%A8%EC%95%84%ED%94%84%EB%A6%AC%EC%B9%B4_%EA%B3%B5%ED%99%94%EA%B5%AD" title="남아프리카 공화국">남아프리카 공화국</a>의 정당이었다. <a href="/wiki/1994%EB%85%84" title="1994년">1994년</a>까지 실권을 쥐고 있었으며, 강력한 인종 차별 정책을 실시하여 비난의 대상이 되었다. <a href="/wiki/1997%EB%85%84" title="1997년">1997년</a> <a href="/wiki/%EC%8B%A0%EA%B5%AD%EB%AF%BC%EB%8B%B9_(%EB%82%A8%EC%95%84%ED%94%84%EB%A6%AC%EC%B9%B4_%EA%B3%B5%ED%99%94%EA%B5%AD)" title="신국민당 (남아프리카 공화국)">신국당</a>이 창설됨에 따라 해체되었다.</p>
<h2><span class="mw-headline" id=".EC.97.AD.EC.82.AC">역사</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EA%B5%AD%EB%AF%BC%EB%8B%B9_(%EB%82%A8%EC%95%84%ED%94%84%EB%A6%AC%EC%B9%B4_%EA%B3%B5%ED%99%94%EA%B5%AD)&amp;action=edit&amp;section=1" title="부분 편집: 역사">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p><a class="new" href="/w/index.php?title=%EB%A3%A8%EC%9D%B4%EC%8A%A4_%EB%B3%B4%ED%83%80&amp;action=edit&amp;redlink=1" title="루이스 보타 (없는 문서)">루이스 보타</a>와 <a class="new" href="/w/index.php?title=%EC%96%80_%EC%8A%A4%EB%AC%B4%ED%8A%B8&amp;action=edit&amp;redlink=1" title="얀 스무트 (없는 문서)">얀 스무트</a>는 백인들이 남아프리카의 지배를 확고하게 하기 위해 <a class="new" href="/w/index.php?title=%EC%95%84%ED%94%84%EB%A6%AC%EC%B9%B4%EB%84%88_%EB%AF%BC%EC%A1%B1%EC%A3%BC%EC%9D%98&amp;action=edit&amp;redlink=1" title="아프리카너 민족주의 (없는 문서)">아프리카너 민족주의</a> 사상을 발전시켜 이 정당을 창설하게 된다. 아프리카너 농민들과 가난한 아프리카너들을 기반으로 하여 <a href="/wiki/1924%EB%85%84" title="1924년">1924년</a>에 백인 노동자 보호단체를 슬로건 노동당과 연합하여 총선에 승리하였고 <sup class="정리_필요 noprint">[<a href="/wiki/%EC%9C%84%ED%82%A4%EB%B0%B1%EA%B3%BC:%EB%AA%A8%ED%98%B8%ED%95%9C_%ED%91%9C%ED%98%84%EC%9D%84_%EC%82%BC%EA%B0%80_%EC%A3%BC%EC%84%B8%EC%9A%94" title="위키백과:모호한 표현을 삼가 주세요">모호한 표현</a>]</sup>, 권력을 장악하게 된다.</p>
<p>그러나 <a href="/wiki/1934%EB%85%84" title="1934년">1934년</a> 국민당과 <a class="new" href="/w/index.php?title=%EC%96%80_%EC%8A%A4%EB%AC%B4%ED%8A%B8&amp;action=edit&amp;redlink=1" title="얀 스무트 (없는 문서)">얀 스무트</a>의 통일당은 합병되었다. <a href="/wiki/%EB%8B%A4%EB%8B%88%EC%97%98_%ED%94%84%EB%9E%91%EC%88%98%EC%95%84_%EB%A7%90%EB%9E%80" title="다니엘 프랑수아 말란">다니엘 프랑수아 말란</a> 등은 합병에 강력하게 저항했고, 19명의 정치인들을 모아 끝내 이 정당을 결성한다. 통일당은 <a class="mw-redirect" href="/wiki/%EC%A0%9C2%EC%B0%A8_%EC%84%B8%EA%B3%84%EB%8C%80%EC%A0%84" title="제2차 세계대전">제2차 세계대전</a> 당시 연합국들을 지지하며 참전하였으나, 아프리카너들은 이러지 않았다는 이유 때문에 비난을 받았다. <a href="/wiki/1948%EB%85%84" title="1948년">1948년</a> 실권을 쥐게 되고, 마침내 <a href="/wiki/%EC%95%84%ED%8C%8C%EB%A5%B4%ED%8A%B8%ED%97%A4%EC%9D%B4%ED%8A%B8" title="아파르트헤이트">아파르트헤이트</a>를 실시하게 된다.</p>
<p><a href="/wiki/1978%EB%85%84" title="1978년">1978년</a> <a href="/wiki/9%EC%9B%94" title="9월">9월</a> 정권을 잡은 <a href="/wiki/%ED%94%BC%ED%84%B0%EB%A5%B4_%EB%B9%8C%EB%9F%BC_%EB%B3%B4%ED%83%80" title="피터르 빌럼 보타">피터르 빌럼 보타</a>는 인종차별을 없애야 한다고 주장했다. 그러나 이에 반대한 소수가 <a class="new" href="/w/index.php?title=%EB%82%A8%EC%95%84%EA%B3%B5_%EB%B3%B4%EC%88%98%EB%8B%B9&amp;action=edit&amp;redlink=1" title="남아공 보수당 (없는 문서)">남아공 보수당</a>을 창설하게 된다. 그러다 인종 차별이 서서히 풀리고, <a href="/wiki/1994%EB%85%84" title="1994년">1994년</a> <a href="/wiki/%EC%95%84%ED%94%84%EB%A6%AC%EC%B9%B4_%EB%AF%BC%EC%A1%B1%ED%9A%8C%EC%9D%98" title="아프리카 민족회의">아프리카 민족회의</a> 당이 정권을 잡게 된다. <a href="/wiki/1997%EB%85%84" title="1997년">1997년</a> 해체되었다.</p>
