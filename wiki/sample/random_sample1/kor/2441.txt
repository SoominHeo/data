<h2><span class="mw-headline" id=".EC.9D.B4.EB.B2.A4.ED.8A.B8_.EB.8C.80.ED.9A.8C.EC.9D.98_.EC.97.AD.EC.82.AC">이벤트 대회의 역사</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EA%B5%AD%EC%A0%9C_%EC%9E%90%EB%8F%99%EC%B0%A8_%EC%97%B0%EB%A7%B9&amp;action=edit&amp;section=3" title="부분 편집: 이벤트 대회의 역사">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<ul>
<li><a href="/wiki/1950%EB%85%84" title="1950년">1950년</a> - FIA가 처음으로 월드 챔피언십 자동차 경주 대회를 개최했다. 이 대회가 오늘 날의 <a href="/wiki/%ED%8F%AC%EB%AE%AC%EB%9F%AC_%EC%9B%90" title="포뮬러 원">F1</a>이다.</li>
<li><a href="/wiki/1955%EB%85%84" title="1955년">1955년</a> - FIA가 최초의 <a class="mw-redirect" href="/wiki/%EC%8A%A4%ED%8F%AC%EC%B8%A0_%EC%9E%90%EB%8F%99%EC%B0%A8" title="스포츠 자동차">스포츠 자동차</a> <a href="/wiki/%EC%9E%90%EB%8F%99%EC%B0%A8_%EA%B2%BD%EC%A3%BC" title="자동차 경주">경주</a> 대회인 <a class="new" href="/w/index.php?title=%EC%9B%94%EB%93%9C_%EC%97%94%EB%93%80%EB%9F%B0%EC%8A%A4_%EC%B1%94%ED%94%BC%EC%96%B8%EC%8B%AD&amp;action=edit&amp;redlink=1" title="월드 엔듀런스 챔피언십 (없는 문서)">월드 엔듀런스 챔피언십</a>(World Endurance Championship)을 개최했다. 이 대회는 <a href="/wiki/1981%EB%85%84" title="1981년">1981년</a>까지 자동차 제조 회사에만 우승 트로피를 주었으나, <a href="/wiki/1982%EB%85%84" title="1982년">1982년</a> 새로운 <a class="new" href="/w/index.php?title=%EA%B7%B8%EB%A3%B9_C&amp;action=edit&amp;redlink=1" title="그룹 C (없는 문서)">그룹 C</a> 규정이 만들어 지면서 자동차 운전자에게도 시상이 이루어졌다.</li>
<li><a href="/wiki/1973%EB%85%84" title="1973년">1973년</a> - FIA가 <a href="/wiki/%EC%84%B8%EA%B3%84_%EB%9E%A0%EB%A6%AC_%EC%84%A0%EC%88%98%EA%B6%8C_%EB%8C%80%ED%9A%8C" title="세계 랠리 선수권 대회">월드 랠리 챔피언십</a>(World Rally Championship)을 개최했다.</li>
</ul>
<h2><span class="mw-headline" id=".EC.9E.90.EB.8F.99.EC.B0.A8_.EA.B2.BD.EC.A3.BC_.EB.8C.80.ED.9A.8C"><a href="/wiki/%EC%9E%90%EB%8F%99%EC%B0%A8_%EA%B2%BD%EC%A3%BC" title="자동차 경주">자동차 경주</a> 대회</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EA%B5%AD%EC%A0%9C_%EC%9E%90%EB%8F%99%EC%B0%A8_%EC%97%B0%EB%A7%B9&amp;action=edit&amp;section=4" title="부분 편집: 자동차 경주 대회">편집</a><span class="mw-editsection-bracket">]</span></span></h2>

<li>FIA <a href="/wiki/%EC%9B%94%EB%93%9C_%ED%88%AC%EC%96%B4%EB%A7%81%EC%B9%B4_%EC%B1%94%ED%94%BC%EC%96%B8%EC%8B%AD" title="월드 투어링카 챔피언십">월드 투어링카 챔피언십</a>(World Touring Car Championship)</li>

<li>CIK-FIA <a class="new" href="/w/index.php?title=%EC%B9%B4%ED%8A%B8_%EC%9B%94%EB%93%9C_%EC%B1%94%ED%94%BC%EC%96%B8%EC%8B%AD&amp;action=edit&amp;redlink=1" title="카트 월드 챔피언십 (없는 문서)">카트 월드 챔피언십</a>(Karting World Championship)</li>
