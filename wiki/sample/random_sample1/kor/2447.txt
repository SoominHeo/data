<p><b>국제우주정거장</b>(國際宇宙停車場, <span style="font-size: smaller;"><a href="/wiki/%EC%98%81%EC%96%B4" title="영어">영어</a>:</span> <span lang="en" xml:lang="en">International Space Station, <b>ISS</b></span>)은 <a href="/wiki/%EB%9F%AC%EC%8B%9C%EC%95%84" title="러시아">러시아</a>와 <a href="/wiki/%EB%AF%B8%EA%B5%AD" title="미국">미국</a>을 비롯한 세계 각국이 참여하여 <a href="/wiki/1998%EB%85%84" title="1998년">1998년</a>에 건설이 시작된, 연구시설을 갖춘 다국적 <a href="/wiki/%EC%9A%B0%EC%A3%BC%EC%A0%95%EA%B1%B0%EC%9E%A5" title="우주정거장">우주정거장</a>이다. <a href="/wiki/2010%EB%85%84" title="2010년">2010년</a>까지 완성되어 최소한 <a href="/wiki/2016%EB%85%84" title="2016년">2016년</a>까지는 운영될 계획이다. <a href="/wiki/2008%EB%85%84" title="2008년">2008년</a> 기준으로 가장 큰 <a class="mw-redirect" href="/wiki/%EC%9A%B0%EC%A3%BC_%EC%A0%95%EA%B1%B0%EC%9E%A5" title="우주 정거장">우주 정거장</a>으로, 지상에서 육안으로 볼 수 있다.</p>
<p>ISS는 크게 러시아 섹션(ROS)과 미국 섹션(USOS)의 두 구역으로 나뉜다. ROS는 ISS 전체에 대한 유도, 항법, 통제, 메인 추진기관, 메인 생명유지장치를 담당한다. USOS는 가장 큰 실험실, 일본의 키보, 유럽의 콜럼버스, 2500 제곱미터 면적의 태양전지판, 추가적인 생명유지장치(산소발생기), 2번째 화장실을 담당한다. ISS는 <a href="/wiki/%EC%A7%80%EA%B5%AC_%EC%A0%80%EA%B6%A4%EB%8F%84" title="지구 저궤도">지구 저궤도</a>에 속하는 400km 고도에 떠 있으며, 시속 27,743.8 km의 속도로 매일 지구를 15.7 바퀴 돌고 있다(참고: 달은 약 384,400km, 태양은 약 147,000,000km 떨어져 있고 지구 대기권은 고도 1,000km까지이다).<sup class="reference" id="cite_ref-4"><a href="#cite_note-4">[4]</a></sup></p>
<p>2011년 4월 7일, 푸틴 러시아 총리는 3개의 모듈을 새로 추가해 ISS의 러시아 섹션이 2016년까지 완공될 것이며, ISS의 이용 기간을 2020년까지 연장하기로 파트너 국가들과 합의했다고 밝혔다.<sup class="reference" id="cite_ref-5"><a href="#cite_note-5">[5]</a></sup></p>

<h2><span class="mw-headline" id=".EC.97.AD.EC.82.AC">역사</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EA%B5%AD%EC%A0%9C%EC%9A%B0%EC%A3%BC%EC%A0%95%EA%B1%B0%EC%9E%A5&amp;action=edit&amp;section=1" title="부분 편집: 역사">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<div class="thumb tleft">
<div class="thumbinner" style="width:152px;"><a class="image" href="/wiki/%ED%8C%8C%EC%9D%BC:ISS_June_1999.jpg"><img alt="" class="thumbimage" data-file-height="3000" data-file-width="1997" height="225" src="//upload.wikimedia.org/wikipedia/commons/thumb/c/c0/ISS_June_1999.jpg/150px-ISS_June_1999.jpg" srcset="//upload.wikimedia.org/wikipedia/commons/thumb/c/c0/ISS_June_1999.jpg/225px-ISS_June_1999.jpg 1.5x, //upload.wikimedia.org/wikipedia/commons/thumb/c/c0/ISS_June_1999.jpg/300px-ISS_June_1999.jpg 2x" width="150"/></a>
<div class="thumbcaption">
<div class="magnify"><a class="internal" href="/wiki/%ED%8C%8C%EC%9D%BC:ISS_June_1999.jpg" title="실제 크기로"></a></div>
1999년 6월의 ISS. <a href="/wiki/%EC%9E%90%EB%9E%B4_%EB%AA%A8%EB%93%88" title="자랴 모듈">자랴</a>와 <a href="/wiki/%EC%9C%A0%EB%8B%88%ED%8B%B0_%EB%AA%A8%EB%93%88" title="유니티 모듈">유니티</a>(노드 원)의 2개 모듈뿐이다</div>
</div>
</div>
<p><a href="/wiki/1980%EB%85%84%EB%8C%80" title="1980년대">1980년대</a> 초반, <a href="/wiki/%EB%AF%B8%EA%B5%AD" title="미국">미국</a>의 <a class="mw-redirect" href="/wiki/NASA" title="NASA">NASA</a>는 <a href="/wiki/%EC%86%8C%EB%A0%A8" title="소련">소련</a>의 살류트와 <a href="/wiki/%EB%AF%B8%EB%A5%B4" title="미르">미르</a>에 대항하는 "<a class="mw-redirect" href="/wiki/%ED%94%84%EB%A6%AC%EB%8D%A4_%EC%9A%B0%EC%A3%BC_%EC%A0%95%EA%B1%B0%EC%9E%A5" title="프리덤 우주 정거장">프리덤 우주 정거장</a>"을 계획한다. 이 계획은 <a href="/wiki/1990%EB%85%84%EB%8C%80" title="1990년대">1990년대</a>까지 지속되다가, <a class="mw-redirect" href="/wiki/%EC%86%8C%EB%B9%84%EC%97%90%ED%8A%B8_%EC%97%B0%EB%B0%A9" title="소비에트 연방">소련</a>이 해체되면서 취소되었다.</p>
<p>대신, 계획 말기에 <a href="/wiki/%EB%AF%B8%EA%B5%AD" title="미국">미국</a> 정부는 <a class="mw-redirect" href="/wiki/%EC%9A%B0%EC%A3%BC_%EC%A0%95%EA%B1%B0%EC%9E%A5" title="우주 정거장">우주 정거장</a> 건설에 여러 나라를 파트너로 참여시키기 위해 협상을 시작했고 <a href="/wiki/1990%EB%85%84%EB%8C%80" title="1990년대">1990년대</a> 초반에 <a href="/wiki/%EB%AF%B8%EA%B5%AD" title="미국">미국</a>, <a href="/wiki/%EC%9C%A0%EB%9F%BD" title="유럽">유럽</a>, <a href="/wiki/%EB%9F%AC%EC%8B%9C%EC%95%84" title="러시아">러시아</a>, <a href="/wiki/%EC%9D%BC%EB%B3%B8" title="일본">일본</a>, <a href="/wiki/%EC%BA%90%EB%82%98%EB%8B%A4" title="캐나다">캐나다</a>의 국제 <a class="mw-redirect" href="/wiki/%EC%9A%B0%EC%A3%BC_%EC%A0%95%EA%B1%B0%EC%9E%A5" title="우주 정거장">우주 정거장</a> 계획이 수립되기에 이르렀다.</p>
<p>이 프로젝트는 <a href="/wiki/1993%EB%85%84" title="1993년">1993년</a>에 "알파 우주 정거장"으로 발표되었다가 취소되었다.</p>
<p>이 계획은 아래 모든 우주 기구의 <a class="mw-redirect" href="/wiki/%EC%9A%B0%EC%A3%BC_%EC%A0%95%EA%B1%B0%EC%9E%A5" title="우주 정거장">우주 정거장</a> 계획을 하나로 합친 것이다:</p>
<ul>
<li><a href="/wiki/%EB%AF%B8%EA%B5%AD_%ED%95%AD%EA%B3%B5%EC%9A%B0%EC%A3%BC%EA%B5%AD" title="미국 항공우주국">NASA</a>의 <a class="mw-redirect" href="/wiki/%ED%94%84%EB%A6%AC%EB%8D%A4_%EC%9A%B0%EC%A3%BC_%EC%A0%95%EA%B1%B0%EC%9E%A5" title="프리덤 우주 정거장">프리덤 우주 정거장</a></li>
<li><a href="/wiki/%EB%9F%AC%EC%8B%9C%EC%95%84" title="러시아">러시아</a>의 <a class="new" href="/w/index.php?title=%EB%AF%B8%EB%A5%B4_2&amp;action=edit&amp;redlink=1" title="미르 2 (없는 문서)">미르 2</a> 우주 정거장</li>
<li><a class="mw-redirect" href="/wiki/%EC%9A%B0%EC%A3%BC%ED%95%AD%EA%B3%B5%EC%97%B0%EA%B5%AC%EA%B0%9C%EB%B0%9C%EA%B8%B0%EA%B5%AC_(%EC%9D%BC%EB%B3%B8)" title="우주항공연구개발기구 (일본)">JAXA</a>의 <a class="mw-disambig" href="/wiki/%ED%82%A4%EB%B3%B4" title="키보">키보 연구실 모듈</a></li>
<li><a href="/wiki/%EC%9C%A0%EB%9F%BD_%EC%9A%B0%EC%A3%BC%EA%B5%AD" title="유럽 우주국">ESA</a>의 <a href="/wiki/%EC%BD%9C%EB%A1%AC%EB%B2%84%EC%8A%A4_%EB%AA%A8%EB%93%88" title="콜롬버스 모듈">콜럼버스 연구실 모듈</a>
<ul>
<li>원래는 <a href="/wiki/%EC%8A%A4%EC%B9%B4%EC%9D%B4%EB%9E%A9" title="스카이랩">스카이랩</a>처럼 독립형으로 발사될 계획이었다.</li>
</ul>
</li>
</ul>
<h2><span class="mw-headline" id=".EC.B0.B8.EC.97.AC.EA.B5.AD.EA.B0.80">참여국가</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EA%B5%AD%EC%A0%9C%EC%9A%B0%EC%A3%BC%EC%A0%95%EA%B1%B0%EC%9E%A5&amp;action=edit&amp;section=2" title="부분 편집: 참여국가">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>참여 국가는 <a href="/wiki/%EB%AF%B8%EA%B5%AD" title="미국">미국</a>, <a href="/wiki/%EB%9F%AC%EC%8B%9C%EC%95%84" title="러시아">러시아</a>, <a href="/wiki/%ED%94%84%EB%9E%91%EC%8A%A4" title="프랑스">프랑스</a>, <a href="/wiki/%EB%8F%85%EC%9D%BC" title="독일">독일</a>, <a href="/wiki/%EC%9D%BC%EB%B3%B8" title="일본">일본</a>, <a href="/wiki/%EC%9D%B4%ED%83%88%EB%A6%AC%EC%95%84" title="이탈리아">이탈리아</a>, <a href="/wiki/%EC%98%81%EA%B5%AD" title="영국">영국</a>, <a href="/wiki/%EB%B2%A8%EA%B8%B0%EC%97%90" title="벨기에">벨기에</a>, <a href="/wiki/%EB%8D%B4%EB%A7%88%ED%81%AC" title="덴마크">덴마크</a>, <a href="/wiki/%EC%8A%A4%EC%9B%A8%EB%8D%B4" title="스웨덴">스웨덴</a>, <a href="/wiki/%EC%8A%A4%ED%8E%98%EC%9D%B8" title="스페인">스페인</a>, <a href="/wiki/%EB%85%B8%EB%A5%B4%EC%9B%A8%EC%9D%B4" title="노르웨이">노르웨이</a>, <a href="/wiki/%EB%84%A4%EB%8D%9C%EB%9E%80%EB%93%9C" title="네덜란드">네덜란드</a>, <a href="/wiki/%EC%8A%A4%EC%9C%84%EC%8A%A4" title="스위스">스위스</a>, <a href="/wiki/%EC%BA%90%EB%82%98%EB%8B%A4" title="캐나다">캐나다</a>, <a href="/wiki/%EB%B8%8C%EB%9D%BC%EC%A7%88" title="브라질">브라질</a>의 16개국이다.</p>
<h3><span class="mw-headline" id=".EB.8C.80.ED.95.9C.EB.AF.BC.EA.B5.AD.EC.9D.98_.EC.B0.B8.EC.97.AC">대한민국의 참여</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EA%B5%AD%EC%A0%9C%EC%9A%B0%EC%A3%BC%EC%A0%95%EA%B1%B0%EC%9E%A5&amp;action=edit&amp;section=3" title="부분 편집: 대한민국의 참여">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>2000년 미국은 한국에 우주관측장비를 함께 만들자는 제안을 하면서, 한국에 사람을 보내 기술검증까지 했으나, 예산을 확보하지 못해 무산되었다.<sup class="reference" id="cite_ref-6"><a href="#cite_note-6">[6]</a></sup> 2001년 말 미국이 한국에 실험 모듈 건설을 제안해 온 적이 있는데, 비용이 2억 달러에 달해 참여를 포기했다.<sup class="reference" id="cite_ref-7"><a href="#cite_note-7">[7]</a></sup> 2002년 4월 한국은 우주입자검출기사업을 추진했으나, 무산되었다. 2003년 다시 미국 <a class="mw-redirect" href="/wiki/NASA" title="NASA">NASA</a>에 우주입자검출기사업 제안서를 제출했으나, 무산되었다.<sup class="reference" id="cite_ref-8"><a href="#cite_note-8">[8]</a></sup> 러시아도 한국에 ISS 모듈 제작 사업을 제안했으나, 역시 무산되었다.</p>
<h2><span class="mw-headline" id=".EA.B1.B4.EC.84.A4_.EA.B3.BC.EC.A0.95">건설 과정</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EA%B5%AD%EC%A0%9C%EC%9A%B0%EC%A3%BC%EC%A0%95%EA%B1%B0%EC%9E%A5&amp;action=edit&amp;section=4" title="부분 편집: 건설 과정">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p><a href="/wiki/2006%EB%85%84" title="2006년">2006년</a> 초까지, 당초 계획에 많은 변경이 있었다. 모듈 등 구조물들의 계획이 취소되거나 변경되었고, 우주왕복선의 비행계획도 대폭 축소되었다. 그러나 <a href="/wiki/1990%EB%85%84%EB%8C%80" title="1990년대">1990년대</a> 후반의 ISS 계획에 80% 이상의 하드웨어 완공률을 보이고 있으며, 완공일자도 대폭 연기되었지만, 현재 완공된 상태이다.</p>

<p>ISS 건설에는 40 회 이상의 <a class="mw-redirect" href="/wiki/%EC%9A%B0%EC%A3%BC_%EC%99%95%EB%B3%B5%EC%84%A0" title="우주 왕복선">미우주왕복선</a>의 비행이 요구되는데 이 중 33회의 <a href="/wiki/%EC%9A%B0%EC%A3%BC%EC%99%95%EB%B3%B5%EC%84%A0" title="우주왕복선">우주왕복선</a> 비행이 완료되었다. 다른 조립 비행은 러시아의 <a href="/wiki/%ED%94%84%EB%A1%9C%ED%86%A4_%EB%A1%9C%EC%BC%93" title="프로톤 로켓">프로톤 로켓</a>이나 <a class="mw-redirect" href="/wiki/%EC%86%8C%EC%9C%A0%EC%8A%A4_%EB%A1%9C%EC%BC%93" title="소유스 로켓">소유스 로켓</a>에 의해 이루어졌다. 또, 미국의 우주왕복선, 러시아의 <a class="new" href="/w/index.php?title=%ED%94%84%EB%A1%9C%EA%B7%B8%EB%A0%88%EC%8A%A4_%ED%99%94%EB%AC%BC%EC%84%A0&amp;action=edit&amp;redlink=1" title="프로그레스 화물선 (없는 문서)">프로그레스 화물선</a>, 유럽의 <a class="mw-disambig" href="/wiki/ATV" title="ATV">ATV</a>(Automated Transfer Vehicle), 일본의 <a class="mw-redirect" href="/wiki/HTV" title="HTV">HTV</a>(H-II Transfer Vehicle) 등이 실험기구, 연료, 식량 등을 ISS에 운반할 것이다.</p>
<p>ISS는 부피가 약 1,000 m³,무게가 약 400,000 kg, 전기 출력은 약 100 kw, 구조물 길이는 약 108.4 m, 모듈 길이는 74 m가 되었고, 6명의 승무원이 생활할 수 있다.</p>

<h2><span class="mw-headline" id=".EC.97.B0.EB.A3.8C_.EC.9E.AC.EB.B3.B4.EA.B8.89.EA.B3.BC_.EC.B6.94.EC.A7.84">연료 재보급과 추진</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EA%B5%AD%EC%A0%9C%EC%9A%B0%EC%A3%BC%EC%A0%95%EA%B1%B0%EC%9E%A5&amp;action=edit&amp;section=6" title="부분 편집: 연료 재보급과 추진">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<div class="rellink relarticle mainarticle"><img alt="" data-file-height="128" data-file-width="128" height="16" src="//upload.wikimedia.org/wikipedia/commons/thumb/e/ec/Crystal_Clear_app_xmag.svg/16px-Crystal_Clear_app_xmag.svg.png" srcset="//upload.wikimedia.org/wikipedia/commons/thumb/e/ec/Crystal_Clear_app_xmag.svg/24px-Crystal_Clear_app_xmag.svg.png 1.5x, //upload.wikimedia.org/wikipedia/commons/thumb/e/ec/Crystal_Clear_app_xmag.svg/32px-Crystal_Clear_app_xmag.svg.png 2x" width="16"/> 이 부분의 본문은 <a href="/wiki/ISS_%EC%B6%94%EC%A7%84_%EB%AA%A8%EB%93%88" title="ISS 추진 모듈">ISS 추진 모듈</a>입니다.</div>
<p><a href="/wiki/2007%EB%85%84" title="2007년">2007년</a> 기준으로 유도, 항법, 조종, 추진을 담당하는 핵심적인 ISS 모듈은 <a href="/wiki/%EB%9F%AC%EC%8B%9C%EC%95%84" title="러시아">러시아</a>의 <a class="mw-disambig" href="/wiki/%EC%A6%88%EB%B2%A0%EC%A6%88%EB%8B%A4" title="즈베즈다">즈베즈다</a> 서비스 모듈과 <a class="mw-redirect" href="/wiki/%ED%94%84%EB%A1%9C%EA%B7%B8%EB%A0%88%EC%8A%A4_%EC%9A%B0%EC%A3%BC%EC%84%A0" title="프로그레스 우주선">프로그레스 우주선</a>이다. 미래에는 <a href="/wiki/%EC%9C%A0%EB%9F%BD_%EC%9A%B0%EC%A3%BC%EA%B5%AD" title="유럽 우주국">ESA</a>의 <a href="/wiki/ATV_(%EC%9A%B0%EC%A3%BC%EC%84%A0)" title="ATV (우주선)">ATV</a>도 이 역할을 함께 할 것이다.</p>
<p>ISS는 고도 유지, 우주 쓰레기 회피, 고도 조정을 위해 매년 평균 7000 kg의 추진제(propellant)를 필요로 한다. <a href="/wiki/2014%EB%85%84" title="2014년">2014년</a>경에는 105,000 kg의 추진체가 매년 필요할 것이다. 7000 kg의 추진제를 보급하기 위해서는 여러 번의 화물 공급 우주선의 발사가 필요하다. 현재는 <a class="mw-redirect" href="/wiki/%ED%94%84%EB%A1%9C%EA%B7%B8%EB%A0%88%EC%8A%A4_%EC%9A%B0%EC%A3%BC%EC%84%A0" title="프로그레스 우주선">프로그레스 우주선</a>을 매년 6회 발사하여 이를 충당한다.</p>
<p><a class="new" href="/w/index.php?title=%ED%94%84%EB%A1%9C%EA%B7%B8%EB%A0%88%EC%8A%A4_M_%ED%99%94%EB%AC%BC%EC%9A%B0%EC%A3%BC%EC%84%A0&amp;action=edit&amp;redlink=1" title="프로그레스 M 화물우주선 (없는 문서)">프로그레스 M 화물우주선</a>은 1100 kg, <a class="new" href="/w/index.php?title=%ED%94%84%EB%A1%9C%EA%B7%B8%EB%A0%88%EC%8A%A4_M1_%ED%99%94%EB%AC%BC%EC%9A%B0%EC%A3%BC%EC%84%A0&amp;action=edit&amp;redlink=1" title="프로그레스 M1 화물우주선 (없는 문서)">프로그레스 M1 화물우주선</a> 은 1950 kg, <a href="/wiki/%EC%9C%A0%EB%9F%BD_%EC%9A%B0%EC%A3%BC%EA%B5%AD" title="유럽 우주국">유럽 우주국</a>의 <a href="/wiki/ATV_(%EC%9A%B0%EC%A3%BC%EC%84%A0)" title="ATV (우주선)">ATV</a>는 4000 kg, 미국의 <a class="new" href="/w/index.php?title=Interim_Control_Module&amp;action=edit&amp;redlink=1" title="Interim Control Module (없는 문서)">Interim Control Module</a>은 5000 kg의 연료(fuel)를 탑재한다.</p>
<p><a href="/wiki/%EB%AF%B8%EA%B5%AD" title="미국">미국</a>의 <a href="/wiki/%EC%9A%B0%EC%A3%BC%EC%99%95%EB%B3%B5%EC%84%A0" title="우주왕복선">우주왕복선</a>은 232 kg의 연료를 ISS의 추진을 위해 사용할 수 있고, 특별히 ISS 추진을 위한 임무를 수행하는 경우 1626 kg의 reboost fuel이 가능하다.</p>
<p><a href="/wiki/%EC%9E%90%EB%9E%B4_%EB%AA%A8%EB%93%88" title="자랴 모듈">자랴 모듈</a>의 FGB는 5500 kg, <a href="/wiki/%EC%A6%88%EB%B2%A0%EC%A6%88%EB%8B%A4_%EB%AA%A8%EB%93%88" title="즈베즈다 모듈">즈베즈다 모듈</a>은 860 kg의 연료를 탑재한다. 그러나 이 두 모듈의 추진기는 예비용으로, ISS의 수명이 다하는 경우 마지막으로 사용하게 된다.</p>
<h2><span class="mw-headline" id=".EA.B5.90.EC.8B.A0">교신</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EA%B5%AD%EC%A0%9C%EC%9A%B0%EC%A3%BC%EC%A0%95%EA%B1%B0%EC%9E%A5&amp;action=edit&amp;section=7" title="부분 편집: 교신">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<div class="thumb tright">
<div class="thumbinner" style="width:222px;"><a class="image" href="/wiki/%ED%8C%8C%EC%9D%BC:Yaesu-vx-7r.jpg"><img alt="" class="thumbimage" data-file-height="904" data-file-width="1207" height="165" src="//upload.wikimedia.org/wikipedia/commons/thumb/b/b1/Yaesu-vx-7r.jpg/220px-Yaesu-vx-7r.jpg" srcset="//upload.wikimedia.org/wikipedia/commons/thumb/b/b1/Yaesu-vx-7r.jpg/330px-Yaesu-vx-7r.jpg 1.5x, //upload.wikimedia.org/wikipedia/commons/thumb/b/b1/Yaesu-vx-7r.jpg/440px-Yaesu-vx-7r.jpg 2x" width="220"/></a>
<div class="thumbcaption">
<div class="magnify"><a class="internal" href="/wiki/%ED%8C%8C%EC%9D%BC:Yaesu-vx-7r.jpg" title="실제 크기로"></a></div>
일본 <a class="new" href="/w/index.php?title=%EC%95%BC%EC%97%90%EC%8A%A4&amp;action=edit&amp;redlink=1" title="야에스 (없는 문서)">야에스</a>사의 VX-7R 무전기. 이런 햄용 핸디무전기로 전 세계인은 국제우주정거장과 교신한다</div>
</div>
</div>
<p>ISS는 전 세계인과 햄(<a href="/wiki/%EC%95%84%EB%A7%88%EC%B6%94%EC%96%B4_%EB%AC%B4%EC%84%A0" title="아마추어 무선">아마추어 무선</a>)으로 교신한다. 햄들은 보통 송신출력 5W의 핸디무전기로 ISS와 교신한다. 평일 0900UTC, 1200UTC, 1900UTC, 주말 토요일 1200UTC 부터 일요일 1900UTC의 네 시간대를 통해 우주인들과 교신할 수 있다. 145.800 Mhz에 수신주파수를 맞추고, ITU Region 2(미국)와 Region 3(한국)에서는 144.490 Mhz로 음성 송신을 하면 된다.<sup class="reference" id="cite_ref-12"><a href="#cite_note-12">[12]</a></sup> 우주인들이 무선교신을 하는 네 시간대 중에서도, ISS가 근처를 지나가는 시간대여야만 교신이 가능하며, 이것은 인터넷에 궤도 계산 사이트를 통해 알 수 있다.<sup class="reference" id="cite_ref-13"><a href="#cite_note-13">[13]</a></sup></p>
<p>2008년, 한국인 최초의 우주인 <a href="/wiki/%EC%9D%B4%EC%86%8C%EC%97%B0_(%EA%B3%B5%ED%95%99%EC%9E%90)" title="이소연 (공학자)">이소연</a>은 한국인들과 햄교신 행사를 가졌다.<sup class="reference" id="cite_ref-14"><a href="#cite_note-14">[14]</a></sup></p>
<h2><span class="mw-headline" id=".EC.B7.A8.EC.86.8C.EB.90.9C_.EA.B3.84.ED.9A.8D">취소된 계획</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EA%B5%AD%EC%A0%9C%EC%9A%B0%EC%A3%BC%EC%A0%95%EA%B1%B0%EC%9E%A5&amp;action=edit&amp;section=8" title="부분 편집: 취소된 계획">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<ul>
<li><a class="new" href="/w/index.php?title=%EB%B6%84%EB%A6%AC%EA%B8%B0_%EC%84%A4%EB%B9%84_%EB%AA%A8%EB%93%88&amp;action=edit&amp;redlink=1" title="분리기 설비 모듈 (없는 문서)">분리기 설비 모듈</a>(Centrifuge Accommodations Module) - Node 2에 부착될 예정이었다.</li>
<li><a class="new" href="/w/index.php?title=%EB%9F%AC%EC%8B%9C%EC%95%84_%EC%97%B0%EA%B5%AC_%EB%AA%A8%EB%93%88&amp;action=edit&amp;redlink=1" title="러시아 연구 모듈 (없는 문서)">러시아 연구 모듈</a>(Russian Research Module) - 러시아의 2번째 연구실 모듈이었는데, 예산 문제로 범용 도킹 모듈과 함께 취소되었다. 러시아는 따라서 1개의 연구실 모듈과 1개의 다목적 연구실 모듈만 운영한다.</li>
<li><a class="new" href="/w/index.php?title=%EB%B2%94%EC%9A%A9_%EB%8F%84%ED%82%B9_%EB%AA%A8%EB%93%88&amp;action=edit&amp;redlink=1" title="범용 도킹 모듈 (없는 문서)">범용 도킹 모듈</a>(Universal Docking Module) - 다목적 연구실 모듈(Multipurpose Laboratory Module)로 대체되었다.</li>
<li><a class="new" href="/w/index.php?title=%EB%8F%84%ED%82%B9_%EB%B0%8F_%EC%A0%81%EC%9E%AC_%EB%AA%A8%EB%93%88&amp;action=edit&amp;redlink=1" title="도킹 및 적재 모듈 (없는 문서)">도킹 및 적재 모듈</a>(Docking and Stowage Module) - 다목적 연구실 모듈(Multipurpose Laboratory Module)로 대체되었다.</li>
<li><a class="new" href="/w/index.php?title=%EA%B1%B0%EC%A3%BC_%EB%AA%A8%EB%93%88&amp;action=edit&amp;redlink=1" title="거주 모듈 (없는 문서)">거주 모듈</a>(Habitation Module)</li>
<li><a class="new" href="/w/index.php?title=%EB%8C%80%EC%9B%90_%EA%B7%80%ED%99%98%EC%84%A0_(CRV)&amp;action=edit&amp;redlink=1" title="대원 귀환선 (CRV) (없는 문서)">대원 귀환선 (CRV)</a>(Crew Return Vehicle (CRV))</li>
<li><a class="new" href="/w/index.php?title=%EC%A4%91%EA%B0%84_%EC%A1%B0%EC%A2%85_%EB%AA%A8%EB%93%88&amp;action=edit&amp;redlink=1" title="중간 조종 모듈 (없는 문서)">중간 조종 모듈</a>(Interim Control Module) - <a class="mw-disambig" href="/wiki/%EC%A6%88%EB%B2%A0%EC%A6%88%EB%8B%A4" title="즈베즈다">즈베즈다</a>를 교체할 필요가 없어졌다. (필요시 신속한 발사를 위해 보관될 것이다.)</li>
<li><a href="/wiki/ISS_%EC%B6%94%EC%A7%84_%EB%AA%A8%EB%93%88" title="ISS 추진 모듈">ISS 추진 모듈</a>(ISS Propulsion Module) - 즈베즈다를 교체할 필요가 없어졌다.</li>
<li><a class="new" href="/w/index.php?title=%EC%82%AC%EC%9D%B4%EC%96%B8%EC%8A%A4_%ED%8C%8C%EC%9B%8C_%ED%94%8C%EB%9E%AB%ED%8F%BC&amp;action=edit&amp;redlink=1" title="사이언스 파워 플랫폼 (없는 문서)">사이언스 파워 플랫폼</a>(Science Power Platform) - <a href="/wiki/%EB%9F%AC%EC%8B%9C%EC%95%84" title="러시아">러시아</a> 설비로부터 전기가 공급될 것이다. <a href="/wiki/%EB%AF%B8%EA%B5%AD" title="미국">미국</a> 태양전지판에서도 일부 공급될 것이다.</li>
</ul>
<h2><span class="mw-headline" id=".EB.B0.A9.EB.AC.B8.ED.95.98.EB.8A.94_.EC.9A.B0.EC.A3.BC.EC.84.A0">방문하는 우주선</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EA%B5%AD%EC%A0%9C%EC%9A%B0%EC%A3%BC%EC%A0%95%EA%B1%B0%EC%9E%A5&amp;action=edit&amp;section=9" title="부분 편집: 방문하는 우주선">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<ul>
<li><a class="mw-redirect" href="/wiki/%EC%9A%B0%EC%A3%BC_%EC%99%95%EB%B3%B5%EC%84%A0" title="우주 왕복선">우주 왕복선</a> - 보급, 조립, 승무원 교체 등을 위해 비행한다.</li>
<li><a class="mw-redirect" href="/wiki/%EC%86%8C%EC%9C%A0%EC%A6%88_%EC%9A%B0%EC%A3%BC%EC%84%A0" title="소유즈 우주선">소유즈 우주선</a> - 승무원 교체와 비상탈출에 사용된다. 6개월마다 교체된다.</li>
<li><a class="new" href="/w/index.php?title=%ED%94%84%EB%A1%9C%EA%B7%B8%EB%A0%88%EC%8A%A4_%ED%99%94%EB%AC%BC%EC%9A%B0%EC%A3%BC%EC%84%A0&amp;action=edit&amp;redlink=1" title="프로그레스 화물우주선 (없는 문서)">프로그레스 화물우주선</a> - 무인우주선이다. 보급에 사용된다. 2007년 기준으로, 매년 6회 발사되어 연료를 공급한다.</li>
<li><a href="/wiki/%EC%9C%A0%EB%9F%BD_%EC%9A%B0%EC%A3%BC%EA%B5%AD" title="유럽 우주국">유럽 우주국</a>의 <a class="mw-disambig" href="/wiki/ATV" title="ATV">ATV</a>(Automated Transfer Vehicle). 보급선이다. <a class="mw-disambig" href="/wiki/ATV" title="ATV">ATV</a>는 개발이 완료되었으며, <a href="/wiki/2008%EB%85%84" title="2008년">2008년</a> <a href="/wiki/3%EC%9B%94_9%EC%9D%BC" title="3월 9일">3월 9일</a> <a href="/wiki/%EC%95%84%EB%A6%AC%EC%95%88_5" title="아리안 5">아리안 5</a> 로켓에 의해 최초로 발사되었다.</li>

<li>예정: <a class="new" href="/w/index.php?title=%ED%81%B4%EB%A6%AC%ED%8D%BC_(%EC%9A%B0%EC%A3%BC%EC%84%A0)&amp;action=edit&amp;redlink=1" title="클리퍼 (우주선) (없는 문서)">클리퍼</a> 러시아 우주 왕복선. 승무원 교체와 보급 수송에 쓰일 예정이다. <a href="/wiki/2012%EB%85%84" title="2012년">2012년</a>에 계획되어 있다.</li>
<li>예정: <a href="/wiki/%EC%98%A4%EB%A6%AC%EC%98%A8_(%EC%9A%B0%EC%A3%BC%EC%84%A0)" title="오리온 (우주선)">CEV</a>(Crew Exploration Vehicle). 승무원 교체와 보급 수송에 쓰일 예정이다. <a href="/wiki/2014%EB%85%84" title="2014년">2014년</a>에 계획되어 있다.</li>
