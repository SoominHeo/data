<p><b>군다 니만슈티르네만</b>(<span style="font-size: smaller;"><a href="/wiki/%EB%8F%85%EC%9D%BC%EC%96%B4" title="독일어">독일어</a>:</span> <span lang="de" xml:lang="de">Gunda Niemann-Stirnemann</span>, <a href="/wiki/1966%EB%85%84" title="1966년">1966년</a> <a href="/wiki/9%EC%9B%94_7%EC%9D%BC" title="9월 7일">9월 7일</a> ~ )은 <a href="/wiki/%EB%8F%85%EC%9D%BC" title="독일">독일</a>의 여자 <a href="/wiki/%EC%8A%A4%ED%94%BC%EB%93%9C_%EC%8A%A4%EC%BC%80%EC%9D%B4%ED%8C%85" title="스피드 스케이팅">스피드 스케이팅</a> 선수이다.</p>
<p><a href="/wiki/%EB%8F%99%EB%8F%85" title="동독">동독</a> 출신으로, <a href="/wiki/1988%EB%85%84_%EB%8F%99%EA%B3%84_%EC%98%AC%EB%A6%BC%ED%94%BD" title="1988년 동계 올림픽">1988년 동계 올림픽</a>에 처음 참가한 이후 세계적인 여자 장거리 선수 중 하나로 이름을 알렸다. 90년대의 세 차례 <a href="/wiki/%EB%8F%99%EA%B3%84_%EC%98%AC%EB%A6%BC%ED%94%BD" title="동계 올림픽">동계 올림픽</a>(<a href="/wiki/1992%EB%85%84_%EB%8F%99%EA%B3%84_%EC%98%AC%EB%A6%BC%ED%94%BD" title="1992년 동계 올림픽">1992년 동계 올림픽</a>·<a href="/wiki/1994%EB%85%84_%EB%8F%99%EA%B3%84_%EC%98%AC%EB%A6%BC%ED%94%BD" title="1994년 동계 올림픽">1994년 동계 올림픽</a>·<a href="/wiki/1998%EB%85%84_%EB%8F%99%EA%B3%84_%EC%98%AC%EB%A6%BC%ED%94%BD" title="1998년 동계 올림픽">1998년 동계 올림픽</a>)에서 금메달 3개, 은메달 4개, 동메달 1개를 획득했다. 세계 종합 선수권에서 8차례 우승하여 남녀를 통틀어 이 대회 우승 기록이 가장 많다.</p>
<p><a href="/wiki/2001%EB%85%84" title="2001년">2001년</a> 출산으로 빙판을 떠나 <a href="/wiki/2002%EB%85%84_%EB%8F%99%EA%B3%84_%EC%98%AC%EB%A6%BC%ED%94%BD" title="2002년 동계 올림픽">2002년 동계 올림픽</a>에 참가하지 않았으나, <a href="/wiki/2003%EB%85%84" title="2003년">2003년</a> 하반기에 복귀하기로 하여<sup class="reference" id="cite_ref-1"><a href="#cite_note-1">[1]</a></sup> 30대 후반의 많은 나이에도 다시 독일 대표로 선발되어 <a href="/wiki/2004%EB%85%84" title="2004년">2004년</a> <a href="/wiki/%EB%8C%80%ED%95%9C%EB%AF%BC%EA%B5%AD" title="대한민국">대한민국</a> <a class="mw-redirect" href="/wiki/%EC%84%9C%EC%9A%B8" title="서울">서울</a> <a href="/wiki/%ED%83%9C%EB%A6%89_%EA%B5%AD%EC%A0%9C_%EC%8A%A4%EC%BC%80%EC%9D%B4%ED%8A%B8%EC%9E%A5" title="태릉 국제 스케이트장">태릉 국제 스케이트장</a>에서 열린 <a class="mw-redirect" href="/wiki/%EC%84%B8%EA%B3%84_%EC%8A%A4%ED%94%BC%EB%93%9C_%EC%8A%A4%EC%BC%80%EC%9D%B4%ED%8C%85_%EC%A2%85%EB%AA%A9%EB%B3%84_%EC%84%A0%EC%88%98%EA%B6%8C_%EB%8C%80%ED%9A%8C" title="세계 스피드 스케이팅 종목별 선수권 대회">세계 종목별 선수권 대회</a>에 참가하여 3000m와 5000에서 각각 5위, 4위를 한 후 은퇴했다.</p>
<p>결혼 전 성은 <b>클레만</b>(Kleemann)으로 처음에는 이 성으로 경기에 출전하다, 데틀레프 니만과 결혼한 후 군다 니만으로 이름을 바꿔 활동했고, <a href="/wiki/1997%EB%85%84" title="1997년">1997년</a> 올리버 슈티르네만과 재혼한 후로는 군다 니만슈티르네만으로 이름을 변경하여 활동하였다. <a href="/wiki/%EC%97%90%EB%A5%B4%ED%91%B8%EB%A5%B4%ED%8A%B8" title="에르푸르트">에르푸르트</a>의 빙상장은 스피드 스케이팅에서 대단한 활약을 펼친 그의 업적을 기념하여 군다니만슈티르네만 할레(Gunda-Niemann-Stirnemann-Halle)로 이름을 바꾸었다.</p>
