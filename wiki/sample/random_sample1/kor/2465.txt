<p><b>굴부딘 헤크마티아르</b>(<span style="font-size: smaller;"><a href="/wiki/%ED%8C%8C%EC%8A%88%ED%86%A0%EC%96%B4" title="파슈토어">파슈토어</a>:</span> <span lang="ps" xml:lang="ps">گلبدین حکمتیار</span>, Gulbuddin Hekmatyar, <a href="/wiki/1947%EB%85%84" title="1947년">1947년</a>(다른 이설에 따르면, <a href="/wiki/1944%EB%85%84" title="1944년">1944년</a><sup class="reference" id="cite_ref-1"><a href="#cite_note-1">[1]</a></sup> 또는 <a href="/wiki/1948%EB%85%84" title="1948년">1948년</a><sup class="reference" id="cite_ref-2"><a href="#cite_note-2">[2]</a></sup>) ~ )는 <a href="/wiki/%EC%95%84%ED%94%84%EA%B0%80%EB%8B%88%EC%8A%A4%ED%83%84" title="아프가니스탄">아프가니스탄</a>의 정치인이자 <a href="/wiki/%EB%AC%B4%EC%9E%90%ED%97%A4%EB%94%98" title="무자헤딘">무자헤딘</a> 지도자이다. <a href="/wiki/%EC%95%84%ED%94%84%EA%B0%80%EB%8B%88%EC%8A%A4%ED%83%84_%EC%9D%B4%EC%8A%AC%EB%9E%8C%EB%8B%B9" title="아프가니스탄 이슬람당">아프가니스탄 이슬람당</a>의 지도자이며, <a href="/wiki/%EB%B6%80%EB%A5%B4%ED%95%98%EB%88%84%EB%94%98_%EB%9D%BC%EB%B0%94%EB%8B%88" title="부르하누딘 라바니">부르하누딘 라바니</a> 정부에서 <a class="new" href="/w/index.php?title=%EC%95%84%ED%94%84%EA%B0%80%EB%8B%88%EC%8A%A4%ED%83%84%EC%9D%98_%EC%B4%9D%EB%A6%AC&amp;action=edit&amp;redlink=1" title="아프가니스탄의 총리 (없는 문서)">총리</a>를 지냈다.</p>
<p><a class="mw-redirect" href="/wiki/%EC%86%8C%EB%A0%A8%EC%9D%98_%EC%95%84%ED%94%84%EA%B0%80%EB%8B%88%EC%8A%A4%ED%83%84_%EC%B9%A8%EA%B3%B5" title="소련의 아프가니스탄 침공">소련의 아프가니스탄 침공</a> 때 가장 강력했던 군벌 중 하나로 불리며<sup class="reference" id="cite_ref-3"><a href="#cite_note-3">[3]</a></sup>, <a class="new" href="/w/index.php?title=1996%EB%85%84_%EC%95%84%ED%94%84%EA%B0%80%EB%8B%88%EC%8A%A4%ED%83%84_%EB%82%B4%EC%A0%84&amp;action=edit&amp;redlink=1" title="1996년 아프가니스탄 내전 (없는 문서)">1996년 아프가니스탄 내전</a> 당시 <a href="/wiki/%EB%AF%B8%EA%B5%AD" title="미국">미국</a>으로부터 <a class="new" href="/w/index.php?title=%EC%8A%A4%ED%8C%85%EC%96%B4_%EB%AF%B8%EC%82%AC%EC%9D%BC&amp;action=edit&amp;redlink=1" title="스팅어 미사일 (없는 문서)">스팅어 미사일</a>을 대거 지원 받아 지금도 이를 보유하고 있는 것으로 알려졌다.<sup class="reference" id="cite_ref-4"><a href="#cite_note-4">[4]</a></sup></p>
<h2><span class="mw-headline" id=".EC.83.9D.EC.95.A0">생애</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EA%B5%B4%EB%B6%80%EB%94%98_%ED%97%A4%ED%81%AC%EB%A7%88%ED%8B%B0%EC%95%84%EB%A5%B4&amp;action=edit&amp;section=1" title="부분 편집: 생애">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p><a href="/wiki/%EB%AC%B4%ED%95%A8%EB%A7%88%EB%93%9C_%EB%8B%A4%EC%9A%B0%EB%93%9C" title="무함마드 다우드">무함마드 다우드</a> 총리 시절 탄압을 피해 파키스탄으로 망명한 뒤 이슬람주의 운동을 주도했고 <a class="mw-redirect" href="/wiki/%EC%86%8C%EB%B9%84%EC%97%90%ED%8A%B8_%EC%97%B0%EB%B0%A9%EC%9D%98_%EC%95%84%ED%94%84%EA%B0%80%EB%8B%88%EC%8A%A4%ED%83%84_%EC%B9%A8%EA%B3%B5" title="소비에트 연방의 아프가니스탄 침공">소비에트 연방의 아프가니스탄 침공</a> 이후에 아프가니스탄 이슬람당을 조직해 무자헤딘 지휘관으로 활동했다. 전쟁 와중에 <a href="/wiki/%EB%B6%80%EB%A5%B4%ED%95%98%EB%88%84%EB%94%98_%EB%9D%BC%EB%B0%94%EB%8B%88" title="부르하누딘 라바니">부르하누딘 라바니</a>와 <a href="/wiki/%EC%95%84%ED%9D%90%EB%A7%88%EB%93%9C_%EC%83%A4_%EB%A7%88%EC%88%98%EB%93%9C" title="아흐마드 샤 마수드">아흐마드 샤 마수드</a>가 이끄는 자미아트 이 이슬라미와 연합했다.</p>
<p><a href="/wiki/1989%EB%85%84" title="1989년">1989년</a> 3월 <a href="/wiki/%ED%8C%8D%ED%8B%B0%EC%95%BC_%EC%A3%BC" title="팍티야 주">팍티야 주</a> <a class="new" href="/w/index.php?title=%EC%89%AC%EC%99%80&amp;action=edit&amp;redlink=1" title="쉬와 (없는 문서)">쉬와</a>에서 소집된 임시 정부 회의에서 외무부 장관이 되었고 <a href="/wiki/7%EC%9B%94_9%EC%9D%BC" title="7월 9일">7월 9일</a> 부하 <a class="new" href="/w/index.php?title=%EC%82%AC%EC%98%88%EB%93%9C_%EC%9E%90%EB%A7%90&amp;action=edit&amp;redlink=1" title="사예드 자말 (없는 문서)">사예드 자말</a>을 시켜 마수드를 기습 공격하나 패하고 <a href="/wiki/1990%EB%85%84" title="1990년">1990년</a> 7월 <a href="/wiki/%EB%AA%A8%ED%95%98%EB%A7%88%EB%93%9C_%EB%82%98%EC%A7%80%EB%B6%88%EB%9D%BC" title="모하마드 나지불라">모하마드 나지불라</a> 정부의 국방부 장관을 사주해 쿠데타를 기도했으나 실패했다.</p>
<p><a href="/wiki/1992%EB%85%84" title="1992년">1992년</a> 4월 나지불라 정권이 붕괴되고 무자헤딘 연합 정권인 <a class="new" href="/w/index.php?title=%EC%95%84%ED%94%84%EA%B0%80%EB%8B%88%EC%8A%A4%ED%83%84_%EC%9D%B4%EC%8A%AC%EB%9E%8C_%EC%97%B0%EB%B0%A9_%EA%B3%B5%ED%99%94%EA%B5%AD&amp;action=edit&amp;redlink=1" title="아프가니스탄 이슬람 연방 공화국 (없는 문서)">아프가니스탄 이슬람 연방 공화국</a>이 수립되면서 국무총리직이 주어졌으나 헤크마티아르는 모든 수단을 동원해 수도 <a href="/wiki/%EC%B9%B4%EB%B6%88" title="카불">카불</a>을 무력 점령하려 했다. <a href="/wiki/4%EC%9B%94_25%EC%9D%BC" title="4월 25일">4월 25일</a> <a class="new" href="/w/index.php?title=%EC%95%84%ED%94%84%EA%B0%80%EB%8B%88%EC%8A%A4%ED%83%84_%EB%82%B4%EC%A0%84_(1992%EB%85%84_~_1996%EB%85%84)&amp;action=edit&amp;redlink=1" title="아프가니스탄 내전 (1992년 ~ 1996년) (없는 문서)">제2차 아프가니스탄 내전</a>이 발발해 수도 카불을 일시적으로 점령했으나 마수드와 <a href="/wiki/%EC%95%95%EB%91%98_%EB%9D%BC%EC%8B%9C%EB%93%9C_%EB%8F%84%EC%8A%A4%ED%88%BC" title="압둘 라시드 도스툼">압둘 라시드 도스툼</a> 연합군에게 패배했으며 이 과정에서 카불을 무차별 공격해 폐허로 만들고 무고한 시민 1만 명을 죽였다.</p>
<p><a href="/wiki/1994%EB%85%84" title="1994년">1994년</a> 1월 4개 정당을 연합하면서 원수 지간이던 도스툼과도 협력했으며 <a href="/wiki/%ED%95%98%EC%9E%90%EB%9D%BC%EC%A1%B1" title="하자라족">하자라족</a> 무장 집단인 헤즈비 와흐다트와도 동맹을 맺었다. <a href="/wiki/7%EC%9B%94_20%EC%9D%BC" title="7월 20일">7월 20일</a> <a href="/wiki/%ED%97%A4%EB%9D%BC%ED%8A%B8" title="헤라트">헤라트</a>에서 마수드와 격전을 벌이나 대패했고 <a href="/wiki/1995%EB%85%84" title="1995년">1995년</a> <a href="/wiki/3%EC%9B%94_6%EC%9D%BC" title="3월 6일">3월 6일</a> <a href="/wiki/%ED%83%88%EB%A0%88%EB%B0%98" title="탈레반">탈레반</a>의 진군으로 혼란한 틈을 타 마수드가 다시 공격을 해 오면서 큰 피해를 입었다.</p>
<p>이후 카불 장악을 위해 몇 번이고 카불을 폭격해대던 헤크마티아르는 <a href="/wiki/1996%EB%85%84" title="1996년">1996년</a> <a href="/wiki/6%EC%9B%94_25%EC%9D%BC" title="6월 25일">6월 25일</a> 국무총리직을 수락하고 카불로 들어갔으며 탈레반이 대규모 공세를 감행하자 <a href="/wiki/1997%EB%85%84" title="1997년">1997년</a> 7월경에는 카불에서 20km 떨어진 샤르 아시아브에 은거하다가 탈레반에 투항해 협력했다.</p>
<p><a href="/wiki/2001%EB%85%84" title="2001년">2001년</a> 미군이 침공해 탈레반 정권이 붕괴되자 무장 조직인 아프가니스탄 이슬람당을 조직해 미군과 나토군에 맞서 항전을 벌였고 이에 <a href="/wiki/%EB%AF%B8%EA%B5%AD" title="미국">미국</a> <a class="mw-redirect" href="/wiki/CIA" title="CIA">CIA</a>가 <a href="/wiki/2002%EB%85%84" title="2002년">2002년</a> 4월 헤크마티아르 암살을 계획했으나 실패로 돌아갔다.</p>
<p><a href="/wiki/2015%EB%85%84" title="2015년">2015년</a> 7월 <a href="/wiki/%EC%9D%B4%EC%8A%AC%EB%9E%8C_%EA%B5%AD%EA%B0%80_(%EB%8B%A8%EC%B2%B4)" title="이슬람 국가 (단체)">이슬람 국가</a>에 대한 지지를 선언했다.<sup class="reference" id="cite_ref-5"><a href="#cite_note-5">[5]</a></sup></p>
