<p><b>굿샤로 호</b>(<span style="font-size: smaller;"><a href="/wiki/%EC%9D%BC%EB%B3%B8%EC%96%B4" title="일본어">일본어</a>:</span> <span lang="ja"><span lang="ja"><ruby><rb>屈斜路湖</rb> <rp>(</rp><rt>くっしゃろこ</rt><rp>)</rp></ruby></span></span> <small>굿샤로코<sup>[<a href="/wiki/%EC%9C%84%ED%82%A4%EB%B0%B1%EA%B3%BC:%EC%9D%BC%EB%B3%B8%EC%96%B4%EC%9D%98_%ED%95%9C%EA%B8%80_%ED%91%9C%EA%B8%B0" title="위키백과:일본어의 한글 표기">*</a>]</sup></small>)는 <a href="/wiki/%ED%99%8B%EC%B9%B4%EC%9D%B4%EB%8F%84" title="홋카이도">홋카이도</a> 동부의 <a href="/wiki/%EB%8D%B0%EC%8B%9C%EC%B9%B4%EA%B0%80_%EC%A0%95" title="데시카가 정">데시카가 정</a>에 있는 <a href="/wiki/%ED%98%B8%EC%88%98" title="호수">호수</a>이다. 호수 전역이 <a href="/wiki/%EC%95%84%EC%B9%B8_%EA%B5%AD%EB%A6%BD%EA%B3%B5%EC%9B%90" title="아칸 국립공원">아칸 국립공원</a>에 속해있다. 미확인 생명체인 <a class="new" href="/w/index.php?title=%EA%B5%BF%EC%8B%9C&amp;action=edit&amp;redlink=1" title="굿시 (없는 문서)">굿시</a>가 목격된 호수로 잠시나마 화제가 된 적이 있다.</p>
<h2><span class="mw-headline" id=".EC.A7.80.EB.A6.AC">지리</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EA%B5%BF%EC%83%A4%EB%A1%9C_%ED%98%B8&amp;action=edit&amp;section=1" title="부분 편집: 지리">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p><a class="new" href="/w/index.php?title=%EB%AA%A8%EC%BD%94%ED%86%A0_%EC%82%B0&amp;action=edit&amp;redlink=1" title="모코토 산 (없는 문서)">모코토 산</a>, 사맛카리누푸리 등을 <a class="new" href="/w/index.php?title=%EC%99%B8%EB%A5%9C%EC%82%B0&amp;action=edit&amp;redlink=1" title="외륜산 (없는 문서)">외륜산</a>으로 해 굿샤로 <a href="/wiki/%EC%B9%BC%EB%8D%B0%EB%9D%BC" title="칼데라">칼데라</a> 내에 생성된 일본 최대의 <a class="mw-redirect" href="/wiki/%EC%B9%BC%EB%8D%B0%EB%9D%BC_%ED%98%B8" title="칼데라 호">칼데라 호</a>이다. 덧붙여 굿샤로 칼레라의 직경은 긴 곳이 26km, 짧은 곳이 20km에 이르는 일본 최대의 칼레라이다. 일본의 호소 중 6번째로 큰 면적을 자랑한다. 주위에 작은 하천이 유입되어 남단에 있는 <a class="new" href="/w/index.php?title=%EA%B5%AC%EC%8B%9C%EB%A1%9C_%EA%B0%95&amp;action=edit&amp;redlink=1" title="구시로 강 (없는 문서)">구시로 강</a>으로 빠져나간다. 호수와 구시로 강이 만나는 상류에는 마을이 있고, 이를 아이누어로〈굿챠로〉라고 부른 데서 유래된 말이다. 아이누어로 굿챠로는“목구멍, 호수가 강으로 빠져나가는 부분”을 의미한다.</p>
<p>호수 중앙에는 일본의 호수안에 있는 섬중 가장 큰 섬인 <a class="new" href="/w/index.php?title=%EB%82%98%EC%B9%B4%EC%8B%9C%EB%A7%88_%EC%84%AC&amp;action=edit&amp;redlink=1" title="나카시마 섬 (없는 문서)">나카시마 섬</a>이 떠있다. 면적은 5.7㎢이며, 둘레는 12km에 이른다. 나카시마 섬 자체가 2중식 화산으로 중앙의 용암원정구의 최고점은 355m이다.</p>
<p>호수 남쪽변은 <a class="new" href="/w/index.php?title=%EC%99%80%EC%BD%94%ED%86%A0_%EB%B0%98%EB%8F%84&amp;action=edit&amp;redlink=1" title="와코토 반도 (없는 문서)">와코토 반도</a>가 돌출되어 있다. 나카시마 섬과 같이 화산 정상이 호수에 떠있는 섬이지만, <a class="new" href="/w/index.php?title=%EC%98%A4%EC%82%AC%EC%93%B0%EB%B2%A0_%EA%B0%95&amp;action=edit&amp;redlink=1" title="오사쓰베 강 (없는 문서)">오사쓰베 강</a>의 <a href="/wiki/%EC%84%A0%EC%83%81%EC%A7%80" title="선상지">선상지</a>로부터 자라난 <a href="/wiki/%EC%82%AC%EC%A3%BC_(%EC%A7%80%EC%A7%88%ED%95%99)" title="사주 (지질학)">사주</a>에 의해 <a class="mw-redirect" href="/wiki/%EC%9C%A1%EA%B3%84%EB%8F%84" title="육계도">육계도</a>가 되었다.</p>
<ul>
<li>섬 : 나카시마 섬</li>
<li>유입하천 : 유카와 강, 오사쓰베 강, 온네나이 강, 아토사 강, 도이코이 강, 온네시레토 강, 시케레펜베쓰 강</li>
<li>유출하천 : 구시로 강</li>
</ul>
<h2><span class="mw-headline" id=".EC.83.9D.ED.83.9C">생태</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EA%B5%BF%EC%83%A4%EB%A1%9C_%ED%98%B8&amp;action=edit&amp;section=2" title="부분 편집: 생태">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>아토사누푸리에서 강산성인 <a href="/wiki/%EC%88%98%EC%86%8C_%EC%9D%B4%EC%98%A8_%EB%86%8D%EB%8F%84" title="수소 이온 농도">pH</a>2전후인 온천수가 호수 북쪽의 유카와 강을 통해 유입되 호수 전체의 pH는 5를 전후한 산성호이다. 이런 까닭에 <a class="mw-redirect" href="/wiki/%EC%96%B4%EB%A5%98" title="어류">어류</a>는 개체수는 적은 편이다. 과거에는 중성에 가까워 어류가 많이 번식한 시기도 있었지만, <a href="/wiki/1938%EB%85%84" title="1938년">1938년</a> 굿샤로 지진으로 호수 밑바닥으로부터 <a href="/wiki/%ED%99%A9" title="황">황</a>이 분출되어 pH가 4정도로 떨어졌기 때문에 대부분의 어류는 전멸했었다. 그 후, 산성도는 줄어들어 호수에 방류한 <a href="/wiki/%EB%AC%B4%EC%A7%80%EA%B0%9C%EC%86%A1%EC%96%B4" title="무지개송어">무지개송어</a> 등이 살고 있다.</p>
<p>현재 호수에는 대형수생생물이 존재할 가능성이 없는 것으로 보인다. 그러나, <a href="/wiki/1970%EB%85%84" title="1970년">1970년</a>대부터 큰 물체가 유영하는 모습의 그림자와 호면의 파문이 계속해서 목격되었고, 전설상에도 거대 <a href="/wiki/%ED%99%8D%EC%86%A1%EC%96%B4" title="홍송어">홍송어</a> 이야기가 전해 오는 것을 근거로 호수에 미확인 거대 생물이 살고 있다는 의견이 있다. 이 미확인 생물을 <a class="new" href="/w/index.php?title=%EA%B5%BF%EC%8B%9C&amp;action=edit&amp;redlink=1" title="굿시 (없는 문서)">굿시</a>로 부르고 있다.</p>
<p>또, 와코토 반도는 <a href="/wiki/%EB%AF%BC%EB%AF%BC%EB%A7%A4%EB%AF%B8" title="민민매미">민민매미</a>의 생식의 북방한계로 <a href="/wiki/1951%EB%85%84" title="1951년">1951년</a>〈와코토 민민매미 발생지〉라는 명칭으로 천연기념물에 지정되었다. <a href="/wiki/%ED%81%B0%EA%B3%A0%EB%8B%88" title="큰고니">큰고니</a>의 서식지로도 알려져 있다.</p>
