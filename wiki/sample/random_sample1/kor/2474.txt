<p><b>궈러 티베트족 자치주</b>(<span style="font-size: smaller;"><a href="/wiki/%ED%8B%B0%EB%B2%A0%ED%8A%B8%EC%96%B4" title="티베트어">티베트어</a>:</span> <span lang="bo" xml:lang="bo">མགོ་ལོག་བོད་རིགས་རང་སྐྱོང་ཁུལ་</span> <small>고로그 티베트족 자치주</small>, <small><a href="/wiki/%EC%A4%91%EA%B5%AD%EC%96%B4" title="중국어">중국어</a></small><small>:</small> <span lang="zh" xml:lang="zh">果洛藏族自治州</span>, <small><a href="/wiki/%ED%95%9C%EC%96%B4_%EB%B3%91%EC%9D%8C" title="한어 병음">병음</a>:</small> <font face="Lucida Sans Unicode">Guǒluò Zàngzú Zìzhìzhōu</font>)는 <a href="/wiki/%EC%A4%91%ED%99%94%EC%9D%B8%EB%AF%BC%EA%B3%B5%ED%99%94%EA%B5%AD" title="중화인민공화국">중화인민공화국</a> <a href="/wiki/%EC%B9%AD%ED%95%98%EC%9D%B4_%EC%84%B1" title="칭하이 성">칭하이 성</a>에 설치된 티베트족 자치주의 하나이다. 중국명 "궈뤄(果洛)"는 티베트명 "고로그"를 음차한 것이다. 티베트의 전통적인 지리 구분에서는 <a href="/wiki/%EC%95%94%EB%8F%84_%EC%A7%80%EA%B5%AC" title="암도 지구">암도 지구</a>의 남부에 해당한다.</p>

<h2><span class="mw-headline" id=".EC.A7.80.EB.A6.AC">지리</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EA%B6%88%EB%A4%84_%ED%8B%B0%EB%B2%A0%ED%8A%B8%EC%A1%B1_%EC%9E%90%EC%B9%98%EC%A3%BC&amp;action=edit&amp;section=1" title="부분 편집: 지리">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>북쪽에서 동쪽으로 걸쳐 칭하이 성의 <a href="/wiki/%ED%95%98%EC%9D%B4%EC%8B%9C_%EB%AA%BD%EA%B3%A8%EC%A1%B1_%ED%8B%B0%EB%B2%A0%ED%8A%B8%EC%A1%B1_%EC%9E%90%EC%B9%98%EC%A3%BC" title="하이시 몽골족 티베트족 자치주">하이시 몽골족 티베트족 자치주</a>, <a href="/wiki/%ED%95%98%EC%9D%B4%EB%82%9C_%ED%8B%B0%EB%B2%A0%ED%8A%B8%EC%A1%B1_%EC%9E%90%EC%B9%98%EC%A3%BC" title="하이난 티베트족 자치주">하이난 티베트족 자치주</a>, <a href="/wiki/%ED%99%A9%EB%82%9C_%ED%8B%B0%EB%B2%A0%ED%8A%B8%EC%A1%B1_%EC%9E%90%EC%B9%98%EC%A3%BC" title="황난 티베트족 자치주">황난 티베트족 자치주</a>, <a href="/wiki/%EA%B0%84%EC%91%A4_%EC%84%B1" title="간쑤 성">간쑤 성</a>의 <a href="/wiki/%EA%B0%84%EB%82%9C_%ED%8B%B0%EB%B2%A0%ED%8A%B8%EC%A1%B1_%EC%9E%90%EC%B9%98%EC%A3%BC" title="간난 티베트족 자치주">간난 티베트족 자치주</a>와 동남쪽은 <a href="/wiki/%EC%93%B0%EC%B4%A8_%EC%84%B1" title="쓰촨 성">쓰촨 성</a>의 <a href="/wiki/%EC%95%84%EB%B0%94_%ED%8B%B0%EB%B2%A0%ED%8A%B8%EC%A1%B1_%EC%B0%BD%EC%A1%B1_%EC%9E%90%EC%B9%98%EC%A3%BC" title="아바 티베트족 창족 자치주">아바 티베트족 창족 자치주</a>와 남쪽은 <a class="mw-redirect" href="/wiki/%EA%B0%80%EC%98%A4%EC%A6%88_%ED%8B%B0%EB%B2%A0%ED%8A%B8%EC%A1%B1_%EC%9E%90%EC%B9%98%EC%A3%BC" title="가오즈 티베트족 자치주">가오즈 티베트족 자치주</a>와 서쪽은 칭하이 성의 <a class="mw-redirect" href="/wiki/%EC%9C%84%EC%8A%88_%ED%8B%B0%EB%B2%A0%ED%8A%B8%EC%A1%B1_%EC%9E%90%EC%B9%98%EC%A3%BC" title="위슈 티베트족 자치주">위슈 티베트족 자치주</a>와 경계를 마주한다.</p>

<h2><span class="mw-headline" id=".ED.96.89.EC.A0.95_.EA.B5.AC.EC.97.AD">행정 구역</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EA%B6%88%EB%A4%84_%ED%8B%B0%EB%B2%A0%ED%8A%B8%EC%A1%B1_%EC%9E%90%EC%B9%98%EC%A3%BC&amp;action=edit&amp;section=3" title="부분 편집: 행정 구역">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
