<p><b>제6회 <a href="/wiki/%EC%95%84%EC%8B%9C%EC%95%88_%EA%B2%8C%EC%9E%84" title="아시안 게임">아시안 게임</a></b>은 <a href="/wiki/%EC%95%84%EC%8B%9C%EC%95%84_%EC%98%AC%EB%A6%BC%ED%94%BD_%ED%8F%89%EC%9D%98%ED%9A%8C" title="아시아 올림픽 평의회">아시아 올림픽 평의회</a>(AGF) 주관으로 <a href="/wiki/1970%EB%85%84" title="1970년">1970년</a> <a href="/wiki/12%EC%9B%94_9%EC%9D%BC" title="12월 9일">12월 9일</a>부터 <a href="/wiki/12%EC%9B%94_20%EC%9D%BC" title="12월 20일">12월 20일</a>까지 <a href="/wiki/%ED%83%80%EC%9D%B4" title="타이">타이</a> <a href="/wiki/%EB%B0%A9%EC%BD%95" title="방콕">방콕</a>에서 열렸다. <a href="/wiki/%EC%95%84%EC%8B%9C%EC%95%84" title="아시아">아시아</a> 18개국이 참가하였으며, 참가 선수는 약 2,400명이다. 원래는 <a href="/wiki/%EB%8C%80%ED%95%9C%EB%AF%BC%EA%B5%AD" title="대한민국">대한민국</a> <a href="/wiki/%EC%84%9C%EC%9A%B8%ED%8A%B9%EB%B3%84%EC%8B%9C" title="서울특별시">서울</a>에서 개최하기로 결정되었으나, 조선민주주의인민공화국의 무력 도발 위협, 국가 경제 발전을 최우선 과제로 삼고 있던 대한민국이 과도한 대회 개최 비용을 감당하기 어렵다는 <a href="/wiki/%EB%B0%95%EC%A0%95%ED%9D%AC" title="박정희">박정희</a> 대통령의 판단으로 개최권을 반납했다.</p>

<h2><span class="mw-headline" id=".EB.8C.80.ED.9A.8C_.EC.9D.B4.EB.AA.A8.EC.A0.80.EB.AA.A8">대회 이모저모</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=1970%EB%85%84_%EC%95%84%EC%8B%9C%EC%95%88_%EA%B2%8C%EC%9E%84&amp;action=edit&amp;section=1" title="부분 편집: 대회 이모저모">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<ul>
<li>아시안 게임 개최비를 대한민국을 중심으로 한 약10개국에서 원조하여 직전개최지였던 <a href="/wiki/%ED%83%80%EC%9D%B4" title="타이">타이</a> <a href="/wiki/%EB%B0%A9%EC%BD%95" title="방콕">방콕</a>에서 다시 아시안 게임을 개최하게 되었다. <sup class="reference" id="cite_ref-1"><a href="#cite_note-1">[1]</a></sup></li>
<li><a href="/wiki/%EC%9A%94%ED%8A%B8" title="요트">요트</a>가 처음으로 정식종목으로 채택되었다.</li>
<li>대한민국 남자 농구가 대회사상 처음으로 우승하였다.</li>
</ul>
