<p><b>100식 기관단총</b>(<span style="font-size: smaller;"><a href="/wiki/%EC%9D%BC%EB%B3%B8%EC%96%B4" title="일본어">일본어</a>:</span> <span lang="ja"><span lang="ja"><ruby><rb>一〇〇式機関短銃</rb> <rp>(</rp><rt>ひゃくしききかんたんじゅう</rt><rp>)</rp></ruby></span></span> <small>햐쿠시키키칸탄주<sup>[<a href="/wiki/%EC%9C%84%ED%82%A4%EB%B0%B1%EA%B3%BC:%EC%9D%BC%EB%B3%B8%EC%96%B4%EC%9D%98_%ED%95%9C%EA%B8%80_%ED%91%9C%EA%B8%B0" title="위키백과:일본어의 한글 표기">*</a>]</sup></small>)은 <a class="mw-redirect" href="/wiki/2%EC%B0%A8_%EC%84%B8%EA%B3%84%EB%8C%80%EC%A0%84" title="2차 세계대전">2차 세계대전</a> 당시 <a href="/wiki/%EC%9D%BC%EB%B3%B8_%EC%A0%9C%EA%B5%AD" title="일본 제국">일본 제국</a>의 <a href="/wiki/%EA%B8%B0%EA%B4%80%EB%8B%A8%EC%B4%9D" title="기관단총">기관단총</a>이다. 삼국동맹 당시, 일본이 독일의 <a class="mw-redirect" href="/wiki/MP18" title="MP18">MP18</a>을 수입하면서 그것을 모티브로 삼아 개발하였으며, 대량생산하려 했지만 소총에 착검을 해 쓰던 일본군에게는 기관단총은 너무 생소한 무기였다. 그래서 생산량은 1만정을 넘기지 못하고 일부 특수부대에만 지급되었다. 발사속도는 분당 300~350발, 중량은 6kg. 탄창은 30발 탄창이있다.</p>
<h2><span class="mw-headline" id=".EC.97.AD.EC.82.AC">역사</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=100%EC%8B%9D_%EA%B8%B0%EA%B4%80%EB%8B%A8%EC%B4%9D&amp;action=edit&amp;section=1" title="부분 편집: 역사">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>스위스 SIG사의 <a class="mw-redirect" href="/wiki/MP18" title="MP18">MP18</a>(SIG M1920) 을 수입하여 모티브로 <a class="mw-redirect" href="/wiki/%EB%82%A8%EB%B6%80_%EA%B8%B0%EC%A7%80%EB%A1%9C" title="남부 기지로">남부 기지로</a>에 의해 <a href="/wiki/1939%EB%85%84" title="1939년">1939년</a> 설계되었다. 8x22 mm 남부 권총탄을 탄약으로 채택하고, <a href="/wiki/%EC%8A%A4%ED%85%90_%EA%B8%B0%EA%B4%80%EB%8B%A8%EC%B4%9D" title="스텐 기관단총">스텐 기관단총</a>처럼 총열 옆에서 급탄받는 구조로 설계된 최초의 100식은 소염기, 사거리를 조절할 수 있는 가늠자. 양각대, 대검꽂이를 갖추었다. <a href="/wiki/1942%EB%85%84" title="1942년">1942년</a>부터 일본 제국군에 보급되기 시작하였으며, 공수부대에는 낙하시에 걸리적거리지 않게 접이식 개머리판으로 보급되었다. 전쟁이 심화되면서 자원이 고갈되면서 <a href="/wiki/1944%EB%85%84" title="1944년">1944년</a>부터 고정된 가늠자와 단순화된 대검꽂이로 설계가 단순화되었다. 그에 따라 발사 속도가 800 발 / 분으로 빨라졌지만, 보급로가 자주 기습당하는 등의 이유로 전선으로의 보급률은 매우 저조했다.</p>
<h2><span class="mw-headline" id=".EC.A0.9C.EC.9B.90">제원</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=100%EC%8B%9D_%EA%B8%B0%EA%B4%80%EB%8B%A8%EC%B4%9D&amp;action=edit&amp;section=2" title="부분 편집: 제원">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<ul>
<li>설계 / 생산: 남부 기지로 / 나고야 조병창 생산회사, 도리이마츠 제작소</li>
<li>개발: 1939년</li>
<li>생산/사용 기간: 1942년 ~ 1945년</li>
<li>구경: <a class="new" href="/w/index.php?title=8x22_mm_%EB%82%A8%EB%B6%80&amp;action=edit&amp;redlink=1" title="8x22 mm 남부 (없는 문서)">8x22 mm 남부</a></li>
<li>작동 방식: <a class="mw-redirect" href="/wiki/%EA%B3%B5%EB%83%89" title="공냉">공냉</a>, <a class="new" href="/w/index.php?title=%EC%A0%84%EC%9E%90%EB%8F%99&amp;action=edit&amp;redlink=1" title="전자동 (없는 문서)">전자동</a>, <a class="mw-redirect" href="/wiki/%EC%98%A4%ED%94%88%EB%B3%BC%ED%8A%B8" title="오픈볼트">오픈볼트</a>, <a class="new" href="/w/index.php?title=%EB%B8%94%EB%A1%9C%EC%9A%B0%EB%B0%B1&amp;action=edit&amp;redlink=1" title="블로우백 (없는 문서)">블로우백</a></li>
<li>강선: 6조 우선(right twist)</li>
<li>급탄: 30발들이 곡선형 <a href="/wiki/%ED%83%84%EC%B0%BD" title="탄창">탄창</a></li>
<li>길이(총열/전장): 228 / 900 mm</li>
<li>무게: 4.22 kg (비장전) ~ 6 kg (장전)</li>
<li>발사 속도: 450 800 발 / 분</li>
<li>총구 속도: 335 미터 / 초</li>
<li>유효 사거리: 100 m</li>
</ul>
