<h2><span class="mw-headline" id=".EB.8C.80.ED.9A.8C_.EC.9D.B4.EB.AA.A8.EC.A0.80.EB.AA.A8">대회 이모저모</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=1972%EB%85%84_%EB%8F%99%EA%B3%84_%EC%98%AC%EB%A6%BC%ED%94%BD&amp;action=edit&amp;section=4" title="부분 편집: 대회 이모저모">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<ul>
<li><a href="/wiki/1940%EB%85%84" title="1940년">1940년</a>에 <a class="mw-redirect" href="/wiki/%EC%82%BF%ED%8F%AC%EB%A1%9C" title="삿포로">삿포로</a>에서 <a href="/wiki/%EB%8F%99%EA%B3%84_%EC%98%AC%EB%A6%BC%ED%94%BD" title="동계 올림픽">동계 올림픽</a>이 열리기로 했으나, <a href="/wiki/%EC%A4%91%EC%9D%BC_%EC%A0%84%EC%9F%81" title="중일 전쟁">중일 전쟁</a>의 발발로 개최권을 반납했다.</li>
<li><a href="/wiki/%EC%9C%A0%EB%9F%BD" title="유럽">유럽</a>과 <a href="/wiki/%EB%AF%B8%EA%B5%AD" title="미국">미국</a> 이외의 지역에서 처음으로 열린 동계 올림픽 대회이다.</li>
<li><a href="/wiki/%EB%84%A4%EB%8D%9C%EB%9E%80%EB%93%9C" title="네덜란드">네덜란드</a>의 <a class="new" href="/w/index.php?title=%EC%86%85%ED%81%AC&amp;action=edit&amp;redlink=1" title="솅크 (없는 문서)">솅크</a>가 남자 <a href="/wiki/%EC%8A%A4%ED%94%BC%EB%93%9C_%EC%8A%A4%EC%BC%80%EC%9D%B4%ED%8C%85" title="스피드 스케이팅">스피드 스케이팅</a> 3관왕에 올랐다.</li>
<li><a class="mw-redirect" href="/wiki/%EC%8A%A4%ED%82%A4%EC%A0%90%ED%94%84" title="스키점프">스키점프</a> 노멀힐 경기에서 <a href="/wiki/%EC%9D%BC%EB%B3%B8" title="일본">일본</a> 선수들이 <a href="/wiki/%EA%B8%88%EB%A9%94%EB%8B%AC" title="금메달">금메달</a>, <a href="/wiki/%EC%9D%80%EB%A9%94%EB%8B%AC" title="은메달">은메달</a>, <a href="/wiki/%EB%8F%99%EB%A9%94%EB%8B%AC" title="동메달">동메달</a>을 휩쓸었다.</li>
</ul>
