<h2><span class="mw-headline" id=".EB.B0.B0.EA.B2.BD">배경</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EA%B7%B8%EB%A0%88%EB%82%98%EB%8B%A4_%EC%B9%A8%EA%B3%B5&amp;action=edit&amp;section=1" title="부분 편집: 배경">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>미국의 침공이 있기 직전 그레나다에서는 <a href="/wiki/%EC%86%8C%EB%A0%A8" title="소련">소련</a>과 <a href="/wiki/%EC%BF%A0%EB%B0%94" title="쿠바">쿠바</a>의 지원을 받은 군부에 의해 <a href="/wiki/%EC%BF%A0%EB%8D%B0%ED%83%80" title="쿠데타">쿠데타</a>가 일어나 <a class="new" href="/w/index.php?title=%EB%AA%A8%EB%A6%AC%EC%8A%A4_%EB%B9%84%EC%88%8D&amp;action=edit&amp;redlink=1" title="모리스 비숍 (없는 문서)">모리스 비숍</a> 총리와 세 명의 각료가 처형됐다. 쿠바는 그레나다의 새 정권에 대해 강력한 재정적 지원 및 무장된 건설노동자를 보냈다. 이는 미국으로 하여금 "제2의 쿠바"를 우려하게 하여, 강한 위기감을 주었다.</p>
<h2><span class="mw-headline" id=".EA.B2.BD.EA.B3.BC">경과</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EA%B7%B8%EB%A0%88%EB%82%98%EB%8B%A4_%EC%B9%A8%EA%B3%B5&amp;action=edit&amp;section=2" title="부분 편집: 경과">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>그레나다 침공에는 미군 병력 7천명, 동카리브 제국(諸國)기구(OECS) 6개국에서 300〜500명이 투입되었다. 그레나다측은 병력 1,500명에 쿠바인 700명(고도의 군사훈련을 받은 무장 건설노동자 및 군사고문)이었다. 이외에 그레나다 국내의 <a class="mw-redirect" href="/wiki/%EC%86%8C%EB%B9%84%EC%97%90%ED%8A%B8_%EC%97%B0%EB%B0%A9" title="소비에트 연방">소련</a>, <a href="/wiki/%EC%A1%B0%EC%84%A0%EB%AF%BC%EC%A3%BC%EC%A3%BC%EC%9D%98%EC%9D%B8%EB%AF%BC%EA%B3%B5%ED%99%94%EA%B5%AD" title="조선민주주의인민공화국">조선민주주의인민공화국</a>, <a class="mw-redirect" href="/wiki/%EB%8F%85%EC%9D%BC%EB%AF%BC%EC%A3%BC%EA%B3%B5%ED%99%94%EA%B5%AD" title="독일민주공화국">독일민주공화국</a>, <a href="/wiki/%EB%B6%88%EA%B0%80%EB%A6%AC%EC%95%84" title="불가리아">불가리아</a>, <a href="/wiki/%EB%A6%AC%EB%B9%84%EC%95%84" title="리비아">리비아</a>에서 온 60명 가량의 고문(顧問)이 있었다. 또 <a href="/wiki/%EC%9D%B8%EB%8F%84" title="인도">인도</a>에서 200명, <a href="/wiki/%EC%A4%91%ED%99%94%EC%9D%B8%EB%AF%BC%EA%B3%B5%ED%99%94%EA%B5%AD" title="중화인민공화국">중화인민공화국</a>에서 82명의 보병(歩兵)이 참전했다는 정보도 있다.</p>
<p>10월 25일 오전 5시에 선전포고 없이 시작된 그레나다 침공은, 미국에게 있어서는 <a class="mw-redirect" href="/wiki/%EC%9B%94%EB%82%A8%EC%A0%84" title="월남전">월남전</a> 이후 최초의 대규모 군사행동이었다. 12월 15일 전도(全島)가 제압되고, 섬 안의 <a href="/wiki/%EC%BF%A0%EB%B0%94" title="쿠바">쿠바</a>, <a href="/wiki/%EC%A1%B0%EC%84%A0%EB%AF%BC%EC%A3%BC%EC%A3%BC%EC%9D%98%EC%9D%B8%EB%AF%BC%EA%B3%B5%ED%99%94%EA%B5%AD" title="조선민주주의인민공화국">조선민주주의인민공화국</a>, <a href="/wiki/%EB%A6%AC%EB%B9%84%EC%95%84" title="리비아">리비아</a>, <a class="mw-redirect" href="/wiki/%EB%8F%85%EC%9D%BC%EB%AF%BC%EC%A3%BC%EA%B3%B5%ED%99%94%EA%B5%AD" title="독일민주공화국">독일민주공화국</a>, <a href="/wiki/%EB%B6%88%EA%B0%80%EB%A6%AC%EC%95%84" title="불가리아">불가리아</a> 국적자들은 연금(軟禁)되었다.</p>
<h2><span class="mw-headline" id=".EC.98.81.ED.96.A5">영향</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EA%B7%B8%EB%A0%88%EB%82%98%EB%8B%A4_%EC%B9%A8%EA%B3%B5&amp;action=edit&amp;section=3" title="부분 편집: 영향">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>미국은 그레나다의 공산화를 막기 위해 기습공격을 단행했다고 주장하면서 주권 침해를 정당화했으나, <a href="/wiki/%EC%98%81%EA%B5%AD" title="영국">영국</a> 등 동맹국을 포함한 국제사회는 미국의 그레나다 침공을 비판했고, 언론보도까지도 제한되는 비민주적인 일까지 벌어지면서 세계 언론사들의 분노를 사기도 했다.</p>
<p>미국은 군사목표를 달성한 뒤 그레나다에 <a href="/wiki/1985%EB%85%84" title="1985년">1985년</a> <a href="/wiki/6%EC%9B%94" title="6월">6월</a>까지 미군을 주둔시켰다. 미국의 그레나다 침공은 당시 대통령 <a class="mw-redirect" href="/wiki/%EB%A0%88%EC%9D%B4%EA%B1%B4" title="레이건">레이건</a>의 잘못 중 하나로 비판받고 있다.</p>
