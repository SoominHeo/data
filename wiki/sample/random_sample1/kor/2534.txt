<p><b>조지프 그레이엄 "그레이" 데이비스</b>(<span style="font-size: smaller;"><a href="/wiki/%EC%98%81%EC%96%B4" title="영어">영어</a>:</span> <span lang="en" xml:lang="en">Joseph Graham "Gray" Davis, Jr.</span>, <a href="/wiki/1942%EB%85%84" title="1942년">1942년</a> <a href="/wiki/12%EC%9B%94_26%EC%9D%BC" title="12월 26일">12월 26일</a> ~ )는 <a href="/wiki/%EB%AF%B8%EA%B5%AD" title="미국">미국</a>의 정치인이다.</p>
<p><a href="/wiki/%EB%89%B4%EC%9A%95" title="뉴욕">뉴욕</a>에서 태어났으며, 어릴 때 <a href="/wiki/%EC%BA%98%EB%A6%AC%ED%8F%AC%EB%8B%88%EC%95%84_%EC%A3%BC" title="캘리포니아 주">캘리포니아 주</a>로 이주하였다. <a href="/wiki/%EC%8A%A4%ED%83%A0%ED%8D%BC%EB%93%9C_%EB%8C%80%ED%95%99%EA%B5%90" title="스탠퍼드 대학교">스탠퍼드 대학교</a>에서 역사학을 전공으로 학부를 마치고 <a href="/wiki/%EC%BB%AC%EB%9F%BC%EB%B9%84%EC%95%84_%EB%B2%95%ED%95%99%EB%8C%80%ED%95%99%EC%9B%90" title="컬럼비아 법학대학원">컬럼비아 로스쿨</a>에서 법무박사(J.D.) 학위를 취득한 후 <a href="/wiki/%EB%B3%80%ED%98%B8%EC%82%AC" title="변호사">변호사</a> 자격을 얻었다. <a href="/wiki/%EB%B2%A0%ED%8A%B8%EB%82%A8_%EC%A0%84%EC%9F%81" title="베트남 전쟁">베트남 전쟁</a>에 참전했고, 캘리포니아 주로 돌아와 <a href="/wiki/%EB%AF%BC%EC%A3%BC%EB%8B%B9_(%EB%AF%B8%EA%B5%AD)" title="민주당 (미국)">민주당</a>과 관계를 맺으며 주정부, 주의회에서 일했다. 주의회 의원, 주 감사원장, 부지사를 거쳐 <a href="/wiki/1998%EB%85%84" title="1998년">1998년</a> 주지사 선거에 당선, <a href="/wiki/1999%EB%85%84" title="1999년">1999년</a>부터 재직했고, <a href="/wiki/2002%EB%85%84" title="2002년">2002년</a> 재선되었다. 그러나 주의 재정적자, 전력난, 불법체류자 문제 등에 대한 부적절한 대처로 인기가 떨어져 <a href="/wiki/2003%EB%85%84" title="2003년">2003년</a> 그에 대한 <a class="mw-redirect" href="/wiki/%EC%A3%BC%EB%AF%BC%EC%86%8C%ED%99%98" title="주민소환">주민소환</a>이 발의되었다. <a href="/wiki/2003%EB%85%84" title="2003년">2003년</a> <a href="/wiki/10%EC%9B%94_7%EC%9D%BC" title="10월 7일">10월 7일</a>, 그에 대한 소환투표와 주지사 선거가 함께 치러져, 투표 결과 주민 소환이 확정되어 그는 주지사직을 상실하였고, 동시에 후임 주지사로 <a href="/wiki/%EC%98%81%ED%99%94%EB%B0%B0%EC%9A%B0" title="영화배우">영화배우</a> 출신인 <a href="/wiki/%EA%B3%B5%ED%99%94%EB%8B%B9_(%EB%AF%B8%EA%B5%AD)" title="공화당 (미국)">공화당</a> 소속의 <a href="/wiki/%EC%95%84%EB%84%90%EB%93%9C_%EC%8A%88%EC%9B%8C%EC%A0%9C%EB%84%A4%EA%B1%B0" title="아널드 슈워제네거">아널드 슈워제네거</a>가 선출되었다.<sup class="reference" id="cite_ref-1"><a href="#cite_note-1">[1]</a></sup> 그 후 데이비스는 <a href="/wiki/%EC%BA%98%EB%A6%AC%ED%8F%AC%EB%8B%88%EC%95%84_%EB%8C%80%ED%95%99%EA%B5%90_%EB%A1%9C%EC%8A%A4%EC%95%A4%EC%A0%A4%EB%A0%88%EC%8A%A4" title="캘리포니아 대학교 로스앤젤레스">캘리포니아 대학교 로스앤젤레스</a> 공공정책 대학원 강사, <a href="/wiki/%EB%A1%9C%ED%8E%8C" title="로펌">로펌</a> 로엡 &amp; 로엡(Loeb &amp; Loeb) 소속 변호사, 그리고 애니메이션 회사 <a href="/wiki/DIC_%EC%97%94%ED%84%B0%ED%85%8C%EC%9D%B8%EB%A8%BC%ED%8A%B8" title="DIC 엔터테인먼트">DIC 엔터테인먼트</a> 이사회 멤버로 활동했다.</p>
