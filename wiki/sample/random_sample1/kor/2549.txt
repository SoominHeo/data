<p><b>그리스-페르시아 전쟁</b>(또는 <b>페르시아 전쟁</b>으로 칭하기도 한다.)은 <a href="/wiki/%EA%B3%A0%EB%8C%80_%EA%B7%B8%EB%A6%AC%EC%8A%A4" title="고대 그리스">고대 그리스</a>의 도시국가 연합과 <a href="/wiki/%EC%95%84%EC%BC%80%EB%A9%94%EB%84%A4%EC%8A%A4_%EC%A0%9C%EA%B5%AD" title="아케메네스 제국">페르시아 제국</a>이 격돌한 전쟁을 말하며, 기원전 499년에 일어나 450년까지 이어졌다. 소규모 도시국가로 이루어진 그리스 세계와 거대한 페르시아 제국이 충돌한 것은 기원전 550년경 <a class="mw-redirect" href="/wiki/%ED%82%A4%EB%A3%A8%EC%8A%A4_%EB%8C%80%EC%99%95" title="키루스 대왕">키루스 대왕</a>이 <a href="/wiki/%EC%9D%B4%EC%98%A4%EB%8B%88%EC%95%84" title="이오니아">이오니아</a>를 정복하면서 일어났다. 이오니아의 독립적인 도시국가들을 다스리기 위하여 페르시아는 각 나라에 <a href="/wiki/%EC%B0%B8%EC%A3%BC" title="참주">참주</a>를 보내었다. 그러나 이들은 그리스인과 페르시아인 모두에게 문제거리가 되어버렸다.</p>
<p>기원전 499년 당시 <a href="/wiki/%EB%B0%80%EB%A0%88%ED%86%A0%EC%8A%A4" title="밀레토스">밀레토스</a>의 참주 <a href="/wiki/%EC%95%84%EB%A6%AC%EC%8A%A4%ED%83%80%EA%B3%A0%EB%9D%BC%EC%8A%A4" title="아리스타고라스">아리스타고라스</a>는 페르시아의 도움을 받아 <a href="/wiki/%EB%82%99%EC%86%8C%EC%8A%A4_%EC%84%AC" title="낙소스 섬">낙소스 섬</a>을 정복하고자 <a class="new" href="/w/index.php?title=%EB%82%99%EC%86%8C%EC%8A%A4_%EA%B3%B5%EB%B0%A9%EC%A0%84_(%EA%B8%B0%EC%9B%90%EC%A0%84_499%EB%85%84)&amp;action=edit&amp;redlink=1" title="낙소스 공방전 (기원전 499년) (없는 문서)">원정대</a>를 보내었다. 그러나 원정대는 패주하였고, 아리스타고라스는 생각을 바꾸어 (별 어려움 없이) 페르시아에 대항하여 <a class="mw-redirect" href="/wiki/%EC%86%8C%EC%95%84%EC%8B%9C%EC%95%84" title="소아시아">소아시아</a>의 <a href="/wiki/%EA%B3%A0%EB%8C%80_%EA%B7%B8%EB%A6%AC%EC%8A%A4" title="고대 그리스">헬라스</a> 지역 전체가 반란을 일으키도록 선동하였다. 그리하여 <a href="/wiki/%EC%9D%B4%EC%98%A4%EB%8B%88%EC%95%84_%EB%B0%98%EB%9E%80" title="이오니아 반란">이오니아 반란</a>이 일어나 기원전 493년까지 이어졌는데, 점점 소아시아의 더 많은 지역이 전쟁에 뛰어들었다. 아리스타고라스는 <a class="mw-redirect" href="/wiki/%EC%95%84%ED%85%8C%EB%82%98%EC%9D%B4" title="아테나이">아테나이</a>와 <a href="/wiki/%EC%97%90%EB%A0%88%ED%8A%B8%EB%A6%AC%EC%95%84" title="에레트리아">에레트리아</a>의 군사적 지원을 확보하고 기원전 498년에 이들 군대의 도움으로 페르시아의 지역 도읍인 <a class="mw-redirect" href="/wiki/%EC%82%AC%EB%A5%B4%EB%8D%B0%EC%9D%B4%EC%8A%A4" title="사르데이스">사르데이스</a>를 점령하여 불태웠다. 페르시아의 <a class="mw-redirect" href="/wiki/%EB%8B%A4%EB%A6%AC%EC%9A%B0%EC%8A%A4_%EB%8C%80%EC%99%95" title="다리우스 대왕">다리우스 대왕</a>은 이런 일을 한 아테나이와 에레트리아에 복수하기로 맹세하였다. 반란은 계속되어 기원전 497년~495년 사이에 양측은 교착 상태에 빠졌다. 기원전 494년에 페르시아는 다시 군대를 일으켜 반란의 중심지인 밀레토스를 공격하였다. <a class="new" href="/w/index.php?title=%EB%9D%BC%EB%8D%B0_%EC%A0%84%ED%88%AC&amp;action=edit&amp;redlink=1" title="라데 전투 (없는 문서)">라데 전투</a>에서 이오니아는 결정적인 패배를 당하여 반란군은 몰락하였고, 이듬해에 마지막 저항도 진압되었다.</p>
<p>다리우스는 자신의 제국 땅에서 다시는 반란이 일어나고 본토 그리스인이 개입하는 일을 막기 위하여 그리스를 달래고 사르데이스를 불태운 아테나이와 에레트리아를 주벌할 계획에 착수하였다. 기원전 492년에 <a class="new" href="/w/index.php?title=%EC%A0%9C1%EC%B0%A8_%EA%B7%B8%EB%A6%AC%EC%8A%A4-%ED%8E%98%EB%A5%B4%EC%8B%9C%EC%95%84_%EC%A0%84%EC%9F%81&amp;action=edit&amp;redlink=1" title="제1차 그리스-페르시아 전쟁 (없는 문서)">페르시아는 처음으로 그리스를 침공</a>하였는데, 페르시아 장군 <a href="/wiki/%EB%A7%88%EB%A5%B4%EB%8F%84%EB%8B%88%EC%98%A4%EC%8A%A4" title="마르도니오스">마르도니오스</a>는 <a class="mw-redirect" href="/wiki/%ED%8A%B8%EB%9D%BC%EC%BC%80" title="트라케">트라케</a>와 <a href="/wiki/%EB%A7%88%EC%BC%80%EB%8F%84%EB%8B%88%EC%95%84_%EC%99%95%EA%B5%AD" title="마케도니아 왕국">마케도니아</a>를 복속하였으나 여러 불상사로 조기에 작전을 끝내야하였다. 기원전 490년에 페르시아의 두 번째 군대가 <a class="new" href="/w/index.php?title=%EB%8B%A4%ED%8B%B0%EC%8A%A4&amp;action=edit&amp;redlink=1" title="다티스 (없는 문서)">다티스</a>와 <a class="new" href="/w/index.php?title=%EC%95%84%EB%A5%B4%ED%83%80%ED%8E%98%EB%A5%B4%EB%84%A4%EC%8A%A4_(%EC%95%84%EB%A5%B4%ED%83%80%ED%8E%98%EB%A5%B4%EB%84%A4%EC%8A%A4%EC%9D%98_%EC%95%84%EB%93%A4)&amp;action=edit&amp;redlink=1" title="아르타페르네스 (아르타페르네스의 아들) (없는 문서)">아르타페르네스</a>의 지휘 하에 그리스로 출정하였는데, 이번에는 <a href="/wiki/%EC%97%90%EA%B2%8C_%ED%95%B4" title="에게 해">에게 해</a>를 건넜다. 페르시아 원정군은 <a class="mw-redirect" href="/wiki/%ED%80%B4%ED%81%B4%EB%9D%BC%EB%8D%B0%EC%8A%A4_%EC%A0%9C%EB%8F%84" title="퀴클라데스 제도">퀴클라데스 제도</a>를 점령하고 <a href="/wiki/%EC%97%90%EB%A0%88%ED%8A%B8%EB%A6%AC%EC%95%84" title="에레트리아">에레트리아</a>를 포위한 끝에 함락하여 파괴해버렸다. 그러나 페르시아군은 아테나이를 공격하는 길에 <a href="/wiki/%EB%A7%88%EB%9D%BC%ED%86%A4_%EC%A0%84%ED%88%AC" title="마라톤 전투">마라톤 전투</a>에서 아테나이에 결정적인 패배를 당하고 말아서 페르시아의 노력은 당분간 물거품으로 돌아가게 된다. 그러자 다리우스는 그리스를 완전히 정복할 계획을 세웠으나, 기원전 486년에 세상을 떠났고, 그리스 정복의 과업은 아들 <a href="/wiki/%ED%81%AC%EC%84%B8%EB%A5%B4%ED%81%AC%EC%84%B8%EC%8A%A4_1%EC%84%B8" title="크세르크세스 1세">크세르크세스 1세</a>에게 넘어갔다. 기원전 480년에 크세르크세스는 몸소 원정길에 나서 터무니없이 거대한 군대를 이끌고 <a class="new" href="/w/index.php?title=%EC%A0%9C2%EC%B0%A8_%EA%B7%B8%EB%A6%AC%EC%8A%A4-%ED%8E%98%EB%A5%B4%EC%8B%9C%EC%95%84_%EC%A0%84%EC%9F%81&amp;action=edit&amp;redlink=1" title="제2차 그리스-페르시아 전쟁 (없는 문서)">두 번째로 그리스를 침공</a>하였다. 페르시아는 유명한 <a class="mw-redirect" href="/wiki/%ED%85%8C%EB%A5%B4%EB%AA%A8%ED%93%94%EB%9D%BC%EC%9D%B4_%EC%A0%84%ED%88%AC" title="테르모퓔라이 전투">테르모퓔라이 전투</a>에서 (<a href="/wiki/%EC%8A%A4%ED%8C%8C%EB%A5%B4%ED%83%80" title="스파르타">스파르타</a>와 아테나이가 주도한) 그리스 연합군을 무찔러 그리스 대부분의 지역을 장악하였으나, 연합군 함대를 파괴하려다 페르시아는 <a href="/wiki/%EC%82%B4%EB%9D%BC%EB%AF%B8%EC%8A%A4_%ED%95%B4%EC%A0%84" title="살라미스 해전">살라미스 해전</a>에서 대패하였다. 이듬해 헬라스 연합군은 반격에 나서 <a class="new" href="/w/index.php?title=%ED%94%8C%EB%9D%BC%ED%83%80%EC%9D%B4%EC%95%84%EC%9D%B4_%EC%A0%84%ED%88%AC&amp;action=edit&amp;redlink=1" title="플라타이아이 전투 (없는 문서)">플라타이아이 전투</a>에서 페르시아군을 격퇴하고 그리스 침략을 막아내었다.</p>
<p>헬라스 사람들은 여세를 몰아 <a class="new" href="/w/index.php?title=%EC%86%8C%EC%8A%A4%ED%85%8C%EC%8A%A4&amp;action=edit&amp;redlink=1" title="소스테스 (없는 문서)">소스테스</a>(기원전 479년)과 <a class="mw-redirect" href="/wiki/%EB%B7%94%EC%9E%94%ED%8B%B0%EC%98%A8" title="뷔잔티온">뷔잔티온</a>(기원전 478년)의 페르시아 주둔군을 몰아내고 <a class="new" href="/w/index.php?title=%EB%AE%88%EC%B9%BC%EB%A0%88_%EC%A0%84%ED%88%AC&amp;action=edit&amp;redlink=1" title="뮈칼레 전투 (없는 문서)">뮈칼레 전투</a>에서 남은 페르시아 함대를 격침하였다. 뷔잔티온 공성전에서 <a class="new" href="/w/index.php?title=%ED%8C%8C%EC%9A%B0%EC%82%AC%EB%8B%88%EC%95%84%EC%8A%A4_(%EC%9E%A5%EA%B5%B0)&amp;action=edit&amp;redlink=1" title="파우사니아스 (장군) (없는 문서)">파우사니아스</a> 장군의 행동으로 여러 그리스 국가가 스파르타를 멀리하게 되었으며, 反페르시아 연합은 아테나이가 주도하는 <a href="/wiki/%EB%8D%B8%EB%A1%9C%EC%8A%A4_%EB%8F%99%EB%A7%B9" title="델로스 동맹">델로스 동맹</a>으로 재조직되었다. 델로스 동맹은 이후 30년간 페르시아에 대항한 전쟁을 계속하여 유럽에 남은 페르시아 주둔군을 몰아냈다. 기원전 466년 <a class="new" href="/w/index.php?title=%EC%97%90%EC%9A%B0%EB%A4%BC%EB%A9%94%EB%8F%88_%EC%A0%84%ED%88%AC&amp;action=edit&amp;redlink=1" title="에우뤼메돈 전투 (없는 문서)">에우뤼메돈 전투</a>에서 델로스 연맹은 두 번의 승리를 거두어 결국 이오니아 도시들의 독립을 쟁취하였다. 그러나 이집트에서 일어난 반란(기원전 460년~454년)을 지원하려고 델로스 동맹이 개입하다가 크게 패하고 원정을 중단하였다. 기원전 451년에는 <a class="mw-redirect" href="/wiki/%ED%80%B4%ED%94%84%EB%A1%9C%EC%8A%A4" title="퀴프로스">퀴프로스</a>에 함대가 파견되었으나 별 소득을 얻지 못하고 철수하면서 그리스와 페르시아의 전쟁은 조용하게 끝나버렸다. 어떤 사료에서는 전쟁 상태가 종료된 것이 아테나이와 페르시아간의 소위 칼리아스 조약이라는 평화 협정에 따라 이루어졌다고 하기도 한다.</p>
<p>오늘날 이 전쟁에 대해서 우리가 알 수 있는 사료의 대부분은 그리스 역사가들(특히 <a href="/wiki/%ED%97%A4%EB%A1%9C%EB%8F%84%ED%86%A0%EC%8A%A4" title="헤로도토스">헤로도토스</a>)과 일부 <a href="/wiki/%EA%B3%A0%EB%8C%80_%EB%A1%9C%EB%A7%88" title="고대 로마">로마</a> 역사가들의 사료이다.</p>

<h2><span class="mw-headline" id=".EC.A0.84.EC.9F.81.EC.9D.98_.EB.B0.B0.EA.B2.BD">전쟁의 배경</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EA%B7%B8%EB%A6%AC%EC%8A%A4-%ED%8E%98%EB%A5%B4%EC%8B%9C%EC%95%84_%EC%A0%84%EC%9F%81&amp;action=edit&amp;section=1" title="부분 편집: 전쟁의 배경">원본 편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>페르시아가 그리스 역사에 등장한 것은 페르시아가 <a class="mw-redirect" href="/wiki/%EB%A4%BC%EB%94%94%EC%95%84" title="뤼디아">뤼디아</a>와 <a href="/wiki/%EC%9D%B4%EC%98%A4%EB%8B%88%EC%95%84" title="이오니아">이오니아</a>의 몇몇 그리스계 도시국가를 지배하게 되면서부터이다. 뤼디아는 소아시아의 <a href="/wiki/%EB%B0%80%EB%A0%88%ED%86%A0%EC%8A%A4" title="밀레토스">밀레토스</a> 같은 그리스계 도시국가를 정복했고 이들 그리스계 도시국가는 내부적으로는 자치가 허용되면서도 외교적으로는 뤼디아의 속국으로 공물을 뤼디아에 바치고 뤼디아가 메디아와 전쟁을 벌일 때 뤼디아를 위해 군대를 파병해야 했다.</p>
<p><a href="/wiki/%EA%B8%B0%EC%9B%90%EC%A0%84_544%EB%85%84" title="기원전 544년">기원전 544년</a>경 <a href="/wiki/%ED%82%A4%EB%A3%A8%EC%8A%A4_2%EC%84%B8" title="키루스 2세">키루스 2세</a>가 메디아에서 반란을 일으켜 <a href="/wiki/%EB%A9%94%EB%94%94%EC%95%84" title="메디아">메디아</a>를 정복하고 페르시아 제국을 세웠다. 뤼디아는 페르시아와 전쟁을 벌였으나 패하고 이오니아의 그리스계 도시들은 페르시아 제국의 수중에 들어갔다. 페르시아의 총독은 이들 그리스계 도시들에 전제군주를 심어놓고 '왕중의 왕'인 페르시아의 황제에게 세금을 바치게 했다. <a href="/wiki/%EA%B8%B0%EC%9B%90%EC%A0%84_525%EB%85%84" title="기원전 525년">기원전 525년</a> 페르시아의 황제 <a href="/wiki/%EC%BA%84%EB%B9%84%EC%84%B8%EC%8A%A4_2%EC%84%B8" title="캄비세스 2세">캄비세스 2세</a>가 이집트원정에 성공하자 <a class="mw-redirect" href="/wiki/%EC%86%8C%EC%95%84%EC%8B%9C%EC%95%84" title="소아시아">소아시아</a>의 그리스계 도시들의 페르시아에 대해 불만이 더 커졌는데 많은 상업과 무역을 일찍부터 페르사아에 편입된 <a href="/wiki/%ED%8E%98%EB%8B%88%ED%82%A4%EC%95%84" title="페니키아">페니키아</a>인들이 차지하였기 때문이었다. 캄비세스 2세의 뒤를 이은 다리우스 1세는 <a href="/wiki/%EB%A6%AC%EB%B9%84%EC%95%84" title="리비아">리비아</a>를 정복하고 인도에 이르기까지 페르시아 제국의 판도를 넓혔다.</p>
<div class="rellink relarticle mainarticle"><img alt="" data-file-height="128" data-file-width="128" height="16" src="//upload.wikimedia.org/wikipedia/commons/thumb/e/ec/Crystal_Clear_app_xmag.svg/16px-Crystal_Clear_app_xmag.svg.png" srcset="//upload.wikimedia.org/wikipedia/commons/thumb/e/ec/Crystal_Clear_app_xmag.svg/24px-Crystal_Clear_app_xmag.svg.png 1.5x, //upload.wikimedia.org/wikipedia/commons/thumb/e/ec/Crystal_Clear_app_xmag.svg/32px-Crystal_Clear_app_xmag.svg.png 2x" width="16"/> 이 부분의 본문은 <a href="/wiki/%EC%9D%B4%EC%98%A4%EB%8B%88%EC%95%84_%EB%B0%98%EB%9E%80" title="이오니아 반란">이오니아 반란</a>입니다.</div>
<p><a href="/wiki/%EA%B8%B0%EC%9B%90%EC%A0%84_513%EB%85%84" title="기원전 513년">기원전 513년</a> 다리우스 1세는 발칸반도 원정을 단행한다. 그는 <a class="mw-redirect" href="/wiki/%ED%8A%B8%EB%9D%BC%EC%BC%80" title="트라케">트라케</a>와 <a href="/wiki/%EB%A7%88%EC%BC%80%EB%8F%84%EB%8B%88%EC%95%84_%EC%99%95%EA%B5%AD" title="마케도니아 왕국">마케도니아 왕국</a>을 정복했고 마케도니아의 <a href="/wiki/%EC%95%84%EB%AF%BC%ED%83%80%EC%8A%A4_1%EC%84%B8" title="아민타스 1세">아민타스 1세</a>는 페르시아의 신하가 되었다. 이 원정에서 <a class="new" href="/w/index.php?title=%EB%B0%80%ED%8B%B0%EC%95%84%EB%8D%B0%EC%8A%A4(%E5%B0%8F)&amp;action=edit&amp;redlink=1" title="밀티아데스(小) (없는 문서)">밀티아데스</a>는 다리우스의 부하로 참전했는데 다리우스가 <a class="mw-redirect" href="/wiki/%EB%8B%A4%EB%89%B4%EB%B8%8C_%EA%B0%95" title="다뉴브 강">다뉴브 강</a>을 건너 <a class="mw-redirect" href="/wiki/%EC%8A%A4%ED%82%A4%ED%83%80%EC%9D%B4%EC%9D%B8" title="스키타이인">스키타이인</a>들의 지역으로 넘어갔을 때 다리를 끊어 함정에 빠트리고 페르시아를 공격하자는 그리스인들이 있었다. 그러나 이 계획은 실행되지 않았다.</p>
<p><a href="/wiki/%EA%B8%B0%EC%9B%90%EC%A0%84_499%EB%85%84" title="기원전 499년">기원전 499년</a> <a href="/wiki/%EB%B0%80%EB%A0%88%ED%86%A0%EC%8A%A4" title="밀레토스">밀레토스</a>의 <a href="/wiki/%EC%95%84%EB%A6%AC%EC%8A%A4%ED%83%80%EA%B3%A0%EB%9D%BC%EC%8A%A4" title="아리스타고라스">아리스타고라스</a>가 주도하여 <a href="/wiki/%EC%9D%B4%EC%98%A4%EB%8B%88%EC%95%84_%EB%B0%98%EB%9E%80" title="이오니아 반란">이오니아 반란</a>을 일으켰다. 소아이사의 그리스 도시들은 페르시아가 임명한 폭군을 축출하고 그리스 본토의 도시국가의 지원을 받았는데 <a class="mw-redirect" href="/wiki/%EC%95%84%ED%85%8C%EB%82%98%EC%9D%B4" title="아테나이">아테나이</a>는 배 20척을, <a href="/wiki/%EC%97%90%EB%A0%88%ED%8A%B8%EB%A6%AC%EC%95%84" title="에레트리아">에레트리아</a>는 5척을 지원했다. 이 반란은 소아시아의 해변과 퀴프로스에까지 번졌지만 그리스인들에게는 통일된 지도력이 없었고 비-그리스인들은 반란에 참여하지 않았기 때문에 오래가지 않았다. 페르시아는 3개의 부대를 보내어 이 반란을 진압했다. 해군은 이오니아의 그리스 해군에 의해 패배했지만 육군은 <a class="mw-redirect" href="/wiki/%ED%80%B4%ED%94%84%EB%A1%9C%EC%8A%A4" title="퀴프로스">퀴프로스</a>를 점령하고 카리아와 다른 여러 도시들의 항복을 받았다. 결국 이오니아는 고립되었고 페르시아의 대군에 밀려서 <a href="/wiki/%EA%B8%B0%EC%9B%90%EC%A0%84_494%EB%85%84" title="기원전 494년">기원전 494년</a> 함락되었다.</p>
<h2><span class="mw-headline" id=".EC.A0.84.EC.9F.81.EC.9D.98_.EA.B2.BD.EA.B3.BC">전쟁의 경과</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EA%B7%B8%EB%A6%AC%EC%8A%A4-%ED%8E%98%EB%A5%B4%EC%8B%9C%EC%95%84_%EC%A0%84%EC%9F%81&amp;action=edit&amp;section=2" title="부분 편집: 전쟁의 경과">원본 편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<h3><span class="mw-headline" id=".EB.8B.A4.EB.A6.AC.EC.9A.B0.EC.8A.A4.EC.9D.98_.EC.B9.A8.EA.B3.B5">다리우스의 침공</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EA%B7%B8%EB%A6%AC%EC%8A%A4-%ED%8E%98%EB%A5%B4%EC%8B%9C%EC%95%84_%EC%A0%84%EC%9F%81&amp;action=edit&amp;section=3" title="부분 편집: 다리우스의 침공">원본 편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p><a href="/wiki/%EA%B8%B0%EC%9B%90%EC%A0%84_493%EB%85%84" title="기원전 493년">기원전 493년</a>까지 마지막까지 남은 반란군도 페르시아의 함대에 의해 괴멸되었고 페르시아는 이 기회로 <a class="mw-redirect" href="/wiki/%EC%97%90%EA%B2%8C%ED%95%B4" title="에게해">에게해</a> 동쪽의 섬들까지 제국의 판도를 넓혀갔다. 다리우스는 반란이 진압된 후 <a href="/wiki/%EB%B0%80%EB%A0%88%ED%86%A0%EC%8A%A4" title="밀레토스">밀레토스</a>에서는 신전을 약탈하고 주민은 모두 노예로 만들거나 강제로 이주시켰지만 나머지 반란 도시들은 놀랍게도 하나도 건드리지 않았다. 그는 사위인 <a href="/wiki/%EB%A7%88%EB%A5%B4%EB%8F%84%EB%8B%88%EC%98%A4%EC%8A%A4" title="마르도니오스">마르도니오스</a>를 이 지역의 총독으로 임명해 뒷처리를 맡겼는데 지역별로 공평한 세금을 부과하고 민주주의도 전부는 아니지만 일부 부활시켰고 죄인들을 각자 자기 고향도시로 돌아가도록 하였다. 다리우스 1세의 이러한 유화책은 그리스 본토에 대한 일종의 선전으로 보인다.</p>
<p><a href="/wiki/%EA%B8%B0%EC%9B%90%EC%A0%84_492%EB%85%84" title="기원전 492년">기원전 492년</a> 봄 엄청난 대군이 마르도니오스의 지휘 아래 그리스 원정에 나섰다. 해군은 에게해 해안의 해안국가들을 휩쓸었고 육군은 <a class="mw-redirect" href="/wiki/%ED%97%AC%EB%A0%88%EC%8A%A4%ED%8F%B0%ED%86%A0%EC%8A%A4_%ED%95%B4%ED%98%91" title="헬레스폰토스 해협">헬레스폰토스 해협</a>을 건너 트라케와 마케도니아로 진군해 들어가서 복속시켰다. 그러나 <a href="/wiki/%EC%95%84%ED%86%A0%EC%8A%A4_%EC%82%B0" title="아토스 산">아토스 산</a>에서 페르시아 해군 선단은 폭풍을 만나 몰살했는데 헤로도토스에 따르면 300척의 배가 침몰하고 20,000명의 병사가 목숨을 잃었다고 한다. <a href="/wiki/%EB%A7%88%EB%A5%B4%EB%8F%84%EB%8B%88%EC%98%A4%EC%8A%A4" title="마르도니오스">마르도니오스</a>는 남은 군대를 소아시아로 퇴각시켰다. <a class="mw-redirect" href="/wiki/%ED%8A%B8%EB%9D%BC%EC%BC%80" title="트라케">트라케</a>의 부족인 브리간스는 퇴각하는 페르시아 군을 맞아 격렬하게 저항하여 마드로니우스에게 부상까지 입혔는데 결국 항복했다. 마르도니오스에 의한 이 원정이 과연 그리스 침공이었느냐에 대해서는 논란이 있다. 헤로도토스는 아테나이를 공격하기 위해서 라고 말하고 있지만 일단 트라케와 마케도니아를 다시 정복하여 이오니아 반란의 싹을 잘랐다는 점에서 더 이상의 진군이 필요했는가에 대한 의문이 있다.</p>

<p><a href="/wiki/%EA%B8%B0%EC%9B%90%EC%A0%84_490%EB%85%84" title="기원전 490년">기원전 490년</a> 페르시아는 다시 한번 원정군을 조직하여 이오니아 반란을 지원한 아테나이와 <a href="/wiki/%EC%97%90%EB%A0%88%ED%8A%B8%EB%A6%AC%EC%95%84" title="에레트리아">에레트리아</a>를 정벌하기 위해 떠났다. 정확한 원정군의 규모는 알려져 있지 않다. 헤로도토스는 600척의 겔리선이라고만 밝혔을 뿐이고 후대의 사가들의 기록은 200,000에서 500,000만까지 다양하게 기록하고 있다. 페르시아 원정군은 <a class="mw-disambig" href="/wiki/%EB%82%99%EC%86%8C%EC%8A%A4" title="낙소스">낙소스</a>로 상륙하여 별다른 저항없이 에레트리아까지 진군하여 6일만에 에레트리아를 함락시켰다. 도시는 폐허가 되었고 신전들은 파괴당했다. 헤로도토스에 따르면 대부분의 에레트리아 주민이 소개되어 끌려갔다고 하는데 10년후 벌어진 <a href="/wiki/%EC%82%B4%EB%9D%BC%EB%AF%B8%EC%8A%A4_%ED%95%B4%EC%A0%84" title="살라미스 해전">살라미스 해전</a>에 에레트리아에서 7척의 전함은 지원한 것을 보면 상당수 주민이 살아남아 도시를 재건한 것으로 보인다.</p>
<p>이후 페르시아군은 아테나이 공략에 나섰다. <a class="mw-redirect" href="/wiki/%EC%95%84%ED%8B%B0%EC%B9%B4" title="아티카">아티카</a> 지방 동쪽의 <a class="new" href="/w/index.php?title=%EB%A7%88%EB%9D%BC%ED%86%A4_%ED%8F%89%EC%95%BC&amp;action=edit&amp;redlink=1" title="마라톤 평야 (없는 문서)">마라톤 평야</a>에 상륙한 페르시아군은 아테나이를 향해 진군했다. 페르시아군의 침략에 아테나이는 <a href="/wiki/%EC%8A%A4%ED%8C%8C%EB%A5%B4%ED%83%80" title="스파르타">스파르타</a>에 전령을 보내어 지원군을 파견해 달라고 했으나 스파르타는 종교적인 이유로 지원군을 보내지 않았다고 한다. 아테나이는 자력으로 방위에 나섰다. 수적으로 우세한 페르시아군은 바다와 육지 양쪽에서 아테나이를 공략했는데 당시 아테나이의 병력은 약 10,000명정도로 추산된다. 육지에서 두 세력은 마라톤 평야에서 싸웠는데 아테나이군은 길게 보병을 배치하고 좌우 양날개에 최정예군을 배치했다. 전투가 시작되자 아테나이군의 중앙이 밀렸으나 좌우의 부대가 페르시아군을 협공하는 데 성공했다. 아테나이군의 전사자는 200명 남짓, 페르시아군은 6,400이 전사했다고 한다. 아테나이군은 수적인 열세에도 불구하고 이 전투에서 승리하고 바다쪽에서 아테나이 해군도 페르시아의 함대를 막는 데 성공하여 페르시아는 후퇴했다. 이 전투의 승전보를 가지고 아테나이로 달려온 한 병사에 대한 전설이 있는데 이것이 현대 <a href="/wiki/%EC%98%AC%EB%A6%BC%ED%94%BD" title="올림픽">올림픽</a>의 <a href="/wiki/%EB%A7%88%EB%9D%BC%ED%86%A4" title="마라톤">마라톤</a>의 유래가 된다.</p>
<p>마라톤 전투의 승리로 페르시아군이 더이상 무적이 아니고 그리스 국가들도 연합하여 싸우면 승리한다는 것을 알게 되었다. 페르시아에 굴복했던 많은 그리스 도시국가들이 아테나이와 스파르타의 편으로 돌아섰다. <a href="/wiki/%ED%82%A4%EB%A3%A8%EC%8A%A4_2%EC%84%B8" title="키루스 2세">키루스 2세</a> 이래로 정규군의 육전에서 한번도 패한적 없는 페르시아 육군은 마라톤 전투에서 유일하게 패배함으로 자존심의 상처를 입었고 이오니아의 그리스계 국가들에 대한 영향력의 약화를 우려할 처지가 되었다.</p>
<h3><span class="mw-headline" id=".EC.9D.B4.ED.9B.84_10.EB.85.84.EA.B0.84.EC.9D.98_.EC.A0.95.EC.84.B8">이후 10년간의 정세</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EA%B7%B8%EB%A6%AC%EC%8A%A4-%ED%8E%98%EB%A5%B4%EC%8B%9C%EC%95%84_%EC%A0%84%EC%9F%81&amp;action=edit&amp;section=4" title="부분 편집: 이후 10년간의 정세">원본 편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>마라톤 전투이후 10년간 그리스의 정세는 크게 변한 것은 없다. 마케도니아의 <a class="mw-redirect mw-disambig" href="/wiki/%EC%95%8C%EB%A0%89%EC%82%B0%EB%93%9C%EB%A1%9C%EC%8A%A4_1%EC%84%B8" title="알렉산드로스 1세">알렉산드로스 1세</a>는 이때 페르시아로부터 독립을 선언한 것으로 여겨진다. 그는 그리스의 <a class="mw-redirect" href="/wiki/%EA%B3%A0%EB%8C%80_%EC%98%AC%EB%A6%BC%ED%94%BD_%EA%B2%BD%EA%B8%B0" title="고대 올림픽 경기">고대 올림픽 경기</a>에 참가했는데 올림픽 경기에는 순수 그리스인들만 참가할 수 있었다. 스파르타에서는 <a class="mw-redirect" href="/wiki/%EB%A0%88%EC%98%A4%EB%8B%88%EB%8B%A4%EC%8A%A4" title="레오니다스">레오니다스</a>가 권좌에 올랐다. 아테나이는 다가오는 페르시아의 침략에 대비하기 위해 준비에 나섰다. <a href="/wiki/%EA%B8%B0%EC%9B%90%EC%A0%84_488%EB%85%84" title="기원전 488년">기원전 488년</a> <a class="mw-redirect" href="/wiki/%EB%8F%84%ED%8E%B8%EC%B6%94%EB%B0%A9%EC%A0%9C" title="도편추방제">도편추방제</a>가 처음으로 실시되어 페르시아에 항복을 주장하는 사람들을 추방시켰고 아테나이는 <a href="/wiki/%ED%85%8C%EB%AF%B8%EC%8A%A4%ED%86%A0%ED%81%B4%EB%A0%88%EC%8A%A4" title="테미스토클레스">테미스토클레스</a>가 이끄는 강경민중파와 <a href="/wiki/%EC%95%84%EB%A6%AC%EC%8A%A4%ED%8B%B0%EB%8D%B0%EC%8A%A4" title="아리스티데스">아리스티데스</a>의 온건귀족파로 나뉘었다. 페르시아의 위협에 맞서 군비증강에 따른 자력방위론을 주장하는 테미스토클레스는 도편추방제를 활용하여 정적인 아리스티데스를 추방하고 정권을 잡은 후 광산수익을 국고에 귀속시키고 군비증강에 힘썼다. 페르시아의 재침공이 점차 현실화 되자 북부의 일부 그리스 국가는 페르시아에 항복했하고 페르시아의 편으로 붙었지만 아테나이와 스파르타를 중심으로 몇몇 그리스 동맹국은 서로 반목을 그치고 페르시아의 위협 앞에 연합전선을 구축했다.</p>
<p>한편 페르시아는 그리스의 재침공을 위해 준비에 나섰으나 때마침 <a href="/wiki/%EB%B0%94%EB%B9%8C%EB%A1%9C%EB%8B%88%EC%95%84" title="바빌로니아">바빌로니아</a>와 <a href="/wiki/%EC%9D%B4%EC%A7%91%ED%8A%B8" title="이집트">이집트</a>에서 일어난 반란때문에 준비가 늦어졌고 <a href="/wiki/%EA%B8%B0%EC%9B%90%EC%A0%84_485%EB%85%84" title="기원전 485년">기원전 485년</a> 다리우스 1세가 죽고 그의 아들 <a href="/wiki/%ED%81%AC%EC%84%B8%EB%A5%B4%ED%81%AC%EC%84%B8%EC%8A%A4_1%EC%84%B8" title="크세르크세스 1세">크세르크세스 1세</a>가 제위에 올랐다. 크세르크세스 1세는 아버지의 뒤를 이어 그리스 원정을 준비했고 대략 4년에 걸친 전쟁 준비 이후 그리스 공략에 나섰다.</p>
<h3><span class="mw-headline" id=".ED.81.AC.EC.84.B8.EB.A5.B4.ED.81.AC.EC.84.B8.EC.8A.A4.EC.9D.98_.EC.B9.A8.EA.B3.B5">크세르크세스의 침공</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EA%B7%B8%EB%A6%AC%EC%8A%A4-%ED%8E%98%EB%A5%B4%EC%8B%9C%EC%95%84_%EC%A0%84%EC%9F%81&amp;action=edit&amp;section=5" title="부분 편집: 크세르크세스의 침공">원본 편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>기원전 480년 크세르크세스가 친히 이끄는 페르시아의 대군은 그리스 원정에 나섰다. 원정군의 병력 규모에 대해서는 많은 논란이 있다. 헤로도토스의 기록은 보병만 170만 명, 기병 8만 명, 그리스의 페르시아 동맹군 32만 명 등 총 260만 명 이상의 규모라고 적고 있으나, 후대의 사가들은 80만 명이라고 적었고, 현대의 연구자들은 9만에서 30만 명으로 본다.</p>
<div class="rellink relarticle mainarticle"><img alt="" data-file-height="128" data-file-width="128" height="16" src="//upload.wikimedia.org/wikipedia/commons/thumb/e/ec/Crystal_Clear_app_xmag.svg/16px-Crystal_Clear_app_xmag.svg.png" srcset="//upload.wikimedia.org/wikipedia/commons/thumb/e/ec/Crystal_Clear_app_xmag.svg/24px-Crystal_Clear_app_xmag.svg.png 1.5x, //upload.wikimedia.org/wikipedia/commons/thumb/e/ec/Crystal_Clear_app_xmag.svg/32px-Crystal_Clear_app_xmag.svg.png 2x" width="16"/> 이 부분의 본문은 <a class="mw-redirect" href="/wiki/%ED%85%8C%EB%A5%B4%EB%AA%A8%ED%93%94%EB%9D%BC%EC%9D%B4_%EC%A0%84%ED%88%AC" title="테르모퓔라이 전투">테르모퓔라이 전투</a>, <a href="/wiki/%EC%82%B4%EB%9D%BC%EB%AF%B8%EC%8A%A4_%ED%95%B4%EC%A0%84" title="살라미스 해전">살라미스 해전</a>입니다.</div>
<p>크세르크세스는 <a href="/wiki/%EB%8B%A4%EB%A5%B4%EB%8B%A4%EB%84%AC%EC%8A%A4_%ED%95%B4%ED%98%91" title="다르다넬스 해협">헬레스폰토스 해협</a>에 이집트와 페니키아의 군함을 연결하여 두개의 다리를 만들라고 명령하는데 폭풍으로 유실되었으나 다시 지어졌고 페르시아의 대군이 모두 건너는 데 7일이 걸렸다고 한다. 페르시아 군은 <a class="mw-redirect" href="/wiki/%ED%8A%B8%EB%9D%BC%EC%BC%80" title="트라케">트라케</a>와 마케도니아를 지나 남하했고 해군도 해안선을 따라 육군과 보조를 맞추어 남하했다. 그리스 측은 아테나이와 스파르타를 중심으로 연합군을 편성했고 각각 해군과 육군을 지휘했다. 그리스 연합군의 지휘관인 아테나이의 <a href="/wiki/%ED%85%8C%EB%AF%B8%EC%8A%A4%ED%86%A0%ED%81%B4%EB%A0%88%EC%8A%A4" title="테미스토클레스">테미스토클레스</a>는 넓은 <a href="/wiki/%ED%85%8C%EC%82%B4%EB%A6%AC%EC%95%84" title="테살리아">테살리아</a>의 평원에서의 전투가 불리하다고 판단하고 최촉의 방어선을 중부지방의 좁고 험한 산악지역인 테르모퓔라이에서 세우기로 하였다. 이 방어선에는 스파르타의 <a class="mw-redirect" href="/wiki/%EB%A0%88%EC%98%A4%EB%8B%88%EB%8B%A4%EC%8A%A4" title="레오니다스">레오니다스</a>가 이끄는 300명의 스파르타 전사와 그리스 각지에서 모인 4,000명의 병사가 투입되었고 테미스토클레스가 이끄는 해군은 에우보이아 곶에서 페르시아군을 맞서기로 결정했다.</p>
<p>페르시아의 해군은 폭풍을 만나 손실을 입었고 <a href="/wiki/%EC%95%84%EB%A5%B4%ED%85%8C%EB%AF%B8%EC%8B%9C%EC%98%A8" title="아르테미시온">아르테미시온</a>에서 그리스 해군에게 저지당했고 페르시아 육군은 테르모퓔라이에서 스파르타 300명의 병사과 그리스군의 격렬한 저항을 받았다. 페르시아 대군은 협곡에서 2일 동안 저지당하면서 많은 손실을 보았는데 3일째 되는 날, 그리스의 한 배신자가 협곡을 우회하는 샛길을 페르시아군에게 알려주어 페르시아군은 테르모퓔라이 정면돌파를 피하고 우회공격에 나서게 되었다. 이에 스파르타의 왕 레오니다스는 스파르타의 300명과 700명의 테스피아인, 포로로 잡혀있던 400명의 테베인, 900명의 헬롯을 제외한 다른 그리스인들을 철수시키고 그곳에서 전원이 전사하였다.</p>
<p>테르모퓔라이에서 승리한 페르시아군은 그리스 본토를 유린하며 <a href="/wiki/%EA%B8%B0%EC%9B%90%EC%A0%84_480%EB%85%84" title="기원전 480년">기원전 480년</a> 9월경 아테나이에 입성하는데 아테나이는 이미 소개되어 비어 있었다. 이것은 아테나이의 테미스토클레스의 작전으로 수도를 방어하기보다는 비워두고 시민과 병력을 살라미스 섬으로 이주시킨 것이다. 페르시아의 해군의 큰 군선은 좁은 살라미스의 바다에서 기동력을 잃었고 상대적으로 가벼운 그리스 갤리선에 의해 괴멸당했고 퇴각해야했다. 크세르크세스는 마침 칼키디키에서 일어난 반란으로 그리스에서 고립되는 것을 우려하여 대부분의 병사를 소아시아로 퇴각시키고 <a class="mw-redirect" href="/wiki/%EB%A7%88%EB%A5%B4%EB%8F%84%EB%8B%88%EC%9A%B0%EC%8A%A4" title="마르도니우스">마르도니우스</a>를 그리스 본토에 남겨주었다.</p>
<div class="rellink relarticle mainarticle"><img alt="" data-file-height="128" data-file-width="128" height="16" src="//upload.wikimedia.org/wikipedia/commons/thumb/e/ec/Crystal_Clear_app_xmag.svg/16px-Crystal_Clear_app_xmag.svg.png" srcset="//upload.wikimedia.org/wikipedia/commons/thumb/e/ec/Crystal_Clear_app_xmag.svg/24px-Crystal_Clear_app_xmag.svg.png 1.5x, //upload.wikimedia.org/wikipedia/commons/thumb/e/ec/Crystal_Clear_app_xmag.svg/32px-Crystal_Clear_app_xmag.svg.png 2x" width="16"/> 이 부분의 본문은 <a class="new" href="/w/index.php?title=%ED%94%8C%EB%9D%BC%ED%83%80%EC%9D%B4%EC%95%84%EC%9D%B4_%EC%A0%84%ED%88%AC&amp;action=edit&amp;redlink=1" title="플라타이아이 전투 (없는 문서)">플라타이아이 전투</a>입니다.</div>
<p>이듬해인 <a href="/wiki/%EA%B8%B0%EC%9B%90%EC%A0%84_479%EB%85%84" title="기원전 479년">기원전 479년</a> 봄, 마르도니우스는 마케도니아의 알레산드로스 1세를 보내어 그리스와 협정을 맺으려 하였으나 그리스인들은 이를 거부했다. 아테나이는 이번에도 시가지를 비우고 해상에서 승부를 거는 작전을 세웠고 스파르타의 <a class="mw-disambig" href="/wiki/%ED%8C%8C%EC%9A%B0%EC%82%AC%EB%8B%88%EC%95%84%EC%8A%A4" title="파우사니아스">파우사니아스</a> 왕이 지휘하는 5만 명의 그리스 연합군은 테바이 근처의 플라타이아 평원에서 페르시아 군과 전투를 벌였다. 전투는 혼전이었지만 스파르타와 테게아군의 용맹으로 대세는 그리스쪽으로 기울었고 결국 마르도니우스는 전사하고 페르시아군은 퇴각했다. 페르시아군은 퇴각하면서 마케도니아의 알렉산드로스 1세의 매복에 걸려 대부분이 소아시아로 건너가지 못하고 죽었다. 한편 바다쪽에서 해군도 플라타이아 전투가 벌어진 날 페르시아의 해군을 섬멸했다.</p>
<p>같은해 그리스는 반격에 나섰는데 아테나이의 해군을 주축으로한 그리스 연합함대는 에게해를 지나 소아시아로 침공했고 그리스계 도시국가를 해방시켰다. 이후 30년간 크고 작은 페르시아와 그리스사이에 크고 작은 전투가 벌어졌는데 아테나이의 주도로 새로 결성된 <a href="/wiki/%EB%8D%B8%EB%A1%9C%EC%8A%A4_%EB%8F%99%EB%A7%B9" title="델로스 동맹">델로스 동맹</a>은 아나톨리아 해안에 있는 이오니아계 도시국가들을 페르시아의 지배로부터 해방하기 위해 계속 공세를 펼쳤는데 대체로 성공을 거두었다. <a href="/wiki/%EA%B8%B0%EC%9B%90%EC%A0%84_448%EB%85%84" title="기원전 448년">기원전 448년</a>경에는 마침내 아테나이와 그 동맹국 및 페르시아 왕 <a class="new" href="/w/index.php?title=%EC%95%84%EB%A5%B4%ED%83%80%ED%81%AC%EC%84%B8%EB%A5%B4%ED%81%AC%EC%84%B8%EC%8A%A4_1%EC%84%B8&amp;action=edit&amp;redlink=1" title="아르타크세르크세스 1세 (없는 문서)">아르타크세르크세스 1세</a> 사이에 협정이 맺어졌다.</p>
