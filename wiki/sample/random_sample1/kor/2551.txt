<p><b>그리스 왕국</b>(<span style="font-size: smaller;"><a href="/wiki/%EA%B7%B8%EB%A6%AC%EC%8A%A4%EC%96%B4" title="그리스어">그리스어</a>:</span> <span lang="el" xml:lang="el">Βασίλειον της Ελλάδος</span> <small>바실리온 티스 엘라도스<sup>[<a href="/wiki/%EC%9C%84%ED%82%A4%EB%B0%B1%EA%B3%BC:%EC%99%B8%EB%9E%98%EC%96%B4_%ED%91%9C%EA%B8%B0%EB%B2%95/%EA%B7%B8%EB%A6%AC%EC%8A%A4%EC%96%B4_%EC%8B%9C%EC%95%88" title="위키백과:외래어 표기법/그리스어 시안">*</a>]</sup></small>)은 <a href="/wiki/1832%EB%85%84" title="1832년">1832년</a> 유럽 열강(<a href="/wiki/%EB%8C%80%EC%98%81_%EC%A0%9C%EA%B5%AD" title="대영 제국">대영 제국</a>, <a href="/wiki/%ED%94%84%EB%9E%91%EC%8A%A4" title="프랑스">프랑스</a>, <a href="/wiki/%EB%9F%AC%EC%8B%9C%EC%95%84_%EC%A0%9C%EA%B5%AD" title="러시아 제국">러시아 제국</a>)이 <a href="/wiki/%EB%9F%B0%EB%8D%98_%ED%9A%8C%EC%9D%98_(1832%EB%85%84)" title="런던 회의 (1832년)">런던 회의</a>로 <a href="/wiki/%EA%B7%B8%EB%A6%AC%EC%8A%A4" title="그리스">그리스</a>에 세운 나라이다. 그리스 왕국은 <a class="new" href="/w/index.php?title=%EC%BD%98%EC%8A%A4%ED%83%84%ED%8B%B0%EB%85%B8%ED%8F%B4%EB%A6%AC%EC%8A%A4_%EC%A1%B0%EC%95%BD_(1832%EB%85%84)&amp;action=edit&amp;redlink=1" title="콘스탄티노폴리스 조약 (1832년) (없는 문서)">콘스탄티노폴리스 조약</a>에서 <a href="/wiki/%EC%98%A4%EC%8A%A4%EB%A7%8C_%EC%A0%9C%EA%B5%AD" title="오스만 제국">오스만 제국</a>에서 완전히 독립을 확보하여 국제적으로 승인받았으며, 15세기 중반 <a href="/wiki/%EB%B9%84%EC%9E%94%ED%8B%B0%EC%9B%80_%EC%A0%9C%EA%B5%AD" title="비잔티움 제국">비잔티움 제국</a>이 멸망한 이래 최초의 완전한 그리스의 독립 국가를 이루었다. 왕국은 <a href="/wiki/%EA%B7%B8%EB%A6%AC%EC%8A%A4_%EB%8F%85%EB%A6%BD_%EC%A0%84%EC%9F%81" title="그리스 독립 전쟁">그리스 독립 전쟁</a> 당시 <a href="/wiki/%EA%B7%B8%EB%A6%AC%EC%8A%A4_%EC%A0%9C1%EA%B3%B5%ED%99%94%EA%B5%AD" title="그리스 제1공화국">그리스 과도 정부</a>를 계승하였으며, <a href="/wiki/1924%EB%85%84" title="1924년">1924년</a> 왕정이 폐지되었다가(<a href="/wiki/%EA%B7%B8%EB%A6%AC%EC%8A%A4_%EC%A0%9C2%EA%B3%B5%ED%99%94%EA%B5%AD" title="그리스 제2공화국">그리스 제2공화국</a>) 1935년에 복고되었으며, <a href="/wiki/%EA%B7%B8%EB%A6%AC%EC%8A%A4_%EA%B5%B0%EC%82%AC_%EC%A0%95%EA%B6%8C" title="그리스 군사 정권">군사 정권의 7년 독재</a>를 거친 후 <a href="/wiki/1974%EB%85%84" title="1974년">1974년</a>에 다시 폐지되어 오늘날의 <a href="/wiki/%EA%B7%B8%EB%A6%AC%EC%8A%A4" title="그리스">그리스 제3공화국</a>으로 이어지고 있다. 1832년부터 <a href="/wiki/1863%EB%85%84" title="1863년">1863년</a>까지는 <a href="/wiki/%EB%8F%85%EC%9D%BC" title="독일">독일</a>계인 <a class="mw-redirect" href="/wiki/%EB%B9%84%ED%85%94%EC%8A%A4%EB%B0%94%ED%9D%90_%EC%99%95%EA%B0%80" title="비텔스바흐 왕가">비텔스바흐 왕가</a>의 <a href="/wiki/%EC%98%A4%ED%86%A4_(%EA%B7%B8%EB%A6%AC%EC%8A%A4)" title="오톤 (그리스)">오톤</a>이 왕국을 다스렸으며, 그 이후부터 1974년까지는 <a href="/wiki/%EB%8D%B4%EB%A7%88%ED%81%AC" title="덴마크">덴마크</a>계 <a class="mw-redirect" href="/wiki/%EA%B8%80%EB%A4%BD%EC%8A%A4%EB%B6%80%EB%A5%B4%ED%81%AC_%EC%99%95%EA%B0%80" title="글뤽스부르크 왕가">글뤽스부르크 왕가</a> 출신이 왕이 되었다.</p>

<h2><span class="mw-headline" id=".EB.B9.84.ED.85.94.EC.8A.A4.EB.B0.94.ED.9D.90_.EC.99.95.EA.B0.80">비텔스바흐 왕가</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EA%B7%B8%EB%A6%AC%EC%8A%A4_%EC%99%95%EA%B5%AD&amp;action=edit&amp;section=1" title="부분 편집: 비텔스바흐 왕가">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p><a href="/wiki/1821%EB%85%84" title="1821년">1821년</a>부터 <a href="/wiki/%EA%B7%B8%EB%A6%AC%EC%8A%A4%EC%9D%B8" title="그리스인">그리스인</a>들은 <a href="/wiki/%EC%98%A4%EC%8A%A4%EB%A7%8C_%EC%A0%9C%EA%B5%AD" title="오스만 제국">오스만 제국</a>에 대항하여 봉기하였으며, 1829년까지 독립 투쟁을 하였다. 그리스의 독립 국가 정부로는 1828년 <a href="/wiki/%EC%9A%94%EC%95%88%EB%8B%88%EC%8A%A4_%EC%B9%B4%ED%8F%AC%EB%94%94%EC%8A%A4%ED%8A%B8%EB%A6%AC%EC%95%84%EC%8A%A4" title="요안니스 카포디스트리아스">요안니스 카포디스트리아스</a> 백작의 과도 정부가 있었으나, 1831년 그가 암살당하면서 그리스는 내전 상태에 빠졌다. 이 때 열강이 간섭하여 그리스에 왕국을 세우기로 결정하였다. <a href="/wiki/1832%EB%85%84" title="1832년">1832년</a> <a href="/wiki/%EB%9F%B0%EB%8D%98_%ED%9A%8C%EC%9D%98_(1832%EB%85%84)" title="런던 회의 (1832년)">런던 회의 (1832년)</a>에서 <a href="/wiki/%EC%98%81%EA%B5%AD" title="영국">영국</a>, <a href="/wiki/%ED%94%84%EB%9E%91%EC%8A%A4" title="프랑스">프랑스</a>, <a href="/wiki/%EB%9F%AC%EC%8B%9C%EC%95%84_%EC%A0%9C%EA%B5%AD" title="러시아 제국">러시아 제국</a>은 당시 17세였던 <a class="mw-redirect" href="/wiki/%EB%B9%84%ED%85%94%EC%8A%A4%EB%B0%94%ED%9D%90_%EC%99%95%EA%B0%80" title="비텔스바흐 왕가">비텔스바흐 왕가</a>의 <a class="mw-redirect" href="/wiki/%EB%B0%94%EC%9D%B4%EC%97%90%EB%A5%B8" title="바이에른">바이에른</a> 공 <a href="/wiki/%EC%98%A4%ED%86%A4_(%EA%B7%B8%EB%A6%AC%EC%8A%A4)" title="오톤 (그리스)">오톤</a>(오토)을 그리스 왕국의 초대 국왕으로 세웠다. 그리스에 올 당시 오톤은 성년이 아니었기 때문에 1835년까지 섭정 회의가 그의 이름으로 나라를 다스렸다. 1835년에 오톤은 친정을 시작하면서 <a class="mw-redirect" href="/wiki/%EB%B0%94%EC%9D%B4%EC%97%90%EB%A5%B8" title="바이에른">바이에른</a> 사람을 고문으로 선택하여 국가 평의회 의장으로 임명하였다. 때때로 국왕이 친히 평의회 의장을 맡기도 하였다.</p>
<h2><span class="mw-headline" id="9.EC.9B.94_3.EC.9D.BC_.ED.98.81.EB.AA.85">9월 3일 혁명</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EA%B7%B8%EB%A6%AC%EC%8A%A4_%EC%99%95%EA%B5%AD&amp;action=edit&amp;section=2" title="부분 편집: 9월 3일 혁명">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>1843년 오톤과 "바이에른인의 지배"에 대한 불만이 극에 달하여 <a href="/wiki/%EA%B7%B8%EB%A6%AC%EC%8A%A4%EC%9D%B8" title="그리스인">그리스인</a>들이 헌법 제정을 요구하기 시작하였다. 처음에 오톤은 헌법 승인을 거부하였는데, 독일군이 왕국에서 철수하자마자 곧 <a class="mw-redirect" href="/wiki/%EA%B5%B0%EC%82%AC_%EC%A0%95%EB%B3%80" title="군사 정변">군사 정변</a>이 일어났다. 1843년 9월 3일 <a class="new" href="/w/index.php?title=%EB%94%94%EB%AF%B8%ED%8A%B8%EB%A6%AC%EC%98%A4%EC%8A%A4_%EC%B9%BC%EB%A0%88%EB%A5%B4%EC%9D%B4%EC%8A%A4&amp;action=edit&amp;redlink=1" title="디미트리오스 칼레르이스 (없는 문서)">디미트리오스 칼레르이스</a> 대령과 존경받은 독립 투사 <a class="new" href="/w/index.php?title=%EC%9A%94%EC%95%88%EB%8B%88%EC%8A%A4_%EB%A7%88%ED%81%AC%EB%A6%AC%EC%96%80%EB%8B%88%EC%8A%A4&amp;action=edit&amp;redlink=1" title="요안니스 마크리얀니스 (없는 문서)">요안니스 마크리얀니스</a>가 이끄는 보병대가 아테네의 궁궐 앞 광장에 집결하였다. 반란군은 왕이 헌법을 승인하는 데 동의할 때까지 해산하기를 거부하였으며, 국가 평의회에 <a href="/wiki/%EA%B7%B8%EB%A6%AC%EC%8A%A4%EC%9D%B8" title="그리스인">그리스인</a>을 들여 보내고 상설 국회를 소집하며, 오톤이 직접 반란의 지도자들을 용서하도록 요구하였다. 오톤은 군대의 압력에 굴복하여 대중의 요구를 들어주었다.</p>

<p>1862년 오톤이 폐위되자 당시 17세였던 <a href="/wiki/%EB%8D%B4%EB%A7%88%ED%81%AC" title="덴마크">덴마크</a>의 빌헬름 공이 <a class="mw-redirect" href="/wiki/%EC%9E%85%ED%97%8C_%EA%B5%B0%EC%A3%BC%EC%A0%9C" title="입헌 군주제">입헌 군주</a>로 선출되었는데, 이 사람이 <a href="/wiki/%EC%9A%94%EB%A5%B4%EC%9A%94%EC%8A%A4_1%EC%84%B8" title="요르요스 1세">요르요스 1세</a>이다. 그는 50년간 재위하였으며, 치세 동안 그리스의 영토는 확대되고(그가 즉위하자 영국은 <a href="/wiki/%EC%9D%B4%EC%98%A4%EB%8B%88%EC%95%84_%EC%A0%9C%EB%8F%84" title="이오니아 제도">이오니아 제도</a>를 그리스에 할양하였다), 경제 발전을 이루었으며, 왕실과는 상관없이 선거에서 가장 많이 득표한 정당 지도자가 정부의 수반이 된다는 생각이 받아들여졌다. 그러나, 요르요스 1세도 정치에 적극적으로 활동하였다. 요르요스는 <a href="/wiki/%EC%A0%9C1%EC%B0%A8_%EB%B0%9C%EC%B9%B8_%EC%A0%84%EC%9F%81" title="제1차 발칸 전쟁">제1차 발칸 전쟁</a>에서 그리스가 승리하여 병합한 지 얼마 안 된 <a href="/wiki/%ED%85%8C%EC%82%B4%EB%A1%9C%EB%8B%88%ED%82%A4" title="테살로니키">테살로니키</a>에서 1913년에 암살되었다.</p>
<p>요르요스 1세의 뒤를 이어 아들 <a href="/wiki/%EC%BD%98%EC%8A%A4%ED%83%84%ED%8B%B0%EB%85%B8%EC%8A%A4_1%EC%84%B8" title="콘스탄티노스 1세">콘스탄티노스 1세</a>가 즉위하였다. 그는 친히 군사령관으로 활약하여 그리스의 영토를 넓혔다. 그는 <a href="/wiki/%EB%8F%85%EC%9D%BC" title="독일">독일</a>에서 교육받았고, 독일 황제의 딸 <a class="new" href="/w/index.php?title=%EC%86%8C%ED%94%BC%EC%95%84_(%ED%94%84%EB%A1%9C%EC%9D%B4%EC%84%BC)&amp;action=edit&amp;redlink=1" title="소피아 (프로이센) (없는 문서)">소피아</a>와 혼인하였다. 콘스탄티노스는 친독일파로 여겨졌으며, <a href="/wiki/%EC%82%BC%EA%B5%AD_%ED%98%91%EC%83%81" title="삼국 협상">삼국 협상</a>을 지지하는 <a href="/wiki/%EC%97%98%EB%A0%88%ED%94%84%ED%85%8C%EB%A6%AC%EC%98%A4%EC%8A%A4_%EB%B2%A0%EB%8B%88%EC%A0%A4%EB%A1%9C%EC%8A%A4" title="엘레프테리오스 베니젤로스">엘레프테리오스 베니젤로스</a>와 대립하였다. <a href="/wiki/%EC%A0%9C1%EC%B0%A8_%EC%84%B8%EA%B3%84_%EB%8C%80%EC%A0%84" title="제1차 세계 대전">제1차 세계 대전</a>에서 콘스탄티노스는 그리스가 중립을 지키게끔 공공연히 노력하면서 이러한 시각을 뒷받침하였다. 연합국은 엘레프테리오스 베니젤로스를 지원하였으며, 이후 <a href="/wiki/%EA%B5%AD%EB%A1%A0_%EB%B6%84%EC%97%B4" title="국론 분열">국론 분열</a>의 시대에 <a href="/wiki/%EC%95%84%ED%85%8C%EB%84%A4" title="아테네">아테네</a>와 <a href="/wiki/%ED%85%8C%EC%82%B4%EB%A1%9C%EB%8B%88%ED%82%A4" title="테살로니키">테살로니키</a>에 각각 두 정부가 수립되었다. 그리스는 삼국 협상에 가입하였으며, 콘스탄티노스 1세는 1917년 퇴위당하고 아들 <a class="mw-redirect" href="/wiki/%EC%95%8C%EB%A0%89%EC%82%B0%EB%93%9C%EB%A1%9C%EC%8A%A4_1%EC%84%B8_(%EA%B7%B8%EB%A6%AC%EC%8A%A4_%EC%99%95%EA%B5%AD)" title="알렉산드로스 1세 (그리스 왕국)">알렉산드로스 1세</a>가 즉위하였다. 그리스는 승전국 편에 선 덕분에 <a href="/wiki/%EC%8A%A4%EB%AF%B8%EB%A5%B4%EB%82%98" title="스미르나">스미르나</a>를 비롯한 <a class="mw-redirect" href="/wiki/%EC%86%8C%EC%95%84%EC%8B%9C%EC%95%84" title="소아시아">소아시아</a> 영토를 얻었다. 그러나, 1920년 알렉산드로스 1세는 원숭이에 물려 죽었고, 아버지 콘스탄티노스 1세가 복위하였다가 <a class="mw-redirect" href="/wiki/%EA%B7%B8%EB%A6%AC%EC%8A%A4-%ED%84%B0%ED%82%A4_%EC%A0%84%EC%9F%81_(1919%EB%85%84~1922%EB%85%84)" title="그리스-터키 전쟁 (1919년~1922년)">그리스-터키 전쟁 (1919년~1922년)</a>에서 그리스가 패배하면서 콘스탄티노스 1세는 다시 폐위되어 <a href="/wiki/%EC%8B%9C%EC%B9%A0%EB%A6%AC%EC%95%84" title="시칠리아">시칠리아</a>에서 망명 중에 세상을 떠났다.</p>
<p>콘스탄티노스의 뒤를 이어 장자 <a href="/wiki/%EC%9A%94%EB%A5%B4%EC%9A%94%EC%8A%A4_2%EC%84%B8" title="요르요스 2세">요르요스 2세</a>가 즉위하였으나, 1924년 <a href="/wiki/%EA%B7%B8%EB%A6%AC%EC%8A%A4_%EC%A0%9C2%EA%B3%B5%ED%99%94%EA%B5%AD" title="그리스 제2공화국">그리스 제2공화국</a>이 선포되면서 나라를 떠나야 했다. 1935년 <a href="/wiki/%EC%9A%94%EB%A5%B4%EC%9A%94%EC%8A%A4_%EC%BD%98%EB%94%9C%EB%A6%AC%EC%8A%A4" title="요르요스 콘딜리스">요르요스 콘딜리스</a>가 군사 정변을 일으켜 공화제를 폐지하였으며, <a class="new" href="/w/index.php?title=%EA%B7%B8%EB%A6%AC%EC%8A%A4_%EA%B5%AD%EB%AF%BC%ED%88%AC%ED%91%9C_(1935%EB%85%84)&amp;action=edit&amp;redlink=1" title="그리스 국민투표 (1935년) (없는 문서)">국민 투표</a>로 왕정 복고를 승인받았다. 요르요스 2세는 다시 나라로 돌아왔으며, <a href="/wiki/8%EC%9B%94_4%EC%9D%BC_%EC%B2%B4%EC%A0%9C" title="8월 4일 체제">메탁사스의 독재 정권</a>을 적극 지지하였다. 1941년 그리스는 <a href="/wiki/%EA%B7%B8%EB%A6%AC%EC%8A%A4_%EC%A0%84%ED%88%AC" title="그리스 전투">독일의 침략</a>을 받아 그는 정부와 함께 이집트로 망명하였다. 요르요스는 1946년에야 그리스로 귀환하였으며, 1947년에 죽을 때까지 재위하였다.</p>
<p>그 뒤 요르요스 2세의 막내 동생 <a class="mw-redirect" href="/wiki/%ED%8C%8C%EB%B8%94%EB%A1%9C%EC%8A%A4_1%EC%84%B8" title="파블로스 1세">파블로스 1세</a>가 즉위하여 1947년부터 1964년까지 재위하였다. 아들 <a href="/wiki/%EC%BD%98%EC%8A%A4%ED%83%84%ED%8B%B0%EB%85%B8%EC%8A%A4_2%EC%84%B8" title="콘스탄티노스 2세">콘스탄티노스 2세</a>가 뒤를 이었으나, 1967년 12월 <a class="mw-redirect" href="/wiki/%EA%B7%B8%EB%A6%AC%EC%8A%A4_%EA%B5%B0%EC%82%AC_%EC%A0%95%EA%B6%8C_(1967~1974)" title="그리스 군사 정권 (1967~1974)">그리스 군사 정권</a>에 대항하는 역쿠데타가 실패한 뒤 망명하게 된다. 군사 정권은 국왕 대신 불법으로 섭정을 임명하였으며, 1973년에 결국 신중하게 조작한 <a class="new" href="/w/index.php?title=%EA%B7%B8%EB%A6%AC%EC%8A%A4_%EA%B5%AD%EB%AF%BC%ED%88%AC%ED%91%9C_(1973%EB%85%84)&amp;action=edit&amp;redlink=1" title="그리스 국민투표 (1973년) (없는 문서)">국민 투표</a>로 군주정을 불법적으로 폐지하였다. 1973년 6월 1일 <a href="/wiki/%EC%9A%94%EB%A5%B4%EC%9A%94%EC%8A%A4_%ED%8C%8C%ED%8C%8C%EB%8F%84%ED%92%80%EB%A1%9C%EC%8A%A4" title="요르요스 파파도풀로스">요르요스 파파도풀로스</a>는 <a href="/wiki/%EA%B7%B8%EB%A6%AC%EC%8A%A4%EC%9D%98_%EB%8C%80%ED%86%B5%EB%A0%B9" title="그리스의 대통령">그리스의 대통령</a>으로 취임하였다.</p>
<p>군사 정권이 사실상 종식된 후에도 콘스탄티노스 2세는 복위되지 못하였다. <a href="/wiki/1974%EB%85%84" title="1974년">1974년</a> 왕정 복고를 놓고 <a class="new" href="/w/index.php?title=%EA%B7%B8%EB%A6%AC%EC%8A%A4_%EA%B5%AD%EB%AF%BC%ED%88%AC%ED%91%9C_(1974%EB%85%84)&amp;action=edit&amp;redlink=1" title="그리스 국민투표 (1974년) (없는 문서)">국민 투표</a>가 시행되었는데, 여기서 그리스인 69%가 군주정 폐지에 찬성하였다.</p>
