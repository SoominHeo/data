그리스의 불을 사용하는 비잔티움 해군(<a href="/wiki/11%EC%84%B8%EA%B8%B0" title="11세기">11세기</a>)</div>

<p><b>그리스 화약</b> 또는 <b>그리스의 불</b>(<span style="font-size: smaller;"><a href="/wiki/%EA%B7%B8%EB%A6%AC%EC%8A%A4%EC%96%B4" title="그리스어">그리스어</a>:</span> <span lang="el" xml:lang="el">υγρόν πυρ</span>)은 <a href="/wiki/%EB%B9%84%EC%9E%94%ED%8B%B0%EC%9B%80_%EC%A0%9C%EA%B5%AD" title="비잔티움 제국">비잔티움 제국</a>의 군대에서 사용하던 화기를 말한다. 이 <a href="/wiki/%EB%B6%88" title="불">불</a>은 <a href="/wiki/%EB%AC%BC" title="물">물</a>로 잘 꺼지지 않았고 수면에서도 불이 계속 타오르는 특성 때문에 비잔티움 해군에 의거해 해전에서 주로 사용되었다. 비잔티움 제국이 다수히 외침당하면서도 약 천년간 굿굿히 버틴 이유 중에는 바로 이 그리스의 불이라는 훌륭한 병기에 힘입은 바가 크다.</p>
<p>비잔티움 제국 역사가 <a class="new" href="/w/index.php?title=%ED%85%8C%EC%98%A4%ED%8C%8C%EB%84%A4%EC%8A%A4&amp;action=edit&amp;redlink=1" title="테오파네스 (없는 문서)">테오파네스</a>는 이 화기를 처음 만든 것은 670년경 <a href="/wiki/%EC%9C%A0%EB%8C%80" title="유대">유대</a>의 <a class="mw-disambig" href="/wiki/%ED%97%AC%EB%A6%AC%EC%98%A4%ED%8F%B4%EB%A6%AC%EC%8A%A4" title="헬리오폴리스">헬리오폴리스</a> 출신 건축가이자 화학자이고 기술자인 칼리키누스라고 말한다. 이 화기를 만드는 비법은 비잔티움 제국에서 일급비밀로 간주되었고 워낙에 비밀이 철통같이 잘 유지되었던 탓에 현재까지도 그 정확한 성분을 알지 못한다.</p>
<p><a href="/wiki/10%EC%84%B8%EA%B8%B0" title="10세기">10세기</a> 작가 마르쿠스 그라이쿠스는 그리스 화약 제조 방법을 “순수한 황, 주석, 사르코콜라[페르시아 고무], 역청, 녹인 초석, 석유, 송진을 준비한다. 이것들을 끓이고서 밧줄에 스며들게끔 해 불을 붙인다. 이 불은 소변, 식초, 모래로만 끌 수 있다.”라고 설명했다.</p>
<p>그리스 화약은 폄프나 관을 이용하여 적함에 뿌릴 수도 있고 항아리에 담아 던져서 터뜨릴 수도 있었으며, 길고 좁은 관(管)으로써 목표물을 향해 발사도 가능했다. 이렇게 발사되면 기름 같은 액체가 불에 타는 것으로 해상에 그대로 떠 있어서 목제 선체에 불이 쉽게 옮겨붙었다.</p>
<p>이 신무기는 <a href="/wiki/%EC%BD%98%EC%8A%A4%ED%83%84%ED%8B%B0%EB%88%84%EC%8A%A4_4%EC%84%B8" title="콘스탄티누스 4세">콘스탄티누스 4세</a> 치세인 <a href="/wiki/674%EB%85%84" title="674년">674년</a> <a class="mw-redirect" href="/wiki/%EC%9D%B4%EC%8A%AC%EB%9E%8C" title="이슬람">이슬람</a> 세력이 <a href="/wiki/%EC%BD%98%EC%8A%A4%ED%83%84%ED%8B%B0%EB%85%B8%ED%8F%B4%EB%A6%AC%EC%8A%A4" title="콘스탄티노폴리스">콘스탄티노폴리스</a>를 침공했을 때 처음 사용됐다. 비잔티움 제국은 당시 무서운 기세로 세력을 확장하던 이슬람군의 공격을 이 그리스 화약으로 물리칠 수 있었다. 그 후에도 <a href="/wiki/%EC%95%84%EB%9E%8D%EC%9D%B8" title="아랍인">아랍인</a>과 수차 벌인 전투나 <a class="mw-redirect" href="/wiki/%EB%A3%A8%EC%8A%A4%EC%A1%B1" title="루스족">루스족</a>의 침입과 <a href="/wiki/%EC%A0%9C4%EC%B0%A8_%EC%8B%AD%EC%9E%90%EA%B5%B0" title="제4차 십자군">제4차 십자군</a>의 침입에서 <a href="/wiki/%EB%B2%A0%EB%84%A4%EC%B9%98%EC%95%84_%EA%B3%B5%ED%99%94%EA%B5%AD" title="베네치아 공화국">베네치아 공화국</a>을 상대로 크게 활약하였다. 그리스 화약은 <a href="/wiki/%EC%A4%91%EC%84%B8" title="중세">중세</a> 최강 비밀 무기였고 비잔티움의 적들은 이 무기를 늘 두려워했다. <a href="/wiki/1453%EB%85%84" title="1453년">1453년</a> 비잔티움 제국이 <a href="/wiki/%EC%98%A4%EC%8A%A4%EB%A7%8C_%EC%A0%9C%EA%B5%AD" title="오스만 제국">오스만 제국</a>에 멸망할 당시 <a href="/wiki/%EC%BD%98%EC%8A%A4%ED%83%84%ED%8B%B0%EB%85%B8%ED%8F%B4%EB%A6%AC%EC%8A%A4%EC%9D%98_%ED%95%A8%EB%9D%BD" title="콘스탄티노폴리스의 함락">콘스탄티노폴리스 공방전</a>에서도 사용되었다.</p>
