<p><b>그웬 토런스</b>(Gwen Torrence, <a href="/wiki/1965%EB%85%84" title="1965년">1965년</a> <a href="/wiki/6%EC%9B%94_12%EC%9D%BC" title="6월 12일">6월 12일</a>~ )는 현재 은퇴한 <a href="/wiki/%EB%AF%B8%EA%B5%AD" title="미국">미국</a>의 <a class="mw-redirect" href="/wiki/%EC%9C%A1%EC%83%81" title="육상">육상</a> 선수이며, 3개의 <a href="/wiki/%EC%98%AC%EB%A6%BC%ED%94%BD" title="올림픽">올림픽</a> 금메달을 획득하였다.</p>
<p><a href="/wiki/%EC%A1%B0%EC%A7%80%EC%95%84_%EC%A3%BC" title="조지아 주">조지아 주</a> 드카터에서 태어난 토런스는 컬럼비아 고등학교와 <a href="/wiki/%EC%A1%B0%EC%A7%80%EC%95%84_%EB%8C%80%ED%95%99%EA%B5%90" title="조지아 대학교">조지아 대학교</a>에서 수학하였다. 토런스는 <a href="/wiki/%ED%95%98%EA%B3%84_%EC%98%AC%EB%A6%BC%ED%94%BD" title="하계 올림픽">하계 올림픽</a>, 실내와 실외 <a href="/wiki/%EC%84%B8%EA%B3%84_%EC%9C%A1%EC%83%81_%EC%84%A0%EC%88%98%EA%B6%8C_%EB%8C%80%ED%9A%8C" title="세계 육상 선수권 대회">세계 육상 선수권 대회</a>, <a href="/wiki/%ED%8C%AC%EC%95%84%EB%A9%94%EB%A6%AC%EC%B9%B8_%EA%B2%8C%EC%9E%84" title="팬아메리칸 게임">팬아메리칸 게임</a>, 친선 경기, 세계 대학 경기에서 메달을 획득하였다.</p>
<p><a href="/wiki/1988%EB%85%84" title="1988년">1988년</a> 국가 실내 선수권 대회 55m 경주에서 <a class="mw-redirect" href="/wiki/%EC%97%90%EB%B8%94%EB%A6%B0_%EC%95%A0%EC%89%AC%ED%8F%AC%EB%93%9C" title="에블린 애쉬포드">에블린 애쉬포드</a>와 함께 동점을 얻었다. 그녀는 또한 <a class="mw-redirect" href="/wiki/%ED%94%8C%EB%A1%9C%EB%A0%8C%EC%8A%A4_%EA%B7%B8%EB%A6%AC%ED%94%BC%EC%8A%A4_%EC%A1%B0%EC%9D%B4%EB%84%88" title="플로렌스 그리피스 조이너">플로렌스 그리피스 조이너</a>와 함께 육상과 기자 회견에서 많이 싸우기도 하였다.</p>
<p><a href="/wiki/1995%EB%85%84" title="1995년">1995년</a> <a href="/wiki/%EC%98%88%ED%85%8C%EB%B3%B4%EB%A6%AC" title="예테보리">예테보리</a>에서 열린 세계 육상 선수권 대회에서 100m와 400m 계주를 우승하였으나, 200m에서는 자신이 1위로 들어온 레인 밖으로 나간 이유로 실격당하고 말았다.</p>
<p><a href="/wiki/1992%EB%85%84" title="1992년">1992년</a>과 <a href="/wiki/1996%EB%85%84" title="1996년">1996년</a> 2개의 올림픽에서 400m 계주 2연승을 한 그녀는 육상에서 은퇴한 후에 <a href="/wiki/%EB%AF%B8%EC%9A%A9%EC%82%AC" title="미용사">미용사</a>가 되었다.</p>
