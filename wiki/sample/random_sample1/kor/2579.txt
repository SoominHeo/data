<p><b>극락조자리 알파</b>(α Apodis, α Aps)는 <a href="/wiki/%EA%B7%B9%EB%9D%BD%EC%A1%B0%EC%9E%90%EB%A6%AC" title="극락조자리">극락조자리</a>에서 가장 밝은 별로 <a class="mw-redirect" href="/wiki/%EA%B2%89%EB%B3%B4%EA%B8%B0_%EB%93%B1%EA%B8%89" title="겉보기 등급">겉보기 등급</a>은 약 3.825이다.<sup class="reference" id="cite_ref-kaler_4-2"><a href="#cite_note-kaler-4">[4]</a></sup> 1603년 <a href="/wiki/%EC%9A%94%ED%95%9C_%EB%B0%94%EC%9D%B4%EC%96%B4" title="요한 바이어">요한 바이어</a>는 성표 <a href="/wiki/%EC%9A%B0%EB%9D%BC%EB%85%B8%EB%A9%94%ED%8A%B8%EB%A6%AC%EC%95%84" title="우라노메트리아">우라노메트리아</a>에 <i>Apis Indica</i> 별자리를 실었는데 여기에서 가장 밝은 별에 ‘알파’를 부여했다.<sup class="reference" id="cite_ref-ridpath_8-0"><a href="#cite_note-ridpath-8">[8]</a></sup> 알파는 위도 –79°에 있는 <a href="/wiki/%EC%A3%BC%EA%B7%B9%EC%84%B1" title="주극성">주극성</a>이다. <a href="/wiki/%EC%84%BC%ED%83%80%EC%9A%B0%EB%A3%A8%EC%8A%A4%EC%9E%90%EB%A6%AC_%EC%95%8C%ED%8C%8C" title="센타우루스자리 알파">센타우루스자리 알파</a>와 <a href="/wiki/%EC%BB%B4%ED%8D%BC%EC%8A%A4%EC%9E%90%EB%A6%AC_%EC%95%8C%ED%8C%8C" title="컴퍼스자리 알파">컴퍼스자리 알파</a>를 가상의 선으로 이은 뒤 이를 천구 남극까지 연장하면 극락조자리 알파를 찾을 수 있다.<sup class="reference" id="cite_ref-moore2000_9-0"><a href="#cite_note-moore2000-9">[9]</a></sup></p>
<p>극락조자리 알파는 분광형 K2.5III의 <a href="/wiki/%EA%B1%B0%EC%84%B1" title="거성">거성</a>으로 중심핵에 있던 수소를 모두 태우고 <a class="mw-redirect" href="/wiki/%EC%A3%BC%EA%B3%84%EC%97%B4%EC%84%B1" title="주계열성">주계열성</a> 단계로부터 벗어나 죽어가는 중에 있다. 알파는 태양 반경의 48배까지 부풀어 있으며 태양보다 980배 더 밝다. 알파의 <a href="/wiki/%EA%B4%91%EA%B5%AC" title="광구">광구</a> <a class="mw-redirect" href="/wiki/%EC%9C%A0%ED%9A%A8_%EC%98%A8%EB%8F%84" title="유효 온도">유효 온도</a>는 4256켈빈으로 오렌지색 빛이 나는 K형 항성이다.<sup class="reference" id="cite_ref-csiro_10-0"><a href="#cite_note-csiro-10">[10]</a></sup> 시차 자료에 따르면 알파는 지구로부터 약 447광년 떨어져 있다.<sup class="reference" id="cite_ref-aaa474_2_653_2-3"><a href="#cite_note-aaa474_2_653-2">[2]</a></sup> 동반천체가 있는지는 밝혀지지 않았다.<sup class="reference" id="cite_ref-mnras389_2_869_11-0"><a href="#cite_note-mnras389_2_869-11">[11]</a></sup></p>
<h2><span class="mw-headline" id=".EC.9D.B4.EB.A6.84">이름</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EA%B7%B9%EB%9D%BD%EC%A1%B0%EC%9E%90%EB%A6%AC_%EC%95%8C%ED%8C%8C&amp;action=edit&amp;section=1" title="부분 편집: 이름">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>극락조자리의 <a href="/wiki/%EC%A4%91%EA%B5%AD%EC%96%B4" title="중국어">중국식</a> 명칭은 ‘이작’(<span lang="zh" xml:lang="zh">異雀</span>, <span lang="zh-Latn" xml:lang="zh-Latn">Yì Què</span>, ‘이국적인 새’라는 의미임)이며 이 별자리를 구성하는 대표적인 항성으로는 극락조자리 알파, <a href="/wiki/%EA%B7%B9%EB%9D%BD%EC%A1%B0%EC%9E%90%EB%A6%AC_%EC%A0%9C%ED%83%80" title="극락조자리 제타">제타</a>, <a href="/wiki/%EA%B7%B9%EB%9D%BD%EC%A1%B0%EC%9E%90%EB%A6%AC_%EC%9A%94%ED%83%80" title="극락조자리 요타">요타</a>, <a href="/wiki/%EA%B7%B9%EB%9D%BD%EC%A1%B0%EC%9E%90%EB%A6%AC_%EB%B2%A0%ED%83%80" title="극락조자리 베타">베타</a>, <a href="/wiki/%EA%B7%B9%EB%9D%BD%EC%A1%B0%EC%9E%90%EB%A6%AC_%EA%B0%90%EB%A7%88" title="극락조자리 감마">감마</a>, <a href="/wiki/%EA%B7%B9%EB%9D%BD%EC%A1%B0%EC%9E%90%EB%A6%AC_%EB%8D%B8%ED%83%80" title="극락조자리 델타">델타</a>, <a class="new" href="/w/index.php?title=%EA%B7%B9%EC%9E%91%EC%A1%B0%EC%9E%90%EB%A6%AC_%EB%8D%B8%ED%83%80&amp;action=edit&amp;redlink=1" title="극작조자리 델타 (없는 문서)">델타1</a>, <a href="/wiki/%EA%B7%B9%EB%9D%BD%EC%A1%B0%EC%9E%90%EB%A6%AC_%EC%97%90%ED%83%80" title="극락조자리 에타">에타</a>, <a href="/wiki/%EA%B7%B9%EB%9D%BD%EC%A1%B0%EC%9E%90%EB%A6%AC_%EC%97%A1%EC%8B%A4%EB%A1%A0" title="극락조자리 엡실론">엡실론</a>이 있다. 알파별의 명칭은 ‘이작팔’(<span lang="zh" xml:lang="zh">異雀八</span>, <span lang="zh-Latn" xml:lang="zh-Latn">Yì Què bā</span>, 이작의 여덟 번째 별)이다.<sup class="reference" id="cite_ref-12"><a href="#cite_note-12">[12]</a></sup></p>
