<td class="mbox-text"><span class="mbox-text-span"><b>이 문서는 <a class="extiw" href="https://en.wikipedia.org/wiki/" title="en:">영어 위키백과</a>의 <a class="extiw" href="https://en.wikipedia.org/wiki/Neighbourhood" title="en:Neighbourhood">Neighbourhood</a> 문서를 번역하여 문서의 내용을 확장할 필요가 있습니다.</b><span class="hide-when-compact"><br/>
<a href="/wiki/%EC%9C%84%ED%82%A4%EB%B0%B1%EA%B3%BC:%EC%8B%A0%EB%A2%B0%ED%95%A0_%EC%88%98_%EC%9E%88%EB%8A%94_%EC%B6%9C%EC%B2%98" title="위키백과:신뢰할 수 있는 출처">신뢰성 있고</a> <a href="/wiki/%EC%9C%84%ED%82%A4%EB%B0%B1%EA%B3%BC:%ED%99%95%EC%9D%B8_%EA%B0%80%EB%8A%A5" title="위키백과:확인 가능">확인할 수 있는</a> 출처가 제시되도록 번역하여 주십시오. 번역을 완료한 후에는 저작자 표시를 하는 목적으로 {{<a href="/wiki/%ED%8B%80:%EB%B2%88%EC%97%AD%EB%90%9C_%EB%AC%B8%EC%84%9C" title="틀:번역된 문서">번역된 문서</a>}} 틀을 토론창에 표기해 주십시오.</span></span></td>
</tr>

<p><b>근린주구</b>(近隣主區)는 적절한 <a href="/wiki/%EB%8F%84%EC%8B%9C_%EA%B3%84%ED%9A%8D" title="도시 계획">도시 계획</a>에 의하여 거주자의 <a href="/wiki/%EB%AC%B8%ED%99%94" title="문화">문화</a>적인 일상생활과 <a href="/wiki/%EC%82%AC%ED%9A%8C" title="사회">사회</a>적 생활을 확보할 수 있는 이상적 <a class="mw-redirect" href="/wiki/%EC%A3%BC%ED%83%9D" title="주택">주택</a>지의 단위를 말한다. 일반적으로 <a href="/wiki/%EC%B4%88%EB%93%B1%ED%95%99%EA%B5%90" title="초등학교">초등학교</a> 하나와 점포, <a class="new" href="/w/index.php?title=%EB%85%B9%EC%A7%80&amp;action=edit&amp;redlink=1" title="녹지 (없는 문서)">녹지</a> 따위의 <a class="mw-disambig" href="/wiki/%EC%8B%9C%EC%84%A4" title="시설">시설</a>을 갖추고 있다.</p>
