<p><b>글리제 876</b>은 <a href="/wiki/%EB%AC%BC%EB%B3%91%EC%9E%90%EB%A6%AC" title="물병자리">물병자리</a>에 있는 별로 <a href="/wiki/%EC%A7%80%EA%B5%AC" title="지구">지구</a>에서 15<a href="/wiki/%EA%B4%91%EB%85%84" title="광년">광년</a> 정도 떨어져 있는 <a class="mw-redirect" href="/wiki/%EC%A0%81%EC%83%89_%EC%99%9C%EC%84%B1" title="적색 왜성">적색 왜성</a>이다. 글리제 876의 <a class="mw-redirect" href="/wiki/%ED%94%8C%EB%9E%A8%EC%8A%A4%ED%8B%B0%EB%93%9C_%EA%B8%B0%ED%98%B8" title="플램스티드 기호">플램스티드 기호</a>는 <b>물병자리 IL</b>이며, <a href="/wiki/%EB%B3%80%EA%B4%91%EC%84%B1" title="변광성">변광성</a>적 특징으로 볼 때 물병자리 IL형 변광성의 원형 별이기도 하다. <a href="/wiki/2006%EB%85%84" title="2006년">2006년</a> 기준으로 이 별은 세 개의 <a href="/wiki/%EC%99%B8%EA%B3%84_%ED%96%89%EC%84%B1" title="외계 행성">외계 행성</a>을 거느리고 있는 것으로 확인되었다. 두 개는 <a href="/wiki/%EB%AA%A9%EC%84%B1%ED%98%95_%ED%96%89%EC%84%B1" title="목성형 행성">목성형 행성</a>이며 한 개는 <a href="/wiki/%ED%95%B4%EC%99%95%EC%84%B1" title="해왕성">해왕성</a> 질량 절반 정도 크기의 행성이다.</p>

<h2><span class="mw-headline" id=".EA.B1.B0.EB.A6.AC.EC.99.80_.EA.B0.80.EC.8B.9C.EC.84.B1">거리와 가시성</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EA%B8%80%EB%A6%AC%EC%A0%9C_876&amp;action=edit&amp;section=1" title="부분 편집: 거리와 가시성">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>글리제 876은 <a href="/wiki/%ED%83%9C%EC%96%91%EA%B3%84" title="태양계">태양계</a>에서 매우 가까운 곳에 있다. <a class="mw-redirect" href="/wiki/%ED%9E%88%ED%8C%8C%EB%A5%B4%EC%BD%94%EC%8A%A4_%EC%9C%84%EC%84%B1" title="히파르코스 위성">히파르코스 위성</a>의 <a class="new" href="/w/index.php?title=%EC%B8%A1%EC%A0%95_%EC%B2%9C%EB%AC%B8%ED%95%99&amp;action=edit&amp;redlink=1" title="측정 천문학 (없는 문서)">측정 천문학</a>적 관측 기구 조사에 따르면 이 별의 <a class="mw-disambig" href="/wiki/%EC%8B%9C%EC%B0%A8" title="시차">시차</a>는 212.59밀리초각으로<sup class="reference" id="cite_ref-1"><a href="#cite_note-1">[1]</a></sup> 여기서 추산된 거리는 4.70<a href="/wiki/%ED%8C%8C%EC%84%B9" title="파섹">파섹</a>(15.3광년)이다. 가까이 있음에도 이 별 자체는 어둡기 때문에 인간의 눈으로 볼 수는 없으며 <a href="/wiki/%EB%A7%9D%EC%9B%90%EA%B2%BD" title="망원경">망원경</a>을 이용해야 관측할 수 있다.</p>
<h2><span class="mw-headline" id=".ED.95.AD.EC.84.B1.EC.9D.98_.ED.8A.B9.EC.A7.95">항성의 특징</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EA%B8%80%EB%A6%AC%EC%A0%9C_876&amp;action=edit&amp;section=2" title="부분 편집: 항성의 특징">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>글리제 876은 <a class="mw-redirect" href="/wiki/%EC%A0%81%EC%83%89_%EC%99%9C%EC%84%B1" title="적색 왜성">적색 왜성</a>으로 우리 <a href="/wiki/%ED%83%9C%EC%96%91" title="태양">태양</a>보다 훨씬 더 어둡다. <a href="/wiki/%EC%A7%88%EB%9F%89" title="질량">질량</a>은 태양의 32퍼센트 정도이다.<sup class="reference" id="cite_ref-marcy_2-0"><a href="#cite_note-marcy-2">[2]</a></sup> 글리제 876의 <a class="mw-redirect" href="/wiki/%ED%91%9C%EB%A9%B4_%EC%98%A8%EB%8F%84" title="표면 온도">표면 온도</a>는 태양보다 차갑고 반지름 또한 작다.<sup class="reference" id="cite_ref-3"><a href="#cite_note-3">[3]</a></sup> 이런 요인들이 복합적으로 작용하여 글리제 876의 <a class="mw-redirect" href="/wiki/%EB%B0%9D%EA%B8%B0" title="밝기">밝기</a>는 태양의 1.38퍼센트에 불과하며 이 별이 발산하는 복사 에너지는 대부분 <a href="/wiki/%EC%A0%81%EC%99%B8%EC%84%A0" title="적외선">적외선</a> 파장에 몰려 있다.</p>
<p>대기 속의 <a href="/wiki/%EC%9D%B4%EC%9B%90%EC%9E%90_%EB%B6%84%EC%9E%90" title="이원자 분자">이원자 분자</a>가 <a href="/wiki/%EC%8A%A4%ED%8E%99%ED%8A%B8%EB%9F%BC" title="스펙트럼">스펙트럼</a>을 매우 복잡하게 만들기 때문에, 차가운 별들의 나이 및 <a class="mw-redirect" href="/wiki/%EC%A4%91%EC%9B%90%EC%86%8C_%ED%95%A8%EB%9F%89" title="중원소 함량">중원소 함량</a>을 측정하기는 어렵다. 관측된 스펙트럼을 스펙트럼 모형에 맞추면 글리제 876의 중원소 함량은 태양의 75퍼센트 수준이다.<sup class="reference" id="cite_ref-4"><a href="#cite_note-4">[4]</a></sup> <a href="/wiki/%EC%B1%84%EC%B8%B5" title="채층">채층</a> 활동을 통해 추측한 글리제 876의 나이는 이론 적 모형에 따르면 65억 ~ 99억 년 사이이다.<sup class="reference" id="cite_ref-5"><a href="#cite_note-5">[5]</a></sup></p>
<p>많은 작고 어두운 별들처럼 글리제 876은 <a href="/wiki/%EB%B3%80%EA%B4%91%EC%84%B1" title="변광성">변광성</a>이다. 이 별의 변광형은 <a class="new" href="/w/index.php?title=%EC%9A%A9%EC%9E%90%EB%A6%AC_BY_%EB%B3%80%EA%B4%91%EC%84%B1&amp;action=edit&amp;redlink=1" title="용자리 BY 변광성 (없는 문서)">용자리 BY 변광성</a>으로, 밝기는 겉보기 등급으로 0.04 정도 변동을 보인다.<sup class="reference" id="cite_ref-6"><a href="#cite_note-6">[6]</a></sup> 이러한 밝기의 변화는 별 표면에 있는 거대한 <a href="/wiki/%ED%9D%91%EC%A0%90" title="흑점">흑점</a>들이 자전과 함께 관측자의 시야에 나타나고 사라지기 때문으로 추측된다.<sup class="reference" id="cite_ref-7"><a href="#cite_note-7">[7]</a></sup></p>
<h2><span class="mw-headline" id=".ED.96.89.EC.84.B1.EA.B3.84">행성계</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EA%B8%80%EB%A6%AC%EC%A0%9C_876&amp;action=edit&amp;section=3" title="부분 편집: 행성계">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<div class="thumb tleft">

천체 예술가가 <a href="/wiki/%EA%B8%80%EB%A6%AC%EC%A0%9C_876_b" title="글리제 876 b">글리제 876 b</a>를 상상한 것.</div>

<p><a href="/wiki/1998%EB%85%84" title="1998년">1998년</a> 글리제 876 주위를 도는 <a href="/wiki/%EC%99%B8%EA%B3%84_%ED%96%89%EC%84%B1" title="외계 행성">외계 행성</a>이 제프리 마시 연구팀<sup class="reference" id="cite_ref-marcy_2-1"><a href="#cite_note-marcy-2">[2]</a></sup> 과 자비어 델포세 연구팀에 의해 각각 독립적으로 존재가 입증되었다.<sup class="reference" id="cite_ref-8"><a href="#cite_note-8">[8]</a></sup> 이 외계 행성의 이름은 <a href="/wiki/%EA%B8%80%EB%A6%AC%EC%A0%9C_876_b" title="글리제 876 b">글리제 876 b</a>로 지어졌으며, 발견 방법은 행성의 중력이 어머니 항성을 흔들리게 만드는 것에서 착안한, <a class="mw-redirect" href="/wiki/%EC%8B%9C%EC%84%A0_%EC%86%8D%EB%8F%84" title="시선 속도">시선 속도</a>법이었다. 이 행성의 질량은 <a href="/wiki/%EB%AA%A9%EC%84%B1" title="목성">목성</a>의 두 배 정도로 공전 주기는 61일이며 항성에서의 거리는 0.208 천문단위(이는 <a href="/wiki/%EC%88%98%EC%84%B1" title="수성">수성</a>과 태양 사이 거리의 절반이 되지 않는다)였다.<sup class="reference" id="cite_ref-cne_9-0"><a href="#cite_note-cne-9">[9]</a></sup></p>
<p><a href="/wiki/2001%EB%85%84" title="2001년">2001년</a> b의 안쪽 궤도를 도는 새로운 행성이 발견되었다.<sup class="reference" id="cite_ref-10"><a href="#cite_note-10">[10]</a></sup> 이 행성의 질량은 목성의 62퍼센트 정도였으며 이름은 <a href="/wiki/%EA%B8%80%EB%A6%AC%EC%A0%9C_876_c" title="글리제 876 c">글리제 876 c</a>로 지어졌다. c는 b와 1대 2의 <a href="/wiki/%EA%B6%A4%EB%8F%84_%EA%B3%B5%EB%AA%85" title="궤도 공명">궤도 공명</a> 상태에 있었으며, 항성을 30.34일 주기로 1회 공전하고 있었다. c를 발견하기 전까지는 b의 궤도가 큰 <a href="/wiki/%EC%9D%B4%EC%8B%AC%EB%A5%A0" title="이심률">이심률</a>을 그리고 있는 것으로 생각했으나, c의 발견으로 이는 궤도 공명 현상이었음이 밝혀졌다. 두 행성은 서로 강한 중력적 상호 작용을 하고 있으며 궤도 요소를 빠르게 변화시키고 있다.<sup class="reference" id="cite_ref-11"><a href="#cite_note-11">[11]</a></sup></p>
<p><a href="/wiki/2005%EB%85%84" title="2005년">2005년</a> <a class="new" href="/w/index.php?title=%EC%97%90%EC%9A%B0%EC%A0%9C%EB%8B%88%EC%98%A4_%EB%A6%AC%EB%B2%A0%EB%9D%BC&amp;action=edit&amp;redlink=1" title="에우제니오 리베라 (없는 문서)">에우제니오 리베라</a> 팀이 세 번째 행성을 발견했다. 이 행성은 앞서 발견된 둘보다 더 항성에 가까이 붙어 있었다.<sup class="reference" id="cite_ref-12"><a href="#cite_note-12">[12]</a></sup> 세 번째 행성은 <a href="/wiki/%EA%B8%80%EB%A6%AC%EC%A0%9C_876_d" title="글리제 876 d">글리제 876 d</a>로 이름지어졌다. 이 행성의 질량은 지구의 5.88배에 불과했으며, <a class="mw-redirect" href="/wiki/%EC%95%94%EC%84%9D_%ED%96%89%EC%84%B1" title="암석 행성">암석 행성</a>으로 추측된다. 시선 속도 및 두 가스 행성 사이의 상호 작용에 의거하면, 글리제 876 행성계의 공전궤도 경사각은 <a href="/wiki/%EC%B2%9C%EA%B5%AC" title="천구">천구</a>면에 대해 50도 정도 기울어져 있는 것으로 보인다. 만약 이 수치가 정확하고 세 행성의 공전면이 같은 평면상에 있다면, 이들의 질량은 시선 속도법으로 측정한 최소 질량값보다 약 30퍼센트 더 무거울 것이다. 여기에 따르면 가장 안쪽의 행성 질량은 지구의 7.5배가 된다. 반대로 <a class="new" href="/w/index.php?title=%EC%B8%A1%EC%A0%95_%EC%B2%9C%EB%AC%B8%ED%95%99&amp;action=edit&amp;redlink=1" title="측정 천문학 (없는 문서)">측정 천문학</a>적 방법으로 잰 가장 바깥쪽 행성의 공전 궤도 경사각은 84도 정도 기울어져 있으며, 여기에 따르면 실제 질량은 최솟값보다 근소하게 큰 수준일 것이다.<sup class="reference" id="cite_ref-13"><a href="#cite_note-13">[13]</a></sup>. 폴 섕크랜드(리베라 및 기타 팀과 공동 작업) 팀의 조사에 따르면 이들 행성 중 어떤 것도 항성 표면을 지나가지(<a class="new" href="/w/index.php?title=%EB%A1%9C%EC%8B%9C%ED%84%B0-%EB%A7%A5%EB%9E%98%ED%94%8C%EB%A6%B0_%ED%9A%A8%EA%B3%BC&amp;action=edit&amp;redlink=1" title="로시터-맥래플린 효과 (없는 문서)">로시터-맥래플린 효과</a>에 의해 시선속도 기울기 90도를 보여줌) 않음이 밝혀졌다. 따라서 공전면이 90도 근처의 경사각을 이룰 가능성은 더 희박해졌다.<sup class="reference" id="cite_ref-14"><a href="#cite_note-14">[14]</a></sup></p>
<p><a href="/wiki/%EB%AA%A9%EC%84%B1%ED%98%95_%ED%96%89%EC%84%B1" title="목성형 행성">목성형 행성</a> 둘 다 글리제 876의 '전통적' <a href="/wiki/%EC%83%9D%EB%AA%85%EC%B2%B4_%EA%B1%B0%EC%A3%BC%EA%B0%80%EB%8A%A5_%EC%98%81%EC%97%AD" title="생명체 거주가능 영역">생명체 거주가능 영역</a>(0.116 ~ 0.227 <a href="/wiki/%EC%B2%9C%EB%AC%B8%EB%8B%A8%EC%9C%84" title="천문단위">천문단위</a>) 내에 있다.<sup class="reference" id="cite_ref-15"><a href="#cite_note-15">[15]</a></sup> 이들의 존재는 글리제 876 행성계 생물권 내에 '지구 크기의 생명체가 존재 가능한 행성'이 있을 가능성을 날려 버린다. 반면 이들 가스행성 주위에 거대한 <a href="/wiki/%EC%9C%84%EC%84%B1" title="위성">위성</a>이 존재한다면 생명이 여기서 살 수 있을지도 모른다. 게다가 <a class="new" href="/w/index.php?title=%EC%A1%B0%EC%84%9D%EC%A0%81_%EA%B3%A0%EC%A0%95&amp;action=edit&amp;redlink=1" title="조석적 고정 (없는 문서)">조석적 고정</a>이 된 행성의 생물권 범위가 전통적 학설보다 더 늘어난다면, 글리제 876 행성계 내 어딘가에 생명체가 존재하는 천체가 있을 수도 있다<sup class="reference" id="cite_ref-16"><a href="#cite_note-16">[16]</a></sup>.</p>
