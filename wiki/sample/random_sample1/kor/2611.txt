<p><b>금 마르크</b>(<span style="font-size: smaller;"><a href="/wiki/%EB%8F%85%EC%9D%BC%EC%96%B4" title="독일어">독일어</a>:</span> <span lang="de" xml:lang="de">Goldmark</span> <small>골트마르크<sup>[<a href="/wiki/%EC%9C%84%ED%82%A4%EB%B0%B1%EA%B3%BC:%EB%8F%85%EC%9D%BC%EC%96%B4%EC%9D%98_%ED%95%9C%EA%B8%80_%ED%91%9C%EA%B8%B0" title="위키백과:독일어의 한글 표기">*</a>]</sup></small>, 공식적인 통화 이름은 <b>마르크</b>(Mark))는 <a href="/wiki/1873%EB%85%84" title="1873년">1873년</a>부터 <a href="/wiki/1914%EB%85%84" title="1914년">1914년</a>까지 통용된 <a href="/wiki/%EB%8F%85%EC%9D%BC_%EC%A0%9C%EA%B5%AD" title="독일 제국">독일 제국</a>의 통화이다.</p>
<p><a href="/wiki/%EB%8F%85%EC%9D%BC%EC%9D%98_%ED%86%B5%EC%9D%BC" title="독일의 통일">독일의 통일</a> 이전까지 <a href="/wiki/%EB%8F%85%EC%9D%BC" title="독일">독일</a>에 있던 여러 국가들은 서로 다른 자신들의 통화를 발행했다. <a href="/wiki/1876%EB%85%84" title="1876년">1876년</a> <a href="/wiki/1%EC%9B%94_1%EC%9D%BC" title="1월 1일">1월 1일</a>을 기해 마르크는 독일 제국의 유일한 법정 통화가 되었다. <a href="/wiki/%EC%A0%9C1%EC%B0%A8_%EC%84%B8%EA%B3%84_%EB%8C%80%EC%A0%84" title="제1차 세계 대전">제1차 세계 대전</a>의 발발로 인해 <a href="/wiki/1914%EB%85%84" title="1914년">1914년</a> <a href="/wiki/8%EC%9B%94_4%EC%9D%BC" title="8월 4일">8월 4일</a>을 기해 마르크와 <a href="/wiki/%EA%B8%88" title="금">금</a> 간의 태환이 정지되면서 금 마르크는 <a href="/wiki/%ED%8C%8C%ED%94%BC%EC%96%B4%EB%A7%88%EB%A5%B4%ED%81%AC" title="파피어마르크">파피어마르크</a>로 대체되었다.</p>
