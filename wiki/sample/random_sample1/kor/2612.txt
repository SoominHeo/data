<p><b>금 태조 완안 아골타</b>(金太祖 完顔 阿骨打, <a href="/wiki/1068%EB%85%84" title="1068년">1068년</a> ~ <a href="/wiki/1123%EB%85%84" title="1123년">1123년</a>)는 <a href="/wiki/%EA%B8%88%EB%82%98%EB%9D%BC" title="금나라">금나라</a>의 초대 황제(재위:1115년 ~ 1123년)이다. 중국식의 휘는 민(旻)이다. 완안(完顔)은 <a href="/wiki/%EC%97%AC%EC%A7%84%EC%96%B4" title="여진어">여진어</a>(지금 현재의 <a href="/wiki/%EB%A7%8C%EC%A3%BC%EC%96%B4" title="만주어">만주어</a>)로 왕기얀(Wanggiyan)을 가차한 것이며 한어로 번역하면 '왕(王)'이라는 뜻이고, 아골타(阿骨打)의 뜻은 여진어 아구다(Aguda)를 가차(假借)한 것이며, 너그럽고 넓은 아량이나 모양(寬闊 - 관활)라는 뜻이라고 한다. 묘호는 태조(太祖)이고 시호는 응건흥운소덕정공인명장효대성무원황제(應乾興運昭德定功仁明莊孝大聖武元皇帝)이다.</p>
<p><a href="/wiki/%EA%B8%88%EC%82%AC" title="금사">금사</a>(金史)에 따르면, <a href="/wiki/%EC%97%AC%EC%A7%84_%EB%AC%B8%EC%9E%90" title="여진 문자">여진 문자</a>는 1119년에 금태조 아골타의 명에 의하여 완안희우(完顔希于)가 만들었고, 1145년에 반포되었다고 한다.</p>

<h2><span class="mw-headline" id=".EC.83.9D.EC.95.A0">생애</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EA%B8%88_%ED%83%9C%EC%A1%B0&amp;action=edit&amp;section=1" title="부분 편집: 생애">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p><a class="mw-redirect" href="/wiki/%ED%95%B5%EB%A6%AC%EB%B0%9C" title="핵리발">핵리발</a>의 둘째 아들로 태어났다. 숙부와 형을 도와서 <a href="/wiki/%EC%99%84%EC%95%88%EB%B6%80" title="완안부">완안부</a>가 영토를 넓히는데 크게 힘을 보탰다. <a href="/wiki/1113%EB%85%84" title="1113년">1113년</a>에는, 동만주의 <a href="/wiki/%EC%83%9D%EC%97%AC%EC%A7%84" title="생여진">생여진</a>과 <a href="/wiki/%EC%B2%9C%EB%A6%AC%EC%9E%A5%EC%84%B1_(%EA%B3%A0%EB%A0%A4)" title="천리장성 (고려)">고려 장성</a> 밖의 여진을 통합했다. <a href="/wiki/1114%EB%85%84" title="1114년">1114년</a>부터 납림하(拉林河)와 출하점(出河店)(일명 : 팔리성 전투)에서 요나라의 진압군을 상대로 대승을 거두었다. 그는 요군을 영강주에서 격파했고 또 뒤이어 여러 주를 점령했다.</p>
<p>여진족은 <a href="/wiki/1115%EB%85%84" title="1115년">1115년</a> 스스로 <a href="/wiki/%EA%B8%88%EB%82%98%EB%9D%BC" title="금나라">금나라</a>를 세웠고, 황제가 된 아골타는 여세를 몰아 요동 지역의 황룡부(黃龍府)를 함락시켰다. 당시 요나라의 황제인 <a class="mw-redirect" href="/wiki/%EC%B2%9C%EC%A1%B0%EC%A0%9C" title="천조제">천조제</a>(天祚帝) 야율연희(耶律延禧)는 70만의 대군을 편성하여 금나라를 공격했으나 도리어 대패를 당했다.</p>
<p>그는 <a href="/wiki/%EC%9A%94%EB%82%98%EB%9D%BC" title="요나라">요나라</a>를 물리치고 요동으로 진출했다. 금나라는 <a href="/wiki/%EC%86%A1%EB%82%98%EB%9D%BC" title="송나라">송나라</a>와 <a href="/wiki/%EC%9A%94%EB%82%98%EB%9D%BC" title="요나라">요나라</a>를 공격하자는 동맹을 맺었다(해상의 맹약).</p>
<p><a href="/wiki/1120%EB%85%84" title="1120년">1120년</a> 요나라의 수도인 <a class="mw-redirect" href="/wiki/%EC%83%81%EA%B2%BD_%EC%9E%84%ED%99%A9%EB%B6%80" title="상경 임황부">상경 임황부</a>(上京臨潢府), <a href="/wiki/1121%EB%85%84" title="1121년">1121년</a>말에는 중경 대정부(中京大定府)를 함락했다. 강남에서 일어난 <a href="/wiki/%EB%B0%A9%EB%9E%8D%EC%9D%98_%EB%82%9C" title="방랍의 난">방랍의 난</a>으로 인해 그해에 겨우 <a href="/wiki/%EB%B6%81%EC%86%A1" title="북송">북송</a>은 북방에 출병하여 요나라의 <a class="mw-redirect" href="/wiki/%EC%B2%9C%EC%A1%B0%EC%A0%9C" title="천조제">천조제</a>가 있는 <a href="/wiki/%EC%97%B0%EA%B2%BD" title="연경">연경</a>을 공격했다. 송나라 군대는 요군에 연패를 당하고, 성과를 올릴 수 없자 송군의 지휘관 동관은 금에 원군을 요청. 해상의 맹에 의해 금나라는 장성 이남에 출병하지 않기로 약속하였지만, 송나라의 요청에 의해 출병하여 순식간에 연경을 함락시켰다. 이 결과 맹약대로 연운16주 중 연경 이남의 6주는 송나라에 할양되었지만, 금군은 연운 이북을 약탈하고 연운 16주의 백성들도 금나라로 이주시켰다.</p>
<p>그 때문에 이곳에서는 세수를 징수할 수 없는 상태에 빠졌다. 또한 금은 연경 출병의 대가로 은 20만냥, 비단 30만 필, 전 100만관, 군량 20만석을 요구했고, <a href="/wiki/%EB%B6%81%EC%86%A1" title="북송">북송</a>은 이것을 수락하지 않을 수 없었다.</p>
<p><a href="/wiki/1123%EB%85%84" title="1123년">1123년</a>에 병에 걸려서 <a href="/wiki/%ED%9A%8C%EB%A0%B9%EB%B6%80" title="회령부">회령부</a>로 가던 중에 객사했다. 후사는 <a href="/wiki/%EA%B8%88_%ED%83%9C%EC%A2%85" title="금 태종">금 태종</a>이 계승하였다.</p>
