<p><b>금천구</b>(衿川區)는 <a href="/wiki/%EB%8C%80%ED%95%9C%EB%AF%BC%EA%B5%AD" title="대한민국">대한민국</a> <a href="/wiki/%EC%84%9C%EC%9A%B8%ED%8A%B9%EB%B3%84%EC%8B%9C" title="서울특별시">서울특별시</a> 남서부에 있는 <a href="/wiki/%EA%B5%AC_(%ED%96%89%EC%A0%95_%EA%B5%AC%EC%97%AD)" title="구 (행정 구역)">구</a>다. <a href="/wiki/%EC%84%9C%EC%9A%B8%ED%8A%B9%EB%B3%84%EC%8B%9C" title="서울특별시">서울</a> 면적의 약 2.1%를 차지하며, <a href="/wiki/%EC%A4%91%EA%B5%AC_(%EC%84%9C%EC%9A%B8%ED%8A%B9%EB%B3%84%EC%8B%9C)" title="중구 (서울특별시)">중구</a>에 이어 <a href="/wiki/%EC%84%9C%EC%9A%B8%ED%8A%B9%EB%B3%84%EC%8B%9C" title="서울특별시">서울</a>에서 두 번째로 면적이 작다. 10개의 <a href="/wiki/%EB%8F%99_(%ED%96%89%EC%A0%95_%EA%B5%AC%EC%97%AD)" title="동 (행정 구역)">행정동</a>과 3개의 <a href="/wiki/%EB%B2%95%EC%A0%95%EB%8F%99" title="법정동">법정동</a>으로 구성되어 있으며, <a href="/wiki/%EB%8F%99_(%ED%96%89%EC%A0%95_%EA%B5%AC%EC%97%AD)" title="동 (행정 구역)">행정동</a> 중에서는 <a href="/wiki/%EA%B0%80%EC%82%B0%EB%8F%99" title="가산동">가산동</a>의 면적이 가장 크고, <a href="/wiki/%EB%8F%85%EC%82%B0%EB%8F%99" title="독산동">독산4동</a>이 가장 작다. 구명은 <a href="/wiki/%EC%8B%9C%ED%9D%A5%EA%B5%B0" title="시흥군">시흥군</a>의 전 명칭이던 금천현(衿川縣)에서 유래하였다. <a href="/wiki/1995%EB%85%84" title="1995년">1995년</a> <a href="/wiki/3%EC%9B%94_1%EC%9D%BC" title="3월 1일">3월 1일</a>자로 <a href="/wiki/%EA%B5%AC%EB%A1%9C%EA%B5%AC" title="구로구">구로구</a>에서 분구되어 설치되었다. 금천이란 명칭은 조선 시대의 지명인 금천에서 유래되었다.<sup class="reference" id="cite_ref-1"><a href="#cite_note-1">[1]</a></sup></p>

<h2><span class="mw-headline" id=".EC.A7.80.EB.A6.AC">지리</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EA%B8%88%EC%B2%9C%EA%B5%AC&amp;action=edit&amp;section=1" title="부분 편집: 지리">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>동쪽 및 북동쪽으로 <a href="/wiki/%EA%B4%80%EC%95%85%EA%B5%AC" title="관악구">관악구</a>, 서쪽으로 <a href="/wiki/%EA%B2%BD%EA%B8%B0%EB%8F%84" title="경기도">경기도</a> <a href="/wiki/%EA%B4%91%EB%AA%85%EC%8B%9C" title="광명시">광명시</a>, 남쪽으로 <a href="/wiki/%EA%B2%BD%EA%B8%B0%EB%8F%84" title="경기도">경기도</a> <a href="/wiki/%EC%95%88%EC%96%91%EC%8B%9C" title="안양시">안양시</a> <a href="/wiki/%EB%A7%8C%EC%95%88%EA%B5%AC" title="만안구">만안구</a>, 북서쪽으로 <a href="/wiki/%EA%B5%AC%EB%A1%9C%EA%B5%AC" title="구로구">구로구</a>와 접해있으며, <a href="/wiki/1995%EB%85%84" title="1995년">1995년</a> <a href="/wiki/%EA%B5%AC%EB%A1%9C%EA%B5%AC" title="구로구">구로구</a>에서 분구된 후 현재의 지형을 이루었다. 지역의 형태는 신발 모양을 하고 있으며, <a href="/wiki/1978%EB%85%84" title="1978년">1978년</a>에 개통된 <a href="/wiki/%EB%82%A8%EB%B6%80%EC%88%9C%ED%99%98%EB%A1%9C" title="남부순환로">남부순환로</a>가 동ㆍ서간을 따라 <a href="/wiki/%EA%B5%AC%EB%A1%9C%EA%B5%AC" title="구로구">구로구</a>, <a href="/wiki/%EA%B4%80%EC%95%85%EA%B5%AC" title="관악구">관악구</a>와 경계를 이루고, 남북간으로는 길게 <a href="/wiki/%EC%95%88%EC%96%91%EC%B2%9C" title="안양천">안양천</a>이 <a href="/wiki/%EA%B4%91%EB%AA%85%EC%8B%9C" title="광명시">광명시</a>와 경계를 이루면서 <a href="/wiki/%EA%B2%BD%EB%B6%80%EC%84%A0" title="경부선">경부선</a>이 나란히 지나고 있으며, 이 구간 지하에 새로 건설될 <a class="mw-redirect" href="/wiki/%EA%B0%95%EB%82%A8%EB%8F%84%EC%8B%9C%EA%B3%A0%EC%86%8D%ED%99%94%EB%8F%84%EB%A1%9C" title="강남도시고속화도로">강남도시고속화도로</a>가 통과할 예정이다. 또한 <a <h2><span class="mw-headline" id=".EB.AC.B8.ED.99.94">문화</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EA%B8%88%EC%B2%9C%EA%B5%AC&amp;action=edit&amp;section=9" title="부분 편집: 문화">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>독산3동에서는 <a href="/wiki/2006%EB%85%84" title="2006년">2006년</a>부터 "문화마을 만들기" 사업을 추진하고 있다. 처음엔 구청이 행정 업무로 시작했으나 2007년 10월, 20여 명이 참여하는 주민협의회를 결성한 뒤에는 주민들이 주도권을 쥐고, 공공미술을 설치할 장소 선정 등을 결정한다. <a href="/wiki/%EB%8F%85%EC%82%B0%EB%8F%99" title="독산동">독산3동</a>에서 "민둥산 독에 담긴 <a href="/wiki/%EB%8F%85%EC%82%B0%EB%8F%99" title="독산동">독산동</a> 사람들의 푸른 이야기"라는 주제로 다양한 공공미술 사업이 진행 중이다.<sup class="reference" id="cite_ref-8"><a href="#cite_note-8">[8]</a></sup></p>
