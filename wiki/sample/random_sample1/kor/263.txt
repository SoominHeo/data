<p><b>유닉스 셸</b>(Unix shell)은 <a class="mw-redirect" href="/wiki/%EB%AA%85%EB%A0%B9_%EC%A4%84" title="명령 줄">명령 줄</a>이라고도 하며 <a href="/wiki/%EC%9C%A0%EB%8B%89%EC%8A%A4" title="유닉스">유닉스</a> <a href="/wiki/%EC%9A%B4%EC%98%81_%EC%B2%B4%EC%A0%9C" title="운영 체제">운영 체제</a>와 <a class="mw-redirect" href="/wiki/%EC%9C%A0%EB%8B%89%EC%8A%A4_%EA%B3%84%ED%86%B5" title="유닉스 계통">유닉스 계통</a>의 시스템을 위한 전통적인 <a href="/wiki/%EC%82%AC%EC%9A%A9%EC%9E%90_%EC%9D%B8%ED%84%B0%ED%8E%98%EC%9D%B4%EC%8A%A4" title="사용자 인터페이스">사용자 인터페이스</a>를 제공한다. 사용자들은 명령어를 문자열로 입력함으로써 셸이 실행되게끔 <a href="/wiki/%EC%BB%B4%ED%93%A8%ED%84%B0" title="컴퓨터">컴퓨터</a>의 동작을 다룰 수 있다. <a class="mw-redirect" href="/wiki/%EB%A7%88%EC%9D%B4%ED%81%AC%EB%A1%9C%EC%86%8C%ED%94%84%ED%8A%B8_%EC%9C%88%EB%8F%84" title="마이크로소프트 윈도">마이크로소프트 윈도</a> 운영 체제 제품군 안에도 이와 비슷한 프로그램 <a class="mw-redirect" href="/wiki/Command.com" title="Command.com">command.com</a> 또는 <a href="/wiki/Cmd.exe" title="Cmd.exe">cmd.exe</a>(<a class="mw-redirect" href="/wiki/%EC%9C%88%EB%8F%84_NT" title="윈도 NT">윈도 NT</a> 기반 운영 체제)이 있다.</p>

<p>설명:</p>
<ul>
<li>빈 공백은 셸이 파일을 읽지 않는다는 것을 의미한다.</li>
<li>"yes"는 시작 시 셸이 언제나 파일을 읽어들인다는 것을 의미한다.</li>
<li>"login"은 셸이 로그인 셸인 경우 파일을 읽어들인다는 것을 의미한다.</li>
<li>"n/login"은 셸이 로그인 셸이 아닌 경우 파일을 읽어들인다는 것을 의미한다.</li>
<li>"int."는 셸이 상호작용하는 경우 파일을 읽어들인다는 것을 의미한다.</li>
</ul>
