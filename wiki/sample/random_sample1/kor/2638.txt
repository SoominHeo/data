<p><b>기노사키</b> (<span style="font-size: smaller;"><a href="/wiki/%EC%9D%BC%EB%B3%B8%EC%96%B4" title="일본어">일본어</a>:</span> <span lang="ja" xml:lang="ja">きのさき</span> <small>키노사키<sup>[<a href="/wiki/%EC%9C%84%ED%82%A4%EB%B0%B1%EA%B3%BC:%EC%9D%BC%EB%B3%B8%EC%96%B4%EC%9D%98_%ED%95%9C%EA%B8%80_%ED%91%9C%EA%B8%B0" title="위키백과:일본어의 한글 표기">*</a>]</sup></small>)는 <a href="/wiki/%EC%9D%BC%EB%B3%B8" title="일본">일본</a>의 철도 회사 <a class="mw-redirect" href="/wiki/%EC%84%9C%EC%9D%BC%EB%B3%B8_%EC%97%AC%EA%B0%9D%EC%B2%A0%EB%8F%84" title="서일본 여객철도">서일본 여객철도</a>가 운행하는 특급 열차이다. <a href="/wiki/%EA%B5%90%ED%86%A0_%EC%97%AD" title="교토 역">교토 역</a>과 <a href="/wiki/%ED%9B%84%EC%BF%A0%EC%B9%98%EC%95%BC%EB%A7%88_%EC%97%AD" title="후쿠치야마 역">후쿠치야마 역</a> <span style="font-weight:bold;">·</span> <a href="/wiki/%EB%8F%84%EC%9A%94%EC%98%A4%EC%B9%B4_%EC%97%AD_(%ED%9A%A8%EA%B3%A0_%ED%98%84)" title="도요오카 역 (효고 현)">도요오카 역</a> <span style="font-weight:bold;">·</span> <a href="/wiki/%EA%B8%B0%EB%85%B8%EC%82%AC%ED%82%A4%EC%98%A8%EC%84%BC_%EC%97%AD" title="기노사키온센 역">기노사키온센 역</a> 사이를 <a href="/wiki/%EC%82%AC%EA%B0%80%EB%85%B8_%EC%84%A0" title="사가노 선">사가노 선</a>·<a href="/wiki/%EC%82%B0%EC%9D%B8_%EB%B3%B8%EC%84%A0" title="산인 본선">산인 본선</a>을 통해 잇고 있다. 열차의 이름은 <a href="/wiki/%ED%9A%A8%EA%B3%A0_%ED%98%84" title="효고 현">효고 현</a> <a href="/wiki/%EB%8F%84%EC%9A%94%EC%98%A4%EC%B9%B4_%EC%8B%9C" title="도요오카 시">도요오카 시</a>에 있는 <a class="new" href="/w/index.php?title=%EA%B8%B0%EB%85%B8%EC%82%AC%ED%82%A4_%EC%98%A8%EC%B2%9C&amp;action=edit&amp;redlink=1" title="기노사키 온천 (없는 문서)">기노사키 온천</a>에서 유래했으며, 상징색은 사가노 선과 같은 보라색 계통 (<span style="color:#9370db;">●</span>) 이다.</p>

<h2><span class="mw-headline" id=".EA.B0.9C.EC.9A.94">개요</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EA%B8%B0%EB%85%B8%EC%82%AC%ED%82%A4_(%EC%97%B4%EC%B0%A8)&amp;action=edit&amp;section=1" title="부분 편집: 개요">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p><a href="/wiki/%EA%B5%90%ED%86%A0_%EC%8B%9C" title="교토 시">교토 시</a>·<a href="/wiki/%EA%B0%80%EB%A9%94%EC%98%A4%EC%B9%B4_%EC%8B%9C" title="가메오카 시">가메오카 시</a>를 비롯한 <a href="/wiki/%EA%B5%90%ED%86%A0_%EB%B6%80" title="교토 부">교토 부</a> 중부와 <a href="/wiki/%ED%9B%84%EC%BF%A0%EC%B9%98%EC%95%BC%EB%A7%88_%EC%8B%9C" title="후쿠치야마 시">후쿠치야마 시</a>·<a href="/wiki/%EB%8F%84%EC%9A%94%EC%98%A4%EC%B9%B4_%EC%8B%9C" title="도요오카 시">도요오카 시</a>를 비롯한 기타킨키 지방 (주단(中丹)·<a href="/wiki/%EB%8B%A4%EC%A7%80%EB%A7%88_%EA%B5%AD" title="다지마 국">다지마</a> 지방)의 도시들을 잇고 있다. 원래 "기노사키"라는 명칭을 쓰던 열차는 <a href="/wiki/1962%EB%85%84" title="1962년">1962년</a>부터 <a href="/wiki/1968%EB%85%84" title="1968년">1968년</a> "단고"에 통합될 때까지 교토 역 ~ 후쿠치야마 역 사이를 운행하던 급행 열차였다. 기노사키는 통폐합 이후 사라졌다가, <a href="/wiki/1996%EB%85%84" title="1996년">1996년</a> <a href="/wiki/3%EC%9B%94_16%EC%9D%BC" title="3월 16일">3월 16일</a> 산인 본선·기타킨키 단고 철도의 <a href="/wiki/%EC%86%8C%EB%85%B8%EB%B2%A0_%EC%97%AD" title="소노베 역">소노베 역</a> ~ <a href="/wiki/%ED%9B%84%EC%BF%A0%EC%B9%98%EC%95%BC%EB%A7%88_%EC%97%AD" title="후쿠치야마 역">후쿠치야마 역</a> ~ <a class="new" href="/w/index.php?title=%EC%95%84%EB%A7%88%EB%85%B8%ED%95%98%EC%8B%9C%EB%8B%A4%ED%85%8C_%EC%97%AD&amp;action=edit&amp;redlink=1" title="아마노하시다테 역 (없는 문서)">아마노하시다테 역</a> 구간의 전철화와 함께 이루어진 시간표 개정을 통해, 그 때까지 기동차로 운행되고 있던 특급 "아사시오"와 급행 "단고"를 통합함으로써 특급 "기노사키"로서 첫 운행을 시작했다. 그리고 2011년 3월 12일 시간표 개정에 따라 특급 "단바"를 통합하였다.</p>
<h2><span class="mw-headline" id=".EC.9A.B4.ED.96.89">운행</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EA%B8%B0%EB%85%B8%EC%82%AC%ED%82%A4_(%EC%97%B4%EC%B0%A8)&amp;action=edit&amp;section=2" title="부분 편집: 운행">편집</a><span class="mw-editsection-bracket">]</span></span></h2>

287계 전동차</div>

</div>
<p>보통 교토 역 ~ 기노사키온센 역 사이를 운행하지만, 후쿠치야마 역에서 시종착하는 열차도 운행되고 있다. 성수기에는 교토 역 ~ 기노사키온센 역 구간에서 임시 열차 81·82호가 전 좌석 지정석제로 운행된다.</p>
<p>일부 열차는 교토 역 ~ <a href="/wiki/%EC%95%84%EC%95%BC%EB%B2%A0_%EC%97%AD" title="아야베 역">아야베 역</a> 구간에서 특급 <a class="new" href="/w/index.php?title=%EB%A7%88%EC%9D%B4%EC%A6%88%EB%A3%A8_(%EC%97%B4%EC%B0%A8)&amp;action=edit&amp;redlink=1" title="마이즈루 (열차) (없는 문서)">마이즈루</a>와 병결 운행되며, 병결 운행이 이루어질 때는 <a class="new" href="/w/index.php?title=%EC%84%9C%EC%9D%BC%EB%B3%B8_%EC%97%AC%EA%B0%9D%EC%B2%A0%EB%8F%84_287%EA%B3%84_%EC%A0%84%EB%8F%99%EC%B0%A8&amp;action=edit&amp;redlink=1" title="서일본 여객철도 287계 전동차 (없는 문서)">서일본 여객철도 287계 전동차</a>가 투입된다.</p>
<p>아래는 기노사키의 정차역이다. 괄호 안의 역은 일부 열차만 정차한다.</p>
<ul>
<li>교토 역 ~ <a href="/wiki/%EB%8B%88%EC%A1%B0_%EC%97%AD_(%EA%B5%90%ED%86%A0_%EB%B6%80)" title="니조 역 (교토 부)">니조 역</a> - <a href="/wiki/%EA%B0%80%EB%A9%94%EC%98%A4%EC%B9%B4_%EC%97%AD" title="가메오카 역">가메오카 역</a> - <a href="/wiki/%EC%86%8C%EB%85%B8%EB%B2%A0_%EC%97%AD" title="소노베 역">소노베 역</a> - <a href="/wiki/%ED%9E%88%EC%9A%94%EC%8B%9C_%EC%97%AD_(%EA%B5%90%ED%86%A0_%EB%B6%80)" title="히요시 역 (교토 부)">히요시 역</a> - <a href="/wiki/%EC%95%84%EC%95%BC%EB%B2%A0_%EC%97%AD" title="아야베 역">아야베 역</a> - 후쿠치야마 역 - <a href="/wiki/%EC%99%80%EB%8B%A4%EC%95%BC%EB%A7%88_%EC%97%AD" title="와다야마 역">와다야마 역</a> - <a href="/wiki/%EC%9A%94%EC%B9%B4_%EC%97%AD" title="요카 역">요카 역</a> - <a href="/wiki/%EC%97%90%EB%B0%94%EB%9D%BC_%EC%97%AD" title="에바라 역">에바라 역</a> - 도요오카 역 - 기노사키온센 역</li>
