<p><b>주식회사 기노쿠니야 쇼텐</b>(<span style="font-size: smaller;"><a href="/wiki/%EC%9D%BC%EB%B3%B8%EC%96%B4" title="일본어">일본어</a>:</span> <span lang="ja" xml:lang="ja">株式会社紀伊國屋書店</span>, <i>KINOKUNIYA COMPANY, LTD.</i>)은 <a href="/wiki/%EC%9D%BC%EB%B3%B8" title="일본">일본</a>의 <a href="/wiki/%EC%84%9C%EC%A0%90" title="서점">서점</a>, <a href="/wiki/%EC%B6%9C%ED%8C%90%EC%82%AC" title="출판사">출판사</a>이다. <a href="/wiki/1927%EB%85%84" title="1927년">1927년</a> <a href="/wiki/1%EC%9B%94_22%EC%9D%BC" title="1월 22일">1월 22일</a>에 창업하였다.</p>
<h2><span class="mw-headline" id=".EA.B0.9C.EC.9A.94">개요</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EA%B8%B0%EB%85%B8%EC%BF%A0%EB%8B%88%EC%95%BC_%EC%87%BC%ED%85%90&amp;action=edit&amp;section=1" title="부분 편집: 개요">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<div class="thumb tleft">
<div class="thumbinner" style="width:202px;"><a class="image" href="/wiki/%ED%8C%8C%EC%9D%BC:Books_Kinokuniya_ShinjukuMinami.jpg"><img alt="" class="thumbimage" data-file-height="1055" data-file-width="801" height="263" src="//upload.wikimedia.org/wikipedia/commons/thumb/4/47/Books_Kinokuniya_ShinjukuMinami.jpg/200px-Books_Kinokuniya_ShinjukuMinami.jpg" srcset="//upload.wikimedia.org/wikipedia/commons/thumb/4/47/Books_Kinokuniya_ShinjukuMinami.jpg/300px-Books_Kinokuniya_ShinjukuMinami.jpg 1.5x, //upload.wikimedia.org/wikipedia/commons/thumb/4/47/Books_Kinokuniya_ShinjukuMinami.jpg/400px-Books_Kinokuniya_ShinjukuMinami.jpg 2x" width="200"/></a>
<div class="thumbcaption">
<div class="magnify"><a class="internal" href="/wiki/%ED%8C%8C%EC%9D%BC:Books_Kinokuniya_ShinjukuMinami.jpg" title="실제 크기로"></a></div>
신주쿠 미나미점</div>
</div>
</div>
<p>등기 상의 본점은 <a href="/wiki/%EB%8F%84%EC%BF%84_%EB%8F%84" title="도쿄 도">도쿄 도</a> <a href="/wiki/%EC%8B%A0%EC%A3%BC%EC%BF%A0_%EA%B5%AC" title="신주쿠 구">신주쿠 구</a> 신주쿠 3초메(신주쿠 본점)에 위치해 있으며, 본사 사무소는 <a href="/wiki/%EB%8F%84%EC%BF%84_%EB%8F%84" title="도쿄 도">도쿄 도</a> <a href="/wiki/%EC%8B%9C%EB%B6%80%EC%95%BC_%EA%B5%AC" title="시부야 구">시부야 구</a> 히가시 3초메에 위치해 있다.</p>
<p>이른바‘내셔널 체인’으로 불리는 전국적인 체인망을 일본에 형성하고 있으며, <a href="/wiki/%EB%AF%B8%EA%B5%AD" title="미국">미국</a>, <a href="/wiki/%EC%98%A4%EC%8A%A4%ED%8A%B8%EB%A0%88%EC%9D%BC%EB%A6%AC%EC%95%84" title="오스트레일리아">오스트레일리아</a>, <a href="/wiki/%EB%8F%99%EC%95%84%EC%8B%9C%EC%95%84" title="동아시아">동아시아</a> 등에도 진출해 있다. 수퍼마켓 체인 <a class="extiw" href="https://ja.wikipedia.org/wiki/%E7%B4%80%E3%83%8E%E5%9B%BD%E5%B1%8B" title="ja:紀ノ国屋">기노쿠니야</a> (紀ノ国屋)와는 무관하다.</p>
<p>창업자인 <a class="new" href="/w/index.php?title=%EB%8B%A4%EB%82%98%EB%B2%A0_%EB%AA%A8%EC%9D%B4%EC%B9%98&amp;action=edit&amp;redlink=1" title="다나베 모이치 (없는 문서)">다나베 모이치</a>는 서점 업계의 실력자일뿐만 아니라, 문화인으로도 유명하다. 또한 다나베의 보좌역으로 활동한 <a class="new" href="/w/index.php?title=%EB%A7%88%EC%93%B0%EB%B0%94%EB%9D%BC_%EC%98%A4%EC%82%AC%EB%AC%B4&amp;action=edit&amp;redlink=1" title="마쓰바라 오사무 (없는 문서)">마쓰바라 오사무</a>는 지금까지도 일본 서점 업계의 중심에서 활약하고 있다.</p>
<p>연극 활동에도 적극적이어서 <a class="new" href="/w/index.php?title=%EA%B8%B0%EB%85%B8%EC%BF%A0%EB%8B%88%EC%95%BC_%EC%97%B0%EA%B7%B9%EC%83%81&amp;action=edit&amp;redlink=1" title="기노쿠니야 연극상 (없는 문서)">기노쿠니야 연극상</a>을 주최하고, <a class="new" href="/w/index.php?title=%EA%B8%B0%EB%85%B8%EC%BF%A0%EB%8B%88%EC%95%BC_%ED%99%80&amp;action=edit&amp;redlink=1" title="기노쿠니야 홀 (없는 문서)">기노쿠니야 홀</a>, <a class="new" href="/w/index.php?title=%EA%B8%B0%EB%85%B8%EC%BF%A0%EB%8B%88%EC%95%BC_%EC%84%9C%EB%8D%98%EC%8B%9C%EC%96%B4%ED%84%B0&amp;action=edit&amp;redlink=1" title="기노쿠니야 서던시어터 (없는 문서)">기노쿠니야 서던시어터</a>와 2개의 극장을 운영하고 있다.</p>
<p>출판은 인문서를 중심으로 하며, 영업·판매를 담당하는 부서를 ‘홀 세일부’라고 부른다.</p>
