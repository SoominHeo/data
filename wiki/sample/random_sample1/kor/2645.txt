<a class="extiw" href="https://en.wikisource.org/wiki/Bible_(American_Standard)/John#1:29" title="wikisource:Bible (American Standard)/John">요한 1:29</a>, 예수 (왼쪽 위)가 "세상 죄를 지고 가는 <a href="/wiki/%ED%95%98%EB%8A%90%EB%8B%98%EC%9D%98_%EC%96%B4%EB%A6%B0_%EC%96%91" title="하느님의 어린 양">하느님의 어린 양</a>"으로 <a href="/wiki/%EC%84%B8%EB%A1%80%EC%9E%90_%EC%9A%94%ED%95%9C" title="세례자 요한">세례자 요한</a>에 의해 확인되고 있다.<sup class="reference" id="cite_ref-1"><a href="#cite_note-1">[1]</a></sup> <a class="new" href="/w/index.php?title=%EC%98%A4%ED%83%80%EB%B9%84%EC%98%A4_%EB%B0%94%EB%8B%88%EB%8B%88&amp;action=edit&amp;redlink=1" title="오타비오 바니니 (없는 문서)">바니니</a>에 의해 17세기 묘사.</div>
</div>

<p><a href="/wiki/%EA%B8%B0%EB%8F%85%EA%B5%90%EC%9D%B8" title="기독교인">기독교인</a>들은 <a href="/wiki/%EC%98%88%EC%88%98" title="예수">예수</a>가 <a href="/wiki/%EA%B7%B8%EB%A6%AC%EC%8A%A4%EB%8F%84" title="그리스도">그리스도</a>라고 생각하며 그의 <a href="/wiki/%EC%98%88%EC%88%98%EC%9D%98_%EB%B6%80%ED%99%9C" title="예수의 부활">죽음과 부활</a>을 통해, 인간이 <a class="new" href="/w/index.php?title=%EA%B8%B0%EB%8F%85%EA%B5%90_%EC%86%8D_%ED%95%98%EB%8A%90%EB%8B%98&amp;action=edit&amp;redlink=1" title="기독교 속 하느님 (없는 문서)">하느님</a>과 화목하여 <a class="new" href="/w/index.php?title=%EA%B5%AC%EC%9B%90_(%EA%B8%B0%EB%8F%85%EA%B5%90)&amp;action=edit&amp;redlink=1" title="구원 (기독교) (없는 문서)">구원</a>과 <a class="new" href="/w/index.php?title=%EC%98%81%EC%83%9D_(%EA%B8%B0%EB%8F%85%EA%B5%90)&amp;action=edit&amp;redlink=1" title="영생 (기독교) (없는 문서)">영생</a>의 약속을 제공 받았다고 믿는다.<sup class="reference" id="cite_ref-2"><a href="#cite_note-2">[2]</a></sup> 이런 가르침은 "하느님의 대리인이자 종"으로서, 예수가 <a href="/wiki/%EC%84%B1%EB%B6%80" title="성부">영원한 아버지</a>의 뜻에 자신의 완전한 순종의 표시로 <a href="/wiki/%EA%B3%A8%EA%B3%A0%ED%83%80" title="골고타">골고타</a>의 십자가에서 고통을 받기로 결정한, <a href="/wiki/%ED%95%98%EB%8A%90%EB%8B%98%EC%9D%98_%EC%96%B4%EB%A6%B0_%EC%96%91" title="하느님의 어린 양">하느님의 어린 양</a>으로 <a href="/wiki/%EC%9E%90%EC%9C%A0%EC%9D%98%EC%A7%80" title="자유의지">나섰음</a>을 강조한다.<sup class="reference" id="cite_ref-anselm_3-0"><a href="#cite_note-anselm-3">[3]</a></sup><sup class="reference" id="cite_ref-Cullmann_4-0"><a href="#cite_note-Cullmann-4">[4]</a></sup> 그 선택은 예수를 <a href="/wiki/%EC%95%84%EB%8B%B4" title="아담">아담</a>의 불순종에 대비되는, 도덕성과 순종의 새 사람이라는 역위치에 두었다.<sup class="reference" id="cite_ref-Pannenberg_5-0"><a href="#cite_note-Pannenberg-5">[5]</a></sup></p>
