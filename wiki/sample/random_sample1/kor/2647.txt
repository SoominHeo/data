<p><b>기독교의 교파</b>는 <a href="/wiki/%EC%98%88%EC%88%98" title="예수">예수</a>를 <a href="/wiki/%EA%B7%B8%EB%A6%AC%EC%8A%A4%EB%8F%84" title="그리스도">그리스도</a>로 고백하는 <a href="/wiki/%EA%B8%B0%EB%8F%85%EA%B5%90" title="기독교">기독교</a>의 여러 종파를 가리키는 말이다.</p>
<h2><span class="mw-headline" id=".EA.B0.9C.EC.9A.94">개요</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EA%B8%B0%EB%8F%85%EA%B5%90%EC%9D%98_%EA%B5%90%ED%8C%8C&amp;action=edit&amp;section=1" title="부분 편집: 개요">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>기독교는 <a class="mw-redirect" href="/wiki/%EC%98%88%EC%88%98_%EA%B7%B8%EB%A6%AC%EC%8A%A4%EB%8F%84%EC%9D%98_%EB%B6%80%ED%99%9C" title="예수 그리스도의 부활">예수 그리스도의 부활</a> 승천 후 사도들을 중심으로 운영된 초대교회에 근원을 두고 있다. 그런데 초대교회의 사도들의 권능을 전승하였다고 믿거나 주장하는 공교회 또는 보편교회(catholic church)에서 시작하여 언어의 차이로 인한 헬라어(그리스어)권역의 <a class="mw-redirect" href="/wiki/%EB%8F%99%EB%B0%A9%EA%B5%90%ED%9A%8C" title="동방교회">동방교회</a>와 라틴어 권역의 <a class="mw-redirect" href="/wiki/%EC%84%9C%EB%B0%A9%EA%B5%90%ED%9A%8C" title="서방교회">서방교회</a>로 구분되었다. 11세기에 공교회의 5개 총대주교 교회 연합이 정치적 대립, 문화적, 신학적 입장 차이(필리오케 논쟁, 성상 논쟁, 상호 파문)로 인하여 깨어져 로마 교회가 4개 교회(콘스타니노플, 알렉산드리아, 안티오키아, 예루살렘)와 연합을 거부해 <a class="mw-redirect" href="/wiki/%EC%84%9C%EB%B0%A9%EA%B5%90%ED%9A%8C" title="서방교회">서방교회</a>과 <a class="mw-redirect" href="/wiki/%EB%8F%99%EB%B0%A9%EA%B5%90%ED%9A%8C" title="동방교회">동방교회</a>로 완전히 나뉘는 교회 대분열이 발생한다. 이후 5백년 후 <a href="/wiki/%EA%B5%90%ED%99%A9" title="교황">교황</a>이 관할하는 로마교구를 중심으로 성장해왔지만 16세기에 들어 교회내 부패와 비기독교적 요소의 만연, 인문주의를 배경으로 교회 갱신 운동이 전개되면서 교황을 중심으로 한 로마가톨릭교회와 독일 루터파교회를 시작으로 개혁교회(장로교회), 영국국교회, 감리교회 등으로 다양한 개신교회로 발전한다.</p>
<h2><span class="mw-headline" id=".EA.B8.B0.EB.8F.85.EA.B5.90_.EA.B5.90.ED.8C.8C.EC.9D.98_.EB.B6.84.EB.A5.98">기독교 교파의 분류</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EA%B8%B0%EB%8F%85%EA%B5%90%EC%9D%98_%EA%B5%90%ED%8C%8C&amp;action=edit&amp;section=2" title="부분 편집: 기독교 교파의 분류">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>현재의 기독교 교파는 크게 3갈래로 구분하며 <a href="/wiki/%EB%A1%9C%EB%A7%88_%EA%B0%80%ED%86%A8%EB%A6%AD%EA%B5%90%ED%9A%8C" title="로마 가톨릭교회">로마 가톨릭교회</a>와 <a href="/wiki/%EB%8F%99%EB%B0%A9_%EC%A0%95%EA%B5%90%ED%9A%8C" title="동방 정교회">동방 정교회</a>, 그리고 16세기 종교개혁 운동 이후 발생한 <a href="/wiki/%EA%B0%9C%EC%8B%A0%EA%B5%90" title="개신교">개신교</a>(<a href="/wiki/%EB%A3%A8%ED%84%B0%EA%B5%90" title="루터교">루터교</a>, <a href="/wiki/%EC%B9%A8%EB%A1%80%EA%B5%90" title="침례교">침례교</a>, <a href="/wiki/%EC%84%B1%EA%B3%B5%ED%9A%8C" title="성공회">성공회</a>, <a href="/wiki/%EA%B0%90%EB%A6%AC%EA%B5%90" title="감리교">감리교</a>, <a href="/wiki/%EC%9E%A5%EB%A1%9C%EA%B5%90" title="장로교">장로교</a>, <a class="mw-redirect" href="/wiki/%EA%B0%9C%ED%98%81%EA%B5%90%ED%9A%8C" title="개혁교회">개혁교회</a> 등)로 구분된다. <a href="/wiki/%EC%98%A4%EB%A6%AC%EC%97%94%ED%8A%B8_%EC%A0%95%EA%B5%90%ED%9A%8C" title="오리엔트 정교회">오리엔트 정교회</a>에는 <a href="/wiki/%EC%9D%B4%EC%A7%91%ED%8A%B8" title="이집트">이집트</a>의 <a class="mw-redirect" href="/wiki/%EC%BD%A5%ED%8A%B8_%EA%B5%90%ED%9A%8C" title="콥트 교회">콥트 교회</a>와 시리아의 <a href="/wiki/%EC%8B%9C%EB%A6%AC%EC%95%84_%EC%A0%95%EA%B5%90%ED%9A%8C" title="시리아 정교회">시리아 정교회</a>, <a href="/wiki/%EC%97%90%ED%8B%B0%EC%98%A4%ED%94%BC%EC%95%84_%EC%A0%95%EA%B5%90%ED%9A%8C" title="에티오피아 정교회">에티오피아 교회</a>, <a class="mw-redirect" href="/wiki/%EC%95%84%EB%A5%B4%EB%A9%94%EB%8B%88%EC%95%84_%EC%82%AC%EB%8F%84_%EA%B5%90%ED%9A%8C" title="아르메니아 사도 교회">아르메니아 교회</a> 등이 존재한다. 이외에도 중동 유일의 기독교 국가인 <a href="/wiki/%EB%A0%88%EB%B0%94%EB%85%BC" title="레바논">레바논</a>과 그외 중동 <a class="mw-redirect" href="/wiki/%EC%9D%B4%EC%8A%AC%EB%9E%8C" title="이슬람">이슬람</a> 국가, <a href="/wiki/%EC%9D%B8%EB%8F%84" title="인도">인도</a> 등에 이슬람이 발생하기 전부터 <a href="/wiki/%EB%84%A4%EC%8A%A4%ED%86%A0%EB%A6%AC%EC%9A%B0%EC%8A%A4%ED%8C%8C" title="네스토리우스파">네스토리우스파</a>, <a href="/wiki/%EB%82%A8_%EC%9D%B8%EB%94%94%EC%95%84_%EA%B5%90%ED%9A%8C" title="남 인디아 교회">남 인디아 교회</a> 등이 자생했다.</p>
