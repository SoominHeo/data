<h1 class="firstHeading" id="firstHeading" lang="ko">기무라 모토</h1>
<div class="mw-body-content" id="bodyContent">
<div id="siteSub">위키백과, 우리 모두의 백과사전.</div>
<div id="contentSub"></div>
<div class="mw-jump" id="jump-to-nav">
					이동:					<a href="#mw-head">둘러보기</a>, 					<a href="#p-search">검색</a>
</div>
<div class="mw-content-ltr" dir="ltr" id="mw-content-text" lang="ko"><script>function mfTempOpenSection(id){var block=document.getElementById("mf-section-"+id);block.className+=" open-block";block.previousSibling.className+=" open-block";}</script><p><b>기무라 모토</b>(<span style="font-size: smaller;"><a href="/wiki/%EC%9D%BC%EB%B3%B8%EC%96%B4" title="일본어">일본어</a>:</span> <span lang="ja" xml:lang="ja">木村資生</span>, <a href="/wiki/1924%EB%85%84" title="1924년">1924년</a> <a href="/wiki/11%EC%9B%94_13%EC%9D%BC" title="11월 13일">11월 13일</a> - <a href="/wiki/1994%EB%85%84" title="1994년">1994년</a> <a href="/wiki/11%EC%9B%94_13%EC%9D%BC" title="11월 13일">11월 13일</a>)는 1968년 <a href="/wiki/%EC%A4%91%EB%A6%BD_%EC%A7%84%ED%99%94_%EC%9D%B4%EB%A1%A0" title="중립 진화 이론">중립 진화 이론</a>을 발표하여 세계적으로 널리 알려진 <a href="/wiki/%EC%9D%BC%EB%B3%B8" title="일본">일본</a>의 <a href="/wiki/%EC%83%9D%EB%AC%BC%ED%95%99" title="생물학">생물학자</a>이다.<sup class="reference" id="cite_ref-1"><a href="#cite_note-1">[1]</a></sup> 이로써 기무라 모토는 <a class="mw-redirect" href="/wiki/%EC%A7%91%EB%8B%A8%EC%9C%A0%EC%A0%84%ED%95%99" title="집단유전학">집단유전학</a> 이론에 가장 영향력 있는 학자가 되었다. 또한 기무라 모토가 제시한 <a class="new" href="/w/index.php?title=%ED%99%95%EC%82%B0_%EB%B0%A9%EC%A0%95%EC%8B%9D&amp;action=edit&amp;redlink=1" title="확산 방정식 (없는 문서)">확산 방정식</a>은 <a href="/wiki/%EC%9C%A0%EC%A0%84%ED%95%99" title="유전학">유전학</a>의 새로운 연구방법으로 널리 사용되고 있다. 확산 방정식은 개체의 생존과 재생산에 대하여 유리하거나 불리 또는 중립적으로 작용하는 각각의 <a class="mw-redirect" href="/wiki/%EB%8C%80%EB%A6%BD%ED%98%95%EC%A7%88" title="대립형질">대립형질</a>이 <a class="new" href="/w/index.php?title=%EA%B3%A0%EC%B0%A9_(%EC%83%9D%EB%AC%BC%ED%95%99)&amp;action=edit&amp;redlink=1" title="고착 (생물학) (없는 문서)">고착</a>될 가능성을 계산하는 수식이다.<sup class="reference" id="cite_ref-2"><a href="#cite_note-2">[2]</a></sup></p>
<p>기무라 모토는 <a class="mw-redirect" href="/wiki/%EC%A7%91%EB%8B%A8%EC%9C%A0%EC%A0%84%ED%95%99" title="집단유전학">집단유전학</a>과 <a href="/wiki/%EB%B6%84%EC%9E%90%EC%83%9D%EB%AC%BC%ED%95%99" title="분자생물학">분자생물학</a>의 데이터를 활용하여 <a href="/wiki/%EC%9C%A0%EC%A0%84%EC%9E%90_%EB%B6%80%EB%8F%99" title="유전자 부동">유전자 부동</a>에 의한 <a class="mw-redirect" href="/wiki/%EB%8C%80%EB%A6%BD%ED%98%95%EC%A7%88_%EB%B0%9C%ED%98%84%EB%B9%88%EB%8F%84" title="대립형질 발현빈도">대립형질 발현빈도</a>의 변화가 <a href="/wiki/%EC%A7%84%ED%99%94" title="진화">진화</a>의 가장 큰 요인으로 작용한다는 <a href="/wiki/%EC%A4%91%EB%A6%BD_%EC%A7%84%ED%99%94_%EC%9D%B4%EB%A1%A0" title="중립 진화 이론">중립 진화 이론</a>을 발표하였다.<sup class="reference" id="cite_ref-3"><a href="#cite_note-3">[3]</a></sup> 위스콘신 주립대학의 석좌교수이자 널리 알려진 집단유전학자인 제임스 F 크로우는 기무라 모토의 업적을 <a href="/wiki/%ED%98%84%EB%8C%80_%EC%A7%84%ED%99%94_%EC%9D%B4%EB%A1%A0" title="현대 진화 이론">현대 진화 이론</a>의 선구자인 <a class="new" href="/w/index.php?title=%EC%A1%B4_%EB%B2%84%EB%8D%98_%EC%83%8C%EB%8D%94%EC%8A%A8_%ED%95%A0%EB%8D%B0%EC%9D%B8&amp;action=edit&amp;redlink=1" title="존 버던 샌더슨 할데인 (없는 문서)">할데인</a>, <a class="new" href="/w/index.php?title=%EC%8A%A4%EC%9B%B0_%EB%9D%BC%EC%9D%B4%ED%8A%B8&amp;action=edit&amp;redlink=1" title="스웰 라이트 (없는 문서)">라이트</a>, <a href="/wiki/%EB%A1%9C%EB%84%90%EB%93%9C_%ED%94%BC%EC%85%94" title="로널드 피셔">피셔</a>와 동등한 위치에 있다고 평가하였다.<sup class="reference" id="cite_ref-4"><a href="#cite_note-4">[4]</a></sup></p>
<h2><span class="mw-headline" id=".EC.83.9D.EC.95.A0.EC.99.80_.EC.97.85.EC.A0.81">생애와 업적</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EA%B8%B0%EB%AC%B4%EB%9D%BC_%EB%AA%A8%ED%86%A0&amp;action=edit&amp;section=1" title="부분 편집: 생애와 업적">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>기무라 모토는 1924년 <a href="/wiki/%EC%9D%BC%EB%B3%B8" title="일본">일본</a>의 <a href="/wiki/%EC%95%84%EC%9D%B4%EC%B9%98_%ED%98%84" title="아이치 현">아이치 현</a> <a href="/wiki/%EC%98%A4%EC%B9%B4%EC%9E%90%ED%82%A4_%EC%8B%9C" title="오카자키 시">오카자키 시</a>에서 태어났다. 어려서부터 식물학과 수학에 재능을 보였다. <a class="mw-redirect" href="/wiki/%EB%82%98%EA%B3%A0%EC%95%BC" title="나고야">나고야</a>에서 고등학교를 다니면서 구마자키 연구소에서 <a href="/wiki/%EB%82%98%EB%A6%AC%EC%86%8D" title="나리속">백합</a>의 <a href="/wiki/%EC%97%BC%EC%83%89%EC%B2%B4" title="염색체">염색체</a> 구조에 대한 연구를 하였다. 연구소장이었던 구마자키는 기무라의 식물과 수학에 대한 재능을 한데 묶을 수 있는 학문으로 <a class="new" href="/w/index.php?title=%EC%83%9D%EB%AC%BC%ED%86%B5%EA%B3%84%ED%95%99&amp;action=edit&amp;redlink=1" title="생물통계학 (없는 문서)">생물통계학</a>을 추천하였다.<sup class="reference" id="cite_ref-Crow_5-0"><a href="#cite_note-Crow-5">[5]</a></sup></p>
<p><a class="mw-redirect" href="/wiki/%EC%A0%9C2%EC%B0%A8_%EC%84%B8%EA%B3%84%EB%8C%80%EC%A0%84" title="제2차 세계대전">제2차 세계대전</a> 중이던 1944년 기무라는 <a href="/wiki/%EA%B5%90%ED%86%A0_%EB%8C%80%ED%95%99" title="교토 대학">교토 제국대학</a>에 입학하였다. 지도교수인 기하라 히토시가 징병을 피하기 위해선 세포학 보다는 당시 농업 개량의 필요성 때문에 중요하게 여겨지고 있던 식물학을 전공으로 하는 것이 좋겠다고 충고했기 때문에 기무라는 식물학을 전공으로 택했다. 전쟁 후 기하라의 연구실에서 일하게 된 그는 외래 식물에 의한 유전자의 유입에 대해 연구하였고 집단유전학을 공부하였다. 1949년 기무라는 <a href="/wiki/%EB%AF%B8%EC%8B%9C%EB%A7%88_%EC%8B%9C" title="미시마 시">미시마 시</a>의 국립 유전자 연구소로 자리를 옮겼다. 1953년 기무라 모토는 집단유전학에 대한 첫 논문을 발표하였다. 그 논문에서 기무라는 <a class="new" href="/w/index.php?title=%EC%8A%A4%EC%9B%B0_%EB%9D%BC%EC%9D%B4%ED%8A%B8&amp;action=edit&amp;redlink=1" title="스웰 라이트 (없는 문서)">스웰 라이트</a>의 "섬 모델"보다 복잡한 패턴을 보이는 집단 유전학 모델인 "징검다리 모델"을 제시하였다. 1953년 기무라는 <a href="/wiki/%EB%AF%B8%EA%B5%AD" title="미국">미국</a>을 방문하였으며 <a class="new" href="/w/index.php?title=%EC%9B%90%ED%8F%AD_%ED%94%BC%ED%95%B4%EC%9E%90_%EC%9C%84%EC%9B%90%ED%9A%8C&amp;action=edit&amp;redlink=1" title="원폭 피해자 위원회 (없는 문서)">원폭 피해자 위원회</a>에서 활동하던 유전학자 던컨 맥도날드의 소개로 <a href="/wiki/%EC%95%84%EC%9D%B4%EC%98%A4%EC%99%80_%EC%A3%BC%EB%A6%BD_%EB%8C%80%ED%95%99%EA%B5%90" title="아이오와 주립 대학교">아이오와 주립 대학교</a>에 자리를 잡게 되었다.<sup class="reference" id="cite_ref-Crow_5-1"><a href="#cite_note-Crow-5">[5]</a></sup></p>
<p>1960년 기무라 모토는 집단유전학에 대한 연구성과를 정리하여 《집단유전학 개론》<sup class="reference" id="cite_ref-6"><a href="#cite_note-6">[6]</a></sup> 을 출간하였다. 이 책에는 기무라가 그 동안 연구하였던 "무한 공간"에서 발생하는 "무한개의 대립형질"의 유전자 부동에 대한 연구와 <a href="/wiki/%EB%8B%A8%EB%B0%B1%EC%A7%88_%EC%84%9C%EC%97%B4" title="단백질 서열">단백질 서열</a>과 <a class="new" href="/w/index.php?title=%EC%9C%A0%EC%A0%84%EC%9E%90_%EC%84%9C%EC%97%B4&amp;action=edit&amp;redlink=1" title="유전자 서열 (없는 문서)">유전자 서열</a>에 대한 <a class="mw-redirect" href="/wiki/%EB%B6%84%EC%9E%90%EC%A7%84%ED%99%94%ED%95%99" title="분자진화학">분자진화학</a>적 연구, 그리고 <a href="/wiki/%EC%83%81%EB%8F%99%EC%84%B1" title="상동성">상동성</a> 단백질에 대한 <a class="new" href="/w/index.php?title=%EC%A0%84%EA%B8%B0%EC%98%81%EB%8F%99&amp;action=edit&amp;redlink=1" title="전기영동 (없는 문서)">전기영동</a> 실험 결과 등이 수록되어 있다. 또한 1961년에는 <a class="new" href="/w/index.php?title=%EC%9C%A0%EC%A0%84%EC%A0%81_%ED%95%98%EC%A4%91&amp;action=edit&amp;redlink=1" title="유전적 하중 (없는 문서)">유전적 하중</a>에 대한 간략한 논문을 발표하였다.</p>
<p>1968년 기무라 모토는 <a href="/wiki/%EC%A4%91%EB%A6%BD_%EC%A7%84%ED%99%94_%EC%9D%B4%EB%A1%A0" title="중립 진화 이론">중립 진화 이론</a>을 발표하였다. 이 이론은 분자 단위에서 보면 <a href="/wiki/%EC%9E%90%EC%97%B0%EC%84%A0%ED%83%9D" title="자연선택">자연선택</a>보다는 <a href="/wiki/%EC%9C%A0%EC%A0%84%EC%9E%90_%EB%B6%80%EB%8F%99" title="유전자 부동">유전자 부동</a>이 <a href="/wiki/%EC%A7%84%ED%99%94" title="진화">진화</a>의 주된 요인으로 작용한다는 것이다.<sup class="reference" id="cite_ref-Kimura_7-0"><a href="#cite_note-Kimura-7">[7]</a></sup> 이 이론은 당시 생물학의 그 어떤 분야보다도 급격한 발전을 이루고 있던 분자생물학의 성과들을 진화 이론에 반영한 것으로 발표 후에 커다란 논쟁이 뒤따랐다. 이 논쟁에서 분자생물학자들의 다수가 기무라 모토를 지지하였다.</p>
<p>이후 기무라 모토는 생애의 대부분을 중립 진화 이론을 발전시키고 방어하는 데 보냈다. 1983년 《분자진화학의 중립이론》<sup class="reference" id="cite_ref-Kimura_7-1"><a href="#cite_note-Kimura-7">[7]</a></sup> 를 발간하였으며, 1988년 일본어로 발간된 《생물진화를 생각한다》<sup class="reference" id="cite_ref-8"><a href="#cite_note-8">[8]</a></sup> 는 일본내에서 베스트셀러가 되었다.</p>
<p>많은 학문적 논쟁의 결과 중립 진화 이론은 오늘날 분자진화학의 현대 이론으로 인정되고 있다.<sup class="reference" id="cite_ref-9"><a href="#cite_note-9">[9]</a></sup></p>
<p>1992년 기무라 모토는 <a href="/wiki/%EC%98%81%EA%B5%AD" title="영국">영국</a> <a class="mw-redirect" href="/wiki/%EC%99%95%EB%A6%BD_%ED%95%99%ED%9A%8C" title="왕립 학회">왕립 학회</a>의 외국인 회원이 되었으며 <a class="new" href="/w/index.php?title=%EB%8B%A4%EC%9C%88_%EB%A9%94%EB%8B%AC&amp;action=edit&amp;redlink=1" title="다윈 메달 (없는 문서)">다윈 메달</a>을 수여받았다. 기무라는 기무라 히로코와 결혼하였으며 사이에 한 명의 자식이 있다.<sup class="reference" id="cite_ref-10"><a href="#cite_note-10">[10]</a></sup></p>
