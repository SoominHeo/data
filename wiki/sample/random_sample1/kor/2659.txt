<h2><span class="mw-headline" id=".EA.B0.9C.EB.9E.B5">개략</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EA%B8%B0%EB%AC%B4%EB%9D%BC_%EC%B9%B4%EC%97%90%EB%9D%BC&amp;action=edit&amp;section=1" title="부분 편집: 개략">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p><a href="/wiki/%EB%8F%84%EC%BF%84_%EB%8F%84" title="도쿄 도">도쿄 도</a> <a href="/wiki/%EC%95%84%EB%8B%A4%EC%B9%98_%EA%B5%AC" title="아다치 구">아다치 구</a> 출신으로 아버지는 <a href="/wiki/%EC%98%81%EA%B5%AD%EC%9D%B8" title="영국인">영국 사람</a>, 어머니는 <a href="/wiki/%EC%9D%BC%EB%B3%B8%EC%9D%B8" title="일본인">일본 사람</a>이다. "카엘라"(Kaela)는 "최고의 사랑"을 의미하는 <a href="/wiki/%ED%9E%88%EB%B8%8C%EB%A6%AC%EC%96%B4" title="히브리어">히브리어</a>로 유래되는 <a href="/wiki/%EC%98%81%EC%96%B4" title="영어">영어</a>의 이름이다. 일본 사람은 "카에라"(<span lang="ja" xml:lang="ja">かえら</span>, Kaera)라는 발음으로 부른다.</p>
<p>패션 모델로서 활약한 뒤 <a href="/wiki/2004%EB%85%84" title="2004년">2004년</a> <a href="/wiki/6%EC%9B%94_23%EC%9D%BC" title="6월 23일">6월 23일</a> 싱글 "Level 42"로 가수 데뷔했다. <a href="/wiki/2005%EB%85%84" title="2005년">2005년</a>에 발매한 싱글 "<span lang="ja" xml:lang="ja">リルラ リルハ</span>"(Real Life, Real Heart)가 <a href="/wiki/%EC%98%A4%EB%A6%AC%EC%BD%98" title="오리콘">오리콘</a> 3위를 기록하고 일약 유명해졌다.</p>
<h2><span class="mw-headline" id=".EA.B2.B0.ED.98.BC">결혼</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EA%B8%B0%EB%AC%B4%EB%9D%BC_%EC%B9%B4%EC%97%90%EB%9D%BC&amp;action=edit&amp;section=2" title="부분 편집: 결혼">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p><a href="/wiki/2010%EB%85%84" title="2010년">2010년</a> <a href="/wiki/6%EC%9B%94_1%EC%9D%BC" title="6월 1일">6월 1일</a> 배우 <a href="/wiki/%EC%97%90%EC%9D%B4%ED%83%80" title="에이타">에이타</a>와 결혼 예정임을 각 공식 사이트에서 보고했다. 당시 카에라는 임신 5개월로, 지난해 말부터 결혼을 결심하고 있었다라고 밝혔다. 두 사람의 첫 만남은 <a href="/wiki/2009%EB%85%84" title="2009년">2009년</a>이라고한다. 그 후 <a href="/wiki/9%EC%9B%94_1%EC%9D%BC" title="9월 1일">9월 1일</a> 정식으로 입적한 것을 보고, <a href="/wiki/10%EC%9B%94_28%EC%9D%BC" title="10월 28일">10월 28일</a> 장남의 출산을 발표했다. 2013년, 둘째 임신 5개월임을 발표했다.</p>

<h2><span class="mw-headline" id=".EB.B0.B0.EC.9A.A9.EC.A4.80_.EB.B9.84.ED.95.98.EB.B0.9C.EC.96.B8_.EB.85.BC.EB.9E.80">배용준 비하발언 논란</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EA%B8%B0%EB%AC%B4%EB%9D%BC_%EC%B9%B4%EC%97%90%EB%9D%BC&amp;action=edit&amp;section=11" title="부분 편집: 배용준 비하발언 논란">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p><a href="/wiki/%ED%95%9C%EA%B5%AD" title="한국">한국</a>에선 <a href="/wiki/2005%EB%85%84" title="2005년">2005년</a> <a href="/wiki/5%EC%9B%94_24%EC%9D%BC" title="5월 24일">5월 24일</a>, 일본의 <a href="/wiki/TV_%EA%B0%80%EB%82%98%EA%B0%80%EC%99%80" title="TV 가나가와">TV 가나가와</a>(tvk)의 "<a href="/wiki/%EC%82%AC%EC%BF%A0%EC%82%AC%EC%BF%A0" title="사쿠사쿠">사쿠사쿠</a>"(<a class="extiw" href="https://ja.wikipedia.org/wiki/Saku_saku" title="ja:Saku saku">saku saku</a>)라는 프로그램에서 <a href="/wiki/%ED%95%9C%EB%A5%98_(%EB%AC%B8%ED%99%94)" title="한류 (문화)">한류</a>, <a href="/wiki/%EB%B0%B0%EC%9A%A9%EC%A4%80" title="배용준">배용준</a>, 가정주부 등을 비하하는 발언으로 인해 <a class="mw-redirect" href="/wiki/%EC%9D%BC%EB%B3%B8%EC%9D%98_%EC%9A%B0%EC%9D%B5_%EB%8B%A8%EC%B2%B4" title="일본의 우익 단체">일본 우익</a>계 네티즌팬들이 또한 상당하다. 하지만 그녀의 공식 블로그를 방문해 보면 "배 카엘라"라는 제목의 글로 삼겹살과 김치가 너무 맛있으며, 상추쌈에 대해서도 호의적인 글을 올려놓고 있는 것을 알 수 있었다. 심지어는 식사후 남은 김치를 싸서 집에까지 가져오며 "김치 너무 좋아"라고 이야기 하고 있지만, 배용준의 초상화를 못생기게 그려서 올리는 모습을 보여주기도 하였다. 그러나 평소 그녀의 그림을 볼 때 그런 극단적인 평가를 내리는 것은 문제가 있다고 생각된다.</p>
