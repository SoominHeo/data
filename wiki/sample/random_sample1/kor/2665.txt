<h1 class="firstHeading" id="firstHeading" lang="ko">기소 (일본사)</h1>
<div class="mw-body-content" id="bodyContent">
<div id="siteSub">위키백과, 우리 모두의 백과사전.</div>
<div id="contentSub"></div>
<div class="mw-jump" id="jump-to-nav">
					이동:					<a href="#mw-head">둘러보기</a>, 					<a href="#p-search">검색</a>
</div>
<div class="mw-content-ltr" dir="ltr" id="mw-content-text" lang="ko"><script>function mfTempOpenSection(id){var block=document.getElementById("mf-section-"+id);block.className+=" open-block";block.previousSibling.className+=" open-block";}</script><p><b>기소</b>(<span style="font-size: smaller;"><a href="/wiki/%EC%9D%BC%EB%B3%B8%EC%96%B4" title="일본어">일본어</a>:</span> <span lang="ja"><span lang="ja"><ruby><rb>碁所</rb> <rp>(</rp><rt>ごどころ</rt><rp>)</rp></ruby></span></span> <small>고도코로<sup>[<a href="/wiki/%EC%9C%84%ED%82%A4%EB%B0%B1%EA%B3%BC:%EC%9D%BC%EB%B3%B8%EC%96%B4%EC%9D%98_%ED%95%9C%EA%B8%80_%ED%91%9C%EA%B8%B0" title="위키백과:일본어의 한글 표기">*</a>]</sup></small>)는 <a href="/wiki/%EC%97%90%EB%8F%84_%EB%A7%89%EB%B6%80" title="에도 막부">에도 막부</a>의 직책이다. 그 직무는 <a class="new" href="/w/index.php?title=%EC%96%B4%EC%84%B1%EA%B8%B0&amp;action=edit&amp;redlink=1" title="어성기 (없는 문서)">어성기</a>(막부배 바둑대회)의 관리, 전국의 바둑 기사의 총람 등이었다. 오늘날의 기원 총재직에 해당한다고 볼 수 있다.</p>
<p><a href="/wiki/%EC%82%AC%EC%82%AC%EB%B4%89%ED%96%89" title="사사봉행">사사봉행</a>의 관할 하에 있었으며 정원은 1명이고, 때론 임명되지 않는 경우도 있었다. 바둑 <a class="mw-redirect" href="/wiki/%EA%B0%80%EC%9B%90" title="가원">가원</a>인 <a href="/wiki/%ED%98%BC%EC%9D%B8%EB%B3%B4" title="혼인보">혼인보 가</a>, <a class="new" href="/w/index.php?title=%EC%9D%B4%EB%85%B8%EC%9A%B0%EC%97%90_%EA%B0%80&amp;action=edit&amp;redlink=1" title="이노우에 가 (없는 문서)">이노우에 가</a>, <a class="new" href="/w/index.php?title=%EC%95%BC%EC%8A%A4%EC%9D%B4_%EA%B0%80&amp;action=edit&amp;redlink=1" title="야스이 가 (없는 문서)">야스이 가</a>, <a class="new" href="/w/index.php?title=%ED%95%98%EC%95%BC%EC%8B%9C_%EA%B0%80&amp;action=edit&amp;redlink=1" title="하야시 가 (없는 문서)">하야시 가</a>의 4개 가문에서 담당자가 뽑혀 취임했고, 취임하기 위해서는 <a class="new" href="/w/index.php?title=%EB%AA%85%EC%9D%B8_(%EB%B0%94%EB%91%91)&amp;action=edit&amp;redlink=1" title="명인 (바둑) (없는 문서)">명인</a>의 기량을 가지고 있어야 했다. <a href="/wiki/%EB%8F%84%EC%BF%A0%EA%B0%80%EC%99%80_%EC%9D%B4%EC%97%90%EC%95%BC%EC%8A%A4" title="도쿠가와 이에야스">도쿠가와 이에야스</a>가 바둑을 애호했기 때문에 <a href="/wiki/%EC%87%BC%EA%B8%B0" title="쇼기">쇼기</a>를 관리하는 벼슬인 <a class="new" href="/w/index.php?title=%EC%9E%A5%EA%B8%B0%EC%86%8C&amp;action=edit&amp;redlink=1" title="장기소 (없는 문서)">장기소</a>보다 서열이 높았다.</p>
<p>간분 2년(서기 1662년)에 바둑과 장기가 사사봉행의 관할하에 놓이게 되었고, 간분 8년(서기 1668년) 10월 18일 막부가 <a class="new" href="/w/index.php?title=%EC%95%BC%EC%8A%A4%EC%9D%B4_%EC%82%B0%EC%B9%98&amp;action=edit&amp;redlink=1" title="야스이 산치 (없는 문서)">야스이 산치</a>를 최초의 기소로 임명한 것이 시초이다.</p>
<p>바둑 4가의 각 당주는 이 기소 자리를 둘러싸고 각종 정치공작을 벌이기도 했다. <a href="/wiki/%EB%AF%B8%ED%86%A0_%EB%B2%88" title="미토 번">미토 번</a>주 <a href="/wiki/%EB%8F%84%EC%BF%A0%EA%B0%80%EC%99%80_%EB%82%98%EB%A6%AC%EC%95%84%ED%82%A4" title="도쿠가와 나리아키">도쿠가와 나리아키</a>, <a class="mw-redirect" href="/wiki/%EB%85%B8%EC%A4%91" title="노중">노중</a> <a class="new" href="/w/index.php?title=%EB%A7%88%EC%93%B0%EB%8B%A4%EC%9D%B4%EB%9D%BC_%EC%95%BC%EC%8A%A4%ED%86%A0%EC%98%A4&amp;action=edit&amp;redlink=1" title="마쓰다이라 야스토오 (없는 문서)">마쓰다이라 야스토오</a>, 사사봉행 등이 연루된 <a class="new" href="/w/index.php?title=%ED%98%BC%EC%9D%B8%EB%B3%B4_%EC%A3%A0%EC%9A%B0%EC%99%80&amp;action=edit&amp;redlink=1" title="혼인보 죠우와 (없는 문서)">혼인보 죠우와</a>와 <a class="new" href="/w/index.php?title=%EC%9D%B4%EB%85%B8%EC%9A%B0%EC%97%90_%EA%B2%90%EB%82%9C%EC%9D%B8%EC%84%B8%ED%82%A4&amp;action=edit&amp;redlink=1" title="이노우에 겐난인세키 (없는 문서)">이노우에 겐난인세키</a>의 암투가 유명하다.</p>
