<h2><span class="mw-headline" id="1.EC.84.B8.EB.8C.80.28BC.29">1세대(BC)</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EA%B8%B0%EC%95%84_%EB%A6%AC%EC%98%A4&amp;action=edit&amp;section=1" title="부분 편집: 1세대(BC)">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<h3><span class="mw-headline" id=".EB.A6.AC.EC.98.A4.281999.EB.85.84_11.EC.9B.94.7E2002.EB.85.84_8.EC.9B.94.29">리오(1999년 11월~2002년 8월)</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EA%B8%B0%EC%95%84_%EB%A6%AC%EC%98%A4&amp;action=edit&amp;section=2" title="부분 편집: 리오(1999년 11월~2002년 8월)">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p><a href="/wiki/1996%EB%85%84_9%EC%9B%94" title="1996년 9월">1996년 9월</a>에 컨셉트를 잡고, 이듬해인 <a href="/wiki/1997%EB%85%84_2%EC%9B%94" title="1997년 2월">1997년 2월</a>에 프로젝트 명 B-Ⅲ로 개발에 들어가 총 33개월의 개발 기간 동안 1,100억 원의 연구 개발 및 부품 구매비와 1,800억원의 투자 비용이 든 큰 프로젝트가 진행되었다. 총 241대의 프로토 타입을 만들어 테스트를 거쳤고, 글로벌 시장을 겨냥한 만큼 많은 차량을 벤치마킹했다. <a href="/wiki/%EC%84%B8%EB%8B%A8" title="세단">세단</a>은 엣지 스타일, <a href="/wiki/%EC%8A%A4%ED%85%8C%EC%9D%B4%EC%85%98_%EC%99%9C%EA%B1%B4" title="스테이션 왜건">스테이션 왜건</a>은 크로스오버 스타일로 만들어졌다. <a href="/wiki/%EB%8C%80%ED%95%9C%EB%AF%BC%EA%B5%AD" title="대한민국">대한민국</a>에서는 <a href="/wiki/1999%EB%85%84" title="1999년">1999년</a> <a href="/wiki/11%EC%9B%94_8%EC%9D%BC" title="11월 8일">11월 8일</a>에 소하리 공장에서 열린 보도 발표회 때 언론에 노출되었고, 이틀 후인 <a href="/wiki/11%EC%9B%94_10%EC%9D%BC" title="11월 10일">11월 10일</a>에 첫 선을 보였다. 기존의 <a href="/wiki/%EA%B8%B0%EC%95%84_%ED%94%84%EB%9D%BC%EC%9D%B4%EB%93%9C" title="기아 프라이드">프라이드</a>와 <a href="/wiki/%EA%B8%B0%EC%95%84_%EC%95%84%EB%B2%A8%EB%9D%BC" title="기아 아벨라">아벨라</a>가 <a href="/wiki/%EA%B8%B0%EC%95%84%EC%9E%90%EB%8F%99%EC%B0%A8" title="기아자동차">기아</a>, <a class="mw-redirect" href="/wiki/%ED%8F%AC%EB%93%9C_%EC%9E%90%EB%8F%99%EC%B0%A8_%ED%9A%8C%EC%82%AC" title="포드 자동차 회사">포드</a>, <a href="/wiki/%EB%A7%88%EC%93%B0%EB%8B%A4" title="마쓰다">마쓰다</a>의 합작으로 만들어진 것과 달리 리오는 <a href="/wiki/%EA%B8%B0%EC%95%84%EC%9E%90%EB%8F%99%EC%B0%A8" title="기아자동차">기아</a>가 독자 개발하여 <a href="/wiki/%EA%B8%B0%EC%95%84_%ED%94%84%EB%9D%BC%EC%9D%B4%EB%93%9C" title="기아 프라이드">프라이드</a>나 <a href="/wiki/%EA%B8%B0%EC%95%84_%EC%95%84%EB%B2%A8%EB%9D%BC" title="기아 아벨라">아벨라</a>와는 확연히 다른 느낌을 강조한 21세기적인 디자인으로 탈바꿈하였으나, 어정쩡한 디자인과 준중형차 수요의 증가에 따라 전작인 <a href="/wiki/%EA%B8%B0%EC%95%84_%ED%94%84%EB%9D%BC%EC%9D%B4%EB%93%9C#1.EC.84.B8.EB.8C.80.28Y.29" title="기아 프라이드">프라이드</a> 만큼의 인기를 끌지 못하였다. 차명인 리오는 <a href="/wiki/%EC%8A%A4%ED%8E%98%EC%9D%B8%EC%96%B4" title="스페인어">스페인어</a>로 활기차다, 역동적이다라는 의미를 지닌다. 바디 타입은 <a href="/wiki/%EC%84%B8%EB%8B%A8" title="세단">세단</a>과 <a href="/wiki/%EC%8A%A4%ED%85%8C%EC%9D%B4%EC%85%98_%EC%99%9C%EA%B1%B4" title="스테이션 왜건">스테이션 왜건</a> 등 2가지이며, <a href="/wiki/%EC%8A%A4%ED%85%8C%EC%9D%B4%EC%85%98_%EC%99%9C%EA%B1%B4" title="스테이션 왜건">스테이션 왜건</a>은 RX-V라는 서브 네임이 달렸다. 엔진은 1.3ℓ <a href="/wiki/SOHC" title="SOHC">SOHC</a>, 1.5ℓ <a href="/wiki/SOHC" title="SOHC">SOHC</a>, 1.5ℓ <a href="/wiki/DOHC" title="DOHC">DOHC</a> 등 3가지가 있었고, <a href="/wiki/2001%EB%85%84" title="2001년">2001년</a>에 <a href="/wiki/%EB%8C%80%ED%95%9C%EB%AF%BC%EA%B5%AD_%EA%B5%AD%ED%86%A0%ED%95%B4%EC%96%91%EB%B6%80" title="대한민국 국토해양부">건설교통부(현 국토해양부</a>)가 주관한 <a href="/wiki/%EB%8C%80%ED%95%9C%EB%AF%BC%EA%B5%AD" title="대한민국">대한민국</a> 소형차의 충돌 테스트에서 <a href="/wiki/%ED%98%84%EB%8C%80_%EB%B2%A0%EB%A5%B4%EB%82%98" title="현대 베르나">현대 베르나</a>, <a href="/wiki/%EB%8C%80%EC%9A%B0_%EB%9D%BC%EB%85%B8%EC%8A%A4" title="대우 라노스">대우 라노스</a>보다 높은 점수를 받았다.<sup class="reference" id="cite_ref-1"><a href="#cite_note-1">[1]</a></sup></p>
<h4><span class="mw-headline" id=".EC.A0.9C.EC.9B.90">제원</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EA%B8%B0%EC%95%84_%EB%A6%AC%EC%98%A4&amp;action=edit&amp;section=3" title="부분 편집: 제원">편집</a><span class="mw-editsection-bracket">]</span></span></h4>

<h3><span class="mw-headline" id=".EB.A6.AC.EC.98.A4_SF.282002.EB.85.84_8.EC.9B.94.7E2005.EB.85.84_4.EC.9B.94.29">리오 SF(2002년 8월~2005년 4월)</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EA%B8%B0%EC%95%84_%EB%A6%AC%EC%98%A4&amp;action=edit&amp;section=5" title="부분 편집: 리오 SF(2002년 8월~2005년 4월)">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p><a href="/wiki/2002%EB%85%84" title="2002년">2002년</a> <a href="/wiki/8%EC%9B%94_1%EC%9D%BC" title="8월 1일">8월 1일</a>에는 페이스 리프트를 거친 리오 SF가 출시되었다. 서브 네임인 SF는 미래 지향적인 소형차라는 의미(Science Fiction)와 소형차 중 최고의 안전성이 있다는 의미(Safety First)를 갖고 있다.<sup class="reference" id="cite_ref-2"><a href="#cite_note-2">[2]</a></sup> 번호판의 위치는 범퍼에서 트렁크로 옮겨졌다. 리오 SF의 <a href="/wiki/%EC%8A%A4%ED%85%8C%EC%9D%B4%EC%85%98_%EC%99%9C%EA%B1%B4" title="스테이션 왜건">스테이션 왜건</a>은 RX-V라는 서브 네임 없이 그냥 5도어로 불렸다. 기존보다 개선된 디자인에도 불구하고, 여전히 판매가 부진함과 같이 <a href="/wiki/2005%EB%85%84_4%EC%9B%94" title="2005년 4월">2005년 4월</a>에 후속 차종인 <a href="/wiki/%EA%B8%B0%EC%95%84_%ED%94%84%EB%9D%BC%EC%9D%B4%EB%93%9C#2.EC.84.B8.EB.8C.80.28JB.29" title="기아 프라이드">뉴 프라이드</a>가 출시되어 단종되었다. 그러나 현재까지 <a href="/wiki/%EA%B8%B0%EC%95%84_%ED%94%84%EB%9D%BC%EC%9D%B4%EB%93%9C" title="기아 프라이드">프라이드</a>의 수출명은 리오이다.</p>
<h4><span class="mw-headline" id=".EC.A0.9C.EC.9B.90_2">제원</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EA%B8%B0%EC%95%84_%EB%A6%AC%EC%98%A4&amp;action=edit&amp;section=6" title="부분 편집: 제원">편집</a><span class="mw-editsection-bracket">]</span></span></h4>
