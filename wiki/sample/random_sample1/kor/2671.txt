<p>당초 <a href="/wiki/%ED%98%84%EB%8C%80_%EA%B7%B8%EB%9E%9C%EC%A0%80#3.EC.84.B8.EB.8C.80.28XG.29" title="현대 그랜저">현대 그랜저 XG</a>의 플랫폼을 활용해 <a href="/wiki/%ED%98%84%EB%8C%80_%EB%8B%A4%EC%9D%B4%EB%84%88%EC%8A%A4%ED%8B%B0" title="현대 다이너스티">현대 다이너스티</a>의 후속 차종으로 개발되고 있었으나, 개발 도중 <a href="/wiki/%EA%B8%B0%EC%95%84%EC%9E%90%EB%8F%99%EC%B0%A8" title="기아자동차">기아자동차</a>에 넘겨져 <a href="/wiki/%EA%B8%B0%EC%95%84_%EC%97%94%ED%84%B0%ED%94%84%EB%9D%BC%EC%9D%B4%EC%A6%88" title="기아 엔터프라이즈">엔터프라이즈</a>를 대체하는 차종으로 출시됐다. 차명인 오피러스는 <a href="/wiki/%EC%98%81%EC%96%B4" title="영어">영어</a>로 '의견 주도층(Opinion Leader Of Us)'을 의미하며, 동시에 <a href="/wiki/%EB%9D%BC%ED%8B%B4%EC%96%B4" title="라틴어">라틴어</a>로 '황금의 땅(Ophir Rus)'을 의미하기도 한다.<sup class="reference" id="cite_ref-1"><a href="#cite_note-1">[1]</a></sup> <a href="/wiki/2003%EB%85%84" title="2003년">2003년</a> <a href="/wiki/3%EC%9B%94_12%EC%9D%BC" title="3월 12일">3월 12일</a>에 출시됐고, 출시 전에 <a class="mw-disambig" href="/wiki/SBS" title="SBS">SBS</a> <a href="/wiki/%EC%98%AC%EC%9D%B8" title="올인">드라마 올인</a>에 먼저 등장하기도 했다.<sup class="reference" id="cite_ref-2"><a href="#cite_note-2">[2]</a></sup> <a href="/wiki/%EB%AF%B8%EA%B5%AD" title="미국">미국</a> 시장에서는 <a href="/wiki/2003%EB%85%84_10%EC%9B%94" title="2003년 10월">2003년 10월</a>에 수출됐으며, V6 3.5ℓ 시그마 엔진을 장착해서 아만티라는 차명으로 판매됐다. <a href="/wiki/2005%EB%85%84" title="2005년">2005년</a> <a href="/wiki/3%EC%9B%94_16%EC%9D%BC" title="3월 16일">3월 16일</a>에 선보인 <a href="/wiki/2005%EB%85%84" title="2005년">2005년</a>형은 V6 3.8ℓ 람다 엔진이 기존의 V6 3.5ℓ 시그마 엔진을 대체했고, 방향 지시등 내장 아웃 사이드 미러과 후방 카메라 등이 적용됐다.</p>

<p><a href="/wiki/2006%EB%85%84" title="2006년">2006년</a> <a href="/wiki/5%EC%9B%94_23%EC%9D%BC" title="5월 23일">5월 23일</a>에 페이스 리프트를 거친 뉴 오피러스가 출시됐으며, 내·외장 디자인 등이 바뀌는 등 많은 부분에서 개선되었다. 새로 개발된 V6 2.7ℓ 뮤 엔진, V6 3.3ℓ 람다 엔진이 적용됐고, 알루미늄 엔진 블록과 알루미늄 마그네슘 등 신소재 사용을 늘려 공차 중량을 줄였다. 특히 페이스 리프트만으로 전체적인 밸런스를 맞추고, 디자인을 크게 개선하여 호평을 얻었다. <a href="/wiki/2007%EB%85%84" title="2007년">2007년</a> <a href="/wiki/7%EC%9B%94_20%EC%9D%BC" title="7월 20일">7월 20일</a>에 선보인 <a href="/wiki/2008%EB%85%84" title="2008년">2008년</a>형은 뒤에도 길어진 신규 번호판을 장착할 수 있도록 했고, DVD 내비게이션에 위험 지역 알림 기능 추가, 보조 매트 후면부 재질 개선(미끄럼 방지) 등의 변화가 이뤄졌다. <a href="/wiki/2008%EB%85%84" title="2008년">2008년</a> <a href="/wiki/4%EC%9B%94_28%EC%9D%BC" title="4월 28일">4월 28일</a>에 선보인 <a href="/wiki/2009%EB%85%84" title="2009년">2009년</a>형은 자동 요금 징수 시스템, MP3 기능과 AUX/USB/iPod 단자가 추가된 오디오 등이 적용됐다.</p>

<p><a href="/wiki/2009%EB%85%84" title="2009년">2009년</a> <a href="/wiki/5%EC%9B%94_23%EC%9D%BC" title="5월 23일">5월 23일</a>에는 패밀리 룩이 적용된 라디에이터 그릴과 범퍼 매립형 듀얼 머플러, 6단 자동변속기 등을 갖춘 오피러스 프리미엄이 선보였다. 또한 <a href="/wiki/%ED%99%94%EC%84%B1%EC%8B%9C" title="화성시">화성</a> 공장에서 <a href="/wiki/%EA%B4%91%EB%AA%85%EC%8B%9C" title="광명시">소하리</a> 공장으로 옮겨 생산되었다. 같은 해 <a href="/wiki/12%EC%9B%94" title="12월">12월</a>에 수출이 중단됐고, <a href="/wiki/%EB%8C%80%ED%95%9C%EB%AF%BC%EA%B5%AD" title="대한민국">대한민국</a>을 제외한 국가에서는 <a href="/wiki/%EA%B8%B0%EC%95%84_K7" title="기아 K7">K7</a>으로 대체되었다.<sup class="reference" id="cite_ref-3"><a href="#cite_note-3">[3]</a></sup> <a href="/wiki/2010%EB%85%84" title="2010년">2010년</a> <a href="/wiki/10%EC%9B%94_7%EC%9D%BC" title="10월 7일">10월 7일</a>에는 크루즈 컨트롤, 멀티 통합 전자식 룸 미러(하이 패스, 후방 카메라), 웰컴 시스템 등이 적용된 <a href="/wiki/2011%EB%85%84" title="2011년">2011년</a>형이 선보였다. <a href="/wiki/2011%EB%85%84_12%EC%9B%94" title="2011년 12월">2011년 12월</a>에 생산이 중단된 후로 한동안 재고 차량이 판매되었다. 이후 <a href="/wiki/%EA%B8%B0%EC%95%84%EC%9E%90%EB%8F%99%EC%B0%A8" title="기아자동차">기아자동차</a>의 기함 자리는 <a href="/wiki/2012%EB%85%84" title="2012년">2012년</a> <a href="/wiki/5%EC%9B%94_2%EC%9D%BC" title="5월 2일">5월 2일</a>에 선보인 상급 차종인 <a href="/wiki/%EA%B8%B0%EC%95%84_K9" title="기아 K9">K9</a>으로 이관되었다.</p>
