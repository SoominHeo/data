<p><b>기아 챌린저스 필드</b>(Kia Challengers Field)는 <a href="/wiki/%EC%A0%84%EB%9D%BC%EB%82%A8%EB%8F%84" title="전라남도">전라남도</a> <a href="/wiki/%ED%95%A8%ED%8F%89%EA%B5%B0" title="함평군">함평군</a> <a href="/wiki/%ED%95%99%EA%B5%90%EB%A9%B4" title="학교면">학교면</a> 곡창리에 있는 <a href="/wiki/%EC%95%BC%EA%B5%AC%EC%9E%A5" title="야구장">야구장</a>이다. 현재 <a href="/wiki/KIA_%ED%83%80%EC%9D%B4%EA%B1%B0%EC%A6%88" title="KIA 타이거즈">KIA 타이거즈</a>가 퓨처스팀,3군 홈구장이자 구단 전용 훈련장으로 사용하고 있다.</p>
<p>기존 <a href="/wiki/KIA_%ED%83%80%EC%9D%B4%EA%B1%B0%EC%A6%88" title="KIA 타이거즈">KIA 타이거즈</a> 2군 야구장이었던 <a href="/wiki/%EB%8C%80%EB%8F%99%EB%A9%B4_(%ED%95%A8%ED%8F%89%EA%B5%B0)" title="대동면 (함평군)">대동면</a> 용성리 <a href="/wiki/%EC%A0%84%EB%82%A8%EC%95%BC%EA%B5%AC%EC%9E%A5" title="전남야구장">전남야구장</a> 시설이 노후하여 선수들이 뛰기 어렵다는 지적이 많아, <a href="/wiki/%EC%A0%84%EB%9D%BC%EB%82%A8%EB%8F%84" title="전라남도">전라남도</a> <a href="/wiki/%ED%95%A8%ED%8F%89%EA%B5%B0" title="함평군">함평군</a> <a href="/wiki/%ED%95%99%EA%B5%90%EB%A9%B4" title="학교면">학교면</a> 곡창리에 2만 4천평 규모로 <a href="/wiki/2010%EB%85%84" title="2010년">2010년</a>에 착공, <a href="/wiki/2012%EB%85%84_4%EC%9B%94" title="2012년 4월">2012년 4월</a>에 완공하였다.<sup class="reference" id="cite_ref-1"><a href="#cite_note-1">[1]</a></sup> 클럽 하우스가 완공된 <a href="/wiki/2013%EB%85%84" title="2013년">2013년</a> <a href="/wiki/8%EC%9B%94_28%EC%9D%BC" title="8월 28일">8월 28일</a>에 정식 개관했다.<sup class="reference" id="cite_ref-2"><a href="#cite_note-2">[2]</a></sup></p>
<p>주 경기장은 좌우 펜스까지 100m, 중앙 펜스는 120m 규모. 전자 점수판을 설치하였으며 내야 연습장은 가로 세로가 각 70m 규모이고, 실내 연습장은 가로 세로가 각 50m, 높이 15m로 투구, 타격, 수비, 작전 등 모든 훈련을 할 수 있다. 숙소동에는 객실 20개와 라커룸, 식당, 휴게실, 샤워실, 헬스장, 의무실 등을 마련했다.</p>
<p>주 경기장에 조명 시설이 설치되어 있지 않아 야간 경기, 야간 훈련은 할 수 없다.</p>
<p><a href="/wiki/2012%EB%85%84" title="2012년">2012년</a> 주 경기장과 보조 경기장은 완공이 되어 선수들이 경기를 펼치는데 문제가 없으며 클럽하우스와 실내연습장은 <a href="/wiki/2013%EB%85%84" title="2013년">2013년</a> <a href="/wiki/8%EC%9B%94_28%EC%9D%BC" title="8월 28일">8월 28일</a>에 완공했다.</p>
<p><a href="/wiki/2013%EB%85%84" title="2013년">2013년</a> <a href="/wiki/7%EC%9B%94_8%EC%9D%BC" title="7월 8일">7월 8일</a>, <a href="/wiki/%EA%B8%B0%EC%95%84%EC%9E%90%EB%8F%99%EC%B0%A8" title="기아자동차">기아자동차</a>는 항상 1군에 오르기 위해 도전하는 2군 선수들의 도전 정신을 담아 2군 야구장의 명칭을 '함평-기아 챌린저스 필드'로 선정했다.<sup class="reference" id="cite_ref-3"><a href="#cite_note-3">[3]</a></sup></p>
<p>2015 시즌 후 3군 선수단 전용 훈련장 신설을 위해 챌린저스필드에 야구장 1면을 증축하고, 야간 조명을 설치해 육성 선수들의 야간 경기 적응력을 높일 예정이다. 더불어 챌린저스필드 부지 내에 등 속성 측정기와 3차원 척추안정화 장비·산소탱크 치료기·아쿠아 마사지 장비·돗토리 재활센터 장비 등을 갖춘 재활센터를 신축할 계획이다.</p>
