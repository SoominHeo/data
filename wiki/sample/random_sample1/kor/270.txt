<p><b>Vorbis</b>는 <a class="mw-redirect" href="/wiki/Xiph.org" title="Xiph.org">Xiph.org</a>에서 개발한 오디오 코덱이다. <a href="/wiki/%EC%98%A4%ED%94%88_%EC%86%8C%EC%8A%A4" title="오픈 소스">오픈 소스</a>며 누구나 무료로 사용할 수 있는 <a href="/wiki/%EC%86%90%EC%8B%A4_%EC%95%95%EC%B6%95" title="손실 압축">손실 압축</a> 오디오 코덱이다. 확장자는 보통 .ogg를 사용하지만, <a href="/wiki/%ED%85%8C%EC%98%A4%EB%9D%BC" title="테오라">테오라</a> 파일로 혼동하는 것을 막기 위해 .oga 확장자를 사용하기도 한다.</p>

<h2><span class="mw-headline" id=".EC.9D.B4.EB.A6.84">이름</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Vorbis&amp;action=edit&amp;section=1" title="부분 편집: 이름">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>'Vorbis'라는 이름은 <a href="/wiki/%ED%85%8C%EB%A6%AC_%ED%94%84%EB%9E%98%EC%B3%87" title="테리 프래쳇">테리 프래쳇</a>의 소설, <a href="/wiki/%EB%94%94%EC%8A%A4%ED%81%AC%EC%9B%94%EB%93%9C" title="디스크월드">디스크월드</a>의 등장인물인 Exquisitor Vorbis의 이름에서 따왔다.<sup class="reference" id="cite_ref-xiphname_4-0"><a href="#cite_note-xiphname-4">[4]</a></sup></p>
<h2><span class="mw-headline" id=".EC.9A.94.EC.95.BD">요약</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Vorbis&amp;action=edit&amp;section=2" title="부분 편집: 요약">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<ul>
<li><a href="/wiki/%EA%B0%80%EB%B3%80_%EB%B9%84%ED%8A%B8%EB%A0%88%EC%9D%B4%ED%8A%B8" title="가변 비트레이트">VBR</a>과 <a href="/wiki/%ED%8F%89%EA%B7%A0_%EB%B9%84%ED%8A%B8%EB%A0%88%EC%9D%B4%ED%8A%B8" title="평균 비트레이트">ABR</a>를 지원한다.</li>
<li><a href="/wiki/%EC%98%A4%ED%94%88_%EC%86%8C%EC%8A%A4" title="오픈 소스">오픈 소스</a>이므로 별도의 사용료가 없다.</li>

<h2><span class="mw-headline" id=".EC.97.AD.EC.82.AC">역사</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Vorbis&amp;action=edit&amp;section=4" title="부분 편집: 역사">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>Vorbis는 <a href="/wiki/1998%EB%85%84" title="1998년">1998년</a> 9월에 처음으로 연구가 진행되었다. 이때 <a href="/wiki/MP3" title="MP3">MP3</a> 포맷은 라이선스가 등록되어 있어서 일정한 금액을 지불해야 했다. 이에 Vorbis 개발자는 완전 무료의 차세대 오디오 코덱을 개발하기로 결심한다. 약 2년 동안 다른 개발자와 함께 협업하여 <a href="/wiki/2001%EB%85%84" title="2001년">2001년</a> <a href="/wiki/6%EC%9B%94_15%EC%9D%BC" title="6월 15일">6월 15일</a>, RC1 버전을 내놓는다. <a href="/wiki/2001%EB%85%84" title="2001년">2001년</a> <a href="/wiki/8%EC%9B%94_30%EC%9D%BC" title="8월 30일">8월 30일</a>에는 RC2 버전을 선보이는데, 이 버전은 최고 <a href="/wiki/%EB%B9%84%ED%8A%B8%EB%A0%88%EC%9D%B4%ED%8A%B8" title="비트레이트">비트레이트</a>가 350kbps로 확장했다. 당시에 Vorbis는 MP3를 뛰어넘을 획기적인 압축 기술로 주목받았다. <a href="/wiki/2002%EB%85%84" title="2002년">2002년</a> <a href="/wiki/7%EC%9B%94_17%EC%9D%BC" title="7월 17일">7월 17일</a> 공식판 1.0이 발표되었다. 공식판의 미비한 점을 보완하고자 개발자 aoTuV는 <a href="/wiki/2004%EB%85%84" title="2004년">2004년</a>부터 공식판을 토대로 미비한 부분을 겹겹이 튜닝한 버전을 내놓는다.</p>
<h2><span class="mw-headline" id=".ED.8A.B9.EC.A7.95">특징</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Vorbis&amp;action=edit&amp;section=5" title="부분 편집: 특징">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<h3><span class="mw-headline" id=".EB.A9.94.ED.83.80.EB.8D.B0.EC.9D.B4.ED.84.B0">메타데이터</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Vorbis&amp;action=edit&amp;section=6" title="부분 편집: 메타데이터">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>Ogg Vorbis에도 MP3의 ID3에 비견되는 메타데이터 표준이 있으며, 이를 <a href="/wiki/%EB%B3%B4%EB%B9%84%EC%8A%A4_%EC%BD%94%EB%A9%98%ED%8A%B8" title="보비스 코멘트">Vorbis 코멘트</a>라고 부른다. 메타데이터는 키 값과 커멘트 내용의 짝으로 저장이 되며, 키와 커멘트의 길이는 2<sup>32</sup>-1 바이트까지 가능하다. 이 정보는 Vorbis 비트 스트림의 처음 부분에 있는 시간 값이 담긴 헤더에 저장된다.</p>
<p>키 값은 대소문자 구분이 없는 7 bit ASCII 값으로 저장되며, 일부 문자는 허용되지 않는다. 커멘트 내용은 <a href="/wiki/UTF-8" title="UTF-8">UTF-8</a>로 저장된다. 음악 태그는 일반적으로 "[태그 키]=[커멘트 내용]"과 같은 형식으로 저장된다.</p>
<pre>
"ARTIST=Various Artist"
</pre>
<p>ID3도 마찬가지로 태그를 입력하는 데 자유로운 편이다. 또한 Replaygain과 Discnumber 등의 2차적 태그를 지원한다.</p>
<h3><span class="mw-headline" id=".EC.9E.A5.EC.A0.90">장점</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Vorbis&amp;action=edit&amp;section=7" title="부분 편집: 장점">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>Vorbis는 모든 비트레이트를 자유롭게 지정할 수 있고 기존 코덱(MP3)의 한계를 넘을 수 있도록 설계되어 있다. Vorbis는 같은 비트레이트의 <a href="/wiki/MP3" title="MP3">MP3</a>보다 음질이 좋고, <a href="/wiki/%EA%B3%A0%EA%B8%89_%EC%98%A4%EB%94%94%EC%98%A4_%EB%B6%80%ED%98%B8%ED%99%94" title="고급 오디오 부호화">AAC</a>와는 음질이 비슷하거나 그 이상의 음질을 보장한다.단,CBR의 경우 MP3보다 안좋은 결과를 내놓기도 한다.</p>
<p>압축률의 지정은 일반적으로 퀄리티 레벨로 불리는 수치로 지정한다. 범위는 -1에서 10까지 있다. 44.1kHz Stereo (2ch)의 소스의 경우에 표준 퀄리티 레벨은 Q3(112kbps)가 된다. 최고 퀄리티 레벨은 Q10(500kbps), 최저 퀄리티 레벨은 Q-1(48kbps)가 된다. (<a href="#aoTuV">aoTuV</a> 인코더에서는 최저 퀄리티 레벨인 Q-1의 하위인 Q-2를 지정할 수 있어 32kbps로 인코딩 할 수 있다).</p>
<p>이전에는 인코더 속도가 느리다는 지적이 있었지만, Blacksword에 의한 Ogg Vorbis 고속화 프로젝트에 의해 크게 개선되고 있다. 음질은 확장성의 높이를 살려 겹겹이 튜닝을 한 aoTuV가 긴 세월에 걸쳐 높은 평가를 거두고 있어 공식판의 최신 버전에 aoTuV 튜닝 기술이 들어갔다.</p>
<p>Vorbis는 프로그램에서는 샘플 단위로 위치를 지정해 정확하게 디코드할 수 있고, 별도의 소스 사용 금액이 없어서, PC 게임 등에서 많이 사용하고 있다.<sup class="reference" id="cite_ref-5"><a href="#cite_note-5">[5]</a></sup></p>
<h3><span class="mw-headline" id=".EB.8B.A8.EC.A0.90">단점</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Vorbis&amp;action=edit&amp;section=8" title="부분 편집: 단점">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<ul>
<li>Vorbis는 가변 비트레이트가 기본이므로 AVI 같은 VBR 음성 코덱을 따로 지원하지 않는 곳에 사용하면 문제가 생길 수 있다.</li>
<li>Vorbis는 MP3보다 더 많은 인코더 시간을 요구한다.
<ul>
<li>Ogg Vorbis 고속화 프로젝트로 인코더 시간을 줄일 수 있다.<sup class="reference" id="cite_ref-6"><a href="#cite_note-6">[6]</a></sup></li>
</ul>
</li>
<li>재생할 수 있는 플레이어가 적다.</li>
<li>MP3에 비해 디코더 메모리가 많이 필요하다. 현대의 개인용 컴퓨터에서는 아무런 문제가 되지 않지만 MP3 플레이어 같은 휴대 매체에서는 전지를 빨리 소비하는 단점이 있다.</li>
<li><a href="/wiki/%EB%8C%80%ED%95%9C%EB%AF%BC%EA%B5%AD" title="대한민국">대한민국</a> 대부분의 음원 제공 사이트는 MP3 형식의 음원만을 제공한다. Vorbis 형식의 음원은 구하기 힘들다.</li>
<li>MP3와 달리 앨범아트 기능을 지원하지 않는다</li>
<li>무손실을 지원하지 않는다</li>
<li>고음역대가 착색되었다</li>
</ul>
<h2><span class="mw-headline" id=".EC.A2.85.EB.A5.98">종류</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Vorbis&amp;action=edit&amp;section=9" title="부분 편집: 종류">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<h3><span class="mw-headline" id=".EA.B3.B5.EC.8B.9D.ED.8C.90">공식판</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Vorbis&amp;action=edit&amp;section=10" title="부분 편집: 공식판">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>Xiph.Org 재단에서 개발하고 관리하고 있으며 <a href="/wiki/BSD" title="BSD">BSD</a> 라이선스와 <a href="/wiki/GNU" title="GNU">GNU</a> 라이선스를 따른다. 2007년 6월 22일 1.2.0 버전이 올라와있다.</p>
<h3><span class="mw-headline" id="aoTuV">aoTuV</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Vorbis&amp;action=edit&amp;section=11" title="부분 편집: aoTuV">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>aoTuV는 공식판 버전을 새로 튜닝한 버전으로 공식판 버전에 비해 괜찮은 음질을 얻을 수 있다. 대체적으로 공식판 버전이 판올림될 때 같이 판올림되는 특징을 보인다. 현재 최신 버전은 <a href="/wiki/2009%EB%85%84" title="2009년">2009년</a> <a href="/wiki/3%EC%9B%94_3%EC%9D%BC" title="3월 3일">3월 3일</a>에 나온 베타 5.7 버전이다. 고속화 프로젝트 팀은 aoTuV를 바탕으로 최적화된 고속화 버전으로 개량하는데, aoTuV로 인코딩하는 속도보다 고속화 버전으로 개량한 버전이 인코딩 속도가 더 빠르다.</p>
<h2><span class="mw-headline" id=".EC.A7.80.EC.9B.90">지원</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Vorbis&amp;action=edit&amp;section=12" title="부분 편집: 지원">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<h3><span class="mw-headline" id=".ED.95.98.EB.93.9C.EC.9B.A8.EC.96.B4">하드웨어</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Vorbis&amp;action=edit&amp;section=13" title="부분 편집: 하드웨어">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<ul>
<li><a href="/wiki/%EC%BD%94%EC%9B%90%EC%8B%9C%EC%8A%A4%ED%85%9C" title="코원시스템">코원시스템</a> iAudio</li>
<li><a class="mw-redirect" href="/wiki/%EC%82%BC%EC%84%B1" title="삼성">삼성</a> 옙 MP3 플레이어</li>
<li><a href="/wiki/%EC%95%84%EC%9D%B4%EB%A6%AC%EB%B2%84" title="아이리버">아이리버</a></li>
</ul>
<p><a href="/wiki/%EC%95%A0%ED%94%8C_(%EA%B8%B0%EC%97%85)" title="애플 (기업)">애플</a>의 <a href="/wiki/%EC%95%84%EC%9D%B4%ED%8C%9F" title="아이팟">아이팟</a>은 아직 정식으로 Vorbis를 지원하지 않지만, <a href="/wiki/%EB%9D%BD%EB%B0%95%EC%8A%A4" title="락박스">락박스</a>를 이용하여 Vorbis 파일을 재생할 수 있다. Xiph.org 재단 위키에서 vorbis를 지원하는 하드웨어 목록을 올리고 있다.<sup class="reference" id="cite_ref-7"><a href="#cite_note-7">[7]</a></sup></p>
<h3><span class="mw-headline" id=".EC.86.8C.ED.94.84.ED.8A.B8.EC.9B.A8.EC.96.B4">소프트웨어</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Vorbis&amp;action=edit&amp;section=14" title="부분 편집: 소프트웨어">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>애플의 <a class="mw-redirect" href="/wiki/%EC%95%84%EC%9D%B4%ED%8A%A0%EC%A6%88" title="아이튠즈">아이튠즈</a>는 Vorbis를 지원하지 않지만, Xiph.org 재단에서 개발한 컴포넌트로 Vorbis를 재생할 수 있다. 컴포넌트는 <a class="mw-redirect" href="/wiki/%EB%A7%88%EC%9D%B4%ED%81%AC%EB%A1%9C%EC%86%8C%ED%94%84%ED%8A%B8_%EC%9C%88%EB%8F%84" title="마이크로소프트 윈도">윈도</a> 버전과 <a href="/wiki/%EB%A7%A4%ED%82%A8%ED%86%A0%EC%8B%9C" title="매킨토시">매킨토시</a> 버전 둘 다 나와있다. <a class="mw-redirect" href="/wiki/%EC%9C%88%EB%8F%84_%EB%AF%B8%EB%94%94%EC%96%B4_%ED%94%8C%EB%A0%88%EC%9D%B4%EC%96%B4" title="윈도 미디어 플레이어">윈도 미디어 플레이어</a>도 애플처럼 Vorbis를 지원하지 않지만, 다이렉트 쇼 필터로 Vorbis를 재생할 수 있다. <a href="/wiki/%EB%A6%AC%EB%88%85%EC%8A%A4" title="리눅스">리눅스</a>에서는 리눅스 기반 프로그램인 XMMS나 xine 등으로 Vorbis 재생을 지원한다. 더 많은 정보는 재단 홈페이지에 있다.<sup class="reference" id="cite_ref-8"><a href="#cite_note-8">[8]</a></sup> 윈도 버전에서는 <a href="/wiki/%EC%9C%88%EC%95%B0%ED%94%84" title="윈앰프">윈앰프</a>와 <a class="mw-redirect" href="/wiki/%ED%91%B8%EB%B0%942000" title="푸바2000">푸바2000</a>, <a href="/wiki/%EC%A0%9C%ED%8A%B8%EC%98%A4%EB%94%94%EC%98%A4" title="제트오디오">제트오디오</a>, <a href="/wiki/VLC_%EB%AF%B8%EB%94%94%EC%96%B4_%ED%94%8C%EB%A0%88%EC%9D%B4%EC%96%B4" title="VLC 미디어 플레이어">VLC 미디어 플레이어</a> 등이 vorbis 재생을 지원한다.</p>
