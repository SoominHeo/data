<h1 class="firstHeading" id="firstHeading" lang="ko">기원전 제4천년기</h1>
<div class="mw-body-content" id="bodyContent">
<div id="siteSub">위키백과, 우리 모두의 백과사전.</div>
<div id="contentSub"></div>
<div class="mw-jump" id="jump-to-nav">
					이동:					<a href="#mw-head">둘러보기</a>, 					<a href="#p-search">검색</a>
</div>
<div class="mw-content-ltr" dir="ltr" id="mw-content-text" lang="ko"><script>function mfTempOpenSection(id){var block=document.getElementById("mf-section-"+id);block.className+=" open-block";block.previousSibling.className+=" open-block";}</script><p><b>기원전 제4천년기</b>는 기원전 4000년부터 기원전 3001년까지를 일컫는다. 이 시기에 인간의 문화는 몇가지 중대한 변화를 맞이한다. <a class="mw-redirect" href="/wiki/%EC%B2%AD%EB%8F%99%EA%B8%B0" title="청동기">청동기</a>의 시작과 <a href="/wiki/%EB%AC%B8%EC%9E%90" title="문자">문자</a>의 발명이 그것으로, 특히 문자는 <a href="/wiki/%EC%97%AD%EC%82%AC_%EC%8B%9C%EB%8C%80" title="역사 시대">역사 시대</a>로 들어서는데 큰 역할을 하게 되었다.</p>
<div class="thumb tright">

<div class="thumbcaption">

<a class="new" href="/w/index.php?title=%EB%AA%AC%ED%85%8C_%EB%8B%A4%EC%BD%94%EB%94%94&amp;action=edit&amp;redlink=1" title="몬테 다코디 (없는 문서)">몬테 다코디</a>는 <a href="/wiki/%EC%9D%B4%ED%83%88%EB%A6%AC%EC%95%84" title="이탈리아">이탈리아</a> <a class="mw-redirect" href="/wiki/%EC%82%AC%EB%A5%B4%EB%8D%B0%EB%83%90_%EC%A3%BC" title="사르데냐 주">사르데냐 주</a>에 위치한 고고학 유적지다. 기원전 4000년경 건축.</div>

<p>기원전 제4천년기에는 <a href="/wiki/%EC%88%98%EB%A9%94%EB%A5%B4" title="수메르">수메르</a>의 도시 국가들과 <a href="/wiki/%EA%B3%A0%EB%8C%80_%EC%9D%B4%EC%A7%91%ED%8A%B8" title="고대 이집트">이집트</a> 왕국이 건국되고 이후 큰 문명으로 발전하게 되었다. <a class="mw-redirect" href="/wiki/%EB%86%8D%EA%B2%BD" title="농경">농경</a>은 <a href="/wiki/%EC%9C%A0%EB%9D%BC%EC%8B%9C%EC%95%84" title="유라시아">유라시아</a> 전역에 널리 퍼져나갔다. 이 1000년간 <a href="/wiki/%EC%84%B8%EA%B3%84_%EC%9D%B8%EA%B5%AC" title="세계 인구">세계 인구</a>는 700만 명에서 1400만 명 정도로 늘어나 두 배로 증가했다.</p>
<h2><span class="mw-headline" id=".EB.AC.B8.ED.99.94">문화</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EA%B8%B0%EC%9B%90%EC%A0%84_%EC%A0%9C4%EC%B2%9C%EB%85%84%EA%B8%B0&amp;action=edit&amp;section=1" title="부분 편집: 문화">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<ul>
<li><a href="/wiki/%EB%A9%94%EC%86%8C%ED%8F%AC%ED%83%80%EB%AF%B8%EC%95%84" title="메소포타미아">메소포타미아</a>
<ul>
<li><a href="/wiki/%EC%9A%B0%EB%A3%A8%ED%81%AC_%EA%B8%B0" title="우루크 기">우루크 기</a> (<a href="/wiki/%EC%88%98%EB%A9%94%EB%A5%B4" title="수메르">수메르</a>의 선대 문화) - 기원전 4100~3100년.</li>
<li><a href="/wiki/%EC%9B%90%EC%8B%9C_%EC%97%98%EB%9E%8C" title="원시 엘람">원시 엘람</a> - 기원전 3200년~.</li>
<li><a href="/wiki/%ED%9B%84%EB%A5%B4%EB%A6%AC%EC%9D%B8" title="후르리인">후르리인</a>들이 <a class="new" href="/w/index.php?title=%EC%9A%B0%EB%A5%B4%EC%BC%80%EC%89%AC&amp;action=edit&amp;redlink=1" title="우르케쉬 (없는 문서)">우르케쉬</a> (시리아 북부)를 건국.</li>
</ul>
</li>
<li>아시아
<ul>
<li>한반도에서는 <a href="/wiki/%EB%B9%97%EC%82%B4%EB%AC%B4%EB%8A%AC%ED%86%A0%EA%B8%B0_%EC%8B%9C%EB%8C%80" title="빗살무늬토기 시대">빗살무늬토기 시대</a>를 이어갔다. (기원전 8000년~2000년)</li>
<li>중국에서는 신석기 문명이 출현했다. 이들은 천과 도자기 (주로 <a href="/wiki/%EC%96%91%EC%82%AC%EC%98%A4_%EB%AC%B8%ED%99%94" title="양사오 문화">양사오 문화</a>와 룽산 문화)를 제작하고 삼베옷을 입었으며 돼지와 개를 길렀다.</li>
<li>베트남에서는 처음으로 청동기 문화가 출현했다. 이를 <a class="new" href="/w/index.php?title=%EB%8F%99%EB%8B%A4%EC%9A%B0_%EB%AC%B8%ED%99%94&amp;action=edit&amp;redlink=1" title="동다우 문화 (없는 문서)">동다우 문화</a> (기원전 4000년~<a class="mw-redirect" href="/wiki/%EA%B8%B0%EC%9B%90%EC%A0%84_2500%EB%85%84" title="기원전 2500년">2500년</a>)라고 부르며 화려한 청동기가 많이 만들어졌다.</li>
<li>오스트로네시아인들이 발전된 항해술을 바탕으로 중국 대륙에서 <a href="/wiki/%ED%83%80%EC%9D%B4%EC%99%84" title="타이완">타이완</a> 섬으로 이동 (기원전 4000~3000년).</li>
</ul>
</li>
<li><a class="new" href="/w/index.php?title=%EC%9C%A0%EB%9F%BD%EC%9D%98_%EC%8B%A0%EC%84%9D%EA%B8%B0_%EC%8B%9C%EB%8C%80&amp;action=edit&amp;redlink=1" title="유럽의 신석기 시대 (없는 문서)">신석기 시대 유럽</a>과 서유라시아
<ul>
<li><a class="mw-redirect" href="/wiki/%ED%81%AC%EB%A0%88%ED%83%80_%EC%84%AC" title="크레타 섬">크레타 섬</a> - <a class="mw-redirect" href="/wiki/%EB%AF%B8%EB%85%B8%EC%95%84_%EB%AC%B8%EB%AA%85" title="미노아 문명">미노아 문명</a> 발생.</li>
<li><a class="new" href="/w/index.php?title=%EC%96%8C%EB%82%98_%EB%AC%B8%ED%99%94&amp;action=edit&amp;redlink=1" title="얌나 문화 (없는 문서)">얌나 문화</a> (쿠르간 문화) - <a class="new" href="/w/index.php?title=%EC%8A%A4%EB%A0%88%EB%93%9C%EB%8B%88_%EC%8A%A4%ED%86%A0%EA%B7%B8_%EB%AC%B8%ED%99%94&amp;action=edit&amp;redlink=1" title="스레드니 스토그 문화 (없는 문서)">스레드니 스토그 문화</a>의 뒤를 잇는 문화. <a class="new" href="/w/index.php?title=%EC%BF%A0%EB%A5%B4%EA%B0%84_%ED%95%99%EC%84%A4&amp;action=edit&amp;redlink=1" title="쿠르간 학설 (없는 문서)">쿠르간 학설</a>에서는 <a class="new" href="/w/index.php?title=%EC%9B%90%EC%9D%B8%EB%8F%84%EC%9C%A0%EB%9F%BD%EC%9D%B8&amp;action=edit&amp;redlink=1" title="원인도유럽인 (없는 문서)">원인도유럽인</a> 문화의 중심지로 봄.</li>
<li><a class="new" href="/w/index.php?title=%EC%88%98%ED%98%88_%EB%AC%B4%EB%8D%A4&amp;action=edit&amp;redlink=1" title="수혈 무덤 (없는 문서)">수혈 무덤</a> (쿠르간 문화) - <a class="new" href="/w/index.php?title=%EC%8A%A4%EB%A0%88%EB%93%9C%EB%8B%88_%EC%8A%A4%ED%86%A0%ED%81%AC_%EB%AC%B8%ED%99%94&amp;action=edit&amp;redlink=1" title="스레드니 스토크 문화 (없는 문서)">스레드니 스토크 문화</a>의 뒤를 잇는 유적지. <a class="new" href="/w/index.php?title=%EA%B5%AC%EC%84%9D%EA%B8%B0_%EC%A7%80%EC%86%8D%EB%A1%A0&amp;action=edit&amp;redlink=1" title="구석기 지속론 (없는 문서)">구석기 지속론</a>에서는 <a class="new" href="/w/index.php?title=%EC%9B%90%EC%9D%B8%EB%8F%84%EC%9C%A0%EB%9F%BD%EC%9D%B8&amp;action=edit&amp;redlink=1" title="원인도유럽인 (없는 문서)">원인도유럽인</a> 문화의 중심지로 봄.</li>
<li><a href="/wiki/%EC%BA%85%EC%B9%B4%EC%8A%A4" title="캅카스">캅카스</a>의 <a class="new" href="/w/index.php?title=%EB%A9%94%EC%9D%B4%EC%BD%94%ED%94%84_%EB%AC%B8%ED%99%94&amp;action=edit&amp;redlink=1" title="메이코프 문화 (없는 문서)">메이코프 문화</a> (기원전 3700~3000년경) - 당대 쿠르간 문화에 속했으며 유럽의 <a href="/wiki/%EC%B2%AD%EB%8F%99" title="청동">청동</a> 제작 및 <a href="/wiki/%EC%B2%AD%EB%8F%99%EA%B8%B0_%EC%8B%9C%EB%8C%80" title="청동기 시대">청동기 시대</a>의 기원지로 거론되는 곳 중 하나.</li>
<li><a class="new" href="/w/index.php?title=%EB%B9%88%EC%B0%A8_%EB%AC%B8%ED%99%94&amp;action=edit&amp;redlink=1" title="빈차 문화 (없는 문서)">빈차 문화</a></li>
<li><a class="new" href="/w/index.php?title=%EC%95%84%ED%8C%8C%EB%82%98%EC%84%B8%EB%B3%B4_%EB%AC%B8%ED%99%94&amp;action=edit&amp;redlink=1" title="아파나세보 문화 (없는 문서)">아파나세보 문화</a> (기원전 3500~2500년경) - 시베리아, 몽골, 신장, 카자흐스탄 지역에 있었던 후기 구리 시대 및 초기 청동기 시대 문화.</li>
<li><a class="new" href="/w/index.php?title=%ED%8F%B0%EC%8A%A4%EC%B9%B4%EC%8A%A4%ED%94%BC_%EC%8A%A4%ED%85%9D_%EC%A7%80%EB%8C%80&amp;action=edit&amp;redlink=1" title="폰스카스피 스텝 지대 (없는 문서)">얌나/쿠르간 문화</a> (기원전 3500–2300년경) - 폰스카스피 지역 (흑해 동부).</li>
<li><a class="new" href="/w/index.php?title=%EC%BF%A0%EB%9D%BC-%EC%95%84%EB%9D%BD%EC%84%B8%EC%8A%A4_%EB%AC%B8%ED%99%94&amp;action=edit&amp;redlink=1" title="쿠라-아락세스 문화 (없는 문서)">쿠라-아락세스 문화</a> (기원전 3400~2000년경) — 아라라트 평원에 있던 초창기 문화</li>
</ul>
</li>
<li>유럽 본토

<li><a class="new" href="/w/index.php?title=%EA%B8%B0%EC%9B%90%EC%A0%84_3250%EB%85%84&amp;action=edit&amp;redlink=1" title="기원전 3250년 (없는 문서)">기원전 3250년</a>경
<ul>
<li><a href="/wiki/%EA%B3%A0%EB%8C%80_%EA%B7%BC%EB%8F%99" title="고대 근동">고대 근동</a> 지역에서 손물레 발명.</li>

<ul>
<li><a href="/wiki/%EC%9C%A0%EB%9F%BD" title="유럽">유럽</a>에서 처음으로 바퀴 출현. <a href="/wiki/%EC%8A%AC%EB%A1%9C%EB%B2%A0%EB%8B%88%EC%95%84" title="슬로베니아">슬로베니아</a> <a class="new" href="/w/index.php?title=%EB%A5%98%EB%B8%94%EB%9E%B4%EB%82%98_%EC%8A%B5%EC%A7%80&amp;action=edit&amp;redlink=1" title="류블랴나 습지 (없는 문서)">류블랴나 습지</a>에서 발견된 나무 바퀴인 <a class="new" href="/w/index.php?title=%EB%A5%98%EB%B8%94%EB%9E%B4%EB%82%98_%EC%8A%B5%EC%A7%80_%EB%B0%94%ED%80%B4&amp;action=edit&amp;redlink=1" title="류블랴나 습지 바퀴 (없는 문서)">류블랴나 습지 바퀴</a>가 바로 그것이다.<sup class="reference" id="cite_ref-Slovenia_2-0"><a href="#cite_note-Slovenia-2">[2]</a></sup> 탄소연대 측정으로는 약 5,150년 전의 것으로 추정되며 현재까지 발견된 것 중에서 가장 오래된 나무 바퀴이기도 하다.</li>

<ul>
<li>이 시기 직후 메소포타미아에서 <a class="mw-disambig" href="/wiki/%EC%A3%BC%EC%84%9D" title="주석">주석</a>을 다루기 시작.<sup class="reference" id="cite_ref-3"><a href="#cite_note-3">[3]</a></sup></li>
</ul>
</li>
<li><a href="/wiki/%EB%A9%94%EC%86%8C%ED%8F%AC%ED%83%80%EB%AF%B8%EC%95%84" title="메소포타미아">메소포타미아</a>의 <a href="/wiki/%EC%88%98%EB%A9%94%EB%A5%B4" title="수메르">수메르</a>와 <a href="/wiki/%EC%9D%B4%EC%A7%91%ED%8A%B8" title="이집트">이집트</a>에서 <a href="/wiki/%EB%8F%84%EC%8B%9C%ED%99%94" title="도시화">도시화</a>가 시작.</li>
<li><a href="/wiki/%EC%9A%B0%EB%A3%A8%ED%81%AC" title="우루크">우루크</a>와 <a class="mw-disambig" href="/wiki/%EC%88%98%EC%82%AC" title="수사">수사</a>의 도시국가에서 처음으로 <a href="/wiki/%EB%AC%B8%EC%9E%90" title="문자">문자</a>를 사용 (<a href="/wiki/%EC%90%90%EA%B8%B0_%EB%AC%B8%EC%9E%90" title="쐐기 문자">쐐기 문자</a>). <a href="/wiki/%EC%9D%B4%EC%A7%91%ED%8A%B8" title="이집트">이집트</a>에서는 <a class="mw-redirect" href="/wiki/%EC%9D%B4%EC%A7%91%ED%8A%B8_%EC%83%81%ED%98%95%EB%AC%B8%EC%9E%90" title="이집트 상형문자">상형문자</a>가 본격적으로 발전.
<ul>
<li>이 시기 수메르 문자는 점토판에 기록됐으며 2,000여개의 상징부호가 등장했다.</li>
</ul>
</li>
<li>지금의 <a href="/wiki/%EB%82%A8%EB%B6%80_%EC%97%B0%EB%B0%A9%EA%B4%80%EA%B5%AC" title="남부 연방관구">러시아 남부</a>와 <a href="/wiki/%EC%9A%B0%ED%81%AC%EB%9D%BC%EC%9D%B4%EB%82%98" title="우크라이나">우크라이나</a>의 <a class="new" href="/w/index.php?title=%EC%BF%A0%EB%A5%B4%EA%B0%84_%EB%AC%B8%ED%99%94&amp;action=edit&amp;redlink=1" title="쿠르간 문화 (없는 문서)">쿠르간 문화</a>에서 처음으로 <a class="new" href="/w/index.php?title=%EB%A7%90%EC%9D%98_%EA%B0%80%EC%B6%95%ED%99%94&amp;action=edit&amp;redlink=1" title="말의 가축화 (없는 문서)">말 사육</a>을 시작한 것으로 추정.</li>
<li><a href="/wiki/%EB%82%98%EC%9D%BC_%EA%B0%95" title="나일 강">나일 강</a>에서 <a href="/wiki/%EB%8F%9B" title="돛">돛</a> 사용 시작.</li>
<li>잉글랜드에서 <a class="new" href="/w/index.php?title=%EC%8A%A4%EC%9C%84%ED%8A%B8_%ED%8A%B8%EB%9E%99&amp;action=edit&amp;redlink=1" title="스위트 트랙 (없는 문서)">스위트 트랙</a> 건설. 세계 최초로 기술공법으로 만들어진 도로.</li>
<li><a href="/wiki/%EC%9D%B8%EB%8D%94%EC%8A%A4_%EB%AC%B8%EB%AA%85" title="인더스 문명">인더스 문명</a>에서 <a href="/wiki/%EB%B0%B0%EC%88%98_(%EA%B3%B5%ED%95%99)" title="배수 (공학)">배수로</a>와 저수지, 하수처리장 출현.</li>
<li><a href="/wiki/%EC%88%98%EB%A9%94%EB%A5%B4" title="수메르">수메르</a>에서 <a href="/wiki/%EB%B9%97%EB%A9%B4" title="빗면">빗면</a>과 <a href="/wiki/%EC%A7%80%EB%A0%88" title="지레">지레</a>를 이용해 <a href="/wiki/%EB%8C%90" title="댐">댐</a>, <a href="/wiki/%EC%9A%B4%ED%95%98" title="운하">운하</a>, <a class="new" href="/w/index.php?title=%EC%84%9D%EC%A1%B0%EB%AC%BC&amp;action=edit&amp;redlink=1" title="석조물 (없는 문서)">석조물</a> 건설.</li>
<li><a href="/wiki/%EA%B5%AC%EB%A6%AC" title="구리">구리</a>가 도구와 무기로 사용.</li>
<li><a href="/wiki/%EC%B2%AD%EB%8F%99" title="청동">청동</a>이 <a class="new" href="/w/index.php?title=%EB%A9%94%EC%9D%B4%EC%BD%94%ED%94%84_%EB%AC%B8%ED%99%94&amp;action=edit&amp;redlink=1" title="메이코프 문화 (없는 문서)">메이코프 문화</a>에서 사용.</li>
<li><a class="new" href="/w/index.php?title=%EC%9D%B4%EC%A7%91%ED%8A%B8_%ED%94%BC%EB%9D%BC%EB%AF%B8%EB%93%9C&amp;action=edit&amp;redlink=1" title="이집트 피라미드 (없는 문서)">이집트 피라미드</a>의 원조인 <a href="/wiki/%EB%A7%88%EC%8A%A4%ED%83%80%EB%B0%94" title="마스타바">마스타바</a> 건설.</li>
<li><a href="/wiki/%EC%8A%A4%ED%86%A4%ED%97%A8%EC%A7%80" title="스톤헨지">스톤헨지</a>의 첫단계 (흙으로 쌓은 원형 둑과 수로) 건설. <a class="mw-redirect" href="/wiki/%EA%B8%B0%EC%9B%90%EC%A0%84_3100%EB%85%84" title="기원전 3100년">기원전 3100년</a>경.</li>
<li>아일랜드의 <a class="new" href="/w/index.php?title=%EC%84%B8%EC%9D%B4%EB%8D%B0_%ED%95%84%EB%93%9C&amp;action=edit&amp;redlink=1" title="세이데 필드 (없는 문서)">세이데 필드</a> 조성. 세계에서 가장 오래된 사육장이라는 주장이 제기되고 있다.</li>
<li>이집트와 남동유럽에서 흰색 <a href="/wiki/%EB%8F%84%EC%9E%90%EA%B8%B0" title="도자기">도자기</a>를 처음으로 제작.</li>
<li>이집트에서 <a href="/wiki/%ED%95%98%ED%94%84" title="하프">하프</a>와 <a href="/wiki/%ED%94%BC%EB%A6%AC" title="피리">피리</a> 연주. <a href="/wiki/%EB%A6%AC%EB%9D%BC_(%EC%95%85%EA%B8%B0)" title="리라 (악기)">리라</a>와 이중 피리 (<a class="new" href="/w/index.php?title=%EC%95%84%EB%A5%B4%EA%B5%B4&amp;action=edit&amp;redlink=1" title="아르굴 (없는 문서)">아르굴</a>, <a class="new" href="/w/index.php?title=%EB%AF%B8%EC%A5%90%EC%A6%88&amp;action=edit&amp;redlink=1" title="미쥐즈 (없는 문서)">미쥐즈</a>)도 만들어졌다.</li>
<li>이집트와 수메르에서 구리합금 이용. 금과 은 합금법도 널리 알려졌다.</li>
<li>이집트에서 초창기 <a href="/wiki/%EA%B8%B0%EC%88%98%EB%B2%95" title="기수법">기수법</a> 사용.</li>
<li>중동에서 <a href="/wiki/%EC%95%84%EB%A7%88%ED%8F%AC" title="아마포">아마포</a> 생산.</li>
