<p><b>기침</b>은 갑작스럽게 숨을 내뿜는 반사 행동으로, <a class="mw-redirect" href="/wiki/%EC%88%A8%EA%B8%B8" title="숨길">기도</a>에 놓인 분비물, 자극을 주는 물질, 외부 먼지, <a href="/wiki/%EB%AF%B8%EC%83%9D%EB%AC%BC" title="미생물">미생물</a>들을 제거한다. 종종 반복적으로 연달아 행해지기도 한다. <a class="new" href="/w/index.php?title=%EA%B8%B0%EC%B9%A8_%EB%B0%98%EC%82%AC&amp;action=edit&amp;redlink=1" title="기침 반사 (없는 문서)">기침 반사</a>는 들이쉬고, <a class="mw-disambig" href="/wiki/%EC%84%B1%EB%AC%B8" title="성문">성문</a>을 향해 내쉬고, 성문을 열면서 폐로부터 공기를 강하게 내보내는 3단계로 구성되며, 종종 특유의 소리를 동반하기도 한다.<sup class="reference" id="cite_ref-Lancet-causes_1-0"><a href="#cite_note-Lancet-causes-1">[1]</a></sup> 기침은 보통 본인의 의사와 관계 없이 발생하나, 일부러 기침(헛기침)을 하기도 한다.</p>
<p>기침이 잦다는 것은 보통 질병이 있음을 시사한다. 많은 <a href="/wiki/%EB%B0%94%EC%9D%B4%EB%9F%AC%EC%8A%A4" title="바이러스">바이러스</a>와 <a href="/wiki/%EC%84%B8%EA%B7%A0" title="세균">세균</a>은 <a href="/wiki/%EC%88%99%EC%A3%BC" title="숙주">숙주</a>가 기침을 하는 것으로 이익을 얻는다. 병균이 새로운 숙주에 전염되는 것을 돕는 것이다. 대부분의 경우 기침은 기도 감염에 의한 것이나, 숨 막힘, <a href="/wiki/%ED%9D%A1%EC%97%B0" title="흡연">흡연</a>, <a href="/wiki/%EB%8C%80%EA%B8%B0_%EC%98%A4%EC%97%BC" title="대기 오염">대기 오염</a><sup class="reference" id="cite_ref-Lancet-causes_1-1"><a href="#cite_note-Lancet-causes-1">[1]</a></sup>, <a href="/wiki/%EC%B2%9C%EC%8B%9D" title="천식">천식</a>, <a class="mw-redirect" href="/wiki/%EC%97%AD%EB%A5%98%EC%84%B1_%EC%8B%9D%EB%8F%84%EC%97%BC" title="역류성 식도염">역류성 식도염</a>, <a class="new" href="/w/index.php?title=%ED%9B%84%EB%B9%84%EB%A3%A8&amp;action=edit&amp;redlink=1" title="후비루 (없는 문서)">후비루</a>(Post-nasal drip), <a class="mw-redirect" href="/wiki/%EB%A7%8C%EC%84%B1_%EA%B8%B0%EA%B4%80%EC%A7%80%EC%97%BC" title="만성 기관지염">만성 기관지염</a>, 폐 종양, <a href="/wiki/%EC%8B%AC%EB%B6%80%EC%A0%84" title="심부전">심부전</a>(heart failure), 그리고, <a href="/wiki/ACE_%EC%96%B5%EC%A0%9C%EC%A0%9C" title="ACE 억제제">ACE 억제제</a>와 같은 약제에 의해서도 발생한다.</p>
<p>치료는 원인에 중점을 두어야 한다. 예를 들면, 흡연 중지나 ACE 억제제의 사용 중단 등이다.</p>
<h2><span class="mw-headline" id=".EC.A2.85.EB.A5.98">종류</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EA%B8%B0%EC%B9%A8&amp;action=edit&amp;section=1" title="부분 편집: 종류">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>기침은 또한 그 지속기간, 특징, 시기에 따라 나눌 수 있다.<sup class="reference" id="cite_ref-Review10_2-0"><a href="#cite_note-Review10-2">[2]</a></sup> 기간에 따라 3주보다 짧으면 <a href="/wiki/%EA%B8%89%EC%84%B1" title="급성">급성</a>, 3에서 8주 사이이면 아급성, 8주보다 길면 <a href="/wiki/%EB%A7%8C%EC%84%B1" title="만성">만성</a>으로 분류한다.<sup class="reference" id="cite_ref-Review10_2-1"><a href="#cite_note-Review10-2">[2]</a></sup> 기침은 또한 건성(乾性)과 습성(濕性)으로 나눌 수 있다. 습성 기침은 <a href="/wiki/%EA%B0%80%EB%9E%98_(%EC%9D%98%ED%95%99)" title="가래 (의학)">가래</a>를 동반하는 것이며, 건성 기침은 그렇지 않은 것이다. 기침은 밤이나 낮에만 있을 수 있고, 낮과 밤 모두 있을 수 있다.<sup class="reference" id="cite_ref-Review10_2-2"><a href="#cite_note-Review10-2">[2]</a></sup></p>
