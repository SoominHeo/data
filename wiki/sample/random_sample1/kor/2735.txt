<p><b>기타카미 선</b>(<span style="font-size: smaller;"><a href="/wiki/%EC%9D%BC%EB%B3%B8%EC%96%B4" title="일본어">일본어</a>:</span> <span lang="ja" xml:lang="ja">北上線</span>)은 <a href="/wiki/%EC%9D%BC%EB%B3%B8" title="일본">일본</a> <a href="/wiki/%EC%9D%B4%EC%99%80%ED%85%8C_%ED%98%84" title="이와테 현">이와테 현</a> <a href="/wiki/%EA%B8%B0%ED%83%80%EC%B9%B4%EB%AF%B8_%EC%8B%9C" title="기타카미 시">기타카미 시</a>의 <a href="/wiki/%EA%B8%B0%ED%83%80%EC%B9%B4%EB%AF%B8_%EC%97%AD" title="기타카미 역">기타카미 역</a>과 <a href="/wiki/%EC%95%84%ED%82%A4%ED%83%80_%ED%98%84" title="아키타 현">아키타 현</a> <a href="/wiki/%EC%9A%94%EC%BD%94%ED%85%8C_%EC%8B%9C" title="요코테 시">요코테 시</a>에 있는 <a href="/wiki/%EC%9A%94%EC%BD%94%ED%85%8C_%EC%97%AD" title="요코테 역">요코테 역</a>을 잇는 <a class="mw-redirect" href="/wiki/%EB%8F%99%EC%9D%BC%EB%B3%B8_%EC%97%AC%EA%B0%9D%EC%B2%A0%EB%8F%84" title="동일본 여객철도">동일본 여객철도</a>의 철도 노선(지방 교통선)이다.</p>
<h2><span class="mw-headline" id=".EB.85.B8.EC.84.A0_.EC.A0.95.EB.B3.B4">노선 정보</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EA%B8%B0%ED%83%80%EC%B9%B4%EB%AF%B8_%EC%84%A0&amp;action=edit&amp;section=1" title="부분 편집: 노선 정보">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<ul>
<li>관할 (사업 종별) : 동일본 여객철도 (<a href="/wiki/%EC%9D%BC%EB%B3%B8%EC%9D%98_%EC%B2%A0%EB%8F%84%EC%82%AC%EC%97%85%EC%9E%90" title="일본의 철도사업자">제 1 종 철도사업자</a>) · 일본화물철도 (<a href="/wiki/%EC%9D%BC%EB%B3%B8%EC%9D%98_%EC%B2%A0%EB%8F%84%EC%82%AC%EC%97%85%EC%9E%90" title="일본의 철도사업자">제 2 종 철도사업자</a>)</li>
<li>구간 (영업 거리) : 기타카미 역 - 요코테 역 61.1km</li>
<li><a href="/wiki/%EA%B6%A4%EA%B0%84" title="궤간">궤간</a> : 1067mm</li>
<li>역 수 : 17(기 · 종점역 포함)
<ul>
<li>기타카미 선 소속 역으로 한정된 경우, 기종점역(기타카미 역은 도호쿠 본선, 요코테 역은 오우 본선의 소속)이 제외되어, 15역이 된다.</li>
</ul>
</li>
<li>복선 구간 : 없음 (전 구간 <a href="/wiki/%EB%8B%A8%EC%84%A0_(%EC%B2%A0%EB%8F%84)" title="단선 (철도)">단선</a>)</li>
<li>전철화 구간 : 없음 (전 구간 비전철화)</li>
<li><a href="/wiki/%ED%8F%90%EC%83%89" title="폐색">폐색방식</a> : 특수 자동 폐색식(궤도 회로 검지식)</li>
<li>종합지령소 :
<ul>
<li>기타카미 역 - 구로사와 역 간 : 모리오카 종합 지령실</li>
<li>구로사와 역 - 요코테 역 간 : 아키타 종합 지령실</li>
</ul>
</li>
<li>최고속도 : 85km/h</li>
</ul>
<p>기타카미 역 - 유다코겐 역 간은 <a class="new" href="/w/index.php?title=%EB%8F%99%EC%9D%BC%EB%B3%B8_%EC%97%AC%EA%B0%9D%EC%B2%A0%EB%8F%84_%EB%AA%A8%EB%A6%AC%EC%98%A4%EC%B9%B4_%EC%A7%80%EC%82%AC&amp;action=edit&amp;redlink=1" title="동일본 여객철도 모리오카 지사 (없는 문서)">동일본 여객철도 모리오카 지사</a>, 구로사와 역 - 요코테 역 간은 <a class="new" href="/w/index.php?title=%EB%8F%99%EC%9D%BC%EB%B3%B8_%EC%97%AC%EA%B0%9D%EC%B2%A0%EB%8F%84_%EC%95%84%ED%82%A4%ED%83%80_%EC%A7%80%EC%82%AC&amp;action=edit&amp;redlink=1" title="동일본 여객철도 아키타 지사 (없는 문서)">동일본 여객철도 아키타 지사</a>의 관할이다.</p>
