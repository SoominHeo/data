<h1 class="firstHeading" id="firstHeading" lang="ko">기타칸바라 군</h1>
<div class="mw-body-content" id="bodyContent">

기타칸바라 군의 위치</div>

<p><b>기타칸바라 군</b>(<span style="font-size: smaller;"><a href="/wiki/%EC%9D%BC%EB%B3%B8%EC%96%B4" title="일본어">일본어</a>:</span> <span lang="ja" xml:lang="ja">北蒲原郡, きたかんばらぐん</span>)은 <a href="/wiki/%EC%9D%BC%EB%B3%B8" title="일본">일본</a> <a href="/wiki/%EB%8B%88%EA%B0%80%ED%83%80_%ED%98%84" title="니가타 현">니가타 현</a>의 군이다. 면적은 37.99km²이고 인구는 <a href="/wiki/2009%EB%85%84" title="2009년">2009년</a> <a href="/wiki/12%EC%9B%94_1%EC%9D%BC" title="12월 1일">12월 1일</a> 기준으로 13,707명이다.</p>
<p>한 개의 정으로 이루어져 있다.</p>
<ul>
<li><a href="/wiki/%EC%84%B8%EC%9D%B4%EB%A1%9C_%EC%A0%95" title="세이로 정">세이로 정</a>(<span lang="ja" xml:lang="ja">聖籠町</span>)</li>
</ul>
<h2><span class="mw-headline" id=".EA.B0.9C.EC.9A.94">개요</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EA%B8%B0%ED%83%80%EC%B9%B8%EB%B0%94%EB%9D%BC_%EA%B5%B0&amp;action=edit&amp;section=1" title="부분 편집: 개요">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>1878년에 제정된 군구정촌 편제법에 의해서 구 간바라 군을 기타칸바라군, <a class="new" href="/w/index.php?title=%EB%82%98%EC%B9%B4%EC%B9%B8%EB%B0%94%EB%9D%BC_%EA%B5%B0&amp;action=edit&amp;redlink=1" title="나카칸바라 군 (없는 문서)">나카칸바라 군</a>, <a href="/wiki/%ED%9E%88%EA%B0%80%EC%8B%9C%EC%B9%B8%EB%B0%94%EB%9D%BC_%EA%B5%B0" title="히가시칸바라 군">히가시칸바라 군</a>, <a href="/wiki/%EB%8B%88%EA%B0%80%ED%83%80_%EC%8B%9C" title="니가타 시">니가타 구</a>, <a href="/wiki/%EB%8B%88%EC%8B%9C%EC%B9%B8%EB%B0%94%EB%9D%BC_%EA%B5%B0" title="니시칸바라 군">니시칸바라 군</a>, <a href="/wiki/%EB%AF%B8%EB%82%98%EB%AF%B8%EC%B9%B8%EB%B0%94%EB%9D%BC_%EA%B5%B0" title="미나미칸바라 군">미나미칸바라 군</a>의 6개로 분할해 성립하였다. 성립 당시의 군역은 현재의 군역을 포함해 현재의 <a href="/wiki/%EC%8B%9C%EB%B0%94%ED%83%80_%EC%8B%9C" title="시바타 시">시바타 시</a>, <a href="/wiki/%EC%95%84%EA%B0%80%EB%85%B8_%EC%8B%9C" title="아가노 시">아가노 시</a>, <a href="/wiki/%EB%8B%A4%EC%9D%B4%EB%82%98%EC%9D%B4_%EC%8B%9C" title="다이나이 시">다이나이 시</a>, <a href="/wiki/%EB%8B%88%EA%B0%80%ED%83%80_%EC%8B%9C" title="니가타 시">니가타 시</a>의 일부(<a href="/wiki/%EC%95%84%EA%B0%80%EB%85%B8_%EA%B0%95" title="아가노 강">아가노 강</a> 이북과 <a href="/wiki/%EB%8B%88%EA%B0%80%ED%83%80_%EA%B3%B5%ED%95%AD" title="니가타 공항">니가타 공항</a> 주변의 마쓰하마 정), 히가시칸바라 군 <a href="/wiki/%EC%95%84%EA%B0%80_%EC%A0%95" title="아가 정">아가 정</a>의 일부를 포함하고 있었다.</p>
<p>덧붙여 옛 군 관공서는 시바타 본촌(현 시바타 시)에 놓였다.</p>
<h2><span class="mw-headline" id=".EC.97.AD.EC.82.AC">역사</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EA%B8%B0%ED%83%80%EC%B9%B8%EB%B0%94%EB%9D%BC_%EA%B5%B0&amp;action=edit&amp;section=2" title="부분 편집: 역사">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>메이지의 대합병에 의해 아이즈 가도(현도 14호선) 주변의 촌락의 일부가 히가시칸바라 군 미카와 촌(현 : 아가 정)에 편입되었다. 전후 <a href="/wiki/1947%EB%85%84" title="1947년">1947년</a> <a href="/wiki/1%EC%9B%94_1%EC%9D%BC" title="1월 1일">1월 1일</a>, 시바타 정이 시로 승격했다. 그 후 쇼와의 대합병기에 시바타 시·니가타 시에 가까운 지역은 두 시로 편입되었고 나머지는 나카조 정·스이바라 정·구즈쓰카 정을 중심으로서 합병을 거듭했다. <a href="/wiki/1970%EB%85%84" title="1970년">1970년</a> <a href="/wiki/11%EC%9B%94_1%EC%9D%BC" title="11월 1일">11월 1일</a>, 구즈쓰카 정을 중심으로 한 도요사카 정이 시로 승격해 <a href="/wiki/%EB%8F%84%EC%9A%94%EC%82%AC%EC%B9%B4_%EC%8B%9C" title="도요사카 시">도요사카 시</a>(현 : 니가타 시)가 성립했다.</p>
<p>헤이세이의 대합병 때는 이하의 합병을 했다.</p>
<ul>
<li>2003년 7월 7일 - 도요우라 정이 시바타 시에 편입되었다.</li>
<li>2004년 4월 1일 - 야스다 정, 스이바라 정, 교가세 촌, 사사카미 촌의 2정 2촌이 합병해 아가노 시가 성립하였다.</li>
<li>2005년 5월 1일 - 시운지 정, 가지카와 촌이 시바타 시에 편입되었다.</li>
<li>2005년 9월 1일 - 나카조 정, 구로가와 촌가 합병해 다이나이 시가 성립하였다.</li>
</ul>
<p>현재 기타칸바라 군에 포함되는 것은 세이로 정뿐이다.</p>
