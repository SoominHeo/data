<p><b>긴코몽구스</b>(<i>Xenogale naso</i>)는 <a href="/wiki/%EC%8B%9D%EC%9C%A1%EB%AA%A9" title="식육목">식육목</a> <a href="/wiki/%EB%AA%BD%EA%B5%AC%EC%8A%A4" title="몽구스">몽구스과</a>에 속하는 포유류의 일종이다. <a href="/wiki/%EC%B9%B4%EB%A9%94%EB%A3%AC" title="카메룬">카메룬</a>과 <a class="mw-redirect" href="/wiki/%EC%BD%A9%EA%B3%A0%EA%B3%B5%ED%99%94%EA%B5%AD" title="콩고공화국">콩고공화국</a>, <a class="mw-redirect" href="/wiki/%EC%BD%A9%EA%B3%A0%EB%AF%BC%EC%A3%BC%EA%B3%B5%ED%99%94%EA%B5%AD" title="콩고민주공화국">콩고민주공화국</a>, <a class="mw-redirect" href="/wiki/%EC%A0%81%EB%8F%84%EA%B8%B0%EB%8B%88" title="적도기니">적도기니</a>, <a href="/wiki/%EA%B0%80%EB%B4%89" title="가봉">가봉</a>, <a href="/wiki/%EC%BC%80%EB%83%90" title="케냐">케냐</a>, <a href="/wiki/%EB%8B%88%EC%A0%9C%EB%A5%B4" title="니제르">니제르</a> 그리고 <a href="/wiki/%ED%83%84%EC%9E%90%EB%8B%88%EC%95%84" title="탄자니아">탄자니아</a>에서 발견된다.<sup class="reference" id="cite_ref-iucn_2-1"><a href="#cite_note-iucn-2">[2]</a></sup> <a href="/wiki/%EB%8A%AA%EB%AA%BD%EA%B5%AC%EC%8A%A4" title="늪몽구스">늪몽구스</a>와 밀접한 관계에 있으며, <b>긴코몽구스속</b>(<i>Xenogale</i>)의 유일종이다. <a href="/wiki/%EB%AA%BD%EA%B5%AC%EC%8A%A4%EC%86%8D" title="몽구스속">몽구스속</a>으로 분류하기도 한다.</p>
