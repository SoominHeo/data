<h2><span class="mw-headline" id=".ED.94.84.EB.A1.9C.EC.95.BC.EA.B5.AC_.EC.84.A0.EC.88.98_.EC.8B.9C.EC.A0.88">프로야구 선수 시절</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EA%B9%80%EA%B2%BD%EB%AC%B8&amp;action=edit&amp;section=1" title="부분 편집: 프로야구 선수 시절">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<h3><span class="mw-headline" id="OB_.EB.B2.A0.EC.96.B4.EC.8A.A4_.EC.8B.9C.EC.A0.88"><a class="mw-redirect" href="/wiki/OB_%EB%B2%A0%EC%96%B4%EC%8A%A4" title="OB 베어스">OB 베어스</a> 시절</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EA%B9%80%EA%B2%BD%EB%AC%B8&amp;action=edit&amp;section=2" title="부분 편집: OB 베어스 시절">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p><a href="/wiki/%EA%B3%A0%EB%A0%A4%EB%8C%80%ED%95%99%EA%B5%90" title="고려대학교">고려대</a> 경영학과 졸업 후 원년 멤버로 입단하여 <a href="/wiki/%EB%91%90%EC%82%B0_%EB%B2%A0%EC%96%B4%EC%8A%A4" title="두산 베어스">OB 베어스</a>의 <a href="/wiki/%ED%8F%AC%EC%88%98" title="포수">포수</a>로 활약했다. 창단 첫 해인 <a href="/wiki/1982%EB%85%84" title="1982년">1982년</a> <a href="/wiki/%ED%88%AC%EC%88%98" title="투수">투수</a> <a href="/wiki/%EB%B0%95%EC%B2%A0%EC%88%9C" title="박철순">박철순</a>과 배터리를 이루어 팀의 <a class="mw-redirect" href="/wiki/%ED%95%9C%EA%B5%AD%EC%8B%9C%EB%A6%AC%EC%A6%88" title="한국시리즈">한국시리즈</a> 초대 우승을 이끌었다. 초대 <a href="/wiki/%EA%B9%80%EC%98%81%EB%8D%95_(%EC%95%BC%EA%B5%AC%EC%9D%B8)" title="김영덕 (야구인)">김영덕</a> 감독 시절에는 주전을 꿰찼으나, <a href="/wiki/%EA%B9%80%EC%84%B1%EA%B7%BC_(%EC%95%BC%EA%B5%AC%EC%9D%B8)" title="김성근 (야구인)">김성근</a> 감독 부임 이후에는 <a href="/wiki/%EC%A1%B0%EB%B2%94%ED%98%84" title="조범현">조범현</a>에게 주전 자리에서 밀리기도 했다.</p>
<h3><span class="mw-headline" id=".ED.83.9C.ED.8F.89.EC.96.91_.EB.8F.8C.ED.95.80.EC.8A.A4_.EC.8B.9C.EC.A0.88"><a class="mw-redirect" href="/wiki/%ED%83%9C%ED%8F%89%EC%96%91_%EB%8F%8C%ED%95%80%EC%8A%A4" title="태평양 돌핀스">태평양 돌핀스</a> 시절</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EA%B9%80%EA%B2%BD%EB%AC%B8&amp;action=edit&amp;section=3" title="부분 편집: 태평양 돌핀스 시절">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p><a href="/wiki/1989%EB%85%84" title="1989년">1989년</a> <a class="mw-redirect" href="/wiki/%ED%83%9C%ED%8F%89%EC%96%91_%EB%8F%8C%ED%95%80%EC%8A%A4" title="태평양 돌핀스">태평양 돌핀스</a>로 자리를 옮긴 <a href="/wiki/%EA%B9%80%EC%84%B1%EA%B7%BC_(%EC%95%BC%EA%B5%AC%EC%9D%B8)" title="김성근 (야구인)">김성근</a> 감독이 이듬해 <a href="/wiki/1990%EB%85%84" title="1990년">1990년</a> <a href="/wiki/%EA%B9%80%EB%8F%99%EA%B8%B0_(%EC%95%BC%EA%B5%AC_%EC%84%A0%EC%88%98)" title="김동기 (야구 선수)">김동기</a>를 받쳐 줄 백업 <a href="/wiki/%ED%8F%AC%EC%88%98" title="포수">포수</a>를 찾던 중 자금 사정이 좋지 않았던 <a href="/wiki/%EB%91%90%EC%82%B0_%EB%B2%A0%EC%96%B4%EC%8A%A4" title="두산 베어스">OB 베어스</a>와의 상황이 맞아 떨어져 <a class="mw-redirect" href="/wiki/%ED%83%9C%ED%8F%89%EC%96%91_%EB%8F%8C%ED%95%80%EC%8A%A4" title="태평양 돌핀스">태평양 돌핀스</a>로 현금 트레이드되었으나, <a href="/wiki/%EB%91%90%EC%82%B0_%EB%B2%A0%EC%96%B4%EC%8A%A4" title="두산 베어스">OB 베어스</a> 시절에 이미 사이가 그다지 좋지 않았던 <a href="/wiki/%EA%B9%80%EC%84%B1%EA%B7%BC_(%EC%95%BC%EA%B5%AC%EC%9D%B8)" title="김성근 (야구인)">김성근</a> 감독과 불편한 관계를 유지하고 만다.<sup class="reference" id="cite_ref-1"><a href="#cite_note-1">[1]</a></sup> 이러한 <a href="/wiki/%EA%B9%80%EC%84%B1%EA%B7%BC_(%EC%95%BC%EA%B5%AC%EC%9D%B8)" title="김성근 (야구인)">김성근</a> 감독과의 악연은 <a href="/wiki/2007%EB%85%84" title="2007년">2007년</a> <a href="/wiki/%EC%9C%A0%EA%B2%A9%EC%88%98" title="유격수">유격수</a> <a href="/wiki/%EC%9D%B4%EB%8C%80%EC%88%98" title="이대수">이대수</a>와 <a href="/wiki/%EB%82%98%EC%A3%BC%ED%99%98" title="나주환">나주환</a>의 트레이드 때에도 나타나기도 했다.<sup class="reference" id="cite_ref-2"><a href="#cite_note-2">[2]</a></sup></p>
<h3><span class="mw-headline" id="OB_.EB.B2.A0.EC.96.B4.EC.8A.A4_.EB.B3.B5.EA.B7.80"><a class="mw-redirect" href="/wiki/OB_%EB%B2%A0%EC%96%B4%EC%8A%A4" title="OB 베어스">OB 베어스</a> 복귀</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EA%B9%80%EA%B2%BD%EB%AC%B8&amp;action=edit&amp;section=4" title="부분 편집: OB 베어스 복귀">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>결국 <a href="/wiki/1991%EB%85%84" title="1991년">1991년</a>에 <a href="/wiki/%EC%99%B8%EC%95%BC%EC%88%98" title="외야수">외야수</a> <a href="/wiki/%EC%86%A1%EC%9E%AC%EB%B0%95" title="송재박">송재박</a>을 상대로 다시 트레이드되어 친정 팀으로 돌아와 1년을 더 뛴 뒤 <a href="/wiki/1991%EB%85%84" title="1991년">1991년</a>에 은퇴했다.</p>
<h2><span class="mw-headline" id=".ED.94.84.EB.A1.9C.EC.95.BC.EA.B5.AC_.EA.B0.90.EB.8F.85_.EC.8B.9C.EC.A0.88">프로야구 감독 시절</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EA%B9%80%EA%B2%BD%EB%AC%B8&amp;action=edit&amp;section=5" title="부분 편집: 프로야구 감독 시절">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<h3><span class="mw-headline" id=".EB.91.90.EC.82.B0_.EB.B2.A0.EC.96.B4.EC.8A.A4_.EC.8B.9C.EC.A0.88">두산 베어스 시절</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EA%B9%80%EA%B2%BD%EB%AC%B8&amp;action=edit&amp;section=6" title="부분 편집: 두산 베어스 시절">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p><a href="/wiki/1992%EB%85%84" title="1992년">1992년</a>부터 2년 간 <a href="/wiki/%EB%AF%B8%EA%B5%AD" title="미국">미국</a> <a href="/wiki/%EC%95%A0%ED%8B%80%EB%9E%9C%ED%83%80" title="애틀랜타">애틀랜타</a>에서 지도자 연수를 받았고, <a href="/wiki/1994%EB%85%84" title="1994년">1994년</a>부터 <a href="/wiki/%EC%82%BC%EC%84%B1_%EB%9D%BC%EC%9D%B4%EC%98%A8%EC%A6%88" title="삼성 라이온즈">삼성</a>의 코치로 일했다. <a href="/wiki/1998%EB%85%84" title="1998년">1998년</a>부터 친정 팀 <a href="/wiki/%EB%91%90%EC%82%B0_%EB%B2%A0%EC%96%B4%EC%8A%A4" title="두산 베어스">두산 베어스</a>의 배터리 코치로 일했다. 2003년 시즌 후 롯데 자이언츠의 수석코치로 내정이 되어 있었으나, <a href="/wiki/%EC%84%A0%EB%8F%99%EC%97%B4" title="선동열">선동열</a> 영입에 실패한 두산 베어스로부터 영입 제의를 받아서,<sup class="reference" id="cite_ref-3"><a href="#cite_note-3">[3]</a></sup> <a href="/wiki/2003%EB%85%84" title="2003년">2003년</a> 10월 <a href="/wiki/%EA%B9%80%EC%9D%B8%EC%8B%9D_(1947%EB%85%84)" title="김인식 (1947년)">김인식</a> 감독의 뒤를 이어 계약기간 2년 계약금 1억 3천만원, 연봉 1억 3천만원(2년 3억 9천만원)의 조건으로 <a href="/wiki/%EB%91%90%EC%82%B0_%EB%B2%A0%EC%96%B4%EC%8A%A4" title="두산 베어스">두산 베어스</a>의 감독으로 선임되었다.<sup class="reference" id="cite_ref-4"><a href="#cite_note-4">[4]</a></sup> <a href="/wiki/2004%EB%85%84" title="2004년">2004년</a>과 <a href="/wiki/2005%EB%85%84" title="2005년">2005년</a> 두 해 모두 팀을 포스트 시즌에 진출시키는 활약을 보여 <a href="/wiki/2005%EB%85%84" title="2005년">2005년</a> 10월 <a href="/wiki/%EB%91%90%EC%82%B0_%EB%B2%A0%EC%96%B4%EC%8A%A4" title="두산 베어스">두산 베어스</a> 역대 감독 최고액인 계약금 2억 원, 연봉 2억 원 등 총 8억 원으로 3년간 재계약했다.</p>
<p><a href="/wiki/2007%EB%85%84" title="2007년">2007년</a>과 <a href="/wiki/2008%EB%85%84" title="2008년">2008년</a>에는 연속으로 <a class="mw-redirect" href="/wiki/%ED%95%9C%EA%B5%AD%EC%8B%9C%EB%A6%AC%EC%A6%88" title="한국시리즈">한국시리즈</a>에 진출하였으나 <a href="/wiki/%EA%B9%80%EC%84%B1%EA%B7%BC_(%EC%95%BC%EA%B5%AC%EC%9D%B8)" title="김성근 (야구인)">김성근</a> 감독의 <a href="/wiki/SK_%EC%99%80%EC%9D%B4%EB%B2%88%EC%8A%A4" title="SK 와이번스">SK 와이번스</a>에 밀려 2년 연속 준우승에 그쳤다. <a href="/wiki/2011%EB%85%84" title="2011년">2011년</a> <a href="/wiki/6%EC%9B%94_13%EC%9D%BC" title="6월 13일">6월 13일</a> 좋지 않았던 팀 분위기와 성적 부진을 이유로 사령탑을 수석코치 <a href="/wiki/%EA%B9%80%EA%B4%91%EC%88%98_(1959%EB%85%84)" title="김광수 (1959년)">김광수</a>에게 넘기고 감독직에서 사임했다.</p>

<p><a href="/wiki/2008%EB%85%84_%ED%95%98%EA%B3%84_%EC%98%AC%EB%A6%BC%ED%94%BD" title="2008년 하계 올림픽">베이징 올림픽</a> 국가대표팀 감독으로 선임, 9전 전승으로 우승하여 올림픽 금메달을 이끌며 감독 경력을 시작한 이래 처음으로 국제 대회 우승을 맛보았다. <a href="/wiki/%EC%98%AC%EB%A6%BC%ED%94%BD" title="올림픽">올림픽</a>에서의 그의 전술은 '믿음의 야구'라는 평을 받았다.<sup class="reference" id="cite_ref-5"><a href="#cite_note-5">[5]</a></sup></p>

<p><a href="/wiki/2013%EB%85%84" title="2013년">2013년</a> <a href="/wiki/5%EC%9B%94_24%EC%9D%BC" title="5월 24일">5월 24일</a> <a href="/wiki/%EA%B4%91%EC%A3%BC%EB%AC%B4%EB%93%B1%EA%B2%BD%EA%B8%B0%EC%9E%A5_%EC%95%BC%EA%B5%AC%EC%9E%A5" title="광주무등경기장 야구장">광주 무등야구장</a>에서 열린 <a href="/wiki/KIA_%ED%83%80%EC%9D%B4%EA%B1%B0%EC%A6%88" title="KIA 타이거즈">KIA 타이거즈</a>와의 경기에서 역대 9번째로 1,000경기 출장을 기록하였다. 이 경기에서 팀 타선 폭발로 10-5로 승리하며 1,000경기 출장을 자축하였다.<sup class="reference" id="cite_ref-7"><a href="#cite_note-7">[7]</a></sup> 2013년 처음으로 1군에 합류한 NC 다이노스를 52승72패4무 승률 4할1푼9리로 7위에 오르며 기대이상 성적을 냈다.</p>

<p><a href="/wiki/2014%EB%85%84" title="2014년">2014년</a> <a href="/wiki/1%EC%9B%94_22%EC%9D%BC" title="1월 22일">1월 22일</a> <a href="/wiki/NC_%EB%8B%A4%EC%9D%B4%EB%85%B8%EC%8A%A4" title="NC 다이노스">NC 다이노스</a>와 3년 연장계약을 체결하였다. 계약 조건은 잔여기간 포함 3년(2014년 ~ 2016년) 계약금 5억원, 연봉 4억원으로 총액 17억원이었다.<sup class="reference" id="cite_ref-8"><a href="#cite_note-8">[8]</a></sup> 2014년 <a href="/wiki/6%EC%9B%94_15%EC%9D%BC" title="6월 15일">6월 15일</a> <a class="mw-redirect" href="/wiki/%EB%A7%88%EC%82%B0%EC%95%BC%EA%B5%AC%EC%9E%A5" title="마산야구장">마산야구장</a>에서 열린 <a href="/wiki/%ED%95%9C%ED%99%94_%EC%9D%B4%EA%B8%80%EC%8A%A4" title="한화 이글스">한화 이글스</a>와의 경기에서 선발투수 <a href="/wiki/%EC%9D%B4%EC%84%B1%EB%AF%BC_(%EC%95%BC%EA%B5%AC_%EC%84%A0%EC%88%98)" title="이성민 (야구 선수)">이성민</a>의 호투와 타선 폭발로 <a href="/wiki/NC_%EB%8B%A4%EC%9D%B4%EB%85%B8%EC%8A%A4" title="NC 다이노스">NC 다이노스</a>가 11-2로 승리하면서 통산 8번째 감독 600승을 기록하였다.<sup class="reference" id="cite_ref-9"><a href="#cite_note-9">[9]</a></sup> <a href="/wiki/9%EC%9B%94_6%EC%9D%BC" title="9월 6일">9월 6일</a> <a class="mw-redirect" href="/wiki/%EB%A7%88%EC%82%B0%EC%95%BC%EA%B5%AC%EC%9E%A5" title="마산야구장">마산야구장</a>에서 열린 <a href="/wiki/KIA_%ED%83%80%EC%9D%B4%EA%B1%B0%EC%A6%88" title="KIA 타이거즈">KIA 타이거즈</a>와의 경기에서 역대 8번째 감독 통산 1,200경기 출장을 기록했다.<sup class="reference" id="cite_ref-10"><a href="#cite_note-10">[10]</a></sup> 이 경기에서 선발 투수 <a href="/wiki/%ED%85%8C%EB%93%9C_%EC%9B%A8%EB%B2%84" title="테드 웨버">테드 웨버</a>가 5 1/3이닝 5실점으로 부진하면서 9-0으로 패배하면서 NC 다이노스는 7연패를 기록하였다.</p>
<h4><span class="mw-headline" id="2015.EB.85.84"><a href="/wiki/2015%EB%85%84_NC_%EB%8B%A4%EC%9D%B4%EB%85%B8%EC%8A%A4_%EC%8B%9C%EC%A6%8C" title="2015년 NC 다이노스 시즌">2015년</a></span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EA%B9%80%EA%B2%BD%EB%AC%B8&amp;action=edit&amp;section=12" title="부분 편집: 2015년">편집</a><span class="mw-editsection-bracket">]</span></span></h4>
<p><a href="/wiki/2015%EB%85%84" title="2015년">2015년</a> <a href="/wiki/7%EC%9B%94_22%EC%9D%BC" title="7월 22일">7월 22일</a> <a href="/wiki/%EC%9A%B8%EC%82%B0%EB%AC%B8%EC%88%98%EC%95%BC%EA%B5%AC%EC%9E%A5" title="울산문수야구장">울산문수야구장</a>에서 열린 <a href="/wiki/%EB%A1%AF%EB%8D%B0_%EC%9E%90%EC%9D%B4%EC%96%B8%EC%B8%A0" title="롯데 자이언츠">롯데 자이언츠</a>와의 경기에서 역대 6번째로 감독 1,300경기 출장을 기록했다.<sup class="reference" id="cite_ref-11"><a href="#cite_note-11">[11]</a></sup> 경기에서는 <a href="/wiki/%ED%95%B4%EC%BB%A4" title="해커">에릭 해커</a>의 호투로 3-2로 승리했다. <a href="/wiki/8%EC%9B%94_27%EC%9D%BC" title="8월 27일">8월 27일</a> <a class="mw-redirect" href="/wiki/%EB%A7%88%EC%82%B0%EC%95%BC%EA%B5%AC%EC%9E%A5" title="마산야구장">마산야구장</a>에서 열린 <a href="/wiki/%ED%95%9C%ED%99%94_%EC%9D%B4%EA%B8%80%EC%8A%A4" title="한화 이글스">한화 이글스</a>와의 경기에서 역대 7번째 감독 700승을 기록했다.<sup class="reference" id="cite_ref-12"><a href="#cite_note-12">[12]</a></sup> 경기에서는 <a href="/wiki/%ED%95%B4%EC%BB%A4" title="해커">에릭 해커</a>의 8이닝 1실점 호투로 4-1로 승리했다.</p>
<p>페넌트레이스 최종 순위 2위로 4위 <a href="/wiki/%EB%84%A5%EC%84%BC_%ED%9E%88%EC%96%B4%EB%A1%9C%EC%A6%88" title="넥센 히어로즈">넥센 히어로즈</a>를 꺾고 올라온 <a href="/wiki/%EB%91%90%EC%82%B0_%EB%B2%A0%EC%96%B4%EC%8A%A4" title="두산 베어스">두산 베어스</a>와 플레이오프 경기를 했으나, 2-3의 스코어로 시리즈에서 패배하면서 최종 순위는 3위로 마감했다.</p>
<h4><span class="mw-headline" id="2016.EB.85.84"><a href="/wiki/2016%EB%85%84_NC_%EB%8B%A4%EC%9D%B4%EB%85%B8%EC%8A%A4_%EC%8B%9C%EC%A6%8C" title="2016년 NC 다이노스 시즌">2016년</a></span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EA%B9%80%EA%B2%BD%EB%AC%B8&amp;action=edit&amp;section=13" title="부분 편집: 2016년">편집</a><span class="mw-editsection-bracket">]</span></span></h4>
<p><a href="/wiki/2016%EB%85%84" title="2016년">2016년</a> <a href="/wiki/5%EC%9B%94_22%EC%9D%BC" title="5월 22일">5월 22일</a> <a class="mw-redirect" href="/wiki/%EB%A7%88%EC%82%B0%EC%95%BC%EA%B5%AC%EC%9E%A5" title="마산야구장">마산야구장</a>에서 열린 <a href="/wiki/%EC%82%BC%EC%84%B1_%EB%9D%BC%EC%9D%B4%EC%98%A8%EC%A6%88" title="삼성 라이온즈">삼성 라이온즈</a>와의 경기에서 역대 6번째로 감독 1,400경기 출장을 기록했다.<sup class="reference" id="cite_ref-13"><a href="#cite_note-13">[13]</a></sup> 경기에서는 <a href="/wiki/%EB%82%98%EC%84%B1%EB%B2%94" title="나성범">나성범</a>이 5타수 4안타(1홈런) 1타점 3득점, 박석민이 4타수 2안타 3타점 1득점을 기록하면서 9-8로 승리했다.</p>
<p><a href="/wiki/%EB%91%90%EC%82%B0_%EB%B2%A0%EC%96%B4%EC%8A%A4" title="두산 베어스">두산 베어스</a> 시절부터 <a href="/wiki/%EB%B0%95%EC%A2%85%ED%9B%88_(1959%EB%85%84)" title="박종훈 (1959년)">박종훈</a>과 함께 <b>화수분 야구</b>를 통하여 무명 선수들을 육성하여 팀의 주축 선수로 성장시켰다. <a href="/wiki/%EC%8B%A0%EA%B3%A0%EC%84%A0%EC%88%98" title="신고선수">신고선수</a>로 입단한 <a class="mw-redirect" href="/wiki/%EA%B9%80%ED%98%84%EC%88%98_(1988%EB%85%84)" title="김현수 (1988년)">김현수</a>, 제대하자마자 <a href="/wiki/%ED%98%84%EB%8C%80_%EC%9C%A0%EB%8B%88%EC%BD%98%EC%8A%A4" title="현대 유니콘스">현대 유니콘스</a>에서 방출당해 선수 생활에 위기를 맞았던 <a href="/wiki/%EC%9D%B4%EC%A2%85%EC%9A%B1_(%EC%95%BC%EA%B5%AC_%EC%84%A0%EC%88%98)" title="이종욱 (야구 선수)">이종욱</a>, <a href="/wiki/%EC%82%BC%EC%84%B1_%EB%9D%BC%EC%9D%B4%EC%98%A8%EC%A6%88" title="삼성 라이온즈">삼성 라이온즈</a> 시절 만년 2군 선수였던 <a href="/wiki/%EA%B9%80%EC%A2%85%ED%98%B8_(1984%EB%85%84)" title="김종호 (1984년)">김종호</a> 등이 이에 해당된다.<sup class="reference" id="cite_ref-14"><a href="#cite_note-14">[14]</a></sup></p>

<h2><span class="mw-headline" id=".EC.97.90.ED.94.BC.EC.86.8C.EB.93.9C">에피소드</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EA%B9%80%EA%B2%BD%EB%AC%B8&amp;action=edit&amp;section=19" title="부분 편집: 에피소드">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>영문명의 Moon을 본따서 "<a href="/wiki/%EB%8B%AC" title="달">달</a>감독"이라는 애칭이 있다.<sup class="reference" id="cite_ref-15"><a href="#cite_note-15">[15]</a></sup><br/>
전 <a href="/wiki/%EB%91%90%EC%82%B0" title="두산">두산 베어스</a>소속의 <a href="/wiki/%EC%9D%B4%ED%98%9C%EC%B2%9C" title="이혜천">이혜천</a>이 <a href="/wiki/2013%EB%85%84" title="2013년">2013년</a> <a href="/wiki/11%EC%9B%94_22%EC%9D%BC" title="11월 22일">11월 22일</a>에 열린 <a href="/wiki/2014%EB%85%84_KBO_%EB%A6%AC%EA%B7%B8_2%EC%B0%A8_%EB%93%9C%EB%9E%98%ED%94%84%ED%8A%B8" title="2014년 KBO 리그 2차 드래프트">2014년 2차 드래프트</a>를 통하여 1라운드 지명을 통해 <a href="/wiki/NC_%EB%8B%A4%EC%9D%B4%EB%85%B8%EC%8A%A4" title="NC 다이노스">NC 다이노스</a>로 이적하자 김경문 감독의 별명인 "<a href="/wiki/%EB%8B%AC" title="달">달</a>"과, 이전 부진했던 <a href="/wiki/%EC%9D%B4%ED%98%9C%EC%B2%9C" title="이혜천">이혜천</a>의 별명인 "이핵천", "혜르노빌" 등의 별명을 함께 빗대어<br/>
<a class="mw-redirect" href="/wiki/MBC" title="MBC">MBC</a>드라마 "<a class="mw-disambig" href="/wiki/%ED%95%B4%EB%A5%BC_%ED%92%88%EC%9D%80_%EB%8B%AC" title="해를 품은 달">해를 품은 달</a>"을 패러디한, "핵을 품은 달" 이라는 유행어가 나오기도 하였다 .<sup class="reference" id="cite_ref-16"><a href="#cite_note-16">[16]</a></sup></p>
