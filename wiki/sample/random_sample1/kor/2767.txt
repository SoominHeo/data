<p><b>김경천</b>(金擎天, <a href="/wiki/1888%EB%85%84" title="1888년">1888년</a> <a href="/wiki/6%EC%9B%94_5%EC%9D%BC" title="6월 5일">6월 5일</a> ~ <a href="/wiki/1942%EB%85%84" title="1942년">1942년</a> <a href="/wiki/1%EC%9B%94_2%EC%9D%BC" title="1월 2일">1월 2일</a>)은 <a href="/wiki/%EC%9D%BC%EB%B3%B8%EA%B5%B0" title="일본군">일본군</a> 장교 출신으로서 해외로 망명하여 <a href="/wiki/%EC%9D%BC%EC%A0%9C_%EA%B0%95%EC%A0%90%EA%B8%B0" title="일제 강점기">일제 강점기</a>에 무장 독립 운동을 벌인 독립 운동가이다. 본명은 <b>김광서</b>, 별칭은 ‘조선의 나폴레옹’<sup class="reference" id="cite_ref-1"><a href="#cite_note-1">[1]</a></sup>, 김응천(金應天), 김현충(金顯忠)이었는데, <a href="/wiki/%EB%A7%8C%EC%A3%BC" title="만주">만주</a>와 <a class="mw-redirect" href="/wiki/%EC%97%B0%ED%95%B4%EC%A3%BC" title="연해주">연해주</a> 일대에서는 <b>백마 탄 김장군</b>으로 더 유명했다. <a href="/wiki/1998%EB%85%84" title="1998년">1998년</a> <a href="/wiki/%EA%B1%B4%EA%B5%AD%ED%9B%88%EC%9E%A5" title="건국훈장">건국훈장</a> 대통령장이 추서됐다. 본관은 <a href="/wiki/%EC%8B%9C%ED%9D%A5_%EA%B9%80%EC%94%A8" title="시흥 김씨">시흥 김씨</a><sup class="reference" id="cite_ref-2"><a href="#cite_note-2">[2]</a></sup> 이다.</p>

<h2><span class="mw-headline" id=".EC.83.9D.EC.95.A0">생애</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EA%B9%80%EA%B2%BD%EC%B2%9C_(1888%EB%85%84)&amp;action=edit&amp;section=1" title="부분 편집: 생애">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p><a href="/wiki/%ED%95%A8%EA%B2%BD%EB%82%A8%EB%8F%84" title="함경남도">함경남도</a> <a href="/wiki/%EB%B6%81%EC%B2%AD%EA%B5%B0" title="북청군">북청</a>에서 무관 가문의 막내 아들로 태어났다. 본명은 김광서(金光瑞)이다. 후에 <a href="/wiki/%EC%8B%A0%ED%8C%94%EA%B7%A0" title="신팔균">신팔균</a>(신동천), <a href="/wiki/%EC%A7%80%EC%B2%AD%EC%B2%9C" title="지청천">지청천</a>과 함께 '하늘 천(天)'자를 넣어 지은 김경천(金擎天, 金警天 또는 金敬天), 김응천(金應天)은 별명이다.</p>
<p>아버지인 김정우(金鼎禹)는 <a href="/wiki/%EC%9D%BC%EB%B3%B8" title="일본">일본</a>에 유학을 다녀온 구한국 육군의 엘리트 인사였다. 김정우가 만학의 나이에 큰 아들 성은과 함께 일본에 유학하여 김정우는 동경공업학교를, 김성은은 일본육사를 졸업했다. 그리고 귀국하여 고급장교가 되었다. 김경천은 군인인 아버지 및 형들의 영향을 받아 어릴 때부터 군인이 되기를 꿈꾸었고, <a href="/wiki/%ED%95%9C%EC%84%B1%EB%B6%80" title="한성부">한성부</a>에서 중학교를 마친 뒤 관비 유학생으로 일본으로 건너간다. 아버지와 형은 공업을 배우라고 권했지만 ‘나폴레옹’ 책을 탐독하던 그는 결국 군인의 길을 선택한다. 그는 그해 바로 <a class="mw-redirect" href="/wiki/%EC%9D%BC%EB%B3%B8%EC%9C%A1%EA%B5%B0%EC%82%AC%EA%B4%80%ED%95%99%EA%B5%90" title="일본육군사관학교">일본육군사관학교</a>에 입학했다. 일본 유학 중 아버지 김정우와 형 김성은의 사망 소식을 접한다.</p>
<p><a href="/wiki/1911%EB%85%84" title="1911년">1911년</a> 일본 육사를 최우등으로 졸업(제23기)하고 <a href="/wiki/%EB%8F%84%EC%BF%84" title="도쿄">도쿄</a>에서 기병 소위로 임관했다. 일본육사를 졸업하면 바로 소위로 임관하는 것이 통례였으나 그는 소위 임관을 처음에 거부하여 말썽이 되었다. 결국 <a href="/wiki/%EA%B2%BD%EC%84%B1%EB%B6%80" title="경성부">경성부</a>로 소환되어 조선총독 테라우치 마다사케와 면담하였다. 테라우치한테 임관 권고를 받은 김경천은 아버지와 형의 죽음으로 가정 생계도 꾸려나가야 했고, '독립전쟁을 벌이려면 육사 졸업한 것 갖고는 안 된다. 일본군 장교 생활을 하여 일본 군사기밀을 알아내야 한다'는 생각으로 마음을 고쳐먹고 그 권고를 받아들였다.</p>
<p>이후 기병 장교로 근무하였는데, 수 년간 기회를 엿보다가 <a href="/wiki/1919%EB%85%84" title="1919년">1919년</a> <a href="/wiki/2%C2%B78_%EB%8F%85%EB%A6%BD_%EC%84%A0%EC%96%B8" title="2·8 독립 선언">2·8 독립 선언</a>을 계기로 탈출을 결심하고 귀국했다. 그해 6월 초 <a href="/wiki/%EC%A7%80%EC%B2%AD%EC%B2%9C" title="지청천">지청천</a>과 함께 <a href="/wiki/%EB%A7%8C%EC%A3%BC" title="만주">만주</a>로 망명하여 <a class="new" href="/w/index.php?title=%EB%8C%80%ED%95%9C%EB%8F%85%EB%A6%BD%EC%B2%AD%EB%85%84%EB%8B%A8&amp;action=edit&amp;redlink=1" title="대한독립청년단 (없는 문서)">대한독립청년단</a>(총재 <a class="new" href="/w/index.php?title=%EC%95%88%EB%B3%91%EC%B0%AC&amp;action=edit&amp;redlink=1" title="안병찬 (없는 문서)">안병찬</a>)에 가입해 활동했고, <a class="mw-redirect" href="/wiki/%EC%84%9C%EA%B0%84%EB%8F%84" title="서간도">서간도</a>의 <a href="/wiki/%EC%8B%A0%ED%9D%A5%EB%AC%B4%EA%B4%80%ED%95%99%EA%B5%90" title="신흥무관학교">신흥무관학교</a>에서 교관으로 근무했다.</p>
<p>1919년에 <a class="mw-redirect" href="/wiki/3.1_%EB%A7%8C%EC%84%B8_%EC%9A%B4%EB%8F%99" title="3.1 만세 운동">3.1 만세 운동</a>이 발발하자 김경천은 육사 삼 년 후배 이청천과 함께 만주 삼원포에 위치해 있는, 독립운동가 이회영, 이상룡이 독립군을 양성하기 위해 세워놓은 신흥무관학교에 갔다. 일본육사출신 김경천, 이청천이 그 무관학교의 교관으로 있다는 소식이 국내에 들어가자 <a class="mw-redirect" href="/wiki/3.1%EC%9A%B4%EB%8F%99" title="3.1운동">3.1운동</a> 이후 독립군이 되려던 학생들은 학업을 중단하고 삼원포로 갔다. 이에 크게 당황한 일제의 온갖 수단을 강구한 김경천 매장 공작과 중국인과의 외부적 문제, 한국인끼리의 내부적 문제로 인해, 신흥무관학교를 주도하던 중 그는 신흥무관학교 내에서 따돌림을 당하게 되었다.</p>
<p>이후 <a href="/wiki/1919%EB%85%84" title="1919년">1919년</a> 연말 경에 김경천은 만주 삼원포를 떠나 <a href="/wiki/%EB%9F%AC%EC%8B%9C%EC%95%84" title="러시아">러시아</a> 지역으로 이동하여 <a href="/wiki/%EB%B8%94%EB%9D%BC%EB%94%94%EB%B3%B4%EC%8A%A4%ED%86%A0%ED%81%AC" title="블라디보스토크">블라디보스토크</a>에 머물렀다. 그는 이곳에서 의용군을 모집하여 일본군의 지원을 받는 중국인 마적단과 싸웠으며, <a class="new" href="/w/index.php?title=%EC%B0%BD%ED%95%B4%EC%B2%AD%EB%85%84%EB%8B%A8&amp;action=edit&amp;redlink=1" title="창해청년단 (없는 문서)">창해청년단</a>(단장 <a class="new" href="/w/index.php?title=%EA%B9%80%EA%B7%9C%EB%A9%B4&amp;action=edit&amp;redlink=1" title="김규면 (없는 문서)">김규면</a>) 총사령관으로서 전투를 거듭하면서 <a href="/wiki/%EC%8B%9C%EB%B2%A0%EB%A6%AC%EC%95%84" title="시베리아">시베리아</a> 지역에서 이름이 널리 알려지게 되었다. 한편 <a href="/wiki/1920%EB%85%84" title="1920년">1920년</a> <a href="/wiki/5%EC%9B%94" title="5월">5월</a> 국내에서 개설된 <a class="new" href="/w/index.php?title=%EB%8C%80%EA%B5%AC%EC%B2%AD%EB%85%84%ED%9A%8C&amp;action=edit&amp;redlink=1" title="대구청년회 (없는 문서)">대구청년회</a> 발전 기금으로 5원을 기부하였다.<sup class="reference" id="cite_ref-3"><a href="#cite_note-3">[3]</a></sup> <a href="/wiki/1921%EB%85%84" title="1921년">1921년</a>에는 <a class="new" href="/w/index.php?title=%EC%88%98%EC%B2%AD%EC%9D%98%EB%B3%91%EB%8C%80&amp;action=edit&amp;redlink=1" title="수청의병대 (없는 문서)">수청의병대</a>의 지도자가 되었고 러시아의 혁명 세력과 연합하면서 <a class="mw-redirect" href="/wiki/%EC%97%B0%ED%95%B4%EC%A3%BC" title="연해주">연해주</a> 지역의 조선인 지도자로 <a href="/wiki/%EC%86%8C%EB%A0%A8" title="소련">소련</a>의 인정도 받게 되었다.</p>
<p><a href="/wiki/1922%EB%85%84" title="1922년">1922년</a> 수청의병대는 <a class="new" href="/w/index.php?title=%EB%8C%80%ED%95%9C%ED%98%81%EB%AA%85%EB%8B%A8&amp;action=edit&amp;redlink=1" title="대한혁명단 (없는 문서)">대한혁명단</a>으로 개칭하였으며, 김경천은 사령관을 맡았다. 그해 말에는 <a class="new" href="/w/index.php?title=%EA%B3%A0%EB%A0%A4%ED%98%81%EB%AA%85%EA%B5%B0&amp;action=edit&amp;redlink=1" title="고려혁명군 (없는 문서)">고려혁명군</a>(사령관 <a href="/wiki/%EA%B9%80%EA%B7%9C%EC%8B%9D_(%EB%B6%81%EB%A1%9C%EA%B5%B0%EC%A0%95%EC%84%9C)" title="김규식 (북로군정서)">김규식</a>)이 조직되었고 김경천은 동부사령관을 맡았다. 그러나 이후 정세 변화로 러시아 지역에서의 독립 운동이 소강 상태에 빠지면서, 노령(露領) 무장 독립 운동의 선도격이던 그의 입지는 좁아졌다. 1923년 상해에서 상해임시정부를 개편하기 위한 국민대표회의가 개최될 때 군사담당 위원으로 내정되었다. 김경천은 상해 국민대표회의에 참석했다. <a href="/wiki/1923%EB%85%84" title="1923년">1923년</a> 이후로는 블라디보스토크의 <a class="new" href="/w/index.php?title=%EA%B7%B9%EB%8F%99%EA%B3%A0%EB%A0%A4%EC%82%AC%EB%B2%94%EB%8C%80%ED%95%99&amp;action=edit&amp;redlink=1" title="극동고려사범대학 (없는 문서)">극동고려사범대학</a>에서 강의를 하였고 국경경비대의 장교로 일했다는 정도만이 알려져 있다.</p>
<p><a href="/wiki/1936%EB%85%84" title="1936년">1936년</a> 소련 당국의 한인 인텔리 피검정책과 관련하여 체포되었고, 9월 29일 국경수비대 군법회의에서 3년 금고형을 선고받았다. 그의 체포 원인으로는 2가지 추측이 있다. 첫째, 연해주 한인 지도자는 이르쿠르츠파, 상해파 둘로 나뉘었는데 당시 상해파 지도자들이 대거 체포되는 과정 중 상해파 공산주의자로 오인받았을 수 있다. 김경천은 파당을 좋아하지 않아 어느 파에도 소속되지 않았다. 둘째, 그는 소련식 공산주의 운동에 적극 가담하지 않았다. 그는 민족주의자였지만 공산주의자는 아니었다 (이는 2012년 12월 06일자 KBS 역사스페셜 '백마 탄 김장군, 김경천! 시베리아의 전설이 되다' 에서도 밝혀진다.)</p>
<p><a href="/wiki/1937%EB%85%84" title="1937년">1937년</a> 연해주 거주 전체 한인에게 카자흐스탄과 중앙아시아 이주 정책이 시작된다. 김경천은 이 당시 2년 반을 복역한다.</p>
<p><a href="/wiki/1939%EB%85%84" title="1939년">1939년</a> 2월 석방되어 카자흐스탄 까라간다에 있는 집으로 돌아온다. 독일인 농장 잡부로 일했으나 4월 5일 재차 체포되어 까라간다 정치범수용소에서 복역했다. 6월 25일 모스크바로 이송되었고 간첩죄가 적용되어 강제노동수감소 8년형을 언도 받았고, 러시아 북부철도수용소로 이송되어 매일 철도건설 공사장에 동원되었다.</p>
<p><a href="/wiki/1942%EB%85%84" title="1942년">1942년</a> 1월 14일 비타민 결핍으로 인한 심장질환으로 사망했고, 시신은 수용소 근처에 묻혔다고 하나 정확한 장소는 아무도 모른다.</p>
<h2><span class="mw-headline" id=".EC.82.AC.ED.9B.84">사후</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EA%B9%80%EA%B2%BD%EC%B2%9C_(1888%EB%85%84)&amp;action=edit&amp;section=2" title="부분 편집: 사후">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>일본 육사를 졸업한 장교로서 보장된 앞길을 버리고 홀연히 망명한 뒤 <a href="/wiki/%EB%A7%8C%EC%A3%BC" title="만주">만주</a>, <a class="mw-redirect" href="/wiki/%EC%97%B0%ED%95%B4%EC%A3%BC" title="연해주">연해주</a> 일대에서 '백마 탄 <b>김장군'</b>으로 불리며 유명했던 그는 흰 말을 타고 만주와 시베리아를 누비는 전설적인 항일 영웅으로서의 이미지로 기억되고 있다. 특히 김경천은 여러 가명을 사용했고 일찍부터 '장군'으로 불린 인물이기에, <b><a href="/wiki/%EA%B9%80%EC%9D%BC%EC%84%B1" title="김일성">김일성</a>이 그의 이러한 명성과 항일 투쟁 경력, 전설적인 이미지 등을 도용했다는 주장</b>이 꾸준히 제기되어 왔다.</p>
<p><a href="/wiki/1936%EB%85%84" title="1936년">1936년</a> 김경천이 3년형을 언도받은 사건은 1956년 재심되어 무죄선고가 내려졌다.</p>
<p><a href="/wiki/1939%EB%85%84" title="1939년">1939년</a> 체포되어 8년형을 받은 사건은 1959년 모스크바 군사재판소에서 재심하여 무죄를 선언하고 다음날 17일에 사후복권 시켰다.</p>
<p><a href="/wiki/1993%EB%85%84" title="1993년">1993년</a> 카자흐스탄에서는 '정치적 탄압에 의한 희생자의 명예회복' 관련 법률에 의해 그의 명예를 회복시켰다.</p>
<p><a href="/wiki/1998%EB%85%84" title="1998년">1998년</a> 한국정부는 그에게 <a href="/wiki/%EA%B1%B4%EA%B5%AD%ED%9B%88%EC%9E%A5" title="건국훈장">건국훈장</a> 대통령장을 추서했다.</p>

<h2><span class="mw-headline" id=".EA.B8.B0.ED.83.80">기타</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EA%B9%80%EA%B2%BD%EC%B2%9C_(1888%EB%85%84)&amp;action=edit&amp;section=5" title="부분 편집: 기타">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<h3><span class="mw-headline" id=".EB.B3.B8.EA.B4.80_.EB.AC.B8.EC.A0.9C">본관 문제</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EA%B9%80%EA%B2%BD%EC%B2%9C_(1888%EB%85%84)&amp;action=edit&amp;section=6" title="부분 편집: 본관 문제">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>경천아일록에서 김경천은 본인의 본관을 김해 김씨라 밝히고 있다. 1916년 10월 10일에 자신의 본관을 시흥(始興) 김씨로 정정하였다.<sup class="reference" id="cite_ref-7"><a href="#cite_note-7">[7]</a></sup> 시흥 김씨 연원에는 김지(金智)가 평안도 대동군(大同郡) 추울미(秋蔚美)에 살면서 세계를 계승하였다.</p>
<p>일부에서는 김경천이 본관을 시흥으로 정정한 것이 독립 의지를 표현하기 위한 것이라는 주장을 하나, 한국의 독립과 김경천의 본관은 상관관계가 없고 본관의 정정은 오류가 없는 경우에는 이루지지 않는 것이 일반적이다.</p>
