<p><b>김담</b>(金淡, <a href="/wiki/1416%EB%85%84" title="1416년">1416년</a> ~ <a href="/wiki/1464%EB%85%84" title="1464년">1464년</a> <a href="/wiki/7%EC%9B%94_10%EC%9D%BC" title="7월 10일">7월 10일</a>)은 <a href="/wiki/%EC%A1%B0%EC%84%A0" title="조선">조선</a> 전기의 문신, 유학자, 수학자, 천문학자이다. 세종, 문종, 세조 때의 문신이자 천문학자, <a href="/wiki/%EC%A7%80%EB%A6%AC%ED%95%99" title="지리학">지리학</a>자이자 월력 연구가이다. <a href="/wiki/%EC%9D%B4%EC%88%9C%EC%A7%80" title="이순지">이순지</a> 등과 함께 조선 전기의 월력과 천문을 연구하였다.</p>
<p><a href="/wiki/1435%EB%85%84" title="1435년">1435년</a>(세종 17) 과거에 급제했다. 그해 문과에서 성삼문이 1등, 김담이 2등, 이개가 3등을 했다. 급제 이후 <a href="/wiki/%ED%99%8D%EB%AC%B8%EA%B4%80" title="홍문관">홍문관</a>정자, <a href="/wiki/%EC%A7%91%ED%98%84%EC%A0%84" title="집현전">집현전</a> 정자가 되었으며 <a href="/wiki/1433%EB%85%84" title="1433년">1433년</a> <a href="/wiki/%EC%9D%B4%EC%88%9C%EC%A7%80" title="이순지">이순지</a>와 함께 조선의 달력인 칠정산내편을 편찬하였다. 이후 중시에 2등으로 급제하였다. <a href="/wiki/%EC%9D%B4%EC%88%9C%EC%A7%80" title="이순지">이순지</a>와 함께 당대에 가장 뛰어난 천문학자로서, 세종대의 천문ㆍ역법사업에 크게 공헌하였다. <a href="/wiki/%EC%9E%A5%EC%98%81%EC%8B%A4" title="장영실">장영실</a>, <a href="/wiki/%EC%9D%B4%EC%88%9C%EC%A7%80" title="이순지">이순지</a>, 이위 등과 함께 간의규표, 앙부일철, 자격누등의 천문 관측 기구의 개발에도 참여하였다. <a href="/wiki/%EB%B0%95%ED%8C%BD%EB%85%84" title="박팽년">박팽년</a>, <a href="/wiki/%ED%95%98%EC%9C%84%EC%A7%80" title="하위지">하위지</a> 등과 친분이 있었으며 이들과 담론하며 시류에 대한 불만을 토로했다가 문제시되기도 하였다. <a class="mw-redirect" href="/wiki/%EC%84%B8%EC%A1%B0_%EB%B0%98%EC%A0%95" title="세조 반정">세조 반정</a> 이후 반정에 동조하지는 않았으나 관직에 있게 되었다. 이후 그는 외직을 자청하여 <a class="mw-redirect" href="/wiki/%EC%95%88%EB%8F%99" title="안동">안동</a><a href="/wiki/%EB%B6%80%EC%82%AC" title="부사">부사</a>, 예조 참의, <a class="mw-redirect" href="/wiki/%EA%B2%BD%EC%A3%BC" title="경주">경주</a><a href="/wiki/%EB%B6%80%EC%9C%A4" title="부윤">부윤</a>, <a class="mw-redirect" href="/wiki/%EC%B6%A9%EC%A3%BC" title="충주">충주</a><a href="/wiki/%EB%AA%A9%EC%82%AC" title="목사">목사</a> 등을 거쳐 <a class="mw-redirect" href="/wiki/%EC%9D%B4%EC%A1%B0%ED%8C%90%EC%84%9C" title="이조판서">이조판서</a>로 승진했고 <a class="mw-disambig" href="/wiki/%EC%A4%91%EC%B6%94%EC%9B%90" title="중추원">중추원</a>사에 이르렀다.</p>
<p>본관은 <a class="mw-redirect" href="/wiki/%EC%98%88%EC%95%88(%EC%84%A0%EC%84%B1)_%EA%B9%80%EC%94%A8" title="예안(선성) 김씨">예안</a>으로, 자는 거원(巨源), 호는 무송헌(撫松軒)이다. 조부 김로(金輅)는 고려때 좌우위보승랑장으로 고려 절신이고, 아버지는 영유현령을 지낸 김소량(金小良)이다. 어머니는 평해 황씨(平海 黃氏)로 고려 때 공조판서를 지낸 황유정의 딸이다.</p>
<p>영주시 가흥동에 소재한 삼판서 고택(정도전의 본가)</p>
<p>삼판서 가운데 가장 먼저 그 집에 살았던 사람은 봉화가 본관인 정도전의 부친 정운경(1305~66)이었다. 공민왕 때에 형부상서(형조판서)를 지낸 그는 사위인 황유정(1343~?)에게 자신의 집을 물려주었다. 평해황씨로 공조전서(공조판서)를 역임한 황유정은 다시 사위인 선성(예안)김씨 김소량(1384~1449)에게 이 집을 물려주었고 김소량의 둘째 아들 김담(1416~64)이 이조판서에 올랐다. 김담의 외조모 정씨(정운경의 외동딸)는 영의정을 지낸 정도전과는 오누이 사이다. 아울러 현재 김담의 후손이 사는 무송헌(撫松軒)종택은 영주시 무섬마을에 있다.</p>

<h2><span class="mw-headline" id=".EC.83.9D.EC.95.A0">생애</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EA%B9%80%EB%8B%B4&amp;action=edit&amp;section=1" title="부분 편집: 생애">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<h3><span class="mw-headline" id=".EC.83.9D.EC.95.A0_.EC.B4.88.EB.B0.98">생애 초반</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EA%B9%80%EB%8B%B4&amp;action=edit&amp;section=2" title="부분 편집: 생애 초반">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>김담은 지금의 <a href="/wiki/%EA%B2%BD%EC%83%81%EB%B6%81%EB%8F%84" title="경상북도">경상북도</a> <a href="/wiki/%EC%98%81%EC%A3%BC%EC%8B%9C" title="영주시">영주시</a>에서 영유현령으로 사후 병조판서에 추증된 김소량의 3형제의 중 둘째 아들로 태어났다. 그의 형 김증(金曾)에 이어 그가 현달함으로써 집안 가세를 일으켰다. 후일 형 김증은 <a href="/wiki/%EC%84%B1%EC%82%BC%EB%AC%B8" title="성삼문">성삼문</a>, <a href="/wiki/%EC%8B%A0%EC%88%99%EC%A3%BC" title="신숙주">신숙주</a>, <a href="/wiki/%EC%A0%95%EC%9D%B8%EC%A7%80" title="정인지">정인지</a> 등과 함께 홍무정운을 번역하고 역대병요를 편찬하기도 했다.</p>
<p>그는 어려서부터 총명하고 독서를 좋아했다 한다.</p>
<h3><span class="mw-headline" id=".EA.B4.80.EB.A3.8C.EC.83.9D.ED.99.9C_.EC.B4.88.EB.B0.98">관료생활 초반</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EA%B9%80%EB%8B%B4&amp;action=edit&amp;section=3" title="부분 편집: 관료생활 초반">편집</a><span class="mw-editsection-bracket">]</span></span></h3>

<div class="rellink boilerplate seealso"><img alt="" data-file-height="128" data-file-width="128" height="16" src="//upload.wikimedia.org/wikipedia/commons/thumb/e/ec/Crystal_Clear_app_xmag.svg/16px-Crystal_Clear_app_xmag.svg.png" srcset="//upload.wikimedia.org/wikipedia/commons/thumb/e/ec/Crystal_Clear_app_xmag.svg/24px-Crystal_Clear_app_xmag.svg.png 1.5x, //upload.wikimedia.org/wikipedia/commons/thumb/e/ec/Crystal_Clear_app_xmag.svg/32px-Crystal_Clear_app_xmag.svg.png 2x" width="16"/> <a href="/wiki/%EC%A7%91%ED%98%84%EC%A0%84" title="집현전">집현전</a>, <a href="/wiki/%EA%B0%84%EC%9D%98%EB%8C%80" title="간의대">간의대</a>, <a class="new" href="/w/index.php?title=%EC%9B%94%EB%A0%A5&amp;action=edit&amp;redlink=1" title="월력 (없는 문서)">월력</a> 문서를 참고하십시오.</div>
<p>이후 <a href="/wiki/1435%EB%85%84" title="1435년">1435년</a>(세종 17) 문과 정시에 병과로 급제하여 집현전(후에 홍문관) 정자로 임명되었다. 이듬해 이순지(李純之)가 <a href="/wiki/%EA%B0%84%EC%9D%98%EB%8C%80" title="간의대">간의대</a>(簡儀臺)에 나가 천문학을 관측하던 중 어머니가 상(喪)을 당하자 그 일을 대신 맡아보기도 했다.</p>
<p>집현전학사로 있으면서 훈민정음 창제에도 참여했고, 특히 천문과 관측 분야에 뛰어나 <a href="/wiki/%EA%B0%84%EC%9D%98%EB%8C%80" title="간의대">간의대</a>(簡儀臺)로 발령받아 이후 간의대에서 활약, 1433년(세종 15년)에는 17세에 세종의 영을 받아 <a href="/wiki/%EC%9D%B4%EC%88%9C%EC%A7%80" title="이순지">이순지</a>와 함께 달력 연구에 참여하였다. 이후 그는 <a href="/wiki/%EC%9D%B4%EC%88%9C%EC%A7%80" title="이순지">이순지</a>와 함께 해와 달의 기울음을 연구한 뒤 <a href="/wiki/%EC%9B%90%EB%82%98%EB%9D%BC" title="원나라">원나라</a>의 접시력, <a href="/wiki/%EB%AA%85%EB%82%98%EB%9D%BC" title="명나라">명나라</a>의 <a class="new" href="/w/index.php?title=%EB%8C%80%ED%86%B5%EB%A0%A5&amp;action=edit&amp;redlink=1" title="대통력 (없는 문서)">대통력</a>을 종합하여 <a href="/wiki/%EC%A1%B0%EC%84%A0" title="조선">조선</a>편 달력을 칠정산내편을 간행하였다. 이후 세종대왕의 전폭적인 신뢰를 얻어 이순지, 이위, 장영실 등과 함께 간의규표, 앙부일철, 자격누등의 천문 관측 기구를 개발하였다. 김담은 천문·지리·풍수등에 두루 박식하여 세종의 총애를 받았고 <a href="/wiki/%EC%84%B1%EC%82%BC%EB%AC%B8" title="성삼문">성삼문</a>, <a href="/wiki/%EC%8B%A0%EC%88%99%EC%A3%BC" title="신숙주">신숙주</a>, <a href="/wiki/%EC%9E%A5%EC%98%81%EC%8B%A4" title="장영실">장영실</a>, <a href="/wiki/%EC%A0%95%EC%9D%B8%EC%A7%80" title="정인지">정인지</a> 등에 버금가는 명성을 얻었다.</p>
<p>그 뒤 다시 <a class="mw-redirect" href="/wiki/%EA%B3%BC%EA%B1%B0_%EC%A0%9C%EB%8F%84" title="과거 제도">중시</a>(重試)에 제 2등으로 합격하였다.<sup class="reference" id="cite_ref-1"><a href="#cite_note-1">[1]</a></sup></p>
<h3><span class="mw-headline" id=".EC.96.91.EC.A0.84_.EC.8B.9C.ED.96.89.EA.B3.BC_.EC.A7.80.EB.B0.A9.EA.B4.80_.EC.8B.9C.EC.A0.88">양전 시행과 지방관 시절</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EA%B9%80%EB%8B%B4&amp;action=edit&amp;section=4" title="부분 편집: 양전 시행과 지방관 시절">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>김담은 나이 25세에 전료(殿僚)와 더불어 국어와 음의(音義)를 보정해 올렸다. 28세 때에는 <a href="/wiki/%EB%B4%89%EC%83%81%EC%8B%9C" title="봉상시">봉상시</a><a href="/wiki/%EC%A3%BC%EB%B6%80" title="주부">주부</a>가 되었고, 이듬해 <a href="/wiki/%EC%A0%95%EC%9D%B8%EC%A7%80" title="정인지">정인지</a>, <a href="/wiki/%EC%9D%B4%EC%88%9C%EC%A7%80" title="이순지">이순지</a>, <a class="new" href="/w/index.php?title=%EB%B0%95%EC%9C%A4%EC%9E%A5&amp;action=edit&amp;redlink=1" title="박윤장 (없는 문서)">박윤장</a> 등과 함께 <a href="/wiki/%EA%B2%BD%EA%B8%B0%EB%8F%84" title="경기도">경기도</a> <a class="mw-redirect" href="/wiki/%EC%95%88%EC%82%B0" title="안산">안산</a>에서 양전을 시행했다. 29세 때는 이조정랑이 되었고, 제언종사관이 되어 이순지와 더불어 언제공사(堰堤工事)에서 계산을 맡았다.</p>
<p><a href="/wiki/1434%EB%85%84" title="1434년">1434년</a>(세종 16년) 이조정랑이 되고, 그해 제언종사관(堤堰從事官)이 되어 <a href="/wiki/%EC%9D%B4%EC%88%9C%EC%A7%80" title="이순지">이순지</a>와 더불어 언제공사(堰堤工事)에 참여하여 계산을 맡았다. <a href="/wiki/1437%EB%85%84" title="1437년">1437년</a> <a href="/wiki/%EC%8A%B9%EB%AC%B8%EC%9B%90" title="승문원">승문원</a>부교리가 되었다. <a href="/wiki/%EC%8A%B9%EB%AC%B8%EC%9B%90" title="승문원">승문원</a>부교리로 있을 때, 명을 받아 토지를 측정 연구하여 토지분할 정책인 「전부구등지법(田賦九等之法)」을 찬정하였다. 이후 이조 정랑, <a href="/wiki/%EC%A7%91%ED%98%84%EC%A0%84" title="집현전">집현전</a> 직제학 등을 지냈다.</p>
<p><a href="/wiki/1437%EB%85%84" title="1437년">1437년</a> <a href="/wiki/%EC%A7%91%ED%98%84%EC%A0%84" title="집현전">집현전</a>저작랑을 거쳐 <a class="new" href="/w/index.php?title=1439&amp;action=edit&amp;redlink=1" title="1439 (없는 문서)">1439</a> <a href="/wiki/%EC%A7%91%ED%98%84%EC%A0%84" title="집현전">집현전</a>박사가 되었다. 그해 <a href="/wiki/%EC%9D%B4%EC%88%9C%EC%A7%80" title="이순지">이순지</a>와 함께 《칠정산외편 七政算外篇》을 교정해서 올렸다. 그 뒤 <a class="mw-redirect" href="/wiki/%EC%84%B8%EC%A2%85%EB%8C%80%EC%99%95" title="세종대왕">세종대왕</a>의 명으로 <a href="/wiki/%EC%A0%95%EC%9D%B8%EC%A7%80" title="정인지">정인지</a>(鄭麟趾), <a class="new" href="/w/index.php?title=%EC%A0%95%ED%9D%A0%EC%A7%80&amp;action=edit&amp;redlink=1" title="정흠지 (없는 문서)">정흠지</a>(鄭欽之), <a class="mw-disambig" href="/wiki/%EC%A0%95%EC%B4%88" title="정초">정초</a>(鄭招) 등과 함께 수시력법(授時曆法)과 대통력태양태음통궤(大統曆太陽太陰通軌) 등에 대하여 그 계산법을 밝히고, 약간의 수정을 가해서 《칠정산내편 七政算內篇》을 만들었는데, 《칠정산내편》과 《태양통궤》·《태음통궤》 등이 모두 이순지와 김담이 편찬한 편찬본의 <a href="/wiki/%EA%B7%9C%EC%9E%A5%EA%B0%81" title="규장각">규장각</a>판 사본만이 현재 전한다.</p>
<p>1448년 서운관 부정이 되었고, 이듬해 친상을 당하였으나 계속 출사하여 역법과 측후의 일을 하였다. 1448년 전라도 관찰사, 1451년 사헌부장령이 되어 불사를 배척하는 상소를 여러번 올렸으며, 1452년(단종 즉위년) 집현전 직제학이 되었고, 이어 상주목사, 충주목사를 지냈다.</p>
<h3><span class="mw-headline" id=".EC.B2.9C.EB.AC.B8.ED.95.99.EA.B3.BC_.EC.9B.94.EB.A0.A5_.EC.97.B0.EA.B5.AC.2C_.EC.84.9C.EC.A0.81_.EA.B0.84.ED.96.89">천문학과 월력 연구, 서적 간행</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EA%B9%80%EB%8B%B4&amp;action=edit&amp;section=5" title="부분 편집: 천문학과 월력 연구, 서적 간행">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>그는 이순지와 더불어 당대에 가장 뛰어난 천문학자로서 천문·역법 사업에 크게 공헌하였으며, 정인지·정초·정흠지·이순지 등과 더불어 《칠정산내편》·《칠정산내편정묘년교식가령 七政算內篇丁卯年交食假令》·《칠정산외편》·《칠정산외편정묘년교식가령》·《대통력일통궤 大統曆日通軌》·《태양통궤 太陽通軌》·《태음통궤 太陰通軌》·《교식통궤 交食通軌》·《오성통궤 五星通軌》·《사여전도통궤 四餘纏度通軌》·《중수대명력 重修大明曆》·《경오원력 庚午元曆》·《선덕십년월오성릉범 宣德十年月五星陵犯》등 많은 천문역서를 교정·편찬하였다.</p>
<p><a href="/wiki/%EC%97%AD%EB%B2%95" title="역법">역법</a>에 재능이 있어 당대의 천문학자인 <a href="/wiki/%EC%9D%B4%EC%88%9C%EC%A7%80" title="이순지">이순지</a>와 더불어 《<a href="/wiki/%EC%B9%A0%EC%A0%95%EC%82%B0" title="칠정산">칠정산내편</a>》, 《<a href="/wiki/%EC%B9%A0%EC%A0%95%EC%82%B0" title="칠정산">칠정산외편</a>》 등의 편찬에 관계하였으며, 이밖에 토지정리 사업과 《<a href="/wiki/%EC%97%AD%EB%8C%80%EB%B3%91%EC%9A%94" title="역대병요">역대병요</a>》 등의 <a class="mw-redirect" href="/wiki/%EC%84%9C%EC%A0%81" title="서적">서적</a> 편찬에도 참여하였다.</p>
<p><a href="/wiki/1451%EB%85%84" title="1451년">1451년</a> <a href="/wiki/%EC%82%AC%ED%97%8C%EB%B6%80" title="사헌부">사헌부</a>장령이 되어 불사(佛事)를 배척하는 소를 여러 번 올렸고, <a href="/wiki/1452%EB%85%84" title="1452년">1452년</a>(단종 즉위년) <a href="/wiki/%EC%A7%91%ED%98%84%EC%A0%84" title="집현전">집현전</a><a class="new" href="/w/index.php?title=%EC%A7%81%EC%A0%9C%ED%95%99&amp;action=edit&amp;redlink=1" title="직제학 (없는 문서)">직제학</a>이 되었다. 그는 <a href="/wiki/%EB%B0%95%ED%8C%BD%EB%85%84" title="박팽년">박팽년</a>, <a href="/wiki/%ED%95%98%EC%9C%84%EC%A7%80" title="하위지">하위지</a> 등과 교분이 있었는데 이들을 만나 시류에 대한 불만을 토로한 것이 후일 문제가 되어 비판거리가 되기도 했다.</p>
<h3><span class="mw-headline" id=".EC.A7.80.EB.B0.A9.EA.B4.80_.EC.83.9D.ED.99.9C">지방관 생활</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EA%B9%80%EB%8B%B4&amp;action=edit&amp;section=6" title="부분 편집: 지방관 생활">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<div class="rellink boilerplate seealso"><img alt="" data-file-height="128" data-file-width="128" height="16" src="//upload.wikimedia.org/wikipedia/commons/thumb/e/ec/Crystal_Clear_app_xmag.svg/16px-Crystal_Clear_app_xmag.svg.png" srcset="//upload.wikimedia.org/wikipedia/commons/thumb/e/ec/Crystal_Clear_app_xmag.svg/24px-Crystal_Clear_app_xmag.svg.png 1.5x, //upload.wikimedia.org/wikipedia/commons/thumb/e/ec/Crystal_Clear_app_xmag.svg/32px-Crystal_Clear_app_xmag.svg.png 2x" width="16"/> <a href="/wiki/%EA%B3%84%EC%9C%A0%EC%A0%95%EB%82%9C" title="계유정난">계유정난</a>, <a class="mw-redirect" href="/wiki/%EC%84%B8%EC%A1%B0_%EC%B0%AC%EC%9C%84" title="세조 찬위">세조 찬위</a> 문서를 참고하십시오.</div>
<p><a href="/wiki/%EA%B3%84%EC%9C%A0%EC%A0%95%EB%82%9C" title="계유정난">계유정난</a>으로 정권을 잡은 수양대군이 <a href="/wiki/1455%EB%85%84" title="1455년">1455년</a> 단종을 축출하고 왕위에 오르자 중앙 벼슬을 피하여 자청하여 안동부사, 충주목사등의 외직으로 나갔다. 1456년 이후 안동부사로 나갔다가 <a href="/wiki/%EC%98%88%EC%A1%B0" title="예조">예조</a> <a href="/wiki/%EC%B0%B8%EC%9D%98" title="참의">참의</a>를 거쳐 1458년 경주부윤 등을 역임했다. 그의 사위 <a href="/wiki/%EC%9D%B4%EC%88%98%ED%98%95_(1435%EB%85%84)" title="이수형 (1435년)">이수형</a>은 단종 양위에 분개하여 벼슬을 버리고 낙향했는데, 그와 긴밀히 교류하였다. <a href="/wiki/%EC%9D%B4%EC%88%98%ED%98%95_(1435%EB%85%84)" title="이수형 (1435년)">이수형</a>은 장인의 고향인 영주 도촌으로 낙향하였다.</p>
<p>경주부윤으로 있을 때는 금강산에 있던 매월당 김시습과 자주 만나 동병상련의 정을 달래며 신라의 고적을 중수하는데 힘쓰기도 했다. 당시 세조가 이조판서의 벼슬을 내리며 두번씩이나 불렀으나 사양, 승정원일기에는 「사판삼신이일」로 기록되어 있다. <a href="/wiki/%EC%84%9C%EA%B1%B0%EC%A0%95" title="서거정">서거정</a>과도 교유하였는데, 경주부윤으로 있을 때 봉명사신으로 파견되었던 <a href="/wiki/%EC%84%9C%EA%B1%B0%EC%A0%95" title="서거정">서거정</a> 등이 오자 감사와 함께 그를 환대하며 고적 정비에 나서는 것으로 시름을 달랜다 하였다.</p>
<p>'자네가 내 마음을 먼저 알고 있네. 이미 통판과 의논하여 장차 객관을 중수(重修)하기로 하고, 재목을 축적하고 기와를 구우면서 시일을 기다리고 있을 뿐일세.'</p>

<p>그가 충주목사로 재직할 때 관내에 도적이 많아 고을 백성들이 몹시 두려워하였는데 그는 이 도적을 잘 다스렸으며, 장물의 증거를 발견하면 비록 그 양이 적더라도 이를 용서하지 않았다 한다.</p>
<p>1463년 이조판서로 승진했다가 1464년 중추원사(中樞院使)에 이르렀다. 문하에서는 <a href="/wiki/%EC%9D%B4%EC%88%98%ED%98%95_(1435%EB%85%84)" title="이수형 (1435년)">이수형</a> 등을 배출했는데, <a href="/wiki/%EC%9D%B4%EC%88%98%ED%98%95_(1435%EB%85%84)" title="이수형 (1435년)">이수형</a>은 그의 제자이자 사위였다. 그해 <a href="/wiki/7%EC%9B%94_10%EC%9D%BC" title="7월 10일">7월 10일</a>에 병으로 사망하였다. 당시 그의 나이 향년 48세였다.</p>

<h2><span class="mw-headline" id=".EC.82.AC.EC.83.81.EA.B3.BC_.EC.B9.98.EC.A0.81">사상과 치적</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EA%B9%80%EB%8B%B4&amp;action=edit&amp;section=13" title="부분 편집: 사상과 치적">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<h3><span class="mw-headline" id=".EB.8F.84.EC.A0.81_.EC.97.84.EB.8B.A8">도적 엄단</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EA%B9%80%EB%8B%B4&amp;action=edit&amp;section=14" title="부분 편집: 도적 엄단">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>실록에 의하면 그는 범죄자에게 엄격했다 한다. "그때 경내(境內)에 도적이 많아서 백성들이 매우 괴로와하였는데, 김담이 이들을 원수(怨讎)같이 싫어하여 도적의 자취를 잘 다스렸고, 간구(姦寇)가 있는 곳을 알아서 적발(摘發)하는 것이 귀신 같았고, 장물(臟物)의 증거를 발견하면 비록 적더라도 용서하지 않았다. 그들을 죽이지 않으면 그만두지 않으니, 이로 말미암아 도적들이 서로 경계하여 경내(境內)에는 들어오지 않았고, 고을 백성들이 그 덕을 입었다.<sup class="reference" id="cite_ref-3"><a href="#cite_note-3">[3]</a></sup>"는 것이다.</p>
<h2><span class="mw-headline" id=".ED.8F.89.EA.B0.80.EC.99.80_.EB.B9.84.ED.8C.90">평가와 비판</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EA%B9%80%EB%8B%B4&amp;action=edit&amp;section=15" title="부분 편집: 평가와 비판">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p><a class="mw-redirect" href="/wiki/%EC%84%B8%EC%A2%85" title="세종">세종</a> 때 <a href="/wiki/%EC%A0%95%EC%9D%B8%EC%A7%80" title="정인지">정인지</a>·<a href="/wiki/%EC%9D%B4%EC%88%9C%EC%A7%80" title="이순지">이순지</a>·<a class="new" href="/w/index.php?title=%EB%B0%95%EC%9C%A4%EC%B0%BD&amp;action=edit&amp;redlink=1" title="박윤창 (없는 문서)">박윤창</a>(朴允昌)과 더불어 경기도 안산(安山)에서 양전(量田)을 시행할 때도 공이 컸다<sup class="reference" id="cite_ref-nate01_4-0"><a href="#cite_note-nate01-4">[4]</a></sup>고 평가된다.</p>
<p>또한 <a href="/wiki/%EC%9D%B4%EC%88%9C%EC%A7%80" title="이순지">이순지</a>와 더불어 당대에 가장 뛰어난 천문학자로서 세종대의 천문·역법 사업에 크게 공헌했다<sup class="reference" id="cite_ref-nate01_4-1"><a href="#cite_note-nate01-4">[4]</a></sup>는 시각도 있다.</p>
