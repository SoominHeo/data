<h2><span class="mw-headline" id=".EC.83.9D.EC.95.A0">생애</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EA%B9%80%EB%B3%B4%EA%B2%BD_(%EC%B6%95%EA%B5%AC_%EC%84%A0%EC%88%98)&amp;action=edit&amp;section=1" title="부분 편집: 생애">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p><a href="/wiki/%EC%A0%84%EB%9D%BC%EB%82%A8%EB%8F%84" title="전라남도">전라남도</a> <a href="/wiki/%EA%B5%AC%EB%A1%80%EA%B5%B0" title="구례군">구례군</a> <a href="/wiki/%EA%B4%91%EC%9D%98%EB%A9%B4" title="광의면">광의면</a> 지천리에서 태어난 김보경은 <a href="/wiki/%EC%84%9C%EC%9A%B8%ED%8A%B9%EB%B3%84%EC%8B%9C" title="서울특별시">서울특별시</a> <a href="/wiki/%EA%B5%AC%EB%A1%9C%EA%B5%AC" title="구로구">구로구</a>의 오류초등학교에서 축구를 시작했으며, <a class="new" href="/w/index.php?title=%EC%9A%A9%EC%9D%B8_FC&amp;action=edit&amp;redlink=1" title="용인 FC (없는 문서)">용인 FC</a> 산하 원삼중학교, <a href="/wiki/%EC%8B%A0%EA%B0%88%EA%B3%A0%EB%93%B1%ED%95%99%EA%B5%90" title="신갈고등학교">신갈고등학교</a>에서 <a href="/wiki/%ED%97%88%EC%A0%95%EB%AC%B4" title="허정무">허정무</a> 감독 등으로부터 지도를 받았다. 그 후 <a href="/wiki/%ED%99%8D%EC%9D%B5%EB%8C%80%ED%95%99%EA%B5%90" title="홍익대학교">홍익대학교</a>를 거쳐 <a href="/wiki/%EC%9D%BC%EB%B3%B8" title="일본">일본</a> <a href="/wiki/J%EB%A6%AC%EA%B7%B8" title="J리그">J리그</a> <a href="/wiki/%EC%84%B8%EB%A0%88%EC%86%8C_%EC%98%A4%EC%82%AC%EC%B9%B4" title="세레소 오사카">세레소 오사카</a>에 입단했다.</p>
<h2><span class="mw-headline" id=".EC.84.A0.EC.88.98_.EC.83.9D.ED.99.9C">선수 생활</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EA%B9%80%EB%B3%B4%EA%B2%BD_(%EC%B6%95%EA%B5%AC_%EC%84%A0%EC%88%98)&amp;action=edit&amp;section=2" title="부분 편집: 선수 생활">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<h3><span class="mw-headline" id=".ED.81.B4.EB.9F.BD_.EC.83.9D.ED.99.9C">클럽 생활</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EA%B9%80%EB%B3%B4%EA%B2%BD_(%EC%B6%95%EA%B5%AC_%EC%84%A0%EC%88%98)&amp;action=edit&amp;section=3" title="부분 편집: 클럽 생활">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p><a href="/wiki/%ED%99%8D%EC%9D%B5%EB%8C%80%ED%95%99%EA%B5%90" title="홍익대학교">홍익대학교</a> 중퇴 후 프로 진출을 계획한 김보경은 <a href="/wiki/K%EB%A6%AC%EA%B7%B8_%EB%93%9C%EB%9E%98%ED%94%84%ED%8A%B8" title="K리그 드래프트">K리그 드래프트</a> 제도로 인해 원하는 팀에 입단할 수 없게 되자 결국 해외 진출로 진로를 바꿔 <a href="/wiki/2009%EB%85%84_12%EC%9B%94" title="2009년 12월">2009년 12월</a>, <a href="/wiki/J%EB%A6%AC%EA%B7%B8" title="J리그">J리그</a> <a href="/wiki/%EC%84%B8%EB%A0%88%EC%86%8C_%EC%98%A4%EC%82%AC%EC%B9%B4" title="세레소 오사카">세레소 오사카</a>에 입단하였으나 당시 <a href="/wiki/%EC%84%B8%EB%A0%88%EC%86%8C_%EC%98%A4%EC%82%AC%EC%B9%B4" title="세레소 오사카">세레소 오사카</a>의 외국인 선수 쿼터가 모두 차 있어, <a href="/wiki/%ED%99%A9%EB%B3%B4%EA%B4%80" title="황보관">황보관</a> 감독이 지도하는 <a class="mw-redirect" href="/wiki/J%EB%A6%AC%EA%B7%B8_%EB%94%94%EB%B9%84%EC%A0%84_2" title="J리그 디비전 2">J리그 디비전 2</a> <a href="/wiki/%EC%98%A4%EC%9D%B4%ED%83%80_%ED%8A%B8%EB%A6%AC%EB%8B%88%ED%83%80" title="오이타 트리니타">오이타 트리니타</a>에 잠시 임대되었다. 오이타에서 개막 직후 5경기에서 6골을 기록하는 좋은 모습을 보였으며, 그 해 열린 <a href="/wiki/2010%EB%85%84_FIFA_%EC%9B%94%EB%93%9C%EC%BB%B5" title="2010년 FIFA 월드컵">남아공 월드컵</a> 최종 명단에 포함되며 오이타 트리니타 클럽 역사상 처음으로 오이타 소속으로 월드컵에 출전한 선수가 되었다. 그러나 경기에 출전하지않아 아쉬움을 남겼다. 2011년 원 소속팀으로 복귀하하여 세레소의 주전 미드필더로 활약하며 41경기에서 15골을 기록하였다.</p>
<p><a href="/wiki/2012%EB%85%84_%ED%95%98%EA%B3%84_%EC%98%AC%EB%A6%BC%ED%94%BD" title="2012년 하계 올림픽">런던 올림픽</a>이 끝난 후인 <a href="/wiki/2012%EB%85%84" title="2012년">2012년</a> <a href="/wiki/%EC%97%AC%EB%A6%84" title="여름">여름</a>, <a href="/wiki/%ED%92%8B%EB%B3%BC_%EB%A6%AC%EA%B7%B8_%EC%B1%94%ED%94%BC%EC%96%B8%EC%8B%AD" title="풋볼 리그 챔피언십">풋볼 리그 챔피언십</a>의 <a href="/wiki/%EC%B9%B4%EB%94%94%ED%94%84_%EC%8B%9C%ED%8B%B0_FC" title="카디프 시티 FC">카디프 시티 FC</a>로 이적하였다. 2012-13 시즌 28경기에 출전하여 2골을 기록하며 팀의 <a href="/wiki/%ED%94%84%EB%A6%AC%EB%AF%B8%EC%96%B4%EB%A6%AC%EA%B7%B8" title="프리미어리그">프리미어리그</a> 승격에 기여하였다. 프리미어리그에서의 첫 시즌 <a class="mw-redirect" href="/wiki/%EB%A7%A8%EC%B2%B4%EC%8A%A4%ED%84%B0_%EC%9C%A0%EB%82%98%EC%9D%B4%ED%8B%B0%EB%93%9C" title="맨체스터 유나이티드">맨체스터 유나이티드</a>와의 경기에서 극적인 동점골을 기록하며 골닷컴 독자들이 선정한 최우수 선수로 뽑히기도 하는 등<sup class="reference" id="cite_ref-1"><a href="#cite_note-1">[1]</a></sup> 좋은 모습을 보였으나. 중반기부터 주전 경쟁에서 밀리는 모습을 보였고 팀이 다시 풋볼 리그 챔피언십으로 강등된 2014-15 시즌에도 출전 기회가 늘어나지 않자 구단과의 상호 계약 해지를 통해 팀을 나오게 되었다.<sup class="reference" id="cite_ref-2"><a href="#cite_note-2">[2]</a></sup> 2015년 겨울, 풋볼 리그 챔피언십 소속의 <a class="mw-redirect" href="/wiki/%EC%9C%84%EA%B1%B4_%EC%95%A0%EC%8A%AC%EB%A0%88%ED%8B%B1" title="위건 애슬레틱">위건 애슬레틱</a>에 입단하였다. 2015년 5월에 위건이 <a href="/wiki/%ED%92%8B%EB%B3%BC_%EB%A6%AC%EA%B7%B8_1" title="풋볼 리그 1">풋볼 리그 1</a>으로 강등된 후 상호 계약 해지하였다. 이후 <a href="/wiki/%EB%B8%94%EB%9E%99%EB%B2%88_%EB%A1%9C%EB%B2%84%EC%8A%A4_FC" title="블랙번 로버스 FC">블랙번</a> 입단을 타진하였으나 강화된 영국 정부의 취업허가 기준(유럽연합 회원국이 아닌 역외 외국인은 소속 국가 축구협회의 FIFA 랭킹이 상위 50위 안에 들어야 하고 소속 국가대표팀 경기에 최근 2년 간 75% 이상 참가해서 뛰어야 한다.) 때문에 입단이 무효화되었다.</p>
<p>2015년 8월 <a href="/wiki/%EB%A7%88%EC%93%B0%EB%AA%A8%ED%86%A0_%EC%95%BC%EB%A7%88%EA%B0%80_FC" title="마쓰모토 야마가 FC">마쓰모토 야마가</a>에 3개월 단기계약으로 입단하며 일본 무대로 복귀하였지만 팀의 <a class="mw-redirect" href="/wiki/J%EB%A6%AC%EA%B7%B8_%EB%94%94%EB%B9%84%EC%A0%84_2" title="J리그 디비전 2">J리그 디비전 2</a> 강등을 막지 못한채 계약이 종료되었다.</p>
<p>2016년 1월 3일 <a href="/wiki/%EC%A0%84%EB%B6%81_%ED%98%84%EB%8C%80_%EB%AA%A8%ED%84%B0%EC%8A%A4" title="전북 현대 모터스">전북 현대 모터스</a>에 입단하며 K리그 무대에서 뛰게 되었다. 2016년 4월 16일 <a href="/wiki/%EC%84%B1%EB%82%A8_FC" title="성남 FC">성남 FC</a>를 상대로 K리그 클래식 데뷔골을 넣었다</p>
<h3><span class="mw-headline" id=".EA.B5.AD.EA.B0.80.EB.8C.80.ED.91.9C_.EC.83.9D.ED.99.9C">국가대표 생활</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EA%B9%80%EB%B3%B4%EA%B2%BD_(%EC%B6%95%EA%B5%AC_%EC%84%A0%EC%88%98)&amp;action=edit&amp;section=4" title="부분 편집: 국가대표 생활">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p><a href="/wiki/%EB%8C%80%ED%95%9C%EB%AF%BC%EA%B5%AD_U-20_%EC%B6%95%EA%B5%AC_%EA%B5%AD%EA%B0%80%EB%8C%80%ED%91%9C%ED%8C%80" title="대한민국 U-20 축구 국가대표팀">대한민국 U-20 국가대표팀</a> 소속으로 <a href="/wiki/2009%EB%85%84_FIFA_U-20_%EC%9B%94%EB%93%9C%EC%BB%B5" title="2009년 FIFA U-20 월드컵">2009년 FIFA U-20 월드컵</a>에 출전하여 대회에서 총 2골을 터뜨렸다.</p>
<p><a href="/wiki/2010%EB%85%84" title="2010년">2010년</a> <a href="/wiki/1%EC%9B%94_9%EC%9D%BC" title="1월 9일">1월 9일</a>, <a href="/wiki/%EC%9E%A0%EB%B9%84%EC%95%84_%EC%B6%95%EA%B5%AC_%EA%B5%AD%EA%B0%80%EB%8C%80%ED%91%9C%ED%8C%80" title="잠비아 축구 국가대표팀">잠비아</a>와의 친선경기에 출전하여 성인 국가대표팀 소속으로 <a href="/wiki/A%EB%A7%A4%EC%B9%98" title="A매치">A매치</a> 데뷔전을 치렀다. <a href="/wiki/2010%EB%85%84_FIFA_%EC%9B%94%EB%93%9C%EC%BB%B5" title="2010년 FIFA 월드컵">2010년 FIFA 월드컵</a>, <a href="/wiki/2011%EB%85%84_AFC_%EC%95%84%EC%8B%9C%EC%95%88%EC%BB%B5" title="2011년 AFC 아시안컵">2011년 AFC 아시안컵</a>에 꾸준히 참가하여 대표팀에서 점차 자리를잡았고 <a href="/wiki/2012%EB%85%84" title="2012년">2012년</a> <a href="/wiki/6%EC%9B%94_12%EC%9D%BC" title="6월 12일">6월 12일</a> <a href="/wiki/%EB%A0%88%EB%B0%94%EB%85%BC_%EC%B6%95%EA%B5%AC_%EA%B5%AD%EA%B0%80%EB%8C%80%ED%91%9C%ED%8C%80" title="레바논 축구 국가대표팀">레바논</a>과의 <a href="/wiki/2014%EB%85%84_FIFA_%EC%9B%94%EB%93%9C%EC%BB%B5_%EC%95%84%EC%8B%9C%EC%95%84_%EC%A7%80%EC%97%AD_%EC%B5%9C%EC%A2%85_%EC%98%88%EC%84%A0" title="2014년 FIFA 월드컵 아시아 지역 최종 예선">2014년 FIFA 월드컵 예선전</a>에서 2골을 넣으며 <a href="/wiki/%EB%8C%80%ED%95%9C%EB%AF%BC%EA%B5%AD_%EC%B6%95%EA%B5%AC_%EA%B5%AD%EA%B0%80%EB%8C%80%ED%91%9C%ED%8C%80" title="대한민국 축구 국가대표팀">국가대표팀</a> 데뷔골을 기록함과 동시에 <a href="/wiki/%EB%8C%80%ED%95%9C%EB%AF%BC%EA%B5%AD_%EC%B6%95%EA%B5%AC_%EA%B5%AD%EA%B0%80%EB%8C%80%ED%91%9C%ED%8C%80" title="대한민국 축구 국가대표팀">대한민국</a>의 3-0 승리를 이끌었다.</p>
<p><a href="/wiki/2012%EB%85%84_%ED%95%98%EA%B3%84_%EC%98%AC%EB%A6%BC%ED%94%BD" title="2012년 하계 올림픽">2012년 하계 올림픽</a> <a href="/wiki/%EB%8C%80%ED%95%9C%EB%AF%BC%EA%B5%AD_U-23_%EC%B6%95%EA%B5%AC_%EA%B5%AD%EA%B0%80%EB%8C%80%ED%91%9C%ED%8C%80" title="대한민국 U-23 축구 국가대표팀">대한민국 U-23 축구 국가대표팀</a> 미드필더로 맹활약하며 <a href="/wiki/%EC%8A%A4%EC%9C%84%EC%8A%A4_%EC%B6%95%EA%B5%AC_%EA%B5%AD%EA%B0%80%EB%8C%80%ED%91%9C%ED%8C%80" title="스위스 축구 국가대표팀">스위스</a>전에서 역전골이자 결승골을 성공시켰고, 3·4위전이자 <a href="/wiki/%EB%8F%99%EB%A9%94%EB%8B%AC" title="동메달">동메달</a> 결정전인 <a href="/wiki/%EC%9D%BC%EB%B3%B8_%EC%B6%95%EA%B5%AC_%EA%B5%AD%EA%B0%80%EB%8C%80%ED%91%9C%ED%8C%80" title="일본 축구 국가대표팀">일본</a>전에서는 상대수비를 공략하고 중거리슛을 날려 골대를 맞추는등 맹활약하여 2:0으로 승리하며 동메달 획득에 기여했다.</p>
<p><a href="/wiki/2014%EB%85%84_FIFA_%EC%9B%94%EB%93%9C%EC%BB%B5" title="2014년 FIFA 월드컵">2014년 FIFA 월드컵</a>에도 발탁되어 조별리그 1차전인 <a href="/wiki/%EB%9F%AC%EC%8B%9C%EC%95%84" title="러시아">러시아</a>전과 3차전인 <a href="/wiki/%EB%B2%A8%EA%B8%B0%EC%97%90" title="벨기에">벨기에</a>전에 교체 출전하였다.</p>
<p>월드컵 종료 후엔 한동안 발탁되지 못하다가 2015년 3월 <a href="/wiki/%EC%9A%B0%EC%A6%88%EB%B2%A0%ED%82%A4%EC%8A%A4%ED%83%84_%EC%B6%95%EA%B5%AC_%EA%B5%AD%EA%B0%80%EB%8C%80%ED%91%9C%ED%8C%80" title="우즈베키스탄 축구 국가대표팀">우즈베키스탄</a>과 <a href="/wiki/%EB%89%B4%EC%A7%88%EB%9E%9C%EB%93%9C_%EC%B6%95%EA%B5%AC_%EA%B5%AD%EA%B0%80%EB%8C%80%ED%91%9C%ED%8C%80" title="뉴질랜드 축구 국가대표팀">뉴질랜드</a>와의 친선 경기에 약 8개월만에 대표팀에 소집되었다. <a href="/wiki/%EB%89%B4%EC%A7%88%EB%9E%9C%EB%93%9C_%EC%B6%95%EA%B5%AC_%EA%B5%AD%EA%B0%80%EB%8C%80%ED%91%9C%ED%8C%80" title="뉴질랜드 축구 국가대표팀">뉴질랜드</a>전에 선발 출전하여 <a href="/wiki/%EC%9D%B4%EC%9E%AC%EC%84%B1_(1992%EB%85%84)" title="이재성 (1992년)">이재성</a>의 결승골을 어시스트했다.</p>
<p><a href="/wiki/2016%EB%85%84_11%EC%9B%94" title="2016년 11월">2016년 11월</a>에 발표된 <a href="/wiki/%EC%BA%90%EB%82%98%EB%8B%A4" title="캐나다">캐나다</a>와의 친선경기 및 <a href="/wiki/%EC%9A%B0%EC%A6%88%EB%B2%A0%ED%82%A4%EC%8A%A4%ED%83%84" title="우즈베키스탄">우즈베키스탄</a>과의 <a href="/wiki/2018%EB%85%84_FIFA_%EC%9B%94%EB%93%9C%EC%BB%B5_%EC%98%88%EC%84%A0" title="2018년 FIFA 월드컵 예선">2018년 FIFA 월드컵 예선</a> 경기 명단에 포함되었고 <a href="/wiki/%EC%BA%90%EB%82%98%EB%8B%A4" title="캐나다">캐나다</a> 전에 선발출장하여 선제골을 기록하였는데, 이 골은 2013년 10월 <a href="/wiki/%EB%A7%90%EB%A6%AC" title="말리">말리</a> 전에서 기록한 골 이후 3년만에 넣은 <a href="/wiki/A%EB%A7%A4%EC%B9%98" title="A매치">A매치</a> 골이었다. 이후 <a class="mw-disambig" href="/wiki/%EC%9D%B4%EC%A0%95%ED%98%91" title="이정협">이정협</a>의 추가골로 2:0으로 승리하였다.</p>
