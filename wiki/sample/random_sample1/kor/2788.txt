<p><b>김보성</b>(金甫城, <span style="font-size:85%;">본명</span>: <b>허석</b>, <span style="font-size:85%;">본명 한자 음역 표기</span>: 許碩), <a href="/wiki/1966%EB%85%84" title="1966년">1966년</a> <a href="/wiki/6%EC%9B%94_27%EC%9D%BC" title="6월 27일">6월 27일</a> ~ )은 <a href="/wiki/%EB%8C%80%ED%95%9C%EB%AF%BC%EA%B5%AD" title="대한민국">대한민국</a>의 <a href="/wiki/%EB%B0%B0%EC%9A%B0" title="배우">배우</a>이다.</p>
<p><a href="/wiki/%EA%B0%95%EC%9B%90%EB%8F%84" title="강원도">강원도</a> <a href="/wiki/%EA%B0%95%EB%A6%89%EC%8B%9C" title="강릉시">강릉시</a> 출신으로서, <a href="/wiki/%EC%9D%BC%EC%A0%9C_%EA%B0%95%EC%A0%90%EA%B8%B0" title="일제 강점기">일제 강점기</a> 말기 <a href="/wiki/%EC%8B%9C_(%EB%AC%B8%ED%95%99)" title="시 (문학)">시인</a> <a href="/wiki/%EC%84%A4%EC%A0%95%EC%8B%9D" title="설정식">설정식</a>의 외손자이다. <a href="/wiki/1987%EB%85%84" title="1987년">1987년</a> <a href="/wiki/%EC%97%B0%EA%B7%B9" title="연극">연극</a> 《<a href="/wiki/%ED%96%84%EB%A6%BF" title="햄릿">햄릿</a>》의 단역 출연으로 <a class="mw-redirect" href="/wiki/%EC%97%B0%EA%B7%B9%EB%B0%B0%EC%9A%B0" title="연극배우">연극배우</a>로 데뷔했고 <a href="/wiki/1988%EB%85%84" title="1988년">1988년</a> <a class="mw-redirect" href="/wiki/%EB%8C%80%ED%95%9C%EB%AF%BC%EA%B5%AD_%EC%98%81%ED%99%94" title="대한민국 영화">영화</a> 《그대 원하면》의 단역을 통하여 <a href="/wiki/%EC%98%81%ED%99%94%EB%B0%B0%EC%9A%B0" title="영화배우">영화배우</a>로 데뷔했다. 본명을 사용하다가 <a href="/wiki/%EC%98%88%EB%AA%85" title="예명">예명</a>인 <b>김보성</b>을 사용하며 활동하고 있다.</p>

<h2><span class="mw-headline" id=".EA.B8.B0.ED.83.80">기타</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EA%B9%80%EB%B3%B4%EC%84%B1_(%EB%B0%B0%EC%9A%B0)&amp;action=edit&amp;section=2" title="부분 편집: 기타">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>친구를 구하기 위해 17명에 맞서 싸우다 한쪽 눈을 실명해서 <a class="mw-redirect" href="/wiki/%EC%8B%9C%EA%B0%81%EC%9E%A5%EC%95%A0" title="시각장애">시각장애</a> 6급으로 병역이 면제되었다고 방송에서 밝혔다.<sup class="reference" id="cite_ref-1"><a href="#cite_note-1">[1]</a></sup> 이런 시각장애 사실을 숨기고 <a class="mw-redirect" href="/wiki/%ED%8A%B9%EC%A0%84%EC%82%AC" title="특전사">특전사</a>에 지원했지만 신체검사에서 시각장애가 있다는 사실이 밝혀져 떨어졌다.<sup class="reference" id="cite_ref-2"><a href="#cite_note-2">[2]</a></sup></p>
<h2><span class="mw-headline" id=".EA.B2.A9.ED.88.AC.EA.B8.B0">격투기</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EA%B9%80%EB%B3%B4%EC%84%B1_(%EB%B0%B0%EC%9A%B0)&amp;action=edit&amp;section=3" title="부분 편집: 격투기">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>2016년 12월 10일 <a href="/wiki/%EC%9E%A5%EC%B6%A9%EC%B2%B4%EC%9C%A1%EA%B4%80" title="장충체육관">장충체육관</a>에서 <a href="/wiki/%EC%9D%BC%EB%B3%B8" title="일본">일본</a>의 콘도 테츠오를 상대로 <a href="/wiki/%EC%A2%85%ED%95%A9%EA%B2%A9%ED%88%AC%EA%B8%B0" title="종합격투기">종합격투기</a> <a href="/wiki/%EB%A1%9C%EB%93%9C_FC" title="로드 FC">로드 FC</a> 데뷔전을 치뤘다. 대전료 전액은 소아암 환자 수술비로 기부하며, 소아암 환자의 아픔을 함께하고 결의를 보이기 위해 삭발식까지 거행하였다.<sup class="reference" id="cite_ref-3"><a href="#cite_note-3">[3]</a></sup> 경기 결과는 1라운드 경기 포기에 의한 TKO패배를 당했고 오른쪽 눈 주위의 뼈가 함몰되는 <a href="/wiki/%EC%95%88%EC%99%80" title="안와">안와</a><a href="/wiki/%EA%B3%A8%EC%A0%88" title="골절">골절</a> 부상을 입었지만 시력 보호를 위해 <a href="/wiki/%EC%88%98%EC%88%A0_(%EC%9D%98%ED%95%99)" title="수술 (의학)">수술</a>을 포기했다.<sup class="reference" id="cite_ref-4"><a href="#cite_note-4">[4]</a></sup><sup class="reference" id="cite_ref-5"><a href="#cite_note-5">[5]</a></sup></p>
