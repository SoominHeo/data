<p><a href="/wiki/%EA%B2%BD%EA%B8%B0%EB%8F%84" title="경기도">경기도</a> <a href="/wiki/%EC%95%88%EC%84%B1%EC%8B%9C" title="안성시">안성</a>에서 출생하였고 <a href="/wiki/%EC%9D%B4%EC%B2%9C%EC%8B%9C" title="이천시">경기도 이천</a>과 <a href="/wiki/%ED%8F%89%ED%83%9D%EC%8B%9C" title="평택시">경기도 평택</a>에서 잠시 유년기를 보낸 적이 있는 그는 <a href="/wiki/%ED%95%9C%EA%B2%BD%EB%8C%80%ED%95%99%EA%B5%90" title="한경대학교">경기도 안성고등농림학교</a>를 거쳐 <a href="/wiki/%EC%84%9C%EC%9A%B8%EB%8C%80%ED%95%99%EA%B5%90" title="서울대학교">서울대학교</a> 사범대학의 전신인 <a href="/wiki/%EC%84%9C%EC%9A%B8%EB%8C%80%ED%95%99%EA%B5%90" title="서울대학교">서울사범학교</a>를 졸업하였다. 청소년 시절 문학에 관심을 가져 습작 <a href="/wiki/%EC%86%8C%EC%84%A4" title="소설">소설</a>을 쓰기도 했고, 자작 <a href="/wiki/%ED%9D%AC%EA%B3%A1" title="희곡">희곡</a>을 무대에 올려 <a href="/wiki/%EC%97%B0%EC%B6%9C%EA%B0%80" title="연출가">연출</a>과 <a class="mw-redirect" href="/wiki/%EC%97%B0%EA%B7%B9_%EB%B0%B0%EC%9A%B0" title="연극 배우">주연</a>을 겸하는 등 <a href="/wiki/%EC%97%B0%EA%B7%B9" title="연극">연극</a>에도 자질을 보였다.</p>
<p><a href="/wiki/1946%EB%85%84" title="1946년">1946년</a>경부터 <a href="/wiki/%EC%84%9C%EC%9A%B8%EA%B5%90%EC%9C%A1%EB%8C%80%ED%95%99%EA%B5%90" title="서울교육대학교">서울사범학교</a> 연극부 부장 연출가로 활동하다가 <a href="/wiki/%ED%95%9C%EA%B5%AD_%EC%A0%84%EC%9F%81" title="한국 전쟁">한국 전쟁</a> 중이던 <a href="/wiki/1951%EB%85%84" title="1951년">1951년</a>에 <a href="/wiki/%EB%8C%80%ED%95%9C%EB%AF%BC%EA%B5%AD_%EC%9C%A1%EA%B5%B0" title="대한민국 육군">대한민국 육군</a> <a href="/wiki/%EC%86%8C%EC%9C%84" title="소위">소위</a>로 <a href="/wiki/%EA%B0%91%EC%A2%85%EC%9E%A5%EA%B5%90" title="갑종장교">육군 갑종장교</a> 임관하여 <a href="/wiki/%EB%8C%80%ED%95%9C%EB%AF%BC%EA%B5%AD_%EA%B5%AD%EB%B0%A9%EB%B6%80" title="대한민국 국방부">대한민국 국방부</a> 정훈국 영화과에 배속이 되면서 처음 영화와 인연을 맺게 되었다. 정훈국에서는 작가 <a href="/wiki/%EC%84%A0%EC%9A%B0%ED%9C%98" title="선우휘">선우 휘</a>의 지휘 아래 군인 교육용 영화를 약 30여 편 만들었다.</p>
<p>당시 문관으로 근무하던 영화감독 <a href="/wiki/%EC%96%91%EC%A3%BC%EB%82%A8" title="양주남">양주남</a>의 주선으로 그의 조감독이 되면서 《배뱅이굿》(1957) 촬영에 참가한 것을 계기로 본격적인 영화 작업에 뛰어들게 되었고, 이듬해 코미디 영화인 《공처가》로 영화감독이 되었다. 초기에는 《공처가》와 같은 계열의 희극 영화를 촬영했으나, <a href="/wiki/1963%EB%85%84" title="1963년">1963년</a> <a href="/wiki/%EA%B9%80%EC%8A%B9%ED%98%B8" title="김승호">김승호</a>와 <a href="/wiki/%ED%99%A9%EC%A0%95%EC%88%9C" title="황정순">황정순</a> 주연의 《굴비》라는 영화 작품을 전환점으로 삼아 좀 더 진지한 영화로 전환했다.</p>
<p>대표작으로는 《굴비》 이후 관심을 갖게 된 사회 문제에 대한 비판 의식을 담은 《저 하늘에도 슬픔이》(1965)와 《갯마을》(1965)을 비롯하여 1970년대의 《야행》(1977), 《화려한 외출》(1978), 1980년대 작품으로는 《도시로 간 처녀》(1981), 《만추》(1982) 등이 있다. 기행 승려인 <a href="/wiki/%EC%A4%91%EA%B4%91_(%EC%8A%B9%EB%A0%A4)" title="중광 (승려)">중광</a>을 다룬 《허튼소리》(1986)는 <a href="/wiki/%EB%B6%88%EA%B5%90" title="불교">불교</a>계의 반발로 사회적인 논란 거리가 되기도 했다. 총 연출 작품 수가 100편이 넘는 다작 연출가이지만 대중성과 흥행성이 조화를 이루며 질적으로 떨어지지 않는다는 평을 받고 있다.</p>
<p><a href="/wiki/1991%EB%85%84" title="1991년">1991년</a>까지 <a href="/wiki/%EC%B2%AD%EC%A3%BC%EB%8C%80%ED%95%99%EA%B5%90" title="청주대학교">청주대학교 연극영화학과</a> 교수를 역임하기도 하였고 <a href="/wiki/1992%EB%85%84" title="1992년">1992년</a>까지 <a href="/wiki/%EC%84%9C%EC%9A%B8%EC%98%88%EC%88%A0%EB%8C%80%ED%95%99%EA%B5%90" title="서울예술대학교">서울예술전문대학 영화학과</a> 특임강사 겸 <a href="/wiki/%EC%A4%91%EC%95%99%EB%8C%80%ED%95%99%EA%B5%90" title="중앙대학교">중앙대학교 영화학과</a> 특임교수를 지내기도 한 그는 <a href="/wiki/%EB%8C%80%ED%95%9C%EB%AF%BC%EA%B5%AD%EC%98%88%EC%88%A0%EC%9B%90" title="대한민국예술원">대한민국예술원</a> 회원이며, 영화감독으로는 최초로 <a href="/wiki/%EB%8C%80%ED%95%9C%EB%AF%BC%EA%B5%AD%EC%98%88%EC%88%A0%EC%9B%90" title="대한민국예술원">대한민국예술원</a> 회장에 오르기도 했다.</p>
