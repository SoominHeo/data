<p><b>김용택</b>(金龍澤, <a href="/wiki/1948%EB%85%84" title="1948년">1948년</a> <a href="/wiki/8%EC%9B%94_26%EC%9D%BC" title="8월 26일">8월 26일</a> ~ )은 <a href="/wiki/%EB%8C%80%ED%95%9C%EB%AF%BC%EA%B5%AD" title="대한민국">대한민국</a>의 <a href="/wiki/%EC%8B%9C_(%EB%AC%B8%ED%95%99)" title="시 (문학)">시인</a> 겸 <a href="/wiki/%EC%88%98%ED%95%84" title="수필">수필가</a>이며 약 40년간 초등교사생활을 은퇴하였고 <a href="/wiki/%EB%B3%B8%EA%B4%80" title="본관">본관</a>은 <a href="/wiki/%EA%B9%80%ED%95%B4_%EA%B9%80%EC%94%A8" title="김해 김씨">김해(金海)</a>이며 <a href="/wiki/%ED%98%B8_(%EC%9D%B4%EB%A6%84)" title="호 (이름)">호(號)</a>는 섬호(蟾滸)이다.</p>

<h2><span class="mw-headline" id=".EC.83.9D.EC.95.A0">생애</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EA%B9%80%EC%9A%A9%ED%83%9D&amp;action=edit&amp;section=1" title="부분 편집: 생애">원본 편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>그는 섬진강 연작으로 유명하여 ‘<a href="/wiki/%EC%84%AC%EC%A7%84%EA%B0%95" title="섬진강">섬진강</a> 시인’이라는 별칭이 있다.<sup class="reference" id="cite_ref-1"><a href="#cite_note-1">[1]</a></sup></p>
<p><a href="/wiki/%EC%A0%84%EB%9D%BC%EB%B6%81%EB%8F%84" title="전라북도">전라북도</a> <a href="/wiki/%EC%9E%84%EC%8B%A4%EA%B5%B0" title="임실군">임실군</a> 진메마을에서 태어나 <a href="/wiki/%EC%88%9C%EC%B0%BD%EC%A0%9C%EC%9D%BC%EA%B3%A0%EB%93%B1%ED%95%99%EA%B5%90" title="순창제일고등학교">순창농림고등학교</a>를 졸업했다. 그 이듬해에 우연히 친구들을 따라<sup class="reference" id="cite_ref-ReferenceA_2-0"><a href="#cite_note-ReferenceA-2">[2]</a></sup> 교사시험을 보고 스물한 살에 초등학교 교사가 되었고,<sup class="reference" id="cite_ref-3"><a href="#cite_note-3">[3]</a></sup> 교직 기간 동안 자신의 모교이기도 한 임실 운암초등학교 마암분교(현 <a href="/wiki/%EB%A7%88%EC%95%94%EC%B4%88%EB%93%B1%ED%95%99%EA%B5%90_(%EC%A0%84%EB%B6%81)" title="마암초등학교 (전북)">마암초등학교</a>)에서 아이들을 가르치며 시를 썼다. 교직 기간 동안 종종 가르치는 아이들의 시를 모아 펴내기도 하였으며,<sup class="reference" id="cite_ref-4"><a href="#cite_note-4">[4]</a></sup> 그가 교사로 재직하고 있다는 사실이 알려지자 학생들이 전학오면서 2005년에 마암분교가 마암초등학교로 승격되었다. <a href="/wiki/2009%EB%85%84" title="2009년">2009년</a> <a href="/wiki/2%EC%9B%94_28%EC%9D%BC" title="2월 28일">2월 28일</a>자로 교직을 정년 퇴임했다.</p>
<h3><span class="mw-headline" id=".EC.95.BD.EB.A0.A5">약력</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EA%B9%80%EC%9A%A9%ED%83%9D&amp;action=edit&amp;section=2" title="부분 편집: 약력">원본 편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p><a href="/wiki/1982%EB%85%84" title="1982년">1982년</a> 《창작과비평사》에서 펴낸 21인 신작 시집 《꺼지지 않는 횃불로》에 〈섬진강 1〉외 8편의 시를 발표하면서 등단했다. 이 때가 나이 35세였다.<sup class="reference" id="cite_ref-ReferenceA_2-1"><a href="#cite_note-ReferenceA-2">[2]</a></sup> <a href="/wiki/1986%EB%85%84" title="1986년">1986년</a> 6회「<a class="mw-redirect" href="/wiki/%EA%B9%80%EC%88%98%EC%98%81_%EB%AC%B8%ED%95%99%EC%83%81" title="김수영 문학상">김수영문학상</a>」, <a href="/wiki/1998%EB%85%84" title="1998년">1998년</a> 12회 「<a href="/wiki/%EC%86%8C%EC%9B%94%EC%8B%9C%EB%AC%B8%ED%95%99%EC%83%81" title="소월시문학상">소월시문학상</a>」을 수상했다.</p>
<p><a href="/wiki/2001%EB%85%84" title="2001년">2001년</a>에는 <a href="/wiki/%EC%98%81%ED%99%94" title="영화">독립단편영화</a> 《들》의 조연으로 <a class="mw-redirect" href="/wiki/%EC%98%81%ED%99%94_%EB%B0%B0%EC%9A%B0" title="영화 배우">영화배우</a>에 데뷔하였고 <a href="/wiki/2010%EB%85%84" title="2010년">2010년</a>에는 <a href="/wiki/%EC%9D%B4%EC%B0%BD%EB%8F%99" title="이창동">이창동</a>이 <a href="/wiki/%EC%98%81%ED%99%94_%EA%B0%90%EB%8F%85" title="영화 감독">감독</a>한 <a href="/wiki/%EC%98%81%ED%99%94" title="영화">영화</a> 《<a href="/wiki/%EC%8B%9C_(%EC%98%81%ED%99%94)" title="시 (영화)">시</a>》에 조연하기도 하였다.</p>
