<p><b>김유성</b>(<a href="/wiki/1988%EB%85%84" title="1988년">1988년</a> <a href="/wiki/12%EC%9B%94_4%EC%9D%BC" title="12월 4일">12월 4일</a> ~ )은 <a href="/wiki/%EB%8C%80%ED%95%9C%EB%AF%BC%EA%B5%AD" title="대한민국">대한민국</a>의 <a href="/wiki/%EC%B6%95%EA%B5%AC" title="축구">축구</a> 선수이며 포지션은 <a href="/wiki/%EB%AF%B8%EB%93%9C%ED%95%84%EB%8D%94" title="미드필더">미드필더</a>이다. 현재 <a href="/wiki/%EB%8C%80%ED%95%9C%EB%AF%BC%EA%B5%AD" title="대한민국">대한민국</a> <a href="/wiki/%EA%B3%A0%EC%96%91_%EC%9E%90%EC%9D%B4%ED%81%AC%EB%A1%9C_FC" title="고양 자이크로 FC">고양 자이크로 FC</a>에서 활약하고 있다.</p>
<h2><span class="mw-headline" id=".EC.B6.95.EA.B5.AC.EC.9D.B8_.EA.B2.BD.EB.A0.A5">축구인 경력</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EA%B9%80%EC%9C%A0%EC%84%B1_(%EC%B6%95%EA%B5%AC_%EC%84%A0%EC%88%98)&amp;action=edit&amp;section=1" title="부분 편집: 축구인 경력">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<h3><span class="mw-headline" id=".EC.84.A0.EC.88.98_.EA.B2.BD.EB.A0.A5">선수 경력</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EA%B9%80%EC%9C%A0%EC%84%B1_(%EC%B6%95%EA%B5%AC_%EC%84%A0%EC%88%98)&amp;action=edit&amp;section=2" title="부분 편집: 선수 경력">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p><a href="/wiki/2010_K%EB%A6%AC%EA%B7%B8_%EB%93%9C%EB%9E%98%ED%94%84%ED%8A%B8" title="2010 K리그 드래프트">2010 K리그 드래프트</a>에서 <a href="/wiki/%EA%B2%BD%EB%82%A8_FC" title="경남 FC">경남 FC</a>에 지명되며 <a href="/wiki/K%EB%A6%AC%EA%B7%B8" title="K리그">K리그</a> 무대에 발을 들여 놓았다. 데뷔 시즌 리그 3경기에 출전하였고 <a href="/wiki/K%EB%A6%AC%EA%B7%B8_2011" title="K리그 2011">2011 시즌</a> 중반 <a href="/wiki/%EB%8C%80%EA%B5%AC_FC" title="대구 FC">대구 FC</a>로 이적하였다. <a href="/wiki/K%EB%A6%AC%EA%B7%B8_2012" title="K리그 2012">2012 시즌</a> <a href="/wiki/5%EC%9B%94_5%EC%9D%BC" title="5월 5일">5월 5일</a> <a href="/wiki/%EA%B4%91%EC%A3%BC_FC" title="광주 FC">광주</a>와의 경기에 출전하여 시즌 첫 출전을 기록하였고 <a href="/wiki/8%EC%9B%94_19%EC%9D%BC" title="8월 19일">8월 19일</a> <a href="/wiki/%ED%8F%AC%ED%95%AD_%EC%8A%A4%ED%8B%B8%EB%9F%AC%EC%8A%A4" title="포항 스틸러스">포항</a>과의 경기에선 득점을 성공시키며 K리그 데뷔골을 기록하였다.</p>
