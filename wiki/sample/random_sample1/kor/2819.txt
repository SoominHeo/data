<p><b>김일제</b>(金日磾, <a href="/wiki/%EA%B8%B0%EC%9B%90%EC%A0%84_134%EB%85%84" title="기원전 134년">기원전 134년</a> ~ <a href="/wiki/%EA%B8%B0%EC%9B%90%EC%A0%84_86%EB%85%84" title="기원전 86년">기원전 86년</a> <a href="/wiki/%EC%9D%8C%EB%A0%A5_8%EC%9B%94" title="음력 8월">음력 8월</a>)는 <a href="/wiki/%EC%A0%84%ED%95%9C" title="전한">전한</a>의 정치가로, <a href="/wiki/%EC%9E%90_(%EC%9D%B4%EB%A6%84)" title="자 (이름)">자</a>는 <b>옹숙</b>(翁叔)이다. 본래 <a href="/wiki/%ED%9D%89%EB%85%B8" title="흉노">흉노</a> 출신으로 흉노의 <a class="mw-redirect" href="/wiki/%EB%B2%88%EC%99%95" title="번왕">번왕</a>인 <a href="/wiki/%ED%9C%B4%EB%8F%84%EC%99%95" title="휴도왕">휴도왕</a>(休屠王)의 장남으로 태어났으며, 열네 살 때 부왕이 <a href="/wiki/%EC%A0%84%ED%95%9C_%EB%AC%B4%EC%A0%9C" title="전한 무제">무제</a>(武帝)와의 전투에서 패하면서 전한에 포로로 끌려왔다. 이후 무제의 신임을 받아 전한의 관료로 일하면서 <a class="mw-redirect" href="/wiki/%EA%B9%80%EC%94%A8" title="김씨">김씨</a>(金氏) 성을 받았으며, 말년에 <a class="mw-redirect" href="/wiki/%ED%88%AC%ED%9B%84" title="투후">투후</a>(秺侯)에 봉해졌다. 신라의 왕족 김씨 역시 김일제의 후손이라는 기록이 있으나,<sup class="reference" id="cite_ref-.EB.AC.B8.EA.B2.BD.ED.98.84_1-0"><a href="#cite_note-.EB.AC.B8.EA.B2.BD.ED.98.84-1">[1]</a></sup> 당시 신라인들이 스스로 김일제의 후손임을 자처했다는 증거는 될 수 있어도, 실제로 그의 후손일 가능성은 크지 않다고 보고 있다.</p>

<h2><span class="mw-headline" id=".EC.83.9D.EC.95.A0">생애</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EA%B9%80%EC%9D%BC%EC%A0%9C&amp;action=edit&amp;section=1" title="부분 편집: 생애">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<h3><span class="mw-headline" id=".EC.B4.88.EA.B8.B0_.EC.82.B6">초기 삶</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EA%B9%80%EC%9D%BC%EC%A0%9C&amp;action=edit&amp;section=2" title="부분 편집: 초기 삶">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>김일제는 <a href="/wiki/%ED%9D%89%EB%85%B8" title="흉노">흉노</a>의 <a class="mw-redirect" href="/wiki/%EB%B2%88%EC%99%95" title="번왕">번왕</a>인 <a href="/wiki/%ED%9C%B4%EB%8F%84%EC%99%95" title="휴도왕">휴도왕</a>(休屠王)의 장남으로 태어났다. <a href="/wiki/%EA%B8%B0%EC%9B%90%EC%A0%84_150%EB%85%84" title="기원전 150년">기원전 150년</a> 경 <a class="mw-redirect" href="/wiki/%EC%9D%B4%EC%B9%98%EC%82%AC_%EC%84%A0%EC%9A%B0" title="이치사 선우">이치사 선우</a>의 번왕이었던 <a href="/wiki/%ED%9C%B4%EB%8F%84%EC%99%95" title="휴도왕">휴도왕</a>(休屠王)과 곤야왕(昆邪王)이 계속하여 <a href="/wiki/%ED%95%9C%EB%82%98%EB%9D%BC" title="한나라">한나라</a>에 패배하자, 그들을 송환하여 사형으로 그 죄를 물으려고 하였다. 곤야왕은 <a class="mw-redirect" href="/wiki/%EC%9D%B4%EC%B9%98%EC%82%AC_%EC%84%A0%EC%9A%B0" title="이치사 선우">이치사 선우</a>에게 끌려가면 목숨에 위험하다고 판단하고 <a href="/wiki/%ED%9C%B4%EB%8F%84%EC%99%95" title="휴도왕">휴도왕</a>을 설득하여 <a href="/wiki/%ED%95%9C%EB%82%98%EB%9D%BC" title="한나라">한나라</a>에 투항하려고 하였다. 곤야왕은 <a href="/wiki/%ED%9C%B4%EB%8F%84%EC%99%95" title="휴도왕">휴도왕</a>이 <a href="/wiki/%ED%95%9C%EB%82%98%EB%9D%BC" title="한나라">한나라</a> 투항에 반대하자, 그를 살해하고 <a href="/wiki/%ED%95%9C%EB%82%98%EB%9D%BC" title="한나라">한나라</a>에 투항하였으며, <a href="/wiki/%ED%9C%B4%EB%8F%84%EC%99%95" title="휴도왕">휴도왕</a>의 부인과 두 아들을 <a href="/wiki/%ED%95%9C%EB%82%98%EB%9D%BC" title="한나라">한나라</a> 장수인 <a href="/wiki/%EA%B3%BD%EA%B1%B0%EB%B3%91" title="곽거병">곽거병</a>에게 포로로 넘겨주었다.</p>
<h3><span class="mw-headline" id=".EC.A4.91.EA.B5.AD_.ED.8F.AC.EB.A1.9C_.EC.83.9D.ED.99.9C">중국 포로 생활</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EA%B9%80%EC%9D%BC%EC%A0%9C&amp;action=edit&amp;section=3" title="부분 편집: 중국 포로 생활">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p><a href="/wiki/%ED%95%9C%EB%82%98%EB%9D%BC" title="한나라">한나라</a>의 <a href="/wiki/%EA%B3%BD%EA%B1%B0%EB%B3%91" title="곽거병">곽거병</a>(霍去病) 장군은 <a href="/wiki/%ED%9C%B4%EB%8F%84%EC%99%95" title="휴도왕">휴도왕</a>을 토벌하고 그 부인 연지(閼氏)와 장남 일제 및 어린 동생 윤(胤)을 사로잡아 <a href="/wiki/%ED%95%9C%EB%82%98%EB%9D%BC" title="한나라">한나라</a>의 수도인 <a href="/wiki/%EC%9E%A5%EC%95%88" title="장안">장안</a>으로 돌아왔다. 이 때 <a href="/wiki/%ED%95%9C%EB%82%98%EB%9D%BC" title="한나라">한나라</a>에 포로로 잡혀온 일제의 나이는 14세였다. 말 기르는 노예가 된 일제는 우연히 <a href="/wiki/%EC%A0%84%ED%95%9C_%EB%AC%B4%EC%A0%9C" title="전한 무제">무제</a>의 눈에 띄어 노예에서 해방되고 마감(馬監)으로 임명되었으며, 이어 시중(侍中), 부마도위(附馬都衛), 광록대부(光祿大夫)에 올랐다.</p>
<h3><span class="mw-headline" id=".EA.B9.80.EC.94.A8.EC.84.B1_.EC.82.AC.EC.84.B1">김씨성 사성</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EA%B9%80%EC%9D%BC%EC%A0%9C&amp;action=edit&amp;section=4" title="부분 편집: 김씨성 사성">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>일제는 망하라(莽何羅) 등의 <a href="/wiki/%EC%A0%84%ED%95%9C_%EB%AC%B4%EC%A0%9C" title="전한 무제">무제</a> 암살 시도를 막아 그 공으로 거기장군(車騎將軍)이 되었고, <a class="mw-redirect" href="/wiki/%EA%B9%80%EC%94%A8" title="김씨">김(金)씨</a> 성을 하사받았다. 김(金)이란 금(金) 즉 황금(黃金)을 뜻하는데, 김일제의 아버지인 <a href="/wiki/%ED%9C%B4%EB%8F%84%EC%99%95" title="휴도왕">휴도왕</a>이 금인(金人)을 가지고 천주(天主)에 제사 지냈던 일에서 비롯하여 성을 김(金)으로 하였다.</p>
<p>김일제에게는 두 명의 아들이 있었는데, 그 가운데 한 명이 성장하여 공주를 유혹하였다. 김일제는 그 행동이 매우 불순하다고 생각하여 자식을 죽이고 <a href="/wiki/%EC%A0%84%ED%95%9C_%EB%AC%B4%EC%A0%9C" title="전한 무제">무제</a>에게 보고하였다. <a href="/wiki/%EC%A0%84%ED%95%9C_%EB%AC%B4%EC%A0%9C" title="전한 무제">무제</a>는 상심하였지만 김일제의 충성심에 더욱 감명되었다. <a href="/wiki/%EC%A0%84%ED%95%9C_%EB%AC%B4%EC%A0%9C" title="전한 무제">무제</a>가 죽으면서 김일제와 <a href="/wiki/%EA%B3%BD%EA%B4%91" title="곽광">곽광</a>(霍光), <a href="/wiki/%EC%83%81%EA%B4%80%EA%B1%B8" title="상관걸">상관걸</a>(上官桀) 등을 어린 아들 <a class="mw-redirect" href="/wiki/%EC%9C%A0%EB%B6%88%EB%A6%89" title="유불릉">유불릉</a>의 후견인으로 지목하여 <a href="/wiki/%EC%A0%84%ED%95%9C_%EC%86%8C%EC%A0%9C_(8%EB%8C%80)" title="전한 소제 (8대)">소제</a>를 보필하기도 했다.</p>
<h3><span class="mw-headline" id=".EC.B5.9C.ED.9B.84">최후</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EA%B9%80%EC%9D%BC%EC%A0%9C&amp;action=edit&amp;section=5" title="부분 편집: 최후">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>김일제가 병이 들어 죽기 직전 <a href="/wiki/%EC%A0%84%ED%95%9C_%EC%86%8C%EC%A0%9C_(8%EB%8C%80)" title="전한 소제 (8대)">소제</a>는 <a href="/wiki/%EA%B3%BD%EA%B4%91" title="곽광">곽광</a>과 의논하여 김일제를 투후에 봉하였고, 김일제는 조서를 받은 지 하루 만에 죽었다. 작위는 아들 <a href="/wiki/%EA%B9%80%EC%83%81_(%EA%B4%91%EB%A1%9D%ED%9B%88)" title="김상 (광록훈)">김상</a>이 이었으나 김상이 죽은 후 아들이 없어 봉국이 몰수되었고, 이후 김일제의 증손 김당(金當)이 다시 투후에 봉해졌다.</p>
<p>김일제의 무덤은 <a href="/wiki/%EC%A0%84%ED%95%9C_%EB%AC%B4%EC%A0%9C" title="전한 무제">한무제</a> 무릉의 배장묘 가운데 하나로서 <a href="/wiki/%EA%B3%BD%EA%B1%B0%EB%B3%91" title="곽거병">곽거병</a>의 묘 오른쪽에 있는데, 오늘날 <a class="mw-redirect" href="/wiki/%EA%B0%90%EC%88%99%EC%84%B1" title="감숙성">감숙성</a> 흥평현(兴平县) 남귀향 도상촌에 있다.<sup class="reference" id="cite_ref-2"><a href="#cite_note-2">[2]</a></sup> 감숙성 무위시에 김일제의 동상이 세워져 있으며 마신(馬神)이라 전해져 내려온다.</p>
<h3><span class="mw-headline" id=".EC.82.AC.ED.9B.84">사후</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EA%B9%80%EC%9D%BC%EC%A0%9C&amp;action=edit&amp;section=6" title="부분 편집: 사후">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>김일제의 자손들은 <a href="/wiki/%EC%8B%A0%EB%82%98%EB%9D%BC" title="신나라">신나라</a> 때 <a href="/wiki/%EC%99%95%EB%A7%9D" title="왕망">왕망</a>(王莽)의 외가로서 주요 관직에 올랐는데, 후한 <a href="/wiki/%ED%9B%84%ED%95%9C_%EA%B4%91%EB%AC%B4%EC%A0%9C" title="후한 광무제">광무제</a>가 <a href="/wiki/%EC%99%95%EB%A7%9D" title="왕망">왕망</a>을 토벌할 때 역사에서 사라지고, 봉국도 몰수되었다.</p>
<p><a href="/wiki/%EC%A4%91%EA%B5%AD" title="중국">중국</a> <a class="mw-redirect" href="/wiki/%EC%82%B0%EB%91%A5%EC%84%B1" title="산둥성">산둥성</a> 하택시 성무현 옥화묘촌은 김일제가 봉지로 받은 투현으로 현재 입구에 이 사실을 알리는 표지석이 남아 있다. 투후 유적지에는 김일제를 기리던 <a href="/wiki/%EC%82%AC%EB%8B%B9" title="사당">사당</a>(祠堂)인 투후사(秺侯祠)가 있었다고 하나 현재는 존재하지 않는다. 김일제는 중국 김씨의 시조이다. 김일제의 자손들은 작은 벼슬을 습직하면서 <a href="/wiki/%ED%95%9C%EB%82%98%EB%9D%BC" title="한나라">한나라</a>에서 살았고, <a href="/wiki/%EC%82%BC%EA%B5%AD%EC%A7%80" title="삼국지">삼국지</a>에도 김일제의 후손인 <a href="/wiki/%EA%B9%80%EC%84%A0_(%ED%9B%84%ED%95%9C)" title="김선 (후한)">김선</a>(金旋)과 <a href="/wiki/%EA%B9%80%EC%9C%84" title="김위">김위</a>(金褘)가 등장한다.</p>
<h2><span class="mw-headline" id=".EA.B9.80.EC.9D.BC.EC.A0.9C.EC.99.80_.EC.8B.A0.EB.9D.BC_.EA.B9.80.EC.94.A8.EC.9D.98_.EA.B4.80.EA.B3.84">김일제와 신라 김씨의 관계</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EA%B9%80%EC%9D%BC%EC%A0%9C&amp;action=edit&amp;section=7" title="부분 편집: 김일제와 신라 김씨의 관계">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p><a href="/wiki/%EC%8B%A0%EB%9D%BC" title="신라">신라</a> 문무왕릉비(文武王陵碑)에는 '투후(秺侯) 제천지윤(祭天之胤)이 7대를 전하여<sup class="reference" id="cite_ref-3"><a href="#cite_note-3">[3]</a></sup>'(5행), '15대조 <a href="/wiki/%EC%84%B1%ED%95%9C_(%EC%8B%A0%EB%9D%BC)" title="성한 (신라)">성한왕</a>(星漢王)은 그 바탕이 하늘에서 <a href="/wiki/%EC%8B%A0%EB%9D%BC" title="신라">신라</a>로 내려왔고'(6행)라는 구절이 있다. 또한 당나라에 살았던 신라인 김씨부인의 업적을 기리는 <a href="/wiki/%EB%8C%80%EB%8B%B9%EA%B3%A0%EA%B9%80%EC%94%A8%EB%B6%80%EC%9D%B8%EB%AC%98%EB%AA%85" title="대당고김씨부인묘명">대당고김씨부인묘명</a>(大唐故金氏夫人墓銘)에도 신라 김씨의 뿌리가 <a class="mw-redirect" href="/wiki/%ED%88%AC%ED%9B%84" title="투후">투후</a> 김일제라고 기록되어 있다. 이러한 기록을 볼 때 신라 김씨 왕족은 자신의 조상을 중국 한나라 때 <a class="mw-redirect" href="/wiki/%ED%88%AC%ED%9B%84" title="투후">투후</a>를 지낸 김일제라고 생각했다는 것을 알 수 있다.</p>
<p>이러한 기록에 대해서 역사학자들은 7세기 후반 들어 형성된 신라 김씨의 관념적인 시조의식의 소산이지 실제로 김일제가 김씨의 시조인 것은 아니라고 판단하고 있다.<sup class="reference" id="cite_ref-4"><a href="#cite_note-4">[4]</a></sup><sup class="reference" id="cite_ref-5"><a href="#cite_note-5">[5]</a></sup> 전근대에는 가계를 신성시하기 위해 고대의 전설적인 제왕 또는 유명한 위인들을 시조로 간주하는 일이 많았으므로 김일제 후손설은 사실로 보기 어렵다는 것이다.<sup class="reference" id="cite_ref-.EB.AC.B8.EA.B2.BD.ED.98.84_1-1"><a href="#cite_note-.EB.AC.B8.EA.B2.BD.ED.98.84-1">[1]</a></sup></p>
<p>대한민국의 일부 역사 다큐멘터리나 재야사학자들은 김일제가 신라 김씨의 선조라는 주장을 하고 있다. 문헌 기록 외에 <a class="new" href="/w/index.php?title=%EB%8F%8C%EB%AC%B4%EC%A7%80%EB%8D%A7%EB%84%90%EB%AC%B4%EB%8D%A4&amp;action=edit&amp;redlink=1" title="돌무지덧널무덤 (없는 문서)">돌무지덧널무덤</a>(적석목곽분) 및 부장품이 일부 북방민족 계통과 유사하다는 사실도 증거로 거론된다.<sup class="reference" id="cite_ref-6"><a href="#cite_note-6">[6]</a></sup> <sup class="reference" id="cite_ref-7"><a href="#cite_note-7">[7]</a></sup> 하지만 2000년대 이후 고고학 발굴 성과의 누적을 통해 돌무지덧널무덤과 부장품의 발전 단계가 모두 밝혀지면서 4세기 신라 묘제의 북방민족 유래설도 역사학계 및 고고학계에서 인정되지 않고 있다.<sup class="reference" id="cite_ref-8"><a href="#cite_note-8">[8]</a></sup></p>
