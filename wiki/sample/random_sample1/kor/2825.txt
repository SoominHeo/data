<h2><span class="mw-headline" id=".EB.84.A5.EC.84.BC_.ED.9E.88.EC.96.B4.EB.A1.9C.EC.A6.88_.EC.8B.9C.EC.A0.88"><a href="/wiki/%EB%84%A5%EC%84%BC_%ED%9E%88%EC%96%B4%EB%A1%9C%EC%A6%88" title="넥센 히어로즈">넥센 히어로즈</a> 시절</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EA%B9%80%EC%A7%80%EC%88%98_(1986%EB%85%84)&amp;action=edit&amp;section=1" title="부분 편집: 넥센 히어로즈 시절">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>입단 후 2009, 2010 시즌 막판 몇 경기에 주로 대타, 대수비, 대주자로 투입되었고 선발로 출장한 경기는 단 한 번이였다. 그렇게 2군 무대만 전전하였다.</p>
<h2><span class="mw-headline" id=".EA.B2.BD.EC.B0.B0_.EC.95.BC.EA.B5.AC.EB.8B.A8_.EC.8B.9C.EC.A0.88"><a href="/wiki/%EA%B2%BD%EC%B0%B0_%EC%95%BC%EA%B5%AC%EB%8B%A8" title="경찰 야구단">경찰 야구단</a> 시절</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EA%B9%80%EC%A7%80%EC%88%98_(1986%EB%85%84)&amp;action=edit&amp;section=2" title="부분 편집: 경찰 야구단 시절">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p><a href="/wiki/2010%EB%85%84" title="2010년">2010년</a> 시즌 후 <a href="/wiki/%EA%B2%BD%EC%B0%B0_%EC%95%BC%EA%B5%AC%EB%8B%A8" title="경찰 야구단">경찰 야구단</a>에 입대하여 군 복무를 마쳤다.</p>
<h2><span class="mw-headline" id=".EB.84.A5.EC.84.BC_.ED.9E.88.EC.96.B4.EB.A1.9C.EC.A6.88_.EB.B3.B5.EA.B7.80"><a href="/wiki/%EB%84%A5%EC%84%BC_%ED%9E%88%EC%96%B4%EB%A1%9C%EC%A6%88" title="넥센 히어로즈">넥센 히어로즈</a> 복귀</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EA%B9%80%EC%A7%80%EC%88%98_(1986%EB%85%84)&amp;action=edit&amp;section=3" title="부분 편집: 넥센 히어로즈 복귀">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>복귀한 후 <a href="/wiki/2013%EB%85%84" title="2013년">2013년</a> 시즌 중 <a href="/wiki/%EA%B9%80%EB%AF%BC%EC%9A%B0_(1979%EB%85%84)" title="김민우 (1979년)">김민우</a>와 <a href="/wiki/%EC%8B%A0%ED%98%84%EC%B2%A0_(%EC%95%BC%EA%B5%AC_%EC%84%A0%EC%88%98)" title="신현철 (야구 선수)">신현철</a>의 음주운전 사건으로 인해 기회를 잡기 시작했다. <a href="/wiki/%EC%84%9C%EA%B1%B4%EC%B0%BD" title="서건창">서건창</a>의 부상과 그 자리를 대신해 <a href="/wiki/%EC%84%9C%EB%8F%99%EC%9A%B1_(%EC%95%BC%EA%B5%AC_%EC%84%A0%EC%88%98)" title="서동욱 (야구 선수)">서동욱</a>과 같이 내야를 번갈아 가며 수비하던 <a href="/wiki/%EC%9C%A0%EC%9E%AC%EC%8B%A0" title="유재신">유재신</a>마저 도루 과정에서 손가락 부상을 당하자 1군에 등록되었다. <a href="/wiki/7%EC%9B%94_3%EC%9D%BC" title="7월 3일">7월 3일</a> NC전 9회에 대주자로 투입되었다. <a href="/wiki/7%EC%9B%94_5%EC%9D%BC" title="7월 5일">7월 5일</a> <a class="mw-redirect" href="/wiki/%EC%9E%A0%EC%8B%A4%EC%95%BC%EA%B5%AC%EC%9E%A5" title="잠실야구장">잠실</a> <a href="/wiki/LG_%ED%8A%B8%EC%9C%88%EC%8A%A4" title="LG 트윈스">LG</a>전에서는 <a class="mw-redirect" href="/wiki/%EC%A7%80%EB%AA%85%ED%83%80%EC%9E%90" title="지명타자">지명타자</a> <a href="/wiki/%EC%9D%B4%EC%84%B1%EC%97%B4_(%EC%95%BC%EA%B5%AC_%EC%84%A0%EC%88%98)" title="이성열 (야구 선수)">이성열</a>이 <a href="/wiki/%ED%8F%AC%EC%88%98" title="포수">포수</a>로 수비 위치에 들어가며 <a class="mw-redirect" href="/wiki/%EC%A7%80%EB%AA%85%ED%83%80%EC%9E%90" title="지명타자">지명타자</a>가 사라져 <a href="/wiki/%ED%88%AC%EC%88%98" title="투수">투수</a>가 타순에 포함되게 되었고 8회 말 <a href="/wiki/%ED%95%9C%ED%98%84%ED%9D%AC" title="한현희">한현희</a>의 대타로 나와 만루 찬스에서 파울 홈런을 치는 등 상대 <a href="/wiki/%ED%88%AC%EC%88%98" title="투수">투수</a> <a href="/wiki/%EB%B4%89%EC%A4%91%EA%B7%BC" title="봉중근">봉중근</a>과 끈질긴 싸움을 했고, 넥센의 3중도루 작전이 나오는데 큰 공헌을 했다<sup class="reference" id="cite_ref-2"><a href="#cite_note-2">[2]</a></sup>. 풀 카운트 승부 끝에 볼넷으로 출루했다. 그 해 1군 37경기에 출장하였다. 시즌을 마치고 포스트 시즌 명단에 합류하여 데뷔 후 처음으로 포스트 시즌에 출전했다. <a href="/wiki/2013%EB%85%84" title="2013년">2013년</a> <a href="/wiki/%EB%AA%A9%EB%8F%99%EC%95%BC%EA%B5%AC%EC%9E%A5" title="목동야구장">목동야구장</a>에서 열린 준 플레이오프 2차전에서 <a href="/wiki/%EA%B9%80%EB%AF%BC%EC%84%B1_(1988%EB%85%84_12%EC%9B%94)" title="김민성 (1988년 12월)">김민성</a>의 3루 대수비로 연장 10회 말 1사 1루 상황에서 <a href="/wiki/%ED%88%AC%EC%88%98" title="투수">투수</a> <a href="/wiki/%EC%98%A4%ED%98%84%ED%83%9D" title="오현택">오현택</a>의 견제 실책으로 1루 주자 <a href="/wiki/%EB%B0%95%EB%B3%91%ED%98%B8_(1986%EB%85%84)" title="박병호 (1986년)">박병호</a>가 3루까지 진루한 후, <a href="/wiki/%EC%A4%91%EA%B2%AC%EC%88%98" title="중견수">중견수</a> 앞으로 가는 끝내기 적시타를 때리면서 이 경기의 MVP로 선정되었다. 경기 후 소감에서 그는 <b>"PS 엔트리에 들어가는 게 목표였다"</b> 라고 밝혔다.<sup class="reference" id="cite_ref-3"><a href="#cite_note-3">[3]</a></sup> <a href="/wiki/2014" title="2014">2014</a> 시즌 28경기에 출전해 타율 0.174, 4안타, 3타점을 기록하였다. <a href="/wiki/2015" title="2015">2015</a> 시즌은 개인적으로 역대 가장 많은 경기에 출전한 시즌이었다. 선발과 백업으로 번갈아 출전하며 104경기에 출전해 타율 0,235, 31안타, 13타점을 기록하였다. 신체가 마른 편이 아닌데도 홈런이 단 한 개도 없었다가, 2016년 7월 28일 <a href="/wiki/%EB%91%90%EC%82%B0_%EB%B2%A0%EC%96%B4%EC%8A%A4" title="두산 베어스">두산 베어스</a>와의 경기에서 드디어 데뷔 첫 홈런을 기록하였다.</p>
