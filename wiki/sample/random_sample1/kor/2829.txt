<p><b>김질</b>(金礩, <a href="/wiki/1422%EB%85%84" title="1422년">1422년</a> ~ <a href="/wiki/1478%EB%85%84" title="1478년">1478년</a> <a href="/wiki/2%EC%9B%94_24%EC%9D%BC" title="2월 24일">2월 24일</a>)은 <a href="/wiki/%EC%A1%B0%EC%84%A0" title="조선">조선</a>시대 전기의 무신, 문신, 정치인이다. <a href="/wiki/%EC%82%AC%EC%9C%A1%EC%8B%A0" title="사육신">사육신</a>과 함께 <a class="mw-redirect" href="/wiki/%EB%8B%A8%EC%A2%85_%EB%B3%B5%EC%9C%84_%EC%9A%B4%EB%8F%99" title="단종 복위 운동">단종 복위 운동</a>에 참여했으나 실패를 예상하고 이를 고변하였다. 자(字)는 가안(可安), 호는 쌍곡(雙谷), 시호는 <b>문정</b>(文靖). 본관은 <a href="/wiki/%EA%B5%AC_%EC%95%88%EB%8F%99_%EA%B9%80%EC%94%A8" title="구 안동 김씨">(구) 안동</a>이다.</p>
<p><a href="/wiki/%EC%9D%8C%EC%84%9C" title="음서">음서</a>로 무관이 되고, 부사직(副司直)에 이르렀을 때 사가독서(賜暇讀書)를 청원하여, 특별히 허락받았다. 그 뒤 <a href="/wiki/1447%EB%85%84" title="1447년">1447년</a> <a class="new" href="/w/index.php?title=%EC%83%9D%EC%9B%90%EC%8B%9C&amp;action=edit&amp;redlink=1" title="생원시 (없는 문서)">생원시</a>에 2등으로 합격했으며 <a href="/wiki/1450%EB%85%84" title="1450년">1450년</a>(문종 즉위년) 문과에 급제하여 집현전 수찬, 사간원 우정언, 성균관 사예 등을 거쳐 <a href="/wiki/1455%EB%85%84" title="1455년">1455년</a> <a href="/wiki/%EC%84%B1%EC%82%BC%EB%AC%B8" title="성삼문">성삼문</a>(成三問)·<a href="/wiki/%EB%B0%95%ED%8C%BD%EB%85%84" title="박팽년">박팽년</a>(朴彭年) 등과 함께 <a class="mw-redirect" href="/wiki/%EB%8B%A8%EC%A2%85_%EB%B3%B5%EC%9C%84_%EC%9A%B4%EB%8F%99" title="단종 복위 운동">단종 복위 운동</a>에 가담했다가 여러 번 기회를 놓치고 위험을 느끼게 되자, 뜻을 바꾸어 <a href="/wiki/%EC%A1%B0%EC%84%A0_%EC%84%B8%EC%A1%B0" title="조선 세조">세조</a>에게 사육신의 거사를 알렸으며 이 공로로 1456년(세조 2) 6월 25일 판군기감사(判軍器監事)가 되고,<sup class="reference" id="cite_ref-1"><a href="#cite_note-1">[1]</a></sup> 1455년 9월 5일 <a href="/wiki/%EC%A2%8C%EC%9D%B5%EA%B3%B5%EC%8B%A0" title="좌익공신">좌익공신</a>(佐翼功臣) 3등과<sup class="reference" id="cite_ref-2"><a href="#cite_note-2">[2]</a></sup> 1459년(세조 5) 8월 25일 상낙군(上洛君)에 봉해졌고<sup class="reference" id="cite_ref-3"><a href="#cite_note-3">[3]</a></sup> 뒤에 부원군으로 진봉되었다.</p>
<p>그 뒤 <a href="/wiki/%ED%8F%89%EC%95%88%EB%8F%84" title="평안도">평안도</a><a href="/wiki/%EA%B4%80%EC%B0%B0%EC%82%AC" title="관찰사">관찰사</a>, <a class="mw-redirect" href="/wiki/%EA%B3%B5%EC%A1%B0%ED%8C%90%EC%84%9C" title="공조판서">공조판서</a>, <a class="mw-redirect" href="/wiki/%EB%B3%91%EC%A1%B0%ED%8C%90%EC%84%9C" title="병조판서">병조판서</a>, <a href="/wiki/%EC%9A%B0%EC%B0%B8%EC%B0%AC" title="우참찬">우참찬</a>, <a href="/wiki/%EA%B2%BD%EC%83%81%EB%8F%84" title="경상도">경상도</a><a href="/wiki/%EA%B4%80%EC%B0%B0%EC%82%AC" title="관찰사">관찰사</a> 등을 지내고, <a href="/wiki/1468%EB%85%84" title="1468년">1468년</a> <a href="/wiki/%EC%9A%B0%EC%9D%98%EC%A0%95" title="우의정">우의정</a>, <a href="/wiki/%EC%A2%8C%EC%9D%98%EC%A0%95" title="좌의정">좌의정</a>이 되었다. <a href="/wiki/1468%EB%85%84" title="1468년">1468년</a> <a class="mw-disambig" href="/wiki/%EC%84%B8%EC%A1%B0" title="세조">세조</a>의 사망과 <a href="/wiki/1469%EB%85%84" title="1469년">1469년</a> <a class="mw-disambig" href="/wiki/%EC%98%88%EC%A2%85" title="예종">예종</a>의 사망 당시에는 <a class="mw-disambig" href="/wiki/%EC%9B%90%EC%83%81" title="원상">원상</a>의 한사람으로 정무를 처결하였다. 그 뒤 <a href="/wiki/%EC%A1%B0%EC%84%A0_%EC%84%B1%EC%A2%85" title="조선 성종">자을산군 혈</a>을 지지하고, <a href="/wiki/%EC%A1%B0%EC%84%A0_%EC%84%B1%EC%A2%85" title="조선 성종">성종</a> 즉위 직후에는 왕족 <a href="/wiki/%EA%B7%80%EC%84%B1%EA%B5%B0" title="귀성군">귀성군</a>을 숙청하였다. <a href="/wiki/1471%EB%85%84" title="1471년">1471년</a>(성종 2) 좌리공신(佐理功臣) 2등으로 상락부원군(上洛府院君)에 봉군되었다.</p>
<p>관직은 <a href="/wiki/%EC%9D%98%EC%A0%95%EB%B6%80" title="의정부">의정부</a><a href="/wiki/%EC%A2%8C%EC%9D%98%EC%A0%95" title="좌의정">좌의정</a>에 이르렀고, <a href="/wiki/%EC%A1%B0%EC%84%A0_%EC%84%B1%EC%A2%85" title="조선 성종">성종</a> 초에 〈경국대전〉의 편참에 참여하였다. 고려의 장군 충렬공 <a href="/wiki/%EA%B9%80%EB%B0%A9%EA%B2%BD" title="김방경">김방경</a>의 후손으로, <a class="new" href="/w/index.php?title=%EA%B0%9C%EA%B5%AD%EA%B3%B5%EC%8B%A0&amp;action=edit&amp;redlink=1" title="개국공신 (없는 문서)">개국공신</a> 익원공 <a href="/wiki/%EA%B9%80%EC%82%AC%ED%98%95" title="김사형">김사형</a>의 증손자이자 <a href="/wiki/%EC%9D%98%EC%A0%95%EB%B6%80" title="의정부">의정부</a><a href="/wiki/%EC%98%81%EC%9D%98%EC%A0%95" title="영의정">영의정</a>을 지낸 <a href="/wiki/%EC%A0%95%EC%B0%BD%EC%86%90" title="정창손">정창손</a>의 사위이다. 또한 <a href="/wiki/%EA%B0%95%ED%9D%AC%EB%A7%B9" title="강희맹">강희맹</a>의 사돈으로 <a href="/wiki/%EA%B0%95%ED%9D%AC%EB%A7%B9" title="강희맹">강희맹</a>의 딸은 그의 아들 <a href="/wiki/%EA%B9%80%EC%84%B1%EB%8F%99_(1452%EB%85%84)" title="김성동 (1452년)">김성동</a>과 결혼했다. <a href="/wiki/%EC%A1%B0%EC%84%A0_%EC%84%B8%EC%A1%B0" title="조선 세조">세조</a> 사후 <a href="/wiki/%EC%98%81%EC%9D%98%EC%A0%95" title="영의정">영의정</a> <a href="/wiki/%ED%95%9C%EB%AA%85%ED%9A%8C" title="한명회">한명회</a>, <a href="/wiki/%EC%8B%A0%EC%88%99%EC%A3%BC" title="신숙주">신숙주</a> 등과 함께 원상(院相)세력을 형성하였으며, <a class="new" href="/w/index.php?title=%EC%9B%90%EC%83%81%EC%A0%9C&amp;action=edit&amp;redlink=1" title="원상제 (없는 문서)">원상제</a>의 전범이 되었다. 낙풍부원군 <a href="/wiki/%EA%B9%80%EC%9E%90%EC%A0%90" title="김자점">김자점</a>의 5대 선조이며 백범 <a href="/wiki/%EA%B9%80%EA%B5%AC" title="김구">김구</a>의 20대 방조에 해당된다.</p>

<h2><span class="mw-headline" id=".EC.83.9D.EC.95.A0">생애</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EA%B9%80%EC%A7%88_(1422%EB%85%84)&amp;action=edit&amp;section=1" title="부분 편집: 생애">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<h3><span class="mw-headline" id=".EC.83.9D.EC.95.A0_.EC.B4.88.EA.B8.B0">생애 초기</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EA%B9%80%EC%A7%88_(1422%EB%85%84)&amp;action=edit&amp;section=2" title="부분 편집: 생애 초기">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<h4><span class="mw-headline" id=".EC.B6.9C.EC.83.9D.EA.B3.BC_.EC.86.8C.EB.85.84.EA.B8.B0">출생과 소년기</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EA%B9%80%EC%A7%88_(1422%EB%85%84)&amp;action=edit&amp;section=3" title="부분 편집: 출생과 소년기">편집</a><span class="mw-editsection-bracket">]</span></span></h4>
<p>쌍곡 김질은 <a href="/wiki/1422%EB%85%84" title="1422년">1422년</a> 조선 개국 공신 <a href="/wiki/%EA%B9%80%EC%82%AC%ED%98%95" title="김사형">김사형</a>(金士衡)의 증손이며 <a class="new" href="/w/index.php?title=%EB%8F%99%EC%A7%80%EC%A4%91%EC%B6%94%EB%B6%80%EC%82%AC&amp;action=edit&amp;redlink=1" title="동지중추부사 (없는 문서)">동지중추부사</a> <a class="mw-redirect" href="/wiki/%EC%A6%9D%EC%A7%81" title="증직">증</a> <a href="/wiki/%EC%9D%98%EC%A0%95%EB%B6%80" title="의정부">의정부</a><a href="/wiki/%EC%98%81%EC%9D%98%EC%A0%95" title="영의정">영의정</a> <a class="new" href="/w/index.php?title=%EA%B9%80%EC%A2%85%EC%88%99&amp;action=edit&amp;redlink=1" title="김종숙 (없는 문서)">김종숙</a>의 아들로 태어났다.</p>
<p>음서제로 관직에 입문하여 <a class="new" href="/w/index.php?title=%EC%B6%A9%EC%9D%98%EC%9C%84&amp;action=edit&amp;redlink=1" title="충의위 (없는 문서)">충의위</a>(忠義衛) 부사직 등을 지냈다. 그 뒤 부사직으로 재직 중 <a href="/wiki/1447%EB%85%84" title="1447년">1447년</a>(세종 29년) <a class="new" href="/w/index.php?title=%EC%8B%9D%EB%85%84%EA%B3%BC&amp;action=edit&amp;redlink=1" title="식년과 (없는 문서)">식년과</a>의 생원시에 2등으로 합격하여 <a href="/wiki/%EC%83%9D%EC%9B%90" title="생원">생원</a>(生員)이 되었다. <a href="/wiki/1450%EB%85%84" title="1450년">1450년</a> 부사직으로 재직 중 휴가를 얻어 <a href="/wiki/%EC%84%B1%EA%B7%A0%EA%B4%80" title="성균관">성균관</a>에 입학, 성균관에서 수학하였다.</p>
<p>그가 성균관에 수학하려 하자 전례가 없어서 들어갈수 없었다. 그러나 <a href="/wiki/%EC%9D%98%EC%A0%95%EB%B6%80" title="의정부">의정부</a>의 특별 건의로 <a href="/wiki/%EC%84%B1%EA%B7%A0%EA%B4%80" title="성균관">성균관</a>에 입학할 수 있게 되었다.<sup class="reference" id="cite_ref-4"><a href="#cite_note-4">[4]</a></sup> 그의 성균관 수학으로부터 참상관의 성균관 수학제도가 비롯되었다는 의견도 있다.<sup class="reference" id="cite_ref-5"><a href="#cite_note-5">[5]</a></sup></p>
<p>뒤에 <a href="/wiki/%EC%9D%98%EC%A0%95%EB%B6%80" title="의정부">의정부</a><a href="/wiki/%EC%98%81%EC%9D%98%EC%A0%95" title="영의정">영의정</a>을 지낸 <a href="/wiki/%EC%A0%95%EC%B0%BD%EC%86%90" title="정창손">정창손</a>의 딸 정씨와 결혼하였다.</p>
<h4><span class="mw-headline" id=".EC.B2.AD.EB.85.84_.EC.8B.9C.EC.A0.88">청년 시절</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EA%B9%80%EC%A7%88_(1422%EB%85%84)&amp;action=edit&amp;section=4" title="부분 편집: 청년 시절">편집</a><span class="mw-editsection-bracket">]</span></span></h4>
<p>음보로 출사하여 <a class="new" href="/w/index.php?title=%EC%B6%A9%EC%9D%98%EC%9C%84&amp;action=edit&amp;redlink=1" title="충의위 (없는 문서)">충의위</a>(忠義衛)가 되었으며 그 뒤 여러 벼슬을 거쳐 <a class="new" href="/w/index.php?title=%EC%B6%A9%EC%9D%98%EC%9C%84&amp;action=edit&amp;redlink=1" title="충의위 (없는 문서)">충의위</a> 부사직(副司直)에 이르렀을 때 사가독서(賜暇讀書)를 청원했다가, 특별히 허락받았다. 그는 과거 급제자가 아닌데도 참상관의 자격으로 사가독서를 청하여 <a class="mw-redirect" href="/wiki/%EC%84%B8%EC%A2%85%EB%8C%80%EC%99%95" title="세종대왕">세종대왕</a>이 이를 허락했다. 그가 과거 급제자가 아닌데도 사가독서를 한 일은 후일 성균관에서 참상관의 수학제도의 전례가 되었다. 그 뒤 <a href="/wiki/%EC%82%AC%EC%A7%81" title="사직">사직</a>(司直)으로 승진했다.</p>
<p><a href="/wiki/1450%EB%85%84" title="1450년">1450년</a>(문종 즉위년) <a href="/wiki/%EC%82%AC%EC%A7%81" title="사직">사직</a>(司直) 재직 중, 추장<a class="mw-redirect" href="/wiki/%EB%AC%B8%EA%B3%BC" title="문과">문과</a>에 응시하여 정과로 급제, 바로 <a href="/wiki/%EC%84%B1%EA%B7%A0%EA%B4%80" title="성균관">성균관</a>주부가 되었다. 그 뒤 집현전 학사가 되었으며, 사간원우정언, 병조좌랑을 지내며 <a href="/wiki/%EC%A7%91%ED%98%84%EC%A0%84" title="집현전">집현전</a> 학사를 겸임하였다. 김질은 글씨를 잘 썼고 고전에 능하였으며 지식이 해박하였다. 그 무렵 <a href="/wiki/%EC%84%B1%EC%82%BC%EB%AC%B8" title="성삼문">성삼문</a>(成三問), <a href="/wiki/%EB%B0%95%ED%8C%BD%EB%85%84" title="박팽년">박팽년</a>, 최항(崔恒), <a href="/wiki/%EC%8B%A0%EC%88%99%EC%A3%BC" title="신숙주">신숙주</a>(申叔舟) 등 몇몇 <a href="/wiki/%EC%A7%91%ED%98%84%EC%A0%84" title="집현전">집현전</a> 학사들과 함께 <a class="mw-disambig" href="/wiki/%EB%AC%B8%EC%A2%85" title="문종">문종</a>의 각별한 사랑과 총애를 받았다.</p>
<p><a href="/wiki/%EC%A1%B0%EC%84%A0_%EB%AC%B8%EC%A2%85" title="조선 문종">문종</a>은 병세가 깊어지자 부왕 <a href="/wiki/%EC%A1%B0%EC%84%A0_%EC%84%B8%EC%A2%85" title="조선 세종">세종대왕</a>이 그랬듯이 <a href="/wiki/%EC%A7%91%ED%98%84%EC%A0%84" title="집현전">집현전</a> 학사들을 불러서 어린 세자를 잘 부탁한다는 부탁을 거듭 하였다. 그러나 뒤에 그는 <a class="mw-redirect" href="/wiki/%EB%8B%A8%EC%A2%85_%EB%B3%B5%EC%9C%84_%EC%9A%B4%EB%8F%99" title="단종 복위 운동">단종 복위 운동</a>에 가담했다가 직전에 등을 돌리게 된다.</p>
<h3><span class="mw-headline" id=".EA.B4.80.EB.A3.8C_.EC.83.9D.ED.99.9C.EA.B3.BC_.EC.A0.95.EC.B9.98_.ED.99.9C.EB.8F.99">관료 생활과 정치 활동</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EA%B9%80%EC%A7%88_(1422%EB%85%84)&amp;action=edit&amp;section=5" title="부분 편집: 관료 생활과 정치 활동">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<h4><span class="mw-headline" id=".EA.B3.BC.EA.B1.B0_.EA.B8.89.EC.A0.9C.EC.99.80_.EA.B4.80.EB.A3.8C.EC.83.9D.ED.99.9C_.EC.B4.88.EB.B0.98">과거 급제와 관료생활 초반</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EA%B9%80%EC%A7%88_(1422%EB%85%84)&amp;action=edit&amp;section=6" title="부분 편집: 과거 급제와 관료생활 초반">편집</a><span class="mw-editsection-bracket">]</span></span></h4>
<p>그해 <a href="/wiki/%EC%A7%91%ED%98%84%EC%A0%84" title="집현전">집현전</a> <a class="new" href="/w/index.php?title=%EC%88%98%EC%B0%AC&amp;action=edit&amp;redlink=1" title="수찬 (없는 문서)">수찬</a>에 뽑혔고, 병조 좌랑(兵曹佐郞)이 되었다가 <a href="/wiki/1452%EB%85%84" title="1452년">1452년</a>(문종 2) <a href="/wiki/2%EC%9B%94" title="2월">2월</a> 정기인사 때 승의랑(承議郞) <a href="/wiki/%EC%82%AC%EA%B0%84%EC%9B%90" title="사간원">사간원</a> <a class="new" href="/w/index.php?title=%EC%9A%B0%EC%A0%95%EC%96%B8&amp;action=edit&amp;redlink=1" title="우정언 (없는 문서)">우정언</a>(司諫院右正言) 등을 지냈다. 그 무렵 그는 <a href="/wiki/%EC%84%B1%EC%82%BC%EB%AC%B8" title="성삼문">성삼문</a>, <a class="mw-disambig" href="/wiki/%EC%B5%9C%ED%95%AD" title="최항">최항</a>, <a href="/wiki/%EC%8B%A0%EC%88%99%EC%A3%BC" title="신숙주">신숙주</a> 등과 함께 문종의 각별한 지우와 총애를 받았다.</p>
<p>그해 <a href="/wiki/2%EC%9B%94" title="2월">2월</a> 헌납(獻納) 조원희(趙元禧)가 관직 제수의 형평성에 대한 의문을 제기하면서 승의랑(承議郞)인데도 우정언에 임명된 것을 문제삼기도 했다.<sup class="reference" id="cite_ref-6"><a href="#cite_note-6">[6]</a></sup> 그러나 왕이 무마시켰다. 그러나 <a href="/wiki/%EC%A1%B0%EC%9B%90%ED%9D%AC" title="조원희">조원희</a>의 계속된 탄핵으로 체직되었다.<sup class="reference" id="cite_ref-7"><a href="#cite_note-7">[7]</a></sup></p>
<p><a href="/wiki/1452%EB%85%84" title="1452년">1452년</a> 다시 병조 좌랑(兵曹佐郞)이 되었다. 그해 <a href="/wiki/8%EC%9B%94" title="8월">8월</a> 비가 내린 뒤 왕명으로 진무(鎭撫) 남육(南陸)과 함께 대행대왕(문종)의 현궁을 공사하는 역부들을 구료하게 했다.<sup class="reference" id="cite_ref-8"><a href="#cite_note-8">[8]</a></sup></p>
<h4><span class="mw-headline" id=".EA.B3.84.EC.9C.A0.EC.A0.95.EB.82.9C.EA.B3.BC_.EC.84.B8.EC.A1.B0_.EB.B0.98.EC.A0.95">계유정난과 세조 반정</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EA%B9%80%EC%A7%88_(1422%EB%85%84)&amp;action=edit&amp;section=7" title="부분 편집: 계유정난과 세조 반정">편집</a><span class="mw-editsection-bracket">]</span></span></h4>
<div class="rellink boilerplate seealso"><img alt="" data-file-height="128" data-file-width="128" height="16" src="//upload.wikimedia.org/wikipedia/commons/thumb/e/ec/Crystal_Clear_app_xmag.svg/16px-Crystal_Clear_app_xmag.svg.png" srcset="//upload.wikimedia.org/wikipedia/commons/thumb/e/ec/Crystal_Clear_app_xmag.svg/24px-Crystal_Clear_app_xmag.svg.png 1.5x, //upload.wikimedia.org/wikipedia/commons/thumb/e/ec/Crystal_Clear_app_xmag.svg/32px-Crystal_Clear_app_xmag.svg.png 2x" width="16"/> <a href="/wiki/%EA%B3%84%EC%9C%A0%EC%A0%95%EB%82%9C" title="계유정난">계유정난</a>, <a class="mw-redirect" href="/wiki/%EC%84%B8%EC%A1%B0_%EC%B0%AC%EC%9C%84" title="세조 찬위">세조 찬위</a> 문서를 참고하십시오.</div>
<p><a href="/wiki/1453%EB%85%84" title="1453년">1453년</a>(단종 2년) <a href="/wiki/%EC%9D%B4%EC%A1%B0" title="이조">이조</a>낭관을 거쳐 <a href="/wiki/1454%EB%85%84" title="1454년">1454년</a> <a href="/wiki/8%EC%9B%94" title="8월">8월</a> <a href="/wiki/%EC%A1%B0%EC%84%A0_%EC%84%B8%EC%A1%B0" title="조선 세조">수양대군</a>의 추천<sup class="reference" id="cite_ref-dan122_9-0"><a href="#cite_note-dan122-9">[9]</a></sup>으로 6품에서 4품으로 특별 승진하였다.<sup class="reference" id="cite_ref-10"><a href="#cite_note-10">[10]</a></sup> 뒤에 <a href="/wiki/%EC%A1%B0%EC%84%A0_%EB%8B%A8%EC%A2%85" title="조선 단종">단종</a>이 그를 6품에서 4품으로 특별 승진시킨 것을 <a class="mw-redirect" href="/wiki/%EC%88%98%EC%96%91%EB%8C%80%EA%B5%B0" title="수양대군">수양대군</a>에게 문제삼자 <a class="mw-redirect" href="/wiki/%EC%88%98%EC%96%91%EB%8C%80%EA%B5%B0" title="수양대군">수양대군</a>은 이를 극력 변호하였다.<sup class="reference" id="cite_ref-dan122_9-1"><a href="#cite_note-dan122-9">[9]</a></sup></p>

<p>이후 그는 <a href="/wiki/%EC%A1%B0%EC%84%A0_%EC%84%B8%EC%A1%B0" title="조선 세조">수양대군</a>의 심복이 되어 활약하였다. <a href="/wiki/1455%EB%85%84" title="1455년">1455년</a>(세조 1년) <a href="/wiki/%EC%84%B1%EA%B7%A0%EA%B4%80" title="성균관">성균관</a> 사예(成均館司藝)가 되었다.</p>
<p>사정전(思政殿)의 정사를 마친 뒤 술좌석에 참석하였다. 이때 집의(執義) 이예(李芮)가 술이 취하여 어탑(御榻)으로 올라가 <a href="/wiki/%EB%8B%B9%EB%82%98%EB%9D%BC" title="당나라">당나라</a> <a href="/wiki/%EB%8B%B9_%ED%83%9C%EC%A2%85" title="당 태종">태종</a>(唐太宗)의 고사(故事)를 끌어서 <a href="/wiki/%EA%B8%88%EC%84%B1%EB%8C%80%EA%B5%B0" title="금성대군">금성대군</a> 이유(李瑜) 등의 죄를 강력히 청하자 이를 불쾌하게 여긴 <a href="/wiki/%EC%A1%B0%EC%84%A0_%EC%84%B8%EC%A1%B0" title="조선 세조">세조</a>는 김질과 한계미(韓繼美), 박수미(朴壽彌), 김석제(金石梯)에게 술을 남용했다는 이유로 파면하였다.<sup class="reference" id="cite_ref-11"><a href="#cite_note-11">[11]</a></sup></p>
<p>그해 <a href="/wiki/12%EC%9B%94" title="12월">12월</a> <a href="/wiki/%EC%A2%8C%EC%9D%B5%EA%B3%B5%EC%8B%A0" title="좌익공신">좌익</a><a href="/wiki/%EC%9B%90%EC%A2%85%EA%B3%B5%EC%8B%A0" title="원종공신">원종공신</a>(佐翼原從功臣) 2등에 책록되었다.<sup class="reference" id="cite_ref-12"><a href="#cite_note-12">[12]</a></sup></p>
<h4><span class="mw-headline" id=".EC.82.AC.EC.9C.A1.EC.8B.A0.EC.9D.98_.EB.82.9C_.EA.B3.A0.EB.B3.80">사육신의 난 고변</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EA%B9%80%EC%A7%88_(1422%EB%85%84)&amp;action=edit&amp;section=8" title="부분 편집: 사육신의 난 고변">편집</a><span class="mw-editsection-bracket">]</span></span></h4>
<div class="rellink boilerplate seealso"><img alt="" data-file-height="128" data-file-width="128" height="16" src="//upload.wikimedia.org/wikipedia/commons/thumb/e/ec/Crystal_Clear_app_xmag.svg/16px-Crystal_Clear_app_xmag.svg.png" srcset="//upload.wikimedia.org/wikipedia/commons/thumb/e/ec/Crystal_Clear_app_xmag.svg/24px-Crystal_Clear_app_xmag.svg.png 1.5x, //upload.wikimedia.org/wikipedia/commons/thumb/e/ec/Crystal_Clear_app_xmag.svg/32px-Crystal_Clear_app_xmag.svg.png 2x" width="16"/> <a class="new" href="/w/index.php?title=%EC%82%AC%EC%9C%A1%EC%8B%A0%EC%9D%98_%EB%82%9C&amp;action=edit&amp;redlink=1" title="사육신의 난 (없는 문서)">사육신의 난</a>, <a class="mw-redirect" href="/wiki/%EB%8B%A8%EC%A2%85_%EB%B3%B5%EC%9C%84_%EC%9A%B4%EB%8F%99" title="단종 복위 운동">단종 복위 운동</a> 문서를 참고하십시오.</div>
<p><a href="/wiki/%EC%A1%B0%EC%84%A0_%EB%8B%A8%EC%A2%85" title="조선 단종">단종</a> 폐위 이후 <a href="/wiki/1456%EB%85%84" title="1456년">1456년</a> <a href="/wiki/%EC%84%B1%EC%82%BC%EB%AC%B8" title="성삼문">성삼문</a>·<a href="/wiki/%EB%B0%95%ED%8C%BD%EB%85%84" title="박팽년">박팽년</a>·<a href="/wiki/%ED%95%98%EC%9C%84%EC%A7%80" title="하위지">하위지</a>·<a href="/wiki/%EC%9C%A0%EC%84%B1%EC%9B%90" title="유성원">유성원</a> 등과 함께 단종 복위운동에 가담하였으나 마음을 바꾸어, 이를 <a href="/wiki/%EC%A1%B0%EC%84%A0_%EC%84%B8%EC%A1%B0" title="조선 세조">세조</a>에게 알렸다. <a href="/wiki/1456%EB%85%84" title="1456년">1456년</a> <a href="/wiki/%EC%84%B1%EC%82%BC%EB%AC%B8" title="성삼문">성삼문</a> 등의 단종복위운동에 김질이 가담하자, 장인 <a href="/wiki/%EC%A0%95%EC%B0%BD%EC%86%90" title="정창손">정창손</a>은 이를 설득하였다. 김질은 거사가 성공하지 못할 줄로 알고 장인인 정창손에게 달려가 수시로 상의하였다. 그해 <a href="/wiki/6%EC%9B%94" title="6월">6월</a> <a class="new" href="/w/index.php?title=%EC%9A%B4%EA%B2%80&amp;action=edit&amp;redlink=1" title="운검 (없는 문서)">운검</a> 설치가 취소되면서 일이 여의치 않자 불안감을 느꼈던 김질은 전향, 이 사실을 <a href="/wiki/%EC%A0%95%EC%B0%BD%EC%86%90" title="정창손">정창손</a>에게 알렸다.</p>

<p>김질은 장인 <a href="/wiki/%EC%A0%95%EC%B0%BD%EC%86%90" title="정창손">정창손</a>에게 일부 집현전 학자들의 단종 복위 계획을 알리고고, <a href="/wiki/%EC%A0%95%EC%B0%BD%EC%86%90" title="정창손">정창손</a>은 <a href="/wiki/%EC%84%B1%EC%82%BC%EB%AC%B8" title="성삼문">성삼문</a>(成三問), <a href="/wiki/%EB%B0%95%ED%8C%BD%EB%85%84" title="박팽년">박팽년</a>(朴彭年), <a href="/wiki/%EC%9D%B4%EA%B0%9C" title="이개">이개</a>(李塏), <a href="/wiki/%ED%95%98%EC%9C%84%EC%A7%80" title="하위지">하위지</a>(河緯地), <a href="/wiki/%EC%9C%A0%EC%84%B1%EC%9B%90" title="유성원">유성원</a>(柳誠源), <a href="/wiki/%EC%9C%A0%EC%9D%91%EB%B6%80" title="유응부">유응부</a>(兪應孚), <a href="/wiki/%EC%A1%B0%EC%84%A0_%EB%8B%A8%EC%A2%85" title="조선 단종">단종</a>의 외삼촌인 <a class="new" href="/w/index.php?title=%EA%B6%8C%EC%9E%90%EC%8B%A0&amp;action=edit&amp;redlink=1" title="권자신 (없는 문서)">권자신</a>(權自愼), <a href="/wiki/%EC%84%B1%EC%8A%B9" title="성승">성승</a> 등이 가담했음을 확인하였다. 정창손은 이를 바로 <a href="/wiki/%EC%A1%B0%EC%84%A0_%EC%84%B8%EC%A1%B0" title="조선 세조">세조</a>에게 알렸다.</p>
<p>그해 <a href="/wiki/6%EC%9B%94_2%EC%9D%BC" title="6월 2일">6월 2일</a> 김질은 <a href="/wiki/%EC%9D%98%EC%A0%95%EB%B6%80" title="의정부">의정부</a><a href="/wiki/%EC%9A%B0%EC%B0%AC%EC%84%B1" title="우찬성">우찬성</a>인 장인 <a href="/wiki/%EC%A0%95%EC%B0%BD%EC%86%90" title="정창손">정창손</a>과 함께 궐로 찾아가 사육신의 거사를 폭로하였다.<sup class="reference" id="cite_ref-13"><a href="#cite_note-13">[13]</a></sup> 정창손은 김질과 같이 대궐로 달려가 고변하며 '신은 실로 모르고 김질만 혼자 참여하였는데, 김질의 죄는 만번 죽어 마땅합니다.'라고 하니, 세조가 특별히 김질을 사면하였다.</p>
<p>그러나 그가 중간에 <a href="/wiki/%EC%82%AC%EC%9C%A1%EC%8B%A0" title="사육신">사육신</a>의 고변을 밀고한 점을 들어 여러 신하가, 김질이 함께 모반하였는데 성패를 관망하다가 일이 이루어지지 못하게 되어서야 고한 것이라 하여 주살하기를 청하였으나, 세조는 듣지 않고 그를 신뢰하였다.<sup class="reference" id="cite_ref-seong09_14-0"><a href="#cite_note-seong09-14">[14]</a></sup></p>
<h4><span class="mw-headline" id=".EC.82.AC.EC.9C.A1.EC.8B.A0.EC.9D.98_.EB.82.9C_.EC.A7.84.EC.95.95_.EC.A7.81.ED.9B.84">사육신의 난 진압 직후</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EA%B9%80%EC%A7%88_(1422%EB%85%84)&amp;action=edit&amp;section=9" title="부분 편집: 사육신의 난 진압 직후">편집</a><span class="mw-editsection-bracket">]</span></span></h4>
<p>사육신의 거사를 고변한 공으로 판군기감사(判軍器監事)로 승진했고 <a href="/wiki/%EC%A1%B0%EC%84%A0_%EC%84%B8%EC%A1%B0" title="조선 세조">세조</a>의 각별한 신임을 받아 원종공신에서 정공신인 <a href="/wiki/%EC%A2%8C%EC%9D%B5%EA%B3%B5%EC%8B%A0" title="좌익공신">좌익공신</a>(佐翼功臣) 3등으로 승급되었으며<sup class="reference" id="cite_ref-15"><a href="#cite_note-15">[15]</a></sup> 특별히 말 한필을 선물로 받았으며<sup class="reference" id="cite_ref-16"><a href="#cite_note-16">[16]</a></sup> 상락군(上洛君)에 봉작되었다.</p>
<p><a href="/wiki/9%EC%9B%94" title="9월">9월</a> <a href="/wiki/%EC%82%AC%EC%9C%A1%EC%8B%A0" title="사육신">사육신</a>가문의 부녀자를 하사할 때, 민보흥(閔甫興)의 아내 석비(石非), 이윤원(李潤源)의 아내 대비(大非)를 하사받았다.<sup class="reference" id="cite_ref-17"><a href="#cite_note-17">[17]</a></sup></p>
<p><a href="/wiki/1456%EB%85%84" title="1456년">1456년</a> <a href="/wiki/10%EC%9B%94" title="10월">10월</a> <a href="/wiki/%EC%8A%B9%EC%A0%95%EC%9B%90" title="승정원">승정원</a> 동부승지가 되고 <a href="/wiki/11%EC%9B%94" title="11월">11월</a> 함길도 관찰사(咸吉道觀察使) 이인손(李仁孫)·병조 판서(兵曹判書) 홍달손(洪達孫)·좌승지(文承旨) 조석문(曹錫文)·좌부승지(左副承旨) 한계미(韓繼美)·우부승지(右副承旨) 권지(權摯)·판통례문사(判通禮門事) 이극배(李克培)를 명소(命召)하여 속로 첩목아(速魯帖木兒)와 마흥귀(馬興貴) 등이 서로 싸워서 소란을 일으킨 일에 대한 대책을 논의하였다. <a href="/wiki/12%EC%9B%94" title="12월">12월</a> 다시 <a href="/wiki/%EC%8A%B9%EC%A0%95%EC%9B%90" title="승정원">승정원</a><a class="new" href="/w/index.php?title=%EB%8F%99%EB%B6%80%EC%8A%B9%EC%A7%80&amp;action=edit&amp;redlink=1" title="동부승지 (없는 문서)">동부승지</a>가 되고, <a href="/wiki/12%EC%9B%94_14%EC%9D%BC" title="12월 14일">12월 14일</a> 공신 대표로 <a href="/wiki/%EB%AA%85%EB%82%98%EB%9D%BC" title="명나라">명나라</a>에 파견되었다.</p>
<p><a href="/wiki/1457%EB%85%84" title="1457년">1457년</a> <a href="/wiki/1%EC%9B%94" title="1월">1월</a> <a class="mw-redirect" href="/wiki/%EC%84%B8%EC%9E%90" title="세자">세자</a>가 환구에 나가 의식을 익히는데 대동하였고, <a href="/wiki/1%EC%9B%94" title="1월">1월</a>말 임금이 성균관의 유생들을 시험할 때 대독관(對讀官)의 한사람으로 임명되어 시험을 주관하였다.</p>
<h4><span class="mw-headline" id=".EC.96.91.EB.B0.98.EC.97.90.EA.B2.8C_.EA.B5.B0.EC.97.AD.EC.9D.84_.EB.B6.80.EA.B3.BC">양반에게 군역을 부과</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EA%B9%80%EC%A7%88_(1422%EB%85%84)&amp;action=edit&amp;section=10" title="부분 편집: 양반에게 군역을 부과">편집</a><span class="mw-editsection-bracket">]</span></span></h4>
<p>이후 그는 <a href="/wiki/%EC%A1%B0%EC%84%A0_%EC%84%B8%EC%A1%B0" title="조선 세조">세조</a>가 주관한 활쏘기에 여러번 활을 잘 쏘아 상을 받기도 했다. 그해 세자의 병을 간호한 공로로 1자급 특진하였다. <a href="/wiki/%EC%8A%B9%EC%A0%95%EC%9B%90" title="승정원">승정원</a>우부승지, 좌부승지, <a class="new" href="/w/index.php?title=%EC%A2%8C%EC%8A%B9%EC%A7%80&amp;action=edit&amp;redlink=1" title="좌승지 (없는 문서)">좌승지</a> 등을 거쳐 <a href="/wiki/1458%EB%85%84" title="1458년">1458년</a> 활을 잘 쏘았다 하여 돌솥 1개를 선물로 받았다.<sup class="reference" id="cite_ref-18"><a href="#cite_note-18">[18]</a></sup></p>
<p><a href="/wiki/1458%EB%85%84" title="1458년">1458년</a> <a href="/wiki/1%EC%9B%94" title="1월">1월</a> <a href="/wiki/%EC%8A%B9%EC%A0%95%EC%9B%90" title="승정원">승정원</a><a class="new" href="/w/index.php?title=%EC%9A%B0%EC%8A%B9%EC%A7%80&amp;action=edit&amp;redlink=1" title="우승지 (없는 문서)">우승지</a>, <a href="/wiki/3%EC%9B%94" title="3월">3월</a> <a class="new" href="/w/index.php?title=%EC%9A%B0%EB%B6%80%EC%8A%B9%EC%A7%80&amp;action=edit&amp;redlink=1" title="우부승지 (없는 문서)">우부승지</a>, <a href="/wiki/10%EC%9B%94" title="10월">10월</a> <a class="new" href="/w/index.php?title=%EC%A2%8C%EB%B6%80%EC%8A%B9%EC%A7%80&amp;action=edit&amp;redlink=1" title="좌부승지 (없는 문서)">좌부승지</a>, <a href="/wiki/1459%EB%85%84" title="1459년">1459년</a> <a href="/wiki/3%EC%9B%94" title="3월">3월</a> <a class="new" href="/w/index.php?title=%EC%A2%8C%EC%8A%B9%EC%A7%80&amp;action=edit&amp;redlink=1" title="좌승지 (없는 문서)">좌승지</a>, 그해 <a href="/wiki/4%EC%9B%94" title="4월">4월</a> 우승지가 되었다. <a href="/wiki/4%EC%9B%94" title="4월">4월</a> 우승지로 <a href="/wiki/%EB%AA%85%EB%82%98%EB%9D%BC" title="명나라">명나라</a>에서 파견된 사신을 인견하였다.</p>
<p><a href="/wiki/1459%EB%85%84" title="1459년">1459년</a> <a href="/wiki/4%EC%9B%94" title="4월">4월</a> 왕명으로 선물인 연회에 쓰이는 탁자들을 2품 이상의 대신들에게 나누어주었다.<sup class="reference" id="cite_ref-19"><a href="#cite_note-19">[19]</a></sup>, 그해 <a href="/wiki/6%EC%9B%94" title="6월">6월</a> 다시 좌승지가 되었다. 그해 <a href="/wiki/7%EC%9B%94" title="7월">7월</a>에는 대신들과 함께 군정의 점고를 의논하였다.<sup class="reference" id="cite_ref-20"><a href="#cite_note-20">[20]</a></sup> 그는 변방을 약탈하는 <a class="mw-redirect" href="/wiki/%EC%97%AC%EC%A7%84%EC%A1%B1" title="여진족">여진족</a>에 대한 강경 대응을 건의하는 한편 북방에 성곽을 쌓고 정병을 파견하여 미구에 있을 <a class="mw-redirect" href="/wiki/%EC%97%AC%EC%A7%84%EC%A1%B1" title="여진족">여진족</a>의 침략에 대비해야 함을 역설하였다. 또한 삼남의 남해안, 동해안에도 병력을 배치하고 성곽을 개보수하여 왜구의 침입과 노략질에 방비할 것을 건의하였다.</p>
<h4><span class="mw-headline" id=".EA.B5.AD.EB.B0.A9.2C_.EA.B5.B0.EC.82.AC_.ED.99.9C.EB.8F.99">국방, 군사 활동</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EA%B9%80%EC%A7%88_(1422%EB%85%84)&amp;action=edit&amp;section=11" title="부분 편집: 국방, 군사 활동">편집</a><span class="mw-editsection-bracket">]</span></span></h4>
<p><a href="/wiki/1459%EB%85%84" title="1459년">1459년</a> <a href="/wiki/8%EC%9B%94" title="8월">8월</a> <a href="/wiki/%EB%B3%91%EC%A1%B0" title="병조">병조</a><a href="/wiki/%EC%B0%B8%ED%8C%90" title="참판">참판</a> 상락군(上洛君)이 되어 <a href="/wiki/%EC%A1%B0%EC%84%A0_%EC%84%B8%EC%A1%B0" title="조선 세조">세조</a>의 국방정책 수립에 참여하고 그 뒤 다시 상락군(上洛君)에 봉하여졌다. 그해 <a href="/wiki/12%EC%9B%94" title="12월">12월</a> <a class="mw-redirect" href="/wiki/%EC%9D%B4%EC%A1%B0%ED%8C%90%EC%84%9C" title="이조판서">이조판서</a> <a href="/wiki/%EA%B5%AC%EC%B9%98%EA%B4%80" title="구치관">구치관</a>·<a class="mw-redirect" href="/wiki/%EC%98%88%EC%A1%B0%ED%8C%90%EC%84%9C" title="예조판서">예조판서</a> <a href="/wiki/%ED%99%8D%EC%9C%A4%EC%84%B1" title="홍윤성">홍윤성</a>과 함께 무경(武經)과 병서(兵書), 진법 등을 강론하였다.</p>
<p><a href="/wiki/1460%EB%85%84" title="1460년">1460년</a> <a href="/wiki/%ED%95%9C%EB%AA%85%ED%9A%8C" title="한명회">한명회</a>의 처가 병이 들자 그와 임무를 교체하라는 명을 받았다.<sup class="reference" id="cite_ref-21"><a href="#cite_note-21">[21]</a></sup></p>
<p>그해 <a href="/wiki/2%EC%9B%94" title="2월">2월</a> <a href="/wiki/%ED%8F%89%EC%95%88%EB%8F%84" title="평안도">평안도</a>와 <a href="/wiki/%ED%99%A9%ED%95%B4%EB%8F%84" title="황해도">황해도</a>의 도순찰사를 겸직하여 파견되었다.<sup class="reference" id="cite_ref-22"><a href="#cite_note-22">[22]</a></sup> 이어 <a href="/wiki/%ED%95%A8%EA%B2%BD%EB%8F%84" title="함경도">함경도</a>의 도순찰사를 겸하고 서북면 도체찰사가 되어, 서북면 도체찰사 겸 평안, 황해, 함경도 순찰사가 되어 부임하였다. 그 해, <a class="mw-disambig" href="/wiki/%EC%84%B8%EC%A1%B0" title="세조">세조</a>의 서쪽지방 순행 시 도체찰사 겸 평안, 황해도 도순찰사로 세조의 서북 지방 순찰을 수행하였다.</p>
<p>그는 <a href="/wiki/%EC%A1%B0%EC%84%A0_%EC%84%B8%EC%A1%B0" title="조선 세조">세조</a>의 병력 증강책의 추진에 앞장섰으며 그는 <a href="/wiki/%EA%B2%BD%EC%83%81%EB%8F%84" title="경상도">경상도</a>와 <a href="/wiki/%EC%A0%84%EB%9D%BC%EB%8F%84" title="전라도">전라도</a>의 군적 순찰사(軍籍巡察使)가 되어 병역을 면제받거나 회피하는 양반과 평민들을 단속하여 군역을 부과했다. 양반이나 평민 면제자에게 군역을 부과하려는 세조의 뜻을 잘 받들고 군액(軍額)과 병력 충원에 힘써서, 한산(閑散)한 문무과(文武科) 출신과 생원(生員)·진사(進士)들도 모두 군역에 편입하였으며, 비록 심한 폐질이 있는 자라도 혹 면제되지 못하는 수가 있었는데, 그의 종사관(從事官) 양진손(梁震孫)은 더욱 각박하였으므로, 사관이 그의 졸기에 논하기를 이 때문에 "원망이 여러 곳에서 일어났다.<sup class="reference" id="cite_ref-seong09_14-1"><a href="#cite_note-seong09-14">[14]</a></sup>"고 하기도 하였다.</p>
<p><a href="/wiki/1460%EB%85%84" title="1460년">1460년</a> <a href="/wiki/10%EC%9B%94" title="10월">10월</a>에는 대장으로서 각지의 병마를 점고하여 관리가 허술한 방백들을 탄핵하기도 하였다.</p>
<h4><span class="mw-headline" id=".EA.B4.80.EB.A3.8C_.EC.83.9D.ED.99.9C">관료 생활</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EA%B9%80%EC%A7%88_(1422%EB%85%84)&amp;action=edit&amp;section=12" title="부분 편집: 관료 생활">편집</a><span class="mw-editsection-bracket">]</span></span></h4>
<p><a href="/wiki/1460%EB%85%84" title="1460년">1460년</a>(세조 6년) 세조가 두 차례에 걸쳐 서도(西道)지역 순행(巡幸)을 하자, 그는 왕의 서도 순시를 위해 특별히 <a href="/wiki/%ED%99%A9%ED%95%B4%EB%8F%84" title="황해도">황해도</a><a class="new" href="/w/index.php?title=%EB%8F%84%EC%88%9C%EC%B0%B0%EC%82%AC&amp;action=edit&amp;redlink=1" title="도순찰사 (없는 문서)">도순찰사</a>와 <a href="/wiki/%ED%8F%89%EC%95%88%EB%8F%84" title="평안도">평안도</a>도순찰사가 되고, 그해 <a href="/wiki/7%EC%9B%94" title="7월">7월</a> 다시 <a href="/wiki/%EB%B3%91%EC%A1%B0" title="병조">병조</a><a href="/wiki/%EC%B0%B8%ED%8C%90" title="참판">참판</a>이 되었다. <a href="/wiki/1461%EB%85%84" title="1461년">1461년</a> <a href="/wiki/1%EC%9B%94" title="1월">1월</a> 다시 <a href="/wiki/%ED%8F%89%EC%95%88%EB%8F%84" title="평안도">평안도</a><a class="new" href="/w/index.php?title=%EB%8F%84%EC%88%9C%EC%B0%B0%EC%82%AC&amp;action=edit&amp;redlink=1" title="도순찰사 (없는 문서)">도순찰사</a> 겸 <a href="/wiki/%ED%8F%89%EC%95%88%EB%8F%84" title="평안도">평안도</a><a href="/wiki/%EA%B4%80%EC%B0%B0%EC%82%AC" title="관찰사">관찰사</a>로 나갔다가 <a href="/wiki/2%EC%9B%94" title="2월">2월</a> 다시 <a href="/wiki/%EB%B3%91%EC%A1%B0" title="병조">병조</a><a href="/wiki/%EC%B0%B8%ED%8C%90" title="참판">참판</a>이 되었다. 그해 <a href="/wiki/5%EC%9B%94" title="5월">5월</a>에 <a href="/wiki/%ED%8F%89%EC%95%88%EB%8F%84" title="평안도">평안도</a> <a class="new" href="/w/index.php?title=%EB%8F%84%EA%B4%80%EC%B0%B0%EC%82%AC&amp;action=edit&amp;redlink=1" title="도관찰사 (없는 문서)">도관찰사</a>로 나갔다.</p>
<p><a href="/wiki/1463%EB%85%84" title="1463년">1463년</a> <a href="/wiki/3%EC%9B%94" title="3월">3월</a> <a href="/wiki/%EA%B3%B5%EC%A1%B0" title="공조">공조</a><a href="/wiki/%ED%8C%90%EC%84%9C" title="판서">판서</a>로 승진하고 이후 병조판서·형조판서를 거쳐 <a href="/wiki/11%EC%9B%94" title="11월">11월</a> 도진무(都鎭撫)가 되었다가 다시 <a href="/wiki/%ED%98%95%EC%A1%B0" title="형조">형조</a><a href="/wiki/%ED%8C%90%EC%84%9C" title="판서">판서</a>가 되었다. <a href="/wiki/1464%EB%85%84" title="1464년">1464년</a> <a href="/wiki/2%EC%9B%94" title="2월">2월</a> 우상 대장(右廂大將), <a href="/wiki/3%EC%9B%94" title="3월">3월</a> 다시 형조판서가 되었다.</p>
<p><a href="/wiki/1464%EB%85%84" title="1464년">1464년</a> <a href="/wiki/8%EC%9B%94" title="8월">8월</a> 도진무(都鎭撫), <a href="/wiki/9%EC%9B%94" title="9월">9월</a> 형조판서, <a href="/wiki/1465%EB%85%84" title="1465년">1465년</a> <a href="/wiki/4%EC%9B%94" title="4월">4월</a> 다시 병조판서, <a href="/wiki/1466%EB%85%84" title="1466년">1466년</a> <a href="/wiki/%EC%9D%98%EC%A0%95%EB%B6%80" title="의정부">의정부</a> <a href="/wiki/%EC%9A%B0%EC%B0%B8%EC%B0%AC" title="우참찬">우참찬</a> 등을 지내고 숭정 대부(崇政大夫)로 승진하였다. <a href="/wiki/1467%EB%85%84" title="1467년">1467년</a> 행 좌참찬(行左參贊)이 되었다가 그해 <a href="/wiki/7%EC%9B%94" title="7월">7월</a> 겸 의금부 판사(兼義禁府判事)를 겸임하였다. 그해 <a href="/wiki/%EA%B2%BD%EC%83%81%EB%8F%84" title="경상도">경상도</a> <a href="/wiki/%EA%B4%80%EC%B0%B0%EC%82%AC" title="관찰사">관찰사</a>가 되었다.</p>
<p><a href="/wiki/1468%EB%85%84" title="1468년">1468년</a>(세조 14년) 다시 상락군에 봉해진 뒤 <a href="/wiki/%EC%A1%B0%EC%84%A0_%EC%98%88%EC%A2%85" title="조선 예종">예종</a>이 즉위하자 <a href="/wiki/%EC%9D%98%EC%A0%95%EB%B6%80" title="의정부">의정부</a> <a href="/wiki/%EC%9A%B0%EC%9D%98%EC%A0%95" title="우의정">우의정</a>에 발탁 되었으며 <a href="/wiki/%EC%A1%B0%EC%84%A0_%EC%84%B8%EC%A1%B0" title="조선 세조">세조</a>의 능침을 정하는 산릉 도감 제조(右議政山陵都監提調)를 겸임하였다.</p>
<p>그 뒤 겸 경연 영사(兼經筵領事)를 겸하다가 곧 <a href="/wiki/%EC%A2%8C%EC%9D%98%EC%A0%95" title="좌의정">좌의정</a>으로 승진했다. 이어 상락군에서 상락부원군으로 진봉되었다.</p>
<p>이때 <a class="mw-redirect" href="/wiki/%EC%84%B8%EC%A2%85%EB%8C%80%EC%99%95" title="세종대왕">세종대왕</a>의 능침을 이장할 때 <a class="new" href="/w/index.php?title=%EC%B2%9C%EB%A6%89_%EB%8F%84%EA%B0%90&amp;action=edit&amp;redlink=1" title="천릉 도감 (없는 문서)">천릉 도감</a> 제조(左議政遷陵都監提調)의 한사람으로 선발되었고, <a href="/wiki/1469%EB%85%84" title="1469년">1469년</a> <a href="/wiki/1%EC%9B%94" title="1월">1월</a> 평안도 도체찰사(左議政平安道都體察使)로 다시 부임하였다. 그해 <a href="/wiki/2%EC%9B%94" title="2월">2월</a> 다시 <a href="/wiki/%EC%A2%8C%EC%9D%98%EC%A0%95" title="좌의정">좌의정</a>(右議政)에 임명되었다.</p>
<p><a href="/wiki/1468%EB%85%84" title="1468년">1468년</a> <a href="/wiki/%EC%A1%B0%EC%84%A0_%EC%84%B8%EC%A1%B0" title="조선 세조">세조</a>가 붕어하였을 때 <a href="/wiki/%ED%95%9C%EB%AA%85%ED%9A%8C" title="한명회">한명회</a>(韓明澮)·<a href="/wiki/%EC%8B%A0%EC%88%99%EC%A3%BC" title="신숙주">신숙주</a>(申叔舟) 등과 함께 <a class="mw-disambig" href="/wiki/%EC%9B%90%EC%83%81" title="원상">원상</a>이 됐고, <a href="/wiki/1469%EB%85%84" title="1469년">1469년</a> <a href="/wiki/%EC%A1%B0%EC%84%A0_%EC%98%88%EC%A2%85" title="조선 예종">예종</a>이 죽었을 때도 <a href="/wiki/%ED%95%9C%EB%AA%85%ED%9A%8C" title="한명회">한명회</a>·<a href="/wiki/%EC%8B%A0%EC%88%99%EC%A3%BC" title="신숙주">신숙주</a>와 함께 원상직을 수행하였다.<sup class="reference" id="cite_ref-23"><a href="#cite_note-23">[23]</a></sup></p>
<p><a href="/wiki/%EC%98%81%EC%9D%98%EC%A0%95" title="영의정">영의정</a> <a href="/wiki/%ED%95%9C%EB%AA%85%ED%9A%8C" title="한명회">한명회</a>, <a href="/wiki/%EC%8B%A0%EC%88%99%EC%A3%BC" title="신숙주">신숙주</a> 등과 함께 원상(院相)으로서 서정을 주관하였다. 이들 원상세력과 함께 나이 어린 예종과 성종 초기에 국정서무를 주관하여 권력공백기의 혼란을 차단하였으나, <a href="/wiki/%EC%A1%B0%EC%84%A0_%EC%98%88%EC%A2%85" title="조선 예종">예종</a> 말년 <a href="/wiki/%EC%A1%B0%EC%84%A0_%EC%84%B1%EC%A2%85" title="조선 성종">자을산군</a>의 강력한 경쟁자였던 왕족 구성군(龜城君) 이준(李浚)을 제거하고 <a href="/wiki/%EC%A1%B0%EC%84%A0_%EC%84%B1%EC%A2%85" title="조선 성종">자을산군</a>의 즉위를 지지하였다.</p>
<h3><span class="mw-headline" id=".EC.83.9D.EC.95.A0_.ED.9B.84.EB.B0.98">생애 후반</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EA%B9%80%EC%A7%88_(1422%EB%85%84)&amp;action=edit&amp;section=13" title="부분 편집: 생애 후반">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<h4><span class="mw-headline" id=".EC.84.9C.EC.A0.81_.ED.8E.B8.EC.B0.AC_.ED.99.9C.EB.8F.99">서적 편찬 활동</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EA%B9%80%EC%A7%88_(1422%EB%85%84)&amp;action=edit&amp;section=14" title="부분 편집: 서적 편찬 활동">편집</a><span class="mw-editsection-bracket">]</span></span></h4>
<p>한편 그는 영의정 <a href="/wiki/%ED%95%9C%EB%AA%85%ED%9A%8C" title="한명회">한명회</a>(韓明澮), <a href="/wiki/%EC%8B%A0%EC%88%99%EC%A3%BC" title="신숙주">신숙주</a> 등과 함께 원상 세력을 형성하여 왕실을 견제하였다. <a href="/wiki/1469%EB%85%84" title="1469년">1469년</a> 성종 즉위후 경연청영사(經筵廳領事)에 임명되고, <a href="/wiki/1470%EB%85%84" title="1470년">1470년</a> 사옹원 제조(司饔院提調)가 되었다. 그 뒤 정치적 영향력이 큰 구성군 준(龜城君 浚)을 공격, 숙청하였다.</p>
<p><a href="/wiki/1471%EB%85%84" title="1471년">1471년</a>에는 성종의 즉위를 도운 <a class="new" href="/w/index.php?title=%EC%A2%8C%EB%A6%AC%EA%B3%B5%EC%8B%A0&amp;action=edit&amp;redlink=1" title="좌리공신 (없는 문서)">좌리공신</a>을 서훈할 때 순성명량경제좌리공신 2등(純誠明亮經濟佐理功臣2等)으로 서훈되고 상락부원군(上洛府院君)에 봉해졌다. <a href="/wiki/%EC%A1%B0%EC%84%A0_%EC%84%B1%EC%A2%85" title="조선 성종">성종</a> 때는 《고려사절요》의 증보 편집, 《경국대전》의 편찬에 참여하기도 했다.</p>
<p>세조의 공신으로 특히 세조의 병제개편에 깊이 관여하였으며, 1465년에 병전(兵典)을 고열(考閱)하였으며, 1470년에는 이전(吏典)의 교정, 교열 작업에도 참여하였다. 성종 초반, 관료들이 사적으로 보유한 사노비들의 수를 점검하게 하여 이를 감시, 사병을 형성하는 것을 방지하게 했다. <a href="/wiki/1472%EB%85%84" title="1472년">1472년</a> 대광 보국 숭록 대부(大匡輔國崇祿大夫) 상락 부원군(上洛府院君)이 되었다.</p>
<p><br/></p>
<h4><span class="mw-headline" id=".EC.B5.9C.ED.9B.84">최후</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EA%B9%80%EC%A7%88_(1422%EB%85%84)&amp;action=edit&amp;section=15" title="부분 편집: 최후">편집</a><span class="mw-editsection-bracket">]</span></span></h4>
<p><a href="/wiki/1473%EB%85%84" title="1473년">1473년</a> 다시 사옹원 도제조(司饔院都提調)가 되었다가 <a href="/wiki/1474%EB%85%84" title="1474년">1474년</a> 소환되어 <a href="/wiki/%EC%9D%98%EC%A0%95%EB%B6%80" title="의정부">의정부</a><a href="/wiki/%EC%9A%B0%EC%9D%98%EC%A0%95" title="우의정">우의정</a>에 재임명되었다.</p>
<p>글을 잘 썼으므로 <a href="/wiki/1474%EB%85%84" title="1474년">1474년</a> 우의정으로 <a href="/wiki/%EB%AA%85%EB%82%98%EB%9D%BC" title="명나라">명나라</a>에 <a href="/wiki/%EC%A1%B0%EC%84%A0_%EB%8D%95%EC%A2%85" title="조선 덕종">의경세자</a>를 왕으로 추존하면서 시호를 얻으러 파견되는 주문사(奏聞使)에 임명되어 <a class="mw-redirect" href="/wiki/%EB%B6%81%EA%B2%BD" title="북경">북경</a>에 다녀왔다. 의경왕(懿敬王)의 시호를 허락을 받아가지고 돌아오니, <a href="/wiki/%EC%A1%B0%EC%84%A0_%EC%84%B1%EC%A2%85" title="조선 성종">성종</a>은 특별히 상으로 그에게 토지와 노비를 내렸다. 법률 개정에도 참여하여 <a href="/wiki/1465%EB%85%84" title="1465년">1465년</a>에 병전(兵典)을 고열(考閱)하고, <a href="/wiki/1470%EB%85%84" title="1470년">1470년</a> 이전(吏典)의 교정과 감수도 맡아보았다.</p>
<p><a href="/wiki/1477%EB%85%84" title="1477년">1477년</a>(성종 8년) 병을 이유로 사직을 청하였으나 <a href="/wiki/%EC%A1%B0%EC%84%A0_%EC%84%B1%EC%A2%85" title="조선 성종">성종</a>이 자신을 더 도와달라며 이를 받아들이지 않았다. <a href="/wiki/1478%EB%85%84" title="1478년">1478년</a>(성종 9년) <a href="/wiki/2%EC%9B%94_24%EC%9D%BC" title="2월 24일">2월 24일</a> 병으로 사망하니<sup class="reference" id="cite_ref-24"><a href="#cite_note-24">[24]</a></sup> 사망당시 향년 56세였다. 문정(文靖)의 시호가 내려졌다.</p>

<h2><span class="mw-headline" id=".ED.8F.89.EA.B0.80.EC.99.80_.EB.B9.84.ED.8C.90">평가와 비판</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EA%B9%80%EC%A7%88_(1422%EB%85%84)&amp;action=edit&amp;section=20" title="부분 편집: 평가와 비판">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>장인인 <a href="/wiki/%EC%A0%95%EC%B0%BD%EC%86%90" title="정창손">정창손</a>의 강압을 받아 그의 사실을 고변하여 훼절하기도 하였으나 세조의 두터운 신뢰를 받았다.</p>
<p>김질은 풍의(風儀)가 아름답고 언론(言論)을 잘하였으며, 부모를 효성으로 섬기고 형제를 우애(友愛)로 대하였다.<sup class="reference" id="cite_ref-seong09_14-2"><a href="#cite_note-seong09-14">[14]</a></sup>고 평하였다.</p>
<p>사신은 실록에서 그가 양반, 생원, 진사시 합격자들도 모두 군역을 부과하여 원망이 심했다고 한다. “김질의 사람됨은 기국(器局)이 관후하고, 어진이를 좋아하고 선비를 좋아해서 재상의 풍도가 있었으나, 나라를 경륜하는 것은 능한 바가 아니었다. 일찍이 하이도(下二道)의 군적 순찰사(軍籍巡察使)가 되었는데, 세조(世祖)의 뜻이 사려(師旅)를 확장하는 데 있는 것을 알고 오직 받들어 순종하여 군액(軍額)을 늘리기에만 힘써서, 한산(閑散)한 문무과(文武科) 출신과 생원(生員)·진사(進士)를 다 군열(軍列)에 편입하여 비록 심한 폐질이 있는 자라도 혹 면제되지 못하는 수가 있었으며, 종사관(從事官) 양진손(梁震孫)은 더욱 각박하였으므로, 원망이 여러 곳에서 일어났다.<sup class="reference" id="cite_ref-seong09_14-3"><a href="#cite_note-seong09-14">[14]</a></sup>”는 것이다.</p>
<p>다른 <a href="/wiki/%EC%A7%91%ED%98%84%EC%A0%84" title="집현전">집현전</a>의 학사들과 함께 <a class="mw-redirect" href="/wiki/%EB%8B%A8%EC%A2%85_%EB%B3%B5%EC%9C%84_%EC%9A%B4%EB%8F%99" title="단종 복위 운동">단종 복위 운동</a> 거사에 참여하였으나 <a href="/wiki/%EC%8B%A0%EC%88%99%EC%A3%BC" title="신숙주">신숙주</a>와 함께 <a href="/wiki/%EC%A1%B0%EC%84%A0_%EC%84%B8%EC%A1%B0" title="조선 세조">세조</a>의 편에 가담함으로써, <a class="mw-disambig" href="/wiki/%EC%84%B8%EC%A1%B0" title="세조">세조</a> 말기 이후 정계한 <a href="/wiki/%EC%82%AC%EB%A6%BC%ED%8C%8C" title="사림파">사림파</a>로부터 심한 비판을 받았다.</p>
<h2><span class="mw-headline" id=".EA.B8.B0.ED.83.80">기타</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EA%B9%80%EC%A7%88_(1422%EB%85%84)&amp;action=edit&amp;section=21" title="부분 편집: 기타">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>무예에 재주가 있어 여러번 활쏘기 대회에서 우수한 성적을 거두어 상과 칭찬을 들었고, 무경과 병법, 진법 등을 유창하게 강론하였다.</p>
<p>동명이인인 효자 영모당 <a href="/wiki/%EA%B9%80%EC%A7%88_(1496%EB%85%84)" title="김질 (1496년)">김질</a>(金質)은 그의 종증손이었다.</p>
<h3><span class="mw-headline" id=".EA.B9.80.EC.A2.85.EC.A7.81.EC.9D.98_.ED.97.8C.EC.8B.9C">김종직의 헌시</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EA%B9%80%EC%A7%88_(1422%EB%85%84)&amp;action=edit&amp;section=22" title="부분 편집: 김종직의 헌시">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
