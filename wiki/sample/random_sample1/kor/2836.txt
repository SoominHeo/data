<p>신일고, 단국대를 거쳐 1990년 <a class="mw-redirect" href="/wiki/OB_%EB%B2%A0%EC%96%B4%EC%8A%A4" title="OB 베어스">OB 베어스</a>에 입단하면서 1980년대를 주름잡던 <a class="mw-redirect" href="/wiki/OB_%EB%B2%A0%EC%96%B4%EC%8A%A4" title="OB 베어스">OB 베어스</a>의 포수 듀오 <a href="/wiki/%EC%A1%B0%EB%B2%94%ED%98%84" title="조범현">조범현</a>과 <a href="/wiki/%EA%B9%80%EA%B2%BD%EB%AC%B8" title="김경문">김경문</a>을 밀어내고 주전 포수 자리를 차지했다. 1990년대를 대표하는 최고의 수비형 포수 중 하나였고, <a class="mw-redirect" href="/wiki/OB_%EB%B2%A0%EC%96%B4%EC%8A%A4" title="OB 베어스">OB 베어스</a>의 주전으로 뛰면서 상대적으로 장타력이 좋은 박현영과 출장 시간을 이상적으로 배분했으며, <a href="/wiki/%EA%B9%80%EC%9D%B8%EC%8B%9D_(1947%EB%85%84)" title="김인식 (1947년)">김인식</a> 감독 취임 이후 일발 장타력을 인정받던 <a href="/wiki/%EC%9D%B4%EB%8F%84%ED%98%95_(%EC%95%BC%EA%B5%AC%EC%9D%B8)" title="이도형 (야구인)">이도형</a>까지 포수 진에 가세한 덕분에, OB의 포수 진 은 현재까지도 내려오는 전통의 포수 왕국의 컬러를 확립하게 되었다. 아무래도 공격력 부분에선 백업인 박현영, <a href="/wiki/%EC%9D%B4%EB%8F%84%ED%98%95_(%EC%95%BC%EA%B5%AC%EC%9D%B8)" title="이도형 (야구인)">이도형</a>에 비해 열세인 덕분에 특출난 시즌 성적을 기록하지는 못했으나, 1995년 한국시리즈 우승에 크게 기여한 일등공신이었다. 이후 국가대표 출신 초대형 포수 유망주 <a href="/wiki/%EC%B5%9C%EA%B8%B0%EB%AC%B8_(%EC%95%BC%EA%B5%AC%EC%9D%B8)" title="최기문 (야구인)">최기문</a>, <a href="/wiki/%EC%A7%84%EA%B0%91%EC%9A%A9" title="진갑용">진갑용</a>, 홍성흔이 연달아 입단하며, <strong class="selflink">김태형</strong>의 주전 포수 자리를 자연스럽게 세대교체하는 시도가 진행되었는데, 의외로 <a href="/wiki/%EC%B5%9C%EA%B8%B0%EB%AC%B8_(%EC%95%BC%EA%B5%AC%EC%9D%B8)" title="최기문 (야구인)">최기문</a>을 밀어낸 <a href="/wiki/%EC%A7%84%EA%B0%91%EC%9A%A9" title="진갑용">진갑용</a>이 타격, 수비 양쪽에서 부진을 거듭하며 확실한 후계자로 믿음을 주지 못했다.</p>
<h3><span class="mw-headline" id=".EB.91.90.EC.82.B0_.EB.B2.A0.EC.96.B4.EC.8A.A4_.EC.8B.9C.EC.A0.88"><a href="/wiki/%EB%91%90%EC%82%B0_%EB%B2%A0%EC%96%B4%EC%8A%A4" title="두산 베어스">두산 베어스</a> 시절</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EA%B9%80%ED%83%9C%ED%98%95_(%EC%95%BC%EA%B5%AC%EC%9D%B8)&amp;action=edit&amp;section=3" title="부분 편집: 두산 베어스 시절">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>1999년부터 신인 열세로 평가받던 <a href="/wiki/%ED%99%8D%EC%84%B1%ED%9D%94" title="홍성흔">홍성흔</a>이 의외로 주전 포수를 차지하면서, <a href="/wiki/%ED%99%8D%EC%84%B1%ED%9D%94" title="홍성흔">홍성흔</a>의 백업과 수비 강화를 위한 역할로 자연스럽게 세대교체가 진행되었다. 2001년 플레잉코치로 보직이 변경되었으며, 6경기 출장을 끝으로 그대로 은퇴했다. 선수 시절 비슷한 커리어의 수비형 포수였고, 이후 현역 감독으로 경쟁 중인 대선배 <a href="/wiki/%EC%A1%B0%EB%B2%94%ED%98%84" title="조범현">조범현</a>, <a href="/wiki/%EA%B9%80%EA%B2%BD%EB%AC%B8" title="김경문">김경문</a>보다는 현역 시절 타격을 잘했다.</p>
<p>한국 프로야구 선수협회 5적 네임밸류므로 불렸으나 당시 구단주가 KBO 회장이면서 두산그룹 오너였던 박용오 회장으로 선수협에 가입하면 구단 자체를 없애 버린다고 엄포를 놓자 김태형은 선수들의 안전을 보장받는 조건으로 구단과 선수협 가입 불가에 합의하였으나 구단이 <a href="/wiki/%EC%8B%AC%EC%A0%95%EC%88%98" title="심정수">심정수</a>를 트레이드 시키면서 구단의 약속이 이행되지 않아 다시 선수들을 이끌고 선수협에 가입한다.</p>
<h2><span class="mw-headline" id=".EC.A7.80.EB.8F.84.EC.9E.90_.EA.B2.BD.EB.A0.A5">지도자 경력</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EA%B9%80%ED%83%9C%ED%98%95_(%EC%95%BC%EA%B5%AC%EC%9D%B8)&amp;action=edit&amp;section=4" title="부분 편집: 지도자 경력">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<h3><span class="mw-headline" id=".EB.91.90.EC.82.B0_.EB.B2.A0.EC.96.B4.EC.8A.A4_.EC.BD.94.EC.B9.98_.EC.8B.9C.EC.A0.88"><a href="/wiki/%EB%91%90%EC%82%B0_%EB%B2%A0%EC%96%B4%EC%8A%A4" title="두산 베어스">두산 베어스</a> 코치 시절</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EA%B9%80%ED%83%9C%ED%98%95_(%EC%95%BC%EA%B5%AC%EC%9D%B8)&amp;action=edit&amp;section=5" title="부분 편집: 두산 베어스 코치 시절">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>현역 마지막 시즌부터 플레잉 코치로 뛰기 시작했고 다음 해 은퇴하면서 두산에서만 10년 동안 배터리코치로만 재직하면서 <a href="/wiki/%ED%99%8D%EC%84%B1%ED%9D%94" title="홍성흔">홍성흔</a> 이후의 포수 자원인 <a href="/wiki/%EC%9A%A9%EB%8D%95%ED%95%9C" title="용덕한">용덕한</a>, <a href="/wiki/%EC%96%91%EC%9D%98%EC%A7%80" title="양의지">양의지</a> 등을 강인권 코치와 프로에 안착시키는데 큰 영향을 주었다. <a href="/wiki/%EB%91%90%EC%82%B0_%EB%B2%A0%EC%96%B4%EC%8A%A4" title="두산 베어스">두산 베어스</a>가 <a href="/wiki/%EA%B9%80%EA%B4%91%EC%88%98_(1959%EB%85%84)" title="김광수 (1959년)">김광수</a> 감독대행 체제로 2011 시즌이 끝난 후 감독 후보군에 올랐지만, <a href="/wiki/%EA%B9%80%EC%A7%84%EC%9A%B1" title="김진욱">김진욱</a> 투수코치가 신임 감독이 됨에 따라 팀을 떠나서 <a class="mw-redirect" href="/wiki/SK%EC%99%80%EC%9D%B4%EB%B2%88%EC%8A%A4" title="SK와이번스">SK와이번스</a>에 1군 배터리코치로 옮겼다.</p>
<h3><span class="mw-headline" id="SK_.EC.99.80.EC.9D.B4.EB.B2.88.EC.8A.A4_.EC.BD.94.EC.B9.98_.EC.8B.9C.EC.A0.88"><a href="/wiki/SK_%EC%99%80%EC%9D%B4%EB%B2%88%EC%8A%A4" title="SK 와이번스">SK 와이번스</a> 코치 시절</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EA%B9%80%ED%83%9C%ED%98%95_(%EC%95%BC%EA%B5%AC%EC%9D%B8)&amp;action=edit&amp;section=6" title="부분 편집: SK 와이번스 코치 시절">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p><a class="mw-redirect" href="/wiki/SK%EC%99%80%EC%9D%B4%EB%B2%88%EC%8A%A4" title="SK와이번스">SK와이번스</a> 배터리 코치 시절 코치로서 능력은 유능하다고 인정받는다. <a class="mw-redirect" href="/wiki/SK%EC%99%80%EC%9D%B4%EB%B2%88%EC%8A%A4" title="SK와이번스">SK와이번스</a>로 자리를 옮긴 후에 수비적으로는 아직 모자란 모습을 보이지만 포수보다 플래툰으로 많이 나왔던 <a href="/wiki/%EC%9D%B4%EC%9E%AC%EC%9B%90_(%EC%95%BC%EA%B5%AC_%EC%84%A0%EC%88%98)" title="이재원 (야구 선수)">이재원</a>이 주전 포수로 발돋움할 수 있었던 데는 김태형의 공을 무시할 수 없다. 3년간 <a class="mw-redirect" href="/wiki/SK%EC%99%80%EC%9D%B4%EB%B2%88%EC%8A%A4" title="SK와이번스">SK와이번스</a>의 배터리코치로 있다가 2015년, <a href="/wiki/%EC%86%A1%EC%9D%BC%EC%88%98" title="송일수">송일수</a> 감독이 성적 부진으로 경질되자 친정팀으로 돌아와 감독을 맡았다.</p>
<h3><span class="mw-headline" id=".EB.91.90.EC.82.B0_.EB.B2.A0.EC.96.B4.EC.8A.A4_.EA.B0.90.EB.8F.85"><a href="/wiki/%EB%91%90%EC%82%B0_%EB%B2%A0%EC%96%B4%EC%8A%A4" title="두산 베어스">두산 베어스</a> 감독</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EA%B9%80%ED%83%9C%ED%98%95_(%EC%95%BC%EA%B5%AC%EC%9D%B8)&amp;action=edit&amp;section=7" title="부분 편집: 두산 베어스 감독">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>취임식 때 그는 공격적인 야구를 선보이겠다는 포부를 밝혔다. <a href="/wiki/2015%EB%85%84" title="2015년">2015년</a> 감독 부임 첫해 페넌트레이스 성적 79승 65패, 3위로 마쳤고 <a href="/wiki/2015%EB%85%84_KBO_%ED%8F%AC%EC%8A%A4%ED%8A%B8%EC%8B%9C%EC%A6%8C" title="2015년 KBO 포스트시즌">포스트시즌</a> 와일드카드에서 <a class="mw-redirect" href="/wiki/SK%EC%99%80%EC%9D%B4%EB%B2%88%EC%8A%A4" title="SK와이번스">SK</a>를 꺾고 올라온 <a href="/wiki/%EB%84%A5%EC%84%BC_%ED%9E%88%EC%96%B4%EB%A1%9C%EC%A6%88" title="넥센 히어로즈">넥센</a> 과의 준플레이오프에서 4차전 2-9로 크게 뒤지던 경기를 7회부터 9회까지 연속 9득점을 올리며 7점 차 대역전승, 3승 1패를 기록하며 플레이오프에 진출하였다. 이어 <a href="/wiki/NC_%EB%8B%A4%EC%9D%B4%EB%85%B8%EC%8A%A4" title="NC 다이노스">NC</a>와의 플레이오프에서도 벼랑 끝에 몰렸으나 4차전 <a href="/wiki/%EB%8D%94%EC%8A%A4%ED%8B%B4_%EB%8B%88%ED%8D%BC%ED%8A%B8" title="더스틴 니퍼트">더스틴 니퍼트</a>, 5차전 <a href="/wiki/%EC%9E%A5%EC%9B%90%EC%A4%80_(%EC%95%BC%EA%B5%AC_%EC%84%A0%EC%88%98)" title="장원준 (야구 선수)">장원준</a>의 활약으로 3승 2패로 <a href="/wiki/NC_%EB%8B%A4%EC%9D%B4%EB%85%B8%EC%8A%A4" title="NC 다이노스">NC</a>를 꺾고 감독 부임 첫해에 한국시리즈에 진출하였다. <a href="/wiki/2015%EB%85%84_%ED%95%9C%EA%B5%AD%EC%8B%9C%EB%A6%AC%EC%A6%88" title="2015년 한국시리즈">한국시리즈</a>에서 1차전 역전패를 당했으나 2차전부터 5차전을 내리 이기며 14년 만에 팀에 우승을 안겨주었다. 이 우승으로 김태형 감독은 역대 4번째로 감독 부임 첫해에 우승을 이끈 주인공이 되었고, 이러한 공로로 2015시즌 감독상을 받았다. <a href="/wiki/1995%EB%85%84" title="1995년">1995년</a> 현역 시절, <a href="/wiki/2001%EB%85%84" title="2001년">2001년</a> 코치 시절, <a href="/wiki/2015%EB%85%84" title="2015년">2015년</a> 감독 시절 우승하여 한팀에서 선수, 코치, 감독으로 모두 우승하는 기록을 세웠다. <a href="/wiki/2016%EB%85%84" title="2016년">2016년</a> <a href="/wiki/5%EC%9B%94_11%EC%9D%BC" title="5월 11일">5월 11일</a> <a class="mw-redirect" href="/wiki/SK%EC%99%80%EC%9D%B4%EB%B2%88%EC%8A%A4" title="SK와이번스">SK</a>와의 경기에서 통산 100승을 달성하였다. 전반기 종료 후 후반기를 앞두고 3년 재계약을 체결하였다.</p>
<h2><span class="mw-headline" id=".EA.B8.B0.ED.83.80">기타</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EA%B9%80%ED%83%9C%ED%98%95_(%EC%95%BC%EA%B5%AC%EC%9D%B8)&amp;action=edit&amp;section=8" title="부분 편집: 기타">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p><a class="mw-redirect" href="/wiki/OB_%EB%B2%A0%EC%96%B4%EC%8A%A4" title="OB 베어스">OB 베어스</a> 주장 시절에는 정말 무서운 군기반장이기도 했다. 경기 끝나고 선수들을 소집해서 엄하게 다그치는 일도 자주 있었다. 이광용의 옐로카드에 출연해서 해명하기로는 "당시 감독이던 김인식 감독이 성격이 부드러우셔서 군기반장을 해야 할 필요가 있었다"고 밝혔다.</p>
