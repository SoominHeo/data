<p><b>김한길</b>(金한길, <a href="/wiki/1953%EB%85%84" title="1953년">1953년</a> <a href="/wiki/9%EC%9B%94_17%EC%9D%BC" title="9월 17일">9월 17일</a> ~ )은 <a href="/wiki/%EB%8C%80%ED%95%9C%EB%AF%BC%EA%B5%AD" title="대한민국">대한민국</a>의 소설가, <a class="mw-redirect" href="/wiki/%EC%A0%95%EC%B9%98%EC%9D%B8" title="정치인">정치인</a>이다. 1992년 <a href="/wiki/%EB%8C%80%ED%95%9C%EB%AF%BC%EA%B5%AD_%EC%A0%9C14%EB%8C%80_%EB%8C%80%ED%86%B5%EB%A0%B9_%EC%84%A0%EA%B1%B0" title="대한민국 제14대 대통령 선거">대선</a>에서 <a href="/wiki/%EC%A0%95%EC%A3%BC%EC%98%81" title="정주영">정주영</a>의 <a href="/wiki/%ED%86%B5%EC%9D%BC%EA%B5%AD%EB%AF%BC%EB%8B%B9_(%EB%8C%80%ED%95%9C%EB%AF%BC%EA%B5%AD)" title="통일국민당 (대한민국)">통일국민당</a> 부대변인으로 정치계에 입문했다. 국회의원 경력은 1992년 통일국민당 소속으로 <a href="/wiki/%EB%8C%80%ED%95%9C%EB%AF%BC%EA%B5%AD_%EC%A0%9C14%EB%8C%80_%EA%B5%AD%ED%9A%8C%EC%9D%98%EC%9B%90_%EC%84%A0%EA%B1%B0" title="대한민국 제14대 국회의원 선거">제14대 총선</a>에 출마했으나 낙선했고 1995년 <a href="/wiki/%EA%B9%80%EC%98%81%EC%82%BC" title="김영삼">김영삼</a>의 <a href="/wiki/%EC%8B%A0%ED%95%9C%EA%B5%AD%EB%8B%B9" title="신한국당">신한국당</a> 후보로 <a href="/wiki/%EB%8C%80%ED%95%9C%EB%AF%BC%EA%B5%AD_%EC%A0%9C15%EB%8C%80_%EA%B5%AD%ED%9A%8C%EC%9D%98%EC%9B%90_%EC%84%A0%EA%B1%B0" title="대한민국 제15대 국회의원 선거">제15대 총선</a>에 출마할 것이라는 언론 보도 속에 진행하던 방송도 그만두었으나 1996년 <a href="/wiki/%EA%B9%80%EB%8C%80%EC%A4%91" title="김대중">김대중</a>의 <a href="/wiki/%EC%83%88%EC%A0%95%EC%B9%98%EA%B5%AD%EB%AF%BC%ED%9A%8C%EC%9D%98" title="새정치국민회의">새정치국민회의</a> 소속으로 <a <h2><span class="mw-headline" id=".EC.83.9D.EC.95.A0">생애</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EA%B9%80%ED%95%9C%EA%B8%B8&amp;action=edit&amp;section=1" title="부분 편집: 생애">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>김한길은 <a href="/wiki/1953%EB%85%84" title="1953년">1953년</a> <a href="/wiki/9%EC%9B%94_17%EC%9D%BC" title="9월 17일">9월 17일</a>에 <a href="/wiki/%EC%9D%BC%EB%B3%B8" title="일본">일본</a> <a href="/wiki/%EB%8F%84%EC%BF%84" title="도쿄">도쿄</a>에서 아버지 당산 <a href="/wiki/%EA%B9%80%EC%B2%A0_(1926%EB%85%84)" title="김철 (1926년)">김철</a>과 어머니 윤초옥 사이에서 태어났다. <a href="/wiki/1976%EB%85%84" title="1976년">1976년</a> <a href="/wiki/%EA%B1%B4%EA%B5%AD%EB%8C%80%ED%95%99%EA%B5%90" title="건국대학교">건국대</a> 정치외교학과를 졸업하고 <a href="/wiki/%EC%A4%91%EC%95%99%EC%97%AC%EC%9E%90%EA%B3%A0%EB%93%B1%ED%95%99%EA%B5%90" title="중앙여자고등학교">중앙여고</a>에서 잠시 교편을 잡았다가 <a href="/wiki/1981%EB%85%84" title="1981년">1981년</a>에 소설《바람과 박제》가 문학사상에서 소설부문 신인상을 수상하여 등단했다. 미국으로 건너가 미주 <a href="/wiki/%ED%95%9C%EA%B5%AD%EC%9D%BC%EB%B3%B4" title="한국일보">한국일보</a>, 미주 <a href="/wiki/%EC%A4%91%EC%95%99%EC%9D%BC%EB%B3%B4" title="중앙일보">중앙일보</a> 등에서 언론인으로 일했다. 귀국한 후에는 방송위원회 기획국 국장, 방송위원회 사무총장 서리와 기조실장을 지내다가 <a href="/wiki/%EC%86%8C%EC%84%A4%EA%B0%80" title="소설가">소설가</a>로 활동했다.<sup class="reference" id="cite_ref-4"><a href="#cite_note-4">[4]</a></sup></p>
<p><a href="/wiki/1987%EB%85%84" title="1987년">1987년</a> 화개장터에 대한 신문기사로 <a href="/wiki/%EC%A1%B0%EC%98%81%EB%82%A8" title="조영남">조영남</a>과 함께 '화개장터'를 작사하여 큰 화제가 되었다. <a href="/wiki/%EC%86%8C%EC%84%A4%EA%B0%80" title="소설가">소설가</a> 활동 외에 <a class="mw-redirect" href="/wiki/%EB%B0%A9%EC%86%A1%EC%9D%B8" title="방송인">방송인</a>으로 자신의 이름을 내건 토크쇼《김한길과 사람들》의 진행을 비롯하여 방송활동을 했다. <a href="/wiki/1981%EB%85%84" title="1981년">1981년</a>에 <a href="/wiki/%EC%9D%B4%EC%96%B4%EB%A0%B9" title="이어령">이어령</a>의 딸 <a href="/wiki/%EC%9D%B4%EB%AF%BC%EC%95%84_(%EB%AA%A9%ED%9A%8C%EC%9E%90)" title="이민아 (목회자)">이민아</a>와 결혼하였고 아들 김유진을 두었으며 1986년 미국에서 이혼하고 한국으로 돌아왔다. <a href="/wiki/2007%EB%85%84" title="2007년">2007년</a>에 김유진이 원인 모를 병으로 사망했다. <a href="/wiki/1992%EB%85%84" title="1992년">1992년</a> <a href="/wiki/%EB%8C%80%ED%95%9C%EB%AF%BC%EA%B5%AD_%EC%A0%9C14%EB%8C%80_%EA%B5%AD%ED%9A%8C%EC%9D%98%EC%9B%90_%EC%84%A0%EA%B1%B0" title="대한민국 제14대 국회의원 선거">제14대 총선</a>에 <a href="/wiki/%EC%84%9C%EC%9A%B8_%EB%8F%99%EC%9E%91%EA%B5%AC%EC%9D%98_%EA%B5%AD%ED%9A%8C%EC%9D%98%EC%9B%90" title="서울 동작구의 국회의원">서울 동작구 을</a>에 <a class="mw-disambig" href="/wiki/%ED%86%B5%EC%9D%BC%EA%B5%AD%EB%AF%BC%EB%8B%B9" title="통일국민당">통일국민당</a> 소속으로 출마하였으나 낙선하였다.<sup class="reference" id="cite_ref-5"><a href="#cite_note-5">[5]</a></sup>이후 <a href="/wiki/%EB%8C%80%ED%95%9C%EB%AF%BC%EA%B5%AD_%EC%A0%9C14%EB%8C%80_%EB%8C%80%ED%86%B5%EB%A0%B9_%EC%84%A0%EA%B1%B0" title="대한민국 제14대 대통령 선거">제14대 대선</a>에 출마한 <a href="/wiki/%EC%A0%95%EC%A3%BC%EC%98%81" title="정주영">정주영</a>의 공보특별보좌관을 맡았다.<sup class="reference" id="cite_ref-6"><a href="#cite_note-6">[6]</a></sup><sup class="reference" id="cite_ref-7"><a href="#cite_note-7">[7]</a></sup> <a href="/wiki/1995%EB%85%84" title="1995년">1995년</a> 43세에 배우 <a href="/wiki/%EC%B5%9C%EB%AA%85%EA%B8%B8_(%EB%B0%B0%EC%9A%B0)" title="최명길 (배우)">최명길</a>과 재혼하고 슬하에 2남을 두었다. 1996년 <a href="/wiki/%EB%8C%80%ED%95%9C%EB%AF%BC%EA%B5%AD_%EC%A0%9C15%EB%8C%80_%EA%B5%AD%ED%9A%8C%EC%9D%98%EC%9B%90_%EC%84%A0%EA%B1%B0" title="대한민국 제15대 국회의원 선거">제15대 총선</a>에 새정치국민회의 소속 전국구 의원으로 출마하여 당선됐고 1998년 김대중 정부 출범뒤 <a href="/wiki/%EC%B2%AD%EC%99%80%EB%8C%80" title="청와대">청와대</a>정책기획수석비서관과 <a href="/wiki/%EB%8C%80%ED%95%9C%EB%AF%BC%EA%B5%AD%EC%9D%98_%EB%AC%B8%ED%99%94%EC%B2%B4%EC%9C%A1%EA%B4%80%EA%B4%91%EB%B6%80_%EC%9E%A5%EA%B4%80" title="대한민국의 문화체육관광부 장관">문화관광부 장관</a>을 지냈다. <a href="/wiki/2006%EB%85%84" title="2006년">2006년</a>에 <a href="/wiki/%EC%97%B4%EB%A6%B0%EC%9A%B0%EB%A6%AC%EB%8B%B9" title="열린우리당">열린우리당</a> 원내대표로 선출됐으나 이듬해 <a href="/wiki/2007%EB%85%84" title="2007년">2007년</a>에 탈당하고 <a class="mw-redirect" href="/wiki/%EC%A4%91%EB%8F%84%EA%B0%9C%ED%98%81%ED%86%B5%ED%95%A9%EC%8B%A0%EB%8B%B9" title="중도개혁통합신당">중도개혁통합신당</a>을 창당하고 대표로 취임했다. <a href="/wiki/2008%EB%85%84" title="2008년">2008년</a> <a href="/wiki/1%EC%9B%94_6%EC%9D%BC" title="1월 6일">1월 6일</a> 대선 패배로 "노무현 프레임 극복의 필요성을 강조하며 서민과 중산층을 위한 정치가 실패한 데 대한 책임을 지겠다."며 <a href="/wiki/%EB%8C%80%ED%95%9C%EB%AF%BC%EA%B5%AD_%EC%A0%9C18%EB%8C%80_%EA%B5%AD%ED%9A%8C%EC%9D%98%EC%9B%90_%EC%84%A0%EA%B1%B0" title="대한민국 제18대 국회의원 선거">제18대 총선</a> 불출마와 정계은퇴를 선언하였다.</p>
<p><a href="/wiki/2012%EB%85%84" title="2012년">2012년</a> <a href="/wiki/4%EC%9B%94_11%EC%9D%BC" title="4월 11일">4월 11일</a>에 열린 <a href="/wiki/%EB%8C%80%ED%95%9C%EB%AF%BC%EA%B5%AD_%EC%A0%9C19%EB%8C%80_%EA%B5%AD%ED%9A%8C%EC%9D%98%EC%9B%90_%EC%84%A0%EA%B1%B0" title="대한민국 제19대 국회의원 선거">제19대 총선</a>에 출마하여 <a href="/wiki/%EC%84%9C%EC%9A%B8_%EA%B4%91%EC%A7%84%EA%B5%AC%EC%9D%98_%EA%B5%AD%ED%9A%8C%EC%9D%98%EC%9B%90" title="서울 광진구의 국회의원">광진구 갑</a>에서 당선되었다. <a href="/wiki/2012%EB%85%84" title="2012년">2012년</a> <a href="/wiki/6%EC%9B%94_9%EC%9D%BC" title="6월 9일">6월 9일</a> 실시된 <a class="mw-redirect" href="/wiki/%EB%AF%BC%EC%A3%BC%ED%86%B5%ED%95%A9%EB%8B%B9" title="민주통합당">민주통합당</a> 전당대회에서 8명의 후보 중 근소한 차이로 2위를 하여 최고위원에 선출됐다. <a href="/wiki/2013%EB%85%84" title="2013년">2013년</a> <a href="/wiki/5%EC%9B%94_4%EC%9D%BC" title="5월 4일">5월 4일</a> 실시된 <a href="/wiki/%EB%AF%BC%EC%A3%BC%EB%8B%B9_(%EB%8C%80%ED%95%9C%EB%AF%BC%EA%B5%AD,_2011%EB%85%84)" title="민주당 (대한민국, 2011년)">민주당</a> 전당대회에서 대표로 선출되었다. 다음해 <a href="/wiki/3%EC%9B%94_26%EC%9D%BC" title="3월 26일">3월 26일</a>에 <a href="/wiki/%EC%95%88%EC%B2%A0%EC%88%98" title="안철수">안철수</a>와 <a class="mw-redirect" href="/wiki/%EC%83%88%EC%A0%95%EC%B9%98%EB%AF%BC%EC%A3%BC%EC%97%B0%ED%95%A9" title="새정치민주연합">새정치민주연합</a>의 공동대표에 취임하였으나 <a href="/wiki/2014%EB%85%84_%EB%8C%80%ED%95%9C%EB%AF%BC%EA%B5%AD_%EC%9E%AC%EB%B3%B4%EA%B6%90%EC%84%A0%EA%B1%B0" title="2014년 대한민국 재보궐선거">2014년 7·30 재보궐선거</a>에 패배한 책임을 지고 <a href="/wiki/%EC%95%88%EC%B2%A0%EC%88%98" title="안철수">안철수</a>와 함께 동반사퇴하였다. 2016년 1월 3일 <a href="/wiki/%EB%8D%94%EB%B6%88%EC%96%B4%EB%AF%BC%EC%A3%BC%EB%8B%B9" title="더불어민주당">더불어민주당</a>을 탈당했다.</p>
<p>더불어민주당을 탈당한 후 국민의당에 합류하였다. 하지만 2016년 3월 17일 20대 총선에서 야권 연대가 무산된 책임을 지고 이번 총선에 불출마하겠다고 선언하였다.<sup class="reference" id="cite_ref-8"><a href="#cite_note-8">[8]</a></sup></p>
