<h2><span class="mw-headline" id=".EA.B0.9C.EC.9A.94">개요</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EA%B9%80%ED%95%B4%EC%9A%B4&amp;action=edit&amp;section=1" title="부분 편집: 개요">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p><a href="/wiki/%EA%B2%BD%EC%83%81%EB%82%A8%EB%8F%84" title="경상남도">경상남도</a> <a href="/wiki/%EB%A7%88%EC%82%B0%EC%8B%9C" title="마산시">마산시</a> 출생으로 <a href="/wiki/%EB%A7%88%EC%82%B0%EA%B3%B5%EC%97%85%EA%B3%A0%EB%93%B1%ED%95%99%EA%B5%90" title="마산공업고등학교">마산공업고등학교</a>, <a href="/wiki/%EB%8C%80%EA%B5%AC%EB%8C%80%ED%95%99%EA%B5%90" title="대구대학교">대구대학교</a>를 졸업하였다. 팀의 문전을 든든히 지켜 '슈퍼 거미'라는 별명을 얻었다.</p>
<h2><span class="mw-headline" id=".EC.B6.95.EA.B5.AC.EC.9D.B8_.EC.83.9D.ED.99.9C">축구인 생활</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EA%B9%80%ED%95%B4%EC%9A%B4&amp;action=edit&amp;section=2" title="부분 편집: 축구인 생활">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<h3><span class="mw-headline" id=".EC.84.A0.EC.88.98_.EC.83.9D.ED.99.9C">선수 생활</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EA%B9%80%ED%95%B4%EC%9A%B4&amp;action=edit&amp;section=3" title="부분 편집: 선수 생활">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p><a href="/wiki/1996%EB%85%84" title="1996년">1996년</a> <a class="mw-redirect" href="/wiki/%EC%84%B1%EB%82%A8_%EC%9D%BC%ED%99%94_%EC%B2%9C%EB%A7%88" title="성남 일화 천마">성남 일화 천마</a>에서 데뷔하여, <a href="/wiki/2008%EB%85%84" title="2008년">2008년</a>까지 몸담은 뒤 <a href="/wiki/2009%EB%85%84" title="2009년">2009년</a> <a href="/wiki/5%EC%9B%94_23%EC%9D%BC" title="5월 23일">5월 23일</a> <a href="/wiki/%EC%A0%84%EB%82%A8_%EB%93%9C%EB%9E%98%EA%B3%A4%EC%A6%88" title="전남 드래곤즈">전남 드래곤즈</a>와의 <a class="mw-redirect" href="/wiki/K-%EB%A6%AC%EA%B7%B8" title="K-리그">K-리그</a> 홈 경기에서 은퇴식을 가져 공식적으로 은퇴하였다.<sup class="reference" id="cite_ref-1"><a href="#cite_note-1">[1]</a></sup> <a href="/wiki/2001%EB%85%84" title="2001년">2001년</a>부터 <a href="/wiki/2003%EB%85%84" title="2003년">2003년</a>까지 <a class="mw-redirect" href="/wiki/%EC%84%B1%EB%82%A8_%EC%9D%BC%ED%99%94_%EC%B2%9C%EB%A7%88" title="성남 일화 천마">성남 일화 천마</a>의 <a class="mw-redirect" href="/wiki/K-%EB%A6%AC%EA%B7%B8" title="K-리그">K-리그</a> 3연패와 <a href="/wiki/1996%EB%85%84" title="1996년">1996년</a> <a href="/wiki/FA%EC%BB%B5_(%EB%8C%80%ED%95%9C%EB%AF%BC%EA%B5%AD)" title="FA컵 (대한민국)">FA컵</a> 우승 등에 큰 공헌을 하였다.</p>
<p><a href="/wiki/2008%EB%85%84" title="2008년">2008년</a> '<a class="mw-redirect" href="/wiki/K-%EB%A6%AC%EA%B7%B8" title="K-리그">K-리그</a> 대상' 시상식에서 공로상을 받았다.</p>
<h3><span class="mw-headline" id=".EA.B5.AD.EA.B0.80_.EB.8C.80.ED.91.9C_.EC.83.9D.ED.99.9C">국가 대표 생활</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EA%B9%80%ED%95%B4%EC%9A%B4&amp;action=edit&amp;section=4" title="부분 편집: 국가 대표 생활">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p><a href="/wiki/2000%EB%85%84_AFC_%EC%95%84%EC%8B%9C%EC%95%88%EC%BB%B5" title="2000년 AFC 아시안컵">2000년 AFC 아시안컵</a>에 최종 엔트리에 포함되었지만 본선 경기에서 기용되지는 않았다. <a href="/wiki/2003%EB%85%84" title="2003년">2003년</a> <a href="/wiki/%EC%9B%80%EB%B2%A0%EB%A5%B4%ED%88%AC_%EC%BD%94%EC%97%98%EB%A5%98" title="움베르투 코엘류">움베르투 코엘류</a> 감독에 의해 <a href="/wiki/%EB%8C%80%ED%95%9C%EB%AF%BC%EA%B5%AD_%EC%B6%95%EA%B5%AC_%EA%B5%AD%EA%B0%80%EB%8C%80%ED%91%9C%ED%8C%80" title="대한민국 축구 국가대표팀">국가대표팀</a>에 발탁되기도 하였지만, 부상으로 인해 경기에 나서지 못하였다.</p>
