<p><b>김현수</b> (金鉉洙, <a href="/wiki/1973%EB%85%84" title="1973년">1973년</a> <a href="/wiki/3%EC%9B%94_13%EC%9D%BC" title="3월 13일">3월 13일</a> ~)는 <a href="/wiki/%EB%8C%80%ED%95%9C%EB%AF%BC%EA%B5%AD" title="대한민국">대한민국</a>의 전 <a href="/wiki/%EC%B6%95%EA%B5%AC" title="축구">축구</a> 선수이자 지도자 이다. 선수 시절 포지션은 <a href="/wiki/%EC%88%98%EB%B9%84%EC%88%98" title="수비수">수비수</a>이다. 현재 <a href="/wiki/%EB%8C%80%EA%B5%AC_FC" title="대구 FC">대구 FC</a> 코치이다.</p>

<h2><span class="mw-headline" id=".EC.B6.95.EA.B5.AC.EC.9D.B8_.EC.83.9D.ED.99.9C">축구인 생활</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EA%B9%80%ED%98%84%EC%88%98_(1973%EB%85%84_3%EC%9B%94)&amp;action=edit&amp;section=1" title="부분 편집: 축구인 생활">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<h3><span class="mw-headline" id=".EC.84.A0.EC.88.98_.EC.83.9D.ED.99.9C">선수 생활</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EA%B9%80%ED%98%84%EC%88%98_(1973%EB%85%84_3%EC%9B%94)&amp;action=edit&amp;section=2" title="부분 편집: 선수 생활">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p><a href="/wiki/1995%EB%85%84" title="1995년">1995년</a> <a class="mw-redirect" href="/wiki/%EB%B6%80%EC%82%B0_%EB%8C%80%EC%9A%B0_%EB%A1%9C%EC%96%84%EC%A6%88" title="부산 대우 로얄즈">부산 대우 로얄즈</a>에서 선수 생활을 시작했으며 <a href="/wiki/1997%EB%85%84" title="1997년">1997년</a> <a class="mw-redirect" href="/wiki/%EB%B6%80%EC%82%B0_%EB%8C%80%EC%9A%B0_%EB%A1%9C%EC%96%84%EC%A6%88" title="부산 대우 로얄즈">부산</a>의 3관왕을 이끌었다. <a href="/wiki/2000%EB%85%84" title="2000년">2000년</a>엔 <a class="mw-redirect" href="/wiki/%EC%84%B1%EB%82%A8_%EC%9D%BC%ED%99%94_%EC%B2%9C%EB%A7%88" title="성남 일화 천마">성남 일화 천마</a>로 이적했다. 성남에서 보낸 첫 시즌에 <a href="/wiki/K%EB%A6%AC%EA%B7%B8_%EB%B2%A0%EC%8A%A4%ED%8A%B8_11" title="K리그 베스트 11">K리그 베스트 11</a>에 선정되었으며 <a href="/wiki/2002%EB%85%84" title="2002년">2002년</a>까지 3년 연속 <a href="/wiki/K%EB%A6%AC%EA%B7%B8_%EB%B2%A0%EC%8A%A4%ED%8A%B8_11" title="K리그 베스트 11">베스트 11</a>에 선정되었다.</p>
<p><a href="/wiki/2004%EB%85%84" title="2004년">2004년</a> 신생팀인 <a class="mw-redirect" href="/wiki/%EC%9D%B8%EC%B2%9C_%EC%9C%A0%EB%82%98%EC%9D%B4%ED%8B%B0%EB%93%9C" title="인천 유나이티드">인천 유나이티드</a>로 이적하여 플레잉 코치로 20경기에 출장한 뒤 <a href="/wiki/%EC%A0%84%EB%82%A8_%EB%93%9C%EB%9E%98%EA%B3%A4%EC%A6%88" title="전남 드래곤즈">전남 드래곤즈</a>로 이적했으나 6개월만에 상호 합의하에 계약을 해지하였다. 이후 <a class="mw-redirect" href="/wiki/%ED%81%AC%EB%A1%9C%EC%95%84%ED%8B%B0%EC%95%84_1.HNL" title="크로아티아 1.HNL">크로아티아 1.HNL</a>의 <a class="mw-redirect" href="/wiki/NK_%EB%94%94%EB%82%98%EB%AA%A8_%EC%9E%90%EA%B7%B8%EB%A0%88%EB%B8%8C" title="NK 디나모 자그레브">NK 디나모 자그레브</a> 입단을 타진하였으나 불발되어 <a href="/wiki/2006%EB%85%84" title="2006년">2006년</a> <a href="/wiki/%EB%8C%80%EA%B5%AC_FC" title="대구 FC">대구 FC</a>에 입단하였다. <a href="/wiki/%EB%8C%80%EA%B5%AC_FC" title="대구 FC">대구</a>에서 2시즌간 활약한 후 <a href="/wiki/2007%EB%85%84" title="2007년">2007년</a> <a href="/wiki/10%EC%9B%94_14%EC%9D%BC" title="10월 14일">10월 14일</a> <a href="/wiki/FC_%EC%84%9C%EC%9A%B8" title="FC 서울">FC 서울</a>과의 경기를 끝으로 은퇴하였다.</p>
<p><a href="/wiki/2008%EB%85%84" title="2008년">2008년</a>, <a href="/wiki/%EB%8C%80%EA%B5%AC_FC" title="대구 FC">대구 FC</a> U-18팀인 <a href="/wiki/%ED%98%84%ED%92%8D%EA%B3%A0%EB%93%B1%ED%95%99%EA%B5%90" title="현풍고등학교">현풍고등학교</a> 축구부 감독이 되었다.</p>
