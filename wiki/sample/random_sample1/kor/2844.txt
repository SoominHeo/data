<p><b>김희라</b>(金熙羅, <a href="/wiki/1947%EB%85%84" title="1947년">1947년</a> <a href="/wiki/3%EC%9B%94_23%EC%9D%BC" title="3월 23일">3월 23일</a> ~ )는 <a href="/wiki/%EB%8C%80%ED%95%9C%EB%AF%BC%EA%B5%AD" title="대한민국">대한민국</a>의 <a href="/wiki/%EC%98%81%ED%99%94%EB%B0%B0%EC%9A%B0" title="영화배우">영화배우</a>이다. <a href="/wiki/%EC%98%81%ED%99%94%EB%B0%B0%EC%9A%B0" title="영화배우">영화배우</a> <a href="/wiki/%EA%B9%80%EC%8A%B9%ED%98%B8" title="김승호">김승호</a>(金勝鎬)는 선친이며 <a href="/wiki/%EA%B0%80%EC%88%98" title="가수">가수</a> 겸 <a href="/wiki/%EC%98%81%ED%99%94%EB%B0%B0%EC%9A%B0" title="영화배우">영화배우</a> <a href="/wiki/%EA%B8%88%EC%84%B1_(%EB%B0%B0%EC%9A%B0)" title="금성 (배우)">금성</a>은 둘째아들이다.</p>
<p>그는 <a href="/wiki/1969%EB%85%84" title="1969년">1969년</a> 영화 《<a class="mw-redirect" href="/wiki/%EB%B9%84%EB%82%98%EB%A6%AC%EB%8A%94_%EA%B3%A0%EB%AA%A8%EB%A0%B9" title="비나리는 고모령">비나리는 고모령</a>》으로 영화배우 첫 데뷔하였다.</p>
