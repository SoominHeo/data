<p><b>깃</b>(Git <span class="IPA" title="IPA 발음">/ɡɪt/</span><sup class="reference" id="cite_ref-1"><a href="#cite_note-1">[1]</a></sup>)은 프로그램 등의 소스 코드 관리를 위한 <a class="mw-redirect" href="/wiki/%EB%B6%84%EC%82%B0_%EB%B2%84%EC%A0%84_%EA%B4%80%EB%A6%AC_%EC%8B%9C%EC%8A%A4%ED%85%9C" title="분산 버전 관리 시스템">분산 버전 관리 시스템</a>이다. <a class="new" href="/w/index.php?title=%EA%B8%B0%ED%95%98%ED%95%99%EC%A0%81_%EB%B6%88%EB%B3%80_%EC%9D%B4%EB%A1%A0&amp;action=edit&amp;redlink=1" title="기하학적 불변 이론 (없는 문서)">기하학적 불변 이론</a>을 바탕으로 설계됐고, 빠른 수행 속도에 중점을 두고 있는 것이 특징이다. 최초에는 <a href="/wiki/%EB%A6%AC%EB%88%84%EC%8A%A4_%ED%86%A0%EB%A5%B4%EB%B0%9C%EC%8A%A4" title="리누스 토르발스">리누스 토르발스</a>가 <a href="/wiki/%EB%A6%AC%EB%88%85%EC%8A%A4_%EC%BB%A4%EB%84%90" title="리눅스 커널">리눅스 커널</a> 개발에 이용하려고 개발하였으며, 현재는 다른 곳에도 널리 사용되고 있다.</p>
<p>깃의 작업 폴더는 모두, 전체 기록과 각 기록을 추적할 수 있는 정보를 포함하고 있으며, 완전한 형태의 저장소이다. 네트워크에 접근하거나 중앙 서버에 의존하지 않는다.</p>
<p>현재 주니오 하마노(Junio Hamano)가 <a class="new" href="/w/index.php?title=%EC%86%8C%ED%94%84%ED%8A%B8%EC%9B%A8%EC%96%B4_%EA%B4%80%EB%A6%AC&amp;action=edit&amp;redlink=1" title="소프트웨어 관리 (없는 문서)">소프트웨어 관리</a>를 감독하고 있다. 깃은 <a href="/wiki/GNU_%EC%9D%BC%EB%B0%98_%EA%B3%B5%EC%A4%91_%EC%82%AC%EC%9A%A9_%ED%97%88%EA%B0%80%EC%84%9C" title="GNU 일반 공중 사용 허가서">GNU 일반 공중 사용 허가서</a> v2 하에 배포되는 <a href="/wiki/%EC%9E%90%EC%9C%A0_%EC%86%8C%ED%94%84%ED%8A%B8%EC%9B%A8%EC%96%B4" title="자유 소프트웨어">자유 소프트웨어</a>이다.</p>
