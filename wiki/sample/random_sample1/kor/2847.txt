<p><br/>
<b>깃대종</b>(<span lang="en" xml:lang="en">flagship species</span>)는 <a class="mw-redirect" href="/wiki/%EC%9C%A0%EC%97%94%ED%99%98%EA%B2%BD%EA%B3%84%ED%9A%8D" title="유엔환경계획">유엔환경계획</a>이 만든 개념으로서, 특정 지역의 생태계를 대표할 수 있는 중요 동·식물을 뜻한다. '깃대'라는 단어는 해당 지역 생태계 회복의 개척자적인 이미지를 부여한 상징적 표현이다.<sup class="reference" id="cite_ref-lag_1-0"><a href="#cite_note-lag-1">[1]</a></sup></p>
<h2><span class="mw-headline" id=".EB.8C.80.ED.95.9C.EB.AF.BC.EA.B5.AD">대한민국</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EA%B9%83%EB%8C%80%EC%A2%85&amp;action=edit&amp;section=1" title="부분 편집: 대한민국">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>대표적인 대한민국의 깃대종으로는 <a class="mw-redirect" href="/wiki/%ED%99%8D%EC%B2%9C" title="홍천">홍천</a>의 <a href="/wiki/%EC%97%B4%EB%AA%A9%EC%96%B4" title="열목어">열목어</a>, <a href="/wiki/%EA%B1%B0%EC%A0%9C%EB%8F%84" title="거제도">거제도</a>의 <a href="/wiki/%EA%B3%A0%EB%9E%80%EC%B4%88" title="고란초">고란초</a>. <a href="/wiki/%EB%8D%95%EC%9C%A0%EC%82%B0" title="덕유산">덕유산</a>의 <a class="mw-redirect" href="/wiki/%EB%B0%98%EB%94%A7%EB%B6%88" title="반딧불">반딧불</a>, <a href="/wiki/%ED%83%9C%ED%99%94%EA%B0%95" title="태화강">태화강</a>의 <a href="/wiki/%EA%B0%81%EC%8B%9C%EB%B6%95%EC%96%B4" title="각시붕어">각시붕어</a>, <a class="mw-redirect" href="/wiki/%EB%B6%80%EC%B2%9C" title="부천">부천</a>의 <a class="mw-redirect" href="/wiki/%EB%B3%B5%EC%82%AC%EA%BD%83" title="복사꽃">복사꽃</a>, <a class="mw-redirect" href="/wiki/%EB%B6%80%EC%82%B0" title="부산">부산</a>의 <a class="mw-redirect" href="/wiki/%EB%8F%99%EB%B0%B1%EA%BD%83" title="동백꽃">동백꽃</a>을 들 수 있다.<sup class="reference" id="cite_ref-lag_1-1"><a href="#cite_note-lag-1">[1]</a></sup></p>
