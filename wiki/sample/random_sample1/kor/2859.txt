<p>《<b>꽃보다 男子</b>》(<b>꽃보다 남자</b>)는 <a href="/wiki/%EB%8C%80%ED%95%9C%EB%AF%BC%EA%B5%AD" title="대한민국">대한민국</a>의 제작사인 <a class="mw-redirect" href="/wiki/%EA%B7%B8%EB%A3%B9%EC%97%90%EC%9D%B4%ED%8A%B8" title="그룹에이트">그룹에이트</a>가 제작하여 <a href="/wiki/2009%EB%85%84" title="2009년">2009년</a> <a href="/wiki/1%EC%9B%94_5%EC%9D%BC" title="1월 5일">1월 5일</a>부터 <a href="/wiki/2009%EB%85%84" title="2009년">2009년</a> <a href="/wiki/3%EC%9B%94_31%EC%9D%BC" title="3월 31일">3월 31일</a>까지 <a href="/wiki/%ED%95%9C%EA%B5%AD%EB%B0%A9%EC%86%A1%EA%B3%B5%EC%82%AC" title="한국방송공사">한국방송공사</a> <a href="/wiki/KBS_2TV" title="KBS 2TV">KBS 2TV</a>에서 방영된 드라마이다<sup class="reference" id="cite_ref-4"><a href="#cite_note-4">[4]</a></sup>. 이 드라마는 <a href="/wiki/%EC%B9%B4%EB%AF%B8%EC%98%A4_%EC%9A%94%EC%BD%94" title="카미오 요코">카미오 요코</a>의 만화 《<a href="/wiki/%EA%BD%83%EB%B3%B4%EB%8B%A4_%EB%82%A8%EC%9E%90" title="꽃보다 남자">꽃보다 남자</a>》(<span style="font-size: smaller;"><a href="/wiki/%EC%9D%BC%EB%B3%B8%EC%96%B4" title="일본어">일본어</a>:</span> <span lang="ja" xml:lang="ja">花より男子</span>)를 원작으로 제작되어 <a href="/wiki/%EB%8C%80%ED%95%9C%EB%AF%BC%EA%B5%AD" title="대한민국">대한민국</a>-<a href="/wiki/%EC%9D%BC%EB%B3%B8" title="일본">일본</a>-<a href="/wiki/%EC%A4%91%ED%99%94%EB%AF%BC%EA%B5%AD" title="중화민국">중화민국</a><sup class="reference" id="cite_ref-5"><a href="#cite_note-5">[5]</a></sup> 3개국에서 같은 원작으로 제작된 드라마이다.</p>
<p>드라마 제작 전부터 관심을 끌었던 주요 등장인물인 극중 F4(Flower 4)로 칭해지는 남자 출연진은 <a href="/wiki/%EC%9D%B4%EB%AF%BC%ED%98%B8_(1987%EB%85%84)" title="이민호 (1987년)">이민호</a>(구준표), <a href="/wiki/%EA%B9%80%ED%98%84%EC%A4%91" title="김현중">김현중</a>(윤지후), <a href="/wiki/%EA%B9%80%EB%B2%94" title="김범">김범</a>(소이정), <a href="/wiki/%ED%8B%B0%EB%A7%A5%EC%8A%A4_(%EC%9D%8C%EC%95%85_%EA%B7%B8%EB%A3%B9)" title="티맥스 (음악 그룹)">김준</a>(송우빈)이며, 여자 출연진은 <a href="/wiki/%EA%B5%AC%ED%98%9C%EC%84%A0" title="구혜선">구혜선</a>(금잔디)이다.</p>

<h2><span class="mw-headline" id=".EA.B0.9C.EC.9A.94">개요</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EA%BD%83%EB%B3%B4%EB%8B%A4_%EB%82%A8%EC%9E%90_(%EB%8C%80%ED%95%9C%EB%AF%BC%EA%B5%AD_%EB%93%9C%EB%9D%BC%EB%A7%88)&amp;action=edit&amp;section=1" title="부분 편집: 개요">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<h3><span class="mw-headline" id=".EC.A0.9C.EC.9E.91">제작</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EA%BD%83%EB%B3%B4%EB%8B%A4_%EB%82%A8%EC%9E%90_(%EB%8C%80%ED%95%9C%EB%AF%BC%EA%B5%AD_%EB%93%9C%EB%9D%BC%EB%A7%88)&amp;action=edit&amp;section=2" title="부분 편집: 제작">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>《꽃보다 남자》 대한민국판 드라마 제작권을 놓고 그동안 여러 제작사들이 경합을 벌였으나 원작자인 가미오 요코와 일본의 출판사인 <a href="/wiki/%EC%8A%88%EC%97%90%EC%9D%B4%EC%83%A4" title="슈에이샤">슈에이샤</a>가 《<a href="/wiki/%EA%B6%81S" title="궁S">궁S</a>》의 제작 노하우를 가진 그룹에이트에 가장 높은 점수를 주었다고 제작사 측은 설명하였다<sup class="reference" id="cite_ref-6"><a href="#cite_note-6">[6]</a></sup>.</p>
<p>방송된 1화와 2화는 시작 배경을 제외하고는 원작대로 이야기가 진행되었으나, 제작 초기에 당시 제작사 측에서는 원작자의 의견과 <a class="mw-redirect" href="/wiki/%EA%BD%83%EB%B3%B4%EB%8B%A4_%EB%82%A8%EC%9E%90_(%EC%9D%BC%EB%B3%B8%EC%9D%98_%EB%93%9C%EB%9D%BC%EB%A7%88)" title="꽃보다 남자 (일본의 드라마)">일본의 드라마</a>와 중화민국의 드라마에서 방송되지 못한 에피소드를 중심으로 스토리를 짤 것이라고 밝힌 바 있다<sup class="reference" id="cite_ref-7"><a href="#cite_note-7">[7]</a></sup>. 한편, 25부작으로 제작 예정인 《꽃보다 남자》는 1화부터 12화까지는 주인공들이 고등학생 때의 이야기. 즉, 일본판으로는 《<a class="mw-redirect" href="/wiki/%EA%BD%83%EB%B3%B4%EB%8B%A4_%EB%82%A8%EC%9E%90_(%EC%9D%BC%EB%B3%B8%EC%9D%98_%EB%93%9C%EB%9D%BC%EB%A7%88)" title="꽃보다 남자 (일본의 드라마)">꽃보다 남자 -1</a>》의 이야기를, 13화부터 24화까지는 대학생 후의 이야기. 즉, 일본판으로는 《<a class="mw-redirect" href="/wiki/%EA%BD%83%EB%B3%B4%EB%8B%A4_%EB%82%A8%EC%9E%90_%EB%A6%AC%ED%84%B4%EC%A6%88" title="꽃보다 남자 리턴즈">꽃보다 남자 리턴즈</a>》에 해당하는 부분을 다룰것이라고 밝혔다<sup class="reference" id="cite_ref-8"><a href="#cite_note-8">[8]</a></sup>.</p>

<h2><span class="mw-headline" id=".EC.9D.B4.EB.AA.A8.EC.A0.80.EB.AA.A8">이모저모</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EA%BD%83%EB%B3%B4%EB%8B%A4_%EB%82%A8%EC%9E%90_(%EB%8C%80%ED%95%9C%EB%AF%BC%EA%B5%AD_%EB%93%9C%EB%9D%BC%EB%A7%88)&amp;action=edit&amp;section=13" title="부분 편집: 이모저모">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<ul>
<li><a href="/wiki/%EA%B9%80%ED%98%84%EC%A3%BC_(1977%EB%85%84)" title="김현주 (1977년)">김현주</a>, <a href="/wiki/%ED%95%9C%EC%B1%84%EC%98%81" title="한채영">한채영</a>, <a href="/wiki/%EA%B9%80%EC%A2%85%EC%A7%84_(%EA%B0%80%EC%88%98)" title="김종진 (가수)">김종진</a>, 그룹 <a href="/wiki/SS501" title="SS501">SS501</a>, <a href="/wiki/%EC%A0%95%EC%9D%98%EC%B2%A0" title="정의철">정의철</a>, <a href="/wiki/%EC%9E%84%EC%A3%BC%ED%99%98" title="임주환">임주환</a> 등이 카메오로 출연하였다.<sup class="reference" id="cite_ref-20"><a href="#cite_note-20">[20]</a></sup><sup class="reference" id="cite_ref-21"><a href="#cite_note-21">[21]</a></sup><sup class="reference" id="cite_ref-22"><a href="#cite_note-22">[22]</a></sup></li>
<li>주요 다운로드 사이트에서 드라마 유료 다운로드 순위 1위를 기록했다<sup class="reference" id="cite_ref-23"><a href="#cite_note-23">[23]</a></sup>.</li>
<li>이 드라마의 OST 메인테마곡인 '파라다이스'는 극중 송우빈 역의 <a class="mw-redirect" href="/wiki/%EA%B9%80%EC%A4%80_(%EA%B0%80%EC%88%98)" title="김준 (가수)">김준</a>이 속한이 속한 그룹인 <a href="/wiki/%ED%8B%B0%EB%A7%A5%EC%8A%A4_(%EC%9D%8C%EC%95%85_%EA%B7%B8%EB%A3%B9)" title="티맥스 (음악 그룹)">티맥스</a>가 불렀다.<sup class="reference" id="cite_ref-dailian1_24-0"><a href="#cite_note-dailian1-24">[24]</a></sup>. 또한 OST 중 '내 머리가 나빠서'는 극중 윤지후 역의 김현중이 속한 그룹인 <a href="/wiki/SS501" title="SS501">SS501</a>이 불렀다.<sup class="reference" id="cite_ref-dailian1_24-1"><a href="#cite_note-dailian1-24">[24]</a></sup>. 드라마에 쓰인 OST들은 온라인 음악 차트 사이트에서 상위권 순위를 차지하는 등 호응을 얻었다<sup class="reference" id="cite_ref-25"><a href="#cite_note-25">[25]</a></sup>.</li>
<li><a href="/wiki/%EB%AC%B8%ED%99%94%EB%B0%A9%EC%86%A1" title="문화방송">문화방송</a>의 예능 프로그램 《<a href="/wiki/%EB%AC%B4%ED%95%9C%EB%8F%84%EC%A0%84" title="무한도전">무한도전</a>》에서 '꽃보다 남자'를 패러디 하는 등<sup class="reference" id="cite_ref-26"><a href="#cite_note-26">[26]</a></sup><sup class="reference" id="cite_ref-27"><a href="#cite_note-27">[27]</a></sup><sup class="reference" id="cite_ref-28"><a href="#cite_note-28">[28]</a></sup>, 여러 예능 프로그램들에서 '꽃보다 남자'를 패러디 하였다<sup class="reference" id="cite_ref-29"><a href="#cite_note-29">[29]</a></sup>.</li>
<li>드라마의 배경 장소인 '신화고등학교'의 촬영지는 <a href="/wiki/%EB%8C%80%EA%B5%AC%EA%B4%91%EC%97%AD%EC%8B%9C" title="대구광역시">대구광역시</a>에 위치한 <a href="/wiki/%EA%B3%84%EB%AA%85%EB%8C%80%ED%95%99%EA%B5%90" title="계명대학교">계명대학교</a> 성서 캠퍼스이다<sup class="reference" id="cite_ref-30"><a href="#cite_note-30">[30]</a></sup>. '신화고등학교'의 내부는 <a href="/wiki/%EA%B2%BD%EA%B8%B0%EB%8F%84" title="경기도">경기도</a> <a class="mw-disambig" href="/wiki/%EC%96%91%ED%8F%89" title="양평">양평</a>에 있는 영어 마을에서 촬영하였다.<sup class="reference" id="cite_ref-31"><a href="#cite_note-31">[31]</a></sup>.</li>
<li>드라마에서 헬기 장면과 오리 장면 등 엉성한 CG 처리에 대한 비판이 있었다.<sup class="reference" id="cite_ref-32"><a href="#cite_note-32">[32]</a></sup>. 이에 대해 제작진은 제작 시간이 부족했다고 해명하며, 디테일하게 보강하지 못한 아쉬움이 있다고 밝혔다<sup class="reference" id="cite_ref-yonhap1_33-0"><a href="#cite_note-yonhap1-33">[33]</a></sup>.</li>
<li>극중 청소년 나이에 나이트 클럽 출입과 음주, 호텔방, 몰래카메라, 해외 장기 수학 여행 등의 설정으로 인하여 비판적인 의견이 있었으며<sup class="reference" id="cite_ref-34"><a href="#cite_note-34">[34]</a></sup> 이러한 행위는 대한민국의 현행 법률들을 위반한 행위라는 비판이 있었다<sup class="reference" id="cite_ref-35"><a href="#cite_note-35">[35]</a></sup><sup class="reference" id="cite_ref-36"><a href="#cite_note-36">[36]</a></sup>. <a href="/wiki/2009%EB%85%84" title="2009년">2009년</a> <a href="/wiki/2%EC%9B%94_25%EC%9D%BC" title="2월 25일">2월 25일</a>에 <a href="/wiki/KBS_1TV" title="KBS 1TV">KBS 1TV</a>에서 방송된 《여성공감》이라는 프로그램에서는 꽃보다 남자에서 나오는 주인공들의 행동이 각각 어떤 법의 적용을 받는지를 논의하는 내용이 방송되기도 하였다<sup class="reference" id="cite_ref-37"><a href="#cite_note-37">[37]</a></sup>. 이러한 비판에 대해 《꽃보다 남자》 제작진은 일본의 원작에 비해 수위 조절을 한 것이라고 밝혔으나 이른바 '막장 드라마' 2위에 선정되는 등 비판적 여론이 있었다<sup class="reference" id="cite_ref-yonhap1_33-1"><a href="#cite_note-yonhap1-33">[33]</a></sup>. 이런 논란에 대해 드라마 특성인 “판타지 드라마”라는 특징을 배제한 것이라며 막장 논란이 무의미하다는 의견도 있었다<sup class="reference" id="cite_ref-38"><a href="#cite_note-38">[38]</a></sup><sup class="reference" id="cite_ref-39"><a href="#cite_note-39">[39]</a></sup>.</li>
<li>외모 지상주의, 재벌 우상화, <a class="mw-redirect" href="/wiki/%EC%99%95%EB%94%B0" title="왕따">왕따</a> 조장, 폭력성 등에 대한 비판과<sup class="reference" id="cite_ref-40"><a href="#cite_note-40">[40]</a></sup><sup class="reference" id="cite_ref-41"><a href="#cite_note-41">[41]</a></sup><sup class="reference" id="cite_ref-42"><a href="#cite_note-42">[42]</a></sup> 납치, 감금 등의 극단적인 설정에 대해 선정성 논란이 있었다<sup class="reference" id="cite_ref-43"><a href="#cite_note-43">[43]</a></sup>. 이에 대해 <a href="/wiki/%EB%8C%80%ED%95%9C%EB%AF%BC%EA%B5%AD" title="대한민국">대한민국</a> <a href="/wiki/%EB%8C%80%ED%95%9C%EB%AF%BC%EA%B5%AD_%EB%B0%A9%EC%86%A1%ED%86%B5%EC%8B%A0%EC%8B%AC%EC%9D%98%EC%9C%84%EC%9B%90%ED%9A%8C" title="대한민국 방송통신심의위원회">방송통신심의위원회</a>는 <a href="/wiki/2009%EB%85%84" title="2009년">2009년</a> <a href="/wiki/3%EC%9B%94_1%EC%9D%BC" title="3월 1일">3월 1일</a>에 '1월 시청자 동향 분석'을 발표하며 "왕따 조장, 학교폭력, 성희롱, 인신모욕, 계층 간 위화감 조성, 물질만능주의 및 사치 조장, 간접광고 등의 내용이 청소년에게 악영향을 끼칠 수 있다는 우려의 민원이 20여 건 접수됐다"며 "당사자 의견진술 후 제재 수위 등을 결정할 것"이라고 밝히며 '꽃보다 남자'에 대한 제재를 가할 의도를 밝혔다<sup class="reference" id="cite_ref-44"><a href="#cite_note-44">[44]</a></sup>. 결국 <a href="/wiki/3%EC%9B%94_18%EC%9D%BC" title="3월 18일">3월 18일</a>에 방송통신심의위원회로부터 '경고' 처분을 받았다<sup class="reference" id="cite_ref-45"><a href="#cite_note-45">[45]</a></sup>.</li>
<li>극중에서 F4로 칭해지는 남자 출연진들이 접촉사고 등의 교통사고를 당했다<sup class="reference" id="cite_ref-isplus1_46-0"><a href="#cite_note-isplus1-46">[46]</a></sup><sup class="reference" id="cite_ref-47"><a href="#cite_note-47">[47]</a></sup><sup class="reference" id="cite_ref-donga1_48-0"><a href="#cite_note-donga1-48">[48]</a></sup>. 또한 극중 여자 주인공인 <a href="/wiki/%EA%B5%AC%ED%98%9C%EC%84%A0" title="구혜선">구혜선</a>도 촬영중 부상을 당했다<sup class="reference" id="cite_ref-49"><a href="#cite_note-49">[49]</a></sup>. 이로 인해 꽃보다 남자 제17회를 정상적으로 방영하지 못하게 되었고, 이날 방영분은 'F4 스페셜 토크쇼'로 대체되었다.<sup class="reference" id="cite_ref-50"><a href="#cite_note-50">[50]</a></sup> 사고가 빈발하는 이유는 지나친 촬영 일정 때문에 출연진들의 피로가 누적되었기 때문이라는 지적이 있다<sup class="reference" id="cite_ref-isplus1_46-1"><a href="#cite_note-isplus1-46">[46]</a></sup><sup class="reference" id="cite_ref-donga1_48-1"><a href="#cite_note-donga1-48">[48]</a></sup>.</li>
<li>조연 출연자 중 한명인 <a href="/wiki/%EC%9E%A5%EC%9E%90%EC%97%B0" title="장자연">장자연</a>이 자살하는 사건이 발생하여 사회적으로 큰 파장이 일어났다<sup class="reference" id="cite_ref-51"><a href="#cite_note-51">[51]</a></sup>.</li>
</ul>
