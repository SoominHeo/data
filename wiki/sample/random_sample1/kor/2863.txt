<p><b>꾸란</b>(<span style="font-size: smaller;"><a href="/wiki/%EC%95%84%EB%9E%8D%EC%96%B4" title="아랍어">아랍어</a>:</span> <span lang="ar" xml:lang="ar">أَلْقُرآن</span>)또는 <b>쿠란</b>, <b>코란</b>은 예언자 <a href="/wiki/%EB%AC%B4%ED%95%A8%EB%A7%88%EB%93%9C" title="무함마드">무함마드</a>가 <a href="/wiki/610%EB%85%84" title="610년">610년</a> 이후 23년간 알라(<a href="/wiki/%EC%95%8C%EB%9D%BC" title="알라">알라</a>는 '하나님'이라는 의미이다.)에게 받은 계시를 구전으로 전하다가 그의 가르침을 받은 제자들이 여러 장소에서 여러 시대를 걸쳐 기록한 기록물들을 모아서 집대성한 <a href="/wiki/%EC%B1%85" title="책">책</a>으로, 이 계시는 무함마드가 40세 경 현재 <a href="/wiki/%EC%82%AC%EC%9A%B0%EB%94%94%EC%95%84%EB%9D%BC%EB%B9%84%EC%95%84" title="사우디아라비아">사우디아라비아</a>에 있는 히라산 동굴에서 천사 <a href="/wiki/%EA%B0%80%EB%B8%8C%EB%A6%AC%EC%97%98" title="가브리엘">지브릴</a>을 통해 처음 받았다.</p>
<p>꾸란은 까라아(َقَرَأ)(읽다)라는 동사에서 파생된 명사로 '낭송'이라는 뜻을 지닌다. '파알라'(َفَعَل)라는 동사에서 '푸을란'(فُعْلَان)이라는 명사로 변형된 형태이다. 꾸란은 계시되어 가르쳐지면, 제자들이 그것을 낙타의 골편이나 야자의 엽피, 양피지 등에 기록하였으며, (무함마드가 죽자마자) 이후 제1대 <a class="mw-redirect" href="/wiki/%EC%B9%BC%EB%A6%AC%ED%94%84" title="칼리프">칼리프</a> <a class="mw-redirect" href="/wiki/%EC%95%84%EB%B6%80_%EB%B0%94%ED%81%AC%EB%A5%B4" title="아부 바크르">아부 바크르</a> 시절에 꾸란을 한 권으로 집대성해 보관하였고 , 제3대 <a class="mw-redirect" href="/wiki/%EC%B9%BC%EB%A6%AC%ED%94%84" title="칼리프">칼리프</a> <a href="/wiki/%EC%9A%B0%EC%8A%A4%EB%A7%8C_%EC%9D%B4%EB%B8%90_%EC%95%84%ED%8C%90" title="우스만 이븐 아판">우스만 이븐 아판</a>은 꾸란 결집위원회를 조직하여 최종적으로 집대성 작업을 완성했다. 이 과정에서 꾸라이쉬 부족의 언어로 그 기재 방법을 통일하였으며, 정통본을 암송자인 <a href="/wiki/%ED%95%98%ED%94%BC%EC%A6%88" title="하피즈">하피즈</a>와 함께 이슬람 각지로 파견하여 전파하도록 했으며, 이것이 현재 사용되고 있는 꾸란의 정본(定本)이다. 또한 이것을 달리 '이맘본' 또는 '오스만본'이라 부르기도 한다.</p>
<p>오늘날 쓰이는 꾸란은 이 정본을 원본으로 삼고 있다. 9세기 말부터 모음 기호(타쉬킬)가 붙어 표기되기 시작했으며. 오늘날 쓰이는 꾸란은 최초의 정본에서 타쉬킬(모음기호)이 추가된 형태이다(문법에 따라, 운율에 따른 읽는 법에 따를 수 있으나 그 내용은 같다). <a href="/wiki/%EC%82%AC%EC%9A%B0%EB%94%94%EC%95%84%EB%9D%BC%EB%B9%84%EC%95%84%EC%9D%98_%EA%B5%AD%EA%B8%B0" title="사우디아라비아의 국기">사우디아라비아의 국기</a>는 꾸란의 구절로서 <a href="/wiki/%EC%9D%B4%EC%8A%AC%EB%9E%8C%EC%9D%98_%EB%8B%A4%EC%84%AF_%EA%B8%B0%EB%91%A5" title="이슬람의 다섯 기둥">이슬람의 다섯 기둥</a> 중의 하나인 <a href="/wiki/%EC%83%A4%ED%95%98%EB%8B%A4" title="샤하다">샤하다</a>(하나님 이외에는 경배받을 존재가 없으며, <a href="/wiki/%EB%AC%B4%ED%95%A8%EB%A7%88%EB%93%9C" title="무함마드">무함마드</a>는 하나님의 사도이다)가 새겨져 있기 때문에 <a href="/wiki/%EC%A1%B0%EA%B8%B0_(%EA%B8%B0%EB%A1%80)" title="조기 (기례)">조기</a>를 게양하지 않는다는 특징이 있다.</p>

<h2><span class="mw-headline" id=".EA.B5.AC.EC.84.B1">구성</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EA%BE%B8%EB%9E%80&amp;action=edit&amp;section=1" title="부분 편집: 구성">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>문체는 사즈아이고 일종의 운을 단 산문체로 되었고, 독송 할 때 리듬감을 느낀다. 6342개의 아야(절)와 114의 수라(장)로 되어 있으며 각 수라는 암소·이므란일가·여자등과 같은 제목으로 분류된다. 꾸란에서 사용되는 단어는 점 하나까지도 리듬과 운율로 연결되어 소홀히 할 수 없다.</p>
<h2><span class="mw-headline" id=".EC.A3.BC.EC.84.9D.EC.84.9C_.28.ED.83.80.ED.94.84.EC.94.A8.EB.A5.B4.29">주석서 (타프씨르)</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EA%BE%B8%EB%9E%80&amp;action=edit&amp;section=2" title="부분 편집: 주석서 (타프씨르)">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>꾸란의 주석서는 타프시르(Tafsir)로 불리며 어근은 아랍어 fassara(설명하다)이다. 꾸란 구절을 설명할 때 가장 먼저 사용되는 것은 먼저 꾸란 그 자체의 다른 구절이며, 두 번째로는 <a href="/wiki/%ED%95%98%EB%94%94%EC%8A%A4" title="하디스">하디스</a>(무함마드의 언행록)이며, 마지막으로는 <a href="/wiki/%EC%82%AC%ED%95%98%EB%B0%94" title="사하바">사하바</a>(무함마드의 교우)의 전승이다.</p>
<h2><span class="mw-headline" id=".EB.8F.85.EC.86.A1">독송</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EA%BE%B8%EB%9E%80&amp;action=edit&amp;section=3" title="부분 편집: 독송">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>하루 5번 <a href="/wiki/%EA%B8%B0%EB%8F%84" title="기도">기도</a> <sup class="reference" id="cite_ref-1"><a href="#cite_note-1">[1]</a></sup>시 <a href="/wiki/%EB%AC%B4%EC%8A%AC%EB%A6%BC" title="무슬림">무슬림</a>은 정확한 발음으로 꾸란을 암송하여야 하며 꾸란 전체를 암기하는 자를 <a href="/wiki/%ED%95%98%ED%94%BC%EC%A6%88" title="하피즈">하피즈</a>(Hafiz)라고 한다. 각종 의식에서는 정규 독송자가 소리 높이 독송한다. 10여 개의 유파가 각기 다른 정지나 성조로 독송했으며 10세기경 이후는 7파로 통일되어 부르고 있다. 모든 예배 시에 <a class="mw-redirect" href="/wiki/%EC%95%8C_%ED%8C%8C%ED%8B%B0%ED%95%98" title="알 파티하">알 파티하</a>는 반드시 독송하며, 하루 다섯 차례 기도를 올리는 동안 최소한 17회 이상 개경장을 암송하게 된다.</p>
<h2><span class="mw-headline" id=".EB.B2.88.EC.97.AD">번역</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EA%BE%B8%EB%9E%80&amp;action=edit&amp;section=4" title="부분 편집: 번역">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>알라가 계시한 꾸란은 문학적 아랍어로 되어있으며 <a href="/wiki/%EC%95%84%EB%9E%8D%EC%96%B4" title="아랍어">아랍어</a> 자체로써 꾸란을 이해할 때만이 그 의미에 가장 정확히 접근할 수 있다고 한다. 원뜻을 훼손하고 왜곡할 수 있다는 우려로 인하여, 꾸란은 타언어로 번역된 꾸란을 인정하지 않는다. 비아랍어를 사용하는 독자, 예를 들어 한국인을 위해 한국어로 번역된 꾸란은 ‘한국어 꾸란’이 아닌 ‘꾸란 의미의 한국어 해설’로 불린다.</p>
<p><a class="mw-redirect" href="/wiki/%EC%8B%AD%EC%9E%90%EA%B5%B0_%EC%A0%84%EC%9F%81" title="십자군 전쟁">십자군 전쟁</a> 이후 <a href="/wiki/%EC%9C%A0%EB%9F%BD" title="유럽">유럽</a> <a href="/wiki/%EA%B8%B0%EB%8F%85%EA%B5%90%EC%9D%B8" title="기독교인">기독교인</a>들이 이슬람에 관심을 가져 12세기에 꾸란이 <a href="/wiki/%EB%9D%BC%ED%8B%B4%EC%96%B4" title="라틴어">라틴어</a>로 번역되었고, 이것이 다시 <a href="/wiki/%EB%8F%85%EC%9D%BC%EC%96%B4" title="독일어">독일어</a>·<a href="/wiki/%EC%9D%B4%ED%83%88%EB%A6%AC%EC%95%84%EC%96%B4" title="이탈리아어">이탈리아어</a>·<a href="/wiki/%EB%84%A4%EB%8D%9C%EB%9E%80%EB%93%9C%EC%96%B4" title="네덜란드어">네덜란드어</a> 등으로 중역되었다. <a href="/wiki/%ED%95%9C%EA%B5%AD%EC%96%B4" title="한국어">한국어</a>로는 <a href="/wiki/1980%EB%85%84" title="1980년">1980년</a> 김용선 교수가 펴낸 《한역주해 코란 역편》이 있으며, <a href="/wiki/%EC%82%AC%EC%9A%B0%EB%94%94%EC%95%84%EB%9D%BC%EB%B9%84%EC%95%84" title="사우디아라비아">사우디아라비아</a>의 지원을 받아 최영길 교수에 의해 이루어진 《성 꾸란 의미의 한국어 번역》이 번역본으로 나와 있으며, 이후 한국어로 완역하고 주석과 해석을 첨가한 《꾸란 주해》를 다시 펴내었다.</p>
<h2><span class="mw-headline" id=".EA.BE.B8.EB.9E.80.EC.9D.98_.EC.9E.A5_.28.EC.88.98.EB.9D.BC.29">꾸란의 장 (수라)</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EA%BE%B8%EB%9E%80&amp;action=edit&amp;section=5" title="부분 편집: 꾸란의 장 (수라)">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<div class="rellink relarticle mainarticle"><img alt="" data-file-height="128" data-file-width="128" height="16" src="//upload.wikimedia.org/wikipedia/commons/thumb/e/ec/Crystal_Clear_app_xmag.svg/16px-Crystal_Clear_app_xmag.svg.png" srcset="//upload.wikimedia.org/wikipedia/commons/thumb/e/ec/Crystal_Clear_app_xmag.svg/24px-Crystal_Clear_app_xmag.svg.png 1.5x, //upload.wikimedia.org/wikipedia/commons/thumb/e/ec/Crystal_Clear_app_xmag.svg/32px-Crystal_Clear_app_xmag.svg.png 2x" width="16"/> 이 부분의 본문은 <a href="/wiki/%EC%88%98%EB%9D%BC" title="수라">수라</a>입니다.</div>
<p>여기서의 수라(<span style="font-size: smaller;"><a href="/wiki/%EC%95%84%EB%9E%8D%EC%96%B4" title="아랍어">아랍어</a>:</span> <span lang="ar" xml:lang="ar">سورة</span>)는 "<a class="mw-redirect" href="/wiki/%EA%BE%B8%EB%A5%B4%EC%95%88" title="꾸르안">꾸르안</a>의 장"에 대한 <a href="/wiki/%EC%95%84%EB%9E%8D%EC%96%B4" title="아랍어">아랍어</a> 표기이다. 이는 보통 <a class="new" href="/w/index.php?title=%EB%A9%94%EB%94%94%EB%82%98_%EC%88%98%EB%9D%BC&amp;action=edit&amp;redlink=1" title="메디나 수라 (없는 문서)">메디나 수라</a>와 <a class="new" href="/w/index.php?title=%EB%A9%94%EC%B9%B4_%EC%88%98%EB%9D%BC&amp;action=edit&amp;redlink=1" title="메카 수라 (없는 문서)">메카 수라</a>로 구분되나 그 구분은 확실하지 않다. 모든 수라는 <a class="new" href="/w/index.php?title=%EC%95%84%EC%95%BC%ED%8A%B8&amp;action=edit&amp;redlink=1" title="아야트 (없는 문서)">아야트</a>, 즉 절로 나뉜다.</p>
<p><a href="/wiki/114" title="114">114</a>개의 수라는 다음과 같다. 이때 "알"은 정관사이고, 그에 따라, 예를 들면 "알 진"은 "정령 장"이라는 뜻을 가진다.</p>
