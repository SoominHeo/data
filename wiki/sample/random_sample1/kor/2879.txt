<h2><span class="mw-headline" id=".EC.84.A0.EC.88.98_.EA.B2.BD.EB.A0.A5">선수 경력</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%82%98%EA%B0%80%ED%86%A0%EB%AA%A8_%EC%9C%A0%ED%86%A0&amp;action=edit&amp;section=1" title="부분 편집: 선수 경력">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<h3><span class="mw-headline" id=".ED.81.B4.EB.9F.BD">클럽</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%82%98%EA%B0%80%ED%86%A0%EB%AA%A8_%EC%9C%A0%ED%86%A0&amp;action=edit&amp;section=2" title="부분 편집: 클럽">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p><a href="/wiki/2007%EB%85%84" title="2007년">2007년</a> 유스 특별지정선수로 <a href="/wiki/J%EB%A6%AC%EA%B7%B8%EC%BB%B5" title="J리그컵">J리그컵</a>에 한 경기 출전을 하였으며 이 후 <a href="/wiki/2008%EB%85%84" title="2008년">2008년</a> <a href="/wiki/J1%EB%A6%AC%EA%B7%B8" title="J1리그">J1리그</a>의 <a href="/wiki/FC_%EB%8F%84%EC%BF%84" title="FC 도쿄">FC 도쿄</a>에 공식으로 입단하였다. <a href="/wiki/2008%EB%85%84" title="2008년">2008년</a>부터 주전으로 중용받기 시작하여 <a href="/wiki/2008%EB%85%84" title="2008년">2008년</a> 정규 리그 29경기에 출전하였고, <a href="/wiki/2009%EB%85%84" title="2009년">2009년</a>엔 31경기에 출전하였다.</p>
<p><a href="/wiki/2010%EB%85%84_FIFA_%EC%9B%94%EB%93%9C%EC%BB%B5" title="2010년 FIFA 월드컵">2010년 FIFA 월드컵</a>에 출전한 이후 <a href="/wiki/%EC%84%B8%EB%A6%AC%EC%97%90_A" title="세리에 A">세리에 A</a> <a href="/wiki/AC_%EC%B2%B4%EC%84%B8%EB%82%98" title="AC 체세나">AC 체세나</a>로 임대 이적하였고, <a href="/wiki/2011%EB%85%84_AFC_%EC%95%84%EC%8B%9C%EC%95%88%EC%BB%B5" title="2011년 AFC 아시안컵">2011년 AFC 아시안컵</a> 이후 완전 이적했다.</p>
<p><a href="/wiki/2011%EB%85%84" title="2011년">2011년</a> <a href="/wiki/2%EC%9B%94_1%EC%9D%BC" title="2월 1일">2월 1일</a> 겨울 이적시장에서 <a href="/wiki/FC_%EC%9D%B8%ED%85%8C%EB%A5%B4%EB%82%98%EC%B9%98%EC%98%A4%EB%82%A0%EB%A0%88_%EB%B0%80%EB%9D%BC%EB%85%B8" title="FC 인테르나치오날레 밀라노">FC 인테르나치오날레 밀라노</a>로 임대로 이적함으로써 인테르에서 뛴 최초의 아시아 선수가 되었다.</p>
<h3><span class="mw-headline" id=".EA.B5.AD.EA.B0.80.EB.8C.80.ED.91.9C">국가대표</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%82%98%EA%B0%80%ED%86%A0%EB%AA%A8_%EC%9C%A0%ED%86%A0&amp;action=edit&amp;section=3" title="부분 편집: 국가대표">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p><a href="/wiki/2008%EB%85%84_%ED%95%98%EA%B3%84_%EC%98%AC%EB%A6%BC%ED%94%BD" title="2008년 하계 올림픽">2008년 하계 올림픽</a>에 <a href="/wiki/%EC%9D%BC%EB%B3%B8_%EC%B6%95%EA%B5%AC_%EA%B5%AD%EA%B0%80%EB%8C%80%ED%91%9C%ED%8C%80" title="일본 축구 국가대표팀">일본 올림픽 축구 국가대표팀</a> 멤버로 출전하였다.</p>
<p><a href="/wiki/2008%EB%85%84" title="2008년">2008년</a> <a href="/wiki/5%EC%9B%94_24%EC%9D%BC" title="5월 24일">5월 24일</a>, <a href="/wiki/%EC%BD%94%ED%8A%B8%EB%94%94%EB%B6%80%EC%95%84%EB%A5%B4_%EC%B6%95%EA%B5%AC_%EA%B5%AD%EA%B0%80%EB%8C%80%ED%91%9C%ED%8C%80" title="코트디부아르 축구 국가대표팀">코트디부아르</a>와의 친선 경기에 출전하며 <a href="/wiki/A%EB%A7%A4%EC%B9%98" title="A매치">A매치</a> 데뷔전을 치렀다.</p>
