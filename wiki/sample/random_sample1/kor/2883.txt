<p><b>구루 나나크</b>(<span style="font-size: smaller;"><a href="/wiki/%ED%8E%80%EC%9E%90%EB%B8%8C%EC%96%B4" title="펀자브어">펀자브어</a>:</span> <span lang="pa" xml:lang="pa">ਗੁਰੂ ਨਾਨਕ ਦੇਵ</span>, <span style="font-size: smaller;"><a href="/wiki/%ED%9E%8C%EB%94%94%EC%96%B4" title="힌디어">힌디어</a>:</span> <span lang="hi" xml:lang="hi">गुरु नानक</span>, <span style="font-size: smaller;"><a href="/wiki/%EC%9A%B0%EB%A5%B4%EB%91%90%EC%96%B4" title="우르두어">우르두어</a>:</span> <span lang="ur" xml:lang="ur">گرونانک</span> <i>Gurū Nānak</i>, <a href="/wiki/1469%EB%85%84" title="1469년">1469년</a> <a href="/wiki/4%EC%9B%94_15%EC%9D%BC" title="4월 15일">4월 15일</a> ~ <a href="/wiki/1539%EB%85%84" title="1539년">1539년</a> <a href="/wiki/9%EC%9B%94_22%EC%9D%BC" title="9월 22일">9월 22일</a>)는 <a href="/wiki/%EC%9D%B8%EB%8F%84" title="인도">인도</a>의 종교가이자 <a href="/wiki/%EC%8B%9C%ED%81%AC%EA%B5%90" title="시크교">시크교</a>의 창시자이다. <a href="/wiki/1469%EB%85%84" title="1469년">1469년</a> <a href="/wiki/%ED%8E%80%EC%9E%90%EB%B8%8C" title="펀자브">펀자브</a> 지방 <a href="/wiki/%EB%9D%BC%ED%98%B8%EB%A5%B4" title="라호르">라호르</a> 근교(현 <a href="/wiki/%ED%8C%8C%ED%82%A4%EC%8A%A4%ED%83%84" title="파키스탄">파키스탄</a>)에서 태어났다. <a class="mw-redirect" href="/wiki/%EC%B9%B4%EC%8A%A4%ED%8A%B8_%EC%A0%9C%EB%8F%84" title="카스트 제도">카스트 제도</a>를 반대하였고 <a href="/wiki/%EC%9D%B4%EC%8A%AC%EB%9E%8C%EA%B5%90" title="이슬람교">이슬람교</a>의 영향을 받아 <a href="/wiki/%ED%9E%8C%EB%91%90%EA%B5%90" title="힌두교">힌두교</a>의 개혁을 시도한 <a href="/wiki/%EC%8B%9C%ED%81%AC%EA%B5%90" title="시크교">시크교</a>를 창시하였다. <a href="/wiki/%EC%8B%9C%ED%81%AC%EA%B5%90" title="시크교">시크교</a>의 10명의 <a class="mw-disambig" href="/wiki/%EA%B5%AC%EB%A3%A8" title="구루">구루</a> 중 첫 번째 구루이다.</p>
<p>신이 유일 영원한 존재이며 각종 종교에서는 각각 다르게 말하지만 신은 모두 동일한 것으로 계급과 종족의 차별없이 접근할 수 있다고 주장하였다. 또 죄를 지으면 그 후세에 응보를 받는다는 <a href="/wiki/%EC%9D%B8%EA%B3%BC%EC%9D%91%EB%B3%B4" title="인과응보">인과응보</a>, 업과 <a href="/wiki/%EC%9C%A4%ED%9A%8C" title="윤회">윤회</a>의 사상을 가르쳤다. 또 <a href="/wiki/%EC%9A%B0%EC%83%81%EC%88%AD%EB%B0%B0" title="우상숭배">우상숭배</a>와 <a class="new" href="/w/index.php?title=%EA%B3%A0%ED%96%89&amp;action=edit&amp;redlink=1" title="고행 (없는 문서)">고행</a>을 반대하고 묵상으로 신을 섬길 것을 역설하였다. 시크교는 인도의 <a class="mw-redirect" href="/wiki/%ED%8E%80%EC%9E%A1" title="펀잡">펀잡</a> 지방에 널리 퍼졌다.<sup class="reference" id="cite_ref-1"><a href="#cite_note-1">[1]</a></sup></p>
