<p><b>나르메르</b>는 <a href="/wiki/%EC%9D%B4%EC%A7%91%ED%8A%B8_%EC%A0%9C1%EC%99%95%EC%A1%B0" title="이집트 제1왕조">이집트 제1왕조</a>의 초대 <a href="/wiki/%ED%8C%8C%EB%9D%BC%EC%98%A4" title="파라오">파라오</a>이며, <a class="new" href="/w/index.php?title=%ED%98%B8%EB%A3%A8%EC%8A%A4_%EC%9D%B4%EB%A6%84&amp;action=edit&amp;redlink=1" title="호루스 이름 (없는 문서)">호루스 이름</a>으로 "공격하는 메기"라는 뜻이다. <a href="/wiki/%EC%83%81%EC%9D%B4%EC%A7%91%ED%8A%B8" title="상이집트">상</a>·<a href="/wiki/%ED%95%98%EC%9D%B4%EC%A7%91%ED%8A%B8" title="하이집트">하이집트</a>를 통일한 첫 번째 파라오로 알려져 있다.</p>
<h2><span class="mw-headline" id=".EB.82.98.EB.A5.B4.EB.A9.94.EB.A5.B4.EC.9D.98_.EC.9C.A0.EC.A0.81">나르메르의 유적</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%82%98%EB%A5%B4%EB%A9%94%EB%A5%B4&amp;action=edit&amp;section=1" title="부분 편집: 나르메르의 유적">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>나르메르와 관련된 중요한 유적은 바로 <a href="/wiki/%EB%82%98%EB%A5%B4%EB%A9%94%EB%A5%B4_%ED%8C%94%EB%A0%88%ED%8A%B8" title="나르메르 팔레트">나르메르 팔레트</a>이다. 이 짙은 녹색 팔레트 조각은 <a href="/wiki/%EA%B3%A0%EB%8C%80_%EC%9D%B4%EC%A7%91%ED%8A%B8" title="고대 이집트">고대 이집트</a>의 기념물 중 역사적 기록이 담긴 최초의 기념물이다.</p>
<p>첫 번째 팔레트에서 나르메르는 <a href="/wiki/%EC%83%81%EC%9D%B4%EC%A7%91%ED%8A%B8" title="상이집트">상이집트</a>의 상징인 <a class="new" href="/w/index.php?title=%EB%B0%B1%EA%B4%80&amp;action=edit&amp;redlink=1" title="백관 (없는 문서)">백관</a>(<a class="new" href="/w/index.php?title=%ED%97%A4%EC%A0%9C%ED%8A%B8&amp;action=edit&amp;redlink=1" title="헤제트 (없는 문서)">헤제트</a>)을 쓴 채 상대방의 앞머리를 잡고 몽둥이로 내려 치려는 듯한 모습으로 등장한다.(여기에서 상대방은 <a href="/wiki/%ED%95%98%EC%9D%B4%EC%A7%91%ED%8A%B8" title="하이집트">하이집트</a>의 도시국가인 <a class="new" href="/w/index.php?title=%EB%82%98%EC%B9%B4%EB%8B%A4&amp;action=edit&amp;redlink=1" title="나카다 (없는 문서)">나카다</a>의 추장으로 예상된다.) 이 장면은 서양 역사에서 왕의 정복장면으로 반복되어 나타난다. 두 번째 팔레트에서는 <a href="/wiki/%ED%95%98%EC%9D%B4%EC%A7%91%ED%8A%B8" title="하이집트">하이집트</a>의 상징인 <a class="new" href="/w/index.php?title=%EC%A0%81%EA%B4%80&amp;action=edit&amp;redlink=1" title="적관 (없는 문서)">적관</a>(<a class="new" href="/w/index.php?title=%EB%8D%B0%EC%8A%88%EB%A0%88%ED%8A%B8&amp;action=edit&amp;redlink=1" title="데슈레트 (없는 문서)">데슈레트</a>)을 쓴 채로 병사들의 사열을 지켜보는 모습으로 등장한다. 이러한 모습은 그가 양 쪽의 왕권을 모두 가지고 있었음을 나타낸다.</p>
<p>또다른 유적인 <a class="new" href="/w/index.php?title=%EB%82%98%EB%A5%B4%EB%A9%94%EB%A5%B4%EC%9D%98_%EC%A7%80%ED%8C%A1%EC%9D%B4_%EB%A8%B8%EB%A6%AC&amp;action=edit&amp;redlink=1" title="나르메르의 지팡이 머리 (없는 문서)">나르메르의 지팡이 머리</a>(Narmer Macehead)에는 파라오의 왕위 갱신제인 <a class="new" href="/w/index.php?title=%ED%97%A4%EB%B8%8C-%EC%84%B8%EB%93%9C&amp;action=edit&amp;redlink=1" title="헤브-세드 (없는 문서)">헤브-세드</a>를 기념하는 내용이 담겨 있다. 헤브-세드가 대체적으로 왕이 된 지 30년이 된 것을 기념하는 것으로 보아, 나르메르는 30년 이상을 통치한 것으로 보인다.</p>
<h2><span class="mw-headline" id=".EA.B8.B0_.EB.A1.9D">기 록</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%82%98%EB%A5%B4%EB%A9%94%EB%A5%B4&amp;action=edit&amp;section=2" title="부분 편집: 기 록">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p><a class="new" href="/w/index.php?title=%EB%A7%88%EB%84%A4%ED%86%A0%EC%9D%98_%EA%B8%B0%EB%A1%9D&amp;action=edit&amp;redlink=1" title="마네토의 기록 (없는 문서)">마네토의 기록</a>에 의하면 나르메르는 <a href="/wiki/%EB%A9%94%EB%84%A4%EC%8A%A4" title="메네스">메네스</a>와 같은 인물로 여겨질 수도 있지만, <a href="/wiki/%EA%B3%A0%EA%B3%A0%ED%95%99" title="고고학">고고학</a>적 증거는 그가 <a href="/wiki/%EC%83%81%EC%9D%B4%EC%A7%91%ED%8A%B8" title="상이집트">상</a>·<a href="/wiki/%ED%95%98%EC%9D%B4%EC%A7%91%ED%8A%B8" title="하이집트">하이집트</a>의 통일자이자, <a href="/wiki/%ED%98%B8%EB%A5%B4-%EC%95%84%ED%95%98" title="호르-아하">호르-아하</a>의 전대 파라오라는 사실을 보여주고 있다.</p>
