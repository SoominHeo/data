<p><b>나르바</b>(<span style="font-size: smaller;"><a href="/wiki/%EC%97%90%EC%8A%A4%ED%86%A0%EB%8B%88%EC%95%84%EC%96%B4" title="에스토니아어">에스토니아어</a>:</span> <span lang="et" xml:lang="et">Narva</span>, <span style="font-size: smaller;"><a href="/wiki/%EB%9F%AC%EC%8B%9C%EC%95%84%EC%96%B4" title="러시아어">러시아어</a>:</span> <span lang="ru" xml:lang="ru">Нарва</span>)는 <a href="/wiki/%EC%97%90%EC%8A%A4%ED%86%A0%EB%8B%88%EC%95%84" title="에스토니아">에스토니아</a>에서 세 번째로 큰 도시이자 에스토니아 최동단에 위치한 도시이다. <a href="/wiki/%EB%9F%AC%EC%8B%9C%EC%95%84" title="러시아">러시아</a> <a href="/wiki/%EC%9D%B4%EB%B0%98%EA%B3%A0%EB%A1%9C%EB%93%9C" title="이반고로드">이반고로드</a>와 맞닿은 곳에 위치하고 있으며 <a href="/wiki/%EB%82%98%EB%A5%B4%EB%B0%94_%EA%B0%95" title="나르바 강">나르바 강</a>은 이 곳을 거쳐서 <a href="/wiki/%ED%8E%98%EC%9D%B4%ED%91%B8%EC%8A%A4_%ED%98%B8" title="페이푸스 호">페이푸스 호</a>로 흘러간다.</p>
<p>인구는 65,886명(<a href="/wiki/2009%EB%85%84" title="2009년">2009년</a> 기준)이며 면적은 84.54㎢이다. 도시 인구 가운데 93.85%는 <a href="/wiki/%EB%9F%AC%EC%8B%9C%EC%95%84%EC%96%B4" title="러시아어">러시아어</a>를 구사할 수 있으며 <a href="/wiki/%EB%9F%AC%EC%8B%9C%EC%95%84%EC%9D%B8" title="러시아인">러시아인</a>은 도시 인구의 80%를 차지한다.</p>
<h2><span class="mw-headline" id=".EC.97.AD.EC.82.AC">역사</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%82%98%EB%A5%B4%EB%B0%94&amp;action=edit&amp;section=1" title="부분 편집: 역사">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p><a href="/wiki/13%EC%84%B8%EA%B8%B0" title="13세기">13세기</a> 후반 <a href="/wiki/%EB%8D%B4%EB%A7%88%ED%81%AC" title="덴마크">덴마크</a>가 에스토니아 북부 지역을 정복하면서 덴마크의 영토가 되었으며 <a href="/wiki/1347%EB%85%84" title="1347년">1347년</a>에 덴마크 국왕이 이 곳을 <a href="/wiki/%ED%8A%9C%ED%84%B4_%EA%B8%B0%EC%82%AC%EB%8B%A8" title="튜턴 기사단">튜턴 기사단</a>에게 넘겨주면서 튜턴 기사단의 영지가 된다. 그 후 <a href="/wiki/1558%EB%85%84" title="1558년">1558년</a>에 러시아가 이 곳을 점령했으며 <a href="/wiki/1581%EB%85%84" title="1581년">1581년</a>에 <a href="/wiki/%EC%8A%A4%EC%9B%A8%EB%8D%B4" title="스웨덴">스웨덴</a>의 영토가 되었다.</p>
<p><a href="/wiki/1704%EB%85%84" title="1704년">1704년</a>에 일어난 <a href="/wiki/%EB%8C%80%EB%B6%81%EB%B0%A9_%EC%A0%84%EC%9F%81" title="대북방 전쟁">대북방 전쟁</a> 때 러시아에 정복당했으며 <a href="/wiki/1918%EB%85%84" title="1918년">1918년</a>에 에스토니아가 독립하면서 에스토니아의 영토가 되었다. 그 후 <a href="/wiki/1940%EB%85%84" title="1940년">1940년</a>에 에스토니아가 <a class="mw-redirect" href="/wiki/%EC%86%8C%EB%B9%84%EC%97%90%ED%8A%B8_%EC%97%B0%EB%B0%A9" title="소비에트 연방">소련</a>에 합병되면서 소련의 영토가 되었다가 <a href="/wiki/1991%EB%85%84" title="1991년">1991년</a> 8월 에스토니아가 소련으로부터 분리 독립하면서 다시 에스토니아의 영토가 되었다.</p>
