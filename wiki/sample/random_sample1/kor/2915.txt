<p><b>나세르주의</b>는 <a href="/wiki/%EC%9D%B4%EC%A7%91%ED%8A%B8" title="이집트">이집트</a>의 <a href="/wiki/%EB%8C%80%ED%86%B5%EB%A0%B9" title="대통령">대통령</a>이자 <a href="/wiki/%EC%95%84%EB%9E%8D%EC%9D%B8" title="아랍인">아랍인</a>들의 정신적 국부(國父)인 <a href="/wiki/%EA%B0%80%EB%A7%90_%EC%95%95%EB%8D%B8_%EB%82%98%EC%84%B8%EB%A5%B4" title="가말 압델 나세르">가말 압델 나세르</a>가 주창한 <a href="/wiki/%EC%82%AC%ED%9A%8C%EC%A3%BC%EC%9D%98" title="사회주의">사회주의</a> 이념이다.</p>
<h2><span class="mw-headline" id=".EC.9D.B4.EB.8D.B0.EC.98.AC.EB.A1.9C.EA.B8.B0">이데올로기</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%82%98%EC%84%B8%EB%A5%B4%EC%A3%BC%EC%9D%98&amp;action=edit&amp;section=1" title="부분 편집: 이데올로기">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p><a href="/wiki/%EC%82%AC%ED%9A%8C%EC%A3%BC%EC%9D%98" title="사회주의">사회주의</a>, <a href="/wiki/%EC%95%84%EB%9E%8D_%EC%82%AC%ED%9A%8C%EC%A3%BC%EC%9D%98" title="아랍 사회주의">아랍 사회주의</a>, <a href="/wiki/%EB%B2%94%EC%95%84%EB%9E%8D%EC%A3%BC%EC%9D%98" title="범아랍주의">범아랍주의</a>, 반(反)<a class="mw-redirect" href="/wiki/%EC%8B%9C%EC%98%A4%EB%8B%88%EC%A6%98" title="시오니즘">시오니즘</a>, <a href="/wiki/%EC%84%B8%EC%86%8D%EC%A3%BC%EC%9D%98" title="세속주의">세속주의</a> <a href="/wiki/%EB%B9%84%EB%8F%99%EB%A7%B9_%EC%9A%B4%EB%8F%99" title="비동맹 운동">비동맹주의</a> 등이 결합된 <a class="mw-redirect" href="/wiki/%EC%A2%8C%ED%8C%8C" title="좌파">좌파</a> 이념이다. <a class="mw-redirect" href="/wiki/%EC%82%AC%EC%9C%A0%EC%9E%AC%EC%82%B0" title="사유재산">사유재산</a>을 인정하며 <a href="/wiki/%EB%85%B8%EB%8F%99%EC%9E%90" title="노동자">노동자</a>의 권리를 강력히 주장하고 <a href="/wiki/%EB%B3%B5%EC%A7%80" title="복지">복지</a> 지향적이라는 점에서 <a href="/wiki/%EC%82%AC%ED%9A%8C%EB%AF%BC%EC%A3%BC%EC%A3%BC%EC%9D%98" title="사회민주주의">사회민주주의</a>와도 유사한 면모를 보인다.</p>
<p><a class="mw-disambig" href="/wiki/%EB%82%98%EC%84%B8%EB%A5%B4" title="나세르">나세르</a> 대통령은 무슬림이지만 <a href="/wiki/%EC%84%B8%EC%86%8D%EC%A3%BC%EC%9D%98" title="세속주의">세속주의</a>적인 인물이였기 때문에 이슬람주의 <a href="/wiki/%EC%9A%B0%EC%9D%B5" title="우익">우익</a>들이 정치에 간섭하는것을 반대했다.</p>
