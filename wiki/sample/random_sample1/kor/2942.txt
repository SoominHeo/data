<p><b>나카 시</b>(<span style="font-size: smaller;"><a href="/wiki/%EC%9D%BC%EB%B3%B8%EC%96%B4" title="일본어">일본어</a>:</span> <span lang="ja" xml:lang="ja">那珂市</span>)는 <a href="/wiki/%EC%9D%BC%EB%B3%B8" title="일본">일본</a> <a href="/wiki/%EC%9D%B4%EB%B0%94%EB%9D%BC%ED%82%A4_%ED%98%84" title="이바라키 현">이바라키 현</a> 북부에 있는 <a href="/wiki/%EC%8B%9C_(%ED%96%89%EC%A0%95_%EA%B5%AC%EC%97%AD)" title="시 (행정 구역)">시</a>이다. 2005년 1월 21일에 <a href="/wiki/%EB%82%98%EC%B9%B4_%EA%B5%B0_(%EC%9D%B4%EB%B0%94%EB%9D%BC%ED%82%A4_%ED%98%84)" title="나카 군 (이바라키 현)">나카 군</a> 나카 정이 우리즈라 정을 편입, 합병, 시로 승격해 탄생했다.</p>
<p>나카 대지를 차지하고 북쪽은 <a href="/wiki/%ED%9E%88%ED%83%80%EC%B9%98%EC%98%A4%ED%83%80_%EC%8B%9C" title="히타치오타 시">히타치오타 시</a>와 <a href="/wiki/%ED%9E%88%ED%83%80%EC%B9%98%EC%98%A4%EB%AF%B8%EC%95%BC_%EC%8B%9C" title="히타치오미야 시">히타치오미야 시</a>, 북동쪽은 <a href="/wiki/%ED%9E%88%ED%83%80%EC%B9%98_%EC%8B%9C" title="히타치 시">히타치 시</a>, 동쪽은 <a href="/wiki/%ED%9E%88%ED%83%80%EC%B9%98%EB%82%98%EC%B9%B4_%EC%8B%9C" title="히타치나카 시">히타치나카 시</a>와 <a href="/wiki/%EB%82%98%EC%B9%B4_%EA%B5%B0_(%EC%9D%B4%EB%B0%94%EB%9D%BC%ED%82%A4_%ED%98%84)" title="나카 군 (이바라키 현)">나카 군</a> <a href="/wiki/%EB%8F%84%EC%B9%B4%EC%9D%B4_%EC%B4%8C" title="도카이 촌">도카이 촌</a>, 남쪽은 현청 소재지인 <a href="/wiki/%EB%AF%B8%ED%86%A0_%EC%8B%9C" title="미토 시">미토 시</a>, 서쪽은 <a href="/wiki/%ED%9E%88%EA%B0%80%EC%8B%9C%EC%9D%B4%EB%B0%94%EB%9D%BC%ED%82%A4_%EA%B5%B0" title="히가시이바라키 군">히가시이바라키 군</a> <a href="/wiki/%EC%8B%9C%EB%A1%9C%EC%82%AC%ED%86%A0_%EC%A0%95" title="시로사토 정">시로사토 정</a>과 각각 인접해있고 최근에 미토 시나 히타치나카 시의 주택 지역으로서 인구가 증가했다.</p>

<h2><span class="mw-headline" id=".EC.97.AD.EC.82.AC">역사</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%82%98%EC%B9%B4_%EC%8B%9C&amp;action=edit&amp;section=1" title="부분 편집: 역사">편집</a><span class="mw-editsection-bracket">]</span></span></h2>

<li>1889년 4월 1일 - 정촌제 시행과 함께 나카 군에 7개의 촌이 성립하였다.</li>
<li>1942년 9월 8일 - 스가야 촌이 정으로 승격해 스가타니 정이 되었다.</li>
<li>1955년 3월 31일 - 1정6촌이 합병해 나카 군 나카 정이 되었다.</li>
<li>2005년 1월 21일：나카 정의 북서쪽에 위치하는 나카 군 우리즈라 정을 편입, 합병, 시로 승격해 나카 시가 되었다.</li>
