<h2><span class="mw-headline" id=".EC.B6.95.EA.B5.AC.EC.9D.B8_.EC.83.9D.ED.99.9C">축구인 생활</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%82%98%EC%B9%B4%EB%AC%B4%EB%9D%BC_%EA%B2%90%EA%B3%A0&amp;action=edit&amp;section=1" title="부분 편집: 축구인 생활">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<h3><span class="mw-headline" id=".EC.84.A0.EC.88.98_.EC.83.9D.ED.99.9C">선수 생활</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%82%98%EC%B9%B4%EB%AC%B4%EB%9D%BC_%EA%B2%90%EA%B3%A0&amp;action=edit&amp;section=2" title="부분 편집: 선수 생활">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p><a href="/wiki/2003%EB%85%84" title="2003년">2003년</a>, <a href="/wiki/%EA%B0%80%EC%99%80%EC%82%AC%ED%82%A4_%ED%94%84%EB%A1%A0%ED%83%88%EB%A0%88" title="가와사키 프론탈레">가와사키 프론탈레</a>의 입단 테스트를 받은 후 입단에 성공하였으며 리그 개막전에 선발 출장하며 데뷔전을 치렀다. <a href="/wiki/2003%EB%85%84" title="2003년">2003년</a> <a href="/wiki/4%EC%9B%94_9%EC%9D%BC" title="4월 9일">4월 9일</a> <a href="/wiki/%EC%95%8C%EB%B9%84%EB%A0%89%EC%8A%A4_%EB%8B%88%EA%B0%80%ED%83%80" title="알비렉스 니가타">알비렉스 니가타</a>와의 경기에서 프로 데뷔 후 첫 골을 성공시켰다.</p>
<p><a href="/wiki/2004%EB%85%84" title="2004년">2004년</a>, <a href="/wiki/%EB%AF%B8%EB%93%9C%ED%95%84%EB%8D%94#.EA.B3.B5.EA.B2.A9.ED.98.95_.EB.AF.B8.EB.93.9C.ED.95.84.EB.8D.94" title="미드필더">공격형 미드필더</a>에서 <a href="/wiki/%EB%AF%B8%EB%93%9C%ED%95%84%EB%8D%94#.EC.88.98.EB.B9.84.ED.98.95_.EB.AF.B8.EB.93.9C.ED.95.84.EB.8D.94" title="미드필더">수비형 미드필더</a>로 보직을 바꿔 팀의 핵심 선수로 자리잡기 시작하였다. <a href="/wiki/2006%EB%85%84" title="2006년">2006년</a> 처음으로 <a href="/wiki/J%EB%A6%AC%EA%B7%B8" title="J리그">J리그</a> 베스트 11에 뽑혔으며 <a href="/wiki/2009%EB%85%84" title="2009년">2009년</a>까지 꾸준히 베스트 11에 뽑혔다.</p>
<h3><span class="mw-headline" id=".EA.B5.AD.EA.B0.80.EB.8C.80.ED.91.9C_.EC.83.9D.ED.99.9C">국가대표 생활</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%82%98%EC%B9%B4%EB%AC%B4%EB%9D%BC_%EA%B2%90%EA%B3%A0&amp;action=edit&amp;section=3" title="부분 편집: 국가대표 생활">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p><a href="/wiki/2006%EB%85%84" title="2006년">2006년</a> <a href="/wiki/8%EC%9B%94_4%EC%9D%BC" title="8월 4일">8월 4일</a> <a href="/wiki/%EA%B0%80%EB%82%98_%EC%B6%95%EA%B5%AC_%EA%B5%AD%EA%B0%80%EB%8C%80%ED%91%9C%ED%8C%80" title="가나 축구 국가대표팀">가나</a>와의 경기에서 <a href="/wiki/A%EB%A7%A4%EC%B9%98" title="A매치">A매치</a> 데뷔전을 치렀고 <a href="/wiki/2006%EB%85%84" title="2006년">2006년</a> <a href="/wiki/8%EC%9B%94_11%EC%9D%BC" title="8월 11일">8월 11일</a> 벌어진 <a href="/wiki/AFC_%EC%95%84%EC%8B%9C%EC%95%88%EC%BB%B5" title="AFC 아시안컵">AFC 아시안컵</a> 예선 <a href="/wiki/%EC%9D%B8%EB%8F%84_%EC%B6%95%EA%B5%AC_%EA%B5%AD%EA%B0%80%EB%8C%80%ED%91%9C%ED%8C%80" title="인도 축구 국가대표팀">인도</a>와의 경기에서 <a href="/wiki/A%EB%A7%A4%EC%B9%98" title="A매치">A매치</a> 데뷔 후 첫 골을 넣었다.</p>
