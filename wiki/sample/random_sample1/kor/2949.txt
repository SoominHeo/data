<div class="mw-content-ltr" dir="ltr" id="mw-content-text" lang="ko"><script>function mfTempOpenSection(id){var block=document.getElementById("mf-section-"+id);block.className+=" open-block";block.previousSibling.className+=" open-block";}</script><p><b>나카시마 아미</b>(中島麻未, <a href="/wiki/1988%EB%85%84" title="1988년">1988년</a> <a href="/wiki/5%EC%9B%94_11%EC%9D%BC" title="5월 11일">5월 11일</a>)는 <a href="/wiki/%EC%9D%BC%EB%B3%B8" title="일본">일본</a>의 <a href="/wiki/%EA%B0%80%EC%88%98" title="가수">가수</a>로 걸그룹 <a class="mw-redirect" href="/wiki/%EB%93%9C%EB%A6%BC_(%EC%9D%8C%EC%95%85_%EA%B7%B8%EB%A3%B9)" title="드림 (음악 그룹)">Dream</a>의 리더 이자 <a href="/wiki/E-girls" title="E-girls">E-girls</a> 의 센터이다.</p>
<p><a href="/wiki/%EB%8B%88%EC%8B%9C%EB%8B%A4_%EC%8B%9C%EC%A6%88%EC%B9%B4" title="니시다 시즈카">니시다 시즈카</a>랑 같은 <a href="/wiki/%EC%98%A4%EC%82%AC%EC%B9%B4" title="오사카">오사카</a> 출신이자 오사카의 댄스 스쿨 <b>캬레스</b>(キャレス)출신이며 <a href="/wiki/2002%EB%85%84_7%EC%9B%94" title="2002년 7월">2002년 7월</a>에 Dream에 합류했다. 또한 <a class="mw-redirect" href="/wiki/%EB%8B%A4%EC%B9%B4%EB%AA%A8%ED%86%A0_%EC%95%84%EC%95%BC" title="다카모토 아야">다카모토 아야</a>랑 개그콤비 <b>ぺちゃぱぁいず</b>을 결성해 팬들에게 큰 웃음을 주며 <b>카요 아이코</b>(嘉陽愛子)랑 <b>A★Girls</b>를 결성해 활동한 바 있다. <a href="/wiki/%EB%B3%B4%EC%95%84" title="보아">보아</a>와 함께 <a href="/wiki/%EB%B3%B4%EC%95%84" title="보아">보아</a>의 일본싱글 중에서 14번째 싱글앨범인 <a href="/wiki/%EB%A9%94%EB%A6%AC%ED%81%AC%EB%A6%AC" title="메리크리">メリクリ</a>를 듀엣으로 불렀다.</p>
