<p><b>나카쓰 시</b>(<span style="font-size: smaller;"><a href="/wiki/%EC%9D%BC%EB%B3%B8%EC%96%B4" title="일본어">일본어</a>:</span> <span lang="ja"><span lang="ja"><ruby><rb>中津市</rb> <rp>(</rp><rt>なかつし</rt><rp>)</rp></ruby></span></span>)는 <a href="/wiki/%EC%9D%BC%EB%B3%B8" title="일본">일본</a> <a href="/wiki/%EC%98%A4%EC%9D%B4%ED%83%80_%ED%98%84" title="오이타 현">오이타 현</a>의 북서단에 있는 <a class="mw-disambig" href="/wiki/%EC%8B%9C" title="시">시</a>이다. 예전에는 <a href="/wiki/%EB%B6%80%EC%A0%A0_%EA%B5%AD" title="부젠 국">부젠 국</a>에 속해 있었다.</p>
<p>오이타 현내에서는 <a href="/wiki/%EC%98%A4%EC%9D%B4%ED%83%80_%EC%8B%9C" title="오이타 시">오이타 시</a>, <a href="/wiki/%EB%B2%B3%ED%91%B8_%EC%8B%9C" title="벳푸 시">벳푸 시</a>에 이어 인구가 3번째로 많은 도시이며 <a href="/wiki/%EC%A1%B0%EC%B9%B4%EB%A7%88%EC%B9%98" title="조카마치">조카마치</a>에서, 아오노도몬, 라칸지, <a href="/wiki/%ED%9B%84%EC%BF%A0%EC%9E%90%EC%99%80_%EC%9C%A0%ED%82%A4%EC%B9%98" title="후쿠자와 유키치">후쿠자와 유키치</a>의 예전의 집, <a class="new" href="/w/index.php?title=%EB%82%98%EC%B9%B4%EC%93%B0_%EC%84%B1&amp;action=edit&amp;redlink=1" title="나카쓰 성 (없는 문서)">나카쓰 성</a> 등의 문화재나 역사적 건조물, 시역 남부에는 경승지인 야바 계곡이 있는 관광도시이다.</p>
<p>나카쓰 시는 옛 부젠 국에 해당되기 때문에 후쿠오카 현 기타큐슈 지방(<a href="/wiki/%EA%B8%B0%ED%83%80%ED%81%90%EC%8A%88_%EC%8B%9C" title="기타큐슈 시">기타큐슈 시</a>, <a href="/wiki/%EC%9C%A0%EC%BF%A0%ED%95%98%EC%8B%9C_%EC%8B%9C" title="유쿠하시 시">유쿠하시 시</a>, <a href="/wiki/%EB%B6%80%EC%A0%A0_%EC%8B%9C" title="부젠 시">부젠 시</a>, <a href="/wiki/%EC%A7%80%EC%BF%A0%EC%A1%B0_%EA%B5%B0" title="지쿠조 군">지쿠조 군</a>, <a href="/wiki/%EB%AF%B8%EC%95%BC%EC%BD%94_%EA%B5%B0_(%ED%9B%84%EC%BF%A0%EC%98%A4%EC%B9%B4_%ED%98%84)" title="미야코 군 (후쿠오카 현)">미야코 군</a> 등)과의 관계가 강하고 후쿠오카 현으로부터의 통근·통학 인구가 매우 많다. 특히 구 고게 군 지역이었던 부젠 시, 고겐 정, <a href="/wiki/%EC%9A%94%EC%8B%9C%ED%86%A0%EB%AF%B8_%EC%A0%95" title="요시토미 정">요시토미 정</a>은 옛날에는 구 시모게 군과 하나의 군이었기 때문에 나카쓰 시와의 관계가 깊고 경제·문화·생활면에서 나카쓰 시와 하나이다.</p>
<p><a href="/wiki/2004%EB%85%84" title="2004년">2004년</a> 말에 다이하쓰 차체 주식회사가 나카쓰 시 쇼와신덴에 본사, 공장을 이전했기 때문에 자동차 관련 공장의 집적이 진행되고 있다.</p>

<h2><span class="mw-headline" id=".EC.A7.80.EB.A6.AC">지리</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%82%98%EC%B9%B4%EC%93%B0_%EC%8B%9C&amp;action=edit&amp;section=1" title="부분 편집: 지리">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>북부는 스오나다(<a href="/wiki/%EC%84%B8%ED%86%A0_%EB%82%B4%ED%95%B4" title="세토 내해">세토 내해</a>)에 접하고 남부의 혼야바케이 정, 야바케이 정, 야마쿠니 정은 대부분이 산간부로 이 지역은 "야바 계곡"이라 불리며 경승지로서 유명하다. 동부에서 <a href="/wiki/%EC%9A%B0%EC%82%AC_%EC%8B%9C" title="우사 시">우사 시</a>까지는 광범위하게 논이 펼쳐져 현내 최대 규모의 농업 지대이다. 야바 계곡·하치멘 산 기슭의 야마쿠니 강 유역에는 현내 최대 규모의 평야인 나카쓰 평야가 펼쳐진다.</p>
<h3><span class="mw-headline" id=".EA.B8.B0.ED.9B.84">기후</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%82%98%EC%B9%B4%EC%93%B0_%EC%8B%9C&amp;action=edit&amp;section=2" title="부분 편집: 기후">편집</a><span class="mw-editsection-bracket">]</span></span></h3>

<h2><span class="mw-headline" id=".EC.97.AD.EC.82.AC">역사</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%82%98%EC%B9%B4%EC%93%B0_%EC%8B%9C&amp;action=edit&amp;section=4" title="부분 편집: 역사">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<div class="thumb tright">
<div class="thumbinner" style="width:252px;"><a class="image" href="/wiki/%ED%8C%8C%EC%9D%BC:NakatsuCastle.JPG"><img alt="" class="thumbimage" data-file-height="2304" data-file-width="3456" height="167" src="//upload.wikimedia.org/wikipedia/commons/thumb/b/b3/NakatsuCastle.JPG/250px-NakatsuCastle.JPG" srcset="//upload.wikimedia.org/wikipedia/commons/thumb/b/b3/NakatsuCastle.JPG/375px-NakatsuCastle.JPG 1.5x, //upload.wikimedia.org/wikipedia/commons/thumb/b/b3/NakatsuCastle.JPG/500px-NakatsuCastle.JPG 2x" width="250"/></a>
<div class="thumbcaption">
<div class="magnify"><a class="internal" href="/wiki/%ED%8C%8C%EC%9D%BC:NakatsuCastle.JPG" title="실제 크기로"></a></div>
<a class="new" href="/w/index.php?title=%EB%82%98%EC%B9%B4%EC%93%B0_%EC%84%B1&amp;action=edit&amp;redlink=1" title="나카쓰 성 (없는 문서)">나카쓰 성</a></div>
</div>
</div>
<p>성시로서의 역사는 <a href="/wiki/1587%EB%85%84" title="1587년">1587년</a>에 <a href="/wiki/%EA%B5%AC%EB%A1%9C%EB%8B%A4_%EC%9A%94%EC%8B%9C%ED%83%80%EC%B9%B4" title="구로다 요시타카">구로다 요시타카</a>가 <a href="/wiki/%EB%8F%84%EC%9A%94%ED%86%A0%EB%AF%B8_%ED%9E%88%EB%8D%B0%EC%9A%94%EC%8B%9C" title="도요토미 히데요시">도요토미 히데요시</a>로부터 부젠 6군의 16만석을 받은 후에 거점을 야마쿠니 강 하구로 옮겨 <a class="new" href="/w/index.php?title=%EB%82%98%EC%B9%B4%EC%93%B0_%EC%84%B1&amp;action=edit&amp;redlink=1" title="나카쓰 성 (없는 문서)">나카쓰 성</a>을 축성하면서 시작된다. 시가지가 거의 완성된 것은 <a href="/wiki/1600%EB%85%84%EB%8C%80" title="1600년대">1600년대</a> 중순으로 여겨진다. 성주는 후에 <a class="new" href="/w/index.php?title=%ED%98%B8%EC%86%8C%EC%B9%B4%EC%99%80_%EC%94%A8&amp;action=edit&amp;redlink=1" title="호소카와 씨 (없는 문서)">호소카와 씨</a>·<a href="/wiki/%EC%98%A4%EA%B0%80%EC%82%AC%EC%99%80%EB%9D%BC_%EC%94%A8" title="오가사와라 씨">오가사와라 씨</a>를 거쳐 <a href="/wiki/1717%EB%85%84" title="1717년">1717년</a>에 오쿠다이라 씨가 입봉해 <a href="/wiki/%EB%A9%94%EC%9D%B4%EC%A7%80_%EC%9C%A0%EC%8B%A0" title="메이지 유신">메이지 유신</a>을 맞이하게 된다.</p>
<p><a href="/wiki/1876%EB%85%84" title="1876년">1876년</a>에 나카쓰 지청이 설치되어 오이타 현북의 중심지로서 중요성을 띠게 되었다. 근대 공업의 시작은 <a href="/wiki/1879%EB%85%84" title="1879년">1879년</a>부터 <a href="/wiki/1880%EB%85%84" title="1880년">1880년</a>에 걸쳐 공장이 설치된 것이 최초로 여겨지고 <a href="/wiki/1890%EB%85%84%EB%8C%80" title="1890년대">1890년대</a>에는 섬유 공업이 집적하게 되었다. <a href="/wiki/1944%EB%85%84" title="1944년">1944년</a>의 고베 제강소의 진출은 <a href="/wiki/%EC%A0%9C2%EC%B0%A8_%EC%84%B8%EA%B3%84_%EB%8C%80%EC%A0%84" title="제2차 세계 대전">제2차 세계 대전</a> 후의 중공업화의 계기가 되었고 <a href="/wiki/1970%EB%85%84" title="1970년">1970년</a>경까지는 철강업과 위생 도기 제조가 특히 번성했다. 1970년대 후반부터는 자동차 관련 공장의 진출이 중심이 되어 <a href="/wiki/1984%EB%85%84" title="1984년">1984년</a>에는 "도요노쿠니 테크노폴리스"의 중심 도시로서 지정되었다.</p>
<ul>
<li>1888년 - 정으로 승격하였다.</li>
<li>1929년 4월 20일 - 오이타 현에서 3번째로 시로 승격하였다.</li>
<li>1943년 - 쓰루이 촌, 오하타마을, 조스이 촌을 편입하였다.</li>
<li>1951년 - 미호 촌을 편입하였다.</li>
<li>1954년 - 와다 촌을 편입하였다.</li>
<li>1955년 - 이마즈 정을 편입하였다.</li>
<li>2005년 3월 1일 - 시모게 군(산코 촌, 혼야바케이 정, 야바케이 정, 야마쿠니 정)을 편입하였다.</li>
</ul>
