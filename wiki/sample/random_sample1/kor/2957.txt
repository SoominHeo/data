<p><b>나콘파놈 주</b>(<span style="font-size: smaller;"><a href="/wiki/%ED%83%9C%EA%B5%AD%EC%96%B4" title="태국어">태국어</a>:</span> <span lang="th" xml:lang="th">นครพนม</span>)는 <a href="/wiki/%ED%83%80%EC%9D%B4" title="타이">타이</a> <a href="/wiki/%EC%9D%B4%EC%82%B0_(%ED%83%80%EC%9D%B4)" title="이산 (타이)">북동부</a>의 주(창왓)의 하나이다. 이웃한 주는 남쪽부터 시계 방향으로 <a href="/wiki/%EB%AC%B5%EB%8B%A4%ED%95%9C_%EC%A3%BC" title="묵다한 주">묵다한 주</a>, <a href="/wiki/%EC%82%AC%EA%BC%B0%EB%82%98%EC%BD%98_%EC%A3%BC" title="사꼰나콘 주">사꼰나콘 주</a>, <a href="/wiki/%EB%86%8D%EC%B9%B4%EC%9D%B4_%EC%A3%BC" title="농카이 주">농카이 주</a>이다. 동쪽은 <a href="/wiki/%EB%9D%BC%EC%98%A4%EC%8A%A4" title="라오스">라오스</a>의 <a href="/wiki/%EC%BA%84%EB%AC%B4%EC%95%88_%EC%A3%BC" title="캄무안 주">캄무안 주</a>와 접한다.</p>

<h2><span class="mw-headline" id=".EC.A7.80.EB.A6.AC">지리</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%82%98%EC%BD%98%ED%8C%8C%EB%86%88_%EC%A3%BC&amp;action=edit&amp;section=1" title="부분 편집: 지리">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>주는 <a href="/wiki/%EB%A9%94%EC%BD%A9_%EA%B0%95" title="메콩 강">메콩 강</a>의 계곡에 위치하고 대부분이 평야이다. 주의 북부는 고도가 높고 숲으로 덮여있다. 북부의 주요 강은 <a class="new" href="/w/index.php?title=%EC%86%A1%ED%81%AC%EB%9E%8C_%EA%B0%95&amp;action=edit&amp;redlink=1" title="송크람 강 (없는 문서)">송크람 강</a>과 오운 강이다. 남부는 더 평평하고 중요한 강으로 꿈 강이 있다. 주도인 <a href="/wiki/%EB%82%98%EC%BD%98%ED%8C%8C%EB%86%88" title="나콘파놈">나콘파놈</a>은 메콩 강의 기슭에 위치한다.</p>
<p>숲과 구릉으로 이루어진 50km²면적의 푸랑까 국립공원에는 폭포인 땃 캄과 가장 인기있는 볼거리인 땃 포가 있다. 또한 Lady Slipper Orchid 꽃밭과 다양한 야생동물을 볼 수 있다.</p>
<h2><span class="mw-headline" id=".EC.97.AD.EC.82.AC">역사</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%82%98%EC%BD%98%ED%8C%8C%EB%86%88_%EC%A3%BC&amp;action=edit&amp;section=2" title="부분 편집: 역사">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>나콘파놈은 '산의 도시'를 의미하고 <a href="/wiki/%EB%9D%BC%EB%A7%88_1%EC%84%B8" title="라마 1세">라마 1세</a>에 의해 붙여진 것이다. 그러나 실제로는 나콘파놈에 산은 없고 석회암 산이 <a href="/wiki/%EB%A9%94%EC%BD%A9_%EA%B0%95" title="메콩 강">메콩 강</a> 건너의 <a href="/wiki/%EB%9D%BC%EC%98%A4%EC%8A%A4" title="라오스">라오스</a>의 <a class="new" href="/w/index.php?title=%ED%83%81%ED%97%A5&amp;action=edit&amp;redlink=1" title="탁헥 (없는 문서)">탁헥</a>에 집중되어있다. 따라서 나콘파놈은 아마도 '산이 보이는 도시'를 의미할 것이다.</p>
<p>이 지역은 오랫동안 <a href="/wiki/%EB%9D%BC%EC%98%A4%EC%A1%B1" title="라오족">라오족</a>이 거주했고 <a class="mw-redirect" href="/wiki/%EB%9E%80%EC%83%81_%EC%99%95%EA%B5%AD" title="란상 왕국">란상 왕국</a>에 속했다. 또한 <a class="mw-redirect" href="/wiki/%EC%95%84%EC%9C%A0%ED%83%80%EC%95%BC_%EC%99%95%EA%B5%AD" title="아유타야 왕국">아유타야 왕국</a>의 지배를 받게 된 후에도 라오족의 우세는 지속되었다.</p>
<p><a href="/wiki/%EB%B2%A0%ED%8A%B8%EB%82%A8_%EC%A0%84%EC%9F%81" title="베트남 전쟁">베트남 전쟁</a>에 의해서 난민이 되어, 라오스에서 도망쳐 온 베트남인이 살고 있는 지역이 <a href="/wiki/%EB%82%98%EC%BD%98%ED%8C%8C%EB%86%88" title="나콘파놈">나콘파놈</a> 시의 일각에 있다.</p>
<h2><span class="mw-headline" id=".ED.96.89.EC.A0.95_.EA.B5.AC.EC.97.AD">행정 구역</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%82%98%EC%BD%98%ED%8C%8C%EB%86%88_%EC%A3%BC&amp;action=edit&amp;section=3" title="부분 편집: 행정 구역">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<div class="floatleft"><a class="image" href="/wiki/%ED%8C%8C%EC%9D%BC:Amphoe_Nakhon_Phanom.png"><img alt="Amphoe Nakhon Phanom.png" data-file-height="430" data-file-width="345" height="187" src="//upload.wikimedia.org/wikipedia/commons/thumb/2/23/Amphoe_Nakhon_Phanom.png/150px-Amphoe_Nakhon_Phanom.png" srcset="//upload.wikimedia.org/wikipedia/commons/thumb/2/23/Amphoe_Nakhon_Phanom.png/225px-Amphoe_Nakhon_Phanom.png 1.5x, //upload.wikimedia.org/wikipedia/commons/thumb/2/23/Amphoe_Nakhon_Phanom.png/300px-Amphoe_Nakhon_Phanom.png 2x" width="150"/></a></div>
<p>주에는 12개의 <a href="/wiki/%EC%95%94%ED%94%84" title="암프">암프</a>(군)와 더 세부 행정구역인 97개의 <a href="/wiki/%EB%95%80%EB%B3%B8" title="땀본">땀본</a> 그리고 1040개의 <a class="mw-redirect" href="/wiki/%EB%AC%B4%EB%B0%98" title="무반">무반</a>이 있다.</p>
