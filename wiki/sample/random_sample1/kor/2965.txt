<h1 class="firstHeading" id="firstHeading" lang="ko">나투람 고드세</h1>
<div class="mw-body-content" id="bodyContent">

<p><b>나투람 비나야크 고드세</b> (<span lang="mr" xml:lang="mr">नथूराम विनायक गोडसे</span>, <a href="/wiki/1910%EB%85%84" title="1910년">1910년</a> <a href="/wiki/5%EC%9B%94_19%EC%9D%BC" title="5월 19일">5월 19일</a> ~ <a href="/wiki/1949%EB%85%84" title="1949년">1949년</a> <a href="/wiki/11%EC%9B%94_15%EC%9D%BC" title="11월 15일">11월 15일</a>) 는 반이슬람 광신적 힌두교도로, <a href="/wiki/%EB%A7%88%ED%95%98%ED%8A%B8%EB%A7%88_%EA%B0%84%EB%94%94" title="마하트마 간디">마하트마 간디</a>를 암살한 사람이다. 그는 <a href="/wiki/%ED%91%B8%EB%84%A4" title="푸네">푸네</a> Baramati출생이고 아버지 Vinayak Vamanrao는 <a href="/wiki/%EC%9A%B0%EC%B2%B4%EA%B5%AD" title="우체국">우체국</a> 직원이었다.</p>
<p>그는 간디가 인도의 분단을 막기 위해 힌두교와 이슬람교의 통합을 노력하자 그것에 불만을 품고 <a href="/wiki/1948%EB%85%84" title="1948년">1948년</a> <a href="/wiki/1%EC%9B%94_30%EC%9D%BC" title="1월 30일">1월 30일</a>에 <a href="/wiki/%EB%89%B4%EB%8D%B8%EB%A6%AC" title="뉴델리">뉴델리</a>에 열린 <a class="new" href="/w/index.php?title=%EA%B8%B0%EB%8F%84%ED%9A%8C&amp;action=edit&amp;redlink=1" title="기도회 (없는 문서)">기도회</a>장에 참석한 <a class="mw-redirect" href="/wiki/%EA%B0%84%EB%94%94" title="간디">간디</a>를 <a href="/wiki/%EC%B4%9D" title="총">총</a>으로 쏴 <a href="/wiki/%EC%95%94%EC%82%B4" title="암살">암살</a>했으며 그 자리에서 체포되었다. 이후 <a href="/wiki/%EA%B5%90%EC%88%98%ED%98%95" title="교수형">교수형</a>으로 <a class="mw-redirect" href="/wiki/%EC%B2%98%ED%98%95" title="처형">처형</a>되었다.</p>
