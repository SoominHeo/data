<h2><span class="mw-headline" id=".EC.83.9D.EC.95.A0">생애</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%82%98%ED%9B%88%EC%95%84&amp;action=edit&amp;section=1" title="부분 편집: 생애">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>나훈아는 <a href="/wiki/1950%EB%85%84" title="1950년">1950년</a> <a href="/wiki/2%EC%9B%94_11%EC%9D%BC" title="2월 11일">2월 11일</a>에 <a href="/wiki/%EB%B6%80%EC%82%B0%EA%B4%91%EC%97%AD%EC%8B%9C" title="부산광역시">부산</a>에서 선원이자 무역상이던 아버지와 어머니 사이에서 2남 2녀 중 차남으로 태어났다.<sup class="reference" id="cite_ref-1"><a href="#cite_note-1">[1]</a></sup>학창 시절의 나훈아는 노래를 좋아해 고향 뒷산에서 친구들과 함께 기타를 즐겨쳤다고 하는데, 나훈아의 지인들은 나훈아가 악기를 다루는데 능숙하고 그 중에서도 피아노 실력이 수준급이라고 밝혔다. <a href="/wiki/1965%EB%85%84" title="1965년">1965년</a>에 형을 따라 <a href="/wiki/%EC%84%9C%EC%9A%B8%ED%8A%B9%EB%B3%84%EC%8B%9C" title="서울특별시">서울</a>에 상경하여 <a class="mw-redirect" href="/wiki/%EC%84%9C%EB%9D%BC%EB%B2%8C%EC%98%88%EC%88%A0%EA%B3%A0%EB%93%B1%ED%95%99%EA%B5%90" title="서라벌예술고등학교">서라벌예고</a>에 입학하여 <a href="/wiki/1968%EB%85%84" title="1968년">1968년</a>에 졸업하였다. 고교 2학년리 되던 해 나훈아는 오아시스레코드와 계약하여 〈천리길〉 을 발표하면서 공식적으로 가요계에 데뷔하였다. 자신만의 독특한 간드러진 꺾기 창법이 매력적이었던 나훈아는 <a href="/wiki/1968%EB%85%84" title="1968년">1968년</a>에 발표한 〈사랑은 눈물의 씨앗〉 은 크게 히트하여 나훈아는 대중가수 대열에 합류하게 되었으며 <a href="/wiki/1970%EB%85%84%EB%8C%80" title="1970년대">1970년대</a>에는 <a href="/wiki/%EB%82%A8%EC%A7%84" title="남진">남진</a>과 함께 라이벌 구도를 이루었고 <a class="mw-redirect" href="/wiki/%EB%8C%80%EC%A4%91%EA%B0%80%EC%9A%94" title="대중가요">대중가요</a>를 주름잡았다. <a href="/wiki/%EB%82%A8%EC%A7%84" title="남진">남진</a>과 나훈아는 <a href="/wiki/1970%EB%85%84%EB%8C%80" title="1970년대">1970년대</a>의 가요계를 장악하면서 서로 경쟁을 벌여 보통 <a href="/wiki/%EB%82%A8%EC%A7%84" title="남진">남진</a> 아니면 나훈아가 가수왕상을 수상하였다. 하지만 실제 방송사 기록을 보면 <a href="/wiki/%EB%82%A8%EC%A7%84" title="남진">남진</a>이 주로 1위와 동시에 가수왕상을 수상하였으며 나훈아는 주로 2위를 한 것으로 집계되었다.</p>
<p>나훈아는 <a href="/wiki/%EB%82%A8%EC%A7%84" title="남진">남진</a>과 함께 경쟁하면서 그 시기에 침체된 <a href="/wiki/%EB%8C%80%EC%A4%91%EC%9D%8C%EC%95%85" title="대중음악">대중음악</a>에 활력을 불어넣어 대중가요 활성화를 이끌었다. <a href="/wiki/1970%EB%85%84%EB%8C%80" title="1970년대">1970년대</a>는 나훈아가 주요 히트곡을 동시에 발표하면서 <a href="/wiki/%EB%82%A8%EC%A7%84" title="남진">남진</a>이 발표하는 노래와 동시에 인기를 얻게 되자 그 어느 때보다도 치열한 경합이 펼쳐졌다.</p>
<p>나훈아는 <a href="/wiki/%EB%82%A8%EC%A7%84" title="남진">남진</a>과 달리 대부분 차분하고 조용한 노래를 선곡하였으며 당시 무대에서 다양한 포즈와 꺾기 창법으로 <a href="/wiki/%ED%8A%B8%EB%A1%9C%ED%8A%B8" title="트로트">트로트</a>를 불러서 중년층에게 많은 인기를 얻었다. 그의 인기를 알 수 있듯이 <a href="/wiki/%EC%82%BC%EC%84%B1%EC%A0%84%EC%9E%90" title="삼성전자">삼성전자</a>의 회장 <a href="/wiki/%EC%9D%B4%EA%B1%B4%ED%9D%AC" title="이건희">이건희</a>의 연회에 참석할 것을 초청받았으나 나훈아는 자신의 공연을 보기 위하여 표를 사는 사람들에게만 노래를 한다며 단호히 거절하였다. 전성기를 한창 누비고 있었던 나훈아는 <a href="/wiki/1972%EB%85%84" title="1972년">1972년</a>에 <a class="mw-redirect" href="/wiki/%EC%84%9C%EC%9A%B8%EC%8B%9C%EB%AF%BC%ED%9A%8C%EA%B4%80" title="서울시민회관">서울시민회관</a>에서 공연하던 중 한 남자에게 병 파편으로 피습을 당해 몇 개월을 입원하였다. 이 같은 소식을 들은 나훈아의 팬들은 <a href="/wiki/%EB%82%A8%EC%A7%84" title="남진">남진</a>의 팬이 나훈아를 다치게 했다는 루머를 믿고 서로 패싸움을 벌이기도 했지만 양측 모두 사실을 부인하면서 루머는 일단락되었다. 이후 나훈아는 건강을 회복한 뒤 <a href="/wiki/1973%EB%85%84" title="1973년">1973년</a> 비밀리에 <a href="/wiki/%EB%8C%80%ED%95%9C%EB%AF%BC%EA%B5%AD_%EA%B3%B5%EA%B5%B0" title="대한민국 공군">공군</a>에 입대하여 큰 화제를 불러모았다. 입대 직전 배우 <a href="/wiki/%EA%B3%A0%EC%9D%80%EC%95%84_(1946%EB%85%84)" title="고은아 (1946년)">고은아</a>의 사촌인 이숙희와 결혼했다가 전역을 1년 앞둔 <a href="/wiki/1975%EB%85%84" title="1975년">1975년</a>에 이혼하였다.</p>
<p><a href="/wiki/1976%EB%85%84" title="1976년">1976년</a> 전역한 뒤 얼마되지 않아 영화계에서 최고의 주가를 달리던 여배우 <a href="/wiki/%EA%B9%80%EC%A7%80%EB%AF%B8" title="김지미">김지미</a>와 결혼을 발표하며 큰 화제가 되었다. 그해 <a href="/wiki/%EC%8B%AC%EC%88%98%EB%B4%89" title="심수봉">심수봉</a>이 남산에 위치했었던 도쿄호텔 스카이라운지 에서 피아노를 치면서 노래를 부르는 모습을 보고 실력에 감탄한 나훈아는 <a href="/wiki/%EC%8B%AC%EC%88%98%EB%B4%89" title="심수봉">심수봉</a>에게 본명인 최홍기로 쓴 자작곡 〈여자이니까〉 를 주면서 <a href="/wiki/%EA%B0%80%EC%88%98" title="가수">가수</a>로서의 데뷔에 힘을 실어 주었다. 그리고 <a href="/wiki/1977%EB%85%84" title="1977년">1977년</a>에 〈애정이 꽃피던 시절〉 이란 곡을 발표했는데 이 곡은 <a href="/wiki/1974%EB%85%84" title="1974년">1974년</a>에 작곡가 <a class="mw-disambig" href="/wiki/%EB%B0%95%EC%84%B1%EA%B7%9C" title="박성규">박성규</a>가 가수 <a href="/wiki/%EB%B0%A9%EC%A3%BC%EC%97%B0" title="방주연">방주연</a>에게 청혼하며 주었던 가사없이 멜로디만 주어서 <a href="/wiki/%EB%B0%A9%EC%A3%BC%EC%97%B0" title="방주연">방주연</a>이 작사하여 발표한 〈생각해보세요〉(<a href="/wiki/%EB%B0%A9%EC%A3%BC%EC%97%B0" title="방주연">방주연</a>/<a class="mw-disambig" href="/wiki/%EB%B0%95%EC%84%B1%EA%B7%9C" title="박성규">박성규</a>/원곡:<a href="/wiki/%EB%B0%A9%EC%A3%BC%EC%97%B0" title="방주연">방주연</a>) 라는 곡이 원곡 이었는데 <a href="/wiki/%EB%B0%A9%EC%A3%BC%EC%97%B0" title="방주연">방주연</a>이 청혼을 거절하자 <a class="mw-disambig" href="/wiki/%EB%B0%95%EC%84%B1%EA%B7%9C" title="박성규">박성규</a>는 <a href="/wiki/%EB%B0%A9%EC%A3%BC%EC%97%B0" title="방주연">방주연</a>에게 줬던 악보를 다시 가져가서 가사를 새로 지어 3년 뒤에 <a href="/wiki/1977%EB%85%84" title="1977년">1977년</a>에 나훈아에게 주었고 이후 〈애정이 꽃피던 시절〉(<a class="mw-disambig" href="/wiki/%EB%B0%95%EC%84%B1%EA%B7%9C" title="박성규">박성규</a>) 로 발표되었다.</p>
<p>나훈아와 <a href="/wiki/%EA%B9%80%EC%A7%80%EB%AF%B8" title="김지미">김지미</a>는 <a href="/wiki/%EA%B9%80%EC%A7%80%EB%AF%B8" title="김지미">김지미</a>의 고향인 <a href="/wiki/%EB%8C%80%EC%A0%84%EA%B4%91%EC%97%AD%EC%8B%9C" title="대전광역시">대전</a> <a href="/wiki/%EC%8B%A0%ED%83%84%EC%A7%84%EB%8F%99" title="신탄진동">신탄진동</a>에서 신혼집을 마련하여 거주하게 되었다. 그로 인해 방송에 출연하지 않던 나훈아는 <a href="/wiki/1981%EB%85%84" title="1981년">1981년</a>에 〈대동강 편지〉 를 발표하면서 가요계에 복귀하였으며 <a class="mw-redirect" href="/wiki/MBC" title="MBC">MBC</a> 10대 가수상을 수상하였다. <a href="/wiki/1982%EB%85%84" title="1982년">1982년</a>에 〈울긴 왜 울어〉 를 발표하며 다시금 대중들의 인기를 얻었다. 하지만 가요계 복귀로 인해 <a href="/wiki/%EA%B9%80%EC%A7%80%EB%AF%B8" title="김지미">김지미</a>와 사이가 나빠지며 <a href="/wiki/1982%EB%85%84" title="1982년">1982년</a>에 <a href="/wiki/%EA%B9%80%EC%A7%80%EB%AF%B8" title="김지미">김지미</a>와 이혼하였고 이듬해 가수 출신의<a href="/wiki/%EC%A0%95%EC%88%98%EA%B2%BD" title="정수경">정수경</a>과 결혼한 후 1남 1녀를 두었다. <a href="/wiki/1984%EB%85%84" title="1984년">1984년</a>에 <a href="/wiki/%EC%A1%B0%EC%9A%A9%ED%95%84" title="조용필">조용필</a> 다음으로 <a href="/wiki/%EC%9D%BC%EB%B3%B8" title="일본">일본</a>에 진출하여 데이지쿠레코드와의 계약을 성공적으로 이끌었다.</p>
<p>나훈아는 <a href="/wiki/1980%EB%85%84%EB%8C%80" title="1980년대">1980년대</a> 후반 방송에 모습을 드러내지 않았음에도 나훈아의 공연장은 인산인해를 이루었다. 방송에서 나훈아를 볼 수 있는 기회는 명절 특집 프로그램 밖에 없었기 때문이다. 홍보와 마케팅에 집중하는 현대 가수들은 위험할 수도 있는 은둔 활동이라는 일종의 희소 가치는 그의 공연에 관객을 몰려들게 했고 그의 존재감은 전설로 상승했다. 또한 주로 서정적이면서 슬픈 로맨스틱한 본인의 자작곡들을 불렀는데 〈갈무리〉, 〈영영〉, 〈내 삶을 눈물로 채워도〉, 〈홍시〉 가 대표적이다. 나훈아의 자작곡 중 <a href="/wiki/1987%EB%85%84" title="1987년">1987년</a>에 발표한 〈땡벌〉 은 발표 당시 별다른 인기를 얻지 못하였고 13년이 지난 <a href="/wiki/2000%EB%85%84" title="2000년">2000년</a>에 <a href="/wiki/%EA%B0%95%EC%A7%84_(%EA%B0%80%EC%88%98)" title="강진 (가수)">강진</a>이 리메이크하여 발표하였는데 역시 발표 직후 인기를 얻지 못하였으나 <a href="/wiki/2006%EB%85%84" title="2006년">2006년</a>에 <a href="/wiki/%EC%A1%B0%EC%9D%B8%EC%84%B1_(%EB%B0%B0%EC%9A%B0)" title="조인성 (배우)">조인성</a>이《<a href="/wiki/%EB%B9%84%EC%97%B4%ED%95%9C_%EA%B1%B0%EB%A6%AC_(2006%EB%85%84_%EC%98%81%ED%99%94)" title="비열한 거리 (2006년 영화)">비열한 거리</a>》에서 이 곡을 불러 화제가 되었고, <a class="mw-redirect" href="/wiki/KBS" title="KBS">KBS</a> 드라마《<a href="/wiki/%EC%86%8C%EB%AC%B8%EB%82%9C_%EC%B9%A0%EA%B3%B5%EC%A3%BC" title="소문난 칠공주">소문난 칠공주</a>》에서 <a href="/wiki/%EC%9D%B4%EC%8A%B9%EA%B8%B0" title="이승기">이승기</a>가 극 중에서 어머니를 위로하는 곡으로 〈땡벌〉 을 부르는 장면이 등장하며 인지도가 상승하게 되었고 <a href="/wiki/2007%EB%85%84" title="2007년">2007년</a> <a href="/wiki/9%EC%9B%94_21%EC%9D%BC" title="9월 21일">9월 21일</a>에 방송된 <a class="mw-redirect" href="/wiki/KBS" title="KBS">KBS</a>《<a href="/wiki/%EB%AE%A4%EC%A7%81%EB%B1%85%ED%81%AC" title="뮤직뱅크">뮤직뱅크</a>》에서 <a href="/wiki/%EA%B0%95%EC%A7%84_(%EA%B0%80%EC%88%98)" title="강진 (가수)">강진</a>이 1위를 차지하며 〈땡벌〉 은 발표된 지 20년 후에야 많은 인기를 얻었다.</p>
<p>나훈아는 호소력 짙은 목소리와 시대를 달리하는 끊임없는 히트곡 양산과 더불어 작곡과 작사 능력으로 <a href="/wiki/%EC%8B%B1%EC%96%B4%EC%86%A1%EB%9D%BC%EC%9D%B4%ED%84%B0" title="싱어송라이터">싱어송라이터</a>로 자리매김했으며 <a href="/wiki/1990%EB%85%84%EB%8C%80" title="1990년대">1990년대</a> 초반 '한국 가요계의 살아 있는 전설'로 불리게 되었던 것이다. 데뷔 이후 현재까지 약 2500여곡을 취입하고 정규 앨범 19장을 포함한 200여개의 앨범을 발표했으며 나훈아가 직접 작사하거나 작곡한 노래는 약 800여곡 이상으로 추정하고 있다. <a href="/wiki/2007%EB%85%84_3%EC%9B%94" title="2007년 3월">2007년 3월</a>에 <a href="/wiki/%EC%84%B8%EC%A2%85%EB%AC%B8%ED%99%94%ED%9A%8C%EA%B4%80" title="세종문화회관">세종문화회관</a>에서 예정된 콘서트가 갑작스럽게 취소되며 <a href="/wiki/2006%EB%85%84" title="2006년">2006년</a> 순회 콘서트가 나훈아의 마지막 활동이 되었고 이후 나훈아는 가요계에서 잠정 은퇴하였다.</p>

<h2><span class="mw-headline" id=".EC.82.AC.EA.B1.B4">사건</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%82%98%ED%9B%88%EC%95%84&amp;action=edit&amp;section=7" title="부분 편집: 사건">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<h3><span class="mw-headline" id=".ED.94.BC.EC.8A.B5_.EC.82.AC.EA.B1.B4">피습 사건</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%82%98%ED%9B%88%EC%95%84&amp;action=edit&amp;section=8" title="부분 편집: 피습 사건">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p><a href="/wiki/1972%EB%85%84" title="1972년">1972년</a> <a href="/wiki/7%EC%9B%94_4%EC%9D%BC" title="7월 4일">7월 4일</a>에 나훈아는 <a class="mw-redirect" href="/wiki/%EC%84%9C%EC%9A%B8%EC%8B%9C%EB%AF%BC%ED%9A%8C%EA%B4%80" title="서울시민회관">서울시민회관</a>에서 리사이틀 공연을 하고 있는 중에 <a href="/wiki/%EB%82%A8%EC%A7%84" title="남진">남진</a>의 사주를 받았다는 한 남자가 무대에 뛰어올라 깨진 사이다 병 파편으로 나훈아를 피습하였다. 갑작스런 습격으로 나훈아는 왼쪽 얼굴에 5cm 가량의 큰 부상을 입었고 72바늘을 꿰매는 대수술을 받아야 했다. 언론 매체들은 용의자가 당시 라이벌이었던 <a href="/wiki/%EB%82%A8%EC%A7%84" title="남진">남진</a>의 사주를 받았을 것이라고 몰고 가는 바람에 <a href="/wiki/%EB%82%A8%EC%A7%84" title="남진">남진</a>의 팬과 나훈아의 팬들이 크게 대립해 긴장감이 조성되었었다. 이후 나훈아 측과 <a href="/wiki/%EB%82%A8%EC%A7%84" title="남진">남진</a> 측이 사실이 아니라고 주장하여 사건은 일단락되었다.</p>
<h3><span class="mw-headline" id=".EB.A3.A8.EB.A8.B8.EC.99.80_.EC.9D.B4.ED.98.BC_.EC.86.8C.EC.86.A1">루머와 이혼 소송</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%82%98%ED%9B%88%EC%95%84&amp;action=edit&amp;section=9" title="부분 편집: 루머와 이혼 소송">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p><a href="/wiki/2008%EB%85%84_1%EC%9B%94" title="2008년 1월">2008년 1월</a>에 모 스포츠 기자가 블로그에 '중견 가수가 젊은 여배우와 스캔들이 나서 야쿠자에게 중요 부위를 훼손당하는 보복을 당했다'는 글을 올렸고 이것이 부풀려져 나훈아 쪽으로 몰아갔다. 결국 나훈아가 직접 기자회견을 열고 사실무근이라고 해명했으며 이 사건 이후로 현재까지 잠적한 상태이다. 당시 기자회견 때 "제가 5분간 보여드리면 믿겠습니까?"라는 말과 함께 벨트를 풀고 바지를 벗으려는 행동을 보여 엄청난 화제가 되었다. 이 기자회견 이후 잠정 은퇴하였고 그 와중에도 팬들을 위해 공연을 준비하려고 했으나, 기자들을 피해 다니다가 결국 모든 일정을 취소하고 칩거하는 중이다. 칩거 와중에도 나훈아는 <a href="/wiki/%EA%B0%80%EC%88%98" title="가수">가수</a>로 복직하려고 여러 시도하였으나 이것 때문에 부부 사이가 나빠져 결국 <a href="/wiki/2011%EB%85%84_8%EC%9B%94" title="2011년 8월">2011년 8월</a>에 아내 <a href="/wiki/%EC%A0%95%EC%88%98%EA%B2%BD" title="정수경">정수경</a>이 이혼 및 재산분할 청구 소송을 제기하면서 <a href="/wiki/%EA%B0%80%EC%88%98" title="가수">가수</a>로의 복직은 불분명해졌다.</p>
<h3><span class="mw-headline" id=".EC.9D.B4.ED.98.BC_.EC.86.8C.EC.86.A1">이혼 소송</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%82%98%ED%9B%88%EC%95%84&amp;action=edit&amp;section=10" title="부분 편집: 이혼 소송">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p><a href="/wiki/2011%EB%85%84_8%EC%9B%94" title="2011년 8월">2011년 8월</a> 아내 <a href="/wiki/%EC%A0%95%EC%88%98%EA%B2%BD" title="정수경">정수경</a>이 이혼 및 재산분할 소송을 제기했으나 <a href="/wiki/2013%EB%85%84_9%EC%9B%94" title="2013년 9월">2013년 9월</a>에 <a href="/wiki/%EB%8C%80%ED%95%9C%EB%AF%BC%EA%B5%AD_%EB%8C%80%EB%B2%95%EC%9B%90" title="대한민국 대법원">대법원</a>에서 기각되었으며,<sup class="reference" id="cite_ref-2"><a href="#cite_note-2">[2]</a></sup>그 이후로 <a href="/wiki/%EA%B0%80%ED%8F%89%EA%B5%B0" title="가평군">가평</a>의 자택에서 칩거를 하다가 결국 <a href="/wiki/2016%EB%85%84" title="2016년">2016년</a> <a href="/wiki/10%EC%9B%94_31%EC%9D%BC" title="10월 31일">10월 31일</a>에 법원에서는 이혼하라고 판결했고 이어 12억을 위자료 배상하라고 하여 5년 동안 진행한 이혼소송에서 패소했다.<sup class="reference" id="cite_ref-3"><a href="#cite_note-3">[3]</a></sup></p>
<h3><span class="mw-headline" id=".EB.87.8C.EA.B2.BD.EC.83.89_.EC.A6.9D.EC.83.81">뇌경색 증상</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%82%98%ED%9B%88%EC%95%84&amp;action=edit&amp;section=11" title="부분 편집: 뇌경색 증상">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p><a href="/wiki/2013%EB%85%84" title="2013년">2013년</a> <a href="/wiki/5%EC%9B%94_23%EC%9D%BC" title="5월 23일">5월 23일</a>에 <a href="/wiki/%EC%9D%B4%EB%8D%B0%EC%9D%BC%EB%A6%AC" title="이데일리">이데일리</a>에서 나훈아는 <a href="/wiki/%EA%B2%BD%EA%B8%B0%EB%8F%84" title="경기도">경기도</a> <a href="/wiki/%EC%84%B1%EB%82%A8%EC%8B%9C" title="성남시">성남</a>에 위치한 요양병원에 머물고 있다고 보도하였다. 나훈아는 한동안 말을 어눌하게 하는 등 뇌경색 증상을 보였지만 양·한방 치료를 받으며 의사표시를 할 수 있을 정도로 상당히 쾌유한 것으로 밝혀졌다.<sup class="reference" id="cite_ref-4"><a href="#cite_note-4">[4]</a></sup></p>
