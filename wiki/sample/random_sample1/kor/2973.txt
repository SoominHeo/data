<p><a href="/wiki/671%EB%85%84" title="671년">671년</a>(<a class="mw-redirect" href="/wiki/%EC%8B%A0%EB%9D%BC_%EB%AC%B8%EB%AC%B4%EC%99%95" title="신라 문무왕">신라 문무왕</a> 11년) <a class="mw-redirect" href="/wiki/%EC%9D%98%EC%83%81" title="의상">의상</a>이 ‘<a class="mw-redirect" href="/wiki/%EA%B4%80%EC%9D%8C%EB%B3%B4%EC%82%B4" title="관음보살">관음보살</a>’의 진신이 이 해변의 굴 안에 머문다는 말을 듣고 굴 속에 들어가 예불하던 중 관음보살이 수정으로 만든 염주를 주면서 절을 지을 곳을 알려 주어 이곳에 사찰을 창건하고 ‘낙산사’라 하였다 한다.<sup class="reference" id="cite_ref-.EC.82.BC.EA.B5.AD.EC.9C.A0.EC.82.AC_4-1"><a href="#cite_note-.EC.82.BC.EA.B5.AD.EC.9C.A0.EC.82.AC-4">[4]</a></sup> <a href="/wiki/858%EB%85%84" title="858년">858년</a> <a class="mw-redirect" href="/wiki/%EC%8B%A0%EB%9D%BC_%ED%97%8C%EC%95%88%EC%99%95" title="신라 헌안왕">신라 헌안왕</a> 2년 <a class="new" href="/w/index.php?title=%EB%B2%94%EC%9D%BC%EB%8C%80%EC%82%AC&amp;action=edit&amp;redlink=1" title="범일대사 (없는 문서)">범일대사</a>가 중창하였으나<sup class="reference" id="cite_ref-5"><a href="#cite_note-5">[5]</a></sup> <a href="/wiki/1231%EB%85%84" title="1231년">1231년</a> 몽골의 침입으로 소실되었다.<sup class="reference" id="cite_ref-6"><a href="#cite_note-6">[6]</a></sup></p>
<p><a href="/wiki/1467%EB%85%84" title="1467년">1467년</a> <a href="/wiki/%EC%A1%B0%EC%84%A0" title="조선">조선</a> <a href="/wiki/%EC%A1%B0%EC%84%A0_%EC%84%B8%EC%A1%B0" title="조선 세조">세조</a>는 <a href="/wiki/%EA%B8%88%EA%B0%95%EC%82%B0" title="금강산">금강산</a>을 둘러 보고 돌아오는 길에 낙산사가 몽골 전란으로 쇠락한 채 방치된 것을 보고 중창을 명령하였다.<sup class="reference" id="cite_ref-7"><a href="#cite_note-7">[7]</a></sup> 중창은 학열(學悅)이 책임지었고<sup class="reference" id="cite_ref-8"><a href="#cite_note-8">[8]</a></sup>, 칠층석탑이 이 때 지어졌다.<sup class="reference" id="cite_ref-9"><a href="#cite_note-9">[9]</a></sup> 1469년(<a href="/wiki/%EC%A1%B0%EC%84%A0_%EC%98%88%EC%A2%85" title="조선 예종">예종</a> 1년) <a href="/wiki/%EC%A0%95%ED%9D%AC%EC%99%95%ED%9B%84" title="정희왕후">정희왕후</a>가 세조를 위해 발원하여 범종을 만들었다. 건칠관음보살상도 이 무렵에 제작된 것으로 보인다.<sup class="reference" id="cite_ref-10"><a href="#cite_note-10">[10]</a></sup></p>
<p><a href="/wiki/%EC%A1%B0%EC%84%A0_%EC%84%B1%EC%A2%85" title="조선 성종">성종</a> 대에 이르러 학열이 노쇠하게 되자 절의 운영권을 두고 다툼이 있었다. 1477년 강릉대도호부사는 성종의 윤허를 받고 쟁송에 대해 입안을 내려 수도하는 수행승이 절의 운영권을 갖도록하였다.<sup class="reference" id="cite_ref-11"><a href="#cite_note-11">[11]</a></sup></p>
<p><a href="/wiki/%EC%9E%84%EC%A7%84%EC%99%9C%EB%9E%80" title="임진왜란">임진왜란</a>과 <a href="/wiki/%EB%B3%91%EC%9E%90%ED%98%B8%EB%9E%80" title="병자호란">병자호란</a> 때 화재를 겪었다. 다시 <a href="/wiki/1624%EB%85%84" title="1624년">1624년</a> <a href="/wiki/%EC%A1%B0%EC%84%A0_%EC%9D%B8%EC%A1%B0" title="조선 인조">인조</a> 원년, 9년(1631)과 21년(1643) 재차 중건이 있었으나 <a href="/wiki/1777%EB%85%84" title="1777년">1777년</a> <a href="/wiki/%EC%A1%B0%EC%84%A0_%EC%A0%95%EC%A1%B0" title="조선 정조">정조</a> 원년 화재를 당하여 다음해 다시 중건하였다. <sup class="reference" id="cite_ref-12"><a href="#cite_note-12">[12]</a></sup></p>
<p><a href="/wiki/%ED%95%9C%EA%B5%AD_%EC%A0%84%EC%9F%81" title="한국 전쟁">한국 전쟁</a> 때 화재로 소실된 것을 <a href="/wiki/1953%EB%85%84" title="1953년">1953년</a>에 다시 지었지만, <a href="/wiki/2005%EB%85%84" title="2005년">2005년</a> <a href="/wiki/4%EC%9B%94_5%EC%9D%BC" title="4월 5일">4월 5일</a> 23시 53분 경 <a href="/wiki/%EA%B0%95%EC%9B%90%EB%8F%84_(%EB%82%A8)" title="강원도 (남)">강원도</a> <a href="/wiki/%EC%96%91%EC%96%91%EA%B5%B0" title="양양군">양양군</a> <a href="/wiki/%EC%96%91%EC%96%91%EC%9D%8D" title="양양읍">양양읍</a> 화일리에서 화재가 일어났다. <a href="/wiki/4%EC%9B%94_5%EC%9D%BC" title="4월 5일">4월 5일</a> 아침 <a class="mw-redirect" href="/wiki/%EC%82%B0%EB%A6%BC%EC%B2%AD" title="산림청">산림청</a>과 <a href="/wiki/%EB%8C%80%ED%95%9C%EB%AF%BC%EA%B5%AD_%EA%B5%AD%EA%B5%B0" title="대한민국 국군">대한민국 국군</a>은 <a href="/wiki/%ED%97%AC%EB%A6%AC%EC%BD%A5%ED%84%B0" title="헬리콥터">헬기</a> 10여 대를 긴급 투입해 진화 작업에 나섰다. 산불은 아침 7시 경 바람을 타고 <a href="/wiki/%EB%82%99%EC%82%B0%ED%95%B4%EC%88%98%EC%9A%95%EC%9E%A5" title="낙산해수욕장">낙산해수욕장</a>까지 번졌고 오전 11시 20분경 큰 불이 거의 잡힌 듯 보여 주민들은 속속 집으로 돌아와 가재 도구를 정리하기 시작했고 진화 작업을 위해 투입됐던 헬기는 고성 산불 진화를 위해 기수를 돌린다. 하지만 잦아지던 불길은 오후 1시경 강풍을 타고 되살아났으며 헬기도 양양으로 방향을 돌린다. 오후 2시 30분 <a href="/wiki/%EC%96%91%EC%96%91%EA%B5%B0" title="양양군">양양군</a>은 재난 경보를 발령했지만 15시 30분 <strong class="selflink">낙산사</strong>에 산불이 확산되어 방화선이 무너졌고 산불이 일어난지 불과 1시간 만에 낙산사 대부분의 전각이 화재로 소실되었다. <a href="/wiki/4%EC%9B%94_7%EC%9D%BC" title="4월 7일">4월 7일</a> <a href="/wiki/%EC%96%91%EC%96%91%EA%B5%B0" title="양양군">양양군</a>은 특별재난지역으로 선포된다. 이 화재로 21채의 건물이 불타고 보물 479호로 지정되어 있던 낙산사 동종이 소실되면서 보물에서 지정 해제되었다.<sup class="reference" id="cite_ref-.EB.A8.B8.EB.8B.88.ED.88.AC.EB.8D.B0.EC.9D.B4_050705_13-0"><a href="#cite_note-.EB.A8.B8.EB.8B.88.ED.88.AC.EB.8D.B0.EC.9D.B4_050705-13">[13]</a></sup> 산불 이후 동종은 복원되었지만 다시 보물로 지정되지는 않았다. 이 때문에 보물 제479호는 결번으로 남아 있다. 낙산사 동종은 2006년 9월 29일 새로 주조되어<sup class="reference" id="cite_ref-14"><a href="#cite_note-14">[14]</a></sup> 10월 13일 충북 진천에서 시험 타종을 거친 후<sup
