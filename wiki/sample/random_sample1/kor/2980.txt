<p><b>난부 시게나오</b>(<span style="font-size: smaller;"><a href="/wiki/%EC%9D%BC%EB%B3%B8%EC%96%B4" title="일본어">일본어</a>:</span> <span lang="ja" xml:lang="ja">南部重直</span>, <a href="/wiki/1606%EB%85%84" title="1606년">1606년</a> <a href="/wiki/4%EC%9B%94_16%EC%9D%BC" title="4월 16일">4월 16일</a> ~ <a href="/wiki/1664%EB%85%84" title="1664년">1664년</a> <a href="/wiki/10%EC%9B%94_30%EC%9D%BC" title="10월 30일">10월 30일</a>)는 <a href="/wiki/%EB%AA%A8%EB%A6%AC%EC%98%A4%EC%B9%B4_%EB%B2%88" title="모리오카 번">모리오카 번</a>의 2대 번주이다. 통칭은 곤페이(<span lang="ja" xml:lang="ja">権平</span>)이며, 관위는 종5위하, <a class="mw-redirect" href="/wiki/%EC%95%BC%EB%A7%88%EC%8B%9C%EB%A1%9C%EB%85%B8%EC%BF%A0%EB%8B%88" title="야마시로노쿠니">야마시로</a>노카미(<span lang="ja" xml:lang="ja">山城守</span>)이다.</p>
<p>초대 번주 <a href="/wiki/%EB%82%9C%EB%B6%80_%EB%8F%84%EC%8B%9C%EB%82%98%EC%98%A4" title="난부 도시나오">난부 도시나오</a>의 셋째 아들로 <a href="/wiki/%EC%97%90%EB%8F%84" title="에도">에도</a>에서 태어났다. <a href="/wiki/1632%EB%85%84" title="1632년">1632년</a>, 아버지가 사망하면서 그 뒤를 이어 번주가 되었다. 선대부터 시작된 <a href="/wiki/%EB%AA%A8%EB%A6%AC%EC%98%A4%EC%B9%B4_%EC%84%B1" title="모리오카 성">모리오카 성</a> 축성 공사를 완성하였고, 번 통치의 기반을 닦았다. <a href="/wiki/1664%EB%85%84" title="1664년">1664년</a> 음력 9월 12일, 에도에서 59세의 나이로 사망하였는데, 죽기 전에 <a class="mw-redirect" href="/wiki/%EC%87%BC%EA%B5%B0" title="쇼군">쇼군</a> <a href="/wiki/%EB%8F%84%EC%BF%A0%EA%B0%80%EC%99%80_%EC%9D%B4%EC%97%90%EC%93%B0%EB%82%98" title="도쿠가와 이에쓰나">도쿠가와 이에쓰나</a>에게 후계자의 선정과 가문의 존속에 대한 부탁을 하였다. 하지만 시게나오와 이에쓰나와의 대화 내용이 비밀에 부쳐진 관계로 번 내부에서 후계 문제를 놓고 혼란이 발생했다. 이에쓰나는 이에 따라 중재에 나서, 이모제 <a class="mw-redirect" href="/wiki/%EC%8B%9C%EC%B9%98%EB%85%B8%ED%97%A4_%EC%8B%9C%EA%B2%8C%EB%85%B8%EB%B6%80" title="시치노헤 시게노부">시치노헤 시게노부</a>에게 2만 석을 삭감한 모리오카 8만 석 영지를 주어 가문을 잇게 하고, 또다른 이모제 <a class="mw-redirect" href="/wiki/%EB%82%98%EC%B9%B4%EC%82%AC%ED%86%A0_%EB%82%98%EC%98%A4%EC%9A%94%EC%8B%9C" title="나카사토 나오요시">나카사토 나오요시</a>에게 하치노헤 2만 석 영지를 주어 사실상의 분할 상속을 행하였다.</p>
