<p>《<b>난징의 강간: 제2차 세계대전의 잊혀진 홀로코스트</b>》(<span lang="en" xml:lang="en">The Rape of Nanking: The Forgotten Holocaust of World War II</span>)는 <a href="/wiki/%EC%95%84%EC%9D%B4%EB%A6%AC%EC%8A%A4_%EC%9E%A5" title="아이리스 장">아이리스 장</a>이 <a href="/wiki/%EB%82%9C%EC%A7%95_%EB%8C%80%ED%95%99%EC%82%B4" title="난징 대학살">난징 대학살</a>에 관해 쓴 논픽션 책이다. 《난징의 강간》으로도 불리며 <a href="/wiki/1997%EB%85%84" title="1997년">1997년</a>에 <a href="/wiki/%EB%B2%A0%EC%8A%A4%ED%8A%B8%EC%85%80%EB%9F%AC" title="베스트셀러">베스트셀러</a>가 되었다. 난징 대학살은 <a href="/wiki/1937%EB%85%84" title="1937년">1937년</a>과 <a href="/wiki/1938%EB%85%84" title="1938년">1938년</a> 사이에 <a href="/wiki/%EC%A4%91%EC%9D%BC_%EC%A0%84%EC%9F%81" title="중일 전쟁">중일 전쟁</a> 동안 <a href="/wiki/%EC%A4%91%ED%99%94%EB%AF%BC%EA%B5%AD" title="중화민국">중화민국</a>의 수도인 <a class="mw-redirect" href="/wiki/%EB%82%9C%EC%A7%95" title="난징">난징</a>을 <a href="/wiki/%EC%9D%BC%EB%B3%B8%EA%B5%B0" title="일본군">일본군</a>이 함락한 이후, 일본군이 자행한 대학살과 잔혹한 범죄 행위를 칭한다. 저자는 난징 대학살에 관한 자료조사를 바탕으로 책을 서술했다. 저자는 책을 통해 <a class="mw-redirect" href="/wiki/%EC%9D%BC%EB%B3%B8_%EC%A0%95%EB%B6%80" title="일본 정부">일본 정부</a>가 범죄를 부끄러워할 만큼 충분히 사과하지 않았다는 관점을 제시한다. 《난징의 강간》은 난징 대학살을 서구에 처음으로 소개한 영어로 쓰인 중요한 책 중 하나로 평가되며, 여러 언어로 번역되었다. <a href="/wiki/%EB%8C%80%ED%95%9C%EB%AF%BC%EA%B5%AD" title="대한민국">대한민국</a>에서는 윤치환의 번역으로 미다스 북스에서 《역사는 힘있는 자가 쓰는가 - 난징의 강간, 그 진실의 기록》라는 제목으로 <a href="/wiki/2006%EB%85%84" title="2006년">2006년</a> <a href="/wiki/5%EC%9B%94" title="5월">5월</a> 재출판되었다.<sup class="reference" id="cite_ref-1"><a href="#cite_note-1">[1]</a></sup></p>
<p>책은 아이리스 장에게 명성과 논란을 가져다 주었다. 뿐만 아니라 책 자체에 대해서도 대중과 학계로부터 칭송과 비판을 받았다. "일본이 자행한 짓을 서술한 과거의 어떠한 책보다 분명하게 보여주는 책"이라는 긍정적 평가가 있었지만<sup class="reference" id="cite_ref-KirbyForward_2-0"><a href="#cite_note-KirbyForward-2">[2]</a></sup>, 반면 "심각하게 결점이 많다", "오해와 경솔한 설명으로 가득하다."라는 부정적 평가도 있었다.<sup class="reference" id="cite_ref-.ED.8F.AC.EA.B2.94_3-0"><a href="#cite_note-.ED.8F.AC.EA.B2.94-3">[3]</a></sup> 이 책의 업적으로는 난징 대학살 동안 중국시민들을 보호하기 위해 지정된 지역인 <a class="mw-redirect" href="/wiki/%EB%82%9C%EC%A7%95%EC%95%88%EC%A0%84%EC%A7%80%EA%B5%AC" title="난징안전지구">난징안전지구</a>(<small><a href="/wiki/%EC%A4%91%EA%B5%AD%EC%96%B4" title="중국어">중국어</a></small> <small><a href="/wiki/%EA%B0%84%EC%B2%B4%EC%9E%90" title="간체자">간체</a>:</small> <span lang="zh-hans" xml:lang="zh-hans">南京安全区</span>, <small><a href="/wiki/%EC%A0%95%EC%B2%B4%EC%9E%90" title="정체자">정체</a>:</small> <span lang="zh-hant" xml:lang="zh-hant">南京安全區</span>, <small><a href="/wiki/%ED%95%9C%EC%96%B4_%EB%B3%91%EC%9D%8C" title="한어 병음">병음</a>:</small> <font face="Lucida Sans Unicode">Nánjīng Anquán Qǖ</font>)를 설립하는데 중요한 역할을 담당했던 <a href="/wiki/%EC%9A%98_%EB%9D%BC%EB%B2%A0" title="욘 라베">욘 라베</a>(<span lang="de" xml:lang="de">John Rabe</span>)와 <a href="/wiki/%EB%AF%B8%EB%8B%88_%EB%B3%B4%ED%8A%B8%EB%A6%B0" title="미니 보트린">미니 보트린</a>(<span lang="en" xml:lang="en">Minnie Vautrin</span>)의 일기를 발견한 것을 <sup class="reference" id="cite_ref-SFGate2005-04-17_4-0"><a href="#cite_note-SFGate2005-04-17-4">[4]</a></sup> 들 수 있다.</p>

<h2><span class="mw-headline" id=".EA.B3.84.EA.B8.B0">계기</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%82%9C%EC%A7%95%EC%9D%98_%EA%B0%95%EA%B0%84&amp;action=edit&amp;section=1" title="부분 편집: 계기">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>그녀의 부모는 <a href="/wiki/%EC%A0%9C2%EC%B0%A8_%EC%84%B8%EA%B3%84_%EB%8C%80%EC%A0%84" title="제2차 세계 대전">제2차 세계 대전</a> 당시 <a href="/wiki/%EC%A4%91%ED%99%94%EB%AF%BC%EA%B5%AD" title="중화민국">중화민국</a>에서 도망쳐, <a href="/wiki/%ED%83%80%EC%9D%B4%EC%99%84" title="타이완">타이완</a> 섬을 거쳐 <a href="/wiki/%EB%AF%B8%EA%B5%AD" title="미국">미국</a>으로 간 이민자였다. 그녀는 부모로부터 <a class="mw-redirect" href="/wiki/%EB%82%9C%EC%A7%95%EB%8C%80%ED%95%99%EC%82%B4" title="난징대학살">난징대학살</a> 당시 <a href="/wiki/%EC%9D%BC%EB%B3%B8%EC%9D%B8" title="일본인">일본인</a>들이 "아이를 반토막이 아닌 세, 네토막으로 토막내었다."는 말을 들었다. 그녀는 《난징의 강간》의 서두에서 "어린 시절, 난징 대학살은 말을 해서는 안되는 악마처럼 마음 속 깊이 묻혀져 있었다."라고 말하였다. 학교에 있는 지역 공공 도서관에서 자료를 찾아 보았음에도 불구하고 아무것도 발견할 수 없자, 그녀는 왜 아무도 <a href="/wiki/%EB%82%9C%EC%A7%95_%EB%8C%80%ED%95%99%EC%82%B4" title="난징 대학살">난징 대학살</a>에 관한 책을 쓰지 않았는지 궁금했다고 했다.<sup class="reference" id="cite_ref-RapeOfNanking7-8_6-0"><a href="#cite_note-RapeOfNanking7-8-6">[6]</a></sup></p>
<p>거의 20년이 지난 후, <a href="/wiki/%EB%82%9C%EC%A7%95_%EB%8C%80%ED%95%99%EC%82%B4" title="난징 대학살">난징 대학살</a>에 관한 <a class="mw-redirect" href="/wiki/%EB%8B%A4%ED%81%90%EB%A9%98%ED%84%B0%EB%A6%AC" title="다큐멘터리">다큐멘타리 영화</a>를 완성한 제작자들을 알게 된 것을 계기로 <a href="/wiki/%EB%82%9C%EC%A7%95_%EB%8C%80%ED%95%99%EC%82%B4" title="난징 대학살">난징 대학살</a>은 그녀의 삶 속으로 다시 들어오게 된다. 제작자들 중 한 명은 샤오 추핑(<span lang="en" xml:lang="en">Shao Tzuping</span>)으로 <a href="/wiki/%EC%84%A0%EA%B5%90%EC%82%AC" title="선교사">선교사</a>인 <a class="new" href="/w/index.php?title=%EC%A1%B4_%EB%A7%A4%EA%B8%B0&amp;action=edit&amp;redlink=1" title="존 매기 (없는 문서)">존 매기</a>(<span lang="en" xml:lang="en">John Magee</span>)가 찍은 난징 대학살의 현장 영상이 들어간 《매기의 증언(<span lang="en" xml:lang="en">Magee's Testament</span>)》이란 영화를 제작하는 데 도움을 주었다.<sup class="reference" id="cite_ref-.ED.94.84.EB.A6.B0.EC.8A.A4.ED.84.B4_7-0"><a href="#cite_note-.ED.94.84.EB.A6.B0.EC.8A.A4.ED.84.B4-7">[7]</a></sup> 다른 제작자는 낸시 통(<span lang="en" xml:lang="en">Nancy Tong</span>)으로 크리스틴 초이(<span lang="en" xml:lang="en">Christine Choy</span>)와 함께 공동으로 <a href="/wiki/%EC%A4%91%EA%B5%AD" title="중국">중국</a>, <a href="/wiki/%EB%AF%B8%EA%B5%AD" title="미국">미국</a>, <a href="/wiki/%EC%9D%BC%EB%B3%B8" title="일본">일본</a> 시민과의 일련의 인터뷰를 담은 《황제의 이름으로 (<span lang="en" xml:lang="en">In The Name of the Emperor</span>)》라는 영화를 제작하고 감독하였다.<sup class="reference" id="cite_ref-.ED.94.84.EB.A6.B0.EC.8A.A4.ED.84.B4_7-1"><a href="#cite_note-.ED.94.84.EB.A6.B0.EC.8A.A4.ED.84.B4-7">[7]</a></sup> 샤오와 통과 이야기하기 시작하면서, 그녀는 곧 <a href="/wiki/%EB%82%9C%EC%A7%95_%EB%8C%80%ED%95%99%EC%82%B4" title="난징 대학살">난징 대학살</a>을 공론화하고 문서화할 필요성을 느끼는 행동주의자들의 모임과 연결되게 되었다.<sup class="reference" id="cite_ref-RapeOfNanking8-9_8-0"><a href="#cite_note-RapeOfNanking8-9-8">[8]</a></sup> 1994년 12월, 아이리스 장은 <a href="/wiki/%EC%BA%98%EB%A6%AC%ED%8F%AC%EB%8B%88%EC%95%84_%EC%A3%BC" title="캘리포니아 주">캘리포니아 주</a> <a href="/wiki/%EC%BF%A0%ED%8D%BC%ED%8B%B0%EB%85%B8" title="쿠퍼티노">쿠퍼티노</a>에서 열린 난징 대학살에 관한 <a class="mw-disambig" href="/wiki/%EC%BD%98%ED%8D%BC%EB%9F%B0%EC%8A%A4" title="콘퍼런스">콘퍼런스</a>에 참석하였고, 그곳에서 본 것과 들은 것이 《난징의 강간》을 쓰도록 자극하였다.<sup class="reference" id="cite_ref-Penguin_9-0"><a href="#cite_note-Penguin-9">[9]</a></sup> 그녀는 책 서두에서 "그 콘퍼런스에 있는 동안 누군가 세상이 이 대학살을 기억하도록 힘쓰지 않는 한, 생명에 대한 끔찍할 정도의 경시와 인간 사회의 발전에 대한 명백한 퇴보가 한낱 역사책의 주석으로 남아 마치 문제를 일으킬 수도, 없을 수도 있는 컴퓨터 프로그램상의 사소한 문제처럼 다뤄질 지도 모르겠다는 생각이 나를 극심한 공포에 시달리게 했다"라고 말하였다.<sup class="reference" id="cite_ref-RapeOfNanking10_10-0"><a href="#cite_note-RapeOfNanking10-10">[10]</a></sup></p>
<h2><span class="mw-headline" id=".EC.A1.B0.EC.82.AC">조사</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%82%9C%EC%A7%95%EC%9D%98_%EA%B0%95%EA%B0%84&amp;action=edit&amp;section=2" title="부분 편집: 조사">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>아이리스 장은 《난징의 강간》을 쓰기 위해서 조사와 자료를 수집하는 데 2년간의 시간을 보냈다.<sup class="reference" id="cite_ref-SFGate2005-04-17_4-1"><a href="#cite_note-SFGate2005-04-17-4">[4]</a></sup> 그녀는 미국에서 난징 대학살 기간 동안에 <a class="mw-redirect" href="/wiki/%EB%82%9C%EC%A7%95" title="난징">난징</a>에 있었던 <a href="/wiki/%EB%AF%B8%EA%B5%AD%EC%9D%B8" title="미국인">미국인</a> <a href="/wiki/%EC%84%A0%EA%B5%90%EC%82%AC" title="선교사">선교사</a>와 <a href="/wiki/%EC%A0%80%EB%84%90%EB%A6%AC%EC%8A%A4%ED%8A%B8" title="저널리스트">저널리스트</a>, <a class="new" href="/w/index.php?title=%EA%B5%B0%EC%82%AC_%EC%9A%94%EC%9B%90&amp;action=edit&amp;redlink=1" title="군사 요원 (없는 문서)">군사 요원</a>의 일기와 영화, 사진 등 생생한 자료를 발견하였다.<sup class="reference" id="cite_ref-RapeOfNanking11_11-0"><a href="#cite_note-RapeOfNanking11-11">[11]</a></sup> 그녀는 난징 대학살에서 살아남은 이들과 인터뷰와 중국인들이 바라보는 관점과, <a href="/wiki/%EC%9D%BC%EB%B3%B8%EC%9D%B8" title="일본인">일본인</a> <a class="new" href="/w/index.php?title=%ED%87%B4%EC%97%AD%EB%B3%91&amp;action=edit&amp;redlink=1" title="퇴역병 (없는 문서)">퇴역병</a>의 고백을 읽기 위해서 난징으로 갔다.<sup class="reference" id="cite_ref-SFGate1998_12-0"><a href="#cite_note-SFGate1998-12">[12]</a></sup> 그러나 장은 <a href="/wiki/%EC%9D%BC%EB%B3%B8" title="일본">일본</a>에서 조사를 하지는 않았으며, 이는 일본이 <a href="/wiki/%EC%A0%9C2%EC%B0%A8_%EC%84%B8%EA%B3%84_%EB%8C%80%EC%A0%84" title="제2차 세계 대전">제2차 세계 대전</a>의 과거를 어떻게 다루었는지에 관한 문맥 안에서, 그녀가 근대 일본을 어떻게 묘사했는가에 대한 비판에 공격받기 쉽게 되었다.<sup class="reference" id="cite_ref-SFGate1998_12-1"><a href="#cite_note-SFGate1998-12">[12]</a></sup></p>
<p>아이리스 장의 조사는 <a href="/wiki/%EC%83%8C%ED%94%84%EB%9E%80%EC%8B%9C%EC%8A%A4%EC%BD%94_%ED%81%AC%EB%A1%9C%EB%8B%88%ED%81%B4" title="샌프란시스코 크로니클">샌프란시스코 크로니클</a> 기사에서 일본군의 침략 기간 동안 생명을 구하기 위해 주도적으로 나선 두 <a class="new" href="/w/index.php?title=%EC%84%9C%EA%B5%AC%EC%9D%B8&amp;action=edit&amp;redlink=1" title="서구인 (없는 문서)">서구인</a>의 일기의 형식을 빌어, 난징 대학살을 주제로 한 "놀라운 발견"이라고 일컬었다.<sup class="reference" id="cite_ref-SFGate2005-04-17_4-2"><a href="#cite_note-SFGate2005-04-17-4">[4]</a></sup> 첫 번째 일기는 <a href="/wiki/%EB%8F%85%EC%9D%BC" title="독일">독일</a> <a class="mw-redirect" href="/wiki/%EB%82%98%EC%B9%98_%EB%8B%B9" title="나치 당">나치 당</a>의 일원이자, 비군사지역인 <a class="mw-redirect" href="/wiki/%EB%82%9C%EC%A7%95%EC%95%88%EC%A0%84%EC%A7%80%EA%B5%AC" title="난징안전지구">난징안전지구</a>의 지도자였던 <a href="/wiki/%EC%9A%98_%EB%9D%BC%EB%B2%A0" title="욘 라베">욘 라베</a>의 것으로, 그 곳은 난징에서 중국 시민들을 보호하기 위해서 라베와 다른 서구인들이 설치한 것이었다.<sup class="reference" id="cite_ref-13"><a href="#cite_note-13">[13]</a></sup> 다른 일기는 미국의 선교사였던 <a href="/wiki/%EB%AF%B8%EB%8B%88_%EB%B3%B4%ED%8A%B8%EB%A6%B0" title="미니 보트린">미니 보트린</a>으로 <a class="new" href="/w/index.php?title=%EC%A7%84%EB%A7%81_%EB%8C%80%ED%95%99%EA%B5%90&amp;action=edit&amp;redlink=1" title="진링 대학교 (없는 문서)">진링 대학교</a>안에 <a class="new" href="/w/index.php?title=%EB%8C%80%ED%94%BC%EC%86%8C&amp;action=edit&amp;redlink=1" title="대피소 (없는 문서)">대피소</a>를 제공하면서, 약 만 명의 여자와 어린 아이의 생명을 살렸다.<sup class="reference" id="cite_ref-.EB.B3.B4.ED.8A.B8.EB.A6.B0_14-0"><a href="#cite_note-.EB.B3.B4.ED.8A.B8.EB.A6.B0-14">[14]</a></sup> 이 일기들은 저자의 시점으로 난징 대학살의 사건들을 보고하며, 그들이 보았던 <a href="/wiki/%EC%9D%BC%EB%B3%B8%EA%B5%B0" title="일본군">일본군</a>의 잔혹한 행위를 자세히 설명할 뿐만 아니라, 난징안전지구의 둘러싼 환경에 대한 정보를 제공하였다. 아이리스 장은 라베를 난징의 <a class="mw-redirect" href="/wiki/%EC%98%A4%EC%8A%A4%EC%B9%B4_%EC%89%B0%EB%93%A4%EB%9F%AC" title="오스카 쉰들러">오스카 쉰들러</a>로, 보트린을 난징의 <a href="/wiki/%EC%95%88%EB%84%A4_%ED%94%84%EB%9E%91%ED%81%AC" title="안네 프랑크">안네 프랑크</a>라 칭했다.<sup class="reference" id="cite_ref-SFGate2005-04-17_4-3"><a href="#cite_note-SFGate2005-04-17-4">[4]</a></sup> 라베의 일기는 800쪽이 넘으며, 난징 대학살의 가장 상세히 묘사한 설명 중 하나를 포함한다.<sup class="reference" id="cite_ref-15"><a href="#cite_note-15">[15]</a></sup> 라베의 일기는 영어로 번역되어, <a href="/wiki/1998%EB%85%84" title="1998년">1998년</a> <a class="mw-redirect" href="/wiki/%EB%9E%9C%EB%8D%A4_%ED%95%98%EC%9A%B0%EC%8A%A4" title="랜덤 하우스">랜덤 하우스</a>에서 그 자체로 출간되며, 《난징의 선한 이: 욘 라베의 일기》라 제목이 달렸다.<sup class="reference" id="cite_ref-16"><a href="#cite_note-16">[16]</a></sup> 보트린의 일기는 <a href="/wiki/%EB%82%9C%EC%A7%95_%EB%8C%80%ED%95%99%EC%82%B4" title="난징 대학살">난징 대학살</a>에 관해 그녀의 개인적인 경험과 감정을 자세히 열거하였다. 그 안에서 초두 부분에 오늘날 이 도시에서 자행되었던 죄악보다 끔직한 <a href="/wiki/%EB%B2%94%EC%A3%84" title="범죄">범죄</a>는 아마도 없을 것이다.라고 적었다.<sup class="reference" id="cite_ref-17"><a href="#cite_note-17">[17]</a></sup> 이 일기는 보트린과 난징 대학살 동안의 그녀의 역할에 관한 전기를 쓰는 데 자료로 사용되었으며, 휴아링 휴이는 《난징 대학살에서 미국의 여신: 미니 보트린의 용기》란 책을 펴냈다.<sup class="reference" id="cite_ref-18"><a href="#cite_note-18">[18]</a></sup></p>
<h2><span class="mw-headline" id=".EC.B1.85">책</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%82%9C%EC%A7%95%EC%9D%98_%EA%B0%95%EA%B0%84&amp;action=edit&amp;section=3" title="부분 편집: 책">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>《난징의 강간》은 3단락의 주요부분으로 구성된다. 첫 번째 부분은 난징 대학살에서 일어난 일들에 중점을 두고, <a href="/wiki/%EC%9D%BC%EB%B3%B8%EA%B5%B0" title="일본군">일본군</a>, 중국인 희생자, 중국 시민을 도우려고 노력했던 서구인들 등 세가지 다른 관점에서 이야기가 전개된다. 두 번째 부분은 대학살에 관해 전후의 반응, 특히 <a href="/wiki/%EB%AF%B8%EA%B5%AD" title="미국">미국</a>과 <a href="/wiki/%EC%9C%A0%EB%9F%BD" title="유럽">유럽</a> 정부에 반응에 중점적으로 썼다. 세 번째 부분은 아이리스 장이 생각하기에 전쟁이 끝난 뒤 수십 년간 대중의 의식구조에서 대학살에 관한 지식에서 멀어진 환경에 대해 측정하는 것에 중점을 둔다.<sup class="reference" id="cite_ref-RapeOfNanking14-15_19-0"><a href="#cite_note-RapeOfNanking14-15-19">[19]</a></sup></p>
<h3><span class="mw-headline" id=".EC.9D.BC.EB.B3.B8.EA.B5.B0.EC.9D.98_.EC.9E.94.ED.98.B9.ED.96.89.EC.9C.84">일본군의 잔혹행위</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%82%9C%EC%A7%95%EC%9D%98_%EA%B0%95%EA%B0%84&amp;action=edit&amp;section=4" title="부분 편집: 일본군의 잔혹행위">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>이 책은 <a href="/wiki/%EB%82%9C%EC%A7%95_%EB%8C%80%ED%95%99%EC%82%B4" title="난징 대학살">난징 대학살</a> 동안 일어난 <a href="/wiki/%EC%82%B4%EC%9D%B8" title="살인">살인</a>, <a href="/wiki/%EA%B3%A0%EB%AC%B8" title="고문">고문</a>, <a href="/wiki/%EA%B0%95%EA%B0%84" title="강간">강간</a>을 상세하게 묘사하였다. 아이리스 장은 산 채로 불태우기, <a href="/wiki/%EC%A0%88%EB%8B%A8" title="절단">절단</a>, 방화로 인한 죽음, 얼음에 의한 죽음, 개에 의한 죽음 등 그곳의 시민에게 가해진 모든 종류의 고문을 나열하고 설명하였다. 그 학살에서 살아남은 한 사람의 고백에 기초하여, 아이리스 장은 일본군인의 한 무리 가운데서 누가 더 빨리 죽일 수 있는지 ‘살인 경진 대회’를 묘사하였다.<sup class="reference" id="cite_ref-RapeOfNanking83-88_20-0"><a href="#cite_note-RapeOfNanking83-88-20">[20]</a></sup> 장은 난징 대학살 기간 동안 일어난 강간에 관하여 쓰기를 "확실히 이는 세계사에서 가장 많은 집단 강간 중 하나였다"라고 적었다. 그녀는 강간을 당한 여성을 약 2만 명에서부터 많게는 8만 명까지 수를 추정하였으며<sup class="reference" id="cite_ref-RapeOfNanking89_21-0"><a href="#cite_note-RapeOfNanking89-21">[21]</a></sup>, <a href="/wiki/%EB%B9%84%EA%B5%AC%EB%8B%88" title="비구니">비구니</a>를 포함하여 모든 계급의 여성들이 강간당했다고 서술한다.<sup class="reference" id="cite_ref-RapeOfNanking90_22-0"><a href="#cite_note-RapeOfNanking90-22">[22]</a></sup> 더욱이 강간은 모든 장소에서 모든 시간에서 벌어졌으며,<sup class="reference" id="cite_ref-RapeOfNanking90-91_23-0"><a href="#cite_note-RapeOfNanking90-91-23">[23]</a></sup> 젊은 여성 늙은 여성 할 것없이 전부 강간당했다.<sup class="reference" id="cite_ref-RapeOfNanking91a_24-0"><a href="#cite_note-RapeOfNanking91a-24">[24]</a></sup> 심지어는 임산부도 예외일 수가 없으며, 집단 강간 뒤에 일본군들은 때때로 <a class="mw-redirect" href="/wiki/%EC%9E%84%EC%82%B0%EB%B6%80" title="임산부">임산부</a>의 배를 갈라서, 여흥을 위해 <a href="/wiki/%ED%83%9C%EC%95%84" title="태아">태아</a>를 잡아째었다.<sup class="reference" id="cite_ref-RapeOfNanking91b_25-0"><a href="#cite_note-RapeOfNanking91b-25">[25]</a></sup> 이 책에 따르면 모든 강간의 희생자들은 여자뿐만 아니라, 남자도 강간당하고, 불쾌한 <a class="mw-redirect" href="/wiki/%EC%84%B1%ED%96%89%EC%9C%84" title="성행위">성행위</a>를 하도록 강요하였다.<sup class="reference" id="cite_ref-RapeOfNanking95a_26-0"><a href="#cite_note-RapeOfNanking95a-26">[26]</a></sup> 일부는 <a href="/wiki/%EA%B7%BC%EC%B9%9C%EC%83%81%EA%B0%84" title="근친상간">근친상간</a>을 하도록 강요되어서, 아버지가 딸을 강간하고, 남자 형제가 여자 형제를 아들이 어머니를 강간하도록 강요하였다.<sup class="reference" id="cite_ref-RapeOfNanking95b_27-0"><a href="#cite_note-RapeOfNanking95b-27">[27]</a></sup></p>
<h3><span class="mw-headline" id=".EC.82.AC.EB.A7.9D.EC.9E.90_.EC.88.98">사망자 수</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%82%9C%EC%A7%95%EC%9D%98_%EA%B0%95%EA%B0%84&amp;action=edit&amp;section=5" title="부분 편집: 사망자 수">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>아이리스 장은 여러 자료에 의거해 측정한 사망자 수를 적었다. <a href="/wiki/%EC%A4%91%EA%B5%AD" title="중국">중국</a> <a class="new" href="/w/index.php?title=%EA%B5%B0%EC%82%AC_%EC%A0%84%EB%AC%B8%EA%B0%80&amp;action=edit&amp;redlink=1" title="군사 전문가 (없는 문서)">군사 전문가</a>인 리우 팡추는 43만 명으로 추정하였고, 1946년 난징 지역 재판소의 <a class="mw-redirect" href="/wiki/%EA%B2%80%EC%82%AC_(%EB%B2%95)" title="검사 (법)">검사</a>와 <a class="new" href="/w/index.php?title=%EB%82%9C%EC%A7%95_%EB%8C%80%ED%95%99%EC%82%B4_%EA%B8%B0%EB%85%90_%ED%99%80&amp;action=edit&amp;redlink=1" title="난징 대학살 기념 홀 (없는 문서)">난징 대학살 기념 홀</a>의 관리들은 적어도 30만 명이 살해되었다고 주장하며, <a class="mw-redirect" href="/wiki/%EA%B7%B9%EB%8F%99_%EA%B5%AD%EC%A0%9C_%EA%B5%B0%EC%82%AC_%EC%9E%AC%ED%8C%90" title="극동 국제 군사 재판">극동 국제 군사 재판</a>의 심판관들은 26만 명 이상이 살해되었으며, 일본의 역사학자인 <a class="new" href="/w/index.php?title=%ED%9B%84%EC%A7%80%EC%99%80%EB%9D%BC_%EC%95%84%ED%82%A4%EB%9D%BC&amp;action=edit&amp;redlink=1" title="후지와라 아키라 (없는 문서)">후지와라 아키라</a>는 대략 20만 명으로, "체계적으로 계산한 적이 없고, 2월에 난징을 떠났던 <a href="/wiki/%EC%9A%98_%EB%9D%BC%EB%B2%A0" title="욘 라베">욘 라베</a>"는 고작 5만 명에서 6만 명으로 추정하였으며, 일본의 저자 <a class="new" href="/w/index.php?title=%EC%9D%B4%EC%BF%A0%ED%9E%88%EC%BD%94_%ED%95%98%ED%83%80&amp;action=edit&amp;redlink=1" title="이쿠히코 하타 (없는 문서)">이쿠히코 하타</a>는 사망자가 3만 8천명에서 4만 2천명이라 주장하였다.<sup class="reference" id="cite_ref-RapeOfNanking100_28-0"><a href="#cite_note-RapeOfNanking100-28">[28]</a></sup></p>
<p>이 책은 <a class="new" href="/w/index.php?title=%EC%A7%80%EC%95%99%EC%88%98_%EC%82%AC%ED%9A%8C%EA%B3%BC%ED%95%99_%EC%95%84%EC%B9%B4%EB%8D%B0%EB%AF%B8&amp;action=edit&amp;redlink=1" title="지앙수 사회과학 아카데미 (없는 문서)">지앙수 사회과학 아카데미</a>의 <a class="mw-redirect" href="/wiki/%EC%97%AD%EC%82%AC%ED%95%99%EC%9E%90" title="역사학자">역사학자</a> 순 자이웨이의 연구를 다룬다. <a href="/wiki/1990%EB%85%84" title="1990년">1990년</a> 《난징 대학살과 난징의 인구》라는 제목의 <a href="/wiki/%EB%85%BC%EB%AC%B8" title="논문">논문</a>에서 순은 살해된 총 인원을 377,400이라 측정하였다. 중국인의 <a href="/wiki/%EB%AC%B4%EB%8D%A4" title="무덤">무덤</a>수 기록을 사용하면서, 그는 사망자가 227,400를 넘는다고 측정하였다. 그는 <a href="/wiki/%EC%9D%BC%EB%B3%B8%EA%B5%B0" title="일본군">일본군</a> <a href="/wiki/%EC%86%8C%EB%A0%B9" title="소령">소령</a>인 <a class="new" href="/w/index.php?title=%EC%98%A4%ED%83%80_%ED%9E%88%EC%82%AC%EC%98%A4&amp;action=edit&amp;redlink=1" title="오타 히사오 (없는 문서)">오타 히사오</a>가 시체 처분을 위한 일본군의 노력에 대해 고백하는 보고서에서 총 십만 오천명을 측정하고 더하여, 총 377,400명이라고 추정하였다.<sup class="reference" id="cite_ref-RapeOfNanking101_29-0"><a href="#cite_note-RapeOfNanking101-29">[29]</a></sup></p>
<p>장은 그 당시 일본인 그들 스스로가 사망자 수가 최대 30만 명일 것이라고 믿었다는 강력한 증거가 있다고 적었다. 그녀는 <a class="new" href="/w/index.php?title=%EC%9D%BC%EB%B3%B8_%EC%99%B8%EB%AC%B4%EB%B6%80&amp;action=edit&amp;redlink=1" title="일본 외무부 (없는 문서)">일본 외무부</a> <a href="/wiki/%EC%9E%AC%EC%83%81" title="재상">재상</a>인 히로타 코키가 난징 대학살이 일어난 첫 번째 달인 <a href="/wiki/1938%EB%85%84" title="1938년">1938년</a> <a href="/wiki/1%EC%9B%94_17%EC%9D%BC" title="1월 17일">1월 17일</a> <a class="mw-redirect" href="/wiki/%EC%9B%8C%EC%8B%B1%ED%84%B4_%EB%94%94%EC%94%A8" title="워싱턴 디씨">워싱턴 디씨</a>에서 그의 연락책에게 중계한 전갈을 인용하였다. 이 전갈은 "적어도 30만 명의 중국인 시민이 학살되었고, 많은 경우에서 냉혈하게 다루었다는 것을 인지한다.<sup class="reference" id="cite_ref-RapeOfNanking103-104_30-0"><a href="#cite_note-RapeOfNanking103-104-30">[30]</a></sup></p>
<h2><span class="mw-headline" id=".EC.B9.AD.EC.86.A1">칭송</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%82%9C%EC%A7%95%EC%9D%98_%EA%B0%95%EA%B0%84&amp;action=edit&amp;section=6" title="부분 편집: 칭송">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>《난징의 강간》은 미국에서 처음으로 출판된 뒤 50만부 이상이 팔렸으며, <a class="mw-redirect" href="/wiki/%EB%89%B4%EC%9A%95_%ED%83%80%EC%9E%84%EC%A6%88" title="뉴욕 타임즈">뉴욕 타임즈</a>에 따르면 일반적인 비평적인 칭송을 받았다.<sup class="reference" id="cite_ref-NewYorkTimes_31-0"><a href="#cite_note-NewYorkTimes-31">[31]</a></sup> <a href="/wiki/%EC%95%84%EC%9D%B4%EB%A6%AC%EC%8A%A4_%EC%9E%A5" title="아이리스 장">아이리스 장</a>은 즉시 미국내에서 <a class="mw-redirect" href="/wiki/%EC%9C%A0%EB%AA%85%EC%9D%B8%EC%82%AC" title="유명인사">유명인사</a>가 되었다.<sup class="reference" id="cite_ref-TimesOnline_32-0"><a href="#cite_note-TimesOnline-32">[32]</a></sup> 그녀는 <a class="new" href="/w/index.php?title=%EB%AA%85%EC%98%88%ED%95%99%EC%9C%84&amp;action=edit&amp;redlink=1" title="명예학위 (없는 문서)">명예학위</a>를 수여받으며,<sup class="reference" id="cite_ref-SFGate2004-11-11_33-0"><a href="#cite_note-SFGate2004-11-11-33">[33]</a></sup> 난징 대학살에 대해 강의를 하거나 〈굿 모닝 아메리카〉, 〈나이트라인〉, 《짐 레러와 함께한 뉴스시간》 등의 <a class="mw-redirect" href="/wiki/%ED%86%A0%ED%81%AC%EC%87%BC" title="토크쇼">토크쇼</a>에서 토론하는 자리에 초대를 받으며, <a class="mw-redirect" href="/wiki/%EB%89%B4%EC%9A%95_%ED%83%80%EC%9E%84%EC%A6%88" title="뉴욕 타임즈">뉴욕 타임즈</a>의 인물소개에 오르며, <a href="/wiki/%EB%A6%AC%EB%8D%94%EC%8A%A4_%EB%8B%A4%EC%9D%B4%EC%A0%9C%EC%8A%A4%ED%8A%B8" title="리더스 다이제스트">리더스 다이제스트</a> 표지에 실리기도 하였다.<sup class="reference" id="cite_ref-Penguin_9-1"><a href="#cite_note-Penguin-9">[9]</a></sup> 그 책은 뉴욕타임즈 베스트셀러 목록에서 10주간 있으며, 4개월 동안 12만 5천부 이상이 팔렸다. <a href="/wiki/%ED%9E%90%EB%9F%AC%EB%A6%AC_%ED%81%B4%EB%A6%B0%ED%84%B4" title="힐러리 클린턴">힐러리 클린턴</a>은 그녀를 <a href="/wiki/%EB%B0%B1%EC%95%85%EA%B4%80" title="백악관">백악관</a>으로 초대하여, 미국의 역사학자인 스티븐 앰브로즈는 "우리가 만난 젊은 역사학자 중 가장 최고일 지도 모른다"<sup class="reference" id="cite_ref-TimesOnline_32-1"><a href="#cite_note-TimesOnline-32">[32]</a></sup> 라고 묘사하였으며, 중국계 미국인 협회에서 장을 그 해의 국가의 여성이라 불렀다.<sup class="reference" id="cite_ref-SFGate2004-11-11_33-1"><a href="#cite_note-SFGate2004-11-11-33">[33]</a></sup> 《난징의 강간》의 인기는 긴 책홍보 투어로 촉진하였고, 아이리스 장은 1년 6개월간 65곳의 도시를 방문하였다.<sup class="reference" id="cite_ref-SFGate2005-04-17_4-4"><a href="#cite_note-SFGate2005-04-17-4">[4]</a></sup></p>
<p>이 책은 <a class="new" href="/w/index.php?title=%EB%89%B4%EC%8A%A4_%EB%AF%B8%EB%94%94%EC%96%B4&amp;action=edit&amp;redlink=1" title="뉴스 미디어 (없는 문서)">뉴스 미디어</a>에서 칭송을 얻었다. <a href="/wiki/%EC%9B%94%EC%8A%A4%ED%8A%B8%EB%A6%AC%ED%8A%B8_%EC%A0%80%EB%84%90" title="월스트리트 저널">월스트리트 저널</a>은 이 책은 "처음으로 중국의 제국 도시의 붕괴를 넓게 처음으로 측정하였다."아이리스 장이 끔찍한 사건이 벌어진 것에 <a href="/wiki/%EB%A7%9D%EA%B0%81" title="망각">망각</a>을 솜씨좋게 파헤쳤다라고 말했다. <a class="new" href="/w/index.php?title=%EC%9B%94%EA%B0%84_%EC%95%84%ED%8B%80%EB%9E%9C%ED%8B%B1&amp;action=edit&amp;redlink=1" title="월간 아틀랜틱 (없는 문서)">월간 아틀랜틱</a>은 그 책을‘일본군의 행위를 압도적으로 고발한다’라고 말했다. <a href="/wiki/%EC%8B%9C%EC%B9%B4%EA%B3%A0_%ED%8A%B8%EB%A6%AC%EB%B7%B4" title="시카고 트리뷴">시카고 트리뷴</a>은 “그 책은 역사와 도덕적인 질문에 강력한 새 작품이며, 아이리스 장은 폭력의 차원을 정확하게 설명하기 위해서 대단히 주의를 기울였다.”라고 말했다. <a class="mw-redirect" href="/wiki/%ED%95%84%EB%9D%BC%EB%8D%B8%ED%94%BC%EC%95%84_%EC%9D%B8%EC%BF%BC%EB%9F%AC" title="필라델피아 인쿼러">필라델피아 인쿼러</a>는 “최근까지 거의 잊혀진 끔직한 사건을 칭찬할 만한 해석이며”, “동물도 일본군이 행동했던 식으로 처신하지 않는다”라고 말했다.<sup class="reference" id="cite_ref-MediaPraise_34-0"><a href="#cite_note-MediaPraise-34">[34]</a></sup></p>
<p><a href="/wiki/%ED%95%98%EB%B2%84%EB%93%9C_%EB%8C%80%ED%95%99%EA%B5%90" title="하버드 대학교">하버드 대학교</a> <a href="/wiki/%EC%97%AD%EC%82%AC%ED%95%99" title="역사학">역사학</a> <a href="/wiki/%EA%B5%90%EC%88%98" title="교수">교수</a>인 윌리엄 C. 커크비에 따르면, “아이리스 장은 일본인이 저지른 짓을 고발하는 이전의 어떠한 해석보다 더 분명하게 보여준다. 그리고 제2차 세계대전 동안 수백만의 죄없는 사람들이 유럽에서 학살된 것과 아시아에서 사건을 연결시켰다.”라고 말했다.<sup class="reference" id="cite_ref-KirbyForward_2-1"><a href="#cite_note-KirbyForward-2">[2]</a></sup> <a href="/wiki/%ED%95%98%EB%B2%84%EB%93%9C_%EB%8C%80%ED%95%99%EA%B5%90" title="하버드 대학교">하버드 대학교</a> <a class="new" href="/w/index.php?title=%EB%8F%99%EC%95%84%EC%8B%9C%EC%95%84_%EC%97%B0%EA%B5%AC%EC%86%8C&amp;action=edit&amp;redlink=1" title="동아시아 연구소 (없는 문서)">동아시아 연구소</a>를 위한 페이뱅크 센터의 연구에 관련이 있는 로스 테릴은 “이 책은 학자의 입장에서 놀라운 연구조사이며, 열정을 담은 작품이다”라고 말했다.<sup class="reference" id="cite_ref-35"><a href="#cite_note-35">[35]</a></sup><sup class="reference" id="cite_ref-Quotes_36-0"><a href="#cite_note-Quotes-36">[36]</a></sup> <a href="/wiki/%EC%98%88%EC%9D%BC_%EB%8C%80%ED%95%99%EA%B5%90" title="예일 대학교">예일 대학교</a> <a href="/wiki/%EC%97%AD%EC%82%AC%ED%95%99" title="역사학">역사학과</a>의 <a class="new" href="/w/index.php?title=%EB%AA%85%EC%98%88%EA%B5%90%EC%88%98&amp;action=edit&amp;redlink=1" title="명예교수 (없는 문서)">명예교수</a>인 비아트리스 S. 바틀렛<sup class="reference" id="cite_ref-37"><a href="#cite_note-37">[37]</a></sup> 은 “아이리스 장의 난징 홀로코스트에 대한 연구는 제2차 세계대전의 잔혹행위에 대해 새롭고 넓혀진 이야기를 보여주며, 조사를 반영하였다”.라고 말했다.<sup class="reference" id="cite_ref-Quotes_36-1"><a href="#cite_note-Quotes-36">[36]</a></sup></p>
<h2><span class="mw-headline" id=".EB.B9.84.ED.8C.90">비판</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%82%9C%EC%A7%95%EC%9D%98_%EA%B0%95%EA%B0%84&amp;action=edit&amp;section=7" title="부분 편집: 비판">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p><a class="mw-disambig" href="/wiki/%EC%9A%94%ED%81%AC_%EB%8C%80%ED%95%99%EA%B5%90" title="요크 대학교">요크 대학교</a>의 캐나다 연구소(Canada Research Chair)의 <a class="mw-redirect" href="/wiki/%EB%B9%84%ED%8F%89%EA%B0%80" title="비평가">비평가</a> 조슈아 A. 포겔(Joshua A. Fogel)<sup class="reference" id="cite_ref-38"><a href="#cite_note-38">[38]</a></sup> 은 난징의 강간이 "심각하게 결함이 있으며, 오해와 경솔한 설명으로 가득차있다."라고 주장하였다.<sup class="reference" id="cite_ref-.ED.8F.AC.EA.B2.94_3-1"><a href="#cite_note-.ED.8F.AC.EA.B2.94-3">[3]</a></sup> 그는 아이리스 장이 왜 그 대학살이 일어났는지 설명하려고 할 때, 비록 도입 부분에 그녀가 "일본인의 성질이나 그러한 행위를 자행할 수 있는 사람들의 일반적인 꾸밈에 대해 칭찬할 생각이 조금도 없다"라고 적었지만, 그녀는 <a href="/wiki/%EC%9D%BC%EB%B3%B8%EC%9D%B8" title="일본인">일본인</a>의 '정신상태'를 반복해 언급하는 것에서 그 책이 시작부터 산산조각 난다고 말했다.</p>
<p><a href="/wiki/%EC%8A%A4%ED%83%A0%ED%8D%BC%EB%93%9C_%EB%8C%80%ED%95%99%EA%B5%90" title="스탠퍼드 대학교">스탠퍼드 대학교</a> <a href="/wiki/%EC%97%AD%EC%82%AC%ED%95%99" title="역사학">역사학과</a> 교수인 <a class="new" href="/w/index.php?title=%EB%8D%B0%EC%9D%B4%EB%B9%84%EB%93%9C_M._%EC%BC%80%EB%84%A4%EB%94%94&amp;action=edit&amp;redlink=1" title="데이비드 M. 케네디 (없는 문서)">데이비드 M. 케네디</a>(David M. Kennedy)는 아이리스 장이 일본인의 성격에 관해서 칭찬할 의도가 없음을 적은 반면, 그녀는 "일본인의 정체성"에 대해서 썼다고 주장하였다. 그녀의 평가에서 무술 경진대회, <a href="/wiki/%EC%82%AC%EB%AC%B4%EB%9D%BC%EC%9D%B4" title="사무라이">사무라이</a> 윤리, <a href="/wiki/%EB%AC%B4%EC%82%AC%EB%8F%84" title="무사도">무사도</a>의 공포스러운 전사의 코드로 충만한 피가 흥건함으로 난징으로의 길이 일본 문화의 정수 사이로 흐른다는 추론을 내었다고 지적하였다. 케네디는 분석과 이해보다는 "고발과 분노"가 이 책의 압도적인 동기이며, 비록 분노가 도덕적으로 <a class="mw-redirect" href="/wiki/%EB%82%9C%EC%A7%95%EB%8C%80%ED%95%99%EC%82%B4" title="난징대학살">난징대학살</a>에 필요한 반응이지만, 이 책은 지적으로 불충분하다라고 주장하였다. 그는 아이리스 장의 서구에 무관심에 대한 비난과 일본의 대학살에 대한 거부를 "과장했다"며, 서구세계에서는 실상 그때도 그리고 난징의 강간을 무시한 적이 없으며, 아이리스 장이 일본이 고집스럽게 전쟁기간의 죄악을 인식하기를 거부하고 있으며, 그들 스스로 유감을 표현하도록 했다는 지적이 맞지도 않다고 주장하였다.<sup class="reference" id="cite_ref-Kennedy_39-0"><a href="#cite_note-Kennedy-39">[39]</a></sup></p>
<p>세인트 올라프 대학교(St. Olaf College)의 역사학과 교수인 루버트 엔텐만(Robert Entenmann)은 아이리스 장이 제시한 일본의 역사적 배경은 상투적이고 단순하며, 정형화되고, 종종 부정확하다고 비난하였다.<sup class="reference" id="cite_ref-40"><a href="#cite_note-40">[40]</a></sup> 그는 난징대학살을 향한 근대 일본인들의 반응에 대한 아이리스 장의 해석에 관해서, "그녀는 일부 극렬국수주의 극단론자들들과 다른 일본인들 사이의 구별을 못하는 듯 싶으며, 그녀 자신의 민족적인 편견이 그녀의 책에 암암리에 침범하였다"라고 말했다. 아이리스 장의 대학살에 관한 설명은 비판에 열려있다라고 말하며, 장은 그 대학살이 왜 일어났는지 충분히 설명하지 못한다고 평했다.<sup class="reference" id="cite_ref-41"><a href="#cite_note-41">[41]</a></sup></p>
<p>샌프란시스코 크로니클의 기자인 찰스 부레스(Charles Burress)가 아이리스 장의 책을 비난하는 기사가 1998년 7월에 실렸다.<sup class="reference" id="cite_ref-SFGate1998_12-2"><a href="#cite_note-SFGate1998-12">[12]</a></sup> <a href="/wiki/%EC%9D%BC%EB%B3%B8" title="일본">일본</a> <a class="mw-redirect" href="/wiki/%EB%8B%88%ED%98%BC%EB%8C%80%ED%95%99%EA%B5%90" title="니혼대학교">니혼대학교</a>의 역사학 교수인 하타 이쿠히코(秦郁彦)는 책 속의 11개의 사진이 잘못 제시되거나, 가짜라고 주장하였다. 예를 들어 여자들과 아이들이 일본군들과 다리를 따라 건너는 한 사진에서, "일본군들이 수천명의 여성을 소집하고, 대부분이 집단강간을 당하거나, 군대의 성노예가 되도록 강요되었다."라고 주석을 달았다. 하타는 이 사진이 원래 1937년 일본의 한 신문에 개재된 것으로, 이 사진의 연작의 일부로 일본의 점령기간 동안 중국 마을 사람들의 평화로운 장면을 보여주고 있다고 항변하였다. 부레스는 또한 그 자신의 분석을 제공하면서, 책안에서 보여지는 실수에 대해 지적하였다. 그는 아이리스 장이 인용한 1938년 <a href="/wiki/%EC%9D%BC%EB%B3%B8" title="일본">일본</a>의 <a class="new" href="/w/index.php?title=%EC%99%B8%EB%AC%B4%EB%B6%80_%EC%88%98%EC%83%81&amp;action=edit&amp;redlink=1" title="외무부 수상 (없는 문서)">외무부 수상</a>에 의해 보내진 비밀 전보는 난징에서 일본 시민들이 적어도 30만 명을 죽였을 것이라는 "강력한 증거"로 인용하는 것은 부정확하다고 말했다.<sup class="reference" id="cite_ref-SFGate1998_12-3"><a href="#cite_note-SFGate1998-12">[12]</a></sup> 부레스의 따르면, 죽음을 당한 30만 명의 중국 시민들의 숫자는 실상 <a href="/wiki/%EC%98%81%EA%B5%AD" title="영국">영국인</a> <a href="/wiki/%EA%B8%B0%EC%9E%90_(%EC%A7%81%EC%97%85)" title="기자 (직업)">기자</a>에 의해 보내진 전갈로, 난징 뿐만 아니라, 다른 장소에서 살해된 사람수를 포함한 것에서 온 것이라고 말했다. 부레스는 부가적으로 그글을 쓰기 위한 그녀의 동기가 행동주의자로서 그 책을 쓴 것인지, 역사가의 입장에서 쓴 것인지에 여부에 질문을 던지며, 난징의 강간은 그녀의 선고로부터 세상에서 난징대학살이 잊혀지지 않도록 감정적인 자극을 그려냈다고 비판하였다.<sup class="reference" id="cite_ref-SFGate1998_12-4"><a href="#cite_note-SFGate1998-12">[12]</a></sup></p>
<h3><span class="mw-headline" id=".EB.B9.84.ED.8C.90.EC.97.90_.EB.8C.80.ED.95.9C_.EC.95.84.EC.9D.B4.EB.A6.AC.EC.8A.A4_.EC.9E.A5.EC.9D.98_.EB.B0.98.EC.9D.91">비판에 대한 아이리스 장의 반응</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%82%9C%EC%A7%95%EC%9D%98_%EA%B0%95%EA%B0%84&amp;action=edit&amp;section=8" title="부분 편집: 비판에 대한 아이리스 장의 반응">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>아이리스 장은 샌프란시스코 크로니클에 편지를 보내 부레스의 비판에 응답하려고 시도했지만, 그 편지는 그 신문사에 실리지 않았다.<sup class="reference" id="cite_ref-42"><a href="#cite_note-42">[42]</a></sup> 그 편지에서 그녀는 부레스의 기사를 그녀 스스로 관계하는 비판을 제시하였다.<sup class="reference" id="cite_ref-ChangLetter_43-0"><a href="#cite_note-ChangLetter-43">[43]</a></sup></p>
<p>아이리스 장은 부레스의 비판이 "<a href="/wiki/%EA%B7%B9%EC%9A%B0" title="극우">극우</a> 일본 평론가들의 억측을 뒷받침할 만한 증거 요구도 없이" 그들의 말을 인용하는 "불온한 경향"이라고 발견하였다. 더욱이 그녀는 부레스가 인용한 참고인인 하타 이쿠히코는 일본에서나 미국에서 명망있는 학자로 여겨지지 않으며, 그가 "극렬우파" 일본 출판지 등에 고정적인 기고자이기 때문이라고 지적하였다. 그러한 극렬우파의 한 일본 출판지는 한 홀로코스트를 부인하는 사람으로부터 기사를 실으며, <a href="/wiki/%EB%8F%85%EC%9D%BC" title="독일">독일</a>에서 <a class="mw-redirect" href="/wiki/%EC%9C%A0%ED%83%9C%EC%9D%B8" title="유태인">유태인</a>들을 죽일 때 <a href="/wiki/%EA%B0%80%EC%8A%A4%EC%8B%A4" title="가스실">가스실</a>이 없었다고 주장하였다. 그녀의 부정확한 사진 주석에 관한 부레스의 비판에, 아이리스 장은 그 주석이 잘못되었다는 말다툼에 논박하였다. 그녀는 자신의 책이 일본이 중국을 침략한 공포를 다루고 있으며, ""일본군들이 수천명의 여성을 소집하고, 대부분이 집단강간을 당하거나, 군대의 성노예가 되도록 강요되었다"는 주석은 논박할 수 없는 사실의 두가지 설명을 담고 있다고 반박하였다. 장은 일본 외교부 수상이 보낸 전보의 부정확한 인용이라는 부레스의 주장에 역시 반박하였다. 그녀는 난징에서의 중국인 사상자의 원래 숫자가 한 영국인 기자에게 보고되었지만, 그 수는 그 일본 외교부 수상이 워싱턴 D.C에 그의 연락책에게 보내질 때 인용되었다고 말했다. 그러한 숫자가 일본의 고위 정부관료에 의해 사용되었으며, 아이리스 장은 이는 일본 정부가 중국인 사망자 수를 30만 명으로 인식했음을 드러내는 증거라고 말했다. 마지막으로 그녀는 부레스가 난징 대학살의 범위와 크기에서 주위를 벗어나서, 사소한 부분을 트집을 잡는다면서 이러한 글쓰기는 <a href="/wiki/%ED%99%80%EB%A1%9C%EC%BD%94%EC%8A%A4%ED%8A%B8" title="홀로코스트">홀로코스트</a>를 부인하는 사람들의 흔한 수법이라고 비판하였다.<sup class="reference" id="cite_ref-ChangLetter_43-1"><a href="#cite_note-ChangLetter-43">[43]</a></sup></p>
<h2><span class="mw-headline" id=".EC.95.84.EC.9D.B4.EB.A6.AC.EC.8A.A4_.EC.9E.A5.EC.9D.98_.EC.A3.BD.EC.9D.8C">아이리스 장의 죽음</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%82%9C%EC%A7%95%EC%9D%98_%EA%B0%95%EA%B0%84&amp;action=edit&amp;section=9" title="부분 편집: 아이리스 장의 죽음">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>《난징의 강간》은 <a href="/wiki/%EC%95%84%EC%9D%B4%EB%A6%AC%EC%8A%A4_%EC%9E%A5" title="아이리스 장">아이리스 장</a>에게 명성을 가져다 주었으며, <a href="/wiki/%EC%A4%91%EA%B5%AD" title="중국">중국</a>에서는 그녀가 <a href="/wiki/%EC%84%9C%EC%9C%A0%EB%9F%BD" title="서유럽">서유럽</a> 세계에 <a class="mw-redirect" href="/wiki/%EB%82%9C%EC%A7%95%EB%8C%80%ED%95%99%EC%82%B4" title="난징대학살">난징대학살</a>에 관한 인식을 일깨운 점 때문에 좋은 평가를 받았다.<sup class="reference" id="cite_ref-SFGate2004-11-11_33-2"><a href="#cite_note-SFGate2004-11-11-33">[33]</a></sup> 동시에 장은 수많은 <a href="/wiki/%ED%98%91%EB%B0%95" title="협박">협박</a> 편지를, 특히 <a href="/wiki/%EC%9D%BC%EB%B3%B8" title="일본">일본</a> <a class="new" href="/w/index.php?title=%EA%B7%B9%EC%9A%B0%EC%A3%BC%EC%9D%98%EC%9E%90&amp;action=edit&amp;redlink=1" title="극우주의자 (없는 문서)">극우주의자</a>들에게서 받았다. 아이리스 장은 그녀의 차안에서 협박성 쪽지를 발견하고, 그녀의 전화가 <a href="/wiki/%EB%8F%84%EC%B2%AD" title="도청">도청</a>되었다고 생각했다. 그녀는 자신의 작업에 타당성에 관해 제기되는 질문에 흥분조로 응대하곤 하였다.<sup class="reference" id="cite_ref-44"><a href="#cite_note-44">[44]</a></sup> 아이리스 장의 어머니는 “그 책이 아이리스를 슬프게 만들었다.”라고 말했다. 아이리스 장은 <a href="/wiki/%EC%9A%B0%EC%9A%B8%EC%A6%9D" title="우울증">우울증</a>을 겪고, 2004년 8월 “단기 민감성 정신이상”이라고 진단받았다.<sup class="reference" id="cite_ref-SFGate2005-04-17_4-5"><a href="#cite_note-SFGate2005-04-17-4">[4]</a></sup> 아이리스 장은 그녀의 기분을 안정시키기 위해서 약을 복용하기 시작했다. 아이리스 장이 쓰기를</p>
<blockquote>
<p>나는 내가 기운을 되찾고, 후에 내가 상상할 수 없는 더 강력한 압력에 <a href="/wiki/%EB%B0%95%ED%95%B4" title="박해">박해</a>받더라도 나의 믿음을 결코 흔들린 적이 없다. <a class="mw-redirect" href="/wiki/CIA" title="CIA">CIA</a>가 되던, 어떤 다른 조직이던간에 나는 결코 알지 못할 것이다. 내가 살아 있는 한, 이러한 압력들이 사냥개처럼 나를 쫓는 것을 결코 멈추게 할 수 없을 것이다.<sup class="reference" id="cite_ref-SFGate2005-04-17_4-6"><a href="#cite_note-SFGate2005-04-17-4">[4]</a></sup></p>
</blockquote>
<p>그녀는 우울증과 외부 압력의 싸움에 굴복하여, <a href="/wiki/2004%EB%85%84" title="2004년">2004년</a> <a href="/wiki/11%EC%9B%94" title="11월">11월</a> <a href="/wiki/%EC%9E%90%EC%82%B4" title="자살">자살</a>로 추정되는 죽음으로 생을 마감하였다.<sup class="reference" id="cite_ref-SFGate2005-04-17_4-7"><a href="#cite_note-SFGate2005-04-17-4">[4]</a></sup></p>
<p>그녀의 사망 이후 <a href="/wiki/%EC%BA%98%EB%A6%AC%ED%8F%AC%EB%8B%88%EC%95%84_%EC%A3%BC" title="캘리포니아 주">캘리포니아 주</a> <a class="mw-disambig" href="/wiki/%EB%A1%9C%EC%8A%A4%EC%95%8C%ED%86%A0%EC%8A%A4" title="로스알토스">로스알토스</a>에서 거행된 그녀의 <a class="mw-redirect" href="/wiki/%EC%9E%A5%EB%A1%80%EC%8B%9D" title="장례식">장례식</a>과 동시에 중국에서 <a class="mw-redirect" href="/wiki/%EB%82%9C%EC%A7%95%EB%8C%80%ED%95%99%EC%82%B4" title="난징대학살">난징대학살</a>의 희생자들을 추도하기 위해 난징대학살 장소에 세워진 <a class="new" href="/w/index.php?title=%EB%82%9C%EC%A7%95%EB%8C%80%ED%95%99%EC%82%B4_%EA%B8%B0%EB%85%90%ED%99%80&amp;action=edit&amp;redlink=1" title="난징대학살 기념홀 (없는 문서)">난징대학살 기념홀</a>에서 대학살의 <a class="new" 점령한 동안, 마침내 <a href="/wiki/1940%EB%85%84" title="1940년">1940년</a> <a href="/wiki/%EC%8B%A0%EA%B2%BD%EC%87%A0%EC%95%BD" title="신경쇠약">신경쇠약</a>을 겪었다. 보트린은 미국으로 돌아와 의학 치료를 받았지만, 일년 뒤 자살하였다.<sup class="reference" id="cite_ref-.EB.B3.B4.ED.8A.B8.EB.A6.B0_14-1"><a href="#cite_note-.EB.B3.B4.ED.8A.B8.EB.A6.B0-14">[14]</a></sup></p>
<h2><span class="mw-headline" id=".EC.9D.BC.EB.B3.B8.EC.97.90.EC.84.9C.EC.9D.98_.EB.B0.98.EC.9D.91">일본에서의 반응</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%82%9C%EC%A7%95%EC%9D%98_%EA%B0%95%EA%B0%84&amp;action=edit&amp;section=10" title="부분 편집: 일본에서의 반응">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p><a class="mw-redirect" href="/wiki/%EB%A1%9C%EC%8A%A4%EC%95%A4%EC%A0%A4%EB%A0%88%EC%8A%A4_%ED%83%80%EC%9E%84%EC%A6%88" title="로스앤젤레스 타임즈">로스앤젤레스 타임즈</a>의 <a href="/wiki/%EA%B8%B0%EC%9E%90_(%EC%A7%81%EC%97%85)" title="기자 (직업)">기자</a> 소니 에프론(Sonni Efron)<sup class="reference" id="cite_ref-47"><a href="#cite_note-47">[47]</a></sup> 은 <a class="mw-redirect" href="/wiki/%EB%82%9C%EC%A7%95" title="난징">난징</a>에서 대학살이 절대로 일어나지 않았다고 단언하는 <a href="/wiki/%EC%9D%BC%EB%B3%B8%EC%9D%B8" title="일본인">일본인</a> "<a class="mw-redirect" href="/wiki/%EA%B5%AD%EC%88%98%EC%A3%BC%EC%9D%98%EC%9E%90" title="국수주의자">극렬국수주의자</a>"과 함께, "그 대학살이 일어나긴 했지만, 아이리스 장의 흠이 있는 학문이 그들의 논점을 손상시켰다."라고 우겨대는 일본 <a class="mw-redirect" href="/wiki/%EB%AC%B8%EC%9D%B8" title="문인">문인</a>들에게 비난을 받았다.<sup class="reference" id="cite_ref-CHGS_48-0"><a href="#cite_note-CHGS-48">[48]</a></sup> <a class="mw-redirect" href="/wiki/%EB%A6%AC%EC%93%B0%EB%A9%94%EC%9D%B4%EC%B9%B8_%EB%8C%80%ED%95%99%EA%B5%90" title="리쓰메이칸 대학교">리쓰메이칸 아시아·태평양 대학</a>(<span lang="ja" xml:lang="ja">立命館 アジア·太平洋 大学</span>)의 조교수인 데이드 아스키우(David Askew)는 아이리스 장의 책은 제2차 세계 대전 동안 자행한 범죄 때문에 일제의 지도자들을 심판하기 위해 소환된 재판인 도쿄 재판의 타당성의 발견을 홍보하는 ‘대학살 학파’에게 심각한 일격을 가하는 것을 다뤘다고 말했다. 아스키우는 대학살 학파는 더 많은 사망자 수를 주장한 책을 비난하는 (일반적이지 않는) 위치로 밀려났다고 주장하였다.<sup class="reference" id="cite_ref-Askew_49-0"><a href="#cite_note-Askew-49">[49]</a></sup></p>
<p>이 책은 2007년 12월까지 <a href="/wiki/%EC%9D%BC%EB%B3%B8%EC%96%B4" title="일본어">일본어</a>로 <a href="/wiki/%EB%B2%88%EC%97%AD" title="번역">번역</a>되어 출판되지 않았다.<sup class="reference" id="cite_ref-50"><a href="#cite_note-50">[50]</a></sup><sup class="reference" id="cite_ref-51"><a href="#cite_note-51">[51]</a></sup> 난징의 강간을 출간하려는 일본인 출판사의 계약을 마친 직후 번역을 위한 노력에 문제가 즉각적으로 표면에 떠올랐다. 한 일본의 문인 에이전시는 <a href="/wiki/%EC%95%84%EC%9D%B4%EB%A6%AC%EC%8A%A4_%EC%9E%A5" title="아이리스 장">아이리스 장</a>에게 여러 일본인 학자들이 그 번역을 리뷰하는 것을 거절하였으며, 한 교수는 그의 가족이 정체를 알 수 없는 단체로부터 압력을 받았다는 이유로 뒤로 물러섰다고 알려줬다.<sup class="reference" id="cite_ref-NewYorkTimes_31-1"><a href="#cite_note-NewYorkTimes-31">[31]</a></sup></p>
<p><a href="/wiki/%EC%9D%BC%EB%B3%B8" title="일본">일본학</a> 학자인 이반 P. 홀(Ivan P. Hall)에 따르면<sup class="reference" id="cite_ref-52"><a href="#cite_note-52">[52]</a></sup>, 일본 내에 <a class="new" href="/w/index.php?title=%EC%88%98%EC%A0%95%EC%A3%BC%EC%9D%98_%EC%97%AD%EC%82%AC%ED%95%99&amp;action=edit&amp;redlink=1" title="수정주의 역사학 (없는 문서)">수정주의</a> <a class="mw-redirect" href="/wiki/%EC%97%AD%EC%82%AC%ED%95%99%EC%9E%90" title="역사학자">역사학자</a>들이 극우파 학자들을 위한 위원회를 조성하여, 《난징의 강간》을 힐난하기 위해서 <a href="/wiki/%EB%8F%84%EC%BF%84" title="도쿄">도쿄</a>와 <a href="/wiki/%EC%9D%BC%EB%B3%B8" title="일본">일본</a>을 통틀어 <a class="new" href="/w/index.php?title=%EC%99%B8%EC%8B%A0%EA%B8%B0%EC%9E%90_%ED%81%B4%EB%9F%BD&amp;action=edit&amp;redlink=1" title="외신기자 클럽 (없는 문서)">외신기자 클럽</a>과 수차례 나타났다. 그들은 그 책을 계약했던 카시와 쇼보를 설복하여, 아이리스 장의 책을 그들이 원하는 대로, 사진들을 지우고 <a href="/wiki/%EC%A7%80%EB%8F%84" title="지도">지도</a>를 고치면서 "교정"하며, 아이리스 장의 책을 반박하는 책을 낼 것을 끈질기게 주장했다. 아이리스 장은 책이 수정되는 것에 동의하지 않았고, 그 결과 그 일본 출판사는 책을 출간할 것을 철회하였다.<sup class="reference" id="cite_ref-Hall_53-0"><a href="#cite_note-Hall-53">[53]</a></sup> 이후, 《난징의 강간》에 대해 반박하기 위하여 <a class="mw-redirect" href="/wiki/%EB%8F%84%EC%BF%84_%EB%8C%80%ED%95%99%EA%B5%90" title="도쿄 대학교">도쿄 대학교</a>의 교육학 교수이자 <a href="/wiki/%EC%9D%BC%EB%B3%B8" title="일본">일본</a>의 <a class="mw-redirect" href="/wiki/%EC%9D%BC%EB%B3%B8%EC%9D%98_%EC%9A%B0%EC%9D%B5_%EB%8B%A8%EC%B2%B4" title="일본의 우익 단체">우익 단체</a> <a href="/wiki/%EC%83%88%EB%A1%9C%EC%9A%B4_%EC%97%AD%EC%82%AC_%EA%B5%90%EA%B3%BC%EC%84%9C%EB%A5%BC_%EB%A7%8C%EB%93%9C%EB%8A%94_%EB%AA%A8%EC%9E%84" title="새로운 역사 교과서를 만드는 모임">새로운 역사 교과서를 만드는 모임</a>의 부회장인 <a class="new" href="/w/index.php?title=%ED%9B%84%EC%A7%80%EC%98%A4%EC%B9%B4_%EB%85%B8%EB%B6%80%EC%B9%B4%EC%93%B0&amp;action=edit&amp;redlink=1" title="후지오카 노부카쓰 (없는 문서)">후지오카 노부카쓰</a>(<span lang="ja" xml:lang="ja">藤岡 信勝</span>)와 <a class="new" href="/w/index.php?title=%EC%95%84%EC%84%B8%EC%95%84_%EB%8C%80%ED%95%99&amp;action=edit&amp;redlink=1" title="아세아 대학 (없는 문서)">아세아 대학</a>(<span lang="ja" xml:lang="ja">亜細亜大学</span>, 아시아 대학) 지적역사학과 교수인 <a class="new" href="/w/index.php?title=%ED%9E%88%EA%B0%80%EC%8B%9C%EB%82%98%EC%B9%B4%EB%85%B8_%EC%8A%88%EB%8F%84&amp;action=edit&amp;redlink=1" title="히가시나카노 슈도 (없는 문서)">히가시나카노 슈도</a> (<span lang="ja" xml:lang="ja">東中野 修道</span>)가 《난징의 강간에 대한 연구》라는 책을 출간했다.<sup class="reference" id="cite_ref-Hall_53-1"><a href="#cite_note-Hall-53">[53]</a></sup><sup class="reference" id="cite_ref-Higashinakano_54-0"><a href="#cite_note-Higashinakano-54">[54]</a></sup><sup class="reference" id="cite_ref-55"><a href="#cite_note-55">[55]</a></sup></p>
<p>히가시나카노는 난징의 강간의 처음 64페이지에서 90가지의 사실상의 오류를 지적하였으며, 이 중 일부는 <a href="/wiki/1998%EB%85%84" title="1998년">1998년</a> <a class="new" href="/w/index.php?title=%ED%8E%AD%EA%B7%84_%EC%B6%9C%ED%8C%90%EC%82%AC&amp;action=edit&amp;redlink=1" title="펭귄 출판사 (없는 문서)">펭귄 출판사</a> 책에서 교정되었다.<sup class="reference" id="cite_ref-Mistakes_56-0"><a href="#cite_note-Mistakes-56">[56]</a></sup> 가장 실수로 지적되는 것은 예를 들어, 난징의 강간은 서술하기를 250년 동안 일본군 기술은 활과 칼을 뛰어넘도록 개선하는 것에 실패하였다. 그러나 사실 일본은 <a class="mw-redirect" href="/wiki/%EB%9D%BC%EC%9D%B4%ED%94%8C" title="라이플">라이플</a>이 일본에 소개된 직후 자체적으로 그들만의 총을 생산하였다. 또다른 실수로 지적되는 것은 그 책은 <a class="mw-redirect" href="/wiki/%EB%A7%A4%EC%8A%88_C._%ED%8E%98%EB%A6%AC" title="매슈 C. 페리">매슈 C. 페리</a>(Matthew C. Perry)를 "<a href="/wiki/%EC%82%AC%EB%A0%B9%EA%B4%80" title="사령관">사령관</a> 매슈 페리"로 일컬었으나, 그는 사실 <a href="/wiki/%ED%95%A8%EB%8C%80%EC%82%AC%EB%A0%B9%EA%B4%80" title="함대사령관">함대사령관</a>이며, "함대사령관 매슈 페리"로 불려야 한다고 지적하였다.<sup class="reference" id="cite_ref-Mistakes_56-1"><a href="#cite_note-Mistakes-56">[56]</a></sup> 아이리스 장에 관한 더 이상의 비판으로 히가시나카노는 <a href="/wiki/%EC%82%B0%EC%BC%80%EC%9D%B4_%EC%8B%A0%EB%AC%B8" title="산케이 신문">산케이 신문</a>에 실린 의견난에서 "난징의 강간은 순전히 허튼소리이며, 불법적인 처형이나 살인에 아무런 목격자가 없었으며, 도쿄 재판에 의해 주장된 난징의 강간은 존재하지 않았다."라고 주장하였다.<sup class="reference" id="cite_ref-Higashinakano_54-1"><a href="#cite_note-Higashinakano-54">[54]</a></sup> 그러나 한편 히가시나카노와 그의 책, &lt;난징학살의 철저 검증&gt;을 출판했던 덴텐사 (展転社)는 난징대학살 피해자인 <a class="new" href="/w/index.php?title=%EC%83%A4%EC%88%98%EC%B9%9C&amp;action=edit&amp;redlink=1" title="샤수친 (없는 문서)">샤수친</a>(夏淑琴)이 명예훼손을 이유로 제기한 소송에서 패소하여 400만엔을 지불하라는 판결을 받았다. <a class="new" href="/w/index.php?title=%EB%8F%84%EC%BF%84%EC%A7%80%EB%B0%A9%EC%9E%AC%ED%8C%90%EC%86%8C&amp;action=edit&amp;redlink=1" title="도쿄지방재판소 (없는 문서)">도쿄지방재판소</a>는“피고의 원자료 해석은 타당한 것이라고 말하기 어렵고, 학문연구의 성과라고 할 만한 값어치가 없다고 해도 과언이 아니다"라고 판결했다.<sup class="reference" id="cite_ref-57"><a href="#cite_note-57">[57]</a></sup><sup class="reference" id="cite_ref-58"><a href="#cite_note-58">[58]</a></sup><sup class="reference" id="cite_ref-59"><a href="#cite_note-59">[59]</a></sup></p>
