<p><b>난카이 방송</b>(<span style="font-size: smaller;"><a href="/wiki/%EC%9D%BC%EB%B3%B8%EC%96%B4" title="일본어">일본어</a>:</span> <span lang="ja"><span lang="ja"><ruby><rb>南海放送</rb> <rp>(</rp><rt>なんかいほうそう</rt><rp>)</rp></ruby></span></span>)은 <a href="/wiki/%EC%97%90%ED%9E%88%EB%A9%94_%ED%98%84" title="에히메 현">에히메 현</a>에서 처음 개국한 민영방송국이며, 라디오 방송국은 <a href="/wiki/1953%EB%85%84" title="1953년">1953년</a> <a href="/wiki/10%EC%9B%94_1%EC%9D%BC" title="10월 1일">10월 1일</a>, 텔레비전 방송국은 <a href="/wiki/1958%EB%85%84" title="1958년">1958년</a> <a href="/wiki/12%EC%9B%94_1%EC%9D%BC" title="12월 1일">12월 1일</a>에 개국했다.</p>
<p>약칭은 이전 회사명 <b>라디오 난카이</b>에서 따온 약칭 RNB, 라디오 콜사인은 JOAF, 텔레비전 콜사인은 JOAF-DTV이다.</p>

<h2><span class="mw-headline" id=".EA.B0.9C.EC.9A.94">개요</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%82%9C%EC%B9%B4%EC%9D%B4_%EB%B0%A9%EC%86%A1&amp;action=edit&amp;section=1" title="부분 편집: 개요">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<ul>
<li>예전에 지역별 프로그램을 방송하고 있는 시간대가 있어서 라디오·텔레비전 모두 복수의 콜사인이 존재한다.</li>
<li><a href="/wiki/%ED%85%94%EB%A0%88%EB%B9%84%EC%A0%84" title="텔레비전">텔레비전</a>은<a class="mw-disambig" href="/wiki/NNN" title="NNN">NNN</a>계열.<a href="/wiki/%EB%9D%BC%EB%94%94%EC%98%A4" title="라디오">라디오</a>는 <a class="mw-redirect" href="/wiki/JRN" title="JRN">JRN</a>계열과 <a href="/wiki/NRN" title="NRN">NRN</a>계열의 크로스넷.</li>
<li>에히메현에 <a href="/wiki/TV_%EB%8F%84%EC%BF%84" title="TV 도쿄">TV 도쿄</a> 계열국이 없기 때문에 TV 도쿄 프로그램 중 일부를 시간차방송 형태로 내보내고 있다.</li>
<li>애니메이션의 방영에 적극적이며, 심야 애니메이션도 <a href="/wiki/%EC%97%90%ED%9E%88%EB%A9%94_%ED%98%84" title="에히메 현">에히메 현</a>에 있는 방송국들중 가장 많이 방송했다.</li>
</ul>
<h2><span class="mw-headline" id=".EC.97.B0.ED.98.81">연혁</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%82%9C%EC%B9%B4%EC%9D%B4_%EB%B0%A9%EC%86%A1&amp;action=edit&amp;section=2" title="부분 편집: 연혁">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<ul>
<li><a href="/wiki/1953%EB%85%84" title="1953년">1953년</a> <a href="/wiki/8%EC%9B%94_1%EC%9D%BC" title="8월 1일">8월 1일</a> 예비 면허 교부</li>
<li><a href="/wiki/1953%EB%85%84" title="1953년">1953년</a> <a href="/wiki/9%EC%9B%94_28%EC%9D%BC" title="9월 28일">9월 28일</a> 「난카이방송 주식회사」설립</li>

<li><a href="/wiki/1959%EB%85%84" title="1959년">1959년</a> <a href="/wiki/11%EC%9B%94_10%EC%9D%BC" title="11월 10일">11월 10일</a> 마쓰야마 라디오 본국, 3kw로 증력</li>

<h3><span class="mw-headline" id=".EA.B0.9C.EC.9A.94_2">개요</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%82%9C%EC%B9%B4%EC%9D%B4_%EB%B0%A9%EC%86%A1&amp;action=edit&amp;section=5" title="부분 편집: 개요">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<ul>
<li>전국적인 국면 동일 주파수 방송을 실시.월요일 2:00~4:00휴지를 제외한 24시간 방송.(4시 기점)　</li>
<li>이전에는 <a href="/wiki/%EC%9D%B4%EB%A7%88%EB%B0%94%EB%A6%AC_%EC%8B%9C" title="이마바리 시">이마바리 시</a>에 이마지 중계국(1116 kHz, 100 W)이 있었지만 <a href="/wiki/1996%EB%85%84" title="1996년">1996년</a> 니하마 중계국 주파수 변경 및 출력증강 이후 전파 상호 간섭을 막으려는 목적으로 2001년 7월 폐국 되었다.</li>
<li>에히메현 전역과 <a href="/wiki/%ED%9E%88%EB%A1%9C%EC%8B%9C%EB%A7%88_%ED%98%84" title="히로시마 현">히로시마 현</a>남부,<a href="/wiki/%EC%95%BC%EB%A7%88%EA%B5%AC%EC%B9%98_%ED%98%84" title="야마구치 현">야마구치 현</a>전역,<a href="/wiki/%EA%B0%80%EA%B0%80%EC%99%80_%ED%98%84" title="가가와 현">가가와 현</a>서부,<a href="/wiki/%ED%9B%84%EC%BF%A0%EC%98%A4%EC%B9%B4_%ED%98%84" title="후쿠오카 현">후쿠오카 현</a>북동부,<a href="/wiki/%EC%98%A4%EC%9D%B4%ED%83%80_%ED%98%84" title="오이타 현">오이타 현</a>동부에서도 수신이 가능하나 오이타현의 경우 현지 방송국인 <a href="/wiki/%EC%98%A4%EC%9D%B4%ED%83%80_%EB%B0%A9%EC%86%A1" title="오이타 방송">오이타 방송</a>의 주파수가 1098kHz로 난카이방송의 1116kHz에 가깝기 때문에 혼신하기 쉬우며 동일본에서는 <a href="/wiki/%EB%8B%88%EA%B0%80%ED%83%80_%EB%B0%A9%EC%86%A1" title="니가타 방송">니가타 방송</a>과 동일 주파수이므로 야간에도 수신하기 곤란하다.</li>
</ul>

<h3><span class="mw-headline" id=".ED.85.94.EB.A0.88.EB.B9.84.EC.A0.84_.EB.84.A4.ED.8A.B8.EC.9B.8C.ED.81.AC.EC.9D.98_.EB.B3.80.EC.B2.9C">텔레비전 네트워크의 변천</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%82%9C%EC%B9%B4%EC%9D%B4_%EB%B0%A9%EC%86%A1&amp;action=edit&amp;section=7" title="부분 편집: 텔레비전 네트워크의 변천">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<ul>
<li><a href="/wiki/1958%EB%85%84" title="1958년">1958년</a> <a href="/wiki/12%EC%9B%94_1%EC%9D%BC" title="12월 1일">12월 1일</a> 텔레비전 방송 개시.<a class="mw-redirect" href="/wiki/%EB%8B%88%ED%98%BCTV" title="니혼TV">니혼TV</a>·<a class="mw-redirect" href="/wiki/%EB%8F%84%EC%BF%84_%EB%B0%A9%EC%86%A1" title="도쿄 방송">라디오도쿄 TV</a>와 네트워크관계를 맺는다.</li>
<li><a href="/wiki/1959%EB%85%84" title="1959년">1959년</a> <a href="/wiki/3%EC%9B%94_1%EC%9D%BC" title="3월 1일">3월 1일</a> <a href="/wiki/TV_%EC%95%84%EC%82%AC%ED%9E%88" title="TV 아사히">NET-TV</a>와 이날 개국한<a href="/wiki/%ED%9B%84%EC%A7%80_TV" title="후지 TV">후지 TV</a>와 네트워크 관계를 맺는다.</li>
<li><a href="/wiki/1959%EB%85%84" title="1959년">1959년</a> <a href="/wiki/8%EC%9B%94_1%EC%9D%BC" title="8월 1일">8월 1일</a> <a class="mw-redirect" href="/wiki/%EB%8F%84%EC%BF%84_%EB%B0%A9%EC%86%A1" title="도쿄 방송">라디오도쿄 TV</a>주체 편성으로 하기 위해 뉴스 네트워크 <a class="mw-redirect" href="/wiki/JNN" title="JNN">JNN</a>에 가맹을 신청하지만 거절 당했기 때문에 프로그램 판매를 통한 참가만 했으며 이를 계기로 편성을 <a class="mw-redirect" href="/wiki/%EB%8B%88%ED%98%BCTV" title="니혼TV">니혼TV</a> 중심으로 한다.</li>
<li><a href="/wiki/1966%EB%85%84" title="1966년">1966년</a> <a href="/wiki/4%EC%9B%94_1%EC%9D%BC" title="4월 1일">4월 1일</a> 뉴스 네트워크<a class="mw-disambig" href="/wiki/NNN" title="NNN">NNN</a>에 가맹하면서 일부 JNN계 낮뉴스 방송을 취소한다.</li>
<li><a href="/wiki/1967%EB%85%84" title="1967년">1967년</a> <a href="/wiki/6%EC%9B%94" title="6월">6월</a> <a href="/wiki/%EB%AF%BC%EA%B0%84%EB%B0%A9%EC%86%A1_%EA%B5%90%EC%9C%A1%ED%98%91%ED%9A%8C" title="민간방송 교육협회">민간방송 교육협회</a>에 가맹.</li>
<li><a href="/wiki/1969%EB%85%84" title="1969년">1969년</a> <a href="/wiki/12%EC%9B%94_10%EC%9D%BC" title="12월 10일">12월 10일</a> <a href="/wiki/TV_%EC%97%90%ED%9E%88%EB%A9%94" title="TV 에히메">TV 에히메</a> 개국에 의해 <a href="/wiki/%ED%9B%84%EC%A7%80_TV" title="후지 TV">후지 TV</a> 프로그램이 중단되는 동시에<a class="mw-redirect" href="/wiki/%EB%8F%84%EC%BF%84%EB%B0%A9%EC%86%A1" title="도쿄방송">도쿄방송</a>·<a href="/wiki/TV_%EC%95%84%EC%82%AC%ED%9E%88" title="TV 아사히">NET-TV</a>의 일부 프로그램도 중단되었다.</li>
<li><a href="/wiki/1992%EB%85%84" title="1992년">1992년</a> <a href="/wiki/10%EC%9B%94_1%EC%9D%BC" title="10월 1일">10월 1일</a> <a href="/wiki/%EC%95%84%EC%9D%B4_TV" title="아이 TV">아이 TV</a> 개국에 의해 대부분의 <a class="mw-redirect" href="/wiki/%EB%8F%84%EC%BF%84_%EB%B0%A9%EC%86%A1" title="도쿄 방송">도쿄 방송</a>의 프로그램이 이행.</li>
<li><a href="/wiki/1993%EB%85%84" title="1993년">1993년</a> <a href="/wiki/10%EC%9B%94_1%EC%9D%BC" title="10월 1일">10월 1일</a> 이 날을 기점으로 <a class="mw-redirect" href="/wiki/%EB%8F%84%EC%BF%84%EB%B0%A9%EC%86%A1" title="도쿄방송">도쿄방송</a>의 프로그램이 <a href="/wiki/%EC%95%84%EC%9D%B4_TV" title="아이 TV">아이 TV</a>로 완전히 이행되어 모두 중단되었다.</li>
<li><a href="/wiki/1995%EB%85%84" title="1995년">1995년</a> <a href="/wiki/4%EC%9B%94_1%EC%9D%BC" title="4월 1일">4월 1일</a> <a href="/wiki/%EC%97%90%ED%9E%88%EB%A9%94_%EC%95%84%EC%82%AC%ED%9E%88_TV" title="에히메 아사히 TV">에히메 아사히 TV</a> 개국에 따라 프로그램 판매에 의한 ANN뉴스 방송을 취소하고 <a href="/wiki/%EB%AF%BC%EA%B0%84%EB%B0%A9%EC%86%A1_%EA%B5%90%EC%9C%A1%ED%98%91%ED%9A%8C" title="민간방송 교육협회">민간방송 교육협회</a> 프로그램을 제외한 모든 <a href="/wiki/TV_%EC%95%84%EC%82%AC%ED%9E%88" title="TV 아사히">TV 아사히</a> 프로그램이 중단되면서 <a class="mw-redirect" href="/wiki/%EB%8B%88%ED%98%BCTV" title="니혼TV">니혼TV</a> 완전가맹국화를 완료하였다.</li>
</ul>
<h2><span class="mw-headline" id=".EB.B9.84.EA.B3.A0">비고</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%82%9C%EC%B9%B4%EC%9D%B4_%EB%B0%A9%EC%86%A1&amp;action=edit&amp;section=8" title="부분 편집: 비고">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<ul>
<li>원래 텔레비전 방송국은<a class="mw-redirect" href="/wiki/%EB%8F%84%EC%BF%84_%EB%B0%A9%EC%86%A1" title="도쿄 방송">도쿄 방송</a>(당시는 라디오도쿄)와의 네트워크를 희망하고 있었지만 결국 니혼TV 계열로 개국했다.
<ul>
<li>이는 개국당시 방송회선 사정(마이크로 회선이 1개밖에 없었으며 시코쿠 지역 최초의 민영 텔레비전 방송사인 <a class="mw-redirect" href="/wiki/%EC%8B%9C%EC%BD%94%EC%BF%A0%EB%B0%A9%EC%86%A1" title="시코쿠방송">시코쿠방송</a>이 니혼TV 계열로 들어갔기 때문이었다.)으로 되어 있지만 이와는 별도로 네트워크 연결 교섭을 하러간 RNB 간부를 라디오 도쿄측에서 오랫동안 기다리게 했기 때문에 무례하다 여긴 간부가 협상을 그만두었다는 설도 있다(덧붙여 후자의 경우 방송구역이 중복되었던 히로시마의 <a href="/wiki/%EC%A3%BC%EA%B3%A0%EC%BF%A0_%EB%B0%A9%EC%86%A1" title="주고쿠 방송">라디오 주고쿠</a>의 요청을 받아 일부러 라디오 도쿄측이 이러한 태도를 취했다고 하는 설까지 있다).</li>
</ul>
</li>
<li>JNN협정 문제로 취재·소재 전달에는 참가할 수 없었지만 「JNN뉴스」를 하나의 「프로그램」으로서 방송하는 것이 인정되었고 이런 이유로 아침에는 TBS의 프로그램을 방송하고 있었다.<br/>
그 때문에 <a href="/wiki/1992%EB%85%84" title="1992년">1992년</a> <a href="/wiki/10%EC%9B%94" title="10월">10월</a> <a href="/wiki/%EC%95%84%EC%9D%B4_TV" title="아이 TV">아이 TV</a> 개국전 까지 오전에 주로 TBS의 프로그램을 방송하면서 제작에도 협력하고 있었다.
<ul>
<li>또한 황금시간대에 광고주 포함 동시연결에 의한 TBS 계열 동시 네트워크 프로그램이 상당수 방송되어 크로스넷 방송국 같은 편성비율이 이루어진 것으로 미루어 양자간의 친밀한 모습을 엿볼 수 있는데 이는 프로그램 공급 조직을 JNN이 겸하고 있어 RNB를 정식으로 계열국 취급 할 수 없었기 때문에 광고주 포함 동시연결에 의한 동시 네트워크 방송을 늘려 실질적으로 계열국에 준하는 취급을 하고 있었다고 생각된다.<br/>
그 때문인지 당시 라이벌 민영방송국인 <a href="/wiki/TV_%EC%97%90%ED%9E%88%EB%A9%94" title="TV 에히메">TV 에히메</a>는 RNB에서 방송하고 있지 않은 일부 니혼TV 계열 프로그램이 방송되고 있었다.</li>
</ul>
</li>
<li>니하마 텔레비전 중계국에는 호출부호 JOAL-TV가 부여되어 있었으나 지상파 아날로그 텔레비전 방송이 종료되면서 폐지되었다.</li>
<li>라디오 방송국의 경우 지역 방송국으로서는 드물게 일요일 오후나 밤에도 자사 제작 생방송 프로그램을 방송하고 있는데 이는 <a href="/wiki/%EC%A3%BC%EA%B3%A0%EC%BF%A0_%EB%B0%A9%EC%86%A1" title="주고쿠 방송">RCC</a>·<a href="/wiki/%EC%95%BC%EB%A7%88%EA%B5%AC%EC%B9%98_%EB%B0%A9%EC%86%A1" title="야마구치 방송">KRY</a>·<a href="/wiki/%EC%98%A4%EC%9D%B4%ED%83%80_%EB%B0%A9%EC%86%A1" title="오이타 방송">OBS</a>등 다른 지역의 라디오 방송국을 수신하기 쉬운 것이 한 가지 이유라고 추정된다.</li>
<li>고교 야구 중계의 경우 개회식부터 결승전까지 대부분의 시합을 라디오로 중계하고 있었고 지금도 라디오 고교 야구 중계에는 적극적이여서 남성 아나운서의 야구 실황은 지방국 중에서 레벨이 높다고 평가되며 그 경험을 살려 지역 야구리그인 시코쿠 아일랜드 리그의 중계도 적극적으로 행하면 좋겠다는 의견도 많다.</li>
</ul>
