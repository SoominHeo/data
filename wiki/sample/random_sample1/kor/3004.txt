<p><b>남대문로</b>(南大門路)는 <a href="/wiki/%EC%84%9C%EC%9A%B8%ED%8A%B9%EB%B3%84%EC%8B%9C" title="서울특별시">서울특별시</a> <a href="/wiki/%EC%A4%91%EA%B5%AC_(%EC%84%9C%EC%9A%B8%ED%8A%B9%EB%B3%84%EC%8B%9C)" title="중구 (서울특별시)">중구</a> <a href="/wiki/%EC%84%9C%EC%9A%B8%EC%97%AD" title="서울역">서울역</a>에서 시작하여 <a href="/wiki/%EC%A2%85%EB%A1%9C%EA%B5%AC" title="종로구">종로구</a> <a href="/wiki/%EC%A2%85%EA%B0%81%EC%97%AD" title="종각역">종각역 사거리</a>에서 끝나는 왕복 8차선 도로로, 총연장은 1.5 km이다. 이 도로는 기점인 국보 제1호 <a href="/wiki/%EC%88%AD%EB%A1%80%EB%AC%B8" title="숭례문">숭례문</a>(崇禮門)의 별칭에서 도로명을 삼았다.</p>
<p><a href="/wiki/%EA%B5%AD%EB%AF%BC%EC%9D%80%ED%96%89" title="국민은행">국민은행</a> 본점이 남대문로2가에, <a href="/wiki/%ED%95%9C%EA%B5%AD%EC%9D%80%ED%96%89" title="한국은행">한국은행</a>은 남대문로3가에, <a class="mw-disambig" href="/wiki/CJ" title="CJ">CJ</a>는 남대문로5가에 위치하고 있다.</p>

<h2><span class="mw-headline" id=".EC.97.B0.EA.B2.B0_.EB.8F.84.EB.A1.9C">연결 도로</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%82%A8%EB%8C%80%EB%AC%B8%EB%A1%9C&amp;action=edit&amp;section=5" title="부분 편집: 연결 도로">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>이 도로의 시작점인 <a href="/wiki/%EC%84%9C%EC%9A%B8%EC%97%AD" title="서울역">서울역</a>에서는 <a href="/wiki/%ED%95%9C%EA%B0%95%EB%8C%80%EB%A1%9C" title="한강대로">한강대로</a>, <a href="/wiki/%ED%86%B5%EC%9D%BC%EB%A1%9C" title="통일로">통일로</a>, <a href="/wiki/%EC%84%B8%EC%A2%85%EB%8C%80%EB%A1%9C" title="세종대로">세종대로</a>가 만나고, 종점인 <a href="/wiki/%EC%A2%85%EA%B0%81%EC%97%AD" title="종각역">종각역 사거리</a>에서는 <a href="/wiki/%EC%A2%85%EB%A1%9C_(%EC%84%9C%EC%9A%B8)" title="종로 (서울)">종로</a>와 <a href="/wiki/%EC%9A%B0%EC%A0%95%EA%B5%AD%EB%A1%9C" title="우정국로">우정국로</a>가 접한다.</p>
