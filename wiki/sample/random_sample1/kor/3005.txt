<h2><span class="mw-headline" id=".EC.9D.B4.EB.A0.A5">이력</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%82%A8%EB%8D%95%EC%9A%B0&amp;action=edit&amp;section=1" title="부분 편집: 이력">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p><a href="/wiki/1924%EB%85%84" title="1924년">1924년</a> <a href="/wiki/4%EC%9B%94_22%EC%9D%BC" title="4월 22일">4월 22일</a> <a href="/wiki/%EC%9D%BC%EC%A0%9C_%EA%B0%95%EC%A0%90%EA%B8%B0" title="일제 강점기">일제 강점기</a> <a href="/wiki/%EA%B2%BD%EA%B8%B0%EB%8F%84" title="경기도">경기도</a> <a href="/wiki/%EA%B4%91%EC%A3%BC%EC%8B%9C_(%EA%B2%BD%EA%B8%B0%EB%8F%84)" title="광주시 (경기도)">광주</a>에서 출생한 그는 <a href="/wiki/%EA%B5%AD%EB%AF%BC%EB%8C%80%ED%95%99%EA%B5%90" title="국민대학교">국민대학교</a> 정치학과를 졸업하였고, <a href="/wiki/%EC%84%9C%EC%9A%B8%EB%8C%80%ED%95%99%EA%B5%90" title="서울대학교">서울대학교</a> 대학원에서 경제학 석사 학위를 받은 이후 <a href="/wiki/%EB%AF%B8%EA%B5%AD" title="미국">미국</a> 오클라호마 주립 대학교 대학원에서 경제학 석사·경제학 박사 학위를 취득하였다.</p>
<p><a href="/wiki/1974%EB%85%84" title="1974년">1974년</a> 9월부터 <a href="/wiki/1978%EB%85%84" title="1978년">1978년</a> 12월까지 <a class="mw-redirect" href="/wiki/%EA%B2%BD%EC%A0%9C%EA%B8%B0%ED%9A%8D%EC%9B%90" title="경제기획원">경제기획원</a> 장관(부총리)을 지내며 <a href="/wiki/%EB%8C%80%ED%95%9C%EB%AF%BC%EA%B5%AD" title="대한민국">대한민국</a>의 100억 달러 수출 달성(<a href="/wiki/1977%EB%85%84" title="1977년">1977년</a>), 1인당 <a class="mw-redirect" href="/wiki/%EA%B5%AD%EB%82%B4_%EC%B4%9D%EC%83%9D%EC%82%B0" title="국내 총생산">국내 총생산</a> (GDP) 1,000달러 달성, 쌀 4,000만섬 생산 달성에 공헌을 한 인물이다. 또한 그는 <a href="/wiki/%EB%B0%95%EC%A0%95%ED%9D%AC" title="박정희">박정희</a> 정부의 수출 드라이브 정책을 지속적으로 밀고 나간 인물이었다. 1980년~1982년에는 제14대 대한민국 <a href="/wiki/%EB%8C%80%ED%95%9C%EB%AF%BC%EA%B5%AD%EC%9D%98_%EA%B5%AD%EB%AC%B4%EC%B4%9D%EB%A6%AC" title="대한민국의 국무총리">국무총리</a>를 지냈으며, 1983~1991년에는 한국무역협회장을, 1983~2007년에는 산학협동재단 이사장을 역임했다. 2013년 5월 18일에 향년 90세로 사망하였다. <a href="/wiki/%EB%8C%80%ED%95%9C%EB%AF%BC%EA%B5%AD_%EC%A0%95%EB%B6%80" title="대한민국 정부">대한민국 정부</a>는 <a class="mw-redirect" href="/wiki/%EA%B5%AD%EB%AF%BC%ED%9B%88%EC%9E%A5" title="국민훈장">국민훈장</a> 1등급 무궁화장을 추서하였다. 본관은 <a href="/wiki/%EC%9D%98%EB%A0%B9_%EB%82%A8%EC%94%A8" title="의령 남씨">의령</a>이다.</p>
