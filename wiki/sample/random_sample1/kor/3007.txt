<p><b>남보헤미아 주</b>(<span style="font-size: smaller;"><a href="/wiki/%EC%B2%B4%EC%BD%94%EC%96%B4" title="체코어">체코어</a>:</span> <span lang="cs" xml:lang="cs">Jihočeský kraj</span>)는 <a href="/wiki/%EC%B2%B4%EC%BD%94" title="체코">체코</a> <a href="/wiki/%EB%B3%B4%ED%97%A4%EB%AF%B8%EC%95%84" title="보헤미아">보헤미아</a> 지방 남부에 위치한 주로, 주도는 <a href="/wiki/%EC%B2%B4%EC%8A%A4%EC%BC%80%EB%B6%80%EB%8D%B0%EC%9A%94%EB%B9%84%EC%B2%B4" title="체스케부데요비체">체스케부데요비체</a>이며 면적은 10,056㎢, 인구는 643,160명(2011년 3월 기준), 인구 밀도는 64명/㎢이다. 7개 행정구를 관할한다.</p>
<p>남쪽으로는 <a href="/wiki/%EC%98%A4%EC%8A%A4%ED%8A%B8%EB%A6%AC%EC%95%84" title="오스트리아">오스트리아</a>, <a href="/wiki/%EB%8F%85%EC%9D%BC" title="독일">독일</a>과 국경을 접하며 서쪽으로는 <a href="/wiki/%ED%94%8C%EC%A0%A0_%EC%A3%BC" title="플젠 주">플젠 주</a>, 북쪽으로는 <a href="/wiki/%EC%A4%91%EC%95%99%EB%B3%B4%ED%97%A4%EB%AF%B8%EC%95%84_%EC%A3%BC" title="중앙보헤미아 주">중앙보헤미아 주</a>, 동쪽으로는 <a href="/wiki/%EB%B9%84%EC%86%8C%EC%B9%98%EB%82%98_%EC%A3%BC" title="비소치나 주">비소치나 주</a>, <a href="/wiki/%EB%82%A8%EB%AA%A8%EB%9D%BC%EB%B0%94_%EC%A3%BC" title="남모라바 주">남모라바 주</a>와 접한다.</p>
