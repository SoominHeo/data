<p><b>남부 독일 공세</b>는 <a href="/wiki/%EC%A0%9C2%EC%B0%A8_%EC%84%B8%EA%B3%84_%EB%8C%80%EC%A0%84" title="제2차 세계 대전">제2차 세계 대전</a>의 <a href="/wiki/%EC%9C%A0%EB%9F%BD_%EC%A0%84%EA%B5%AC_(%EC%A0%9C2%EC%B0%A8_%EC%84%B8%EA%B3%84_%EB%8C%80%EC%A0%84)" title="유럽 전구 (제2차 세계 대전)">유럽 전구</a>에서 마지막 공세 중 하나이다. 이 공세는 <a href="/wiki/%EB%AF%B8%EA%B5%B0" title="미군">미군</a> <a class="mw-redirect" href="/wiki/7_%EA%B5%B0_(%EB%AF%B8%EA%B5%AD)" title="7 군 (미국)">7 군</a>과 <a class="new" href="/w/index.php?title=3_%EA%B5%B0_(%EB%AF%B8%EA%B5%AD)&amp;action=edit&amp;redlink=1" title="3 군 (미국) (없는 문서)">3 군</a>, <a href="/wiki/%ED%94%84%EB%9E%91%EC%8A%A4_%EA%B3%B5%ED%99%94%EA%B5%AD_%EC%9E%84%EC%8B%9C_%EC%A0%95%EB%B6%80" title="프랑스 공화국 임시 정부">프랑스 공화국 임시 정부</a>의 <a class="new" href="/w/index.php?title=1_%EA%B5%B0_(%ED%94%84%EB%9E%91%EC%8A%A4)&amp;action=edit&amp;redlink=1" title="1 군 (프랑스) (없는 문서)">1 군</a>이 주도했다. <a href="/wiki/%EC%86%8C%EB%A0%A8%EA%B5%B0" title="소련군">소련군</a>은 <a href="/wiki/%EC%B2%B4%EC%BD%94%EC%8A%AC%EB%A1%9C%EB%B0%94%ED%82%A4%EC%95%84" title="체코슬로바키아">체코슬로바키아</a>에서 <a class="new" href="/w/index.php?title=%EC%8B%A4%EB%A6%AC%EB%B9%84%EC%B2%B4_%EC%A0%84%ED%88%AC&amp;action=edit&amp;redlink=1" title="실리비체 전투 (없는 문서)">실리비체 전투</a>와 함께 미군과 조우했다.<sup class="reference" id="cite_ref-1"><a href="#cite_note-1">[1]</a></sup> 이 공세에서 <a class="new" href="/w/index.php?title=%EB%AF%B8%EA%B5%AD_6%EA%B5%B0%EB%8B%A8&amp;action=edit&amp;redlink=1" title="미국 6군단 (없는 문서)">미국 6군단</a>이 <a class="new" href="/w/index.php?title=%EB%AF%B8%EA%B5%AD_12%EA%B5%B0%EB%8B%A8&amp;action=edit&amp;redlink=1" title="미국 12군단 (없는 문서)">미국 12군단</a>의 우측 측면을 보호하고 <a href="/wiki/%EC%95%8C%ED%94%84%EC%8A%A4_%EC%82%B0%EB%A7%A5" title="알프스 산맥">알프스 산맥</a>에서 독일군의 최후 저항을 막기 위해 진격했다. 그러나, 독일군의 저항은 북쪽에서보다 훨씬 더 치열하여 6군단의 진격이 지연되었다. 그러나, 이후 4월 말까지 많은 독일군이 항복하고 미군은 피해 없이 빠르게 전진할 수 있게 되었다. 미군 7 군의 VI 군단은 <a href="/wiki/%EC%9D%B4%ED%83%88%EB%A6%AC%EC%95%84" title="이탈리아">이탈리아</a>에서 진격하는 <a href="/wiki/%EB%AF%B8%EA%B5%AD_%EB%B6%81%EB%B6%80_%EC%9C%A1%EA%B5%B0" title="미국 북부 육군">5 군</a>과 조우했고, 3 군은 알프스 산맥에서 <a href="/wiki/%EC%98%A4%EC%8A%A4%ED%8A%B8%EB%A6%AC%EC%95%84" title="오스트리아">오스트리아</a>와 <a href="/wiki/%EC%B2%B4%EC%BD%94%EC%8A%AC%EB%A1%9C%EB%B0%94%ED%82%A4%EC%95%84" title="체코슬로바키아">체코슬로바키아</a>로 진격하여 동쪽에서 오던 소련군과 조우했다. 5월 8일 <a class="new" href="/w/index.php?title=%EB%8F%85%EC%9D%BC%EC%9D%98_%ED%95%AD%EB%B3%B5%EB%AC%B8%EC%84%9C&amp;action=edit&amp;redlink=1" title="독일의 항복문서 (없는 문서)">독일의 항복</a> 이후에도 독일군은 미군에게는 항복하면서 소련군과는 전투하며 며칠 동안은 전투가 계속되었다.</p>
