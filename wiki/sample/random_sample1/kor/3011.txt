<p><b>남부 주</b>(Southern Region)는 <a href="/wiki/%EB%A7%90%EB%9D%BC%EC%9C%84" title="말라위">말라위</a>의 주로, 주도는 <a href="/wiki/%EB%B8%94%EB%9E%9C%ED%83%80%EC%9D%B4%EC%96%B4_(%EB%A7%90%EB%9D%BC%EC%9C%84)" title="블랜타이어 (말라위)">블랜타이어</a>이며 면적은 31,753㎢, 인구는 5,876,784명(2008년 기준)이다. 13개 현(<a href="/wiki/%EB%B0%9C%EB%9D%BC%EC%B9%B4_%ED%98%84" title="발라카 현">발라카 현</a>, <a href="/wiki/%EB%B8%94%EB%9E%9C%ED%83%80%EC%9D%B4%EC%96%B4_%ED%98%84" title="블랜타이어 현">블랜타이어 현</a>, <a href="/wiki/%EC%B9%98%EC%BD%B0%EC%99%80_%ED%98%84" title="치콰와 현">치콰와 현</a>, <a href="/wiki/%EC%B9%98%EB%9D%BC%EC%A4%84%EB%A3%A8_%ED%98%84" title="치라줄루 현">치라줄루 현</a>, <a href="/wiki/%EB%A7%88%EC%B9%AD%EA%B0%80_%ED%98%84" title="마칭가 현">마칭가 현</a>, <a href="/wiki/%EB%A7%9D%EA%B3%A0%EC%B9%98_%ED%98%84" title="망고치 현">망고치 현</a>, <a href="/wiki/%EB%AC%BC%EB%9E%80%EC%A0%9C_%ED%98%84" title="물란제 현">물란제 현</a>, <a href="/wiki/%EC%9D%8C%EC%99%84%EC%9E%90_%ED%98%84" title="음완자 현">음완자 현</a>, <a href="/wiki/%EB%84%A4%EB%85%B8_%ED%98%84" title="네노 현">네노 현</a>, <a href="/wiki/%EC%9D%80%EC%82%B0%EC%A0%9C_%ED%98%84" title="은산제 현">은산제 현</a>, <a href="/wiki/%ED%8C%94%EB%A1%AC%EB%B2%A0_%ED%98%84" title="팔롬베 현">팔롬베 현</a>, <a href="/wiki/%ED%8B%B0%EC%98%AC%EB%A1%9C_%ED%98%84" title="티올로 현">티올로 현</a>, <a href="/wiki/%EC%A2%80%EB%B0%94_%ED%98%84" title="좀바 현">좀바 현</a>)을 관할한다.</p>
