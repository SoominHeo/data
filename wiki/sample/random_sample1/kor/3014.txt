<p><b>남서부 전선군</b>은 <a href="/wiki/%EC%A0%9C1%EC%B0%A8_%EC%84%B8%EA%B3%84_%EB%8C%80%EC%A0%84" title="제1차 세계 대전">제1차 세계 대전</a> 기간 <a class="new" href="/w/index.php?title=%EB%9F%AC%EC%8B%9C%EC%95%84_%EC%A0%9C%EA%B5%AD%EA%B5%B0&amp;action=edit&amp;redlink=1" title="러시아 제국군 (없는 문서)">러시아 제국군</a>, <a class="mw-redirect" href="/wiki/%EC%A0%81%EB%B0%B1%EB%82%B4%EC%A0%84" title="적백내전">적백내전</a> 기간 <a href="/wiki/%EB%9F%AC%EC%8B%9C%EC%95%84_%EC%86%8C%EB%B9%84%EC%97%90%ED%8A%B8_%EC%97%B0%EB%B0%A9_%EC%82%AC%ED%9A%8C%EC%A3%BC%EC%9D%98_%EA%B3%B5%ED%99%94%EA%B5%AD" title="러시아 소비에트 연방 사회주의 공화국">러시아 소비에트 연방 사회주의 공화국</a> 육군, <a href="/wiki/%EC%A0%9C2%EC%B0%A8_%EC%84%B8%EA%B3%84_%EB%8C%80%EC%A0%84" title="제2차 세계 대전">제2차 세계 대전</a> 기간 <a href="/wiki/%EB%B6%89%EC%9D%80_%EA%B5%B0%EB%8C%80" title="붉은 군대">붉은 군대</a>의 <a href="/wiki/%EC%A0%84%EC%84%A0%EA%B5%B0" title="전선군">전선군</a>이다.</p>
<p>이 문서는 제2차 세계 대전 기간 붉은 군대의 남서부 전선군에 대해 설명한다. 붉은 군대는 1941년 6월 22일 <a class="new" href="/w/index.php?title=%ED%82%A4%EC%98%88%ED%94%84_%ED%8A%B9%EB%B3%84_%EA%B5%B0%EC%82%AC_%EC%A7%80%EA%B5%AC&amp;action=edit&amp;redlink=1" title="키예프 특별 군사 지구 (없는 문서)">키예프 특별 군사 지구</a>를 창설했다. 1941년 6월 이 전선군이 맡은 지역은 <a class="new" href="/w/index.php?title=%ED%94%84%EB%A6%AC%ED%8D%84%ED%8A%B8_%EA%B0%95&amp;action=edit&amp;redlink=1" title="프리퍄트 강 (없는 문서)">프리퍄트 강</a>과 <a href="/wiki/%ED%94%84%EB%A3%A8%ED%8A%B8_%EA%B0%95" title="프루트 강">프루트 강</a>의 <a class="new" href="/w/index.php?title=%EC%98%AC%EB%A1%9C%EB%8B%A4%EC%99%80&amp;action=edit&amp;redlink=1" title="올로다와 (없는 문서)">올로다와</a> 마을부터 루마니아 국경의 <a class="new" href="/w/index.php?title=%EB%A6%AC%ED%94%84%EC%B9%B4%EB%8B%88&amp;action=edit&amp;redlink=1" title="리프카니 (없는 문서)">리프카니</a>까지 길이가 865km에 이르렀다. 이 전선군은 리투아니아 국경까지 있는 <a class="new" href="/w/index.php?title=%EC%84%9C%EB%B6%80_%EC%A0%84%EC%84%A0%EA%B5%B0_(%EC%86%8C%EB%B9%84%EC%97%90%ED%8A%B8_%EC%97%B0%EB%B0%A9)&amp;action=edit&amp;redlink=1" title="서부 전선군 (소비에트 연방) (없는 문서)">서부 전선군</a>, 흑해와 <a href="/wiki/%EC%98%A4%EB%8D%B0%EC%82%AC" title="오데사">오데사</a>까지 이어져 있는 <a class="new" href="/w/index.php?title=%EB%82%A8%EB%B6%80_%EC%A0%84%EC%84%A0%EA%B5%B0_(%EC%86%8C%EB%B9%84%EC%97%90%ED%8A%B8_%EC%97%B0%EB%B0%A9)&amp;action=edit&amp;redlink=1" title="남부 전선군 (소비에트 연방) (없는 문서)">남부 전선군</a>까지 이어져 있었다.</p>
