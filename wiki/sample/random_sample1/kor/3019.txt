<p><b>남송</b>(南宋 : <a href="/wiki/1127%EB%85%84" title="1127년">1127년</a>-<a href="/wiki/1279%EB%85%84" title="1279년">1279년</a>)은 중국의 왕조 중 하나이다. <a class="mw-redirect" href="/wiki/%EC%A1%B0%EA%B4%91%EC%9C%A4" title="조광윤">조광윤</a>(趙匡胤)이 건국한 <a href="/wiki/%EC%86%A1%EB%82%98%EB%9D%BC" title="송나라">송나라</a> 왕조가 여진족의 <a href="/wiki/%EA%B8%88%EB%82%98%EB%9D%BC" title="금나라">금나라</a>에게 <a href="/wiki/1126%EB%85%84" title="1126년">1126년</a> <a href="/wiki/%EC%A0%95%EA%B0%95%EC%9D%98_%EB%B3%80" title="정강의 변">정강의 변</a>을 겪으면서 화북을 빼앗긴 후, 종실 강왕(康王) <a class="mw-redirect" href="/wiki/%EC%A1%B0%EA%B5%AC" title="조구">조구</a>(趙構) (훗날 <a href="/wiki/%EB%82%A8%EC%86%A1_%EA%B3%A0%EC%A2%85" title="남송 고종">고종</a>)가 남쪽으로 옮겨 양자강 이남의 땅 <a class="mw-redirect" href="/wiki/%EC%9E%84%EC%95%88" title="임안">임안</a>(현재의 <a class="mw-redirect" href="/wiki/%ED%95%AD%EC%A0%80%EC%9A%B0" title="항저우">항저우</a>)으로 천도하여 재흥한 정권을 가리켜 남송이라고 부른다.</p>

남송 행정도</div>

<p><a href="/wiki/1126%EB%85%84" title="1126년">1126년</a> <a class="mw-redirect" href="/wiki/%EC%B9%B4%EC%9D%B4%ED%8E%91" title="카이펑">개봉부</a>가 금병에 함락되어 <a href="/wiki/%EB%B6%81%EC%86%A1" title="북송">북송</a>의 <a class="mw-redirect" href="/wiki/%ED%9C%98%EC%A2%85" title="휘종">휘종</a>(徽宗)과 <a class="mw-redirect" href="/wiki/%ED%9D%A0%EC%A2%85" title="흠종">흠종</a>(欽宗) 두 황제가 금나라에 의해 <a class="mw-redirect" href="/wiki/%EC%B9%B4%EC%9D%B4%ED%8E%91" title="카이펑">카이펑</a>에서 북쪽으로 잡혀가는 <a href="/wiki/%EC%A0%95%EA%B0%95%EC%9D%98_%EB%B3%80" title="정강의 변">정강의 변</a>이 일어나 북송이 멸망한 후, <a class="mw-redirect" href="/wiki/%ED%9D%A0%EC%A2%85" title="흠종">흠종</a>의 동생 강왕 <a class="mw-redirect" href="/wiki/%EC%A1%B0%EA%B5%AC" title="조구">조구</a>는 남쪽으로 도망쳐 <a href="/wiki/1127%EB%85%84" title="1127년">1127년</a> 난징에서 등극하여 <a href="/wiki/%EB%82%A8%EC%86%A1_%EA%B3%A0%EC%A2%85" title="남송 고종">고종</a>(高宗)이 되어 송나라를 재흥했다. 처음엔 <a href="/wiki/%EC%95%85%EB%B9%84" title="악비">악비</a>(岳飛), <a href="/wiki/%ED%95%9C%EC%84%B8%EC%B6%A9" title="한세충">한세충</a>(韓世忠) 등의 활약으로 인해 금나라에게 강력하게 저항했으나, <a class="mw-disambig" href="/wiki/%EC%A7%84%ED%9A%8C" title="진회">진회</a>(秦檜)가 재상으로 취임하면서 주전론(主戰論)을 누르고, 금나라와 화평을 맺었다.</p>
<p>화평 후 <a class="mw-disambig" href="/wiki/%EC%A7%84%ED%9A%8C" title="진회">진회</a>는 완전히 권력을 장악하고, 이때까지 <a href="/wiki/%EC%95%85%EB%B9%84" title="악비">악비</a> 등의 군벌의 손에 장악당한 군의 지휘권을 조정으로 되돌렸다. 또 한편으로는 주전론자들을 탄압하여 <a href="/wiki/%EC%95%85%EB%B9%84" title="악비">악비</a>는 살해되고, <a href="/wiki/%ED%95%9C%EC%84%B8%EC%B6%A9" title="한세충">한세충</a>은 은거를 하였다.</p>
<h3><span class="mw-headline" id=".ED.9A.A8.EC.A2.85.EC.9D.98_.EC.B9.98.EC.84.B8">효종의 치세</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%82%A8%EC%86%A1&amp;action=edit&amp;section=2" title="부분 편집: 효종의 치세">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p><a class="mw-disambig" href="/wiki/%EC%A7%84%ED%9A%8C" title="진회">진회</a>의 사후 금나라의 4대 황제 <a class="mw-redirect" href="/wiki/%ED%95%B4%EB%A6%89%EC%99%95" title="해릉왕">해릉왕</a>(海陵王)이 침공을 개시했다. 금나라군은 대군이었으나, 해릉왕은 권력 확립을 위해 수많은 사람을 살해했기 때문에 금나라의 황족 중 한 사람인 완안옹(完顔雍)이 해릉왕에 대해 반란을 일으켰다. 금나라의 유력자들은 속속 완안옹의 곁으로 모이게 되었고, 해릉왕은 군중에서 살해되었다. 완안옹은 금나라의 <a href="/wiki/%EA%B8%88_%EC%84%B8%EC%A2%85" title="금 세종">세종</a>(世宗)이 되어 송나라와 화평을 맺었다. 같은해 고종은 퇴위하여 태상황이 되고, 양자인 조신(趙昚)이 즉위하여 <a href="/wiki/%EB%82%A8%EC%86%A1_%ED%9A%A8%EC%A2%85" title="남송 효종">효종</a>(孝宗)이 되었다.</p>
<p>금의 세종, 남송의 효종은 각자가 속한 왕조에서 가장 최고의 명군으로 불렸던 인물들이었는데, 우연하게도 동시대에 2명의 명군이 남쪽과 북쪽에 나타나게 되어 평화가 찾아오게 되었다.</p>
<p>효종은 불필요한 관리의 숫자를 줄이고, 당시 남발기미가 보이던 회자(會子;지폐)의 절제 및 농촌의 체력회복, 강남경제의 활성화 등 여러가지 개혁을 추진해 남송은 번영을 구가하게 되었다.</p>
<h3><span class="mw-headline" id=".ED.95.9C.ED.83.81.EC.A3.BC_.EC.8B.9C.EB.8C.80">한탁주 시대</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%82%A8%EC%86%A1&amp;action=edit&amp;section=3" title="부분 편집: 한탁주 시대">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>효종이 <a href="/wiki/1189%EB%85%84" title="1189년">1189년</a> 퇴위하여 상황이 되고, 조돈(趙惇)이 즉위하여 <a class="mw-redirect" href="/wiki/%EC%86%A1_%EA%B4%91%EC%A2%85" title="송 광종">광종</a>(光宗)이 되었다. 광종은 아버지와는 다르게 우둔하였기에 황후 이씨의 말밖에는 듣지않았다. 황제에 불만을 품은 종실 재상 조여우(趙汝愚)와 <a href="/wiki/%ED%95%9C%ED%83%81%EC%A3%BC" title="한탁주">한탁주</a>(韓侂胄) 등에 의해 광종은 퇴위당했다. 한탁주는 이 공적으로 권력의 자리에 가까이 다가가게 될 것이라 생각했지만, 한탁주의 인격을 좋아하지 않은 조여우 등은 한탁주를 멀리했다.</p>
<p>이것에 원한을 품은 한탁주는 조여우 등을 몰아내기 위한 운동을 시작해 <a href="/wiki/1195%EB%85%84" title="1195년">1195년</a>(경원 원년) 조여우는 재상직에서 물러났고, 경원 3년에는 조여우편에 섰던 주필대(周必大), 유정(留正), 왕난(王藺), <a href="/wiki/%EC%A3%BC%ED%9D%AC" title="주희">주자</a>(朱子), 팽귀년(彭龜年) 등 59명이 금고(禁錮)에 처해졌다. 그 이듬해에는 주자의 <a href="/wiki/%EC%84%B1%EB%A6%AC%ED%95%99" title="성리학">성리학</a>(性理學;당시 도학(道學)이라고 불렸다) 역시 위학이라고하여 탄압받는 지경에 이르렀다.(경원위학의 금(慶元僞學之禁)) 이와 같은 일련의 사건을 <a class="new" href="/w/index.php?title=%EA%B2%BD%EC%9B%90%EC%9D%98_%EB%8B%B9%EA%B8%88&amp;action=edit&amp;redlink=1" title="경원의 당금 (없는 문서)">경원의 당금</a>(慶元之黨禁)이라 불렀다.</p>
<p>한탁주는 그 후 10년동안 권력을 유지했으나, 뒤를 보호해주던 황후와 황태후가 차례로 붕어하자, 권력이 서서히 줄어들게 되었다. 이때 금나라가 북쪽의 <a class="mw-disambig" href="/wiki/%ED%83%80%ED%83%80%EB%A5%B4" title="타타르">타타르</a> 등의 침입에 괴로워하는 모습을 보고 금나라는 약체화되었다고 본 한탁주는 남송의 비원인 금나라 타도을 성공시킨다면 권력을 확고부동할 수 있을거라 생각하고 북벌을 감행했으나 실패했다. 실제 금나라는 곤란한 상황이었으나, 그 이상으로 남송군의 약체화가 현저하였던 것이다.</p>
<p><a href="/wiki/1207%EB%85%84" title="1207년">1207년</a> 금나라는 조기 화평을 희망하고, 한탁주의 머리를 요구하자, 이것을 들은 예부시랑(문부대신) <a href="/wiki/%EC%82%AC%EB%AF%B8%EC%9B%90" title="사미원">사미원</a>(史弥遠)에 의해 한탁주는 살해되고, 그의 머리는 소금에 절여져 금나라로 보내지는 것으로 금나라와 화의을 맺었다.</p>
<h3><span class="mw-headline" id=".EB.AA.BD.EA.B3.A8.EC.9D.98_.EC.9C.84.ED.98.91">몽골의 위협</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%82%A8%EC%86%A1&amp;action=edit&amp;section=4" title="부분 편집: 몽골의 위협">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p><a href="/wiki/%ED%95%9C%ED%83%81%EC%A3%BC" title="한탁주">한탁주</a>을 살해한 <a href="/wiki/%EC%82%AC%EB%AF%B8%EC%9B%90" title="사미원">사미원</a>이 이번에 권력을 장악하고 그 후 26년 동안 재상의 지위를 차지했다. 그 시기 북쪽의 몽골고원에서는 <a href="/wiki/%EB%AA%BD%EA%B3%A8_%EC%A0%9C%EA%B5%AD" title="몽골 제국">몽골 제국</a>이 급속도로 세력을 확대하였다. <a href="/wiki/%EC%82%AC%EB%AF%B8%EC%9B%90" title="사미원">사미원</a>이 사망한 <a href="/wiki/1233%EB%85%84" title="1233년">1233년</a> <a href="/wiki/%EB%AA%BD%EA%B3%A8_%EC%A0%9C%EA%B5%AD" title="몽골 제국">몽골 제국</a>이 금나라의 수도 <a class="mw-redirect" href="/wiki/%EC%B9%B4%EC%9D%B4%ED%8E%91" title="카이펑">카이펑</a>을 함락하자, 남쪽으로 도망친 <a href="/wiki/%EA%B8%88%EB%82%98%EB%9D%BC" title="금나라">금나라</a> 최후의 황제 <a class="mw-redirect" href="/wiki/%EC%95%A0%EC%A2%85" title="애종">애종</a>(哀宗)을 송나라군과 협력하여 사로잡아 <a href="/wiki/1234%EB%85%84" title="1234년">1234년</a> 금나라는 멸망했다.</p>
<p>그 후 <a href="/wiki/%EB%AA%BD%EA%B3%A8" title="몽골">몽골</a>은 일시 북쪽으로 물러나고, 그 뒤를 송나라군이 북상하여 <a class="mw-redirect" href="/wiki/%EB%82%99%EC%96%91" title="낙양">낙양</a>과 <a class="mw-redirect" href="/wiki/%EC%B9%B4%EC%9D%B4%ED%8E%91" title="카이펑">카이펑</a>(개봉)을 손에 넣었다. 그러나 이것은 몽골과의 조약위반이 되었기에 격노한 몽골군과 전투상태에 들어가게 되었다. 그러나 명장으로 널리 알려진 <a class="new" href="/w/index.php?title=%EB%A7%B9%EA%B3%B5&amp;action=edit&amp;redlink=1" title="맹공 (없는 문서)">맹공</a>(孟珙)의 앞에서 고전을 면치 못해, 전투는 일진일퇴를 거듭했다. 이윽고 <a href="/wiki/1260%EB%85%84" title="1260년">1260년</a> <a href="/wiki/%EB%AA%BD%EC%BC%80_%EC%B9%B8" title="몽케 칸">몽케 칸</a>의 친정군이 들이닥쳤다.</p>
<h3><span class="mw-headline" id=".EB.A9.B8.EB.A7.9D">멸망</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%82%A8%EC%86%A1&amp;action=edit&amp;section=5" title="부분 편집: 멸망">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>그러나 <a href="/wiki/%EB%AA%BD%EC%BC%80_%EC%B9%B8" title="몽케 칸">몽케 칸</a>은 이 원정 도중에 병사한다. 이때 <a href="/wiki/%EC%BF%A0%EB%B9%8C%EB%9D%BC%EC%9D%B4_%EC%B9%B8" title="쿠빌라이 칸">쿠빌라이 칸</a>이 공격한 무창에 원군으로 온 <a class="mw-disambig" href="/wiki/%EA%B0%80%EC%82%AC%EB%8F%84" title="가사도">가사도</a>(賈似道)는 이들을 퇴각시켰다. 이 전투에서 가사도와 쿠빌라이 칸 사이에서 밀약이 있었던 것은 훗날 알려지게 되었다.</p>
<p>몽골을 격퇴한 영웅으로써 환영받게 된 <a class="mw-disambig" href="/wiki/%EA%B0%80%EC%82%AC%EB%8F%84" title="가사도">가사도</a>는 그 인기에 힘입어 재상이 되어, 전권을 임명받았다. <a class="mw-disambig" href="/wiki/%EA%B0%80%EC%82%AC%EB%8F%84" title="가사도">가사도</a>는 교묘한 정치수완을 발휘하고, 공전법 등 농정 개혁에 힘쓰는 한편, 인기를 얻는 것도 잊지 않아 그 후 15년에 걸쳐 정권을 장악했다.</p>
<p>그러나 몽골 평원에서 <a class="mw-redirect" href="/wiki/%EC%95%84%EB%A6%AC%ED%81%AC%EB%B6%80%EC%BC%80" title="아리크부케">아리크부케</a>를 쓰러뜨리고 권력을 장악한 <a href="/wiki/%EC%BF%A0%EB%B9%8C%EB%9D%BC%EC%9D%B4_%EC%B9%B8" title="쿠빌라이 칸">쿠빌라이 칸</a>이 다시 침공을 개시하여, 남송이 국력을 총동원하여 국토 방어의 거점으로 삼았던 <a class="mw-redirect" href="/wiki/%EC%96%91%EC%96%91_(%ED%9B%84%EB%B2%A0%EC%9D%B4_%EC%84%B1)" title="양양 (후베이 성)">양양</a>(襄陽)을 <a href="/wiki/1268%EB%85%84" title="1268년">1268년</a>부터 <a href="/wiki/1273%EB%85%84" title="1273년">1273년</a>까지 5년간에 걸친 포위전(<a href="/wiki/%EC%96%91%EC%96%91_%EA%B3%B5%EB%B0%A9%EC%A0%84" title="양양 공방전">양양 공방전</a>) 끝에 함락하면서 남송은 더이상의 저항할 힘이 없어졌고, <a class="mw-disambig" href="/wiki/%EA%B0%80%EC%82%AC%EB%8F%84" title="가사도">가사도</a>는 주위의 목소리에 몽골과 싸우기 위해 출발했으나 대패했다.</p>
<p><a href="/wiki/1276%EB%85%84" title="1276년">1276년</a> 몽골의 장군 <a class="mw-redirect" href="/wiki/%EB%B0%94%EB%A6%B0%EC%9D%98_%EB%B0%94%EC%96%80" title="바린의 바얀">바얀</a>에 의해 수도 <a class="mw-redirect" href="/wiki/%EC%9E%84%EC%95%88" title="임안">임안</a>이 점령되면서 사실상 송나라는 멸망했다. 그 시기 <a class="new" href="/w/index.php?title=%EC%9E%A5%EC%84%B8%EA%B1%B8&amp;action=edit&amp;redlink=1" title="장세걸 (없는 문서)">장세걸</a>(張世傑), <a class="new" href="/w/index.php?title=%EC%9C%A1%EC%88%98%EB%B6%80&amp;action=edit&amp;redlink=1" title="육수부 (없는 문서)">육수부</a>(陸秀夫) 등 일부 군인과 관료는 어린 황자를 데리고 나와 황제로 옹립하고, 남쪽으로 도망쳐 철저 항전을 계속하였다. <a href="/wiki/1279%EB%85%84" title="1279년">1279년</a> 그들은 광주만(廣州灣)의 애산(厓山)에서 <a class="mw-disambig" href="/wiki/%EC%9B%90" title="원">원</a>군에게 격멸당해 이것으로 인해 송나라는 완전히 멸망하였다(<a href="/wiki/%EC%95%A0%EC%82%B0_%EC%A0%84%ED%88%AC" title="애산 전투">애산 전투</a>).</p>
<p>충신의 귀감으로 칭해지던 <a href="/wiki/%EB%AC%B8%EC%B2%9C%EC%83%81" title="문천상">문천상</a>(文天祥)도 2년이상 각지에서 저항전을 계속했으나, <a href="/wiki/1278%EB%85%84" title="1278년">1278년</a>에 원나라에 붙잡혀 누차의 회유에도 불응하고 절개를 지켜 옥중에서 <a class="new" href="/w/index.php?title=%EC%A0%95%EA%B8%B0%EA%B0%80&amp;action=edit&amp;redlink=1" title="정기가 (없는 문서)">정기가</a>(正氣歌)를 읊고 1282년에 처형되었다.</p>
<p>남송이 멸망할 때 나라에 순사한 충신은 다른 왕조에 비해 대단히 많았으나, 원나라의 통치 아래에서 송나라의 유민으로써 살아남기를 계속한 사대부도 있어 <a class="new" href="/w/index.php?title=%EB%AC%B8%EC%9E%A5%EA%B6%A4%EB%B2%94&amp;action=edit&amp;redlink=1" title="문장궤범 (없는 문서)">문장궤범</a>을 편찬한 사방득, <a href="/wiki/%EC%8B%AD%ED%8C%94%EC%82%AC%EB%9E%B5" title="십팔사략">십팔사략</a>을 저술한 증선지, <a class="new" href="/w/index.php?title=%EC%9E%90%EC%B9%98%ED%86%B5%EA%B0%90%EC%9D%8C%EC%A3%BC&amp;action=edit&amp;redlink=1" title="자치통감음주 (없는 문서)">자치통감음주</a>(<a href="/wiki/%EC%9E%90%EC%B9%98%ED%86%B5%EA%B0%90" title="자치통감">자치통감</a>의 주석서)을 저술한 호삼성 등, 문학 사학에서 이름을 남긴 송나라의 유민도 많았다.</p>
<h2><span class="mw-headline" id=".EB.AC.B8.ED.99.94">문화</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%82%A8%EC%86%A1&amp;action=edit&amp;section=6" title="부분 편집: 문화">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>국제교역 및 농지개량 등에 의해 고도의 문화를 쌓아 화중의 개발에 중요한 역할을 담당했다. 절강성 용천애산의 청자 및 복건성 건애의 [건애]의 별명으로 알려진 [천목다완]에 최고의 명품이 만들어진 시기이기도 하고, 또 일본과의 사이에서 일송교역이 왕성해 선종도 이때 중국에서 일본으로 전해지는 등 일본에 많은 영향을 미쳤다.</p>
