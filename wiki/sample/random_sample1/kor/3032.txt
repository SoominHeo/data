<p><b>남항대교</b>(南港大橋)는 <a href="/wiki/%EB%B6%80%EC%82%B0%EA%B4%91%EC%97%AD%EC%8B%9C" title="부산광역시">부산광역시</a> <a href="/wiki/%EC%98%81%EB%8F%84%EA%B5%AC" title="영도구">영도구</a> 영선동과 <a href="/wiki/%EC%84%9C%EA%B5%AC_(%EB%B6%80%EC%82%B0%EA%B4%91%EC%97%AD%EC%8B%9C)" title="서구 (부산광역시)">서구</a> 암남동을 잇는 총연장 1.9km의 다리이다. 기존 <a href="/wiki/%EC%98%81%EB%8F%84%EB%8C%80%EA%B5%90" title="영도대교">영도대교</a> 노후에 따른 대체 교량의 건설의 필요성과 <a href="/wiki/%EC%98%81%EB%8F%84" title="영도">영도</a> 동삼지구 개발 및 항만확충에 따른 물동량 처리를 위해 건설되었다.<sup class="reference" id="cite_ref-.EA.B8.80.EB.A1.9C.EB.B2.8C_.EB.82.A8.ED.95.AD.EB.8C.80.EA.B5.90_.EA.B1.B4.EC.84.A4_1-0"><a href="#cite_note-.EA.B8.80.EB.A1.9C.EB.B2.8C_.EB.82.A8.ED.95.AD.EB.8C.80.EA.B5.90_.EA.B1.B4.EC.84.A4-1">[1]</a></sup> <a href="/wiki/2008%EB%85%84" title="2008년">2008년</a> <a href="/wiki/7%EC%9B%94_9%EC%9D%BC" title="7월 9일">7월 9일</a>에 개통한 남항대교는<a href="/wiki/%EC%84%9C%EA%B5%AC_(%EB%B6%80%EC%82%B0%EA%B4%91%EC%97%AD%EC%8B%9C)" title="서구 (부산광역시)">서구</a> 암남동과 <a href="/wiki/%EC%98%81%EB%8F%84%EA%B5%AC" title="영도구">영도구</a> 영선동을 연결하는 길이 1,941ｍ, 폭 18.5~38.6m (왕복 4차로)로 준공된 해상교량으로, 3,550억원의 사업비를 들여 <a href="/wiki/1997%EB%85%84" title="1997년">1997년</a>부터 약 10년간 공사를 추진했다.</p>
<p>남항대교의 개통으로 21세기 동북아 국제 물류 중심 항으로 떠오르고 있는 부산신항과 녹산공단, <a href="/wiki/%EC%9D%84%EC%88%99%EB%8F%84%EB%8C%80%EA%B5%90" title="을숙도대교">을숙도대교</a>, 남항대교, <a href="/wiki/%EB%B6%80%EC%82%B0%ED%95%AD%EB%8C%80%EA%B5%90" title="부산항대교">부산항대교</a>, <a class="mw-redirect" href="/wiki/%EA%B4%91%EC%95%88%EB%8C%80%EB%A1%9C" title="광안대로">광안대로</a>, 수영강변대로를 거쳐 <a href="/wiki/%EA%B2%BD%EB%B6%80%EA%B3%A0%EC%86%8D%EB%8F%84%EB%A1%9C" title="경부고속도로">경부고속도로</a>까지 연결되는 <a href="/wiki/%ED%95%AD%EB%A7%8C%EB%B0%B0%ED%9B%84%EB%8F%84%EB%A1%9C" title="항만배후도로">항만배후도로</a>를 완성하여 물류의 원활한 흐름이 가능해졌을 뿐 아니라 <a href="/wiki/%EC%84%9C%EA%B5%AC_(%EB%B6%80%EC%82%B0%EA%B4%91%EC%97%AD%EC%8B%9C)" title="서구 (부산광역시)">서구</a>와 <a href="/wiki/%EC%98%81%EB%8F%84" title="영도">영도</a>를 가깝게 잇는 길이 뚫림으로서 <a href="/wiki/%EC%84%9C%EA%B5%AC_(%EB%B6%80%EC%82%B0%EA%B4%91%EC%97%AD%EC%8B%9C)" title="서구 (부산광역시)">서구</a>와 <a href="/wiki/%EC%98%81%EB%8F%84%EA%B5%AC" title="영도구">영도구</a>를 동일 생활권으로 묶어주었다. 그리고 ‘신세기를 열어가는 관문’을 주제로 한 경관조명은 남항의 밤바다를 아름답게 수놓고 있으며, 내항 쪽에는 폭 3m의 산책로가 설치돼 시민들이 자유롭게 왕복하면서 남항일원의 풍광과 <a class="mw-redirect" href="/wiki/%EB%B6%80%EC%82%B0" title="부산">부산</a>의 해안을 한 눈에 볼 수 있다.</p>

<h2><span class="mw-headline" id=".EA.B5.AC.EC.A1.B0">구조</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%82%A8%ED%95%AD%EB%8C%80%EA%B5%90&amp;action=edit&amp;section=2" title="부분 편집: 구조">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<ul>
<li>영선램프~암남램프 : 편도 2차로(왕복 4차로)</li>
</ul>
<h2><span class="mw-headline" id=".EA.B3.84.ED.9A.8D">계획</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%82%A8%ED%95%AD%EB%8C%80%EA%B5%90&amp;action=edit&amp;section=3" title="부분 편집: 계획">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<ul>
<li>남항대교는 <a href="/wiki/%EB%B6%80%EC%82%B0%EA%B4%91%EC%97%AD%EC%8B%9C%EB%8F%84_%EC%A0%9C66%ED%98%B8%EC%84%A0" title="부산광역시도 제66호선">부산광역시도 제66호선</a>의 일부를 구성한다.</li>
<li>추후 <b>천마터널</b>을 통하여 <a href="/wiki/%EC%9D%84%EC%88%99%EB%8F%84%EB%8C%80%EA%B5%90" title="을숙도대교">을숙도대교</a>와 연결되는 지하차도가 이어서 건설될 예정이다.</li>
<li>추후 <b>천마터널</b>이 개통되면 현재의 편도 2차로(왕복 4차로)의 도로를 편도3차로(왕복 6차로)로 확장할 예정이다.</li>
</ul>
<h2><span class="mw-headline" id=".ED.9A.A8.EA.B3.BC">효과</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%82%A8%ED%95%AD%EB%8C%80%EA%B5%90&amp;action=edit&amp;section=4" title="부분 편집: 효과">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<ul>
<li>남항대교가 개통됨에 따라 기존 암남동-영도 7 km 30분에서 2 km, 3분이라는 효과를 내고 있으며 영도에서 서부산권으로 나가는 시간이 단축되는 효과가 나타났다.</li>
</ul>
<h2><span class="mw-headline" id=".EC.A3.BC.EC.9D.98.EC.82.AC.ED.95.AD">주의사항</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%82%A8%ED%95%AD%EB%8C%80%EA%B5%90&amp;action=edit&amp;section=5" title="부분 편집: 주의사항">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<ul>
<li>남항대교는 <a class="mw-redirect" href="/wiki/%EC%9E%90%EB%8F%99%EC%B0%A8%EC%A0%84%EC%9A%A9%EB%8F%84%EB%A1%9C" title="자동차전용도로">자동차전용도로</a>이나, 산책로를 이용하여 <a class="new" href="/w/index.php?title=%EB%B3%B4%ED%96%89%EC%9E%90&amp;action=edit&amp;redlink=1" title="보행자 (없는 문서)">보행자</a>와 <a href="/wiki/%EC%9E%90%EC%A0%84%EA%B1%B0" title="자전거">자전거</a>가 통행할 수 있다. 이륜자동차(모터사이클 혹은 오토바이)는 긴급자동차(싸이카 및 소방용 모터사이클 등)에 한해 통행이 가능하며, 그 밖의 이륜자동차는 통행할 수 없다.</li>
</ul>
