<p><b>남해</b>(南海, <a href="/wiki/%EB%AC%B8%ED%99%94%EC%96%B4" title="문화어">문화어</a>: 조선남해/朝鮮南海)는 <a href="/wiki/%ED%95%9C%EB%B0%98%EB%8F%84" title="한반도">한반도</a>의 남쪽 바다를 지칭하는 말로, 일반적으로 <a href="/wiki/%EC%A0%84%EB%9D%BC%EB%82%A8%EB%8F%84" title="전라남도">전라남도</a> <a href="/wiki/%EC%A7%84%EB%8F%84_(%EC%84%AC)" title="진도 (섬)">진도</a>부터 <a href="/wiki/%EB%B6%80%EC%82%B0%EA%B4%91%EC%97%AD%EC%8B%9C" title="부산광역시">부산광역시</a> <a href="/wiki/%ED%95%B4%EC%9A%B4%EB%8C%80%EA%B5%AC" title="해운대구">해운대구</a>까지 사이의 바다를 가리킨다. 국제적으로 이 해역의 대부분(<a href="/wiki/%EC%A0%84%EB%9D%BC%EB%82%A8%EB%8F%84" title="전라남도">전라남도</a> <a href="/wiki/%ED%95%B4%EB%82%A8%EA%B5%B0" title="해남군">해남군</a> 남쪽 끝 ~ <a href="/wiki/%EB%B6%80%EC%82%B0%EA%B4%91%EC%97%AD%EC%8B%9C" title="부산광역시">부산광역시</a>)은 <a href="/wiki/%EB%8F%99%ED%95%B4" title="동해">동해</a>에 속하고, 나머지는 <a href="/wiki/%ED%99%A9%ED%95%B4" title="황해">황해</a>에 속한다.</p>
<p><a href="/wiki/%EA%B5%AD%EC%A0%9C_%EC%88%98%EB%A1%9C_%EA%B8%B0%EA%B5%AC" title="국제 수로 기구">국제 수로 기구</a>는 <a href="/wiki/1953%EB%85%84" title="1953년">1953년</a> 발간한 《해양과 바다의 경계》(Limits of Oceans and Seas) 제3판에서 <a href="/wiki/%EC%A0%9C%EC%A3%BC%EB%8F%84" title="제주도">제주도</a>를 경계로 서북쪽을 <a href="/wiki/%ED%99%A9%ED%95%B4" title="황해">황해</a>, 동북쪽을 <a href="/wiki/%EB%8F%99%ED%95%B4" title="동해">동해</a> , 제주도 남쪽 바다를 <a href="/wiki/%EB%8F%99%EC%A4%91%EA%B5%AD%ED%95%B4" title="동중국해">동중국해</a>로 설정하고 있다.<sup class="reference" id="cite_ref-1"><a href="#cite_note-1">[1]</a></sup><sup class="reference" id="cite_ref-2"><a href="#cite_note-2">[2]</a></sup></p>
<h2><span class="mw-headline" id=".ED.95.B4.EC.97.AD.EC.9D.98_.EB.B2.94.EC.9C.84">해역의 범위</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%82%A8%ED%95%B4&amp;action=edit&amp;section=1" title="부분 편집: 해역의 범위">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p><a href="/wiki/%EB%8C%80%ED%95%9C%EB%AF%BC%EA%B5%AD" title="대한민국">대한민국</a>의 정부 기관들은 남해의 범위를 제각각 다르게 정하고 있다.<sup class="reference" id="cite_ref-3"><a href="#cite_note-3">[3]</a></sup></p>
<p><a href="/wiki/1997%EB%85%84" title="1997년">1997년</a>, <a href="/wiki/%EB%8C%80%ED%95%9C%EB%AF%BC%EA%B5%AD_%ED%95%B4%EC%96%91%EC%88%98%EC%82%B0%EB%B6%80" title="대한민국 해양수산부">대한민국 해양수산부</a>는 <a href="/wiki/%EB%8F%99%ED%95%B4" title="동해">동해</a>와 남해의 경계를 <a href="/wiki/%EC%9A%B8%EC%82%B0%EA%B4%91%EC%97%AD%EC%8B%9C" title="울산광역시">울산광역시</a> <a href="/wiki/%ED%83%9C%ED%99%94%EA%B0%95" title="태화강">태화강</a> 하구의 <a href="/wiki/%EB%B0%A9%EC%96%B4%EC%A7%84%ED%95%AD" title="방어진항">방어진항</a> 남쪽과 <a href="/wiki/%EC%9D%BC%EB%B3%B8" title="일본">일본</a> <a href="/wiki/%EC%9D%B4%EC%A6%88%EB%AA%A8_%EC%8B%9C" title="이즈모 시">이즈모 시</a>를 동서로 이은 직선(북위 35도 28분)으로 정했다. 또한, <a class="mw-redirect" href="/wiki/%EC%84%9C%ED%95%B4" title="서해">서해</a>와 남해의 경계를 <a href="/wiki/%EC%A0%84%EB%9D%BC%EB%82%A8%EB%8F%84" title="전라남도">전라남도</a> <a class="mw-redirect" href="/wiki/%EC%A7%84%EB%8F%84_(%EC%A7%84%EB%8F%84%EA%B5%B0)" title="진도 (진도군)">진도</a> 서쪽 끝과 <a href="/wiki/%EC%A0%9C%EC%A3%BC%EB%8F%84" title="제주도">제주도</a> 서쪽 끝(<a href="/wiki/%EC%B0%A8%EA%B7%80%EB%8F%84" title="차귀도">차귀도</a>)을 남북으로 이은 직선으로, <a href="/wiki/%EB%8F%99%EC%A4%91%EA%B5%AD%ED%95%B4" title="동중국해">동중국해</a>와 남해의 경계를 <a href="/wiki/%EC%A0%9C%EC%A3%BC%EB%8F%84" title="제주도">제주도</a> 동쪽 끝(<a href="/wiki/%EC%9A%B0%EB%8F%84" title="우도">우도</a>)에서 <a href="/wiki/%EC%9D%BC%EB%B3%B8" title="일본">일본</a> <a href="/wiki/%ED%9B%84%EC%BF%A0%EC%97%90_%EC%84%AC" title="후쿠에 섬">후쿠에 섬</a> 남쪽 끝을 이은 직선으로 정했다. 이는 국립수산진흥원(현 <a href="/wiki/%EA%B5%AD%EB%A6%BD%EC%88%98%EC%82%B0%EA%B3%BC%ED%95%99%EC%9B%90" title="국립수산과학원">국립수산과학원</a>)이 <a href="/wiki/1979%EB%85%84" title="1979년">1979년</a> 한국해양편람에서 썼던 기준이다.</p>
<p>반면, <a href="/wiki/%EB%8C%80%ED%95%9C%EB%AF%BC%EA%B5%AD_%ED%95%B4%EC%96%91%EC%88%98%EC%82%B0%EB%B6%80" title="대한민국 해양수산부">대한민국 해양수산부</a>의 소속 기관인 <a class="mw-redirect" href="/wiki/%EA%B5%AD%EB%A6%BD%ED%95%B4%EC%96%91%EC%A1%B0%EC%82%AC%EC%9B%90" title="국립해양조사원">국립해양조사원</a>은 <a href="/wiki/1992%EB%85%84" title="1992년">1992년</a> 수로업무편람에서 동해와 남해의 경계를 <a class="mw-redirect" href="/wiki/%EB%B6%80%EC%82%B0_%EB%82%A8%EA%B5%AC" title="부산 남구">부산 남구</a> 승두말(<a href="/wiki/%EC%98%A4%EB%A5%99%EB%8F%84" title="오륙도">오륙도</a> 바로 위)로, 서해와 남해의 경계를 <a href="/wiki/%EC%A0%84%EB%9D%BC%EB%82%A8%EB%8F%84" title="전라남도">전라남도</a> <a href="/wiki/%ED%95%B4%EB%82%A8%EA%B5%B0" title="해남군">해남 반도</a>의 남쪽 끝으로 정하였다. 그러나, 이 기관이 현재 <a href="/wiki/%EC%A7%80%EB%A6%AC_%EC%A2%8C%ED%91%9C%EA%B3%84" title="지리 좌표계">좌표</a>상 정하고 있는 동해와 남해의 경계는 <a href="/wiki/%EB%B6%80%EC%82%B0%EA%B4%91%EC%97%AD%EC%8B%9C" title="부산광역시">부산광역시</a> <a href="/wiki/%ED%95%B4%EC%9A%B4%EB%8C%80%EA%B5%AC" title="해운대구">해운대구</a>를 지난다.<sup class="reference" id="cite_ref-4"><a href="#cite_note-4">[4]</a></sup><sup class="reference" id="cite_ref-5"><a href="#cite_note-5">[5]</a></sup></p>
<p><a href="/wiki/%EB%8C%80%ED%95%9C%EB%AF%BC%EA%B5%AD_%EA%B8%B0%EC%83%81%EC%B2%AD" title="대한민국 기상청">대한민국 기상청</a>은 국립해양조사원과 같이 서해와 남해의 경계를 전라남도 해남 반도 남쪽 끝으로 정하고 있지만, 동해와 남해의 경계는 <a href="/wiki/%EB%B6%80%EC%82%B0%EA%B4%91%EC%97%AD%EC%8B%9C" title="부산광역시">부산광역시</a>와 <a href="/wiki/%EC%9A%B8%EC%82%B0%EA%B4%91%EC%97%AD%EC%8B%9C" title="울산광역시">울산광역시</a>의 해안 경계점으로 정하고 있다.</p>

<h2><span class="mw-headline" id=".EA.B0.81.EC.A3.BC">각주</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%82%A8%ED%95%B4&amp;action=edit&amp;section=3" title="부분 편집: 각주">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
