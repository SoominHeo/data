<p>《<b>내 머리 속의 지우개</b>》는 <a href="/wiki/2004%EB%85%84" title="2004년">2004년</a>에 개봉된 <a href="/wiki/%EC%9D%B4%EC%9E%AC%ED%95%9C_(%EC%98%81%ED%99%94_%EA%B0%90%EB%8F%85)" title="이재한 (영화 감독)">이재한</a>의 <a href="/wiki/%EC%98%81%ED%99%94" title="영화">영화</a>이다. <a href="/wiki/%EC%A0%95%EC%9A%B0%EC%84%B1" title="정우성">정우성</a>, <a href="/wiki/%EC%86%90%EC%98%88%EC%A7%84" title="손예진">손예진</a>이 주연을 맡았고, <a href="/wiki/2001%EB%85%84" title="2001년">2001년</a> <a href="/wiki/%EC%9D%BC%EB%B3%B8" title="일본">일본</a> <a class="mw-disambig" href="/wiki/YTV" title="YTV">요미우리 TV</a>에서 방송된 <a href="/wiki/%EB%93%9C%EB%9D%BC%EB%A7%88" title="드라마">드라마</a> 《퓨어 소울: 네가 나를 잊어도》가 원작이며, 그 원작을 토대로 각색하였다.</p>
<p><a href="/wiki/2005%EB%85%84" title="2005년">2005년</a> <a href="/wiki/%EC%9D%BC%EB%B3%B8" title="일본">일본</a>에서 개봉되어 300만이 넘는 관객수를 불러들으며, 일본내 30억 엔의 흥행수입을 기록했다. <a href="/wiki/2007%EB%85%84" title="2007년">2007년</a>에는 <a href="/wiki/%ED%85%94%EB%A0%88%EB%B9%84%EC%A0%84_%EB%93%9C%EB%9D%BC%EB%A7%88" title="텔레비전 드라마">드라마</a>로 만들어져 방송되었고, <a href="/wiki/2010%EB%85%84" title="2010년">2010년</a>에는 <a href="/wiki/%EC%97%B0%EA%B7%B9" title="연극">연극</a>으로 제작되었다.</p>
