<p><b><a href="/wiki/%EB%82%B4%EC%85%94%EB%84%90_%EB%A6%AC%EA%B7%B8_(%EC%95%BC%EA%B5%AC)" title="내셔널 리그 (야구)">내셔널 리그</a> 중부</b>(National League Central)은 <a href="/wiki/%EB%A9%94%EC%9D%B4%EC%A0%80_%EB%A6%AC%EA%B7%B8_%EB%B2%A0%EC%9D%B4%EC%8A%A4%EB%B3%BC" title="메이저 리그 베이스볼">메이저 리그 베이스볼</a>의 5개팀으로 이루어진 지구이다. <a href="/wiki/1994%EB%85%84" title="1994년">1994년</a>, <a href="/wiki/%EB%82%B4%EC%85%94%EB%84%90_%EB%A6%AC%EA%B7%B8_%EC%84%9C%EB%B6%80" title="내셔널 리그 서부">NL 서부</a>에서 2팀(<a href="/wiki/%EC%8B%A0%EC%8B%9C%EB%82%B4%ED%8B%B0_%EB%A0%88%EC%A6%88" title="신시내티 레즈">신시내티 레즈</a>, <a href="/wiki/%ED%9C%B4%EC%8A%A4%ED%84%B4_%EC%95%A0%EC%8A%A4%ED%8A%B8%EB%A1%9C%EC%8A%A4" title="휴스턴 애스트로스">휴스턴 애스트로스</a>), <a href="/wiki/%EB%82%B4%EC%85%94%EB%84%90_%EB%A6%AC%EA%B7%B8_%EB%8F%99%EB%B6%80" title="내셔널 리그 동부">NL 동부</a>에서 3팀(<a href="/wiki/%EC%8B%9C%EC%B9%B4%EA%B3%A0_%EC%BB%B5%EC%8A%A4" title="시카고 컵스">시카고 컵스</a>, <a href="/wiki/%ED%94%BC%EC%B8%A0%EB%B2%84%EA%B7%B8_%ED%8C%8C%EC%9D%B4%EB%A6%AC%EC%B8%A0" title="피츠버그 파이리츠">피츠버그 파이리츠</a>, <a href="/wiki/%EC%84%B8%EC%9D%B8%ED%8A%B8%EB%A3%A8%EC%9D%B4%EC%8A%A4_%EC%B9%B4%EB%94%94%EB%84%90%EC%8A%A4" title="세인트루이스 카디널스">세인트루이스 카디널스</a>)가 참가하면서 설치되었다. 1998년, 중부 지구는 <a href="/wiki/%EC%95%84%EB%A9%94%EB%A6%AC%EC%B9%B8_%EB%A6%AC%EA%B7%B8_%EC%A4%91%EB%B6%80" title="아메리칸 리그 중부">AL 중부</a>에서 <a href="/wiki/%EB%B0%80%EC%9B%8C%ED%82%A4_%EB%B8%8C%EB%A3%A8%EC%96%B4%EC%8A%A4" title="밀워키 브루어스">밀워키 브루어스</a>가 참가하여 메이저 리그에서 가장 큰 지구가 되었다. 2013년, <a href="/wiki/%ED%9C%B4%EC%8A%A4%ED%84%B4_%EC%95%A0%EC%8A%A4%ED%8A%B8%EB%A1%9C%EC%8A%A4" title="휴스턴 애스트로스">휴스턴 애스트로스</a>가 <a href="/wiki/%EC%95%84%EB%A9%94%EB%A6%AC%EC%B9%B8_%EB%A6%AC%EA%B7%B8_%EC%84%9C%EB%B6%80" title="아메리칸 리그 서부">AL 서부</a>로 이동했다.</p>
<p>이 지구는 <a href="/wiki/%EC%84%B8%EC%9D%B8%ED%8A%B8%EB%A3%A8%EC%9D%B4%EC%8A%A4_%EC%B9%B4%EB%94%94%EB%84%90%EC%8A%A4" title="세인트루이스 카디널스">세인트루이스 카디널스</a>와 <a href="/wiki/%ED%9C%B4%EC%8A%A4%ED%84%B4_%EC%95%A0%EC%8A%A4%ED%8A%B8%EB%A1%9C%EC%8A%A4" title="휴스턴 애스트로스">휴스턴 애스트로스</a>가 지배해왔는데, 이 팀들은 지금까지 지구 우승을 5시즌 만을 제외하고 모두 차지했다.</p>
<p>2016년 지구 우승팀은 <a href="/wiki/%EC%8B%9C%EC%B9%B4%EA%B3%A0_%EC%BB%B5%EC%8A%A4" title="시카고 컵스">시카고 컵스</a>이다.</p>
