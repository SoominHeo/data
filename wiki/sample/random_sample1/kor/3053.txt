<h2><span class="mw-headline" id=".EA.B0.9C.EC.9A.94">개요</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%82%B4%EC%85%94%EB%84%90%EB%A6%AC%EC%A6%98&amp;action=edit&amp;section=1" title="부분 편집: 개요">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>제일 중요한 것으로는 "민족이 <a href="/wiki/%EA%B5%AD%EA%B0%80" title="국가">국가</a>를 합리화하는 유일무이한 근간"이라는 주장과 "개별 민족은 국가를 형성할 권리가 있다"라는 주장이다. 이러한 형태를 갖출 때 국민주의는 보편적인 이념이다. 그러나 국민주의는 또한 민족적 움직임의 특수 이념을 의미하기도 하는데 이런 경우 '민족주의'라고 불리며, 특정 민족의 입장을 지지하게 된다.</p>
<p>국민주의, 민족주의는 <a href="/wiki/%EC%95%A0%EA%B5%AD%EC%A3%BC%EC%9D%98" title="애국주의">애국주의</a>와 지나친 애국주의로 생기는 <a href="/wiki/%EA%B5%AD%EC%88%98%EC%A3%BC%EC%9D%98" title="국수주의">국수주의</a>와 혼동하기 쉬운 개념이다.</p>
<h2><span class="mw-headline" id=".EA.B5.AD.EB.AF.BC.EC.A3.BC.EC.9D.98.EC.9D.98_.EA.B8.B0.EB.B3.B8_.EC.9B.90.EC.B9.99">국민주의의 기본 원칙</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%82%B4%EC%85%94%EB%84%90%EB%A6%AC%EC%A6%98&amp;action=edit&amp;section=2" title="부분 편집: 국민주의의 기본 원칙">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>국민주의는 또한 16세기 영국에서 발현되어 타 지역 사회로 퍼지게 되어 "근대성을 구성"하게 된 "특정 관념적 시선"으로 정의되기도 한다. 여러 국민주의적 움직임들은 서로의 입장에 의의를 제기할 수 도 있지만 동일한 국민주의적 사상을 공유한다.</p>
<p>국민주의에 대한 기본적인 학술 연구는 <a href="/wiki/%EB%B2%A0%EB%84%A4%EB%94%95%ED%8A%B8_%EC%95%A4%EB%8D%94%EC%8A%A8" title="베네딕트 앤더슨">베네딕트 앤더슨</a>의 《상상의 공동체》와 <a class="new" href="/w/index.php?title=%EB%A6%AC%EC%95%84_%EA%B7%B8%EB%A6%B0%ED%8E%A0%EB%93%9C&amp;action=edit&amp;redlink=1" title="리아 그린펠드 (없는 문서)">리아 그린펠드</a>의 《민족주의: 근대에 이르는 다섯 갈래 길》이 있다.</p>
<p>일반적으로 국민주의자들은 개별 민족을 특정 기준에 맞춰 정의하며 민족을 타 민족에서 구분하고, 개개인의 "민족의 일원 여부"를 판단한다. 이러한 기준들은 공유된 <a href="/wiki/%EC%96%B8%EC%96%B4" title="언어">언어</a>, <a href="/wiki/%EB%AC%B8%ED%99%94" title="문화">문화</a>, 또는 <a href="/wiki/%EA%B0%80%EC%B9%98%EA%B4%80" title="가치관">가치관</a>등이 있으나 많은 경우 제일 중요하게 여겨지는 것은 하나의 <a href="/wiki/%EB%AF%BC%EC%A1%B1" title="민족">민족</a>에 속해있는것이다. 민족적 "정체성"이라는 것은 한편으로는 이러한 선언적 정의를 포함하며 다른 한편으로는 이러한 무리의 "일원이 되었다는 느낌"을 포함한다. 국민주의자들은 민족의 일원이 되는 것을 배타적이며 개인의 의지와는 독립적인것으로 보는데, 이것은 개인이 원한다고 해서 일반 <a class="mw-redirect" href="/wiki/%EB%8B%A8%EC%B2%B4" title="단체">단체</a>처럼 "가입"할 수는 없는 것이다.</p>
<p>국민주의는 인류의 활동 대부분을 그 특징에 있어서 민족주의적으로 파악한다. 이들에게 있어서 민족은 고유한 <a class="new" href="/w/index.php?title=%EA%B5%AD%EA%B0%80_%EC%83%81%EC%A7%95&amp;action=edit&amp;redlink=1" title="국가 상징 (없는 문서)">국가 상징</a>, 민족적 특성, <a class="new" href="/w/index.php?title=%EB%AF%BC%EC%A1%B1%EC%A0%81_%EB%AC%B8%ED%99%94&amp;action=edit&amp;redlink=1" title="민족적 문화 (없는 문서)">민족적 문화</a>, 민족 노래와 민족 문학을 가지고 있으며 민족적 <a href="/wiki/%EC%8B%A0%ED%99%94" title="신화">신화</a>와 소수 사례의 경우 <a href="/wiki/%EB%AF%BC%EC%A1%B1_%EC%A2%85%EA%B5%90" title="민족 종교">민족 종교</a> 또한 가지고 있다. 개인들은 <a class="new" href="/w/index.php?title=%EB%AF%BC%EC%A1%B1_%EA%B0%80%EC%B9%98%EA%B4%80&amp;action=edit&amp;redlink=1" title="민족 가치관 (없는 문서)">민족 가치관</a>과 <a class="new" href="/w/index.php?title=%EB%AC%B8%ED%99%94%EC%A0%81_%EC%A0%95%EC%B2%B4%EC%84%B1&amp;action=edit&amp;redlink=1" title="문화적 정체성 (없는 문서)">문화적 정체성</a>을 공유하며 <a class="new" href="/w/index.php?title=%EB%AF%BC%EC%A1%B1_%EC%98%81%EC%9B%85&amp;action=edit&amp;redlink=1" title="민족 영웅 (없는 문서)">민족 영웅</a>을 우러러보고 <a class="new" href="/w/index.php?title=%EB%AF%BC%EC%A1%B1_%EC%9D%8C%EC%8B%9D&amp;action=edit&amp;redlink=1" title="민족 음식 (없는 문서)">민족 음식</a>을 먹으며 <a class="new" href="/w/index.php?title=%EB%AF%BC%EC%A1%B1_%EC%8A%A4%ED%8F%AC%EC%B8%A0&amp;action=edit&amp;redlink=1" title="민족 스포츠 (없는 문서)">민족 스포츠</a>를 즐긴다.</p>
<p>국민주의는 <a class="mw-redirect" href="/wiki/%EB%AF%BC%EC%A1%B1%EA%B5%AD%EA%B0%80" title="민족국가">민족국가</a>를 지배적인 <a href="/wiki/%EA%B5%AD%EA%B0%80" title="국가">국가</a> 형태로 만들어버림으로서 <a class="mw-redirect" href="/wiki/%EC%84%B8%EA%B3%84_%EC%97%AD%EC%82%AC" title="세계 역사">세계 역사</a>와 <a class="new" href="/w/index.php?title=%EC%A7%80%EB%A6%AC%EC%A0%95%EC%B9%98%ED%95%99&amp;action=edit&amp;redlink=1" title="지리정치학 (없는 문서)">지리정치학</a>에 엄청난 영향을 끼쳤다. 오늘날 세계 대부분의 사람들이 살고 있는 국가들은 최소한 문서상으로는 민족국가이다. 그 결과로 이들 민족국가를 "민족"로 부르는, 이론적으로 부정확한 전례가 생겨났다. 민족국가는 민족의 존재 여부를 보장하며 구분된 정체성을 유지하며 지배적인 민족적 문화와 <a href="/wiki/%EC%97%90%ED%86%A0%EC%8A%A4" title="에토스">에토스</a>가 형성 될 수 있는 영토를 제공하도록 의도되었다. 수많은 민족국가는 문화·역사적인 <a href="/wiki/%EC%8B%A0%ED%99%94" title="신화">신화</a>의 힘을 빌려 자신의 존재를 방어하여 <a class="new" href="/w/index.php?title=%EC%A0%95%EB%8B%B9%EC%84%B1_(%EC%A0%95%EC%B9%98%ED%95%99)&amp;action=edit&amp;redlink=1" title="정당성 (정치학) (없는 문서)">정당성</a>을 확보하려고 한다.</p>
<p>국민주의자들은 "비(非)민족적" 국가가 존재한다는 사실을 인정한다. 실제로 초기 민족주의 운동들은 <a class="mw-redirect" href="/wiki/%EC%98%A4%EC%8A%A4%ED%8A%B8%EB%A6%AC%EC%95%84-%ED%97%9D%EA%B0%80%EB%A6%AC" title="오스트리아-헝가리">오스트리아-헝가리</a> 등의 <a href="/wiki/%EC%A0%9C%EA%B5%AD" title="제국">제국</a>들을 겨냥하였다. <a class="mw-redirect" href="/wiki/%EB%B0%94%ED%8B%B0%EC%B9%B8" title="바티칸">바티칸</a>은 <a class="mw-redirect" href="/wiki/%EB%A1%9C%EB%A7%88_%EA%B0%80%ED%86%A8%EB%A6%AD" title="로마 가톨릭">로마 가톨릭</a>의 지도하에 있는 <a href="/wiki/%EC%A3%BC%EA%B6%8C" title="주권">주권적 국가</a>를 제공하기 위하여 존재하지만 이는 <a href="/wiki/%EB%AF%BC%EC%A1%B1" title="민족">민족</a>이 없다. 많은 <a href="/wiki/%EC%9D%B4%EC%8A%AC%EB%9E%8C%EC%A3%BC%EC%9D%98" title="이슬람주의">이슬람주의</a>자들이 추구하는 범세계적 <a href="/wiki/%EC%B9%BC%EB%A6%AC%ED%8C%8C" title="칼리파">칼리파</a>국(Caliphate)은 또한 비민족적 국가의 한 예가 될 수 있다.</p>
<p>자신을 민족의 일부로 파악하며 민족국가들이 정통성을 지닌다고 생각하는 이들은 형식적으로 "민족주의자"로 정의될 수 있다. 이러한 맥락에서 대부분의 어른들은 수동적 민족주의자이다. 그러나 현대의 생활 언어에 있어서 "민족주의"는 민족주의적 요구를 지지하는 정치적 (그리고 가끔 군사적) 행동을 의미하게 되었다. 이러한 일련의 행동들은 <a href="/wiki/%EB%B6%84%EB%A6%AC%EC%A3%BC%EC%9D%98" title="분리주의">분리주의</a>, 민족병합주의(irredentism), <a class="mw-redirect" href="/wiki/%EA%B5%B0%EC%82%AC%EC%A3%BC%EC%9D%98" title="군사주의">군사주의</a>등을 포함할 수 있으며 극단적인 경우 <a class="mw-redirect" href="/wiki/%EB%AF%BC%EC%A1%B1%EC%B2%AD%EC%86%8C" title="민족청소">민족청소</a>의 형태를 취한다. 정치학자들(그리고 언론 매체들은) 일반적으로 이러한 극단적인 민족주의의 발현에 집중하는 경향이 있다.</p>
<h2><span class="mw-headline" id=".EA.B5.AD.EB.AF.BC.EC.A3.BC.EC.9D.98.EC.9D.98_.EC.97.AD.EC.82.AC.EC.A0.81_.EB.B0.9C.EC.A0.84">국민주의의 역사적 발전</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%82%B4%EC%85%94%EB%84%90%EB%A6%AC%EC%A6%98&amp;action=edit&amp;section=3" title="부분 편집: 국민주의의 역사적 발전">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<h3><span class="mw-headline" id=".EA.B3.A0.EB.8C.80">고대</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%82%B4%EC%85%94%EB%84%90%EB%A6%AC%EC%A6%98&amp;action=edit&amp;section=4" title="부분 편집: 고대">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>국민주의는 처음에 <a href="/wiki/%EC%8B%A0%ED%99%94" title="신화">신화</a>등을 통해 <a class="mw-redirect" href="/wiki/%EB%AF%BC%EC%A1%B1%EC%A2%85%EA%B5%90" title="민족종교">민족종교</a>로 세계 여러 <a href="/wiki/%EC%A2%85%EA%B5%90" title="종교">종교</a>들의 시초가 되었다.</p>
<h3><span class="mw-headline" id="1900.EB.85.84.EB.8C.80_.EC.9D.B4.EC.A0.84">1900년대 이전</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%82%B4%EC%85%94%EB%84%90%EB%A6%AC%EC%A6%98&amp;action=edit&amp;section=5" title="부분 편집: 1900년대 이전">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>17세기 영국에서 시작된 반봉건적 정치이론으로 왕이나 일부 특권층에 속한 국가를 민족의 것으로 바꾼 이론으로, 18세기 <a class="mw-redirect" href="/wiki/%ED%94%84%EB%9E%91%EC%8A%A4_%EB%8C%80%ED%98%81%EB%AA%85" title="프랑스 대혁명">프랑스 대혁명</a>을 통해 발전하여, 이때 프랑스에서의 국민주의는 애민,애국심으로 혁명을 일으키고 이를 지키는 것이었다.</p>
<h3><span class="mw-headline" id="20.EC.84.B8.EA.B8.B0_.EA.B5.AD.EB.AF.BC.EC.A3.BC.EC.9D.98">20세기 국민주의</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%82%B4%EC%85%94%EB%84%90%EB%A6%AC%EC%A6%98&amp;action=edit&amp;section=6" title="부분 편집: 20세기 국민주의">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>18세기 이후 국민주의는 <a href="/wiki/%EB%AF%B8%EA%B5%AD_%EB%8F%85%EB%A6%BD_%ED%98%81%EB%AA%85" title="미국 독립 혁명">미국 독립 혁명</a> 등에도 기여를 한 반면, 민족국가의 이익을 위해 다른 약소국가를 침략,지배해야한다는 <a href="/wiki/%EC%A0%9C%EA%B5%AD%EC%A3%BC%EC%9D%98" title="제국주의">제국주의</a>, <a href="/wiki/%ED%8C%8C%EC%8B%9C%EC%A6%98" title="파시즘">파시즘</a>, <a href="/wiki/%EB%82%98%EC%B9%98%EC%A6%98" title="나치즘">나치즘</a> 등의 국가주의로 변질되고,사회의 빈부격차를 은폐해서 계급투쟁을 희석시키는 데 악용되는 등 서구에서는 부정적 의미로 받아들여진다. <a href="/wiki/%EB%AF%B8%EA%B5%AD" title="미국">미국</a>, <a href="/wiki/%EC%A4%91%EA%B5%AD" title="중국">중국</a>, <a href="/wiki/%EB%9F%AC%EC%8B%9C%EC%95%84" title="러시아">러시아</a>등의 국가에서 국민주의가 <a class="mw-redirect" href="/wiki/%EC%9D%B8%EC%A2%85%EC%B0%A8%EB%B3%84%EC%A3%BC%EC%9D%98" title="인종차별주의">인종차별주의</a>로 변모되어, 이를 주도하는 정부, 단체, 자본가들로 인해 사회문제가 되기도 하였다.</p>
<p>여러 <a class="mw-redirect" href="/wiki/%EC%84%B8%EA%B3%84%EB%8C%80%EC%A0%84" title="세계대전">세계대전</a>을 거치고 이후 식민지 해방의 합의사안으로 <a class="mw-redirect" href="/wiki/%EB%AF%BC%EC%A1%B1%EC%9E%90%EA%B2%B0%EC%A3%BC%EC%9D%98" title="민족자결주의">민족자결주의</a>가 합의되었으며, <a class="mw-redirect" href="/wiki/%EC%84%B8%EA%B3%84%EB%8C%80%EC%A0%84" title="세계대전">세계대전</a> 당시 제국주의를 비판하면서 <a class="mw-redirect" href="/wiki/%EB%B3%BC%EC%85%B0%EB%B9%84%ED%82%A4_%ED%98%81%EB%AA%85" title="볼셰비키 혁명">볼셰비키 혁명</a>과 <a class="mw-redirect" href="/wiki/UN" title="UN">UN</a><a class="mw-redirect" href="/wiki/%EC%84%B8%EA%B3%84%EC%A3%BC%EC%9D%98" title="세계주의">세계주의</a> (<a href="/wiki/%EB%83%89%EC%A0%84" title="냉전">냉전</a>)이념에도 영향을 주었다.</p>
<h3><span class="mw-headline" id="21.EC.84.B8.EA.B8.B0_.EA.B5.AD.EB.AF.BC.EC.A3.BC.EC.9D.98">21세기 국민주의</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%82%B4%EC%85%94%EB%84%90%EB%A6%AC%EC%A6%98&amp;action=edit&amp;section=7" title="부분 편집: 21세기 국민주의">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p><a href="/wiki/%EC%8B%A0%EC%9E%90%EC%9C%A0%EC%A3%BC%EC%9D%98" title="신자유주의">신자유주의</a>와 <a href="/wiki/%EA%B3%B5%EC%82%B0%EC%A3%BC%EC%9D%98" title="공산주의">공산주의</a>가 비판받으면서 국민주의의 올바른 발전을 위해 열린국민주의와 자주적세계화라는 대안이 제시되기도하였다.</p>
<h2><span class="mw-headline" id=".EA.B5.AD.EB.AF.BC.EC.A3.BC.EC.9D.98.EC.9D.98_.EC.A2.85.EB.A5.98">국민주의의 종류</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%82%B4%EC%85%94%EB%84%90%EB%A6%AC%EC%A6%98&amp;action=edit&amp;section=8" title="부분 편집: 국민주의의 종류">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>고전적 의미의 국가주의와 달리, 식민지에서는 국민주의는 반제국주의적 계급지향주의적 성격으로, 19세기 말에서 20세기 초에 걸쳐서 아시아에서, 아프리카, 라틴아메리카 등의 식민지·반식민지 세계의 전역에 퍼져 나가 민족해방의 이데올로기가 되어 통일 독립국가 건설에 기여하였다.</p>
