<p><a href="/wiki/%EC%84%A0%ED%98%95%EB%8C%80%EC%88%98%ED%95%99" title="선형대수학">선형대수학</a>에서, <b>내적 공간</b>(內積空間, <span style="font-size: smaller;"><a href="/wiki/%EC%98%81%EC%96%B4" title="영어">영어</a>:</span> <span lang="en" xml:lang="en">inner product space</span>)은 두 벡터를 곱해 <a class="mw-disambig" href="/wiki/%EC%8A%A4%EC%B9%BC%EB%9D%BC" title="스칼라">스칼라</a>를 얻는 <b>내적</b>이라는 <a class="mw-redirect" href="/wiki/%EC%9D%B4%ED%95%AD%EC%97%B0%EC%82%B0" title="이항연산">이항연산</a>이 주어진 <a href="/wiki/%EB%B2%A1%ED%84%B0_%EA%B3%B5%EA%B0%84" title="벡터 공간">벡터 공간</a>이다. 벡터 공간에 내적이 주어지면 이를 이용해 <a href="/wiki/%EA%B8%B8%EC%9D%B4" title="길이">길이</a>나 <a href="/wiki/%EA%B0%81%EB%8F%84" title="각도">각도</a> 등의 개념을 정의할 수 있으며, 이는 <a href="/wiki/%EC%9C%A0%ED%81%B4%EB%A6%AC%EB%93%9C_%EA%B3%B5%EA%B0%84" title="유클리드 공간">유클리드 공간</a>의 <a class="mw-redirect" href="/wiki/%EC%8A%A4%EC%B9%BC%EB%9D%BC_%EA%B3%B1" title="스칼라 곱">스칼라 곱</a>을 일반화한 것으로 볼 수 있다. (주의: 책에 따라 내적과 스칼라 곱을 동의어로 여기기도 한다.). 내적공간의 개념은 <a href="/wiki/%ED%95%A8%EC%88%98%ED%95%B4%EC%84%9D%ED%95%99" title="함수해석학">함수해석학</a>에서 중요하게 다루어진다.</p>
<h2><span class="mw-headline" id=".EC.A0.95.EC.9D.98">정의</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%82%B4%EC%A0%81%EA%B3%B5%EA%B0%84&amp;action=edit&amp;section=1" title="부분 편집: 정의">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>(이 글에서 <a href="/wiki/%EC%8A%A4%EC%B9%BC%EB%9D%BC_(%EC%88%98%ED%95%99)" title="스칼라 (수학)">스칼라</a>들의 <a href="/wiki/%EC%B2%B4_(%EC%88%98%ED%95%99)" title="체 (수학)">체</a> <b>F</b>는 <a class="mw-redirect" href="/wiki/%EC%8B%A4%EC%88%98%EC%B2%B4" title="실수체">실수체</a> <b>R</b> 혹은 <a class="mw-redirect" href="/wiki/%EB%B3%B5%EC%86%8C%EC%88%98%EC%B2%B4" title="복소수체">복소수체</a> <b>C</b>이다.) 체 F 상의 벡터 공간 V에 <a class="new" href="/w/index.php?title=%EC%A0%95%EB%B6%80%ED%98%B8&amp;action=edit&amp;redlink=1" title="정부호 (없는 문서)">정부호</a> <a class="mw-redirect" href="/wiki/%EB%B9%84%ED%87%B4%ED%99%94_%EC%8C%8D%EC%84%A0%ED%98%95_%ED%98%95%EC%8B%9D" title="비퇴화 쌍선형 형식">비퇴화</a> <a class="new" href="/w/index.php?title=%EC%A0%95%EB%B0%98%EC%84%A0%ED%98%95_%ED%98%95%EC%8B%9D&amp;action=edit&amp;redlink=1" title="정반선형 형식 (없는 문서)">정반선형 형식</a> &lt;·,·&gt;이 주어지면 이 공간을 <b>내적 공간</b>이라 하고, &lt;·,·&gt;를 <b>내적</b>이라 한다. 이는 실수 벡터 공간에 대해서는 정부호 <a class="mw-redirect" href="/wiki/%EB%B9%84%ED%87%B4%ED%99%94_%EC%8C%8D%EC%84%A0%ED%98%95_%ED%98%95%EC%8B%9D" title="비퇴화 쌍선형 형식">비퇴화</a> <a class="mw-redirect" href="/wiki/%EB%8C%80%EC%B9%AD_%EC%8C%8D%EC%84%A0%ED%98%95_%ED%98%95%EC%8B%9D" title="대칭 쌍선형 형식">대칭 쌍선형 형식</a>이 된다. 위의 내적의 정의를 보다 기초적인 용어들로 아래와 같이 풀어 쓸 수 있다.</p>
