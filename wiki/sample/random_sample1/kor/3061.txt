<p><b>낸시 패트리샤 달레산드로 펠로시</b>(Nancy Patricia D'Alesandro Pelosi, <a href="/wiki/1940%EB%85%84" title="1940년">1940년</a> <a href="/wiki/3%EC%9B%94_26%EC%9D%BC" title="3월 26일">3월 26일</a> ~ )는 <a href="/wiki/%EB%AF%B8%EA%B5%AD" title="미국">미국</a>의 여성 정치인이다. <a href="/wiki/%EB%AF%BC%EC%A3%BC%EB%8B%B9_(%EB%AF%B8%EA%B5%AD)" title="민주당 (미국)">민주당</a> 소속으로 <a href="/wiki/1987%EB%85%84" title="1987년">1987년</a> 연방 <a href="/wiki/%EB%AF%B8%EA%B5%AD_%ED%95%98%EC%9B%90" title="미국 하원">하원의원</a>으로 선출된 이후 하원에서 폭넓은 활동을 해오고 있으며, <a href="/wiki/2007%EB%85%84" title="2007년">2007년</a> ~ <a href="/wiki/2011%EB%85%84" title="2011년">2011년</a> 미국에서 여성으로는 사상 처음으로 <a class="mw-redirect" href="/wiki/%EB%AF%B8%EA%B5%AD%EC%9D%98_%EC%97%B0%EB%B0%A9_%ED%95%98%EC%9B%90%EC%9D%98%EC%9E%A5" title="미국의 연방 하원의장">연방 하원의장</a>으로 재직하기도 하였다. 이로써 그는 현재까지 미국에서 가장 높은 지위에 오른 여성으로 기록되어 있다.</p>
<h2><span class="mw-headline" id=".EC.83.9D.EC.95.A0">생애</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%82%B8%EC%8B%9C_%ED%8E%A0%EB%A1%9C%EC%8B%9C&amp;action=edit&amp;section=1" title="부분 편집: 생애">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p><a href="/wiki/%EB%A9%94%EB%A6%B4%EB%9E%9C%EB%93%9C_%EC%A3%BC" title="메릴랜드 주">메릴랜드 주</a> <a href="/wiki/%EB%B3%BC%ED%8B%B0%EB%AA%A8%EC%96%B4" title="볼티모어">볼티모어</a> 출신이다. <a href="/wiki/%EC%9D%B4%ED%83%88%EB%A6%AC%EC%95%84" title="이탈리아">이탈리아</a> 이민의 후손으로, 부친 <a class="new" href="/w/index.php?title=%ED%86%A0%EB%A8%B8%EC%8A%A4_%EB%8B%AC%EB%A0%88%EC%82%B0%EB%93%9C%EB%A1%9C&amp;action=edit&amp;redlink=1" title="토머스 달레산드로 (없는 문서)">토머스 달레산드로</a>는 그가 태어났을 당시 하원의원으로 재직 중이었다. 부친은 <a href="/wiki/1947%EB%85%84" title="1947년">1947년</a> ~ <a href="/wiki/1959%EB%85%84" title="1959년">1959년</a> 볼티모어 시장을 지냈고, 메릴랜드 주 대표 <a href="/wiki/%EB%AF%B8%EA%B5%AD_%EC%83%81%EC%9B%90" title="미국 상원">상원의원</a> 선거에 출마했으나, 낙선했다. 이에 따라 낸시는 아버지의 선거 과정을 지켜보며 나름대로 정치에 대한 꿈을 키웠다. 그러나 <a class="mw-disambig" href="/wiki/%ED%8A%B8%EB%A6%AC%EB%8B%88%ED%8B%B0_%EB%8C%80%ED%95%99%EA%B5%90" title="트리니티 대학교">트리니티 대학교</a>를 졸업하고 폴 펠로시와 결혼한 후로는 한동안 자녀들을 낳고 주부로 지냈다.</p>
<p>낸시 펠로시는 금융업에 종사하는 남편을 따라 <a class="mw-redirect" href="/wiki/%EB%89%B4%EC%9A%95_(%EB%8F%84%EC%8B%9C)" title="뉴욕 (도시)">뉴욕</a>에서 거주하다가 남편의 고향인 <a href="/wiki/%EC%BA%98%EB%A6%AC%ED%8F%AC%EB%8B%88%EC%95%84_%EC%A3%BC" title="캘리포니아 주">캘리포니아 주</a> <a href="/wiki/%EC%83%8C%ED%94%84%EB%9E%80%EC%8B%9C%EC%8A%A4%EC%BD%94" title="샌프란시스코">샌프란시스코</a>로 옮겼다. 다섯 자녀를 낳고 민주당에 가입하여 샌프란시스코 지역사회 활동을 했으나, 적극적인 정치활동은 하지 않고 차분히 지역에서 인맥을 쌓았다. <a href="/wiki/1987%EB%85%84" title="1987년">1987년</a> 샌프란시스코를 지역구로 하는 연방 하원의원으로 여성인 <a class="new" href="/w/index.php?title=%EC%83%90%EB%9D%BC_%EB%B2%84%ED%8A%BC&amp;action=edit&amp;redlink=1" title="샐라 버튼 (없는 문서)">샐라 버튼</a>이 사망하자 보궐 선거에 출마하여 민주당 지지율이 높은 지역에서 어렵지 않게 당선되었다. 이후 펠로시는 2년 주기로 치러지는 선거에서 70~80%의 압도적인 득표율로 계속 당선되었다.</p>
<p>하원 진출 후 경력을 쌓은 펠로시는 <a href="/wiki/2001%EB%85%84" title="2001년">2001년</a> 하원 민주당 원내총무로 선출되어, 하원에서 민주당 내 서열 2위에 올랐으며, 미국에서 여성으로는 처음으로 양대 정당의 원내 총무가 되었다. <a href="/wiki/2003%EB%85%84" title="2003년">2003년</a> 하원 민주당 대표로 선출되어 하원에서 민주당 최고 자리에 올랐으며, <a href="/wiki/%EA%B3%B5%ED%99%94%EB%8B%B9_(%EB%AF%B8%EA%B5%AD)" title="공화당 (미국)">공화당</a> 정권과 대립하였다. 그는 민주당 내에서도 가장 진보적인 성향을 가진 의원으로 알려져 있으며, <a href="/wiki/%EC%A1%B0%EC%A7%80_W._%EB%B6%80%EC%8B%9C" title="조지 W. 부시">조지 W. 부시</a> 대통령의 <a href="/wiki/%EC%9D%B4%EB%9D%BC%ED%81%AC_%EC%A0%84%EC%9F%81" title="이라크 전쟁">이라크 전쟁</a> 정책을 비판하고 <a href="/wiki/%EA%B0%80%ED%86%A8%EB%A6%AD" title="가톨릭">가톨릭</a> 신자면서도 <a class="mw-redirect" href="/wiki/%EB%8F%99%EC%84%B1_%EA%B2%B0%ED%98%BC" title="동성 결혼">동성 결혼</a>이나 <a href="/wiki/%EB%82%99%ED%83%9C" title="낙태">낙태</a>를 적극적으로 찬성하였다. 이라크 전쟁이 큰 이슈였던 <a href="/wiki/2006%EB%85%84" title="2006년">2006년</a> 중간선거에서 민주당은 31석을 추가하며 총 233석을 확보, 202석을 확보한 공화당을 압도하였다. 이에 따라 하원 민주당 대표인 그는 <a href="/wiki/2007%EB%85%84" title="2007년">2007년</a> 시작되는 제110대 회기부터 미국 연방 하원의장에 취임하였다. 연방 하원의장은 <a href="/wiki/%EB%AF%B8%EA%B5%AD%EC%9D%98_%EB%8C%80%ED%86%B5%EB%A0%B9" title="미국의 대통령">대통령</a>과 <a href="/wiki/%EB%AF%B8%EA%B5%AD%EC%9D%98_%EB%B6%80%ED%86%B5%EB%A0%B9" title="미국의 부통령">부통령</a> 다음가는 권력 서열 3위로, 그는 미국에서 가장 높은 지위에 오른 여성이 되었다.</p>
<p>하원의장 취임 후 그는 이전에 소극적인 처리로 통과가 늦어졌던 <a class="mw-redirect" href="/wiki/%EC%9D%BC%EB%B3%B8%EA%B5%B0_%EC%9C%84%EC%95%88%EB%B6%80" title="일본군 위안부">일본군 위안부</a> 결의안을 통과시키도록 하는 데 큰 영향력을 행사하여, <a href="/wiki/2007%EB%85%84" title="2007년">2007년</a> <a href="/wiki/7%EC%9B%94_30%EC%9D%BC" title="7월 30일">7월 30일</a> 결의안은 하원에서 만장일치로 통과되었다. <a href="/wiki/2008%EB%85%84_%EB%AF%B8%EA%B5%AD_%EB%8C%80%ED%86%B5%EB%A0%B9_%EC%84%A0%EA%B1%B0" title="2008년 미국 대통령 선거">2008년 대통령 선거</a>에서는 <a href="/wiki/%EB%B2%84%EB%9D%BD_%EC%98%A4%EB%B0%94%EB%A7%88" title="버락 오바마">버락 오바마</a> 후보를 적극적으로 지지하였다. <a href="/wiki/2010%EB%85%84" title="2010년">2010년</a> 중간선거에서 하원 다수당 지위를 공화당에 넘겨주어, 하원의장 자리는 공화당의 <a href="/wiki/%EC%A1%B4_%EB%B2%A0%EC%9D%B4%EB%84%88" title="존 베이너">존 베이너</a>에게 넘겨주게 되었다. 대신 그는 다시 하원 민주당 대표로 선출되어<sup class="reference" id="cite_ref-1"><a href="#cite_note-1">[1]</a></sup>, <a href="/wiki/2011%EB%85%84" title="2011년">2011년</a> <a href="/wiki/1%EC%9B%94_3%EC%9D%BC" title="1월 3일">1월 3일</a> 하원의장 자리에서 물러나고 하원 민주당 다수당 대표가 되었다.</p>
