<p><b>네마냐 마티치</b>(<span style="font-size: smaller;"><a href="/wiki/%EC%84%B8%EB%A5%B4%EB%B9%84%EC%95%84%EC%96%B4" title="세르비아어">세르비아어</a>:</span> <span lang="sr" xml:lang="sr">Немања Матић, Nemanja Matić</span>, <a href="/wiki/1988%EB%85%84" title="1988년">1988년</a> <a href="/wiki/8%EC%9B%94_1%EC%9D%BC" title="8월 1일">8월 1일</a> <a href="/wiki/%EC%83%A4%EB%B0%94%EC%B8%A0" title="샤바츠">샤바츠</a> ~ )는 <a href="/wiki/%EC%84%B8%EB%A5%B4%EB%B9%84%EC%95%84" title="세르비아">세르비아</a>의 <a href="/wiki/%EC%B6%95%EA%B5%AC_%EC%84%A0%EC%88%98" title="축구 선수">축구 선수</a>로, 현재 <a href="/wiki/%EC%9E%89%EA%B8%80%EB%9E%9C%EB%93%9C" title="잉글랜드">잉글랜드</a> <a href="/wiki/%ED%94%84%EB%A6%AC%EB%AF%B8%EC%96%B4%EB%A6%AC%EA%B7%B8" title="프리미어리그">프리미어리그</a>의 <a href="/wiki/%EC%B2%BC%EC%8B%9C_FC" title="첼시 FC">첼시</a>에서 <a class="mw-redirect" href="/wiki/%EC%88%98%EB%B9%84%ED%98%95_%EB%AF%B8%EB%93%9C%ED%95%84%EB%8D%94" title="수비형 미드필더">수비형 미드필더</a>로 활약하고 있다.</p>

<p><a href="/wiki/2004%EB%85%84" title="2004년">2004년</a>, 16세의 나이로 마티치는 FK 콜루바라의 1군에 합류하였다. 마티치는 3시즌동안 16경기에 출장하였다. 그는 베오그라드 세르비아 리그 (3부리그) 에서의 인상적인 활약 후, 그는 <a href="/wiki/%EC%8A%AC%EB%A1%9C%EB%B0%94%ED%82%A4%EC%95%84_%EC%88%98%ED%8E%98%EB%A5%B4%EB%A6%AC%EA%B0%80" title="슬로바키아 수페르리가">슬로바키아 수페르리가</a>의 <a class="mw-redirect" href="/wiki/MFK_%EC%BD%94%EC%8B%9C%EC%B2%B4" title="MFK 코시체">MFK 코시체</a>로 이적하였다. 그는 67번의 슬로바키아 리그 경기에 참여하여 4골을 기록하였고, 소속팀이 2008-09 시즌을 4위로 마감하는데 공을 세웠다. 그는 세넥의 NTC 세넥에서 벌어진 슬로박 컵 결승전에서도 <a class="mw-redirect" href="/wiki/FC_%ED%8E%98%ED%8A%B8%EB%A5%B4%EC%9E%98%EC%B9%B4_1898" title="FC 페트르잘카 1898">FC 페트르잘카 1898</a>를 꺾고 소속팀이 우승을 할 수 있도록 하였다. 코시체는 3-1로 승리하여 <a href="/wiki/UEFA_%EC%9C%A0%EB%A1%9C%ED%8C%8C%EB%A6%AC%EA%B7%B8" title="UEFA 유로파리그">UEFA 유로파리그</a>의 첫 시즌에 우승할 수 있게 되었다.</p>
<p>네마냐는 이중국적자이다. 그는 <a href="/wiki/%EC%84%B8%EB%A5%B4%EB%B9%84%EC%95%84" title="세르비아">세르비아</a> 출신임에 따라 세르비아 시민권을 가지고 있고, 나중에 <a href="/wiki/%EC%8A%AC%EB%A1%9C%EB%B0%94%ED%82%A4%EC%95%84" title="슬로바키아">슬로바키아</a> 시민권도 획득하였다. 2008-09 시즌 <a href="/wiki/%ED%94%84%EB%A6%AC%EB%AF%B8%EC%96%B4%EB%A6%AC%EA%B7%B8" title="프리미어리그">프리미어리그</a> 트라이얼을 거친 뒤 <a href="/wiki/%EB%AF%B8%EB%93%A4%EC%A6%88%EB%B8%8C%EB%9F%AC_FC" title="미들즈브러 FC">미들즈브러 FC</a>가 가장 먼저 그를 영입하였다. 그러나, 트라이얼은 <a href="/wiki/%EB%8D%B4%EB%A7%88%ED%81%AC_%EC%B6%95%EA%B5%AC_%EA%B5%AD%EA%B0%80%EB%8C%80%ED%91%9C%ED%8C%80" title="덴마크 축구 국가대표팀">덴마크</a>전을 앞둔 <a href="/wiki/%EC%84%B8%EB%A5%B4%EB%B9%84%EC%95%84_%EC%B6%95%EA%B5%AC_%EA%B5%AD%EA%B0%80%EB%8C%80%ED%91%9C%ED%8C%80" title="세르비아 축구 국가대표팀">세르비아 국가대표팀</a>의 차출로 인해 중단되었다. <a href="/wiki/2009%EB%85%84" title="2009년">2009년</a> 여름, 마티치가 <a href="/wiki/%ED%94%84%EB%A6%AC%EB%AF%B8%EC%96%B4%EB%A6%AC%EA%B7%B8" title="프리미어리그">프리미어리그</a>의 <a href="/wiki/%ED%86%A0%ED%8A%B8%EB%84%98_%ED%99%8B%EC%8A%A4%ED%8D%BC_FC" title="토트넘 홋스퍼 FC">토트넘 홋스퍼 FC</a>나 <a href="/wiki/%EC%B2%BC%EC%8B%9C_FC" title="첼시 FC">첼시 FC</a>로 이적할 것이라는 소문이 돌았다.</p>
<h3><span class="mw-headline" id=".EC.B2.BC.EC.8B.9C_FC">첼시 FC</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%84%A4%EB%A7%88%EB%83%90_%EB%A7%88%ED%8B%B0%EC%B9%98&amp;action=edit&amp;section=2" title="부분 편집: 첼시 FC">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p><a href="/wiki/2009%EB%85%84" title="2009년">2009년</a> <a href="/wiki/8%EC%9B%94_18%EC%9D%BC" title="8월 18일">8월 18일</a>, 마티치는 £1.5M의 가격에 <a href="/wiki/%EC%B2%BC%EC%8B%9C_FC" title="첼시 FC">첼시 FC</a>와 4년 계약을 체결하였다.<sup class="reference" id="cite_ref-CFC1_1-0"><a href="#cite_note-CFC1-1">[1]</a></sup><sup class="reference" id="cite_ref-CFC2_2-0"><a href="#cite_note-CFC2-2">[2]</a></sup> 그는 24번 유니폼을 받았지만, 그는 UEFA U-21 챔피언쉽에서 부상을 당하는 바람에 데뷔가 연기되었다. 그는 <a href="/wiki/2009%EB%85%84" title="2009년">2009년</a> <a href="/wiki/9%EC%9B%94_23%EC%9D%BC" title="9월 23일">9월 23일</a>, 1-0 승리로 끝난 <a class="mw-redirect" href="/wiki/%ED%80%B8%EC%A6%88_%ED%8C%8C%ED%81%AC_%EB%A0%88%EC%9D%B8%EC%A0%80%EC%8A%A4_FC" title="퀸즈 파크 레인저스 FC">퀸즈 파크 레인저스 FC</a>와의 <a class="mw-redirect" href="/wiki/%ED%92%8B%EB%B3%BC_%EB%A6%AC%EA%B7%B8_%EC%BB%B5" title="풋볼 리그 컵">리그 컵</a> 경기에서 벤치를 지켰다. 마티치는 <a href="/wiki/2009%EB%85%84" title="2009년">2009년</a> <a href="/wiki/11%EC%9B%94_21%EC%9D%BC" title="11월 21일">11월 21일</a>, <a href="/wiki/%EC%9A%B8%EB%B2%84%ED%96%84%ED%94%84%ED%84%B4_%EC%9B%90%EB%8D%94%EB%9F%AC%EC%8A%A4_FC" title="울버햄프턴 원더러스 FC">울버햄프턴 원더러스 FC</a>전에서 69분 <a href="/wiki/%ED%94%8C%EB%A1%9C%EB%9E%91_%EB%A7%90%EB%A3%A8%EB%8B%A4" title="플로랑 말루다">플로랑 말루다</a>와 교체 투입되어 데뷔전을 치루었고, 팀은 4-0 승리를 거두었다.<sup class="reference" id="cite_ref-EPLdebut_3-0"><a href="#cite_note-EPLdebut-3">[3]</a></sup> 그는 <a href="/wiki/%EC%9C%84%EA%B1%B4_%EC%95%A0%EC%8A%AC%EB%A0%88%ED%8B%B1_FC" title="위건 애슬레틱 FC">위건 애슬레틱 FC</a>와의 2009-10시즌 리그 마지막 경기에 출장하였고, 이날 첼시는 8-0로 크게 이겨 프리미어리그 우승 트로피를 들어올렸다.</p>
<h3><span class="mw-headline" id=".ED.94.BC.ED.85.8C.EC.84.9C">피테서</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%84%A4%EB%A7%88%EB%83%90_%EB%A7%88%ED%8B%B0%EC%B9%98&amp;action=edit&amp;section=3" title="부분 편집: 피테서">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p><a href="/wiki/2010%EB%85%84" title="2010년">2010년</a> <a href="/wiki/8%EC%9B%94_23%EC%9D%BC" title="8월 23일">8월 23일</a>, 마티치는 첼시 동료 <a class="new" href="/w/index.php?title=%EC%8A%AC%EB%A1%9C%EB%B3%B4%EB%8B%A8_%EB%9D%BC%EC%9D%B4%EC%BD%94%EB%B9%84%EC%B9%98&amp;action=edit&amp;redlink=1" title="슬로보단 라이코비치 (없는 문서)">슬로보단 라이코비치</a>와 <a href="/wiki/%EB%A7%88%ED%85%8C%EC%9D%B4_%EB%8D%B8%EB%9D%BC%EC%B9%98" title="마테이 델라치">마테이 델라치</a>와 함께 <a href="/wiki/%EC%97%90%EB%A0%88%EB%94%94%EB%B9%84%EC%8B%9C" title="에레디비시">에레디비시</a>의 <a href="/wiki/%ED%94%BC%ED%85%8C%EC%84%9C" title="피테서">피테서</a>로 1년동안 임대되었다.<sup class="reference" id="cite_ref-Vitesse_4-0"><a href="#cite_note-Vitesse-4">[4]</a></sup></p>
<h3><span class="mw-headline" id=".EB.B2.A4.ED.94.BC.EC.B9.B4">벤피카</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%84%A4%EB%A7%88%EB%83%90_%EB%A7%88%ED%8B%B0%EC%B9%98&amp;action=edit&amp;section=4" title="부분 편집: 벤피카">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>2011년에는 첼시가 <a href="/wiki/%EB%8B%A4%EB%B9%84%EB%93%9C_%EB%A3%A8%EC%9D%B4%EC%8A%A4" title="다비드 루이스">다비드 루이스</a>를 영입하면서 그 대가로 벤피카로 팀을 옮겼다.</p>
<h3><span class="mw-headline" id=".EC.B2.BC.EC.8B.9C.EB.A1.9C.EC.9D.98_.EB.B3.B5.EA.B7.80">첼시로의 복귀</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%84%A4%EB%A7%88%EB%83%90_%EB%A7%88%ED%8B%B0%EC%B9%98&amp;action=edit&amp;section=5" title="부분 편집: 첼시로의 복귀">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>1월 겨울이적시장을 통해 첼시로 다시 복귀하였다. 복귀 후 첫 선발인 fa컵 <a href="/wiki/%EC%8A%A4%ED%86%A0%ED%81%AC_%EC%8B%9C%ED%8B%B0_FC" title="스토크 시티 FC">스토크 시티 FC</a>전에서는 팀의 1:0 무실점 승리를 이끌어내는 활약을 펼치며 좋은 신호탄을 올렸다. 그리고 그는 2014년 2월 4일 <a class="mw-redirect" href="/wiki/%EB%A7%A8%EC%B2%B4%EC%8A%A4%ED%84%B0%EC%8B%9C%ED%8B%B0" title="맨체스터시티">맨체스터시티</a>와의 리그 경기에서 <a href="/wiki/%EB%8B%A4%EB%B9%84%EB%93%9C_%EB%A3%A8%EC%9D%B4%EC%8A%A4" title="다비드 루이스">다비드 루이스</a>와 더블볼란치를 이뤘는데 엄청난 활동량과 피지컬,높이(다비드루이스 189 cm,마티치 194 cm)로 상대 맨체스터시티의 최고 선수 야야투레를 완벽히 봉쇄하면서 밀리지 않는 중원 싸움을 보여주었다. 리그레이스에 중요한 결전인 이 맞대결 mom으로 선정 되면서 마티치 라는 이름 세글자를 세상에 알렸다. 그 후로 그는 매경기 기복 없는 좋은 경기력을 펼치며 첼시의 수비불안,중원문제를 한꺼번에 덜어 내줬고 2014/2015시즌 시작전에 영입된 <a href="/wiki/%EC%84%B8%EC%8A%A4%ED%81%AC_%ED%8C%8C%EB%B8%8C%EB%A0%88%EA%B0%80%EC%8A%A4" title="세스크 파브레가스">세스크 파브레가스</a>와 더블 볼란치를 이뤄 현재 세계 최고수준의 중원을 이루고 있다.</p>
<h2><span class="mw-headline" id=".EA.B5.AD.EA.B0.80.EB.8C.80.ED.91.9C_.EA.B2.BD.EB.A0.A5">국가대표 경력</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%84%A4%EB%A7%88%EB%83%90_%EB%A7%88%ED%8B%B0%EC%B9%98&amp;action=edit&amp;section=6" title="부분 편집: 국가대표 경력">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>2008-09 시즌 <a href="/wiki/FC_%EC%8A%A4%ED%8C%8C%EB%A5%B4%ED%83%80%ED%81%AC_%ED%8A%B8%EB%A5%B4%EB%82%98%EB%B0%94" title="FC 스파르타크 트르나바">FC 스파르타크 트르나바</a>의 감독 <a class="new" href="/w/index.php?title=%EB%B8%94%EB%9D%BC%EB%94%94%EB%AF%B8%EB%A5%B4_%EB%B2%A0%EB%A5%B4%EB%A9%94%EC%A1%B0%EB%B9%84%EC%B9%98&amp;action=edit&amp;redlink=1" title="블라디미르 베르메조비치 (없는 문서)">블라디미르 베르메조비치</a>는 그를 세르비아 감독에게 추천하였다. 네마냐는 당시 세르비아 국가대표팀에서 생소한 이름이었다. <a href="/wiki/2008%EB%85%84" title="2008년">2008년</a> <a href="/wiki/10%EC%9B%94_11%EC%9D%BC" title="10월 11일">10월 11일</a>, 마티치는 <a href="/wiki/%EC%84%B8%EB%A5%B4%EB%B9%84%EC%95%84_%EC%B6%95%EA%B5%AC_%EA%B5%AD%EA%B0%80%EB%8C%80%ED%91%9C%ED%8C%80" title="세르비아 축구 국가대표팀">세르비아 U-21 대표팀</a>에 차출되어 <a href="/wiki/%EB%8D%B4%EB%A7%88%ED%81%AC_%EC%B6%95%EA%B5%AC_%EA%B5%AD%EA%B0%80%EB%8C%80%ED%91%9C%ED%8C%80" title="덴마크 축구 국가대표팀">덴마크 U-21 대표팀</a>과의 경기에 출전하였다.<sup class="reference" id="cite_ref-U21_5-0"><a href="#cite_note-U21-5">[5]</a></sup> 세르비아 U-21 대표팀 소속으로 3경기 출장과 2골을 기록한 그는 <a href="/wiki/2008%EB%85%84" title="2008년">2008년</a> <a href="/wiki/12%EC%9B%94_14%EC%9D%BC" title="12월 14일">12월 14일</a>, <a href="/wiki/%ED%8F%B4%EB%9E%80%EB%93%9C_%EC%B6%95%EA%B5%AC_%EA%B5%AD%EA%B0%80%EB%8C%80%ED%91%9C%ED%8C%80" title="폴란드 축구 국가대표팀">폴란드</a>와의 친선경기를 앞두고 <a href="/wiki/%EC%84%B8%EB%A5%B4%EB%B9%84%EC%95%84_%EC%B6%95%EA%B5%AC_%EA%B5%AD%EA%B0%80%EB%8C%80%ED%91%9C%ED%8C%80" title="세르비아 축구 국가대표팀">세르비아 성인대표팀</a>에 차출되었다.<sup class="reference" id="cite_ref-SRBsenior_6-0"><a href="#cite_note-SRBsenior-6">[6]</a></sup></p>
<p>마티치는 2009년 UEFA U-21 챔피언쉽에 참가하여, <a href="/wiki/%EC%9D%B4%ED%83%88%EB%A6%AC%EC%95%84_%EC%B6%95%EA%B5%AC_%EA%B5%AD%EA%B0%80%EB%8C%80%ED%91%9C%ED%8C%80" title="이탈리아 축구 국가대표팀">이탈리아</a>와의 첫 조별리그 경기 (0-0) 에서 85분 출장하였다. 마티치는 이탈리아의 <a href="/wiki/%EC%84%B8%EB%B0%94%EC%8A%A4%ED%8B%B0%EC%95%88_%EC%A1%B0%EB%B9%88%EC%BD%94" title="세바스티안 조빈코">세바스티안 조빈코</a>와 충돌하여 부상당하였다. 그는 오른발의 5번째 중족골 골절로 수술을 받았다.<sup class="reference" id="cite_ref-injury_7-0"><a href="#cite_note-injury-7">[7]</a></sup></p>
