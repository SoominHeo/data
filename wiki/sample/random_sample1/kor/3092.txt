<p><b>네벤 수보티치</b> (Neven Subotić, <a href="/wiki/1988%EB%85%84" title="1988년">1988년</a> <a href="/wiki/12%EC%9B%94_10%EC%9D%BC" title="12월 10일">12월 10일</a>, <a href="/wiki/%EB%B0%94%EB%83%90%EB%A3%A8%EC%B9%B4" title="바냐루카">바냐루카</a> ~ )는 <a href="/wiki/%EC%84%B8%EB%A5%B4%EB%B9%84%EC%95%84" title="세르비아">세르비아</a>의 축구 선수로, 현재 <a class="mw-redirect" href="/wiki/%EB%8F%85%EC%9D%BC_%EB%B6%84%EB%8D%B0%EC%8A%A4%EB%A6%AC%EA%B0%80" title="독일 분데스리가">독일 분데스리가</a>의 <a href="/wiki/1._FC_%EC%BE%B0%EB%A5%B8" title="1. FC 쾰른">쾰른</a> 소속 <a href="/wiki/%EC%88%98%EB%B9%84%EC%88%98" title="수비수">수비수</a>이다. 2006-07 시즌 <a href="/wiki/1._FSV_%EB%A7%88%EC%9D%B8%EC%B8%A0_05" title="1. FSV 마인츠 05">마인츠</a>의 선수로 데뷔하였고, 2008년부터 현재까지 <a href="/wiki/%EB%B3%B4%EB%A3%A8%EC%8B%9C%EC%95%84_%EB%8F%84%EB%A5%B4%ED%8A%B8%EB%AC%B8%ED%8A%B8" title="보루시아 도르트문트">보루시아 도르트문트</a>의 선수로 활동하고 있다. <a href="/wiki/%EC%84%B8%EB%A5%B4%EB%B9%84%EC%95%84_%EC%B6%95%EA%B5%AC_%EA%B5%AD%EA%B0%80%EB%8C%80%ED%91%9C%ED%8C%80" title="세르비아 축구 국가대표팀">세르비아 국가 대표</a> 선수이기도 하다.</p>

<h2><span class="mw-headline" id=".EC.96.B4.EB.A6.B0_.EC.8B.9C.EC.A0.88">어린 시절</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%84%A4%EB%B2%A4_%EC%88%98%EB%B3%B4%ED%8B%B0%EC%B9%98&amp;action=edit&amp;section=1" title="부분 편집: 어린 시절">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>보스니아계 혈통의 세르비아인으로, <a href="/wiki/%EB%B0%94%EB%83%90%EB%A3%A8%EC%B9%B4" title="바냐루카">바냐루카</a>에서 태어났다. 1994년, 그가 5살일 때 가족은 독일의 쇰베르크로 이주하였다. 7살 때 네벤은 TSV 슈바르첸부르크에서 축구하였다.</p>
<p>90년대 말, 그의 가족의 체류 허가 기한이 만료되자, 보스니아로 돌아가는 대신 1999년에 다시 <a href="/wiki/%EB%AF%B8%EA%B5%AD" title="미국">미국</a>으로 건너갔다. 그의 가족은 네벤의 5촌이 사는 <a href="/wiki/%EC%9C%A0%ED%83%80_%EC%A3%BC" title="유타 주">유타 주</a> <a href="/wiki/%EC%86%94%ED%8A%B8%EB%A0%88%EC%9D%B4%ED%81%AC%EC%8B%9C%ED%8B%B0" title="솔트레이크시티">솔트레이크시티</a>에 정착하였다. 네벤은 그곳에서 스파르타 골드와 임팩트 블랙의 유소년팀에서 뛰기도 하였다.</p>
<p>2년 뒤 가족은 또다시 <a href="/wiki/%ED%94%8C%EB%A1%9C%EB%A6%AC%EB%8B%A4_%EC%A3%BC" title="플로리다 주">플로리다 주</a>의 <a class="new" href="/w/index.php?title=%EB%B8%8C%EB%9E%98%EB%8D%B4%ED%84%B4&amp;action=edit&amp;redlink=1" title="브래덴턴 (없는 문서)">브래덴턴</a>으로 이주하였다. 네벤의 자매 나탈리야는 그곳에서 닉 볼레티에리 테니스 아카데미에 들어가 테니스를 배웠다. 그곳에 있는 동안 그는 G. T. 브레이 파크에서 미국 청소년 국가대표팀의 수석코치 케이스 펄크의 주목을 받고, 존 엘링어 감독이 그를 시험한 뒤 미국의 U-17 대표팀으로 차출하였다. 당시 그는 소속이 없었지만, 얼마 지나지 않아 남 플로리다 대학의 사우스 플로리다에 합류하게 되었다.</p>
<p>네덜란드의 U-17 국가대표팀과의 경기 후, 수보티치는 스티브 켈리로부터 유럽에서 뛸 것을 제안받았다. <a href="/wiki/1._FSV_%EB%A7%88%EC%9D%B8%EC%B8%A0_05" title="1. FSV 마인츠 05">1. FSV 마인츠 05</a>에서 시험을 받은 수보티치는 곧 팀에 합류하였고 마인츠 2군이 있는 4부리그에서 프로 데뷔를 하였다.</p>
<h2><span class="mw-headline" id=".ED.81.B4.EB.9F.BD_.EA.B2.BD.EB.A0.A5">클럽 경력</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%84%A4%EB%B2%A4_%EC%88%98%EB%B3%B4%ED%8B%B0%EC%B9%98&amp;action=edit&amp;section=2" title="부분 편집: 클럽 경력">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<h3><span class="mw-headline" id=".EB.A7.88.EC.9D.B8.EC.B8.A0">마인츠</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%84%A4%EB%B2%A4_%EC%88%98%EB%B3%B4%ED%8B%B0%EC%B9%98&amp;action=edit&amp;section=3" title="부분 편집: 마인츠">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>2006-07시즌의 데뷔는 분데스리가 마지막 상대인 바이에른 뮌헨이었고, 팀은 패하여 강등당했다. 다음 시즌 2부리그에서, 그는 인상적인 모습을 보이며 팀이 37골만을 허용하게 하였다. 팀은 4위로 마무리하여 1부리그 재승격에 실패하였다. 당시 3위 팀과의 승점차는 불과 2점이었다.</p>
<p>2008년 여름, 마인츠의 감독 <a href="/wiki/%EC%9C%84%EB%A5%B4%EA%B2%90_%ED%81%B4%EB%A1%9C%ED%94%84" title="위르겐 클로프">위르겐 클로프</a>가 <a href="/wiki/%EB%B3%B4%EB%A3%A8%EC%8B%9C%EC%95%84_%EB%8F%84%EB%A5%B4%ED%8A%B8%EB%AC%B8%ED%8A%B8" title="보루시아 도르트문트">보루시아 도르트문트</a>의 새 감독으로 내정되었고, 그는 감독을 따라 이적하였다.</p>
<h3><span class="mw-headline" id=".EB.B3.B4.EB.A3.A8.EC.8B.9C.EC.95.84_.EB.8F.84.EB.A5.B4.ED.8A.B8.EB.AC.B8.ED.8A.B8">보루시아 도르트문트</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%84%A4%EB%B2%A4_%EC%88%98%EB%B3%B4%ED%8B%B0%EC%B9%98&amp;action=edit&amp;section=4" title="부분 편집: 보루시아 도르트문트">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p><a href="/wiki/2008%EB%85%84" title="2008년">2008년</a> <a href="/wiki/6%EC%9B%94_4%EC%9D%BC" title="6월 4일">6월 4일</a>, 수보티치는 <a href="/wiki/%EB%B3%B4%EB%A3%A8%EC%8B%9C%EC%95%84_%EB%8F%84%EB%A5%B4%ED%8A%B8%EB%AC%B8%ED%8A%B8" title="보루시아 도르트문트">보루시아 도르트문트</a>와 5년 계약을 맺었다. 그는 바이에른 뮌헨과의 슈퍼컵 경기에서 데뷔하였고, 팀은 2-1로 승리하였다. 이 경기에서 수보티치의 수비는 관심을 모았다. 2008년 12월, 그는 <a href="/wiki/%EB%A3%A8%EC%8B%9C%EC%9A%B0" title="루시우">루시우</a>와 <a href="/wiki/%ED%95%84%EB%A6%AC%ED%94%84_%EB%9E%8C" title="필리프 람">필리프 람</a> 등의 쟁쟁한 수비수들과 더불어 2008-09시즌 전반기 베스트 11으로 뽑혔다. 그 후로도 좋은 모습을 보이며, 저돌적인 공격력도 보였다 (시즌 6골).</p>
<p><a href="/wiki/2009%EB%85%84" title="2009년">2009년</a> 6월, 여름 2014년에 만료되는 새 계약서에 서명하였다. 2009-10시즌, 수보티치는 34경기에 모두 출장한 4명중 1명이 되었다.</p>
<p><a href="/wiki/2010%EB%85%84" title="2010년">2010년</a> <a href="/wiki/12%EC%9B%94_15%EC%9D%BC" title="12월 15일">12월 15일</a>, 수보티치는 유럽대항전 첫 골을 <a href="/wiki/UEFA_%EC%9C%A0%EB%A1%9C%ED%8C%8C%EB%A6%AC%EA%B7%B8_2010-11" title="UEFA 유로파리그 2010-11">UEFA 유로파리그 2010-11</a>의 K조 조별리그 마지막 경기인 <a href="/wiki/%EC%84%B8%EB%B9%84%EC%95%BC_FC" title="세비야 FC">세비야 FC</a>전에서 득점하였다. 그는 헤딩골을 넣었고 경기는 2-2로 종료하였다. 세비야 원정에서 반드시 승리해야 32강에 진출할 수 있는 도르트문트였지만, 무승부를 거두어 최종 조 3위로 32강 진출에 실패하였다. 2010-11 겨울이적시장에서, 부상당한 <a class="mw-redirect" href="/wiki/%EC%95%8C%EB%A0%89%EC%8A%A4_%ED%98%B8%EB%93%9C%EB%A6%AC%EA%B5%AC_%EB%94%94%EC%95%84%EC%8A%A4_%EB%8B%A4_%EC%BD%94%EC%8A%A4%ED%83%80" title="알렉스 호드리구 디아스 다 코스타">알렉스</a>의 대체자를 물색하던 <a href="/wiki/%EB%9F%B0%EB%8D%98" title="런던">런던</a>의 <a href="/wiki/%EC%B2%BC%EC%8B%9C_FC" title="첼시 FC">첼시 FC</a>로부터 제의를 받았고, 2011년 1월 중순에는 £17.5M의 제안을 하였으나 보루시아는 이적을 무산시켰다.</p>
