<p><b>1996년 CONCACAF 골드컵</b>(1996 CONCACAF Gold Cup)은 <a class="mw-redirect" href="/wiki/CONCACAF" title="CONCACAF">CONCACAF</a>가 개최한 세 번째 골드컵으로, <a href="/wiki/1996%EB%85%84" title="1996년">1996년</a> <a href="/wiki/1%EC%9B%94_10%EC%9D%BC" title="1월 10일">1월 10일</a>부터 <a href="/wiki/1%EC%9B%94_21%EC%9D%BC" title="1월 21일">1월 21일</a>까지 <a href="/wiki/%EB%AF%B8%EA%B5%AD" title="미국">미국</a> <a href="/wiki/%EC%BA%98%EB%A6%AC%ED%8F%AC%EB%8B%88%EC%95%84_%EC%A3%BC" title="캘리포니아 주">캘리포니아 주</a> <a href="/wiki/%EB%A1%9C%EC%8A%A4%EC%95%A4%EC%A0%A4%EB%A0%88%EC%8A%A4" title="로스앤젤레스">로스앤젤레스</a>와 <a href="/wiki/%EC%83%8C%EB%94%94%EC%97%90%EC%9D%B4%EA%B3%A0" title="샌디에이고">샌디에이고</a>, <a href="/wiki/%EC%95%A0%EB%84%88%ED%95%98%EC%9E%84" title="애너하임">애너하임</a>에서 개최되었다.</p>
<p>대회의 형식은 9개국이 세 개의 조로 나뉘어 각 조의 1위 팀이 4강에 직행하고, 2위 팀 가운데 성적이 가장 높은 팀이 4강에 진출하는 방식이었다. 이 대회 결승전에서는 <a href="/wiki/%EB%A9%95%EC%8B%9C%EC%BD%94_%EC%B6%95%EA%B5%AC_%EA%B5%AD%EA%B0%80%EB%8C%80%ED%91%9C%ED%8C%80" title="멕시코 축구 국가대표팀">멕시코</a>가 <a href="/wiki/%EB%B8%8C%EB%9D%BC%EC%A7%88_%EC%B6%95%EA%B5%AC_%EA%B5%AD%EA%B0%80%EB%8C%80%ED%91%9C%ED%8C%80" title="브라질 축구 국가대표팀">브라질</a>을 2-0으로 누르고 다섯 번째 우승을 차지했다.</p>
<p>이 대회부터 타 대륙에서 팀을 초청하는 제도가 도입되었다. 이에 따라 <a href="/wiki/%EB%B8%8C%EB%9D%BC%EC%A7%88_%EC%B6%95%EA%B5%AC_%EA%B5%AD%EA%B0%80%EB%8C%80%ED%91%9C%ED%8C%80" title="브라질 축구 국가대표팀">브라질</a>이 초청되었다.</p>
