<h1 class="firstHeading" id="firstHeading" lang="ko">네팔 궁중 학살</h1>
<div class="mw-body-content" id="bodyContent">
<div id="siteSub">위키백과, 우리 모두의 백과사전.</div>
<div id="contentSub"></div>
<div class="mw-jump" id="jump-to-nav">
					이동:					<a href="#mw-head">둘러보기</a>, 					<a href="#p-search">검색</a>
</div>
<div class="mw-content-ltr" dir="ltr" id="mw-content-text" lang="ko"><script>function mfTempOpenSection(id){var block=document.getElementById("mf-section-"+id);block.className+=" open-block";block.previousSibling.className+=" open-block";}</script><p><b>네팔 궁중 학살</b>(Nepalese royal massacre)은 <a href="/wiki/2001%EB%85%84" title="2001년">2001년</a> <a href="/wiki/6%EC%9B%94_1%EC%9D%BC" title="6월 1일">6월 1일</a>에 <a href="/wiki/%EB%84%A4%ED%8C%94" title="네팔">네팔</a>의 <a href="/wiki/%EB%82%98%EB%9D%BC%EC%96%80%ED%9E%88%ED%8B%B0_%EC%99%95%EA%B6%81" title="나라얀히티 왕궁">나라얀히티 왕궁</a>에서 발생한 참극이다. 이 사건으로 <a href="/wiki/%EB%B9%84%EB%A0%8C%EB%93%9C%EB%9D%BC" title="비렌드라">비렌드라</a> 왕과 <a href="/wiki/%EC%95%84%EC%9D%B4%EC%8A%88%EC%99%80%EB%9D%BC" title="아이슈와라">아이슈와라</a> 왕비를 비롯한 9명의 왕족들이 사망하였다. 병원에 실려간 디펜드라 왕세자는 법에 의해 왕이 되었으나, 학살 3일째인 <a href="/wiki/6%EC%9B%94_4%EC%9D%BC" title="6월 4일">6월 4일</a>에 사망했으며 이 사건으로 <a href="/wiki/%EA%B0%B8%EB%84%A8%EB%93%9C%EB%9D%BC" title="갸넨드라">갸넨드라</a>가 다음 왕이 되었다.</p>
<h2><span class="mw-headline" id=".EC.A0.84.EA.B0.9C">전개</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%84%A4%ED%8C%94_%EA%B6%81%EC%A4%91_%ED%95%99%EC%82%B4&amp;action=edit&amp;section=1" title="부분 편집: 전개">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>조사 결과에 의하면, <a href="/wiki/%EB%94%94%ED%8E%9C%EB%93%9C%EB%9D%BC" title="디펜드라">디펜드라</a> 왕자가 취한 채 손님들과 함께 버릇 없이 행동하였고, 그 결과로 그의 아버지인 <a href="/wiki/%EB%B9%84%EB%A0%8C%EB%93%9C%EB%9D%BC" title="비렌드라">비렌드라</a> 왕, 그의 아들은 만찬회장을 떠났다. 취한 디펜드라 왕자는 그의 동생 <a class="new" href="/w/index.php?title=%EB%8B%88%EB%9D%BC%EC%9E%94&amp;action=edit&amp;redlink=1" title="니라잔 (없는 문서)">니라잔</a> 왕자와 사촌 <a class="new" href="/w/index.php?title=%ED%8C%8C%EB%9D%BC%EC%8A%A4_(%EB%84%A4%ED%8C%94%EC%9D%98_%EC%99%95%EC%A1%B1)&amp;action=edit&amp;redlink=1" title="파라스 (네팔의 왕족) (없는 문서)">파라스</a> 왕자에 의해 자신의 방으로 끌려갔다.<sup class="reference" id="cite_ref-one_1-0"><a href="#cite_note-one-1">[1]</a></sup></p>
<p>한 시간 후, 디펜드라는 한 자루의 <a class="mw-redirect" href="/wiki/%ED%97%A4%ED%81%B4%EB%9F%AC%26%EC%BD%94%ED%9D%90_MP5K" title="헤클러&amp;코흐 MP5K">MP5K</a>과 한 자루의 <a href="/wiki/M16_%EC%86%8C%EC%B4%9D" title="M16 소총">M16 소총</a>을 소지하여, 만찬회장에 되돌아왔다. 총을 단발로 천장을 쏜 후 그의 아버지 비렌드라 왕에게 겨누었고 몇 초 후, 디펜드라는 그의 고모를 쏘았고, 그 후 그를 막으려는 삼촌 <a class="new" href="/w/index.php?title=%EB%94%94%EB%A0%8C%EB%93%9C%EB%9D%BC&amp;action=edit&amp;redlink=1" title="디렌드라 (없는 문서)">디렌드라</a>의 흉부를 근거리에서 쏘았다.<sup class="reference" id="cite_ref-one_1-1"><a href="#cite_note-one-1">[1]</a></sup> 총격이 진행되는 동안, 니라잔 왕자의 <a class="new" href="/w/index.php?title=%EC%8A%88%EB%A5%B4%ED%8B%B0&amp;action=edit&amp;redlink=1" title="슈르티 (없는 문서)">슈르티</a> 공주도 총에 맞고 파라스 왕자는 부상이 더 악화되었으나, 소파로 두 아이를 포함해서 적어도 세 명의 왕족을 데려옴으로써, 용케 구할 수 있었다.<sup class="reference" id="cite_ref-one_1-2"><a href="#cite_note-one-1">[1]</a></sup></p>
<h2><span class="mw-headline" id=".EA.B2.B0.EA.B3.BC">결과</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%84%A4%ED%8C%94_%EA%B6%81%EC%A4%91_%ED%95%99%EC%82%B4&amp;action=edit&amp;section=2" title="부분 편집: 결과">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>사건 이후, 비렌드라 국왕의 동생 <a href="/wiki/%EA%B0%B8%EB%84%A8%EB%93%9C%EB%9D%BC" title="갸넨드라">갸넨드라</a>가 즉위하였다.</p>
<h2><span class="mw-headline" id=".EA.B0.81.EC.A3.BC">각주</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%84%A4%ED%8C%94_%EA%B6%81%EC%A4%91_%ED%95%99%EC%82%B4&amp;action=edit&amp;section=3" title="부분 편집: 각주">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
