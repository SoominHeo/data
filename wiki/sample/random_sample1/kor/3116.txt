<p><b>넥센 히어로즈</b>(Nexen Heroes)는 <a href="/wiki/%EB%8C%80%ED%95%9C%EB%AF%BC%EA%B5%AD" title="대한민국">대한민국</a> <a href="/wiki/%EC%84%9C%EC%9A%B8%ED%8A%B9%EB%B3%84%EC%8B%9C" title="서울특별시">서울특별시</a>를 연고지로 하는 프로 <a href="/wiki/%EC%95%BC%EA%B5%AC" title="야구">야구</a>팀이다. <a class="mw-redirect" href="/wiki/KBO_(%EC%95%BC%EA%B5%AC)" title="KBO (야구)">KBO</a> 소속이며, <a href="/wiki/%EC%84%9C%EC%9A%B8%ED%8A%B9%EB%B3%84%EC%8B%9C" title="서울특별시">서울특별시</a> <a href="/wiki/%EA%B5%AC%EB%A1%9C%EA%B5%AC" title="구로구">구로구</a> <a href="/wiki/%EA%B3%A0%EC%B2%99%EB%8F%99" title="고척동">고척동</a>에 있는 <a href="/wiki/%EA%B3%A0%EC%B2%99%EC%8A%A4%EC%B9%B4%EC%9D%B4%EB%8F%94" title="고척스카이돔">고척스카이돔</a>을 홈 구장으로 사용한다. 야구단 자체가 곧 기업인 팀이며, 메인 네이밍 스폰서 기업을 유치하여 운영하고 그 외의 몇몇 서브 스폰서 주체를 유치하여 운영하기 때문에 <a href="/wiki/%EA%B3%A0%EC%B2%99%EC%8A%A4%EC%B9%B4%EC%9D%B4%EB%8F%94" title="고척스카이돔">고척스카이돔</a>에는 메인 스폰서를 포함한 스폰서 기업들의 광고가 붙어 있다.</p>

<h2><span class="mw-headline" id=".EC.97.AD.EC.82.AC">역사</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%84%A5%EC%84%BC_%ED%9E%88%EC%96%B4%EB%A1%9C%EC%A6%88&amp;action=edit&amp;section=1" title="부분 편집: 역사">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p><a href="/wiki/2008%EB%85%84" title="2008년">2008년</a> 3월에 해체되었던 <a href="/wiki/%ED%98%84%EB%8C%80_%EC%9C%A0%EB%8B%88%EC%BD%98%EC%8A%A4" title="현대 유니콘스">현대 유니콘스</a> 선수단의 선수들을 계약하고 재창단하여 만들어졌으며 <a href="/wiki/2008%EB%85%84" title="2008년">2008년</a>부터 <a href="/wiki/2010%EB%85%84" title="2010년">2010년</a>까지 <a href="/wiki/%EB%8C%80%ED%95%9C%EB%AF%BC%EA%B5%AD" title="대한민국">대한민국</a>의 민간 <a href="/wiki/%EB%8B%B4%EB%B0%B0" title="담배">담배</a> 회사인 <a href="/wiki/%EC%9A%B0%EB%A6%AC%EB%8B%B4%EB%B0%B0" title="우리담배">우리담배</a>가 네이밍 스폰서를 맡아 팀 이름이 <b>우리 히어로즈</b>로 결정되었으나 <a href="/wiki/2008%EB%85%84" title="2008년">2008년</a> 시즌 중 구단의 <a class="mw-redirect" href="/wiki/KBO_(%EC%95%BC%EA%B5%AC)" title="KBO (야구)">KBO</a> 가입금 미납을 이유로 <a href="/wiki/%EC%9A%B0%EB%A6%AC%EB%8B%B4%EB%B0%B0" title="우리담배">우리담배</a> 측에서 스폰서 계약을 전격 해지했다. 삼미 슈퍼스타즈, 청보 핀토스, 태평양 돌핀스, 현대 유니콘스 이 네팀 과는 아무런 관련이 없는 팀이다.<sup class="reference" id="cite_ref-2"><a href="#cite_note-2">[2]</a></sup> <a href="/wiki/2008%EB%85%84" title="2008년">2008년</a> 8월부터 스폰서명인 '우리'를 삭제하고 <b>히어로즈</b>로 활동하였다. 대신 이러한 자금난 등으로 인해 몇몇 선수들을 현금 트레이드로 이적시킨 일이 있었다. 구단의 수익 사업의 일환으로 네이밍 스폰서를 유치하고자 팀 명칭상에 연고지인 서울을 생략한 채 활동하였다. <a href="/wiki/2009%EB%85%84" title="2009년">2009년</a> 시즌 종료후 팀 명칭을 공식적으로 "서울 히어로즈"로 확정한다고 선언하였으나, <a href="/wiki/2010%EB%85%84" title="2010년">2010년</a> <a href="/wiki/2%EC%9B%94_8%EC%9D%BC" title="2월 8일">2월 8일</a> 타이어 회사인 <a href="/wiki/%EB%84%A5%EC%84%BC%ED%83%80%EC%9D%B4%EC%96%B4" title="넥센타이어">넥센타이어</a>와 2년간 메인 스폰서십을 계약함으로써 "<b>넥센 히어로즈</b>"로 변경하였다. <a href="/wiki/2011%EB%85%84" title="2011년">2011년</a>과 <a href="/wiki/2013%EB%85%84" title="2013년">2013년</a>에 <a href="/wiki/%EB%84%A5%EC%84%BC%ED%83%80%EC%9D%B4%EC%96%B4" title="넥센타이어">넥센타이어</a>와 메인 스폰서 계약을 더 연장했다. <a href="/wiki/2010%EB%85%84" title="2010년">2010년</a> <a href="/wiki/2%EC%9B%94_19%EC%9D%BC" title="2월 19일">2월 19일</a>에 2군 훈련장을 원당야구장에서 <a href="/wiki/%EC%A0%84%EB%9D%BC%EB%82%A8%EB%8F%84" title="전라남도">전라남도</a> <a href="/wiki/%EA%B0%95%EC%A7%84%EA%B5%B0" title="강진군">강진군</a> <a href="/wiki/%EB%8F%84%EC%95%94%EB%A9%B4_(%EA%B0%95%EC%A7%84%EA%B5%B0)" title="도암면 (강진군)">도암면</a> 학장리의 <a href="/wiki/%EA%B0%95%EC%A7%84_%EB%B2%A0%EC%9D%B4%EC%8A%A4%EB%B3%BC_%ED%8C%8C%ED%81%AC" title="강진 베이스볼 파크">강진 베이스볼 파크</a>로 옮겨 2013 시즌까지 사용하였고, 2014 시즌부터는 <a href="/wiki/%ED%99%94%EC%84%B1%EC%8B%9C" title="화성시">화성시</a>와 협약을 체결하여 비봉면에 2군 구장을 새로 마련하기로 결정하여 <a href="/wiki/%ED%99%94%EC%84%B1_%ED%9E%88%EC%96%B4%EB%A1%9C%EC%A6%88" title="화성 히어로즈">화성 히어로즈</a>라는 팀명으로 2군을 운영한다.<sup class="reference" id="cite_ref-3"><a href="#cite_note-3">[3]</a></sup></p>
<p><a href="/wiki/2008%EB%85%84" title="2008년">2008년</a> <a class="mw-redirect" href="/wiki/2008%EB%85%84_%ED%95%9C%EA%B5%AD_%ED%94%84%EB%A1%9C_%EC%95%BC%EA%B5%AC" title="2008년 한국 프로 야구">시즌</a> 7위에 그친 후 초대 <a href="/wiki/%EC%95%BC%EA%B5%AC_%EA%B0%90%EB%8F%85" title="야구 감독">감독</a>인 <a href="/wiki/%EC%9D%B4%EA%B4%91%ED%99%98" title="이광환">이광환</a>을 경질하고, <a href="/wiki/%EA%B9%80%EC%8B%9C%EC%A7%84" title="김시진">김시진</a> <a class="mw-redirect" href="/wiki/KBO_(%EC%95%BC%EA%B5%AC)" title="KBO (야구)">KBO</a> 감독관을 다시 불러들여 <a href="/wiki/2008%EB%85%84" title="2008년">2008년</a> <a href="/wiki/10%EC%9B%94_10%EC%9D%BC" title="10월 10일">10월 10일</a>에 취임식을 열었다. 하지만 성적 부진으로 인해 <a href="/wiki/%EA%B9%80%EC%8B%9C%EC%A7%84" title="김시진">김시진</a>을 경질하고, <a href="/wiki/2012%EB%85%84" title="2012년">2012년</a> 시즌이 끝난 후 작전·주루코치를 맡고 있었던 <a href="/wiki/%EC%97%BC%EA%B2%BD%EC%97%BD" title="염경엽">염경엽</a>이 신임 <a href="/wiki/%EC%95%BC%EA%B5%AC_%EA%B0%90%EB%8F%85" title="야구 감독">감독</a>으로 부임하여, 재창단된 후 <a href="/wiki/2013%EB%85%84" title="2013년">2013년</a> 처음으로 포스트 시즌 진출에 성공했다. 이어, <a href="/wiki/2014%EB%85%84" title="2014년">2014년</a> <a href="/wiki/10%EC%9B%94_31%EC%9D%BC" title="10월 31일">10월 31일</a> <a href="/wiki/2014%EB%85%84_%ED%95%9C%EA%B5%AD%ED%94%84%EB%A1%9C%EC%95%BC%EA%B5%AC" title="2014년 한국프로야구">한국프로야구</a> <a href="/wiki/2014%EB%85%84_%ED%95%9C%EA%B5%AD%ED%94%84%EB%A1%9C%EC%95%BC%EA%B5%AC_%ED%8F%AC%EC%8A%A4%ED%8A%B8%EC%8B%9C%EC%A6%8C" title="2014년 한국프로야구 포스트시즌">플레이오프</a>에서 <a href="/wiki/LG_%ED%8A%B8%EC%9C%88%EC%8A%A4" title="LG 트윈스">LG 트윈스</a>를 꺾고 <a href="/wiki/2008%EB%85%84" title="2008년">2008년</a> 창단 후 처음으로 <a class="mw-redirect" href="/wiki/%ED%95%9C%EA%B5%AD%EC%8B%9C%EB%A6%AC%EC%A6%88" title="한국시리즈">한국시리즈</a> 진출에 성공했다. 하지만 <a href="/wiki/%EC%82%BC%EC%84%B1_%EB%9D%BC%EC%9D%B4%EC%98%A8%EC%A6%88" title="삼성 라이온즈">삼성 라이온즈</a>에게 2승 4패로 밀려 준우승에 그쳤다. <a href="/wiki/2015%EB%85%84" title="2015년">2015년</a> 정규 시즌 4위를 기록하여 3년 연속 <a href="/wiki/KBO_%ED%8F%AC%EC%8A%A4%ED%8A%B8%EC%8B%9C%EC%A6%8C" title="KBO 포스트시즌">KBO 포스트시즌</a> 진출에 성공하였다. 하지만 <a href="/wiki/%EB%91%90%EC%82%B0_%EB%B2%A0%EC%96%B4%EC%8A%A4" title="두산 베어스">두산 베어스</a>와의 <a href="/wiki/2015%EB%85%84_KBO_%ED%8F%AC%EC%8A%A4%ED%8A%B8%EC%8B%9C%EC%A6%8C" title="2015년 KBO 포스트시즌">준플레이오프 4차전</a>에서 9-2로 크게 리드하다가 9-11로 역전패를 당하며 시즌을 마감하고 만다. <a href="/wiki/2015%EB%85%84" title="2015년">2015년</a> 시즌 이후 <a class="mw-redirect" href="/wiki/%EB%84%A5%EC%84%BC_%ED%83%80%EC%9D%B4%EC%96%B4" title="넥센 타이어">넥센 타이어</a>와의 네이밍 스폰서 계약이 만료되면서 <a class="new" href="/w/index.php?title=J%ED%8A%B8%EB%9F%AC%EC%8A%A4%ED%8A%B8&amp;action=edit&amp;redlink=1" title="J트러스트 (없는 문서)">J트러스트</a>를 비롯해 다른 기업들과 네이밍 스폰서 협상에 들어갔으나, 결국 <a href="/wiki/2015%EB%85%84" title="2015년">2015년</a> <a href="/wiki/11%EC%9B%94_5%EC%9D%BC" title="11월 5일">11월 5일</a> 연간 100억원에 넥센 타이어와의 관계를 계속해서 이어간다고 발표한다.<sup class="reference" id="cite_ref-4"><a href="#cite_note-4">[4]</a></sup></p>

<h2><span class="mw-headline" id=".ED.95.9C.EA.B5.AD_.ED.94.84.EB.A1.9C_.EC.95.BC.EA.B5.AC.EC.83.81_.EC.88.98.EC.83.81.EC.9E.90">한국 프로 야구상 수상자</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%84%A5%EC%84%BC_%ED%9E%88%EC%96%B4%EB%A1%9C%EC%A6%88&amp;action=edit&amp;section=10" title="부분 편집: 한국 프로 야구상 수상자">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<div class="rellink relarticle mainarticle"><img alt="" data-file-height="128" data-file-width="128" height="16" src="//upload.wikimedia.org/wikipedia/commons/thumb/e/ec/Crystal_Clear_app_xmag.svg/16px-Crystal_Clear_app_xmag.svg.png" srcset="//upload.wikimedia.org/wikipedia/commons/thumb/e/ec/Crystal_Clear_app_xmag.svg/24px-Crystal_Clear_app_xmag.svg.png 1.5x, //upload.wikimedia.org/wikipedia/commons/thumb/e/ec/Crystal_Clear_app_xmag.svg/32px-Crystal_Clear_app_xmag.svg.png 2x" width="16"/> 이 부분의 본문은 <a class="mw-redirect" href="/wiki/%EB%84%A5%EC%84%BC%EC%9D%98_%ED%95%9C%EA%B5%AD_%ED%94%84%EB%A1%9C_%EC%95%BC%EA%B5%AC%EC%83%81_%EC%88%98%EC%83%81%EC%9E%90" title="넥센의 한국 프로 야구상 수상자">넥센의 한국 프로 야구상 수상자</a>입니다.</div>
<p>히어로즈의 <a class="mw-redirect" href="/wiki/%ED%95%9C%EA%B5%AD_%ED%94%84%EB%A1%9C_%EC%95%BC%EA%B5%AC_%EC%B5%9C%EC%9A%B0%EC%88%98_%EC%84%A0%EC%88%98" title="한국 프로 야구 최우수 선수">한국 프로 야구 최우수 선수</a>는 <a href="/wiki/%EB%B0%95%EB%B3%91%ED%98%B8_(1986%EB%85%84)" title="박병호 (1986년)">박병호</a>와 <a href="/wiki/%EC%84%9C%EA%B1%B4%EC%B0%BD" title="서건창">서건창</a>이 있다. 1루수인 <a href="/wiki/%EB%B0%95%EB%B3%91%ED%98%B8_(1986%EB%85%84)" title="박병호 (1986년)">박병호</a>는 2011년 시즌 중 트레이드되어 넥센 히어로즈에 입단한 후, 잠재력을 폭발시키며 2012년과 2013년 연속으로 리그 최우수 선수가 되었으며, 신고선수 출신이자 <a href="/wiki/2012%EB%85%84" title="2012년">2012년</a> 시즌 신인왕 출신인 <a href="/wiki/%EC%84%9C%EA%B1%B4%EC%B0%BD" title="서건창">서건창</a>은 2014년 최우수선수에 선정되어 3년 연속으로 넥센 히어로즈 선수가 최우수선수상을 휩쓸었다.</p>

<h2><span class="mw-headline" id=".EB.A7.88.EC.8A.A4.EC.BD.94.ED.8A.B8">마스코트</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%84%A5%EC%84%BC_%ED%9E%88%EC%96%B4%EB%A1%9C%EC%A6%88&amp;action=edit&amp;section=21" title="부분 편집: 마스코트">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<div class="thumb tright">
<div class="thumbinner" style="width:252px;"><a class="image" href="/wiki/%ED%8C%8C%EC%9D%BC:Nexon_Heroes_Tukdori.jpg"><img alt="" class="thumbimage" data-file-height="1067" data-file-width="1600" height="167" src="//upload.wikimedia.org/wikipedia/commons/thumb/f/fc/Nexon_Heroes_Tukdori.jpg/250px-Nexon_Heroes_Tukdori.jpg" srcset="//upload.wikimedia.org/wikipedia/commons/thumb/f/fc/Nexon_Heroes_Tukdori.jpg/375px-Nexon_Heroes_Tukdori.jpg 1.5x, //upload.wikimedia.org/wikipedia/commons/thumb/f/fc/Nexon_Heroes_Tukdori.jpg/500px-Nexon_Heroes_Tukdori.jpg 2x" width="250"/></a>
<div class="thumbcaption">
<div class="magnify"><a class="internal" href="/wiki/%ED%8C%8C%EC%9D%BC:Nexon_Heroes_Tukdori.jpg" title="실제 크기로"></a></div>
넥센 히어로즈의 마스코트인 일명 턱돌이</div>
</div>
</div>
<p>넥센 히어로즈의 마스코트는 공식적인 이름은 <b>히어로</b>이지만, <b><a href="/wiki/%ED%84%B1%EB%8F%8C%EC%9D%B4" title="턱돌이">턱돌이</a></b>라는 애칭으로 더 많이 알려져 있다. 김진의 탈을 쓴 캐릭터에 한해 현재는 공식적으로 턱돌이로 칭하고 있다. 현재 마스코트 실물을 위해 착용하는 유니폼에도 턱돌이로 표기되어있다. 넥센 히어로즈는 몰라도 마스코트는 알 정도로, 현재 프로야구 구단 마스코트 중 가장 인지도가 높은 마스코트다. <a href="/wiki/2015%EB%85%84" title="2015년">2015년</a> <a href="/wiki/4%EC%9B%94_9%EC%9D%BC" title="4월 9일">4월 9일</a> 신규 마스코트의 애칭을 <b>동글이</b>를 발표했다. 그리고 <b>동글이</b>는 귀여움과 강인한 매력을 동시에 지닌 여성 마스코트이다. <a href="/wiki/4%EC%9B%94_10%EC%9D%BC" title="4월 10일">4월 10일</a> <a href="/wiki/%EB%AA%A9%EB%8F%99%EC%95%BC%EA%B5%AC%EC%9E%A5" title="목동야구장">목동야구장</a> <a href="/wiki/Kt_%EC%9C%84%EC%A6%88" title="Kt 위즈">kt</a>전에서 <b><a href="/wiki/%ED%84%B1%EB%8F%8C%EC%9D%B4" title="턱돌이">턱돌이</a></b>와 함께 시구 및 시타를 하였다.<sup class="reference" id="cite_ref-6"><a href="#cite_note-6">[6]</a></sup></p>
