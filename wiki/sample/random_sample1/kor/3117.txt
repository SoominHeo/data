<a href="/wiki/%EA%B7%B8%EB%86%88" title="그놈">그놈</a> 2.14를 실행 중인 넥센타 OS 알파 5</div>

<p><b>넥센타 OS</b>(Nexenta OS, 넥센타 코어 플랫폼/Nexenta Core Platform)은 <a href="/wiki/IA-32" title="IA-32">IA-32</a>, <a href="/wiki/X86-64" title="X86-64">x86-64</a> 기반 시스템에서 동작하는 <a href="/wiki/%EC%98%A4%ED%94%88%EC%86%94%EB%9D%BC%EB%A6%AC%EC%8A%A4" title="오픈솔라리스">오픈솔라리스</a> 및 <a href="/wiki/%EC%9A%B0%EB%B6%84%ED%88%AC_(%EC%9A%B4%EC%98%81_%EC%B2%B4%EC%A0%9C)" title="우분투 (운영 체제)">우분투</a>에 기반한 과거의 컴퓨터 <a href="/wiki/%EC%9A%B4%EC%98%81_%EC%B2%B4%EC%A0%9C" title="운영 체제">운영 체제</a>이다. <a href="/wiki/%EC%8D%AC_%EB%A7%88%EC%9D%B4%ED%81%AC%EB%A1%9C%EC%8B%9C%EC%8A%A4%ED%85%9C%EC%A6%88" title="썬 마이크로시스템즈">썬 마이크로시스템즈</a>가 <a href="/wiki/%EC%98%A4%ED%94%88%EC%86%94%EB%9D%BC%EB%A6%AC%EC%8A%A4" title="오픈솔라리스">오픈솔라리스</a> 프로젝트를 2005년 6월 시작한 이후인 같은 해 가을에 모습을 드러냈다.<sup class="reference" id="cite_ref-1"><a href="#cite_note-1">[1]</a></sup></p>
<p>2011년 말 넥센타 OS 브랜드는 종료되었으며, <a class="new" href="/w/index.php?title=%EC%9D%BC%EB%A3%A8%EB%AA%A8%EC%8A%A4&amp;action=edit&amp;redlink=1" title="일루모스 (없는 문서)">일루모스</a>, <a class="new" href="/w/index.php?title=%EC%98%A4%ED%94%88%EC%9D%B8%EB%94%94%EC%95%A0%EB%82%98&amp;action=edit&amp;redlink=1" title="오픈인디애나 (없는 문서)">오픈인디애나</a> 개발 커뮤니티에서 파생한 <a class="external text" href="https://www.illumos.org/projects/illumian" rel="nofollow">일루미안</a>으로 대체되었다.</p>
