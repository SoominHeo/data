<p><b>노동절</b>(勞動節) 또는 <b>메이 데이</b>(<span style="font-size: smaller;"><a href="/wiki/%EC%98%81%EC%96%B4" title="영어">영어</a>:</span> <span lang="en" xml:lang="en">May Day</span>)는 노동자의 권익과 복지를 향상하고 안정된 삶을 도모하기 위하여 제정한 날이다. 전 세계적으로 노동자의 연대와 단결을 과시한다. <a href="/wiki/1886%EB%85%84" title="1886년">1886년</a> <a href="/wiki/5%EC%9B%94_1%EC%9D%BC" title="5월 1일">5월 1일</a> <a href="/wiki/%EB%AF%B8%EA%B5%AD" title="미국">미국</a>의 <a href="/wiki/%EC%B4%9D%ED%8C%8C%EC%97%85" title="총파업">총파업</a>을 노동절의 시초로 본다. 1889년에 <a href="/wiki/%EC%A0%9C2%EC%9D%B8%ED%84%B0%EB%82%B4%EC%85%94%EB%84%90" title="제2인터내셔널">제2인터내셔널</a>은 <a href="/wiki/5%EC%9B%94_1%EC%9D%BC" title="5월 1일">5월 1일</a>을 노동 운동을 기념하는 날로 정하였고, 이후 전 세계로 확산되었다.</p>

<h3><span class="mw-headline" id="8.EC.8B.9C.EA.B0.84_.EB.85.B8.EB.8F.99_.EC.9F.81.EC.B7.A8">8시간 노동 쟁취</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%85%B8%EB%8F%99%EC%A0%88&amp;action=edit&amp;section=2" title="부분 편집: 8시간 노동 쟁취">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p><a href="/wiki/1886%EB%85%84" title="1886년">1886년</a> <a href="/wiki/5%EC%9B%94_1%EC%9D%BC" title="5월 1일">5월 1일</a> <a href="/wiki/%EB%AF%B8%EA%B5%AD" title="미국">미국</a> <a href="/wiki/%EC%8B%9C%EC%B9%B4%EA%B3%A0" title="시카고">시카고</a>에서는 8만 명의 노동자들과 그들의 가족들이 미시건 거리에서 파업 집회를 열었다. 이들이 집회를 연 이유는 장시간 노동에 대항하여 8시간 노동을 보장받기 위해서였지만, 경찰과 군대의 발포로 유혈 사태가 발생하였고, 결국은 자본가들은 단결투쟁하는 노동자들의 요구를 들어주었다. 당시 <a class="new" href="/w/index.php?title=%EC%8B%9C%EC%B9%B4%EA%B3%A0_%EB%8D%B0%EC%9D%BC%EB%A6%AC_%EB%89%B4%EC%8A%A4&amp;action=edit&amp;redlink=1" title="시카고 데일리 뉴스 (없는 문서)">시카고 데일리 뉴스</a>에서는 공산폭동이 일어날 것이라며, 이들을 <a class="mw-redirect" href="/wiki/%EA%B3%B5%EC%82%B0%EC%A3%BC%EC%9D%98%EC%9E%90" title="공산주의자">공산주의자</a> 취급했으나 당시 미국 노동자들의 노동 운동은 의식화된 사회주의자들의 쟁투이기 이전에, 장시간 노동을 극복함으로써 사람답게 살고자 한 노동자들의 단결투쟁이었다. 미국 노동 운동은 <a class="mw-redirect" href="/wiki/%EC%9E%90%EB%B3%B8%EA%B0%80" title="자본가">자본가</a>, 정부, 자본가와 결탁한 <a class="mw-disambig" href="/wiki/%EB%B3%B4%EC%88%98" title="보수">보수</a>언론들의 탄압과 색깔론을 주장하는 왜곡 보도가 증가되어 큰 힘을 발휘하지 못하였으나,8시간 노동이라는 노동<a href="/wiki/%EC%9D%B8%EA%B6%8C" title="인권">인권</a>을 단결투쟁으로써 쟁취했다는 의미가 있는,노동운동 역사에서 중요한 사건이다.</p>
<h3><span class="mw-headline" id=".EB.85.B8.EB.8F.99.EC.A0.88">노동절</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%85%B8%EB%8F%99%EC%A0%88&amp;action=edit&amp;section=3" title="부분 편집: 노동절">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>20세기 초부터 미국 정부가 매년 <a href="/wiki/5%EC%9B%94_1%EC%9D%BC" title="5월 1일">5월 1일</a>이 사회주의의 냄새를 풍긴다는 이유로, 노동절을 <a href="/wiki/9%EC%9B%94" title="9월">9월</a> 첫 번째 <a href="/wiki/%EC%9B%94%EC%9A%94%EC%9D%BC" title="월요일">월요일</a>로 바꿔 놓았다.<sup class="reference" id="cite_ref-1"><a href="#cite_note-1">[1]</a></sup></p>
<h2><span class="mw-headline" id=".ED.95.9C.EA.B5.AD.EC.9D.98_.EB.85.B8.EB.8F.99.EC.A0.88">한국의 노동절</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%85%B8%EB%8F%99%EC%A0%88&amp;action=edit&amp;section=4" title="부분 편집: 한국의 노동절">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p><a href="/wiki/%ED%95%9C%EA%B5%AD" title="한국">한국</a>에서는 <a class="mw-redirect" href="/wiki/%EC%9D%BC%EC%A0%9C%EA%B0%95%EC%A0%90%EA%B8%B0" title="일제강점기">일제강점기 조선</a> 시기인 1923년부터 노동절 행사가 <a class="new" href="/w/index.php?title=%EC%A1%B0%EC%84%A0%EB%85%B8%EB%8F%99%EC%B4%9D%EB%8F%99%EB%A7%B9&amp;action=edit&amp;redlink=1" title="조선노동총동맹 (없는 문서)">조선노동총동맹</a>의 주도로 시작되었다.<sup class="reference" id="cite_ref-NodongNet_2-0"><a href="#cite_note-NodongNet-2">[2]</a></sup> 독립 직후에는 <a class="mw-redirect" href="/wiki/2%EC%B0%A8_%EC%84%B8%EA%B3%84_%EB%8C%80%EC%A0%84" title="2차 세계 대전">2차 세계 대전</a>이 끝난 시점인 1945년 결성된 <a href="/wiki/%EC%A1%B0%EC%84%A0%EB%85%B8%EB%8F%99%EC%A1%B0%ED%95%A9%EC%A0%84%EA%B5%AD%ED%8F%89%EC%9D%98%ED%9A%8C" title="조선노동조합전국평의회">전평</a>과 1946년 결성된 <a href="/wiki/%EB%8C%80%ED%95%9C%EB%8F%85%EB%A6%BD%EC%B4%89%EC%84%B1%EB%85%B8%EB%8F%99%EC%B4%9D%EC%97%B0%EB%A7%B9" title="대한독립촉성노동총연맹">대한노총</a>이 1946년에 각각 노동절 행사를 치르게 되었다.<sup class="reference" id="cite_ref-NodongNet_2-1"><a href="#cite_note-NodongNet-2">[2]</a></sup></p>

<p>대한민국에서는 <b>근로자의 날</b>이라고 부른다. 역사적으로는 <a href="/wiki/1958%EB%85%84" title="1958년">1958년</a> 이후, <a href="/wiki/%EB%8C%80%ED%95%9C%EB%8F%85%EB%A6%BD%EC%B4%89%EC%84%B1%EB%85%B8%EB%8F%99%EC%B4%9D%EC%97%B0%EB%A7%B9" title="대한독립촉성노동총연맹">대한노총</a> 창립일인 <a href="/wiki/3%EC%9B%94_10%EC%9D%BC" title="3월 10일">3월 10일</a>을 ‘노동절’로 정했으며, <a href="/wiki/1963%EB%85%84" title="1963년">1963년</a> <a href="/wiki/4%EC%9B%94_17%EC%9D%BC" title="4월 17일">4월 17일</a>에는 ‘근로자의 날’로 이름을 바꾸었다. 이것이 <a href="/wiki/1973%EB%85%84" title="1973년">1973년</a> <a href="/wiki/3%EC%9B%94_30%EC%9D%BC" title="3월 30일">3월 30일</a>에 제정·공포되었으며, <a href="/wiki/1994%EB%85%84" title="1994년">1994년</a> 다시 <a href="/wiki/5%EC%9B%94_1%EC%9D%BC" title="5월 1일">5월 1일</a>로 바뀌었다. 노동절은 근로자의 날로서 노동자의 휴일로 지정되어 있으며, 유급휴가로 인정된다. 하지만, 이주공동행동은 대한민국에서 일하는 이주노동자들이 노동절에도 쉬지 못하고 일을 한다고 밝혔다.<sup class="reference" id="cite_ref-3"><a href="#cite_note-3">[3]</a></sup></p>
