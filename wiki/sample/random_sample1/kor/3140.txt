<h1 class="firstHeading" id="firstHeading" lang="ko">노라 아스토르가</h1>
<div class="mw-body-content" id="bodyContent">
<div id="siteSub">위키백과, 우리 모두의 백과사전.</div>
<div id="contentSub"></div>
<div class="mw-jump" id="jump-to-nav">
					이동:					<a href="#mw-head">둘러보기</a>, 					<a href="#p-search">검색</a>
</div>
<div class="mw-content-ltr" dir="ltr" id="mw-content-text" lang="ko"><script>function mfTempOpenSection(id){var block=document.getElementById("mf-section-"+id);block.className+=" open-block";block.previousSibling.className+=" open-block";}</script><p><b>노라 아스토르가</b>(Nora Astorga Gadea de Jenkins, <a href="/wiki/1948%EB%85%84" title="1948년">1948년</a> ~ <a href="/wiki/1988%EB%85%84" title="1988년">1988년</a> <a href="/wiki/1%EC%9B%94_14%EC%9D%BC" title="1월 14일">1월 14일</a>)은 <a href="/wiki/%EB%8B%88%EC%B9%B4%EB%9D%BC%EA%B3%BC" title="니카라과">니카라과</a>의 혁명 게릴라, 변호사, 판사, 정치인, 외교관이다.</p>
<p><a href="/wiki/1948%EB%85%84" title="1948년">1948년</a> <a href="/wiki/%EB%8B%88%EC%B9%B4%EB%9D%BC%EA%B3%BC" title="니카라과">니카라과</a>의 수도 <a href="/wiki/%EB%A7%88%EB%82%98%EA%B3%BC" title="마나과">마나과</a>에서 <a class="mw-redirect" href="/wiki/%EB%A1%9C%EB%A7%88_%EA%B0%80%ED%86%A8%EB%A6%AD" title="로마 가톨릭">로마 가톨릭</a> 신봉 중산층 가정에서 태어났다.</p>
<p>대학생 시절부터 혁명 활동을 시작하여 <a href="/wiki/%EC%82%B0%EB%94%94%EB%8B%88%EC%8A%A4%ED%83%80_%EB%AF%BC%EC%A1%B1%ED%95%B4%EB%B0%A9%EC%A0%84%EC%84%A0" title="산디니스타 민족해방전선">산디니스타 민족해방전선</a>에서 핵심적으로 활약하였다. <a href="/wiki/1979%EB%85%84" title="1979년">1979년</a> 7월, <a href="/wiki/%EC%82%B0%EB%94%94%EB%8B%88%EC%8A%A4%ED%83%80_%EB%AF%BC%EC%A1%B1%ED%95%B4%EB%B0%A9%EC%A0%84%EC%84%A0" title="산디니스타 민족해방전선">산디니스타 민족해방전선</a>이 니카라과 정권을 장악한 이후 법무부 각료로 임명되었다. <a href="/wiki/1986%EB%85%84" title="1986년">1986년</a>부터 <a href="/wiki/1988%EB%85%84" title="1988년">1988년</a>에 사망할 때까지 산디니스타 민족해방전선의 <a href="/wiki/%EB%8B%A4%EB%8B%88%EC%97%98_%EC%98%A4%EB%A5%B4%ED%85%8C%EA%B0%80" title="다니엘 오르테가">다니엘 오르테가</a> 정권의 니카라과를 대표하여 <a href="/wiki/%EC%9C%A0%EC%97%94" title="유엔">유엔</a> 대사로 활동했다.</p>
<p><a href="/wiki/1988%EB%85%84" title="1988년">1988년</a> <a href="/wiki/1%EC%9B%94_14%EC%9D%BC" title="1월 14일">1월 14일</a>, 노라 아스토르가는 <a href="/wiki/%EC%9E%90%EA%B6%81%EA%B2%BD%EB%B6%80%EC%95%94" title="자궁경부암">자궁경부암</a>으로 사망했다.</p>
