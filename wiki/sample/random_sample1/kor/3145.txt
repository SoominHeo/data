<p><b>노량진동</b>(鷺梁津洞)은 <a href="/wiki/%EC%84%9C%EC%9A%B8%ED%8A%B9%EB%B3%84%EC%8B%9C" title="서울특별시">서울특별시</a> <a href="/wiki/%EB%8F%99%EC%9E%91%EA%B5%AC" title="동작구">동작구</a>에 있으며 <a href="/wiki/%EB%8F%99_(%ED%96%89%EC%A0%95_%EA%B5%AC%EC%97%AD)" title="동 (행정 구역)">행정동</a>으로 노량진1동과 노량진2동으로 나뉜다. 노량진1동은 법정동 노량진동과 함께 본동을 포함하고 있다.</p>

<h2><span class="mw-headline" id=".EA.B0.9C.EC.9A.94">개요</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%85%B8%EB%9F%89%EC%A7%84%EB%8F%99&amp;action=edit&amp;section=1" title="부분 편집: 개요">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p><a href="/wiki/%EA%B2%BD%EB%B6%80%EC%84%A0" title="경부선">경부선</a> <a href="/wiki/%EB%85%B8%EB%9F%89%EC%A7%84%EC%97%AD" title="노량진역">노량진역</a>을 중심으로 일대에 상가가 발달하였고 <a href="/wiki/1974%EB%85%84" title="1974년">1974년</a>에 <a href="/wiki/%EC%A0%84%EA%B8%B0%EC%B2%A0%EB%8F%84" title="전기철도">전철</a>이 개통되었으며 강북지역 및 강남과 영등포, 시흥, 안양 등지를 잇는 교통의 요충지로서 <a href="/wiki/%ED%95%9C%EA%B0%95%EB%8C%80%EA%B5%90" title="한강대교">한강대교</a>와 <a href="/wiki/%EC%83%81%EB%8F%84%ED%84%B0%EB%84%90" title="상도터널">상도터널</a>이 있다.</p>
<p>본래는 경기도 과천군 하북면 소속이었으나 1914년 부군면 통폐합으로 시흥군 북면으로 개편되었다.</p>
<p><a href="/wiki/1936%EB%85%84" title="1936년">1936년</a>에 <a href="/wiki/%EA%B2%BD%EA%B8%B0%EB%8F%84" title="경기도">경기도</a> <a href="/wiki/%EC%8B%9C%ED%9D%A5%EA%B5%B0" title="시흥군">시흥군</a> 북면(北面)에서 서울로 편입되었으며 <a href="/wiki/1899%EB%85%84" title="1899년">1899년</a>에 한국 최초의 <a href="/wiki/%EC%B2%A0%EB%8F%84" title="철도">철도</a>인 <a href="/wiki/%EA%B2%BD%EC%9D%B8%EC%84%A0" title="경인선">경인선</a>이 이 곳에서 <a href="/wiki/%EC%9D%B8%EC%B2%9C%EA%B4%91%EC%97%AD%EC%8B%9C" title="인천광역시">인천</a>의 <a class="mw-disambig" href="/wiki/%EC%A0%9C%EB%AC%BC%ED%8F%AC" title="제물포">제물포</a>까지 개통되어 철도 시발지로서 철도 역사의 중요한 지명이 되었으며, <a href="/wiki/1900%EB%85%84" title="1900년">1900년</a>에 <a href="/wiki/%ED%95%9C%EA%B0%95%EC%B2%A0%EA%B5%90" title="한강철교">한강철교</a>, <a href="/wiki/1917%EB%85%84" title="1917년">1917년</a>에 <a href="/wiki/%ED%95%9C%EA%B0%95%EB%8C%80%EA%B5%90" title="한강대교">인도교</a>가 건설되면서 서울의 부도심으로 발달하게 되었다.</p>
<p>노량진역 북쪽에 수산물 도매센터인 <a href="/wiki/%EB%85%B8%EB%9F%89%EC%A7%84%EC%88%98%EC%82%B0%EC%8B%9C%EC%9E%A5" title="노량진수산시장">노량진수산시장</a>이 있고, <a href="/wiki/%ED%95%9C%EA%B0%95%EB%8C%80%EA%B5%90" title="한강대교">한강대교</a>의 남단에 있는 노량진 상수도 수원지는 서울에서 가장 오래된 상수도 수원지이다. 그 밖에 <a href="/wiki/%EB%85%B8%EB%9F%89%EC%A7%84%EC%97%AD" title="노량진역">노량진역</a> 동쪽 언덕 위에는 <a href="/wiki/%EC%82%AC%EC%9C%A1%EC%8B%A0%EA%B3%B5%EC%9B%90" title="사육신공원">사육신공원</a>(지방유형문화재)이 있다.</p>
<p>동작구청은 노량진2동에 자리하고 있으며, 노량진1동은 고시학원, 입시학원 등 각종 대규모 학원이 많이 분포하고 학원생들을 유치하기 위한 고시원과 고시생들을 대상으로 하는 저렴한 가격의 식당이 골목마다 들어서 있다.</p>
<h2><span class="mw-headline" id=".EA.B4.80.EA.B4.91">관광</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%85%B8%EB%9F%89%EC%A7%84%EB%8F%99&amp;action=edit&amp;section=2" title="부분 편집: 관광">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<ul>
<li><a href="/wiki/%EC%82%AC%EC%9C%A1%EC%8B%A0%EA%B3%B5%EC%9B%90" title="사육신공원">사육신공원</a></li>
</ul>
<p>노량진동에 있는 묘와 그 외 사육신비 등이 안장된 공원이다. 단종 복위에 힘 쓰다 목숨을 잃은 <a href="/wiki/%EC%84%B1%EC%82%BC%EB%AC%B8" title="성삼문">성삼문</a>, <a href="/wiki/%EB%B0%95%ED%8C%BD%EB%85%84" title="박팽년">박팽년</a>, <a href="/wiki/%ED%95%98%EC%9C%84%EC%A7%80" title="하위지">하위지</a>, <a href="/wiki/%EC%9D%B4%EA%B0%9C" title="이개">이개</a>, <a href="/wiki/%EC%9C%A0%EC%9D%91%EB%B6%80" title="유응부">유응부</a>, <a href="/wiki/%EC%9C%A0%EC%84%B1%EC%9B%90" title="유성원">유성원</a>에 추가로 <a href="/wiki/%EA%B9%80%EB%AC%B8%EA%B8%B0" title="김문기">김문기</a>(金文起)를 현창(顯彰)하여 모신 곳이다.</p>
