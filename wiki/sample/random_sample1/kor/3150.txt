<h2><span class="mw-headline" id=".ED.95.84.EB.AA.85">필명</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%85%B8%EB%A5%B4%EB%B2%A0%EB%A5%B4%ED%86%A0_%EB%B3%B4%EB%B9%84%EC%98%A4&amp;action=edit&amp;section=1" title="부분 편집: 필명">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p><a href="/wiki/%ED%94%BC%EC%97%90%EB%AA%AC%ED%85%8C_%EC%A3%BC" title="피에몬테 주">피에몬테 주</a>에 있는 중산층 가정에서 태어났다. 고등학교에 다닐 때 <a href="/wiki/%EC%B2%B4%EC%82%AC%EB%A0%88_%ED%8C%8C%EB%B2%A0%EC%84%B8" title="체사레 파베세">체사레 파베세</a>와 <a href="/wiki/%EB%A0%88%EC%98%A4%EB%84%A4_%EA%B8%B4%EC%B8%A0%EB%B6%80%EB%A5%B4%EA%B7%B8" title="레오네 긴츠부르그">레오네 긴츠부르그</a>와 교류하였다. <a href="/wiki/%ED%86%A0%EB%A6%AC%EB%85%B8_%EB%8C%80%ED%95%99%EA%B5%90" title="토리노 대학교">토리노 대학교</a>에 진학하여 <a href="/wiki/%EB%B2%95%EC%B2%A0%ED%95%99" title="법철학">법철학</a>을 전공하였다. <a href="/wiki/%ED%86%A0%EB%A6%AC%EB%85%B8_%EB%8C%80%ED%95%99%EA%B5%90" title="토리노 대학교">토리노 대학교</a>에서 노르베르또 봅비오는 당시에 유행하던, 실제이고 세부까지 포함한 현실을 좇지 않는, 언행 따위가 현실과 동떨어져 막연한 표상에 경도된 정체를 불신하게 되었고 자유주의에 토대한 제도도 비판하였다. <a href="/wiki/%ED%86%A0%EB%A6%AC%EB%85%B8_%EB%8C%80%ED%95%99%EA%B5%90" title="토리노 대학교">토리노 대학교</a> 강사가 되고서 <a href="/wiki/%EC%9D%B4%ED%83%88%EB%A6%AC%EC%95%84" title="이탈리아">이탈리아</a> 중부 <a href="/wiki/%EB%A7%88%EB%A5%B4%EC%BC%80_%EC%A3%BC" title="마르케 주">마르케 주</a>에 소재한 카메리노대학과 <a href="/wiki/%ED%86%A0%EC%8A%A4%EC%B9%B4%EB%82%98_%EC%A3%BC" title="토스카나 주">토스카나 주</a>에 있는 <a href="/wiki/%EC%8B%9C%EC%97%90%EB%82%98" title="시에나">시에나</a>대학, <a href="/wiki/%EC%9D%B4%ED%83%88%EB%A6%AC%EC%95%84" title="이탈리아">이탈리아</a> 동북부 <a href="/wiki/%EB%B2%A0%EB%84%A4%ED%86%A0_%EC%A3%BC" title="베네토 주">베네토 주</a> <a href="/wiki/%ED%8C%8C%EB%8F%84%EB%B0%94" title="파도바">파도바</a>에 소재한 <a href="/wiki/%ED%8C%8C%EB%8F%84%EB%B0%94_%EB%8C%80%ED%95%99%EA%B5%90" title="파도바 대학교">파도바 대학교</a>에서 교편을 잡았다.</p>
<p><a href="/wiki/%EC%A0%9C2%EC%B0%A8_%EC%84%B8%EA%B3%84_%EB%8C%80%EC%A0%84" title="제2차 세계 대전">제2차 세계 대전</a> 중 비합법 조직인 「행동당」에 참가하여 1943년에서 1944년까지 징역을 살았다. 1946년, <a href="/wiki/%EC%9D%B4%ED%83%88%EB%A6%AC%EC%95%84" title="이탈리아">이탈리아</a> 의회 선거에 입후보하지만 낙선하고 당 활동에서도 멀어지면서, 사회현상과 별도로 학문이 당면한 처지를 순수하게 견지하는 태도로 실행을 중시하지 않는 학문 지상주의다운 학풍의 세계로 복귀하여 1948년에 <a href="/wiki/%ED%86%A0%EB%A6%AC%EB%85%B8_%EB%8C%80%ED%95%99%EA%B5%90" title="토리노 대학교">토리노 대학교</a>에서 교편을 잡기 시작하였다. 1984년에 종신 <a class="new" href="/w/index.php?title=%EC%83%81%EC%9B%90_%EC%9D%98%EC%9B%90&amp;action=edit&amp;redlink=1" title="상원 의원 (없는 문서)">상원 의원</a>에 지명되었다.</p>
<p>법치와 권력분립과 권력 제한을 두둔하고 편들어 지키는 사람으로서 노르베르또 봅비오는 <a href="/wiki/%EC%82%AC%ED%9A%8C%EC%A3%BC%EC%9D%98" title="사회주의">사회주의</a>를 믿고 주장하였지만, <a href="/wiki/%EB%A7%88%EB%A5%B4%ED%81%AC%EC%8A%A4%EC%A3%BC%EC%9D%98" title="마르크스주의">마르크스주의</a>를 신봉하거나 추종하는 사람들의 행동 양식에서 허다반한, <a href="/wiki/%EB%AF%BC%EC%A3%BC%EC%A3%BC%EC%9D%98" title="민주주의">민주주의</a>와 상충하거나 <a href="/wiki/%EA%B6%8C%EC%9C%84%EC%A3%BC%EC%9D%98" title="권위주의">권위주의</a> 성격을 띤 요소에는 반대하였다. 노르베르또 봅비오는 1970년대에 <a href="/wiki/%EC%9D%B4%ED%83%88%EB%A6%AC%EC%95%84" title="이탈리아">이탈리아</a><a href="/wiki/%EA%B3%B5%EC%82%B0%EB%8B%B9" title="공산당">공산당</a>과 <a href="/wiki/%EA%B8%B0%EB%8F%85%EA%B5%90" title="기독교">기독교</a>민주당이 체결한 「역사성을 띤 타협」을 열렬히 지지하였다.</p>
