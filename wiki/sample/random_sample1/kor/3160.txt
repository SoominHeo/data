<p><b>노릴스크</b>(<span style="font-size: smaller;"><a href="/wiki/%EB%9F%AC%EC%8B%9C%EC%95%84%EC%96%B4" title="러시아어">러시아어</a>:</span> <span lang="ru" xml:lang="ru">Норильск</span>)는 <a href="/wiki/%ED%81%AC%EB%9D%BC%EC%8A%A4%EB%85%B8%EC%95%BC%EB%A5%B4%EC%8A%A4%ED%81%AC_%EC%A7%80%EB%B0%A9" title="크라스노야르스크 지방">크라스노야르스크 지방</a>의 도시이자, <a href="/wiki/%ED%83%80%EC%9D%B4%EB%AF%B8%EB%A5%B4_%EC%9E%90%EC%B9%98%EA%B5%AC" title="타이미르 자치구">타이미르 자치구</a>의 도시이다. 크라스노야르스크 지방의 직할시이다. <a class="mw-redirect" href="/wiki/%EC%A4%91%EC%95%99%EC%8B%9C%EB%B2%A0%EB%A6%AC%EC%95%84_%EA%B3%A0%EC%9B%90" title="중앙시베리아 고원">중앙시베리아 고원</a>의 북서부, <a href="/wiki/%ED%83%80%EC%9D%B4%EB%AF%B8%EB%A5%B4_%EB%B0%98%EB%8F%84" title="타이미르 반도">타이미르 반도</a>의 남쪽에 위치하고 <a href="/wiki/%EC%98%88%EB%8B%88%EC%84%B8%EC%9D%B4_%EA%B0%95" title="예니세이 강">예니세이 강</a>에서 동쪽으로 90km지점에 위치하고 있다. 인구는 12만9,800명(2004년)이다. 인구가 10만명을 넘는 도시로, 유일하게 세계에서 가장 최북단에 위치한다(북위 69도 22부, 동경 88도 06부).</p>
<p><a href="/wiki/%EB%8B%88%EC%BC%88" title="니켈">니켈</a>광산이 있는 것 외에도, <a href="/wiki/%EA%B5%AC%EB%A6%AC" title="구리">동</a>이나 <a href="/wiki/%EC%BD%94%EB%B0%9C%ED%8A%B8" title="코발트">코발트</a> 등 여러 가지의 금속을 생산된다. 노릴스크의 콤비나트(combinat)는 세계 <a href="/wiki/%ED%8C%94%EB%9D%BC%EB%93%90" title="팔라듐">팔라듐</a>의 35%, <a href="/wiki/%EB%B0%B1%EA%B8%88" title="백금">백금</a>의 25%, 니켈의 20%, <a href="/wiki/%EB%A1%9C%EB%93%90" title="로듐">로듐</a>의 20%, 코발트의 10%를 생산하고 있다. 또, 러시아 연방에서 생산되는 니켈의 96%, 코발트의 95%, 동의 55%가 노릴스크의 콤비나트(combinat)에서 생산되고 있다.</p>
<p><a href="/wiki/%EC%9E%90%EB%8F%99%EC%B0%A8" title="자동차">자동차</a>도로나 <a href="/wiki/%EC%B2%A0%EB%8F%84" title="철도">철도</a>로 예니세이 강 하류의 <a href="/wiki/%EB%91%90%EB%94%98%EC%B9%B4" title="두딘카">두딘카</a>하고도 연결되어 있다.</p>
<p>고등교육 기관으로는 노릴스크 공업 대학, 노릴스크 경제 대학이 있다. 극장이나 박물관이 있다.</p>
<p>노릴스크에는 정교회성당과 세계에서 가장 북쪽에 있는 <a href="/wiki/%EB%AA%A8%EC%8A%A4%ED%81%AC" title="모스크">모스크</a>인 <a class="new" href="/w/index.php?title=%EB%88%84%EB%A5%B4%EB%93%9C%EC%B9%B4%EB%A7%90&amp;action=edit&amp;redlink=1" title="누르드카말 (없는 문서)">누르드카말</a>(<span style="font-size: smaller;"><a href="/wiki/%EB%9F%AC%EC%8B%9C%EC%95%84%EC%96%B4" title="러시아어">러시아어</a>:</span> <span lang="ru" xml:lang="ru">Нурд-Камаль</span>)이 위치해 있다.<sup class="reference" id="cite_ref-1"><a href="#cite_note-1">[1]</a></sup></p>

<h2><span class="mw-headline" id=".EC.97.AD.EC.82.AC">역사</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%85%B8%EB%A6%B4%EC%8A%A4%ED%81%AC&amp;action=edit&amp;section=2" title="부분 편집: 역사">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p><a href="/wiki/1935%EB%85%84" title="1935년">1935년</a>에는 강제 노동에 의해서, <a href="/wiki/%EC%BD%A4%EB%B9%84%EB%82%98%ED%8A%B8" title="콤비나트">콤비나트</a>의 건설이 개시되었는데 <a href="/wiki/1939%EB%85%84" title="1939년">1939년</a> 3월에 공장 가동을 시작했다. <a href="/wiki/1953%EB%85%84" title="1953년">1953년</a>에 도시로 등록될 때까지 초기의 주민은 오로지 강제 노동을 위한 죄수이며, 죄수의 수는 <a href="/wiki/1950%EB%85%84%EB%8C%80" title="1950년대">1950년대</a>초에는 약 7만명에 이르렀다.</p>
