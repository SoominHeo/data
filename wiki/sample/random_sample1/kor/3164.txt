<p><b>노면전차</b>(路面電車), <b>시가전차</b>(市街電車), 또는 <b>트램</b>(tram, tramcar, trolley 또는 streetcar)은 <a href="/wiki/%EA%B5%90%ED%86%B5%EC%88%98%EB%8B%A8" title="교통수단">교통수단</a>의 하나로, 주로 도로상에 부설된 <a class="mw-disambig" href="/wiki/%EB%A0%88%EC%9D%BC" title="레일">레일</a>을 따라 움직이는 <a class="mw-redirect" href="/wiki/%EC%A0%84%EB%8F%99%EC%B0%A8" title="전동차">전동차</a>를 일컫는다.</p>

<h2><span class="mw-headline" id=".EA.B0.9C.EC.9A.94">개요</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%85%B8%EB%A9%B4%EC%A0%84%EC%B0%A8&amp;action=edit&amp;section=1" title="부분 편집: 개요">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<h3><span class="mw-headline" id=".EC.9C.A0.EB.9F.BD.EC.9D.98_.EC.A0.84.EC.B0.A8">유럽의 전차</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%85%B8%EB%A9%B4%EC%A0%84%EC%B0%A8&amp;action=edit&amp;section=2" title="부분 편집: 유럽의 전차">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p><a href="/wiki/%EB%8F%85%EC%9D%BC" title="독일">독일</a>, <a href="/wiki/%EB%9F%AC%EC%8B%9C%EC%95%84" title="러시아">러시아</a>, <a href="/wiki/%EC%9A%B0%ED%81%AC%EB%9D%BC%EC%9D%B4%EB%82%98" title="우크라이나">우크라이나</a>를 비롯한 세계 약 50개국의 약 400도시에 존재한다. 도시 내 및 그 근교에서 여객의 이동 수단으로서 이용되고 있는 것이 많다. 현재의 <a href="/wiki/%EC%9D%BC%EB%B3%B8" title="일본">일본</a>에서는 약 20개소에서 노면 전차가 존재하고 있다. 노선은 일반적으로 노면을 주행하지만, 교외에서는 별개의 전용 궤도를 마련하는 경우가 많다. 한편, 시가지에 있어도 지하화 혹은 고가화로 도로와의 분리를 도모한 구간도 많지만 20세기말 이후는 반대로 노면으로부터 승강할 수 있는 것이 평가되어 유럽을 중심으로 보급되어 있다.</p>
<h3><span class="mw-headline" id=".EB.8C.80.ED.95.9C.EB.AF.BC.EA.B5.AD.EA.B3.BC_.EC.A1.B0.EC.84.A0.EB.AF.BC.EC.A3.BC.EC.A3.BC.EC.9D.98.EC.9D.B8.EB.AF.BC.EA.B3.B5.ED.99.94.EA.B5.AD">대한민국과 조선민주주의인민공화국</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%85%B8%EB%A9%B4%EC%A0%84%EC%B0%A8&amp;action=edit&amp;section=3" title="부분 편집: 대한민국과 조선민주주의인민공화국">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p><a href="/wiki/%ED%95%9C%EA%B5%AD" title="한국">한국</a>에서는 <a class="mw-redirect" href="/wiki/%EA%B2%BD%EC%84%B1%EC%A0%84%EC%B0%A8" title="경성전차">경성전차</a>가 <a href="/wiki/1898%EB%85%84" title="1898년">1898년</a> <a href="/wiki/12%EC%9B%94" title="12월">12월</a>에 <a class="mw-redirect" href="/wiki/%EC%84%9C%EC%9A%B8" title="서울">서울</a> 서대문-청량리 구간에 처음 개통된 뒤 사대문 안을 중심으로 연결되었고, <a class="mw-redirect" href="/wiki/%EB%B6%80%EC%82%B0" title="부산">부산</a>에도 건설되었다. 경성전차는 <a href="/wiki/1910%EB%85%84" title="1910년">1910년</a> 이후 <a class="mw-disambig" href="/wiki/%EC%9A%A9%EC%82%B0" title="용산">용산</a>, <a href="/wiki/%EC%9B%90%ED%9A%A8%EB%A1%9C" title="원효로">원효로</a>, <a class="mw-disambig" href="/wiki/%EC%99%95%EC%8B%AD%EB%A6%AC" title="왕십리">왕십리</a>, 영천, <a class="mw-redirect" href="/wiki/%EB%85%B8%EB%9F%89%EC%A7%84" title="노량진">노량진</a> 등 외곽지대로 확장되었으며 <a href="/wiki/1941%EB%85%84" title="1941년">1941년</a>에는 <a class="mw-redirect" href="/wiki/%EC%B0%BD%EA%B2%BD%EC%9B%90" title="창경원">창경원</a>-돈암동 네거리까지 연장되었다.<sup class="reference" id="cite_ref-1"><a href="#cite_note-1">[1]</a></sup> 그러나 <a class="mw-redirect" href="/wiki/%EA%B4%91%EB%B3%B5" title="광복">광복</a> 이후 <a href="/wiki/%EC%9E%90%EB%8F%99%EC%B0%A8" title="자동차">자동차</a>를 필두로 한 교통량의 증가로 <a href="/wiki/1968%EB%85%84" title="1968년">1968년</a> 버스로 대체 모두 폐기되어, 노면전차는 자취를 감추었지만 , <a href="/wiki/%EC%84%B1%EB%82%A8%EC%8B%9C" title="성남시">성남시</a>, <a href="/wiki/%EC%9C%84%EB%A1%80%EC%8B%A0%EB%8F%84%EC%8B%9C" title="위례신도시">위례신도시</a><sup class="reference" id="cite_ref-wirye_tram_2-0"><a href="#cite_note-wirye_tram-2">[2]</a></sup>,<a href="/wiki/%EC%88%98%EC%9B%90%EC%8B%9C" title="수원시">수원시</a>, <a href="/wiki/%EB%8C%80%EC%A0%84%EA%B4%91%EC%97%AD%EC%8B%9C" title="대전광역시">대전광역시</a>에서 노면전차가 부활할 예정이다.</p>
<p><a class="mw-redirect" href="/wiki/%EB%B6%81%ED%95%9C" title="북한">북한</a>에서는 <a class="mw-redirect" href="/wiki/%EC%9D%BC%EC%A0%9C%EA%B0%95%EC%A0%90%EA%B8%B0" title="일제강점기">일제강점기</a> 초반 <a class="mw-redirect" href="/wiki/%ED%8F%89%EC%96%91" title="평양">평양</a>에 건설되었다. 이후 폐지되었다가, <a href="/wiki/1991%EB%85%84" title="1991년">1991년</a>에 부활되었고 이는 오늘날에도 주요 교통수단 중의 하나이다. <a href="/wiki/%EC%B2%AD%EC%A7%84%EC%8B%9C" title="청진시">청진</a>의 경우, <a href="/wiki/1999%EB%85%84" title="1999년">1999년</a>도부터 노면 전차가 운행되고 있다.</p>

<h3><span class="mw-headline" id=".EB.8C.80.EC.B2.B4.EA.B5.90.ED.86.B5.EC.88.98.EB.8B.A8_.EA.B2.80.ED.86.A0">대체교통수단 검토</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%85%B8%EB%A9%B4%EC%A0%84%EC%B0%A8&amp;action=edit&amp;section=4" title="부분 편집: 대체교통수단 검토">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>최근에 천문학적인 예산이 투입되는 지하철의 대체 교통수단으로 검토되고 있다. 노면전차는 경량전철(輕量電鐵·경전철)의 여러 종류 중 하나로 <a href="/wiki/%EA%B2%BD%EC%A0%84%EC%B2%A0" title="경전철">경전철</a>은 <a class="new" href="/w/index.php?title=%EC%A4%91%EB%9F%89%EC%A0%84%EC%B2%A0&amp;action=edit&amp;redlink=1" title="중량전철 (없는 문서)">중량전철</a>인 지하철과 구분된다. 지하철보다 건설비가 적게 들고 공사 기간이 짧지만, 수송량은 지하철보다 소량이며, 버스보다는 많다. 한 칸에서 여섯 칸까지 만든다. 속도는 35∼40㎞이다.</p>
<h2><span class="mw-headline" id=".EC.97.AD.EC.82.AC">역사</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%85%B8%EB%A9%B4%EC%A0%84%EC%B0%A8&amp;action=edit&amp;section=5" title="부분 편집: 역사">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<h3><span class="mw-headline" id=".EC.A0.84.EC.B0.A8.EB.B0.9C.EB.AA.85">전차발명</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%85%B8%EB%A9%B4%EC%A0%84%EC%B0%A8&amp;action=edit&amp;section=6" title="부분 편집: 전차발명">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>전차는 <a href="/wiki/%EB%A7%88%EC%B0%A8%EC%B2%A0%EB%8F%84" title="마차철도">마차철도</a>의 동력인 마력을 대체할 <a href="/wiki/%EC%97%90%EB%84%88%EC%A7%80" title="에너지">에너지</a>를 생각함으로써 개발되었다. <a href="/wiki/1879%EB%85%84" title="1879년">1879년</a>에 <a href="/wiki/%EB%8F%85%EC%9D%BC" title="독일">독일</a>의 <a href="/wiki/%EC%A0%84%EA%B8%B0" title="전기">전기</a> 회사인 <a class="mw-disambig" href="/wiki/%EC%A7%80%EB%A9%98%EC%8A%A4" title="지멘스">지멘스</a>가 <a href="/wiki/%EB%B2%A0%EB%A5%BC%EB%A6%B0" title="베를린">베를린</a> 박람회에서의 시험주행을 시작으로, <a href="/wiki/1881%EB%85%84" title="1881년">1881년</a>에는 베를린 교외에서의 운행이 개시되었다. 그 후 <a href="/wiki/1887%EB%85%84" title="1887년">1887년</a>에 <a href="/wiki/%EB%AF%B8%EA%B5%AD" title="미국">미국</a>에서 현재와 같은 방식의 노면 전차가 개발되어 사람들의 신뢰를 받으며 보급되었다.</p>
<h3><span class="mw-headline" id=".EC.9C.A0.EB.9F.BD">유럽</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%85%B8%EB%A9%B4%EC%A0%84%EC%B0%A8&amp;action=edit&amp;section=7" title="부분 편집: 유럽">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>전기 궤도(노면 전차)는 먼저 미국의 각 도시에서 보급되어 유럽 각국으로 뻗어나갔으나 1920년대 미국에서는 <a class="mw-redirect" href="/wiki/%EC%9E%90%EA%B0%80%EC%9A%A9" title="자가용">자가용</a>이 보급되어 많은 <a href="/wiki/%EB%8F%84%EC%8B%9C" title="도시">도시</a>에서 폐지되어 간다. <a href="/wiki/%EC%9C%A0%EB%9F%BD" title="유럽">유럽</a>의 일부에서도 <a href="/wiki/%EB%9F%B0%EB%8D%98" title="런던">런던</a>, <a class="mw-redirect" href="/wiki/%ED%8C%8C%EB%A6%AC_(%EB%8F%84%EC%8B%9C)" title="파리 (도시)">파리</a><sup class="reference" id="cite_ref-3"><a href="#cite_note-3">[3]</a></sup> 등의 도시에서 폐지되었다.</p>
<h3><span class="mw-headline" id=".EA.B5.AC_.EC.86.8C.EB.A0.A8.EA.B3.BC_.EC.84.9C.EB.8F.85">구 소련과 서독</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%85%B8%EB%A9%B4%EC%A0%84%EC%B0%A8&amp;action=edit&amp;section=8" title="부분 편집: 구 소련과 서독">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>한편, 구 <a href="/wiki/%EC%86%8C%EB%A0%A8" title="소련">소련</a>과 <a href="/wiki/%EC%84%9C%EB%8F%85" title="서독">서독</a> 등에서 <a class="mw-redirect" href="/wiki/%EC%A0%9C2%EC%B0%A8_%EC%84%B8%EA%B3%84%EB%8C%80%EC%A0%84" title="제2차 세계대전">제2차 세계대전</a> 후에도 노면 전차를 활용했다. 서독에서 소규모 노면 전차는 폐지 되었지만, 인구 20만 이상의 도시에서는 환승전철의 투입, 신용카드를 이용한 운임 납부 등의 서비스를 제공하였다. 노선망의 폐지 및 신설도 빈번히 이루어졌고, 교외로의 노선 연장을 추진할 뿐 아니라 중심지의 노선의 지하화 등도 이루어졌다.</p>
<p>이 중 일부는 1960년대 후반 이후 전용 궤도화 등 노선의 고규격화를 추진해 속도의 향상이나 정시성 확보를 실시했다. <a class="new" href="/w/index.php?title=%ED%94%84%EB%9E%91%ED%81%AC%ED%91%B8%EB%A5%B4%ED%8A%B8_U-%EB%B0%98&amp;action=edit&amp;redlink=1" title="프랑크푸르트 U-반 (없는 문서)">프랑크푸르트 U-반</a>이나 <a href="/wiki/%EC%BE%B0%EB%A5%B8" title="쾰른">쾰른</a>, <a href="/wiki/%EC%8A%88%ED%88%AC%ED%8A%B8%EA%B0%80%EB%A5%B4%ED%8A%B8" title="슈투트가르트">슈투트가르트</a> 등에 대표되는 이 방식은 슈타트반으로 불리게 되었다.</p>
<p>이 변화는 미국에 영향을 주었다. 1970년대에 들어 미 연방대중교통국에 의해서 '<a href="/wiki/%EB%9D%BC%EC%9D%B4%ED%8A%B8_%EB%A0%88%EC%9D%BC" title="라이트 레일">라이트 레일</a>'의 정의가 내려진다. 그 내용은 '대부분을 전용 궤도로 주행하고, 부분적으로 도로상(병용 궤도)을 1량 또는 수 량의 열차가 전기에 의해서 구동되는, 누구라도 용이하게 이용할 수 있는 교통 시스템'이다. 즉 본래 '<a href="/wiki/%EB%9D%BC%EC%9D%B4%ED%8A%B8_%EB%A0%88%EC%9D%BC" title="라이트 레일">라이트 레일</a> (LRT)란 시스템이며, 또 노면을 주행하는 전철인 필요성은 부족한 점에 주의를 필요로 한다. 이것에 수반해 미국 및 (<a href="/wiki/%EC%BA%90%EB%82%98%EB%8B%A4" title="캐나다">캐나다</a>)에서 라이트 레일이 정비되어 가게 되지만, 안에는 전용 궤도뿐인 노선도 있다. 전반적으로 고속 철도 지향이 강하다.</p>
<p>1980년대가 되면, <a href="/wiki/%ED%94%84%EB%9E%91%EC%8A%A4" title="프랑스">프랑스</a>에서 도시내에서의 궤도 시스템 정비의 움직임이 강해져 온다. 프랑스에서는 슈타트반~라이트 레일이 아닌, 병용 궤도율의 비싼 노면 전차를 채용했다. 이것은 후에 다른 유럽에도 퍼져 가지만, 이것들도 일부에서는 <a href="/wiki/%EB%9D%BC%EC%9D%B4%ED%8A%B8_%EB%A0%88%EC%9D%BC" title="라이트 레일">라이트 레일</a>이라고 부르게 되었다.</p>
<p>따라서, 라이트 레일이라는 말은 미국에서의 원의·실례와 유럽에서의 호칭 사례의 사이에 차이가 있는 것에 주의를 필요로 한다.</p>
<p>이러한 궤도계 교통이 구미에서 주목받는 배경에는 시가 중심지에의 사람의 흐름을 확보해 진흥하는 수단으로서 또 환경 파괴를 막는 면으로부터 가치가 있다라고 하는 것이 생각할 수 있기 때문에 있다. 따라서, 그 정비는 토지 이용이나 인구 분포등의 점으로 도시 정책에 제대로 짜넣어지고 있는 것이 상이다. 또, 자가용차를 교외의 주차장에 두어 노면 전차나 버스 등에 갈아 타중심 시가지에 들어가는 파크 앤드 라이드 방식이나, 중심 시가에서 자동차의 노선연장을 금지해 공공 교통과 보행자만을 통행 가능하게 하는 아이디어가 태어나 실행되었다.</p>
<p>차량면에서는 고상 홈이 적은 유럽에서는 1980년대 후반보다 높이를 20~30cm정도로 낮춘 초저상 전철 의 개발이 진행되어 바리어 프리화가 촉진되었다. 노면전차는 <a href="/wiki/%EC%A0%84%EA%B8%B0" title="전기">전기</a>(電氣)로써 움직이는 <a class="mw-redirect" href="/wiki/%EC%A0%84%EA%B8%B0_%EC%B2%A0%EB%8F%84" title="전기 철도">전기 철도</a>의 일종이기 때문에 대기오염이 없다는 장점이 있다. 그러나 <a href="/wiki/%EB%B2%84%EC%8A%A4" title="버스">버스</a>에 비해 속력이 느려 교통을 방해할 수도 있다는 단점으로 있다. 또 전기를 배송하기 위한 전선을 설비해야 한다. 한편, 레일을 따라 움직이지 않고 타이어가 있는 바퀴를 쓰는 무궤도전차도 있다(보통의 전차는 궤도전차로 불린다.).</p>
<h3><span class="mw-headline" id=".ED.95.9C.EA.B5.AD.EC.9D.98_.EB.85.B8.EB.A9.B4.EC.A0.84.EC.B0.A8">한국의 노면전차</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%85%B8%EB%A9%B4%EC%A0%84%EC%B0%A8&amp;action=edit&amp;section=9" title="부분 편집: 한국의 노면전차">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p><a href="/wiki/%EC%84%9C%EC%9A%B8%EC%A0%84%EC%B0%A8" title="서울전차">서울</a>과 <a class="new" href="/w/index.php?title=%ED%8F%89%EC%96%91%EC%A0%84%EC%B0%A8&amp;action=edit&amp;redlink=1" title="평양전차 (없는 문서)">평양</a>, 그리고 <a href="/wiki/%EB%B6%80%EC%82%B0%EC%A0%84%EC%B0%A8" title="부산전차">부산</a>에서 노면 전차가 운행된 적이 있다. 1882년 미국과 수교를 한 후 1898년에 <a class="mw-redirect" href="/wiki/%ED%95%9C%EC%84%B1%EC%A0%84%EA%B8%B0%ED%9A%8C%EC%82%AC" title="한성전기회사">한성전기회사</a>가 설립되었고, 1899년 석가탄신일에 서대문과 동대문 간의 전차를 개통하였다. 미국인 콜브란은 교토 전철의 설계자인 마키 헤이이치로에게 전차부설의 책임을 맡겼고 그는 오하타 코우노스케와 이시쿠마 노부노유 등을 동반하여 1898년 10월 서울에 들어와 설계와 공사를 시행하였다. 발전소는 동대문 문루 아래에 건설을 하였고, 서대문에서 홍릉까지 단선의 궤도가 부설되었다. 전차는 정원 40인의 일반차 8대와 귀빈용차 1대였다. 그러나, 개통식 후 전차가 어린아이를 치여 죽이고 달아나는 사건이 발생하였는데, 격분한 군중들이 전차를 불태워버렸다. 이 일때문에 전차는 몇달간 운행을 중지하게 되었다.<sup class="정리_필요 noprint">[<a href="/wiki/%EC%9C%84%ED%82%A4%EB%B0%B1%EA%B3%BC:%EC%B6%9C%EC%B2%98_%EB%B0%9D%ED%9E%88%EA%B8%B0" title="위키백과:출처 밝히기">출처 필요</a>]</sup></p>
<p>개통된 전차는 일본의 <a href="/wiki/%EA%B5%90%ED%86%A0_%EC%8B%9C" title="교토 시">교토</a>보다 4년이 늦었지만 <a href="/wiki/%EB%8F%84%EC%BF%84_%EB%8F%84" title="도쿄 도">도쿄</a>보다 3년이 빨랐다. 당시 동경은 비교적 속도가 빠르고 유지비용이 적은 <a href="/wiki/%EB%A7%88%EC%B0%A8%EC%B2%A0%EB%8F%84" title="마차철도">마차철도</a>가 사용되고 있었다. 태국의 수도 방콕도 역시 1894년 철도마차를 대신하여 전기식 전차를 개통하였다.</p>
<p>근래에 천문학적인 규모의 예산이 들어가는 중량지하철을 대신하는 대중교통수단으로 도입이 건설/검토되고 있다. <a href="/wiki/%EC%9C%84%EB%A1%80%EC%8B%A0%EB%8F%84%EC%8B%9C" title="위례신도시">위례신도시</a>에서는 2017년부터 노면전차의 일종인 <a class="mw-redirect" href="/wiki/%ED%8A%B8%EB%9E%A8" title="트램">트램</a>이 운행 할 예정이며, <sup class="reference" id="cite_ref-wirye_tram_2-1"><a href="#cite_note-wirye_tram-2">[2]</a></sup> 한국교통연구원은 경상남도의 의뢰로 용역을 수행한 결과 <a href="/wiki/%EC%B0%BD%EC%9B%90_%EB%8F%84%EC%8B%9C%EC%B2%A0%EB%8F%84" title="창원 도시철도">창원 도시철도</a>의 최적 차량시스템으로 '노면전차'를 꼽았다. <sup class="reference" id="cite_ref-4"><a href="#cite_note-4">[4]</a></sup></p>
