<p><b>노비사드</b>(<span style="font-size: smaller;"><a href="/wiki/%EC%84%B8%EB%A5%B4%EB%B9%84%EC%95%84%EC%96%B4" title="세르비아어">세르비아어</a>:</span> <span lang="sr" xml:lang="sr">Novi Sad / Нови Сад</span>, <span style="font-size: smaller;"><a href="/wiki/%ED%97%9D%EA%B0%80%EB%A6%AC%EC%96%B4" title="헝가리어">헝가리어</a>:</span> <span lang="hu" xml:lang="hu">Újvidék</span> <small>우이비데크<sup>[<a href="/wiki/%EC%9C%84%ED%82%A4%EB%B0%B1%EA%B3%BC:%ED%97%9D%EA%B0%80%EB%A6%AC%EC%96%B4%EC%9D%98_%ED%95%9C%EA%B8%80_%ED%91%9C%EA%B8%B0" title="위키백과:헝가리어의 한글 표기">*</a>]</sup></small>)는 <a href="/wiki/%EC%84%B8%EB%A5%B4%EB%B9%84%EC%95%84" title="세르비아">세르비아</a> 북부 <a class="mw-redirect" href="/wiki/%EB%B3%B4%EC%9D%B4%EB%B3%B4%EB%94%94%EB%82%98_%EC%9E%90%EC%B9%98%EC%A3%BC" title="보이보디나 자치주">보이보디나 자치주</a>에 있는 도시이다. 인구 400,000(2010). 보이보디나 자치주의 주도이자 <a href="/wiki/%EB%82%A8%EB%B0%94%EC%B9%98%EC%B9%B4_%EA%B5%AC" title="남바치카 구">남바치카 구</a>의 행정 중심지이며, 세르비아에서 수도 <a href="/wiki/%EB%B2%A0%EC%98%A4%EA%B7%B8%EB%9D%BC%EB%93%9C" title="베오그라드">베오그라드</a> 다음으로 큰 도시이다.</p>
<p>베오그라드 북쪽 30km 지점의 <a href="/wiki/%EB%8F%84%EB%82%98%EC%9A%B0_%EA%B0%95" title="도나우 강">도나우 강</a> 유역에 면한다. <a href="/wiki/17%EC%84%B8%EA%B8%B0" title="17세기">17세기</a>에 건설되었고, 강 연안의 어촌이었으나, <a class="mw-redirect" href="/wiki/%EC%98%A4%EC%8A%A4%EB%A7%8C_%ED%88%AC%EB%A5%B4%ED%81%AC" title="오스만 투르크">오스만 투르크</a>와 <a href="/wiki/%EC%98%A4%EC%8A%A4%ED%8A%B8%EB%A6%AC%EC%95%84" title="오스트리아">오스트리아</a>의 경계 지점에 있어 군사적으로 중요해지면서 발전하였다. <a href="/wiki/19%EC%84%B8%EA%B8%B0" title="19세기">19세기</a> 오스만 투르크 치하의 <a href="/wiki/%EC%84%B8%EB%A5%B4%EB%B9%84%EC%95%84%EC%9D%B8" title="세르비아인">세르비아인</a>들이 도나우 강을 넘어 이 곳으로 정착하여 <a href="/wiki/%EC%98%A4%EC%8A%A4%ED%8A%B8%EB%A6%AC%EC%95%84-%ED%97%9D%EA%B0%80%EB%A6%AC_%EC%A0%9C%EA%B5%AD" title="오스트리아-헝가리 제국">오스트리아-헝가리 제국</a>에서 세르비아인의 중심지가 되었다. 오스트리아-헝가리 제국 시절에는 독일어로 노이자츠(<span style="font-size: smaller;"><a href="/wiki/%EB%8F%85%EC%9D%BC%EC%96%B4" title="독일어">독일어</a>:</span> <span lang="de" xml:lang="de">Neusatz</span>)라고 했다.</p>
<p>다양한 공업이 이루어지는 공업 도시이며, 도나우 강 연안의 중요한 도시이다. <a href="/wiki/%EB%B0%9C%EC%B9%B8_%EB%B0%98%EB%8F%84" title="발칸 반도">발칸 반도</a>와 <a href="/wiki/%EC%A4%91%EC%95%99%EC%9C%A0%EB%9F%BD" title="중앙유럽">중앙유럽</a>을 잇는 교통의 요지이다. 주민은 75%가 세르비아인, 5%가 <a href="/wiki/%ED%97%9D%EA%B0%80%EB%A6%AC%EC%9D%B8" title="헝가리인">헝가리인</a>이며, 기타 여러 소수 민족이 있다.</p>
