<h1 class="firstHeading" id="firstHeading" lang="ko">노섬브리아 왕국</h1>
<div class="mw-body-content" id="bodyContent">
<div id="siteSub">위키백과, 우리 모두의 백과사전.</div>
<div id="contentSub"></div>
<div class="mw-jump" id="jump-to-nav">
					이동:					<a href="#mw-head">둘러보기</a>, 					<a href="#p-search">검색</a>
</div>
<div class="mw-content-ltr" dir="ltr" id="mw-content-text" lang="ko"><script>function mfTempOpenSection(id){var block=document.getElementById("mf-section-"+id);block.className+=" open-block";block.previousSibling.className+=" open-block";}</script><p><b>노섬브리아 왕국</b>(<span style="font-size: smaller;"><a href="/wiki/%EA%B3%A0%EB%8C%80_%EC%98%81%EC%96%B4" title="고대 영어">고대 영어</a>:</span> <span lang="ang" xml:lang="ang">Norþhymbra rīce</span>)은 <a href="/wiki/%EC%98%81%EA%B5%AD" title="영국">영국</a> <a class="mw-redirect" href="/wiki/%EC%95%B5%EA%B8%80%EB%A1%9C%EC%83%89%EC%8A%A8" title="앵글로색슨">앵글로색슨족</a>의 <a class="mw-redirect" href="/wiki/%EC%B9%A0%EC%99%95%EA%B5%AD" title="칠왕국">7왕국</a>(王國) 중 하나이다.</p>
<h2><span class="mw-headline" id=".EA.B0.9C.EC.9A.94">개요</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%85%B8%EC%84%AC%EB%B8%8C%EB%A6%AC%EC%95%84_%EC%99%95%EA%B5%AD&amp;action=edit&amp;section=1" title="부분 편집: 개요">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p><a class="mw-redirect" href="/wiki/%EC%95%B5%EA%B8%80%EC%A1%B1" title="앵글족">앵글족</a>의 왕국 가운데 하나였던 베르니시아는 7세기 서부 <a class="mw-redirect" href="/wiki/%ED%94%BD%ED%8A%B8%EC%A1%B1" title="픽트족">픽트족</a>의 <a class="mw-redirect" href="/wiki/%EB%8B%AC_%EB%A6%AC%EC%95%84%ED%83%80" title="달 리아타">달 리아타 왕국</a>을 통합하였고, 후에 노섬브리아 왕국으로 발전한다.</p>
<p><a class="new" href="/w/index.php?title=%EB%B2%A0%EB%A5%B4%EB%8B%88%EC%8B%9C%EC%95%84&amp;action=edit&amp;redlink=1" title="베르니시아 (없는 문서)">베르니시아</a>에서 <a href="/wiki/547%EB%85%84" title="547년">547년</a>에 건국하였다. <a href="/wiki/610%EB%85%84" title="610년">610년</a>경 <a class="new" href="/w/index.php?title=%EB%8D%B0%EC%9D%B4%EB%9D%BC_%EC%99%95%EA%B5%AD&amp;action=edit&amp;redlink=1" title="데이라 왕국 (없는 문서)">데이라 왕국</a>을 지배하고, <a class="new" href="/w/index.php?title=%EB%85%B8%EC%84%AC%EB%B8%8C%EB%A6%AC%EC%95%84_%EC%A3%BC&amp;action=edit&amp;redlink=1" title="노섬브리아 주 (없는 문서)">노섬브리아 지방</a>을 제패하였으나 그의 아들인 <a class="new" href="/w/index.php?title=%EC%98%A4%EC%A6%88%EC%99%88%EB%93%9C&amp;action=edit&amp;redlink=1" title="오즈왈드 (없는 문서)">오즈왈드 왕</a>이 <a href="/wiki/641%EB%85%84" title="641년">641년</a>에 <a href="/wiki/%EB%A8%B8%EC%8B%9C%EC%95%84_%EC%99%95%EA%B5%AD" title="머시아 왕국">머시아 왕국</a>의 국왕 <a class="new" href="/w/index.php?title=%ED%8E%9C%EB%8B%A4&amp;action=edit&amp;redlink=1" title="펜다 (없는 문서)">펜다</a>에게 패배하고 만다. 그 후 <a href="/wiki/%ED%94%84%EB%9E%91%ED%81%AC_%EC%99%95%EA%B5%AD" title="프랑크 왕국">프랑크 왕국</a>으로부터 원조를 받아 머시아에 대항하였으나, <a href="/wiki/825%EB%85%84" title="825년">825년</a> <a href="/wiki/%EC%9B%A8%EC%84%B9%EC%8A%A4_%EC%99%95%EA%B5%AD" title="웨섹스 왕국">웨섹스 왕국</a>의 국왕인 <a href="/wiki/%EC%97%90%EA%B7%B8%EB%B2%84%ED%8A%B8" title="에그버트">에그버트</a>에게 항복하였다. 북방의 <a class="mw-redirect" href="/wiki/%EB%B8%8C%EB%A6%AC%ED%8A%BC%EC%A1%B1" title="브리튼족">브리튼족</a>의 침입, 국내 귀족의 반항 등으로 고통을 받았으나, <a href="/wiki/5%EC%84%B8%EA%B8%B0" title="5세기">5세기</a>경부터는 <a href="/wiki/%EA%B8%B0%EB%8F%85%EA%B5%90" title="기독교">기독교</a>를 받아들이고, <a href="/wiki/%EC%95%84%EC%9D%BC%EB%9E%9C%EB%93%9C%EC%9D%B8" title="아일랜드인">아일랜드인</a> 등의 도래가 이어져, 문화는 번영하였다.</p>
