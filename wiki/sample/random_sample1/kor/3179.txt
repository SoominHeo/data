<p><b>노세 정</b>(<span style="font-size: smaller;"><a href="/wiki/%EC%9D%BC%EB%B3%B8%EC%96%B4" title="일본어">일본어</a>:</span> <span lang="ja" xml:lang="ja">能勢町</span>)은 <a href="/wiki/%EC%98%A4%EC%82%AC%EC%B9%B4_%EB%B6%80" title="오사카 부">오사카 부</a> 북서단에 있는 <a href="/wiki/%EB%8F%84%EC%9A%94%EB%85%B8_%EA%B5%B0" title="도요노 군">도요노 군</a>의 정이다. 면적은 98.68km²이고 인구는 <a href="/wiki/2009%EB%85%84" title="2009년">2009년</a> <a href="/wiki/10%EC%9B%94_1%EC%9D%BC" title="10월 1일">10월 1일</a> 기준으로 11,874명이다.</p>
<p><a href="/wiki/%EA%B5%90%ED%86%A0_%EB%B6%80" title="교토 부">교토 부</a> 남서부의 <a href="/wiki/%EC%98%A4%ED%86%A0%EC%BF%A0%EB%8B%88_%EA%B5%B0" title="오토쿠니 군">오토쿠니 군</a> <a href="/wiki/%EC%98%A4%EC%95%BC%EB%A7%88%EC%9E%90%ED%82%A4_%EC%A0%95" title="오야마자키 정">오야마자키 정</a>과 인접해있다. 가쓰라가와 강, 우지 강, 기즈 강이 야마자키 지협에서 합류해 <a class="mw-redirect" href="/wiki/%EC%9A%94%EB%8F%84%EA%B0%80%EC%99%80_%EA%B0%95" title="요도가와 강">요도가와 강</a>이 되는 장소에 있어 교토 분지에서 오사카 평야로 빠지는 예부터 교통의 요충지였다. <a href="/wiki/%EC%98%A4%EC%82%AC%EC%B9%B4_%EC%8B%9C" title="오사카 시">오사카 시</a>나 <a href="/wiki/%EA%B5%90%ED%86%A0_%EC%8B%9C" title="교토 시">교토 시</a>의 주택 지역으로서 주택지 개발도 진행되고 있다.</p>

<h2><span class="mw-headline" id=".EC.A7.80.EB.A6.AC">지리</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%85%B8%EC%84%B8_%EC%A0%95&amp;action=edit&amp;section=1" title="부분 편집: 지리">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>오사카 부 최북단에 위치하고 정의 대부분은 산림, 전답이다. "오사카의 가루이자와", "오사카의 홋카이도", "오사카의 티베트"라고 표현되듯이 정내 전역이 높이 200m 지대로 오사카 시내와 5~6℃기온차이가 있어 여름에는 시원하지만 겨울에는 매일 같이 영하까지 차가워진다.</p>
<p>정은 중앙의 산을 경계로 서쪽과 동쪽으로 나뉜다. 근처 역은 <a class="new" href="/w/index.php?title=%EB%85%B8%EC%84%B8_%EC%A0%84%EC%B2%A0&amp;action=edit&amp;redlink=1" title="노세 전철 (없는 문서)">노세 전철</a>의 <a class="new" href="/w/index.php?title=%EC%82%B0%EC%B9%B4_%EC%97%AD&amp;action=edit&amp;redlink=1" title="산카 역 (없는 문서)">산카 역</a>, <a class="new" href="/w/index.php?title=%EB%AC%98%EC%BC%84%EA%B5%AC_%EC%97%AD&amp;action=edit&amp;redlink=1" title="묘켄구 역 (없는 문서)">묘켄구 역</a>뿐이지만 고개를 넘어 가메오카, 교토 시내나 미타, 시노야마 방면으로도 향한다. 상업 시설이나 행정 시설은 서쪽 지구에 집중하지만 도요노 경찰서가 동쪽 지구에 존재하는 등 정내에 각 시설이 분산하고 있다.</p>
<p>철도역으로부터 거리가 있고 국도가 정비되고 있어서 자동차·오토바이·노선버스가 중요한 이동 수단이 되고 있다.</p>
<h2><span class="mw-headline" id=".EC.97.AD.EC.82.AC">역사</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%85%B8%EC%84%B8_%EC%A0%95&amp;action=edit&amp;section=2" title="부분 편집: 역사">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<ul>
<li>1896년 : 도요노 군이 성립하였다.</li>
<li>1956년 : 우타카키 촌, 다지리 촌, 니시노세 촌이 합병, 정으로 승격해 도요노 군 노세 정이 탄생하였다.</li>
</ul>
