<p><b>F-100</b>는 <a href="/wiki/%EB%AF%B8%EA%B5%AD" title="미국">미국</a>의 <a class="new" href="/w/index.php?title=%EB%85%B8%EC%8A%A4_%EC%95%84%EB%A9%94%EB%A6%AC%EC%B9%B8&amp;action=edit&amp;redlink=1" title="노스 아메리칸 (없는 문서)">노스 아메리칸</a>사에서 개발된 세계 최초의 <a href="/wiki/%EC%B4%88%EC%9D%8C%EC%86%8D" title="초음속">초음속</a> <a href="/wiki/%EC%A0%84%ED%88%AC%EA%B8%B0" title="전투기">전투기</a>로, <a href="/wiki/F-86_%EC%84%B8%EC%9D%B4%EB%B2%84" title="F-86 세이버">F-86 세이버</a>의 후속기로 개발되었다. 별칭은 슈퍼 세이버(Super Sabre)이다.</p>

<h2><span class="mw-headline" id=".EA.B0.9C.EB.B0.9C">개발</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%85%B8%EC%8A%A4%EC%95%84%EB%A9%94%EB%A6%AC%EC%B9%B8_F-100_%EC%8A%88%ED%8D%BC%EC%84%B8%EC%9D%B4%EB%B2%84&amp;action=edit&amp;section=1" title="부분 편집: 개발">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p><a href="/wiki/1949%EB%85%84" title="1949년">1949년</a> 2월, F-86의 후계기로 주 날개 후퇴각이 F-86의 35도에서 45도로 변경된 것을 의미한 '세이버 45' 라는 명칭을 가지고 개발을 시작하였다. <a class="new" href="/w/index.php?title=1951%EB%85%84_11%EC%9B%94&amp;action=edit&amp;redlink=1" title="1951년 11월 (없는 문서)">1951년 11월</a>, 미국 공군과 노스 아메리칸사와의 사이에 시험 제작기 YF-100 A·2기와 양산기 F-100 A·203기의 제조 계약이 이루어졌으며, <a href="/wiki/1953%EB%85%84" title="1953년">1953년</a> <a href="/wiki/5%EC%9B%94_25%EC%9D%BC" title="5월 25일">5월 25일</a>에 XJ57-P-7 터보 제트 엔진을 장착한 최초의 YF-100A 시험 제작기가 음속을 돌파하면서 첫 비행에 성공하였다. <a href="/wiki/1953%EB%85%84" title="1953년">1953년</a> <a href="/wiki/10%EC%9B%94_29%EC%9D%BC" title="10월 29일">10월 29일</a>, YF-100A의 순항속도가 당시로서는 세계 기록인 1,215 km/h를 기록하였으며 개선을 통해 미국 공군에 도입 되었다.</p>
<p>노스 아메리칸사에서는 발전형의 YF-107도 시작되었지만, 채용에 이르지는 않았다.</p>
<h2><span class="mw-headline" id=".EA.B8.B0.EC.B2.B4">기체</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%85%B8%EC%8A%A4%EC%95%84%EB%A9%94%EB%A6%AC%EC%B9%B8_F-100_%EC%8A%88%ED%8D%BC%EC%84%B8%EC%9D%B4%EB%B2%84&amp;action=edit&amp;section=2" title="부분 편집: 기체">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>기수에 인 테이크를 가지고 있으며, 후퇴각을 가지는 주 날개는 약간 높은 저날개로 배치가 되었다. 공중 급유 능력이 없었지만, 후에 개수에 의해 수유 프로브가 증설되었다.</p>
<h2><span class="mw-headline" id=".EC.9A.B4.EC.9A.A9">운용</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%85%B8%EC%8A%A4%EC%95%84%EB%A9%94%EB%A6%AC%EC%B9%B8_F-100_%EC%8A%88%ED%8D%BC%EC%84%B8%EC%9D%B4%EB%B2%84&amp;action=edit&amp;section=3" title="부분 편집: 운용">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<h3><span class="mw-headline" id=".EB.AF.B8.EA.B5.AD">미국</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%85%B8%EC%8A%A4%EC%95%84%EB%A9%94%EB%A6%AC%EC%B9%B8_F-100_%EC%8A%88%ED%8D%BC%EC%84%B8%EC%9D%B4%EB%B2%84&amp;action=edit&amp;section=4" title="부분 편집: 미국">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>본래 영공 방어를 위해 개발되었으나, <a class="new" href="/w/index.php?title=%EC%A0%84%ED%88%AC_%ED%8F%AD%EA%B2%A9%EA%B8%B0&amp;action=edit&amp;redlink=1" title="전투 폭격기 (없는 문서)">전투 폭격기</a>로서의 운용이 많았다.</p>
<p><a href="/wiki/1953%EB%85%84" title="1953년">1953년</a>, F-100A가 군에 인도된 뒤 시험비행을 거쳐 <a class="new" href="/w/index.php?title=1954%EB%85%84_9%EC%9B%94&amp;action=edit&amp;redlink=1" title="1954년 9월 (없는 문서)">1954년 9월</a>부터 운용을 시작했다. 도입 당시, <a href="/wiki/%EB%83%89%EC%A0%84" title="냉전">냉전</a>으로 <a class="new" href="/w/index.php?title=MiG-19_(%ED%95%AD%EA%B3%B5%EA%B8%B0)&amp;action=edit&amp;redlink=1" title="MiG-19 (항공기) (없는 문서)">MiG-19</a>의 존재를 인식하지 못했던 미국에서는 세계에서 가장 고성능의 전투기로 평가하였다. 이후 롤 중에 사고가 잇따랐으며, <a href="/wiki/1954%EB%85%84" title="1954년">1954년</a> <a href="/wiki/10%EC%9B%94_12%EC%9D%BC" title="10월 12일">10월 12일</a>에는 급강하 사고가 일어나 조종사가 회복 불능상태가 되어 <a class="new" href="/w/index.php?title=1954%EB%85%84_11%EC%9B%94&amp;action=edit&amp;redlink=1" title="1954년 11월 (없는 문서)">1954년 11월</a>부터 <a class="new" href="/w/index.php?title=1955%EB%85%84_2%EC%9B%94&amp;action=edit&amp;redlink=1" title="1955년 2월 (없는 문서)">1955년 2월</a>까지 비행 정지 조치를 취해, 수직꼬리 면적을 27% 확대함과 동시에 주 날개의 연장 등의 개수를 했다. <a href="/wiki/1955%EB%85%84" title="1955년">1955년</a>, <b>F-100C</b>가 시험 비행을 거쳐 도입되었으며, <a href="/wiki/1956%EB%85%84" title="1956년">1956년</a>에는 날개 면적 등을 증가시킨 전투 폭격기형 <b>F-100D</b>가 도입되었다. 도입 이후 <a href="/wiki/1968%EB%85%84" title="1968년">1968년</a>까지 미국 공군의 <a class="new" href="/w/index.php?title=%EA%B3%A1%EC%98%88%EB%B9%84%ED%96%89%EB%8C%80&amp;action=edit&amp;redlink=1" title="곡예비행대 (없는 문서)">곡예비행대</a> · <a class="mw-disambig" href="/wiki/%EC%84%A0%EB%8D%94%EB%B2%84%EB%93%9C" title="선더버드">선더버드</a>에서 사용된 것 외에 <a class="new" href="/w/index.php?title=%EC%A0%9C%EB%A1%9C_%EA%B1%B0%EB%A6%AC_%EB%B0%9C%EC%A7%84&amp;action=edit&amp;redlink=1" title="제로 거리 발진 (없는 문서)">제로 거리 발진</a>의 시험 등 실용적인 측면에서도 다양하게 이용되었다.</p>
<p>이후, F-100D가 <a href="/wiki/%EB%B2%A0%ED%8A%B8%EB%82%A8_%EC%A0%84%EC%9F%81" title="베트남 전쟁">베트남 전쟁</a>에서 전투 폭격기로 잠시나마 이용되나, <a class="new" href="/w/index.php?title=F-105&amp;action=edit&amp;redlink=1" title="F-105 (없는 문서)">F-105</a>가 도입된 이후 폭격 임무에서는 배제된다. F-105가 <a class="mw-redirect" href="/wiki/MiG-17" title="MiG-17">MiG-17</a>에 격추되어 손실이 커지자 호위 전투기로 사용하게 되었는데, 당시 공대공 전투의 필수 장비가 되고 있던 수색 레이더를 가지지 않았고, 조준기도 F-86A가 사용했던 APG-30을 장비하고 있어 F-105와는 달리 MiG-17을 격추할 수는 없었다.</p>
<p><a class="mw-redirect" href="/wiki/F-4" title="F-4">F-4</a>가 등장하면서 <a class="mw-redirect" href="/wiki/%EB%82%A8%EB%B2%A0%ED%8A%B8%EB%82%A8" title="남베트남">남베트남</a>의 <a class="new" href="/w/index.php?title=%ED%95%AD%EA%B3%B5_%EC%9E%91%EC%A0%84&amp;action=edit&amp;redlink=1" title="항공 작전 (없는 문서)">근접 항공 지원</a>임무로 돌려져 <a href="/wiki/1971%EB%85%84" title="1971년">1971년</a>까지 사용되었으나, 여기에서 또한 무수한 단점이 발견되어 미국 공군이 <a class="mw-redirect mw-disambig" href="/wiki/A-10" title="A-10">A-10</a>을 개발하는 계기가 된다.</p>
<p>베트남 전쟁이 끝난 이후 <a href="/wiki/1979%EB%85%84" title="1979년">1979년</a>까지 미국 공군 주 방위군에서 사용하였으며, 퇴역 처리된 기체 중 일부를 터키 공군에 양도하였다.</p>
<h3><span class="mw-headline" id=".ED.84.B0.ED.82.A4">터키</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%85%B8%EC%8A%A4%EC%95%84%EB%A9%94%EB%A6%AC%EC%B9%B8_F-100_%EC%8A%88%ED%8D%BC%EC%84%B8%EC%9D%B4%EB%B2%84&amp;action=edit&amp;section=5" title="부분 편집: 터키">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p><a class="mw-redirect" href="/wiki/F-4" title="F-4">F-4E</a>나 <a class="mw-redirect" href="/wiki/F-104" title="F-104">F-104G</a>, <a class="new" href="/w/index.php?title=F-102&amp;action=edit&amp;redlink=1" title="F-102 (없는 문서)">F-102A</a>, <a class="new" href="/w/index.php?title=F-84&amp;action=edit&amp;redlink=1" title="F-84 (없는 문서)">RF-84F</a>등과 함께 <a href="/wiki/%EC%82%AC%EC%9D%B4%ED%94%84%EB%9F%AC%EC%8A%A4" title="사이프러스">사이프러스</a>를 둘러싼 <a href="/wiki/%EA%B7%B8%EB%A6%AC%EC%8A%A4" title="그리스">그리스</a>와의 전쟁에 투입되었다. 대지 공격에는 큰 힘을 발휘하였지만 장비가 구식이라 손실도 만만치 않았다. (<a href="/wiki/1974%EB%85%84" title="1974년">1974년</a> <a href="/wiki/7%EC%9B%94_21%EC%9D%BC" title="7월 21일">7월 21일</a>, 그리스 해군 <a class="new" href="/w/index.php?title=%ED%94%8C%EB%A0%88%EC%B2%98%EA%B8%89_%EA%B5%AC%EC%B6%95%ED%95%A8&amp;action=edit&amp;redlink=1" title="플레처급 구축함 (없는 문서)">플레처급 구축함</a>의 40mm 대공포에 의한 포화로 한 번에 4기가 격추되기도 하였다). 미국 공군 및 덴마크 공군에서 <a class="new" href="/w/index.php?title=%ED%87%B4%EC%97%AD&amp;action=edit&amp;redlink=1" title="퇴역 (없는 문서)">퇴역</a> 처리된 기체를 양도받아 <a href="/wiki/1990%EB%85%84%EB%8C%80" title="1990년대">1990년대</a> 후반까지 사용하였다.</p>
