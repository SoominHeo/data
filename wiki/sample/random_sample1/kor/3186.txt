<p><b>노스파이어</b>(Norsefire)는 <a href="/wiki/%EC%95%A8%EB%9F%B0_%EB%AC%B4%EC%96%B4" title="앨런 무어">앨런 무어</a>와 <a href="/wiki/%EB%8D%B0%EC%9D%B4%EB%B9%84%EB%93%9C_%EB%A1%9C%EC%9D%B4%EB%93%9C" title="데이비드 로이드">데이비드 로이드</a>의 《<a href="/wiki/%EB%B8%8C%EC%9D%B4_%ED%8F%AC_%EB%B2%A4%EB%8D%B0%ED%83%80" title="브이 포 벤데타">브이 포 벤데타</a>》 세계관에서 영국을 지배하는 가공의 <a href="/wiki/%ED%8C%8C%EC%8B%9C%EC%A6%98" title="파시즘">파시즘</a> <a href="/wiki/%EC%A0%95%EB%8B%B9" title="정당">정당</a>이다. 무어와 로이드는 이 독재 정당의 이름을 왜 "노스파이어"라고 지었는지 설명한 적이 없는데, <a class="mw-disambig" href="/wiki/%EA%B5%AD%EB%AF%BC%EC%A0%84%EC%84%A0" title="국민전선">국민전선</a>의 약자가 NF이고, 불타는 횃불이 그 상징이기 때문에, 국민 전선이 모티브일 수도 있다. 국민 전선은 1980년대 초기에 두드러지게 강해졌고, 그들의 신문 이름이 <i>The Flame</i>이었다.</p>
<p>노스파이어의 상징은 검은색 바탕에 N 글씨가 있는 깃발과 날개달린 십자가다. <a href="/wiki/%EB%B8%8C%EC%9D%B4_%ED%8F%AC_%EB%B2%A4%EB%8D%B0%ED%83%80_(%EC%98%81%ED%99%94)" title="브이 포 벤데타 (영화)">실사 영화</a>에서는 <a href="/wiki/%EB%A1%9C%EB%A0%8C_%EC%8B%AD%EC%9E%90" title="로렌 십자">로렌 십자</a>로 바뀌었다.</p>

<h2><span class="mw-headline" id=".EB.B0.B0.EA.B2.BD">배경</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%85%B8%EC%8A%A4%ED%8C%8C%EC%9D%B4%EC%96%B4&amp;action=edit&amp;section=1" title="부분 편집: 배경">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>이야기는 <a href="/wiki/%EC%A2%85%EB%A7%90%EB%AC%BC" title="종말물">종말물</a>로, <a href="/wiki/%EC%A0%9C3%EC%B0%A8_%EC%84%B8%EA%B3%84_%EB%8C%80%EC%A0%84" title="제3차 세계 대전">제3차 세계 대전</a>, <a class="mw-redirect" href="/wiki/%ED%95%B5_%EC%A0%84%EC%9F%81" title="핵 전쟁">핵 전쟁</a> 이후의 희망이 없는 <a href="/wiki/%EC%98%81%EA%B5%AD" title="영국">영국</a>을 묘사하고 있다. 집권 정당인 노스파이어는 <a href="/wiki/%EC%95%A0%EB%8D%A4_%EC%88%98%EC%A0%84" title="애덤 수전">애덤 수전</a>이 이끄는 정당이었는데, <a href="/wiki/%EC%95%A0%EB%8D%A4_%EC%88%98%EC%A0%84" title="애덤 수전">애덤 수전</a>은 주로 작에서 <b>애덤 서틀러</b> (<a href="/wiki/%EC%95%84%EB%8F%8C%ED%94%84_%ED%9E%88%ED%8B%80%EB%9F%AC" title="아돌프 히틀러">아돌프 히틀러</a>의 패러디) 로 묘사되었다. 당원들은 <a href="/wiki/%EB%82%98%EC%B9%98_%EA%B2%BD%EB%A1%80" title="나치 경례">나치 경례</a>를 하였고, 노스파이어는 작에서 <a href="/wiki/%EC%9E%89%EA%B8%80%EB%9E%9C%EB%93%9C_%EC%84%B1%EA%B3%B5%ED%9A%8C" title="잉글랜드 성공회">잉글랜드 성공회</a>의 지원을 받았으며, <a href="/wiki/%EA%B7%B9%EC%9A%B0" title="극우">극우</a>와 <a href="/wiki/%ED%8C%8C%EC%8B%9C%EC%A6%98" title="파시즘">파시즘</a> 적 정책을 폈다. 노스파이어는 <i><a class="new" href="/w/index.php?title=%EB%85%B8%EC%8A%A4%ED%8C%8C%EC%9D%B4%EC%96%B4_%ED%85%94%EB%A0%88%EB%B9%84%EC%A0%84&amp;action=edit&amp;redlink=1" title="노스파이어 텔레비전 (없는 문서)">노스파이어 텔레비전</a></i> 이나 기타 텔레비전 방송사들을 소유하고 <a class="mw-redirect" href="/wiki/%ED%94%84%EB%A1%9C%ED%8C%8C%EA%B0%84%EB%8B%A4" title="프로파간다">프로파간다</a>를 배포하고 거리 곳곳에 붙였으며, <a class="mw-redirect" href="/wiki/%EB%B9%84%EB%B0%80_%EA%B2%BD%EC%B0%B0" title="비밀 경찰">비밀 경찰</a> 운용, <a href="/wiki/%EB%B0%B1%EC%9D%B8" title="백인">백인</a>이 아닌 사람, <a href="/wiki/%EA%B8%B0%EB%8F%85%EA%B5%90" title="기독교">기독교</a>인이 아닌 사람, <a class="mw-redirect" href="/wiki/%EC%9D%B4%EC%84%B1%EC%95%A0%EC%9E%90" title="이성애자">이성애자</a>가 아닌 사람들을 모두 <a href="/wiki/%EA%B0%95%EC%A0%9C_%EC%88%98%EC%9A%A9%EC%86%8C" title="강제 수용소">강제 수용소</a>로 이송하였으며, <a href="/wiki/%EC%A2%8C%EC%9D%B5" title="좌익">좌익</a>이나 반대자들은 모조리 제거되기도 하였다. 이에 작에서 <b>V</b> 로 묘사되는 <a class="mw-redirect" href="/wiki/%EC%95%84%EB%82%98%ED%82%A4%EC%8A%A4%ED%8A%B8" title="아나키스트">아나키스트</a>는, 노스파이어를 붕괴시키고 <a href="/wiki/%EC%98%81%EA%B5%AD" title="영국">영국</a>에 <a href="/wiki/%EC%9E%90%EC%9C%A0" title="자유">자유</a>를 되찾기 위해 싸우는 이야기가 그려지게 된다.</p>
<h3><span class="mw-headline" id=".ED.91.9C.EC.96.B4">표어</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%85%B8%EC%8A%A4%ED%8C%8C%EC%9D%B4%EC%96%B4&amp;action=edit&amp;section=2" title="부분 편집: 표어">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<div class="thumb tright">
<div class="thumbinner" style="width:252px;"><a class="image" href="/wiki/%ED%8C%8C%EC%9D%BC:NorsefirePoster.png"><img alt="" class="thumbimage" data-file-height="622" data-file-width="493" height="315" src="//upload.wikimedia.org/wikipedia/commons/thumb/f/f4/NorsefirePoster.png/250px-NorsefirePoster.png" srcset="//upload.wikimedia.org/wikipedia/commons/thumb/f/f4/NorsefirePoster.png/375px-NorsefirePoster.png 1.5x, //upload.wikimedia.org/wikipedia/commons/f/f4/NorsefirePoster.png 2x" width="250"/></a>
<div class="thumbcaption">
<div class="magnify"><a class="internal" href="/wiki/%ED%8C%8C%EC%9D%BC:NorsefirePoster.png" title="실제 크기로"></a></div>
작 내 노스파이어의 <a href="/wiki/%ED%8F%AC%EC%8A%A4%ED%84%B0" title="포스터">포스터</a>. 노스파이어의 표어가 쓰여져 있다.</div>
</div>
</div>
<p>노스파이어의 작 내 슬로건은, <i>힘은 순수에서 오며, 순수는 신념에서부터 온다</i> (만화), <i>힘은 연합에서 오며, 연합은 신념에서부터 온다</i> (영화) 였다. 이는, <a href="/wiki/%EC%98%81%EA%B5%AD" title="영국">영국</a> 내에 실존했던 정당인 <a href="/wiki/%EC%98%81%EA%B5%AD_%ED%8C%8C%EC%8B%9C%EC%8A%A4%ED%8A%B8_%EC%97%B0%ED%95%A9" title="영국 파시스트 연합">영국 파시스트 연합</a>의 표어 "연합을 통한 행동" 에서 따온 것이다.</p>
<h3><span class="mw-headline" id=".EC.83.81.EC.A7.95">상징</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%85%B8%EC%8A%A4%ED%8C%8C%EC%9D%B4%EC%96%B4&amp;action=edit&amp;section=3" title="부분 편집: 상징">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>책에서 나오는 "N" 이나 "NF" 와 같은 상징들은, <a href="/wiki/%EC%98%81%EA%B5%AD" title="영국">영국</a> 내의 실존 정당인 <a href="/wiki/%EA%B5%AD%EB%AF%BC%EC%A0%84%EC%84%A0_(%EC%98%81%EA%B5%AD)" title="국민전선 (영국)">국민전선</a> (National Front, NF) 을 풍자한 것이며, <a href="/wiki/%EA%B8%B0%EB%8F%85%EA%B5%90" title="기독교">기독교</a> 상징이나 <a href="/wiki/%EB%B6%88%EC%82%AC%EC%A1%B0" title="불사조">불사조</a>의 상징, <a href="/wiki/%EC%98%81%EA%B5%AD%EC%9D%98_%EA%B5%AD%EA%B8%B0" title="영국의 국기">영국의 국기</a>색인 <a href="/wiki/%EB%B9%A8%EA%B0%95" title="빨강">빨강</a>, <a href="/wiki/%ED%95%98%EC%96%91" title="하양">하양</a>, <a href="/wiki/%ED%8C%8C%EB%9E%91" title="파랑">파랑</a>이 상징으로 쓰이기도 하였다.</p>
<p>영화에서는, 노스파이어의 상징이 <a href="/wiki/%EC%98%81%EA%B5%AD%EC%9D%98_%EA%B5%AD%EA%B8%B0" title="영국의 국기">영국의 국기</a>나, <a href="/wiki/%EC%8A%A4%ED%82%A8%ED%97%A4%EB%93%9C" title="스킨헤드">스킨헤드</a> 들의 상징으로도 쓰이는 <a href="/wiki/%EB%A1%9C%EB%A0%8C_%EC%8B%AD%EC%9E%90" title="로렌 십자">로렌 십자</a>가 사용되었으며, 이는 작 내에서 깃발, 경찰 배지, <a class="mw-disambig" href="/wiki/%EA%B5%AD%EC%9E%A5" title="국장">국장</a>, <a class="mw-redirect" href="/wiki/%ED%83%B1%ED%81%AC" title="탱크">탱크</a>, 군 <a class="mw-redirect" href="/wiki/%EB%B2%A0%EB%A0%88%EB%AA%A8" title="베레모">베레모</a> 배지에서도 종종 보이기도 하였다. 또한, 작 내에서 <a href="/wiki/%EB%A1%9C%EB%A0%8C_%EC%8B%AD%EC%9E%90" title="로렌 십자">로렌 십자</a>는 <a class="mw-redirect" href="/wiki/%EA%B2%80%EC%A0%95%EC%83%89" title="검정색">검정색</a> 바탕의 <a class="mw-redirect" href="/wiki/%EB%B9%A8%EA%B0%95%EC%83%89" title="빨강색">빨강색</a> 십자로 사용되었다.</p>
