<p><b>1998년 FIFA 월드컵 결승전</b> (<span style="font-size: smaller;"><a href="/wiki/%ED%94%84%EB%9E%91%EC%8A%A4%EC%96%B4" title="프랑스어">프랑스어</a>:</span> <span lang="fr" xml:lang="fr">Finale de la Coupe du monde de football de 1998</span>) 은 1998년 7월 12일, <a href="/wiki/%EC%83%9D%EB%93%9C%EB%8B%88" title="생드니">생드니</a>의 <a href="/wiki/%EC%8A%A4%ED%83%80%EB%93%9C_%EB%93%9C_%ED%94%84%EB%9E%91%EC%8A%A4" title="스타드 드 프랑스">스타드 드 프랑스</a>에서 <a href="/wiki/1998%EB%85%84_FIFA_%EC%9B%94%EB%93%9C%EC%BB%B5" title="1998년 FIFA 월드컵">1998년 FIFA 월드컵</a> 우승팀을 가리기 위해 열린 <a href="/wiki/%EC%B6%95%EA%B5%AC" title="축구">축구</a> 경기이다. 결승전은 <a href="/wiki/1994%EB%85%84_FIFA_%EC%9B%94%EB%93%9C%EC%BB%B5" title="1994년 FIFA 월드컵">1994년</a>에 <a href="/wiki/FIFA_%EC%9B%94%EB%93%9C%EC%BB%B5" title="FIFA 월드컵">FIFA 월드컵</a> 우승을 차지한 <a href="/wiki/%EB%B8%8C%EB%9D%BC%EC%A7%88_%EC%B6%95%EA%B5%AC_%EA%B5%AD%EA%B0%80%EB%8C%80%ED%91%9C%ED%8C%80" title="브라질 축구 국가대표팀">브라질</a><sup class="reference" id="cite_ref-2"><a href="#cite_note-2">[2]</a></sup> 과 개최국 <a href="/wiki/%ED%94%84%EB%9E%91%EC%8A%A4_%EC%B6%95%EA%B5%AC_%EA%B5%AD%EA%B0%80%EB%8C%80%ED%91%9C%ED%8C%80" title="프랑스 축구 국가대표팀">프랑스</a>의 대결로, 후자의 팀은 대회 결승전에 처음으로 올랐다.<sup class="reference" id="cite_ref-3"><a href="#cite_note-3">[3]</a></sup> 프랑스는 3-0 완승을 거두며 사상 첫 FIFA 월드컵을 거두었고, <a href="/wiki/%ED%98%81%EB%AA%85_%EA%B8%B0%EB%85%90%EC%9D%BC" title="혁명 기념일">혁명 기념일</a> 이틀 전에 이룩한 업적이어서 프랑스는 이 우승에 더 큰 의의를 두게 되었다.<sup class="reference" id="cite_ref-France_98_BBC_4-0"><a href="#cite_note-France_98_BBC-4">[4]</a></sup><sup class="reference" id="cite_ref-Rewind_5-0"><a href="#cite_note-Rewind-5">[5]</a></sup> <a href="/wiki/%EC%A7%80%EB%84%A4%EB%94%98_%EC%A7%80%EB%8B%A8" title="지네딘 지단">지네딘 지단</a>이 이 경기의 MVP로 선정되었는데, 그는 전반에 2골을 기록하였고, 프랑스는 <a href="/wiki/%EC%97%90%EB%A7%88%EB%89%98%EC%97%98_%ED%94%84%ED%8B%B0" title="에마뉘엘 프티">에마뉘엘 프티</a>의 3번째 골을 막판에 추가하였다. 관중은 80,000명 가량이 웅집하였다.<sup class="reference" id="cite_ref-France_98_BBC_4-1"><a href="#cite_note-France_98_BBC-4">[4]</a></sup></p>
<p>양팀 모두 고전 끝에 결승 무대에 도달하였다. 브라질은 <a href="/wiki/1998%EB%85%84_FIFA_%EC%9B%94%EB%93%9C%EC%BB%B5#.EC.A1.B0.EB.B3.84_.EB.A6.AC.EA.B7.B8" title="1998년 FIFA 월드컵">조별 리그</a> 3경기에서 6점을 적립하였는데, 이때 1패는 <a href="/wiki/%EB%85%B8%EB%A5%B4%EC%9B%A8%EC%9D%B4_%EC%B6%95%EA%B5%AC_%EA%B5%AD%EA%B0%80%EB%8C%80%ED%91%9C%ED%8C%80" title="노르웨이 축구 국가대표팀">노르웨이</a>에 당한 것이었다. 16강에서 <a href="/wiki/%EC%B9%A0%EB%A0%88_%EC%B6%95%EA%B5%AC_%EA%B5%AD%EA%B0%80%EB%8C%80%ED%91%9C%ED%8C%80" title="칠레 축구 국가대표팀">칠레</a>를 4-1, 8강에서 <a href="/wiki/%EB%8D%B4%EB%A7%88%ED%81%AC_%EC%B6%95%EA%B5%AC_%EA%B5%AD%EA%B0%80%EB%8C%80%ED%91%9C%ED%8C%80" title="덴마크 축구 국가대표팀">덴마크</a>를 3-2로 꺾은 브라질은 <a href="/wiki/%EB%84%A4%EB%8D%9C%EB%9E%80%EB%93%9C_%EC%B6%95%EA%B5%AC_%EA%B5%AD%EA%B0%80%EB%8C%80%ED%91%9C%ED%8C%80" title="네덜란드 축구 국가대표팀">네덜란드</a>와의 준결승전에서 승부차기 끝에 승리해 결승 진출을 확정지었다. 한편 프랑스는 조별 리그에서 3경기를 모두 이기며 순항하였고, <a href="/wiki/%ED%8C%8C%EB%9D%BC%EA%B3%BC%EC%9D%B4_%EC%B6%95%EA%B5%AC_%EA%B5%AD%EA%B0%80%EB%8C%80%ED%91%9C%ED%8C%80" title="파라과이 축구 국가대표팀">파라과이</a>와의 16강전에서 골든골로 신승을 거두었다. 8강에서는 <a href="/wiki/%EC%9D%B4%ED%83%88%EB%A6%AC%EC%95%84_%EC%B6%95%EA%B5%AC_%EA%B5%AD%EA%B0%80%EB%8C%80%ED%91%9C%ED%8C%80" title="이탈리아 축구 국가대표팀">이탈리아</a>를 상대로 승부차기 끝에 이겼고, 준결승전에서는 처녀 출전국 <a href="/wiki/%ED%81%AC%EB%A1%9C%EC%95%84%ED%8B%B0%EC%95%84_%EC%B6%95%EA%B5%AC_%EA%B5%AD%EA%B0%80%EB%8C%80%ED%91%9C%ED%8C%80" title="크로아티아 축구 국가대표팀">크로아티아</a>를 따돌리고 결승전 진출을 이룩하였다.</p>
<p>결승전 직전, 브라질 스트라이커 <a href="/wiki/%ED%98%B8%EB%82%98%EC%9A%B0%EB%91%90" title="호나우두">호나우두</a>가 <a class="mw-disambig" href="/wiki/%EB%B0%9C%EC%9E%91" title="발작">발작</a>을 일으킴에 따라 그의 몸상태에 의혹이 제기되었다.<sup class="reference" id="cite_ref-Mystery_6-0"><a href="#cite_note-Mystery-6">[6]</a></sup><sup class="reference" id="cite_ref-7"><a href="#cite_note-7">[7]</a></sup> 본래 결승전 출전 명단에서 제외되었으나, 그의 몸상태에도 불구하고, 경기 72분전에 그는 결승전 선발 명단에 포함되었다.<sup class="reference" id="cite_ref-France_98_BBC_4-2"><a href="#cite_note-France_98_BBC-4">[4]</a></sup> 본 경기에서, 그는 <a href="/wiki/%ED%8C%8C%EB%B9%84%EC%95%B5_%EB%B0%94%EB%A5%B4%ED%85%8C%EC%A6%88" title="파비앵 바르테즈">파비앵 바르테즈</a> 프랑스 수문장과의 충돌로 부상을 당하였다. 비록 호나우두의 출전을 강행한 결정으로 효과를 보지 못하였으나, 그가 대회 기간동안 주축 선수로 활동해 4번 득점하고 3번의 득점 기회를 창출한 것을 한 것이 신빙성 있는 결정으로 증명되고 있다.<sup class="reference" id="cite_ref-8"><a href="#cite_note-8">[8]</a></sup></p>
<p>FIFA 월드컵 우승 후, 프랑스는 <a href="/wiki/%EB%84%A4%EB%8D%9C%EB%9E%80%EB%93%9C" title="네덜란드">네덜란드</a>와 <a href="/wiki/%EB%B2%A8%EA%B8%B0%EC%97%90" title="벨기에">벨기에</a>에서 열린 <a href="/wiki/UEFA_%EC%9C%A0%EB%A1%9C_2000" title="UEFA 유로 2000">UEFA 유로 2000</a>의 <a class="new" href="/w/index.php?title=UEFA_%EC%9C%A0%EB%A1%9C_2000_%EC%98%88%EC%84%A0%EC%A0%84&amp;action=edit&amp;redlink=1" title="UEFA 유로 2000 예선전 (없는 문서)">본선에 진출</a>해 <a href="/wiki/UEFA_%EC%9C%A0%EB%A1%9C_2000_%EA%B2%B0%EC%8A%B9%EC%A0%84" title="UEFA 유로 2000 결승전">우승</a>을 거두었다.<sup class="reference" id="cite_ref-9"><a href="#cite_note-9">[9]</a></sup> 브라질은 <a href="/wiki/1999%EB%85%84_%EC%BD%94%ED%8C%8C_%EC%95%84%EB%A9%94%EB%A6%AC%EC%B9%B4" title="1999년 코파 아메리카">1999년</a>에 <a href="/wiki/%EC%BD%94%ED%8C%8C_%EC%95%84%EB%A9%94%EB%A6%AC%EC%B9%B4" title="코파 아메리카">코파 아메리카</a> 우승을 차지하였고, <a href="/wiki/%EB%8C%80%ED%95%9C%EB%AF%BC%EA%B5%AD" title="대한민국">대한민국</a>과 <a href="/wiki/%EC%9D%BC%EB%B3%B8" title="일본">일본</a>에서 <a href="/wiki/2002%EB%85%84_FIFA_%EC%9B%94%EB%93%9C%EC%BB%B5" title="2002년 FIFA 월드컵">2002년</a>에 열린 다음 FIFA 월드컵에서 정상에 올랐다.<sup class="reference" id="cite_ref-10"><a href="#cite_note-10">[10]</a></sup><sup class="reference" id="cite_ref-11"><a href="#cite_note-11">[11]</a></sup> 호나우두는 그 후로 FIFA 월드컵 최다 득점 신기록을 깼으나, 이 기록은 2014년 FIFA 월드컵에 <a href="/wiki/%EB%AF%B8%EB%A1%9C%EC%8A%AC%EB%9D%BC%ED%94%84_%ED%81%B4%EB%A1%9C%EC%A0%9C" title="미로슬라프 클로제">미로슬라프 클로제</a>가 또다시 경신하였다.<sup class="reference" id="cite_ref-12"><a href="#cite_note-12">[12]</a></sup></p>

<h3><span class="mw-headline" id=".EB.B8.8C.EB.9D.BC.EC.A7.88">브라질</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=1998%EB%85%84_FIFA_%EC%9B%94%EB%93%9C%EC%BB%B5_%EA%B2%B0%EC%8A%B9%EC%A0%84&amp;action=edit&amp;section=2" title="부분 편집: 브라질">원본 편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>브라질은 <a href="/wiki/%EC%8A%A4%EC%BD%94%ED%8B%80%EB%9E%9C%EB%93%9C_%EC%B6%95%EA%B5%AC_%EA%B5%AD%EA%B0%80%EB%8C%80%ED%91%9C%ED%8C%80" title="스코틀랜드 축구 국가대표팀">스코틀랜드</a>, <a href="/wiki/%EB%AA%A8%EB%A1%9C%EC%BD%94_%EC%B6%95%EA%B5%AC_%EA%B5%AD%EA%B0%80%EB%8C%80%ED%91%9C%ED%8C%80" title="모로코 축구 국가대표팀">모로코</a>, 그리고 <a href="/wiki/%EB%85%B8%EB%A5%B4%EC%9B%A8%EC%9D%B4_%EC%B6%95%EA%B5%AC_%EA%B5%AD%EA%B0%80%EB%8C%80%ED%91%9C%ED%8C%80" title="노르웨이 축구 국가대표팀">노르웨이</a>와 함께 <a href="/wiki/1998%EB%85%84_FIFA_%EC%9B%94%EB%93%9C%EC%BB%B5_A%EC%A1%B0" title="1998년 FIFA 월드컵 A조">A조</a>에 편성되었다. 브라질은 스코틀랜드와 모로코를 상대로 각각 2-1, 3-0으로 이겨 16강 진출을 확정지었으나, 노르웨이와의 조별 리그 최종전에서 1-2로 역전패하였다.</p>
<p>브라질의 16강 상대는 B조 2위를 차지한 <a href="/wiki/%EC%B9%A0%EB%A0%88_%EC%B6%95%EA%B5%AC_%EA%B5%AD%EA%B0%80%EB%8C%80%ED%91%9C%ED%8C%80" title="칠레 축구 국가대표팀">칠레</a>로, <a href="/wiki/%ED%98%B8%EB%82%98%EC%9A%B0%EB%91%90" title="호나우두">호나우두</a>와 <a class="new" href="/w/index.php?title=%EC%84%B8%EC%9E%90%EB%A5%B4_%EC%83%81%ED%8C%8C%EC%9D%B4%EC%9A%B0&amp;action=edit&amp;redlink=1" title="세자르 상파이우 (없는 문서)">세자르 상파이우</a>가 각각 2골씩 득점해 4-1 대승에 일조하였다. 8강전의 브라질 상대는 <a href="/wiki/%EB%8D%B4%EB%A7%88%ED%81%AC_%EC%B6%95%EA%B5%AC_%EA%B5%AD%EA%B0%80%EB%8C%80%ED%91%9C%ED%8C%80" title="덴마크 축구 국가대표팀">덴마크</a>로, 앞서 <a href="/wiki/%EB%82%98%EC%9D%B4%EC%A7%80%EB%A6%AC%EC%95%84_%EC%B6%95%EA%B5%AC_%EA%B5%AD%EA%B0%80%EB%8C%80%ED%91%9C%ED%8C%80" title="나이지리아 축구 국가대표팀">나이지리아</a>와의 16강전에서 4-1로 이긴 팀이었으나, 격전 끝에 3-2 승리를 쟁취하였다. <a href="/wiki/%EB%A7%88%EB%A5%B4%ED%8B%B4_%EC%98%88%EB%A5%B4%EA%B2%90%EC%84%BC" title="마르틴 예르겐센">마르틴 예르겐센</a>에게 0-1 선제골을 2분만에 내주었으나, 브라질은 자신의 페이스대로 경기를 끌고가 11분에 <a href="/wiki/%EB%B2%A0%EB%B2%A0%ED%88%AC" title="베베투">베베투</a>, 27분에 <a href="/wiki/%ED%9E%88%EB%B0%94%EC%9A%B0%EB%91%90" title="히바우두">히바우두</a>가 연달아 득점을 성공시켰다. <a href="/wiki/%EB%B8%8C%EB%A6%AC%EC%95%88_%EB%9D%BC%EC%9A%B0%EB%93%9C%EB%A3%A8%ED%94%84" title="브리안 라우드루프">브리안 라우드루프</a>는 50분경에 덴마크의 동점골을 넣었으나, 10분 후에 히바우두가 결승골로 경기의 종지부를 찍었다.</p>
<p>준결승전에서 브라질은 <a href="/wiki/%EB%A7%88%EB%A5%B4%EC%84%B8%EC%9C%A0" title="마르세유">마르세유</a>에서 <a href="/wiki/%EB%84%A4%EB%8D%9C%EB%9E%80%EB%93%9C_%EC%B6%95%EA%B5%AC_%EA%B5%AD%EA%B0%80%EB%8C%80%ED%91%9C%ED%8C%80" title="네덜란드 축구 국가대표팀">네덜란드</a>를 상대하였다. 경기는 정규시간 끝에 1-1로 끝났는데, <a href="/wiki/%ED%98%B8%EB%82%98%EC%9A%B0%EB%91%90" title="호나우두">호나우두</a>가 후반 시작과 함께 선제골을 넣은 뒤, <a class="mw-redirect" href="/wiki/%ED%8C%8C%ED%8A%B8%EB%A6%AC%ED%81%AC_%ED%81%B4%EB%9D%BC%EC%9C%84%EB%B2%84%EB%A5%B4%ED%8A%B8" title="파트리크 클라위버르트">파트리크 클라위버르트</a>의 87분 동점골로 경기가 연장전으로 이어졌고, 120분 안에 승부가 나지 않았다. 경기는 승부차기를 통해 승자가 가려졌는데, 브라질이 4-2로 이기면서 FIFA 월드컵 2대회 연속 결승 진출에 성공하였다.</p>
<h3><span class="mw-headline" id=".ED.94.84.EB.9E.91.EC.8A.A4">프랑스</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=1998%EB%85%84_FIFA_%EC%9B%94%EB%93%9C%EC%BB%B5_%EA%B2%B0%EC%8A%B9%EC%A0%84&amp;action=edit&amp;section=3" title="부분 편집: 프랑스">원본 편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>프랑스는 <a href="/wiki/%EB%82%A8%EC%95%84%ED%94%84%EB%A6%AC%EC%B9%B4_%EA%B3%B5%ED%99%94%EA%B5%AD_%EC%B6%95%EA%B5%AC_%EA%B5%AD%EA%B0%80%EB%8C%80%ED%91%9C%ED%8C%80" title="남아프리카 공화국 축구 국가대표팀">남아프리카 공화국</a>, <a href="/wiki/%EC%82%AC%EC%9A%B0%EB%94%94%EC%95%84%EB%9D%BC%EB%B9%84%EC%95%84_%EC%B6%95%EA%B5%AC_%EA%B5%AD%EA%B0%80%EB%8C%80%ED%91%9C%ED%8C%80" title="사우디아라비아 축구 국가대표팀">사우디아라비아</a>, 그리고 <a href="/wiki/%EB%8D%B4%EB%A7%88%ED%81%AC_%EC%B6%95%EA%B5%AC_%EA%B5%AD%EA%B0%80%EB%8C%80%ED%91%9C%ED%8C%80" title="덴마크 축구 국가대표팀">덴마크</a>와 함께 <a href="/wiki/1998%EB%85%84_FIFA_%EC%9B%94%EB%93%9C%EC%BB%B5_C%EC%A1%B0" title="1998년 FIFA 월드컵 C조">C조</a>에 편성되었다. 프랑스는 남아프리카 공화국을 상대로 3-0 대승을 거두며 가볍게 대회를 시작하였고, 이어지는 사우디아라비아전에서 4-0의 대승을 일구어냈다. 프랑스는 덴마크와의 조별 리그 최종전에서 <a href="/wiki/%EC%9C%A0%EB%A6%AC_%EC%A1%B0%EB%A5%B4%EC%B9%B4%EC%97%90%ED%94%84" title="유리 조르카에프">유리 조르카에프</a>와 <a href="/wiki/%EC%97%90%EB%A7%88%EB%89%98%EC%97%98_%ED%94%84%ED%8B%B0" title="에마뉘엘 프티">에마뉘엘 프티</a>의 연속골을 통해 2-1로 이겨 조 1위를 확정지었다.</p>
<p>프랑스의 16강전 상대는 D조 2위 <a href="/wiki/%ED%8C%8C%EB%9D%BC%EA%B3%BC%EC%9D%B4_%EC%B6%95%EA%B5%AC_%EA%B5%AD%EA%B0%80%EB%8C%80%ED%91%9C%ED%8C%80" title="파라과이 축구 국가대표팀">파라과이</a>였다. 프랑스는 접전 끝에 연장전에 터진 <a href="/wiki/%EB%A1%9C%EB%9E%91_%EB%B8%94%EB%9E%91" title="로랑 블랑">로랑 블랑</a>의 골든골로 1-0 승리를 쟁취하는데 성공했다. 프랑스는 8강전에서 <a href="/wiki/%EC%9D%B4%ED%83%88%EB%A6%AC%EC%95%84_%EC%B6%95%EA%B5%AC_%EA%B5%AD%EA%B0%80%EB%8C%80%ED%91%9C%ED%8C%80" title="이탈리아 축구 국가대표팀">이탈리아</a>를 상대하였는데, 이들은 <a href="/wiki/%EB%85%B8%EB%A5%B4%EC%9B%A8%EC%9D%B4_%EC%B6%95%EA%B5%AC_%EA%B5%AD%EA%B0%80%EB%8C%80%ED%91%9C%ED%8C%80" title="노르웨이 축구 국가대표팀">노르웨이</a>전의 1-0 신승을 통해 올라왔다. 두 팀은 연장전까지 지속된 120분간의 혈전 끝에 0-0으로 비겼고, 이탈리아의 <a href="/wiki/%EB%A3%A8%EC%9D%B4%EC%A7%80_%EB%94%94_%EB%B9%84%EC%95%84%EC%A1%B0" title="루이지 디 비아조">루이지 디 비아조</a>가 승부차기 키커로 나서 크로스바를 강타한 덕에 프랑스가 승부차기에서 4-3으로 이겼다.</p>
<p>준결승전에서, 프랑스는 이 대회에서 돌풍을 일으킨 <a href="/wiki/%ED%81%AC%EB%A1%9C%EC%95%84%ED%8B%B0%EC%95%84_%EC%B6%95%EA%B5%AC_%EA%B5%AD%EA%B0%80%EB%8C%80%ED%91%9C%ED%8C%80" title="크로아티아 축구 국가대표팀">크로아티아</a>를 만났다. 득점 없는 전반전 후, 프랑스는 후반전 시작과 함께 터진 크로아티아의 <a href="/wiki/%EB%8B%A4%EB%B3%B4%EB%A5%B4_%EC%8A%88%EC%BC%80%EB%A5%B4" title="다보르 슈케르">다보르 슈케르</a> 대회 5호골과 함께 리드를 헌납하였다. 프랑스는 <a href="/wiki/%EB%A6%B4%EB%A6%AC%EC%95%99_%ED%8A%80%EB%9E%91" title="릴리앙 튀랑">릴리앙 튀랑</a>의 동점골이자 그의 첫 국제대회 골로 응수하였다. 튀랑은 그로부터 20분 후에 또다시 한골을 추가하여 역전승을 거둠으로써 프랑스의 사상 첫 FIFA 월드컵 결승전 진출을 견인하였다. 그러나, 경기는 <a href="/wiki/%EB%A1%9C%EB%9E%91_%EB%B8%94%EB%9E%91" title="로랑 블랑">로랑 블랑</a>이 크로아티아의 <a href="/wiki/%EC%8A%AC%EB%9D%BC%EB%B2%A4_%EB%B9%8C%EB%A6%AC%EC%B9%98" title="슬라벤 빌리치">슬라벤 빌리치</a>와 다툼으로 퇴장당해 논란을 야기하였다. 빌리치는 무릎쪽으로 굴러 고통스러운 표정을 지었다. 그러나, 비디오에서는 선수들간에 큰 충돌이 일어나지 않은 것으로 보였다. 블랑은 퇴장으로 인해 결승전에 결장하게 되었다.</p>
<h2><span class="mw-headline" id=".EA.B2.BD.EA.B8.B0">경기</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=1998%EB%85%84_FIFA_%EC%9B%94%EB%93%9C%EC%BB%B5_%EA%B2%B0%EC%8A%B9%EC%A0%84&amp;action=edit&amp;section=4" title="부분 편집: 경기">원본 편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<h3><span class="mw-headline" id=".EC.9A.94.EC.95.BD">요약</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=1998%EB%85%84_FIFA_%EC%9B%94%EB%93%9C%EC%BB%B5_%EA%B2%B0%EC%8A%B9%EC%A0%84&amp;action=edit&amp;section=5" title="부분 편집: 요약">원본 편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>경기 전, 브라질의 주축 스트라이커 <a href="/wiki/%ED%98%B8%EB%82%98%EC%9A%B0%EB%91%90" title="호나우두">호나우두</a>의 몸상태에 의혹이 제기되었고, 몇몇 보고내용에 따르면 그는 경기 전에 건강상 문제가 있는 것으로 나타났다.<sup class="reference" id="cite_ref-Mystery_6-1"><a href="#cite_note-Mystery-6">[6]</a></sup></p>
<p><a href="/wiki/%EC%A7%80%EB%84%A4%EB%94%98_%EC%A7%80%EB%8B%A8" title="지네딘 지단">지네딘 지단</a>은 전반 27분에 에마뉘엘차 프티가 오른쪽 코너플래그에서 올린 코너킥을 헤딩으로 연결해 프랑스에게 리드를 가져다 주었다. 몇분 후, 호나우두는 <a href="/wiki/%EB%91%A5%EA%B0%80" title="둥가">둥가</a>의 롱볼을 받아 득점 찬스를 얻어냈으나, 쇄도하는 <a href="/wiki/%ED%8C%8C%EB%B9%84%EC%95%B5_%EB%B0%94%EB%A5%B4%ED%85%8C%EC%A6%88" title="파비앵 바르테즈">파비앵 바르테즈</a>를 보지 못하였고, 결국 브라질 스트라이커는 그와 충돌하였다. 두 팀은 선수단 의료진의 도움이 필요했으나, 얼마 지나지 않아 회복하였다. 브라질의 플레이메이커 <a href="/wiki/%EB%A0%88%EC%98%A4%EB%82%98%EB%A5%B4%EB%91%90_%EC%95%84%EB%9D%BC%EC%9A%B0%EC%A3%BC" title="레오나르두 아라우주">레오나르두</a>와 <a href="/wiki/%ED%9E%88%EB%B0%94%EC%9A%B0%EB%91%90" title="히바우두">히바우두</a>는 <a href="/wiki/%EB%94%94%EB%94%94%EC%97%90_%EB%8D%B0%EC%83%B9" title="디디에 데샹">디디에 데샹</a>과 <a href="/wiki/%ED%81%AC%EB%A6%AC%EC%8A%A4%ED%8B%B0%EC%95%99_%EC%B9%B4%EB%9E%91%EB%B5%88" title="크리스티앙 카랑뵈">크리스티앙 카랑뵈</a>에 막혀 침묵하였다. 전반 종료를 앞두고, 지단은 왼쪽 코너플래그에서 날아온 코너킥을 비슷한 패턴으로 추가골을 성공해 리드를 두골차로 늘렸다. 후반전에 호나우두는 점수차를 1골차로 줄일 기회를 잡았다. 공은 페널티 박스 안으로 배급되었으나, 그의 슛은 바르테즈 팔 사이로 날아갔다. 미드필더 <a href="/wiki/%EC%97%90%EB%A7%88%EB%89%98%EC%97%98_%ED%94%84%ED%8B%B0" title="에마뉘엘 프티">에마뉘엘 프티</a>는 90분에 <a href="/wiki/%EC%95%84%EC%8A%A4%EB%84%90_FC" title="아스널 FC">아스널</a> 팀동료인 <a href="/wiki/%ED%8C%8C%ED%8A%B8%EB%A6%AC%ED%81%AC_%EB%B9%84%EC%97%90%EB%9D%BC" title="파트리크 비에라">파트리크 비에라</a>의 전진패스를 받아 골망 아래쪽으로 집어넣어 쐐기골을 성공시켰고, 모든 의혹을 잠재울 승리를 선사하였다. 프랑스는 <a href="/wiki/%EB%A7%88%EB%A5%B4%EC%85%80_%EB%93%9C%EC%82%AC%EC%9D%B4" title="마르셀 드사이">마르셀 드사이</a>의 <a href="/wiki/%ED%8C%8C%EC%9A%B8_(%EC%B6%95%EA%B5%AC)" title="파울 (축구)">반칙</a>으로 인한 경고 누적으로 퇴장당함에 따라 막판 20분동안 10명으로 경기에 임하였다.<sup class="reference" id="cite_ref-13"><a href="#cite_note-13">[13]</a></sup></p>
