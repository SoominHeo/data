<p><b>노예주</b>(奴隸州, slave state)는 <a href="/wiki/%EB%AF%B8%EA%B5%AD_%EB%82%A8%EB%B6%81_%EC%A0%84%EC%9F%81" title="미국 남북 전쟁">미국 남북 전쟁</a> 이전에 있었던 <a class="mw-redirect" href="/wiki/%EB%85%B8%EC%98%88_%EC%A0%9C%EB%8F%84" title="노예 제도">노예 제도</a>가 합법적이었던 주를 말한다. 이 개념과 반대로 <b>자유주</b>(自由州, free state) 또는 <b>해방주</b>(解放州)는 노예 제도가 금지되었거나 그 전에 폐지되었던 주를 말한다. 노예제도는 <a class="mw-redirect mw-disambig" href="/wiki/%EB%82%A8%EB%B6%81%EC%A0%84%EC%9F%81" title="남북전쟁">남북전쟁</a>의 원인이 되었으며, <a href="/wiki/1865%EB%85%84" title="1865년">1865년</a> ‘<a class="mw-redirect" href="/wiki/%EB%AF%B8%EA%B5%AD_%ED%97%8C%EB%B2%95_%EC%88%98%EC%A0%95_%EC%A0%9C13%EC%A1%B0" title="미국 헌법 수정 제13조">미국 헌법 수정 제13조</a>’에 의해 폐지되었다.</p>

<h2><span class="mw-headline" id=".EB.B0.B0.EA.B2.BD">배경</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%85%B8%EC%98%88%EC%A3%BC%EC%99%80_%EC%9E%90%EC%9C%A0%EC%A3%BC&amp;action=edit&amp;section=1" title="부분 편집: 배경">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p><a href="/wiki/%EB%A7%A4%EC%82%AC%EC%B6%94%EC%84%B8%EC%B8%A0_%EC%A3%BC" title="매사추세츠 주">매사추세츠</a>, <a href="/wiki/%EB%89%B4%EC%9A%95_%EC%A3%BC" title="뉴욕 주">뉴욕</a>, <a class="mw-redirect" href="/wiki/%ED%8E%9C%EC%8B%A4%EB%B2%A0%EB%8B%88%EC%95%84_%EC%A3%BC" title="펜실베니아 주">펜실바니아</a>, <a href="/wiki/%EB%89%B4%EC%A0%80%EC%A7%80_%EC%A3%BC" title="뉴저지 주">뉴저지</a>를 포함한 <a href="/wiki/%EB%AF%B8%EA%B5%AD_%EB%B6%81%EB%8F%99%EB%B6%80" title="미국 북동부">미국 북동부</a>와 중부 대서양 연안 주들은 17세기와 18세기 심지어 19세기 까지도 노예제도가 합법화된 곳이었다. 남북전쟁 이전의 세대에서 일련의 조치들로 인해 노예가 해방되기 시작했다.</p>
<p>미국 내에서 <a class="mw-redirect" href="/wiki/%EB%85%B8%EC%98%88%EC%A0%9C%EB%8F%84" title="노예제도">노예제도</a>를 완전히 철폐한 최초의 지역은 미국 북서부였다. 이러한 조치는 미국 헌법이 비준되기도 전에 <a href="/wiki/1787%EB%85%84" title="1787년">1787년</a> 북서부 법령으로 해방이 예정된 것이었다. <a href="/wiki/%EC%98%A4%ED%95%98%EC%9D%B4%EC%98%A4_%EC%A3%BC" title="오하이오 주">오하이오 주</a>, <a href="/wiki/%EC%9D%B8%EB%94%94%EC%95%A0%EB%82%98_%EC%A3%BC" title="인디애나 주">인디애나 주</a>, <a href="/wiki/%EC%9D%BC%EB%A6%AC%EB%85%B8%EC%9D%B4_%EC%A3%BC" title="일리노이 주">일리노이 주</a>, <a href="/wiki/%EC%9C%84%EC%8A%A4%EC%BD%98%EC%8B%A0_%EC%A3%BC" title="위스콘신 주">위스콘신 주</a>, <a href="/wiki/%EB%AF%B8%EB%84%A4%EC%86%8C%ED%83%80_%EC%A3%BC" title="미네소타 주">미네소타 주</a>를 포함한 이 지역에는 뉴잉글랜드 출신들이나 미국독립전쟁 참전 용사들이 주로 거주하는 곳이었다. 남쪽의 <a href="/wiki/%EC%98%A4%ED%95%98%EC%9D%B4%EC%98%A4_%EA%B0%95" title="오하이오 강">오하이오 강</a>을 기준으로 이 지역에 노예가 완전히 해방되었기 때문에 서부로 노예제도가 합법화된 곳에 영향을 미치고 있었다. 그리하여 ‘자유주’의 개념에 반대되는 ‘노예주’라는 개념이 확립되게 되었다.</p>

<h2><span class="mw-headline" id=".EB.85.B8.EC.98.88_.EC.A3.BC.EC.9D.98_.EC.A2.85.EB.A7.90">노예 주의 종말</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%85%B8%EC%98%88%EC%A3%BC%EC%99%80_%EC%9E%90%EC%9C%A0%EC%A3%BC&amp;action=edit&amp;section=3" title="부분 편집: 노예 주의 종말">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p><a class="mw-redirect" href="/wiki/%EB%A7%A4%EB%A6%B4%EB%9E%9C%EB%93%9C_%EC%A3%BC" title="매릴랜드 주">매릴랜드 주</a>,<sup class="reference" id="cite_ref-1"><a href="#cite_note-1">[1]</a></sup> <a href="/wiki/%EB%AF%B8%EC%A3%BC%EB%A6%AC_%EC%A3%BC" title="미주리 주">미주리 주</a>,<sup class="reference" id="cite_ref-2"><a href="#cite_note-2">[2]</a></sup> <a class="mw-redirect" href="/wiki/%ED%85%8C%EB%8B%88%EC%8B%9C_%EC%A3%BC" title="테니시 주">테니시 주</a>,<sup class="reference" id="cite_ref-3"><a href="#cite_note-3">[3]</a></sup> <a class="mw-redirect" href="/wiki/%EC%9B%A8%EC%8A%A4%ED%8A%B8_%EB%B2%84%EC%A7%80%EB%8B%88%EC%95%84" title="웨스트 버지니아">웨스트 버지니아</a>의 새로운 주,<sup class="reference" id="cite_ref-4"><a href="#cite_note-4">[4]</a></sup> 그리고 <a href="/wiki/%EC%9B%8C%EC%8B%B1%ED%84%B4_D.C." title="워싱턴 D.C.">컬럼비아</a>는 남북전쟁 종전 이전에 노예제도를 금지시켰다. 그러나 <a href="/wiki/%EB%8D%B8%EB%9D%BC%EC%9B%A8%EC%96%B4_%EC%A3%BC" title="델라웨어 주">델라웨어 주</a>,<sup class="reference" id="cite_ref-5"><a href="#cite_note-5">[5]</a></sup><a href="/wiki/%EC%BC%84%ED%84%B0%ED%82%A4_%EC%A3%BC" title="켄터키 주">켄터키 주</a>,<sup class="reference" id="cite_ref-6"><a href="#cite_note-6">[6]</a></sup> 그리고 대부분의 <a href="/wiki/%EB%AF%B8%EA%B5%AD_%EB%82%A8%EB%B6%80" title="미국 남부">남부 주</a>들에게 노예제도는 <a href="/wiki/1865%EB%85%84" title="1865년">1865년</a> 12월 미국 전역에 미국수정헌법 제13조가 적용되어 노예제도를 폐지할 때까지 지속되었으며, 그로 인해 자유 주와 노예 주라는 구분은 없어지게 되었다. <a class="mw-redirect" href="/wiki/%EB%AF%B8%EA%B5%AD_%ED%97%8C%EB%B2%95_%EC%88%98%EC%A0%95_%EC%A0%9C13%EC%A1%B0" title="미국 헌법 수정 제13조">미국 헌법 수정 제13조</a>의 비준은 이러한 지역적인 통치를 돌려주는 조건이었다.</p>
