<p><b>노요리 료지</b>(<span style="font-size: smaller;"><a href="/wiki/%EC%9D%BC%EB%B3%B8%EC%96%B4" title="일본어">일본어</a>:</span> <span lang="ja"><span lang="ja"><ruby><rb>野依 良治</rb> <rp>(</rp><rt>のより りょうじ</rt><rp>)</rp></ruby></span></span>, <a href="/wiki/1938%EB%85%84" title="1938년">1938년</a> <a href="/wiki/9%EC%9B%94_3%EC%9D%BC" title="9월 3일">9월 3일</a><sup class="reference" id="cite_ref-kahaku_1-0"><a href="#cite_note-kahaku-1">[1]</a></sup> ~ )는 <a href="/wiki/%EC%9D%BC%EB%B3%B8" title="일본">일본</a>의 <a href="/wiki/%ED%99%94%ED%95%99%EC%9E%90" title="화학자">화학자</a>(<a href="/wiki/%EC%9C%A0%EA%B8%B0%ED%99%94%ED%95%99" title="유기화학">유기화학</a>)이다. 학위는 공학박사(<a href="/wiki/%EA%B5%90%ED%86%A0_%EB%8C%80%ED%95%99" title="교토 대학">교토 대학</a>, 1967년).</p>
<p><a href="/wiki/%EC%9D%BC%EB%B3%B8_%ED%95%99%EC%82%AC%EC%9B%90" title="일본 학사원">일본 학사원</a> 회원, 국립연구개발법인 <a class="new" href="/w/index.php?title=%EA%B3%BC%ED%95%99%EA%B8%B0%EC%88%A0%EC%A7%84%ED%9D%A5%EA%B8%B0%EA%B5%AC&amp;action=edit&amp;redlink=1" title="과학기술진흥기구 (없는 문서)">과학기술진흥기구</a> 연구개발전략센터장,<sup class="reference" id="cite_ref-2"><a href="#cite_note-2">[2]</a></sup> <a href="/wiki/%EB%82%98%EA%B3%A0%EC%95%BC_%EB%8C%80%ED%95%99" title="나고야 대학">나고야 대학</a> 특임교수, <a href="/wiki/%EB%A9%94%EC%9D%B4%EC%A1%B0_%EB%8C%80%ED%95%99" title="메이조 대학">메이조 대학</a> 객원교수, <a href="/wiki/%EB%8B%A4%EC%B9%B4%EC%82%AC%EA%B3%A0_%ED%96%A5%EB%A3%8C%EA%B3%B5%EC%97%85" title="다카사고 향료공업">다카사고 향료공업</a> 주식회사 이사, 나고야 대학 대학원 이학연구과 연구과장, 이학부 학부장, 물질 과학 국제 연구 센터장, <a href="/wiki/%EB%8F%85%EB%A6%BD%ED%96%89%EC%A0%95%EB%B2%95%EC%9D%B8" title="독립행정법인">독립행정법인</a> <a href="/wiki/%EC%9D%B4%ED%99%94%ED%95%99%EC%97%B0%EA%B5%AC%EC%86%8C" title="이화학연구소">이화학연구소</a> 이사장 등을 역임했다.</p>
<p>2001년에 ‘<a href="/wiki/%EC%B9%B4%EC%9D%B4%EB%9E%84%EC%84%B1_(%ED%99%94%ED%95%99)" title="카이랄성 (화학)">카이랄성</a>을 갖고 <a href="/wiki/%EC%B4%89%EB%A7%A4" title="촉매">촉매</a>되는 <a href="/wiki/%EC%88%98%EC%86%8C" title="수소">수소</a> 첨가 반응에 대한 연구’에 의한 공로로 <a href="/wiki/%EB%85%B8%EB%B2%A8_%ED%99%94%ED%95%99%EC%83%81" title="노벨 화학상">노벨 화학상</a>을 수상했다.</p>

<h2><span class="mw-headline" id=".EC.9D.B8.EB.AC.BC">인물</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%85%B8%EC%9A%94%EB%A6%AC_%EB%A3%8C%EC%A7%80&amp;action=edit&amp;section=1" title="부분 편집: 인물">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<h3><span class="mw-headline" id=".ED.95.99.EC.B0.BD_.EC.8B.9C.EC.A0.88">학창 시절</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%85%B8%EC%9A%94%EB%A6%AC_%EB%A3%8C%EC%A7%80&amp;action=edit&amp;section=2" title="부분 편집: 학창 시절">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>1938년 9월, <a href="/wiki/%ED%9A%A8%EA%B3%A0_%ED%98%84" title="효고 현">효고 현</a> 무코 군 세이도 촌(현재의 <a href="/wiki/%EC%95%84%EC%8B%9C%EC%95%BC_%EC%8B%9C" title="아시야 시">아시야 시</a>)에서 노요리 가네키·노요리 스즈코의 장남으로 태어났다.<sup class="reference" id="cite_ref-kahaku_1-1"><a href="#cite_note-kahaku-1">[1]</a></sup> 1945년에 종전을 맞아 피난했던 곳에서 <a href="/wiki/%EA%B3%A0%EB%B2%A0_%EC%8B%9C" title="고베 시">고베</a>의 <a class="new" href="/w/index.php?title=%EB%A1%AF%EC%BD%94_%EC%82%B0&amp;action=edit&amp;redlink=1" title="롯코 산 (없는 문서)">롯코</a>로 되돌아와서 효고 사범학교 남자부 부속초등학교(현재의 고베 대학 부속 스미요시 초등학교)에 입학했다.<sup class="reference" id="cite_ref-kahaku_1-2"><a href="#cite_note-kahaku-1">[1]</a></sup> 1951년 4월에는 사립 나다 중학교에 입학하여<sup class="reference" id="cite_ref-kahaku_1-3"><a href="#cite_note-kahaku-1">[1]</a></sup> <a href="/wiki/%EB%82%98%EC%9D%BC%EB%A1%A0" title="나일론">나일론</a>은 <a href="/wiki/%EC%84%9D%ED%83%84" title="석탄">석탄</a>과 <a href="/wiki/%EB%AC%BC" title="물">물</a>과 <a href="/wiki/%EA%B3%B5%EA%B8%B0" title="공기">공기</a>에서 만들어졌다는 이야기를 듣고 <a href="/wiki/%ED%99%94%ED%95%99" title="화학">화학</a>에 대한 뜻을 품게됐다. 그 후 나다 고등학교를 거쳐 1957년 4월에 <a href="/wiki/%EA%B5%90%ED%86%A0_%EB%8C%80%ED%95%99" title="교토 대학">교토 대학</a> 공학부에 진학했다.<sup class="reference" id="cite_ref-kahaku_1-4"><a href="#cite_note-kahaku-1">[1]</a></sup></p>
<p>1961년 3월에 교토 대학 공학부 공업화학과를 졸업했고 2년 뒤인 1963년 3월에 교토 대학 대학원에서 공학연구과 공업화학 전공으로 석사과정을 수료했다.<sup class="reference" id="cite_ref-kahaku_1-5"><a href="#cite_note-kahaku-1">[1]</a></sup></p>
<h3><span class="mw-headline" id=".ED.95.99.EC.88.A0_.ED.99.9C.EB.8F.99">학술 활동</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%85%B8%EC%9A%94%EB%A6%AC_%EB%A3%8C%EC%A7%80&amp;action=edit&amp;section=3" title="부분 편집: 학술 활동">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>1963년 4월, 교토 대학 공학부에서 <a class="new" href="/w/index.php?title=%EB%85%B8%EC%9E%90%ED%82%A4_%ED%9E%88%ED%86%A0%EC%8B%9C&amp;action=edit&amp;redlink=1" title="노자키 히토시 (없는 문서)">노자키 히토시</a> 연구실의 조수로 발탁됐고 1967년 9월에 공학박사 학위를 취득했다. 논문 제목은 《The chemistry of metastable species leading to strained homocyclic molecules》(비틀림을 갖는 탄소고리의 합성에 있어서 불안정체의 화학).</p>
<p>1968년 2월 <a href="/wiki/%EB%82%98%EA%B3%A0%EC%95%BC_%EB%8C%80%ED%95%99" title="나고야 대학">나고야 대학</a>으로 자리를 옮겨 이학부에서 조교수로 부임했고<sup class="reference" id="cite_ref-kahaku_1-6"><a href="#cite_note-kahaku-1">[1]</a></sup> 1969년 1월에는 <a href="/wiki/%EB%AF%B8%EA%B5%AD" title="미국">미국</a>으로 건너가서 <a class="mw-redirect" href="/wiki/%ED%95%98%EB%B2%84%EB%93%9C_%EB%8C%80%ED%95%99" title="하버드 대학">하버드 대학</a> 박사연구원으로서 <a href="/wiki/%EC%9D%BC%EB%9D%BC%EC%9D%B4%EC%96%B4%EC%8A%A4_%EC%A0%9C%EC%9E%84%EC%8A%A4_%EC%BD%94%EB%A6%AC" title="일라이어스 제임스 코리">일라이어스 제임스 코리</a>(1990년 <a href="/wiki/%EB%85%B8%EB%B2%A8_%ED%99%94%ED%95%99%EC%83%81" title="노벨 화학상">노벨 화학상</a> 수상)의 지도 하에 1970년 3월까지 연구를 했다.<sup class="reference" id="cite_ref-kahaku_1-7"><a href="#cite_note-kahaku-1">[1]</a></sup> 이 시기에 훗날 노벨 화학상 공동 수상자가 되는 <a class="new" href="/w/index.php?title=%EB%B0%B0%EB%A6%AC_%EC%83%A4%ED%94%8C%EB%A6%AC%EC%8A%A4&amp;action=edit&amp;redlink=1" title="배리 샤플리스 (없는 문서)">배리 샤플리스</a>와의 교류가 시작됐다. 귀국 후인 1972년 8월에 나고야 대학의 이학부 교수로 승진했고<sup class="reference" id="cite_ref-kahaku_1-8"><a href="#cite_note-kahaku-1">[1]</a></sup> 같은 해 노요리 히로코와 결혼했다. 그 후 1983년에는 <a class="mw-redirect" href="/wiki/%EB%A9%98%ED%86%A8" title="멘톨">멘톨</a>의 양산화에 성공하고 1986년에 <a class="new" href="/w/index.php?title=BINAP&amp;action=edit&amp;redlink=1" title="BINAP (없는 문서)">BINAP</a>-<a href="/wiki/%EB%A3%A8%ED%85%8C%EB%8A%84" title="루테늄">루테늄</a> 촉매를 발명하는 등의 업적을 남겼다.</p>
<p>1996년 2월부터 나고야 대학의 대학원에서 이학연구과 교수를 맡았고<sup class="reference" id="cite_ref-kahaku_1-9"><a href="#cite_note-kahaku-1">[1]</a></sup> 1997년 1월에는 이학연구과장 및 이학부장에, 2000년 4월에는 같은 대학의 물질과학국제연구센터에서 센터장으로 부임했다.<sup class="reference" id="cite_ref-kahaku_1-10"><a href="#cite_note-kahaku-1">[1]</a></sup> 2003년 7월부터 독립행정법인 <a href="/wiki/%EC%9D%B4%ED%99%94%ED%95%99%EC%97%B0%EA%B5%AC%EC%86%8C" title="이화학연구소">이화학연구소</a>에서 이사장직을 맡고 있었지만<sup class="reference" id="cite_ref-kahaku_1-11"><a href="#cite_note-kahaku-1">[1]</a></sup> <a href="/wiki/%EC%98%A4%EB%B3%B4%EC%B9%B4%ED%83%80_%ED%95%98%EB%A3%A8%EC%BD%94" title="오보카타 하루코">오보카타 하루코</a> 등에 의한 <a href="/wiki/%EC%9E%90%EA%B7%B9_%EC%95%BC%EA%B8%B0%EC%84%B1_%EB%8B%A4%EA%B8%B0%EB%8A%A5%EC%84%B1_%ED%9A%8D%EB%93%9D%EC%84%B8%ED%8F%AC" title="자극 야기성 다기능성 획득세포">STAP 세포</a>와 관련된 논문 조작 사건에 대한 책임을 지고 임기 도중인 2015년 3월에 이사장직에서 물러났다.<sup class="reference" id="cite_ref-yonhapnews20150307_3-0"><a href="#cite_note-yonhapnews20150307-3">[3]</a></sup></p>
<h3><span class="mw-headline" id=".EA.B3.B5.EC.A0.81_.ED.99.9C.EB.8F.99">공적 활동</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%85%B8%EC%9A%94%EB%A6%AC_%EB%A3%8C%EC%A7%80&amp;action=edit&amp;section=4" title="부분 편집: 공적 활동">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>노요리 료지는 또한 공적 활동에도 종사하여 <a class="mw-redirect" href="/wiki/%EC%9D%BC%EB%B3%B8_%EB%AC%B8%EB%B6%80%EC%84%B1" title="일본 문부성">문부성</a> 학술심의회 위원, <a href="/wiki/%EC%9D%BC%EB%B3%B8_%EB%AC%B8%EB%B6%80%EA%B3%BC%ED%95%99%EC%84%B1" title="일본 문부과학성">문부과학성</a> 과학기술·학술심의회 위원, <a class="new" href="/w/index.php?title=%EC%9D%BC%EB%B3%B8_%ED%95%99%EC%88%A0%EC%A7%84%ED%9D%A5%ED%9A%8C&amp;action=edit&amp;redlink=1" title="일본 학술진흥회 (없는 문서)">일본 학술진흥회</a> 학술고문 등의 직무를 맡았다. 2006년에는 <a href="/wiki/%EC%A0%9C1%EC%B0%A8_%EC%95%84%EB%B2%A0_%EC%8B%A0%EC%A1%B0_%EB%82%B4%EA%B0%81" title="제1차 아베 신조 내각">아베 신조 내각</a>에서 설치된 정부의 교육재생회의에서 단장으로 취임했다. 2007년에 <a href="/wiki/%EC%95%84%ED%82%A4%ED%9E%88%ED%86%A0" title="아키히토">아키히토</a>·<a href="/wiki/%EB%AF%B8%EC%B9%98%EC%BD%94_%ED%99%A9%ED%9B%84" title="미치코 황후">미치코 황후</a>의 유럽 5개국 방문에 즈음하여 수석 수행원을 맡았다.</p>

나고야 대학 히가시야마 캠퍼스에 세워진 노요리 료지의 노벨상 수상 기념비. 비석에 ‘연구는 신선하고, 단순 명쾌하게’(<span lang="ja">「研究は瑞々しく、単純明快に」</span>) 라는 문구가 새겨져 있다.</div>

<li><a href="/wiki/%EC%95%8C%EC%BC%84" title="알켄">알켄</a>의 비대칭 수소화(<a class="new" href="/w/index.php?title=%EB%85%B8%EC%9A%94%EB%A6%AC_%EB%B9%84%EB%8C%80%EC%B9%AD_%EC%88%98%EC%86%8C%ED%99%94_%EB%B0%98%EC%9D%91&amp;action=edit&amp;redlink=1" title="노요리 비대칭 수소화 반응 (없는 문서)">노요리 비대칭 수소화 반응</a>) 등에 <a href="/wiki/%EB%A6%AC%EA%B0%84%EB%93%9C" title="리간드">리간드</a>로서 이용되는 <a class="new" href="/w/index.php?title=BINAP&amp;action=edit&amp;redlink=1" title="BINAP (없는 문서)">BINAP</a>을 개발했다.</li>
<li>Rh-BINAP을 이용한 비대칭 합성 반응에 의해 <a href="/wiki/%EB%B0%95%ED%95%98%EB%87%8C" title="박하뇌">멘톨</a> 합성의 공업화를 가능케 했다.</li>
<li><a href="/wiki/%EC%B9%B4%EB%B3%B4%EB%8B%90%EA%B8%B0" title="카보닐기">카보닐기 화합물</a>을 카이랄 선택적으로 <a href="/wiki/%EC%95%8C%EC%BD%94%EC%98%AC" title="알코올">알코올</a>로 변환할 수 있는 금속 착체 촉매(BINAP-디아민루테늄(II) 촉매)를 개발했다.<sup class="reference" id="cite_ref-4"><a href="#cite_note-4">[4]</a></sup> 이 촉매는 매우 많은 카보닐기 화합물에 적용 가능한 범용성이 높은 것이며 <a href="/wiki/%EC%9D%98%EC%95%BD%ED%92%88" title="의약품">의약품</a>, <a href="/wiki/%EB%86%8D%EC%95%BD" title="농약">농약</a>, <a class="new" href="/w/index.php?title=%ED%96%A5%EB%A3%8C&amp;action=edit&amp;redlink=1" title="향료 (없는 문서)">향료</a> 등을 만들 때 열쇠가 되는 비대칭 합성 반응에 널리 이용되고 있다. 이러한 성과가 노벨 화학상 수상에 크게 기여하고 있다.</li>
</ul>
<h2><span class="mw-headline" id=".EB.B0.9C.EC.96.B8">발언</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%85%B8%EC%9A%94%EB%A6%AC_%EB%A3%8C%EC%A7%80&amp;action=edit&amp;section=6" title="부분 편집: 발언">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<ul>
<li>2006년 12월 8일에 열린 교육재생회의 ‘규범 의식·가족·지역 교육재생분과회’(제2분과회)<sup class="reference" id="cite_ref-5"><a href="#cite_note-5">[5]</a></sup>에서 “<a href="/wiki/%ED%95%99%EC%8A%B5%EC%88%99" title="학습숙">학습숙</a>은 성적이 나쁜 아이가 가기 위해서는 필요하지만 보통 이상의 아이는 금지에 해야 한다”라고 주장해 논란을 불러 일으켰다. 노요리의 발언에 동조하는 위원도 있었지만 다른 위원에게서는 ‘학문의 영역이 넓어졌다’, ‘수학의 레벨은 학원에 의해서 유지되고 있다’는 등의 반론을 받았다. 노요리의 주장은 이 회의가 12월 21일에 정리한 제1차 보고의 원안에는 들어있지 않았다.</li>
<li>2009년 11월 25일에 열린 ‘첨단과학조사회’(문부과학성의 정책회의가 학습회로 설치한 것)에서는 정부의 사업 구분으로 과학 기술 관련 사업의 예산 삭감이 잇따르고 있는 것에 대해 “과학 기술은 일본이 국제 경쟁을 살리는 수단이며 국제 협조의 기둥이다. 이것을 삭감하는 것은 견식이 없다”고 강도 높게 비판했다. 이 가운데는 다른 선진국과 비교해서 과학 기술 관련 예산이 현저하게 적은 것과 미국에서 박사 학위를 취득한 사람이 <a href="/wiki/%EC%A4%91%ED%99%94%EC%9D%B8%EB%AF%BC%EA%B3%B5%ED%99%94%EA%B5%AD" title="중화인민공화국">중화인민공화국</a>의 20분의 1, <a href="/wiki/%EB%8C%80%ED%95%9C%EB%AF%BC%EA%B5%AD" title="대한민국">대한민국</a>의 6분의 1이라는 현상을 지적하고 “10년 후, 각국에 거대한 과학 국제 인맥이 형성되면서 일본은 뒤처질 가능성이 있다”, “(사업 구분은)장래에 역사의 법정에 설 각오로 하고 있는지 묻고 싶다”라고 비판했다.<sup class="reference" id="cite_ref-6"><a href="#cite_note-6">[6]</a></sup> 노요리의 비판에 가토 히데키 행정 쇄신 회의 사무국장은 “(구분의 논의를) 보지도, 듣지도, 알지도 못하고 ‘견식이 없다’고 말하는 것은 비과학적”이라고 반박했다.<sup class="reference" id="cite_ref-7"><a href="#cite_note-7">[7]</a></sup></li>
</ul>

<ul>
<li>실업가인 <a class="new" href="/w/index.php?title=%EC%95%84%EC%82%AC%EB%B6%80%ED%82%A4_%EC%97%90%EC%9D%B4%EC%A7%80&amp;action=edit&amp;redlink=1" title="아사부키 에이지 (없는 문서)">아사부키 에이지</a>, <a class="new" href="/w/index.php?title=%EC%95%84%EC%82%AC%EB%B6%80%ED%82%A4_%EC%93%B0%EB%84%A4%ED%82%A4%EC%B9%98&amp;action=edit&amp;redlink=1" title="아사부키 쓰네키치 (없는 문서)">아사부키 쓰네키치</a>, <a href="/wiki/%EA%B2%8C%EC%9D%B4%EC%98%A4%EA%B8%B0%EC%A3%BC%EC%BF%A0_%EB%8C%80%ED%95%99" title="게이오기주쿠 대학">게이오기주쿠 대학</a> 교수인 <a class="new" href="/w/index.php?title=%EC%95%84%EC%82%AC%EB%B6%80%ED%82%A4_%EB%A3%8C%EC%A7%80&amp;action=edit&amp;redlink=1" title="아사부키 료지 (없는 문서)">아사부키 료지</a>, <a class="new" href="/w/index.php?title=%EC%95%84%EC%82%AC%EB%B6%80%ED%82%A4_%EC%82%B0%ED%82%A4%EC%B9%98&amp;action=edit&amp;redlink=1" title="아사부키 산키치 (없는 문서)">아사부키 산키치</a>와는 친족 관계이다. 또한 아버지 가네키의 친아버지 시오카와 산시로(은행가)의 아내 시오카와 지나쓰는 <a class="mw-redirect" href="/wiki/%EC%99%80%ED%83%80%EB%82%98%EB%B2%A0_%EC%B9%98%EC%95%84%ED%82%A4" title="와타나베 치아키">와타나베 치아키</a>(궁내상 백작)의 3녀이다. 따라서 <a href="/wiki/%EA%B7%80%EC%A1%B1%EC%9B%90_(%EC%9D%BC%EB%B3%B8)" title="귀족원 (일본)">귀족원</a> 의원 와타나베 아키라, <a href="/wiki/%EC%95%84%EC%8B%9C%EC%95%84_%EA%B0%9C%EB%B0%9C_%EC%9D%80%ED%96%89" title="아시아 개발 은행">아시아 개발 은행</a> 초대총재 <a class="new" href="/w/index.php?title=%EC%99%80%ED%83%80%EB%82%98%EB%B2%A0_%EB%8B%A4%EC%BC%80%EC%8B%9C_(1906%EB%85%84)&amp;action=edit&amp;redlink=1" title="와타나베 다케시 (1906년) (없는 문서)">와타나베 다케시</a>, <a class="new" href="/w/index.php?title=%EC%99%80%ED%83%80%EB%82%98%EB%B2%A0_%EC%82%AC%ED%86%A0%EC%8B%9C_(%EB%AC%BC%EB%A6%AC%ED%95%99%EC%9E%90)&amp;action=edit&amp;redlink=1" title="와타나베 사토시 (물리학자) (없는 문서)">와타나베 사토시</a>(물리학자) 등은 아버지의 사촌에 해당한다. 와타나베가는 <a href="/wiki/%EC%98%A4%EC%95%BC%EB%A7%88_%EC%9D%B4%EC%99%80%EC%98%A4" title="오야마 이와오">오야마 이와오</a>의 집안과도 친척이다.</li>
