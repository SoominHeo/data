<p><b>노이브란덴부르크 구</b>(<span style="font-size: smaller;"><a href="/wiki/%EB%8F%85%EC%9D%BC%EC%96%B4" title="독일어">독일어</a>:</span> <span lang="de" xml:lang="de">Bezirk Neubrandenburg</span>)는 과거 <a href="/wiki/%EB%8F%99%EB%8F%85" title="동독">동독</a>(독일 민주 공화국)에 존재했던 14개 구 가운데 하나로 구청 소재지는 <a href="/wiki/%EB%85%B8%EC%9D%B4%EB%B8%8C%EB%9E%80%EB%8D%B4%EB%B6%80%EB%A5%B4%ED%81%AC" title="노이브란덴부르크">노이브란덴부르크</a>였다.</p>

<h2><span class="mw-headline" id=".ED.96.89.EC.A0.95_.EA.B5.AC.EC.97.AD_.EC.A0.95.EB.B3.B4">행정 구역 정보</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%85%B8%EC%9D%B4%EB%B8%8C%EB%9E%80%EB%8D%B4%EB%B6%80%EB%A5%B4%ED%81%AC_%EA%B5%AC&amp;action=edit&amp;section=1" title="부분 편집: 행정 구역 정보">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<ul>
<li>구청 소재지: <a href="/wiki/%EB%85%B8%EC%9D%B4%EB%B8%8C%EB%9E%80%EB%8D%B4%EB%B6%80%EB%A5%B4%ED%81%AC" title="노이브란덴부르크">노이브란덴부르크</a></li>
<li>면적: 10,948㎢</li>
<li>인구: 620,500명 (1989년 당시 기준)</li>
<li>인구 밀도: 57명/㎢</li>
<li>차량 번호판 코드: C</li>
<li>하위 행정 구역: 14개 군, 1개 시</li>
</ul>
<h2><span class="mw-headline" id=".EC.97.AD.EC.82.AC">역사</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%85%B8%EC%9D%B4%EB%B8%8C%EB%9E%80%EB%8D%B4%EB%B6%80%EB%A5%B4%ED%81%AC_%EA%B5%AC&amp;action=edit&amp;section=2" title="부분 편집: 역사">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p><a href="/wiki/1952%EB%85%84" title="1952년">1952년</a> <a href="/wiki/7%EC%9B%94_25%EC%9D%BC" title="7월 25일">7월 25일</a>에 시행된 행정 구역 개편 당시에 폐지된 주를 대신하여 신설된 14개 구 가운데 하나였다. <a href="/wiki/1990%EB%85%84" title="1990년">1990년</a> <a href="/wiki/%EB%8F%85%EC%9D%BC%EC%9D%98_%EC%9E%AC%ED%86%B5%EC%9D%BC" title="독일의 재통일">동서독 통일</a> 당시에 <a href="/wiki/%EB%A9%94%ED%81%B4%EB%A0%8C%EB%B6%80%EB%A5%B4%ED%81%AC%ED%8F%AC%EC%96%B4%ED%8F%AC%EB%A9%94%EB%A5%B8_%EC%A3%BC" title="메클렌부르크포어포메른 주">메클렌부르크포어포메른 주</a>에 편입되었다.</p>
<h2><span class="mw-headline" id=".EC.A7.80.EB.A6.AC">지리</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%85%B8%EC%9D%B4%EB%B8%8C%EB%9E%80%EB%8D%B4%EB%B6%80%EB%A5%B4%ED%81%AC_%EA%B5%AC&amp;action=edit&amp;section=3" title="부분 편집: 지리">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>동독 북동부에 위치한 구였으며 동쪽으로는 <a href="/wiki/%ED%8F%B4%EB%9E%80%EB%93%9C" title="폴란드">폴란드</a>와 국경을 접하고 있었다. 북쪽으로는 <a href="/wiki/%EB%A1%9C%EC%8A%A4%ED%86%A0%ED%81%AC_%EA%B5%AC" title="로스토크 구">로스토크 구</a>, 서쪽으로는 <a href="/wiki/%EC%8A%88%EB%B2%A0%EB%A6%B0_%EA%B5%AC" title="슈베린 구">슈베린 구</a>, 남쪽으로는 <a href="/wiki/%ED%8F%AC%EC%B8%A0%EB%8B%B4_%EA%B5%AC" title="포츠담 구">포츠담 구</a>, <a href="/wiki/%ED%94%84%EB%9E%91%ED%81%AC%ED%91%B8%EB%A5%B4%ED%8A%B8_%EA%B5%AC" title="프랑크푸르트 구">프랑크푸르트 구</a>와 접하고 있었다.</p>
<h3><span class="mw-headline" id=".ED.95.98.EC.9C.84_.ED.96.89.EC.A0.95_.EA.B5.AC.EC.97.AD">하위 행정 구역</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%85%B8%EC%9D%B4%EB%B8%8C%EB%9E%80%EB%8D%B4%EB%B6%80%EB%A5%B4%ED%81%AC_%EA%B5%AC&amp;action=edit&amp;section=4" title="부분 편집: 하위 행정 구역">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<ul>
<li>14개 군(Landkreise): 알텐트레프토프 군(Altentreptow), 앙클람 군(Anklam), 뎀민 군(Demmin), 말힌 군(Malchin), 노이브란덴부르크란트 군(Neubrandenburg-Land), 노이슈트렐리츠 군(Neustrelitz), 파제발크 군(Pasewalk), 프렌츨라우 군(Prenzlau), 뢰벨/뮈리츠 군(Röbel/Müritz), 슈트라스부르크 군(Strasburg), 템플린 군(Templin), 테테로프 군(Teterow), 위커뮌데 군(Ueckermünde), 바렌 군(Waren)</li>
<li>1개 시(Stadtkreise): <a href="/wiki/%EB%85%B8%EC%9D%B4%EB%B8%8C%EB%9E%80%EB%8D%B4%EB%B6%80%EB%A5%B4%ED%81%AC" title="노이브란덴부르크">노이브란덴부르크 시</a>(Neubrandenburg)</li>
</ul>
