<h1 class="firstHeading" id="firstHeading" lang="ko">노자와 다이조</h1>
<div class="mw-body-content" id="bodyContent">
<div id="siteSub">위키백과, 우리 모두의 백과사전.</div>
<div id="contentSub"></div>
<div class="mw-jump" id="jump-to-nav">
					이동:					<a href="#mw-head">둘러보기</a>, 					<a href="#p-search">검색</a>
</div>
<div class="mw-content-ltr" dir="ltr" id="mw-content-text" lang="ko"><script>function mfTempOpenSection(id){var block=document.getElementById("mf-section-"+id);block.className+=" open-block";block.previousSibling.className+=" open-block";}</script><p><b>노자와 다이조</b> (野沢 太三, のざわ だいぞう, 1933년 5월 6일 ~) 는 <a href="/wiki/%EC%9D%BC%EB%B3%B8" title="일본">일본</a>의 기술사·공학박사·정치인으로, 전 <a href="/wiki/%EC%B0%B8%EC%9D%98%EC%9B%90_(%EC%9D%BC%EB%B3%B8)" title="참의원 (일본)">참의원</a> 의원이다. <a class="mw-redirect" href="/wiki/%EC%A0%9C1%EC%B0%A8_%EA%B3%A0%EC%9D%B4%EC%A6%88%EB%AF%B8_%EB%82%B4%EA%B0%81_%EC%A0%9C2%EC%B0%A8_%EA%B0%9C%EC%A1%B0%EB%82%B4%EA%B0%81" title="제1차 고이즈미 내각 제2차 개조내각">제1차 고이즈미 내각 제2차 개조내각</a>, <a href="/wiki/%EC%A0%9C2%EC%B0%A8_%EA%B3%A0%EC%9D%B4%EC%A6%88%EB%AF%B8_%EB%82%B4%EA%B0%81" title="제2차 고이즈미 내각">제2차 고이즈미 내각</a>에서 법무대신을 지냈으며, 현재 <a class="extiw" href="https://ja.wikipedia.org/wiki/%E6%97%A5%E9%9F%93%E3%83%88%E3%83%B3%E3%83%8D%E3%83%AB%E7%A0%94%E7%A9%B6%E4%BC%9A" title="ja:日韓トンネル研究会">한일해저터널 연구회</a>의 회장을 맡고 있다.</p>
<p><a href="/wiki/%EB%82%98%EA%B0%80%EB%85%B8_%ED%98%84" title="나가노 현">나가노 현</a> <a href="/wiki/%EA%B0%80%EB%AF%B8%EC%9D%B4%EB%82%98_%EA%B5%B0" title="가미이나 군">가미이나 군</a> <a href="/wiki/%EB%8B%A4%EC%93%B0%EB%85%B8_%EC%A0%95" title="다쓰노 정">다쓰노 정</a> 출신으로, <a href="/wiki/%EB%8F%84%EC%BF%84_%EB%8C%80%ED%95%99" title="도쿄 대학">도쿄 대학</a> 공학부 토목공학과를 졸업한 후, <a class="mw-redirect" href="/wiki/%EC%9D%BC%EB%B3%B8_%EA%B5%AD%EC%9C%A0_%EC%B2%A0%EB%8F%84" title="일본 국유 철도">일본 국유 철도</a>에 입사해 오사카 철도 관리국 시설 부장, 나가노 철도 관리 국장, 본부 시설 국장을 지냈다.</p>
<p>1986년, 국회의원 선거 비례구에서 <a href="/wiki/%EC%9E%90%EC%9C%A0%EB%AF%BC%EC%A3%BC%EB%8B%B9_(%EC%9D%BC%EB%B3%B8)" title="자유민주당 (일본)">자유민주당</a> 공인으로 출마해 첫 당선되었다. 당선 후 <a href="/wiki/%ED%99%8B%EC%B9%B4%EC%9D%B4%EB%8F%84" title="홋카이도">홋카이도</a> 개발 정무차관을 시작으로, 참의원에서 외무 위원장, 결산 위원장을 지냈다. 2002년, 참의원 헌법 연구 회장이 되었고, 이후 <a class="mw-redirect" href="/wiki/%EC%A0%9C1%EC%B0%A8_%EA%B3%A0%EC%9D%B4%EC%A6%88%EB%AF%B8_%EB%82%B4%EA%B0%81_%EC%A0%9C2%EC%B0%A8_%EA%B0%9C%EC%A1%B0%EB%82%B4%EA%B0%81" title="제1차 고이즈미 내각 제2차 개조내각">제1차 고이즈미 내각 제2차 개조내각</a>에서 법무장관이 되었다. 제20회 일본 참의원 의원 통상 선거에 출마하지 않고, 참의원 의원 임기 만료 후인 2004년 9월, 법무대신에서 물러났다.</p>
