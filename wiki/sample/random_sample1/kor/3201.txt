<p><b>노즈 미치쓰라</b>(<span style="font-size: smaller;"><a href="/wiki/%EC%9D%BC%EB%B3%B8%EC%96%B4" title="일본어">일본어</a>:</span> <span lang="ja"><span lang="ja"><ruby><rb>野津道貫</rb> <rp>(</rp><rt>のづ みちつら</rt><rp>)</rp></ruby></span></span> <small>노즈 미치츠라<sup>[<a href="/wiki/%EC%9C%84%ED%82%A4%EB%B0%B1%EA%B3%BC:%EC%9D%BC%EB%B3%B8%EC%96%B4%EC%9D%98_%ED%95%9C%EA%B8%80_%ED%91%9C%EA%B8%B0" title="위키백과:일본어의 한글 표기">*</a>]</sup></small>, <a href="/wiki/1841%EB%85%84" title="1841년">1841년</a> <a href="/wiki/12%EC%9B%94_17%EC%9D%BC" title="12월 17일">12월 17일</a>(<a href="/wiki/%EB%8D%B4%ED%8F%AC" title="덴포">덴포</a> 12년 <a href="/wiki/11%EC%9B%94_5%EC%9D%BC" title="11월 5일">11월 5일</a>) ~ <a href="/wiki/1908%EB%85%84" title="1908년">1908년</a>(<a class="mw-disambig" href="/wiki/%EB%A9%94%EC%9D%B4%EC%A7%80" title="메이지">메이지</a> 41년) <a href="/wiki/10%EC%9B%94_18%EC%9D%BC" title="10월 18일">10월 18일</a>)는 <a href="/wiki/%EC%97%90%EB%8F%84_%EB%A7%89%EB%B6%80" title="에도 막부">에도 막부</a> 말기 <a href="/wiki/%EC%82%AC%EC%93%B0%EB%A7%88_%EB%B2%88" title="사쓰마 번">사쓰마 번</a>의 번사 출신으로 <a href="/wiki/%EB%A9%94%EC%9D%B4%EC%A7%80_%EC%8B%9C%EB%8C%80" title="메이지 시대">메이지 시대</a>에 활동한 <a href="/wiki/%EC%9D%BC%EB%B3%B8_%EC%A0%9C%EA%B5%AD" title="일본 제국">일본 제국</a>의 군인이다. 귀족원의원, 동부도독, 교육총감, 제4군사령관을 역임하였다. 통칭은 시치지로(七次)이며 휘는 미치쓰라(道貫)이다. 최종 계급은 원수·육군대장이며 정이위 대훈위 공일급의 후작이다.</p>
