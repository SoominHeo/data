<p><b>노키아 루미아 710</b>(Nokia Lumia 710)은 <a href="/wiki/%EB%85%B8%ED%82%A4%EC%95%84" title="노키아">노키아</a>가 <a href="/wiki/%EB%A7%88%EC%9D%B4%ED%81%AC%EB%A1%9C%EC%86%8C%ED%94%84%ED%8A%B8" title="마이크로소프트">마이크로소프트</a>와 공동으로 개발한 <a href="/wiki/%EC%8A%A4%EB%A7%88%ED%8A%B8%ED%8F%B0" title="스마트폰">스마트폰</a>으로, 노키아의 첫 <a href="/wiki/%EC%9C%88%EB%8F%84%EC%9A%B0_%ED%8F%B0_7" title="윈도우 폰 7">윈도우 폰 7</a> 플랫폼 제품이며 노키아 루미아 윈도우 폰 7 라인업 중 보급형 기종이다. <a href="/wiki/%EC%9C%A0%EB%9F%BD" title="유럽">유럽</a>에서는 <a href="/wiki/2011%EB%85%84" title="2011년">2011년</a> 10월 <a class="new" href="/w/index.php?title=%EB%85%B8%ED%82%A4%EC%95%84_%EB%A3%A8%EB%AF%B8%EC%95%84_800&amp;action=edit&amp;redlink=1" title="노키아 루미아 800 (없는 문서)">루미아 800</a>과 함께 출시되었고<sup class="reference" id="cite_ref-1"><a href="#cite_note-1">[1]</a></sup>, <a href="/wiki/2012%EB%85%84" title="2012년">2012년</a> 1월에는 <a href="/wiki/T-%EB%AA%A8%EB%B0%94%EC%9D%BC" title="T-모바일">T-모바일</a>을 통해서 <a href="/wiki/%EB%AF%B8%EA%B5%AD" title="미국">미국</a>에도 출시되었다.<sup class="reference" id="cite_ref-2"><a href="#cite_note-2">[2]</a></sup> <a href="/wiki/%EB%8C%80%ED%95%9C%EB%AF%BC%EA%B5%AD" title="대한민국">대한민국</a>에서도 <a href="/wiki/2011%EB%85%84" title="2011년">2011년</a> 12월 26일 <a href="/wiki/KT" title="KT">KT</a>를 통해 출시 하였으며, 대한민국에서 정식 출시된 <a href="/wiki/%EC%9C%88%EB%8F%84%EC%9A%B0_%ED%8F%B0_7" title="윈도우 폰 7">윈도우 폰 7</a> 플랫폼의 첫 제품이다.</p>
