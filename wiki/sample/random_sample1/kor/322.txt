<h1 class="firstHeading" id="firstHeading" lang="ko">ICO (파일 포맷)</h1>
<div class="mw-body-content" id="bodyContent">
<div id="siteSub">위키백과, 우리 모두의 백과사전.</div>
<div id="contentSub"><span class="mw-redirectedfrom">(<a class="mw-redirect" href="/w/index.php?title=.ico&amp;redirect=no" title=".ico">.ico</a>에서 넘어옴)</span></div>
<div class="mw-jump" id="jump-to-nav">
					이동:					<a href="#mw-head">둘러보기</a>, 					<a href="#p-search">검색</a>
</div>
<div class="mw-content-ltr" dir="ltr" id="mw-content-text" lang="ko"><script>function mfTempOpenSection(id){var block=document.getElementById("mf-section-"+id);block.className+=" open-block";block.previousSibling.className+=" open-block";}</script><p><b>ICO 파일 포맷</b>은 <a class="mw-redirect" href="/wiki/%EB%A7%88%EC%9D%B4%ED%81%AC%EB%A1%9C%EC%86%8C%ED%94%84%ED%8A%B8_%EC%9C%88%EB%8F%84" title="마이크로소프트 윈도">마이크로소프트 윈도</a>의 <a href="/wiki/%EC%95%84%EC%9D%B4%EC%BD%98" title="아이콘">아이콘</a>에 쓰이는 그림 <a class="mw-redirect" href="/wiki/%ED%8C%8C%EC%9D%BC_%ED%8F%AC%EB%A7%B7" title="파일 포맷">파일 포맷</a>이다. <a class="mw-disambig" href="/wiki/%ED%97%A4%EB%8D%94" title="헤더">헤더</a>의 인증 바이트가 다른 것만 제외하고는 <a class="mw-redirect" href="/wiki/%EC%BB%B4%ED%93%A8%ED%84%B0_%EC%BB%A4%EC%84%9C" title="컴퓨터 커서">컴퓨터 커서</a>에 쓰이는 <b>CUR</b> 파일 포맷과 비슷하다.</p>
<p>ICO 파일은 여러 크기와 <a href="/wiki/%EC%83%89_%EA%B9%8A%EC%9D%B4" title="색 깊이">색 깊이</a>를 가진, 하나 이상의 작은 그림을 담고 있다.</p>
<h2><span class="mw-headline" id=".EA.B6.8C.EC.9E.A5_.EC.95.84.EC.9D.B4.EC.BD.98_.ED.81.AC.EA.B8.B0">권장 아이콘 크기</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=ICO_(%ED%8C%8C%EC%9D%BC_%ED%8F%AC%EB%A7%B7)&amp;action=edit&amp;section=1" title="부분 편집: 권장 아이콘 크기">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>완전한 윈도 비스타 / 윈도 7 호환성을 위해 제공되어야 하는 표준 아이콘 크기의 모임은 다음과 같다:</p>
<ol>
<li>256×256, 32비트 색, PNG 압축됨</li>
<li>256×256, 8비트 색, PNG 압축됨</li>
<li>256×256, 4비트 색, PNG 압축됨</li>
<li>48×48, 32비트 색, 압축되지 않음</li>
<li>48×48, 8비트 색, 압축되지 않음</li>
<li>48×48, 4비트 색, 압축되지 않음</li>
<li>32×32, 32비트 색, 압축되지 않음</li>
<li>32×32, 8비트 색, 압축되지 않음</li>
<li>32×32, 4비트 색, 압축되지 않음</li>
<li>16×16, 32비트 색, 압축되지 않음</li>
<li>16×16, 8비트 색, 압축되지 않음</li>
<li>16×16, 4비트 색, 압축되지 않음</li>
</ol>
<h2><span class="mw-headline" id=".EC.95.84.EC.9D.B4.EC.BD.98_.EB.9D.BC.EC.9D.B4.EB.B8.8C.EB.9F.AC.EB.A6.AC">아이콘 라이브러리</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=ICO_(%ED%8C%8C%EC%9D%BC_%ED%8F%AC%EB%A7%B7)&amp;action=edit&amp;section=2" title="부분 편집: 아이콘 라이브러리">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p><b>아이콘 라이브러리</b>는 윈도 아이콘들을 한데 묶는 방법이다. 여러 개의 아이콘을 묶고 있는 아이콘 <a class="new" href="/w/index.php?title=%EB%A6%AC%EC%86%8C%EC%8A%A4_(%EC%9C%88%EB%8F%84)&amp;action=edit&amp;redlink=1" title="리소스 (윈도) (없는 문서)">리소스</a>를 가진, .ICL 확장자를 갖는 <a class="new" href="/w/index.php?title=NE_%ED%8F%AC%EB%A7%B7&amp;action=edit&amp;redlink=1" title="NE 포맷 (없는 문서)">NE 포맷</a> 이진 파일이다.</p>
