<p><b>녹사평(용산구청)역</b>(綠莎坪(龍山區廳)驛)은 <a href="/wiki/%EC%84%9C%EC%9A%B8%ED%8A%B9%EB%B3%84%EC%8B%9C" title="서울특별시">서울특별시</a> <a href="/wiki/%EC%9A%A9%EC%82%B0%EA%B5%AC" title="용산구">용산구</a> <a href="/wiki/%EC%9A%A9%EC%82%B0%EB%8F%99_(%EC%84%9C%EC%9A%B8)" title="용산동 (서울)">용산동4가</a>에 있는 <a href="/wiki/%EC%84%9C%EC%9A%B8_%EC%A7%80%ED%95%98%EC%B2%A0_6%ED%98%B8%EC%84%A0" title="서울 지하철 6호선">서울 지하철 6호선</a>의 <a class="mw-redirect" href="/wiki/%EC%A7%80%ED%95%98%EC%B2%A0%EC%97%AD" title="지하철역">지하철역</a>이다. 부역명은 <b>용산구청</b>으로, 인근에 <a class="mw-redirect" href="/wiki/%EC%9A%A9%EC%82%B0%EA%B5%AC%EC%B2%AD" title="용산구청">용산구청</a>이 있다.</p>

<h2><span class="mw-headline" id=".EC.97.AD.EB.AA.85_.EC.9C.A0.EB.9E.98">역명 유래</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%85%B9%EC%82%AC%ED%8F%89%EC%97%AD&amp;action=edit&amp;section=4" title="부분 편집: 역명 유래">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>녹사평이라는 이름은 <a class="mw-redirect" href="/wiki/%EC%A1%B0%EC%84%A0%EC%8B%9C%EB%8C%80" title="조선시대">조선시대</a> 말까지만 하더라도 이 일대에 잡초가 무성하여 사람이 살지 않았기에 <b>푸른 풀이 무성한 들판</b>이라는 뜻으로 붙여진 이름이다. <a href="/wiki/2000%EB%85%84" title="2000년">2000년</a> <a href="/wiki/%EC%84%9C%EC%9A%B8_%EC%A7%80%ED%95%98%EC%B2%A0_6%ED%98%B8%EC%84%A0" title="서울 지하철 6호선">서울 지하철 6호선</a>이 개통될 때 예전 지명을 살려 녹사평역으로 명명되었다. 한편 인근에 <a href="/wiki/%EC%9A%A9%EC%82%B0%EA%B5%AC" title="용산구">용산구</a>청 신청사가 건설되고 구청의 기능이 <a href="/wiki/2010%EB%85%84" title="2010년">2010년</a> <a href="/wiki/4%EC%9B%94_8%EC%9D%BC" title="4월 8일">4월 8일</a>부터 이전됨에 따라 <a href="/wiki/%EC%9A%A9%EC%82%B0%EA%B5%AC" title="용산구">용산구</a>청이 용산구청역으로 역명 변경을 추진하였고,<sup class="reference" id="cite_ref-2"><a href="#cite_note-2">[2]</a></sup> <a href="/wiki/2013%EB%85%84" title="2013년">2013년</a> <a href="/wiki/12%EC%9B%94_26%EC%9D%BC" title="12월 26일">12월 26일</a> 역명이 <b>녹사평역</b>에서 <b>녹사평(용산구청)역</b>으로 변경되었다.<sup class="reference" id="cite_ref-3"><a href="#cite_note-3">[3]</a></sup></p>
<h2><span class="mw-headline" id=".EC.97.AD_.EA.B5.AC.EC.A1.B0_2">역 구조</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%85%B9%EC%82%AC%ED%8F%89%EC%97%AD&amp;action=edit&amp;section=5" title="부분 편집: 역 구조">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>역사는 지상 4층에서 지하 5층까지 이루어져 있으며, 지상 4층에는 자연 채광이 가능한 <a href="/wiki/%EC%9C%A0%EB%A6%AC" title="유리">유리</a> 돔이 있다. 영화 <a href="/wiki/%EB%A7%90%EC%95%84%ED%86%A4" title="말아톤">말아톤</a>을 비롯하여, <a class="mw-disambig" href="/wiki/%EC%B2%9C%EA%B5%AD%EC%9D%98_%EA%B3%84%EB%8B%A8" title="천국의 계단">천국의 계단</a> 등 여러 드라마의 촬영지였으며, 또한 역사 내부에서 무료 결혼식이 이루어지기도 한다.</p>
<p>당초 서울시청사의 이전 계획과 11호선 환승 계획으로 인해 지금과 같은 모습으로 건설되었으나, 현재는 백지화되었다.<sup class="정리_필요 noprint">[<a href="/wiki/%EC%9C%84%ED%82%A4%EB%B0%B1%EA%B3%BC:%EC%B6%9C%EC%B2%98_%EB%B0%9D%ED%9E%88%EA%B8%B0" title="위키백과:출처 밝히기">출처 필요</a>]</sup></p>
