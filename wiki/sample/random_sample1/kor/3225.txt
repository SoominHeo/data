<h1 class="firstHeading" id="firstHeading" lang="ko">논리실증주의</h1>
<div class="mw-body-content" id="bodyContent">
<div id="siteSub">위키백과, 우리 모두의 백과사전.</div>
<div id="contentSub"></div>
<div class="mw-jump" id="jump-to-nav">
					이동:					<a href="#mw-head">둘러보기</a>, 					<a href="#p-search">검색</a>
</div>
<div class="mw-content-ltr" dir="ltr" id="mw-content-text" lang="ko"><script>function mfTempOpenSection(id){var block=document.getElementById("mf-section-"+id);block.className+=" open-block";block.previousSibling.className+=" open-block";}</script><p><b>논리실증주의</b>(論理實證主義, <span style="font-size: smaller;"><a href="/wiki/%EB%8F%85%EC%9D%BC%EC%96%B4" title="독일어">독일어</a>:</span> <span lang="de" xml:lang="de">Logischer Empirismus</span>, <span style="font-size: smaller;"><a href="/wiki/%EC%98%81%EC%96%B4" title="영어">영어</a>:</span> <span lang="en" xml:lang="en">Logical positivism</span>)는 <a href="/wiki/%EA%B3%BC%ED%95%99" title="과학">과학</a>의 논리적 분석 방법을 <a href="/wiki/%EC%B2%A0%ED%95%99" title="철학">철학</a>에 적용하고자 하는 사상이다. 현대 <a href="/wiki/%EB%B6%84%EC%84%9D%EC%B2%A0%ED%95%99" title="분석철학">분석철학</a>의 주류의 하나이다.</p>
<p><a href="/wiki/1922%EB%85%84" title="1922년">1922년</a> <a href="/wiki/%EB%AA%A8%EB%A6%AC%EC%B8%A0_%EC%8A%90%EB%A6%AC%ED%81%AC" title="모리츠 슐리크">모리츠 슐리크</a>의 <a class="mw-redirect" href="/wiki/%EB%B9%88_%EB%8C%80%ED%95%99" title="빈 대학">빈 대학</a> 교수 취임과 함께 그의 세미나를 중심으로 여러 전공 분야의 학자가 모여 <a class="mw-redirect" href="/wiki/%EB%A7%88%ED%9D%90" title="마흐">마흐</a>의 실증정신(實證精神)을 계승하고 과학을 <a href="/wiki/%ED%98%95%EC%9D%B4%EC%83%81%ED%95%99" title="형이상학">형이상학</a>으로부터 해방하며 <a href="/wiki/%EC%84%B8%EA%B3%84" title="세계">세계</a>를 과학적으로 파악하는 것을 공통 목표로 삼는 사상 운동을 개시하였다. 주요 멤버로서는 슐리크 이외에는 전문 철학자가 거의 없고 과학자가 많았다. <a class="mw-redirect" href="/wiki/%EB%B9%84%ED%8A%B8%EA%B2%90%EC%8A%88%ED%83%80%EC%9D%B8" title="비트겐슈타인">비트겐슈타인</a>은 <a href="/wiki/%EB%B9%88" title="빈">빈</a> 체재 중에도 이 파와의 교섭이 없었다. 그러나 그의 저서 및 소수 멤버와의 접촉에 의한 영향은 매우 컸다. <a href="/wiki/1926%EB%85%84" title="1926년">1926년</a>에 <a class="mw-redirect" href="/wiki/%EC%B9%B4%EB%A5%B4%EB%82%98%ED%94%84" title="카르나프">카르나프</a>가 빈 대학에 초빙된 후 급속히 발전하여 그 그룹은 '마흐 협회'를 창설하고, 이윽고 그들 자신의 입장을 '논리실증주의', 그리고 협회를 '빈 학파'이라고 호칭하게 되었다. <a href="/wiki/%EB%B2%A0%EB%A5%BC%EB%A6%B0" title="베를린">베를린</a>에서 동일한 운동을 지도하고 있던 <a href="/wiki/%ED%95%9C%EC%8A%A4_%EB%9D%BC%EC%9D%B4%ED%97%A8%EB%B0%94%ED%9D%90" title="한스 라이헨바흐">한스 라이헨바흐</a>가 가담하고 <a href="/wiki/1930%EB%85%84" title="1930년">1930년</a>부터는 기관지 《인식》을 발행. 또 《통일 과학 백과전서》라는 제목 아래 일련의 연구 논문을 <a class="new" href="/w/index.php?title=%EB%85%B8%EC%9D%B4%EB%9D%BC%ED%8A%B8&amp;action=edit&amp;redlink=1" title="노이라트 (없는 문서)">노이라트</a>의 지도를 받아 <a href="/wiki/%EC%8B%9C%EC%B9%B4%EA%B3%A0" title="시카고">시카고</a>에서 발간, 통일 과학 운동을 전개했으며, 각지에서 대회를 열었다. 그러나 그들의 세계시민주의(코스모폴리터니즘)가 <a class="mw-redirect" href="/wiki/%EB%82%98%EC%B9%98%EC%8A%A4" title="나치스">나치스</a>와 맞지 않고, 1936년 슐리크의 서거 등도 있어서 이 학파는 <a href="/wiki/1938%EB%85%84" title="1938년">1938년</a>경에 실질적으로 해체되고, 멤버의 대부분은 <a href="/wiki/%EC%98%81%EA%B5%AD" title="영국">영국</a>, <a href="/wiki/%EB%AF%B8%EA%B5%AD" title="미국">미국</a>, 기타 나라로 옮겨갔다.</p>
