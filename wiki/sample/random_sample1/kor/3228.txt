<p><b>논현동</b>(論峴洞)은 <a href="/wiki/%EC%84%9C%EC%9A%B8%ED%8A%B9%EB%B3%84%EC%8B%9C" title="서울특별시">서울특별시</a> <a href="/wiki/%EA%B0%95%EB%82%A8%EA%B5%AC" title="강남구">강남구</a>에 속한 <a href="/wiki/%EB%B2%95%EC%A0%95%EB%8F%99" title="법정동">법정동</a>이다. <a href="/wiki/%EB%8F%99_(%ED%96%89%EC%A0%95_%EA%B5%AC%EC%97%AD)" title="동 (행정 구역)">행정동</a> 논현1동, 논현2동으로 나뉘어 있다. 행정동 기준으로 동쪽으로는 <a href="/wiki/%EC%82%BC%EC%84%B1%EB%8F%99_(%EA%B0%95%EB%82%A8%EA%B5%AC)" title="삼성동 (강남구)">삼성2동</a>과 <a href="/wiki/%EC%B2%AD%EB%8B%B4%EB%8F%99" title="청담동">청담동</a>, 서쪽으로는 <a href="/wiki/%EC%84%9C%EC%B4%88%EA%B5%AC" title="서초구">서초구</a> <a href="/wiki/%EB%B0%98%ED%8F%AC%EB%8F%99" title="반포동">반포1동</a>과 <a href="/wiki/%EC%9E%A0%EC%9B%90%EB%8F%99" title="잠원동">잠원동</a>, 남쪽으로는 <a href="/wiki/%EC%97%AD%EC%82%BC%EB%8F%99_(%EC%84%9C%EC%9A%B8)" title="역삼동 (서울)">역삼1동</a>과 서초구 <a href="/wiki/%EC%84%9C%EC%B4%88%EB%8F%99" title="서초동">서초4동</a>, 북쪽으로는 <a href="/wiki/%EC%8B%A0%EC%82%AC%EB%8F%99_(%EA%B0%95%EB%82%A8%EA%B5%AC)" title="신사동 (강남구)">신사동</a>과 <a href="/wiki/%EC%95%95%EA%B5%AC%EC%A0%95%EB%8F%99" title="압구정동">압구정동</a>을 접한다. 논현동의 면적은 2.72㎢, 인구는 2001년 기준으로 48,917명, 인구밀도는 17,984명/㎢이다.</p>

<h2><span class="mw-headline" id=".EC.9D.B4.EB.A6.84.EC.9D.98_.EC.9C.A0.EB.9E.98">이름의 유래</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%85%BC%ED%98%84%EB%8F%99_(%EC%84%9C%EC%9A%B8)&amp;action=edit&amp;section=1" title="부분 편집: 이름의 유래">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>논현동의 유래는 논고개에서 붙여진 이름이다. 이 고개는 지금의 <a class="new" href="/w/index.php?title=%EB%85%BC%ED%98%84%EC%84%B1%EB%8B%B9&amp;action=edit&amp;redlink=1" title="논현성당 (없는 문서)">논현성당</a> 근처이며 특히 강남우체국 위에서 반포아파트까지의 산골짜기 좌우 벌판이 논과 밭으로 연결되어 있었기 때문에 논고개라 부르게 되었다고 한다.</p>
<h2><span class="mw-headline" id=".EC.97.AD.EC.82.AC">역사</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%85%BC%ED%98%84%EB%8F%99_(%EC%84%9C%EC%9A%B8)&amp;action=edit&amp;section=2" title="부분 편집: 역사">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p><a href="/wiki/%EC%A1%B0%EC%84%A0" title="조선">조선</a>시대에는 <a href="/wiki/%EA%B2%BD%EA%B8%B0%EB%8F%84" title="경기도">경기도</a> <a href="/wiki/%EA%B4%91%EC%A3%BC%EA%B5%B0_(%EA%B2%BD%EA%B8%B0%EB%8F%84)" title="광주군 (경기도)">광주군</a> 언주면 논현리였다. <a href="/wiki/1963%EB%85%84" title="1963년">1963년</a> <a href="/wiki/1%EC%9B%94_1%EC%9D%BC" title="1월 1일">1월 1일</a> <a href="/wiki/%EC%84%9C%EC%9A%B8%ED%8A%B9%EB%B3%84%EC%8B%9C" title="서울특별시">서울특별시</a> 행정구역 확장에 따라 <a href="/wiki/%EC%84%9C%EC%9A%B8%ED%8A%B9%EB%B3%84%EC%8B%9C" title="서울특별시">서울특별시</a>로 편입되면서 <a href="/wiki/%EC%84%9C%EC%9A%B8%ED%8A%B9%EB%B3%84%EC%8B%9C" title="서울특별시">서울특별시</a> <a href="/wiki/%EC%84%B1%EB%8F%99%EA%B5%AC" title="성동구">성동구</a> 논현동이 되어 <a href="/wiki/%EC%84%B1%EB%8F%99%EA%B5%AC" title="성동구">성동구</a> 언주출장소 사평동사무소 관할이 되었다. 1970년 사평동사무소가 신사동사무소로 명칭이 변경되었고, 1973년 영동출장소 관할이 되었다가 1975년 10월 1일 <a href="/wiki/%EA%B0%95%EB%82%A8%EA%B5%AC" title="강남구">강남구</a>가 신설되면서 강남구 관할이 되었다. <a href="/wiki/1977%EB%85%84" title="1977년">1977년</a> <a href="/wiki/%EC%8B%A0%EC%82%AC%EB%8F%99_(%EA%B0%95%EB%82%A8%EA%B5%AC)" title="신사동 (강남구)">신사동</a>에서 논현동으로 분동되었으며, <a href="/wiki/1980%EB%85%84" title="1980년">1980년</a> 논현동에서 학동이 분동되었고, 1982년 4월 10일 강남지역 일대의 구획정리 사업이 완료되면서 학동 지번 전체가 논현동으로 편입되었으며, 1985년 법정동으로서의 학동은 완전히 폐지되고 <a href="/wiki/%EB%8F%99_(%ED%96%89%EC%A0%95_%EA%B5%AC%EC%97%AD)" title="동 (행정 구역)">행정동</a> 명칭만 존속하게 되었으나 1992년 명칭이 논현2동으로 바뀌었다. 논현동의 행정은 <a href="/wiki/%EB%85%BC%ED%98%84%EB%A1%9C" title="논현로">논현로</a>를 경계로 논현1동과 논현2동이 나누어 맡고 있다.</p>
<h2><span class="mw-headline" id=".ED.8A.B9.EC.A7.95">특징</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%85%BC%ED%98%84%EB%8F%99_(%EC%84%9C%EC%9A%B8)&amp;action=edit&amp;section=3" title="부분 편집: 특징">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>논현1동은 강남대로가 인접하여 유동인구가 많으며, 주거 중심 지역으로 생활 수준이 높다. 학동로에는 고급 가구 상가가 형성되어 있다.</p>
<p>논현2동은 공공기관과 업무용 빌딩, 금융기관 등이 밀집하여 고품질의 행정 수요가 요구되는 지역이다.</p>
<h2><span class="mw-headline" id=".ED.95.99.EB.8F.99">학동</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%85%BC%ED%98%84%EB%8F%99_(%EC%84%9C%EC%9A%B8)&amp;action=edit&amp;section=4" title="부분 편집: 학동">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>마을 모양이 <a class="mw-redirect" href="/wiki/%ED%95%99" title="학">학</a>과 같이 생겼다고 하여 학실(鶴室), 학곡(鶴谷), 학리(鶴里)라고 하던 <a href="/wiki/%EA%B4%91%EC%A3%BC%EA%B5%B0_(%EA%B2%BD%EA%B8%B0%EB%8F%84)" title="광주군 (경기도)">광주군</a> 언주면 학동(鶴洞)이 <a href="/wiki/1914%EB%85%84" title="1914년">1914년</a> <a href="/wiki/%EA%B4%91%EC%A3%BC%EA%B5%B0_(%EA%B2%BD%EA%B8%B0%EB%8F%84)" title="광주군 (경기도)">광주군</a> 언주면 학리로 개편하였다. <a href="/wiki/1963%EB%85%84" title="1963년">1963년</a> <a href="/wiki/%EC%84%9C%EC%9A%B8%ED%8A%B9%EB%B3%84%EC%8B%9C" title="서울특별시">서울특별시</a>에 편입되어 <a href="/wiki/%EC%84%B1%EB%8F%99%EA%B5%AC" title="성동구">성동구</a> 학동이 되고, <a href="/wiki/1975%EB%85%84" title="1975년">1975년</a> 신설 <a href="/wiki/%EA%B0%95%EB%82%A8%EA%B5%AC" title="강남구">강남구</a>에 편입되었다. <a href="/wiki/1980%EB%85%84" title="1980년">1980년</a> 행정동 학동이 설치되었으나, <a href="/wiki/1982%EB%85%84" title="1982년">1982년</a> 지번 전체를 논현동으로 편입하였고, <a href="/wiki/1985%EB%85%84" title="1985년">1985년</a> <a href="/wiki/%EB%B2%95%EC%A0%95%EB%8F%99" title="법정동">법정동</a> 학동을 폐지하기에 이르렀다. <a href="/wiki/1992%EB%85%84" title="1992년">1992년</a> <a href="/wiki/%EB%8F%99_(%ED%96%89%EC%A0%95_%EA%B5%AC%EC%97%AD)" title="동 (행정 구역)">행정동</a> 학동이 논현2동으로 개칭함으로써, 학동은 행정구역명에서 완전히 사라졌다.</p>
