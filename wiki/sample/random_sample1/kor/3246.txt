<p><b>누에바에스파냐 부왕령</b>(<span style="font-size: smaller;"><a href="/wiki/%EC%8A%A4%ED%8E%98%EC%9D%B8%EC%96%B4" title="스페인어">스페인어</a>:</span> <span lang="es" xml:lang="es">Virreinato de Nueva España</span>)은 <a href="/wiki/%EB%B6%81%EC%95%84%EB%A9%94%EB%A6%AC%EC%B9%B4" title="북아메리카">북아메리카</a>와 <a href="/wiki/%EC%95%84%EC%8B%9C%EC%95%84" title="아시아">아시아</a>-<a href="/wiki/%ED%83%9C%ED%8F%89%EC%96%91" title="태평양">태평양</a>에 위치한 <a href="/wiki/%EC%8A%A4%ED%8E%98%EC%9D%B8" title="스페인">스페인</a>의 영토 행정 단위였다. 누에바에스파냐의 영토는 오늘날 <a href="/wiki/%EB%AF%B8%EA%B5%AD_%EB%82%A8%EC%84%9C%EB%B6%80" title="미국 남서부">미국 남서부</a>, <a href="/wiki/%EB%A9%95%EC%8B%9C%EC%BD%94" title="멕시코">멕시코</a>, <a href="/wiki/%EC%A4%91%EC%95%99%EC%95%84%EB%A9%94%EB%A6%AC%EC%B9%B4" title="중앙아메리카">중앙아메리카</a>(<a href="/wiki/%ED%8C%8C%EB%82%98%EB%A7%88" title="파나마">파나마</a> 제외), <a href="/wiki/%EC%B9%B4%EB%A6%AC%EB%B8%8C_%ED%95%B4" title="카리브 해">카리브 해</a>, <a href="/wiki/%ED%95%84%EB%A6%AC%ED%95%80" title="필리핀">필리핀</a>을 아울렀다. 이곳 부왕령은 <a class="mw-redirect" href="/wiki/%EC%97%90%EC%8A%A4%ED%8C%8C%EB%83%90%EC%9D%98_%EA%B5%B0%EC%A3%BC" title="에스파냐의 군주">에스파냐 임금</a>을 대신하여 <a class="mw-redirect" href="/wiki/%EB%A9%95%EC%8B%9C%EC%BD%94_%EC%8B%9C" title="멕시코 시">멕시코 시</a>에서 <a href="/wiki/%EB%B6%80%EC%99%95" title="부왕">부왕</a>이 다스렸다. 누에바에스파냐 부왕령은 1535년부터 1821년까지 존속하였으며, <a href="/wiki/%ED%8E%98%EB%A3%A8_%EB%B6%80%EC%99%95%EB%A0%B9" title="페루 부왕령">페루 부왕령</a>과 더불어 16세기에 <a class="mw-redirect" href="/wiki/%EC%97%90%EC%8A%A4%ED%8C%8C%EB%83%90_%EC%A0%9C%EA%B5%AD" title="에스파냐 제국">에스파냐 제국</a>이 <a class="mw-disambig" href="/wiki/%EC%8B%A0%EC%84%B8%EA%B3%84" title="신세계">신세계</a>의 영토에 세운 초기 두 부왕령 가운데 하나이다. 18세기에 <a class="new" href="/w/index.php?title=%EB%88%84%EC%97%90%EB%B0%94_%EA%B7%B8%EB%9D%BC%EB%82%98%EB%8B%A4_%EB%B6%80%EC%99%95%EB%A0%B9&amp;action=edit&amp;redlink=1" title="누에바 그라나다 부왕령 (없는 문서)">누에바 그라나다 부왕령</a>과 <a class="mw-redirect" href="/wiki/%EB%A6%AC%EC%98%A4_%EB%8D%B0_%EB%9D%BC_%ED%94%8C%EB%9D%BC%ED%83%80_%EB%B6%80%EC%99%95%EB%A0%B9" title="리오 데 라 플라타 부왕령">리오 데 라 플라타 부왕령</a>이 설치되었다. 누에바에스파냐의 주요 정부는 "북부 아메리카"(América Septentrional)이라는 곳에 자리 잡고 있었다.<sup class="reference" id="cite_ref-1"><a href="#cite_note-1">[1]</a></sup><sup class="reference" id="cite_ref-2"><a href="#cite_note-2">[2]</a></sup><sup class="reference" id="cite_ref-3"><a href="#cite_note-3">[3]</a></sup><sup class="reference" id="cite_ref-4"><a href="#cite_note-4">[4]</a></sup> 1821년, <a href="/wiki/%EB%A9%95%EC%8B%9C%EC%BD%94" title="멕시코">멕시코</a>와 중앙 아메리카는 300여년의 에스파냐 지배를 뒤로하고 독립을 쟁취하였다. 그러나 <a href="/wiki/%EC%BF%A0%EB%B0%94" title="쿠바">쿠바</a>, <a href="/wiki/%ED%91%B8%EC%97%90%EB%A5%B4%ED%86%A0%EB%A6%AC%EC%BD%94" title="푸에르토리코">푸에르토리코</a>를 비롯한 <a class="new" href="/w/index.php?title=%EC%97%90%EC%8A%A4%ED%8C%8C%EB%83%90%EB%A0%B9_%EC%84%9C%EC%9D%B8%EB%8F%84&amp;action=edit&amp;redlink=1" title="에스파냐령 서인도 (없는 문서)">에스파냐령 서인도</a> 대부분과 필리핀 제도, <a href="/wiki/%EA%B4%8C" title="괌">괌</a>, <a href="/wiki/%EB%A7%88%EB%A6%AC%EC%95%84%EB%82%98_%EC%A0%9C%EB%8F%84" title="마리아나 제도">마리아나 제도</a> 등 <a href="/wiki/%EC%8A%A4%ED%8E%98%EC%9D%B8%EB%A0%B9_%EB%8F%99%EC%9D%B8%EB%8F%84" title="스페인령 동인도">에스파냐령 동인도</a>는 1898년 <a class="mw-redirect" href="/wiki/%EB%AF%B8%EA%B5%AD-%EC%97%90%EC%8A%A4%ED%8C%8C%EB%83%90_%EC%A0%84%EC%9F%81" title="미국-에스파냐 전쟁">미국-에스파냐 전쟁</a>까지 에스파냐 제국의 영토로 남아있었다.</p>
