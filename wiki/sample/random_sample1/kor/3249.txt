<h1 class="firstHeading" id="firstHeading" lang="ko">누자베스</h1>
<div class="mw-body-content" id="bodyContent">
<div id="siteSub">위키백과, 우리 모두의 백과사전.</div>
<div id="contentSub"></div>
<div class="mw-jump" id="jump-to-nav">
					이동:					<a href="#mw-head">둘러보기</a>, 					<a href="#p-search">검색</a>

<div class="mw-content-ltr" dir="ltr" id="mw-content-text" lang="ko"><script>function mfTempOpenSection(id){var block=document.getElementById("mf-section-"+id);block.className+=" open-block";block.previousSibling.className+=" open-block";}</script><p><b>누자베스</b> (Nujabes, <a href="/wiki/1974%EB%85%84" title="1974년">1974년</a> ~ <a href="/wiki/2010%EB%85%84" title="2010년">2010년</a> <a href="/wiki/2%EC%9B%94_26%EC%9D%BC" title="2월 26일">2월 26일</a><sup class="reference" id="cite_ref-nujabes_1-0"><a href="#cite_note-nujabes-1">[1]</a></sup>)는 <a href="/wiki/%EC%9D%BC%EB%B3%B8" title="일본">일본</a>의 <a href="/wiki/%ED%9E%99%ED%95%A9" title="힙합">힙합</a> <a class="mw-disambig" href="/wiki/%ED%94%84%EB%A1%9C%EB%93%80%EC%84%9C" title="프로듀서">프로듀서</a>이자 <a class="mw-redirect" href="/wiki/%EB%94%94%EC%8A%A4%ED%81%AC%EC%9E%90%ED%82%A4" title="디스크자키">DJ</a>이다. 누자베스라는 이름은 그의 실명 〈세바 준〉(seba jun,せば　じゅん)의 영어 철자를 거꾸로 읽은 것이다. 국내에서는 《<a class="new" href="/w/index.php?title=%EC%82%AC%EB%AC%B4%EB%9D%BC%EC%9D%B4_%EC%B0%B8%ED%94%84%EB%A3%A8&amp;action=edit&amp;redlink=1" title="사무라이 참프루 (없는 문서)">사무라이 참프루</a>》OST를 통해 이름을 알렸다.<sup class="reference" id="cite_ref-nujabes_1-1"><a href="#cite_note-nujabes-1">[1]</a></sup> <a href="/wiki/%EC%8B%9C%EB%B6%80%EC%95%BC" title="시부야">시부야</a>에 위치한 T 레코드와 기네스 레코드 가게의 주인이며 독립 레이블 <a class="extiw" href="https://en.wikipedia.org/wiki/Hydeout_Productions" title="en:Hydeout Productions">하이드아웃 프로덕션</a>의 설립자이다.</p>
<p><a href="/wiki/2010%EB%85%84" title="2010년">2010년</a> <a href="/wiki/2%EC%9B%94_26%EC%9D%BC" title="2월 26일">2월 26일</a> <a href="/wiki/%EC%9D%BC%EB%B3%B8" title="일본">일본</a>, <a class="mw-redirect" href="/wiki/%EB%8F%84%EC%BF%84%EB%8F%84" title="도쿄도">도쿄도</a>, <a href="/wiki/%EB%AF%B8%EB%82%98%ED%86%A0_%EA%B5%AC_(%EB%8F%84%EC%BF%84_%EB%8F%84)" title="미나토 구 (도쿄 도)">미나토 구</a>의 고속도로 근처에서 교통사고를 당하여 사망하였다. 향년 37세<sup class="reference" id="cite_ref-nujabes_1-2"><a href="#cite_note-nujabes-1">[1]</a></sup> 그의 사망은 하이드아웃 프로덕션 홈페이지에 공문으로 전해진 상태이다.<sup class="reference" id="cite_ref-nujabes_1-3"><a href="#cite_note-nujabes-1">[1]</a></sup></p>
