<p><b>제13회 <a href="/wiki/%EC%95%84%EC%8B%9C%EC%95%88_%EA%B2%8C%EC%9E%84" title="아시안 게임">아시안 게임</a></b>은 <a href="/wiki/1998%EB%85%84" title="1998년">1998년</a> <a href="/wiki/12%EC%9B%94_6%EC%9D%BC" title="12월 6일">12월 6일</a>부터 <a href="/wiki/12%EC%9B%94_20%EC%9D%BC" title="12월 20일">12월 20일</a>까지 <a href="/wiki/%ED%83%80%EC%9D%B4" title="타이">타이</a> <a href="/wiki/%EB%B0%A9%EC%BD%95" title="방콕">방콕</a>에서 열렸다. 15일 동안 열린 이 대회는 방콕이 반납된 두 차례의 대회(1970년, 1978년)를 대신 개최 이후, 처음으로 유치 경쟁에 뛰어든 후 개최권을 얻어 연 대회이다. 참가국은 모두 41개국이었다.</p>

<h2><span class="mw-headline" id=".EC.97.A0.EB.B8.94.EB.A0.98">엠블렘</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=1998%EB%85%84_%EC%95%84%EC%8B%9C%EC%95%88_%EA%B2%8C%EC%9E%84&amp;action=edit&amp;section=1" title="부분 편집: 엠블렘">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>제13회 아시안 게임의 공식 엠블렘은 아시아와 타이의 특징을 나타내는 요소들을 결합해 만들어졌다. 이 엠블렘은 아시아(Asia)와 육상(Athletes)의 머리글자인 A를 바탕으로 했다. 중앙의 <a href="/wiki/%ED%83%91" title="탑">탑</a>은 타이를 상징한다. 탑 위로는 <a href="/wiki/%EC%95%84%EC%8B%9C%EC%95%84_%EC%98%AC%EB%A6%BC%ED%94%BD_%ED%8F%89%EC%9D%98%ED%9A%8C" title="아시아 올림픽 평의회">아시아 올림픽 평의회</a>(OCA) 로고가 있다.</p>
<h2><span class="mw-headline" id=".EB.8C.80.ED.9A.8C_.EC.9D.B4.EB.AA.A8.EC.A0.80.EB.AA.A8">대회 이모저모</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=1998%EB%85%84_%EC%95%84%EC%8B%9C%EC%95%88_%EA%B2%8C%EC%9E%84&amp;action=edit&amp;section=2" title="부분 편집: 대회 이모저모">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<ul>
<li>태국 방콕에서 4번째로 열린 아시안 게임이다. 방콕은 1966년, 1970년, 1978년에도 아시안 게임을 개최하였다.</li>
<li><a href="/wiki/%EC%9D%B8%EB%8F%84" title="인도">인도</a>의 세계적인 여자 육상스타 <a class="new" href="/w/index.php?title=P._T._%EC%9A%B0%EC%83%A4&amp;action=edit&amp;redlink=1" title="P. T. 우샤 (없는 문서)">P. T. 우샤</a>(Pilavullakandi Thekkeparambil Usha)가 아시안 게임 400m를 끝으로 은퇴를 하였다. 우샤는 13세 때 인도 국가대표 육상선수로 선발돼 세계를 깜짝 놀라게 했고 숱한 기록을 남기며 ‘아시아의 마녀’라는 별명을 얻었다.</li>
<li><a href="/wiki/%EC%82%AC%EC%9A%B0%EB%94%94%EC%95%84%EB%9D%BC%EB%B9%84%EC%95%84" title="사우디아라비아">사우디아라비아</a>, <a href="/wiki/%EC%95%84%ED%94%84%EA%B0%80%EB%8B%88%EC%8A%A4%ED%83%84" title="아프가니스탄">아프가니스탄</a>, <a href="/wiki/%EC%9D%B4%EB%9D%BC%ED%81%AC" title="이라크">이라크</a> 3개국은 출전하지 않았다. 사우디아라비아는 대회 기간이 이슬람교의 금식기간인 <a href="/wiki/%EB%9D%BC%EB%A7%88%EB%8B%A8" title="라마단">라마단</a>과 겹친다는 점, 자국 내에 국가적인 큰 행사가 있다는 점을 내세워 불참했다. 아프가니스탄은 <a href="/wiki/%ED%83%88%EB%A0%88%EB%B0%98" title="탈레반">탈레반</a> 정권, 이라크는 <a href="/wiki/%EA%B1%B8%ED%94%84_%EC%A0%84%EC%9F%81" title="걸프 전쟁">걸프 전쟁</a>으로 인하여 OCA로부터 자격 정지 처분을 받은 상태였기 때문에 불참하였다. 단, 사우디아라비아는 개막식 때 기수 1명만 파견하였다.</li>
</ul>
<h2><span class="mw-headline" id=".EC.B0.B8.EA.B0.80.EA.B5.AD">참가국</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=1998%EB%85%84_%EC%95%84%EC%8B%9C%EC%95%88_%EA%B2%8C%EC%9E%84&amp;action=edit&amp;section=3" title="부분 편집: 참가국">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<table class="multicol" role="presentation" style="border-collapse: collapse; padding: 0; border: 0; background:transparent; width:100%;">
<tr>
