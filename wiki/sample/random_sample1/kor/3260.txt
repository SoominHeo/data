<p>《<b>뉴 슈퍼 마리오브라더스</b>》<sup class="reference" id="cite_ref-1"><a href="#cite_note-1">[1]</a></sup>(<span style="font-size: smaller;"><a href="/wiki/%EC%9D%BC%EB%B3%B8%EC%96%B4" title="일본어">일본어</a>:</span> <span lang="ja" xml:lang="ja">ニュースーパーマリオブラザーズ</span>, New Super Mario Bros.)는 <a href="/wiki/%EB%8B%8C%ED%85%90%EB%8F%84" title="닌텐도">닌텐도</a>가 개발한 <a href="/wiki/%EB%8B%8C%ED%85%90%EB%8F%84_DS" title="닌텐도 DS">닌텐도 DS</a>용 게임 소프트웨어이다. <a href="/wiki/%EB%AF%B8%EA%B5%AD" title="미국">미국</a>에서 <a href="/wiki/2006%EB%85%84" title="2006년">2006년</a> <a href="/wiki/5%EC%9B%94_15%EC%9D%BC" title="5월 15일">5월 15일</a> 먼저 발매했으며, <a href="/wiki/%EB%8C%80%ED%95%9C%EB%AF%BC%EA%B5%AD" title="대한민국">대한민국</a>에서는 <a href="/wiki/%EC%9D%BC%EB%B3%B8" title="일본">일본</a>과 동시에 <a href="/wiki/2006%EB%85%84" title="2006년">2006년</a> <a href="/wiki/5%EC%9B%94_25%EC%9D%BC" title="5월 25일">5월 25일</a>에 <a href="/wiki/%EB%8C%80%EC%9B%90%EC%94%A8%EC%95%84%EC%9D%B4" title="대원씨아이">대원씨아이</a>에서 발매했고 <a href="/wiki/2007%EB%85%84" title="2007년">2007년</a> <a href="/wiki/3%EC%9B%94_8%EC%9D%BC" title="3월 8일">3월 8일</a>에는 그동안 닌텐도의 한국 발매를 담당했던 대원씨아이를 대신해서 <a href="/wiki/%ED%95%9C%EA%B5%AD%EB%8B%8C%ED%85%90%EB%8F%84" title="한국닌텐도">한국닌텐도</a>에서 한글화를 해서 다시 발매했다.</p>

<h2><span class="mw-headline" id=".EC.A4.84.EA.B1.B0.EB.A6.AC">줄거리</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%89%B4_%EC%8A%88%ED%8D%BC_%EB%A7%88%EB%A6%AC%EC%98%A4%EB%B8%8C%EB%9D%BC%EB%8D%94%EC%8A%A4&amp;action=edit&amp;section=1" title="부분 편집: 줄거리">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>뉴 슈퍼 마리오브라더스의 줄거리는 <a href="/wiki/%EC%8A%88%ED%8D%BC_%EB%A7%88%EB%A6%AC%EC%98%A4%EB%B8%8C%EB%9D%BC%EB%8D%94%EC%8A%A4" title="슈퍼 마리오브라더스">슈퍼 마리오브라더스</a> 그리고 다른 마리오 게임들과 같다. 납치된 피치공주를 <a class="mw-redirect" href="/wiki/%EC%BF%A0%ED%8C%8C_%EC%A3%BC%EB%8B%88%EC%96%B4" title="쿠파 주니어">쿠파 주니어</a>와 <a href="/wiki/%EC%BF%A0%ED%8C%8C_(%EA%B2%8C%EC%9E%84_%EC%BA%90%EB%A6%AD%ED%84%B0)" title="쿠파 (게임 캐릭터)">쿠파</a>로부터 구하기 위해 <a href="/wiki/%EB%A7%88%EB%A6%AC%EC%98%A4" title="마리오">마리오</a>는 여덟 개의 월드(<a href="/wiki/%EC%B4%88%EC%9B%90" title="초원">초원</a>, <a href="/wiki/%EC%82%AC%EB%A7%89" title="사막">사막</a>, <a href="/wiki/%ED%95%B4%EB%B3%80" title="해변">해변</a>, <a class="mw-redirect" href="/wiki/%EC%A0%95%EA%B8%80" title="정글">정글</a>, <a href="/wiki/%EB%B9%99%ED%95%98" title="빙하">빙하</a>, <a href="/wiki/%EC%82%B0" title="산">산</a>, <a href="/wiki/%ED%95%98%EB%8A%98" title="하늘">하늘</a>, <a href="/wiki/%ED%99%94%EC%82%B0" title="화산">화산</a>)(쿠파왕국)를 탐험하게 된다.</p>
<h2><span class="mw-headline" id=".EA.B8.B0.EC.88.A0">기술</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%89%B4_%EC%8A%88%ED%8D%BC_%EB%A7%88%EB%A6%AC%EC%98%A4%EB%B8%8C%EB%9D%BC%EB%8D%94%EC%8A%A4&amp;action=edit&amp;section=2" title="부분 편집: 기술">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>이 게임에서 마리오가 사용할 수 있는 기술은 <a href="/wiki/%EC%8A%88%ED%8D%BC_%EB%A7%88%EB%A6%AC%EC%98%A4%EB%B8%8C%EB%9D%BC%EB%8D%94%EC%8A%A4" title="슈퍼 마리오브라더스">슈퍼 마리오브라더스</a>에서 보였던 점프와 대시뿐만 아니라, <a href="/wiki/%EC%8A%88%ED%8D%BC_%EB%A7%88%EB%A6%AC%EC%98%A4_64" title="슈퍼 마리오 64">슈퍼 마리오 64</a>에서 등장하였던 이·삼단 점프, 벽차기, 엉덩이 찍기 또한 등장하였다.</p>
<h2><span class="mw-headline" id=".EC.95.84.EC.9D.B4.ED.85.9C">아이템</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%89%B4_%EC%8A%88%ED%8D%BC_%EB%A7%88%EB%A6%AC%EC%98%A4%EB%B8%8C%EB%9D%BC%EB%8D%94%EC%8A%A4&amp;action=edit&amp;section=3" title="부분 편집: 아이템">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>게임 도중에는 몇 가지 아이템을 습득할 수 있다. 예를 들어 거대버섯을 먹으면 마리오는 거대해져 화면을 꽉 채우며 마리오는 무적 상태가 되기 때문에 바로 전진만 하면 된다.(단, 절대 부술 수 없는 것(바위, 땅, 일부 인공 구조물)이 있으면 전진이 거의 불가능하며, 거대마리오 크기 이상의 구멍이 있으면 점프를 하여 넘어가야 한다.)부순 물건만큼 화면 위에 있는 게이지가 차는데, 차는 만큼 1-UP버섯을 준다. 반대로 땅콩버섯의 경우 마리오를 굉장히 작게 만들어 버리는데, 이 경우 마리오는 높은 점프력은 물론 굉장히 작은 토관(파이프)에 들어갈 수도 있다. 또 대시 버튼을 누르고 있으면 물위를 걸을 수 있다. 또 등껍질마리오의 경우에는 대쉬버튼을 누르고 있으면 일반마리오로는 뚫을 수 없는 벽돌을 뚫게 된다. 스타를 먹으면 일정시간동안 무적상태가 된다. 일부 맵에서는 스타를 먹으면 블루 코인이 생성되는 스테이지도 있다.</p>
