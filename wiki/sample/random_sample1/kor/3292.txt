<p><b>뉴질랜드 축구 선수권 대회</b>(<span lang="en" xml:lang="en">New Zealand Football Championship</span>, 약칭 <b>NZFC</b>)는 <a href="/wiki/%EB%89%B4%EC%A7%88%EB%9E%9C%EB%93%9C" title="뉴질랜드">뉴질랜드</a>의 <a href="/wiki/%EC%84%B8%EB%AF%B8%ED%94%84%EB%A1%9C" title="세미프로">세미프로</a> <a href="/wiki/%EC%B6%95%EA%B5%AC" title="축구">축구</a> <a href="/wiki/%EB%A6%AC%EA%B7%B8_(%EC%8A%A4%ED%8F%AC%EC%B8%A0)" title="리그 (스포츠)">리그</a>이다. <a class="new" href="/w/index.php?title=2004-05_%EB%89%B4%EC%A7%88%EB%9E%9C%EB%93%9C_%EC%B6%95%EA%B5%AC_%EC%84%A0%EC%88%98%EA%B6%8C_%EB%8C%80%ED%9A%8C&amp;action=edit&amp;redlink=1" title="2004-05 뉴질랜드 축구 선수권 대회 (없는 문서)">2004-05</a>시즌이 첫 시즌이며, 2010-11시즌부터 ASB 뱅크의 후원을 받아 <b>ASB 프리미어십</b>으로 대회 이름을 바꾸었다. 총 아홉 개 팀이 참가하며, 정규리그가 끝난 후에 상위 네 개 팀이 <b>그랜드 파이널</b>이라는 이름으로 <a href="/wiki/%ED%94%8C%EB%A0%88%EC%9D%B4%EC%98%A4%ED%94%84" title="플레이오프">플레이오프</a>를 벌여 우승 팀을 결정한다. <a href="/wiki/%EC%8A%B9%EA%B0%95%EC%A0%9C" title="승강제">승강제</a>는 아직 실시하지 않고 있다. <a href="/wiki/%EC%98%A4%EC%8A%A4%ED%8A%B8%EB%A0%88%EC%9D%BC%EB%A6%AC%EC%95%84" title="오스트레일리아">오스트레일리아</a> <a class="mw-redirect" href="/wiki/A-%EB%A6%AC%EA%B7%B8" title="A-리그">A-리그</a>에 참가하는 <a href="/wiki/%EC%9B%B0%EB%A7%81%ED%84%B4_%ED%94%BC%EB%8B%89%EC%8A%A4_FC" title="웰링턴 피닉스 FC">웰링턴 피닉스 FC</a>는 리저브 팀을 이 리그에 참가시키고,과거에 <a class="mw-redirect" href="/wiki/A-%EB%A6%AC%EA%B7%B8" title="A-리그">A-리그</a>에 참가했던 <a href="/wiki/%EB%89%B4%EC%A7%88%EB%9E%9C%EB%93%9C_%EB%82%98%EC%9D%B4%EC%B8%A0_FC" title="뉴질랜드 나이츠 FC">뉴질랜드 나이츠 FC</a>는 현재 뉴질랜드 축구 선수권에 참여하지 않고 있다.</p>

<h2><span class="mw-headline" id=".EA.B5.AC.EC.84.B1">구성</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%89%B4%EC%A7%88%EB%9E%9C%EB%93%9C_%EC%B6%95%EA%B5%AC_%EC%84%A0%EC%88%98%EA%B6%8C_%EB%8C%80%ED%9A%8C&amp;action=edit&amp;section=1" title="부분 편집: 구성">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>ASB 프리미어십은 정규 시즌과 그랜드 파이널로 구분되는데, 총 참가하는 9개 팀이 다른 팀과 각각 2번씩 <a href="/wiki/%ED%99%88_%EC%95%A4_%EC%96%B4%EC%9B%A8%EC%9D%B4" title="홈 앤 어웨이">홈 앤 어웨이</a> 방식으로 경기를 펼쳐 순위를 가린다. 정규 시즌이 끝난 후, 상위 4개 팀은 <a href="/wiki/%ED%94%8C%EB%A0%88%EC%9D%B4%EC%98%A4%ED%94%84" title="플레이오프">플레이오프</a>에 진출한다. 정규 시즌 1위와 4위, 2위와 3위가 각각 [홈 앤 어웨이]]의 방식으로 준결승전을 치르고, 준결승전의 승자끼리 <b>그랜드 파이널</b>이라는 이름으로 단판으로 결승전을 치른다.</p>
<h3><span class="mw-headline" id=".EA.B5.AC.EC.84.B1.EC.9D.98_.EB.B3.80.ED.99.94">구성의 변화</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%89%B4%EC%A7%88%EB%9E%9C%EB%93%9C_%EC%B6%95%EA%B5%AC_%EC%84%A0%EC%88%98%EA%B6%8C_%EB%8C%80%ED%9A%8C&amp;action=edit&amp;section=2" title="부분 편집: 구성의 변화">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p><a href="/wiki/OFC_%EC%B1%94%ED%94%BC%EC%96%B8%EC%8A%A4%EB%A6%AC%EA%B7%B8" title="OFC 챔피언스리그">OFC 챔피언스리그</a> 진출권은 기본적으로 정규 시즌 우승팀과 그랜드 파이널 우승팀이 가지지만, 정규 시즌 우승팀이 그랜드 파이널에서도 우승할 경우 그 진출권은 정규 시즌 2위팀에게 간다.</p>

<h2><span class="mw-headline" id=".EB.A6.AC.EA.B7.B8_.EC.B0.B8.EA.B0.80.ED.8C.80">리그 참가팀</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%89%B4%EC%A7%88%EB%9E%9C%EB%93%9C_%EC%B6%95%EA%B5%AC_%EC%84%A0%EC%88%98%EA%B6%8C_%EB%8C%80%ED%9A%8C&amp;action=edit&amp;section=3" title="부분 편집: 리그 참가팀">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>다음 9개 팀은 현재 <a href="/wiki/%EB%89%B4%EC%A7%88%EB%9E%9C%EB%93%9C" title="뉴질랜드">뉴질랜드</a>에서 <a class="new" href="/w/index.php?title=2014-15_ASB_%ED%94%84%EB%A6%AC%EB%AF%B8%EC%96%B4%EC%8B%AD&amp;action=edit&amp;redlink=1" title="2014-15 ASB 프리미어십 (없는 문서)">2014-15 ASB 프리미어십</a>에 참가 중인 팀이다. 이 리그는 유럽 축구와는 다르게 <a href="/wiki/%EC%8A%B9%EA%B0%95%EC%A0%9C" title="승강제">승강제</a>가 아직 존재하지 않는다. 그리고 이 리그는 <a href="/wiki/%EC%98%A4%EC%8A%A4%ED%8A%B8%EB%A0%88%EC%9D%BC%EB%A6%AC%EC%95%84" title="오스트레일리아">오스트레일리아</a>의 <a href="/wiki/A%EB%A6%AC%EA%B7%B8" title="A리그">A리그</a>와 <a href="/wiki/%EB%AF%B8%EA%B5%AD" title="미국">미국</a>의 <a href="/wiki/%EB%A9%94%EC%9D%B4%EC%A0%80_%EB%A6%AC%EA%B7%B8_%EC%82%AC%EC%BB%A4" title="메이저 리그 사커">메이저 리그 사커</a>와 비슷한 체계이다.</p>
