<p><b>뉴턴 르로이 "뉴트" 깅리치</b>(<span style="font-size: smaller;"><a href="/wiki/%EC%98%81%EC%96%B4" title="영어">영어</a>:</span> <span lang="en" xml:lang="en">Newton Leroy "Newt" Gingrich</span>, <a href="/wiki/1943%EB%85%84" title="1943년">1943년</a> <a href="/wiki/6%EC%9B%94_17%EC%9D%BC" title="6월 17일">6월 17일</a> ~ )는 <a href="/wiki/%EB%AF%B8%EA%B5%AD" title="미국">미국</a>의 정치인이다.</p>
<h2><span class="mw-headline" id=".EC.9D.B8.EB.AC.BC">인물</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%89%B4%ED%8A%B8_%EA%B9%85%EB%A6%AC%EC%B9%98&amp;action=edit&amp;section=1" title="부분 편집: 인물">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p><a href="/wiki/%ED%8E%9C%EC%8B%A4%EB%B2%A0%EC%9D%B4%EB%8B%88%EC%95%84_%EC%A3%BC" title="펜실베이니아 주">펜실베이니아 주</a> <a href="/wiki/%ED%95%B4%EB%A6%AC%EC%8A%A4%EB%B2%84%EA%B7%B8" title="해리스버그">해리스버그</a>에서 태어났으며, 원래 성은 <b>맥퍼슨</b>(McPherson)이다. 모친의 재혼 후 계부의 집안에 입적되어 깅리치라는 새 성을 얻었으며, <a href="/wiki/%EC%A1%B0%EC%A7%80%EC%95%84_%EC%A3%BC" title="조지아 주">조지아 주</a>에서 성장하였다. <a href="/wiki/%EC%97%90%EB%AA%A8%EB%A6%AC_%EB%8C%80%ED%95%99%EA%B5%90" title="에모리 대학교">에모리 대학교</a>와 <a href="/wiki/%ED%88%B4%EB%A0%88%EC%9D%B8_%EB%8C%80%ED%95%99%EA%B5%90" title="툴레인 대학교">툴레인 대학교</a>에서 사학 석·박사학위를 받았으며, <a href="/wiki/1970%EB%85%84" title="1970년">1970년</a>부터 <a class="new" href="/w/index.php?title=%EC%9B%A8%EC%8A%A4%ED%8A%B8%EC%A1%B0%EC%A7%80%EC%95%84_%EB%8C%80%ED%95%99%EA%B5%90&amp;action=edit&amp;redlink=1" title="웨스트조지아 대학교 (없는 문서)">웨스트조지아 대학교</a>에서 교수로 일했다. <a href="/wiki/%EA%B3%B5%ED%99%94%EB%8B%B9_(%EB%AF%B8%EA%B5%AD)" title="공화당 (미국)">공화당</a> 소속으로 <a href="/wiki/1974%EB%85%84" title="1974년">1974년</a>, <a href="/wiki/1976%EB%85%84" title="1976년">1976년</a> 두 차례 연방 하원의원 선거에 낙선한 후 <a href="/wiki/1978%EB%85%84" title="1978년">1978년</a> 당선되어 <a href="/wiki/1979%EB%85%84" title="1979년">1979년</a>부터 하원의원으로 재직했다. <a href="/wiki/%EB%B3%B4%EC%88%98%EC%A3%BC%EC%9D%98" title="보수주의">보수주의</a>를 주도하며 이름이 알려졌고, <a href="/wiki/1988%EB%85%84" title="1988년">1988년</a> <a href="/wiki/%EB%AF%BC%EC%A3%BC%EB%8B%B9_(%EB%AF%B8%EA%B5%AD)" title="민주당 (미국)">민주당</a> 출신의 <a class="new" href="/w/index.php?title=%EC%A7%90_%EB%9D%BC%EC%9D%B4%ED%8A%B8&amp;action=edit&amp;redlink=1" title="짐 라이트 (없는 문서)">짐 라이트</a> 하원의장이 수많은 독직사건에 연루되어 있다고 주장, 특별검사 임명을 강력하게 밀어붙여 전국적으로 큰 주목을 받았고, 결국 라이트 하원의장은 <a href="/wiki/1989%EB%85%84" title="1989년">1989년</a> 6월에 의장직을 사임하였다<sup class="reference" id="cite_ref-1"><a href="#cite_note-1">[1]</a></sup>. 그에 앞서 깅리치는 공화당 하원 원내총무로 선출되었다. <a href="/wiki/1993%EB%85%84" title="1993년">1993년</a> <a href="/wiki/%EB%B9%8C_%ED%81%B4%EB%A6%B0%ED%84%B4" title="빌 클린턴">빌 클린턴</a> 대통령이 취임한 후 클린턴 정부를 강하게 비판하기 시작했다.</p>
<p><a href="/wiki/1994%EB%85%84" title="1994년">1994년</a> 중간 선거를 앞두고 ‘미국과의 계약’이라는 보수주의적 공약 입안을 주도하여 인기를 모아 공화당이 하원 선거에서 40년 만에 승리하는 데 크게 기여하였다. 그가 주도한 공약으로 공화당의 승리는 미국의 변혁을 이끈 깅리치 혁명으로 큰 주목을 받았고, 그는 새 하원의장으로 선출되어 <a href="/wiki/1995%EB%85%84" title="1995년">1995년</a> 1월 40년만의 공화당 출신 하원의장이 되었다. 그러나 재임 기간 동안 클린턴 정부와의 불편한 관계와 그 자신의 윤리적인 문제로 계속 논란의 대상이 되었다. 이러한 끊임없는 대립으로 <a href="/wiki/1996%EB%85%84" title="1996년">1996년</a> 하원 선거에서는 어려움이 예상되었으나, 9석만 잃고 다수당 지위를 유지, 하원의장으로 계속 재임했다. 그 후 ‘<a href="/wiki/%EB%AA%A8%EB%8B%88%EC%B9%B4_%EB%A5%B4%EC%9C%88%EC%8A%A4%ED%82%A4" title="모니카 르윈스키">르윈스키</a> 스캔들’로 클린턴 대통령의 성추문 논란이 계속되자 그는 클린턴 대통령의 거취 문제를 법적으로 심판하자는 논리로 탄핵을 추진하고 이를 <a href="/wiki/1998%EB%85%84" title="1998년">1998년</a> 11월 중간 선거에 정략적으로 이용했다.</p>
<p>선거에서 공화당은 하원 의석 5석을 잃고 다수당 지위는 유지했으나 선거 패배에 대한 책임을 추궁당하고, 그의 개인적인 문제가 불거지면서 공화당 내에서도 비판의 목소리가 커지면서 영향력을 급속히 잃었고 곧바로 하원의장직 사임을 발표하였다. 그러나 클린턴 대통령에 대한 탄핵소추안은 <a href="/wiki/12%EC%9B%94_20%EC%9D%BC" title="12월 20일">12월 20일</a>, 하원을 통과했다<sup class="reference" id="cite_ref-2"><a href="#cite_note-2">[2]</a></sup>. 깅리치는 <a href="/wiki/1999%EB%85%84" title="1999년">1999년</a> <a href="/wiki/1%EC%9B%94_3%EC%9D%BC" title="1월 3일">1월 3일</a>, 하원의장과 의원직을 동시에 내놓고 20년 간의 하원의원 생활을 마감하였다. 정계 은퇴 후 깅리치는 강연과 저술 활동으로 보수적인 견해를 계속 주장하고 있다. <a href="/wiki/2008%EB%85%84_%EB%AF%B8%EA%B5%AD_%EB%8C%80%ED%86%B5%EB%A0%B9_%EC%84%A0%EA%B1%B0" title="2008년 미국 대통령 선거">2008년 대통령 선거</a>에서 <a href="/wiki/%EB%B2%84%EB%9D%BD_%EC%98%A4%EB%B0%94%EB%A7%88" title="버락 오바마">버락 오바마</a> 후보의 우세를 보며 공화당의 개혁을 촉구했으며, <a href="/wiki/2012%EB%85%84_%EB%AF%B8%EA%B5%AD_%EB%8C%80%ED%86%B5%EB%A0%B9_%EC%84%A0%EA%B1%B0" title="2012년 미국 대통령 선거">2012년 대통령 선거</a> 출마를 고려하기도 했다.</p>
