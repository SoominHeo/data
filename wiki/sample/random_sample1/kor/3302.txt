<p><br/>
<b>뉴햄프셔 주</b>(<span style="font-size: smaller;"><a href="/wiki/%EC%98%81%EC%96%B4" title="영어">영어</a>:</span> <span lang="en" xml:lang="en">State of New Hampshire</span>)는 <a href="/wiki/%EB%89%B4%EC%9E%89%EA%B8%80%EB%9E%9C%EB%93%9C" title="뉴잉글랜드">뉴잉글랜드</a> 북부에 있는 <a href="/wiki/%EB%AF%B8%EA%B5%AD%EC%9D%98_%EC%A3%BC" title="미국의 주">미국의 주</a>이다. 북쪽으로 <a href="/wiki/%EC%BA%90%EB%82%98%EB%8B%A4" title="캐나다">캐나다</a>의 <a href="/wiki/%ED%80%98%EB%B2%A1_%EC%A3%BC" title="퀘벡 주">퀘벡 주</a>와 <a href="/wiki/%EA%B5%AD%EA%B2%BD" title="국경">국경</a>을 접하며, 동쪽으로 <a href="/wiki/%EB%A9%94%EC%9D%B8_%EC%A3%BC" title="메인 주">메인 주</a>와 <a href="/wiki/%EB%8C%80%EC%84%9C%EC%96%91" title="대서양">대서양</a>, 남쪽으로 <a href="/wiki/%EB%A7%A4%EC%82%AC%EC%B6%94%EC%84%B8%EC%B8%A0_%EC%A3%BC" title="매사추세츠 주">매사추세츠 주</a>, 서쪽으로 <a href="/wiki/%EB%B2%84%EB%AA%AC%ED%8A%B8_%EC%A3%BC" title="버몬트 주">버몬트 주</a>와 접한다.</p>
<p>미국 50개 주 가운데 면적으로 43번째, 인구로 41번째의 작은 주지만 미국 대통령 예비 선거 중 가장 먼저 열리는 뉴햄프셔 예비 선거와 <a class="mw-redirect" href="/wiki/%EC%95%84%EC%9D%B4%EB%B9%84%EB%A6%AC%EA%B7%B8" title="아이비리그">아이비리그</a> 소속 대학의 하나인 <a href="/wiki/%EB%8B%A4%ED%8A%B8%EB%A8%B8%EC%8A%A4_%EB%8C%80%ED%95%99%EA%B5%90" title="다트머스 대학교">다트머스 대학교</a>로 유명하다. 14대 대통령 <a href="/wiki/%ED%94%84%EB%9E%AD%ED%81%B4%EB%A6%B0_%ED%94%BC%EC%96%B4%EC%8A%A4" title="프랭클린 피어스">프랭클린 피어스</a>의 고향이다.</p>

<h2><span class="mw-headline" id=".EC.97.AD.EC.82.AC">역사</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%89%B4%ED%96%84%ED%94%84%EC%85%94_%EC%A3%BC&amp;action=edit&amp;section=1" title="부분 편집: 역사">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<h3><span class="mw-headline" id=".EC.B4.88.EA.B8.B0_.EC.97.AD.EC.82.AC">초기 역사</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%89%B4%ED%96%84%ED%94%84%EC%85%94_%EC%A3%BC&amp;action=edit&amp;section=2" title="부분 편집: 초기 역사">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p><a href="/wiki/%EC%9C%A0%EB%9F%BD%EC%9D%B8" title="유럽인">유럽인</a>들이 오기 전에 지금의 뉴햄프셔에는 대략 5,000명의 <a class="mw-redirect" href="/wiki/%EC%95%84%EB%A9%94%EB%A6%AC%EC%B9%B4_%EC%9D%B8%EB%94%94%EC%96%B8" title="아메리카 인디언">아메리카 인디언</a>들이 아마 살았을 것이다. 대부분은 <a href="/wiki/%EC%95%8C%EA%B3%A4%ED%82%A8_%EC%A1%B1" title="알곤킨 족">알곤킨 족</a>에 속하는 <a href="/wiki/%EC%95%84%EB%B2%A0%EB%82%98%ED%82%A4_%EC%A1%B1" title="아베나키 족">아베나키 족</a>과 페나쿡 족의 2개의 분파였다. 아베나키 분파는 오시피 족과 페쿼케트 족을 포함하였다. 페나쿡 분파는 애모스케아그 족, 내슈아 족, 피스캐터콰 족, 수히건 족과 스퀌스코트 족을 포함하였다. 뉴햄프셔 인디언들은 나무껍질로 오막집을 지었다. 그들은 <a href="/wiki/%EC%82%AC%EB%83%A5" title="사냥">사냥</a>과 <a href="/wiki/%EB%82%9A%EC%8B%9C" title="낚시">낚시</a>를 하거나 <a href="/wiki/%EC%98%A5%EC%88%98%EC%88%98" title="옥수수">옥수수</a>의 작은 농작을 하였다. 종족들은 평화롭게 함께 살았으나 가끔 자신들의 공통의 적인 <a class="mw-redirect" href="/wiki/%EC%9D%B4%EB%A1%9C%EC%BF%BC%EC%9D%B4_%EC%96%B4%EC%A1%B1" title="이로쿼이 어족">이로쿼이 어족</a>에 대항하여 전쟁을 벌였다.</p>
<h3><span class="mw-headline" id=".ED.83.90.ED.97.98">탐험</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%89%B4%ED%96%84%ED%94%84%EC%85%94_%EC%A3%BC&amp;action=edit&amp;section=3" title="부분 편집: 탐험">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>역사가들은 뉴햄프셔에 도달한 첫 유럽인들이 누구였는지 모른다. 그러나 지역의 넓은 탐험은 1600년대 초반에 시작되었다. <a href="/wiki/1603%EB%85%84" title="1603년">1603년</a> <a href="/wiki/%EC%9E%89%EA%B8%80%EB%9E%9C%EB%93%9C" title="잉글랜드">잉글랜드</a>의 <a class="new" href="/w/index.php?title=%EB%A7%88%ED%8B%B4_%ED%94%84%EB%A7%81&amp;action=edit&amp;redlink=1" title="마틴 프링 (없는 문서)">마틴 프링</a>이 상선을 타고 피스캐터콰 강을 항해하였다. 프링과 그의 선원이 현재의 <a href="/wiki/%ED%8F%AC%EC%B8%A0%EB%A8%B8%EC%8A%A4_(%EB%89%B4%ED%96%84%ED%94%84%EC%85%94_%EC%A3%BC)" title="포츠머스 (뉴햄프셔 주)">포츠머스</a>에 상륙하였을 것이다. <a href="/wiki/%ED%94%84%EB%9E%91%EC%8A%A4" title="프랑스">프랑스</a>의 탐험가 <a href="/wiki/%EC%82%AC%EB%AE%88%EC%97%98_%EB%93%9C_%EC%83%B9%ED%94%8C%EB%9E%AD" title="사뮈엘 드 샹플랭">사뮈엘 드 샹플랭</a>이 <a href="/wiki/1605%EB%85%84" title="1605년">1605년</a>에 뉴햄프셔 해안에 상륙하였다. <a href="/wiki/1614%EB%85%84" title="1614년">1614년</a> 잉글랜드의 선장 <a class="mw-disambig" href="/wiki/%EC%A1%B4_%EC%8A%A4%EB%AF%B8%EC%8A%A4" title="존 스미스">존 스미스</a>가 숄즈 섬에 도달하여, 스미스 섬으로 이름을 붙였다.</p>
<h3><span class="mw-headline" id=".EC.A0.95.EC.B0.A9">정착</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%89%B4%ED%96%84%ED%94%84%EC%85%94_%EC%A3%BC&amp;action=edit&amp;section=4" title="부분 편집: 정착">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p><a href="/wiki/1619%EB%85%84" title="1619년">1619년</a> 잉글랜드의 <a class="mw-disambig" href="/wiki/%EC%A0%9C%EC%9E%84%EC%8A%A4_1%EC%84%B8" title="제임스 1세">제임스 1세</a>는 아메리카의 정착에 용기를 주는 데 뉴잉글랜드 회의를 창립하였다. 회의는 <a class="mw-redirect" href="/wiki/%EB%8D%B0%EC%9D%B4%EB%B9%84%EB%93%9C_%ED%86%B0%EC%8A%A8" title="데이비드 톰슨">데이비드 톰슨</a>이 오늘날 뉴햄프셔에 상륙하는 데 인정하였다. <a href="/wiki/1623%EB%85%84" title="1623년">1623년</a> 톰슨과 그의 부하들이 현재 라이에 속하는 오디온즈포인트에 정착하였다. <a class="new" href="/w/index.php?title=%EC%97%90%EB%93%9C%EC%9B%8C%EB%93%9C_%ED%9E%90%ED%8A%BC&amp;action=edit&amp;redlink=1" title="에드워드 힐튼 (없는 문서)">에드워드 힐튼</a>은 1620년대에 현재 도버에 속하는 힐튼즈포인트에 정착하였다. 어떤 역사가들은 이 정착들이 같은 시기에 시작되었다고 믿는다. 다른이들은 힐튼의 정착이 나중에 시작되었다고 생각한다. 다른 정착지들은 <a href="/wiki/1630%EB%85%84" title="1630년">1630년</a>에 설립된 스트로베리뱅크(현재의 포츠머스)와 <a href="/wiki/1638%EB%85%84" title="1638년">1638년</a>에 창립된 엑스터와 햄프턴을 포함하였다.</p>
<p><a href="/wiki/1622%EB%85%84" title="1622년">1622년</a> 뉴잉글랜드 회의는 <a class="new" href="/w/index.php?title=%EC%A1%B4_%EB%A9%94%EC%9D%B4%EC%8A%A8&amp;action=edit&amp;redlink=1" title="존 메이슨 (없는 문서)">존 메이슨</a>과 <a class="new" href="/w/index.php?title=%ED%8E%98%EB%A5%B4%EB%94%94%EB%82%9C%EB%8F%84_%EA%B3%A0%EB%A5%B4%ED%97%A4%EC%8A%A4&amp;action=edit&amp;redlink=1" title="페르디난도 고르헤스 (없는 문서)">페르디난도 고르헤스</a> 경에게 오늘날 뉴햄프셔와 <a class="mw-disambig" href="/wiki/%EB%A9%94%EC%9D%B8" title="메인">메인</a>에 있는 대지의 큰 지대를 주었다. <a href="/wiki/1629%EB%85%84" title="1629년">1629년</a> 대지는 두 사람들 사이에 나뉘었다. 메이슨은 자신의 고향 잉글랜드의 <a href="/wiki/%ED%96%84%ED%94%84%EC%85%94_%EC%A3%BC" title="햄프셔 주">햄프셔 주</a>을 따서 자신의 대지를 뉴햄프셔라고 이름을 지었다. <a href="/wiki/1641%EB%85%84" title="1641년">1641년</a> 뉴햄프셔는 <a class="mw-redirect" href="/wiki/%EB%A7%A4%EC%82%AC%EC%B6%94%EC%84%B8%EC%B8%A0" title="매사추세츠">매사추세츠</a>의 일부로 만들어졌다. 그러나 <a href="/wiki/1680%EB%85%84" title="1680년">1680년</a> 잉글랜드의 <a href="/wiki/%EC%B0%B0%EC%8A%A4_2%EC%84%B8" title="찰스 2세">찰스 2세</a>는 뉴햄프셔를 갈라진 식민지로 만들었다. 국왕은 <a class="new" href="/w/index.php?title=%EC%A1%B4_%EC%BB%A4%ED%8A%B8&amp;action=edit&amp;redlink=1" title="존 커트 (없는 문서)">존 커트</a>를 뉴햄프셔의 초대 식민지 총독으로 임명하였다.</p>
<h3><span class="mw-headline" id=".ED.94.84.EB.A0.8C.EC.B9.98_.EC.9D.B8.EB.94.94.EC.96.B8_.EC.A0.84.EC.9F.81">프렌치 인디언 전쟁</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%89%B4%ED%96%84%ED%94%84%EC%85%94_%EC%A3%BC&amp;action=edit&amp;section=5" title="부분 편집: 프렌치 인디언 전쟁">편집</a><span class="mw-editsection-bracket">]</span></span></h3>

<p><a href="/wiki/%ED%94%84%EB%A0%8C%EC%B9%98_%EC%9D%B8%EB%94%94%EC%96%B8_%EC%A0%84%EC%9F%81" title="프렌치 인디언 전쟁">프렌치 인디언 전쟁</a>은 뉴햄프셔와 뉴잉글랜드의 나머지에서 싸워졌으며, <a href="/wiki/1689%EB%85%84" title="1689년">1689년</a>부터 <a href="/wiki/1763%EB%85%84" title="1763년">1763년</a>까지 중단되었다가 다시 시작되었다. 프랑스군과 그들의 인디언 동맹들은 영국군으로부터 지역의 통치를 얻기위하여 전투를 벌였다. 인디언들의 안내를 받으면서 프랑스군은 <a href="/wiki/%EC%BA%90%EB%82%98%EB%8B%A4" title="캐나다">캐나다</a>로부터 밀려 내려왔다. 뉴햄프셔의 로저스의 유격 대원이라 불리는 군인들의 단체의 지도자 <a class="new" href="/w/index.php?title=%EB%A1%9C%EB%B2%84%ED%8A%B8_%EB%A1%9C%EC%A0%80%EC%8A%A4&amp;action=edit&amp;redlink=1" title="로버트 로저스 (없는 문서)">로버트 로저스</a>와 <a class="new" href="/w/index.php?title=%EC%A1%B4_%EC%8A%A4%ED%83%80%ED%81%AC&amp;action=edit&amp;redlink=1" title="존 스타크 (없는 문서)">존 스타크</a>는 전쟁이 일어나는 동안에 식민지의 군사 지도자들로서 명성을 얻었다.</p>
<p>식민주의적 뉴햄프셔는 도시의 사회였다. 주민들의 대부분은 악조건을 제거하고, 주택들을 짓고, 식량을 재배하는 데 계속 바쁘게 지냈다. <a href="/wiki/1767%EB%85%84" title="1767년">1767년</a> 뉴햄프셔가 그 첫 인구 조사를 가졌을 때에 식민지에는 52,700명의 주민들이 있었다. <a href="/wiki/%EC%98%81%EA%B5%AD" title="영국">영국</a>의 국왕은 총독과 그 추밀원을 임명하였다. 그러나 주민들은 식민지의 정세에서 자신들을 대표하는 데 의원들을 선출하였다.</p>
<h3><span class="mw-headline" id=".EB.8F.85.EB.A6.BD_.EC.A0.84.EC.9F.81">독립 전쟁</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%89%B4%ED%96%84%ED%94%84%EC%85%94_%EC%A3%BC&amp;action=edit&amp;section=6" title="부분 편집: 독립 전쟁">편집</a><span class="mw-editsection-bracket">]</span></span></h3>

<div class="thumbinner" style="width:222px;"><a class="image" href="/wiki/%ED%8C%8C%EC%9D%BC:Nhcolony.png"><img alt="" class="thumbimage" data-file-height="500" data-file-width="480" height="229" src="//upload.wikimedia.org/wikipedia/commons/thumb/2/28/Nhcolony.png/220px-Nhcolony.png" srcset="//upload.wikimedia.org/wikipedia/commons/thumb/2/28/Nhcolony.png/330px-Nhcolony.png 1.5x, //upload.wikimedia.org/wikipedia/commons/thumb/2/28/Nhcolony.png/440px-Nhcolony.png 2x" width="220"/></a>

<p>1760년대 동안에 영국은 뉴햄프셔와 다른 아메리카 식민지들에서 불안을 일으킨 법률들의 시리즈를 통과시켰다. 이 법률들의 대부분은 엄격한 세금을 과하거나 식민지의 교역을 제한시켰다. <a href="/wiki/%EC%A1%B4_%EC%84%A4%EB%A6%AC%EB%B2%88" title="존 설리번">존 설리번</a>이 이끄는 뉴햄프셔 주민들의 단체는 뉴캐슬에서 영국군의 요새로부터 군사적 공급물들을 포착하였다. 이 사건은 식민주의자들이 영국에 대항하면서 첫 무장한 실행들 중의 하나였다.</p>
<p><a href="/wiki/1775%EB%85%84" title="1775년">1775년</a> 매사추세츠에서 <a href="/wiki/%EB%AF%B8%EA%B5%AD_%EB%8F%85%EB%A6%BD_%EC%A0%84%EC%9F%81" title="미국 독립 전쟁">미국 독립 전쟁</a>이 벌어진 후에 수백명의 뉴햄프셔의 민병이 영국군에 싸우려고 <a href="/wiki/%EB%B3%B4%EC%8A%A4%ED%84%B4" title="보스턴">보스턴</a>으로 서둘러 갔다. 뉴햄프셔는 싸움이 일어나지 않은 13개의 초기 식민지들 중의 단 하나였다.</p>
<p>뉴햄프셔는 영국의 완전 독립한 정부를 형성한 첫 식민지가 되었다. <a href="/wiki/1776%EB%85%84" title="1776년">1776년</a> <a href="/wiki/1%EC%9B%94_5%EC%9D%BC" title="1월 5일">1월 5일</a> 일시적 헌법을 채택하였을 때 그 일이 생겼다. <a href="/wiki/1778%EB%85%84" title="1778년">1778년</a> <a href="/wiki/7%EC%9B%94_9%EC%9D%BC" title="7월 9일">7월 9일</a> 뉴햄프셔는 연합 규약을 비준하였다. <a href="/wiki/1788%EB%85%84" title="1788년">1788년</a> <a href="/wiki/6%EC%9B%94_21%EC%9D%BC" title="6월 21일">6월 21일</a> 뉴햄프셔는 <a class="mw-redirect" href="/wiki/%EB%AF%B8%EA%B5%AD_%ED%97%8C%EB%B2%95" title="미국 헌법">미국 헌법</a>을 비준한 9번째 주가 되었다. 뉴햄프셔의 비준은 헌법을 효과적으로 놓았다.</p>
<h3><span class="mw-headline" id="19.EC.84.B8.EA.B8.B0">19세기</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%89%B4%ED%96%84%ED%94%84%EC%85%94_%EC%A3%BC&amp;action=edit&amp;section=7" title="부분 편집: 19세기">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>뉴햄프셔 주는 <a href="/wiki/1800%EB%85%84" title="1800년">1800년</a>부터 <a href="/wiki/1861%EB%85%84" title="1861년">1861년</a> <a class="mw-disambig" href="/wiki/%EB%82%A8%EB%B6%81_%EC%A0%84%EC%9F%81" title="남북 전쟁">남북 전쟁</a>이 일어날 때까지 농업의 주로 남아있었다. 그러고나서 주는 오늘날까지 지속된 산업의 발전이 시작되었다. 1830년대에 애모스케아그 제조업 회사가 메리맥 강의 양쪽을 따라서 <a href="/wiki/%EB%A7%A8%EC%B2%B4%EC%8A%A4%ED%84%B0_(%EB%89%B4%ED%96%84%ED%94%84%EC%85%94_%EC%A3%BC)" title="맨체스터 (뉴햄프셔 주)">맨체스터</a>에 30척의 직물 제재소들을 지었다. 맨체스터는 뉴햄프셔의 가장 큰 산업의 중심지가 되었고, <a href="/wiki/1846%EB%85%84" title="1846년">1846년</a> 주의 첫 합병된 도시가 되었다. 포츠머스는 1800년대 초반에 지도적인 쾌속선의 항구로서 발달하였고, <a href="/wiki/1838%EB%85%84" title="1838년">1838년</a>에 첫 <a href="/wiki/%EC%B2%A0%EB%8F%84" title="철도">철도</a>가 개장되었다. 1850년대 동안에 비지니스의 지도자들이 양말, 메리아스 제조업 지대, 직물 제재소와 부츠와 구두, 기계 도구와 목재품을 만드는 공장들을 지었다.</p>
<p>뉴햄프셔 주에서 온 몇몇의 주민들이 이 시기의 동안에 국가적 명성을 얻었다. 그들은 14대 대통령 <a href="/wiki/%ED%94%84%EB%9E%AD%ED%81%B4%EB%A6%B0_%ED%94%BC%EC%96%B4%EC%8A%A4" title="프랭클린 피어스">프랭클린 피어스</a>와 지도적인 웅변가이며 <a href="/wiki/%EB%AF%B8%EA%B5%AD_%EC%83%81%EC%9B%90" title="미국 상원">미국 상원</a>이 된 <a href="/wiki/%EB%8C%80%EB%8B%88%EC%96%BC_%EC%9B%B9%EC%8A%A4%ED%84%B0" title="대니얼 웹스터">대니얼 웹스터</a>를 포함하였다.</p>
<p>뉴햄프셔 주는 <a class="mw-redirect" href="/wiki/%EB%85%B8%EC%98%88_%EC%A0%9C%EB%8F%84" title="노예 제도">노예 제도</a>에 반대하는 데 지도적인 주였다. 남북 전쟁이 일어나는 동안에 뉴햄프셔 주민의 대략 34,000명이 북군에 종군하였고 포츠머스의 해군 조선소는 남부의 항구들을 봉쇄한 군함들을 지었다.</p>
<p><a href="/wiki/1865%EB%85%84" title="1865년">1865년</a> 전쟁이 끝난 후에 산업적 발전이 거대하게 증가하였다. 직물, 목공과 가죽 산업들이 기록의 비율에서 번창하였다. 제재소와 공장들의 확장에 의한 노동력의 필요함을 채우는 데 수천명의 <a href="/wiki/%ED%94%84%EB%9E%91%EC%8A%A4%EA%B3%84_%EC%BA%90%EB%82%98%EB%8B%A4%EC%9D%B8" title="프랑스계 캐나다인">프랑스계 캐나다인</a>과 <a href="/wiki/%EC%9C%A0%EB%9F%BD" title="유럽">유럽</a>의 이민들이 뉴햄프셔 주에 왔다. 동시에 많은 농부들은 서부에서 자유의 대지를 주장하는 데 주를 떠났다. 산업이 번창하면서 뉴햄프셔 주에서 농업이 감소하였다.</p>
<h3><span class="mw-headline" id="1900.EB.85.84.EB.8C.80_.EC.B4.88.EB.B0.98">1900년대 초반</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%89%B4%ED%96%84%ED%94%84%EC%85%94_%EC%A3%BC&amp;action=edit&amp;section=8" title="부분 편집: 1900년대 초반">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p><a href="/wiki/%EC%A0%9C1%EC%B0%A8_%EC%84%B8%EA%B3%84_%EB%8C%80%EC%A0%84" title="제1차 세계 대전">제1차 세계 대전</a>이 일어나는 동안에 포츠머스 해군 조선소는 <a href="/wiki/%EA%B5%B0%ED%95%A8" title="군함">군함</a>들을 지었다. 뉴햄프셔의 면과 섬유 제재업들이 1920년대와 1930년대에 쇠퇴하였다. 가죽과 구두의 제조업이 주의 지도적인 산업이 되었다. 1920년대 동안에 뉴햄프셔 주는 그 고속도로의 시스템을 향상시키고, 개인적 유용 회사들은 주에 <a class="mw-redirect" href="/wiki/%EC%88%98%EB%A0%A5%EB%B0%9C%EC%A0%84%EC%86%8C" title="수력발전소">수력발전소</a>들을 지었다. 1930년대의 <a href="/wiki/%EB%8C%80%EA%B3%B5%ED%99%A9" title="대공황">대공황</a>이 주 경제의 번창을 느리게 하였다. 1930년대 후반에 대공황이 진정된 후에 상태가 향상되었다.</p>
<h3><span class="mw-headline" id="1900.EB.85.84.EB.8C.80_.EC.A4.91.EB.B0.98">1900년대 중반</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%89%B4%ED%96%84%ED%94%84%EC%85%94_%EC%A3%BC&amp;action=edit&amp;section=9" title="부분 편집: 1900년대 중반">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p><a href="/wiki/%EC%A0%9C2%EC%B0%A8_%EC%84%B8%EA%B3%84_%EB%8C%80%EC%A0%84" title="제2차 세계 대전">제2차 세계 대전</a>이 일어나는 동안에 포츠머스는 <a href="/wiki/%EC%9E%A0%EC%88%98%ED%95%A8" title="잠수함">잠수함</a>과 재수리된 군함들을 짓고 뉴햄프셔 주의 직물 제재소들은 군사 제복을 위한 물자들을 공급하였다. 이 경제적 활동들은 공황 상태를 끝내는 데 도움을 주었다. <a href="/wiki/1944%EB%85%84" title="1944년">1944년</a> 44개국의 대표들이 화이트 산맥에 있는 브레튼우즈에서 역사적 국제 통화 회의를 열었다. 그들은 국가들 사이에 전쟁 이후의 세계 무역을 계획하고 돈의 융통을 단순화하였다. 추가로 그들은 국제적 기구인 <a href="/wiki/%EA%B5%AD%EC%A0%9C_%ED%86%B5%ED%99%94_%EA%B8%B0%EA%B8%88" title="국제 통화 기금">국제 통화 기금</a>과 <a href="/wiki/%EC%84%B8%EA%B3%84%EC%9D%80%ED%96%89" title="세계은행">세계은행</a>을 세웠다.</p>
<p>뉴햄프셔 주는 1900년대 중반 동안에 증가적으로 도시와 산업적이 되었다. 1950년대에 주는 비지니스 개발 주식 회사의 형성을 찬성하고 산업적 공원 권위를 설립하였다. 이 2개의 대리들은 새로운 비지니스들을 원조하는 일을 하였다. 그들은 또한 뉴햄프셔 주에 산업을 끌어들이는 데 노력하였다.</p>
<p>뉴햄프셔 주의 한번의 중요한 구두 산업은 다른 나라들 특히 <a href="/wiki/%EC%9D%B4%ED%83%88%EB%A6%AC%EC%95%84" title="이탈리아">이탈리아</a>는 물론 다른 주들로부터 증가한 경쟁의 이유로 날카롭게 쇠퇴하였다. 하지만, 뉴햄프셔 주의 전자 산업의 빠른 번창은 구두 산업에서 손해들을 위하여 이루어진 것보다 더 많았다.</p>
<p><a href="/wiki/1961%EB%85%84" title="1961년">1961년</a> 뉴햄프셔 주의 <a href="/wiki/%EC%95%A8%EB%9F%B0_%EC%85%B0%ED%8D%BC%EB%93%9C" title="앨런 셰퍼드">앨런 셰퍼드</a>는 <a href="/wiki/%EC%9A%B0%EC%A3%BC" title="우주">우주</a>를 여행한 첫 미국인이 되었다. 그는 이스트데리에서 태어났다.</p>
<p><a href="/wiki/1963%EB%85%84" title="1963년">1963년</a> 뉴햄프셔 주는 1890년대 이래 미국에서 첫 입법적 추첨 분배를 채택하였다. 주의 추첨 분배는 "스윕스테익스"로 알려졌으며, 주는 공공 교육을 위한 돈을 내는 데 스윕스테익스로부터 얻은 이득을 썼다.</p>
<h3><span class="mw-headline" id="1900.EB.85.84.EB.8C.80_.ED.9B.84.EB.B0.98">1900년대 후반</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%89%B4%ED%96%84%ED%94%84%EC%85%94_%EC%A3%BC&amp;action=edit&amp;section=10" title="부분 편집: 1900년대 후반">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>콩코그 고등학교의 교사 <a href="/wiki/%ED%81%AC%EB%A6%AC%EC%8A%A4%ED%83%80_%EB%A7%A4%EC%BD%9C%EB%A6%AC%ED%94%84" title="크리스타 매콜리프">크리스타 매콜리프</a>는 우주에서 첫 "평범한 시민"이 되는 데 선택되었다. 그녀는 <a href="/wiki/1986%EB%85%84" title="1986년">1986년</a> <a href="/wiki/1%EC%9B%94_28%EC%9D%BC" title="1월 28일">1월 28일</a>에 챌린저 우주선이 발사된 후 부서졌을 때 사망한 7명의 승무원들 중에 하나였다.</p>
<p>1980년대 동안에 뉴햄프셔 주는 특히 주의 남부에서 경제적 번영과 재빠른 인구의 증가를 경험하였다. 하지만, 1980년대 후반과 1990년대 초반에 국가적 불경기가 특히 <a href="/wiki/%EB%B6%80%EB%8F%99%EC%82%B0" title="부동산">부동산</a>, <a href="/wiki/%EA%B8%88%EC%9C%B5" title="금융">금융</a>과 <a href="/wiki/%EA%B5%AD%EB%B0%A9" title="국방">국방</a> 산업에서 많은 직업들의 손실의 결과를 가져왔다. 1990년대 중반에 뉴햄프셔 주의 경제는 향상되었다.</p>
<p><a href="/wiki/1996%EB%85%84" title="1996년">1996년</a> 투표인들은 뉴햄프셔 주의 첫 여성 주지사 <a class="new" href="/w/index.php?title=%EC%A7%84_%EC%84%80%ED%9E%8C&amp;action=edit&amp;redlink=1" title="진 섀힌 (없는 문서)">진 섀힌</a>을 선출하였다. 그녀는 2번이나 재선되어 <a href="/wiki/2003%EB%85%84" title="2003년">2003년</a>까지 그 직에 있었다.</p>
<h3><span class="mw-headline" id="21.EC.84.B8.EA.B8.B0">21세기</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%89%B4%ED%96%84%ED%94%84%EC%85%94_%EC%A3%BC&amp;action=edit&amp;section=11" title="부분 편집: 21세기">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<div class="thumb tright">

<p>뉴햄프셔 주는 일반 개인적 세금 혹은 일반 판매세를 걷지 않는 몇몇의 주들 중의 하나로 남아있었다. 주민들은 보통의 소득이 아닌 분배된 이익 소득에만 개인적 소득세를 낸다. 이 저세금 정책은 새로운 산업들과 다른 비지니스들을 끌어들이는 데 도움을 준다. 그러나 저세금들은 주의 기금된 프로그램과 서비스를 위하여 내는 데 적은 돈과 학교 기금과 지방적 공동체들에 다른 원조를 위하여 적은 돈의 결과를 가져온다.</p>
<p><a href="/wiki/2003%EB%85%84" title="2003년">2003년</a> 뉴햄프셔 주는 주의 가장 잘 알려진 상징 올드맨 산의 손해에 고통을 겪었다. 그레이트 스톤 페이스로도 불린 올드맨 산은 프로파일 산에서 새겨져 부식된 남자 얼굴의 자연적 돌 옆모습이었다. 옆얼굴을 형성한 바위들이 그해 <a href="/wiki/5%EC%9B%94" title="5월">5월</a>에 떨어졌다. <a href="/wiki/2009%EB%85%84" title="2009년">2009년</a> 뉴햄프셔 주의 입법부는 동성의 결혼을 적법화하였다.</p>
<h2><span class="mw-headline" id=".EA.B2.BD.EC.A0.9C">경제</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%89%B4%ED%96%84%ED%94%84%EC%85%94_%EC%A3%BC&amp;action=edit&amp;section=12" title="부분 편집: 경제">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<h3><span class="mw-headline" id=".EC.84.9C.EB.B9.84.EC.8A.A4.EC.97.85">서비스업</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%89%B4%ED%96%84%ED%94%84%EC%85%94_%EC%A3%BC&amp;action=edit&amp;section=13" title="부분 편집: 서비스업">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p><a class="mw-redirect" href="/wiki/%EC%84%9C%EB%B9%84%EC%8A%A4%EC%97%85" title="서비스업">서비스업</a>은 뉴햄프셔 주의 고용과 국내 총생산의 대략 5분의 4를 차지한다. 이 산업들은 주를 통하여, 특히 주의 남부에 있는 도시와 타운들에서 운영된다.</p>
<p>주도인 <a href="/wiki/%EC%BD%A9%EC%BD%94%EB%93%9C_(%EB%89%B4%ED%96%84%ED%94%84%EC%85%94_%EC%A3%BC)" title="콩코드 (뉴햄프셔 주)">콩코드</a>는 정부 활동의 중심지이다. <a href="/wiki/%EB%A7%A8%EC%B2%B4%EC%8A%A4%ED%84%B0_(%EB%89%B4%ED%96%84%ED%94%84%EC%85%94_%EC%A3%BC)" title="맨체스터 (뉴햄프셔 주)">맨체스터</a>는 주요 보험과 금융의 중심지이다. 많은 <a href="/wiki/%ED%98%B8%ED%85%94" title="호텔">호텔</a>, 식당과 소매상들도 맨체스터 지역에 있다. 화이트 산맥과 뉴햄프셔 주의 호수와 연안 지역들은 많은 <a href="/wiki/%EB%A6%AC%EC%A1%B0%ED%8A%B8" title="리조트">리조트</a>들이 있다. <a href="/wiki/%ED%8F%AC%EC%B8%A0%EB%A8%B8%EC%8A%A4_(%EB%89%B4%ED%96%84%ED%94%84%EC%85%94_%EC%A3%BC)" title="포츠머스 (뉴햄프셔 주)">포츠머스</a> 지역은 포츠머스 해군 조선소의 기지이다.</p>
<h3><span class="mw-headline" id=".EC.A0.9C.EC.A1.B0.EC.97.85">제조업</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%89%B4%ED%96%84%ED%94%84%EC%85%94_%EC%A3%BC&amp;action=edit&amp;section=14" title="부분 편집: 제조업">편집</a><span class="mw-editsection-bracket">]</span></span></h3>

<p><a href="/wiki/%EC%BB%B4%ED%93%A8%ED%84%B0" title="컴퓨터">컴퓨터</a>와 <a href="/wiki/%EC%A0%84%EC%9E%90" title="전자">전자</a> 제품들은 주의 지도적인 생산품들이다. 마이크로칩과 네트워킹 용품 같은 전자 제품들은 맨체스터 지역에서 만들어진다. <a href="/wiki/%EB%82%B4%EC%8A%88%EC%95%84_(%EB%89%B4%ED%96%84%ED%94%84%EC%85%94_%EC%A3%BC)" title="내슈아 (뉴햄프셔 주)">내슈아</a>에 있는 공장 지대들은 군사 통신 조직망을 만든다.</p>
<p>볼베어링, <a href="/wiki/%EC%86%8C%ED%99%94%EA%B8%B0" title="소화기">소화기</a>와 상점의 기계품들을 포함한 합성 금속 제품들은 맨체스터 지역에서 만들어진다. 뉴햄프셔의 남부에서는 <a href="/wiki/%EC%9D%B8%EC%87%84" title="인쇄">인쇄</a>와 금속 가공과 다른 기계 제품들을 생산한다. 다른 기계 제품들은 <a href="/wiki/%EB%A0%8C%EC%A6%88" title="렌즈">렌즈</a>와 다른 광학 기구들을 위한 용품들을 포함한다. 뉴햄프셔 주는 <a href="/wiki/%EC%B4%88%EC%BD%9C%EB%A6%BF" title="초콜릿">초콜릿</a>, <a href="/wiki/%EC%BD%98%ED%81%AC%EB%A6%AC%ED%8A%B8" title="콘크리트">콘크리트</a> 제품, <a href="/wiki/%EC%9C%A0%EB%A6%AC" title="유리">유리</a> 제품, <a href="/wiki/%EC%A0%84%EA%B8%B0" title="전기">전기</a> 제품과 <a href="/wiki/%EC%9D%98%EB%A3%8C" title="의료">의료</a> 용품들을 생산하기도 한다.</p>
<h3><span class="mw-headline" id=".EB.86.8D.EC.97.85">농업</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%89%B4%ED%96%84%ED%94%84%EC%85%94_%EC%A3%BC&amp;action=edit&amp;section=15" title="부분 편집: 농업">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<div class="thumb tright">
<div class="thumbinner" style="width:222px;"><a class="image" href="/wiki/%ED%8C%8C%EC%9D%BC:Massey_Ferguson_230,_hay_rakes_and_round_bales_in_New_Hampshire.jpg"><img alt="" class="thumbimage" data-file-height="541" data-file-width="1024" height="116" src="//upload.wikimedia.org/wikipedia/commons/thumb/b/ba/Massey_Ferguson_230%2C_hay_rakes_and_round_bales_in_New_Hampshire.jpg/220px-Massey_Ferguson_230%2C_hay_rakes_and_round_bales_in_New_Hampshire.jpg" srcset="//upload.wikimedia.org/wikipedia/commons/thumb/b/ba/Massey_Ferguson_230%2C_hay_rakes_and_round_bales_in_New_Hampshire.jpg/330px-Massey_Ferguson_230%2C_hay_rakes_and_round_bales_in_New_Hampshire.jpg 1.5x, //upload.wikimedia.org/wikipedia/commons/thumb/b/ba/Massey_Ferguson_230%2C_hay_rakes_and_round_bales_in_New_Hampshire.jpg/440px-Massey_Ferguson_230%2C_hay_rakes_and_round_bales_in_New_Hampshire.jpg 2x" width="220"/></a>
<div class="thumbcaption">
<div class="magnify"><a class="internal" href="/wiki/%ED%8C%8C%EC%9D%BC:Massey_Ferguson_230,_hay_rakes_and_round_bales_in_New_Hampshire.jpg" title="실제 크기로"></a></div>
<a href="/wiki/%EA%B1%B4%EC%B4%88" title="건초">건초</a>의 경작</div>
</div>
</div>
<p>농장 지대는 뉴햄프셔의 8 퍼센트 만을 덮고 있다. 온실과 종묘 산품들은 농장 소득의 주의 지도적인 근원이다. <a href="/wiki/%EA%B1%B4%EC%B4%88" title="건초">건초</a>는 뉴햄프셔의 지도적인 곡물이며, <a class="mw-redirect" href="/wiki/%EC%82%AC%EA%B3%BC" title="사과">사과</a>가 지도적인 과일이고 사탕 옥수수가 지도적인 채소이다.</p>
<p><a class="mw-redirect" href="/wiki/%EB%82%99%EB%86%8D%EC%97%85" title="낙농업">낙농업</a>은 뉴햄프셔의 가장 중요한 축산 활동이다. 낙농 지대는 주의 전역을 통하여 운영되나 특히 <a href="/wiki/%EC%BD%94%EB%84%A4%ED%8B%B0%EC%BB%B7_%EA%B0%95" title="코네티컷 강">코네티컷 강</a>을 따라 서부에서 다수로 운영되고 있다. 뉴햄프셔의 다른 축산물은 <a href="/wiki/%EC%9C%A1%EC%9A%B0" title="육우">육우</a>, <a href="/wiki/%EB%8B%AC%EA%B1%80" title="달걀">달걀</a>과 <a href="/wiki/%EB%8F%BC%EC%A7%80" title="돼지">돼지</a>를 포함한다.</p>
<h3><span class="mw-headline" id=".EA.B4.91.EC.97.85">광업</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%89%B4%ED%96%84%ED%94%84%EC%85%94_%EC%A3%BC&amp;action=edit&amp;section=16" title="부분 편집: 광업">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p><a href="/wiki/%EB%AA%A8%EB%9E%98" title="모래">모래</a>와 <a href="/wiki/%EC%9E%90%EA%B0%88" title="자갈">자갈</a>은 주의 주요 체굴품들이다. 그들은 주의 카운티들의 전부에서 찾아지나 생산은 주의 남중부에 있는 힐스보로와 메리맥 카운티들에서 가장 무거운 편이다.</p>
<p>깨진 돌도 또한 중요하고, 주의 몇몇의 부분들은 <a href="/wiki/%ED%99%94%EA%B0%95%EC%95%94" title="화강암">화강암</a> 채석장들이 있다. 메리맥 카운티에 있는 콩코드 채석장은 미국에서 많은 유명한 건물들을 위한 화강암을 공급한다.</p>
<h2><span class="mw-headline" id=".EC.A3.BC.EB.AF.BC">주민</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%89%B4%ED%96%84%ED%94%84%EC%85%94_%EC%A3%BC&amp;action=edit&amp;section=17" title="부분 편집: 주민">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<div class="thumb tright">
<div class="thumbinner" style="width:222px;"><a class="image" href="/wiki/%ED%8C%8C%EC%9D%BC:Manch-DownTown.jpg"><img alt="" class="thumbimage" data-file-height="1600" data-file-width="1200" height="293" src="//upload.wikimedia.org/wikipedia/commons/thumb/a/a4/Manch-DownTown.jpg/220px-Manch-DownTown.jpg" srcset="//upload.wikimedia.org/wikipedia/commons/thumb/a/a4/Manch-DownTown.jpg/330px-Manch-DownTown.jpg 1.5x, //upload.wikimedia.org/wikipedia/commons/thumb/a/a4/Manch-DownTown.jpg/440px-Manch-DownTown.jpg 2x" width="220"/></a>
<div class="thumbcaption">
<div class="magnify"><a class="internal" href="/wiki/%ED%8C%8C%EC%9D%BC:Manch-DownTown.jpg" title="실제 크기로"></a></div>
다운타운 맨체스터</div>
</div>
</div>
<p><a href="/wiki/2000%EB%85%84" title="2000년">2000년</a> <a class="mw-redirect" href="/wiki/%EB%AF%B8%EA%B5%AD_%EC%9D%B8%EA%B5%AC_%EC%A1%B0%EC%82%AC%EA%B5%AD" title="미국 인구 조사국">미국 인구 조사국</a>은 뉴햄프셔의 인구가 1,235,786명이라고 보고하였다. 주의 인구는 <a href="/wiki/1990%EB%85%84" title="1990년">1990년</a>으로 본 1,109,252명에서 대략 11 퍼센트나 증가하였다. <a href="/wiki/2000%EB%85%84" title="2000년">2000년</a> 조사국에 의하면 뉴햄프셔는 50개의 주들 사이에 인구에서 41위로 들어와있다.</p>
<p>뉴햄프셔 주민의 대략 5분의 3은 메트로폴리탄 지역들에 산다. 뉴햄프셔 주는 2개의 메트로폴리탄 지역들을 가지고 있다. 이 지역들은 <a href="/wiki/%EB%B3%B4%EC%8A%A4%ED%84%B4" title="보스턴">보스턴</a>(<a class="mw-redirect" href="/wiki/%EB%A7%A4%EC%82%AC%EC%B6%94%EC%84%B8%EC%B8%A0" title="매사추세츠">매사추세츠</a>)-<a href="/wiki/%EC%BC%80%EC%9E%84%EB%B8%8C%EB%A6%AC%EC%A7%80_(%EB%A7%A4%EC%82%AC%EC%B6%94%EC%84%B8%EC%B8%A0_%EC%A3%BC)" title="케임브리지 (매사추세츠 주)">케임브리지</a>(매사추세츠)-<a href="/wiki/%ED%80%B8%EC%8B%9C_(%EB%A7%A4%EC%82%AC%EC%B6%94%EC%84%B8%EC%B8%A0_%EC%A3%BC)" title="퀸시 (매사추세츠 주)">퀸시</a>(매사추세츠)와 <a href="/wiki/%EB%A7%A8%EC%B2%B4%EC%8A%A4%ED%84%B0_(%EB%89%B4%ED%96%84%ED%94%84%EC%85%94_%EC%A3%BC)" title="맨체스터 (뉴햄프셔 주)">맨체스터</a>-<a href="/wiki/%EB%82%B4%EC%8A%88%EC%95%84_(%EB%89%B4%ED%96%84%ED%94%84%EC%85%94_%EC%A3%BC)" title="내슈아 (뉴햄프셔 주)">내슈아</a>이다. 맨체스터는 주에서 가장 큰 도시이다.</p>
<p>맨체스터에 추가로 뉴햄프셔 주는 12개의 다른 합병된 도시들을 가지고 있다. 크기의 순서로 이 도시들은 내슈아, <a href="/wiki/%EC%BD%A9%EC%BD%94%EB%93%9C_(%EB%89%B4%ED%96%84%ED%94%84%EC%85%94_%EC%A3%BC)" title="콩코드 (뉴햄프셔 주)">콩코드</a>, <a href="/wiki/%EB%A1%9C%EC%B2%B4%EC%8A%A4%ED%84%B0_(%EB%89%B4%ED%96%84%ED%94%84%EC%85%94_%EC%A3%BC)" title="로체스터 (뉴햄프셔 주)">로체스터</a>, 도버, 킨, <a href="/wiki/%ED%8F%AC%EC%B8%A0%EB%A8%B8%EC%8A%A4_(%EB%89%B4%ED%96%84%ED%94%84%EC%85%94_%EC%A3%BC)" title="포츠머스 (뉴햄프셔 주)">포츠머스</a>, 라코니아, 클레어몬트, 레버넌, 소머스워스, <a href="/wiki/%EB%B2%8C%EB%A6%B0_(%EB%89%B4%ED%96%84%ED%94%84%EC%85%94_%EC%A3%BC)" title="벌린 (뉴햄프셔 주)">벌린</a>과 프랭클린이다.</p>
<p>뉴햄프셔의 많은 주민들은 <a href="/wiki/%EC%BA%90%EB%82%98%EB%8B%A4" title="캐나다">캐나다</a>와 많은 <a href="/wiki/%EC%9C%A0%EB%9F%BD" title="유럽">유럽</a>의 나라들에서 온 정착자들의 자손들이다. 뉴햄프셔의 초기 정착자들은 주로 <a href="/wiki/%EC%9E%89%EA%B8%80%EB%9E%9C%EB%93%9C" title="잉글랜드">잉글랜드</a>, <a href="/wiki/%EC%95%84%EC%9D%BC%EB%9E%9C%EB%93%9C" title="아일랜드">아일랜드</a>, <a href="/wiki/%EC%8A%A4%EC%BD%94%ED%8B%80%EB%9E%9C%EB%93%9C" title="스코틀랜드">스코틀랜드</a>, <a class="mw-redirect" href="/wiki/%EC%9B%A8%EC%9D%BC%EC%A6%88" title="웨일즈">웨일즈</a>에서 왔다. <a class="mw-disambig" href="/wiki/%EB%82%A8%EB%B6%81_%EC%A0%84%EC%9F%81" title="남북 전쟁">남북 전쟁</a> 이후에는 수천명의 <a href="/wiki/%ED%94%84%EB%9E%91%EC%8A%A4%EA%B3%84_%EC%BA%90%EB%82%98%EB%8B%A4%EC%9D%B8" title="프랑스계 캐나다인">프랑스계 캐나다인</a>과 유럽의 이민들이 제재소, 상점과 공장들에 일하러 뉴햄프셔에 왔다.</p>
<p>오늘날 뉴햄프셔 주의 가장 큰 인구 단체들은 <a href="/wiki/%EC%95%84%EC%9D%BC%EB%9E%9C%EB%93%9C" title="아일랜드">아일랜드</a>, <a href="/wiki/%EC%98%81%EA%B5%AD" title="영국">영국</a>, <a href="/wiki/%ED%94%84%EB%9E%91%EC%8A%A4" title="프랑스">프랑스</a>, <a href="/wiki/%EB%8F%85%EC%9D%BC" title="독일">독일</a>, <a href="/wiki/%EC%9D%B4%ED%83%88%EB%A6%AC%EC%95%84" title="이탈리아">이탈리아</a> 계통과 프랑스계 캐나다인의 후손이다. <a href="/wiki/%EC%95%84%EC%8B%9C%EC%95%84" title="아시아">아시아</a>인, <a class="mw-redirect" href="/wiki/%EC%95%84%EB%A9%94%EB%A6%AC%EC%B9%B4_%EC%9D%B8%EB%94%94%EC%96%B8" title="아메리카 인디언">아메리카 인디언</a>, <a href="/wiki/%ED%9E%88%EC%8A%A4%ED%8C%A8%EB%8B%89" title="히스패닉">히스패닉</a>은 주 인구의 5 퍼센트 이하를 차지한다.</p>
