<p><b>줄리아노 디 로렌초 데 메디치</b>(Giuliano di Lorenzo de' Medici, <a href="/wiki/1479%EB%85%84" title="1479년">1479년</a> <a href="/wiki/3%EC%9B%94_12%EC%9D%BC" title="3월 12일">3월 12일</a> ~ <a href="/wiki/1516%EB%85%84" title="1516년">1516년</a> <a href="/wiki/3%EC%9B%94_17%EC%9D%BC" title="3월 17일">3월 17일</a>)는 이탈리아의 귀족으로, <a href="/wiki/%EB%A1%9C%EB%A0%8C%EC%B4%88_%EB%8D%B0_%EB%A9%94%EB%94%94%EC%B9%98" title="로렌초 데 메디치">로렌초 데 메디치</a>의 아들이다. 줄리아노의 집안인 <a href="/wiki/%EB%A9%94%EB%94%94%EC%B9%98_%EA%B0%80%EB%AC%B8" title="메디치 가문">메디치 가문</a>은 대대로 피렌체를 통치해 왔는데 뛰어난 통치자였던 부친 로렌초와는 달리 그 뒤를 이은 큰형 <a class="mw-redirect" href="/wiki/%ED%94%BC%EC%97%90%EB%A1%9C_%EB%94%94_%EB%A1%9C%EB%A0%8C%EC%B4%88_%EB%8D%B0_%EB%A9%94%EB%94%94%EC%B9%98" title="피에로 디 로렌초 데 메디치">피에로 2세 데 메디치</a>는 무능한 인물로, 그의 대에 이르러 메디치 가문은 피렌체에서 쫓겨나는 수모를 당하게 되었다. 줄리아노는 <a href="/wiki/1512%EB%85%84" title="1512년">1512년</a>까지 유럽 곳곳을 돌아다녔고, <a href="/wiki/1515%EB%85%84" title="1515년">1515년</a>에는 <a class="new" href="/w/index.php?title=%EC%82%AC%EB%B3%B4%EC%9D%B4%EC%9D%98_%EA%B5%B0%EC%A3%BC&amp;action=edit&amp;redlink=1" title="사보이의 군주 (없는 문서)">사보이 공작</a> <a class="new" href="/w/index.php?title=%EC%82%AC%EB%B3%B4%EC%9D%B4%EC%9D%98_%ED%95%84%EB%A6%AC%ED%8F%AC_2%EC%84%B8&amp;action=edit&amp;redlink=1" title="사보이의 필리포 2세 (없는 문서)">필리포 2세</a>의 딸 <a class="new" href="/w/index.php?title=%EC%82%AC%EB%B3%B4%EC%9D%B4%EC%9D%98_%ED%95%84%EB%B2%A0%EB%A5%B4%ED%83%80&amp;action=edit&amp;redlink=1" title="사보이의 필베르타 (없는 문서)">필베르타</a>와 결혼했다. 필베르타의 언니 <a class="new" href="/w/index.php?title=%EC%82%AC%EB%B3%B4%EC%9D%B4%EC%9D%98_%EB%A3%A8%EC%9D%B4%EC%A6%88&amp;action=edit&amp;redlink=1" title="사보이의 루이즈 (없는 문서)">사보이의 루이즈</a>는 <a href="/wiki/%ED%94%84%EB%9E%91%EC%88%98%EC%95%84_1%EC%84%B8_(%ED%94%84%EB%9E%91%EC%8A%A4)" title="프랑수아 1세 (프랑스)">프랑수아 1세</a>의 어머니였는데, 프랑수아 1세는 이모부 줄리아노에게 느무르 공작의 작위를 주었다. 줄리아노는 <a href="/wiki/1513%EB%85%84" title="1513년">1513년</a>부터 3년간 죽은 형을 대신하여 피렌체를 다스렸고, <a class="mw-redirect" href="/wiki/%EB%9D%BC%ED%8C%8C%EC%97%98%EB%A1%9C" title="라파엘로">라파엘로</a>, <a href="/wiki/%EB%A3%A8%EB%8F%84%EB%B9%84%EC%BD%94_%EC%95%84%EB%A6%AC%EC%98%A4%EC%8A%A4%ED%86%A0" title="루도비코 아리오스토">루도비코 아리오스토</a>, <a href="/wiki/%EB%A0%88%EC%98%A4%EB%82%98%EB%A5%B4%EB%8F%84_%EB%8B%A4_%EB%B9%88%EC%B9%98" title="레오나르도 다 빈치">레오나르도 다 빈치</a> 등을 후원하였다. <a href="/wiki/1516%EB%85%84" title="1516년">1516년</a> 갑작스러운 죽음을 맞이하였고, <a href="/wiki/%EC%82%B0_%EB%A1%9C%EB%A0%8C%EC%B4%88_%EC%84%B1%EB%8B%B9_(%ED%94%BC%EB%A0%8C%EC%B2%B4)" title="산 로렌초 성당 (피렌체)">산 로렌초 성당</a>에 묻혔다.</p>
