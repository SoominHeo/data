<p><b>늘보로리스</b>(또는 <b>슬로우로리스</b>)는 <b>늘보로리스속</b> 또는 <b>니크티케부스속</b>(<i>Nycticebus</i>)에 속하는 3 종의 로리스의 총칭이다. <a href="/wiki/%EB%B3%B4%EB%A5%B4%EB%84%A4%EC%98%A4_%EC%84%AC" title="보르네오 섬">보르네오 섬</a>과 <a href="/wiki/%EB%8F%99%EB%82%A8%EC%95%84%EC%8B%9C%EC%95%84" title="동남아시아">동남아시아</a>의 <a href="/wiki/%ED%95%84%EB%A6%AC%ED%95%80" title="필리핀">필리핀</a>에서부터 <a href="/wiki/%EB%B0%A9%EA%B8%80%EB%9D%BC%EB%8D%B0%EC%8B%9C" title="방글라데시">방글라데시</a>, <a href="/wiki/%EB%B2%A0%ED%8A%B8%EB%82%A8" title="베트남">베트남</a>, <a href="/wiki/%EC%9D%B8%EB%8F%84%EB%84%A4%EC%8B%9C%EC%95%84" title="인도네시아">인도네시아</a>, <a href="/wiki/%EC%9D%B8%EB%8F%84" title="인도">인도</a> (인도 북동부, 벵갈), <a href="/wiki/%EC%A4%91%EA%B5%AD" title="중국">중국</a> 남부 (<a href="/wiki/%EC%9C%88%EB%82%9C_%EC%84%B1" title="윈난 성">윈난 성</a> 지역)와 <a class="mw-redirect" href="/wiki/%ED%83%9C%EA%B5%AD" title="태국">태국</a>에 걸쳐 분포하는 <a href="/wiki/%EA%B3%A1%EB%B9%84%EC%9B%90%EB%A5%98" title="곡비원류">곡비원류</a> <a href="/wiki/%EC%98%81%EC%9E%A5%EB%A5%98" title="영장류">영장류</a>로 느리게 움직이는 특징이 있다.</p>
<p>이들의 큰 눈이 현지의 전통 의약품에서 높이 평가되어 사냥되기 때문에, <a href="/wiki/%EB%B3%B4%EC%A0%84_%EC%83%81%ED%83%9C" title="보전 상태">멸종위기등급</a>에서 취약 등급 또는 위기 등급으로 분류되고 있다.<sup class="reference" id="cite_ref-2"><a href="#cite_note-2">[2]</a></sup></p>
<p>인도네시아에서의 이름은 "말루말루"(<i>malu malu</i>)로 번역<sup class="reference" id="cite_ref-3"><a href="#cite_note-3">[3]</a></sup> 하면, "부끄럼타는 놈"이라는 뜻이다.</p>
<p>미국의 어류 및 야생동물 관리국은 늘보로리스를 멸종위기종으로 분류하고 있다.<sup class="reference" id="cite_ref-4"><a href="#cite_note-4">[4]</a></sup></p>
<p>그 밖에도 늘보로리스는 <a href="/wiki/SBS_TV" title="SBS TV">SBS TV</a>의 <a href="/wiki/%EC%A0%95%EA%B8%80%EC%9D%98_%EB%B2%95%EC%B9%99" title="정글의 법칙">정글의 법칙</a> in 보르네오와 <a class="mw-redirect" href="/wiki/EBS" title="EBS">EBS</a> <a href="/wiki/%ED%95%98%EB%82%98%EB%BF%90%EC%9D%B8_%EC%A7%80%EA%B5%AC" title="하나뿐인 지구">하나뿐인 지구</a>, <a href="/wiki/TV_%EB%8F%99%EB%AC%BC%EB%86%8D%EC%9E%A5" title="TV 동물농장">TV 동물농장</a>에서도 간혹 소개되어 온 것으로 드러나고 있으며, IUCN에서 지정받고 있는 멸종위기종으로 분류되어 오게 되는 등 보호 관찰이 가장 시급하다고 여겨지는 것으로 보인다.</p>
