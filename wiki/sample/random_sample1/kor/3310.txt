<p><b>늪사슴</b>(<i>Blastocerus dichotomus</i>)은 <a href="/wiki/%EB%82%A8%EC%95%84%EB%A9%94%EB%A6%AC%EC%B9%B4" title="남아메리카">남아메리카</a>에 서식하는 가장 큰 <a href="/wiki/%EC%82%AC%EC%8A%B4" title="사슴">사슴</a>으로 어깨 높이가 최대 1.2m이다. <a href="/wiki/%EC%95%84%EB%A5%B4%ED%97%A8%ED%8B%B0%EB%82%98" title="아르헨티나">아르헨티나</a>와 <a href="/wiki/%EB%B3%BC%EB%A6%AC%EB%B9%84%EC%95%84" title="볼리비아">볼리비아</a>, <a href="/wiki/%ED%8E%98%EB%A3%A8" title="페루">페루</a>, <a href="/wiki/%EB%B8%8C%EB%9D%BC%EC%A7%88" title="브라질">브라질</a>, <a href="/wiki/%EC%9A%B0%EB%A3%A8%EA%B3%BC%EC%9D%B4" title="우루과이">우루과이</a>, <a href="/wiki/%ED%8C%8C%EB%9D%BC%EA%B3%BC%EC%9D%B4" title="파라과이">파라과이</a>에서 발견된다. <b>늪사슴속</b>(<i>Blastocerus</i>)의 유일종이다. 이전에는 남아메리카 열대와 아열대 기후 지대의 상당수 지역에서 발견되어, <a href="/wiki/%EC%95%88%EB%8D%B0%EC%8A%A4_%EC%82%B0%EB%A7%A5" title="안데스 산맥">안데스 산맥</a> 동부와 아마존 우림의 남쪽부터 브라질 <a href="/wiki/%EB%8C%80%EC%84%9C%EC%96%91%EB%A6%BC" title="대서양림">대서양림</a>과 아르헨티나 <a href="/wiki/%ED%8C%9C%ED%8C%8C%EC%8A%A4" title="팜파스">팜파스</a> 지역 북부까지 지역에 분포했다. 현재는 고립된 개체군으로 크게 줄어 들어서 <a href="/wiki/%ED%8C%8C%EB%9D%BC%EB%82%98_%EA%B0%95" title="파라나 강">파라나 강</a>과 <a href="/wiki/%ED%8C%8C%EB%9D%BC%EA%B3%BC%EC%9D%B4_%EA%B0%95" title="파라과이 강">파라과이 강</a>, <a href="/wiki/%EC%95%84%EB%9D%BC%EA%B3%BC%EC%9D%B4%EC%95%84_%EA%B0%95" title="아라과이아 강">아라과이아 강</a>, 과포레 강 분지의 습지와 개펄 지역에서 서식한다. 또한 페루 바우아하-소네네 국립공원의 보호 지역을 포함하여 아마존 남부 지역에서도 작은 개체군으로 발견된다.<sup class="reference" id="cite_ref-Thornback_2-0"><a href="#cite_note-Thornback-2">[2]</a></sup><sup class="reference" id="cite_ref-Cabrera_3-0"><a href="#cite_note-Cabrera-3">[3]</a></sup><sup class="reference" id="cite_ref-Thomas1997_4-0"><a href="#cite_note-Thomas1997-4">[4]</a></sup> <a href="/wiki/%EA%B5%AD%EC%A0%9C_%EC%9E%90%EC%97%B0_%EB%B3%B4%EC%A0%84_%EC%97%B0%EB%A7%B9" title="국제 자연 보전 연맹">국제 자연 보전 연맹</a>(IUCN)과 <a href="/wiki/%EB%A9%B8%EC%A2%85%EC%9C%84%EA%B8%B0%EC%97%90_%EC%B2%98%ED%95%9C_%EC%95%BC%EC%83%9D%EB%8F%99%EC%8B%9D%EB%AC%BC%EC%A2%85%EC%9D%98_%EA%B5%AD%EC%A0%9C%EA%B1%B0%EB%9E%98%EC%97%90_%EA%B4%80%ED%95%9C_%ED%98%91%EC%95%BD" title="멸종위기에 처한 야생동식물종의 국제거래에 관한 협약">멸종위기에 처한 야생동식물종의 국제거래에 관한 협약</a>(CITES) 부속서 I는 이 종을 멸종 취약종으로 분류하고 있다.<sup class="reference" id="cite_ref-iucn_1-1"><a href="#cite_note-iucn-1">[1]</a></sup></p>
