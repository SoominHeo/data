<p><b>니나와 주</b>(<span style="font-size: smaller;"><a href="/wiki/%EC%95%84%EB%9E%8D%EC%96%B4" title="아랍어">아랍어</a>:</span> <span lang="ar" xml:lang="ar">نینوى</span>, <span style="font-size: smaller;"><a href="/wiki/%EC%BF%A0%EB%A5%B4%EB%93%9C%EC%96%B4" title="쿠르드어">쿠르드어</a>:</span> <span lang="ku" xml:lang="ku">Neynewa</span>)는 <a href="/wiki/%EC%9D%B4%EB%9D%BC%ED%81%AC" title="이라크">이라크</a> 북부에 위치한 주로 주도는 <a href="/wiki/%EB%AA%A8%EC%88%A0" title="모술">모술</a>이며 면적은 37,323㎢, 인구는 2,453,000명(2003년 기준)이다. 북쪽으로는 <a href="/wiki/%EB%8B%A4%ED%9B%84%ED%81%AC_%EC%A3%BC" title="다후크 주">다후크 주</a>, 동쪽으로는 <a href="/wiki/%EC%95%84%EB%A5%B4%EB%B9%8C_%EC%A3%BC" title="아르빌 주">아르빌 주</a>, 남동쪽으로는 <a href="/wiki/%EC%82%B4%EB%9D%BC%EB%94%98_%EC%A3%BC" title="살라딘 주">살라딘 주</a>, 남서쪽으로는 <a href="/wiki/%EC%95%88%EB%B0%94%EB%A5%B4_%EC%A3%BC" title="안바르 주">안바르 주</a>와 접하며 서쪽으로는 <a href="/wiki/%EC%8B%9C%EB%A6%AC%EC%95%84" title="시리아">시리아</a>와 국경을 접한다.</p>
<p>주 이름은 고대 <a href="/wiki/%EC%95%84%EC%8B%9C%EB%A6%AC%EC%95%84" title="아시리아">아시리아</a>의 도시인 <a href="/wiki/%EB%8B%88%EB%84%A4%EB%B2%A0" title="니네베">니네베</a>에서 유래된 이름이다. <a href="/wiki/1976%EB%85%84" title="1976년">1976년</a> 이전까지는 <b>모술 주</b>라는 이름으로 알려져 있었으며 현재의 <a href="/wiki/%EB%8B%A4%ED%9B%84%ED%81%AC_%EC%A3%BC" title="다후크 주">다후크 주</a>는 과거 니나와 주의 일부였다.</p>
