<p><b>니시무라 유이치</b>(<span style="font-size: smaller;"><a href="/wiki/%EC%9D%BC%EB%B3%B8%EC%96%B4" title="일본어">일본어</a>:</span> <span lang="ja"><span lang="ja"><ruby><rb>西村雄一</rb> <rp>(</rp><rt>にしむらゆういち</rt><rp>)</rp></ruby></span></span>, <a href="/wiki/1972%EB%85%84" title="1972년">1972년</a> <a href="/wiki/4%EC%9B%94_17%EC%9D%BC" title="4월 17일">4월 17일</a> ~ )는 <a href="/wiki/%EC%9D%BC%EB%B3%B8" title="일본">일본</a>의 <a href="/wiki/%EC%B6%95%EA%B5%AC" title="축구">축구</a> 심판이다.</p>
<p>그는 <a href="/wiki/2004%EB%85%84" title="2004년">2004년</a>에 <a class="mw-redirect" href="/wiki/%EA%B5%AD%EC%A0%9C_%EC%B6%95%EA%B5%AC_%EC%97%B0%EB%A7%B9" title="국제 축구 연맹">국제 축구 연맹</a>으로부터 국제 심판 자격을 취득하였으며, <a href="/wiki/2007%EB%85%84_FIFA_U-17_%EC%9B%94%EB%93%9C%EC%BB%B5" title="2007년 FIFA U-17 월드컵">2007년 FIFA U-17 월드컵</a>에서 벨기에와 튀니지, 아르헨티나와 온두라스, 독일과 트리니다드 토바고의 조별 예선 세 경기, 페루와 가나의 8강전 경기, 스페인과 나이지리아의 결승전 경기의 주심을 맡았다. 또한 그는 <a href="/wiki/2008%EB%85%84_%EC%95%84%ED%94%84%EB%A6%AC%EC%B9%B4_%EB%84%A4%EC%9D%B4%EC%85%98%EC%8A%A4%EC%BB%B5" title="2008년 아프리카 네이션스컵">2008년 아프리카 네이션스컵</a>에서 아시아 국가 출신의 유일한 축구 심판으로 활동했으며, <a href="/wiki/2010%EB%85%84_FIFA_%EC%9B%94%EB%93%9C%EC%BB%B5" title="2010년 FIFA 월드컵">2010년 FIFA 월드컵</a>에서 우루과이와 프랑스, 스페인과 온두라스, 파라과이와 뉴질랜드의 조별 예선 세 경기, 네덜란드와 브라질의 8강전 경기의 주심을 맡기도 하였다.</p>
<p>그러나 그는 <a href="/wiki/2008%EB%85%84" title="2008년">2008년</a> <a href="/wiki/4%EC%9B%94_29%EC%9D%BC" title="4월 29일">4월 29일</a>에 열린 <a href="/wiki/FC_%EB%8F%84%EC%BF%84" title="FC 도쿄">FC 도쿄</a>와 <a href="/wiki/%EC%98%A4%EC%9D%B4%ED%83%80_%ED%8A%B8%EB%A6%AC%EB%8B%88%ED%83%80" title="오이타 트리니타">오이타 트리니타</a>의 <a href="/wiki/J%EB%A6%AC%EA%B7%B8" title="J리그">J리그</a> 9라운드 경기에서 주심에 항의하는 오이타의 우에모토 다이카이에게 "입 다물고 있어라"와 "죽어라"라는 내용의 폭언을 함으로써 구설수에 오르기도 하였다.<sup class="reference" id="cite_ref-1"><a href="#cite_note-1">[1]</a></sup></p>
<p>또한 <a href="/wiki/2014%EB%85%84_FIFA_%EC%9B%94%EB%93%9C%EC%BB%B5" title="2014년 FIFA 월드컵">2014년 FIFA 월드컵</a>에서 브라질과 크로아티아와의 개막전에서, 크로아티아 측 페널티 에어리어 내에 있는 <a href="/wiki/%ED%94%84%EB%A0%88%EB%8D%B0%EB%A6%AC%EC%BF%A0_%EC%83%A4%EB%B2%A0%EC%8A%A4_%EA%B2%8C%EC%A7%80%EC%8A%A4" title="프레데리쿠 샤베스 게지스">프레드</a>의 헐리우드 액션을 크로아티아의 반칙으로 판정하여 브라질 <a href="/wiki/%EB%84%A4%EC%9D%B4%EB%A7%88%EB%A5%B4" title="네이마르">네이마르</a>에게 논란의 <a class="mw-redirect" href="/wiki/%ED%8E%98%EB%84%90%ED%8B%B0%ED%82%A5" title="페널티킥">페널티킥</a>을 주어 성공시킴으로써 크로아티아 선수들과 축구팬들로부터 격렬한 항의를 받기도 하였다.</p>
<p>2014년 브라질 월드컵</p>
<p>브라질과 크로아티아의 월드컵 개회식 경기에서 오심을 일으켜 브라질이 월드컵 첫승을 가져가는데 큰 디여를 간접적 브라질 대표팀 일원이다.</p>
