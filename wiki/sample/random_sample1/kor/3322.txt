<p><b>니시야쓰시로 군</b>(<span style="font-size: smaller;"><a href="/wiki/%EC%9D%BC%EB%B3%B8%EC%96%B4" title="일본어">일본어</a>:</span> <span lang="ja" xml:lang="ja">西八代郡, にしやつしろぐん</span>)은 <a href="/wiki/%EC%9D%BC%EB%B3%B8" title="일본">일본</a> <a href="/wiki/%EC%95%BC%EB%A7%88%EB%82%98%EC%8B%9C_%ED%98%84" title="야마나시 현">야마나시 현</a> 북서부에 있는 군이다. 면적은 75.07km²이고 인구는 <a href="/wiki/2009%EB%85%84" title="2009년">2009년</a> <a href="/wiki/12%EC%9B%94_1%EC%9D%BC" title="12월 1일">12월 1일</a> 기준으로 17,222명이다.</p>
<p>한 개의 정으로 이루어져 있다.</p>
<ul>
<li><a href="/wiki/%EC%9D%B4%EC%B9%98%EC%B9%B4%EC%99%80%EB%AF%B8%EC%82%AC%ED%86%A0_%EC%A0%95" title="이치카와미사토 정">이치카와미사토 정</a>(<span lang="ja" xml:lang="ja">市川三郷町</span>)</li>
</ul>
<h2><span class="mw-headline" id=".EC.97.AD.EC.82.AC">역사</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%8B%88%EC%8B%9C%EC%95%BC%EC%93%B0%EC%8B%9C%EB%A1%9C_%EA%B5%B0&amp;action=edit&amp;section=1" title="부분 편집: 역사">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>쇼와의 대합병까지는 <a href="/wiki/%ED%9B%84%EC%A7%80_%EA%B0%95" title="후지 강">후지 강</a>의 좌안이 니시야쓰시로 군, 우안이 <a href="/wiki/%EB%AF%B8%EB%82%98%EB%AF%B8%EC%BD%94%EB%A7%88_%EA%B5%B0" title="미나미코마 군">미나미코마 군</a>이었지만 <a href="/wiki/1955%EB%85%84" title="1955년">1955년</a> <a href="/wiki/2%EC%9B%94_11%EC%9D%BC" title="2월 11일">2월 11일</a>에 오카와우치 정이 미나미코마 군 미노부 정 등 3정촌과의 합병으로 새롭게 미나미코마 군 <a href="/wiki/%EB%AF%B8%EB%85%B8%EB%B6%80_%EC%A0%95" title="미노부 정">미노부 정</a>이 되었고 동년 <a href="/wiki/4%EC%9B%94_1%EC%9D%BC" title="4월 1일">4월 1일</a>에는 사카에 촌이 미나미코마 군 무쓰아이 촌과의 합병으로 미나미코마군 <a href="/wiki/%EB%82%9C%EB%B6%80_%EC%A0%95_(%EC%95%BC%EB%A7%88%EB%82%98%EC%8B%9C_%ED%98%84)" title="난부 정 (야마나시 현)">난부 정</a>이 되었다. <a href="/wiki/1956%EB%85%84" title="1956년">1956년</a> <a href="/wiki/9%EC%9B%94_30%EC%9D%BC" title="9월 30일">9월 30일</a>에는 다이도 촌의 일부가, <a href="/wiki/1958%EB%85%84" title="1958년">1958년</a> <a href="/wiki/1%EC%9B%94_1%EC%9D%BC" title="1월 1일">1월 1일</a>에는 이치카와다이몬 정의 일부가 각각 미나미코마 군 <a href="/wiki/%EA%B0%80%EC%A7%80%EC%B9%B4%EC%9E%90%EC%99%80_%EC%A0%95" title="가지카자와 정">가지카자와 정</a>에, 또 1958년 4월 1일에는 시모베 정의 일부가 미나미코마 군 나카토미 정에 편입되어 니시야쓰시로 군의 구역은 서서히 축소되어 갔다.</p>
<p>게다가 <a href="/wiki/2004%EB%85%84" title="2004년">2004년</a> <a href="/wiki/9%EC%9B%94_13%EC%9D%BC" title="9월 13일">9월 13일</a>에는 시모베 정과 미나미코마 군 나카토미 정·미노부 정이 합병해 새롭게 미나미코마 군 미노부 정이 되었고 이후 <a href="/wiki/2006%EB%85%84" title="2006년">2006년</a> <a href="/wiki/3%EC%9B%94_1%EC%9D%BC" title="3월 1일">3월 1일</a>에는 <a href="/wiki/%EA%B0%80%EB%AF%B8%EC%BF%A0%EC%9D%B4%EC%8B%9C%ED%82%A4_%EC%B4%8C" title="가미쿠이시키 촌">가미쿠이시키 촌</a> 남부의 쇼지, 모토스 및 후지가네의 각지구는 미나미쓰루 군 <a href="/wiki/%ED%9B%84%EC%A7%80%EC%B9%B4%EC%99%80%EA%B5%AC%EC%B9%98%EC%BD%94_%EC%A0%95" title="후지카와구치코 정">후지카와구치코 정</a>, 북부의 가케하시 및 후루세키의 두 지방구는 <a href="/wiki/%EA%B3%A0%ED%9B%84_%EC%8B%9C" title="고후 시">고후 시</a>에 편입되어 후지 강 좌안의 광대한 구역을 가지고 있던 니시야쓰시로 군은 이치카와미사토 정의 1정만 남게 되었다.</p>
