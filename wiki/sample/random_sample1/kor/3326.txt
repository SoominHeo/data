<p><b>니시카스가이 군</b>(<span style="font-size: smaller;"><a href="/wiki/%EC%9D%BC%EB%B3%B8%EC%96%B4" title="일본어">일본어</a>:</span> <span lang="ja" xml:lang="ja">西春日井郡</span>)은 일본 <a href="/wiki/%EC%95%84%EC%9D%B4%EC%B9%98_%ED%98%84" title="아이치 현">아이치 현</a>에 있는 <a href="/wiki/%EA%B5%B0_(%ED%96%89%EC%A0%95_%EA%B5%AC%EC%97%AD)" title="군 (행정 구역)">군</a>이다. 오와리 지방의 중앙부에 위치한다.</p>
<p>한 개의 정으로 이루어져 있다.</p>
<ul>
<li><a href="/wiki/%EB%8F%84%EC%9A%94%EC%95%BC%EB%A7%88_%EC%A0%95" title="도요야마 정">도요야마 정</a>(<span lang="ja" xml:lang="ja">豊山町</span>)</li>
</ul>
<h2><span class="mw-headline" id=".EC.97.AD.EC.82.AC">역사</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%8B%88%EC%8B%9C%EC%B9%B4%EC%8A%A4%EA%B0%80%EC%9D%B4_%EA%B5%B0&amp;action=edit&amp;section=1" title="부분 편집: 역사">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>이전에는 <a class="mw-redirect" href="/wiki/%EB%82%98%EA%B3%A0%EC%95%BC_%EC%84%B1" title="나고야 성">나고야 성</a>(현재의 <a href="/wiki/%EB%82%98%EA%B3%A0%EC%95%BC_%EC%8B%9C" title="나고야 시">나고야 시</a> <a href="/wiki/%EB%8B%88%EC%8B%9C_%EA%B5%AC_(%EB%82%98%EA%B3%A0%EC%95%BC_%EC%8B%9C)" title="니시 구 (나고야 시)">니시 구</a> 및 <a href="/wiki/%EA%B8%B0%ED%83%80_%EA%B5%AC_(%EB%82%98%EA%B3%A0%EC%95%BC_%EC%8B%9C)" title="기타 구 (나고야 시)">기타 구</a>)의 근처까지 니시카스가이 군에 포함되어 있었다. 그러나 <a href="/wiki/%EB%8B%A4%EC%9D%B4%EC%87%BC_%EC%8B%9C%EB%8C%80" title="다이쇼 시대">다이쇼 시대</a>부터 <a href="/wiki/%EC%A0%9C2%EC%B0%A8_%EC%84%B8%EA%B3%84_%EB%8C%80%EC%A0%84" title="제2차 세계 대전">제2차 세계 대전</a> 후의 <a href="/wiki/1950%EB%85%84%EB%8C%80" title="1950년대">1950년대</a>에 걸쳐 행해진 몇 차례의 합병으로 서서히 나고야 시에 편입 되어 갔다.</p>
<p>군 전체를 1개의 시를 목표로 하는 구상이 있었지만 신중파와 적극파가 대립해 백지화 되었다. 그 후 남부의 니시비와지마 정·신카와 정·기요스 정의 3정이 <a href="/wiki/2005%EB%85%84" title="2005년">2005년</a>에 합병해 <a href="/wiki/%EA%B8%B0%EC%9A%94%EC%8A%A4_%EC%8B%9C" title="기요스 시">기요스 시</a>가 되었다. 또 <a href="/wiki/2006%EB%85%84" title="2006년">2006년</a>에는 니시하루 정과 시카쓰 정이 합병해 <a href="/wiki/%EA%B8%B0%ED%83%80%EB%82%98%EA%B3%A0%EC%95%BC_%EC%8B%9C" title="기타나고야 시">기타나고야 시</a>가 되었다. <a href="/wiki/2009%EB%85%84" title="2009년">2009년</a>에는 <a href="/wiki/%ED%95%98%EB%A3%A8%ED%9E%88_%EC%A0%95" title="하루히 정">하루히 정</a>이 기요스 시에 편입되어 현재는 도요야마 정만이 남는다.</p>
<ul>
<li><a href="/wiki/1878%EB%85%84" title="1878년">1878년</a> <a href="/wiki/7%EC%9B%94_22%EC%9D%BC" title="7월 22일">7월 22일</a> - 니시카스가이 군 성립하였다.</li>
<li><a href="/wiki/1963%EB%85%84" title="1963년">1963년</a> <a href="/wiki/11%EC%9B%94_1%EC%9D%BC" title="11월 1일">11월 1일</a> - 니시하루 촌이 정으로 승격해 니시하루 정이 되었다.(5정 2촌)</li>
<li><a href="/wiki/1972%EB%85%84" title="1972년">1972년</a> <a href="/wiki/4%EC%9B%94_1%EC%9D%BC" title="4월 1일">4월 1일</a> - 도요야마 촌이 정으로 승격해 도요야마 정이 되었다.(6정 1촌)</li>
<li><a href="/wiki/1990%EB%85%84" title="1990년">1990년</a> <a href="/wiki/4%EC%9B%94_1%EC%9D%BC" title="4월 1일">4월 1일</a> - 하루히 촌이 정으로 승격해 하루히 정이 되었다.(7정)</li>
<li><a href="/wiki/2005%EB%85%84" title="2005년">2005년</a> <a href="/wiki/7%EC%9B%94_7%EC%9D%BC" title="7월 7일">7월 7일</a> - 기요스 정·신카와 정·니시비와지마 정이 합병해 <a href="/wiki/%EA%B8%B0%EC%9A%94%EC%8A%A4_%EC%8B%9C" title="기요스 시">기요스 시</a>가 성립, 군에서 이탈하였다.(4정)</li>
<li><a href="/wiki/2006%EB%85%84" title="2006년">2006년</a> <a href="/wiki/3%EC%9B%94_20%EC%9D%BC" title="3월 20일">3월 20일</a> - 시카쓰 정·니시하루 정이 합병해 <a href="/wiki/%EA%B8%B0%ED%83%80%EB%82%98%EA%B3%A0%EC%95%BC_%EC%8B%9C" title="기타나고야 시">기타나고야 시</a>가 성립, 군에서 이탈하였다.(2정)</li>
<li><a href="/wiki/2009%EB%85%84" title="2009년">2009년</a> <a href="/wiki/10%EC%9B%94_1%EC%9D%BC" title="10월 1일">10월 1일</a> - 하루히 정이 기요스 시에 편입되었다.(1정)</li>
</ul>
