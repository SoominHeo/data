<p><b>니시키 정</b>(<span style="font-size: smaller;"><a href="/wiki/%EC%9D%BC%EB%B3%B8%EC%96%B4" title="일본어">일본어</a>:</span> <span lang="ja" xml:lang="ja">錦町</span>)은 <a href="/wiki/%EC%9D%BC%EB%B3%B8" title="일본">일본</a> <a href="/wiki/%EA%B5%AC%EB%A7%88%EB%AA%A8%ED%86%A0_%ED%98%84" title="구마모토 현">구마모토 현</a>의 남부에 위치한 <a href="/wiki/%EA%B5%AC%EB%A7%88_%EA%B5%B0" title="구마 군">구마 군</a>의 정이다. 면적은 84.93km²이고 인구는 <a href="/wiki/2010%EB%85%84" title="2010년">2010년</a> <a href="/wiki/3%EC%9B%94_1%EC%9D%BC" title="3월 1일">3월 1일</a> 기준으로 11,216명이다.</p>

<h2><span class="mw-headline" id=".EC.A7.80.EB.A6.AC">지리</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%8B%88%EC%8B%9C%ED%82%A4_%EC%A0%95&amp;action=edit&amp;section=1" title="부분 편집: 지리">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>구마모토 현의 남단부에 위치한다. <a href="/wiki/%EA%B5%AC%EB%A7%88%EB%AA%A8%ED%86%A0_%EC%8B%9C" title="구마모토 시">구마모토 시</a>에서는 남남동쪽으로 약 70km 떨어진 곳에 해당하며 정의 북부는 히토요시 분지의 일부에 포함되는 평탄한 지형으로 <a href="/wiki/%EA%B5%AC%EB%A7%88_%EA%B0%95" title="구마 강">구마 강</a>이 동서로 흐른다. 남부는 규슈 산지의 일부를 이루는 산이 많은 지형이다. 정역 남단부는 <a href="/wiki/%EB%AF%B8%EC%95%BC%EC%9E%90%ED%82%A4_%ED%98%84" title="미야자키 현">미야자키 현</a>과 접하고 있다.</p>
<h3><span class="mw-headline" id=".EC.9D.B8.EC.A0.91.ED.95.9C_.EC.9E.90.EC.B9.98.EC.B2.B4">인접한 자치체</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%8B%88%EC%8B%9C%ED%82%A4_%EC%A0%95&amp;action=edit&amp;section=2" title="부분 편집: 인접한 자치체">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<ul>
<li>구마모토 현
<ul>

<li>구마 군 <a href="/wiki/%EC%82%AC%EA%B0%80%EB%9D%BC_%EC%B4%8C" title="사가라 촌">사가라 촌</a>·<a href="/wiki/%EC%95%84%EC%82%AC%EA%B8%B0%EB%A6%AC_%EC%A0%95" title="아사기리 정">아사기리 정</a></li>
</ul>

<h2><span class="mw-headline" id=".EC.97.AD.EC.82.AC">역사</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%8B%88%EC%8B%9C%ED%82%A4_%EC%A0%95&amp;action=edit&amp;section=3" title="부분 편집: 역사">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<ul>
<li>1955년 7월 1일 - 니시 촌·이치부 촌·기노우에 촌이 신설 합병해 니시키 촌이 성립하였다.</li>
<li>1965년 4월 1일 - 정으로 승격하였다.</li>
</ul>
<h2><span class="mw-headline" id=".EA.B2.BD.EC.A0.9C">경제</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%8B%88%EC%8B%9C%ED%82%A4_%EC%A0%95&amp;action=edit&amp;section=4" title="부분 편집: 경제">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>2004년도 정내 총생산은 407억엔이었다. 농업은 정의 남부의 산이 많은 지역에서는 배·복숭아의 생산이 <a href="/wiki/%EA%B5%AC%EB%A7%88_%EA%B0%95" title="구마 강">구마 강</a>가의 평야에서는 쌀의 생산이 많다. 골프 등의 레저 관광도 이루어진다.</p>
