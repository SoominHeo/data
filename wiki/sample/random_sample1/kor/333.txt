<p>이 은 <a href="/wiki/1968%EB%85%84" title="1968년">1968년</a> <a href="/wiki/1%EC%9B%94_13%EC%9D%BC" title="1월 13일">1월 13일</a>에 <a href="/wiki/%EC%A1%B0%EC%84%A0%EB%AF%BC%EC%A3%BC%EC%A3%BC%EC%9D%98%EC%9D%B8%EB%AF%BC%EA%B3%B5%ED%99%94%EA%B5%AD" title="조선민주주의인민공화국">조선민주주의인민공화국</a> <a href="/wiki/%EC%A1%B0%EC%84%A0%EB%AF%BC%EC%A3%BC%EC%A3%BC%EC%9D%98%EC%9D%B8%EB%AF%BC%EA%B3%B5%ED%99%94%EA%B5%AD_%EB%AF%BC%EC%A1%B1%EB%B3%B4%EC%9C%84%EC%84%B1" title="조선민주주의인민공화국 민족보위성">민족보위성</a> <a class="mw-redirect" href="/wiki/%EB%AF%BC%EC%A1%B1%EB%B3%B4%EC%9C%84%EC%84%B1_%EC%A0%95%EC%B0%B0%EA%B5%AD" title="민족보위성 정찰국">정찰국</a>의 124부대 소속 31명이 조선인민군 정찰국장인 김정태로부터 <a href="/wiki/%EC%B2%AD%EC%99%80%EB%8C%80" title="청와대">청와대</a> 습격과 요인 암살 지령을 받아, <a href="/wiki/%EB%8C%80%ED%95%9C%EB%AF%BC%EA%B5%AD_%EA%B5%AD%EA%B5%B0" title="대한민국 국군">대한민국 국군</a>의 복장과 <a href="/wiki/%EC%88%98%EB%A5%98%ED%83%84" title="수류탄">수류탄</a> 및 <a href="/wiki/%EA%B8%B0%EA%B4%80%EB%8B%A8%EC%B4%9D" title="기관단총">기관단총</a>으로 무장하고, <a href="/wiki/1%EC%9B%94_17%EC%9D%BC" title="1월 17일">1월 17일</a> 자정을 기해 <a class="mw-redirect" href="/wiki/%ED%9C%B4%EC%A0%84%EC%84%A0" title="휴전선">휴전선</a> 군사분계선을 넘어 야간을 이용하여 <a class="mw-redirect" href="/wiki/%EB%8C%80%ED%95%9C%EB%AF%BC%EA%B5%AD_%EC%88%98%EB%8F%84%EA%B6%8C" title="대한민국 수도권">대한민국 수도권</a>에 잠입하였다. 이들은 <a href="/wiki/%EC%B2%AD%EC%9A%B4%EB%8F%99" title="청운동">청운동</a>의 <a href="/wiki/%EC%84%B8%EA%B2%80%EC%A0%95_(%EC%84%9C%EC%9A%B8)" title="세검정 (서울)">세검정</a> 고개의 <a href="/wiki/%EC%B0%BD%EC%9D%98%EB%AC%B8" title="창의문">창의문</a>을 통과하려다 비상근무 중이던 <a href="/wiki/%EA%B2%BD%EC%B0%B0" title="경찰">경찰</a>의 불심검문으로 정체가 드러나자, <a href="/wiki/%EC%88%98%EB%A5%98%ED%83%84" title="수류탄">수류탄</a> 및 기관단총을 쏘면서 저항하였다.</p>
<p>대한민국 군·경은 비상경계태세를 확립하고 현장으로 출동하여 소탕 작전을 벌였으며, <a href="/wiki/%EA%B2%BD%EA%B8%B0%EB%8F%84" title="경기도">경기도</a> 일원에 걸쳐 군경합동수색전을 <a href="/wiki/1%EC%9B%94_31%EC%9D%BC" title="1월 31일">1월 31일</a>까지 전개하였다. 현장에서 비상근무를 지휘하던 <a class="mw-redirect" href="/wiki/%EC%A2%85%EB%A1%9C%EA%B2%BD%EC%B0%B0%EC%84%9C" title="종로경찰서">종로경찰서</a>장 <a href="/wiki/%EC%B5%9C%EA%B7%9C%EC%8B%9D_(%EA%B2%BD%EC%B0%B0%EA%B4%80)" title="최규식 (경찰관)">최규식</a> 총경은 총탄에 맞아 사망하였고, 124부대 소속 31명 중 29명이 사살되고 <a href="/wiki/%EA%B9%80%EC%8B%A0%EC%A1%B0" title="김신조">김신조</a>는 투항하였으며, 한 명은 도주하여 북으로 넘어갔다. 도주한 이는 조선인민군 대장인 <a href="/wiki/%EB%B0%95%EC%9E%AC%EA%B2%BD" title="박재경">박재경</a>으로, 총정치국 부총국장을 역임하고 2000년, 2007년에 방한하여 송이버섯을 선물하기도 했다.<sup class="reference" id="cite_ref-1"><a href="#cite_note-1">[1]</a></sup></p>
<h2><span class="mw-headline" id=".EA.B2.B0.EA.B3.BC">결과</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=1%C2%B721_%EC%82%AC%ED%83%9C&amp;action=edit&amp;section=2" title="부분 편집: 결과">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>시도는 미수에 그쳤으나, 이 사건 이후에 남-북 간의 군사적 긴장관계를 이유로 하여, 박정희는 ‘국가안보 우선주의’를 선언하였다. 안보우선주의는 <a href="/wiki/%EB%85%B8%EB%8F%99%EC%A1%B0%ED%95%A9" title="노동조합">노동조합</a>과 <a class="mw-redirect" href="/wiki/%EB%AF%BC%EC%A3%BC%ED%99%94_%EC%9A%B4%EB%8F%99" title="민주화 운동">민주화 운동</a>을 탄압하는 근거가 되었으며, <a href="/wiki/%EB%8C%80%ED%95%9C%EB%AF%BC%EA%B5%AD_%EC%98%88%EB%B9%84%EA%B5%B0" title="대한민국 예비군">대한민국 예비군</a>과 <a href="/wiki/%EC%9C%A1%EA%B5%B03%EC%82%AC%EA%B4%80%ED%95%99%EA%B5%90" title="육군3사관학교">육군3사관학교</a>가 창설되고 <a href="/wiki/%EA%B3%A0%EB%93%B1%ED%95%99%EA%B5%90" title="고등학교">고등학교</a>와 <a class="mw-redirect" href="/wiki/%EB%8C%80%ED%95%99%EA%B5%90" title="대학교">대학교</a>에서 <a href="/wiki/%EA%B5%90%EB%A0%A8" title="교련">교련</a> 교육이 실시되는 계기가 되었다. 또한, <a href="/wiki/%EB%8C%80%ED%95%9C%EB%AF%BC%EA%B5%AD" title="대한민국">대한민국</a>은 특수부대인 <a class="mw-redirect" href="/wiki/684_%EB%B6%80%EB%8C%80" title="684 부대">684 부대</a>를 비밀리에 조직하여 보복성 공격을 계획하였으나, <a href="/wiki/%EB%AF%B8%EA%B5%AD" title="미국">미국</a>과 <a href="/wiki/%EC%86%8C%EB%A0%A8" title="소련">소련</a> 사이에 <a href="/wiki/%EB%8D%B0%ED%83%95%ED%8A%B8" title="데탕트">데탕트</a>가 조성되면서 불발에 그쳤다.<sup class="정리_필요 noprint">[<a href="/wiki/%EC%9C%84%ED%82%A4%EB%B0%B1%EA%B3%BC:%EC%B6%9C%EC%B2%98_%EB%B0%9D%ED%9E%88%EA%B8%B0" title="위키백과:출처 밝히기">출처 필요</a>]</sup></p>
<h2><span class="mw-headline" id=".EA.B8.B0.ED.83.80">기타</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=1%C2%B721_%EC%82%AC%ED%83%9C&amp;action=edit&amp;section=3" title="부분 편집: 기타">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p><a href="/wiki/%EA%B9%80%EC%9D%BC%EC%84%B1" title="김일성">김일성</a>은 <a href="/wiki/1972%EB%85%84" title="1972년">1972년</a> <a href="/wiki/7%C2%B74_%EB%82%A8%EB%B6%81_%EA%B3%B5%EB%8F%99_%EC%84%B1%EB%AA%85" title="7·4 남북 공동 성명">7·4 남북 공동 성명</a>으로, <a href="/wiki/%ED%8F%89%EC%96%91%EC%A7%81%ED%95%A0%EC%8B%9C" title="평양직할시">평양</a>을 방문한 <a href="/wiki/%EC%9D%B4%ED%9B%84%EB%9D%BD" title="이후락">이후락</a> 전 <a href="/wiki/%EB%8C%80%ED%95%9C%EB%AF%BC%EA%B5%AD" title="대한민국">대한민국</a> 중앙정보부장과의 오찬 자리에서 <strong class="selflink">1·21 청와대 기습 사건</strong>에 대해 자신은 아는 바가 없었다고 말하였으나, 1.21 사태 당시 남파공작원이었던 <a href="/wiki/%EA%B9%80%EC%8B%A0%EC%A1%B0" title="김신조">김신조</a>는 이를 북한의 상투적 거짓말이라고 말했다.<sup class="reference" id="cite_ref-2"><a href="#cite_note-2">[2]</a></sup></p>
<p><a href="/wiki/2013%EB%85%84" title="2013년">2013년</a> <a href="/wiki/10%EC%9B%94_24%EC%9D%BC" title="10월 24일">10월 24일</a> 평화문제연구소 창립 30주년 기념식에서 청샤오허(成曉河) 중국 인민대학 교수는 김일성이 <a href="/wiki/1965%EB%85%84" title="1965년">1965년</a>에 제2차 <a href="/wiki/%ED%95%9C%EA%B5%AD_%EC%A0%84%EC%9F%81" title="한국 전쟁">한국 전쟁</a>을 계획했지만 <a href="/wiki/%EC%A4%91%EA%B5%AD" title="중국">중국</a>이 거부했다는 중국 외교부 기밀해제 문서(No.106-01480-07)를 공개했다.<sup class="reference" id="cite_ref-3"><a href="#cite_note-3">[3]</a></sup></p>
