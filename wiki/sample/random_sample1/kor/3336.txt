<p><b>니조 정</b>(<span style="font-size: smaller;"><a href="/wiki/%EC%9D%BC%EB%B3%B8%EC%96%B4" title="일본어">일본어</a>:</span> <span lang="ja" xml:lang="ja">二丈町</span>)은 <a href="/wiki/%ED%9B%84%EC%BF%A0%EC%98%A4%EC%B9%B4_%ED%98%84" title="후쿠오카 현">후쿠오카 현</a> 서부에 있었던 <a href="/wiki/%EC%9D%B4%ED%86%A0%EC%8B%9C%EB%A7%88_%EA%B5%B0" title="이토시마 군">이토시마 군</a>의 정이다. 2010년 1월 1일, 인접하는 <a href="/wiki/%EB%A7%88%EC%97%90%EB%B0%94%EB%A3%A8_%EC%8B%9C" title="마에바루 시">마에바루 시</a>·이토시마 군 <a href="/wiki/%EC%8B%9C%EB%A7%88_%EC%A0%95_(%ED%9B%84%EC%BF%A0%EC%98%A4%EC%B9%B4_%ED%98%84)" title="시마 정 (후쿠오카 현)">시마 정</a>과 신설 합병해 소멸했다. 새로운 시명은 <a href="/wiki/%EC%9D%B4%ED%86%A0%EC%8B%9C%EB%A7%88_%EC%8B%9C" title="이토시마 시">이토시마 시</a>이다.</p>

<h2><span class="mw-headline" id=".EC.A7.80.EB.A6.AC">지리</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%8B%88%EC%A1%B0_%EC%A0%95&amp;action=edit&amp;section=1" title="부분 편집: 지리">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>후쿠오카 현 최서단에 해당한다. <a href="/wiki/%EC%82%AC%EA%B0%80_%ED%98%84" title="사가 현">사가 현</a>과의 현 경계에 접하고 있어 <a href="/wiki/%ED%9B%84%EC%BF%A0%EC%98%A4%EC%B9%B4_%EC%8B%9C" title="후쿠오카 시">후쿠오카 시</a>와 <a href="/wiki/%EA%B0%80%EB%9D%BC%EC%93%B0_%EC%8B%9C" title="가라쓰 시">가라쓰 시</a>를 연결하는 루트의 중간에 위치한다. 정의 중심부에 해당하는 <a href="/wiki/%EC%A7%80%EC%BF%A0%EC%A0%A0%ED%9B%84%EC%B9%B4%EC%97%90_%EC%97%AD" title="지쿠젠후카에 역">지쿠젠후카에 역</a>에서 동쪽은 후쿠오카 시의 <a class="mw-redirect" href="/wiki/%EB%B2%A0%EB%93%9C%ED%83%80%EC%9A%B4" title="베드타운">베드타운</a>으로서 주택지가 개발되고 있지만 서쪽은 온화한 어촌의 이미지가 남아 있다.</p>
<p>북쪽은 <a href="/wiki/%EA%B2%90%EC%B9%B4%EC%9D%B4%EB%82%98%EB%8B%A4" title="겐카이나다">겐카이나다</a>에 접하고 있고 남쪽은 세후리 산지에 의해서 사가 현과 가로막혀 있다.</p>
