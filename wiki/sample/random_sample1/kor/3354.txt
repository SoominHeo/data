<p><b>니콘 D200</b>(<span lang="en" xml:lang="en">Nikon D200</span>)은 <a href="/wiki/2005%EB%85%84" title="2005년">2005년</a> 11월 1일에 출시된 <a href="/wiki/%EB%8B%88%EC%BD%98" title="니콘">니콘</a>의 준 전문가용 <a class="mw-redirect" href="/wiki/%EB%94%94%EC%A7%80%ED%84%B8_SLR" title="디지털 SLR">디지털 SLR</a>이다. D200은 보급형 DSLR(<a href="/wiki/%EB%8B%88%EC%BD%98_D50" title="니콘 D50">니콘 D50</a>, <a class="mw-redirect" href="/wiki/%EB%8B%88%EC%BD%98_D70s" title="니콘 D70s">니콘 D70s</a>)과 최고급 전문가용 DSLR(<a class="mw-redirect" href="/wiki/%EB%8B%88%EC%BD%98_D2Hs" title="니콘 D2Hs">니콘 D2Hs</a>, <a class="mw-redirect" href="/wiki/%EB%8B%88%EC%BD%98_D2Xs" title="니콘 D2Xs">니콘 D2Xs</a>) 사이에 위치한다. D200은 <a href="/wiki/%EB%8B%88%EC%BD%98_D100" title="니콘 D100">니콘 D100</a>의 후속 모델로 볼 수 있다. 1,020만 <a class="mw-redirect" href="/wiki/%EC%9C%A0%ED%9A%A8_%ED%99%94%EC%86%8C" title="유효 화소">유효 화소</a>를 가지는 <a class="mw-redirect" href="/wiki/CCD" title="CCD">CCD</a>를 사용한다.</p>
<p>니콘의 D200은 <a href="/wiki/%EC%86%8C%EB%8B%88" title="소니">소니</a>의 <a class="mw-redirect" href="/wiki/CCD" title="CCD">CCD</a>를 사용하고 있다. 이 센서는 <a href="/wiki/%EB%8B%88%EC%BD%98_D80" title="니콘 D80">니콘 D80</a>, <a href="/wiki/%EC%86%8C%EB%8B%88_DSLR-A100" title="소니 DSLR-A100">소니 DSLR-A100</a> 등에 사용되는 ICX493AQA와는 다르다. <a href="/wiki/%ED%9B%84%EC%A7%80%ED%95%84%EB%A6%84" title="후지필름">후지필름</a>은 니콘에게서 D200의 바디를 넘겨받아 <a href="/wiki/%ED%8C%8C%EC%9D%B8%ED%94%BD%EC%8A%A4_S5_Pro" title="파인픽스 S5 Pro">파인픽스 S5 Pro</a>를 생산하고 있다.</p>
