<p><b>니콜라 알렉시 쥘리오 은쿨루 은두베나</b> (<span style="font-size: smaller;"><a href="/wiki/%ED%94%84%EB%9E%91%EC%8A%A4%EC%96%B4" title="프랑스어">프랑스어</a>:</span> <span lang="fr" xml:lang="fr">Nicolas Alexis Julio N'Koulou N'Doubena</span>, <a href="/wiki/1990%EB%85%84" title="1990년">1990년</a> <a href="/wiki/3%EC%9B%94_27%EC%9D%BC" title="3월 27일">3월 27일</a>, <a href="/wiki/%EC%95%BC%EC%9A%B4%EB%8D%B0" title="야운데">야운데</a> ~)는 불리는 <a href="/wiki/%EC%B9%B4%EB%A9%94%EB%A3%AC" title="카메룬">카메룬</a>의 축구 선수로, 현재 <a href="/wiki/%EB%A6%AC%EA%B7%B8_1" title="리그 1">리그 1</a>의 <a href="/wiki/%EC%98%AC%EB%9E%AD%ED%94%BC%ED%81%AC_%EB%A6%AC%EC%98%B9" title="올랭피크 리옹">올랭피크 리옹</a> 소속이다.</p>

<h2><span class="mw-headline" id=".EA.B2.BD.EB.A0.A5">경력</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%8B%88%EC%BD%9C%EB%9D%BC_%EC%9D%80%EC%BF%A8%EB%A3%A8&amp;action=edit&amp;section=1" title="부분 편집: 경력">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>은쿨루는 <a href="/wiki/%EB%91%90%EC%95%8C%EB%9D%BC" title="두알라">두알라</a>의 카지 스포츠 아카데미에서 축구계에 입문하였다. 이 아카데미는 <a href="/wiki/%EC%B9%B4%EB%A9%94%EB%A3%AC" title="카메룬">카메룬</a>의 주요 선수들인 <a href="/wiki/%EC%82%AC%EB%AE%88%EC%97%98_%EC%97%90%ED%86%A0" title="사뮈엘 에토">사뮈엘 에토</a>, <a href="/wiki/%EC%9D%B4%EB%93%9C%EB%A6%AC%EC%8A%A4_%EC%B9%B4%EB%A5%BC%EB%A1%9C%EC%8A%A4_%EC%B9%B4%EB%A9%94%EB%8B%88" title="이드리스 카를로스 카메니">이드리스 카를로스 카메니</a>, <a class="new" href="/w/index.php?title=%EC%97%90%EB%A6%AD_%EC%A0%AC%EB%B0%94-%EC%A0%AC%EB%B0%94&amp;action=edit&amp;redlink=1" title="에릭 젬바-젬바 (없는 문서)">에릭 젬바-젬바</a> 등을 배출한 곳이다. 2007년 말, 그는 <a href="/wiki/%EB%AA%A8%EB%82%98%EC%BD%94" title="모나코">모나코</a>를 연고로 하는 <a href="/wiki/AS_%EB%AA%A8%EB%82%98%EC%BD%94_FC" title="AS 모나코 FC">AS 모나코 FC</a>로 이적하였다. 그는 3년 계약을 체결하였다.<sup class="reference" id="cite_ref-ASM_1-0"><a href="#cite_note-ASM-1">[1]</a></sup> 그는 2007-08 시즌의 후반을 모나코의 CFA 팀에서 활약하였고, 2008년 7월 프리시즌에 프로팀에 합류하여, 등번호 3번을 부여받았다.</p>
<p>그는 프리시즌에 2경기 출장하였는데, 그 경기들은 <a href="/wiki/%EB%A6%AC%EA%B7%B8_1" title="리그 1">리그 1</a>에 속한 <a href="/wiki/%ED%88%B4%EB%A3%A8%EC%A6%88_FC" title="툴루즈 FC">툴루즈 FC</a>와의 첫 프리시즌 경기와, <a href="/wiki/%ED%81%AC%EB%A1%9C%EC%95%84%ED%8B%B0%EC%95%84" title="크로아티아">크로아티아</a>의 <a class="mw-redirect" href="/wiki/NK_%EB%94%94%EB%82%98%EB%AA%A8_%EC%9E%90%EA%B7%B8%EB%A0%88%EB%B8%8C" title="NK 디나모 자그레브">NK 디나모 자그레브</a>와의 경기였다. 그는 2008년 9월 13일 프로 데뷔전을 <a href="/wiki/FC_%EB%A1%9C%EB%A6%AC%EC%95%99" title="FC 로리앙">FC 로리앙</a> 전에서 65분에 교체로 필드를 나갈때까지 선발로 출전하였다. 그는 모든 잔여 경기에서 90분 풀타임을 소화하였다. 그러나 그는 한골도 넣지 못하였다.</p>
<p>그의 모나코에서의 성과는 유럽 명문 클럽 스카우트의 주목을 받았다. <a href="/wiki/%ED%94%84%EB%A6%AC%EB%AF%B8%EC%96%B4%EB%A6%AC%EA%B7%B8" title="프리미어리그">프리미어리그</a>의 <a href="/wiki/%EC%95%84%EC%8A%A4%EB%84%90_FC" title="아스널 FC">아스널 FC</a><sup class="reference" id="cite_ref-Arsenal_2-0"><a href="#cite_note-Arsenal-2">[2]</a></sup> 와 <a href="/wiki/%ED%97%90_%EC%8B%9C%ED%8B%B0_AFC" title="헐 시티 AFC">헐 시티 AFC</a>,<sup class="reference" id="cite_ref-HCFC_3-0"><a href="#cite_note-HCFC-3">[3]</a></sup> <a class="mw-redirect" href="/wiki/%ED%94%84%EB%A6%AC%EB%A9%94%EB%9D%BC%EB%A6%AC%EA%B0%80" title="프리메라리가">라 리가</a>의 <a href="/wiki/%EC%84%B8%EB%B9%84%EC%95%BC_FC" title="세비야 FC">세비야 FC</a>와 <a href="/wiki/%EC%95%84%ED%8B%80%EB%A0%88%ED%8B%B0%EC%BD%94_%EB%A7%88%EB%93%9C%EB%A6%AC%EB%93%9C" title="아틀레티코 마드리드">아틀레티코 마드리드</a>,<sup class="reference" id="cite_ref-SFCnATM_4-0"><a href="#cite_note-SFCnATM-4">[4]</a></sup> <a href="/wiki/%ED%94%84%EB%9E%91%EC%8A%A4" title="프랑스">프랑스</a> 거함 <a href="/wiki/%EC%98%AC%EB%9E%AD%ED%94%BC%ED%81%AC_%EB%A6%AC%EC%98%B9" title="올랭피크 리옹">올랭피크 리옹</a>은 모두 이 카메룬의 수비수에 주목하였다.<sup class="reference" id="cite_ref-scout_5-0"><a href="#cite_note-scout-5">[5]</a></sup> 2011년 6월 29일, 그는 <a href="/wiki/%EC%98%AC%EB%9E%AD%ED%94%BC%ED%81%AC_%EB%93%9C_%EB%A7%88%EB%A5%B4%EC%84%B8%EC%9C%A0" title="올랭피크 드 마르세유">올랭피크 드 마르세유</a>의 메디컬 테스트를 통과하였고, 4년계약을 체결하였다. 그러나 <a href="/wiki/%EC%95%84%EC%8A%A4%EB%84%90_FC" title="아스널 FC">아스널 FC</a>는 2012년 1월 이적시장에서 그를 원한다고 밝혔다.<sup class="reference" id="cite_ref-WTM2012_6-0"><a href="#cite_note-WTM2012-6">[6]</a></sup></p>
<h2><span class="mw-headline" id=".EA.B5.AD.EA.B0.80.EB.8C.80.ED.91.9C.ED.8C.80_.EA.B2.BD.EB.A0.A5">국가대표팀 경력</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%8B%88%EC%BD%9C%EB%9D%BC_%EC%9D%80%EC%BF%A8%EB%A3%A8&amp;action=edit&amp;section=2" title="부분 편집: 국가대표팀 경력">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>은쿨루는 카메룬 U-23 대표팀에 차출되어 <a href="/wiki/2008%EB%85%84_%ED%95%98%EA%B3%84_%EC%98%AC%EB%A6%BC%ED%94%BD" title="2008년 하계 올림픽">2008년 하계 올림픽</a>에 참가하였다. 카메룬은 8강에 진출하였으나, 그 단계에서 <a href="/wiki/%EB%B8%8C%EB%9D%BC%EC%A7%88_%EC%B6%95%EA%B5%AC_%EA%B5%AD%EA%B0%80%EB%8C%80%ED%91%9C%ED%8C%80" title="브라질 축구 국가대표팀">브라질</a>을 만나 탈락하였다. 그는 2008년 11월 19일, <a href="/wiki/%EB%82%A8%EC%95%84%ED%94%84%EB%A6%AC%EC%B9%B4_%EA%B3%B5%ED%99%94%EA%B5%AD_%EC%B6%95%EA%B5%AC_%EA%B5%AD%EA%B0%80%EB%8C%80%ED%91%9C%ED%8C%80" title="남아프리카 공화국 축구 국가대표팀">남아공</a>전에서 성인 대표팀 데뷔전을 치뤘다.</p>
