<h1 class="firstHeading" id="firstHeading" lang="ko">니콜라오스 플라스티라스</h1>
<div class="mw-body-content" id="bodyContent">
<div id="siteSub">위키백과, 우리 모두의 백과사전.</div>
<div id="contentSub"></div>
<div class="mw-jump" id="jump-to-nav">
					이동:					<a href="#mw-head">둘러보기</a>, 					<a href="#p-search">검색</a>
</div>
<div class="mw-content-ltr" dir="ltr" id="mw-content-text" lang="ko"><script>function mfTempOpenSection(id){var block=document.getElementById("mf-section-"+id);block.className+=" open-block";block.previousSibling.className+=" open-block";}</script><p><b>니콜라오스 플라스티라스</b>(<a href="/wiki/%EA%B7%B8%EB%A6%AC%EC%8A%A4%EC%96%B4" title="그리스어">그리스어</a>: Νικόλαος Πλαστήρας, 1883년 11월 4일 ~ 1953년 7월 26일)은 <a href="/wiki/%EA%B7%B8%EB%A6%AC%EC%8A%A4" title="그리스">그리스</a>의 장군이며, <a href="/wiki/%EA%B7%B8%EB%A6%AC%EC%8A%A4%EC%9D%98_%EC%B4%9D%EB%A6%AC" title="그리스의 총리">그리스의 총리</a>를 세 번 역임한 인물이다. 용맹함으로 두각을 드러내어 <a class="mw-redirect" href="/wiki/%EA%B7%B8%EB%A6%AC%EC%8A%A4-%ED%84%B0%ED%82%A4_%EC%A0%84%EC%9F%81_(1919%EB%85%84_~_1922%EB%85%84)" title="그리스-터키 전쟁 (1919년 ~ 1922년)">그리스-터키 전쟁 (1919년 ~ 1922년)</a> 당시 '검은 기사'로 알려지기도 하였다. 그리스가 전쟁에서 패배하자 다른 <a class="new" href="/w/index.php?title=%EB%B2%A0%EB%8B%88%EC%A0%A4%EB%A1%9C%EC%8A%A4%ED%8C%8C&amp;action=edit&amp;redlink=1" title="베니젤로스파 (없는 문서)">베니젤로스파</a> 군 장교들과 함께 <a href="/wiki/%EC%BD%98%EC%8A%A4%ED%83%84%ED%8B%B0%EB%85%B8%EC%8A%A4_1%EC%84%B8" title="콘스탄티노스 1세">콘스탄티노스 1세</a> 임금과 왕정 정부에 대항해 정변을 일으켰다. 군부가 주도한 정권은 1924년 1월까지 집권하였으며, 권력을 의회에 이양하여 <a href="/wiki/%EA%B7%B8%EB%A6%AC%EC%8A%A4_%EC%A0%9C2%EA%B3%B5%ED%99%94%EA%B5%AD" title="그리스 제2공화국">그리스 제2공화국</a>이 성립되었다. <a href="/wiki/%EC%A0%84%EA%B0%84%EA%B8%B0" title="전간기">전간기</a>에 플라스티라스는 헌신적인 베니젤로스파이자 공화주의자로 남았다. 왕당파였던 <a class="new" href="/w/index.php?title=%EC%9D%B8%EB%AF%BC%EB%8B%B9_(%EA%B7%B8%EB%A6%AC%EC%8A%A4)&amp;action=edit&amp;redlink=1" title="인민당 (그리스) (없는 문서)">인민당</a>의 발흥과 왕정 복고를 막고자 그는 1933년과 1935년에 두 차례에 걸쳐 정변을 일으켰으나 실패하고 <a href="/wiki/%ED%94%84%EB%9E%91%EC%8A%A4" title="프랑스">프랑스</a>로 망명하였다.</p>
<p><a href="/wiki/%EC%A0%9C2%EC%B0%A8_%EC%84%B8%EA%B3%84_%EB%8C%80%EC%A0%84" title="제2차 세계 대전">제2차 세계 대전</a> 당시 <a href="/wiki/%EC%B6%94%EC%B6%95%EA%B5%AD_%EC%A0%90%EB%A0%B9%EA%B8%B0%EC%9D%98_%EA%B7%B8%EB%A6%AC%EC%8A%A4" title="추축국 점령기의 그리스">추축국이 그리스를 점령</a>했을 때, 그는 <a href="/wiki/%EB%A7%88%EB%A5%B4%EC%84%B8%EC%9C%A0" title="마르세유">마르세유</a>에서 망명 생활을 하였으나 저항 단체 <a class="new" href="/w/index.php?title=%EA%B7%B8%EB%A6%AC%EC%8A%A4_%EA%B5%AD%EB%AF%BC_%EB%AF%BC%EC%A3%BC_%EC%97%B0%EB%8C%80&amp;action=edit&amp;redlink=1" title="그리스 국민 민주 연대 (없는 문서)">그리스 국민 민주 연대</a>의 명목상 지도자였다. 그리스가 해방된 뒤 그는 귀국하여 중도파로 총리직을 세 번 역임하였으며, <a class="new" href="/w/index.php?title=%EC%9E%90%EC%9C%A0%EB%8B%B9_(%EA%B7%B8%EB%A6%AC%EC%8A%A4)&amp;action=edit&amp;redlink=1" title="자유당 (그리스) (없는 문서)">자유당</a>과 연정을 구성하기도 하였다. 그의 마지막 두 정부에서 그는 <a href="/wiki/%EA%B7%B8%EB%A6%AC%EC%8A%A4_%EB%82%B4%EC%A0%84" title="그리스 내전">그리스 내전</a>에 따른 사회 갈등을 해소하고자 노력하였으나 실패하였다.</p>
