<p><b>니콜라이 겐나디예비치 바소프</b>(<span style="font-size: smaller;"><a href="/wiki/%EB%9F%AC%EC%8B%9C%EC%95%84%EC%96%B4" title="러시아어">러시아어</a>:</span> <span lang="ru" xml:lang="ru">Никола́й Генна́диевич Ба́сов</span>, <a href="/wiki/1922%EB%85%84" title="1922년">1922년</a> <a href="/wiki/12%EC%9B%94_14%EC%9D%BC" title="12월 14일">12월 14일</a> ~ <a href="/wiki/2001%EB%85%84" title="2001년">2001년</a> <a href="/wiki/7%EC%9B%94_1%EC%9D%BC" title="7월 1일">7월 1일</a>) 는 <a href="/wiki/%EB%9F%AC%EC%8B%9C%EC%95%84" title="러시아">러시아</a>의 물리학자이다. <a href="/wiki/1953%EB%85%84" title="1953년">1953년</a> 러시아의 물리학자 <a href="/wiki/%EC%95%8C%EB%A0%89%EC%82%B0%EB%93%9C%EB%A5%B4_%ED%94%84%EB%A1%9C%ED%98%B8%EB%A1%9C%ED%94%84" title="알렉산드르 프로호로프">프로호로프</a>와 함께 <a href="/wiki/%EC%A0%84%ED%8C%8C" title="전파">전파</a>를 증폭시킬 때 <a href="/wiki/%EB%B6%84%EC%9E%90" title="분자">분자</a>의 <a href="/wiki/%EC%97%90%EB%84%88%EC%A7%80" title="에너지">에너지</a>를 이용하는 원리를 알아냈다. 이 두 사람은 또한 1955년에 이 증폭기의 작동 방법을 고쳐서 메이저라는 새로운 증폭기를 발명했다. 이 업적으로 두 사람은 <a href="/wiki/1964%EB%85%84" title="1964년">1964년</a> <a href="/wiki/%EB%AF%B8%EA%B5%AD" title="미국">미국</a>의 물리학자 <a class="mw-redirect" href="/wiki/%ED%83%80%EC%9A%B4%EC%8A%A4" title="타운스">타운스</a>와 함께 <a class="mw-redirect" href="/wiki/%EB%85%B8%EB%B2%A8%EB%AC%BC%EB%A6%AC%ED%95%99%EC%83%81" title="노벨물리학상">노벨물리학상</a>을 받았다.</p>
