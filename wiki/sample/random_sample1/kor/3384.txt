<p><b>닌코 천황</b>(<span style="font-size: smaller;"><a href="/wiki/%EC%9D%BC%EB%B3%B8%EC%96%B4" title="일본어">일본어</a>:</span> <span lang="ja" xml:lang="ja">仁孝天皇</span>, <a href="/wiki/1800%EB%85%84" title="1800년">1800년</a> <a href="/wiki/3%EC%9B%94_16%EC%9D%BC" title="3월 16일">3월 16일</a> ~ <a href="/wiki/1846%EB%85%84" title="1846년">1846년</a> <a href="/wiki/2%EC%9B%94_21%EC%9D%BC" title="2월 21일">2월 21일</a>)은 <a href="/wiki/%EC%9D%BC%EB%B3%B8" title="일본">일본</a>의 제120대 <a href="/wiki/%EC%9D%BC%EB%B3%B8_%EC%B2%9C%ED%99%A9" title="일본 천황">천황</a>이다.</p>
<p>1817년부터 1846년까지 재위하는 동안 1817년부터 1837년까지 <a href="/wiki/%EB%8F%84%EC%BF%A0%EA%B0%80%EC%99%80_%EC%9D%B4%EC%97%90%EB%82%98%EB%A6%AC" title="도쿠가와 이에나리">도쿠가와 이에나리</a>가 <a href="/wiki/%EC%84%AD%EC%A0%95" title="섭정">섭정</a>하였고 1837년부터 1846년 붕어할 때까지 <a href="/wiki/%EB%8F%84%EC%BF%A0%EA%B0%80%EC%99%80_%EC%9D%B4%EC%97%90%EC%9A%94%EC%8B%9C" title="도쿠가와 이에요시">도쿠가와 이에요시</a>가 <a href="/wiki/%EB%8C%80%EB%A6%AC%EC%B2%AD%EC%A0%95" title="대리청정">대리청정</a>하였다.</p>
<p>교토 어소(<span lang="ja" xml:lang="ja">京都御所</span>)의 내부에 학문 연구 기관을 두어 훗날 <a href="/wiki/%EA%B0%80%EC%BF%A0%EC%8A%88%EC%9D%B8" title="가쿠슈인">가쿠슈인</a>의 전신이 되었다.</p>
<h2><span class="mw-headline" id=".EA.B0.80.EC.A1.B1">가족</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%8B%8C%EC%BD%94_%EC%B2%9C%ED%99%A9&amp;action=edit&amp;section=1" title="부분 편집: 가족">원본 편집</a><span class="mw-editsection-bracket">]</span></span></h2>
