<p><b>닌텐도 2DS</b>(Nintendo 2DS)는 <a href="/wiki/%EB%8B%8C%ED%85%90%EB%8F%84" title="닌텐도">닌텐도</a>가 발매한 <a href="/wiki/%ED%9C%B4%EB%8C%80%EC%9A%A9_%EA%B2%8C%EC%9E%84%EA%B8%B0" title="휴대용 게임기">휴대용 게임기</a>이며, <a href="/wiki/%EB%8B%8C%ED%85%90%EB%8F%84_3DS_XL" title="닌텐도 3DS XL">닌텐도 3DS XL</a>의 하위기종이다. 기존 3DS, 3DS XL의 3D기능을 뺀 저가형 모델이다.</p>
<p><a href="/wiki/2013%EB%85%84" title="2013년">2013년</a> <a href="/wiki/8%EC%9B%94_28%EC%9D%BC" title="8월 28일">8월 28일</a>에 IGN 공식 채널을 통해 발표되었다. 가격은 129.99달러로, <a href="/wiki/%EB%AF%B8%EA%B5%AD" title="미국">미국</a>과 <a href="/wiki/%EC%9C%A0%EB%9F%BD_%EC%97%B0%ED%95%A9" title="유럽 연합">유럽 연합</a>에서 <a href="/wiki/2013%EB%85%84" title="2013년">2013년</a> <a href="/wiki/10%EC%9B%94_12%EC%9D%BC" title="10월 12일">10월 12일</a>에 먼저 발매하였다. 대한민국에서는 <a href="/wiki/%EA%B5%AD%EB%A6%BD%EC%A0%84%ED%8C%8C%EC%97%B0%EA%B5%AC%EC%9B%90" title="국립전파연구원">국립전파연구원</a>에서 2013년 7월 26일 FTR-001라는 제품명으로 전파인증을 받은 바 있으나, <a href="/wiki/%ED%95%9C%EA%B5%AD%EB%8B%8C%ED%85%90%EB%8F%84" title="한국닌텐도">한국닌텐도</a>측에서는 레드X화이트 색상과 블랙X블루 색상으로 발매된 유럽 연합판을 역수출하는 형식으로 발매일을 <a href="/wiki/2013%EB%85%84" title="2013년">2013년</a> <a href="/wiki/12%EC%9B%94_7%EC%9D%BC" title="12월 7일">12월 7일</a>로 결정했다. 그리고 미국 및 유럽 연합에서는 핑크×화이트와 그린×화이트가 발매된다. 일본에서는 다수 국가에서만 발매했던 이후로 <a href="/wiki/2016%EB%85%84" title="2016년">2016년</a> <a href="/wiki/2%EC%9B%94_27%EC%9D%BC" title="2월 27일">2월 27일</a>에 3년으로 인해 늦게 발매되었다.</p>
<h2><span class="mw-headline" id=".EA.B0.9C.EC.9A.94">개요</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%8B%8C%ED%85%90%EB%8F%84_2DS&amp;action=edit&amp;section=1" title="부분 편집: 개요">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p><a href="/wiki/%EB%8B%8C%ED%85%90%EB%8F%84_3DS" title="닌텐도 3DS">닌텐도 3DS</a>에서 3D 영상 기능과 폴딩 기능을 제거하고 <a href="/wiki/%EC%8A%A4%ED%85%8C%EB%A0%88%EC%98%A4" title="스테레오">스테레오</a> 스피커를 <a class="mw-disambig" href="/wiki/%EB%AA%A8%EB%85%B8" title="모노">모노</a>로 바꿈으로써, 닌텐도 3DS보다 가격을 낮춘 제품이다. 후면의 3D 카메라는 남아 있고, 화면은 2개이지만 비용 절감을 위해서 액정 자체는 1장이고 불필요한 부분을 커버로 가리는 것으로 화면이 2개처럼 보이는 것이다. 폴딩 기능을 제거함으로써 슬립모드로 전환되는 스위치가 생겼다. 이와 같은 것으로, 닌텐도 <a href="/wiki/Wii" title="Wii">Wii</a>의 저가형 모델인 <a class="new" href="/w/index.php?title=Wii_mini&amp;action=edit&amp;redlink=1" title="Wii mini (없는 문서)">Wii mini</a>가 있다. 또 아시아를 제외한 유럽, 아프리카, 북미 지역에 핑크랑 화이트(PINK+WHITE)가 섞인 닌텐도 2DS를 출시 예정이다.</p>
