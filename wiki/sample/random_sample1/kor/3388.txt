<p><b>닐스 빌헬름 가데</b>(Niels Wilhelm Gade, <a href="/wiki/1817%EB%85%84" title="1817년">1817년</a> <a href="/wiki/2%EC%9B%94_22%EC%9D%BC" title="2월 22일">2월 22일</a>~<a href="/wiki/1890%EB%85%84" title="1890년">1890년</a> <a href="/wiki/12%EC%9B%94_21%EC%9D%BC" title="12월 21일">12월 21일</a>)는 <a href="/wiki/%EB%8D%B4%EB%A7%88%ED%81%AC" title="덴마크">덴마크</a>의 작곡가이자 지휘자, 바이올리니스트, 오르가니스트이다. 자기 나라에서는 게제로 불린다. 덴마크 국민음악의 최초의 대음악가로 인접국 <a href="/wiki/%EB%85%B8%EB%A5%B4%EC%9B%A8%EC%9D%B4" title="노르웨이">노르웨이</a>의 <a href="/wiki/%EC%97%90%EB%93%9C%EB%B0%94%EB%A5%B4_%EA%B7%B8%EB%A6%AC%EA%B7%B8" title="에드바르 그리그">그리그</a> 등에도 정신적인 영향을 준 사람이나, 그의 작풍은 반드시 민족주의적은 아니며 도리어 그가 사숙하던 <a href="/wiki/%ED%8E%A0%EB%A6%AD%EC%8A%A4_%EB%A9%98%EB%8D%B8%EC%8A%A4%EC%A1%B4" title="펠릭스 멘델스존">멘델스존</a>에 가까운 조기(早期) 낭만적 음악이다. <a href="/wiki/%EC%BD%94%ED%8E%9C%ED%95%98%EA%B2%90" title="코펜하겐">코펜하겐</a>의 악기 제조업자의 아들로 태어나 A. P. 베르그렌에게 작곡을 배워 24세 때 멘델스존의 서곡 &lt;핑갈의 동굴&gt;의 덴마크판이라고 할 서곡 &lt;오시안의 여운&gt;으로 새로 창설된 코펜하겐 음악협회의 상금을 획득하여 <a href="/wiki/%EB%8F%85%EC%9D%BC" title="독일">독일</a>과 <a href="/wiki/%EC%9D%B4%ED%83%88%EB%A6%AC%EC%95%84" title="이탈리아">이탈리아</a>에 유학하였다. 그 후 제1교향곡으로 국가의 장학금을 탔고, <a href="/wiki/%EB%9D%BC%EC%9D%B4%ED%94%84%EC%B9%98%ED%9E%88" title="라이프치히">라이프치히</a>에 유학하였으며 거기서 멘델스존과 알게 되어 라이프치히 음악원의 교수와 게반트하우스 관현악단의 지휘자로 있었으며 1847년에 멘델스존이 죽은 뒤 약 1년 동안 그의 후임으로 게반트하우스의 연주회에 관한 전권을 위임맡았다. 1848년, 전쟁 때문에 귀국하여 1850년에 코펜하겐 음악협회의 지휘자에 취임하는 등 여러 가지 요직을 겸하면서 연주활동과 창작을 같이하였다. 연주가로서는 <a class="mw-redirect" href="/wiki/%EB%B0%94%ED%9D%90" title="바흐">바흐</a>의 &lt;마태 수난곡&gt;, <a class="mw-redirect" href="/wiki/%EB%B2%A0%ED%86%A0%EB%B2%A4" title="베토벤">베토벤</a>의 제9교향곡의 덴마크 초연을 비롯하여 <a class="mw-redirect" href="/wiki/%EB%B2%A0%EB%A5%BC%EB%A6%AC%EC%98%A4%EC%A6%88" title="베를리오즈">베를리오즈</a>, <a href="/wiki/%EB%A6%AC%ED%95%98%EB%A5%B4%ED%8A%B8_%EB%B0%94%EA%B7%B8%EB%84%88" title="리하르트 바그너">바그너</a>, <a href="/wiki/%ED%94%84%EB%9E%80%EC%B8%A0_%EB%A6%AC%EC%8A%A4%ED%8A%B8" title="프란츠 리스트">리스트</a>, <a class="mw-redirect" href="/wiki/%EB%B8%8C%EB%9E%8C%EC%8A%A4" title="브람스">브람스</a> 등의 명곡을 차례로 소개하였지만, 작곡가로서도 8번까지 있는 교향곡, 다수의 칸타타와 오라토리오, 피아노곡, 실내악곡 등을 많이 남겼다.</p>
