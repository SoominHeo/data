<p><b>닛산 자동차</b>(<span style="font-size: smaller;"><a href="/wiki/%EC%98%81%EC%96%B4" title="영어">영어</a>:</span> <span lang="en" xml:lang="en">Nissan Motor</span>, <span style="font-size: smaller;"><a href="/wiki/%EC%9D%BC%EB%B3%B8%EC%96%B4" title="일본어">일본어</a>:</span> <span lang="ja" xml:lang="ja">日産自動車</span>)는 <a href="/wiki/%EC%9D%BC%EB%B3%B8" title="일본">일본</a>의 자동차 회사로 본부는 <a href="/wiki/%EA%B0%80%EB%82%98%EA%B0%80%EC%99%80_%ED%98%84" title="가나가와 현">가나가와 현</a> <a href="/wiki/%EC%9A%94%EC%BD%94%ED%95%98%EB%A7%88_%EC%8B%9C" title="요코하마 시">요코하마 시</a> 니시 구에 있다. 닛산과 <a href="/wiki/%EB%A5%B4%EB%85%B8" title="르노">르노</a>는 서로 상대의 주식을 보유하고 있어, <a class="extiw" href="https://en.wikipedia.org/wiki/Renault%E2%80%93Nissan_Alliance" title="en:Renault–Nissan Alliance">르노-닛산 얼라인언스</a>으로 대등한 관계를 맺고이지만 실제로는 <a href="/wiki/%ED%94%84%EB%9E%91%EC%8A%A4" title="프랑스">프랑스</a> 국내법의 제한으로 닛산은 <a href="/wiki/%EB%A5%B4%EB%85%B8" title="르노">르노</a>의 의결권을 행사할 수 없다. 현재의 사장은 카를로스 곤이며, <a href="/wiki/%EC%9D%BC%EB%B3%B8" title="일본">일본</a> 내에서는 <a href="/wiki/%ED%9B%84%EC%9A%94_%EA%B7%B8%EB%A3%B9" title="후요 그룹">후요 그룹</a>에 속한다.</p>

<h3><span class="mw-headline" id=".EC.9D.B8.ED.94.BC.EB.8B.88.ED.8B.B0">인피니티</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%8B%9B%EC%82%B0_%EC%9E%90%EB%8F%99%EC%B0%A8&amp;action=edit&amp;section=3" title="부분 편집: 인피니티">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<div class="rellink relarticle mainarticle"><img alt="" data-file-height="128" data-file-width="128" height="16" src="//upload.wikimedia.org/wikipedia/commons/thumb/e/ec/Crystal_Clear_app_xmag.svg/16px-Crystal_Clear_app_xmag.svg.png" srcset="//upload.wikimedia.org/wikipedia/commons/thumb/e/ec/Crystal_Clear_app_xmag.svg/24px-Crystal_Clear_app_xmag.svg.png 1.5x, //upload.wikimedia.org/wikipedia/commons/thumb/e/ec/Crystal_Clear_app_xmag.svg/32px-Crystal_Clear_app_xmag.svg.png 2x" width="16"/> 이 부분의 본문은 <a href="/wiki/%EC%9D%B8%ED%94%BC%EB%8B%88%ED%8B%B0" title="인피니티">인피니티</a>입니다.</div>
<p><a href="/wiki/1989%EB%85%84" title="1989년">1989년</a>에 <a href="/wiki/%ED%86%A0%EC%9A%94%ED%83%80_%EC%9E%90%EB%8F%99%EC%B0%A8" title="토요타 자동차">도요타</a>의 <a href="/wiki/%EB%A0%89%EC%84%9C%EC%8A%A4" title="렉서스">렉서스</a>와 함께 출시된 <a class="mw-redirect" href="/wiki/%EB%8B%9B%EC%82%B0" title="닛산">닛산</a>의 고급차 브랜드이다. <a href="/wiki/%EB%A0%89%EC%84%9C%EC%8A%A4" title="렉서스">렉서스</a>와 달리 우아함보다는 스포티함을 추구하였지만, 일본적임을 보여주는 광고 때문에 판매 경쟁에서 밀려났다. 그러나 최근 들어 다시 부활하고 있으며, <a href="/wiki/%EB%8C%80%ED%95%9C%EB%AF%BC%EA%B5%AD" title="대한민국">대한민국</a>에는 <a href="/wiki/2005%EB%85%84" title="2005년">2005년</a>부터 공식 판매 중이다. <a href="/wiki/%EB%8C%80%ED%95%9C%EB%AF%BC%EA%B5%AD" title="대한민국">대한민국</a>에서는 Q45와 M35hL (현 Q70L), M45와 M37, M35, M35h, M30d (현 Q70), G37과 G35 (세단 / 쿠페, 현 세단은 Q50으로 현 쿠페는 Q60으로 판매 중), G25 (현 Q40), EX35와 EX37 (현 QX50), JX35와 JX35h (현 QX60), FX30d와 FX35, FX37, FX45, FX50 (현 QX70), QX56 (현 QX80) 모델을 판매하고 있다.</p>
