<p><b>닛산 푸가</b>(Nissan Fuga)는 <a href="/wiki/%EB%8B%9B%EC%82%B0_%EC%9E%90%EB%8F%99%EC%B0%A8" title="닛산 자동차">닛산</a>의 <a href="/wiki/%ED%9B%84%EB%A5%9C%EA%B5%AC%EB%8F%99" title="후륜구동">후륜구동</a> 승용차로, <a class="new" href="/w/index.php?title=%EB%8B%9B%EC%82%B0_%EA%B8%80%EB%A1%9C%EB%A6%AC%EC%95%84&amp;action=edit&amp;redlink=1" title="닛산 글로리아 (없는 문서)">글로리아</a>와 <a class="new" href="/w/index.php?title=%EB%8B%9B%EC%82%B0_%EC%84%B8%EB%93%9C%EB%A6%AD&amp;action=edit&amp;redlink=1" title="닛산 세드릭 (없는 문서)">세드릭</a>의 후속 차종이다. <a href="/wiki/%EC%9D%BC%EB%B3%B8" title="일본">일본</a>을 제외한 거의 대부분의 국가에서는 <a href="/wiki/%EC%9D%B8%ED%94%BC%EB%8B%88%ED%8B%B0_M" title="인피니티 M">인피니티 M</a>으로 판매되었고, <a href="/wiki/%EC%9D%B8%ED%94%BC%EB%8B%88%ED%8B%B0" title="인피니티">인피니티</a>에서 차명 체계를 바꾸어 <a href="/wiki/2014%EB%85%84" title="2014년">2014년</a>부터는 <a href="/wiki/%EC%9D%B8%ED%94%BC%EB%8B%88%ED%8B%B0_M#4.EC.84.B8.EB.8C.80.28Y51.29" title="인피니티 M">Q70</a>로 판매된다.</p>

<p><a href="/wiki/2004%EB%85%84_11%EC%9B%94" title="2004년 11월">2004년 11월</a>에 출시되었고, 출시되기 전인 <a href="/wiki/2003%EB%85%84" title="2003년">2003년</a>에 개최된 <a href="/wiki/%EB%8F%84%EC%BF%84_%EB%AA%A8%ED%84%B0%EC%87%BC" title="도쿄 모터쇼">도쿄 모터쇼</a>에서 발표된 컨셉트 카인 푸가의 양산형이다. V6 2.5ℓ VQ25DE 엔진과 280마력 V6 3.5ℓ VQ35DE 엔진이 탑재되었고, <a href="/wiki/2005%EB%85%84_8%EC%9B%94" title="2005년 8월">2005년 8월</a>부터는 338마력 V8 4.5ℓ VK45DE 엔진이 추가되었다. 모든 엔진에 5단 <a href="/wiki/%EC%9E%90%EB%8F%99%EB%B3%80%EC%86%8D%EA%B8%B0" title="자동변속기">자동변속기</a>를 조합했다. <a href="/wiki/2007%EB%85%84_12%EC%9B%94" title="2007년 12월">2007년 12월</a>에는 페이스 리프트를 거쳐 V6 2.5ℓ VQ25DE 엔진과 V6 3.5ℓ VQ35DE 엔진을 각각 V6 2.5ℓ VQ25HR 엔진과 302마력 V6 3.5ℓ VQ35HR 엔진으로 대체했다. <a href="/wiki/%EC%9D%BC%EB%B3%B8" title="일본">일본</a>을 제외하면 유일하게 <a href="/wiki/%EC%A4%91%ED%99%94%EC%9D%B8%EB%AF%BC%EA%B3%B5%ED%99%94%EA%B5%AD" title="중화인민공화국">중국</a>에서도 <a href="/wiki/2005%EB%85%84_6%EC%9B%94" title="2005년 6월">2005년 6월</a>부터 <a href="/wiki/%EB%8B%9B%EC%82%B0_%EC%9E%90%EB%8F%99%EC%B0%A8" title="닛산 자동차">닛산</a>의 브랜드로 판매되었으나, <a href="/wiki/2007%EB%85%84_1%EC%9B%94" title="2007년 1월">2007년 1월</a>에 <a href="/wiki/%EC%A4%91%ED%99%94%EC%9D%B8%EB%AF%BC%EA%B3%B5%ED%99%94%EA%B5%AD" title="중화인민공화국">중국</a>에서도 <a href="/wiki/%EC%9D%B8%ED%94%BC%EB%8B%88%ED%8B%B0" title="인피니티">인피니티</a> 브랜드가 런칭하여 페이스 리프트를 거친 후인 <a href="/wiki/2008%EB%85%84_1%EC%9B%94" title="2008년 1월">2008년 1월</a>부터는 <a href="/wiki/%EC%9D%B8%ED%94%BC%EB%8B%88%ED%8B%B0" title="인피니티">인피니티</a> 브랜드로 판매되었다. <a href="/wiki/%EB%8C%80%ED%95%9C%EB%AF%BC%EA%B5%AD" title="대한민국">대한민국</a>에서는 <a href="/wiki/2005%EB%85%84" title="2005년">2005년</a>부터 <a href="/wiki/%EC%9D%B8%ED%94%BC%EB%8B%88%ED%8B%B0_M#3.EC.84.B8.EB.8C.80.28Y50.29" title="인피니티 M">인피니티 M</a>으로 공식 수입되기 시작하였다.</p>

<p><a href="/wiki/2009%EB%85%84" title="2009년">2009년</a>에 개최된 <a href="/wiki/%EB%8F%84%EC%BF%84_%EB%AA%A8%ED%84%B0%EC%87%BC" title="도쿄 모터쇼">도쿄 모터쇼</a>에 2세대 신형 푸가의 양산형이 발표되었고, 같은 해 <a href="/wiki/11%EC%9B%94" title="11월">11월</a>부터 <a href="/wiki/%EC%9D%BC%EB%B3%B8" title="일본">일본</a>에서 판매가 시작되었다. 기존의 V6 2.5ℓ VQ25HR 엔진 외에 V6 3.7ℓ VQ37VHR 엔진이 새로 탑재되었다. 1세대처럼 <a href="/wiki/%ED%9B%84%EB%A5%9C%EA%B5%AC%EB%8F%99" title="후륜구동">후륜구동</a>이 기본이며, <a href="/wiki/4%EB%A5%9C%EA%B5%AC%EB%8F%99" title="4륜구동">4륜구동(상시)</a>을 선택할 수 있다. <a href="/wiki/%EB%8B%9B%EC%82%B0_%EC%9E%90%EB%8F%99%EC%B0%A8" title="닛산 자동차">닛산 자동차</a>의 계열사인 <a class="new" href="/w/index.php?title=%EC%9E%90%ED%8A%B8%EC%BD%94&amp;action=edit&amp;redlink=1" title="자트코 (없는 문서)">자트코</a>에서 새로 개발된 7단 <a href="/wiki/%EC%9E%90%EB%8F%99%EB%B3%80%EC%86%8D%EA%B8%B0" title="자동변속기">자동변속기</a>가 조합된다. <a href="/wiki/2010%EB%85%84_11%EC%9B%94" title="2010년 11월">2010년 11월</a>에는 V6 3.5ℓ VQ35HR 엔진이 탑재된 <a href="/wiki/%ED%95%98%EC%9D%B4%EB%B8%8C%EB%A6%AC%EB%93%9C_(%EB%8F%99%EB%A0%A5)" title="하이브리드 (동력)">하이브리드</a> 사양이 추가되었다. <a href="/wiki/2012%EB%85%84_4%EC%9B%94" title="2012년 4월">2012년 4월</a>에는 롱 휠 베이스 버전이 <a href="/wiki/%EB%8B%9B%EC%82%B0_%EC%8B%9C%EB%A7%88#5.EC.84.B8.EB.8C.80.28Y51.29" title="닛산 시마">시마(Y51)</a>로 부활하였다. <a href="/wiki/%EB%AF%B8%EC%93%B0%EB%B9%84%EC%8B%9C%EC%9E%90%EB%8F%99%EC%B0%A8%EA%B3%B5%EC%97%85" title="미쓰비시자동차공업">미쓰비시자동차공업</a>이 고급 승용차 라인업을 강화하는 차원에서 <a href="/wiki/%EB%8B%9B%EC%82%B0_%EC%9E%90%EB%8F%99%EC%B0%A8" title="닛산 자동차">닛산 자동차</a>와 제휴함에 따라 뱃지 엔지니어링을 통하여 <a href="/wiki/%ED%98%84%EB%8C%80_%EC%97%90%EC%BF%A0%EC%8A%A4" title="현대 에쿠스">현대 에쿠스</a>와 공동 개발되어 널리 알려진 <a href="/wiki/%EB%AF%B8%EC%93%B0%EB%B9%84%EC%8B%9C_%ED%94%84%EB%9D%BC%EC%9A%B0%EB%94%94%EC%95%84" title="미쓰비시 프라우디아">미쓰비시 프라우디아</a>의 2세대 차종으로도 판매되고 있다. <a href="/wiki/2015%EB%85%84_2%EC%9B%94" title="2015년 2월">2015년 2월</a>에 페이스 리프트를 거침과 동시에 <a href="/wiki/%EC%9D%BC%EB%B3%B8" title="일본">일본</a>에서 판매되는 푸가도 <a href="/wiki/%EB%8B%9B%EC%82%B0_%EC%8A%A4%EC%B9%B4%EC%9D%B4%EB%9D%BC%EC%9D%B8#13.EC.84.B8.EB.8C.80.28V37.29" title="닛산 스카이라인">스카이라인(V37)</a>처럼 <a href="/wiki/%EC%9D%B8%ED%94%BC%EB%8B%88%ED%8B%B0" title="인피니티">인피니티</a> 엠블럼이 적용되었다.</p>
