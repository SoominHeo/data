<p><b>닛코 시</b>(일광시, <span style="font-size: smaller;"><a href="/wiki/%EC%9D%BC%EB%B3%B8%EC%96%B4" title="일본어">일본어</a>:</span> <span lang="ja" xml:lang="ja">日光市</span>)는 <a href="/wiki/%EC%9D%BC%EB%B3%B8" title="일본">일본</a> <a href="/wiki/%ED%98%BC%EC%8A%88" title="혼슈">혼슈</a> <a href="/wiki/%EB%8F%84%EC%B9%98%EA%B8%B0_%ED%98%84" title="도치기 현">도치기 현</a>에 있는 <a href="/wiki/%EC%8B%9C_(%ED%96%89%EC%A0%95_%EA%B5%AC%EC%97%AD)" title="시 (행정 구역)">시</a>이다. <a href="/wiki/%EB%8F%84%EC%BF%84" title="도쿄">도쿄</a>에서 북쪽으로 약 140km, 도치기 현도 <a href="/wiki/%EC%9A%B0%EC%93%B0%EB%85%B8%EB%AF%B8%EC%95%BC_%EC%8B%9C" title="우쓰노미야 시">우쓰노미야 시</a>에서 서쪽으로 약 35km 떨어져 있고 <a href="/wiki/%EB%8F%84%EC%BF%A0%EA%B0%80%EC%99%80_%EC%9D%B4%EC%97%90%EC%95%BC%EC%8A%A4" title="도쿠가와 이에야스">도쿠가와 이에야스</a>의 능이 있는 <a href="/wiki/%EB%8B%9B%EC%BD%94_%EB%8F%99%EC%A1%B0%EA%B6%81" title="닛코 동조궁">닛코 동조궁</a>과 <a href="/wiki/767%EB%85%84" title="767년">767년</a>에 세워진 <a class="mw-redirect" href="/wiki/%ED%9B%84%ED%83%80%EB%9D%BC%EC%82%B0_%EC%8B%A0%EC%82%AC" title="후타라산 신사">후타라산 신사</a>가 있는 일본과 해외 관광객들에게 인기있는 목적지이다. 이 지역에는 또한 많은 유명한 온천들이 있다.</p>

<h2><span class="mw-headline" id=".EA.B0.9C.EC.9A.94">개요</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%8B%9B%EC%BD%94_%EC%8B%9C&amp;action=edit&amp;section=1" title="부분 편집: 개요">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>도시 중앙에 닛코 국립공원이 있다. 예로부터 일본에서는 "닛코를 보기 전에 훌륭하다고 하지 말라"는 말이 있다. 이미 <a href="/wiki/8%EC%84%B8%EA%B8%B0" title="8세기">8세기</a>부터 <a href="/wiki/%EB%B6%88%EA%B5%90" title="불교">불교</a>사원이 세워졌다. <a href="/wiki/1616%EB%85%84" title="1616년">1616년</a> <a href="/wiki/%EB%8F%84%EC%BF%A0%EA%B0%80%EC%99%80_%EC%9D%B4%EC%97%90%EC%95%BC%EC%8A%A4" title="도쿠가와 이에야스">도쿠가와 이에야스</a>가 세상을 떠나자 <a href="/wiki/%EB%8F%99%EC%A1%B0%EA%B6%81" title="동조궁">동조궁</a>을 건립하고 <a href="/wiki/1617%EB%85%84" title="1617년">1617년</a> 그의 유골을 이곳에 안장했다. 그의 손자 <a href="/wiki/%EB%8F%84%EC%BF%A0%EA%B0%80%EC%99%80_%EC%9D%B4%EC%97%90%EB%AF%B8%EC%93%B0" title="도쿠가와 이에미쓰">도쿠가와 이에미쓰</a>가 안장된 <a class="new" href="/w/index.php?title=%EB%8C%80%EC%9C%A0%EC%9B%90&amp;action=edit&amp;redlink=1" title="대유원 (없는 문서)">대유원</a>(大猶院)도 유명하다. 동조궁, 대유원 등의 영향으로 오늘날에도 동조대권현(東照大権現:이에야스)에 대한 존경 의식이 강하다.</p>

<h2><span class="mw-headline" id=".EC.A7.80.EB.A6.AC">지리</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%8B%9B%EC%BD%94_%EC%8B%9C&amp;action=edit&amp;section=2" title="부분 편집: 지리">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>통합 후의 닛코 시는 도치기 현의 북서쪽의 광대한 농촌 지역을 포함해 면적도 1,449.87km²로 늘어났다. 이것은 일본에서 <a href="/wiki/%EB%8B%A4%EC%B9%B4%EC%95%BC%EB%A7%88_%EC%8B%9C" title="다카야마 시">다카야마 시</a>, <a href="/wiki/%ED%95%98%EB%A7%88%EB%A7%88%EC%93%B0_%EC%8B%9C" title="하마마쓰 시">하마마쓰 시</a>에 이어 세 번째로 넓은 것이다.</p>
<p><a href="/wiki/%EC%A3%BC%EC%A0%A0%EC%A7%80_%ED%98%B8" title="주젠지 호">주젠지 호</a>와 <a href="/wiki/%EA%B2%8C%EA%B3%A4_%ED%8F%AD%ED%8F%AC" title="게곤 폭포">게곤 폭포</a>, <a class="mw-redirect" href="/wiki/%EB%8B%9B%EC%BD%94_%EC%8B%9D%EB%AC%BC%EC%9B%90" title="닛코 식물원">닛코 식물원</a>이 있으며, 도시의 많은 산과 폭포는 수력 발전의 중요한 자원이 된다. 이 지역은 또한 구리, 알루미늄, 콘크리트 등을 채굴한다. 닛코의 날씨는 <a href="/wiki/%EB%8F%84%EC%BF%84" title="도쿄">도쿄</a>와 가까이 있기는 하지만 오히려 <a href="/wiki/%ED%99%8B%EC%B9%B4%EC%9D%B4%EB%8F%84" title="홋카이도">홋카이도</a>와 비슷하다. 이러한 이유는 닛코 시의 고도 때문이다. 닛코의 연평균 기온은 7 °C 이고 가장 더운 달에도 22 °C까지 밖에 안 오르고 가장 추운 달에는 -8 °C까지 떨어진다.</p>

<h2><span class="mw-headline" id=".EC.97.AD.EC.82.AC">역사</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%8B%9B%EC%BD%94_%EC%8B%9C&amp;action=edit&amp;section=4" title="부분 편집: 역사">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>쇼도 쇼닌(勝道上人)은 <a href="/wiki/766%EB%85%84" title="766년">766년</a>에 <a href="/wiki/%EB%A6%B0%EB%85%B8%EC%A7%80" title="린노지">린노지</a>를 세웠고 이후 <a href="/wiki/784%EB%85%84" title="784년">784년</a>에는 주젠지를 세웠다. 닛코 마을은 이러한 절들을 주변으로 발전하였다. <a href="/wiki/1617%EB%85%84" title="1617년">1617년</a>에 <a href="/wiki/%EB%8B%9B%EC%BD%94_%EB%8F%99%EC%A1%B0%EA%B6%81" title="닛코 동조궁">닛코 동조궁</a>이 완성되었고 <a href="/wiki/%EC%97%90%EB%8F%84_%EC%8B%9C%EB%8C%80" title="에도 시대">에도 시대</a>에 이 지역에 방문객을 오게하는 주요 시설이 되었다. 많은 새로운 길들이 이 시기에 세워져 주변 지역에서 닛코로 접근하는 것을 편리하게 하였다. 닛코 동조궁, 후타라산 신사, 린노지는 현재 <a class="mw-redirect" href="/wiki/%EC%9C%A0%EB%84%A4%EC%8A%A4%EC%BD%94_%EC%84%B8%EA%B3%84%EC%9C%A0%EC%82%B0" title="유네스코 세계유산">유네스코 세계유산</a>으로 지정되어있다.</p>
<p><a href="/wiki/%EB%A9%94%EC%9D%B4%EC%A7%80_%EC%8B%9C%EB%8C%80" title="메이지 시대">메이지 시대</a>에 닛코는 산 속의 휴양지로 개발되었고 특히 일본을 방문하는 외국인들에게 인기있는 장소가 되었다. <a href="/wiki/1890%EB%85%84" title="1890년">1890년</a>에 <a class="new" href="/w/index.php?title=%EC%9D%BC%EB%B3%B8_%EC%B2%A0%EB%8F%84&amp;action=edit&amp;redlink=1" title="일본 철도 (없는 문서)">일본 철도</a>(현재의 <a href="/wiki/%EB%8B%9B%EC%BD%94_%EC%84%A0" title="닛코 선">닛코 선</a>)이 개업하였고 <a href="/wiki/1929%EB%85%84" title="1929년">1929년</a>에는 <a class="mw-redirect" href="/wiki/%EB%8F%84%EB%B6%80_%EC%B2%A0%EB%8F%84_%EB%8B%9B%EC%BD%94_%EC%84%A0" title="도부 철도 닛코 선">도부 철도 닛코 선</a>이 개업하였다.</p>
<p><a href="/wiki/1889%EB%85%84" title="1889년">1889년</a> <a href="/wiki/4%EC%9B%94_1%EC%9D%BC" title="4월 1일">4월 1일</a>에 <a href="/wiki/%EA%B0%80%EB%AF%B8%EC%93%B0%EA%B0%80_%EA%B5%B0" title="가미쓰가 군">가미쓰가 군</a> 닛코 정이 성립하였으며, <a href="/wiki/1954%EB%85%84" title="1954년">1954년</a>에 이웃한 오코로가와 촌을 합병해 시로 승격되었다. <a href="/wiki/2006%EB%85%84" title="2006년">2006년</a> <a href="/wiki/3%EC%9B%94_20%EC%9D%BC" title="3월 20일">3월 20일</a>, 주변에 있는 <a href="/wiki/%EC%9D%B4%EB%A7%88%EC%9D%B4%EC%B9%98_%EC%8B%9C" title="이마이치 시">이마이치 시</a>(<span lang="ja" xml:lang="ja">今市市</span>), 가미쓰가 군 아시오 정(<span lang="ja" xml:lang="ja">足尾町</span>), 시오야 군 후지하라 정(<span lang="ja" xml:lang="ja">藤原町</span>), 구리야마 촌(<span lang="ja" xml:lang="ja">栗山村</span>)과 통합하여 새로운 닛코 시가 발족하였다. 시청은 지역의 중심지이자 인구가 가장 많은 옛 <a href="/wiki/%EC%9D%B4%EB%A7%88%EC%9D%B4%EC%B9%98_%EC%8B%9C" title="이마이치 시">이마이치 시</a>에 세워졌고 시 이름은 지명도가 높은 "닛코"가 채택되었다. <a class="new" href="/w/index.php?title=%EB%8B%9B%EC%BD%94_%ED%98%B8%ED%85%94&amp;action=edit&amp;redlink=1" title="닛코 호텔 (없는 문서)">닛코 호텔</a>도 여기서 유래하였다.</p>
