<p><b>다그마라 워즈니악</b> (Dagmara Woźniak, <a href="/wiki/1988%EB%85%84" title="1988년">1988년</a> <a href="/wiki/7%EC%9B%94_1%EC%9D%BC" title="7월 1일">7월 1일</a>, <a href="/wiki/%EB%B8%8C%EB%A1%9C%EC%B8%A0%EC%99%80%ED%94%84" title="브로츠와프">브로츠와프</a> ~) 은 <a href="/wiki/%ED%8F%B4%EB%9E%80%EB%93%9C" title="폴란드">폴란드</a> 태생의 <a href="/wiki/%EB%AF%B8%EA%B5%AD" title="미국">미국</a>인 <a href="/wiki/%EC%82%AC%EB%B8%8C%EB%A5%B4_(%ED%8E%9C%EC%8B%B1)" title="사브르 (펜싱)">사브르</a> <a href="/wiki/%ED%8E%9C%EC%8B%B1" title="펜싱">펜싱</a> 선수이다. 그녀는 <a href="/wiki/2008%EB%85%84_%ED%95%98%EA%B3%84_%EC%98%AC%EB%A6%BC%ED%94%BD" title="2008년 하계 올림픽">2008년 하계 올림픽</a>의 <a href="/wiki/2008%EB%85%84_%ED%95%98%EA%B3%84_%EC%98%AC%EB%A6%BC%ED%94%BD_%ED%8E%9C%EC%8B%B1_%EC%97%AC%EC%9E%90_%EC%82%AC%EB%B8%8C%EB%A5%B4_%EB%8B%A8%EC%B2%B4%EC%A0%84" title="2008년 하계 올림픽 펜싱 여자 사브르 단체전">여자 사브르 단체전</a>에서 교체 요원으로 참가한 경험이 있다. 그러나, 이 대회에서 그녀는 출전 기회를 잡지 못하였다. 이후, 그녀는 <a href="/wiki/2012%EB%85%84_%ED%95%98%EA%B3%84_%EC%98%AC%EB%A6%BC%ED%94%BD" title="2012년 하계 올림픽">2012년 하계 올림픽</a>에서 <a class="new" href="/w/index.php?title=2012%EB%85%84_%ED%95%98%EA%B3%84_%EC%98%AC%EB%A6%BC%ED%94%BD_%ED%8E%9C%EC%8B%B1_%EC%97%AC%EC%9E%90_%EC%82%AC%EB%B8%8C%EB%A5%B4_%EB%8B%A8%EC%B2%B4%EC%A0%84&amp;action=edit&amp;redlink=1" title="2012년 하계 올림픽 펜싱 여자 사브르 단체전 (없는 문서)">여자 사브르 단체전</a>에 이 부문에서 2연패를 거둔 <a href="/wiki/%EB%A7%A4%EB%A6%AC%EC%96%BC_%EC%9E%AC%EA%B1%B0%EB%8B%88%EC%8A%A4" title="매리얼 재거니스">매리얼 재거니스</a>와 함께 출전하였다.<sup class="reference" id="cite_ref-fencingteam_1-0"><a href="#cite_note-fencingteam-1">[1]</a></sup><sup class="reference" id="cite_ref-workethic_2-0"><a href="#cite_note-workethic-2">[2]</a></sup></p>
<p>워즈니악은 그녀가 1세였을 때 그녀의 부모님과 함께 미국으로 이민갔다. 워즈니악이 처음 검을 잡았을 때 나이는 9세였다.<sup class="reference" id="cite_ref-3"><a href="#cite_note-3">[3]</a></sup> 워즈니악은 <a href="/wiki/%EB%89%B4%EC%A0%80%EC%A7%80_%EC%A3%BC" title="뉴저지 주">뉴저지 주</a> <a class="new" href="/w/index.php?title=%EC%9A%B0%EB%93%9C%EB%B8%8C%EB%A6%BF%EC%A7%80_(%EB%89%B4%EC%A0%80%EC%A7%80_%EC%A3%BC)&amp;action=edit&amp;redlink=1" title="우드브릿지 (뉴저지 주) (없는 문서)">우드브릿지</a>의 <a class="new" href="/w/index.php?title=%EC%BD%9C%EB%A1%9C%EB%8B%88%EC%95%84_%EA%B3%A0%EB%93%B1%ED%95%99%EA%B5%90&amp;action=edit&amp;redlink=1" title="콜로니아 고등학교 (없는 문서)">콜로니아 고등학교</a>를 2006년에 졸업한 뒤, <a href="/wiki/%EC%84%B8%EC%9D%B8%ED%8A%B8_%EC%A1%B4%EC%8A%A4_%EB%8C%80%ED%95%99%EA%B5%90" title="세인트 존스 대학교">세인트 존스 대학교</a>에 입학하였다.<sup class="reference" id="cite_ref-4"><a href="#cite_note-4">[4]</a></sup></p>
<p>워즈니악이 거둔 가장 큰 성과들 중 하나는 <a class="new" href="/w/index.php?title=2011%EB%85%84_FIE_%ED%8E%9C%EC%8B%B1_%EC%84%B8%EA%B3%84%EC%84%A0%EC%88%98%EA%B6%8C&amp;action=edit&amp;redlink=1" title="2011년 FIE 펜싱 세계선수권 (없는 문서)">2011년 FIE 펜싱 세계선수권</a> 대회의 여자 사브르 단체전에서 동메달을 획득한 것이다. 2008년 백업 멤버였던 그는 <a href="/wiki/2012%EB%85%84_%ED%95%98%EA%B3%84_%EC%98%AC%EB%A6%BC%ED%94%BD" title="2012년 하계 올림픽">2012년 하계 올림픽</a> 때에 들어서 국가대표팀 주전이 되었다.<sup class="reference" id="cite_ref-workethic_2-1"><a href="#cite_note-workethic-2">[2]</a></sup></p>
<p><a href="/wiki/2012%EB%85%84_%ED%95%98%EA%B3%84_%EC%98%AC%EB%A6%BC%ED%94%BD" title="2012년 하계 올림픽">2012년 하계 올림픽</a>의 <a class="mw-redirect" href="/wiki/2012%EB%85%84_%ED%95%98%EA%B3%84_%EC%98%AC%EB%A6%BC%ED%94%BD_%ED%8E%9C%EC%8B%B1_%EC%97%AC%EC%9E%90_%EC%82%AC%EB%B8%8C%EB%A5%B4_%EA%B0%9C%EC%9D%B8%EC%A0%84" title="2012년 하계 올림픽 펜싱 여자 사브르 개인전">여자 사브르 개인전</a>에서 그녀는 16강전에서 튀니지 출신의 난적 <a href="/wiki/%EC%95%84%EC%9E%90_%EB%B2%A0%EC%8A%A4%EB%B2%A0%EC%8A%A4" title="아자 베스베스">아자 베스베스</a>를 15-13으로 이겼으나, 러시아의 <a class="mw-redirect" href="/wiki/%EC%86%8C%ED%94%BC%EC%95%84_%EB%B2%A8%EB%A6%AC%EC%B9%B4%EC%95%BC" title="소피아 벨리카야">소피아 벨리카야</a>와의 8강전에서 13-15로 석패하며 탈락하였다.<sup class="reference" id="cite_ref-5"><a href="#cite_note-5">[5]</a></sup> 이후 벨리카야는 이 토너먼트의 은메달을 획득하였다.</p>
