<p><b>다니엘 곤살레스 귀사</b> (<span style="font-size: smaller;"><a href="/wiki/%EC%8A%A4%ED%8E%98%EC%9D%B8%EC%96%B4" title="스페인어">스페인어</a>:</span> <span lang="es" xml:lang="es">Daniel González Güiza</span>, <a href="/wiki/1980%EB%85%84" title="1980년">1980년</a> <a href="/wiki/8%EC%9B%94_17%EC%9D%BC" title="8월 17일">8월 17일</a>, <a href="/wiki/%EC%B9%B4%EB%94%94%EC%8A%A4_%EC%A3%BC" title="카디스 주">카디스 주</a> <a href="/wiki/%ED%97%A4%EB%A0%88%EC%8A%A4%EB%8D%B0%EB%9D%BC%ED%94%84%EB%A1%A0%ED%85%8C%EB%9D%BC" title="헤레스데라프론테라">헤레스데라프론테라</a> ~ )는 <a href="/wiki/%EC%8A%A4%ED%8E%98%EC%9D%B8" title="스페인">스페인</a>의 축구 선수로, <a class="mw-redirect" href="/wiki/%EC%A7%91%EC%8B%9C" title="집시">집시</a> 혈통이다. 현재 <a href="/wiki/%EC%8A%A4%ED%8E%98%EC%9D%B8" title="스페인">스페인</a>의 <a href="/wiki/%EC%B9%B4%EB%94%94%EC%8A%A4_CF" title="카디스 CF">카디스 CF</a>에서 <a href="/wiki/%EA%B3%B5%EA%B2%A9%EC%88%98#.EC.A4.91.EC.95.99_.EA.B3.B5.EA.B2.A9.EC.88.98" title="공격수">스트라이커</a>로 활약하고 있으며, <a class="mw-redirect" href="/wiki/%ED%94%84%EB%A6%AC%EB%A9%94%EB%9D%BC%EB%A6%AC%EA%B0%80" title="프리메라리가">프리메라리가</a> 2007-08 시즌에 <a href="/wiki/RCD_%EB%A7%88%EC%9A%94%EB%A5%B4%EC%B9%B4" title="RCD 마요르카">마요르카</a> 소속으로 27골이라는 가공할만한 득점력을 선보이며, 그 시즌 최다 득점자가 되기도 했다.</p>

<h2><span class="mw-headline" id=".ED.81.B4.EB.9F.BD_.EA.B2.BD.EB.A0.A5">클럽 경력</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%8B%A4%EB%8B%88%EC%97%98_%EA%B7%80%EC%82%AC&amp;action=edit&amp;section=1" title="부분 편집: 클럽 경력">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<h3><span class="mw-headline" id=".EA.B2.BD.EB.A0.A5_.EC.B4.88.EA.B8.B0">경력 초기</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%8B%A4%EB%8B%88%EC%97%98_%EA%B7%80%EC%82%AC&amp;action=edit&amp;section=2" title="부분 편집: 경력 초기">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p><a href="/wiki/%ED%97%A4%EB%A0%88%EC%8A%A4_CD" title="헤레스 CD">헤레스</a>에서 축구를 시작한 귀사는 마요르카에 영입되었으나 첫 세 시즌 동안 클럽 관계자들에게 관심을 받지 못하였다. 임대로 팀을 번갈아가며 2부리그에서 세 시즌을 더 허비한 귀사는 <a href="/wiki/%ED%97%A4%ED%83%80%ED%8E%98_CF" title="헤타페 CF">헤타페</a>에 영입되어 주전으로 뛰기 시작하며 축구 인생을 다시 열게 되었다.</p>
<h3><span class="mw-headline" id=".EB.A7.88.EC.9A.94.EB.A5.B4.EC.B9.B4_.EB.B3.B5.EA.B7.80">마요르카 복귀</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%8B%A4%EB%8B%88%EC%97%98_%EA%B7%80%EC%82%AC&amp;action=edit&amp;section=3" title="부분 편집: 마요르카 복귀">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p><a href="/wiki/2007%EB%85%84" title="2007년">2007년</a> 귀사는 <a href="/wiki/RCD_%EB%A7%88%EC%9A%94%EB%A5%B4%EC%B9%B4" title="RCD 마요르카">마요르카</a>로 복귀한 이후 37경기에서 페널티킥을 한번도 차지 않고 27득점을 기록, 2007-08 <a class="mw-redirect" href="/wiki/%EB%9D%BC_%EB%A6%AC%EA%B0%80" title="라 리가">라 리가</a> 득점왕을 차지하면서 시즌을 마쳤다. 스페인의 공격수 귀사는 이 활약으로 인해 많은 유럽 팀들에게 관심을 받았고 결국 <a href="/wiki/2008%EB%85%84" title="2008년">2008년</a> 여름 이적 시장에서 1400만 유로의 이적료로 <a class="mw-redirect" href="/wiki/%ED%8E%98%EB%84%A4%EB%A5%B4%EB%B0%94%EC%B2%B4_SK" title="페네르바체 SK">페네르바체</a>에 4년 계약으로 이적 되었다.</p>
<h3><span class="mw-headline" id=".ED.8E.98.EB.84.A4.EB.A5.B4.EB.B0.94.EC.B2.B4">페네르바체</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%8B%A4%EB%8B%88%EC%97%98_%EA%B7%80%EC%82%AC&amp;action=edit&amp;section=4" title="부분 편집: 페네르바체">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p><a class="mw-redirect" href="/wiki/%EC%9C%A0%EB%A1%9C_2008" title="유로 2008">유로 2008</a>에서 <a href="/wiki/%EC%8A%A4%ED%8E%98%EC%9D%B8_%EC%B6%95%EA%B5%AC_%EA%B5%AD%EA%B0%80%EB%8C%80%ED%91%9C%ED%8C%80" title="스페인 축구 국가대표팀">스페인</a>을 우승 시킨 명장 <a href="/wiki/%EB%A3%A8%EC%9D%B4%EC%8A%A4_%EC%95%84%EB%9D%BC%EA%B3%A0%EB%84%A4%EC%8A%A4" title="루이스 아라고네스">루이스 아라고네스</a>와 함께 <a class="mw-redirect" href="/wiki/%ED%8E%98%EB%84%A4%EB%A5%B4%EB%B0%94%EC%B2%B4_SK" title="페네르바체 SK">페네르바체</a>의 황금기를 다시 이끌어낼 것이라고 기대 하였으나 기대 이하의 활약과 성적으로 많은 비판을 받기도 하였다.</p>
<p>그러나 2008-09 시즌이 끝나갈 무렵, <a href="/wiki/%EC%8A%A4%ED%8E%98%EC%9D%B8_%EC%B6%95%EA%B5%AC_%EA%B5%AD%EA%B0%80%EB%8C%80%ED%91%9C%ED%8C%80" title="스페인 축구 국가대표팀">스페인 축구 국가대표팀</a>에 소집 되어 6경기동안 6득점을 하는 괴력을 보여주었다.</p>
<p>2009-2010 시즌에는 27경기 11골이라는 준수한 성적을 거두었으나, 중요한 찬스들을 놓치는 많은 실수들로 인해 <a class="mw-redirect" href="/wiki/%ED%8E%98%EB%84%A4%EB%A5%B4%EB%B0%94%EC%B2%B4_SK" title="페네르바체 SK">페네르바체</a> 서포터들 사이에서는 형편 없는 결정력을 가지고 있는 공격수로 낙인 찍혔다.</p>
<p>그리고 2010-2011 시즌에는 장기부상을 당해 좀처럼 경기에 나서지 못했다. 결국 페네르바체에서 설 자리를 잃은 귀사는 3년 만에 터키 무대와 결별을 고하게 됐다.</p>
<h3><span class="mw-headline" id=".ED.97.A4.ED.83.80.ED.8E.98_.EB.B3.B5.EA.B7.80">헤타페 복귀</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%8B%A4%EB%8B%88%EC%97%98_%EA%B7%80%EC%82%AC&amp;action=edit&amp;section=5" title="부분 편집: 헤타페 복귀">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p><a href="/wiki/2011%EB%85%84" title="2011년">2011년</a> 페네르바체가 귀사의 헤타페 이적을 공식 발표했다. 지난 시즌 아킬레스건 부상으로 긴 공백기를 가졌던 귀사는 친정팀 헤타페에서 부활에 도전장을 내밀 예정이다.</p>
<p>페네르바체는 구단 홈페이지를 통해 "페네르바체에서 3시즌을 보낸 귀사가 스페인의 헤타페로 이적하게 됐다. 그의 미래에 행운이 가득하길 빈다" 라며 이 같은 소식을 전했다. 헤타페 측이 지급하게 될 이적료는 공개되지 않았다.</p>
<p>그러나 32경기에 출전하였음에도 3골에 그치면서 그는 한 시즌을 <a href="/wiki/%EB%A7%90%EB%A0%88%EC%9D%B4%EC%8B%9C%EC%95%84" title="말레이시아">말레이시아</a>의 <a href="/wiki/%EC%A1%B0%ED%98%B8%EB%A5%B4_%EB%8B%A4%EB%A3%B0_%ED%83%81%EC%A7%90_FC" title="조호르 다룰 탁짐 FC">조호르 다룰 탁짐 FC</a>으로 임대되었고 거기에서도 10경기 6골에 그치면서 임대 종료 후 <a href="/wiki/%ED%8C%8C%EB%9D%BC%EA%B3%BC%EC%9D%B4" title="파라과이">파라과이</a>의 <a href="/wiki/%EC%84%B8%EB%A1%9C_%ED%8F%AC%EB%A5%B4%ED%85%8C%EB%87%A8" title="세로 포르테뇨">세로 포르테뇨</a>로 이적하였다.</p>
<h2><span class="mw-headline" id=".EA.B5.AD.EA.B0.80.EB.8C.80.ED.91.9C_.EA.B2.BD.EB.A0.A5">국가대표 경력</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%8B%A4%EB%8B%88%EC%97%98_%EA%B7%80%EC%82%AC&amp;action=edit&amp;section=6" title="부분 편집: 국가대표 경력">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p><a href="/wiki/2007%EB%85%84" title="2007년">2007년</a>, <a class="mw-redirect" href="/wiki/%EC%9C%A0%EB%A1%9C_2008" title="유로 2008">유로 2008</a> 예선에서 처음으로 소집된 귀사는 <a href="/wiki/%EC%8A%A4%EC%9C%84%EC%8A%A4_%EC%B6%95%EA%B5%AC_%EA%B5%AD%EA%B0%80%EB%8C%80%ED%91%9C%ED%8C%80" title="스위스 축구 국가대표팀">스위스</a>와의 경기에선 뛰지 못하였으나 <a href="/wiki/%EB%B6%81%EC%95%84%EC%9D%BC%EB%9E%9C%EB%93%9C_%EC%B6%95%EA%B5%AC_%EA%B5%AD%EA%B0%80%EB%8C%80%ED%91%9C%ED%8C%80" title="북아일랜드 축구 국가대표팀">북아일랜드</a>와의 경기에서 국가대표 데뷔전을 치렀고 팀은 1-0으로 승리를 했다.</p>
<p>그 뒤 <a class="mw-redirect" href="/wiki/%EC%9C%A0%EB%A1%9C_2008" title="유로 2008">유로 2008</a> 선수 명단에 들게 된 귀사는 대회에서 2번 밖에 경기에 나서지 못했다. 그러나 그는 대회에서 2골을 득점했는데 디펜딩 챔피언, <a href="/wiki/%EA%B7%B8%EB%A6%AC%EC%8A%A4_%EC%B6%95%EA%B5%AC_%EA%B5%AD%EA%B0%80%EB%8C%80%ED%91%9C%ED%8C%80" title="그리스 축구 국가대표팀">그리스</a>와의 경기에서 선발 출장 해 자신의 첫 국가대표 골을 넣었고 준결승전, <a href="/wiki/%EB%9F%AC%EC%8B%9C%EC%95%84_%EC%B6%95%EA%B5%AC_%EA%B5%AD%EA%B0%80%EB%8C%80%ED%91%9C%ED%8C%80" title="러시아 축구 국가대표팀">러시아</a>와의 3-0으로 승리한 경기에서 <a href="/wiki/%EC%8A%A4%ED%8E%98%EC%9D%B8_%EC%B6%95%EA%B5%AC_%EA%B5%AD%EA%B0%80%EB%8C%80%ED%91%9C%ED%8C%80" title="스페인 축구 국가대표팀">스페인</a>의 2번째 골이자 자신의 국가대표 커리어의 2번째 골을 득점했다. 그의 활약과 더불어 <a href="/wiki/%EC%8A%A4%ED%8E%98%EC%9D%B8_%EC%B6%95%EA%B5%AC_%EA%B5%AD%EA%B0%80%EB%8C%80%ED%91%9C%ED%8C%80" title="스페인 축구 국가대표팀">스페인</a>은 <a class="mw-redirect" href="/wiki/%EC%9C%A0%EB%A1%9C_2008" title="유로 2008">유로 2008</a>에서 우승한다.</p>
<p><a href="/wiki/2009%EB%85%84" title="2009년">2009년</a> 펼쳐진 <a href="/wiki/FIFA_%EC%BB%A8%ED%8E%98%EB%8D%94%EB%A0%88%EC%9D%B4%EC%85%98%EC%8A%A4%EC%BB%B5" title="FIFA 컨페더레이션스컵">FIFA 컨페더레이션스컵</a>, <a href="/wiki/%EB%82%A8%EC%95%84%ED%94%84%EB%A6%AC%EC%B9%B4_%EA%B3%B5%ED%99%94%EA%B5%AD_%EC%B6%95%EA%B5%AC_%EA%B5%AD%EA%B0%80%EB%8C%80%ED%91%9C%ED%8C%80" title="남아프리카 공화국 축구 국가대표팀">남아프리카 공화국</a>과의 경기에서 1-0으로 지고 있는 팀을 귀사가 88분, 89분 연속 2골을 뽑아내며 2-1를 만들어 내었으나 다시 실점을 한 까닭에 연장전에 돌입했고 결국 팀은 3-2로 승리, <a href="/wiki/%EC%8A%A4%ED%8E%98%EC%9D%B8_%EC%B6%95%EA%B5%AC_%EA%B5%AD%EA%B0%80%EB%8C%80%ED%91%9C%ED%8C%80" title="스페인 축구 국가대표팀">스페인</a>은 3위를 하였다.</p>
