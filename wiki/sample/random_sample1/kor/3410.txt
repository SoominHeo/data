<p><b>다니엘 반 바이텐</b> (Daniel van Buyten, <a href="/wiki/1978%EB%85%84" title="1978년">1978년</a> <a href="/wiki/2%EC%9B%94_7%EC%9D%BC" title="2월 7일">2월 7일</a>, <a href="/wiki/%EC%8B%9C%EB%A9%94" title="시메">시메</a> ~) 는 <a href="/wiki/%EB%B2%A8%EA%B8%B0%EC%97%90" title="벨기에">벨기에</a>의 전 축구 선수이다. <a href="/wiki/2006%EB%85%84" title="2006년">2006년</a>에 <a href="/wiki/%ED%95%A8%EB%B6%80%EB%A5%B4%ED%81%AC_SV" title="함부르크 SV">함부르크 SV</a>에서 <a href="/wiki/FC_%EB%B0%94%EC%9D%B4%EC%97%90%EB%A5%B8_%EB%AE%8C%ED%97%A8" title="FC 바이에른 뮌헨">FC 바이에른 뮌헨</a>으로 자유 이적하여, 현재까지 소속팀의 <a class="mw-redirect" href="/wiki/%EC%84%BC%ED%84%B0%EB%B0%B1" title="센터백">센터백</a>으로 활약 한다. 그의 아버지, 프란츠 판 바위턴은 1960년대에서 1980년대까지 활약하던 유명한 레슬링 선수이며, 어머니 레나테는 독일인 이다.</p>

<h3><span class="mw-headline" id=".EB.A7.A8.EC.B2.B4.EC.8A.A4.ED.84.B0_.EC.8B.9C.ED.8B.B0_FC">맨체스터 시티 FC</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%8B%A4%EB%8B%88%EC%97%98_%ED%8C%90_%EB%B0%94%EC%9C%84%ED%84%B4&amp;action=edit&amp;section=2" title="부분 편집: 맨체스터 시티 FC">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>2003-04 시즌, 판 바위턴은 <a href="/wiki/%EC%98%AC%EB%9E%AD%ED%94%BC%ED%81%AC_%EB%93%9C_%EB%A7%88%EB%A5%B4%EC%84%B8%EC%9C%A0" title="올랭피크 드 마르세유">올랭피크 드 마르세유</a> 선수로, 단기간동안 <a href="/wiki/%EB%A7%A8%EC%B2%B4%EC%8A%A4%ED%84%B0_%EC%8B%9C%ED%8B%B0_FC" title="맨체스터 시티 FC">맨체스터 시티 FC</a>로 임대되었으나, <a href="/wiki/%ED%94%84%EB%A6%AC%EB%AF%B8%EC%96%B4%EB%A6%AC%EA%B7%B8" title="프리미어리그">프리미어리그</a> 풀타임 출장 경기는 없었다.</p>
<h3><span class="mw-headline" id="FC_.EB.B0.94.EC.9D.B4.EC.97.90.EB.A5.B8_.EB.AE.8C.ED.97.A8">FC 바이에른 뮌헨</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%8B%A4%EB%8B%88%EC%97%98_%ED%8C%90_%EB%B0%94%EC%9C%84%ED%84%B4&amp;action=edit&amp;section=3" title="부분 편집: FC 바이에른 뮌헨">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p><a href="/wiki/2006%EB%85%84" title="2006년">2006년</a> <a href="/wiki/6%EC%9B%94" title="6월">6월</a>, 바이에른 뮌헨은 타이틀 경쟁자 <a href="/wiki/%ED%95%A8%EB%B6%80%EB%A5%B4%ED%81%AC_SV" title="함부르크 SV">함부르크 SV</a>로부터 판 바위턴을 영입하였다고 발표했으며, 판 바위턴이 2009-10 시즌까지 유효한 계약을 하였다고 하였다. 판 바위턴은 뮌헨 첫 시즌에 <a href="/wiki/%EB%A3%A8%EC%8B%9C%EC%9A%B0" title="루시우">루시우</a>와 센터백 듀오를 이루며 성공적인 시즌을 보냈다. 그의 뮌헨 첫 골은 <a href="/wiki/2006%EB%85%84" title="2006년">2006년</a> <a href="/wiki/12%EC%9B%94_9%EC%9D%BC" title="12월 9일">12월 9일</a>, <a href="/wiki/FC_%EC%97%90%EB%84%A4%EB%A5%B4%EA%B8%B0_%EC%BD%A7%EB%B6%80%EC%8A%A4" title="FC 에네르기 콧부스">FC 에네르기 콧부스</a>전으로, 판 바위턴은 2-1 결승골의 주인공이었다. 판 바위턴은 그의 2번째골을 패한 <a href="/wiki/%EB%B3%B4%EB%A3%A8%EC%8B%9C%EC%95%84_%EB%8F%84%EB%A5%B4%ED%8A%B8%EB%AC%B8%ED%8A%B8" title="보루시아 도르트문트">보루시아 도르트문트</a>전에서 기록하였다. 그러나, 판 바위턴의 가장 중요한 업적은 <a href="/wiki/2007%EB%85%84" title="2007년">2007년</a> <a href="/wiki/4%EC%9B%94_3%EC%9D%BC" title="4월 3일">4월 3일</a>, <a href="/wiki/AC_%EB%B0%80%EB%9E%80" title="AC 밀란">AC 밀란</a>을 상대로 한 <a href="/wiki/UEFA_%EC%B1%94%ED%94%BC%EC%96%B8%EC%8A%A4%EB%A6%AC%EA%B7%B8" title="UEFA 챔피언스리그">UEFA 챔피언스리그</a> 8강전에서 이루어졌다. 판 바위턴은 이 원정경기에서 2골을 기록하였고, 경기는 무승부로 끝났다. 그의 단거리 2골은 그 시즌에 득점한 골수를 4개로 늘렸다.</p>
<p><a href="/wiki/2010%EB%85%84" title="2010년">2010년</a> <a href="/wiki/2%EC%9B%94_4%EC%9D%BC" title="2월 4일">2월 4일</a>, 바이에른 뮌헨은 판 바위턴이 <a href="/wiki/2012%EB%85%84" title="2012년">2012년</a>까지 유효한 연장 계약을 하였다고 발표하였다.<sup class="reference" id="cite_ref-Ext1_1-0"><a href="#cite_note-Ext1-1">[1]</a></sup><sup class="reference" id="cite_ref-Ext2_2-0"><a href="#cite_note-Ext2-2">[2]</a></sup> 그는 <a class="mw-redirect" href="/wiki/%ED%91%B8%EC%8A%A4%EB%B3%BC-%EB%B6%84%EB%8D%B0%EC%8A%A4%EB%A6%AC%EA%B0%80_2010-11" title="푸스볼-분데스리가 2010-11">2010-11 시즌</a> 후반 벤치로 밀렸으나, 시즌 종료를 앞두고 주전 자리를 되찾았다. <a href="/wiki/2011%EB%85%84" title="2011년">2011년</a> <a href="/wiki/12%EC%9B%94_17%EC%9D%BC" title="12월 17일">12월 17일</a>, <a href="/wiki/1._FC_%EC%BE%B0%EB%A5%B8" title="1. FC 쾰른">1. FC 쾰른</a>과의 전반기 마지막 리그 경기에서, 판 바위턴은 프리킥을 지나치게 강력하게 차서 공이 터지도록 만들었다.<sup class="reference" id="cite_ref-burstball_3-0"><a href="#cite_note-burstball-3">[3]</a></sup> 13-14시즌을 끝으로 바이에른과 계약이 종료되는데 본인이 재계약을 거부했다고 한다. 나이가 나이인지라 더 이상 빅클럽에서의 젊고 쟁쟁한 선수들과의 주전경쟁이 무리가 있다고 생각하고, 커리어를 마치기 적당한 조국 벨기에 클럽들을 물색중이라고한다. 수비력에서는 바이에른 팬들을 그다지 만족시키지 못하며 그리 좋은 소리를 듣진 못했고, 무려 8시즌을 바이에른의 1군으로 보냈음에도 닥주전으로 뛴 세월은 그리 길진 않았지만, 그래도 세계 최정상급의 뛰어난 세트피스 가담능력을 바이에른 팬들은 두고두고 그리워할듯 하다. 2014년 8월 13일 선수 생활에서 은퇴하였다.</p>
<h3><span class="mw-headline" id=".EC.B1.94.ED.94.BC.EC.96.B8.EC.8A.A4.EB.A6.AC.EA.B7.B8">챔피언스리그</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%8B%A4%EB%8B%88%EC%97%98_%ED%8C%90_%EB%B0%94%EC%9C%84%ED%84%B4&amp;action=edit&amp;section=4" title="부분 편집: 챔피언스리그">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p><a href="/wiki/UEFA_%EC%B1%94%ED%94%BC%EC%96%B8%EC%8A%A4%EB%A6%AC%EA%B7%B8_2006-07" title="UEFA 챔피언스리그 2006-07">UEFA 챔피언스리그 2006-07</a> <a href="/wiki/%EC%8A%A4%ED%83%80%EB%94%94%EC%98%A4_%EC%A3%BC%EC%84%B8%ED%8E%98_%EB%A9%94%EC%95%84%EC%B0%A8" title="스타디오 주세페 메아차">산 시로 경기장</a>에서 시행된 <a href="/wiki/AC_%EB%B0%80%EB%9E%80" title="AC 밀란">AC 밀란</a>과의 8강전에서, 그는 왼발로 2골을 기록하였다. 첫 골은 78분에 터진 1-1 골이었고, 2번째 골은 하프 볼리슛이었다. 이 2골 덕에 그의 챔피언스리그 통산 골수는 3골로 증가하였다. 그는 <a href="/wiki/2003%EB%85%84" title="2003년">2003년</a>에 <a class="mw-redirect" href="/wiki/%EC%98%AC%EB%9E%AD%ED%94%BC%ED%81%AC_%EB%A7%88%EB%A5%B4%EC%84%B8%EC%9C%A0" title="올랭피크 마르세유">올랭피크 마르세유</a> 소속으로 <a href="/wiki/%EB%A0%88%EC%95%8C_%EB%A7%88%EB%93%9C%EB%A6%AC%EB%93%9C_CF" title="레알 마드리드 CF">레알 마드리드 CF</a>를 상대로 골을 넣은 적이 있다.</p>
<h2><span class="mw-headline" id=".EA.B5.AD.EA.B0.80.EB.8C.80.ED.91.9C_.EA.B2.BD.EB.A0.A5">국가대표 경력</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%8B%A4%EB%8B%88%EC%97%98_%ED%8C%90_%EB%B0%94%EC%9C%84%ED%84%B4&amp;action=edit&amp;section=5" title="부분 편집: 국가대표 경력">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>판 바위턴은 <a href="/wiki/2001%EB%85%84" title="2001년">2001년</a>에 처음으로 <a href="/wiki/%EB%B2%A8%EA%B8%B0%EC%97%90_%EC%B6%95%EA%B5%AC_%EA%B5%AD%EA%B0%80%EB%8C%80%ED%91%9C%ED%8C%80" title="벨기에 축구 국가대표팀">벨기에 축구 국가대표팀</a> 일원으로 승선하였으며, <a href="/wiki/2002%EB%85%84_FIFA_%EC%9B%94%EB%93%9C%EC%BB%B5" title="2002년 FIFA 월드컵">2002년 FIFA 월드컵</a> 최종 엔트리에도 등록되었다. <a href="/wiki/2009%EB%85%84" title="2009년">2009년</a> <a href="/wiki/2%EC%9B%94_11%EC%9D%BC" title="2월 11일">2월 11일</a>, 그는 <a href="/wiki/%EC%8A%AC%EB%A1%9C%EB%B0%94%ED%82%A4%EC%95%84_%EC%B6%95%EA%B5%AC_%EA%B5%AD%EA%B0%80%EB%8C%80%ED%91%9C%ED%8C%80" title="슬로바키아 축구 국가대표팀">슬로바키아</a>전에서 2골을 기록하여, 벨기에의 2-0 승리를 견인하였다.<sup class="reference" id="cite_ref-Slovak_4-0"><a href="#cite_note-Slovak-4">[4]</a></sup></p>
<h2><span class="mw-headline" id=".EC.9D.80.ED.96.89.EC.A0.88.EB.8F.84_.EC.82.AC.EA.B1.B4">은행절도 사건</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%8B%A4%EB%8B%88%EC%97%98_%ED%8C%90_%EB%B0%94%EC%9C%84%ED%84%B4&amp;action=edit&amp;section=6" title="부분 편집: 은행절도 사건">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p><a href="/wiki/%EB%A7%A8%EC%B2%B4%EC%8A%A4%ED%84%B0_%EC%8B%9C%ED%8B%B0_FC" title="맨체스터 시티 FC">맨체스터 시티 FC</a> 시절, 그는 팀 동료 <a class="new" href="/w/index.php?title=%EC%9E%90%EB%A9%9C_%EB%B2%A8%EB%A7%88%EB%94%94&amp;action=edit&amp;redlink=1" title="자멜 벨마디 (없는 문서)">자멜 벨마디</a>와 <a class="new" href="/w/index.php?title=%EB%B9%84%EC%84%BC%ED%85%8C_%EB%A7%88%ED%8B%B0%EC%95%84%EC%8A%A4_%EB%B6%80%EC%98%A4%EC%86%8C&amp;action=edit&amp;redlink=1" title="비센테 마티아스 부오소 (없는 문서)">비센테 마티아스 부오소</a>와 더불어 은행털이 희생양이 되었다. 은행털이범 2명은 은행원이었고, 이들은 이 세 선수의 계좌로부터 £350,000의 돈을 빼돌렸다.</p>
<p><a href="/wiki/2006%EB%85%84" title="2006년">2006년</a> <a href="/wiki/1%EC%9B%94" title="1월">1월</a>,<sup class="reference" id="cite_ref-theft1_5-0"><a href="#cite_note-theft1-5">[5]</a></sup><sup class="reference" id="cite_ref-theft2_6-0"><a href="#cite_note-theft2-6">[6]</a></sup> 은행털이범 폴 셰어우드와 폴 헨리는 각각 32개월, 12개월간 구속되었다.</p>
