<p><b>다니엘 하르케 곤살레스</b>(<span style="font-size: smaller;"><a href="/wiki/%EC%8A%A4%ED%8E%98%EC%9D%B8%EC%96%B4" title="스페인어">스페인어</a>:</span> <span lang="es" xml:lang="es">Daniel Jarque i González</span>, <a href="/wiki/1983%EB%85%84" title="1983년">1983년</a> <a href="/wiki/1%EC%9B%94_1%EC%9D%BC" title="1월 1일">1월 1일</a> ~ <a href="/wiki/2009%EB%85%84" title="2009년">2009년</a> <a href="/wiki/8%EC%9B%94_8%EC%9D%BC" title="8월 8일">8월 8일</a>)는 <a href="/wiki/%EC%8A%A4%ED%8E%98%EC%9D%B8" title="스페인">스페인</a>의 전 축구 <a href="/wiki/%EC%88%98%EB%B9%84%EC%88%98" title="수비수">수비수</a>이다. <b>다니 하르케</b>(Dani Jarque)라고도 알려져 있다.</p>
<p><a class="mw-redirect" href="/wiki/%EC%B9%B4%ED%83%88%EB%A3%A8%EB%83%90" title="카탈루냐">카탈루냐</a>의 <a href="/wiki/%EB%B0%94%EB%A5%B4%EC%85%80%EB%A1%9C%EB%82%98" title="바르셀로나">바르셀로나</a>에서 태어났으며, 청소년 시절부터 바르셀로나를 연고지로 한 <a href="/wiki/RCD_%EC%97%90%EC%8A%A4%ED%8C%8C%EB%87%B0" title="RCD 에스파뇰">RCD 에스파뇰</a>에서 뛰었다. 이후 <a href="/wiki/%EB%A0%88%ED%81%AC%EB%A0%88%EC%95%84%ED%8B%B0%EB%B3%B4_%EC%9A%B0%EC%97%98%EB%B0%94" title="레크레아티보 우엘바">레크레아티보 우엘바</a>와의 2002-03 시즌 리그 경기에서 프로 무대에 데뷔했으며, 그 뒤 1군과 <a class="new" href="/w/index.php?title=RCD_%EC%97%90%EC%8A%A4%ED%8C%8C%EB%87%B0_B&amp;action=edit&amp;redlink=1" title="RCD 에스파뇰 B (없는 문서)">에스파뇰 B</a>팀을 오가며 출전 횟수를 늘리다 2004-05 시즌부터는 팀의 수비의 중심적인 역할을 수행하게 되었다. 이후 2005-06 시즌 팀의 <a href="/wiki/%EC%BD%94%ED%8C%8C_%EB%8D%B8_%EB%A0%88%EC%9D%B4" title="코파 델 레이">코파 델 레이</a> 우승에 공헌했으며, 그 시즌 <a href="/wiki/%EB%A0%88%EC%95%8C_%EB%A7%88%EB%93%9C%EB%A6%AC%EB%93%9C_CF" title="레알 마드리드 CF">레알 마드리드 CF</a>와의 경기에서 득점을 기록하기도 했다. 그 뒤 2006-07 시즌을 앞두고 소속팀과 2009년까지 재계약을 맺었으며, 그 시즌 팀의 <a class="mw-redirect" href="/wiki/UEFA%EC%BB%B5" title="UEFA컵">UEFA컵</a> 준우승에 기여하였다. 이러한 활약으로 2007-08 시즌 <a href="/wiki/%ED%86%A0%ED%8A%B8%EB%84%98_%ED%99%8B%EC%8A%A4%ED%8D%BC_FC" title="토트넘 홋스퍼 FC">토트넘 홋스퍼 FC</a>의 관심을 받기도 했으며,<sup class="reference" id="cite_ref-1"><a href="#cite_note-1">[1]</a></sup> 2009-10 시즌을 앞두고 <a class="new" href="/w/index.php?title=%EB%9D%BC%EC%9A%B8_%ED%83%80%EB%AC%B4%EB%8F%84&amp;action=edit&amp;redlink=1" title="라울 타무도 (없는 문서)">라울 타무도</a>를 대신해 팀의 주장에 선임되었다.<sup class="reference" id="cite_ref-2"><a href="#cite_note-2">[2]</a></sup></p>
<p><a href="/wiki/2009%EB%85%84" title="2009년">2009년</a> <a href="/wiki/8%EC%9B%94_8%EC%9D%BC" title="8월 8일">8월 8일</a> <a href="/wiki/%EC%9D%B4%ED%83%88%EB%A6%AC%EC%95%84" title="이탈리아">이탈리아</a> <a href="/wiki/%ED%94%BC%EB%A0%8C%EC%B2%B4" title="피렌체">피렌체</a>에 있는 팀의 전지훈련장에서 여자친구와 전화 통화를 하던 중 갑작스러운 <a class="mw-disambig" href="/wiki/%EC%8B%AC%EC%9E%A5%EB%A7%88%EB%B9%84" title="심장마비">심장마비</a>로 의식을 잃었으며, 발견되었던 당시에는 이미 사망한 뒤였다.<sup class="reference" id="cite_ref-3"><a href="#cite_note-3">[3]</a></sup></p>
<p>사망 이후, 에스파뇰의 팬들은 구단의 홈구장인 <a class="mw-redirect" href="/wiki/%EC%97%90%EC%8A%A4%ED%83%80%EB%94%94_%EC%BD%94%EB%A5%B4%EB%84%A4%EC%95%BC-%EC%97%98_%ED%94%84%EB%9D%BC%ED%8A%B8" title="에스타디 코르네야-엘 프라트">에스타디 코르네야-엘 프라트</a>에 모여 조의를 표했으며,<sup class="reference" id="cite_ref-4"><a href="#cite_note-4">[4]</a></sup> <a href="/wiki/%EB%8D%B0%EB%A5%B4%EB%B9%84_%EB%B0%94%EB%A5%B4%EC%85%80%EB%A3%A8%EB%8B%88" title="데르비 바르셀루니">데르비 바르셀루니</a> 라이벌인 <a href="/wiki/FC_%EB%B0%94%EB%A5%B4%EC%85%80%EB%A1%9C%EB%82%98" title="FC 바르셀로나">FC 바르셀로나</a> 또한 애도를 표했다.<sup class="reference" id="cite_ref-5"><a href="#cite_note-5">[5]</a></sup> 또한 그 해 <a href="/wiki/8%EC%9B%94_15%EC%9D%BC" title="8월 15일">8월 15일</a> 열린 <a class="mw-redirect" href="/wiki/%EB%A7%88%EC%BC%80%EB%8F%84%EB%8B%88%EC%95%84_%EC%B6%95%EA%B5%AC_%EA%B5%AD%EA%B0%80%EB%8C%80%ED%91%9C%ED%8C%80" title="마케도니아 축구 국가대표팀">마케도니아</a>와의 <a href="/wiki/A%EB%A7%A4%EC%B9%98" title="A매치">A매치</a>에서 승리를 장식한 <a href="/wiki/%EC%8A%A4%ED%8E%98%EC%9D%B8_%EC%B6%95%EA%B5%AC_%EA%B5%AD%EA%B0%80%EB%8C%80%ED%91%9C%ED%8C%80" title="스페인 축구 국가대표팀">스페인 축구 국가대표팀</a> 선수들이 승리의 영광을 하르케에게 바치며 그를 추도했으며,<sup class="reference" id="cite_ref-6"><a href="#cite_note-6">[6]</a></sup> <a href="/wiki/8%EC%9B%94_15%EC%9D%BC" title="8월 15일">8월 15일</a> 열린 <a href="/wiki/%EC%95%84%EC%8A%A4%EB%84%90_FC" title="아스널 FC">아스널 FC</a>와 <a href="/wiki/%EC%97%90%EB%B2%84%ED%84%B4_FC" title="에버턴 FC">에버턴 FC</a>와의 <a class="mw-redirect" href="/wiki/%EC%9E%89%EA%B8%80%EB%9E%9C%EB%93%9C_%ED%94%84%EB%A6%AC%EB%AF%B8%EC%96%B4%EB%A6%AC%EA%B7%B8" title="잉글랜드 프리미어리그">잉글랜드 프리미어리그</a> 경기에서 <a href="/wiki/2010%EB%85%84" title="2010년">2010년</a> <a href="/wiki/7%EC%9B%94_11%EC%9D%BC" title="7월 11일">7월 11일</a> 열린 <a href="/wiki/%EC%95%88%EB%93%9C%EB%A0%88%EC%8A%A4_%EC%9D%B4%EB%8B%88%EC%97%90%EC%8A%A4%ED%83%80" title="안드레스 이니에스타">안드레스 이니에스타</a>는 결승골을 득점한 후 <i>다니 하르케는 언제나 우리와 함께 있다</i>(<span style="font-size: smaller;"><a href="/wiki/%EC%8A%A4%ED%8E%98%EC%9D%B8%EC%96%B4" title="스페인어">스페인어</a>:</span> <span lang="es" xml:lang="es">Dani Jarque siempre con nosotros</span>)라는 메시지가 적힌 셔츠를 보이는 세레모니를 하여 그를 추모했으며,<sup class="reference" id="cite_ref-7"><a href="#cite_note-7">[7]</a></sup> <a href="/wiki/2010%EB%85%84" title="2010년">2010년</a> <a href="/wiki/7%EC%9B%94_1%EC%9D%BC" title="7월 1일">7월 1일</a> 열린 <a href="/wiki/UEFA_%EC%9C%A0%EB%A1%9C_2012_%EA%B2%B0%EC%8A%B9%EC%A0%84" title="UEFA 유로 2012 결승전">UEFA 유로 2012 결승전</a> 당시 선발 출전했던 파브레가스는 우승 확정 이후 하르케의 얼굴이 페인팅된 셔츠를 입으며 그를 추도하였다.</p>
