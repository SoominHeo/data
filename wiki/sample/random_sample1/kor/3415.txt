<p><b>다니옐 프라니치</b>(<span style="font-size: smaller;"><a href="/wiki/%ED%81%AC%EB%A1%9C%EC%95%84%ED%8B%B0%EC%95%84%EC%96%B4" title="크로아티아어">크로아티아어</a>:</span> <span lang="hr" xml:lang="hr">Danijel Pranjić</span>, <a class="mw-redirect" href="/wiki/IPA" title="IPA">IPA</a>: <span class="IPA" title="IPA 발음">[ˈdanijɛl ˈpraɲitɕ]</span>, <a href="/wiki/1981%EB%85%84" title="1981년">1981년</a> <a href="/wiki/12%EC%9B%94_2%EC%9D%BC" title="12월 2일">12월 2일</a>, 나시체 ~ )는 <a href="/wiki/%ED%81%AC%EB%A1%9C%EC%95%84%ED%8B%B0%EC%95%84" title="크로아티아">크로아티아</a>의 축구 선수로, 현재 <a href="/wiki/%EC%8A%AC%EB%A1%9C%EB%B2%A0%EB%8B%88%EC%95%84" title="슬로베니아">슬로베니아</a>의 <a href="/wiki/FC_%EC%BD%94%ED%8E%98%EB%A5%B4" title="FC 코페르">코페르</a> 소속이다.</p>
<p><a class="new" href="/w/index.php?title=NK_NA%C5%A0K&amp;action=edit&amp;redlink=1" title="NK NAŠK (없는 문서)">NK NAŠK</a> (1998년~2000년), <a class="new" href="/w/index.php?title=NK_%ED%8C%8C%ED%91%B8%ED%81%AC&amp;action=edit&amp;redlink=1" title="NK 파푸크 (없는 문서)">NK 파푸크</a> (NK Papuk, 2000년~2001년), <a class="new" href="/w/index.php?title=NK_%EB%B2%A8%EB%A6%AC%EC%8A%88%EC%B2%B4&amp;action=edit&amp;redlink=1" title="NK 벨리슈체 (없는 문서)">NK 벨리슈체</a> (NK Belišće, 2001년~2002년), <a href="/wiki/NK_%EC%98%A4%EC%8B%9C%EC%98%88%ED%81%AC" title="NK 오시예크">NK 오시예크</a> (NK Osijek, 2002년~2004년)에서 선수 생활을 하고, <a href="/wiki/2004%EB%85%84" title="2004년">2004년</a> <a class="mw-redirect" href="/wiki/NK_%EB%94%94%EB%82%98%EB%AA%A8_%EC%9E%90%EA%B7%B8%EB%A0%88%EB%B8%8C" title="NK 디나모 자그레브">디나모 자그레브</a>로 이적하였다. <a href="/wiki/2005%EB%85%84" title="2005년">2005년</a> <a href="/wiki/SC_%ED%97%A4%EC%9D%B4%EB%A0%8C%EB%B2%A0%EC%9D%B8" title="SC 헤이렌베인">SC 헤이렌베인</a>으로 이적했으며, 2008-09 시즌에는 24경기에서 16골을 넣는 활약을 펼쳤다. 이 해에 <a href="/wiki/KNVB%EC%BB%B5" title="KNVB컵">KNVB컵</a>에서도 활약하며, 헤이렌베인의 KNVB컵 우승을 이끌었다.</p>
<p><a href="/wiki/2009%EB%85%84" title="2009년">2009년</a> <a href="/wiki/FC_%EB%B0%94%EC%9D%B4%EC%97%90%EB%A5%B8_%EB%AE%8C%ED%97%A8" title="FC 바이에른 뮌헨">바이에른 뮌헨</a>으로 이적했으며, 같은 해 12월 <a href="/wiki/VfL_%EB%B3%B4%ED%9B%94" title="VfL 보훔">VfL 보훔</a> 전에서 분데스리가 첫 골을 넣었다. 2009-10 시즌에는 왼쪽 수비수, 왼쪽 하프, 센터 하프 등 다양한 포지션을 소화하며 <a class="mw-redirect" href="/wiki/%EB%8F%85%EC%9D%BC_%EB%B6%84%EB%8D%B0%EC%8A%A4%EB%A6%AC%EA%B0%80" title="독일 분데스리가">분데스리가</a>와 <a href="/wiki/DFB-%ED%8F%AC%EC%B9%BC" title="DFB-포칼">DFB-포칼</a>에서 바이에른 뮌헨의 우승에 공헌하였다.</p>
<p><a href="/wiki/%ED%81%AC%EB%A1%9C%EC%95%84%ED%8B%B0%EC%95%84_%EC%B6%95%EA%B5%AC_%EA%B5%AD%EA%B0%80%EB%8C%80%ED%91%9C%ED%8C%80" title="크로아티아 축구 국가대표팀">크로아티아 축구 국가대표팀</a>에는 <a href="/wiki/2004%EB%85%84" title="2004년">2004년</a> <a href="/wiki/%EC%95%84%EC%9D%BC%EB%9E%9C%EB%93%9C_%EC%B6%95%EA%B5%AC_%EA%B5%AD%EA%B0%80%EB%8C%80%ED%91%9C%ED%8C%80" title="아일랜드 축구 국가대표팀">아일랜드 대표팀</a>과의 경기에서 데뷔했으며, 이 경기에서 크로아티아는 아일랜드에게 0-1로 패하였다. <a href="/wiki/2008%EB%85%84" title="2008년">2008년</a> <a href="/wiki/UEFA_%EC%9C%A0%EB%A1%9C_2008" title="UEFA 유로 2008">UEFA 유로 2008</a>에서 감독 <a href="/wiki/%EC%8A%AC%EB%9D%BC%EB%B2%A4_%EB%B9%8C%EB%A6%AC%EC%B9%98" title="슬라벤 빌리치">슬라벤 빌리치</a> (Slaven Bilić) 에 의해 수비 역할을 했으며, <a href="/wiki/%EB%8F%85%EC%9D%BC_%EC%B6%95%EA%B5%AC_%EA%B5%AD%EA%B0%80%EB%8C%80%ED%91%9C%ED%8C%80" title="독일 축구 국가대표팀">독일</a> 전에서는 <a href="/wiki/%EB%8B%A4%EB%A6%AC%EC%9A%94_%EC%8A%A4%EB%A5%B4%EB%82%98" title="다리요 스르나">다리요 스르나</a>의 선제골을 어시스트하고, <a href="/wiki/%ED%8F%B4%EB%9E%80%EB%93%9C_%EC%B6%95%EA%B5%AC_%EA%B5%AD%EA%B0%80%EB%8C%80%ED%91%9C%ED%8C%80" title="폴란드 축구 국가대표팀">폴란드</a> 전에서는 <a href="/wiki/%EC%9D%B4%EB%B0%98_%ED%81%B4%EB%9D%BC%EC%8A%A4%EB%8B%88%EC%B9%98" title="이반 클라스니치">이반 클라스니치</a>의 골을 어시스트하였다.</p>
