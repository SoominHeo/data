<p><b>다니엘 문테 아게르</b> (<span style="font-size: smaller;"><a href="/wiki/%EB%8D%B4%EB%A7%88%ED%81%AC%EC%96%B4" title="덴마크어">덴마크어</a>:</span> <span lang="da" xml:lang="da">Daniel Munthe Agger</span>, <a href="/wiki/1984%EB%85%84" title="1984년">1984년</a> <a href="/wiki/12%EC%9B%94_12%EC%9D%BC" title="12월 12일">12월 12일</a> ~ )는 <a href="/wiki/%EB%8D%B4%EB%A7%88%ED%81%AC" title="덴마크">덴마크</a>의 은퇴한 축구 선수이다. 그는 현재 <a class="mw-redirect" href="/wiki/%EC%9E%89%EA%B8%80%EB%9E%9C%EB%93%9C_%ED%94%84%EB%A6%AC%EB%AF%B8%EC%96%B4%EB%A6%AC%EA%B7%B8" title="잉글랜드 프리미어리그">잉글랜드 프리미어리그</a>의 <a href="/wiki/%EB%A6%AC%EB%B2%84%ED%92%80_FC" title="리버풀 FC">리버풀</a>에서 9년간의 생활을 마치고 친정팀인 <a href="/wiki/%EB%8D%B4%EB%A7%88%ED%81%AC_%EC%88%98%ED%8E%98%EB%A5%B4%EB%A6%AC%EA%B0%80" title="덴마크 수페르리가">덴마크 수페르리가</a>의 <a href="/wiki/%EB%B8%8C%EB%A2%B4%EB%B7%94_IF" title="브뢴뷔 IF">브뢴뷔 IF</a>에서 뛰었다. <a href="/wiki/2004%EB%85%84_7%EC%9B%94" title="2004년 7월">2004년 7월</a> 덴마크의 클럽인 <a href="/wiki/%EB%B8%8C%EB%A2%B4%EB%B7%94_IF" title="브뢴뷔 IF">브뢴뷔 IF</a>에서 프로 무대에 데뷔했으며 <a href="/wiki/2006%EB%85%84" title="2006년">2006년</a> 빅클럽 <a href="/wiki/%EB%A6%AC%EB%B2%84%ED%92%80_FC" title="리버풀 FC">리버풀</a>로 이적했다. 그리고 <a href="/wiki/2014%EB%85%84_9%EC%9B%94" title="2014년 9월">2014년 9월</a> 자신의 친정팀인 <a href="/wiki/%EB%B8%8C%EB%A2%B4%EB%B7%94_IF" title="브뢴뷔 IF">브뢴뷔 IF</a>로 돌아왔다. 2005년 대표팀에 처음으로 모습을 드러낸 이후로 75경기에 나섰으며 12골을 기록하기도 했다.</p>
<p>그는 <a href="/wiki/2010%EB%85%84_FIFA_%EC%9B%94%EB%93%9C%EC%BB%B5" title="2010년 FIFA 월드컵">2010년 FIFA 월드컵</a>과 <a href="/wiki/UEFA_%EC%9C%A0%EB%A1%9C_2012" title="UEFA 유로 2012">UEFA 유로 2012</a>에서 <a href="/wiki/%EB%8D%B4%EB%A7%88%ED%81%AC_%EC%B6%95%EA%B5%AC_%EA%B5%AD%EA%B0%80%EB%8C%80%ED%91%9C%ED%8C%80" title="덴마크 축구 국가대표팀">덴마크 대표</a>로 참가했다.</p>

<p><a href="/wiki/2004%EB%85%84_7%EC%9B%94" title="2004년 7월">2004년 7월</a>에 덴마크의 클럽인 <a href="/wiki/%EB%B8%8C%EB%A2%B4%EB%B7%94_IF" title="브뢴뷔 IF">브뢴뷔 IF</a>에서 프로 계약을 맺고 경력을 시작한 그는 얼마 지나지 않아 자신의 능력을 보이며 팀의 2004-05 시즌 <a href="/wiki/%EB%8D%B4%EB%A7%88%ED%81%AC_%EC%88%98%ED%8E%98%EB%A5%B4%EB%A6%AC%EA%B0%80" title="덴마크 수페르리가">덴마크 수페르리가</a> 우승에 기여한다. 성공적인 시즌을 보내고 대표팀 경기에 나서며 좋은 활약을 보이던 아게르는 오랜 기간동안 빅클럽들과 링크되어 왔으며, 결국 <a href="/wiki/2006%EB%85%84_1%EC%9B%94" title="2006년 1월">2006년 1월</a>에 당시 <a href="/wiki/UEFA_%EC%B1%94%ED%94%BC%EC%96%B8%EC%8A%A4%EB%A6%AC%EA%B7%B8" title="UEFA 챔피언스리그">UEFA 챔피언스리그</a> 디펜딩 챔피언이었던 <a href="/wiki/%EB%A6%AC%EB%B2%84%ED%92%80_FC" title="리버풀 FC">리버풀</a>과 4년 반 계약을 맺고 <a class="mw-redirect" href="/wiki/%EC%95%A4%ED%95%84%EB%93%9C" title="앤필드">앤필드</a>에 입성하였다. 당시 그의 이적료였던 5백만 파운드는 덴마크 클럽에서 해외 클럽으로 진출하는 선수들 중 최고액으로 기록되었으며, 당시 <a href="/wiki/%EB%A6%AC%EB%B2%84%ED%92%80_FC" title="리버풀 FC">리버풀</a> 수비수들 가운데서도 가장 비싼 몸값이었다. 이적 초반 부상으로 인해 4경기 밖에 나서지 못한 그는 <a href="/wiki/2006%EB%85%84" title="2006년">2006년</a> <a href="/wiki/FA_%EC%BB%A4%EB%AE%A4%EB%8B%88%ED%8B%B0_%EC%8B%A4%EB%93%9C" title="FA 커뮤니티 실드">FA 커뮤니티 실드</a>에 풀타임 출장하며 본격적으로 팀 동료인 <a href="/wiki/%EC%A0%9C%EC%9D%B4%EB%AF%B8_%EC%BA%90%EB%9F%AC%EA%B1%B0" title="제이미 캐러거">제이미 캐러거</a>, <a class="mw-redirect" href="/wiki/%EC%82%AC%EB%AF%B8_%ED%9E%88%ED%94%BC%EC%95%A0" title="사미 히피애">사미 히피애</a> 등과 주전 경쟁을 시작하였다. 아게르는 <a href="/wiki/2006%EB%85%84" title="2006년">2006년</a> <a href="/wiki/8%EC%9B%94_26%EC%9D%BC" title="8월 26일">8월 26일</a> <a class="mw-redirect" href="/wiki/%EC%9B%A8%EC%8A%A4%ED%8A%B8%ED%96%84_%EC%9C%A0%EB%82%98%EC%9D%B4%ED%8B%B0%EB%93%9C" title="웨스트햄 유나이티드">웨스트햄 유나이티드</a>와의 경기에서 32m 중거리슛으로 자신의 <a href="/wiki/%EB%A6%AC%EB%B2%84%ED%92%80_FC" title="리버풀 FC">리버풀</a> 첫 득점을 기록한다. 이 골은 <a class="mw-redirect" href="/wiki/BBC" title="BBC">BBC</a>의 '맨 오브 더 매치'로 부터 이 달의 골로 선정되기도 했다.</p>
<p><a href="/wiki/2009%EB%85%84_5%EC%9B%94" title="2009년 5월">2009년 5월</a>, 아게르는 구단과 4년의 새로운 계약에 서명했으며 이로써 그는 <a href="/wiki/2014%EB%85%84" title="2014년">2014년</a>까지 <a href="/wiki/%EB%A6%AC%EB%B2%84%ED%92%80_FC" title="리버풀 FC">리버풀</a>의 일원으로 남게 되었다.</p>
<p><a href="/wiki/2014%EB%85%84_8%EC%9B%94" title="2014년 8월">2014년 8월</a>, 9년간의 <a href="/wiki/%EB%A6%AC%EB%B2%84%ED%92%80_FC" title="리버풀 FC">리버풀</a> 생활을 정리하고 여러 클럽들의 오퍼를 거절하고 <a href="/wiki/%EB%A6%AC%EB%B2%84%ED%92%80_FC" title="리버풀 FC">리버풀</a>에 대한 충성심을 내비치면서 자신의 고향팀인 <a href="/wiki/%EB%B8%8C%EB%A2%B4%EB%B7%94_IF" title="브뢴뷔 IF">브뢴뷔 IF</a>에 헐값에 2년 계약으로 선수 생활을 이어가기로 결정했다. <a href="/wiki/%EB%A6%AC%EB%B2%84%ED%92%80_FC" title="리버풀 FC">리버풀</a>에서 지내는 9년동안 232경기에 출전했고 14골을 기록했으며 2012년 리그컵 트로피를 들어올렸다. <a href="/wiki/2016%EB%85%84" title="2016년">2016년</a> <a href="/wiki/6%EC%9B%94_9%EC%9D%BC" title="6월 9일">6월 9일</a> 현역 은퇴를 선언했다.</p>
<h2><span class="mw-headline" id=".EA.B5.AD.EA.B0.80.EB.8C.80.ED.91.9C_.EA.B2.BD.EB.A0.A5">국가대표 경력</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%8B%A4%EB%8B%90_%EC%95%84%EA%B2%8C%EB%A5%B4&amp;action=edit&amp;section=2" title="부분 편집: 국가대표 경력">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>2004-05 시즌의 활약을 기반으로 <a href="/wiki/%EB%8D%B4%EB%A7%88%ED%81%AC_%EC%B6%95%EA%B5%AC_%EA%B5%AD%EA%B0%80%EB%8C%80%ED%91%9C%ED%8C%80" title="덴마크 축구 국가대표팀">덴마크 대표팀</a>에 입성한 그는 <a href="/wiki/2005%EB%85%84_6%EC%9B%94" title="2005년 6월">2005년 6월</a> <a href="/wiki/%ED%95%80%EB%9E%80%EB%93%9C_%EC%B6%95%EA%B5%AC_%EA%B5%AD%EA%B0%80%EB%8C%80%ED%91%9C%ED%8C%80" title="핀란드 축구 국가대표팀">핀란드</a>를 상대로 1-0 승리를 거둔 경기에 선발출장하며 국가대표 공식 데뷔를 하게 된다. 이어 8월에는 <a href="/wiki/%EC%9E%89%EA%B8%80%EB%9E%9C%EB%93%9C_%EC%B6%95%EA%B5%AC_%EA%B5%AD%EA%B0%80%EB%8C%80%ED%91%9C%ED%8C%80" title="잉글랜드 축구 국가대표팀">잉글랜드</a>와의 경기에서 <a href="/wiki/%EC%9B%A8%EC%9D%B8_%EB%A3%A8%EB%8B%88" title="웨인 루니">웨인 루니</a>를 꽁꽁 묶으며 팀의 4-1 대승에 기여하였다. 이후 소속팀과 대표팀을 오가며 꾸준히 활약하던 아게르는 <a href="/wiki/%EB%8D%B4%EB%A7%88%ED%81%AC_%EC%B6%95%EA%B5%AC_%EA%B5%AD%EA%B0%80%EB%8C%80%ED%91%9C%ED%8C%80" title="덴마크 축구 국가대표팀">덴마크</a>가 <a href="/wiki/2010%EB%85%84_FIFA_%EC%9B%94%EB%93%9C%EC%BB%B5_%EC%98%88%EC%84%A0" title="2010년 FIFA 월드컵 예선">2010년 FIFA 월드컵 예선</a>을 통과하며 자신의 생애 첫 월드컵 출전을 이루게 된다. 아게르는 조별 예선 세 경기에 모두 나섰으며, <a href="/wiki/%EB%84%A4%EB%8D%9C%EB%9E%80%EB%93%9C_%EC%B6%95%EA%B5%AC_%EA%B5%AD%EA%B0%80%EB%8C%80%ED%91%9C%ED%8C%80" title="네덜란드 축구 국가대표팀">네덜란드</a>와의 조별예선 첫 경기에서는 <a href="/wiki/%EC%9E%90%EC%B1%85%EA%B3%A8" title="자책골">자책골</a>을 기록하기도 했다.</p>
<p><a href="/wiki/UEFA_%EC%9C%A0%EB%A1%9C_2012" title="UEFA 유로 2012">UEFA 유로 2012</a>에 주장으로 참가한 그는 조국을 위해 헌신했고, 비록 팀은 3위로 조별리그에서 탈락했지만 <a href="/wiki/%EB%84%A4%EB%8D%9C%EB%9E%80%EB%93%9C_%EC%B6%95%EA%B5%AC_%EA%B5%AD%EA%B0%80%EB%8C%80%ED%91%9C%ED%8C%80" title="네덜란드 축구 국가대표팀">네덜란드</a>와의 경기에서 45년 만에 승리를 거두는데 일조하였으며, 이와 같은 활약을 통해 아게르는 이 경기에서 <a href="/wiki/%EB%A7%A8_%EC%98%A4%EB%B8%8C_%EB%8D%94_%EB%A7%A4%EC%B9%98" title="맨 오브 더 매치">맨 오브 더 매치</a>에 뽑혔다.</p>
<p>이후, <a href="/wiki/2014%EB%85%84_FIFA_%EC%9B%94%EB%93%9C%EC%BB%B5" title="2014년 FIFA 월드컵">2014년 FIFA 월드컵</a>에서는 <a href="/wiki/%EB%8D%B4%EB%A7%88%ED%81%AC_%EC%B6%95%EA%B5%AC_%EA%B5%AD%EA%B0%80%EB%8C%80%ED%91%9C%ED%8C%80" title="덴마크 축구 국가대표팀">덴마크 대표팀</a>이 본선에 오르지 못했고, 마지막 메이저 대회가 될 수 있었던 <a href="/wiki/UEFA_%EC%9C%A0%EB%A1%9C_2016" title="UEFA 유로 2016">UEFA 유로 2016</a>에서는 예선 7경기에 모두 풀타임 출전하여 분투하였으나, 플레이오프에서 <a href="/wiki/%EC%8A%A4%EC%9B%A8%EB%8D%B4_%EC%B6%95%EA%B5%AC_%EA%B5%AD%EA%B0%80%EB%8C%80%ED%91%9C%ED%8C%80" title="스웨덴 축구 국가대표팀">스웨덴</a>에 1무 1패로 밀리며 본선 무대를 밟지 못하면서 <a href="/wiki/%EB%8D%B4%EB%A7%88%ED%81%AC_%EC%B6%95%EA%B5%AC_%EA%B5%AD%EA%B0%80%EB%8C%80%ED%91%9C%ED%8C%80" title="덴마크 축구 국가대표팀">덴마크 대표팀</a>에서의 메이저 대회 커리어을 마감하였다.</p>
