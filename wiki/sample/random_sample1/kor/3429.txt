<h1 class="firstHeading" id="firstHeading" lang="ko">다름슈타트</h1>
<div class="mw-body-content" id="bodyContent">

<div class="mw-content-ltr" dir="ltr" id="mw-content-text" lang="ko"><script>function mfTempOpenSection(id){var block=document.getElementById("mf-section-"+id);block.className+=" open-block";block.previousSibling.className+=" open-block";}</script><div class="floatright"><a class="image" href="/wiki/%ED%8C%8C%EC%9D%BC:Darmstadt_in_Germany.png" title="다름슈타트의 위치"><img alt="다름슈타트의 위치" data-file-height="235" data-file-width="188" height="235" src="//upload.wikimedia.org/wikipedia/commons/9/93/Darmstadt_in_Germany.png" width="188"/></a></div>
<p><b>다름슈타트</b>(Darmstadt)는 <a href="/wiki/%EB%8F%85%EC%9D%BC" title="독일">독일</a> <a href="/wiki/%ED%97%A4%EC%84%BC_%EC%A3%BC" title="헤센 주">헤센 주</a>의 도시이다. <a href="/wiki/2005%EB%85%84" title="2005년">2005년</a>의 인구는 140,129명이다.</p>
<p>다름슈타트란 이름이 처음 등장한 것은 <a href="/wiki/11%EC%84%B8%EA%B8%B0" title="11세기">11세기</a> 말이었다. <a href="/wiki/%EC%8B%A0%EC%84%B1_%EB%A1%9C%EB%A7%88_%EC%A0%9C%EA%B5%AD" title="신성 로마 제국">신성 로마 제국</a> 시대인 1330년 도시권을 획득했다. 지금의 <a class="new" href="/w/index.php?title=%EB%8B%A4%EB%A6%84%EC%8A%88%ED%83%80%ED%8A%B8_%EA%B3%B5%EA%B3%BC%EB%8C%80%ED%95%99%EA%B5%90&amp;action=edit&amp;redlink=1" title="다름슈타트 공과대학교 (없는 문서)">다름슈타트 공과대학교</a>(TU Darmstadt)의 기원이 된 기술학교가 1877년 문을 열었다. 20세기 초, 다름슈타트는 독일의 <a class="mw-redirect" href="/wiki/%EC%95%84%EB%A5%B4%EB%88%84%EB%B3%B4" title="아르누보">아르누보</a> 운동인 <a class="mw-redirect" href="/wiki/%EC%9C%A0%EA%B2%90%ED%8A%B8%EC%8A%88%ED%8B%B8" title="유겐트슈틸">유겐트슈틸</a>의 중심지다.</p>
