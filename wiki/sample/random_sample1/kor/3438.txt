<p><b>다미르 카흐리만</b> (<span style="font-size: smaller;"><a href="/wiki/%EC%84%B8%EB%A5%B4%EB%B9%84%EC%95%84%EC%96%B4" title="세르비아어">세르비아어</a>:</span> <span lang="sr" xml:lang="sr">Дамир Кахриман</span>, <a href="/wiki/1984%EB%85%84" title="1984년">1984년</a> <a href="/wiki/9%EC%9B%94_11%EC%9D%BC" title="9월 11일">9월 11일</a>, <a href="/wiki/%EB%B2%A0%EC%98%A4%EA%B7%B8%EB%9D%BC%EB%93%9C" title="베오그라드">베오그라드</a>) 는 <a href="/wiki/%EC%84%B8%EB%A5%B4%EB%B9%84%EC%95%84" title="세르비아">세르비아</a>의 <a href="/wiki/%EC%B6%95%EA%B5%AC" title="축구">축구</a> 선수로, 포지션은 <a href="/wiki/%EA%B3%A8%ED%82%A4%ED%8D%BC" title="골키퍼">골키퍼</a>이다. 현재 <a href="/wiki/%EC%84%B8%EB%A5%B4%EB%B9%84%EC%95%84" title="세르비아">세르비아</a>의 <a class="new" href="/w/index.php?title=FK_%ED%95%98%EB%B9%84%ED%8A%B8_%ED%8C%8C%EB%A6%84_%EC%95%BC%EB%B3%B4%EB%A5%B4_%EC%9D%B4%EB%B0%94%EB%8B%88%EC%B0%A8&amp;action=edit&amp;redlink=1" title="FK 하비트 파름 야보르 이바니차 (없는 문서)">하비트 파름 야보르 이바니차</a> 소속이다.</p>
<p>그는 <a href="/wiki/2007%EB%85%84" title="2007년">2007년</a>, <a href="/wiki/%EB%8F%85%EC%9D%BC" title="독일">독일</a>에서 열린 UEEA U-21 유럽축구선수권대회에서 세르비아 대표팀의 일원으로 출전했고, 이 대회에서 세르비아를 준우승으로 이끌어, 2007년 UEFA U-21 유럽축구선수권대회 베스트 11에 선정되기도 했다. 또한 <a href="/wiki/2008%EB%85%84_%ED%95%98%EA%B3%84_%EC%98%AC%EB%A6%BC%ED%94%BD" title="2008년 하계 올림픽">2008년 하계 올림픽</a> 본선 진출에도 기여하였다.</p>
