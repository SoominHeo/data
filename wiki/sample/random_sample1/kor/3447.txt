<h2><span class="mw-headline" id=".ED.81.B4.EB.9F.BD_.EA.B2.BD.EB.A0.A5">클럽 경력</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%8B%A4%EB%B9%84%EB%93%9C_%EB%A7%88%ED%85%8C%EC%98%A4%EC%8A%A4&amp;action=edit&amp;section=1" title="부분 편집: 클럽 경력">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<h3><span class="mw-headline" id=".EB.A0.88.EC.95.8C_.EB.A7.88.EB.93.9C.EB.A6.AC.EB.93.9C">레알 마드리드</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%8B%A4%EB%B9%84%EB%93%9C_%EB%A7%88%ED%85%8C%EC%98%A4%EC%8A%A4&amp;action=edit&amp;section=2" title="부분 편집: 레알 마드리드">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p><a href="/wiki/%EB%A7%88%EB%93%9C%EB%A6%AC%EB%93%9C" title="마드리드">마드리드</a> 출신으로, 마테오스는 12세의 나이에 <a href="/wiki/%EB%A0%88%EC%95%8C_%EB%A7%88%EB%93%9C%EB%A6%AC%EB%93%9C_CF" title="레알 마드리드 CF">레알 마드리드</a> 유소년 아카데미에 합류했다. <a class="new" href="/w/index.php?title=%EC%84%B8%EA%B5%B0%EB%8B%A4_%EB%94%94%EB%B9%84%EC%8B%9C%EC%98%A8_B_2007-08&amp;action=edit&amp;redlink=1" title="세군다 디비시온 B 2007-08 (없는 문서)">2007-08 시즌</a>, 그는 성인 무대 첫 경기를 치렀는데, 그는 이어지는 3ㅜ 시즌 동안 <a href="/wiki/%EC%84%B8%EA%B5%B0%EB%8B%A4_%EB%94%94%EB%B9%84%EC%8B%9C%EC%98%A8_B" title="세군다 디비시온 B">3부 리그</a>의 <a href="/wiki/%EB%A0%88%EC%95%8C_%EB%A7%88%EB%93%9C%EB%A6%AC%EB%93%9C_%EC%B9%B4%EC%8A%A4%ED%8B%B0%EC%95%BC" title="레알 마드리드 카스티야">2군</a> 경기에 출전했다.</p>
<p>2010년 1월 말, 마테오스는 <a class="new" href="/w/index.php?title=%EB%A7%88%EB%88%84%EC%97%98_%ED%8E%98%EC%98%88%EA%B7%B8%EB%A6%AC%EB%8B%88_%EA%B0%90%EB%8F%85&amp;action=edit&amp;redlink=1" title="마누엘 페예그리니 감독 (없는 문서)">마누엘 페예그리니 감독</a>에 의해 <a href="/wiki/%EB%8D%B0%ED%8F%AC%EB%A5%B4%ED%8B%B0%EB%B3%B4_%EB%9D%BC%EC%BD%94%EB%A3%A8%EB%83%90" title="데포르티보 라코루냐">데포르티보</a>와의 <a href="/wiki/%EB%9D%BC%EB%A6%AC%EA%B0%80" title="라리가">라 리가</a> 경기를 앞두고 1군에 차출되었다.<sup class="reference" id="cite_ref-1"><a href="#cite_note-1">[1]</a></sup> 같은 해 4월 3일, 그는 <a href="/wiki/%EB%9D%BC%EC%8B%B1_%EC%82%B0%ED%83%84%EB%8D%B0%EB%A5%B4" title="라싱 산탄데르">라싱 산탄데르</a>와의 경기에서 최종 명단에 이름을 올렸지만,<sup class="reference" id="cite_ref-2"><a href="#cite_note-2">[2]</a></sup> 두 경기 모두 <a class="new" href="/w/index.php?title=%ED%9B%84%EB%B3%B4_(%EC%B6%95%EA%B5%AC)&amp;action=edit&amp;redlink=1" title="후보 (축구) (없는 문서)">후보</a>로 대기하는데 그쳤다.</p>
<p><a class="new" href="/w/index.php?title=%EB%9D%BC%EB%A6%AC%EA%B0%80_2010-11&amp;action=edit&amp;redlink=1" title="라리가 2010-11 (없는 문서)">2010-11 시즌</a> 전 훈련에서, <a class="mw-redirect" href="/wiki/%EC%A3%BC%EC%A0%9C_%EB%AA%A8%EB%A6%AC%EB%89%B4" title="주제 모리뉴">주제 모리뉴</a> 레알 마드리드 1군 감독은:</p>
<blockquote class="templatequote">
<p>마테오스는 제가 마음에 드는 선수이지만 우리 선수단에 합류하지 못한 것이 안타깝습니다. 그는 수비가 좋고 후방에서 경기를 전개하는 법을 압니다. 그러한 능력이 마드리드 선수단이 필요한 것입니다. 우리가 마주친 어려움으로 인해 그가 뛰지 못하는 것이 안타깝지만, 그가 1군에 합류하기를 바랍니다.<sup class="reference" id="cite_ref-3"><a href="#cite_note-3">[3]</a></sup><sup class="reference" id="cite_ref-4"><a href="#cite_note-4">[4]</a></sup></p>
</blockquote>
<p>감독의 주장은 레알 마드리드가 그와 프로 계약을 맺게 했고,<sup class="reference" id="cite_ref-5"><a href="#cite_note-5">[5]</a></sup> 이후 마테오스는 <a href="/wiki/RCD_%EB%A7%88%EC%9A%94%EB%A5%B4%EC%B9%B4" title="RCD 마요르카">마요르카</a>와의 개막전에 차출되었다.<sup class="reference" id="cite_ref-6"><a href="#cite_note-6">[6]</a></sup> <a href="/wiki/%EB%9D%BC%ED%8C%8C%EC%97%98_%ED%8C%90_%EB%8D%B0%EB%A5%B4_%ED%8C%8C%EB%A5%B4%ED%8A%B8" title="라파엘 판 데르 파르트">라파엘 판 데르 파르트</a>와 <a class="mw-redirect" href="/wiki/%EB%A1%9C%EC%9D%B4%EC%8A%A4%ED%86%A4_%EB%93%9C%EB%A0%8C%ED%85%8C" title="로이스톤 드렌테">로이스톤 드렌테</a>가 각각 <a href="/wiki/%ED%86%A0%ED%8A%B8%EB%84%98_%ED%99%8B%EC%8A%A4%ED%8D%BC_FC" title="토트넘 홋스퍼 FC">토트넘 홋스퍼</a>와 <a href="/wiki/%EC%97%90%EB%A5%B4%EC%BF%A8%EB%A0%88%EC%8A%A4_CF" title="에르쿨레스 CF">에르쿨레스</a>로 각각 떠나면서, 그는 1군으로 승격되어 후자의 선수가 달던 15번을 받았다.<sup class="reference" id="cite_ref-7"><a href="#cite_note-7">[7]</a></sup></p>
<p>2010년 11월 23일, 마테오스는 <a href="/wiki/AFC_%EC%95%84%EC%95%BD%EC%8A%A4" title="AFC 아약스">아약스</a>와의 <a href="/wiki/UEFA_%EC%B1%94%ED%94%BC%EC%96%B8%EC%8A%A4%EB%A6%AC%EA%B7%B8_2010-11" title="UEFA 챔피언스리그 2010-11">그 시즌</a> <a href="/wiki/UEFA_%EC%B1%94%ED%94%BC%EC%96%B8%EC%8A%A4%EB%A6%AC%EA%B7%B8" title="UEFA 챔피언스리그">UEFA 챔피언스리그</a> 경기에 공식 첫 경기를 치렀는데, 경기는 원정 4-0 승리로 끝났다. 경기 후, 그는 "저는 이 날을 잊지 않을 것입니다"라고 말했다.<sup class="reference" id="cite_ref-8"><a href="#cite_note-8">[8]</a></sup></p>
<p>2011년 1월 29일, <a href="/wiki/%EA%B7%B8%EB%A6%AC%EC%8A%A4" title="그리스">그리스</a>의 <a href="/wiki/AEK_%EC%95%84%ED%85%8C%EB%84%A4_FC" title="AEK 아테네 FC">AEK</a>가 마테오스의 6개월 임대에 합의를 보았고, 임대 기간을 1년 연장할 권한도 가져갔다.<sup class="reference" id="cite_ref-9"><a href="#cite_note-9">[9]</a></sup>그러나, 그 해 여름, 그는 같은 방식으로 <a href="/wiki/%EB%A0%88%EC%95%8C_%EC%82%AC%EB%9D%BC%EA%B3%A0%EC%82%AC" title="레알 사라고사">사라고사</a>에 입단했고,<sup class="reference" id="cite_ref-10"><a href="#cite_note-10">[10]</a></sup> 9월 11일, 0-0으로 비긴 <a href="/wiki/%EB%9D%BC%EC%9A%94_%EB%B0%94%EC%98%88%EC%B9%B4%EB%85%B8" title="라요 바예카노">라요 바예카노</a>와의 경기에서 1군 첫 경기를 치렀지만,<sup class="reference" id="cite_ref-11"><a href="#cite_note-11">[11]</a></sup> <a class="new" href="/w/index.php?title=%EB%9D%BC%EB%A6%AC%EA%B0%80_2011-12&amp;action=edit&amp;redlink=1" title="라리가 2011-12 (없는 문서)">시즌</a> 대부분 기간 동안 부상으로 나서지 못했다.<sup class="reference" id="cite_ref-12"><a href="#cite_note-12">[12]</a></sup><sup class="reference" id="cite_ref-13"><a href="#cite_note-13">[13]</a></sup></p>
<p>2012년 6월 말, 마테오스는 <a class="new" href="/w/index.php?title=%EC%84%B8%EA%B5%B0%EB%8B%A4_%EB%94%94%EB%B9%84%EC%8B%9C%EC%98%A8_2012-13&amp;action=edit&amp;redlink=1" title="세군다 디비시온 2012-13 (없는 문서)">2012-13 시즌</a>에 카스티야 복귀에 합의를 보았다. 2군은 5년 만에 <a href="/wiki/%EC%84%B8%EA%B5%B0%EB%8B%A4_%EB%94%94%EB%B9%84%EC%8B%9C%EC%98%A8" title="세군다 디비시온">2부 리그</a>로 <a class="new" href="/w/index.php?title=%EC%84%B8%EA%B5%B0%EB%8B%A4_%EB%94%94%EB%B9%84%EC%8B%9C%EC%98%A8_B_2011-12&amp;action=edit&amp;redlink=1" title="세군다 디비시온 B 2011-12 (없는 문서)">승격</a>되었다.<sup class="reference" id="cite_ref-14"><a href="#cite_note-14">[14]</a></sup></p>
<h3><span class="mw-headline" id=".ED.8E.98.EB.A0.8C.EC.B8.A0.EB.B0.94.EB.A1.9C.EC.8B.9C">페렌츠바로시</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%8B%A4%EB%B9%84%EB%93%9C_%EB%A7%88%ED%85%8C%EC%98%A4%EC%8A%A4&amp;action=edit&amp;section=3" title="부분 편집: 페렌츠바로시">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>2013년 9월 2일, 마테오스는 <a href="/wiki/%ED%97%9D%EA%B0%80%EB%A6%AC" title="헝가리">헝가리</a>의 <a href="/wiki/%ED%8E%98%EB%A0%8C%EC%B8%A0%EB%B0%94%EB%A1%9C%EC%8B%9C_TC" title="페렌츠바로시 TC">페렌츠바로시</a>에 입단했다.<sup class="reference" id="cite_ref-15"><a href="#cite_note-15">[15]</a></sup> 그는 <a class="new" href="/w/index.php?title=%EB%84%B4%EC%A0%9C%ED%8B%B0_%EB%B2%84%EC%9D%B4%EB%85%B9%EC%83%A4%EA%B7%B8_I_2013-14&amp;action=edit&amp;redlink=1" title="넴제티 버이녹샤그 I 2013-14 (없는 문서)">첫 시즌</a>에 21경기에 나서 2골을 기록했고, 소속 구단은 3위를 차지했다.</p>
<h3><span class="mw-headline" id=".EC.98.AC.EB.9E.9C.EB.8F.84_.EC.8B.9C.ED.8B.B0">올랜도 시티</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%8B%A4%EB%B9%84%EB%93%9C_%EB%A7%88%ED%85%8C%EC%98%A4%EC%8A%A4&amp;action=edit&amp;section=4" title="부분 편집: 올랜도 시티">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>2015년 7월 29일, 마테오스는 <a href="/wiki/%EB%A9%94%EC%9D%B4%EC%A0%80_%EB%A6%AC%EA%B7%B8_%EC%82%AC%EC%BB%A4" title="메이저 리그 사커">메이저 리그 사커</a>의 <a href="/wiki/%EC%98%AC%EB%9E%9C%EB%8F%84_%EC%8B%9C%ED%8B%B0_SC" title="올랜도 시티 SC">올랜도 시티</a>에 입단했고, 폴 맥도너 <a class="mw-redirect" href="/wiki/%EB%8B%A8%EC%9E%A5" title="단장">단장</a>은 그를 "뛰어난 속도"를 지닌 선수로 "즉시 영향을 줄 수 있는" 선수로 기대되었다.<sup class="reference" id="cite_ref-16"><a href="#cite_note-16">[16]</a></sup></p>
