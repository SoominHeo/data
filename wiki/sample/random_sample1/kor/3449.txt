<p><b>다비드 마르셀로 피사로 코르테스</b>(<span style="font-size: smaller;"><a href="/wiki/%EC%8A%A4%ED%8E%98%EC%9D%B8%EC%96%B4" title="스페인어">스페인어</a>:</span> <span lang="es" xml:lang="es">David Marcelo Pizarro Cortés</span>, <a href="/wiki/1979%EB%85%84" title="1979년">1979년</a> <a href="/wiki/9%EC%9B%94_11%EC%9D%BC" title="9월 11일">9월 11일</a>, <a href="/wiki/%EC%B9%A0%EB%A0%88" title="칠레">칠레</a> <a href="/wiki/%EB%B0%9C%ED%8C%8C%EB%9D%BC%EC%9D%B4%EC%86%8C" title="발파라이소">발파라이소</a> ~) 는 <a href="/wiki/%EC%B9%A0%EB%A0%88" title="칠레">칠레</a>의 축구 선수로, 포지션은 <a href="/wiki/%EB%AF%B8%EB%93%9C%ED%95%84%EB%8D%94" title="미드필더">미드필더</a>이며, 현재 <a href="/wiki/%EC%B9%A0%EB%A0%88" title="칠레">칠레</a>의 산티아고 원더러스 소속이다.</p>
<p><a href="/wiki/1999%EB%85%84_%EC%BD%94%ED%8C%8C_%EC%95%84%EB%A9%94%EB%A6%AC%EC%B9%B4" title="1999년 코파 아메리카">1999년 코파 아메리카</a>와 <a href="/wiki/%EC%8B%9C%EB%93%9C%EB%8B%88" title="시드니">시드니</a>에서 열렸던 <a href="/wiki/2000%EB%85%84_%ED%95%98%EA%B3%84_%EC%98%AC%EB%A6%BC%ED%94%BD" title="2000년 하계 올림픽">2000년 하계 올림픽</a>에 칠레 대표로 출전한 바 있다. 2006년 그는 대표팀 은퇴를 밝혔으나 2013년 국가대표팀 감독인 <a href="/wiki/%ED%98%B8%EB%A5%B4%ED%97%A4_%EC%82%BC%ED%8C%8C%EC%98%AC%EB%A6%AC" title="호르헤 삼파올리">호르헤 삼파올리</a>의 설득으로 7년 여만에 대표팀에 복귀했다.</p>
