<h1 class="firstHeading" id="firstHeading" lang="ko">다싱안링 산맥</h1>
<div class="mw-body-content" id="bodyContent">
<div id="siteSub">위키백과, 우리 모두의 백과사전.</div>
<div id="contentSub"></div>
<div class="mw-jump" id="jump-to-nav">
					이동:					<a href="#mw-head">둘러보기</a>, 					<a href="#p-search">검색</a>
</div>
<div class="mw-content-ltr" dir="ltr" id="mw-content-text" lang="ko"><script>function mfTempOpenSection(id){var block=document.getElementById("mf-section-"+id);block.className+=" open-block";block.previousSibling.className+=" open-block";}</script><p><b>다싱안링 산맥</b>(대흥안령산맥, <small><a href="/wiki/%EC%A4%91%EA%B5%AD%EC%96%B4" title="중국어">중국어</a></small> <small><a href="/wiki/%EA%B0%84%EC%B2%B4%EC%9E%90" title="간체자">간체</a>:</small> <span lang="zh-hans" xml:lang="zh-hans">大兴安岭</span>, <small><a href="/wiki/%EC%A0%95%EC%B2%B4%EC%9E%90" title="정체자">정체</a>:</small> <span lang="zh-hant" xml:lang="zh-hant">大興安嶺</span>, <small><a href="/wiki/%ED%95%9C%EC%96%B4_%EB%B3%91%EC%9D%8C" title="한어 병음">병음</a>:</small> <font face="Lucida Sans Unicode">Dàxīng'ānlǐng</font>, <a href="/wiki/%EB%A7%8C%EC%A3%BC%EC%96%B4" title="만주어">만주어</a>:Amba Hinggan)은 중국 북동부 <a class="mw-redirect" href="/wiki/%EB%82%B4%EB%AA%BD%EA%B3%A8" title="내몽골">내몽골</a> 지역의 화산 산맥이다. 대 힝간 산맥(Greater Hinggan Range)으로도 불린다. 남북으로 약 1200km에 걸쳐 뻗어있으며 남쪽으로 갈수록 좁아진다. 산맥은 동쪽의 <a href="/wiki/%EB%91%A5%EB%B2%A0%EC%9D%B4_%ED%8F%89%EC%9B%90" title="둥베이 평원">둥베이 평원</a>과 서쪽의 <a href="/wiki/%EB%AA%BD%EA%B3%A8_%EA%B3%A0%EC%9B%90" title="몽골 고원">몽골 고원</a>을 나눈다. 이 지역의 해발고도는 1200~1300m이고 가장 높은 봉우리는 2035m이다. 산맥의 폭은 북쪽이 306km로 남쪽의 97km보다 훨씬 넓다. 산맥은 약 2억년에서 1억 4천 5백만년 전 사이의 <a href="/wiki/%EC%A5%90%EB%9D%BC%EA%B8%B0" title="쥐라기">쥐라기</a> 때 형성되었고 본래 단층으로 고대의 단층선은 둥베이 평원과 만나는 동쪽의 가장자리에 형성되었다. 산맥은 두드러지게 비대칭적으로 동쪽면은 경사가 가파른 반면 서쪽면은 경사가 완만하며 몽골 고원과 이어진다. 동쪽면은 <a class="mw-redirect" href="/wiki/%EB%84%8C_%EA%B0%95" title="넌 강">넌 강</a>과 <a href="/wiki/%EC%91%B9%ED%99%94_%EA%B0%95" title="쑹화 강">쑹화 강</a>의 수많은 지류들에 의해 크게 절개되어있지만 일반적으로 산맥은 편평한 봉우리에 둘러싸여있다. 산맥은 주로 화성암으로 이루어져있다.</p>
<p>산맥은 울창한 숲이다. 시베리아와 만주 식물군 사이에 있는 <a href="/wiki/%EC%9E%90%EB%B0%94%EC%9D%B4%EC%B9%BC" title="자바이칼">자바이칼</a> 식물군으로 유명한 생태 지역이다. 산맥은 주요한 기후 경계를 형성한다. 동쪽은 남동풍이 불어와 상대적으로 습한 기후(연평균 강수량 500mm 이상)인 반면 서쪽은 건조한 지방이다. 산맥의 북부는 중국 동부에서 가장 추운 지역으로 겨울의 기온은 -28 °C까지 떨어지고 많은 지역이 영구 동토층이다. 이 지역의 고산 지대는 낙엽송, 자작나무, 사시나무, 소나무 숲과 관목으로 덮여있다. 사슴, 엘크, 담비, 기타 모피 동물을 포함한 야생 동물이 많다. 산맥의 중남부는 북부보다 더 온화하고 건조하여 1월 기온은 −21 °C이고 연 강수량은 250~300mm이다. 북쪽에서 남쪽으로 내려올수록 침엽수림은 점차 활엽수립과 삼림 사이로 산재한 초원에 자리를 내준다.</p>
<p>산비탈은 상대적으로 풍요로운 목초지이자 10세기에 <a href="/wiki/%EC%9A%94%EB%82%98%EB%9D%BC" title="요나라">요나라</a>를 세운 <a class="mw-redirect" href="/wiki/%EA%B1%B0%EB%9E%80%EC%A1%B1" title="거란족">거란족</a>의 발흥 지역이다.</p>
<p>다싱안링 지역은 20세기까지 미지의 땅이었다. 이 지역 북부의 개발은 20세기 초에 <a href="/wiki/%ED%97%A4%EC%9D%B4%EB%A3%BD%EC%9E%A5_%EC%84%B1" title="헤이룽장 성">헤이룽장 성</a> <a class="mw-redirect" href="/wiki/%EC%B9%98%EC%B9%98%ED%95%98%EC%96%BC" title="치치하얼">치치하얼</a>부터 산맥을 통과해 내몽골 북동부의 러시아와 인접한 <a class="mw-redirect" href="/wiki/%EB%A7%8C%EC%A0%80%EC%9A%B0%EB%A6%AC" title="만저우리">만저우리</a>, <a href="/wiki/%ED%9B%84%EB%A3%AC_%ED%98%B8" title="후룬 호">후룬 호</a> 북쪽으로 가는 철도가 부설되면서 시작되었다. 만주의 일본 점령기(1931~1945)에 이 지역의 삼림 벌채를 위해 산맥 남북으로 수많은 철도가 부설되었다. 이후 산맥 남쪽으로 타오얼 강 계곡을 따라 <a href="/wiki/%EC%A7%80%EB%A6%B0_%EC%84%B1" title="지린 성">지린 성</a> <a class="mw-redirect" href="/wiki/%EB%B0%94%EC%9D%B4%EC%B2%AD" title="바이청">바이청</a>과 <a class="mw-redirect" href="/wiki/%EB%82%B4%EB%AA%BD%EA%B3%A8" title="내몽골">내몽골</a> <a class="mw-redirect" href="/wiki/%EC%95%84%EC%96%BC%EC%82%B0" title="아얼산">아얼산</a>의 온천을 잇는 노선이 부설되었다.</p>
<p>이 지역의 대부분은 <a href="/wiki/%EB%AA%BD%EA%B3%A8%EC%96%B4" title="몽골어">몽골어</a>를 사용하는 사람들이 거주하고 북쪽으로 만주-퉁구스 계통의 언어를 사용하는 <a class="new" href="/w/index.php?title=%EC%98%A4%EB%A1%9C%EC%B4%8C&amp;action=edit&amp;redlink=1" title="오로촌 (없는 문서)">오로촌</a>과 <a class="mw-redirect" href="/wiki/%EC%98%88%EB%B2%A4%ED%82%A4%EC%A1%B1" title="예벤키족">예벤키족</a>이 살고있다.</p>
<p>벌채는 여전히 이 지역의 중요한 경제 활동이다.</p>
