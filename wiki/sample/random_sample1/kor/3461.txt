<p><b>다우다 카이라바 자와라</b> 경(Sir Dawda Kairaba Jawara, <a class="mw-redirect" href="/wiki/%EC%84%B8%EC%9D%B8%ED%8A%B8%EB%A7%88%EC%9D%B4%ED%81%B4%EC%95%A4%EB%93%9C%EC%84%B8%EC%9D%B8%ED%8A%B8%EC%A1%B0%EC%A7%80%ED%9B%88%EC%9E%A5" title="세인트마이클앤드세인트조지훈장">GCMG</a>, <a href="/wiki/1924%EB%85%84" title="1924년">1924년</a> <a href="/wiki/5%EC%9B%94_16%EC%9D%BC" title="5월 16일">5월 16일</a> ~)은 <a href="/wiki/%EA%B0%90%EB%B9%84%EC%95%84" title="감비아">감비아</a>의 <a class="mw-redirect" href="/wiki/%EC%A0%95%EC%B9%98%EC%9D%B8" title="정치인">정치인</a>이다.</p>
<p><a href="/wiki/%EC%98%81%EA%B5%AD" title="영국">영국</a>의 <a href="/wiki/%EA%B8%80%EB%9E%98%EC%8A%A4%EA%B3%A0" title="글래스고">글래스고</a>, <a href="/wiki/%EB%A6%AC%EB%B2%84%ED%92%80" title="리버풀">리버풀</a>에서 <a href="/wiki/%EC%88%98%EC%9D%98%ED%95%99" title="수의학">수의학</a>을 배웠으며, 1962년 ~ 1970년 <a class="new" href="/w/index.php?title=%EA%B0%90%EB%B9%84%EC%95%84%EC%9D%98_%EC%B4%9D%EB%A6%AC&amp;action=edit&amp;redlink=1" title="감비아의 총리 (없는 문서)">감비아의 총리</a>를 지내고, 1970년 <a href="/wiki/%EA%B0%90%EB%B9%84%EC%95%84%EC%9D%98_%EB%8C%80%ED%86%B5%EB%A0%B9" title="감비아의 대통령">감비아의 제1대 대통령</a>이 되었다.</p>
<p>1981년 <a class="mw-redirect" href="/wiki/%EC%B0%B0%EC%8A%A4_%EC%99%95%EC%84%B8%EC%9E%90" title="찰스 왕세자">찰스 왕세자</a>와 <a class="mw-redirect" href="/wiki/%EB%8B%A4%EC%9D%B4%EC%95%A0%EB%82%98_%EB%B9%84" title="다이애나 비">다이애나 비</a>의 결혼식에 참석하는 동안, 쿠코이 삼바 사냥 (Kukoi Sanyang) 이 쿠데타를 일으켰으나, 세네갈 군에 의해 진압되었다. 1994년 <a class="mw-redirect" href="/wiki/%EC%95%BC%EC%95%BC_%EC%9E%90%EB%A9%94" title="야야 자메">야야 자메</a> 중위가 일으킨 쿠데타로 실각했으며, 2002년까지 망명 생활을 하다가 감비아로 돌아왔다.</p>
<p>현재 감비아의 파자라 (Fajara) 에 거주하고 있으며, 1984년 <a href="/wiki/%EB%8C%80%ED%95%9C%EB%AF%BC%EA%B5%AD" title="대한민국">대한민국</a>을 방문한 바 있다.</p>
