<p><b>다이쇼 천황</b>(<span style="font-size: smaller;"><a href="/wiki/%EC%9D%BC%EB%B3%B8%EC%96%B4" title="일본어">일본어</a>:</span> <span lang="ja"><span lang="ja"><ruby><rb>大正天皇</rb> <rp>(</rp><rt>たいしょうてんのう</rt><rp>)</rp></ruby></span></span> <small>타이쇼 텐노<sup>[<a href="/wiki/%EC%9C%84%ED%82%A4%EB%B0%B1%EA%B3%BC:%EC%9D%BC%EB%B3%B8%EC%96%B4%EC%9D%98_%ED%95%9C%EA%B8%80_%ED%91%9C%EA%B8%B0" title="위키백과:일본어의 한글 표기">*</a>]</sup></small>, <a href="/wiki/1879%EB%85%84" title="1879년">1879년</a> <a href="/wiki/8%EC%9B%94_31%EC%9D%BC" title="8월 31일">8월 31일</a> ~ <a href="/wiki/1926%EB%85%84" title="1926년">1926년</a> <a href="/wiki/12%EC%9B%94_25%EC%9D%BC" title="12월 25일">12월 25일</a>)은 <a href="/wiki/%EC%9D%BC%EB%B3%B8" title="일본">일본</a>의 제123대 <a href="/wiki/%EC%9D%BC%EB%B3%B8_%EC%B2%9C%ED%99%A9" title="일본 천황">천황</a>으로 본명은 <b>요시히토</b>(<span lang="ja"><ruby><rb>嘉仁</rb> <rp>(</rp><rt>よしひと</rt><rp>)</rp></ruby></span>), 어릴 적 칭호는 하루노미야(<span lang="ja"><ruby><rb>明宮</rb> <rp>(</rp><rt>はるのみや</rt><rp>)</rp></ruby></span>)이다.</p>
<p><a href="/wiki/1912%EB%85%84" title="1912년">1912년</a>부터 <a href="/wiki/1926%EB%85%84" title="1926년">1926년</a>까지 재위하였고 <a href="/wiki/1912%EB%85%84" title="1912년">1912년</a>부터 <a href="/wiki/1921%EB%85%84" title="1921년">1921년</a>까지 친정을 하였으며 <a href="/wiki/1921%EB%85%84" title="1921년">1921년</a>부터 <a href="/wiki/1926%EB%85%84" title="1926년">1926년</a> 붕어할 때까지는 병으로 정사를 수행할 수 없어서 그의 아들인 <a href="/wiki/%EC%87%BC%EC%99%80_%EC%B2%9C%ED%99%A9" title="쇼와 천황">히로히토</a> 황태자가 사실상 <a href="/wiki/%EB%8C%80%EB%A6%AC%EC%B2%AD%EC%A0%95" title="대리청정">대리청정</a>하였다.</p>

<p><a href="/wiki/1879%EB%85%84" title="1879년">1879년</a> <a href="/wiki/8%EC%9B%94_31%EC%9D%BC" title="8월 31일">8월 31일</a>, <a href="/wiki/%EB%8F%84%EC%BF%84" title="도쿄">도쿄</a>의 아오야마(靑山) 궁전에서 <a href="/wiki/%EB%A9%94%EC%9D%B4%EC%A7%80_%EC%B2%9C%ED%99%A9" title="메이지 천황">메이지 천황</a>과 그의 측실 야나기하라 나루코(柳原德子) 후궁 사이에서 <a class="mw-redirect" href="/wiki/%EC%84%9C%EC%9E%90" title="서자">서자</a>로 출생하였다. 생후 <a href="/wiki/%EC%8A%B5%EC%A7%84" title="습진">습진</a>을 겪는 등 건강이 좋지 못해 이듬해까지 무거운 병을 앓았다. 메이지 천황과 그의 정실인 하루코(美子) 황후 사이에서의 자녀가 없는 데다가 측실인 후궁들 소생의 4명의 자녀까지 잇따라 사망한 상태였으므로 후에 황태자에 올랐으며 어릴 적에는 하루코 황후의 친자식이라 알고 있었기 때문에 생모가 야나기하라 나루코 후궁이라는 것을 알았을 때 큰 충격을 받았다고 한다.</p>

<h4><span class="mw-headline" id=".EC.96.B4.EB.A6.B0_.EC.8B.9C.EC.A0.88">어린 시절</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%8B%A4%EC%9D%B4%EC%87%BC_%EC%B2%9C%ED%99%A9&amp;action=edit&amp;section=4" title="부분 편집: 어린 시절">편집</a><span class="mw-editsection-bracket">]</span></span></h4>
<p><a href="/wiki/1880%EB%85%84" title="1880년">1880년</a>, 화족(華族) 출신의 나카야마 다다요시(中山忠敬)의 양자가 되었으나 <a href="/wiki/1885%EB%85%84" title="1885년">1885년</a> 3월에 파양된 후 다시 아오야마로 돌아와 쓸쓸한 어린 시절을 보냈다. <a href="/wiki/1887%EB%85%84" title="1887년">1887년</a>, 부황 <a href="/wiki/%EB%A9%94%EC%9D%B4%EC%A7%80_%EC%B2%9C%ED%99%A9" title="메이지 천황">메이지 천황</a>의 정실인 적모(嫡母) 하루코 황후의 정식 양자가 되었고 같은 해 9월, <a class="mw-redirect" href="/wiki/%ED%95%99%EC%8A%B5%EC%9B%90" title="학습원">학습원</a>에 입학했지만 건강이 좋지 못해 <a class="new" href="/w/index.php?title=%EC%9C%A0%EA%B8%89&amp;action=edit&amp;redlink=1" title="유급 (없는 문서)">유급</a>되는 일이 있었으며 학습원의 어려운 규칙에 적응하지 못해 <a href="/wiki/1889%EB%85%84" title="1889년">1889년</a>, <a class="new" href="/w/index.php?title=%EC%95%84%ED%83%80%EB%AF%B8&amp;action=edit&amp;redlink=1" title="아타미 (없는 문서)">아타미</a>(熱海)에서 정양을 하기도 하였다. <a href="/wiki/1889%EB%85%84" title="1889년">1889년</a>, 입태자례(立太子禮)를 치렀으나 이수계의 교과를 어려워하는 등 학습원에서의 학습이 여전히 진행되지 않자 결국 중퇴를 선택, 아카사카(赤坂) 별궁에서 <a href="/wiki/%ED%94%84%EB%9E%91%EC%8A%A4%EC%96%B4" title="프랑스어">프랑스어</a>, <a href="/wiki/%EC%9C%A0%EA%B5%90" title="유교">유교학</a>, <a href="/wiki/%ED%95%9C%EB%AC%B8" title="한문">한문</a> 등 개인 교습을 받았다. 그러나 이도 곧 여의치 않았으므로 메이지 천황은 새로운 <a class="new" href="/w/index.php?title=%EB%8F%99%EA%B6%81%EB%B3%B4%EB%8F%84&amp;action=edit&amp;redlink=1" title="동궁보도 (없는 문서)">동궁보도</a>(東宮輔導)에 방계 왕족인 <a class="mw-redirect" href="/wiki/%EC%95%84%EB%A6%AC%EC%8A%A4%EA%B0%80%EC%99%80%EB%85%B8%EB%AF%B8%EC%95%BC_%EB%8B%A4%EC%BC%80%ED%9E%88%ED%86%A0" title="아리스가와노미야 다케히토">아리스가와노미야 다케히토</a>를 임명하였고 이후 요시히토는 다케히토를 벗 삼아 학업을 계속하였다.</p>
<h4><span class="mw-headline" id=".EA.B2.B0.ED.98.BC">결혼</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%8B%A4%EC%9D%B4%EC%87%BC_%EC%B2%9C%ED%99%A9&amp;action=edit&amp;section=5" title="부분 편집: 결혼">편집</a><span class="mw-editsection-bracket">]</span></span></h4>
<p>병약한 황태자를 빨리 결혼시키자는 목소리가 높아짐에 따라 <a href="/wiki/1900%EB%85%84" title="1900년">1900년</a> <a href="/wiki/5%EC%9B%94_10%EC%9D%BC" title="5월 10일">5월 10일</a>, 요시히토는 화족(華族) 출신의 <a href="/wiki/%EB%8D%B0%EC%9D%B4%EB%A9%94%EC%9D%B4_%ED%99%A9%ED%9B%84" title="데이메이 황후">구조 사다코</a>와 혼인했다. 그 뒤에 좋지 않았던 건강이 점차 회복되어 갔다. 다이쇼 천황은 후궁을 거느리지 않았으며, 일부일처제의 모범이 되었다.<sup class="reference" id="cite_ref-1"><a href="#cite_note-1">[1]</a></sup></p>
<h3><span class="mw-headline" id=".EC.B2.9C.ED.99.A9_.EC.8B.9C.EC.A0.88">천황 시절</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%8B%A4%EC%9D%B4%EC%87%BC_%EC%B2%9C%ED%99%A9&amp;action=edit&amp;section=6" title="부분 편집: 천황 시절">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<h4><span class="mw-headline" id=".EC.A6.89.EC.9C.84">즉위</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%8B%A4%EC%9D%B4%EC%87%BC_%EC%B2%9C%ED%99%A9&amp;action=edit&amp;section=7" title="부분 편집: 즉위">편집</a><span class="mw-editsection-bracket">]</span></span></h4>
<p><a href="/wiki/1912%EB%85%84" title="1912년">1912년</a> <a href="/wiki/7%EC%9B%94_30%EC%9D%BC" title="7월 30일">7월 30일</a>, 부황 메이지 천황이 죽자 황위를 계승했다. 즉위 후에도 병약함과 생각한 것을 곧바로 언동에 나타내는 버릇 등으로 "믿음직스럽지 못하다", "머리가 나쁘다"란 인식이 정계를 비롯한 일본 각계로 퍼져 나갔으며 때문에 후에는 <b>존재감이 없는 천황</b>(<span lang="ja">影の薄い天皇</span>)으로 불렸다. <a href="/wiki/1917%EB%85%84" title="1917년">1917년</a>경부터 공무로 받은 스트레스와 심로 등이 겹쳐 다시 건강이 악화, 급기야 <a href="/wiki/1921%EB%85%84" title="1921년">1921년</a>에는 황태자 <a href="/wiki/%EC%87%BC%EC%99%80_%EC%B2%9C%ED%99%A9" title="쇼와 천황">히로히토</a>가 <a href="/wiki/%EB%8C%80%EB%A6%AC%EC%B2%AD%EC%A0%95" title="대리청정">대리청정</a>으로 취임하기에 이르렀다. 이 때문에 일본 정치인들은 그를 무시하는 경우가 종종 있었다고 한다.</p>
<h4><span class="mw-headline" id=".EC.82.AC.EB.A7.9D">사망</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%8B%A4%EC%9D%B4%EC%87%BC_%EC%B2%9C%ED%99%A9&amp;action=edit&amp;section=8" title="부분 편집: 사망">편집</a><span class="mw-editsection-bracket">]</span></span></h4>
<p><a href="/wiki/1926%EB%85%84" title="1926년">1926년</a> <a href="/wiki/12%EC%9B%94_25%EC%9D%BC" title="12월 25일">12월 25일</a> 새벽 1시 25분 경 하야마(葉山) 황실 별장에서 친어머니의 손을 잡은 채 <a class="mw-disambig" href="/wiki/%EC%8B%AC%EC%9E%A5%EB%A7%88%EB%B9%84" title="심장마비">심장마비</a>로 사망하였다. 천황의 유해는 전례를 깨고 영구차로 도쿄 서쪽 근교의 <a href="/wiki/%ED%95%98%EC%B9%98%EC%98%A4%EC%A7%80_%EC%8B%9C" title="하치오지 시">하치오지 시</a>에 있는 타마 어릉에 안치되었다.</p>
<h2><span class="mw-headline" id=".EB.8B.B9.EC.8B.9C_.ED.95.9C.EA.B5.AD.EA.B3.BC.EC.9D.98_.EC.83.81.ED.99.A9">당시 한국과의 상황</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%8B%A4%EC%9D%B4%EC%87%BC_%EC%B2%9C%ED%99%A9&amp;action=edit&amp;section=9" title="부분 편집: 당시 한국과의 상황">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p><a href="/wiki/1907%EB%85%84" title="1907년">1907년</a> <a href="/wiki/%EB%8C%80%ED%95%9C%EC%A0%9C%EA%B5%AD" title="대한제국">대한제국</a>을 방문해 <a class="mw-redirect" href="/wiki/%EC%9D%98%EB%AF%BC%ED%99%A9%ED%83%9C%EC%9E%90" title="의민황태자">의민황태자</a> 이은(李垠)을 만난 요시히토는 그를 매우 마음에 들어하여 이튿날부터 <a href="/wiki/%ED%95%9C%EA%B5%AD%EC%96%B4" title="한국어">한국어</a>를 배우기 시작했다고 하는데 그의 한국어 학습이 어느 정도까지 진행되었는가는 확실치 않으나 천황 즉위 후에도 계속되었다고 전해지고 있으며 작고 직전 의민태자가 그를 문병했을 당시 한국어를 하려고 하였으나 이미 병으로 말을 하지 못하는 상태였기 때문에 한국어 같지 않은 한국어를 했다고 전해진다.</p>

<h2><span class="mw-headline" id=".EC.9E.90.EB.85.80">자녀</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%8B%A4%EC%9D%B4%EC%87%BC_%EC%B2%9C%ED%99%A9&amp;action=edit&amp;section=10" title="부분 편집: 자녀">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
