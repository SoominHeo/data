<p>《<b>다즐링 주식회사</b>》(<span lang="en" xml:lang="en"><i>The Darjeeling Limited</i></span>)은 2007년에 제작된 <a href="/wiki/%EC%9B%A8%EC%8A%A4_%EC%95%A4%EB%8D%94%EC%8A%A8" title="웨스 앤더슨">웨스 앤더슨</a> 감독의 <a href="/wiki/%EC%BD%94%EB%AF%B8%EB%94%94_%EB%93%9C%EB%9D%BC%EB%A7%88" title="코미디 드라마">코미디 드라마</a> 장르의 영화로, <a href="/wiki/%EC%98%A4%EC%96%B8_%EC%9C%8C%EC%8A%A8" title="오언 윌슨">오언 윌슨</a>, <a href="/wiki/%EC%97%90%EC%9D%B4%EB%93%9C%EB%A6%AC%EC%96%B8_%EB%B8%8C%EB%A1%9C%EB%94%94" title="에이드리언 브로디">에이드리언 브로디</a>, <a href="/wiki/%EC%A0%9C%EC%9D%B4%EC%8A%A8_%EC%8A%88%EC%9B%8C%EC%B8%A0%EB%A8%BC" title="제이슨 슈워츠먼">제이슨 슈워츠먼</a>이 주연으로 출연하였다. 또한 <a class="new" href="/w/index.php?title=%EC%99%80%EB%A6%AC%EC%8A%A4_%EC%95%8C%EB%A3%A8%EC%99%80%EB%A6%AC%EC%95%84&amp;action=edit&amp;redlink=1" title="와리스 알루와리아 (없는 문서)">와리스 알루와리아</a>, <a class="new" href="/w/index.php?title=%EC%95%84%EB%A7%88%EB%9D%BC_%EC%B9%B4%EB%A0%8C&amp;action=edit&amp;redlink=1" title="아마라 카렌 (없는 문서)">아마라 카렌</a>, <a href="/wiki/%EB%B0%94%EB%A5%B4%EB%B2%A0_%EC%8A%88%EB%A2%B0%EB%8D%94" title="바르베 슈뢰더">바르베 슈뢰더</a>, <a href="/wiki/%EC%95%A4%EC%A0%A4%EB%A6%AC%EC%B9%B4_%ED%9C%B4%EC%8A%A4%ED%84%B4" title="앤젤리카 휴스턴">앤젤리카 휴스턴</a>, <a href="/wiki/%EB%82%98%ED%83%88%EB%A6%AC_%ED%8F%AC%ED%8A%B8%EB%A7%8C" title="나탈리 포트만">나탈리 포트만</a>, <a class="new" href="/w/index.php?title=%EC%B9%B4%EB%B0%80%EB%9D%BC_%EB%A3%A8%EB%8D%94%ED%8F%AC%EB%93%9C&amp;action=edit&amp;redlink=1" title="카밀라 루더포드 (없는 문서)">카밀라 루더포드</a>, <a href="/wiki/%EC%9D%B4%EB%A5%B4%ED%8C%90_%EC%B9%B8" title="이르판 칸">이르판 칸</a>, <a href="/wiki/%EB%B9%8C_%EB%A8%B8%EB%A6%AC" title="빌 머리">빌 머리</a> 등이 <a href="/wiki/%EC%B9%B4%EB%A9%94%EC%98%A4_%EC%B6%9C%EC%97%B0" title="카메오 출연">카메오 출연</a>하였다. 원제 The Darjeeling Limited는 '다즐링 특급열차'라고 해석되는 것이 옳다.</p>
