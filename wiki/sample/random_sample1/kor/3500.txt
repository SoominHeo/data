<p><b>다카르</b>(<span style="font-size: smaller;"><a href="/wiki/%ED%94%84%EB%9E%91%EC%8A%A4%EC%96%B4" title="프랑스어">프랑스어</a>:</span> <span lang="fr" xml:lang="fr">Dakar</span>, <a href="/wiki/%EB%AC%B8%ED%99%94%EC%96%B4" title="문화어">문화어</a>: 다까르)는 <a href="/wiki/%EC%84%B8%EB%84%A4%EA%B0%88" title="세네갈">세네갈</a>의 <a href="/wiki/%EC%88%98%EB%8F%84" title="수도">수도</a>로 인구 207만9,000명(2000년)이 거주하고 있다. <a href="/wiki/%EB%8C%80%EC%84%9C%EC%96%91" title="대서양">대서양</a>에 접해 있는 무역항이다. <a href="/wiki/%EC%95%84%ED%94%84%EB%A6%AC%EC%B9%B4" title="아프리카">아프리카</a>의 최서단에 위치해 있고 대서양 횡단 무역 및 유럽 무역에 있어서, 중요한 위치를 차지하는 중요한 <a href="/wiki/%ED%95%AD%EB%A7%8C" title="항만">항만</a>이다.</p>
<p>다카르는 <a href="/wiki/%ED%94%84%EB%9E%91%EC%8A%A4" title="프랑스">프랑스</a>에 의해 도시를 이루었다. <a href="/wiki/1902%EB%85%84" title="1902년">1902년</a>에 <a href="/wiki/%EC%83%9D%EB%A3%A8%EC%9D%B4_(%EC%84%B8%EB%84%A4%EA%B0%88)" title="생루이 (세네갈)">생루이</a>로 바뀌었고 <a href="/wiki/%ED%94%84%EB%9E%91%EC%8A%A4%EB%A0%B9_%EC%84%9C%EC%95%84%ED%94%84%EB%A6%AC%EC%B9%B4" title="프랑스령 서아프리카">프랑스령 서아프리카</a>의 수도가 되었다. <a href="/wiki/%EC%A0%9C2%EC%B0%A8_%EC%84%B8%EA%B3%84_%EB%8C%80%EC%A0%84" title="제2차 세계 대전">제2차 세계 대전</a> 도중에 세네갈 식민지는 <a class="mw-redirect" href="/wiki/%EB%B9%84%EC%8B%9C_%EC%A0%95%EB%B6%80" title="비시 정부">비시 정부</a>에 충성을 맹세했기 때문에, <a href="/wiki/1940%EB%85%84" title="1940년">1940년</a> <a href="/wiki/9%EC%9B%94" title="9월">9월</a> <a class="mw-redirect" href="/wiki/%EB%93%9C%EA%B3%A8" title="드골">드골</a>의 <a href="/wiki/%EC%9E%90%EC%9C%A0_%ED%94%84%EB%9E%91%EC%8A%A4" title="자유 프랑스">자유 프랑스</a>군이 다카르를 공격했지만, 실패에 끝났던 적이 있다(<a class="new" href="/w/index.php?title=%EB%8B%A4%EC%B9%B4%EB%A5%B4_%ED%95%B4%EC%A0%84&amp;action=edit&amp;redlink=1" title="다카르 해전 (없는 문서)">다카르 해전</a>). <a href="/wiki/1959%EB%85%84" title="1959년">1959년</a>부터 <a href="/wiki/1960%EB%85%84" title="1960년">1960년</a>까지는 <a href="/wiki/%EB%A7%90%EB%A6%AC_%EC%97%B0%EB%B0%A9" title="말리 연방">말리 연방</a>의 수도였고 나중에 세네갈의 수도가 되어 현재에 이른다.</p>
<p>유럽풍의 아름다운 시가가 있다. 3대륙의 십자로에 위치하여 해·공 교통의 요충이 되어 있다. <a href="/wiki/%EC%88%98%EC%97%90%EC%A6%88_%EC%9A%B4%ED%95%98" title="수에즈 운하">수에즈 운하</a> 봉쇄 이후에는 특히 <a href="/wiki/%EC%BC%80%EC%9D%B4%ED%94%84%ED%83%80%EC%9A%B4" title="케이프타운">케이프타운</a>을 회항하는 선박들이 보급을 위해 많이 기항하고 있다. 검은 아프리카 연구의 중심지로 다카르 대학교가 있고, 항구 밖의 골레 섬에는 노예 수용소 자리가 있다. 세네갈 강 하류의 생루이는 18세기까지 무역 근거지로서 발전한 거리로 인구는 120만이다.<sup class="reference" id="cite_ref-1"><a href="#cite_note-1">[1]</a></sup> <a class="new" href="/w/index.php?title=%EB%8B%A4%EC%B9%B4%EB%A5%B4_%EB%9E%A0%EB%A6%AC&amp;action=edit&amp;redlink=1" title="다카르 랠리 (없는 문서)">다카르 랠리</a>의 종착점으로 유명한 도시이다.</p>
