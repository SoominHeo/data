<p><b>다카치호</b>(高千穂)는 <a href="/wiki/%EC%9D%BC%EB%B3%B8_%EC%A0%9C%EA%B5%AD_%ED%95%B4%EA%B5%B0" title="일본 제국 해군">일본 제국 해군</a>의 방호순양함으로 <a class="new" href="/w/index.php?title=%EB%82%98%EB%8B%88%EC%99%80%ED%98%95_%EB%B0%A9%ED%98%B8%EC%88%9C%EC%96%91%ED%95%A8&amp;action=edit&amp;redlink=1" title="나니와형 방호순양함 (없는 문서)">나니와형 방호순양함</a>의 2번함이다. 함 이름은 ‘천손이 강림한 곳’이라는 신화가 내려오고 있는 <a href="/wiki/%EB%AF%B8%EC%95%BC%EC%9E%90%ED%82%A4_%ED%98%84" title="미야자키 현">미야자키 현</a>의 ‘<a class="new" href="/w/index.php?title=%EB%8B%A4%EC%B9%B4%EC%B9%98%ED%98%B8_%EB%B4%89&amp;action=edit&amp;redlink=1" title="다카치호 봉 (없는 문서)">다카치호 봉</a>’(高千穂峰)을 따서 이름 붙여졌다.</p>
<h2><span class="mw-headline" id=".EA.B0.9C.EC.9A.94">개요</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%8B%A4%EC%B9%B4%EC%B9%98%ED%98%B8_(%EB%B0%A9%ED%98%B8%EC%88%9C%EC%96%91%ED%95%A8)&amp;action=edit&amp;section=1" title="부분 편집: 개요">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>타카치호는 <a href="/wiki/1884%EB%85%84" title="1884년">1884년</a> <a href="/wiki/%EC%98%81%EA%B5%AD" title="영국">영국</a>, <a class="mw-disambig" href="/wiki/%EB%89%B4%EC%BA%90%EC%8A%AC" title="뉴캐슬">뉴캐슬</a>에 있는 <a href="/wiki/%EC%95%94%EC%8A%A4%ED%8A%B8%EB%A1%B1_%ED%9C%98%ED%8A%B8%EC%9B%8C%EC%8A%A4" title="암스트롱 휘트워스">암스트롱 휘트워스</a>의 로우워커 조선소에서 기공되어, <a href="/wiki/1886%EB%85%84" title="1886년">1886년</a> 4월말에 준공되어 이등함으로 정해졌다. 직후 일본으로 인도되어 같은 해 <a href="/wiki/7%EC%9B%94_3%EC%9D%BC" title="7월 3일">7월 3일</a>, <a class="mw-redirect" href="/wiki/%EC%8B%9C%EB%82%98%EA%B0%80%EC%99%80" title="시나가와">시나가와</a>에 도착했다. <a href="/wiki/1890%EB%85%84" title="1890년">1890년</a> <a href="/wiki/8%EC%9B%94_23%EC%9D%BC" title="8월 23일">8월 23일</a>, 제1종 군함으로 정해졌다.</p>
<p><a href="/wiki/1894%EB%85%84" title="1894년">1894년</a> 6월부터 다음 달까지 타카치호는 <a class="new" href="/w/index.php?title=%ED%95%98%EC%99%80%EC%9D%B4_%ED%98%81%EB%AA%85&amp;action=edit&amp;redlink=1" title="하와이 혁명 (없는 문서)">하와이 혁명</a>이 발발함에 따라 일본인 보호를 위해 <a href="/wiki/%ED%98%B8%EB%86%80%EB%A3%B0%EB%A3%A8" title="호놀룰루">호놀룰루</a>에 파견되었다.</p>
<p><a href="/wiki/1894%EB%85%84" title="1894년">1894년</a> <a href="/wiki/%EC%B2%AD%EC%9D%BC_%EC%A0%84%EC%9F%81" title="청일 전쟁">청일 전쟁</a>이 발발하자 <a href="/wiki/%ED%99%A9%ED%95%B4_%ED%95%B4%EC%A0%84_(1894%EB%85%84)" title="황해 해전 (1894년)">황해 해전</a>, <a class="mw-redirect" href="/wiki/%EB%8B%A4%EB%A1%84" title="다롄">다롄</a>, <a class="mw-redirect" href="/wiki/%EB%A4%BC%EC%88%9C" title="뤼순">뤼순</a>, <a class="mw-redirect" href="/wiki/%EC%9B%A8%EC%9D%B4%ED%95%98%EC%9D%B4" title="웨이하이">웨이하이</a>, <a class="mw-redirect" href="/wiki/%ED%8E%91%ED%9B%84_%EC%84%AC" title="펑후 섬">펑후 섬</a> 공략 작전 등 여러 전투에 참가한다.</p>
<p><a href="/wiki/1898%EB%85%84" title="1898년">1898년</a> <a href="/wiki/3%EC%9B%94_21%EC%9D%BC" title="3월 21일">3월 21일</a>, 이등순양함으로 분류된다. <a href="/wiki/%EC%9D%98%ED%99%94%EB%8B%A8_%EC%9A%B4%EB%8F%99" title="의화단 운동">의화단 운동</a>이 일어난 <a href="/wiki/1900%EB%85%84" title="1900년">1900년</a> 8월부터 10월까지 <a class="mw-disambig" href="/wiki/%EC%83%A4%EB%A8%BC" title="샤먼">샤먼</a> 지역의 경계를 담당했다.</p>
<p><a href="/wiki/1904%EB%85%84" title="1904년">1904년</a> 1월에 우 조병창에서 기뢰 투하 장치를 장착, <a href="/wiki/%EB%9F%AC%EC%9D%BC_%EC%A0%84%EC%9F%81" title="러일 전쟁">러일 전쟁</a> 시에는 <a href="/wiki/%EC%A0%9C%EB%AC%BC%ED%8F%AC_%ED%95%B4%EC%A0%84" title="제물포 해전">제물포 해전</a>, <a href="/wiki/%EC%9A%B8%EC%82%B0_%ED%95%B4%EC%A0%84" title="울산 해전">울산 해전</a>, <a href="/wiki/%EC%93%B0%EC%8B%9C%EB%A7%88_%ED%95%B4%EC%A0%84" title="쓰시마 해전">쓰시마 해전</a> 등에 참가, <a class="mw-redirect" href="/wiki/%EC%9B%90%EC%82%B0" title="원산">원산</a> 앞바다에서 기뢰 부설작업을 담당했다.</p>
<p><a href="/wiki/1911%EB%85%84" title="1911년">1911년</a>, 다카치호는 <a class="mw-redirect" href="/wiki/%EC%9A%94%EC%BD%94%EC%8A%A4%EC%B9%B4_%ED%95%B4%EA%B5%B0_%EC%A1%B0%EC%84%A0%EC%86%8C" title="요코스카 해군 조선소">요코스카 해군 조선소</a>에 정박하여 군함으로 개조되어 <a href="/wiki/1912%EB%85%84" title="1912년">1912년</a> <a href="/wiki/8%EC%9B%94_28%EC%9D%BC" title="8월 28일">8월 28일</a>, 이등해방함으로 등급이 변경되었다.</p>
<p><a href="/wiki/%EC%A0%9C1%EC%B0%A8_%EC%84%B8%EA%B3%84_%EB%8C%80%EC%A0%84" title="제1차 세계 대전">제1차 세계 대전</a>이 발발하자 <a href="/wiki/1914%EB%85%84" title="1914년">1914년</a> 8월부터 9월까지 <a href="/wiki/%EC%B9%AD%EB%8B%A4%EC%98%A4_%EC%A0%84%ED%88%AC" title="칭다오 전투">칭다오 공략전</a>에 참가했지만, <a href="/wiki/10%EC%9B%94_17%EC%9D%BC" title="10월 17일">10월 17일</a> 해상 봉쇄 중인 <a href="/wiki/%EC%9E%90%EC%98%A4%EC%A0%80%EC%9A%B0_%EB%A7%8C" title="자오저우 만">자오저우 만</a> 밖에서 <a href="/wiki/%EB%8F%85%EC%9D%BC_%ED%95%B4%EA%B5%B0" title="독일 해군">독일 해군</a>의 <a href="/wiki/%EC%96%B4%EB%A2%B0%EC%A0%95" title="어뢰정">어뢰정</a> S90(SMS S90)의 뇌격에 타카치호에 탑재된 기뢰가 유폭을 일으켜 침몰했다. 타카치호는 일본 해군의 군함 중 적과 교전 중에 격침된 최초의 군함이기도 했다.</p>
