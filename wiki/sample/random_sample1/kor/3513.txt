<p><a href="/wiki/1854%EB%85%84" title="1854년">1854년</a> <a href="/wiki/9%EC%9B%94_19%EC%9D%BC" title="9월 19일">9월 19일</a> <a href="/wiki/%EC%97%90%EB%8F%84_%EB%A7%89%EB%B6%80" title="에도 막부">에도 막부</a>의 어용 화가 가와무라의 아들로 태어났으며,곧 <a href="/wiki/%EC%84%BC%EB%8B%A4%EC%9D%B4_%EB%B2%88" title="센다이 번">센다이 번</a>의 최하급 <a href="/wiki/%EB%AC%B4%EC%82%AC" title="무사">무사</a> 다카하시의 <a class="mw-disambig" href="/wiki/%EC%96%91%EC%9E%90" title="양자">양자</a>로 보내졌다.</p>
<h3><span class="mw-headline" id=".EB.B0.B0.EC.9B.80">배움</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%8B%A4%EC%B9%B4%ED%95%98%EC%8B%9C_%EA%B3%A0%EB%A0%88%ED%82%A4%EC%9A%94&amp;action=edit&amp;section=3" title="부분 편집: 배움">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>지금의 <a class="extiw" href="https://ja.wikipedia.org/wiki/%E6%98%8E%E6%B2%BB%E5%AD%A6%E9%99%A2%E5%A4%A7%E5%AD%A6" title="ja:明治学院大学">메이지 가쿠인 대학</a>인 <a href="/wiki/%EC%9A%94%EC%BD%94%ED%95%98%EB%A7%88_%EC%8B%9C" title="요코하마 시">요코하마</a>의 미국인 <a href="/wiki/%EC%9D%98%EC%82%AC" title="의사">의사</a>이자 <a href="/wiki/%EA%B0%9C%ED%98%81_%EA%B5%90%ED%9A%8C" title="개혁 교회">개혁교회</a>(Reformed Church,<a href="/wiki/%EC%9E%A5%EB%A1%9C%EA%B5%90" title="장로교">장로교회</a>)<a href="/wiki/%EC%84%A0%EA%B5%90%EC%82%AC" title="선교사">선교사</a>인 <a class="extiw" href="https://en.wikipedia.org/wiki/James_Curtis_Hepburn" title="en:James Curtis Hepburn">제임스 커티스 헵번</a><sup class="reference" id="cite_ref-1"><a href="#cite_note-1">[1]</a></sup>의 사숙에서 배웠으며, <a href="/wiki/1867%EB%85%84" title="1867년">1867년</a> <a href="/wiki/%EC%84%BC%EB%8B%A4%EC%9D%B4_%EB%B2%88" title="센다이 번">센다이 번</a>의 명령에 의해 <a href="/wiki/%EA%B0%80%EC%93%B0_%EA%B0%80%EC%9D%B4%EC%8A%88" title="가쓰 가이슈">가쓰 가이슈</a>의 아들과 함께 <a href="/wiki/%EB%AF%B8%EA%B5%AD" title="미국">미국</a>으로 유학을 가 어렵게 공부하고 <a href="/wiki/%EC%9D%BC%EB%B3%B8" title="일본">일본</a>으로 돌아왔다. 귀국 후에는 <a class="extiw" href="https://ja.wikipedia.org/wiki/%E6%96%87%E9%83%A8%E7%9C%81" title="ja:文部省">문부성</a>과 <a class="extiw" href="https://ja.wikipedia.org/wiki/%E8%BE%B2%E5%95%86%E5%8B%99%E7%9C%81_(%E6%97%A5%E6%9C%AC)" title="ja:農商務省 (日本)">농상무성</a>의 관료로 활약했으며, 그 뒤에는 특허국장을 거쳐 <a href="/wiki/1892%EB%85%84" title="1892년">1892년</a> <a href="/wiki/%EC%9D%BC%EB%B3%B8%EC%9D%80%ED%96%89" title="일본은행">일본 은행</a>에 들어가 일본 은행 부총재와 일본 은행 총재 등을 맡았다.</p>

<h3><span class="mw-headline" id=".EC.A0.95.EC.B9.98_.EC.9E.85.EB.AC.B8">정치 입문</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%8B%A4%EC%B9%B4%ED%95%98%EC%8B%9C_%EA%B3%A0%EB%A0%88%ED%82%A4%EC%9A%94&amp;action=edit&amp;section=4" title="부분 편집: 정치 입문">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p><a href="/wiki/1913%EB%85%84" title="1913년">1913년</a> 제1차 <a href="/wiki/%EC%95%BC%EB%A7%88%EB%AA%A8%ED%86%A0_%EA%B3%A4%EB%85%B8%ED%9A%A8%EC%97%90" title="야마모토 곤노효에">야마모토 곤노효에</a> 내각에서 대장 대신으로 취임하였고, 이때 입헌정우회에 입당하였다. 또, 정우회 소속의 <a href="/wiki/%ED%95%98%EB%9D%BC_%EB%8B%A4%EC%B9%B4%EC%8B%9C" title="하라 다카시">하라 다카시</a>가 내각을 조성 했을 때도 대장 대신이 되었고, 하라 다카시가 암살된 직후에는 재정 정책의 수완이 평가돼 제20대 <a href="/wiki/%EC%9D%BC%EB%B3%B8%EC%9D%98_%EB%82%B4%EA%B0%81%EC%B4%9D%EB%A6%AC%EB%8C%80%EC%8B%A0" title="일본의 내각총리대신">내각총리대신</a>에 올랐다. 그러나, 중심인 <a href="/wiki/%ED%95%98%EB%9D%BC_%EB%8B%A4%EC%B9%B4%EC%8B%9C" title="하라 다카시">하라 다카시</a>를 잃고 혼란스러워 하는 정우회를 다시 일으키지 못하고, 내각 내의 불통일 결과 내각은 반년 만에 무너졌다.</p>
<h3><span class="mw-headline" id=".EC.A0.95.EC.9A.B0.ED.9A.8C_.ED.98.B8.ED.97.8C.EC.9A.B4.EB.8F.99">정우회 호헌운동</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%8B%A4%EC%B9%B4%ED%95%98%EC%8B%9C_%EA%B3%A0%EB%A0%88%ED%82%A4%EC%9A%94&amp;action=edit&amp;section=5" title="부분 편집: 정우회 호헌운동">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>다카하시 내각이 무너진 이후, 다카하시가 인솔하는 정우회는 호헌 운동을 일으켜 <a href="/wiki/%EA%B8%B0%EC%9A%94%EC%9A%B0%EB%9D%BC_%EA%B2%8C%EC%9D%B4%EA%B3%A0" title="기요우라 게이고">기요우라 게이고</a> 내각의 타도에 성공하였다. 새롭게 총리대신에 오른 헌정회 초재 <a href="/wiki/%EA%B0%80%ED%86%A0_%EB%8B%A4%EC%B9%B4%EC%95%84%ED%82%A4" title="가토 다카아키">가토 다카아키</a>는 다카하시 고레키요를 <a class="extiw" href="https://ja.wikipedia.org/wiki/%E8%BE%B2%E5%95%86%E5%8B%99%E7%9C%81_(%E6%97%A5%E6%9C%AC)" title="ja:農商務省 (日本)">농상무성</a>에 임명하였다.</p>
<h3><span class="mw-headline" id=".EC.87.BC.EC.99.80_.EA.B3.B5.ED.99.A9">쇼와 공황</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%8B%A4%EC%B9%B4%ED%95%98%EC%8B%9C_%EA%B3%A0%EB%A0%88%ED%82%A4%EC%9A%94&amp;action=edit&amp;section=6" title="부분 편집: 쇼와 공황">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>그 후에 다카하시는 정우회 총재를 <a href="/wiki/%EB%8B%A4%EB%82%98%EC%B9%B4_%EA%B8%B0%EC%9D%B4%EC%B9%98" title="다나카 기이치">다나카 기이치</a>에게 양보해 일본 정계를 은퇴했지만, <a href="/wiki/1927%EB%85%84" title="1927년">1927년</a> <a class="extiw" href="https://ja.wikipedia.org/wiki/%E6%98%AD%E5%92%8C%E9%87%91%E8%9E%8D%E6%81%90%E6%85%8C" title="ja:昭和金融恐慌">쇼와 금융 공황</a>의 발생으로 무너진 제1차 <a href="/wiki/%EC%99%80%EC%B9%B4%EC%93%B0%ED%82%A4_%EB%A0%88%EC%9D%B4%EC%A7%80%EB%A1%9C" title="와카쓰키 레이지로">와카쓰키 레이지로</a> 내각을 대신해 내각을 조직한 <a href="/wiki/%EB%8B%A4%EB%82%98%EC%B9%B4_%EA%B8%B0%EC%9D%B4%EC%B9%98" title="다나카 기이치">다나카 기이치</a>의 청으로 대장 대신에 올랐다. 다카하시는 <a href="/wiki/%EC%9D%BC%EB%B3%B8%EC%9D%80%ED%96%89" title="일본은행">일본 은행</a> 총재와 협력해 <a href="/wiki/%EB%AA%A8%EB%9D%BC%ED%86%A0%EB%A6%AC%EC%97%84" title="모라토리엄">지급 유예</a> 조치(모라토리엄, Moratorium) 을 실시함과 동시에 제반 조치를 취해 쇼와 공황을 극복하였다.</p>
<h3><span class="mw-headline" id=".EB.8C.80.EC.9E.A5_.EB.8C.80.EC.8B.A0">대장 대신</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%8B%A4%EC%B9%B4%ED%95%98%EC%8B%9C_%EA%B3%A0%EB%A0%88%ED%82%A4%EC%9A%94&amp;action=edit&amp;section=7" title="부분 편집: 대장 대신">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>정우회 총재 <a href="/wiki/%EC%9D%B4%EB%88%84%EC%B9%B4%EC%9D%B4_%EC%93%B0%EC%9A%94%EC%8B%9C" title="이누카이 쓰요시">이누카이 쓰요시</a>가 내각을 조성할 때도 이누카이의 요구로 대장대신에 올라 재정 문제 처리에 뛰어난 실력을 발휘하였다. 또, <a href="/wiki/5%C2%B715_%EC%82%AC%EA%B1%B4" title="5·15 사건">5·15 사건</a>으로 이누카이가 암살됐을 때에는 총리대신을 임시 겸임하였다. 계속해서 <a href="/wiki/%EC%82%AC%EC%9D%B4%ED%86%A0_%EB%A7%88%EC%BD%94%ED%86%A0" title="사이토 마코토">사이토</a> 내각과 <a href="/wiki/%EC%98%A4%EC%B9%B4%EB%8B%A4_%EA%B2%8C%EC%9D%B4%EC%8A%A4%EC%BC%80" title="오카다 게이스케">오카다</a> 내각에서도 대장 대신으로 일했다.</p>
<h3><span class="mw-headline" id=".EC.95.94.EC.82.B4">암살</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%8B%A4%EC%B9%B4%ED%95%98%EC%8B%9C_%EA%B3%A0%EB%A0%88%ED%82%A4%EC%9A%94&amp;action=edit&amp;section=8" title="부분 편집: 암살">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>그러나 <a href="/wiki/%EC%9D%B8%ED%94%8C%EB%A0%88%EC%9D%B4%EC%85%98" title="인플레이션">인플레이션</a>을 억제하기 위해 군사 예산을 축소하려고 했던 것이 일본 군부와의 충돌을 불렀고, <a href="/wiki/2%C2%B726_%EC%82%AC%EA%B1%B4" title="2·26 사건">2·26 사건</a> 당시 <a class="extiw" href="https://ja.wikipedia.org/wiki/%E8%B5%A4%E5%9D%82_(%E6%9D%B1%E4%BA%AC%E9%83%BD%E6%B8%AF%E5%8C%BA)" title="ja:赤坂 (東京都港区)">아카사카</a>의 자택 2층에서 청년 장교들에게 암살되었다.</p>
