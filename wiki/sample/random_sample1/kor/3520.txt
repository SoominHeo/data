<p><b>다케다 노부토라</b> (<span style="font-size: smaller;"><a href="/wiki/%EC%9D%BC%EB%B3%B8%EC%96%B4" title="일본어">일본어</a>:</span> <span lang="ja"><span lang="ja"><ruby><rb>武田信虎</rb> <rp>(</rp><rt>たけだ のぶとら</rt><rp>)</rp></ruby></span></span> <small>타케다 노부토라<sup>[<a href="/wiki/%EC%9C%84%ED%82%A4%EB%B0%B1%EA%B3%BC:%EC%9D%BC%EB%B3%B8%EC%96%B4%EC%9D%98_%ED%95%9C%EA%B8%80_%ED%91%9C%EA%B8%B0" title="위키백과:일본어의 한글 표기">*</a>]</sup></small>, <a href="/wiki/1494%EB%85%84" title="1494년">1494년</a> <a href="/wiki/2%EC%9B%94_12%EC%9D%BC" title="2월 12일">2월 12일</a> ~ <a href="/wiki/1574%EB%85%84" title="1574년">1574년</a> <a href="/wiki/3%EC%9B%94_27%EC%9D%BC" title="3월 27일">3월 27일</a>)는 <a href="/wiki/%EA%B0%80%EC%9D%B4_%EA%B5%AD" title="가이 국">가이</a>의 <a class="mw-redirect" href="/wiki/%EC%84%BC%EA%B3%A0%EC%BF%A0_%EB%8B%A4%EC%9D%B4%EB%AC%98" title="센고쿠 다이묘">센고쿠 다이묘</a> 이며 <a href="/wiki/%EB%8B%A4%EC%BC%80%EB%8B%A4_%EC%8B%A0%EA%B2%90" title="다케다 신겐">다케다 신겐</a>(武田信玄)의 아버지이다. 가이 겐지 다케다 가문의 종가이며, 다케다 가문 제18대 당주이다.</p>

<h2><span class="mw-headline" id=".EC.83.9D.EC.95.A0">생애</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%8B%A4%EC%BC%80%EB%8B%A4_%EB%85%B8%EB%B6%80%ED%86%A0%EB%9D%BC&amp;action=edit&amp;section=1" title="부분 편집: 생애">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<h3><span class="mw-headline" id=".EA.B0.80.EB.8F.85.EC.83.81.EC.86.8D.EA.B3.BC_.EA.B0.80.EC.9D.B4_.EA.B5.AD_.ED.86.B5.EC.9D.BC">가독상속과 가이 국 통일</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%8B%A4%EC%BC%80%EB%8B%A4_%EB%85%B8%EB%B6%80%ED%86%A0%EB%9D%BC&amp;action=edit&amp;section=2" title="부분 편집: 가독상속과 가이 국 통일">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p><a href="/wiki/%EB%A9%94%EC%9D%B4%EC%98%A4" title="메이오">메이오</a> 3년 (<a href="/wiki/1494%EB%85%84" title="1494년">1494년</a>) <a href="/wiki/1%EC%9B%94_6%EC%9D%BC" title="1월 6일">1월 6일</a> 다케다 가문 제17대 당주 <a class="new" href="/w/index.php?title=%EB%8B%A4%EC%BC%80%EB%8B%A4_%EB%85%B8%EB%B6%80%EC%93%B0%EB%82%98&amp;action=edit&amp;redlink=1" title="다케다 노부쓰나 (없는 문서)">다케다 노부쓰나</a>(武田信縄)의 적자로 태어났다. 에이쇼 4년 (<a href="/wiki/1507%EB%85%84" title="1507년">1507년</a>) 아버지가 병으로 죽자 14세의 나이로 가독을 상속받는다. 에이쇼 5년 (<a href="/wiki/1508%EB%85%84" title="1508년">1508년</a>) 가문내에 대립하던 숙부 <a class="mw-disambig" href="/wiki/%EB%8B%A4%EC%BC%80%EB%8B%A4_%EB%85%B8%EB%B6%80%EC%8B%9C%EA%B2%8C" title="다케다 노부시게">다케다 노부시게</a>(武田信恵)를 보네 전투(坊峰合戦 현재 <a href="/wiki/%ED%9B%84%EC%97%90%ED%9B%84%ED%82%A4_%EC%8B%9C" title="후에후키 시">후에후키 시</a>)에서 토벌 멸문시킨다. 에이쇼 7년 (<a href="/wiki/1510%EB%85%84" title="1510년">1510년</a>) 가이 동부의 <a class="new" href="/w/index.php?title=%EA%B5%B0%EB%82%98%EC%9D%B4_%EC%A7%80%EB%B0%A9&amp;action=edit&amp;redlink=1" title="군나이 지방 (없는 문서)">군나이 지방</a>(郡内地方)에서 세력을 과시하던 오야마다 가문을 압박해, 자신의 동생을 <a class="new" href="/w/index.php?title=%EC%98%A4%EC%95%BC%EB%A7%88%EB%8B%A4_%EB%85%B8%EB%B6%80%EC%95%84%EB%A6%AC&amp;action=edit&amp;redlink=1" title="오야마다 노부아리 (없는 문서)">오야마다 노부아리</a>(小山田信有)에게 시집을 보내 오야마다 가문과 동맹을 맺는다. 군나이에 인접지역인 가쓰누마 지역은 동생인 <a class="new" href="/w/index.php?title=%EA%B0%80%EC%93%B0%EB%88%84%EB%A7%88_%EB%85%B8%EB%B6%80%ED%86%A0%EB%AA%A8&amp;action=edit&amp;redlink=1" title="가쓰누마 노부토모 (없는 문서)">가쓰누마 노부토모</a>(勝沼信友)에게 맡긴다. 에이쇼 12년 (<a href="/wiki/1515%EB%85%84" title="1515년">1515년</a>) <a class="mw-redirect" href="/wiki/%EC%8A%A4%EB%A3%A8%EA%B0%80%EB%85%B8%EC%BF%A0%EB%8B%88" title="스루가노쿠니">스루가</a> 이마가와 가문을 지지한 군나이 서부 지역의 호족 오이 가문 (현재 <a href="/wiki/%EB%AF%B8%EB%82%98%EB%AF%B8%EC%95%8C%ED%94%84%EC%8A%A4_%EC%8B%9C" title="미나미알프스 시">미나미알프스 시</a>)을 공격하고, 에이쇼 14년 (<a href="/wiki/1517%EB%85%84" title="1517년">1517년</a>)에는 잠시나마 이마가와 가문과 화친했다. 에이쇼 17년 (<a href="/wiki/1520%EB%85%84" title="1520년">1520년</a>) <a class="new" href="/w/index.php?title=%EC%98%A4%EC%9D%B4_%EC%94%A8&amp;action=edit&amp;redlink=1" title="오이 씨 (없는 문서)">오이 가문</a>을 압박해 <a class="new" href="/w/index.php?title=%EC%98%A4%EC%9D%B4_%EB%85%B8%EB%B6%80%EC%82%AC%ED%86%A0&amp;action=edit&amp;redlink=1" title="오이 노부사토 (없는 문서)">오이 노부사토</a>(大井信達)의 딸(<a class="new" href="/w/index.php?title=%EC%98%A4%EC%9D%B4%EB%85%B8%EA%B0%80%ED%83%80&amp;action=edit&amp;redlink=1" title="오이노가타 (없는 문서)">오이노가타</a>)을 아내로 맞이해 혼인동맹을 맺는다.</p>
<p>또, 에이쇼 16년 (<a href="/wiki/1519%EB%85%84" title="1519년">1519년</a>) 이 때까지 다케다 가문의 연고 였던 이사와로부터 서쪽에 있는 <a href="/wiki/%EA%B3%A0%ED%9B%84_%EC%8B%9C" title="고후 시">고후</a>로 거처를 옮긴다. 그리고, 여기에 <a class="new" href="/w/index.php?title=%EC%93%B0%EC%93%B0%EC%A7%80%EA%B0%80%EC%82%AC%ED%82%A4%EC%95%BC%EC%B9%B4%ED%83%80&amp;action=edit&amp;redlink=1" title="쓰쓰지가사키야카타 (없는 문서)">쓰쓰지가사키야카타</a>를 짓고, <a href="/wiki/%EC%A1%B0%EC%B9%B4%EB%A7%88%EC%B9%98" title="조카마치">성하 마을</a>을 정비해 가신들을 이주시킨다.</p>
<p>그 후에도 호족 이마이 가문과 시나노의 스와 가문의 다툼에 참견했다. 다이에 원년 (<a href="/wiki/1521%EB%85%84" title="1521년">1521년</a>) 스루가의 <a class="new" href="/w/index.php?title=%EB%8B%A4%EC%B9%B4%EB%8D%B4%EC%A7%84_%EC%84%B1&amp;action=edit&amp;redlink=1" title="다카덴진 성 (없는 문서)">다카덴진 성</a>주 <a class="new" href="/w/index.php?title=%EA%B5%AC%EC%8B%9C%EB%A7%88_%EB%A7%88%EC%82%AC%EC%8B%9C%EA%B2%8C&amp;action=edit&amp;redlink=1" title="구시마 마사시게 (없는 문서)">구시마 마사시게</a>(福島正成)를 본대로 한 이마가와군이 <a href="/wiki/%ED%9B%84%EC%A7%80_%EA%B0%95" title="후지 강">후지 강</a>과 접해있는 군나이 서부 지역까지 쳐들어와 고후에 이르자, 노부토라는 쓰쓰지가사키야카타의 북동쪽에 있는 <a class="new" href="/w/index.php?title=%EC%9A%94%EC%B9%B4%EC%9D%B4_%EC%82%B0%EC%84%B1&amp;action=edit&amp;redlink=1" title="요카이 산성 (없는 문서)">요카이 산성</a>으로 퇴각해 일전을 준비한다. 이다가와라 전투(飯田河原合戦)와 가미조가와라 전투(上条河原合戦)에서 이마가와 군을 격퇴하고, 적장 구시마 마사시게는 전사한다. <a class="new" href="/w/index.php?title=%ED%95%98%EB%9D%BC_%EB%8F%84%EB%9D%BC%ED%83%80%EB%84%A4&amp;action=edit&amp;redlink=1" title="하라 도라타네 (없는 문서)">하라 도라타네</a>(原虎胤)가 수급을 베었다고 하나 이설도 많다. 그 와중에 요카이 산성에서 적자 <a href="/wiki/%EB%8B%A4%EC%BC%80%EB%8B%A4_%EC%8B%A0%EA%B2%90" title="다케다 신겐">다케다 하루노부</a>가 태어났다.</p>
<p>다이에 4년 (<a href="/wiki/1524%EB%85%84" title="1524년">1524년</a>) <a class="mw-redirect" href="/wiki/%EA%B0%84%ED%86%A0" title="간토">간토</a>에 있어서 두 우에스기 가문과 신흥 세력인 고호조 가문의 다툼에 개입하고, <a class="mw-redirect" href="/wiki/%EC%82%AC%EA%B0%80%EB%AF%B8%EB%85%B8%EC%BF%A0%EB%8B%88" title="사가미노쿠니">사가미</a> 국경에서 호조군과 싸운다. 이 해부터 스루가의 이마가와 가문과 사가미의 호조 가문, 시나노의 스와 가문 등 주변 여러 다이묘와 간헐적인 교전이 오갔다. 그러나, 이 같은 전투에도 좀처럼 세력을 확장할 수 없었다고 한다.</p>
<p>교로쿠 원년 (<a href="/wiki/1528%EB%85%84" title="1528년">1528년</a>) <a class="mw-redirect" href="/wiki/%EC%8A%A4%EC%99%80%EB%85%B8%EC%BF%A0%EB%8B%88" title="스와노쿠니">스와</a>로 공격해 가지만, 고베=사카이가와 전투(神戸・堺川合戦)에서 패배한다. 스와 가문은 반 노부토라 세력인 이마이 가문 등과 연합해 적대하지만, 교로쿠 4년 (<a href="/wiki/1531%EB%85%84" title="1531년">1531년</a>) 가와라베 전투(河原部合戦 현재 <a class="new" href="/w/index.php?title=%EB%8B%88%EB%9D%BC%EC%9E%90%ED%82%A4_%EC%8B%9C&amp;action=edit&amp;redlink=1" title="니라자키 시 (없는 문서)">니라자키 시</a>)에서 호족 연합을 격파했다. 또, 교로쿠 3년 (<a href="/wiki/1530%EB%85%84" title="1530년">1530년</a>) <a class="new" href="/w/index.php?title=%EC%9A%B0%EC%97%90%EC%8A%A4%EA%B8%B0_%EB%8F%84%EB%AA%A8%EC%98%A4%ED%82%A4&amp;action=edit&amp;redlink=1" title="우에스기 도모오키 (없는 문서)">우에스기 도모오키</a>(上杉朝興)의 알선으로 <a class="new" href="/w/index.php?title=%EC%9A%B0%EC%97%90%EC%8A%A4%EA%B8%B0_%EB%85%B8%EB%A6%AC%ED%9B%84%EC%82%AC&amp;action=edit&amp;redlink=1" title="우에스기 노리후사 (없는 문서)">우에스기 노리후사</a>(上杉憲房)의 부인을 측실로 맞이한다.</p>
<p>덴분 4년 (<a href="/wiki/1535%EB%85%84" title="1535년">1535년</a>) 이마가와 가문을 공격하러 국경 만자와에서 전투를 치르지만, 이마가와 가문과 친척관계로 있는 호조 가문이 <a class="new" href="/w/index.php?title=%EA%B0%80%EA%B3%A0%EC%82%AC%EC%B9%B4_%EA%B3%A0%EA%B0%9C&amp;action=edit&amp;redlink=1" title="가고사카 고개 (없는 문서)">가고사카 고개</a>를 넘어 야마나카로 공격해와 오야마다 가문과 가쓰누마 가문이 대항해 보지만 패배한다.</p>
<p>이 같이 <a class="mw-redirect" href="/wiki/%EB%AC%B4%EC%82%AC%EC%8B%9C%EB%85%B8%EC%BF%A0%EB%8B%88" title="무사시노쿠니">무사시</a>의 오기가야쓰 우에스기 가문을 제외한 주변 세력과 적대했었지만, 덴분 5년 <a class="new" href="/w/index.php?title=%EC%9D%B4%EB%A7%88%EA%B0%80%EC%99%80_%EC%9A%B0%EC%A7%80%ED%85%8C%EB%A3%A8&amp;action=edit&amp;redlink=1" title="이마가와 우지테루 (없는 문서)">이마가와 우지테루</a>(今川氏輝) 사후 발발한 이마가와 가문의 내분(<a class="new" href="/w/index.php?title=%ED%95%98%EB%82%98%EC%BF%A0%EB%9D%BC%EC%9D%98_%EB%82%9C&amp;action=edit&amp;redlink=1" title="하나쿠라의 난 (없는 문서)">하나쿠라의 난</a>)에서 <a href="/wiki/%EC%9D%B4%EB%A7%88%EA%B0%80%EC%99%80_%EC%9A%94%EC%8B%9C%EB%AA%A8%ED%86%A0" title="이마가와 요시모토">이마가와 요시모토</a>(今川義元)를 지지해 이마가와 가문과 관계를 호전시킨다. 덴분 6년 (<a href="/wiki/1537%EB%85%84" title="1537년">1537년</a>) 장녀를 <a href="/wiki/%EC%9D%B4%EB%A7%88%EA%B0%80%EC%99%80_%EC%9A%94%EC%8B%9C%EB%AA%A8%ED%86%A0" title="이마가와 요시모토">이마가와 요시모토</a>에게 시집보내고, 적자 하루노부는 이마가와 가문의 중매로 <a class="mw-redirect" href="/wiki/%EA%B5%AC%EA%B2%8C" title="구게">구게</a> 산조 가문의 딸을 며느리로 맞이한다. 게다가 호조 가문과도 화친하였다. 덴분 9년 (<a href="/wiki/1540%EB%85%84" title="1540년">1540년</a>) <a class="new" href="/w/index.php?title=%EC%9D%B4%EB%A7%88%EC%9D%B4_%EB%85%B8%EB%B6%80%EB%AA%A8%ED%86%A0&amp;action=edit&amp;redlink=1" title="이마이 노부모토 (없는 문서)">이마이 노부모토</a>를 우라 성에서 굴복시겼고, 스와 가문과는 셋째 딸 네네를 <a class="mw-disambig" href="/wiki/%EC%8A%A4%EC%99%80_%EC%9A%94%EB%A6%AC%EC%8B%9C%EA%B2%8C" title="스와 요리시게">스와 요리시게</a>(諏訪頼重)에게 시집을 보내 스와 가문과 화친한다.</p>
<p>덴분 10년 (<a href="/wiki/1541%EB%85%84" title="1541년">1541년</a>) 스와 요리시게와 <a href="/wiki/%EB%AC%B4%EB%9D%BC%EC%B9%B4%EB%AF%B8_%EC%9A%94%EC%8B%9C%ED%82%A4%EC%9A%94" title="무라카미 요시키요">무라카미 요시키요</a>(村上義清) 등 시나노 호족등과 연합해 사쿠 군(현재 <a href="/wiki/%EC%82%AC%EC%BF%A0_%EC%8B%9C" title="사쿠 시">사쿠 시</a>)에서 시게노 일족(운노 가문, 모치즈키 가문, 사나다 가문 등)과 격돌, 이들을 시나노에서 축출한다. (<a class="new" href="/w/index.php?title=%EC%9A%B4%EB%85%B8%ED%83%80%EC%9D%B4%EB%9D%BC_%EC%A0%84%ED%88%AC&amp;action=edit&amp;redlink=1" title="운노타이라 전투 (없는 문서)">운노타이라 전투</a> (海野平合戦)) 그 후, 세력을 시나노 동부인 사쿠 군과 지사가타 군까지 세력을 뻗친다.</p>
<h3><span class="mw-headline" id=".EC.B6.94.EB.B0.A9">추방</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%8B%A4%EC%BC%80%EB%8B%A4_%EB%85%B8%EB%B6%80%ED%86%A0%EB%9D%BC&amp;action=edit&amp;section=3" title="부분 편집: 추방">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>덴분 10년 (<a href="/wiki/1541%EB%85%84" title="1541년">1541년</a>) <a href="/wiki/6%EC%9B%94_14%EC%9D%BC" title="6월 14일">6월 14일</a> 노부토라는 시나노로부터 개선하여 사위 <a href="/wiki/%EC%9D%B4%EB%A7%88%EA%B0%80%EC%99%80_%EC%9A%94%EC%8B%9C%EB%AA%A8%ED%86%A0" title="이마가와 요시모토">이마가와 요시모토</a>를 만나러 <a class="new" href="/w/index.php?title=%EC%8A%A4%EB%A3%A8%EA%B0%80&amp;action=edit&amp;redlink=1" title="스루가 (없는 문서)">스루가</a>로 향한다. 그러나, <a href="/wiki/%EC%9D%B4%ED%83%80%EA%B0%80%ED%82%A4_%EB%85%B8%EB%B6%80%EC%B9%B4%ED%83%80" title="이타가키 노부카타">이타가키 노부카타</a>(板垣信方), <a href="/wiki/%EC%95%84%EB%A7%88%EB%A6%AC_%EB%8F%84%EB%9D%BC%EC%95%BC%EC%8A%A4" title="아마리 도라야스">아마리 도라야스</a>(甘利虎泰) 등 후다이 가신들로부터 지지를 받은 적자 하루노부 일파에 의해 가이로 돌아갈수 없었다. 노부토라를 추방한 하루노부는 다케다 가문의 가독과 슈고직을 계승한다.</p>
<p>사건의 경위는 《<a class="new" href="/w/index.php?title=%EA%B0%80%EC%93%B0%EC%95%BC%EB%A7%88%ED%82%A4&amp;action=edit&amp;redlink=1" title="가쓰야마키 (없는 문서)">가쓰야마키</a>(勝山記)》, 《<a class="new" href="/w/index.php?title=%EA%B3%A0%ED%95%98%EC%BF%A0%EC%82%AC%EC%9D%B4%ED%82%A4&amp;action=edit&amp;redlink=1" title="고하쿠사이키 (없는 문서)">고하쿠사이키</a>(高白斎記)》 등에서 볼 수 있다. 사건의 배경에 대해서는 여러 설이 있다. 노부토라가 적자 하루노부를 홀대하고 차남 노부시게를 편애해, 언젠간 폐적이 될지 모른다고 생각한 하루노부와의 불화설과, 《<a class="new" href="/w/index.php?title=%EA%B3%A0%EC%9A%94%EA%B5%B0%EC%B9%B8&amp;action=edit&amp;redlink=1" title="고요군칸 (없는 문서)">고요군칸</a>(甲陽軍鑑)》에서는 하루노부와 중신, 그리고, <a href="/wiki/%EC%9D%B4%EB%A7%88%EA%B0%80%EC%99%80_%EC%9A%94%EC%8B%9C%EB%AA%A8%ED%86%A0" title="이마가와 요시모토">이마가와 요시모토</a>가 공모했다는 설이 있다. 또, 《가쓰야마키(勝山記)》 등에 따르면, 노부토라의 치세는 여러번에 걸친 정벌사업에 따른 군자금 확보를 위해 농민과 호족들에 과중한 세액을 부담시켰고, 이 결과 백성들의 원성이 하늘을 찔렀다고 한다. 그래서, 노부토라의 추방은 영민들도 환영했다고 한다.</p>
<h3><span class="mw-headline" id=".EC.9C.A0.EB.9E.91">유랑</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%8B%A4%EC%BC%80%EB%8B%A4_%EB%85%B8%EB%B6%80%ED%86%A0%EB%9D%BC&amp;action=edit&amp;section=4" title="부분 편집: 유랑">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>그 후, <a href="/wiki/%EC%9D%B4%EB%A7%88%EA%B0%80%EC%99%80_%EC%9A%94%EC%8B%9C%EB%AA%A8%ED%86%A0" title="이마가와 요시모토">이마가와 요시모토</a>의 비호 아래 스루가에서 한적한 생활을 보내고 있었다. 그러나, 에이로쿠 3년 (<a href="/wiki/1560%EB%85%84" title="1560년">1560년</a>) <a href="/wiki/%EC%98%A4%EC%BC%80%ED%95%98%EC%9E%90%EB%A7%88_%EC%A0%84%ED%88%AC" title="오케하자마 전투">오케하자마 전투</a>(桶狭間の戦い)에서 요시모토가 이 전투에서 전사하자, 그 뒤를 이은 <a href="/wiki/%EC%9D%B4%EB%A7%88%EA%B0%80%EC%99%80_%EC%9A%B0%EC%A7%80%EC%9E%90%EB%84%A4" title="이마가와 우지자네">이마가와 우지자네</a>(今川氏真)와 관계가 틀어졌다. 일설에는 우지자네를 폐적시키고 자신이 이마가와 가문을 계승할 계획이었다고 한다. 이 때문에, 노부토라는 스루가를 떠났다. 이 시기 신겐은 이마가와 가문의 가운이 얼마 남지 않았음을 직감했다고 한다.</p>
<p>노부토라가 스루가를 떠난 후, <a class="mw-redirect" href="/wiki/%EC%9D%B4%EC%84%B8%EB%85%B8%EC%BF%A0%EB%8B%88" title="이세노쿠니">이세</a>로 갔지만, 당시 이곳에는 이세 <a class="mw-redirect" href="/wiki/%EA%B3%A0%EC%BF%A0%EC%8B%9C" title="고쿠시">고쿠시</a> <a class="new" href="/w/index.php?title=%EA%B8%B0%ED%83%80%EB%B0%94%ED%83%80%EC%BC%80_%EB%8F%84%EB%AA%A8%EB%85%B8%EB%A6%AC&amp;action=edit&amp;redlink=1" title="기타바타케 도모노리 (없는 문서)">기타바타케 도모노리</a>(北畠具教)의 후원을 받은 <a class="mw-redirect" href="/wiki/%EC%8B%9C%EB%A7%88%EB%85%B8%EC%BF%A0%EB%8B%88" title="시마노쿠니">시마</a>의 관료와 <a class="new" href="/w/index.php?title=%EB%8F%84%EB%B0%94_%EC%84%B1&amp;action=edit&amp;redlink=1" title="도바 성 (없는 문서)">도바 성</a>주 <a class="mw-disambig" href="/wiki/%EA%B5%AC%ED%82%A4_%EC%9A%94%EC%8B%9C%ED%83%80%EC%B9%B4" title="구키 요시타카">구키 요시타카</a>(九鬼嘉隆)가 시마의 패권을 둘러싸고 다투고 있었다. 이때 노부토라는 기타바타케 도모노리의 부탁을 받고 시마의 관료들 편에서 참모로써 참전하게 된다. 그의 명성은 주변국에 익히 잘 알려졌기 때문에 노부토라는 <a class="mw-disambig" href="/wiki/%EA%B5%AC%ED%82%A4_%EC%9A%94%EC%8B%9C%ED%83%80%EC%B9%B4" title="구키 요시타카">구키 요시타카</a>를 격파하고 시마의 관료측의 승리에 공헌한다. 이때 패한 <a class="mw-disambig" href="/wiki/%EA%B5%AC%ED%82%A4_%EC%9A%94%EC%8B%9C%ED%83%80%EC%B9%B4" title="구키 요시타카">구키 요시타카</a>는 <a class="mw-redirect" href="/wiki/%EC%98%A4%EC%99%80%EB%A6%AC%EB%85%B8%EC%BF%A0%EB%8B%88" title="오와리노쿠니">오와리</a>의 <a href="/wiki/%EC%98%A4%EB%8B%A4_%EB%85%B8%EB%B6%80%EB%82%98%EA%B0%80" title="오다 노부나가">오다 노부나가</a>(織田信長)를 의지해 그의 가신이 된다.</p>
<p>그 후, 조정과 관계가 깊었던 기타바타케 도모노리의 후원으로 <a href="/wiki/%EA%B5%90%ED%86%A0_%EC%8B%9C" title="교토 시">교토</a>로 올라가 사돈인 <a class="new" href="/w/index.php?title=%EC%82%B0%EC%A1%B0_%EC%82%AC%EB%84%A4%EC%93%B0%EB%82%98&amp;action=edit&amp;redlink=1" title="산조 사네쓰나 (없는 문서)">산조 사네쓰나</a>(三条実綱)의 비호를 받으며 교토 고조(五条)에 살았다. 이때 유력인사와 친분을 쌓았다. 또, 신겐도 노부토라의 생활비 등을 송금했다고 전해진다.</p>
<h3><span class="mw-headline" id=".EB.A7.90.EB.85.84">말년</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%8B%A4%EC%BC%80%EB%8B%A4_%EB%85%B8%EB%B6%80%ED%86%A0%EB%9D%BC&amp;action=edit&amp;section=5" title="부분 편집: 말년">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>겐나 4년 (<a href="/wiki/1573%EB%85%84" title="1573년">1573년</a>) 신겐이 병사하자, 손자 <a href="/wiki/%EB%8B%A4%EC%BC%80%EB%8B%A4_%EA%B0%80%EC%93%B0%EC%9A%94%EB%A6%AC" title="다케다 가쓰요리">다케다 가쓰요리</a>(武田勝頼)에 의탁하여 다케다령으로 돌아간다. 셋째 아들 <a class="new" href="/w/index.php?title=%EB%8B%A4%EC%BC%80%EB%8B%A4_%EB%85%B8%EB%B6%80%EC%B9%B4%EB%8F%84&amp;action=edit&amp;redlink=1" title="다케다 노부카도 (없는 문서)">다케다 노부카도</a>(武田信廉)의 거성 <a class="new" href="/w/index.php?title=%EB%8B%A4%EC%B9%B4%ED%86%A0_%EC%84%B1&amp;action=edit&amp;redlink=1" title="다카토 성 (없는 문서)">다카토 성</a>에서 생활한다. 덴쇼 2년 (<a href="/wiki/1574%EB%85%84" title="1574년">1574년</a>) <a href="/wiki/3%EC%9B%94_5%EC%9D%BC" title="3월 5일">3월 5일</a>, 이나의 사위 禰津神平의 비호를 받으며 시나노 다카토 성에서 생을 마감한다. 향년 81세. 장례는 노부토라가 창건한 고후 <a class="new" href="/w/index.php?title=%EB%8B%A4%EC%9D%B4%EC%84%BC%EC%A7%80&amp;action=edit&amp;redlink=1" title="다이센지 (없는 문서)">다이센지</a>에서 거행되었다.</p>
<p>일설에는 가쓰요리와 대면할 때에 노부토라가 나란히 앉아 있는 군신들 앞에서 칼을 뽑고, 80세라고는 믿기지 않을 정도로 능숙하게 휘둘렀다고 한다. 그리고, 가쓰요리 앞에서 가이로의 귀환을 부탁했다고 한다. 그러나, 노부토라를 잘 아는 노신들은 경악을 하며 완강히 반대했다고 한다. 가쓰요리도 쓸데없는 혼란을 피하기 위해 숙부 노부카도에게 할아버지 노부토라를 보살필 것을 부탁했다고 전해진다.</p>
<h2><span class="mw-headline" id=".EC.9D.B8.EB.AC.BC">인물</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%8B%A4%EC%BC%80%EB%8B%A4_%EB%85%B8%EB%B6%80%ED%86%A0%EB%9D%BC&amp;action=edit&amp;section=6" title="부분 편집: 인물">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<ul>
<li>적은 물론 모반을 일으킨 사람까지도 항복만 하면 용서했다고 한다. 노부토라 통치 초기에는 인재가 부족했기 때문에 죽이려고 해도 죽일 수 없었다는 견해가 있지만, 폭군으로 알려진 노부토라의 다른 일면을 나타낸 일화로 이상하진 않다.</li>
<li>전술가로서는 탁월했고, 싸움에도 능숙해 <a href="/wiki/%EC%98%A4%EB%8B%A4_%EB%85%B8%EB%B6%80%ED%9E%88%EB%8D%B0" title="오다 노부히데">오다 노부히데</a>(織田信秀)나 <a href="/wiki/%EC%82%AC%EC%9D%B4%ED%86%A0_%EB%8F%84%EC%82%B0" title="사이토 도산">사이토 도산</a>(斎藤道三)과 비교해도 손색이 없다. 특히 다이에 2년에 이마가와 가문의 무장 <a class="new" href="/w/index.php?title=%EA%B5%AC%EC%8B%9C%EB%A7%88_%EB%A7%88%EC%82%AC%EC%8B%9C%EA%B2%8C&amp;action=edit&amp;redlink=1" title="구시마 마사시게 (없는 문서)">구시마 마사시게</a>가 1만 5천의 병력을 이끌고 가이로 침공할 때 노부토라의 병력은 2천이었지만, 본대를 기습하여 격파하고 적장 마사시게를 죽이는 등 대승을 거두었다. 이 전투는 규모와, 경위도 <a href="/wiki/%EC%98%A4%EC%BC%80%ED%95%98%EC%9E%90%EB%A7%88_%EC%A0%84%ED%88%AC" title="오케하자마 전투">오케하자마 전투</a>와 유사해 <a href="/wiki/%EC%98%A4%EB%8B%A4_%EB%85%B8%EB%B6%80%EB%82%98%EA%B0%80" title="오다 노부나가">오다 노부나가</a>(織田信長)가 이 전투를 본보기로 작전을 세웠다고 전해진다.</li>
<li><a href="/wiki/%EC%97%90%EB%8F%84_%EC%8B%9C%EB%8C%80" title="에도 시대">에도 시대</a>에 편찬된 《<a class="new" href="/w/index.php?title=%EA%B3%A0%EC%9A%94%EA%B5%B0%EC%B9%B8&amp;action=edit&amp;redlink=1" title="고요군칸 (없는 문서)">고요군칸</a>(甲陽軍鑑)》에 따르면, 성격이 난폭하고 교만했다고 한다. 진언한 가신을 숙청하거나, 임산부의 배를 산 채로 가르는 등 악행을 저질렀다고 한다. 그 예로 <a class="new" href="/w/index.php?title=%EB%82%98%EC%9D%B4%ED%86%A0_%EB%8F%84%EB%9D%BC%EC%8A%A4%EC%BC%80&amp;action=edit&amp;redlink=1" title="나이토 도라스케 (없는 문서)">나이토 도라스케</a>(内藤虎資), <a class="new" href="/w/index.php?title=%EB%B0%94%EB%B0%94_%EB%8F%84%EB%9D%BC%EC%82%AC%EB%8B%A4&amp;action=edit&amp;redlink=1" title="바바 도라사다 (없는 문서)">바바 도라사다</a>(馬場虎貞), <a class="new" href="/w/index.php?title=%EC%95%BC%EB%A7%88%EA%B0%80%ED%83%80_%EB%8F%84%EB%9D%BC%ED%82%A4%EC%9A%94&amp;action=edit&amp;redlink=1" title="야마가타 도라키요 (없는 문서)">야마가타 도라키요</a>(山県虎清), <a class="new" href="/w/index.php?title=%EC%97%90%ED%86%A0_%EB%8F%84%EB%9D%BC%ED%86%A0%EC%9A%94&amp;action=edit&amp;redlink=1" title="에토 도라토요 (없는 문서)">에토 도라토요</a>(工藤虎豊) 등을 숙청했다. 후에 그의 아들 신겐이 이들 가문을 부활시키게 된다. <sup class="reference" id="cite_ref-1"><a href="#cite_note-1">[1]</a></sup> 유력 가신이나 친척 척결은 <a class="mw-redirect" href="/wiki/%EC%84%BC%EA%B3%A0%EC%BF%A0_%EB%8B%A4%EC%9D%B4%EB%AC%98" title="센고쿠 다이묘">센고쿠 다이묘</a>가 권력을 집중시키기 위해 널리 행해지던 일이었다. 그 예로 <a href="/wiki/%EB%AA%A8%EB%A6%AC_%EB%AA%A8%ED%86%A0%EB%82%98%EB%A6%AC" title="모리 모토나리">모리 모토나리</a>(毛利元就), 오다 노부나가(織田信長)등을 들 수 있다. 단지 정책이 실패하여 부정적 평가가 이어지는 면이 있다.</li>
<li><a class="mw-redirect" href="/wiki/%EA%B0%80%EC%9D%B4%EB%85%B8%EC%BF%A0%EB%8B%88" title="가이노쿠니">가이</a>를 통일하고, 다케다 가문을 슈고 다이묘에서 센고쿠 다이묘로 탈피시켜 강력한 중앙집권화를 단행했다. 신겐에 의한 쿠데타는 이 중앙집권화에 반발한 호족 무리의 주도로 행해진 것이다. 다케다 가문은 호족들의 연합맹주로 존재했고, 신겐 시기에 행해진 합의제가 이를 뒷받침하는 증거다.</li>
<li>여러 지방으로부터 유능한 인재를 모아 아시카루 다이쇼(고요 5명신)로 키우는 한편, 후에 다케다 가문을 지지하는 <a class="mw-redirect" href="/wiki/%EA%B3%A0%EC%82%AC%EC%B9%B4_%EB%A7%88%EC%82%AC%EB%85%B8%EB%B6%80" title="고사카 마사노부">고사카 마사노부</a>(高坂昌信), <a href="/wiki/%EB%B0%94%EB%B0%94_%EB%85%B8%EB%B6%80%ED%95%98%EB%A3%A8" title="바바 노부하루">바바 노부하루</a>(馬場信春) 등의 부장도 노부토라가 발탁한 것이라 사료된다.</li>
<li>적극적인 외교정책을 폈다. 적자 신겐에게 구게 <a class="new" href="/w/index.php?title=%EC%82%B0%EC%A1%B0_%EA%B8%B4%EC%9A%94%EB%A6%AC&amp;action=edit&amp;redlink=1" title="산조 긴요리 (없는 문서)">산조 긴요리</a>(三条公頼)의 딸을 맞하는 등, 혼인정책을 폈고, 이를 통해 주변 세력과 동맹을 할 수 있었다. 또 이 혼인을 통해 신겐은 혼간지 <a href="/wiki/%EA%B2%90%EB%87%A8" title="겐뇨">겐뇨</a>(本願寺顕如)를 통해 종교의 힘을 끌어들일 수 있었다. 혼간지 겐뇨의 처와 신겐의 처는 자매지간이다.</li>
<li>아들인 다케다 노부카도가 그린 노부토라의 초상화가 현존해 있다. 이를 볼 때 신겐 이외의 부자관계는 좋았다고 사료된다.</li>
<li>명도 <a class="new" href="/w/index.php?title=%EC%86%8C%EC%9E%94%EC%82%AC%EB%AA%AC%EC%A7%80&amp;action=edit&amp;redlink=1" title="소잔사몬지 (없는 문서)">소잔사몬지</a>(宗三左文字)를 소유해, 이 칼을 <a href="/wiki/%EC%9D%B4%EB%A7%88%EA%B0%80%EC%99%80_%EC%9A%94%EC%8B%9C%EB%AA%A8%ED%86%A0" title="이마가와 요시모토">이마가와 요시모토</a>에게 주었다. 오케하자마 전투 후에 <a href="/wiki/%EC%98%A4%EB%8B%A4_%EB%85%B8%EB%B6%80%EB%82%98%EA%B0%80" title="오다 노부나가">오다 노부나가</a>가 가지고 있었지만, <a href="/wiki/%ED%98%BC%EB%85%B8%EC%A7%80%EC%9D%98_%EB%B3%80" title="혼노지의 변">혼노지의 변</a>에서 소실된후, <a href="/wiki/%EB%8F%84%EC%9A%94%ED%86%A0%EB%AF%B8_%ED%9E%88%EB%8D%B0%EC%9A%94%EC%8B%9C" title="도요토미 히데요시">도요토미 히데요시</a>(豊臣秀吉)가 소실된 터에서 이 칼을 회수했고 그 후, <a href="/wiki/%EB%8F%84%EC%9A%94%ED%86%A0%EB%AF%B8_%ED%9E%88%EB%8D%B0%EC%9A%94%EB%A6%AC" title="도요토미 히데요리">도요토미 히데요리</a>(豊臣秀頼)로부터 <a href="/wiki/%EB%8F%84%EC%BF%A0%EA%B0%80%EC%99%80_%EC%9D%B4%EC%97%90%EC%95%BC%EC%8A%A4" title="도쿠가와 이에야스">도쿠가와 이에야스</a>(徳川家康)가 받아 현재에 이르고 있다.</li>
</ul>
