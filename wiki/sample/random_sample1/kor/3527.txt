<h1 class="firstHeading" id="firstHeading" lang="ko">다테 요시무라</h1>
<div class="mw-body-content" id="bodyContent">
<div id="siteSub">위키백과, 우리 모두의 백과사전.</div>
<div id="contentSub"></div>
<div class="mw-jump" id="jump-to-nav">
					이동:					<a href="#mw-head">둘러보기</a>, 					<a href="#p-search">검색</a>
</div>
<div class="mw-content-ltr" dir="ltr" id="mw-content-text" lang="ko"><script>function mfTempOpenSection(id){var block=document.getElementById("mf-section-"+id);block.className+=" open-block";block.previousSibling.className+=" open-block";}</script><p><b>다테 요시무라</b>(<span style="font-size: smaller;"><a href="/wiki/%EC%9D%BC%EB%B3%B8%EC%96%B4" title="일본어">일본어</a>:</span> <span lang="ja" xml:lang="ja">伊達吉村</span>, <a href="/wiki/1680%EB%85%84" title="1680년">1680년</a> <a href="/wiki/7%EC%9B%94_23%EC%9D%BC" title="7월 23일">7월 23일</a> ~ <a href="/wiki/1752%EB%85%84" title="1752년">1752년</a> <a href="/wiki/2%EC%9B%94_8%EC%9D%BC" title="2월 8일">2월 8일</a>)는 <a href="/wiki/%EC%97%90%EB%8F%84_%EC%8B%9C%EB%8C%80" title="에도 시대">에도 시대</a>의 <a href="/wiki/%EB%8B%A4%EC%9D%B4%EB%AC%98" title="다이묘">다이묘</a>로, <a href="/wiki/%EC%84%BC%EB%8B%A4%EC%9D%B4_%EB%B2%88" title="센다이 번">센다이 번</a>의 5대 번주이다. 어릴적 이름은 스케사부로(<span lang="ja" xml:lang="ja">助三郎</span>)이며, 초명은 무라후사(<span lang="ja" xml:lang="ja">村房</span>)이다. 관위는 <a class="mw-redirect" href="/wiki/%EC%A2%85%EC%82%AC%EC%9C%84" title="종사위">종4위상</a>, 좌근위 임시 중장, <a href="/wiki/%EB%AC%B4%EC%93%B0_%EA%B5%AD" title="무쓰 국">무쓰</a>노카미이다.</p>
<p><a href="/wiki/%EC%97%94%ED%8F%AC" title="엔포">엔포</a> 8년, 2대 번주 <a href="/wiki/%EB%8B%A4%ED%85%8C_%EB%8B%A4%EB%8B%A4%EB%AC%B4%EB%84%A4" title="다테 다다무네">다테 다다무네</a>의 여덟 번째 아들인 <a class="new" href="/w/index.php?title=%EB%8B%A4%ED%85%8C_%EB%AC%B4%EB%84%A4%ED%9B%84%EC%82%AC&amp;action=edit&amp;redlink=1" title="다테 무네후사 (없는 문서)">다테 무네후사</a>의 맏아들로 태어났다. 4대 번주 <a href="/wiki/%EB%8B%A4%ED%85%8C_%EC%93%B0%EB%82%98%EB%AC%B4%EB%9D%BC" title="다테 쓰나무라">다테 쓰나무라</a>의 맏아들 오기치요마루가 일찍 사망하면서 양자가 되었다. <a href="/wiki/%EA%B2%90%EB%A1%9C%EC%BF%A0" title="겐로쿠">겐로쿠</a> 16년(<a href="/wiki/1703%EB%85%84" title="1703년">1703년</a>), 쓰나무라가 은거하면서 가문을 잇게 되었다. 선대 번주 쓰나무라가 시행한 개혁으로 인해, 요시무라의 재위기에 이르러 센다이 번은 극도의 재정적 궁핍을 겪게 되었고, 요시무라는 재정 재건을 위해 번정 개혁을 단행했다. 우선 <a href="/wiki/%EA%B5%90%ED%98%B8" title="교호">교호</a> 12년(<a href="/wiki/1727%EB%85%84" title="1727년">1727년</a>)에 막부의 허가를 얻어 <a class="new" href="/w/index.php?title=%EA%B0%84%EC%97%90%EC%9D%B4%EC%93%B0%ED%98%B8&amp;action=edit&amp;redlink=1" title="간에이쓰호 (없는 문서)">간에이쓰호</a>를 <a href="/wiki/%EC%9D%B4%EC%8B%9C%EB%85%B8%EB%A7%88%ED%82%A4_%EC%8B%9C" title="이시노마키 시">이시노마키</a>에서 주조하였고, 그것을 영내에서 유통시켜 이윤을 얻었다. 또 쌀 매입을 재편 강화해서, 농민으로부터 잉여 곡물을 강제적으로 공출시켰고, 그것을 <a href="/wiki/%EC%97%90%EB%8F%84" title="에도">에도</a>로 운반시켜 이익을 증대해 번 재정을 윤택하게 했다. 이 때문에 18세기 초부터 중엽에 걸쳐 에도 시장에 돌던 쌀의 대부분이 센다이 번의 것이었다고 할 정도였다. 또 요시무라는 책, 그림, 와카 등 문학적 방면에 소질이 있었고, 번 내에 학문소를 설치하여 학예를 장려했다. 특히 와카에 조예가 깊어 교토의 귀족들과 친교를 다졌다.</p>
<p><a href="/wiki/%EA%B0%84%ED%8F%AC" title="간포">간포</a> 3년(<a href="/wiki/1743%EB%85%84" title="1743년">1743년</a>), 넷째 아들인 <a class="mw-redirect" href="/wiki/%EB%8B%A4%ED%85%8C_%EB%AC%B4%EB%84%A4%EB%AC%B4%EB%9D%BC_(%EC%97%90%EB%8F%84)" title="다테 무네무라 (에도)">무네무라</a>에게 자리를 물려주고 은거하였으며, <a href="/wiki/%ED%98%B8%EB%A0%88%ED%82%A4" title="호레키">호레키</a> 원년 12월 24일에 72세의 나이로 사망하였다. 요시무네는 센다이 번의 재정을 재건하여, 쓰나무라와 함께 중흥의 명군으로 칭송받았으며, <a href="/wiki/%EC%87%BC%EC%99%80_%EC%8B%9C%EB%8C%80" title="쇼와 시대">쇼와</a> 3년(<a href="/wiki/1928%EB%85%84" title="1928년">1928년</a>)에는 <a href="/wiki/%EC%A2%853%EC%9C%84" title="종3위">종3위</a>로 <a href="/wiki/%EC%B6%94%EC%A6%9D" title="추증">추증</a>되었다.</p>
