<p><b>다프너 스히퍼르스</b>(<span style="font-size: smaller;"><a href="/wiki/%EB%84%A4%EB%8D%9C%EB%9E%80%EB%93%9C%EC%96%B4" title="네덜란드어">네덜란드어</a>:</span> <span lang="nl" xml:lang="nl">Dafne Schippers</span>, <a href="/wiki/1992%EB%85%84" title="1992년">1992년</a> <a href="/wiki/6%EC%9B%94_15%EC%9D%BC" title="6월 15일">6월 15일</a> ~ )는 <a href="/wiki/%EB%84%A4%EB%8D%9C%EB%9E%80%EB%93%9C" title="네덜란드">네덜란드</a>의 <a class="mw-redirect" href="/wiki/%EC%9C%A1%EC%83%81" title="육상">육상</a> 선수로 <a class="mw-redirect" href="/wiki/%EB%8B%A8%EA%B1%B0%EB%A6%AC%EB%8B%AC%EB%A6%AC%EA%B8%B0" title="단거리달리기">단거리달리기</a>와 7종 경기를 주종목으로 활약하였다. <a href="/wiki/2013%EB%85%84_%EC%84%B8%EA%B3%84_%EC%9C%A1%EC%83%81_%EC%84%A0%EC%88%98%EA%B6%8C_%EB%8C%80%ED%9A%8C" title="2013년 세계 육상 선수권 대회">2013년 세계 육상 선수권 대회</a>에서 칠종 경기 <a href="/wiki/%EB%8F%99%EB%A9%94%EB%8B%AC" title="동메달">동메달</a>을 획득하였다.</p>
<p><a href="/wiki/%EC%9C%84%ED%8A%B8%EB%A0%88%ED%9D%90%ED%8A%B8" title="위트레흐트">위트레흐트</a>에서 태어난 스히퍼르스는 9세 때 육상을 시작하였다.</p>
<p>그녀는 <a href="/wiki/2010%EB%85%84" title="2010년">2010년</a> 세계 주니어 육상 선수권과 <a href="/wiki/2011%EB%85%84" title="2011년">2011년</a> 유럽 주니어 육상 선수권에서 칠종 경기를 우승하였다. 2년 후에 유럽 23세 이하 선수권에서 100m <a href="/wiki/%EA%B8%88%EB%A9%94%EB%8B%AC" title="금메달">금메달</a>과 <a href="/wiki/%EB%A9%80%EB%A6%AC%EB%9B%B0%EA%B8%B0" title="멀리뛰기">멀리뛰기</a> 동메달을 획득하였다.</p>
<p><a href="/wiki/2011%EB%85%84" title="2011년">2011년</a> <a href="/wiki/%EB%8C%80%EA%B5%AC%EA%B4%91%EC%97%AD%EC%8B%9C" title="대구광역시">대구</a>에서 스히퍼르스는 200m에서 네덜란드 국내 기록을 깼다. 그녀는 400m 릴레이에서 합동 기록 보유자이다. 그녀는 칠종 경기의 <a href="/wiki/2014%EB%85%84" title="2014년">2014년</a> 편이 있는 동안에 괴치스에서 열린 하이포 대회에서 200m 기록을 향상시켰으며, 그녀의 시간 22.35초는 칠종 경기에서 최고의 200m 상연들 중의 하나였다.</p>
<p><a href="/wiki/2014%EB%85%84" title="2014년">2014년</a> 유럽 선수권 대회에서 스히퍼르스는 100m와 200m에서 금메달을 획득하였다. 거기서 그녀의 성공은 자신의 장기적 전망들과 자신이 단거리달리기에 전념하거나 칠종 경기에서 자신의 경력을 지속하느냐에 토론을 격려하였다.</p>
<p><a href="/wiki/2015%EB%85%84" title="2015년">2015년</a> 스히퍼르스는 <a href="/wiki/%ED%8A%B8%EC%9C%84%ED%84%B0" title="트위터">트위터</a>를 통하여 그해의 <a href="/wiki/%EC%84%B8%EA%B3%84_%EC%9C%A1%EC%83%81_%EC%84%A0%EC%88%98%EA%B6%8C_%EB%8C%80%ED%9A%8C" title="세계 육상 선수권 대회">세계 육상 선수권 대회</a>와 <a class="mw-redirect" href="/wiki/2016%EB%85%84_%EB%A6%AC%EC%9A%B0%EB%8D%B0%EC%9E%90%EB%84%A4%EC%9D%B4%EB%A3%A8_%EC%98%AC%EB%A6%BC%ED%94%BD" title="2016년 리우데자네이루 올림픽">2016년 리우데자네이루 올림픽</a>에 달하는 데 단거리달리기에 전념할 것이라고 발표하였다.</p>
<p><a class="mw-redirect" href="/wiki/%EB%B2%A0%EC%9D%B4%EC%A7%95" title="베이징">베이징</a>에서 열린 세계 육상 선수권 대회에서 그녀는 100m <a href="/wiki/%EC%9D%80%EB%A9%94%EB%8B%AC" title="은메달">은메달</a>과 200m 금메달을 땄다. 21.63초의 그녀의 200m 우승 시간은 유럽 신기록이었고, 역사상 그 거리에 그녀를 3번째로 가장 빠른 여성 선수로 만들었다.</p>
<p><a class="mw-redirect" href="/wiki/2016%EB%85%84_%EB%A6%AC%EC%9A%B0%EB%8D%B0%EC%9E%90%EB%84%A4%EC%9D%B4%EB%A3%A8_%EC%98%AC%EB%A6%BC%ED%94%BD" title="2016년 리우데자네이루 올림픽">2016년 리우데자네이루 올림픽</a>에 나간 스히퍼르스는 100m와 200m 결승전에 출전하여 각각 5위(10.90초)와 2위(21.88초)를 하였다.</p>
