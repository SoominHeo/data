<p><b>단속 평형이론</b>(斷續平衡理論)은 <a class="mw-redirect" href="/wiki/%EC%9C%A0%EC%84%B1_%EC%83%9D%EC%8B%9D" title="유성 생식">유성 생식</a>을 하는 <a href="/wiki/%EC%83%9D%EB%AC%BC" title="생물">생물</a> <a class="mw-redirect" href="/wiki/%EC%A2%85_(%EC%83%9D%EB%AC%BC)" title="종 (생물)">종</a>의 <a href="/wiki/%EC%A7%84%ED%99%94" title="진화">진화</a> 양상은 대부분의 기간 동안 큰 변화 없는 안정기와 비교적 짧은 시간에 급속한 <a href="/wiki/%EC%A2%85%EB%B6%84%ED%99%94" title="종분화">종분화</a>가 이루어지는 <a class="new" href="/w/index.php?title=%EB%B6%84%EA%B8%B0%EC%A7%84%ED%99%94&amp;action=edit&amp;redlink=1" title="분기진화 (없는 문서)">분화기</a>로 나뉜다는 진화 이론이다.<sup class="reference" id="cite_ref-1"><a href="#cite_note-1">[1]</a></sup></p>
<p>단속평형이론은 종의 진화가 매우 오랜 시간 동안 세대에 걸쳐 점진적으로 이루어진다는 기존의 <a class="new" href="/w/index.php?title=%EA%B3%84%ED%86%B5%EC%A0%90%EC%A7%84%EC%9D%B4%EB%A1%A0&amp;action=edit&amp;redlink=1" title="계통점진이론 (없는 문서)">계통점진이론</a>을 정면으로 반박한다. 1972년 <a href="/wiki/%EC%8A%A4%ED%8B%B0%EB%B8%90_%EC%A0%9C%EC%9D%B4_%EA%B5%B4%EB%93%9C" title="스티븐 제이 굴드">스티븐 제이 굴드</a>와 <a class="new" href="/w/index.php?title=%EB%8B%90%EC%8A%A4_%EC%97%98%EB%93%9C%EB%A6%AC%EC%A7%80&amp;action=edit&amp;redlink=1" title="닐스 엘드리지 (없는 문서)">닐스 엘드리지</a>가 《단속평형》<sup class="reference" id="cite_ref-EnG_2-0"><a href="#cite_note-EnG-2">[2]</a></sup> 을 발표하였다. 단속평형이론은 <a href="/wiki/%EC%97%90%EB%A5%B8%EC%8A%A4%ED%8A%B8_%EB%A7%88%EC%9D%B4%EC%96%B4" title="에른스트 마이어">에른스트 마이어</a>의 <a class="new" href="/w/index.php?title=%EC%9D%B4%EC%86%8C%EC%A0%81_%EA%B2%A9%EB%A6%AC&amp;action=edit&amp;redlink=1" title="이소적 격리 (없는 문서)">이소적 격리</a> 이론<sup class="reference" id="cite_ref-Mayr_3-0"><a href="#cite_note-Mayr-3">[3]</a></sup> 과 <a class="new" href="/w/index.php?title=%EC%9D%B4%EC%8A%A4%EB%9D%BC%EC%97%98_%EB%A7%88%EC%9D%B4%ED%81%B4_%EB%9F%AC%EB%84%88&amp;action=edit&amp;redlink=1" title="이스라엘 마이클 러너 (없는 문서)">이스라엘 마이클 러너</a>의 유전적 <a href="/wiki/%EC%83%81%EB%8F%99%EC%84%B1" title="상동성">상동성</a> 이론을 바탕으로 하고 있다.<sup class="reference" id="cite_ref-4"><a href="#cite_note-4">[4]</a></sup></p>
<p>굴드와 엘드리지가 단속평형이론을 처음으로 발표한 1972년 당시 "표준" 진화 이론에서는 <a class="new" href="/w/index.php?title=%EC%9D%B4%EC%86%8C%EC%A0%81_%EC%A2%85%EB%B6%84%ED%99%94&amp;action=edit&amp;redlink=1" title="이소적 종분화 (없는 문서)">이소적 종분화</a>를 <a href="/wiki/%EC%A2%85%EB%B6%84%ED%99%94" title="종분화">종분화</a>의 대표적 원인으로 파악하고 있었다.<sup class="reference" id="cite_ref-EnG_2-1"><a href="#cite_note-EnG-2">[2]</a></sup><sup class="reference" id="cite_ref-5"><a href="#cite_note-5">[5]</a></sup> 이소적 종분화는 <a href="/wiki/%EC%97%90%EB%A5%B8%EC%8A%A4%ED%8A%B8_%EB%A7%88%EC%9D%B4%EC%96%B4" title="에른스트 마이어">에른스트 마이어</a>가 1954년 발표한 〈유전환경의 변화와 진화〉<sup class="reference" id="cite_ref-Mayr_3-1"><a href="#cite_note-Mayr-3">[3]</a></sup> 와 고전으로 평가받고 있는 마이어의 또 다른 저서 《동물 종과 진화》(1963년)<sup class="reference" id="cite_ref-6"><a href="#cite_note-6">[6]</a></sup> 에 의해 널리 알려져 있었다.</p>
<p>일정 규모 이상의 집단을 갖는 종들은 상호간의 <a href="/wiki/%EC%9C%A0%EC%A0%84%EC%9E%90_%EC%9D%B4%EB%8F%99" title="유전자 이동">유전자 이동</a>을 통해 종의 정체성을 지켜나간다. 이러한 상황에서는 종의 생존에 이로운 <a href="/wiki/%EB%8F%8C%EC%97%B0%EB%B3%80%EC%9D%B4" title="돌연변이">돌연변이</a>가 발생한다고 하더라도 매우큰 집단간의 유전자 이동으로 별다른 영향을 미치지 못하게 되고 집단의 종분화는 억제된다. 고생물의 <a href="/wiki/%ED%99%94%EC%84%9D" title="화석">화석</a> 기록은 이러한 것이 실제로 작동했음을 증명한다. 반면 소규모 무리가 집단에서 떨어져 나와 지리적으로 격리되면 유전자 이동이 불가능하므로 돌연변이에 의한 종분화가 가속화된다. 여기에 새로운 <a class="mw-redirect" href="/wiki/%EC%9E%90%EC%97%B0%ED%99%98%EA%B2%BD" title="자연환경">자연환경</a>이 가하는 <a href="/wiki/%EC%9E%90%EC%97%B0%EC%84%A0%ED%83%9D" title="자연선택">자연선택</a>이 더해지면 기존의 집단과는 다른 새로운 <a href="/wiki/%EC%A0%81%EC%9D%91" title="적응">적응</a>이 일어나게 된다. 마이어는 1954년 논문의 결론에서 <a class="new" href="/w/index.php?title=%EC%A7%80%EB%A6%AC%EC%A0%81_%EA%B2%A9%EB%A6%AC&amp;action=edit&amp;redlink=1" title="지리적 격리 (없는 문서)">지리적 격리</a>에 의한 <a class="new" href="/w/index.php?title=%EC%9D%B4%EC%86%8C%EC%A0%81_%EC%A2%85%EB%B6%84%ED%99%94&amp;action=edit&amp;redlink=1" title="이소적 종분화 (없는 문서)">이소적 종분화</a>가 <a href="/wiki/%EC%A2%85%EB%B6%84%ED%99%94" title="종분화">종분화</a>의 유일한 형태라고 가정하면, 지리적 격리가 일어나지 않는 상당한 기간동안 생물 종은 안정적인 상태를 유지할 것이라 예상하였다.</p>
