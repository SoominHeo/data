<h2><span class="mw-headline" id="2003.EB.85.84_10.EC.9B.94_28.EC.9D.BC"><a href="/wiki/2003%EB%85%84" title="2003년">2003년</a> <a href="/wiki/10%EC%9B%94_28%EC%9D%BC" title="10월 28일">10월 28일</a></span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=2003%EB%85%84_10%EC%9B%94&amp;action=edit&amp;section=1" title="부분 편집: 2003년 10월 28일">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<ul>
<li>프와티에(<a class="extiw" href="https://fr.wikipedia.org/wiki/Poitiers" title="fr:Poitiers">fr</a>)에서 <a href="/wiki/%EA%B2%8C%EB%A5%B4%ED%95%98%EB%A5%B4%ED%8A%B8_%EC%8A%88%EB%A2%B0%EB%8D%94" title="게르하르트 슈뢰더">게르하르트 슈뢰더</a> 독일 수상, 장 피에르 라파랭 프랑스 수상을 중심으로 프랑스의 각 <a href="/wiki/%ED%94%84%EB%9E%91%EC%8A%A4%EC%9D%98_%EB%A0%88%EC%A7%80%EC%98%B9" title="프랑스의 레지옹">레지옹</a>의 지사(22명 중 13명)와 독일의 각 란트의 지사(16명 중 6명)가 참석한 회의가 열리다.</li>
</ul>

<li><a href="/wiki/%EB%85%B8%EB%AC%B4%ED%98%84" title="노무현">노무현</a> 대통령이 재신임 <a href="/wiki/%EA%B5%AD%EB%AF%BC%ED%88%AC%ED%91%9C" title="국민투표">국민투표</a>를 제안하다.</li>
</ul>
<h2><span class="mw-headline" id="2003.EB.85.84_10.EC.9B.94_9.EC.9D.BC"><a href="/wiki/2003%EB%85%84" title="2003년">2003년</a> <a href="/wiki/10%EC%9B%94_9%EC%9D%BC" title="10월 9일">10월 9일</a></span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=2003%EB%85%84_10%EC%9B%94&amp;action=edit&amp;section=14" title="부분 편집: 2003년 10월 9일">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<ul>
<li>GSI 연구소에서 발견한 110번 <a class="mw-redirect" href="/wiki/%ED%99%94%ED%95%99_%EC%9B%90%EC%86%8C" title="화학 원소">원소</a>가 <a class="mw-redirect" href="/wiki/%EA%B5%AD%EC%A0%9C%EC%88%9C%EC%88%98%C2%B7%EC%9D%91%EC%9A%A9%ED%99%94%ED%95%99%EC%97%B0%ED%95%A9" title="국제순수·응용화학연합">국제순수·응용화학연합</a> 및 국제순수·응용물리학연합(<a class="extiw" href="https://en.wikipedia.org/wiki/International_Union_of_Pure_and_Applied_Physics" title="en:International Union of Pure and Applied Physics">en</a>)에 의해 <a href="/wiki/%EB%8B%A4%EB%A6%84%EC%8A%88%ED%83%80%ED%8A%AC" title="다름슈타튬">다름슈타튬</a>으로 공식 명명되다. 이는 GSI 연구소가 위치한 <a href="/wiki/%EB%8B%A4%EB%A6%84%EC%8A%88%ED%83%80%ED%8A%B8" title="다름슈타트">다름슈타트</a>에서 유래된 이름이다.</li>
</ul>
<h2><span class="mw-headline" id="2003.EB.85.84_10.EC.9B.94_8.EC.9D.BC"><a href="/wiki/2003%EB%85%84" title="2003년">2003년</a> <a href="/wiki/10%EC%9B%94_8%EC%9D%BC" title="10월 8일">10월 8일</a></span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=2003%EB%85%84_10%EC%9B%94&amp;action=edit&amp;section=15" title="부분 편집: 2003년 10월 8일">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<ul>
<li><a href="/wiki/%EB%8C%80%ED%95%9C%EB%AF%BC%EA%B5%AD_%EA%B2%80%EC%B0%B0%EC%B2%AD" title="대한민국 검찰청">대한민국 검찰</a>, 최도술이 <a href="/wiki/SK%EA%B7%B8%EB%A3%B9" title="SK그룹">SK그룹</a>으로부터 11억을 수수한 사실을 확인하다.</li>
<li>연예인 비리와 관련하여 <a href="/wiki/%EC%84%9C%EC%84%B8%EC%9B%90" title="서세원">서세원</a>과 <a href="/wiki/%EC%9D%B4%EC%88%98%EB%A7%8C" title="이수만">이수만</a>이 구속되다.</li>
<li><a href="/wiki/%ED%94%84%EB%A6%AC%ED%86%A0%EB%A6%AC%EC%95%84" title="프리토리아">프리토리아</a>에서 도미티앵 은다이제예(<a class="extiw" href="https://fr.wikipedia.org/wiki/Domitien_Ndayizeye" title="fr:Domitien Ndayizeye">fr</a>) <a href="/wiki/%EB%B6%80%EB%A3%AC%EB%94%94" title="부룬디">부룬디</a> 대통령과 <a class="mw-redirect" href="/wiki/%ED%94%BC%EC%97%90%EB%A5%B4_%EC%9D%91%EC%BF%A0%EB%A3%AC%EC%A7%80%EC%9E%90" title="피에르 응쿠룬지자">피에르 응쿠룬지자</a> 민주방위국민평의회(FDD) 사이에 평화 협정이 체결되다. 부룬디 내전은 2년 뒤인 <a href="/wiki/2005%EB%85%84" title="2005년">2005년</a>에 이르러서야 종식되었다.</li>
</ul>
<h2><span class="mw-headline" id="2003.EB.85.84_10.EC.9B.94_7.EC.9D.BC"><a href="/wiki/2003%EB%85%84" title="2003년">2003년</a> <a href="/wiki/10%EC%9B%94_7%EC%9D%BC" title="10월 7일">10월 7일</a></span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=2003%EB%85%84_10%EC%9B%94&amp;action=edit&amp;section=16" title="부분 편집: 2003년 10월 7일">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<ul>
<li>대한민국의 <a href="/wiki/%EB%85%B8%EB%AC%B4%ED%98%84" title="노무현">노무현</a> 대통령, 일본의 <a href="/wiki/%EA%B3%A0%EC%9D%B4%EC%A6%88%EB%AF%B8_%EC%A4%80%EC%9D%B4%EC%B9%98%EB%A1%9C" title="고이즈미 준이치로">고이즈미 준이치로</a> 총리, 중국의 <a href="/wiki/%EC%9B%90%EC%9E%90%EB%B0%94%EC%98%A4" title="원자바오">원자바오</a> 총리가 인도네시아 <a href="/wiki/%EB%B0%9C%EB%A6%AC_%EC%84%AC" title="발리 섬">발리 섬</a>에서 3국 정상회담을 갖고 14개 분야에 합의하다.</li>
</ul>
<h2><span class="mw-headline" id="2003.EB.85.84_10.EC.9B.94_5.EC.9D.BC"><a href="/wiki/2003%EB%85%84" title="2003년">2003년</a> <a href="/wiki/10%EC%9B%94_5%EC%9D%BC" title="10월 5일">10월 5일</a></span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=2003%EB%85%84_10%EC%9B%94&amp;action=edit&amp;section=17" title="부분 편집: 2003년 10월 5일">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<ul>
<li>미국 <a href="/wiki/%EC%A1%B0%EC%A7%80_W._%EB%B6%80%EC%8B%9C" title="조지 W. 부시">조지 W. 부시</a> 대통령, 부분분만 낙태 금지법(Partial-Birth Abortion Ban Act)에 서명하다.</li>
<li><a href="/wiki/%EC%95%84%ED%9D%90%EB%A7%88%ED%8A%B8_%EC%B9%B4%EB%94%94%EB%A1%9C%ED%94%84" title="아흐마트 카디로프">아흐마트 카디로프</a>가 선거에서 82.5%의 지지를 받아 <a href="/wiki/%EC%B2%B4%EC%B2%B8_%EA%B3%B5%ED%99%94%EA%B5%AD" title="체첸 공화국">체첸 공화국</a>의 대통령으로 선출되다.</li>
</ul>
<h2><span class="mw-headline" id="2003.EB.85.84_10.EC.9B.94_2.EC.9D.BC"><a href="/wiki/2003%EB%85%84" title="2003년">2003년</a> <a href="/wiki/10%EC%9B%94_2%EC%9D%BC" title="10월 2일">10월 2일</a></span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=2003%EB%85%84_10%EC%9B%94&amp;action=edit&amp;section=18" title="부분 편집: 2003년 10월 2일">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<ul>
<li>프로야구 선수인 <a href="/wiki/%EC%9D%B4%EC%8A%B9%EC%97%BD" title="이승엽">이승엽</a>, 56호 홈런으로 아시아 최다기록을 수립하다.</li>
</ul>
