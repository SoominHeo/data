<p><b>달의 교점</b>(月之 交点, lunar nodes)은 달의 <a class="new" href="/w/index.php?title=%EA%B6%A4%EB%8F%84_%EA%B5%90%EC%A0%90&amp;action=edit&amp;redlink=1" title="궤도 교점 (없는 문서)">궤도 교점</a>으로, <a href="/wiki/%EB%8B%AC" title="달">달</a>의 <a href="/wiki/%EA%B6%A4%EB%8F%84" title="궤도">궤도</a>가 <a href="/wiki/%ED%99%A9%EB%8F%84" title="황도">황도</a>와 교차하는 점이다. <b>승교점</b>(昇交點, ascending node)은 달이 황도를 북쪽으로 가로지는 곳이다. <b>강교점</b>(降交點, descending node)은 달이 황도를 남쪽으로 가로지는 곳이다.</p>
<p><a href="/wiki/%EC%8B%9D_(%EC%B2%9C%EB%AC%B8)" title="식 (천문)">식</a>은 오직 교점에서만 일어난다. <a href="/wiki/%EC%9D%BC%EC%8B%9D" title="일식">일식</a>은 <a class="mw-redirect" href="/wiki/%EA%B7%B8%EB%AF%90" title="그믐">그믐</a> 때 인데다가, 교점에서 달이 해를 가로지를 때 일어난다. <a href="/wiki/%EC%9B%94%EC%8B%9D" title="월식">월식</a>은 <a href="/wiki/%EB%B3%B4%EB%A6%84%EB%8B%AC" title="보름달">보름달</a>인 달이 교점을 통과할 때 일어난다. 만일 (황경) 11° 38'의 교점 내에서 보름달이 뜬다면, 월식이 일어날 수도 있으며, 만일 17° 25' 이내의 교점에 그믐달이 있다면, 일식이 일어날 수도 있다.</p>
<p>달의 교점은 6793.5일 또는 18.5996년에 걸쳐 황도를 <a class="new" href="/w/index.php?title=%ED%83%9C%EC%9D%8C_%EC%84%B8%EC%B0%A8&amp;action=edit&amp;redlink=1" title="태음 세차 (없는 문서)">역행</a>하며 (교점 주기라고도 불리는 <a href="/wiki/%EC%9E%A5%EB%8F%99_(%EC%A7%80%EA%B5%AC%EA%B3%BC%ED%95%99)" title="장동 (지구과학)">장동</a> 주기의) 한번의 공전을 완료한다.(이 기간은 <a href="/wiki/%EC%82%AC%EB%A1%9C%EC%8A%A4_%EC%A3%BC%EA%B8%B0" title="사로스 주기">사로스 주기</a>와는 다르다.)</p>

1550년 <a href="/wiki/%EA%B5%AC%EC%9D%B4%EB%8F%84_%EB%B3%B4%EB%82%98%ED%8B%B0" title="구이도 보나티">구이도 보나티</a>의《천문학의 서(Liber Astronomiae)》에 있는 용의 머리와 꼬리.</div>

<p>교점은 세계의 지역에 따라 다른 이름으로 불린다.</p>
<p>승교점은 달이 남쪽에서 북쪽으로 상향하는 황도면과 달의 궤도 사이의 교차점이며, 그것은 때때로 북교점(北交点, North node)이라고 불린다. 고대 유럽 문헌들에서, 그것은 (용의 머리를 뜻하는 <i>카푸트 드라코니스</i>(<i>Caput Draconis</i>)나 <i>아나비바존</i>(<i>Anabibazon</i>) 즉,) <b>용두</b>(龍頭, dragon's head)로 표기되어 있다. 승교점의 기호는 <a href="/wiki/%EC%9A%A9" title="용">용</a>의 머리를 뜻하는 <a href="/wiki/%EC%B2%9C%EB%AC%B8%ED%95%99" title="천문학">천문학</a>과 <a href="/wiki/%EC%A0%90%EC%84%B1%EC%88%A0" title="점성술">점성술</a>의 기호 <a class="image" href="/wiki/%ED%8C%8C%EC%9D%BC:U%2B260A.svg"><img alt="U+260A.svg" data-file-height="180" data-file-width="180" height="16" src="//upload.wikimedia.org/wikipedia/commons/thumb/8/8e/U%2B260A.svg/16px-U%2B260A.svg.png" srcset="//upload.wikimedia.org/wikipedia/commons/thumb/8/8e/U%2B260A.svg/24px-U%2B260A.svg.png 1.5x, //upload.wikimedia.org/wikipedia/commons/thumb/8/8e/U%2B260A.svg/32px-U%2B260A.svg.png 2x" width="16"/></a>이다. 유사하게, 강교점은 달이 북쪽에서 남쪽으로 하향하는 교점이며, 때때로 남교점(南交点, South node)라 불린다. 그것은 (용의 꼬리를 뜻하는 <i>카우다 드라코니스</i>(<i>Cauda Draconis</i>)나 <i>카타비바존</i>(<i>Catabibazon</i>) 즉,) <b>용미</b>(龍尾, dragon's tail)라고 불리며, 그것의 기호는 승교점의 것과는 정 반대인 <a class="image" href="/wiki/%ED%8C%8C%EC%9D%BC:U%2B260B.svg"><img alt="U+260B.svg" data-file-height="180" data-file-width="180" height="16" src="//upload.wikimedia.org/wikipedia/commons/thumb/b/b4/U%2B260B.svg/16px-U%2B260B.svg.png" srcset="//upload.wikimedia.org/wikipedia/commons/thumb/b/b4/U%2B260B.svg/24px-U%2B260B.svg.png 1.5x, //upload.wikimedia.org/wikipedia/commons/thumb/b/b4/U%2B260B.svg/32px-U%2B260B.svg.png 2x" width="16"/></a>이다. 사실상, 교점 주기 동안에 북교점이 남교점보다 남쪽에 있을 수도 있다. 그러므로, 승교점은 북향 교점, 강교점은 남향 교점이 더 정확한 표현이 될 수도 있다.</p>
<p><a href="/wiki/%ED%9E%8C%EB%91%90_%EC%A0%90%EC%84%B1%EC%88%A0" title="힌두 점성술">힌두 점성술</a>에서, 승교점 ☊은 <a class="new" href="/w/index.php?title=%EB%9D%BC%ED%9B%84&amp;action=edit&amp;redlink=1" title="라후 (없는 문서)">라후</a>라고 불리며, 강교점 ☋은 <a class="new" href="/w/index.php?title=%EC%BC%80%ED%88%AC&amp;action=edit&amp;redlink=1" title="케투 (없는 문서)">케투</a>라고 불린다. (일부 <a class="new" href="/w/index.php?title=%EC%B9%BC%EB%9D%BC%EC%B0%A8%ED%81%AC%EB%9D%BC&amp;action=edit&amp;redlink=1" title="칼라차크라 (없는 문서)">칼라차크라 탄트라</a>에 근거하는) <a class="new" href="/w/index.php?title=%ED%8B%B0%EB%B2%A0%ED%8A%B8_%EC%A0%90%EC%84%B1%EC%88%A0&amp;action=edit&amp;redlink=1" title="티베트 점성술 (없는 문서)">티베트 점성술</a>에서, 북교점은 칼라그니라는 이름이다.</p>
<h2><span class="mw-headline" id=".EC.B5.9C.EB.8C.80_.EA.B2.BD.EC.82.AC.EA.B0.81">최대 경사각</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%8B%AC%EC%9D%98_%EA%B5%90%EC%A0%90&amp;action=edit&amp;section=2" title="부분 편집: 최대 경사각">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<div class="rellink boilerplate seealso"><img alt="" data-file-height="128" data-file-width="128" height="16" src="//upload.wikimedia.org/wikipedia/commons/thumb/e/ec/Crystal_Clear_app_xmag.svg/16px-Crystal_Clear_app_xmag.svg.png" srcset="//upload.wikimedia.org/wikipedia/commons/thumb/e/ec/Crystal_Clear_app_xmag.svg/24px-Crystal_Clear_app_xmag.svg.png 1.5x, //upload.wikimedia.org/wikipedia/commons/thumb/e/ec/Crystal_Clear_app_xmag.svg/32px-Crystal_Clear_app_xmag.svg.png 2x" width="16"/> <a class="new" href="/w/index.php?title=%EA%B5%90%EC%A0%90_%EC%A3%BC%EA%B8%B0&amp;action=edit&amp;redlink=1" title="교점 주기 (없는 문서)">교점 주기#최대 편위</a> 문서를 참고하십시오.</div>
<p>달의 궤도는 황도에서 약 5도 기울어 있다. 그러므로, 달은 황도의 남쪽이나 북쪽으로 약 5도씩 오르락내리락하는 것이라고도 할 수 있다. 황도는 지구 자전축의 직각면인 천구 적도에 비해 23.4°가 기울어져 있다. 그 결과, 18.6년인 교점 주기 동안에 단 한번, 달의 궤도상 승교점이 <a href="/wiki/%EC%B6%98%EB%B6%84%EC%A0%90" title="춘분점">춘분점</a>과 일치 할 때면, 달은 남쪽이과 북쪽의 최대 <a href="/wiki/%EC%A0%81%EC%9C%84" title="적위">적위</a>에 이르게 된다. 그러한 기간에 그믐달이 지평선에서 출몰 한다면, 그것은 남쪽과 북쪽의 최대 <a href="/wiki/%EB%B0%A9%EC%9C%84%EA%B0%81" title="방위각">방위각</a>을 가지게 되며, <a class="mw-redirect" href="/wiki/%EC%9E%90%EC%98%A4%EC%84%A0" title="자오선">자오선</a>을 교차할 때는 그것의 최고나 최하의 <a href="/wiki/%EA%B3%A0%EB%8F%84_(%EB%86%92%EC%9D%B4)" title="고도 (높이)">고도</a>에 있게 된다. 게다가, 매 교점 주기마다 단 한 번의 그 짦은 기간 중에 황도로부터 북위 4° 에 있는 밝은 <a class="mw-redirect" href="/wiki/%ED%94%8C%EB%A0%88%EC%9D%B4%EC%95%84%EB%8D%B0%EC%8A%A4_%EC%84%B1%EB%8B%A8" title="플레이아데스 성단">플레이아데스 성단</a>의 달에 의한 <a class="mw-redirect" href="/wiki/%EC%97%84%ED%8F%90_(%EC%B2%9C%EB%AC%B8%ED%95%99)" title="엄폐 (천문학)">엄폐</a>가 일어나게 된다.</p>
<h2><span class="mw-headline" id=".EC.A0.90.EC.84.B1.ED.95.99.EC.A0.81_.EC.9D.98.EB.AF.B8">점성학적 의미</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%8B%AC%EC%9D%98_%EA%B5%90%EC%A0%90&amp;action=edit&amp;section=3" title="부분 편집: 점성학적 의미">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p><a href="/wiki/%EC%84%9C%EC%96%91_%EC%A0%90%EC%84%B1%EC%88%A0" title="서양 점성술">서양 점성술</a>에서는 일반적으로 승교점만을 <a href="/wiki/%EC%B2%9C%EA%B6%81%EB%8F%84" title="천궁도">천궁도</a>에 표시하는데, 따라서 차트에서 강교점은 그것의 정반대쪽 점으로 정의된다. <a class="mw-redirect" href="/wiki/%EB%B2%A0%EB%8B%A4_%EC%A0%90%EC%84%B1%EC%88%A0" title="베다 점성술">베다 점성술</a>에서, 승교점과 강교점은 각각 라후와 케투라 불리며, 둘 모두 차트에 표시된다.</p>
<p>교점은 언제나 역행하므로, 본질적으로 불길하다고 여겨진다. 그것들의 특정한 각맺음에 대해서는 일치가 없다.</p>
