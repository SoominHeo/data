<h1 class="firstHeading" id="firstHeading" lang="ko">당자</h1>
<div class="mw-body-content" id="bodyContent">
<div id="siteSub">위키백과, 우리 모두의 백과사전.</div>
<div id="contentSub"></div>
<div class="mw-jump" id="jump-to-nav">
					이동:					<a href="#mw-head">둘러보기</a>, 					<a href="#p-search">검색</a>
</div>
<div class="mw-content-ltr" dir="ltr" id="mw-content-text" lang="ko"><script>function mfTempOpenSection(id){var block=document.getElementById("mf-section-"+id);block.className+=" open-block";block.previousSibling.className+=" open-block";}</script><p><b>당자</b>(唐咨, ? ~ ?)는 <a href="/wiki/%EC%A4%91%EA%B5%AD" title="중국">중국</a> <a href="/wiki/%EC%82%BC%EA%B5%AD_%EC%8B%9C%EB%8C%80_(%EC%A4%91%EA%B5%AD)" title="삼국 시대 (중국)">삼국시대</a> <a href="/wiki/%EC%98%A4_(%EC%82%BC%EA%B5%AD)" title="오 (삼국)">오나라</a>와 <a href="/wiki/%EC%9C%84_(%EC%82%BC%EA%B5%AD)" title="위 (삼국)">위나라</a>의 장수이며, <a href="/wiki/%EC%82%B0%EB%91%A5_%EC%84%B1" title="산둥 성">산동</a>(山東)군 출신이다.</p>
<h2><span class="mw-headline" id=".EC.83.9D.EC.95.A0">생애</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%8B%B9%EC%9E%90&amp;action=edit&amp;section=1" title="부분 편집: 생애">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p><a href="/wiki/225%EB%85%84" title="225년">225년</a>, 산동 지역의 이성군(利成郡)의 병사 채방(蔡方) 등이 반란을 일으켜 태수 서기(徐箕)<sup class="reference" id="cite_ref-1"><a href="#cite_note-1">[1]</a></sup> 를 죽이고 당자를 추대하여 자신들의 우두머리로 받들었다. 위나라 조정에서 둔기교위(屯騎校尉) 임복(任福)으로 하여금 당자의 무리를 공격하게 하니 얼마 가지 않아 진압당하여 배를 타고 오나라로 달아났다. 오나라 조정에서는 당자를 장군으로 임명하여 오나라의 장수가 되었다.</p>
<p><a href="/wiki/239%EB%85%84" title="239년">239년</a>, 오나라의 장수 요식(廖植)이 임하(臨賀)태수 엄강(嚴綱)을 죽이고 스스로를 평남장군(平南將軍)이라 칭하며 형주 남부와 교주 일대를 혼란스럽게 만들자, 오나라에서는 여대(呂岱)가 진압에 나섰다. 당자는 <a href="/wiki/%EC%86%90%EA%B6%8C" title="손권">손권</a>(孫權)의 명령을 받아 여대를 도와서 반란을 진압하는 데 이바지했다. <a href="/wiki/252%EB%85%84" title="252년">252년</a> 11월에 위나라의 <a href="/wiki/%EC%99%95%EC%B0%BD_(%EC%A1%B0%EC%9C%84)" title="왕창 (조위)">왕창</a>(王昶), <a href="/wiki/%EA%B4%80%EA%B5%AC%EA%B2%80" title="관구검">관구검</a>(毌丘倹), <a href="/wiki/%EC%A0%9C%EA%B0%88%ED%83%84" title="제갈탄">제갈탄</a>(諸葛誕), 호준 등이 오나라의 동흥(東興)을 공격해 왔을 때 <a href="/wiki/%EC%A0%9C%EA%B0%88%EA%B0%81" title="제갈각">제갈각</a>(諸葛恪)에 의해 선봉장으로 참전했다.</p>
<p><a href="/wiki/256%EB%85%84" title="256년">256년</a>, <a href="/wiki/%EA%B4%80%EA%B5%AC%EA%B2%80" title="관구검">관구검</a>의 반란을 돕다가 오나라로 망명해온 <a href="/wiki/%EB%AC%B8%ED%9D%A0" title="문흠">문흠</a>(文欽)이 위나라를 공격할 것을 청하였다. 이를 받아들인 오나라의 실권자 <a href="/wiki/%EC%86%90%EC%A4%80" title="손준">손준</a>(孫峻)이 편성한 정벌군에 전장군(前將軍)으로써 참전하여 서주와 청주를 공략하는 임무를 받았으나, <a href="/wiki/%EC%86%90%EC%A4%80" title="손준">손준</a>이 갑작스런 병으로 죽는 바람에 원정이 중단되었다.</p>
<p><a href="/wiki/257%EB%85%84" title="257년">257년</a>, <a href="/wiki/%EC%A0%9C%EA%B0%88%ED%83%84" title="제갈탄">제갈탄</a>이 <a href="/wiki/%EC%82%AC%EB%A7%88%EC%86%8C" title="사마소">사마소</a>(司馬昭)의 야심에 반발하여 수춘에서 반란을 일으키고, 오나라에 아들 <a href="/wiki/%EC%A0%9C%EA%B0%88%EC%A0%95" title="제갈정">제갈정</a>(諸葛靚)을 인질로 보내고 구원을 청하자 오나라에서는 3만 명의 원군을 보냈다. 당자는 <a href="/wiki/%EB%AC%B8%ED%9D%A0" title="문흠">문흠</a>, <a href="/wiki/%EC%A0%84%EC%97%AD_(%EB%8F%99%EC%98%A4)" title="전역 (동오)">전역</a>(全懌) 등과 함께 수춘으로 가서 <a href="/wiki/%EC%82%AC%EB%A7%88%EC%86%8C" title="사마소">사마소</a>의 진압군에 맞섰으나 <a href="/wiki/%EC%A0%84%EC%97%AD_(%EB%8F%99%EC%98%A4)" title="전역 (동오)">전역</a>, 전단이 위나라에 항복하는 등 궁지에 몰리게 되었다.</p>
<p><a href="/wiki/258%EB%85%84" title="258년">258년</a>, 남아있는 무기를 총동원하여 5, 6일간 밤낮으로 남쪽 포위망을 공격하지만 위나라의 투석기와 불화살에 공격 무기들마저 다 파괴되면서 탈주병이 속출했으며, 제갈탄이 문흠과의 의견 차이로 문흠을 죽이는 사태가 벌어져 문앙과 문호가 위나라로 투항한데다가 제갈탄과 당자의 계책도 다하니 결국 위나라의 총공격에 제갈탄은 죽고 성은 함락되어 당자는 적에게 사로잡힌다.</p>
<p><a href="/wiki/262%EB%85%84" title="262년">262년</a>, <a href="/wiki/%EC%A2%85%ED%9A%8C" title="종회">종회</a>와 <a href="/wiki/%EC%82%AC%EB%A7%88%EC%86%8C" title="사마소">사마소</a>의 명에 따라 바다도 건널 수 있는 큰 배를 만드니 외부에서는 오나라를 토벌하기 위하여 배를 만든다고 여겼다.</p>
<p><a href="/wiki/263%EB%85%84" title="263년">263년</a>, 유선이 항복하자 종회에게 항복한 강유 및 촉의 관리들을 포로들 앞에서 종회가 말할 때 <b>"당자는 포위당해 포로로 붙잡힌 후에 항복하고도 지금 정사에 참여하고 있으니 다른 사람들은 걱정할 필요없다"</b>라고 말했다.</p>
<p>그 이후의 행적은 알 수 없다.</p>
<h2><span class="mw-headline" id=".EC.82.BC.EA.B5.AD.EC.A7.80.EC.97.B0.EC.9D.98.EC.97.90.EC.84.9C.EC.9D.98_.EB.8B.B9.EC.9E.90">삼국지연의에서의 당자</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%8B%B9%EC%9E%90&amp;action=edit&amp;section=2" title="부분 편집: 삼국지연의에서의 당자">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>이성군에서 반란을 일으킨 것은 나오지 않지만 진나라와의 싸움과 배를 만드는 장면에서 등장하며, 정사와는 달리 항복한 당자의 가족들이 손침에게 살해당한다는 점이 나온다.</p>
