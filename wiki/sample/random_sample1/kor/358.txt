<h2><span class="mw-headline" id="2004.EB.85.84_1.EC.9B.94_15.EC.9D.BC"><a href="/wiki/2004%EB%85%84" title="2004년">2004년</a> <a href="/wiki/1%EC%9B%94_15%EC%9D%BC" title="1월 15일">1월 15일</a></span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=2004%EB%85%84_1%EC%9B%94&amp;action=edit&amp;section=1" title="부분 편집: 2004년 1월 15일">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<ul>
<li><a href="/wiki/%EB%8C%80%ED%95%9C%EB%AF%BC%EA%B5%AD_%EC%99%B8%EA%B5%90%ED%86%B5%EC%83%81%EB%B6%80" title="대한민국 외교통상부">외교통상부</a> 직원의 이른바 부적절한 발언 파문으로 <a href="/wiki/%EC%9C%A4%EC%98%81%EA%B4%80" title="윤영관">윤영관</a> 장관이 사임하다.</li>
</ul>
<h2><span class="mw-headline" id="2004.EB.85.84_1.EC.9B.94_11.EC.9D.BC"><a href="/wiki/2004%EB%85%84" title="2004년">2004년</a> <a href="/wiki/1%EC%9B%94_11%EC%9D%BC" title="1월 11일">1월 11일</a></span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=2004%EB%85%84_1%EC%9B%94&amp;action=edit&amp;section=2" title="부분 편집: 2004년 1월 11일">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<ul>
<li><a href="/wiki/%EC%97%B4%EB%A6%B0%EC%9A%B0%EB%A6%AC%EB%8B%B9" title="열린우리당">열린우리당</a>은 전당대회를 열어 <a href="/wiki/%EC%A0%95%EB%8F%99%EC%98%81" title="정동영">정동영</a>을 새 당 의장으로 선출하다.</li>
</ul>
<h2><span class="mw-headline" id="2004.EB.85.84_1.EC.9B.94_5.EC.9D.BC"><a href="/wiki/2004%EB%85%84" title="2004년">2004년</a> <a href="/wiki/1%EC%9B%94_5%EC%9D%BC" title="1월 5일">1월 5일</a></span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=2004%EB%85%84_1%EC%9B%94&amp;action=edit&amp;section=3" title="부분 편집: 2004년 1월 5일">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<ul>
<li><a href="/wiki/%EB%B6%80%EB%A3%AC%EB%94%94" title="부룬디">부룬디</a>의 <a href="/wiki/%ED%9B%84%ED%88%AC%EC%A1%B1" title="후투족">후투족</a> 무장 조직인 FNL(<a class="extiw" href="https://fr.wikipedia.org/wiki/Forces_nationales_de_lib%C3%A9ration_(Burundi)" title="fr:Forces nationales de libération (Burundi)">fr</a>)이 12월 31일 발표한 <a href="/wiki/%EC%84%A0%EC%A0%84_%ED%8F%AC%EA%B3%A0" title="선전 포고">선전 포고</a>를 철회하고 도미티앵 은다이제예(<a class="extiw" href="https://fr.wikipedia.org/wiki/Domitien_Ndayizeye" title="fr:Domitien Ndayizeye">fr</a>) 대통령과 협상할 것을 받아들이다.</li>
</ul>
<h2><span class="mw-headline" id="2004.EB.85.84_1.EC.9B.94_3.EC.9D.BC"><a href="/wiki/2004%EB%85%84" title="2004년">2004년</a> <a href="/wiki/1%EC%9B%94_3%EC%9D%BC" title="1월 3일">1월 3일</a></span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=2004%EB%85%84_1%EC%9B%94&amp;action=edit&amp;section=4" title="부분 편집: 2004년 1월 3일">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<ul>
<li><a href="/wiki/%EC%9D%B4%EC%A7%91%ED%8A%B8" title="이집트">이집트</a>의 <a href="/wiki/%EC%A0%80%EA%B0%80_%ED%95%AD%EA%B3%B5%EC%82%AC" title="저가 항공사">저가 항공사</a>인 플래시 항공(<a class="extiw" href="https://en.wikipedia.org/wiki/Flash_Airlines" title="en:Flash Airlines">en</a>) 소속의 604편(<a href="/wiki/%EB%B3%B4%EC%9E%89_737" title="보잉 737">보잉 737-300</a>형)이 <a href="/wiki/%EC%8B%9C%EB%82%98%EC%9D%B4_%EB%B0%98%EB%8F%84" title="시나이 반도">시나이 반도</a> 남단의 <a class="mw-redirect" href="/wiki/%EC%83%A4%EB%A6%84_%EC%97%98_%EC%85%B0%EC%9D%B4%ED%81%AC" title="샤름 엘 셰이크">샤름 엘 셰이크</a> 근처 <a href="/wiki/%ED%99%8D%ED%95%B4" title="홍해">홍해</a> 상공에서 추락하여 탑승자 148명 전원이 사망하다.(<a class="extiw" href="https://en.wikipedia.org/wiki/Flash_Airlines_Flight_604" title="en:Flash Airlines Flight 604">en:Flash Airlines Flight 604</a>)</li>
</ul>
<h2><span class="mw-headline" id="2004.EB.85.84_1.EC.9B.94_1.EC.9D.BC"><a href="/wiki/2004%EB%85%84" title="2004년">2004년</a> <a href="/wiki/1%EC%9B%94_1%EC%9D%BC" title="1월 1일">1월 1일</a></span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=2004%EB%85%84_1%EC%9B%94&amp;action=edit&amp;section=5" title="부분 편집: 2004년 1월 1일">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<ul>
<li><a href="/wiki/%ED%95%9C%EA%B5%AD%EC%B2%A0%EB%8F%84%EC%8B%9C%EC%84%A4%EA%B3%B5%EB%8B%A8" title="한국철도시설공단">한국철도시설공단</a>이 공식 출범하다.</li>
<li><a href="/wiki/%ED%94%84%EB%9E%91%EC%8A%A4" title="프랑스">프랑스</a> <a class="mw-redirect" href="/wiki/%ED%8C%8C%EB%A6%AC_(%EB%8F%84%EC%8B%9C)" title="파리 (도시)">파리</a> 도심에서 새해를 축하하던 2천여 명의 군중이 갑자기 폭도화하여 경찰 4500여 명에 의해 진압되었다. 총 214명이 부상당하였다.</li>
<li><a href="/wiki/%EA%B3%A0%EC%9D%B4%EC%A6%88%EB%AF%B8_%EC%A4%80%EC%9D%B4%EC%B9%98%EB%A1%9C" title="고이즈미 준이치로">고이즈미 준이치로</a> 일본 총리가 <a href="/wiki/%EC%95%BC%EC%8A%A4%EC%BF%A0%EB%8B%88_%EC%8B%A0%EC%82%AC" title="야스쿠니 신사">야스쿠니 신사</a>를 참배하다.</li>
</ul>
