<p><b>대니엘 패너베이커</b>(<span style="font-size: smaller;"><a href="/wiki/%EC%98%81%EC%96%B4" title="영어">영어</a>:</span> <span lang="en">Danielle Panabaker</span>, <a href="/wiki/1987%EB%85%84" title="1987년">1987년</a> <a href="/wiki/9%EC%9B%94_19%EC%9D%BC" title="9월 19일">9월 19일</a> ~ )는 <a href="/wiki/%EB%AF%B8%EA%B5%AD" title="미국">미국</a>의 <a href="/wiki/%EB%B0%B0%EC%9A%B0" title="배우">배우</a>이다. 10대 시절부터 연기를 시작하였으며, 본격적으로 이름을 알린 것은 영화 《<a href="/wiki/%EB%B8%8C%EB%A6%AC%ED%8A%B8%EB%8B%88%EC%9D%98_%EC%9D%BC%EC%83%81%ED%83%88%EC%B6%9C" title="브리트니의 일상탈출">브리트니의 일상탈출</a>》과 《<a class="mw-redirect mw-disambig" href="/wiki/%EC%8A%A4%EC%B9%B4%EC%9D%B4_%ED%95%98%EC%9D%B4" title="스카이 하이">스카이 하이</a>》, <a href="/wiki/HBO" title="HBO">HBO</a>의 미니시리즈 《<a class="new" href="/w/index.php?title=%EC%97%A0%ED%8C%8C%EC%9D%B4%EC%96%B4_%ED%8F%B4%EC%8A%A4&amp;action=edit&amp;redlink=1" title="엠파이어 폴스 (없는 문서)">엠파이어 폴스</a>》(Empire Falls) 등에 출연하면서부터이다.</p>
<p>대니엘 패너베이커는 <a class="mw-redirect" href="/wiki/%EA%B3%B5%ED%8F%AC%EC%98%81%ED%99%94" title="공포영화">공포영화</a>의 단골 배우로도 유명하다. 2007년 《<a href="/wiki/%EB%AF%B8%EC%8A%A4%ED%84%B0_%EB%B8%8C%EB%A3%A9%EC%8A%A4" title="미스터 브룩스">미스터 브룩스</a>》를 시작으로 《<a href="/wiki/13%EC%9D%BC%EC%9D%98_%EA%B8%88%EC%9A%94%EC%9D%BC_(2009%EB%85%84_%EC%98%81%ED%99%94)" title="13일의 금요일 (2009년 영화)">13일의 금요일</a>》, 《<a class="mw-disambig" href="/wiki/%ED%81%AC%EB%A0%88%EC%9D%B4%EC%A7%80" title="크레이지">크레이지</a>》, 《<a href="/wiki/%EB%8D%94_%EC%9B%8C%EB%93%9C" title="더 워드">더 워드</a>》, 《<a href="/wiki/%ED%94%BC%EB%9D%BC%EB%83%90_3DD" title="피라냐 3DD">피라냐 3DD</a>》 외 다수의 영화에서 '비명의 여왕(scream queen)'으로서 존재감을 뽐냈다.</p>
<p>2014년에는 <a href="/wiki/The_CW" title="The CW">The CW</a> 방송의 슈퍼히어로 드라마 《<a class="mw-redirect mw-disambig" href="/wiki/%EC%95%A0%EB%A1%9C%EC%9A%B0" title="애로우">애로우</a>》에 <a class="new" href="/w/index.php?title=%EC%BC%80%EC%9D%B4%ED%8B%80%EB%A6%B0_%EC%8A%A4%EB%85%B8&amp;action=edit&amp;redlink=1" title="케이틀린 스노 (없는 문서)">케이틀린 스노</a> 역으로 출연하였고, 같은 세계관을 다룬 《<a href="/wiki/%ED%94%8C%EB%9E%98%EC%8B%9C_(%EB%93%9C%EB%9D%BC%EB%A7%88)" title="플래시 (드라마)">플래시</a>》에서 주연으로 발탁되었다.<sup class="reference" id="cite_ref-1"><a href="#cite_note-1">[1]</a></sup></p>

<h2><span class="mw-headline" id=".EC.83.9D.EC.95.A0">생애</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%8C%80%EB%8B%88%EC%97%98_%ED%8C%A8%EB%84%88%EB%B2%A0%EC%9D%B4%EC%BB%A4&amp;action=edit&amp;section=1" title="부분 편집: 생애">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>대니엘 패너베이커는 <a href="/wiki/%EC%A1%B0%EC%A7%80%EC%95%84_%EC%A3%BC" title="조지아 주">조지아 주</a> <a class="mw-disambig" href="/wiki/%EC%98%A4%EA%B1%B0%EC%8A%A4%ED%83%80" title="오거스타">오거스타</a>에서, 도나 메이옥(Mayock)과 해럴드 패너베이커 사이에서 태어났다.<sup class="reference" id="cite_ref-2"><a href="#cite_note-2">[2]</a></sup> 이후 <a href="/wiki/%ED%85%8D%EC%82%AC%EC%8A%A4_%EC%A3%BC" title="텍사스 주">텍사스 주</a> <a href="/wiki/%EC%98%A4%EB%A0%8C%EC%A7%80" title="오렌지">오렌지</a>로 이사하여, 여름 캠프의 연극 수업에서 연기의 재능을 발견했다. 14세에 고등학교를 졸업한 패너베이커는<sup class="reference" id="cite_ref-3"><a href="#cite_note-3">[3]</a></sup> <a href="/wiki/%EB%A1%9C%EC%8A%A4%EC%95%A4%EC%A0%A4%EB%A0%88%EC%8A%A4" title="로스앤젤레스">로스앤젤레스</a>에서 배우가 되기로 결심하고 글렌데일 전문대학(Glendale Community College)에서 연기를 공부한다. 이후 2006년 <a href="/wiki/%EC%BA%98%EB%A6%AC%ED%8F%AC%EB%8B%88%EC%95%84_%EB%8C%80%ED%95%99%EA%B5%90_%EB%A1%9C%EC%8A%A4%EC%95%A4%EC%A0%A4%EB%A0%88%EC%8A%A4" title="캘리포니아 대학교 로스앤젤레스">캘리포니아 대학교 로스앤젤레스</a>에서 <a href="/wiki/%EB%AC%B8%ED%95%99%EC%82%AC" title="문학사">문학사</a>를 수학하였다.</p>
