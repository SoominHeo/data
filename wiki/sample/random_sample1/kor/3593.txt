<p><b>대런 영</b>(Darren Young, <a href="/wiki/1983%EB%85%84" title="1983년">1983년</a> <a href="/wiki/11%EC%9B%94_2%EC%9D%BC" title="11월 2일">11월 2일</a> ~ )은 남자 <a class="mw-redirect" href="/wiki/%ED%94%84%EB%A1%9C%EB%A0%88%EC%8A%AC%EB%A7%81%EC%84%A0%EC%88%98" title="프로레슬링선수">프로레슬링선수</a>다. <a href="/wiki/WWE" title="WWE">WWE</a> 제휴단체인 <a class="mw-disambig" href="/wiki/FCW" title="FCW">FCW</a>를 거쳐 <a class="mw-disambig" href="/wiki/NXT" title="NXT">NXT</a> 시즌1에 참가하였고 <a href="/wiki/%EB%84%A5%EC%84%9C%EC%8A%A4_(%ED%94%84%EB%A1%9C%EB%A0%88%EC%8A%AC%EB%A7%81)" title="넥서스 (프로레슬링)">넥서스 (프로레슬링)</a>의 멤버였다. <a href="/wiki/2012%EB%85%84" title="2012년">2012년</a> <a href="/wiki/4%EC%9B%94_20%EC%9D%BC" title="4월 20일">4월 20일</a> 스맥다운에서 <a href="/wiki/%ED%83%80%EC%9D%B4%ED%84%B0%EC%8A%A4_%EC%98%A4%EB%8B%90" title="타이터스 오닐">타이터스 오닐</a>하고 태그팀으로 악역으로 활동 중이었다가 나중에는 <a href="/wiki/2013%EB%85%84" title="2013년">2013년</a> <a href="/wiki/8%EC%9B%94_19%EC%9D%BC" title="8월 19일">8월 19일</a> 선역으로 돌아오게 되었지만 <a href="/wiki/2014%EB%85%84" title="2014년">2014년</a> <a href="/wiki/1%EC%9B%94_31%EC%9D%BC" title="1월 31일">1월 31일</a> 끝내 <a href="/wiki/%ED%83%80%EC%9D%B4%ED%84%B0%EC%8A%A4_%EC%98%A4%EB%8B%90" title="타이터스 오닐">타이터스 오닐</a>한테 배신을 당해 <a class="mw-redirect" href="/wiki/%EB%8D%94_%ED%94%84%EB%9D%BC%EC%9E%84_%ED%83%80%EC%9E%84_%ED%94%8C%EB%A0%88%EC%9D%B4%EC%96%B4%EC%A6%88" title="더 프라임 타임 플레이어즈">더 프라임 타임 플레이어즈</a>를 해제 되고 만다. <a href="/wiki/2015" title="2015">2015</a>년 <a href="/wiki/2%EC%9B%94_16%EC%9D%BC" title="2월 16일">2월 16일</a> 오닐이 선역으로 탄생하게 되었고 부활하게 되었다. 하지만 16년 12월 27일에 스트로우먼 한테 구타당한다. 키는 186cm 몸무게는 108kg이다.</p>
