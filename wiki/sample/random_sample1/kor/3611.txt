<p><b>대중운동연합</b>(大衆運動聯合, <span style="font-size: smaller;"><a href="/wiki/%ED%94%84%EB%9E%91%EC%8A%A4%EC%96%B4" title="프랑스어">프랑스어</a>:</span> <span lang="fr" xml:lang="fr">Union pour un Mouvement Populaire</span>, 약칭 <b>UMP</b>)는 <a href="/wiki/%EC%A4%91%EB%8F%84%EC%9A%B0%ED%8C%8C" title="중도우파">중도우파</a>의 성격을 가진 <a href="/wiki/%ED%94%84%EB%9E%91%EC%8A%A4" title="프랑스">프랑스</a>의 정당이다.</p>
<p><a href="/wiki/2002%EB%85%84" title="2002년">2002년</a>에 창당되어, <a href="/wiki/2007%EB%85%84" title="2007년">2007년</a>의 대선에서는 <a href="/wiki/%EB%8B%88%EC%BD%9C%EB%9D%BC_%EC%82%AC%EB%A5%B4%EC%BD%94%EC%A7%80" title="니콜라 사르코지">니콜라 사르코지</a>를 후보로 내세워 당선되었다. 2015년 <a href="/wiki/%EA%B3%B5%ED%99%94%EB%8B%B9_(%ED%94%84%EB%9E%91%EC%8A%A4)" title="공화당 (프랑스)">공화당</a>으로 재창당되었다.</p>

<h2><span class="mw-headline" id=".EC.97.AD.EC.82.AC">역사</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%8C%80%EC%A4%91%EC%9A%B4%EB%8F%99%EC%97%B0%ED%95%A9&amp;action=edit&amp;section=1" title="부분 편집: 역사">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<h3><span class="mw-headline" id=".EC.84.9C.EB.A1.A0">서론</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%8C%80%EC%A4%91%EC%9A%B4%EB%8F%99%EC%97%B0%ED%95%A9&amp;action=edit&amp;section=2" title="부분 편집: 서론">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>1980년대부터 프랑스의 우파 세력들은 유럽의 건설과 경제적 자유주의란 가치를 중심으로 세력을 형성해나가기 시작하였다. 사회당을 비롯한 그들의 이념적·정치적 경쟁자들은 <a href="/wiki/1981%EB%85%84" title="1981년">1981년</a>과 <a href="/wiki/1988%EB%85%84" title="1988년">1988년</a>의 총선에서 승리를 거두자, 몇몇 정치인들을 중심으로 우익 연합의 필요성을 강조하는 목소리가 등장하였다.</p>
<p><a href="/wiki/1993%EB%85%84" title="1993년">1993년</a>의 총선을 앞두고 <a href="/wiki/%EB%93%9C%EA%B3%A8%EC%A3%BC%EC%9D%98" title="드골주의">드골주의</a> 보수정당인 <a href="/wiki/%EA%B3%B5%ED%99%94%EA%B5%AD%EC%97%B0%ED%95%A9" title="공화국연합">공화국연합</a> (RPR)과 중도정당인 <a class="new" href="/w/index.php?title=%ED%94%84%EB%9E%91%EC%8A%A4_%EB%AF%BC%EC%A3%BC%EB%8F%99%EB%A7%B9&amp;action=edit&amp;redlink=1" title="프랑스 민주동맹 (없는 문서)">프랑스 민주동맹</a> (UDF)이 <a class="new" href="/w/index.php?title=%ED%94%84%EB%9E%91%EC%8A%A4%EB%A5%BC_%EC%9C%84%ED%95%9C_%EC%97%B0%ED%95%A9&amp;action=edit&amp;redlink=1" title="프랑스를 위한 연합 (없는 문서)">프랑스를 위한 연합</a>이란 이름의 선거연합을 결성하였다. 그러나 <a href="/wiki/1995%EB%85%84" title="1995년">1995년</a>의 총선 기간 동안, 이 선거연합은 다시 <a href="/wiki/%EC%9E%90%ED%81%AC_%EC%8B%9C%EB%9D%BC%ED%81%AC" title="자크 시라크">자크 시라크</a>를 따르는 무리와 <a href="/wiki/%EC%97%90%EB%91%90%EC%95%84%EB%A5%B4_%EB%B0%9C%EB%9D%BC%EB%92%A4%EB%A5%B4" title="에두아르 발라뒤르">에두아르 발라뒤르</a>를 지지하는 무리로 갈리게 된다. 결국 이들은 <a href="/wiki/1997%EB%85%84" title="1997년">1997년</a>의 총선에서 패배한 뒤에 공동 교섭단체인 '<a class="new" href="/w/index.php?title=%ED%94%84%EB%9E%91%EC%8A%A4%EB%A5%BC_%EC%9C%84%ED%95%9C_%EB%8F%99%EB%A7%B9&amp;action=edit&amp;redlink=1" title="프랑스를 위한 동맹 (없는 문서)">프랑스를 위한 동맹</a>'을 다시 구성하게 된다.</p>
<h3><span class="mw-headline" id=".EC.B0.BD.EB.8B.B9">창당</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%8C%80%EC%A4%91%EC%9A%B4%EB%8F%99%EC%97%B0%ED%95%A9&amp;action=edit&amp;section=3" title="부분 편집: 창당">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p><a href="/wiki/2002%EB%85%84" title="2002년">2002년</a>의 대선 즈음에, 당시 현직 대통령이었던 <a href="/wiki/%EC%9E%90%ED%81%AC_%EC%8B%9C%EB%9D%BC%ED%81%AC" title="자크 시라크">시라크</a>의 지지자들은 세 개의 우익 정당으로 갈라져 있는 상태였다. 이들은 시라크의 재선을 위해 <b>행동하는 연대</b>(<i>Union en mouvement</i>)를 조직한다. 시라크의 재집권 성공 이후에는 <a href="/wiki/2002%EB%85%84" title="2002년">2002년</a>의 총선을 위해, 이들을 중심으로 다시 <b>여권</b>(與權) <b>강화를 위한 연대</b>(<i>Union pour la majorité présidentielle</i>)가 조직된다. 이 조직은 몇 달 후에 이름을 <b>대중운동연합</b>으로 바꾸고 단순히 자크 시라크 지지자들을 아우르던 <a class="new" href="/w/index.php?title=%EC%9A%B0%EC%82%B0_%EC%A1%B0%EC%A7%81&amp;action=edit&amp;redlink=1" title="우산 조직 (없는 문서)">우산 조직</a>에서 벗어나 항구적인 조직으로 거듭난다.</p>
<p>대중운동연합은 <a href="/wiki/%EB%93%9C%EA%B3%A8%EC%A3%BC%EC%9D%98" title="드골주의">드골</a>-<a href="/wiki/%EB%B3%B4%EC%88%98%EC%A3%BC%EC%9D%98" title="보수주의">보수주의</a> 정당인 <a href="/wiki/%EA%B3%B5%ED%99%94%EA%B5%AD%EC%97%B0%ED%95%A9" title="공화국연합">공화국연합</a> (RPR)과 <a class="mw-redirect" href="/wiki/%EC%9E%90%EC%9C%A0_%EB%B3%B4%EC%88%98%EC%A3%BC%EC%9D%98" title="자유 보수주의">자유 보수주의</a> 정당인 <a class="new" href="/w/index.php?title=%EC%9E%90%EC%9C%A0_%EB%AF%BC%EC%A3%BC%EC%A3%BC%EC%9D%98_(%ED%94%84%EB%9E%91%EC%8A%A4)&amp;action=edit&amp;redlink=1" title="자유 민주주의 (프랑스) (없는 문서)">자유 민주주의</a> (DL), 중도 성향의 정당인 <a class="new" href="/w/index.php?title=%ED%94%84%EB%9E%91%EC%8A%A4_%EB%AF%BC%EC%A3%BC%EB%8F%99%EB%A7%B9&amp;action=edit&amp;redlink=1" title="프랑스 민주동맹 (없는 문서)">프랑스 민주동맹</a> (UDF)의 상당수, 더 정확히는 <a class="new" href="/w/index.php?title=%ED%95%84%EB%A6%BD_%EB%91%90%EC%8A%A4%ED%8A%B8-%EB%B8%94%EB%9D%BC%EC%A7%80&amp;action=edit&amp;redlink=1" title="필립 두스트-블라지 (없는 문서)">필립 두스트-블라지</a>나 <a class="new" href="/w/index.php?title=%EC%9E%90%ED%81%AC_%EB%B0%94%EB%A1%9C&amp;action=edit&amp;redlink=1" title="자크 바로 (없는 문서)">자크 바로</a>와 같은 프랑스 민주동맹 내 기독교 민주주의자들을 흡수·합병하는 한편, <a href="/wiki/2002%EB%85%84" title="2002년">2002년</a>까지 민주동맹과 연합관계였던 <a href="/wiki/%EC%82%AC%ED%9A%8C%EC%9E%90%EC%9C%A0%EC%A3%BC%EC%9D%98" title="사회자유주의">사회자유주의</a> 성향의 <a href="/wiki/%EA%B8%89%EC%A7%84%EB%8B%B9_(%ED%94%84%EB%9E%91%EC%8A%A4)" title="급진당 (프랑스)">급진당</a>과 중도 성향의 정당인 <a class="new" href="/w/index.php?title=%ED%94%84%EB%9E%91%EC%8A%A4_%EB%AF%BC%EC%A3%BC%EB%8C%80%EC%A4%91%EB%8B%B9&amp;action=edit&amp;redlink=1" title="프랑스 민주대중당 (없는 문서)">프랑스 민주대중당</a>과도 통합하였다.</p>
<p>대중운동연합의 창당에는 <a href="/wiki/%EB%93%9C%EA%B3%A8%EC%A3%BC%EC%9D%98" title="드골주의">드골주의</a>, <a href="/wiki/%EC%9E%90%EC%9C%A0%EC%A3%BC%EC%9D%98" title="자유주의">자유주의</a> (프랑스 내에서는 <a href="/wiki/%EA%B3%B5%ED%99%94%EC%A3%BC%EC%9D%98" title="공화주의">공화주의</a>라 일컫기도 한다), <a href="/wiki/%EA%B8%B0%EB%8F%85%EA%B5%90_%EB%AF%BC%EC%A3%BC%EC%A3%BC%EC%9D%98" title="기독교 민주주의">기독교 민주주의</a> 그리고 <a href="/wiki/%EA%B8%89%EC%A7%84%EC%A3%BC%EC%9D%98" title="급진주의">급진주의</a> 등 네 개의 프랑스를 대표하는 정치철학이 만났다는 데에 의의가 있다.</p>
<p>당의 첫 이름('여권 강화를 위한 연대')이 의미하듯, 대중운동연합은 <a href="/wiki/%EC%9E%90%ED%81%AC_%EC%8B%9C%EB%9D%BC%ED%81%AC" title="자크 시라크">자크 시라크</a> 대통령의 정책을 일반적으로 지지하였다. 그러나 <a href="/wiki/2004%EB%85%84" title="2004년">2004년</a>에 대중운동연합은 시라크로부터 독립하려는 의미있는 움직임을 보이기 시작하였다. 대중운동연합의 당원들이 유권자들로부터 인기를 잃은 자크 시라크와 <a class="mw-redirect" href="/wiki/%EC%9E%A5_%ED%94%BC%EC%97%90%EB%A5%B4_%EB%9D%BC%ED%8C%8C%EB%9E%AD" title="장 피에르 라파랭">장 피에르 라파랭</a> 내각보다 시라크의 라이벌이었던 <a href="/wiki/%EB%8B%88%EC%BD%9C%EB%9D%BC_%EC%82%AC%EB%A5%B4%EC%BD%94%EC%A7%80" title="니콜라 사르코지">니콜라 사르코지</a>를 지지하고 나섰던 것이었다. 또한 시라크가 이전에 몇 번이나 공개적으로 지지한 <a href="/wiki/%ED%84%B0%ED%82%A4" title="터키">터키</a>의 <a href="/wiki/%EC%9C%A0%EB%9F%BD_%EC%97%B0%ED%95%A9" title="유럽 연합">유럽 연합</a> 가입에 대해 반대표를 던지기도 하였다.</p>
<h3><span class="mw-headline" id="2004.EB.85.84.EA.B9.8C.EC.A7.80">2004년까지</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%8C%80%EC%A4%91%EC%9A%B4%EB%8F%99%EC%97%B0%ED%95%A9&amp;action=edit&amp;section=4" title="부분 편집: 2004년까지">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>대중운동연합의 첫 당수였던 <a href="/wiki/%EC%95%8C%EB%9E%AD_%EC%A5%90%ED%8E%98" title="알랭 쥐페">알랭 쥐페</a>는 <a href="/wiki/%EC%9E%90%ED%81%AC_%EC%8B%9C%EB%9D%BC%ED%81%AC" title="자크 시라크">자크 시라크</a>의 최측근 중 하나였으나, <a href="/wiki/2004%EB%85%84" title="2004년">2004년</a> 1월에 있었던 부패 사건에 연루되어 같은 해 <a href="/wiki/7%EC%9B%94_15%EC%9D%BC" title="7월 15일">7월 15일</a>에 사임했다. <a href="/wiki/2004%EB%85%84" title="2004년">2004년</a> <a href="/wiki/11%EC%9B%94_29%EC%9D%BC" title="11월 29일">11월 29일</a>에는 <a href="/wiki/%EB%8B%88%EC%BD%9C%EB%9D%BC_%EC%82%AC%EB%A5%B4%EC%BD%94%EC%A7%80" title="니콜라 사르코지">니콜라 사르코지</a>가 당수직을 이어받겠다 선언하였으나 재무장관에 임명됨에 따라 사르코지의 당수직 승계는 없던 일이 되고 말았다.</p>
<p><a href="/wiki/2004%EB%85%84" title="2004년">2004년</a>의 프랑스 지방선거에서 대중운동연합은 <a href="/wiki/%EC%95%8C%EC%9E%90%EC%8A%A4" title="알자스">알자스</a>와 <a href="/wiki/%EC%BD%94%EB%A5%B4%EC%8B%9C%EC%B9%B4" title="코르시카">코르시카</a>를 포함하는 <a href="/wiki/%ED%94%84%EB%9E%91%EC%8A%A4_%EB%B3%B8%ED%86%A0" title="프랑스 본토">프랑스 본토</a>의 22개 선거구 중 단 2곳에서 겨우 승리하는 한편, 현(縣) 단위의 선거에서는 절반만을 건져 참패하고 만다. (이전에는 우익 정당들이 현 중 다수를 차지하고 있었다.) 이어 열린 <a href="/wiki/%EC%9C%A0%EB%9F%BD_%EC%9D%98%ED%9A%8C" title="유럽 의회">유럽 의회</a> 총선에서도 <a href="/wiki/%EC%82%AC%ED%9A%8C%EB%8B%B9_(%ED%94%84%EB%9E%91%EC%8A%A4)" title="사회당 (프랑스)">사회당</a>의 득표율에 크게 못미치는 16.6%를 획득하고, 의석에서도 고작 16석만을 얻는 등 참패하게 된다. <a href="/wiki/2005%EB%85%84" title="2005년">2005년</a> <a href="/wiki/5%EC%9B%94_25%EC%9D%BC" title="5월 25일">5월 25일</a>에 열린 <a href="/wiki/%EC%9C%A0%EB%9F%BD_%ED%97%8C%EB%B2%95" title="유럽 헌법">유럽 헌법</a> 인준을 위한 <a class="mw-redirect" href="/wiki/%EA%B5%AD%EB%AF%BC_%ED%88%AC%ED%91%9C" title="국민 투표">국민 투표</a>의 부결은 <a class="mw-redirect" href="/wiki/%EC%9E%A5_%ED%94%BC%EC%97%90%EB%A5%B4_%EB%9D%BC%ED%8C%8C%EB%9E%AD" title="장 피에르 라파랭">장 피에르 라파랭</a> 내각의 몰락을 가져와 UMP의 다른 정치인인 <a href="/wiki/%EB%8F%84%EB%AF%B8%EB%8B%88%ED%81%AC_%EB%93%9C_%EB%B9%8C%ED%8C%BD" title="도미니크 드 빌팽">도미니크 드 빌팽</a>의 새 내각이 구성되는 배경이 되기도 하였다.</p>
<p>그러나 <a href="/wiki/2007%EB%85%84" title="2007년">2007년</a> <a href="/wiki/4%EC%9B%94_22%EC%9D%BC" title="4월 22일">4월 22일</a>에 열린 프랑스 대선 1차투표에서 UMP의 후보로 나선 <a href="/wiki/%EB%8B%88%EC%BD%9C%EB%9D%BC_%EC%82%AC%EB%A5%B4%EC%BD%94%EC%A7%80" title="니콜라 사르코지">니콜라 사르코지</a>가 승리한다. 이어 <a href="/wiki/5%EC%9B%94_6%EC%9D%BC" title="5월 6일">5월 6일</a>에 열린 대선 2차투표에서는 <a href="/wiki/%EC%82%AC%ED%9A%8C%EB%8B%B9_(%ED%94%84%EB%9E%91%EC%8A%A4)" title="사회당 (프랑스)">사회당</a> 후보 <a href="/wiki/%EC%84%B8%EA%B3%A8%EB%A0%8C_%EB%A3%A8%EC%95%84%EC%96%84" title="세골렌 루아얄">세골렌 루아얄</a>을 맞아 53.1%를 획득해 대선에서 최종 승리하게 된다. 이에 따라, <a href="/wiki/%EB%8B%88%EC%BD%9C%EB%9D%BC_%EC%82%AC%EB%A5%B4%EC%BD%94%EC%A7%80" title="니콜라 사르코지">사르코지</a>는 대통령직에 오르기 이틀 전인 <a href="/wiki/5%EC%9B%94_14%EC%9D%BC" title="5월 14일">5월 14일</a> UMP의 당수직에서 물러난다. <a href="/wiki/6%EC%9B%94_17%EC%9D%BC" title="6월 17일">6월 17일</a>에 열린 총선에서 UMP는 하원 <a href="/wiki/%ED%94%84%EB%9E%91%EC%8A%A4_%EA%B5%AD%EB%AF%BC%EC%9D%98%ED%9A%8C" title="프랑스 국민의회">국민의회</a>의 총 의석수인 577석 중 313석을 차지하여 다시 원내 제1당으로 복귀하게 된다.</p>
<p><a href="/wiki/2008%EB%85%84" title="2008년">2008년</a> 3월에 열린 지방선거에서 대중운동연합은 다시 한 번 패배의 쓴 잔을 마신다. 시장선거에서 <a href="/wiki/%ED%88%B4%EB%A3%A8%EC%A6%88" title="툴루즈">툴루즈</a>나 <a href="/wiki/%EC%8A%A4%ED%8A%B8%EB%9D%BC%EC%8A%A4%EB%B6%80%EB%A5%B4" title="스트라스부르">스트라스부르</a> 등 다수의 선거구와 현(縣) 들에서 좌파들에게 내주었던 것이다.</p>
<p>이후, <a href="/wiki/%ED%94%84%EB%9E%91%EC%88%98%EC%95%84_%ED%94%BC%EC%9A%A9" title="프랑수아 피용">프랑수아 피용</a> 내각 아래서 사회연대부의 장관이었던 <a class="new" href="/w/index.php?title=%EC%9E%90%EB%B9%84%EC%97%90_%EB%B2%A0%ED%8A%B8%ED%8A%B8%EB%9E%91&amp;action=edit&amp;redlink=1" title="자비에 베트트랑 (없는 문서)">자비에 베트트랑</a>이 입각을 위해 당수직을 사임한 <a class="new" href="/w/index.php?title=%ED%8C%A8%ED%8A%B8%EB%A6%AD_%EB%93%9C%EB%B8%8C%EC%A7%80%EC%95%99&amp;action=edit&amp;redlink=1" title="패트릭 드브지앙 (없는 문서)">패트릭 드브지앙</a>을 계승하여 <a href="/wiki/2008%EB%85%84" title="2008년">2008년</a> 말 임시 당수에 오른다.</p>
<p><a href="/wiki/2009%EB%85%84" title="2009년">2009년</a>에 실시된 <a href="/wiki/%EC%9C%A0%EB%9F%BD_%EC%9D%98%ED%9A%8C" title="유럽 의회">유럽 의회</a> 총선에서 대중운동연합은 "여권의 강화" 시절처럼, 이제는 군소정당이 된 <a href="/wiki/%EA%B8%89%EC%A7%84%EB%8B%B9_(%ED%94%84%EB%9E%91%EC%8A%A4)" title="급진당 (프랑스)">급진당</a>과 <a class="new" href="/w/index.php?title=%EC%A4%91%EB%8F%84%EC%A2%8C%ED%8C%8C%EB%8B%B9_(%ED%94%84%EB%9E%91%EC%8A%A4)&amp;action=edit&amp;redlink=1" title="중도좌파당 (프랑스) (없는 문서)">중도좌파당</a>과 함께 선거 연합을 구성한다. 이들은 27.9%를 득표하고 29명의 유럽 의회 의원을 내는등 아주 좋은 결과를 냈다. 이 결과는 <a href="/wiki/2004%EB%85%84" title="2004년">2004년</a>에 비해 비약적으로 성장한 대중운동연합의 모습을 실감하게 했다.</p>
<p><a href="/wiki/2015" title="2015">2015</a>년 <a href="/wiki/5%EC%9B%94_30%EC%9D%BC" title="5월 30일">5월 30일</a> 해산되었고, <a href="/wiki/%EA%B3%B5%ED%99%94%EB%8B%B9_(%ED%94%84%EB%9E%91%EC%8A%A4)" title="공화당 (프랑스)">공화당</a>으로 제창당되었다.</p>
