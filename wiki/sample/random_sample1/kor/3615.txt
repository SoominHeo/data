<p><b>대통합민주신당</b>(大統合民主新黨, 약칭 : 대통합신당)은 <a href="/wiki/%EB%8C%80%ED%95%9C%EB%AF%BC%EA%B5%AD" title="대한민국">대한민국</a>의 중도개혁세력임을 주장하는 제 세력이, 민주·개혁·평화·중도·미래세력의 대통합이라는 명분으로 <a href="/wiki/2007%EB%85%84" title="2007년">2007년</a> <a href="/wiki/8%EC%9B%94_5%EC%9D%BC" title="8월 5일">8월 5일</a>에 창당한 정당이다. <a href="/wiki/%EC%97%B4%EB%A6%B0%EC%9A%B0%EB%A6%AC%EB%8B%B9" title="열린우리당">열린우리당</a> 탈당파 80명, 당시 <a class="mw-redirect" href="/wiki/%EC%A4%91%EB%8F%84%ED%86%B5%ED%95%A9%EB%AF%BC%EC%A3%BC%EB%8B%B9" title="중도통합민주당">중도통합민주당</a> 탈당파 4명, <a href="/wiki/%EC%86%90%ED%95%99%EA%B7%9C" title="손학규">손학규</a> 전 <a href="/wiki/%EA%B2%BD%EA%B8%B0%EB%8F%84%EC%A7%80%EC%82%AC" title="경기도지사">경기도지사</a>를 주축으로 한 <a class="mw-redirect" href="/wiki/%ED%95%9C%EB%82%98%EB%9D%BC%EB%8B%B9" title="한나라당">한나라당</a>의 일부 탈당세력과 시민사회세력을 주축으로 출범했다. 출범 직후 <a href="/wiki/%EB%AF%BC%EC%A3%BC%EB%8B%B9_(%EB%8C%80%ED%95%9C%EB%AF%BC%EA%B5%AD,_2007%EB%85%84)" title="민주당 (대한민국, 2007년)">민주당</a>은 유사한 당명이라며 비난하였다. 이후 <a href="/wiki/2007%EB%85%84" title="2007년">2007년</a> <a href="/wiki/8%EC%9B%94_20%EC%9D%BC" title="8월 20일">8월 20일</a> 신당 출범으로 상대적으로 약해진 <a href="/wiki/%EC%97%B4%EB%A6%B0%EC%9A%B0%EB%A6%AC%EB%8B%B9" title="열린우리당">열린우리당</a>과의 합당을 통해 의석수 143석으로 원내 제1당이 되었다. <a href="/wiki/2008%EB%85%84" title="2008년">2008년</a> <a href="/wiki/2%EC%9B%94_11%EC%9D%BC" title="2월 11일">2월 11일</a>에 <a href="/wiki/%EB%AF%BC%EC%A3%BC%EB%8B%B9_(%EB%8C%80%ED%95%9C%EB%AF%BC%EA%B5%AD,_2007%EB%85%84)" title="민주당 (대한민국, 2007년)">민주당</a>과 합당을 선언하고 <a href="/wiki/%EB%AF%BC%EC%A3%BC%EB%8B%B9_(%EB%8C%80%ED%95%9C%EB%AF%BC%EA%B5%AD,_2008%EB%85%84)" title="민주당 (대한민국, 2008년)">통합민주당</a>을 창당하여 소멸하였다.</p>

<h2><span class="mw-headline" id=".EC.A3.BC.EC.9A.94_.ED.99.9C.EB.8F.99">주요 활동</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%8C%80%ED%86%B5%ED%95%A9%EB%AF%BC%EC%A3%BC%EC%8B%A0%EB%8B%B9&amp;action=edit&amp;section=2" title="부분 편집: 주요 활동">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>대통합민주신당은 <a href="/wiki/%EC%97%B4%EB%A6%B0%EC%9A%B0%EB%A6%AC%EB%8B%B9" title="열린우리당">열린우리당</a>을 흡수한 후에 <a href="/wiki/%EC%A0%95%EB%8F%99%EC%98%81" title="정동영">정동영</a>을 대선 후보로 내세워 <a href="/wiki/%EB%8C%80%ED%95%9C%EB%AF%BC%EA%B5%AD_%EC%A0%9C17%EB%8C%80_%EB%8C%80%ED%86%B5%EB%A0%B9_%EC%84%A0%EA%B1%B0" title="대한민국 제17대 대통령 선거">제17대 대선</a>에 참여했다. 부단히 <a href="/wiki/%EC%B9%9C%EB%85%B8" title="친노">친노</a> 이미지를 벗기위한 명분없는 차별화를 시도했으나, 이에 대한 국민들에 대한 반감이 작용하여 대통령 선거에서는 <a class="mw-redirect" href="/wiki/%ED%95%9C%EB%82%98%EB%9D%BC%EB%8B%B9" title="한나라당">한나라당</a>의 <a href="/wiki/%EC%9D%B4%EB%AA%85%EB%B0%95" title="이명박">이명박</a> 후보에게 큰 표차로 패배하였다. 이로 인해 <a href="/wiki/%EC%B9%9C%EB%85%B8" title="친노">친노</a> 세력은 정치적으로 큰 타격을 입게 되었다. 대선 이후에는 <a href="/wiki/%EC%86%90%ED%95%99%EA%B7%9C" title="손학규">손학규</a>를 대표최고위원으로 선출하고 <a href="/wiki/%EC%9D%B4%EB%AA%85%EB%B0%95_%EC%A0%95%EB%B6%80" title="이명박 정부">이명박 정부</a>에 대한 견제를 지속하였다. 그 후 <a href="/wiki/%EB%AF%BC%EC%A3%BC%EB%8B%B9_(%EB%8C%80%ED%95%9C%EB%AF%BC%EA%B5%AD,_2007%EB%85%84)" title="민주당 (대한민국, 2007년)">민주당</a>과 합당하여 <a href="/wiki/%EB%AF%BC%EC%A3%BC%EB%8B%B9_(%EB%8C%80%ED%95%9C%EB%AF%BC%EA%B5%AD,_2008%EB%85%84)" title="민주당 (대한민국, 2008년)">통합민주당</a>을 창당하였다.</p>

<p>2007년 <a href="/wiki/8%EC%9B%94_20%EC%9D%BC" title="8월 20일">8월 20일</a>, 통합신당과 우리당 합당수임기관 합동회의는 열린우리당의 <strong class="selflink">대통합민주신당</strong> 흡수합당을 결의하였다.</p>
