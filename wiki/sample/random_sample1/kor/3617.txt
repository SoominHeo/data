<h2><span class="mw-headline" id=".EC.A4.84.EA.B1.B0.EB.A6.AC">줄거리</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%8C%80%ED%92%8D%EC%88%98&amp;action=edit&amp;section=1" title="부분 편집: 줄거리">원본 편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>극 중 <a href="/wiki/%EC%9E%90%EB%AF%B8%EC%9B%90%EA%B5%AD" title="자미원국">자미원국</a>(紫微垣局)은 왕실의 번영을 가져올 최고의 <a href="/wiki/%EB%AA%85%EB%8B%B9" title="명당">명당</a>으로, 모든 사건의 시작이자 중심이다. <a href="/wiki/%EC%9E%90%EB%AF%B8%EC%9B%90%EA%B5%AD" title="자미원국">자미원국</a>의 존재와 위치는 등장인물간의 극심한 갈등을 만든다.</p>
<p><a href="/wiki/%EA%B3%A0%EB%A0%A4" title="고려">고려</a>의 국무는 <a href="/wiki/%EC%9E%90%EB%AF%B8%EC%9B%90%EA%B5%AD" title="자미원국">자미원국</a>이 나타나리란 신탁을 받고 반드시 이를 찾아 고려왕실의 번영을 도모하여야 한다고 주장한다. 그러나 <a href="/wiki/%EC%9E%90%EB%AF%B8%EC%9B%90%EA%B5%AD" title="자미원국">자미원국</a>은 <a href="/wiki/%EA%B3%A0%EB%A0%A4" title="고려">고려</a>의 희망이자 두려움이기도 하다. <a href="/wiki/%EC%9B%90%EB%82%98%EB%9D%BC" title="원나라">원나라</a>의 지배를 받는 입장에서 고려왕실이 이를 찾는다는 사실만으로도 <a href="/wiki/%EC%9B%90%EB%82%98%EB%9D%BC" title="원나라">원나라</a>의 노여움을 살 수 있기 때문이다.</p>
<p>결국 <a class="mw-redirect" href="/wiki/%EA%B3%A0%EB%A0%A4%EC%8B%9C%EB%8C%80" title="고려시대">고려시대</a> <a href="/wiki/%ED%92%8D%EC%88%98" title="풍수">풍수</a>를 관장하는 <a href="/wiki/%EC%84%9C%EC%9A%B4%EA%B4%80" title="서운관">서운관</a>(書雲觀)의 일관 동륜이 비밀리에 <a href="/wiki/%EC%9E%90%EB%AF%B8%EC%9B%90%EA%B5%AD" title="자미원국">자미원국</a>을 찾아 떠난다. '<a href="/wiki/%ED%92%8D%EC%88%98" title="풍수">풍수</a>를 보는 신안을 가졌으면서도 입이 무거운 자'로서 동륜이 선발된 것이다.</p>
<p>동륜은 천신만고 끝에 <a href="/wiki/%EC%9E%90%EB%AF%B8%EC%9B%90%EA%B5%AD" title="자미원국">자미원국</a>을 발견하고 환호한다. 그런데 그가 <a class="mw-redirect" href="/wiki/%EA%B3%B5%EB%AF%BC%EC%99%95" title="공민왕">공민왕</a>을 위한 표지석을 묻기 위해 땅을 파다 발견한 것은 '이 땅은 50년 뒤에야 임자가 나타나니 절대 훼손하지 말라'는 예언과도 같은 석표(石標)였다. 결국 동륜은 끝까지 <a href="/wiki/%EC%9E%90%EB%AF%B8%EC%9B%90%EA%B5%AD" title="자미원국">자미원국</a>의 위치를 숨겼고, 이에 모진 고문을 당하며 10년이나 감옥에 갇히는 고초를 겪는다.</p>
<p>목동륜, 왕영지의 아들 목지상과 윤효명의 딸 윤해인 그리고 <a href="/wiki/%EC%9D%B4%EC%9D%B8%EC%9E%84" title="이인임">이인임</a> 아들 이정근의 삼각관계를 극중 중심 판도로 이끌면서 동륜과 영지 그리고 <a href="/wiki/%EC%9D%B4%EC%9D%B8%EC%9E%84" title="이인임">이인임</a>과 수련개의 극중 초반의 연적관계의 갈등으로 시작하여 왕영지를 두고 이정근과 목지상의 모자관계의 사랑과 반야와 목지상, 윤해인과 이정근 그리고 목지상 등의 사랑을 엮어냈다. 그리고 <a href="/wiki/%EC%9E%90%EB%AF%B8%EC%9B%90%EA%B5%AD" title="자미원국">자미원국</a>은 줄거리를 이끄는 등장인물의 구동력으로 작용할 뿐이었다.</p>
<p>그러나 목동륜의 인생을 수렁으로 몰아넣은 <a href="/wiki/%EC%9E%90%EB%AF%B8%EC%9B%90%EA%B5%AD" title="자미원국">자미원국</a>은 동륜을 연모한 왕족 왕영지의 삶까지 송두리째 바꿨다. 나아가 <a href="/wiki/%EC%9E%90%EB%AF%B8%EC%9B%90%EA%B5%AD" title="자미원국">자미원국</a>은 국무 수련개, <a href="/wiki/%EC%9D%B4%EC%9D%B8%EC%9E%84" title="이인임">이인임</a> 세력과 <a href="/wiki/%EC%A1%B0%EC%84%A0_%ED%83%9C%EC%A1%B0" title="조선 태조">이성계</a> 세력간 충돌의 이유가 돼 극 전체를 이끌어 가는 중심 줄거리가 된다. 결국 <a href="/wiki/%EC%9E%90%EB%AF%B8%EC%9B%90%EA%B5%AD" title="자미원국">자미원국</a>의 신탁에 의해 <a href="/wiki/%EC%A1%B0%EC%84%A0_%ED%83%9C%EC%A1%B0" title="조선 태조">이성계</a>는 <a class="mw-redirect" href="/wiki/%EB%AC%B4%ED%95%99%EB%8C%80%EC%82%AC" title="무학대사">무학대사</a>, <a href="/wiki/%EC%9D%B4%EC%A7%80%EB%9E%80" title="이지란">이지란</a>, <a href="/wiki/%EC%A0%95%EB%8F%84%EC%A0%84" title="정도전">정도전</a> 그리고 아들 <a href="/wiki/%EC%A1%B0%EC%84%A0_%ED%83%9C%EC%A2%85" title="조선 태종">이방원</a>과 함께 <a href="/wiki/%EC%A1%B0%EC%84%A0" title="조선">조선</a>이라는 새로운 나라를 건설하게 된다는 내용이다.</p>
<p><a class="mw-redirect" href="/wiki/%EA%B3%B5%EB%AF%BC%EC%99%95" title="공민왕">공민왕</a>, <a class="mw-redirect" href="/wiki/%EB%85%B8%EA%B5%AD%EA%B3%B5%EC%A3%BC" title="노국공주">노국공주</a>, <a class="mw-redirect" href="/wiki/%EB%AA%85%EB%8D%95%ED%83%9C%ED%9B%84" title="명덕태후">명덕태후</a>, <a href="/wiki/%EC%8B%A0%EB%8F%88" title="신돈">신돈</a>, <a class="mw-redirect" href="/wiki/%EC%9A%B0%EC%99%95" title="우왕">우왕</a>, <a href="/wiki/%EB%B0%98%EC%95%BC" title="반야">반야</a>를 비롯하여 <a href="/wiki/%EC%B5%9C%EC%98%81" title="최영">최영</a>, <a href="/wiki/%EC%9D%B4%EC%9D%B8%EC%9E%84" title="이인임">이인임</a>, <a href="/wiki/%EC%A1%B0%EB%AF%BC%EC%88%98" title="조민수">조민수</a>, <a href="/wiki/%EC%A0%95%EB%8F%84%EC%A0%84" title="정도전">정도전</a>, <a href="/wiki/%EC%9D%B4%EC%83%89" title="이색">이색</a>, <a href="/wiki/%EC%A0%95%EB%AA%BD%EC%A3%BC" title="정몽주">정몽주</a> 등 역사에 나오는 인물로 이야기를 이끌어가는 극중의 갈등을 조명하면서 목지상, 윤해인, 이정근, 목동륜, 왕영지, 윤효명, 수련개, 홍종대, 봉춘, 우야숙 등 가공인물들을 내세워 이야기를 엮어내었다.</p>
<p>대풍수는 이같이 국운이 쇠한 <a href="/wiki/%EA%B3%A0%EB%A0%A4" title="고려">고려</a> 말기 권력의 주변에 있던 도사들이 난세의 영웅인 <a href="/wiki/%EC%A1%B0%EC%84%A0_%ED%83%9C%EC%A1%B0" title="조선 태조">이성계</a>를 내세워 <a href="/wiki/%EC%A1%B0%EC%84%A0" title="조선">조선</a>을 건국하며 벌어지는 이야기를 다룬 <a href="/wiki/%ED%8C%A9%EC%85%98" title="팩션">팩션</a> <a href="/wiki/%EC%82%AC%EA%B7%B9" title="사극">사극</a>이다.</p>
