<div class="mw-body-content" id="bodyContent">

					이동:					<a href="#mw-head">둘러보기</a>, 					<a href="#p-search">검색</a>

<div class="mw-content-ltr" dir="ltr" id="mw-content-text" lang="ko"><script>function mfTempOpenSection(id){var block=document.getElementById("mf-section-"+id);block.className+=" open-block";block.previousSibling.className+=" open-block";}</script><p><b>대한민국농구협회</b>(大韓民國籠球協會)는 농구경기를 널리 보급하여 국민의 체력향상과 정신력을 배양함과 동시에 우수한경기자를 양성하여 국위선영 및 국민체육발전에 기여할 목적으로 <a href="/wiki/2005%EB%85%84" title="2005년">2005년</a> <a href="/wiki/9%EC%9B%94_14%EC%9D%BC" title="9월 14일">9월 14일</a> 설립 허가된 <a href="/wiki/%EB%8C%80%ED%95%9C%EB%AF%BC%EA%B5%AD_%EB%AC%B8%ED%99%94%EC%B2%B4%EC%9C%A1%EA%B4%80%EA%B4%91%EB%B6%80" title="대한민국 문화체육관광부">대한민국 문화체육관광부</a> 소관의 <a href="/wiki/%EC%82%AC%EB%8B%A8%EB%B2%95%EC%9D%B8" title="사단법인">사단법인</a>으로 <a href="/wiki/%EB%8C%80%ED%95%9C%EB%AF%BC%EA%B5%AD" title="대한민국">대한민국</a>의 아마추어 <a href="/wiki/%EB%86%8D%EA%B5%AC" title="농구">농구</a>를 총괄하는 단체이다. <a href="/wiki/%EC%9D%BC%EC%A0%9C_%EA%B0%95%EC%A0%90%EA%B8%B0" title="일제 강점기">일제 강점기</a>인 <a href="/wiki/1925%EB%85%84" title="1925년">1925년</a>에 <b>조선바스켓볼협회</b>라는 명칭으로 출발한 것이 연원이다. 사무실은 <a href="/wiki/%EC%84%9C%EC%9A%B8%ED%8A%B9%EB%B3%84%EC%8B%9C" title="서울특별시">서울특별시</a> <a href="/wiki/%EC%86%A1%ED%8C%8C%EA%B5%AC" title="송파구">송파구</a> 올림픽로 424 (<a href="/wiki/%EB%B0%A9%EC%9D%B4%EB%8F%99" title="방이동">방이동</a>, 편익시설A-3동)에 있다. 현 제32대 대한농구협회장은 <a href="/wiki/%EB%B0%A9%EC%97%B4_(%EB%86%8D%EA%B5%AC%EC%9D%B8)" title="방열 (농구인)">방열</a>(方烈) 회장이다.</p>

<h2><span class="mw-headline" id=".EC.A3.BC.EC.9A.94_.ED.99.9C.EB.8F.99">주요 활동</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%8C%80%ED%95%9C%EB%86%8D%EA%B5%AC%ED%98%91%ED%9A%8C&amp;action=edit&amp;section=3" title="부분 편집: 주요 활동">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>대한민국농구협회는 <a href="/wiki/%EB%86%8D%EA%B5%AC%EB%8C%80%EC%9E%94%EC%B9%98" title="농구대잔치">농구대잔치</a>와 전국남녀종별선수권대회를 매년 개최하고, <a class="new" href="/w/index.php?title=%EC%95%84%EC%8B%9C%EC%95%84%EB%86%8D%EA%B5%AC%EC%84%A0%EC%88%98%EA%B6%8C%EB%8C%80%ED%9A%8C&amp;action=edit&amp;redlink=1" title="아시아농구선수권대회 (없는 문서)">아시아농구선수권대회</a>, <a class="mw-redirect" href="/wiki/%EC%84%B8%EA%B3%84%EB%86%8D%EA%B5%AC%EC%84%A0%EC%88%98%EA%B6%8C%EB%8C%80%ED%9A%8C" title="세계농구선수권대회">세계농구선수권대회</a>, <a href="/wiki/%EC%98%AC%EB%A6%BC%ED%94%BD_%EB%86%8D%EA%B5%AC" title="올림픽 농구">올림픽 농구</a> 경기에 대표팀을 선발해 출전하고 있다.</p>
<p>국내외 아마추어 농구 경기대회를 개최, 참가하며, 산하연맹 및 각 시도지부의 지도, 관리하며 경기기술에 관한 연구 및 자료를 수집한다. 국제대회에 출전할 국가대표 선수와 국제회의 대표를 선정하여 파견하는 사업도 벌인다.</p>
