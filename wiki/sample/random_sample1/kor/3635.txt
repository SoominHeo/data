<h1 class="firstHeading" id="firstHeading" lang="ko">대한민국의 뉴라이트</h1>
<div class="mw-body-content" id="bodyContent">

<div class="mw-content-ltr" dir="ltr" id="mw-content-text" lang="ko"><script>function mfTempOpenSection(id){var block=document.getElementById("mf-section-"+id);block.className+=" open-block";block.previousSibling.className+=" open-block";}</script><div class="dablink"><a href="/wiki/%EC%9C%84%ED%82%A4%EB%B0%B1%EA%B3%BC:%EB%8F%99%EC%9D%8C%EC%9D%B4%EC%9D%98%EC%96%B4_%EB%AC%B8%EC%84%9C" title="위키백과:동음이의어 문서"><img alt="" data-file-height="200" data-file-width="260" height="18" src="//upload.wikimedia.org/wikipedia/commons/thumb/4/4a/Disambig_grey.svg/23px-Disambig_grey.svg.png" srcset="//upload.wikimedia.org/wikipedia/commons/thumb/4/4a/Disambig_grey.svg/35px-Disambig_grey.svg.png 1.5x, //upload.wikimedia.org/wikipedia/commons/thumb/4/4a/Disambig_grey.svg/46px-Disambig_grey.svg.png 2x" width="23"/></a> 이 문서는 대한민국의 뉴라이트에 관한 것입니다. <a href="/wiki/1980%EB%85%84%EB%8C%80" title="1980년대">1980년대</a>에 각국에서 일어난 신자유주의 혹은 신보수주의 운동에 대해서는 <a class="mw-redirect" href="/wiki/%EB%89%B4%EB%9D%BC%EC%9D%B4%ED%8A%B8" title="뉴라이트">뉴라이트</a> 문서를 참조하십시오.</div>
<p><b>대한민국의 뉴라이트</b>(New Right)는 '신흥 우파'를 표방하는 이념이며, <a href="/wiki/1991%EB%85%84" title="1991년">1991년</a> <a class="mw-redirect" href="/wiki/%EC%86%8C%EB%A0%A8_%EB%B6%95%EA%B4%B4" title="소련 붕괴">소련 붕괴</a> 이후 <a class="mw-redirect" href="/wiki/%EC%A3%BC%EC%82%AC%ED%8C%8C" title="주사파">주사파</a>에서 <a href="/wiki/%EC%9A%B0%EC%9D%B5" title="우익">우익</a>으로 전향한 <a href="/wiki/%EB%8C%80%ED%95%9C%EB%AF%BC%EA%B5%AD" title="대한민국">대한민국</a>의 정치 분파를 포괄적으로 뜻하는 용어이기도 하다.</p>
<p><a href="/wiki/2000%EB%85%84%EB%8C%80" title="2000년대">2000년대</a> 이후로 본격적인 활동을 시작하였으며, <a class="mw-redirect" href="/wiki/%EB%89%B4%EB%9D%BC%EC%9D%B4%ED%8A%B8" title="뉴라이트">뉴라이트</a>를 이끄는 상당수 인사들은 기존의 우파가 아닌 <a class="mw-redirect" href="/wiki/%EC%A2%8C%ED%8C%8C" title="좌파">사이비 좌파</a>와 <a class="mw-redirect" href="/wiki/%EC%A3%BC%EC%82%AC%ED%8C%8C" title="주사파">주사파</a> 등 <a href="/wiki/%EC%9A%B4%EB%8F%99%EA%B6%8C" title="운동권">운동권</a> 출신에서 전향을 한 사람들이 대부분으로, 운동권 출신의 뉴라이트 인사 중에는 주사파였던 인사가 많다.<sup class="reference" id="cite_ref-1"><a href="#cite_note-1">[1]</a></sup> 진보세력의 정치적 주도권 확보에 대한 반작용과 기존 보수층의 퇴행적 행태와 성격에 대한 자성 등을 내세운 것이 뉴라이트 그룹이 태동한 요인이다.<sup class="reference" id="cite_ref-2"><a href="#cite_note-2">[2]</a></sup></p>

<h2><span class="mw-headline" id=".EC.9D.B4.EB.85.90">이념</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%8C%80%ED%95%9C%EB%AF%BC%EA%B5%AD%EC%9D%98_%EB%89%B4%EB%9D%BC%EC%9D%B4%ED%8A%B8&amp;action=edit&amp;section=1" title="부분 편집: 이념">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>대한민국의 뉴라이트는 <a href="/wiki/%EC%8B%A0%EC%9E%90%EC%9C%A0%EC%A3%BC%EC%9D%98" title="신자유주의">신자유주의</a>, <a href="/wiki/%EC%8B%9D%EB%AF%BC%EC%82%AC%EA%B4%80" title="식민사관">식민사관</a>, <a href="/wiki/%EC%82%AC%ED%9A%8C%EC%A7%84%ED%99%94%EB%A1%A0" title="사회진화론">사회진화론</a> 등을 표방한다. 이들은 서구와 미국의 신우파 운동과 다른 출발점에 서 있다.<sup class="reference" id="cite_ref-3"><a href="#cite_note-3">[3]</a></sup> 미국과 서구가 복지국가가 누적시킨 문제를 해결하고자 했다면 뉴라이트는 구보수와 구진보의 낡은 이념과 극단적 대립을 주요한 극복대상으로 삼아 시대로 나아가고자 한다고 주장한다.<sup class="reference" id="cite_ref-4"><a href="#cite_note-4">[4]</a></sup></p>
<p>과거의 보수와 달리 <a href="/wiki/%EC%9E%91%EC%9D%80_%EC%A0%95%EB%B6%80" title="작은 정부">작은 정부</a>를 추구해 국가-시장간의 관계에 등에 있어서는 상당한 차별화를 이루었다고 평가받고 있으나, 대북인식과 대미인식에는 구체적인 차별성을 발견하기 힘들다는 비판이 있다 <sup class="reference" id="cite_ref-5"><a href="#cite_note-5">[5]</a></sup></p>
<p>그러나 국민의 정부와 참여정부가 신자유주의적 경제정책을 다수 시행하면서 뉴라이트가 내세운 경제적 차별화 또한 의미를 잃는다는 비판도 있다.<sup class="reference" id="cite_ref-6"><a href="#cite_note-6">[6]</a></sup></p>
<h2><span class="mw-headline" id=".EC.97.B0.ED.98.81">연혁</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%8C%80%ED%95%9C%EB%AF%BC%EA%B5%AD%EC%9D%98_%EB%89%B4%EB%9D%BC%EC%9D%B4%ED%8A%B8&amp;action=edit&amp;section=2" title="부분 편집: 연혁">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<h3><span class="mw-headline" id=".EB.8C.80.ED.95.9C.EB.AF.BC.EA.B5.AD_.EB.89.B4.EB.9D.BC.EC.9D.B4.ED.8A.B8_.EC.9A.B4.EB.8F.99.EC.9D.98_.EC.8B.9C.EC.9E.91">대한민국 뉴라이트 운동의 시작</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%8C%80%ED%95%9C%EB%AF%BC%EA%B5%AD%EC%9D%98_%EB%89%B4%EB%9D%BC%EC%9D%B4%ED%8A%B8&amp;action=edit&amp;section=3" title="부분 편집: 대한민국 뉴라이트 운동의 시작">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>2000년대 이후 일부 <a class="mw-redirect" href="/wiki/%EC%A3%BC%EC%82%AC%ED%8C%8C" title="주사파">주사파</a>와 <a class="mw-redirect" href="/wiki/%EC%A2%8C%ED%8C%8C" title="좌파">좌파</a> 중 전향자들에 의해 뉴라이트 운동이 시작되었으며 <a href="/wiki/2004%EB%85%84" title="2004년">2004년</a> 11월 자유주의연대를 출범으로 뉴라이트 운동이 본격화되기 시작했다. 자유주의연대는 <a href="/wiki/%EC%8B%A0%EC%A7%80%ED%98%B8_(%EC%A0%95%EC%B9%98%EC%9D%B8)" title="신지호 (정치인)">신지호</a> 대표와 <a href="/wiki/%ED%99%8D%EC%A7%84%ED%91%9C" title="홍진표">홍진표</a> 사무총장, <a href="/wiki/%EC%B5%9C%ED%99%8D%EC%9E%AC" title="최홍재">최홍재</a> 전 자유주의연대 조직위원장 등이 이끌어 가고 있다. (<a href="/wiki/2007%EB%85%84" title="2007년">2007년</a> 현재.) 또한 <a href="/wiki/2005%EB%85%84" title="2005년">2005년</a> 11월에는 <a href="/wiki/%EA%B9%80%EC%A7%84%ED%99%8D_(%EB%AA%A9%ED%9A%8C%EC%9E%90)" title="김진홍 (목회자)">김진홍</a> 대표가 이끄는 <a href="/wiki/%EB%89%B4%EB%9D%BC%EC%9D%B4%ED%8A%B8%EC%A0%84%EA%B5%AD%EC%97%B0%ED%95%A9" title="뉴라이트전국연합">뉴라이트전국연합</a>이 창설되어 보수와 진보의 틀을 넘어서는 일방적인 보수운동을 표방했다.</p>
<h3><span class="mw-headline" id=".EC.A0.95.EC.B9.98_.ED.99.9C.EB.8F.99">정치 활동</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%8C%80%ED%95%9C%EB%AF%BC%EA%B5%AD%EC%9D%98_%EB%89%B4%EB%9D%BC%EC%9D%B4%ED%8A%B8&amp;action=edit&amp;section=4" title="부분 편집: 정치 활동">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>이들은 보수와 진보를 넘어서는 실용적 중도를 주장하며 일부는 사실상 미국의 <a href="/wiki/%EC%8B%A0%EB%B3%B4%EC%88%98%EC%A3%BC%EC%9D%98" title="신보수주의">신보수주의</a>과 비견될만한 극단적 보수주의사상을 표방하기도 한다. 이들은 구 <a href="/wiki/%EB%B3%B4%EC%88%98%EC%A3%BC%EC%9D%98" title="보수주의">보수주의</a>의 <a href="/wiki/%EB%B0%98%EA%B3%B5%EC%A3%BC%EC%9D%98" title="반공주의">반공주의</a>나 <a href="/wiki/%EA%B6%8C%EC%9C%84%EC%A3%BC%EC%9D%98" title="권위주의">권위주의</a>를 비판함과 동시에 구 <a href="/wiki/%EC%A7%84%EB%B3%B4%EC%A3%BC%EC%9D%98" title="진보주의">진보주의</a>의 시대착오적 <a class="mw-redirect" href="/wiki/%EC%A2%8C%ED%8C%8C" title="좌파">좌파</a>정책을 넘어서고자 한다고 주장한다. 반면 이들이 내세우는 뉴라이트 교과서 포럼은 새로운 역사해석으로 논란이 되었다.<sup class="reference" id="cite_ref-7"><a href="#cite_note-7">[7]</a></sup></p>
<p>뉴라이트 계열 인사들 중 2008년 4월 국회의원 선거에서 정계진출을 시도하였으며 자유주의연대 대표 <a href="/wiki/%EC%8B%A0%EC%A7%80%ED%98%B8_(%EC%A0%95%EC%B9%98%EC%9D%B8)" title="신지호 (정치인)">신지호</a>, 뉴라이트경기안보연합 대표 <a href="/wiki/%EA%B9%80%EC%84%B1%ED%9A%8C" title="김성회">김성회</a>, 인천 남동을의 자유주의교육운동연합 대표 <a href="/wiki/%EC%A1%B0%EC%A0%84%ED%98%81" title="조전혁">조전혁</a> 등은 정계진출에 성공했다.<sup class="reference" id="cite_ref-8"><a href="#cite_note-8">[8]</a></sup></p>
<h2><span class="mw-headline" id=".EC.82.AC.EA.B1.B4_.EC.82.AC.EA.B3.A0">사건 사고</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%8C%80%ED%95%9C%EB%AF%BC%EA%B5%AD%EC%9D%98_%EB%89%B4%EB%9D%BC%EC%9D%B4%ED%8A%B8&amp;action=edit&amp;section=5" title="부분 편집: 사건 사고">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<h3><span class="mw-headline" id=".EC.97.AD.EC.82.AC.EA.B4.80_.EA.B4.80.EB.A0.A8">역사관 관련</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%8C%80%ED%95%9C%EB%AF%BC%EA%B5%AD%EC%9D%98_%EB%89%B4%EB%9D%BC%EC%9D%B4%ED%8A%B8&amp;action=edit&amp;section=6" title="부분 편집: 역사관 관련">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>뉴라이트 인사들의 발언과 관련하여 그들의 역사관이 논란이 되고 있다. 관련 인사들의 발언 가운데 논란이 되고 있는 주장은 다음과 같다.</p>
<ul>
<li><a href="/wiki/2008%EB%85%84" title="2008년">2008년</a> <a href="/wiki/%EC%95%88%EB%B3%91%EC%A7%81" title="안병직">안병직</a>은, '우리가 <a href="/wiki/%EC%9D%BC%EB%B3%B8" title="일본">일본</a>보다 꼭 법적, 사료적 증거가 많다고 할 수 없다'며 '<a href="/wiki/%EC%9D%BC%EB%B3%B8" title="일본">일본</a>은 <a href="/wiki/%EB%8F%85%EB%8F%84" title="독도">독도</a>를 자기 것이라고 주장할 법적, 사료적 근거가 있다'고 주장했다.<sup class="reference" id="cite_ref-9"><a href="#cite_note-9">[9]</a></sup></li>
<li><a href="/wiki/2009%EB%85%84" title="2009년">2009년</a> 뉴라이트전국연합 단체가 <a href="/wiki/%EC%B9%9C%EC%9D%BC%EC%9D%B8%EB%AA%85%EC%82%AC%EC%A0%84" title="친일인명사전">친일인명사전</a>발간을 '대한민국의 선진화를 저해하는 행위'라 주장했다.<sup class="reference" id="cite_ref-10"><a href="#cite_note-10">[10]</a></sup></li>
<li><a href="/wiki/2010%EB%85%84" title="2010년">2010년</a> <a href="/wiki/%EC%9D%B4%EC%98%81%EC%A1%B0" title="이영조">이영조</a>는, <a class="mw-redirect" href="/wiki/%EC%A0%9C%EC%A3%BC_4.3_%EC%82%AC%EA%B1%B4" title="제주 4.3 사건">제주 4.3 사건</a>을 폭동, <a class="mw-redirect" href="/wiki/%EA%B4%91%EC%A3%BC_5.18_%EB%AF%BC%EC%A3%BC%ED%99%94_%EC%9A%B4%EB%8F%99" title="광주 5.18 민주화 운동">광주 5.18 민주화 운동</a>을 민중반란이라 망언했다.<sup class="reference" id="cite_ref-11"><a href="#cite_note-11">[11]</a></sup></li>
<li><a href="/wiki/2013%EB%85%84" title="2013년">2013년</a> <a href="/wiki/%EC%9C%A0%EC%98%81%EC%9D%B5" title="유영익">유영익</a>은, 한국인은 '짐승같이 저열'하며, '도덕 수준이 낮아서' 일제의 식민지가 되었다고 주장했다.<sup class="reference" id="cite_ref-12"><a href="#cite_note-12">[12]</a></sup><sup class="reference" id="cite_ref-13"><a href="#cite_note-13">[13]</a></sup></li>
<li><a href="/wiki/2014%EB%85%84" title="2014년">2014년</a> <a class="mw-disambig" href="/wiki/%EC%9D%B4%EC%98%81%ED%9B%88" title="이영훈">이영훈</a> 교수는 '젊은이들에게 독재라는 말을 쓰면 안 된다고 교육해야 한다'고 주장했다.<sup class="reference" id="cite_ref-14"><a href="#cite_note-14">[14]</a></sup></li>
<li><a href="/wiki/2015%EB%85%84" title="2015년">2015년</a> <a class="mw-disambig" href="/wiki/%EC%9D%B4%EB%AA%85%ED%9B%88" title="이명훈">이명훈</a>은, 아베담화는 '진중하게 쓰인 훌륭한 문장'이라 주장했다.<sup class="reference" id="cite_ref-15"><a href="#cite_note-15">[15]</a></sup></li>
</ul>
<h3><span class="mw-headline" id=".EC.B9.9C.EC.9D.BC.EB.AA.85.EB.8B.A8_.EA.B3.B5.EA.B0.9C_.EB.B0.98.EB.8C.80">친일명단 공개 반대</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%8C%80%ED%95%9C%EB%AF%BC%EA%B5%AD%EC%9D%98_%EB%89%B4%EB%9D%BC%EC%9D%B4%ED%8A%B8&amp;action=edit&amp;section=7" title="부분 편집: 친일명단 공개 반대">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p><a href="/wiki/2008%EB%85%84" title="2008년">2008년</a> 뉴라이트계열 단체인 <a href="/wiki/%EB%89%B4%EB%9D%BC%EC%9D%B4%ED%8A%B8%EC%A0%84%EA%B5%AD%EC%97%B0%ED%95%A9" title="뉴라이트전국연합">뉴라이트전국연합</a>은 친일인명사전편찬위원회와 <a href="/wiki/%EB%AF%BC%EC%A1%B1%EB%AC%B8%EC%A0%9C%EC%97%B0%EA%B5%AC%EC%86%8C" title="민족문제연구소">민족문제연구소</a>가 <a href="/wiki/%EC%B9%9C%EC%9D%BC%EC%9D%B8%EB%AA%85%EC%82%AC%EC%A0%84" title="친일인명사전">친일인명사전</a>에 수록할 명단을 발표하자, 이는 '대한민국의 선진화를 저해하는 행위'라고 주장하며 반발하였다.<sup class="reference" id="cite_ref-16"><a href="#cite_note-16">[16]</a></sup></p>

<p><a href="/wiki/2012%EB%85%84" title="2012년">2012년</a> <a class="mw-redirect" href="/wiki/%EC%84%9C%EC%9A%B8%EC%A4%91%EC%95%99%EC%A7%80%EB%B2%95" title="서울중앙지법">서울중앙지법</a> 형사합의23부는, 뉴라이트계열 단체 <a class="new" href="/w/index.php?title=%EB%AF%BC%EC%83%9D%ED%8F%AC%EB%9F%BC&amp;action=edit&amp;redlink=1" title="민생포럼 (없는 문서)">민생포럼</a>의 대표 김범수가 <a href="/wiki/%EB%AF%B8%EC%86%8C%EA%B8%88%EC%9C%B5" title="미소금융">미소금융</a>의 자본을 개인적인 용도로 쓴 죄로 징역 5년을 선고하였다.<sup class="reference" id="cite_ref-17"><a href="#cite_note-17">[17]</a></sup></p>
