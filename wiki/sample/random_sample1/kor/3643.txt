<p><b>대한민국의 자유주의</b>는 <a href="/wiki/%EB%8C%80%ED%95%9C%EB%AF%BC%EA%B5%AD" title="대한민국">대한민국</a>의 <a href="/wiki/%EC%9E%90%EC%9C%A0%EC%A3%BC%EC%9D%98" title="자유주의">자유주의</a> 사상을 말한다. 광복이후, 대한민국의 자유주의자들은 좌우 대립이 심했기 때문에 <a href="/wiki/%EB%B3%B4%EC%88%98%EC%A3%BC%EC%9D%98" title="보수주의">보수주의</a> 또는 <a href="/wiki/%EB%AF%BC%EC%A1%B1%EC%A3%BC%EC%9D%98" title="민족주의">민족주의</a>와 동의어처럼 쓰였으며<sup class="reference" id="cite_ref-1"><a href="#cite_note-1">[1]</a></sup>, <a href="/wiki/%EC%9D%B4%EC%8A%B9%EB%A7%8C" title="이승만">이승만</a> 정권 후반에는 주로 이승만 독재정권에 반대하는 우익계열 <a href="/wiki/%EB%AF%BC%EC%A3%BC%ED%99%94" title="민주화">민주화</a> 운동가들을 일컫는다. 초기 <b>대한민국의 자유주의</b>자들은 <a href="/wiki/%EB%B0%98%EA%B3%B5%EC%A3%BC%EC%9D%98" title="반공주의">반공주의</a>자들이 대다수였기 때문에 <a href="/wiki/%EC%A1%B0%EC%84%A0%EB%AF%BC%EC%A3%BC%EC%A3%BC%EC%9D%98%EC%9D%B8%EB%AF%BC%EA%B3%B5%ED%99%94%EA%B5%AD" title="조선민주주의인민공화국">조선민주주의인민공화국</a>에 대해 부정적인 관점을 취하고 있었으나, 김대중 정부의 <a href="/wiki/%ED%96%87%EB%B3%95%EC%A0%95%EC%B1%85" title="햇볕정책">햇볕정책</a> 이후, 현재는 <a href="/wiki/%EC%A1%B0%EC%84%A0%EB%AF%BC%EC%A3%BC%EC%A3%BC%EC%9D%98%EC%9D%B8%EB%AF%BC%EA%B3%B5%ED%99%94%EA%B5%AD" title="조선민주주의인민공화국">조선민주주의인민공화국</a>에 대해 호의적인 시각으로 바뀌었고, <a href="/wiki/%ED%8F%89%ED%99%94%ED%86%B5%EC%9D%BC" title="평화통일">평화통일</a>을 지향한다.</p>

<h2><span class="mw-headline" id=".EC.97.AD.EC.82.AC">역사</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%8C%80%ED%95%9C%EB%AF%BC%EA%B5%AD%EC%9D%98_%EC%9E%90%EC%9C%A0%EC%A3%BC%EC%9D%98&amp;action=edit&amp;section=1" title="부분 편집: 역사">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>대한민국의 자유주의 사상은 구한말에 최초로 도입되었으며, 일제강점기의 자유주의자들은 주로 민족주의자들이였으며, 해방 이후에는 <a href="/wiki/%EB%8C%80%ED%95%9C%EB%AF%BC%EA%B5%AD%EC%9D%98_%EB%B3%B4%EC%88%98%EC%A3%BC%EC%9D%98" title="대한민국의 보수주의">보수주의 계열</a>과 타협하여, <a href="/wiki/%EA%B3%B5%EC%82%B0%EC%A3%BC%EC%9D%98" title="공산주의">공산주의</a>, <a href="/wiki/%EC%82%AC%ED%9A%8C%EC%A3%BC%EC%9D%98" title="사회주의">사회주의</a>세력을 탄압하였다.</p>

<h3><span class="mw-headline" id=".EA.B4.91.EB.B3.B5_.EC.A7.81.ED.9B.84">광복 직후</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%8C%80%ED%95%9C%EB%AF%BC%EA%B5%AD%EC%9D%98_%EC%9E%90%EC%9C%A0%EC%A3%BC%EC%9D%98&amp;action=edit&amp;section=3" title="부분 편집: 광복 직후">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>사실 광복직후 대한민국의 자유주의자들은 주로 <a href="/wiki/%EB%B0%98%EA%B3%B5%EC%A3%BC%EC%9D%98" title="반공주의">반공주의</a>성향이 가미된 사람들이 대부분이였거나, 또는 <a href="/wiki/%EB%B3%B4%EC%88%98%EC%A3%BC%EC%9D%98" title="보수주의">보수주의</a>자들과 타협한 사람들이 대부분이 였기 때문에 그들이 창당한 <a href="/wiki/%ED%95%9C%EA%B5%AD%EB%AF%BC%EC%A3%BC%EB%8B%B9" title="한국민주당">한국민주당</a>을 <a class="mw-redirect" href="/wiki/%EB%8C%80%ED%95%9C%EB%AF%BC%EA%B5%AD%EC%9D%98_%EC%9E%90%EC%9C%A0%EC%A3%BC%EC%9D%98_%EC%A0%95%EB%8B%B9" title="대한민국의 자유주의 정당">대한민국의 자유주의 정당</a>과 <a href="/wiki/%EB%8C%80%ED%95%9C%EB%AF%BC%EA%B5%AD%EC%9D%98_%EB%B3%B4%EC%88%98%EC%A0%95%EB%8B%B9" title="대한민국의 보수정당">보수주의 정당</a> 모두의 뿌리로 본다.</p>
<h3><span class="mw-headline" id=".ED.95.9C.EA.B5.AD.EC.A0.84.EC.9F.81.EA.B3.BC_.EC.A0.9C1.EA.B3.B5.ED.99.94.EA.B5.AD">한국전쟁과 제1공화국</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%8C%80%ED%95%9C%EB%AF%BC%EA%B5%AD%EC%9D%98_%EC%9E%90%EC%9C%A0%EC%A3%BC%EC%9D%98&amp;action=edit&amp;section=4" title="부분 편집: 한국전쟁과 제1공화국">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p><a class="mw-redirect" href="/wiki/%ED%95%9C%EA%B5%AD%EC%A0%84%EC%9F%81" title="한국전쟁">한국전쟁</a>이후, 이승만 정부시절때부터는 자유주의자들과 보수주의자들의 구분이 나타나기 시작하였다. 그러나 한편으로는 당시 자유주의자들은 이승만의 독재만 반대했을뿐, 보수주의적 색깔이 짙었다.</p>
