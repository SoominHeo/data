<p><b>대한항공 803편 추락사고</b>는 <a href="/wiki/1989%EB%85%84" title="1989년">1989년</a> <a href="/wiki/7%EC%9B%94_27%EC%9D%BC" title="7월 27일">7월 27일</a> <a href="/wiki/%EB%8C%80%ED%95%9C%EB%AF%BC%EA%B5%AD" title="대한민국">대한민국</a> <a href="/wiki/%EA%B9%80%ED%8F%AC%EA%B5%AD%EC%A0%9C%EA%B3%B5%ED%95%AD" title="김포국제공항">김포국제공항</a>을 이륙, <a href="/wiki/%ED%83%80%EC%9D%B4" title="타이">타이</a>의 <a href="/wiki/%EB%B0%A9%EC%BD%95" title="방콕">방콕</a> <a href="/wiki/%EB%8F%88%EB%AF%80%EC%95%99_%EA%B5%AD%EC%A0%9C%EA%B3%B5%ED%95%AD" title="돈므앙 국제공항">돈므앙 국제공항</a>과 <a href="/wiki/%EC%82%AC%EC%9A%B0%EB%94%94%EC%95%84%EB%9D%BC%EB%B9%84%EC%95%84" title="사우디아라비아">사우디아라비아</a>의 <a href="/wiki/%ED%82%B9_%EC%95%95%EB%91%98%EC%95%84%EC%A7%80%EC%A6%88_%EA%B5%AD%EC%A0%9C%EA%B3%B5%ED%95%AD" title="킹 압둘아지즈 국제공항">제다</a>를 거쳐 <a href="/wiki/%EB%A6%AC%EB%B9%84%EC%95%84" title="리비아">리비아</a>의 <a href="/wiki/%ED%8A%B8%EB%A6%AC%ED%8F%B4%EB%A6%AC_%EA%B5%AD%EC%A0%9C%EA%B3%B5%ED%95%AD" title="트리폴리 국제공항">트리폴리</a>에 도착할 예정이던 <a href="/wiki/%EB%8C%80%ED%95%9C%ED%95%AD%EA%B3%B5" title="대한항공">대한항공</a>의 803편(<a href="/wiki/%EB%A7%A5%EB%8F%84%EB%84%AC%EB%8D%94%EA%B8%80%EB%9F%AC%EC%8A%A4_DC-10" title="맥도넬더글러스 DC-10">맥도넬더글러스 DC-10</a> 여객기)이 악천후와 시야 미확보, <a href="/wiki/%EC%95%88%EA%B0%9C" title="안개">안개</a> 등의 무리한 착륙을 시도하다가 추락, 탑승객 72명이 사망한 사건이다.</p>

<p>사고의 원인은 시계 240m의 짙은 안개 속에서 조종사가 무리한 착륙시도를 하다 일어난 것이다.<sup class="reference" id="cite_ref-1"><a href="#cite_note-1">[1]</a></sup></p>
<h2><span class="mw-headline" id=".EC.82.AC.EA.B1.B4_.EA.B0.9C.EC.9A.94">사건 개요</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%8C%80%ED%95%9C%ED%95%AD%EA%B3%B5_803%ED%8E%B8_%EC%B6%94%EB%9D%BD_%EC%82%AC%EA%B3%A0&amp;action=edit&amp;section=2" title="부분 편집: 사건 개요">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>승객과 승무원 200명을 태운 <a href="/wiki/%EB%8C%80%ED%95%9C%ED%95%AD%EA%B3%B5" title="대한항공">대한항공</a> <a href="/wiki/%EB%A7%A5%EB%8F%84%EB%84%AC%EB%8D%94%EA%B8%80%EB%9F%AC%EC%8A%A4_DC-10" title="맥도넬더글러스 DC-10">DC-10</a> 여객기가 한국시간을 기준으로 이날 오후 2시 30분 경 추락하여 탑승객 72명이 사망하고 139명이 부상했다(리비아 현지주민도 포함).<sup class="reference" id="cite_ref-2"><a href="#cite_note-2">[2]</a></sup> 탑승객 대부분은 중동 현지에서 근로하는 근로자들로 대부분을 차지하고 있으며, <a href="/wiki/%ED%98%84%EB%8C%80%EA%B1%B4%EC%84%A4" title="현대건설">현대건설</a>과 <a href="/wiki/%EB%8C%80%EC%9A%B0%EA%B1%B4%EC%84%A4" title="대우건설">대우건설</a>, <a class="mw-redirect" href="/wiki/%EB%8F%99%EC%95%84%EA%B1%B4%EC%84%A4" title="동아건설">동아건설</a> (공영토건 포함) 등 총 150명이 탑승하였다.</p>
