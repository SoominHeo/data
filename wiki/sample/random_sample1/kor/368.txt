<h2><span class="mw-headline" id=".EB.B0.9C.EA.B2.AC.EA.B3.BC_.EB.AA.85.EB.AA.85">발견과 명명</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%ED%95%98%EC%9A%B0%EB%A9%94%EC%95%84_(%EC%99%9C%ED%96%89%EC%84%B1)&amp;action=edit&amp;section=1" title="부분 편집: 발견과 명명">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>하우메아는 <a href="/wiki/2004%EB%85%84" title="2004년">2004년</a> <a href="/wiki/%EB%AF%B8%EA%B5%AD" title="미국">미국</a> <a href="/wiki/%EC%BA%98%EB%A6%AC%ED%8F%AC%EB%8B%88%EC%95%84_%EA%B3%B5%EA%B3%BC%EB%8C%80%ED%95%99%EA%B5%90" title="캘리포니아 공과대학교">캘리포니아 공과대학교</a>의 <a class="mw-redirect mw-disambig" href="/wiki/%EB%A7%88%EC%9D%B4%ED%81%AC_%EB%B8%8C%EB%9D%BC%EC%9A%B4" title="마이크 브라운">마이크 브라운</a>(Mike Brown) 교수 팀과 <a href="/wiki/2005%EB%85%84" title="2005년">2005년</a> <a class="new" href="/w/index.php?title=%EC%98%A4%EB%A5%B4%ED%8B%B0%EC%8A%A4&amp;action=edit&amp;redlink=1" title="오르티스 (없는 문서)">오르티스</a>(J. L. Ortiz)가 이끄는 <a href="/wiki/%EC%8A%A4%ED%8E%98%EC%9D%B8" title="스페인">스페인</a>의 과학자들에 의해 발견되었다. 마이크 브라운 교수 팀이 먼저 발견했지만 그들은 발견 발표를 지연했기 때문에, 누가 먼저 발견했는지에 대해서는 논란이 있다. 스페인 과학자들이 이 천체의 발견을 <a href="/wiki/%EA%B5%AD%EC%A0%9C%EC%B2%9C%EB%AC%B8%EC%97%B0%EB%A7%B9" title="국제천문연맹">국제천문연맹</a>(IAU)에 발표했을 때, 천체의 임시 명칭은 (136108) 2003 EL<sub>61</sub>로 주어졌었다. 그러다가 <a href="/wiki/2008%EB%85%84" title="2008년">2008년</a> <a href="/wiki/9%EC%9B%94_17%EC%9D%BC" title="9월 17일">9월 17일</a>, <a href="/wiki/%EA%B5%AD%EC%A0%9C%EC%B2%9C%EB%AC%B8%EC%97%B0%EB%A7%B9" title="국제천문연맹">국제천문연맹</a>은 이 천체를 <a href="/wiki/%EC%99%9C%ED%96%89%EC%84%B1" title="왜행성">왜행성</a>으로 공식 분류하고, <a href="/wiki/%ED%95%98%EC%99%80%EC%9D%B4_%EC%A0%9C%EB%8F%84" title="하와이 제도">하와이</a> 신화 속의 풍요와 출산을 상징하는 여신의 이름인 하우메아의 이름을 따서 명명하게 된 것이다. 이로써 하우메아는 <a href="/wiki/%EC%84%B8%EB%A0%88%EC%8A%A4_(%EC%99%9C%ED%96%89%EC%84%B1)" title="세레스 (왜행성)">세레스</a>와 <a href="/wiki/%EB%AA%85%EC%99%95%EC%84%B1" title="명왕성">명왕성</a>, <a href="/wiki/%EC%97%90%EB%A6%AC%EC%8A%A4_(%EC%99%9C%ED%96%89%EC%84%B1)" title="에리스 (왜행성)">에리스</a>, <a href="/wiki/%EB%A7%88%EC%BC%80%EB%A7%88%EC%BC%80_(%EC%99%9C%ED%96%89%EC%84%B1)" title="마케마케 (왜행성)">마케마케</a>에 이어 태양계의 5번째 <a href="/wiki/%EC%99%9C%ED%96%89%EC%84%B1" title="왜행성">왜행성</a>으로 등록되었다.</p>
<h2><span class="mw-headline" id=".EA.B6.A4.EB.8F.84.EC.99.80_.EA.B3.B5.EC.A0.84">궤도와 공전</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%ED%95%98%EC%9A%B0%EB%A9%94%EC%95%84_(%EC%99%9C%ED%96%89%EC%84%B1)&amp;action=edit&amp;section=2" title="부분 편집: 궤도와 공전">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>하우메아는 <a class="mw-redirect" href="/wiki/%ED%81%90%EB%B9%84%EC%9B%90%EC%A1%B1" title="큐비원족">큐비원족</a>에 속하는 <a href="/wiki/%ED%95%B4%EC%99%95%EC%84%B1_%EB%B0%94%EA%B9%A5_%EC%B2%9C%EC%B2%B4" title="해왕성 바깥 천체">해왕성 바깥 천체</a>의 하나로 분류된다. 하우메아와 태양간의 현재 거리는 약 50 <a href="/wiki/%EC%B2%9C%EB%AC%B8%EB%8B%A8%EC%9C%84" title="천문단위">천문단위</a> 정도이지만 <a class="mw-redirect" href="/wiki/%EA%B7%BC%EC%9D%BC%EC%A0%90" title="근일점">근일점</a>일 때는 35 천문단위까지도 접근한다. <a class="mw-redirect" href="/wiki/%EA%B6%A4%EB%8F%84%EA%B2%BD%EC%82%AC" title="궤도경사">궤도경사</a>는 28°로, 명왕성의 17°에 비해 65% 가량 높다.</p>
<h2><span class="mw-headline" id=".EB.AC.BC.EB.A6.AC.EC.A0.81_.ED.8A.B9.EC.A7.95">물리적 특징</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%ED%95%98%EC%9A%B0%EB%A9%94%EC%95%84_(%EC%99%9C%ED%96%89%EC%84%B1)&amp;action=edit&amp;section=3" title="부분 편집: 물리적 특징">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>하우메아의 크기는 1960 × 1518 × 996 km 정도로 명왕성과 비슷하나 질량은 약 4.2 × 10<sup>21</sup> kg로 명왕성의 약 32% 정도이고 긴 타원형이다. 이 같은 형태를 띠게 된 것은 수십억 년 전 다른 천체와 충돌한 뒤 4시간 주기의 빠른 자전 속도를 갖게 되었기 때문이라고 분석되었다.</p>
<h2><span class="mw-headline" id=".EC.9C.84.EC.84.B1">위성</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%ED%95%98%EC%9A%B0%EB%A9%94%EC%95%84_(%EC%99%9C%ED%96%89%EC%84%B1)&amp;action=edit&amp;section=4" title="부분 편집: 위성">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>하우메아에는 2개의 <a href="/wiki/%EC%9C%84%EC%84%B1" title="위성">위성</a>이 있다. 충돌에 의해 하우메아로부터 떨어져 나간 것으로 보이는 이 두 위성은 하와이 신화 속 하우메아의 자식들의 이름인 “<a href="/wiki/%ED%9E%88%EC%9D%B4%EC%95%84%EC%B9%B4_(%EC%9C%84%EC%84%B1)" title="히이아카 (위성)">히이아카</a>(Hi'iaka)”와 “<a href="/wiki/%EB%82%98%EB%A7%88%EC%B9%B4_(%EC%9C%84%EC%84%B1)" title="나마카 (위성)">나마카</a>(Namaka)”로 각각 명명됐다. 위성들은 모두 <a href="/wiki/2005%EB%85%84" title="2005년">2005년</a>에 발견되었다.</p>
