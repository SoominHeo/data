<p><b>.cat</b>은 <a href="/wiki/%EC%9D%BC%EB%B0%98_%EC%B5%9C%EC%83%81%EC%9C%84_%EB%8F%84%EB%A9%94%EC%9D%B8" title="일반 최상위 도메인">일반 최상위 도메인</a>의 일종인 <a class="new" href="/w/index.php?title=%EC%8A%A4%ED%8F%B0%EC%84%9C_%EC%B5%9C%EC%83%81%EC%9C%84_%EB%8F%84%EB%A9%94%EC%9D%B8%EC%9D%98_%ED%95%98%EB%82%98&amp;action=edit&amp;redlink=1" title="스폰서 최상위 도메인의 하나 (없는 문서)">스폰서 최상위 도메인의 하나</a>로, <a href="/wiki/%EC%B9%B4%ED%83%88%EB%A3%A8%EB%83%90%EC%96%B4" title="카탈루냐어">카탈루냐어</a> 화자를 대상으로 한다. 사실상 일부 지역을 대상으로 하고 있으나, <a href="/wiki/%EA%B5%AD%EA%B0%80_%EC%BD%94%EB%93%9C_%EC%B5%9C%EC%83%81%EC%9C%84_%EB%8F%84%EB%A9%94%EC%9D%B8" title="국가 코드 최상위 도메인">국가 코드 최상위 도메인</a>은 아니다. 이 방침은 <a href="/wiki/ICANN" title="ICANN">ICANN</a>과 푼카트 재단(<span style="font-size: smaller;"><a href="/wiki/%EC%B9%B4%ED%83%88%EB%A3%A8%EB%83%90%EC%96%B4" title="카탈루냐어">카탈루냐어</a>:</span> <span lang="ca" xml:lang="ca">Fundació puntCAT</span>)에 의해 만들어졌다.<sup class="reference" id="cite_ref-1"><a href="#cite_note-1">[1]</a></sup></p>

<h2><span class="mw-headline" id=".EC.97.AD.EC.82.AC">역사</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=.cat&amp;action=edit&amp;section=1" title="부분 편집: 역사">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>.cat 도메인이 만들어지기 전에는 카탈루냐 민족의 기관·회사·개인은 어쩔 수 없이 <a href="/wiki/.es" title=".es">.es</a>, <a href="/wiki/.fr" title=".fr">.fr</a>, <a href="/wiki/.it" title=".it">.it</a>, <a href="/wiki/.ad" title=".ad">.ad</a>등의 도메인을 사용해 왔다. 그 예로 <a href="/wiki/%EC%B9%B4%ED%83%88%EB%A3%A8%EB%83%90_%EC%A7%80%EB%B0%A9" title="카탈루냐 지방">카탈루냐 지방</a> <a href="/wiki/%EC%A7%80%EB%A1%9C%EB%82%98" title="지로나">지로나</a> 시청의 주소는 <a href="/wiki/.gi" title=".gi">.gi</a>도메인을 사용한 <a class="external free" href="http://www.ajuntament.gi/" rel="nofollow">http://www.ajuntament.gi/</a> ("ajuntament"는 시청을 뜻한다)이다. 이 최상위 도메인은 <a href="/wiki/%EC%A7%80%EB%B8%8C%EB%A1%A4%ED%84%B0" title="지브롤터">지브롤터</a>의 것으로, 스페인의 도시가 지브롤터의 도메인을 사용한다는 것은 현상(現狀)을 인정한다는 의미로도 연결될 수 있어, 지브롤터의 주권을 주장해 온 스페인 정부를 곤혹하게 만들었다. 기묘하게도 이 등록은 카탈루냐 지방의 독립과 영국이 지브롤터의 영유권을 가지는 데에 반대하는 <a class="mw-redirect" href="/wiki/%EC%8A%A4%ED%8E%98%EC%9D%B8_%EC%82%AC%ED%9A%8C%EB%85%B8%EB%8F%99%EB%8B%B9" title="스페인 사회노동당">스페인 사회노동당</a>의 동맹격인 <a class="new" href="/w/index.php?title=%EC%B9%B4%ED%83%88%EB%A3%A8%EB%83%90_%EC%82%AC%ED%9A%8C%EB%8B%B9&amp;action=edit&amp;redlink=1" title="카탈루냐 사회당 (없는 문서)">카탈루냐 사회당</a> 사무소에서 시행한 일이다.</p>
<p>이 문제를 해결하고 카탈루냐어권 문화 인터넷 커뮤니티의 수요에 대응하기 위해 <a href="/wiki/2005%EB%85%84_9%EC%9B%94" title="2005년 9월">2005년 9월</a> .cat 도메인이 신설되었다. 이 커뮤니티는 사용자가 커뮤니티 내에서 카탈루냐어를 사용하기 위해서 사용하거나 다른 문화권의 사람들에게 카탈루냐의 문화를 알리기 위한 목적 등에 사용하기 위해 시작되었다. 최초 등록 기간은<a href="/wiki/2006%EB%85%84" title="2006년">2006년</a> <a href="/wiki/2%EC%9B%94_13%EC%9D%BC" title="2월 13일">2월 13일</a>부터 <a href="/wiki/4%EC%9B%94_21%EC%9D%BC" title="4월 21일">4월 21일</a>까지였으며, 같은 해 <a href="/wiki/4%EC%9B%94_23%EC%9D%BC" title="4월 23일">4월 23일</a> 일반 등록을 개시하였다.<sup class="reference" id="cite_ref-en_faq_2-0"><a href="#cite_note-en_faq-2">[2]</a></sup></p>
<h2><span class="mw-headline" id=".EA.B7.9C.EC.A0.9C">규제</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=.cat&amp;action=edit&amp;section=2" title="부분 편집: 규제">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>거점 사이트가 카탈루냐에 위치해 있으나, .cat 도메인에는 대상 지역의 한정은 없고, 모든 카탈루냐어 화자 커뮤니티를 대상으로 한다. 단, 카탈루냐 문화에 소속하는 개인이나 단체만 허용된다.<sup class="reference" id="cite_ref-en_faq_2-1"><a href="#cite_note-en_faq-2">[2]</a></sup></p>
