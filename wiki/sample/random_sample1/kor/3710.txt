<h2><span class="mw-headline" id=".EA.B0.9C.EC.9A.94">개요</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%8D%95_%EB%8B%A4%EC%A0%80%EC%8A%A4&amp;action=edit&amp;section=1" title="부분 편집: 개요">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>덕 다저스는 <a href="/wiki/%EB%A3%A8%EB%8B%88_%ED%88%B0" title="루니 툰">루니 툰</a>의 <a href="/wiki/%EB%8C%80%ED%94%BC_%EB%8D%95" title="대피 덕">대피 덕</a>을 주인공으로 한 에피소드 Duck Dodgers in 24 1/2 Century를 토대로 만든 리메이크 작품이다.</p>
<h2><span class="mw-headline" id=".EC.A4.84.EA.B1.B0.EB.A6.AC">줄거리</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%8D%95_%EB%8B%A4%EC%A0%80%EC%8A%A4&amp;action=edit&amp;section=2" title="부분 편집: 줄거리">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>21세기에 사고로 냉동이 된 오리 덕 다저스는 24세기에 잠에서 깨어나 자기 자신이 21세기의 히어로라고 사람들에게 속인다. 조수인 카뎃과 함께 우주방위국의 일원으로 활약하지만,화성제국의 X-2사령관이 그와 대치한다.</p>
<h2><span class="mw-headline" id=".EC.A3.BC.EC.9A.94_.EB.93.B1.EC.9E.A5.EC.9D.B8.EB.AC.BC">주요 등장인물</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%8D%95_%EB%8B%A4%EC%A0%80%EC%8A%A4&amp;action=edit&amp;section=3" title="부분 편집: 주요 등장인물">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<ul>
<li>덕 다저스</li>
</ul>
<p>성우 - <a class="new" href="/w/index.php?title=%EC%A1%B0_%EC%95%8C%EB%9D%BC%EC%8A%A4%ED%82%A4&amp;action=edit&amp;redlink=1" title="조 알라스키 (없는 문서)">조 알라스키</a></p>
<p>주인공.대피 덕이 연기하는 우주방위군의 함장. 함장이지만 약하고 게으르며 이기적이어서 항상 문제를 일으킨다. 또한 임무를 불성실하게 수행한다. 여자와 돈을 좋아한다. 첫 에피소드에서는 정찰임무를 제대로 수행하지 않고 지구가 화성제국의 침공에 무방비하게되는 사태가 일어났다.하지만 화성군대의 레이저포에 감자를 발사해 지구가 공격당하는 것을 저지했다. 항상 지구를 위험에서 구하지만 그 결과로 자기 자신이나 주변인물들이 좋지 않은 꼴이 되는 경우가 많다. 21세기에 냉동되기 전에는 대학의 풋볼팀에서 워터보이로 활동했었다.</p>
<ul>
<li>스페이스 카뎃</li>
</ul>
<p>성우- 밥 버겐</p>
<p>포키 피그가 연기하는 다저스의 조수.다저스가 일으키는 문제에 휘둘리는 일이 잦다.다저스보다 유능하기 때문에 사실상 대부분의 업무는 그가 맡고 있다.</p>
<ul>
<li>I.Q. 하이 박사</li>
</ul>
<p>성우-리차드 맥고나글</p>
<p>우주방위국 소속의 박사. 다저스를 해동시킨 장본인이자 그의 상사이다. 머리에 전구를 달고 있다. 개인적으로 실험이나 발명을 하고 있어 그의 창고에는 여러 발명품들이 있으며 가끔 다저스가 일부 훔쳐가기도 한다.</p>
<ul>
<li>X-2 사령관</li>
</ul>
<p>성우- 조 알라스키</p>
<p><a href="/wiki/%ED%99%94%EC%84%B1%EC%9D%B8_%EB%A7%88%EB%B9%88" title="화성인 마빈">화성인 마빈</a>이 연기하는 화성제국군대의 총사령관. 센트리온이라는 로봇군대를 지휘하고 있다. 다저스와는 숙적관계이지만 공동의 적이 나타났을 경우에는 다저스 혹은 카뎃과 협동작전을 수행하기도 한다. 티라니 여왕을 짝사랑하여 그녀가 관심을 보이는 다저스를 질투하고 있다.</p>
<ul>
<li>K-9</li>
</ul>
<p>성우- <a href="/wiki/%ED%94%84%EB%9E%AD%ED%81%AC_%EC%9B%B0%EC%BB%A4" title="프랭크 웰커">프랭크 웰커</a></p>
<p>X-2 사령관의 직속부하인 <a href="/wiki/%EA%B0%9C" title="개">개</a>.루니툰 본편에도 등장하지만 이 작품에서 유일하게 이름이 달라지지 않았다.</p>
<ul>
<li>센트리온</li>
</ul>
<p>성우- <a href="/wiki/%EB%A7%88%EC%9D%B4%ED%81%B4_%EB%8F%88" title="마이클 돈">마이클 돈</a></p>
<p>X-2사령관의 부하인 로봇들. 공중에 떠다니며 손에 레이저 포를 장착하고 있다.</p>
<ul>
<li>티라니 여왕</li>
</ul>

<h2><span class="mw-headline" id=".EC.A3.BC.EC.A0.9C.EA.B0.80">주제가</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%8D%95_%EB%8B%A4%EC%A0%80%EC%8A%A4&amp;action=edit&amp;section=5" title="부분 편집: 주제가">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>이 작품의 주제가는 <a class="mw-disambig" href="/wiki/%ED%86%B0_%EC%A1%B4%EC%8A%A4" title="톰 존스">톰 존스</a>가 불렀으며 <a class="new" href="/w/index.php?title=%EB%8D%94_%ED%94%8C%EB%A0%88%EB%B0%8D_%EB%A6%BD%EC%8A%A4&amp;action=edit&amp;redlink=1" title="더 플레밍 립스 (없는 문서)">더 플레밍 립스</a>가 어레인지 했다. 존스는 또한 시즌2의 에피소드 <i>Talent Show A Go-Go</i>에 게스트 성우로 출연해 그의 대표곡 <i>It's Not Unusual</i>을 불렀다.</p>
<h2><span class="mw-headline" id=".EC.88.98.EC.83.81">수상</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%8D%95_%EB%8B%A4%EC%A0%80%EC%8A%A4&amp;action=edit&amp;section=6" title="부분 편집: 수상">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>덕 다저스는 2004년 <a class="mw-redirect" href="/wiki/%EC%95%A0%EB%8B%88_%EC%83%81" title="애니 상">애니 상</a> <i>Outstanding Achievement in an Animated Television Production Produced For Children</i>부문에 노미네이트 되었고 작품의 음악,프로덕션 디자인,목소리 연기 부문 또한 노미네이트 되었다. 2004년 덕 다저스는 음악 부문에서 애니 상을 수상했다.</p>
<p>또한 덕 다저스는 2004년 <a class="mw-redirect" href="/wiki/%EC%97%90%EB%AF%B8_%EC%83%81" title="에미 상">에미 상</a> Outstanding Sound Editing 부문, Live Action and Animation and Special Class Animated Program 부문에 노미네이트 되었고 2005년 각각 다시 같은 부문에 노미네이트 되었다.같은해 이 작품은 Outstanding Performer in an Animated Program(조 알라스키)에서 수상했다.</p>
