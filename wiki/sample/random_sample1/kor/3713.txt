<p><b>덕양구</b>(德陽區)는 <a href="/wiki/%EA%B2%BD%EA%B8%B0%EB%8F%84" title="경기도">경기도</a> <a href="/wiki/%EA%B3%A0%EC%96%91%EC%8B%9C" title="고양시">고양시</a> 동부에 있는 <a href="/wiki/%EA%B5%AC_(%ED%96%89%EC%A0%95_%EA%B5%AC%EC%97%AD)" title="구 (행정 구역)">구</a>이다. <a href="/wiki/%EB%B6%81%ED%95%9C%EC%82%B0" title="북한산">북한산</a>과 <a href="/wiki/%ED%95%9C%EA%B0%95" title="한강">한강</a>으로 대표되는 자연 환경, <a class="mw-redirect" href="/wiki/%EC%84%B8%EA%B3%84%EB%AC%B8%ED%99%94%EC%9C%A0%EC%82%B0" title="세계문화유산">세계문화유산</a>인 <a href="/wiki/%EC%84%9C%EC%82%BC%EB%A6%89" title="서삼릉">서삼릉</a>과 <a href="/wiki/%EC%84%9C%EC%98%A4%EB%A6%89" title="서오릉">서오릉</a>, <a href="/wiki/%ED%96%89%EC%A3%BC%EC%82%B0%EC%84%B1" title="행주산성">행주산성</a> 등의 문화재로 유명한 곳이다. 구의 명칭은 덕양구 지역의 옛 이름인 '덕양(德陽)현'에서 따 왔으며, 행주산성이 <a class="new" href="/w/index.php?title=%EB%8D%95%EC%96%91%EC%82%B0&amp;action=edit&amp;redlink=1" title="덕양산 (없는 문서)">덕양산</a>에 있다.</p>

<h2><span class="mw-headline" id=".EC.97.AD.EC.82.AC">역사</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%8D%95%EC%96%91%EA%B5%AC&amp;action=edit&amp;section=1" title="부분 편집: 역사">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<div class="rellink boilerplate seealso"><img alt="" data-file-height="128" data-file-width="128" height="16" src="//upload.wikimedia.org/wikipedia/commons/thumb/e/ec/Crystal_Clear_app_xmag.svg/16px-Crystal_Clear_app_xmag.svg.png" srcset="//upload.wikimedia.org/wikipedia/commons/thumb/e/ec/Crystal_Clear_app_xmag.svg/24px-Crystal_Clear_app_xmag.svg.png 1.5x, //upload.wikimedia.org/wikipedia/commons/thumb/e/ec/Crystal_Clear_app_xmag.svg/32px-Crystal_Clear_app_xmag.svg.png 2x" width="16"/> 행정구역의 역사는 <a href="/wiki/%EB%8D%95%EC%96%91%EA%B5%AC%EC%9D%98_%ED%96%89%EC%A0%95_%EA%B5%AC%EC%97%AD#.EC.97.B0.ED.98.81" title="덕양구의 행정 구역">덕양구의 행정 구역#연혁</a> 문서를 참고하십시오.</div>
<ul>
<li><a href="/wiki/%EA%B3%A0%EA%B5%AC%EB%A0%A4" title="고구려">고구려</a> : 개백현, 달을성현</li>
<li><a class="mw-redirect" href="/wiki/%EC%8B%A0%EB%9D%BC%EC%8B%9C%EB%8C%80" title="신라시대">신라시대</a> : 고봉현, 무왕현</li>
<li><a href="/wiki/1413%EB%85%84" title="1413년">1413년</a> <a href="/wiki/%EC%A1%B0%EC%84%A0_%ED%83%9C%EC%A2%85" title="조선 태종">조선 태종</a> 13년 : 덕양현</li>
</ul>
<p>본디 ‘덕양현’으로 불렸다. <a href="/wiki/1413%EB%85%84" title="1413년">1413년</a> <a href="/wiki/%EC%A1%B0%EC%84%A0_%ED%83%9C%EC%A2%85" title="조선 태종">조선 태종</a> 13년에 <a class="new" href="/w/index.php?title=%EA%B3%A0%EB%B4%89%ED%98%84&amp;action=edit&amp;redlink=1" title="고봉현 (없는 문서)">고봉현</a>과 덕양현의 글자를 각각 따서 <a href="/wiki/%EA%B3%A0%EC%96%91%EA%B5%B0" title="고양군">고양군</a>이 설립되었다. 그 뒤, <a href="/wiki/%EC%A1%B0%EC%84%A0_%EC%97%B0%EC%82%B0%EA%B5%B0" title="조선 연산군">조선 연산군</a> 때에는 왕의 사냥터로 바뀌어 주민들이 모두 소개(疏開)되기도 하였다.</p>
<ul>
<li><a href="/wiki/1471%EB%85%84" title="1471년">1471년</a> <a href="/wiki/%EC%A1%B0%EC%84%A0_%EC%84%B1%EC%A2%85" title="조선 성종">조선 성종</a> 2년 : 고양군으로 승격하였다.</li>
<li><a href="/wiki/1992%EB%85%84" title="1992년">1992년</a> <a href="/wiki/2%EC%9B%94_1%EC%9D%BC" title="2월 1일">2월 1일</a> : <a href="/wiki/%EA%B3%A0%EC%96%91%EC%8B%9C" title="고양시">고양시</a>로 승격하였다.</li>
<li><a href="/wiki/1996%EB%85%84" title="1996년">1996년</a> <a href="/wiki/3%EC%9B%94_1%EC%9D%BC" title="3월 1일">3월 1일</a> : 덕양구를 설치하였다.</li>
<li><a href="/wiki/1996%EB%85%84" title="1996년">1996년</a> <a href="/wiki/10%EC%9B%94_21%EC%9D%BC" title="10월 21일">10월 21일</a> : <a class="mw-disambig" href="/wiki/%ED%99%94%EC%A0%95%EB%8F%99" title="화정동">화정동</a>을 화정1동, 2동으로 분동하였다.</li>
<li><a href="/wiki/2003%EB%85%84" title="2003년">2003년</a> <a href="/wiki/2%EC%9B%94_3%EC%9D%BC" title="2월 3일">2월 3일</a> : <a href="/wiki/%ED%96%89%EC%8B%A0%EB%8F%99" title="행신동">행신1동</a>과 <a href="/wiki/%ED%96%89%EC%8B%A0%EB%8F%99" title="행신동">2동</a>에서 행신3동을 분동하였다.</li>
<li><a href="/wiki/2015%EB%85%84" title="2015년">2015년</a> <a href="/wiki/4%EC%9B%94_22%EC%9D%BC" title="4월 22일">4월 22일</a> : 덕양구청에 고양시 덕양여권민원실이 설치되었다. 이로써 고양시는 국내 최초로 한 기초자치단체에 두 여권민원실을 두게 되었다.</li>

<h2><span class="mw-headline" id=".ED.96.89.EC.A0.95_.EA.B5.AC.EC.97.AD">행정 구역</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%8D%95%EC%96%91%EA%B5%AC&amp;action=edit&amp;section=2" title="부분 편집: 행정 구역">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<div class="rellink relarticle mainarticle"><img alt="" data-file-height="128" data-file-width="128" height="16" src="//upload.wikimedia.org/wikipedia/commons/thumb/e/ec/Crystal_Clear_app_xmag.svg/16px-Crystal_Clear_app_xmag.svg.png" srcset="//upload.wikimedia.org/wikipedia/commons/thumb/e/ec/Crystal_Clear_app_xmag.svg/24px-Crystal_Clear_app_xmag.svg.png 1.5x, //upload.wikimedia.org/wikipedia/commons/thumb/e/ec/Crystal_Clear_app_xmag.svg/32px-Crystal_Clear_app_xmag.svg.png 2x" width="16"/> 이 부분의 본문은 <a href="/wiki/%EB%8D%95%EC%96%91%EA%B5%AC%EC%9D%98_%ED%96%89%EC%A0%95_%EA%B5%AC%EC%97%AD" title="덕양구의 행정 구역">덕양구의 행정 구역</a>입니다.</div>
<p>덕양구의 행정 구역은 32개의 <a href="/wiki/%EB%B2%95%EC%A0%95%EB%8F%99" title="법정동">법정동</a>을 19개의 <a class="mw-redirect" href="/wiki/%ED%96%89%EC%A0%95%EB%8F%99" title="행정동">행정동</a>으로 나누어 관리를 하고 있으며, 19동 397통 2,387반으로 구성되어 있다. 덕양구의 면적은 165.51㎢로 시의 61.8%에 해당되고, 인구는 <a href="/wiki/2014%EB%85%84" title="2014년">2014년</a> <a href="/wiki/12%EC%9B%94" title="12월">12월</a> 말 기준으로 42만4423 명, 16만6253 세대이다. 행신3동은 5만 명 이상이 거주하고 있으며, 화정1동도 4만 명 이상이 거주하고 있다.<sup class="reference" id="cite_ref-3"><a href="#cite_note-3">[3]</a></sup></p>
