<h2><span class="mw-headline" id="2006.EB.85.84_3.EC.9B.94_25.EC.9D.BC"><a href="/wiki/2006%EB%85%84" title="2006년">2006년</a> <a href="/wiki/3%EC%9B%94_25%EC%9D%BC" title="3월 25일">3월 25일</a></span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=2006%EB%85%84_3%EC%9B%94&amp;action=edit&amp;section=1" title="부분 편집: 2006년 3월 25일">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<ul>
<li>다음달 2일 총선을 앞두고 <a href="/wiki/%ED%83%80%EC%9D%B4" title="타이">타이</a> 수도 <a href="/wiki/%EB%B0%A9%EC%BD%95" title="방콕">방콕</a>에서 <a href="/wiki/%ED%83%81%EC%8B%A0_%EC%B9%9C%EB%82%98%EC%99%93" title="탁신 친나왓">탁신 친나왓</a> 총리의 사임을 요구하는 100만 명 이상이 참여한 대규모 시위가 벌어졌다.</li>
</ul>
<h2><span class="mw-headline" id="2006.EB.85.84_3.EC.9B.94_24.EC.9D.BC"><a href="/wiki/2006%EB%85%84" title="2006년">2006년</a> <a href="/wiki/3%EC%9B%94_24%EC%9D%BC" title="3월 24일">3월 24일</a></span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=2006%EB%85%84_3%EC%9B%94&amp;action=edit&amp;section=2" title="부분 편집: 2006년 3월 24일">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<ul>
<li><a href="/wiki/%EB%85%B8%EB%AC%B4%ED%98%84" title="노무현">노무현</a> 대통령은 <a href="/wiki/%EC%9D%B4%ED%95%B4%EC%B0%AC" title="이해찬">이해찬</a> 전<a class="mw-redirect" href="/wiki/%EB%8C%80%ED%95%9C%EB%AF%BC%EA%B5%AD%EC%9D%98_%EC%B4%9D%EB%A6%AC" title="대한민국의 총리">총리</a> 후임으로 <a href="/wiki/%EC%97%B4%EB%A6%B0%EC%9A%B0%EB%A6%AC%EB%8B%B9" title="열린우리당">열린우리당</a> <a href="/wiki/%ED%95%9C%EB%AA%85%EC%88%99" title="한명숙">한명숙</a> 의원을 지명했다.</li>
<li><a href="/wiki/%EC%A0%95%EC%A7%84%EC%84%9D_(%EC%B6%94%EA%B8%B0%EA%B2%BD)" title="정진석 (추기경)">정진석</a> 추기경이 <a class="mw-redirect" href="/wiki/%EB%B0%94%ED%8B%B0%EC%B9%B8" title="바티칸">바티칸</a>에서 열린 서임 예식에서 한국인으로는 두 번째로 <a href="/wiki/%EA%B0%80%ED%86%A8%EB%A6%AD" title="가톨릭">가톨릭교회</a> 추기경에 공식 임명됐다.</li>
</ul>

<li><a href="/wiki/%ED%94%84%EB%9E%91%EC%8A%A4" title="프랑스">프랑스</a>에서 정부에서 추진 중인 청년 실업 정책인 <a href="/wiki/%EC%B5%9C%EC%B4%88%EA%B3%A0%EC%9A%A9%EA%B3%84%EC%95%BD" title="최초고용계약">최초고용계약</a>(CPE) 제도의 취소를 요구하며 전국적으로 수십만 명의 학생, 노조원, 사회당과 공산당등 야당 당원들이 연대시위를 벌였다.</li>
<li><a href="/wiki/%ED%8C%94%EB%A0%88%EC%8A%A4%ED%83%80%EC%9D%B8_%EC%9E%90%EC%B9%98_%EC%A0%95%EB%B6%80" title="팔레스타인 자치 정부">팔레스타인</a> <a href="/wiki/%ED%95%98%EB%A7%88%EC%8A%A4" title="하마스">하마스</a>의 <a class="new" href="/w/index.php?title=%EC%9D%B4%EC%8A%A4%EB%A7%88%EC%9D%BC_%ED%95%98%EB%8B%88%EC%95%BC&amp;action=edit&amp;redlink=1" title="이스마일 하니야 (없는 문서)">이스마일 하니야</a> 총리 내정자는 핵심 각료 자리에 하마스 내부 인사들을 내정하는 등 <a href="/wiki/%ED%8C%94%EB%A0%88%EC%8A%A4%ED%83%80%EC%9D%B8_%EC%9E%90%EC%B9%98_%EC%A0%95%EB%B6%80" title="팔레스타인 자치 정부">팔레스타인 자치 정부</a> 내각 구성을 완료하였다. 한편 <a class="mw-redirect" href="/wiki/%EB%A7%88%ED%9D%90%EB%AC%B4%EB%93%9C_%EC%95%84%EB%B0%94%EC%8A%A4" title="마흐무드 아바스">마흐무드 아바스</a> 수반이 이끄는 <a href="/wiki/%ED%8C%8C%ED%83%80" title="파타">파타</a>는 하마스와의 연립 정부 구성을 거부하였다.</li>
</ul>

<li><a href="/wiki/%EC%9D%B4%EB%9D%BC%ED%81%AC_%EC%A0%84%EC%9F%81" title="이라크 전쟁">이라크 전쟁</a> : 미군이 <a href="/wiki/%EB%B0%94%EA%B7%B8%EB%8B%A4%EB%93%9C" title="바그다드">바그다드</a> 북쪽으로 100 km 가량 떨어진 이샤카의 한 가정을 공습하고 집에서 자고 있던 11명의 이라크인을 사살해 이라크 경찰이 미군을 고발했다.</li>

<li>개그맨 <a href="/wiki/%EA%B9%80%ED%98%95%EA%B3%A4" title="김형곤">김형곤</a>이 11일 오전 갑작스럽게 세상을 떠났다.</li>

<li><a href="/wiki/%EC%8A%AC%EB%A1%9C%EB%B3%B4%EB%8B%A8_%EB%B0%80%EB%A1%9C%EC%85%B0%EB%B9%84%EC%B9%98" title="슬로보단 밀로셰비치">슬로보단 밀로셰비치</a> 전 <a href="/wiki/%EC%9C%A0%EA%B3%A0%EC%8A%AC%EB%9D%BC%EB%B9%84%EC%95%84" title="유고슬라비아">유고슬라비아</a> 대통령이 <a href="/wiki/%ED%94%84%EB%9D%BC%ED%95%98" title="프라하">프라하</a>의 감방에서 숨진 채 발견되었다.</li>
<li><a class="mw-redirect" href="/wiki/%EB%AF%B8%EC%B2%BC%EB%A0%88_%EB%B0%94%EC%B2%BC%EB%A0%88%ED%8A%B8" title="미첼레 바첼레트">미첼레 바첼레트</a>가 <a href="/wiki/%EC%B9%A0%EB%A0%88" title="칠레">칠레</a>의 첫 여성 대통령으로 공식 취임했다.</li>
<li>추종자들로부터 <a href="/wiki/%EC%84%9D%EA%B0%80%EB%AA%A8%EB%8B%88" title="석가모니">석가모니</a>의 화신으로 믿어졌던 10대 소년 <a class="new" href="/w/index.php?title=%EB%9E%8C_%EB%B0%94%ED%95%98%EB%91%90%EB%A5%B4_%EB%B0%A4%EC%A1%B4&amp;action=edit&amp;redlink=1" title="람 바하두르 밤존 (없는 문서)">람 바하두르 밤존</a>이 10개월 이상을 단식 수행하던 보리수 나무 밑에서 사라졌다. <a class="external text" href="http://edition.cnn.com/2006/WORLD/asiapcf/03/11/nepal.missing.ap/" rel="nofollow">(CNN)</a></li>

<h2><span class="mw-headline" id="2006.EB.85.84_3.EC.9B.94_8.EC.9D.BC"><a href="/wiki/2006%EB%85%84" title="2006년">2006년</a> <a href="/wiki/3%EC%9B%94_8%EC%9D%BC" title="3월 8일">3월 8일</a></span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=2006%EB%85%84_3%EC%9B%94&amp;action=edit&amp;section=17" title="부분 편집: 2006년 3월 8일">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<ul>
<li>133개 학술,시민단체로 이뤄진 <b>&lt;강정구 교수 탄압 반대 공동대책위원회&gt;</b>는 일주일에 한 차례씩 <a href="/wiki/%EB%8F%99%EA%B5%AD%EB%8C%80%ED%95%99%EA%B5%90" title="동국대학교">동국대</a> 본관 앞에서 천막강의를 열기로 결의함에 따라 &lt;한국사회 냉전 성역 허물기&gt;라는 주제로 <a href="/wiki/%EA%B0%95%EC%A0%95%EA%B5%AC_(1945%EB%85%84)" title="강정구 (1945년)">강정구</a> 교수가 첫 강연을 했다.</li>

<ul>
<li><a href="/wiki/%EC%9D%B4%ED%95%B4%EC%B0%AC" title="이해찬">이해찬</a> <a class="mw-redirect" href="/wiki/%EB%8C%80%ED%95%9C%EB%AF%BC%EA%B5%AD%EC%9D%98_%EC%B4%9D%EB%A6%AC" title="대한민국의 총리">국무총리</a>는 <a class="mw-redirect" href="/wiki/%EC%82%BC%EC%9D%BC%EC%A0%88" title="삼일절">삼일절</a>에 <a href="/wiki/%EA%B3%A8%ED%94%84" title="골프">골프</a>를 치며 정치적인 비리를 일으켰다는 의혹에 대해 대국민 사과를 발표했다. 또 <a href="/wiki/%EB%85%B8%EB%AC%B4%ED%98%84" title="노무현">노무현</a> 대통령에게 총리직 사의를 표명한 것으로 알려졌다.</li>
<li><a href="/wiki/%EC%9B%94%EB%93%9C_%EB%B2%A0%EC%9D%B4%EC%8A%A4%EB%B3%BC_%ED%81%B4%EB%9E%98%EC%8B%9D" title="월드 베이스볼 클래식">월드 베이스볼 클래식</a>에서 <a href="/wiki/%EB%8C%80%ED%95%9C%EB%AF%BC%EA%B5%AD" title="대한민국">대한민국</a>이 <a href="/wiki/%EC%9D%BC%EB%B3%B8" title="일본">일본</a>을 3:2로 누르고 1라운드 조 1위로 제2라운드(8강)에 진출했다.</li>
<li><a href="/wiki/%EC%A4%91%EA%B5%AD" title="중국">중국</a>의 입법기관인 제10기 <a href="/wiki/%EC%A0%84%EA%B5%AD%EC%9D%B8%EB%AF%BC%EB%8C%80%ED%91%9C%EB%8C%80%ED%9A%8C" title="전국인민대표대회">전국인민대표대회</a>(전인대) 제4차 회의가 <a href="/wiki/3%EC%9B%94_14%EC%9D%BC" title="3월 14일">3월 14일</a>까지의 일정으로 <a class="mw-redirect" href="/wiki/%EB%B2%A0%EC%9D%B4%EC%A7%95" title="베이징">베이징</a> 인민대회당에서 개최되었다.</li>
<li>영화 <a href="/wiki/%EC%99%95%EC%9D%98_%EB%82%A8%EC%9E%90" title="왕의 남자">왕의 남자</a>가 관객 1천180만여 명을 동원하여 <a class="mw-redirect" href="/wiki/%EB%8C%80%ED%95%9C%EB%AF%BC%EA%B5%AD_%EC%98%81%ED%99%94_%ED%9D%A5%ED%96%89_%EA%B8%B0%EB%A1%9D" title="대한민국 영화 흥행 기록">역대 한국 영화 중 최고 흥행 기록</a>을 달성했다고 영화 배급사 "시네마서비스"가 밝혔다.</li>
</ul>
<h2><span class="mw-headline" id="2006.EB.85.84_3.EC.9B.94_4.EC.9D.BC"><a href="/wiki/2006%EB%85%84" title="2006년">2006년</a> <a href="/wiki/3%EC%9B%94_4%EC%9D%BC" title="3월 4일">3월 4일</a></span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=2006%EB%85%84_3%EC%9B%94&amp;action=edit&amp;section=20" title="부분 편집: 2006년 3월 4일">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<ul>
<li><a href="/wiki/%ED%95%9C%EA%B5%AD%EC%B2%A0%EB%8F%84%EA%B3%B5%EC%82%AC" title="한국철도공사">한국철도공사</a> <a href="/wiki/%EB%85%B8%EB%8F%99%EC%A1%B0%ED%95%A9" title="노동조합">노동조합</a>이 사흘만에 <a href="/wiki/%ED%8C%8C%EC%97%85" title="파업">파업</a>을 철회하고 업무에 복귀했다.</li>
</ul>
<h2><span class="mw-headline" id="2006.EB.85.84_3.EC.9B.94_2.EC.9D.BC"><a href="/wiki/2006%EB%85%84" title="2006년">2006년</a> <a href="/wiki/3%EC%9B%94_2%EC%9D%BC" title="3월 2일">3월 2일</a></span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=2006%EB%85%84_3%EC%9B%94&amp;action=edit&amp;section=21" title="부분 편집: 2006년 3월 2일">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<ul>
<li><a href="/wiki/%EB%85%B8%EB%AC%B4%ED%98%84" title="노무현">노무현</a> 대통령은 신임 <a href="/wiki/%EB%8C%80%ED%95%9C%EB%AF%BC%EA%B5%AD_%ED%96%89%EC%A0%95%EC%9E%90%EC%B9%98%EB%B6%80" title="대한민국 행정자치부">행정자치부</a>장관에 <a href="/wiki/%EC%9D%B4%EC%9A%A9%EC%84%AD" title="이용섭">이용섭</a>, <a href="/wiki/%EB%8C%80%ED%95%9C%EB%AF%BC%EA%B5%AD_%EB%AC%B8%ED%99%94%EA%B4%80%EA%B4%91%EB%B6%80" title="대한민국 문화관광부">문화관광부</a>장관에 <a href="/wiki/%EA%B9%80%EB%AA%85%EA%B3%A4" title="김명곤">김명곤</a>, <a href="/wiki/%EB%8C%80%ED%95%9C%EB%AF%BC%EA%B5%AD_%EC%A0%95%EB%B3%B4%ED%86%B5%EC%8B%A0%EB%B6%80" title="대한민국 정보통신부">정보통신부</a>장관에 <a href="/wiki/%EB%85%B8%EC%A4%80%ED%98%95" title="노준형">노준형</a>, <a href="/wiki/%EB%8C%80%ED%95%9C%EB%AF%BC%EA%B5%AD_%ED%95%B4%EC%96%91%EC%88%98%EC%82%B0%EB%B6%80" title="대한민국 해양수산부">해양수산부</a>장관에 <a class="new" href="/w/index.php?title=%EA%B9%80%EC%84%B1%EC%A7%84_(1956%EB%85%84)&amp;action=edit&amp;redlink=1" title="김성진 (1956년) (없는 문서)">김성진</a>을 내정하여 발표했다.</li>
</ul>
<h2><span class="mw-headline" id="2006.EB.85.84_3.EC.9B.94_1.EC.9D.BC"><a href="/wiki/2006%EB%85%84" title="2006년">2006년</a> <a href="/wiki/3%EC%9B%94_1%EC%9D%BC" title="3월 1일">3월 1일</a></span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=2006%EB%85%84_3%EC%9B%94&amp;action=edit&amp;section=22" title="부분 편집: 2006년 3월 1일">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<ul>
<li><a class="mw-redirect" href="/wiki/%ED%95%9C%EB%82%98%EB%9D%BC%EB%8B%B9" title="한나라당">한나라당</a> <a href="/wiki/%EC%A0%95%EC%9D%98%ED%99%94" title="정의화">정의화</a> 의원은 여기자 성추행 파문을 일으킨 <a href="/wiki/%EC%B5%9C%EC%97%B0%ED%9D%AC" title="최연희">최연희</a> 전 사무총장에 대해 "<b>급성알콜중독증</b>과 당무로 인한 스트레스 및 과도한 흡연 등으로 정신적·육체적으로 무척 피곤한 상태에 저질러진 우발적 사건"일 가능성을 거듭 제기하면서 사건조사를 제안하였고 최연희 의원에 대한 "정상참작"을 호소했다.</li>
<li><a class="mw-redirect" href="/wiki/%EB%89%B4%EC%9A%95%ED%83%80%EC%9E%84%EC%A6%88" title="뉴욕타임즈">뉴욕타임즈</a>와 CBS TV가 공동으로 조사한 여론조사에서 <a href="/wiki/%EC%A1%B0%EC%A7%80_W._%EB%B6%80%EC%8B%9C" title="조지 W. 부시">조지 W. 부시</a> <a href="/wiki/%EB%AF%B8%EA%B5%AD" title="미국">미국</a> 대통령의 직무 수행 지지도가 집권 이후 최저인 34%로 나타났다.</li>
<li><a href="/wiki/%EB%AF%BC%EC%A1%B1%EB%AC%B8%EC%A0%9C%EC%97%B0%EA%B5%AC%EC%86%8C" title="민족문제연구소">민족문제연구소</a>와 국립묘지령 개정 및 반민족행위자 김창룡 묘 이장 추진 시민연대 회원들이 대전국립묘지 정문에서 집회를 갖고 "장군묘역에 안장된 친일파 <a href="/wiki/%EA%B9%80%EC%B0%BD%EB%A3%A1" title="김창룡">김창룡</a>의 묘를 당장 이장하라"고 촉구했다.</li>
<li><a href="/wiki/%EA%B3%A0%EC%9D%B4%EC%A6%88%EB%AF%B8_%EC%A4%80%EC%9D%B4%EC%B9%98%EB%A1%9C" title="고이즈미 준이치로">고이즈미</a> <a href="/wiki/%EC%9D%BC%EB%B3%B8" title="일본">일본</a> 총리는 <a href="/wiki/%EB%85%B8%EB%AC%B4%ED%98%84" title="노무현">노무현</a> 대통령이 <a href="/wiki/3%C2%B71_%EC%9A%B4%EB%8F%99" title="3·1 운동">3.1절</a> 기념사에서 "일본이 <i>보통국가'</i>, 나아가서는 <i>세계의 지도적인 국가</i>가 되려고 한다면 법을 바꾸고 군비를 강화할 것이 아니라, 먼저 인류의 양심과 도리에 맞게 행동하여 국제사회의 신뢰를 확보해야 할 것"이라고 비판한 것에 대해 "2차대전후 60년 동안 일본이 걸어온 길을 잘 보고 앞으로도 한.일우호를 위해 노력해 주기 바란다"라고 반박했다.</li>
<li>한국철도공사 노동조합이 파업에 돌입했다.</li>
<li><a href="/wiki/%EB%A7%88%EC%9D%B4%ED%81%AC%EB%A1%9C%EC%86%8C%ED%94%84%ED%8A%B8" title="마이크로소프트">마이크로소프트</a>(MicroSoft)는 6개월 이내에 미.영 시장에서 <a href="/wiki/%EA%B5%AC%EA%B8%80" title="구글">구글</a>보다 우수한 <a class="mw-redirect" href="/wiki/%EA%B2%80%EC%83%89%EC%97%94%EC%A7%84" title="검색엔진">검색엔진</a>을 도입할 예정이라고 닐 할로웨이 MS 유럽.중동.아프리카담당 사장이 밝혔다.</li>
</ul>
