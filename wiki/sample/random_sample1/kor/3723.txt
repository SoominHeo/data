<p><b>프레더릭 덩컨 마이클 홀데인</b> (<a href="/wiki/%EC%98%81%EC%96%B4" title="영어">영어</a>: Frederick Duncan Michael Haldane, <a href="/wiki/%EC%99%95%EB%A6%BD%ED%95%99%ED%9A%8C" title="왕립학회">FRS</a>, 1951년 9월 14일 ~ )은 <a href="/wiki/%EC%98%81%EA%B5%AD" title="영국">영국</a>의 <a href="/wiki/%EB%AC%BC%EB%A6%AC%ED%95%99" title="물리학">물리학자</a>로 프린스턴 대학교의 유진 히긴스 물리학 교수이자,  <a class="new" href="/w/index.php?title=%ED%8E%98%EB%A6%AC%EB%AF%B8%ED%84%B0_%EC%9D%B4%EB%A1%A0_%EB%AC%BC%EB%A6%AC%ED%95%99_%EC%97%B0%EA%B5%AC%EC%86%8C&amp;action=edit&amp;redlink=1" title="페리미터 이론 물리학 연구소 (없는 문서)">페리미터 이론 물리학 연구소</a>의 석좌 연구 학과장이다.(Distinguished Visiting Research Chair)<sup class="reference" id="cite_ref-2"><a href="#cite_note-2">[2]</a></sup> 2016년 <a class="mw-redirect" href="/wiki/%EB%8D%B0%EC%9D%B4%EB%B9%84%EB%93%9C_%EC%82%AC%EC%9A%B8%EB%A6%AC%EC%8A%A4" title="데이비드 사울리스">데이비드 사울리스</a>, <a href="/wiki/%EB%A7%88%EC%9D%B4%ED%81%B4_%EC%BD%94%EC%8A%A4%ED%84%B8%EB%A6%AC%EC%B8%A0" title="마이클 코스털리츠">마이클 코스털리츠</a>와 함께 노벨 물리학상을 수상하였다.<sup class="reference" id="cite_ref-Devlin_Sample_2016_3-0"><a href="#cite_note-Devlin_Sample_2016-3">[3]</a></sup></p>

<p>런던 <a class="new" href="/w/index.php?title=St_Paul%27s_School&amp;action=edit&amp;redlink=1" title="St Paul's School (없는 문서)">St Paul's School</a>과 <a class="new" href="/w/index.php?title=%EC%BA%A0%EB%B8%8C%EB%A6%AC%EC%A7%80_%ED%81%AC%EB%9D%BC%EC%9D%B4%EC%8A%A4%ED%8A%B8_%EC%B9%BC%EB%A6%AC%EC%A7%80&amp;action=edit&amp;redlink=1" title="캠브리지 크라이스트 칼리지 (없는 문서)">캠브리지 크라이스트 칼리지</a>에서 교육을 받았다.<sup class="reference" id="cite_ref-4"><a href="#cite_note-4">[4]</a></sup></p>
<h2><span class="mw-headline" id=".EC.97.B0.EA.B5.AC">연구</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%8D%A9%EC%BB%A8_%ED%99%80%EB%8D%B0%EC%9D%B8&amp;action=edit&amp;section=2" title="부분 편집: 연구">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p><a href="/wiki/%EC%9D%91%EC%A7%91%EB%AC%BC%EC%A7%88%EB%AC%BC%EB%A6%AC%ED%95%99" title="응집물질물리학">응집물질물리학</a>에서의 광범위한 기초적인 기여로 알려져있다. <a class="new" href="/w/index.php?title=%EB%A3%A8%ED%8B%B4%EC%A0%80_%EC%95%A1%EC%B2%B4_%EC%9D%B4%EB%A1%A0&amp;action=edit&amp;redlink=1" title="루틴저 액체 이론 (없는 문서)">루틴저 액체 이론</a>과 일차원적 <a class="new" href="/w/index.php?title=%EC%8A%A4%ED%95%80_%EB%AA%A8%EB%8D%B8&amp;action=edit&amp;redlink=1" title="스핀 모델 (없는 문서)">스핀 사슬</a>, <a class="new" href="/w/index.php?title=%EB%B6%84%EC%88%98_%EC%96%91%EC%9E%90_%ED%99%80_%ED%98%84%EC%83%81&amp;action=edit&amp;redlink=1" title="분수 양자 홀 현상 (없는 문서)">분수 양자 홀 현상</a>과 entanglement spectra, exclusion statistics 등의 수많은 연구에 기여를 했다.</p>
<p>2011년 6월 현재 "unimodular" 를 통해 "혼성 보손"의 "shape"을 도입하는 분수 양자 홀 현상의 새로운 기하학적 설명을 연구중에 있다.  (determinant 1) spatial metric-tensor field as the fundmamental collective degree of freedom of FQHE states.<sup class="reference" id="cite_ref-5"><a href="#cite_note-5">[5]</a></sup> This new "Chern-Simons + quantum geometry" description is a replacement for the "Chern-Simons + Ginzburg-Landau" paradigm introduced c.1990. Unlike its predecessor, it provides a description of the FQHE collective mode that agrees with the Girvin-Macdonald-Platzman "single-mode approximation".<sup class="reference" id="cite_ref-6"><a href="#cite_note-6">[6]</a></sup></p>
<h2><span class="mw-headline" id=".EC.88.98.EC.83.81.EA.B3.BC_.EC.98.81.EC.98.88">수상과 영예</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%8D%A9%EC%BB%A8_%ED%99%80%EB%8D%B0%EC%9D%B8&amp;action=edit&amp;section=3" title="부분 편집: 수상과 영예">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>1996년, 런던의 <a class="mw-redirect" href="/wiki/%EC%99%95%EB%A6%BD_%ED%95%99%ED%9A%8C" title="왕립 학회">왕립 학회</a> 회원이 되었으며, <sup class="reference" id="cite_ref-7"><a href="#cite_note-7">[7]</a></sup> 1992년 <a href="/wiki/%EB%AF%B8%EA%B5%AD_%EC%98%88%EC%88%A0_%EA%B3%BC%ED%95%99_%EC%95%84%EC%B9%B4%EB%8D%B0%EB%AF%B8" title="미국 예술 과학 아카데미">미국 예술 과학 아카데미</a>의 회원이 되었다. <sup class="reference" id="cite_ref-8"><a href="#cite_note-8">[8]</a></sup> 또한 <a href="/wiki/%EB%AF%B8%EA%B5%AD_%EB%AC%BC%EB%A6%AC%ED%95%99%ED%9A%8C" title="미국 물리학회">미국 물리학회</a>의 회원이자,<sup class="reference" id="cite_ref-9"><a href="#cite_note-9">[9]</a></sup> 영국 Institute of Physics의 회원이며, American Association for the Advancement of Science의 회원이다. 1993년 미국 물리학회의 올리버 E.버클리 상을 수상하였으며, <a href="/wiki/%EC%95%A8%ED%94%84%EB%A6%AC%EB%93%9C_P._%EC%8A%AC%EB%A1%A0_%EC%9E%AC%EB%8B%A8" title="앨프리드 P. 슬론 재단">앨프리드 P. 슬론 재단</a>의 회원이였다.(1984 ~ 1988)  Lorentz Chair이었으며 (2008)<sup class="reference" id="cite_ref-10"><a href="#cite_note-10">[10]</a></sup>, <a class="new" href="/w/index.php?title=%EB%94%94%EB%9E%99_%EB%A9%94%EB%8B%AC&amp;action=edit&amp;redlink=1" title="디랙 메달 (없는 문서)">디랙 메달</a>(2012)<sup class="reference" id="cite_ref-11"><a href="#cite_note-11">[11]</a></sup>,Doctor Honoris Causae of the Université de Cergy-Pontoise (2015) <sup class="reference" id="cite_ref-12"><a href="#cite_note-12">[12]</a></sup> 그리고 <a href="/wiki/%EB%85%B8%EB%B2%A8_%EB%AC%BC%EB%A6%AC%ED%95%99%EC%83%81" title="노벨 물리학상">노벨 물리학상</a>(2016)등을 수상하였다.</p>
