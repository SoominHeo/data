<p><b>데니우송 마르틴스 나시멘투</b>(<span style="font-size: smaller;"><a href="/wiki/%ED%8F%AC%EB%A5%B4%ED%88%AC%EA%B0%88%EC%96%B4" title="포르투갈어">포르투갈어</a>:</span> <span lang="pt" xml:lang="pt">Denilson Martins Nascimento</span>, <a href="/wiki/1976%EB%85%84" title="1976년">1976년</a> <a href="/wiki/9%EC%9B%94_4%EC%9D%BC" title="9월 4일">9월 4일</a> ~ )는 <b>데니우송</b>으로 잘 알려진 <a href="/wiki/%EB%B8%8C%EB%9D%BC%EC%A7%88" title="브라질">브라질</a> <a href="/wiki/%EC%82%AC%EC%9A%B0%EB%B0%94%EB%8F%84%EB%A5%B4" title="사우바도르">사우바도르</a> 출신의 <a href="/wiki/%EC%B6%95%EA%B5%AC" title="축구">축구</a> 선수이다. <a href="/wiki/2006%EB%85%84" title="2006년">2006년</a>부터 <a href="/wiki/2009%EB%85%84" title="2009년">2009년</a>까지 <a href="/wiki/K%EB%A6%AC%EA%B7%B8" title="K리그">K리그</a>의 <a href="/wiki/%EB%8C%80%EC%A0%84_%EC%8B%9C%ED%8B%B0%EC%A6%8C" title="대전 시티즌">대전 시티즌</a>, <a href="/wiki/%ED%8F%AC%ED%95%AD_%EC%8A%A4%ED%8B%B8%EB%9F%AC%EC%8A%A4" title="포항 스틸러스">포항 스틸러스</a>에서 <b>데닐손</b>이라는 등록명으로 활약하였다.</p>

<h2><span class="mw-headline" id=".ED.81.B4.EB.9F.BD_.EA.B2.BD.EB.A0.A5">클럽 경력</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%8D%B0%EB%8B%88%EC%9A%B0%EC%86%A1_%EB%A7%88%EB%A5%B4%ED%8B%B4%EC%8A%A4_%EB%82%98%EC%8B%9C%EB%A9%98%ED%88%AC&amp;action=edit&amp;section=1" title="부분 편집: 클럽 경력">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p><a href="/wiki/2006%EB%85%84" title="2006년">2006년</a> 초에 <a class="mw-redirect" href="/wiki/K-%EB%A6%AC%EA%B7%B8" title="K-리그">K-리그</a> <a href="/wiki/%EB%8C%80%EC%A0%84_%EC%8B%9C%ED%8B%B0%EC%A6%8C" title="대전 시티즌">대전 시티즌</a>에 3개월 임대로 영입되어 인상적인 모습을 보였으며 <a href="/wiki/%EB%B8%8C%EB%9D%BC%EC%A7%88" title="브라질">브라질</a>로 복귀하였다가 <a href="/wiki/%EC%B5%9C%EC%9C%A4%EA%B2%B8" title="최윤겸">최윤겸</a> 감독의 연락<sup class="reference" id="cite_ref-1"><a href="#cite_note-1">[1]</a></sup> 을 받고 <a href="/wiki/2007%EB%85%84" title="2007년">2007년</a> 말까지 1년 6개월 동안 정식 계약하였다. 임대기간에는 15번을 달고 있었는데, 정식 계약을 하면서 <a href="/wiki/%EC%9D%B4%EA%B4%80%EC%9A%B0" title="이관우">이관우</a> 선수의 배번인 등번호 8번을 받았다.</p>
<p><a href="/wiki/2007%EB%85%84" title="2007년">2007년</a> <a href="/wiki/9%EC%9B%94_22%EC%9D%BC" title="9월 22일">9월 22일</a> <a href="/wiki/%EB%8C%80%EA%B5%AC_FC" title="대구 FC">대구 FC</a>와의 경기에서 <a href="/wiki/%ED%95%B4%ED%8A%B8%ED%8A%B8%EB%A6%AD" title="해트트릭">해트트릭</a>을 기록하는 등 뛰어난 활약으로 <a href="/wiki/%EB%8C%80%EC%A0%84_%EC%8B%9C%ED%8B%B0%EC%A6%8C" title="대전 시티즌">대전 시티즌</a>의 플레이오프 진출을 이끌어 <a class="mw-redirect" href="/wiki/2007%EB%85%84_K-%EB%A6%AC%EA%B7%B8" title="2007년 K-리그">2007년 K-리그</a> MVP 후보에 올랐으며, 골을 넣으면 코미디 프로그램의 마빡이를 응용한 세레모니, 탈춤 세레모니 등 독특한 팬 서비스를 제공하며 팬들의 사랑을 받았다. (<a href="/wiki/2006%EB%85%84" title="2006년">2006년</a> 26경기 9득점 3도움, <a href="/wiki/2007%EB%85%84" title="2007년">2007년</a> 34경기 19득점 5도움)</p>
<p>계약 종료후 해외 리그로 이적한다는 소식과 높은 몸값에 <a href="/wiki/%EB%8C%80%EC%A0%84_%EC%8B%9C%ED%8B%B0%EC%A6%8C" title="대전 시티즌">대전 시티즌</a>은 재계약을 포기했지만 <a href="/wiki/%ED%8F%AC%ED%95%AD_%EC%8A%A4%ED%8B%B8%EB%9F%AC%EC%8A%A4" title="포항 스틸러스">포항 스틸러스</a>로 이적하였다.<sup class="reference" id="cite_ref-2"><a href="#cite_note-2">[2]</a></sup> <a href="/wiki/2009%EB%85%84" title="2009년">2009년</a> 포항의 <a href="/wiki/AFC_%EC%B1%94%ED%94%BC%EC%96%B8%EC%8A%A4%EB%A6%AC%EA%B7%B8" title="AFC 챔피언스리그">AFC 챔피언스리그</a> 우승에 공헌한 뒤, <a href="/wiki/2010%EB%85%84_1%EC%9B%94" title="2010년 1월">2010년 1월</a> <a href="/wiki/%EC%9A%B0%EC%A6%88%EB%B2%A0%ED%82%A4%EC%8A%A4%ED%83%84" title="우즈베키스탄">우즈베키스탄</a>의 <a href="/wiki/FC_%EB%B6%80%EB%87%A8%EB%93%9C%EC%BD%94%EB%A5%B4" title="FC 부뇨드코르">FC 부뇨드코르</a>로 이적하였다. 이적 후 정규 리그 6경기에서 5골을 넣었고, <a href="/wiki/AFC_%EC%B1%94%ED%94%BC%EC%96%B8%EC%8A%A4%EB%A6%AC%EA%B7%B8_2010" title="AFC 챔피언스리그 2010">AFC 챔피언스리그</a>에서도 5골을 넣었으나 그해 여름 방출 당하였다.</p>
