<p><b>데드 프린세스</b>는 <a href="/wiki/%EC%9D%BC%EB%B3%B8" title="일본">일본</a>의 <a href="/wiki/%EB%A7%8C%ED%99%94%EA%B0%80" title="만화가">만화가</a>인 <a class="new" href="/w/index.php?title=%EC%95%84%ED%82%A4%ED%9E%88%ED%86%A0_%EC%9A%94%EC%8B%9C%EC%9D%B4%EC%B9%98&amp;action=edit&amp;redlink=1" title="아키히토 요시이치 (없는 문서)">아키히토 요시이치</a>가 <a class="new" href="/w/index.php?title=%EC%9B%94%EA%B0%84_%EC%86%8C%EB%85%84_%EA%B0%95%EA%B0%95&amp;action=edit&amp;redlink=1" title="월간 소년 강강 (없는 문서)">월간 소년 강강</a>에서 연재 중인 <a href="/wiki/%EB%A7%8C%ED%99%94" title="만화">만화</a>이자, 동명의 만화를 바탕으로 제작된 애니메이션 작품이다. '데드 프린세스'라는 제목은 <a href="/wiki/%EB%8C%80%ED%95%9C%EB%AF%BC%EA%B5%AD" title="대한민국">대한민국</a>에서 발행된 한국어 정식판의 명칭으로, <a href="/wiki/%EC%9D%BC%EB%B3%B8%EC%96%B4" title="일본어">일본어</a>판의 원제는 <b>시희</b>(<span style="font-size: smaller;"><a href="/wiki/%EC%9D%BC%EB%B3%B8%EC%96%B4" title="일본어">일본어</a>:</span> <span lang="ja" xml:lang="ja">屍姫</span> <small>시카바네 히메<sup>[<a href="/wiki/%EC%9C%84%ED%82%A4%EB%B0%B1%EA%B3%BC:%EC%9D%BC%EB%B3%B8%EC%96%B4%EC%9D%98_%ED%95%9C%EA%B8%80_%ED%91%9C%EA%B8%B0" title="위키백과:일본어의 한글 표기">*</a>]</sup></small>)이다.</p>
<p><a href="/wiki/%EC%9D%BC%EB%B3%B8" title="일본">일본</a>을 기준으로 <a href="/wiki/2005%EB%85%84" title="2005년">2005년</a>에 첫 단행본이 출간되었으며, <a href="/wiki/2008%EB%85%84" title="2008년">2008년</a> <a href="/wiki/10%EC%9B%94" title="10월">10월</a>부터 <a href="/wiki/12%EC%9B%94" title="12월">12월</a>까지 <b>적</b>(<span lang="ja" xml:lang="ja">赫</span> <small>아카<sup>[<a href="/wiki/%EC%9C%84%ED%82%A4%EB%B0%B1%EA%B3%BC:%EC%9D%BC%EB%B3%B8%EC%96%B4%EC%9D%98_%ED%95%9C%EA%B8%80_%ED%91%9C%EA%B8%B0" title="위키백과:일본어의 한글 표기">*</a>]</sup></small>)편, <a href="/wiki/2009%EB%85%84" title="2009년">2009년</a> <a href="/wiki/1%EC%9B%94" title="1월">1월</a>부터 <a href="/wiki/3%EC%9B%94" title="3월">3월</a>까지 <b>흑</b>(<span lang="ja" xml:lang="ja">玄</span> <small>쿠로<sup>[<a href="/wiki/%EC%9C%84%ED%82%A4%EB%B0%B1%EA%B3%BC:%EC%9D%BC%EB%B3%B8%EC%96%B4%EC%9D%98_%ED%95%9C%EA%B8%80_%ED%91%9C%EA%B8%B0" title="위키백과:일본어의 한글 표기">*</a>]</sup></small>)편으로 각각 나뉘어 애니메이션이 방송되었다. <a href="/wiki/%EB%8C%80%ED%95%9C%EB%AF%BC%EA%B5%AD" title="대한민국">대한민국</a>에서는 <a href="/wiki/2007%EB%85%84" title="2007년">2007년</a> <a href="/wiki/10%EC%9B%94" title="10월">10월</a>부터 한국어로 번역 출간되고 있으며, 역자는 서현아이다.</p>
