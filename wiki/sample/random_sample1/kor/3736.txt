<p><b>데릭 월컷</b>(Derek Walcott, <a href="/wiki/%EB%8C%80%EC%98%81_%EC%A0%9C%EA%B5%AD_%ED%9B%88%EC%9E%A5" title="대영 제국 훈장">OBE</a>, <a href="/wiki/1930%EB%85%84" title="1930년">1930년</a> <a href="/wiki/1%EC%9B%94_23%EC%9D%BC" title="1월 23일">1월 23일</a> ~ )은 <a href="/wiki/%EC%84%B8%EC%9D%B8%ED%8A%B8%EB%A3%A8%EC%8B%9C%EC%95%84" title="세인트루시아">세인트루시아</a>의 <a href="/wiki/%EC%8B%9C%EC%9D%B8" title="시인">시인</a>이자 <a href="/wiki/%EA%B7%B9%EC%9E%91%EA%B0%80" title="극작가">극작가</a>로 <a href="/wiki/1992%EB%85%84" title="1992년">1992년</a> <a href="/wiki/%EB%85%B8%EB%B2%A8_%EB%AC%B8%ED%95%99%EC%83%81" title="노벨 문학상">노벨 문학상</a> 수상자이다.</p>
<p>그의 작업은 식민지 시절의 <a class="mw-redirect" href="/wiki/%EB%85%B8%EC%98%88" title="노예">노예</a> 제도부터 독립까지 <a href="/wiki/%EC%B9%B4%EB%A6%AC%EB%B8%8C_%ED%95%B4" title="카리브 해">카리브 해</a>의 경험, 색다른 문화와 전통의 혼합이 담긴 카리브 해의 식민적 지위의 자연을 탐구하였다. 그의 작품들은 아프리카인과 유럽인 둘다 포함하는 자신의 풍부한 인종적 유산과 문화적 배경을 반영하는 편이다.</p>
<p>세인트루시아의 캐스트리즈에서 태어났다. 처음으로 그의 상업적으로 발간된 시집은 〈푸른 밤에: 1948-1960〉(<a href="/wiki/1962%EB%85%84" title="1962년">1962년</a>)이었다. 다른 편들로는 긴 이야기가 담긴 <a href="/wiki/%EC%8B%9C_(%EB%AC%B8%ED%95%99)" title="시 (문학)">시</a> 〈다른 인생〉(<a href="/wiki/1973%EB%85%84" title="1973년">1973년</a>), 〈별사과 왕국〉(<a href="/wiki/1979%EB%85%84" title="1979년">1979년</a>), 〈행운적 여행인〉 (<a href="/wiki/1981%EB%85%84" title="1981년">1981년</a>), 〈한 여름 중〉(<a href="/wiki/1984%EB%85%84" title="1984년">1984년</a>), 〈아칸소 유언〉(<a href="/wiki/1987%EB%85%84" title="1987년">1987년</a>)과 〈하사금〉(<a href="/wiki/1997%EB%85%84" title="1997년">1997년</a>)을 포함한다. 〈오메로스〉(<a href="/wiki/1990%EB%85%84" title="1990년">1990년</a>)는 개인적, 집단적 추억들 사이에 균형을 찾는데, 시인의 내적인 투쟁을 묘사하는 <a href="/wiki/%EC%84%9C%EC%82%AC%EC%8B%9C" title="서사시">서사시</a>였다. 다른 서사시 〈티에폴로의 사냥개〉(<a href="/wiki/2000%EB%85%84" title="2000년">2000년</a>)는 <a class="mw-redirect" href="/wiki/%EB%B2%84%EC%A7%84_%EC%95%84%EC%9D%BC%EB%9E%9C%EB%93%9C" title="버진 아일랜드">버진 아일랜드</a>에서 태어난 <a href="/wiki/%EC%9D%B8%EC%83%81%EC%A3%BC%EC%9D%98" title="인상주의">인상주의</a> 화가 <a class="mw-redirect" href="/wiki/%EC%B9%B4%EB%B0%80_%ED%94%BC%EC%82%AC%EB%A1%9C" title="카밀 피사로">카밀 피사로</a>에 중심을 두고 있다. 연작 시집 〈방탕자〉(<a href="/wiki/2004%EB%85%84" title="2004년">2004년</a>)는 그의 여행책과 저서를 겸하였다. 그의 시 전집은 <a href="/wiki/2007%EB%85%84" title="2007년">2007년</a>에 발간되었다.</p>
<p>그의 극작으로는 〈원숭이 산에 대한 꿈〉(<a href="/wiki/1967%EB%85%84" title="1967년">1967년</a>), 〈기념〉(<a href="/wiki/1977%EB%85%84" title="1977년">1977년</a>), 〈무연극〉(<a href="/wiki/1978%EB%85%84" title="1978년">1978년</a>)과 〈오디세이: 무대극판〉(<a href="/wiki/1993%EB%85%84" title="1993년">1993년</a>)을 포함한다. 그 작품들은 위선, 탐험, 권의에 대해 인간의 투쟁에 집중시키고 있다. 그는 또한 <a href="/wiki/%EB%AF%B8%EA%B5%AD" title="미국">미국</a>의 가수 <a href="/wiki/%ED%8F%B4_%EC%82%AC%EC%9D%B4%EB%A8%BC" title="폴 사이먼">폴 사이먼</a>과 함께 음악적 작품 〈동굴인〉(<a href="/wiki/1997%EB%85%84" title="1997년">1997년</a>)을 쓰기도 하였다.</p>
