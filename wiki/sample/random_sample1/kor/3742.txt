<p><b>데빈 스타 테일즈</b>(1989년 7월 2일 ~ )는 <b>데브</b>(Dev)라는 예명으로 활동하는 <a href="/wiki/%EB%AF%B8%EA%B5%AD" title="미국">미국</a>의 <a href="/wiki/%EA%B0%80%EC%88%98" title="가수">가수</a> 겸 <a class="mw-redirect" href="/wiki/%EB%9E%A9%ED%8D%BC" title="랩퍼">랩퍼</a>이다. <a href="/wiki/%EC%9D%8C%EC%95%85_%ED%94%84%EB%A1%9C%EB%93%80%EC%84%9C" title="음악 프로듀서">음악 프로듀서</a>인 <a class="new" href="/w/index.php?title=%EB%8D%94_%EC%BA%90%ED%84%B0%EB%9D%BD%EC%8A%A4&amp;action=edit&amp;redlink=1" title="더 캐터락스 (없는 문서)">더 캐터락스</a>에 의해 <a href="/wiki/%EB%A7%88%EC%9D%B4%EC%8A%A4%ED%8E%98%EC%9D%B4%EC%8A%A4" title="마이스페이스">마이스페이스</a>에서 발굴되었다. 데브의 노래 "부티 바운스(Booty Bounce)"는 <a href="/wiki/%ED%8C%8C_%EC%9D%B4%EC%8A%A4%ED%8A%B8_%EB%AC%B4%EB%B8%8C%EB%A8%BC%ED%8A%B8" title="파 이스트 무브먼트">파 이스트 무브먼트</a>의 싱글 "라이크 어 지식스(Like A G6)"에 <a class="mw-disambig" href="/wiki/%EC%83%98%ED%94%8C%EB%A7%81" title="샘플링">샘플링</a>으로 사용되어, <a href="/wiki/%EB%AF%B8%EA%B5%AD" title="미국">미국</a> <a href="/wiki/%EB%B9%8C%EB%B3%B4%EB%93%9C_%ED%95%AB_100" title="빌보드 핫 100">빌보드 핫 100</a> 차트 1위와 함께 미국 내 500만 회 이상의 다운로드를 기록했다.</p>
<p>데브는 <a href="/wiki/2010%EB%85%84" title="2010년">2010년</a> 10월 <a class="mw-redirect" href="/wiki/%EC%9C%A0%EB%8B%88%EB%B2%84%EC%84%A4_%EB%A6%AC%ED%8D%BC%EB%B8%94%EB%A6%AD" title="유니버설 리퍼블릭">유니버설 리퍼블릭</a>과 계약하고 첫 공식 싱글 "베이스 다운 로우(Bass Down Low)"를 12월에 발표했다. 이 곡은 최고로 <a href="/wiki/%EB%AF%B8%EA%B5%AD" title="미국">미국</a>에서 61위, 영국에서 10위를 기록했다. 데브의 데뷔 앨범 &lt;<a class="new" href="/w/index.php?title=%EB%8D%94_%EB%82%98%EC%9D%B4%ED%8A%B8_%EB%8D%94_%EC%84%A0_%EC%BC%80%EC%9E%84_%EC%97%85&amp;action=edit&amp;redlink=1" title="더 나이트 더 선 케임 업 (없는 문서)">더 나이트 더 선 케임 업</a>(The Night The Sun Came Up)&gt;은 <a href="/wiki/2011%EB%85%84_9%EC%9B%94" title="2011년 9월">2011년 9월</a> 20일 발매되었다.</p>
<h2><span class="mw-headline" id=".EC.9D.8C.EC.95.85_.EC.8A.A4.ED.83.80.EC.9D.BC">음악 스타일</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%8D%B0%EB%B8%8C&amp;action=edit&amp;section=1" title="부분 편집: 음악 스타일">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>데브의 <a href="/wiki/%EC%9D%BC%EB%A0%89%ED%8A%B8%EB%A1%9C%EB%8B%88%EC%B9%B4" title="일렉트로니카">일렉트로니카</a> 및 "노래하며-말하기(sing-talk)" 식의 <a href="/wiki/%EB%B3%B4%EC%BB%AC" title="보컬">보컬</a> 스타일은 동료 팝 가수인 <a href="/wiki/%EC%BC%80%EC%83%A4" title="케샤">케샤</a>와 비교되고 있다. <sup class="reference" id="cite_ref-Idolator_1-0"><a href="#cite_note-Idolator-1">[1]</a></sup> 데브는 이에 대해 "나는 내가 하는 것에 대해 충분히 자신이 있으므로 사람들은 나 그대로를 볼 것이다. (…) 난 왜 사람들이 우릴 비교하는지 이해하지만, 사람들이 내 라이브를 본다면 그렇게 우릴 연결짓지는 못할 것이다."<sup class="reference" id="cite_ref-Hollywood_2-0"><a href="#cite_note-Hollywood-2">[2]</a></sup> 음악 <a href="/wiki/%EB%B8%94%EB%A1%9C%EA%B7%B8" title="블로그">블로그</a> <a class="new" href="/w/index.php?title=%EC%95%84%EC%9D%B4%EB%8F%8C%EB%A0%88%EC%9D%B4%ED%84%B0&amp;action=edit&amp;redlink=1" title="아이돌레이터 (없는 문서)">아이돌레이터</a>는 데브를 <a href="/wiki/%EC%BC%80%EC%83%A4" title="케샤">케샤</a>에 비교하며 "데브의 <a href="/wiki/%EB%B9%84%ED%8A%B8" title="비트">비트</a>는 약간 더 부드럽게 끊어지고 데브의 노래는 약간 더 <a class="mw-disambig" href="/wiki/%ED%8C%9D" title="팝">팝</a>보다 <a href="/wiki/%EC%9D%BC%EB%A0%89%ED%8A%B8%EB%A1%9C%EB%8B%88%EC%B9%B4" title="일렉트로니카">일렉트로</a>적"<sup class="reference" id="cite_ref-Idolator_1-1"><a href="#cite_note-Idolator-1">[1]</a></sup>이라고 지적했다. <a href="/wiki/%EC%BC%80%EC%83%A4" title="케샤">케샤</a>와 <a href="/wiki/%EC%96%B4%ED%94%BC" title="어피">어피</a> 둘에 비교되는 점에 대해 데브는 이렇게 말했다. "나는 좀 더 할 말이 많고 다방면의 사운드를 가질 능력이 있다고 느낄 뿐이다. 난 어떤 아티스트도 공격하지 않지만, 그들의 작품들로부터 여러분이 들어온 것보다 내가 여러분에게 약간 더 많이 보여드릴 수 있을 것이다. 내 앨범이 나오면 또 다른 언급이나 비교를 들을 필요가 없기를 바란다."<sup class="reference" id="cite_ref-Fashion_Indie_3-0"><a href="#cite_note-Fashion_Indie-3">[3]</a></sup></p>
<p>"<a class="mw-disambig" href="/wiki/%ED%8C%9D" title="팝">팝</a> 음악이지만 <a class="mw-disambig" href="/wiki/%EB%9E%A9" title="랩">랩</a>의 뿌리"에 팬들이 반응한다는 믿음에 따라, 데브 음악의 많은 부분은 <a class="mw-redirect" href="/wiki/%EB%8D%A5%EC%8A%A4%ED%85%9D" title="덥스텝">덥스텝</a>의 영감을 받았다. 데브는 "난 항상 <a href="/wiki/%ED%9E%99%ED%95%A9" title="힙합">힙합</a>에 빠져 있었다"며 "내가 어릴 때 <a href="/wiki/%EC%97%90%EB%AF%B8%EB%84%B4" title="에미넴">에미넴</a>은 내 삶을 바꿔놓았다"<sup class="reference" id="cite_ref-Hollywood_2-1"><a href="#cite_note-Hollywood-2">[2]</a></sup>고 고백하기도 했다. 연예잡지 <a href="/wiki/%EB%8D%94_%ED%95%A0%EB%A6%AC%EC%9A%B0%EB%93%9C_%EB%A6%AC%ED%8F%AC%ED%84%B0" title="더 할리우드 리포터">더 할리우드 리포터</a>는 데브의 "미래적"이고 "고동치는" 사운드를 <a class="mw-disambig" href="/wiki/%EC%96%B8%EB%8D%94%EA%B7%B8%EB%9D%BC%EC%9A%B4%EB%93%9C" title="언더그라운드">언더그라운드</a> <a href="/wiki/%EC%9D%BC%EB%A0%89%ED%8A%B8%EB%A1%9C%ED%81%B4%EB%9E%98%EC%8B%9C" title="일렉트로클래시">일렉트로클래시</a> 뮤지션인 <a href="/wiki/%ED%94%BC%EC%B9%98%EC%8A%A4" title="피치스">피치스</a>와 <a href="/wiki/%EC%98%90" title="옐">옐</a>에 비교했다.<sup class="reference" id="cite_ref-Hollywood_2-2"><a href="#cite_note-Hollywood-2">[2]</a></sup> 데브의 음악 장르는 <a class="new" href="/w/index.php?title=%EC%9D%BC%EB%A0%89%ED%8A%B8%EB%A1%9C%ED%81%AC%EB%A0%81%ED%81%AC&amp;action=edit&amp;redlink=1" title="일렉트로크렁크 (없는 문서)">일렉트로크렁크</a><sup class="reference" id="cite_ref-4"><a href="#cite_note-4">[4]</a></sup>, <a href="/wiki/%EC%9D%BC%EB%A0%89%ED%8A%B8%EB%A1%9C%ED%8C%9D" title="일렉트로팝">일렉트로팝</a>,<sup class="reference" id="cite_ref-Idolator_1-2"><a href="#cite_note-Idolator-1">[1]</a></sup> 혹은 <a class="new" href="/w/index.php?title=%EC%9D%BC%EB%A0%89%ED%8A%B8%EB%A1%9C%ED%95%A9&amp;action=edit&amp;redlink=1" title="일렉트로합 (없는 문서)">일렉트로합</a>으로 불린다.<sup class="reference" id="cite_ref-Hollywood_2-3"><a href="#cite_note-Hollywood-2">[2]</a></sup></p>
