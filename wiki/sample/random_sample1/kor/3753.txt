<p><b>데스크톱 컴퓨터</b>(desktop computer, <span style="font-size: smaller;"><a href="/wiki/%EB%AC%B8%ED%99%94%EC%96%B4" title="문화어">문화어</a>:</span> 탁상콤퓨터, 탁상형콤퓨터)는 가공되지 않은 데이터를 뜻이 있는 정보로 변환하는 전자 기기 <a href="/wiki/%EC%BB%B4%ED%93%A8%ED%84%B0" title="컴퓨터">컴퓨터</a>로 <a class="mw-redirect" href="/wiki/%EC%82%AC%EB%AC%B4" title="사무">사무</a>용이나 <a href="/wiki/%EA%B0%80%EC%A0%95" title="가정">가정</a>용으로 <a class="new" href="/w/index.php?title=%EC%BB%B4%ED%93%A8%ED%84%B0_%EC%B1%85%EC%83%81&amp;action=edit&amp;redlink=1" title="컴퓨터 책상 (없는 문서)">책상</a> 위에 올려 놓고 쓸 수 있게 만들어져 있다. 그리고 <a class="mw-redirect" href="/wiki/%EB%85%B8%ED%8A%B8%EB%B6%81_%EC%BB%B4%ED%93%A8%ED%84%B0" title="노트북 컴퓨터">노트북 컴퓨터</a>, <a class="mw-redirect" href="/wiki/PDA" title="PDA">PDA</a>와 같은 <a class="mw-redirect" href="/wiki/%ED%9C%B4%EB%8C%80%EC%9A%A9_%EC%BB%B4%ED%93%A8%ED%84%B0" title="휴대용 컴퓨터">휴대용 컴퓨터</a>와 구별된다. 데스크톱 컴퓨터들은 또한 <a href="/wiki/%EB%A7%88%EC%9D%B4%ED%81%AC%EB%A1%9C%EC%BB%B4%ED%93%A8%ED%84%B0" title="마이크로컴퓨터">마이크로컴퓨터</a>라고도 알려져 있다. 데스크톱 컴퓨터에는 다음과 같이 네 가지 종류가 있다.</p>
<ul>
<li><a class="mw-redirect" href="/wiki/%ED%99%88_%EC%BB%B4%ED%93%A8%ED%84%B0" title="홈 컴퓨터">홈 컴퓨터</a></li>
<li><a href="/wiki/%EA%B0%9C%EC%9D%B8%EC%9A%A9_%EC%BB%B4%ED%93%A8%ED%84%B0" title="개인용 컴퓨터">개인용 컴퓨터</a></li>
<li><a href="/wiki/%EC%9B%8C%ED%81%AC%EC%8A%A4%ED%85%8C%EC%9D%B4%EC%85%98" title="워크스테이션">워크스테이션</a></li>
<li><a href="/wiki/%EC%84%9C%EB%B2%84" title="서버">인터넷 서버</a> (특수 목적의 통신 컴퓨터)</li>

<h2><span class="mw-headline" id=".EC.97.AD.EC.82.AC">역사</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%8D%B0%EC%8A%A4%ED%81%AC%ED%86%B1_%EC%BB%B4%ED%93%A8%ED%84%B0&amp;action=edit&amp;section=1" title="부분 편집: 역사">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>초기의 컴퓨터들은 방 하나를 가득 메웠다. 미니컴퓨터들은 일반적으로 한 대 이상의 냉장고 크기와 맞먹었다. 그러다가 1970년대 들어 <a class="new" href="/w/index.php?title=HP_9800_%EC%8B%9C%EB%A6%AC%EC%A6%88_%EB%8D%B0%EC%8A%A4%ED%81%AC%ED%86%B1_%EC%BB%B4%ED%93%A8%ED%84%B0&amp;action=edit&amp;redlink=1" title="HP 9800 시리즈 데스크톱 컴퓨터 (없는 문서)">HP 9800 시리즈 데스크톱 컴퓨터</a>와 같은 컴퓨터들이 온전히 책상 위에 놓일 수 있게 되었다. 최초의 대형 계산기는 1971년에 도입되어 1972년에 프로그래밍이 가능한 <a href="/wiki/%EB%B2%A0%EC%9D%B4%EC%A7%81" title="베이직">베이직</a> 모델로 이어졌다. 이들은 ROM 메모리를 기반으로 하는 <a href="/wiki/%EB%AF%B8%EB%8B%88%EC%BB%B4%ED%93%A8%ED%84%B0" title="미니컴퓨터">미니컴퓨터</a>의 소형판이었으며 조그마한 온라인 LED 디스플레이가 장착되어 있었다. 이들은 플로터로 컴퓨터 그래픽스를 그리는 기능을 제공하였다. 1973년 <a href="/wiki/%EC%99%95_%EC%97%B0%EA%B5%AC%EC%86%8C" title="왕 연구소">왕 연구소</a>사의 <a class="new" href="/w/index.php?title=%EC%99%95_2200&amp;action=edit&amp;redlink=1" title="왕 2200 (없는 문서)">왕 2200</a>은 완전한 크기의 CRT 및 카세트 테이프 기억 장치가 있었다. 1975년의 <a href="/wiki/IBM_5100" title="IBM 5100">IBM 5100</a>는 조그마한 <a href="/wiki/%EC%9D%8C%EA%B7%B9%EC%84%A0%EA%B4%80" title="음극선관">CRT</a> 디스플레이가 있었으며 베이직과 <a class="mw-disambig" href="/wiki/APL" title="APL">APL</a> 프로그래밍 언어로 프로그래밍할 수 있었다. 이러한 컴퓨터들은 너무 비쌌기에 비즈니스와 과학용으로 판매되었다. 1970년대와 1980년대 말 들어 <a href="/wiki/%EC%95%A0%ED%94%8C_II" title="애플 II">애플 II</a> 시리즈와 <a href="/wiki/IBM_PC" title="IBM PC">IBM PC</a>와 같은 <a href="/wiki/%EA%B0%9C%EC%9D%B8%EC%9A%A9_%EC%BB%B4%ED%93%A8%ED%84%B0" title="개인용 컴퓨터">개인용 컴퓨터</a>는 표준 프로세서를 사용하고 별도의 모니터를 책상 위에 놓음으로써 가격을 절감하였다. 이로써 비즈니스와 산업뿐 아니라 집에서도 이용이 가능하게 되었다. 나중에는 맥(<a href="/wiki/%EB%A7%A4%ED%82%A8%ED%86%A0%EC%8B%9C" title="매킨토시">매킨토시</a>)과 윈도의 운영 체제를 통해 그래픽 사용자 인터페이스를 이용할 수 있게 되었다.</p>
<h3><span class="mw-headline" id=".ED.98.84.EC.9E.AC">현재</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%8D%B0%EC%8A%A4%ED%81%AC%ED%86%B1_%EC%BB%B4%ED%93%A8%ED%84%B0&amp;action=edit&amp;section=2" title="부분 편집: 현재">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>현대의 데스크톱 컴퓨터들은 (다른 종류의 컴퓨터에 비해) 돈을 들이기 쉽고 사업, 학교, 가정 안에 어디든 존재한다. 거의 모든 데스크톱 컴퓨터들은 바꾸거나 업그레이드하기 쉬운 부품들을 포함하고 있다. 데스크톱 컴퓨터들은 다양한 <a href="/wiki/%EC%BB%B4%ED%93%A8%ED%84%B0_%EC%BC%80%EC%9D%B4%EC%8A%A4" title="컴퓨터 케이스">케이스</a> 스타일로 출시된다. 이를테면 대형 수직 타워부터 <a class="mw-redirect" href="/wiki/LCD_%EB%AA%A8%EB%8B%88%ED%84%B0" title="LCD 모니터">LCD 모니터</a>가 뒤에 놓여 있는 소형 <a class="new" href="/w/index.php?title=%ED%8F%BC_%ED%8C%A9%ED%84%B0_(%EC%BB%B4%ED%93%A8%ED%8C%85)&amp;action=edit&amp;redlink=1" title="폼 팩터 (컴퓨팅) (없는 문서)">폼 팩터</a> 모델까지이다. 데스크톱 컴퓨터들은 보통 <a class="mw-redirect" href="/wiki/%ED%8C%8C%EC%9B%8C_%EC%8A%A4%ED%8A%B8%EB%A6%BD" title="파워 스트립">파워 스트립</a>에 연결하기도 한다. 모니터와 본체가 결합하여 나온 일체형 PC가 있다.</p>
