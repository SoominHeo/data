<h1 class="firstHeading" id="firstHeading" lang="ko">데시벨</h1>
<div class="mw-body-content" id="bodyContent">
<div id="siteSub">위키백과, 우리 모두의 백과사전.</div>
<div id="contentSub"></div>
<div class="mw-jump" id="jump-to-nav">
					이동:					<a href="#mw-head">둘러보기</a>, 					<a href="#p-search">검색</a>
</div>
<div class="mw-content-ltr" dir="ltr" id="mw-content-text" lang="ko"><script>function mfTempOpenSection(id){var block=document.getElementById("mf-section-"+id);block.className+=" open-block";block.previousSibling.className+=" open-block";}</script><p><b>데시벨</b>(decibel, <b>㏈</b>)은 전기공학이나 진동·음향공학 등에서 사용되는 무차원의 단위이다. 데시벨은 <a href="/wiki/%EA%B5%AD%EC%A0%9C%EB%8B%A8%EC%9C%84%EA%B3%84" title="국제단위계">국제단위계</a>(SI)에서 'SI와 함께 쓰지만, SI에 속하지 않는 단위'로 규정되어 있다.<sup class="reference" id="cite_ref-1"><a href="#cite_note-1">[1]</a></sup></p>
<p>데시벨은 소리의 어떤 기준 <a href="/wiki/%EC%A0%84%EB%A0%A5" title="전력">전력</a>에 대한 전력 비의 <a href="/wiki/%EC%83%81%EC%9A%A9%EB%A1%9C%EA%B7%B8" title="상용로그">상용로그</a> 값을 벨(bel)으로서, 그것을 다시 10분의 1배(=데시[d])한 변환이다. 벨(bel)의 10분의 일이란 의미에서 데시벨[dB]이며, 벨이 상용에서는 너무 큰 값이기에 그대로 쓰기는 힘들기 때문에 통상적으로는 데시벨을 이용한다. 소리의 강함(<a class="mw-redirect" href="/wiki/%EC%9D%8C%EC%95%95_%EB%A0%88%EB%B2%A8" title="음압 레벨">음압 레벨</a>, SPL)·전력 등의 비교나 감쇠량 등을 에너지 비로 나타낼 때에도 사용된다. 어떤 기준치 <i>A</i>에 대하여 <i>B</i>의 데시벨 값 <i>L <sub>B</sub></i>은</p>

<p>이 된다.</p>
<p><a href="/wiki/%EC%97%B0%EC%82%B0_%EC%A6%9D%ED%8F%AD%EA%B8%B0" title="연산 증폭기">연산 증폭기</a> 등의 증폭기의 입력 및 출력 전압비(<a class="new" href="/w/index.php?title=%EC%9D%B4%EB%93%9D_(%EC%A0%84%EA%B8%B0_%EA%B3%B5%ED%95%99)&amp;action=edit&amp;redlink=1" title="이득 (전기 공학) (없는 문서)">이득</a>, gain)을 나타내는 단위로도 사용하며, 다음 환산식에 적용하여 구한다.</p>

<p>(단, G:데시벨로 환산한 입력/출력비 x:입력, y:출력)</p>
<p>전압비의 제곱이 전력비이므로, 앞의 전력비의 정의식을 전압비로 다시 쓰면 로그의 계수가 2배의 20이 된다. 입력과 출력의 차이는 0에 가까운 수치에서부터 100만 단위의 큰 수치까지 폭넓은 범위를 가질 수 있으므로, 이러한 방법을 쓴다.</p>
<p>전력은 전압(혹은 전류)의 제곱에 비례하므로, 10배의 전압(전류)비는 100배의 전력비가 되고, 데시벨로 나타내면 20dB가 된다. 음압 레벨에 대해서도 같다. <a class="mw-redirect" href="/wiki/%EC%9D%8C%EC%95%95_%EB%A0%88%EB%B2%A8" title="음압 레벨">음압 레벨</a>, 진동 가속도 레벨은 각각 음압(Pa) 및 진동의 가속도(m/s²)에 기초해 정의된, 레벨화하여 표현한 양(절대데시벨)이다. 전자파의 감쇠량은 상대비를 데시벨로 표현한 것(상대데시벨)이다.</p>
<h2><span class="mw-headline" id=".EC.A0.88.EB.8C.80.EA.B0.92.EC.9C.BC.EB.A1.9C.EC.84.9C.EC.9D.98_.EB.8D.B0.EC.8B.9C.EB.B2.A8">절대값으로서의 데시벨</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%8D%B0%EC%8B%9C%EB%B2%A8&amp;action=edit&amp;section=1" title="부분 편집: 절대값으로서의 데시벨">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>데시벨은 원래 두 가지의 전력의 비를 나타내는 무차원의 양이지만, 공학에서는 관습적으로 절대기준값을 정하여 절대 단위로 사용하는 경우가 많다.</p>
<ul>
<li>dBSPL(sound pressure level, 음압) : <a href="/wiki/%EC%9D%8C%EC%95%95" title="음압">음압</a>(소리의 매질인 공기의 압력의 <a class="new" href="/w/index.php?title=%EC%8B%A4%ED%9A%A8%EC%B6%9C%EB%A0%A5&amp;action=edit&amp;redlink=1" title="실효출력 (없는 문서)">실효출력</a>)의 단위는 <a href="/wiki/%ED%8C%8C%EC%8A%A4%EC%B9%BC_(%EB%8B%A8%EC%9C%84)" title="파스칼 (단위)">파스칼</a>(Pa)이지만, 2×10<sup>−5</sup>Pa를 기준값(0dBSPL)으로 한다.</li>
</ul>
<ul>
<li>마찬가지로 음의 강한 수준(단위 단면적을 단위 시간에 통과하는 음향의 <a href="/wiki/%EC%97%90%EB%84%88%EC%A7%80" title="에너지">에너지</a>)의 단위는 W・m<sup>−2</sup>이지만, 10<sup>−12</sup>W・m<sup>−2</sup>를 기준값(0 dB)으로 한 값을 dB로 표현한다.</li>
</ul>
<ul>
<li>dBm : 1mW를 0dB로 한 것. 예를 들면 600옴의 회로에 1mW의 전력을 공급하는 데 필요한 교류 전압은 약 0.775V<sub>rms</sub>다.</li>
</ul>
<ul>
<li>dBV : 1V<sub>rms</sub>를 0dBV로 한 전압(저항과 관련 없음). 주로 가정용 오디오 기기에서 사용되는 음성 신호 레벨의 기준. 통상 마이크의 출력이 -40 dB(10mV<sub>rms</sub>)에서 -50 dB(3mV<sub>rms</sub>) 정도이다.</li>
</ul>
<ul>
<li>dBu : 0.775V를 0dB로 한 것으로, 전압의 강함을 dB로 나타낸 것. 부하의 <a class="mw-redirect" href="/wiki/%EC%9E%84%ED%94%BC%EB%8D%98%EC%8A%A4" title="임피던스">임피던스</a>는 관계없다. 주로 업무용 오디오 기기에서 사용되는 음성 신호 레벨의 기준.</li>
</ul>
<ul>
<li>dBi : 등방성 안테나(전 방향에 균등한 전파를 쏘는 가상의 안테나)를 기준으로 한 안테나의 이득. <a class="new" href="/w/index.php?title=%EC%8C%8D%EA%B7%B9_%EC%95%88%ED%85%8C%EB%82%98&amp;action=edit&amp;redlink=1" title="쌍극 안테나 (없는 문서)">쌍극 안테나</a>를 기준으로 하는 경우에는 dBd나 dB라고 쓴다. dBi 단위는 dBd보다 2.14 크다.</li>
</ul>
<ul>
<li>dBμ : 1μV<sub>rms</sub>(0．000001V<sub>rms</sub>)를 0dB로 한 것. 주로 무선 통신에서 사용된다.</li>
</ul>
<ul>
<li>dBμEMF, dBμ(emf) : 무선 통신에서 SG(고주파 신호발생기)의 출력 전압을 표현하는 경우, SG의 출력을 신호원 임피던스로 종단했을 때의 전압(종단전압)으로 표현하는 경우와, SG의 출력을 개방했을 때의 전압(개방전압)으로 표현하는 경우가 있다. 양자는 6dB의 차이가 있기 때문에, 개방전압으로 표현하는 경우에는 dBμ에 이어 EMF(Electro Motive Force의 약자)를 써주거나 따로 설명해 줄 필요가 있다. 예를 들면 50Ω의 경우 113dBμEMF과 107dBμ은 같은 표현으로 약 0dBm이다. 일본에서는 업무용 무선기나 <a class="new" href="/w/index.php?title=PDC&amp;action=edit&amp;redlink=1" title="PDC (없는 문서)">PDC</a>방식 휴대폰기기에서 dBμEMF가 자주 사용된다. 미국이나 아마추어 무선에서는 dBμ가 자주 사용된다. 규격이나 사양에서는 EMF가 생략되는 경우도 많기 때문에 주의를 요한다. 예를 들면 -6dBμ의 수신 감도의 업무 무선기와 -12dBμ의 수신 감도의 아마추어 무선기는 같은 수신 감도를 갖지만, 아마추어 무선기쪽이 더 좋은 수신 감도를 가진다고 오인할 우려가 있다. dBm으로 표기하면 틀릴 우려는 없다.</li>
</ul>
