<p><b>데어라 그레이스 토러스</b>(Dara Grace Torres, <a href="/wiki/1967%EB%85%84" title="1967년">1967년</a> <a href="/wiki/4%EC%9B%94_15%EC%9D%BC" title="4월 15일">4월 15일</a> ~ )는 <a href="/wiki/%EB%AF%B8%EA%B5%AD" title="미국">미국</a>의 국제적 <a href="/wiki/%EC%88%98%EC%98%81" title="수영">수영</a> 선수이자, 12개의 <a href="/wiki/%EC%98%AC%EB%A6%BC%ED%94%BD" title="올림픽">올림픽</a> 메달을 획득하였다. 그녀는 5개의 올림픽(1984, 1988, 1992, 2000, 2008)에 참가한 첫 미국 수영 선수였으며, 41세의 나이로 미국 올림픽 팀에 나간 연장자 수영 선수이기도 하다. <a class="mw-redirect" href="/wiki/2008%EB%85%84_%EB%B2%A0%EC%9D%B4%EC%A7%95_%EC%98%AC%EB%A6%BC%ED%94%BD" title="2008년 베이징 올림픽">2008년 베이징 올림픽</a>에서 50m <a href="/wiki/%EC%9E%90%EC%9C%A0%ED%98%95_(%EC%88%98%EC%98%81)" title="자유형 (수영)">자유형</a>, 400m 혼계영, 400m 자유형 계주에서 모두 <a href="/wiki/%EC%9D%80%EB%A9%94%EB%8B%AC" title="은메달">은메달</a>을 획득하였다.</p>
<p><a href="/wiki/2007%EB%85%84" title="2007년">2007년</a> <a href="/wiki/8%EC%9B%94_1%EC%9D%BC" title="8월 1일">8월 1일</a> 40세의 나이로 <a href="/wiki/%EC%9D%B8%EB%94%94%EC%95%A0%EB%82%98%ED%8F%B4%EB%A6%AC%EC%8A%A4" title="인디애나폴리스">인디애나폴리스</a>에서 열린 미국 국립 대회에서 100m 자유형 종목을 우승하였다. <a href="/wiki/8%EC%9B%94_4%EC%9D%BC" title="8월 4일">8월 4일</a>에는 50m 자유형에서 자신의 미국 기록을 깼다.</p>

<h2><span class="mw-headline" id=".EC.B4.88.EA.B8.B0_.EC.83.9D.EC.95.A0.EC.99.80_.EA.B5.90.EC.9C.A1">초기 생애와 교육</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%8D%B0%EC%96%B4%EB%9D%BC_%ED%86%A0%EB%9F%AC%EC%8A%A4&amp;action=edit&amp;section=1" title="부분 편집: 초기 생애와 교육">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p><a href="/wiki/%EC%BA%98%EB%A6%AC%ED%8F%AC%EB%8B%88%EC%95%84_%EC%A3%BC" title="캘리포니아 주">캘리포니아 주</a> <a class="mw-redirect" href="/wiki/%EB%B9%84%EB%B2%8C%EB%A6%AC%ED%9E%90%EC%8A%A4" title="비벌리힐스">비벌리힐스</a>에서 태어난 토러스는 7세의 나이로 지방 <a class="mw-redirect" href="/wiki/YMCA" title="YMCA">YMCA</a>에 수영을 연습하러 자신의 형제들을 따라다녔으며 컬버 시티 수영 팀에 가입하였다.</p>
<p>웨스트레이크 여자 학교(현재의 하버드웨스트레이크 학교)에서 수학하면서 학교 수영 팀을 위해 뛰었으며, 캘리포니아 학교 간 연맹 기록들을 세웠다. 1980년대에 10대로서 그녀는 후에 미션 비에조에서 <a href="/wiki/2008%EB%85%84_%ED%95%98%EA%B3%84_%EC%98%AC%EB%A6%BC%ED%94%BD" title="2008년 하계 올림픽">베이징 올림픽</a> 수영 코치가 된 마크 슈버트 아래 미션 비에조 나다도레스를 위해 활약하였다.</p>
<p>토러스는 <a href="/wiki/%ED%94%8C%EB%A1%9C%EB%A6%AC%EB%8B%A4_%EB%8C%80%ED%95%99%EA%B5%90" title="플로리다 대학교">플로리다 대학교</a>에 입학하는 데 선수 <a href="/wiki/%EC%9E%A5%ED%95%99%EA%B8%88" title="장학금">장학금</a>을 받고, <a href="/wiki/1985%EB%85%84" title="1985년">1985년</a>부터 <a href="/wiki/1989%EB%85%84" title="1989년">1989년</a>까지 랜디 리즈 코치 아래에서 플로리다 케이터즈 수영 팀과 NCAA 선수권 대회의 <a href="/wiki/%EB%8B%A4%EC%9D%B4%EB%B9%99" title="다이빙">다이빙</a> 팀에서 활약하였다. 게이터 수영 선수로서 4년 동안에 그녀는 <a href="/wiki/1986%EB%85%84" title="1986년">1986년</a>의 400m 자유형 계주, <a href="/wiki/1988%EB%85%84" title="1988년">1988년</a>의 50m 자유형, 100m 자유형, 100m 접영, 200m 혼계영, 400m 혼계영과 400m 자유형 계주, <a href="/wiki/1989%EB%85%84" title="1989년">1989년</a>의 200m 혼계영과 400m 혼계영을 포함한 9개의 SEC 선수권 대회와 9개의 NCAA 선수권 대회의 타이틀을 우승하였다. <a href="/wiki/%ED%94%8C%EB%A1%9C%EB%A6%AC%EB%8B%A4_%EC%A3%BC" title="플로리다 주">플로리다 주</a>에서 "올해의 SEC 선수"(1988)로 선정되었다.</p>
<p><a href="/wiki/1990%EB%85%84" title="1990년">1990년</a> <a href="/wiki/%ED%86%B5%EC%8B%A0%EA%B3%B5%ED%95%99" title="통신공학">통신공학</a> 학사 학위를 받고 졸업하였으며, <a href="/wiki/1999%EB%85%84" title="1999년">1999년</a> "게이터의 위대한 선수"로서 플로리다 대학교 선수 명예의 전당에 헌액되었다.</p>
<h2><span class="mw-headline" id=".EC.98.AC.EB.A6.BC.ED.94.BD_.EA.B2.BD.EB.A0.A5">올림픽 경력</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%8D%B0%EC%96%B4%EB%9D%BC_%ED%86%A0%EB%9F%AC%EC%8A%A4&amp;action=edit&amp;section=2" title="부분 편집: 올림픽 경력">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<h3><span class="mw-headline" id="1984.EB.85.84_.ED.95.98.EA.B3.84_.EC.98.AC.EB.A6.BC.ED.94.BD">1984년 하계 올림픽</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%8D%B0%EC%96%B4%EB%9D%BC_%ED%86%A0%EB%9F%AC%EC%8A%A4&amp;action=edit&amp;section=3" title="부분 편집: 1984년 하계 올림픽">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p><a class="mw-redirect" href="/wiki/1984%EB%85%84_%EB%A1%9C%EC%8A%A4%EC%95%A4%EC%A0%A4%EB%A0%88%EC%8A%A4_%EC%98%AC%EB%A6%BC%ED%94%BD" title="1984년 로스앤젤레스 올림픽">1984년 로스앤젤레스 올림픽</a>에 17세의 나이로 참가한 토러스는 400m 자유형 계주에서 <a href="/wiki/%EA%B8%88%EB%A9%94%EB%8B%AC" title="금메달">금메달</a>을 획득한 미국 팀의 일원이었다. 그녀의 팀 돌료들로는 <a class="new" href="/w/index.php?title=%EB%82%B8%EC%8B%9C_%ED%98%B8%EA%B7%B8%EC%85%B0%EB%93%9C&amp;action=edit&amp;redlink=1" title="낸시 호그셰드 (없는 문서)">낸시 호그셰드</a>, <a class="new" href="/w/index.php?title=%EC%A0%9C%EB%82%98_%EC%A1%B4%EC%8A%A8&amp;action=edit&amp;redlink=1" title="제나 존슨 (없는 문서)">제나 존슨</a>, <a class="new" href="/w/index.php?title=%EC%BA%90%EB%A6%AC_%EC%8A%A4%ED%83%80%EC%9D%B8%EC%82%AC%EC%9D%B4%ED%8D%BC&amp;action=edit&amp;redlink=1" title="캐리 스타인사이퍼 (없는 문서)">캐리 스타인사이퍼</a>였으며, <a class="new" href="/w/index.php?title=%EC%A7%88_%EC%8A%A4%ED%84%B0%EC%BC%88&amp;action=edit&amp;redlink=1" title="질 스터켈 (없는 문서)">질 스터켈</a>과 <a class="new" href="/w/index.php?title=%EB%A9%94%EB%A6%AC_%EC%9B%A8%EC%9D%B4%ED%8A%B8&amp;action=edit&amp;redlink=1" title="메리 웨이트 (없는 문서)">메리 웨이트</a>도 두 번째 라운드 예선에서 뛰었다.</p>
<h3><span class="mw-headline" id="1988.EB.85.84_.ED.95.98.EA.B3.84_.EC.98.AC.EB.A6.BC.ED.94.BD">1988년 하계 올림픽</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%8D%B0%EC%96%B4%EB%9D%BC_%ED%86%A0%EB%9F%AC%EC%8A%A4&amp;action=edit&amp;section=4" title="부분 편집: 1988년 하계 올림픽">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p><a class="mw-redirect" href="/wiki/1988%EB%85%84_%EC%84%9C%EC%9A%B8_%EC%98%AC%EB%A6%BC%ED%94%BD" title="1988년 서울 올림픽">1988년 서울 올림픽</a>을 위하여 토러스는 미국 여자 수영팀을 위해 1개의 개인 종복과 2개의 계영 종목들에 합격하였다. 그녀는 메리 웨이트, <a class="new" href="/w/index.php?title=%EB%A1%9C%EB%9D%BC_%EC%9B%8C%EC%BB%A4&amp;action=edit&amp;redlink=1" title="로라 워커 (없는 문서)">로라 워커</a>, <a class="new" href="/w/index.php?title=%EB%AF%B8%EC%B9%98_%ED%81%AC%EB%A0%88%EB%A8%B8&amp;action=edit&amp;redlink=1" title="미치 크레머 (없는 문서)">미치 크레머</a>와 함께 400m 자유형 계주에서 <a href="/wiki/%EB%8F%99%EB%A9%94%EB%8B%AC" title="동메달">동메달</a>을 따고, 400m 혼계영에서 <a href="/wiki/%EC%9D%80%EB%A9%94%EB%8B%AC" title="은메달">은메달</a>을 땄다. 100m 자유형 종목에서는 7위에 그쳤다.</p>
<h3><span class="mw-headline" id="1992.EB.85.84_.ED.95.98.EA.B3.84_.EC.98.AC.EB.A6.BC.ED.94.BD">1992년 하계 올림픽</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%8D%B0%EC%96%B4%EB%9D%BC_%ED%86%A0%EB%9F%AC%EC%8A%A4&amp;action=edit&amp;section=5" title="부분 편집: 1992년 하계 올림픽">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>토러스는 <a class="mw-redirect" href="/wiki/1992%EB%85%84_%EB%B0%94%EB%A5%B4%EC%85%80%EB%A1%9C%EB%82%98_%EC%98%AC%EB%A6%BC%ED%94%BD" title="1992년 바르셀로나 올림픽">1992년 바르셀로나 올림픽</a>에 출전하여 <a href="/wiki/%EB%8B%88%EC%BD%9C_%ED%97%A4%EC%9D%B4%EC%A6%90%EB%A0%9B" title="니콜 헤이즐렛">니콜 헤이즐렛</a>, <a class="new" href="/w/index.php?title=%EC%95%A4%EC%A0%A4_%EB%A7%88%ED%8B%B0%EB%85%B8&amp;action=edit&amp;redlink=1" title="앤젤 마티노 (없는 문서)">앤젤 마티노</a>, <a href="/wiki/%EC%A0%9C%EB%8B%88_%ED%86%B0%EC%8A%A8" title="제니 톰슨">제니 톰슨</a>을 포함한 400m 자유형 계주 팀에서 2번째 주자로 뛰었으며 결승전과 첫 라운드 예선에서 뛰어난 실력을 보임으로서 금메달을 획득하였다. <a class="new" href="/w/index.php?title=%EC%95%A0%EC%89%B4%EB%A6%AC_%ED%83%9C%ED%95%80&amp;action=edit&amp;redlink=1" title="애쉴리 태핀 (없는 문서)">애쉴리 태핀</a>과 <a class="new" href="/w/index.php?title=%ED%81%AC%EB%A6%AC%EC%8A%A4_%EC%95%84%EB%A7%A8&amp;action=edit&amp;redlink=1" title="크리스 아맨 (없는 문서)">크리스 아맨</a>도 예선전에서 팀을 위하여 뛰었다.</p>
<h3><span class="mw-headline" id="2000.EB.85.84_.ED.95.98.EA.B3.84_.EC.98.AC.EB.A6.BC.ED.94.BD">2000년 하계 올림픽</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%8D%B0%EC%96%B4%EB%9D%BC_%ED%86%A0%EB%9F%AC%EC%8A%A4&amp;action=edit&amp;section=6" title="부분 편집: 2000년 하계 올림픽">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p><a class="mw-redirect" href="/wiki/2000%EB%85%84_%EC%8B%9C%EB%93%9C%EB%8B%88_%EC%98%AC%EB%A6%BC%ED%94%BD" title="2000년 시드니 올림픽">2000년 시드니 올림픽</a>에서 토러스는 5개의 메달을 획득하였으나, 이전과 마찬가지로 2개의 계영 종목을 우승하였다. <a class="new" href="/w/index.php?title=%EC%97%90%EC%9D%B4%EB%AF%B8_%EB%B0%98_%EB%8B%A4%EC%9D%B4%EC%BC%84&amp;action=edit&amp;redlink=1" title="에이미 반 다이켄 (없는 문서)">에이미 반 다이켄</a>, <a class="new" href="/w/index.php?title=%EC%BB%A4%ED%8A%B8%EB%8B%88_%EC%89%B4%EB%A6%AC&amp;action=edit&amp;redlink=1" title="커트니 쉴리 (없는 문서)">커트니 쉴리</a>와 제니 톰슨이 속하였던 400m 계주에서 다시 2번째 주자로 뛰었으며, 애쉴리 태핀과 <a class="new" href="/w/index.php?title=%EC%97%90%EB%A6%B0_%ED%94%BC%EB%8B%89%EC%8A%A4&amp;action=edit&amp;redlink=1" title="에린 피닉스 (없는 문서)">에린 피닉스</a>도 예선에서 뛰었다. 또한 반 다이켄, 태핀, 쉴리, <a class="new" href="/w/index.php?title=%EC%8A%A4%ED%83%9C%EC%83%A4%EB%82%98_%EC%8A%A4%ED%8B%B0%EC%B8%A0&amp;action=edit&amp;redlink=1" title="스태샤나 스티츠 (없는 문서)">스태샤나 스티츠</a>가 예선에서 뛰고, 톰슨, <a class="new" href="/w/index.php?title=B.J._%EB%B2%A0%EB%93%9C%ED%8F%AC%EB%93%9C&amp;action=edit&amp;redlink=1" title="B.J. 베드포드 (없는 문서)">B.J. 베드포드</a>, <a class="mw-redirect" href="/wiki/%EB%A9%94%EA%B0%84_%EC%A0%A0%EB%93%9C%EB%A6%AD" title="메간 젠드릭">메간 쿠안</a>이 결승전에 뛰었던 400m 혼계영에서도 2번째 금메달을 땄다.</p>
<p>토러스는 또한 50m, 100m 자유형, 100m <a href="/wiki/%EC%A0%91%EC%98%81" title="접영">접영</a>에서 3개의 동메달을 따기도 하였다. 33세의 나이로 토러스는 미국 수영 팀 선수들 중에 연장자였으나 다른 동료들보다 더 많은 메달을 따냈다.</p>
<h3><span class="mw-headline" id="2008.EB.85.84_.ED.95.98.EA.B3.84_.EC.98.AC.EB.A6.BC.ED.94.BD">2008년 하계 올림픽</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%8D%B0%EC%96%B4%EB%9D%BC_%ED%86%A0%EB%9F%AC%EC%8A%A4&amp;action=edit&amp;section=7" title="부분 편집: 2008년 하계 올림픽">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>41세의 나이로 토러스는 5번째 올림픽 출전을 하게 되는 데 올림픽 역사상 40세 이상의 나이로 출전한 첫 여성 선수이다.</p>
<p><a href="/wiki/2008%EB%85%84" title="2008년">2008년</a> <a href="/wiki/7%EC%9B%94_5%EC%9D%BC" title="7월 5일">7월 5일</a> 50m 자유형에 출전하는 데 합격한 토러스는 준결승전에서 24.38초와 함께 미국 기록을 깼다. <a href="/wiki/7%EC%9B%94_6%EC%9D%BC" title="7월 6일">7월 6일</a> 24.25초를 세우면서 9번째 기록을 깨고 그 종목에서 정상 미국 여성의 순위를 차지한다. <a href="/wiki/7%EC%9B%94_30%EC%9D%BC" title="7월 30일">7월 30일</a> <a href="/wiki/%EC%8B%B1%EA%B0%80%ED%8F%AC%EB%A5%B4" title="싱가포르">싱가포르</a>에서 미국 수영 팀의 마지막 훈련에서 토러스는 <a href="/wiki/%EC%96%B4%EB%A7%A8%EB%8B%A4_%EB%B9%84%EC%96%B4%EB%93%9C" title="어맨다 비어드">어맨다 비어드</a>와 <a href="/wiki/%EB%82%B4%ED%84%B8%EB%A6%AC_%EC%BD%94%EA%B8%80%EB%A6%B0" title="내털리 코글린">내털리 코글린</a>과 함께 미국 올림픽 여자 수영 팀의 주장으로 뽑혔다.</p>
<p><a href="/wiki/2008%EB%85%84_%ED%95%98%EA%B3%84_%EC%98%AC%EB%A6%BC%ED%94%BD" title="2008년 하계 올림픽">베이징 올림픽</a>에서 토러스는 400m 자유형 계주의 최종주자로 뛰면서 은메달을 땄다. 이것은 그녀가 그 올림픽 종목의 5개 시도에서 5번째 메달을 따는 기록이 되었다.</p>
<p><a href="/wiki/8%EC%9B%94_17%EC%9D%BC" title="8월 17일">8월 17일</a>에는 50m 자유형에서 은메달을 따는 데 미국 신기록 24.07초를 세우면서 <a href="/wiki/%EB%B8%8C%EB%A6%AC%ED%83%80_%EC%8A%88%ED%85%8C%ED%8E%9C" title="브리타 슈테펜">브리타 슈테펜</a>에게 0.01초 차이로 들어왔다. 35분 후에 그녀는 400m 혼계영에서 또다른 은메달을 땄다. 400m 혼계영에서 토러스의 전속력(52.27초)은 계영 역사상 제일 빠른 100m 자유형 속력이었다. 개인적 100m 자유형에서 세운 미국 기록 53.39초는 토러스의 시간을 두 번째로 빠르게 만들었다.</p>
<h3><span class="mw-headline" id="2009.EB.85.84_.EA.B5.AD.EB.A6.BD_.EC.84.A0.EC.88.98.EA.B6.8C_.EB.8C.80.ED.9A.8C.EC.99.80_.EC.84.B8.EA.B3.84_.EC.84.A0.EC.88.98.EA.B6.8C_.EB.8C.80.ED.9A.8C">2009년 국립 선수권 대회와 세계 선수권 대회</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%8D%B0%EC%96%B4%EB%9D%BC_%ED%86%A0%EB%9F%AC%EC%8A%A4&amp;action=edit&amp;section=8" title="부분 편집: 2009년 국립 선수권 대회와 세계 선수권 대회">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>미국 국립 선수권 대회에서 토러스는 <a href="/wiki/2009%EB%85%84" title="2009년">2009년</a> <a href="/wiki/%EB%A1%9C%EB%A7%88" title="로마">로마</a>에서 열리는 세계 선수권 대회에 50m 자유형과 50m 접영에 출전하게 되었다. 이것은 <a href="/wiki/1986%EB%85%84" title="1986년">1986년</a> 이래 토러스가 세계 선수권 대회 출전하는 데 처음이었다. 자유형에서 8위에 머물고, 접영에서는 예선에서 탈락하였다.</p>

<h2><span class="mw-headline" id=".EA.B7.B8.EB.B0.96.EC.9D.98_.ED.99.9C.EB.8F.99">그밖의 활동</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%8D%B0%EC%96%B4%EB%9D%BC_%ED%86%A0%EB%9F%AC%EC%8A%A4&amp;action=edit&amp;section=9" title="부분 편집: 그밖의 활동">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>토러스는 <a href="/wiki/NBC" title="NBC">NBC</a>, <a href="/wiki/ESPN" title="ESPN">ESPN</a>, <a class="mw-disambig" href="/wiki/TNT" title="TNT">TNT</a>, OLN 같은 방송국과 폭스 뉴스 채널에서 <a class="mw-redirect" href="/wiki/%EB%A6%AC%ED%8F%AC%ED%84%B0" title="리포터">리포터</a>와 <a href="/wiki/%EC%95%84%EB%82%98%EC%9A%B4%EC%84%9C" title="아나운서">아나운서</a>에서 일하였다. 그녀는 현재 리조트 스포츠 네트워크에서 <a href="/wiki/%EA%B3%A8%ED%94%84" title="골프">골프</a> 쇼 "The Clubhouse"의 진행을 맡고 있다. 또한 임시 <a class="mw-disambig" href="/wiki/%EB%AA%A8%EB%8D%B8" title="모델">모델</a>이기도 한 그녀는 <a href="/wiki/1994%EB%85%84" title="1994년">1994년</a> 수영복 발행을 위한 <a href="/wiki/%EC%8A%A4%ED%8F%AC%EC%B8%A0_%EC%9D%BC%EB%9F%AC%EC%8A%A4%ED%8A%B8%EB%A0%88%EC%9D%B4%ED%8B%B0%EB%93%9C" title="스포츠 일러스트레이티드">스포츠 일러스트레이티드</a>에 나오기도 하였다.</p>
<p>그녀는 또한 영감적 회고록 "나이는 그냥 숫자이다(Age is Just a Number)"(<a href="/wiki/2009%EB%85%84" title="2009년">2009년</a> <a href="/wiki/4%EC%9B%94" title="4월">4월</a>에 발간)의 저자이며, 다른 저서 "금메달 건강(Gold Medal Fitness)"는 <a href="/wiki/2010%EB%85%84" title="2010년">2010년</a> <a href="/wiki/5%EC%9B%94" title="5월">5월</a>에 발간되었다.</p>
<p><a href="/wiki/2005%EB%85%84" title="2005년">2005년</a> 국제 유대인 스포츠 명예의 전당에 입성되었다.</p>
