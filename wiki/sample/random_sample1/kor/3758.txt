<h2><span class="mw-headline" id=".EC.97.AD.EC.82.AC">역사</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%8D%B0%EC%98%A4%EB%8F%84%EB%9E%80%ED%8A%B8&amp;action=edit&amp;section=1" title="부분 편집: 역사">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>1888년, 최초의 상용 데오도란트 제품인 Mum이 미국 <a href="/wiki/%ED%8E%9C%EC%8B%A4%EB%B2%A0%EC%9D%B4%EB%8B%88%EC%95%84_%EC%A3%BC" title="펜실베이니아 주">펜실베이니아 주</a> <a href="/wiki/%ED%95%84%EB%9D%BC%EB%8D%B8%ED%94%BC%EC%95%84" title="필라델피아">필라델피아</a>의 한 발명가에 의해 개발, 특허화되었다.<sup class="reference" id="cite_ref-Joey_Green_1-0"><a href="#cite_note-Joey_Green-1">[1]</a></sup></p>
<h2><span class="mw-headline" id=".EB.8D.B0.EC.98.A4.EB.8F.84.EB.9E.80.ED.8A.B8_.EA.B4.80.EB.A0.A8_.EA.B8.B0.EC.82.AC">데오도란트 관련 기사</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%8D%B0%EC%98%A4%EB%8F%84%EB%9E%80%ED%8A%B8&amp;action=edit&amp;section=2" title="부분 편집: 데오도란트 관련 기사">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<ul>
<li><a href="/wiki/2011%EB%85%84" title="2011년">2011년</a> <a href="/wiki/8%EC%9B%94_16%EC%9D%BC" title="8월 16일">8월 16일</a> <a href="/wiki/%EB%8C%80%ED%95%9C%EB%AF%BC%EA%B5%AD" title="대한민국">대한민국</a>의 <a href="/wiki/%EB%8C%80%ED%95%9C%EB%AF%BC%EA%B5%AD_%EC%8B%9D%ED%92%88%EC%9D%98%EC%95%BD%ED%92%88%EC%95%88%EC%A0%84%EC%B2%AD" title="대한민국 식품의약품안전청">식품의약품안전청</a>은 <a class="new" href="/w/index.php?title=%EC%A0%9C%EB%AA%A8%EC%A0%9C&amp;action=edit&amp;redlink=1" title="제모제 (없는 문서)">제모제</a>와 데오도란트를 동시에 사용할 경우 <a href="/wiki/%ED%94%BC%EB%B6%80" title="피부">피부</a>에 부작용이 생길 수 있음을 경고하였다. 제모제는 제모 작용을 하는 <a class="new" href="/w/index.php?title=%EC%B9%98%EC%98%A4%EA%B8%80%EB%A6%AC%EC%BD%9C%EC%82%B0&amp;action=edit&amp;redlink=1" title="치오글리콜산 (없는 문서)">치오글리콜산</a> 그리고 제모 작용을 돕는 <a class="new" href="/w/index.php?title=%EC%82%B0%EC%84%B1%EB%8F%84_%EC%A1%B0%EC%A0%95%EC%A0%9C&amp;action=edit&amp;redlink=1" title="산성도 조정제 (없는 문서)">산성도 조정제</a>, <a href="/wiki/%EA%B3%84%EB%A9%B4%ED%99%9C%EC%84%B1%EC%A0%9C" title="계면활성제">계면활성제</a> 등의 첨가제로 구성되어 있는데, 제모제를 사용하고 즉시 데오드란트를 사용하면 피부가 받는 자극이 심해져서 발적이 생길 수 있으니, 적어도 24시간의 시차를 두고 사용해야 한다.<sup class="reference" id="cite_ref-2"><a href="#cite_note-2">[2]</a></sup></li>
</ul>
<ul>
<li><a href="/wiki/2005%EB%85%84" title="2005년">2005년</a> <a href="/wiki/5%EC%9B%94_25%EC%9D%BC" title="5월 25일">5월 25일</a> <a href="/wiki/%EB%8C%80%ED%95%9C%EB%AF%BC%EA%B5%AD" title="대한민국">대한민국</a>의 환경 관련 단체인 <a class="new" href="/w/index.php?title=%EC%97%AC%EC%84%B1%ED%99%98%EA%B2%BD%EC%97%B0%EB%8C%80&amp;action=edit&amp;redlink=1" title="여성환경연대 (없는 문서)">여성환경연대</a>에 따르면 한국 내 시중에서 유통되는 6개사(한국기업 3사, 외국기업 3사)의 데오도란트에서 <a class="mw-redirect" href="/wiki/%ED%99%98%EA%B2%BD%ED%98%B8%EB%A5%B4%EB%AA%AC" title="환경호르몬">환경호르몬</a>인 <a class="new" href="/w/index.php?title=%ED%94%84%ED%83%88%EB%A0%88%EC%9D%B4%ED%8A%B8&amp;action=edit&amp;redlink=1" title="프탈레이트 (없는 문서)">프탈레이트</a> 1가지 이상이 검출되었다고 보도 되었다.<sup class="reference" id="cite_ref-3"><a href="#cite_note-3">[3]</a></sup></li>
</ul>
