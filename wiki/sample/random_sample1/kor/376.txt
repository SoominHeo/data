<p><b>2006년 미국 상원의원 선거</b>는 <a href="/wiki/2006%EB%85%84" title="2006년">2006년</a> <a href="/wiki/11%EC%9B%94_7%EC%9D%BC" title="11월 7일">11월 7일</a>에 <a href="/wiki/%EB%AF%B8%EA%B5%AD_%EC%83%81%EC%9B%90" title="미국 상원">미국 상원</a>의 <a class="new" href="/w/index.php?title=%EB%AF%B8%EA%B5%AD_%EC%83%81%EC%9B%90%EC%9D%98_%ED%81%B4%EB%9E%98%EC%8A%A4&amp;action=edit&amp;redlink=1" title="미국 상원의 클래스 (없는 문서)">클래스 1의 의원</a> 33명을 선출하기 위해 실시된 선거이다. 이 선거는 60,839,144명의 유권자가 투표에 참여하여 투표율 29.7%를 기록하였다.</p>
<p>선거 전 <a href="/wiki/%EA%B3%B5%ED%99%94%EB%8B%B9_(%EB%AF%B8%EA%B5%AD)" title="공화당 (미국)">공화당</a>은 지난 <a href="/wiki/2004%EB%85%84_%EB%AF%B8%EA%B5%AD_%EC%83%81%EC%9B%90%EC%9D%98%EC%9B%90_%EC%84%A0%EA%B1%B0" title="2004년 미국 상원의원 선거">2004년 선거</a>에서 55석을 확보해 다수당 위치에 있었다. 그러나 <a href="/wiki/2004%EB%85%84_%EB%AF%B8%EA%B5%AD_%EB%8C%80%ED%86%B5%EB%A0%B9_%EC%84%A0%EA%B1%B0" title="2004년 미국 대통령 선거">2004년 대선</a> 이후 <a href="/wiki/%EC%9D%B4%EB%9D%BC%ED%81%AC_%EC%A0%84%EC%9F%81" title="이라크 전쟁">이라크 전쟁</a>의 후유증과 <a href="/wiki/%ED%97%88%EB%A6%AC%EC%BC%80%EC%9D%B8_%EC%B9%B4%ED%8A%B8%EB%A6%AC%EB%82%98" title="허리케인 카트리나">카트리나</a> 피해 복구 지연으로 인한 <a href="/wiki/%EC%A1%B0%EC%A7%80_W._%EB%B6%80%EC%8B%9C" title="조지 W. 부시">조지 W. 부시</a> 대통령의 지지도 하락과 공화당 내의 지도층 비리 문제로 인해 지지율이 하락해 다수당 지위를 상실할 위기에 처하였다.</p>
<p>선거 결과, 공화당은 6개의 지역을 민주당에 빼앗겨 49석으로 줄어들었다. 특히 <a href="/wiki/%ED%8E%9C%EC%8B%A4%EB%B2%A0%EC%9D%B4%EB%8B%88%EC%95%84_%EC%A3%BC" title="펜실베이니아 주">펜실베이니아 주</a>의 <a href="/wiki/%EB%A6%AD_%EC%83%8C%ED%86%A0%EB%9F%BC" title="릭 샌토럼">릭 샌토럼</a> 후보와 <a href="/wiki/%EB%B2%84%EC%A7%80%EB%8B%88%EC%95%84_%EC%A3%BC" title="버지니아 주">버지니아 주</a>의 <a class="new" href="/w/index.php?title=%EC%A1%B0%EC%A7%80_%EC%95%A8%EB%9F%B0_(%EC%A0%95%EC%B9%98%EC%9D%B8&amp;action=edit&amp;redlink=1" title="조지 앨런 (정치인 (없는 문서)">조지 앨런</a> 후보 등 주요 중진들이 낙선하면서 공화당은 <a href="/wiki/2002%EB%85%84_%EB%AF%B8%EA%B5%AD_%EC%83%81%EC%9B%90%EC%9D%98%EC%9B%90_%EC%84%A0%EA%B1%B0" title="2002년 미국 상원의원 선거">2002년 선거</a> 이후 다수당 위치를 4년 만에 빼앗겼다. 반대로 <a href="/wiki/%EB%AF%BC%EC%A3%BC%EB%8B%B9_(%EB%AF%B8%EA%B5%AD)" title="민주당 (미국)">민주당</a>은 기존의 지역구를 잃지 않으면서 공화당이 차지하고 있던 6개의 지역을 빼앗아 오면서 공화당과 동석인 49석을 차지하였다. 기존 민주당 소속인 <a href="/wiki/%EC%A1%B0_%EB%A6%AC%EB%B2%84%EB%A8%BC" title="조 리버먼">조 리버먼</a>이 민주당 공천 탈락 반발 속에 <a href="/wiki/%EB%AC%B4%EC%86%8C%EC%86%8D" title="무소속">무소속</a>으로 출마한 후 당선되면서 무소속 의원이 2명이 되었다. 따라서 <a href="/wiki/1954%EB%85%84_%EB%AF%B8%EA%B5%AD_%EC%83%81%EC%9B%90%EC%9D%98%EC%9B%90_%EC%84%A0%EA%B1%B0" title="1954년 미국 상원의원 선거">1954년 선거</a> 이후 민주당과 공화당은 처음으로 어느 정당도 과반선을 넘기지 못하였다. 그러나 무소속 2명이 모두 민주당과 협조하기로 하면서 민주당과 공화당은 51:49 비율을 이루게 되었다. 따라서 <a href="/wiki/2007%EB%85%84" title="2007년">2007년</a> <a href="/wiki/1%EC%9B%94_3%EC%9D%BC" title="1월 3일">1월 3일</a>에 시작한 의회에서 다수당 대표로 민주당의 <a href="/wiki/%ED%95%B4%EB%A6%AC_%EB%A6%AC%EB%93%9C" title="해리 리드">해리 리드</a>가 선출되었다. 이 구도는 <a class="new" href="/w/index.php?title=2014%EB%85%84_%EB%AF%B8%EA%B5%AD_%EC%83%81%EC%9B%90%EC%9D%98%EC%9B%90_%EC%84%A0%EA%B1%B0&amp;action=edit&amp;redlink=1" title="2014년 미국 상원의원 선거 (없는 문서)">2014년 선거</a>를 끝으로 깨지게 된다.</p>
