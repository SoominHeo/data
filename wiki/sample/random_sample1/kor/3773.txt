<h2><span class="mw-headline" id=".EC.83.9D.EC.95.A0">생애</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%8D%B0%EC%9D%B4%EB%B9%84%EB%93%9C_%EB%A6%AC_(%EB%86%8D%EA%B5%AC_%EC%84%A0%EC%88%98)&amp;action=edit&amp;section=1" title="부분 편집: 생애">원본 편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>데이비드 리는 <a href="/wiki/%EB%AF%B8%EC%A3%BC%EB%A6%AC_%EC%A3%BC" title="미주리 주">미주리 주</a> <a href="/wiki/%EC%84%B8%EC%9D%B8%ED%8A%B8%EB%A3%A8%EC%9D%B4%EC%8A%A4" title="세인트루이스">세인트루이스</a>에서 태어났다. 세인트 루이스 외곽에 있는 중학교와 고등학교에 다녔다. 태생적으로 왼손잡이지만 왼손이 부러졌을 때 오른손 쓰는 것도 익혔기 때문에 양손을 쓰는 것에 능숙하다. 맥도날드 올 어메리칸 대회에 참가했고, 2001년 슬램덩크 컨테스트에서 우승을 했다. 퍼레이드 잡지가 선정한 전미 고등학교 퍼스트팀에 선정되기도 했다.</p>
<h2><span class="mw-headline" id=".EB.8C.80.ED.95.99.EA.B5.90_.EC.8B.9C.EC.A0.88">대학교 시절</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%8D%B0%EC%9D%B4%EB%B9%84%EB%93%9C_%EB%A6%AC_(%EB%86%8D%EA%B5%AC_%EC%84%A0%EC%88%98)&amp;action=edit&amp;section=2" title="부분 편집: 대학교 시절">원본 편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p><a href="/wiki/%ED%94%8C%EB%A1%9C%EB%A6%AC%EB%8B%A4_%EC%A3%BC" title="플로리다 주">플로리다 주</a> <a class="mw-disambig" href="/wiki/%EA%B2%8C%EC%9D%B8%EC%A6%88%EB%B9%8C" title="게인즈빌">게인즈빌</a>에 위치한 <a href="/wiki/%ED%94%8C%EB%A1%9C%EB%A6%AC%EB%8B%A4_%EB%8C%80%ED%95%99%EA%B5%90" title="플로리다 대학교">플로리다 대학교</a>에 운동선수 장학금을 받고 입학한다. 신입생이었던 2002 시즌에 컨퍼런스 코치들이 선정한 사우스 이스턴 컨퍼런스 신인선수에 선정된다. 2학년 시즌이었던 2002~03 시즌에 경기당 평균 두 자리 수 득점과 7개의 리바운드를 기록했다. 3학년 시즌이었던 2003~04 시즌에는 경기당 13득점과 7개의 리바운드를 기록하면서 사우스 이스턴 컨퍼런스 세컨드 팀에 선정된다.</p>

<h3><span class="mw-headline" id=".EB.89.B4.EC.9A.95_.EB.8B.89.EC.8A.A4_.282005.7E2010.29">뉴욕 닉스 (2005~2010)</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%8D%B0%EC%9D%B4%EB%B9%84%EB%93%9C_%EB%A6%AC_(%EB%86%8D%EA%B5%AC_%EC%84%A0%EC%88%98)&amp;action=edit&amp;section=4" title="부분 편집: 뉴욕 닉스 (2005~2010)">원본 편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<h4><span class="mw-headline" id="2005.7E06">2005~06</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%8D%B0%EC%9D%B4%EB%B9%84%EB%93%9C_%EB%A6%AC_(%EB%86%8D%EA%B5%AC_%EC%84%A0%EC%88%98)&amp;action=edit&amp;section=5" title="부분 편집: 2005~06">원본 편집</a><span class="mw-editsection-bracket">]</span></span></h4>
<p>2005년 드래프트에서 <a href="/wiki/%EB%89%B4%EC%9A%95_%EB%8B%89%EC%8A%A4" title="뉴욕 닉스">뉴욕 닉스</a>의 1라운드 두 번째 지명(전체 30위)을 받았는데, 뉴욕 닉스의 1라운드 첫 번째 지명은 전체 8위의 <a class="new" href="/w/index.php?title=%EC%B1%84%EB%8B%9D_%ED%94%84%EB%9D%BC%EC%9D%B4&amp;action=edit&amp;redlink=1" title="채닝 프라이 (없는 문서)">채닝 프라이</a>였다. 리는 <a href="/wiki/2005%EB%85%84" title="2005년">2005년</a> <a href="/wiki/7%EC%9B%94_1%EC%9D%BC" title="7월 1일">7월 1일</a> 뉴욕 닉스와 입단 계약을 체결한다.</p>
<p>2005년 12월부터 2006년 1월까지 13경기 연속으로 포워드로 선발 출장했다. <a href="/wiki/2006%EB%85%84" title="2006년">2006년</a> <a href="/wiki/1%EC%9B%94_2%EC%9D%BC" title="1월 2일">1월 2일</a> <a href="/wiki/%ED%94%BC%EB%8B%89%EC%8A%A4_%EC%84%A0%EC%8A%A4" title="피닉스 선스">피닉스 선스</a>와의 3번의 연장전까지 가는 접전에서 52분 동안 11개의 필드골을 시도해서 10개를 성공시키면서 23득점을 올렸고, 리바운드는 15개, 가로채기는 3개를 기록했다. 23득점은 <a href="/wiki/2007%EB%85%84" title="2007년">2007년</a> <a href="/wiki/11%EC%9B%94_16%EC%9D%BC" title="11월 16일">11월 16일</a> <a href="/wiki/%EC%83%88%ED%81%AC%EB%9D%BC%EB%A9%98%ED%86%A0_%ED%82%B9%EC%8A%A4" title="새크라멘토 킹스">새크라멘토 킹스</a>와의 경기에서 24득점을 올리기 전까지 리의 개인 통산 한 경기 최다 득점이었다. 15개의 리바운드 또한 2006년 <a href="/wiki/11%EC%9B%94_15%EC%9D%BC" title="11월 15일">11월 15일</a> <a href="/wiki/%EC%9B%8C%EC%8B%B1%ED%84%B4_%EC%9C%84%EC%A0%80%EC%A6%88" title="워싱턴 위저즈">워싱턴 위저즈</a>와의 경기에서 기록한 15개와 같은 기록이었는데, <a href="/wiki/2008%EB%85%84" title="2008년">2008년</a> <a href="/wiki/11%EC%9B%94_29%EC%9D%BC" title="11월 29일">11월 29일</a> <a href="/wiki/%EA%B3%A8%EB%93%A0%EC%8A%A4%ED%85%8C%EC%9D%B4%ED%8A%B8_%EC%9B%8C%EB%A6%AC%EC%96%B4%EC%8A%A4" title="골든스테이트 워리어스">골든스테이트 워리어스</a>와의 경기에서 21개의 리바운드를 잡으면서 개인 통산 한 경기 최다 리바운드 기록을 경신했다. 신인으로 67경기에 나와 평균 16분을 뛰면서 5.2득점(필드골 성공률 59.6%), 4.5 리바운드를 기록했다.<sup class="reference" id="cite_ref-1"><a href="#cite_note-1">[1]</a></sup></p>
<h4><span class="mw-headline" id="2006.7E07">2006~07</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%8D%B0%EC%9D%B4%EB%B9%84%EB%93%9C_%EB%A6%AC_(%EB%86%8D%EA%B5%AC_%EC%84%A0%EC%88%98)&amp;action=edit&amp;section=6" title="부분 편집: 2006~07">원본 편집</a><span class="mw-editsection-bracket">]</span></span></h4>
<p>채닝 프라이의 부상으로 <a href="/wiki/2006%EB%85%84" title="2006년">2006년</a> <a href="/wiki/11%EC%9B%94_28%EC%9D%BC" title="11월 28일">11월 28일</a> <a href="/wiki/%EC%8B%9C%EC%B9%B4%EA%B3%A0_%EB%B6%88%EC%8A%A4" title="시카고 불스">시카고 불스</a>와의 경기에서 시즌 첫 선발 출장을 한다. 2006년 <a href="/wiki/12%EC%9B%94_16%EC%9D%BC" title="12월 16일">12월 16일</a> 뉴욕 닉스 선수들과 <a href="/wiki/%EB%8D%B4%EB%B2%84_%EB%84%88%EA%B8%B0%EC%B8%A0" title="덴버 너기츠">덴버 너기츠</a> 선수들간의 말다툼으로 총 10명의 선수들이 퇴장당했는데, 리도 포함되었다. 하지만 말다툼에 직접 참가하지 않았기에 NBA 사무국의 징계는 받지 않았다. 2006년 12월 20일 샬럿 밥캐츠와의 두번의 연장전까지 가는 접전에서 0.1초가 남은 상태에서 트렌트 터커 룰을 위반하지 않고 결승 팁인 슛을 성공시키면서 팀을 승리로 이끌었다. 올스타 전 휴식기까지 리는 경기당 평균 30.9분을 뛰면서 평균 11.1 득점, 61.05% 필드골 성공률(리그 1위), 80%의 자유투 성공률, 10.8개의 리바운드(리그 8위), 1.8 어시스트를 기록했다. <a href="/wiki/2007%EB%85%84" title="2007년">2007년</a> <a href="/wiki/2%EC%9B%94_16%EC%9D%BC" title="2월 16일">2월 16일</a> 루키 챌린지에서 2년차 팀 소속으로 경기에 출장해 14개의 야투를 시도해 모두 성공시키면서 30득점을 올렸고 11개의 리바운드를 잡아내면서 경기 최우수 선수에 뽑힌다.</p>

<h4><span class="mw-headline" id="2009.7E10">2009~10</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%8D%B0%EC%9D%B4%EB%B9%84%EB%93%9C_%EB%A6%AC_(%EB%86%8D%EA%B5%AC_%EC%84%A0%EC%88%98)&amp;action=edit&amp;section=9" title="부분 편집: 2009~10">원본 편집</a><span class="mw-editsection-bracket">]</span></span></h4>
<h3><span class="mw-headline" id=".EA.B3.A8.EB.93.A0.EC.8A.A4.ED.85.8C.EC.9D.B4.ED.8A.B8_.EC.9B.8C.EB.A6.AC.EC.96.B4.EC.8A.A4_.282010.7E.ED.98.84.EC.9E.AC.29">골든스테이트 워리어스 (2010~현재)</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%8D%B0%EC%9D%B4%EB%B9%84%EB%93%9C_%EB%A6%AC_(%EB%86%8D%EA%B5%AC_%EC%84%A0%EC%88%98)&amp;action=edit&amp;section=10" title="부분 편집: 골든스테이트 워리어스 (2010~현재)">원본 편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>2010년 오프시즌에 데이비드 리는 비제한적 자유계약선수가 되었다. 7월 9일 사인 앤 트레이드 방식으로 골든스테이트 워리어스로 트레이드 되는 데, 워리어스에서는 앤서니 랜돌프, 켈레나 아주부이크, 로니 터리아프 그리고 향후 드래프트 2라운드 지명권을 닉스에 양도했다. 계약 조건은 6년, 8천만 달러였다.</p>

<p><a href="/wiki/2013%EB%85%84" title="2013년">2013년</a> <a href="/wiki/1%EC%9B%94_24%EC%9D%BC" title="1월 24일">1월 24일</a> 데이비드 리는 2013년 NBA 올스타전 예비선수로 선출된다. 리의 올스타 선정은 워리어스 팀 소속으로 <a href="/wiki/1997%EB%85%84" title="1997년">1997년</a> <a href="/wiki/%EB%9D%BC%ED%8A%B8%EB%A0%90_%EC%8A%A4%ED%94%84%EB%A6%AC%EC%9B%B0" title="라트렐 스프리웰">라트렐 스프리웰</a>이 선정된 이후로 처음이었다.</p>
