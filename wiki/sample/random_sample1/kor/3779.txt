<p><b>데이비드 제임스</b>(<span style="font-size: smaller;"><a href="/wiki/%EC%98%81%EC%96%B4" title="영어">영어</a>:</span> <span lang="en" xml:lang="en">David James, <a href="/wiki/%EB%8C%80%EC%98%81_%EC%A0%9C%EA%B5%AD_%ED%9B%88%EC%9E%A5" title="대영 제국 훈장">MBE</a><sup class="reference" id="cite_ref-.ED.9B.88.EC.9E.A5_1-0"><a href="#cite_note-.ED.9B.88.EC.9E.A5-1">[1]</a></sup></span>, <a href="/wiki/1970%EB%85%84" title="1970년">1970년</a> <a href="/wiki/8%EC%9B%94_1%EC%9D%BC" title="8월 1일">8월 1일</a> ~ )는 <a href="/wiki/%EB%8F%84%EB%AF%B8%EB%8B%88%EC%B9%B4_%EC%97%B0%EB%B0%A9" title="도미니카 연방">도미니카 연방</a>계 <a href="/wiki/%EC%9E%89%EA%B8%80%EB%9E%9C%EB%93%9C" title="잉글랜드">잉글랜드</a>인 <a href="/wiki/%EC%B6%95%EA%B5%AC_%EC%84%A0%EC%88%98" title="축구 선수">축구 선수</a>로, 선수 시절 포지션은 <a href="/wiki/%EA%B3%A8%ED%82%A4%ED%8D%BC_(%EC%B6%95%EA%B5%AC)" title="골키퍼 (축구)">골키퍼</a>다.</p>
<p>제임스는 <a href="/wiki/2009%EB%85%84" title="2009년">2009년</a> <a href="/wiki/2%EC%9B%94_14%EC%9D%BC" title="2월 14일">2월 14일</a>에 536번째 <a href="/wiki/%ED%94%84%EB%A6%AC%EB%AF%B8%EC%96%B4%EB%A6%AC%EA%B7%B8" title="프리미어리그">프리미어리그</a> 경기에 출장함으로써 기존에 <a href="/wiki/%EA%B2%8C%EB%A6%AC_%EC%8A%A4%ED%94%BC%EB%93%9C" title="게리 스피드">게리 스피드</a>가 세웠던 535경기 출장을 넘어서 프리미어리그 경기 최다 출장 기록을 새로 세웠다. 또한 제임스는 프리미어리그 역사상 가장 많은 클린 시트인 173회의 클린 시트를 기록했다.</p>
