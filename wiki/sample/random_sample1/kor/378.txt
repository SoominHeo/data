<h1 class="firstHeading" id="firstHeading" lang="ko">2006년 유엔 안전 보장 이사회 선거</h1>
<div class="mw-body-content" id="bodyContent">
<div id="siteSub">위키백과, 우리 모두의 백과사전.</div>
<div id="contentSub"></div>
<div class="mw-jump" id="jump-to-nav">
					이동:					<a href="#mw-head">둘러보기</a>, 					<a href="#p-search">검색</a>
</div>
<div class="mw-content-ltr" dir="ltr" id="mw-content-text" lang="ko"><script>function mfTempOpenSection(id){var block=document.getElementById("mf-section-"+id);block.className+=" open-block";block.previousSibling.className+=" open-block";}</script><p><b>2006년 유엔 안전 보장 이사회 선거</b>는 <a href="/wiki/%EB%AF%B8%EA%B5%AD" title="미국">미국</a> <a href="/wiki/%EB%89%B4%EC%9A%95" title="뉴욕">뉴욕</a>의 <a href="/wiki/%EC%9C%A0%EC%97%94_%EB%B3%B8%EB%B6%80" title="유엔 본부">유엔 본부</a>에서 제61차 <a href="/wiki/%EC%9C%A0%EC%97%94_%EC%B4%9D%ED%9A%8C" title="유엔 총회">유엔 총회</a> 회기 동안인 <a href="/wiki/2006%EB%85%84" title="2006년">2006년</a> <a href="/wiki/10%EC%9B%94_16%EC%9D%BC" title="10월 16일">10월 16일</a>에 시작되었다. <a href="/wiki/2007%EB%85%84" title="2007년">2007년</a> <a href="/wiki/1%EC%9B%94_1%EC%9D%BC" title="1월 1일">1월 1일</a>부터 2년의 임기가 시작되는 5개국의 <a href="/wiki/%EC%9C%A0%EC%97%94_%EC%95%88%EC%A0%84_%EB%B3%B4%EC%9E%A5_%EC%9D%B4%EC%82%AC%ED%9A%8C" title="유엔 안전 보장 이사회">유엔 안전 보장 이사회</a> 비상임 이사국을 선출하는 선거이다.</p>
<p>비상임 이사국의 임기는 2년이며 <a href="/wiki/%EC%9C%A0%EC%97%94_%EC%B4%9D%ED%9A%8C" title="유엔 총회">유엔 총회</a>에서 지리적 균형을 감안해 선출된다. <a href="/wiki/%EC%95%84%ED%94%84%EB%A6%AC%EC%B9%B4" title="아프리카">아프리카</a>와 <a class="mw-redirect" href="/wiki/%EC%A4%91%EB%82%A8%EB%AF%B8" title="중남미">중남미</a>, <a href="/wiki/%EC%95%84%EC%8B%9C%EC%95%84" title="아시아">아시아</a>에서 두 자리, <a href="/wiki/%EB%8F%99%EC%9C%A0%EB%9F%BD" title="동유럽">동유럽</a>에서 한 자리, <a href="/wiki/%EC%84%9C%EC%9C%A0%EB%9F%BD" title="서유럽">서유럽</a>과 그 외의 나라에서 두 자리를 차지한다. 남은 한 자리는 아시아와 아프리카에서 교대로 차지한다.</p>
<p>선거를 통해 선출 예정인 5개의 자리는 지역별로 다음과 같다.</p>
<ul>
<li><a href="/wiki/%EC%84%9C%EC%9C%A0%EB%9F%BD" title="서유럽">서유럽</a>과 그 외의 나라에서 2석</li>
<li><a href="/wiki/%EC%95%84%EC%8B%9C%EC%95%84" title="아시아">아시아</a>지역에서 1석</li>
<li><a href="/wiki/%EC%95%84%ED%94%84%EB%A6%AC%EC%B9%B4" title="아프리카">아프리카</a>지역에서 1석</li>
<li><a class="mw-redirect" href="/wiki/%EC%A4%91%EB%82%A8%EB%AF%B8" title="중남미">중남미</a>에서 1석</li>
</ul>

<h2><span class="mw-headline" id=".EC.84.9C.EC.9C.A0.EB.9F.BD.EA.B3.BC_.EA.B7.B8_.EC.99.B8.EC.9D.98_.EB.82.98.EB.9D.BC">서유럽과 그 외의 나라</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=2006%EB%85%84_%EC%9C%A0%EC%97%94_%EC%95%88%EC%A0%84_%EB%B3%B4%EC%9E%A5_%EC%9D%B4%EC%82%AC%ED%9A%8C_%EC%84%A0%EA%B1%B0&amp;action=edit&amp;section=2" title="부분 편집: 서유럽과 그 외의 나라">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p><a href="/wiki/%EC%84%9C%EC%9C%A0%EB%9F%BD" title="서유럽">서유럽</a>과 그 외의 나라에 배분되어 있는 2석의 자리에, 180표와 186표의 지지를 받은 <a href="/wiki/%EB%B2%A8%EA%B8%B0%EC%97%90" title="벨기에">벨기에</a>와 <a href="/wiki/%EC%9D%B4%ED%83%88%EB%A6%AC%EC%95%84" title="이탈리아">이탈리아</a>가 각각 선출되었다. <a href="/wiki/2007%EB%85%84" title="2007년">2007년</a> <a href="/wiki/1%EC%9B%94_1%EC%9D%BC" title="1월 1일">1월 1일</a>부터 <a href="/wiki/%EA%B7%B8%EB%A6%AC%EC%8A%A4" title="그리스">그리스</a>와 <a href="/wiki/%EB%8D%B4%EB%A7%88%ED%81%AC" title="덴마크">덴마크</a>를 대신할 예정이다.</p>
<h2><span class="mw-headline" id=".EC.95.84.EC.8B.9C.EC.95.84_.EC.A7.80.EC.97.AD">아시아 지역</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=2006%EB%85%84_%EC%9C%A0%EC%97%94_%EC%95%88%EC%A0%84_%EB%B3%B4%EC%9E%A5_%EC%9D%B4%EC%82%AC%ED%9A%8C_%EC%84%A0%EA%B1%B0&amp;action=edit&amp;section=3" title="부분 편집: 아시아 지역">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p><a href="/wiki/%EC%95%84%EC%8B%9C%EC%95%84" title="아시아">아시아</a> 지역 1석의 자리를 놓고 2개국이 경쟁하였다. 158표를 얻은 <a href="/wiki/%EC%9D%B8%EB%8F%84%EB%84%A4%EC%8B%9C%EC%95%84" title="인도네시아">인도네시아</a>가 28표를 얻은 <a href="/wiki/%EB%84%A4%ED%8C%94" title="네팔">네팔</a>을 누르고 비상임 이사국 자리를 차지했다. <a href="/wiki/2007%EB%85%84" title="2007년">2007년</a> <a href="/wiki/1%EC%9B%94_1%EC%9D%BC" title="1월 1일">1월 1일</a>부터 현재의 <a href="/wiki/%EC%9D%BC%EB%B3%B8" title="일본">일본</a>을 대신한다.<sup class="reference" id="cite_ref-grabs_1-0"><a href="#cite_note-grabs-1">[1]</a></sup></p>
<h2><span class="mw-headline" id=".EC.95.84.ED.94.84.EB.A6.AC.EC.B9.B4_.EC.A7.80.EC.97.AD">아프리카 지역</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=2006%EB%85%84_%EC%9C%A0%EC%97%94_%EC%95%88%EC%A0%84_%EB%B3%B4%EC%9E%A5_%EC%9D%B4%EC%82%AC%ED%9A%8C_%EC%84%A0%EA%B1%B0&amp;action=edit&amp;section=4" title="부분 편집: 아프리카 지역">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p><a href="/wiki/%EC%95%84%ED%94%84%EB%A6%AC%EC%B9%B4" title="아프리카">아프리카</a> 지역에 배정된 1석에 <a class="mw-redirect" href="/wiki/%EB%82%A8%EC%95%84%ED%94%84%EB%A6%AC%EC%B9%B4%EA%B3%B5%ED%99%94%EA%B5%AD" title="남아프리카공화국">남아프리카공화국</a>이 단독 출마, 186표를 얻어 선출되었다. 현재의 <a href="/wiki/%ED%83%84%EC%9E%90%EB%8B%88%EC%95%84" title="탄자니아">탄자니아</a>를 교체할 예정이다.<sup class="reference" id="cite_ref-grabs_1-1"><a href="#cite_note-grabs-1">[1]</a></sup></p>
<h2><span class="mw-headline" id=".EC.A4.91.EB.82.A8.EB.AF.B8_.EC.A7.80.EC.97.AD">중남미 지역</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=2006%EB%85%84_%EC%9C%A0%EC%97%94_%EC%95%88%EC%A0%84_%EB%B3%B4%EC%9E%A5_%EC%9D%B4%EC%82%AC%ED%9A%8C_%EC%84%A0%EA%B1%B0&amp;action=edit&amp;section=5" title="부분 편집: 중남미 지역">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>중남미 지역 몫으로 배정된 1석의 자리를 차지하기 위해 <a href="/wiki/%EA%B3%BC%ED%85%8C%EB%A7%90%EB%9D%BC" title="과테말라">과테말라</a>와 <a href="/wiki/%EB%B2%A0%EB%84%A4%EC%88%98%EC%97%98%EB%9D%BC" title="베네수엘라">베네수엘라</a>가 경쟁하였다. 47차례의 선거를 치르는 표 대결 끝에 결국, 양측은 막후 협상을 통해 이사국 후보에서 동반 퇴진하는 대신 제3의 후보인 <a href="/wiki/%ED%8C%8C%EB%82%98%EB%A7%88" title="파나마">파나마</a>를 공동 지지하기로 합의하였고 <a href="/wiki/%ED%8C%8C%EB%82%98%EB%A7%88" title="파나마">파나마</a>가 선출되었다.</p>
<p>파나마는 <a href="/wiki/%EC%95%84%EB%A5%B4%ED%97%A8%ED%8B%B0%EB%82%98" title="아르헨티나">아르헨티나</a>의 임기가 만료되는 2007년 1월 1일부터 중남미 지역 몫의 비상임 이사국이 된다.</p>
