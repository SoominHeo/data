<p><b>데이비드 코레시</b>(<span style="font-size: smaller;"><a href="/wiki/%EC%98%81%EC%96%B4" title="영어">영어</a>:</span> <span lang="en" xml:lang="en">David Koresh</span>, <a href="/wiki/1959%EB%85%84" title="1959년">1959년</a> <a href="/wiki/8%EC%9B%94_17%EC%9D%BC" title="8월 17일">8월 17일</a> – <a href="/wiki/1993%EB%85%84" title="1993년">1993년</a> <a href="/wiki/4%EC%9B%94_19%EC%9D%BC" title="4월 19일">4월 19일</a>)는 <a href="/wiki/%EB%AF%B8%EA%B5%AD" title="미국">미국</a>의 종교인이자, 범죄자로, <a href="/wiki/%EB%8B%A4%EC%9C%97%EA%B5%90" title="다윗교">다윗교</a>라는 광신적 종교집단의 교주였다. 본명은 <b>버논 웨인 하웰</b> (Vernon Wayne Howell)이었으나 <a href="/wiki/1990%EB%85%84" title="1990년">1990년</a> <a href="/wiki/5%EC%9B%94_15%EC%9D%BC" title="5월 15일">5월 15일</a> <a href="/wiki/%EC%9D%B4%EC%8A%A4%EB%9D%BC%EC%97%98" title="이스라엘">이스라엘</a>의 왕 <a href="/wiki/%EB%8B%A4%EC%9C%97" title="다윗">다윗</a>과 <a href="/wiki/%EC%9C%A0%EB%8C%80%EC%9D%B8" title="유대인">유대인</a>을 <a href="/wiki/%EB%B0%94%EB%B9%8C%EB%A1%9C%EB%8B%88%EC%95%84" title="바빌로니아">바빌로니아</a>로부터 해방시킨 <a class="mw-disambig" href="/wiki/%ED%8E%98%EB%A5%B4%EC%8B%9C%EC%95%84" title="페르시아">페르시아</a>의 왕 <a href="/wiki/%ED%82%A4%EB%A3%A8%EC%8A%A4_2%EC%84%B8" title="키루스 2세">키루스 2세</a>의 이름을 조합하여 데이비드 코레시로 개명했다. <a href="/wiki/%EC%A2%85%EB%A7%90%EB%A1%A0" title="종말론">종말론</a>에 심취하여 <a class="mw-redirect" href="/wiki/%EC%9E%AC%EB%A6%BC_%EC%98%88%EC%88%98" title="재림 예수">재림 예수</a>를 자칭하며 <a class="mw-redirect" href="/wiki/%EC%95%84%EB%A7%88%EA%B2%8C%EB%8F%88" title="아마게돈">아마게돈</a>에 대비한다며 <a href="/wiki/%ED%85%8D%EC%82%AC%EC%8A%A4_%EC%A3%BC" title="텍사스 주">텍사스 주</a>의 <a href="/wiki/%EC%9B%A8%EC%9D%B4%EC%BD%94" title="웨이코">웨이코</a>의 다윗파 본부에 수많은 총기와 탄약을 비축하였다. <a href="/wiki/1993%EB%85%84" title="1993년">1993년</a> <a href="/wiki/2%EC%9B%94_28%EC%9D%BC" title="2월 28일">2월 28일</a>, <a href="/wiki/%EC%A3%BC%EB%A5%98%C2%B7%EB%8B%B4%EB%B0%B0%C2%B7%ED%99%94%EA%B8%B0_%EB%B0%8F_%ED%8F%AD%EB%B0%9C%EB%AC%BC_%EB%8B%A8%EC%86%8D%EA%B5%AD" title="주류·담배·화기 및 폭발물 단속국">ATF</a>의 압수 수색이 총격전으로 이어지면서 51일 동안 대치하다가 건물에 발생한 화재로 데이비드 코레시를 비롯한 성인 55명과 28명의 아이가 사망하는 참사를 일으켰다. (<a class="mw-redirect" href="/wiki/%EC%9B%A8%EC%9D%B4%EC%BD%94_%EC%82%AC%ED%83%9C" title="웨이코 사태">웨이코 사태</a>)</p>
<p>이 사건으로 미국 <a class="new" href="/w/index.php?title=%EA%B8%B0%EB%8F%85%EA%B5%90_%EC%9B%90%EB%A6%AC%EC%A3%BC%EC%9D%98%EC%9E%90&amp;action=edit&amp;redlink=1" title="기독교 원리주의자 (없는 문서)">기독교 원리주의자</a> 및 백인 극우 집단이 연방 정부에 대한 깊은 반감을 가지게 되었으며, 대표적으로 <a href="/wiki/%ED%8B%B0%EB%A8%B8%EC%8B%9C_%EB%A7%A5%EB%B2%A0%EC%9D%B4" title="티머시 맥베이">티머시 맥베이</a>는 이 사건에 대한 개인적 보복의 일환으로 웨이코 사태 2주년인 <a href="/wiki/1995%EB%85%84" title="1995년">1995년</a> <a href="/wiki/4%EC%9B%94_19%EC%9D%BC" title="4월 19일">4월 19일</a>에 <a class="mw-redirect" href="/wiki/%EC%98%A4%ED%81%B4%EB%9D%BC%ED%98%B8%EB%A7%88_%ED%8F%AD%ED%83%84_%ED%85%8C%EB%9F%AC" title="오클라호마 폭탄 테러">오클라호마 폭탄 테러</a>를 일으켰다.</p>
