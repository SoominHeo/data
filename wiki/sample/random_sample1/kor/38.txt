<h2><span class="mw-headline" id=".EA.B0.81.EC.A3.BC">각주</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=10%EC%9B%94_9%EC%9D%BC&amp;action=edit&amp;section=7" title="부분 편집: 각주">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<ol class="references">
<li id="cite_note-1"><span class="mw-cite-backlink"><a href="#cite_ref-1">↑</a></span> <span class="reference-text">사실 <a href="/wiki/%ED%9B%88%EB%AF%BC%EC%A0%95%EC%9D%8C" title="훈민정음">훈민정음</a> 반포일의 정확한 날짜는 모른다. <a class="external text" href="http://sillok.history.go.kr/inspection/insp_king.jsp?id=kda_12809029&amp;tabid=k" rel="nofollow">조선왕조실록</a>에 따르면, <a href="/wiki/1446%EB%85%84" title="1446년">1446년</a> <a href="/wiki/%EC%9D%8C%EB%A0%A5_9%EC%9B%94_29%EC%9D%BC" title="음력 9월 29일">음력 9월 29일</a>에 <a href="/wiki/%ED%9B%88%EB%AF%BC%EC%A0%95%EC%9D%8C" title="훈민정음">훈민정음</a>이 이루어졌다고 되어 있는데, <a href="/wiki/1940%EB%85%84" title="1940년">1940년</a>에 발견된 《<a href="/wiki/%ED%9B%88%EB%AF%BC%EC%A0%95%EC%9D%8C" title="훈민정음">훈민정음</a>》 해례본 말문(末文)에는 ‘정통 11년 9월 상한-세종 28년 9월’ (正統十一年九月上澣)에 책으로 펴냈다고 되어 있다. 그래서 <a href="/wiki/1446%EB%85%84" title="1446년">1446년</a> 9월 상한(上澣)의 마지막 날인 <a href="/wiki/%EC%9D%8C%EB%A0%A5_9%EC%9B%94_10%EC%9D%BC" title="음력 9월 10일">음력 9월 10일</a>을 <a href="/wiki/%ED%9B%88%EB%AF%BC%EC%A0%95%EC%9D%8C" title="훈민정음">훈민정음</a> 반포일로 정했는데, 이를 <a href="/wiki/%EA%B7%B8%EB%A0%88%EA%B3%A0%EB%A6%AC%EB%A0%A5" title="그레고리력">그레고리력</a>으로 환산하면 10월 9일이 되므로, 10월 9일을 <a href="/wiki/%ED%9B%88%EB%AF%BC%EC%A0%95%EC%9D%8C" title="훈민정음">훈민정음</a> 반포일로 정했다. 그러나 지금 우리가 사용하는 <a href="/wiki/%EA%B7%B8%EB%A0%88%EA%B3%A0%EB%A6%AC%EB%A0%A5" title="그레고리력">그레고리력</a>이 아닌, <a href="/wiki/1446%EB%85%84" title="1446년">1446년</a> 당시 서양이 사용했던 <a href="/wiki/%EC%9C%A8%EB%A6%AC%EC%9A%B0%EC%8A%A4%EB%A0%A5" title="율리우스력">율리우스력</a>으로 환산하면, <a href="/wiki/1446%EB%85%84" title="1446년">1446년</a> <a href="/wiki/%EC%9D%8C%EB%A0%A5_9%EC%9B%94_10%EC%9D%BC" title="음력 9월 10일">음력 9월 10일</a>은 <a href="/wiki/9%EC%9B%94_30%EC%9D%BC" title="9월 30일">9월 30일</a>이 된다. (더 자세한 것은 <a class="external text" href="http://astro.kasi.re.kr/Life/ConvertSolarLunarForm.aspx?MenuID=115" rel="nofollow">한국천문연구원 천문우주지식정보 사이트</a> 혹은 <a class="external text" href="http://astro.kasi.re.kr/Community/QuestionViewForm.aspx?MenuID=1593&amp;QuestionID=22978" rel="nofollow">천문참여관 질문상자</a> 를 참조하라.)</span></li>
<li id="cite_note-2"><span class="mw-cite-backlink"><a href="#cite_ref-2">↑</a></span> <span class="reference-text">'민음 한국사' 15세기-조선의 때 이른 절정 166쪽~168쪽</span></li>
