<h2><span class="mw-headline" id="2007.EB.85.84_12.EC.9B.94_31.EC.9D.BC"><a href="/wiki/2007%EB%85%84" title="2007년">2007년</a> <a href="/wiki/12%EC%9B%94_31%EC%9D%BC" title="12월 31일">12월 31일</a></span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=2007%EB%85%84_12%EC%9B%94&amp;action=edit&amp;section=1" title="부분 편집: 2007년 12월 31일">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<ul>
<li><a href="/wiki/%EB%8C%80%ED%95%9C%EB%AF%BC%EA%B5%AD" title="대한민국">대한민국</a> <a href="/wiki/%EB%85%B8%EB%AC%B4%ED%98%84" title="노무현">노무현</a> 대통령은 <a href="/wiki/%EA%B9%80%EC%9A%B0%EC%A4%91" title="김우중">김우중</a>·<a class="mw-redirect" href="/wiki/%EB%B0%95%EC%A7%80%EC%9B%90_(%EC%A0%95%EC%B9%98%EC%9D%B8)" title="박지원 (정치인)">박지원</a>·<a href="/wiki/%ED%95%9C%ED%99%94%EA%B0%91" title="한화갑">한화갑</a> 등 총 75명에 대해 특별사면을 단행하였다.</li>
<li><a href="/wiki/%ED%95%9C%EA%B5%AD%EC%9D%80%ED%96%89" title="한국은행">한국은행</a>은 <a href="/wiki/2009%EB%85%84" title="2009년">2009년</a> 발행 예정인 5만원권·10만원권의 주조색을 각각 <a href="/wiki/%EB%85%B8%EB%9E%91" title="노랑">노랑</a>과 <a href="/wiki/%ED%9A%8C%EC%83%89" title="회색">회색</a>으로 결정하였다.</li>
</ul>
<h2><span class="mw-headline" id="2007.EB.85.84_12.EC.9B.94_30.EC.9D.BC"><a href="/wiki/2007%EB%85%84" title="2007년">2007년</a> <a href="/wiki/12%EC%9B%94_30%EC%9D%BC" title="12월 30일">12월 30일</a></span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=2007%EB%85%84_12%EC%9B%94&amp;action=edit&amp;section=2" title="부분 편집: 2007년 12월 30일">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<ul>
<li><a href="/wiki/%EB%8C%80%ED%95%9C%EB%AF%BC%EA%B5%AD" title="대한민국">대한민국</a>이 <a href="/wiki/1997%EB%85%84" title="1997년">1997년</a> <a href="/wiki/12%EC%9B%94_30%EC%9D%BC" title="12월 30일">12월 30일</a>을 사형 집행 이후, 10년 동안 단 한 명의 사형 집행을 하지 않아 <a class="mw-redirect" href="/wiki/%EC%95%B0%EB%84%A4%EC%8A%A4%ED%8B%B0" title="앰네스티">앰네스티</a>가 분류하는 “<a href="/wiki/%EC%84%B8%EA%B3%84_%EA%B0%81_%EB%82%98%EB%9D%BC%EC%9D%98_%EC%82%AC%ED%98%95%EC%A0%9C%EB%8F%84_%ED%98%84%ED%99%A9" title="세계 각 나라의 사형제도 현황">실질적 사형폐지국</a>”이 되었다. 한편, <a href="/wiki/%EB%85%B8%EB%AC%B4%ED%98%84" title="노무현">노무현</a> 대통령이 31일 단행할 임기 중 마지막 특별사면에서 일부 사형수를 무기징역으로 감형하는 조치가 단행할 예정이라고 <a href="/wiki/%EC%97%B0%ED%95%A9%EB%89%B4%EC%8A%A4" title="연합뉴스">연합뉴스</a>가 보도하였다.</li>
</ul>
<h2><span class="mw-headline" id="2007.EB.85.84_12.EC.9B.94_29.EC.9D.BC"><a href="/wiki/2007%EB%85%84" title="2007년">2007년</a> <a href="/wiki/12%EC%9B%94_29%EC%9D%BC" title="12월 29일">12월 29일</a></span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=2007%EB%85%84_12%EC%9B%94&amp;action=edit&amp;section=3" title="부분 편집: 2007년 12월 29일">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<ul>
<li>암살된 <a href="/wiki/%EB%B2%A0%EB%82%98%EC%A7%80%EB%A5%B4_%EB%B6%80%ED%86%A0" title="베나지르 부토">베나지르 부토</a> <a href="/wiki/%ED%8C%8C%ED%82%A4%EC%8A%A4%ED%83%84" title="파키스탄">파키스탄</a> 전 총리는 유언장에서 그녀의 19세 아들 빌라왈을 자신의 정치적 후계자 겸 <a class="new" href="/w/index.php?title=%ED%8C%8C%ED%82%A4%EC%8A%A4%ED%83%84%EC%9D%B8%EB%AF%BC%EB%8B%B9&amp;action=edit&amp;redlink=1" title="파키스탄인민당 (없는 문서)">파키스탄인민당</a>의 새 지도자로 지명했다고 뉴스위크가 보도하였다.</li>
<li><a href="/wiki/%EC%BC%80%EB%83%90" title="케냐">케냐</a>의 대통령 선거 중간 개표 결과 여야의 두 후보가 박빙의 접전을 펼치고 있는 가운데 최종 결과 발표가 지연되자 야당 지지자들을 중심으로 전국에서 시위가 발생하였다.</li>
</ul>
<h2><span class="mw-headline" id="2007.EB.85.84_12.EC.9B.94_28.EC.9D.BC"><a href="/wiki/2007%EB%85%84" title="2007년">2007년</a> <a href="/wiki/12%EC%9B%94_28%EC%9D%BC" title="12월 28일">12월 28일</a></span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=2007%EB%85%84_12%EC%9B%94&amp;action=edit&amp;section=4" title="부분 편집: 2007년 12월 28일">편집</a><span class="mw-editsection-bracket">]</span></span></h2>

<h2><span class="mw-headline" id="2007.EB.85.84_12.EC.9B.94_16.EC.9D.BC"><a href="/wiki/2007%EB%85%84" title="2007년">2007년</a> <a href="/wiki/12%EC%9B%94_16%EC%9D%BC" title="12월 16일">12월 16일</a></span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=2007%EB%85%84_12%EC%9B%94&amp;action=edit&amp;section=16" title="부분 편집: 2007년 12월 16일">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<ul>
<li><a href="/wiki/BBK_%EC%A3%BC%EA%B0%80_%EC%A1%B0%EC%9E%91_%EC%82%AC%EA%B1%B4" title="BBK 주가 조작 사건">BBK 주가 조작 사건</a> :
<ul>
<li><a href="/wiki/%EB%8C%80%ED%86%B5%ED%95%A9%EB%AF%BC%EC%A3%BC%EC%8B%A0%EB%8B%B9" title="대통합민주신당">대통합민주신당</a>은 “<a href="/wiki/%EC%9D%B4%EB%AA%85%EB%B0%95" title="이명박">이명박</a> 후보가 <a href="/wiki/2000%EB%85%84" title="2000년">2000년</a> <a href="/wiki/10%EC%9B%94_17%EC%9D%BC" title="10월 17일">10월 17일</a> 서울 <a href="/wiki/%EA%B4%91%EC%9A%B4%EB%8C%80%ED%95%99%EA%B5%90" title="광운대학교">광운대학교</a> 경영대학원에서 주최한 특강에서 본인이 BBK를 설립했다고 밝히는 동영상을 확보해 관련 내용을 확인했다”고 밝혔다.</li>
<li>서울마포경찰서는 지난 2000년 이명박 후보의 광운대 특강을 촬영한 모 미디어 회사 대표 여모 씨 등 3명을 붙잡아 조사중이라고 밝혔다.</li>

</li>

<li><a href="/wiki/%EC%A1%B0%EC%A7%80_W._%EB%B6%80%EC%8B%9C" title="조지 W. 부시">조지 W. 부시</a> <a class="mw-redirect" href="/wiki/%EB%AF%B8%EA%B5%AD_%EB%8C%80%ED%86%B5%EB%A0%B9" title="미국 대통령">미국 대통령</a>이 기자회견을 갖고 2차 <a class="mw-redirect" href="/wiki/%EB%AF%B8%EA%B5%AD_%EC%84%9C%EB%B8%8C%ED%94%84%EB%9D%BC%EC%9E%84_%EB%AA%A8%EA%B8%B0%EC%A7%80%EB%A1%A0_%EC%82%AC%ED%83%9C" title="미국 서브프라임 모기지론 사태">서브프라임 모기지론 사태</a>의 대책을 발표하였다.</li>
<li><a href="/wiki/%EB%8C%80%ED%95%9C%EB%AF%BC%EA%B5%AD_%EC%B9%9C%EC%9D%BC%EB%B0%98%EB%AF%BC%EC%A1%B1%ED%96%89%EC%9C%84%EC%A7%84%EC%83%81%EA%B7%9C%EB%AA%85%EC%9C%84%EC%9B%90%ED%9A%8C" title="대한민국 친일반민족행위진상규명위원회">대한민국 친일반민족행위진상규명위원회</a>는 <a href="/wiki/%EC%9D%BC%EC%A0%9C_%EA%B0%95%EC%A0%90%EA%B8%B0" title="일제 강점기">일제 강점기</a> 중기의 <b><a href="/wiki/%EC%B9%9C%EC%9D%BC%EB%B0%98%EB%AF%BC%EC%A1%B1%ED%96%89%EC%9C%84_195%EC%9D%B8_%EB%AA%85%EB%8B%A8" title="친일반민족행위 195인 명단">친일반민족행위 195인 명단</a></b>을 확정해 <a class="mw-redirect" href="/wiki/%EB%8C%80%ED%95%9C%EB%AF%BC%EA%B5%AD_%EB%8C%80%ED%86%B5%EB%A0%B9" title="대한민국 대통령">대통령</a>과 <a href="/wiki/%EB%8C%80%ED%95%9C%EB%AF%BC%EA%B5%AD_%EA%B5%AD%ED%9A%8C" title="대한민국 국회">국회</a>에 보고하였다.</li>
</ul>
<h2><span class="mw-headline" id="2007.EB.85.84_12.EC.9B.94_5.EC.9D.BC"><a href="/wiki/2007%EB%85%84" title="2007년">2007년</a> <a href="/wiki/12%EC%9B%94_5%EC%9D%BC" title="12월 5일">12월 5일</a></span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=2007%EB%85%84_12%EC%9B%94&amp;action=edit&amp;section=24" title="부분 편집: 2007년 12월 5일">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<ul>
<li><a class="mw-redirect" href="/wiki/2007%EB%85%84_%EB%8C%80%ED%95%9C%EB%AF%BC%EA%B5%AD_%EB%8C%80%ED%86%B5%EB%A0%B9_%EC%84%A0%EA%B1%B0" title="2007년 대한민국 대통령 선거">2007년 대한민국 대통령 선거</a> :
<ul>
<li>검찰은 <a href="/wiki/%EC%9D%B4%EB%AA%85%EB%B0%95" title="이명박">이명박</a> 대선후보의 <a href="/wiki/BBK_%EC%A3%BC%EA%B0%80_%EC%A1%B0%EC%9E%91_%EC%82%AC%EA%B1%B4" title="BBK 주가 조작 사건">BBK 주가 조작 사건</a>에 대해 무혐의로 발표하였다.</li>
</ul>
</li>
<li><a href="/wiki/%EB%A1%9C%EB%B2%84%ED%8A%B8_%EA%B2%8C%EC%9D%B4%EC%B8%A0" title="로버트 게이츠">로버트 게이츠</a> <a href="/wiki/%EB%AF%B8%EA%B5%AD" title="미국">미국</a> 국방장관이 <a href="/wiki/%EC%9D%B4%EB%9D%BC%ED%81%AC" title="이라크">이라크</a>를 방문한 가운데, 잇따른 <a class="new" href="/w/index.php?title=%ED%8F%AD%ED%83%84_%ED%85%8C%EB%9F%AC&amp;action=edit&amp;redlink=1" title="폭탄 테러 (없는 문서)">폭탄 테러</a>로 모두 23명이 사망하고 30명이 부상당했다.</li>
<li><a href="/wiki/%EA%B0%95%ED%99%94%EB%8F%84" title="강화도">강화도</a>에서 <b><a class="mw-redirect" href="/wiki/%EA%B0%95%ED%99%94_%EC%B4%9D%EA%B8%B0_%ED%83%88%EC%B7%A8_%EC%82%AC%EA%B1%B4" title="강화 총기 탈취 사건">총기 탈취 사건</a></b>이 일어나 해병대 군인 1명이 사망하였다.</li>
</ul>
<h2><span class="mw-headline" id="2007.EB.85.84_12.EC.9B.94_3.EC.9D.BC"><a href="/wiki/2007%EB%85%84" title="2007년">2007년</a> <a href="/wiki/12%EC%9B%94_3%EC%9D%BC" title="12월 3일">12월 3일</a></span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=2007%EB%85%84_12%EC%9B%94&amp;action=edit&amp;section=25" title="부분 편집: 2007년 12월 3일">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<ul>
<li><a class="mw-redirect" href="/wiki/%EB%8C%80%ED%95%9C%EB%AF%BC%EA%B5%AD_%EC%95%BC%EA%B5%AC_%EB%8C%80%ED%91%9C%ED%8C%80" title="대한민국 야구 대표팀">대한민국 야구 올림픽 대표팀</a>이 필리핀과의 경기에서 13-1로 7회 콜드게임승을 거두었다.</li>
<li><a class="mw-redirect" href="/wiki/2007%EB%85%84_%EB%8C%80%ED%95%9C%EB%AF%BC%EA%B5%AD_%EB%8C%80%ED%86%B5%EB%A0%B9_%EC%84%A0%EA%B1%B0" title="2007년 대한민국 대통령 선거">2007년 대한민국 대통령 선거</a> :
<ul>
<li><a href="/wiki/%EC%A0%95%EB%AA%BD%EC%A4%80" title="정몽준">정몽준</a> 의원이 <a href="/wiki/%EC%9D%B4%EB%AA%85%EB%B0%95" title="이명박">이명박</a> 후보 지지와 <a class="mw-redirect" href="/wiki/%ED%95%9C%EB%82%98%EB%9D%BC%EB%8B%B9" title="한나라당">한나라당</a> 입당을 선언하였다.</li>
<li><a href="/wiki/%EA%B5%AD%EB%AF%BC%EC%A4%91%EC%8B%AC%EB%8B%B9" title="국민중심당">국민중심당</a>의 <a href="/wiki/%EC%8B%AC%EB%8C%80%ED%8F%89" title="심대평">심대평</a> 대선후보가 무소속의 <a href="/wiki/%EC%9D%B4%ED%9A%8C%EC%B0%BD" title="이회창">이회창</a> 후보 지지를 공식적으로 선언하였다.</li>
</ul>
</li>
<li><a href="/wiki/%EC%98%A4%EC%8A%A4%ED%8A%B8%EB%A0%88%EC%9D%BC%EB%A6%AC%EC%95%84" title="오스트레일리아">오스트레일리아</a>의 <a class="mw-redirect" href="/wiki/2007%EB%85%84_%EC%98%A4%EC%8A%A4%ED%8A%B8%EB%A0%88%EC%9D%BC%EB%A6%AC%EC%95%84_%EA%B5%AD%ED%9A%8C%EC%9D%98%EC%9B%90_%EC%B4%9D%EC%84%A0%EA%B1%B0" title="2007년 오스트레일리아 국회의원 총선거">총선</a>에서 승리한 <a href="/wiki/%EC%BC%80%EB%B9%88_%EB%9F%AC%EB%93%9C" title="케빈 러드">케빈 러드</a>가 제26대 연방<a href="/wiki/%EC%B4%9D%EB%A6%AC" title="총리">총리</a>직에 취임하였다. 그는 첫 공식집무로 <a href="/wiki/%EA%B5%90%ED%86%A0_%EC%9D%98%EC%A0%95%EC%84%9C" title="교토 의정서">교토 의정서</a>에 서명하였다.</li>
</ul>
<h2><span class="mw-headline" id="2007.EB.85.84_12.EC.9B.94_2.EC.9D.BC"><a href="/wiki/2007%EB%85%84" title="2007년">2007년</a> <a href="/wiki/12%EC%9B%94_2%EC%9D%BC" title="12월 2일">12월 2일</a></span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=2007%EB%85%84_12%EC%9B%94&amp;action=edit&amp;section=26" title="부분 편집: 2007년 12월 2일">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<ul>
<li><a href="/wiki/%EC%9D%BC%EB%B3%B8" title="일본">일본</a> 야구 대표팀이 <a class="mw-redirect" href="/wiki/%EB%8C%80%ED%95%9C%EB%AF%BC%EA%B5%AD_%EC%95%BC%EA%B5%AC_%EB%8C%80%ED%91%9C%ED%8C%80" title="대한민국 야구 대표팀">대한민국</a>과의 경기에서 4-3로 승리하였다.</li>
<li><a href="/wiki/%EB%B2%A0%EB%84%A4%EC%88%98%EC%97%98%EB%9D%BC" title="베네수엘라">베네수엘라</a>에선 언론통제, <a href="/wiki/%EC%A2%85%EC%8B%A0_%EB%8C%80%ED%86%B5%EB%A0%B9" title="종신 대통령">종신 대통령</a> 등을 포함하고 있는 <a href="/wiki/%EC%9A%B0%EA%B3%A0_%EC%B0%A8%EB%B2%A0%EC%8A%A4" title="우고 차베스">우고 차베스</a> <a href="/wiki/%EB%B2%A0%EB%84%A4%EC%88%98%EC%97%98%EB%9D%BC" title="베네수엘라">베네수엘라</a> 대통령의 개헌안이 <a class="mw-redirect" href="/wiki/%EA%B5%AD%EB%AF%BC_%ED%88%AC%ED%91%9C" title="국민 투표">국민 투표</a>에 의해 부결되었다.</li>
</ul>
<h2><span class="mw-headline" id="2007.EB.85.84_12.EC.9B.94_1.EC.9D.BC"><a href="/wiki/2007%EB%85%84" title="2007년">2007년</a> <a href="/wiki/12%EC%9B%94_1%EC%9D%BC" title="12월 1일">12월 1일</a></span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=2007%EB%85%84_12%EC%9B%94&amp;action=edit&amp;section=27" title="부분 편집: 2007년 12월 1일">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
