<p><b>데코폰</b>(<span style="font-size: smaller;"><a href="/wiki/%EC%9D%BC%EB%B3%B8%EC%96%B4" title="일본어">일본어</a>:</span> <span lang="ja" xml:lang="ja">デコポン</span>)은 1972년에 일본에서 개발된 교잡종 감귤이다. 흔히 <b>시라누이</b>(<span lang="ja" xml:lang="ja">不知火</span>)라는 품종명으로 불리며, 1990년대에 이를 수입해 간 <a href="/wiki/%ED%95%9C%EA%B5%AD" title="한국">한국</a>에서는 <b>한라봉</b>(漢拏峰)이라고 부른다. <a href="/wiki/%EC%9D%BC%EB%B3%B8" title="일본">일본</a>과 <a href="/wiki/%EC%A0%9C%EC%A3%BC%ED%8A%B9%EB%B3%84%EC%9E%90%EC%B9%98%EB%8F%84" title="제주특별자치도">제주특별자치도</a> 등지에서 주로 재배된다.</p>

<h2><span class="mw-headline" id=".EA.B0.9C.EC.9A.94">개요</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%8D%B0%EC%BD%94%ED%8F%B0&amp;action=edit&amp;section=1" title="부분 편집: 개요">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p><a href="/wiki/1972%EB%85%84" title="1972년">1972년</a>에 <a href="/wiki/%EC%9D%BC%EB%B3%B8_%EB%86%8D%EB%A6%BC%EC%88%98%EC%82%B0%EC%84%B1" title="일본 농림수산성">일본 농림수산성</a> 과수시험장에서 <a class="new" href="/w/index.php?title=%ED%82%A4%EC%9A%94%EB%AF%B8&amp;action=edit&amp;redlink=1" title="키요미 (없는 문서)">키요미</a>과 <a class="new" href="/w/index.php?title=%ED%8F%B0%EC%B9%B8&amp;action=edit&amp;redlink=1" title="폰칸 (없는 문서)">폰칸</a>을 교배하여 육성한 <a class="mw-redirect" href="/wiki/%EB%A7%8C%EB%8B%A4%EB%A6%B0_%EC%98%A4%EB%A0%8C%EC%A7%80" title="만다린 오렌지">만다린</a>계의 <a class="mw-redirect" href="/wiki/%EA%B5%90%EC%9E%A1%EC%A2%85" title="교잡종">교잡종</a> <a class="mw-redirect" href="/wiki/%EA%B0%90%EA%B7%A4" title="감귤">감귤</a>이다. 당시 <a class="new" href="/w/index.php?title=%EC%8B%A4%EC%83%9D%EB%B2%88%ED%98%B8&amp;action=edit&amp;redlink=1" title="실생번호 (없는 문서)">실생번호</a>는 키요미*폰칸 32호였다. 당시에는 과실의 형태가 좋지 않고 특이하며, 과면이 거칠고 과피색이 희미하다 하여 주목받지 못했다. 그러나 점차 품질이 개선되어 지금은 당도가 높고 과육 상태와 향기가 좋으며, 상품 또한 다양해졌다.</p>
<h2><span class="mw-headline" id=".EB.8C.80.ED.95.9C.EB.AF.BC.EA.B5.AD">대한민국</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%8D%B0%EC%BD%94%ED%8F%B0&amp;action=edit&amp;section=2" title="부분 편집: 대한민국">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>대한민국에서는 <a href="/wiki/1990%EB%85%84%EB%8C%80" title="1990년대">1990년대</a>에 일본에서 수입하여 재배하기 시작했다. 이 <a href="/wiki/%EA%B3%BC%EC%9D%BC" title="과일">과일</a>이 제주특별자치도에서 주로 재배되고 과일 꼭지의 튀어나온 부분이 <a href="/wiki/%EC%82%B0" title="산">산</a>을 연상시킨다고 하여, <a href="/wiki/%ED%95%9C%EB%9D%BC%EC%82%B0" title="한라산">한라산</a>의 <a href="/wiki/%EC%9D%B4%EB%A6%84" title="이름">이름</a>을 따 한라봉이라는 이름이 붙여졌다. 2000년대 후반에 들어서는 제주도 이외에 전라남도 등지에서도 재배되고 있다.<sup class="reference" id="cite_ref-1"><a href="#cite_note-1">[1]</a></sup></p>
