<h2><span class="mw-headline" id=".EA.B2.BD.EB.A0.A5">경력</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%8D%B0%ED%8A%B8%EB%A7%88%EC%96%B4_%ED%81%AC%EB%9D%BC%EB%A8%B8&amp;action=edit&amp;section=1" title="부분 편집: 경력">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>상세한 것에 대한 관심으로 "축구 교수", 161cm의 단신으로 "<a class="mw-redirect" href="/wiki/%EB%82%98%ED%8F%B4%EB%A0%88%EC%98%B9" title="나폴레옹">나폴레옹</a>"이라 불린 크라머는 빅토리아 도르트문트와 게르마니아 비스바덴에서 선수로 경력을 보냈다. 그의 감독 경력은 토이토니아 립슈타트, VfL 게세케, <a href="/wiki/SC_%ED%8C%8C%EB%8D%94%EB%B3%B4%EB%A5%B8_07" title="SC 파더보른 07">SC 파더보른 07</a>, TuS 에빙-린덴호르스트에서 시작하였다. 1948-49 시즌을 기점으로, 크라머는 <a href="/wiki/%EB%8F%85%EC%9D%BC_%EC%B6%95%EA%B5%AC_%ED%98%91%ED%9A%8C" title="독일 축구 협회">독일 축구 협회</a>(DFB)의 관련직을 처음으로 맡았다. <a href="/wiki/1963%EB%85%84" title="1963년">1963년</a>까지, 그는 <a href="/wiki/%EB%92%A4%EC%8A%A4%EB%B6%80%EB%A5%B4%ED%81%AC" title="뒤스부르크">뒤스부르크</a>의 DFB 위원으로 <a href="/wiki/%EB%8F%85%EC%9D%BC_%EC%B6%95%EA%B5%AC_%EA%B5%AD%EA%B0%80%EB%8C%80%ED%91%9C%ED%8C%80" title="독일 축구 국가대표팀">서독 국가대표팀</a> 감독을 맡았다.</p>
<p>이시기에 크라머는 언론계에 입문하기도 하였다. 그는 독일 국영방송 <a class="mw-redirect" href="/wiki/ZDF" title="ZDF">ZDF</a>의 스포츠 리드 편집가로 근무하였다. 그는 축구계에서 고립되었다 생각한 후, 크라머는 겨우 6개월만에 언론직을 그만두었다. 이후, 그는 <a href="/wiki/%EB%8F%85%EC%9D%BC_%EC%B6%95%EA%B5%AC_%ED%98%91%ED%9A%8C" title="독일 축구 협회">독일 축구 협회</a> (DFB) 에 의해 <a href="/wiki/%EC%9D%BC%EB%B3%B8" title="일본">일본</a>으로 보내졌고, 그곳에서 축구 코치로 일하였다.</p>
<p><a href="/wiki/1964%EB%85%84_%ED%95%98%EA%B3%84_%EC%98%AC%EB%A6%BC%ED%94%BD" title="1964년 하계 올림픽">1964년 도쿄 올림픽</a>을 앞두고, <a href="/wiki/%EC%9D%BC%EB%B3%B8_%EC%B6%95%EA%B5%AC_%ED%98%91%ED%9A%8C" title="일본 축구 협회">일본 축구 협회</a>는 <a href="/wiki/1960%EB%85%84" title="1960년">1960년</a>에 외국의 프로 감독을 모셔오기로 하였다. 크라머는 결국 이 일을 맡게 되었다. 크라머의 코칭과 리더립 역량, 그의 선수에 대한 노력 등에 힘입어, <a href="/wiki/%EC%9D%BC%EB%B3%B8" title="일본">일본</a>은 <a href="/wiki/%EC%95%84%EB%A5%B4%ED%97%A8%ED%8B%B0%EB%82%98_%EC%B6%95%EA%B5%AC_%EA%B5%AD%EA%B0%80%EB%8C%80%ED%91%9C%ED%8C%80" title="아르헨티나 축구 국가대표팀">아르헨티나</a>를 상대로 도쿄 올림픽에서 이변을 일으켰다. 이 역사적인 결과는 일본 축구계의 발전을 의미하였다. 크라머의 노력은 국가대표팀에 그치지 않았으며, 발전을 위한 행정에도 신경을 썼다. 국가 리그가 처음으로 출범하였고, 코칭스태프를 훈련하였으며, 이에 따라 일본 국가대표는 <a href="/wiki/1968%EB%85%84_%ED%95%98%EA%B3%84_%EC%98%AC%EB%A6%BC%ED%94%BD" title="1968년 하계 올림픽">1968년 멕시코 시티 올림픽</a>에서 동메달을 획득하며 성공을 거두었다.</p>
<p><a href="/wiki/1964%EB%85%84" title="1964년">1964년</a> <a href="/wiki/1%EC%9B%94_1%EC%9D%BC" title="1월 1일">1월 1일</a>, 크라머는 <a href="/wiki/%EC%84%9C%EB%8F%85" title="서독">서독</a>으로 복귀하여 <a href="/wiki/%ED%97%AC%EB%AC%B4%ED%8A%B8_%EC%87%A4" title="헬무트 쇤">헬무트 쇤</a>, <a href="/wiki/%EB%8F%85%EC%9D%BC_%EC%B6%95%EA%B5%AC_%EA%B5%AD%EA%B0%80%EB%8C%80%ED%91%9C%ED%8C%80" title="독일 축구 국가대표팀">서독 국가대표</a> 감독의 수석 코치가 되었다. 이 시기에 서독은 <a href="/wiki/1966%EB%85%84_FIFA_%EC%9B%94%EB%93%9C%EC%BB%B5" title="1966년 FIFA 월드컵">1966년 FIFA 월드컵</a>에 참여하였고, 서독은 홈팀 <a href="/wiki/%EC%9E%89%EA%B8%80%EB%9E%9C%EB%93%9C_%EC%B6%95%EA%B5%AC_%EA%B5%AD%EA%B0%80%EB%8C%80%ED%91%9C%ED%8C%80" title="잉글랜드 축구 국가대표팀">잉글랜드</a>와의 결승에서 석패하였다. 그의 재능은 <a class="mw-redirect" href="/wiki/FIFA" title="FIFA">FIFA</a>의 인정을 받았고, <a href="/wiki/1967%EB%85%84" title="1967년">1967년</a>에서 <a href="/wiki/1974%EB%85%84" title="1974년">1974년</a>까지 지구를 돌았고, 이 시기에 크라머는 <a href="/wiki/1969%EB%85%84" title="1969년">1969년</a>, <a href="/wiki/%EC%9D%BC%EB%B3%B8" title="일본">일본</a>에서 FIFA 감독 훈련을 시행하였고, <a href="/wiki/%EC%9D%BC%EB%B3%B8" title="일본">일본</a>의 감독 잠재력을 가진 자들을 발굴하였다. 이후, <a href="/wiki/1974%EB%85%84" title="1974년">1974년</a> <a href="/wiki/8%EC%9B%94_1%EC%9D%BC" title="8월 1일">8월 1일</a>, 크라머는 <a href="/wiki/%EB%AF%B8%EA%B5%AD_%EC%B6%95%EA%B5%AC_%EA%B5%AD%EA%B0%80%EB%8C%80%ED%91%9C%ED%8C%80" title="미국 축구 국가대표팀">미국 축구 국가대표팀</a>의 사령탑에 올랐다.</p>
<p>이 시기에 크라머는 <a class="mw-redirect" href="/wiki/%ED%91%B8%EC%8A%A4%EB%B3%BC-%EB%B6%84%EB%8D%B0%EC%8A%A4%EB%A6%AC%EA%B0%80" title="푸스볼-분데스리가">독일 분데스리가</a>의 여러 클럽으로부터 감독 제의를 받았으나, 처음에는 계속적으로 거절하였다. 결국 <a href="/wiki/1975%EB%85%84" title="1975년">1975년</a> <a href="/wiki/1%EC%9B%94_16%EC%9D%BC" title="1월 16일">1월 16일</a>, 그는 독일의 거함 <a href="/wiki/FC_%EB%B0%94%EC%9D%B4%EC%97%90%EB%A5%B8_%EB%AE%8C%ED%97%A8" title="FC 바이에른 뮌헨">FC 바이에른 뮌헨</a>의 새 사령탑이 되었다. 초기에 크라머는 수동적인 태도로 <a href="/wiki/%EB%AE%8C%ED%97%A8" title="뮌헨">뮌헨</a>에서 격렬한 비난을 받았다. 크라머의 경질설이 돌았을 때, 뮌헨의 주장 <a href="/wiki/%ED%94%84%EB%9E%80%EC%B8%A0_%EB%B2%A0%EC%BC%84%EB%B0%94%EC%9B%8C" title="프란츠 베켄바워">프란츠 베켄바워</a>는 크라머를 옹호하였다. 그는 스캔들로 <a href="/wiki/%EB%8F%85%EC%9D%BC_%EC%B6%95%EA%B5%AC_%EA%B5%AD%EA%B0%80%EB%8C%80%ED%91%9C%ED%8C%80" title="독일 축구 국가대표팀">서독 국가대표</a>에서 추방당하였을 때, 크라머는 그에게 신세를 졌다고 생각하였다.</p>
<p>이후 크라머는 <a href="/wiki/%EC%9C%A0%EB%9F%AC%ED%94%BC%EC%96%B8%EC%BB%B5_1975-76" title="유러피언컵 1975-76">유러피언컵 1975-76</a>을 우승하였고 이후 1976 <a href="/wiki/%EC%9D%B8%ED%84%B0%EC%BD%98%ED%8B%B0%EB%84%A8%ED%84%B8%EC%BB%B5_(%EC%B6%95%EA%B5%AC)" title="인터콘티넨털컵 (축구)">인터콘티넨털컵</a>에서도 우승을 거두었다. 그러나 75-76 시즌의 <a class="mw-redirect" href="/wiki/%ED%91%B8%EC%8A%A4%EB%B3%BC-%EB%B6%84%EB%8D%B0%EC%8A%A4%EB%A6%AC%EA%B0%80" title="푸스볼-분데스리가">분데스리가</a>에서 바이에른은 부진한 모습을 보였고, 시즌 종료 후 타이틀 탈환에 실패하였고, 크라머는 해임되었다. 이후, 그는 <a href="/wiki/%EC%95%84%EC%9D%B8%ED%8A%B8%EB%9D%BC%ED%9D%90%ED%8A%B8_%ED%94%84%EB%9E%91%ED%81%AC%ED%91%B8%EB%A5%B4%ED%8A%B8" title="아인트라흐트 프랑크푸르트">아인트라흐트 프랑크푸르트</a>의 감독을 맡게 된 반면, 프랑크푸르트의 역대 최고 성적을 낸 <a href="/wiki/%ED%97%9D%EA%B0%80%EB%A6%AC" title="헝가리">헝가리</a>인 감독 <a href="/wiki/%EB%A1%9C%EB%9E%80%ED%8A%B8_%EC%A4%84%EB%9F%AC" title="로란트 줄러">로란트 줄러</a> 감독이 바이에른의 감독이 되었다. 이 스왑딜은 양쪽 팀 모두에게 손해였고, 뮌헨은 역대 최악인 리그 12위로, 프랑크푸르트는 중위권으로 1977-78 시즌을 마무리하였다. 그 결과, 크라머는 프랑크푸르트와 <a href="/wiki/1978%EB%85%84" title="1978년">1978년</a> <a href="/wiki/6%EC%9B%94_30%EC%9D%BC" title="6월 30일">6월 30일</a>에 결별하였다.</p>
<p>이후, 그는 1982-83 시즌 이전까지 <a href="/wiki/%EB%8F%85%EC%9D%BC" title="독일">독일</a> 밖을 돌아다녔다. 그는 <a href="/wiki/1982%EB%85%84" title="1982년">1982년</a>에 <a href="/wiki/%EB%B0%94%EC%9D%B4%EC%96%B4_04_%EB%A0%88%EB%B2%84%EC%BF%A0%EC%A0%A0" title="바이어 04 레버쿠젠">바이어 04 레버쿠젠</a>과 계약을 하였다. 레버쿠젠에서 그는 팀을 2번째 해에 리그 순위를 9위 안으로 올렸다. 이후 그는 더 나은 성적을 내지 못하고 3시즌만에 해임되었다.</p>
<p>그는 이후에 다시 <a class="mw-redirect" href="/wiki/%ED%91%B8%EC%8A%A4%EB%B3%BC-%EB%B6%84%EB%8D%B0%EC%8A%A4%EB%A6%AC%EA%B0%80" title="푸스볼-분데스리가">분데스리가</a>에서 감독직을 맡지 않았으며, 크라머는 다양한 해외의 클럽과 국가대표팀 감독을 맡았다. 그 중, <a href="/wiki/1991%EB%85%84" title="1991년">1991년</a>부터 <a href="/wiki/1992%EB%85%84" title="1992년">1992년</a> 2월까지 <a href="/wiki/%EB%8C%80%ED%95%9C%EB%AF%BC%EA%B5%AD_U-23_%EC%B6%95%EA%B5%AC_%EA%B5%AD%EA%B0%80%EB%8C%80%ED%91%9C%ED%8C%80" title="대한민국 U-23 축구 국가대표팀">대한민국 U-23 국가대표팀</a>의 총감독 즉 기술고문을 맡아 28년만의 올림픽 본선 자력 진출의 쾌거에 일조하였다. 그러나 국내 축구계와의 갈등을 극복하지 못해 정작 <a href="/wiki/1992%EB%85%84_%ED%95%98%EA%B3%84_%EC%98%AC%EB%A6%BC%ED%94%BD" title="1992년 하계 올림픽">1992년 하계 올림픽</a> 본선을 치르지 못하고 <a href="/wiki/%EB%8C%80%ED%95%9C%EB%AF%BC%EA%B5%AD" title="대한민국">대한민국</a>을 떠나야 했다. <a href="/wiki/2002%EB%85%84" title="2002년">2002년</a>에 크라머는 축구계에서 공식으로 은퇴하였고, <a href="/wiki/2005%EB%85%84" title="2005년">2005년</a>에는 <a href="/wiki/%EC%9D%BC%EB%B3%B8" title="일본">일본</a> 축구 명예의 전당에 등록되기도 하였다.</p>
<h2><span class="mw-headline" id=".EC.82.AC.EC.83.9D.ED.99.9C">사생활</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%8D%B0%ED%8A%B8%EB%A7%88%EC%96%B4_%ED%81%AC%EB%9D%BC%EB%A8%B8&amp;action=edit&amp;section=2" title="부분 편집: 사생활">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>크라머는 <a href="/wiki/%EC%A0%9C2%EC%B0%A8_%EC%84%B8%EA%B3%84_%EB%8C%80%EC%A0%84" title="제2차 세계 대전">제2차 세계 대전</a>에서 낙하산 부대 <a href="/wiki/%EB%8C%80%EC%9C%84" title="대위">대위</a>로 참여하였다. 그는 현재 두번 결혼하였으며, <a href="/wiki/%EB%B0%94%EC%9D%B4%EC%97%90%EB%A5%B8_%EC%A3%BC" title="바이에른 주">바이에른 주</a>의 라이트 임 빈클에서 죽을 때까지 그의 가족과 거주하고 있다.</p>
<p>그는 국제적 업적을 달성하며, 크라머는 2차례 명예의 훈장을 수여받았다. 그는 독일 명예의 십자가 (Bundesverdienstkreuz) 와 일본 명예의 훈장을 수여받았다. 더 나아가서, 크라머는 <a class="mw-redirect" href="/wiki/%EC%95%84%EB%A9%94%EB%A6%AC%EC%B9%B4_%ED%86%A0%EC%B0%A9%EB%AF%BC" title="아메리카 토착민">미국 원주민</a>의 <a class="mw-redirect" href="/wiki/%EC%88%98%EC%A1%B1" title="수족">수족</a>과 <a class="mw-disambig" href="/wiki/%EB%AA%A8%ED%9E%88%EC%B9%B8" title="모히칸">모히칸</a>족의 명예 추장이다.</p>
