<h1 class="firstHeading" id="firstHeading" lang="ko">데플림픽</h1>
<div class="mw-body-content" id="bodyContent">
<div id="siteSub">위키백과, 우리 모두의 백과사전.</div>
<div id="contentSub"></div>
<div class="mw-jump" id="jump-to-nav">
					이동:					<a href="#mw-head">둘러보기</a>, 					<a href="#p-search">검색</a>
</div>
<div class="mw-content-ltr" dir="ltr" id="mw-content-text" lang="ko"><script>function mfTempOpenSection(id){var block=document.getElementById("mf-section-"+id);block.className+=" open-block";block.previousSibling.className+=" open-block";}</script><p><b>데플림픽</b>(<span lang="en" xml:lang="en">Deaflympics</span>)은 4년마다 개최되는 <a class="mw-redirect" href="/wiki/%EC%B2%AD%EA%B0%81_%EC%9E%A5%EC%95%A0%EC%9D%B8" title="청각 장애인">청각 장애인</a>을 위한 <a href="/wiki/%EC%98%AC%EB%A6%BC%ED%94%BD" title="올림픽">올림픽</a>과 같은 국제경기대회로서 스포츠를 통한 심신을 단련하고 세계농아간의 친목도모와 유대 강화를 목적으로 하고 있다.</p>
<p>청각장애인들을 위한 스포츠기구는 제1차 대전 이전에 조직되었지만 1924년 이전까지는 그다지 큰활동을 하지 못하였다. <a href="/wiki/%ED%94%84%EB%9E%91%EC%8A%A4" title="프랑스">프랑스</a> 청각장애인인 E.Rubens Alcais는 당시 활동하는 6개 국가의 청각 장애인을 위한 스포츠 기구와 함께 농아스포츠의 국제화에 크게 이바지하였다. 그리하여 1924년 8월에 <a href="/wiki/%ED%94%84%EB%9E%91%EC%8A%A4" title="프랑스">프랑스</a> <a class="mw-redirect" href="/wiki/%ED%8C%8C%EB%A6%AC_%EC%8B%9C" title="파리 시">파리</a>에서 <a href="/wiki/%EB%B2%A8%EA%B8%B0%EC%97%90" title="벨기에">벨기에</a>, <a href="/wiki/%EC%B2%B4%EC%BD%94%EC%8A%AC%EB%A1%9C%EB%B0%94%ED%82%A4%EC%95%84" title="체코슬로바키아">체코슬로바키아</a>, <a href="/wiki/%ED%94%84%EB%9E%91%EC%8A%A4" title="프랑스">프랑스</a>, <a href="/wiki/%EC%98%81%EA%B5%AD" title="영국">영국</a>, <a href="/wiki/%EB%84%A4%EB%8D%9C%EB%9E%80%EB%93%9C" title="네덜란드">네덜란드</a>, <a href="/wiki/%ED%8F%B4%EB%9E%80%EB%93%9C" title="폴란드">폴란드</a>, <a href="/wiki/%ED%97%9D%EA%B0%80%EB%A6%AC" title="헝가리">헝가리</a>, <a href="/wiki/%EC%9D%B4%ED%83%88%EB%A6%AC%EC%95%84" title="이탈리아">이탈리아</a>, <a href="/wiki/%EB%A3%A8%EB%A7%88%EB%8B%88%EC%95%84" title="루마니아">루마니아</a> 등 9개국 133명의 선수들이 참가한 가운데 제1회 하계 데플림픽이 열렸다. <a href="/wiki/%EB%8C%80%ED%95%9C%EB%AF%BC%EA%B5%AD" title="대한민국">대한민국</a>은 1985년 <a href="/wiki/%EB%AF%B8%EA%B5%AD" title="미국">미국</a> <a href="/wiki/%EB%A1%9C%EC%8A%A4%EC%95%A4%EC%A0%A4%EB%A0%88%EC%8A%A4" title="로스앤젤레스">로스앤젤레스</a>에서 개최된 제15회 하계 데플림픽에 처음으로 선수단을 파견하였다.</p>
