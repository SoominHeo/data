<p><b>덴버</b> (Denver)는 <a href="/wiki/%EC%BD%9C%EB%A1%9C%EB%9D%BC%EB%8F%84_%EC%A3%BC" title="콜로라도 주">콜로라도 주</a>에서 가장 큰 도시이자 주도이다. 2014년을 기준으로 덴버는 콜로라도 주에서 가장 인구가 많은 도시이다. 덴버는 <a href="/wiki/%EB%A1%9C%ED%82%A4_%EC%82%B0%EB%A7%A5" title="로키 산맥">로키 산맥</a>의 <a class="new" href="/w/index.php?title=%ED%94%84%EB%A1%A0%ED%8A%B8_%EC%82%B0%EB%A7%A5&amp;action=edit&amp;redlink=1" title="프론트 산맥 (없는 문서)">프론트 산맥</a> 동쪽과 <a class="new" href="/w/index.php?title=%ED%95%98%EC%9D%B4_%ED%8F%89%EC%9B%90&amp;action=edit&amp;redlink=1" title="하이 평원 (없는 문서)">하이 평원</a> 서쪽 끝자락에 있는 사우스플랫 리버 밸리에 걸쳐 있다. 도심은 체리크리크와 로키 산맥 고원의 동쪽으로 12 마일 (19 km) 떨어진 사우스플랫강 합류 지점에 있다. 덴버는 해수면으로부터 정확히 1 마일 (5,280 ft 또는 1,610 m) 고도에 있어 미국의 주요 도시 중 가장 높은 곳에 있기 때문에 마일 하이 시티 (Mile-High City)라는 별명이 붙었다.<sup class="reference" id="cite_ref-denfacts_1-0"><a href="#cite_note-denfacts-1">[1]</a></sup> 또한 <a href="/wiki/%EC%82%B0%EC%95%85_%ED%91%9C%EC%A4%80%EC%8B%9C" title="산악 표준시">산악 표준시</a>의 기준이 되는 그리니치 자오선 105도가 <a class="new" href="/w/index.php?title=%EB%8D%B4%EB%B2%84_%EC%9C%A0%EB%8B%88%EC%98%A8%EC%97%AD&amp;action=edit&amp;redlink=1" title="덴버 유니온역 (없는 문서)">덴버 유니온역</a>을 통화한다.</p>
<p>덴버는 글로벌리제이션 앤 월드 시티스 리서치 네크워크(Globalization and World Cities Research Network)에서 발표한 베타 월드 시티에 올랐다. 2014년을 기준으로 인구는 663,862명이며, 미국에서 21번째로 인구가 많은 도시이다.<sup class="reference" id="cite_ref-QF_2-0"><a href="#cite_note-QF-2">[2]</a></sup> 덴버-오로라-레이크우드 콜로라도 대도시 통계지구는 2013년 2,697,476명의 인구를 기록했고, 미국에서 21번째로 큰 대도시 통계지구이다.<sup class="reference" id="cite_ref-PopEstCBSA_3-0"><a href="#cite_note-PopEstCBSA-3">[3]</a></sup> 덴버-아우라 콜로라도 통계지구는 2013년 3,277,309의 인구를 기록했고, 미국에서 16번째로 큰 통계지구이다.<sup class="reference" id="cite_ref-PopEstCBSA_3-1"><a href="#cite_note-PopEstCBSA-3">[3]</a></sup> 덴버는 2010년을 기준으로 5,467,633명의 인구와 함께 두 개의 주에 걸쳐 직사각형으로 도시가 펼쳐져 있는 프론트 산맥 어반 코리더 중 가장 인구가 많다.<sup class="reference" id="cite_ref-PopEstCounties_4-0"><a href="#cite_note-PopEstCounties-4">[4]</a></sup> 또한 500-마일 (800 km) 반경 내 웨스트 마운틴 중 가장 거대하며, <a href="/wiki/%EB%AF%B8%EA%B5%AD_%EB%82%A8%EC%84%9C%EB%B6%80" title="미국 남서부">미국 남서부</a> 도시 중에서는 <a class="mw-disambig" href="/wiki/%ED%94%BC%EB%8B%89%EC%8A%A4" title="피닉스">피닉스</a>, <a class="mw-redirect" href="/wiki/%EC%97%98%ED%8C%8C%EC%86%8C" title="엘파소">엘파소</a>에 이어 세 번째로 거대하다. 덴버 대도시권은 남서부에서 피닉스 대도시권에 이어 두 번째로 크다.</p>

<h2><span class="mw-headline" id=".EC.97.AD.EC.82.AC">역사</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%8D%B4%EB%B2%84&amp;action=edit&amp;section=1" title="부분 편집: 역사">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>과거 덴버는 <a href="/wiki/%EC%8A%A4%ED%8E%98%EC%9D%B8" title="스페인">스페인</a>의 영토였다. 시굴자들이 이 지역에서 <a href="/wiki/%EA%B8%88" title="금">금</a>을 찾은 후인 <a href="/wiki/1858%EB%85%84" title="1858년">1858년</a>에 설립되었다. <a href="/wiki/1859%EB%85%84" title="1859년">1859년</a> <a href="/wiki/%EA%B3%A8%EB%93%9C_%EB%9F%AC%EC%8B%9C" title="골드 러시">골드 러시</a>의 시기에 체굴 정착자들을 위한 공급 기지가 되었다. <a href="/wiki/1860%EB%85%84" title="1860년">1860년</a> <a href="/wiki/4%EC%9B%94" title="4월">4월</a>에 덴버와 근처에 있는 오로라가 병합되었고, 이듬해에 도시로 승격되었다. <a href="/wiki/1867%EB%85%84" title="1867년">1867년</a> 콜로라도 영토의 수도가 되었다가, <a href="/wiki/1876%EB%85%84" title="1876년">1876년</a> 콜로라도가 정식 주로 승격되자 그 주도가 되었다.</p>
<p><a href="/wiki/1870%EB%85%84" title="1870년">1870년</a> 덴버 패시픽 철도가 완공되면서 도시가 더욱 넓혀졌다. <a href="/wiki/1880%EB%85%84" title="1880년">1880년</a>대와 <a href="/wiki/1890%EB%85%84" title="1890년">1890년</a>대를 거쳐 <a href="/wiki/%EC%9D%80" title="은">은</a>의 채굴 <a class="mw-disambig" href="/wiki/%EB%B6%90" title="붐">붐</a>이 일어나 도시의 재산을 추가하였다.</p>
<p><a href="/wiki/1900%EB%85%84" title="1900년">1900년</a>대 초반에 덴버는 평야의 읍에서 아름다운 도시로 변하였다. 도시의 정부는 도시 전역에 <a href="/wiki/%EB%82%98%EB%AC%B4" title="나무">나무</a>를 심고, 많은 <a href="/wiki/%EA%B3%B5%EC%9B%90" title="공원">공원</a>을 설립하였다. <a href="/wiki/1910%EB%85%84" title="1910년">1910년</a>에는 <a href="/wiki/%EB%A1%9C%ED%82%A4_%EC%82%B0%EB%A7%A5" title="로키 산맥">로키 산맥</a> 지방의 상업의 중심지가 되었다. <a href="/wiki/1927%EB%85%84" title="1927년">1927년</a>에는 모패트 터널이 완공되었다.</p>
<p><a class="mw-redirect" href="/wiki/%EC%A0%9C2%EC%B0%A8_%EC%84%B8%EA%B3%84%EB%8C%80%EC%A0%84" title="제2차 세계대전">제2차 세계대전</a> 중에는 덴버의 인구가 크게 증가하였다. 전쟁 후에 미국의 군사 조직들이 모두 덴버에 주둔하기 시작하였다. <a href="/wiki/1970%EB%85%84" title="1970년">1970년</a>대에 덴버는 도시의 산업을 계속 늘이는 동안 도시의 아름다운 자연을 보호하는 문제를 향하였다. <a href="/wiki/1968%EB%85%84" title="1968년">1968년</a>에는 공해를 줄이기 위하여 쓰레기를 태우는 일을 금지하였다.</p>
<p>덴버 도시 부흥 권위는 다운타운 지역의 낡고 버려진 건물들을 재건하기 위하여 <a class="mw-disambig" href="/wiki/%EC%8A%A4%EC%B9%B4%EC%9D%B4%EB%9D%BC%EC%9D%B8" title="스카이라인">스카이라인</a> <a href="/wiki/%ED%94%84%EB%A1%9C%EC%A0%9D%ED%8A%B8" title="프로젝트">프로젝트</a>를 시작하였다. 프로젝트는 <a href="/wiki/%EC%95%84%ED%8C%8C%ED%8A%B8" title="아파트">아파트</a>와 <a class="mw-redirect" href="/wiki/%EC%82%AC%EB%AC%B4%EC%8B%A4" title="사무실">사무실</a> 건물과 예술공연 단지 등을 포함하였다. 스카이라인 프로젝트는 <a href="/wiki/1985%EB%85%84" title="1985년">1985년</a>에 완성하였다. <a href="/wiki/1990%EB%85%84" title="1990년">1990년</a>에는 콜로라도 컨벤션 센터가 열렸다.</p>
<p>덴버의 가장 성공적 외곽 비지니스 개발 중의 하나는 도시의 남동부에 위치한 사무실 지대인 덴버 테크놀로지컬 센터이다. <a href="/wiki/1960%EB%85%84" title="1960년">1960년</a>대에 시작되어 지금도 계속 개발 중이다. 오늘날, 덴버 테크놀로지컬 센터는 수백 기업들의 본사이다.</p>
<h2><span class="mw-headline" id=".EC.8A.A4.ED.8F.AC.EC.B8.A0">스포츠</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%8D%B4%EB%B2%84&amp;action=edit&amp;section=2" title="부분 편집: 스포츠">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>덴버를 연고지로 하는 프로스포츠팀은 메이저 리그 야구에서는 <a href="/wiki/%EC%BD%9C%EB%A1%9C%EB%9D%BC%EB%8F%84_%EB%A1%9C%ED%82%A4%EC%8A%A4" title="콜로라도 로키스">콜로라도 로키스</a>. NBA 농구에서는 <a href="/wiki/%EB%8D%B4%EB%B2%84_%EB%84%88%EA%B8%B0%EC%B8%A0" title="덴버 너기츠">덴버 너기츠</a>, NFL 풋볼에서는 <a href="/wiki/%EB%8D%B4%EB%B2%84_%EB%B8%8C%EB%A1%B1%EC%BD%94%EC%8A%A4" title="덴버 브롱코스">덴버 브롱코스</a>, NHL 아이스하키에서는 <a href="/wiki/%EC%BD%9C%EB%A1%9C%EB%9D%BC%EB%8F%84_%EC%95%A0%EB%B2%8C%EB%9E%9C%EC%B9%98" title="콜로라도 애벌랜치">콜로라도 애벌랜치</a>가 있다.</p>
