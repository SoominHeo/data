<p>델 연산자와 <a href="/wiki/%EB%B0%9C%EC%82%B0_(%EB%B2%A1%ED%84%B0)" title="발산 (벡터)">발산</a>, <a href="/wiki/%ED%9A%8C%EC%A0%84_(%EB%B2%A1%ED%84%B0)" title="회전 (벡터)">회전</a>의 물리적 의미를 처음으로 발견한 사람은 <a href="/wiki/%EC%A0%9C%EC%9E%84%EC%8A%A4_%ED%81%B4%EB%9F%AC%ED%81%AC_%EB%A7%A5%EC%8A%A4%EC%9B%B0" title="제임스 클러크 맥스웰">제임스 클러크 맥스웰</a>이다. <a href="/wiki/%EC%A0%9C%EC%9E%84%EC%8A%A4_%ED%81%B4%EB%9F%AC%ED%81%AC_%EB%A7%A5%EC%8A%A4%EC%9B%B0" title="제임스 클러크 맥스웰">맥스웰</a>은 그의 논문 &lt;&lt;전기와 자기에 관한 논문&gt;&gt;에서는 <a href="/wiki/%EB%B0%9C%EC%82%B0_(%EB%B2%A1%ED%84%B0)" title="발산 (벡터)">발산</a>과 <a href="/wiki/%ED%9A%8C%EC%A0%84_(%EB%B2%A1%ED%84%B0)" title="회전 (벡터)">회전</a>을 각각 그 당시 많이 사용되던 단어인 컨버전스(convergence)와 로테이션(rotation)이라 이름 붙이고 <a href="/wiki/%EC%A0%84%EA%B8%B0%EC%9E%A5" title="전기장">전기장</a>과 <a href="/wiki/%EC%9E%90%EA%B8%B0%EC%9E%A5" title="자기장">자기장</a> 사이의 상호작용을 설명하였다. 그는 <a href="/wiki/%EB%B0%9C%EC%82%B0_(%EB%B2%A1%ED%84%B0)" title="발산 (벡터)">발산</a>의 물리적 의미를 <a href="/wiki/%EB%B0%9C%EC%82%B0%EC%A0%95%EB%A6%AC" title="발산정리">가우스의 발산정리</a>를 이용하여 설명하였으나 <a href="/wiki/%ED%9A%8C%EC%A0%84_(%EB%B2%A1%ED%84%B0)" title="회전 (벡터)">회전</a>의 경우 깊은 물리적 의미를 찾지는 못하였다.</p>
<p>지금의 이름인 ‘<a href="/wiki/%EB%B0%9C%EC%82%B0_(%EB%B2%A1%ED%84%B0)" title="발산 (벡터)">발산</a>(<span style="font-size: smaller;"><a href="/wiki/%EC%98%81%EC%96%B4" title="영어">영어</a>:</span> <span lang="en" xml:lang="en">divergence</span>)’과 ‘<a href="/wiki/%ED%9A%8C%EC%A0%84_(%EB%B2%A1%ED%84%B0)" title="회전 (벡터)">회전</a>(<span style="font-size: smaller;"><a href="/wiki/%EC%98%81%EC%96%B4" title="영어">영어</a>:</span> <span lang="en" xml:lang="en">curl</span>)’을 붙인 것은 <a href="/wiki/%EC%A1%B0%EC%82%AC%EC%9D%B4%EC%96%B4_%EC%9C%8C%EB%9F%AC%EB%93%9C_%EA%B8%B0%EB%B8%8C%EC%8A%A4" title="조사이어 윌러드 기브스">조사이어 윌러드 기브스</a>이다. 그는 <a href="/wiki/%EC%A0%9C%EC%9E%84%EC%8A%A4_%ED%81%B4%EB%9F%AC%ED%81%AC_%EB%A7%A5%EC%8A%A4%EC%9B%B0" title="제임스 클러크 맥스웰">맥스웰</a>보다 <a href="/wiki/%EB%B0%9C%EC%82%B0_(%EB%B2%A1%ED%84%B0)" title="발산 (벡터)">발산</a>과 <a href="/wiki/%ED%9A%8C%EC%A0%84_(%EB%B2%A1%ED%84%B0)" title="회전 (벡터)">회전</a>의 훨씬 더 근본적인 물리적 의미를 찾아냈다. 그가 찾아낸 <a href="/wiki/%EB%B0%9C%EC%82%B0_(%EB%B2%A1%ED%84%B0)" title="발산 (벡터)">발산</a>의 의미는 공간에서 유체의 속도벡터와 공간 상의 어느 한 점에서 유체가 빠져나가는 속도를 잇는 연산자였고, <a href="/wiki/%ED%9A%8C%EC%A0%84_(%EB%B2%A1%ED%84%B0)" title="회전 (벡터)">회전</a>의 의미는 어떤 강체 각 지점의 속도 벡터와 강체의 각속도를 연결짓는 연산자였다. 하지만 그는 <a href="/wiki/%ED%9A%8C%EC%A0%84_(%EB%B2%A1%ED%84%B0)" title="회전 (벡터)">회전</a>의 더욱 깊은 의미를 찾는 데에는 실패하였고, 그 의미는 아직까지도 명확하지 않다.</p>
