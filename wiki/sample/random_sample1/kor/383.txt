<h2><span class="mw-headline" id="2008.EB.85.84_10.EC.9B.94_31.EC.9D.BC"><a href="/wiki/2008%EB%85%84" title="2008년">2008년</a> <a href="/wiki/10%EC%9B%94_31%EC%9D%BC" title="10월 31일">10월 31일</a></span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=2008%EB%85%84_10%EC%9B%94&amp;action=edit&amp;section=1" title="부분 편집: 2008년 10월 31일">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<ul>
<li><a class="mw-redirect" href="/wiki/%EC%84%9C%EC%9A%B8" title="서울">서울</a>에서 옛 나산백화점 재건축지가 붕괴돼 1명이 매몰되는 사고가 발생했다.</li>
<li><a href="/wiki/%EB%A6%AC%EB%B9%84%EC%95%84" title="리비아">리비아</a>가 대미 테러 보상금으로 15억 달러를 지급해 양국 외교 관계가 정상화 기로를 맞게 됐다.</li>
<li>티베트 역사상 최악의 눈폭풍으로 7명이 사망했다.</li>
<li><a href="/wiki/%EC%95%84%ED%94%84%EB%A6%AC%EC%B9%B4" title="아프리카">아프리카</a>의 기존 경제 블록(COMESA, SADC, EAC)이 하나로 규모를 합치는 데 합의함에 따라 26개국 7억 5,700만 명의 규모로 확장됐다.</li>
</ul>
<h2><span class="mw-headline" id="2008.EB.85.84_10.EC.9B.94_30.EC.9D.BC"><a href="/wiki/2008%EB%85%84" title="2008년">2008년</a> <a href="/wiki/10%EC%9B%94_30%EC%9D%BC" title="10월 30일">10월 30일</a></span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=2008%EB%85%84_10%EC%9B%94&amp;action=edit&amp;section=2" title="부분 편집: 2008년 10월 30일">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<ul>
<li><a href="/wiki/%EC%8B%9C%EB%A6%AC%EC%95%84" title="시리아">시리아</a>가 <a href="/wiki/%EB%AF%B8%EA%B5%AD" title="미국">미국</a>의 동부 공격에 따른 이의 제기를 <a href="/wiki/%EC%9C%A0%EC%97%94" title="유엔">유엔</a>에 공식 제기했다.</li>
<li><a href="/wiki/%EB%A7%90%EB%A0%88%EC%9D%B4%EC%8B%9C%EC%95%84" title="말레이시아">말레이시아</a>가 코소보를 주권 국가로 인정했다.</li>
<li><a href="/wiki/%EC%9D%B8%EB%8F%84" title="인도">인도</a> <a href="/wiki/%EC%95%84%EC%82%BC_%EC%A3%BC" title="아삼 주">아삼 주</a> 연쇄 테러로 66명이 죽고 470명 이상이 부상당했다.</li>
<li><a href="/wiki/%EC%98%81%EA%B5%AD" title="영국">영국</a> 이스트 데본 지역에서 우박 폭풍이 발생해 세인트 메리와 호니튼 지역에 홍수가 발생했다.</li>
<li><a href="/wiki/%EC%BA%90%EB%82%98%EB%8B%A4" title="캐나다">캐나다</a>의 총리인 스티픈 하퍼의 새 내각이 10월 14일 연방 총선 후 출범했다.</li>
</ul>
<h2><span class="mw-headline" id="2008.EB.85.84_10.EC.9B.94_27.EC.9D.BC"><a href="/wiki/2008%EB%85%84" title="2008년">2008년</a> <a href="/wiki/10%EC%9B%94_27%EC%9D%BC" title="10월 27일">10월 27일</a></span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=2008%EB%85%84_10%EC%9B%94&amp;action=edit&amp;section=3" title="부분 편집: 2008년 10월 27일">편집</a><span class="mw-editsection-bracket">]</span></span></h2>

<li><a href="/wiki/%EB%AF%B8%EA%B5%AD" title="미국">미국</a> <a class="mw-redirect" href="/wiki/%EC%97%B0%EB%B0%A9%EC%A4%80%EB%B9%84%EC%A0%9C%EB%8F%84%EC%9D%B4%EC%82%AC%ED%9A%8C" title="연방준비제도이사회">연방준비제도이사회</a>를 비롯한 세계 7개 주요국 중앙은행들이 일제히 전격적인 금리인하를 실시했다.</li>
<li>2008년 <a href="/wiki/%EB%85%B8%EB%B2%A8_%ED%99%94%ED%95%99%EC%83%81" title="노벨 화학상">노벨 화학상</a> 수상자로 <a href="/wiki/%EB%AF%B8%EA%B5%AD" title="미국">미국</a>의 <a class="mw-redirect" href="/wiki/%EB%A7%88%ED%8B%B4_%EC%B2%BC%ED%94%BC" title="마틴 첼피">마틴 첼피</a>, <a href="/wiki/%EB%A1%9C%EC%A0%80_%EC%B2%B8" title="로저 첸">로저 첸</a>과 <a href="/wiki/%EC%9D%BC%EB%B3%B8" title="일본">일본</a>의 <a href="/wiki/%EC%8B%9C%EB%AA%A8%EB%AC%B4%EB%9D%BC_%EC%98%A4%EC%82%AC%EB%AC%B4" title="시모무라 오사무">시모무라 오사무</a> 3인이 공동 선정되었다.</li>
<li><a href="/wiki/%EB%AA%B0%EB%94%94%EB%B8%8C" title="몰디브">몰디브</a>에서 최초로 민주주의 대통령선거가 진행 중이며, 현직 대통령인 <a href="/wiki/%EB%A7%88%EC%9A%B0%EB%AC%B8_%EC%95%95%EB%91%98_%EA%B0%80%EC%9C%B0" title="마우문 압둘 가윰">마우문 압둘 가윰</a>을 포함하여 6명의 후보자가 경선에 나섰다.</li>
<li>예티 항공 103편 비행기가 <a href="/wiki/%EB%84%A4%ED%8C%94" title="네팔">네팔</a> 영토의 에베레스트 인근에 부딪혀 폭발해 18명의 승객이 숨졌다.</li>
<li>우크라이나 의회가 대통령령으로 해산돼 조기 선거가 이뤄질 전망이다.</li>
</ul>
