<p><b>도다 성</b>(<span style="font-size: smaller;"><a href="/wiki/%EC%9D%BC%EB%B3%B8%EC%96%B4" title="일본어">일본어</a>:</span> <span lang="ja"><span lang="ja"><ruby><rb>富田城</rb> <rp>(</rp><rt>とだじょう</rt><rp>)</rp></ruby></span></span>)은 <a href="/wiki/%EC%8B%9C%EB%A7%88%EB%84%A4_%ED%98%84" title="시마네 현">시마네 현</a> <a href="/wiki/%EC%95%BC%EC%8A%A4%EA%B8%B0_%EC%8B%9C" title="야스기 시">야스기 시</a> <a class="new" href="/w/index.php?title=%ED%9E%88%EB%A1%9C%EC%84%B8_%EC%A0%95&amp;action=edit&amp;redlink=1" title="히로세 정 (없는 문서)">히로세 정</a> 도다에 있는 산성이다. 갓산 산에 축성되었기 때문에 <b>갓산 산성</b>(月山城), <b>갓산토다 성</b>(月山富田城)이란 이명이 존재한다. 성터는 국가 사적으로 지정되어 있다.</p>
<h2><span class="mw-headline" id=".EA.B0.9C.EC.9A.94">개요</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%8F%84%EB%8B%A4_%EC%84%B1&amp;action=edit&amp;section=1" title="부분 편집: 개요">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>산기슭에 거관을 조성하였고, 거관 뒤 높이 197m의 갓산 산정에 성을 쌓아 전시에 대비하였다. 산정에는 <a href="/wiki/%ED%98%BC%EB%A7%88%EB%A3%A8" title="혼마루">혼마루</a>, <a class="mw-redirect" href="/wiki/%EB%8B%88%EB%85%B8%EB%A7%88%EB%A3%A8" title="니노마루">니노마루</a>, <a class="mw-redirect" href="/wiki/%EC%82%B0%EB%85%B8%EB%A7%88%EB%A3%A8" title="산노마루">산노마루</a>를 일렬로 배치하였고, 갓산 산의 능성에 여러 <a href="/wiki/%EA%B5%AC%EB%A3%A8%EC%99%80" title="구루와">구루와</a>를 두었다. 특히 혼마루와 니노마루 사이에는 <a href="/wiki/%ED%95%B4%EC%9E%90" title="해자">호리키리</a>가 혼마루와 니노마루 사이를 가르고 있다. <a href="/wiki/%EB%82%9C%EB%B3%B4%EC%BF%A0%EC%B4%88_%EC%8B%9C%EB%8C%80" title="난보쿠초 시대">남북조</a>, <a href="/wiki/%EB%AC%B4%EB%A1%9C%EB%A7%88%EC%B9%98_%EC%8B%9C%EB%8C%80" title="무로마치 시대">무로마치 시대</a> <a class="mw-redirect" href="/wiki/%EC%9D%B4%EC%A6%88%EB%AA%A8%EB%85%B8%EC%BF%A0%EB%8B%88" title="이즈모노쿠니">이즈모 국</a> <a class="mw-redirect" href="/wiki/%EC%8A%88%EA%B3%A0" title="슈고">슈고</a>의 거성으로 알려져 있으며, <a href="/wiki/%EC%84%BC%EA%B3%A0%EC%BF%A0_%EC%8B%9C%EB%8C%80" title="센고쿠 시대">센고쿠 시대</a>에는 <a href="/wiki/%EC%A3%BC%EA%B3%A0%EC%BF%A0_%EC%A7%80%EB%B0%A9" title="주고쿠 지방">주고쿠 지방</a>을 호령한 아마고 가문(尼子氏)의 본거지이기도 하다. <a class="new" href="/w/index.php?title=%EC%95%84%EB%A7%88%EA%B3%A0_%ED%95%98%EB%A3%A8%ED%9E%88%EC%82%AC&amp;action=edit&amp;redlink=1" title="아마고 하루히사 (없는 문서)">아마고 가문</a>과 <a href="/wiki/%EC%98%A4%EC%9A%B0%EC%B9%98_%EC%9A%94%EC%8B%9C%ED%83%80%EC%B9%B4" title="오우치 요시타카">오우치 가문</a> 및 <a href="/wiki/%EB%AA%A8%EB%A6%AC_%EB%AA%A8%ED%86%A0%EB%82%98%EB%A6%AC" title="모리 모토나리">모리 가문</a>의 전투에서 적의 공격을 번번이 격퇴한 센고쿠 시대의 견성으로 알려져 있다(<a href="/wiki/%EA%B0%93%EC%82%B0%ED%86%A0%EB%8B%A4_%EC%84%B1_%EC%A0%84%ED%88%AC" title="갓산토다 성 전투">갓산토다 성 전투</a>). 이런 연유로 천공의 성(天空の城)으로도 불리며, 일본 5대 산악성 중에 한곳이기도 하다. <a href="/wiki/1600%EB%85%84" title="1600년">1600년</a> 이후 호리오 가문이 부임하였고, <a href="/wiki/1611%EB%85%84" title="1611년">1611년</a> <a class="new" href="/w/index.php?title=%ED%98%B8%EB%A6%AC%EC%98%A4_%EB%8B%A4%EB%8B%A4%ED%95%98%EB%A3%A8&amp;action=edit&amp;redlink=1" title="호리오 다다하루 (없는 문서)">호리오 다다하루</a>가 <a href="/wiki/%EB%A7%88%EC%93%B0%EC%97%90_%EC%84%B1" title="마쓰에 성">마쓰에 성</a>을 거점으로 삼았기 때문에 폐성되었다. <a href="/wiki/1934%EB%85%84" title="1934년">1934년</a> 국가 사적으로 지정되었다.</p>
