<p><b>도도 다카토라</b>(<span style="font-size: smaller;"><a href="/wiki/%EC%9D%BC%EB%B3%B8%EC%96%B4" title="일본어">일본어</a>:</span> <span lang="ja"><span lang="ja"><ruby><rb>藤堂高虎</rb> <rp>(</rp><rt>とうどう たかとら</rt><rp>)</rp></ruby></span></span> <small>토도 타카토라<sup>[<a href="/wiki/%EC%9C%84%ED%82%A4%EB%B0%B1%EA%B3%BC:%EC%9D%BC%EB%B3%B8%EC%96%B4%EC%9D%98_%ED%95%9C%EA%B8%80_%ED%91%9C%EA%B8%B0" title="위키백과:일본어의 한글 표기">*</a>]</sup></small>, <a href="/wiki/1556%EB%85%84" title="1556년">1556년</a> <a href="/wiki/2%EC%9B%94_16%EC%9D%BC" title="2월 16일">2월 16일</a> ~ <a href="/wiki/1630%EB%85%84" title="1630년">1630년</a> <a href="/wiki/11%EC%9B%94_9%EC%9D%BC" title="11월 9일">11월 9일</a>)는 <a href="/wiki/%EC%84%BC%EA%B3%A0%EC%BF%A0_%EC%8B%9C%EB%8C%80" title="센고쿠 시대">센고쿠 시대</a>부터 <a href="/wiki/%EC%97%90%EB%8F%84_%EC%8B%9C%EB%8C%80" title="에도 시대">에도 시대</a> 전기에 활약한 <a href="/wiki/%EB%8B%A4%EC%9D%B4%EB%AC%98" title="다이묘">다이묘</a>이다. <a href="/wiki/%EB%B2%95%EB%AA%85" title="법명">법명</a>은 <b>도모요시</b>(智吉). 주군을 여러번 바꾼 <a class="new" href="/w/index.php?title=%EC%84%BC%EA%B3%A0%EC%BF%A0_%EB%AC%B4%EC%9E%A5&amp;action=edit&amp;redlink=1" title="센고쿠 무장 (없는 문서)">센고쿠 무장</a>으로 잘 알려져 있다.</p>
<p>축성술이 뛰어나 <a href="/wiki/%EC%9A%B0%EC%99%80%EC%A7%80%EB%A7%88_%EC%84%B1" title="우와지마 성">우와지마 성</a>, <a href="/wiki/%EC%9D%B4%EB%A7%88%EB%B0%94%EB%A6%AC_%EC%84%B1" title="이마바리 성">이마바리 성</a>, <a class="new" href="/w/index.php?title=%EC%82%AC%EC%82%AC%EC%95%BC%EB%A7%88_%EC%84%B1&amp;action=edit&amp;redlink=1" title="사사야마 성 (없는 문서)">사사야마 성</a>, <a class="new" href="/w/index.php?title=%EC%93%B0_%EC%84%B1&amp;action=edit&amp;redlink=1" title="쓰 성 (없는 문서)">쓰 성</a>, <a class="mw-redirect" href="/wiki/%EC%9D%B4%EA%B0%80_%EC%9A%B0%EC%97%90%EB%85%B8_%EC%84%B1" title="이가 우에노 성">이가 우에노 성</a>, <a class="new" href="/w/index.php?title=%EC%A0%9C%EC%A0%9C_%EC%84%B1&amp;action=edit&amp;redlink=1" title="제제 성 (없는 문서)">제제 성</a> 등을 축성했다. 다카토라의 축성은 석벽을 높게 쌓는 것과 <a href="/wiki/%ED%95%B4%EC%9E%90" title="해자">해자</a>의 설계에 특징이 있다. 같은 축성의 명수 <a href="/wiki/%EA%B0%80%ED%86%A0_%EA%B8%B0%EC%9A%94%EB%A7%88%EC%82%AC" title="가토 기요마사">가토 기요마사</a>는 석벽의 휨을 중시한다는 것에서 대비된다.</p>

<p><a href="/wiki/%EA%B3%A0%EC%A7%80" title="고지">고지</a> 2년 <a href="/wiki/%EC%9D%8C%EB%A0%A5_1%EC%9B%94_6%EC%9D%BC" title="음력 1월 6일">음력 1월 6일</a>, <a href="/wiki/%EC%98%A4%EB%AF%B8_%EA%B5%AD" title="오미 국">오미</a> 이누카미 군 도도무라(현재 <a href="/wiki/%EC%8B%9C%EA%B0%80_%ED%98%84" title="시가 현">시가 현</a> <a href="/wiki/%EC%9D%B4%EB%88%84%EC%B9%B4%EB%AF%B8_%EA%B5%B0" title="이누카미 군">이누카미 군</a> <a href="/wiki/%EA%B3%A0%EB%9D%BC_%EC%A0%95" title="고라 정">고라 정</a>)의 토호 <a class="new" href="/w/index.php?title=%EB%8F%84%EB%8F%84_%EB%8F%84%EB%9D%BC%ED%83%80%EC%B9%B4&amp;action=edit&amp;redlink=1" title="도도 도라타카 (없는 문서)">도도 도라타카</a>(藤堂虎高)의 아들로 태어난다. 도도 씨는 선조 대대로 영주였지만, 센고쿠 시대에 들어서 서서히 몰락했고, 다카토라가 태어날 즈음에는 일개 농민에 불과했다. 아명은 요요시? 혹은 도모요시? 이다.</p>
<p>처음 오미의 <a class="mw-redirect" href="/wiki/%EC%84%BC%EA%B3%A0%EC%BF%A0_%EB%8B%A4%EC%9D%B4%EB%AC%98" title="센고쿠 다이묘">센고쿠 다이묘</a> <a href="/wiki/%EC%95%84%EC%9E%90%EC%9D%B4_%EB%82%98%EA%B0%80%EB%A7%88%EC%82%AC" title="아자이 나가마사">아자이 나가마사</a>의 가신이 되었고, <a href="/wiki/%EA%B2%90%ED%82%A4" title="겐키">겐키</a> 원년(<a href="/wiki/1570%EB%85%84" title="1570년">1570년</a>) <a href="/wiki/%EC%95%84%EB%84%A4%EA%B0%80%EC%99%80_%EC%A0%84%ED%88%AC" title="아네가와 전투">아네가와 전투</a>에 참전해 무공을 세워 나가마사를 기쁘게 하였다. <a class="mw-disambig" href="/wiki/%EB%8D%B4%EC%87%BC" title="덴쇼">덴쇼</a> 원년(<a href="/wiki/1573%EB%85%84" title="1573년">1573년</a>) 아자이 씨가 <a href="/wiki/%EC%98%A4%EB%8B%A4_%EB%85%B8%EB%B6%80%EB%82%98%EA%B0%80" title="오다 노부나가">오다 노부나가</a>에 의해 멸문되자, 아자이 가의 옛 가신인 <a class="new" href="/w/index.php?title=%EC%95%84%EC%93%B0%EC%A7%80_%EC%82%AC%EB%8B%A4%EC%9C%A0%ED%82%A4&amp;action=edit&amp;redlink=1" title="아쓰지 사다유키 (없는 문서)">아쓰지 사다유키</a>를 거쳐 <a class="new" href="/w/index.php?title=%EC%9D%B4%EC%86%8C%EB%85%B8_%EA%B0%80%EC%A6%88%EB%A7%88%EC%82%AC&amp;action=edit&amp;redlink=1" title="이소노 가즈마사 (없는 문서)">이소노 가즈마사</a>를 섬겼다.</p>
<h3><span class="mw-headline" id=".EC.98.A4.EB.8B.A4.2C_.ED.95.98.EC.8B.9C.EB.B0.94.EC.9D.98_.EA.B0.80.EC.8B.A0.EC.9C.BC.EB.A1.9C">오다, 하시바의 가신으로</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%8F%84%EB%8F%84_%EB%8B%A4%EC%B9%B4%ED%86%A0%EB%9D%BC&amp;action=edit&amp;section=3" title="부분 편집: 오다, 하시바의 가신으로">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>뒤이어 오미를 벗어나 노부나가의 조카인 <a class="new" href="/w/index.php?title=%EC%98%A4%EB%8B%A4_%EB%85%B8%EB%B6%80%EC%A6%88%EB%AF%B8&amp;action=edit&amp;redlink=1" title="오다 노부즈미 (없는 문서)">오다 노부즈미</a>를 모신다. 그러나 그리 오래 가지는 못해 덴쇼 4년(<a href="/wiki/1576%EB%85%84" title="1576년">1576년</a>) <a class="mw-redirect" href="/wiki/%ED%95%98%EC%8B%9C%EB%B0%94_%ED%9E%88%EB%8D%B0%EB%82%98%EA%B0%80" title="하시바 히데나가">하시바 히데나가</a>를 섬겨, 3000석의 작은 영토를 받는다. 히데나가의 수하에서 <a class="new" href="/w/index.php?title=%EC%A3%BC%EA%B3%A0%EC%BF%A0_%EC%A0%95%EB%B2%8C&amp;action=edit&amp;redlink=1" title="주고쿠 정벌 (없는 문서)">주고쿠 정벌</a>, <a href="/wiki/%EC%8B%9C%EC%A6%88%EA%B0%80%ED%83%80%EC%BC%80_%EC%A0%84%ED%88%AC" title="시즈가타케 전투">시즈가타케 전투</a>등에 종군한다. 시즈가타케 전투에서 발군의 기량을 뽐내 무공을 세웠기 때문에 2000석이 가증되었다. 후에 <a href="/wiki/%EB%8F%84%EC%9A%94%ED%86%A0%EB%AF%B8_%ED%9E%88%EB%8D%B0%EC%9A%94%EC%8B%9C" title="도요토미 히데요시">히데요시</a>로부터 5000석을 더 받아 합계 1만 석의 다이묘로 승진했다.</p>
<p>덴쇼 13년(<a href="/wiki/1585%EB%85%84" title="1585년">1585년</a>) <a href="/wiki/%EA%B8%B0%EC%9D%B4_%EA%B5%AD" title="기이 국">기이</a> 사이카 공격에 종군하고, 이 즈음 히데요시의 명령으로 사이카토의 수장으로 있는 <a class="new" href="/w/index.php?title=%EC%8A%A4%EC%A6%88%ED%82%A4_%EC%8B%9C%EA%B2%8C%EC%98%A4%ED%82%A4&amp;action=edit&amp;redlink=1" title="스즈키 시게오키 (없는 문서)">스즈키 시게오키</a>를 모략으로 자결로 몰고간다. 덴쇼 15년(<a href="/wiki/1587%EB%85%84" title="1587년">1587년</a>) <a href="/wiki/%EA%B7%9C%EC%8A%88_%EC%A0%95%EB%B2%8C" title="규슈 정벌">규슈 정벌</a>에도 종군해, 1만 석이 가증되어 좀 전 것의 배인 2만 석이 되었고, 이 즈음 히데요시의 천거로 정오위하, 사도노카미에 위계되었다.</p>
<p>덴쇼 19년(<a href="/wiki/1591%EB%85%84" title="1591년">1591년</a>) 히데나가가 죽자, 그의 <a class="mw-redirect" href="/wiki/%EC%96%91%EC%9E%90_(%EA%B0%80%EC%A1%B1)" title="양자 (가족)">양자</a> <a class="new" href="/w/index.php?title=%ED%95%98%EC%8B%9C%EB%B0%94_%ED%9E%88%EB%8D%B0%EC%93%B0%EA%B5%AC&amp;action=edit&amp;redlink=1" title="하시바 히데쓰구 (없는 문서)">하시바 히데쓰구</a>를 주군으로 모신다. 히데쓰구의 대리로 다음 해인 <a href="/wiki/%EB%B6%84%EB%A1%9C%EC%BF%A0" title="분로쿠">분로쿠</a> 원년 <a href="/wiki/%EC%9E%84%EC%A7%84%EC%99%9C%EB%9E%80" title="임진왜란">임진왜란</a> 때 군을 이끌고 출정한다. 수군 9000여 명을 이끌고 참전하였으나, <a href="/wiki/%EC%9D%B4%EC%88%9C%EC%8B%A0" title="이순신">이순신</a>이 지휘한 <a href="/wiki/%EC%A1%B0%EC%84%A0" title="조선">조선</a> <a href="/wiki/%EC%88%98%EA%B5%B0" title="수군">수군</a>에게 여러 차례 패전하였다. 일본 수군의 첫 패배였던 <a href="/wiki/%EC%98%A5%ED%8F%AC_%ED%95%B4%EC%A0%84" title="옥포 해전">옥포 해전</a> 당시 지휘관이기도 했다.</p>
<p>분로쿠 4년(<a href="/wiki/1595%EB%85%84" title="1595년">1595년</a>) 히데쓰구가 처형당하자 <a href="/wiki/%EA%B3%A0%EC%95%BC_%EC%82%B0" title="고야 산">고야 산</a>으로 출가한다. 하지만 그의 재능을 아까워한 <a href="/wiki/%EB%8F%84%EC%9A%94%ED%86%A0%EB%AF%B8_%ED%9E%88%EB%8D%B0%EC%9A%94%EC%8B%9C" title="도요토미 히데요시">도요토미 히데요시</a>의 소환에 응해 환속한다. 5만 석을 가증받아 <a href="/wiki/%EC%9D%B4%EC%9A%94_%EA%B5%AD" title="이요 국">이요</a> 사카지마(현재 <a href="/wiki/%EC%9A%B0%EC%99%80%EC%A7%80%EB%A7%88_%EC%8B%9C" title="우와지마 시">우와지마 시</a>) 7만 석 다이묘로 승진했다. 이때 히데요시로부터 니혼마루라는 군함을 받았다고 전해진다.</p>
<p><a href="/wiki/%EA%B2%8C%EC%9D%B4%EC%B4%88" title="게이초">게이초</a> 2년(<a href="/wiki/1597%EB%85%84" title="1597년">1597년</a>)부터 <a href="/wiki/%EC%A0%95%EC%9C%A0%EC%9E%AC%EB%9E%80" title="정유재란">정유재란</a>때 수군을 이끌고 참가한다. 조선 수군의 장수 <a href="/wiki/%EC%9B%90%EA%B7%A0" title="원균">원균</a>이 이끄는 수군을 칠천량에서 섬멸한다(<a href="/wiki/%EC%B9%A0%EC%B2%9C%EB%9F%89_%ED%95%B4%EC%A0%84" title="칠천량 해전">칠천량 해전</a>). 그리고 귀국 후 1만 석이 가증되어 8만 석의 다이묘가 된다. 그리고 거성이었던 사카지마 마루토키 성을 대규모로 개축한다. 완성후 <a href="/wiki/%EC%9A%B0%EC%99%80%EC%A7%80%EB%A7%88_%EC%84%B1" title="우와지마 성">우와지마 성</a>으로 개명한다.</p>
<h3><span class="mw-headline" id=".EC.84.B8.ED.82.A4.EA.B0.80.ED.95.98.EB.9D.BC">세키가하라</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%8F%84%EB%8F%84_%EB%8B%A4%EC%B9%B4%ED%86%A0%EB%9D%BC&amp;action=edit&amp;section=4" title="부분 편집: 세키가하라">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>게이초 3년(<a href="/wiki/1598%EB%85%84" title="1598년">1598년</a>) 8월, 히데요시가 죽자 다카토라는 다음 번 패자(覇者)는 <a href="/wiki/%EB%8F%84%EC%BF%A0%EA%B0%80%EC%99%80_%EC%9D%B4%EC%97%90%EC%95%BC%EC%8A%A4" title="도쿠가와 이에야스">도쿠가와 이에야스</a>라는 걸 판단하고 재빨리 이에야스에게로 접근한다. 그리고 도요토미 씨의 가신단이 무단파와 문치파로 분열하자, 다카토라는 무단파와 함께 <a href="/wiki/%EB%8F%84%EC%BF%A0%EA%B0%80%EC%99%80_%EC%9D%B4%EC%97%90%EC%95%BC%EC%8A%A4" title="도쿠가와 이에야스">도쿠가와 이에야스</a> 측으로 붙는다.</p>
<p>게이초 5년(<a href="/wiki/1600%EB%85%84" title="1600년">1600년</a>) <a href="/wiki/%EC%9A%B0%EC%97%90%EC%8A%A4%EA%B8%B0_%EA%B0%80%EA%B2%8C%EC%B9%B4%EC%93%B0" title="우에스기 가게카쓰">우에스기 가게카쓰</a>에 대해 이에야스가 거병하자, 이에야스의 <a class="new" href="/w/index.php?title=%EC%95%84%EC%9D%B4%EC%A6%88_%EC%A0%95%EB%B2%8C&amp;action=edit&amp;redlink=1" title="아이즈 정벌 (없는 문서)">아이즈 정벌</a>에 병력을 이끌고 종군한다.[아이즈 정벌로 인해 당시 정치의 중심이었던 오사카가 텅비게 되자 이 틈을 노린 <a href="/wiki/%EC%9D%B4%EC%8B%9C%EB%8B%A4_%EB%AF%B8%EC%93%B0%EB%82%98%EB%A6%AC" title="이시다 미쓰나리">이시다 미쓰나리</a> 등의 서군이 거병하였다. 다카토라는 이에야스의 동군에 속해 <a href="/wiki/%EC%98%A4%EB%8B%A4_%ED%9E%88%EB%8D%B0%EB%85%B8%EB%B6%80" title="오다 히데노부">오다 히데노부</a>가 지키고 있는 <a href="/wiki/%EA%B8%B0%ED%9B%84_%EC%84%B1" title="기후 성">기후 성</a>공격에 참전하고, 9월 15일 <a href="/wiki/%EC%84%B8%ED%82%A4%EA%B0%80%ED%95%98%EB%9D%BC_%EC%A0%84%ED%88%AC" title="세키가하라 전투">세키가하라 전투</a>에도 참가해 <a href="/wiki/%EC%98%A4%ED%83%80%EB%8B%88_%EC%9A%94%EC%8B%9C%EC%93%B0%EA%B5%AC" title="오타니 요시쓰구">오타니 요시쓰구</a>의 부대와 겨뤘지만 그 부대에 걸려 꼼짝못했다. 또, <a href="/wiki/%EC%99%80%ED%82%A4%EC%9E%90%EC%B9%B4_%EC%95%BC%EC%8A%A4%ED%95%98%EB%A3%A8" title="와키자카 야스하루">와키자카 야스하루</a>, <a href="/wiki/%EC%98%A4%EA%B0%80%EC%99%80_%EC%8A%A4%EC%BC%80%ED%83%80%EB%8B%A4" title="오가와 스케타다">오가와 스케타다</a>, <a href="/wiki/%EA%B5%AC%EC%93%B0%ED%82%A4_%EB%AA%A8%ED%86%A0%EC%93%B0%EB%82%98" title="구쓰키 모토쓰나">구쓰키 모토쓰나</a>, <a href="/wiki/%EC%95%84%EC%B9%B4%EC%9E%90_%EB%82%98%EC%98%A4%EC%95%BC%EC%8A%A4" title="아카자 나오야스">아카자 나오야스</a> 등을 조력으로 서군을 배신하게 만든 공으로, 전후에 이에야스로부터 이요 이마바리 20만 석으로 가증 이봉되었다.</p>
<h3><span class="mw-headline" id=".EC.97.90.EB.8F.84_.EC.8B.9C.EB.8C.80">에도 시대</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%8F%84%EB%8F%84_%EB%8B%A4%EC%B9%B4%ED%86%A0%EB%9D%BC&amp;action=edit&amp;section=5" title="부분 편집: 에도 시대">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>그 후 다카토라는 <a href="/wiki/%EC%97%90%EB%8F%84_%EB%A7%89%EB%B6%80" title="에도 막부">에도 막부</a>의 중신이 되었다. <a href="/wiki/%EC%97%90%EB%8F%84_%EC%84%B1" title="에도 성">에도 성</a> 개축 등에도 힘썼기 때문에 게이초 13년(<a href="/wiki/1608%EB%85%84" title="1608년">1608년</a>) <a class="new" href="/w/index.php?title=%EC%9D%B4%EC%B9%B4_%EA%B5%AD&amp;action=edit&amp;redlink=1" title="이카 국 (없는 문서)">이카</a> 일국 및 <a href="/wiki/%EC%9D%B4%EC%84%B8_%EA%B5%AD" title="이세 국">이세</a> 8개 군 22만 석에 가증 이동된다. 이세 <a href="/wiki/%EC%93%B0_%EB%B2%88" title="쓰 번">쓰 번</a>의 번주가 되었다. 이에야스는 다카토라의 능력과 충의를 높게 평가해서, <a href="/wiki/%EB%8F%84%EC%9E%90%EB%A7%88_%EB%8B%A4%EC%9D%B4%EB%AC%98" title="도자마 다이묘">도자마 다이묘</a>임에 불구하고 대우는 <a href="/wiki/%ED%9B%84%EB%8B%A4%EC%9D%B4_%EB%8B%A4%EC%9D%B4%EB%AC%98" title="후다이 다이묘">후다이 다이묘</a>로 중용한다.</p>
<p>게이초 19년(<a href="/wiki/1614%EB%85%84" title="1614년">1614년</a>) <a href="/wiki/%EC%98%A4%EC%82%AC%EC%B9%B4_%EC%A0%84%ED%88%AC" title="오사카 전투">오사카 겨울 전투</a>에 막부군으로 참전한다. 이듬해 오사카 여름 전투에서도 막부규으로 참전, 스스로 <a href="/wiki/%EA%B0%80%EC%99%80%EC%B9%98_%EA%B5%AD" title="가와치 국">가와치</a> 방면의 선봉으로 지원해, 야오에서 도요토미 측 <a href="/wiki/%EC%A1%B0%EC%86%8C%EC%B9%B4%EB%B2%A0_%EB%AA%A8%EB%A6%AC%EC%B9%98%EC%B9%B4" title="조소카베 모리치카">조소카베 모리치카</a> 부대와 전투를 벌인다(야오 전투). 이 전투에서 조소카베 군의 맹공으로 도도 군은 조소카베 군에 고전했고, 무장 <a class="new" href="/w/index.php?title=%EB%8F%84%EB%8F%84_%EC%9A%94%EC%8B%9C%EC%B9%B4%EC%93%B0&amp;action=edit&amp;redlink=1" title="도도 요시카쓰 (없는 문서)">도도 요시카쓰</a>와 <a class="new" href="/w/index.php?title=%EB%8F%84%EB%8F%84_%EB%8B%A4%EC%B9%B4%EB%85%B8%EB%A6%AC&amp;action=edit&amp;redlink=1" title="도도 다카노리 (없는 문서)">도도 다카노리</a>를 비롯 600여명의 사상자가 나왔다. 하지만 전투에서는 승리해 그 공적으로 전후 32만 석이 가증되어 동년 윤6월 종사위하로 위계되었다.</p>
<p>이에야스 임종 직전에 침상 곁에 있는 것이 허락되었다. 이에야스 사후 쇼군 <a href="/wiki/%EB%8F%84%EC%BF%A0%EA%B0%80%EC%99%80_%ED%9E%88%EB%8D%B0%ED%83%80%EB%8B%A4" title="도쿠가와 히데타다">히데타다</a>를 섬겼고, <a href="/wiki/%EA%B2%90%EB%82%98" title="겐나">겐나</a> 6년(<a href="/wiki/1620%EB%85%84" title="1620년">1620년</a>) <a href="/wiki/%EB%8F%84%EC%BF%A0%EA%B0%80%EC%99%80_%ED%9E%88%EB%8D%B0%ED%83%80%EB%8B%A4" title="도쿠가와 히데타다">도쿠가와 히데타다</a>의 딸인 <a href="/wiki/%EB%8F%84%EC%BF%A0%EA%B0%80%EC%99%80_%EB%A7%88%EC%82%AC%EC%BD%94" title="도쿠가와 마사코">마사코</a>가 조정으로 입궁할 때 스스로 지원해 선도역을 맡았다. 궁궐에 들어가, 마사코의 궁내 입궁을 반대하는 귀족들에게 「마사코 님이 입궁하지 못한다면 책임을 지고 어소(御所)에서 할복하겠다.」라고 말하며 강력히 밀어붙였다고 한다.</p>
<h3><span class="mw-headline" id=".EB.A7.90.EB.85.84_.EB.B0.8F_.EC.82.AC.ED.9B.84">말년 및 사후</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%8F%84%EB%8F%84_%EB%8B%A4%EC%B9%B4%ED%86%A0%EB%9D%BC&amp;action=edit&amp;section=6" title="부분 편집: 말년 및 사후">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>말년에는 눈병을 앓아 실명했으며, <a href="/wiki/%EA%B0%84%EC%97%90%EC%9D%B4" title="간에이">간에이</a> 7년 (<a href="/wiki/1630%EB%85%84" title="1630년">1630년</a> <a href="/wiki/%EC%9D%8C%EB%A0%A5_10%EC%9B%94_5%EC%9D%BC" title="음력 10월 5일">음력 10월 5일</a>)에 죽었다. 향년 75세. 후에 아들 <a class="mw-disambig" href="/wiki/%EB%8F%84%EB%8F%84_%EB%8B%A4%EC%B9%B4%EC%93%B0%EA%B5%AC" title="도도 다카쓰구">다카쓰구</a>가 대를 이었다.</p>
<h2><span class="mw-headline" id=".EC.9D.B8.EB.AC.BC">인물</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%8F%84%EB%8F%84_%EB%8B%A4%EC%B9%B4%ED%86%A0%EB%9D%BC&amp;action=edit&amp;section=7" title="부분 편집: 인물">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<ul>
<li>6척 2치(약 190cm)의 키를 자랑하는 거한이었다고도 한다.</li>
<li>다카토라는 몇 번이나 주군을 바꿔, '변절자' 혹은 '권력의 앞잡이'라는 소리가 많아서, 역사 소설등에서 부정적으로 묘사되는 경우가 많다. 사실이기는 하지만, 에도 시대에 유교의 가르침을 받은 무사가 나타나기 이전 일본에서는 가신은 자신의 일한 만큼 은상을 받고, 장래성이 있는 주군을 스스로 선택하는 것이 당연했다<sup class="reference" id="cite_ref-1"><a href="#cite_note-1">[1]</a></sup>. 그러나 <a href="/wiki/%EC%86%8C_%EC%9A%94%EC%8B%9C%ED%86%A0%EC%8B%9C" title="소 요시토시">소 요시토시</a>의 경우도 자신의 세력에 누를 끼치지 않기 위해 어쩔 수 없이 실시한 이혼 한 번이 그렇게나 변절자 취급을 받던 관점과 같은 기준으로 판단했기 때문에 도도 다카토라가 변절자 취급을 당한 것일 수 있다.</li>
<li>무용뿐만 아니라, 쓰 번의 번정에 기반을 쌓았고 내정수완 및 문학과 다도에도 뛰어난 문화인이었다는 설도 있다.</li>
<li>『항상 죽기를 각오하면 마음의 동요가 없는 법이다.』라는 말을 남겼다고 전해진다.</li>
</ul>
<h2><span class="mw-headline" id=".EC.9D.BC.ED.99.94">일화</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%8F%84%EB%8F%84_%EB%8B%A4%EC%B9%B4%ED%86%A0%EB%9D%BC&amp;action=edit&amp;section=8" title="부분 편집: 일화">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<ul>
<li>고단<sup class="reference" id="cite_ref-2"><a href="#cite_note-2">[2]</a></sup>, 로쿄쿠<sup class="reference" id="cite_ref-3"><a href="#cite_note-3">[3]</a></sup>의 「도도 다카토라, 출세의 떡」에서는 아쓰지 씨 슬하에서 떠나 낭인생활을 하는 젊은 시절 다카토라가 <a href="/wiki/%EB%AF%B8%EC%B9%B4%EC%99%80_%EA%B5%AD" title="미카와 국">미카와 국</a> 요시다주쿠(현재 <a href="/wiki/%EC%95%84%EC%9D%B4%EC%B9%98_%ED%98%84" title="아이치 현">아이치 현</a> <a href="/wiki/%EB%8F%84%EC%9A%94%ED%95%98%EC%8B%9C_%EC%8B%9C" title="도요하시 시">도요하시 시</a>)까지 가서, 요시다야 라는 떡집에서 미카와 떡을 무전취식했다. 주인 요시다야 히코베에게 고향에 돌아가 부모님을 모셔야 한다고 말하고 노잣돈까지 받았다고 한다. 때마침 요시다야의 부인이 오미로 외출했었다고 한다. 나중에 다이묘로 출세한 다카토라가 <a class="mw-redirect" href="/wiki/%EC%82%B0%ED%82%A8%EC%BD%94%ED%83%80%EC%9D%B4" title="산킨코타이">산킨코타이</a><sup class="reference" id="cite_ref-4"><a href="#cite_note-4">[4]</a></sup> 할 때 들러 떡값을 주었다는 이야기가 전해지고 있다.</li>
<li>어느날, 2사람의 가신(여색을 좋아하는 가신과 노름을 좋아하는 가신) 사이에 다툼이 벌어져, 그것을 다카토라가 중재를 하게 되었다. 이 때, 다카토라는 여색을 좋아하는 가신을 추방하고 노름을 좋아하는 가신은 아직 전망이 있다고 생각해 그대로 두었다고 한다. 노름은 계산이 빨라야 되기 때문인 것 같다.</li>
<li>"무사란 주군을 7번 바꾸지 않는다면, 무사라고 말할 수 없다."라는 발언했다고 한다.</li>
</ul>
<h2><span class="mw-headline" id=".EC.8B.A4.EB.A0.A5.EC.A3.BC.EC.9D.98.EC.9E.90">실력주의자</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%8F%84%EB%8F%84_%EB%8B%A4%EC%B9%B4%ED%86%A0%EB%9D%BC&amp;action=edit&amp;section=9" title="부분 편집: 실력주의자">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<ul>
<li>다카토라는 소위 실력주의자다. 어디까지나 혈연이 좋은 것도 아니었다. 그와 관계없이 그는 자기 실력만으로 살아 남아왔다. 오다 노부즈미를 모시고 있을 때 큰 공적을 올렸지만, 노부즈미는 다카토라를 싫어해서 가증하지 않았다. 이 때문에, 다카토라는 자신의 영지를 버리고 낭인이 되었다가 후에 하시바 히데나가를 모셨다고 전해진다.</li>
<li>히데요시 사후, 도요토미 씨의 은공을 받은 다이묘로 있으면서, 재빨리 <a href="/wiki/%EB%8F%84%EC%BF%A0%EA%B0%80%EC%99%80_%EC%9D%B4%EC%97%90%EC%95%BC%EC%8A%A4" title="도쿠가와 이에야스">도쿠가와 이에야스</a>에게 접근한 것에 여러 다이묘가 힐난했다. 확연히 충의로 중시하는 무사의 풍토에서는 결코 칭찬받을 수 없는 일이다. 이에 다카토라가 말하기를 「자기 입장이 명확하지 않는 자야 말로, 만일의 경우 무슨 일이 생기면 제일 먼저 의지할 곳이 없게 된다.」라고 사서에 기록되어 있다. 다카토라는 도쿠가와 히데나가를 섬길 시에는 충실한 가신이었고, <a class="new" href="/w/index.php?title=%EC%8B%9C%EC%BD%94%EC%BF%A0_%EC%A0%95%EB%B2%8C&amp;action=edit&amp;redlink=1" title="시코쿠 정벌 (없는 문서)">시코쿠 정벌</a> 때에는 히데나가를 따라 많은 공을 세웠다.</li>
<li>이에야스는 오사카 진에서 공을 올린 다카토라를 은상하고, 「국가에 무슨 일이 생기면, 가장 먼저 다카토라의 손을 빌리겠다.」라고 말했다고 한다. 도쿠가와가의 가신 대부분은 주군을 여러차례 바꾼 다카토라를 그다지 좋아하지 않았지만, 이에야스는 그의 실력을 인정한 것 같다. 오사카 여름 진에서 다카토라는 선봉으로 나서 그 충성심을 보였다. 이에 이에야스는 그것을 인정하고 다카토라에 대한 신뢰가 깊어 졌다고 한다. 그리고, 이에야스 임종 시, 도자마 다이묘로써는 유일하게 다카토라만이 침상 곁에 있는 것을 허락받았다고 한다.</li>
<li>단, <a href="/wiki/%EB%A7%89%EB%A7%90" title="막말">막말</a>의 <a href="/wiki/%EB%8F%84%EB%B0%94%C2%B7%ED%9B%84%EC%8B%9C%EB%AF%B8_%EC%A0%84%ED%88%AC" title="도바·후시미 전투">도바·후시미 전투</a>에서 막부군의 열세를 깨닫자, 곧바로 관군을 배신해, 막부 측에 포격을 가한 것은 도도 가문였다고 기록되어 있다. 이때 막부군에서는 「역시 번조의 가르침이군」 이라고 비아냥거렸다고 한다.</li>
<li>본령 쓰 번 외에 막부의 명에 의해 여식의 결혼으로 사돈 관계로 있는 <a href="/wiki/%EC%95%84%EC%9D%B4%EC%A6%88_%EB%B2%88" title="아이즈 번">아이즈 번</a>의 가모 씨와 <a href="/wiki/%EB%8B%A4%EC%B9%B4%EB%A7%88%EC%93%B0_%EB%B2%88" title="다카마쓰 번">다카마쓰 번</a>의 이코마 씨, 그리고, <a href="/wiki/%EA%B0%80%ED%86%A0_%EA%B8%B0%EC%9A%94%EB%A7%88%EC%82%AC" title="가토 기요마사">가토 기요마사</a> 사후, <a href="/wiki/%EA%B5%AC%EB%A7%88%EB%AA%A8%ED%86%A0_%EB%B2%88" title="구마모토 번">구마모토 번</a>의 정무까지 맡아 가신단의 대립을 조정하고, 160여만 석을 통치했다. 하지만 다카토라의 존재 하나만으로 간신히 유지되었다고 말할 수 있고, 다카토라 사후, 전부 <a href="/wiki/%EA%B0%9C%EC%97%AD" title="개역">개역</a><sup class="reference" id="cite_ref-5"><a href="#cite_note-5">[5]</a></sup>되었다고 한다.</li>
</ul>
