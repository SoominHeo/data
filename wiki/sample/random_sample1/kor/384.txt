<h2><span class="mw-headline" id="2008.EB.85.84_7.EC.9B.94_31.EC.9D.BC"><a href="/wiki/2008%EB%85%84" title="2008년">2008년</a> <a href="/wiki/7%EC%9B%94_31%EC%9D%BC" title="7월 31일">7월 31일</a></span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=2008%EB%85%84_7%EC%9B%94&amp;action=edit&amp;section=1" title="부분 편집: 2008년 7월 31일">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<ul>
<li><a href="/wiki/%EB%8C%80%ED%95%9C%EB%AF%BC%EA%B5%AD" title="대한민국">대한민국</a> <a class="mw-redirect" href="/wiki/2008%EB%85%84_%EC%84%9C%EC%9A%B8%EC%8B%9C_%EA%B5%90%EC%9C%A1%EA%B0%90_%EC%84%A0%EA%B1%B0" title="2008년 서울시 교육감 선거">서울시 교육감 선거</a>에서 <a href="/wiki/%EA%B3%B5%EC%A0%95%ED%83%9D" title="공정택">공정택</a> 현 서울시교육감이 2위와 근소한 표차로 당선되었다.</li>
<li><a href="/wiki/%EB%8C%80%ED%95%9C%EB%AF%BC%EA%B5%AD" title="대한민국">대한민국</a>의 <a href="/wiki/%EC%86%8C%EC%84%A4%EA%B0%80" title="소설가">소설가</a> <a href="/wiki/%EC%9D%B4%EC%B2%AD%EC%A4%80" title="이청준">이청준</a>이 지병인 <a href="/wiki/%ED%8F%90%EC%95%94" title="폐암">폐암</a>으로 타계했다.</li>
<li><a href="/wiki/%EC%A7%90%EB%B0%94%EB%B8%8C%EC%9B%A8" title="짐바브웨">짐바브웨</a>에서 100억 <a href="/wiki/%EC%A7%90%EB%B0%94%EB%B8%8C%EC%9B%A8_%EB%8B%AC%EB%9F%AC" title="짐바브웨 달러">짐바브웨 달러</a>를 1 짐바브웨 달러로 바꾸는 <a class="new" href="/w/index.php?title=%EB%A6%AC%EB%94%94%EB%85%B8%EB%AF%B8%EB%84%A4%EC%9D%B4%EC%85%98&amp;action=edit&amp;redlink=1" title="리디노미네이션 (없는 문서)">리디노미네이션</a>을 실시했다.</li>
<li><a href="/wiki/%EB%AF%B8%EA%B5%AD" title="미국">미국</a>의 지명위원회(BGN)가 <a href="/wiki/%EC%A1%B0%EC%A7%80_W._%EB%B6%80%EC%8B%9C" title="조지 W. 부시">조지 W. 부시</a> 대통령의 지시로 독도의 영유권 표기를 "미지정국가"에서 다시 "<a href="/wiki/%EB%8C%80%ED%95%9C%EB%AF%BC%EA%B5%AD" title="대한민국">대한민국</a>"(South Korea)으로 바꾸었다.</li>
</ul>

<h2><span class="mw-headline" id="2008.EB.85.84_7.EC.9B.94_6.EC.9D.BC"><a href="/wiki/2008%EB%85%84" title="2008년">2008년</a> <a href="/wiki/7%EC%9B%94_6%EC%9D%BC" title="7월 6일">7월 6일</a></span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=2008%EB%85%84_7%EC%9B%94&amp;action=edit&amp;section=18" title="부분 편집: 2008년 7월 6일">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<ul>
<li><a href="/wiki/%EB%9F%B0%EB%8D%98" title="런던">런던</a> 마담 투소 <a class="mw-redirect" href="/wiki/%EB%B0%80%EB%9E%8D%EC%9D%B8%ED%98%95" title="밀랍인형">밀랍인형</a> 박물관이 세계에서 7번째 분관을 <a href="/wiki/%EB%B2%A0%EB%A5%BC%EB%A6%B0" title="베를린">베를린</a>에 개관했다. <a class="mw-redirect" href="/wiki/%EC%95%84%EC%9D%B8%EC%8A%88%ED%83%80%EC%9D%B8" title="아인슈타인">아인슈타인</a>, <a class="mw-redirect" href="/wiki/%EB%B2%A0%ED%86%A0%EB%B2%A4" title="베토벤">베토벤</a> 등 75명의 밀납인형을 전시했지만 40대 남성이 <a class="mw-redirect" href="/wiki/%EB%82%98%EC%B9%98" title="나치">나치</a>의 <a href="/wiki/%EC%95%84%EB%8F%8C%ED%94%84_%ED%9E%88%ED%8B%80%EB%9F%AC" title="아돌프 히틀러">히틀러</a>의 전신 밀랍인형의 목을 잘랐고 이 과정에서 직원 한 명이 가벼운 상처를 입었다.</li>
<li><a class="mw-disambig" href="/wiki/%ED%86%B5%ED%95%A9%EB%AF%BC%EC%A3%BC%EB%8B%B9" title="통합민주당">통합민주당</a>의 제1차 전당대회에서 새 대표에 <a href="/wiki/%EC%A0%95%EC%84%B8%EA%B7%A0" title="정세균">정세균</a> 의원이, 5명의 선출직 최고위원에 <a href="/wiki/%EC%86%A1%EC%98%81%EA%B8%B8_(%EC%A0%95%EC%B9%98%EC%9D%B8)" title="송영길 (정치인)">송영길</a>, <a class="mw-redirect" href="/wiki/%EA%B9%80%EB%AF%BC%EC%84%9D_(1964%EB%85%84)" title="김민석 (1964년)">김민석</a>, <a href="/wiki/%EB%B0%95%EC%A3%BC%EC%84%A0" title="박주선">박주선</a>, <a href="/wiki/%EC%95%88%ED%9D%AC%EC%A0%95" title="안희정">안희정</a>, <a href="/wiki/%EA%B9%80%EC%A7%84%ED%91%9C_(%EC%A0%95%EC%B9%98%EC%9D%B8)" title="김진표 (정치인)">김진표</a> 후보가 선출되었다.</li>
</ul>
<h2><span class="mw-headline" id="2008.EB.85.84_7.EC.9B.94_5.EC.9D.BC"><a href="/wiki/2008%EB%85%84" title="2008년">2008년</a> <a href="/wiki/7%EC%9B%94_5%EC%9D%BC" title="7월 5일">7월 5일</a></span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=2008%EB%85%84_7%EC%9B%94&amp;action=edit&amp;section=19" title="부분 편집: 2008년 7월 5일">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<ul>
<li><a href="/wiki/6%EC%9B%94_27%EC%9D%BC" title="6월 27일">6월 27일</a> 실시된 <a href="/wiki/%EC%A7%90%EB%B0%94%EB%B8%8C%EC%9B%A8" title="짐바브웨">짐바브웨</a>의 결선투표가 부정선거였다는 것을 증명하는 한 교도관이 비밀리에 촬영한 필름이 공개되었다.</li>
<li>리히터 규모 7.5에 이르는 강진이 오전 10시 19분 <a class="mw-redirect" href="/wiki/%EC%98%A4%ED%98%B8%EC%B8%A0%ED%81%AC%ED%95%B4" title="오호츠크해">오호츠크해</a>에서 발생했다고 <a href="/wiki/%EB%AF%B8%EA%B5%AD" title="미국">미국</a>과 <a href="/wiki/%ED%99%8D%EC%BD%A9" title="홍콩">홍콩</a>의 관측 당국이 각각 밝혔다.</li>
</ul>
<h2><span class="mw-headline" id="2008.EB.85.84_7.EC.9B.94_3.EC.9D.BC"><a href="/wiki/2008%EB%85%84" title="2008년">2008년</a> <a href="/wiki/7%EC%9B%94_3%EC%9D%BC" title="7월 3일">7월 3일</a></span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=2008%EB%85%84_7%EC%9B%94&amp;action=edit&amp;section=20" title="부분 편집: 2008년 7월 3일">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<ul>
<li><a class="mw-redirect" href="/wiki/%ED%95%9C%EB%82%98%EB%9D%BC%EB%8B%B9" title="한나라당">한나라당</a>의 제10차 전당대회에서 새 대표최고위원에 <a href="/wiki/%EB%B0%95%ED%9D%AC%ED%83%9C" title="박희태">박희태</a> 후보가, 4명의 선출직 최고위원에 <a href="/wiki/%EC%A0%95%EB%AA%BD%EC%A4%80" title="정몽준">정몽준</a>, <a href="/wiki/%ED%97%88%ED%83%9C%EC%97%B4" title="허태열">허태열</a>, <a href="/wiki/%EA%B3%B5%EC%84%B1%EC%A7%84" title="공성진">공성진</a>, <a href="/wiki/%EB%B0%95%EC%88%9C%EC%9E%90" title="박순자">박순자</a> 의원이 선출되었다.</li>
</ul>
<h2><span class="mw-headline" id="2008.EB.85.84_7.EC.9B.94_1.EC.9D.BC"><a href="/wiki/2008%EB%85%84" title="2008년">2008년</a> <a href="/wiki/7%EC%9B%94_1%EC%9D%BC" title="7월 1일">7월 1일</a></span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=2008%EB%85%84_7%EC%9B%94&amp;action=edit&amp;section=21" title="부분 편집: 2008년 7월 1일">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<ul>
<li>이날 밤 10시 경, 여의도에 있는 <a class="mw-disambig" href="/wiki/%EC%A7%84%EB%B3%B4%EC%8B%A0%EB%8B%B9" title="진보신당">진보신당</a> 당사에 <a class="mw-redirect" href="/wiki/%EC%9C%A1%EA%B5%B0%EC%B2%A9%EB%B3%B4%EB%B6%80%EB%8C%80" title="육군첩보부대">HID</a>(특수임무수행자회)의 사무총장 <a class="new" href="/w/index.php?title=%EC%98%A4%EB%B3%B5%EC%84%AD&amp;action=edit&amp;redlink=1" title="오복섭 (없는 문서)">오복섭</a> 및 다른 몇몇 회원들이 무단으로 침입하여 난동을 일으켰다. 이 과정에서 칼라TV 진행자인 <a href="/wiki/%EC%A7%84%EC%A4%91%EA%B6%8C" title="진중권">진중권</a> 중앙대 겸임교수를 포함한 당원 여러명이 부상을 입었으며, 당 입구에 있는 현판을 비롯한 기물들이 파손되었다. 현재 폭행을 행사한 이들 중 <a class="new" href="/w/index.php?title=%EC%98%A4%EB%B3%B5%EC%84%AD&amp;action=edit&amp;redlink=1" title="오복섭 (없는 문서)">오복섭</a> 사무총장 등 2명은 구속 수감되었다.</li>
<li><a href="/wiki/6%EC%9B%94_17%EC%9D%BC" title="6월 17일">6월 17일</a>에 실종됐던 <a class="mw-redirect" href="/wiki/%EA%B0%95%ED%99%94%EB%8F%84_%EB%AA%A8%EB%85%80_%EC%8B%A4%EC%A2%85%EC%82%AC%EA%B1%B4" title="강화도 모녀 실종사건">강화 두 모녀</a> 윤복희와 김선영 양이 숨진 채 발견되었다.</li>
</ul>
