<h1 class="firstHeading" id="firstHeading" lang="ko">도리스 데이</h1>

<p><b>도리스 데이</b>(Doris Day, <a href="/wiki/1922%EB%85%84" title="1922년">1922년</a> <a href="/wiki/4%EC%9B%94_2%EC%9D%BC" title="4월 2일">4월 2일</a>~)는 <a href="/wiki/%EB%AF%B8%EA%B5%AD" title="미국">미국</a>의 <a href="/wiki/%EA%B0%80%EC%88%98" title="가수">가수</a>이다. <a href="/wiki/%EC%98%A4%ED%95%98%EC%9D%B4%EC%98%A4_%EC%A3%BC" title="오하이오 주">오하이오 주</a> <a href="/wiki/%EC%8B%A0%EC%8B%9C%EB%82%B4%ED%8B%B0" title="신시내티">신시내티</a>에서 태어났다.</p>
<p><a href="/wiki/%EC%95%84%EB%B2%84%EC%A7%80" title="아버지">부친</a>이 <a href="/wiki/%EC%9D%8C%EC%95%85%EA%B0%80" title="음악가">음악가</a>였기 때문에 어릴 적부터 음악을 배웠으나, 댄서를 동경하여 <a href="/wiki/%EC%B6%A4" title="춤">춤</a> 연습에 열중했다. 하지만 다리를 다쳐 그녀의 희망은 좌절되었고 결국 <a href="/wiki/%EC%84%B1%EC%95%85" title="성악">성악</a>에 전념하였다. 타고난 끼를 발휘하여 방송국의 오디션에 합격한 후 가수경력을 시작하였다.</p>
<p><a class="new" href="/w/index.php?title=%EB%B0%A5_%ED%81%AC%EB%A1%9C%EC%8A%A4%EB%B9%84&amp;action=edit&amp;redlink=1" title="밥 크로스비 (없는 문서)">밥 크로스비</a>(Bob Crosby)와 <a class="new" href="/w/index.php?title=%ED%94%84%EB%A0%88%EB%93%9C_%EC%9B%8C%EB%A7%81&amp;action=edit&amp;redlink=1" title="프레드 워링 (없는 문서)">프레드 워링</a>(Fred Waring)의 악단을 거쳐 <a href="/wiki/1941%EB%85%84" title="1941년">1941년</a>에는 <a class="new" href="/w/index.php?title=%EB%A0%88%EC%8A%A4_%EB%B8%8C%EB%9D%BC%EC%9A%B4&amp;action=edit&amp;redlink=1" title="레스 브라운 (없는 문서)">레스 브라운</a>(Les Brown) 악단의 전속 가수가 되었으며, 〈Sentimental Journey〉로 크게 인기를 얻어 실력파 가수로 자리잡았다. 안정된 창법과 선천적인 미성이 매력적이다. 음반으로는 《Day by Day》, 《Show Time》, 《Doris Day's Sentimental Journey》 등 다수가 있다.</p>
<p><a href="/wiki/1948%EB%85%84" title="1948년">1948년</a>에는 <a href="/wiki/%EC%98%81%ED%99%94" title="영화">영화</a>계에 들어가 수많은 <a class="new" href="/w/index.php?title=%EC%9D%8C%EC%95%85_%EC%98%81%ED%99%94&amp;action=edit&amp;redlink=1" title="음악 영화 (없는 문서)">음악 영화</a>에 출연하여, <a href="/wiki/%EC%95%A8%ED%94%84%EB%A6%AC%EB%93%9C_%ED%9E%88%EC%B9%98%EC%BD%95" title="앨프리드 히치콕">앨프리드 히치콕</a> 감독의 《나는 비밀을 알고 있다》 등을 통해 연기파 <a href="/wiki/%EB%B0%B0%EC%9A%B0" title="배우">배우</a>로서의 명성을 떨쳤다.</p>
