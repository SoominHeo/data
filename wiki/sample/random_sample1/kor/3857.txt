<h1 class="firstHeading" id="firstHeading" lang="ko">도모에</h1>
<div class="mw-body-content" id="bodyContent">
<div id="siteSub">위키백과, 우리 모두의 백과사전.</div>
<div id="contentSub"></div>
<div class="mw-jump" id="jump-to-nav">
					이동:					<a href="#mw-head">둘러보기</a>, 					<a href="#p-search">검색</a>
</div>
<div class="mw-content-ltr" dir="ltr" id="mw-content-text" lang="ko"><script>function mfTempOpenSection(id){var block=document.getElementById("mf-section-"+id);block.className+=" open-block";block.previousSibling.className+=" open-block";}</script><p><b>토모에</b>(<span style="font-size: smaller;"><a href="/wiki/%EC%9D%BC%EB%B3%B8%EC%96%B4" title="일본어">일본어</a>:</span> <span lang="ja"><span lang="ja"><ruby><rb>鞆絵</rb> <rp>(</rp><rt>ともえ</rt><rp>)</rp></ruby></span></span> <small>토모에<sup>[<a href="/wiki/%EC%9C%84%ED%82%A4%EB%B0%B1%EA%B3%BC:%EC%9D%BC%EB%B3%B8%EC%96%B4%EC%9D%98_%ED%95%9C%EA%B8%80_%ED%91%9C%EA%B8%B0" title="위키백과:일본어의 한글 표기">*</a>]</sup></small>)는 <a href="/wiki/%EC%9D%BC%EB%B3%B8" title="일본">일본</a>의 전통적인 <a href="/wiki/%EC%9E%A5%EC%8B%9D" title="장식">장식</a>으로 <a class="mw-redirect" href="/wiki/%EA%B3%A1%EC%98%A5" title="곡옥">곡옥</a>의 <a href="/wiki/%EC%86%8C%EC%9A%A9%EB%8F%8C%EC%9D%B4" title="소용돌이">소용돌이</a> 모양을 띄고 있다. 토모에의 기원은 불확실하나 일부 사람들은 토모에(鞆絵) 혹은 토모(鞆)를 그리는 것이 궁수들이 사용하는 팔 보호대를 의미한다고 주장하지만, 다른 사람들은 토모에가 <a class="mw-redirect" href="/wiki/%EA%B3%A1%EC%98%A5" title="곡옥">곡옥</a>을 양식화한거라고 주장한다.<sup class="reference" id="cite_ref-Nihon_no_Kamon_Taizen_1-0"><a href="#cite_note-Nihon_no_Kamon_Taizen-1">[1]</a></sup></p>
<p>이것은 일본의 <a href="/wiki/%EA%B0%80%EB%AA%AC" title="가몬">가몬</a>(家紋-가문의 문장)에서 공통적으로 보이는 문양이며, 회사 <a href="/wiki/%EB%A1%9C%EA%B3%A0" title="로고">로고</a>에도 종종 쓰인다. 특히 삼중으로 된 토모에는 <span style="font-size: smaller;"><a href="/wiki/%EC%9D%BC%EB%B3%B8%EC%96%B4" title="일본어">일본어</a>:</span> <span lang="ja"><span lang="ja"><ruby><rb>三つ巴</rb> <rp>(</rp><rt>みつともゑ</rt><rp>)</rp></ruby></span></span> <small>미츠토모에<sup>[<a href="/wiki/%EC%9C%84%ED%82%A4%EB%B0%B1%EA%B3%BC:%EC%9D%BC%EB%B3%B8%EC%96%B4%EC%9D%98_%ED%95%9C%EA%B8%80_%ED%91%9C%EA%B8%B0" title="위키백과:일본어의 한글 표기">*</a>]</sup></small>라 알려졌다. 일부 사람들은 미츠토모에를 <a href="/wiki/%EC%8B%A0%ED%86%A0" title="신토">신토</a>의 중요한 3 요소로 대표되는 <a href="/wiki/%ED%95%98%EB%8A%98" title="하늘">하늘</a>, <a href="/wiki/%EB%95%85" title="땅">땅</a>, <a href="/wiki/%EC%82%AC%EB%9E%8C" title="사람">사람</a>을 의미한다고 보고 있다. 근본적으로, 이는 <a href="/wiki/%EC%8B%A0%ED%86%A0" title="신토">신토</a>에서 전쟁의 신으로 알려진 <a class="new" href="/w/index.php?title=%ED%95%98%EC%B9%98%EB%A7%8C&amp;action=edit&amp;redlink=1" title="하치만 (없는 문서)">하치만</a>과 연관 되어있다. 이러한 사실을 통해 미츠토모에는 <a href="/wiki/%EC%82%AC%EB%AC%B4%EB%9D%BC%EC%9D%B4" title="사무라이">사무라이</a>의 전통적인 문양으로 적용 되어왔다. 미츠토모에의 변형인 <a class="new" href="/w/index.php?title=%ED%9E%88%EB%8B%A4%EB%A6%AC_%EA%B3%A0%EB%AA%AC&amp;action=edit&amp;redlink=1" title="히다리 고몬 (없는 문서)">히다리 고몬</a>은 <a class="mw-redirect" href="/wiki/%EB%A5%98%ED%81%90_%EC%99%95%EA%B5%AD" title="류큐 왕국">류큐 왕국</a>의 전통 문양으로, <a class="mw-redirect" href="/wiki/%EC%9D%BC%EB%B3%B8_%EB%B6%88%EA%B5%90" title="일본 불교">일본 불교</a> <a href="/wiki/%EC%A7%84%EC%96%B8%EC%A2%85" title="진언종">진언종</a>의 성지인 <a href="/wiki/%EA%B3%A0%EC%95%BC_%EC%82%B0" title="고야 산">고야 산</a>에서 <a class="new" href="/w/index.php?title=%ED%9E%88%EB%8B%A4%EB%A6%AC_%EA%B3%A0%EB%AA%AC&amp;action=edit&amp;redlink=1" title="히다리 고몬 (없는 문서)">히다리 고몬</a>은 <a href="/wiki/%EC%9C%A4%ED%9A%8C" title="윤회">윤회</a> 사상을 대표하는 시각적 이미지로 활용 되어왔다.</p>
<p>이중으로 된 토모에는 거의 <a href="/wiki/%EC%A4%91%EA%B5%AD" title="중국">중국</a>의 <a class="mw-disambig" href="/wiki/%ED%83%80%EC%9D%B4%EC%A7%80" title="타이지">타이지</a>(太極-태극)에서 기원한 무늬라고 알려져왔다. 반면에 삼중으로 된 미츠토모에는 <a href="/wiki/%ED%95%9C%EA%B5%AD" title="한국">한국</a>의 <a class="mw-redirect" href="/wiki/%EC%82%BC%ED%83%9C%EA%B7%B9" title="삼태극">삼태극</a> 문양과 흡사하다. 또한 사중으로 된 토모에는 <a class="mw-redirect" href="/wiki/%EC%9E%90%EC%9D%B4%EB%8B%88%EC%A6%98" title="자이니즘">자이니즘</a>과 <a href="/wiki/%ED%9E%8C%EB%91%90%EA%B5%90" title="힌두교">힌두교</a>에서 <a href="/wiki/%EC%A2%85%EA%B5%90" title="종교">종교</a>적으로 중요하게 쓰이는 <a class="mw-redirect" href="/wiki/%EC%8A%A4%EC%99%80%EC%8A%A4%ED%8B%B0%EC%B9%B4" title="스와스티카">스와스티카</a>와 형태가 비슷하다. 한편 이와 비슷한 문양을 <a href="/wiki/%EC%9C%A0%EB%9F%BD" title="유럽">유럽</a>에서도 찾아볼 수 있는데 <a class="mw-redirect" href="/wiki/%EB%B0%94%EC%8A%A4%ED%81%AC%EC%A1%B1" title="바스크족">바스크족</a>의 <a class="new" href="/w/index.php?title=%EB%A1%9C%EB%B8%8C%EB%A5%B4&amp;action=edit&amp;redlink=1" title="로브르 (없는 문서)">로브르</a>(바스크 십자가)나 <a href="/wiki/%EC%BC%88%ED%8A%B8%EC%A1%B1" title="켈트족">켈트족</a>의 <a class="new" href="/w/index.php?title=%ED%8A%B8%EB%A6%AC%EC%8A%A4%EC%BC%80%EB%A5%B4&amp;action=edit&amp;redlink=1" title="트리스케르 (없는 문서)">트리스케르</a>와 유사한 모습을 띄고 있다.</p>
