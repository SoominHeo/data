<p><b>2008년 AFC 챌린지컵</b>(<span style="font-size: smaller;"><a href="/wiki/%EC%98%81%EC%96%B4" title="영어">영어</a>:</span> <span lang="en" xml:lang="en">2008 AFC Challenge Cup</span>)은 <a href="/wiki/%EC%95%84%EC%8B%9C%EC%95%84_%EC%B6%95%EA%B5%AC_%EC%97%B0%EB%A7%B9" title="아시아 축구 연맹">아시아 축구 연맹</a>에서 주관하는 대회로, <a href="/wiki/2008%EB%85%84" title="2008년">2008년</a> <a href="/wiki/7%EC%9B%94_30%EC%9D%BC" title="7월 30일">7월 30일</a>부터 <a href="/wiki/8%EC%9B%94_13%EC%9D%BC" title="8월 13일">8월 13일</a>까지 <a href="/wiki/%EC%9D%B8%EB%8F%84" title="인도">인도</a>의 <a class="mw-disambig" href="/wiki/%ED%95%98%EC%9D%B4%EB%8D%B0%EB%9D%BC%EB%B0%94%EB%93%9C" title="하이데라바드">하이데라바드</a>, <a href="/wiki/%EB%89%B4%EB%8D%B8%EB%A6%AC" title="뉴델리">뉴델리</a>에서 개최되었다. 이번 대회의 우승국인 <a href="/wiki/%EC%9D%B8%EB%8F%84_%EC%B6%95%EA%B5%AC_%EA%B5%AD%EA%B0%80%EB%8C%80%ED%91%9C%ED%8C%80" title="인도 축구 국가대표팀">인도</a>에게는 <a href="/wiki/2011%EB%85%84_AFC_%EC%95%84%EC%8B%9C%EC%95%88%EC%BB%B5" title="2011년 AFC 아시안컵">2011년 AFC 아시안컵</a> 본선진출권이 주어졌다.</p>
