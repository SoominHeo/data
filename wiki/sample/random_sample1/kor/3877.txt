<p><b>도사</b>(Dosa)는 주로 <a href="/wiki/%EC%8C%80" title="쌀">쌀</a> 반죽과 검은 <a class="mw-redirect" href="/wiki/%EB%A0%8C%ED%8B%B8%EC%BD%A9" title="렌틸콩">렌틸콩</a>으로 만들어지는 발효 <a href="/wiki/%ED%81%AC%EB%A0%88%EC%9D%B4%ED%94%84" title="크레이프">크레이프</a>이다. <a href="/wiki/%EC%9D%B8%EB%8F%84" title="인도">인도</a> 남부 <a href="/wiki/%ED%83%80%EB%B0%80%EB%82%98%EB%91%90_%EC%A3%BC" title="타밀나두 주">타밀나두 주</a>, <a href="/wiki/%EC%B9%B4%EB%A5%B4%EB%82%98%ED%83%80%EC%B9%B4_%EC%A3%BC" title="카르나타카 주">카르나타카 주</a>, <a href="/wiki/%ED%85%94%EB%9E%91%EA%B0%80%EB%82%98_%EC%A3%BC" title="텔랑가나 주">텔랑가나 주</a>, <a href="/wiki/%EC%95%88%EB%93%9C%EB%9D%BC%ED%94%84%EB%9D%BC%EB%8D%B0%EC%8B%9C_%EC%A3%BC" title="안드라프라데시 주">안드라프라데시 주</a>, <a href="/wiki/%EC%BC%80%EB%9E%84%EB%9D%BC_%EC%A3%BC" title="케랄라 주">케랄라 주</a>의 주식이며 인도의 다른 지방이나 <a href="/wiki/%EC%8A%A4%EB%A6%AC%EB%9E%91%EC%B9%B4" title="스리랑카">스리랑카</a>, <a href="/wiki/%EB%84%A4%ED%8C%94" title="네팔">네팔</a>, <a href="/wiki/%EB%A7%90%EB%A0%88%EC%9D%B4%EC%8B%9C%EC%95%84" title="말레이시아">말레이시아</a>, <a href="/wiki/%EC%8B%B1%EA%B0%80%ED%8F%AC%EB%A5%B4" title="싱가포르">싱가포르</a>에서도 찾아볼 수 있다.</p>

<h2><span class="mw-headline" id=".EC.97.AD.EC.82.AC">역사</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%8F%84%EC%82%AC_(%EB%B9%B5)&amp;action=edit&amp;section=1" title="부분 편집: 역사">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>도사는 남인도의 요리이다. 한 서적에 따르면 <a href="/wiki/1%EC%84%B8%EA%B8%B0" title="1세기">1세기</a>부터 고대 <a class="mw-disambig" href="/wiki/%ED%83%80%EB%B0%80" title="타밀">타밀</a> 지방에서는 도사가 만들어지기 시작했다고 한다. 전통적인 도사는 현재의 도사보다 부드럽고 두꺼웠다. 현재는 얇고 살짝 딱딱한 것이 특징이다.</p>
<h2><span class="mw-headline" id=".EC.98.81.EC.96.91">영양</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%8F%84%EC%82%AC_(%EB%B9%B5)&amp;action=edit&amp;section=2" title="부분 편집: 영양">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>유명한 아침 식사이자 <a href="/wiki/%EA%B8%B8%EA%B1%B0%EB%A6%AC_%EC%9D%8C%EC%8B%9D" title="길거리 음식">길거리 음식</a>인 도사는 <a href="/wiki/%ED%83%84%EC%88%98%ED%99%94%EB%AC%BC" title="탄수화물">탄수화물</a>이 풍부하다. 또한 <a class="mw-redirect" href="/wiki/%ED%8F%AC%ED%99%94%EC%A7%80%EB%B0%A9" title="포화지방">포화지방</a>과 <a href="/wiki/%EC%84%A4%ED%83%95" title="설탕">설탕</a>을 포함하지 않고 있다. 탄수화물이 풍부하고 지방 함량이 적은 이 영양 구조는 <a href="/wiki/%EB%B0%80%EA%B0%80%EB%A3%A8" title="밀가루">밀가루</a> 대신 <a href="/wiki/%EC%8C%80" title="쌀">쌀</a>로 만들어서이다. 도사의 발효 과정은 <a class="mw-redirect" href="/wiki/%EB%B9%84%ED%83%80%EB%AF%BC_B" title="비타민 B">비타민 B</a>와 <a href="/wiki/%EB%B9%84%ED%83%80%EB%AF%BC_C" title="비타민 C">비타민 C</a> 또한 늘려 준다.</p>
<h2><span class="mw-headline" id=".EC.A4.80.EB.B9.84">준비</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%8F%84%EC%82%AC_(%EB%B9%B5)&amp;action=edit&amp;section=3" title="부분 편집: 준비">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>도사를 준비하기 위해서는 <a href="/wiki/%EC%8C%80" title="쌀">쌀</a>과 렌틸콩이 필요하다. 보통 쌀과 렌틸콩의 혼합물은 갈리기 전 물에 빠트려지며 약 4:1의 비율로 만들어지는 경우가 많다. 반죽이 만들어진 후 발효 과정을 거치기도 한다. 원하는 빵 두께를 만들기 위해서는 다시한번 도사를 물에 빠트린다. 만들어진 도사는 곧 <i>타바</i>라고 불리는 작은 <a href="/wiki/%ED%94%84%EB%9D%BC%EC%9D%B4%ED%8C%AC" title="프라이팬">프라이팬</a>에 부어진다. 기름을 두르고 빵을 구우면 곧 빵 반죽이 퍼지며 빵의 형태를 갖출 것이다. 보통 도사는 뜨거운 상태로 제공되는 경우가 많다. <a href="/wiki/%EC%B2%98%ED%8A%B8%EB%8B%88" title="처트니">처트니</a>와 함께 서빙되기도 한다.</p>
<h2><span class="mw-headline" id=".EC.84.9C.EB.B9.99">서빙</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%8F%84%EC%82%AC_(%EB%B9%B5)&amp;action=edit&amp;section=4" title="부분 편집: 서빙">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>도사는 종종 채소와 함께 서빙되며 소스가 같이 제공되기도 한다. 대체적으로 아래와 같은 음식들이 곁들여지거나 함께 제공된다.</p>
