<p><b>도이나 오펠리아 멜린테</b>(<span style="font-size: smaller;"><a href="/wiki/%EB%A3%A8%EB%A7%88%EB%8B%88%EC%95%84%EC%96%B4" title="루마니아어">루마니아어</a>:</span> <span lang="ro" xml:lang="ro">Doina Ofelia Melinte</span>, <a href="/wiki/1956%EB%85%84" title="1956년">1956년</a> <a href="/wiki/12%EC%9B%94_27%EC%9D%BC" title="12월 27일">12월 27일</a> ~ )는 은퇴한 <a href="/wiki/%EB%A3%A8%EB%A7%88%EB%8B%88%EC%95%84" title="루마니아">루마니아</a>의 <a class="mw-redirect" href="/wiki/%EC%A4%91%EA%B1%B0%EB%A6%AC%EB%8B%AC%EB%A6%AC%EA%B8%B0" title="중거리달리기">중거리달리기</a> 선수이다. <a class="mw-redirect" href="/wiki/1984%EB%85%84_%EB%A1%9C%EC%8A%A4%EC%95%A4%EC%A0%A4%EB%A0%88%EC%8A%A4_%EC%98%AC%EB%A6%BC%ED%94%BD" title="1984년 로스앤젤레스 올림픽">1984년 로스앤젤레스 올림픽</a> 800m 챔피언으로 <a href="/wiki/1990%EB%85%84" title="1990년">1990년</a> 4분 17.14초와 함께 마일 종목의 세계 기록을 깼다.</p>
<h2><span class="mw-headline" id=".EA.B2.BD.EB.A0.A5">경력</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%8F%84%EC%9D%B4%EB%82%98_%EB%A9%9C%EB%A6%B0%ED%85%8C&amp;action=edit&amp;section=1" title="부분 편집: 경력">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p><a href="/wiki/%EB%B0%94%EC%BB%A4%EC%9A%B0" title="바커우">바커우</a>에서 태어난 벨린테는 <a class="mw-redirect" href="/wiki/1980%EB%85%84_%EB%AA%A8%EC%8A%A4%ED%81%AC%EB%B0%94_%EC%98%AC%EB%A6%BC%ED%94%BD" title="1980년 모스크바 올림픽">1980년 모스크바 올림픽</a>에서 돌리나 베슬리우라는 이름으로 나가 800m 준결승전에 도달하였다. <a href="/wiki/1982%EB%85%84" title="1982년">1982년</a> 그녀는 800m 종목에서 세계의 가장 빠른 여성이었으나 <a href="/wiki/%EC%95%84%ED%85%8C%EB%84%A4" title="아테네">아테네</a>에서 열린 유럽 선수권에서 6위를 하였다. 그녀는 또한 <a href="/wiki/1983%EB%85%84_%EC%84%B8%EA%B3%84_%EC%9C%A1%EC%83%81_%EC%84%A0%EC%88%98%EA%B6%8C_%EB%8C%80%ED%9A%8C" title="1983년 세계 육상 선수권 대회">1983년 세계 육상 선수권 대회</a>에서도 800m 6위를 하였다.</p>
<p><a class="mw-redirect" href="/wiki/1984%EB%85%84_%EB%A1%9C%EC%8A%A4%EC%95%A4%EC%A0%A4%EB%A0%88%EC%8A%A4_%EC%98%AC%EB%A6%BC%ED%94%BD" title="1984년 로스앤젤레스 올림픽">1984년 로스앤젤레스 올림픽</a>에서 <a href="/wiki/%EB%AF%B8%EA%B5%AD" title="미국">미국</a>의 <a class="new" href="/w/index.php?title=%ED%82%B4_%EA%B0%A4%EB%9F%AC%EA%B1%B0&amp;action=edit&amp;redlink=1" title="킴 갤러거 (없는 문서)">킴 갤러거</a>와 동료 선수 <a class="new" href="/w/index.php?title=%ED%94%BC%ED%83%80_%EB%A1%9C%EB%B9%88&amp;action=edit&amp;redlink=1" title="피타 로빈 (없는 문서)">피타 로빈</a>을 앞서 800m <a href="/wiki/%EA%B8%88%EB%A9%94%EB%8B%AC" title="금메달">금메달</a>을 획득하였다. 1500m 결승전에도 진출한 벨린테는 <a href="/wiki/%EC%9D%B4%ED%83%88%EB%A6%AC%EC%95%84" title="이탈리아">이탈리아</a>의 <a href="/wiki/%EA%B0%80%EB%B8%8C%EB%A6%AC%EC%97%98%EB%9D%BC_%EB%8F%84%EB%A6%AC%EC%98%A4" title="가브리엘라 도리오">가브리엘라 도리오</a>에 밀리고 동료 선수 <a href="/wiki/%EB%A7%88%EB%A6%AC%EC%8B%9C%EC%B9%B4_%ED%91%B8%EC%9D%B4%EC%BB%A4" title="마리시카 푸이커">마리시카 푸이커</a>를 앞서 <a href="/wiki/%EC%9D%80%EB%A9%94%EB%8B%AC" title="은메달">은메달</a>을 땄다. <a href="/wiki/1986%EB%85%84" title="1986년">1986년</a> 유럽 선수권에서는 <a href="/wiki/%EC%86%8C%EB%A0%A8" title="소련">소련</a>의 <a class="new" href="/w/index.php?title=%EB%9D%BC%EB%B9%8C%EB%A6%AC%EC%95%BC_%EC%95%84%EA%B8%80%EB%A0%88%ED%8B%B0%EB%85%B8%EB%B0%94&amp;action=edit&amp;redlink=1" title="라빌리야 아글레티노바 (없는 문서)">라빌리야 아글레티노바</a>와 <a class="new" href="/w/index.php?title=%ED%83%80%ED%8B%B0%EC%95%BC%EB%82%98_%EC%82%AC%EB%AA%B0%EB%A0%8C%EC%BD%94&amp;action=edit&amp;redlink=1" title="타티야나 사몰렌코 (없는 문서)">타티야나 사몰렌코</a>에 밀려 <a href="/wiki/%EB%8F%99%EB%A9%94%EB%8B%AC" title="동메달">동메달</a>을 땄다. <a href="/wiki/1987%EB%85%84" title="1987년">1987년</a> <a href="/wiki/%EC%9D%B8%EB%94%94%EC%95%A0%EB%82%98%ED%8F%B4%EB%A6%AC%EC%8A%A4" title="인디애나폴리스">인디애나폴리스</a>에서 열린 세계 실내 선수권에서 사몰렌코를 앞서 1500m를 우승하였다. 그해 <a href="/wiki/9%EC%9B%94" title="9월">9월</a> <a href="/wiki/%EB%A1%9C%EB%A7%88" title="로마">로마</a>에서 열린 세계 육상 선수권 대회에서 그 종목의 동메달을 땄다. 원래 그녀는 4위에 왔으나, 3위의 <a class="new" href="/w/index.php?title=%EC%9E%94%EB%93%9C%EB%9D%BC_%EA%B0%80%EC%84%9C&amp;action=edit&amp;redlink=1" title="잔드라 가서 (없는 문서)">잔드라 가서</a>가 실격당하면서 한 단계로 승진되었다.</p>
<p><a class="mw-redirect" href="/wiki/1988%EB%85%84_%EC%84%9C%EC%9A%B8_%EC%98%AC%EB%A6%BC%ED%94%BD" title="1988년 서울 올림픽">1988년 서울 올림픽</a>에서 그녀는 동료 선수 <a href="/wiki/%ED%8C%8C%EC%9A%B8%EB%9D%BC_%EC%9D%B4%EB%B0%98" title="파울라 이반">파울라 이반</a>이 우승한 1500m에서 실망스러운 9위를 하였다. <a href="/wiki/1989%EB%85%84" title="1989년">1989년</a> <a href="/wiki/%EB%B6%80%EB%8B%A4%ED%8E%98%EC%8A%A4%ED%8A%B8" title="부다페스트">부다페스트</a>에서 4분 04.79초의 선수권 기록과 함께 1500m 세계 실내 타이틀을 성공적으로 유지하였다. <a href="/wiki/1990%EB%85%84" title="1990년">1990년</a> <a href="/wiki/2%EC%9B%94" title="2월">2월</a> 이스트 러더퍼드에서 4분 17.14초와 함께 세계 실내 마일 기록을 깼다. 1달 후에 <a href="/wiki/%EA%B8%80%EB%9E%98%EC%8A%A4%EA%B3%A0" title="글래스고">글래스고</a>에서 3번째 유럽 실내 타이틀을 우승하였다. <a href="/wiki/1990%EB%85%84" title="1990년">1990년</a> 유럽 선수권에서 메달 후보 선수로 보이던 그녀는 1500m 6위로 들어왔다. <a href="/wiki/1991%EB%85%84" title="1991년">1991년</a> <a href="/wiki/%EC%84%B8%EB%B9%84%EC%95%BC" title="세비야">세비야</a>에서 열린 세계 실내 선수권과 <a href="/wiki/%EB%8F%84%EC%BF%84" title="도쿄">도쿄</a>에서 열린 세계 육상 선수권 양대회에서 1500m 4위를 하였다.</p>
<p>멜린테는 1500m 결승전에서 자신이 300m가 남은 자리에서 떨어져 나갔던 <a class="mw-redirect" href="/wiki/1992%EB%85%84_%EB%B0%94%EB%A5%B4%EC%85%80%EB%A1%9C%EB%82%98_%EC%98%AC%EB%A6%BC%ED%94%BD" title="1992년 바르셀로나 올림픽">1992년 바르셀로나 올림픽</a>에서 자신의 국제 경력을 끝냈다.</p>
