<div class="mw-content-ltr" dir="ltr" id="mw-content-text" lang="ko"><script>function mfTempOpenSection(id){var block=document.getElementById("mf-section-"+id);block.className+=" open-block";block.previousSibling.className+=" open-block";}</script><p><b>도잉크 더 크라운</b>(Doink The Clown)은 <a href="/wiki/1957%EB%85%84" title="1957년">1957년</a> <a href="/wiki/7%EC%9B%94_27%EC%9D%BC" title="7월 27일">7월 27일</a>~<a href="/wiki/2013%EB%85%84" title="2013년">2013년</a> <a href="/wiki/6%EC%9B%94_28%EC%9D%BC" title="6월 28일">6월 28일</a> <a href="/wiki/%EB%AF%B8%EA%B5%AD" title="미국">미국</a>의 프로레슬링선수다. 1984년 프로레슬링에 입문하였으며 <a href="/wiki/2012%EB%85%84" title="2012년">2012년</a> <a href="/wiki/7%EC%9B%94_2%EC%9D%BC" title="7월 2일">7월 2일</a> <a class="mw-redirect" href="/wiki/WWE_RAW" title="WWE RAW">WWE RAW</a>에서 <a href="/wiki/%ED%9E%88%EC%8A%A4_%EC%8A%AC%EB%A0%88%EC%9D%B4%ED%84%B0" title="히스 슬레이터">히스 슬레이터</a>를 상대로 깜짝 등장하지만, 승리를 거두진 못했다. 항상 <a href="/wiki/%ED%94%BC%EC%97%90%EB%A1%9C" title="피에로">피에로</a> 분장을하고 경기에 출현하는걸로 유명하다. <a href="/wiki/2013%EB%85%84" title="2013년">2013년</a> <a href="/wiki/6%EC%9B%94_28%EC%9D%BC" title="6월 28일">6월 28일</a> <a href="/wiki/%EB%AF%B8%EA%B5%AD" title="미국">미국</a> <a href="/wiki/%ED%85%8D%EC%82%AC%EC%8A%A4_%EC%A3%BC" title="텍사스 주">텍사스 주</a>에서 WWE 일화성 복귀때 입은 뇌진탕으로 자택에서 숨지고 말았다. 피니쉬는 우피 쿠션(시티드 센톤), 스텀프 풀러(싱글 레그 인버티드 보스턴 크랩)로 두가지 피니쉬로 사용을 한다. 키는 182cm 몸무게는 110kg이다.</p>
