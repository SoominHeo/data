<p>《<b>도쿄!</b>》(<span style="font-size: smaller;"><a href="/wiki/%EC%98%81%EC%96%B4" title="영어">영어</a>:</span> <span lang="en" xml:lang="en">Tokyo!</span>)는 <a href="/wiki/%ED%94%84%EB%9E%91%EC%8A%A4" title="프랑스">프랑스</a>, <a href="/wiki/%EC%9D%BC%EB%B3%B8" title="일본">일본</a>, <a href="/wiki/%EB%8F%85%EC%9D%BC" title="독일">독일</a>, <a href="/wiki/%EB%8C%80%ED%95%9C%EB%AF%BC%EA%B5%AD" title="대한민국">대한민국</a>의 합작의 <a href="/wiki/2008%EB%85%84" title="2008년">2008년</a> 제작, 개봉된 <a href="/wiki/%EC%98%B4%EB%8B%88%EB%B2%84%EC%8A%A4_%EC%98%81%ED%99%94" title="옴니버스 영화">옴니버스 영화</a>이다. <a href="/wiki/%EB%AF%B8%EC%85%B8_%EA%B3%B5%EB%93%9C%EB%A6%AC" title="미셸 공드리">미셸 공드리</a> 감독의 〈인테리어 디자인〉 (Interior Design)과 <a class="mw-redirect" href="/wiki/%EB%A0%88%EC%98%A4_%EC%B9%B4%EB%9D%BD%EC%8A%A4" title="레오 카락스">레오 카락스</a> 감독의 〈메르도〉 (Merde), <a href="/wiki/%EB%B4%89%EC%A4%80%ED%98%B8" title="봉준호">봉준호</a> 감독의 〈흔들리는 도쿄〉 (Shaking Tokyo) 세 편으로 구성되어 있으며 모두 <a href="/wiki/%EC%9D%BC%EB%B3%B8" title="일본">일본</a>의 <a href="/wiki/%EB%8F%84%EC%BF%84_%EB%8F%84" title="도쿄 도">도쿄 도</a>가 무대이다. 제61회 칸 국제 영화제 '주목할만한 시선' 부문에 출품되었다.</p>
<p>비터즈 엔드, 꼼 데 시네마, 유러스페이스, 오프 스크린 등 에서 제작하여, <a href="/wiki/%EB%8C%80%ED%95%9C%EB%AF%BC%EA%B5%AD" title="대한민국">대한민국</a>에서는 스폰지이엔티에서 수입해, 싸이더스FNH를 통해 배급되어 개봉되었다.</p>
