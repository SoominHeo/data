<p><b>도쿄 지하철 08계 전동차</b>(<span style="font-size: smaller;"><a href="/wiki/%EC%9D%BC%EB%B3%B8%EC%96%B4" title="일본어">일본어</a>:</span> <span lang="ja" xml:lang="ja">東京地下鉄08系電車</span>)는 <a href="/wiki/2003%EB%85%84" title="2003년">2003년</a>부터 운행하고 있는 <a class="mw-redirect" href="/wiki/%EB%8F%84%EC%BF%84_%EB%A9%94%ED%8A%B8%EB%A1%9C" title="도쿄 메트로">도쿄 메트로</a> (이전 • <a href="/wiki/%EC%A0%9C%EB%8F%84%EA%B3%A0%EC%86%8D%EB%8F%84%EA%B5%90%ED%86%B5%EC%98%81%EB%8B%A8" title="제도고속도교통영단">제도고속도교통영단</a>) <a href="/wiki/%EB%8F%84%EC%BF%84_%EC%A7%80%ED%95%98%EC%B2%A0_%ED%95%9C%EC%A1%B0%EB%AA%AC_%EC%84%A0" title="도쿄 지하철 한조몬 선">한조몬 선</a>의 전동차이다.</p>
<p><a href="/wiki/%EB%8F%84%EC%BF%84_%EC%A7%80%ED%95%98%EC%B2%A0_%EB%8F%84%EC%9E%90%EC%9D%B4_%EC%84%A0" title="도쿄 지하철 도자이 선">도자이 선</a>의 05계를 기반으로 개발되었다.</p>
<p>이 차량은 <a href="/wiki/%EB%8F%84%EC%BF%84_%EA%B8%89%ED%96%89_%EC%A0%84%EC%B2%A0" title="도쿄 급행 전철">도쿄 급행 전철</a> <a href="/wiki/%EB%8F%84%EC%BF%84_%EA%B8%89%ED%96%89_%EC%A0%84%EC%B2%A0" title="도쿄 급행 전철">덴엔토시 선</a>, <a href="/wiki/%EB%8F%84%EB%B6%80_%EC%B2%A0%EB%8F%84" title="도부 철도">도부 철도</a> <a href="/wiki/%EB%8F%84%EB%B6%80_%EC%B2%A0%EB%8F%84" title="도부 철도">이세사키 선</a>과의 <a href="/wiki/%EC%A7%81%EA%B2%B0_%EC%9A%B4%ED%96%89" title="직결 운행">직결 운행</a> 및 2003년 <a href="/wiki/%EC%98%A4%EC%8B%9C%EC%95%84%EA%B2%8C_%EC%97%AD" title="오시아게 역">오시아게</a>~<a href="/wiki/%EC%8A%A4%EC%9D%B4%ED%85%90%EA%B5%AC%EB%A7%88%EC%97%90_%EC%97%AD" title="스이텐구마에 역">스이텐구마에</a> 구간 개통에 따른 증편 목적으로 제작되었다.</p>
<p><br/></p>
<h2><span class="mw-headline" id=".ED.8E.B8.EC.84.B1.2C_.EC.9A.B4.EC.9A.A9">편성, 운용</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%8F%84%EC%BF%84_%EC%A7%80%ED%95%98%EC%B2%A0_08%EA%B3%84_%EC%A0%84%EB%8F%99%EC%B0%A8&amp;action=edit&amp;section=1" title="부분 편집: 편성, 운용">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p><a class="mw-redirect" href="/wiki/%EB%8F%84%EC%BF%84_%EA%B8%89%ED%96%89_%EC%A0%84%EC%B2%A0_%EB%8D%B4%EC%97%94%ED%86%A0%EC%8B%9C_%EC%84%A0" title="도쿄 급행 전철 덴엔토시 선">덴엔토시 선</a> <a href="/wiki/%EC%A3%BC%EC%98%A4%EB%A6%B0%EC%B9%B8_%EC%97%AD" title="주오린칸 역">주오린칸 역</a>에서 시작해, <a href="/wiki/%EB%8F%84%EC%BF%84_%EC%A7%80%ED%95%98%EC%B2%A0_%ED%95%9C%EC%A1%B0%EB%AA%AC_%EC%84%A0" title="도쿄 지하철 한조몬 선">한조몬 선</a>을 거쳐 <a class="mw-redirect" href="/wiki/%EB%8F%84%EB%B6%80_%EC%B2%A0%EB%8F%84_%EC%9D%B4%EC%84%B8%EC%82%AC%ED%82%A4_%EC%84%A0" title="도부 철도 이세사키 선">이세사키 선</a> <a class="mw-disambig" href="/wiki/%EA%B5%AC%ED%82%A4_%EC%97%AD" title="구키 역">구키 역</a>, 혹은 <a class="mw-redirect" href="/wiki/%EB%8F%84%EB%B6%80_%EC%B2%A0%EB%8F%84_%EB%8B%9B%EC%BD%94_%EC%84%A0" title="도부 철도 닛코 선">닛코 선</a>의 <a href="/wiki/%EB%AF%B8%EB%82%98%EB%AF%B8%EC%BF%A0%EB%A6%AC%ED%95%98%EC%8B%9C_%EC%97%AD" title="미나미쿠리하시 역">미나미쿠리하시 역</a>까지 운행한다. 운행거리는 총 98.5km로 일본 지하철로는 최대의 운행거리를 자랑한다. 그리고 6편성만 제작되었으며, 사기누마 차량기지에서 검수를 한다.</p>
