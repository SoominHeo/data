<p><b>도쿠시마 보르티스</b>(<span style="font-size: smaller;"><a href="/wiki/%EC%9D%BC%EB%B3%B8%EC%96%B4" title="일본어">일본어</a>:</span> <span lang="ja" xml:lang="ja">徳島ヴォルティス</span> <small>도쿠시마 보루티스<sup>[<a href="/wiki/%EC%9C%84%ED%82%A4%EB%B0%B1%EA%B3%BC:%EC%9D%BC%EB%B3%B8%EC%96%B4%EC%9D%98_%ED%95%9C%EA%B8%80_%ED%91%9C%EA%B8%B0" title="위키백과:일본어의 한글 표기">*</a>]</sup></small>, Tokushima Vortis)는 <a href="/wiki/J%EB%A6%AC%EA%B7%B8" title="J리그">J리그</a>에 소속된 축구팀이다. <a href="/wiki/%EB%8F%84%EC%BF%A0%EC%8B%9C%EB%A7%88_%ED%98%84" title="도쿠시마 현">도쿠시마 현</a>을 연고지로 하고 있다. 팀 명칭 '보르티스'(Vortis)는 경기장이 위치한 <a href="/wiki/%EB%82%98%EB%A3%A8%ED%86%A0_%EC%8B%9C" title="나루토 시">나루토 시</a> 주변 <a href="/wiki/%EB%82%98%EB%A3%A8%ED%86%A0_%ED%95%B4%ED%98%91" title="나루토 해협">나루토 해협</a>의 명물인 소용돌이의 <a href="/wiki/%EC%9D%B4%ED%83%88%EB%A6%AC%EC%95%84%EC%96%B4" title="이탈리아어">이탈리아어</a>인 '<b>Vor</b>tice'의 앞 세 글자와 '<a href="/wiki/%EB%8F%84%EC%82%AC_%EA%B5%AD" title="도사 국">도사 국</a>'(<b>T</b>osa), '<a href="/wiki/%EC%9D%B4%EC%9A%94_%EA%B5%AD" title="이요 국">이요 국</a>'(<b>I</b>yo), '<a href="/wiki/%EC%82%AC%EB%88%84%ED%82%A4_%EA%B5%AD" title="사누키 국">사누키 국</a>'(<b>S</b>anuki)의 영문 앞글자를 따서 만들어졌다. 모 기업은 <a class="new" href="/w/index.php?title=%EC%98%A4%EC%B8%A0%EC%B9%B4_%EC%A0%9C%EC%95%BD&amp;action=edit&amp;redlink=1" title="오츠카 제약 (없는 문서)">오츠카 제약</a>이다.</p>

<h2><span class="mw-headline" id=".EC.97.AD.EC.82.AC">역사</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%8F%84%EC%BF%A0%EC%8B%9C%EB%A7%88_%EB%B3%B4%EB%A5%B4%ED%8B%B0%EC%8A%A4&amp;action=edit&amp;section=1" title="부분 편집: 역사">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p><a href="/wiki/1955%EB%85%84" title="1955년">1955년</a> <a class="new" href="/w/index.php?title=%EC%98%A4%EC%B8%A0%EC%B9%B4_%EC%A0%9C%EC%95%BD&amp;action=edit&amp;redlink=1" title="오츠카 제약 (없는 문서)">오츠카 제약</a> 축구단으로 창단하였으며 <a href="/wiki/2005%EB%85%84" title="2005년">2005년</a> <a href="/wiki/J%EB%A6%AC%EA%B7%B8" title="J리그">J리그</a>에 입성하였다.</p>
<p><a href="/wiki/2005%EB%85%84" title="2005년">2005년</a>, <a href="/wiki/J%EB%A6%AC%EA%B7%B8" title="J리그">J2</a>로 승격하면서 지역민과의 유착을 강화하기 위하여 팀 명칭을 지역명이 앞에 오도록 <b>도쿠시마 보르티스</b>로 바꾸었다. 유니폼 앞면에는 '<a href="/wiki/%ED%8F%AC%EC%B9%B4%EB%A6%AC_%EC%8A%A4%EC%9B%A8%ED%8A%B8" title="포카리 스웨트">포카리 스웨트</a>'가 붙어있다.</p>
<p><a href="/wiki/2011%EB%85%84" title="2011년">2011년</a> J2에서 선전하며 J1 승격을 노렸으나 3위 <a class="mw-redirect" href="/wiki/%EC%BD%98%EC%82%AC%EB%8F%84%EB%A0%88_%EC%82%BF%ED%8F%AC%EB%A1%9C" title="콘사도레 삿포로">콘사도레 삿포로</a>에게 승점 3점이 뒤진 4위로 시즌을 마감하여 아깝게 승격하지 못하였고(당시는 승격 플레이오프가 없었다) <a href="/wiki/2012%EB%85%84" title="2012년">2012년</a>을 부진을 면치 못하고 15위로 시즌을 마감하였다. 그러나 <a href="/wiki/2013%EB%85%84" title="2013년">2013년</a> J2에서 4위를 차지, <a href="/wiki/J%EB%A6%AC%EA%B7%B8_%EC%8A%B9%EA%B2%A9_%ED%94%8C%EB%A0%88%EC%9D%B4%EC%98%A4%ED%94%84" title="J리그 승격 플레이오프">J리그 승격 플레이오프</a>에서 <a class="mw-redirect" href="/wiki/%EC%A0%9C%ED%94%84_%EC%9C%A0%EB%82%98%EC%9D%B4%ED%8B%B0%EB%93%9C" title="제프 유나이티드">제프 유나이티드</a>와 <a href="/wiki/%EA%B5%90%ED%86%A0_%EC%83%81%EA%B0%80_FC" title="교토 상가 FC">교토 상가</a>를 누르고 팀 사상 최초의 J1 승격을 확정했다.</p>
