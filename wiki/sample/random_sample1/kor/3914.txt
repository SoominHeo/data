<h1 class="firstHeading" id="firstHeading" lang="ko">도토리묵</h1>
<div class="mw-body-content" id="bodyContent">

<p><b>도토리묵</b>은 <a href="/wiki/%EB%8F%84%ED%86%A0%EB%A6%AC" title="도토리">도토리</a>의 <a class="mw-redirect" href="/wiki/%EC%A0%84%EB%B6%84" title="전분">전분</a>을 이용하여 만든 <a href="/wiki/%EB%AC%B5" title="묵">묵</a>류의 음식이다. 도토리묵은 산간지방에서 기원하였는데, <a href="/wiki/%EB%96%A1%EA%B0%88%EB%82%98%EB%AC%B4" title="떡갈나무">떡갈나무</a>가 많은 지역은 <a href="/wiki/%EA%B0%80%EC%9D%84" title="가을">가을</a>에 도토리가 풍부하게 생산되므로 도토리를 음식 재료로 사용하였다. 다른 묵들과 마찬가지로 도토리묵은 <a class="new" href="/w/index.php?title=%EB%AC%B4%EC%B9%A8&amp;action=edit&amp;redlink=1" title="무침 (없는 문서)">무침</a>으로 먹거나 <a href="/wiki/%EC%96%91%EB%85%90%EC%9E%A5" title="양념장">양념장</a>을 위에 뿌려 먹는다. 양념장은 <a href="/wiki/%EA%B0%84%EC%9E%A5" title="간장">간장</a>, <a href="/wiki/%EC%B0%B8%EA%B8%B0%EB%A6%84" title="참기름">참기름</a>, <a href="/wiki/%EB%8B%B9%EA%B7%BC" title="당근">당근</a>, <a href="/wiki/%EC%96%91%ED%8C%8C" title="양파">양파</a>, <a href="/wiki/%EB%A7%88%EB%8A%98" title="마늘">마늘</a>, <a class="mw-redirect" href="/wiki/%EA%B3%A0%EC%B6%A7%EA%B0%80%EB%A3%A8" title="고춧가루">고춧가루</a>, <a href="/wiki/%EA%B9%A8%EC%86%8C%EA%B8%88" title="깨소금">깨소금</a> 등을 섞어 만든다.</p>
<p>도토리묵은 수백만의 사람들이 헐벗고 굶주렸던 <a class="mw-redirect" href="/wiki/%ED%95%9C%EA%B5%AD%EC%A0%84%EC%9F%81" title="한국전쟁">한국전쟁</a> 기간동안 많이 먹었으나, 세월이 흐르며 도토리묵은 빈곤의 상징으로 인식되기 시작했고 <a href="/wiki/%EB%A9%94%EB%B0%80%EB%AC%B5" title="메밀묵">메밀묵</a> 등의 다른 묵을 쉽게 구할 수 있게 되면서, 점차적으로 수요가 감소했다. 그러나 최근 몇 년 사이 도토리묵은 건강 식품으로 재조명되고 있다.</p>
<h2><span class="mw-headline" id=".EC.83.9D.EC.82.B0_.EA.B3.BC.EC.A0.95">생산 과정</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%8F%84%ED%86%A0%EB%A6%AC%EB%AC%B5&amp;action=edit&amp;section=1" title="부분 편집: 생산 과정">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>도토리에는 전분과 <a href="/wiki/%EB%8B%A8%EB%B0%B1%EC%A7%88" title="단백질">단백질</a>이 풍부하지만, <a class="mw-redirect" href="/wiki/%ED%83%84%EB%8B%8C" title="탄닌">탄닌</a>과 <a href="/wiki/%ED%8F%B4%EB%A6%AC%ED%8E%98%EB%86%80" title="폴리페놀">폴리페놀</a> 역시 풍부하여 소화를 방해할 수 있다. 따라서, 수확한 도토리는 반드시 사용하기전에 탄닌 성분을 적절하게 가공하여야 한다. 도토리는 땅에 떨어지는 걸 줍거나, 가지에 달린 것을 직접 수확한다. 수확한 도토리는 <a href="/wiki/%EA%BB%8D%EC%A7%88" title="껍질">껍질</a>을 까고 내부의 <a href="/wiki/%EA%B2%AC%EA%B3%BC" title="견과">견과</a>부분을 분리하여 곱게 갈아낸다. 도토리 가루를 많은 양의 물과 잘 섞어 우려낸 후, <a class="mw-disambig" href="/wiki/%EC%95%99%EA%B8%88" title="앙금">앙금</a>을 가라앉힌 뒤 걸러내면 도토리 가루 속의 <a class="mw-redirect" href="/wiki/%EC%84%AC%EC%9C%A0%EC%86%8C" title="섬유소">섬유소</a>와 전분 성분을 분리할 수 있다.</p>
<p>이런 과정에서 섬유소와 전분이 분리되는 것 뿐만 아니라, 전분 속의 탄닌 성분도 <a class="new" href="/w/index.php?title=%ED%99%95%EC%82%B0%ED%98%84%EC%83%81&amp;action=edit&amp;redlink=1" title="확산현상 (없는 문서)">확산현상</a>을 통해 빠져나간다. 물 속에서 우려내는 시간의 길이는 도토리가루 속의 탄닌 양에 비례하며, 효과적인 탄닌 제거를 위해서는 여러번 물을 갈아주어야 한다.</p>
<p>탄닌이 완전히 제거된 도토리 전분가루는 색이 희어진다. 이 상태의 앙금은 그릇의 가장 바닥에 가라앉게 되므로, 윗물만 모두 따라낸 후 앙금을 말린다. 물기가 제거된 앙금은 곱게 빻아 가루의 형태로 판매된다. 가루 형태의 도토리 전분 가루는, 물과 섞어 걸쭉한 농도가 되도록 끓여, 평평한 접시에 부어 식히면 도토리묵으로 만들 수 있다.</p>
