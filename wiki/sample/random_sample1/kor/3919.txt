<h2><span class="mw-headline" id=".EA.B0.9C.EC.9A.94">개요</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%8F%85%EB%A6%BD%EA%B5%AD%EA%B0%80%EC%97%B0%ED%95%A9_%EC%B6%95%EA%B5%AC_%EA%B5%AD%EA%B0%80%EB%8C%80%ED%91%9C%ED%8C%80&amp;action=edit&amp;section=1" title="부분 편집: 개요">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<div class="thumb tright">
<div class="thumbinner" style="width:202px;"><a class="image" href="/wiki/%ED%8C%8C%EC%9D%BC:Flag_of_the_CIS_(UEFA_Euro_1992).svg"><img alt="" class="thumbimage" data-file-height="600" data-file-width="900" height="133" src="//upload.wikimedia.org/wikipedia/commons/thumb/3/33/Flag_of_the_CIS_%28UEFA_Euro_1992%29.svg/200px-Flag_of_the_CIS_%28UEFA_Euro_1992%29.svg.png" srcset="//upload.wikimedia.org/wikipedia/commons/thumb/3/33/Flag_of_the_CIS_%28UEFA_Euro_1992%29.svg/300px-Flag_of_the_CIS_%28UEFA_Euro_1992%29.svg.png 1.5x, //upload.wikimedia.org/wikipedia/commons/thumb/3/33/Flag_of_the_CIS_%28UEFA_Euro_1992%29.svg/400px-Flag_of_the_CIS_%28UEFA_Euro_1992%29.svg.png 2x" width="200"/></a>
<div class="thumbcaption">
<div class="magnify"><a class="internal" href="/wiki/%ED%8C%8C%EC%9D%BC:Flag_of_the_CIS_(UEFA_Euro_1992).svg" title="실제 크기로"></a></div>
UEFA 유로 1992에 출전한 독립국가연합 축구 국가대표팀의 깃발</div>
</div>
</div>
<p><a href="/wiki/1991%EB%85%84" title="1991년">1991년</a> <a href="/wiki/12%EC%9B%94_26%EC%9D%BC" title="12월 26일">12월 26일</a> <a href="/wiki/%EC%86%8C%EB%A0%A8" title="소련">소련</a>이 <a href="/wiki/%EC%86%8C%EB%A0%A8%EC%9D%98_%EB%B6%95%EA%B4%B4" title="소련의 붕괴">해체</a>되었다. <a href="/wiki/1992%EB%85%84" title="1992년">1992년</a> <a href="/wiki/1%EC%9B%94_1%EC%9D%BC" title="1월 1일">1월 1일</a> <a class="new" href="/w/index.php?title=%EC%86%8C%EB%A0%A8_%EC%B6%95%EA%B5%AC_%ED%98%91%ED%9A%8C&amp;action=edit&amp;redlink=1" title="소련 축구 협회 (없는 문서)">소련 축구 협회</a>와 <a href="/wiki/%EC%86%8C%EB%A0%A8_%EC%B6%95%EA%B5%AC_%EA%B5%AD%EA%B0%80%EB%8C%80%ED%91%9C%ED%8C%80" title="소련 축구 국가대표팀">소련 축구 국가대표팀</a>이 해체되었고 대신 <a class="new" href="/w/index.php?title=%EB%8F%85%EB%A6%BD%EA%B5%AD%EA%B0%80%EC%97%B0%ED%95%A9_%EC%B6%95%EA%B5%AC_%ED%98%91%ED%9A%8C&amp;action=edit&amp;redlink=1" title="독립국가연합 축구 협회 (없는 문서)">독립국가연합 축구 협회</a>가 <a href="/wiki/1992%EB%85%84" title="1992년">1992년</a> <a href="/wiki/1%EC%9B%94_11%EC%9D%BC" title="1월 11일">1월 11일</a>에 신설되었으며, 이틀 후 <a class="mw-redirect" href="/wiki/FIFA" title="FIFA">FIFA</a>로부터 공식 승인을 받았다.</p>
<p><a href="/wiki/%EC%97%90%EC%8A%A4%ED%86%A0%EB%8B%88%EC%95%84_%EC%B6%95%EA%B5%AC_%EA%B5%AD%EA%B0%80%EB%8C%80%ED%91%9C%ED%8C%80" title="에스토니아 축구 국가대표팀">에스토니아</a>, <a href="/wiki/%EB%9D%BC%ED%8A%B8%EB%B9%84%EC%95%84_%EC%B6%95%EA%B5%AC_%EA%B5%AD%EA%B0%80%EB%8C%80%ED%91%9C%ED%8C%80" title="라트비아 축구 국가대표팀">라트비아</a>, <a href="/wiki/%EB%A6%AC%ED%88%AC%EC%95%84%EB%8B%88%EC%95%84_%EC%B6%95%EA%B5%AC_%EA%B5%AD%EA%B0%80%EB%8C%80%ED%91%9C%ED%8C%80" title="리투아니아 축구 국가대표팀">리투아니아</a>는 이미 독자적인 축구 협회를 꾸린 상태였기 때문에 이들 세 나라의 선수들은 독립국가연합 축구 국가대표팀에 뽑히지 않았다. 독립국가연합 축구 국가대표팀은 함께 뭉치는 데에는 큰 지장이 없었으나, 통합적인 축구 리그 계획은 성공적이지 못했고, 각기 나라에서 다른 축구 선수권 대회를 개최하였다.</p>
<p>독립국가연합 축구 국가대표팀은 <a href="/wiki/%EC%8A%A4%EC%9B%A8%EB%8D%B4" title="스웨덴">스웨덴</a>에서 열린 <a href="/wiki/UEFA_%EC%9C%A0%EB%A1%9C_1992" title="UEFA 유로 1992">UEFA 유로 1992</a>에 출전했으며 <a href="/wiki/%EA%B5%AD%EA%B0%80_(%EB%85%B8%EB%9E%98)" title="국가 (노래)">국가</a>는 <a href="/wiki/%EB%A3%A8%ED%8A%B8%EB%B9%84%ED%9E%88_%ED%8C%90_%EB%B2%A0%ED%86%A0%EB%B2%A4" title="루트비히 판 베토벤">베토벤</a>의 <a href="/wiki/%EA%B5%90%ED%96%A5%EA%B3%A1_9%EB%B2%88_(%EB%B2%A0%ED%86%A0%EB%B2%A4)" title="교향곡 9번 (베토벤)">교향곡 9번</a>으로 정하였다. 독립국가연합 축구 국가대표팀의 감독은 <a href="/wiki/%EC%95%84%EB%82%98%ED%86%A8%EB%A6%AC_%EB%B9%84%EC%87%BC%EB%B2%A0%EC%B8%A0" title="아나톨리 비쇼베츠">아나톨리 비쇼베츠</a>였다. UEFA 유로 1992에 출전한 독립국가연합 축구 국가대표팀은 <a href="/wiki/%EB%8F%85%EC%9D%BC_%EC%B6%95%EA%B5%AC_%EA%B5%AD%EA%B0%80%EB%8C%80%ED%91%9C%ED%8C%80" title="독일 축구 국가대표팀">독일</a>과는 1-1, <a href="/wiki/%EB%84%A4%EB%8D%9C%EB%9E%80%EB%93%9C_%EC%B6%95%EA%B5%AC_%EA%B5%AD%EA%B0%80%EB%8C%80%ED%91%9C%ED%8C%80" title="네덜란드 축구 국가대표팀">네덜란드</a>와는 0-0으로 비겼으나, 독립국가연합 축구 국가대표팀의 마지막 경기인 <a href="/wiki/%EC%8A%A4%EC%BD%94%ED%8B%80%EB%9E%9C%EB%93%9C_%EC%B6%95%EA%B5%AC_%EA%B5%AD%EA%B0%80%EB%8C%80%ED%91%9C%ED%8C%80" title="스코틀랜드 축구 국가대표팀">스코틀랜드</a>와의 경기에서 3-0으로 패하여 조별 예선 최하위로 탈락하였다.</p>
<p>대회가 끝난 이후에 독립국가연합 축구 국가대표팀은 각자의 12개 축구 국가대표팀(<a href="/wiki/%EB%9F%AC%EC%8B%9C%EC%95%84_%EC%B6%95%EA%B5%AC_%EA%B5%AD%EA%B0%80%EB%8C%80%ED%91%9C%ED%8C%80" title="러시아 축구 국가대표팀">러시아</a>, <a href="/wiki/%EC%9A%B0%ED%81%AC%EB%9D%BC%EC%9D%B4%EB%82%98_%EC%B6%95%EA%B5%AC_%EA%B5%AD%EA%B0%80%EB%8C%80%ED%91%9C%ED%8C%80" title="우크라이나 축구 국가대표팀">우크라이나</a>, <a
