<p><b>독립기념관</b>(獨立記念館, Independence Hall)은 <a href="/wiki/%EB%AF%B8%EA%B5%AD" title="미국">미국</a> <a href="/wiki/%ED%8E%9C%EC%8B%A4%EB%B2%A0%EC%9D%B4%EB%8B%88%EC%95%84_%EC%A3%BC" title="펜실베이니아 주">펜실베이니아 주</a> <a href="/wiki/%ED%95%84%EB%9D%BC%EB%8D%B8%ED%94%BC%EC%95%84" title="필라델피아">필라델피아</a>에 있는 기념관이다. <a href="/wiki/1753%EB%85%84" title="1753년">1753년</a> 원래는 <a href="/wiki/%ED%8E%9C%EC%8B%A4%EB%B2%A0%EC%9D%B4%EB%8B%88%EC%95%84_%EC%8B%9D%EB%AF%BC%EC%A7%80" title="펜실베이니아 식민지">펜실베이니아 식민지</a>의 의회 건물로 완공되었고, <a href="/wiki/1775%EB%85%84" title="1775년">1775년</a>에서 <a href="/wiki/1783%EB%85%84" title="1783년">1783년</a>까지 <a class="mw-redirect" href="/wiki/%EC%A0%9C2%EC%B0%A8_%EB%8C%80%EB%A5%99_%ED%9A%8C%EC%9D%98" title="제2차 대륙 회의">제2차 대륙 회의</a>의 주요 회의 장소였으며, <a href="/wiki/1776%EB%85%84" title="1776년">1776년</a> <a href="/wiki/7%EC%9B%94_4%EC%9D%BC" title="7월 4일">7월 4일</a> <a href="/wiki/%EB%AF%B8%EA%B5%AD_%EB%8F%85%EB%A6%BD_%EC%84%A0%EC%96%B8" title="미국 독립 선언">미국 독립 선언</a>을 한 건물이다. <a class="mw-redirect" href="/wiki/%ED%95%84%EB%9D%BC%EB%8D%B8%ED%94%BC%EC%95%84_%ED%97%8C%EB%B2%95_%EC%A0%9C%EC%A0%95_%ED%9A%8C%EC%9D%98" title="필라델피아 헌법 제정 회의">필라델피아 제헌회의</a>가 <a href="/wiki/1787%EB%85%84" title="1787년">1787년</a> 여름에 개최되기도 하였다.</p>
<p><a href="/wiki/1979%EB%85%84" title="1979년">1979년</a>, <a class="mw-redirect" href="/wiki/%EC%9C%A0%EB%84%A4%EC%8A%A4%EC%BD%94" title="유네스코">유네스코</a>의 <a class="mw-redirect" href="/wiki/%EC%84%B8%EA%B3%84_%EC%9C%A0%EC%82%B0" title="세계 유산">세계 유산</a>(문화 유산)으로 등록되었다.<sup class="reference" id="cite_ref-WldHeritageSite_1-0"><a href="#cite_note-WldHeritageSite-1">[1]</a></sup></p>

<h2><span class="mw-headline" id=".EC.97.AD.EC.82.AC">역사</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%8F%85%EB%A6%BD%EA%B8%B0%EB%85%90%EA%B4%80_(%EB%AF%B8%EA%B5%AD)&amp;action=edit&amp;section=1" title="부분 편집: 역사">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<ul>
<li><a href="/wiki/1749%EB%85%84" title="1749년">1749년</a>에 <a href="/wiki/%ED%8E%9C%EC%8B%A4%EB%B2%A0%EC%9D%B4%EB%8B%88%EC%95%84_%EC%A3%BC" title="펜실베이니아 주">펜실베이니아 주</a>의 의사당으로 건설되었다.</li>
<li><a href="/wiki/1776%EB%85%84" title="1776년">1776년</a> <a href="/wiki/7%EC%9B%94_4%EC%9D%BC" title="7월 4일">7월 4일</a>에 영국 식민지 하에 있던 <a href="/wiki/13%EA%B0%9C_%EC%8B%9D%EB%AF%BC%EC%A7%80" title="13개 식민지">13개 식민지</a> 대표가 이 건물의 사랑방에 모여 <a href="/wiki/%ED%86%A0%EB%A8%B8%EC%8A%A4_%EC%A0%9C%ED%8D%BC%EC%8A%A8" title="토머스 제퍼슨">토머스 제퍼슨</a>이 기초한 <a class="mw-redirect" href="/wiki/%EB%AF%B8%EA%B5%AD%EB%8F%85%EB%A6%BD%EC%84%A0%EC%96%B8%EB%AC%B8" title="미국독립선언문">미국독립선언문</a>에 서명하였고, 이때부터 ‘독립기념관’이라고 부르게 되었다.</li>
<li><a href="/wiki/1787%EB%85%84" title="1787년">1787년</a>에는 <a class="mw-redirect" href="/wiki/%EB%AF%B8%EA%B5%AD_%ED%97%8C%EB%B2%95" title="미국 헌법">미국 헌법</a>이 제정된 곳이기도 하다.</li>
<li><a href="/wiki/1790%EB%85%84" title="1790년">1790년</a> ~ <a href="/wiki/1800%EB%85%84" title="1800년">1800년</a>에 <a href="/wiki/%ED%95%84%EB%9D%BC%EB%8D%B8%ED%94%BC%EC%95%84" title="필라델피아">필라델피아</a>가 미국의 수도가 되면서, <a href="/wiki/%EB%AF%B8%EA%B5%AD_%EC%9D%98%ED%9A%8C" title="미국 의회">미국 의회</a> 의사당으로 사용된다. 의원 수가 적었던 ‘원로원’이 2층을 사용하였고, "하원"이 1층을 회의장으로 사용한 것이 ‘상원’과 ‘하원’이라는 단어의 어원이 되었다.</li>
</ul>
<h2><span class="mw-headline" id=".EC.A3.BC.EB.B3.80">주변</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%8F%85%EB%A6%BD%EA%B8%B0%EB%85%90%EA%B4%80_(%EB%AF%B8%EA%B5%AD)&amp;action=edit&amp;section=2" title="부분 편집: 주변">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>독립기념관 주변은 ‘독립 기념관 국립 역사 공원’으로 정비되어 있으며, 앞에 있는 ‘리버티 벨 파빌리온’'에는 독립 선언에 명문을 썼다는 ‘자유의 종’이 있다. 이것은 미국의 상징물 중의 하나이다. 독립 기념관 있는 필라델피아 근교에는 독립 전쟁의 랜드 마크인 <a href="/wiki/%EB%B0%B8%EB%A6%AC_%ED%8F%AC%EC%A7%80" title="밸리 포지">밸리 포지</a>가 있다.</p>
<h2><span class="mw-headline" id=".EB.93.B1.EB.A1.9D_.EA.B8.B0.EC.A4.80">등록 기준</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%8F%85%EB%A6%BD%EA%B8%B0%EB%85%90%EA%B4%80_(%EB%AF%B8%EA%B5%AD)&amp;action=edit&amp;section=3" title="부분 편집: 등록 기준">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>이 세계 유산은 세계 유산 등록 기준의 이하의 기준을 충족하는 것으로 간주 되어 등록이 이루어졌다. 아래의 기준은 세계 유산 센터에서 번역, 인용한 것이다.</p>
<ul>
<li>(vi) 현저한 보편적인 의미를 갖는 사건, 현존하는 전통 사상, 신앙 또는 예술, 문학 작품과 직접 또는 명백하게 관련하는 것.</li>
</ul>
