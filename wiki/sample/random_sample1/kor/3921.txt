<p><b>서울 독립문</b>(서울 獨立門)은 <a href="/wiki/%EC%A1%B0%EC%84%A0" title="조선">조선</a> 후기의 건축물로 <a href="/wiki/1896%EB%85%84" title="1896년">1896년</a> <a href="/wiki/11%EC%9B%94" title="11월">11월</a>에 준공되어 <a href="/wiki/1897%EB%85%84" title="1897년">1897년</a>에 완공되었다. <a href="/wiki/%EB%8F%85%EB%A6%BD%ED%98%91%ED%9A%8C" title="독립협회">독립협회</a>가 중심이 되어 <a href="/wiki/%EC%A1%B0%EC%84%A0" title="조선">조선</a>이 청나라의 책봉 체제에서 독립한 것을 상징하기 위하여<sup class="reference" id="cite_ref-1"><a href="#cite_note-1">[1]</a></sup> <a href="/wiki/%EC%98%81%EC%9D%80%EB%AC%B8" title="영은문">영은문</a>을 무너뜨리고 그 터에 지은 문으로<sup class="reference" id="cite_ref-gojong_2-0"><a href="#cite_note-gojong-2">[2]</a></sup><sup class="reference" id="cite_ref-3"><a href="#cite_note-3">[3]</a></sup> <a href="/wiki/%EC%84%9C%EC%9E%AC%ED%95%84" title="서재필">서재필</a>의 주도로 건립되었으며, 서재필의 원작을 배경으로 <a href="/wiki/%EC%95%84%ED%8C%8C%EB%82%98%EC%8B%9C_%EC%84%B8%EB%A0%88%EB%94%98%EC%82%AC%EB%B0%94%ED%8B%B4" title="아파나시 세레딘사바틴">아파나시 세레딘사바틴</a>(Середин-Cабатин, 士巴津(사파진), Sabatin, 흔히 사바틴)이 설계·시공했고, 그 현판은 <a href="/wiki/%EA%B9%80%EA%B0%80%EC%A7%84" title="김가진">김가진</a>의 작품이다.</p>
<p><a href="/wiki/%EC%A1%B0%EC%84%A0" title="조선">조선</a>의 <a href="/wiki/%EC%A4%91%EA%B5%AD" title="중국">중국</a>에 대한 사대의 상징인 <a href="/wiki/%EC%98%81%EC%9D%80%EB%AC%B8" title="영은문">영은문</a>을 헐고 그 뒷편에 건립되었으며, 독립문 전면에는 영은문이 서 있던 기둥돌이 함께 남아 있다. <a href="/wiki/1963%EB%85%84" title="1963년">1963년</a> <a href="/wiki/1%EC%9B%94_21%EC%9D%BC" title="1월 21일">1월 21일</a> <a href="/wiki/%EB%8C%80%ED%95%9C%EB%AF%BC%EA%B5%AD" title="대한민국">대한민국</a> 사적 제32호로 지정되었다.</p>

<h2><span class="mw-headline" id=".EB.B0.B0.EA.B2.BD">배경</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%8F%85%EB%A6%BD%EB%AC%B8&amp;action=edit&amp;section=1" title="부분 편집: 배경">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p><a href="/wiki/%EC%8B%A0%EB%9D%BC" title="신라">신라</a>시대에 <a href="/wiki/%EB%8B%B9%EB%82%98%EB%9D%BC" title="당나라">당나라</a>를 상국으로 받들고 조공을 바친 이후 <a href="/wiki/%EA%B3%A0%EB%A0%A4" title="고려">고려</a> 시대의 <a href="/wiki/%EC%86%A1%EB%82%98%EB%9D%BC" title="송나라">송나라</a>, <a href="/wiki/%EC%9B%90%EB%82%98%EB%9D%BC" title="원나라">원나라</a>와의 관계를 거쳐 <a href="/wiki/%EC%A1%B0%EC%84%A0" title="조선">조선</a>시대에도 <a href="/wiki/%EB%AA%85%EB%82%98%EB%9D%BC" title="명나라">명나라</a>를 상국으로 받들고 조공을 바쳤으며, 속국을 자처하였다. 이를 사대 정책이라 한다. 조선 시대에는 명의 사신들을 맞이하기 위해 <a class="mw-redirect" href="/wiki/%EC%84%9C%EB%8C%80%EB%AC%B8" title="서대문">서대문</a> 근처에 <a href="/wiki/%EC%98%81%EC%9D%80%EB%AC%B8" title="영은문">영은문</a>을 건립하고, 근처에는 사신을 맞이하는 <a class="new" href="/w/index.php?title=%EB%AA%A8%ED%99%94%EA%B4%80&amp;action=edit&amp;redlink=1" title="모화관 (없는 문서)">모화관</a>을 건립하였다. '영은'은 은혜로운 대국의 사신을 맞이한다는 뜻이며, '모화'는 중국을 흠모한다는 뜻이다. 명에 대한 조선의 관계는 소중화사상으로 이어져 명나라가 멸망하고 <a href="/wiki/%EC%B2%AD%EB%82%98%EB%9D%BC" title="청나라">청나라</a>로 교체된 뒤에도 계속되었다.</p>
<p><a href="/wiki/%EB%B3%91%EC%9E%90%ED%98%B8%EB%9E%80" title="병자호란">병자호란</a>과 <a href="/wiki/%EC%A0%95%EB%AC%98%ED%98%B8%EB%9E%80" title="정묘호란">정묘호란</a>으로 인한 <a href="/wiki/%EC%B2%AD%EB%82%98%EB%9D%BC" title="청나라">청나라</a>에 대한 반감에 더하여, 예전부터 <a href="/wiki/%EB%B0%9C%ED%95%B4" title="발해">발해</a>와 <a href="/wiki/%EA%B3%A0%EB%A0%A4" title="고려">고려</a>, 조선을 형님의 나라이며 상국으로 받들던 <a class="mw-redirect" href="/wiki/%EC%97%AC%EC%A7%84%EC%A1%B1" title="여진족">여진족</a>에 대한 멸시 등이 복합되어 반청 감정은 고조되었고, 북벌론이 등장하게 되었다. 청나라에 대한 반감은 19세기까지 이어졌고 청나라가 유럽 열강과의 전쟁에서 패하기 시작하면서 <a href="/wiki/%EC%A1%B0%EC%84%A0" title="조선">조선</a>이 자주독립국임을 선언해야 된다는 여론이 강하게 나타났다.</p>
<p><a href="/wiki/1890%EB%85%84" title="1890년">1890년</a>대에 <a href="/wiki/%EC%84%9C%EC%9E%AC%ED%95%84" title="서재필">서재필</a>은 <a href="/wiki/%EC%98%81%EC%9D%80%EB%AC%B8" title="영은문">영은문</a>과 모화관을 헐고 그 자리에 독립문과 <a class="new" href="/w/index.php?title=%EB%8F%85%EB%A6%BD%EA%B4%80&amp;action=edit&amp;redlink=1" title="독립관 (없는 문서)">독립관</a>을 세워야 한다고 주장하였다. 서재필은 직접 <a href="/wiki/%ED%94%84%EB%9E%91%EC%8A%A4" title="프랑스">프랑스</a>의 <a href="/wiki/%EA%B0%9C%EC%84%A0%EB%AC%B8" title="개선문">개선문</a> 사진을 보고, 이를 참고하여 독립문의 모양새를 직접 스케치했다. 이는 1896년 <a href="/wiki/%EC%8A%A4%EC%9C%84%EC%8A%A4" title="스위스">스위스</a>계<sup class="reference" id="cite_ref-4"><a href="#cite_note-4">[4]</a></sup> 건축가 <a href="/wiki/%EC%95%84%ED%8C%8C%EB%82%98%EC%8B%9C_%EC%84%B8%EB%A0%88%EB%94%98%EC%82%AC%EB%B0%94%ED%8B%B4" title="아파나시 세레딘사바틴">아파나시 세레딘사바틴</a>에게 보내졌다.</p>
<h3><span class="mw-headline" id=".EC.84.9C.EC.9E.AC.ED.95.84.EC.9D.98_.EB.85.BC.ED.8F.89">서재필의 논평</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%8F%85%EB%A6%BD%EB%AC%B8&amp;action=edit&amp;section=2" title="부분 편집: 서재필의 논평">편집</a><span class="mw-editsection-bracket">]</span></span></h3>

<p>서재필은 1896년 7월 4일에 발행된 《독립신문》에 게재한 자신의 논평에서 조선이 청나라의 책봉 체제에서 독립한 것을 상징하는 건축물의 필요성을 시사했다.</p>
<blockquote class="toccolours" style="float:none; padding: 10px 15px 10px 15px; display:table;">
<p>조선이 몇 해를 청나라의 속국으로 있다가 하나님 덕에 독립이 되어 조선 대군주 폐하께서 지금은 세계의 제일 높은 임금들과 동등이 되시고, 조선 인민이 세계에서 자유로운 백성이 되었으니, 이런 경사를 그저 보고 지내는 것이 도리가 아니요, 조선 독립된 것을 세계에 광고도 하며, 또 조선 후생들에게도 이때에 조선이 분명하게 독립된 것을 전하자는 표적이 있어야 할 터이요. 또 조선 인민이 양생을 하려면 맑은 공기를 마셔야 할 터이요, 경치 좋고 맑은 데서 운동도 하여야 할지라.</p>
</blockquote>
<h2><span class="mw-headline" id=".EA.B0.9C.EC.9A.94">개요</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%8F%85%EB%A6%BD%EB%AC%B8&amp;action=edit&amp;section=3" title="부분 편집: 개요">편집</a><span class="mw-editsection-bracket">]</span></span></h2>

<p>독립문의 건립은 1890년대 초 <a href="/wiki/%EC%84%9C%EC%9E%AC%ED%95%84" title="서재필">서재필</a> 등 개화파에 의해 추진되었다. 이들은 <a href="/wiki/%EC%98%81%EC%9D%80%EB%AC%B8" title="영은문">영은문</a>은 <a href="/wiki/%EC%A4%91%EA%B5%AD" title="중국">중국</a>에 사대해온 치욕의 상징이므로 헐고, 중국으로부터의 독립을 상징하지는 건축물을 건설하자고 <a href="/wiki/%EB%8C%80%ED%95%9C%EC%A0%9C%EA%B5%AD_%EA%B3%A0%EC%A2%85" title="대한제국 고종">고종</a>에게 적극적으로 건의하기 시작하였다. 독립문은 <a href="/wiki/%EC%98%81%EC%9D%80%EB%AC%B8" title="영은문">영은문</a> 맞은편이 예정지로 정해지고 시공되었으며, <a href="/wiki/1896%EB%85%84" title="1896년">1896년</a> <a href="/wiki/11%EC%9B%94_20%EC%9D%BC" title="11월 20일">11월 20일</a> 건립되었다.</p>
<p><a href="/wiki/1895%EB%85%84" title="1895년">1895년</a>부터 서재필은 독립문 건립을 위해 백성들의 성금을 모금하였고, 스위스계 우크라이나인인 세레딘사바틴을 영입해왔다. 서재필은 직접 독립문을 스케치하고 이를 토대로 공사를 시작한다.</p>
<p><a href="/wiki/%EC%B2%AD%EC%9D%BC_%EC%A0%84%EC%9F%81" title="청일 전쟁">청일 전쟁</a>에서 일본이 승리, <a href="/wiki/%EC%8B%9C%EB%AA%A8%EB%85%B8%EC%84%B8%ED%82%A4_%EC%A1%B0%EC%95%BD" title="시모노세키 조약">시모노세키 조약</a>에서 조선의 독립이 국제법상 확정<sup class="reference" id="cite_ref-5"><a href="#cite_note-5">[5]</a></sup> 된 후, 건설이 시작되었다. <a href="/wiki/1896%EB%85%84" title="1896년">1896년</a>에 공사를 시작해 <a href="/wiki/1897%EB%85%84" title="1897년">1897년</a>에 완공되었다. 높이는 14.28<a href="/wiki/%EB%AF%B8%ED%84%B0" title="미터">미터</a>, 폭 11.48미터이다. 약 1,850개의 <a href="/wiki/%ED%99%94%EA%B0%95%EC%95%94" title="화강암">화강암</a>으로 구성되어 있으며, <a href="/wiki/%ED%94%84%EB%9E%91%EC%8A%A4" title="프랑스">프랑스</a> <a class="mw-redirect" href="/wiki/%ED%8C%8C%EB%A6%AC_%EC%8B%9C" title="파리 시">파리</a>의 <a href="/wiki/%EA%B0%9C%EC%84%A0%EB%AC%B8" title="개선문">개선문</a>을 모델로 삼아서 만들었다. 현판은 <a href="/wiki/%EA%B9%80%EA%B0%80%EC%A7%84" title="김가진">김가진</a>이 썼고, 현판 바로 아래에는 <a class="mw-redirect" href="/wiki/%EB%8C%80%ED%95%9C%EC%A0%9C%EA%B5%AD_%ED%99%A9%EC%A1%B1" title="대한제국 황족">대한제국 황실</a>을 상징하는 문양인 <a class="mw-redirect" href="/wiki/%EC%98%A4%EC%96%8F" title="오얏">오얏</a>꽃이 장식되어 있다. 앞쪽의 기둥 두 개는 옛날 영은문의 기둥으로서, 철거 후에도 남은 부분이다. 독립문의 기공식 때는 <a href="/wiki/%EB%8C%80%ED%95%9C%EC%A0%9C%EA%B5%AD_%EA%B3%A0%EC%A2%85" title="대한제국 고종">대한제국 고종</a>을 비롯한 황족과 고관들이 모두 참여하였다. 이 중 <a href="/wiki/%EA%B9%80%EA%B0%80%EC%A7%84" title="김가진">김가진</a>은 <a class="mw-redirect" href="/wiki/%EC%82%BC%EC%A0%84%EB%8F%84" title="삼전도">삼전도</a>(<a href="/wiki/%EC%A0%95%EC%B6%95%ED%95%98%EC%84%B1" title="정축하성">정축하성</a>)와 병자년(<a href="/wiki/%EB%B3%91%EC%9E%90%ED%98%B8%EB%9E%80" title="병자호란">병자호란</a>), 정묘년(<a href="/wiki/%EC%A0%95%EB%AC%98%ED%98%B8%EB%9E%80" title="정묘호란">정묘호란</a>)의 굴욕을 이제서야 깨트렸다며 크게 기뻐하였다 한다.</p>
<p><a href="/wiki/1979%EB%85%84" title="1979년">1979년</a>에 <a class="mw-redirect" href="/wiki/%EC%84%B1%EC%82%B0%EB%8C%80%EB%A1%9C" title="성산대로">성산대로</a> 공사로 인해 본래의 위치에서 북서쪽으로 70미터 정도 이전하였고, <a href="/wiki/2009%EB%85%84" title="2009년">2009년</a> <a href="/wiki/10%EC%9B%94_28%EC%9D%BC" title="10월 28일">10월 28일</a>에 <a class="mw-redirect" href="/wiki/%EC%84%9C%EB%8C%80%EB%AC%B8_%EB%8F%85%EB%A6%BD%EA%B3%B5%EC%9B%90" title="서대문 독립공원">서대문 독립공원</a>의 재조성 공사가 완료되면서 동시에 독립문은 일반인에게 개방되었다.</p>
<h2><span class="mw-headline" id=".EA.B8.B0.ED.83.80">기타</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%8F%85%EB%A6%BD%EB%AC%B8&amp;action=edit&amp;section=4" title="부분 편집: 기타">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>독립문은 서재필이 특별히 초빙한 건축사 <a href="/wiki/%EC%95%84%ED%8C%8C%EB%82%98%EC%8B%9C_%EC%84%B8%EB%A0%88%EB%94%98%EC%82%AC%EB%B0%94%ED%8B%B4" title="아파나시 세레딘사바틴">아파나시 세레딘사바틴</a>이 설계하였다. 후일 사적 32호로 지정된 독립문은 <a href="/wiki/%EC%84%9C%EC%9E%AC%ED%95%84" title="서재필">서재필</a>이 독립문의 윤곽을 스케치한 것을 바탕으로 <a href="/wiki/%EB%8F%85%EC%9D%BC" title="독일">독일</a> 공사관의 <a href="/wiki/%EC%8A%A4%EC%9C%84%EC%8A%A4" title="스위스">스위스</a>인 기사가 설계를 담당했다. 토목·건축공사는 한국인 건축 기사 심의석이 담당하고 중국인 노무자들이 노역을 맡았다.<sup class="reference" id="cite_ref-sonns_6-0"><a href="#cite_note-sonns-6">[6]</a></sup> 공사비는 기부금으로 해결했다.<sup class="reference" id="cite_ref-sonns_6-1"><a href="#cite_note-sonns-6">[6]</a></sup></p>
<p>후일 <a href="/wiki/%EC%95%84%ED%8C%8C%EB%82%98%EC%8B%9C_%EC%84%B8%EB%A0%88%EB%94%98%EC%82%AC%EB%B0%94%ED%8B%B4" title="아파나시 세레딘사바틴">아파나시 세레딘사바틴</a>의 출신과 관련, 서재필은 그의 자서전에서 독립문 설계자를 <a href="/wiki/%EC%8A%A4%EC%9C%84%EC%8A%A4" title="스위스">스위스</a>인 기사라고 언급했다.<sup class="reference" id="cite_ref-swiss_7-0"><a href="#cite_note-swiss-7">[7]</a></sup> 후일 <a href="/wiki/%EC%8A%A4%EC%9C%84%EC%8A%A4" title="스위스">스위스</a>인이 아니라는 반론이 제기되었다. <a class="new" href="/w/index.php?title=%EA%B5%AD%EC%A0%9C%ED%95%9C%EA%B5%AD%EC%82%AC%ED%95%99%ED%9A%8C&amp;action=edit&amp;redlink=1" title="국제한국사학회 (없는 문서)">국제한국사학회</a> 공동 연구자의 한사람인 <a class="new" href="/w/index.php?title=%EB%9F%AC%EC%8B%9C%EC%95%84_%EA%B5%AD%EB%A6%BD%EC%9D%B8%EB%AC%B8%EA%B3%BC%ED%95%99%EB%8C%80%ED%95%99&amp;action=edit&amp;redlink=1" title="러시아 국립인문과학대학 (없는 문서)">러시아 국립인문과학대학</a> <a class="new" href="/w/index.php?title=%ED%83%80%EC%B9%98%EC%95%84%EB%82%98_%EC%8B%AC%EB%B9%84%EB%A5%B4%EC%B2%B4%EB%B0%94&amp;action=edit&amp;redlink=1" title="타치아나 심비르체바 (없는 문서)">타치아나 심비르체바</a> <a href="/wiki/%EB%B0%95%EC%82%AC" title="박사">박사</a>는 아파나시 세레딘사바틴이 <a href="/wiki/%EC%9A%B0%ED%81%AC%EB%9D%BC%EC%9D%B4%EB%82%98" title="우크라이나">우크라이나</a> 출신이지만 <a href="/wiki/%ED%8F%B4%EB%9E%80%EB%93%9C" title="폴란드">폴란드</a>, <a href="/wiki/%EB%8F%85%EC%9D%BC" title="독일">독일</a>, <a href="/wiki/%EC%9D%B4%ED%83%88%EB%A6%AC%EC%95%84" title="이탈리아">이탈리아</a>, <a href="/wiki/%EC%8A%A4%EC%9C%84%EC%8A%A4" title="스위스">스위스</a> 등 여러 국가의 혈통을 물려받은 국제인이라는 사실을 밝혀내고, “따라서 서재필 박사가 독립문 설계자를 <a href="/wiki/%EC%8A%A4%EC%9C%84%EC%8A%A4" title="스위스">스위스</a>인 기사라고 언급한 대목이 전혀 근거 없는 것은 아니다”라고 밝혔다.<sup class="reference" id="cite_ref-swiss_7-1"><a href="#cite_note-swiss-7">[7]</a></sup></p>
