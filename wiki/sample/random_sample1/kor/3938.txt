<p><b>독일계 미국인</b>(German American)은 <a href="/wiki/%EB%AF%B8%EA%B5%AD" title="미국">미국</a>에 거주하고 있는 <a href="/wiki/%EB%8F%85%EC%9D%BC" title="독일">독일</a>계 사람들을 가리키는 말이다. 미국 인구에서 약 5천만 명, 곧 약 17%를 차지하며 미국에서 가장 큰 자기 보고된 조상 전래의 민족단이다.<sup class="reference" id="cite_ref-4"><a href="#cite_note-4">[4]</a></sup> <a href="/wiki/%EC%98%A4%ED%95%98%EC%9D%B4%EC%98%A4_%EC%A3%BC" title="오하이오 주">오하이오 주</a>, <a href="/wiki/%EC%95%84%EC%9D%B4%EC%98%A4%EC%99%80_%EC%A3%BC" title="아이오와 주">아이오와 주</a>, <a href="/wiki/%EB%AF%B8%EB%84%A4%EC%86%8C%ED%83%80_%EC%A3%BC" title="미네소타 주">미네소타 주</a>, <a href="/wiki/%EC%9C%84%EC%8A%A4%EC%BD%98%EC%8B%A0_%EC%A3%BC" title="위스콘신 주">위스콘신 주</a>, <a href="/wiki/%EB%84%A4%EB%B8%8C%EB%9E%98%EC%8A%A4%EC%B9%B4_%EC%A3%BC" title="네브래스카 주">네브래스카 주</a>와 <a class="mw-disambig" href="/wiki/%EB%8B%A4%EC%BD%94%ED%83%80" title="다코타">다코타</a> 지방을 포함한 중서부 지방 상류 주들이 3분의 1 이상의 가장 높은 독일계 미국인들의 비율을 가지고 있어도 <a href="/wiki/%EC%BA%98%EB%A6%AC%ED%8F%AC%EB%8B%88%EC%95%84_%EC%A3%BC" title="캘리포니아 주">캘리포니아 주</a>, <a href="/wiki/%ED%85%8D%EC%82%AC%EC%8A%A4_%EC%A3%BC" title="텍사스 주">텍사스 주</a>와 <a href="/wiki/%ED%8E%9C%EC%8B%A4%EB%B2%A0%EC%9D%B4%EB%8B%88%EC%95%84_%EC%A3%BC" title="펜실베이니아 주">펜실베이니아 주</a>가 독일계 주민들이 가장 많이 살고 있는 편이다.</p>
<p><a href="/wiki/2010%EB%85%84" title="2010년">2010년</a> <a href="/wiki/%EB%AF%B8%EA%B5%AD_%EC%9D%B8%EA%B5%AC%EC%A1%B0%EC%82%AC%EA%B5%AD" title="미국 인구조사국">미국 인구조사국</a>에 의하면 독일계 미국인이 총 미국 인구의 17.1%를 차지하며, <a href="/wiki/%EC%95%84%EC%9D%BC%EB%9E%9C%EB%93%9C%EA%B3%84_%EB%AF%B8%EA%B5%AD%EC%9D%B8" title="아일랜드계 미국인">아일랜드계 미국인</a>, <a href="/wiki/%EC%95%84%ED%94%84%EB%A6%AC%EC%B9%B4%EA%B3%84_%EB%AF%B8%EA%B5%AD%EC%9D%B8" title="아프리카계 미국인">아프리카계 미국인</a>, <a href="/wiki/%EC%9E%89%EA%B8%80%EB%9E%9C%EB%93%9C%EA%B3%84_%EB%AF%B8%EA%B5%AD%EC%9D%B8" title="잉글랜드계 미국인">잉글랜드계 미국인</a>을 앞서 가장 큰 조상 전래의 민족 단체를 형성하였다.</p>
<p>역사적인 독일의 주들이 해외 <a href="/wiki/%EC%8B%9D%EB%AF%BC%EC%A7%80" title="식민지">식민지</a>들을 가지지 않았고, 1680년대 <a href="/wiki/%EB%B6%81%EC%95%84%EB%A9%94%EB%A6%AC%EC%B9%B4" title="북아메리카">북아메리카</a>의 <a href="/wiki/%EC%98%81%EA%B5%AD" title="영국">영국</a> 식민지들에 첫 현저한 독일인 이민들이 도착하여 <a class="mw-redirect" href="/wiki/%ED%8E%9C%EC%8B%A4%EB%B2%A0%EC%9D%B4%EB%8B%88%EC%95%84" title="펜실베이니아">펜실베이니아</a>와 <a href="/wiki/%EB%89%B4%EC%9A%95" title="뉴욕">뉴욕</a> 지역에 주로 정착하였다. 독일인들의 이민은 19세기 동안에 8백만 명의 가장 큰 수로서 지속하였다. 그들은 대지의 인력과 종교적 자유에 의하여 당겨졌고, 대지의 부족과 종교적 및 정치적 압박에 의하여 <a href="/wiki/%EC%9C%A0%EB%9F%BD" title="유럽">유럽</a>의 외부로 밀려났다. 미국에 도착한 많은 이들은 종교적 밑 정치적 자유를, 어떤 이들은 유럽보다 더 나은 경제적 기회들을, 다른이들은 신대륙에서 간단히 깨끗하게 시작되는 기회를 위한 길을 찾았다. <a href="/wiki/1850%EB%85%84" title="1850년">1850년</a> 이전에 도착한 이민들은 자신들의 강한 농장 기술들이 지불할 만한 가장 생산적 대지를 찾은 농부들이었다. <a href="/wiki/1840%EB%85%84" title="1840년">1840년</a> 후에 많은 이민들은 <a href="/wiki/%EB%8F%85%EC%9D%BC%EC%96%B4" title="독일어">독일어</a> 사용 구역 "게르마니아"가 곧 생긴 대도시들에 왔다.<sup class="reference" id="cite_ref-5"><a href="#cite_note-5">[5]</a></sup><sup class="reference" id="cite_ref-6"><a href="#cite_note-6">[6]</a></sup><sup class="reference" id="cite_ref-7"><a href="#cite_note-7">[7]</a></sup></p>
<p>독일계 미국인들은 미국에 첫 <a href="/wiki/%EC%9C%A0%EC%B9%98%EC%9B%90" title="유치원">유치원</a>을 설립하였고,<sup class="reference" id="cite_ref-8"><a href="#cite_note-8">[8]</a></sup> <a href="/wiki/%ED%81%AC%EB%A6%AC%EC%8A%A4%EB%A7%88%EC%8A%A4_%ED%8A%B8%EB%A6%AC" title="크리스마스 트리">크리스마스 트리</a> 전통을 소개하였으며,<sup class="reference" id="cite_ref-9"><a href="#cite_note-9">[9]</a></sup><sup class="reference" id="cite_ref-10"><a href="#cite_note-10">[10]</a></sup> <a href="/wiki/%ED%95%AB%EB%8F%84%EA%B7%B8" title="핫도그">핫도그</a>와 <a href="/wiki/%ED%96%84%EB%B2%84%EA%B1%B0" title="햄버거">햄버거</a> 같은 인기있는 미국식 음식들을 창설하였다.<sup class="reference" id="cite_ref-11"><a href="#cite_note-11">[11]</a></sup></p>
<p>독일계 미국인들의 축하식은 전국을 통하여 열리고, <a href="/wiki/%EB%89%B4%EC%9A%95" title="뉴욕">뉴욕</a>에서 매 <a href="/wiki/9%EC%9B%94" title="9월">9월</a>마다 세 번째 <a href="/wiki/%ED%86%A0%EC%9A%94%EC%9D%BC" title="토요일">토요일</a>에 열리는 독일계 미국인 슈토이벤 행렬식이 가장 잘 알려진 것들 가운데 하나이다. 또한 <a href="/wiki/%EC%8B%9C%EC%B9%B4%EA%B3%A0" title="시카고">시카고</a>, <a href="/wiki/%EC%8B%A0%EC%8B%9C%EB%82%B4%ED%8B%B0" title="신시내티">신시내티</a>, <a href="/wiki/%EB%B0%80%EC%9B%8C%ED%82%A4" title="밀워키">밀워키</a>, <a href="/wiki/%EC%84%B8%EC%9D%B8%ED%8A%B8%EB%A3%A8%EC%9D%B4%EC%8A%A4" title="세인트루이스">세인트루이스</a>와 다를 도시들에서 해마다 열리는 주요 이벤트들이 있다. 미국에 온 다른 이민들처럼 압도적인 독일인 혹은 독일계 주민들은 본질적으로 미국화되었다.</p>

<p><a href="/wiki/1607%EB%85%84" title="1607년">1607년</a> <a href="/wiki/%EB%B2%84%EC%A7%80%EB%8B%88%EC%95%84_%EC%A3%BC" title="버지니아 주">버지니아 주</a> <a class="mw-disambig" href="/wiki/%EC%A0%9C%EC%9E%84%EC%8A%A4%ED%83%80%EC%9A%B4" title="제임스타운">제임스타운</a>에 도착한 첫 <a href="/wiki/%EC%98%81%EA%B5%AD%EC%9D%B8" title="영국인">영국인</a> 정착자들은 첫 독일계 미국인 요하네스 플라이셔 박사에 의하여 동행되었다. 플라이셔 박사는 <a href="/wiki/1608%EB%85%84" title="1608년">1608년</a> 5명의 <a href="/wiki/%EC%9C%A0%EB%A6%AC" title="유리">유리</a> 제조업자들과 3명의 <a href="/wiki/%EB%AA%A9%EA%B3%B5" title="목공">목공</a>인들 혹은 가옥 <a href="/wiki/%EA%B1%B4%EC%B6%95%EA%B0%80" title="건축가">건축가</a>들에 의하여 수행되었다.<sup class="reference" id="cite_ref-12"><a href="#cite_note-12">[12]</a></sup> 미국이 된 첫 영구적 독일인 정착은 <a href="/wiki/%ED%8E%9C%EC%8B%A4%EB%B2%A0%EC%9D%B4%EB%8B%88%EC%95%84_%EC%A3%BC" title="펜실베이니아 주">펜실베이니아 주</a> 저먼타운이었으며, <a href="/wiki/1683%EB%85%84" title="1683년">1683년</a> <a href="/wiki/10%EC%9B%94_6%EC%9D%BC" title="10월 6일">10월 6일</a> <a href="/wiki/%ED%95%84%EB%9D%BC%EB%8D%B8%ED%94%BC%EC%95%84" title="필라델피아">필라델피아</a> 근처에서 창립되었다.</p>
<p>독일인들의 큰 수들이 1680년대부터 1760년대까지 펜실베이니아가 호의적 목적지로 되면서 이주하였다. 그들은 여러 이유들로 아메리카에 이주하였는데, 그 예를 들면, 추천 도매상들이 중유럽에서 농장 소유를 위한 악화된 기회들에 연루, 어떤 종교 단체들의 박해와 군사적 징병제, 인력 도매상들이 더 나은 경제적 상태(특히 소유적 대지에 기회와 종교적 자유)이다. 가끔 이민자들은 계약적 하인들로서 세월들의 기간을 위하여 자신들의 노동력을 팔면서 자신들의 통행을 위하여 돈을 냈다.<sup class="reference" id="cite_ref-13"><a href="#cite_note-13">[13]</a></sup></p>
<p>펜실베이니아의 큰 구역들과 <a href="/wiki/%EB%89%B4%EC%9A%95" title="뉴욕">뉴욕</a>의 상부는 독일인들을 끌어들였다. 대부분은 <a href="/wiki/%EB%A3%A8%ED%84%B0%EA%B5%90" title="루터교">루터교</a>와 독일 <a href="/wiki/%EA%B0%9C%ED%98%81_%EA%B5%90%ED%9A%8C" title="개혁 교회">개혁 교회</a> 신자들이었으며, 많은 이들은 <a class="mw-redirect" href="/wiki/%EB%AA%A8%EB%9D%BC%EB%B9%84%EC%95%84" title="모라비아">모라비아</a> 교도들과 <a href="/wiki/%EB%A9%94%EB%85%B8%EB%82%98%EC%9D%B4%ED%8A%B8" title="메노나이트">메노나이트</a> 교도들 같은 작은 종교적 종파에 속하였다. 독일의 <a href="/wiki/%EB%A1%9C%EB%A7%88_%EA%B0%80%ED%86%A8%EB%A6%AD%EA%B5%90%ED%9A%8C" title="로마 가톨릭교회">로마 가톨릭교회</a> 교도들은 <a href="/wiki/1812%EB%85%84_%EC%A0%84%EC%9F%81" title="1812년 전쟁">1812년 전쟁</a> 이후 때까지 도착하지 않았다.<sup class="reference" id="cite_ref-Conzen_14-0"><a href="#cite_note-Conzen-14">[14]</a></sup></p>
<h3><span class="mw-headline" id=".ED.8C.94.EC.B8.A0.EC.9D.B8.EB.93.A4">팔츠인들</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%8F%85%EC%9D%BC%EA%B3%84_%EB%AF%B8%EA%B5%AD%EC%9D%B8&amp;action=edit&amp;section=3" title="부분 편집: 팔츠인들">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p><a href="/wiki/1709%EB%85%84" title="1709년">1709년</a> 독일의 <a href="/wiki/%ED%8C%94%EC%B8%A0" title="팔츠">팔츠</a> 지방에서 온 독일의 <a class="mw-redirect" href="/wiki/%ED%94%84%EB%A1%9C%ED%85%8C%EC%8A%A4%ED%83%84%ED%8A%B8" title="프로테스탄트">프로테스탄트</a>들이 곤란의 상태들을 탈출하여 처음에 <a href="/wiki/%EB%A1%9C%ED%85%8C%EB%A5%B4%EB%8B%B4" title="로테르담">로테르담</a>으로 갔다가, 그러고나서 <a href="/wiki/%EB%9F%B0%EB%8D%98" title="런던">런던</a>으로 갔다. 영국의 <a class="mw-redirect" href="/wiki/%EC%95%A4_%EC%97%AC%EC%99%95" title="앤 여왕">앤 여왕</a>은 그들에게 그녀의 아메리카 식민지들을 얻는 데 도움을 주었다. 음식의 나쁜 품질과 전염되기 쉬운 <a class="new" href="/w/index.php?title=%EB%B0%9C%EB%94%98%ED%8B%B0%ED%91%B8%EC%8A%A4&amp;action=edit&amp;redlink=1" title="발딘티푸스 (없는 문서)">발딘티푸스</a> 병 등의 이유로 여행은 생존하는 데 길고 어려웠다. 많은 이민자들, 특히 어린이들이 <a href="/wiki/1710%EB%85%84" title="1710년">1710년</a> <a href="/wiki/6%EC%9B%94" title="6월">6월</a> 아메리카에 도달하기 전에 사망하였다.<sup class="reference" id="cite_ref-Knittle_15-0"><a href="#cite_note-Knittle-15">[15]</a></sup></p>
<p>생존한 대략 2,100명의 팔츠인 이민은 식민지 시대에 아메리카로 가장 큰 단 하나의 이민이었다. 그들의 대부분은 <a href="/wiki/%ED%97%88%EB%93%9C%EC%8A%A8_%EA%B0%95" title="허드슨 강">허드슨 강</a> 유역의 봉사 캠프들에 정착하여 자신들의 통행을 청산하였다. <a href="/wiki/1711%EB%85%84" title="1711년">1711년</a> 안으로 7개의 마을들이 <a href="/wiki/%EB%A1%9C%EB%B2%84%ED%8A%B8_%EB%A6%AC%EB%B9%99%EC%8A%A4%ED%84%B4" title="로버트 리빙스턴">로버트 리빙스턴</a> 영대 차지에 뉴욕 주에서 설립되었다. <a href="/wiki/1723%EB%85%84" title="1723년">1723년</a> 독일인들은 리틀 폭포들 서부의 모호크 계곡에서 대지를 살 수 있던 첫 유럽인들이 되었다. 100개의 농가들은 버니츠필드 특허권에서 배치되었다. <a href="/wiki/1750%EB%85%84" title="1750년">1750년</a> 안으로 독일은 모호크 강 양쪽에 12 마일(19 킬로미터)의 제거를 차지하였다. <a href="/wiki/%ED%9D%99" title="흙">흙</a>이 가장 좋았으며, 거의 <a class="mw-redirect" href="/wiki/%EB%8F%8C" title="돌">돌</a>과 함께 500척의 집들이 지어졌고 그 지방은 <a class="mw-redirect" href="/wiki/%EC%9D%B8%EB%94%94%EC%96%B8" title="인디언">인디언</a> 습격의 원한에서 번영하였다. 허키머는 오랫동안 "저먼플랫츠"라고 알려지면서 지방에서 가장 잘 알려진 독일인의 정착지들이었다.<sup class="reference" id="cite_ref-Knittle_15-1"><a href="#cite_note-Knittle-15">[15]</a></sup></p>
<p>초기 독일 팔츠 이민들 중에 가장 유명한 인물로는 미국에서 발행의 자유를 위하여 식민주의적 뉴욕에서 싸움을 이끈 편집자 <a class="mw-redirect" href="/wiki/%EC%A1%B4_%ED%94%BC%ED%84%B0_%EC%A0%B1%EA%B1%B0" title="존 피터 젱거">존 피터 젱거</a>였다. 미국 독립 전쟁 이후에 <a href="/wiki/%EB%B0%94%EB%8D%B4" title="바덴">바덴</a>에서 건너온 후기의 이민 <a href="/wiki/%EC%A1%B4_%EC%A0%9C%EC%9D%B4%EC%BB%B5_%EC%95%A0%EC%8A%A4%ED%84%B0" title="존 제이컵 애스터">존 제이컵 애스터</a>는 뉴욕에서 자신의 <a href="/wiki/%EB%AA%A8%ED%94%BC_%EB%AC%B4%EC%97%AD" title="모피 무역">모피 무역</a>과 <a href="/wiki/%EB%B6%80%EB%8F%99%EC%82%B0" title="부동산">부동산</a> 투자들로부터 미국에서 가장 부유한 사람이 되었다.</p>
<h3><span class="mw-headline" id=".EB.A3.A8.EC.9D.B4.EC.A7.80.EC.95.A0.EB.82.98">루이지애나</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%8F%85%EC%9D%BC%EA%B3%84_%EB%AF%B8%EA%B5%AD%EC%9D%B8&amp;action=edit&amp;section=4" title="부분 편집: 루이지애나">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p><a class="mw-redirect" href="/wiki/%EB%AF%B8%EC%8B%9C%EC%8B%9C%ED%94%BC_%ED%9A%8C%EC%82%AC" title="미시시피 회사">미시시피 회사</a>는 <a href="/wiki/1721%EB%85%84" title="1721년">1721년</a> <a href="/wiki/%ED%94%84%EB%9E%91%EC%8A%A4" title="프랑스">프랑스</a>령 <a class="mw-redirect" href="/wiki/%EB%A3%A8%EC%9D%B4%EC%A7%80%EC%95%A0%EB%82%98" title="루이지애나">루이지애나</a>에 수천명의 독일인 개척자들을 정착시켰다. 정착은 독일인들, 최근에 프랑스의 규칙 아래에 떨어진 <a href="/wiki/%EC%95%8C%EC%9E%90%EC%8A%A4" title="알자스">알자스</a> 지방의 독일인들과 <a href="/wiki/%EC%8A%A4%EC%9C%84%EC%8A%A4" title="스위스">스위스</a>인들을 이민가는 데 용기를 주었다. 알자스 지방은 <a href="/wiki/30%EB%85%84_%EC%A0%84%EC%9F%81" title="30년 전쟁">30년 전쟁</a>의 더 위대한 문맥 안으로 프랑스에 팔렸다.</p>
<p><a href="/wiki/%EC%98%88%EC%88%98%ED%9A%8C" title="예수회">예수회</a>의 샤를르부아는 <a href="/wiki/%EC%BA%90%EB%82%98%EB%8B%A4" title="캐나다">캐나다</a>에서 루이지애나로 갔다. 그의 편지는 "팔츠 선제후국에서 자라온 이 9,000명의 독일인들은 <a class="mw-redirect" href="/wiki/%EC%95%84%EC%B9%B8%EC%86%8C" title="아칸소">아칸소</a>에 있었다. 통틀어서 독일인들은 아칸소를 떠났다. 그들은 <a href="/wiki/%EB%89%B4%EC%98%AC%EB%A6%AC%EC%96%B8%EC%8A%A4" title="뉴올리언스">뉴올리언스</a>로 가서 유럽에 통행을 요청하였다. 미시시피 회사는 뉴올리언스에서 위로 약 25 마일(40 킬로미터) 떨어져 있는 <a href="/wiki/%EB%AF%B8%EC%8B%9C%EC%8B%9C%ED%94%BC_%EA%B0%95" title="미시시피 강">미시시피 강</a>의 오른쪽 둑에 있는 부유한 대지들을 독일인들에게 주었다. 지역은 이제 '저먼 코스트'로 알려졌다."</p>
<p>독일인들의 번영하는 인구가 저먼 코스트로 알려진 뉴올리언스로부터 강 상류에 살았다. 그들은 J. 하노 다일러의 "루이지애나:독일인 정착자들을 위한 집" 같은 <a class="mw-redirect" href="/wiki/%ED%8C%9C%ED%94%8C%EB%A0%9B" title="팜플렛">팜플렛</a>들을 통하여 끌어당겨졌다.<sup class="reference" id="cite_ref-J._Hanno_Deiler_16-0"><a href="#cite_note-J._Hanno_Deiler-16">[16]</a></sup></p>
<h3><span class="mw-headline" id=".EB.82.A8.EB.B6.80">남부</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%8F%85%EC%9D%BC%EA%B3%84_%EB%AF%B8%EA%B5%AD%EC%9D%B8&amp;action=edit&amp;section=5" title="부분 편집: 남부">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p><a href="/wiki/1714%EB%85%84" title="1714년">1714년</a>과 <a href="/wiki/1717%EB%85%84" title="1717년">1717년</a> 독일 식민주의자들의 2개의 방향들은 현재 <a href="/wiki/%EB%B2%84%EC%A7%80%EB%8B%88%EC%95%84_%EC%A3%BC" title="버지니아 주">버지니아 주</a> 컬페퍼 근처에 자리잡은 저매너라 불리는 버지니아의 큰 식민지를 설립하였다.<sup class="reference" id="cite_ref-Germanna_Foundation_17-0"><a href="#cite_note-Germanna_Foundation-17">[17]</a></sup></p>
<p><a href="/wiki/%EB%85%B8%EC%8A%A4%EC%BA%90%EB%A1%A4%EB%9D%BC%EC%9D%B4%EB%82%98_%EC%A3%BC" title="노스캐롤라이나 주">노스캐롤라이나 주</a>에서 독일의 모라비아 교도들은 <a href="/wiki/1753%EB%85%84" title="1753년">1753년</a> 노스캐롤라이나의 피드먼트에서 그랜빌 경으로부터 10만 에이커(400 제곱 킬로미터) 가까운 펜실베이니아 주 베들레헴 주위에 살았다. 그들은 특히 현재 윈스턴-세일럼 주위에 있는 지역에서 그 지대에 독일인 정착지들을 설립하였다. 그들을 또한 <a href="/wiki/1759%EB%85%84" title="1759년">1759년</a> 노스캐롤라이나 주의 처음으로 계획된 모라비아 교도 공동체인 "통행의 집"으로 해석된 노스캐롤라이나 주의 베서버라의 과도적인 정착지를 창립하였다. 곧 후로 <a href="/wiki/1766%EB%85%84" title="1766년">1766년</a>에 독일의 모라비아 교도들은 세일럼의 타운과 <a href="/wiki/1772%EB%85%84" title="1772년">1772년</a> 초기의 여성 대학 세일럼 칼리지를 세웠다.</p>
<p>독일 이민들은 <a href="/wiki/%EC%82%AC%EC%9A%B0%EC%8A%A4%EC%BA%90%EB%A1%A4%EB%9D%BC%EC%9D%B4%EB%82%98_%EC%A3%BC" title="사우스캐롤라이나 주">사우스캐롤라이나 주</a><sup class="reference" id="cite_ref-Conzen_14-1"><a href="#cite_note-Conzen-14">[14]</a></sup>의 더치 포크 지역과 <a href="/wiki/%ED%85%8D%EC%82%AC%EC%8A%A4_%EC%A3%BC" title="텍사스 주">텍사스 주</a>, 특히 <a href="/wiki/%EC%98%A4%EC%8A%A4%ED%8B%B4_(%ED%85%8D%EC%82%AC%EC%8A%A4_%EC%A3%BC)" title="오스틴 (텍사스 주)">오스틴</a> 지역을 포함한 <a href="/wiki/%EB%AF%B8%EA%B5%AD_%EB%82%A8%EB%B6%80" title="미국 남부">미국 남부</a>의 다른 지역들에 정착하기도 하였다.</p>
<p>독일의 <a href="/wiki/%EC%9C%A0%EB%8C%80%EC%9D%B8" title="유대인">유대인</a>, 즉 유대계 독일인들은 19세기를 통하여 20세기 초반까지 미국 남부에 이민을 갔다. 그들은 특히 자신들이 가끔 지방적 상인, 가축 거래인, 농업적 상품 교역인, 은행업자와 비지니스 소유인들로 일하였던 도시들과 타운들을 포함한 남부의 많은 지역들에서 작은 유대계 독일인 공동체들을 형성하였다. <a href="/wiki/%EC%95%A8%EB%9D%BC%EB%B0%B0%EB%A7%88_%EC%A3%BC" title="앨라배마 주">앨라배마 주</a>에서 자신의 동생과 함께 <a class="new" href="/w/index.php?title=%EB%A0%88%EB%A8%BC_%EB%B8%8C%EB%9D%BC%EB%8D%94%EC%A6%88_%EC%86%8C%EC%9C%A0%EC%A7%80&amp;action=edit&amp;redlink=1" title="레먼 브라더즈 소유지 (없는 문서)">레먼 브라더즈 소유지</a>를 창립한 핸리 래먼은 부분적으로 유대계 독일인 이민들 중의 두드러진 예이다. 유대계 독일인들은 미국에서 첫 개혁적인 <a href="/wiki/%EC%8B%9C%EB%82%98%EA%B3%A0%EA%B7%B8" title="시나고그">시나고그</a>를 설립한 사우스캐롤라이나 주의 <a href="/wiki/%EC%B0%B0%EC%8A%A4%ED%84%B4_(%EC%82%AC%EC%9A%B0%EC%8A%A4%EC%BA%90%EB%A1%A4%EB%9D%BC%EC%9D%B4%EB%82%98_%EC%A3%BC)" title="찰스턴 (사우스캐롤라이나 주)">찰스턴</a> 같은 도시들에 정착하였다. 유대계 독일인들은 자신들의 시나고그들을 지은 다른 남부의 도시들 등에 정착하기도 하였다.</p>
<h3><span class="mw-headline" id=".EB.89.B4.EC.9E.89.EA.B8.80.EB.9E.9C.EB.93.9C">뉴잉글랜드</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%8F%85%EC%9D%BC%EA%B3%84_%EB%AF%B8%EA%B5%AD%EC%9D%B8&amp;action=edit&amp;section=6" title="부분 편집: 뉴잉글랜드">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p><a href="/wiki/1742%EB%85%84" title="1742년">1742년</a>과 <a href="/wiki/1753%EB%85%84" title="1753년">1753년</a> 사이에 거칠게 1천명의 독일인들이 <a href="/wiki/%EB%A7%A4%EC%82%AC%EC%B6%94%EC%84%B8%EC%B8%A0_%EC%A3%BC" title="매사추세츠 주">매사추세츠 주</a> 브로드 베이(현재의 <a href="/wiki/%EB%A9%94%EC%9D%B8_%EC%A3%BC" title="메인 주">메인 주</a> 월도보로)에 정착하였다. 그들의 집들이 불타고 그들의 이웃들이 인디언들에 의하여 살인당하거나 포로로 잡혀간 후에 많은 식민주의자들이 <a href="/wiki/%EB%B3%B4%EC%8A%A4%ED%84%B4" title="보스턴">보스턴</a>, <a class="mw-redirect" href="/wiki/%EB%85%B8%EB%B0%94%EC%8A%A4%EC%BD%94%EC%83%A4" title="노바스코샤">노바스코샤</a>와 노스캐롤라이나로 도망갔다. 남아있던 독일인들은 농사에 생존하기 힘든 것을 찾아냈고 결국 해운업과 어업으로 돌려졌다.<sup class="reference" id="cite_ref-Faust_18-0"><a href="#cite_note-Faust-18">[18]</a></sup></p>
<h3><span class="mw-headline" id=".ED.8E.9C.EC.8B.A4.EB.B2.A0.EC.9D.B4.EB.8B.88.EC.95.84">펜실베이니아</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%8F%85%EC%9D%BC%EA%B3%84_%EB%AF%B8%EA%B5%AD%EC%9D%B8&amp;action=edit&amp;section=7" title="부분 편집: 펜실베이니아">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p><a href="/wiki/1725%EB%85%84" title="1725년">1725년</a>과 <a href="/wiki/1775%EB%85%84" title="1775년">1775년</a> 사이에 무임 도항 이주자들과 개약적 하인들로서 도착한 이민들과 함께 펜실베이니아로 독일인 이민의 형세가 팽창하였다. <a href="/wiki/1775%EB%85%84" title="1775년">1775년</a> 안으로 독일인들은 펜실베이니아 주의 대략 3분의 1을 구성하였다. 독일인 농부들은 그들의 높은 생산적 동물 경자과 농업적 실습들로 명성을 얻었다. 정치적으로 그들은 후에 <a href="/wiki/%EB%AF%B8%EA%B5%AD_%EB%8F%85%EB%A6%BD_%ED%98%81%EB%AA%85" title="미국 독립 혁명">미국 독립 혁명</a>을 지지한 입법부의 통제를 가진 <a href="/wiki/%ED%80%98%EC%9D%B4%EC%BB%A4" title="퀘이커">퀘이커</a> 교도들이 이끄는 연합에 가입한 <a href="/wiki/1740%EB%85%84" title="1740년">1740년</a>까지 정통적으로 활동하지 않았다. 이럼에 불구하고, 많은 독일인 정착자들은 아마 그들이 자신들의 왕립 대지 양도가 새로운 공화국 정부에 의하여 가져갈 것이라는 이유 혹은 영국인들에게 충성이 자유적 사회에 사는 기회를 마련한 독일 군주 정체라는 이유로 독립 혁명 동안에 영국당원들이었다.<sup class="reference" id="cite_ref-19"><a href="#cite_note-19">[19]</a></sup> 루터교, 개혁 교회, 메노나이트, <a href="/wiki/%EC%95%84%EB%AF%B8%EC%8B%9C%ED%8C%8C" title="아미시파">아미시파</a>와 다른 종파들을 포함한 독일인들은 강한 음악적 문화와 함께 부유한 종교적 생활을 개발하였다. 수집적으로 그들은 <a href="/wiki/%ED%8E%9C%EC%8B%A4%EB%B2%A0%EC%9D%B4%EB%8B%88%EC%95%84_%EB%8D%94%EC%B9%98" title="펜실베이니아 더치">펜실베이니아 더치</a>로 알려지게 되었다.<sup class="reference" id="cite_ref-20"><a href="#cite_note-20">[20]</a></sup><sup class="reference" id="cite_ref-21"><a href="#cite_note-21">[21]</a></sup> 1810년대 이전에는 펜실베이니아에 약간의 독일인 가톨릭 교도들이 있었다.</p>
<p><a href="/wiki/%EC%9E%90%EB%8F%99%EC%B0%A8" title="자동차">자동차</a>와 <a href="/wiki/%EC%88%98%EB%A0%88" title="수레">수레</a> 제조업자들의 조상들 스튜드베이커 형제가 <a href="/wiki/1736%EB%85%84" title="1736년">1736년</a> 단검으로 유명한 <a href="/wiki/%EC%A1%B8%EB%A7%81%EA%B2%90" title="졸링겐">졸링겐</a>으로부터 펜실베이니아에 정착하였다. 그들의 실력들과 함께 그들은 서부로 향하는 변경 개척자들을 싣는 수레들을 만들었고, 그들의 <a href="/wiki/%EB%8C%80%ED%8F%AC" title="대포">대포</a>들은 <a class="mw-disambig" href="/wiki/%EB%82%A8%EB%B6%81_%EC%A0%84%EC%9F%81" title="남북 전쟁">남북 전쟁</a>에서 포병들과 함께 한 북군들에게 마련되었으며 그들의 자동차 회사는 미국에서 가장 큰 회사들 중에 하나가 되어 <a href="/wiki/%EB%AF%B8%EA%B5%AD_%EC%9C%A1%EA%B5%B0" title="미국 육군">육군</a>의 전쟁 노력과 산업적 창립에서 요소였다.</p>
<p>미국 독립 전쟁이 터지자 영국은 식민지군에 대항하는 데 어떤 헤센군들을 모집하는 데 독일의 왕자들과 함께 협정들을 만들었다. 대다수는 <a href="/wiki/%ED%97%A4%EC%84%BC_%EB%8C%80%EA%B3%B5%EA%B5%AD" title="헤센 대공국">헤센 대공국</a>에서 왔다. 많은이들은 영구적으로 미국에 정착한 어떤이들의 농장들에서 포로들이 되었다.</p>
<p><a href="/wiki/1790%EB%85%84" title="1790년">1790년</a> 미국 인구 조사로 봐서, 역사가들은 독일인들이 미국 백인 인구들의 9% 가까이 차지한다고 견적하였다.</p>
<h2><span class="mw-headline" id="19.EC.84.B8.EA.B8.B0">19세기</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%8F%85%EC%9D%BC%EA%B3%84_%EB%AF%B8%EA%B5%AD%EC%9D%B8&amp;action=edit&amp;section=8" title="부분 편집: 19세기">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p><a href="/wiki/1820%EB%85%84" title="1820년">1820년</a>과 <a href="/wiki/%EC%A0%9C1%EC%B0%A8_%EC%84%B8%EA%B3%84_%EB%8C%80%EC%A0%84" title="제1차 세계 대전">제1차 세계 대전</a> 사이에 6백만 명 가까이 되는 독일인들이 미국으로 이주하는 시기 동안에 들어오는 독일 이민들의 가장 큰 흐름이 생겼다. <a href="/wiki/1840%EB%85%84" title="1840년">1840년</a>부터 <a href="/wiki/1880%EB%85%84" title="1880년">1880년</a>까지 그들은 이민 집단 중에 가장 컸다. 독일의 주들에서 일어난 <a href="/wiki/1848%EB%85%84_%ED%98%81%EB%AA%85" title="1848년 혁명">1848년 혁명</a>에 이어서 포티에이터스라고 알려진 정치적 난민들의 바람이 불었다. 그들은 <a href="/wiki/%EC%A0%80%EB%84%90%EB%A6%AC%EC%8A%A4%ED%8A%B8" title="저널리스트">저널리스트</a>, <a class="mw-redirect" href="/wiki/%EC%A0%95%EC%B9%98%EC%9D%B8" title="정치인">정치인</a>, 전문 직업인들을 포함한다. 현명한 포티에이터스는 헨리 빌라드와 칼 셔즈를 포함한다.</p>
<h3><span class="mw-headline" id=".EB.8F.84.EC.8B.9C.EB.93.A4">도시들</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%8F%85%EC%9D%BC%EA%B3%84_%EB%AF%B8%EA%B5%AD%EC%9D%B8&amp;action=edit&amp;section=9" title="부분 편집: 도시들">편집</a><span class="mw-editsection-bracket">]</span></span></h3>

<a href="/wiki/1872%EB%85%84" title="1872년">1872년</a> <a href="/wiki/%EB%8F%85%EC%9D%BC%EC%9D%B8" title="독일인">독일인</a>의 인구 분포를 보이는 <a href="/wiki/%EB%AF%B8%EA%B5%AD" title="미국">미국</a>의 <a href="/wiki/%EC%A7%80%EB%8F%84" title="지도">지도</a></div>

<p><a href="/wiki/%EB%B0%80%EC%9B%8C%ED%82%A4" title="밀워키">밀워키</a>, <a href="/wiki/%EC%8B%A0%EC%8B%9C%EB%82%B4%ED%8B%B0" title="신시내티">신시내티</a>, <a href="/wiki/%EC%84%B8%EC%9D%B8%ED%8A%B8%EB%A3%A8%EC%9D%B4%EC%8A%A4" title="세인트루이스">세인트루이스</a>, <a href="/wiki/%EC%8B%9C%EC%B9%B4%EA%B3%A0" title="시카고">시카고</a>, <a href="/wiki/%EB%89%B4%EC%9A%95" title="뉴욕">뉴욕</a>과 <a href="/wiki/%EB%B3%BC%ED%8B%B0%EB%AA%A8%EC%96%B4" title="볼티모어">볼티모어</a> 같은 도시들은 독일인 이민들의 호의적인 목적지들이었다. <a href="/wiki/1900%EB%85%84" title="1900년">1900년</a> 안으로 <a href="/wiki/%ED%81%B4%EB%A6%AC%EB%B8%94%EB%9E%9C%EB%93%9C" title="클리블랜드">클리블랜드</a>, 밀워키, 호코번과 신시내티의 인구들은 모두 40% 이상이 독일계 미국인들이었다. <a href="/wiki/1910%EB%85%84" title="1910년">1910년</a> <a href="/wiki/%EB%84%A4%EB%B8%8C%EB%9E%98%EC%8A%A4%EC%B9%B4_%EC%A3%BC" title="네브래스카 주">네브래스카 주</a> <a href="/wiki/%EC%98%A4%EB%A7%88%ED%95%98" title="오마하">오마하</a>에서 독일계 미국인의 비율이 57%를 차지하면서 <a href="/wiki/%EC%95%84%EC%9D%B4%EC%98%A4%EC%99%80_%EC%A3%BC" title="아이오와 주">아이오와 주</a>의 듀부크와 대번포트는 더 큰 비율을 가졌다. <a href="/wiki/%EC%9D%B8%EB%94%94%EC%95%A0%EB%82%98_%EC%A3%BC" title="인디애나 주">인디애나 주</a> <a class="mw-redirect" href="/wiki/%ED%8F%AC%ED%8A%B8_%EC%9B%A8%EC%9D%B8" title="포트 웨인">포트 웨인</a> 같은 다른 북부 도시들은 독일계 미국인들이 인구의 30% 이하를 차지한다. 신시내티의 "오버 더 라인" 구역과 <a href="/wiki/%EC%BD%9C%EB%9F%BC%EB%B2%84%EC%8A%A4_(%EC%98%A4%ED%95%98%EC%9D%B4%EC%98%A4_%EC%A3%BC)" title="콜럼버스 (오하이오 주)">콜럼버스</a>의 "저먼 빌리지" 같이 많은 그들의 인구 집중들은 자신들의 유산을 암시한 특유의 이름들을 취득하였다.</p>
<p>가장 좋아하는 목적지는 "독일인들의 아테네"로 알려진 밀워키였다. 이전 국가에서 정치 훈련을 받은 근본적인 독일인들이 이 도시의 <a href="/wiki/%EC%82%AC%ED%9A%8C%EC%A3%BC%EC%9D%98" title="사회주의">사회주의</a> 정치들을 지배하였다. 많은 중개업자들이 <a href="/wiki/%EB%A7%A5%EC%A3%BC" title="맥주">맥주</a> 양조업을 창립하는 동안에 능력있는 근로자들은 많은 기능들을 지배하였다.</p>
<p>도시들에 정착한 독일인 이민들의 절반에 반하여, 다른 절반들은 중서부에 농장들을 설립하였다. 21세기에 들어서 <a href="/wiki/%EC%98%A4%ED%95%98%EC%9D%B4%EC%98%A4_%EC%A3%BC" title="오하이오 주">오하이오 주</a>에서 평원 주들까지 시골 지역들에 무거운 존재들이 지속되었다. 어떤 독일인들은 뉴올리언스에서 떨어진 최남부 지방에 정착하였다.</p>
<h3><span class="mw-headline" id=".ED.85.8D.EC.82.AC.EC.8A.A4">텍사스</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%8F%85%EC%9D%BC%EA%B3%84_%EB%AF%B8%EA%B5%AD%EC%9D%B8&amp;action=edit&amp;section=10" title="부분 편집: 텍사스">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<div class="thumb tright">
<div class="thumbinner" style="width:222px;"><a class="image" href="/wiki/%ED%8C%8C%EC%9D%BC:WahrenbergerHouseAustinTX.JPG"><img alt="" class="thumbimage" data-file-height="1704" data-file-width="2272" height="165" src="//upload.wikimedia.org/wikipedia/commons/thumb/5/5f/WahrenbergerHouseAustinTX.JPG/220px-WahrenbergerHouseAustinTX.JPG" srcset="//upload.wikimedia.org/wikipedia/commons/thumb/5/5f/WahrenbergerHouseAustinTX.JPG/330px-WahrenbergerHouseAustinTX.JPG 1.5x, //upload.wikimedia.org/wikipedia/commons/thumb/5/5f/WahrenbergerHouseAustinTX.JPG/440px-WahrenbergerHouseAustinTX.JPG 2x" width="220"/></a>

독일계 미국인 학교로 쓰였던 <a href="/wiki/%ED%85%8D%EC%82%AC%EC%8A%A4_%EC%A3%BC" title="텍사스 주">텍사스 주</a> <a href="/wiki/%EC%98%A4%EC%8A%A4%ED%8B%B4_(%ED%85%8D%EC%82%AC%EC%8A%A4_%EC%A3%BC)" title="오스틴 (텍사스 주)">오스틴</a>에 있는 워렌버거 하우스</div>

<p><a href="/wiki/%ED%85%8D%EC%82%AC%EC%8A%A4_%EC%A3%BC" title="텍사스 주">텍사스 주</a>는 <a href="/wiki/%EA%B0%A4%EB%B2%84%EC%8A%A4%ED%84%B4" title="갤버스턴">갤버스턴</a>과 인디애놀라를 통하여 들어와 양 타운의 농장으로 오고, 후의 더 많은 이민들이 <a href="/wiki/%ED%9C%B4%EC%8A%A4%ED%84%B4" title="휴스턴">휴스턴</a> 같은 도시들에서 빠르게 공업적 직업들을 가진 많은 독일인들을 끌어들였다. 밀워키에서 같이 휴스턴의 독일인들은 양조업을 지었다. 1920년대 안으로 대학에서 교육을 받은 독일계 미국인들의 첫 세대는 화학과 석유 산업으로 이동해 들어갔다.</p>
<p>1850년대에 텍사스는 약 2만명의 독일계 미국인들이 있었다. 그들은 유니폼 블록을 형성하지 않았으나, 아주 몇몇 안되는 귀족들이나 상중류 클래스의 비지니스맨들을 제외하고는 지리적 지역들과 유럽인 사회의 모든 부문들로부터 높게 다양하고 비겼다.</p>
<h3><span class="mw-headline" id=".EB.9F.AC.EC.8B.9C.EC.95.84.EC.97.90.EC.84.9C_.EA.B1.B4.EB.84.88.EC.98.A8_.EB.8F.85.EC.9D.BC.EC.9D.B8.EB.93.A4">러시아에서 건너온 독일인들</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%8F%85%EC%9D%BC%EA%B3%84_%EB%AF%B8%EA%B5%AD%EC%9D%B8&amp;action=edit&amp;section=11" title="부분 편집: 러시아에서 건너온 독일인들">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<h4><span class="mw-headline" id=".EC.A0.84.ED.86.B5">전통</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%8F%85%EC%9D%BC%EA%B3%84_%EB%AF%B8%EA%B5%AD%EC%9D%B8&amp;action=edit&amp;section=12" title="부분 편집: 전통">편집</a><span class="mw-editsection-bracket">]</span></span></h4>
<p><a href="/wiki/%EB%9F%AC%EC%8B%9C%EC%95%84" title="러시아">러시아</a>에서 온 독일인들은 독일어를 사용하는 도착자들의 가장 전통적이었다. 그들은 <a href="/wiki/%EB%9F%AC%EC%8B%9C%EC%95%84_%EC%A0%9C%EA%B5%AD" title="러시아 제국">러시아 제국</a>을 통하여 많은 세대들을 살아온 독일인들이었으나 특히 <a href="/wiki/%EB%B3%BC%EA%B0%80_%EA%B0%95" title="볼가 강">볼가 강</a> 유역과 현재 <a href="/wiki/%EC%9A%B0%ED%81%AC%EB%9D%BC%EC%9D%B4%EB%82%98" title="우크라이나">우크라이나</a>에 속하는 <a href="/wiki/%ED%81%AC%EB%A6%BC_%EB%B0%98%EB%8F%84" title="크림 반도">크림 반도</a> 근처에 살았다. 그들의 조상들은 <a href="/wiki/1762%EB%85%84" title="1762년">1762년</a>과 <a href="/wiki/1763%EB%85%84" title="1763년">1763년</a> <a class="mw-redirect" href="/wiki/%EC%98%88%EC%B9%B4%ED%85%8C%EB%A6%AC%EB%82%98_%EB%8C%80%EC%A0%9C" title="예카테리나 대제">예카테리나 대제</a>에 의하여 초대를 받아 독일어를 사용하는 모든 세계에서 왔으며, 러시아의 시골 지역에 정착하여 더 진보한 독일식 농업 방법들을 소개하였다.</p>
<h4><span class="mw-headline" id=".EC.96.91.EC.8B.AC.EC.A0.81_.EB.B3.91.EC.97.AD.EA.B1.B0.EB.B6.80">양심적 병역거부</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%8F%85%EC%9D%BC%EA%B3%84_%EB%AF%B8%EA%B5%AD%EC%9D%B8&amp;action=edit&amp;section=13" title="부분 편집: 양심적 병역거부">편집</a><span class="mw-editsection-bracket">]</span></span></h4>
<p>기독교 <a href="/wiki/%ED%8F%89%ED%99%94%EC%A3%BC%EC%9D%98" title="평화주의">평화주의</a>자들인 <a href="/wiki/%EB%A9%94%EB%85%B8%EB%82%98%EC%9D%B4%ED%8A%B8" title="메노나이트">메노나이트</a>들은 19세기에 러시아 정부의 양심적 병역거부 탄압때문에<a href="/wiki/1900%EB%85%84" title="1900년">1900년</a>에 대략 10만 명이 이민을 떠나 주로 <a class="mw-disambig" href="/wiki/%EB%8B%A4%EC%BD%94%ED%83%80" title="다코타">다코타</a> 지방, <a class="mw-redirect" href="/wiki/%EC%BA%94%EC%9E%90%EC%8A%A4" title="캔자스">캔자스</a>와 <a class="mw-redirect" href="/wiki/%EB%84%A4%EB%B8%8C%EB%9E%98%EC%8A%A4%EC%B9%B4" title="네브래스카">네브래스카</a>에 정착하였다. <a href="/wiki/%EB%85%B8%EC%8A%A4%EB%8B%A4%EC%BD%94%ED%83%80_%EC%A3%BC" title="노스다코타 주">노스다코타 주</a>의 남부중앙부는 "독일-러시아의 삼각형"으로 알려졌다. 작은 수들은 서부로 멀리 이주하여 <a href="/wiki/%EC%B9%B4%EC%9A%B0%EB%B3%B4%EC%9D%B4" title="카우보이">카우보이</a>와 목장주같은 직업을 찾았다.</p>
<p>부정적으로 <a href="/wiki/%EC%B0%A8%EB%A5%B4" title="차르">차르</a>에 의하여 그들의 권리들의 폭력과 문화적 박해에 의한 영향으로 미국 중서부의 북부에 정착한 러시아에서 건너온 독일인들은 러시아계 미국인들로부터 갈라진 학대받는 민족으로 자신을 보았고, 독일 땅으로부터 이민을 온 독일계 미국인으로부터 아주 다른 경험을 겪어 자신들의 독일어와 문화를 유지한 빈틈없는 공동체들에 정착하였다. 그들은 큰 가족들을 키웠고, 독일식 교회들을 지어 자신들의 시체를 특유의 묘지들에 안장하는 데 주형의 철 무덤 묘석을 이용하였으며 독일 교회 <a class="mw-redirect" href="/wiki/%EC%B0%AC%EC%86%A1%EA%B0%80" title="찬송가">찬송가</a>를 부르는 합창단을 창조하였다. 많은 농부들은 아직도 <a class="mw-redirect" href="/wiki/%EB%8C%80%ED%8F%89%EC%9B%90" title="대평원">대평원</a> 상부에서 나는 주요 농작물 <a href="/wiki/%EC%82%AC%ED%83%95%EB%AC%B4" title="사탕무">사탕무</a>에 전문적이었다. <a href="/wiki/%EC%A0%9C1%EC%B0%A8_%EC%84%B8%EA%B3%84_%EB%8C%80%EC%A0%84" title="제1차 세계 대전">제1차 세계 대전</a> 중에 그들의 신분은 반독 감정에 의하여 도전되었다. <a href="/wiki/%EC%A0%9C2%EC%B0%A8_%EC%84%B8%EA%B3%84_%EB%8C%80%EC%A0%84" title="제2차 세계 대전">제2차 세계 대전</a>이 끝나면서 공공적과 공식적 일들에서 영어와 함께 쓰이던 독일어가 심각한 거절에 놓였다. 오늘날 독일어는 영어를 우세하게 사용한 대평원 주들의 북부에 사는 러시아에서 건너온 독일인들과 함께 주로 노래부르는 단체들과 방법들을 통하여 보존되었다. 독일어는 다코타 지방에서 두 번째로 많이 쓰이는 언어로 남아있다. 자신들의 언어 손해에 불구하고 그들은 뚜렷하게 남아 미국 서부에 오래가는 감독을 남겼다.</p>
<h3><span class="mw-headline" id=".EB.82.A8.EB.B6.81_.EC.A0.84.EC.9F.81">남북 전쟁</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%8F%85%EC%9D%BC%EA%B3%84_%EB%AF%B8%EA%B5%AD%EC%9D%B8&amp;action=edit&amp;section=14" title="부분 편집: 남북 전쟁">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>독일계 미국인들 사이에 감정은 특히 포티에이터들 사이에 반노예주의였다. 수십만명의 독일계 미국인들은 <a class="mw-disambig" href="/wiki/%EB%82%A8%EB%B6%81_%EC%A0%84%EC%9F%81" title="남북 전쟁">남북 전쟁</a>에서 북군을 위하여 싸우는 데 지원되었다. 독일인들은 남북 전쟁에 참전하는 데 가장 큰 이민 단체들이었으며, 17만 6천명 이상의 미국 군인들이 독일에서 태어났다. 독일인들 사이에 인기있던 북군 사령관 프란츠 지겔 <a href="/wiki/%EC%86%8C%EB%A0%B9" title="소령">소령</a>은 "지겔과 함께 싸움"에 입대하는 데 주장된 많은 독일 이민들과 함께 북군의 최고 계급 독일인 사령관이었다.</p>
<p>4명의 독일인들 중에 1명이 전 독일인 연대에서 싸웠어도 그들은 독일 군인의 공동적 이미지를 창조하였다. 펜실베이니아는 5개, 뉴욕은 11개, 오하이오는 6개의 독일인 연대들을 주둔하였다.</p>
<h3><span class="mw-headline" id=".EB.86.8D.EC.9E.A5.EB.93.A4">농장들</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%8F%85%EC%9D%BC%EA%B3%84_%EB%AF%B8%EA%B5%AD%EC%9D%B8&amp;action=edit&amp;section=15" title="부분 편집: 농장들">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>많은 농부들을 끌어들이는 데 유용한 큰 토지 승인들과 함께 서부 철도들은 <a href="/wiki/%ED%95%A8%EB%B6%80%EB%A5%B4%ED%81%AC" title="함부르크">함부르크</a>와 다른 독일의 도시들에 대리인들을 세워 값이 싼 교통과 쉬운 기간들에 농장지의 판매들을 약속하였다. 예를 들어 <a class="mw-redirect" href="/wiki/%EC%83%8C%ED%83%80%ED%8E%98%EC%9D%B4" title="샌타페이">샌타페이</a> 철도는 이민을 위하여 자신들의 감독관을 기용하였고 30만 에이커(1,200 제곱 킬로미터) 이상을 독일어 사용자 농부들에게 팔았다.</p>
<p>19세기와 20세기를 걸쳐 독일계 미국인들은 농부들이 되는 데 높은 흥미를 보였고 대지에 자신들의 자식들과 손자·손녀들을 지켰다. 그들이 운영하는 데 이득들이 필요하였을 때에 그들은 이득들을 가족의 지속을 유지하는 데 연장으로 사용하였다. 그들은 위험을 반대하는 전략들을 사용하여 조심스럽게 가족에서 대지를 지키는 데 자신들의 상속을 계획세웠다. 그들의 공동체들은 더 작은 평균의 농장 크기, 더 위대한 동등권, 적은 부재 소유권과 더 위대한 지리적 인내력을 보였다.</p>
<h3><span class="mw-headline" id=".EC.A0.95.EC.B9.98">정치</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%8F%85%EC%9D%BC%EA%B3%84_%EB%AF%B8%EA%B5%AD%EC%9D%B8&amp;action=edit&amp;section=16" title="부분 편집: 정치">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<h4><span class="mw-headline" id=".EB.85.B8.EB.8F.99.EC.9A.B4.EB.8F.99">노동운동</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%8F%85%EC%9D%BC%EA%B3%84_%EB%AF%B8%EA%B5%AD%EC%9D%B8&amp;action=edit&amp;section=17" title="부분 편집: 노동운동">편집</a><span class="mw-editsection-bracket">]</span></span></h4>
<p>19세기 후반의 많은 독일계 미국인들은 <a class="mw-redirect" href="/wiki/%EB%AC%B4%EC%A0%95%EB%B6%80%EC%A3%BC%EC%9D%98" title="무정부주의">무정부주의</a>자들이나 <a class="mw-redirect" href="/wiki/%EC%82%AC%ED%9A%8C%EC%A3%BC%EC%9D%98%EC%9E%90" title="사회주의자">사회주의자</a>들이었다. <a href="/wiki/1886%EB%85%84" title="1886년">1886년</a> 5월 1일 8시간 노동쟁취를 위한 <a href="/wiki/%EC%8B%9C%EC%B9%B4%EA%B3%A0" title="시카고">시카고</a> 총<a href="/wiki/%ED%8C%8C%EC%97%85" title="파업">파업</a>이 있었다. 당시 경찰에서는 노동자들을 살상함에 항의하려고 노동자들이 헤이마켓에서 집회를 가졌다. 이때 폭탄<a href="/wiki/%ED%85%8C%EB%9F%AC" title="테러">테러</a>가 일어나 경찰과 노동자들사이의 싸움이 일어나자 <a class="mw-redirect" href="/wiki/%EC%9E%90%EB%B3%B8%EA%B0%80" title="자본가">자본가</a>들과 결탁한 <a class="new" href="/w/index.php?title=%EC%A7%80%EB%B0%B0%EA%B3%84%EA%B8%89&amp;action=edit&amp;redlink=1" title="지배계급 (없는 문서)">지배계급</a>들은 노동자들이 경찰을 살해했다고 조작하여 8명의 노동운동가들을 <a href="/wiki/%EC%82%AC%EB%B2%95%EC%82%B4%EC%9D%B8" title="사법살인">사법살인</a>한 <a href="/wiki/%ED%97%A4%EC%9D%B4%EB%A7%88%EC%BC%93" title="헤이마켓">헤이마켓</a> 사건이 일어났다. <sup class="reference" id="cite_ref-22"><a href="#cite_note-22">[22]</a></sup>이때 8명의 피고인들 중에 6명이 독일인이었고, <a href="/wiki/%EB%8F%85%EC%9D%BC%EC%9D%B8" title="독일인">독일인</a>들은 초기 미국 <a class="mw-redirect" href="/wiki/%EB%85%B8%EB%8F%99_%EC%9A%B4%EB%8F%99" title="노동 운동">노동 운동</a>에서 중요한 역할을 맡았다.</p>
<h4><span class="mw-headline" id=".EC.84.A0.EA.B1.B0">선거</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%8F%85%EC%9D%BC%EA%B3%84_%EB%AF%B8%EA%B5%AD%EC%9D%B8&amp;action=edit&amp;section=18" title="부분 편집: 선거">편집</a><span class="mw-editsection-bracket">]</span></span></h4>
<p>상대적으로 몇몇의 독일계 미국인들이 사무실을 보유하였으나 남자들이 한번 시민들이 되는 데 투표하였다. 정통으로서 <a class="mw-redirect" href="/wiki/%ED%94%84%EB%A1%9C%ED%85%8C%EC%8A%A4%ED%83%84%ED%8A%B8" title="프로테스탄트">프로테스탄트</a>와 유대인들은 <a href="/wiki/%EA%B3%B5%ED%99%94%EB%8B%B9_(%EB%AF%B8%EA%B5%AD)" title="공화당 (미국)">공화당</a> 편에 들었고 <a href="/wiki/%EA%B0%80%ED%86%A8%EB%A6%AD" title="가톨릭">가톨릭</a> 교도들은 강하게 <a href="/wiki/%EB%AF%BC%EC%A3%BC%EB%8B%B9_(%EB%AF%B8%EA%B5%AD)" title="민주당 (미국)">민주당</a>을 성워하였다. 비밀 투표에서 금지령이 있을 때에 독일인들은 그것에 대항하는 데 고체적으로 투표하였다. 그들은 강하게 금주 개혁자들과 많은 인기당원들을 포함한 도덕적 개혁 운동가들을 불신임하였다. 독일인 공동체는 강하게 은화의 자유 주조를 반대하였고 <a href="/wiki/1896%EB%85%84" title="1896년">1896년</a> 개혁 운동가 <a href="/wiki/%EC%9C%8C%EB%A6%AC%EC%97%84_%EC%A0%9C%EB%8B%9D%EC%8A%A4_%EB%B8%8C%EB%9D%BC%EC%9D%B4%EC%96%B8" title="윌리엄 제닝스 브라이언">윌리엄 제닝스 브라이언</a>에 무겁게 대항하며 투표하였다. <a href="/wiki/1900%EB%85%84" title="1900년">1900년</a> 많은 독일계 민주당원들이 자신들의 당에 복귀하여 브라이언을 위하여 투표하였는데, 아마 <a href="/wiki/%EC%9C%8C%EB%A6%AC%EC%97%84_%EB%A7%A4%ED%82%A8%EB%A6%AC" title="윌리엄 매킨리">윌리엄 매킨리</a> 대통령의 외교 정책 때문이었을 것이다.</p>
<h2><span class="mw-headline" id="20.EC.84.B8.EA.B8.B0">20세기</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%8F%85%EC%9D%BC%EA%B3%84_%EB%AF%B8%EA%B5%AD%EC%9D%B8&amp;action=edit&amp;section=19" title="부분 편집: 20세기">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<h3><span class="mw-headline" id=".EC.A0.9C1.EC.B0.A8_.EC.84.B8.EA.B3.84_.EB.8C.80.EC.A0.84.EA.B3.BC_.EB.B0.98.EB.8F.85_.EA.B0.90.EC.A0.95">제1차 세계 대전과 반독 감정</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%8F%85%EC%9D%BC%EA%B3%84_%EB%AF%B8%EA%B5%AD%EC%9D%B8&amp;action=edit&amp;section=20" title="부분 편집: 제1차 세계 대전과 반독 감정">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p><a href="/wiki/%EC%A0%9C1%EC%B0%A8_%EC%84%B8%EA%B3%84_%EB%8C%80%EC%A0%84" title="제1차 세계 대전">제1차 세계 대전</a> 동안에 독일계 미국인들은 어쩌다 <a href="/wiki/%EB%8F%85%EC%9D%BC_%EC%A0%9C%EA%B5%AD" title="독일 제국">독일 제국</a>에 너무 동정심을 가져 고발을 당하였다. <a href="/wiki/%EC%9A%B0%EB%93%9C%EB%A1%9C_%EC%9C%8C%EC%8A%A8" title="우드로 윌슨">우드로 윌슨</a> 전 대통령은 "외국계 미국인주의"를 비난하여 이중의 충성들이 전쟁 시기에 불가능하였다고 주장하였다. 독일을 위한 작은 소수파가 나오거나 영국인들을 비웃기 마련이었다. 비슷하게 <a href="/wiki/%ED%95%98%EB%B2%84%EB%93%9C_%EB%8C%80%ED%95%99%EA%B5%90" title="하버드 대학교">하버드 대학교</a>의 <a class="mw-redirect" href="/wiki/%EC%A0%95%EC%8B%A0%EA%B3%BC" title="정신과">정신과</a> 교수 휴고 뮨스터버그는 미국과 독일 사이의 중재에 자신의 노력들을 떨어뜨렸고, 자신의 노력들을 독일의 이유 뒤로 던져버렸다.</p>
<p>법무부는 모든 독일 외국인들의 명단을 준비하는 데 시도하여 그들의 대략 48만 명을 세워 4천 명 이상은 1917~18년 사이에 투옥되었다. 변증은 독일을 위한 스파이와 독일의 전쟁 수고를 보증함을 포함하였다. 수천명은 자신들의 충성을 보이는 데 전쟁 보증금을 사는 데 강요되었다. <a class="mw-redirect" href="/wiki/%EC%A0%81%EC%8B%AD%EC%9E%90" title="적십자">적십자</a>는 파괴행위의 위협에 가입으로부터 독일인의 성을 가진 개인들을 제외하였다. 한명은 반란자에 의하여 살인되었는 데, <a href="/wiki/%EC%9D%BC%EB%A6%AC%EB%85%B8%EC%9D%B4_%EC%A3%BC" title="일리노이 주">일리노이 주</a> 콜린스빌에서 독일 출신의 로버트 프라거가 간첩으로서 감옥으로부터 끌려나와 린치당하였다.</p>
<p>시카고에서 프레더릭 스톡은 그가 자신의 귀화 문서들을 최종적으로 완성할 때까지 시카고 심포니 오케스트라의 지휘자로서 임시적으로 사임하였다. 오케스트라단들은 <a href="/wiki/%EB%A6%AC%ED%95%98%EB%A5%B4%ED%8A%B8_%EB%B0%94%EA%B7%B8%EB%84%88" title="리하르트 바그너">바그너</a>와 <a href="/wiki/%EC%97%91%ED%86%A0%EB%A5%B4_%EB%B2%A0%EB%A5%BC%EB%A6%AC%EC%98%A4%EC%A6%88" title="엑토르 베를리오즈">베를리오즈</a>에 의한 음악을 되돌렸다. 신시내티에서는 공공 도서관이 자신들의 책꽂이들로부터 독일어 책들을 거두어 들이는 데 요구되었다. 독일어 이름을 진 거리들이 다시 이름을 지었다. <a href="/wiki/%EB%AF%B8%EC%8B%9C%EA%B0%84_%EC%A3%BC" title="미시간 주">미시간 주</a>의 베를린은 마네(마네의 전투에서 싸운 군사들을 기념)로 이름을 바꾸었다. 아이오와 주에서는 <a href="/wiki/1918%EB%85%84" title="1918년">1918년</a>의 바벨 포고에서 주지사가 학교들과 공공 장소들에서 외국어들을 금지시켰다. 네브래스카 주는 영어를 제외한 모든 언어들에서 지시를 금지시켰으나 <a class="mw-redirect" href="/wiki/%EB%AF%B8%EA%B5%AD_%EB%8C%80%EB%B2%95%EC%9B%90" title="미국 대법원">미국 대법원</a>은 <a href="/wiki/1923%EB%85%84" title="1923년">1923년</a> 금지를 불법으로 규칙을 내렸다. 독일계 미국인들의 이런 전술들에 책임에서 가끔 이름들을 미국화("슈미트"에서 "스미스"로, "뮐러"에서 "밀러"로)하고, 공동 장소들, 특히 교회에서 독일어 사용을 제한시켰다.</p>
<h3><span class="mw-headline" id=".EC.A0.9C2.EC.B0.A8_.EC.84.B8.EA.B3.84_.EB.8C.80.EC.A0.84">제2차 세계 대전</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%8F%85%EC%9D%BC%EA%B3%84_%EB%AF%B8%EA%B5%AD%EC%9D%B8&amp;action=edit&amp;section=21" title="부분 편집: 제2차 세계 대전">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p><a href="/wiki/1931%EB%85%84" title="1931년">1931년</a>부터 <a href="/wiki/1940%EB%85%84" title="1940년">1940년</a>까지 11만 4천명의 독일인들이 미국으로 이주하였는 데 <a href="/wiki/%EB%85%B8%EB%B2%A8%EC%83%81" title="노벨상">노벨상</a> 수상자 <a href="/wiki/%EC%95%8C%EB%B2%A0%EB%A5%B4%ED%8A%B8_%EC%95%84%EC%9D%B8%EC%8A%88%ED%83%80%EC%9D%B8" title="알베르트 아인슈타인">알베르트 아인슈타인</a>을 포함한 그들은 유대계 독일인들이나 <a class="mw-redirect" href="/wiki/%EB%82%98%EC%B9%98%EC%A3%BC%EC%9D%98" title="나치주의">나치주의</a>의 반대자들이었다. 대략 2만 5천명은 전쟁 이전의 세월 동안에 친나치 독일계 미국인 동맹의 회원들이 되었다. 독일 외국인들은 편견과 빗나간 수가 <a href="/wiki/%EC%9D%BC%EB%B3%B8%EA%B3%84_%EB%AF%B8%EA%B5%AD%EC%9D%B8" title="일본계 미국인">일본계 미국인</a>들보다 정통적으로 적은 단체로서 고통을 겪은 의미였어도 전쟁 동안에 혐의와 차별의 주제였다. <a href="/wiki/1940%EB%85%84" title="1940년">1940년</a> 외국인 등록 법령은 연방 정부와 함께 독일 시민권을 등록한 독일 출신 외국인들을 요구하여 그들의 여행과 재산 소유권을 제한시켰다. 아직도 활동적인 <a href="/wiki/1798%EB%85%84" title="1798년">1798년</a>의 외국인 적령 아래에서 미국 정부는 <a href="/wiki/1940%EB%85%84" title="1940년">1940년</a>과 <a href="/wiki/1948%EB%85%84" title="1948년">1948년</a> 사이에 1만 1천명 가까이 되는 독일 시민들을 구금하였다. "지원적 구금인"들의 알 수 없는 수는 수용소들에서 자신들의 배우자들과 부모들에게 가입하여 떠나는 데 허락되지 않았다.</p>
<p><a href="/wiki/%ED%94%84%EB%9E%AD%ED%81%B4%EB%A6%B0_D._%EB%A3%A8%EC%8A%A4%EB%B2%A8%ED%8A%B8" title="프랭클린 D. 루스벨트">프랭클린 D. 루스벨트</a> 대통령은 <a href="/wiki/%EB%93%9C%EC%99%80%EC%9D%B4%ED%8A%B8_D._%EC%95%84%EC%9D%B4%EC%A0%A0%ED%95%98%EC%9B%8C" title="드와이트 D. 아이젠하워">드와이트 D. 아이젠하워</a> 장군, <a href="/wiki/%EC%B2%B4%EC%8A%A4%ED%84%B0_%EB%8B%88%EB%AF%B8%EC%B8%A0" title="체스터 니미츠">체스터 니미츠</a> 해군 대장과 칼 앤드루 스파츠를 포함한 독일계 미국인들에게 정상적인 전쟁 직업을 찾았다. 그는 공화당원 <a href="/wiki/%EC%9B%AC%EB%8D%B8_%EC%9C%8C%ED%82%A4" title="웬델 윌키">웬델 윌키</a>를 개인적 대표로 임명하였다. 독일어에 능통한 독일계 미국인들은 전쟁 시기의 지능으로 중요한 감성이었고 들은 미국을 위한 번역자와 스파이로서 근무하였다. 예로 <a href="/wiki/%ED%97%A8%EB%A6%AC_%ED%82%A4%EC%8B%A0%EC%A0%80" title="헨리 키신저">헨리 키신저</a>가 종군 당시에 통역관을 맡았다.</p>
<p>전쟁은 당시 본국에 친척들과 연락한 몇몇의 독일계 미국인 사이에 강한 친미주의 애국적 감정을 일깨웠다.</p>
<h3><span class="mw-headline" id=".EC.A0.84.EC.9F.81_.EC.9D.B4.ED.9B.84.EC.9D.98_.EC.84.B8.EC.9B.94.EB.93.A4">전쟁 이후의 세월들</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%8F%85%EC%9D%BC%EA%B3%84_%EB%AF%B8%EA%B5%AD%EC%9D%B8&amp;action=edit&amp;section=22" title="부분 편집: 전쟁 이후의 세월들">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>제2차 세계 대전 직후의 시기에 <a class="mw-redirect" href="/wiki/%EC%86%8C%EB%B9%84%EC%97%90%ED%8A%B8_%EC%97%B0%EB%B0%A9" title="소비에트 연방">소비에트 연방</a>, <a href="/wiki/%ED%8F%B4%EB%9E%80%EB%93%9C" title="폴란드">폴란드</a>, <a href="/wiki/%EC%B2%B4%EC%BD%94%EC%8A%AC%EB%A1%9C%EB%B0%94%ED%82%A4%EC%95%84" title="체코슬로바키아">체코슬로바키아</a>, <a href="/wiki/%EB%A3%A8%EB%A7%88%EB%8B%88%EC%95%84" title="루마니아">루마니아</a>, <a href="/wiki/%ED%97%9D%EA%B0%80%EB%A6%AC" title="헝가리">헝가리</a>와 <a href="/wiki/%EC%9C%A0%EA%B3%A0%EC%8A%AC%EB%9D%BC%EB%B9%84%EC%95%84" title="유고슬라비아">유고슬라비아</a>를 포함한 다시 그어진 동유럽의 경계들 안에서 100만명의 독일인들이 추방당하였다. 대부분은 독일에 재정착하였으나 다른이들은 1940년대 후반에 미국에 난민들로 들어와 자신들의 새로운 고향들에 문화적 중심지들을 설립하였다. 후에 유고슬라비아 땅(현재의 <a href="/wiki/%EC%84%B8%EB%A5%B4%EB%B9%84%EC%95%84" title="세르비아">세르비아</a>)이 된 헝가리의 <a href="/wiki/%EB%8F%84%EB%82%98%EC%9A%B0_%EA%B0%95" title="도나우 강">도나우 강</a> 유역에 정착 후에 언어와 풍습들을 유지해 온 독일인들, 이를테면 어떤 도나우 강의 <a href="/wiki/%EC%8A%88%EB%B0%94%EB%B2%A4" title="슈바벤">슈바벤</a>인들이 전쟁 후에 미국으로 이민을 갔다.</p>
<p><a href="/wiki/1970%EB%85%84" title="1970년">1970년</a> 후에 제2차 세계 대전에 의하여 자극된 반독 감정이 사라져갔다. 오늘날 제2차 세계 대전 이후에 이민온 독일계 미국인들은 미국에서 다른 서유럽 이민 단체만큼 같은 특성을 나누고 있다. 그들은 대부분 전문적인 이유들로 온 전문인들과 학회원들이었다. 소비에트 연방의 붕괴와 <a href="/wiki/%EB%8F%85%EC%9D%BC%EC%9D%98_%EC%9E%AC%ED%86%B5%EC%9D%BC" title="독일의 재통일">독일의 재통일</a> 이래 독일은 오히려 이민을 떠나는 사람들보다 이민들의 목적지가 되었다.</p>
<p><a href="/wiki/1990%EB%85%84" title="1990년">1990년</a> 미국 인구 조사에서 5천 8백만의 미국인들이 단독으로 혹은 부분적으로 독일계라고 주장되었다. <a href="/wiki/2005%EB%85%84" title="2005년">2005년</a> 인구 조사에 의하면 5천만의 미국인들이 독일 계통을 가졌다고 한다. 독일계 미국인은 총 미국 인구의 17%, <a href="/wiki/%ED%9E%88%EC%8A%A4%ED%8C%A8%EB%8B%89" title="히스패닉">히스패닉</a>이 아닌 백인 인구의 26%를 차지한다.</p>

<p>독일인들은 미국의 막대한 문화와 기술 지역들에 공헌하였다. 전 <a href="/wiki/%ED%94%84%EB%A1%9C%EC%9D%B4%EC%84%BC" title="프로이센">프로이센</a>의 장교 <a href="/wiki/%ED%94%84%EB%A6%AC%EB%93%9C%EB%A6%AC%ED%9E%88_%EB%B9%8C%ED%97%AC%EB%A6%84_%ED%8F%B0_%EC%8A%88%ED%86%A0%EC%9D%B4%EB%B2%A4" title="프리드리히 빌헬름 폰 슈토이벤">슈토이벤 남작</a>은 <a href="/wiki/%EB%AF%B8%EA%B5%AD_%EB%8F%85%EB%A6%BD_%EC%A0%84%EC%9F%81" title="미국 독립 전쟁">미국 독립 전쟁</a> 동안에 <a href="/wiki/%EB%AF%B8%EA%B5%AD_%EC%9C%A1%EA%B5%B0" title="미국 육군">미국 육군</a>의 재결성을 이끌어 영국군에 승리를 거두었다. 스타인웨이 앤드 선즈 <a href="/wiki/%ED%94%BC%EC%95%84%EB%85%B8" title="피아노">피아노</a> 제조 상사는 <a href="/wiki/1853%EB%85%84" title="1853년">1853년</a> 이민자 헨리 E. 스타인웨이에 의하여 창설되었다. 독일인 정착자들은 미국에 <a href="/wiki/%ED%81%AC%EB%A6%AC%EC%8A%A4%EB%A7%88%EC%8A%A4_%ED%8A%B8%EB%A6%AC" title="크리스마스 트리">크리스마스 트리</a> 전통을 가져왔다. 서부로 이동 동안에 스튜드베이커 형제는 대량의 수레들을 지었는 데, 뒤젠버그 형제처럼 후에 중요한 초기 자동차 제조업자가 되었다. 첫 실패적 <a href="/wiki/1848%EB%85%84_%EB%8F%85%EC%9D%BC_%ED%98%81%EB%AA%85" title="1848년 독일 혁명">1848년 독일 혁명</a>으로부터 난민으로 들어온 칼 셔즈는 영향적인 정치인이 되어 처음에 공화당에서, 후에 민주당으로 옮겨 내무 장관을 지냈다.</p>
<p><a href="/wiki/%EC%A0%9C2%EC%B0%A8_%EC%84%B8%EA%B3%84_%EB%8C%80%EC%A0%84" title="제2차 세계 대전">제2차 세계 대전</a> 후에 <a href="/wiki/%EB%B2%A0%EB%A5%B4%EB%84%88_%ED%8F%B0_%EB%B8%8C%EB%9D%BC%EC%9A%B4" title="베르너 폰 브라운">베르너 폰 브라운</a>과 펜네뮌데에 있는 전 독일 V-2 <a class="mw-redirect" href="/wiki/%EB%A1%9C%EC%BC%80%ED%8A%B8" title="로케트">로케트</a> 기지에서 온 대부분의 일류 공학자들이 미국으로 데려왔다. 그들은 <a class="mw-redirect" href="/wiki/%EB%AF%B8%EA%B5%AD_%ED%95%AD%EA%B3%B5_%EC%9A%B0%EC%A3%BC%EA%B5%AD" title="미국 항공 우주국">미국 항공 우주국</a>의 프로그램을 위한 로케트들은 물론 미국 군사 로케트들의 개발에 중대하게 공헌하였다.</p>
<p>독일 음식의 영향은 전국에 걸쳐 미국의 음식에 보였는 데, 특히 <a class="mw-redirect" href="/wiki/%ED%8C%A8%EC%8A%A4%ED%8A%B8%EB%A6%AC" title="패스트리">패스트리</a>, <a class="mw-redirect" href="/wiki/%EC%9C%A1%EB%A5%98" title="육류">육류</a>, <a href="/wiki/%EC%86%8C%EC%8B%9C%EC%A7%80" title="소시지">소시지</a>와 <a href="/wiki/%EB%A7%A5%EC%A3%BC" title="맥주">맥주</a>로 여겨진다. 햄버거, <a href="/wiki/%EC%86%8C%EA%B8%88" title="소금">소금</a>에 절인 <a href="/wiki/%EC%96%91%EB%B0%B0%EC%B6%94" title="양배추">양배추</a>, 스튜루델(<a href="/wiki/%EA%B3%BC%EC%9D%BC" title="과일">과일</a>, <a href="/wiki/%EC%B9%98%EC%A6%88" title="치즈">치즈</a> 등을 <a href="/wiki/%EB%B0%80%EA%B0%80%EB%A3%A8" title="밀가루">밀가루</a> 반죽으로 얇게 싸서 화덕에 구운 과자) 등이 공동적 요리이다. 독일의 빵 굽는 사람들이 <a href="/wiki/%EB%B9%84%EC%8A%A4%ED%82%B7" title="비스킷">비스킷</a>의 일종인 프레첼을 소개하였다. <a href="/wiki/1850%EB%85%84" title="1850년">1850년</a> 이래 독일인들은 맥주 양조업에 지배적인 민족으로 지내왔다.</p>
<p><a href="/wiki/%EB%B0%80%EC%9B%8C%ED%82%A4" title="밀워키">밀워키</a>는 한번 세계에서 가장 큰 독일 양조사들 중에 4개의 고향이었다 - 쉴리츠, 블라츠, 팝스트와 밀러. 그리고 수많은 세월에 세계에서 최고 맥주 생산 도시였다. 미국에서 현재 팔리고 있는 맥주의 절반이 A. 팝스트와 <a href="/wiki/1860%EB%85%84" title="1860년">1860년</a> <a href="/wiki/%EC%84%B8%EC%9D%B8%ED%8A%B8%EB%A3%A8%EC%9D%B4%EC%8A%A4" title="세인트루이스">세인트루이스</a>에서 <a href="/wiki/%EB%B2%84%EB%93%9C%EC%99%80%EC%9D%B4%EC%A0%80" title="버드와이저">버드와이저</a>를 창립한 에베르하르트 안호이저와 아돌푸스 부슈 같은 독일 이민들에 귀착시켰다.</p>
<p><a href="/wiki/%EC%98%A5%ED%86%A0%EB%B2%84%ED%8E%98%EC%8A%A4%ED%8A%B8" title="옥토버페스트">옥토버페스트</a>, 독일계 미국인의 날과 슈토이벤 남작의 날 같은 독일계 미국인의 축제들이 전국을 통하여 정규적으로 열린다. 가장 큰 축제들 중의 하나는 <a href="/wiki/%EB%89%B4%EC%9A%95" title="뉴욕">뉴욕</a>에서 열리는 슈토이벤 행렬이며 <a href="/wiki/9%EC%9B%94" title="9월">9월</a>마다 매일 세 번째 <a href="/wiki/%ED%86%A0%EC%9A%94%EC%9D%BC" title="토요일">토요일</a>에 열린다. <a href="/wiki/%EC%8B%9C%EC%B9%B4%EA%B3%A0" title="시카고">시카고</a>의 링컨 광장 이웃에서도 주요 해마다의 이벤트들이 열리고, 독일인 인구의 전통적 중심지 <a href="/wiki/%EC%8B%A0%EC%8B%9C%EB%82%B4%ED%8B%B0" title="신시내티">신시내티</a>에서는 독일 외부에서 가장 큰 옥토버페스트인 "옥토버페스트 친치나티(<span style="font-size: smaller;"><a href="/wiki/%EB%8F%85%EC%9D%BC%EC%96%B4" title="독일어">독일어</a>:</span> <span lang="de" xml:lang="de">Oktoberfest Zinzinati</span>)"가 열린다. 밀워키에서는 해마다 열리는 독일인 축제와 함께 도시의 독일인 유산을 축제벌이는 편이다. <a href="/wiki/%ED%8E%9C%EC%8B%A4%EB%B2%A0%EC%9D%B4%EB%8B%88%EC%95%84_%EC%A3%BC" title="펜실베이니아 주">펜실베이니아 주</a>에 온 독일과 다른 독일어 사용 국가들에서 많은 이민들이 "앨러게니 시티"(지금의 <a href="/wiki/%ED%94%BC%EC%B8%A0%EB%B2%84%EA%B7%B8" title="피츠버그">피츠버그</a> 시의 노스 사이드)로 알려진 지역에 정착하였다. 많은 독일인들이 도착하면서 지역이 "도이치타운"으로 알려졌다. 도이치타운 안에서와 <a href="/wiki/1854%EB%85%84" title="1854년">1854년</a> 이래 튜토니아 맨너호르는 독일 문화 전통들을 진흥과 촉진을 해왔다.</p>
<p>독일에서 가장 인기있는 카드 게임 스카트는 <a href="/wiki/%EC%9C%84%EC%8A%A4%EC%BD%98%EC%8B%A0_%EC%A3%BC" title="위스콘신 주">위스콘신 주</a>와 <a href="/wiki/%ED%85%8D%EC%82%AC%EC%8A%A4_%EC%A3%BC" title="텍사스 주">텍사스 주</a> 같은 큰 독일 인구와 함께 한 미국의 지역들에서 행하고 있다.</p>
<p>독일식 이름을 가지고 있는 미국의 도시와 타운들은 중서부에서 마저 극단적으로 흔한 편이다. 그러나 독일식 이름의 두가지 예로는 <a href="/wiki/%EC%95%84%EC%9D%B4%EC%98%A4%EC%99%80_%EC%A3%BC" title="아이오와 주">아이오와 주</a>의 베텐도프와 <a href="/wiki/%EC%9D%BC%EB%A6%AC%EB%85%B8%EC%9D%B4_%EC%A3%BC" title="일리노이 주">일리노이 주</a>의 샤움버그를 들 수 있다.</p>

<p>독일계 미국인들은 <a href="/wiki/%EA%B3%BC%ED%95%99" title="과학">과학</a>, <a href="/wiki/%EA%B1%B4%EC%B6%95" title="건축">건축</a>, <a href="/wiki/%EC%82%B0%EC%97%85" title="산업">산업</a>, <a href="/wiki/%EC%8A%A4%ED%8F%AC%EC%B8%A0" title="스포츠">스포츠</a>, <a class="mw-redirect" href="/wiki/%EC%97%B0%EC%98%88%EA%B3%84" title="연예계">연예계</a>, <a href="/wiki/%EC%8B%A0%ED%95%99" title="신학">신학</a>, <a href="/wiki/%EC%A0%95%EB%B6%80" title="정부">정부</a>와 <a href="/wiki/%EA%B5%B0%EC%82%AC" title="군사">군사</a>의 거의 모든 분야에서 영향적이었다.</p>
<p>독일계 미국인 장군들 - <a href="/wiki/%ED%94%84%EB%A6%AC%EB%93%9C%EB%A6%AC%ED%9E%88_%EB%B9%8C%ED%97%AC%EB%A6%84_%ED%8F%B0_%EC%8A%88%ED%86%A0%EC%9D%B4%EB%B2%A4" title="프리드리히 빌헬름 폰 슈토이벤">슈토이벤 남작</a>, <a href="/wiki/%EC%A1%B0%EC%A7%80_%EC%95%94%EC%8A%A4%ED%8A%B8%EB%A1%B1_%EC%BB%A4%EC%8A%A4%ED%84%B0" title="조지 암스트롱 커스터">조지 암스트롱 커스터</a>, <a class="mw-redirect" href="/wiki/%EC%A1%B4_%ED%8D%BC%EC%8B%B1" title="존 퍼싱">존 퍼싱</a>, <a href="/wiki/%EB%93%9C%EC%99%80%EC%9D%B4%ED%8A%B8_D._%EC%95%84%EC%9D%B4%EC%A0%A0%ED%95%98%EC%9B%8C" title="드와이트 D. 아이젠하워">드와이트 D. 아이젠하워</a>와 <a href="/wiki/%EB%85%B8%EB%A8%BC_%EC%8A%88%EC%9B%8C%EC%B8%A0%EC%BD%94%ED%94%84" title="노먼 슈워츠코프">노먼 슈워츠코프</a>는 <a href="/wiki/%EB%AF%B8%EA%B5%AD_%EB%8F%85%EB%A6%BD_%EC%A0%84%EC%9F%81" title="미국 독립 전쟁">미국 독립 전쟁</a>, <a class="mw-disambig" href="/wiki/%EB%82%A8%EB%B6%81_%EC%A0%84%EC%9F%81" title="남북 전쟁">남북 전쟁</a>, <a class="mw-redirect" href="/wiki/%EC%9D%B8%EB%94%94%EC%96%B8" title="인디언">인디언</a> 학살, <a href="/wiki/%EC%A0%9C1%EC%B0%A8_%EC%84%B8%EA%B3%84_%EB%8C%80%EC%A0%84" title="제1차 세계 대전">제1차 세계 대전</a>, <a href="/wiki/%EC%A0%9C2%EC%B0%A8_%EC%84%B8%EA%B3%84_%EB%8C%80%EC%A0%84" title="제2차 세계 대전">제2차 세계 대전</a>과 <a href="/wiki/%EA%B1%B8%ED%94%84_%EC%A0%84%EC%9F%81" title="걸프 전쟁">걸프 전쟁</a>에서 <a href="/wiki/%EB%AF%B8%EA%B5%AD_%EC%9C%A1%EA%B5%B0" title="미국 육군">미국 육군</a>의 사령관들이었다.</p>
<p><a href="/wiki/%EB%8B%90_%EC%95%94%EC%8A%A4%ED%8A%B8%EB%A1%B1" title="닐 암스트롱">닐 암스트롱</a> 같은 다른 이들은 <a href="/wiki/%EB%8B%AC" title="달">달</a>에 착륙하였다.</p>
<p>독일계 미국인들은 칼 셔즈, 프리드리히 헤커, 프레더릭 뮬렌버그, <a href="/wiki/%EB%93%9C%EC%99%80%EC%9D%B4%ED%8A%B8_D._%EC%95%84%EC%9D%B4%EC%A0%A0%ED%95%98%EC%9B%8C" title="드와이트 D. 아이젠하워">드와이트 D. 아이젠하워</a>, <a href="/wiki/%ED%97%A8%EB%A6%AC_%ED%82%A4%EC%8B%A0%EC%A0%80" title="헨리 키신저">헨리 키신저</a> 등을 포함한 유명한 정치인들이었다.</p>
<p>많은 독일계 미국인들은 산업과 <a class="mw-redirect" href="/wiki/%EB%B9%84%EC%A7%80%EB%8B%88%EC%8A%A4" title="비지니스">비지니스</a>에서 두드러진 역할을 하였으며, <a href="/wiki/%EC%A1%B4_D._%EB%A1%9D%ED%8E%A0%EB%9F%AC" title="존 D. 록펠러">존 D. 록펠러</a>(<a class="mw-redirect" href="/wiki/%EC%8A%A4%ED%83%A0%EB%8B%A4%EB%93%9C_%EC%84%9D%EC%9C%A0_%ED%9A%8C%EC%82%AC" title="스탠다드 석유 회사">스탠다드 석유 회사</a>), 윌리엄 보잉(<a href="/wiki/%EB%B3%B4%EC%9E%89" title="보잉">보잉</a> 컴퍼니), <a href="/wiki/%EC%9B%94%ED%84%B0_%ED%81%AC%EB%9D%BC%EC%9D%B4%EC%8A%AC%EB%9F%AC" title="월터 크라이슬러">월터 크라이슬러</a>(크라이슬러 주식회사), 프레더릭과 오거스트 듀센버그(듀센버그 자동차 주식회사), 스튜드베이커 형제(스튜드베이커 자동차 주식회사), <a href="/wiki/%EC%A1%B0%EC%A7%80_%EC%9B%A8%EC%8A%A4%ED%8C%85%ED%95%98%EC%9A%B0%EC%8A%A4" title="조지 웨스팅하우스">조지 웨스팅하우스</a>(<a href="/wiki/%EC%9B%A8%EC%8A%A4%ED%8C%85%ED%95%98%EC%9A%B0%EC%8A%A4" title="웨스팅하우스">웨스팅하우스</a> 전기 주식회사), <a href="/wiki/%EB%A6%AC%EB%B0%94%EC%9D%B4_%EC%8A%A4%ED%8A%B8%EB%9D%BC%EC%9A%B0%EC%8A%A4" title="리바이 스트라우스">리바이 스트라우스</a>(<a href="/wiki/%EB%A6%AC%EB%B0%94%EC%9D%B4%EC%8A%A4" title="리바이스">리바이 스트라우스 앤드 컴퍼니</a>), 헨리 J. 하인즈(<a class="mw-redirect" href="/wiki/H._J._%ED%95%98%EC%9D%B8%EC%A6%88_%EC%BB%B4%ED%8D%BC%EB%8B%88" title="H. J. 하인즈 컴퍼니">H. J. 하인즈 컴퍼니</a>), 찰스 거스(<a href="/wiki/%ED%8E%A9%EC%8B%9C%EC%BD%94" title="펩시코">펩시코</a>), 제임스 L. 크래프트(<a href="/wiki/%ED%81%AC%EB%9E%98%ED%94%84%ED%8A%B8_%ED%91%B8%EC%A6%88" title="크래프트 푸즈">크래프트 푸즈</a>), 헨리 E. 스타인웨이(스타인웨이 앤드 선즈), 찰스 파이저(파이저 단체), <a href="/wiki/%EB%8F%84%EB%84%90%EB%93%9C_%ED%8A%B8%EB%9F%BC%ED%94%84" title="도널드 트럼프">도널드 트럼프</a>(<a class="mw-disambig" href="/wiki/%ED%8A%B8%EB%9F%BC%ED%94%84" title="트럼프">트럼프</a> 기구), <a href="/wiki/%EC%A1%B4_%EC%A0%9C%EC%9D%B4%EC%BB%B5_%EC%95%A0%EC%8A%A4%ED%84%B0" title="존 제이컵 애스터">존 제이컵 애스터</a>(월도프 애스터리아 호텔 앤드 리조트), 콘래드 힐튼(<a class="mw-redirect" href="/wiki/%ED%9E%90%ED%8A%BC_%ED%98%B8%ED%85%94" title="힐튼 호텔">힐튼 호텔</a> 앤드 리조트), <a class="mw-disambig" href="/wiki/%EA%B5%AC%EA%B2%90%ED%95%98%EC%9E%84" title="구겐하임">구겐하임</a> 가(솔로몬 R. 구겐하임 재단), 마커스 골드먼과 새뮤얼 잭스(골드먼 잭스 그룹 단체), 레먼 형제(레먼 브라더즈 소유지), <a href="/wiki/%ED%97%88%EB%A8%BC_%ED%99%80%EB%9F%AC%EB%A6%AC%EC%8A%A4" title="허먼 홀러리스">허먼 홀러리스</a>(<a href="/wiki/IBM" title="IBM">IBM</a>), <a href="/wiki/%EC%8A%A4%ED%8B%B0%EB%B8%8C_%EC%9E%A1%EC%8A%A4" title="스티브 잡스">스티브 잡스</a>(<a href="/wiki/%EC%95%A0%ED%94%8C_(%EA%B8%B0%EC%97%85)" title="애플 (기업)">애플</a>)와 <a class="mw-redirect" href="/wiki/%EC%97%90%EB%A6%AD_%EC%8A%88%EB%AF%B8%ED%8A%B8" title="에릭 슈미트">에릭 슈미트</a>(<a href="/wiki/%EA%B5%AC%EA%B8%80" title="구글">구글</a>) 등을 포함한다.</p>
<p>독일계 미국인들은 미국 역사의 거의로 봐서 <a href="/wiki/%EB%A7%A5%EC%A3%BC" title="맥주">맥주</a> 양조업을 지배하기도 하였다. 아우구스트 셸(아우구스트 셸 양조 회사), 에베르하르트 안호이저와 아돌푸스 부슈(<a href="/wiki/%EB%B2%84%EB%93%9C%EC%99%80%EC%9D%B4%EC%A0%80" title="버드와이저">안호이저-부슈 합동사</a>), 아돌프 코르스(몬손 코르스 양조 회사), 프레더릭 밀러(밀러 양조 회사), 프레더릭 팝스트(팝스트 양조 회사), 베른하르트 슈트로(슈트로 양조 회사)와 조지프 쉴리츠(조지프 쉴리츠 양조 회사) 등의 19세기 독일인 이민들에 의하여 창립된 양조사들과 함께 시작되었다.</p>
<p>어떤 분야에서는 <a href="/wiki/%EB%B8%8C%EB%A3%A8%ED%81%B4%EB%A6%B0_%EA%B5%90" title="브루클린 교">브루클린 교</a>의 <a href="/wiki/%EA%B3%B5%ED%95%99%EC%9E%90" title="공학자">공학자</a> <a href="/wiki/%EC%A1%B4_%EB%A1%9C%EB%B8%94%EB%A7%81" title="존 로블링">존 로블링</a>과 <a href="/wiki/%EB%B0%9C%ED%84%B0_%EA%B7%B8%EB%A1%9C%ED%94%BC%EC%9A%B0%EC%8A%A4" title="발터 그로피우스">발터 그로피우스</a>와 <a href="/wiki/%EB%A3%A8%ED%8A%B8%EB%B9%84%ED%9E%88_%EB%AF%B8%EC%8A%A4_%EB%B0%98_%EB%8D%B0%EC%96%B4_%EB%A1%9C%EC%97%90" title="루트비히 미스 반 데어 로에">루트비히 미스 반 데어 로에</a> 같은 <a href="/wiki/%EA%B1%B4%EC%B6%95%EA%B0%80" title="건축가">건축가</a>들은 볼 수 있는 경계표들 뒤에 남겨졌다.</p>
