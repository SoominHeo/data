<p><b>독일을 위한 대안</b>(獨逸을 爲한 代案, <span style="font-size: smaller;"><a href="/wiki/%EB%8F%85%EC%9D%BC%EC%96%B4" title="독일어">독일어</a>:</span> <span lang="de" xml:lang="de">Alternative für Deutschland</span>, <b>AfD</b>)은 <a href="/wiki/%EB%8F%85%EC%9D%BC" title="독일">독일</a>의 <a href="/wiki/%EC%9A%B0%EC%9D%B5%EB%8C%80%EC%A4%91%EC%A3%BC%EC%9D%98" title="우익대중주의">우익대중주의</a>,<sup class="reference" id="cite_ref-1"><a href="#cite_note-1">[1]</a></sup> <a href="/wiki/%EC%9C%A0%EB%9F%BD%ED%9A%8C%EC%9D%98%EC%A3%BC%EC%9D%98" title="유럽회의주의">유럽회의주의</a><sup class="reference" id="cite_ref-2"><a href="#cite_note-2">[2]</a></sup> 정당으로, 2013년에 창당되었다. AfD는 2013년 투표에서 4.7%의 득표율을 기록해, <a href="/wiki/%EB%8F%85%EC%9D%BC_%EC%97%B0%EB%B0%A9%EC%9D%98%ED%9A%8C" title="독일 연방의회">독일 연방의회</a>에 입성하기 위한 최저 득표율인 <a href="/wiki/5%25_%EC%A0%80%EC%A7%80_%EC%A1%B0%ED%95%AD" title="5% 저지 조항">5%</a>에는 다다르지 못했다. 2014년 AfD는 <a href="/wiki/%EC%9C%A0%EB%9F%BD_%EC%9D%98%ED%9A%8C" title="유럽 의회">유럽 의회</a>에서 7.1%의 득표율을 기록해, 독일에 배정된 96석 중 7석을 확보하였으며, <a class="mw-redirect" href="/wiki/%EC%9C%A0%EB%9F%BD_%EB%B3%B4%EC%88%98%EC%99%80_%EA%B0%9C%ED%98%81" title="유럽 보수와 개혁">유럽 보수와 개혁</a>(European Conservatives and Reformists)에 가입하였지만 2016년 3월에 탈퇴했다.<sup class="reference" id="cite_ref-3"><a href="#cite_note-3">[3]</a></sup> 2016년 3월까지 AfD는 독일 주 의회 중 8곳에 대표를 두고 있으며, 현재 당은 <a href="/wiki/%ED%94%84%EB%9D%BC%EC%9A%B0%EC%BC%80_%ED%8E%98%ED%8A%B8%EB%A6%AC" title="프라우케 페트리">프라우케 페트리</a>(Frauke Petry)와 <a class="new" href="/w/index.php?title=%EC%99%B8%EB%A5%B4%ED%81%AC_%EB%AA%A8%EC%9D%B4%ED%85%90&amp;action=edit&amp;redlink=1" title="외르크 모이텐 (없는 문서)">외르크 모이텐</a>(Jörg Meuthen)이 이끌고 있다. 또한 2016년 1월을 기준으로 <a class="new" href="/w/index.php?title=%EA%B5%AD%EA%B3%A0_%EB%B3%B4%EC%A1%B0%EA%B8%88&amp;action=edit&amp;redlink=1" title="국고 보조금 (없는 문서)">국고 보조금</a>은 5,210,508.91 유로를 받고 있다.<sup class="reference" id="cite_ref-4"><a href="#cite_note-4">[4]</a></sup></p>
<p>처음에는 독일의 <a href="/wiki/%EC%9C%A0%EB%9F%BD_%EC%97%B0%ED%95%A9" title="유럽 연합">유럽 연합</a> 탈퇴를 주장하지는 않고 단지 독일의 <a href="/wiki/%EC%9C%A0%EB%A1%9C" title="유로">유로</a> 사용 중지만을 주장하였다. AfD는 2015년 중순에 페트리가 당권을 장악해 당 노선을 반이민 정책으로 삼은 뒤부터 갈라졌다. 그 후 당의 설립자 <a class="new" href="/w/index.php?title=%EB%B2%A0%EB%A5%B8%ED%8A%B8_%EB%A3%A8%EC%BC%80&amp;action=edit&amp;redlink=1" title="베른트 루케 (없는 문서)">베른트 루케</a>(Bernd Lucke)와 그의 경제 정책 지지자는 탈당해 <a href="/wiki/%EC%A7%84%EB%B3%B4%EC%99%80_%EA%B0%9C%ED%98%81_%EC%97%B0%ED%95%A9" title="진보와 개혁 연합">진보와 개혁 연합</a><sup class="reference" id="cite_ref-5"><a href="#cite_note-5">[5]</a></sup>을 설립하였다.</p>

<h2><span class="mw-headline" id=".EC.84.B1.EC.9E.A5_.EB.B0.B0.EA.B2.BD">성장 배경</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%8F%85%EC%9D%BC%EC%9D%84_%EC%9C%84%ED%95%9C_%EB%8C%80%EC%95%88&amp;action=edit&amp;section=1" title="부분 편집: 성장 배경">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p><a href="/wiki/%EC%A0%9C2%EC%B0%A8_%EC%84%B8%EA%B3%84_%EB%8C%80%EC%A0%84" title="제2차 세계 대전">제2차 세계 대전</a> 이후부터 AfD 이전까지 <a href="/wiki/%EB%8F%85%EC%9D%BC" title="독일">독일</a>에서는 유럽의 다른 국가와는 달리 <a class="mw-redirect" href="/wiki/%EA%B7%B9%EC%9A%B0%ED%8C%8C" title="극우파">극우파</a> 정당이 성공한 경우가 없었다. 또한 <a href="/wiki/%EB%8F%85%EC%9D%BC" title="독일">독일</a>에서 <a href="/wiki/%EB%8F%85%EC%9D%BC_%EC%97%B0%EB%B0%A9%EC%9D%98%ED%9A%8C" title="독일 연방의회">독일 연방의회</a>에 입성한 극우파 정당은 하나도 없었다. 게다가 극우 테러단체인 <a class="new" href="/w/index.php?title=%EA%B5%AD%EA%B0%80%EC%82%AC%ED%9A%8C%EC%A3%BC%EC%9D%98_%EC%A7%80%ED%95%98%EC%A1%B0%EC%A7%81&amp;action=edit&amp;redlink=1" title="국가사회주의 지하조직 (없는 문서)">국가사회주의 지하조직</a>(NSU)의 활동 등으로 인해 <a class="mw-redirect" href="/wiki/%EA%B7%B9%EC%9A%B0%ED%8C%8C" title="극우파">극우파</a>에 대한 인식이 더욱 안 좋아지게 되었다. 초기 AfD는 이러한 독일 및 <a href="/wiki/%EC%9C%A0%EB%9F%BD" title="유럽">유럽</a>의 극우 정당들과는 거리를 두었다. 또한 <a href="/wiki/%EB%8F%85%EC%9D%BC_%EA%B8%B0%EB%8F%85%EA%B5%90%EB%AF%BC%EC%A3%BC%EC%97%B0%ED%95%A9" title="독일 기독교민주연합">CDU</a>나 <a href="/wiki/%EB%8F%85%EC%9D%BC_%EC%82%AC%ED%9A%8C%EB%AF%BC%EC%A3%BC%EB%8B%B9" title="독일 사회민주당">SPD</a> 등의 기존 정당들이 실행하지 않는 문제들을 공략하거나 <a class="mw-redirect" href="/wiki/%EC%9C%A0%EB%A1%9C%ED%99%94" title="유로화">유로화</a>, <a href="/wiki/%EB%8B%A4%EB%AC%B8%ED%99%94%EC%A3%BC%EC%9D%98" title="다문화주의">다문화</a>, <a href="/wiki/%EB%8F%99%EC%84%B1%EC%95%A0" title="동성애">동성애</a> 등에 반대함으로써 득표를 어느 정도 하는데 성공하였다.<sup class="reference" id="cite_ref-6"><a href="#cite_note-6">[6]</a></sup></p>
<p>최근에는 <a href="/wiki/%EC%8B%9C%EB%A6%AC%EC%95%84_%EB%82%B4%EC%A0%84%EC%9D%98_%EB%82%9C%EB%AF%BC" title="시리아 내전의 난민">시리아 난민</a> 문제로 인해 AfD에 대한 지지가 크게 증가하였다. 독일의 난민 수용으로 인한 경제적인 부담과 <a href="/wiki/%EC%BE%B0%EB%A5%B8" title="쾰른">쾰른</a>에서 일어난 <a href="/wiki/%EB%8F%85%EC%9D%BC_%EC%8B%A0%EB%85%84%EC%A0%84%EC%95%BC_%EB%8F%99%EC%8B%9C%EB%8B%A4%EB%B0%9C_%EC%84%B1%ED%8F%AD%ED%96%89_%EC%82%AC%EA%B1%B4" title="독일 신년전야 동시다발 성폭행 사건">난민들의 성폭행 사건</a>과 같은 사회문제의 발생은 독일인들이 난민에 대한 불신을 불러일으키게 되었으며, 이는 AfD를 지지하는데 결정적인 역할을 하였다. 그뿐만 아니라 AfD가 득세하고 있는 지역은 과거 <a href="/wiki/%EB%8F%99%EB%8F%85" title="동독">동독</a> 지역인데, <a href="/wiki/%EB%8F%99%EB%8F%85" title="동독">동독</a> 지역의 주민들은 통일과정에서의 불만과 함께 <a href="/wiki/%EC%84%9C%EB%8F%85" title="서독">서독</a> 지역에 비해 상대적으로 낮은 경제수준으로 인해 박탈감을 느껴왔고,<sup class="reference" id="cite_ref-7"><a href="#cite_note-7">[7]</a></sup> 이러한 상황에서 좌파에 대한 불만과, 불만을 해소시켜주지 못하는 기존 정당에 대한 심판과 더불어 대안을 줄 수 있는 정당을 희망하였다. 2016년의 선거 결과를 보면 <a href="/wiki/%EB%8F%85%EC%9D%BC_%EA%B8%B0%EB%8F%85%EA%B5%90%EB%AF%BC%EC%A3%BC%EC%97%B0%ED%95%A9" title="독일 기독교민주연합">기민당</a>도 피해를 입었지만 <a href="/wiki/%EB%8F%85%EC%9D%BC_%EC%82%AC%ED%9A%8C%EB%AF%BC%EC%A3%BC%EB%8B%B9" title="독일 사회민주당">사민당</a>도 이에 못지 않게 매우 큰 피해를 입은 것을 알 수 있다. 2016년 주의회 선거에서 <a href="/wiki/%EB%8F%85%EC%9D%BC_%EA%B8%B0%EB%8F%85%EA%B5%90%EB%AF%BC%EC%A3%BC%EC%97%B0%ED%95%A9" title="독일 기독교민주연합">기민당</a>에 대한 지지율 하락은 <a href="/wiki/%EB%B0%94%EB%8D%B4%EB%B7%94%EB%A5%B4%ED%85%9C%EB%B2%A0%EB%A5%B4%ED%81%AC_%EC%A3%BC" title="바덴뷔르템베르크 주">바덴뷔르템베르크 주</a>에서 12퍼센트, <a href="/wiki/%EC%9E%91%EC%84%BC%EC%95%88%ED%95%A0%ED%8A%B8_%EC%A3%BC" title="작센안할트 주">작센안할트 주</a>에서 2.7퍼센트였다. 그런데 <a href="/wiki/%EB%8F%85%EC%9D%BC_%EC%82%AC%ED%9A%8C%EB%AF%BC%EC%A3%BC%EB%8B%B9" title="독일 사회민주당">사민당</a>은 <a href="/wiki/%EB%B0%94%EB%8D%B4%EB%B7%94%EB%A5%B4%ED%85%9C%EB%B2%A0%EB%A5%B4%ED%81%AC_%EC%A3%BC" title="바덴뷔르템베르크 주">바덴뷔르템베르크 주</a>에서 10.4퍼센트, <a href="/wiki/%EC%9E%91%EC%84%BC%EC%95%88%ED%95%A0%ED%8A%B8_%EC%A3%BC" title="작센안할트 주">작센안할트 주</a>에서는 10.9퍼센트의 지지율을 잃었다. 이를 통해 AfD의 득세가 <a href="/wiki/%EC%8B%9C%EB%A6%AC%EC%95%84_%EB%82%B4%EC%A0%84%EC%9D%98_%EB%82%9C%EB%AF%BC" title="시리아 내전의 난민">난민</a> 문제에 대한 반대의 결과와 더불어 과거 동독 지역의 어려운 상황 속에서 기존 정당에 대한 불만이 표출된 결과라고 할 수 있다. 또한 AfD의 주요 지지층은 젊은 남성층으로<sup class="reference" id="cite_ref-8"><a href="#cite_note-8">[8]</a></sup>, 이들은 전쟁을 직접 겪지 않은 세대이기 때문에 극우정당이라는 것에 대한 금기가 상당부분 희석되었다. 이와 같은 요인들이 AfD가 극우정당으로서 강력한 돌풍을 일으키는데 작용하였다.<sup class="reference" id="cite_ref-9"><a href="#cite_note-9">[9]</a></sup></p>
<h2><span class="mw-headline" id=".EB.8B.B9_.EC.97.AD.EC.82.AC">당 역사</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%8F%85%EC%9D%BC%EC%9D%84_%EC%9C%84%ED%95%9C_%EB%8C%80%EC%95%88&amp;action=edit&amp;section=2" title="부분 편집: 당 역사">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<h3><span class="mw-headline" id=".EC.84.A4.EB.A6.BD">설립</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%8F%85%EC%9D%BC%EC%9D%84_%EC%9C%84%ED%95%9C_%EB%8C%80%EC%95%88&amp;action=edit&amp;section=3" title="부분 편집: 설립">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>2012년 9월, <a href="/wiki/%ED%97%A4%EC%84%BC_%EC%A3%BC" title="헤센 주">헤센 주</a>의 전 주 사무관인 알렉산더 가울란트(Alexander Gauland), 경제학자 <a class="new" href="/w/index.php?title=%EB%B2%A0%EB%A5%B8%ED%8A%B8_%EB%A3%A8%EC%BC%80&amp;action=edit&amp;redlink=1" title="베른트 루케 (없는 문서)">베른트 루케</a>, <a href="/wiki/%ED%94%84%EB%9E%91%ED%81%AC%ED%91%B8%EB%A5%B4%ED%84%B0_%EC%95%8C%EA%B2%8C%EB%A7%88%EC%9D%B4%EB%84%A4_%EC%B0%A8%EC%9D%B4%ED%89%81" title="프랑크푸르터 알게마이네 차이퉁">프랑크푸르터 알게마이네 차이퉁</a>(Frankfurter Allgemeine Zeitung)의 전 편집자 콘라드 아담(Konrad Adam)은 바트 나우하임(Bad Nauheim)에서 Electoral Alternative 2013 (Wahlalternative 2013)을 창당하였다. 당의 목적은 <a href="/wiki/%EC%9C%A0%EB%A1%9C%EC%A1%B4" title="유로존">유로존</a> 위기에 관련한 독일의 연방 정책을 비판하는 것이며, 그들의 주장은 68명의 경제학자, 언론인 그리고 사업가들에게 지지를 받았다. 그들 중 절반은 교수였으며 그 외 기업가나 기존 정당 일원도 있었다.<sup class="reference" id="cite_ref-10"><a href="#cite_note-10">[10]</a></sup> 그들은 경제 위기를 통해 <a href="/wiki/%EC%9C%A0%EB%A1%9C%EC%A1%B4" title="유로존">유로존</a>의 불안정성이 드러났으며, <a href="/wiki/%EB%82%A8%EC%9C%A0%EB%9F%BD" title="남유럽">남유럽</a> 국가들은 <a class="mw-redirect" href="/wiki/%EC%9C%A0%EB%A1%9C%ED%99%94" title="유로화">유로화</a>의 압박 때문에 가난해지고 있다고 주장하였다.</p>
<h3><span class="mw-headline" id=".ED.83.80_.EC.A0.95.EB.8B.B9.EA.B3.BC.EC.9D.98_.EC.97.B0.EB.8C.80_.EC.8B.9C.EB.8F.84">타 정당과의 연대 시도</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%8F%85%EC%9D%BC%EC%9D%84_%EC%9C%84%ED%95%9C_%EB%8C%80%EC%95%88&amp;action=edit&amp;section=4" title="부분 편집: 타 정당과의 연대 시도">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>후에 설립될 AfD의 몇몇 멤버들은 <a href="/wiki/%EB%8B%88%EB%8D%94%EC%9E%91%EC%84%BC_%EC%A3%BC" title="니더작센 주">니더작센 주</a>의 선거를 위해 <a class="new" href="/w/index.php?title=%EC%9E%90%EC%9C%A0_%ED%88%AC%ED%91%9C%EC%9E%90&amp;action=edit&amp;redlink=1" title="자유 투표자 (없는 문서)">자유 투표자</a>(Freie Wähler)와 연합을 했지만, 연방과 외국 문제에 대한 구체적인 정책이 없다 보니 1%의 득표율 을 기록하는 데 그쳤다. 2013년 2월 당은 2013년 연방 선거를 위해 새로운 당을 창당하려 했는데, 루커의 메일 누출 사건<sup class="reference" id="cite_ref-11"><a href="#cite_note-11">[11]</a></sup>으로 인해 Free voters의 협조를 얻는 데 실패한다. 이 후에 이 정당은 유로의 철폐를 요구하는 등, <a class="new" href="/w/index.php?title=%EC%9E%90%EC%9C%A0_%ED%88%AC%ED%91%9C%EC%9E%90&amp;action=edit&amp;redlink=1" title="자유 투표자 (없는 문서)">자유 투표자</a> 보다 더 급진적인 주장을 펼치게 된다.</p>
<h3><span class="mw-headline" id=".EB.8F.85.EC.9D.BC.EC.9D.84_.EC.9C.84.ED.95.9C_.EB.8C.80.EC.95.88.28AfD.29">독일을 위한 대안(AfD)</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%8F%85%EC%9D%BC%EC%9D%84_%EC%9C%84%ED%95%9C_%EB%8C%80%EC%95%88&amp;action=edit&amp;section=5" title="부분 편집: 독일을 위한 대안(AfD)">편집</a><span class="mw-editsection-bracket">]</span></span></h3>

2013년 왼쪽부터 당수 콘라트 아담, <a href="/wiki/%ED%94%84%EB%9D%BC%EC%9A%B0%EC%BC%80_%ED%8E%98%ED%8A%B8%EB%A6%AC" title="프라우케 페트리">프라우케 페트리</a> 그리고 <a class="new" href="/w/index.php?title=%EB%B2%A0%EB%A5%B8%ED%8A%B8_%EB%A3%A8%EC%BC%80&amp;action=edit&amp;redlink=1" title="베른트 루케 (없는 문서)">베른트 루케</a></div>

<p>AfD 정당은 2013년 2월 6일 정책에 대한 충분한 토론을 거치지 못한 채 비교적 짧은 시간 내에 만들어졌다.<sup class="reference" id="cite_ref-12"><a href="#cite_note-12">[12]</a></sup> 2013년 4월 14,일 AfD는 <a href="/wiki/%EB%B2%A0%EB%A5%BC%EB%A6%B0" title="베를린">베를린</a>에서 첫 전당대회를 열면서 그 존재를 대중에게 알렸다. 이 당 대회에서 AfD는 <a class="new" href="/w/index.php?title=%EB%B2%A0%EB%A5%B8%ED%8A%B8_%EB%A3%A8%EC%BC%80&amp;action=edit&amp;redlink=1" title="베른트 루케 (없는 문서)">베른트 루케</a>, 콘라트 아담 그리고 <a href="/wiki/%ED%94%84%EB%9D%BC%EC%9A%B0%EC%BC%80_%ED%8E%98%ED%8A%B8%EB%A6%AC" title="프라우케 페트리">프라우케 페트리</a>를 대변인으로 선출 하는 등 당의 구성을 완료하였다.<sup class="reference" id="cite_ref-13"><a href="#cite_note-13">[13]</a></sup> 2013년 3월 31일과 5월 12일 사이에 <a href="/wiki/%EB%8F%85%EC%9D%BC%EC%9D%98_%EC%84%A0%EA%B1%B0" title="독일의 선거">독일 주 선거</a>에 참여하기 위해 16개 주 모두에 지부를 설치하였고, 6월 15일에는 <a href="/wiki/%EB%8B%A4%EB%A6%84%EC%8A%88%ED%83%80%ED%8A%B8" title="다름슈타트">다름슈타트</a>(Darmstadt)에서 소년 정당으로 독일을 위한 청소년 대안(JA)를 창당하였다.<sup class="reference" id="cite_ref-14"><a href="#cite_note-14">[14]</a></sup></p>
<h3><span class="mw-headline" id=".EC.9C.A0.EB.9F.BD_.EB.B3.B4.EC.88.98.EC.99.80_.EA.B0.9C.ED.98.81.EA.B3.BC.EC.9D.98_.EC.97.B0.EA.B3.84">유럽 보수와 개혁과의 연계</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%8F%85%EC%9D%BC%EC%9D%84_%EC%9C%84%ED%95%9C_%EB%8C%80%EC%95%88&amp;action=edit&amp;section=6" title="부분 편집: 유럽 보수와 개혁과의 연계">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>2013년 4월, <a class="mw-redirect" href="/wiki/%EC%9C%A0%EB%9F%BD_%EB%B3%B4%EC%88%98%EC%99%80_%EA%B0%9C%ED%98%81" title="유럽 보수와 개혁">유럽 보수와 개혁</a>(European Conservatives and Reformists)의 지도 하에 AfD는 <a class="mw-redirect" href="/wiki/%EC%98%81%EA%B5%AD_%EB%B3%B4%EC%88%98%EB%8B%B9" title="영국 보수당">영국 보수당</a>과 접촉을 하였고<sup class="reference" id="cite_ref-15"><a href="#cite_note-15">[15]</a></sup>, 6월에는 루커가 런던의 <a class="new" href="/w/index.php?title=%ED%8F%AC%ED%8A%B8%EC%BB%AC%EB%A6%AC%EC%8A%A4&amp;action=edit&amp;redlink=1" title="포트컬리스 (없는 문서)">포트컬리스</a>(Portcullis House)에서 청문회를 하였다.<sup class="reference" id="cite_ref-16"><a href="#cite_note-16">[16]</a></sup></p>
<h3><span class="mw-headline" id=".EB.8B.B9.EC.9D.98_.EA.B7.B9.EC.9A.B0.ED.99.94">당의 극우화</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%8F%85%EC%9D%BC%EC%9D%84_%EC%9C%84%ED%95%9C_%EB%8C%80%EC%95%88&amp;action=edit&amp;section=7" title="부분 편집: 당의 극우화">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>2015년 7월 4일, <a class="mw-redirect" href="/wiki/%ED%97%A4%EC%84%BC" title="헤센">헤센</a> 주의 당 대회에서 <a href="/wiki/%ED%94%84%EB%9D%BC%EC%9A%B0%EC%BC%80_%ED%8E%98%ED%8A%B8%EB%A6%AC" title="프라우케 페트리">프라우케 페트리</a>가 60%의 득표율로 <a class="new" href="/w/index.php?title=%EB%B2%A0%EB%A5%B8%ED%8A%B8_%EB%A3%A8%EC%BC%80&amp;action=edit&amp;redlink=1" title="베른트 루케 (없는 문서)">베른트 루케</a>를 제치고 당의 주 대변인으로 선출되었다. 페트리는 AfD에서 이민 문제, <a class="mw-redirect" href="/wiki/%EC%9D%B4%EC%8A%AC%EB%9E%8C" title="이슬람">이슬람</a> 반대 및 <a href="/wiki/%EB%9F%AC%EC%8B%9C%EC%95%84" title="러시아">러시아</a>와의 연대를 주장<sup class="reference" id="cite_ref-17"><a href="#cite_note-17">[17]</a></sup>하는 <a href="/wiki/%EA%B7%B9%EC%9A%B0" title="극우">극우</a> 세력의 인물이었다.<sup class="reference" id="cite_ref-18"><a href="#cite_note-18">[18]</a></sup> 이를 계기로 7월 7일에 7명의 <a href="/wiki/%EC%9C%A0%EB%9F%BD_%EC%9D%98%ED%9A%8C" title="유럽 의회">유럽 의회</a> 의원 중 배아트릭스 폰 스토치(Beatrix von Storch)와 마르커스 프레첼(Marcus Pretzell)을 제외한 4명(루케 제외)이 탈당을 했으며<sup class="reference" id="cite_ref-19"><a href="#cite_note-19">[19]</a></sup>, 그 다음날인 7월 8일에는 루케가 당의 친러화와 외국인 혐오증에 반대하며 자신의 지지세력과 함께 AfD를 탈당 하였다.<sup class="reference" id="cite_ref-20"><a href="#cite_note-20">[20]</a></sup> 탈당한 루커는 7월 19일 자신이 처음 내세웠던 경제 정책을 기조로 하는 <a href="/wiki/%EC%A7%84%EB%B3%B4%EC%99%80_%EA%B0%9C%ED%98%81_%EC%97%B0%ED%95%A9" title="진보와 개혁 연합">진보와 개혁 연합</a>(Alliance for Progress and Renewal, ALFA)를 창당하였다.<sup class="reference" id="cite_ref-21"><a href="#cite_note-21">[21]</a></sup> 루케를 포함해서 탈당했던 유럽의회 의원들은 모두 ALFA에 입당했다.<sup class="reference" id="cite_ref-22"><a href="#cite_note-22">[22]</a></sup></p>
<h2><span class="mw-headline" id=".EC.82.AC.EC.83.81_.EB.B0.8F_.EC.A0.95.EC.B1.85">사상 및 정책</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%8F%85%EC%9D%BC%EC%9D%84_%EC%9C%84%ED%95%9C_%EB%8C%80%EC%95%88&amp;action=edit&amp;section=8" title="부분 편집: 사상 및 정책">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<h3><span class="mw-headline" id=".EA.B2.BD.EC.A0.9C_.EC.A0.95.EC.B1.85">경제 정책</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%8F%85%EC%9D%BC%EC%9D%84_%EC%9C%84%ED%95%9C_%EB%8C%80%EC%95%88&amp;action=edit&amp;section=9" title="부분 편집: 경제 정책">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<h4><span class="mw-headline" id=".EC.9C.A0.EB.9F.BD.EC.97.90_.EB.8C.80.ED.95.9C_.EA.B8.88.EC.9C.B5.EC.A7.80.EC.9B.90">유럽에 대한 금융지원</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%8F%85%EC%9D%BC%EC%9D%84_%EC%9C%84%ED%95%9C_%EB%8C%80%EC%95%88&amp;action=edit&amp;section=10" title="부분 편집: 유럽에 대한 금융지원">편집</a><span class="mw-editsection-bracket">]</span></span></h4>
<p>AfD는 <a href="/wiki/%EB%8F%85%EC%9D%BC" title="독일">독일</a>이 <a href="/wiki/%EB%82%A8%EC%9C%A0%EB%9F%BD" title="남유럽">남유럽</a>과 같은 가난한 <a class="mw-redirect" href="/wiki/EU" title="EU">EU</a> 가입국에 금융 지원을 해 주는 것을 반대하였다. 이들은 <a href="/wiki/%EB%82%A8%EC%9C%A0%EB%9F%BD" title="남유럽">남유럽</a> 국가를 지원해 주는 것은 <a href="/wiki/%EB%8F%85%EC%9D%BC" title="독일">독일</a>이 꼭 해야만 하는 일이 아니라며 <a href="/wiki/%EB%8F%85%EC%9D%BC" title="독일">독일</a> 국민들을 선동했다. 이에 나아가 AfD는 <a href="/wiki/%EA%B7%B8%EB%A6%AC%EC%8A%A4" title="그리스">그리스</a>, <a href="/wiki/%ED%94%84%EB%9E%91%EC%8A%A4" title="프랑스">프랑스</a>와 같이 현재 <a href="/wiki/%EC%9C%A0%EB%A1%9C%EC%A1%B4" title="유로존">유로존</a>의 경제 위기를 초래하고 있는 나라들의 <a class="mw-redirect" href="/wiki/EU" title="EU">EU</a> 탈퇴와 함께, <a href="/wiki/%EB%8F%85%EC%9D%BC" title="독일">독일</a>이 <a href="/wiki/%EB%B8%8C%EB%A4%BC%EC%85%80" title="브뤼셀">브뤼셀</a>로부터 <a class="mw-redirect" href="/wiki/EU" title="EU">EU</a>에 대한 권한을 빼앗아와야 한다고 주장하였다.<sup class="reference" id="cite_ref-23"><a href="#cite_note-23">[23]</a></sup></p>
<h4><span class="mw-headline" id=".EB.B0.98.EC.9C.A0.EB.A1.9C.ED.99.94_.EC.A0.95.EC.B1.85">반유로화 정책</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%8F%85%EC%9D%BC%EC%9D%84_%EC%9C%84%ED%95%9C_%EB%8C%80%EC%95%88&amp;action=edit&amp;section=11" title="부분 편집: 반유로화 정책">편집</a><span class="mw-editsection-bracket">]</span></span></h4>
<p>AfD의 창당 목적이 반<a href="/wiki/%EC%9C%A0%EB%A1%9C" title="유로">유로</a>화 및 <a href="/wiki/%EB%8F%85%EC%9D%BC_%EB%A7%88%EB%A5%B4%ED%81%AC" title="독일 마르크">마르크</a>화로의 회귀이고, 과거 경제학자였던 <a class="new" href="/w/index.php?title=%EB%B2%A0%EB%A5%B8%ED%8A%B8_%EB%A3%A8%EC%BC%80&amp;action=edit&amp;redlink=1" title="베른트 루케 (없는 문서)">베른트 루케</a>가 당수로써 목소리를 높일 때는 이에 관해 많은 문제제기를 했지만, 루커가 탈당한 이후로는 여기에 대해 크게 언급하지는 않고 있다.<sup class="reference" id="cite_ref-24"><a href="#cite_note-24">[24]</a></sup></p>
<h4><span class="mw-headline" id=".EB.B2.94.EB.8C.80.EC.84.9C.EC.96.91_.EB.AC.B4.EC.97.AD_.ED.88.AC.EC.9E.90_.EB.8F.99.EB.B0.98.EC.9E.90_.ED.98.91.EC.A0.95.28TTIP.29_.EB.B0.98.EB.8C.80">범대서양 무역 투자 동반자 협정(TTIP) 반대</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%8F%85%EC%9D%BC%EC%9D%84_%EC%9C%84%ED%95%9C_%EB%8C%80%EC%95%88&amp;action=edit&amp;section=12" title="부분 편집: 범대서양 무역 투자 동반자 협정(TTIP) 반대">편집</a><span class="mw-editsection-bracket">]</span></span></h4>
<p>AfD는 중소기업을 보호해야 하는 것을 강조하면서<sup class="reference" id="cite_ref-25"><a href="#cite_note-25">[25]</a></sup> 조약국 간의 관세를 철폐한다는 <a href="/wiki/%EB%B2%94%EB%8C%80%EC%84%9C%EC%96%91_%EB%AC%B4%EC%97%AD_%ED%88%AC%EC%9E%90_%EB%8F%99%EB%B0%98%EC%9E%90_%ED%98%91%EC%A0%95" title="범대서양 무역 투자 동반자 협정">범대서양 무역 투자 동반자 협정</a>(TTIP)에 반대하고 있다. AfD는 TTIP가 독일 소비자 보호와 헌법 및 정책적인 부분에 위협이 된다고 생각하고 있다.<sup class="reference" id="cite_ref-26"><a href="#cite_note-26">[26]</a></sup></p>
<h3><span class="mw-headline" id=".EB.82.9C.EB.AF.BC.EC.9D.98_.EC.9C.A0.EC.9E.85_.EB.B0.98.EB.8C.80">난민의 유입 반대</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%8F%85%EC%9D%BC%EC%9D%84_%EC%9C%84%ED%95%9C_%EB%8C%80%EC%95%88&amp;action=edit&amp;section=13" title="부분 편집: 난민의 유입 반대">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>기존의 AfD의 경우 일을 하기 위한 이민은 허용해야 하지만 복지만을 보고 오는 이민에 대해 반대 입장을 취했다.<sup class="reference" id="cite_ref-27"><a href="#cite_note-27">[27]</a></sup> 현재는 더욱 반외국인, 특히 최근 <a href="/wiki/%EC%8B%9C%EB%A6%AC%EC%95%84_%EB%82%B4%EC%A0%84%EC%9D%98_%EB%82%9C%EB%AF%BC" title="시리아 내전의 난민">난민</a>문제에 대해서 반대 입장을 취한다. 당수 <a href="/wiki/%ED%94%84%EB%9D%BC%EC%9A%B0%EC%BC%80_%ED%8E%98%ED%8A%B8%EB%A6%AC" title="프라우케 페트리">프라우케 페트리</a>는 <a href="/wiki/%EC%8B%9C%EB%A6%AC%EC%95%84_%EB%82%B4%EC%A0%84%EC%9D%98_%EB%82%9C%EB%AF%BC" title="시리아 내전의 난민">난민</a>문제와 <a href="/wiki/%EC%95%99%EA%B2%94%EB%9D%BC_%EB%A9%94%EB%A5%B4%EC%BC%88" title="앙겔라 메르켈">앙겔라 메르켈</a> 총리의 <a href="/wiki/%EC%8B%9C%EB%A6%AC%EC%95%84_%EB%82%B4%EC%A0%84%EC%9D%98_%EB%82%9C%EB%AF%BC" title="시리아 내전의 난민">난민</a> 정책에 대해 과격한 발언을 여러 번 일삼았다.<sup class="reference" id="cite_ref-28"><a href="#cite_note-28">[28]</a></sup> 페트리 뿐만 아니라 부당수인 배아트릭스도 <a href="/wiki/%EC%8B%9C%EB%A6%AC%EC%95%84_%EB%82%B4%EC%A0%84%EC%9D%98_%EB%82%9C%EB%AF%BC" title="시리아 내전의 난민">난민</a>과 관련해 부정적인 발언을 한 적이 있다.<sup class="reference" id="cite_ref-29"><a href="#cite_note-29">[29]</a></sup> 지난 1월에 <a class="mw-redirect" href="/wiki/%ED%80%BC%EB%A5%B8" title="퀼른">퀼른</a>에서 발생한 <a href="/wiki/%EB%8F%85%EC%9D%BC_%EC%8B%A0%EB%85%84%EC%A0%84%EC%95%BC_%EB%8F%99%EC%8B%9C%EB%8B%A4%EB%B0%9C_%EC%84%B1%ED%8F%AD%ED%96%89_%EC%82%AC%EA%B1%B4" title="독일 신년전야 동시다발 성폭행 사건">성폭행 사건</a>과 같은 범죄들과 유럽 지역에서 발생하는 테러들이 이를 부추기고 있으며, 특히 과거 <a href="/wiki/%EB%8F%99%EB%8F%85" title="동독">동독</a> 지역에서는 자신들이 처한 경제적 어려움을 외국인 탓으로 돌리고 있기 때문에 이는 더욱 심해지고 있다.<sup class="reference" id="cite_ref-30"><a href="#cite_note-30">[30]</a></sup></p>
<h3><span class="mw-headline" id=".EC.B9.9C.EB.9F.AC.EC.8B.9C.EC.95.84_.EC.84.B1.ED.96.A5_.EB.B0.8F_.ED.81.AC.EB.A6.BC_.EB.B0.98.EB.8F.84_.EC.82.AC.ED.83.9C">친러시아 성향 및 크림 반도 사태</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%8F%85%EC%9D%BC%EC%9D%84_%EC%9C%84%ED%95%9C_%EB%8C%80%EC%95%88&amp;action=edit&amp;section=14" title="부분 편집: 친러시아 성향 및 크림 반도 사태">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>기존에도 AfD는 <a href="/wiki/2014%EB%85%84" title="2014년">2014년</a>에 일어난 <a href="/wiki/%ED%81%AC%EB%A6%BC_%EB%B0%98%EB%8F%84" title="크림 반도">크림 반도</a> 사태에 대해 묵인하는 입장을 취해왔지만,<sup class="reference" id="cite_ref-31"><a href="#cite_note-31">[31]</a></sup> 최근에는 당의 친(親)러시아 성향이 강하게 나타나고 있다. 보통 <a href="/wiki/%EC%9C%A0%EB%9F%BD" title="유럽">유럽</a>의 <a class="mw-redirect" href="/wiki/%EA%B7%B9%EC%9A%B0%ED%8C%8C" title="극우파">극우파</a>들이 친러시아 성향을 띠는데, 이는 AfD도 마찬가지이다. 그들은 그들의 혐오 대상인 <a href="/wiki/%EC%8B%9C%EB%A6%AC%EC%95%84_%EB%82%B4%EC%A0%84%EC%9D%98_%EB%82%9C%EB%AF%BC" title="시리아 내전의 난민">난민</a> 문제가 <a href="/wiki/%EB%B8%94%EB%9D%BC%EB%94%94%EB%AF%B8%EB%A5%B4_%ED%91%B8%ED%8B%B4" title="블라디미르 푸틴">블라디미르 푸틴</a>의 <a href="/wiki/%EC%8B%9C%EB%A6%AC%EC%95%84_%EB%82%B4%EC%A0%84" title="시리아 내전">시리아 내전</a> 개입으로 해결이 된다고 믿고 있고, 크림 반도 사태에 대응하여 러시아에 가해진 경제적 제재에서 유럽의 경제 손실이 크다고 생각하기 때문이다.<sup class="reference" id="cite_ref-32"><a href="#cite_note-32">[32]</a></sup> 또한 현재 AfD의 의석이 존재하는 대부분의 지역이 과거 <a href="/wiki/%EB%8F%99%EB%8F%85" title="동독">동독</a> 지역인데, 이 지역의 주민들은 자신의 빈곤이 <a href="/wiki/%EB%8F%99%EB%8F%85" title="동독">동독</a> 시절에 비해 커졌다고 생각하기 때문에,<sup class="reference" id="cite_ref-33"><a href="#cite_note-33">[33]</a></sup> <a href="/wiki/%EA%B3%B5%EC%82%B0%EC%A3%BC%EC%9D%98" title="공산주의">공산주의</a>에 대한 향수로 <a class="mw-redirect" href="/wiki/%EC%B9%9C%EB%9F%AC%EC%8B%9C%EC%95%84" title="친러시아">친러시아</a>적 성향을 띠고 있어서, 이것도 AfD의 <a class="mw-redirect" href="/wiki/%EC%B9%9C%EB%9F%AC%EC%8B%9C%EC%95%84" title="친러시아">친러시아</a> 성향에 영향을 주고있다.</p>
<h3><span class="mw-headline" id=".EA.B8.B0.ED.83.80">기타</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%8F%85%EC%9D%BC%EC%9D%84_%EC%9C%84%ED%95%9C_%EB%8C%80%EC%95%88&amp;action=edit&amp;section=15" title="부분 편집: 기타">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>이 외에도 다른 정책 및 핵심 문제들을 다루고 있다. 먼저 기존 <a class="mw-disambig" href="/wiki/%EB%B3%B4%EC%88%98%EB%8B%B9" title="보수당">보수당</a>이 가지고 있는 반<a href="/wiki/%EB%8F%99%EC%84%B1%EC%95%A0" title="동성애">동성애</a> 정책에 찬성하고 있으며, 최근 <a href="/wiki/%EC%8B%9C%EB%A6%AC%EC%95%84_%EB%82%B4%EC%A0%84%EC%9D%98_%EB%82%9C%EB%AF%BC" title="시리아 내전의 난민">난민</a>문제의 여파로 굵어진 <a class="mw-redirect" href="/wiki/%EC%9D%98%EB%AC%B4%EA%B5%90%EC%9C%A1" title="의무교육">의무교육</a> 폐지 및 <a class="mw-redirect" href="/wiki/%ED%99%88%EC%8A%A4%EC%BF%A8%EB%A7%81" title="홈스쿨링">홈스쿨링</a> 도입 역시 추진하고 있다.<sup class="reference" id="cite_ref-34"><a href="#cite_note-34">[34]</a></sup> 그리고 <a href="/wiki/%EB%8F%85%EC%9D%BC" title="독일">독일</a> 국민의 대다수가 찬성하는 <a href="/wiki/%EC%97%90%EB%93%9C%EC%9B%8C%EB%93%9C_%EC%8A%A4%EB%85%B8%EB%93%A0" title="에드워드 스노든">에드워드 스노든</a>(Edward Snowden)의 <a href="/wiki/%EB%A7%9D%EB%AA%85" title="망명">망명</a>문제에 대해서도 빠르게 허용할 것을 촉구하고 있다.<sup class="reference" id="cite_ref-35"><a href="#cite_note-35">[35]</a></sup></p>
<h2><span class="mw-headline" id=".EA.B4.80.EB.A0.A8.EB.8B.A8.EC.B2.B4">관련단체</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%8F%85%EC%9D%BC%EC%9D%84_%EC%9C%84%ED%95%9C_%EB%8C%80%EC%95%88&amp;action=edit&amp;section=16" title="부분 편집: 관련단체">편집</a><span class="mw-editsection-bracket">]</span></span></h2>

<p>AfD는 <a class="mw-redirect" href="/wiki/%ED%8E%98%EA%B8%B0%EB%8B%A4" title="페기다">페기다</a>(PEGIDA)와 성향이 비슷한데, 당은 비록 <a class="mw-redirect" href="/wiki/%ED%8E%98%EA%B8%B0%EB%8B%A4" title="페기다">페기다</a>와의 연합은 하지 않지만 옹호하고 있으며, <a class="mw-redirect" href="/wiki/%ED%8E%98%EA%B8%B0%EB%8B%A4" title="페기다">페기다</a>의 대부분의 사람들이 AfD를 지지한다.<sup class="reference" id="cite_ref-36"><a href="#cite_note-36">[36]</a></sup> 구 <a href="/wiki/%EB%8F%99%EB%8F%85" title="동독">동독</a>지역이었던 <a href="/wiki/%EB%93%9C%EB%A0%88%EC%8A%A4%EB%8D%B4" title="드레스덴">드레스덴</a>(Dresden)에서 설립된 <a class="mw-redirect" href="/wiki/%ED%8E%98%EA%B8%B0%EB%8B%A4" title="페기다">페기다</a>는 반<a class="mw-redirect" href="/wiki/%EC%9D%B4%EC%8A%AC%EB%9E%8C" title="이슬람">이슬람</a>(<a href="/wiki/%EC%8B%9C%EB%A6%AC%EC%95%84_%EB%82%B4%EC%A0%84%EC%9D%98_%EB%82%9C%EB%AF%BC" title="시리아 내전의 난민">난민</a>), 반외국인 정책을 주장하면서 <a href="/wiki/%EC%95%99%EA%B2%94%EB%9D%BC_%EB%A9%94%EB%A5%B4%EC%BC%88" title="앙겔라 메르켈">메르켈</a>의 퇴진을 요구한다.</p>
<h3><span class="mw-headline" id=".EB.8F.85.EC.9D.BC.EC.9D.84_.EC.9C.84.ED.95.9C_.EC.B2.AD.EC.86.8C.EB.85.84_.EB.8C.80.EC.95.88.28JA.29">독일을 위한 청소년 대안(JA)</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%8F%85%EC%9D%BC%EC%9D%84_%EC%9C%84%ED%95%9C_%EB%8C%80%EC%95%88&amp;action=edit&amp;section=18" title="부분 편집: 독일을 위한 청소년 대안(JA)">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>독일을 위한 청소년 대안(Junge Alternative für Deutschland, JA)은 2013년 6월 <a href="/wiki/%EB%8B%A4%EB%A6%84%EC%8A%88%ED%83%80%ED%8A%B8" title="다름슈타트">다름슈타트</a>에서 설립되었다. JA는 14세부터 35세까지 활동이 가능하지만, 그 외의 연령대의 사람들도 회원으로 허용할 예정이라고 한다. 현재는 900명 정도가 소속되어 있으며, '이념보다는 상식'을 슬로건으로 삼고 있다. 그리고 토론이나 교육활동 등도 제공하고 있다.<sup class="reference" id="cite_ref-37"><a href="#cite_note-37">[37]</a></sup> 표면상으로는 AfD에 부속된 청소년 정당으로 보이지만, 실제로는 법적으로나 구조적으로나 AfD와는 별개의 정당이다.<sup class="reference" id="cite_ref-38"><a href="#cite_note-38">[38]</a></sup> 이런 JA의 독립성은 AfD의 체계가 완전하지 않음을 보여준다. JA는 독일 언론에 따르면 극우주의적 성향을 가진다고 평가되고 있으며,<sup class="reference" id="cite_ref-39"><a href="#cite_note-39">[39]</a></sup> 정치적으로 미숙하며 <a href="/wiki/%EB%B0%98%EC%97%AC%EC%84%B1%EC%A3%BC%EC%9D%98" title="반여성주의">반여성주의</a>적 성향을 보이고 있다.<sup class="reference" id="cite_ref-40"><a href="#cite_note-40">[40]</a></sup></p>
<h4><span class="mw-headline" id=".EC.B2.AD.EC.86.8C.EB.85.84_.EC.A0.95.EB.8B.B9.EC.9D.98_.EB.A7.8C.ED.96.89">청소년 정당의 만행</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%8F%85%EC%9D%BC%EC%9D%84_%EC%9C%84%ED%95%9C_%EB%8C%80%EC%95%88&amp;action=edit&amp;section=19" title="부분 편집: 청소년 정당의 만행">편집</a><span class="mw-editsection-bracket">]</span></span></h4>
<p>2014년 3월, JA는 당의 <a href="/wiki/%EC%BE%B0%EB%A5%B8" title="쾰른">쾰른</a>에 있는 노르트라인베스트팔렌(North Rhine-Westphila) 지부의 문제를 해결하기 위해 영국의 정치인 <a href="/wiki/%EB%82%98%EC%9D%B4%EC%A0%88_%ED%8C%A8%EB%9D%BC%EC%A7%80" title="나이절 패라지">나이절 패라지</a>를 초청했다.<sup class="reference" id="cite_ref-41"><a href="#cite_note-41">[41]</a></sup> 그러나 이 초대는 JA에게 문제가 되었다. 이러한 외부 인사에 대한 초청은 AfD의 연방 위원들에 의해서만 결정될 수 있다는, AfD의 정책에 반하는 것이었기 때문이다.<sup class="reference" id="cite_ref-42"><a href="#cite_note-42">[42]</a></sup> 이에 대해 루커를 비롯한 AfD의 지도자들은 ‘정치적인 미숙함’이라고 비난하였다.<sup class="reference" id="cite_ref-43"><a href="#cite_note-43">[43]</a></sup></p>
<p>또한 JA는 <a href="/wiki/%EA%B5%AD%EC%A0%9C_%EC%97%AC%EC%84%B1%EC%9D%98_%EB%82%A0" title="국제 여성의 날">국제 여성의 날</a>에 대한 지지를 보인 젊은 사회 민주주의(young Social Democrats)에 대응하기 위해 평등주의 대신 같음(Gleichberechtigung statt Gleichmacherei)이라는 캠페인을 시작하고 <a href="/wiki/%ED%8E%98%EC%9D%B4%EC%8A%A4%EB%B6%81" title="페이스북">페이스북</a> 페이지를 만들었다. 이곳에서 JA는 <a href="/wiki/%EC%97%AC%EC%84%B1%EC%A3%BC%EC%9D%98" title="여성주의">여성주의</a>를 <a href="/wiki/%EC%A2%8C%EC%9D%B5" title="좌익">좌익</a>사상에 따른 행동이라고 주장하고, 이를 저지할 것을 주장했다.<sup class="reference" id="cite_ref-44"><a href="#cite_note-44">[44]</a></sup> 또한 JA는 독일에 있는 <a class="new" href="/w/index.php?title=%EC%97%AC%EC%84%B1_%ED%95%A0%EB%8B%B9%EC%A0%9C&amp;action=edit&amp;redlink=1" title="여성 할당제 (없는 문서)">여성 할당제</a>의 철폐를 주장했다.<sup class="reference" id="cite_ref-45"><a href="#cite_note-45">[45]</a></sup> 거기에다 2014년 5월, JA는 범죄에 대한 <a href="/wiki/%EC%9E%90%EA%B2%BD%EB%8B%A8" title="자경단">자경단</a> 활동을 옹호하였다. 이는 폭력행위에 대한 옹호와도 이어질 수 있는 것으로, 이런 주장으로 인해 JA는 AfD의 지도자들과도 충돌하였다.<sup class="reference" id="cite_ref-46"><a href="#cite_note-46">[46]</a></sup></p>
<h2><span class="mw-headline" id=".EC.84.A0.EA.B1.B0_.EA.B2.B0.EA.B3.BC">선거 결과</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%8F%85%EC%9D%BC%EC%9D%84_%EC%9C%84%ED%95%9C_%EB%8C%80%EC%95%88&amp;action=edit&amp;section=20" title="부분 편집: 선거 결과">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<h4><span class="mw-headline" id=".EB.8F.85.EC.9D.BC_.EC.97.B0.EB.B0.A9.EC.9D.98.ED.9A.8C_.EC.84.A0.EA.B1.B0">독일 연방의회 선거</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%8F%85%EC%9D%BC%EC%9D%84_%EC%9C%84%ED%95%9C_%EB%8C%80%EC%95%88&amp;action=edit&amp;section=21" title="부분 편집: 독일 연방의회 선거">편집</a><span class="mw-editsection-bracket">]</span></span></h4>

<p>AfD는 2013년 9월 22일 시행된 제 18대 <a href="/wiki/%EB%8F%85%EC%9D%BC_%EC%97%B0%EB%B0%A9%EC%9D%98%ED%9A%8C" title="독일 연방의회">독일 연방의회</a> 총선거에서 <a href="/wiki/%EC%9C%A0%EB%A1%9C%EC%A1%B4" title="유로존">유로존</a> 탈퇴라는 공약을 내걸고 등장했다. 이 선거에서 4.7%의 지지율을 획득하였으나 0.3%의 차이로 <a href="/wiki/5%25_%EC%A0%80%EC%A7%80_%EC%A1%B0%ED%95%AD" title="5% 저지 조항">5%</a>를 넘지 못해 의회에 진입하지는 못하였다.</p>
<h4><span class="mw-headline" id=".EC.9C.A0.EB.9F.BD.EC.9D.98.ED.9A.8C_.EC.84.A0.EA.B1.B0">유럽의회 선거</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%8F%85%EC%9D%BC%EC%9D%84_%EC%9C%84%ED%95%9C_%EB%8C%80%EC%95%88&amp;action=edit&amp;section=22" title="부분 편집: 유럽의회 선거">편집</a><span class="mw-editsection-bracket">]</span></span></h4>
<p>2014년 5월 25일 <a class="mw-redirect" href="/wiki/%EC%9C%A0%EB%9F%BD%EC%9D%98%ED%9A%8C" title="유럽의회">유럽의회</a> 선거가 치러졌는데, 이 선거에서는 <a href="/wiki/%EC%9C%A0%EB%9F%BD%ED%9A%8C%EC%9D%98%EC%A3%BC%EC%9D%98" title="유럽회의주의">유럽회의주의</a>, <a class="mw-redirect" href="/wiki/%EB%8C%80%EC%A4%91%EC%98%81%ED%95%A9%EC%A3%BC%EC%9D%98" title="대중영합주의">대중영합주의</a> 정당이 큰 인기를 끌었다.<sup class="reference" id="cite_ref-47"><a href="#cite_note-47">[47]</a></sup> AfD 또한 이러한 분위기 속에서 7.1%의 지지율을 얻으면서 7석의 의석을 획득하였다. 현재는 5명이 탈당을 한 상태라 2명만이 남아있다.</p>
<h4><span class="mw-headline" id=".EC.A3.BC.EC.9D.98.ED.9A.8C_.EC.84.A0.EA.B1.B0">주의회 선거</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%8F%85%EC%9D%BC%EC%9D%84_%EC%9C%84%ED%95%9C_%EB%8C%80%EC%95%88&amp;action=edit&amp;section=23" title="부분 편집: 주의회 선거">편집</a><span class="mw-editsection-bracket">]</span></span></h4>
<p>2016년 선거에서는 <a href="/wiki/%EC%95%99%EA%B2%94%EB%9D%BC_%EB%A9%94%EB%A5%B4%EC%BC%88" title="앙겔라 메르켈">메르켈</a> 총리의 난민 정책의 영향이 컸다. <a href="/wiki/%EB%8F%85%EC%9D%BC_%EA%B8%B0%EB%8F%85%EA%B5%90%EB%AF%BC%EC%A3%BC%EC%97%B0%ED%95%A9" title="독일 기독교민주연합">기민당</a> 내부에서도 <a href="/wiki/%EC%95%99%EA%B2%94%EB%9D%BC_%EB%A9%94%EB%A5%B4%EC%BC%88" title="앙겔라 메르켈">메르켈</a> 총리의 개방적인 <a href="/wiki/%EC%8B%9C%EB%A6%AC%EC%95%84_%EB%82%B4%EC%A0%84%EC%9D%98_%EB%82%9C%EB%AF%BC" title="시리아 내전의 난민">난민</a> 정책에 대해서 반대하는 사람이 많을 정도로 이 정책은 개방적인 것이었다.<sup class="reference" id="cite_ref-48"><a href="#cite_note-48">[48]</a></sup> 기존 보수 투표자들의 <a href="/wiki/%EC%95%99%EA%B2%94%EB%9D%BC_%EB%A9%94%EB%A5%B4%EC%BC%88" title="앙겔라 메르켈">메르켈</a> 총리의 급진정책에 대한 반발로 <a href="/wiki/%EB%8F%85%EC%9D%BC_%EA%B8%B0%EB%8F%85%EA%B5%90%EB%AF%BC%EC%A3%BC%EC%97%B0%ED%95%A9" title="독일 기독교민주연합">기민당</a> 표가 분산되어 AfD 지지율이 급상승하게 되었다.<sup class="reference" id="cite_ref-49"><a href="#cite_note-49">[49]</a></sup></p>
