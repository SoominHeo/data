<p><b>돈지덕</b>(頓智德, <a href="/wiki/1980%EB%85%84" title="1980년">1980년</a> <a href="/wiki/4%EC%9B%94_28%EC%9D%BC" title="4월 28일">4월 28일</a> ~ )은 <a href="/wiki/%EB%8C%80%ED%95%9C%EB%AF%BC%EA%B5%AD" title="대한민국">대한민국</a>의 <a href="/wiki/%EC%B6%95%EA%B5%AC" title="축구">축구</a> 선수로서 포지션은 수비수이다.</p>
<h2><span class="mw-headline" id=".EC.B6.95.EA.B5.AC.EC.9D.B8_.EA.B2.BD.EB.A0.A5">축구인 경력</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%8F%88%EC%A7%80%EB%8D%95&amp;action=edit&amp;section=1" title="부분 편집: 축구인 경력">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<h3><span class="mw-headline" id=".EC.84.A0.EC.88.98_.EA.B2.BD.EB.A0.A5">선수 경력</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%8F%88%EC%A7%80%EB%8D%95&amp;action=edit&amp;section=2" title="부분 편집: 선수 경력">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>돈지덕은 <a href="/wiki/%EC%9D%B8%EC%B2%9C%EB%8C%80%ED%95%99%EA%B5%90" title="인천대학교">인천대학교</a>를 졸업하고 <a href="/wiki/2003%EB%85%84" title="2003년">2003년</a> 실업 축구 <a href="/wiki/%EA%B3%A0%EC%96%91_KB%EA%B5%AD%EB%AF%BC%EC%9D%80%ED%96%89_%EC%B6%95%EA%B5%AC%EB%8B%A8" title="고양 KB국민은행 축구단">국민은행 축구단</a>에 입단하였다.</p>
<p><a href="/wiki/2013%EB%85%84" title="2013년">2013년</a> 드래프트에서 <a href="/wiki/FC_%EC%95%88%EC%96%91" title="FC 안양">FC 안양</a>에 우선 지명되어 34세의 나이에 프로 무대에 첫 발을 디뎠다.</p>
<p>2014 시즌을 앞두고 <a href="/wiki/%EB%82%B4%EC%85%94%EB%84%90%EB%A6%AC%EA%B7%B8_(%EB%8C%80%ED%95%9C%EB%AF%BC%EA%B5%AD)" title="내셔널리그 (대한민국)">내셔널리그</a>의 <a href="/wiki/%EA%B2%BD%EC%A3%BC_%ED%95%9C%EA%B5%AD%EC%88%98%EB%A0%A5%EC%9B%90%EC%9E%90%EB%A0%A5_%EC%B6%95%EA%B5%AC%EB%8B%A8" title="경주 한국수력원자력 축구단">경주 한국수력원자력</a>으로 이적하였다.</p>
