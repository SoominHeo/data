<p><b>동</b>(洞)은 <a href="/wiki/%EB%8C%80%ED%95%9C%EB%AF%BC%EA%B5%AD" title="대한민국">대한민국</a>의 <a href="/wiki/%ED%96%89%EC%A0%95_%EA%B5%AC%EC%97%AD" title="행정 구역">행정 구역</a>의 하나로 <a href="/wiki/%EA%B5%AC_(%ED%96%89%EC%A0%95_%EA%B5%AC%EC%97%AD)" title="구 (행정 구역)">구</a> 또는 <a class="mw-redirect" href="/wiki/%EC%9E%90%EC%B9%98%EC%8B%9C" title="자치시">시</a>의 하부 행정 구역이다. <b>행정운영동</b>(行政運營洞) 또는 <b>행정동</b>(行政洞)이라고 불리어 <a href="/wiki/%EB%B2%95%EC%A0%95%EB%8F%99" title="법정동">법정동</a>과 구별하고 있으며, 모든 행정동 청사는 <a class="mw-redirect" href="/wiki/%EB%8F%99%EC%A3%BC%EB%AF%BC%EC%84%BC%ED%84%B0" title="동주민센터">동주민센터</a>(동사무소)라고 부르고 있다.</p>

<li>하나의 <a href="/wiki/%EB%B2%95%EC%A0%95%EB%8F%99" title="법정동">법정동</a>을 하나의 행정동이 관할하는 경우: 대체로 행정동의 명칭은 <a href="/wiki/%EB%B2%95%EC%A0%95%EB%8F%99" title="법정동">법정동</a>과 동일하게 정해진다.<br/>
(예: 행정동인 <a href="/wiki/%EC%88%98%EC%9B%90%EC%8B%9C" title="수원시">수원시</a> <a href="/wiki/%EA%B6%8C%EC%84%A0%EA%B5%AC" title="권선구">권선구</a> <a href="/wiki/%EA%B5%AC%EC%9A%B4%EB%8F%99" title="구운동">구운동</a>은 온전한 구운동 전체만을 관할함)</li>
<li>하나의 법정동을 복수의 행정동이 일부분씩 관할하는 경우: 행정동의 이름은 대체로 해당 법정동의 이름에 숫자를 붙여 구분한다.<br/>
(예: 법정동인 <a href="/wiki/%EB%B6%80%EC%82%B0%EA%B4%91%EC%97%AD%EC%8B%9C" title="부산광역시">부산광역시</a> <a href="/wiki/%EB%8F%99%EB%9E%98%EA%B5%AC" title="동래구">동래구</a> 온천동은 온천1동, 온천2동, 온천3동 세 개의 행정동이 각각 관할함)</li>
<li>복수의 법정동을 하나의 행정동이 관할하는 경우:
<ol>
<li>행정동의 이름은 비교적 규모나 대표성이 큰 법정동명 하나를 따서 짓는다.<br/>
(예: 법정동인 <a href="/wiki/%EA%B2%BD%EA%B8%B0%EB%8F%84" title="경기도">경기도</a> <a href="/wiki/%EA%B3%A0%EC%96%91%EC%8B%9C" title="고양시">고양시</a> <a href="/wiki/%EB%8D%95%EC%96%91%EA%B5%AC" title="덕양구">덕양구</a> <a href="/wiki/%EA%B3%A0%EC%96%91%EB%8F%99" title="고양동">고양동</a>과 <a class="mw-redirect" href="/wiki/%EB%B2%BD%EC%A0%9C%EB%8F%99" title="벽제동">벽제동</a> 그리고 <a class="mw-redirect" href="/wiki/%EB%8C%80%EC%9E%90%EB%8F%99" title="대자동">대자동</a>과 <a class="mw-redirect" href="/wiki/%EC%84%A0%EC%9C%A0%EB%8F%99" title="선유동">선유동</a>은 '<a href="/wiki/%EA%B3%A0%EC%96%91%EB%8F%99" title="고양동">고양동</a>' 하나의 행정동이 관할함)</li>
<li>행정동 관할 내 법정동들의 이름의 맨 앞 글자를 조합하기도 한다.<br/>
(예: 법정동인 <a href="/wiki/%EA%B2%BD%EC%83%81%EB%82%A8%EB%8F%84" title="경상남도">경상남도</a> <a href="/wiki/%EA%B9%80%ED%95%B4%EC%8B%9C" title="김해시">김해시</a> 삼방동과 안동의 행정동 명칭은 <a href="/wiki/%EC%82%BC%EC%95%88%EB%8F%99" title="삼안동">삼안동</a>)</li>
<li>행정동 관할 내 법정동들의 이름의 맨 뒷 글자와 앞 글자 또는 맨 앞 글자와 뒷 글자를 조합하기도 한다.<br/>
(예:법정동인 <a href="/wiki/%EA%B2%BD%EA%B8%B0%EB%8F%84" title="경기도">경기도</a> <a href="/wiki/%EA%B3%A0%EC%96%91%EC%8B%9C" title="고양시">고양시</a> <a href="/wiki/%EB%8D%95%EC%96%91%EA%B5%AC" title="덕양구">덕양구</a> <a class="mw-redirect" href="/wiki/%EC%9B%90%ED%9D%A5%EB%8F%99" title="원흥동">원흥동</a>과 <a class="mw-redirect" href="/wiki/%EB%8F%84%EB%82%B4%EB%8F%99" title="도내동">도내동</a>의 행정동 명칭은 <a href="/wiki/%ED%9D%A5%EB%8F%84%EB%8F%99" title="흥도동">흥도동</a>)</li>
<li>행정동 관할 내 법정동들의 이름 전체를 조합하기도 한다.<br/>
(예: 법정동인 <a href="/wiki/%EC%9D%B8%EC%B2%9C%EA%B4%91%EC%97%AD%EC%8B%9C" title="인천광역시">인천광역시</a> <a href="/wiki/%EB%82%A8%EB%8F%99%EA%B5%AC" title="남동구">남동구</a> <a class="mw-disambig" href="/wiki/%EC%9E%A5%EC%88%98%EB%8F%99" title="장수동">장수동</a> + <a class="mw-disambig" href="/wiki/%EC%84%9C%EC%B0%BD%EB%8F%99" title="서창동">서창동</a> = 행정동인 <a href="/wiki/%EC%9E%A5%EC%88%98%EC%84%9C%EC%B0%BD%EB%8F%99" title="장수서창동">장수서창동</a>)</li>

</li>
<li>하나의 행정동이 어떤 법정동의 일부와 또다른 법정동의 일부에 걸쳐 관할하기도 있다.<br/>
(예: 행정동인 <a href="/wiki/%EC%84%9C%EC%9A%B8%ED%8A%B9%EB%B3%84%EC%8B%9C" title="서울특별시">서울</a> <a href="/wiki/%EA%B0%95%EB%82%A8%EA%B5%AC" title="강남구">강남구</a> <a href="/wiki/%EC%8B%A0%EC%82%AC%EB%8F%99_(%EA%B0%95%EB%82%A8%EA%B5%AC)" title="신사동 (강남구)">신사동</a>은 법정동인 <a href="/wiki/%EC%95%95%EA%B5%AC%EC%A0%95%EB%8F%99" title="압구정동">압구정동</a>의 서쪽 1/3과 신사동의 서쪽 1/3을 관할함)</li>
<li>읍/면을 동으로 전환할 때, 기존 읍/면의 명칭을 따라 정하기도 한다.<br/>
(예 - 김해시 장유면: 장유1동 ~ 장유3동, 양주군 회천읍: 양주시 회천1동 ~ 회천4동, 당진군 당진읍: 당진시 당진1동 ~ 당진3동)</li>
<li>행정동의 이름이 반드시 법정동과 직접 연관되지 않을 수도 있다.<br/>
(예: 행정동인 <a href="/wiki/%EA%B4%91%EC%A3%BC%EA%B4%91%EC%97%AD%EC%8B%9C" title="광주광역시">광주광역시</a> <a href="/wiki/%EA%B4%91%EC%82%B0%EA%B5%AC" title="광산구">광산구</a> <a href="/wiki/%EC%96%B4%EB%A3%A1%EB%8F%99_(%EA%B4%91%EC%A3%BC)" title="어룡동 (광주)">어룡동</a>은 법정동인 박호동, 서봉동, 선암동, 운수동, 소촌동을 관할함. <a href="/wiki/%EA%B2%BD%EA%B8%B0%EB%8F%84" title="경기도">경기도</a> <a href="/wiki/%EC%9D%98%EC%99%95%EC%8B%9C" title="의왕시">의왕시</a> <a href="/wiki/%EB%B6%80%EA%B3%A1%EB%8F%99_(%EC%9D%98%EC%99%95%EC%8B%9C)" title="부곡동 (의왕시)">부곡동</a>은 삼동, 이동, 월암동, 초평동을 관할함.)</li>
</ol>
<h2><span class="mw-headline" id=".EC.97.AD.EC.82.AC">역사</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%8F%99_(%ED%96%89%EC%A0%95_%EA%B5%AC%EC%97%AD)&amp;action=edit&amp;section=2" title="부분 편집: 역사">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>행정동의 기원은 <a href="/wiki/%EC%9D%BC%EC%A0%9C_%EA%B0%95%EC%A0%90%EA%B8%B0" title="일제 강점기">일제 강점기</a>의 정회(町會)에 두고 있다. 정회는 <a href="/wiki/%EC%A0%95_(%ED%96%89%EC%A0%95_%EA%B5%AC%EC%97%AD)" title="정 (행정 구역)">정</a>(<a href="/wiki/%EC%9D%BC%EB%B3%B8%EC%96%B4" title="일본어">일본어</a> : 町), 정목(丁目)을 단위로 하여 설치된 부(府)의 말단협력 보조기관으로 예컨대 당시 <a href="/wiki/%EA%B2%BD%EC%84%B1%EB%B6%80" title="경성부">경성부</a>에는 280여 개의 정회가 있었다. 이 정회 제도는 <a href="/wiki/%EC%9D%BC%EC%A0%9C_%EA%B0%95%EC%A0%90%EA%B8%B0" title="일제 강점기">일제 강점기</a> 말기에 생긴 것으로 군정기에도 그대로 유지되었다.<sup class="reference" id="cite_ref-1"><a href="#cite_note-1">[1]</a></sup> 정(町)을 동(洞)으로, 통(通)을 로(路)로, 정목(丁目)을 가(街)로 명칭 환원이 이루어질 때 동회로 개칭하여 사용하였고, 점차 동으로 명칭 일원화가 되었다.</p>
<p>행정 구역 동은 초기에 법정동과 행정동을 구분하지 않았다. 법정동이 곧 행정 구역 단위였다. 그러나 세분화된 동은 행정력 낭비로 이어졌고, 이를 개선하기 위하여 여러 법정동을 묶을 필요가 생겨났다. 그리하여 기존 동과 구별되는 명칭으로 동회(洞會)를 사용하였다(예: <a href="/wiki/1965%EB%85%84" title="1965년">1965년</a> <a href="/wiki/%EB%8C%80%EA%B5%AC%EA%B4%91%EC%97%AD%EC%8B%9C" title="대구광역시">대구</a> <a href="/wiki/%EC%A4%91%EA%B5%AC_(%EB%8C%80%EA%B5%AC%EA%B4%91%EC%97%AD%EC%8B%9C)" title="중구 (대구광역시)">중구</a>에서는 시내 번화가의 38개의 법정동을 동성동회, 남성동회, 종로동회, 북성동회, 서성동회로 나누었다). 반면 넓은 동을 나누어야 하는 경우도 발생하였고 이 경우에는 동구(洞區)를 사용하였다.(예: 1965년 <a href="/wiki/%EB%8C%80%EA%B5%AC%EA%B4%91%EC%97%AD%EC%8B%9C" title="대구광역시">대구</a> <a href="/wiki/%EC%84%9C%EA%B5%AC_(%EB%8C%80%EA%B5%AC%EA%B4%91%EC%97%AD%EC%8B%9C)" title="서구 (대구광역시)">서구</a>에서는 내당동을 내당동1구, 내당동2구, 내당동3구, 내당동4구로 나누었다.) 이렇게 행정동이 생겨났다. 1960년대 후반 동회는 모두 동으로 명칭 일원화가 이루어졌다.</p>
<p>현재 <a href="/wiki/%EC%A0%84%EB%9D%BC%EB%82%A8%EB%8F%84" title="전라남도">전라남도</a> <a href="/wiki/%EB%B3%B4%EC%84%B1%EA%B5%B0" title="보성군">보성군</a> <a href="/wiki/%EB%B3%B4%EC%84%B1%EC%9D%8D" title="보성읍">보성읍</a>은 <a href="/wiki/%EB%A6%AC_(%ED%96%89%EC%A0%95_%EA%B5%AC%EC%97%AD)" title="리 (행정 구역)">리</a>의 하위 행정 구역에 반 대신 동을 사용하고 있다.</p>
