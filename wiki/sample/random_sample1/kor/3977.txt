<p><b>동경 16도</b>(東經16度)는 <a href="/wiki/%EB%B3%B8%EC%B4%88_%EC%9E%90%EC%98%A4%EC%84%A0" title="본초 자오선">본초 자오선</a>에서 동쪽으로 16도 떨어진 <a href="/wiki/%EA%B2%BD%EC%84%A0" title="경선">경선</a>이다. <a href="/wiki/%EB%B6%81%EA%B7%B9%EC%A0%90" title="북극점">북극점</a>에서 시작하여 <a href="/wiki/%EB%B6%81%EA%B7%B9%ED%95%B4" title="북극해">북극해</a>, <a href="/wiki/%EC%9C%A0%EB%9F%BD" title="유럽">유럽</a>, <a href="/wiki/%EC%95%84%ED%94%84%EB%A6%AC%EC%B9%B4" title="아프리카">아프리카</a>, <a href="/wiki/%EB%8C%80%EC%84%9C%EC%96%91" title="대서양">대서양</a>, <a href="/wiki/%EB%82%A8%EA%B7%B9%ED%95%B4" title="남극해">남극해</a>, <a href="/wiki/%EB%82%A8%EA%B7%B9" title="남극">남극</a>을 거쳐 <a href="/wiki/%EB%82%A8%EA%B7%B9%EC%A0%90" title="남극점">남극점</a>에 이른다.</p>
<p>동경 16도는 <a class="new" href="/w/index.php?title=%EC%84%9C%EA%B2%BD_164%EB%8F%84&amp;action=edit&amp;redlink=1" title="서경 164도 (없는 문서)">서경 164도</a>와 이어져 지구의 둘레를 이룬다.</p>
<p><br/>
<a href="/wiki/%EB%B6%81%EA%B7%B9%EC%A0%90" title="북극점">북극점</a>에서 <a href="/wiki/%EB%82%A8%EA%B7%B9%EC%A0%90" title="남극점">남극점</a>까지 동경 16도선은 다음 지역을 지나간다.</p>
