<p><b>엔싱크</b>('N Sync)는 <a href="/wiki/%EB%AF%B8%EA%B5%AD" title="미국">미국</a>의 <a class="mw-disambig" href="/wiki/%ED%8C%9D" title="팝">팝</a>, <a class="mw-redirect" href="/wiki/R%26B" title="R&amp;B">R&amp;B</a> <a class="mw-redirect" href="/wiki/%EC%9D%8C%EC%95%85_%EA%B7%B8%EB%A3%B9" title="음악 그룹">음악 그룹</a>이다. 그는 음악 매니저인 <a class="new" href="/w/index.php?title=%EB%A1%9C%EC%9A%B0_%ED%8E%84%EB%A8%BC&amp;action=edit&amp;redlink=1" title="로우 펄먼 (없는 문서)">로우 펄먼</a>으로부터 <a href="/wiki/1995%EB%85%84" title="1995년">1995년</a> 픽업됐으며, 지금까지 5600만장 넘는 판매고를 자랑하고 있다.</p>
<p>이 그룹은 <a href="/wiki/1995%EB%85%84" title="1995년">1995년</a> <a href="/wiki/%ED%94%8C%EB%A1%9C%EB%A6%AC%EB%8B%A4_%EC%A3%BC" title="플로리다 주">플로리다 주</a>에서 결성됐으며, 맴버로는 <a href="/wiki/%EC%A0%80%EC%8A%A4%ED%8B%B4_%ED%8C%80%EB%B2%84%EB%A0%88%EC%9D%B4%ED%81%AC" title="저스틴 팀버레이크">저스틴 팀버레이크</a>,<a class="new" href="/w/index.php?title=%EC%A0%9C%EC%9D%B4%EC%8B%9C_%EC%B2%B4%EC%A6%88&amp;action=edit&amp;redlink=1" title="제이시 체즈 (없는 문서)">제이시 체즈</a>, <a class="new" href="/w/index.php?title=%EB%A0%8C%EC%8A%A4_%EB%B0%94%EC%8A%A4&amp;action=edit&amp;redlink=1" title="렌스 바스 (없는 문서)">렌스 바스</a>, <a class="new" href="/w/index.php?title=%EC%A1%B0%EC%9D%B4_%ED%8C%BB%ED%86%A4&amp;action=edit&amp;redlink=1" title="조이 팻톤 (없는 문서)">조이 팻톤</a>, <a class="new" href="/w/index.php?title=%ED%81%AC%EB%A6%AC%EC%8A%A4_%EC%BB%A4%ED%81%AC%ED%8E%98%ED%8A%B8%EB%A6%AD&amp;action=edit&amp;redlink=1" title="크리스 커크페트릭 (없는 문서)">크리스 커크페트릭</a>으로 구성됐었다. 이 그룹의 약자는 구성원 이름의 한 부분을 따서 만들었으며, 그들은 <a href="/wiki/%EC%86%8C%EB%8B%88_%EB%AE%A4%EC%A7%81_%EC%97%94%ED%84%B0%ED%85%8C%EC%9D%B8%EB%A8%BC%ED%8A%B8" title="소니 뮤직 엔터테인먼트">소니 뮤직 엔터테인먼트</a>로부터 픽업됐다. 그들의 두 번째 앨범인 'No Strings Attached'은 첫주에만 110만 장이 팔렸고, 전 세계적으로는 한 달 만에 240만 장을 돌파했다. 그들은 <a href="/wiki/%EC%98%AC%EB%A6%BC%ED%94%BD" title="올림픽">올림픽</a>이나 각종 시상식에서 공연을 선보였다.</p>
<p><a href="/wiki/2002%EB%85%84" title="2002년">2002년</a> 공식 해체했다.</p>
