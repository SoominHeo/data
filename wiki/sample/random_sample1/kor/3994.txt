<h1 class="firstHeading" id="firstHeading" lang="ko">동기 디지털 가입자 회선</h1>
<div class="mw-body-content" id="bodyContent">
<div id="siteSub">위키백과, 우리 모두의 백과사전.</div>
<div id="contentSub"></div>
<div class="mw-jump" id="jump-to-nav">
					이동:					<a href="#mw-head">둘러보기</a>, 					<a href="#p-search">검색</a>
</div>
<div class="mw-content-ltr" dir="ltr" id="mw-content-text" lang="ko"><script>function mfTempOpenSection(id){var block=document.getElementById("mf-section-"+id);block.className+=" open-block";block.previousSibling.className+=" open-block";}</script><p><b>동기 디지털 가입자 회선</b>(<b>S</b>ymmetric <b>D</b>igital <b>S</b>ubscriber <b>L</b>ine, <b>SDSL</b>)은 <a class="new" href="/w/index.php?title=T_%EC%BA%90%EB%A6%AC%EC%96%B4&amp;action=edit&amp;redlink=1" title="T 캐리어 (없는 문서)">E1</a>과 같은 데이터 속도를 사용하는 <a href="/wiki/%EB%94%94%EC%A7%80%ED%84%B8_%EA%B0%80%EC%9E%85%EC%9E%90_%ED%9A%8C%EC%84%A0" title="디지털 가입자 회선">DSL</a>의 변형이다. 한 쌍의 구리선을 통해 최대 3 킬로미터의 거리로 연결된다. <a class="mw-redirect" href="/wiki/ADSL" title="ADSL">ADSL</a>과 SDSL의 주된 차이를 보면, SDSL은 데이터 업로드 속도와 다운로드 속도가 똑같은 데 반해 (<a class="mw-redirect" href="/wiki/%EB%8C%80%EC%B9%AD%EC%84%B1" title="대칭성">대칭형</a>), ADSL은 언제나 업로드 속도가 다운로드 속도보다 느리다. (<a class="new" href="/w/index.php?title=%EB%B9%84%EB%8C%80%EC%B9%AD%EC%84%B1&amp;action=edit&amp;redlink=1" title="비대칭성 (없는 문서)">비대칭형</a>) 그러나 ADSL과는 달리, SDSL은 대역폭을 모두 점유하기 때문에 같은 선에 전통적인 음성 서비스와 공존하지 못한다.<sup class="reference" id="cite_ref-1"><a href="#cite_note-1">[1]</a></sup> SDSL은 또한 사이트 서버를 호스팅(<a class="new" href="/w/index.php?title=%ED%84%B0%EB%AF%B8%EB%84%90_%EC%84%9C%EB%B2%84&amp;action=edit&amp;redlink=1" title="터미널 서버 (없는 문서)">터미널 서버</a>, <a href="/wiki/%EA%B0%80%EC%83%81_%EC%82%AC%EC%84%A4%EB%A7%9D" title="가상 사설망">가상 사설망</a> 참조)하면서도 ADSL을 바라지 않는 중소 기업이 주된 대상이며, <a href="/wiki/%EC%A0%84%EC%9A%A9%EC%84%A0" title="전용선">전용선</a>의 높은 성능을 요구하지도 않는다.</p>
<p>SDSL는 G.991.2 (ex-G.shdsl) 권고안이 <a href="/wiki/ITU-T" title="ITU-T">ITU-T</a>가 승인할 때까지 제대로 표준화되지 않았다. SDSL은 G.SHDSL과 혼동을 일으키기도 한다. <a href="/wiki/%EC%9C%A0%EB%9F%BD" title="유럽">유럽</a>에서 G.SHDSL은 'SDSL'이라는 이름으로 <a class="new" href="/w/index.php?title=ETSI&amp;action=edit&amp;redlink=1" title="ETSI (없는 문서)">ETSI</a>에 의해 표준화되었다. ETSI의 변형은 유럽 전역에 표준화된 <a href="/wiki/%EA%B5%AD%EC%A0%9C_%EC%A0%84%EA%B8%B0_%ED%86%B5%EC%8B%A0_%EC%97%B0%ED%95%A9" title="국제 전기 통신 연합">ITU</a>-T G.SHDSL와 호환된다.</p>
