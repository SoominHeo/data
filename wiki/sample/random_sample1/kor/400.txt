<h2><span class="mw-headline" id="2009.EB.85.84_9.EC.9B.94_30.EC.9D.BC"><a href="/wiki/2009%EB%85%84" title="2009년">2009년</a> <a href="/wiki/9%EC%9B%94_30%EC%9D%BC" title="9월 30일">9월 30일</a></span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=2009%EB%85%84_9%EC%9B%94&amp;action=edit&amp;section=1" title="부분 편집: 2009년 9월 30일">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<ul>
<li><a class="mw-redirect" href="/wiki/%EB%82%A8%ED%83%9C%ED%8F%89%EC%96%91" title="남태평양">남태평양</a>의 <a href="/wiki/%EC%82%AC%EB%AA%A8%EC%95%84" title="사모아">사모아</a> 부근에서 <a href="/wiki/%EB%A6%AD%ED%84%B0_%EA%B7%9C%EB%AA%A8" title="릭터 규모">리히터 규모</a> 8.0의 <a href="/wiki/%EC%A7%80%EC%A7%84" title="지진">지진</a>과 파도 1.5m 정도의 <a href="/wiki/%EC%A7%80%EC%A7%84_%ED%95%B4%EC%9D%BC" title="지진 해일">쓰나미</a>가 발생했다고 <a href="/wiki/%EB%AF%B8%EA%B5%AD_%EC%A7%80%EC%A7%88%EC%A1%B0%EC%82%AC%EA%B5%AD" title="미국 지질조사국">미국 지질조사국</a>이 밝혔다. <a href="/wiki/%ED%83%9C%ED%8F%89%EC%96%91" title="태평양">태평양</a> 전역에 쓰나미 주의보가 발령되었으며, <a href="/wiki/%EC%95%84%EB%A9%94%EB%A6%AC%EC%B9%B8%EC%82%AC%EB%AA%A8%EC%95%84" title="아메리칸사모아">미국령 사모아</a>, 사모아, <a href="/wiki/%ED%86%B5%EA%B0%80" title="통가">통가</a> 등이 쓰나미로 인해 피해를 입었다. 118명의 사망이 확인되었으며, 총 사망자 수는 수백명이 될 것으로 추정된다.</li>
<li><a href="/wiki/%EC%95%84%EB%9E%8D%EC%97%90%EB%AF%B8%EB%A6%AC%ED%8A%B8" title="아랍에미리트">아랍에미리트</a>의 <a href="/wiki/%EC%95%84%EB%B6%80%EB%8B%A4%EB%B9%84" title="아부다비">아부다비</a>에서 <a class="mw-redirect" href="/wiki/%EA%B5%AD%EC%A0%9C_%EC%97%B0%ED%95%A9_%EA%B5%90%EC%9C%A1_%EA%B3%BC%ED%95%99_%EB%AC%B8%ED%99%94_%EA%B8%B0%EA%B5%AC" title="국제 연합 교육 과학 문화 기구">유네스코</a>가 <a class="mw-redirect" href="/wiki/%EC%84%B8%EA%B3%84%EB%AC%B4%ED%98%95%EC%9C%A0%EC%82%B0" title="세계무형유산">인류무형문화유산</a>으로 <a href="/wiki/%EB%8C%80%ED%95%9C%EB%AF%BC%EA%B5%AD" title="대한민국">대한민국</a>의 <a href="/wiki/%EA%B0%95%EA%B0%95%EC%88%A0%EB%9E%98" title="강강술래">강강술래</a>, <a href="/wiki/%EB%82%A8%EC%82%AC%EB%8B%B9%EB%86%80%EC%9D%B4" title="남사당놀이">남사당놀이</a>, <a href="/wiki/%EC%B2%98%EC%9A%A9%EB%AC%B4" title="처용무">처용무</a> 등을 등록하였다.</li>
<li><a href="/wiki/%EC%9D%B8%EB%8F%84%EB%84%A4%EC%8B%9C%EC%95%84" title="인도네시아">인도네시아</a> <a href="/wiki/%EC%88%98%EB%A7%88%ED%8A%B8%EB%9D%BC_%EC%84%AC" title="수마트라 섬">수마트라 섬</a> 부근에서 <a href="/wiki/%EB%A6%AD%ED%84%B0_%EA%B7%9C%EB%AA%A8" title="릭터 규모">리히터 규모</a> 7.9의 <a href="/wiki/%EC%A7%80%EC%A7%84" title="지진">지진</a>과 <a href="/wiki/%EC%A7%80%EC%A7%84_%ED%95%B4%EC%9D%BC" title="지진 해일">쓰나미</a>가 발생했다. 유수프 칼라 인도네시아 부통령은 "이 사태로 적어도 75명이 사망하고 주택과 호텔, 학교, 상점 등 주요 건물이 붕괴됐다"고 전했다. 최소 110명이 넘는 사람이 목숨을 잃은 것으로 집계됐고 건물 잔해에 수천명이 매몰되었다.</li>
<li><a href="/wiki/%EB%8C%80%ED%95%9C%EB%AF%BC%EA%B5%AD" title="대한민국">대한민국</a> <a class="mw-redirect" href="/wiki/%EA%B5%90%EC%9C%A1%EA%B3%BC%ED%95%99%EA%B8%B0%EC%88%A0%EB%B6%80" title="교육과학기술부">교육과학기술부</a>가 뇌연구촉진심의회를 열고 미래 고령사회에 대비하여 각종 <a class="new" href="/w/index.php?title=%EB%87%8C%EC%A7%88%ED%99%98&amp;action=edit&amp;redlink=1" title="뇌질환 (없는 문서)">뇌질환</a> 연구와 뇌 관련 연구를 담당할 <a href="/wiki/%ED%95%9C%EA%B5%AD%EB%87%8C%EC%97%B0%EA%B5%AC%EC%9B%90" title="한국뇌연구원">한국뇌연구원</a>을 설립 추진하기로 확정하였다. 뇌연구원은 <a href="/wiki/2013%EB%85%84" title="2013년">2013년</a> 첫 출범한다.</li>
</ul>
<h2><span class="mw-headline" id="2009.EB.85.84_9.EC.9B.94_29.EC.9D.BC"><a href="/wiki/2009%EB%85%84" title="2009년">2009년</a> <a href="/wiki/9%EC%9B%94_29%EC%9D%BC" title="9월 29일">9월 29일</a></span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=2009%EB%85%84_9%EC%9B%94&amp;action=edit&amp;section=2" title="부분 편집: 2009년 9월 29일">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<ul>
<li><a href="/wiki/%EB%8C%80%ED%95%9C%EB%AF%BC%EA%B5%AD%EC%9D%98_%EA%B5%AD%EB%AC%B4%EC%B4%9D%EB%A6%AC" title="대한민국의 국무총리">대한민국의 국무총리</a>로 <a href="/wiki/%EC%A0%95%EC%9A%B4%EC%B0%AC" title="정운찬">정운찬</a> 전 <a href="/wiki/%EC%84%9C%EC%9A%B8%EB%8C%80%ED%95%99%EA%B5%90" title="서울대학교">서울대학교</a> 교수가 임명 되었다.</li>
<li><a href="/wiki/%EB%8C%80%ED%95%9C%EB%AF%BC%EA%B5%AD_%EA%B5%AD%ED%86%A0%ED%95%B4%EC%96%91%EB%B6%80" title="대한민국 국토해양부">대한민국 국토해양부</a>가 <a href="/wiki/2010%EB%85%84" title="2010년">2010년</a> 도로.철도 신규사업으로 철도 3개 노선과 도시철도 1개 노선, 고속도로 2개 노선, 국도 9개 노선 등을 확정하였으며 예산안에 포함시켰다고 밝혔다.</li>
<li><a href="/wiki/%EB%8C%80%ED%95%9C%EB%AF%BC%EA%B5%AD_%EA%B2%80%EC%B0%B0%EC%B2%AD" title="대한민국 검찰청">대한민국 검찰청</a>은 <a href="/wiki/%EC%9C%A0%EA%B0%80%EC%A6%9D%EA%B6%8C" title="유가증권">유가증권</a>시장 상장업체의 주가를 두배 이상으로 부풀려(조작해) 시세차익을 챙긴 혐의로 D업체 대표이사 지모(42세)씨를 구속, 영업팀장을 추적 중이고 공범 3명을 입건해 조사중이라고 밝혔다.</li>
<li><a href="/wiki/%EB%8C%80%ED%95%9C%EB%AF%BC%EA%B5%AD" title="대한민국">대한민국</a> <a href="/wiki/%EB%8C%80%EA%B5%AC%EA%B3%A0%EB%93%B1%EB%B2%95%EC%9B%90" title="대구고등법원">대구고등법원</a>에 따르면 <a href="/wiki/%EC%8B%A0%EC%B0%BD%EC%9B%90" title="신창원">신창원</a>이 청송3교도소장과 국가를 상대로 4건의 행정,민사소송을 제기했다.</li>
</ul>
<h2><span class="mw-headline" id="2009.EB.85.84_9.EC.9B.94_28.EC.9D.BC"><a href="/wiki/2009%EB%85%84" title="2009년">2009년</a> <a href="/wiki/9%EC%9B%94_28%EC%9D%BC" title="9월 28일">9월 28일</a></span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=2009%EB%85%84_9%EC%9B%94&amp;action=edit&amp;section=3" title="부분 편집: 2009년 9월 28일">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<ul>
<li><a href="/wiki/%EB%8C%80%ED%95%9C%EB%AF%BC%EA%B5%AD_%EA%B5%AD%ED%9A%8C" title="대한민국 국회">대한민국 국회</a>에서 <a href="/wiki/%EC%A0%95%EC%9A%B4%EC%B0%AC" title="정운찬">정운찬</a> <a href="/wiki/%EC%84%9C%EC%9A%B8%EB%8C%80%ED%95%99%EA%B5%90" title="서울대학교">서울대학교</a> 교수에 대한 <a href="/wiki/%EB%8C%80%ED%95%9C%EB%AF%BC%EA%B5%AD%EC%9D%98_%EA%B5%AD%EB%AC%B4%EC%B4%9D%EB%A6%AC" title="대한민국의 국무총리">국무총리</a> 인준안이 여당인 <a class="mw-redirect" href="/wiki/%ED%95%9C%EB%82%98%EB%9D%BC%EB%8B%B9" title="한나라당">한나라당</a> 단독으로 가결되었다. 야당이 인준안의 표결에 참여하지 않은 것은 <a href="/wiki/%EB%8C%80%ED%95%9C%EB%AF%BC%EA%B5%AD_%EA%B5%AD%ED%9A%8C" title="대한민국 국회">국회</a> 역사상 처음이었다.</li>
<li><a href="/wiki/%EB%8C%80%ED%95%9C%EB%AF%BC%EA%B5%AD" title="대한민국">대한민국</a> <a href="/wiki/%EC%88%98%EC%9B%90%EC%8B%9C" title="수원시">수원시</a> 국철 <a href="/wiki/%EC%88%98%EC%9B%90%EC%97%AD" title="수원역">수원역</a> 앞 선로에서 <a class="mw-redirect" href="/wiki/%EC%9D%B4%EC%82%B0%EA%B0%80%EC%A1%B1%EC%83%81%EB%B4%89" title="이산가족상봉">이산가족상봉</a>이 좌절된 실향민 이 모(75세)씨가 진입하는 전동차에 스스로 몸을 던져 그자리에서 목숨을 끊었다.</li>
</ul>
<h2><span class="mw-headline" id="2009.EB.85.84_9.EC.9B.94_27.EC.9D.BC"><a href="/wiki/2009%EB%85%84" title="2009년">2009년</a> <a href="/wiki/9%EC%9B%94_27%EC%9D%BC" title="9월 27일">9월 27일</a></span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=2009%EB%85%84_9%EC%9B%94&amp;action=edit&amp;section=4" title="부분 편집: 2009년 9월 27일">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<ul>
<li><a href="/wiki/%EB%8C%80%ED%95%9C%EB%AF%BC%EA%B5%AD" title="대한민국">대한민국</a> <a class="mw-redirect" href="/wiki/%EC%9D%B8%EC%B2%9C_%EC%84%B8%EA%B3%84%EB%8F%84%EC%8B%9C%EC%B6%95%EC%A0%84" title="인천 세계도시축전">인천 세계도시축전</a> 행사장에서 <a href="/wiki/%EB%B9%84%ED%96%89%EA%B8%B0" title="비행기">경비행기</a>가 추락해 2층 <a href="/wiki/%EB%B2%84%EC%8A%A4" title="버스">버스</a>에 부딪치는 사고가 발생했다. 경비행기 탑승자 2명 중 1명이 숨지고, 조종사는 경상을 입었으며 2층 버스에 탑승했던 관람객 중 10여명이 가벼운 상처를 입었다.</li>
<li><a href="/wiki/%EB%8C%80%ED%95%9C%EB%AF%BC%EA%B5%AD_%EA%B2%80%EC%B0%B0%EC%B2%AD" title="대한민국 검찰청">대한민국 검찰청</a>은 <a href="/wiki/2008%EB%85%84" title="2008년">2008년</a> <a href="/wiki/3%EC%9B%94" title="3월">3월</a>부터 지금까지 정부 예산과 국가보조금, 공공기금을 횡령한 <a href="/wiki/%EA%B3%B5%EB%AC%B4%EC%9B%90" title="공무원">공무원</a> 등 150명을 구속 기소하고 546명을 불구속 기소했다고 밝혔다.</li>

</ul>

<li><a href="/wiki/%EC%A4%91%ED%99%94%EB%AF%BC%EA%B5%AD" title="중화민국">중화민국</a> 타이베이 지방법원에서 열린 1심 재판에서 뇌물 수수 등 비리 혐의로 기소된 <a href="/wiki/%EC%B2%9C%EC%88%98%EC%9D%B4%EB%B3%9C" title="천수이볜">천수이볜</a> 부부에게 종신형이 선고 되었고 5억 <a href="/wiki/%EC%8B%A0_%ED%83%80%EC%9D%B4%EC%99%84_%EB%8B%AC%EB%9F%AC" title="신 타이완 달러">달러</a>(한화 약 187억 2,500만<a href="/wiki/%EB%8C%80%ED%95%9C%EB%AF%BC%EA%B5%AD_%EC%9B%90" title="대한민국 원">원</a>)의 벌금도 부가됐다. 아들 천즈중은 자금세탁 혐의로 2년6개월 형을 선고받았다.</li>
<li><a href="/wiki/%EB%AF%B8%EA%B5%AD" title="미국">미국</a> <a href="/wiki/%EB%85%B8%EC%8A%A4%EC%BA%90%EB%A1%A4%EB%9D%BC%EC%9D%B4%EB%82%98_%EC%A3%BC" title="노스캐롤라이나 주">노스캐롤라이나 주</a>에서 <a href="/wiki/%EC%98%A4%EC%85%80%ED%83%80%EB%AF%B8%EB%B9%84%EC%96%B4" title="오셀타미비어">타미플루</a> 내성 인플루엔자바이러스 A형 감염자가 2명 발생하였다.</li>
<li><a href="/wiki/%EC%98%81%EA%B5%AD" title="영국">영국</a>의 <a href="/wiki/%EA%B3%A0%EB%93%A0_%EB%B8%8C%EB%9D%BC%EC%9A%B4" title="고든 브라운">고든 브라운</a> 총리가 수학자 <a href="/wiki/%EC%95%A8%EB%9F%B0_%ED%8A%9C%EB%A7%81" title="앨런 튜링">앨런 튜링</a>의 거세 사건에 대해 사과했다.</li>
<li><a href="/wiki/%EC%BF%A0%EB%B0%94" title="쿠바">쿠바</a>의 국가 평의회 부의장인 <a href="/wiki/%ED%9B%84%EC%95%88_%EC%95%8C%EB%A9%94%EC%9D%B4%EB%8B%A4_%EB%B3%B4%EC%8A%A4%EC%BC%80" title="후안 알메이다 보스케">후안 알메이다 보스케</a>(<a class="extiw" href="https://en.wikipedia.org/wiki/Juan_Almeida_Bosque" title="en:Juan Almeida Bosque">Juan Almeida Bosque</a>)가 심장마비로 82세의 나이로 사망하였다.</li>
</ul>

</ul>

<li><a href="/wiki/%EC%9D%BC%EB%B3%B8" title="일본">일본</a>의 <a class="mw-redirect" href="/wiki/%EB%AF%BC%EC%A3%BC%EB%8B%B9_(%EC%9D%BC%EB%B3%B8)" title="민주당 (일본)">민주당</a> 정권이 올해 추가 경정 <a href="/wiki/%EC%98%88%EC%82%B0" title="예산">예산</a> 가운데 용도가 불분명한 7조<a href="/wiki/%EC%9D%BC%EB%B3%B8_%EC%97%94" title="일본 엔">엔</a>(한화 약 93조 6,677억<a href="/wiki/%EB%8C%80%ED%95%9C%EB%AF%BC%EA%B5%AD_%EC%9B%90" title="대한민국 원">원</a>)을 <a href="/wiki/%EC%82%AC%ED%9A%8C_%EB%B3%B4%EC%9E%A5" title="사회 보장">복지</a>에 투입하기로 했다고 <a href="/wiki/%EA%B5%90%EB%8F%84_%ED%86%B5%EC%8B%A0%EC%82%AC" title="교도 통신사">교도 통신사</a>가 <a href="/wiki/%EB%89%B4%EC%8A%A4" title="뉴스">보도</a>했다.</li>
<li><a href="/wiki/%EB%B2%A0%ED%8A%B8%EB%82%A8" title="베트남">베트남</a>에 진출한 <a href="/wiki/%EC%BD%94%EC%B9%B4%EC%BD%9C%EB%9D%BC_%EC%BB%B4%ED%8D%BC%EB%8B%88" title="코카콜라 컴퍼니">코카콜라</a> 기업이 <a href="/wiki/%EC%83%9D%EC%82%B0" title="생산">생산</a>시설 확충과 <a href="/wiki/%EB%A7%88%EC%BC%80%ED%8C%85" title="마케팅">마케팅</a> 활동 강화를 위해 2억 <a href="/wiki/%EB%AF%B8%EA%B5%AD_%EB%8B%AC%EB%9F%AC" title="미국 달러">달러</a>(한화 약 2,483억<a href="/wiki/%EB%8C%80%ED%95%9C%EB%AF%BC%EA%B5%AD_%EC%9B%90" title="대한민국 원">원</a>)를 추가로 <a href="/wiki/%ED%88%AC%EC%9E%90" title="투자">투자</a>하기로 했다고 국영통신(VNA)이 <a href="/wiki/%EB%89%B4%EC%8A%A4" title="뉴스">보도</a>했다.</li>
<li><a href="/wiki/%EC%A4%91%EA%B5%AD" title="중국">중국</a> <a href="/wiki/%ED%97%A4%EC%9D%B4%EB%A3%BD%EC%9E%A5_%EC%84%B1" title="헤이룽장 성">헤이룽장 성</a> <a class="mw-redirect" href="/wiki/%EA%B3%B5%EC%95%88%EA%B5%AD" title="공안국">공안국</a>과 <a href="/wiki/%EB%A7%88%EC%95%BD" title="마약">마약</a>단속반이 <a href="/wiki/%EB%AC%B4%EB%8B%A8%EC%9E%A5_%EC%8B%9C" title="무단장 시">무단장 시</a> 일대에서 <a class="mw-redirect" href="/wiki/%EC%BA%90%EB%84%88%EB%B9%84%EC%8A%A4" title="캐너비스">대마초</a>를 <a href="/wiki/%EA%B1%B0%EB%9E%98" title="거래">거래</a>해오던 국제 대마 조직 12명을 검거하고 대마초 잎 2,300<a href="/wiki/%EA%B7%B8%EB%9E%A8" title="그램">ｇ</a>을 압수했다고 <a href="/wiki/%EC%8B%A0%ED%99%94%ED%86%B5%EC%8B%A0%EC%82%AC" title="신화통신사">신화통신사</a>가 <a href="/wiki/%EB%89%B4%EC%8A%A4" title="뉴스">보도</a>했다.</li>
<li><a href="/wiki/%EB%8C%80%ED%95%9C%EB%AF%BC%EA%B5%AD" title="대한민국">대한민국</a>의 <a href="/wiki/%EC%97%AC%EC%84%B1" title="여성">여성</a> 프로복서인 김주희(23세, 거인체육관 소속)가 파프라탄 룩사이콩(20세, 태국 국적)을 상대로 4라운드 TKO승을 거둬서 세계 3개 <a class="mw-redirect" href="/wiki/%EB%B3%B5%EC%8B%B1" title="복싱">복싱</a>기구 통합 타이틀을 획득했다.</li>
</ul>
<h2><span class="mw-headline" id="2009.EB.85.84_9.EC.9B.94_4.EC.9D.BC"><a href="/wiki/2009%EB%85%84" title="2009년">2009년</a> <a href="/wiki/9%EC%9B%94_4%EC%9D%BC" title="9월 4일">9월 4일</a></span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=2009%EB%85%84_9%EC%9B%94&amp;action=edit&amp;section=23" title="부분 편집: 2009년 9월 4일">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<ul>
<li><a href="/wiki/%EB%8C%80%ED%95%9C%EB%AF%BC%EA%B5%AD" title="대한민국">대한민국</a> <a href="/wiki/%EC%84%9C%EC%9A%B8%EC%A4%91%EC%95%99%EC%A7%80%EB%B0%A9%EB%B2%95%EC%9B%90" title="서울중앙지방법원">서울중앙지방법원</a>이 <a class="mw-redirect" href="/wiki/%EA%B5%90%EC%9C%A1%EA%B3%BC%ED%95%99%EA%B8%B0%EC%88%A0%EB%B6%80" title="교육과학기술부">교육과학기술부</a>가 수정을 요청했던 <a href="/wiki/%EA%B8%88%EC%84%B1%EC%B6%9C%ED%8C%90%EC%82%AC" title="금성출판사">금성출판사</a>의 〈한국 근·현대사〉교과서의 발행·배포를 중단하라고 1심 판결했다.</li>
</ul>
<h2><span class="mw-headline" id="2009.EB.85.84_9.EC.9B.94_3.EC.9D.BC"><a href="/wiki/2009%EB%85%84" title="2009년">2009년</a> <a href="/wiki/9%EC%9B%94_3%EC%9D%BC" title="9월 3일">9월 3일</a></span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=2009%EB%85%84_9%EC%9B%94&amp;action=edit&amp;section=24" title="부분 편집: 2009년 9월 3일">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<ul>
<li><a href="/wiki/%EB%AF%B8%EA%B5%AD" title="미국">미국</a>의 제약회사 화이자가 <a href="/wiki/%EB%B2%84%EB%9D%BD_%EC%98%A4%EB%B0%94%EB%A7%88_%ED%96%89%EC%A0%95%EB%B6%80" title="버락 오바마 행정부">행정부</a>의 <a href="/wiki/%ED%97%88%EA%B0%80" title="허가">허가</a> 내용과 다른 용도로 <a href="/wiki/%ED%8C%90%EB%A7%A4" title="판매">판매</a>하고 <a href="/wiki/%EC%9D%98%EC%82%AC" title="의사">의사</a>들에게 향응을 제공했다는 혐의와 관련해 유<a href="/wiki/%EC%A3%84" title="죄">죄</a>를 인정하고 23억 <a href="/wiki/%EB%AF%B8%EA%B5%AD_%EB%8B%AC%EB%9F%AC" title="미국 달러">달러</a>(한화 약 2조 8,646억 <a href="/wiki/%EB%8C%80%ED%95%9C%EB%AF%BC%EA%B5%AD_%EC%9B%90" title="대한민국 원">원</a>)를 내기로 연방검찰과 합의했다. 또한 정부가 의사도 처벌하는 조치를 취할 예정이다.</li>
</ul>
<h2><span class="mw-headline" id="2009.EB.85.84_9.EC.9B.94_2.EC.9D.BC"><a href="/wiki/2009%EB%85%84" title="2009년">2009년</a> <a href="/wiki/9%EC%9B%94_2%EC%9D%BC" title="9월 2일">9월 2일</a></span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=2009%EB%85%84_9%EC%9B%94&amp;action=edit&amp;section=25" title="부분 편집: 2009년 9월 2일">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<ul>
<li><a href="/wiki/%EC%9D%B8%EB%8F%84%EB%84%A4%EC%8B%9C%EC%95%84" title="인도네시아">인도네시아</a> <a href="/wiki/%EC%9E%90%EC%99%80_%EC%84%AC" title="자와 섬">자바섬</a> 인근 지하 38마일(60km) 지점에서 <a href="/wiki/%EB%A6%AD%ED%84%B0_%EA%B7%9C%EB%AA%A8" title="릭터 규모">리히터 규모</a> 7.4 강도의 <a href="/wiki/%EC%A7%80%EC%A7%84" title="지진">지진</a>이 발생했다고 <a href="/wiki/%EB%AF%B8%EA%B5%AD_%EC%A7%80%EC%A7%88%EC%A1%B0%EC%82%AC%EA%B5%AD" title="미국 지질조사국">미국 지질조사국</a>이 밝혔다.</li>
<li><a href="/wiki/%EB%8C%80%ED%95%9C%EB%AF%BC%EA%B5%AD" title="대한민국">대한민국</a>에서 <a class="mw-redirect" href="/wiki/%EC%8B%A0%EC%A2%85%ED%94%8C%EB%A3%A8" title="신종플루">신종플루</a>로 인한 4번째 사망자가 나왔다. 사망자는 고혈압과 당뇨, 신부전증 등을 앓고 있던 47세의 여성으로 앞서 신종플루 사망자와는 달리 호흡기질환 징후가 없었다.</li>
</ul>
<h2><span class="mw-headline" id="2009.EB.85.84_9.EC.9B.94_1.EC.9D.BC"><a href="/wiki/2009%EB%85%84" title="2009년">2009년</a> <a href="/wiki/9%EC%9B%94_1%EC%9D%BC" title="9월 1일">9월 1일</a></span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=2009%EB%85%84_9%EC%9B%94&amp;action=edit&amp;section=26" title="부분 편집: 2009년 9월 1일">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<ul>
<li><a href="/wiki/%EB%AF%B8%EA%B5%AD" title="미국">미국</a> <a class="mw-redirect" href="/wiki/%EC%9B%94%ED%8A%B8_%EB%94%94%EC%A6%88%EB%8B%88_%EC%BB%B4%ED%8C%A8%EB%8B%88" title="월트 디즈니 컴패니">월트 디즈니</a>가 유명 <a href="/wiki/%EB%A7%8C%ED%99%94" title="만화">만화</a>책 회사 그룹인 마블 엔터테인먼트사를 40억 <a href="/wiki/%EB%AF%B8%EA%B5%AD_%EB%8B%AC%EB%9F%AC" title="미국 달러">달러</a> 상당의 <a href="/wiki/%ED%98%84%EA%B8%88" title="현금">현금</a>과 <a href="/wiki/%EC%A3%BC%EC%8B%9D" title="주식">주식</a>을 제공하고 <a href="/wiki/%EC%9D%B8%EC%88%98_%ED%95%A9%EB%B3%91" title="인수 합병">인수</a>하기로 했다고 발표했다고 <a class="mw-redirect" href="/wiki/AP_%ED%86%B5%EC%8B%A0%EC%82%AC" title="AP 통신사">AP 통신사</a>, <a href="/wiki/%ED%94%84%EB%9E%91%EC%8A%A4_%ED%86%B5%EC%8B%A0%EC%82%AC" title="프랑스 통신사">프랑스 통신사</a>가 <a href="/wiki/%EB%89%B4%EC%8A%A4" title="뉴스">보도</a>했다.</li>
<li><a href="/wiki/%EB%8C%80%ED%95%9C%EB%AF%BC%EA%B5%AD" title="대한민국">대한민국</a>의 배우 <a href="/wiki/%EC%9E%A5%EC%A7%84%EC%98%81" title="장진영">장진영</a>이 암투병 끝에 향년 38세로 서울 <a href="/wiki/%EA%B0%80%ED%86%A8%EB%A6%AD%EB%8C%80%ED%95%99%EA%B5%90" title="가톨릭대학교">가톨릭대학교</a> 성모병원에서 생을 마감했다.</li>
</ul>
