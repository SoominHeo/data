<p><b>동대문역</b>(東大門驛)은 <a href="/wiki/%EC%84%9C%EC%9A%B8%ED%8A%B9%EB%B3%84%EC%8B%9C" title="서울특별시">서울특별시</a> <a href="/wiki/%EC%A2%85%EB%A1%9C%EA%B5%AC" title="종로구">종로구</a> 창신동과 종로6가에 있는 <a href="/wiki/%EC%84%9C%EC%9A%B8_%EC%A7%80%ED%95%98%EC%B2%A0_1%ED%98%B8%EC%84%A0" title="서울 지하철 1호선">서울 지하철 1호선</a>과 <a href="/wiki/%EC%84%9C%EC%9A%B8_%EC%A7%80%ED%95%98%EC%B2%A0_4%ED%98%B8%EC%84%A0" title="서울 지하철 4호선">서울 지하철 4호선</a>의 <a href="/wiki/%ED%99%98%EC%8A%B9%EC%97%AD" title="환승역">환승역</a>이다. 인근에 <a href="/wiki/%ED%9D%A5%EC%9D%B8%EC%A7%80%EB%AC%B8" title="흥인지문">동대문</a>이 있어 역명이 제정되었다. 과거에는 <a href="/wiki/%EB%8F%99%EB%8C%80%EB%AC%B8%EA%B5%AC" title="동대문구">동대문구</a>에 속하였으나, 수차례 행정 구역이 전환되어 현재는 <a href="/wiki/%EC%A2%85%EB%A1%9C%EA%B5%AC" title="종로구">종로구</a> 관할에 속하게 되었다.</p>
<p>1호선의 경우, 1974년 8월 15일부터 <a href="/wiki/%EB%8F%99%EB%AC%98%EC%95%9E%EC%97%AD" title="동묘앞역">동묘앞역</a>이 개업한 2005년 12월 21일까지는 이 역이 <a href="/wiki/%EA%B5%B0%EC%9E%90%EC%B0%A8%EB%9F%89%EC%82%AC%EC%97%85%EC%86%8C" title="군자차량사업소">군자차량기지</a> 입·출고 열차의 시·종착역으로 운영되었으며, 동묘앞역 개업 후에는 열차의 입·출고 기능을 전부 동묘앞역으로 이관하였다.</p>

<h2><span class="mw-headline" id=".EC.97.AD_.EA.B5.AC.EC.A1.B0">역 구조</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%8F%99%EB%8C%80%EB%AC%B8%EC%97%AD&amp;action=edit&amp;section=2" title="부분 편집: 역 구조">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>1호선은 2면 2선의 <a class="mw-redirect" href="/wiki/%EC%83%81%EB%8C%80%EC%8B%9D_%EC%8A%B9%EA%B0%95%EC%9E%A5" title="상대식 승강장">상대식 승강장</a>이고 4호선은 1면 2선의 <a class="mw-redirect" href="/wiki/%EC%84%AC%EC%8B%9D_%EC%8A%B9%EA%B0%95%EC%9E%A5" title="섬식 승강장">섬식 승강장</a>이며 모든 <a href="/wiki/%EC%8A%B9%EA%B0%95%EC%9E%A5" title="승강장">승강장</a>에 <a href="/wiki/%EC%8A%A4%ED%81%AC%EB%A6%B0%EB%8F%84%EC%96%B4" title="스크린도어">스크린도어</a>가 설치되어 있다. 출구는 10개다.</p>
