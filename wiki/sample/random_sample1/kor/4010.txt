<p><b>동부 주</b>(<span style="font-size: smaller;"><a href="/wiki/%ED%94%84%EB%9E%91%EC%8A%A4%EC%96%B4" title="프랑스어">프랑스어</a>:</span> <span lang="fr" xml:lang="fr">Est</span>)는 <a href="/wiki/%EB%B6%80%EB%A5%B4%ED%82%A4%EB%82%98%ED%8C%8C%EC%86%8C" title="부르키나파소">부르키나파소</a> 동부에 위치한 주로 주도는 <a href="/wiki/%ED%8C%8C%EB%8B%A4%EC%9D%91%EA%B5%AC%EB%A5%B4%EB%A7%88" title="파다응구르마">파다응구르마</a>이며 면적은 36,256㎢, 인구는 1,209,399명(2006년 기준)이다. 남서쪽으로는 <a href="/wiki/%EC%A4%91%EB%8F%99%EB%B6%80_%EC%A3%BC" title="중동부 주">중동부 주</a>, 북서쪽으로는 <a href="/wiki/%EC%A4%91%EB%B6%81%EB%B6%80_%EC%A3%BC_(%EB%B6%80%EB%A5%B4%ED%82%A4%EB%82%98%ED%8C%8C%EC%86%8C)" title="중북부 주 (부르키나파소)">중북부 주</a>, 북쪽으로는 <a href="/wiki/%EC%82%AC%EC%97%98_%EC%A3%BC" title="사엘 주">사엘 주</a>와 접하며 북동쪽으로는 <a href="/wiki/%EB%8B%88%EC%A0%9C%EB%A5%B4" title="니제르">니제르</a>, 남동쪽으로는 <a href="/wiki/%EB%B2%A0%EB%83%89" title="베냉">베냉</a>, 남쪽으로는 <a href="/wiki/%ED%86%A0%EA%B3%A0" title="토고">토고</a>와 국경을 접한다. 5개 현(<a href="/wiki/%EA%B7%B8%EB%82%98%EA%B7%B8%EB%82%98_%ED%98%84" title="그나그나 현">그나그나 현</a>, <a href="/wiki/%EA%B5%AC%EB%A5%B4%EB%A7%88_%ED%98%84" title="구르마 현">구르마 현</a>, <a href="/wiki/%EC%BD%94%EB%AA%BD%EC%9E%90%EB%A6%AC_%ED%98%84" title="코몽자리 현">코몽자리 현</a>, <a href="/wiki/%EC%BD%A4%ED%94%BC%EC%97%A5%EA%B0%80_%ED%98%84" title="콤피엥가 현">콤피엥가 현</a>, <a href="/wiki/%ED%83%80%ED%8F%AC%EC%95%84_%ED%98%84" title="타포아 현">타포아 현</a>)을 관할한다.</p>
