<h2><span class="mw-headline" id=".EC.97.B0.ED.98.81">연혁</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%8F%99%EB%B6%81%EA%B5%B0&amp;action=edit&amp;section=1" title="부분 편집: 연혁">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<h3><span class="mw-headline" id=".EA.B5.B0.EB.B2.8C.EC.9D.98_.EA.B5.B0.EB.8C.80.EB.A1.9C.EC.84.9C">군벌의 군대로서</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%8F%99%EB%B6%81%EA%B5%B0&amp;action=edit&amp;section=2" title="부분 편집: 군벌의 군대로서">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>동북군의 전신은 <a href="/wiki/%EC%9E%A5%EC%AD%A4%EB%A6%B0" title="장쭤린">장쭤린</a>의 봉계 군벌 군대였다. 이를 봉군(奉軍)이라고 한다. <a href="/wiki/%EC%8B%A0%ED%95%B4%ED%98%81%EB%AA%85" title="신해혁명">신해혁명</a> 이후 만주에 보안부대와 신군이 편성되었다. 1920년 <a href="/wiki/%EC%A7%81%EA%B3%84_%EA%B5%B0%EB%B2%8C" title="직계 군벌">직계 군벌</a>과 연합하여 <a href="/wiki/%ED%99%98%EA%B3%84_%EA%B5%B0%EB%B2%8C" title="환계 군벌">환계 군벌</a>을 패퇴시키고, 세력을 <a href="/wiki/%EC%82%B0%ED%95%B4%EA%B4%80" title="산해관">산해관</a> 안쪽까지 확장하였다.</p>
<p>1922년 직계 군벌과 충돌하여, <a class="new" href="/w/index.php?title=%EC%A0%9C1%EC%B0%A8_%EC%A7%81%EB%B4%89%EC%A0%84%EC%9F%81&amp;action=edit&amp;redlink=1" title="제1차 직봉전쟁 (없는 문서)">제1차 직봉전쟁</a>이 터졌다. 여기서 패해서 산해관 밖으로 물러났다. 1924년 <a class="new" href="/w/index.php?title=%EC%A0%9C2%EC%B0%A8_%EC%A7%81%EB%B4%89%EC%A0%84%EC%9F%81&amp;action=edit&amp;redlink=1" title="제2차 직봉전쟁 (없는 문서)">제2차 직봉전쟁</a>이 발발하였는데, 직계의 내부의 <a href="/wiki/%ED%8E%91%EC%9C%84%EC%83%B9" title="펑위샹">펑위샹</a>이 베이징에서 쿠데타를 일으켜서 직계군벌 <a href="/wiki/%EC%B0%A8%EC%98%A4%EC%BF%A4" title="차오쿤">차오쿤</a>과 <a href="/wiki/%EC%9A%B0%ED%8E%98%EC%9D%B4%ED%91%B8" title="우페이푸">우페이푸</a>가 패해 세력을 잃었고, 이때 봉군은 세력을 다시 회복하여 대부분의 관내를 장악하고 멀리는 상하이까지 세력을 뻗었다.</p>
<p>1925년 우페이푸가 <a class="new" href="/w/index.php?title=%EB%B0%98%EB%B4%89_%EC%A0%84%EC%9F%81&amp;action=edit&amp;redlink=1" title="반봉 전쟁 (없는 문서)">반봉 전쟁</a>을 일으키자, 펑위샹은 봉군의 <a class="new" href="/w/index.php?title=%EA%B6%88%EC%88%AD%EB%A7%81&amp;action=edit&amp;redlink=1" title="궈숭링 (없는 문서)">궈숭링</a>이 모반하도록 하였으나, 봉군은 펑을 패퇴시키고 1926년 봄 <a class="mw-redirect" href="/wiki/%EB%B6%81%EA%B2%BD" title="북경">북경</a>을 점령하였다. 이후 봉군은 직계 군벌과 협상하여 정전했다.</p>
<p>1927년 장쭤린은 스스로 대총통 겸 중화민국 육해공 대원수에 취임하였다. 1928년 봉군은 국민당 소속의 <a href="/wiki/%EC%9E%A5%EC%A0%9C%EC%8A%A4" title="장제스">장제스</a>, <a href="/wiki/%EB%A6%AC%EC%AD%9D%EB%9F%B0" title="리쭝런">리쭝런</a>의 <a href="/wiki/%EA%B5%AD%EB%AF%BC%ED%98%81%EB%AA%85%EA%B5%B0" title="국민혁명군">국민혁명군</a>과 펑위샹, <a href="/wiki/%EC%98%8C%EC%8B%9C%EC%82%B0" title="옌시산">옌시산</a>의 연합군에 패하였다. 장쭤린은 본거지인 만주로 후퇴하다가 일본군의 계략으로 철도가 폭파되어 암살되었고, 그 아들 <a href="/wiki/%EC%9E%A5%EC%89%90%EB%9F%89" title="장쉐량">장쉐량</a>이 봉군의 사령관이 되었다. 장쉐량은 일본이 자신의 아버지를 암살했다는 것을 알게되었고, 일본의 의도와는 달리 국민당과 장제스에게 충성을 맹세하였다.</p>
<h3><span class="mw-headline" id=".EC.A4.91.ED.99.94.EB.AF.BC.EA.B5.AD.EC.9D.98_.EC.A7.80.EB.B0.A9.EA.B5.B0.EC.9C.BC.EB.A1.9C.EC.84.9C">중화민국의 지방군으로서</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%8F%99%EB%B6%81%EA%B5%B0&amp;action=edit&amp;section=3" title="부분 편집: 중화민국의 지방군으로서">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p><a href="/wiki/1929%EB%85%84" title="1929년">1929년</a> 봉군은 형식상 국민혁명군에 편입되어 동북변방군(東北邊防軍)이 되었으며, 약칭 동북군이 되었다. 총 병력은 약 30만명이었고, <a href="/wiki/%EC%9E%A5%EC%89%90%EB%9F%89" title="장쉐량">장쉐량</a>이 총괄하였다. 1930년 각 군벌과 장제스 간에 <a class="new" href="/w/index.php?title=%EC%A4%91%EC%9B%90%EB%8C%80%EC%A0%84&amp;action=edit&amp;redlink=1" title="중원대전 (없는 문서)">중원대전</a>이 벌어졌는데, 동북군은 장제스를 지지했고, 다시 산해관 안으로 출동하여 베이징과 <a class="mw-redirect" href="/wiki/%ED%86%88%EC%A7%84" title="톈진">톈진</a>을 장악하였다.</p>
<p>1930년 만주 철도의 이권을 둘러싸고 <a href="/wiki/%EC%86%8C%EB%A0%A8" title="소련">소련</a>과 장쉐량과 분쟁이 벌어졌다. 동북군은 <a href="/wiki/%EB%B0%94%EC%8B%A4%EB%A6%AC_%EB%B8%94%EB%A5%98%ED%97%A4%EB%A5%B4" title="바실리 블류헤르">바실리 블류헤르</a> 원수가 지위한 소련군에게 참패하였고, 만주 주요 요지가 점령당했다. (<a class="new" href="/w/index.php?title=%EB%B4%89%EC%86%8C%EB%B6%84%EC%9F%81&amp;action=edit&amp;redlink=1" title="봉소분쟁 (없는 문서)">봉소분쟁</a>)</p>
<p>1931년 <a href="/wiki/%EB%A7%8C%EC%A3%BC%EC%82%AC%EB%B3%80" title="만주사변">만주사변</a>이 발발하자, 장쉐량은 <a class="mw-redirect" href="/wiki/%EC%A4%91%ED%99%94%EB%AF%BC%EA%B5%AD_%EC%B4%9D%ED%86%B5" title="중화민국 총통">중화민국 총통</a> <a href="/wiki/%EC%9E%A5%EC%A0%9C%EC%8A%A4" title="장제스">장제스</a>의 명령에 따라 강력한 일본군에 저항하지 않고 산해관 안으로 철퇴하였다.</p>
<p>다만 <a href="/wiki/%ED%97%A4%EC%9D%B4%EB%A3%BD%EC%9E%A5_%EC%84%B1" title="헤이룽장 성">헤이룽장 성</a>의 주석을 맡고 있던 <a class="new" href="/w/index.php?title=%EB%A7%88%EC%9E%94%EC%82%B0&amp;action=edit&amp;redlink=1" title="마잔산 (없는 문서)">마잔산</a>과 소수의 부대들이 장제스의 무저항정책을 거부하고 자발적으로 일본군과 싸웠다.</p>
<p>1933년 휘하의 숭저위안이 <a class="new" href="/w/index.php?title=%EB%A7%8C%EB%A6%AC%EC%9E%A5%EC%84%B1_%EB%B0%A9%EC%96%B4%EC%A0%84&amp;action=edit&amp;redlink=1" title="만리장성 방어전 (없는 문서)">만리장성 방어전</a>에서 일본군과 교전하였고 <a href="/wiki/1934%EB%85%84" title="1934년">1934년</a> 3월 장쉐량은 악예완(악(鄂)은 <a href="/wiki/%ED%9B%84%EB%B2%A0%EC%9D%B4_%EC%84%B1" title="후베이 성">후베이 성</a>, 예(豫)는 <a href="/wiki/%ED%97%88%EB%82%9C_%EC%84%B1" title="허난 성">허난 성</a>, 환(皖)은 <a class="mw-redirect" href="/wiki/%EC%95%88%ED%9C%98_%EC%84%B1" title="안휘 성">안휘 성</a>을 가리킨다) 공비(<a href="/wiki/%ED%99%8D%EA%B5%B0" title="홍군">홍군</a>) 토벌 군의 부사령관이 임명되었고, 자신의 잔류 병력 병력 16만을 이끌고 <a href="/wiki/%EA%B0%84%EC%91%A4_%EC%84%B1" title="간쑤 성">간쑤 성</a>으로 이동하였다.</p>
<p><a href="/wiki/1935%EB%85%84" title="1935년">1935년</a> 장제스의 명령을 받아 간쑤 성 일대에서 <a href="/wiki/%ED%99%8D%EA%B5%B0" title="홍군">홍군</a>을 공격하고, 3번의 전투를 치렀으나 참패하여 3개 사단을 잃었다.</p>
<p>장제스는 우선 홍군을 토벌하는 것이 더 중요하다고 생각했기 때문에, 일본의 침략은 일단 제쳐두고 있었다. 장쉐량과 동북군의 장병들은 홍군에는 전력을 투구하면서 일본에는 무대응으로 일관하는 <a href="/wiki/%EC%9E%A5%EC%A0%9C%EC%8A%A4" title="장제스">장제스</a>의 정책에 반발하였다. 장쉐량은 은밀히 <a class="mw-redirect" href="/wiki/%EC%83%81%ED%95%98%EC%9D%B4" title="상하이">상하이</a>로 가서 <a class="new" href="/w/index.php?title=%EB%91%90%EC%A4%91%EC%9C%84%EC%95%88&amp;action=edit&amp;redlink=1" title="두중위안 (없는 문서)">두중위안</a>과, <a class="new" href="/w/index.php?title=%EC%9D%B4%EC%85%94%EB%8D%A9&amp;action=edit&amp;redlink=1" title="이셔덩 (없는 문서)">이셔덩</a>등을 만났다.</p>
<p>이때 중국 공산당의 비밀당원인 류딩이 장학량에게 와서 동북군과 홍군간의 화평에 대해 논의를 하였다. 1936년 1월 25일 중국공산당은 <a href="/wiki/%EB%A7%88%EC%98%A4%EC%A9%8C%EB%91%A5" title="마오쩌둥">마오쩌둥</a>, <a href="/wiki/%EC%A0%80%EC%9A%B0%EC%96%B8%EB%9D%BC%EC%9D%B4" title="저우언라이">저우언라이</a>, <a href="/wiki/%ED%8E%91%EB%8D%94%ED%99%94%EC%9D%B4" title="펑더화이">펑더화이</a>등의 이름으로 "홍군과 동북군의 연합항일을 위해 동북군 전체장병에게 보내는 성명 《红军为愿意同东北军联合抗日致东北军全体将士书》"을 발표하였다.</p>
<p><a href="/wiki/1936%EB%85%84" title="1936년">1936년</a> <a href="/wiki/12%EC%9B%94_12%EC%9D%BC" title="12월 12일">12월 12일</a> 장쉐량과 <a class="new" href="/w/index.php?title=%EC%84%9C%EB%B6%81%EA%B5%B0&amp;action=edit&amp;redlink=1" title="서북군 (없는 문서)">서북군</a> 사령관 <a href="/wiki/%EC%96%91%ED%9B%84%EC%B2%AD" title="양후청">양후청</a>은 중국공산당의 항일민족통일전선과 정책과 항일을 요구하는 전국의 여론에 영향받아, <a class="mw-redirect" href="/wiki/%EC%8B%9C%EC%95%88%EC%82%AC%EB%B3%80" title="시안사변">시안사변</a>을 일으켰다. 사변후, 제2차 국공합작이 실행되어 항일전선은 실행되었으나, 장쉐량은 장제스에게 억류되어 <a class="mw-redirect" href="/wiki/%EB%82%9C%EC%A7%95" title="난징">난징</a>에 머물렀다.</p>
<h3><span class="mw-headline" id=".EC.A4.91.EC.9D.BC.EC.A0.84.EC.9F.81"><a class="mw-redirect" href="/wiki/%EC%A4%91%EC%9D%BC%EC%A0%84%EC%9F%81" title="중일전쟁">중일전쟁</a></span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%8F%99%EB%B6%81%EA%B5%B0&amp;action=edit&amp;section=4" title="부분 편집: 중일전쟁">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p><a href="/wiki/1937%EB%85%84" title="1937년">1937년</a> 3월 장제스는 동북군을 허난과 안휘로 이동시켜 이를 해체 개편하였는데 <a href="/wiki/1945%EB%85%84" title="1945년">1945년</a>까지 동북군 49군, 57군, 67군은 송후 전투와 <a href="/wiki/%EB%82%9C%EC%A7%95_%EC%A0%84%ED%88%AC" title="난징 전투">난징 전투</a>에 참전하였다. 동북군 51군은 <a href="/wiki/%ED%99%94%EC%9D%B4%ED%97%88" title="화이허">화이허</a>와 <a class="new" href="/w/index.php?title=%EC%89%AC%EC%A0%80%EC%9A%B0_%EC%A0%84%ED%88%AC&amp;action=edit&amp;redlink=1" title="쉬저우 전투 (없는 문서)">쉬저우 전투</a>에 참전하였고, 53군은 후베이, 허난, 안휘, <a href="/wiki/%ED%9B%84%EB%82%9C_%EC%84%B1" title="후난 성">후난 성</a>을 돌아가면서 싸웠다. 53군의 일부는 <a href="/wiki/%ED%8C%94%EB%A1%9C%EA%B5%B0" title="팔로군">팔로군</a>에 편입되었다.</p>
