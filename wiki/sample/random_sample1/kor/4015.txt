<p><b>동성왕</b>(東城王, <a href="/wiki/445%EB%85%84" title="445년">445년</a>~<a href="/wiki/501%EB%85%84" title="501년">501년</a>, 재위 : <a href="/wiki/479%EB%85%84" title="479년">479년</a>~<a href="/wiki/501%EB%85%84" title="501년">501년</a>)은 <a href="/wiki/%EB%B0%B1%EC%A0%9C" title="백제">백제</a>의 제24대 <a href="/wiki/%EA%B5%B0%EC%A3%BC" title="군주">국왕</a>이다. 성은 부여(扶餘), 이름은 모대(牟大)이다. 또 다른 이름은 마모(摩牟)·마제(麻帝)·말다(末多)·여대(餘大)여대(餘大)등이 있다. 제22대 왕인 <a href="/wiki/%EB%AC%B8%EC%A3%BC%EC%99%95" title="문주왕">문주왕</a>(文周王)의 아우인 좌평(佐平) <a href="/wiki/%EA%B3%A4%EC%A7%80" title="곤지">곤지</a>(昆支)의 둘째 아들로서 담력이 있고 활을 쏘는 솜씨가 뛰어났다. <a href="/wiki/%EC%82%BC%EA%B7%BC%EC%99%95" title="삼근왕">삼근왕</a> 때 일어난 해구(解仇)의 반란을 평정한 뒤 정권을 잡은 진씨(眞氏) 세력에 의해 옹립되어 삼근왕의 뒤를 이었다.</p>
<p><a href="/wiki/479%EB%85%84" title="479년">479년</a>부터 <a href="/wiki/501%EB%85%84" title="501년">501년</a>까지 재위하는 동안 <a href="/wiki/479%EB%85%84" title="479년">479년</a>부터 <a href="/wiki/482%EB%85%84" title="482년">482년</a>까지 <a href="/wiki/%EC%A7%84%EB%A1%9C_(%EB%B0%B1%EC%A0%9C)" title="진로 (백제)">진로</a>(眞老)가 <a href="/wiki/%EA%B6%8C%EB%A0%A5" title="권력">실권</a>을 맡았고 <a href="/wiki/482%EB%85%84" title="482년">482년</a>부터 <a href="/wiki/501%EB%85%84" title="501년">501년</a> 붕어할 때까지 친정하였다.</p>

<h2><span class="mw-headline" id=".EC.83.9D.EC.95.A0">생애</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%8F%99%EC%84%B1%EC%99%95&amp;action=edit&amp;section=1" title="부분 편집: 생애">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>동성왕의 본명은 모대 또는 여대, 마모이고, <a href="/wiki/%EC%9D%BC%EB%B3%B8" title="일본">일본</a>의 <a href="/wiki/%EC%9D%BC%EB%B3%B8%EC%84%9C%EA%B8%B0" title="일본서기">일본서기</a>와 <a href="/wiki/%EC%86%8D%EC%9D%BC%EB%B3%B8%EA%B8%B0" title="속일본기">속일본기</a>에는 말다로 기록되어 있다. <a href="/wiki/%EC%82%BC%EA%B5%AD%EC%82%AC%EA%B8%B0" title="삼국사기">삼국사기</a>와 <a href="/wiki/%EC%82%BC%EA%B5%AD%EC%9C%A0%EC%82%AC" title="삼국유사">삼국유사</a>에는 동성왕은 <a href="/wiki/%EC%A0%84%EC%A7%80%EC%99%95" title="전지왕">전지왕</a>의 손자로, <a href="/wiki/%EA%B0%9C%EB%A1%9C%EC%99%95" title="개로왕">개로왕</a>의 동생 <a href="/wiki/%EA%B3%A4%EC%A7%80" title="곤지">곤지</a>의 아들이다. <a href="/wiki/%EC%9D%BC%EB%B3%B8" title="일본">일본</a>의<a href="/wiki/%EC%9D%BC%EB%B3%B8%EC%84%9C%EA%B8%B0" title="일본서기">일본서기</a>에도 동성왕은 곤지의 아들이라 한다. 모후의 이름은 전하지 않는다.</p>
<p><a href="/wiki/%EC%9D%BC%EB%B3%B8" title="일본">일본</a>에서 체류하다 사촌 <a href="/wiki/%EC%82%BC%EA%B7%BC%EC%99%95" title="삼근왕">삼근왕</a>이 살해되자 아버지 <a href="/wiki/%EA%B3%A4%EC%A7%80" title="곤지">곤지</a>에 의해 <a href="/wiki/%EB%B0%B1%EC%A0%9C" title="백제">백제</a>로 보내져 즉위하였다. 동성왕은 《<a href="/wiki/%EC%82%BC%EA%B5%AD%EC%82%AC%EA%B8%B0" title="삼국사기">삼국사기</a>》의 기록에 따르면, <a href="/wiki/481%EB%85%84" title="481년">481년</a>(동성왕 3년)에 <a href="/wiki/%EC%8B%A0%EB%9D%BC" title="신라">신라</a>의 북쪽 변경을 침공한 <a href="/wiki/%EA%B3%A0%EA%B5%AC%EB%A0%A4" title="고구려">고구려</a>와 <a href="/wiki/%EB%A7%90%EA%B0%88" title="말갈">말갈</a>의 연합군을, <a href="/wiki/%EC%8B%A0%EB%9D%BC" title="신라">신라</a> 및 <a href="/wiki/%EA%B0%80%EC%95%BC" title="가야">가야</a>와 연합하여 격퇴했으며, <a href="/wiki/484%EB%85%84" title="484년">484년</a>(동성왕 6년)에는 중국 <a class="mw-disambig" href="/wiki/%EB%82%A8%EC%A1%B0" title="남조">남조</a>(南朝)의 남제에도 사신을 보내고자 시도하고, 이듬해인 <a href="/wiki/485%EB%85%84" title="485년">485년</a>(동성왕 7년)에는 <a href="/wiki/%EC%8B%A0%EB%9D%BC" title="신라">신라</a>에도 사신을 보내는 등 외교에도 힘을 쏟았다.</p>
<p><a href="/wiki/488%EB%85%84" title="488년">488년</a>(동성왕 10년) 백제를 침공한 <a href="/wiki/%EB%B6%81%EC%9C%84_(%EB%B6%81%EC%A1%B0)" title="북위 (북조)">북위</a>의 기병을 사법명 등을 시켜 격퇴했으며, <a href="/wiki/493%EB%85%84" title="493년">493년</a>(동성왕 15년) 신라의 왕녀를 맞이하여 왕비로 삼고, <a href="/wiki/494%EB%85%84" title="494년">494년</a>(동성왕 16년)과 <a href="/wiki/495%EB%85%84" title="495년">495년</a> 두 해에 걸쳐 백제와 신라를 번갈아 침공해오는 고구려의 군대를 신라와 연합해 격퇴하는 등, 신라와의 동맹을 더욱 돈독히 다졌다. <a href="/wiki/498%EB%85%84" title="498년">498년</a>에 공물과 세금을 바치지 않는 <a class="mw-redirect" href="/wiki/%ED%83%90%EB%9D%BC%EA%B5%AD" title="탐라국">탐라국</a>(지금의 <a href="/wiki/%EC%A0%9C%EC%A3%BC%EB%8F%84" title="제주도">제주도</a>)을 친히 정벌하고자 무진주(武珍州: 지금의 <a href="/wiki/%EA%B4%91%EC%A3%BC%EA%B4%91%EC%97%AD%EC%8B%9C" title="광주광역시">광주</a>)에 이르렀다가, <a class="mw-redirect" href="/wiki/%ED%83%90%EB%9D%BC%EA%B5%AD" title="탐라국">탐라국</a>의 항복을 받고 그만두면서 탐라국을 복속시켰다.<sup class="reference" id="cite_ref-1"><a href="#cite_note-1">[1]</a></sup></p>
<p>웅진 천도 이후 계속된 혼란을 수습하는데 많은 업적을 남겼지만, 말기에 이르러서는 점차 향락에 빠져 정치를 돌보지 않으며 놀기만 했다. <a href="/wiki/499%EB%85%84" title="499년">499년</a>(동성왕 21년) 가뭄이 들어 백성들이 굶주려 죽어나가자, 궁궐의 창고를 열어서 백성들을 구제하자는 신하들의 권고도 받아들이지 않고, <a href="/wiki/500%EB%85%84" title="500년">500년</a>(동성왕 22년) 봄에 웅진성 동쪽에 임류각(臨流閣)을 짓고 사치스러운 정원을 만들었다. 또한 신하들이 간언하는 것을 귀찮아하며 궁궐의 문까지 닫아버릴 정도로 사치와 향락을 일삼았다.</p>
<p><a href="/wiki/%EC%82%BC%EA%B5%AD%EC%82%AC%EA%B8%B0" title="삼국사기">삼국사기</a>와 <a href="/wiki/%EC%82%BC%EA%B5%AD%EC%9C%A0%EC%82%AC" title="삼국유사">삼국유사</a>에 의하면 <a href="/wiki/501%EB%85%84" title="501년">501년</a>(동성왕 23년) 겨울 음력 11월, 사냥을 나갔다가 폭설을 만나 근처에서 머무르던 중, 왕의 정책에 반발을 한 <a href="/wiki/%EC%A2%8C%ED%8F%89" title="좌평">위사좌평</a>(衛士佐平) <a href="/wiki/%EB%B0%B1%EA%B0%80_(%EA%B7%80%EC%A1%B1)" title="백가 (귀족)">백가</a>(苩加)가 쿠데타를 일으켜 병사들에게 처형당했다.</p>
<p><a href="/wiki/%EC%9D%BC%EB%B3%B8%EC%84%9C%EA%B8%B0" title="일본서기">일본서기</a>는 백제신찬을 인용하여 "백제신찬에 이르기를 말다왕이 무도하여 폭정을 펼치니, 백성들을 학대하였다. <a href="/wiki/%EB%B0%B1%EC%A0%9C" title="백제">백제</a>의 국인이 함께 그를 제거하였다. 무령왕을 옹립하니 휘는 사마왕인데, 곤지왕자의 아들이다. 즉, 말다왕(동성왕)은 이모형(이복형)이다.(百濟新撰云; 末多王無道暴虐百姓 國人共除 武寧立 諱斯麻王 是 昆支王子之子, 則末多王異母兄也"라고 기록되었다. 그가 죽은 뒤에 동성왕(東城王)이라는 <a href="/wiki/%EC%8B%9C%ED%98%B8" title="시호">시호</a>(諡號)를 받았다.</p>
<h2><span class="mw-headline" id=".EB.AC.B4.EB.A0.B9.EC.99.95.EA.B3.BC.EC.9D.98_.EA.B4.80.EA.B3.84">무령왕과의 관계</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%8F%99%EC%84%B1%EC%99%95&amp;action=edit&amp;section=2" title="부분 편집: 무령왕과의 관계">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p><a href="/wiki/%EC%82%BC%EA%B5%AD%EC%82%AC%EA%B8%B0" title="삼국사기">삼국사기</a>와 <a href="/wiki/%EC%82%BC%EA%B5%AD%EC%9C%A0%EC%82%AC" title="삼국유사">삼국유사</a>의 기록에 의하면 <a class="mw-redirect" href="/wiki/%EB%B0%B1%EC%A0%9C_%EB%AC%B4%EB%A0%B9%EC%99%95" title="백제 무령왕">무령왕</a>은 동성왕의 둘째 아들이라고 한다. 그러나 <a href="/wiki/%EC%9D%BC%EB%B3%B8" title="일본">일본</a>측의 저서인 <a href="/wiki/%EC%9D%BC%EB%B3%B8%EC%84%9C%EA%B8%B0" title="일본서기">일본서기</a>에 의하면 <a class="mw-redirect" href="/wiki/%EB%B0%B1%EC%A0%9C_%EB%AC%B4%EB%A0%B9%EC%99%95" title="백제 무령왕">무령왕</a>은 <a href="/wiki/%EA%B3%A4%EC%A7%80" title="곤지">곤지</a>의 둘째 아들이고 동성왕의 동생이라고 한다. 한편 그의 딸 보과부인(寶菓夫人)은 <a class="mw-redirect" href="/wiki/%EC%8B%A0%EB%9D%BC_%EB%B2%95%ED%9D%A5%EC%99%95" title="신라 법흥왕">신라 법흥왕</a>의 후궁이 되었고 남모의 외조부가 된다.</p>
<p><a href="/wiki/%EC%9D%BC%EB%B3%B8%EC%84%9C%EA%B8%B0" title="일본서기">일본서기</a>에 의하면 동성왕은 <a href="/wiki/%EB%AC%B4%EB%A0%B9%EC%99%95" title="무령왕">무령왕</a>의 이복 형이다. 이 설은 신뢰받지 못하다가, <a href="/wiki/1971%EB%85%84" title="1971년">1971년</a> 공주 송산리 왕릉에서 발견된 지석(誌石)에 의해 <a href="/wiki/%EB%AC%B4%EB%A0%B9%EC%99%95" title="무령왕">무령왕</a>이 <a href="/wiki/462%EB%85%84" title="462년">462년</a>에 태어났음이 밝혀지면서 점차 신뢰를 얻고 있다. <a href="/wiki/%EB%AC%B4%EB%A0%B9%EC%99%95" title="무령왕">무령왕</a>은 <a href="/wiki/462%EB%85%84" title="462년">462년</a> 출생하였고 <a href="/wiki/523%EB%85%84" title="523년">523년</a> 사망함이 묘지명에 기록되었다. 동성왕은 <a href="/wiki/445%EB%85%84" title="445년">445년</a>에 출생했고 <a href="/wiki/501%EB%85%84" title="501년">501년</a>에 백가에게 처형되었다. 동성왕이 무령왕보다 17세 많은데, 동성왕이 17세에 둘째 아들 <a href="/wiki/%EB%AC%B4%EB%A0%B9%EC%99%95" title="무령왕">무령왕</a>을 낳기는 생리적으로 힘들다.</p>
