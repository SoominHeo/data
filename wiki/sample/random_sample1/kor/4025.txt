<p><b>동원령</b>(mobilization)은 국가 체제를 전시 동원 체제로 변환하고, 병력과 보급 물자 생산에 국가 총력을 기울이는 것을 말한다. <a class="mw-redirect" href="/wiki/%ED%94%84%EB%9E%91%EC%8A%A4_%EB%8C%80%ED%98%81%EB%AA%85" title="프랑스 대혁명">프랑스 대혁명</a> 당시 처음 공표된 적은 있었지만, 이 단어가 군사적 용어로 정착되기 시작한 것은 1850년대와 1860년대 러시아 군이 <a href="/wiki/%ED%81%AC%EB%A6%BC_%EC%A0%84%EC%9F%81" title="크림 전쟁">크림 전쟁</a> 등으로 군사 충원을 위해서였던 것으로 알려져 있다. 동원 이론과 기술은 시대에 따라 조금씩 그 의미를 달리했으며 전쟁의 성격 자체가 동원령 실시 이전과 이후가 크게 바뀔만큼 큰 영향을 끼쳤다. 반댓말은 동원 해제(demobilization)이다.</p>
<p>동원령은 많은 수의 징병을 위한 제도적 장치로서, 예비군을 모두 정규군화하고 국내 생산을 모두 군수 물자 생산으로 돌리게 되므로 단기간에 격심한 기술적 사회적 변화가 수반된다. <a href="/wiki/%EC%B4%9D%EB%A0%A5%EC%A0%84" title="총력전">총력전</a> 양상을 띄게 된 전쟁에서 동원령은 필수불가결한 것이 되었다. 점차 <a href="/wiki/%EC%A0%84%ED%99%94" title="전화">전화</a> <a href="/wiki/%EC%A0%84%EC%8B%A0" title="전신">전신</a>의 발달로 국가 훈령이 단시간 내에 넓은 지역에 확산될 수 있었던 것과 더불어 <a href="/wiki/19%EC%84%B8%EA%B8%B0" title="19세기">19세기</a> <a href="/wiki/%EC%B2%A0%EB%8F%84" title="철도">철도</a>의 보급으로 대규모 병력이 빠른 시간 내에 전개될 수 있게 된 점, 그리고 징집 제도로 사전에 훈련시켜 놨던 예비 병력을 전시 상황에 바로바로 준비시켜 전쟁에 쓸 수 있게 됐다는 점이 동원령을 가능케 했다.</p>

적색 점선-독일군 <a href="/wiki/%EB%B2%A8%EA%B8%B0%EC%97%90" title="벨기에">벨기에</a> 우회 진격로</div>

<p><a href="/wiki/1914%EB%85%84" title="1914년">1914년</a> <a href="/wiki/%EB%8C%80%EC%98%81_%EC%A0%9C%EA%B5%AD" title="대영 제국">대영 제국</a>은 <a href="/wiki/%EC%9C%A0%EB%9F%BD" title="유럽">유럽</a> <a class="mw-redirect" href="/wiki/%EC%97%B4%EA%B0%95" title="열강">열강</a> 중 유일하게 징병제를 실시하지 않은 국가였다. <a href="/wiki/%EC%98%A4%EC%8A%A4%ED%8A%B8%EB%A6%AC%EC%95%84-%ED%97%9D%EA%B0%80%EB%A6%AC_%EC%A0%9C%EA%B5%AD" title="오스트리아-헝가리 제국">오스트리아-헝가리 제국</a>, <a href="/wiki/%EC%9D%B4%ED%83%88%EB%A6%AC%EC%95%84_%EC%99%95%EA%B5%AD_(1861%EB%85%84_~_1946%EB%85%84)" title="이탈리아 왕국 (1861년 ~ 1946년)">이탈리아 왕국</a>, <a href="/wiki/%ED%94%84%EB%9E%91%EC%8A%A4_%EC%A0%9C3%EA%B3%B5%ED%99%94%EA%B5%AD" title="프랑스 제3공화국">프랑스 제3공화국</a><sup class="reference" id="cite_ref-1"><a href="#cite_note-1">[1]</a></sup>, <a class="mw-redirect" href="/wiki/%EB%8F%85%EC%9D%BC_%EC%A0%9C2%EC%A0%9C%EA%B5%AD" title="독일 제2제국">독일 제2제국</a>과 <a href="/wiki/%EB%9F%AC%EC%8B%9C%EC%95%84_%EC%A0%9C%EA%B5%AD" title="러시아 제국">러시아 제국</a> 등은 의무 징집으로 수백만의 군대를 유지하며 엄청난 비용을 쓰고 있었다. 각 열강들은 평시에는 전쟁 동원 가능 자원 중 일부만 상비군으로 두고 나머지는 예비군으로 돌려 최소한의 비용으로 주기적인 기초군사훈련만 시켜 언제든 동원할 수 있게 해두었다. 동원령은 평시에 비용을 아낄 수 있게 해주지만 반대로 수백만에 달하는 동원 자원들을 컨트롤하려면 상당히 정교한 법적 행정적 준비가 뒷받침되어야 실제 동원 시 혼란 및 실패를 막을 수 있다. 그래서 대부분 열강들은 전시 비상 계획 등의 매뉴얼을 만들어놓고 차질이 없도록 미리 준비해 두었다<sup class="reference" id="cite_ref-Keegan_2-0"><a href="#cite_note-Keegan-2">[2]</a></sup><sup class="reference" id="cite_ref-3"><a href="#cite_note-3">[3]</a></sup>.</p>
<p>국제 관계에 있어서 일방의 총동원 체제는 상대방에 대한 전쟁의사가 있는 것으로 간주되므로 중요한 개전 이유<sup class="reference" id="cite_ref-4"><a href="#cite_note-4">[4]</a></sup>가 된다. <a href="/wiki/1914%EB%85%84" title="1914년">1914년</a> 시작된 <a class="mw-redirect" href="/wiki/1%EC%B0%A8_%EC%84%B8%EA%B3%84_%EB%8C%80%EC%A0%84" title="1차 세계 대전">1차 세계 대전</a>의 주요한 원인 중 하나가 각국의 총동원령이기도 했다. <a class="mw-redirect" href="/wiki/1%EC%B0%A8_%EB%8C%80%EC%A0%84" title="1차 대전">1차 대전</a> 때 독일 사령부는 프랑스 침공시 러시아를 자극하지 않으려고 동원령을 발동하지 않았다. 물론 <a href="/wiki/%EB%B9%8C%ED%97%AC%EB%A6%84_2%EC%84%B8" title="빌헬름 2세">빌헬름 2세</a>가 승인한 <a href="/wiki/%EC%8A%90%EB%A6%AC%ED%8E%9C_%EA%B3%84%ED%9A%8D" title="슐리펜 계획">슐리펜 계획</a>에는 프·러 양국이 협공할 경우에 대비해 동원령을 발동하는 것도 물론 들어있었지만, 러시아가 도발을 하든말든 실제 침공이 아닌 이상 <a href="/wiki/%EC%8A%90%EB%A6%AC%ED%8E%9C_%EA%B3%84%ED%9A%8D" title="슐리펜 계획">슐리펜 계획</a> 상으로는 한 번에 하나의 적과 싸우는 것으로 되어있었으므로 동원령은 자제되었다.</p>
<p><a href="/wiki/%EC%8A%90%EB%A6%AC%ED%8E%9C_%EA%B3%84%ED%9A%8D" title="슐리펜 계획">슐리펜 계획</a>에 대해 <a href="/wiki/%EB%9F%AC%EC%8B%9C%EC%95%84_%EC%A0%9C%EA%B5%AD" title="러시아 제국">러시아 제국</a>의 <a class="new" href="/w/index.php?title=%EC%8A%A4%EB%94%B0%EB%B8%8C%EC%B9%B4_%EA%B3%84%ED%9A%8D&amp;action=edit&amp;redlink=1" title="스따브카 계획 (없는 문서)">스따브카 계획</a>(Stavka plan)은 <a href="/wiki/%EC%98%A4%EC%8A%A4%ED%8A%B8%EB%A6%AC%EC%95%84_%EC%A0%9C%EA%B5%AD" title="오스트리아 제국">오스트리아 제국</a>이나 <a class="mw-redirect" href="/wiki/%EB%8F%85%EC%9D%BC_%EC%A0%9C2%EC%A0%9C%EA%B5%AD" title="독일 제2제국">독일 제2제국</a>을 주적으로 상정하고, 러시아의 안전을 위협하는 여타 <a class="mw-redirect" href="/wiki/%EC%97%B4%EA%B0%95" title="열강">열강</a>들을 대비하도록 상정돼 있었다. <a href="/wiki/%EC%98%A4%EC%8A%A4%ED%8A%B8%EB%A6%AC%EC%95%84_%EC%A0%9C%EA%B5%AD" title="오스트리아 제국">오스트리아 제국</a>이 <a class="mw-redirect" href="/wiki/%EC%84%A0%EC%A0%84%ED%8F%AC%EA%B3%A0" title="선전포고">선전포고</a>를 해옴에 따라 <a href="/wiki/1914%EB%85%84" title="1914년">1914년</a> <a href="/wiki/6%EC%9B%94_28%EC%9D%BC" title="6월 28일">6월 28일</a> <a href="/wiki/%EB%8B%88%EC%BD%9C%EB%9D%BC%EC%9D%B4_2%EC%84%B8" title="니콜라이 2세">니콜라이 2세</a>는 부분 동원령을 실시하려 했다. 아직 선전포고가 없었던 독일을 자극하지 않으려는 의도였다. 그러나 그의 조언자들은 부분 동원령이 누가 징집될 것인가의 문제로 오히려 국내 혼란만 가중시킬 것이며, 어차피 오스트리아의 동맹국인 독일과 전쟁은 시간 문제이므로 부분 동원령은 소용없다고 간언했다. 따라서 <a href="/wiki/%EC%B0%A8%EB%A5%B4" title="차르">차르</a>는 총동원령 쪽으로 마음을 굳혔으나, 그날 밤 그의 사촌인 독일 황제 <a href="/wiki/%EB%B9%8C%ED%97%AC%EB%A6%84_2%EC%84%B8" title="빌헬름 2세">빌헬름 2세</a>가 급전을 보내 총동원령만은 자제할 것을 당부해왔다. 러시아 지휘부는 아직 독일이 러시아 공격 계획이 없다는 시그널을 받고 독일을 자극하지 않기로 하고 다시 부분 동원령을 결정하려 했다. 그런데 다음날 러시아 외무성 장관 <a class="new" href="/w/index.php?title=%EC%84%B8%EB%A5%B4%EA%B2%8C%EC%9D%B4_%EC%82%AC%EC%A1%B0%EB%85%B8%ED%94%84&amp;action=edit&amp;redlink=1" title="세르게이 사조노프 (없는 문서)">세르게이 사조노프</a>가 입궐해 <a href="/wiki/%EC%B0%A8%EB%A5%B4" title="차르">차르</a>에게 직접 총동원령을 간언했으며 결국 최종적으로 총동원령이 내려졌다.</p>
<p>이에 따라 독일도 러시아에 <a class="mw-redirect" href="/wiki/%EC%84%A0%EC%A0%84%ED%8F%AC%EA%B3%A0" title="선전포고">선전포고</a>했다. 독일은 <a href="/wiki/%EC%8A%90%EB%A6%AC%ED%8E%9C_%EA%B3%84%ED%9A%8D" title="슐리펜 계획">슐리펜 계획</a>의 순서를 폐기하고 <a class="mw-redirect" href="/wiki/%ED%97%AC%EB%AC%B4%ED%8A%B8_%ED%8F%B0_%EB%AA%B0%ED%8A%B8%EC%BC%80_(1848%EB%85%84)" title="헬무트 폰 몰트케 (1848년)">소(小) 몰트케 장군</a>의 수정안<sup class="reference" id="cite_ref-5"><a href="#cite_note-5">[5]</a></sup>에 따라 총동원령을 발동했다. 러시아도 독일도 양면전 계획으로 전쟁 플랜을 바꾼 것이다. 독일은 <a href="/wiki/8%EC%9B%94_3%EC%9D%BC" title="8월 3일">8월 3일</a> 프랑스에 선전포고하고 이어 <a href="/wiki/%EB%B2%A8%EA%B8%B0%EC%97%90" title="벨기에">벨기에</a>에 독일군 통과권을 인정하라는 최후통첩을 보냈다. <a class="mw-redirect" href="/wiki/%EC%A4%91%EB%A6%BD%EA%B5%AD" title="중립국">중립국</a>인 벨기에에 대한 위협 행위에 대해 <a href="/wiki/%EB%8C%80%EC%98%81_%EC%A0%9C%EA%B5%AD" title="대영 제국">대영 제국</a>도 독일에 선전포고하였다. 이렇게 해서 영·불·러 <a class="new" href="/w/index.php?title=%EC%82%BC%EA%B0%81%EB%8F%99%EB%A7%B9_(1882%EB%85%84)&amp;action=edit&amp;redlink=1" title="삼각동맹 (1882년) (없는 문서)">삼각동맹</a>(<span style="font-size: smaller;"><a href="/wiki/%EC%98%81%EC%96%B4" title="영어">영어</a>:</span> <span lang="en" xml:lang="en">Triple Alliance</span>,<span style="font-size: smaller;"><a href="/wiki/%ED%94%84%EB%9E%91%EC%8A%A4%EC%96%B4" title="프랑스어">프랑스어</a>:</span> <span lang="fr" xml:lang="fr">Triple Entente</span>)이 결성됐다. 독일과 러시아가 총동원령을 내리자 유럽 열강들의 연쇄적인 총동원령이 뒤따랐다. 이렇게 유럽 대륙은 세계 대전 속으로 말려들었다.</p>

<p>동원령은 전쟁 경험이 없는 군인들에게 있어서 일상에서의 해방으로 느껴졌던 모양이다. 징집 동원된 독일군들은 총구에 꽃을 달고 행진했고, 공짜 기차로 전선으로 모여들었다. 독일인들은 1만 1천량이 넘는 기차로 <a href="/wiki/%EB%9D%BC%EC%9D%B8_%EA%B0%95" title="라인 강">라인 강</a>을 넘었다. 프랑스는 7천량 정도 들었다. 민간의 말 역시 징집 대상이었다. 영국은 16만 5천 마리 가량이 기마대로 긴급 편성됐고, 오스트리아는 75만 마리, 독일은 71만 5천마리, 러시아는 백만 마리가 동원됐다<sup class="reference" id="cite_ref-Keegan_2-1"><a href="#cite_note-Keegan-2">[2]</a></sup><sup class="reference" id="cite_ref-Keegan2_6-0"><a href="#cite_note-Keegan2-6">[6]</a></sup>.</p>
<p>영연방 국가<sup class="reference" id="cite_ref-7"><a href="#cite_note-7">[7]</a></sup>인 <a class="mw-redirect" href="/wiki/%ED%98%B8%EC%A3%BC" title="호주">호주</a>, <a href="/wiki/%EC%BA%90%EB%82%98%EB%8B%A4" title="캐나다">캐나다</a>, <a href="/wiki/%EB%89%B4%EC%A7%88%EB%9E%9C%EB%93%9C" title="뉴질랜드">뉴질랜드</a> 및 <a class="mw-redirect" href="/wiki/%EB%82%A8%EC%95%84%EA%B3%B5" title="남아공">남아공</a> 국민들은 영국 본토인들과 마찬가지로 징집대상이 됐다. 하지만 영연방내에서의 동원은 또다른 문제를 야기했다. 모든 영연방 국가들은 기존의 군 편성이 아닌 새로운 전시 편성을 위해 거의 창군 수준의 재정비가 필요했다. 캐나다의 경우, 방위청 장관 샘 휴즈(Sir.Sam Hughes) 경은 캐나다 파견군을 위해 <a href="/wiki/%ED%80%98%EB%B2%A1_%EC%A3%BC" title="퀘벡 주">퀘벡 주</a>에 위치한 <a class="new" href="/w/index.php?title=CEF_%EB%B0%9C%EC%B9%B4%ED%8B%B0%EC%97%90_%EA%B8%B0%EC%A7%80&amp;action=edit&amp;redlink=1" title="CEF 발카티에 기지 (없는 문서)">CEF 발카티에 기지</a>(Valcartier) 소속 징집관들의 옷을 맞춰주는 사소한 문제만으로도 총 226통의 전보를 주고받아야 했다. 파견군 편성은 과거 통상적인 주 방위군과는 차원이 달랐다. 당시 샘 휴즈를 비롯 동원령은 캐나다로서도 처음 있는 일이었기 때문에 당시 징집 과정에 수많은 시행착오가 뒤따랐다. <a class="mw-redirect" href="/wiki/%EA%B3%A0%EB%8C%80" title="고대">고대</a> <a href="/wiki/%EC%8A%A4%EC%BD%94%ED%8B%80%EB%9E%9C%EB%93%9C" title="스코틀랜드">스코틀랜드</a>에서 <a class="mw-redirect mw-disambig" href="/wiki/%ED%95%98%EC%9D%B4%EB%9E%9C%EB%8D%94" title="하이랜더">하이랜더</a> 청년들을 소집하는 수준이라는 핀잔까지 받아야할 정도였다. <a href="/wiki/1920%EB%85%84" title="1920년">1920년</a> 오터 위원회는 부득이하게 재정과 행정력 문제로 과거 방식을 포기하던지 새 방식을 포기하던지 양단간에 결정을 내려야 했다. 캐나다의 경우는 새 방식을 채택하되 샘 휴즈가 했던 것보다 좀더 산업화된 방식의 행정적, 법적 정비를 하기로 결정했다.</p>
<p>영국의 지휘하에 모인 '식민지 백성'들은 영연방이라는 제한된 자치권 아래 있었으면서도 영국군의 일부로 편성되기를 거부하고, 독립적인 군 지휘 체계로서 따로 연방군을 결성하는데 성공했다. 그 예로 <a href="/wiki/1918%EB%85%84" title="1918년">1918년</a> 5월 호주 연대가 영국인 지휘관 윌리엄 버드웃을 자국 지휘관 존 모나쉬로 교체하면서 완전한 독립 체계를 수립한 것을 시작으로 곧 다른 영연방 국가들도 이를 따랐다. 그들은 '식민지 군단'으로서 영국 측으로서도 적국 독일 측으로서도 영국 군대와 동일하게 취급되었다.</p>
<p><a href="/wiki/1915%EB%85%84" title="1915년">1915년</a> <a href="/wiki/5%EC%9B%94_23%EC%9D%BC" title="5월 23일">5월 23일</a>, <a href="/wiki/%EC%9D%B4%ED%83%88%EB%A6%AC%EC%95%84" title="이탈리아">이탈리아</a>도 연합군으로 1차 대전에 참전했다. 연합군 중 가장 약했지만 이탈리아는 참전후 130여 개 이상의 대대를 순식간에 증강시켰다. 정규군 숫자는 참전 전 백만에서 참전 후 백오십 만으로 늘었다<sup class="reference" id="cite_ref-Keegan2_6-1"><a href="#cite_note-Keegan2-6">[6]</a></sup>. 야포 숫자도 1,788문에서 2,068문으로 늘었다<sup class="reference" id="cite_ref-Keegan2_6-2"><a href="#cite_note-Keegan2-6">[6]</a></sup>. <a href="/wiki/1916%EB%85%84" title="1916년">1916년</a> <a href="/wiki/8%EC%9B%94_17%EC%9D%BC" title="8월 17일">8월 17일</a> <a href="/wiki/%EB%A3%A8%EB%A7%88%EB%8B%88%EC%95%84" title="루마니아">루마니아</a>도 <a class="mw-redirect" href="/wiki/%EC%98%A4%EC%8A%A4%EB%A7%8C_%ED%88%AC%EB%A5%B4%ED%81%AC" title="오스만 투르크">오스만 투르크</a>와 전쟁을 시작하면서 연합군 측에 붙었고, 23개 사단을 동원령으로 만들어냈다. 루마니아는 그러나 독일, 오스트리아, 불가리아 연합군에 얼마버티지 못하고 항복했다. <a href="/wiki/%EB%B6%88%EA%B0%80%EB%A6%AC%EC%95%84" title="불가리아">불가리아</a>는 무려 전체 인구의 1/4에 달하는 80만을 동원했는데, 이는 인구 대비 비율로만 따지면 1차 세계 대전에서 가장 높은 것이었다.</p>
<p>군수 물자의 생산도 전쟁을 통해 지속적으로 늘려나갔다. <a href="/wiki/%EB%9F%AC%EC%8B%9C%EC%95%84_%EC%A0%9C%EA%B5%AD" title="러시아 제국">러시아 제국</a>은 동원령으로 국내 생산을 순식간에 전시 체제로 전환해, 포탄 생산량에서 전쟁 전보다 2천 퍼센트 증가라는 놀라운 생산량을 보였다. 불과 <a href="/wiki/1915%EB%85%84" title="1915년">1915년</a> 11월에 매달 백오십만 발이 넘는 포탄 생산량을 보이고 있었다<sup class="reference" id="cite_ref-Keegan2_6-3"><a href="#cite_note-Keegan2-6">[6]</a></sup>. 프랑스에서는 여성들을 대량 동원해 군수 물자 공장을 돌리면서 <a href="/wiki/1915%EB%85%84" title="1915년">1915년</a> 매일 10만 개의 포탄 생산량을 보였다. 매달 3백만 발이라는 엄청난 양이었다<sup class="reference" id="cite_ref-8"><a href="#cite_note-8">[8]</a></sup>.</p>
<p>양측은 동원령으로 엄청난 숫자의 군대를 찍어냈다. 영국 국방장관 <a class="new" href="/w/index.php?title=%ED%98%B8%EB%A0%88%EC%9D%B4%EC%87%BC_%ED%82%A4%EC%B9%98%EB%84%88&amp;action=edit&amp;redlink=1" title="호레이쇼 키치너 (없는 문서)">호레이쇼 키치너</a> <a href="/wiki/%EB%B0%B1%EC%9E%91" title="백작">백작</a>은 수십만의 군대가 필요하다고 국민들에게 어필했고 이는 열광적인 반응을 불렀다. 순식간에 30개 사단이 증강됐다. 지원자들이 몰려들어 <a href="/wiki/1916%EB%85%84" title="1916년">1916년</a>까지 영국에는 징병을 따로 하지 않아도 될 정도였다. 뉴질랜드를 시작으로 캐나다 등 영연방 국가들의 동원령도 뒤따랐다.</p>
<p><a href="/wiki/1917%EB%85%84" title="1917년">1917년</a> <a href="/wiki/4%EC%9B%94_6%EC%9D%BC" title="4월 6일">4월 6일</a> 미국이 참전했다. 참전 당시 미국은 107,641명을 동원해 참전국 중 17위에 지나지 않는 동원 병력수를 보였으나, <a class="mw-redirect" href="/wiki/%EB%AF%B8_%ED%95%B4%EA%B5%B0" title="미 해군">미 해군</a>의 동원 규모가 엄청나, 5척의 <a href="/wiki/%EB%93%9C%EB%A0%88%EB%93%9C%EB%85%B8%ED%8A%B8" title="드레드노트">드레드노트</a>가 연합 해군에 공여됐다. 미국은 징집으로 병력을 늘렸다. <a href="/wiki/1918%EB%85%84" title="1918년">1918년</a> 3월에는 31만 8천 명의 병력이 프랑스로 파견됐고, 종전 시까지 3백만이라는 엄청난 수의 미군 병력이 유럽에 동원됐다<sup class="reference" id="cite_ref-9"><a href="#cite_note-9">[9]</a></sup>.</p>

<p><a class="new" href="/w/index.php?title=%ED%94%84%EB%9E%91%EC%8A%A4-%ED%8F%B4%EB%9E%80%EB%93%9C_%EA%B5%B0%EC%82%AC%EB%8F%99%EB%A7%B9&amp;action=edit&amp;redlink=1" title="프랑스-폴란드 군사동맹 (없는 문서)">프랑스-폴란드 군사동맹</a> 조약에 따라 프랑스는 동원령을 실시했다. 프랑스 정부가 공표한 예비 동원령은 <a href="/wiki/8%EC%9B%94_26%EC%9D%BC" title="8월 26일">8월 26일</a>이었고, <a href="/wiki/9%EC%9B%94_1%EC%9D%BC" title="9월 1일">9월 1일</a>에 총동원령이 공표됐었다. 동원령을 실시한 3일 후에는 대규모 반격을 위한 준비가 시작됐다. 프랑스군은 마지노 선과 독일 국경 사이의 완충 지대를 압도적인 병력을 바탕으로 효과적으로 컨트롤하면서 독일의 방어선을 탐색했다. 동원 15일 후인 <a href="/wiki/1939%EB%85%84" title="1939년">1939년</a> <a href="/wiki/9%EC%9B%94_16%EC%9D%BC" title="9월 16일">9월 16일</a>에 프랑스 육군은 독일 진격을 시작했다.</p>
<p><a href="/wiki/%ED%8F%B4%EB%9E%80%EB%93%9C_%EC%A0%9C2%EA%B3%B5%ED%99%94%EA%B5%AD" title="폴란드 제2공화국">폴란드</a>는 <a href="/wiki/%EC%98%81%EA%B5%AD-%ED%8F%B4%EB%9E%80%EB%93%9C_%EA%B5%B0%EC%82%AC%EB%8F%99%EB%A7%B9" title="영국-폴란드 군사동맹">영국-폴란드 군사동맹</a>으로 <a href="/wiki/%EB%82%98%EC%B9%98_%EB%8F%85%EC%9D%BC" title="나치 독일">나치 독일</a>과 긴장이 높아지면서 <a href="/wiki/1939%EB%85%84" title="1939년">1939년</a> <a href="/wiki/8%EC%9B%94_24%EC%9D%BC" title="8월 24일">8월 24일</a> 부분적 동원령을 발동했고, 일주일 후인 <a href="/wiki/8%EC%9B%94_30%EC%9D%BC" title="8월 30일">8월 30일</a>에 총동원령으로 바꾼 바 있다. <a href="/wiki/9%EC%9B%94_1%EC%9D%BC" title="9월 1일">9월 1일</a> 독일군이 폴란드를 침공했고 군사동맹의 자동 참전 조항에 따라 영국과 프랑스가 독일에 선전포고했다. 그러나 영불 연합은 동원령 발동을 주저했고, 이미 폴란드는 <a href="/wiki/%EC%B6%94%EC%B6%95%EA%B5%AD" title="추축국">추축국</a>들에 의해 유린당할대로 당한 지경이었다. 단지 프랑스만 <a class="mw-redirect" href="/wiki/%EC%9E%90%EB%A5%B4_%EA%B3%B5%EC%84%B8" title="자르 공세">자르 공세</a>라는 시덥잖은 공격을 잠깐 했을 뿐이었다.</p>
<p><a href="/wiki/%EC%BA%90%EB%82%98%EB%8B%A4" title="캐나다">캐나다</a>는 <a href="/wiki/1939%EB%85%84" title="1939년">1939년</a> <a href="/wiki/8%EC%9B%94_25%EC%9D%BC" title="8월 25일">8월 25일</a> 부분 동원령을 발동하여 영국 본섬으로 병력을 파견했다. <a href="/wiki/9%EC%9B%94_1%EC%9D%BC" title="9월 1일">9월 1일</a> 독일의 프랑스 침공을 맞아 2개 사단을 동원하고 <a href="/wiki/9%EC%9B%94_10%EC%9D%BC" title="9월 10일">9월 10일</a> 영국으로 1개 사단만 건너갔다. 캐나다 자치정부로서는 외교권, 군사권을 포함 전면적인 자치권을 영국으로부터 위임받은 상태였지만 선전포고를 정식으로 하지도 않았고, 면피 수준으로만 지원을 한 것이었다. <a href="/wiki/%ED%94%84%EB%9E%91%EC%8A%A4_%EA%B3%B5%EB%B0%A9%EC%A0%84" title="프랑스 공방전">프랑스 공방전</a>이 시작된 <a href="/wiki/1940%EB%85%84" title="1940년">1940년</a> 5월이 되어서야 캐나다 자치정부는 전쟁이 본격화되고 있음을 깨닫고 3개 사단을 더 동원해 대서양 건너로 보냈다<sup class="reference" id="cite_ref-10"><a href="#cite_note-10">[10]</a></sup>. <a href="/wiki/1940%EB%85%84" title="1940년">1940년</a> 동원령 관련법을 정비한 캐나다는 <a href="/wiki/%ED%83%9C%ED%8F%89%EC%96%91_%EC%A0%84%EC%9F%81" title="태평양 전쟁">태평양 전쟁</a>에도 <a class="new" href="/w/index.php?title=%EC%95%8C%EB%A3%A8%EC%83%A8_%EC%97%B4%EB%8F%84&amp;action=edit&amp;redlink=1" title="알루샨 열도 (없는 문서)">알루샨 열도</a> 방위를 맡았다.</p>
<h2><span class="mw-headline" id=".EB.83.89.EC.A0.84_.ED.95.98.EC.97.90.EC.84.9C.EC.9D.98_.EC.B4.9D.EB.8F.99.EC.9B.90.EB.A0.B9">냉전 하에서의 총동원령</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%8F%99%EC%9B%90%EB%A0%B9&amp;action=edit&amp;section=3" title="부분 편집: 냉전 하에서의 총동원령">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<div class="rellink relarticle mainarticle"><img alt="" data-file-height="128" data-file-width="128" height="16" src="//upload.wikimedia.org/wikipedia/commons/thumb/e/ec/Crystal_Clear_app_xmag.svg/16px-Crystal_Clear_app_xmag.svg.png" srcset="//upload.wikimedia.org/wikipedia/commons/thumb/e/ec/Crystal_Clear_app_xmag.svg/24px-Crystal_Clear_app_xmag.svg.png 1.5x, //upload.wikimedia.org/wikipedia/commons/thumb/e/ec/Crystal_Clear_app_xmag.svg/32px-Crystal_Clear_app_xmag.svg.png 2x" width="16"/> 이 부분의 본문은 <a href="/wiki/%EB%83%89%EC%A0%84" title="냉전">냉전</a>입니다.</div>
<p>대리전 양상을 띄는 냉전 체제에서 총동원령은 미소 양강이나 그 동맹국들 역시 시행한 적이 없으며 의미있는 규모의 국가 동원은 없었다. <a class="mw-redirect" href="/wiki/6.25_%EC%A0%84%EC%9F%81" title="6.25 전쟁">6.25 전쟁</a> 혹은 <a class="mw-redirect" href="/wiki/%EB%B2%A0%ED%8A%B8%EB%82%A8_%EC%A0%84" title="베트남 전">베트남 전</a>, <a href="/wiki/%EA%B1%B8%ED%94%84_%EC%A0%84%EC%9F%81" title="걸프 전쟁">걸프 전</a> 하다못해 <a href="/wiki/%ED%8F%AC%ED%81%B4%EB%9E%9C%EB%93%9C_%EC%A0%84%EC%9F%81" title="포클랜드 전쟁">포클랜드 전쟁</a> 조차도 국가 총동원령은 없었다.</p>
<h2><span class="mw-headline" id=".EA.B2.BD.EC.A0.9C.EC.A0.81_.EC.B4.9D.EB.8F.99.EC.9B.90">경제적 총동원</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%8F%99%EC%9B%90%EB%A0%B9&amp;action=edit&amp;section=4" title="부분 편집: 경제적 총동원">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p><b>경제적 총동원</b>은 국가 비상 사태를 맞아 국가 경제 구조를 전시 체제로 전환하는 것을 말한다<sup class="reference" id="cite_ref-11"><a href="#cite_note-11">[11]</a></sup>.</p>
