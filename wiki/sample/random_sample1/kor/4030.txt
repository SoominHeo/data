<h1 class="firstHeading" id="firstHeading" lang="ko">동인도 제도</h1>
<div class="mw-body-content" id="bodyContent">
<div id="siteSub">위키백과, 우리 모두의 백과사전.</div>
<div id="contentSub"></div>
<div class="mw-jump" id="jump-to-nav">
					이동:					<a href="#mw-head">둘러보기</a>, 					<a href="#p-search">검색</a>

<div class="thumbinner" style="width:302px;"><a class="image" href="/wiki/%ED%8C%8C%EC%9D%BC:Dampier,_Map_of_the_East_Indies.jpg"><img alt="" class="thumbimage" data-file-height="829" data-file-width="1429" height="174" src="//upload.wikimedia.org/wikipedia/commons/thumb/b/b4/Dampier%2C_Map_of_the_East_Indies.jpg/300px-Dampier%2C_Map_of_the_East_Indies.jpg" srcset="//upload.wikimedia.org/wikipedia/commons/thumb/b/b4/Dampier%2C_Map_of_the_East_Indies.jpg/450px-Dampier%2C_Map_of_the_East_Indies.jpg 1.5x, //upload.wikimedia.org/wikipedia/commons/thumb/b/b4/Dampier%2C_Map_of_the_East_Indies.jpg/600px-Dampier%2C_Map_of_the_East_Indies.jpg 2x" width="300"/></a>

윌리엄 댐피어의 1697년 지도</div>
</div>

<p><b>동인도 제도</b>(East Indies)는 16세기 이후 현재의 <a href="/wiki/%EC%9D%B8%EB%8F%84_%EC%95%84%EB%8C%80%EB%A5%99" title="인도 아대륙">인도 아대륙</a>, <a href="/wiki/%EB%8F%99%EB%82%A8%EC%95%84%EC%8B%9C%EC%95%84" title="동남아시아">동남아시아</a>, <a href="/wiki/%EC%98%A4%EC%84%B8%EC%95%84%EB%8B%88%EC%95%84" title="오세아니아">오세아니아</a> 제도, 그리고 동남아 해양 지역을 구분하기 위해 유럽인들에 의해 사용된 용어이다. 이 용어에서 전통적으로 <a href="/wiki/%EC%A4%91%EA%B5%AD" title="중국">중국</a>, <a href="/wiki/%ED%95%9C%EA%B5%AD" title="한국">한국</a>, <a href="/wiki/%EC%9D%BC%EB%B3%B8" title="일본">일본</a> 그리고 인도의 북부 국가들 그리고 히말라야 지역은 포함되지 않았으며, <a href="/wiki/%EA%B7%B9%EB%8F%99" title="극동">극동</a>이라는 용어로 사용되었다.<sup class="reference" id="cite_ref-1"><a href="#cite_note-1">[1]</a></sup></p>
<p>‘인디아’(<i>India</i>)와 ‘인디즈’(<i>the Indies</i>)와 같은 이름은 오늘날의 <a href="/wiki/%ED%8C%8C%ED%82%A4%EC%8A%A4%ED%83%84" title="파키스탄">파키스탄</a>과 <a href="/wiki/%EC%9D%B8%EB%8F%84" title="인도">인도</a>, <a href="/wiki/%ED%8B%B0%EB%B2%A0%ED%8A%B8" title="티베트">티베트</a> 서부를 흐르는 <a href="/wiki/%EC%9D%B8%EB%8D%94%EC%8A%A4_%EA%B0%95" title="인더스 강">인더스 강</a>에서 유래되었으며, 고대 <a href="/wiki/%EA%B7%B8%EB%A6%AC%EC%8A%A4%EC%9D%B8" title="그리스인">그리스인</a>들이 아시아 대부분의 지역과 <a class="mw-disambig" href="/wiki/%ED%8E%98%EB%A5%B4%EC%8B%9C%EC%95%84" title="페르시아">페르시아</a>의 동부 아시아를 가리키는 말로 사용되었다. 이러한 사용은 적어도 <a class="mw-redirect" href="/wiki/%ED%97%A4%EB%A1%9C%EB%8F%84%ED%88%AC%EC%8A%A4" title="헤로도투스">헤로도투스</a>가 활약하던 기원전 5세기로 거슬러 올라갈 수 있다.</p>
<p>네덜란드 식민지 시대에, <a href="/wiki/%EC%9D%B8%EB%8F%84%EB%84%A4%EC%8B%9C%EC%95%84" title="인도네시아">인도네시아</a>는 <a href="/wiki/%EB%84%A4%EB%8D%9C%EB%9E%80%EB%93%9C%EB%A0%B9_%EB%8F%99%EC%9D%B8%EB%8F%84" title="네덜란드령 동인도">네덜란드령 동인도</a> 또는 네덜란드령 동인도 제도로 불렸다. 인도네시아라는 이름 자체는 그리스 어원인 ‘인도’(Indo, 인더스 또는 인디아에서 나온 인도)에서 온 것이며, Nesos는 섬을 의미한다.</p>
