<p><b>동청철도</b>(東淸鐵道, <span style="font-size: smaller;"><a href="/wiki/%EC%98%81%EC%96%B4" title="영어">영어</a>:</span> <span lang="en" xml:lang="en">Chinese Eastern Railway, <b>CER</b></span>)는 <a class="mw-redirect" href="/wiki/%ED%95%98%EC%96%BC%EB%B9%88" title="하얼빈">하얼빈</a>을 중심으로 하여 <a class="mw-redirect" href="/wiki/%EB%A7%8C%EC%A0%80%EC%9A%B0%EB%A6%AC" title="만저우리">만저우리</a> (만주리)와 <a class="mw-redirect" href="/wiki/%EC%91%A4%EC%9D%B4%ED%8E%80%ED%97%88" title="쑤이펀허">쑤이펀허</a> (수분하), 그리고 <a class="mw-redirect" href="/wiki/%EB%8B%A4%EB%A1%84" title="다롄">다롄</a>을 잇는 철도 노선이다. <a href="/wiki/1911%EB%85%84" title="1911년">1911년</a> <a href="/wiki/%EC%A4%91%ED%99%94%EB%AF%BC%EA%B5%AD" title="중화민국">중화민국</a>이 성립한 뒤에는 <b>중동 철도</b>(中東鐵道)로 불렸다.</p>
<h2><span class="mw-headline" id=".EC.97.AD.EC.82.AC">역사</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%8F%99%EC%B2%AD_%EC%B2%A0%EB%8F%84&amp;action=edit&amp;section=1" title="부분 편집: 역사">편집</a><span class="mw-editsection-bracket">]</span></span></h2>

<td width="auto"><img alt="" data-file-height="204" data-file-width="262" height="19" src="//upload.wikimedia.org/wikipedia/commons/thumb/6/64/Question_book-4.svg/25px-Question_book-4.svg.png" srcset="//upload.wikimedia.org/wikipedia/commons/thumb/6/64/Question_book-4.svg/38px-Question_book-4.svg.png 1.5x, //upload.wikimedia.org/wikipedia/commons/thumb/6/64/Question_book-4.svg/50px-Question_book-4.svg.png 2x" width="25"/> 이 문단의 <b>내용은</b> <b><a href="/wiki/%EC%9C%84%ED%82%A4%EB%B0%B1%EA%B3%BC:%EC%B6%9C%EC%B2%98_%EB%B0%9D%ED%9E%88%EA%B8%B0" title="위키백과:출처 밝히기">출처</a>가 분명하지 않습니다.</b> 지금 바로 이 <span class="plainlinks"><a class="external text" href="//ko.wikipedia.org/w/index.php?title=%EB%8F%99%EC%B2%AD_%EC%B2%A0%EB%8F%84&amp;action=edit">문단을 편집</a></span>하여, 참고하신 문헌이나 <a href="/wiki/%EC%9C%84%ED%82%A4%EB%B0%B1%EA%B3%BC:%EC%8B%A0%EB%A2%B0%ED%95%A0_%EC%88%98_%EC%9E%88%EB%8A%94_%EC%B6%9C%EC%B2%98" title="위키백과:신뢰할 수 있는 출처">신뢰할 수 있는 출처</a>를 각주 등으로 표기해 주세요. <a href="/wiki/%EC%9C%84%ED%82%A4%EB%B0%B1%EA%B3%BC:%ED%99%95%EC%9D%B8_%EA%B0%80%EB%8A%A5" title="위키백과:확인 가능">검증</a>되지 않은 내용은 삭제될 수도 있습니다. 내용에 대한 의견이 있으시다면 <a href="/wiki/%ED%86%A0%EB%A1%A0:%EB%8F%99%EC%B2%AD_%EC%B2%A0%EB%8F%84#.23" title="토론:동청 철도">토론 문서</a>에서 나누어 주세요. <p><a href="/wiki/%EB%9F%AC%EC%8B%9C%EC%95%84_%EC%A0%9C%EA%B5%AD" title="러시아 제국">러시아 제국</a>은 <a href="/wiki/%EC%B9%98%ED%83%80" title="치타">치타</a>를 출발해 만주 북부를 횡단, <a href="/wiki/%EB%B8%94%EB%9D%BC%EB%94%94%EB%B3%B4%EC%8A%A4%ED%86%A0%ED%81%AC" title="블라디보스토크">블라디보스토크</a>에 이르는 철도 노선을 계획하였다. 그리고 러시아는 <a href="/wiki/%EC%B2%AD%EC%9D%BC_%EC%A0%84%EC%9F%81" title="청일 전쟁">청일 전쟁</a>에서 이긴 <a href="/wiki/%EC%9D%BC%EB%B3%B8" title="일본">일본</a>이 요동 반도를 영유하는 것을 막은 대가로 <a href="/wiki/%EC%B2%AD%EB%82%98%EB%9D%BC" title="청나라">청나라</a>와 협상해 동청철도 부설권을 얻는다. <a href="/wiki/1898%EB%85%84" title="1898년">1898년</a> <a href="/wiki/3%EC%9B%94" title="3월">3월</a> 러시아는 여순과 대련 일대를 조차 하고 <a class="mw-redirect" href="/wiki/%ED%95%98%EC%96%BC%EB%B9%88" title="하얼빈">하얼빈</a>에서 <a class="mw-redirect" href="/wiki/%EB%8C%80%EB%A0%A8" title="대련">대련</a>에 이르는 <a href="/wiki/%EB%82%A8%EB%A7%8C%EC%A3%BC%EC%B2%A0%EB%8F%84" title="남만주철도">남만주철도</a>의 부설권도 얻었다. 하지만 <a href="/wiki/%EB%9F%AC%EC%9D%BC_%EC%A0%84%EC%9F%81" title="러일 전쟁">러일 전쟁</a>에서 러시아가 지고, 그에 따라 맺어진 포츠머스 조약에 의해 <a class="mw-redirect" href="/wiki/%EC%9E%A5%EC%B6%98" title="장춘">장춘</a> 이남의 남만주철도는 일본의 소유가 되었다. <a href="/wiki/1918%EB%85%84" title="1918년">1918년</a> <a href="/wiki/%EB%9F%AC%EC%8B%9C%EC%95%84_%EC%A0%9C%EA%B5%AD" title="러시아 제국">러시아 제국</a>이 멸망하자, <a class="mw-redirect" href="/wiki/%EC%86%8C%EB%B9%84%EC%97%90%ED%8A%B8_%EC%97%B0%EB%B0%A9" title="소비에트 연방">소비에트 연방</a> (소련)이 동청철도의 이권을 계승했다.</p>
<p><a href="/wiki/1924%EB%85%84" title="1924년">1924년</a> <a href="/wiki/5%EC%9B%94" title="5월">5월</a> <a href="/wiki/%EC%A4%91%ED%99%94%EB%AF%BC%EA%B5%AD" title="중화민국">중화민국</a>과 <a href="/wiki/%EC%86%8C%EB%A0%A8" title="소련">소련</a> 양국은 국교를 맺기 위해 수교 협정을 체결하였다. 이 수교 협정에서 동청철도의 <a href="/wiki/%EC%86%8C%EB%A0%A8" title="소련">소련</a> 소유가 인정되었다. 하지만 이 협정에 불만을 가진 <a class="mw-redirect" href="/wiki/%EC%9E%A5%EC%9E%91%EB%A6%BC" title="장작림">장작림</a> 군벌 정권은 중화민국 정부와 별도로 <a href="/wiki/1924%EB%85%84" title="1924년">1924년</a> <a href="/wiki/9%EC%9B%94" title="9월">9월</a> <a href="/wiki/%EC%86%8C%EB%A0%A8" title="소련">소련</a>과 협정을 체결하였다. <a href="/wiki/1929%EB%85%84" title="1929년">1929년</a> <a class="mw-redirect" href="/wiki/%EC%9E%A5%ED%95%99%EB%9F%89" title="장학량">장학량</a>은 소련의 협정 위반을 이유로 동청철도 이권 회수를 시도했으나 소련군의 방해에 의해 실패하게 되었다.</p>
<p><a href="/wiki/1931%EB%85%84" title="1931년">1931년</a> <a class="mw-redirect" href="/wiki/%EB%A7%8C%EC%A3%BC_%EC%82%AC%EB%B3%80" title="만주 사변">만주 사변</a>이 발발하고, <a href="/wiki/1932%EB%85%84" title="1932년">1932년</a> 일본에 의해 괴뢰국인 <a href="/wiki/%EB%A7%8C%EC%A3%BC%EA%B5%AD" title="만주국">만주국</a>이 성립하자 <a href="/wiki/%EC%86%8C%EB%A0%A8" title="소련">소련</a>은 <a href="/wiki/%EC%9D%BC%EB%B3%B8" title="일본">일본</a>과의 무력 충돌을 피하기 위해 동청 철도에 관한 권리를 만주국 정부에게 팔았다. 동청철도를 판 소련은 <a href="/wiki/1935%EB%85%84" title="1935년">1935년</a> <a href="/wiki/3%EC%9B%94" title="3월">3월</a> 만주에서 철수하였다. 동청철도가 만주국의 국유 철도가 되자, 일본은 중국 대륙 침공을 원활히 하기 위해 <a href="/wiki/%EA%B4%91%EA%B6%A4" title="광궤">광궤</a>였던 궤도를 <a href="/wiki/%ED%91%9C%EC%A4%80%EA%B6%A4" title="표준궤">표준궤</a>로 바꾸었다.</p>
<p><a href="/wiki/1945%EB%85%84" title="1945년">1945년</a> <a href="/wiki/8%EC%9B%94" title="8월">8월</a> <a href="/wiki/%EC%86%8C%EB%A0%A8" title="소련">소련</a>은 <a class="mw-redirect" href="/wiki/8%EC%9B%94%EC%9D%98_%ED%8F%AD%ED%92%8D_%EC%9E%91%EC%A0%84" title="8월의 폭풍 작전">일본에게 선전 포고를 하는 동시에 만주에 침공</a>하면서 동청철도 및 <a href="/wiki/%EB%82%A8%EB%A7%8C%EC%A3%BC%EC%B2%A0%EB%8F%84" title="남만주철도">남만주철도</a>를 차지하였다. 그리고 이 두 철도를 합해 명칭을 장춘 철도라고 개칭하였다. <a href="/wiki/1945%EB%85%84" title="1945년">1945년</a> <a href="/wiki/8%EC%9B%94" title="8월">8월</a> <a href="/wiki/%EC%86%8C%EB%A0%A8" title="소련">소련</a>은 <a href="/wiki/%EC%A4%91%ED%99%94%EB%AF%BC%EA%B5%AD" title="중화민국">중화민국</a> <a href="/wiki/%EA%B5%AD%EB%AF%BC%EC%A0%95%EB%B6%80" title="국민정부">국민정부</a>의 <a href="/wiki/%EC%9E%A5%EC%A0%9C%EC%8A%A4" title="장제스">장제스</a>와 동맹 조약을 맺어 장춘철도를 소련과 중국이 30여년간 공동으로 사용하는 것을 합의하였다. 그러나 <a href="/wiki/1949%EB%85%84" title="1949년">1949년</a> <a class="mw-redirect" href="/wiki/%EC%A4%91%EA%B5%AD_%EA%B3%B5%EC%82%B0%EB%8B%B9" title="중국 공산당">중국 공산당</a>이 <a href="/wiki/%EC%A4%91%ED%99%94%EC%9D%B8%EB%AF%BC%EA%B3%B5%ED%99%94%EA%B5%AD" title="중화인민공화국">중화인민공화국</a>을 성립하고 <a href="/wiki/%EC%9E%A5%EC%A0%9C%EC%8A%A4" title="장제스">장제스</a>의 <a href="/wiki/%EC%A4%91%ED%99%94%EB%AF%BC%EA%B5%AD" title="중화민국">중화민국</a> 정부가 <a href="/wiki/%ED%83%80%EC%9D%B4%EC%99%84" title="타이완">타이완</a>으로 이전하자, 소련과 중공은 <a href="/wiki/1949%EB%85%84" title="1949년">1949년</a> 동맹 조약을 개정해 장춘 철도의 소유권을 <a href="/wiki/%EC%A4%91%ED%99%94%EC%9D%B8%EB%AF%BC%EA%B3%B5%ED%99%94%EA%B5%AD" title="중화인민공화국">중화인민공화국</a>이 갖는다는 것을 합의함으로써 장춘철도의 모든 권리는 중화인민공화국으로 넘어 갔다.</p>
<p>중화인민공화국 성립 후 만저우리~하얼빈 구간은 <a href="/wiki/%EB%B9%88%EC%A0%80%EC%9A%B0_%EC%B2%A0%EB%A1%9C" title="빈저우 철로">빈저우 철로</a>로, 하얼빈~쑤이펀허 구간은 <a href="/wiki/%EB%B9%88%EC%91%A4%EC%9D%B4_%EC%B2%A0%EB%A1%9C" title="빈쑤이 철로">빈쑤이 철로</a>로, 하얼빈~다롄 구간은 <a class="new" href="/w/index.php?title=%ED%95%98%EB%8B%A4_%EC%B2%A0%EB%A1%9C&amp;action=edit&amp;redlink=1" title="하다 철로 (없는 문서)">하다 철로</a>(<a class="extiw" href="https://zh.wikipedia.org/wiki/%E5%93%88%E5%A4%A7%E9%90%B5%E8%B7%AF" title="zh:哈大鐵路">zh</a>)로 분할되었다.</p>
