<p><b>동치미</b>는 <a class="mw-redirect" href="/wiki/%ED%95%9C%EA%B5%AD%EC%9A%94%EB%A6%AC" title="한국요리">한국요리</a>에서 <a class="mw-redirect" href="/wiki/%EB%AC%BC%EA%B9%80%EC%B9%98" title="물김치">물김치</a>의 일종으로, <a class="mw-redirect" href="/wiki/%EB%AC%B4%EC%9A%B0" title="무우">무우</a>, <a href="/wiki/%EB%B0%B0%EC%B6%94" title="배추">배추</a>, <a href="/wiki/%ED%8C%8C_(%EC%8B%9D%EB%AC%BC)" title="파 (식물)">파</a>, 삭힌 <a href="/wiki/%EA%B3%A0%EC%B6%94" title="고추">고추</a>, <a href="/wiki/%EC%83%9D%EA%B0%95" title="생강">생강</a>, <a class="mw-redirect" href="/wiki/%EB%B0%B0_(%EA%B3%BC%EC%9D%BC)" title="배 (과일)">배</a>와 <a class="new" href="/w/index.php?title=%EA%B5%AD%EB%AC%BC&amp;action=edit&amp;redlink=1" title="국물 (없는 문서)">국물</a>로 구성된다. 동치미는 이름처럼 전통적으로 <a href="/wiki/%EA%B2%A8%EC%9A%B8" title="겨울">겨울</a>에 담궈서 먹는다.<sup class="reference" id="cite_ref-1"><a href="#cite_note-1">[1]</a></sup></p>
<p>동치미도 다른 김치처럼 <a href="/wiki/%EB%B0%9C%ED%9A%A8" title="발효">발효</a>를 시키지만, 2~3일간의 숙성으로 다른 김치에 비해 그 기간이 비교적 짧다. 한국인들은 <a class="new" href="/w/index.php?title=%EA%B9%80%EC%9E%A5%EC%B2%A0&amp;action=edit&amp;redlink=1" title="김장철 (없는 문서)">김장철</a> 바로 직전, 즉 늦은 가을에 동치미를 담근다. 특히 조선민주주의인민공화국에서 <a href="/wiki/%ED%95%A8%EA%B2%BD%EB%8F%84" title="함경도">함경도</a>, <a href="/wiki/%ED%8F%89%EC%95%88%EB%8F%84" title="평안도">평안도</a>를 포함하는 <a class="new" href="/w/index.php?title=%EC%9D%B4%EB%B6%81%EC%A7%80%EB%B0%A9&amp;action=edit&amp;redlink=1" title="이북지방 (없는 문서)">이북지방</a>의 동치미가 유명하다.<sup class="reference" id="cite_ref-2"><a href="#cite_note-2">[2]</a></sup></p>
<p>시원하고, 깔끔한 국물 덕분에, 동치미 국물은 <a class="new" href="/w/index.php?title=%EB%8F%99%EC%B9%98%EB%AF%B8_%EA%B5%AD%EC%88%98&amp;action=edit&amp;redlink=1" title="동치미 국수 (없는 문서)">동치미 국수</a>나 <a href="/wiki/%EB%83%89%EB%A9%B4" title="냉면">냉면</a>을 만드는 데 국물로 사용되기도 한다. 또는 <a href="/wiki/%EB%96%A1" title="떡">떡</a>을 먹거나, 찐 <a href="/wiki/%EA%B3%A0%EA%B5%AC%EB%A7%88" title="고구마">고구마</a>를 먹을 때 같이 먹기도 한다.<sup class="reference" id="cite_ref-3"><a href="#cite_note-3">[3]</a></sup></p>
