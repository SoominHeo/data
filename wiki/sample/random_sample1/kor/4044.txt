<p><b>동탁</b>(董卓, ? ~ <a href="/wiki/192%EB%85%84" title="192년">192년</a> <a href="/wiki/%EC%9D%8C%EB%A0%A5_4%EC%9B%94_23%EC%9D%BC" title="음력 4월 23일">음력 4월 23일</a>)은 <a href="/wiki/%EC%A4%91%EA%B5%AD" title="중국">중국</a> <a href="/wiki/%ED%9B%84%ED%95%9C" title="후한">후한</a> 말의 <a href="/wiki/%EC%9E%A5%EA%B5%B0" title="장군">장군</a> 겸 <a href="/wiki/%EC%A0%95%EC%B9%98%EA%B0%80" title="정치가">정치가</a>로 <a href="/wiki/%EC%9E%90_(%EC%9D%B4%EB%A6%84)" title="자 (이름)">자</a>는 중영(仲穎)이며 <a href="/wiki/%EC%96%91%EC%A3%BC_(%E6%B6%BC%E5%B7%9E)" title="양주 (涼州)">양주</a>(凉州) 농서군(隴西郡) 임조현(臨洮縣) <a href="/wiki/%EC%82%AC%EB%9E%8C" title="사람">사람</a>이다. 변방에서 무공을 쌓았다. <a class="mw-redirect" href="/wiki/%EC%98%81%EC%A0%9C" title="영제">영제</a> 사후 <a href="/wiki/%EC%8B%AD%EC%83%81%EC%8B%9C%EC%9D%98_%EB%82%9C" title="십상시의 난">십상시의 난</a> 등 정치적 혼란을 틈타 정권을 잡았다. <a href="/wiki/%ED%9B%84%ED%95%9C_%EC%86%8C%EC%A0%9C_(13%EB%8C%80)" title="후한 소제 (13대)">소제</a>를 폐하고 <a href="/wiki/%EB%B6%80%EC%A0%95%EB%B6%80%ED%8C%A8" title="부정부패">부패</a>·<a href="/wiki/%EC%82%B4%EC%9D%B8" title="살인">살인</a>·<a href="/wiki/%EC%95%BD%ED%83%88" title="약탈">약탈</a>·<a href="/wiki/%EB%8F%84%EA%B5%B4" title="도굴">도굴</a>·<a href="/wiki/%EB%B0%A9%ED%99%94" title="방화">방화</a>·축재 등 온갖 폭정을 휘둘렀다. 관리들의 대규모 무장 항명사태인 <a class="mw-redirect" href="/wiki/%EB%B0%98%EB%8F%99%ED%83%81_%EC%97%B0%ED%95%A9%EA%B5%B0" title="반동탁 연합군">반동탁 연합군</a>과 각종 암살 시도가 있었고 결국 <a href="/wiki/%EC%99%95%EC%9C%A4" title="왕윤">왕윤</a>·<a href="/wiki/%EC%97%AC%ED%8F%AC" title="여포">여포</a>에게 죽임을 당하였다. 이후 대표적인 악인으로 두루 회자되었다.</p>

<p>젊어서부터 <a class="mw-redirect" href="/wiki/%EA%B0%95%EC%A1%B1" title="강족">강족</a> 지역을 유랑하며 유력자와 친분을 맺었다. 돌아와서는 <a class="mw-redirect" href="/wiki/%EB%86%8D%EC%82%AC" title="농사">농사</a>를 지었는데 친했던 강족 <a href="/wiki/%EB%AC%B4%EB%A6%AC" title="무리">무리</a>가 찾아오자 <a href="/wiki/%EB%B0%AD" title="밭">밭</a> 갈던 <a href="/wiki/%EC%86%8C" title="소">소</a>를 잡아 연회를 베풀고 같이 즐겼다. 그러자 강족들이 감격하여 <a href="/wiki/%EB%A7%90_(%EB%8F%99%EB%AC%BC)" title="말 (동물)">말</a>과 소를 제외한 <a href="/wiki/%EA%B0%80%EC%B6%95" title="가축">가축</a> 1,000여 마리를 선물로 주었다. <a href="/wiki/%ED%9B%84%ED%95%9C_%ED%99%98%EC%A0%9C" title="후한 환제">환제</a>의 말년인 <a href="/wiki/167%EB%85%84" title="167년">167년</a>(<a href="/wiki/%EC%98%81%EA%B0%95_(%ED%9B%84%ED%95%9C)" title="영강 (후한)">영강</a> 원년), 동탁은 양갓집 자제로서 우림랑(羽林郞)으로 뽑혀, <a href="/wiki/%ED%9D%89%EB%85%B8" title="흉노">흉노</a> <a href="/wiki/%EC%A4%91%EB%9E%91%EC%9E%A5" title="중랑장">중랑장</a> <a class="new" href="/w/index.php?title=%EC%9E%A5%ED%99%98&amp;action=edit&amp;redlink=1" title="장환 (없는 문서)">장환</a>(張奐) 밑에서 <a href="/wiki/%EC%82%AC%EB%A7%88_(%EB%B2%BC%EC%8A%AC)" title="사마 (벼슬)">사마</a>로 종군하여 <a href="/wiki/%ED%86%88%EC%88%98%EC%9D%B4_%EC%8B%9C" title="톈수이 시">한양</a>(漢陽, 천수군)에서 강족을 쳐부수었다. 이 공으로 낭중(郎中)이 되고 <a class="mw-redirect" href="/wiki/%EB%B9%84%EB%8B%A8" title="비단">비단</a> 9,000필을 받았다. 비단은 부하들에게 모두 나누어 주었다.</p>
<p><a class="mw-redirect" href="/wiki/%EC%9D%B4%EB%AF%BC%EC%A1%B1" title="이민족">이민족</a>으로부터 변방을 지키며 여러 공을 세워 광무 <a href="/wiki/%ED%98%84%EB%A0%B9" title="현령">현령</a>(廣武―), <a href="/wiki/%EC%B4%89%EA%B5%B0" title="촉군">촉군</a> 북부도위, <a href="/wiki/%EC%84%9C%EC%97%AD" title="서역">서역</a> 무기교위(西域戊己校尉), <a href="/wiki/%EB%B3%91%EC%A3%BC_(%EC%A4%91%EA%B5%AD)" title="병주 (중국)">병주</a> <a href="/wiki/%EC%9E%90%EC%82%AC_(%EA%B4%80%EC%A7%81)" title="자사 (관직)">자사</a>, <a href="/wiki/%ED%95%98%EB%8F%99%EA%B5%B0_(%EC%A4%91%EA%B5%AD)" title="하동군 (중국)">하동</a> <a href="/wiki/%ED%83%9C%EC%88%98" title="태수">태수</a> 등을 거쳤다. <a href="/wiki/184%EB%85%84" title="184년">184년</a>(<a href="/wiki/%EC%A4%91%ED%8F%89" title="중평">중평</a> 원년) <a href="/wiki/%ED%99%A9%EA%B1%B4%EC%A0%81" title="황건적">황건적</a>을 상대로 승전을 거듭하던 <a href="/wiki/%EB%85%B8%EC%8B%9D" title="노식">노식</a>이 좌풍(左豊)에게 <a href="/wiki/%EB%87%8C%EB%AC%BC" title="뇌물">뇌물</a>을 주지 않아서 소환당하고 그 대신, 동탁이 동중랑장(東中郞將)으로서 <a href="/wiki/%EC%9E%A5%EA%B0%81" title="장각">장각</a>을 공격했다가 패하여 벌을 받았다.</p>
<h3><span class="mw-headline" id=".EC.96.91.EC.A3.BC.EC.9D.98_.EB.82.9C">양주의 난</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%8F%99%ED%83%81&amp;action=edit&amp;section=3" title="부분 편집: 양주의 난">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p><a href="/wiki/%EA%B2%A8%EC%9A%B8" title="겨울">겨울</a> 북지군(北地郡), 농서군 부한현(枹罕縣)<sup class="reference" id="cite_ref-1"><a href="#cite_note-1">[1]</a></sup>·하관현(河關縣)의 <a class="mw-redirect" href="/wiki/%EC%84%A0%EB%A0%B9%EA%B0%95" title="선령강">선령강</a>(先零羌)과 <a class="mw-redirect" href="/wiki/%EC%86%8C%EC%9B%94%EC%A7%80" title="소월지">소월지</a><sup class="reference" id="cite_ref-2"><a href="#cite_note-2">[2]</a></sup>가 북궁백옥(北宮伯玉)과 이문후(李文侯)를 장군으로 추대하고 <a href="/wiki/%EC%96%91%EC%A3%BC%EC%9D%98_%EB%82%9C" title="양주의 난">난을 일으</a>켰다. <a href="/wiki/%ED%95%9C%EC%88%98_(%ED%9B%84%ED%95%9C)" title="한수 (후한)">한수</a>와 변장(邊章)은 위협을 받고 군정을 맡았다. <a href="/wiki/185%EB%85%84" title="185년">185년</a> <a href="/wiki/%EB%B4%84" title="봄">봄</a> <a class="mw-disambig" href="/wiki/%EC%82%BC%EB%B3%B4" title="삼보">삼보</a>(三輔)라 불리는 <a href="/wiki/%EA%B2%BD%EC%A1%B0%EC%9C%A4" title="경조윤">경조</a>, <a class="mw-redirect" href="/wiki/%ED%92%8D%EC%9D%B5" title="풍익">풍익</a>, <a class="mw-redirect" href="/wiki/%EB%B6%80%ED%92%8D" title="부풍">부풍</a>까지 <a href="/wiki/%EA%B8%B0%EB%B3%91" title="기병">기병</a> 수만 명으로 침범해오므로 <a class="mw-redirect" href="/wiki/%EC%A2%8C%EA%B1%B0%EA%B8%B0%EC%9E%A5%EA%B5%B0" title="좌거기장군">좌거기장군</a> <a href="/wiki/%ED%99%A9%EB%B3%B4%EC%88%AD" title="황보숭">황보숭</a>의 지휘 하에 그 진압을 맡았다. 7월(<a class="mw-redirect" href="/wiki/%EC%9D%8C%EB%A0%A5" title="음력">음력</a>)<sup class="reference" id="cite_ref-3"><a href="#cite_note-3">[3]</a></sup> 황보숭이 <a href="/wiki/%EC%9E%A5%EC%96%91_(%ED%9B%84%ED%95%9C)" title="장양 (후한)">장양</a>에게 뇌물을 주지 않았기 때문에<sup class="reference" id="cite_ref-.EB.B3.B4.EC.88.AD.EC.9D.B4_4-0"><a href="#cite_note-.EB.B3.B4.EC.88.AD.EC.9D.B4-4">[4]</a></sup> 면직되고 <a href="/wiki/%EC%9E%A5%EC%98%A8_(%ED%9B%84%ED%95%9C)" title="장온 (후한)">장온</a>이 부임하였다. 동탁은 파로장군(破虜―)으로 승진하였다.</p>
<p>고전하다가 11월(음력) 떨어진 <a class="mw-redirect" href="/wiki/%EB%B3%84%EB%98%A5%EB%B3%84" title="별똥별">별똥별</a>에 불안해하는 반군을 바로 공격하여 크게 깨트리고 수천 명을 베었다. 탕구장군(盪寇―) 주신(周愼)이 유중(楡中)으로 추격하였다. 동탁은 주신의 뒤에서 보급선을 지키려 했으나<sup class="reference" id="cite_ref-5"><a href="#cite_note-5">[5]</a></sup> 장온은 선령강 토벌을 명했다. 과연 보급선이 끊기고 주신은 패퇴하였다. 동탁은 수만 명의 강족에게 <a href="/wiki/%ED%8F%AC%EC%9C%84" title="포위">포위</a>당했지만 <a class="mw-redirect" href="/wiki/%EB%91%91" title="둑">둑</a>을 쌓은 후 <a href="/wiki/%EB%AC%BC%EA%B3%A0%EA%B8%B0" title="물고기">물고기</a>를 잡는 척하면서 그 아래로 자맥질해서 빠져나왔다. 그리고 둑을 무너트리니 강족은 <a href="/wiki/%EB%AC%BC" title="물">물</a>이 깊어 쫓아오지 못했다. 진압군이 모두 패퇴하는 와중에도 동탁군만이 전력을 보전한 채 돌아와 부풍에 주둔했으므로 태향후(斄鄕侯)에 봉해지고 <a href="/wiki/%EC%8B%9D%EC%9D%8D" title="식읍">식읍</a> 1,000호를 받았다.</p>
<p><a href="/wiki/186%EB%85%84" title="186년">186년</a> 겨울 한수가 북궁백옥, 이문후, 변장을 죽이고 10여만 명으로 늘어나자 <a href="/wiki/187%EB%85%84" title="187년">187년</a><sup class="reference" id="cite_ref-6"><a href="#cite_note-6">[6]</a></sup> 이상(李相), <a href="/wiki/%EB%A7%88%EB%93%B1" title="마등">마등</a>, <a href="/wiki/%EC%99%95%EA%B5%AD_(%ED%9B%84%ED%95%9C)" title="왕국 (후한)">왕국</a>마저 합세하였다. <a href="/wiki/188%EB%85%84" title="188년">188년</a> 왕국 등이 진창(陳倉)을 포위하자 <a class="mw-redirect" href="/wiki/%EC%A0%84%EC%9E%A5%EA%B5%B0" title="전장군">전장군</a>이 되어 황보숭과 함께 구원하러 갔다. 동탁은 진창으로 바로 나아가야 한다고 했지만 황보숭은 진창이 충분히 버틸 수 있다며 반군이 힘이 빠지기를 기다렸다. <a href="/wiki/189%EB%85%84" title="189년">189년</a> 2월(음력)<sup class="reference" id="cite_ref-.EC.98.81.EC.9E.AC_7-0"><a href="#cite_note-.EC.98.81.EC.9E.AC-7">[7]</a></sup> 왕국 등이 스스로 군을 물리자 황보숭이 때를 놓치지 않고 추격하려 하였다. 동탁이 이번에는 궁지에 몰린 적을 치는 것은 위험하다며 반대하자 그의 부대를 후방에 남겨둔 채 황보숭만이 진격하여 만여 명을 베고 반군을 소탕하였다. 동탁은 몹시 부끄럽고 한스러워 황보숭을 미워하였다.<sup class="reference" id="cite_ref-.EB.B3.B4.EC.88.AD.EC.9D.B4_4-1"><a href="#cite_note-.EB.B3.B4.EC.88.AD.EC.9D.B4-4">[4]</a></sup></p>
<h3><span class="mw-headline" id=".EB.82.99.EC.96.91_.EC.9E.85.EC.84.B1">낙양 입성</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%8F%99%ED%83%81&amp;action=edit&amp;section=4" title="부분 편집: 낙양 입성">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>중앙에서 소부(少府)로 삼아 불렀는데도 ‘호인(胡人) 병사들의 형편이 처참하여 자신을 가지 못 하게 하니 어쩔 수 없이 위무하겠다’며 취임을 거부했다. 조정에서는 어찌 하지 못하다가 이번에는 병<a href="/wiki/%EC%9E%90%EC%82%AC_(%EA%B4%80%EC%A7%81)" title="자사 (관직)">주목</a>에 임명하고 기존의 병력은 <a href="/wiki/%ED%99%A9%EB%B3%B4%EC%88%AD" title="황보숭">황보숭</a>이 맡도록 하였다. 동탁은 ‘이들과 함께 한 지 10년이어서 <a class="mw-redirect" href="/wiki/%EB%AA%A9%EC%88%A8" title="목숨">목숨</a>을 바칠 정도로 우애가 깊다’며 또 거절하였다.<sup class="reference" id="cite_ref-8"><a href="#cite_note-8">[8]</a></sup></p>
<p>4월(음력)<sup class="reference" id="cite_ref-.EC.98.81.EC.9E.AC_7-1"><a href="#cite_note-.EC.98.81.EC.9E.AC-7">[7]</a></sup> <a class="mw-redirect" href="/wiki/%EC%98%81%EC%A0%9C" title="영제">영제</a>가 죽은 후 <a href="/wiki/%EB%8C%80%EC%9E%A5%EA%B5%B0" title="대장군">대장군</a> <a href="/wiki/%ED%95%98%EC%A7%84" title="하진">하진</a>과 <a href="/wiki/%EC%9B%90%EC%86%8C_(%ED%9B%84%ED%95%9C)" title="원소 (후한)">원소</a>가 <a href="/wiki/%ED%99%98%EA%B4%80" title="환관">환관</a>들을 제거하려 했는데 <a class="mw-redirect" href="/wiki/%ED%95%98%ED%83%9C%ED%9B%84" title="하태후">하태후</a>가 반대하였다. 이에 여러 호걸들과 그 군대를 불러와 <a href="/wiki/%ED%83%9C%ED%9B%84" title="태후">태후</a>를 <a href="/wiki/%ED%98%91%EB%B0%95" title="협박">협박</a>할 계획을 세우고<sup class="reference" id="cite_ref-.EB.B0.B1.EC.A0.95_9-0"><a href="#cite_note-.EB.B0.B1.EC.A0.95-9">[9]</a></sup> <a href="/wiki/%EC%A0%95%ED%83%9C" title="정태">정태</a>의 반대에도 불구하고<sup class="reference" id="cite_ref-.EC.A0.95.ED.83.9C_10-0"><a href="#cite_note-.EC.A0.95.ED.83.9C-10">[10]</a></sup> 동탁까지 불렀다. 동탁이 미처 <a class="mw-redirect" href="/wiki/%EB%82%99%EC%96%91" title="낙양">낙양</a>에 도착하기 전에 하진은 <a href="/wiki/%EC%95%94%EC%82%B4" title="암살">암살</a>되고 호분중랑장(虎賁―) <a href="/wiki/%EC%9B%90%EC%88%A0" title="원술">원술</a>과 <a class="mw-redirect" href="/wiki/%EC%82%AC%EB%A1%80%EA%B5%90%EC%9C%84" title="사례교위">사례교위</a> 원소가 <a href="/wiki/%EC%8B%AD%EC%83%81%EC%8B%9C%EC%9D%98_%EB%82%9C" title="십상시의 난">환관들을 몰살</a>하였다. 환관 장양과 <a href="/wiki/%EB%8B%A8%EA%B7%9C" title="단규">단규</a> 등은 <a href="/wiki/%ED%9B%84%ED%95%9C_%EC%86%8C%EC%A0%9C_(13%EB%8C%80)" title="후한 소제 (13대)">소제</a>와 진류왕 유협(훗날 <a class="mw-redirect" href="/wiki/%ED%97%8C%EC%A0%9C" title="헌제">헌제</a>)을 데리고 소평진(小平津)으로 도주하므로 <sup class="reference" id="cite_ref-11"><a href="#cite_note-11">[11]</a></sup> <a href="/wiki/%EC%83%81%EC%84%9C" title="상서">상서</a> <a href="/wiki/%EB%85%B8%EC%8B%9D" title="노식">노식</a>과 <a href="/wiki/%ED%95%98%EB%82%A8%EA%B5%B0" title="하남군">하남</a>중부연(河南中部掾) <a href="/wiki/%EB%AF%BC%EA%B3%B5_(%ED%9B%84%ED%95%9C)" title="민공 (후한)">민공</a>이 좇아가 죽였다.<sup class="reference" id="cite_ref-.EB.B0.B1.EC.A0.95_9-1"><a href="#cite_note-.EB.B0.B1.EC.A0.95-9">[9]</a></sup> 동탁이 <a class="new" href="/w/index.php?title=%EB%B6%81%EB%A7%9D%EC%82%B0&amp;action=edit&amp;redlink=1" title="북망산 (없는 문서)">북망산</a>에서 황제 일행을 영접하여 낙양으로 들어왔다.</p>
<h3><span class="mw-headline" id=".EC.A0.95.EA.B6.8C_.EC.9E.A5.EC.95.85.EA.B3.BC_.ED.9A.A1.ED.8F.AC">정권 장악과 횡포</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%8F%99%ED%83%81&amp;action=edit&amp;section=5" title="부분 편집: 정권 장악과 횡포">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>원래 동탁의 병사는 불과 3,000명뿐이었지만 사오일 간 <a href="/wiki/%EB%B0%A4_(%EC%8B%9C%EA%B0%84)" title="밤 (시간)">밤</a>에는 몰래 밖으로 군사를 보내고 다음날 <a href="/wiki/%EC%95%84%EC%B9%A8" title="아침">아침</a> <a href="/wiki/%EB%B6%81" title="북">북</a>을 울리며 입성시키기를 반복하여 대군인 양 꾸며서 위세를 보였다. 죽은 하진·<a href="/wiki/%ED%95%98%EB%AC%98" title="하묘">하묘</a> <a href="/wiki/%ED%98%95%EC%A0%9C" title="형제">형제</a>의 병력을 자연스레 거두었을 뿐더러 <a href="/wiki/%EC%97%AC%ED%8F%AC" title="여포">여포</a>를 부추겨 <a href="/wiki/%EC%A7%91%EA%B8%88%EC%98%A4" title="집금오">집금오</a> <a href="/wiki/%EC%A0%95%EC%9B%90_(%ED%9B%84%ED%95%9C)" title="정원 (후한)">정원</a>을 제거하고 그 군대까지 흡수하니 정말로 대군이 되었다. <a href="/wiki/%EA%B0%80%EB%AD%84" title="가뭄">가뭄</a>을 이유로 <a href="/wiki/%EC%82%AC%EA%B3%B5_(%EA%B4%80%EC%A7%81)" title="사공 (관직)">사공</a> <a href="/wiki/%EC%9C%A0%ED%99%8D_(%EC%9E%90%EA%B3%A0)" title="유홍 (자고)">유홍</a>을 면직시키고 대신 그 자리에 올랐다.</p>
<p>소제는 어리석은 데 반해 유협은 똑똑한데다 같은 <a href="/wiki/%EC%84%B1%EC%94%A8" title="성씨">성씨</a>인 <a href="/wiki/%EB%8F%99%ED%83%9C%ED%9B%84_(%ED%9B%84%ED%95%9C)" title="동태후 (후한)">동태후</a>가 길렀으므로 폐립할 뜻을 품었다. 먼저 원소에게 그 뜻을 내비치니 반대하기에 ‘유씨(劉氏)<sup class="reference" id="cite_ref-12"><a href="#cite_note-12">[12]</a></sup>는 남겨둘 수 없다’고 화냈다. 원소 역시 발끈하여 ‘천하에 강한 자가 어찌 동탁뿐이겠는가!’라고 큰소리는 쳤지만 이내 <a href="/wiki/%EA%B8%B0%EC%A3%BC" title="기주">기주</a>로 달아났다.<sup class="reference" id="cite_ref-13"><a href="#cite_note-13">[13]</a></sup></p>
<p>9월<sup class="reference" id="cite_ref-.EC.98.81.EC.9E.AC_7-2"><a href="#cite_note-.EC.98.81.EC.9E.AC-7">[7]</a></sup> 대신들과 폐립을 의논하였다. 오직 노식만이 반대했을 뿐 아무도 감히 나서지 못 했다. 소제는 폐위하여 홍농왕으로 강등시키고 유협을 제위에 앉혔으며<sup class="reference" id="cite_ref-14"><a href="#cite_note-14">[14]</a></sup> 하태후와 그 <a href="/wiki/%EC%96%B4%EB%A8%B8%EB%8B%88" title="어머니">어머니</a> 무양군(舞陽君)을 죽였다.<sup class="reference" id="cite_ref-15"><a href="#cite_note-15">[15]</a></sup><sup class="reference" id="cite_ref-.EC.99.95.EC.B0.AC_16-0"><a href="#cite_note-.EC.99.95.EC.B0.AC-16">[16]</a></sup> 이내 <a href="/wiki/%ED%83%9C%EC%9C%84" title="태위">태위</a>로 옮겼다가<sup class="reference" id="cite_ref-17"><a href="#cite_note-17">[17]</a></sup> 헌제를 옹립한 공이 있다 하여 스스로 <a href="/wiki/%EC%83%81%EA%B5%AD" title="상국">상국</a>에 오르고(<a href="/wiki/11%EC%9B%94_26%EC%9D%BC" title="11월 26일">11월 26일</a>)<sup class="reference" id="cite_ref-18"><a href="#cite_note-18">[18]</a></sup>자신은 미후(郿侯), 어머니는 지양군(池陽君)에 봉했다. <a href="/wiki/%ED%99%A9%EC%A0%9C" title="황제">황제</a>를 배알할 때 자신의 <a href="/wiki/%EC%9D%B4%EB%A6%84" title="이름">이름</a>을 부르지 않았으며[贊拜不名] <a class="mw-disambig" href="/wiki/%EC%B9%BC" title="칼">칼</a>을 찬 채 전각에 올랐다.[劍履上殿] 동탁과 그 휘하 군대는 <a href="/wiki/%EC%82%B4%EC%9D%B8" title="살인">살인</a>·<a href="/wiki/%EC%95%BD%ED%83%88" title="약탈">약탈</a>·겁탈·축재 등 온갖 전횡을 저질렀다.</p>
<h3><span class="mw-headline" id=".EB.B0.98.EB.8F.99.ED.83.81_.EC.97.B0.ED.95.A9.EA.B5.B0_.EA.B6.90.EA.B8.B0">반동탁 연합군 궐기</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%8F%99%ED%83%81&amp;action=edit&amp;section=6" title="부분 편집: 반동탁 연합군 궐기">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>환관들이 충성스럽고 어진 이들을 주살했던 것을 천하가 미워했으므로 뭇 <a href="/wiki/%EC%84%A0%EB%B9%84" title="선비">선비</a>들을 발탁하였다. <a class="mw-disambig" href="/wiki/%EC%A3%BC%EB%B9%84" title="주비">주비</a>는 <a class="mw-disambig" href="/wiki/%EC%9D%B4%EB%B6%80" title="이부">이부</a>상서, 오경(伍瓊)은 <a class="mw-disambig" href="/wiki/%EC%8B%9C%EC%A4%91" title="시중">시중</a>(侍中), 정태는 상서에 임명하고 <a href="/wiki/%ED%95%9C%EB%B3%B5_(%ED%9B%84%ED%95%9C)" title="한복 (후한)">한복</a>은 기주, <a href="/wiki/%EC%9C%A0%EB%8C%80_(%ED%9B%84%ED%95%9C)" title="유대 (후한)">유대</a>는 <a href="/wiki/%EC%97%B0%EC%A3%BC_(%EC%A4%91%EA%B5%AD)" title="연주 (중국)">연주</a>, <a href="/wiki/%EA%B3%B5%EC%A3%BC_(%ED%9B%84%ED%95%9C)" title="공주 (후한)">공주</a>는 <a href="/wiki/%EC%98%88%EC%A3%BC" title="예주">예주</a> <a href="/wiki/%EC%9E%90%EC%82%AC_(%EA%B4%80%EC%A7%81)" title="자사 (관직)">자사</a>로 각각 파견하였으며 장자(張咨)는 <a href="/wiki/%EB%82%A8%EC%96%91%EA%B5%B0_(%EC%A4%91%EA%B5%AD)" title="남양군 (중국)">남양</a><a href="/wiki/%ED%83%9C%EC%88%98" title="태수">태수</a>로 삼았다. 사실 이들은 모두 동탁을 몰아내려고 공조하고 있었고 마침내 <a href="/wiki/190%EB%85%84" title="190년">190년</a>(<a href="/wiki/%EC%B4%88%ED%8F%89" title="초평">초평</a> 원년) <a href="/wiki/%EB%B0%9C%ED%95%B4%EA%B5%B0" title="발해군">발해</a> 태수 <a href="/wiki/%EC%9B%90%EC%86%8C_(%ED%9B%84%ED%95%9C)" title="원소 (후한)">원소</a>를 맹주로<sup class="reference" id="cite_ref-19"><a href="#cite_note-19">[19]</a></sup> <a class="mw-redirect" href="/wiki/%EB%B0%98%EB%8F%99%ED%83%81_%EC%97%B0%ED%95%A9%EA%B5%B0" title="반동탁 연합군">반동탁 연합군</a>이 일어났다.</p>
<p>안 그래도 <a href="/wiki/%ED%99%A9%EA%B1%B4%EC%A0%81" title="황건적">황건적</a>이었던 곽태(郭太) 등의 백파적(白波賊)이 서하군(西河郡)에서 봉기하여 그 규모가 10여만 명이었는데 반동탁 연합군까지 일어나자 난감해진 동탁은 <a href="/wiki/%EC%9D%B4%EC%9C%A0_(%ED%9B%84%ED%95%9C)" title="이유 (후한)">이유</a>를 시켜서<sup class="reference" id="cite_ref-20"><a href="#cite_note-20">[20]</a></sup> 홍농왕을 독살하고(190년 1월 25일)<sup id="cite_ref-22"><a href="#cite_note-22">[22]</a></sup> 태위 <a href="/wiki/%ED%99%A9%EC%99%84" title="황완">황완</a>·<a href="/wiki/%EC%82%AC%EB%8F%84_(%EA%B4%80%EC%A7%81)" title="사도 (관직)">사도</a> <a href="/wiki/%EC%96%91%ED%91%9C" title="양표">양표</a>의 반대를 내치면서 <a href="/wiki/%EC%9E%A5%EC%95%88" title="장안">장안</a> 천도를 강행했다.<sup class="reference" id="cite_ref-23"><a href="#cite_note-23">[23]</a></sup> <a class="mw-redirect" href="/wiki/%EB%82%99%EC%96%91" title="낙양">낙양</a> <a href="/wiki/%EC%82%AC%EB%9E%8C" title="사람">사람</a> 수백만 명을 군대로 몰아치며 이주시키니 짓밟히거나 굶어 죽은 <a href="/wiki/%EC%8B%9C%EC%B2%B4" title="리">리</a> 내에는 남은 것이 아무것도 없었으며 여포를 통해 황제와 대신들의 <a class="mw-redirect" href="/wiki/%EB%AC%98" title="묘">묘</a>까지 <a href="/wiki/%EB%8F%84%EA%B5%B4" title="도굴">도굴</a>하였다.<sup class="reference" 남아있던 원소 일가를 멸족하였다.<sup class="reference" id="cite_ref-26"><a href="#cite_note-26">[26]</a></sup><sup class="reference" id="cite_ref-27"><a href="#cite_note-27">[27]</a></sup></p>
<p>6월(음력) 한융(韓融)·<a href="/wiki/%EC%9D%8C%EC%88%98_(%ED%9B%84%ED%95%9C)" title="음수 (후한)">음수</a>·<a href="/wiki/%ED%98%B8%EB%AA%A8%EB%B0%98" title="호모반">호모반</a>·오수(吳脩)·왕괴(王瓌)를 보내 진정시키려 했으나 한융을 제외하고 모두 잡혀 죽었다.<sup class="reference" id="cite_ref-28"><a href="#cite_note-28">[28]</a></sup> 이러는 중에 <a href="/wiki/%EC%98%A4%EC%88%98%EC%A0%84" title="오수전">오수전</a>을 녹이고 대신 소전(小錢)으로 <a href="/wiki/%ED%99%94%ED%8F%90_%EA%B0%9C%ED%98%81" title="화폐 개혁">화폐 개혁</a>을 실시했는데<sup class="reference" id="cite_ref-29"><a href="#cite_note-29">[29]</a></sup> 조악한데다 너무 많이 만들어서 <a href="/wiki/%EC%B4%88%EC%9D%B8%ED%94%8C%EB%A0%88%EC%9D%B4%EC%85%98" title="초인플레이션">초인플레이션</a>이 일어나 <a class="mw-redirect" href="/wiki/%EA%B3%A1%EC%8B%9D" title="곡식">곡식</a> 한 곡(斛)이 수십만 전에 이를 정도로 실패하였다.</p>
<h3><span class="mw-headline" id=".EC.86.8C.EB.93.9D_.EC.97.86.EB.8A.94_.EC.97.B0.ED.95.A9.EA.B5.B0">소득 없는 연합군</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%8F%99%ED%83%81&amp;action=edit&amp;section=7" title="부분 편집: 소득 없는 연합군">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p><a href="/wiki/%ED%95%98%EB%82%B4%EA%B5%B0" title="하내군">하내</a> 태수 <a class="mw-disambig" href="/wiki/%EC%99%95%EA%B4%91" title="왕광">왕광</a>이 <a href="/wiki/%ED%83%9C%EC%82%B0%EA%B5%B0" title="태산군">태산</a>병을 하양진(河陽津)에 주둔시키고 장차 남쪽으로 넘어오려 하기에 평음진(平陰津)에서 <a href="/wiki/%ED%99%A9%ED%95%98" title="황하">황하</a>를 건널 것처럼 속이고 몰래 소평진에서 북상해서 그 배후를 둘러 공격해 거의 다 죽였다. <a href="/wiki/%EC%A1%B0%EC%A1%B0" title="조조">조조</a>는 <a class="mw-redirect" href="/wiki/%EC%A0%9C%EB%B6%81" title="제북">제북</a>상 <a href="/wiki/%ED%8F%AC%EC%8B%A0" title="포신">포신</a>,<sup class="reference" id="cite_ref-30"><a href="#cite_note-30">[30]</a></sup> <a class="mw-redirect" href="/wiki/%EC%A7%84%EB%A5%98" title="진류">진류</a> 태수 <a href="/wiki/%EC%9E%A5%EB%A7%89_(%ED%9B%84%ED%95%9C)" title="장막 (후한)">장막</a>의 부하 <a class="mw-disambig" href="/wiki/%EC%9C%84%EC%9E%90" title="위자">위자</a>와 함께 <a href="/wiki/%EC%8B%B1%EC%96%91_%EC%8B%9C" title="싱양 시">형양</a>(滎陽) 방면으로 오므로 <a href="/wiki/%EC%84%9C%EC%98%81_(%ED%9B%84%ED%95%9C)" title="서영 (후한)">서영</a>을 보내 무찔렀다.<sup class="reference" id="cite_ref-31"><a href="#cite_note-31">[31]</a></sup> 서영과 <a href="/wiki/%EC%9D%B4%EB%AA%BD" title="이몽">이몽</a>을 네 번 출전시켜 일대를 노략질하고 원소의 예주 종사(―從事) <a href="/wiki/%EC%9D%B4%EC%97%B0_(%ED%9B%84%ED%95%9C)" title="이연 (후한)">이연</a>을 삶아 죽였다.<sup class="reference" id="cite_ref-.ED.97.8C.EC.A0.9C.EA.B8.B0_32-0"><a href="#cite_note-.ED.97.8C.EC.A0.9C.EA.B8.B0-32">[32]</a></sup> 겨울에는 노양(魯陽)으로부터<sup class="reference" id="cite_ref-.EA.BF.80.EB.AC.BC_.EC.85.94.ED.8B.80_33-0"><a href="#cite_note-.EA.BF.80.EB.AC.BC_.EC.85.94.ED.8B.80-33">[33]</a></sup> 올라오던 <a href="/wiki/%EC%9E%A5%EC%82%AC%EA%B5%B0" title="장사군">장사</a> 태수 <a href="/wiki/%EC%86%90%EA%B2%AC" title="손견">손견</a>까지 양(梁, 지금의 <a class="mw-redirect" href="/wiki/%ED%97%88%EB%82%9C%EC%84%B1" title="허난성">허난성</a> <a class="mw-redirect" href="/wiki/%EB%A3%A8%EC%A0%80%EC%9A%B0%EC%8B%9C" title="루저우시">루저우시</a>)에서 서영이 대파함으로써 세 방향 모두 성공적으로 방어하였다.</p>
<p><a href="/wiki/191%EB%85%84" title="191년">191년</a> 음력 2월<sup class="reference" id="cite_ref-34"><a href="#cite_note-34">[34]</a></sup> 손견이 흩어진 군사를 수습하고 다시 양인(陽人)으로 진격해오자 <a href="/wiki/%ED%98%B8%EC%A7%84" title="호진">호진</a>을 대독(大督), <a href="/wiki/%EC%97%AC%ED%8F%AC" title="여포">여포</a>를 기독(騎督)으로 삼아 요격하였다.<sup class="reference" id="cite_ref-35"><a href="#cite_note-35">[35]</a></sup> 호진과 여포는 서로 사이가 좋지 않아 내분으로 패하였고 <a href="/wiki/%EC%9D%B4%EA%B0%81_(%ED%9B%84%ED%95%9C)" title="이각 (후한)">이각</a>을 보내 화친을 시도했으나 결렬되었다. 손견이 낙양 가까이까지 진군해오므로 친히 맞붙었지만 패하여 낙양 입성을 허용하고 말았다. 다만 낙양은 이미 황폐해져 보급이 어려웠기에<sup class="reference" id="cite_ref-36"><a href="#cite_note-36">[36]</a></sup> 손견은 곧 노양으로 돌아갔다.<sup class="reference" id="cite_ref-.EA.BF.80.EB.AC.BC_.EC.85.94.ED.8B.80_33-1"><a href="#cite_note-.EA.BF.80.EB.AC.BC_.EC.85.94.ED.8B.80-33">[33]</a></sup> <a href="/wiki/%EC%A3%BC%EC%A4%80" title="주준">주준</a> 역시 한때 낙양에 입성했었으나 이내 <a href="/wiki/%EC%A4%91%EB%AC%B4_%ED%98%84" title="중무 현">중모</a>(中牟)에 주둔하였다. 동탁은 중랑장 동월(董越), <a href="/wiki/%EB%8B%A8%EC%99%B8" title="단외">단외</a>, <a class="mw-disambig" href="/wiki/%EC%9A%B0%EB%B3%B4" title="우보">우보</a>를 각각 면지(黽池), <a href="/wiki/%ED%99%94%EC%9D%B8_%EC%8B%9C" title="화인 시">화음</a>(華陰), 안읍(安邑)에 배치하여 대치시키고 장안으로 들어갔다.</p>
<h3><span class="mw-headline" id=".EC.97.AC.ED.8F.AC_.EB.84.88.EB.A7.88.EC.A0.80">여포 너마저</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%8F%99%ED%83%81&amp;action=edit&amp;section=8" title="부분 편집: 여포 너마저">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p><a href="/wiki/%EC%9E%A5%EC%95%88" title="장안">장안</a>으로 들어오기 전 태사(太師)라 칭했다.(190년 3월 25일(음력 2월 12일, 정축(丁丑)일))<sup class="reference" id="cite_ref-37"><a href="#cite_note-37">[37]</a></sup> <a href="/wiki/%ED%99%A9%ED%83%9C%EC%9E%90" title="황태자">황태자</a> 급의 <a href="/wiki/%EC%88%98%EB%A0%88" title="수레">수레</a><sup class="reference" id="cite_ref-38"><a href="#cite_note-38">[38]</a></sup>와 복식을 사용하였고, <a class="mw-redirect" href="/wiki/%EB%8F%99%EC%83%9D" title="동생">동생</a> 동민(董旻)에게는 <a class="mw-redirect" href="/wiki/%EC%A2%8C%EC%9E%A5%EA%B5%B0" title="좌장군">좌장군</a>과 호후(鄠侯), 큰조카 동황(董璜)에게는 시중과 중군<a class="new" href="/w/index.php?title=%EA%B5%90%EC%9C%84&amp;action=edit&amp;redlink=1" title="교위 (없는 문서)">교위</a>를 주는 등 동탁의 가문 사람은 <a class="mw-redirect" href="/wiki/%EC%96%B4%EB%A6%B0%EC%95%A0" title="어린애">어린애</a>라도 모두 한 자리씩 가졌다. <a class="mw-redirect" href="/wiki/%EB%AF%B8%ED%98%84" title="미현">미현</a>(郿縣)에 오(塢)를 쌓고 만세오(萬歲塢)라 하였다. <a href="/wiki/%EB%86%92%EC%9D%B4" title="높이">높이</a>가 높은 곳은 7장(丈)에 이르렀으며 30년 치 <a class="mw-redirect" href="/wiki/%EC%8B%9D%EB%9F%89" title="식량">식량</a>을 비축했으므로 동탁 스스로 “일이 잘되면 천하를 웅거하고 잘 안되면 이곳을 지키며 여생을 보내겠다”고 자신했다. 법령이 가혹하고 문란하여 억울하게 죽은 자가 많았다. <a href="/wiki/%EA%B4%80%EC%A4%91_(%EC%A7%80%EB%AA%85)" title="관중 (지명)">관중</a>의 귀족들은 반역죄를 뒤집어쓰기도 했다.</p>
<p>동탁은 여포를 매우 신임해서 <a class="mw-redirect" href="/wiki/%EC%96%91%EC%95%84%EB%93%A4" title="양아들">양아들</a>로 삼고 <a href="/wiki/%EA%B2%BD%ED%98%B8" title="경호">경호</a>를 맡겼었다. 어느날 사소한 일로 화가 나서 여포에게 수극을 던졌다. 민첩하게 피했기에 망정이지 죽을 뻔한 여포는 이 일이 앙금으로 남았다. 거기다 동탁의 시중드는 계집종과 몰래 정을 통한 일도 있어 발각될까 두려웠다. <a href="/wiki/%EC%99%95%EC%9C%A4" title="왕윤">왕윤</a>은 <a href="/wiki/%EC%83%81%EC%84%9C%EB%B3%B5%EC%95%BC" title="상서복야">상서복야</a> <a href="/wiki/%EC%82%AC%EC%86%90%EC%84%9C" title="사손서">사손서</a>와 동탁 주살을 모의하고 있었는데 여포가 수극에 죽을 뻔한 일을 듣고는 여포도 회유하였다. 여포는 처음에는 부자 사이에 어떻게 그럴 수 있냐고 펄쩍 뛰었지만 왕윤이 ‘친부도 아닐 뿐더러 <a href="/wiki/%EC%95%84%EB%B2%84%EC%A7%80" title="아버지">아버지</a>가 <a href="/wiki/%EC%95%84%EB%93%A4" title="아들">아들</a>에게 수극을 던지겠느냐’고 설득하니 결국 동참하였다.<sup class="reference" id="cite_ref-39"><a href="#cite_note-39">[39]</a></sup></p>
<p>192년 4월 동탁이 여포 등 호위병을 거느리고 궁궐로 들어가는데 <a href="/wiki/%EA%B8%B0%EB%8F%84%EC%9C%84" title="기도위">기도위</a> <a href="/wiki/%EC%9D%B4%EC%88%99_(%EC%82%BC%EA%B5%AD%EC%A7%80)" title="이숙 (삼국지)">이숙</a> 등 10여 명의 습격을 받았다. 동탁은 속에 <a href="/wiki/%EA%B0%91%EC%98%B7" title="갑옷">갑옷</a>을 입은지라 <a href="/wiki/%ED%8C%94" title="팔">팔</a>에만 부상을 입고 수레에서 떨어졌다. 큰 <a href="/wiki/%EC%86%8C%EB%A6%AC" title="소리">소리</a>로 여포를 찾자 여포는 “조서를 받들어 역적 동탁을 치노라”고 하였고, 동탁은 “개새끼, 감히 이럴 수 있느냐!”<sup class="reference" id="cite_ref-40"><a href="#cite_note-40">[40]</a></sup>란 마지막 말을 남기고 죽었다.(5월 22일, 음력 4월 23일, 신사(辛巳)일) 이를 만방에 알리니 사졸들은 <a href="/wiki/%EB%A7%8C%EC%84%B8" title="만세">만세</a>를 부르고 사람들은 거리로 뛰쳐나와 <a href="/wiki/%EB%85%B8%EB%9E%98" title="노래">노래</a>하고 <a href="/wiki/%EC%B6%A4" title="춤">춤</a>추며 <a href="/wiki/%EC%9E%94%EC%B9%98" title="잔치">잔치</a>를 벌였다. 미오는 <a href="/wiki/%ED%99%A9%EB%B3%B4%EC%88%AD" title="황보숭">황보숭</a>이 점령하였으며 동탁의 시체는 거리에 내버려지고 그 일족은 멸족되었다.<sup class="reference" id="cite_ref-41"><a href="#cite_note-41">[41]</a></sup> 동탁은 <a href="/wiki/%EA%B8%88" title="금">금</a> 이삼만 근, <a href="/wiki/%EC%9D%80" title="은">은</a> 팔구만 근, 비단과 <a href="/wiki/%EA%B5%AC%EC%8A%AC" title="구슬">구슬</a> 등 온갖 보물을 <a href="/wiki/%EC%82%B0" title="산">산</a>처럼 모아 놨었다.</p>
<h2><span class="mw-headline" id=".ED.8A.B9.EC.A7.95.EA.B3.BC_.EC.84.B1.ED.92.88">특징과 성품</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%8F%99%ED%83%81&amp;action=edit&amp;section=9" title="부분 편집: 특징과 성품">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>거칠고 사나웠으며 지략이 있었다. <a class="mw-redirect" href="/wiki/%EA%B0%95%EC%A1%B1" title="강족">강족</a>들도 경외할 정도로 힘이 세고 무예가 뛰어나서 동개 두 개를 차고 <a href="/wiki/%EB%A7%90_(%EB%8F%99%EB%AC%BC)" title="말 (동물)">말</a>을 몰면서 좌우로 <a href="/wiki/%ED%99%9C" title="활">활</a>을 쏠 수 있었다. 오부(伍孚)가 동탁의 <a href="/wiki/%EC%95%94%EC%82%B4" title="암살">암살</a>을 시도했을 때도 힘이 세서 실패한 바 있다.<sup class="reference" id="cite_ref-42"><a href="#cite_note-42">[42]</a></sup> 한편으로 매우 뚱뚱했다. 동탁의 <a href="/wiki/%EC%8B%9C%EC%B2%B4" title="시체">시체</a>에서 <a href="/wiki/%EA%B8%B0%EB%A6%84" title="기름">기름</a>이 흘러 이를 지키는 관리가 그 <a href="/wiki/%EB%B0%B0%EA%BC%BD" title="배꼽">배꼽</a> 위에 심지를 놓고 <a href="/wiki/%EB%B6%88" title="불">불</a>을 붙였는데 며칠씩이나 계속 탔다.<sup class="reference" id="cite_ref-.EC.99.95.EC.B0.AC_16-1"><a href="#cite_note-.EC.99.95.EC.B0.AC-16">[16]</a></sup></p>
<p>흉악하고 방자한데다 잔인했다. 한번은 양성(陽城)에 군대를 보낸 일이 있었는데 마침 <a href="/wiki/%EB%B0%B1%EC%84%B1" title="백성">백성</a>들이 지신제를 지내고 있기에 모두 죽이고 재물과 부녀자를 취하고는 <a class="mw-redirect" href="/wiki/%EB%8F%84%EC%A0%81" title="도적">도적</a>을 격파했다며 만세를 불렀다. 장안에서는 공경대신들에게 연회를 베풀었는데 북지에서 <a href="/wiki/%EB%B0%98%EB%9E%80" title="반란">반란</a>을 일으켰다가 <a href="/wiki/%ED%95%AD%EB%B3%B5" title="항복">항복</a>한 수백 명을 그 자리에서 죽였다. 먼저 <a href="/wiki/%ED%98%80" title="혀">혀</a>를 자른 다음 <a href="/wiki/%EC%86%90" title="손">손</a><a href="/wiki/%EB%B0%9C" title="발">발</a>을 베고 <a class="mw-redirect" href="/wiki/%EB%88%88%EC%95%8C" title="눈알">눈알</a>마저 파낸 후에 삶았으며 미처 죽지 않은 자들은 주안상 사이로 굴러다녔다. 모두가 두려움에 벌벌 떨며 <a href="/wiki/%EC%88%98%EC%A0%80" title="수저">수저</a>를 들지 못 하는데 동탁만이 태연자약하게 먹고 마셨다.</p>
<p>억세고 편협하며 화가 나면 뒷일을 생각하지 않았다.<sup class="reference" id="cite_ref-43"><a href="#cite_note-43">[43]</a></sup> 동탁이 아끼는 호인 부하가 이를 믿고 제멋대로 행동하다가 사례교위 조겸(趙謙)에게 죽었다. 동탁이 대노하여 “내가 <a href="/wiki/%EC%82%AC%EB%9E%91" title="사랑">사랑</a>하는 <a href="/wiki/%EA%B0%9C" title="개">개</a>도 꾸짖지 못하게 하는데 하물며 사람은 아닐쏘냐!”라고 하며 <a class="mw-redirect" href="/wiki/%EC%82%AC%EB%A1%80" title="사례">사례</a>도관(―都官)을 죽였다.<sup class="reference" id="cite_ref-.ED.97.8C.EC.A0.9C.EA.B8.B0_32-1"><a href="#cite_note-.ED.97.8C.EC.A0.9C.EA.B8.B0-32">[32]</a></sup> <a href="/wiki/%EC%9C%84%EC%9C%84" title="위위">위위</a> <a href="/wiki/%EC%9E%A5%EC%98%A8_(%ED%9B%84%ED%95%9C)" title="장온 (후한)">장온</a>은 개인적인 감정 때문에 <a class="mw-redirect" href="/wiki/%ED%83%9C%ED%98%95" title="태형">태형</a>을 받고 목숨을 잃었으며 동탁을 노려보기만 해도 저세상으로 갔다. 호위를 맡긴 여포에게까지 사소한 이유로 수극을 던질 정도였고 결국 이것이 화가 되었다.</p>
<h2><span class="mw-headline" id=".ED.8F.89.EA.B0.80">평가</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%8F%99%ED%83%81&amp;action=edit&amp;section=10" title="부분 편집: 평가">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p><a href="/wiki/%EC%86%90%EA%B2%AC" title="손견">손견</a>은 장온의 호출에 늦게 온 동탁이 그 <a href="/wiki/%ED%83%9C%EB%8F%84" title="태도">태도</a>까지 불손하다며 군율에 따라 <a href="/wiki/%EC%B0%B8%EC%88%98" title="참수">참수</a>할 것을 진언한 적이 있다. 장온은 ‘동탁이 농촉(隴蜀)에서 위세와 명성을 떨치고 있으므로 전투에 꼭 필요하다’며 받아들이지 않았다.<sup class="reference" id="cite_ref-44"><a href="#cite_note-44">[44]</a></sup> <a href="/wiki/%ED%95%98%EC%A7%84" title="하진">하진</a>이 동탁을 부를 때 노식은 ‘흉악하고 사나워 제어가 어려울 것’이라고<sup class="reference" id="cite_ref-.EB.85.B8.EC.8B.9D_45-0"><a href="#cite_note-.EB.85.B8.EC.8B.9D-45">[45]</a></sup> <a href="/wiki/%EC%A0%95%ED%83%9C" title="정태">정태</a>는 ‘욕심이 끝이 없어서 조정이 위태로울 것’이라며 반대하였다.<sup class="reference" id="cite_ref-.EC.A0.95.ED.83.9C_10-1"><a href="#cite_note-.EC.A0.95.ED.83.9C-10">[10]</a></sup> <a href="/wiki/%ED%8F%AC%EC%8B%A0" title="포신">포신</a> 역시 낙양에 갓 입성한 동탁이 다른 뜻이 있으니 조속히 도모해야 하다고 <a href="/wiki/%EC%9B%90%EC%86%8C_(%ED%9B%84%ED%95%9C)" title="원소 (후한)">원소</a>에게 강권하였다.</p>
<p>동탁이 정권을 장악한 후 <a href="/wiki/%EC%88%9C%EC%9C%A0" title="순유">순유</a>는 정태, <a href="/wiki/%ED%95%98%EC%98%B9" title="하옹">하옹</a>, <a href="/wiki/%EC%B6%A9%EC%A7%91" title="충집">충집</a>, 오경과 그 암살을 모의하며 ‘동탁의 무도함은 <a class="mw-redirect" href="/wiki/%EA%B1%B8%EC%99%95" title="걸왕">걸왕</a>, <a class="mw-redirect" href="/wiki/%EC%A3%BC%EC%99%95" title="주왕">주왕</a>보다도 심하니 그를 죽이는 것은 <a href="/wiki/%EC%A0%9C_%ED%99%98%EA%B3%B5" title="제 환공">제 환공</a>이나 <a href="/wiki/%EC%A7%84_%EB%AC%B8%EA%B3%B5" title="진 문공">진 문공</a>의 의거와 같다.’고 하였다.<sup class="reference" id="cite_ref-46"><a href="#cite_note-46">[46]</a></sup> <a class="mw-redirect" href="/wiki/%EC%88%9C%EC%9A%B1_(%ED%9B%84%ED%95%9C)" title="순욱 (후한)">순욱</a>은 191년 <a href="/wiki/%EC%A1%B0%EC%A1%B0" title="조조">조조</a>에게 임관하며 “그 포학함이 지나치게 심하니 필히 변을 당해 망할 것입니다.”라 평하였고<sup class="reference" id="cite_ref-47"><a href="#cite_note-47">[47]</a></sup> 그 말대로 수회 암살 시도가 있었고 결국 <a href="/wiki/%EC%99%95%EC%9C%A4" title="왕윤">왕윤</a>, <a href="/wiki/%EC%97%AC%ED%8F%AC" title="여포">여포</a> 등에게 죽었다.</p>
<p>《<a href="/wiki/%EC%82%BC%EA%B5%AD%EC%A7%80" title="삼국지">삼국지</a>》의 <a class="mw-redirect" href="/wiki/%EC%A0%80%EC%9E%90" title="저자">저자</a> <a href="/wiki/%EC%A7%84%EC%88%98_(%EC%84%9C%EC%A7%84)" title="진수 (서진)">진수</a>는 ‘거칠고 포악하며 사악한데다 잔인하다. <a href="/wiki/%EC%97%AD%EC%82%AC" title="역사">역사</a>를 기록한 이래 이와 같은 자는 없었다.’고 평하였다.<sup class="reference" id="cite_ref-48"><a href="#cite_note-48">[48]</a></sup> 《<a href="/wiki/%ED%9B%84%ED%95%9C%EC%84%9C" title="후한서">후한서</a>》의 저자 <a href="/wiki/%EB%B2%94%EC%97%BD" title="범엽">범엽</a>은 ‘원래 <a href="/wiki/%ED%98%B8%EB%9E%91%EC%9D%B4" title="호랑이">호랑이</a> 같은 성깔로 우연히 난세를 만나 온갖 악행을 저지르면서도 대신들과 정사를 타협하고 찬탈도 주저했으니 어찌 보면 대도라 하겠다.<sup class="reference" id="cite_ref-49"><a href="#cite_note-49">[49]</a></sup>’고 평하였다. <a href="/wiki/%EC%99%95%EC%B0%AC" title="왕찬">왕찬</a>의 《<a href="/wiki/%EC%98%81%EC%9B%85%EA%B8%B0" title="영웅기">영웅기</a>》에서는 사람들이 ‘동탁이 있어서 큰 난리가 생겼고 그 난리가 동탁 자신까지 집어삼켰다’고 말했다고 한다.<sup class="reference" id="cite_ref-50"><a href="#cite_note-50">[50]</a></sup></p>
<p><a href="/wiki/%EB%8F%99%EC%95%84%EC%8B%9C%EC%95%84" title="동아시아">동아시아</a>에서 동탁은 <a href="/wiki/%EC%99%95%EB%A7%9D" title="왕망">왕망</a>, 조조 등과 함께 <a class="mw-redirect" href="/wiki/%EB%82%98%EB%9D%BC" title="나라">나라</a>를 도탄에 빠트린 역적의 대명사로 취급되었다.<sup class="reference" id="cite_ref-51"><a href="#cite_note-51">[51]</a></sup></p>
<h2><span class="mw-headline" id=".EC.82.BC.EA.B5.AD.EC.A7.80.EC.97.B0.EC.9D.98">삼국지연의</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%8F%99%ED%83%81&amp;action=edit&amp;section=11" title="부분 편집: 삼국지연의">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>사서가 아닌 <a href="/wiki/%EC%86%8C%EC%84%A4" title="소설">소설</a> 《<a href="/wiki/%EC%82%BC%EA%B5%AD%EC%A7%80%EC%97%B0%EC%9D%98" title="삼국지연의">삼국지연의</a>》에서도 악인으로 묘사한다. <a href="/wiki/%EC%9E%A5%EA%B0%81" title="장각">장각</a>과의 전투에서 자신을 구해준 <a href="/wiki/%EC%9C%A0%EB%B9%84" title="유비">유비</a> 삼형제를 업신여기는 것으로 첫등장한다. 조조가 칠보도(七寶刀)를 빌려 동탁 암살을 시도하다가 실패하는 장면을 추가하였다. 왕윤이 양녀 <a href="/wiki/%EC%B4%88%EC%84%A0" title="초선">초선</a>을 이용하여 동탁과 여포 사이를 이간질하는 것으로 연출하였으며 죽는 날 <a href="/wiki/%EC%84%A0%EC%96%91_(%EC%97%AD%EC%82%AC)" title="선양 (역사)">선양</a>받는 줄 알고 궁궐로 들어간다.</p>
