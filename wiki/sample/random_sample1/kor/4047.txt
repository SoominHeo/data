<p><b>동티모르 센타부</b>(<span style="font-size: smaller;"><a href="/wiki/%ED%85%8C%ED%88%BC%EC%96%B4" title="테툼어">테툼어</a>:</span> <span lang="tet" xml:lang="tet">Doit Timór-Leste nian</span>, <span style="font-size: smaller;"><a href="/wiki/%ED%8F%AC%EB%A5%B4%ED%88%AC%EA%B0%88%EC%96%B4" title="포르투갈어">포르투갈어</a>:</span> <span lang="pt" xml:lang="pt">Centavo do Timor-Leste</span>)는 <a href="/wiki/%EB%8F%99%ED%8B%B0%EB%AA%A8%EB%A5%B4" title="동티모르">동티모르</a>의 <a href="/wiki/%ED%86%B5%ED%99%94" title="통화">통화</a>이다. <a href="/wiki/2000%EB%85%84" title="2000년">2000년</a>에 <a class="mw-redirect" href="/wiki/UN" title="UN">UN</a>의 행정 통치를 받으며 <a href="/wiki/%EC%9D%B8%EB%8F%84%EB%84%A4%EC%8B%9C%EC%95%84_%EB%A3%A8%ED%94%BC%EC%95%84" title="인도네시아 루피아">인도네시아 루피아</a>를 대체하기 위해 <a href="/wiki/%EB%AF%B8%EA%B5%AD_%EB%8B%AC%EB%9F%AC" title="미국 달러">미국 달러</a>를 도입했으며, <a href="/wiki/2003%EB%85%84" title="2003년">2003년</a>부터 센타부를 <a href="/wiki/%EB%AF%B8%EA%B5%AD_%EB%8B%AC%EB%9F%AC" title="미국 달러">미국 달러</a>와 함께 쓰이기 시작했다. <a href="/wiki/%EC%A7%80%ED%8F%90" title="지폐">지폐</a>는 발행하고 있지 않으며, 1 센타부는 <a href="/wiki/%ED%8E%98%EB%8B%88_(%EB%AF%B8%EA%B5%AD_%EB%8F%99%EC%A0%84)" title="페니 (미국 동전)">1 센트</a>와 가치가 같다. 1, 5, 10, 25, 50 센타부 <a class="mw-redirect" href="/wiki/%EB%8F%99%EC%A0%84" title="동전">동전</a>이 있으며 각 동전의 뒷면에는 <a href="/wiki/%EB%8F%99%ED%8B%B0%EB%AA%A8%EB%A5%B4" title="동티모르">동티모르</a>의 동식물들이 있다.</p>
<p>센타부 <a class="mw-redirect" href="/wiki/%EB%8F%99%EC%A0%84" title="동전">동전</a>들은 <a href="/wiki/%ED%8F%AC%EB%A5%B4%ED%88%AC%EA%B0%88" title="포르투갈">포르투갈</a> <a href="/wiki/%EB%A6%AC%EC%8A%A4%EB%B3%B8" title="리스본">리스본</a>의 국립 조폐국에서 발행하고 있다.</p>
