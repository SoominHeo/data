<p><b>돛자리 감마</b>(Gamma Velorum)는 <a href="/wiki/%EB%8F%9B%EC%9E%90%EB%A6%AC" title="돛자리">돛자리</a>에 있는 840광년 떨어진 <a href="/wiki/%ED%95%AD%EC%84%B1%EA%B3%84" title="항성계">항성계</a>이다. 이 별의 <a class="mw-redirect" href="/wiki/%EA%B2%89%EB%B3%B4%EA%B8%B0_%EB%93%B1%EA%B8%89" title="겉보기 등급">겉보기 등급</a>은 +1.7로, <a href="/wiki/%EB%B0%A4%ED%95%98%EB%8A%98" title="밤하늘">밤하늘</a>에서 밝은 별들 중 하나이다. 이 별의 고유 명칭은 <b>수하일</b> 또는 <b>알 수하일 알 무리프</b>이다.(수하일은 <a href="/wiki/%EB%8F%9B%EC%9E%90%EB%A6%AC_%EB%9E%8C%EB%8B%A4" title="돛자리 람다">돛자리 람다</a>를 가리키는 명칭으로도 쓰이므로 사용자에게 혼란을 준다) 최근 붙은 이름으로 <b>레고르</b>(Regor)가 있는데, 이는 <a href="/wiki/%EC%95%84%ED%8F%B4%EB%A1%9C_1%ED%98%B8" title="아폴로 1호">아폴로 1호</a> 승무원이었던 <a class="new" href="/w/index.php?title=%EA%B1%B0%EC%8A%A4_%EA%B7%B8%EB%A6%AC%EC%84%AC&amp;action=edit&amp;redlink=1" title="거스 그리섬 (없는 문서)">거스 그리섬</a>이 동료 <a class="new" href="/w/index.php?title=%EB%A1%9C%EC%A0%80_%EC%B0%A8%ED%94%BC&amp;action=edit&amp;redlink=1" title="로저 차피 (없는 문서)">로저 차피</a>를 놀려주려는 의도에서 만들어진 단어이다. 이 별은 어두운 <a href="/wiki/%ED%9D%A1%EC%88%98%EC%84%A0" title="흡수선">흡수선</a>상에 밝은 <a class="mw-redirect" href="/wiki/%EB%B0%A9%EC%B6%9C%EC%84%A0" title="방출선">방출선</a>을 보여주기 때문에, '남반구 하늘의 스펙트럼 보석'으로도 불린다.<sup class="reference" id="cite_ref-1"><a href="#cite_note-1">[1]</a></sup></p>
<p>돛자리 감마는 최소 여섯 개의 별로 이루어진 <a class="mw-redirect" href="/wiki/%EB%8B%A4%EC%A4%91%EC%84%B1%EA%B3%84" title="다중성계">다중성계</a>이다. 가장 밝은 돛자리 감마<b>2</b> 또는 돛자리 감마 <b>A</b>는 <a class="mw-redirect" href="/wiki/%EB%B6%84%EA%B4%91%ED%98%95" title="분광형">분광형</a> O9의 <a class="mw-redirect" href="/wiki/%EC%B2%AD%EC%83%89_%EC%B4%88%EA%B1%B0%EC%84%B1" title="청색 초거성">청색 초거성</a>(<a href="/wiki/%ED%83%9C%EC%96%91%EC%A7%88%EB%9F%89" title="태양질량">태양질량</a>의 30배)과, <a class="mw-redirect" href="/wiki/%EC%9A%B8%ED%94%84-%EB%A0%88%EC%9D%B4%EC%97%90%EB%B3%84" title="울프-레이에별">울프-레이에별</a>(태양질량 10배, <a class="new" href="/w/index.php?title=%EC%A7%88%EB%9F%89_%EC%86%90%EC%8B%A4&amp;action=edit&amp;redlink=1" title="질량 손실 (없는 문서)">질량 손실</a>이 있기 전은 태양질량 40배로 추측)로 이루어져 있는 <a class="mw-redirect" href="/wiki/%EB%B6%84%EA%B4%91%EC%8C%8D%EC%84%B1" title="분광쌍성">분광쌍성</a>이다. 이 둘은 1 <a href="/wiki/%EC%B2%9C%EB%AC%B8%EB%8B%A8%EC%9C%84" title="천문단위">천문단위</a> 정도 떨어져 있으며 <a class="mw-redirect" href="/wiki/%EA%B3%B5%EC%A0%84%EC%A3%BC%EA%B8%B0" title="공전주기">공전주기</a>는 78.5일이다. 이들로부터 가장 가까운 동반성은 돛자리 감마<b>1</b> 또는 돛자리 감마 <b>B</b>로, 이 별은 분광형 B의 <a href="/wiki/%EC%A4%80%EA%B1%B0%EC%84%B1" title="준거성">준거성</a>이다. B는 <a href="/wiki/%EC%B2%9C%EA%B5%AC" title="천구">천구</a>상에서 A로부터 41.2" 떨어져 있으며, <a href="/wiki/%EC%8C%8D%EC%95%88%EA%B2%BD" title="쌍안경">쌍안경</a>을 이용하면 분리된 모습을 쉽게 관찰할 수 있다.</p>
<p>이외에도 돛자리 감마 항성계에는 더 어두운 몇 개의 반성들이 있다. 돛자리 감마 <b>C</b>는 겉보기 등급 8.5의 분광형 A 항성으로, A로부터 천구상에서 62.3초각 떨어져 있다. 돛자리 감마 <b>D</b>와 <b>E</b>는 천구상에서 A로부터 93.5초각 떨어져 있다. D는 겉보기 등급 9.4의 분광형 A 항성이며, E는 D로부터 천구상에서 1.8초각 떨어져 있으며 겉보기 등급은 +13이다.</p>
