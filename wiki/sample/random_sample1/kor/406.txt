<p><b>앰퍼샌드</b>(<span lang="en" xml:lang="en">ampersand</span>, <b>&amp;</b>, 앤드 기호)는 ‘~와(과)’를 의미하는 <a href="/wiki/%EA%B8%B0%ED%98%B8" title="기호">기호</a>이다. <a href="/wiki/%EC%98%81%EC%96%B4" title="영어">영어</a>의 <span lang="en" xml:lang="en">"and"</span>에 해당하는 <a href="/wiki/%EB%9D%BC%ED%8B%B4%EC%96%B4" title="라틴어">라틴어</a>의 <span lang="la" xml:lang="la">‘et’</span>의 <a href="/wiki/%ED%95%A9%EC%9E%90" title="합자">합자</a>로, <span lang="en" xml:lang="en">‘etc.’</span>를 <span lang="en" xml:lang="en">‘&amp;c.’</span>로 쓰기도 했다. <a href="/wiki/Z" title="Z">Z</a> 다음에 해당하는 27번째 <a class="mw-redirect" href="/wiki/%EB%9D%BC%ED%8B%B4_%EB%AC%B8%EC%9E%90" title="라틴 문자">라틴 문자</a> <a class="mw-redirect" href="/wiki/%EC%95%8C%ED%8C%8C%EB%B2%B3" title="알파벳">알파벳</a>으로 여겨졌던 시기도 있다.</p>
<h2><span class="mw-headline" id=".EC.82.AC.EC.9A.A9">사용</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%95%B0%ED%8D%BC%EC%83%8C%EB%93%9C&amp;action=edit&amp;section=1" title="부분 편집: 사용">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>한국과 일본에서는 <a href="/wiki/%EA%B0%80%EC%9A%B4%EB%8E%83%EC%A0%90" title="가운뎃점">가운뎃점</a>(·)과 흡사하게, 중국에서는 <a class="mw-redirect" href="/wiki/%EB%B0%98%EC%A0%90%EA%B3%BC_%EB%AA%A8%EC%A0%90" title="반점과 모점">모점</a>(、)과 흡사하게 사용된다. <a class="mw-redirect" href="/wiki/%EC%8A%AC%EB%9E%98%EC%8B%9C" title="슬래시">슬래시</a>(/)처럼 사용되기도 한다.</p>
<h2><span class="mw-headline" id=".ED.94.84.EB.A1.9C.EA.B7.B8.EB.9E.98.EB.B0.8D_.EC.96.B8.EC.96.B4">프로그래밍 언어</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%95%B0%ED%8D%BC%EC%83%8C%EB%93%9C&amp;action=edit&amp;section=2" title="부분 편집: 프로그래밍 언어">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>C 등 여러 프로그래밍 언어에서는 AND 연산자로 사용된다. 다음은 C의 예이다.</p>
<div class="mw-highlight mw-content-ltr" dir="ltr">
<pre>
<span class="n">x</span> <span class="o">=</span> <span class="n">a</span> <span class="o">&amp;&amp;</span> <span class="n">b</span><span class="p">;</span>
</pre></div>
<p>위와 같이 &amp; 2개를 붙여쓴 것은 논리연산자 AND를 나타낸다. 이 경우 A, B가 모두 참이라면 X도 참이 되고, 그 외의 경우는 거짓이 된다.</p>
<div class="mw-highlight mw-content-ltr" dir="ltr">
<pre>
<span class="n">x</span> <span class="o">=</span> <span class="mh">0x12345678</span> <span class="o">&amp;</span> <span class="mh">0x0f0f0f0f</span><span class="p">;</span>
</pre></div>
<p>위와 같이 &amp;가 1개이면 비트연산자 AND를 나타낸다. 이 경우의 결과는 0x02040608이다.</p>
<p>BASIC 계열의 언어로는 문자열의 연결 연산자로서 사용된다.</p>
<div class="mw-highlight mw-content-ltr" dir="ltr">
<pre>
<span class="vg">A$</span><span class="w"> </span><span class="o">=</span><span class="w"> </span><span class="s2">"foo"</span><span class="w"> </span><span class="o">&amp;</span><span class="w"> </span><span class="s2">"bar"</span>
</pre></div>
<p>위의 경우에는 "foobar"를 돌려준다. 또, 주로 마이크로소프트 계열에서는 정수의 16진표기에 &amp;h를 이용해 &amp;h0F (10진으로 15)와 같이 표현한다.</p>
<p>SGML, XML, HTML에서는, 앰퍼샌드(&amp;)를 사용해 SGML 실체를 참조한다.</p>
