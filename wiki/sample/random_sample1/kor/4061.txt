<div class="mw-body-content" id="bodyContent">

<a href="/wiki/%EB%AC%BC%EC%86%8C" title="물소">물소</a>의 <a href="/wiki/%EC%95%84%EC%88%98%EB%9D%BC" title="아수라">아수라</a>인 <a class="new" href="/w/index.php?title=%EB%A7%88%ED%9E%88%EC%89%AC%EC%95%84%EC%88%98%EB%9D%BC&amp;action=edit&amp;redlink=1" title="마히쉬아수라 (없는 문서)">마히쉬아수라</a>(<a class="extiw" href="https://en.wikipedia.org/wiki/Mahishasura" title="en:Mahishasura"><span style="color: black">Mahishasura</span></a>)와 싸우고 있는 두르가: 여덟 개의 팔을 가지고 있으며 <a href="/wiki/%ED%98%B8%EB%9E%91%EC%9D%B4" title="호랑이">호랑이</a>를 타고 있고, 여덟 개의 손에는 <a class="mw-disambig" href="/wiki/%EB%8D%B0%EB%B0%94" title="데바">데바</a>들이 준 여러 무기들이 들려 있다. <a href="/wiki/%EA%B5%AC%EB%A6%84" title="구름">구름</a> 위의 <a href="/wiki/%ED%95%98%EB%8A%98" title="하늘">하늘</a>에서는 <a class="mw-disambig" href="/wiki/%EB%8D%B0%EB%B0%94" title="데바">데바</a>들이 이 싸움의 광경을 보고 있는데, 이것은 두르가가 <a class="mw-disambig" href="/wiki/%EB%8D%B0%EB%B0%94" title="데바">데바</a>들의 여성적 권능들의 한 표현인데, 특히 그 중에서도 수행자 자신의 내면의 <a href="/wiki/%EC%95%85" title="악">악</a>을 무찌르는 힘들의 총체적인 표현 또는 상징이라는 것을 의미한다</div>
</div>

<p><b>두르가</b>(<span style="font-size: smaller;"><a href="/wiki/%EC%82%B0%EC%8A%A4%ED%81%AC%EB%A6%AC%ED%8A%B8%EC%96%B4" title="산스크리트어">산스크리트어</a>:</span> <span lang="sa" xml:lang="sa">दुर्गा Durgā</span> 또는 <b>마 두르가</b>(<span style="font-size: smaller;"><a href="/wiki/%EB%B2%B5%EA%B3%A8%EC%96%B4" title="벵골어">벵골어</a>:</span> <span lang="bn" xml:lang="bn">মা দুর্গা</span> Mā Durgā 어머니 두르가, <span style="font-size: smaller;"><a href="/wiki/%EC%98%81%EC%96%B4" title="영어">영어</a>:</span> <span lang="en" xml:lang="en">Maa Durga</span>)는 <a href="/wiki/%ED%9E%8C%EB%91%90%EA%B5%90" title="힌두교">힌두교</a>의 <a href="/wiki/%EC%97%AC%EC%8B%A0" title="여신">여신</a>으로, 내면의 <a href="/wiki/%EC%95%85" title="악">악</a>에 대한 내면의 <a class="mw-disambig" href="/wiki/%EC%84%A0" title="선">선</a>의 궁극적 승리를 상징한다. 힌두교에서 여신을 뜻하는 <a href="/wiki/%EB%8D%B0%EB%B9%84" title="데비">데비</a>는 모든 <a class="mw-redirect" href="/wiki/%ED%9E%8C%EB%91%90%EA%B5%90_%EC%97%AC%EC%8B%A0%EB%93%A4" title="힌두교 여신들">힌두교 여신들</a>의 궁극적 본질이라고 여겨지는데 이런 면에서 두르가는 <a href="/wiki/%EB%8D%B0%EB%B9%84" title="데비">데비</a>의 한 모습이다.</p>
<p><a href="/wiki/%EC%95%84%EC%88%98%EB%9D%BC" title="아수라">악신</a>인 <a class="new" href="/w/index.php?title=%EB%A7%88%ED%9E%88%EC%89%AC%EC%95%84%EC%88%98%EB%9D%BC&amp;action=edit&amp;redlink=1" title="마히쉬아수라 (없는 문서)">마히쉬아수라</a>(<a class="extiw" href="https://en.wikipedia.org/wiki/Mahishasura" title="en:Mahishasura"><span style="color: black">Mahishasura</span></a>)가 <a href="/wiki/%ED%95%98%EB%8A%98" title="하늘">하늘</a>과 <a href="/wiki/%EB%95%85" title="땅">땅</a>을 어지럽히고 있을 때, <a href="/wiki/%EB%B8%8C%EB%9D%BC%ED%9D%90%EB%A7%88" title="브라흐마">브라흐마</a> <b>·</b> <a href="/wiki/%EB%B9%84%EC%8A%88%EB%88%84" title="비슈누">비슈누</a> <b>·</b> <a href="/wiki/%EC%8B%9C%EB%B0%94" title="시바">시바</a> 등 <a href="/wiki/%ED%9E%8C%EB%91%90%EA%B5%90" title="힌두교">힌두교</a>의 <a class="mw-disambig" href="/wiki/%EB%8D%B0%EB%B0%94" title="데바">데바</a>(<a href="/wiki/%ED%9E%8C%EB%91%90%EA%B5%90%EC%9D%98_%EC%8B%A0#.EB.82.A8.EC.8B.A0" title="힌두교의 신">남신</a>)들의 힘이 구체화된 한 여신으로서의 두르가가 출현하여 <a class="new" href="/w/index.php?title=%EB%A7%88%ED%9E%88%EC%89%AC%EC%95%84%EC%88%98%EB%9D%BC&amp;action=edit&amp;redlink=1" title="마히쉬아수라 (없는 문서)">마히쉬아수라</a>를 무찌르는 신화가 <a class="mw-redirect" href="/wiki/%ED%9E%8C%EB%91%90%EA%B5%90%EC%9D%98_%EA%B2%BD%EC%A0%84" title="힌두교의 경전">힌두교의 경전군</a> 중 18 <a class="mw-redirect" href="/wiki/%EB%A7%88%ED%95%98%ED%91%B8%EB%9D%BC%EB%82%98" title="마하푸라나">마하푸라나</a>의 하나인 《<a class="new" href="/w/index.php?title=%EB%A7%88%EB%A5%B4%EC%B9%B8%EB%8D%B0%EC%95%BC_%ED%91%B8%EB%9D%BC%EB%82%98&amp;action=edit&amp;redlink=1" title="마르칸데야 푸라나 (없는 문서)">마르칸데야 푸라나</a>(<a class="extiw" href="https://en.wikipedia.org/wiki/Markandeya_Purana" title="en:Markandeya Purana"><span style="color: black">Markandeya Purana</span></a>)》의 〈<a class="new" href="/w/index.php?title=%EB%8D%B0%EB%B9%84_%EB%A7%88%ED%95%98%ED%8A%B8%EB%AF%B8%EC%95%94&amp;action=edit&amp;redlink=1" title="데비 마하트미암 (없는 문서)">데비 마하트미암</a>(<a class="extiw" href="https://en.wikipedia.org/wiki/Devi_Mahatmyam" title="en:Devi Mahatmyam"><span style="color: black">Devi Mahatmyam</span></a>)〉 이야기에 자세히 나온다. 이 이야기에 따라 두르가는 "마히쉬아수라를 죽인 자"라는 뜻에서 <b>마하쉬아수르마르디니</b>(Mahishasurmardhini)라고도 불린다.</p>
<p>산스크리트어 두르가(durgā)는 "어려움이 있는"을 뜻하는 두르(dur)와 <span class="nowrap">"가다</span> 또는 오다"를 뜻하는 가(gā)가 합쳐서 이루어진 낱말이다. 이에 따라 <a href="/wiki/%ED%9E%8C%EB%91%90%EA%B5%90" title="힌두교">힌두교</a>에서 두르가 신은 "(악 또는 어려움이) 접근하기 어려운 자, 접근할 수 없는 자"<sup class="reference" id="cite_ref-1"><a href="#cite_note-1">[1]</a></sup>, "(모든 악 또는 어려움을 제압하는) 무적자" 또는 <span class="nowrap">"어려움</span> 속에서도 갈 수 있는 자, 가장 어려운 상황에서도 <a href="/wiki/%EA%B5%AC%EC%9B%90" title="구원">구원</a>의 워크를 행할 수 있는 자"의 뜻을 가진 것으로 해석된다.</p>
<p>두르가는 여덟 개의 팔을 가지고 있으며 <a href="/wiki/%EC%82%AC%EC%9E%90" title="사자">사자</a> 또는 <a href="/wiki/%ED%98%B8%EB%9E%91%EC%9D%B4" title="호랑이">호랑이</a>를 타고 있는 모습으로 묘사되는데, 명상적인 미소를 띤 채 손에는 여러 무기들과 <a href="/wiki/%EC%97%B0%EA%BD%83" title="연꽃">연꽃</a>을 지니고 있으며 손가락으로는 <a class="mw-redirect mw-disambig" href="/wiki/%EB%AC%B4%EB%93%9C%EB%9D%BC" title="무드라">무드라</a>(<a class="mw-disambig" href="/wiki/%EC%9D%B8%EC%83%81" title="인상">인</a>)를 맺고 있는 것으로 표현된다.</p>
<p><a class="new" href="/w/index.php?title=%EB%91%90%EB%A5%B4%EA%B0%80_%ED%91%B8%EC%9E%90&amp;action=edit&amp;redlink=1" title="두르가 푸자 (없는 문서)">두르가 푸자</a>(<a class="extiw" href="https://en.wikipedia.org/wiki/Durga_Puja" title="en:Durga Puja"><span style="color: black">Durga Puja</span></a>) 축제에서, 두르가는 <a href="/wiki/%EA%B0%80%EB%84%A4%EC%83%A4" title="가네샤">가네샤</a> <b>·</b> <a class="mw-redirect" href="/wiki/%EC%B9%B4%EB%A5%B4%ED%8B%B0%EC%BC%80%EC%95%BC" title="카르티케야">카르티케야</a> <b>·</b> <a href="/wiki/%EB%9D%BD%EC%8A%88%EB%AF%B8" title="락슈미">락슈미</a> <b>·</b> <a href="/wiki/%EC%82%AC%EB%9D%BC%EC%8A%A4%EB%B0%94%ED%8B%B0" title="사라스바티">사라스바티</a>의 어머니로 등장한다.<sup class="reference" id="cite_ref-2"><a href="#cite_note-2">[2]</a></sup></p>
