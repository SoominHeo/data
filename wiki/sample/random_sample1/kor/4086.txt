<p><b>둥베이</b>(동북. <small><a href="/wiki/%EC%A4%91%EA%B5%AD%EC%96%B4" title="중국어">중국어</a></small> <small><a href="/wiki/%EA%B0%84%EC%B2%B4%EC%9E%90" title="간체자">간체</a>:</small> <span lang="zh-hans" xml:lang="zh-hans">东北</span>, <small><a href="/wiki/%EC%A0%95%EC%B2%B4%EC%9E%90" title="정체자">정체</a>:</small> <span lang="zh-hant" xml:lang="zh-hant">東北</span>, <small><a href="/wiki/%ED%95%9C%EC%96%B4_%EB%B3%91%EC%9D%8C" title="한어 병음">병음</a>:</small> <font face="Lucida Sans Unicode">Dōngběi</font>) 또는 <b>둥베이 3성</b>(동북3성. <small><a href="/wiki/%EC%A4%91%EA%B5%AD%EC%96%B4" title="중국어">중국어</a></small> <small><a href="/wiki/%EA%B0%84%EC%B2%B4%EC%9E%90" title="간체자">간체</a>:</small> <span lang="zh-hans" xml:lang="zh-hans">东北三省</span>, <small><a href="/wiki/%EC%A0%95%EC%B2%B4%EC%9E%90" title="정체자">정체</a>:</small> <span lang="zh-hant" xml:lang="zh-hant">東北三省</span>)은 <a href="/wiki/%EC%A4%91%EA%B5%AD" title="중국">중국</a>의 6대 중국지리대구(中國地理大區)의 하나로서 동북부 지역을 말한다. 과거에는 <b><a href="/wiki/%EB%A7%8C%EC%A3%BC" title="만주">만주</a></b>(<small><a href="/wiki/%EC%A4%91%EA%B5%AD%EC%96%B4" title="중국어">중국어</a></small> <small><a href="/wiki/%EA%B0%84%EC%B2%B4%EC%9E%90" title="간체자">간체</a>:</small> <span lang="zh-hans" xml:lang="zh-hans">满洲</span>, <small><a href="/wiki/%EC%A0%95%EC%B2%B4%EC%9E%90" title="정체자">정체</a>:</small> <span lang="zh-hant" xml:lang="zh-hant">滿洲</span>, <small><a href="/wiki/%ED%95%9C%EC%96%B4_%EB%B3%91%EC%9D%8C" title="한어 병음">병음</a>:</small> <font face="Lucida Sans Unicode">Mǎnzhōu</font>,  <small>만저우<sup>[<a href="/wiki/%EC%9C%84%ED%82%A4%EB%B0%B1%EA%B3%BC:%EC%A4%91%EA%B5%AD%EC%96%B4%EC%9D%98_%ED%95%9C%EA%B8%80_%ED%91%9C%EA%B8%B0" title="위키백과:중국어의 한글 표기">*</a>]</sup></small>)로 불렸던 지역이다. 행정구역상으로는 <a href="/wiki/%EB%9E%B4%EC%98%A4%EB%8B%9D_%EC%84%B1" title="랴오닝 성">랴오닝 성</a>(遼寧省), <a href="/wiki/%EC%A7%80%EB%A6%B0_%EC%84%B1" title="지린 성">지린 성</a>(吉林省), <a href="/wiki/%ED%97%A4%EC%9D%B4%EB%A3%BD%EC%9E%A5_%EC%84%B1" title="헤이룽장 성">헤이룽장 성</a>(黑龍江省)의 3개 성(省)이 포함되며 <a href="/wiki/%EB%82%B4%EB%AA%BD%EA%B3%A8_%EC%9E%90%EC%B9%98%EA%B5%AC" title="내몽골 자치구">내몽골 자치구</a> 동북부 <a class="new" href="/w/index.php?title=%EB%8F%99%EC%82%AC%EB%A7%B9&amp;action=edit&amp;redlink=1" title="동사맹 (없는 문서)">동사맹</a>지역 (<a href="/wiki/%ED%9B%84%EB%A3%AC%EB%B2%A0%EC%9D%B4%EC%96%BC_%EC%8B%9C" title="후룬베이얼 시">후룬베이얼 시</a>, <a href="/wiki/%EC%8B%B1%EC%95%88_%EB%A7%B9" title="싱안 맹">싱안 맹</a>, <a href="/wiki/%ED%89%81%EB%9E%B4%EC%98%A4_%EC%8B%9C" title="퉁랴오 시">퉁랴오 시</a>, <a href="/wiki/%EC%B8%A0%ED%8E%91_%EC%8B%9C" title="츠펑 시">츠펑 시</a>)을 포함하기도 하고 포함하지 않기도 한다. 이 지역 최대의 도시는 <a href="/wiki/%EC%84%A0%EC%96%91_%EC%8B%9C" title="선양 시">선양</a>(瀋陽)이며, 그 외에 <a class="mw-redirect" href="/wiki/%ED%95%98%EC%96%BC%EB%B9%88" title="하얼빈">하얼빈</a>(哈爾濱), <a class="mw-redirect" href="/wiki/%EC%B0%BD%EC%B6%98" title="창춘">창춘</a>(長春), <a class="mw-redirect" href="/wiki/%EB%8B%A4%EB%A1%84" title="다롄">다롄</a>(大連), <a class="mw-redirect" href="/wiki/%EC%B9%98%EC%B9%98%ED%95%98%EC%96%BC" title="치치하얼">치치하얼</a>(齊齊哈爾), <a class="mw-disambig" href="/wiki/%EC%A7%80%EB%A6%B0" title="지린">지린</a>(吉林) 등의 도시가 있다.</p>
<p>200만 명 이상의 <a href="/wiki/%EC%A1%B0%EC%84%A0%EC%A1%B1" title="조선족">조선족</a>이 거주하고 있고 <a href="/wiki/%EA%B3%A0%EC%A1%B0%EC%84%A0" title="고조선">고조선</a>, <a href="/wiki/%EA%B3%A0%EA%B5%AC%EB%A0%A4" title="고구려">고구려</a>와 <a href="/wiki/%EB%B0%9C%ED%95%B4" title="발해">발해</a>의 유적이 많이 남아있어서 <a href="/wiki/%ED%95%9C%EB%AF%BC%EC%A1%B1" title="한민족">한민족</a>과 매우 밀접한 관련을 맺고 있는 지역이기도 하다.</p>

<h2><span class="mw-headline" id=".EC.97.AD.EC.82.AC">역사</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%91%A5%EB%B2%A0%EC%9D%B4&amp;action=edit&amp;section=1" title="부분 편집: 역사">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>둥베이는 <a href="/wiki/%EC%A4%91%EA%B5%AD" title="중국">중국</a>의 <a href="/wiki/%ED%95%9C%EC%A1%B1" title="한족">한족</a>(漢族)과 <a href="/wiki/%ED%95%9C%EA%B5%AD" title="한국">한국</a>의 <a href="/wiki/%ED%95%9C%EB%AF%BC%EC%A1%B1" title="한민족">한민족</a> 및 여러 북방 민족이 서로 차지하기 위해 쟁탈전을 벌였던 곳이다. 특히 둥베이는 <span class="fix" title="이 부분은 출처가 필요합니다.">한국 역사상 최초의 국가인 <a href="/wiki/%EA%B3%A0%EC%A1%B0%EC%84%A0" title="고조선">고조선</a>이 세워진 곳이며</span><sup class="정리_필요 noprint">[<a href="/wiki/%EC%9C%84%ED%82%A4%EB%B0%B1%EA%B3%BC:%EC%B6%9C%EC%B2%98_%EB%B0%9D%ED%9E%88%EA%B8%B0" title="위키백과:출처 밝히기">출처 필요</a>]</sup> 중국 <a href="/wiki/%EC%B2%AD%EB%82%98%EB%9D%BC" title="청나라">청나라</a>를 세운 <a href="/wiki/%EB%A7%8C%EC%A3%BC%EC%A1%B1" title="만주족">만주족</a>(전신은 <a class="mw-redirect" href="/wiki/%EC%97%AC%EC%A7%84%EC%A1%B1" title="여진족">여진족</a>)의 본거지이기도 하다. <a href="/wiki/%EC%B2%AD%EB%82%98%EB%9D%BC" title="청나라">청나라</a> 말기 중국이 한창 <a href="/wiki/%EC%A0%9C%EA%B5%AD%EC%A3%BC%EC%9D%98" title="제국주의">제국주의</a> 국가들로부터 침략을 당하던 시기에 <a href="/wiki/%EB%9F%AC%EC%8B%9C%EC%95%84" title="러시아">러시아</a>에 <a class="mw-redirect" href="/wiki/%EB%A4%BC%EC%88%9C" title="뤼순">뤼순</a>(旅順)을 조차하였고, 청나라 멸망 이후에는 <a href="/wiki/%EC%A4%91%ED%99%94%EB%AF%BC%EA%B5%AD" title="중화민국">중화민국</a>의 영토가 되었다. <a href="/wiki/1931%EB%85%84" title="1931년">1931년</a>에 <a href="/wiki/%EC%9D%BC%EB%B3%B8" title="일본">일본</a>이 일으킨 <a href="/wiki/%EB%A7%8C%EC%A3%BC%EC%82%AC%EB%B3%80" title="만주사변">만주사변</a>으로 일본에 점령되어 일본의 지배를 받는 괴뢰 국가인 <a href="/wiki/%EB%A7%8C%EC%A3%BC%EA%B5%AD" title="만주국">만주국</a>(滿洲國)이 세워지기도 하였다. <a href="/wiki/1949%EB%85%84" title="1949년">1949년</a> 이후에 <a href="/wiki/%EC%A4%91%ED%99%94%EC%9D%B8%EB%AF%BC%EA%B3%B5%ED%99%94%EA%B5%AD" title="중화인민공화국">중화인민공화국</a>의 영토로 편입되어 오늘날에 이른다.</p>
<h2><span class="mw-headline" id=".EC.A3.BC.EB.AF.BC">주민</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%91%A5%EB%B2%A0%EC%9D%B4&amp;action=edit&amp;section=2" title="부분 편집: 주민">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>둥베이의 인구는 현재 약 1억 명으로, 이는 중국 전체 인구의 8%에 해당한다.</p>
<h3><span class="mw-headline" id=".EB.AF.BC.EC.A1.B1">민족</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%91%A5%EB%B2%A0%EC%9D%B4&amp;action=edit&amp;section=3" title="부분 편집: 민족">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p><a href="/wiki/%ED%95%9C%EC%A1%B1" title="한족">한족</a>(漢族)이 거의 대다수를 차지하며, 그 외에 <a href="/wiki/%EC%A1%B0%EC%84%A0%EC%A1%B1" title="조선족">조선족</a>, <a href="/wiki/%EB%A7%8C%EC%A3%BC%EC%A1%B1" title="만주족">만주족</a>, <a href="/wiki/%EB%AA%BD%EA%B3%A8%EC%A1%B1" title="몽골족">몽골족</a> 등의 소수민족들이 거주한다. 지리적으로 <a href="/wiki/%EC%A1%B0%EC%84%A0%EB%AF%BC%EC%A3%BC%EC%A3%BC%EC%9D%98%EC%9D%B8%EB%AF%BC%EA%B3%B5%ED%99%94%EA%B5%AD" title="조선민주주의인민공화국">조선민주주의인민공화국</a>과 인접해있어 <a class="mw-redirect" href="/wiki/%ED%83%88%EB%B6%81%EC%9E%90" title="탈북자">탈북자</a>들도 있다.</p>
<h3><span class="mw-headline" id=".EC.96.B8.EC.96.B4">언어</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%91%A5%EB%B2%A0%EC%9D%B4&amp;action=edit&amp;section=4" title="부분 편집: 언어">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p><a href="/wiki/%EB%91%A5%EB%B2%A0%EC%9D%B4%EC%96%B4" title="둥베이어">둥베이어</a>나 <a class="mw-redirect" href="/wiki/%EB%B2%A0%EC%9D%B4%EC%A7%95" title="베이징">베이징</a>에서 사용하는 표준 <a href="/wiki/%EC%A4%91%EA%B5%AD%EC%96%B4" title="중국어">중국어</a>가 통용되나, <a href="/wiki/%EC%A1%B0%EC%84%A0%EC%A1%B1" title="조선족">조선족</a> 사이에서는 <a href="/wiki/%ED%95%9C%EA%B5%AD%EC%96%B4" title="한국어">한국어</a>가 사용되기도 한다. 이와는 달리 <a href="/wiki/%EB%A7%8C%EC%A3%BC%EC%96%B4" title="만주어">만주어</a>를 아는 <a href="/wiki/%EB%A7%8C%EC%A3%BC%EC%A1%B1" title="만주족">만주족</a>은 많지 않으며, 대부분의 <a href="/wiki/%EB%A7%8C%EC%A3%BC%EC%A1%B1" title="만주족">만주족</a>들은 <a href="/wiki/%EC%A4%91%EA%B5%AD%EC%96%B4" title="중국어">중국어</a>를 사용한다.</p>
<h2><span class="mw-headline" id=".EC.82.B0.EC.97.85">산업</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%91%A5%EB%B2%A0%EC%9D%B4&amp;action=edit&amp;section=5" title="부분 편집: 산업">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>둥베이는 일본의 지배 시기에 <a href="/wiki/%EA%B3%B5%EC%97%85" title="공업">공업</a>의 기반이 잘 조성되어 초기에는 중국 최대의 중화학 공업 지대가 되었다. 그러나 1980년대에 중국이 외국 자본을 도입하면서 <a href="/wiki/%EA%B4%91%EB%91%A5_%EC%84%B1" title="광둥 성">광둥 성</a>(廣東省) 등지의 산업이 급격히 발달하자 둥베이의 공업은 급격히 쇠퇴하기 시작하였으나, 최근에 <a href="/wiki/%ED%95%9C%EA%B5%AD" title="한국">한국</a>과 <a href="/wiki/%EC%9D%BC%EB%B3%B8" title="일본">일본</a> 등지의 자본이 밀려오면서 다시 활력을 되찾고 있다. 지하자원으로는 <a class="mw-redirect" href="/wiki/%ED%91%B8%EC%88%9C" title="푸순">푸순</a>(撫順)의 탄전과 <a class="mw-redirect" href="/wiki/%EB%8B%A4%EC%B9%AD" title="다칭">다칭</a>(大慶)의 유전이 있다.</p>
