<p><b>2010년 동계 올림픽 개막식</b>은 <a class="mw-redirect" href="/wiki/%ED%83%9C%ED%8F%89%EC%96%91_%EC%8B%9C%EA%B0%84%EB%8C%80" title="태평양 시간대">PST</a> 기준으로 <a href="/wiki/2010%EB%85%84" title="2010년">2010년</a> <a href="/wiki/2%EC%9B%94_12%EC%9D%BC" title="2월 12일">2월 12일</a>에 <a href="/wiki/%EC%BA%90%EB%82%98%EB%8B%A4" title="캐나다">캐나다</a>의 <a href="/wiki/%EB%B8%8C%EB%A6%AC%ED%8B%B0%EC%8B%9C%EC%BB%AC%EB%9F%BC%EB%B9%84%EC%95%84_%EC%A3%BC" title="브리티시컬럼비아 주">브리티시컬럼비아 주</a>에 위치한 <a href="/wiki/%EB%B0%B4%EC%BF%A0%EB%B2%84" title="밴쿠버">밴쿠버</a> 시내에 있는 <a class="mw-redirect" href="/wiki/BC_%ED%94%8C%EB%A0%88%EC%9D%B4%EC%8A%A4_%EC%8A%A4%ED%83%80%EB%94%94%EC%9B%80" title="BC 플레이스 스타디움">BC 플레이스 스타디움</a>에서 열린 <a href="/wiki/2010%EB%85%84_%EB%8F%99%EA%B3%84_%EC%98%AC%EB%A6%BC%ED%94%BD" title="2010년 동계 올림픽">2010년 동계 올림픽</a>의 <a class="mw-redirect" href="/wiki/%EA%B0%9C%EB%A7%89%EC%8B%9D" title="개막식">개막식</a>이다. 연방정부에서는 개회식만을 위해 4000만 달러가 넘는 돈을 쏟아부었으며 이것은 역대 <a href="/wiki/%EB%8F%99%EA%B3%84_%EC%98%AC%EB%A6%BC%ED%94%BD" title="동계 올림픽">동계 올림픽</a>에서는 가장 높은 금액이며 역대 올림픽 중에서는 <a href="/wiki/2008%EB%85%84_%ED%95%98%EA%B3%84_%EC%98%AC%EB%A6%BC%ED%94%BD" title="2008년 하계 올림픽">베이징 올림픽</a>에 이어 두 번째이다. 처음으로 실내에서 개회식을 개최하는 올림픽이기도 하다. 개회식과 폐회식은 <a href="/wiki/2000%EB%85%84_%ED%95%98%EA%B3%84_%EC%98%AC%EB%A6%BC%ED%94%BD" title="2000년 하계 올림픽">시드니 올림픽</a>과 <a href="/wiki/2006%EB%85%84_%EB%8F%99%EA%B3%84_%EC%98%AC%EB%A6%BC%ED%94%BD" title="2006년 동계 올림픽">토리노 올림픽</a>의 총감독을 맡았던 <a class="new" href="/w/index.php?title=%EB%8D%B0%EC%9D%B4%EB%B9%84%EB%93%9C_%EC%95%B3%ED%82%A8%EC%8A%A4&amp;action=edit&amp;redlink=1" title="데이비드 앳킨스 (없는 문서)">데이비드 앳킨스</a>가 지휘했다.</p>
<p>개회식은 <a class="mw-redirect" href="/wiki/%EC%BA%90%EB%82%98%EB%8B%A4%EC%9D%98_%EC%97%B0%EB%B0%A9_%EC%B4%9D%EB%8F%85" title="캐나다의 연방 총독">연방 총독</a>인 <a href="/wiki/%EB%AF%B8%EC%B9%B4%EC%97%98_%EC%9E%A5" title="미카엘 장">미카엘 장</a>에 의해 공식적으로 거행됐다.<sup class="reference" id="cite_ref-GovernorGeneralFamily_1-0"><a href="#cite_note-GovernorGeneralFamily-1">[1]</a></sup><sup class="reference" id="cite_ref-2"><a href="#cite_note-2">[2]</a></sup> 그녀의 남편인 <a class="new" href="/w/index.php?title=%EC%9E%A5_%EB%8B%A4%EB%8B%88%EC%97%98_%EB%9D%BC%ED%90%81&amp;action=edit&amp;redlink=1" title="장 다니엘 라퐁 (없는 문서)">장 다니엘 라퐁</a>과 딸인 마리 에덴도 개회식에 참석했다.<sup class="reference" id="cite_ref-GovernorGeneralFamily_1-1"><a href="#cite_note-GovernorGeneralFamily-1">[1]</a></sup></p>
