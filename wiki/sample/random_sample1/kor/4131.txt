<p><b>드워프 햄스터</b>(<span lang="en" xml:lang="en">dwarf Hamster</span> <small>‘난쟁이 <a href="/wiki/%ED%96%84%EC%8A%A4%ED%84%B0" title="햄스터">햄스터</a>’<sup>[<a href="/wiki/%EC%9C%84%ED%82%A4%EB%B0%B1%EA%B3%BC:%EC%98%81%EC%96%B4%EC%9D%98_%ED%95%9C%EA%B8%80_%ED%91%9C%EA%B8%B0" title="위키백과:영어의 한글 표기">*</a>]</sup></small>)는 <a class="mw-redirect" href="/wiki/%EC%8B%9C%EB%A6%AC%EC%95%88_%ED%96%84%EC%8A%A4%ED%84%B0" title="시리안 햄스터">시리안 햄스터</a>에 비해 상대적으로 몸집이 작은 햄스터들을 일괄적으로 부르는 말이다. 단독 생활을 하는 시리안 햄스터와는 달리 드워프 햄스터는 공동 생활을 하는 것으로 알려져 있지만, 이는 <a class="mw-redirect" href="/wiki/%EC%A7%9D%EC%A7%93%EA%B8%B0" title="짝짓기">짝짓기</a>를 하고 새끼를 암수가 공동 양육하는 기간 동안만 해당되며 소수 가족 단위로 이루어진 무리 생활을 뜻한다. 각각의 무리는 자신의 영역을 지키며 낯선 상대를 쫓아내기 위해서 사정없이 덤벼든다. 평소에 잘 지내던 햄스터들이 갑자기 싸움을 시작하여 서로를 물어 죽이기도 하는데, 서열 싸움이라고 볼 수도 있겠지만 넓게 보면 단독 생활을 하기 때문에 벌어지는 사고라고 할 수 있다. 따라서 드워프 햄스터가 공동 생활을 한다고 하여 작은 우리에 여러 마리를 기르는 것은 옳지 않다.</p>
<p>대표적인 애완용 드워프 햄스터로 캠벨 러시안, 윈터 화이트, 로보로브스키, <a href="/wiki/%EB%B9%84%EB%8B%A8%ED%84%B8%EB%93%B1%EC%A4%84%EC%A5%90%EC%86%8D" title="비단털등줄쥐속">차이니즈 햄스터</a> 등이 있다.</p>
<h2><span class="mw-headline" id=".EC.BA.A0.EB.B2.A8_.EB.9F.AC.EC.8B.9C.EC.95.88">캠벨 러시안</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%93%9C%EC%9B%8C%ED%94%84%ED%96%84%EC%8A%A4%ED%84%B0&amp;action=edit&amp;section=1" title="부분 편집: 캠벨 러시안">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<div class="thumb tleft">
<div class="thumbinner" style="width:182px;"><a class="image" href="/wiki/%ED%8C%8C%EC%9D%BC:Campbell_hamster.jpg"><img alt="" class="thumbimage" data-file-height="238" data-file-width="180" height="238" src="//upload.wikimedia.org/wikipedia/commons/f/f8/Campbell_hamster.jpg" width="180"/></a>
<div class="thumbcaption">
<div class="magnify"><a class="internal" href="/wiki/%ED%8C%8C%EC%9D%BC:Campbell_hamster.jpg" title="실제 크기로"></a></div>
캠밸 러시안 햄스터</div>
</div>
</div>
<p>캠벨 러시안 햄스터(학명:<i>Phodopus sungorus campbelli</i>)는 건조한 사막, 초원 또는 경작지에 서식하며 <a href="/wiki/%EC%8B%9C%EB%B2%A0%EB%A6%AC%EC%95%84" title="시베리아">시베리아</a>의 <a class="mw-redirect" href="/wiki/%EB%B0%94%EC%9D%B4%EC%B9%BC_%ED%98%B8%EC%88%98" title="바이칼 호수">바이칼 호수</a>로부터 <a href="/wiki/%EB%AA%BD%EA%B3%A8" title="몽골">몽골</a>, <a href="/wiki/%EC%A4%91%ED%99%94%EC%9D%B8%EB%AF%BC%EA%B3%B5%ED%99%94%EA%B5%AD" title="중화인민공화국">중국</a>의 <a href="/wiki/%ED%97%A4%EC%9D%B4%EB%A3%BD%EC%9E%A5_%EC%84%B1" title="헤이룽장 성">헤이룽장 성</a>에서 내몽골, <a href="/wiki/%EC%8B%A0%EC%9E%A5_%EC%9C%84%EA%B5%AC%EB%A5%B4_%EC%9E%90%EC%B9%98%EA%B5%AC" title="신장 위구르 자치구">위구르</a>에 이르는 중국 북부 지역에 분포하고 있다. 1905년 중국 북부 지방인 내몽고에서 <a class="new" href="/w/index.php?title=%ED%86%A0%EB%A7%88%EC%8A%A4_%EC%BA%A0%EB%B2%A8&amp;action=edit&amp;redlink=1" title="토마스 캠벨 (없는 문서)">토마스 캠벨</a>에 의해 최초로 발견되어 최초 발견자의 이름을 따 캠벨 러시안 햄스터라 불리게 되었다.</p>
<p><a href="/wiki/1961%EB%85%84" title="1961년">1961년</a> <a href="/wiki/%EC%86%8C%EB%A0%A8" title="소련">소련</a>의 <a class="new" href="/w/index.php?title=%EB%A0%88%EB%8B%8C%EA%B7%B8%EB%9D%BC%EB%93%9C_%EB%8F%99%EB%AC%BC_%ED%95%99%ED%9A%8C&amp;action=edit&amp;redlink=1" title="레닌그라드 동물 학회 (없는 문서)">레닌그라드 동물 학회</a>에서 처음 사육되었으며, 1963년 <a href="/wiki/%EC%98%81%EA%B5%AD" title="영국">영국</a> <a href="/wiki/%EB%9F%B0%EB%8D%98_%EB%8F%99%EB%AC%BC%EC%9B%90" title="런던 동물원">런던 동물원</a>에 들여와 <a href="/wiki/1970%EB%85%84" title="1970년">1970년</a> Percy Parslow로부터 애완 동물로 사육되어 널리 알려졌다. 보통 수컷이 암컷보다 몸집이 큰 편이며 암컷을 도와 새끼를 양육하며 몸길이는 약 10~12센티미터, 몸무게는 약 20~50그램 정도이다.</p>
<p>사람을 잘 물고 사람에 대한 경계심이 많기 때문에 다른 드워프 햄스터에 비해서 인기가 많은 편은 아니다.</p>

<h2><span class="mw-headline" id=".EC.9C.88.ED.84.B0_.ED.99.94.EC.9D.B4.ED.8A.B8">윈터 화이트</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%93%9C%EC%9B%8C%ED%94%84%ED%96%84%EC%8A%A4%ED%84%B0&amp;action=edit&amp;section=2" title="부분 편집: 윈터 화이트">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<div class="thumb tright">
<div class="thumbinner" style="width:222px;"><a class="image" href="/wiki/%ED%8C%8C%EC%9D%BC:Djungarian_Hamster_Pearl_White_run_wheel.jpg"><img alt="" class="thumbimage" data-file-height="886" data-file-width="1083" height="180" src="//upload.wikimedia.org/wikipedia/commons/thumb/6/63/Djungarian_Hamster_Pearl_White_run_wheel.jpg/220px-Djungarian_Hamster_Pearl_White_run_wheel.jpg" srcset="//upload.wikimedia.org/wikipedia/commons/thumb/6/63/Djungarian_Hamster_Pearl_White_run_wheel.jpg/330px-Djungarian_Hamster_Pearl_White_run_wheel.jpg 1.5x, //upload.wikimedia.org/wikipedia/commons/thumb/6/63/Djungarian_Hamster_Pearl_White_run_wheel.jpg/440px-Djungarian_Hamster_Pearl_White_run_wheel.jpg 2x" width="220"/></a>

윈터 화이트 햄스터</div>

<p>윈터 화이트 햄스터(학명:<i>Phodopus sungorus sungorus</i>)는 <a href="/wiki/1773%EB%85%84" title="1773년">1773년</a> 중국의 북서부인 <a class="mw-redirect" href="/wiki/%EC%8B%A0%EA%B0%95%EC%84%B1" title="신강성">신강성</a>, 남부 <a href="/wiki/%EC%95%8C%ED%83%80%EC%9D%B4_%EC%82%B0%EB%A7%A5" title="알타이 산맥">알타이 산맥</a>의 지류인 <a class="mw-redirect" href="/wiki/%EC%A4%80%EA%B0%80%EC%96%BC_%EB%B6%84%EC%A7%80" title="준가얼 분지">준가얼 분지</a>(<a class="mw-redirect" href="/wiki/%EC%A4%91%EA%B0%80%EB%A6%AC%EC%95%84_%EB%B6%84%EC%A7%80" title="중가리아 분지">중가리아 분지</a>)에서 <a class="mw-disambig" href="/wiki/%ED%8C%94%EB%9D%BC%EC%8A%A4" title="팔라스">팔라스</a>에 의해 최초로 발견되었다. <a href="/wiki/%EC%B9%B4%EC%9E%90%ED%9D%90%EC%8A%A4%ED%83%84" title="카자흐스탄">카자흐스탄</a>의 동부와 시베리아 남서쪽에 걸친 초원지대 혹은 경작지에 서식하며, 생김새가 비슷한 캠벨 러시안 햄스터와는 달리 겨울이 되면 털이 하얗게 변하는 특징이 있기 때문에 윈터 화이트(Winter White) 햄스터라고 불리며, 발견지인 <a class="mw-redirect" href="/wiki/%EC%A4%80%EA%B0%80%EC%96%BC_%EB%B6%84%EC%A7%80" title="준가얼 분지">준가얼 분지</a>의 이름을 따서 장가리안 또는 정글리안(Djungarian)이라고 부르기도 한다.</p>
<p><a href="/wiki/1960%EB%85%84" title="1960년">1960년</a>대에 들어서 <a href="/wiki/%EB%8F%85%EC%9D%BC" title="독일">독일</a>의 <a class="mw-redirect" href="/wiki/%EB%A7%89%EC%8A%A4_%ED%94%8C%EB%9E%91%ED%81%AC_%EC%97%B0%EA%B5%AC%EC%86%8C" title="막스 플랑크 연구소">막스 플랑크 연구소</a>의 연구원인 <a class="new" href="/w/index.php?title=%ED%81%B4%EB%9D%BC%EC%9A%B0%EC%8A%A4_%ED%98%B8%ED%94%84%EB%A7%8C&amp;action=edit&amp;redlink=1" title="클라우스 호프만 (없는 문서)">클라우스 호프만</a>에 의해 애완용으로 선보였고 몸길이는 10~12센티미터, 몸무게는 20~60그램 정도이다.</p>
<p>캠벨 러시안 햄스터에 비해서 성질이 온순하고 사람을 잘 따라서 매우 인기가 있다.</p>
<h2><span class="mw-headline" id=".EB.A1.9C.EB.B3.B4.EB.A1.9C.EB.B8.8C.EC.8A.A4.ED.82.A4">로보로브스키</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%93%9C%EC%9B%8C%ED%94%84%ED%96%84%EC%8A%A4%ED%84%B0&amp;action=edit&amp;section=3" title="부분 편집: 로보로브스키">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<div class="thumb tleft">

<p>로보로브스키 햄스터(학명:<i>Phodopus roborovskii</i>)는 애완동물로 키워지는 드워프 햄스터 중 가장 작은 종으로서 주로 <a href="/wiki/%EC%A4%91%ED%99%94%EC%9D%B8%EB%AF%BC%EA%B3%B5%ED%99%94%EA%B5%AD" title="중화인민공화국">중국</a> 북부지역의 <a class="mw-redirect" href="/wiki/%EA%B3%A0%EB%B9%84%EC%82%AC%EB%A7%89" title="고비사막">고비사막</a>과 <a href="/wiki/%EB%AA%BD%EA%B3%A8" title="몽골">몽골</a> 주변에 서식한다. 1894년 러시아인 Lt. Vsevolod Roborovski에 의해 처음으로 발견된 이후 로보로브스키 햄스터라 불리게 되었으며, 이를 줄여서 <a class="mw-redirect" href="/wiki/%EB%A1%9C%EB%B3%B4_%ED%96%84%EC%8A%A4%ED%84%B0" title="로보 햄스터">로보 햄스터</a>라고 하거나 흰 눈썹이 사슴을 닮았다고 해서 <a class="mw-redirect" href="/wiki/%EC%82%AC%EC%8A%B4_%ED%96%84%EC%8A%A4%ED%84%B0" title="사슴 햄스터">사슴 햄스터</a>라고도 불리기도 한다. 1960년대 이후 유럽에서 이에 대한 연구가 본격적으로 진행되다가 1990년대 말에 이르러 한국, 미국 등 여러 나라에 애완동물로 소개가 되었다.</p>
<p>성체의 몸길이는 대개 5~7센티미터로 작은 편이며 일반 햄스터용 케이지의 철장 사이를 쉽게 통과하기 때문에 철장 간격이 7밀리미터 이하의 드워프 햄스터용 케이지에서 키우거나 아예 유리로 된 수조에서 키우는 것이 좋다. 암컷의 경우 생후 5주가 되면 임신이 가능하게 되지만 생후 4개월이 되어 완전한 성체로 자라기 전까지는 새끼를 분만하더라도 양육을 포기하는 경우가 많기 때문에 각별히 주의해야 한다. 또한 다른 드워프 햄스터들보다 임신기간이 긴 편으로 최대 30일이 걸리며 대개 한번에 4~6마리의 새끼를 낳는다. 로보로브스키 햄스터는 다른 햄스터들에 비해 사회성이 좋은 편으로 대개 다른 여러 햄스터들과 같은 케이지에서 함께 생활하는 것이 가능한 것으로 알려져 있다. 그러나, 어린 햄스터의 경우 성체가 되어 본격적인 영역다툼을 벌이기 전까지는 사회성을 알기가 힘들며, 키우는 햄스터 수에 비해 케이지가 너무 작거나 먹이가 부족한 경우에는 영역다툼이 심해지게 되므로 여러 마리를 한 케이지에 같이 키우는 경우에는 지속적인 관찰이 필요하다.</p>
<p>로보로브스키 햄스터는 사람을 잘 물지는 않지만 예민하고 겁이 많아 길들이기가 힘들고 움직임도 상당히 빠른 편이라 대개 관상용으로 적합하다.</p>
