<p><b>들깨</b>(<small><a href="/wiki/%ED%95%99%EB%AA%85" title="학명">학명</a>:</small> <i>Perilla frutescens var. frutescens</i>)는 <a href="/wiki/%EA%BF%80%ED%92%80%EA%B3%BC" title="꿀풀과">꿀풀과</a>의 <a href="/wiki/%EC%9D%BC%EB%85%84%EC%83%9D%EC%8B%9D%EB%AC%BC" title="일년생식물">한해살이풀</a> 또는 그 <a href="/wiki/%EC%94%A8" title="씨">씨</a>이다. <a href="/wiki/%EB%8F%99%EB%82%A8%EC%95%84%EC%8B%9C%EC%95%84" title="동남아시아">동남아시아</a>와 <a href="/wiki/%EC%9D%B8%EB%8F%84" title="인도">인도</a> <a href="/wiki/%EA%B3%A0%EC%A7%80" title="고지">고지</a>가 원산지로, <a href="/wiki/%ED%95%9C%EB%B0%98%EB%8F%84" title="한반도">한반도</a>, <a href="/wiki/%EC%A4%91%EA%B5%AD" title="중국">중국</a> 중남부, <a href="/wiki/%EC%9D%B8%EB%8F%84" title="인도">인도</a> 등지에 분포한다. <a href="/wiki/%ED%95%9C%EA%B5%AD" title="한국">한국</a>에서는 <a href="/wiki/%ED%86%B5%EC%9D%BC%EC%8B%A0%EB%9D%BC" title="통일신라">통일신라</a> 시대에 널리 심었다는 기록이 있으며, 그 이전부터 심었던 것으로 보인다.</p>
<p><a href="/wiki/%EB%8C%80%ED%95%9C%EB%AF%BC%EA%B5%AD" title="대한민국">대한민국</a>에서 재배되는 주요 들깨 품종으로는 동글2호, 남천들깨, 늘보라들깨, 일엽들깨, 보라들깨 등이 있다.<sup class="reference" id="cite_ref-1"><a href="#cite_note-1">[1]</a></sup></p>
<p><a href="/wiki/%EC%B0%B8%EA%B9%A8" title="참깨">참깨</a>는 다른 <a href="/wiki/%EC%A2%85_(%EC%83%9D%EB%AC%BC%ED%95%99)" title="종 (생물학)">종</a>이며, <a href="/wiki/%EC%86%8C%EC%97%BD" title="소엽">소엽</a>은 같은 종에 속하는 다른 <a href="/wiki/%EC%95%84%EC%A2%85" title="아종">아종</a>이다.</p>

<h2><span class="mw-headline" id=".EC.9D.B4.EB.A6.84">이름</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%93%A4%EA%B9%A8&amp;action=edit&amp;section=1" title="부분 편집: 이름">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p><b>들깨</b>는 꿀풀과의 한해살이 풀과 그 씨를 함께 일컫는 말이다. <b>백소</b>(白蘇), <b>수임</b>(水荏), <b>야임</b>(野荏), <b>임자</b>(荏子)로 부르기도 한다.<sup class="reference" id="cite_ref-2"><a href="#cite_note-2">[2]</a></sup></p>
<p>들깨의 잎은 <b>깻잎</b>이라고 부르고, 들깨로 짠 기름은 <b>들기름</b>이라고 부른다.</p>
<p>들깨를 일본어로는 에고마(<span style="font-size: smaller;"><a href="/wiki/%EC%9D%BC%EB%B3%B8%EC%96%B4" title="일본어">일본어</a>:</span> <span lang="ja" xml:lang="ja">エゴマ/荏胡麻</span>)라 부르며, 중국어로는 바이쯔쑤(<small><a href="/wiki/%EC%A4%91%EA%B5%AD%EC%96%B4" title="중국어">중국어</a></small> <small><a href="/wiki/%EA%B0%84%EC%B2%B4%EC%9E%90" title="간체자">간체</a>:</small> <span lang="zh-hans" xml:lang="zh-hans">白紫苏</span>, <small><a href="/wiki/%EC%A0%95%EC%B2%B4%EC%9E%90" title="정체자">정체</a>:</small> <span lang="zh-hant" xml:lang="zh-hant">白紫蘇</span>, <small><a href="/wiki/%ED%95%9C%EC%96%B4_%EB%B3%91%EC%9D%8C" title="한어 병음">병음</a>:</small> <font face="Lucida Sans Unicode">báizǐsū</font>)라 부른다.</p>
<h2><span class="mw-headline" id=".EC.83.9D.EA.B9.80.EC.83.88">생김새</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%93%A4%EA%B9%A8&amp;action=edit&amp;section=2" title="부분 편집: 생김새">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>높이는 60~90<a href="/wiki/%EC%84%BC%ED%8B%B0%EB%AF%B8%ED%84%B0" title="센티미터">센티미터</a>이며 전체적으로 강한 향기를 풍긴다. <a href="/wiki/%EC%A4%84%EA%B8%B0" title="줄기">줄기</a>는 네모지며 곧게 자라고 긴 털이 있다. <a href="/wiki/%EC%9E%8E" title="잎">잎</a>은 마주나고 난상 원형으로 끝이 뾰족하며 밑부분은 둥글다. 길이 7~12센티미터, 너비 5~8센티미터로, 가장자리에 둔한 톱니가 있고 잎자루가 길다. 녹색이지만 때로는 뒷면에 자줏빛이 돈다. 8~9월에 피는 <a href="/wiki/%EA%BD%83" title="꽃">꽃</a>은 흰색이며 가지 끝과 원줄기 끝의 총상<a href="/wiki/%EA%BD%83%EC%B0%A8%EB%A1%80" title="꽃차례">꽃차례</a>에 달린다. 꽃받침은 길이 3~4밀리미터로, 위쪽의 것이 3개로 갈라지며 아래쪽 것은 더욱 길고 2개로 갈라지며 긴 털이 있다. 꽃부리는 길이 4~5밀리미터로서 하순이 약간 길다. 수술 4개 중 2개가 길다. <a href="/wiki/%EC%97%B4%EB%A7%A4" title="열매">열매</a>는 <a class="new" href="/w/index.php?title=%EB%B6%84%EA%B3%BC&amp;action=edit&amp;redlink=1" title="분과 (없는 문서)">분과</a>로, 꽃받침 안에 들어 있으며 둥글고 지름 2밑리미터 정도로서 겉에 그물 무늬가 있다.</p>
<h2><span class="mw-headline" id=".EC.9E.AC.EB.B0.B0_.ED.8A.B9.EC.84.B1">재배 특성</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%93%A4%EA%B9%A8&amp;action=edit&amp;section=3" title="부분 편집: 재배 특성">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>들깨를 자연 그대로 자라게 두면 <a href="/wiki/%EC%97%B4%EB%A7%A4" title="열매">열매</a>를 적게 맺어 수확량이 적다. 여름에 줄기를 밑에서 5센티미터 정도만 남기고 자르면 새 줄기가 자라고 가을에 열매를 더 많이 맺는다. 이때 잘라낸 줄기에서 잎을 수확할 수 있다.</p>
<p>잎은 여름부터 가을에 걸쳐 수확하며, 들깨는 가을에 과실이 성숙하면 풀포기 채로 채취해 <a href="/wiki/%EC%94%A8" title="씨">씨앗</a>을 떨어낸 다음 햇볕에 말린다.</p>
<h2><span class="mw-headline" id=".EC.93.B0.EC.9E.84.EC.83.88">쓰임새</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%93%A4%EA%B9%A8&amp;action=edit&amp;section=4" title="부분 편집: 쓰임새">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<h3><span class="mw-headline" id=".EB.93.A4.EA.B9.A8">들깨</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%93%A4%EA%B9%A8&amp;action=edit&amp;section=5" title="부분 편집: 들깨">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<div class="rellink boilerplate seealso"><img alt="" data-file-height="128" data-file-width="128" height="16" src="//upload.wikimedia.org/wikipedia/commons/thumb/e/ec/Crystal_Clear_app_xmag.svg/16px-Crystal_Clear_app_xmag.svg.png" srcset="//upload.wikimedia.org/wikipedia/commons/thumb/e/ec/Crystal_Clear_app_xmag.svg/24px-Crystal_Clear_app_xmag.svg.png 1.5x, //upload.wikimedia.org/wikipedia/commons/thumb/e/ec/Crystal_Clear_app_xmag.svg/32px-Crystal_Clear_app_xmag.svg.png 2x" width="16"/> <a href="/wiki/%EA%B9%A8" title="깨">깨</a> 문서를 참고하십시오.</div>
<p><a href="/wiki/%EB%93%A4%EA%B8%B0%EB%A6%84" title="들기름">들기름</a>을 짜서 쓰거나, 기름을 짜지 않고 <a href="/wiki/%EC%94%A8" title="씨">씨</a>를 볶아서 가루를 내 <a href="/wiki/%EC%A1%B0%EB%AF%B8%EB%A3%8C" title="조미료">양념</a>이나 고명으로 쓴다. <a class="new" href="/w/index.php?title=%EB%93%A4%EA%B9%A8%EC%97%BF&amp;action=edit&amp;redlink=1" title="들깨엿 (없는 문서)">들깨엿</a>을 만들어 겨울철 간식으로 먹기도 했다.</p>
<p>먹는 <a href="/wiki/%EB%93%A4%EA%B8%B0%EB%A6%84" title="들기름">들기름</a>은 주로 볶은 들깨로 짜며, 국이나 나물 등의 요리에 많이 쓰인다. 들깻가루는 국수, 국, 김치 등에 뿌려 먹는다. 또한 어묵에도 넣고, <a href="/wiki/%EC%83%90%EB%9F%AC%EB%93%9C" title="샐러드">샐러드</a>드레싱으로도 쓴다.</p>
<p>과거에는 <a href="/wiki/%EB%93%A4%EA%B8%B0%EB%A6%84" title="들기름">들기름</a>을 등잔 기름으로 쓰기도 하고, 종이를 들기름에 결어서 장판지로 쓰기도 했다. 들기름은 또한 <a href="/wiki/%EB%8F%84%EB%A3%8C" title="도료">페인트</a>, <a href="/wiki/%EB%B0%94%EB%8B%88%EC%8B%9C" title="바니시">바니시</a>, <a href="/wiki/%EB%A6%AC%EB%86%80%EB%A5%A8" title="리놀륨">리놀륨</a>, 인쇄용 <a href="/wiki/%EC%9E%89%ED%81%AC" title="잉크">잉크</a>, <a href="/wiki/%ED%8F%AC%EB%A7%88%EB%93%9C" title="포마드">포마드</a>, <a href="/wiki/%EB%B9%84%EB%88%84" title="비누">비누</a> 등의 원료로도 쓰인다. 원료로 쓰거나 물건을 겯는 데에 쓰는 들기름은 볶지 않은 들깨에서 짠다.</p>
<p><a href="/wiki/%EB%93%A4%EA%B8%B0%EB%A6%84" title="들기름">들기름</a>을 짜고 남은 <a href="/wiki/%EA%B9%BB%EB%AC%B5" title="깻묵">깻묵</a>은 <a href="/wiki/%EB%B9%84%EB%A3%8C" title="비료">비료</a>나 <a href="/wiki/%EB%A8%B9%EC%9D%B4" title="먹이">사료</a>로 쓰인다.</p>
<h3><span class="mw-headline" id=".EA.B9.BB.EC.9E.8E">깻잎</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%93%A4%EA%B9%A8&amp;action=edit&amp;section=6" title="부분 편집: 깻잎">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p><a href="/wiki/%ED%95%9C%EA%B5%AD" title="한국">한국</a>에서는 깻잎을 따서 <a href="/wiki/%EC%8C%88" title="쌈">쌈</a> 채소로 먹거나, <a href="/wiki/%EA%B0%84%EC%9E%A5" title="간장">간장</a>이나 <a href="/wiki/%EB%90%9C%EC%9E%A5" title="된장">된장</a>에 절여 <a href="/wiki/%EC%9E%A5%EC%95%84%EC%B0%8C" title="장아찌">장아찌</a>를 담가 먹는다. <a href="/wiki/%EA%B9%80%EC%B9%98" title="김치">김치</a>를 담가 먹거나 <a href="/wiki/%EB%82%98%EB%AC%BC" title="나물">나물</a>로 먹기도 한다. <a href="/wiki/%EA%B3%A0%EA%B8%B0" title="고기">고기</a> 요리에 <a href="/wiki/%ED%97%88%EB%B8%8C_(%EC%8B%9D%EB%AC%BC)" title="허브 (식물)">허브</a>로 쓰이기도 하고, <a href="/wiki/%EB%B3%B6%EC%9D%8C%EB%B0%A5" title="볶음밥">볶음밥</a> 등에도 들어간다. <a href="/wiki/%ED%93%A8%EC%A0%84_%EC%9A%94%EB%A6%AC" title="퓨전 요리">퓨전 양식</a>에서 깻잎이 <a href="/wiki/%EB%B0%94%EC%A7%88" title="바질">바질</a> 등을 대체하기도 한다.</p>
