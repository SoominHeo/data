<p><b>등처눌</b>(鄧處訥, ? ~ <a href="/wiki/894%EB%85%84" title="894년">894년</a>)은 <a href="/wiki/%EC%A4%91%EA%B5%AD" title="중국">중국</a> <a href="/wiki/%EB%8B%B9%EB%82%98%EB%9D%BC" title="당나라">당나라</a> 말기에 활약했던 <a href="/wiki/%EA%B5%B0%EB%B2%8C" title="군벌">군벌</a>로, <a href="/wiki/893%EB%85%84" title="893년">893년</a>부터 <a href="/wiki/894%EB%85%84" title="894년">894년</a>까지 절도사의 <a href="/wiki/%EC%8B%A0%EB%B6%84" title="신분">신분</a>으로서 무안군(武安軍, 본거지는 지금의 <a href="/wiki/%ED%9B%84%EB%82%9C_%EC%84%B1" title="후난 성">후난 성</a> <a href="/wiki/%EC%B0%BD%EC%82%AC_%EC%8B%9C" title="창사 시">창사 시</a>) 번진(藩鎭)을 지배하였다. <a href="/wiki/894%EB%85%84" title="894년">894년</a> 그는 <a href="/wiki/%EC%9C%A0%EA%B1%B4%EB%B4%89" title="유건봉">유건봉</a>에게 패하여 죽임을 당했고, 유건봉은 무안군 번진을 점령하였다. <a href="/wiki/%EC%9E%90_(%EC%9D%B4%EB%A6%84)" title="자 (이름)">자</a>는 <b>충온</b>(沖韞).</p>

<h2><span class="mw-headline" id=".EC.83.9D.EC.95.A0">생애</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%93%B1%EC%B2%98%EB%88%8C&amp;action=edit&amp;section=1" title="부분 편집: 생애">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<h3><span class="mw-headline" id=".EC.B6.9C.EC.8B.A0_.EB.B0.B0.EA.B2.BD_.EB.B0.8F_.EC.B4.88.EA.B8.B0_.EA.B2.BD.EB.A0.A5">출신 배경 및 초기 경력</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%93%B1%EC%B2%98%EB%88%8C&amp;action=edit&amp;section=2" title="부분 편집: 출신 배경 및 초기 경력">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>등처눌은 소주(邵州, 지금의 <a href="/wiki/%ED%9B%84%EB%82%9C_%EC%84%B1" title="후난 성">후난 성</a> <a href="/wiki/%EC%82%AC%EC%98%A4%EC%96%91_%EC%8B%9C" title="사오양 시">사오양 시</a>) 출신으로, 언제 태어났는지는 알려진 바 없다. 젊었을 때, 그는 강서도(江西道, 본거지는 지금의 <a href="/wiki/%EC%9E%A5%EC%8B%9C_%EC%84%B1" title="장시 성">장시 성</a> <a href="/wiki/%EB%82%9C%EC%B0%BD_%EC%8B%9C" title="난창 시">난창 시</a>)에서 온 장수였던 <a href="/wiki/%EB%AF%BC%EC%9A%B1_(%EB%8B%B9%EB%82%98%EB%9D%BC)" title="민욱 (당나라)">민욱</a>의 휘하 사병이 되어 <a href="/wiki/%EC%95%88%EB%82%A8" title="안남">안남</a>(安南, 지금의 <a class="mw-redirect" href="/wiki/%EB%B6%81%EB%B2%A0%ED%8A%B8%EB%82%A8" title="북베트남">북베트남</a>)에 배치되었다.<sup class="reference" id="cite_ref-NBT186_1-0"><a href="#cite_note-NBT186-1">[1]</a></sup></p>
<h3><span class="mw-headline" id=".EB.AF.BC.EC.9A.B1_.ED.9C.98.ED.95.98.EC.97.90.EC.84.9C"><a href="/wiki/%EB%AF%BC%EC%9A%B1_(%EB%8B%B9%EB%82%98%EB%9D%BC)" title="민욱 (당나라)">민욱</a> 휘하에서</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%93%B1%EC%B2%98%EB%88%8C&amp;action=edit&amp;section=3" title="부분 편집: 민욱 휘하에서">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p><a href="/wiki/881%EB%85%84" title="881년">881년</a> 민욱과 그의 병사들이 안남에서 돌아오는 길에 담주(潭州, 지금의 <a href="/wiki/%ED%9B%84%EB%82%9C_%EC%84%B1" title="후난 성">후난 성</a> <a href="/wiki/%EC%B0%BD%EC%82%AC_%EC%8B%9C" title="창사 시">창사 시</a>)를 지나게 되자, 민욱은 호남관찰사(湖南觀察使, 본거지는 담주)를 추방하고 호남도를 점령하였다.<sup class="reference" id="cite_ref-ZZTJ254_2-0"><a href="#cite_note-ZZTJ254-2">[2]</a></sup> 그 직후, 민욱은 유후(留後)를 자칭하였고, 등처눌을 소주자사(邵州刺史)에 임명하였다.<sup class="reference" id="cite_ref-NBT186_1-1"><a href="#cite_note-NBT186-1">[1]</a></sup></p>
<h3><span class="mw-headline" id=".EB.AC.B4.EC.95.88.EA.B5.B0_.EB.B2.88.EC.A7.84.EC.9D.84_.ED.83.88.EC.B7.A8.ED.95.98.EB.8B.A4">무안군 번진을 탈취하다</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%93%B1%EC%B2%98%EB%88%8C&amp;action=edit&amp;section=4" title="부분 편집: 무안군 번진을 탈취하다">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p><a href="/wiki/886%EB%85%84" title="886년">886년</a> 담주가 형주자사(衡州刺史, 지금의 <a href="/wiki/%ED%9B%84%EB%82%9C_%EC%84%B1" title="후난 성">후난 성</a> <a href="/wiki/%ED%97%9D%EC%96%91_%EC%8B%9C" title="헝양 시">헝양 시</a>) <a href="/wiki/%EC%A3%BC%EC%95%85" title="주악">주악</a>의 공격을 받고 있었을 무렵, 민욱은 채주(蔡州, 지금의 <a href="/wiki/%ED%97%88%EB%82%9C_%EC%84%B1" title="허난 성">허난 성</a> <a href="/wiki/%EC%A3%BC%EB%A7%88%EB%8E%AC_%EC%8B%9C" title="주마뎬 시">주마뎬 시</a>) 일대를 중심으로 한 새로운 <a class="mw-redirect" href="/wiki/%EB%82%98%EB%9D%BC" title="나라">나라</a>의 <a href="/wiki/%ED%99%A9%EC%A0%9C" title="황제">황제</a>를 자칭했던 <a href="/wiki/%EA%B5%B0%EB%B2%8C" title="군벌">군벌</a> <a class="new" href="/w/index.php?title=%EC%A7%84%EC%A2%85%EA%B6%8C&amp;action=edit&amp;redlink=1" title="진종권 (없는 문서)">진종권</a>의 휘하 장수 황호(黃皓)를 맞이하여, 주악의 공격에 맞서 자신을 도와 지키게 하였다. 하지만, 황호는 담주성에 입성하자마자, 민욱을 암살해 버렸다. 그 후, 주악은 황호를 격파, 살해하고 <a href="/wiki/%ED%9D%A0%ED%99%94%EA%B5%B0_%EC%A0%88%EB%8F%84%EC%82%AC" title="흠화군 절도사">흠화군</a>(欽化軍. 당시에는 이 군호(軍號)로 명명되어 있었다) 번진을 점령하였으며, 최종적으로는 그곳의 절도사(節度使)로 임명되었다.<sup class="reference" id="cite_ref-ZZTJ256_3-0"><a href="#cite_note-ZZTJ256-3">[3]</a></sup></p>
<p>민욱의 죽음을 전해들은 등처눌은 그를 위하여 주악을 비난하면서, 비통하게 민욱을 애도하며 제장들에게 말하였다.<sup class="reference" id="cite_ref-ZZTJ259_4-0"><a href="#cite_note-ZZTJ259-4">[4]</a></sup></p>
<table align="center" style="border-collapse:collapse; border-style:none; background-color:transparent;">
<tr>
<td style="color:#90B5F0; width: 20px; vertical-align: top; font-size:40px;font-family:Georgia;font-weight:bold;text-align:left;">“</td>
<td style="padding:10px; vertical-align: top;">나와 공들은 복야(僕射, 민욱)의 큰 은혜를 받았소. 그런데 주악은 아무런 이유도 없이 그를 죽였소. 나는 공들과 함께 복야의 원수를 갚고자 우리 주의 힘을 다하고 싶소. 우리 할 수 없겠소?<sup class="reference" id="cite_ref-ZZTJ259_4-1"><a href="#cite_note-ZZTJ259-4">[4]</a></sup></td>
<td style="color:#A0B7F2; width: 20px; vertical-align: bottom; font-size:40px;font-family:Georgia;font-weight:bold;text-align:right;">”</td>
</tr>
</table>
<p>제장들은 그에게 동의하였다. 그는 7년간 자신의 <a href="/wiki/%EA%B5%B0%EB%8C%80" title="군대">군대</a>를 조련시켰고, <a href="/wiki/893%EB%85%84" title="893년">893년</a> 낭주자사(朗州刺史, 지금의 <a href="/wiki/%ED%9B%84%EB%82%9C_%EC%84%B1" title="후난 성">후난 성</a> <a href="/wiki/%EC%B0%BD%EB%8D%94_%EC%8B%9C" title="창더 시">창더 시</a>) <a class="new" href="/w/index.php?title=%EB%87%8C%EB%A7%8C&amp;action=edit&amp;redlink=1" title="뇌만 (없는 문서)">뇌만</a>과 결탁하여 담주를 공격하여 그곳을 함락시켰다. 그는 주악을 죽이고 무안군(당시 흠화군은 무안군으로 군호가 다시 명명되어 있었기 때문이었다) 유후를 자칭하였다. <a href="/wiki/894%EB%85%84" title="894년">894년</a> 봄, <a href="/wiki/%EB%8B%B9_%EC%86%8C%EC%A2%85" title="당 소종">당 소종</a>은 그를 정식으로 절도사에 임명하였다.<sup class="reference" id="cite_ref-ZZTJ259_4-2"><a href="#cite_note-ZZTJ259-4">[4]</a></sup></p>
<h3><span class="mw-headline" id=".ED.8C.A8.EB.A7.9D">패망</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%93%B1%EC%B2%98%EB%88%8C&amp;action=edit&amp;section=5" title="부분 편집: 패망">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>그러나 그 해 연말에, <a href="/wiki/%EC%9C%A0%EA%B1%B4%EB%B4%89" title="유건봉">유건봉</a>과 <a href="/wiki/%EB%A7%88%EC%9D%80" title="마은">마은</a>이 이끄는 군대가 무안군에 다가왔다. 등처눌은 그의 장수인 소주지휘사(邵州指揮使) 장훈(蔣勛)과 등계숭(鄧繼崇)을 파견하여 소주 군대와 함께 용회관(龍回關, 지금의 <a href="/wiki/%EC%82%AC%EC%98%A4%EC%96%91_%EC%8B%9C" title="사오양 시">사오양 시</a>에 있었다)에서 공격을 저지하게 하였다. 마은은 장훈과 등계숭에게 유건봉의 성공은 예고되었으며, 저항할 수 없다고 설득하였고, 그들은 군대를 해산시켰다. 마은은 그들이 입었던 <a href="/wiki/%EA%B5%B0%EB%B3%B5" title="군복">군복</a>들을 접수하고, 곧장 담주를 향해 진군하였다. 담주 수비군은 이들이 소주에서 온 병사들이라 믿고 아무런 방비도 하지 않았다. 마은은 곧장 성중과 군부(軍府)로 들어가서, 마침 잔치를 벌이고 있었던 등처눌을 사로잡아 참수하고, 유건봉에게 번진을 점령하게 하였다.<sup class="reference" id="cite_ref-ZZTJ259_4-3"><a href="#cite_note-ZZTJ259-4">[4]</a></sup></p>
