<p><b>디니야르 리나토비치 빌랼레트디노프</b>(<span style="font-size: smaller;"><a href="/wiki/%EB%9F%AC%EC%8B%9C%EC%95%84%EC%96%B4" title="러시아어">러시아어</a>:</span> <span lang="ru" xml:lang="ru">Динияр Ринатович Билялетдинов​</span>, <span style="font-size: smaller;"><a href="/wiki/%ED%83%80%ED%83%80%EB%A5%B4%EC%96%B4" title="타타르어">타타르어</a>:</span> <span lang="tt" xml:lang="tt">Динияр Ринат улы Билалетдинев</span>, <a href="/wiki/1985%EB%85%84" title="1985년">1985년</a> <a href="/wiki/2%EC%9B%94_27%EC%9D%BC" title="2월 27일">2월 27일</a> ~, <a href="/wiki/%EB%AA%A8%EC%8A%A4%ED%81%AC%EB%B0%94" title="모스크바">모스크바</a>)는 <a href="/wiki/%EB%9F%AC%EC%8B%9C%EC%95%84" title="러시아">러시아</a>의 축구 선수이다. 현재 <a href="/wiki/%EB%9F%AC%EC%8B%9C%EC%95%84_%ED%94%84%EB%A6%AC%EB%AF%B8%EC%96%B4%EB%A6%AC%EA%B7%B8" title="러시아 프리미어리그">러시아 프리미어리그</a>의 <a href="/wiki/FC_%EC%8A%A4%ED%8C%8C%EB%A5%B4%ED%83%80%ED%81%AC_%EB%AA%A8%EC%8A%A4%ED%81%AC%EB%B0%94" title="FC 스파르타크 모스크바">FC 스파르타크 모스크바</a> 소속으로, 포지션은 <a class="mw-redirect" href="/wiki/%EA%B3%B5%EA%B2%A9%ED%98%95_%EB%AF%B8%EB%93%9C%ED%95%84%EB%8D%94" title="공격형 미드필더">공격형 미드필더</a>이다.</p>
<h2><span class="mw-headline" id=".EC.84.A0.EC.88.98_.EC.83.9D.ED.99.9C">선수 생활</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%94%94%EB%8B%88%EC%95%BC%EB%A5%B4_%EB%B9%8C%EB%9E%BC%EB%A0%88%ED%8A%B8%EB%94%94%EB%85%B8%ED%94%84&amp;action=edit&amp;section=1" title="부분 편집: 선수 생활">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p><a href="/wiki/%EB%AA%A8%EC%8A%A4%ED%81%AC%EB%B0%94" title="모스크바">모스크바</a>에서 태어난 빌랼레트디노프는 고향의 축구 클럽 중 하나인 <a class="mw-redirect" href="/wiki/%EB%A1%9C%EC%BD%94%EB%AA%A8%ED%8B%B0%EB%B8%8C_%EB%AA%A8%EC%8A%A4%ED%81%AC%EB%B0%94" title="로코모티브 모스크바">로코모티브 모스크바</a>에서 선수 생활을 시작했다. 1군 무대에 데뷔한 <a href="/wiki/2004%EB%85%84" title="2004년">2004년</a>에 러시아 리그 올해의 유망주 상을 수상하면서 두각을 나타낸 빌랼레트디노프는 지속적인 기량 성장을 거듭해 팀의 주축 선수가 되었고, <a href="/wiki/%EB%9F%AC%EC%8B%9C%EC%95%84_%EC%B6%95%EA%B5%AC_%EA%B5%AD%EA%B0%80%EB%8C%80%ED%91%9C%ED%8C%80" title="러시아 축구 국가대표팀">러시아 축구 국가대표팀</a>에 발탁되어 현재까지도 국가 대표팀의 주전 선수로 자리매김 하고 있다. 그러던 중 <a href="/wiki/2009%EB%85%84" title="2009년">2009년</a> <a href="/wiki/8%EC%9B%94" title="8월">8월</a> 이적 시장이 닫히기 며칠 전에 <a href="/wiki/%EC%97%90%EB%B2%84%ED%84%B4_FC" title="에버턴 FC">에버턴 FC</a>와 5년 계약을 맺고 팀을 옮기게 되어 현재에 이르고 있다.</p>
