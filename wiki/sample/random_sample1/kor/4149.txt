<p><b>디미트로브그라드</b>(<span style="font-size: smaller;"><a href="/wiki/%EC%84%B8%EB%A5%B4%EB%B9%84%EC%95%84%EC%96%B4" title="세르비아어">세르비아어</a>:</span> <span lang="sr" xml:lang="sr">Димитровград, Dimitrovgrad</span>, <span style="font-size: smaller;"><a href="/wiki/%EB%B6%88%EA%B0%80%EB%A6%AC%EC%95%84%EC%96%B4" title="불가리아어">불가리아어</a>:</span> <span lang="bg" xml:lang="bg">Цариброд</span> <small>차리브로드</small>)는 <a href="/wiki/%EC%84%B8%EB%A5%B4%EB%B9%84%EC%95%84" title="세르비아">세르비아</a> <a href="/wiki/%ED%94%BC%EB%A1%9C%ED%8A%B8_%EA%B5%AC" title="피로트 구">피로트 구</a>에 위치한 도시이다. "디미트로브그라드"는 <a href="/wiki/%EB%B6%88%EA%B0%80%EB%A6%AC%EC%95%84" title="불가리아">불가리아</a>의 <a href="/wiki/%EA%B3%B5%EC%82%B0%EC%A3%BC%EC%9D%98" title="공산주의">공산주의</a> 지도자인 <a href="/wiki/%EA%B2%8C%EC%98%A4%EB%A5%B4%EA%B8%B0_%EB%94%94%EB%AF%B8%ED%8A%B8%EB%A1%9C%ED%94%84" title="게오르기 디미트로프">게오르기 디미트로프</a>의 이름을 따서 붙여진 이름이다.</p>
<p>디미트로브그라드는 원래 <a href="/wiki/%EB%B6%88%EA%B0%80%EB%A6%AC%EC%95%84" title="불가리아">불가리아</a>의 영토였지만 <a href="/wiki/1919%EB%85%84" title="1919년">1919년</a> <a href="/wiki/11%EC%9B%94_27%EC%9D%BC" title="11월 27일">11월 27일</a>에 체결된 <a href="/wiki/%EB%87%8C%EC%9D%B4_%EC%A1%B0%EC%95%BD" title="뇌이 조약">뇌이 조약</a>에 따라 <a href="/wiki/%EC%9C%A0%EA%B3%A0%EC%8A%AC%EB%9D%BC%EB%B9%84%EC%95%84_%EC%99%95%EA%B5%AD" title="유고슬라비아 왕국">유고슬라비아 왕국</a>의 영토가 되었다.</p>
