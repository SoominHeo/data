<p><b>디에고 트리스탄 에레라</b>(<span style="font-size: smaller;"><a href="/wiki/%EC%8A%A4%ED%8E%98%EC%9D%B8%EC%96%B4" title="스페인어">스페인어</a>:</span> <span lang="es" xml:lang="es">Diego Tristán Herrera‎‎</span>, 1976년 1월 5일 ~ )는 <a href="/wiki/%EC%8A%A4%ED%8E%98%EC%9D%B8" title="스페인">스페인</a>의 은퇴한 <a href="/wiki/%EC%B6%95%EA%B5%AC" title="축구">축구</a> 선수이며 선수 시절 포지션은 <a href="/wiki/%EA%B3%B5%EA%B2%A9%EC%88%98" title="공격수">공격수</a>이다. <a class="mw-redirect" href="/wiki/%ED%94%84%EB%A6%AC%EB%A9%94%EB%9D%BC%EB%A6%AC%EA%B0%80" title="프리메라리가">프리메라리가</a>에서 통산 227경기에 나서 95골을 기록하였고, <a href="/wiki/2002%EB%85%84_FIFA_%EC%9B%94%EB%93%9C%EC%BB%B5" title="2002년 FIFA 월드컵">2002년 월드컵</a>에서 활약하였다.</p>

<h2><span class="mw-headline" id=".EC.B6.95.EA.B5.AC.EC.9D.B8_.EA.B2.BD.EB.A0.A5">축구인 경력</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%94%94%EC%97%90%EA%B3%A0_%ED%8A%B8%EB%A6%AC%EC%8A%A4%ED%83%84&amp;action=edit&amp;section=1" title="부분 편집: 축구인 경력">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<h3><span class="mw-headline" id=".ED.81.B4.EB.9F.BD_.EA.B2.BD.EB.A0.A5">클럽 경력</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%94%94%EC%97%90%EA%B3%A0_%ED%8A%B8%EB%A6%AC%EC%8A%A4%ED%83%84&amp;action=edit&amp;section=2" title="부분 편집: 클럽 경력">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p><a href="/wiki/%EB%A0%88%EC%95%8C_%EB%B2%A0%ED%8B%B0%EC%8A%A4" title="레알 베티스">베티스</a>의 유소년 팀과 B팀을 거쳐 <a href="/wiki/RCD_%EB%A7%88%EC%9A%94%EB%A5%B4%EC%B9%B4" title="RCD 마요르카">마요르카</a>의 B팀으로 이적하였고, 1999년 9월 20일 열린 <a href="/wiki/CD_%EB%88%84%EB%A7%8C%EC%8B%9C%EC%95%84" title="CD 누만시아">누만시아</a>와의 1999-2000 프리메라리가 경기에서 1군 팀 데뷔전을 가졌다. 이후 총 리그 35경기에 나서 18골을 기록하는 좋은 활약으로 데뷔 시즌을 마무리하였고, 이 활약을 바탕으로 <a href="/wiki/%EB%A0%88%EC%95%8C_%EB%A7%88%EB%93%9C%EB%A6%AC%EB%93%9C_CF" title="레알 마드리드 CF">레알 마드리드</a> 이적을 추진하였으나 협상 도중 레알 마드리드의 회장이 교체되며 무산되었고, 디펜딩 챔피언이었던 <a class="mw-redirect" href="/wiki/%EB%8D%B0%ED%8F%AC%EB%A5%B4%ED%8B%B0%EB%B3%B4_%EB%9D%BC_%EC%BD%94%EB%A3%A8%EB%83%90" title="데포르티보 라 코루냐">데포르티보 라 코루냐</a>로 이적하였다.</p>
<p>데포르티보에서 첫 시즌 리그 29경기에서 19골을 기록하는 좋은 활약을 펼쳤다. 2번째 시즌인 2001-02 시즌엔 챔피언스리그에서 6골 코파 델 레이에서 5골을 기록한 것 뿐만 아니라 리그에서는 21골을 기록하며 리그 득점왕에 올랐다. 2002-03 시즌엔 발목 부상을 입어 그 여파로 리그 23경기 출전에 9골을 기록하여 지난 시즌에 비해 부진한 모습을 보였다. 2003-04 시즌엔 경쟁자 <a href="/wiki/%EB%A1%9C%EC%9D%B4_%EB%A7%88%EC%B9%B4%EC%9D%B4" title="로이 마카이">로이 마카이</a>가 이적하였으나 컨디션 회복이 더뎌 출전한 33경기의 리그 경기 중 10경기에서만 풀 타임을 소화하였다.</p>
<p>2006년 여름 팀을 떠난 후 <a href="/wiki/%EB%B3%BC%ED%84%B4_%EC%9B%90%EB%8D%94%EB%9F%AC%EC%8A%A4_FC" title="볼턴 원더러스 FC">볼턴</a> 등이 관심을 보였으나<sup class="reference" id="cite_ref-1"><a href="#cite_note-1">[1]</a></sup> 전 소속팀 마요르카로 복귀하였다. 하지만 2007년 1월, 반 시즌 만에 팀을 떠났고 같은 해 7월 <a href="/wiki/AS_%EB%A6%AC%EB%B3%B4%EB%A5%B4%EB%85%B8_%EC%B9%BC%EC%B4%88" title="AS 리보르노 칼초">리보르노</a>에 입단하였다. 하지만 21경기에 나서 1골을 기록하는 부진한 활약을 펼쳤고, 팀은 <a href="/wiki/%EC%84%B8%EB%A6%AC%EC%97%90_B" title="세리에 B">세리에 B</a>로 강등되었다.</p>
<p>2008년 9월 <a class="mw-redirect" href="/wiki/%EC%9B%A8%EC%8A%A4%ED%8A%B8_%ED%96%84_%EC%9C%A0%EB%82%98%EC%9D%B4%ED%8B%B0%EB%93%9C_FC" title="웨스트 햄 유나이티드 FC">웨스트 햄</a>에서 입단 테스트를 받았고 10월 14일 정식 계약을 체결하였다. 12월 28일 <a class="mw-redirect" href="/wiki/%EC%8A%A4%ED%86%A0%ED%81%AC_%EC%8B%9C%ED%8B%B0" title="스토크 시티">스토크 시티</a>를 상대로 데뷔골을 기록하였지만 1시즌 만에 팀을 떠났다.</p>
<p>2009년 7월 24일 그의 고향인 <a href="/wiki/%EC%95%88%EB%8B%AC%EB%A3%A8%EC%8B%9C%EC%95%84_%EC%A7%80%EB%B0%A9" title="안달루시아 지방">안달루시아 지방</a>의 팀인 <a href="/wiki/%EC%B9%B4%EB%94%94%EC%8A%A4_CF" title="카디스 CF">카디스</a>에 자유 계약으로 입단하였다. 29경기에 나서 8골을 기록하였지만 팀은 3부 리그로 강등되었고, 트리스탄도 은퇴를 선언하였다.</p>
<h3><span class="mw-headline" id=".EB.8C.80.ED.91.9C.ED.8C.80_.EA.B2.BD.EB.A0.A5">대표팀 경력</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%94%94%EC%97%90%EA%B3%A0_%ED%8A%B8%EB%A6%AC%EC%8A%A4%ED%83%84&amp;action=edit&amp;section=3" title="부분 편집: 대표팀 경력">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>2001년 <a href="/wiki/%EB%B3%B4%EC%8A%A4%EB%8B%88%EC%95%84_%ED%97%A4%EB%A5%B4%EC%B2%B4%EA%B3%A0%EB%B9%84%EB%82%98_%EC%B6%95%EA%B5%AC_%EA%B5%AD%EA%B0%80%EB%8C%80%ED%91%9C%ED%8C%80" title="보스니아 헤르체고비나 축구 국가대표팀">보스니아 헤르체고비나</a>와의 <a href="/wiki/2002%EB%85%84_FIFA_%EC%9B%94%EB%93%9C%EC%BB%B5_%EC%98%88%EC%84%A0" title="2002년 FIFA 월드컵 예선">2002년 FIFA 월드컵 예선</a> 경기를 앞둔 <a href="/wiki/%EC%8A%A4%ED%8E%98%EC%9D%B8_%EC%B6%95%EA%B5%AC_%EA%B5%AD%EA%B0%80%EB%8C%80%ED%91%9C%ED%8C%80" title="스페인 축구 국가대표팀">스페인 국가대표팀</a>에 발탁되어 데뷔전을 치렀고, 이 경기에서 데뷔골을 기록하였다. <a href="/wiki/2002%EB%85%84_FIFA_%EC%9B%94%EB%93%9C%EC%BB%B5" title="2002년 FIFA 월드컵">2002년 월드컵</a> 본선 무대에 등번호 10번을 달고 출전하였지만 부상으로 출전 기회를 잡지 못하였다.</p>
