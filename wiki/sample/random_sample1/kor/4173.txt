<p><b>디오니시오스 카스다글리스</b>(<span style="font-size: smaller;"><a href="/wiki/%EA%B7%B8%EB%A6%AC%EC%8A%A4%EC%96%B4" title="그리스어">그리스어</a>:</span> <span lang="el" xml:lang="el">Διονύσιος Κάσδαγλης</span>, <a href="/wiki/1880%EB%85%84" title="1880년">1880년</a> ~ ?)는 <a href="/wiki/%EA%B7%B8%EB%A6%AC%EC%8A%A4" title="그리스">그리스계</a> <a href="/wiki/%EC%9D%B4%EC%A7%91%ED%8A%B8" title="이집트">이집트인</a> <a href="/wiki/%ED%85%8C%EB%8B%88%EC%8A%A4" title="테니스">테니스</a> 선수이다. 그는 <a href="/wiki/%EC%95%84%ED%85%8C%EB%84%A4" title="아테네">아테네</a>에서 열린 <a href="/wiki/1896%EB%85%84_%ED%95%98%EA%B3%84_%EC%98%AC%EB%A6%BC%ED%94%BD" title="1896년 하계 올림픽">1896년 하계 올림픽</a>에 참가했다.</p>

<div class="thumbcaption">

1896년 올림픽 단식 경기 결승, 카스다글리스는 이 경기에서 볼랜드한테 졌다.</div>

</div>
<p>이집트의 유일한 참가 선수인 카스다글리스는 단식과 복식에서 모두 결승에 올랐다. 단식경기에서는 1라운드에서 <a href="/wiki/%ED%94%84%EB%9E%91%EC%8A%A4" title="프랑스">프랑스</a>의 드페르를, 2라운드에서는 <a href="/wiki/%EA%B7%B8%EB%A6%AC%EC%8A%A4" title="그리스">그리스</a>의 <a class="new" href="/w/index.php?title=%EC%BD%98%EC%8A%A4%ED%83%84%ED%8B%B0%EB%85%B8%EC%8A%A4_%EC%95%84%ED%81%AC%EB%A1%9C%ED%86%A0%ED%92%80%EB%A1%9C%EC%8A%A4&amp;action=edit&amp;redlink=1" title="콘스탄티노스 아크로토풀로스 (없는 문서)">콘스탄티노스 아크로토풀로스</a>, 준결승에서는 <a href="/wiki/%ED%97%9D%EA%B0%80%EB%A6%AC" title="헝가리">헝가리</a>의 <a class="mw-redirect" href="/wiki/%ED%84%B0%ED%8D%BC%EB%B9%84%EC%B8%A0%EC%A0%80_%EB%AA%B8%EC%B9%A0%EB%A1%9C" title="터퍼비츠저 몸칠로">터퍼비츠저 몸칠로</a>를 꺾고 결승에 진출했다. 결승에서는 <a href="/wiki/%EC%98%81%EA%B5%AD" title="영국">영국</a>의 <a href="/wiki/%EC%A1%B4_%ED%94%BC%EC%9A%B0%EC%8A%A4_%EB%B3%BC%EB%9E%9C%EB%93%9C" title="존 피우스 볼랜드">존 피우스 볼랜드</a>와 맞붙었으며, 결승전에서는 볼랜드가 최고의 선수라는 것을 입증했다. 카스다글리스는 2위를 차지했다. 이 메달로 <a class="mw-redirect" href="/wiki/IOC" title="IOC">IOC</a>는 카스다글리스를 그리스인으로 해놓았다.</p>
<p>복식경기에서는 <a href="/wiki/%EA%B7%B8%EB%A6%AC%EC%8A%A4" title="그리스">그리스</a>의 <a href="/wiki/%EB%8D%B0%EB%A9%94%ED%8A%B8%EB%A6%AC%EC%98%A4%EC%8A%A4_%ED%8E%98%ED%8A%B8%EB%A1%9C%EC%BD%94%ED%82%A4%EB%85%B8%EC%8A%A4" title="데메트리오스 페트로코키노스">데메트리오스 페트로코키노스</a>와 짝을 이뤄서 <a class="mw-redirect" href="/wiki/1896%EB%85%84_%ED%95%98%EA%B3%84_%EC%98%AC%EB%A6%BC%ED%94%BD_%EB%8B%A4%EA%B5%AD%EC%A0%81%ED%98%BC%EC%84%B1%ED%8C%80_%EC%84%A0%EC%88%98%EB%8B%A8" title="1896년 하계 올림픽 다국적혼성팀 선수단">혼성팀 자격</a>으로 참가했다.<sup class="reference" id="cite_ref-mixed_1-0"><a href="#cite_note-mixed-1">[1]</a></sup> 1라운드에서는 <a href="/wiki/%EA%B7%B8%EB%A6%AC%EC%8A%A4" title="그리스">그리스</a>의 <a href="/wiki/%EC%97%90%EB%B0%98%EA%B2%94%EB%A1%9C%EC%8A%A4_%EB%9E%84%EB%A6%AC%EC%8A%A4" title="에반겔로스 랄리스">에반겔로스 랄리스</a>와 <a href="/wiki/%EC%BD%98%EC%8A%A4%ED%83%84%ED%8B%B0%EB%85%B8%EC%8A%A4_%ED%8C%8C%EC%8A%A4%ED%8C%8C%ED%8B%B0%EC%8A%A4" title="콘스탄티노스 파스파티스">콘스탄티노스 파스파티스</a>를 제압했으며, 준결승에서는 <a href="/wiki/%EC%98%A4%EC%8A%A4%ED%8A%B8%EB%A0%88%EC%9D%BC%EB%A6%AC%EC%95%84" title="오스트레일리아">오스트레일리아</a>의 <a href="/wiki/%EC%97%90%EB%93%9C%EC%9C%88_%ED%94%8C%EB%9E%99" title="에드윈 플랙">에드윈 플랙</a>과 <a href="/wiki/%EC%98%81%EA%B5%AD" title="영국">영국</a>의 <a href="/wiki/%EC%A1%B0%EC%A7%80_S._%EB%A1%9C%EB%B2%84%ED%8A%B8%EC%8A%A8" title="조지 S. 로버트슨">조지 S. 로버트슨</a> 조를 무찔렀다. 결승에서는 또다시 볼랜드와 그의 짝인 <a href="/wiki/%EB%8F%85%EC%9D%BC" title="독일">독일</a>의 <a class="mw-redirect" href="/wiki/%ED%94%84%EB%A6%AC%EB%93%9C%EB%A6%AC%ED%9E%88_%ED%8A%B8%EB%9D%BC%EC%9A%B4" title="프리드리히 트라운">프리드리히 트라운</a>과 만났다. 카스다글리스와 페트로코키노스는 이 경기에서 져서 은메달에 만족해야 했다.</p>
