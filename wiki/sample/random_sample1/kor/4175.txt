<h1 class="firstHeading" id="firstHeading" lang="ko">디옥시리보뉴클레오타이드</h1>
<div class="mw-body-content" id="bodyContent">

<div class="mw-content-ltr" dir="ltr" id="mw-content-text" lang="ko"><script>function mfTempOpenSection(id){var block=document.getElementById("mf-section-"+id);block.className+=" open-block";block.previousSibling.className+=" open-block";}</script><p><b>디옥시리보뉴클레오타이드</b>(<span style="font-size: smaller;"><a href="/wiki/%EC%98%81%EC%96%B4" title="영어">영어</a>:</span> <span lang="en" xml:lang="en">Deoxyribonucleotide</span>)는 <a href="/wiki/DNA" title="DNA">DNA</a>를 구성하는 <a href="/wiki/%EB%89%B4%ED%81%B4%EB%A0%88%EC%98%A4%ED%83%80%EC%9D%B4%EB%93%9C" title="뉴클레오타이드">뉴클레오타이드</a>가 단독으로 존재하는 형태이다.<sup class="reference" id="cite_ref-P48_1-0"><a href="#cite_note-P48-1">[1]</a></sup></p>
<h2><span class="mw-headline" id=".EA.B5.AC.EC.A1.B0.EC.99.80_.EC.97.AD.ED.95.A0">구조와 역할</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%94%94%EC%98%A5%EC%8B%9C%EB%A6%AC%EB%B3%B4%EB%89%B4%ED%81%B4%EB%A0%88%EC%98%A4%ED%83%80%EC%9D%B4%EB%93%9C&amp;action=edit&amp;section=1" title="부분 편집: 구조와 역할">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>디옥시리보뉴클레오타이드는 <a href="/wiki/%EB%94%94%EC%98%A5%EC%8B%9C%EB%A6%AC%EB%B3%B4%EC%8A%A4" title="디옥시리보스">디옥시리보스</a>에 <a href="/wiki/%ED%95%B5%EC%97%BC%EA%B8%B0" title="핵염기">핵염기</a>와 <a href="/wiki/%EC%9D%B8%EC%82%B0" title="인산">인산</a>이 결합되어 있는 구조로 되어 있다. 인산의 개수에 따라 디옥시뉴클레오타이드일인산, 디옥시뉴클레오타이드이인산, 디옥시뉴클레오타이드삼인산 등으로 분류된다.<sup class="reference" id="cite_ref-2"><a href="#cite_note-2">[2]</a></sup></p>
<p>디옥시뉴클레오타이드삼인산 가운데 <a class="new" href="/w/index.php?title=%EB%94%94%EC%98%A5%EC%8B%9C%EC%95%84%EB%8D%B0%EB%85%B8%EC%8B%A0_%EC%82%BC%EC%9D%B8%EC%82%B0&amp;action=edit&amp;redlink=1" title="디옥시아데노신 삼인산 (없는 문서)">dATP</a>, <a class="new" href="/w/index.php?title=%EB%94%94%EC%98%A5%EC%8B%9C%EA%B5%AC%EC%95%84%EB%85%B8%EC%8B%A0_%EC%82%BC%EC%9D%B8%EC%82%B0&amp;action=edit&amp;redlink=1" title="디옥시구아노신 삼인산 (없는 문서)">dGTP</a>, <a class="new" href="/w/index.php?title=%EB%94%94%EC%98%A5%EC%8B%9C%EC%82%AC%EC%9D%B4%ED%8B%B0%EB%94%98_%EC%82%BC%EC%9D%B8%EC%82%B0&amp;action=edit&amp;redlink=1" title="디옥시사이티딘 삼인산 (없는 문서)">dCTP</a>, <a class="new" href="/w/index.php?title=%ED%8B%B0%EB%AF%B8%EB%94%98_%EC%82%BC%EC%9D%B8%EC%82%B0&amp;action=edit&amp;redlink=1" title="티미딘 삼인산 (없는 문서)">dTTP</a>은 <a href="/wiki/DNA_%EB%B3%B5%EC%A0%9C" title="DNA 복제">DNA 복제</a>의 재료로 쓰인다.<sup class="reference" id="cite_ref-P48_1-1"><a href="#cite_note-P48-1">[1]</a></sup></p>
