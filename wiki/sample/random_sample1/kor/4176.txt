<p><b>디온 헤밍스</b> (Deon Hemmings, <a href="/wiki/1968%EB%85%84" title="1968년">1968년</a> <a href="/wiki/10%EC%9B%94_9%EC%9D%BC" title="10월 9일">10월 9일</a> ~ )는 <a href="/wiki/%EC%9E%90%EB%A9%94%EC%9D%B4%EC%B9%B4" title="자메이카">자메이카</a>의 전직 <a class="mw-redirect" href="/wiki/%EC%9C%A1%EC%83%81" title="육상">육상</a> 선수이다.<sup class="reference" id="cite_ref-1"><a href="#cite_note-1">[1]</a></sup></p>
<p>세인트앤에서 태어난 헤밍스는 <a class="mw-redirect" href="/wiki/1996%EB%85%84_%EC%95%A0%ED%8B%80%EB%9E%9C%ED%83%80_%EC%98%AC%EB%A6%BC%ED%94%BD" title="1996년 애틀랜타 올림픽">1996년 애틀랜타 올림픽</a>에서 <a href="/wiki/2004%EB%85%84" title="2004년">2004년</a>까지 지속된 올림픽 기록을 깨고 400m <a class="mw-redirect" href="/wiki/%ED%97%88%EB%93%A4" title="허들">허들</a>을 우승할 때 올림픽 <a href="/wiki/%EA%B8%88%EB%A9%94%EB%8B%AC" title="금메달">금메달</a>을 획득하는 데 첫 자메이카 여성 선수가 되었다. 4년 후, <a href="/wiki/2000%EB%85%84_%ED%95%98%EA%B3%84_%EC%98%AC%EB%A6%BC%ED%94%BD" title="2000년 하계 올림픽">시드니 올림픽</a>에서는 그 종목과 1600m 릴레이에서 <a href="/wiki/%EC%9D%80%EB%A9%94%EB%8B%AC" title="은메달">은메달</a>을 땄다.</p>
<p>그녀는 <a href="/wiki/1994%EB%85%84" title="1994년">1994년</a> <a href="/wiki/%EC%98%81%EC%97%B0%EB%B0%A9_%EA%B2%BD%EA%B8%B0_%EB%8C%80%ED%9A%8C" title="영연방 경기 대회">영연방 경기 대회</a>에서 은메달을 따고, <a href="/wiki/%EC%84%B8%EA%B3%84_%EC%9C%A1%EC%83%81_%EC%84%A0%EC%88%98%EA%B6%8C_%EB%8C%80%ED%9A%8C" title="세계 육상 선수권 대회">세계 육상 선수권 대회</a>에서는 허들에서 1개의 은메달(1997년)과 2개의 <a href="/wiki/%EB%8F%99%EB%A9%94%EB%8B%AC" title="동메달">동메달</a>을 따냈다.</p>
<p><a href="/wiki/2003%EB%85%84" title="2003년">2003년</a> 육상에서 은퇴하고 <a href="/wiki/2004%EB%85%84" title="2004년">2004년</a> 마이클 매캐티와 결혼하였다.</p>
