<h2><span class="mw-headline" id=".EC.97.AD.EC.82.AC">역사</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%94%94%ED%8A%B8%EB%A1%9C%EC%9D%B4%ED%8A%B8_%EA%B5%90%ED%96%A5%EC%95%85%EB%8B%A8&amp;action=edit&amp;section=1" title="부분 편집: 역사">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>1914년에 창단되었으며, 초대 음악 감독으로 웨스턴 게일즈가 임명되었다. 게일즈는 1917년까지 재임했으며, 후임으로는 <a href="/wiki/%EB%9F%AC%EC%8B%9C%EC%95%84" title="러시아">러시아</a> 출신의 오시프 가브릴로비치가 임명되었다. 가브릴로비치는 1936년까지 장기간 재임하면서 악단의 실력 향상에 주력했고, 상주 공연장의 필요성을 역설해 오케스트라 홀의 건립에도 크게 기여했다. 1922년에는 지역 방송국을 통해 세계 최초로 연주회 실황을 라디오로 중계했으며, 1934년부터는 매주 일요일 오후에 미국 전역에 방송되는 공연 실황 프로그램인 '포드 선데이 이브닝 아워(이후 포드 심포니 아워로 개칭)' 에 출연하는 등 방송 매체와 일찍부터 밀접한 관계를 맺기 시작했다.</p>
<p>가브릴로비치가 재임 도중 타계한 후에는 <a href="/wiki/%EC%9D%B4%ED%83%88%EB%A6%AC%EC%95%84" title="이탈리아">이탈리아</a> 출신의 <a class="new" href="/w/index.php?title=%ED%94%84%EB%9E%91%EC%BD%94_%EA%B8%B0%EC%98%A4%EB%84%A4&amp;action=edit&amp;redlink=1" title="프랑코 기오네 (없는 문서)">프랑코 기오네</a>와 <a href="/wiki/%ED%97%9D%EA%B0%80%EB%A6%AC" title="헝가리">헝가리</a> 출신인 빅토르 콜러르, <a href="/wiki/%EB%8F%85%EC%9D%BC" title="독일">독일</a> 출신의 칼 크뤼거가 차례로 음악 감독을 역임했다. 그러나 이 시기에 심각한 재정난을 겪게 되었고, 그로 인해 발생한 운영진 사이의 심각한 대립으로 인해 1944년과 1949년에 해단되는 최악의 상황에 처하기도 했다. 1951년에 디트로이트 소재 자동차 회사들을 중심으로 운영 자금의 공동 출자가 결정되어 다시 재결성되었고, 음악 감독으로 <a href="/wiki/%ED%94%84%EB%9E%91%EC%8A%A4" title="프랑스">프랑스</a> 출신의 <a class="new" href="/w/index.php?title=%ED%8F%B4_%ED%8C%8C%EB%A0%88&amp;action=edit&amp;redlink=1" title="폴 파레 (없는 문서)">폴 파레</a>를 영입했다.</p>
<p>파레는 <a href="/wiki/%EC%A1%B0%EB%A5%B4%EC%A3%BC_%EB%B9%84%EC%A0%9C" title="조르주 비제">비제</a>나 <a href="/wiki/%EC%B9%B4%EB%AF%B8%EC%9C%A0_%EC%83%9D%EC%83%81%EC%8A%A4" title="카미유 생상스">생상스</a>, <a href="/wiki/%EC%97%90%EB%A7%88%EB%89%98%EC%97%98_%EC%83%A4%EB%B8%8C%EB%A6%AC%EC%97%90" title="에마뉘엘 샤브리에">샤브리에</a>, <a href="/wiki/%ED%81%B4%EB%A1%9C%EB%93%9C_%EB%93%9C%EB%B7%94%EC%8B%9C" title="클로드 드뷔시">드뷔시</a>, <a href="/wiki/%EB%AA%A8%EB%A6%AC%EC%8A%A4_%EB%9D%BC%EB%B2%A8" title="모리스 라벨">라벨</a> 등 모국 작곡가들의 작품을 중심으로 악단 재건과 연주력 향상에 힘썼고, <a class="new" href="/w/index.php?title=%EC%83%A4%EB%A5%BC_%EB%AE%8C%EC%8A%88&amp;action=edit&amp;redlink=1" title="샤를 뮌슈 (없는 문서)">샤를 뮌슈</a>가 이끌던 <a href="/wiki/%EB%B3%B4%EC%8A%A4%ED%84%B4_%EA%B5%90%ED%96%A5%EC%95%85%EB%8B%A8" title="보스턴 교향악단">보스턴 교향악단</a>과 함께 미국 관현악단 중 프랑스 작품을 탁월하게 소화하는 악단으로 명성을 얻게 되었다. 파레가 1963년에 사임한 후에는 <a href="/wiki/%EC%8A%A4%EC%9B%A8%EB%8D%B4" title="스웨덴">스웨덴</a> 출신인 <a class="new" href="/w/index.php?title=%EC%8B%9D%EC%8A%A4%ED%85%90_%EC%97%90%EC%96%BC%EB%A7%81&amp;action=edit&amp;redlink=1" title="식스텐 에얼링 (없는 문서)">식스텐 에얼링</a>과 <a href="/wiki/%EC%9D%B4%ED%83%88%EB%A6%AC%EC%95%84" title="이탈리아">이탈리아</a> 출신의 <a class="new" href="/w/index.php?title=%EC%95%8C%EB%8F%84_%EC%B2%B4%EC%B9%B4%ED%86%A0&amp;action=edit&amp;redlink=1" title="알도 체카토 (없는 문서)">알도 체카토</a>, <a href="/wiki/%ED%97%9D%EA%B0%80%EB%A6%AC" title="헝가리">헝가리</a> 출신의 <a href="/wiki/%EC%95%88%ED%83%88_%EB%8F%84%EB%9D%BC%ED%8B%B0" title="안탈 도라티">안탈 도라티</a>, <a href="/wiki/%EB%8F%85%EC%9D%BC" title="독일">독일</a> 출신의 <a class="new" href="/w/index.php?title=%EA%B7%84%ED%84%B0_%ED%97%A4%EB%A5%B4%EB%B9%84%ED%9E%88&amp;action=edit&amp;redlink=1" title="귄터 헤르비히 (없는 문서)">귄터 헤르비히</a>가 차례로 직책을 인계받아 활동했다.</p>
<p>1990년에는 <a href="/wiki/%EC%97%90%EC%8A%A4%ED%86%A0%EB%8B%88%EC%95%84" title="에스토니아">에스토니아</a> 태생의 <a class="new" href="/w/index.php?title=%EB%84%A4%EB%A9%94_%EC%98%88%EB%A5%B4%EB%B9%84&amp;action=edit&amp;redlink=1" title="네메 예르비 (없는 문서)">네메 예르비</a>가 음악 감독으로 부임했고, 예르비는 모국 작품들을 비롯한 다양한 레퍼토리를 소화하면서 청중과 악단 양측에서 호평받았다. 예르비 사임 후에는 한동안 음악 감독 공석 상태로 활동했고, <a href="/wiki/%EC%BA%90%EB%82%98%EB%8B%A4" title="캐나다">캐나다</a> 출신의 <a class="new" href="/w/index.php?title=%ED%94%BC%ED%84%B0_%EC%9A%B4%EC%A7%80%EC%95%88&amp;action=edit&amp;redlink=1" title="피터 운지안 (없는 문서)">피터 운지안</a>과 토머스 윌킨스가 각각 예술 고문 겸 수석 객원 지휘자와 부지휘자로 악단을 이끌고 있다. 2008년 봄부터는 <a class="new" href="/w/index.php?title=%EB%A0%88%EB%84%88%EB%93%9C_%EC%8A%AC%EB%9E%98%ED%8A%B8%ED%82%A8&amp;action=edit&amp;redlink=1" title="레너드 슬래트킨 (없는 문서)">레너드 슬래트킨</a>이 신임 음악 감독으로 부임할 예정이다.</p>
<h2><span class="mw-headline" id=".EC.83.81.EC.A3.BC_.EA.B3.B5.EC.97.B0.EC.9E.A5.EA.B3.BC_.EC.A3.BC.EC.9A.94_.ED.99.9C.EB.8F.99">상주 공연장과 주요 활동</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%94%94%ED%8A%B8%EB%A1%9C%EC%9D%B4%ED%8A%B8_%EA%B5%90%ED%96%A5%EC%95%85%EB%8B%A8&amp;action=edit&amp;section=2" title="부분 편집: 상주 공연장과 주요 활동">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>창단 이후 1919년까지는 디트로이트 오페라 하우스를 주요 공연장으로 사용하고 있었으나, 콘서트 전문 공연장의 필요성이 계속 제기되면서 약 4개월 가량의 단기간 공사로 디트로이트 오케스트라 홀이 완공되면서 옮겨갔다. 그러나 세계 경제 대공황으로 인해 홀 운영이 힘들어지면서 <a href="/wiki/%ED%94%84%EB%A6%AC%EB%A9%94%EC%9D%B4%EC%8A%A8" title="프리메이슨">프리메이슨</a> 회당인 디트로이트 메이스닉 템플로 공연장을 다시 바꾸어야 했다.</p>
<p>1956년에는 신축 공연장인 포드 오디토리엄으로 다시 옮겨 활동했고, 그 사이 오케스트라 홀은 사설 극장으로 이용되다가 폐쇄된 뒤 철거가 예정되어 있었다. 그러나 단원들과 지역 유지들이 정부에 사적 지정을 요구하고 개축 계획을 수립하면서 대규모 장기 보수 공사에 들어갔고, 1989년에는 다시 상주 공연장으로 쓰이게 되었다. 오케스트라 홀은 2003년에 음향 보수 공사를 추가로 거치고 홀 주변에 새로 건립된 리사이틀 홀, 교육 시설과 함께 <a class="new" href="/w/index.php?title=%EB%A7%A5%EC%8A%A4_M._%ED%94%BC%EC%85%94_%EB%AE%A4%EC%A7%81_%EC%84%BC%ED%84%B0&amp;action=edit&amp;redlink=1" title="맥스 M. 피셔 뮤직 센터 (없는 문서)">맥스 M. 피셔 뮤직 센터</a>라는 이름의 복합 공연장으로 통합되었다.</p>
<p>초기에는 디트로이트의 자동차와 철강 업계 호황에 힘입어 안정된 재정 상태를 유지하고 있었으나, 대공황의 여파로 인해 심각한 타격을 받게 되어 위 항목에 서술한 대로 1944년의 일시 해단과 1949년의 장기 해단 사태를 겪기도 했다. 재결성된 후에도 지역 경제의 침체로 인해 계속 크고 작은 재정난을 겪었고, 최근에야 안정세를 보이고 있다. 1920년대 중반부터 유소년과 청소년층을 대상으로 무료 교육 음악회를 개최하는 등 청중 확보에 힘썼고, 라디오와 텔레비전 등의 매체를 적극 활용하면서 명성을 얻었다.</p>
<p>녹음 활동은 1928년에 가브릴로비치의 지휘로 처음 시작되었고, 이후 크뤼거도 피아니스트 <a href="/wiki/%ED%81%B4%EB%9D%BC%EC%9A%B0%EB%94%94%EC%98%A4_%EC%95%84%EB%9D%BC%EC%9A%B0" title="클라우디오 아라우">클라우디오 아라우</a>와 협주곡 녹음을 남기기도 했다. 그러나 본격적인 녹음은 파레 재임기부터 시작되었으며, 최신 스테레오 녹음 기술을 적극 홍보하던 음반사인 머큐리와 계약하고 프랑스 근대 음악을 중심으로 상당량의 녹음을 남겼다. 특히 생상의 교향곡 제 3번은 프랑스의 노장 오르가니스트인 마르셀 뒤프레와 협연해 화제가 되었다.</p>
<p>파레 이후에는 도라티가 전속사였던 데카에 주목할 만한 녹음들을 많이 남겼으며, 특히 <a class="mw-redirect" href="/wiki/%EC%9D%B4%EA%B3%A0%EB%A6%AC_%EC%8A%A4%ED%8A%B8%EB%9D%BC%EB%B9%88%EC%8A%A4%ED%82%A4" title="이고리 스트라빈스키">스트라빈스키</a>의 발레 '봄의 제전' 녹음은 프랑스의 샤를 크로 디스크 대상을 수상한 바 있다. 예르비도 <a href="/wiki/%EC%98%81%EA%B5%AD" title="영국">영국</a> 음반사인 샨도스와 계약하고 다양한 작품들을 녹음했으며, 특히 <a href="/wiki/%EC%B0%B0%EC%8A%A4_%EC%95%84%EC%9D%B4%EB%B8%8C%EC%8A%A4" title="찰스 아이브스">아이브스</a>나 <a href="/wiki/%EC%83%88%EB%AE%A4%EC%96%BC_%EB%B0%94%EB%B2%84" title="새뮤얼 바버">바버</a>, <a class="new" href="/w/index.php?title=%EC%A1%B0%EC%A7%80_%EC%B1%84%EB%93%9C%EC%9C%85&amp;action=edit&amp;redlink=1" title="조지 채드윅 (없는 문서)">채드윅</a>, 랜달 톰슨, 윌리엄 그랜트 스틸 등 미국 작곡가들의 관현악 작품 음반들이 호평을 받았다. 최근에는 악단 이름을 내건 자체 음반사를 통해서도 음반을 발매하고 있다.</p>
