<h1 class="firstHeading" id="firstHeading" lang="ko">떡국</h1>
<div class="mw-body-content" id="bodyContent">

떡국</div>
</div>
</div>
<p><b>떡국</b>은 <a href="/wiki/%ED%95%9C%EA%B5%AD" title="한국">한국</a>의 전통 음식이다. 장국에 일정한 두께로 얇게 썰은 흰떡을 넣고 끓여 만든 음식이며, 고기, <a href="/wiki/%EB%8B%AC%EA%B1%80" title="달걀">달걀</a> 등을 고명으로 얹거나 <a href="/wiki/%EB%A7%8C%EB%91%90" title="만두">만두</a> 등의 재료를 곁들여 먹는다. 한국에는 <a href="/wiki/%ED%95%9C%EA%B5%AD%EC%9D%98_%EC%84%A4%EB%82%A0" title="한국의 설날">설날</a>에 떡국을 먹는 전통이 있다.<sup class="reference" id="cite_ref-1"><a href="#cite_note-1">[1]</a></sup></p>
<h2><span class="mw-headline" id=".EA.B0.99.EC.9D.B4_.EB.B3.B4.EA.B8.B0">같이 보기</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%96%A1%EA%B5%AD&amp;action=edit&amp;section=1" title="부분 편집: 같이 보기">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<ul>
<li><a href="/wiki/%EB%96%A1" title="떡">떡</a></li>
<li><a href="/wiki/%EC%88%98%EC%A0%9C%EB%B9%84" title="수제비">수제비</a></li>
<li><a href="/wiki/%ED%95%9C%EA%B5%AD_%EC%9A%94%EB%A6%AC" title="한국 요리">한국 요리</a></li>
<li><a class="mw-disambig" href="/wiki/%EC%A1%B0%EB%8B%88" title="조니">조니</a>(雑煮) - <a href="/wiki/%EC%9D%BC%EB%B3%B8" title="일본">일본</a>에서 <a href="/wiki/%EC%84%A4%EB%82%A0" title="설날">설날</a>에 먹는 떡국이다. 구운 <a class="mw-redirect" href="/wiki/%EC%B0%B0%EB%96%A1" title="찰떡">찰떡</a>, <a class="new" href="/w/index.php?title=%EC%96%B4%EC%9C%A1&amp;action=edit&amp;redlink=1" title="어육 (없는 문서)">어육</a>, <a class="mw-redirect" href="/wiki/%EC%95%BC%EC%B1%84" title="야채">야채</a>를 재료로 사용한다. <a href="/wiki/%ED%95%9C%EA%B5%AD" title="한국">한국</a>의 떡국과 비슷하며,가래떡이 아니라 <a class="mw-redirect" href="/wiki/%EC%B0%B0%EB%96%A1" title="찰떡">찰떡</a>을 사용하는 게 다르다. <a href="/wiki/%EC%9D%BC%EB%B3%B8" title="일본">일본</a>역사소설《<a href="/wiki/%EC%98%A4%EC%8B%B1" title="오싱">오싱</a>》에서 조니는 <a href="/wiki/%EC%83%88%ED%95%B4" title="새해">새해</a>를 상징하는 음식으로 묘사되었는데, 번역자 김균은 한국 독자들의 이해를 돕기 위해 '떡국'으로 번역하였다.</li>
</ul>
<h2><span class="mw-headline" id=".EA.B0.81.EC.A3.BC">각주</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%96%A1%EA%B5%AD&amp;action=edit&amp;section=2" title="부분 편집: 각주">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<div class="reflist" style="list-style-type: decimal;">
<ol class="references">
<li id="cite_note-1"><span class="mw-cite-backlink"><a href="#cite_ref-1">↑</a></span> <span class="reference-text">떡국은 지방에 따라 다양화되었는데, 예를 들어 <a class="mw-redirect" href="/wiki/%EA%B0%9C%EC%84%B1" title="개성">개성</a>에서는 조랭이 떡국이라 하여 조선 초기에 이성계의 목을 자르고 싶다는 뜻으로 만들었다고 한다. 그러나, <a href="/wiki/%EC%9E%84%EC%A7%84%EC%99%9C%EB%9E%80" title="임진왜란">임진왜란</a> 직후에는 한때 그 대상이 바뀌기도 했었다. 즉, <a href="/wiki/%EB%8F%84%EC%9A%94%ED%86%A0%EB%AF%B8_%ED%9E%88%EB%8D%B0%EC%9A%94%EC%8B%9C" title="도요토미 히데요시">도요토미 히데요시</a>의 목을 자르고 싶다는 의미로 재해석된 것이다.</span></li>
</ol>
</div>
