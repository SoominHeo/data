<p><b>라마 1세</b>(<span style="font-size: smaller;"><a href="/wiki/%ED%83%9C%EA%B5%AD%EC%96%B4" title="태국어">태국어</a>:</span> <span lang="th" xml:lang="th">รัชกาลที่</span>, Rama I, <a href="/wiki/1737%EB%85%84" title="1737년">1737년</a> <a href="/wiki/3%EC%9B%94_20%EC%9D%BC" title="3월 20일">3월 20일</a> - <a href="/wiki/1809%EB%85%84" title="1809년">1809년</a> <a href="/wiki/9%EC%9B%94_7%EC%9D%BC" title="9월 7일">9월 7일</a>)는 <a href="/wiki/%ED%83%80%EC%9D%B4" title="타이">타이</a>의 현 왕조인 <a href="/wiki/%EC%A7%9C%EB%81%84%EB%A6%AC_%EC%99%95%EC%A1%B0" title="짜끄리 왕조">짜끄리 왕조</a>의 창시자로, 초대 국왕(재위 <a href="/wiki/1782%EB%85%84" title="1782년">1782년</a> - <a href="/wiki/1809%EB%85%84" title="1809년">1809년</a>)이다.</p>
<p>본명은 <b>챠오피아짜끄리</b>이며, <a href="/wiki/%EB%94%B1%EC%8B%A0" title="딱신">딱신</a> 왕의 장군으로 전쟁의 공적이 많았다. <a class="new" href="/w/index.php?title=%EC%97%AD%EC%84%B1%ED%98%81%EB%AA%85&amp;action=edit&amp;redlink=1" title="역성혁명 (없는 문서)">역성혁명</a>을 일으켜 <a href="/wiki/%ED%86%A4%EB%B6%80%EB%A6%AC_%EC%99%95%EC%A1%B0" title="톤부리 왕조">톤부리 왕조</a>의 피아딱신 왕을 죽이고 스스로 왕위에 올라 수도를 <a href="/wiki/%EB%B0%A9%EC%BD%95" title="방콕">방콕</a>에 정하였다. 재위 기간 동안 왕조를 튼튼히 하였으며, 과거의 법전을 수집·개정하여 새로운 법전을 편찬해 내기도 하였다.<sup class="reference" id="cite_ref-1"><a href="#cite_note-1">[1]</a></sup></p>

<h2><span class="mw-headline" id=".EC.83.9D.EC.95.A0">생애</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%9D%BC%EB%A7%88_1%EC%84%B8&amp;action=edit&amp;section=1" title="부분 편집: 생애">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>아명은 <b>톤 두원</b>이며, <a href="/wiki/%EC%95%84%EC%9C%A0%ED%83%80%EC%95%BC_%EC%99%95%EC%A1%B0" title="아유타야 왕조">아유타야 왕가</a>의 혈통을 받고 태어난 아유타야 시대의 사관, 차오프라야 코서판의 손자로 알려져 있다. 아유타야 왕조 말기에 사관으로 벼슬을 했으며 붓다 요드파 출라로께의 벼슬도 가지고 있었다. 라마 1세의 칭호는 라마 6세에 의해서 제정된 것이다.</p>
<p>그가 벼슬을 한 아유타야 왕조는 <a href="/wiki/1767%EB%85%84" title="1767년">1767년</a>에 <a class="mw-redirect" href="/wiki/%EB%B2%84%EB%A7%88" title="버마">버마</a>에 점령되었고, 다음 해 중국계 세력인 <a href="/wiki/%EB%94%B1%EC%8B%A0" title="딱신">딱신</a>이 버마군을 몰아내고, <a href="/wiki/%ED%86%A4%EB%B6%80%EB%A6%AC_%EC%99%95%EC%A1%B0" title="톤부리 왕조">톤부리 왕조</a>를 일으켜 타이의 왕으로 등극한다. 이때 붓다 요드파 출라로께(라마 1세)는 딱신왕의 신하가 되었다. 출라로께는 딱신이 <a href="/wiki/%EB%9D%BC%EC%9A%A9" title="라용">라용</a>으로 군사를 일으켰을 때 딱신의 어머니를 보호를 위해 라용에 데려다 준 공훈으로 승진한 것을 시작으로 많은 전과를 올려 차례차례 승진을 했기 때문에, 맹장으로도 알려져 있다.</p>
<h2><span class="mw-headline" id=".EC.A7.9C.EB.81.84.EB.A6.AC_.EC.99.95.EC.A1.B0">짜끄리 왕조</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%9D%BC%EB%A7%88_1%EC%84%B8&amp;action=edit&amp;section=2" title="부분 편집: 짜끄리 왕조">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<div class="thumb tleft">
<div class="thumbinner" style="width:232px;"><a class="image" href="/wiki/%ED%8C%8C%EC%9D%BC:Watpkaeoramakien0605.jpg"><img alt="" class="thumbimage" data-file-height="718" data-file-width="850" height="194" src="//upload.wikimedia.org/wikipedia/commons/thumb/4/4e/Watpkaeoramakien0605.jpg/230px-Watpkaeoramakien0605.jpg" srcset="//upload.wikimedia.org/wikipedia/commons/thumb/4/4e/Watpkaeoramakien0605.jpg/345px-Watpkaeoramakien0605.jpg 1.5x, //upload.wikimedia.org/wikipedia/commons/thumb/4/4e/Watpkaeoramakien0605.jpg/460px-Watpkaeoramakien0605.jpg 2x" width="230"/></a>
<div class="thumbcaption">
<div class="magnify"><a class="internal" href="/wiki/%ED%8C%8C%EC%9D%BC:Watpkaeoramakien0605.jpg" title="실제 크기로"></a></div>
<a href="/wiki/%EB%B0%A9%EC%BD%95_%EC%99%95%EA%B6%81" title="방콕 왕궁">방콕 왕궁</a>에 있는 라마끼엔 벽화</div>
</div>
</div>
<p>톤부리 왕조의 딱신 왕은 말년에 정신착란을 일으켰으며, 차오프라야(라마 1세)는 국내의 혼란을 두려워하여, <a href="/wiki/%EC%BA%84%EB%B3%B4%EB%94%94%EC%95%84" title="캄보디아">캄보디아</a> 원정을 중단하고 톤부리로 돌아와 역성혁명을 일으켜 <a class="mw-redirect" href="/wiki/%EB%94%B1%EC%8B%A0_%EB%8C%80%EC%99%95" title="딱신 대왕">딱신 대왕</a>을 폐위, 처형했다. 이후, 차오프라야는 왕위에 올랐다. 타이의 역사서에서는 차오프라야가 백성들의 지지를 얻어 왕으로 등극하였다고 나온다. 왕이 되었기 때문에 이름을 붓다 요드파 출라로께로 바꿨으며, 버마의 침공을 방어하기 위해 수도를 톤부리에서 <a class="mw-redirect" href="/wiki/%EC%B0%A8%EC%98%A4%ED%94%84%EB%9D%BC%EC%95%BC_%EA%B0%95" title="차오프라야 강">차오프라야 강</a>을 사이에 두고 맞은 편인 동쪽에 방콕(정확하게는 끄룽텝)을 건설하여 천도 한 후, 짜끄리 왕조를 창설했다. 아유타야 왕조의 신권정치를 계승했다.</p>
<h2><span class="mw-headline" id=".EB.AC.B8.ED.95.99">문학</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%9D%BC%EB%A7%88_1%EC%84%B8&amp;action=edit&amp;section=3" title="부분 편집: 문학">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>출라로께 왕(라마 1세)은 즉위 이후에도 몇 차례에 걸쳐서 버마 침공군을 몰아내고, 국내를 안정시켰다. 국내가 안정되자, 《삼표법》을 제정하여 관료 제도를 정비했다. 또 정통성을 확보하기 위해 아유타야 왕조의 후예인 것을 강조하고자 많은 학자들을 등용하여 버마 점령 시대에 사라진 문학 <a class="new" href="/w/index.php?title=%EB%9D%BC%EB%A7%88%EB%81%BC%EC%97%94&amp;action=edit&amp;redlink=1" title="라마끼엔 (없는 문서)">라마끼엔</a>, 이나오를 재편집 했다. 지금도 라마 1세의 라마끼엔은 수많은 라마끼엔 중에서도 뛰어난 작품으로서 알려져 있다.</p>
<h2><span class="mw-headline" id=".EA.B1.B4.EC.B6.95.EB.AC.BC">건축물</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%9D%BC%EB%A7%88_1%EC%84%B8&amp;action=edit&amp;section=4" title="부분 편집: 건축물">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>관광지로 잘 알려진 <a class="mw-redirect" href="/wiki/%EC%99%93_%ED%94%84%EB%9D%BC%EA%B9%A8%EC%9A%B0" title="왓 프라깨우">왓 프라깨우</a>(에메랄드 사원) 등 많은 사원을 지었다. 이곳에 <a class="mw-redirect" href="/wiki/%EC%B6%9C%EB%9D%BC%EB%A1%B1%EA%BC%B0_%EC%99%95" title="출라롱꼰 왕">출라롱꼰 왕</a>(<a class="mw-redirect" href="/wiki/%EB%9D%BC%EB%A7%88_5%EC%84%B8" title="라마 5세">라마 5세</a>)은 <a href="/wiki/%EB%9D%BC%EC%98%A4%EC%8A%A4" title="라오스">라오스</a>에서 빼앗은 에메랄드 불상을 설치했다. 그의 이름을 딴 유명한 건축물은 차오프라야 강 위에 가설된 최초의 대교인 사판 풋 등이 있다.</p>
