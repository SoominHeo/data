<p><b>라에</b>(Lae)는 인구 12만여 명의 <a href="/wiki/%ED%8C%8C%ED%91%B8%EC%95%84%EB%89%B4%EA%B8%B0%EB%8B%88" title="파푸아뉴기니">파푸아뉴기니</a>에서 두 번째로 큰 도시이다. 파푸아 뉴기니 북쪽 해안의 후온 만에 위치해 있으며 <a class="new" href="/w/index.php?title=%EB%A7%88%EC%BA%84_%EA%B0%95&amp;action=edit&amp;redlink=1" title="마캄 강 (없는 문서)">마캄 강</a> 하구에 인접해 있다. <a class="new" href="/w/index.php?title=%EB%AA%A8%EB%A1%9C%EB%B2%A0&amp;action=edit&amp;redlink=1" title="모로베 (없는 문서)">모로베</a> 주의 수도이자 파푸아 뉴기니의 공업 중심지이다. 라에는 하이랜드 지방을 해안과 연결하는 주요 교통망인 <a href="/wiki/%ED%95%98%EC%9D%BC%EB%9E%9C%EC%A6%88_%EA%B3%A0%EC%86%8D%EB%8F%84%EB%A1%9C" title="하일랜즈 고속도로">하일랜즈 고속도로</a>의 출발점이며 파푸아 뉴기니의 가장 큰 항구이기도 하다. <a class="new" href="/w/index.php?title=%ED%8C%8C%ED%91%B8%EC%95%84_%EB%89%B4%EA%B8%B0%EB%8B%88_%EA%B8%B0%EC%88%A0_%EB%8C%80%ED%95%99&amp;action=edit&amp;redlink=1" title="파푸아 뉴기니 기술 대학 (없는 문서)">파푸아 뉴기니 기술 대학</a> 역시 라에에 위치해 있다.</p>
<h2><span class="mw-headline" id=".EC.97.AD.EC.82.AC">역사</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%9D%BC%EC%97%90&amp;action=edit&amp;section=1" title="부분 편집: 역사">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>라에는 <a href="/wiki/1920%EB%85%84%EB%8C%80" title="1920년대">1920년대</a>에서 <a href="/wiki/1930%EB%85%84%EB%8C%80" title="1930년대">1930년대</a> 사이 금광의 발견과 함께 탄생하였다. 파푸아 뉴기니의 많은 도시들처럼 활주로 주변에 솟아났으며, 라에에 도착한 화물은 다시 항공로를 통해 <a class="mw-disambig" href="/wiki/%EC%99%80%EC%9A%B0" title="와우">와우</a>의 금광 지역으로 이동되었다.</p>
<p><a href="/wiki/1937%EB%85%84" title="1937년">1937년</a> <a href="/wiki/%EB%9D%BC%EB%B0%94%EC%9A%B8" title="라바울">라바울</a>의 화산폭발과 함께 뉴기니의 수도가 라바울에서 라에로 옮겨질 것이 결정되었지만 이 과정은 <a href="/wiki/%EC%A0%9C2%EC%B0%A8_%EC%84%B8%EA%B3%84_%EB%8C%80%EC%A0%84" title="제2차 세계 대전">제2차 세계 대전</a> 중 라에가 <a href="/wiki/%EC%9D%BC%EB%B3%B8_%EC%A0%9C%EA%B5%AD" title="일본 제국">일본 제국</a>에 의해 점령되면서 무산되고 만다. 라에, 라바울, <a class="new" href="/w/index.php?title=%EC%82%B4%EB%9D%BC%EB%A7%88%EC%9A%B0%EC%95%84&amp;action=edit&amp;redlink=1" title="살라마우아 (없는 문서)">살라마우아</a>는 후에 뉴기니의 주요 일본군 기지가 되었다. <a href="/wiki/1943%EB%85%84" title="1943년">1943년</a> 중반 <a class="new" href="/w/index.php?title=%EC%BD%94%EC%BD%94%EB%8B%A4_%EC%A0%84%EC%97%AD&amp;action=edit&amp;redlink=1" title="코코다 전역 (없는 문서)">코코다 전역</a>, <a class="new" href="/w/index.php?title=%EB%B6%80%EB%82%98%EA%B3%A0%EB%82%98_%EC%A0%84%ED%88%AC&amp;action=edit&amp;redlink=1" title="부나고나 전투 (없는 문서)">부나고나 전투</a>, <a class="new" href="/w/index.php?title=%EC%99%80%EC%9A%B0_%EC%A0%84%ED%88%AC&amp;action=edit&amp;redlink=1" title="와우 전투 (없는 문서)">와우 전투</a>에서의 패배와 함께 일본군은 라에와 살라마우아로 밀려났다. <a class="new" href="/w/index.php?title=%EB%9D%BC%EC%97%90_%EC%A0%84%ED%88%AC&amp;action=edit&amp;redlink=1" title="라에 전투 (없는 문서)">라에 전투</a>는 여러 주 동안 계속되었으며 <a href="/wiki/9%EC%9B%94_16%EC%9D%BC" title="9월 16일">9월 16일</a> 라에는 연합군에 의해 함락된다.</p>
<p>전후 라에의 발전은 <a href="/wiki/%EC%BB%A4%ED%94%BC" title="커피">커피</a>와 <a href="/wiki/%EC%B0%A8_(%EC%9D%8C%EB%A3%8C)" title="차 (음료)">차</a>의 수출항으로서 하이랜드 지역의 발전과 직접적으로 연결되었으며, 이후 하이랜드 고속도로가 연결되었다. <a href="/wiki/1980%EB%85%84%EB%8C%80" title="1980년대">1980년대</a>부터 <a href="/wiki/1990%EB%85%84%EB%8C%80" title="1990년대">1990년대</a>의 광업 발달은 라에의 경제 중심지로서의 위치를 더욱 부각시켰다.</p>
<p><a href="/wiki/1991%EB%85%84" title="1991년">1991년</a> 라에는 <a class="mw-redirect" href="/wiki/%EB%82%A8%ED%83%9C%ED%8F%89%EC%96%91_%EA%B2%8C%EC%9E%84" title="남태평양 게임">남태평양 게임</a>을 주최하였다. 라에는 현재 높은 범죄율과 도로상의 무수한 구멍들로 알려져 있으며, 이에 따라 "구멍난 도시"라는 별명이 붙었다.</p>
