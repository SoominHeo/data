<p><b>휘세인 라우프 오르바이</b> (Hüseyin Rauf Orbay, 1881년, <a class="mw-redirect" href="/wiki/%EC%BD%98%EC%8A%A4%ED%83%84%ED%8B%B0%EB%85%B8%ED%94%8C" title="콘스탄티노플">콘스탄티노플</a> ~ <a href="/wiki/1964%EB%85%84" title="1964년">1964년</a> <a href="/wiki/7%EC%9B%94_16%EC%9D%BC" title="7월 16일">7월 16일</a>) 는 <a href="/wiki/%ED%84%B0%ED%82%A4" title="터키">터키</a>의 해군 장교이자 정치인으로, <a href="/wiki/%ED%84%B0%ED%82%A4_%EB%8F%85%EB%A6%BD_%EC%A0%84%EC%9F%81" title="터키 독립 전쟁">터키 독립 전쟁</a> 중인 1922년부터 1923년까지 터키의 총리를 지냈다.</p>
<p><a class="mw-redirect" href="/wiki/%EC%95%95%ED%95%98%EC%8A%A4%EC%9D%B8" title="압하스인">압하스인</a>에 뿌리를 둔 아버지와 <a class="mw-redirect" href="/wiki/%ED%81%AC%EB%A0%88%ED%83%80_%EC%84%AC" title="크레타 섬">크레타 섬</a> 출신의 어머니 사이에서 태어났다. <a class="new" href="/w/index.php?title=%EC%98%A4%EC%8A%A4%EB%A7%8C_%ED%95%B4%EA%B5%B0&amp;action=edit&amp;redlink=1" title="오스만 해군 (없는 문서)">오스만 제국 해군</a>의 장교로서, <a href="/wiki/%EC%A0%9C1%EC%B0%A8_%EB%B0%9C%EC%B9%B8_%EC%A0%84%EC%9F%81" title="제1차 발칸 전쟁">제1차 발칸 전쟁</a> 당시에 <a class="extiw" href="https://en.wikipedia.org/wiki/Ottoman_cruiser_Hamidiye" title="en:Ottoman cruiser Hamidiye">하미디예</a> 순양함의 선장으로서의 활약으로 명성을 떨쳤다. 1918년 10월 31일, 그는 해군 장관으로서 연합군과의 <a href="/wiki/%EB%AC%B4%EB%93%9C%EB%A1%9C%EC%8A%A4_%EC%A0%95%EC%A0%84_%ED%98%91%EC%A0%95" title="무드로스 정전 협정">무드로스 정전 협정</a> (<span style="font-size: smaller;"><a href="/wiki/%ED%84%B0%ED%82%A4%EC%96%B4" title="터키어">터키어</a>:</span> <span lang="tr" xml:lang="tr">Mondros Ateşkes Anlaşması</span>) 에 서명하였고, 오스만 제국은 <a href="/wiki/%EC%A0%9C1%EC%B0%A8_%EC%84%B8%EA%B3%84_%EB%8C%80%EC%A0%84" title="제1차 세계 대전">제1차 세계 대전</a>에서 패전국이 되었다. <a href="/wiki/%ED%84%B0%ED%82%A4_%EB%8F%85%EB%A6%BD_%EC%A0%84%EC%9F%81" title="터키 독립 전쟁">터키 독립 전쟁</a>이 시작되었을 때, 그는 자신의 지위에서 물러나, <a href="/wiki/%EC%95%99%EC%B9%B4%EB%9D%BC" title="앙카라">앙카라</a>에서 <a class="mw-redirect" href="/wiki/%EB%AC%B4%EC%8A%A4%ED%83%80%ED%8C%8C_%EC%BC%80%EB%A7%90" title="무스타파 케말">무스타파 케말</a>과 협력하였고, 1919년 7월 23일에 에르주룸 회의 (Erzurum Congress) 의 대표위원회의 구성원, 1919년 9월 4일에 시바스 회의 (Congress of Sivas) 의 부의장으로 선출되었다.</p>
<p>1922년 8월 11일, <a href="/wiki/%ED%84%B0%ED%82%A4_%EB%8F%85%EB%A6%BD_%EC%A0%84%EC%9F%81" title="터키 독립 전쟁">터키 독립 전쟁</a>이 끝났을 때, 터키의 새 공화국의 첫 총리로 취임하였고, 1924년에 터키 진보 공화당 (<span style="font-size: smaller;"><a href="/wiki/%ED%84%B0%ED%82%A4%EC%96%B4" title="터키어">터키어</a>:</span> <span lang="tr" xml:lang="tr">Terakkiperver Cumhuriyet Fırkasi</span>) 설립에 참여했지만, 1925년에 동당이 정부로부터 해산 명령을 받자, 그는 정부의 탄압을 피해 유럽에 10년간 망명하였다. 그 후 그는, 10년 만에 귀국하여 터키 대국민의회의 의원으로 정계에 복귀하였고, <a href="/wiki/%EC%A0%9C2%EC%B0%A8_%EC%84%B8%EA%B3%84_%EB%8C%80%EC%A0%84" title="제2차 세계 대전">제2차 세계 대전</a> 중에는 1942년부터 주<a href="/wiki/%EC%98%81%EA%B5%AD" title="영국">영</a> 대사를 맡았으나, 1944년에 사임하였다. <a href="/wiki/1964%EB%85%84" title="1964년">1964년</a>, <a href="/wiki/%EC%9D%B4%EC%8A%A4%ED%83%84%EB%B6%88" title="이스탄불">이스탄불</a>에서 세상을 떠났다.</p>
