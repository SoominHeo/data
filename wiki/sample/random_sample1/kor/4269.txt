<p><b>라이몬트 아우만</b> (Raimond Aumann, <a href="/wiki/1963%EB%85%84" title="1963년">1963년</a> <a href="/wiki/10%EC%9B%94_12%EC%9D%BC" title="10월 12일">10월 12일</a>, <a href="/wiki/%EB%B0%94%EC%9D%B4%EC%97%90%EB%A5%B8_%EC%A3%BC" title="바이에른 주">바이에른 주</a> <a href="/wiki/%EC%95%84%EC%9A%B0%ED%81%AC%EC%8A%A4%EB%B6%80%EB%A5%B4%ED%81%AC" title="아우크스부르크">아우크스부르크</a> ~) 는 은퇴한 <a href="/wiki/%EB%8F%85%EC%9D%BC" title="독일">독일</a>의 축구 선수이다.<sup class="reference" id="cite_ref-1"><a href="#cite_note-1">[1]</a></sup> 그의 별명은 <b>발루</b> (Balu) 인데, 이 이름은 <a href="/wiki/%EB%9F%AC%EB%94%94%EC%96%B4%EB%93%9C_%ED%82%A4%ED%94%8C%EB%A7%81" title="러디어드 키플링">러디어드 키플링</a>의 <a href="/wiki/%EC%A0%95%EA%B8%80_%EB%B6%81" title="정글 북">정글북</a>에서 나오는 곰인 발루 (Baloo) 의 이름을 딴 것이다.<sup class="reference" id="cite_ref-2"><a href="#cite_note-2">[2]</a></sup></p>
<p>그는 1982년에서 1994년까지 <a class="mw-redirect" href="/wiki/%ED%91%B8%EC%8A%A4%EB%B3%BC-%EB%B6%84%EB%8D%B0%EC%8A%A4%EB%A6%AC%EA%B0%80" title="푸스볼-분데스리가">분데스리가</a>의 <a href="/wiki/%EA%B3%A8%ED%82%A4%ED%8D%BC" title="골키퍼">골키퍼</a>로 활약하였다. 그는 첫 2년을 리저브 키퍼로 활약하였으며, <a href="/wiki/%EC%9E%A5%EB%A7%88%EB%A6%AC_%ED%8C%8C%ED%94%84" title="장마리 파프">장마리 파프</a>의 백업이었다. 1984년, 그는 1985년 11월에 부상당하기 전까지 주전 골키퍼자리를 확보하였다. 1988년, 파프가 뮌헨을 떠나자 아우만은 다시 주전 자리를 탈환하였다. 그는 당시 독일의 최고 골키퍼로 평가받았다.</p>
<p>바이에른 뮌헨 소속으로, 그는 6차례 <a class="mw-redirect" href="/wiki/%ED%91%B8%EC%8A%A4%EB%B3%BC-%EB%B6%84%EB%8D%B0%EC%8A%A4%EB%A6%AC%EA%B0%80" title="푸스볼-분데스리가">독일 분데스리가</a>, 2차례 <a href="/wiki/DFB-%ED%8F%AC%EC%B9%BC" title="DFB-포칼">DFB-포칼</a> 우승을 거두었다. 이후 1994년에 <a class="mw-redirect" href="/wiki/%EB%B2%A0%EC%8B%9D%ED%83%80%EC%8A%88_JK" title="베식타슈 JK">베식타슈 JK</a>로 이적한 그는 1994-95 시즌의 <a href="/wiki/%EC%89%AC%ED%8E%98%EB%A5%B4%EB%A6%AC%EA%B7%B8" title="쉬페르리그">터키 슈퍼리그</a> 우승으로 이끌었다. 1995년 11월, 아우만은 현역 은퇴를 선언하였다.</p>
<p>아우만은 1989년과 1990년 사이 4차례 <a href="/wiki/%EB%8F%85%EC%9D%BC_%EC%B6%95%EA%B5%AC_%EA%B5%AD%EA%B0%80%EB%8C%80%ED%91%9C%ED%8C%80" title="독일 축구 국가대표팀">국가대표팀</a> 경기에 출장하였다. 그는 <a href="/wiki/1990%EB%85%84_FIFA_%EC%9B%94%EB%93%9C%EC%BB%B5" title="1990년 FIFA 월드컵">1990년 FIFA 월드컵</a>의 독일 엔트리에 이름을 올렸으나 단 한경기도 출장하지 못하였다.</p>
<p>현재, 그는 바이에른 뮌헨의 팬이자 팬클럽 인사이다.</p>
