<h2><span class="mw-headline" id=".EA.B2.BD.EB.A0.A5">경력</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%9D%BC%EC%9D%B4%EC%96%B8_%EA%B3%A0%EC%9D%B8%EC%8A%A4&amp;action=edit&amp;section=1" title="부분 편집: 경력">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<h3><span class="mw-headline" id=".EB.A9.94.EC.9D.B4.EC.A0.80_.EB.A6.AC.EA.B7.B8_.EC.9D.B4.EC.A0.84">메이저 리그 이전</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%9D%BC%EC%9D%B4%EC%96%B8_%EA%B3%A0%EC%9D%B8%EC%8A%A4&amp;action=edit&amp;section=2" title="부분 편집: 메이저 리그 이전">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p><a href="/wiki/1988" title="1988">1988</a>년에 <a href="/wiki/%ED%85%8D%EC%82%AC%EC%8A%A4_%EC%A3%BC" title="텍사스 주">텍사스 주</a>에 있는 <a href="/wiki/%EB%9D%BC%EC%9A%B4%EB%93%9C%EB%A1%9D" title="라운드록">라운드록</a>에서 태어났다. 고향에 있는 <a class="new" href="/w/index.php?title=%EC%8A%A4%ED%86%A0%EB%8B%88_%ED%8F%AC%EC%9D%B8%ED%8A%B8_%EA%B3%A0%EB%93%B1%ED%95%99%EA%B5%90&amp;action=edit&amp;redlink=1" title="스토니 포인트 고등학교 (없는 문서)">스토니 포인트 고등학교</a>를 다니면서 <a href="/wiki/%EC%95%BC%EA%B5%AC" title="야구">야구</a>를 했으며, 포지션은 <a href="/wiki/%EC%9C%A0%EA%B2%A9%EC%88%98" title="유격수">유격수</a>였다. 졸업한 후에는 <a class="new" href="/w/index.php?title=%EB%8C%88%EB%9F%AC%EC%8A%A4_%EC%B9%A8%EB%A1%80_%EB%8C%80%ED%95%99%EA%B5%90&amp;action=edit&amp;redlink=1" title="댈러스 침례 대학교 (없는 문서)">댈러스 침례 대학교</a>로 진학하여 야구를 했으며, <a href="/wiki/2008%EB%85%84" title="2008년">2008년</a>에는 노스우드 리그에 소속되어 있는 워털루 벅스에서 뛰었다. 그리고 22살이 되던 <a href="/wiki/2009%EB%85%84" title="2009년">2009년</a>에 있었던 MLB 드래프트에서 4라운드 전체 130위로 <a href="/wiki/%ED%86%A0%EB%A1%A0%ED%86%A0_%EB%B8%94%EB%A3%A8%EC%A0%9C%EC%9D%B4%EC%8A%A4" title="토론토 블루제이스">토론토 블루제이스</a>에게 지명 되어 입단하게 된다. <a href="/wiki/2012%EB%85%84" title="2012년">2012년</a>에는 더블A인 뉴햄프셔 피셔 캣츠에서 뛰었다. 시즌이 끝난 후에는 <a href="/wiki/%ED%86%A0%EB%A1%A0%ED%86%A0_%EB%B8%94%EB%A3%A8%EC%A0%9C%EC%9D%B4%EC%8A%A4" title="토론토 블루제이스">토론토 블루제이스</a>의 40인 로스터에 들어갔으며, 가을에는 유망주들의 리그인 애리조나 가을 리그에 참여한다.</p>
<h3><span class="mw-headline" id=".EB.A9.94.EC.9D.B4.EC.A0.80_.EB.A6.AC.EA.B7.B8">메이저 리그</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%9D%BC%EC%9D%B4%EC%96%B8_%EA%B3%A0%EC%9D%B8%EC%8A%A4&amp;action=edit&amp;section=3" title="부분 편집: 메이저 리그">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p><a href="/wiki/2013%EB%85%84" title="2013년">2013년</a> <a href="/wiki/8%EC%9B%94_22%EC%9D%BC" title="8월 22일">8월 22일</a>에 같은 팀 동료 <a href="/wiki/2%EB%A3%A8%EC%88%98" title="2루수">2루수</a>인 <a href="/wiki/%EB%A9%94%EC%9D%B4%EC%84%9C_%EC%9D%B4%EC%8A%A4%ED%88%AC%EB%A6%AC%EC%8A%A4" title="메이서 이스투리스">메이서 이스투리스</a>가 부상자 명단에 오르자 <a class="mw-disambig" href="/wiki/%EB%A9%94%EC%9D%B4%EC%A0%80_%EB%A6%AC%EA%B7%B8" title="메이저 리그">메이저 리그</a>로 승격 됐다. 다음 날인 <a href="/wiki/8%EC%9B%94_23%EC%9D%BC" title="8월 23일">8월 23일</a>에 있었던 <a href="/wiki/%ED%9C%B4%EC%8A%A4%ED%84%B4_%EC%95%A0%EC%8A%A4%ED%8A%B8%EB%A1%9C%EC%8A%A4" title="휴스턴 애스트로스">휴스턴 애스트로스</a>전에서 9번 타자이자 <a href="/wiki/2%EB%A3%A8%EC%88%98" title="2루수">2루수</a>로 나오며 <a class="mw-disambig" href="/wiki/%EB%A9%94%EC%9D%B4%EC%A0%80_%EB%A6%AC%EA%B7%B8" title="메이저 리그">메이저 리그</a> 데뷔를 한다. 데뷔전에서는 2루타를 포함해서 4타수 2안타를 기록했으며, 경기 도중에 수비위치를 2루에서 유격수로 옮기기도 하지만, 팀은 4-12로 패하고 만다. <a href="/wiki/8%EC%9B%94_25%EC%9D%BC" title="8월 25일">8월 25일</a>에 있었던 경기에서는 9회에 땅볼을 치지만 자신의 첫 타점을 올린다. <a href="/wiki/8%EC%9B%94_31%EC%9D%BC" title="8월 31일">8월 31일</a>에는 <a class="mw-disambig" href="/wiki/%EB%A9%94%EC%9D%B4%EC%A0%80_%EB%A6%AC%EA%B7%B8" title="메이저 리그">메이저 리그</a> 데뷔 이후 8경기 연속 안타를 치면서 데뷔 순간부터 연속 안타 팀 타이 기록(1981년 <a class="new" href="/w/index.php?title=%EC%A0%9C%EC%8B%9C_%EB%B0%94%ED%95%84%EB%93%9C&amp;action=edit&amp;redlink=1" title="제시 바필드 (없는 문서)">제시 바필드</a>도 데뷔 이후 8경기 연속 안타)을 세운다. 하지만, <a href="/wiki/9%EC%9B%94_1%EC%9D%BC" title="9월 1일">9월 1일</a>에 있었던 <a href="/wiki/%EC%BA%94%EC%9E%90%EC%8A%A4%EC%8B%9C%ED%8B%B0_%EB%A1%9C%EC%97%B4%EC%8A%A4" title="캔자스시티 로열스">캔자스시티 로열스</a>전에서 안타 없이 2개의 삼진만을 기록하며 기록은 끊긴다. <a href="/wiki/9%EC%9B%94_18%EC%9D%BC" title="9월 18일">9월 18일</a>에 있었던 <a href="/wiki/%EB%89%B4%EC%9A%95_%EC%96%91%ED%82%A4%EC%8A%A4" title="뉴욕 양키스">뉴욕 양키스</a>전에서는 <a href="/wiki/%EB%8D%B0%EC%9D%B4%EB%B9%84%EB%93%9C_%ED%97%88%ED%94%84" title="데이비드 허프">데이비드 허프</a>를 상대로 자신의 메이저 리그 커리어 첫 홈런을 달성한다. 시즌에서는 34경기에 나와 .252이라는 신인으로서 나쁘지 않은 타율과 2개의 홈런, 8개의 타점을 기록한다.</p>
<p><a href="/wiki/2014%EB%85%84" title="2014년">2014년</a>에 있었던 <a href="/wiki/%EC%8A%A4%ED%94%84%EB%A7%81_%ED%8A%B8%EB%A0%88%EC%9D%B4%EB%8B%9D" title="스프링 트레이닝">스프링 트레이닝</a>에 참여했으며, 주전 <a href="/wiki/2%EB%A3%A8%EC%88%98" title="2루수">2루수</a>로 낙점된다. 그러나 시즌에서는 부진을 겪으면서 트리플A를 왔다갔다하며 67경기에 출장하여 .188의 타율과 1개의 홈런, 15개의 타점을 기록한다.</p>
