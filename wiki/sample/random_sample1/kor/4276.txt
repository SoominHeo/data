<p><b>라이언 제임스 쇼크로스</b>(<span style="font-size: smaller;"><a href="/wiki/%EC%98%81%EC%96%B4" title="영어">영어</a>:</span> <span lang="en" xml:lang="en">Ryan James Shawcross</span>, <a href="/wiki/1987%EB%85%84" title="1987년">1987년</a> <a href="/wiki/10%EC%9B%94_4%EC%9D%BC" title="10월 4일">10월 4일</a> ~ )는 <a href="/wiki/%EC%9E%89%EA%B8%80%EB%9E%9C%EB%93%9C" title="잉글랜드">잉글랜드</a>의 <a href="/wiki/%EC%B6%95%EA%B5%AC_%EC%84%A0%EC%88%98" title="축구 선수">축구 선수</a>로서, 포지션은 <a href="/wiki/%EC%88%98%EB%B9%84%EC%88%98#.EC.84.BC.ED.84.B0.EB.B0.B1" title="수비수">센터백</a>이다. 현재 <a href="/wiki/%EC%8A%A4%ED%86%A0%ED%81%AC_%EC%8B%9C%ED%8B%B0_FC" title="스토크 시티 FC">스토크 시티</a>에서 활약하고 있다.</p>
<h2><span class="mw-headline" id=".EC.B6.95.EA.B5.AC.EC.9D.B8_.EA.B2.BD.EB.A0.A5">축구인 경력</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%9D%BC%EC%9D%B4%EC%96%B8_%EC%87%BC%ED%81%AC%EB%A1%9C%EC%8A%A4&amp;action=edit&amp;section=1" title="부분 편집: 축구인 경력">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<h3><span class="mw-headline" id=".EC.84.A0.EC.88.98_.EA.B2.BD.EB.A0.A5">선수 경력</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%9D%BC%EC%9D%B4%EC%96%B8_%EC%87%BC%ED%81%AC%EB%A1%9C%EC%8A%A4&amp;action=edit&amp;section=2" title="부분 편집: 선수 경력">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p><a href="/wiki/2006%EB%85%84" title="2006년">2006년</a> <a href="/wiki/10%EC%9B%94_25%EC%9D%BC" title="10월 25일">10월 25일</a> <a class="mw-redirect" href="/wiki/%ED%92%8B%EB%B3%BC_%EB%A6%AC%EA%B7%B8_%EC%BB%B5" title="풋볼 리그 컵">풋볼 리그 컵</a> 경기에 출전하여 데뷔전을 치렀으나 리그에는 한 경기도 출전하지 못하였다. 이후 <a href="/wiki/%EB%A1%9C%EC%97%B4_%EC%95%A4%ED%8A%B8%EC%9B%8C%ED%94%84_FC" title="로열 앤트워프 FC">로열 앤트워프</a>에서 임대 생활을 한 후 <a href="/wiki/2007%EB%85%84" title="2007년">2007년</a> <a href="/wiki/8%EC%9B%94" title="8월">8월</a> <a href="/wiki/%EC%8A%A4%ED%86%A0%ED%81%AC_%EC%8B%9C%ED%8B%B0_FC" title="스토크 시티 FC">스토크 시티</a>에 임대 이적하였다. <a href="/wiki/%EC%B9%B4%EB%94%94%ED%94%84_%EC%8B%9C%ED%8B%B0_FC" title="카디프 시티 FC">카디프</a>와의 데뷔 경기에서 데뷔골을 터뜨리며 강한 인상을 심어주었고 이내 스토크의 주전 센터백으로 도약하였다. 2007-08 시즌 <a href="/wiki/%ED%92%8B%EB%B3%BC_%EB%A6%AC%EA%B7%B8_%EC%B1%94%ED%94%BC%EC%96%B8%EC%8B%AD" title="풋볼 리그 챔피언십">풋볼 리그 챔피언십</a> 베스트 11에 선정되는 등 좋은 활약을 펼치자 이에 매료된 스토크는 맨유에 완전 영입을 제의하였고 <a href="/wiki/2008%EB%85%84" title="2008년">2008년</a> <a href="/wiki/1%EC%9B%94_18%EC%9D%BC" title="1월 18일">1월 18일</a> 완전 영입에 성공하였다. 2008-09 시즌 초반 새로 영입된 <a class="new" href="/w/index.php?title=%EC%9D%B4%EB%B8%8C%EB%9D%BC%ED%9E%88%EB%A7%88_%EC%86%A1%EC%BD%94&amp;action=edit&amp;redlink=1" title="이브라히마 송코 (없는 문서)">이브라히마 송코</a>에게 주전 자리를 잠시 내주었으나 다시 주전 자리를 되찾았다. 시즌 2008-09 시즌 3골을 터뜨렸는 데 시즌 막바지 <a href="/wiki/%EB%AF%B8%EB%93%A4%EC%A6%88%EB%B8%8C%EB%9F%AC_FC" title="미들즈브러 FC">미들즈브러</a>와의 경기에서 터뜨린 득점은 팀을 강등에서 구해내는 데 중요한 역할을 하였다.</p>
<p><a href="/wiki/2010%EB%85%84" title="2010년">2010년</a> <a href="/wiki/2%EC%9B%94" title="2월">2월</a> <a href="/wiki/%EC%95%84%EC%8A%A4%EB%84%90_FC" title="아스널 FC">아스널</a>과의 경기에서 <a href="/wiki/%EC%97%90%EB%9F%B0_%EB%9E%A8%EC%A7%80" title="에런 램지">에런 램지</a>에게 가한 태클이 램지에게 장기 부상을 입히면서 논란이 되었다. 2010-11 시즌을 앞두고 팀의 주장으로 선임되었다. <a href="/wiki/2012%EB%85%84" title="2012년">2012년</a> <a href="/wiki/10%EC%9B%94" title="10월">10월</a> <a href="/wiki/%EC%82%B0%EB%A7%88%EB%A6%AC%EB%85%B8_%EC%B6%95%EA%B5%AC_%EA%B5%AD%EA%B0%80%EB%8C%80%ED%91%9C%ED%8C%80" title="산마리노 축구 국가대표팀">산마리노</a>와 <a href="/wiki/%ED%8F%B4%EB%9E%80%EB%93%9C_%EC%B6%95%EA%B5%AC_%EA%B5%AD%EA%B0%80%EB%8C%80%ED%91%9C%ED%8C%80" title="폴란드 축구 국가대표팀">폴란드</a>와의 경기를 앞두고 <a href="/wiki/%EC%9E%89%EA%B8%80%EB%9E%9C%EB%93%9C_%EC%B6%95%EA%B5%AC_%EA%B5%AD%EA%B0%80%EB%8C%80%ED%91%9C%ED%8C%80" title="잉글랜드 축구 국가대표팀">잉글랜드 대표팀</a>에 처음으로 선발되었다.</p>
