<h1 class="firstHeading" id="firstHeading" lang="ko">라이트 펜</h1>
<div class="mw-body-content" id="bodyContent">
<div id="siteSub">위키백과, 우리 모두의 백과사전.</div>

<p><b>라이트 펜</b>(light pen)은 빛에 반응하는 막대 모양의 <a href="/wiki/%EC%BB%B4%ED%93%A8%ED%84%B0" title="컴퓨터">컴퓨터</a> <a href="/wiki/%EC%BB%B4%ED%93%A8%ED%84%B0_%EC%9E%85%EB%A0%A5_%EC%9E%A5%EC%B9%98" title="컴퓨터 입력 장치">입력 장치</a>이다. 컴퓨터 <a href="/wiki/%EC%9D%8C%EA%B7%B9%EC%84%A0%EA%B4%80" title="음극선관">CRT</a> 텔레비전 수상기나 <a class="mw-redirect" href="/wiki/%EC%BB%B4%ED%93%A8%ED%84%B0_%EB%94%94%EC%8A%A4%ED%94%8C%EB%A0%88%EC%9D%B4" title="컴퓨터 디스플레이">모니터</a>에 접촉하여 사용한다. 사용자가 표시 물체를 가리키거나 화면에 그리는 것으로, 마치 <a href="/wiki/%ED%84%B0%EC%B9%98%EC%8A%A4%ED%81%AC%EB%A6%B0" title="터치스크린">터치 스크린</a>과 비슷하지만, 위치 감지에 대한 더 나은 정확도를 제공한다. 한동안 라이트 펜이 CRT 기반의 디스플레이에서만 동작할 수 있으며 <a href="/wiki/%EC%95%A1%EC%A0%95_%EB%94%94%EC%8A%A4%ED%94%8C%EB%A0%88%EC%9D%B4" title="액정 디스플레이">LCD</a> 화면(다만 도시바와 히타치가 일본의 "디즈플레이 2006"에서 비슷한 아이디어를 보여주긴 하였음<sup class="reference" id="cite_ref-1"><a href="#cite_note-1">[1]</a></sup>), 프로젝터, 기타 디스플레이 장치에서는 동작하지 않는다고 여겨졌다. 그러나 2011년에 <a class="new" href="/w/index.php?title=%ED%8E%98%EC%96%B4%EB%9D%BC%EC%9D%B4%ED%8A%B8&amp;action=edit&amp;redlink=1" title="페어라이트 (없는 문서)">페어라이트 인스트루먼츠</a>가 라이트 펜을 사용할 수 있는 17인치 LCD 모니터 <a class="new" href="/w/index.php?title=%ED%8E%98%EC%96%B4%EB%9D%BC%EC%9D%B4%ED%8A%B8_CMI&amp;action=edit&amp;redlink=1" title="페어라이트 CMI (없는 문서)">페어라이트 CMI-30A</a>를 출시하였다.</p>
<p>라이트 펜은 추가하기가 매우 단순하다. 마치 <a href="/wiki/%EB%9D%BC%EC%9D%B4%ED%8A%B8_%EA%B1%B4" title="라이트 건">라이트 건</a>처럼, 라이트 펜은 <a class="new" href="/w/index.php?title=%EC%A0%84%EC%9E%90%EC%B4%9D&amp;action=edit&amp;redlink=1" title="전자총 (없는 문서)">전자총</a>이 화면의 특정 지점을 새로 고칠 때 해당 지점의 밝기의 갑작스런 조그마한 변화에 반응하면서 동작한다.</p>
<h2><span class="mw-headline" id=".EC.97.AD.EC.82.AC">역사</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%9D%BC%EC%9D%B4%ED%8A%B8_%ED%8E%9C&amp;action=edit&amp;section=1" title="부분 편집: 역사">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>최초의 라이트 펜은 1955년 즈음에 <a href="/wiki/%EB%A7%A4%EC%82%AC%EC%B6%94%EC%84%B8%EC%B8%A0_%EA%B3%B5%EA%B3%BC%EB%8C%80%ED%95%99%EA%B5%90" title="매사추세츠 공과대학교">MIT</a>의 <a class="new" href="/w/index.php?title=%ED%9B%A8%EC%9C%88%EB%93%9C&amp;action=edit&amp;redlink=1" title="훨윈드 (없는 문서)">훨윈드</a>(Whirwind) 프로젝트의 일부로 나왔다.<sup class="reference" id="cite_ref-2"><a href="#cite_note-2">[2]</a></sup><sup class="reference" id="cite_ref-3"><a href="#cite_note-3">[3]</a></sup></p>
<p>라이트 펜은 1980년대 초에 어느 정도 인기가 있었다. <a class="new" href="/w/index.php?title=%ED%8E%98%EC%96%B4%EB%9D%BC%EC%9D%B4%ED%8A%B8_CMI&amp;action=edit&amp;redlink=1" title="페어라이트 CMI (없는 문서)">페어라이트 CMI</a>, <a class="new" href="/w/index.php?title=BBC_%EB%A7%88%EC%9D%B4%ED%81%AC%EB%A1%9C&amp;action=edit&amp;redlink=1" title="BBC 마이크로 (없는 문서)">BBC 마이크로</a>를 통해 유명해졌다. <a href="/wiki/IBM_PC" title="IBM PC">IBM PC</a> 호환 <a href="/wiki/%EC%BB%AC%EB%9F%AC_%EA%B7%B8%EB%9E%98%ED%94%BD%EC%8A%A4_%EC%96%B4%EB%8C%91%ED%84%B0" title="컬러 그래픽스 어댑터">CGA</a>, <a href="/wiki/%ED%97%88%ED%81%98%EB%A0%88%EC%8A%A4_%EA%B7%B8%EB%9E%98%ED%94%BD_%EC%B9%B4%EB%93%9C" title="허큘레스 그래픽 카드">HGC</a>, 또 일부 <a href="/wiki/%EA%B0%95%ED%99%94_%EA%B7%B8%EB%9E%98%ED%94%BD_%EC%96%B4%EB%8C%91%ED%84%B0" title="강화 그래픽 어댑터">EGA</a> 그래픽 카드는 라이트 펜을 위한 단자도 갖추고 있었다. 일부 소비자 제품에도 라이트 펜이 제공되었는데, 이 가운데 <a class="new" href="/w/index.php?title=%ED%86%B0%EC%8A%A8_MO5&amp;action=edit&amp;redlink=1" title="톰슨 MO5 (없는 문서)">톰슨 MO5</a> 컴퓨터 계열을 들 수 있다. 사용자가 오랜 시간 동안 화면 위에다 펜을 잡고 있어야 했으므로 라이트 펜은 일반 목적의 입력 장치로서 점차 사용하지 않게 되었다.</p>
