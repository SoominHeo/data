<p><b>라트비아 소비에트 사회주의 공화국</b>(<span style="font-size: smaller;"><a href="/wiki/%EB%9D%BC%ED%8A%B8%EB%B9%84%EC%95%84%EC%96%B4" title="라트비아어">라트비아어</a>:</span> <span lang="lv" xml:lang="lv">Latvijas Padomju Sociālistiskā Republika / Latvijas PSR</span> <small>라트비야스 파도뮤 소치알리스티스카 레푸블리카</small>, <span style="font-size: smaller;"><a href="/wiki/%EB%9F%AC%EC%8B%9C%EC%95%84%EC%96%B4" title="러시아어">러시아어</a>:</span> <span lang="ru" xml:lang="ru">Латвийская Советская Социалистическая Республика / ЛССР</span> <small>라트비스카야 소베츠카야 소치알리스티체스카야 레스푸블리카 / 엘에스에스에르<sup>[<a href="/wiki/%EC%9C%84%ED%82%A4%EB%B0%B1%EA%B3%BC:%EB%9F%AC%EC%8B%9C%EC%95%84%EC%96%B4%EC%9D%98_%ED%95%9C%EA%B8%80_%ED%91%9C%EA%B8%B0" title="위키백과:러시아어의 한글 표기">*</a>]</sup></small>)은 <a href="/wiki/1940%EB%85%84" title="1940년">1940년</a> 당시 <a class="mw-redirect" href="/wiki/%EC%86%8C%EB%B9%84%EC%97%90%ED%8A%B8_%EC%97%B0%EB%B0%A9" title="소비에트 연방">소비에트 연방</a>에 의해 불법 점령을 당하고부터 <a href="/wiki/1990%EB%85%84" title="1990년">1990년</a>에 독립할 때까지 존재했던 옛 공화국이었다. 현재는 주권 국가인 <a href="/wiki/%EB%9D%BC%ED%8A%B8%EB%B9%84%EC%95%84" title="라트비아">라트비아</a>로 독립되어 있으며 라트비아의 반러시아적인 성향은 아직도 남아 있다.</p>

<h2><span class="mw-headline" id=".EC.97.AD.EC.82.AC">역사</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%9D%BC%ED%8A%B8%EB%B9%84%EC%95%84_%EC%86%8C%EB%B9%84%EC%97%90%ED%8A%B8_%EC%82%AC%ED%9A%8C%EC%A3%BC%EC%9D%98_%EA%B3%B5%ED%99%94%EA%B5%AD&amp;action=edit&amp;section=2" title="부분 편집: 역사">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p><a href="/wiki/1918%EB%85%84" title="1918년">1918년</a> 이래 독립국인 <a href="/wiki/%EB%9D%BC%ED%8A%B8%EB%B9%84%EC%95%84" title="라트비아">라트비아</a>였는데, <a href="/wiki/1940%EB%85%84" title="1940년">1940년</a> <a href="/wiki/8%EC%9B%94_5%EC%9D%BC" title="8월 5일">8월 5일</a> <a href="/wiki/%EC%86%8C%EB%A0%A8" title="소련">소련</a>에 강제 합병되어 15번째 공화국이 되었다. 라트비아 SSR은 <a href="/wiki/1990%EB%85%84" title="1990년">1990년</a> 소련 쿠데타 실패 직후 <a href="/wiki/%EC%97%90%EC%8A%A4%ED%86%A0%EB%8B%88%EC%95%84" title="에스토니아">에스토니아</a>, <a href="/wiki/%EB%A6%AC%ED%88%AC%EC%95%84%EB%8B%88%EC%95%84" title="리투아니아">리투아니아</a>와 함께 독립을 선언해 그 해 9월 <a href="/wiki/%EC%9C%A0%EC%97%94" title="유엔">유엔</a>에 가입했다.</p>
<h2><span class="mw-headline" id=".EC.A3.BC.EB.AF.BC">주민</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%9D%BC%ED%8A%B8%EB%B9%84%EC%95%84_%EC%86%8C%EB%B9%84%EC%97%90%ED%8A%B8_%EC%82%AC%ED%9A%8C%EC%A3%BC%EC%9D%98_%EA%B3%B5%ED%99%94%EA%B5%AD&amp;action=edit&amp;section=3" title="부분 편집: 주민">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p><a href="/wiki/%EB%9D%BC%ED%8A%B8%EB%B9%84%EC%95%84%EC%9D%B8" title="라트비아인">라트비아인</a>들은 <a href="/wiki/%EC%9D%B4%EC%98%A4%EC%8B%9C%ED%94%84_%EC%8A%A4%ED%83%88%EB%A6%B0" title="이오시프 스탈린">이오시프 스탈린</a>의 죽음을 환영하였다.</p>
<p><a href="/wiki/%EC%86%8C%EB%A0%A8" title="소련">소련</a> 당국은 정책적으로 이 지역에 <a href="/wiki/%EB%9F%AC%EC%8B%9C%EC%95%84%EC%9D%B8" title="러시아인">러시아인</a>을 많이 이주시켜, 러시아인의 비율이 30%대에 이르렀다.</p>
<h2><span class="mw-headline" id=".EC.96.B8.EC.96.B4.EC.99.80_.EC.A2.85.EA.B5.90">언어와 종교</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%9D%BC%ED%8A%B8%EB%B9%84%EC%95%84_%EC%86%8C%EB%B9%84%EC%97%90%ED%8A%B8_%EC%82%AC%ED%9A%8C%EC%A3%BC%EC%9D%98_%EA%B3%B5%ED%99%94%EA%B5%AD&amp;action=edit&amp;section=4" title="부분 편집: 언어와 종교">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p><a href="/wiki/%EB%9D%BC%ED%8A%B8%EB%B9%84%EC%95%84%EC%96%B4" title="라트비아어">라트비아어</a>와 <a href="/wiki/%EB%9F%AC%EC%8B%9C%EC%95%84%EC%96%B4" title="러시아어">러시아어</a>가 사실상 <a href="/wiki/%EA%B3%B5%EC%9A%A9%EC%96%B4" title="공용어">공용어</a>였고 대부분 <a href="/wiki/%EA%B5%90%ED%9A%8C" title="교회">교회</a>를 믿었다. <sup class="정리_필요 noprint">[<a href="/wiki/%EC%9C%84%ED%82%A4%EB%B0%B1%EA%B3%BC:%EC%B6%9C%EC%B2%98_%EB%B0%9D%ED%9E%88%EA%B8%B0" title="위키백과:출처 밝히기">출처 필요</a>]</sup></p>
