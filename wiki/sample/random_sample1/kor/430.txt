<h1 class="firstHeading" id="firstHeading" lang="ko">백분율 기호</h1>
<div class="mw-body-content" id="bodyContent">
<div id="siteSub">위키백과, 우리 모두의 백과사전.</div>
<div id="contentSub"><span class="mw-redirectedfrom">(<a class="mw-redirect" href="/w/index.php?title=%25&amp;redirect=no" title="%">%</a>에서 넘어옴)</span></div>
<div class="mw-jump" id="jump-to-nav">
					이동:					<a href="#mw-head">둘러보기</a>, 					<a href="#p-search">검색</a>
</div>
<div class="mw-content-ltr" dir="ltr" id="mw-content-text" lang="ko"><script>function mfTempOpenSection(id){var block=document.getElementById("mf-section-"+id);block.className+=" open-block";block.previousSibling.className+=" open-block";}</script><div class="unicode" style="float:right; clear: right; margin: 10px; padding:30px; font-size: 400%; background-color: #ddddff; border: 1px solid #aaaaff;">%</div>
<p><b>백분율 기호</b>(%) 또는 <b>퍼센트 기호</b>는 <a href="/wiki/%EB%B0%B1%EB%B6%84%EC%9C%A8" title="백분율">백분율</a>(앞 수가 100으로 나뉜 것)을 나타내는 데 쓰이는 기호로, <a href="/wiki/%EC%9C%A0%EB%8B%88%EC%BD%94%EB%93%9C" title="유니코드">유니코드</a>에는 U+0025에 할당되어 있다. 관련된 기호로 <a href="/wiki/%EC%B2%9C%EB%B6%84%EC%9C%A8" title="천분율">천분율</a> 기호 <span class="Unicode">‰</span>(U+2030)과 <a href="/wiki/%EB%A7%8C%EB%B6%84%EC%9C%A8" title="만분율">만분율</a> 기호 <span class="Unicode">‱</span>(U+2031, <a class="mw-redirect" href="/wiki/%EB%B2%A0%EC%9D%B4%EC%8B%9C%EC%8A%A4_%ED%8F%AC%EC%9D%B8%ED%8A%B8" title="베이시스 포인트">베이시스 포인트</a>라고도 함)가 있는데, 각각 1000분의 1과 10000분의 1을 가리킨다. 이보다 더 높은 비율은 parts-per 표기법을 사용한다.</p>
<p>숫자와 숫자 뒤의 % 사이에 빈칸을 두지 않는 것이 스타일 지침의 일반적인 관례이다.</p>

<h2><span class="mw-headline" id=".EC.BB.B4.ED.93.A8.ED.84.B0.EC.97.90.EC.84.9C.EC.9D.98_.25">컴퓨터에서의 %</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%B0%B1%EB%B6%84%EC%9C%A8_%EA%B8%B0%ED%98%B8&amp;action=edit&amp;section=2" title="부분 편집: 컴퓨터에서의 %">원본 편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>퍼센트 문자는 <a class="mw-redirect" href="/wiki/%EB%AF%B8%EA%B5%AD_%EC%A0%95%EB%B3%B4_%EA%B5%90%ED%99%98_%ED%91%9C%EC%A4%80_%EB%B6%80%ED%98%B8" title="미국 정보 교환 표준 부호">ASCII</a>, <a href="/wiki/%EC%9C%A0%EB%8B%88%EC%BD%94%EB%93%9C" title="유니코드">유니코드</a>를 비롯한 대부분의 <a href="/wiki/%EB%AC%B8%EC%9E%90_%EC%9D%B8%EC%BD%94%EB%94%A9#.EB.AC.B8.EC.9E.90_.EC.9D.B8.EC.BD.94.EB.94.A9.EC.9D.98_.EA.B5.AC.EC.84.B1" title="문자 인코딩">문자 집합</a> 및 <a href="/wiki/%EB%AC%B8%EC%9E%90_%EC%9D%B8%EC%BD%94%EB%94%A9" title="문자 인코딩">인코딩</a>에 들어 있으며, 보통 0x25(십진수 37)에 배당되어 있다. 유니코드에는 % 기호의 양쪽 귀퉁이의 동그라미 두 개가 사각형 점으로 바뀐 아라비아 퍼센트 기호(٪: U+066A)도 배정되어 있다.</p>
<p>퍼센트 문자는 컴퓨터 <a href="/wiki/%ED%94%84%EB%A1%9C%EA%B7%B8%EB%9E%98%EB%B0%8D_%EC%96%B8%EC%96%B4" title="프로그래밍 언어">프로그래밍 언어</a>에서 <a class="mw-redirect" href="/wiki/%EB%82%98%EB%A8%B8%EC%A7%80_%EC%97%B0%EC%82%B0" title="나머지 연산">나머지 연산</a> 연산자로도 사용된다. 가능한 정밀도 내에서 나머지 없는 나눗셈의 몫을 구하는 <a class="mw-redirect" href="/wiki/%EC%8A%AC%EB%9E%98%EC%8B%9C" title="슬래시">/ (슬래시)</a>와 달리 퍼센트 기호는 나눗셈의 나머지를 구할 때 사용한다. <span class="fix" title="이 부분은 출처가 필요합니다.">이런 관례는 <a href="/wiki/C_(%ED%94%84%EB%A1%9C%EA%B7%B8%EB%9E%98%EB%B0%8D_%EC%96%B8%EC%96%B4)" title="C (프로그래밍 언어)">C</a>에서 유래하여 널리 쓰이게 된 것이지만 그 이전의 <a href="/wiki/B_(%ED%94%84%EB%A1%9C%EA%B7%B8%EB%9E%98%EB%B0%8D_%EC%96%B8%EC%96%B4)" title="B (프로그래밍 언어)">B 프로그래밍 언어</a>에도 존재했다.</span><sup class="정리_필요 noprint">[<a href="/wiki/%EC%9C%84%ED%82%A4%EB%B0%B1%EA%B3%BC:%EC%B6%9C%EC%B2%98_%EB%B0%9D%ED%9E%88%EA%B8%B0" title="위키백과:출처 밝히기">출처 필요</a>]</sup></p>
<p>또한 퍼센트 문자는 <a href="/wiki/%ED%86%B5%ED%95%A9_%EC%9E%90%EC%9B%90_%EC%8B%9D%EB%B3%84%EC%9E%90" title="통합 자원 식별자">URI</a>에서 직접 사용할 수 없는 <a href="/wiki/%EC%98%A5%ED%85%9F_(%EC%BB%B4%ED%93%A8%ED%8C%85)" title="옥텟 (컴퓨팅)">옥텟</a>을 표현하기 위해 사용된다. 즉, <code>%25</code>는 0x25라는 값을 가지는 옥텟, 즉 퍼센트 기호 자신을 나타낸다. <a href="/wiki/SQL" title="SQL">SQL</a>을 비롯한 몇몇 프로그래밍 언어에서는 모든 값을 나타내는 <a class="new" href="/w/index.php?title=%EC%99%80%EC%9D%BC%EB%93%9C_%EC%B9%B4%EB%93%9C_(%EC%BB%B4%ED%93%A8%ED%84%B0)&amp;action=edit&amp;redlink=1" title="와일드 카드 (컴퓨터) (없는 문서)">와일드 카드</a>로 퍼센트 기호를 쓰기도 한다.</p>
