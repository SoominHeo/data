<p><b>라티나</b>(<span style="font-size: smaller;"><a href="/wiki/%EC%9D%B4%ED%83%88%EB%A6%AC%EC%95%84%EC%96%B4" title="이탈리아어">이탈리아어</a>:</span> <span lang="it" xml:lang="it">Latina</span>)는 <a href="/wiki/%EC%9D%B4%ED%83%88%EB%A6%AC%EC%95%84" title="이탈리아">이탈리아</a> <a href="/wiki/%EB%9D%BC%EC%B9%98%EC%98%A4_%EC%A3%BC" title="라치오 주">라치오 주</a> <a href="/wiki/%EB%9D%BC%ED%8B%B0%EB%82%98_%ED%98%84" title="라티나 현">라티나 현</a>에 위치한 코무네다. <a href="/wiki/%EB%9D%BC%ED%8B%B0%EB%82%98_%ED%98%84" title="라티나 현">라티나 현</a>의 현도이며, <a href="/wiki/%EB%A1%9C%EB%A7%88" title="로마">로마</a> 다음으로 주에서 인구가 많은 도시이기도 하다.</p>

<h2><span class="mw-headline" id=".EC.97.AD.EC.82.AC">역사</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%9D%BC%ED%8B%B0%EB%82%98&amp;action=edit&amp;section=1" title="부분 편집: 역사">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p><a href="/wiki/1932%EB%85%84" title="1932년">1932년</a> <a href="/wiki/%EB%B2%A0%EB%8B%88%ED%86%A0_%EB%AC%B4%EC%86%94%EB%A6%AC%EB%8B%88" title="베니토 무솔리니">무솔리니</a> 시대에 <a href="/wiki/%EB%A1%9C%EB%A7%88" title="로마">로마</a> 동남쪽 60km 지점의 간척지 위에 건설된 새로운 계획도시이다. 처음에는 <b>리토리아</b>(<span style="font-size: smaller;"><a href="/wiki/%EC%9D%B4%ED%83%88%EB%A6%AC%EC%95%84%EC%96%B4" title="이탈리아어">이탈리아어</a>:</span> <span lang="it" xml:lang="it">Littoria</span>)라 불렀다. <a href="/wiki/1934%EB%85%84" title="1934년">1934년</a> <a href="/wiki/%EB%A1%9C%EB%A7%88_%ED%98%84" title="로마 현">로마 현</a>에서 분리된 <a class="new" href="/w/index.php?title=%EB%A6%AC%ED%86%A0%EB%A6%AC%EC%95%84_%ED%98%84&amp;action=edit&amp;redlink=1" title="리토리아 현 (없는 문서)">리토리아 현</a>의 현청이 되었다. 8각형으로 구획 정리된 가로 곳곳에 광장과 공원을 배치하였다. <a href="/wiki/1947%EB%85%84" title="1947년">1947년</a> 현과 시의 이름은 라티나로 개칭되었으며, 새로운 간척지의 중심지로 발전하였다.</p>
<h2><span class="mw-headline" id=".EA.B2.BD.EC.A0.9C">경제</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%9D%BC%ED%8B%B0%EB%82%98&amp;action=edit&amp;section=2" title="부분 편집: 경제">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p><a href="/wiki/%EC%A0%9C%EC%95%BD" title="제약">제약</a>, <a class="new" href="/w/index.php?title=%ED%99%94%ED%95%99_%EC%82%B0%EC%97%85&amp;action=edit&amp;redlink=1" title="화학 산업 (없는 문서)">화학 산업</a>, <a href="/wiki/%EC%B9%98%EC%A6%88" title="치즈">치즈</a> 생산업이 있고 뛰어난 서비스 부문을 가지고 있다. 농업 중심지이기도 하다.</p>
<p><a class="new" href="/w/index.php?title=%EB%9D%BC%ED%8B%B0%EB%82%98_%ED%95%B5%EB%B0%9C%EC%A0%84%EC%86%8C&amp;action=edit&amp;redlink=1" title="라티나 핵발전소 (없는 문서)">라티나 핵발전소</a>가 있었지만 지금은 가동을 중단하고 해체 작업을 하고 있다.</p>
<h2><span class="mw-headline" id=".EA.B8.B0.ED.9B.84">기후</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%9D%BC%ED%8B%B0%EB%82%98&amp;action=edit&amp;section=3" title="부분 편집: 기후">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
