<p><b>라틴어</b>(<span lang="la" xml:lang="la">Lingua Latīna</span>)는 <a href="/wiki/%EC%9D%B4%ED%83%88%EB%A6%AC%EC%95%84" title="이탈리아">이탈리아</a> 반도의 중부에 있는 <a class="mw-redirect" href="/wiki/%EA%B3%A0%EB%8C%80_%EB%A1%9C%EB%A7%88%EC%82%AC" title="고대 로마사">고대 로마</a>와 그 주변 지역 <a href="/wiki/%EB%9D%BC%ED%8B%B0%EC%9B%80" title="라티움">라티움</a>(Latium)에 정착하여 살던 라티움 사람들이 쓰던 언어이다. 로마가 지중해를 정복하면서 라틴어는 지중해 전역과 유럽 지역의 상당 부분으로 퍼져나갔다. 오늘날 라틴어는 사어가 되어<sup class="reference" id="cite_ref-1"><a href="#cite_note-1">[1]</a></sup> <a href="/wiki/%EC%9D%B4%ED%83%88%EB%A6%AC%EC%95%84%EC%96%B4" title="이탈리아어">이탈리아어</a>, <a href="/wiki/%ED%94%84%EB%9E%91%EC%8A%A4%EC%96%B4" title="프랑스어">프랑스어</a>, <a href="/wiki/%EC%8A%A4%ED%8E%98%EC%9D%B8%EC%96%B4" title="스페인어">스페인어</a>, <a href="/wiki/%ED%8F%AC%EB%A5%B4%ED%88%AC%EA%B0%88%EC%96%B4" title="포르투갈어">포르투갈어</a>, <a href="/wiki/%EB%A3%A8%EB%A7%88%EB%8B%88%EC%95%84%EC%96%B4" title="루마니아어">루마니아어</a> 등 <a class="mw-redirect" href="/wiki/%EB%A1%9C%EB%A7%9D%EC%8A%A4%EC%96%B4" title="로망스어">로망스어</a>가 라틴어의 후손이다. 그 외 <a href="/wiki/%EC%98%81%EC%96%B4" title="영어">영어</a> 등 다른 언어들도 라틴어에서 많은 어휘를 차용했다.</p>
<p><a href="/wiki/17%EC%84%B8%EA%B8%B0" title="17세기">17세기</a>까지 라틴어에는 두 종류가 있다. 운문과 산문에 쓰는 문학 언어인 <a href="/wiki/%EA%B3%A0%EC%A0%84_%EB%9D%BC%ED%8B%B4%EC%96%B4" title="고전 라틴어">고전 라틴어</a>와 일반 대중이 쓴 <a class="mw-redirect" href="/wiki/%EB%AF%BC%EC%A4%91_%EB%9D%BC%ED%8B%B4%EC%96%B4" title="민중 라틴어">민중 라틴어</a>(Latina Vulgata)가 있다. 민중 라틴어는 로마제국이 멸망한 뒤에서 유럽 많은 지역에서 구어로 남았으며, 9세기에 이르면 <a class="mw-redirect" href="/wiki/%EB%A1%9C%EB%A7%9D%EC%8A%A4%EC%96%B4" title="로망스어">로망스어</a>로 분화된다.</p>
<p>로마 제국 멸망 이후에도 라틴어는 서양 세계의 지식인 사이에서 <a href="/wiki/%EB%A7%81%EA%B5%AC%EC%95%84_%ED%94%84%EB%9E%91%EC%B9%B4" title="링구아 프랑카">링구아 프랑카</a>(lingua franca)로 남았는데, <a href="/wiki/%EB%A1%9C%EB%A7%88_%EA%B0%80%ED%86%A8%EB%A6%AD%EA%B5%90%ED%9A%8C" title="로마 가톨릭교회">로마 가톨릭교회</a>가 라틴어를 채택한 것도 이에 큰 몫을 했다. 이런 환경에서 라틴어는 <a class="mw-redirect" href="/wiki/2%EC%B2%9C%EB%85%84%EA%B8%B0" title="2천년기">2천년기</a>까지 <a class="mw-redirect" href="/wiki/%EB%AA%A8%EA%B5%AD%EC%96%B4" title="모국어">모국어</a>(mother tongue)으로 존속했는데 이것을 <a class="new" href="/w/index.php?title=%EC%A4%91%EC%84%B8_%EB%9D%BC%ED%8B%B4%EC%96%B4&amp;action=edit&amp;redlink=1" title="중세 라틴어 (없는 문서)">중세 라틴어</a>라 한다. <a href="/wiki/%EB%A5%B4%EB%84%A4%EC%83%81%EC%8A%A4" title="르네상스">르네상스</a> 시대에 라틴어를 르네상스 인문주의자이 받아들이면서 그 위상이 잠시 높아졌으나 <a href="/wiki/16%EC%84%B8%EA%B8%B0" title="16세기">16세기</a> 이후 중세 라틴어는 점차 쇠퇴한다.</p>
<p>라틴어는 <a class="new" href="/w/index.php?title=%EA%B5%90%ED%9A%8C_%EB%9D%BC%ED%8B%B4%EC%96%B4&amp;action=edit&amp;redlink=1" title="교회 라틴어 (없는 문서)">교회 라틴어</a>의 형태로 <a href="/wiki/%EB%A1%9C%EB%A7%88_%EA%B0%80%ED%86%A8%EB%A6%AD%EA%B5%90%ED%9A%8C" title="로마 가톨릭교회">로마 가톨릭교회</a>에서 교황의 조서(papal bull)나 칙령에 쓰인다. 라틴어는 과거 <a href="/wiki/%EC%9C%A0%EB%9F%BD" title="유럽">유럽</a>에서 <a href="/wiki/%EA%B3%BC%ED%95%99" title="과학">과학</a>, <a class="mw-redirect" href="/wiki/%ED%95%99%EC%88%A0" title="학술">학술</a>, <a href="/wiki/%EB%B2%95" title="법">법</a> 분야의 언어로 널리 쓰였다. <a href="/wiki/20%EC%84%B8%EA%B8%B0" title="20세기">20세기</a> 초 이래 라틴어의 역할은 줄어들었지만, 로마 공화정 후기에서 제정 초기에 쓰이던 문어인 고전 라틴어를 오늘날에도 서구의 여러 각급 <a href="/wiki/%ED%95%99%EA%B5%90" title="학교">학교</a>에서 가르치며, 고전 연구에서 <a href="/wiki/%EA%B7%B8%EB%A6%AC%EC%8A%A4%EC%96%B4" title="그리스어">그리스어</a>와 함께 다룬다.</p>

<h2><span class="mw-headline" id=".EB.AC.B8.EC.9E.90">문자</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%9D%BC%ED%8B%B4%EC%96%B4&amp;action=edit&amp;section=1" title="부분 편집: 문자">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<div class="thumb tright">

라틴어 알파벳</div>
</div>
</div>
<div class="rellink boilerplate seealso"><img alt="" data-file-height="128" data-file-width="128" height="16" src="//upload.wikimedia.org/wikipedia/commons/thumb/e/ec/Crystal_Clear_app_xmag.svg/16px-Crystal_Clear_app_xmag.svg.png" srcset="//upload.wikimedia.org/wikipedia/commons/thumb/e/ec/Crystal_Clear_app_xmag.svg/24px-Crystal_Clear_app_xmag.svg.png 1.5x, //upload.wikimedia.org/wikipedia/commons/thumb/e/ec/Crystal_Clear_app_xmag.svg/32px-Crystal_Clear_app_xmag.svg.png 2x" width="16"/> <a href="/wiki/%EB%A1%9C%EB%A7%88%EC%9E%90" title="로마자">로마자</a> 문서를 참고하십시오.</div>
<p>라틴어 금석문전은 기원전 6세기부터 출현하고 있으며, <a class="mw-redirect" href="/wiki/%EC%95%8C%ED%8C%8C%EB%B2%B3" title="알파벳">알파벳</a>의 형성은 그리스 알파벳이 에트루스키어 표기를 거쳐 라틴어 알파벳으로 성립한 것으로 추정된다. 고전 라틴어에서 사용되는 알파벳은 23자(A B C D E F G H I K L M N O P Q R S T V X Y Z)였으며, Y, Z가 도입된 것은 기원전 1세기였고 J가 도입된 것은 <a href="/wiki/%EC%A4%91%EC%84%B8" title="중세">중세</a>였다. 발음법은 시대에 따라 상당한 차이를 보인다. 라틴 알파벳은 오늘날에도 로망스어, 켈트어, 게르만어와 몇몇 슬라브(폴란드어 등) 등 여러 유럽 언어에 쓰이고 있다.</p>
<p>고대 로마에는 <a class="mw-redirect" href="/wiki/%EA%B5%AC%EB%91%90%EC%A0%90" title="구두점">구두점</a>이나 장음 표시(macron)를 쓰지 않았다. (다만 장모음과 단모음을 구분하기 위해 양음 악센트 부호 '는 썼다.) 알파벳 <i>j,u</i>도 없었고, 필기체는 있었지만 소문자는 없었다. 또 띄어쓰기도 하지 않았는데, 헷갈리는 낱말 사이에 점을 찍는 경우는 있었다. 따라서 라틴어 문장은 이렇게 쓰게 된다.</p>

<p>이 문장을 현대식으로 쓰자면</p>
<dl>
<dd>"Lugete, O Veneres Cupidinesque"</dd>

<p>장음 표시를 붙이면</p>
<dl>
<dd>"<span lang="la" xml:lang="la">Lūgēte, Ō Venerēs Cupīdinēsque.</span>"</dd>
</dl>
<p>가 된다.</p>
<p>로마의 필기체 글은 주로 밀랍판에서 발견되는데, 이것은 대개 <a href="/wiki/%EC%98%81%EA%B5%AD" title="영국">영국</a> <a class="new" href="/w/index.php?title=%ED%95%98%EB%93%9C%EB%A6%AC%EC%95%84%EB%88%84%EC%8A%A4_%EC%84%B1%EB%B2%BD&amp;action=edit&amp;redlink=1" title="하드리아누스 성벽 (없는 문서)">하드리아누스 성벽</a>의 <a class="new" href="/w/index.php?title=%EB%B9%88%EB%8F%8C%EB%9E%80%EB%8B%A4&amp;action=edit&amp;redlink=1" title="빈돌란다 (없는 문서)">빈돌란다</a> 요새에서 널리 발견되었다. 흥미롭게도 빈돌란다의 밀랍판에서는 띄어쓰기가 나오는데, 동시대의 기념 비문에는 띄어쓰기가 없다.</p>
<h2><span class="mw-headline" id=".EC.97.AD.EC.82.AC">역사</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%9D%BC%ED%8B%B4%EC%96%B4&amp;action=edit&amp;section=2" title="부분 편집: 역사">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<div class="rellink relarticle mainarticle"><img alt="" data-file-height="128" data-file-width="128" height="16" src="//upload.wikimedia.org/wikipedia/commons/thumb/e/ec/Crystal_Clear_app_xmag.svg/16px-Crystal_Clear_app_xmag.svg.png" srcset="//upload.wikimedia.org/wikipedia/commons/thumb/e/ec/Crystal_Clear_app_xmag.svg/24px-Crystal_Clear_app_xmag.svg.png 1.5x, //upload.wikimedia.org/wikipedia/commons/thumb/e/ec/Crystal_Clear_app_xmag.svg/32px-Crystal_Clear_app_xmag.svg.png 2x" width="16"/> 이 부분의 본문은 <a href="/wiki/%EB%9D%BC%ED%8B%B4%EC%96%B4%EC%9D%98_%EC%97%AD%EC%82%AC" title="라틴어의 역사">라틴어의 역사</a>입니다.</div>
<h3><span class="mw-headline" id=".EB.B0.9C.EC.83.9D.EA.B3.BC_.EA.B3.84.ED.86.B5">발생과 계통</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%9D%BC%ED%8B%B4%EC%96%B4&amp;action=edit&amp;section=3" title="부분 편집: 발생과 계통">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>역사적으로 라틴 부족이 언제 어디서 <a href="/wiki/%EC%9D%B4%ED%83%88%EB%A6%AC%EC%95%84" title="이탈리아">이탈리아</a> 반도로 들어와 살게 되었는지는 오늘날 정확히 말할 수 없지만 지금까지 <a href="/wiki/%EC%97%AD%EC%82%AC" title="역사">역사</a>, <a href="/wiki/%EA%B3%A0%EA%B3%A0%ED%95%99" title="고고학">고고학</a> 및 문헌학자들의 연구 결과에 따르면 기원전 약 2000년을 전후로 <a href="/wiki/%EC%9D%B4%ED%83%88%EB%A6%AC%EC%95%84" title="이탈리아">이탈리아</a> 반도의 북동쪽, 오늘날 <a href="/wiki/%EC%95%84%EB%93%9C%EB%A6%AC%EC%95%84_%ED%95%B4" title="아드리아 해">아드리아 해안</a>의 <a href="/wiki/%EB%B0%9C%EC%B9%B8_%EB%B0%98%EB%8F%84" title="발칸 반도">발칸 반도</a> 북부지역에 퍼져 살던 <a href="/wiki/%EC%82%AC%EB%9E%8C" title="사람">사람</a>들이 라틴 부족의 선조였을 것이라고 추측하고 있다.</p>
<p>라틴어는 문법학적으로 볼 때 언어의 계통은 <a href="/wiki/%EC%9D%B8%EB%8F%84%EC%9C%A0%EB%9F%BD%EC%96%B4%EC%A1%B1" title="인도유럽어족">인도유럽어족</a>에 속한다. <a href="/wiki/%EC%9D%B8%EB%8F%84%EC%9C%A0%EB%9F%BD%EC%96%B4%EC%A1%B1" title="인도유럽어족">인도유럽어족</a>에서 라틴어는 다시 <a href="/wiki/%EC%9D%B4%ED%83%88%EB%A6%AC%EC%95%84%EC%96%B4" title="이탈리아어">이탈리아어</a>, <a href="/wiki/%ED%94%84%EB%9E%91%EC%8A%A4%EC%96%B4" title="프랑스어">프랑스어</a>, <a href="/wiki/%EC%8A%A4%ED%8E%98%EC%9D%B8%EC%96%B4" title="스페인어">스페인어</a>, <a href="/wiki/%ED%8F%AC%EB%A5%B4%ED%88%AC%EA%B0%88%EC%96%B4" title="포르투갈어">포르투갈어</a>, <a href="/wiki/%EB%A3%A8%EB%A7%88%EB%8B%88%EC%95%84%EC%96%B4" title="루마니아어">루마니아어</a>를 지칭하는 <a class="mw-redirect" href="/wiki/%EB%A1%9C%EB%A7%9D%EC%8A%A4%EC%96%B4" title="로망스어">로망스어</a>족의 생성과 발달에 결정적인 역할을 한 모체어이다.</p>
<p>라틴어가 한 부족의 언어로 틀을 잡기 시작한 시기는 기원전 약 6-5세기로 여겨지며, 구어에서 문어로 발전된 것은 오늘날 남아 있는 비문을 근거로 할 때 기원전 약 3세기를 전후로 하여 이루어진 것으로 추측된다. 라틴어가 고대 로마의 건립과 함께 로마와 그 주변 지역에서 뿌리를 내리기 전에 낱말이나 문법에서 서로 많은 차이가 있는 두 종류 이상의 언어가 라틴어와 공존하여 쓰인 것으로 보인다. <a href="/wiki/%EC%82%BC%EB%8B%88%EC%9B%80" title="삼니움">삼니움</a> 지역의 <a class="new" href="/w/index.php?title=%EC%98%A4%EC%8A%A4%ED%82%A4%EC%96%B4&amp;action=edit&amp;redlink=1" title="오스키어 (없는 문서)">오스키어</a>와 <a class="mw-redirect" href="/wiki/%EC%9B%80%EB%B8%8C%EB%A6%AC%EC%95%84" title="움브리아">움브리아</a> 지역의 <a class="new" href="/w/index.php?title=%EC%9B%80%EB%B8%8C%EB%A6%AC%EC%96%B4&amp;action=edit&amp;redlink=1" title="움브리어 (없는 문서)">움브리어</a>는 라틴어와 마찬가지로 지역 언어로서 사용되었으며 기원전 3세기 로마인의 세력 확장에 따라 점차 사라지거나 라틴어에 동화된 것으로 보이며 기원전 1세기에 이르러서 완전히 자취를 감춘 것으로 여겨진다.</p>
<h3><span class="mw-headline" id=".EB.9D.BC.ED.8B.B4.EC.96.B4.EC.9D.98_.ED.8C.8C.EA.B8.89.EA.B3.BC_.EC.A4.91.EC.84.B8_.EB.9D.BC.ED.8B.B4.EC.96.B4">라틴어의 파급과 중세 라틴어</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%9D%BC%ED%8B%B4%EC%96%B4&amp;action=edit&amp;section=4" title="부분 편집: 라틴어의 파급과 중세 라틴어">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>라틴어는 고대와 중세를 통하여 유럽의 대표 언어로 자리잡았다. 유럽의 학술, 외교, 종교에서 기본 언어는 라틴어였다. 따라서 고대-중세 유럽에서 <a href="/wiki/%EC%A7%80%EC%8B%9D%EC%9D%B8" title="지식인">지식인</a>이란 바로 라틴어를 할 줄 아는 사람을 뜻했다. 특히 교회사에서 라틴어는 <a href="/wiki/%EB%A1%9C%EB%A7%88_%EA%B0%80%ED%86%A8%EB%A6%AD%EA%B5%90%ED%9A%8C" title="로마 가톨릭교회">로마 가톨릭교회</a>의 언어였는데, 로마 가톨릭교회는 <a class="mw-redirect" href="/wiki/%EC%84%B1%EC%84%9C" title="성서">성서</a>의 남·오용, 혹은 잘못된 번역 때문에 생기는 <a href="/wiki/%EA%B8%B0%EB%8F%85%EA%B5%90" title="기독교">기독교</a> 신앙의 훼손을 막는다는 명분으로 라틴어 성서를 다른 언어로 번역하는 것을 금지하였다. 하지만 라틴어 성서 번역 금지는 <a href="/wiki/%EB%A1%9C%EB%A7%88_%EA%B0%80%ED%86%A8%EB%A6%AD%EA%B5%90%ED%9A%8C" title="로마 가톨릭교회">로마 가톨릭교회</a> <a href="/wiki/%EC%82%AC%EC%A0%9C" title="사제">사제</a>들이 라틴어를 모르는 평신도들을 지배하는 수단이 되었기 때문에, <a class="mw-redirect" href="/wiki/%EC%A2%85%EA%B5%90%EA%B0%9C%ED%98%81" title="종교개혁">종교개혁</a>기에 각 지역에서 자국어 성서로 번역되었다.</p>
<p>학술 언어로도 정착되어 <a href="/wiki/19%EC%84%B8%EA%B8%B0" title="19세기">19세기</a> 말까지도 모든 학술과 학문에서 강의와 기록은 라틴어로 이루어졌다. 그 예로 <a href="/wiki/%EC%9E%A5_%EC%B9%BC%EB%B1%85" title="장 칼뱅">장 칼뱅</a>은 신학 문서인 《기독교 강요》를 <a href="/wiki/%ED%94%84%EB%9E%91%EC%8A%A4%EC%96%B4" title="프랑스어">프랑스어</a>로 저술한 뒤 라틴어로 다시 번역했다. 물론 <a href="/wiki/%EA%B7%B8%EB%A6%AC%EC%8A%A4%EC%96%B4" title="그리스어">그리스어</a>도 사용되었으나 고대 언어를 연구할 때나 그리스어로 쓰인 문헌을 번역하는 데만 쓰일 뿐이었다. 이러한 유습은 오래 남아, <a href="/wiki/1962%EB%85%84" title="1962년">1962년</a> <a href="/wiki/%EC%A0%9C2%EC%B0%A8_%EB%B0%94%ED%8B%B0%EC%B9%B8_%EA%B3%B5%EC%9D%98%ED%9A%8C" title="제2차 바티칸 공의회">제2차 바티칸 공의회</a> 이전까지 로마 가톨릭교회는 <a href="/wiki/%ED%8A%B8%EB%A6%AC%EC%97%94%ED%8A%B8_%EA%B3%B5%EC%9D%98%ED%9A%8C" title="트리엔트 공의회">트리엔트 공의회</a>(1545년)의 결정에 따라, 루브리카 양식으로 불리는 라틴어 <a href="/wiki/%EC%A0%84%EB%A1%80" title="전례">전례</a>로 <a href="/wiki/%EB%AF%B8%EC%82%AC" title="미사">미사</a>를 집전하였으며, <a href="/wiki/%EC%84%B1%EA%B3%B5%ED%9A%8C" title="성공회">성공회</a>에서도 '<a class="mw-redirect" href="/wiki/%ED%95%98%EB%8A%90%EB%8B%98%EC%9D%98_%EC%96%B4%EB%A6%B0%EC%96%91" title="하느님의 어린양">하느님의 어린양</a>(Agnus Dei)', '<a class="mw-redirect" href="/wiki/%EA%B1%B0%EB%A3%A9%ED%95%98%EC%8B%9C%EB%8B%A4" title="거룩하시다">거룩하시다</a>(Sanctus)' 등의 라틴어 <a href="/wiki/%EC%A0%84%EB%A1%80" title="전례">전례</a>용어들을 사용한다. 현재까지도 <a href="/wiki/%EC%83%9D%EB%AC%BC%ED%95%99" title="생물학">생물학</a>에서 생물종을 분류할 때 <a href="/wiki/%EC%B9%BC_%ED%8F%B0_%EB%A6%B0%EB%84%A4" title="칼 폰 린네">린네</a>의 분류법에 따라 라틴어로 표기하고 있으며, 그 밖에 학술 명칭들 중에도 라틴어로 표기하는 것들이 많다.</p>

<h2><span class="mw-headline" id=".EB.9D.BC.ED.8B.B4.EC.96.B4.EC.9D.98_.EC.98.81.ED.96.A5">라틴어의 영향</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%9D%BC%ED%8B%B4%EC%96%B4&amp;action=edit&amp;section=5" title="부분 편집: 라틴어의 영향">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>로마 제국 이후 <a class="mw-redirect" href="/wiki/%EB%AF%BC%EC%A4%91_%EB%9D%BC%ED%8B%B4%EC%96%B4" title="민중 라틴어">민중 라틴어</a>는 9세기경 여러 방언으로 갈라져 <a class="mw-redirect" href="/wiki/%EB%A1%9C%EB%A7%9D%EC%8A%A4%EC%96%B4" title="로망스어">로망스어</a>를 이루게 된다. 수백년간 이들 언어는 오직 구어로만 쓰였지만, 라틴어만은 여전히 문서에도 쓰였다.</p>
<p>예를 들어 1296년에 대체될 때까지 <a href="/wiki/%ED%8F%AC%EB%A5%B4%ED%88%AC%EA%B0%88" title="포르투갈">포르투갈</a>에서는 라틴어가 공식 언어였다. 라틴어에서 파생된 이탈리아어, 프랑스어, 스페인어, 포르투갈어, <a href="/wiki/%EC%B9%B4%ED%83%88%EB%A3%A8%EB%83%90%EC%96%B4" title="카탈루냐어">카탈루냐어</a>, <a href="/wiki/%EC%98%A4%ED%81%AC%EC%96%B4" title="오크어">오크어</a>, <a href="/wiki/%EB%A1%9C%EB%A7%8C%EC%8A%88%EC%96%B4" title="로만슈어">로만슈어</a>, 루마니아어는 시간이 지날수록 더욱 번성하여 더 확대되고 공식화된다.</p>
<p>로망스어 가운데 이탈리아어는 어휘 면에서 가장 보수적인 변화를 보였다. 또 <a href="/wiki/%EC%82%AC%EB%A5%B4%EB%8D%B0%EB%83%90%EC%96%B4" title="사르데냐어">사르데냐어</a>는 음운상 가장 보수적이었다.</p>
<p>고전 라틴어와 로망스어의 몇몇 차이점은 민중 라틴어를 재현하는 데 쓰이기도 했다. 가령 로망스어에는 특정 모음에 고유한 강세가 붙는데, 라틴어는 이 뿐만 아니라 모음의 길이에도 장단의 차이를 두었다. 이탈리아어와 <a class="new" href="/w/index.php?title=%EC%82%AC%EB%A5%B4%EB%8F%84_%EB%A1%9C%EA%B5%AC%EB%8F%84%EB%A0%88%EC%84%B8&amp;action=edit&amp;redlink=1" title="사르도 로구도레세 (없는 문서)">사르도 로구도레세</a>(사르데냐어 방언)에서는 강세와 더불어 자음에서도 장단 차이가 있었다. 프랑스어 (언중 대부분의 경우) 장단과 강세에 차이를 따로 두지 않는다. 로망스어와 라틴어의 가장 주요한 차이 중 하나는 루마니아어를 제외한 거의 대부분의 로망스어에서 명사의 <a href="/wiki/%EA%B2%A9" title="격">격</a>이 없어졌다는 것이다.</p>
<p>영어에서도 라틴어의 영향을 찾아 볼 수 있다. 중세 시대에 <a class="new" href="/w/index.php?title=%EC%BA%94%ED%84%B0%EB%B2%A0%EB%A6%AC%EC%9D%98_%EC%84%B1_%EC%95%84%EC%9A%B0%EA%B5%AC%EC%8A%A4%ED%8B%B0%EB%88%84%EC%8A%A4&amp;action=edit&amp;redlink=1" title="캔터베리의 성 아우구스티누스 (없는 문서)">캔터베리의 성 아우구스티누스</a>가 쓴 교회 언어 사용이나 혹은 <a href="/wiki/%EB%85%B8%EB%A5%B4%EB%A7%8C_%EC%A0%95%EB%B3%B5" title="노르만 정복">노르만 정복</a> 이후 <a class="new" href="/w/index.php?title=%EC%95%B5%EA%B8%80%EB%A1%9C-%EB%85%B8%EB%A5%B4%EB%A7%8C_%EC%96%B8%EC%96%B4&amp;action=edit&amp;redlink=1" title="앵글로-노르만 언어 (없는 문서)">앵글로-노르만 언어</a>를 통해 수많은 라틴어 차용어를 받아들였다.</p>
<p>16세기에서 18세기까지 영어로 글을 쓰는 사람들은 라틴어나 그리스어로 새로운 낱말을 많이 만들었다. 이런 낱말을 <a class="new" href="/w/index.php?title=%EC%9E%89%ED%81%AC%ED%86%B5_%EC%9A%A9%EC%96%B4&amp;action=edit&amp;redlink=1" title="잉크통 용어 (없는 문서)">잉크통 용어</a>(Inkhorn terms)라고 한다. 이런 낱말 상당수는 낱말을 지어낸 본인만 쓰다 잊혀졌지만, 어떤 낱말은 아직까지 살아남아 쓰인다. <i>imbibe</i>나 <i>extrapolate</i>는 라틴어에서 만들어낸 잉크통 언어이다. 영어에서 다음절(polysyllabic) 낱말 중 많은 것들이 라틴어에서 바로 차용한 것들이며, 상당수가 <a class="new" href="/w/index.php?title=%EA%B3%A0%EB%8C%80_%ED%94%84%EB%9E%91%EC%8A%A4%EC%96%B4&amp;action=edit&amp;redlink=1" title="고대 프랑스어 (없는 문서)">고대 프랑스어</a>에서 차용한 것들을 다시 받아들인 것이다.</p>
<p>많은 단체에서 <a class="mw-redirect" href="/wiki/%EB%AA%A8%ED%86%A0" title="모토">모토</a>를 라틴어로 쓴다.</p>
<h2><span class="mw-headline" id=".EB.AC.B8.EB.B2.95">문법</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%9D%BC%ED%8B%B4%EC%96%B4&amp;action=edit&amp;section=6" title="부분 편집: 문법">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<div class="rellink relarticle mainarticle"><img alt="" data-file-height="128" data-file-width="128" height="16" src="//upload.wikimedia.org/wikipedia/commons/thumb/e/ec/Crystal_Clear_app_xmag.svg/16px-Crystal_Clear_app_xmag.svg.png" srcset="//upload.wikimedia.org/wikipedia/commons/thumb/e/ec/Crystal_Clear_app_xmag.svg/24px-Crystal_Clear_app_xmag.svg.png 1.5x, //upload.wikimedia.org/wikipedia/commons/thumb/e/ec/Crystal_Clear_app_xmag.svg/32px-Crystal_Clear_app_xmag.svg.png 2x" width="16"/> 이 부분의 본문은 <a href="/wiki/%EB%9D%BC%ED%8B%B4%EC%96%B4_%EB%AC%B8%EB%B2%95" title="라틴어 문법">라틴어 문법</a>입니다.</div>
<p>유형론상으로 라틴어는 인도-유럽 어족에서 주로 나타나는 <a href="/wiki/%EA%B5%B4%EC%A0%88%EC%96%B4" title="굴절어">굴절어</a> 유형에 속한다. 이와 유사한 형태를 가진 <a class="new" href="/w/index.php?title=%EA%B3%A0%EC%A0%84%EC%96%B4&amp;action=edit&amp;redlink=1" title="고전어 (없는 문서)">고전어</a>로는 <a href="/wiki/%EA%B3%A0%EB%8C%80_%EA%B7%B8%EB%A6%AC%EC%8A%A4%EC%96%B4" title="고대 그리스어">고대 그리스어</a>, <a href="/wiki/%EA%B3%A0%EB%8C%80_%ED%8E%98%EB%A5%B4%EC%8B%9C%EC%95%84%EC%96%B4" title="고대 페르시아어">고대 페르시아어</a>, <a href="/wiki/%EC%82%B0%EC%8A%A4%ED%81%AC%EB%A6%AC%ED%8A%B8%EC%96%B4" title="산스크리트어">산스크리트어</a> 등이 있는데, 특히 작가들이 라틴어의 문법을 정비하면서 고대 그리스어(아틱 방언)의 합리적인 문법요소를 대폭 도입하여 문법적으로 이 두 언어는 유사한 점이 많게 되었다. 대표적으로 고대 그리스어의 <a class="new" href="/w/index.php?title=%EC%A0%88%EB%8C%80_%EC%86%8D%EA%B2%A9&amp;action=edit&amp;redlink=1" title="절대 속격 (없는 문서)">절대 속격</a>과 이의 대응으로서 라틴어의 <a class="new" href="/w/index.php?title=%EC%A0%88%EB%8C%80_%ED%83%88%EA%B2%A9&amp;action=edit&amp;redlink=1" title="절대 탈격 (없는 문서)">절대 탈격</a>의 예가 있다.</p>
<p>라틴어의 명사는 단수, 복수의 2수와 남성, 여성, 중성의 3성이 있으며 슬라브계 언어와 같은 활동체/불활동체의 구별은 없다. 또한 주격, 속격, 여격, 대격, 탈격, 호격으로 6격 변화를 하며 특징적인 어미의 변화에 따라 서로 다른 <a href="/wiki/%EA%B3%A1%EC%9A%A9" title="곡용">곡용</a>의 형태가 존재한다. 라틴어의 형용사는 명사의 성과 격에 따라 명사와 유사하게 규칙적인 곡용을 따라 굴절시키는 <a class="new" href="/w/index.php?title=%EB%AC%B8%EB%B2%95%EC%A0%81_%EC%9D%BC%EC%B9%98&amp;action=edit&amp;redlink=1" title="문법적 일치 (없는 문서)">문법적 일치</a>의 체계를 갖고 있다.</p>
<p>라틴어의 동사는 1, 2, 3인칭과 단수, 복수의 형식에 따라 6가지, 과거, 현재, 미래의 3개 시점과 각각의 완료, 미완료의 구분에 따라 6가지, 또한 능동태와 수동태의 2개 태가 있어서 이에 따라 2가지로 굴절한다. 또한 직설법(기본형), <a class="new" href="/w/index.php?title=%EC%A0%91%EC%86%8D%EB%B2%95&amp;action=edit&amp;redlink=1" title="접속법 (없는 문서)">접속법</a>, 명령법의 3개 서법이 있으며 모두 굴절의 형식이고 이에 따른 <a class="new" href="/w/index.php?title=%EC%84%9C%EB%B2%95_%EC%A1%B0%EB%8F%99%EC%82%AC&amp;action=edit&amp;redlink=1" title="서법 조동사 (없는 문서)">서법 조동사</a>는 없다.<sup class="reference" id="cite_ref-2"><a href="#cite_note-2">[2]</a></sup> 명사의 경우와 유사하게 특징적인 어미의 변화에 대한 <a href="/wiki/%ED%99%9C%EC%9A%A9" title="활용">활용</a>의 형태가 존재한다.</p>
<p>라틴어에서 동사로부터 유도될 수 있는 다른 문법요소로는 <a class="new" href="/w/index.php?title=%EC%9B%90%ED%98%95%EB%8F%99%EC%82%AC&amp;action=edit&amp;redlink=1" title="원형동사 (없는 문서)">원형동사</a>, <a class="new" href="/w/index.php?title=%EB%B6%84%EC%82%AC&amp;action=edit&amp;redlink=1" title="분사 (없는 문서)">분사</a>, <a href="/wiki/%EB%8F%99%EB%AA%85%EC%82%AC" title="동명사">동명사</a>, <a class="new" href="/w/index.php?title=%EB%8F%99%ED%98%95%EC%82%AC&amp;action=edit&amp;redlink=1" title="동형사 (없는 문서)">동형사</a>, <a class="new" href="/w/index.php?title=%EB%AA%A9%EC%A0%81%EB%B6%84%EC%82%AC&amp;action=edit&amp;redlink=1" title="목적분사 (없는 문서)">목적분사</a> 등이 있다. 먼저 원형동사는 동사적인 명사인데, 시제와, <a class="new" href="/w/index.php?title=%EC%83%81_(%EB%AC%B8%EB%B2%95)&amp;action=edit&amp;redlink=1" title="상 (문법) (없는 문서)">상</a>, 태에 따라 몇 가지 이형태(異形態)가 존재한다.<sup class="reference" id="cite_ref-3"><a href="#cite_note-3">[3]</a></sup> 분사는 동사적인 형용사로서, 역시 시제, 상, 태에 따라 몇 가지 형태가 존재한다. 동명사는 원형동사로부터 유도되는 동사의 명사형으로, <a href="/wiki/%EA%B3%A1%EC%9A%A9" title="곡용">곡용</a>된 것이어서 주격이 없다. 동형사<sup class="reference" id="cite_ref-4"><a href="#cite_note-4">[4]</a></sup> 는 당위(~해야만 하는)나 필연의 뜻이 포함된 동사적인 형용사이다. 목적분사는 목적(~하기 위하여)의 뜻이 포함된 동사적인 명사이며 능동형과 수동형이 있다.</p>
<p>라틴어의 문장은 많은 단어를 사용하지 않고 되도록이면 간결하게 표현하는 것이 고급스럽다. 그러나 많은 경우 명사, 형용사, 동사로만 표현하기 곤란한 경우가 생기는데 이때 의미를 명확히 하기 위해 전치사나 부사, 접속사, <a class="new" href="/w/index.php?title=%EA%B0%84%ED%88%AC%EC%82%AC&amp;action=edit&amp;redlink=1" title="간투사 (없는 문서)">간투사</a> 등을 사용한다. 이 네 가지 문법 범주는 기본적으로 <a class="new" href="/w/index.php?title=%EB%B6%88%EB%B3%80%ED%99%94%EC%82%AC&amp;action=edit&amp;redlink=1" title="불변화사 (없는 문서)">불변화사</a>이다. 전치사는 지배받는 명사가 전치사의 성질에 따라 정해진 격을 취하게 하는 <a class="new" href="/w/index.php?title=%EA%B2%A9%EC%A7%80%EB%B0%B0&amp;action=edit&amp;redlink=1" title="격지배 (없는 문서)">격지배</a>라는 성질이 있는데, 이것은 <a href="/wiki/%EA%B3%A1%EC%9A%A9" title="곡용">곡용</a>이 없는 언어(영어, 프랑스어 등)에서는 나타나지 않는 특징이다. 부사는 본래적인 것 외에 형용사에서 규칙적으로 생성하는 방법이 있다.</p>
<p>라틴어의 <a href="/wiki/%EC%96%B4%EC%88%9C" title="어순">어순</a>은 기본적으로 SOV(주어-목적어-동사)인 경우가 많다. 그러나 주어가 인칭대명사일 경우 동사에 인칭과 수가 표지되므로 탈락시키는 것이 일반적이며<sup class="reference" id="cite_ref-5"><a href="#cite_note-5">[5]</a></sup>, 주어와 목적어는 각각 주격과 대격 표지를 받으므로 이 둘이 혼동되지 않는다면 위치를 어디로든 이동시킬 수 있어서 사실상 어순은 매우 자유롭다.<sup class="reference" id="cite_ref-6"><a href="#cite_note-6">[6]</a></sup></p>

<p>서구 사회에서 중등학교 혹은 대학교의 라틴어 수업은 일차적으로 라틴어 텍스트를 현대어로 번역하는 데에 초점을 맞춘다. 라틴어 회화는 거의 다루지 않는데, 현대의 여러 개념(가령 기술과 관련된 말 등)에 부합하는 고대어 표현이 없기도 하거니와 새로운 낱말을 받아들이는 공식적인 기관이 있는 것도 아니기 때문이다. 따라서 라틴어 공부에서는 읽고 쓰는 것이 중시되며 듣고 말하는 것은 거의 무시된다.</p>
<p>그러나 <a class="new" href="/w/index.php?title=%EC%83%9D%ED%99%9C_%EB%9D%BC%ED%8B%B4%EC%96%B4&amp;action=edit&amp;redlink=1" title="생활 라틴어 (없는 문서)">생활 라틴어</a> 운동 같은 시도도 있는데, 생활 라틴어를 지지하는 사람들은 현대어를 가르치는 방식, 즉 대화나 글을 이용한 의사소통으로 라틴어를 가르칠 수도 있다고 본다. 예를 들어 핀란드에서는 1989년 9월부터 'Nuntii Latini'라는 이름의 라틴어 뉴스 방송을 내보내고 있다. 이렇게 라틴어를 활용하려는 접근법은 고대 작가들이 라틴어를 어떤 방식으로 말하고 썼을지 이해하는 데 시사점을 던져준다. 라틴 시나 문학의 양식은 그 낱말들의 성음을 이해하지 못한 상태에서는 알기 어렵기 때문이다.</p>
<p>생활 라틴어 교육은 <a class="mw-redirect" href="/wiki/%EB%B0%94%ED%8B%B0%EC%B9%B8" title="바티칸">바티칸</a>이나 혹은 켄터키 대학과 같은 미국의 몇몇 기관에서 실시하고 있다. 영국에서는 <a class="new" href="/w/index.php?title=%EA%B3%A0%EC%A0%84_%EC%97%B0%EB%A7%B9&amp;action=edit&amp;redlink=1" title="고전 연맹 (없는 문서)">고전 연맹</a>이 이 운동을 장려하고 있으며 ‘미니무스(Minimus)’라는 쥐 캐릭터의 모험을 묘사한 라틴어 책도 출판되었다. 미국에서는 <a class="new" href="/w/index.php?title=%EA%B5%AD%EB%A6%BD_%EC%B2%AD%EC%86%8C%EB%85%84_%EA%B3%A0%EC%A0%84_%EC%97%B0%EB%A7%B9&amp;action=edit&amp;redlink=1" title="국립 청소년 고전 연맹 (없는 문서)">국립 청소년 고전 연맹</a>(National Junior Classical League)<sup class="reference" id="cite_ref-7"><a href="#cite_note-7">[7]</a></sup> 에서 고등학교 학생들에게 라틴어 공부를 권장하고 있으며, <a class="new" href="/w/index.php?title=%EA%B5%AD%EB%A6%BD_%EB%8C%80%ED%95%99%EC%83%9D_%EA%B3%A0%EC%A0%84_%EC%97%B0%EB%A7%B9&amp;action=edit&amp;redlink=1" title="국립 대학생 고전 연맹 (없는 문서)">국립 대학생 고전 연맹</a>(National Senior Classical League)에서도 대학생들이 라틴어 공부를 계속 이어가도록 장려한다.</p>
<p>여러 <a class="mw-redirect" href="/wiki/%EA%B5%AD%EC%A0%9C_%EB%B3%B4%EC%A1%B0%EC%96%B4" title="국제 보조어">국제 보조어</a>가 라틴어의 영향을 크게 받았다. <a href="/wiki/%EC%9D%B8%ED%85%8C%EB%A5%B4%EB%A7%81%EA%B5%AC%EC%95%84" title="인테르링구아">인테르링구아</a>는 간소화된 현대식 라틴어로 간주되기도 한다. 20세기 초에 인기를 끌었던 <a class="new" href="/w/index.php?title=%EB%AC%B4%EA%B5%B4%EC%A0%88_%EB%9D%BC%ED%8B%B4%EC%96%B4&amp;action=edit&amp;redlink=1" title="무굴절 라틴어 (없는 문서)">무굴절 라틴어</a>(Latino sine Flexione)는 라틴어에서 <a href="/wiki/%ED%99%9C%EC%9A%A9" title="활용">활용</a>(inflection)을 배제한 언어이다.</p>
<p>현대 문학을 라틴어로 번역하는 경우도 있는데, <a class="mw-redirect" href="/wiki/%EC%9C%84%EB%8B%88_%EB%8D%94_%ED%91%B8" title="위니 더 푸">위니 더 푸</a>, <a href="/wiki/%EC%95%84%EC%8A%A4%ED%85%8C%EB%A6%AD%EC%8A%A4" title="아스테릭스">아스테릭스</a>, <a href="/wiki/%ED%95%B4%EB%A6%AC_%ED%8F%AC%ED%84%B0" title="해리 포터">해리 포터</a>, <a href="/wiki/%EC%96%B4%EB%A6%B0_%EC%99%95%EC%9E%90" title="어린 왕자">어린 왕자</a> 등 라틴어 공부에 흥미를 높이기 위한 것이다.</p>
<h2><span class="mw-headline" id=".ED.98.84.EB.8C.80.EC.9D.98_.EB.9D.BC.ED.8B.B4.EC.96.B4_.EC.93.B0.EC.9E.84">현대의 라틴어 쓰임</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%9D%BC%ED%8B%B4%EC%96%B4&amp;action=edit&amp;section=8" title="부분 편집: 현대의 라틴어 쓰임">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<div class="thumb tright">

영국 월샌드 전철역의 표지판. 영어와 더불어 라틴어로 "금연(Noli Fumare)"라고 써있다. 로마의 유산인 <a class="new" href="/w/index.php?title=%ED%95%98%EB%93%9C%EB%A6%AC%EC%95%84%EB%88%84%EC%8A%A4_%EC%84%B1%EB%B2%BD&amp;action=edit&amp;redlink=1" title="하드리아누스 성벽 (없는 문서)">하드리아누스 성벽</a>과 가까운 역이라서 이를 기념하는 의미에서 표지판에 두 언어로 표기한 것이다.</div>

<p>오늘날 라틴어로된 전문 용어는 널리 쓰이는데, <a href="/wiki/%EC%B2%A0%ED%95%99" title="철학">철학</a>, <a href="/wiki/%EC%9D%98%ED%95%99" title="의학">의학</a>, <a href="/wiki/%EC%83%9D%EB%AC%BC%ED%95%99" title="생물학">생물학</a>, <a href="/wiki/%EB%B2%95" title="법">법</a> 분야에서도 마찬가지며, 원어나 약어로 쓰는 경우도 많다.</p>
<ul>
<li><i>subpoena duces tecum</i><font color="gray"><small>→"지정한 증거물을 가지고 출두 증언하라는 법원의 명령"</small></font></li>
<li><i>q.i.d</i>(quater in die)<font color="gray"><small>→"하루에 네 번씩"</small></font></li>
<li><i>inter alia</i><font color="gray"><small>→"다른 것들 중에서"</small></font></li>
</ul>
<p>기술 용어처럼 라틴어 용어가 따로 쓰이기도 한다.</p>
<p>오늘날 공식상에서 여전히 라틴어를 쓰는 최대 조직은 <a href="/wiki/%EB%A1%9C%EB%A7%88_%EA%B0%80%ED%86%A8%EB%A6%AD%EA%B5%90%ED%9A%8C" title="로마 가톨릭교회">로마 가톨릭교회</a>이다. (특히 라틴어 <a href="/wiki/%EC%A0%84%EB%A1%80" title="전례">전례</a>에서) <a href="/wiki/%EA%B5%90%ED%99%A9_%EB%B0%94%EC%98%A4%EB%A1%9C_6%EC%84%B8" title="교황 바오로 6세">교황 바오로 6세</a> 이래 <a href="/wiki/%EB%AF%B8%EC%82%AC" title="미사">미사</a>는 보통 각 지역의 제 나라 언어로 집전하지만, 라틴어로 하기도 하며 특히 바티칸에서 그렇다. 라틴어는 <a href="/wiki/%EA%B5%90%ED%99%A9" title="교황">교황</a>의 공식 언어이기도 하다.그러나 현재는 바티칸 내에서도 능숙한 사용자가 줄고 있는 추세이다. <a href="/wiki/%EC%84%B1%EA%B3%B5%ED%9A%8C" title="성공회">성공회</a>에서도 라틴어 전례용어들이 쓰이는데, <a class="mw-redirect" href="/wiki/%ED%95%98%EB%8A%90%EB%8B%98%EC%9D%98_%EC%96%B4%EB%A6%B0%EC%96%91" title="하느님의 어린양">하느님의 어린양</a>(Agnus Dei) 등이 그것이다.물론 라틴어 그대로 사용하는 것이 아니라 <a href="/wiki/%EC%86%8D%EC%A7%80%EC%A3%BC%EC%9D%98" title="속지주의">속지주의</a>에 따라 성공회 교회가 속한 지역의 <a href="/wiki/%EC%96%B8%EC%96%B4" title="언어">언어</a>를 사용한다.한국에서도 일부 가톨릭교회들에서 라틴어로 미사를 집전하는 '전통 라틴 미사 성제' 운동을 벌이고 있다.</p>
<p>언어적 중립성이 필요한 분야에서, 특히 생물체의 <a href="/wiki/%ED%95%99%EB%AA%85" title="학명">학명</a>처럼 라틴어가 많이 쓰인다.</p>
<p>《<a class="new" href="/w/index.php?title=%EC%84%B8%EB%B0%94%EC%8A%A4%ED%8B%B0%EC%95%84%EB%84%A4&amp;action=edit&amp;redlink=1" title="세바스티아네 (없는 문서)">세바스티아네</a>》(1976)이나 《<a href="/wiki/%ED%8C%A8%EC%85%98_%EC%98%A4%EB%B8%8C_%ED%81%AC%EB%9D%BC%EC%9D%B4%EC%8A%A4%ED%8A%B8" title="패션 오브 크라이스트">패션 오브 크라이스트</a>》(2004)처럼 고대사를 배경으로 하는 영화에서 현실성을 살리고자 라틴어로 대사를 만들기도 했다.</p>
<p>오늘날 많은 단체에서 라틴어 모토를 쓰는데, <a href="/wiki/%EB%AF%B8%EA%B5%AD_%ED%95%B4%EC%95%88%EA%B2%BD%EB%B9%84%EB%8C%80" title="미국 해안경비대">미국 해안경비대</a>에서는 "Semper Paratus"<font color="gray"><small>→항상 준비</small></font>를, <a href="/wiki/%EB%AF%B8%EA%B5%AD_%ED%95%B4%EB%B3%91%EB%8C%80" title="미국 해병대">미국 해병대</a>에서는 "Semper fidelis"<font color="gray"><small>→항상 충실하게</small></font>가 모토이다. 미국의 몇몇 주에서도 라틴어 모토가 있는데, <a class="mw-redirect" href="/wiki/%EC%9B%A8%EC%8A%A4%ED%8A%B8_%EB%B2%84%EC%A7%80%EB%8B%88%EC%95%84" title="웨스트 버지니아">웨스트 버지니아</a> 주의 모토는 "Montani Semper Liberi"<font color="gray"><small>→산 사람은 언제나 자유롭다</small></font>이며, <a class="mw-redirect" href="/wiki/%EB%85%B8%EC%8A%A4_%EC%BA%90%EB%A1%A4%EB%9D%BC%EC%9D%B4%EB%82%98" title="노스 캐롤라이나">노스 캐롤라이나</a>에서는 "Esse Quam Videri"<font color="gray"><small>→겉으로 보이는 것 이상으로</small></font>가 모토이다. <a href="/wiki/%EB%8C%80%ED%95%9C%EB%AF%BC%EA%B5%AD" title="대한민국">대한민국</a>의 <a href="/wiki/%EC%84%9C%EC%9A%B8%EB%8C%80%ED%95%99%EA%B5%90" title="서울대학교">서울대학교</a>의 모토는 "Veritas Lux Mea"<font color="gray"><small>→진리는 나의 빛</small></font>이며, <a href="/wiki/%EC%84%9C%EA%B0%95%EB%8C%80%ED%95%99%EA%B5%90" title="서강대학교">서강대학교</a>의 모토는 "Obedire Veritati<font color="gray"><small>→진리에 순종하라</small></font>"이며, <a href="/wiki/%EA%B4%91%EC%9A%B4%EB%8C%80%ED%95%99%EA%B5%90" title="광운대학교">광운대학교</a>의 모토는 "Veritas et Lux"<font color="gray"><small>→참·빛</small></font>이다.</p>

<p>드라마/영화 시리즈 <a href="/wiki/%EC%8A%A4%ED%83%80_%ED%8A%B8%EB%A0%89" title="스타 트렉">스타 트렉</a>에 나오는, 샌프란시스코 소재의 스타플릿 아카데미(Starfleet Academy)의 모토는 "Ex astris, Scientia"(영어로는 From the star, knowledge.ㅡ별로부터의 지식ㅡ)이다.</p>
