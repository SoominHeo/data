<h1 class="firstHeading" id="firstHeading" lang="ko">라틴어의 역사</h1>
<div class="mw-body-content" id="bodyContent">
<div id="siteSub">위키백과, 우리 모두의 백과사전.</div>

<p><b>라틴어</b>는 <a href="/wiki/%EC%9D%B4%ED%83%88%EB%A6%AC%EC%95%84%EC%96%B4%ED%8C%8C" title="이탈리아어파">이탈리아어파</a>의 한 갈래로, 문자는 <a href="/wiki/%EC%97%90%ED%8A%B8%EB%A3%A8%EB%A6%AC%EC%95%84_%EB%AC%B8%EC%9E%90" title="에트루리아 문자">에트루리아 문자</a>에서 나온 <a class="mw-redirect" href="/wiki/%EB%9D%BC%ED%8B%B4_%EB%AC%B8%EC%9E%90" title="라틴 문자">라틴 문자</a>로, 원래 <a href="/wiki/%EA%B7%B8%EB%A6%AC%EC%8A%A4_%EB%AC%B8%EC%9E%90" title="그리스 문자">그리스 문자</a>와 <a href="/wiki/%ED%8E%98%EB%8B%88%ED%82%A4%EC%95%84_%EB%AC%B8%EC%9E%90" title="페니키아 문자">페니키아 문자</a>에서 파생된 것이다. 라틴어는 기원전 9, 8세기경 북쪽에서 <a href="/wiki/%EC%9D%B4%ED%83%88%EB%A6%AC%EC%95%84_%EB%B0%98%EB%8F%84" title="이탈리아 반도">이탈리아 반도</a>로 들어와 <a href="/wiki/%EB%9D%BC%ED%8B%B0%EC%9B%80" title="라티움">라티움</a>, 특히 <a href="/wiki/%EA%B3%A0%EB%8C%80_%EB%A1%9C%EB%A7%88" title="고대 로마">고대 로마</a>의 발상지인 <a href="/wiki/%ED%85%8C%EB%B2%A0%EB%A0%88_%EA%B0%95" title="테베레 강">테베레 강</a> 유역에 정착한 사람들이 전해온 언어이다. <a href="/wiki/%EB%9D%BC%ED%8B%B4%EC%96%B4" title="라틴어">라틴어</a>는 이탈리아 북부의 켈트어 방언과 이탈리아 중부의 <a class="mw-redirect" href="/wiki/%EC%9D%B8%EB%8F%84-%EC%9C%A0%EB%9F%BD%EC%96%B4" title="인도-유럽어">인도-유럽어</a>와 <a href="/wiki/%EC%97%90%ED%8A%B8%EB%A3%A8%EB%A6%AC%EC%95%84%EC%96%B4" title="에트루리아어">에트루리아어</a>, 이탈리아 남부의 <a href="/wiki/%EA%B7%B8%EB%A6%AC%EC%8A%A4%EC%96%B4" title="그리스어">그리스어</a>의 영향을 받았다.</p>
<p>현존하는 <a href="/wiki/%EB%9D%BC%ED%8B%B4_%EB%AC%B8%ED%95%99" title="라틴 문학">라틴 문학</a> 작품은 거의 <a href="/wiki/%EA%B3%A0%EC%A0%84_%EB%9D%BC%ED%8B%B4%EC%96%B4" title="고전 라틴어">고전 라틴어</a>로 남아있는데, 고전 라틴어는 기원전 1세기경 이래 매우 양식화되고 정제된 인위적인 <a href="/wiki/%EB%AC%B8%EC%96%B4" title="문어">문어</a>로, 로마 제국의 실제 구어는 <a class="mw-redirect" href="/wiki/%EB%AF%BC%EC%A4%91_%EB%9D%BC%ED%8B%B4%EC%96%B4" title="민중 라틴어">민중 라틴어</a>인데, 고전 라틴어와는 문법, 어휘, 나중에는 발음 면까지 상당한 차이가 있다. 또 라틴어가 로마 제국의 주요 언어였지만, 로마인들이 학습한 문학의 대부분이 그리스어로 되어 있어서 상류층에서는 그리스어도 구사했다. 이후 <a href="/wiki/%EB%B9%84%EC%9E%94%ED%8B%B0%EC%9B%80_%EC%A0%9C%EA%B5%AD" title="비잔티움 제국">비잔티움 제국</a>이 되는 로마 제국의 동부 지역은 <a href="/wiki/%EC%BD%94%EC%9D%B4%EB%84%A4_%EA%B7%B8%EB%A6%AC%EC%8A%A4%EC%96%B4" title="코이네 그리스어">코이네 그리스어</a>가 라틴어로 대체되지 않고 계속 널리 쓰였다.</p>

<h2><span class="mw-headline" id=".EA.B8.B0.EC.9B.90">기원</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%9D%BC%ED%8B%B4%EC%96%B4%EC%9D%98_%EC%97%AD%EC%82%AC&amp;action=edit&amp;section=1" title="부분 편집: 기원">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<div class="thumb tright">
<div class="thumbinner" style="width:302px;"><a class="image" href="/wiki/%ED%8C%8C%EC%9D%BC:Iron_Age_Italy.png"><img alt="" class="thumbimage" data-file-height="525" data-file-width="432" height="365" src="//upload.wikimedia.org/wikipedia/commons/thumb/d/d8/Iron_Age_Italy.png/300px-Iron_Age_Italy.png" srcset="//upload.wikimedia.org/wikipedia/commons/d/d8/Iron_Age_Italy.png 1.5x" width="300"/></a>

기원전 6세기경 철기 시대 <a href="/wiki/%EC%9D%B4%ED%83%88%EB%A6%AC%EC%95%84" title="이탈리아">이탈리아</a>의 대략적인 언어 분포.</div>

<p><a href="/wiki/%EC%9D%B4%ED%83%88%EB%A6%AC%EC%95%84%EC%96%B4%ED%8C%8C" title="이탈리아어파">이탈리아어파</a>는 <a href="/wiki/%EC%9D%B8%EB%8F%84%EC%9C%A0%EB%9F%BD%EC%96%B4%EC%A1%B1" title="인도유럽어족">인도유럽어족</a>의 <a class="new" href="/w/index.php?title=%EC%BC%84%ED%88%BC%EC%96%B4&amp;action=edit&amp;redlink=1" title="켄툼어 (없는 문서)">켄툼어</a>족에 속한다. 여기에는 <a class="mw-redirect" href="/wiki/%EB%A1%9C%EB%A7%9D%EC%8A%A4%EC%96%B4" title="로망스어">로망스어</a>(<a href="/wiki/%ED%94%84%EB%9E%91%EC%8A%A4%EC%96%B4" title="프랑스어">프랑스어</a>, <a href="/wiki/%EC%9D%B4%ED%83%88%EB%A6%AC%EC%95%84%EC%96%B4" title="이탈리아어">이탈리아어</a>, <a class="mw-redirect" href="/wiki/%EC%97%90%EC%8A%A4%ED%8C%8C%EB%83%90%EC%96%B4" title="에스파냐어">에스파냐어</a>, <a href="/wiki/%ED%8F%AC%EB%A5%B4%ED%88%AC%EA%B0%88%EC%96%B4" title="포르투갈어">포르투갈어</a>, <a href="/wiki/%EB%A3%A8%EB%A7%88%EB%8B%88%EC%95%84%EC%96%B4" title="루마니아어">루마니아어</a> 등)와 여러 멸종한 언어가 있다.</p>
<p><b>초기 라틴어</b>(Old Latin, Early Latin, Archaic Latin)는 <a href="/wiki/%EA%B3%A0%EC%A0%84_%EB%9D%BC%ED%8B%B4%EC%96%B4" title="고전 라틴어">고전 라틴어</a> 이전의 라틴어를 일컫는다.</p>
<p>대체로 강세 음절에서 <a class="mw-redirect" href="/wiki/%EC%9D%B8%EB%8F%84-%EC%9C%A0%EB%9F%BD%EC%96%B4" title="인도-유럽어">인도-유럽어</a>의 모음<sup class="reference" id="cite_ref-1"><a href="#cite_note-1">[1]</a></sup> 은 대개 라틴어에도 남아있다. <a href="/wiki/%EC%8A%88%EC%99%80" title="슈와">슈와</a>(schwa indogermanicum, *ə)는 라틴어에는 'a'로 남아있다.(즉 인도-유럽어의 <i>*pəter</i>는 <span style="font-size: smaller;"><a href="/wiki/%EB%9D%BC%ED%8B%B4%EC%96%B4" title="라틴어">라틴어</a>:</span> <span lang="la" xml:lang="la"><i>pater</i></span>가 된다) <a href="/wiki/%EC%9D%B4%EC%A4%91%EB%AA%A8%EC%9D%8C" title="이중모음">이중모음</a>은 초기 라틴어에 남아 있으나 고전 라틴어에는 단모음화되는 경향이 일부 있다(가령 <i>oi</i> &gt; <i>oe</i>, <i>ei</i> &gt; <i>ē</i> &gt; <i>ī</i>).<sup class="reference" id="cite_ref-2"><a href="#cite_note-2">[2]</a></sup></p>
<p>이전 라틴어에서 보이는 다른 음운상의 특징으로는 격 어미 <i>-os</i>와 <i>-om</i>(나중에 <i>-us</i>와 <i>-um</i>이 된다)이다. 또 고전 라틴어에서는 모음 사이의 /s/가 /r/로 바뀌는 경우가 많다. 격 변화(declension)에 그런 현상이 나타나는데, 초기 고전 라틴어 <i>honos</i>, <i>honoris</i>가 고전 라틴어에서는 <i>honor</i>, <i>honoris</i><font color="gray"><small>→"명예"</small></font>가 된다. 어떤 옛 라틴어 문헌에는 이 위치에 /s/가 남아 있기도 하여, <a class="new" href="/w/index.php?title=%EC%95%84%EB%A5%B4%EB%B0%9C%EB%A0%88%EC%8A%A4&amp;action=edit&amp;redlink=1" title="아르발레스 (없는 문서)">아르발레스</a>(arvales) 축제의 <i>lares</i>를 <i>lases</i>라고 쓰기도 한다.</p>
<p><a href="/wiki/%EC%9D%B8%EB%8F%84%EC%9C%A0%EB%9F%BD%EC%A1%B0%EC%96%B4" title="인도유럽조어">원형 인도-유럽어</a>는 원래 8격인데, 라틴어에는 6격<sup class="reference" id="cite_ref-3"><a href="#cite_note-3">[3]</a></sup> 이 남아있다. <a class="mw-redirect" href="/wiki/%EC%9D%B8%EB%8F%84-%EC%9C%A0%EB%9F%BD%EC%96%B4" title="인도-유럽어">인도-유럽어</a>의 <a class="new" href="/w/index.php?title=%EC%B2%98%EA%B2%A9&amp;action=edit&amp;redlink=1" title="처격 (없는 문서)">처격</a>(locative)은 몇몇 지명과 명사의 격변화에 남아있다. 가령 <span lang="la" xml:lang="la"><i>Roma</i></span> <small><a href="/wiki/%EB%A1%9C%EB%A7%88" title="로마">로마</a><sup>[<a href="/wiki/%EC%9C%84%ED%82%A4%EB%B0%B1%EA%B3%BC:%EC%99%B8%EB%9E%98%EC%96%B4_%ED%91%9C%EA%B8%B0%EB%B2%95/%EB%9D%BC%ED%8B%B4%EC%96%B4_%ED%91%9C%EA%B8%B0_%EC%9B%90%EC%B9%99" title="위키백과:외래어 표기법/라틴어 표기 원칙">*</a>]</sup></small>는 <i>Romae</i>, <i>domus</i><font color="gray"><small>→집</small></font>은 <i>domī</i>가 처격이다. <a class="new" href="/w/index.php?title=%EA%B5%AC%EA%B2%A9&amp;action=edit&amp;redlink=1" title="구격 (없는 문서)">구격</a>(instrumental case)은 부사 형태 어미 <i>-ē</i>로 남아있다.<sup class="reference" id="cite_ref-4"><a href="#cite_note-4">[4]</a></sup></p>
<h2><span class="mw-headline" id=".EA.B3.A0.EC.A0.84_.EB.9D.BC.ED.8B.B4.EC.96.B4">고전 라틴어</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%9D%BC%ED%8B%B4%EC%96%B4%EC%9D%98_%EC%97%AD%EC%82%AC&amp;action=edit&amp;section=2" title="부분 편집: 고전 라틴어">편집</a><span class="mw-editsection-bracket">]</span></span></h2>

<p><b>고전 라틴어</b>는 <a href="/wiki/%EA%B3%A0%EB%8C%80_%EB%A1%9C%EB%A7%88" title="고대 로마">고대 로마</a>에서 보통 "고전적인" <a href="/wiki/%EB%9D%BC%ED%8B%B4_%EB%AC%B8%ED%95%99" title="라틴 문학">라틴 문학</a>으로 여겨지는 <a href="/wiki/%EB%9D%BC%ED%8B%B4%EC%96%B4" title="라틴어">라틴어</a> 형태를 말한다. 고전 라틴어는 라틴 문학의 황금시대(대체로 기원전 1세기부터 기원후 1세기까지)와 백은 시대(기원후 1,2세기)까지를 이른다.</p>
<p>오늘날 "고전 라틴어"라 부르는 것은 매우 정제되고 양식화된 문학 언어로 <a class="new" href="/w/index.php?title=%EC%B4%88%EA%B8%B0_%EB%9D%BC%ED%8B%B4%EC%96%B4&amp;action=edit&amp;redlink=1" title="초기 라틴어 (없는 문서)">초기 라틴어</a>(초기 라틴어 작품은 별로 남아있지 않다)에서 뽑아내어 만들어낸 것이다. 고전 라틴어는 <a class="new" href="/w/index.php?title=%EC%95%84%ED%8B%B0%EC%B9%B4_%EA%B7%B8%EB%A6%AC%EC%8A%A4%EC%96%B4&amp;action=edit&amp;redlink=1" title="아티카 그리스어 (없는 문서)">아티카 그리스어</a>를 원형으로 초기 라틴어를 재구성한 것이다. 고전 라틴어는 <a href="/wiki/%EB%8C%80_%EC%B9%B4%ED%86%A0" title="대 카토">대 카토</a>나 <a class="mw-redirect" href="/wiki/%ED%94%8C%EB%9D%BC%EC%9A%B0%ED%88%AC%EC%8A%A4" title="플라우투스">플라우투스</a>, 혹은 <a href="/wiki/%EB%A3%A8%ED%81%AC%EB%A0%88%ED%8B%B0%EC%9A%B0%EC%8A%A4" title="루크레티우스">루크레티우스</a> 일부 작품까지 초기의 라틴 문학과 여러 면에서 차이가 있다. 이것은 초기 라틴어에서 분화하여, 어미꼴도 <i>-om</i>과 <i>-os</i>에서 <i>-um</i>과 <i>-us</i>로 바뀌었으며, 낱말의 의미가 확대되는(가령 '놀랍게'를 뜻하는 <i>forte</i>가 '열심히', '몹시'도 뜻하게 된다.) 등 어휘면에서도 차이가 생겼다. 고전 라틴어는 고대 그리스어의 <a class="mw-redirect" href="/wiki/%EA%B3%A0%EC%A0%80_%EC%95%85%EC%84%BC%ED%8A%B8" title="고저 악센트">고저 악센트</a>(pitch accent)와 달리 <a class="mw-redirect" href="/wiki/%EA%B0%95%EC%84%B8_%EC%95%85%EC%84%BC%ED%8A%B8" title="강세 악센트">강세 악센트</a>(stress accent)로 발음한다.<sup class="reference" id="cite_ref-5"><a href="#cite_note-5">[5]</a></sup></p>
<p>로마의 일반인들이 특히 기원후 2세기부터 쓰던 구어 라틴어는 보통 <a class="mw-redirect" href="/wiki/%EB%AF%BC%EC%A4%91_%EB%9D%BC%ED%8B%B4%EC%96%B4" title="민중 라틴어">민중 라틴어</a>라고 한다. 민중 라틴어는 어휘, 문법면에서 고전 라틴어와 차이가 있으며, 나중에는 발음도 달라졌다.</p>
<h3><span class="mw-headline" id=".ED.99.A9.EA.B8.88_.EC.8B.9C.EB.8C.80">황금 시대</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%9D%BC%ED%8B%B4%EC%96%B4%EC%9D%98_%EC%97%AD%EC%82%AC&amp;action=edit&amp;section=3" title="부분 편집: 황금 시대">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p><b>라틴 문학의 황금 시대</b>(Latinitas aurea)는 대략 기원전 75년부터 기원후 14년의 기간을 이르며, 대략 <a href="/wiki/%EB%A1%9C%EB%A7%88_%EA%B3%B5%ED%99%94%EC%A0%95" title="로마 공화정">로마 공화정</a>이 몰락하고 <a href="/wiki/%EC%95%84%EC%9A%B0%EA%B5%AC%EC%8A%A4%ED%88%AC%EC%8A%A4" title="아우구스투스">아우구스투스</a>가 집권한 시대이다. 여러 고전 연구가들은 이 시기를 라틴 문학의 정점으로 보고 있으며, 세련되고 양식화된 문어인 고전 라틴어를 쓰는 것은 다른 저자들도 따라야 할 이상적인 규범이라고 생각했다. 고전 라틴어는 기원후 1, 2세기인 라틴 문학의 백은 시대까지 이어졌다.</p>
<h3><span class="mw-headline" id=".EB.B0.B1.EC.9D.80_.EC.8B.9C.EB.8C.80">백은 시대</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%9D%BC%ED%8B%B4%EC%96%B4%EC%9D%98_%EC%97%AD%EC%82%AC&amp;action=edit&amp;section=4" title="부분 편집: 백은 시대">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>로마의 문학사에서 <b>백은 시대</b>는 황금 시대 직후 기원후 1, 2세기를 일컫는다. 이 시대의 문학은 전통적으로 (부당하다고 볼 수도 있으나) 황금 시대 문학보다 저급하다고 여겨졌다. 백은 시대는 두 기간으로 나눌 수 있는데, 첫 기간은 기원후 1세기 후반기의 급진적인 실험기이고, 이후는 2세기에 새롭게 나타난 <a href="/wiki/%EC%8B%A0%EA%B3%A0%EC%A0%84%EC%A3%BC%EC%9D%98" title="신고전주의">신고전주의</a>이다.</p>
<p><a href="/wiki/%EB%84%A4%EB%A1%9C" title="네로">네로</a>와 <a href="/wiki/%EB%8F%84%EB%AF%B8%ED%8B%B0%EC%95%84%EB%88%84%EC%8A%A4" title="도미티아누스">도미티아누스</a> 치세에 <a class="mw-redirect" href="/wiki/%EC%86%8C_%EC%84%B8%EB%84%A4%EC%B9%B4" title="소 세네카">소 세네카</a>와 <a href="/wiki/%EB%A3%A8%EC%B9%B4%EB%88%84%EC%8A%A4" title="루카누스">루카누스</a>, <a href="/wiki/%EC%8A%A4%ED%83%80%ED%8B%B0%EC%9A%B0%EC%8A%A4" title="스타티우스">스타티우스</a> 같은 시인들은 독특한 양식을 추구하여, 후대 비평가들의 찬사와 비난을 함께 받았다.</p>
<h2><span class="mw-headline" id=".EB.AF.BC.EC.A4.91_.EB.9D.BC.ED.8B.B4.EC.96.B4">민중 라틴어</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%9D%BC%ED%8B%B4%EC%96%B4%EC%9D%98_%EC%97%AD%EC%82%AC&amp;action=edit&amp;section=5" title="부분 편집: 민중 라틴어">편집</a><span class="mw-editsection-bracket">]</span></span></h2>

<p>구어 라틴어는 발음, 어휘, 문법면에서 고전 라틴어와 차이가 있다. 민중 라틴어의 어떤 특성은 제정 후기부터 나타났다. 다른 특징은 구어에서 훨씬 이전에 (최소한 상류층 방언에서) 나타난 것 같다. 민중 라틴어는 문어가 아닌 구어인데, 이 시기에 라틴어 구어가 여러 방언으로 갈라졌다는 증거가 있기 때문이다. 음운 자료가 거의 없으므로, 이 시기의 일상적으로 구사하는 라틴어가 어떠했는지는 의문이다.</p>
<h2><span class="mw-headline" id=".EA.B0.81.EC.A3.BC">각주</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%9D%BC%ED%8B%B4%EC%96%B4%EC%9D%98_%EC%97%AD%EC%82%AC&amp;action=edit&amp;section=6" title="부분 편집: 각주">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
