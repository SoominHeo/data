<h2><span class="mw-headline" id="2011.EB.85.84_2.EC.9B.94_27.EC.9D.BC"><a href="/wiki/2011%EB%85%84" title="2011년">2011년</a> <a href="/wiki/2%EC%9B%94_27%EC%9D%BC" title="2월 27일">2월 27일</a></span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=2011%EB%85%84_2%EC%9B%94&amp;action=edit&amp;section=1" title="부분 편집: 2011년 2월 27일">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<ul>
<li>유엔 안전보장이사회는 <a href="/wiki/%EB%A6%AC%EB%B9%84%EC%95%84" title="리비아">리비아</a> 제재 결의를 만장일치로 채택했다.</li>
</ul>
<h2><span class="mw-headline" id="2011.EB.85.84_2.EC.9B.94_26.EC.9D.BC"><a href="/wiki/2011%EB%85%84" title="2011년">2011년</a> <a href="/wiki/2%EC%9B%94_26%EC%9D%BC" title="2월 26일">2월 26일</a></span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=2011%EB%85%84_2%EC%9B%94&amp;action=edit&amp;section=2" title="부분 편집: 2011년 2월 26일">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<ul>
<li><a href="/wiki/%EB%8C%80%ED%95%9C%EB%AF%BC%EA%B5%AD" title="대한민국">대한민국</a> <a href="/wiki/%EC%9A%B8%EC%82%B0%EA%B4%91%EC%97%AD%EC%8B%9C" title="울산광역시">울산광역시</a>에서 <a href="/wiki/%EA%B5%AC%EC%A0%9C%EC%97%AD" title="구제역">구제역</a>이 확인되었다.</li>
</ul>
<h2><span class="mw-headline" id="2011.EB.85.84_2.EC.9B.94_22.EC.9D.BC"><a href="/wiki/2011%EB%85%84" title="2011년">2011년</a> <a href="/wiki/2%EC%9B%94_22%EC%9D%BC" title="2월 22일">2월 22일</a></span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=2011%EB%85%84_2%EC%9B%94&amp;action=edit&amp;section=3" title="부분 편집: 2011년 2월 22일">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<ul>
<li><a href="/wiki/%EB%A6%AC%EB%B9%84%EC%95%84" title="리비아">리비아</a>의 카다피가 방송에 나와 퇴진할 뜻이 없다는 입장을 밝혔다.</li>
<li><a href="/wiki/%EB%89%B4%EC%A7%88%EB%9E%9C%EB%93%9C" title="뉴질랜드">뉴질랜드</a>에서 규모 6.3의 지진이 발생했다.</li>
<li><a href="/wiki/%EB%A7%88%EC%9D%B4%ED%81%AC%EB%A1%9C%EC%86%8C%ED%94%84%ED%8A%B8" title="마이크로소프트">마이크로소프트</a>에서 <a href="/wiki/%EC%9C%88%EB%8F%84%EC%9A%B0_7" title="윈도우 7">윈도우 7 서비스팩 1</a>이 발표되다.</li>
</ul>
<h2><span class="mw-headline" id="2011.EB.85.84_2.EC.9B.94_15.EC.9D.BC"><a href="/wiki/2011%EB%85%84" title="2011년">2011년</a> <a href="/wiki/2%EC%9B%94_15%EC%9D%BC" title="2월 15일">2월 15일</a></span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=2011%EB%85%84_2%EC%9B%94&amp;action=edit&amp;section=4" title="부분 편집: 2011년 2월 15일">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<ul>
<li>브라질의 축구선수 <a href="/wiki/%ED%98%B8%EB%82%98%EC%9A%B0%EB%91%90" title="호나우두">호나우두</a>가 35세로 현역 은퇴를 선언했다.</li>
</ul>
<h2><span class="mw-headline" id="2011.EB.85.84_2.EC.9B.94_14.EC.9D.BC"><a href="/wiki/2011%EB%85%84" title="2011년">2011년</a> <a href="/wiki/2%EC%9B%94_14%EC%9D%BC" title="2월 14일">2월 14일</a></span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=2011%EB%85%84_2%EC%9B%94&amp;action=edit&amp;section=5" title="부분 편집: 2011년 2월 14일">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<ul>
<li><a href="/wiki/%ED%99%94%EC%84%B1" title="화성">화성</a> 탐사 여행 국제 시뮬레이션팀이 <a href="/wiki/%EC%A7%80%EA%B5%AC" title="지구">지구</a>로 부터 출발한지 8개월만에 <a href="/wiki/%ED%99%94%EC%84%B1" title="화성">화성</a>에 도착하였다.</li>
</ul>
<h2><span class="mw-headline" id="2011.EB.85.84_2.EC.9B.94_11.EC.9D.BC"><a href="/wiki/2011%EB%85%84" title="2011년">2011년</a> <a href="/wiki/2%EC%9B%94_11%EC%9D%BC" title="2월 11일">2월 11일</a></span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=2011%EB%85%84_2%EC%9B%94&amp;action=edit&amp;section=6" title="부분 편집: 2011년 2월 11일">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<ul>
<li>오전 10시 45분에 <a href="/wiki/%EB%B6%80%EC%82%B0%EC%97%AD" title="부산역">부산역</a>을 출발하여 <a href="/wiki/%EA%B4%91%EB%AA%85%EC%97%AD" title="광명역">광명역</a>으로 가던 <a href="/wiki/KTX-%EC%82%B0%EC%B2%9C" title="KTX-산천">KTX-산천</a>이 <a href="/wiki/%EA%B4%91%EB%AA%85%EC%97%AD" title="광명역">광명역</a> 270여m 못 미친 지점에서 탈선하는 사고가 발생하였다.</li>
<li>오후 6시경 <a href="/wiki/%EC%84%9C%EC%9A%B8_%EC%A7%80%ED%95%98%EC%B2%A0_2%ED%98%B8%EC%84%A0" title="서울 지하철 2호선">서울 지하철 2호선</a> <a href="/wiki/%ED%99%8D%EB%8C%80%EC%9E%85%EA%B5%AC%EC%97%AD" title="홍대입구역">홍대입구역</a>에서 열차가 고장나는 사고가 발생하였다.</li>
<li><a href="/wiki/%EC%9D%B4%EC%A7%91%ED%8A%B8" title="이집트">이집트</a>의 30년 장기 집권 대통령인 <a href="/wiki/%ED%98%B8%EC%8A%A4%EB%8B%88_%EB%AC%B4%EB%B0%94%EB%9D%BC%ED%81%AC" title="호스니 무바라크">호스니 무바라크</a>가 18일간 이어진 시위 끝에 결국 굴복하고 대통령 직을 사퇴하였다. 권력은 이집트 군부에게 이양되었다.</li>
</ul>
<h2><span class="mw-headline" id="2011.EB.85.84_2.EC.9B.94_10.EC.9D.BC"><a href="/wiki/2011%EB%85%84" title="2011년">2011년</a> <a href="/wiki/2%EC%9B%94_10%EC%9D%BC" title="2월 10일">2월 10일</a></span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=2011%EB%85%84_2%EC%9B%94&amp;action=edit&amp;section=7" title="부분 편집: 2011년 2월 10일">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<ul>
<li><a href="/wiki/%EC%9D%B8%EB%8F%84" title="인도">인도</a> 재벌 <a href="/wiki/%EB%AC%B4%EC%BC%80%EC%8B%9C_%EC%95%94%EB%B0%94%EB%8B%88" title="무케시 암바니">무케시 암바니</a> 회장이 이메일 성명을 통해 "라이벌회사가 퍼뜨린 루머로 인한 주가 하락으로 26억달러의 손실"을 입었다고 밝혔다.</li>
</ul>
<h2><span class="mw-headline" id="2011.EB.85.84_2.EC.9B.94_9.EC.9D.BC"><a href="/wiki/2011%EB%85%84" title="2011년">2011년</a> <a href="/wiki/2%EC%9B%94_9%EC%9D%BC" title="2월 9일">2월 9일</a></span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=2011%EB%85%84_2%EC%9B%94&amp;action=edit&amp;section=8" title="부분 편집: 2011년 2월 9일">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<ul>
<li>남북 군사회담이 입장차만 확인한 채 결렬되었다.</li>
</ul>
<h2><span class="mw-headline" id="2011.EB.85.84_2.EC.9B.94_5.EC.9D.BC"><a href="/wiki/2011%EB%85%84" title="2011년">2011년</a> <a href="/wiki/2%EC%9B%94_5%EC%9D%BC" title="2월 5일">2월 5일</a></span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=2011%EB%85%84_2%EC%9B%94&amp;action=edit&amp;section=9" title="부분 편집: 2011년 2월 5일">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<ul>
<li><a href="/wiki/%EC%A1%B0%EC%84%A0%EB%AF%BC%EC%A3%BC%EC%A3%BC%EC%9D%98%EC%9D%B8%EB%AF%BC%EA%B3%B5%ED%99%94%EA%B5%AD" title="조선민주주의인민공화국">조선민주주의인민공화국</a> 주민 31명이 어선을 타고 <a href="/wiki/%EC%97%B0%ED%8F%89%EB%8F%84" title="연평도">연평도</a> 동북쪽 북방한계선을 넘어와 귀순 의사를 밝힌 것으로 알려졌다.</li>
</ul>
