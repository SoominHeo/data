<p><b>랄프 페어만</b> (Ralf Fährmann, <a href="/wiki/1988%EB%85%84" title="1988년">1988년</a> <a href="/wiki/9%EC%9B%94_27%EC%9D%BC" title="9월 27일">9월 27일</a> <a href="/wiki/%EC%BC%90%EB%8B%88%EC%B8%A0" title="켐니츠">켐니츠</a> ~) 은 <a href="/wiki/%EB%8F%85%EC%9D%BC" title="독일">독일</a>의 <a href="/wiki/%EC%B6%95%EA%B5%AC_%EC%84%A0%EC%88%98" title="축구 선수">축구 선수</a>로, 현재 <a class="mw-redirect" href="/wiki/%ED%91%B8%EC%8A%A4%EB%B3%BC-%EB%B6%84%EB%8D%B0%EC%8A%A4%EB%A6%AC%EA%B0%80" title="푸스볼-분데스리가">독일 분데스리가</a>의 <a href="/wiki/FC_%EC%83%AC%EC%BC%80_04" title="FC 샬케 04">FC 샬케 04</a> 소속 <a href="/wiki/%EA%B3%A8%ED%82%A4%ED%8D%BC" title="골키퍼">골키퍼</a>이다. <a href="/wiki/%EB%A7%88%EB%88%84%EC%97%98_%EB%85%B8%EC%9D%B4%EC%96%B4" title="마누엘 노이어">마누엘 노이어</a>가 <a href="/wiki/FC_%EB%B0%94%EC%9D%B4%EC%97%90%EB%A5%B8_%EB%AE%8C%ED%97%A8" title="FC 바이에른 뮌헨">FC 바이에른 뮌헨</a>으로 이적하게 됨에 따라, 그의 대체자로 영입되었다..</p>

<h2><span class="mw-headline" id=".ED.81.B4.EB.9F.BD_.EA.B2.BD.EB.A0.A5">클럽 경력</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%9E%84%ED%94%84_%ED%8E%98%EC%96%B4%EB%A7%8C&amp;action=edit&amp;section=1" title="부분 편집: 클럽 경력">원본 편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>페어만은 본래 <a href="/wiki/FC_%EC%83%AC%EC%BC%80_04" title="FC 샬케 04">FC 샬케 04</a> 출신으로, <a href="/wiki/2008%EB%85%84" title="2008년">2008년</a> <a href="/wiki/9%EC%9B%94_13%EC%9D%BC" title="9월 13일">9월 13일</a>, <a href="/wiki/%EB%B3%B4%EB%A3%A8%EC%8B%9C%EC%95%84_%EB%8F%84%EB%A5%B4%ED%8A%B8%EB%AC%B8%ED%8A%B8" title="보루시아 도르트문트">보루시아 도르트문트</a>와 3-3으로 비긴 경기에서 데뷔전을 치렀다. <a href="/wiki/2008%EB%85%84" title="2008년">2008년</a> <a href="/wiki/9%EC%9B%94_16%EC%9D%BC" title="9월 16일">9월 16일</a>, 그는 <a class="mw-redirect" href="/wiki/UEFA_%EC%BB%B5" title="UEFA 컵">UEFA 컵</a>의 <a class="mw-redirect" href="/wiki/APOEL_FC" title="APOEL FC">APOEL FC</a>전에서 유럽대항전 첫 경기를 치렀다.</p>
<p>2008-09시즌 종료 후, 그는 <a href="/wiki/FC_%EC%83%AC%EC%BC%80_04" title="FC 샬케 04">FC 샬케 04</a>를 떠나 <a href="/wiki/%EC%95%84%EC%9D%B8%ED%8A%B8%EB%9D%BC%ED%9D%90%ED%8A%B8_%ED%94%84%EB%9E%91%ED%81%AC%ED%91%B8%EB%A5%B4%ED%8A%B8" title="아인트라흐트 프랑크푸르트">아인트라흐트 프랑크푸르트</a>로 자유이적 하였다. 그리고, <a href="/wiki/2012%EB%85%84" title="2012년">2012년</a> <a href="/wiki/6%EC%9B%94_30%EC%9D%BC" title="6월 30일">6월 30일</a>까지 지속되는 계약서에 서명하였다.</p>
<p>그는 <a href="/wiki/2011%EB%85%84" title="2011년">2011년</a>, <a href="/wiki/%EB%A7%88%EB%88%84%EC%97%98_%EB%85%B8%EC%9D%B4%EC%96%B4" title="마누엘 노이어">마누엘 노이어</a>가 <a href="/wiki/FC_%EB%B0%94%EC%9D%B4%EC%97%90%EB%A5%B8_%EB%AE%8C%ED%97%A8" title="FC 바이에른 뮌헨">FC 바이에른 뮌헨</a>에 €22M에 판매됨에 따라, <a href="/wiki/FC_%EC%83%AC%EC%BC%80_04" title="FC 샬케 04">FC 샬케 04</a>로 복귀하였다. <a href="/wiki/2011%EB%85%84" title="2011년">2011년</a>, <a href="/wiki/DFB-%ED%8F%AC%EC%B9%BC" title="DFB-포칼">DFB-포칼</a>우승팀으로 참가한 <a href="/wiki/FC_%EC%83%AC%EC%BC%80_04" title="FC 샬케 04">FC 샬케 04</a>는 <a class="mw-redirect" href="/wiki/%EB%8F%85%EC%9D%BC_%EC%8A%88%ED%8D%BC%EC%BB%B5" title="독일 슈퍼컵">독일 슈퍼컵</a>에서 리그 챔피언인 <a href="/wiki/%EB%B3%B4%EB%A3%A8%EC%8B%9C%EC%95%84_%EB%8F%84%EB%A5%B4%ED%8A%B8%EB%AC%B8%ED%8A%B8" title="보루시아 도르트문트">보루시아 도르트문트</a>를 상대하였다. 페어만은 90분동안 선방쇼를 펼쳤고 0:0으로 끝나 바로 <a href="/wiki/%EC%8A%B9%EB%B6%80%EC%B0%A8%EA%B8%B0" title="승부차기">승부차기</a>에 돌입하였다. 페어만은 <a href="/wiki/%EC%BC%80%EB%B9%88_%EA%B7%B8%EB%A1%9C%EC%8A%A4%ED%81%AC%EB%A1%9C%EC%9D%B4%EC%B8%A0" title="케빈 그로스크로이츠">케빈 그로스크로이츠</a>와 <a href="/wiki/%EC%9D%B4%EB%B0%98_%ED%8E%98%EB%A6%AC%EC%8B%9C%EC%B9%98" title="이반 페리시치">이반 페리시치</a>의 킥을 선방하여<sup class="reference" id="cite_ref-F.C3.A4hrmann_1-0"><a href="#cite_note-F.C3.A4hrmann-1">[1]</a></sup>, 샬케가 처음으로 슈퍼컵을 들어올리도록 하였다. 그는 그 경기의 MVP로 선정되었다.</p>
<h2><span class="mw-headline" id=".EA.B5.AD.EA.B0.80.EB.8C.80.ED.91.9C_.EA.B2.BD.EB.A0.A5">국가대표 경력</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%9E%84%ED%94%84_%ED%8E%98%EC%96%B4%EB%A7%8C&amp;action=edit&amp;section=2" title="부분 편집: 국가대표 경력">원본 편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>그는 <a href="/wiki/%EB%8F%85%EC%9D%BC_%EC%B6%95%EA%B5%AC_%EA%B5%AD%EA%B0%80%EB%8C%80%ED%91%9C%ED%8C%80" title="독일 축구 국가대표팀">독일 청소년 국가대표</a>로 몇차례 차출된 바 있다.</p>
<h2><span class="mw-headline" id=".EC.82.AC.EC.83.9D.ED.99.9C">사생활</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%9E%84%ED%94%84_%ED%8E%98%EC%96%B4%EB%A7%8C&amp;action=edit&amp;section=3" title="부분 편집: 사생활">원본 편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>랄프 페어만의 형제 팔크 페어만 또한 축구선수이며 <a class="new" href="/w/index.php?title=FSV_%EC%B8%A0%EB%B9%84%EC%B9%B4%EC%9A%B0%EC%9D%98&amp;action=edit&amp;redlink=1" title="FSV 츠비카우의 (없는 문서)">FSV 츠비카우의</a> 선수로 활동하고 있다.</p>
