<h2><span class="mw-headline" id="2011.EB.85.84_7.EC.9B.94_30.EC.9D.BC"><a href="/wiki/2011%EB%85%84" title="2011년">2011년</a> <a href="/wiki/7%EC%9B%94_30%EC%9D%BC" title="7월 30일">7월 30일</a></span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=2011%EB%85%84_7%EC%9B%94&amp;action=edit&amp;section=1" title="부분 편집: 2011년 7월 30일">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<ul>
<li><a href="/wiki/%EC%BD%9C%EB%A1%AC%EB%B9%84%EC%95%84" title="콜롬비아">콜롬비아</a>에서 <a href="/wiki/2011%EB%85%84_FIFA_U-20_%EC%9B%94%EB%93%9C%EC%BB%B5" title="2011년 FIFA U-20 월드컵">2011년 FIFA U-20 월드컵</a>이 개막하였다.</li>
<li><a href="/wiki/%EC%84%9C%EC%9A%B8%EC%97%AD" title="서울역">서울</a>발 <a href="/wiki/%EB%B6%80%EC%82%B0%EC%97%AD" title="부산역">부산</a>행 <a class="mw-redirect" href="/wiki/KTX" title="KTX">KTX</a> 607열차와 승용차가 충돌하여 1명이 사망하고 열차 운행에 차질이 생겼다.</li>
</ul>
<h2><span class="mw-headline" id="2011.EB.85.84_7.EC.9B.94_29.EC.9D.BC"><a href="/wiki/2011%EB%85%84" title="2011년">2011년</a> <a href="/wiki/7%EC%9B%94_29%EC%9D%BC" title="7월 29일">7월 29일</a></span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=2011%EB%85%84_7%EC%9B%94&amp;action=edit&amp;section=2" title="부분 편집: 2011년 7월 29일">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<ul>
<li><a class="mw-redirect" href="/wiki/%EC%84%9C%EC%9A%B8%EC%A4%91%EC%95%99%EC%A7%80%EA%B2%80" title="서울중앙지검">서울중앙지검</a> 공안1부는 <a class="mw-redirect" href="/wiki/%EC%A1%B0%EC%84%A0_%EB%85%B8%EB%8F%99%EB%8B%B9" title="조선 노동당">조선 노동당</a> 225국의 지령을 받아 대한민국에 지하당 <a class="mw-redirect" href="/wiki/%EC%99%95%EC%9E%AC%EC%82%B0_%EA%B0%84%EC%B2%A9%EB%8B%A8" title="왕재산 간첩단">왕재산 간첩단</a>을 구성해 간첩활동을 한 혐의로 정치인, 미디어 업체 종사자 등 5명을 구속했으며, 학계와 정계, 노동계 등 각계 인사 수십 명에 대한 수사를 시작하였다.</li>
<li><a class="mw-redirect" href="/wiki/%EC%84%9C%EC%9A%B8" title="서울">서울</a>과 <a href="/wiki/%EA%B0%95%EC%9B%90%EB%8F%84" title="강원도">강원도</a> 등 집중 호우 피해 지역에 대한 본격적인 복구 지원이 시작됐다.</li>
</ul>
<h2><span class="mw-headline" id="2011.EB.85.84_7.EC.9B.94_28.EC.9D.BC"><a href="/wiki/2011%EB%85%84" title="2011년">2011년</a> <a href="/wiki/7%EC%9B%94_28%EC%9D%BC" title="7월 28일">7월 28일</a></span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=2011%EB%85%84_7%EC%9B%94&amp;action=edit&amp;section=3" title="부분 편집: 2011년 7월 28일">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<ul>
<li>오전 4시, <a href="/wiki/%EC%9D%B8%EC%B2%9C%EA%B5%AD%EC%A0%9C%EA%B3%B5%ED%95%AD" title="인천국제공항">인천</a>에서 <a class="mw-redirect" href="/wiki/%ED%91%B8%EB%91%A5_%EA%B5%AD%EC%A0%9C%EA%B3%B5%ED%95%AD" title="푸둥 국제공항">푸둥</a>으로 향하던 <a class="mw-redirect" href="/wiki/%EC%95%84%EC%8B%9C%EC%95%84%EB%82%98%ED%95%AD%EA%B3%B5_991%ED%8E%B8_%EC%B6%94%EB%9D%BD_%EC%82%AC%EA%B3%A0" title="아시아나항공 991편 추락 사고">아시아나 보잉 747 화물기</a>가 <a href="/wiki/%EC%A0%9C%EC%A3%BC%EB%8F%84" title="제주도">제주도</a> 해상에서 추락하였으며, 승무원 2명이 숨졌다.</li>
</ul>
<h2><span class="mw-headline" id="2011.EB.85.84_7.EC.9B.94_27.EC.9D.BC"><a href="/wiki/2011%EB%85%84" title="2011년">2011년</a> <a href="/wiki/7%EC%9B%94_27%EC%9D%BC" title="7월 27일">7월 27일</a></span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=2011%EB%85%84_7%EC%9B%94&amp;action=edit&amp;section=4" title="부분 편집: 2011년 7월 27일">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<ul>
<li><a href="/wiki/%EB%8C%80%ED%95%9C%EB%AF%BC%EA%B5%AD" title="대한민국">대한민국</a>의 <a class="mw-redirect mw-disambig" href="/wiki/%EC%A4%91%EB%B6%80_%EC%A7%80%EB%B0%A9" title="중부 지방">중부 지방</a>에 최고 291mm에 달하는 <a class="mw-redirect" href="/wiki/2011%EB%85%84_7%EC%9B%94_%EB%8C%80%ED%95%9C%EB%AF%BC%EA%B5%AD_%EC%A4%91%EB%B6%80%EC%A7%80%EB%B0%A9_%EC%A7%91%EC%A4%91%ED%98%B8%EC%9A%B0" title="2011년 7월 대한민국 중부지방 집중호우">집중호우가 발생</a>하면서 <a class="mw-redirect" href="/wiki/%EC%84%9C%EC%9A%B8" title="서울">서울</a>을 비롯한 대도시에서 도로가 통제되거나 지하철 역이 수몰되고, 주택이 침수되는 등 많은 피해를 겪었다.
<ul>
<li><a href="/wiki/%EC%B6%98%EC%B2%9C%EC%8B%9C" title="춘천시">춘천시</a>에서 산사태가 발생하여 <a href="/wiki/%EC%9D%B8%ED%95%98%EB%8C%80%ED%95%99%EA%B5%90" title="인하대학교">인하대학교</a> 학생 등 13명이 사망하고 29명이 부상당했다.</li>
<li><a href="/wiki/%EC%9A%B0%EB%A9%B4%EC%82%B0" title="우면산">우면산</a>에서 산사태가 발생하여 5명이 사망하고 1명이 실종되었다. 또한 <a class="mw-redirect" href="/wiki/EBS" title="EBS">EBS</a>는 송출시설 침수로 인한 방송중단 사태를 겪었다.</li>
</ul>
</li>
</ul>

<li><a class="mw-redirect" href="/wiki/%EC%9C%94%EB%B8%94%EB%8D%98_%EC%B1%94%ED%94%BC%EC%96%B8%EC%8B%AD" title="윔블던 챔피언십">윔블던 챔피언십</a>에서 남자 단식은 <a class="mw-redirect" href="/wiki/%EB%85%B8%EB%B0%95_%EC%A1%B0%EC%BD%94%EB%B9%84%EC%B9%98" title="노박 조코비치">노박 조코비치</a>가 우승했고 여자 단식은 <a class="new" href="/w/index.php?title=%ED%8E%98%ED%8A%B8%EB%9D%BC_%ED%81%AC%EB%B9%84%ED%86%A0%EB%B0%94&amp;action=edit&amp;redlink=1" title="페트라 크비토바 (없는 문서)">페트라 크비토바</a>가 우승했다.</li>
</ul>
<h2><span class="mw-headline" id="2011.EB.85.84_7.EC.9B.94_2.EC.9D.BC"><a href="/wiki/2011%EB%85%84" title="2011년">2011년</a> <a href="/wiki/7%EC%9B%94_2%EC%9D%BC" title="7월 2일">7월 2일</a></span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=2011%EB%85%84_7%EC%9B%94&amp;action=edit&amp;section=25" title="부분 편집: 2011년 7월 2일">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<ul>
<li><a class="mw-redirect" href="/wiki/%ED%95%9C%EB%82%98%EB%9D%BC%EB%8B%B9" title="한나라당">한나라당</a> <a href="/wiki/%EC%9D%B4%ED%95%B4%EB%B4%89_(1942%EB%85%84)" title="이해봉 (1942년)">이해봉</a> 전국위원회 의장이 7·4전당대회 당헌 개정안에 관한 법원의 효력 정지 결정에 대해 책임을 지고 사퇴하였다.</li>
<li>모나코 궁에서 모나코 공작 <a class="mw-disambig" href="/wiki/%EC%95%8C%EB%B2%A0%EB%A5%B4_2%EC%84%B8" title="알베르 2세">알베르 2세</a>와 <a class="mw-redirect" href="/wiki/%EC%83%A4%EB%A5%BC%EB%A6%B0_%EC%9C%84%ED%8A%B8%EC%8A%A4%ED%86%A1" title="샤를린 위트스톡">샤를린 위트스톡</a>의 결혼식이 열렸다.</li>
<li><a href="/wiki/%EC%95%84%EB%A5%B4%ED%97%A8%ED%8B%B0%EB%82%98" title="아르헨티나">아르헨티나</a>에서 <a href="/wiki/2011%EB%85%84_%EC%BD%94%ED%8C%8C_%EC%95%84%EB%A9%94%EB%A6%AC%EC%B9%B4" title="2011년 코파 아메리카">2011년 코파 아메리카</a> 대회가 개막하였다.</li>
</ul>
<h2><span class="mw-headline" id="2011.EB.85.84_7.EC.9B.94_1.EC.9D.BC"><a href="/wiki/2011%EB%85%84" title="2011년">2011년</a> <a href="/wiki/7%EC%9B%94_1%EC%9D%BC" title="7월 1일">7월 1일</a></span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=2011%EB%85%84_7%EC%9B%94&amp;action=edit&amp;section=26" title="부분 편집: 2011년 7월 1일">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<ul>
<li><a href="/wiki/%EC%97%AC%ED%98%B8%EC%99%80%EC%9D%98_%EC%A6%9D%EC%9D%B8" title="여호와의 증인">여호와의 증인</a> 등 <a class="mw-redirect" href="/wiki/%EC%96%91%EC%8B%AC%EC%A0%81_%EB%B3%91%EC%97%AD%EA%B1%B0%EB%B6%80%EC%9E%90" title="양심적 병역거부자">양심적 병역거부자</a> <a class="mw-redirect" href="/wiki/%EB%8C%80%EC%B2%B4%EB%B3%B5%EB%AC%B4" title="대체복무">대체복무</a> 허용 법안 <a href="/wiki/%EB%8C%80%ED%95%9C%EB%AF%BC%EA%B5%AD_%EA%B5%AD%ED%9A%8C" title="대한민국 국회">국회</a> 발의, <a class="mw-disambig" href="/wiki/%EB%AF%BC%EC%A3%BC%EB%8B%B9" title="민주당">민주당</a> 김부겸 <a class="mw-redirect" href="/wiki/%EA%B5%AD%ED%9A%8C%EC%9D%98%EC%9B%90" title="국회의원">국회의원</a> 등 다수의 의원들이 공동 발의하였다.</li>
<li><a href="/wiki/%EC%A4%91%ED%99%94%EC%9D%B8%EB%AF%BC%EA%B3%B5%ED%99%94%EA%B5%AD" title="중화인민공화국">중화인민공화국</a>의 <a class="mw-redirect" href="/wiki/%EC%82%B0%EB%91%A5%EC%84%B1" title="산둥성">산둥성</a>에서 세계에서 가장 긴 다리인 자우저우완 대교가 착공되었다.</li>
<li><a href="/wiki/%EB%AA%A8%EB%A1%9C%EC%BD%94" title="모로코">모로코</a> 민주 헌법이 국민투표 98%의 지지로 승인되었다</li>
</ul>
