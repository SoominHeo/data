<p><b>랑랑</b>(郞朗), Lang Lang, <a href="/wiki/1982%EB%85%84" title="1982년">1982년</a> <a href="/wiki/6%EC%9B%94_14%EC%9D%BC" title="6월 14일">6월 14일</a> ~ )은 <a href="/wiki/%EC%A4%91%EA%B5%AD" title="중국">중국</a> <a href="/wiki/%EC%84%A0%EC%96%91_%EC%8B%9C" title="선양 시">선양 시</a>에서 태어난 중국의 <a href="/wiki/%ED%94%BC%EC%95%84%EB%85%B8_%EC%97%B0%EC%A3%BC%EC%9E%90" title="피아노 연주자">피아노 연주자</a>이다. <a href="/wiki/%EB%A7%8C%EC%A3%BC%EC%A1%B1" title="만주족">만주족</a> 출신이다.</p>
<h2><span class="mw-headline" id=".EC.83.9D.EC.95.A0">생애</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%9E%91%EB%9E%91&amp;action=edit&amp;section=1" title="부분 편집: 생애">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p><a class="mw-redirect" href="/wiki/%EC%96%BC%ED%9B%84" title="얼후">얼후</a> 연주자인 아버지와 공장 노무자인 어머니 사이에서 태어난 랑랑은 3세 때부터 센양 음악학교에서 피아노를 배우기 시작하여, 5세 때 센양 콩쿠르에서 우승하였다. 9세 때 <a class="new" href="/w/index.php?title=%EB%B2%A0%EC%9D%B4%EC%A7%95_%EC%A4%91%EC%95%99_%EC%9D%8C%EC%95%85%EC%9B%90&amp;action=edit&amp;redlink=1" title="베이징 중앙 음악원 (없는 문서)">베이징 중앙 음악원</a>에 입학하였고, 이후 미국 <a href="/wiki/%EC%BB%A4%ED%8B%B0%EC%8A%A4_%EC%9D%8C%EC%95%85%EC%9B%90" title="커티스 음악원">커티스 음악원</a>에서 <a class="mw-redirect" href="/wiki/%EA%B2%8C%EB%A6%AC_%EA%B7%B8%EB%9E%98%ED%94%84%EB%A8%BC" title="게리 그래프먼">게리 그래프먼</a>을 사사아였다.</p>
<p>이후 그는 <a class="mw-disambig" href="/wiki/%EB%B2%A0%EC%9D%B4%EC%A7%95_%EC%98%AC%EB%A6%BC%ED%94%BD" title="베이징 올림픽">베이징 올림픽</a> 개막식과 <a class="mw-redirect" href="/wiki/%EA%B4%91%EC%A0%80%EC%9A%B0_%EC%95%84%EC%8B%9C%EC%95%88_%EA%B2%8C%EC%9E%84" title="광저우 아시안 게임">광저우 아시안 게임</a> 개막식에서 피아노 연주를 선보였다.</p>
