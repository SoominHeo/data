<p><b>랑의 베르트라다</b> 또는 <b>레온의 베트트라다</b>(Bertrada of Laon, <a href="/wiki/720%EB%85%84" title="720년">720년</a> ~ <a href="/wiki/783%EB%85%84" title="783년">783년</a>) 또는 <b>왕발 베르타</b>(Bertha Broadfoot)는 <a href="/wiki/%ED%94%84%EB%9E%91%ED%81%AC_%EC%99%95%EA%B5%AD" title="프랑크 왕국">프랑크 왕국</a>의 귀족으로 <a class="mw-disambig" href="/wiki/%EB%A0%88%EC%98%A8" title="레온">레온</a>의 여백작이다. <a href="/wiki/%EC%B9%B4%EB%A1%A4%EB%A7%81%EA%B1%B0_%EC%99%95%EC%A1%B0" title="카롤링거 왕조">카롤링거 왕조</a> 출신 <a href="/wiki/%ED%94%BC%ED%95%80_3%EC%84%B8" title="피핀 3세">피핀 3세</a>의 왕후이자 <a class="mw-redirect" href="/wiki/%EC%83%A4%EB%A5%BC%EB%A7%88%EB%89%B4" title="샤를마뉴">샤를마뉴</a>, <a href="/wiki/%EC%B9%B4%EB%A5%BC%EB%A1%9C%EB%A7%8C_2%EC%84%B8" title="카를로만 2세">카를로만 2세</a>, <a class="new" href="/w/index.php?title=%EB%B2%A0%EB%A5%B4%ED%83%80_4%EC%84%B8&amp;action=edit&amp;redlink=1" title="베르타 4세 (없는 문서)">베르타</a> 남매의 어머니이다. <b>베르타</b>로도 부른다. 별칭은 <b>왕발 베르타</b>(Bertha of the Big Foot).</p>
<p><a class="new" href="/w/index.php?title=%EB%A9%94%EB%A1%9C%EB%B9%99%EA%B1%B0_%EC%99%95%EA%B0%80&amp;action=edit&amp;redlink=1" title="메로빙거 왕가 (없는 문서)">메로빙거 왕가</a>의 외손으로, <a class="new" href="/w/index.php?title=%EB%A0%88%EC%98%A8_%EB%B0%B1%EC%9E%91&amp;action=edit&amp;redlink=1" title="레온 백작 (없는 문서)">레온 백작</a> <a class="new" href="/w/index.php?title=%ED%95%98%EB%A6%AC%EB%B2%A0%EB%A5%B4%ED%8A%B8&amp;action=edit&amp;redlink=1" title="하리베르트 (없는 문서)">하리베르트</a>와 베르타(Bertha)의 딸이며 레온 백작가의 상속녀였다. 할머니 <a class="new" href="/w/index.php?title=%EB%B2%A0%EB%A5%B4%ED%8A%B8%EB%9D%BC%EB%8B%A4&amp;action=edit&amp;redlink=1" title="베르트라다 (없는 문서)">베르타</a>(Bertha)는 <a href="/wiki/%EB%A9%94%EB%A1%9C%EB%B9%99%EA%B1%B0_%EC%99%95%EC%A1%B0" title="메로빙거 왕조">메로빙거 왕조</a> 출신 공주로 <a class="mw-redirect" href="/wiki/%ED%85%8C%EC%98%A4%EB%8D%B0%EB%A6%AC%ED%9E%88_3%EC%84%B8" title="테오데리히 3세">테오데리히 3세</a>와 <a class="new" href="/w/index.php?title=%EC%B9%B4%EB%A1%A4%EB%A7%81%EA%B1%B0_%EA%B0%80&amp;action=edit&amp;redlink=1" title="카롤링거 가 (없는 문서)">카롤링거 가</a> 출신 <a class="new" href="/w/index.php?title=%ED%81%B4%EB%A1%9C%ED%8B%B8%EB%93%9C_(%EC%B9%B4%EB%A1%A4%EB%A7%81%EA%B1%B0)&amp;action=edit&amp;redlink=1" title="클로틸드 (카롤링거) (없는 문서)">클로틸드</a>의 딸이며 외삼촌인 레온 백작 <a class="mw-redirect mw-disambig" href="/wiki/%EB%A7%88%EB%A5%B4%ED%8B%B4" title="마르틴">마르틴</a>과 결혼했다. 그의 시신은 생드니 수도원의 1층 성소에 안치되었다. 남편 <a href="/wiki/%ED%94%BC%ED%95%80_3%EC%84%B8" title="피핀 3세">피핀 3세</a>의 왼편에 안치되었고, 위에는 <a href="/wiki/%EB%A3%A8%EC%9D%B4_3%EC%84%B8" title="루이 3세">루이 3세</a>와 <a href="/wiki/%EC%83%A4%EB%A5%BC%EB%A1%9C%EB%A7%9D" title="샤를로망">샤를로망</a><sup class="reference" id="cite_ref-1"><a href="#cite_note-1">[1]</a></sup>의 시신이 안치되었다.</p>
