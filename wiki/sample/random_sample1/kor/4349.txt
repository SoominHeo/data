<p><b>래클런 매쿼리</b>(Lachlan Macquarie, <a href="/wiki/1762%EB%85%84" title="1762년">1762년</a> <a href="/wiki/1%EC%9B%94_31%EC%9D%BC" title="1월 31일">1월 31일</a> <a href="/wiki/%EC%8A%A4%EC%BD%94%ED%8B%80%EB%9E%9C%EB%93%9C" title="스코틀랜드">스코틀랜드</a> ~ <a href="/wiki/1824%EB%85%84" title="1824년">1824년</a> <a href="/wiki/7%EC%9B%94_1%EC%9D%BC" title="7월 1일">7월 1일</a> <a href="/wiki/%EB%9F%B0%EB%8D%98" title="런던">런던</a>)는 <a href="/wiki/%EC%8A%A4%EC%BD%94%ED%8B%80%EB%9E%9C%EB%93%9C" title="스코틀랜드">스코틀랜드</a> 출신의 영국군 장교로 <a href="/wiki/%EC%98%A4%EC%8A%A4%ED%8A%B8%EB%A0%88%EC%9D%BC%EB%A6%AC%EC%95%84" title="오스트레일리아">오스트레일리아</a> <a class="mw-redirect" href="/wiki/%EB%89%B4%EC%82%AC%EC%9A%B0%EC%8A%A4%EC%9B%A8%EC%9D%BC%EC%A6%88_%EC%A3%BC" title="뉴사우스웨일즈 주">뉴사우스웨일즈 주</a>의 초대 총독(1810년 ~ 1821년)을 역임하였다.</p>
<p>그는 오스트레일리아가 유배 식민지에서 자유로운 식민지에 사회 변화에 중요한 역할을 했다. 따라서 초기 19세기 오스트레일리아 사회의 형성에 중요한 역할을 한 인물로 평가된다.<sup class="reference" id="cite_ref-1"><a href="#cite_note-1">[1]</a></sup><sup class="reference" id="cite_ref-2"><a href="#cite_note-2">[2]</a></sup> 스코틀랜드에 있는 그의 무덤에 비문은 "오스트레일리아(호주)의 아버지"로 설명하고 있다.<sup class="reference" id="cite_ref-3"><a href="#cite_note-3">[3]</a></sup></p>

<h2><span class="mw-headline" id=".EC.83.9D.EC.95.A0">생애</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%9E%98%ED%81%B4%EB%9F%B0_%EB%A7%A4%EC%BF%BC%EB%A6%AC&amp;action=edit&amp;section=1" title="부분 편집: 생애">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>래클런 매쿼리는 <a href="/wiki/%EC%8A%A4%EC%BD%94%ED%8B%80%EB%9E%9C%EB%93%9C" title="스코틀랜드">스코틀랜드</a> 서해안의 이너 헤브라이즈 군도의 멀 섬의 해안에서 멀리 떨어진 울바 섬에서 태어났다. 그의 부친이나 매쿼리의 생가에 대해서는 알려진 바가 거의 없다. 그의 모친은 멀 섬에 성을 가지고 있는 대지주의 딸이었다.<sup class="reference" id="cite_ref-4"><a href="#cite_note-4">[4]</a></sup> 그는 14살의 나이로 이 섬을 떠나게 된다.<sup class="reference" id="cite_ref-Keay_5-0"><a href="#cite_note-Keay-5">[5]</a></sup> 그 집안의 전통대로 ‘에딘버러 왕립 고등학교’에 입학했었다면, 그가 군대에 자원한 나이와 동일하기 때문에 매우 짧은 시기였을 것으로 추측된다.<sup class="reference" id="cite_ref-Ellis.2C_M.H..2C_1952.2C_p._4_6-0"><a href="#cite_note-Ellis.2C_M.H..2C_1952.2C_p._4-6">[6]</a></sup></p>
<p>매쿼리는 1776년 제84보병연대에 입대를 하게 된다. 그 부대를 따라 <a href="/wiki/1777%EB%85%84" title="1777년">1777년</a>에는 <a href="/wiki/%EB%AF%B8%EA%B5%AD" title="미국">미국</a>으로 건너가 <a href="/wiki/%EB%AF%B8%EA%B5%AD_%EB%8F%85%EB%A6%BD_%EC%A0%84%EC%9F%81" title="미국 독립 전쟁">미국 독립 전쟁</a>에 참전하게 된다. 신병으로 그가 참가한 전투는 뉴캐슬 재인 전투였다. 이 전투는 영국의 상선으로 미국 사략선에 승리한 최초의 전투였다. 처음에는 <a href="/wiki/%EB%85%B8%EB%B0%94%EC%8A%A4%EC%BD%94%EC%83%A4_%EC%A3%BC" title="노바스코샤 주">노바스코샤 주</a> <a href="/wiki/%ED%95%BC%EB%A6%AC%ED%8C%A9%EC%8A%A4_(%EB%85%B8%EB%B0%94%EC%8A%A4%EC%BD%94%EC%83%A4_%EC%A3%BC)" title="핼리팩스 (노바스코샤 주)">핼리팩스</a> 에 주둔했으며, 도착 5개월 후에 소위로 임관했다. 1781년에 〈71 보병 연대〉로 전출을 가서, <a href="/wiki/%EB%89%B4%EC%9A%95" title="뉴욕">뉴욕</a>과 <a class="mw-disambig" href="/wiki/%EC%B0%B0%EC%8A%A4%ED%84%B4" title="찰스턴">찰스턴</a>, <a class="mw-redirect" href="/wiki/%EC%9E%90%EB%A7%88%EC%9D%B4%EC%B9%B4" title="자마이카">자마이카</a>에서 복무를 하였다.<sup class="reference" id="cite_ref-Ellis.2C_M.H..2C_1952.2C_p._4_6-1"><a href="#cite_note-Ellis.2C_M.H..2C_1952.2C_p._4-6">[6]</a></sup> 1784년 스코틀랜드로 돌아와 절반의 급여를 받는 중위로 복무를 하게 된다. 그 후 <a href="/wiki/%EC%9D%B8%EB%8F%84" title="인도">인도</a>와 <a href="/wiki/%EC%9D%B4%EC%A7%91%ED%8A%B8" title="이집트">이집트</a>에서 장교로 일을 하게 된다. <a href="/wiki/1793%EB%85%84" title="1793년">1793년</a> 1월 <a href="/wiki/%EC%9D%B8%EB%8F%84" title="인도">인도</a>의 <a class="mw-redirect" href="/wiki/%EB%B4%84%EB%B2%A0%EC%9D%B4" title="봄베이">봄베이</a> 랏지 No1(오늘날의 Grand Lodge No. 139)에서 매쿼리는 <a href="/wiki/%ED%94%84%EB%A6%AC%EB%A9%94%EC%9D%B4%EC%8A%A8" title="프리메이슨">프리메이슨</a>이 되었다. <a href="/wiki/1789%EB%85%84" title="1789년">1789년</a> 대위로 승진하였으며, 2년 후인 <a href="/wiki/1801%EB%85%84" title="1801년">1801년</a> 소령으로 승진을 하였다. 그리고 <a href="/wiki/1805%EB%85%84" title="1805년">1805년</a>에는 73 보병 연대를 지휘하는 중령이 되었다.</p>
<p><a href="/wiki/1793%EB%85%84" title="1793년">1793년</a>에 그는 안티과 사법부 장관의 딸인 제인 자비스(Jane Jarvis)와 결혼을 하였다. 그러나 3년 후 그녀는 <a href="/wiki/%EA%B2%B0%ED%95%B5" title="결핵">결핵</a>으로 사망을 하게 된다.</p>
<h2><span class="mw-headline" id=".EB.89.B4.EC.82.AC.EC.9A.B0.EC.8A.A4.EC.9B.A8.EC.9D.BC.EC.A6.88_.EC.A3.BC.EC.A7.80.EC.82.AC">뉴사우스웨일즈 주지사</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%9E%98%ED%81%B4%EB%9F%B0_%EB%A7%A4%EC%BF%BC%EB%A6%AC&amp;action=edit&amp;section=2" title="부분 편집: 뉴사우스웨일즈 주지사">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p><a href="/wiki/1807%EB%85%84" title="1807년">1807년</a> 11월에 매쿼리의 사촌인 엘리자베스 헨리에타 캠프벨과 두 번째 결혼식을 올린다. 1809년 4월에 매쿼리는 뉴사우스웨일즈 주지사로 임명을 받게 된다.</p>
