<p><b>랜디 듀에인 커투어</b>(Randy Duane Couture, <a href="/wiki/1963%EB%85%84" title="1963년">1963년</a> <a href="/wiki/6%EC%9B%94_22%EC%9D%BC" title="6월 22일">6월 22일</a> 출생)는 <a class="new" href="/w/index.php?title=%EA%B7%B8%EB%A0%88%EC%BD%94%EB%A1%9C%EB%A7%8C_%EB%A0%88%EC%8A%AC%EB%A7%81&amp;action=edit&amp;redlink=1" title="그레코로만 레슬링 (없는 문서)">그레코로만 레슬러</a>이자 <a href="/wiki/%EC%A2%85%ED%95%A9%EA%B2%A9%ED%88%AC%EA%B8%B0" title="종합격투기">종합격투기</a> 선수로, <a href="/wiki/%EC%96%BC%ED%8B%B0%EB%B0%8B_%ED%8C%8C%EC%9D%B4%ED%8C%85_%EC%B1%94%ED%94%BC%EC%96%B8%EC%8B%AD" title="얼티밋 파이팅 챔피언십">얼티밋 파이팅 챔피언십</a>의 헤비급 챔피언이자 UFC 명예의 전당 헌액자이다. 그는 UFC에서 헤비급과 라이트 헤비급의 챔피언 벨트를 모두 차지해본 유일한 선수이자 UFC 역사상 유일하게 다섯 차례 챔피언에 올랐다.</p>
<p>커투어는 한 때 <a href="/wiki/%EC%98%A4%EB%A6%AC%EA%B1%B4_%EC%A3%BC" title="오리건 주">오리건 주</a> <a class="mw-redirect" href="/wiki/%EA%B7%B8%EB%A0%88%EC%85%A4" title="그레셤">그레셤</a>에서 격투가 훈련소인 <a class="new" href="/w/index.php?title=%ED%8C%80_%ED%80%98%EC%8A%A4%ED%8A%B8&amp;action=edit&amp;redlink=1" title="팀 퀘스트 (없는 문서)">팀 퀘스트</a>를 이끌었다. <a href="/wiki/2005%EB%85%84" title="2005년">2005년</a>에는 <a href="/wiki/%EB%84%A4%EB%B0%94%EB%8B%A4_%EC%A3%BC" title="네바다 주">네바다 주</a>의 <a href="/wiki/%EB%9D%BC%EC%8A%A4%EB%B2%A0%EC%9D%B4%EA%B1%B0%EC%8A%A4" title="라스베이거스">라스베이거스</a>로 이주했다. 그는 팀 퀘스트를 나와서 라스베이거스와 <a href="/wiki/%EC%9B%8C%EC%8B%B1%ED%84%B4_%EC%A3%BC" title="워싱턴 주">워싱턴 주</a> <a href="/wiki/%EB%B0%B4%EC%BF%A0%EB%B2%84" title="밴쿠버">밴쿠버</a>, <a href="/wiki/%ED%94%8C%EB%A1%9C%EB%A6%AC%EB%8B%A4_%EC%A3%BC" title="플로리다 주">플로리다 주</a>의 <a href="/wiki/%EC%9B%B0%EB%A7%81%ED%84%B4" title="웰링턴">웰링턴</a>에 익스트림 커투어라는 체육관을 열었고, 현재 그의 라스베이거스 체육관에서 훈련하고 있다.</p>
<p>클린치 상태에서의 끈질긴 공격과 함께 종합격투가로서 그의 격투 스타일을 잘 나타내는 것은 그가 선호하는 그라운드 상태에서의 파운딩이다. 그는 자신의 레슬링 능력을 활용하여 상대를 매트에 메치고, 상위 포지션을 점유한 뒤 효과적으로 상대를 타격하는 전술을 쓴다. 커투어는 레슬링 이외에도 <a class="mw-redirect" href="/wiki/%EB%AC%B4%EC%97%90_%ED%83%80%EC%9D%B4" title="무에 타이">무에 타이</a>와 <a href="/wiki/%EB%B8%8C%EB%9D%BC%EC%A7%88_%EC%9C%A0%EC%88%A0" title="브라질 유술">브라질 유술</a>을 수련했고, 두 명의 상대에게서 쵸크를 통해 승리를 거둔 바 있다.</p>
