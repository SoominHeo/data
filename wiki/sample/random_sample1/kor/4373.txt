<p><b>러브 퍼레이드 압사 참사</b>는 <a href="/wiki/2010%EB%85%84" title="2010년">2010년</a> <a href="/wiki/7%EC%9B%94_24%EC%9D%BC" title="7월 24일">7월 24일</a> 오후 5시경 <a href="/wiki/%EB%8F%85%EC%9D%BC" title="독일">독일</a> <a href="/wiki/%EB%92%A4%EC%8A%A4%EB%B6%80%EB%A5%B4%ED%81%AC" title="뒤스부르크">뒤스부르크</a>에서 열린 음악 축제 <a href="/wiki/%EB%9F%AC%EB%B8%8C_%ED%8D%BC%EB%A0%88%EC%9D%B4%EB%93%9C" title="러브 퍼레이드">러브 퍼레이드</a> 도중 일어난 압사 사고이다. 이 사고로 21명이 숨지고 342명이 부상당하는 참사가 일어났다.<sup class="reference" id="cite_ref-1"><a href="#cite_note-1">[1]</a></sup></p>
<h2><span class="mw-headline" id=".EA.B0.9C.EC.9A.94">개요</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%9F%AC%EB%B8%8C_%ED%8D%BC%EB%A0%88%EC%9D%B4%EB%93%9C_%EC%95%95%EC%82%AC_%EC%B0%B8%EC%82%AC&amp;action=edit&amp;section=1" title="부분 편집: 개요">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>사고는 당일 오후 5시경 축제장으로 들어가는 유일한 출입구인, 카를-레어 거리에서 발생하였다.<sup class="reference" id="cite_ref-2"><a href="#cite_note-2">[2]</a></sup> 러브 퍼레이드는 뒤스부르크의 폐역을 개조해서 만든 축제장을 사용하였는데, 카를-레어 거리는 축제장이 위치한 고속도로와 철로 사이를 지나가기 위하여 터널로 이어져 있다. 오후 4시경이 지나면서 경찰의 통제 속에서도 수 십만명의 사람들이 이 터널을 통하여 축제장으로 들어가려 했으며, 양 터널에서 들어오는 방문객이 축제로 들어가는 중앙 삼거리에서 만나는 구조로 이루어져 있었기 때문에 참사가 불가피했다는 지적도 있다. 대부분의 희생자들은 이 삼거리 벽면에 위치한 좁은 계단을 통해 탈출하려다 압사 사고로 숨졌으며 터널에서도 참사가 발생하였다.<sup class="reference" id="cite_ref-3"><a href="#cite_note-3">[3]</a></sup></p>
<p>사건 직후, 축제 주최측은 갑작스러운 축제 중단으로 참가자들이 다시 출입구로 몰리는 것을 방지 하기 위하여 타 참가자에게 이를 알리지 않고 축제를 강행하였으며, 이번 사고로 앞으로의 축제 개최를 포기함을 선언하였다. 뒤스부르크 시당국과 경찰, 축제 주취측은 현재 책임 공방 중에 있다.</p>
<p>이 사고로 독일인이 14명, <a href="/wiki/%EC%8A%A4%ED%8E%98%EC%9D%B8" title="스페인">스페인</a>인이 2명, <a href="/wiki/%EB%84%A4%EB%8D%9C%EB%9E%80%EB%93%9C" title="네덜란드">네덜란드</a>와 <a class="mw-redirect" href="/wiki/%ED%98%B8%EC%A3%BC" title="호주">호주</a>, <a href="/wiki/%EC%A4%91%EA%B5%AD" title="중국">중국</a>, <a href="/wiki/%EC%9D%B4%ED%83%88%EB%A6%AC%EC%95%84" title="이탈리아">이탈리아</a>, <a href="/wiki/%EB%B3%B4%EC%8A%A4%EB%8B%88%EC%95%84_%ED%97%A4%EB%A5%B4%EC%B2%B4%EA%B3%A0%EB%B9%84%EB%82%98" title="보스니아 헤르체고비나">보스니아 헤르체고비나</a> 각 나라에서 온 5명의 참가자들이 숨졌다. 희생자중 여성이 13명 남성이 8명이다.</p>
