<p><b>2011년 월드 시리즈</b>(2011 World Series)는 제 107회 <a href="/wiki/%EB%A9%94%EC%9D%B4%EC%A0%80_%EB%A6%AC%EA%B7%B8_%EB%B2%A0%EC%9D%B4%EC%8A%A4%EB%B3%BC" title="메이저 리그 베이스볼">메이저 리그 베이스볼</a> <a href="/wiki/%EC%9B%94%EB%93%9C_%EC%8B%9C%EB%A6%AC%EC%A6%88" title="월드 시리즈">월드 시리즈</a>다. 시리즈 일정은 <a href="/wiki/10%EC%9B%94_19%EC%9D%BC" title="10월 19일">10월 19일</a> 시작하여 <a href="/wiki/10%EC%9B%94_28%EC%9D%BC" title="10월 28일">10월 28일</a>까지 진행됐다. <a href="/wiki/%EB%82%B4%EC%85%94%EB%84%90_%EB%A6%AC%EA%B7%B8_(%EC%95%BC%EA%B5%AC)" title="내셔널 리그 (야구)">내셔널 리그</a> 우승팀 <a href="/wiki/%EC%84%B8%EC%9D%B8%ED%8A%B8%EB%A3%A8%EC%9D%B4%EC%8A%A4_%EC%B9%B4%EB%94%94%EB%84%90%EC%8A%A4" title="세인트루이스 카디널스">세인트루이스 카디널스</a>와 <a href="/wiki/%EC%95%84%EB%A9%94%EB%A6%AC%EC%B9%B8_%EB%A6%AC%EA%B7%B8" title="아메리칸 리그">아메리칸 리그</a> 우승팀인 <a href="/wiki/%ED%85%8D%EC%82%AC%EC%8A%A4_%EB%A0%88%EC%9D%B8%EC%A0%80%EC%8A%A4" title="텍사스 레인저스">텍사스 레인저스</a>가 맞붙었다. 총 7전 4선승제로 치루어진 월드시리즈는 내셔널 리그 우승 팀 <b>세인트루이스 카디널스</b>가 아메리칸 리그 우승팀 텍사스를 4승 3패로 누르며 <a href="/wiki/2006%EB%85%84_%EC%9B%94%EB%93%9C_%EC%8B%9C%EB%A6%AC%EC%A6%88" title="2006년 월드 시리즈">2006년</a> 이후 처음이자 통산 11번째 월드시리즈 우승에 성공했다.</p>

<p>세인트루이스 카디널스는 <a class="new" href="/w/index.php?title=2011%EB%85%84_%EB%82%B4%EC%85%94%EB%84%90_%EB%A6%AC%EA%B7%B8_%EB%94%94%EB%B9%84%EC%A0%84_%EC%8B%9C%EB%A6%AC%EC%A6%88&amp;action=edit&amp;redlink=1" title="2011년 내셔널 리그 디비전 시리즈 (없는 문서)">내셔널 리그 디비전 시리즈</a>에서 <a href="/wiki/%ED%95%84%EB%9D%BC%EB%8D%B8%ED%94%BC%EC%95%84_%ED%95%84%EB%A6%AC%EC%8A%A4" title="필라델피아 필리스">필라델피아 필리스</a>와 경기를 치렀으며 3승 2패로 <a class="new" href="/w/index.php?title=2011%EB%85%84_%EB%82%B4%EC%85%94%EB%84%90_%EB%A6%AC%EA%B7%B8_%EC%B1%94%ED%94%BC%EC%96%B8%EC%8B%AD_%EC%8B%9C%EB%A6%AC%EC%A6%88&amp;action=edit&amp;redlink=1" title="2011년 내셔널 리그 챔피언십 시리즈 (없는 문서)">내셔널 리그 챔피언십 시리즈</a>에 진출, <a href="/wiki/%EB%B0%80%EC%9B%8C%ED%82%A4_%EB%B8%8C%EB%A3%A8%EC%96%B4%EC%8A%A4" title="밀워키 브루어스">밀워키 브루어스</a>와 경기를 치러 4승 2패로 월드 시리즈에 진출했다. 이로써 세인트루이스는 2006년 월드시리즈 우승 이후 5년 만에 다시 월드 시리즈 진출에 성공하게 된다.</p>
<h3><span class="mw-headline" id=".ED.85.8D.EC.82.AC.EC.8A.A4_.EB.A0.88.EC.9D.B8.EC.A0.80.EC.8A.A4">텍사스 레인저스</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=2011%EB%85%84_%EC%9B%94%EB%93%9C_%EC%8B%9C%EB%A6%AC%EC%A6%88&amp;action=edit&amp;section=3" title="부분 편집: 텍사스 레인저스">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>텍사스 레인저스는 <a class="new" href="/w/index.php?title=2011%EB%85%84_%EC%95%84%EB%A9%94%EB%A6%AC%EC%B9%B8_%EB%A6%AC%EA%B7%B8_%EB%94%94%EB%B9%84%EC%A0%84_%EC%8B%9C%EB%A6%AC%EC%A6%88&amp;action=edit&amp;redlink=1" title="2011년 아메리칸 리그 디비전 시리즈 (없는 문서)">아메리칸 리그 디비전 시리즈</a>에서 <a href="/wiki/%ED%83%AC%ED%8C%8C%EB%B2%A0%EC%9D%B4_%EB%A0%88%EC%9D%B4%EC%8A%A4" title="탬파베이 레이스">탬파베이 레이스</a>와 경기를 치렀으며 3승 1패로 <a class="new" href="/w/index.php?title=2011%EB%85%84_%EC%95%84%EB%A9%94%EB%A6%AC%EC%B9%B8_%EB%A6%AC%EA%B7%B8_%EC%B1%94%ED%94%BC%EC%96%B8%EC%8B%AD_%EC%8B%9C%EB%A6%AC%EC%A6%88&amp;action=edit&amp;redlink=1" title="2011년 아메리칸 리그 챔피언십 시리즈 (없는 문서)">아메리칸 리그 챔피언십 시리즈</a>에 진출, <a href="/wiki/%EB%94%94%ED%8A%B8%EB%A1%9C%EC%9D%B4%ED%8A%B8_%ED%83%80%EC%9D%B4%EA%B1%B0%EC%8A%A4" title="디트로이트 타이거스">디트로이트 타이거스</a>와 경기를 치러 4승 2패로 월드 시리즈에 진출했다. 이는 텍사스 레인저스 구단 역사상 두 번째 월드 시리즈 진출이다. ALCS MVP는 <a class="mw-redirect" href="/wiki/%EB%84%AC%EC%8A%A8_%ED%81%AC%EB%A3%A8%EC%A6%88" title="넬슨 크루즈">넬슨 크루즈</a></p>
<h2><span class="mw-headline" id=".EC.8B.9C.EB.A6.AC.EC.A6.88_.EC.9A.94.EC.95.BD">시리즈 요약</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=2011%EB%85%84_%EC%9B%94%EB%93%9C_%EC%8B%9C%EB%A6%AC%EC%A6%88&amp;action=edit&amp;section=4" title="부분 편집: 시리즈 요약">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<ul>
<li>2011 월드 시리즈 (4승 3패) : <a href="/wiki/%EC%84%B8%EC%9D%B8%ED%8A%B8%EB%A3%A8%EC%9D%B4%EC%8A%A4_%EC%B9%B4%EB%94%94%EB%84%90%EC%8A%A4" title="세인트루이스 카디널스">세인트루이스 카디널스</a> (내셔널 리그)가 <a href="/wiki/%ED%85%8D%EC%82%AC%EC%8A%A4_%EB%A0%88%EC%9D%B8%EC%A0%80%EC%8A%A4" title="텍사스 레인저스">텍사스 레인저스</a> (아메리칸 리그)에게 승리하였다. 월드 시리즈 MVP는 <a href="/wiki/%EC%84%B8%EC%9D%B8%ED%8A%B8%EB%A3%A8%EC%9D%B4%EC%8A%A4_%EC%B9%B4%EB%94%94%EB%84%90%EC%8A%A4" title="세인트루이스 카디널스">세인트루이스 카디널스</a>의 <a href="/wiki/3%EB%A3%A8%EC%88%98" title="3루수">3루수</a>인 <a class="new" href="/w/index.php?title=%EB%8D%B0%EC%9D%B4%EB%B9%97_%ED%94%84%EB%A6%AC%EC%A6%88&amp;action=edit&amp;redlink=1" title="데이빗 프리즈 (없는 문서)">데이빗 프리즈</a>가 수상하였다.</li>
</ul>
