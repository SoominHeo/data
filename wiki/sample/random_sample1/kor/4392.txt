<h2><span class="mw-headline" id=".EC.83.9D.EC.95.A0">생애</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%9F%AC%ED%82%A4_%EC%8A%A4%ED%83%80_(%EC%98%A4%EC%8A%A4%ED%8A%B8%EB%A0%88%EC%9D%BC%EB%A6%AC%EC%95%84%EC%9D%98_%EA%B0%80%EC%88%98)&amp;action=edit&amp;section=1" title="부분 편집: 생애">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<h3><span class="mw-headline" id=".EC.9D.8C.EC.95.85.EA.B0.80_.ED.99.9C.EB.8F.99">음악가 활동</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%9F%AC%ED%82%A4_%EC%8A%A4%ED%83%80_(%EC%98%A4%EC%8A%A4%ED%8A%B8%EB%A0%88%EC%9D%BC%EB%A6%AC%EC%95%84%EC%9D%98_%EA%B0%80%EC%88%98)&amp;action=edit&amp;section=2" title="부분 편집: 음악가 활동">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p><a href="/wiki/1957%EB%85%84" title="1957년">1957년</a> <a href="/wiki/%EB%A1%9C%ED%81%B0%EB%A1%A4" title="로큰롤">로큰롤</a> <a class="mw-redirect" href="/wiki/%EC%9D%8C%EC%95%85_%EA%B7%B8%EB%A3%B9" title="음악 그룹">음악 밴드</a> "<a class="extiw" href="https://en.wikipedia.org/wiki/The_Hepparays" title="en:The Hepparays">더 헤퍼레이스</a>(The Hepparays)"의 <a href="/wiki/%EB%B3%B4%EC%BB%AC" title="보컬">보컬리스트</a> 겸 <a href="/wiki/%EA%B8%B0%ED%83%80_%EC%97%B0%EC%A3%BC%EC%9E%90" title="기타 연주자">기타리스트</a> 첫 <a href="/wiki/%EB%8D%B0%EB%B7%94" title="데뷔">데뷔</a>를 하였고 <a href="/wiki/1962%EB%85%84" title="1962년">1962년</a> <a class="mw-disambig" href="/wiki/%EC%86%94%EB%A1%9C" title="솔로">솔로</a> <a href="/wiki/%EA%B0%80%EC%88%98" title="가수">가수</a> 데뷔하였다.</p>
<p>그는 <a href="/wiki/1969%EB%85%84" title="1969년">1969년</a> <a href="/wiki/%EB%AF%B8%EA%B5%AD" title="미국">미국</a> 공연 당시 <a href="/wiki/%EC%BB%A8%ED%8A%B8%EB%A6%AC_%EC%9D%8C%EC%95%85" title="컨트리 음악">컨트리 음악</a> <a href="/wiki/%EC%8B%B1%EC%96%B4%EC%86%A1%EB%9D%BC%EC%9D%B4%ED%84%B0" title="싱어송라이터">싱어송라이터</a> <a href="/wiki/%ED%96%89%ED%81%AC_%EC%8A%A4%EB%85%B8" title="행크 스노">행크 스노</a>(Hank Snow)와 서로 악수를 나누는 영광을 얻기도 하였다.</p>
<h2><span class="mw-headline" id=".EC.9C.A0.EB.AA.85.EC.84.B8">유명세</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%9F%AC%ED%82%A4_%EC%8A%A4%ED%83%80_(%EC%98%A4%EC%8A%A4%ED%8A%B8%EB%A0%88%EC%9D%BC%EB%A6%AC%EC%95%84%EC%9D%98_%EA%B0%80%EC%88%98)&amp;action=edit&amp;section=3" title="부분 편집: 유명세">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>그는 보통적으로 우리나라 국내에서는 <a href="/wiki/%EC%98%A4%EC%8A%A4%ED%8A%B8%EB%A0%88%EC%9D%BC%EB%A6%AC%EC%95%84" title="오스트레일리아">오스트레일리아</a> <a href="/wiki/%EC%BB%A8%ED%8A%B8%EB%A6%AC_%EC%9D%8C%EC%95%85" title="컨트리 음악">컨트리 음악</a> <a href="/wiki/%EA%B0%80%EC%88%98" title="가수">가수</a> <a href="/wiki/%EC%A0%9C%ED%94%84_%EB%A7%A5" title="제프 맥">저프 맥</a>(Geoff Mack)의 《I've been everywhere》라는 곡을 <a href="/wiki/%EB%A6%AC%EB%A9%94%EC%9D%B4%ED%81%AC" title="리메이크">리메이크</a>한 가수로 널리 잘 알려져 있다.</p>
