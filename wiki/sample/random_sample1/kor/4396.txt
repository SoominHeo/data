<p><b>럭스 페테르 다비드</b>(<span style="font-size: smaller;"><a href="/wiki/%ED%97%9D%EA%B0%80%EB%A6%AC%EC%96%B4" title="헝가리어">헝가리어</a>:</span> <span lang="hu" xml:lang="hu">Lax Péter Dávid</span> <small><a href="/wiki/%EC%9C%84%ED%82%A4%EB%B0%B1%EA%B3%BC:IPA" title="위키백과:IPA">IPA</a>: </small><span class="IPA" title="IPA 발음">[lɒks ˈpeːtɛr ˈdaːvid]</span>, <span style="font-size: smaller;"><a href="/wiki/%EC%98%81%EC%96%B4" title="영어">영어</a>:</span> <span lang="en" xml:lang="en">Peter David Lax</span> <small>피터 데이비드 랙스<sup>[<a href="/wiki/%EC%9C%84%ED%82%A4%EB%B0%B1%EA%B3%BC:%EC%98%81%EC%96%B4%EC%9D%98_%ED%95%9C%EA%B8%80_%ED%91%9C%EA%B8%B0" title="위키백과:영어의 한글 표기">*</a>]</sup></small>, 1926년 5월 1일 –)는 <a href="/wiki/%ED%97%9D%EA%B0%80%EB%A6%AC" title="헝가리">헝가리</a> 태생 <a href="/wiki/%EB%AF%B8%EA%B5%AD" title="미국">미국</a> <a href="/wiki/%EC%88%98%ED%95%99%EC%9E%90" title="수학자">수학자</a>다. <a href="/wiki/%EC%A0%81%EB%B6%84%EA%B0%80%EB%8A%A5%EA%B3%84" title="적분가능계">적분가능계</a>와 <a href="/wiki/%EC%9C%A0%EC%B2%B4%EC%97%AD%ED%95%99" title="유체역학">유체역학</a>의 연구에 공헌하였다. <a href="/wiki/%EC%9A%B8%ED%94%84%EC%83%81" title="울프상">울프상</a>과 <a class="mw-redirect" href="/wiki/%EC%95%84%EB%B2%A8_%EC%83%81" title="아벨 상">아벨 상</a>을 수상하였다.</p>
<h2><span class="mw-headline" id=".EC.83.9D.EC.95.A0">생애</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%9F%AD%EC%8A%A4_%ED%8E%98%ED%85%8C%EB%A5%B4&amp;action=edit&amp;section=1" title="부분 편집: 생애">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p><a href="/wiki/%EB%B6%80%EB%8B%A4%ED%8E%98%EC%8A%A4%ED%8A%B8" title="부다페스트">부다페스트</a>에서 <a href="/wiki/%EC%9C%A0%EB%8C%80%EC%9D%B8" title="유대인">유대인</a> 가정에서 태어났다. 아버지 럭스 헨리(<span style="font-size: smaller;"><a href="/wiki/%ED%97%9D%EA%B0%80%EB%A6%AC%EC%96%B4" title="헝가리어">헝가리어</a>:</span> <span lang="hu" xml:lang="hu">Lax Henry</span>)와 어머니 코른필트 클라라(<span style="font-size: smaller;"><a href="/wiki/%ED%97%9D%EA%B0%80%EB%A6%AC%EC%96%B4" title="헝가리어">헝가리어</a>:</span> <span lang="hu" xml:lang="hu">Kornfield Klara</span>)는 둘 다 외과의사였다.</p>
<p>럭스 가족은 <a class="mw-redirect" href="/wiki/%EB%82%98%EC%B9%98_%EB%8B%B9" title="나치 당">나치 당</a>의 <a href="/wiki/%EB%B0%98%EC%9C%A0%EB%8C%80%EC%A3%BC%EC%9D%98" title="반유대주의">반유대주의</a>를 피해 1941년 11월 15일 <a href="/wiki/%EB%A6%AC%EC%8A%A4%EB%B3%B8" title="리스본">리스본</a>을 통해 <a href="/wiki/%EB%AF%B8%EA%B5%AD" title="미국">미국</a> <a href="/wiki/%EB%89%B4%EC%9A%95" title="뉴욕">뉴욕</a>으로 이민하였다. 뉴욕의 영재 학교인 <a class="new" href="/w/index.php?title=%EC%8A%A4%ED%83%80%EC%9D%B4%EB%B2%84%EC%84%A0%ED%8A%B8_%EA%B3%A0%EB%93%B1%ED%95%99%EA%B5%90&amp;action=edit&amp;redlink=1" title="스타이버선트 고등학교 (없는 문서)">스타이버선트 고등학교</a>(Stuyvesant High School)에서 럭스는 <a href="/wiki/%EC%A1%B4_%ED%8F%B0_%EB%85%B8%EC%9D%B4%EB%A7%8C" title="존 폰 노이만">존 폰 노이만</a>과 <a href="/wiki/%EC%97%90%EB%A5%B4%EB%90%98%EC%8B%9C_%ED%8C%94" title="에르되시 팔">에르되시 팔</a>, <a href="/wiki/%EB%A6%AC%ED%95%98%EB%A5%B4%ED%8A%B8_%EC%BF%A0%EB%9E%80%ED%8A%B8" title="리하르트 쿠란트">리하르트 쿠란트</a>를 알게 되었다. 이들은 모두 훗날 저명한 수학자가 되었다.</p>
<p>고등학교를 17세에 졸업하였고, <a href="/wiki/%EB%89%B4%EC%9A%95_%EB%8C%80%ED%95%99%EA%B5%90" title="뉴욕 대학교">뉴욕 대학교</a>에 입학하였다. 18세가 되자 럭스는 징병되어 <a href="/wiki/%EB%A7%A8%ED%95%B4%ED%8A%BC_%EA%B3%84%ED%9A%8D" title="맨해튼 계획">맨해튼 계획</a>에 기여하였다. 전후 1947년에 뉴욕 대학교에서 학사 학위를 취득하였다. 1948년에 혼인하였고, 1949년에 뉴욕 대학교에서 박사 학위를 취득하였다.</p>
<p>현재 <a href="/wiki/%EB%89%B4%EC%9A%95_%EB%8C%80%ED%95%99%EA%B5%90" title="뉴욕 대학교">뉴욕 대학교</a> <a href="/wiki/%EC%BF%A0%EB%9E%80%ED%8A%B8_%EC%88%98%ED%95%99%EC%97%B0%EA%B5%AC%EC%86%8C" title="쿠란트 수학연구소">쿠란트 수학연구소</a> 수학과 교수로 있다.</p>
