2014년 11월 홀본의 챈서리 레인 역에 정차된 트럭에서 내려지고 있는 《런던 이브닝 스탠더드》.</div>

<p>《<b>런던 이브닝 스탠더드</b>》 (<i>London Evening Standard</i>)는 <a href="/wiki/%EB%9F%B0%EB%8D%98" title="런던">런던</a>에서 월요일에서 금요일까지 <a href="/wiki/%ED%83%80%EB%B8%94%EB%A1%9C%EC%9D%B4%EB%93%9C" title="타블로이드">타블로이드</a>판으로 발행되는 지역 <a class="new" href="/w/index.php?title=%EB%AC%B4%EB%A3%8C_%EC%9D%BC%EA%B0%84%EC%A7%80&amp;action=edit&amp;redlink=1" title="무료 일간지 (없는 문서)">무료 일간지</a>이다. 2009년 5월까지는 간단하게 《<b>이브닝 스탠더드</b>》 (<i>Evening Standard</i>}였다.</p>
<p>《런던 이브닝 스탠더드》는 런던과 인근 지역을 대상으로 국내외 뉴스와 <a href="/wiki/%EC%8B%9C%ED%8B%B0%EC%98%A4%EB%B8%8C%EB%9F%B0%EB%8D%98" title="시티오브런던">시티오브런던</a>의 금융을 보도하는 우세적인 지역 석간신문이다. 2009년 10월, 《런던 이브닝 스탠더드》는, 사업 계획 변경의 일부로 발행부수를 두 배로 늘리면서, 180년간의 <a class="new" href="/w/index.php?title=%EC%8B%A0%EB%AC%B8_%EB%B0%9C%ED%96%89%EB%B6%80%EC%88%98&amp;action=edit&amp;redlink=1" title="신문 발행부수 (없는 문서)">유료발행</a> 역사를 마무리하고 무가지가 되었다.<sup class="reference" id="cite_ref-eandp09_3-1"><a href="#cite_note-eandp09-3">[3]</a></sup></p>
<h2><span class="mw-headline" id=".EC.97.AD.EC.82.AC">역사</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%9F%B0%EB%8D%98_%EC%9D%B4%EB%B8%8C%EB%8B%9D_%EC%8A%A4%ED%83%A0%EB%8D%94%EB%93%9C&amp;action=edit&amp;section=1" title="부분 편집: 역사">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>《런던 이브닝 스탠더드》는 법정변호사였던 <a class="new" href="/w/index.php?title=%EC%8A%A4%ED%83%A0%EB%A6%AC_%EB%A6%AC%EC%A6%88_%EA%B8%B8%ED%8D%BC%EB%93%9C&amp;action=edit&amp;redlink=1" title="스탠리 리즈 길퍼드 (없는 문서)">스탠리 리즈 길퍼드</a>가 《스탠더드》 (<i>Standard</i>)라는 이름으로 1827년 5월 21일 창간했다.<sup class="reference" id="cite_ref-4"><a href="#cite_note-4">[4]</a></sup> 초창기 소유주는 찰스 벌드윈 (Charles Baldwin)이었다. <a class="new" href="/w/index.php?title=%EC%A0%9C%EC%9E%84%EC%8A%A4_%EC%A1%B4%EC%8A%A4%ED%84%B4_(%EC%B6%9C%ED%8C%90%EC%97%85%EC%9E%90)&amp;action=edit&amp;redlink=1" title="제임스 존스턴 (출판업자) (없는 문서)">제임스 존스턴</a>의 소유 하에 《스탠더드》는 1857년 6월 29일부터 조간신문이 되었다. 《이브닝 스탠더드》는 1859년 6월 11일부터 발행되었다. 《스탠더드》는 상세한 외국 소식, 특히 <a class="mw-disambig" href="/wiki/%EB%82%A8%EB%B6%81_%EC%A0%84%EC%9F%81" title="남북 전쟁">남북 전쟁</a> (1861~1865), <a class="mw-redirect" href="/wiki/%EC%98%A4%EC%8A%A4%ED%8A%B8%EB%A6%AC%EC%95%84-%ED%94%84%EB%A1%9C%EC%9D%B4%EC%84%BC_%EC%A0%84%EC%9F%81" title="오스트리아-프로이센 전쟁">오스트리아-프로이센 전쟁</a> (1866), <a class="mw-redirect" href="/wiki/%ED%94%84%EB%9E%91%EC%8A%A4-%ED%94%84%EB%A1%9C%EC%9D%B4%EC%84%BC_%EC%A0%84%EC%9F%81" title="프랑스-프로이센 전쟁">프랑스-프로이센 전쟁</a> (1870)의 사건 보도로 명성을 얻었으며, 이 모두가 발행부수를 늘리는 데 기여했다.<sup class="reference" id="cite_ref-5"><a href="#cite_note-5">[5]</a></sup></p>
<p>《런던 이브닝 스탠더드》는 매년 열리는 <a class="new" href="/w/index.php?title=%EC%9D%B4%EB%B8%8C%EB%8B%9D_%EC%8A%A4%ED%83%A0%EB%8D%94%EB%93%9C_%EC%83%81&amp;action=edit&amp;redlink=1" title="이브닝 스탠더드 상 (없는 문서)">이브닝 스탠더드 시어터 상</a>을 1950년대부터 후원하고 있다.<sup class="정리_필요 noprint">[<a href="/wiki/%EC%9C%84%ED%82%A4%EB%B0%B1%EA%B3%BC:%EC%B6%9C%EC%B2%98_%EB%B0%9D%ED%9E%88%EA%B8%B0" title="위키백과:출처 밝히기">출처 필요</a>]</sup> 또한 역시 매년 열리는 <a class="new" href="/w/index.php?title=%EC%9D%B4%EB%B8%8C%EB%8B%9D_%EC%8A%A4%ED%83%A0%EB%8D%94%EB%93%9C_%ED%8E%8D_%EC%98%A4%EB%B8%8C_%EB%8D%94_%EC%9D%B4%EC%96%B4&amp;action=edit&amp;redlink=1" title="이브닝 스탠더드 펍 오브 더 이어 (없는 문서)">이브닝 스탠더드 펍 오브 더 이어</a> (2007년 폐지)와 <a class="new" href="/w/index.php?title=%EC%9D%B4%EB%B8%8C%EB%8B%9D_%EC%8A%A4%ED%83%A0%EB%8D%94%EB%93%9C_%EB%B8%8C%EB%A6%AC%ED%8B%B0%EC%8B%9C_%ED%95%84%EB%A6%84_%EC%83%81&amp;action=edit&amp;redlink=1" title="이브닝 스탠더드 브리티시 필름 상 (없는 문서)">이브닝 스탠더드 브리티시 필름 상</a>을 1970년대부터 수여하고 있다.<sup class="정리_필요 noprint">[<a href="/wiki/%EC%9C%84%ED%82%A4%EB%B0%B1%EA%B3%BC:%EC%B6%9C%EC%B2%98_%EB%B0%9D%ED%9E%88%EA%B8%B0" title="위키백과:출처 밝히기">출처 필요</a>]</sup></p>
