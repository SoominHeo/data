<p><b>레이니르 "레네" 람베르튀스 판 더 케르크호프</b> (<span style="font-size: smaller;"><a href="/wiki/%EB%84%A4%EB%8D%9C%EB%9E%80%EB%93%9C%EC%96%B4" title="네덜란드어">네덜란드어</a>:</span> <span lang="nl" xml:lang="nl">Reinier "René" Lambertus van de Kerkhof</span>, <a href="/wiki/1951%EB%85%84" title="1951년">1951년</a> <a href="/wiki/9%EC%9B%94_16%EC%9D%BC" title="9월 16일">9월 16일</a> ~ )는 <a href="/wiki/%EB%84%A4%EB%8D%9C%EB%9E%80%EB%93%9C" title="네덜란드">네덜란드</a>의 전직 <a href="/wiki/%EC%B6%95%EA%B5%AC" title="축구">축구</a> 선수로, 포지션은 <a href="/wiki/%EA%B3%B5%EA%B2%A9%EC%88%98" title="공격수">공격수</a>(오른쪽 윙어)였으며 <a href="/wiki/%EB%B9%8C%EB%A6%AC_%ED%8C%90_%EB%8D%94_%EC%BC%80%EB%A5%B4%ED%81%AC%ED%98%B8%ED%94%84" title="빌리 판 더 케르크호프">빌리 판 더 케르크호프</a>의 쌍둥이 형이다.</p>
<p><a href="/wiki/1973%EB%85%84" title="1973년">1973년</a>부터 <a href="/wiki/1982%EB%85%84" title="1982년">1982년</a>까지 <a href="/wiki/%EB%84%A4%EB%8D%9C%EB%9E%80%EB%93%9C_%EC%B6%95%EA%B5%AC_%EA%B5%AD%EA%B0%80%EB%8C%80%ED%91%9C%ED%8C%80" title="네덜란드 축구 국가대표팀">네덜란드 대표팀</a>에서 뛰는 동안 47경기에 출전하면서 5득점을 기록했으며 그의 쌍둥이 동생인 빌리와 함께 네덜란드 대표팀에서 뛰었다. <a href="/wiki/1974%EB%85%84_FIFA_%EC%9B%94%EB%93%9C%EC%BB%B5" title="1974년 FIFA 월드컵">1974년 FIFA 월드컵</a>과 <a href="/wiki/1978%EB%85%84_FIFA_%EC%9B%94%EB%93%9C%EC%BB%B5" title="1978년 FIFA 월드컵">1978년 FIFA 월드컵</a>에서 네덜란드의 FIFA 월드컵 준우승을 이끌었다. 1978년 FIFA 월드컵 결승전 경기가 시작되기 직전 부상으로 인해 자신의 팔에 감은 붕대 때문에 네덜란드의 상대 팀인 <a href="/wiki/%EC%95%84%EB%A5%B4%ED%97%A8%ED%8B%B0%EB%82%98_%EC%B6%95%EA%B5%AC_%EA%B5%AD%EA%B0%80%EB%8C%80%ED%91%9C%ED%8C%80" title="아르헨티나 축구 국가대표팀">아르헨티나</a>가 강력히 항의하는 바람에 경기 시작이 지연되는 사태가 일어나기도 했지만 다행히 경기 시작과 함께 붕대를 풀면서 사태는 일단락될 수 있었다.</p>
<p><a href="/wiki/2004%EB%85%84" title="2004년">2004년</a> 3월 <a href="/wiki/%ED%8E%A0%EB%A0%88" title="펠레">펠레</a>가 선정한 현존하는 최고의 축구 선수 목록인 <a href="/wiki/FIFA_100" title="FIFA 100">FIFA 100</a>에 선정되었다.</p>
