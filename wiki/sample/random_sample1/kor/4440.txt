<p><b>레번 헬름</b>(Levon Helm, <a href="/wiki/1940%EB%85%84" title="1940년">1940년</a> <a href="/wiki/5%EC%9B%94_26%EC%9D%BC" title="5월 26일">5월 26일</a> ~ <a href="/wiki/2012%EB%85%84" title="2012년">2012년</a> <a href="/wiki/4%EC%9B%94_19%EC%9D%BC" title="4월 19일">4월 19일</a>)은 <a href="/wiki/%EB%AF%B8%EA%B5%AD" title="미국">미국</a>의 <a href="/wiki/%EA%B0%80%EC%88%98" title="가수">가수</a>이자 <a href="/wiki/%EB%B0%B0%EC%9A%B0" title="배우">배우</a>이다. <a href="/wiki/%EB%8D%94_%EB%B0%B4%EB%93%9C" title="더 밴드">더 밴드</a>의 <a href="/wiki/%EB%93%9C%EB%9F%BC_%EC%97%B0%EC%A3%BC%EC%9E%90" title="드럼 연주자">드럼 연주자</a>이자 <a href="/wiki/%EB%A6%AC%EB%93%9C_%EB%B3%B4%EC%BB%AC" title="리드 보컬">리드 보컬</a>로 유명하다.</p>
<p>다양한 악기를 연주했던 <a href="/wiki/%EB%A9%80%ED%8B%B0%ED%94%8C%EB%A0%88%EC%9D%B4%EC%96%B4_(%EC%9D%8C%EC%95%85)" title="멀티플레이어 (음악)">멀티플레이어</a>로, <a href="/wiki/%EC%BB%A8%ED%8A%B8%EB%A6%AC_%EC%9D%8C%EC%95%85" title="컨트리 음악">컨트리</a>풍의 깊은 음색과 창의적인 드럼 연주로 사랑을 받았다.</p>
<p><a href="/wiki/1977%EB%85%84" title="1977년">1977년</a> 더 밴드가 잠시 해체된 뒤로는 솔로로서도 활동했고, <a href="/wiki/1980%EB%85%84" title="1980년">1980년</a>작 《American Son》이 호평을 받았다.</p>
<p><a href="/wiki/1998%EB%85%84" title="1998년">1998년</a> <a class="new" href="/w/index.php?title=%EC%9D%B8%ED%9B%84%EC%95%94&amp;action=edit&amp;redlink=1" title="인후암 (없는 문서)">인후암</a> 진단을 받아 노래를 하기 어려워졌지만, 재활에 성공해 <a href="/wiki/2007%EB%85%84" title="2007년">2007년</a>에 솔로 음반 《Dirt Farmer》를 발표했다. 이 음반으로 레번 헬름은 <a href="/wiki/2008%EB%85%84" title="2008년">2008년</a> <a href="/wiki/%EA%B7%B8%EB%9E%98%EB%AF%B8%EC%83%81" title="그래미상">그래미</a> 최우수 전통 포크 음반상을 받았다.</p>
<p>《<a href="/wiki/%EB%A1%A4%EB%A7%81_%EC%8A%A4%ED%86%A4" title="롤링 스톤">롤링 스톤</a>》지는 레번 헬름을 '가장 위대한 가수 100명' 중 91번째로 선정했다.<sup class="reference" id="cite_ref-1"><a href="#cite_note-1">[1]</a></sup></p>
<p>영화 배우로 활동하기도 했는데, 《<a class="new" href="/w/index.php?title=%EA%B4%91%EB%B6%80%EC%9D%98_%EB%94%B8&amp;action=edit&amp;redlink=1" title="광부의 딸 (없는 문서)">광부의 딸</a>》, 《<a class="new" href="/w/index.php?title=%ED%95%84%EC%82%AC%EC%9D%98_%EB%8F%84%EC%A0%84&amp;action=edit&amp;redlink=1" title="필사의 도전 (없는 문서)">필사의 도전</a>》, 《<a href="/wiki/%EB%8D%94%EB%B8%94_%ED%83%80%EA%B2%9F" title="더블 타겟">더블 타겟</a>》 등에 출연했다.</p>
