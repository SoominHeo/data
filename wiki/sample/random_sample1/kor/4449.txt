<p><b>레시틴</b>(Lecithin)은 <a href="/wiki/%EC%9D%B8%EC%82%B0" title="인산">인산</a>, <a class="new" href="/w/index.php?title=%EC%BD%9C%EB%A6%B0_(%EB%B3%B5%ED%95%A9%EC%B2%B4)&amp;action=edit&amp;redlink=1" title="콜린 (복합체) (없는 문서)">콜린</a>, <a href="/wiki/%EC%A7%80%EB%B0%A9%EC%82%B0" title="지방산">지방산</a>, <a href="/wiki/%EA%B8%80%EB%A6%AC%EC%84%B8%EB%A1%A4" title="글리세롤">글리세롤</a>, <a class="new" href="/w/index.php?title=%EB%8B%B9%EC%A7%80%EC%A7%88&amp;action=edit&amp;redlink=1" title="당지질 (없는 문서)">당지질</a>, <a href="/wiki/%ED%8A%B8%EB%9D%BC%EC%9D%B4%EA%B8%80%EB%A6%AC%EC%84%B8%EB%9D%BC%EC%9D%B4%EB%93%9C" title="트라이글리세라이드">트라이글리세라이드</a>, <a href="/wiki/%EC%9D%B8%EC%A7%80%EC%A7%88" title="인지질">인지질</a>로 구성된 동식물 조직에서 발생하는 황갈빛의 지방 물질군을 두루 가리키는 용어이다. (예: <a class="new" href="/w/index.php?title=%ED%8F%AC%EC%8A%A4%ED%8C%8C%ED%8B%B0%EB%94%9C%EC%BD%9C%EB%A6%B0&amp;action=edit&amp;redlink=1" title="포스파티딜콜린 (없는 문서)">포스파티딜콜린</a>, <a class="new" href="/w/index.php?title=%ED%8F%AC%EC%8A%A4%ED%8C%8C%ED%8B%B0%EB%94%9C%EC%97%90%ED%83%84%EC%98%AC%EC%95%84%EB%AF%BC&amp;action=edit&amp;redlink=1" title="포스파티딜에탄올아민 (없는 문서)">포스파티딜에탄올아민</a>, <a class="new" href="/w/index.php?title=%ED%8F%AC%EC%8A%A4%ED%8C%8C%ED%8B%B0%EB%94%9C%EC%9D%B4%EB%85%B8%EC%8B%9C%ED%86%A8&amp;action=edit&amp;redlink=1" title="포스파티딜이노시톨 (없는 문서)">포스파티딜이노시톨</a>)</p>
<p>레시틴은 1846년 프랑스의 화학자 <a class="new" href="/w/index.php?title=%EC%8B%9C%EC%96%B4%EB%8F%84%EC%96%B4_%EB%8B%88%EC%BD%9C%EB%9D%BC%EC%8A%A4_%EA%B3%A0%EB%B8%94%EB%A6%AC&amp;action=edit&amp;redlink=1" title="시어도어 니콜라스 고블리 (없는 문서)">시어도어 고블리</a>가 분리해냈다.<sup class="reference" id="cite_ref-1"><a href="#cite_note-1">[1]</a></sup> 1850년 그는 포스파티딜콜린(phosphatidylcholine)을 레시틴(léchithine)으로 명명했다.<sup class="reference" id="cite_ref-2"><a href="#cite_note-2">[2]</a></sup> 고블리는 원래 레시틴을 <a href="/wiki/%EB%85%B8%EB%A5%B8%EC%9E%90" title="노른자">달걀 노른자</a>에서 분리하였는데(레키토스(λέκιθος)는 노른자를 가리키는 고대 그리스어 낱말) 1874년에 포스파티딜콜린의 완전한 화학식을 확립했다.<sup class="reference" id="cite_ref-3"><a href="#cite_note-3">[3]</a></sup></p>
<p>레시틴은 화학적(<a href="/wiki/%ED%97%A5%EC%84%B8%EC%9D%B8" title="헥세인">헥세인</a>, <a href="/wiki/%EC%97%90%ED%83%84%EC%98%AC" title="에탄올">에탄올</a>, <a href="/wiki/%EC%95%84%EC%84%B8%ED%86%A4" title="아세톤">아세톤</a>, <a href="/wiki/%EB%B2%A4%EC%A7%84" title="벤진">벤진</a> 등)으로나 기계적으로 쉽게 분리가 가능하다. 일반적으로 콩, 달걀, 젖, 해산물, <a href="/wiki/%EC%9C%A0%EC%B1%84" title="유채">유채</a>, 목화씨, 해바라기와 같은 곳으로부터 구할 수 있다. 수용성은 낮은 편이지만 뛰어난 유화체이다.</p>
