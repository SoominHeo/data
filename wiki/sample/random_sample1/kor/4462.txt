<p><b>레오폴 3세</b>(Leopold III, <a href="/wiki/1901%EB%85%84" title="1901년">1901년</a> <a href="/wiki/11%EC%9B%94_3%EC%9D%BC" title="11월 3일">11월 3일</a> ~ <a href="/wiki/1983%EB%85%84" title="1983년">1983년</a> <a href="/wiki/9%EC%9B%94_25%EC%9D%BC" title="9월 25일">9월 25일</a>)는 <a href="/wiki/%EB%B2%A8%EA%B8%B0%EC%97%90" title="벨기에">벨기에</a>의 국왕(1934~1951)이며, <a class="mw-redirect" href="/wiki/%EC%A0%9C2%EC%B0%A8_%EC%84%B8%EA%B3%84%EB%8C%80%EC%A0%84" title="제2차 세계대전">제2차 세계대전</a> 동안에 벨기에군의 최고 사령관으로 활약하였다.</p>
<h2><span class="mw-headline" id=".EC.83.9D.EC.95.A0">생애</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%A0%88%EC%98%A4%ED%8F%B4_3%EC%84%B8&amp;action=edit&amp;section=1" title="부분 편집: 생애">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p><a href="/wiki/%EC%95%8C%EB%B2%A0%EB%A5%B4_1%EC%84%B8_(%EB%B2%A8%EA%B8%B0%EC%97%90)" title="알베르 1세 (벨기에)">알베르 1세</a>와 <a class="new" href="/w/index.php?title=%EB%B0%94%EC%9D%B4%EC%97%90%EB%A5%B8%EC%9D%98_%EC%97%98%EB%A6%AC%EC%9E%90%EB%B2%A0%ED%8A%B8_(1876%EB%85%84)&amp;action=edit&amp;redlink=1" title="바이에른의 엘리자베트 (1876년) (없는 문서)">바이에른의 엘리자베트</a> 사이에서 태어나 <a class="mw-redirect" href="/wiki/%EC%A0%9C1%EC%B0%A8_%EC%84%B8%EA%B3%84%EB%8C%80%EC%A0%84" title="제1차 세계대전">제1차 세계대전</a>에 졸병으로 참전하였다. <a href="/wiki/1926%EB%85%84" title="1926년">1926년</a> <a href="/wiki/11%EC%9B%94_10%EC%9D%BC" title="11월 10일">11월 10일</a> <a href="/wiki/%EC%8A%A4%EC%9B%A8%EB%8D%B4" title="스웨덴">스웨덴</a>의 공주 아스트리드(<a href="/wiki/1935%EB%85%84" title="1935년">1935년</a> 사망)와 결혼하여 슬하 2남 1녀 - <a class="mw-redirect" href="/wiki/%EB%B3%B4%EB%91%90%EC%95%B5_1%EC%84%B8_(%EB%B2%A8%EA%B8%B0%EC%97%90)" title="보두앵 1세 (벨기에)">보두앵</a>과 <a href="/wiki/%EC%95%8C%EB%B2%A0%EB%A5%B4_2%EC%84%B8_(%EB%B2%A8%EA%B8%B0%EC%97%90)" title="알베르 2세 (벨기에)">알베르</a>와 <a href="/wiki/%EB%B2%A8%EA%B8%B0%EC%97%90%EC%9D%98_%EC%A1%B0%EC%A0%9C%ED%95%80%EC%83%A4%EB%A5%BC%EB%A1%9C%ED%8A%B8" title="벨기에의 조제핀샤를로트">조제핀샤를로트</a>를 두었다.</p>
<p><a href="/wiki/1934%EB%85%84" title="1934년">1934년</a> <a href="/wiki/2%EC%9B%94_17%EC%9D%BC" title="2월 17일">2월 17일</a> 부왕의 사망과 함께 왕위에 올랐다. 순조로운 독립적인 외교 정책을 세웠으나, 중립적이 아닌 적으로 <a href="/wiki/%EB%A1%9C%EC%B9%B4%EB%A5%B4%EB%85%B8_%EC%A1%B0%EC%95%BD" title="로카르노 조약">로카르노 조약</a>으로부터 벨기에를 자퇴시켰다. <a href="/wiki/%EC%98%81%EA%B5%AD" title="영국">영국</a>과 <a href="/wiki/%ED%94%84%EB%9E%91%EC%8A%A4" title="프랑스">프랑스</a>의 도움으로 침략에 저항을 결심시키고, <a href="/wiki/%EC%95%88%ED%8A%B8%EB%B2%A0%EB%A5%B4%ED%8E%9C" title="안트베르펜">안트베르펜</a>부터 <a href="/wiki/%EB%82%98%EB%AE%88%EB%A5%B4" title="나뮈르">나뮈르</a>까지 방어선 요새를 세우는 후원을 하였다.</p>
<p><a class="mw-redirect" href="/wiki/%EC%A0%9C2%EC%B0%A8_%EC%84%B8%EA%B3%84%EB%8C%80%EC%A0%84" title="제2차 세계대전">제2차 세계대전</a>이 일어나자, 레오폴은 벨기에군의 최고 사령관을 지내면서 18일 후 <a href="/wiki/%EB%8F%85%EC%9D%BC" title="독일">독일</a>군이 침입했던 <a href="/wiki/1940%EB%85%84" title="1940년">1940년</a> <a href="/wiki/5%EC%9B%94_28%EC%9D%BC" title="5월 28일">5월 28일</a>에 자신의 군대에 항복 명령을 내렸다. 벨기에 정부는 망명하여 <a href="/wiki/%EB%9F%B0%EB%8D%98" title="런던">런던</a> 정부에 가입하는 대신 그의 항복을 거절하였고, 그의 왕위 승계에 분쟁을 위한 창설을 놓았다.</p>
<p>그는 <a href="/wiki/%EB%B8%8C%EB%A4%BC%EC%85%80" title="브뤼셀">브뤼셀</a> 근처에서 독일군의 포로로 잡혔으며, 전쟁이 끝날 때까지 <a href="/wiki/%EC%98%A4%EC%8A%A4%ED%8A%B8%EB%A6%AC%EC%95%84" title="오스트리아">오스트리아</a>에서 감금되어 있었다. <a href="/wiki/1942%EB%85%84" title="1942년">1942년</a> 그가 <a href="/wiki/%EC%95%84%EB%8F%8C%ED%94%84_%ED%9E%88%ED%8B%80%EB%9F%AC" title="아돌프 히틀러">아돌프 히틀러</a>에게 보낸 <a href="/wiki/%ED%8E%B8%EC%A7%80" title="편지">편지</a>는 대략 500,000명의 벨기에 여성들과 어린이들을 국외 추방으로부터 독일의 군수 공장들로 구하는 명예를 얻었다.</p>
<p><a href="/wiki/1941%EB%85%84" title="1941년">1941년</a> <a class="new" href="/w/index.php?title=%EB%A7%88%EB%A6%AC_%EB%A6%B4%EB%A6%AC%EC%95%99_%EB%B0%94%EC%97%98&amp;action=edit&amp;redlink=1" title="마리 릴리앙 바엘 (없는 문서)">마리 릴리앙 바엘</a>과 결혼하여 슬하 1남 2녀 - <a class="new" href="/w/index.php?title=%EB%B2%A8%EA%B8%B0%EC%97%90_%EC%99%95%EC%9E%90_%EC%95%8C%EB%A0%89%EC%83%81%EB%93%9C%EB%A5%B4&amp;action=edit&amp;redlink=1" title="벨기에 왕자 알렉상드르 (없는 문서)">알렉상드르</a>, <a class="new" href="/w/index.php?title=%EB%B2%A8%EA%B8%B0%EC%97%90_%EA%B3%B5%EC%A3%BC_%EB%A7%88%EB%A6%AC%ED%81%AC%EB%A6%AC%EC%8A%A4%ED%8B%B4&amp;action=edit&amp;redlink=1" title="벨기에 공주 마리크리스틴 (없는 문서)">마리크리스틴</a>, <a class="new" href="/w/index.php?title=%EB%B2%A8%EA%B8%B0%EC%97%90_%EA%B3%B5%EC%A3%BC_%EB%A7%88%EB%A6%AC%EC%97%90%EC%8A%A4%EB%A9%94%EB%9E%84%EB%8B%A4&amp;action=edit&amp;redlink=1" title="벨기에 공주 마리에스메랄다 (없는 문서)">마리에스메랄다</a>를 두었다.</p>
<p>그의 형 샤를이 <a href="/wiki/1944%EB%85%84" title="1944년">1944년</a> 섭정에 임명된 후에 레오폴은 <a href="/wiki/1945%EB%85%84" title="1945년">1945년</a>부터 5년동안 <a href="/wiki/%EC%8A%A4%EC%9C%84%EC%8A%A4" title="스위스">스위스</a>에 머물었다. <a href="/wiki/1950%EB%85%84" title="1950년">1950년</a> <a href="/wiki/3%EC%9B%94_12%EC%9D%BC" title="3월 12일">3월 12일</a>에 실시된 <a class="mw-redirect" href="/wiki/%EA%B5%AD%EB%AF%BC_%ED%88%AC%ED%91%9C" title="국민 투표">국민 투표</a>에서 58% 가까이 왕의 귀국을 찬성하였다. 자유당, 사회당, 왈롱당은 그위 왕위를 거절하여, 레오폴 3세는 <a href="/wiki/1951%EB%85%84" title="1951년">1951년</a>에 왕좌에서 물러나 그의 장남 <a class="mw-redirect" href="/wiki/%EB%B2%A8%EA%B8%B0%EC%97%90%EC%9D%98_%EB%B3%B4%EB%91%90%EC%95%B5_1%EC%84%B8" title="벨기에의 보두앵 1세">보두앵 1세</a>가 그의 뒤를 이어 국왕이 된다.</p>
<p><a href="/wiki/1983%EB%85%84" title="1983년">1983년</a> <a href="/wiki/9%EC%9B%94_25%EC%9D%BC" title="9월 25일">9월 25일</a> 81세의 나이로 브뤼셀에서 사망하였다.</p>
