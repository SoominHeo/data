<h2><span class="mw-headline" id=".EC.B4.88.EC.B0.BD.EA.B8.B0">초창기</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%A0%88%EC%98%A8_%EB%B0%95%EC%8A%A4%ED%8A%B8&amp;action=edit&amp;section=1" title="부분 편집: 초창기">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>레온은 현재의 <a href="/wiki/%EB%B2%A8%EB%9D%BC%EB%A3%A8%EC%8A%A4" title="벨라루스">벨라루스</a> <a href="/wiki/%ED%9D%90%EB%A1%9C%EB%93%9C%EB%82%98" title="흐로드나">흐로드나</a> 지방인 로젠베르크에서 중산층 <a class="mw-redirect" href="/wiki/%EC%9C%A0%ED%83%9C%EC%9D%B8" title="유태인">유태인</a> 가정에서 태어났다. <a href="/wiki/%EA%B9%80%EB%82%98%EC%A7%80%EC%9B%80" title="김나지움">김나지움</a>을 졸업한 후, 그는 <a href="/wiki/%EC%83%81%ED%8A%B8%ED%8E%98%ED%85%8C%EB%A5%B4%EB%B6%80%EB%A5%B4%ED%81%AC" title="상트페테르부르크">상트페테르부르크</a> 미술 학교에서 비정규 학생으로 공부하면서, 시간제로 책 <a class="mw-redirect" href="/wiki/%EC%9D%BC%EB%9F%AC%EC%8A%A4%ED%8A%B8%EB%A0%88%EC%9D%B4%ED%84%B0" title="일러스트레이터">일러스트레이터</a>로 일하였다.</p>
<p>1889년 그의 첫 번째 전시회에서 그는 어머니의 할머니의 이름인 Baxter를 기초로 Bakst라는 이름을 사용하였다. 1890년대 초기에 박스트는 <i>수채화가 조합</i>과 함께 작품을 전시하였다. 그는 1893 - 1897년 동안 파리에 살면서, Académie Julian에서 공부하였으며, 자주 <a href="/wiki/%EC%83%81%ED%8A%B8%ED%8E%98%ED%85%8C%EB%A5%B4%EB%B6%80%EB%A5%B4%ED%81%AC" title="상트페테르부르크">상트페테르부르크</a>를 방문하였다. 1890년대 중반 이후, 그는 <a class="mw-redirect" href="/wiki/%EC%84%B8%EB%A5%B4%EA%B2%8C%EC%9D%B4_%EB%94%94%EC%95%84%EA%B8%B8%EB%A0%88%ED%94%84" title="세르게이 디아길레프">세르게이 디아길레프</a>와 <a class="new" href="/w/index.php?title=%EC%95%8C%EB%A0%89%EC%82%B0%EB%93%9C%EB%A5%B4_%EB%B8%8C%EB%88%84%EC%95%84&amp;action=edit&amp;redlink=1" title="알렉산드르 브누아 (없는 문서)">알렉산드르 브누아</a>가 구축한 작가와 화가들의 집단의 구성원중 하나가 되었다. 이는 <a class="new" href="/w/index.php?title=Mir_Iskusstva&amp;action=edit&amp;redlink=1" title="Mir Iskusstva (없는 문서)">Mir Iskusstva</a> 미술 운동이 되었다.</p>
<p>1899년 박스트는 <a class="mw-redirect" href="/wiki/%EB%94%94%EC%95%84%EA%B8%B8%EB%A0%88%ED%94%84" title="디아길레프">디아길레프</a>와 함께 영향력있는 정기 간행물인《<a class="new" href="/w/index.php?title=%EB%AF%B8%EC%88%A0_%EC%84%B8%EA%B3%84&amp;action=edit&amp;redlink=1" title="미술 세계 (없는 문서)">미술 세계</a>》를 공동으로 설립하였다. 그의 그 잡지를 위한 그래픽 디자인은 박스트의 명성을 가져다 주었다.</p>
<h3><span class="mw-headline" id=".EB.96.A0.EC.98.A4.EB.A5.B4.EB.8A.94_.EB.AA.85.EC.84.B1">떠오르는 명성</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%A0%88%EC%98%A8_%EB%B0%95%EC%8A%A4%ED%8A%B8&amp;action=edit&amp;section=2" title="부분 편집: 떠오르는 명성">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>그는 <a class="new" href="/w/index.php?title=Filipp_Malyavin&amp;action=edit&amp;redlink=1" title="Filipp Malyavin (없는 문서)">Filipp Malyavin</a> (1899년), <a class="new" href="/w/index.php?title=Vasily_Rozanov&amp;action=edit&amp;redlink=1" title="Vasily Rozanov (없는 문서)">Vasily Rozanov</a> (1901년), <a class="new" href="/w/index.php?title=Andrei_Bely&amp;action=edit&amp;redlink=1" title="Andrei Bely (없는 문서)">Andrei Bely</a> (1905년), <a class="new" href="/w/index.php?title=Zinaida_Gippius&amp;action=edit&amp;redlink=1" title="Zinaida Gippius (없는 문서)">Zinaida Gippius</a> (1906년)을 위한 프로덕션 초상화뿐만 아니라, 외부에서 의뢰 받은 초상화도 그렸다. 박스트는 Grand Duke Vladimir Alexandrovich의 아이들의 미술 교사로도 일했다. 1902년 <a class="new" href="/w/index.php?title=%EB%8B%88%EC%BD%9C%EB%9D%BC%EC%9D%B4_%ED%99%A9%EC%A0%9C_2%EC%84%B8&amp;action=edit&amp;redlink=1" title="니콜라이 황제 2세 (없는 문서)">니콜라이 황제 2세</a>로부터 의뢰를 받아 파리에서 《<a class="new" href="/w/index.php?title=%EB%9F%AC%EC%8B%9C%EC%95%84_%ED%95%AD%ED%95%B4%EC%82%AC%EC%9D%98_%EB%A7%8C%EB%82%A8&amp;action=edit&amp;redlink=1" title="러시아 항해사의 만남 (없는 문서)">러시아 항해사의 만남</a>》이란 그림을 그렸다.</p>
<p>1898년 그는 <a class="mw-redirect" href="/wiki/%EB%94%94%EC%95%84%EA%B8%B8%EB%A0%88%ED%94%84" title="디아길레프">디아길레프</a>가 조직한 "러시아와 핀란드 미술가의 첫 번째 전시회"에서 작품을 전시하였다. <a href="/wiki/%EB%AE%8C%ED%97%A8" title="뮌헨">뮌헨</a>에서 <a href="/wiki/%EB%B6%84%EB%A6%AC%EC%A3%BC%EC%9D%98" title="분리주의">분리주의</a>파와 러시아 예술가들의 연합에서도 작품을 전시하였다.</p>
<p>1905년 <a href="/wiki/%EB%9F%AC%EC%8B%9C%EC%95%84_%ED%98%81%EB%AA%85" title="러시아 혁명">러시아 혁명</a> 기간 동안, 박스트는 잡지, <i>Zhupel, Adskaja Pochta, Satyricon</i>과 미술 잡지인 <i>Apollon</i>을 위해 일했다.</p>
<h3><span class="mw-headline" id=".EB.AC.B4.EB.8C.80_.EB.94.94.EC.9E.90.EC.9D.B8">무대 디자인</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%A0%88%EC%98%A8_%EB%B0%95%EC%8A%A4%ED%8A%B8&amp;action=edit&amp;section=3" title="부분 편집: 무대 디자인">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>1900년대 10년 동안 박스트는 대부분 <a class="new" href="/w/index.php?title=%EB%AC%B4%EB%8C%80_%EB%94%94%EC%9E%90%EC%9D%B4%EB%84%88&amp;action=edit&amp;redlink=1" title="무대 디자이너 (없는 문서)">무대 디자이너</a>로 활동하였다. 박스트는 <a class="mw-redirect" href="/wiki/%EA%B7%B8%EB%A6%AC%EC%8A%A4_%EB%B9%84%EA%B7%B9" title="그리스 비극">그리스 비극</a>을 위한 무대를 디자인했고, 1908년에 <a class="mw-redirect" href="/wiki/%EB%94%94%EC%95%84%EA%B8%B8%EB%A0%88%ED%94%84" title="디아길레프">디아길레프</a>와 <a class="mw-redirect" href="/wiki/%EB%B0%9C%EB%A0%88_%EB%A3%A8%EC%8A%A4" title="발레 루스">발레 루스</a>를 위한 <a class="new" href="/w/index.php?title=%EB%AC%B4%EB%8C%80_%ED%99%94%EA%B0%80&amp;action=edit&amp;redlink=1" title="무대 화가 (없는 문서)">무대 화가</a>로 이름을 얻었다. 《<a class="mw-redirect" href="/wiki/%ED%81%B4%EB%A0%88%EC%98%A4%ED%8C%8C%ED%8A%B8%EB%9D%BC" title="클레오파트라">클레오파트라</a>》(1909년), 《<a class="mw-redirect" href="/wiki/%EC%84%B8%ED%97%A4%EB%9D%BC%EC%9E%90%EB%8D%B0" title="세헤라자데">세헤라자데</a>》(1901년), 《<a href="/wiki/%EC%82%AC%EC%9C%A1%EC%A0%9C" title="사육제">사육제</a>》(1910년), 《<a class="mw-redirect" href="/wiki/%EB%82%98%EB%A5%B4%EC%8B%9C%EC%8A%A4" title="나르시스">나르시스</a>》(1911년), 《<a class="new" href="/w/index.php?title=%EB%8B%A4%ED%94%84%EB%84%A4%EC%99%80_%ED%81%B4%EB%A1%9C%EC%97%90&amp;action=edit&amp;redlink=1" title="다프네와 클로에 (없는 문서)">다프네와 클로에</a>》(1912년) 등이 있다. 유태인은 영구적으로 러시아에서 <a class="new" href="/w/index.php?title=%EC%9C%A0%ED%83%9C%EC%9D%B8_%EA%B1%B0%EC%A3%BC%EC%A7%80&amp;action=edit&amp;redlink=1" title="유태인 거주지 (없는 문서)">유태인 거주지</a> 바깥에서는 살 권리가 없었기 때문에 박스트는 이때 유럽에서 살았다.</p>
<p><a class="mw-redirect" href="/wiki/%EC%83%81%ED%8A%B8_%ED%8E%98%ED%85%8C%EB%A5%B4%EB%B6%80%EB%A5%B4%ED%81%AC" title="상트 페테르부르크">상트 페테르부르크</a>에 가서 있는 동안 박스트는 Zvantseva's school에서 학생들을 가르쳤다. <a href="/wiki/%EB%A7%88%EB%A5%B4%ED%81%AC_%EC%83%A4%EA%B0%88" title="마르크 샤갈">마르크 샤갈</a>도 그의 제자 중 한 명이다. 1910년 둘다 파산했다. 박스트는 샤갈에게 파리에 가지 말 것을 충고하였다. 박스트에 따르면 파리로의 이주는 샤갈의 예술과 재정적으로도 위험하기에 아마도 그가 굶어죽을 것이라고 말했다.</p>
<p>1914년 Bakst는 <a class="new" href="/w/index.php?title=%ED%99%A9%EC%8B%A4_%EB%AF%B8%EC%88%A0_%EC%95%84%EC%B9%B4%EB%8D%B0%EB%AF%B8&amp;action=edit&amp;redlink=1" title="황실 미술 아카데미 (없는 문서)">황실 미술 아카데미</a>의 회원으로 선출되었다.</p>
<p>박스트는 1918년 <a class="mw-redirect" href="/wiki/%EB%94%94%EC%95%84%EA%B8%B8%EB%A0%88%ED%94%84" title="디아길레프">디아길레프</a>와 <a class="mw-redirect" href="/wiki/%EB%B0%9C%EB%A0%88_%EB%A3%A8%EC%8A%A4" title="발레 루스">발레 루스</a>의 관계를 끝냈다. 그는 1924년 파리에서 폐 합병증으로 사망하였다.</p>
