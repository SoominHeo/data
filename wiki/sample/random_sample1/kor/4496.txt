<p><b>레이튼 교수 VS 역전재판</b>(<span style="font-size: smaller;"><a href="/wiki/%EC%9D%BC%EB%B3%B8%EC%96%B4" title="일본어">일본어</a>:</span> <span lang="ja"><span lang="ja"><ruby><rb>レイトン教授 VS 逆転裁判</rb> <rp>(</rp><rp>)</rp></ruby></span></span>)은 <a class="mw-redirect" href="/wiki/%EB%A0%88%EB%B2%A8_%ED%8C%8C%EC%9D%B4%EB%B8%8C" title="레벨 파이브">레벨 5</a>와 <a href="/wiki/%EC%BA%A1%EC%BD%A4" title="캡콤">캡콤</a>의 합동 게임 작품으로 <a href="/wiki/%EB%8B%8C%ED%85%90%EB%8F%84_3DS" title="닌텐도 3DS">닌텐도 3DS</a>용 게임이다.</p>
<h2><span class="mw-headline" id=".EA.B0.9C.EC.9A.94">개요</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%A0%88%EC%9D%B4%ED%8A%BC_%EA%B5%90%EC%88%98_VS_%EC%97%AD%EC%A0%84%EC%9E%AC%ED%8C%90&amp;action=edit&amp;section=1" title="부분 편집: 개요">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p><a class="mw-redirect" href="/wiki/%EB%A0%88%EB%B2%A8_%ED%8C%8C%EC%9D%B4%EB%B8%8C" title="레벨 파이브">레벨 5</a>를 대표하는 어드벤처 게임 《<b><a href="/wiki/%EB%A0%88%EC%9D%B4%ED%8A%BC_%EA%B5%90%EC%88%98_%EC%8B%9C%EB%A6%AC%EC%A6%88" title="레이튼 교수 시리즈">레이튼 교수 시리즈</a></b>》와 캡콤을 대표하는 어드벤처 게임《<b><a href="/wiki/%EC%97%AD%EC%A0%84%EC%9E%AC%ED%8C%90" title="역전재판">역전재판 시리즈</a></b>》의 콜라보레이션 작품으로 <a href="/wiki/2010%EB%85%84" title="2010년">2010년</a> <a href="/wiki/10%EC%9B%94_19%EC%9D%BC" title="10월 19일">10월 19일</a> 개최된 LEVEL5 VISION 2010에서 게임 제작을 발표하였다.</p>
<p>시나리오는 《<a href="/wiki/%EC%97%AD%EC%A0%84%EC%9E%AC%ED%8C%90" title="역전재판">역전재판 시리즈</a>》의 아버지라 불리는 <a href="/wiki/%EB%8B%A4%EC%BF%A0%EB%AF%B8_%EC%8A%88" title="다쿠미 슈">다쿠미 슈</a>가 담당했다. 이번 게임은 <a href="/wiki/%EB%A0%88%EC%9D%B4%ED%8A%BC_%EA%B5%90%EC%88%98_%EC%8B%9C%EB%A6%AC%EC%A6%88" title="레이튼 교수 시리즈">레이튼 교수 시리즈</a>에서 흔히 나오는 수수께끼와 <a href="/wiki/%EC%97%AD%EC%A0%84%EC%9E%AC%ED%8C%90" title="역전재판">역전재판 시리즈</a>에서 나오는 모순들이 합쳐져 '마녀재판' 이라는 주제에 <a href="/wiki/%EC%97%AD%EC%A0%84%EC%9E%AC%ED%8C%90" title="역전재판">역전재판 시리즈</a>의 주인공 나루호도 류이치와 파트너 아야사토 마요이, <a href="/wiki/%EB%A0%88%EC%9D%B4%ED%8A%BC_%EA%B5%90%EC%88%98_%EC%8B%9C%EB%A6%AC%EC%A6%88" title="레이튼 교수 시리즈">레이튼 교수 시리즈</a>의 주인공 레이튼 교수와 제자 루크가 등장한다.</p>
