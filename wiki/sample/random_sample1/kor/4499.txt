<p><b>레이펑</b>(뇌봉, <small><a href="/wiki/%EC%A4%91%EA%B5%AD%EC%96%B4" title="중국어">중국어</a></small> <small><a href="/wiki/%EA%B0%84%EC%B2%B4%EC%9E%90" title="간체자">간체</a>:</small> <span lang="zh-hans" xml:lang="zh-hans">雷锋</span>, <small><a href="/wiki/%EC%A0%95%EC%B2%B4%EC%9E%90" title="정체자">정체</a>:</small> <span lang="zh-hant" xml:lang="zh-hant">雷鋒</span>, <small><a href="/wiki/%ED%95%9C%EC%96%B4_%EB%B3%91%EC%9D%8C" title="한어 병음">병음</a>:</small> <font face="Lucida Sans Unicode">Léi Fēng</font>, <a href="/wiki/1940%EB%85%84" title="1940년">1940년</a> <a href="/wiki/12%EC%9B%94_18%EC%9D%BC" title="12월 18일">12월 18일</a> ~ <a href="/wiki/1962%EB%85%84" title="1962년">1962년</a> <a href="/wiki/8%EC%9B%94_15%EC%9D%BC" title="8월 15일">8월 15일</a>)은 <a href="/wiki/%EC%A4%91%EA%B5%AD_%EC%9D%B8%EB%AF%BC%ED%95%B4%EB%B0%A9%EA%B5%B0" title="중국 인민해방군">중국 인민해방군</a>의 모범병사이다. <a href="/wiki/%ED%9B%84%EB%82%9C_%EC%84%B1" title="후난 성">후난 성</a> 창시 출신으로, 아동단과 소년선봉대에 들어가 활동하였으며, <a href="/wiki/1957%EB%85%84" title="1957년">1957년</a>에는 <a class="mw-redirect" href="/wiki/%EC%A4%91%EA%B5%AD%EA%B3%B5%EC%82%B0%EC%A3%BC%EC%9D%98%EC%B2%AD%EB%85%84%EB%8B%A8" title="중국공산주의청년단">중국공산주의청년단</a>에 들어가, 중국 각지의 농장이나 공장에서 작업하는 등 봉사활동을 계속했다. <a href="/wiki/1960%EB%85%84" title="1960년">1960년</a> 인민해방군에 입대, 수송대에 배속되었다. <a href="/wiki/1962%EB%85%84" title="1962년">1962년</a> <a href="/wiki/8%EC%9B%94_15%EC%9D%BC" title="8월 15일">8월 15일</a>, 랴오닝 성 푸순에서 사고로 순직.</p>
<p>사후 <a href="/wiki/%EB%A7%88%EC%98%A4%EC%A9%8C%EB%91%A5" title="마오쩌둥">마오쩌둥</a> 등의 공산당 지도자의 말을 인용한 일기가 발견되었다. 그는 이상적 군인상으로 널리 선전되기 시작해, 1963년 <a href="/wiki/3%EC%9B%94_5%EC%9D%BC" title="3월 5일">3월 5일</a>에는 마오쩌둥이 직접 향뇌봉동지학습(<span lang="zh" xml:lang="zh">向雷鋒同志学習</span>, 레이펑 동지에게 배우라) 운동을 지시하게 이른다. 이 슬로건은 <a href="/wiki/%EB%AC%B8%ED%99%94_%EB%8C%80%ED%98%81%EB%AA%85" title="문화 대혁명">문화 대혁명</a> 중, 각종신문이나 교과서에 수없이 인용되며 우상으로 떠받들어졌다.</p>
<p>그 후로도 오늘날에 이르기까지 정부의 공식 캠페인에 수없이 활용되어, <a href="/wiki/3%EC%9B%94_5%EC%9D%BC" title="3월 5일">3월 5일</a>은 "레이펑에게 배우는 날(Learn from Lei Feng Day)"로 지정돼 학생들이 공원이나 거리를 청소하는 날이 되어 있다. 또한 고향인 창시와 순직지인 푸순에는 레이펑 기념관이 설립되어 있다.</p>
