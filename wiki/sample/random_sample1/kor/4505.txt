<h1 class="firstHeading" id="firstHeading" lang="ko">레일라 요세포비치</h1>
<div class="mw-body-content" id="bodyContent">
<div id="siteSub">위키백과, 우리 모두의 백과사전.</div>
<div id="contentSub"></div>
<div class="mw-jump" id="jump-to-nav">
					이동:					<a href="#mw-head">둘러보기</a>, 					<a href="#p-search">검색</a>
</div>
<div class="mw-content-ltr" dir="ltr" id="mw-content-text" lang="ko"><script>function mfTempOpenSection(id){var block=document.getElementById("mf-section-"+id);block.className+=" open-block";block.previousSibling.className+=" open-block";}</script><p><b>레일라 요세포비치</b>(<span style="font-size: smaller;"><a href="/wiki/%ED%8F%B4%EB%9E%80%EB%93%9C%EC%96%B4" title="폴란드어">폴란드어</a>:</span> <span lang="pl" xml:lang="pl">Leila Josefowicz</span>, <a href="/wiki/1977%EB%85%84" title="1977년">1977년</a> <a href="/wiki/10%EC%9B%94_20%EC%9D%BC" title="10월 20일">10월 20일</a> ~ )는 <a href="/wiki/%EC%BA%90%EB%82%98%EB%8B%A4" title="캐나다">캐나다</a>의 <a href="/wiki/%EB%B0%94%EC%9D%B4%EC%98%AC%EB%A6%B0" title="바이올린">바이올린</a> 연주자이다.</p>
<p>캐나다 <a href="/wiki/%EC%98%A8%ED%83%80%EB%A6%AC%EC%98%A4_%EC%A3%BC" title="온타리오 주">온타리오</a>에서 폴란드계 아버지와 잉글랜드계 어머니 사이에서 태어났다. 어렸을 때 <a href="/wiki/%EC%BA%98%EB%A6%AC%ED%8F%AC%EB%8B%88%EC%95%84_%EC%A3%BC" title="캘리포니아 주">캘리포니아 주</a>로 이사를 가서 3살 반 때 <a class="new" href="/w/index.php?title=%EC%8A%A4%EC%A6%88%ED%82%A4_%EA%B5%90%EC%88%98%EB%B2%95&amp;action=edit&amp;redlink=1" title="스즈키 교수법 (없는 문서)">스즈키 교수법</a>으로 바이올린을 배웠다. 다섯살 때 Idel Low, 여덟살 때 Robert Lipsett에게 바이올린을 배웠다. 13살때 <a href="/wiki/%ED%95%84%EB%9D%BC%EB%8D%B8%ED%94%BC%EC%95%84" title="필라델피아">필라델피아</a>로 옮겨 <a href="/wiki/%EC%BB%A4%ED%8B%B0%EC%8A%A4_%EC%9D%8C%EC%95%85%EC%9B%90" title="커티스 음악원">커티스 음악원</a>을 다녔다.</p>
<p>1994년에 <a href="/wiki/%EC%B9%B4%EB%84%A4%EA%B8%B0_%ED%99%80" title="카네기 홀">카네기 홀</a>에서 <a href="/wiki/%EB%84%A4%EB%B9%8C_%EB%A7%88%EB%A6%AC%EB%84%88" title="네빌 마리너">네빌 마리너</a>와 <a href="/wiki/%EC%84%B8%EC%9D%B8%ED%8A%B8_%EB%A7%88%ED%8B%B4_%EC%9D%B8%EB%8D%94_%ED%95%84%EC%A6%88_%EC%95%84%EC%B9%B4%EB%8D%B0%EB%AF%B8_%EC%8B%A4%EB%82%B4_%EA%B4%80%ED%98%84%EC%95%85%EB%8B%A8" title="세인트 마틴 인더 필즈 아카데미 실내 관현악단">세인트 마틴 인더 필즈 아카데미 실내 관현악단</a>과의 <a href="/wiki/%ED%91%9C%ED%8A%B8%EB%A5%B4_%EC%B0%A8%EC%9D%B4%EC%BD%A5%EC%8A%A4%ED%82%A4" title="표트르 차이콥스키">차이콥스키</a> 협주곡 협연으로 데뷔했다.</p>
