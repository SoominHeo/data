<h1 class="firstHeading" id="firstHeading" lang="ko">레일리 수</h1>
<div class="mw-body-content" id="bodyContent">
<div id="siteSub">위키백과, 우리 모두의 백과사전.</div>
<div id="contentSub"></div>
<div class="mw-jump" id="jump-to-nav">
					이동:					<a href="#mw-head">둘러보기</a>, 					<a href="#p-search">검색</a>
</div>
<div class="mw-content-ltr" dir="ltr" id="mw-content-text" lang="ko"><script>function mfTempOpenSection(id){var block=document.getElementById("mf-section-"+id);block.className+=" open-block";block.previousSibling.className+=" open-block";}</script><p><a class="mw-redirect" href="/wiki/%EC%9C%A0%EC%B2%B4_%EC%97%AD%ED%95%99" title="유체 역학">유체 역학</a>에서, 어떤 유체의 <b>레일리 수</b>는 유체 사이의 열 전달과 관련된 <a class="new" href="/w/index.php?title=%EB%AC%B4%EC%B0%A8%EC%9B%90_%EC%88%98&amp;action=edit&amp;redlink=1" title="무차원 수 (없는 문서)">무차원 수</a>이다. 레일리 수가 유체의 임계값보다 작으면 열은 <a class="mw-redirect" href="/wiki/%EC%97%B4_%EC%A0%84%EB%8F%84" title="열 전도">전도</a>의 형태로 전달되고, 반대로 유체의 임계값보다 크면 <a href="/wiki/%EB%8C%80%EB%A5%98" title="대류">대류</a>의 형태로 전달된다. 레일리 수는 <a class="mw-redirect" href="/wiki/%EA%B7%B8%EB%9E%98%EC%86%8C%ED%94%84_%EC%88%98" title="그래소프 수">그래소프 수</a>와 <a href="/wiki/%ED%94%84%EB%9E%80%ED%8B%80_%EC%88%98" title="프란틀 수">프란틀 수</a>의 곱으로 정의되는데, 그래소프 수는 유체간의 <a href="/wiki/%EB%B6%80%EB%A0%A5" title="부력">부력</a>과 <a href="/wiki/%EA%B4%80%EC%84%B1" title="관성">관성</a> 사이의 관계를 나타내는 수이고, 프란틀 수는 유체의 <a href="/wiki/%EC%A0%90%EC%84%B1" title="점성">점성</a>과 <a href="/wiki/%EC%98%A8%EB%8F%84" title="온도">온도</a> 사이의 관계를 나타내는 수이다.</p>
<p>수직 벽 근처에서의 <a class="new" href="/w/index.php?title=%EC%9E%90%EC%9C%A0_%EB%8C%80%EB%A5%98&amp;action=edit&amp;redlink=1" title="자유 대류 (없는 문서)">자유 대류</a>에 대해, 레일리 수는 다음과 같이 표현된다.</p>

<p>여기서,</p>
<ul>
<li>Ra = 레일리 수</li>
<li>Gr = <a class="mw-redirect" href="/wiki/%EA%B7%B8%EB%9E%98%EC%86%8C%ED%94%84_%EC%88%98" title="그래소프 수">그래소프 수</a></li>

<li>T<sub>∞</sub> = 정지 온도</li>
<li>ν = 운동 점성 계수</li>
<li>α = 열 퍼짐도</li>
<li>β = 열팽창계수</li>
</ul>
<p>를 나타낸다.</p>
<p><a class="mw-redirect" href="/wiki/%EC%A7%80%EA%B5%AC_%EB%AC%BC%EB%A6%AC%ED%95%99" title="지구 물리학">지구 물리학</a>에서 레일리 수는 매우 중요하다. 이 수는 유체 사이의 대류의 존재와 그 세기를 나타내며, 이것은 원래는 고체이지만 지질학적 시간 척도에 따르면 유체처럼 행동하는 지구의 <a href="/wiki/%EB%A7%A8%ED%8B%80" title="맨틀">맨틀</a>에 적용될 수 있다. 맨틀 내에서 레일리 수가 작은 값을 가지면 맨틀 경계면에서뿐만 아니라 맨틀 전체에서 대류가 일어난다는 것을 의미한다.</p>
<p><b>레일리 수</b>는 <a class="mw-redirect" href="/wiki/%EC%A1%B4_%EC%9C%8C%EB%A6%AC%EC%97%84_%EC%8A%A4%ED%8A%B8%EB%9F%BF_%EB%A0%88%EC%9D%BC%EB%A6%AC" title="존 윌리엄 스트럿 레일리">레일리</a>에 의해 이름 지어졌다.</p>
