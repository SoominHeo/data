<p><b>펜탁스 *istD</b>(<span style="font-size: smaller;"><a href="/wiki/%ED%95%9C%EA%B5%AD%EC%96%B4" title="한국어">한국어</a>:</span> <span lang="ko" xml:lang="ko">펜탁스 zxD</span>)는 23.7 x 16.6 mm 크기의 631만 (610만 <a class="mw-redirect" href="/wiki/%EC%9C%A0%ED%9A%A8_%ED%99%94%EC%86%8C" title="유효 화소">유효 화소</a>)<a class="mw-redirect" href="/wiki/CCD" title="CCD">CCD</a>를 가지는 <a href="/wiki/%ED%8E%9C%ED%83%81%EC%8A%A4" title="펜탁스">펜탁스</a> 최초의 <a class="mw-redirect" href="/wiki/%EB%94%94%EC%A7%80%ED%84%B8_SLR" title="디지털 SLR">디지털 SLR</a>카메라이자, 중급기종이다. 이 카메라는 <a href="/wiki/2003%EB%85%84" title="2003년">2003년</a> 가을에 시판되었으며, <a href="/wiki/%EB%8C%80%ED%95%9C%EB%AF%BC%EA%B5%AD" title="대한민국">대한민국</a>에서는 상표권 문제로 <b>PENTAX zxD</b>라는 상품명으로 발매되었다.</p>

<h2><span class="mw-headline" id=".EC.82.AC.EC.96.91">사양</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%ED%8E%9C%ED%83%81%EC%8A%A4_*ist_D&amp;action=edit&amp;section=1" title="부분 편집: 사양">원본 편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<ul>
<li><a class="mw-redirect" href="/wiki/%EC%B8%A1%EA%B4%91" title="측광">측광</a> : A·F·FA·FAJ·DA·DFA 렌즈 사용시 16분할 측광, 중앙중점식, 스팟측광 / K,M 렌즈는 펌웨어 업그레이드 후 옵션을 설정한 뒤 그린 단추로 측광할 수 있음</li>
<li>노출모드 : 하이퍼 프로그램 · 프로그램(P) · 조리개우선(Av) · 셔터우선(Tv) · 매뉴얼(M)</li>
<li><a class="new" href="/w/index.php?title=%EB%85%B8%EC%B6%9C%EB%B3%B4%EC%A0%95&amp;action=edit&amp;redlink=1" title="노출보정 (없는 문서)">노출보정</a> : ±3.0 (1/2EV 씩)</li>
<li><a class="new" href="/w/index.php?title=%EB%B8%8C%EB%9D%BC%EC%BC%80%ED%8C%85&amp;action=edit&amp;redlink=1" title="브라케팅 (없는 문서)">브라케팅</a> : 1/2EV 혹은 1/3EV씩 세장까지 가능</li>
<li><a class="mw-redirect" href="/wiki/CCD" title="CCD">CCD</a> : <a class="mw-redirect" href="/wiki/APS-C" title="APS-C">APS-C</a> 규격의 23.5x15.7mm CCD(1.5배 크롭). 원색필터. 610만/631만화소(유효/실제)</li>
<li>LCD : 1.8" TFT LCD·11만8천화소</li>
<li>ISO 감도 설정: ISO 200-1600· 확장시 3200</li>
<li>저장방식 : <a class="new" href="/w/index.php?title=CF_%EC%B9%B4%EB%93%9C&amp;action=edit&amp;redlink=1" title="CF 카드 (없는 문서)">CF 카드</a></li>
<li>USB 형식 : <a href="/wiki/USB" title="USB">USB</a> 1.1</li>
<li>촬영 파일 포맷 : JPG, TIFF, RAW촬영 가능</li>
<li>셔터 : 30 - 1/4000, 벌브(B/Bulb)</li>
<li>동조속도 : 1/150</li>
<li>셔터타입 : 세로주행 전자식 금속 셔터</li>
<li>연속 촬영 : 2.6매/초, 6장까지 촬영가능</li>
<li>피사계심도 미리보기 기능 : 있음</li>
<li>셀프타이머 : 있음(12초·2초)</li>
<li>미러업 : 있음 (셀프타이머 사용)</li>
<li>노출고정 : 있음 (20초)</li>
<li>내장 플래시 : GN 15.6 / 환산 28mm(ISO200/m), 컨트라스트컨트롤, 저속동조</li>
<li>외장 플래시 : 전용 외장 플래시 사용시 P-TTL, 무선동조, 고속동조, 후막동조</li>
<li>생산년도 : 2003년</li>

<li>배터리 : 1.5V AA Battery x 4 (니켈수소충전지 추천), CRV3 x 2</li>
<li>뷰파인더 : 95%/0.95배(1.5 크롭 APS-C <a class="mw-redirect" href="/wiki/CCD" title="CCD">CCD</a> 기준 시야율/배율), 펜타프리즘 채용, -2.5~+1.5 디옵터 설정</li>

<li>보다 정밀한 SAFOX VIII 자동초점 시스템</li>
