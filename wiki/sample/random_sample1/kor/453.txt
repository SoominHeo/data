<p><b>MPEG-4 파트 14</b>(MPEG-4 Part 14) 또는 <b>MP4</b> (공식적으로 <a href="/wiki/%EA%B5%AD%EC%A0%9C_%ED%91%9C%EC%A4%80%ED%99%94_%EA%B8%B0%EA%B5%AC" title="국제 표준화 기구">ISO</a>/<a href="/wiki/%EA%B5%AD%EC%A0%9C_%EC%A0%84%EA%B8%B0_%ED%91%9C%EC%A4%80_%ED%9A%8C%EC%9D%98" title="국제 전기 표준 회의">IEC</a> 14496-14:2003)는 <a href="/wiki/MPEG-4" title="MPEG-4">MPEG-4</a>의 일부로 규정된 <a href="/wiki/%EB%A9%80%ED%8B%B0%EB%AF%B8%EB%94%94%EC%96%B4" title="멀티미디어">멀티미디어</a> <a class="mw-redirect" href="/wiki/%EC%BB%A8%ED%85%8C%EC%9D%B4%EB%84%88_%ED%8F%AC%EB%A7%B7" title="컨테이너 포맷">컨테이너 포맷</a> 표준이다. <a class="mw-redirect" href="/wiki/%EB%94%94%EC%A7%80%ED%84%B8_%EB%B9%84%EB%94%94%EC%98%A4" title="디지털 비디오">디지털 비디오</a>와 <a href="/wiki/%EB%94%94%EC%A7%80%ED%84%B8_%EC%98%A4%EB%94%94%EC%98%A4" title="디지털 오디오">디지털 오디오</a> 스트림을 저장하는데 사용하는 것이 일반적이며 <a href="/wiki/%EC%9E%90%EB%A7%89" title="자막">자막</a>과 스틸 이미지 따위의 기타 데이터를 저장하는데 사용할 수도 있다. 현대의 대부분의 컨테이너 포맷과 동일하게 MPEG-4 파트 14는 <a href="/wiki/%EC%9D%B8%ED%84%B0%EB%84%B7" title="인터넷">인터넷</a>을 통한 <a href="/wiki/%EC%8A%A4%ED%8A%B8%EB%A6%AC%EB%B0%8D" title="스트리밍">스트리밍</a>을 지원한다. 파일 내 스트리밍 정보를 포함하기 위해 별도의 힌트 트랙이 사용된다. 공식적인 기본 확장자는 .mp4이다. <a href="/wiki/MP4_%ED%94%8C%EB%A0%88%EC%9D%B4%EC%96%B4" title="MP4 플레이어">MP4 플레이어</a>로 광고되는 일부 장치들은 단순히 <a class="mw-redirect" href="/wiki/MP3_%ED%94%8C%EB%A0%88%EC%9D%B4%EC%96%B4" title="MP3 플레이어">MP3 플레이어</a>로서 <a class="mw-disambig" href="/wiki/AMV" title="AMV">AMV</a> 비디오나 일부 기타 비디오 포맷도 재생하지만 반드시 MPEG-4 파트 14 포맷을 재생하는 것은 아니다.<sup class="reference" id="cite_ref-1"><a href="#cite_note-1">[1]</a></sup><sup class="reference" id="cite_ref-2"><a href="#cite_note-2">[2]</a></sup></p>

<p>MPEG-4 파트 14는 간단히 MP4로 줄여서 사용되며 MP4 파일, MP4 컨테이너 등으로 불린다. 대한민국에서는 이와 달리 <a class="mw-redirect" href="/wiki/H.264" title="H.264">H.264</a>기반으로 인코딩되어 휴대전화에서 재생가능하도록 제작한 파일을 MP4라고 부르기도 하며, 단순히 파일의 확장자를 변경하여 .mp4, .k3g(KTF/LGT용 휴대전화), .skm(SKT용 휴대전화)끼리 상호 변환할 수 있다.</p>

<h2><span class="mw-headline" id=".ED.8C.8C.EC.9D.BC_.ED.99.95.EC.9E.A5.EC.9E.90">파일 확장자</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=MPEG-4_%ED%8C%8C%ED%8A%B8_14&amp;action=edit&amp;section=3" title="부분 편집: 파일 확장자">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>공식 <a href="/wiki/%ED%8C%8C%EC%9D%BC_%ED%99%95%EC%9E%A5%EC%9E%90" title="파일 확장자">파일 확장자</a>가 .mp4로 규정되어 있으나 특정 콘텐츠에 적용하기 위해 다양한 파일 확장자가 사용된다.</p>
<ul>
<li>오디오와 비디오가 포함된 MPEG-4 파일들은 표준 .mp4 확장자를 사용한다.</li>
<li>오디오 전용 MPEG-4 파일들은 일반적으로 .m4a 확장자를 갖고 있다.
<ul>
<li><a class="mw-disambig" href="/wiki/%ED%8E%98%EC%96%B4%ED%94%8C%EB%A0%88%EC%9D%B4" title="페어플레이">페어플레이</a> <a href="/wiki/%EB%94%94%EC%A7%80%ED%84%B8_%EA%B6%8C%EB%A6%AC_%EA%B4%80%EB%A6%AC" title="디지털 권리 관리">디지털 권리 관리</a>로 암호화된 오디오 스트림이 있는 MPEG-4 파일은 <a class="mw-redirect" href="/wiki/%EC%95%84%EC%9D%B4%ED%8A%A0%EC%A6%88_%EC%8A%A4%ED%86%A0%EC%96%B4" title="아이튠즈 스토어">아이튠즈 스토어</a>를 통해 판매되며 .m4p 확장자를 사용한다. 아이튠즈 스토어가 현재 판매하는 <a class="new" href="/w/index.php?title=%EC%95%84%EC%9D%B4%ED%8A%A0%EC%A6%88_%ED%94%8C%EB%9F%AC%EC%8A%A4&amp;action=edit&amp;redlink=1" title="아이튠즈 플러스 (없는 문서)">아이튠즈 플러스</a> 트랙들은 암호화되어 있지 않으며 .m4a 확장자를 사용한다.</li>
<li>챕터 마커, 이미지, 하이퍼링크의 메타데이터를 포함하는 <a class="new" href="/w/index.php?title=%EC%98%A4%EB%94%94%EC%98%A4_%EB%B6%81&amp;action=edit&amp;redlink=1" title="오디오 북 (없는 문서)">오디오 북</a>과 <a href="/wiki/%ED%8C%9F%EC%BA%90%EC%8A%A4%ED%8A%B8" title="팟캐스트">팟캐스트</a> 파일은 .m4a 확장자를 사용할 수 있지만 .m4b 확장자가 더 일반적으로 사용된다. .m4a 오디오 파일은 북마크(bookmark: 마지막 청취 지점 기억)할 수 없지만 .m4b 확장자의 파일은 할 수 있다.</li>
<li>애플 <a href="/wiki/%EC%95%84%EC%9D%B4%ED%8F%B0" title="아이폰">아이폰</a>은 벨소리로 MPEG-4 오디오를 사용하지만 .m4a가 아닌 .m4r 확장자를 사용한다.</li>
</ul>
</li>
<li>순수 <a class="mw-redirect" href="/wiki/MPEG-4_%EB%B9%84%EC%A3%BC%EC%96%BC" title="MPEG-4 비주얼">MPEG-4 비주얼</a> 비트스트림은 .m4v라는 이름의 확장자를 사용하지만 이 확장자는 MP4 컨테이너 포맷의 비디오에 사용되기도 한다.<sup class="reference" id="cite_ref-3"><a href="#cite_note-3">[3]</a></sup></li>
<li><a class="mw-redirect" href="/wiki/%ED%9C%B4%EB%8C%80%EC%A0%84%ED%99%94" title="휴대전화">휴대전화</a>는 MPEG-4 파트 12 (MPEG-4/JPEG2000 ISO 기반 미디어 파일 포맷)로서 <a href="/wiki/3GP" title="3GP">3GP</a>를 사용하는데 이는 MP4와 비슷하다. .3gp와 .3g2 확장자를 사용한다. 이 파일들은 비 MPEG-4 데이터(<a href="/wiki/H.263" title="H.263">H.263</a>, AMR, TX3G)를 저장할 수도 있다.</li>
</ul>
