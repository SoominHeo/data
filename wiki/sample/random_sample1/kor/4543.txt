<p><b>옌 로니 욘센</b>(<span style="font-size: smaller;"><a href="/wiki/%EB%85%B8%EB%A5%B4%EC%9B%A8%EC%9D%B4%EC%96%B4" title="노르웨이어">노르웨이어</a>:</span> <span lang="no" xml:lang="no">Jean Ronny Johnsen</span>, <a href="/wiki/1969%EB%85%84" title="1969년">1969년</a> <a href="/wiki/6%EC%9B%94_10%EC%9D%BC" title="6월 10일">6월 10일</a> ~ )은 <a href="/wiki/%EB%85%B8%EB%A5%B4%EC%9B%A8%EC%9D%B4" title="노르웨이">노르웨이</a>의 은퇴한 축구 선수이다. 선수 시절 포지션은 <a href="/wiki/%EC%88%98%EB%B9%84%EC%88%98" title="수비수">센터 백</a>, <a href="/wiki/%EB%AF%B8%EB%93%9C%ED%95%84%EB%8D%94" title="미드필더">수비형 미드필더</a>였으며 <a class="new" href="/w/index.php?title=%EC%97%90%EC%9D%B4%ED%81%AC_%ED%87%B8%EC%8A%A4%EB%B2%A0%EB%A5%B4&amp;action=edit&amp;redlink=1" title="에이크 퇸스베르 (없는 문서)">에이크 퇸스베르</a>, <a href="/wiki/%EB%A5%80_%ED%8F%AC%ED%8A%B8%EB%B0%9C" title="륀 포트발">륀</a>, <a href="/wiki/%EB%A6%B4%EB%A0%88%EC%8A%A4%ED%8A%B8%EB%A3%80_SK" title="릴레스트룀 SK">릴레스트룀</a>, <a class="mw-redirect" href="/wiki/%EB%B2%A0%EC%8B%9D%ED%83%80%EC%89%AC_JK" title="베식타쉬 JK">베식타쉬</a>, <a href="/wiki/%EB%A7%A8%EC%B2%B4%EC%8A%A4%ED%84%B0_%EC%9C%A0%EB%82%98%EC%9D%B4%ED%8B%B0%EB%93%9C_FC" title="맨체스터 유나이티드 FC">맨체스터 유나이티드</a>, <a href="/wiki/%EC%95%A0%EC%8A%A4%ED%84%B4_%EB%B9%8C%EB%9D%BC_FC" title="애스턴 빌라 FC">애스턴 빌라</a>, <a href="/wiki/%EB%89%B4%EC%BA%90%EC%8A%AC_%EC%9C%A0%EB%82%98%EC%9D%B4%ED%8B%B0%EB%93%9C_FC" title="뉴캐슬 유나이티드 FC">뉴캐슬 유나이티드</a>, <a href="/wiki/%EB%B3%BC%EB%A0%88%EB%A0%9D%EC%95%84_%ED%8F%AC%ED%8A%B8%EB%B0%9C" title="볼레렝아 포트발">볼레렝아</a>에서 활약한 바 있다.</p>
<p>욘센은 <a href="/wiki/%EB%85%B8%EB%A5%B4%EC%9B%A8%EC%9D%B4_%EC%B6%95%EA%B5%AC_%EA%B5%AD%EA%B0%80%EB%8C%80%ED%91%9C%ED%8C%80" title="노르웨이 축구 국가대표팀">노르웨이 축구 국가대표팀</a> 소속으로 62경기를 뛰었으며, <a href="/wiki/1998%EB%85%84_FIFA_%EC%9B%94%EB%93%9C%EC%BB%B5" title="1998년 FIFA 월드컵">1998년 FIFA 월드컵</a>에 출전한 바 있다.<sup class="reference" id="cite_ref-HoF_1-0"><a href="#cite_note-HoF-1">[1]</a></sup></p>
