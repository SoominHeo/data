<p><b>로라시아상목</b>(Laurasiatheria)은 DNA 염기서열 분석에 기초한, <a class="mw-redirect" href="/wiki/%ED%83%9C%EB%B0%98%EB%A5%98" title="태반류">태반류</a>(Placentalia) 또는 <a href="/wiki/%EC%A7%84%EC%88%98%ED%95%98%EA%B0%95" title="진수하강">진수하강</a>(Eutheria)에 속하는 <a href="/wiki/%ED%8F%AC%EC%9C%A0%EB%A5%98" title="포유류">포유류</a>의 분류군이다.</p>
<p>이 이름은, <a href="/wiki/%EC%B4%88%EB%8C%80%EB%A5%99" title="초대륙">초대륙</a> <a href="/wiki/%ED%8C%90%EA%B2%8C%EC%95%84" title="판게아">판게아</a>에서 분리된 <a href="/wiki/%EB%A1%9C%EB%9D%BC%EC%8B%9C%EC%95%84" title="로라시아">로라시아</a>와 <a href="/wiki/%EA%B3%A4%EB%93%9C%EC%99%80%EB%82%98" title="곤드와나">곤드와나</a> 중, <a href="/wiki/%EB%A1%9C%EB%9D%BC%EC%8B%9C%EC%95%84" title="로라시아">로라시아</a>에서 이들 포유류가 진화했다는 이론에서 유래하였다. <a href="/wiki/%EC%98%81%EC%9E%A5%EC%83%81%EB%AA%A9" title="영장상목">영장상목</a>(Euarchontoglires)과 자매군이다. 다음과 같은 현존하는 동물의 목을 포함하고 있다.</p>
<ul>
<li><a href="/wiki/%EA%B3%A0%EC%8A%B4%EB%8F%84%EC%B9%98%EA%B3%BC" title="고슴도치과">고슴도치목</a> (Erinaceomorpha) : 고슴도치와 털고슴도치</li>
<li><a href="/wiki/%EB%95%83%EC%A5%90%EB%AA%A9" title="땃쥐목">땃쥐목</a> (Soricomorpha) : 두더쥐, 땃쥐, 솔레노돈</li>
<li><a class="new" href="/w/index.php?title=%EA%B2%BD%EC%9A%B0%EC%A0%9C%EB%A5%98&amp;action=edit&amp;redlink=1" title="경우제류 (없는 문서)">경우제류</a> (Cetartiodactyla) : 전 세계적으로 분포
<ul>
<li><a href="/wiki/%EA%B3%A0%EB%9E%98" title="고래">고래목</a> (Cetacea) : 고래와 돌고래 그리고 참돌고래</li>
<li><a href="/wiki/%EC%86%8C%EB%AA%A9_(%EB%8F%99%EB%AC%BC)" title="소목 (동물)">소목</a> (Artiodactyla) : 발굽이 짝수 개(돼지, 하마, 낙타, 기린, 사슴, 영양, 소, 양, 염소 등)</li>
</ul>
</li>
<li>Pegasoferae:
<ul>
<li><a href="/wiki/%EC%B2%9C%EC%82%B0%EA%B0%91" title="천산갑">유린목</a> (Pholidota) : 천산갑(아프리카, 남아시아)</li>
<li><a href="/wiki/%EB%B0%95%EC%A5%90" title="박쥐">박쥐목</a> (Chiroptera) : 박쥐 등 (전 세계적으로 분포)</li>
<li><a href="/wiki/%EC%8B%9D%EC%9C%A1%EB%AA%A9" title="식육목">식육목</a> (Carnivora) : 개 등(전 세계적으로 분포)</li>
<li><a class="mw-redirect" href="/wiki/%EA%B8%B0%EC%A0%9C%EB%AA%A9" title="기제목">기제목</a> (Perissodactyla) : 발굽이 홀수 개(말, 맥, 코뿔소 등)</li>
</ul>
</li>
</ul>
<p>로라시아상목, 고슴도치목 등은 매우 다양하게 분기되어 진화하였음을 보여 주고 있다. 연구 결과에 따라서, 일부는 말목과 Ferae를 Zooamata 계통군에 넣거나 일부는 말목과 <a class="new" href="/w/index.php?title=%EA%B2%BD%EC%9A%B0%EC%A0%9C%EB%A5%98&amp;action=edit&amp;redlink=1" title="경우제류 (없는 문서)">경우제류</a>(鯨偶蹄類)를 진(眞)-<a href="/wiki/%EC%9C%A0%EC%A0%9C%EB%A5%98" title="유제류">유제류</a>(有蹄類) 계통군에 넣기도 한다. 어느 계통 분류 방법도 완벽하게 지지되고 있지는 않다.</p>
<p>로라시아상목에는 또한 다음과 같은 멸종된 여러 개의 목을 포함하고 있다.</p>
<ul>
<li>Meridiungulata</li>
<li><a class="new" href="/w/index.php?title=%EA%B3%BC%EC%A0%88%EB%A5%98&amp;action=edit&amp;redlink=1" title="과절류 (없는 문서)">과절목</a> (顆節目, Condylarthra)</li>
<li>Dinocerata</li>
<li>Mesonychia</li>
<li>Cimolesta</li>
<li><a class="new" href="/w/index.php?title=%EC%9C%A1%EC%B9%98%EB%A5%98&amp;action=edit&amp;redlink=1" title="육치류 (없는 문서)">육치목</a> (肉齒目, Creodonta)</li>
</ul>
