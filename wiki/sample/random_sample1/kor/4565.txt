<h1 class="firstHeading" id="firstHeading" lang="ko">로런츠 군</h1>
<div class="mw-body-content" id="bodyContent">
<div id="siteSub">위키백과, 우리 모두의 백과사전.</div>
<div id="contentSub"></div>
<div class="mw-jump" id="jump-to-nav">
					이동:					<a href="#mw-head">둘러보기</a>, 					<a href="#p-search">검색</a>
</div>
<div class="mw-content-ltr" dir="ltr" id="mw-content-text" lang="ko"><script>function mfTempOpenSection(id){var block=document.getElementById("mf-section-"+id);block.className+=" open-block";block.previousSibling.className+=" open-block";}</script><p><b>로런츠 군</b>(Lorentz群, <span lang="en" xml:lang="en">Lorentz group</span>)이란 <a href="/wiki/%EB%AF%BC%EC%BD%94%ED%94%84%EC%8A%A4%ED%82%A4_%EA%B3%B5%EA%B0%84" title="민코프스키 공간">민코프스키 공간</a> 상의 <a href="/wiki/%EB%A1%9C%EB%9F%B0%EC%B8%A0_%EB%B3%80%ED%99%98" title="로런츠 변환">로런츠 변환</a>과 <a class="new" href="/w/index.php?title=%ED%9A%8C%EC%A0%84%EB%B3%80%ED%99%98&amp;action=edit&amp;redlink=1" title="회전변환 (없는 문서)">회전변환</a>을 모아놓은 <a href="/wiki/%EA%B5%B0_(%EC%88%98%ED%95%99)" title="군 (수학)">군</a>을 말한다. 중력이 작용하지 않는 경우에는 로런츠 군에 속하는 변환에 대하여 많은 물리학적 법칙들의 형태가 변하지 않는 대칭성을 가지고 있다.</p>
<p>예를 들면,</p>
<ul>
<li><a class="mw-redirect" href="/wiki/%ED%8A%B9%EC%88%98_%EC%83%81%EB%8C%80%EC%84%B1_%EC%9D%B4%EB%A1%A0" title="특수 상대성 이론">특수 상대성 이론</a>의 동역학 법칙들</li>
<li><a href="/wiki/%EC%A0%84%EC%9E%90%EA%B8%B0%ED%95%99" title="전자기학">전자기학</a>의 <a href="/wiki/%EB%A7%A5%EC%8A%A4%EC%9B%B0_%EB%B0%A9%EC%A0%95%EC%8B%9D" title="맥스웰 방정식">맥스웰 방정식</a></li>
<li><a href="/wiki/%EC%96%91%EC%9E%90%EC%97%AD%ED%95%99" title="양자역학">양자역학</a>의 <a href="/wiki/%EC%A0%84%EC%9E%90" title="전자">전자</a>에 대한 <a href="/wiki/%EB%94%94%EB%9E%99_%EB%B0%A9%EC%A0%95%EC%8B%9D" title="디랙 방정식">디랙 방정식</a></li>
</ul>
<p>이 있다. 때문에 로런츠 군의 변환들은 자연의 법칙들이 가져야 할 기본적인 대칭성으로 받아들여지고 있다.</p>
<h2><span class="mw-headline" id=".EC.A0.95.EC.9D.98">정의</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%A1%9C%EB%9F%B0%EC%B8%A0_%EA%B5%B0&amp;action=edit&amp;section=1" title="부분 편집: 정의">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>로런츠 군은 <a href="/wiki/%EB%AF%BC%EC%BD%94%ED%94%84%EC%8A%A4%ED%82%A4_%EA%B3%B5%EA%B0%84" title="민코프스키 공간">민코프스키 공간</a>의 원점을 변화시키기 않는 등거리변환을 모두 모아놓은 군이다. 즉, 선형변환</p>

<p>가 변하지 않고 원점이 변하지 않는 변환을 모아놓은 군이다. 로런츠 군의 원소들은 <a href="/wiki/%EC%A7%81%EA%B5%90%ED%96%89%EB%A0%AC" title="직교행렬">직교행렬</a>, 즉, Λ<sup>−1</sup> = Λ<sup>T</sup> 이고, 계량텐서 <i>η</i><sub><i>μν</i></sub> 의 부호가 (+,-,-,-) 이기 때문에 <a href="/wiki/%EC%A7%81%EA%B5%90%EA%B5%B0" title="직교군">직교군</a> O(1,3) 라 부르기도 한다.</p>
<p>로런츠 군은 군이면서 <a href="/wiki/%EB%A7%A4%EB%81%84%EB%9F%AC%EC%9A%B4_%EB%8B%A4%EC%96%91%EC%B2%B4" title="매끄러운 다양체">매끄러운 다양체</a>이므로 <a href="/wiki/%EB%A6%AC_%EA%B5%B0" title="리 군">리 군</a>을 이룬다.</p>
<h2><span class="mw-headline" id=".EC.97.B0.EA.B2.B0.EC.84.B1.EB.B6.84.EA.B3.BC_.EC.A0.9C.ED.95.9C.EB.90.9C_.EB.A1.9C.EB.9F.B0.EC.B8.A0_.EA.B5.B0">연결성분과 제한된 로런츠 군</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%A1%9C%EB%9F%B0%EC%B8%A0_%EA%B5%B0&amp;action=edit&amp;section=2" title="부분 편집: 연결성분과 제한된 로런츠 군">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>로런츠 군은 총 네 개의 <a class="mw-redirect" href="/wiki/%EC%97%B0%EA%B2%B0_%EC%84%B1%EB%B6%84" title="연결 성분">연결 성분</a>을 갖는다. 즉, 위상수학적으로 서로 분리된 군의 네 부분들을 생각해 볼 수 있다. 간단히 말해, 로런츠 군의 원소들은 다음의 조건에 따라 네 가지로 분류할 수 있다.</p>
<ul>
<li>시간을 역전시키는가?</li>
<li>공간의 방향이 유지되는가?</li>
</ul>
<p>여기서, 시간이 역전되지 않는 변환을 정시적(<span lang="en" xml:lang="en">orthochronous</span>)이라고 하고, 방향이 유지되는, 즉 det Λ = 1 인 변환을 고유(<span lang="en" xml:lang="en">proper</span>)하다고 한다.</p>
<p>군의 <a href="/wiki/%ED%95%AD%EB%93%B1%EC%9B%90" title="항등원">항등원</a>은 정시적이며 고유한 연결 성분에 들어있으며, 시간이 역전되지 않고 방향이 유지되는 변환들이 모두 포함되어 있다. 이 또한 <a href="/wiki/%EB%B6%80%EB%B6%84%EA%B5%B0" title="부분군">부분군</a>을 이루며 리 군이다. 이 연결 성분을 <b>정시적 고유 로런츠 군</b> 또는 <b>제한된 로런츠 군</b>(<span lang="en" xml:lang="en">restricted Lorentz group</span>)이라 하며 SO<sup>+</sup>(1,3)라 쓴다. 경우에 따라선 로런츠 군을 말할때 제한된 로런츠 군을 가리키기도 한다. 또한 제한된 로런츠 군은 로런츠 군의 <a class="mw-redirect" href="/wiki/%EC%A0%95%EA%B7%9C_%EB%B6%80%EB%B6%84%EA%B5%B0" title="정규 부분군">정규 부분군</a>이다.</p>
<p>제한된 로런츠 군에 대한 로런츠 군의 <a class="mw-redirect" href="/wiki/%EB%AA%AB%EA%B5%B0" title="몫군">몫군</a> O(1,3)/SO<sup>+</sup>(1,3)은 <a class="new" href="/w/index.php?title=%EA%B3%B5%EA%B0%84%EB%B0%98%EC%A0%84&amp;action=edit&amp;redlink=1" title="공간반전 (없는 문서)">공간반전</a> P 와 <a class="new" href="/w/index.php?title=%EC%8B%9C%EA%B0%84%EC%97%AD%EC%A0%84&amp;action=edit&amp;redlink=1" title="시간역전 (없는 문서)">시간역전</a> T 로 구성되어 있으며, {1, P, T, PT} 의 네 가지 원소를 가지고 있다. 이 몫군은 <a href="/wiki/%ED%81%B4%EB%9D%BC%EC%9D%B8_4%EC%9B%90%EA%B5%B0" title="클라인 4원군">클라인 4원군</a>과 <a class="mw-redirect" href="/wiki/%EB%8F%99%ED%98%95" title="동형">동형</a>이다.</p>
<h2><span class="mw-headline" id=".EB.A1.9C.EB.9F.B0.EC.B8.A0_.EB.8C.80.EC.88.98">로런츠 대수</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%A1%9C%EB%9F%B0%EC%B8%A0_%EA%B5%B0&amp;action=edit&amp;section=3" title="부분 편집: 로런츠 대수">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
