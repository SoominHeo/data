<h1 class="firstHeading" id="firstHeading" lang="ko">로렌스 맥케온</h1>
<div class="mw-body-content" id="bodyContent">
<div id="siteSub">위키백과, 우리 모두의 백과사전.</div>
<div id="contentSub"></div>
<div class="mw-jump" id="jump-to-nav">
					이동:					<a href="#mw-head">둘러보기</a>, 					<a href="#p-search">검색</a>
</div>
<div class="mw-content-ltr" dir="ltr" id="mw-content-text" lang="ko"><script>function mfTempOpenSection(id){var block=document.getElementById("mf-section-"+id);block.className+=" open-block";block.previousSibling.className+=" open-block";}</script><p><b>로런스 맥키언</b>(<span lang="en" xml:lang="en">Laurence McKeown</span>, 1956년 ~ )은 <a href="/wiki/%EC%95%84%EC%9D%BC%EB%9E%9C%EB%93%9C" title="아일랜드">아일랜드</a>의 극작가이며, 한때 <a href="/wiki/%EC%95%84%EC%9D%BC%EB%9E%9C%EB%93%9C_%EA%B3%B5%ED%99%94%EA%B5%AD%EA%B5%B0_%EC%9E%84%EC%8B%9C%ED%8C%8C" title="아일랜드 공화국군 임시파">아일랜드 공화국군 임시파</a> 요원이었다. 1981년 <a href="/wiki/%EC%95%84%EC%9D%BC%EB%9E%9C%EB%93%9C_%EB%8B%A8%EC%8B%9D%ED%88%AC%EC%9F%81" title="아일랜드 단식투쟁">아일랜드 단식투쟁</a>에도 참여했다.</p>
<p>북아일랜드 <a href="/wiki/%EC%95%A4%ED%8A%B8%EB%A6%BC_%EC%A3%BC" title="앤트림 주">앤트림 주</a> <a class="new" href="/w/index.php?title=%EB%9E%9C%EB%8D%9C%EC%8A%A4%ED%83%80%EC%9A%B4&amp;action=edit&amp;redlink=1" title="랜덜스타운 (없는 문서)">랜덜스타운</a>에서 태어났다.<sup class="reference" id="cite_ref-searc_1-0"><a href="#cite_note-searc-1">[1]</a></sup> 10대 시절 맥키언은 건축가가 되고 싶었고 16살 때 <a class="new" href="/w/index.php?title=%EA%B2%AC%EC%A0%81%EC%82%AC&amp;action=edit&amp;redlink=1" title="견적사 (없는 문서)">견적사</a> 사무실에서 일하기 시작했다.<sup class="reference" id="cite_ref-guardian_2-0"><a href="#cite_note-guardian-2">[2]</a></sup> 17세 때 IRA에 들어갔고 1976년 8월 폭발물 소지 및 <a href="/wiki/%EC%99%95%EB%A6%BD_%EC%96%BC%EC%8A%A4%ED%84%B0_%EA%B2%BD%EC%B0%B0%EB%8C%80" title="왕립 얼스터 경찰대">왕립 얼스터 경찰대</a> 경관 살해미수 혐의로 체포되었다.<sup class="reference" id="cite_ref-searc_1-1"><a href="#cite_note-searc-1">[1]</a></sup><sup class="reference" id="cite_ref-guardian_2-1"><a href="#cite_note-guardian-2">[2]</a></sup> 1977년 4월 재판에서 유죄가 확정되어 종신형을 선고받고 <a class="new" href="/w/index.php?title=%EB%A9%94%EC%9D%B4%EC%A6%88_%EA%B5%90%EB%8F%84%EC%86%8C&amp;action=edit&amp;redlink=1" title="메이즈 교도소 (없는 문서)">메이즈 교도소</a>에 수감되었다.<sup class="reference" id="cite_ref-searc_1-2"><a href="#cite_note-searc-1">[1]</a></sup><sup class="reference" id="cite_ref-guardian_2-2"><a href="#cite_note-guardian-2">[2]</a></sup></p>
<p>맥키언은 <a class="new" href="/w/index.php?title=%ED%8A%B9%EC%88%98%EB%B2%94%EC%A3%BC%EC%A7%80%EC%9C%84&amp;action=edit&amp;redlink=1" title="특수범주지위 (없는 문서)">특수범주지위</a>의 반환을 요구하는 <a href="/wiki/%EB%AA%A8%ED%8F%AC%ED%88%AC%EC%9F%81" title="모포투쟁">모포투쟁</a>과 <a href="/wiki/%EB%B6%88%EA%B2%B0%ED%88%AC%EC%9F%81" title="불결투쟁">불결투쟁</a>에 참여했다. <a href="/wiki/%EB%B0%94%EB%B9%84_%EC%83%8C%EC%A6%88" title="바비 샌즈">바비 샌즈</a>가 주도하는 제2차 단식투쟁이 시작되었고, 맥키언은 샌즈 외 세 명의 참여자가 아사한 이후인 6월 29일 단식을 시작했다.<sup class="reference" id="cite_ref-cainlist_3-0"><a href="#cite_note-cainlist-3">[3]</a></sup> 여섯 명이 더 죽고 맥키언의 가족들은 단식 70일차인 9월 6일 의료진을 개입시켜 아들의 단식을 중단시켰다.<sup class="reference" id="cite_ref-cainlist_3-1"><a href="#cite_note-cainlist-3">[3]</a></sup></p>
<p>맥키언은 옥중에서 <a class="new" href="/w/index.php?title=%EA%B0%9C%EB%B0%A9%EB%8C%80%ED%95%99&amp;action=edit&amp;redlink=1" title="개방대학 (없는 문서)">개방대학</a> 과정을 밟아 사회과학 학사 학위를 취득했고 1992년 석방되었다. 그 뒤 <a class="mw-redirect" href="/wiki/%ED%80%B8%EC%A6%88_%EB%8C%80%ED%95%99%EA%B5%90_%EB%B2%A8%ED%8C%A8%EC%8A%A4%ED%8A%B8" title="퀸즈 대학교 벨패스트">퀸즈 대학교 벨패스트</a>에서 박사학위까지 마쳤다.<sup class="reference" id="cite_ref-guardian_2-3"><a href="#cite_note-guardian-2">[2]</a></sup><sup class="reference" id="cite_ref-4"><a href="#cite_note-4">[4]</a></sup><sup class="reference" id="cite_ref-irb_5-0"><a href="#cite_note-irb-5">[5]</a></sup> 1990년대 중반 맥키언은 <a class="new" href="/w/index.php?title=%EB%B2%A8%ED%8C%8C%EC%8A%A4%ED%8A%B8_%EC%98%81%ED%99%94%EC%A0%9C&amp;action=edit&amp;redlink=1" title="벨파스트 영화제 (없는 문서)">벨파스트 영화제</a>를 공동설립했고<sup class="reference" id="cite_ref-guardian_2-4"><a href="#cite_note-guardian-2">[2]</a></sup> 메이즈 교도소의 <a href="/wiki/%EC%95%84%EC%9D%BC%EB%9E%9C%EB%93%9C_%EA%B3%B5%ED%99%94%EC%A3%BC%EC%9D%98" title="아일랜드 공화주의">아일랜드 공화주의자</a> 수감자들에 관한 책을 두 권 썼다.<sup class="reference" id="cite_ref-irb_5-1"><a href="#cite_note-irb-5">[5]</a></sup></p>
