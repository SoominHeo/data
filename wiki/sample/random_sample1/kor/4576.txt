<p><b>고대 로마 신전</b>은 <a class="mw-redirect" href="/wiki/%EB%A1%9C%EB%A7%88_%EB%AC%B8%ED%99%94" title="로마 문화">로마 문화</a>의 고고학적 유물들 가장 시각적인 것들에 속하며 <a href="/wiki/%EB%A1%9C%EB%A7%88_%EA%B1%B4%EC%B6%95" title="로마 건축">로마 건축</a>을 이해함에 있어 중요한 요소이다. 신전(神殿)을 만드고 유지하는 것은 <a class="mw-redirect" href="/wiki/%EA%B3%A0%EB%8C%80_%EB%A1%9C%EB%A7%88_%EC%A2%85%EA%B5%90" title="고대 로마 종교">고대 로마 종교</a>의 주요한 한 부분이었다. <a href="/wiki/%EC%BC%88%EB%9D%BC" title="켈라">켈라</a>(<a class="extiw" href="https://en.wikipedia.org/wiki/cella" title="en:cella"><span style="color: black">cella</span></a>)라고 불린 주실(主室)에는 해당 신전이 모시는 <a class="mw-redirect" href="/wiki/%EB%A1%9C%EB%A7%88_%EC%8B%A0%EB%93%A4%EC%9D%98_%EB%AA%A9%EB%A1%9D" title="로마 신들의 목록">신</a>의 신상(神像)이 놓여 있었으며 종종 <a href="/wiki/%ED%96%A5" title="향">향</a>을 피우거나 <a class="mw-disambig" href="/wiki/%EC%A0%9C%EC%A3%BC" title="제주">제주</a>(祭酒)를 올리는 작은 제단이 함께 놓여 있었다. <a href="/wiki/%EC%BC%88%EB%9D%BC" title="켈라">켈라</a> 뒤에는 신전의 수행원들과 관리인들의 용품이나 봉헌자들의 봉헌물을 보관하는 용도로 사용한 하나 혹은 그 이상의 방이 있었다.</p>
<p><a class="mw-redirect" href="/wiki/%EC%8B%A0%EC%A0%84" title="신전">신전</a>(神殿)을 뜻하는 <a href="/wiki/%EC%98%81%EC%96%B4" title="영어">영어</a> 단어 템플(temple)은 <a href="/wiki/%EB%9D%BC%ED%8B%B4%EC%96%B4" title="라틴어">라틴어</a> 템플룸(templum)에서 유래하였다. 원래 템플룸(templum)은 빌딩을 뜻하는 것이 아니라 의식을 통해 측량되고 구획된 신성한 장소를 뜻하는 낱말이었다. <a href="/wiki/%EB%A1%9C%EB%A7%88_%EA%B1%B4%EC%B6%95" title="로마 건축">로마 건축가</a> <a href="/wiki/%EB%B9%84%ED%8A%B8%EB%A3%A8%EB%B9%84%EC%9A%B0%EC%8A%A4" title="비트루비우스">비트루비우스</a>는 항상 템플룸이라는 낱말을 신성한 장소 또는 구역, 즉 성역(聖域 <b>·</b> sacred precinct)을 가리키는데 사용하였으며 건물을 가리키는데 사용하지는 않았다. 건물로서의 신전 또는 전당을 가리키는 용도로 자주 사용된 일반적인 라틴어 단어는 아에데스(aedes), 데루브룸(delubrum) 또는 파눔(fanum)이었다. (이 문서에서는 이러한 건물을 가리키는데 "신전"이라는 낱말을 사용하고 있으며 신성한 장소 또는 구역을 가리키는데는 "성역"를 사용하고 있다).</p>
<p>공공의 종교 의식들은 <a class="mw-redirect" href="/wiki/%EC%8B%A0%EC%A0%84" title="신전">신전</a> 건물 내부가 아닌 밖에서 열렸다. 어떤 의식들은 어떤 신전 또는 전당에서 출발하거나 혹은 어떤 신전 또는 전당을 중간에 방문하거나 혹은 어떤 신전 또는 전당에서 끝마치거나 하는 행렬 의식이었는데, 이것은 그 신전 또는 전당에 의식에 사용되는 용구나 봉헌물이 저장되어 있었다는 것을 시사한다. <a class="new" href="/w/index.php?title=%ED%9D%AC%EC%83%9D_%EC%9D%98%EC%8B%9D&amp;action=edit&amp;redlink=1" title="희생 의식 (없는 문서)">희생 의식</a>(<a class="extiw" href="https://en.wikipedia.org/wiki/Sacrifice" title="en:Sacrifice"><span style="color: black">Sacrifice</span></a>)이 성역 내의 옥외 <a href="/wiki/%EC%A0%9C%EB%8B%A8" title="제단">제단</a>에서 열리곤 했는데 주로 <a class="new" href="/w/index.php?title=%EB%8F%99%EB%AC%BC_%ED%9D%AC%EC%83%9D_%EC%9D%98%EC%8B%9D&amp;action=edit&amp;redlink=1" title="동물 희생 의식 (없는 문서)">동물 희생 의식</a>(<a class="extiw" href="https://en.wikipedia.org/wiki/Animal_sacrifice" title="en:Animal sacrifice"><span style="color: black">Animal sacrifice</span></a>)이 행해졌다.</p>

<h2><span class="mw-headline" id=".EA.B8.B0.EC.9B.90.EA.B3.BC_.EB.B0.9C.EC.A0.84">기원과 발전</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%A1%9C%EB%A7%88_%EC%8B%A0%EC%A0%84&amp;action=edit&amp;section=1" title="부분 편집: 기원과 발전">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>로마 신전의 건축 양식은 <a href="/wiki/%EC%97%90%ED%8A%B8%EB%A3%A8%EB%A6%AC%EC%95%84" title="에트루리아">에트루리아</a> 양식에서 기원한 것으로, <a href="/wiki/%EC%97%90%ED%8A%B8%EB%A3%A8%EB%A6%AC%EC%95%84%EC%9D%B8" title="에트루리아인">에트루리아인</a>은 <a href="/wiki/%EA%B8%B0%EC%9B%90%EC%A0%84_7%EC%84%B8%EA%B8%B0" title="기원전 7세기">기원전 7세기</a>에 정정기를 맞았던 토착 <a href="/wiki/%EC%9D%B4%ED%83%88%EB%A6%AC%EC%95%84%EC%9D%B8" title="이탈리아인">이탈리아인</a>이다. <a href="/wiki/%EC%97%90%ED%8A%B8%EB%A3%A8%EB%A6%AC%EC%95%84%EC%9D%B8" title="에트루리아인">에트루리아인</a>은 다른 양식들을 자신들의 신전에 도입했는데 그 중 가장 큰 영향을 미친 것을 <a class="mw-redirect" href="/wiki/%EA%B7%B8%EB%A6%AC%EC%8A%A4_%EA%B1%B4%EC%B6%95" title="그리스 건축">그리스 건축</a>이었다. 따라서 로마 신전은 다른 나라의 신전들과 구분되는 특징을 가지고 있으면서도 <a href="/wiki/%EC%97%90%ED%8A%B8%EB%A3%A8%EB%A6%AC%EC%95%84" title="에트루리아">에트루리아</a>와 <a href="/wiki/%EA%B3%A0%EB%8C%80_%EA%B7%B8%EB%A6%AC%EC%8A%A4" title="고대 그리스">그리스</a>의 <a class="mw-redirect" href="/wiki/%EC%8B%A0%EC%A0%84" title="신전">신전</a> 건축 양식을 기반으로 하고 있다.<sup class="reference" id="cite_ref-Campbell_1-0"><a href="#cite_note-Campbell-1">[1]</a></sup></p>
<p>로마 신전은 건물의 앞부분을 강조했는데, 신전의 앞부분은 <a class="mw-redirect" href="/wiki/%EC%A3%BC%EB%9E%91_%EC%97%B4%EC%A3%BC" title="주랑 열주">주랑 열주</a>라고 불리는 <a href="/wiki/%EA%B8%B0%EB%91%A5" title="기둥">기둥</a>들이 있는 <a href="/wiki/%EC%A3%BC%EB%9E%91_%ED%98%84%EA%B4%80" title="주랑 현관">주랑 현관</a>(<a class="extiw" href="https://en.wikipedia.org/wiki/portico" title="en:portico"><span style="color: black">portico</span></a>)과 <a class="mw-redirect" href="/wiki/%EC%A3%BC%EB%9E%91_%EC%97%B4%EC%A3%BC" title="주랑 열주">주랑 열주</a>에서 <a href="/wiki/%EC%BC%88%EB%9D%BC" title="켈라">켈라</a>(<a class="extiw" href="https://en.wikipedia.org/wiki/cella" title="en:cella"><span style="color: black">cella</span></a>) 입구 사이의 공간인 <a class="mw-redirect" href="/wiki/%ED%94%84%EB%A1%9C%EB%82%98%EC%98%A4%EC%8A%A4" title="프로나오스">프로나오스</a>(<a class="extiw" href="https://en.wikipedia.org/wiki/pronaos" title="en:pronaos"><span style="color: black">pronaos</span></a>)로 이루어져 있다. 이렇게 <a class="mw-redirect" href="/wiki/%EC%8B%A0%EC%A0%84" title="신전">신전</a>의 앞부분을 강조한 것은 <a href="/wiki/%EA%B7%B8%EB%A6%AC%EC%8A%A4" title="그리스">그리스</a> 신전 양식에서 벗어난 것인데, <a href="/wiki/%EA%B7%B8%EB%A6%AC%EC%8A%A4_%EC%8B%A0%EC%A0%84" title="그리스 신전">그리스 신전</a>에서는 <a class="mw-redirect" href="/wiki/%EC%8B%A0%EC%A0%84" title="신전">신전</a>의 모든 부분이 동등하게 중요시되었고 모든 방향에서 보거나 접근할 수 있도록 되어 있다.</p>
