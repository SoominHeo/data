<h1 class="firstHeading" id="firstHeading" lang="ko">로버트 가네</h1>
<div class="mw-body-content" id="bodyContent">
<div id="siteSub">위키백과, 우리 모두의 백과사전.</div>
<div id="contentSub"></div>
<div class="mw-jump" id="jump-to-nav">
					이동:					<a href="#mw-head">둘러보기</a>, 					<a href="#p-search">검색</a>
</div>
<div class="mw-content-ltr" dir="ltr" id="mw-content-text" lang="ko"><script>function mfTempOpenSection(id){var block=document.getElementById("mf-section-"+id);block.className+=" open-block";block.previousSibling.className+=" open-block";}</script><p><b>로버트 가네</b>(Robert M. Gagné, <a href="/wiki/1916%EB%85%84" title="1916년">1916년</a> <a href="/wiki/8%EC%9B%94_21%EC%9D%BC" title="8월 21일">8월 21일</a> ~ <a href="/wiki/2002%EB%85%84" title="2002년">2002년</a> <a href="/wiki/4%EC%9B%94_28%EC%9D%BC" title="4월 28일">4월 28일</a>)는 '학습의 조건'을 연구한 <a href="/wiki/%EB%AF%B8%EA%B5%AD" title="미국">미국</a>의 <a href="/wiki/%EA%B5%90%EC%9C%A1%EC%8B%AC%EB%A6%AC%ED%95%99" title="교육심리학">교육심리학자</a>이다. 그는 <a href="/wiki/%EC%A0%9C2%EC%B0%A8_%EC%84%B8%EA%B3%84_%EB%8C%80%EC%A0%84" title="제2차 세계 대전">제2차 세계 대전</a> 당시 <a href="/wiki/%EA%B3%B5%EA%B5%B0" title="공군">공군</a> <a class="mw-redirect" href="/wiki/%ED%8C%8C%EC%9D%BC%EB%9F%BF" title="파일럿">파일럿</a>들을 대상으로 명령 과학을 선구적으로 연구하였다. 이후 그는 '좋은 수업'을 하기 위해서 고려해야 할 요소들과, 컴퓨터를 기반으로 한 훈련과 <a href="/wiki/%EB%A9%80%ED%8B%B0%EB%AF%B8%EB%94%94%EC%96%B4" title="멀티미디어">멀티미디어</a> 기반 학습을 위한 교수 이론의 개념을 정립하고자 하였다.</p>
<p>가네의 업적 중 가장 대표적인 것은 학습에는 여러 유형들이 존재하고, 서로 다른 환경은 다른 학습 유형을 이끌어낸다는 <b>가네의 가정</b>이다.</p>

<h2><span class="mw-headline" id=".EC.A0.84.EA.B8.B0">전기</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%A1%9C%EB%B2%84%ED%8A%B8_%EA%B0%80%EB%84%A4&amp;action=edit&amp;section=1" title="부분 편집: 전기">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<ul>
<li><a href="/wiki/%EB%A7%A4%EC%82%AC%EC%B6%94%EC%84%B8%EC%B8%A0_%EC%A3%BC" title="매사추세츠 주">매사추세츠 주</a> 노스 앤도버에 있는 한 고등학교에서, 가네는 <a class="mw-redirect" href="/wiki/%EC%8B%AC%EB%A6%AC%ED%95%99%EC%9E%90" title="심리학자">심리학자</a>가 되기로 마음 먹었다. 1932년 고등학교 졸업식에서 그는 <a href="/wiki/%EC%8B%AC%EB%A6%AC%ED%95%99" title="심리학">심리학</a>이 인간의 삶의 부담을 완화하는데 사용되어야 한다는 연설을 했다.</li>
<li><a href="/wiki/%EC%98%88%EC%9D%BC_%EB%8C%80%ED%95%99%EA%B5%90" title="예일 대학교">예일 대학교</a>에 장학금을 받고 입학. 1937년 졸업.</li>
<li><a href="/wiki/%EB%B8%8C%EB%9D%BC%EC%9A%B4_%EB%8C%80%ED%95%99%EA%B5%90" title="브라운 대학교">브라운 대학교</a>에서 쥐의 조건 반응과 관련된 연구로 박사 학위 취득.</li>
<li>1940년 코네티컷 여자 대학교에서 처음으로 교수직을 맡게 됨. <a href="/wiki/%EC%A0%9C2%EC%B0%A8_%EC%84%B8%EA%B3%84_%EB%8C%80%EC%A0%84" title="제2차 세계 대전">제2차 세계 대전</a>으로 연구 중단.</li>
<li>전쟁이 끝난 후, <a href="/wiki/%ED%94%8C%EB%A1%9C%EB%A6%AC%EB%8B%A4_%EC%A3%BC%EB%A6%BD_%EB%8C%80%ED%95%99%EA%B5%90" title="플로리다 주립 대학교">플로리다 주립 대학교</a>에서 임시 교수직을 맡음.</li>
<li>1949년 <a href="/wiki/%EB%AF%B8%EA%B5%AD" title="미국">미국</a> <a href="/wiki/%EA%B3%B5%EA%B5%B0" title="공군">공군</a> 엔진 기술 연구 책임자 자리를 맡음.</li>
<li>1958년 <a href="/wiki/%ED%94%84%EB%A6%B0%EC%8A%A4%ED%84%B4_%EB%8C%80%ED%95%99%EA%B5%90" title="프린스턴 대학교">프린스턴 대학교</a> 교수직으로 학계로 복귀함.</li>
<li>1962년 '학습의 조건' 저술.</li>
<li>1969년부터 <a href="/wiki/%ED%94%8C%EB%A1%9C%EB%A6%AC%EB%8B%A4_%EC%A3%BC%EB%A6%BD_%EB%8C%80%ED%95%99%EA%B5%90" title="플로리다 주립 대학교">플로리다 주립 대학교</a>에서 있으며 L. J. 브릭스의 도움으로 '학습의 조건' 개정판을 출판함.</li>
</ul>
