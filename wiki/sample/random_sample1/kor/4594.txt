<p><b>로버트 로스</b>(Robert Ross, 1766년 – 1814년 9월 12일)는 <a href="/wiki/%EB%82%98%ED%8F%B4%EB%A0%88%EC%98%B9_%EC%A0%84%EC%9F%81" title="나폴레옹 전쟁">나폴레옹 전쟁</a>과 <a class="mw-redirect" href="/wiki/%EB%AF%B8%EC%98%81%EC%A0%84%EC%9F%81" title="미영전쟁">미영전쟁</a>에 참가한 앵글로 아일랜드계 영국군 장교이다. 그는 <a href="/wiki/%EC%9B%8C%EC%8B%B1%ED%84%B4_%EB%B0%A9%ED%99%94" title="워싱턴 방화">워싱턴 방화</a>로 가장 잘 알려진 인물이며, <a href="/wiki/%EB%B0%B1%EC%95%85%EA%B4%80" title="백악관">백악관</a>과 <a class="mw-redirect" href="/wiki/%EB%AF%B8%EA%B5%AD_%EA%B5%AD%ED%9A%8C%EC%9D%98%EC%82%AC%EB%8B%B9" title="미국 국회의사당">미국 국회의사당</a>을 불태우고, 볼티모어에 투입은 실패로 돌아간다. 그는 불명예스러운 <a href="/wiki/%EB%A7%A5%ED%97%A8%EB%A6%AC_%EC%9A%94%EC%83%88" title="맥헨리 요새">맥헨리 요새</a> 포격 전날에 벌어진 <a class="new" href="/w/index.php?title=%EB%85%B8%EC%8A%A4%ED%8F%AC%EC%9D%B8%ED%8A%B8_%EC%A0%84%ED%88%AC&amp;action=edit&amp;redlink=1" title="노스포인트 전투 (없는 문서)">노스포인트 전투</a>에서 전사를 한다.</p>

<h2><span class="mw-headline" id=".EC.B4.88.EA.B8.B0_.EC.83.9D.EC.95.A0">초기 생애</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%A1%9C%EB%B2%84%ED%8A%B8_%EB%A1%9C%EC%8A%A4&amp;action=edit&amp;section=1" title="부분 편집: 초기 생애">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>로버트 로스는 <a href="/wiki/%EC%95%84%EC%9D%BC%EB%9E%9C%EB%93%9C_%EC%99%95%EA%B5%AD" title="아일랜드 왕국">아일랜드 왕국</a> 다운 군 노스트레버에서 <a href="/wiki/7%EB%85%84_%EC%A0%84%EC%9F%81" title="7년 전쟁">7년 전쟁</a>에서 장교를 지낸 아버지 데이비드 로스 소령과 어머니 샤르몽 배작의 누이 사이에서 태어났다. <a href="/wiki/%EB%8D%94%EB%B8%94%EB%A6%B0" title="더블린">더블린</a> <a class="mw-redirect mw-disambig" href="/wiki/%ED%8A%B8%EB%A6%AC%EB%8B%88%ED%8B%B0_%EC%BB%AC%EB%A6%AC%EC%A7%80" title="트리니티 컬리지">트리니티 컬리지</a>에서 교육을 받았으며, 컬리지 역사 학회의 총무를 맡았으며, 제25 보병연대의 수습장교로 <a href="/wiki/1789%EB%85%84" title="1789년">1789년</a>에 입대했다.<sup class="reference" id="cite_ref-autogenerated1_1-0"><a href="#cite_note-autogenerated1-1">[1]</a></sup></p>
<h2><span class="mw-headline" id=".EB.82.98.ED.8F.B4.EB.A0.88.EC.98.B9_.EC.A0.84.EC.9F.81">나폴레옹 전쟁</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%A1%9C%EB%B2%84%ED%8A%B8_%EB%A1%9C%EC%8A%A4&amp;action=edit&amp;section=2" title="부분 편집: 나폴레옹 전쟁">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>1799년 네덜란드에서 크레블댐 전투와 1801년 이집트에서 알렉산드리아 전투에 초급 장교로 싸웠다. 1803년 소령으로 진급하여 20보병연대를 맡게 된다. 그후 그는 1806년 나폴리 왕국 마이다에서 싸웠다. 1808년 말에 중령으로 진급하였고, 1809년 초 스페인 코루나 전투에 참전하였다. 1810년 로스는 대령 진급과 함께 옹의 참모가 되었다.</p>
<h3><span class="mw-headline" id=".EB.B0.98.EB.8F.84_.EC.A0.84.EC.9F.81">반도 전쟁</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%A1%9C%EB%B2%84%ED%8A%B8_%EB%A1%9C%EC%8A%A4&amp;action=edit&amp;section=3" title="부분 편집: 반도 전쟁">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p><a href="/wiki/1813%EB%85%84" title="1813년">1813년</a> <a href="/wiki/%EB%B0%98%EB%8F%84_%EC%A0%84%EC%9F%81" title="반도 전쟁">반도 전쟁</a>에서 <a href="/wiki/%EC%A0%9C1%EB%8C%80_%EC%9B%B0%EB%A7%81%ED%84%B4_%EA%B3%B5%EC%9E%91_%EC%95%84%EC%84%9C_%EC%9B%B0%EC%A6%90%EB%A6%AC" title="제1대 웰링턴 공작 아서 웰즐리">아서 웰슬리</a> 휘하로 파견되어 그해 <a href="/wiki/%EB%B9%84%ED%86%A0%EB%A6%AC%EC%95%84_%EC%A0%84%ED%88%AC" title="비토리아 전투">비토리아 전투</a>, 론세스발레 전투, 소라우렌 전투에서 그의 연대를 지휘했다. <a href="/wiki/1814%EB%85%84" title="1814년">1814년</a> <a href="/wiki/2%EC%9B%94_27%EC%9D%BC" title="2월 27일">2월 27일</a>, 오테스 전투에서 목좌측에 중상을 당해 후송되어 미국 파견군 사령관을 맡았다.</p>
<h2><span class="mw-headline" id="1812.EB.85.84_.EC.A0.84.EC.9F.81">1812년 전쟁</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%A1%9C%EB%B2%84%ED%8A%B8_%EB%A1%9C%EC%8A%A4&amp;action=edit&amp;section=4" title="부분 편집: 1812년 전쟁">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>로스는 전 영국 부대 총책임자를 맡아 소장으로 미국 동부 해안으로 향해했다. 그는 친히 영국 부대를 이끌고 메릴랜드 주 베네딕트에 상륙하여 <a href="/wiki/1814%EB%85%84" title="1814년">1814년</a> <a href="/wiki/8%EC%9B%94_24%EC%9D%BC" title="8월 24일">8월 24일</a> <a href="/wiki/%EB%B8%94%EB%9E%98%EB%8D%98%EC%8A%A4%EB%B2%84%EA%B7%B8_%EC%A0%84%ED%88%AC" title="블래던스버그 전투">블래던스버그 전투</a>에서 미군을 기습하기 위해 어퍼 말보로로 진군을 했다. 이 기습을 통해 도중에 급조된 미군 민병대를 붕괴시켰다.<sup class="reference" id="cite_ref-2"><a href="#cite_note-2">[2]</a></sup><sup class="reference" id="cite_ref-3"><a href="#cite_note-3">[3]</a></sup> 블래던스버그에서 계속 이동을 하여, 로스는 근처의 <a href="/wiki/%EC%9B%8C%EC%8B%B1%ED%84%B4_D.C." title="워싱턴 D.C.">워싱턴 D.C.</a>로 교전을 치뤘으며 그의 말이 총탄을 맞게 된다. 그는 캐나다 침공하여 <a href="/wiki/1813%EB%85%84" title="1813년">1813년</a> 미국이 요크 시(오늘 날의 토론토)를 불태운데 대한 보복으로 <a href="/wiki/%EB%B0%B1%EC%95%85%EA%B4%80" title="백악관">백악관</a>과 <a class="mw-redirect" href="/wiki/%EB%AF%B8%EA%B5%AD_%EA%B5%AD%ED%9A%8C%EC%9D%98%EC%82%AC%EB%8B%B9" title="미국 국회의사당">미국 국회의사당</a>과 같은 공공건물과 시설과 해군 조병창을 불태웠다. 공적 자산을 불태운 로스의 결정에 여전히 논란이 많지만, 그 방화 중에도 민간 자산은 보존해 주었다.<sup class="reference" id="cite_ref-autogenerated1_1-1"><a href="#cite_note-autogenerated1-1">[1]</a></sup></p>

<p>그후 로스는 매릴랜드 볼티모어를 습격하자고 설득을 했다. <a href="/wiki/1814%EB%85%84" title="1814년">1814년</a> <a href="/wiki/9%EC%9B%94_12%EC%9D%BC" title="9월 12일">9월 12일</a> 그의 부대는 시에서 12마일 떨어진 볼티모어 군의 노스 포인트 남동쪽에 있는 파탑스코 넥 반도의 남단에 상륙을 했다. <a href="/wiki/%EB%B3%BC%ED%8B%B0%EB%AA%A8%EC%96%B4_%EC%A0%84%ED%88%AC" title="볼티모어 전투">볼티모어 전투</a>의 일부로 도중에 노스포인트 전투라고 불리는 전투를 벌이게 되는데, 영국군은 미군 교전대를 마주쳤고, 로스 장군은 그의 부대를 이끌고 친히 전진을 했다. 미군 저격병이 그를 쏘아 오른쪽 팔을 뚫고 왼쪽 가슴을 쏘았다. 볼티모어 전설에 따르면, 당시 18살과 19살이었던 두 명의 미군 라이플 병인 대니얼 웰즈와 헨리 맥코마스가 각각 로스를 죽였다고 전한다. 로스는 함대로 이송되는 도중에 사망을 했다.</p>
<p>그의 죽음 이후, 장군의 시신은 토넌트 호의 자마이카 럼주를 담는 129 갤런 통에 보관이 되었다. 그 배가 다가올 전투를 위해 1815년 1월 뉴올리언스로 방향을 돌렸을때, 그의 시선은 영국 배 로열 오크 호에 선적되어 노바스코샤의 <a class="mw-disambig" href="/wiki/%ED%95%BC%EB%A6%AC%ED%8C%A9%EC%8A%A4" title="핼리팩스">핼리팩스</a>로 이송되었고, 그곳에서 그의 시신이 1814년 9월 29일에 오울드 베링 그라운드에 매장되었다.</p>
