<p><b>로버트 루빈</b>(Robert Rubin, 1938년 8월 29일~ )은 <a href="/wiki/%EB%AF%B8%EA%B5%AD" title="미국">미국</a>의 금융인이다. 1966년부터 1992년까지 <a class="mw-redirect" href="/wiki/%EA%B3%A8%EB%93%9C%EB%A7%8C_%EC%82%AD%EC%8A%A4" title="골드만 삭스">골드만 삭스</a>에서 일하며 회장을 역임하였으며, 이후 1993년부터 1995년까지 백악관 경제정책 보좌관, 1995년부터 1999년까지 <a href="/wiki/%EB%AF%B8%EA%B5%AD_%EC%9E%AC%EB%AC%B4%EB%B6%80" title="미국 재무부">재무장관</a>을 지냈다. 2008년 미국 금융위기의 주범 중 한명이라는 비난을 받기도 한다.<sup class="reference" id="cite_ref-1"><a href="#cite_note-1">[1]</a></sup></p>
<h2><span class="mw-headline" id=".EC.83.9D.EC.95.A0">생애</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%A1%9C%EB%B2%84%ED%8A%B8_%EB%A3%A8%EB%B9%88&amp;action=edit&amp;section=1" title="부분 편집: 생애">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p><a class="mw-redirect" href="/wiki/%EB%89%B4%EC%9A%95_%EC%8B%9C" title="뉴욕 시">뉴욕 시</a>에서 태어난 그는 마이애미 비치 고등학교를 졸업하고 <a href="/wiki/%ED%95%98%EB%B2%84%EB%93%9C_%EB%8C%80%ED%95%99%EA%B5%90" title="하버드 대학교">하버드 대학교</a>로 진학하여 1960년에 경제학 전공으로 <a class="mw-redirect" href="/wiki/%EC%88%98%EB%A7%88_%EC%BF%B0_%EB%9D%BC%EC%9A%B0%EB%8D%B0" title="수마 쿰 라우데">수마 쿰 라우데</a>(최우수) 졸업했다. 그후 그는 <a href="/wiki/%ED%95%98%EB%B2%84%EB%93%9C_%EB%A1%9C%EC%8A%A4%EC%BF%A8" title="하버드 로스쿨">하버드 로스쿨</a>에 입학하여 3일간 다니다가 견문을 넓히기 위해 학교를 <a href="/wiki/%EC%98%81%EA%B5%AD" title="영국">영국</a> <a href="/wiki/%EB%9F%B0%EB%8D%98_%EC%A0%95%EC%B9%98%EA%B2%BD%EC%A0%9C%EB%8C%80%ED%95%99%EA%B5%90" title="런던 정치경제대학교">LSE</a>로 유학을 갔다. 귀국한 후 <a href="/wiki/%EC%98%88%EC%9D%BC_%EB%B2%95%ED%95%99%EB%8C%80%ED%95%99%EC%9B%90" title="예일 법학대학원">예일 로스쿨</a>에 입학하여 3년후 1964년 법학 학사(LL.B.) 학위를 취득하였다.</p>
<p>첫 직장으로는 뉴욕에 위치한 <a href="/wiki/%ED%81%B4%EB%A6%AC%EC%96%B4%EB%A6%AC_%EA%B3%A0%ED%8B%80%EB%A6%BD" title="클리어리 고틀립">클리어리 고틀립</a> <a href="/wiki/%EB%A1%9C%ED%8E%8C" title="로펌">로펌</a>에서 변호사로 일했다. 1966년에 그는 <a href="/wiki/%ED%88%AC%EC%9E%90%EC%9D%80%ED%96%89" title="투자은행">투자은행</a> <a class="mw-redirect" href="/wiki/%EA%B3%A8%EB%93%9C%EB%A7%8C_%EC%82%AD%EC%8A%A4" title="골드만 삭스">골드만 삭스</a>에 입사했고 1990년부터 1992년까지 공동 회장과 공동 수석파트너 자리를 역임했다. 1993년부터 1995년까지 <a href="/wiki/%EB%B9%8C_%ED%81%B4%EB%A6%B0%ED%84%B4" title="빌 클린턴">빌 클린턴</a> 정부의 <a href="/wiki/%EB%AF%B8%EA%B5%AD_%EB%8C%80%ED%86%B5%EB%A0%B9_%EA%B2%BD%EC%A0%9C%EC%9E%90%EB%AC%B8%EC%9C%84%EC%9B%90%ED%9A%8C" title="미국 대통령 경제자문위원회">경제자문위원회</a>를 이끌었다. 1995년에는 재무장관으로 임명되었고 1999년에 <a href="/wiki/%EB%A1%9C%EB%9F%B0%EC%8A%A4_%EC%84%9C%EB%A8%B8%EC%8A%A4" title="로런스 서머스">로런스 서머스</a>가 그의 자리를 이었다. 2007년부터 외교평의회(Council on Foreign Relations, CFR) 회장으로 활동하고 있다.</p>
