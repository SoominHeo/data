<p><b>로버트 브라운</b>(Robert Brown, <a href="/wiki/1773%EB%85%84" title="1773년">1773년</a> <a href="/wiki/12%EC%9B%94_21%EC%9D%BC" title="12월 21일">12월 21일</a> ~ <a href="/wiki/1858%EB%85%84" title="1858년">1858년</a> <a href="/wiki/6%EC%9B%94_10%EC%9D%BC" title="6월 10일">6월 10일</a>)은 <a href="/wiki/%EC%98%81%EA%B5%AD" title="영국">영국</a>의 식물학자이다. <a class="mw-redirect" href="/wiki/%EC%97%90%EB%93%A0%EB%B2%84%EB%9F%AC_%EB%8C%80%ED%95%99" title="에든버러 대학">에든버러 대학</a> 의학부를 나와 <a href="/wiki/%EC%98%A4%EC%8A%A4%ED%8A%B8%EB%A0%88%EC%9D%BC%EB%A6%AC%EC%95%84" title="오스트레일리아">오스트레일리아</a>로 탐험을 가서 식물에 대한 조사를 하였다. 후에 <a class="mw-redirect" href="/wiki/%EB%8C%80%EC%98%81_%EB%B0%95%EB%AC%BC%EA%B4%80" title="대영 박물관">대영 박물관</a> 식물부 주임이 되었다. <a href="/wiki/1827%EB%85%84" title="1827년">1827년</a> 물에 넣은 꽃가루에서 나온 미세한 입자가 움직이는 것을 현미경으로 발견했는데, 이것은 움직이는 물 분자가 입자에 부딪혀 일어나는 것이다. 액체 속에서 움직이는 매우 작은 분자의 이런 움직임을 <a href="/wiki/%EB%B8%8C%EB%9D%BC%EC%9A%B4_%EC%9A%B4%EB%8F%99" title="브라운 운동">브라운 운동</a>이라 부른다. 이 이론은 후에 <a class="mw-redirect" href="/wiki/%EC%95%84%EC%9D%B8%EC%8A%88%ED%83%80%EC%9D%B8" title="아인슈타인">아인슈타인</a>에 의해 증명되었다. <a href="/wiki/1831%EB%85%84" title="1831년">1831년</a> <a href="/wiki/%EC%84%B8%ED%8F%AC" title="세포">세포</a> 속의 <a class="mw-redirect" href="/wiki/%EC%84%B8%ED%8F%AC%ED%95%B5" title="세포핵">핵</a>을 발견하였다.</p>
