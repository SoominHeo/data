<p><b>노트워시 컴포저</b>(NoteWorthy Composer, NWC)는 노트워시 소프트웨어(NoteWorthy Software)가 개발한 <a href="/wiki/%EC%95%85%EB%B3%B4_%EC%9E%91%EC%84%B1_%ED%94%84%EB%A1%9C%EA%B7%B8%EB%9E%A8" title="악보 작성 프로그램">악보 작성 프로그램</a>이다. 사용자가 원하는 악보를 만들어 재생 및 녹음, 편집, 인쇄 등의 기능을 지원한다.</p>
<p>노트워시 컴포저는 <a class="mw-redirect" href="/wiki/%EB%A7%88%EC%9D%B4%ED%81%AC%EB%A1%9C%EC%86%8C%ED%94%84%ED%8A%B8_%EC%9C%88%EB%8F%84" title="마이크로소프트 윈도">마이크로소프트 윈도</a>용 그래픽 악보 편집기로서 <a class="mw-redirect" href="/wiki/%EC%9C%88%EB%8F%84_95" title="윈도 95">윈도 95</a>부터 <a class="mw-redirect" href="/wiki/%EC%9C%88%EB%8F%84_7" title="윈도 7">윈도 7</a>까지 지원한다. <a href="/wiki/%EC%99%80%EC%9D%B8_(%EC%86%8C%ED%94%84%ED%8A%B8%EC%9B%A8%EC%96%B4)" title="와인 (소프트웨어)">와인</a>을 이용하면 <a href="/wiki/%EB%A6%AC%EB%88%85%EC%8A%A4" title="리눅스">리눅스</a>나 <a class="mw-redirect" href="/wiki/OS_X" title="OS X">OS X</a>에서도 이용할 수 있다는 보고가 있다. NWC의 버전 1은 1994년 10월에 출시되었으며 버전 2는 2008년 9월에 출시되었다.</p>
<p>NWC는 <a href="/wiki/%EC%95%85%EB%B3%B4" title="악보">악보</a> 작성을 위해 고안되었으나 <a class="mw-redirect" href="/wiki/%EB%AF%B8%EB%94%94" title="미디">미디</a>와 가라오케 파일을 가져오고 내보낼 수 있으며 그래픽 <a class="mw-redirect" href="/wiki/%EC%9C%88%EB%8F%84_%EB%A9%94%ED%83%80%ED%8C%8C%EC%9D%BC" title="윈도 메타파일">WMF</a>을 내보낼 수도 있다. <a href="/wiki/%EC%82%AC%EC%9A%A9%EC%9E%90_%EC%9D%B8%ED%84%B0%ED%8E%98%EC%9D%B4%EC%8A%A4" title="사용자 인터페이스">사용자 인터페이스</a>는 키보드나 마우스로 작업할 수 있게 구성되어 있다. 시각적인 결과가 바로 나타나며 언제든지 소리로 들을 수 있다. 설정이 제대로 되어 있다면 악보는 미디 장치를 재생하여 바로 입력할 수도 있다.</p>
<p>NWC <a href="/wiki/%ED%8C%8C%EC%9D%BC_%ED%98%95%EC%8B%9D" title="파일 형식">파일 형식</a>은 문서화되어 있지 않으며 텍스트 형식으로의 변환은 제한적이다. NWC 버전 2부터 텍스트 버전의 파일인 NWCTXT가 도입되었다. <a class="new" href="/w/index.php?title=GNU_LilyPond&amp;action=edit&amp;redlink=1" title="GNU LilyPond (없는 문서)">GNU LilyPond</a>와 같은 다른 형식으로의 변환에도 사용할 수 있다.<sup class="reference" id="cite_ref-1"><a href="#cite_note-1">[1]</a></sup></p>
<p>이 소프트웨어의 가격은 시벨리우스(Sibelius, 955달러)나 피날레(Finale, 600달러)보다도 훨씬 값이 싼 49달러로 구매할 수 있다. 완전한 기능의 <a href="/wiki/%EC%85%B0%EC%96%B4%EC%9B%A8%EC%96%B4" title="셰어웨어">데모 버전</a>도 제공되는데 저장이 불가능하고 각 인쇄 페이지마다 조그마한 꼬릿말이 추가되며 각 인쇄 악보마다 등록 폼을 인쇄한다. 데모 프로그램뿐 아니라 <a href="/wiki/%EC%9C%88%EC%95%B0%ED%94%84" title="윈앰프">윈앰프</a>용 플러그인도 제공되는데 이를 내려받아 이용하면 NWC가 내보낸 파일을 재생할 수 있다.</p>
<p>무료 뷰어도 제공된다. 이에 따라 노트워시는 대형 <a class="mw-redirect" href="/wiki/%EC%B0%AC%EC%86%A1%EA%B0%80" title="찬송가">찬송가</a> 데이터베이스 <a class="external text" href="http://www.hymntime.com/tch/" rel="nofollow">Cyber Hymnal</a>의 표준 악보 배포판으로 채택되었다.</p>
