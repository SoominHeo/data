<p><b>로버트 패리</b>(Robert Parry, <a href="/wiki/1933%EB%85%84" title="1933년">1933년</a> <a href="/wiki/1%EC%9B%94_8%EC%9D%BC" title="1월 8일">1월 8일</a> ~ <a href="/wiki/2000%EB%85%84" title="2000년">2000년</a> <a href="/wiki/3%EC%9B%94_9%EC%9D%BC" title="3월 9일">3월 9일</a>)는 <a href="/wiki/%EC%9E%89%EA%B8%80%EB%9E%9C%EB%93%9C" title="잉글랜드">영국 잉글랜드</a> <a href="/wiki/%EB%A8%B8%EC%A7%80%EC%82%AC%EC%9D%B4%EB%93%9C_%EC%A3%BC" title="머지사이드 주">머지사이드 주</a> <a href="/wiki/%EB%A6%AC%EB%B2%84%ED%92%80" title="리버풀">리버풀</a> <a href="/wiki/%EC%98%81%EA%B5%AD_%EC%9D%98%ED%9A%8C" title="영국 의회">의회의원</a>을 지낸 <a href="/wiki/%EC%98%81%EA%B5%AD" title="영국">영국</a>의 <a href="/wiki/%EC%A0%95%EC%B9%98%EA%B0%80" title="정치가">정치가</a> 겸 <a href="/wiki/%EC%82%AC%ED%9A%8C_%EC%9A%B4%EB%8F%99" title="사회 운동">시민사회운동가</a>이다.</p>
<h2><span class="mw-headline" id=".EC.83.9D.EC.95.A0">생애</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%A1%9C%EB%B2%84%ED%8A%B8_%ED%8C%A8%EB%A6%AC_(1933%EB%85%84)&amp;action=edit&amp;section=1" title="부분 편집: 생애">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>그는 <a href="/wiki/1952%EB%85%84" title="1952년">1952년</a>부터 <a href="/wiki/%EC%82%AC%ED%9A%8C_%EC%9A%B4%EB%8F%99" title="사회 운동">시민사회운동가</a>로 활약하다가 <a href="/wiki/1970%EB%85%84" title="1970년">1970년</a> <a href="/wiki/%EC%98%81%EA%B5%AD" title="영국">영국</a> <a href="/wiki/%EC%9E%89%EA%B8%80%EB%9E%9C%EB%93%9C" title="잉글랜드">잉글랜드</a> <a href="/wiki/%EB%A8%B8%EC%A7%80%EC%82%AC%EC%9D%B4%EB%93%9C_%EC%A3%BC" title="머지사이드 주">머지사이드 주</a> <a href="/wiki/%EB%A6%AC%EB%B2%84%ED%92%80" title="리버풀">리버풀</a> 의회의원에 당선되어 <a href="/wiki/1997%EB%85%84" title="1997년">1997년</a>까지 27년간 의회의원 직위에 재임하였고 의회의원 재임 시기이던 <a href="/wiki/1975%EB%85%84" title="1975년">1975년</a>에는 예비역 영국 해군 상사 <a class="mw-redirect" href="/wiki/%EC%95%A8%ED%94%84%EB%A0%9B_%EB%A0%88%EB%85%BC" title="앨프렛 레논">앨프렛 레논</a>(Alfred Lennon)을 예방(禮訪)한 적도 있으며 <a href="/wiki/1989%EB%85%84" title="1989년">1989년</a>에는 <a href="/wiki/%EC%A4%91%ED%99%94%EC%9D%B8%EB%AF%BC%EA%B3%B5%ED%99%94%EA%B5%AD" title="중화인민공화국">중화인민공화국</a> <a href="/wiki/%EB%B2%A0%EC%9D%B4%EC%A7%95_%EC%8B%9C" title="베이징 시">베이징</a>의 <a href="/wiki/1989%EB%85%84_%ED%86%88%EC%95%88%EB%A8%BC_%EC%82%AC%EA%B1%B4" title="1989년 톈안먼 사건">톈안먼 사건</a> 현장을 시찰하기도 하였다.</p>
