<p><b>2013 UEFA 챔피언스리그 결승전</b>은 <a class="mw-redirect" href="/wiki/UEFA_%EC%B1%94%ED%94%BC%EC%96%B8%EC%8A%A4_%EB%A6%AC%EA%B7%B8" title="UEFA 챔피언스 리그">UEFA 챔피언스 리그</a>의 58번째 결승전이자 현재의 대회명으로 개명 후 21번째 결승 경기이다. 경기는 2013년 5월 25일 토요일, <a href="/wiki/%EC%9E%89%EA%B8%80%EB%9E%9C%EB%93%9C" title="잉글랜드">잉글랜드</a>의 <a href="/wiki/%EC%9B%B8%EB%B8%94%EB%A6%AC_%EC%8A%A4%ED%83%80%EB%94%94%EC%9B%80" title="웸블리 스타디움">웸블리 스타디움</a>에서 열렸다.</p>
<p>이 대회는 처음으로 <a href="/wiki/%EB%8F%85%EC%9D%BC" title="독일">독일</a>에서 올라온 두 팀끼리 붙었으며, 챔피언스리그가 만들어진 후 같은 나라의 팀이 붙은 것은 <a href="/wiki/2008_UEFA_%EC%B1%94%ED%94%BC%EC%96%B8%EC%8A%A4%EB%A6%AC%EA%B7%B8_%EA%B2%B0%EC%8A%B9%EC%A0%84" title="2008 UEFA 챔피언스리그 결승전">2008 UEFA 챔피언스리그 결승전</a> 이후 4번째이다.</p>
<p>승리팀인 <a href="/wiki/FC_%EB%B0%94%EC%9D%B4%EC%97%90%EB%A5%B8_%EB%AE%8C%ED%97%A8" title="FC 바이에른 뮌헨">바이에른 뮌헨</a>은 <a href="/wiki/UEFA_%EC%8A%88%ED%8D%BC%EC%BB%B5" title="UEFA 슈퍼컵">UEFA 슈퍼컵</a> 2013에서 <a href="/wiki/UEFA_%EC%9C%A0%EB%A1%9C%ED%8C%8C%EB%A6%AC%EA%B7%B8_2012-13" title="UEFA 유로파리그 2012-13">UEFA 유로파리그 2012-13</a> 우승팀인 <a href="/wiki/%EC%B2%BC%EC%8B%9C_FC" title="첼시 FC">첼시 FC</a>와 경기를 펼치며 <a class="mw-redirect" href="/wiki/%EC%9C%A0%EB%9F%BD_%EC%B6%95%EA%B5%AC_%EC%97%B0%EB%A7%B9" title="유럽 축구 연맹">유럽 축구 연맹</a>을 대표하여 <a href="/wiki/FIFA_%ED%81%B4%EB%9F%BD_%EC%9B%94%EB%93%9C%EC%BB%B5_2013" title="FIFA 클럽 월드컵 2013">FIFA 클럽 월드컵 2013</a>에 참가하게 된다.</p>
