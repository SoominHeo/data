<p><b>프랑스의 로베르 2세</b>, 일명 <i>경건왕</i> 또는 <i>현명왕 로베르</i> (<a href="/wiki/%ED%94%84%EB%9E%91%EC%8A%A4%EC%96%B4" title="프랑스어">프랑스어</a>:<i>Robert II le Pieux</i> , <a href="/wiki/972%EB%85%84" title="972년">972년</a> <a href="/wiki/3%EC%9B%94_27%EC%9D%BC" title="3월 27일">3월 27일</a> – <a href="/wiki/1031%EB%85%84" title="1031년">1031년</a> <a href="/wiki/7%EC%9B%94_20%EC%9D%BC" title="7월 20일">7월 20일</a>)은, <a href="/wiki/%EC%9C%84%EA%B7%B8_%EC%B9%B4%ED%8E%98" title="위그 카페">위그 카페</a>와 아키텐 공 <a class="new" href="/w/index.php?title=%EC%95%84%ED%82%A4%ED%85%90%EC%9D%98_%EA%B8%B0%EC%9A%A4_3%EC%84%B8&amp;action=edit&amp;redlink=1" title="아키텐의 기욤 3세 (없는 문서)">기욤 3세</a>의 딸인 아델라이드의 아들이며, <a href="/wiki/%EC%98%A4%EB%A5%BC%EB%A0%88%EC%95%99" title="오를레앙">오를레앙</a>에서 태어났다. <a href="/wiki/988%EB%85%84" title="988년">988년</a>에 이탈리아 왕녀로 프란돌 백작인 아르눌프 2세의 미망인이었던 로잘라와 결혼했으나 <a href="/wiki/992%EB%85%84" title="992년">992년</a>에 이혼했고, <a href="/wiki/996%EB%85%84" title="996년">996년</a>에는 결혼할 수 없는 친척(사촌)인 부르고뉴의 왕녀이자 블루아 백작 <a class="new" href="/w/index.php?title=%EB%B8%94%EB%A3%A8%EC%95%84%EC%9D%98_%EC%98%A4%EB%8F%84_1%EC%84%B8&amp;action=edit&amp;redlink=1" title="블루아의 오도 1세 (없는 문서)">블루아의 오도 1세</a>의 미망인인 베르타와 결혼하여 일시적으로 <a href="/wiki/%EA%B5%90%ED%99%A9_%EA%B7%B8%EB%A0%88%EA%B3%A0%EB%A6%AC%EC%98%A4_5%EC%84%B8" title="교황 그레고리오 5세">교황 그레고리오 5세</a>에 의해 파문당했다. <a href="/wiki/%EA%B5%90%ED%99%A9_%EC%8B%A4%EB%B2%A0%EC%8A%A4%ED%85%8C%EB%A5%B4_2%EC%84%B8" title="교황 실베스테르 2세">교황 실베스테르 2세</a>와 교섭한 끝에, 혼인 무효가 선언되었다. <a href="/wiki/1001%EB%85%84" title="1001년">1001년</a>에 결혼한 아를 백작 기욤 1세의 딸인 콩스탕스와의 사이에서 알릭스, 위그, <a href="/wiki/%EC%95%99%EB%A6%AC_1%EC%84%B8" title="앙리 1세">앙리 1세</a>, 아델, 부르고뉴 공작 로베르 1세 등이 태어났다. 그는 극히 신앙심이 깊은 왕이었으며, 궁전의 한 쪽에 예배소를 만들고, 이교도에 대해 엄격한 태도를 취했다. <a href="/wiki/1016%EB%85%84" title="1016년">1016년</a>에는 부르고뉴를 손에 넣었다.</p>
