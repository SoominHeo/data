<p>로베르토 만치니는 포지션이 <a href="/wiki/%EA%B3%B5%EA%B2%A9%EC%88%98" title="공격수">스트라이커</a>였으며 <a href="/wiki/UC_%EC%82%BC%ED%94%84%EB%8F%84%EB%A6%AC%EC%95%84" title="UC 삼프도리아">UC 삼프도리아</a>와 <a href="/wiki/SS_%EB%9D%BC%EC%B9%98%EC%98%A4" title="SS 라치오">SS 라치오</a>에서 4차례의 <a href="/wiki/%EC%BD%94%ED%8C%8C_%EC%9D%B4%ED%83%88%EB%A6%AC%EC%95%84" title="코파 이탈리아">코파 이탈리아</a> 우승과 1차례의 리그 우승, 1991-92 <a class="mw-redirect" href="/wiki/%EC%9C%A0%EB%9F%AC%ED%94%BC%EC%96%B8%EC%BB%B5" title="유러피언컵">유러피언컵</a> 결승 진출에 일조하였다. <a href="/wiki/2001%EB%85%84" title="2001년">2001년</a> <a href="/wiki/%EB%A0%88%EC%8A%A4%ED%84%B0_%EC%8B%9C%ED%8B%B0_FC" title="레스터 시티 FC">레스터 시티</a>를 마지막으로 선수 생활을 마감하였다.</p>
<h2><span class="mw-headline" id=".EA.B0.90.EB.8F.85_.EA.B2.BD.EB.A0.A5">감독 경력</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%A1%9C%EB%B2%A0%EB%A5%B4%ED%86%A0_%EB%A7%8C%EC%B9%98%EB%8B%88&amp;action=edit&amp;section=2" title="부분 편집: 감독 경력">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p><a href="/wiki/ACF_%ED%94%BC%EC%98%A4%EB%A0%8C%ED%8B%B0%EB%82%98" title="ACF 피오렌티나">피오렌티나</a>에서 감독 생활을 시작하여 코파 이탈리아 우승으로 올려놓았다. 이후 <a href="/wiki/SS_%EB%9D%BC%EC%B9%98%EC%98%A4" title="SS 라치오">SS 라치오</a>를 이끌며 <a href="/wiki/2004%EB%85%84" title="2004년">2004년</a> 코파 이탈리아 대회 우승으로 장식하였으며, <a href="/wiki/2004%EB%85%84" title="2004년">2004년</a> <a href="/wiki/FC_%EC%9D%B8%ED%85%8C%EB%A5%B4%EB%82%98%EC%B9%98%EC%98%A4%EB%82%A0%EB%A0%88_%EB%B0%80%EB%9D%BC%EB%85%B8" title="FC 인테르나치오날레 밀라노">FC 인테르나치오날레 밀라노</a>의 감독에 부임하여 <a href="/wiki/2005%EB%85%84" title="2005년">2005년</a>부터 <a href="/wiki/2008%EB%85%84" title="2008년">2008년</a>까지 3시즌 연속 리그 우승과 2차례의 코파 이탈리아 우승, 2차례의 <a class="mw-redirect" href="/wiki/%EC%9D%B4%ED%83%88%EB%A6%AC%EC%95%84_%EC%8A%88%ED%8D%BC%EC%BB%B5" title="이탈리아 슈퍼컵">이탈리아 슈퍼컵</a> 우승 등을 이룩하여 이름값을 하였다. 그러나 <a href="/wiki/UEFA_%EC%B1%94%ED%94%BC%EC%96%B8%EC%8A%A4%EB%A6%AC%EA%B7%B8" title="UEFA 챔피언스리그">UEFA 챔피언스리그</a>에서 별다른 성과를 이루지 못하였기 때문에 2007-08 시즌 이후 경질되었다.<sup class="reference" id="cite_ref-1"><a href="#cite_note-1">[1]</a></sup></p>
<p><a href="/wiki/2009%EB%85%84" title="2009년">2009년</a> 12월에 <a href="/wiki/%EB%A7%A8%EC%B2%B4%EC%8A%A4%ED%84%B0_%EC%8B%9C%ED%8B%B0_FC" title="맨체스터 시티 FC">맨체스터 시티</a>의 감독으로 부임하여 팀을 지휘하고 있다. 그는 <a href="/wiki/%EB%A7%A8%EC%B2%B4%EC%8A%A4%ED%84%B0_%EC%8B%9C%ED%8B%B0_FC" title="맨체스터 시티 FC">맨체스터 시티 FC</a>의 고질적인 문제점으로 지적된 수비 불안을 보완하여, 초반부터 무실점 완승 행진을 거두었다.<sup class="reference" id="cite_ref-2"><a href="#cite_note-2">[2]</a></sup> <a href="/wiki/2012%EB%85%84" title="2012년">2012년</a> <a href="/wiki/5%EC%9B%94_14%EC%9D%BC" title="5월 14일">5월 14일</a>, 2011-2012시즌 마지막 38라운드 <a class="mw-redirect" href="/wiki/%ED%80%B8%EC%A6%88_%ED%8C%8C%ED%81%AC_%EB%A0%88%EC%9D%B8%EC%A0%80%EC%8A%A4_FC" title="퀸즈 파크 레인저스 FC">퀸즈 파크 레인저스 FC</a>와의 홈 경기 때 모두들 놀라움을 감출 수가 없었다. 전후반 90분이 지나고 1-2로 지고 있을 무렵, 추가 시간에 <a href="/wiki/%EC%97%90%EB%94%98_%EC%A0%9C%EC%BD%94" title="에딘 제코">에딘 제코</a>가 두 번째 골을 성공시켰다. 그 다음 <a class="mw-redirect" href="/wiki/%EC%84%B8%EB%A5%B4%ED%9E%88%EC%98%A4_%EC%95%84%EA%B5%AC%EC%97%90%EB%A1%9C" title="세르히오 아구에로">세르히오 아구에로</a>가 결승골인 세 번째 골을 성공시켰고 결국 <a href="/wiki/%EB%A7%A8%EC%B2%B4%EC%8A%A4%ED%84%B0_%EC%8B%9C%ED%8B%B0_FC" title="맨체스터 시티 FC">맨체스터 시티</a>가 <a href="/wiki/%ED%94%84%EB%A6%AC%EB%AF%B8%EC%96%B4%EB%A6%AC%EA%B7%B8" title="프리미어리그">프리미어리그</a>에서 우승하였다. 하지만 드라마틱한 우승에도 불구하고 경기력에 대한 의문은 꾸준히 제기되었으며 위건에게 2012/2013 FA컵 우승을 내주고 시즌을 무관으로 마치자 즉각 경질되었다. 그뒤 2013년 9월 터키 쉬페르리그의 명문구단 갈라타사라이의 감독으로 부임했다.<sup class="reference" id="cite_ref-3"><a href="#cite_note-3">[3]</a></sup></p>
<p>그러나 갈라타사라이에서 1년만에 나와야 했으며, 이후 <a href="/wiki/2014%EB%85%84" title="2014년">2014년</a> <a href="/wiki/11%EC%9B%94_15%EC%9D%BC" title="11월 15일">11월 15일</a> 그가 감독을 했었던 <a href="/wiki/FC_%EC%9D%B8%ED%85%8C%EB%A5%B4%EB%82%98%EC%B9%98%EC%98%A4%EB%82%A0%EB%A0%88_%EB%B0%80%EB%9D%BC%EB%85%B8" title="FC 인테르나치오날레 밀라노">FC 인테르나치오날레 밀라노</a>로 다시 돌아오게 되었다.</p>
