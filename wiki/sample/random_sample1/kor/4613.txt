<p><b>로베르토 비티엘로</b>(<span style="font-size: smaller;"><a href="/wiki/%EC%9D%B4%ED%83%88%EB%A6%AC%EC%95%84%EC%96%B4" title="이탈리아어">이탈리아어</a>:</span> <span lang="it" xml:lang="it">Roberto Vitiello</span>, <a href="/wiki/1983%EB%85%84" title="1983년">1983년</a> <a href="/wiki/5%EC%9B%94_8%EC%9D%BC" title="5월 8일">5월 8일</a>, <a href="/wiki/%EC%9D%B4%ED%83%88%EB%A6%AC%EC%95%84" title="이탈리아">이탈리아</a> <a href="/wiki/%EC%8A%A4%EC%B9%B4%ED%8C%8C%ED%8B%B0" title="스카파티">스카파티</a> ~)는 <a href="/wiki/%EC%84%B8%EB%A6%AC%EC%97%90_A" title="세리에 A">세리에 A</a>에 참가중인 <a class="mw-redirect" href="/wiki/US_%ED%8C%94%EB%A0%88%EB%A5%B4%EB%AA%A8" title="US 팔레르모">팔레르모</a>에서 뛰는 <a href="/wiki/%EC%9D%B4%ED%83%88%EB%A6%AC%EC%95%84" title="이탈리아">이탈리아</a>의 축구 선수이다.</p>
<h2><span class="mw-headline" id=".ED.81.B4.EB.9F.BD_.EA.B2.BD.EB.A0.A5">클럽 경력</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%A1%9C%EB%B2%A0%EB%A5%B4%ED%86%A0_%EB%B9%84%ED%8B%B0%EC%97%98%EB%A1%9C&amp;action=edit&amp;section=1" title="부분 편집: 클럽 경력">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>비티엘로는 <a class="mw-redirect" href="/wiki/%ED%8C%8C%EB%A5%B4%EB%A7%88_FC" title="파르마 FC">파르마</a> 소속으로 <a class="new" href="/w/index.php?title=%ED%86%A0%EB%A5%B4%EB%84%A4%EC%98%A4_%EB%94%94_%EB%B9%84%EC%95%84%EB%A0%88%EC%A1%B0&amp;action=edit&amp;redlink=1" title="토르네오 디 비아레조 (없는 문서)">토르네오 디 비아레조</a>에 참가하여 높은 평가를 받았고, 2002년 선수 생활을 시작했다.<sup class="reference" id="cite_ref-1"><a href="#cite_note-1">[1]</a></sup> 그는 등번호 34번을 받았다.<sup class="reference" id="cite_ref-2"><a href="#cite_note-2">[2]</a></sup></p>
<p><a class="mw-redirect" href="/wiki/%EC%84%B8%EB%A6%AC%EC%97%90_C1" title="세리에 C1">세리에 C1</a>의 <a href="/wiki/AC_%EC%B2%B4%EC%84%B8%EB%82%98" title="AC 체세나">체세나</a>에 임대된 후, 그는 2003년 7월에 <a class="new" href="/w/index.php?title=%EC%A3%BC%EC%84%B8%ED%8E%98_%EC%B9%B4%EB%A5%B4%EB%8F%84%EB%84%A4&amp;action=edit&amp;redlink=1" title="주세페 카르도네 (없는 문서)">주세페 카르도네</a>와 <a href="/wiki/%ED%81%AC%EB%A6%AC%EC%8A%A4%ED%8B%B0%EC%95%88_%EB%A7%88%EC%A1%B0" title="크리스티안 마조">크리스티안 마조</a>의 보상금의 일종으로 <a href="/wiki/%EC%84%B8%EB%A6%AC%EC%97%90_B" title="세리에 B">세리에 B</a>의 <a href="/wiki/%EB%B9%84%EC%B2%B8%EC%B0%A8_%EC%B9%BC%EC%B4%88" title="비첸차 칼초">비첸차</a>로 임대되었다. 2004년 여름에 그의 완전 영입이 이루어졌다. 2006년 7월, 그는 <a href="/wiki/%EC%84%B8%EB%A6%AC%EC%97%90_B" title="세리에 B">세리에 B</a>의 <a class="mw-redirect" href="/wiki/%EB%A6%AC%EB%AF%B8%EB%8B%88_%EC%B9%BC%EC%B4%88_FC" title="리미니 칼초 FC">리미니</a>로 이적하였다.<sup class="reference" id="cite_ref-3"><a href="#cite_note-3">[3]</a></sup><sup class="reference" id="cite_ref-4"><a href="#cite_note-4">[4]</a></sup></p>
<p>2010년 7월에 그는 세리에 B로 강등된 <a class="mw-redirect" href="/wiki/AC_%EC%8B%9C%EC%97%90%EB%82%98" title="AC 시에나">시에나</a>로 3년 계약을 맺고 이적하였다.<sup class="reference" id="cite_ref-5"><a href="#cite_note-5">[5]</a></sup></p>
<p>2014년 1월 <a class="mw-redirect" href="/wiki/US_%ED%8C%94%EB%A0%88%EB%A5%B4%EB%AA%A8" title="US 팔레르모">팔레르모</a>로 팀을 옮겼다.<sup class="reference" id="cite_ref-6"><a href="#cite_note-6">[6]</a></sup></p>
<h2><span class="mw-headline" id=".EC.9D.B4.ED.83.88.EB.A6.AC.EC.95.84_.EC.B6.95.EA.B5.AC_.EC.8A.A4.EC.BA.94.EB.93.A4">이탈리아 축구 스캔들</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%A1%9C%EB%B2%A0%EB%A5%B4%ED%86%A0_%EB%B9%84%ED%8B%B0%EC%97%98%EB%A1%9C&amp;action=edit&amp;section=2" title="부분 편집: 이탈리아 축구 스캔들">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
