<h1 class="firstHeading" id="firstHeading" lang="ko">로베르트 무질</h1>
<div class="mw-body-content" id="bodyContent">
<div id="siteSub">위키백과, 우리 모두의 백과사전.</div>
<div id="contentSub"></div>
<div class="mw-jump" id="jump-to-nav">
					이동:					<a href="#mw-head">둘러보기</a>, 					<a href="#p-search">검색</a>
</div>
<div class="mw-content-ltr" dir="ltr" id="mw-content-text" lang="ko"><script>function mfTempOpenSection(id){var block=document.getElementById("mf-section-"+id);block.className+=" open-block";block.previousSibling.className+=" open-block";}</script><p><b>로베르트 무질</b>(<span style="font-size: smaller;"><a href="/wiki/%EB%8F%85%EC%9D%BC%EC%96%B4" title="독일어">독일어</a>:</span> <span lang="de" xml:lang="de">Robert Musil</span> <a href="/wiki/1880%EB%85%84" title="1880년">1880년</a> <a href="/wiki/11%EC%9B%94_6%EC%9D%BC" title="11월 6일">11월 6일</a> <a href="/wiki/%EC%98%A4%EC%8A%A4%ED%8A%B8%EB%A6%AC%EC%95%84" title="오스트리아">오스트리아</a> <a href="/wiki/%ED%81%B4%EB%9D%BC%EA%B2%90%ED%91%B8%EB%A5%B4%ED%8A%B8" title="클라겐푸르트">클라겐푸르트</a> - <a href="/wiki/1942%EB%85%84" title="1942년">1942년</a> <a href="/wiki/4%EC%9B%94_15%EC%9D%BC" title="4월 15일">4월 15일</a> <a href="/wiki/%EC%8A%A4%EC%9C%84%EC%8A%A4" title="스위스">스위스</a> <a href="/wiki/%EC%A0%9C%EB%84%A4%EB%B0%94" title="제네바">제네바</a>)은 <a href="/wiki/%EC%98%A4%EC%8A%A4%ED%8A%B8%EB%A6%AC%EC%95%84" title="오스트리아">오스트리아</a>의 작가이다. 근대 소설의 주요 작품인 미완성 장편소설 《<a class="new" href="/w/index.php?title=%ED%8A%B9%EC%84%B1%EC%97%86%EB%8A%94_%EB%82%A8%EC%9E%90&amp;action=edit&amp;redlink=1" title="특성없는 남자 (없는 문서)">특성없는 남자</a>》(<span style="font-size: smaller;"><a href="/wiki/%EB%8F%85%EC%9D%BC%EC%96%B4" title="독일어">독일어</a>:</span> <span lang="de" xml:lang="de">Der Mann ohne Eigenschaften</span>)의 작가이다.</p>
<p>이 소설의 주인공인 울리히의 눈을 통해 본 <a class="mw-redirect" href="/wiki/%EC%98%A4%EC%8A%A4%ED%8A%B8%EB%A6%AC%EC%95%84-%ED%97%9D%EA%B0%80%EB%A6%AC" title="오스트리아-헝가리">오스트리아-헝가리 제국</a>의 도덕적 및 지적 쇠퇴를 다루고 있다. 배경은 <a href="/wiki/%EC%A0%9C1%EC%B0%A8_%EC%84%B8%EA%B3%84_%EB%8C%80%EC%A0%84" title="제1차 세계 대전">제1차 세계 대전</a> 직전의 <a class="mw-redirect" href="/wiki/%EB%B9%84%EC%97%94%EB%82%98" title="비엔나">비엔나</a>이다. 무질은 <a href="/wiki/1914%EB%85%84" title="1914년">1914년</a>에서 <a href="/wiki/1918%EB%85%84" title="1918년">1918년</a> 사이에 <a class="mw-redirect" href="/wiki/%EC%98%A4%EC%8A%A4%ED%8A%B8%EB%A6%AC%EC%95%84-%ED%97%9D%EA%B0%80%EB%A6%AC" title="오스트리아-헝가리">오스트리아-헝가리 제국</a> 육군에서 사무관으로 종사하였었다.</p>
<p>무질은 알프레드 무질(<span style="font-size: smaller;"><a href="/wiki/%EB%8F%85%EC%9D%BC%EC%96%B4" title="독일어">독일어</a>:</span> <span lang="de" xml:lang="de">Alfred Musil</span>, <a href="/wiki/1846%EB%85%84" title="1846년">1846년</a>-<a href="/wiki/1924%EB%85%84" title="1924년">1924년</a>)과 그의 아내 헤르미네(<span style="font-size: smaller;"><a href="/wiki/%EB%8F%85%EC%9D%BC%EC%96%B4" title="독일어">독일어</a>:</span> <span lang="de" xml:lang="de">Hermine</span>, <a href="/wiki/1853%EB%85%84" title="1853년">1853년</a>-<a href="/wiki/1924%EB%85%84" title="1924년">1924년</a>)의 아들이다. 이들은 혈연은 아니지만 "삼촌"이라 불리는 하인리히 라이터(<span style="font-size: smaller;"><a href="/wiki/%EB%8F%85%EC%9D%BC%EC%96%B4" title="독일어">독일어</a>:</span> <span lang="de" xml:lang="de">Heinrich Reiter</span>, <a href="/wiki/1856%EB%85%84" title="1856년">1856년</a>생)과 함께 살았다.</p>
