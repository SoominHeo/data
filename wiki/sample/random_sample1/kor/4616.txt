<p><b>하인리히 헤르만 로베르트 코흐</b>(Heinrich Hermann Robert Koch, <a href="/wiki/1843%EB%85%84" title="1843년">1843년</a> <a href="/wiki/12%EC%9B%94_11%EC%9D%BC" title="12월 11일">12월 11일</a> ~ <a href="/wiki/1910%EB%85%84" title="1910년">1910년</a> <a href="/wiki/5%EC%9B%94_27%EC%9D%BC" title="5월 27일">5월 27일</a>)은 <a href="/wiki/%EB%8F%85%EC%9D%BC" title="독일">독일</a>의 의사, 미생물학자이다. <a href="/wiki/%ED%83%84%EC%A0%80%EA%B7%A0" title="탄저균">탄저균</a>(1877년), <a href="/wiki/%EA%B2%B0%ED%95%B5%EA%B7%A0" title="결핵균">결핵균</a>(1882년), <a class="new" href="/w/index.php?title=%EC%BD%9C%EB%A0%88%EB%9D%BC%EA%B7%A0&amp;action=edit&amp;redlink=1" title="콜레라균 (없는 문서)">콜레라균</a>(1885년) 등을 발견한 것으로 유명하다. '세균학의 아버지'로 평가된다. 결핵균의 발견으로 1905년 <a href="/wiki/%EB%85%B8%EB%B2%A8_%EC%83%9D%EB%A6%AC%ED%95%99%C2%B7%EC%9D%98%ED%95%99%EC%83%81" title="노벨 생리학·의학상">노벨 생리학·의학상</a>을 수상했다.</p>

<h2><span class="mw-headline" id=".EC.83.9D.EC.95.A0">생애</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%A1%9C%EB%B2%A0%EB%A5%B4%ED%8A%B8_%EC%BD%94%ED%9D%90&amp;action=edit&amp;section=1" title="부분 편집: 생애">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p><a href="/wiki/%EB%8F%85%EC%9D%BC" title="독일">독일</a> 클라우스탈에서 <a href="/wiki/%EA%B4%91%EC%82%B0" title="광산">광산</a> 기사의 아들로 태어났다. <a href="/wiki/%EA%B4%B4%ED%8C%85%EA%B2%90_%EB%8C%80%ED%95%99%EA%B5%90" title="괴팅겐 대학교">괴팅겐 대학교</a>에서 <a class="new" href="/w/index.php?title=%ED%94%84%EB%A6%AC%EB%93%9C%EB%A6%AC%ED%9E%88_%EA%B5%AC%EC%8A%A4%ED%83%80%ED%94%84_%EC%95%BC%EC%BD%94%ED%94%84_%ED%97%A8%EB%A0%88&amp;action=edit&amp;redlink=1" title="프리드리히 구스타프 야코프 헨레 (없는 문서)">프리드리히 구스타프 야코프 헨레</a> 밑에서 <a href="/wiki/%EC%9D%98%ED%95%99" title="의학">의학</a>을 공부하고 1866년 졸업했는데, 의대생시절 <a href="/wiki/%EB%A3%A8%EC%9D%B4_%ED%8C%8C%EC%8A%A4%ED%87%B4%EB%A5%B4" title="루이 파스퇴르">루이 파스퇴르</a>의 <a href="/wiki/%EB%AF%B8%EC%83%9D%EB%AC%BC" title="미생물">미생물</a> 연구에 관심을 갖게 되어 <a class="mw-redirect" href="/wiki/%EB%B2%A0%EB%A5%BC%EB%A6%B0_%EB%8C%80%ED%95%99%EA%B5%90" title="베를린 대학교">베를린 대학교</a>에서 미생물연구를 하였다. <a href="/wiki/%EC%9D%98%EC%82%AC" title="의사">의사</a>자격시험에 합격, 의사가 된 코흐는 시골에서 근무하다가, <a href="/wiki/%ED%94%84%EB%A1%9C%EC%9D%B4%EC%84%BC-%ED%94%84%EB%9E%91%EC%8A%A4_%EC%A0%84%EC%9F%81" title="프로이센-프랑스 전쟁">프로이센-프랑스 전쟁</a>당시 군의관으로 근무하였다. 종전후 볼슈타인(지금의 Wolsztyn)에서 지역 의사로 일하면서 <a href="/wiki/%ED%83%84%EC%A0%80" title="탄저">탄저</a>에 대해 연구, 1876년 <a class="new" href="/w/index.php?title=%EB%B8%8C%EB%A0%88%EC%8A%AC%EB%9D%BC%EC%9A%B0_%EB%8C%80%ED%95%99%EA%B5%90&amp;action=edit&amp;redlink=1" title="브레슬라우 대학교 (없는 문서)">브레슬라우 대학교</a>에서 자신의 연구결과를 <a href="/wiki/%EA%B0%80%EC%B6%95" title="가축">가축</a>실험으로 발표하였다. 코흐의 연구결과는 여러 나라에 알려져, <a href="/wiki/%EB%AF%B8%EC%83%9D%EB%AC%BC%ED%95%99" title="미생물학">미생물학</a> 발전에 공헌하였다.</p>
<p>1881년 <a class="new" href="/w/index.php?title=%EB%B2%A0%EB%A5%BC%EB%A6%B0_%EA%B5%AD%EB%A6%BD_%EC%9C%84%EC%83%9D%EC%9B%90&amp;action=edit&amp;redlink=1" title="베를린 국립 위생원 (없는 문서)">베를린 국립 위생원</a>으로 자리를 옮긴 그는 세균의 표본 고정법·염색법·현미경 촬영법을 창시하였다. 이듬해 1882년 3월 24일 결핵균을 발견했으며, <a class="mw-redirect" href="/wiki/%EA%B3%B5%EA%B8%B0_(%EA%B8%B0%EC%B2%B4)" title="공기 (기체)">공기</a>를 통해 전염된다는 것도 알았다. 1883년에는 <a href="/wiki/%EC%9D%B4%EC%A7%91%ED%8A%B8" title="이집트">이집트</a>를 거쳐 <a href="/wiki/%EC%9D%B8%EB%8F%84" title="인도">인도</a>까지 여행하면서 콜레라 병원균을 발견했다. 1890년 결핵균에 대한 항원인 <a href="/wiki/%ED%88%AC%EB%B2%A0%EB%A5%B4%EC%BF%A8%EB%A6%B0" title="투베르쿨린">투베르쿨린</a>을 발견했다. 그러나 머지않아 투베르쿨린이 효과가 없음이 드러나자 학자로서 코흐의 명성은 일시적으로 하락하게 된다. 코흐는 그가 발견한 투베르쿨린의 개량을 위해 온힘을 쏟다가 그의 제자인 <a href="/wiki/%EC%97%90%EB%B0%80_%EC%95%84%EB%8F%8C%ED%94%84_%ED%8F%B0_%EB%B2%A0%EB%A7%81" title="에밀 아돌프 폰 베링">에밀 아돌프 폰 베링</a>에게 학자로서의 입지마저 위협당하게 된다. 결국 베링은 디프테리아에 대한 혈청 면역 연구로 1901년 최초의 노벨 생리학상을 수여받게 되는데, 이는 코흐보다 4년 먼저의 수상이었다. 그러나 이러한 베링의 수상은 논란의 여지가 많았던 것으로, 파상풍 혈청 면역 연구와 혐기성 세균의 순수 배양, 세균의 독소 생성 등 코흐의 또 다른 일본 출신 제자인 <a href="/wiki/%EA%B8%B0%ED%83%80%EC%82%AC%ED%86%A0_%EC%8B%9C%EB%B0%94%EC%82%AC%EB%B6%80%EB%A1%9C" title="기타사토 시바사부로">기타사토 시바사부로</a>가 이뤄낸 연구 성과의 역할과 중요성이 수상자 결정에서 무시되었다는 의혹이 일고 있다.</p>
<p>1891년에는 베를린 국립전염병연구소의 초대 소장이 되었으며, 1905년 세균학 창시에 대한 공로로 <a href="/wiki/%EB%85%B8%EB%B2%A8_%EC%83%9D%EB%A6%AC%ED%95%99%C2%B7%EC%9D%98%ED%95%99%EC%83%81" title="노벨 생리학·의학상">노벨 생리학·의학상</a> 생리학 부문 상을 받았다. 1910년 바덴바덴에서 별세하였다.</p>
