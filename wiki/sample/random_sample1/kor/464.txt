<h2><span class="mw-headline" id=".ED.8C.8C.EC.9D.BC_.EA.B5.AC.EC.A1.B0">파일 구조</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=EAR_(%ED%8C%8C%EC%9D%BC_%ED%8F%AC%EB%A7%B7)&amp;action=edit&amp;section=1" title="부분 편집: 파일 구조">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>EAR 파일은 .ear 확장자를 가진 표준 <a href="/wiki/JAR_(%ED%8C%8C%EC%9D%BC_%ED%8F%AC%EB%A7%B7)" title="JAR (파일 포맷)">JAR 파일</a>(ZIP 파일)이며, 애플리케이션 모듈을 대표하는 하나 이상의 엔트리 및 하나 이상의 배치 기술자를 포함하는 META-INF라는 이름의 메타데이터 디렉터리가 있다.</p>
<h3><span class="mw-headline" id=".EB.AA.A8.EB.93.88">모듈</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=EAR_(%ED%8C%8C%EC%9D%BC_%ED%8F%AC%EB%A7%B7)&amp;action=edit&amp;section=2" title="부분 편집: 모듈">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<ul>
<li>웹 모듈은 <a href="/wiki/WAR_(%ED%8C%8C%EC%9D%BC_%ED%8F%AC%EB%A7%B7)" title="WAR (파일 포맷)">.war</a> 확장자를 지닌다.</li>
<li><a class="mw-redirect" href="/wiki/POJO" title="POJO">POJO</a> 자바 클래스는 <a class="mw-redirect" href="/wiki/.jar" title=".jar">.jar</a> 파일 안에 배치될 수 있다. 하나 이상의 웹 구성 요소, 다른 리소스, <a href="/wiki/%EC%9B%B9_%EC%95%A0%ED%94%8C%EB%A6%AC%EC%BC%80%EC%9D%B4%EC%85%98" title="웹 애플리케이션">웹 애플리케이션</a> <a class="new" href="/w/index.php?title=%EB%B0%B0%EC%B9%98_%EC%84%9C%EC%88%A0%EC%9E%90&amp;action=edit&amp;redlink=1" title="배치 서술자 (없는 문서)">배치 서술자</a>로 이루어진 배치 가능한 단위이다. 이 웹 모듈은 표준 웹 애플리케이션 포맷 내의 디렉터리, 파일 <a href="/wiki/%EA%B3%84%EC%B8%B5" title="계층">계층</a> 안에 포함된다.</li>
<li><a href="/wiki/%EC%97%94%ED%84%B0%ED%94%84%EB%9D%BC%EC%9D%B4%EC%A6%88_%EC%9E%90%EB%B0%94%EB%B9%88%EC%A6%88" title="엔터프라이즈 자바빈즈">엔터프라이즈 자바빈즈</a> 모듈은 <a class="mw-redirect" href="/wiki/.jar" title=".jar">.jar</a> 확장자를 지니며 영구적으로 배치된 클래스를 기술하는 자신만의 META-INF 디렉터리 서술자가 있다.</li>
<li><a href="/wiki/JCA" title="JCA">리소스 어댑터</a> 모듈은 .rar 확장자를 지닌다.</li>
</ul>
<h3><span class="mw-headline" id="META-INF_.EB.94.94.EB.A0.89.ED.84.B0.EB.A6.AC">META-INF 디렉터리</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=EAR_(%ED%8C%8C%EC%9D%BC_%ED%8F%AC%EB%A7%B7)&amp;action=edit&amp;section=3" title="부분 편집: META-INF 디렉터리">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>META-INF 디렉터리에는 적어도 자바 EE 배치 서술자(Java EE Deployment Descriptor)로 알려져 있는 application.xml 배치 서술자가 포함된다. 여기에는 다음의 XML 엔티티가 포함된다:</p>
<ul>
<li><code>icon</code>: 애플리케이션을 대표하는 이미지를 위한 위치를 지정한다. 하부 구역은 <code>small-icon</code>, <code>large-icon</code>을 위해 생성된다.</li>
<li><code>display-name</code>: 애플리케이션을 식별한다.</li>
<li><code>description</code></li>
<li>각 아카이브 내의 <code>module</code> 요소</li>
<li>0개 이상의 <code>security-role</code> 요소: 애플리케이션 내의 전역 보안 역할</li>
</ul>
