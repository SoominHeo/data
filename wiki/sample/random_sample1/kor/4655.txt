<p><b>로저 킹덤</b>(Roger Kingdom, <a href="/wiki/1962%EB%85%84" title="1962년">1962년</a> <a href="/wiki/8%EC%9B%94_26%EC%9D%BC" title="8월 26일">8월 26일</a> ~ )은 <a href="/wiki/%EB%AF%B8%EA%B5%AD" title="미국">미국</a>의 허들 선수로 <a href="/wiki/%ED%95%98%EA%B3%84_%EC%98%AC%EB%A6%BC%ED%94%BD" title="하계 올림픽">하계 올림픽</a> 110m 허들 2연승(1984, 1988)을 하였다.</p>
<p><a href="/wiki/%EC%A1%B0%EC%A7%80%EC%95%84_%EC%A3%BC" title="조지아 주">조지아 주</a> 비엔나에서 태어난 킹덤은 <a href="/wiki/%EB%86%92%EC%9D%B4%EB%9B%B0%EA%B8%B0" title="높이뛰기">높이뛰기</a>와 <a href="/wiki/%EC%9B%90%EB%B0%98%EB%8D%98%EC%A7%80%EA%B8%B0" title="원반던지기">원반던지기</a>에 뛰어나면서 두드러진 <a href="/wiki/%EB%AF%B8%EC%8B%9D%EC%B6%95%EA%B5%AC" title="미식축구">미식축구</a> 선수로도 알려졌다. 미식축구 장학금을 받아 <a href="/wiki/%ED%94%BC%EC%B8%A0%EB%B2%84%EA%B7%B8_%EB%8C%80%ED%95%99%EA%B5%90" title="피츠버그 대학교">피츠버그 대학교</a>에서 수학한 그는 <a href="/wiki/1983%EB%85%84" title="1983년">1983년</a> 110m 허들 NCAA 실외 타이틀과 이듬해 NCAA 실내 대회 55m 허들을 우승하였다. 그는 아직도 <a href="/wiki/%ED%94%BC%EC%B8%A0%EB%B2%84%EA%B7%B8" title="피츠버그">피츠버그</a>의 외곽 먼로빌에 살고 있다.</p>
<p>그는 110m 허들에서 육상에 두드러진 장기적 경력을 가지면서 하계 올림픽 2연속 <a href="/wiki/%EA%B8%88%EB%A9%94%EB%8B%AC" title="금메달">금메달</a>을 획득하였다. <a href="/wiki/1988%EB%85%84" title="1988년">1988년</a> 전 시즌을 통하여 패배한 적이 없고 <a href="/wiki/1988%EB%85%84_%ED%95%98%EA%B3%84_%EC%98%AC%EB%A6%BC%ED%94%BD" title="1988년 하계 올림픽">서울 올림픽</a>에서 자신의 올림픽 타이틀을 유지하는 데 호의를 얻었다. 기술, 힘과 속력의 매력적인 표명으로 3m 차이로 우승한 킹덤은 12.98초에 달려 13초 아래로 달린 첫 선수가 되었다. 이 기록은 <a class="mw-redirect" href="/wiki/1996%EB%85%84_%EC%95%A0%ED%8B%80%EB%9E%9C%ED%83%80_%EC%98%AC%EB%A6%BC%ED%94%BD" title="1996년 애틀랜타 올림픽">1996년 애틀랜타 올림픽</a>에서 <a href="/wiki/%EC%95%A8%EB%9F%B0_%EC%A1%B4%EC%8A%A8" title="앨런 존슨">앨런 존슨</a>에 의하여 경신될 때까지 보유되었다. 킹덤은 <a class="mw-redirect" href="/wiki/%EB%A6%AC_%EC%BB%AC%ED%9B%88" title="리 컬훈">리 컬훈</a>에 이어 110m 허들의 올림픽 타이틀을 성공적으로 유지한 두 번째 선수가 되었다.</p>
<p><a href="/wiki/1989%EB%85%84" title="1989년">1989년</a> <a href="/wiki/%EC%B7%A8%EB%A6%AC%ED%9E%88" title="취리히">취리히</a>에서 110m 허들의 세계 기록 12.92초를 세웠다. 이 기록은 <a href="/wiki/1993%EB%85%84" title="1993년">1993년</a> <a href="/wiki/8%EC%9B%94_20%EC%9D%BC" title="8월 20일">8월 20일</a> <a href="/wiki/%EC%BD%9C%EB%A6%B0_%EC%9E%AD%EC%8A%A8" title="콜린 잭슨">콜린 잭슨</a>에 의하여 1과 100분의 1초 차이로 깨질 때까지 보유되었다.</p>
<p><a href="/wiki/1991%EB%85%84" title="1991년">1991년</a> 자신의 <a href="/wiki/%EC%9D%B8%EB%8C%80" title="인대">인대</a> 손해를 회복하고 <a href="/wiki/%EB%AC%B4%EB%A6%8E" title="무릎">무릎</a>으로부터 뼈조각을 치우는 데 수술을 받을 때 그의 진보가 훼방놓이고 말았다.</p>
<p>현재 <a href="/wiki/%ED%8E%9C%EC%8B%A4%EB%B2%A0%EC%9D%B4%EB%8B%88%EC%95%84_%EC%A3%BC" title="펜실베이니아 주">펜실베이니아 주</a> 캘리포니아 대학교의 육상 총감독은 물론, 육상과 크로스컨트리 관리자로 있다.</p>
