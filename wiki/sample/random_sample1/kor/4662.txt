<p><b>로즈 장학금</b>(Rhodes Scholarship)은 <a href="/wiki/%EC%98%81%EA%B5%AD" title="영국">영국</a> 자선사업가이자 <a class="new" href="/w/index.php?title=%EC%A0%9C%EA%B5%AD%EC%A3%BC%EC%9D%98%EC%9E%90&amp;action=edit&amp;redlink=1" title="제국주의자 (없는 문서)">제국주의자</a>였던 <a href="/wiki/%EC%84%B8%EC%8B%A4_%EB%A1%9C%EC%A6%88" title="세실 로즈">세실 로즈</a>(Cecil Rhodes)의 유언에 의해 설립된 로즈 장학재단(The Rhodes Trust)에서 주어진다. 매년 <a href="/wiki/%EB%AF%B8%EA%B5%AD" title="미국">미국</a>·<a href="/wiki/%EB%8F%85%EC%9D%BC" title="독일">독일</a>·<a class="mw-redirect" href="/wiki/%EC%98%81%EC%97%B0%EB%B0%A9" title="영연방">영연방</a> 국가의 젊은이 85여 명을 선발해 영국 <a href="/wiki/%EC%98%A5%EC%8A%A4%ED%8D%BC%EB%93%9C_%EB%8C%80%ED%95%99%EA%B5%90" title="옥스퍼드 대학교">옥스퍼드 대학교</a>에서 무료로 공부할 기회를 주는 제도이다.<sup class="reference" id="cite_ref-1"><a href="#cite_note-1">[1]</a></sup></p>
<p>로즈 장학금은 세계에서 가장 영예로운 <a href="/wiki/%EC%9E%A5%ED%95%99%EA%B8%88" title="장학금">장학금</a> 중 하나로 인정받고 있으며<sup class="reference" id="cite_ref-2"><a href="#cite_note-2">[2]</a></sup>, 선발된 학생들은 평생 '로즈 장학생'이었다는 말이 따라다닐 정도로 사회적, 학문적 인지도가 높다.</p>

<p>1902년에 시작된 이 국제 장학금 프로그램은 역사가 1백 년이 넘었으며<sup class="reference" id="cite_ref-19"><a href="#cite_note-19">[19]</a></sup>, 선발과정이 까다롭기로 유명하다. 이렇게 뽑힌 로즈 <a class="new" href="/w/index.php?title=%EC%9E%A5%ED%95%99%EC%83%9D&amp;action=edit&amp;redlink=1" title="장학생 (없는 문서)">장학생</a>(Rhodes Scholar)들은 영국 옥스퍼드 대학에서 무료로 공부할 수 있는 특혜가 주어진다. 이들에게는 전공별로 2~3년간 <a class="mw-redirect" href="/wiki/%EC%98%A5%EC%8A%A4%ED%8D%BC%EB%93%9C_%EB%8C%80%ED%95%99" title="옥스퍼드 대학">옥스퍼드 대학</a>에서의 학비는 물론 기숙사 시설과 생활비, 심지어는 여행비까지 일체가 지급된다. 이 기간동안 학생들은 옥스퍼드에서 어떤 대학원에서든지 공부하고 졸업할 수 있으며, 원하면 옥스퍼드 학부 졸업장도 얻을 수 있다.</p>
<p>매년 <a href="/wiki/%EB%AF%B8%EA%B5%AD" title="미국">미국</a>에서 32명의 <a class="mw-redirect" href="/wiki/%EB%8C%80%ED%95%99%EC%83%9D" title="대학생">대학생</a>들이 선발되며, 그외에도 <a href="/wiki/%EC%BA%90%EB%82%98%EB%8B%A4" title="캐나다">캐나다</a>(11명), <a class="mw-redirect" href="/wiki/%EB%82%A8%EC%95%84%ED%94%84%EB%A6%AC%EC%B9%B4%EA%B3%B5%ED%99%94%EA%B5%AD" title="남아프리카공화국">남아프리카공화국</a>(10명), <a href="/wiki/%EC%98%A4%EC%8A%A4%ED%8A%B8%EB%A0%88%EC%9D%BC%EB%A6%AC%EC%95%84" title="오스트레일리아">오스트레일리아</a>(9명), <a href="/wiki/%EC%9D%B8%EB%8F%84" title="인도">인도</a>(5명), <a href="/wiki/%EB%89%B4%EC%A7%88%EB%9E%9C%EB%93%9C" title="뉴질랜드">뉴질랜드</a>(3명), <a href="/wiki/%EB%8F%85%EC%9D%BC" title="독일">독일</a>(2명), <a href="/wiki/%EB%B2%84%EB%AE%A4%EB%8B%A4" title="버뮤다">버뮤다</a>, <a href="/wiki/%ED%99%8D%EC%BD%A9" title="홍콩">홍콩</a>, <a href="/wiki/%EC%9E%90%EB%A9%94%EC%9D%B4%EC%B9%B4" title="자메이카">자메이카</a>, <a href="/wiki/%EC%BC%80%EB%83%90" title="케냐">케냐</a>, <a href="/wiki/%ED%8C%8C%ED%82%A4%EC%8A%A4%ED%83%84" title="파키스탄">파키스탄</a>, <a href="/wiki/%EC%A7%90%EB%B0%94%EB%B8%8C%EC%9B%A8" title="짐바브웨">짐바브웨</a>, <a href="/wiki/%EC%9E%A0%EB%B9%84%EC%95%84" title="잠비아">잠비아</a> 등에서 51여 명이 선발돼 함께 옥스퍼드 대학에 입학하게 된다. 세실 로즈의 유언에 따라 <a href="/wiki/%EB%AF%B8%EA%B5%AD" title="미국">미국</a>·<a href="/wiki/%EB%8F%85%EC%9D%BC" title="독일">독일</a>·<a class="mw-redirect" href="/wiki/%EC%98%81%EC%97%B0%EB%B0%A9" title="영연방">영연방</a>의 젊은이들을 대상으로 하는 장학금 제도이므로, 선발 과정은 이들 지역에 국한되어 있어 <a href="/wiki/%EB%8C%80%ED%95%9C%EB%AF%BC%EA%B5%AD" title="대한민국">대한민국</a> 국적의 학생들은 지원할 수 없다. <a href="/wiki/%EB%AF%B8%EA%B5%AD" title="미국">미국</a> 국적의 <a href="/wiki/%ED%95%9C%EA%B5%AD%EA%B3%84_%EB%AF%B8%EA%B5%AD%EC%9D%B8" title="한국계 미국인">한국</a> 학생들이 선발된 사례는 있다.</p>
<h2><span class="mw-headline" id=".EC.84.A0.EB.B0.9C_.EA.B8.B0.EC.A4.80">선발 기준</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%A1%9C%EC%A6%88_%EC%9E%A5%ED%95%99%EA%B8%88&amp;action=edit&amp;section=2" title="부분 편집: 선발 기준">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>로즈 장학생 선발은 그 선발 기준이 일반적인 장학재단과는 적지 않은 차이를 두고 있다. <a class="new" href="/w/index.php?title=%EB%A1%9C%EC%A6%88_%EC%9E%A5%ED%95%99%EC%9E%AC%EB%8B%A8&amp;action=edit&amp;redlink=1" title="로즈 장학재단 (없는 문서)">로즈 장학재단</a>은 장학생을 뽑을때 우수한 학업성적 외에도 인격과 봉사활동, 특정 주제에의 열정적 관심에 많은 무게를 둔다. 로즈장학생에 지원하려면 최소한 추천서만 8장이 필요하며 전공분야의 탁월성, 남다른 <a class="mw-redirect" href="/wiki/%EB%B4%89%EC%82%AC%ED%99%9C%EB%8F%99" title="봉사활동">봉사활동</a>, <a class="mw-redirect" href="/wiki/%EB%A6%AC%EB%8D%94%EC%8B%AD" title="리더십">리더십</a>, <a href="/wiki/%EC%8A%A4%ED%8F%AC%EC%B8%A0" title="스포츠">스포츠</a> 등 지덕체가 모두 요구되므로 보통 1학년때부터 신경써야 할 정도로 경쟁률이 심하다.</p>
<p>로즈 장학재단은 장학생을 뽑을때 "높은 학업 성취도는 기본이고 성실성과 진지함, 이타적 자세, 타인과 세상에 대한 진지한 관심, 육체적 활력을 중시한다"고 한다.</p>
<p>미국 내에서 로즈 장학생에 지원하려면 우선 각 대학에서 추천을 받아 주별로 결성된 로즈 장학금 위원회에서 뽑혀야 한다. 이 심사위원들과 함께 하는 칵테일 디너 파티에서 말솜씨와 인품, 매너 등에서 좋은 인상을 주어야 하며 그 후 정식인터뷰를 통해 각 주대표로 뽑히면 다시 미 50개 주를 8개 지역으로 나눠 각 지역위원회로부터 최종 인터뷰를 합격하면 미국 전체에서 32명의 장학생이 결정된다. 미국에서는 매년 최고의 엘리트 대학생 1000여명이 로즈장학금에 지원하는데 이 중 32명만이 최종 선발되는 영광을 누리며, 여기에 몇 명이 선발되느냐에 따라 대학의 평판과 후원금 규모가 달라질 정도로 상징성과 영향력이 큰 장학금이다.</p>
<h2><span class="mw-headline" id=".EB.A1.9C.EC.A6.88_.EC.9E.A5.ED.95.99.EC.83.9D_.EC.B6.9C.EC.8B.A0.EC.9D.98_.EC.A7.80.EB.8F.84.EC.9E.90.EB.93.A4">로즈 장학생 출신의 지도자들</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%A1%9C%EC%A6%88_%EC%9E%A5%ED%95%99%EA%B8%88&amp;action=edit&amp;section=3" title="부분 편집: 로즈 장학생 출신의 지도자들">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p><a href="/wiki/%EB%B9%8C_%ED%81%B4%EB%A6%B0%ED%84%B4" title="빌 클린턴">빌 클린턴</a> 전 미국 대통령을 비롯해 <a class="mw-redirect" href="/wiki/%EB%B3%B4%EB%B9%84_%EC%A7%84%EB%8B%AC" title="보비 진달">보비 진달</a> 현 미국 <a href="/wiki/%EB%A3%A8%EC%9D%B4%EC%A7%80%EC%95%A0%EB%82%98_%EC%A3%BC" title="루이지애나 주">루이지애나 주</a> 주지사, 웨슬리 클라크 전 <a class="mw-redirect" href="/wiki/%EB%82%98%ED%86%A0" title="나토">나토</a> 사령관, <a href="/wiki/%EB%B0%A5_%ED%98%B8%ED%81%AC" title="밥 호크">밥 호크</a> 전 호주 총리, 바이런 와이트 전 <a class="new" href="/w/index.php?title=%EB%AF%B8%EA%B5%AD_%EC%97%B0%EB%B0%A9%EB%8C%80%EB%B2%95%EA%B4%80&amp;action=edit&amp;redlink=1" title="미국 연방대법관 (없는 문서)">미국 연방대법관</a>, <a class="new" href="/w/index.php?title=%EB%8D%B0%EC%9D%B4%EB%B9%84%EB%93%9C_%EC%84%9C%ED%84%B0&amp;action=edit&amp;redlink=1" title="데이비드 서터 (없는 문서)">데이비드 서터</a> 현 미국 연방대법관, <a class="new" href="/w/index.php?title=%EB%9F%AC%EC%8A%A4_%ED%8C%8C%EC%9D%B8%EA%B3%A8%EB%93%9C&amp;action=edit&amp;redlink=1" title="러스 파인골드 (없는 문서)">러스 파인골드</a> 현 미국 상원의원, 빌 브래들리 전 상원의원, 한국 대학생들에게도 잘 알려져있는 <a class="new" href="/w/index.php?title=%ED%92%80%EB%B8%8C%EB%9D%BC%EC%9D%B4%ED%8A%B8&amp;action=edit&amp;redlink=1" title="풀브라이트 (없는 문서)">풀브라이트</a> 장학재단(Fulbright Fellowships)을 설립한 <a class="new" href="/w/index.php?title=J._%EC%9C%8C%EB%A6%AC%EC%97%84_%ED%92%80%EB%B8%8C%EB%9D%BC%EC%9D%B4%ED%8A%B8&amp;action=edit&amp;redlink=1" title="J. 윌리엄 풀브라이트 (없는 문서)">J. 윌리엄 풀브라이트</a> 전 상원의원, 시사 주간지 &lt;<a href="/wiki/%ED%83%80%EC%9E%84_(%EC%9E%A1%EC%A7%80)" title="타임 (잡지)">타임</a>&gt;의 편집국장이었던 <a href="/wiki/%EC%9B%94%ED%84%B0_%EC%95%84%EC%9D%B4%EC%9E%91%EC%8A%A8" title="월터 아이작슨">월터 아이작슨</a> 등 각 분야의 <a class="mw-redirect" href="/wiki/%EC%A7%80%EB%8F%84%EC%9E%90" title="지도자">지도자</a>들이 로즈 장학금을 받았을 만큼 이 프로그램을 통해 세계적인 지도자들이 많이 배출됐다.</p>
