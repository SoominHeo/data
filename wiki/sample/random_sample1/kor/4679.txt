<p><b>로토 다리</b> (<span style="font-size: smaller;"><a href="/wiki/%EC%9D%B4%ED%83%88%EB%A6%AC%EC%95%84%EC%96%B4" title="이탈리아어">이탈리아어</a>:</span> <span lang="it" xml:lang="it">Ponte Rotto</span>)는 과거 <b>아이밀리우스 다리</b>(<span style="font-size: smaller;"><a href="/wiki/%EB%9D%BC%ED%8B%B4%EC%96%B4" title="라틴어">라틴어</a>:</span> <span lang="la" xml:lang="la">Pons Aemilius</span>, <span style="font-size: smaller;"><a href="/wiki/%EC%9D%B4%ED%83%88%EB%A6%AC%EC%95%84%EC%96%B4" title="이탈리아어">이탈리아어</a>:</span> <span lang="it" xml:lang="it">Ponte Emilio</span>)라고 불린 <a href="/wiki/%EC%9D%B4%ED%83%88%EB%A6%AC%EC%95%84" title="이탈리아">이탈리아</a> <a href="/wiki/%EB%A1%9C%EB%A7%88" title="로마">로마</a>에 있는 옛 <a class="new" href="/w/index.php?title=%EA%B3%A0%EB%8C%80_%EB%A1%9C%EB%A7%88%EC%9D%98_%EB%8B%A4%EB%A6%AC&amp;action=edit&amp;redlink=1" title="고대 로마의 다리 (없는 문서)">로마 시대의 석제 다리</a>이다. 처음에는 목제로 지어졌지만, 기원전 2세기 경에 석제 다리로 다시 지었다. <a class="new" href="/w/index.php?title=%ED%8A%B8%EB%9D%BC%EC%8A%A4%ED%85%8C%EB%B2%A0%EB%A0%88&amp;action=edit&amp;redlink=1" title="트라스테베레 (없는 문서)">트라스테베레</a>와 <a class="new" href="/w/index.php?title=%EB%B3%B4%EC%95%84%EB%A6%AC%EC%9B%80_%EA%B4%91%EC%9E%A5&amp;action=edit&amp;redlink=1" title="보아리움 광장 (없는 문서)">보아리움 광장</a> 사이에 <a href="/wiki/%ED%85%8C%EB%B2%A0%EB%A0%88_%EA%B0%95" title="테베레 강">테베레 강</a>을 연결하는 유일한 다리였고, 중류 지역에 있는 아치 한 개가 오늘날에 남아있는 다리의 전부이며, 이렇게 불안전한 모습에서 이름을 따 <i>로토 다리</i>(Ponte Rotto, 부서진 다리) 라고 부른다.</p>

<h2><span class="mw-headline" id=".EC.97.AD.EC.82.AC">역사</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%A1%9C%ED%86%A0_%EB%8B%A4%EB%A6%AC&amp;action=edit&amp;section=1" title="부분 편집: 역사">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<h3><span class="mw-headline" id=".EA.B3.A0.EC.A0.84_.EC.8B.9C.EB.8C.80">고전 시대</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%A1%9C%ED%86%A0_%EB%8B%A4%EB%A6%AC&amp;action=edit&amp;section=2" title="부분 편집: 고전 시대">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>다리의 가장 오래된 교각은 아마 기원전 2세기 중반에 <a class="new" href="/w/index.php?title=%EC%95%84%EC%9A%B0%EB%A0%90%EB%A6%AC%EC%95%84_%EA%B0%80%EB%8F%84&amp;action=edit&amp;redlink=1" title="아우렐리아 가도 (없는 문서)">아우렐리아 가도</a>가 건설되던 때에 놓여졌을 것이다.<sup class="reference" id="cite_ref-oxford_1-0"><a href="#cite_note-oxford-1">[1]</a></sup> <a href="/wiki/%ED%8B%B0%ED%88%AC%EC%8A%A4_%EB%A6%AC%EB%B9%84%EC%9A%B0%EC%8A%A4" title="티투스 리비우스">티투스 리비우스</a>는 기원전 192년에 이 다리와 똑같은 위치에 다리가 존재했었다고 기록을 남겼다. 최초의 석제 다리는 기원전 179년에 <a href="/wiki/%EA%B0%90%EC%B0%B0%EA%B4%80" title="감찰관">감찰관</a> <a class="new" href="/w/index.php?title=%EB%A7%88%EB%A5%B4%EC%BF%A0%EC%8A%A4_%ED%92%80%EB%B9%84%EC%9A%B0%EC%8A%A4_%EB%85%B8%EB%B9%8C%EB%A6%AC%EC%98%A4%EB%A5%B4&amp;action=edit&amp;redlink=1" title="마르쿠스 풀비우스 노빌리오르 (없는 문서)">마르쿠스 풀비우스 노빌리오르</a>가 건설에 착수했으며, 기원전 151년까지 완공을 하지 못했었다.<sup class="reference" id="cite_ref-train_2-0"><a href="#cite_note-train-2">[2]</a></sup> 교각 부분은 이 시기의 초창기에 지어졌지만, 그럼에도 아치는 기원전 142년에 지어졌다.<sup class="reference" id="cite_ref-3"><a href="#cite_note-3">[3]</a></sup> 이렇게 로토 다리는 수백년간 있었다가, <a href="/wiki/%EC%95%84%EC%9A%B0%EA%B5%AC%EC%8A%A4%ED%88%AC%EC%8A%A4" title="아우구스투스">아우구스투스</a> 보수와 재건을 했었고,<sup class="reference" id="cite_ref-4"><a href="#cite_note-4">[4]</a></sup> 이후에는 서기 280년에 <a class="new" href="/w/index.php?title=%EB%A7%88%EB%A5%B4%EC%BF%A0%EC%8A%A4_%EC%95%84%EC%9A%B0%EB%A0%90%EB%A6%AC%EC%9A%B0%EC%8A%A4_%ED%94%84%EB%A1%9C%EB%B6%80%EC%8A%A4&amp;action=edit&amp;redlink=1" title="마르쿠스 아우렐리우스 프로부스 (없는 문서)">마르쿠스 아우렐리우스 프로부스</a> 시기에도 보수가 이뤄졌다.</p>

<p>로마 제국이 붕괴하고 나서, 로토 다리는 모든 건물에 엄청난 피해를 입힌 홍수들로 인해 손상을 입었다. 최초의 심각한 피해는 1230년에 입었으며, <a href="/wiki/%EA%B5%90%ED%99%A9_%EA%B7%B8%EB%A0%88%EA%B3%A0%EB%A6%AC%EC%98%A4_11%EC%84%B8" title="교황 그레고리오 11세">교황 그레고리오 11세</a>가 재건하였다.<sup class="reference" id="cite_ref-train_2-1"><a href="#cite_note-train-2">[2]</a></sup> 그후 로토 다리는 1557년에 더 심각한 피해를 입었고, 다시 <a href="/wiki/%EA%B5%90%ED%99%A9_%EA%B7%B8%EB%A0%88%EA%B3%A0%EB%A6%AC%EC%98%A4_13%EC%84%B8" title="교황 그레고리오 13세">교황 그레고리오 13세</a> 시기에 재건되었으며, 그레고리오 13세가 다리에 남긴 보수에 관한 라틴어 비문은 오늘날까지 남아있다.<sup class="reference" id="cite_ref-5"><a href="#cite_note-5">[5]</a></sup> 마지막으로 1575년과 1598년에 발생한 홍수가 다리의 동쪽 지역을 날려버려, 결론적으로 수 세기 동안 다리로서의 기능은 버려졌다.<sup class="reference" id="cite_ref-train_2-2"><a href="#cite_note-train-2">[2]</a></sup><sup class="reference" id="cite_ref-encyclopedia_6-0"><a href="#cite_note-encyclopedia-6">[6]</a></sup> 그후로 많은 시간 동안 낚시터로서 사용되었다<sup class="reference" id="cite_ref-website_7-0"><a href="#cite_note-website-7">[7]</a></sup> 1853년에 <a href="/wiki/%EA%B5%90%ED%99%A9_%EB%B9%84%EC%98%A4_9%EC%84%B8" title="교황 비오 9세">교황 비오 9세</a>가 다리의 남은 부분을 철제로 육지와 연결하였지만, 무거운 금속이 석제 다리의 구조적 안정성을 약화시키고 말았다<sup class="reference" id="cite_ref-train_2-3"><a href="#cite_note-train-2">[2]</a></sup><sup class="reference" id="cite_ref-website_7-1"><a href="#cite_note-website-7">[7]</a></sup> 남아있는 절반은 1887년에 <a class="new" href="/w/index.php?title=%ED%8C%94%EB%9D%BC%ED%8B%B0%EB%85%B8_%EB%8B%A4%EB%A6%AC&amp;action=edit&amp;redlink=1" title="팔라티노 다리 (없는 문서)">팔라티노 다리</a>의 공간을 확보하기 위해 무너트렸다<sup class="reference" id="cite_ref-encyclopedia_6-1"><a href="#cite_note-encyclopedia-6">[6]</a></sup></p>
