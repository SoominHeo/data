<p><b>론 폴</b>(<span lang="en" xml:lang="en">Ronald Ernest "Ron" Paul</span>, <a href="/wiki/1935%EB%85%84" title="1935년">1935년</a> <a href="/wiki/8%EC%9B%94_20%EC%9D%BC" title="8월 20일">8월 20일</a> ~ )은 <a href="/wiki/%EB%AF%B8%EA%B5%AD" title="미국">미국</a>의 정치인이다.</p>
<p><a href="/wiki/%ED%8E%9C%EC%8B%A4%EB%B2%A0%EC%9D%B4%EB%8B%88%EC%95%84_%EC%A3%BC" title="펜실베이니아 주">펜실베이니아 주</a> 출신이다. 게티즈버그 대학교를 거쳐 <a href="/wiki/%EB%93%80%ED%81%AC_%EB%8C%80%ED%95%99%EA%B5%90" title="듀크 대학교">듀크 대학교</a> 메디컬 스쿨을 졸업하였다. <a href="/wiki/%ED%85%8D%EC%82%AC%EC%8A%A4_%EC%A3%BC" title="텍사스 주">텍사스 주</a>로 옮겨 <a href="/wiki/%EC%82%B0%EB%B6%80%EC%9D%B8%EA%B3%BC" title="산부인과">산부인과</a> 의사로 활동했다. 본래 정치에 뜻을 두고 있던 그는 <a href="/wiki/1960%EB%85%84%EB%8C%80" title="1960년대">1960년대</a>에 <a href="/wiki/%EA%B3%B5%ED%99%94%EB%8B%B9_(%EB%AF%B8%EA%B5%AD)" title="공화당 (미국)">공화당</a> 소속 정치인의 선거운동에 참여했고, <a href="/wiki/1974%EB%85%84" title="1974년">1974년</a>과 <a href="/wiki/1976%EB%85%84" title="1976년">1976년</a> 텍사스 주 남부 지역구의 연방 하원의원 선거에 출마했으나, <a href="/wiki/%EB%AF%BC%EC%A3%BC%EB%8B%B9_(%EB%AF%B8%EA%B5%AD)" title="민주당 (미국)">민주당</a> 후보에게 패했다. <a href="/wiki/1978%EB%85%84" title="1978년">1978년</a> 하원의원 선거에 출마하여 당선됐으며, <a href="/wiki/1980%EB%85%84" title="1980년">1980년</a>, <a href="/wiki/1982%EB%85%84" title="1982년">1982년</a> 재선되었다. <a href="/wiki/1984%EB%85%84" title="1984년">1984년</a> 연방 상원의원 선거에 도전했으나, 공화당 예비선거에서 패했고, <a href="/wiki/1985%EB%85%84" title="1985년">1985년</a> 하원의원 임기 종료와 함께 의회를 떠났다. 자유의지주의자로 전쟁에 반대하고 감세를 지지하는 그는 <a href="/wiki/1988%EB%85%84" title="1988년">1988년</a> <a href="/wiki/1988%EB%85%84_%EB%AF%B8%EA%B5%AD_%EB%8C%80%ED%86%B5%EB%A0%B9_%EC%84%A0%EA%B1%B0" title="1988년 미국 대통령 선거">대통령 선거</a>에 <a href="/wiki/%EC%9E%90%EC%9C%A0%EB%8B%B9_(%EB%AF%B8%EA%B5%AD)" title="자유당 (미국)">자유당</a>의 지지로 출마, 전국적으로 0.5%의 표를 얻었다. <a href="/wiki/1996%EB%85%84" title="1996년">1996년</a> 공화당 소속으로 다시 하원의원 선거에 출마, 당선되었고, 이후 <a href="/wiki/2008%EB%85%84" title="2008년">2008년</a>까지 2년 주기의 선거에 계속 당선되어 하원의원으로 재직 중이다. 그는 독특한 경력과 주장으로 주목을 받았고 전쟁에 반대하는 입장을 고수하여 공화당 소속으로는 드물게 <a href="/wiki/%EC%9D%B4%EB%9D%BC%ED%81%AC_%EC%A0%84%EC%9F%81" title="이라크 전쟁">이라크 전쟁</a>에도 반대했다. 그러나 공화당에서는 주류의 입장에 속하지 않으므로 공화당에서 큰 위치를 차지하지는 못한 가운데, <a href="/wiki/2008%EB%85%84_%EB%AF%B8%EA%B5%AD_%EB%8C%80%ED%86%B5%EB%A0%B9_%EC%84%A0%EA%B1%B0" title="2008년 미국 대통령 선거">2008년 대통령 선거</a> 출마를 선언했다. 공화당원 전체의 지지율은 낮아 후보로 선출될 가능성은 없었고 <a href="/wiki/%EC%A1%B4_%EB%A7%A4%EC%BC%80%EC%9D%B8" title="존 매케인">존 매케인</a>이 일찌감치 후보로 확정되었지만 그를 지지하는 일정한 층이 존재하는 가운데 그는 가장 늦게까지 경선을 포기하지 않았다. 미국에서 대표적인 <a class="mw-redirect" href="/wiki/%EC%9E%90%EC%9C%A0%EC%9D%98%EC%A7%80%EC%A3%BC%EC%9D%98" title="자유의지주의">자유의지주의</a> 신봉자로 알려진 저술가로 유명하며 독특한 행보로 눈길을 끌고 있는 가운데 4월 27일 2012년 대선 행보를 공식적으로 발표하였다.</p>
