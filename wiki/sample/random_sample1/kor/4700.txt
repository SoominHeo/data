<h1 class="firstHeading" id="firstHeading" lang="ko">롤러코스터 타이쿤 3D</h1>
<div class="mw-body-content" id="bodyContent">
<div id="siteSub">위키백과, 우리 모두의 백과사전.</div>

<div class="mw-jump" id="jump-to-nav">
					이동:					<a href="#mw-head">둘러보기</a>, 					<a href="#p-search">검색</a>
</div>
<div class="mw-content-ltr" dir="ltr" id="mw-content-text" lang="ko"><script>function mfTempOpenSection(id){var block=document.getElementById("mf-section-"+id);block.className+=" open-block";block.previousSibling.className+=" open-block";}</script><p><b>롤러코스터 타이쿤 3D</b>(RollerCoaster Tycoon 3D)는 <a class="new" href="/w/index.php?title=%EC%97%94-%EC%8A%A4%ED%8E%98%EC%9D%B4%EC%8A%A4&amp;action=edit&amp;redlink=1" title="엔-스페이스 (없는 문서)">엔-스페이스</a>가 제작하고 <a href="/wiki/%EC%95%84%ED%83%80%EB%A6%AC" title="아타리">아타리</a>가 유통한 <a href="/wiki/%EB%8B%8C%ED%85%90%EB%8F%84_3DS" title="닌텐도 3DS">닌텐도 3DS</a> 전용 게임으로, <a href="/wiki/%EB%A1%A4%EB%9F%AC%EC%BD%94%EC%8A%A4%ED%84%B0_%ED%83%80%EC%9D%B4%EC%BF%A4_(%EC%8B%9C%EB%A6%AC%EC%A6%88)" title="롤러코스터 타이쿤 (시리즈)">롤러코스터 타이쿤 시리즈</a>의 4번째 작품이자 <a class="mw-redirect" href="/wiki/%EC%8A%A4%ED%95%80%EC%98%A4%ED%94%84" title="스핀오프">스핀오프</a> 작이다. <a href="/wiki/2012%EB%85%84" title="2012년">2012년</a> <a href="/wiki/10%EC%9B%94_16%EC%9D%BC" title="10월 16일">10월 16일</a>에 출시되었으며, <a href="/wiki/%EB%A1%A4%EB%9F%AC%EC%BD%94%EC%8A%A4%ED%84%B0_%ED%83%80%EC%9D%B4%EC%BF%A4_3" title="롤러코스터 타이쿤 3">3편</a>과 마찬가지로 <a class="mw-redirect" href="/wiki/%ED%81%AC%EB%A6%AC%EC%8A%A4_%EC%86%8C%EC%9D%B4%EC%96%B4" title="크리스 소이어">크리스 소이어</a>는 개발에 참여하지 않았다. 터치 스크린 조작으로 닌텐도 3DS의 기능을 사용해 롤러코스터를 3D로 체험하는 기능이 포함되어 있다고 밝혔다. 기존의 시리즈처럼 철저히 게이머의 조작으로 진행되는 테마 파크 운영 방식을 계승하되 기존의 소스를 재활용하지는 않았다.</p>
<p>그러나 <a href="/wiki/%EB%8B%8C%ED%85%90%EB%8F%84_3DS" title="닌텐도 3DS">3DS</a>에서 <a href="/wiki/%EB%8B%8C%ED%85%90%EB%8F%84_DS" title="닌텐도 DS">NDS</a> 수준 그래픽을 만들고 있다는 혹평까지 나올 정도로 퇴보된 그래픽의 티저로 실망감을 안겨주었다. 나아져서 나올 줄 알았는데, 실제로 출시되고 받은 평은 더 처참했다. <a href="/wiki/%EB%A1%A4%EB%9F%AC%EC%BD%94%EC%8A%A4%ED%84%B0_%ED%83%80%EC%9D%B4%EC%BF%A4_3" title="롤러코스터 타이쿤 3">3편</a>에서 제공되었던 <a href="/wiki/%EB%B6%88%EA%BD%83%EB%86%80%EC%9D%B4" title="불꽃놀이">불꽃놀이</a>도 없고, <a href="/wiki/%EB%82%A0%EC%94%A8" title="날씨">날씨</a>도 없고, 배너도 없고, 심지어 지형 변경 불가에 터널도 없고, 놀이기구 이름을 짓는 것도 불가능했다. 이로 인해 <a href="/wiki/%EB%A9%94%ED%83%80%ED%81%AC%EB%A6%AC%ED%8B%B1" title="메타크리틱">메타크리틱</a>에 100점 만점에 39점을 받았으며, 결국 역대 최악의 <a href="/wiki/%EB%A1%A4%EB%9F%AC%EC%BD%94%EC%8A%A4%ED%84%B0_%ED%83%80%EC%9D%B4%EC%BF%A4_(%EC%8B%9C%EB%A6%AC%EC%A6%88)" title="롤러코스터 타이쿤 (시리즈)">롤러코스터 타이쿤</a>이 되었다.</p>
