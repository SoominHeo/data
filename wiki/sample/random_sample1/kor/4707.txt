<h1 class="firstHeading" id="firstHeading" lang="ko">롬바르디아 주</h1>
<div class="mw-body-content" id="bodyContent">
<div id="siteSub">위키백과, 우리 모두의 백과사전.</div>
<div id="contentSub"></div>
<div class="mw-jump" id="jump-to-nav">
					이동:					<a href="#mw-head">둘러보기</a>, 					<a href="#p-search">검색</a>
</div>
<div class="mw-content-ltr" dir="ltr" id="mw-content-text" lang="ko"><script>function mfTempOpenSection(id){var block=document.getElementById("mf-section-"+id);block.className+=" open-block";block.previousSibling.className+=" open-block";}</script><div class="floatright"><a class="image" href="/wiki/%ED%8C%8C%EC%9D%BC:Italy_Regions_Lombardy_Map.png"><img alt="Italy Regions Lombardy Map.png" data-file-height="257" data-file-width="220" height="257" src="//upload.wikimedia.org/wikipedia/commons/1/18/Italy_Regions_Lombardy_Map.png" width="220"/></a></div>
<p><b>롬바르디아 주</b>(Lombardia)는 <a href="/wiki/%EC%9D%B4%ED%83%88%EB%A6%AC%EC%95%84" title="이탈리아">이탈리아</a> 북부의 주로 주도는 <a href="/wiki/%EB%B0%80%EB%9D%BC%EB%85%B8" title="밀라노">밀라노</a>이다.</p>
<p>지리상으로 북쪽에는 <a href="/wiki/%EC%95%8C%ED%94%84%EC%8A%A4_%EC%82%B0%EB%A7%A5" title="알프스 산맥">알프스 산맥</a>의 꼭대기와 계곡으로 이루어져있고, 남쪽은 최대의 곡창 <a class="new" href="/w/index.php?title=%EB%A1%AC%EB%B0%94%EB%A5%B4%EB%94%94%EC%95%84_%ED%8F%89%EC%9B%90&amp;action=edit&amp;redlink=1" title="롬바르디아 평원 (없는 문서)">롬바르디아 평원</a>이 펼쳐져 있다.</p>
<p>롬바르디아 주는 <a href="/wiki/%EB%AF%BC%EC%A1%B1_%EB%8C%80%EC%9D%B4%EB%8F%99" title="민족 대이동">민족 대이동</a> 때, 그곳에 침입한 <a class="mw-redirect" href="/wiki/%EB%A1%AC%EB%B0%94%EB%A5%B4%EB%93%9C%EC%A1%B1" title="롬바르드족">롬바르드족</a>의 이름에서 비롯되었다. 이 주는 12개의 현으로 나뉜다 - <a href="/wiki/%EB%B0%80%EB%9D%BC%EB%85%B8_%ED%98%84" title="밀라노 현">밀라노 현</a>, <a href="/wiki/%EB%B8%8C%EB%A0%88%EC%8B%9C%EC%95%84_%ED%98%84" title="브레시아 현">브레시아 현</a>, <a href="/wiki/%EC%BD%94%EB%AA%A8_%ED%98%84" title="코모 현">코모 현</a>, <a href="/wiki/%ED%81%AC%EB%A0%88%EB%AA%A8%EB%82%98_%ED%98%84" title="크레모나 현">크레모나 현</a>, <a href="/wiki/%EB%A0%88%EC%BD%94_%ED%98%84" title="레코 현">레코 현</a>, <a href="/wiki/%EB%A1%9C%EB%94%94_%ED%98%84" title="로디 현">로디 현</a>, <a href="/wiki/%EB%A7%8C%ED%86%A0%EB%B0%94_%ED%98%84" title="만토바 현">만토바 현</a>, <a class="new" href="/w/index.php?title=%EB%AA%AC%EC%B0%A8_%ED%98%84&amp;action=edit&amp;redlink=1" title="몬차 현 (없는 문서)">몬차 현</a>, <a class="new" href="/w/index.php?title=%EB%B8%8C%EB%A6%AC%EC%95%88%EC%B0%A8_%ED%98%84&amp;action=edit&amp;redlink=1" title="브리안차 현 (없는 문서)">브리안차 현</a>, <a href="/wiki/%ED%8C%8C%EB%B9%84%EC%95%84_%ED%98%84" title="파비아 현">파비아 현</a>, <a class="new" href="/w/index.php?title=%EC%86%90%EB%94%94%EB%A6%AC%EC%98%A4_%ED%98%84&amp;action=edit&amp;redlink=1" title="손디리오 현 (없는 문서)">손디리오 현</a>과 <a href="/wiki/%EB%B0%94%EB%A0%88%EC%84%B8_%ED%98%84" title="바레세 현">바레세 현</a>.</p>
<p>밀라노는 이탈리아 최대의 상공업, 금융의 중심지이며, 많은 기업들의 본사이다. 공업이 롬바르디아 주의 가장 큰 비율을 차지하고, 롬바르디아 평원에는 <a class="new" href="/w/index.php?title=%EB%B2%BC%EB%86%8D%EC%82%AC&amp;action=edit&amp;redlink=1" title="벼농사 (없는 문서)">벼농사</a>가 발달하였다. 또한 <a href="/wiki/%EC%B9%98%EC%A6%88" title="치즈">치즈</a>, <a href="/wiki/%ED%8F%AC%EB%8F%84%EC%A3%BC" title="포도주">포도주</a> 등을 생산한다.</p>
