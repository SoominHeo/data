<h1 class="firstHeading" id="firstHeading" lang="ko">루돌프 카르나프</h1>
<div class="mw-body-content" id="bodyContent">
<div id="siteSub">위키백과, 우리 모두의 백과사전.</div>
<div id="contentSub"></div>
<div class="mw-jump" id="jump-to-nav">
					이동:					<a href="#mw-head">둘러보기</a>, 					<a href="#p-search">검색</a>
</div>
<div class="mw-content-ltr" dir="ltr" id="mw-content-text" lang="ko"><script>function mfTempOpenSection(id){var block=document.getElementById("mf-section-"+id);block.className+=" open-block";block.previousSibling.className+=" open-block";}</script><p><b>루돌프 카르나프</b>(<span style="font-size: smaller;"><a href="/wiki/%EB%8F%85%EC%9D%BC%EC%96%B4" title="독일어">독일어</a>:</span> <span lang="de" xml:lang="de">Rudolf Carnap</span>, <a href="/wiki/1891%EB%85%84" title="1891년">1891년</a> <a href="/wiki/5%EC%9B%94_18%EC%9D%BC" title="5월 18일">5월 18일</a> ~ <a href="/wiki/1970%EB%85%84" title="1970년">1970년</a> <a href="/wiki/9%EC%9B%94_14%EC%9D%BC" title="9월 14일">9월 14일</a>)는 현대 <a href="/wiki/%EB%8F%85%EC%9D%BC" title="독일">독일</a>의 <a href="/wiki/%EC%B2%A0%ED%95%99%EC%9E%90" title="철학자">철학자</a>이다.</p>
<h2><span class="mw-headline" id=".EC.83.9D.EC.95.A0">생애</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%A3%A8%EB%8F%8C%ED%94%84_%EC%B9%B4%EB%A5%B4%EB%82%98%ED%94%84&amp;action=edit&amp;section=1" title="부분 편집: 생애">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>독일의 <a href="/wiki/%EC%BE%B0%EB%A5%B8" title="쾰른">쾰른</a> 근교 <a href="/wiki/%EB%B6%80%ED%8D%BC%ED%83%88" title="부퍼탈">부퍼탈</a>에서 태어났다. <a href="/wiki/%EB%B9%88" title="빈">빈</a>과 <a href="/wiki/%ED%94%84%EB%9D%BC%ED%95%98" title="프라하">프라하</a> 등의 대학에서 가르쳤다. 처음에는 <a href="/wiki/%EA%B3%A0%ED%8B%80%EB%A1%9C%ED%94%84_%ED%94%84%EB%A0%88%EA%B2%8C" title="고틀로프 프레게">고틀로프 프레게</a>, <a href="/wiki/%EB%B2%84%ED%8A%B8%EB%9F%B0%EB%93%9C_%EB%9F%AC%EC%85%80" title="버트런드 러셀">버트런드 러셀</a>의 영향을 받아 <a class="mw-redirect" href="/wiki/%EA%B8%B0%ED%98%B8%EB%85%BC%EB%A6%AC%ED%95%99" title="기호논리학">기호논리학</a>에 의한 '세계의 논리적 구성' 파악을 위해 노력했다. 이어서 수학자 <a class="mw-redirect" href="/wiki/%EA%B4%B4%EB%8D%B8" title="괴델">괴델</a>의 이론에서 힌트를 얻어 언어의 논리적 통사론을 연구했다. <a href="/wiki/1936%EB%85%84" title="1936년">1936년</a>부터 <a href="/wiki/%EC%8B%9C%EC%B9%B4%EA%B3%A0_%EB%8C%80%ED%95%99%EA%B5%90" title="시카고 대학교">시카고 대학교</a> 교수를 지냈다.</p>
<h2><span class="mw-headline" id=".EC.97.85.EC.A0.81">업적</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%A3%A8%EB%8F%8C%ED%94%84_%EC%B9%B4%EB%A5%B4%EB%82%98%ED%94%84&amp;action=edit&amp;section=2" title="부분 편집: 업적">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p><a href="/wiki/%EC%95%8C%ED%94%84%EB%A0%88%ED%8A%B8_%ED%83%80%EB%A5%B4%EC%8A%A4%ED%82%A4" title="알프레트 타르스키">알프레트 타르스키</a>의 <a href="/wiki/%EC%9D%98%EB%AF%B8%EB%A1%A0" title="의미론">의미론</a>(意味論)을 배우고 <a href="/wiki/%EB%85%BC%EB%A6%AC%ED%95%99" title="논리학">논리학</a> 및 <a href="/wiki/%EC%88%98%ED%95%99" title="수학">수학</a>의 기초를 구명, <a href="/wiki/%EB%85%BC%EB%A6%AC%EC%8B%A4%EC%A6%9D%EC%A3%BC%EC%9D%98" title="논리실증주의">논리실증주의</a>자 중에서도 형식화된 인공적인 기호논리학에 의해 명제의 분석을 철저히 한 학자이다.</p>
