<p><b>루돌프 에른스트 파이얼스</b>(<span style="font-size: smaller;"><a href="/wiki/%EB%8F%85%EC%9D%BC%EC%96%B4" title="독일어">독일어</a>:</span> <span lang="de" xml:lang="de">Rudolf Ernst Peierls</span>, 1907–1995)는 <a href="/wiki/%EB%8F%85%EC%9D%BC" title="독일">독일</a> 태생의 물리학자다. <a href="/wiki/%EB%82%98%EC%B9%98_%EB%8F%85%EC%9D%BC" title="나치 독일">나치 독일</a>을 피해 <a href="/wiki/%EC%98%81%EA%B5%AD" title="영국">영국</a>으로 이민하였고, 영국의 <a href="/wiki/%ED%95%B5%EB%AC%B4%EA%B8%B0" title="핵무기">핵무기</a> 개발에 참여하였다. 군사적 연구 밖에도, 물리학의 여러 분야에 공헌하였다.</p>
<h2><span class="mw-headline" id=".EC.83.9D.EC.95.A0">생애</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%A3%A8%EB%8F%8C%ED%94%84_%ED%8C%8C%EC%9D%B4%EC%96%BC%EC%8A%A4&amp;action=edit&amp;section=1" title="부분 편집: 생애">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p><a href="/wiki/%EB%B2%A0%EB%A5%BC%EB%A6%B0" title="베를린">베를린</a>에서 <a href="/wiki/%EC%9C%A0%EB%8C%80%EC%9D%B8" title="유대인">유대인</a> 가정에 태어났다. 아버지는 하인리히 파이얼스(<span style="font-size: smaller;"><a href="/wiki/%EB%8F%85%EC%9D%BC%EC%96%B4" title="독일어">독일어</a>:</span> <span lang="de" xml:lang="de">Heinrich Peierls</span>)였고, 어머니는 엘리차베트 바이게르트(Elizabeth Weigert)였다. 1929년에는 <a href="/wiki/%EC%B7%A8%EB%A6%AC%ED%9E%88" title="취리히">취리히</a>에서 <a href="/wiki/%EB%B2%A0%EB%A5%B4%EB%84%88_%ED%95%98%EC%9D%B4%EC%A0%A0%EB%B2%A0%EB%A5%B4%ED%81%AC" title="베르너 하이젠베르크">베르너 하이젠베르크</a>와 <a href="/wiki/%EB%B3%BC%ED%94%84%EA%B0%95_%ED%8C%8C%EC%9A%B8%EB%A6%AC" title="볼프강 파울리">볼프강 파울리</a> 아래 <a href="/wiki/%EC%9D%91%EC%A7%91%EB%AC%BC%EC%A7%88%EB%AC%BC%EB%A6%AC%ED%95%99" title="응집물질물리학">응집물질물리학</a>을 연구하였다.</p>
<p>그 뒤 영국 <a href="/wiki/%EC%BC%80%EC%9E%84%EB%B8%8C%EB%A6%AC%EC%A7%80_%EB%8C%80%ED%95%99%EA%B5%90" title="케임브리지 대학교">케임브리지 대학교</a>에서 유학하였다. 이 도중 <a class="mw-redirect" href="/wiki/%EB%82%98%EC%B9%98_%EB%8B%B9" title="나치 당">나치 당</a>이 정권을 잡자 할 수 없이 영국에 남아 있게 되었다. 1939년부터 <a class="new" href="/w/index.php?title=%EC%98%A4%ED%86%A0_%ED%94%84%EB%A6%AC%EC%8A%88&amp;action=edit&amp;redlink=1" title="오토 프리슈 (없는 문서)">오토 프리슈</a>(Otto Frisch)와 <a href="/wiki/%EC%A0%9C%EC%9E%84%EC%8A%A4_%EC%B1%84%EB%93%9C%EC%9C%85" title="제임스 채드윅">제임스 채드윅</a>과 같이 <a href="/wiki/%ED%95%B5%EB%AC%B4%EA%B8%B0" title="핵무기">핵무기</a> 연구를 시작하였다. 1940년 3월에는 <b>프리슈-파이얼스 보고서</b>(<span style="font-size: smaller;"><a href="/wiki/%EC%98%81%EC%96%B4" title="영어">영어</a>:</span> <span lang="en" xml:lang="en">Frisch–Peierls memorandum</span>)를 작성하였다. 이 보고서에서 프리슈와 파이얼스는 불과 1 <a href="/wiki/%ED%82%AC%EB%A1%9C%EA%B7%B8%EB%9E%A8" title="킬로그램">킬로그램</a>의 <a href="/wiki/%EC%9A%B0%EB%9D%BC%EB%8A%84" title="우라늄">우라늄</a>을 통해 <a class="mw-redirect" href="/wiki/%EC%9B%90%EC%9E%90_%ED%8F%AD%ED%83%84" title="원자 폭탄">원자 폭탄</a>을 만들 수 있다는 사실을 설명하였고, <a href="/wiki/%EC%98%81%EA%B5%AD" title="영국">영국</a>과 이후 <a href="/wiki/%EB%AF%B8%EA%B5%AD" title="미국">미국</a>의 핵무기 연구가 시작되는 발단이 되었다. 이듬해 (1941년) 이 보고서는 미국으로 전해졌고, 미국은 <a href="/wiki/%EB%A7%A8%ED%95%B4%ED%8A%BC_%EA%B3%84%ED%9A%8D" title="맨해튼 계획">맨해튼 계획</a>을 시작하였다.</p>
<p>1943년 8월부터 파이얼스는 미국으로 건너가 <a href="/wiki/%EB%A7%A8%ED%95%B4%ED%8A%BC_%EA%B3%84%ED%9A%8D" title="맨해튼 계획">맨해튼 계획</a>에 참여하였다. 1945년 영국 작위(Commander of the Order of the British Empire (Civil Division))를 수여받았다.</p>
<p>전후 파이얼스는 <a href="/wiki/%EB%B2%84%EB%B0%8D%EC%97%84_%EB%8C%80%ED%95%99%EA%B5%90" title="버밍엄 대학교">버밍엄 대학교</a> 교수로 있다가, 1963년 <a href="/wiki/%EC%98%A5%EC%8A%A4%ED%8D%BC%EB%93%9C_%EB%8C%80%ED%95%99%EA%B5%90" title="옥스퍼드 대학교">옥스퍼드 대학교</a> 교수가 되었다. 1974년 은퇴하였고, 1995년 <a href="/wiki/%EC%98%A5%EC%8A%A4%ED%8D%BC%EB%93%9C" title="옥스퍼드">옥스퍼드</a>에서 사망하였다.</p>
