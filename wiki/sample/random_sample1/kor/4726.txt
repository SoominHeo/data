<p><b><a href="/wiki/%EB%A3%A8%EB%A7%88%EB%8B%88%EC%95%84" title="루마니아">루마니아</a>의 <a href="/wiki/%EA%B5%AD%EA%B8%B0" title="국기">국기</a></b>인 <b>트레이 쿨로리</b>(이는 <a class="mw-redirect" href="/wiki/%EB%A3%A8%EB%A7%88%EB%8B%88%EC%95%84_%EC%9D%B8%EB%AF%BC_%EA%B3%B5%ED%99%94%EA%B5%AD%EC%9D%98_%EA%B5%AD%EA%B0%803" title="루마니아 인민 공화국의 국가3">루마니아 인민 공화국의 국가</a>이기도 하였다)는 파랑, 노랑, 빨강이 같은 폭으로 되어 있는 <a href="/wiki/%EC%82%BC%EC%83%89%EA%B8%B0" title="삼색기">삼색기</a>이다. 1861년 최초의 루마니아 당시에는 국기가 빨강-노랑-파랑으로 가로 순으로 배치되어 있었다. 그러나 왕국이 세워진 후인 1877년에 지금과 같은 형태가 세워졌다. <a href="/wiki/1947%EB%85%84" title="1947년">1947년</a> 공산주의 정권이 들어선 후, 공산주의를 상징하는 문장이 추가되었으나, <a href="/wiki/1989%EB%85%84" title="1989년">1989년</a> <a href="/wiki/12%EC%9B%94_27%EC%9D%BC" title="12월 27일">12월 27일</a> <a href="/wiki/%EB%8B%88%EC%BD%9C%EB%9D%BC%EC%97%90_%EC%B0%A8%EC%9A%B0%EC%85%B0%EC%8A%A4%EC%BF%A0" title="니콜라에 차우셰스쿠">니콜라에 차우셰스쿠</a> 부부 처형 이후, 문장을 삭제하고 지금과 같은 모습으로 다시 정식으로 채택되었다. 이 깃발은 옛날의 <a href="/wiki/%EB%AA%B0%EB%8B%A4%EB%B9%84%EC%95%84" title="몰다비아">몰다비아</a>(파랑), <a href="/wiki/%EC%99%88%EB%9D%BC%ED%82%A4%EC%95%84" title="왈라키아">왈라키아</a>(노랑), <a href="/wiki/%ED%8A%B8%EB%9E%80%EC%8B%A4%EB%B0%94%EB%8B%88%EC%95%84" title="트란실바니아">트란실바니아</a>(빨강)의 통합을 상징한다.</p>

<h2><span class="mw-headline" id=".EB.B9.84.EC.8A.B7.ED.95.9C_.EA.B5.AD.EA.B8.B0">비슷한 국기</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%A3%A8%EB%A7%88%EB%8B%88%EC%95%84%EC%9D%98_%EA%B5%AD%EA%B8%B0&amp;action=edit&amp;section=2" title="부분 편집: 비슷한 국기">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>루마니아의 국기는 아프리카 <a href="/wiki/%EC%B0%A8%EB%93%9C%EC%9D%98_%EA%B5%AD%EA%B8%B0" title="차드의 국기">차드의 국기</a>와 파란색의 명도 차이외에는 똑같다.(차드 국기의 파란 색이 더 진함) 공산주의 시대에는 루마니아 국기의 중앙에 문장이 있어 구별되었으나, 이후에 공산주의 문장을 제거하면서 현재의 모습이 되었다. 또 <a href="/wiki/%EB%AA%B0%EB%8F%84%EB%B0%94%EC%9D%98_%EA%B5%AD%EA%B8%B0" title="몰도바의 국기">몰도바의 국기</a>와 <a href="/wiki/%EC%95%88%EB%8F%84%EB%9D%BC%EC%9D%98_%EA%B5%AD%EA%B8%B0" title="안도라의 국기">안도라의 국기</a>와는 색조, 문장, 국기 비율의 차이만이 있을 뿐이다.</p>
<p><a href="/wiki/%EB%AA%B0%EB%8F%84%EB%B0%94" title="몰도바">몰도바</a>는 1918년에 <a href="/wiki/%EB%8C%80%EB%A3%A8%EB%A7%88%EB%8B%88%EC%95%84" title="대루마니아">대루마니아</a>로 통합되었으나, <a href="/wiki/1924%EB%85%84" title="1924년">1924년</a> <a class="mw-redirect" href="/wiki/%EC%86%8C%EB%B9%84%EC%97%90%ED%8A%B8_%EC%97%B0%EB%B0%A9" title="소비에트 연방">소비에트 연방</a>에 병합되었다. <a href="/wiki/%EB%AA%B0%EB%8F%84%EB%B0%94_%EC%86%8C%EB%B9%84%EC%97%90%ED%8A%B8_%EC%82%AC%ED%9A%8C%EC%A3%BC%EC%9D%98_%EA%B3%B5%ED%99%94%EA%B5%AD" title="몰도바 소비에트 사회주의 공화국">몰도바 소비에트 사회주의 공화국</a>은 현재 몰도바의 미승인 국가 지역인 <a href="/wiki/%ED%8A%B8%EB%9E%80%EC%8A%A4%EB%8B%88%EC%8A%A4%ED%8A%B8%EB%A6%AC%EC%95%84" title="트란스니스트리아">트란스니스트리아</a>의 기와 유사한 빨강, 초록, 빨강의 기에 소비에트 문장을 추가한 것을 사용하다가 <a href="/wiki/1991%EB%85%84" title="1991년">1991년</a> <a href="/wiki/8%EC%9B%94_27%EC%9D%BC" title="8월 27일">8월 27일</a> 독립하면서 루마니아의 국기에 몰도바의 문장을 넣은 국기를 채택하였다.</p>
