<p><b>루이 12세</b>(<span style="font-size: smaller;"><a href="/wiki/%ED%94%84%EB%9E%91%EC%8A%A4%EC%96%B4" title="프랑스어">프랑스어</a>:</span> <span lang="fr" xml:lang="fr">Louis XII</span>, <a href="/wiki/1462%EB%85%84" title="1462년">1462년</a> <a href="/wiki/6%EC%9B%94_27%EC%9D%BC" title="6월 27일">6월 27일</a> - <a href="/wiki/1515%EB%85%84" title="1515년">1515년</a> <a href="/wiki/1%EC%9B%94_1%EC%9D%BC" title="1월 1일">1월 1일</a>)은 <a href="/wiki/%ED%94%84%EB%9E%91%EC%8A%A4" title="프랑스">프랑스</a>의 왕이다. <a href="/wiki/%EC%83%A4%EB%A5%BC_6%EC%84%B8" title="샤를 6세">샤를 6세</a>의 동생인 <a class="mw-redirect" href="/wiki/%EB%A3%A8%EC%9D%B4_1%EC%84%B8_%EB%93%9C_%EB%B0%9C%EB%A3%A8%EC%95%84" title="루이 1세 드 발루아">루이 1세 드 발루아</a>의 손자이며, <a href="/wiki/%EC%95%84%EC%9F%81%EC%BF%A0%EB%A5%B4_%EC%A0%84%ED%88%AC" title="아쟁쿠르 전투">아쟁쿠르 전투</a> 이후 <a href="/wiki/%EC%9E%89%EA%B8%80%EB%9E%9C%EB%93%9C" title="잉글랜드">잉글랜드</a>에서 25년을 보낸 <a class="mw-redirect" href="/wiki/%EC%83%A4%EB%A5%BC_%EB%8F%84%EB%A5%BC%EB%A0%88%EC%95%99" title="샤를 도를레앙">샤를 도를레앙</a>의 아들이다.</p>

<p>루이 12세는 <a href="/wiki/1462%EB%85%84" title="1462년">1462년</a> <a href="/wiki/6%EC%9B%94_27%EC%9D%BC" title="6월 27일">6월 27일</a> <a class="mw-redirect" href="/wiki/%ED%88%AC%EB%A0%8C" title="투렌">투렌</a>의 <a href="/wiki/%EB%B8%94%EB%A3%A8%EC%95%84" title="블루아">블루아</a>에 위치한 <a href="/wiki/%EB%B8%94%EB%A3%A8%EC%95%84_%EC%84%B1" title="블루아 성">블루아 성</a>에서 태어났다. <a href="/wiki/1465%EB%85%84" title="1465년">1465년</a> 작위를 물려받아 <a href="/wiki/%EC%98%A4%EB%A5%BC%EB%A0%88%EC%95%99_%EA%B3%B5%EC%9E%91" title="오를레앙 공작">오를레앙 공작</a>이 되었다. 루이 12세가 14살이 되던 <a href="/wiki/1476%EB%85%84" title="1476년">1476년</a> 당시 <a href="/wiki/%ED%94%84%EB%9E%91%EC%8A%A4" title="프랑스">프랑스</a>의 왕이었던 <a href="/wiki/%EB%A3%A8%EC%9D%B4_11%EC%84%B8" title="루이 11세">루이 11세</a>는 그의 둘째 딸인 <a href="/wiki/%EC%9E%94_%EB%93%9C_%ED%94%84%EB%9E%91%EC%8A%A4" title="잔 드 프랑스">잔 드 프랑스</a>를 루이 12세와 결혼시켰다. <a href="/wiki/%EC%83%A4%EB%A5%BC_8%EC%84%B8" title="샤를 8세">샤를 8세</a>의 제위 원년인 <a href="/wiki/1483%EB%85%84" title="1483년">1483년</a> 루이 12세는 샤를 8세에 맞서서 반란을 일으켰으나 <a href="/wiki/1488%EB%85%84" title="1488년">1488년</a> <a class="new" href="/w/index.php?title=%EC%83%9D%ED%86%A0%EB%B0%A9%EB%92%A4%EC%BD%94%EB%A5%B4%EB%AF%B8%EC%97%90_%EC%A0%84%ED%88%AC&amp;action=edit&amp;redlink=1" title="생토방뒤코르미에 전투 (없는 문서)">생토방뒤코르미에 전투</a>에서 패배한 후 붙잡혀 투옥되었다. 이후 <a href="/wiki/1491%EB%85%84" title="1491년">1491년</a> 샤를 8세의 명령으로 석방되었다. <a href="/wiki/1494%EB%85%84" title="1494년">1494년</a> 루이 12세는 <a href="/wiki/%EC%9D%B4%ED%83%88%EB%A6%AC%EC%95%84" title="이탈리아">이탈리아</a> 원정대를 보조하여 <a href="/wiki/%EC%A0%9C%EB%85%B8%EB%B0%94" title="제노바">제노바</a>를 점령하고 북 이탈리아에 남아 <a href="/wiki/%EB%B0%80%EB%9D%BC%EB%85%B8" title="밀라노">밀라노</a>를 위협하였다. 당시 <a href="/wiki/%EC%83%A4%EB%A5%BC_8%EC%84%B8" title="샤를 8세">샤를 8세</a>의 아이들은 모두 유아기에 사망하여, 루이 12세는 <a href="/wiki/%ED%94%84%EB%9E%91%EC%8A%A4" title="프랑스">프랑스</a> 왕위의 제 1 계승자가 되었고 <a href="/wiki/1498%EB%85%84" title="1498년">1498년</a> 샤를 8세에 이어 왕위에 오르게 되었다.</p>
<h2><span class="mw-headline" id=".EC.9E.AC.EC.9C.84">재위</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%A3%A8%EC%9D%B4_12%EC%84%B8&amp;action=edit&amp;section=2" title="부분 편집: 재위">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>루이 12세는 왕위에 오른 후 <a href="/wiki/1499%EB%85%84" title="1499년">1499년</a> <a href="/wiki/%EC%83%A4%EB%A5%BC_8%EC%84%B8" title="샤를 8세">샤를 8세</a>의 미망인이었던 <a href="/wiki/%EC%95%88_%EB%93%9C_%EB%B8%8C%EB%A5%B4%ED%83%80%EB%89%B4" title="안 드 브르타뉴">안 드 브르타뉴</a>와 결혼하였는데, 그녀는 <a href="/wiki/%EC%B6%94%EA%B8%B0%EA%B2%BD" title="추기경">추기경</a> <a class="new" href="/w/index.php?title=%EC%A1%B0%EB%A5%B4%EC%A3%BC_%EB%8B%B9%EB%B6%80%EC%99%80%EC%A6%88&amp;action=edit&amp;redlink=1" title="조르주 당부와즈 (없는 문서)">조르주 당부와즈</a>와 함께 루이 12세에게 큰 영향을 끼칠 수 있었다. 당시 루이 12세는 <a href="/wiki/%EB%82%98%ED%8F%B4%EB%A6%AC_%EC%99%95%EA%B5%AD" title="나폴리 왕국">나폴리 왕국</a>에 대한 소유권을 요구하고 있었는데, <a href="/wiki/1500%EB%85%84" title="1500년">1500년</a> <a class="new" href="/w/index.php?title=%EA%B7%B8%EB%9D%BC%EB%82%98%EB%8B%A4_%ED%98%91%EC%95%BD&amp;action=edit&amp;redlink=1" title="그라나다 협약 (없는 문서)">그라나다 협약</a>에 의해서 <a href="/wiki/%EC%95%84%EB%9D%BC%EA%B3%A4_%EC%99%95%EA%B5%AD" title="아라곤 왕국">아라곤</a>의 왕 <a href="/wiki/%ED%8E%98%EB%A5%B4%EB%82%9C%EB%8F%84_2%EC%84%B8_(%EC%95%84%EB%9D%BC%EA%B3%A4)" title="페르난도 2세 (아라곤)">페르난도 2세</a>와 <a href="/wiki/%EB%82%98%ED%8F%B4%EB%A6%AC_%EC%99%95%EA%B5%AD" title="나폴리 왕국">나폴리 왕국</a>을 분할하기로 하였으나 결국에는 <a href="/wiki/1501%EB%85%84" title="1501년">1501년</a> <a href="/wiki/%ED%94%84%EB%9E%91%EC%8A%A4" title="프랑스">프랑스</a>와 아라곤 왕국 사이에 전쟁이 발발하고 결국끝에 <a href="/wiki/1504%EB%85%84" title="1504년">1504년</a> 프랑스는 나폴리를 모두 잃어버렸다.</p>
<p>루이 12세는 나폴리를 요구한 것에다가 할아버지인 <a href="/wiki/%EC%98%A4%EB%A5%BC%EB%A0%88%EC%95%99_%EA%B3%B5%EC%9E%91_%EB%A3%A8%EC%9D%B4_1%EC%84%B8_%EB%93%9C_%EB%B0%9C%EB%A3%A8%EC%95%84" title="오를레앙 공작 루이 1세 드 발루아">오를레앙 공작 루이 1세</a>가 <a class="new" href="/w/index.php?title=%EB%B0%9C%EB%A0%8C%ED%8B%B0%EB%82%98_%EB%B9%84%EC%8A%A4%EC%BD%98%ED%8B%B0&amp;action=edit&amp;redlink=1" title="발렌티나 비스콘티 (없는 문서)">발렌티나 비스콘티</a>와 결혼 한 것을 빌미로 <a href="/wiki/%EB%B0%80%EB%9D%BC%EB%85%B8" title="밀라노">밀라노</a>까지 요구하기 시작하였고, 이를 위하여 그는 몇 번의 <a href="/wiki/%EC%9D%B4%ED%83%88%EB%A6%AC%EC%95%84" title="이탈리아">이탈리아</a> 원정을 일으키기도 하였다. <a href="/wiki/1499%EB%85%84" title="1499년">1499년</a> 루이 12세는 <a href="/wiki/%EB%A3%A8%EB%8F%84%EB%B9%84%EC%BD%94_%EC%8A%A4%ED%8F%AC%EB%A5%B4%EC%B0%A8" title="루도비코 스포르차">루도비코 스포르차</a>에게서 밀라노를 빼앗았고, 12년동안 통지하였다. 또한 <a href="/wiki/%EB%B2%A0%EB%84%A4%EC%B9%98%EC%95%84_%EA%B3%B5%ED%99%94%EA%B5%AD" title="베네치아 공화국">베네치아 공화국</a>에 대항하기 위하여 <a href="/wiki/1508%EB%85%84" title="1508년">1508년</a> <a href="/wiki/%EA%B5%90%ED%99%A9" title="교황">교황</a> <a class="mw-redirect" href="/wiki/%EC%9C%A8%EB%A6%AC%EC%98%A4_2%EC%84%B8" title="율리오 2세">율리오 2세</a>, <a href="/wiki/%EC%8B%A0%EC%84%B1_%EB%A1%9C%EB%A7%88_%EC%A0%9C%EA%B5%AD" title="신성 로마 제국">신성 로마 제국</a> 황제 <a href="/wiki/%EB%A7%89%EC%8B%9C%EB%B0%80%EB%A6%AC%EC%95%88_1%EC%84%B8_(%EC%8B%A0%EC%84%B1_%EB%A1%9C%EB%A7%88_%EC%A0%9C%EA%B5%AD)" title="막시밀리안 1세 (신성 로마 제국)">막시밀리안 1세</a>, 아라곤의 왕 <a href="/wiki/%ED%8E%98%EB%A5%B4%EB%82%9C%EB%8F%84_2%EC%84%B8_(%EC%95%84%EB%9D%BC%EA%B3%A4)" title="페르난도 2세 (아라곤)">페르난도 2세</a> 등과 함께 <a class="new" href="/w/index.php?title=%EC%BA%89%EB%B8%8C%EB%A0%88_%EB%8F%99%EB%A7%B9&amp;action=edit&amp;redlink=1" title="캉브레 동맹 (없는 문서)">캉브레 동맹</a>을 조직하여 <a class="new" href="/w/index.php?title=%EC%BA%89%EB%B8%8C%EB%A0%88_%EB%8F%99%EB%A7%B9_%EC%A0%84%EC%9F%81&amp;action=edit&amp;redlink=1" title="캉브레 동맹 전쟁 (없는 문서)">캉브레 동맹 전쟁</a>을 일으키며 <a href="/wiki/1509%EB%85%84" title="1509년">1509년</a> <a class="new" href="/w/index.php?title=%EC%95%84%EB%83%90%EB%8D%B8%EB%A1%9C_%EC%A0%84%ED%88%AC&amp;action=edit&amp;redlink=1" title="아냐델로 전투 (없는 문서)">아냐델로 전투</a>에서 승리하였으나, 베네치아와 교황이 동맹을 맺고 이후 교황 율리오 2세가 신성 동맹을 조직하여 프랑스를 저지하기 시작한데다가, 신성 로마 제국, 아라곤 왕국, 잉글랜드 왕국 등이 신성 동맹에 가담하자 프랑스는 이탈리아에서 밀려나기 시작하여 결국 <a href="/wiki/1513%EB%85%84" title="1513년">1513년</a> <a <p>이러한 전쟁이 있었음에도 불구하고 국내 문제에서는 <a href="/wiki/1504%EB%85%84" title="1504년">1504년</a> 재정 개혁과 <a href="/wiki/1508%EB%85%84" title="1508년">1508년</a> 세금 동결 및 세금 징수과정 개선 등 많은 노력을 기울여 <a href="/wiki/%ED%94%84%EB%9E%91%EC%8A%A4" title="프랑스">프랑스</a> 국내에서 큰 지지를 받아 <a href="/wiki/1506%EB%85%84" title="1506년">1506년</a> <a href="/wiki/%ED%88%AC%EB%A5%B4" title="투르">투르</a>에서 열린 <a href="/wiki/%EC%82%BC%EB%B6%80%ED%9A%8C" title="삼부회">삼부회</a>에서 루이 12세는 '국민의 아버지'(<span style="font-size: smaller;"><a href="/wiki/%ED%94%84%EB%9E%91%EC%8A%A4%EC%96%B4" title="프랑스어">프랑스어</a>:</span> <span lang="fr" xml:lang="fr">Père Du Peuple</span>)이라는 칭호를 얻었다.</p>
<h2><span class="mw-headline" id=".EA.B0.80.EC.A1.B1_.EA.B4.80.EA.B3.84">가족 관계</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%A3%A8%EC%9D%B4_12%EC%84%B8&amp;action=edit&amp;section=3" title="부분 편집: 가족 관계">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>루이 12세는 세 번 결혼하였다. 애초부터 첫 번째 부인인 <a href="/wiki/%EC%9E%94_%EB%93%9C_%ED%94%84%EB%9E%91%EC%8A%A4" title="잔 드 프랑스">잔 드 프랑스</a>를 마음에 들지 않아했던 루이 12세는 왕위에 오른 후 잔과 이혼하고 <a href="/wiki/%EC%83%A4%EB%A5%BC_8%EC%84%B8" title="샤를 8세">샤를 8세</a>의 미망인인 <a href="/wiki/%EC%95%88_%EB%93%9C_%EB%B8%8C%EB%A5%B4%ED%83%80%EB%89%B4" title="안 드 브르타뉴">안 드 브르타뉴</a>와 재혼하여 <a href="/wiki/%ED%94%84%EB%9E%91%EC%88%98%EC%95%84_1%EC%84%B8_(%ED%94%84%EB%9E%91%EC%8A%A4)" title="프랑수아 1세 (프랑스)">프랑수아 1세</a>의 왕비가 <a href="/wiki/%ED%94%84%EB%9E%91%EC%8A%A4%EC%9D%98_%ED%81%B4%EB%A1%9C%EB%93%9C" title="프랑스의 클로드">클로드</a>를 비롯한 두 명의 딸을 두었다. <a href="/wiki/1514%EB%85%84" title="1514년">1514년</a> <a href="/wiki/1%EC%9B%94_9%EC%9D%BC" title="1월 9일">1월 9일</a> 안이 사망하자 <a href="/wiki/%EC%9E%89%EA%B8%80%EB%9E%9C%EB%93%9C" title="잉글랜드">잉글랜드</a>를 신성 동맹으로부터 떼어내기 위하여 잉글랜드의 국왕 <a href="/wiki/%ED%97%A8%EB%A6%AC_8%EC%84%B8" title="헨리 8세">헨리 8세</a>의 여동생이었던 <a class="mw-redirect" href="/wiki/%EB%A9%94%EB%A6%AC_%ED%8A%9C%EB%8D%94_(%ED%94%84%EB%9E%91%EC%8A%A4%EC%9D%98_%EC%99%95%EB%B9%84)" title="메리 튜더 (프랑스의 왕비)">메리 튜더</a>와 세 번째 결혼을 하였다. 루이 12세는 <a href="/wiki/1515%EB%85%84" title="1515년">1515년</a> <a href="/wiki/1%EC%9B%94_1%EC%9D%BC" title="1월 1일">1월 1일</a>에 사망하였다.</p>
