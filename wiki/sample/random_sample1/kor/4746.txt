<p><b>루이 니콜라 보클랭</b>(<span style="font-size: smaller;"><a href="/wiki/%ED%94%84%EB%9E%91%EC%8A%A4%EC%96%B4" title="프랑스어">프랑스어</a>:</span> <span lang="fr" xml:lang="fr">Louis-Nicolas Vauquelin</span>, <a href="/wiki/1763%EB%85%84" title="1763년">1763년</a> <a href="/wiki/5%EC%9B%94_16%EC%9D%BC" title="5월 16일">5월 16일</a> ~ <a href="/wiki/1829%EB%85%84" title="1829년">1829년</a> <a href="/wiki/11%EC%9B%94_14%EC%9D%BC" title="11월 14일">11월 14일</a>)은 <a href="/wiki/%ED%94%84%EB%9E%91%EC%8A%A4" title="프랑스">프랑스</a>의 <a href="/wiki/%EC%95%BD%EC%82%AC" title="약사">약사</a>이자 <a href="/wiki/%ED%99%94%ED%95%99%EC%9E%90" title="화학자">화학자</a>이다. <a href="/wiki/1797%EB%85%84" title="1797년">1797년</a>에는 <a href="/wiki/%ED%81%AC%EB%A1%9C%EB%AE%B4" title="크로뮴">크로뮴</a>을, <a href="/wiki/1798%EB%85%84" title="1798년">1798년</a>에는 <a href="/wiki/%EB%B2%A0%EB%A6%B4%EB%A5%A8" title="베릴륨">베릴륨</a>을 발견하는 데 기여하였다.</p>

<h2><span class="mw-headline" id=".EC.83.9D.EC.95.A0">생애</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%A3%A8%EC%9D%B4_%EB%8B%88%EC%BD%9C%EB%9D%BC_%EB%B3%B4%ED%81%B4%EB%9E%AD&amp;action=edit&amp;section=1" title="부분 편집: 생애">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>보클랭은 <a href="/wiki/1763%EB%85%84" title="1763년">1763년</a> <a href="/wiki/5%EC%9B%94_16%EC%9D%BC" title="5월 16일">5월 16일</a> <a href="/wiki/%ED%94%84%EB%9E%91%EC%8A%A4_%EC%99%95%EA%B5%AD" title="프랑스 왕국">프랑스</a> <a href="/wiki/%EB%85%B8%EB%A5%B4%EB%A7%9D%EB%94%94" title="노르망디">노르망디</a>의 생앙드레데베르토(Saint-André-d'Hébertot)에서 농부의 아들로 태어났다. 1777년에서 1779년 사이에 <a href="/wiki/%EB%A3%A8%EC%95%99" title="루앙">루앙</a>의 한 약제상에서 일하면서 처음으로 <a href="/wiki/%ED%99%94%ED%95%99" title="화학">화학</a>을 접하게 되었다. 이후 우여곡절 끝에 <a href="/wiki/%EC%95%99%ED%88%AC%EC%95%88_%ED%94%84%EB%9E%91%EC%88%98%EC%95%84_%ED%91%B8%EB%A5%B4%ED%81%AC%EB%A3%A8%EC%95%84" title="앙투안 프랑수아 푸르크루아">앙투안 프랑수아 푸르크루아</a>를 만나 친해지게 되었고, 1783년에서 1791년 사이에는 그의 실험실 조수가 되기도 하였다. <a href="/wiki/1829%EB%85%84" title="1829년">1829년</a>에 고향을 방문하던 중 사망하였다.</p>
<h2><span class="mw-headline" id=".EC.97.85.EC.A0.81">업적</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%A3%A8%EC%9D%B4_%EB%8B%88%EC%BD%9C%EB%9D%BC_%EB%B3%B4%ED%81%B4%EB%9E%AD&amp;action=edit&amp;section=2" title="부분 편집: 업적">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p><a href="/wiki/1797%EB%85%84" title="1797년">1797년</a>에는 <a href="/wiki/%EC%8B%9C%EB%B2%A0%EB%A6%AC%EC%95%84" title="시베리아">시베리아</a>산 <a class="new" href="/w/index.php?title=%ED%99%8D%EC%97%B0%EC%84%9D&amp;action=edit&amp;redlink=1" title="홍연석 (없는 문서)">홍연석</a>에서 <a href="/wiki/%ED%81%AC%EB%A1%9C%EB%AE%B4" title="크로뮴">크로뮴</a>을 발견하였으며, 이듬해에는 <a href="/wiki/%EB%85%B9%EC%A3%BC%EC%84%9D" title="녹주석">녹주석</a>에서 <a href="/wiki/%EB%8B%A8%EB%A7%9B" title="단맛">단맛</a>이 나는 <a href="/wiki/%EB%B2%A0%EB%A6%B4%EB%A5%A8" title="베릴륨">베릴륨</a> 염을 최초로 발견하였다. 또, <a href="/wiki/%EB%8C%80%EA%B8%B0%EC%95%95" title="대기압">대기압</a>에서 액체 <a href="/wiki/%EC%95%94%EB%AA%A8%EB%8B%88%EC%95%84" title="암모니아">암모니아</a>를 얻는 데 성공하기도 하였다. <a href="/wiki/1806%EB%85%84" title="1806년">1806년</a>에는 조수인 <a class="new" href="/w/index.php?title=%ED%94%BC%EC%97%90%EB%A5%B4_%EB%A1%9C%EB%B9%84%EC%BC%80&amp;action=edit&amp;redlink=1" title="피에르 로비케 (없는 문서)">피에르 로비케</a>(Pierre Jean Robiquet)와 함께 <a href="/wiki/%EC%95%84%EC%8A%A4%ED%8C%8C%EB%9D%BC%EA%B1%B0%EC%8A%A4" title="아스파라거스">아스파라거스</a>를 연구하여 최초로 발견된 <a href="/wiki/%EC%95%84%EB%AF%B8%EB%85%B8%EC%82%B0" title="아미노산">아미노산</a>인 <a href="/wiki/%EC%95%84%EC%8A%A4%ED%8C%8C%EB%9D%BC%EA%B8%B4" title="아스파라긴">아스파라긴</a>을 발견하였다. 보클랭은 이 밖에도 <a class="mw-redirect" href="/wiki/%EC%82%AC%EA%B3%BC" title="사과">사과</a>에서 <a href="/wiki/%ED%8E%99%ED%8B%B4" title="펙틴">펙틴</a>과 <a href="/wiki/%EB%A7%90%EC%82%B0" title="말산">말산</a>(사과산)을 발견하기도 하였다.</p>
