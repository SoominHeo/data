<p><b>루이스 펠리페 몬티</b>(<span style="font-size: smaller;"><a href="/wiki/%EC%9D%B4%ED%83%88%EB%A6%AC%EC%95%84%EC%96%B4" title="이탈리아어">이탈리아어</a>:</span> <span lang="it" xml:lang="it">Luis Felipe Monti</span>, <span style="font-size: smaller;"><a href="/wiki/%EC%8A%A4%ED%8E%98%EC%9D%B8%EC%96%B4" title="스페인어">스페인어</a>:</span> <span lang="es" xml:lang="es">Luis Felipe Monti</span>, <a href="/wiki/1901%EB%85%84" title="1901년">1901년</a> <a href="/wiki/5%EC%9B%94_15%EC%9D%BC" title="5월 15일">5월 15일</a>, <a href="/wiki/%EC%95%84%EB%A5%B4%ED%97%A8%ED%8B%B0%EB%82%98" title="아르헨티나">아르헨티나</a> <a href="/wiki/%EB%B6%80%EC%97%90%EB%85%B8%EC%8A%A4%EC%95%84%EC%9D%B4%EB%A0%88%EC%8A%A4" title="부에노스아이레스">부에노스아이레스</a> ~ <a href="/wiki/1983%EB%85%84" title="1983년">1983년</a> <a href="/wiki/9%EC%9B%94_9%EC%9D%BC" title="9월 9일">9월 9일</a>)는 전 축구 선수이자 전 축구 감독으로, 이탈리아계 아르헨티나인이다.</p>
<p>그는 <a href="/wiki/1920%EB%85%84%EB%8C%80" title="1920년대">1920년대</a>부터 <a href="/wiki/%EC%95%84%EB%A5%B4%ED%97%A8%ED%8B%B0%EB%82%98_%EC%B6%95%EA%B5%AC_%EA%B5%AD%EA%B0%80%EB%8C%80%ED%91%9C%ED%8C%80" title="아르헨티나 축구 국가대표팀">아르헨티나 축구 국가대표팀</a>에서 활약했으며, 최초의 월드컵인 <a href="/wiki/1930%EB%85%84_FIFA_%EC%9B%94%EB%93%9C%EC%BB%B5" title="1930년 FIFA 월드컵">1930년 FIFA 월드컵</a>에서 몬티는 <a href="/wiki/%EC%95%84%EB%A5%B4%ED%97%A8%ED%8B%B0%EB%82%98_%EC%B6%95%EA%B5%AC_%EA%B5%AD%EA%B0%80%EB%8C%80%ED%91%9C%ED%8C%80" title="아르헨티나 축구 국가대표팀">아르헨티나</a>로 출전하여 준우승을 차지하였으나 <a href="/wiki/1934%EB%85%84_FIFA_%EC%9B%94%EB%93%9C%EC%BB%B5" title="1934년 FIFA 월드컵">1934년 FIFA 월드컵</a>이 개최되자 조국인 아르헨티나가 이 월드컵에 출전함에도 불구하고 <a href="/wiki/%EC%9D%B4%ED%83%88%EB%A6%AC%EC%95%84_%EC%B6%95%EA%B5%AC_%EA%B5%AD%EA%B0%80%EB%8C%80%ED%91%9C%ED%8C%80" title="이탈리아 축구 국가대표팀">이탈리아</a>로 출전하여 우승을 차지했다. 몬티가 뛴 이탈리아는 이 월드컵에서 우승했으나 몬티가 빠져나간 아르헨티나는 약체인 스웨덴에게 패하며 초반에 탈락했다. 몬티를 시작으로 아르헨티나 축구 국가대표팀은 <a href="/wiki/1978%EB%85%84_FIFA_%EC%9B%94%EB%93%9C%EC%BB%B5" title="1978년 FIFA 월드컵">1978년 FIFA 월드컵</a>에서 우승하기까지 끊임없이 극심한 선수유출에 시달려야 했다.</p>
