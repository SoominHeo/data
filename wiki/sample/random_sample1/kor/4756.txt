<p><b>루이스 알베르토 쿠비야 알메이다</b>(<span style="font-size: smaller;"><a href="/wiki/%EC%8A%A4%ED%8E%98%EC%9D%B8%EC%96%B4" title="스페인어">스페인어</a>:</span> <span lang="es" xml:lang="es">Luis Alberto Cubilla Almeida</span>, <a href="/wiki/1940%EB%85%84" title="1940년">1940년</a> <a href="/wiki/3%EC%9B%94_28%EC%9D%BC" title="3월 28일">3월 28일</a> ~ <a href="/wiki/2013%EB%85%84" title="2013년">2013년</a> <a href="/wiki/3%EC%9B%94_3%EC%9D%BC" title="3월 3일">3월 3일</a>)는 전 <a href="/wiki/%EC%9A%B0%EB%A3%A8%EA%B3%BC%EC%9D%B4" title="우루과이">우루과이</a>의 축구 선수이자 축구 감독이었다. 쿠비야는 선수 시절 16번의 우승을 이루어냈고, 감독으로서는 17번의 우승으로 <a href="/wiki/%EB%82%A8%EC%95%84%EB%A9%94%EB%A6%AC%EC%B9%B4" title="남아메리카">남아메리카</a> 감독들 사이에서 가장 많은 우승을 한 감독 가운데 한 명이 되었다. <a class="extiw" href="https://en.wikipedia.org/wiki/FIFA_Player_of_the_Century" title="en:FIFA Player of the Century">FIFA 선정 20세기 최우수 선수</a>에서 IFFHS는 쿠비야를 64번째에 뽑았다.</p>

<h2><span class="mw-headline" id=".ED.81.B4.EB.9F.BD_.EA.B2.BD.EB.A0.A5">클럽 경력</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%A3%A8%EC%9D%B4%EC%8A%A4_%EC%BF%A0%EB%B9%84%EC%95%BC&amp;action=edit&amp;section=1" title="부분 편집: 클럽 경력">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>쿠비야는 콜론 데 파이산두의 유소년 팀에서 축구를 시작했으며, <a href="/wiki/1957%EB%85%84" title="1957년">1957년</a> <a href="/wiki/%ED%8E%98%EB%83%90%EB%A1%A4" title="페냐롤">페냐롤</a>에 입단, 4번의 <a href="/wiki/%EC%9A%B0%EB%A3%A8%EA%B3%BC%EC%9D%B4_%ED%94%84%EB%A6%AC%EB%A9%94%EB%9D%BC_%EB%94%94%EB%B9%84%EC%8B%9C%EC%98%A8" title="우루과이 프리메라 디비시온">리그</a> 우승(<a href="/wiki/1958%EB%85%84" title="1958년">1958년</a>, <a href="/wiki/1959%EB%85%84" title="1959년">1959년</a>, <a href="/wiki/1960%EB%85%84" title="1960년">1960년</a>, <a href="/wiki/1961%EB%85%84" title="1961년">1961년</a>)과 2번의 <a href="/wiki/%EC%BD%94%ED%8C%8C_%EB%A6%AC%EB%B2%A0%EB%A5%B4%ED%83%80%EB%8F%84%EB%A0%88%EC%8A%A4" title="코파 리베르타도레스">코파 리베르타도레스</a> 우승(<a href="/wiki/1960%EB%85%84" title="1960년">1960년</a>, <a href="/wiki/1961%EB%85%84" title="1961년">1961년</a>), 그리고 한 번의 <a href="/wiki/%EC%9D%B8%ED%84%B0%EC%BD%98%ED%8B%B0%EB%84%A8%ED%84%B8%EC%BB%B5_(%EC%B6%95%EA%B5%AC)" title="인터콘티넨털컵 (축구)">인터콘티넨털컵</a> 우승(<a href="/wiki/1961%EB%85%84" title="1961년">1961년</a>)을 이루었다. <a href="/wiki/1962%EB%85%84" title="1962년">1962년</a>에는 <a href="/wiki/FC_%EB%B0%94%EB%A5%B4%EC%85%80%EB%A1%9C%EB%82%98" title="FC 바르셀로나">FC 바르셀로나</a>에 입단해 <a href="/wiki/1963%EB%85%84" title="1963년">1963년</a> <a href="/wiki/%EC%BD%94%ED%8C%8C_%EB%8D%B8_%EB%A0%88%EC%9D%B4" title="코파 델 레이">코파 델 레이</a>에서 우승했다.</p>
<p>쿠비야는 <a href="/wiki/1964%EB%85%84" title="1964년">1964년</a> <a href="/wiki/%EB%82%A8%EC%95%84%EB%A9%94%EB%A6%AC%EC%B9%B4" title="남아메리카">남아메리카</a>로 돌아와 <a class="new" href="/w/index.php?title=%EB%A6%AC%EB%B2%A0_%ED%94%8C%EB%A0%88%EC%9D%B4%ED%8A%B8&amp;action=edit&amp;redlink=1" title="리베 플레이트 (없는 문서)">리베 플레이트</a>에 들어갔고, <a href="/wiki/1969%EB%85%84" title="1969년">1969년</a> <a class="mw-redirect mw-disambig" href="/wiki/%ED%81%B4%EB%9F%BD_%EB%82%98%EC%8B%9C%EC%98%A4%EB%82%A0" title="클럽 나시오날">클럽 나시오날</a>로 이적해 4번의 <a href="/wiki/%EC%9A%B0%EB%A3%A8%EA%B3%BC%EC%9D%B4_%ED%94%84%EB%A6%AC%EB%A9%94%EB%9D%BC_%EB%94%94%EB%B9%84%EC%8B%9C%EC%98%A8" title="우루과이 프리메라 디비시온">리그</a> 우승(<a href="/wiki/1969%EB%85%84" title="1969년">1969년</a>, <a href="/wiki/1970%EB%85%84" title="1970년">1970년</a>, <a href="/wiki/1971%EB%85%84" title="1971년">1971년</a>, <a href="/wiki/1972%EB%85%84" title="1972년">1972년</a>)과 한 번의 <a href="/wiki/%EC%BD%94%ED%8C%8C_%EB%A6%AC%EB%B2%A0%EB%A5%B4%ED%83%80%EB%8F%84%EB%A0%88%EC%8A%A4" title="코파 리베르타도레스">코파 리베르타도레스</a> 우승(<a href="/wiki/1971%EB%85%84" title="1971년">1971년</a>), 한 번의 <a href="/wiki/%EC%9D%B8%ED%84%B0%EC%BD%98%ED%8B%B0%EB%84%A8%ED%84%B8%EC%BB%B5_(%EC%B6%95%EA%B5%AC)" title="인터콘티넨털컵 (축구)">인터콘티넨털컵</a> 우승(<a href="/wiki/1971%EB%85%84" title="1971년">1971년</a>), 한 번의 <a href="/wiki/%EC%BD%94%ED%8C%8C_%EC%9D%B8%ED%85%8C%EB%9D%BC%EB%A9%94%EB%A6%AC%EC%B9%B4%EB%82%98" title="코파 인테라메리카나">코파 인테라메리카나</a> 우승(<a href="/wiki/1972%EB%85%84" title="1972년">1972년</a>)을 달성했다.</p>
<p>이후 <a href="/wiki/%EC%82%B0%ED%8B%B0%EC%95%84%EA%B3%A0_%EB%AA%A8%EB%8B%9D" title="산티아고 모닝">산티아고 모닝</a>과 <a class="mw-redirect" href="/wiki/%EB%8D%B0%ED%8E%9C%EC%86%8C%EB%A5%B4_%EC%8A%A4%ED%8F%AC%EB%A5%B4%ED%8C%85_%ED%81%B4%EB%9F%BD" title="데펜소르 스포르팅 클럽">데펜소르 스포르팅 클럽</a>에서 뛰었으며, <a href="/wiki/1976%EB%85%84" title="1976년">1976년</a> <a class="mw-redirect" href="/wiki/%EB%8D%B0%ED%8E%9C%EC%86%8C%EB%A5%B4_%EC%8A%A4%ED%8F%AC%EB%A5%B4%ED%8C%85_%ED%81%B4%EB%9F%BD" title="데펜소르 스포르팅 클럽">데펜소르 스포르팅 클럽</a> 소속으로 <a href="/wiki/%ED%8E%98%EB%83%90%EB%A1%A4" title="페냐롤">페냐롤</a>과 <a class="mw-redirect mw-disambig" href="/wiki/%ED%81%B4%EB%9F%BD_%EB%82%98%EC%8B%9C%EC%98%A4%EB%82%A0" title="클럽 나시오날">클럽 나시오날</a>의 <a href="/wiki/%EC%9A%B0%EB%A3%A8%EA%B3%BC%EC%9D%B4_%ED%94%84%EB%A6%AC%EB%A9%94%EB%9D%BC_%EB%94%94%EB%B9%84%EC%8B%9C%EC%98%A8" title="우루과이 프리메라 디비시온">리그</a> 2강 체제를 깨고 <a href="/wiki/%EC%9A%B0%EB%A3%A8%EA%B3%BC%EC%9D%B4_%ED%94%84%EB%A6%AC%EB%A9%94%EB%9D%BC_%EB%94%94%EB%B9%84%EC%8B%9C%EC%98%A8" title="우루과이 프리메라 디비시온">리그</a> 우승을 차지했다.</p>
<h2><span class="mw-headline" id=".EA.B5.AD.EA.B0.80.EB.8C.80.ED.91.9C.ED.8C.80_.EA.B2.BD.EB.A0.A5">국가대표팀 경력</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%A3%A8%EC%9D%B4%EC%8A%A4_%EC%BF%A0%EB%B9%84%EC%95%BC&amp;action=edit&amp;section=2" title="부분 편집: 국가대표팀 경력">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>쿠비야는 <a href="/wiki/1959%EB%85%84" title="1959년">1959년</a>부터 <a href="/wiki/1974%EB%85%84" title="1974년">1974년</a>까지 <a href="/wiki/%EC%9A%B0%EB%A3%A8%EA%B3%BC%EC%9D%B4_%EC%B6%95%EA%B5%AC_%EA%B5%AD%EA%B0%80%EB%8C%80%ED%91%9C%ED%8C%80" title="우루과이 축구 국가대표팀">우루과이 축구 국가대표팀</a>의 일원으로서 활약했다. <a href="/wiki/1962%EB%85%84_FIFA_%EC%9B%94%EB%93%9C%EC%BB%B5" title="1962년 FIFA 월드컵">1962년</a>, <a href="/wiki/1970%EB%85%84_FIFA_%EC%9B%94%EB%93%9C%EC%BB%B5" title="1970년 FIFA 월드컵">1970년</a>, <a href="/wiki/1974%EB%85%84_FIFA_%EC%9B%94%EB%93%9C%EC%BB%B5" title="1974년 FIFA 월드컵">1974년 FIFA 월드컵</a>에 출전했으며, <a href="/wiki/1970%EB%85%84_FIFA_%EC%9B%94%EB%93%9C%EC%BB%B5" title="1970년 FIFA 월드컵">1970년 FIFA 월드컵</a>에서는 4위를 기록했다.</p>
<h2><span class="mw-headline" id=".EA.B0.90.EB.8F.85_.EA.B2.BD.EB.A0.A5">감독 경력</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%A3%A8%EC%9D%B4%EC%8A%A4_%EC%BF%A0%EB%B9%84%EC%95%BC&amp;action=edit&amp;section=3" title="부분 편집: 감독 경력">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>쿠비야는 감독 경력을 <a href="/wiki/1978%EB%85%84" title="1978년">1978년</a> <a class="mw-redirect" href="/wiki/%ED%81%B4%EB%9F%BD_%EC%98%AC%EB%A6%BC%ED%94%BC%EC%95%84" title="클럽 올림피아">클럽 올림피아</a>에서 시작한 뒤 <a class="mw-redirect" href="/wiki/%ED%81%B4%EB%9F%BD_%EC%98%AC%EB%A6%BC%ED%94%BC%EC%95%84" title="클럽 올림피아">클럽 올림피아</a>에서 많은 우승을 이루어냈다. 이후 여러 클럽 감독을 거친 뒤 <a href="/wiki/1990%EB%85%84" title="1990년">1990년</a>부터 <a href="/wiki/1993%EB%85%84" title="1993년">1993년</a>까지 <a href="/wiki/%EC%9A%B0%EB%A3%A8%EA%B3%BC%EC%9D%B4_%EC%B6%95%EA%B5%AC_%EA%B5%AD%EA%B0%80%EB%8C%80%ED%91%9C%ED%8C%80" title="우루과이 축구 국가대표팀">우루과이 축구 국가대표팀</a> 감독을 지내며 형 <a class="new" href="/w/index.php?title=%ED%8E%98%EB%93%9C%EB%A1%9C_%EC%BF%A0%EB%B9%84%EC%95%BC&amp;action=edit&amp;redlink=1" title="페드로 쿠비야 (없는 문서)">페드로 쿠비야</a>와 함께 일했다. <a href="/wiki/1991%EB%85%84_%EC%BD%94%ED%8C%8C_%EC%95%84%EB%A9%94%EB%A6%AC%EC%B9%B4" title="1991년 코파 아메리카">1991년 코파 아메리카</a>에서는 조별 예선 탈락, <a href="/wiki/1993%EB%85%84_%EC%BD%94%ED%8C%8C_%EC%95%84%EB%A9%94%EB%A6%AC%EC%B9%B4" title="1993년 코파 아메리카">1993년 코파 아메리카</a>에서는 8강까지 진출했으나 <a href="/wiki/%EC%BD%9C%EB%A1%AC%EB%B9%84%EC%95%84_%EC%B6%95%EA%B5%AC_%EA%B5%AD%EA%B0%80%EB%8C%80%ED%91%9C%ED%8C%80" title="콜롬비아 축구 국가대표팀">콜롬비아</a>에게 승부차기 끝에 졌다.</p>
<p>이후 다시 여러 클럽의 감독으로 있은 뒤 <a href="/wiki/2010%EB%85%84" title="2010년">2010년</a> 다시 <a class="mw-redirect" href="/wiki/%ED%81%B4%EB%9F%BD_%EC%98%AC%EB%A6%BC%ED%94%BC%EC%95%84" title="클럽 올림피아">클럽 올림피아</a>의 감독을 맡은 뒤 <a href="/wiki/2012%EB%85%84" title="2012년">2012년</a> <a href="/wiki/%ED%8C%8C%EB%9D%BC%EA%B3%BC%EC%9D%B4" title="파라과이">파라과이</a>의 <a class="new" href="/w/index.php?title=%ED%83%80%EC%B9%B4%EB%A6%AC_FC&amp;action=edit&amp;redlink=1" title="타카리 FC (없는 문서)">타카리 FC</a>의 감독을 지냈고, <a href="/wiki/2013%EB%85%84" title="2013년">2013년</a> <a href="/wiki/3%EC%9B%94_3%EC%9D%BC" title="3월 3일">3월 3일</a> <a href="/wiki/%ED%8C%8C%EB%9D%BC%EA%B3%BC%EC%9D%B4" title="파라과이">파라과이</a> <a href="/wiki/%EC%95%84%EC%88%9C%EC%8B%9C%EC%98%A8" title="아순시온">아순시온</a>에서 사망했다.</p>
