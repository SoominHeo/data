<p><b>루이스 펠리피 리마 올리베이라</b> ( <a href="/wiki/1993%EB%85%84" title="1993년">1993년</a> <a href="/wiki/1%EC%9B%94_21%EC%9D%BC" title="1월 21일">1월 21일</a> ~ )는 <a href="/wiki/%EB%B8%8C%EB%9D%BC%EC%A7%88" title="브라질">브라질</a>의 <a href="/wiki/%EC%B6%95%EA%B5%AC" title="축구">축구</a> 선수이며 포지션은 <a href="/wiki/%EB%AF%B8%EB%93%9C%ED%95%84%EB%8D%94" title="미드필더">미드필더</a>이다. 현재 <a href="/wiki/K%EB%A6%AC%EA%B7%B8_%ED%81%B4%EB%9E%98%EC%8B%9D" title="K리그 클래식">K리그 클래식</a> <a href="/wiki/%ED%8F%AC%ED%95%AD_%EC%8A%A4%ED%8B%B8%EB%9F%AC%EC%8A%A4" title="포항 스틸러스">포항 스틸러스</a>에서 활약하고 있으며 K리그 등록명은 <b>무랄랴</b>이다.</p>
<p><a href="/wiki/2016%EB%85%84" title="2016년">2016년</a> 여름, 포항에 임대영입되었고, <a href="/wiki/10%EC%9B%94_2%EC%9D%BC" title="10월 2일">10월 2일</a> 열린 <a href="/wiki/%EC%84%B1%EB%82%A8_FC" title="성남 FC">성남 FC</a> 원정 경기에서 중거리슛으로 데뷔골을 성공시켰다.</p>
