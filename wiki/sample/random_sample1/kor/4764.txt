<p><b>루이지 디 비아조</b> (<span style="font-size: smaller;"><a href="/wiki/%EC%9D%B4%ED%83%88%EB%A6%AC%EC%95%84%EC%96%B4" title="이탈리아어">이탈리아어</a>:</span> <span lang="it" xml:lang="it">Luigi Di Biagio</span>, <a href="/wiki/1971%EB%85%84" title="1971년">1971년</a> <a href="/wiki/6%EC%9B%94_3%EC%9D%BC" title="6월 3일">6월 3일</a>, <a href="/wiki/%EC%9D%B4%ED%83%88%EB%A6%AC%EC%95%84" title="이탈리아">이탈리아</a> <a href="/wiki/%EB%A1%9C%EB%A7%88" title="로마">로마</a> ~ )는 <a href="/wiki/%EC%9D%B4%ED%83%88%EB%A6%AC%EC%95%84" title="이탈리아">이탈리아</a>의 은퇴한 <a href="/wiki/%EC%B6%95%EA%B5%AC_%EC%84%A0%EC%88%98" title="축구 선수">축구 선수</a>이자, 현재 <a href="/wiki/%EC%9D%B4%ED%83%88%EB%A6%AC%EC%95%84_U-21_%EC%B6%95%EA%B5%AC_%EA%B5%AD%EA%B0%80%EB%8C%80%ED%91%9C%ED%8C%80" title="이탈리아 U-21 축구 국가대표팀">이탈리아 U-21 축구 국가대표팀</a>의 감독이다.</p>
<h2><span class="mw-headline" id="1998.EB.85.84_.ED.94.84.EB.9E.91.EC.8A.A4_.EC.9B.94.EB.93.9C.EC.BB.B5">1998년 프랑스 월드컵</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%A3%A8%EC%9D%B4%EC%A7%80_%EB%94%94_%EB%B9%84%EC%95%84%EC%A1%B0&amp;action=edit&amp;section=1" title="부분 편집: 1998년 프랑스 월드컵">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>디 비아조는 1998년 28살에 국가대표에 데뷔했고, 그 해 6월 <a href="/wiki/1998%EB%85%84_FIFA_%EC%9B%94%EB%93%9C%EC%BB%B5" title="1998년 FIFA 월드컵">프랑스 월드컵</a>에서 출전했다. 특히 디 비아조는 이탈리아의 주전 <a class="mw-redirect" href="/wiki/%EC%88%98%EB%B9%84%ED%98%95_%EB%AF%B8%EB%93%9C%ED%95%84%EB%8D%94" title="수비형 미드필더">수비형 미드필더</a>로 맹활약을 하면서 팀의 8강 진출을 이끌었고, <a href="/wiki/%EC%B9%B4%EB%A9%94%EB%A3%AC_%EC%B6%95%EA%B5%AC_%EA%B5%AD%EA%B0%80%EB%8C%80%ED%91%9C%ED%8C%80" title="카메룬 축구 국가대표팀">카메룬</a>과의 2차전에서는 득점을 기록하기도 했었다. 하지만 <a href="/wiki/%ED%94%84%EB%9E%91%EC%8A%A4_%EC%B6%95%EA%B5%AC_%EA%B5%AD%EA%B0%80%EB%8C%80%ED%91%9C%ED%8C%80" title="프랑스 축구 국가대표팀">프랑스</a>와의 8강전 승부차기에서 마지막 키커로 나섰으나 슛은 크로스바를 맞고 나오면서 이탈리아는 또다시 승부차기 징크스를 이어가야만 했었다.</p>
<h2><span class="mw-headline" id=".EC.9C.A0.EB.A1.9C_2000">유로 2000</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%A3%A8%EC%9D%B4%EC%A7%80_%EB%94%94_%EB%B9%84%EC%95%84%EC%A1%B0&amp;action=edit&amp;section=2" title="부분 편집: 유로 2000">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>디 비아조는 앞서 두 경기에는 결장했지만 <a href="/wiki/%EB%B2%A8%EA%B8%B0%EC%97%90_%EC%B6%95%EA%B5%AC_%EA%B5%AD%EA%B0%80%EB%8C%80%ED%91%9C%ED%8C%80" title="벨기에 축구 국가대표팀">벨기에</a>와의 최종전에서 처음으로 선발로 출전했고, 이 경기에서 선제 득점을 기록했다. 그리고 특히 개최국 <a href="/wiki/%EB%84%A4%EB%8D%9C%EB%9E%80%EB%93%9C_%EC%B6%95%EA%B5%AC_%EA%B5%AD%EA%B0%80%EB%8C%80%ED%91%9C%ED%8C%80" title="네덜란드 축구 국가대표팀">네덜란드</a>와의 준결승전에서는 수적 열세 속에서도 120분을 무득점으로 버텼고, 승부차기에서 1번 키커로 나가 기선을 제압했다. 이탈리아는 <a href="/wiki/%ED%94%84%EB%9E%80%EC%B2%B4%EC%8A%A4%EC%BD%94_%ED%86%A8%EB%8F%84" title="프란체스코 톨도">톨도</a> 골키퍼의 활약으로 3대 1로 이기고 결승전에 진출했지만, <a href="/wiki/%ED%94%84%EB%9E%91%EC%8A%A4_%EC%B6%95%EA%B5%AC_%EA%B5%AD%EA%B0%80%EB%8C%80%ED%91%9C%ED%8C%80" title="프랑스 축구 국가대표팀">프랑스</a>의 <a href="/wiki/%EB%8B%A4%EB%B9%84%EB%93%9C_%ED%8A%B8%EB%A0%88%EC%A0%9C%EA%B2%8C" title="다비드 트레제게">트레제게</a>에 <a href="/wiki/%EA%B3%A8%EB%93%A0%EA%B3%A8" title="골든골">골든골</a>을 허용하면서 준우승에 머물렀다.</p>
