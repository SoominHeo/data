<p><b>루이지 카도르나</b>(<span style="font-size: smaller;"><a href="/wiki/%EC%9D%B4%ED%83%88%EB%A6%AC%EC%95%84%EC%96%B4" title="이탈리아어">이탈리아어</a>:</span> <span lang="it" xml:lang="it">Luigi Cadorna</span>, <a href="/wiki/1850%EB%85%84" title="1850년">1850년</a> <a href="/wiki/9%EC%9B%94_4%EC%9D%BC" title="9월 4일">9월 4일</a>, <a href="/wiki/%EB%B2%A0%EB%A5%B4%EB%B0%94%EB%8B%88%EC%95%84" title="베르바니아">베르바니아</a> ~ <a href="/wiki/1928%EB%85%84" title="1928년">1928년</a> <a href="/wiki/12%EC%9B%94_21%EC%9D%BC" title="12월 21일">12월 21일</a>, <a class="new" href="/w/index.php?title=%EB%B3%B4%EB%A5%B4%EB%94%94%EA%B2%8C%EB%9D%BC&amp;action=edit&amp;redlink=1" title="보르디게라 (없는 문서)">보르디게라</a>) 는 <a href="/wiki/%EC%9D%B4%ED%83%88%EB%A6%AC%EC%95%84_%EC%99%95%EA%B5%AD_(1861%EB%85%84_~_1946%EB%85%84)" title="이탈리아 왕국 (1861년 ~ 1946년)">이탈리아</a>의 육군 원수로, <a href="/wiki/%EC%A0%9C1%EC%B0%A8_%EC%84%B8%EA%B3%84_%EB%8C%80%EC%A0%84" title="제1차 세계 대전">제1차 세계 대전</a> 동안 이탈리아군의 참모총장으로 활약하였다.</p>
<p><a class="mw-redirect" href="/wiki/%EC%A0%9C1%EC%B0%A8_%EC%84%B8%EA%B3%84%EB%8C%80%EC%A0%84" title="제1차 세계대전">제1차 세계대전</a> 당시, 30개월 동안 참모총장을 지내면서 이탈리아군을 지휘해 <a href="/wiki/1915%EB%85%84" title="1915년">1915년</a>부터 이손초 강에서 <a href="/wiki/%EC%98%A4%EC%8A%A4%ED%8A%B8%EB%A6%AC%EC%95%84-%ED%97%9D%EA%B0%80%EB%A6%AC_%EC%A0%9C%EA%B5%AD" title="오스트리아-헝가리 제국">오스트리아</a>의 <a href="/wiki/%EC%98%A4%EC%8A%A4%ED%8A%B8%EB%A6%AC%EC%95%84-%ED%97%9D%EA%B0%80%EB%A6%AC_%EC%A0%9C%EA%B5%AD_%EC%9C%A1%EA%B5%B0" title="오스트리아-헝가리 제국 육군">군대</a>와 전투를 벌여서 5월 14일부터 6월까지 <a href="/wiki/%ED%8A%B8%EB%A0%8C%ED%8B%B0%EB%85%B8_%EA%B3%B5%EC%84%B8" title="트렌티노 공세">트렌티노 공세</a>에서 반격을 하여 오스트리아군의 공세를 저지하는 데 성공했다. 그러나, <a href="/wiki/5%EC%9B%94_24%EC%9D%BC" title="5월 24일">5월 24일</a>에 벌어진 <a class="mw-redirect" href="/wiki/1%EC%B0%A8_%EC%9D%B4%EC%86%90%EC%B4%88_%EC%A0%84%ED%88%AC" title="1차 이손초 전투">1차 이손초 전투</a>가 홍수로 대패하고 <a href="/wiki/7%EC%9B%94_29%EC%9D%BC" title="7월 29일">7월 29일</a>에는 오스트리아에게 2~4 이손초 전투까지 대패하고 28만 명의 병사가 전사했다. <a href="/wiki/1916%EB%85%84" title="1916년">1916년</a> <a href="/wiki/3%EC%9B%94_11%EC%9D%BC" title="3월 11일">3월 11일</a>에 5~9차 이손초 전투가 오스트리아에게 다시 대패해 50만 명의 사상자를 냈다. <a href="/wiki/1916%EB%85%84" title="1916년">1916년</a> <a href="/wiki/8%EC%9B%94" title="8월">8월</a>에 <a class="new" href="/w/index.php?title=%EA%B3%A0%EB%A6%AC%EC%B9%98%EC%95%84_%EC%A0%84%ED%88%AC&amp;action=edit&amp;redlink=1" title="고리치아 전투 (없는 문서)">고리치아 전투</a>에서 오스트리아를 무찌르고 <a href="/wiki/%EA%B3%A0%EB%A6%AC%EC%B9%98%EC%95%84" title="고리치아">고리치아</a>를 점령해 다시 회복을 했으나, <a href="/wiki/1917%EB%85%84" title="1917년">1917년</a> <a href="/wiki/5%EC%9B%94_12%EC%9D%BC" title="5월 12일">5월 12일</a>에 10~12차 이손초 전투를 치르고 대패했다. 하지만 <a class="new" href="/w/index.php?title=%EB%B0%94%EC%97%94%EC%84%B8%EC%B0%A8_%EC%A0%84%ED%88%AC&amp;action=edit&amp;redlink=1" title="바엔세차 전투 (없는 문서)">바엔세차 전투</a>에서 승리해 약간의 부끄러움을 덜 수는 있었다.</p>
<p><a href="/wiki/1917%EB%85%84" title="1917년">1917년</a> <a href="/wiki/10%EC%9B%94_24%EC%9D%BC" title="10월 24일">10월 24일</a>에 오스트리아와 <a href="/wiki/%EB%8F%85%EC%9D%BC_%EC%A0%9C%EA%B5%AD" title="독일 제국">독일</a>과 <a href="/wiki/%EC%B9%B4%ED%8F%AC%EB%A0%88%ED%86%A0_%EC%A0%84%ED%88%AC" title="카포레토 전투">카포레토 전투</a>를 벌여 참패함으로써 본국으로 송환, <a href="/wiki/%EC%95%84%EB%A5%B4%EB%A7%8C%EB%8F%84_%EB%94%94%EC%95%84%EC%B8%A0" title="아르만도 디아츠">아르만도 디아츠</a>가 지휘권을 갖게 되었다. 종전 후에 <a class="mw-redirect" href="/wiki/%EC%9D%B4%ED%83%88%EB%A6%AC%EC%95%84_%EC%9C%A1%EA%B5%B0_%EC%9B%90%EC%88%98" title="이탈리아 육군 원수">육군 원수</a>가 되었다.</p>
