<p>《<b>루키즈</b>》(<span style="font-size: smaller;"><a href="/wiki/%EC%98%81%EC%96%B4" title="영어">영어</a>:</span> <span lang="en" xml:lang="en">ROOKIES</span>)는 <a class="new" href="/w/index.php?title=%EB%AA%A8%EB%A6%AC%ED%83%80_%EB%A7%88%EC%82%AC%EB%85%B8%EB%A6%AC&amp;action=edit&amp;redlink=1" title="모리타 마사노리 (없는 문서)">모리타 마사노리</a>에 의한 <a href="/wiki/%EC%9D%BC%EB%B3%B8" title="일본">일본</a>의 <a href="/wiki/%EC%95%BC%EA%B5%AC" title="야구">야구</a> <a href="/wiki/%EB%A7%8C%ED%99%94" title="만화">만화</a>이다. <a href="/wiki/1998%EB%85%84" title="1998년">1998년</a>부터 <a href="/wiki/2003%EB%85%84" title="2003년">2003년</a>까지 《<a href="/wiki/%EC%A3%BC%EA%B0%84_%EC%86%8C%EB%85%84_%EC%A0%90%ED%94%84" title="주간 소년 점프">주간 소년 점프</a>》(<a href="/wiki/%EC%8A%88%EC%97%90%EC%9D%B4%EC%83%A4" title="슈에이샤">슈에이샤</a>)에 연재되었다. <a href="/wiki/2008%EB%85%84" title="2008년">2008년</a>에 <a class="mw-redirect" href="/wiki/%EB%8F%84%EC%BF%84%EB%B0%A9%EC%86%A1" title="도쿄방송">TBS</a>에서 <a href="/wiki/%ED%85%94%EB%A0%88%EB%B9%84%EC%A0%84_%EB%93%9C%EB%9D%BC%EB%A7%88" title="텔레비전 드라마">텔레비전 드라마</a>화 되어 방송되었고, <a href="/wiki/2009%EB%85%84" title="2009년">2009년</a> <a href="/wiki/5%EC%9B%94_30%EC%9D%BC" title="5월 30일">5월 30일</a>에는 극장판이 개봉되었다.</p>

<h2><span class="mw-headline" id=".EC.A4.84.EA.B1.B0.EB.A6.AC">줄거리</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%A3%A8%ED%82%A4%EC%A6%88&amp;action=edit&amp;section=1" title="부분 편집: 줄거리">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>니코타마가와 학원 고등학교(통칭: 니코가쿠)에 발령받은 신인 교사 가와토 고이치. 그가 그곳에서 만난 것은 부원이 사고를 일으켜 활동 정지중인 야구부이다. 한 때는 <a class="mw-redirect" href="/wiki/%EC%9D%BC%EB%B3%B8%EC%9D%98_%EC%84%A0%EB%B0%9C_%EA%B3%A0%EB%93%B1%ED%95%99%EA%B5%90_%EC%95%BC%EA%B5%AC_%EB%8C%80%ED%9A%8C" title="일본의 선발 고등학교 야구 대회">봄의 고시엔</a> 출장까지 했던 전통있는 야구부였으나, 현재는 불량학생들의 아지트화되어 부원들은 자포자기 상태이다. 그러나 부원들의 마음 속에 남아있는 열정을 알아챈 가와토. 야구의 '야'자로 몰랐던 가와토는 스스로 야구부 고문이 되어, 야구부의 재건을 위해 움직인다. 이런 가와토의 태도에 마음을 움직인 불량학생들이, 꿈의 <a class="mw-disambig" href="/wiki/%EA%B3%A0%EC%8B%9C%EC%97%94" title="고시엔">고시엔</a>을 목표로 한 이야기이다.</p>

<p>《<b>루키즈</b>》(<span style="font-size: smaller;"><a href="/wiki/%EC%98%81%EC%96%B4" title="영어">영어</a>:</span> <span lang="en" xml:lang="en">ROOKIES</span>)는 <a class="new" href="/w/index.php?title=%EB%AA%A8%EB%A6%AC%ED%83%80_%EB%A7%88%EC%82%AC%EB%85%B8%EB%A6%AC&amp;action=edit&amp;redlink=1" title="모리타 마사노리 (없는 문서)">모리타 마사노리</a>의 원작 만화를 드라마화하여, <a href="/wiki/2008%EB%85%84" title="2008년">2008년</a> <a href="/wiki/4%EC%9B%94_19%EC%9D%BC" title="4월 19일">4월 19일</a>부터 <a href="/wiki/7%EC%9B%94_26%EC%9D%BC" title="7월 26일">7월 26일</a>까지 <a class="mw-redirect" href="/wiki/%EB%8F%84%EC%BF%84_%EB%B0%A9%EC%86%A1" title="도쿄 방송">TBS</a>에서 방송되었다. 주연은 <a href="/wiki/%EC%82%AC%ED%86%A0_%EB%A5%98%ED%83%80" title="사토 류타">사토 류타</a>. 보통 <a href="/wiki/%EC%9D%BC%EB%B3%B8" title="일본">일본</a>의 <a href="/wiki/%ED%85%94%EB%A0%88%EB%B9%84%EC%A0%84_%EB%93%9C%EB%9D%BC%EB%A7%88" title="텔레비전 드라마">텔레비전 드라마</a>는 3개월 동안 방송되지만, <a href="/wiki/%EB%B0%B0%EA%B5%AC" title="배구">배구</a> 경기의 중계 방송이 겹쳐, 지연되었고 보통 드라마보다 1개월 늦게 종영되었다. 내용은 원작의 설정과는 다른 점이 있다.</p>

<p>《<b>영화 "루키즈 -졸업-"</b>》(<span style="font-size: smaller;"><a href="/wiki/%EC%9D%BC%EB%B3%B8%EC%96%B4" title="일본어">일본어</a>:</span> <span lang="ja" xml:lang="ja">映画 "ROOKIES -卒業-"</span>)은 <a href="/wiki/2008%EB%85%84" title="2008년">2008년</a> 4월기 <a class="mw-redirect" href="/wiki/%EB%8F%84%EC%BF%84%EB%B0%A9%EC%86%A1" title="도쿄방송">TBS</a>에서 방송된 드라마 《루키즈》의 극장판 <a href="/wiki/%EC%98%81%ED%99%94" title="영화">영화</a>로, <a href="/wiki/%EC%9D%BC%EB%B3%B8" title="일본">일본</a>에서 <a href="/wiki/2009%EB%85%84" title="2009년">2009년</a> <a href="/wiki/5%EC%9B%94_30%EC%9D%BC" title="5월 30일">5월 30일</a>에 개봉되었다. 영화는 총 흥행수입 85.5억엔을 기록하며, <a href="/wiki/2009%EB%85%84" title="2009년">2009년</a> 개봉된 영화 흥행 1위를 차지했고, 역대 <a href="/wiki/%EC%9D%BC%EB%B3%B8" title="일본">일본</a> 방화 흥행 10위안에 들며 대히트를 하였다.</p>
