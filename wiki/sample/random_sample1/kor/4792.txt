<p><b>루트비히 3세</b>(Ludwig III, Louis III) 또는 <b>소 루트비히</b> (Ludwig the Younger <a href="/wiki/830%EB%85%84" title="830년">830년</a> - <a href="/wiki/882%EB%85%84" title="882년">882년</a> <a href="/wiki/1%EC%9B%94_20%EC%9D%BC" title="1월 20일">1월 20일</a>) 또는 <b>흘로도비쿠스 3세</b>(Hludowicus III, Hludovicus III) <a class="mw-redirect" href="/wiki/%ED%94%84%EB%9E%91%ED%81%AC%EC%9D%98_%EB%A3%A8%ED%8A%B8%EB%B9%84%ED%9E%88_2%EC%84%B8" title="프랑크의 루트비히 2세">독일왕 루트비히 2세</a>의 둘째 아들이며 동프랑크의 2대 국왕이다. <a href="/wiki/876%EB%85%84" title="876년">876년</a> 아버지의 죽음 이후 <a class="new" href="/w/index.php?title=%EC%9E%91%EC%84%BC%EC%9D%98_%EA%B5%B0%EC%A3%BC&amp;action=edit&amp;redlink=1" title="작센의 군주 (없는 문서)">작센의 군주</a>였고 <a class="mw-redirect" href="/wiki/%ED%8A%80%EB%A7%81%EA%B2%90" title="튀링겐">튀링겐</a>의 군주, <a class="new" href="/w/index.php?title=%EB%8F%99%ED%94%84%EB%9E%91%ED%81%AC%EC%9D%98_%EA%B5%AD%EC%99%95&amp;action=edit&amp;redlink=1" title="동프랑크의 국왕 (없는 문서)">동프랑크의 국왕</a> 칭호를 물려받았으며, <a href="/wiki/880%EB%85%84" title="880년">880년</a>부터는 형 <a class="new" href="/w/index.php?title=%EC%B9%B4%EB%A5%BC%EB%A7%8C_3%EC%84%B8&amp;action=edit&amp;redlink=1" title="카를만 3세 (없는 문서)">카를만 3세</a>로부터 <a class="mw-redirect" href="/wiki/%EB%B0%94%EC%9D%B4%EC%97%90%EB%A5%B8" title="바이에른">바이에른</a>을 넘겨받아 <a href="/wiki/%EB%B0%94%EC%9D%B4%EC%97%90%EB%A5%B8_%EC%99%95%EA%B5%AD" title="바이에른 왕국">바이에른 왕</a>이었으므로 <a href="/wiki/%EB%8F%85%EC%9D%BC%EC%9D%98_%EA%B5%B0%EC%A3%BC" title="독일의 군주">독일 왕</a>으로 본다.</p>
<p><a href="/wiki/%EC%B9%B4%EB%A5%BC%EB%A7%8C" title="카를만">카를만</a>의 사망 전 880년 <a href="/wiki/2%EC%9B%94" title="2월">2월</a> <a class="mw-redirect" href="/wiki/%EC%84%9C%ED%94%84%EB%9E%91%ED%81%AC" title="서프랑크">서프랑크</a>의 <a href="/wiki/%EB%A3%A8%EC%9D%B4_3%EC%84%B8" title="루이 3세">루이 3세</a>, <a href="/wiki/%EC%83%A4%EB%A5%BC%EB%A1%9C%EB%A7%9D" title="샤를로망">샤를로망</a> 형제와 <a class="new" href="/w/index.php?title=%EB%A6%AC%EB%B2%A0%EB%AA%BD&amp;action=edit&amp;redlink=1" title="리베몽 (없는 문서)">리베몽</a>에서 조약을 체결하여 <a href="/wiki/%EB%A1%9C%EB%A0%8C" title="로렌">로트링겐(로렌)</a>을 차지하였다. 형의 죽음 이후 <a class="mw-redirect" href="/wiki/%EB%B0%94%EC%9D%B4%EC%97%90%EB%A5%B8" title="바이에른">바이에른</a> 작센 공작 <a class="new" href="/w/index.php?title=%EB%A6%AC%EC%9A%B0%EB%8F%8C%ED%94%84_(%EC%9E%91%EC%84%BC)&amp;action=edit&amp;redlink=1" title="리우돌프 (작센) (없는 문서)">루돌프(하인리히 1세의 조부)</a>의 딸 리우트가르트와 결혼하여 아들 루트비히와 베른하르트를 얻었으나 모두 어려서 사망하였다. 재위기간 내내 <a class="new" href="/w/index.php?title=%EB%B0%94%EC%9D%B4%ED%82%B9_%EC%A1%B1&amp;action=edit&amp;redlink=1" title="바이킹 족 (없는 문서)">바이킹 족</a>의 침략에 시달림당하였다. 그의 재직 중 <a class="mw-redirect" href="/wiki/%EC%84%9C%ED%94%84%EB%9E%91%ED%81%AC" title="서프랑크">서프랑크</a>의 <a class="mw-disambig" href="/wiki/%EC%B9%B4%EB%A5%BC_2%EC%84%B8" title="카를 2세">카를 2세</a>는 그의 영지를 탐냈고, <a class="new" href="/w/index.php?title=%EC%95%88%EB%8D%94%EB%82%98%ED%9D%90_%EC%A0%84%ED%88%AC&amp;action=edit&amp;redlink=1" title="안더나흐 전투 (없는 문서)">안더나흐 전투</a>에서 자신의 병사들을 유령으로 위장시켜 <a class="mw-disambig" href="/wiki/%EC%B9%B4%EB%A5%BC_2%EC%84%B8" title="카를 2세">카를 2세</a>의 군대를 퇴각하게 했다. 한편 그는 카를 사후, <a class="mw-redirect" href="/wiki/%EC%84%9C%ED%94%84%EB%9E%91%ED%81%AC" title="서프랑크">서프랑크</a> 측에 <a class="mw-redirect" href="/wiki/%EB%A1%9C%ED%8A%B8%EB%A7%81%EA%B2%90" title="로트링겐">로트링겐</a>의 영토를 계속 요구하였고, <a href="/wiki/880%EB%85%84" title="880년">880년</a>의 <a href="/wiki/%EB%A6%AC%EB%B2%A0%EB%AA%BD_%EC%A1%B0%EC%95%BD" title="리베몽 조약">리베몽 조약</a>을 통해 <a class="mw-redirect" href="/wiki/%EB%A1%9C%ED%8A%B8%EB%A7%81%EA%B2%90" title="로트링겐">로트링겐</a>의 서부 영토를 동프랑크로 병합하여 로트링겐 전체를 차지하였다. 아들들은 모두 일찍 죽어 그는 후계자를 남기지 못해 결국 그가 죽은 이후 그의 영토인 <a href="/wiki/%EB%8F%99%ED%94%84%EB%9E%91%ED%81%AC_%EC%99%95%EA%B5%AD" title="동프랑크 왕국">동프랑크 왕국</a>은 동생 <a href="/wiki/%EC%B9%B4%EB%A5%BC_3%EC%84%B8" title="카를 3세">비만왕 카를</a>에게 계승되었다.</p>

<h2><span class="mw-headline" id=".EC.83.9D.EC.95.A0">생애</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%A3%A8%ED%8A%B8%EB%B9%84%ED%9E%88_3%EC%84%B8_(%EB%8F%99%ED%94%84%EB%9E%91%ED%81%AC)&amp;action=edit&amp;section=1" title="부분 편집: 생애">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<h3><span class="mw-headline" id=".EC.99.95.EC.9E.90_.EC.8B.9C.EC.A0.88">왕자 시절</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%A3%A8%ED%8A%B8%EB%B9%84%ED%9E%88_3%EC%84%B8_(%EB%8F%99%ED%94%84%EB%9E%91%ED%81%AC)&amp;action=edit&amp;section=2" title="부분 편집: 왕자 시절">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>루트비히 3세는 <a href="/wiki/830%EB%85%84" title="830년">830년</a> <a class="mw-redirect" href="/wiki/%ED%94%84%EB%9E%91%ED%81%AC%EC%9D%98_%EB%A3%A8%ED%8A%B8%EB%B9%84%ED%9E%88_2%EC%84%B8" title="프랑크의 루트비히 2세">루트비히 독일왕</a>과 <a class="mw-redirect" href="/wiki/%EB%B0%94%EC%9D%B4%EC%97%90%EB%A5%B8" title="바이에른">바이에른</a>과 <a class="new" href="/w/index.php?title=%EC%95%84%EB%A5%B4%ED%86%A8%ED%94%84&amp;action=edit&amp;redlink=1" title="아르톨프 (없는 문서)">아르톨프</a>의 <a href="/wiki/%EA%B3%B5%EC%9E%91_(%EC%9E%91%EC%9C%84)" title="공작 (작위)">공작</a> <a class="new" href="/w/index.php?title=%EB%B2%A8%ED%94%84_1%EC%84%B8&amp;action=edit&amp;redlink=1" title="벨프 1세 (없는 문서)">벨프 1세</a>의 딸 <a href="/wiki/%EB%B0%94%EC%9D%B4%EC%97%90%EB%A5%B8%EC%9D%98_%EC%97%A0%EB%AF%80" title="바이에른의 엠므">바이에른의 엠므</a>의 아들로 태어났다. 형은 <a class="mw-redirect" href="/wiki/%EC%B9%B4%EB%A5%BC%EB%A1%9C%EB%A7%8C_3%EC%84%B8" title="카를로만 3세">카를로만 3세</a>였고 동생은 <a href="/wiki/%EC%B9%B4%EB%A5%BC_3%EC%84%B8" title="카를 3세">뚱보왕 카를 3세</a>였다. <a class="mw-disambig" href="/wiki/%EB%B0%94%EC%9D%B4%EC%97%90%EB%A5%B8%EC%9D%98_%EC%9C%A0%EB%94%94%ED%8A%B8" title="바이에른의 유디트">바이에른의 유디트</a>는 그의 이모이자 계조모이기도 했다.<sup class="reference" id="cite_ref-1"><a href="#cite_note-1">[1]</a></sup></p>
<p>루트비히는 북방 슬라브계 민족인 오보트리스 족의 침략으로 <a href="/wiki/858%EB%85%84" title="858년">858년</a>과 <a href="/wiki/862%EB%85%84" title="862년">862년</a> <a class="mw-redirect" href="/wiki/%ED%94%84%EB%9E%91%ED%81%AC%EC%9D%98_%EB%A3%A8%ED%8A%B8%EB%B9%84%ED%9E%88_2%EC%84%B8" title="프랑크의 루트비히 2세">독일인 루트비히</a>가 오보트리스(Abodrites)를 정벌할 때, 군사 훈련을 받고 동부 오보트리스에 대한 군사 작전에 투입되었습니다. <a href="/wiki/854%EB%85%84" title="854년">854년</a> <a href="/wiki/%EC%95%84%ED%82%A4%ED%85%90" title="아키텐">아키텐</a>을 두고 <a class="mw-redirect" href="/wiki/%EB%8C%80%EB%A8%B8%EB%A6%AC%EC%99%95_%EC%B9%B4%EB%A5%BC" title="대머리왕 카를">대머리왕 카를</a>과 <a class="mw-redirect" href="/wiki/%EC%95%84%ED%82%A4%ED%85%90%EC%9D%98_%ED%94%BC%ED%95%80_2%EC%84%B8" title="아키텐의 피핀 2세">피핀</a> 간에 전쟁이 벌어지자 피핀을 지지하던 <a href="/wiki/%EC%95%84%ED%82%A4%ED%85%90" title="아키텐">아키텐</a> 귀족들의 요청으로 개입하게 된다. 군사를 이끌고 <a href="/wiki/%EC%95%84%ED%82%A4%ED%85%90" title="아키텐">아키텐</a>에 들어가 <a class="mw-redirect" href="/wiki/%EC%95%84%ED%82%A4%ED%85%90%EC%9D%98_%ED%94%BC%ED%95%80_2%EC%84%B8" title="아키텐의 피핀 2세">피핀</a>을 지원하였다.</p>
<p><a class="mw-redirect" href="/wiki/%EB%8C%80%EB%A8%B8%EB%A6%AC%EC%99%95_%EC%B9%B4%EB%A5%BC" title="대머리왕 카를">대머리왕 카를</a>을 공격하는 한편 <a class="new" href="/w/index.php?title=%EB%B0%94%EC%9D%B4%ED%82%B9_%EC%A1%B1&amp;action=edit&amp;redlink=1" title="바이킹 족 (없는 문서)">바이킹 족</a>이 정착한 <a class="mw-redirect" href="/wiki/%EC%84%9C%ED%94%84%EB%9E%91%ED%81%AC" title="서프랑크">서프랑크</a>의 <a class="new" href="/w/index.php?title=%EB%A6%AC%EB%AA%A8_%EC%A3%BC&amp;action=edit&amp;redlink=1" title="리모 주 (없는 문서)">리모 주</a>를 공격하고 <a class="new" href="/w/index.php?title=%EB%A3%A8%EC%95%84%EB%A5%B4_%EB%B0%B8%EB%A6%AC&amp;action=edit&amp;redlink=1" title="루아르 밸리 (없는 문서)">루아르 밸리</a>(Loire Valley)를 파괴한 뒤 <a href="/wiki/%ED%91%B8%EC%95%84%ED%8B%B0%EC%97%90" title="푸아티에">푸아티에</a>, <a href="/wiki/%EC%95%99%EA%B5%B4%EB%A0%98" title="앙굴렘">앙굴렘</a>, 페리제(Perigueux_), <a class="new" href="/w/index.php?title=%EB%A6%AC%EB%AA%A8_%EC%A3%BC&amp;action=edit&amp;redlink=1" title="리모 주 (없는 문서)">리모 주</a>, <a class="mw-redirect" href="/wiki/%ED%81%B4%EB%A0%88%EB%A5%B4%EB%AA%BD" title="클레르몽">클레르몽</a>과 <a class="new" href="/w/index.php?title=%EB%B6%80%EB%A5%B4%EC%A5%AC%EA%B0%80&amp;action=edit&amp;redlink=1" title="부르쥬가 (없는 문서)">부르쥬가</a> 등을 점령하였다. 이어 수도원에 투옥된 <a class="mw-redirect" href="/wiki/%EC%95%84%ED%82%A4%ED%85%90%EC%9D%98_%ED%94%BC%ED%95%80_2%EC%84%B8" title="아키텐의 피핀 2세">피핀</a>을 풀어주었다. 그는 사촌인 마인츠(Mainz)의 대주교 <a href="/wiki/%EC%83%A4%EB%A5%BC_(%EC%95%84%ED%82%A4%ED%85%90%EC%9D%98_%EC%99%95%EC%9E%90)" title="샤를 (아키텐의 왕자)">샤를</a>에게 왕관을 전달한 뒤 군사들을 이끌고 다시 <a class="mw-redirect" href="/wiki/%EB%8F%99%ED%94%84%EB%9E%91%ED%81%AC" title="동프랑크">동프랑크</a>로 퇴각, <a class="mw-redirect" href="/wiki/%EB%B0%94%EC%9D%B4%EC%97%90%EB%A5%B8" title="바이에른">바이에른</a>으로 되돌아왔다. <a href="/wiki/853%EB%85%84" title="853년">853년</a> <a class="mw-redirect" href="/wiki/%EC%84%9C%ED%94%84%EB%9E%91%ED%81%AC" title="서프랑크">서프랑크</a>의 <a class="mw-redirect" href="/wiki/%EB%8C%80%EB%A8%B8%EB%A6%AC%EC%99%95_%EC%B9%B4%EB%A5%BC" title="대머리왕 카를">대머리왕 카를</a>은 청년 루트비히를 회유할 생각으로 그를 <a href="/wiki/%ED%94%BC%ED%95%80_2%EC%84%B8_(%EC%95%84%ED%82%A4%ED%85%90)" title="피핀 2세 (아키텐)">아키텐의 피핀 2세</a>를 대신할 <a href="/wiki/%EC%95%84%ED%82%A4%ED%85%90" title="아키텐">아키텐의 군주</a>로 지명하였다. <a class="mw-redirect" href="/wiki/%EC%84%9C%ED%94%84%EB%9E%91%ED%81%AC" title="서프랑크">서프랑크</a>의 대머리 <a class="mw-disambig" href="/wiki/%EC%B9%B4%EB%A5%BC_2%EC%84%B8" title="카를 2세">카를 2세</a>는 청년 루트비히 3세의 아버지인 독일인 루트비히 2세에게 사자를 보내, <a href="/wiki/%EC%95%84%ED%82%A4%ED%85%90" title="아키텐">아키텐</a>의 왕관을 주겠다고 제의하였다. 그러나 <a href="/wiki/%EC%95%84%ED%82%A4%ED%85%90" title="아키텐">아키텐</a>의 귀족들은 청년 루트비히를 환영하지 않았다.</p>
<p>청년 루트비히는 삼촌 <a class="mw-redirect" href="/wiki/%EB%8C%80%EB%A8%B8%EB%A6%AC_%EC%B9%B4%EB%A5%BC" title="대머리 카를">대머리 카를</a>의 제의를 거절하고 오히려 삼촌 <a class="mw-redirect" href="/wiki/%EB%8C%80%EB%A8%B8%EB%A6%AC_%EC%B9%B4%EB%A5%BC" title="대머리 카를">대머리 카를</a>에 의해 수감된 <a href="/wiki/%ED%94%BC%ED%95%80_2%EC%84%B8_(%EC%95%84%ED%82%A4%ED%85%90)" title="피핀 2세 (아키텐)">아키텐의 피핀 2세</a>와 <a href="/wiki/%EC%83%A4%EB%A5%BC_(%EC%95%84%ED%82%A4%ED%85%90%EC%9D%98_%EC%99%95%EC%9E%90)" title="샤를 (아키텐의 왕자)">샤를</a> 형제를 탈출시켰다. <a href="/wiki/%ED%94%BC%ED%95%80_2%EC%84%B8_(%EC%95%84%ED%82%A4%ED%85%90)" title="피핀 2세 (아키텐)">아키텐의 피핀 2세</a>와 <a href="/wiki/%EC%83%A4%EB%A5%BC_(%EC%95%84%ED%82%A4%ED%85%90%EC%9D%98_%EC%99%95%EC%9E%90)" title="샤를 (아키텐의 왕자)">샤를</a>의 탈옥을 확정지은 후, 청년 루트비히는 <a class="new" href="/w/index.php?title=%EB%A6%AC%EB%AA%A8_%EC%A3%BC&amp;action=edit&amp;redlink=1" title="리모 주 (없는 문서)">리모 주</a>, <a class="mw-redirect" href="/wiki/%EB%85%B8%EB%A5%B4%EB%A7%8C_%EC%A1%B1" title="노르만 족">노르만 족</a>이 정착한 <a href="/wiki/%EB%A3%A8%EC%95%84%EB%A5%B4_%EA%B3%84%EA%B3%A1" title="루아르 계곡">루아르 계곡</a> 일대, 그밖에 <a href="/wiki/%ED%91%B8%EC%95%84%ED%8B%B0%EC%97%90" title="푸아티에">푸아티에</a>, <a href="/wiki/%EC%95%99%EA%B5%B4%EB%A0%98" title="앙굴렘">앙굴렘</a>, <a class="mw-disambig" href="/wiki/%ED%8E%98%EB%A6%AC" title="페리">페리</a>, <a href="/wiki/%EB%A6%AC%EB%AA%A8" title="리모">리모</a>, <a class="mw-redirect" href="/wiki/%ED%81%B4%EB%A0%88%EB%A5%B4%EB%AA%BD" title="클레르몽">클레르몽</a>과 <a class="new" href="/w/index.php?title=%EB%B6%80%EB%A5%B4%EC%A0%9C&amp;action=edit&amp;redlink=1" title="부르제 (없는 문서)">부르제</a>를 공격하고 돌아갔다.</p>

<p>루트비히는 은근히 <a class="mw-redirect" href="/wiki/%EB%8F%99%ED%94%84%EB%9E%91%ED%81%AC" title="동프랑크">동프랑크</a>의 귀족들과 긴밀한 유대 관계를 형성하는 한편 그의 아버지의 통제를 거부하고 점점 더 독립적으로 행동하였다. 그는 집안의 허락 없이 단독으로 <a href="/wiki/%EB%84%A4%EC%9A%B0%EC%8A%A4%ED%8A%B8%EB%A6%AC%EC%95%84" title="네우스트리아">네우스트리아</a>(Neustria) <a href="/wiki/%EB%B0%B1%EC%9E%91" title="백작">백작</a> 아달하르트(Adalhard)의 딸과 약혼하였으나 아버지 <a class="mw-redirect" href="/wiki/%ED%94%84%EB%9E%91%ED%81%AC%EC%9D%98_%EB%A3%A8%ED%8A%B8%EB%B9%84%ED%9E%88_2%EC%84%B8" title="프랑크의 루트비히 2세">독일인 루트비히</a>의 반대로 결혼은 파탄났고, 약혼녀는 일찍 사망하였다. <a href="/wiki/864%EB%85%84" title="864년">864년</a> <a class="mw-redirect" href="/wiki/%EC%9E%91%EC%84%BC" title="작센">작센</a> <a class="mw-redirect" href="/wiki/%EA%B5%AD%EC%99%95" title="국왕">분국왕</a>과 <a class="mw-redirect" href="/wiki/%ED%8A%80%EB%A7%81%EA%B2%90" title="튀링겐">튀링겐</a> <a href="/wiki/%EA%B3%B5%EC%9E%91_(%EC%9E%91%EC%9C%84)" title="공작 (작위)">공작</a>에 임명되어 아버지에게 <a class="mw-redirect" href="/wiki/%EC%9E%91%EC%84%BC" title="작센">작센</a>과 <a class="mw-redirect" href="/wiki/%ED%8A%80%EB%A7%81%EA%B2%90" title="튀링겐">튀링겐</a>의 상속권을 약속받았다. 그러나 <a href="/wiki/865%EB%85%84" title="865년">865년</a> 루트비히는 동생 <a href="/wiki/%EC%B9%B4%EB%A5%BC_3%EC%84%B8" title="카를 3세">카를 3세</a>와 함께 아버지 <a class="mw-redirect" href="/wiki/%ED%94%84%EB%9E%91%ED%81%AC%EC%9D%98_%EB%A3%A8%ED%8A%B8%EB%B9%84%ED%9E%88_2%EC%84%B8" title="프랑크의 루트비히 2세">독일인 루트비히</a>에 대한 반란에 합류하였다.</p>
<p><a href="/wiki/865%EB%85%84" title="865년">865년</a> <a class="mw-redirect" href="/wiki/%ED%94%84%EB%9E%91%ED%81%AC%ED%91%B8%EB%A5%B4%ED%8A%B8" title="프랑크푸르트">프랑크푸르트</a>와 <a class="mw-redirect" href="/wiki/%ED%94%84%EB%9E%91%EC%BD%94%EB%8B%88%EC%95%84" title="프랑코니아">프랑코니아</a>를 차지했고 <a class="mw-redirect" href="/wiki/%EC%9E%91%EC%84%BC" title="작센">작센</a>과 <a class="mw-redirect" href="/wiki/%ED%8A%80%EB%A7%81%EA%B2%90" title="튀링겐">튀링겐</a>에서 자신의 영향력을 증가시켰다. 865년 초 청년 루트비히는 <a href="/wiki/%EB%84%A4%EC%9A%B0%EC%8A%A4%ED%8A%B8%EB%A6%AC%EC%95%84" title="네우스트리아">네우스트리아</a> <a href="/wiki/%EB%B0%B1%EC%9E%91" title="백작">백작</a> 아달하르트의 딸과 약혼하였으나, 아버지 독일인 루트비히의 반대로 파혼하였다. 청년 루트비히는 몇 명의 첩에게서 아들들을 얻기도 했다. 뒤에 869년 <a href="/wiki/11%EC%9B%94_24%EC%9D%BC" title="11월 24일">11월 24일</a> 청년 루트비히는 <a class="mw-redirect" href="/wiki/%EC%9E%91%EC%84%BC" title="작센">작센</a>의 실력자인 <a class="new" href="/w/index.php?title=%EB%A6%AC%EC%9A%B0%EB%8F%8C%ED%95%91_%EA%B0%80%EB%AC%B8&amp;action=edit&amp;redlink=1" title="리우돌핑 가문 (없는 문서)">리우돌핑 가문</a> 출신 <a class="mw-redirect" href="/wiki/%EC%9E%91%EC%84%BC" title="작센">작센</a>과 이스트팔리아(Eastphalia)의 공작 <a class="mw-redirect" href="/wiki/%EB%A6%AC%EC%9A%B0%EB%8F%8C%ED%94%84" title="리우돌프">리우돌프</a>(Liudolf)의 딸 <a class="mw-disambig" href="/wiki/%EB%A6%AC%EC%9A%B0%ED%8A%B8%EA%B0%80%EB%A5%B4%ED%8A%B8" title="리우트가르트">리우트가르트</a>(Liutgard)와 <a href="/wiki/%EC%95%84%EC%83%A4%ED%8E%9C%EB%B6%80%EB%A5%B4%ED%81%AC" title="아샤펜부르크">아샤펜부르크</a>(Aschaffenburg)에서 결혼하였다. <a class="mw-redirect" href="/wiki/%EB%A6%AC%EC%9A%B0%EB%8F%8C%ED%94%84" title="리우돌프">리우돌프</a>는 후일 <a href="/wiki/918%EB%85%84" title="918년">918년</a> <a href="/wiki/%EB%8F%85%EC%9D%BC%EC%9D%98_%EA%B5%B0%EC%A3%BC" title="독일의 군주">독일의 군주</a>로 즉위하는 <a href="/wiki/%ED%95%98%EC%9D%B8%EB%A6%AC%ED%9E%88_1%EC%84%B8" title="하인리히 1세">하인리히 1세</a>의 할아버지이자 <a class="mw-redirect" href="/wiki/%EC%98%A4%ED%86%A0_%EB%8C%80%EC%A0%9C" title="오토 대제">오토 대제</a>의 증조부가 된다. <a class="mw-redirect" href="/wiki/%EC%9E%91%EC%84%BC" title="작센">작센</a>의 유력 귀족가문의 딸이기도 한 그의 왕비 리우트가르트는 강한 의지와 정치적 야망을 가진 여성으로 그녀의 남편의 정치적 야심을 부추겼으며, 자신의 야심을 달성하기 위해 노력했다.</p>
<p>865년 아버지 <a class="mw-redirect" href="/wiki/%ED%94%84%EB%9E%91%ED%81%AC%EC%9D%98_%EB%A3%A8%ED%8A%B8%EB%B9%84%ED%9E%88_2%EC%84%B8" title="프랑크의 루트비히 2세">독일인 루트비히</a>는 세 아들에게 각각 자신이 차지할 땅을 지목하도록 강요했는데, <a href="/wiki/%EC%B9%B4%EB%A5%BC%EB%A7%8C" title="카를만">카를만</a>은 <a class="mw-redirect" href="/wiki/%EB%B0%94%EC%9D%B4%EC%97%90%EB%A5%B8" title="바이에른">바이에른</a>을, 청년 루트비히는 <a class="mw-redirect" href="/wiki/%EC%9E%91%EC%84%BC" title="작센">작센</a>과 <a href="/wiki/%ED%94%84%EB%9E%91%EC%BC%84" title="프랑켄">프랑켄</a>, <a class="mw-redirect" href="/wiki/%ED%8A%80%EB%A7%81%EA%B2%90" title="튀링겐">튀링겐</a>을, 뚱보 카를은 <a class="new" href="/w/index.php?title=%EC%95%8C%EB%A0%88%EB%A7%8C%EB%8B%88%EC%95%84&amp;action=edit&amp;redlink=1" title="알레만니아 (없는 문서)">알레만니아</a>와 <a href="/wiki/%EC%8A%88%EB%B0%94%EB%B2%A4" title="슈바벤">슈바벤</a>, <a href="/wiki/%EB%9D%BC%EC%9D%B4%ED%8B%B0%EC%95%84" title="라이티아">라이티아</a>(Raetia, <a href="/wiki/%EB%8F%85%EC%9D%BC" title="독일">독일</a> <a href="/wiki/%EB%B0%94%EC%9D%B4%EC%97%90%EB%A5%B8_%EC%A3%BC" title="바이에른 주">바이에른 주</a>의 남부와 <a href="/wiki/%EC%98%A4%EC%8A%A4%ED%8A%B8%EB%A6%AC%EC%95%84" title="오스트리아">오스트리아</a> <a href="/wiki/%ED%8F%AC%EC%96%B4%EC%95%84%EB%A5%BC%EB%B2%A0%EB%A5%B4%ED%81%AC_%EC%A3%BC" title="포어아를베르크 주">포어아를베르크 주</a> 전역, 오스트리아와 이탈리아에 거쳐 있는 <a href="/wiki/%ED%8B%B0%EB%A1%A4" title="티롤">티롤</a> 지방의 전역 및 <a href="/wiki/%EC%9D%B4%ED%83%88%EB%A6%AC%EC%95%84" title="이탈리아">이탈리아</a> <a href="/wiki/%EB%A1%AC%EB%B0%94%EB%A5%B4%EB%94%94%EC%95%84_%EC%A3%BC" title="롬바르디아 주">롬바르디아 주</a>의 북동부)를 지목했다. 그리고 <a class="mw-redirect" href="/wiki/%ED%94%84%EB%9E%91%ED%81%AC%ED%91%B8%EB%A5%B4%ED%8A%B8" title="프랑크푸르트">프랑크푸르트</a> 지역은 삼형제가 공유하기로 했다. 몇달 후 청년 루트비히는 아버지 독일인 루트비히에 대항해 반란을 일으켰으나 진압되었다. <a href="/wiki/865%EB%85%84" title="865년">865년</a> <a href="/wiki/10%EC%9B%94" title="10월">10월</a> <a href="/wiki/%EC%84%9C%ED%94%84%EB%9E%91%ED%81%AC_%EC%99%95%EA%B5%AD" title="서프랑크 왕국">서프랑크 왕국</a>의 <a class="mw-disambig" href="/wiki/%EC%B9%B4%EB%A5%BC_2%EC%84%B8" title="카를 2세">카를 2세</a>는 <a href="/wiki/%EC%BE%B0%EB%A5%B8" title="쾰른">쾰른</a>에서 소 루트비히와 독일인 루트비히 부자의 화해를 주선하였다. <a href="/wiki/870%EB%85%84" title="870년">870년</a> <a href="/wiki/%EB%A1%9C%ED%83%80%EB%A5%B4_2%EC%84%B8" title="로타르 2세">로타르 2세</a>의 영지의 일부로 <a href="/wiki/%EC%84%9C%ED%94%84%EB%9E%91%ED%81%AC_%EC%99%95%EA%B5%AD" title="서프랑크 왕국">서프랑크 왕국</a>에 넘어갔던 <a href="/wiki/%EB%A7%88%EC%8A%A4%ED%8A%B8%EB%A6%AC%ED%9D%90%ED%8A%B8" title="마스트리흐트">마스트리흐트</a> 일대를 빼앗아 점령하였다. <a href="/wiki/871%EB%85%84" title="871년">871년</a>과 <a href="/wiki/873%EB%85%84" title="873년">873년</a> 청년 루트비히는 반란을 일으켜 부왕과의 사이에 불화가 증가하지만, 그 때마다 그는 기적적으로 부왕과 화해하였다. <a href="/wiki/872%EB%85%84" title="872년">872년</a> 루트비히는 자신의 다른 두 형제와 함께 유산으로 정해진 지역의 상속을 재확인받았다.</p>

<p><a href="/wiki/876%EB%85%84" title="876년">876년</a> <a href="/wiki/8%EC%9B%94_28%EC%9D%BC" title="8월 28일">8월 28일</a> 독일인 루트비히의 사후, 청년 루트비히는 그해 <a href="/wiki/11%EC%9B%94" title="11월">11월</a>의 영토 분할에서 그는 작센과 튀링겐, 동 로트링겐을 상속받았고, <a href="/wiki/%EB%A9%94%EB%A5%B4%EC%84%BC_%EC%A1%B0%EC%95%BD" title="메르센 조약">메르센 조약</a>으로 추가된 영토도 추가적으로 상속받았으며, 동프랑크의 전체의 국왕(Rex Francorum) 칭호도 물려받았다. 그밖에 <a href="/wiki/%EB%A3%A8%ED%8A%B8%EB%B9%84%ED%9E%88_2%EC%84%B8_(%EB%8F%99%ED%94%84%EB%9E%91%ED%81%AC)" title="루트비히 2세 (동프랑크)">독일인 루트비히</a>가 <a href="/wiki/870%EB%85%84" title="870년">870년</a> <a href="/wiki/%EB%A9%94%EB%A5%B4%EC%84%BC_%EC%A1%B0%EC%95%BD" title="메르센 조약">메르센 조약</a>으로 획득한 동부 <a class="mw-redirect" href="/wiki/%EB%A1%9C%ED%8A%B8%EB%A7%81%EA%B2%90" title="로트링겐">로트링겐</a>은 이때 처음으로 넘겨받았다. 또한 청년 루트비히는 자신의 아버지에게 유능한 참모인 마인츠 대주교 리우트베르트(889년 사망)를 물려받았다. 즉위 초반부터 루트비히는 바이킹의 잦은 침입에 적극적으로 맞섰으며 자국 내 귀족들과 좋은 관계를 유지했다. 비록 그가 형식적으로 <a class="mw-redirect" href="/wiki/%EB%8F%99%ED%94%84%EB%9E%91%ED%81%AC" title="동프랑크">동프랑크</a> 전체의 국왕 칭호를 물려받았지만 형제들은 각자 <a href="/wiki/%EB%8F%99%ED%94%84%EB%9E%91%ED%81%AC_%EC%99%95%EA%B5%AD" title="동프랑크 왕국">동프랑크 왕국</a> 내 개인 영지를 독립적으로 통치하였고, 전쟁에서도 서로 협력하지는 않았다. 그는 주로 <a href="/wiki/%EB%9D%BC%EC%9D%B8%EB%9E%80%ED%8A%B8" title="라인란트">라인란트</a>에 머물렀으며 자신의 영지 <a class="mw-redirect" href="/wiki/%EC%9E%91%EC%84%BC" title="작센">작센</a>이나 동부 변경으로 가는 것은 가급적이면 피했다.</p>
<p>청년 루트비히는 왕이 되자마자 곧 삼촌 <a class="mw-disambig" href="/wiki/%EC%B9%B4%EB%A5%BC_2%EC%84%B8" title="카를 2세">카를 2세</a>의 위협을 받았다. 대머리 카를은 동부 로타링기아를 합병하려 했고, <a href="/wiki/%EB%A3%A8%ED%8A%B8%EB%B9%84%ED%9E%88_2%EC%84%B8_(%EB%8F%99%ED%94%84%EB%9E%91%ED%81%AC)" title="루트비히 2세 (동프랑크)">독일인 루트비히 2세</a>의 사후 <a href="/wiki/876%EB%85%84" title="876년">876년</a> 가을, <a class="mw-redirect" href="/wiki/%EC%84%9C%ED%94%84%EB%9E%91%ED%81%AC" title="서프랑크">서프랑크</a>의 대머리 <a class="mw-disambig" href="/wiki/%EC%B9%B4%EB%A5%BC_2%EC%84%B8" title="카를 2세">카를 2세</a>가 동프랑크로 쳐들어왔다. 동 로트링겐에서 <a class="mw-disambig" href="/wiki/%EC%B9%B4%EB%A5%BC_2%EC%84%B8" title="카를 2세">대머리 카를</a>의 군대와 교전하여 물리친 뒤 청년 루트비히와 <a class="mw-disambig" href="/wiki/%EC%B9%B4%EB%A5%BC_2%EC%84%B8" title="카를 2세">대머리 카를</a>은 876년 <a href="/wiki/10%EC%9B%94" title="10월">10월</a> <a class="new" href="/w/index.php?title=%EC%95%88%EB%8D%94%EB%82%98%ED%9D%90&amp;action=edit&amp;redlink=1" title="안더나흐 (없는 문서)">안더나흐</a>에서 맞붙었다. 877년 <a href="/wiki/10%EC%9B%94_8%EC%9D%BC" title="10월 8일">10월 8일</a> 안더나흐에서 서프랑크 군을 최종적으로 격퇴하였다. 전략과 수적으로 모두 우월했던 동프랑크의 군이었으나 루트비히는 군사들에게 모두 흰 옷을 입혀 망령 군대처럼 보이게 했고, 밤에 이들을 본 서프랑크의 군사들은 귀신으로 보고 놀라서 달아났다. 병사들은 우왕좌왕 도주하거나 실신해서 쓰러졌고 <a class="mw-disambig" href="/wiki/%EC%B9%B4%EB%A5%BC_2%EC%84%B8" title="카를 2세">카를 2세</a>는 당황해하며 되돌아갔다. 루트비히의 기지로 훨씬 많은 수의 서프랑크군을 작은 부대 몇 개만 움직여서 물리칠 수 있었다.</p>

<p>대머리 카를은 자신의 형 루트비히가 살아 있을 때에도 황제 직위를 욕심내어 세력을 이탈리아까지 확대했고, 형 독일인 루트비히가 <a class="mw-redirect" href="/wiki/%ED%94%84%EB%9E%91%ED%81%AC%ED%91%B8%EB%A5%B4%ED%8A%B8" title="프랑크푸르트">프랑크푸르트</a>에서 죽자 <a href="/wiki/%EB%9D%BC%EC%9D%B8_%EA%B0%95" title="라인 강">라인 강</a>까지 진출하려 했다.<sup class="reference" id="cite_ref-ppkk_2-0"><a href="#cite_note-ppkk-2">[2]</a></sup> 독일인 루트비히의 둘째 아들인 청년 루트비히가 작은 아버지인 카를과 담판하여 <a href="/wiki/%ED%94%84%EB%9E%91%EC%BC%84" title="프랑켄">프랑켄</a>과 <a class="mw-redirect" href="/wiki/%EC%9E%91%EC%84%BC" title="작센">작센</a>에서 군대 소집권을 주기로 하고 전쟁을 종결했다.<sup class="reference" id="cite_ref-ppkk_2-1"><a href="#cite_note-ppkk-2">[2]</a></sup> 카를은 전쟁 노획물과 많은 선물을 받아 서프랑크로 귀환했다.<sup class="reference" id="cite_ref-ppkk_2-2"><a href="#cite_note-ppkk-2">[2]</a></sup></p>
<p>877년 청년 루트비히는 교회에 일정 부분 재산을 기증했는데, 이는 왕비인 리우트가르트의 명의로 기부되었다. <a href="/wiki/877%EB%85%84" title="877년">877년</a> 말 루트비히는 형 <a class="mw-redirect" href="/wiki/%EC%B9%B4%EB%A5%BC%EB%A1%9C%EB%A7%8C_3%EC%84%B8" title="카를로만 3세">카를로만 3세</a>, 동생 <a href="/wiki/%EC%B9%B4%EB%A5%BC_3%EC%84%B8" title="카를 3세">카를 3세</a>와 함께 동부 로타링기아를 어떻게 분할하는가를 논의하기 위해 모였다. 그러나 <a href="/wiki/%EB%87%8C%EC%A1%B8%EC%A4%91" title="뇌졸중">뇌졸중</a> 등으로 병세가 위중했던 카를로만이 로타링기아를 포기하자 로타링기아는 루트비히와 카를이 분할하기로 했으며, <a href="/wiki/878%EB%85%84" title="878년">878년</a> <a href="/wiki/10%EC%9B%94" title="10월">10월</a> <a href="/wiki/%EC%95%8C%EC%9E%90%EC%8A%A4" title="알자스">알자스</a>에서 루트비히는 카를과 로타링기아를 분할했다.</p>

<div class="rellink boilerplate seealso"><img alt="" data-file-height="128" data-file-width="128" height="16" src="//upload.wikimedia.org/wikipedia/commons/thumb/e/ec/Crystal_Clear_app_xmag.svg/16px-Crystal_Clear_app_xmag.svg.png" srcset="//upload.wikimedia.org/wikipedia/commons/thumb/e/ec/Crystal_Clear_app_xmag.svg/24px-Crystal_Clear_app_xmag.svg.png 1.5x, //upload.wikimedia.org/wikipedia/commons/thumb/e/ec/Crystal_Clear_app_xmag.svg/32px-Crystal_Clear_app_xmag.svg.png 2x" width="16"/> <a class="new" href="/w/index.php?title=%ED%8F%AC%EC%97%90%EB%A5%B8_%EC%A1%B0%EC%95%BD&amp;action=edit&amp;redlink=1" title="포에른 조약 (없는 문서)">포에른 조약</a>, <a href="/wiki/%EB%B0%94%EC%9D%B4%ED%82%B9" title="바이킹">바이킹</a> 문서를 참고하십시오.</div>
<p>877년 10월 삼촌 <a class="mw-redirect" href="/wiki/%EB%8C%80%EB%A8%B8%EB%A6%AC%EC%99%95_%EC%B9%B4%EB%A5%BC" title="대머리왕 카를">대머리왕 카를</a>이 죽자, 그를 계승한 사촌 <a class="mw-disambig" href="/wiki/%EB%A3%A8%EC%9D%B4_2%EC%84%B8" title="루이 2세">루이 2세</a>와 로타링기아(로렌)에서 만나, <a class="mw-redirect" href="/wiki/%EB%A1%9C%ED%8A%B8%EB%A7%81%EA%B2%90" title="로트링겐">로트링겐</a>의 분할을 유지하는 데 동의했는데 이것은 그들의 아버지인 <a href="/wiki/%EC%B9%B4%EB%A5%BC_2%EC%84%B8_(%EC%84%9C%ED%94%84%EB%9E%91%ED%81%AC)" title="카를 2세 (서프랑크)">카를 2세 대머리왕</a>과 <a href="/wiki/%EB%A3%A8%ED%8A%B8%EB%B9%84%ED%9E%88_2%EC%84%B8_(%EB%8F%99%ED%94%84%EB%9E%91%ED%81%AC)" title="루트비히 2세 (동프랑크)">루트비히 게르만인</a>가 <a href="/wiki/870%EB%85%84" title="870년">870년</a> <a class="new" href="/w/index.php?title=%EB%A9%94%EB%A5%B4%EC%84%BC&amp;action=edit&amp;redlink=1" title="메르센 (없는 문서)">메르센</a>에서 <a href="/wiki/%EB%A1%9C%EB%A0%8C" title="로렌">로렌</a>을 분할한 조약을 재확인하는 것이었다. 바로 <a class="new" href="/w/index.php?title=%EB%A6%AC%EC%97%90_%EC%A3%BC&amp;action=edit&amp;redlink=1" title="리에 주 (없는 문서)">리에 주</a>의 푀렌에서 <a class="new" href="/w/index.php?title=%ED%8F%AC%EC%97%90%EB%A0%8C_%EC%A1%B0%EC%95%BD&amp;action=edit&amp;redlink=1" title="포에렌 조약 (없는 문서)">포에렌 조약</a>을 맺어 <a href="/wiki/%EB%A9%94%EB%A5%B4%EC%84%BC_%EC%A1%B0%EC%95%BD" title="메르센 조약">메르센 조약</a>으로 분할된 <a class="mw-redirect" href="/wiki/%EB%A1%9C%ED%83%80%EB%A7%81%EA%B8%B0%EC%95%84" title="로타링기아">로타링기아</a>의 영토를 서로 유지하는 데 동의했는다.</p>
<p>그러나 <a class="new" href="/w/index.php?title=%ED%91%80%EB%A0%8C_%EC%A1%B0%EC%95%BD&amp;action=edit&amp;redlink=1" title="푀렌 조약 (없는 문서)">푀렌 조약</a>이 발효되려는 참에 <a class="mw-disambig" href="/wiki/%EB%A3%A8%EC%9D%B4_2%EC%84%B8" title="루이 2세">루이 2세</a>가 사망했다. 요슬랭(Joscelin) 대주교가 이끄는 서프랑크의 일부 귀족이 그를 서프랑크 왕으로 추대했고, 바로 초청하였다. 또 아내 <a class="mw-disambig" href="/wiki/%EB%A6%AC%EC%9A%B0%ED%8A%B8%EA%B0%80%EB%A5%B4%ED%8A%B8" title="리우트가르트">리우트가르트</a>의 부추김까지 더해져 루트비히는 <a class="mw-redirect" href="/wiki/%EC%84%9C%ED%94%84%EB%9E%91%ED%81%AC" title="서프랑크">서프랑크</a>를 침공했다. <a class="mw-redirect" href="/wiki/%EB%A1%9C%ED%83%80%EB%A7%81%EA%B8%B0%EC%95%84" title="로타링기아">로타링기아</a>를 손에 넣은 루트비히 3세는 <a class="mw-redirect" href="/wiki/%EB%B2%A0%EB%A5%B4%EB%8D%A9" title="베르덩">베르덩</a> 전투에서 승리하여 <a class="mw-disambig" href="/wiki/%EB%A9%94%EC%B8%A0" title="메츠">메츠</a>와 <a class="mw-redirect" href="/wiki/%EB%B2%A0%EB%A5%B4%EB%8D%A9" title="베르덩">베르덩</a>, Scheldt을 수중에 넣었지만, 나중에 내부 반란을 수습하고 출정한 새로운 <a class="mw-redirect" href="/wiki/%EC%84%9C%ED%94%84%EB%9E%91%ED%81%AC" title="서프랑크">서프랑크</a>의 왕 <a href="/wiki/%EB%A3%A8%EC%9D%B4_3%EC%84%B8" title="루이 3세">루이 3세</a>와 <a href="/wiki/%EC%83%A4%EB%A5%BC%EB%A1%9C%EB%A7%9D" title="샤를로망">샤를로망</a> 형제에게 <a class="mw-redirect" href="/wiki/%EB%A1%9C%ED%83%80%EB%A7%81%EA%B8%B0%EC%95%84" title="로타링기아">로타링기아</a>에서 격퇴당했다.</p>
<p>879년 서프랑크의 루트비히가 죽고 그의 아들 루트비히 3세와 카를로만이 서프랑크의 공동 국왕이 되자, 청년 루트비히는 <a href="/wiki/870%EB%85%84" title="870년">870년</a>의 <a href="/wiki/%EB%A9%94%EB%A5%B4%EC%84%BC_%EC%A1%B0%EC%95%BD" title="메르센 조약">메르센 조약</a>은 불법이라며 계속 로트링겐의 영유권을 주장하였다. 노르망디와 브르타뉴 해안가로 쳐들어오는 <a href="/wiki/%EB%B0%94%EC%9D%B4%ED%82%B9" title="바이킹">바이킹</a> 해적을 상대하던 서프랑크인 <a href="/wiki/%EB%A3%A8%EC%9D%B4_3%EC%84%B8" title="루이 3세">루이 3세</a>는 자신의 동생 <a class="mw-redirect" href="/wiki/%EC%83%A4%EB%A5%BC%EB%A1%9C%EB%A7%9D_3%EC%84%B8" title="샤를로망 3세">샤를로망 3세</a>와 함께 <a class="mw-redirect" href="/wiki/%EB%A1%9C%ED%8A%B8%EB%A7%81%EA%B2%90" title="로트링겐">로트링겐</a>의 영유권을 주장하던 <a href="/wiki/%EC%9C%84%EA%B7%B8_(%EB%A1%9C%ED%8A%B8%EB%A7%81%EA%B2%90%EC%9D%98_%EC%99%95%EC%9E%90)" title="위그 (로트링겐의 왕자)">위그</a>를 추방했지만 계속 바이킹을 상대해야 했고, 청년 루트비히와의 전쟁을 피하려 했다. 결국 루이3세와 샤를로망 형제는 청년 루트비히와의 리베몽에서 영토 조약을 체결하기로 합의하고, 로트링겐을 사실상 동프랑크에 복속시켰다. <a href="/wiki/879%EB%85%84" title="879년">879년</a> 겨울, 형 <a class="mw-redirect" href="/wiki/%EB%B0%94%EC%9D%B4%EC%97%90%EB%A5%B8" title="바이에른">바이에른</a>의 카를로만이 <a href="/wiki/%EB%87%8C%EC%A1%B8%EC%A4%91" title="뇌졸중">뇌졸중</a>으로 쓰러지자 그는 형에게서 슈바벤과 바이에른을 물려받았다. 그러나 루트비히는 <a href="/wiki/879%EB%85%84" title="879년">879년</a>부터 이미 바이에른의 실질적 통치자 노릇을 하고 있던, 형 <a class="mw-redirect" href="/wiki/%EC%B9%B4%EB%A5%BC%EB%A1%9C%EB%A7%8C_3%EC%84%B8" title="카를로만 3세">카를로만 3세</a>의 서자 <a class="mw-disambig" href="/wiki/%EC%95%84%EB%A5%B4%EB%88%8C%ED%94%84" title="아르눌프">아르눌프</a>에게 바이에른 공작 자격으로 <a class="mw-redirect" href="/wiki/%EB%B0%94%EC%9D%B4%EC%97%90%EB%A5%B8" title="바이에른">바이에른</a>을 직접 통치하게 했다.</p>
<p><a href="/wiki/879%EB%85%84" title="879년">879년</a>부터 다시 <a href="/wiki/%EB%B0%94%EC%9D%B4%ED%82%B9" title="바이킹">바이킹</a>의 침략이 <a class="mw-disambig" href="/wiki/%ED%94%84%EB%9E%91%ED%81%AC" title="프랑크">프랑크</a> 왕국 내 각국에서 계속됐고 일부는 내륙 깊숙히 쳐들어와 공격, 약탈을 감행하였다. 루트비히는 <a href="/wiki/880%EB%85%84" title="880년">880년</a> <a href="/wiki/2%EC%9B%94" title="2월">2월</a> 티에몬(현재의 벨기에 샤를루아 일대)에 들어온 바이킹과 교전 <a class="new" href="/w/index.php?title=%ED%8B%B0%EC%97%90%EB%AA%AC_%EC%A0%84%ED%88%AC&amp;action=edit&amp;redlink=1" title="티에몬 전투 (없는 문서)">티에몬 전투</a>에서 바이킹을 크게 물리쳤지만, 이 전쟁에서 서자 위그가 <a class="new" href="/w/index.php?title=%EC%8A%A4%EC%BC%88%ED%8A%B8%EA%B0%95&amp;action=edit&amp;redlink=1" title="스켈트강 (없는 문서)">스켈트강</a>변(river Scheldt)에서 전사했다. <a href="/wiki/%EB%B0%94%EC%9D%B4%ED%82%B9" title="바이킹">바이킹</a>을 나이메헨 바깥까지 몰아냈지만 처남 <a class="mw-disambig" href="/wiki/%EB%B8%8C%EB%A3%A8%EB%85%B8" title="브루노">브루노</a>의 지휘 하에 출정한 작센인 군사들은 <a href="/wiki/%ED%95%A8%EB%B6%80%EB%A5%B4%ED%81%AC" title="함부르크">함부르크</a>에서 바이킹에게 크게 패하고 브루노와 많은 작센 귀족, 병사들이 전사했다. <a class="new" href="/w/index.php?title=%ED%8B%B0%EC%97%90%EB%AA%AC_%EC%A0%84%ED%88%AC&amp;action=edit&amp;redlink=1" title="티에몬 전투 (없는 문서)">티에몬 전투</a>에서 승리했음에도 불구하고 바이킹의 침략을 막지는 못했다.</p>

<p>880년 <a href="/wiki/2%EC%9B%94" title="2월">2월</a> 루트비히 3세는 <a class="mw-redirect" href="/wiki/%EC%84%9C%ED%94%84%EB%9E%91%ED%81%AC" title="서프랑크">서프랑크</a> <a href="/wiki/%EB%A3%A8%EC%9D%B4_3%EC%84%B8" title="루이 3세">루이 3세</a>, <a href="/wiki/%EC%83%A4%EB%A5%BC%EB%A1%9C%EB%A7%9D" title="샤를로망">샤를로망</a> 형제와 함께 영토 문제를 협상, <a href="/wiki/%EB%A6%AC%EB%B2%A0%EB%AA%BD_%EC%A1%B0%EC%95%BD" title="리베몽 조약">리베몽 조약</a>을 체결했다. 청년 루트비히는 자신의 동생 뚱보 <a href="/wiki/%EC%B9%B4%EB%A5%BC_3%EC%84%B8" title="카를 3세">카를 3세</a> 등과 함께 참석했다. 이때 청년 루트비히가 점령한 곳을 국경지대로 하기로 결정, 그가 점령한 지역을 서프랑크와 동프랑크간의 국경으로 확정하여 동프랑크는 로타링기아를 흡수했다. 한편 동생 <a href="/wiki/%EC%B9%B4%EB%A5%BC_3%EC%84%B8" title="카를 3세">뚱보 카를</a>이 로타링기아에 대한 영유권을 포기함으로써 그는 로타링기아 전역을 차지할 수 있었다. <a href="/wiki/%EB%A6%AC%EB%B2%A0%EB%AA%BD_%EC%A1%B0%EC%95%BD" title="리베몽 조약">리베몽 조약</a>으로 결정된 <a href="/wiki/%EB%8F%85%EC%9D%BC" title="독일">독일</a>-<a href="/wiki/%ED%94%84%EB%9E%91%EC%8A%A4" title="프랑스">프랑스</a>간 영토, 국경은 14세기까지 유지되었다. 자신의 영지 내에서 청년 루트비히는 아버지와 반대로, 왕실의 이익을 위해 귀족들의 권한을 감소시켰다. 그는 처가인 작센의 리우돌핑 가문을 포함해 왕가의 힘을 왕을 중심으로 강하게 결속시켰다. 형으로부터 물려받은 바이에른에도 2번밖에 방문하지 않았다. 한편 형 <a href="/wiki/%EC%B9%B4%EB%A5%BC%EB%A7%8C" title="카를만">카를만</a>이 사망하자, 청년 루트비히는 <a href="/wiki/880%EB%85%84" title="880년">880년</a> 내내 형 <a href="/wiki/%EC%B9%B4%EB%A5%BC%EB%A7%8C" title="카를만">카를만</a>의 서자이자 자신의 조카인 <a href="/wiki/%EC%95%84%EB%A5%B4%EB%88%8C%ED%94%84_%ED%8F%B0_%EC%BC%80%EB%A5%B8%ED%85%90" title="아르눌프 폰 케른텐">아르눌프</a>를 제거하려 시도했으나 실패했다.</p>
<p>그는 작센 공작 오토의 딸 리우트가르트와 결혼, 아들 루트비히와 베른하르트를 두었으나 모두 어린 나이에 일찍 요절했다. 아들 루트비히는 <a href="/wiki/879%EB%85%84" title="879년">879년</a> 가을 <a href="/wiki/%EB%A0%88%EA%B2%90%EC%8A%A4%EB%B6%80%EB%A5%B4%ED%81%AC" title="레겐스부르크">레겐스부르크</a> 성의 난간에서 떨어져 머리를 크게 다쳐 죽었다. 서자였던 <a class="new" href="/w/index.php?title=%EC%9C%84%EA%B7%B8_(%EB%8F%99%ED%94%84%EB%9E%91%ED%81%AC)&amp;action=edit&amp;redlink=1" title="위그 (동프랑크) (없는 문서)">위그</a>는 <a href="/wiki/880%EB%85%84" title="880년">880년</a> <a href="/wiki/2%EC%9B%94" title="2월">2월</a> <a href="/wiki/%EB%B0%94%EC%9D%B4%ED%82%B9" title="바이킹">바이킹</a> 족과 싸우다가 전사하고 말았다. 서자로 당시 중년이었던 작센 백작 <a href="/wiki/%EC%9C%84%EA%B7%B8_(%EB%8F%99%ED%94%84%EB%9E%91%ED%81%AC%EC%9D%98_%EC%99%95%EC%9E%90)" title="위그 (동프랑크의 왕자)">위그</a>에게 기대를 걸었으나 위그가 전사한 뒤 루트비히는 실의에 빠졌다. <a href="/wiki/880%EB%85%84" title="880년">880년</a> <a href="/wiki/%EB%87%8C%EC%A1%B8%EC%A4%91" title="뇌졸중">뇌졸중</a>과 <a href="/wiki/%EC%A4%91%ED%92%8D" title="중풍">중풍</a> 등으로 고생하던 친형 <a class="new" href="/w/index.php?title=%EB%B0%94%EC%9D%B4%EC%97%90%EB%A5%B8%EC%9D%98_%EA%B5%AD%EC%99%95&amp;action=edit&amp;redlink=1" title="바이에른의 국왕 (없는 문서)">바이에른의 국왕</a> <a href="/wiki/%EC%B9%B4%EB%A5%BC%EB%A7%8C" title="카를만">카를만</a>이 사망하자, 청년 루트비히는 카를만의 영지 중 <a class="mw-redirect" href="/wiki/%EB%B0%94%EC%9D%B4%EC%97%90%EB%A5%B8" title="바이에른">바이에른</a>을 물려받았다. 그러나 <a href="/wiki/881%EB%85%84" title="881년">881년</a> 겨울, 청년 루트비히는 갑자기 <a class="new" href="/w/index.php?title=%EC%97%B4%EB%B3%91&amp;action=edit&amp;redlink=1" title="열병 (없는 문서)">열병</a>으로 쓰러져 <a href="/wiki/882%EB%85%84" title="882년">882년</a> <a href="/wiki/1%EC%9B%94_20%EC%9D%BC" title="1월 20일">1월 20일</a> <a href="/wiki/%ED%94%84%EB%9E%91%ED%81%AC%ED%91%B8%EB%A5%B4%ED%8A%B8%EC%95%94%EB%A7%88%EC%9D%B8" title="프랑크푸르트암마인">프랑크푸르트암마인</a>에서 사망했다.</p>
<h3><span class="mw-headline" id=".EC.82.AC.ED.9B.84">사후</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%A3%A8%ED%8A%B8%EB%B9%84%ED%9E%88_3%EC%84%B8_(%EB%8F%99%ED%94%84%EB%9E%91%ED%81%AC)&amp;action=edit&amp;section=7" title="부분 편집: 사후">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>시신은 아버지 <a class="mw-redirect" href="/wiki/%EB%8F%85%EC%9D%BC%EC%9D%B8_%EB%A3%A8%ED%8A%B8%EB%B9%84%ED%9E%88" title="독일인 루트비히">독일인 루트비히</a>, 서자 <a href="/wiki/%EC%9C%84%EA%B7%B8_(%EB%8F%99%ED%94%84%EB%9E%91%ED%81%AC%EC%9D%98_%EC%99%95%EC%9E%90)" title="위그 (동프랑크의 왕자)">위그</a>가 안치된 <a class="new" href="/w/index.php?title=%EB%A1%9C%EC%89%AC_%EC%88%98%EB%8F%84%EC%9B%90&amp;action=edit&amp;redlink=1" title="로쉬 수도원 (없는 문서)">로쉬 수도원</a>(abbey of Lorsch)에 함께 안치되었다. 청년 루트비히에게는 남자 후손이 없었으므로, 그의 사후 작센과 로렌, 바이에른의 영지는 모두 친동생 <a href="/wiki/%EC%8A%88%EB%B0%94%EB%B2%A4" title="슈바벤">슈바벤</a>과 <a class="new" href="/w/index.php?title=%EC%95%8C%EB%A0%88%EB%A7%8C%EB%8B%88%EC%95%84&amp;action=edit&amp;redlink=1" title="알레만니아 (없는 문서)">알레만니아</a>를 다스리던 뚱보 <a href="/wiki/%EC%B9%B4%EB%A5%BC_3%EC%84%B8" title="카를 3세">카를 3세</a>에게로 넘어갔다. 부인인 <a class="new" href="/w/index.php?title=%EC%9E%91%EC%84%BC%EC%9D%98_%EB%A6%AC%EC%9A%B0%ED%8A%B8%EA%B0%80%EB%A5%B4%ED%8A%B8&amp;action=edit&amp;redlink=1" title="작센의 리우트가르트 (없는 문서)">작센의 리우트가르트</a>는 <a href="/wiki/882%EB%85%84" title="882년">882년</a>경 <a href="/wiki/%EC%8A%88%EB%B0%94%EB%B2%A4" title="슈바벤">슈바벤</a> <a class="mw-disambig" href="/wiki/%EA%B3%B5%EC%9E%91" title="공작">공작</a> <a class="new" href="/w/index.php?title=%EB%B6%80%EB%A5%B4%ED%95%98%EB%A5%B4%ED%8A%B8_1%EC%84%B8_%ED%8F%B0_%EC%8A%88%EB%B0%94%EB%B2%A4&amp;action=edit&amp;redlink=1" title="부르하르트 1세 폰 슈바벤 (없는 문서)">부르하르트 1세 폰 슈바벤</a>과 재혼했으나 <a href="/wiki/885%EB%85%84" title="885년">885년</a> 사망하여, 나중에 <a href="/wiki/%EC%95%84%EC%83%A4%ED%8E%9C%EB%B6%80%EB%A5%B4%ED%81%AC" title="아샤펜부르크">아샤펜부르크</a>에 안장되었다.</p>
<p>딸 힐데가르트는 <a href="/wiki/%EB%B0%94%EC%9D%B4%EC%97%90%EB%A5%B8_%EC%A3%BC" title="바이에른 주">바이에른 주</a> 킴제(Chiemsee)의 킴제 수도원의 수녀가 되었다. 힐데가르트는 895년 플랑드르의 안네(Annales of Fuldenses)의 밀고로 사촌인 황제 <a href="/wiki/%EC%95%84%EB%A5%B4%EB%88%8C%ED%94%84_%ED%8F%B0_%EC%BC%80%EB%A5%B8%ED%85%90" title="아르눌프 폰 케른텐">아르눌프</a>에 대한 반역 혐의로 기소된 뒤, <a class="mw-disambig" href="/wiki/%EC%95%84%EB%A5%B4%EB%88%8C%ED%94%84" title="아르눌프">아르눌프</a>에 의해 모든 영예를 박탈당하고 호수 변의 섬에 유폐되었다가 <a class="mw-redirect" href="/wiki/%EB%B0%94%EC%9D%B4%EC%97%90%EB%A5%B8" title="바이에른">바이에른</a>의 킴제 수도원으로 갔다. 힐데가르트는 <a class="mw-disambig" href="/wiki/%EC%95%84%EB%A5%B4%EB%88%8C%ED%94%84" title="아르눌프">아르눌프</a> 사후 <a href="/wiki/899%EB%85%84" title="899년">899년</a>에 명예를 회복했다. 19세기 무렵 루트비히 3세와 서자 <a href="/wiki/%EC%9C%84%EA%B7%B8_(%EB%8F%99%ED%94%84%EB%9E%91%ED%81%AC%EC%9D%98_%EC%99%95%EC%9E%90)" title="위그 (동프랑크의 왕자)">위그</a>가 안치된 석관은 실종되었다.</p>

<h2><span class="mw-headline" id=".EA.B8.B0.ED.83.80">기타</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%A3%A8%ED%8A%B8%EB%B9%84%ED%9E%88_3%EC%84%B8_(%EB%8F%99%ED%94%84%EB%9E%91%ED%81%AC)&amp;action=edit&amp;section=9" title="부분 편집: 기타">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p><a class="mw-redirect" href="/wiki/%EC%84%9C%ED%94%84%EB%9E%91%ED%81%AC" title="서프랑크">서프랑크</a>의 대머리왕 카를은 자신의 아들 <a class="mw-disambig" href="/wiki/%EB%A3%A8%EC%9D%B4_2%EC%84%B8" title="루이 2세">루이 2세</a>를 <a href="/wiki/855%EB%85%84" title="855년">855년</a> <a class="mw-redirect" href="/wiki/%EB%8F%99%ED%94%84%EB%9E%91%ED%81%AC" title="동프랑크">동프랑크</a>의 왕이라는 직책을 부여하고는 호시탐탐 <a href="/wiki/%EB%8F%99%ED%94%84%EB%9E%91%ED%81%AC_%EC%99%95%EA%B5%AD" title="동프랑크 왕국">동프랑크 왕국</a>의 영토를 노리고 있었다.</p>
<p><a href="/wiki/877%EB%85%84" title="877년">877년</a> <a class="mw-redirect" href="/wiki/%EC%84%9C%ED%94%84%EB%9E%91%ED%81%AC" title="서프랑크">서프랑크</a>의 <a class="mw-redirect" href="/wiki/%EB%8C%80%EB%A8%B8%EB%A6%AC%EC%99%95_%EC%B9%B4%EB%A5%BC" title="대머리왕 카를">대머리왕 카를</a>이 그의 영토에 침략했을 때 루트비히는 군사들에게 모두 흰 옷을 입히거나 흰 천을 준비하여 유령 집단처럼 보이게 했고, 밤에는 약간의 불을 밝혀서 유령과 비슷하게 행동하였다. 야밤에 기습공격을 하려고 오던 서프랑크의 군사들은 <a class="mw-redirect" href="/wiki/%EB%8F%99%ED%94%84%EB%9E%91%ED%81%AC" title="동프랑크">동프랑크</a>의 군대를 귀신으로 보고 놀라서 달아났다.</p>
