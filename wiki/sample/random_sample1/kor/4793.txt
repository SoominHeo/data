<p><b>루트비히 요제프 요한 비트겐슈타인</b>(<span style="font-size: smaller;"><a href="/wiki/%EB%8F%85%EC%9D%BC%EC%96%B4" title="독일어">독일어</a>:</span> <span lang="de" xml:lang="de">Ludwig Josef Johann Wittgenstein</span>, <a href="/wiki/1889%EB%85%84" title="1889년">1889년</a> <a href="/wiki/4%EC%9B%94_26%EC%9D%BC" title="4월 26일">4월 26일</a> <a href="/wiki/%EC%98%A4%EC%8A%A4%ED%8A%B8%EB%A6%AC%EC%95%84" title="오스트리아">오스트리아</a> – <a href="/wiki/1951%EB%85%84" title="1951년">1951년</a> <a href="/wiki/4%EC%9B%94_29%EC%9D%BC" title="4월 29일">4월 29일</a>)은 <a href="/wiki/%EC%98%A4%EC%8A%A4%ED%8A%B8%EB%A6%AC%EC%95%84" title="오스트리아">오스트리아</a>와 <a href="/wiki/%EC%98%81%EA%B5%AD" title="영국">영국</a>에서 활동한 <a href="/wiki/%EC%B2%A0%ED%95%99%EC%9E%90" title="철학자">철학자</a>이며, <a href="/wiki/%EB%85%BC%EB%A6%AC%ED%95%99" title="논리학">논리학</a>, <a class="mw-redirect" href="/wiki/%EC%88%98%ED%95%99_%EC%B2%A0%ED%95%99" title="수학 철학">수학 철학</a>, <a class="mw-redirect" href="/wiki/%EC%8B%AC%EB%A6%AC_%EC%B2%A0%ED%95%99" title="심리 철학">심리 철학</a>, <a class="mw-redirect" href="/wiki/%EC%96%B8%EC%96%B4_%EC%B2%A0%ED%95%99" title="언어 철학">언어 철학</a> 분야에 업적을 남겼다.<sup class="reference" id="cite_ref-Time_1-0"><a href="#cite_note-Time-1">[1]</a></sup> 많은 사람들이 20세기의 가장 위대한 철학자로 손꼽는다.<sup class="reference" id="cite_ref-2"><a href="#cite_note-2">[2]</a></sup><sup class="reference" id="cite_ref-3"><a href="#cite_note-3">[3]</a></sup></p>
<p>비트겐슈타인은 <a class="mw-redirect" href="/wiki/%EB%85%BC%EB%A6%AC_%EC%8B%A4%EC%A6%9D%EC%A3%BC%EC%9D%98" title="논리 실증주의">논리 실증주의</a>와 <a href="/wiki/%EC%9D%BC%EC%83%81_%EC%96%B8%EC%96%B4_%EC%B2%A0%ED%95%99" title="일상 언어 철학">일상 언어 철학</a>에 영향을 끼쳤고<sup class="reference" id="cite_ref-Time_1-1"><a href="#cite_note-Time-1">[1]</a></sup> <a class="mw-redirect" href="/wiki/%EB%B6%84%EC%84%9D_%EC%B2%A0%ED%95%99" title="분석 철학">분석 철학</a>을 대표하는 인재로도 알려져 있다. 20세기 말 한 철학 포럼이 정리한 세기를 대표하는 철학에서도 비트겐슈타인의 《<a class="mw-redirect" href="/wiki/%EB%85%BC%EB%A6%AC_%EC%B2%A0%ED%95%99_%EB%85%BC%EA%B3%A0" title="논리 철학 논고">논리 철학 논고</a>》와 《<a href="/wiki/%EC%B2%A0%ED%95%99_%ED%83%90%EA%B5%AC" title="철학 탐구">철학 탐구</a>》는 상위 5위권에 모두 선정되었다.<sup class="reference" id="cite_ref-4"><a href="#cite_note-4">[4]</a></sup> 비트겐슈타인의 사상은 후일 <a href="/wiki/%EC%9D%B8%EB%AC%B8%ED%95%99" title="인문학">인문학</a>과 <a class="mw-redirect" href="/wiki/%EC%82%AC%ED%9A%8C_%EA%B3%BC%ED%95%99" title="사회 과학">사회 과학</a>의 여러 방면에 영향을 주었고 분석 철학자로서는 예외로 예술가들에게 전파되었다.<sup class="reference" id="cite_ref-5"><a href="#cite_note-5">[5]</a></sup> 비트겐슈타인이라는 이름은 종종 천재를 가리키는 낱말로 쓰인다.<sup class="reference" id="cite_ref-6"><a href="#cite_note-6">[6]</a></sup><sup class="reference" id="cite_ref-7"><a href="#cite_note-7">[7]</a></sup></p>
<p>비트겐슈타인의 사상은 《<a class="mw-redirect" href="/wiki/%EB%85%BC%EB%A6%AC_%EC%B2%A0%ED%95%99_%EB%85%BC%EA%B3%A0" title="논리 철학 논고">논리 철학 논고</a>》로 대표되는 전기와 《<a href="/wiki/%EC%B2%A0%ED%95%99_%ED%83%90%EA%B5%AC" title="철학 탐구">철학 탐구</a>》로 대표되는 후기로 나뉜다. 《논리 철학 논고》에 나타난 전기 사상이 <a href="/wiki/%EB%AA%85%EC%A0%9C" title="명제">명제</a>에 사용된 <a href="/wiki/%EB%82%B1%EB%A7%90" title="낱말">낱말</a>의 <a href="/wiki/%EC%9D%80%EC%9C%A0" title="은유">은유</a>다운 관계를 분석하여 기존 철학에서 잘못된 개념 탓에 빚어진 논리에 상충하는 점을 지목하는 데 집중된 반면, 후기 사상은 <a href="/wiki/%EC%96%B8%EC%96%B4-%EB%86%80%EC%9D%B4" title="언어-놀이">언어-놀이</a>에서 상호 변환되는 <a class="mw-redirect" href="/wiki/%EC%9E%90%EC%97%B0_%EC%96%B8%EC%96%B4" title="자연 언어">자연 언어</a>가 논리에 부합한 구조로 정형화한 언어와는 다른 의미가 있다는 점을 역설하는데 중심이 놓여있다. 비트겐슈타인은 “언어의 의미는 사용에 있다”고 주장하였는데 이것은 비트겐슈타인의 후기 사상을 대표하는 말이다.</p>

<h2><span class="mw-headline" id=".EC.83.9D.EC.95.A0">생애</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%A3%A8%ED%8A%B8%EB%B9%84%ED%9E%88_%EB%B9%84%ED%8A%B8%EA%B2%90%EC%8A%88%ED%83%80%EC%9D%B8&amp;action=edit&amp;section=1" title="부분 편집: 생애">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<div class="thumb tright">
<div class="thumbinner" style="width:179px;"><a class="image" href="/wiki/%ED%8C%8C%EC%9D%BC:Karl_Wittgenstein.jpg"><img alt="" class="thumbimage" data-file-height="277" data-file-width="177" height="277" src="//upload.wikimedia.org/wikipedia/commons/e/ef/Karl_Wittgenstein.jpg" width="177"/></a>
<div class="thumbcaption">
<div class="magnify"><a class="internal" href="/wiki/%ED%8C%8C%EC%9D%BC:Karl_Wittgenstein.jpg" title="실제 크기로"></a></div>
카를 비트겐슈타인은 오스트리아-헝가리 제국에서 뛰어난 갑부였다.<sup class="reference" id="cite_ref-8"><a href="#cite_note-8">[8]</a></sup> 1890년 촬영</div>
</div>
</div>
<p>비트겐슈타인은 <a href="/wiki/1889%EB%85%84" title="1889년">1889년</a> <a href="/wiki/4%EC%9B%94_26%EC%9D%BC" title="4월 26일">4월 26일</a>에 <a href="/wiki/%EB%B9%88" title="빈">빈</a>에서 태어났다. 아버지 <a href="/wiki/%EC%B9%B4%EB%A5%BC_%EB%B9%84%ED%8A%B8%EA%B2%90%EC%8A%88%ED%83%80%EC%9D%B8" title="카를 비트겐슈타인">카를 비트겐슈타인</a>은 <a href="/wiki/%EC%98%A4%EC%8A%A4%ED%8A%B8%EB%A6%AC%EC%95%84-%ED%97%9D%EA%B0%80%EB%A6%AC_%EC%A0%9C%EA%B5%AD" title="오스트리아-헝가리 제국">오스트리아 헝가리 제국</a>에서 손꼽히는 부자였고 어머니는 레오폴디네 비트겐슈타인이다. 루트비히는 8남매 가운데 막내였다. 할아버지와 할머니인 헤르만 크리스티안과 파니 비트겐슈타인은 <a href="/wiki/%EC%9C%A0%EB%8C%80%EC%9D%B8" title="유대인">유대인</a>이었으나 <a href="/wiki/%EA%B0%9C%EC%8B%A0%EA%B5%90" title="개신교">개신교</a>로 개종하였고 1850년대에 <a class="mw-redirect" href="/wiki/%EC%9E%91%EC%84%BC" title="작센">작센</a>에서 빈으로 이주하였다. 할머니 파니 비트겐슈타인은 유명한 바이올린 연주가인 <a class="mw-redirect" href="/wiki/%EC%9A%94%EC%A0%9C%ED%94%84_%EC%9A%94%ED%95%98%EC%9E%84" title="요제프 요하임">요제프 요하임</a>의 사촌이다.<sup class="reference" id="cite_ref-9"><a href="#cite_note-9">[9]</a></sup><sup class="reference" id="cite_ref-10"><a href="#cite_note-10">[10]</a></sup> 아버지 카를 비트겐슈타인은 제철업을 하여 많은 돈을 벌었다. 1880년대 후반 무렵 카를 비트겐슈타인은 오스트리아 헝가리 제국에서 <a href="/wiki/%EB%8F%85%EC%A0%90" title="독점">독점</a>하는 제철 사업가이자 세계에서 뛰어난 부자가 되었다.<sup class="reference" id="cite_ref-11"><a href="#cite_note-11">[11]</a></sup> 카를 비트겐슈타인의 자산은 부동산, 주식, 귀금속, 외화 형태로 스위스, 오스트리아, 네덜란드, 북미 등지에 분산되어 있었다. 재산이 분산되어 있었기에 막대한 인플레이션이 일어난 뒤에도 부를 유지할 수 있었다.<sup class="reference" id="cite_ref-12"><a href="#cite_note-12">[12]</a></sup> 비트겐슈타인의 어머니인 레오폴디네 칼무스는 유대인 아버지와 천주교 신자인 어머니 사이에서 태어났으며 노벨상을 받은 <a href="/wiki/%ED%94%84%EB%A6%AC%EB%93%9C%EB%A6%AC%ED%9E%88_%ED%95%98%EC%9D%B4%EC%97%90%ED%81%AC" title="프리드리히 하이에크">프리드리히 하이에크</a>의 이모이기도 하다. 이런 가족력에 따라 비트겐슈타인의 가족은 개신교와 가톨릭(천주교)에서 세례받았는데 다른 형제들은 <a href="/wiki/%EC%B9%A8%EB%A1%80%EA%B5%90" title="침례교">침례교</a>를 신봉하였고 비트겐슈타인은 할머니의 신앙을 좇아 <a href="/wiki/%EA%B0%80%ED%86%A8%EB%A6%AD" title="가톨릭">가톨릭</a>(<a class="mw-redirect" href="/wiki/%EC%B2%9C%EC%A3%BC%EA%B5%90" title="천주교">천주교</a>)에서 세례받았다.<sup class="reference" id="cite_ref-13"><a href="#cite_note-13">[13]</a></sup></p>

<p>예술가와 지식인을 존중하는 집안 분위기에서 성장한 비트겐슈타인의 아버지와 어머니는 둘다 고등교육을 받았고 음악 같은 예술을 사랑하였다. 제철 사업가로서뿐만 아니라 예술 애호자로서도 유명했던 카를 비트겐슈타인은 <a href="/wiki/%EC%98%A4%EA%B7%80%EC%8A%A4%ED%8A%B8_%EB%A1%9C%EB%8C%95" title="오귀스트 로댕">오귀스트 로댕</a>, <a href="/wiki/%EA%B5%AC%EC%8A%A4%ED%83%80%ED%94%84_%ED%81%B4%EB%A6%BC%ED%8A%B8" title="구스타프 클림트">구스타프 클림트</a> 같은 예술가를 후원하였고 <a href="/wiki/%EB%B9%88_%EB%B6%84%EB%A6%AC%ED%8C%8C_%EC%A0%84%EC%8B%9C%EA%B4%80" title="빈 분리파 전시관">빈 분리파 전시관</a>을 설립하려는 재정을 책임졌다.<sup class="reference" id="cite_ref-14"><a href="#cite_note-14">[14]</a></sup></p>

<div class="thumbinner" style="width:222px;"><a class="image" href="/wiki/%ED%8C%8C%EC%9D%BC:Palais_Wittgenstein_Rote_Salon_1910.JPG"><img alt="" class="thumbimage" data-file-height="1429" data-file-width="1920" height="164" src="//upload.wikimedia.org/wikipedia/commons/thumb/2/20/Palais_Wittgenstein_Rote_Salon_1910.JPG/220px-Palais_Wittgenstein_Rote_Salon_1910.JPG" srcset="//upload.wikimedia.org/wikipedia/commons/thumb/2/20/Palais_Wittgenstein_Rote_Salon_1910.JPG/330px-Palais_Wittgenstein_Rote_Salon_1910.JPG 1.5x, //upload.wikimedia.org/wikipedia/commons/thumb/2/20/Palais_Wittgenstein_Rote_Salon_1910.JPG/440px-Palais_Wittgenstein_Rote_Salon_1910.JPG 2x" width="220"/></a>

<p>비트겐슈타인의 집에는 <a href="/wiki/%EC%9A%94%ED%95%98%EB%84%A4%EC%8A%A4_%EB%B8%8C%EB%9E%8C%EC%8A%A4" title="요하네스 브람스">요하네스 브람스</a>, <a href="/wiki/%EB%A6%AC%ED%95%98%EB%A5%B4%ED%8A%B8_%EC%8A%88%ED%8A%B8%EB%9D%BC%EC%9A%B0%EC%8A%A4" title="리하르트 슈트라우스">리하르트 슈트라우스</a>, <a href="/wiki/%EA%B5%AC%EC%8A%A4%ED%83%80%ED%94%84_%EB%A7%90%EB%9F%AC" title="구스타프 말러">구스타프 말러</a> 같은 예술가가 늘 초대되었다. 요하네스 브람스는 비트겐슈타인보다 두 살 위의 누나에게 피아노를 가르쳤고 요하네스 브람스의 주요한 작품 가운데 몇몇을 비트겐슈타인 집의 음악실에서 초연하였다.<sup class="reference" id="cite_ref-15"><a href="#cite_note-15">[15]</a></sup> 비트겐슈타인의 형 <a href="/wiki/%ED%8C%8C%EC%9A%B8_%EB%B9%84%ED%8A%B8%EA%B2%90%EC%8A%88%ED%83%80%EC%9D%B8" title="파울 비트겐슈타인">파울 비트겐슈타인</a>은 후일 세계적으로 유명한 피아니스트가 되었다. 파울은 <a class="mw-redirect" href="/wiki/%EC%A0%9C1%EC%B0%A8_%EC%84%B8%EA%B3%84%EB%8C%80%EC%A0%84" title="제1차 세계대전">제1차 세계대전</a> 참전 중 오른팔을 잃고 러시아에서 포로수용소에 갇혀서도 왼손만으로 피아노를 연습하여 명성을 유지하였다.<sup class="reference" id="cite_ref-16"><a href="#cite_note-16">[16]</a></sup> 비트겐슈타인도 <a class="mw-redirect" href="/wiki/%EC%A0%88%EB%8C%80%EC%9D%8C%EA%B0%90" title="절대음감">절대음감</a>이 있었고<sup class="reference" id="cite_ref-ReferenceA_17-0"><a href="#cite_note-ReferenceA-17">[17]</a></sup>, 음악을 향한 열정은 비트겐슈타인의 생활과 사상에 큰 영향을 주었다. 비트겐슈타인은 자신의 철학 저작에서도 음악에 빗댄 예를 자주 사용하였고 <a href="/wiki/%ED%81%B4%EB%9D%BC%EB%A6%AC%EB%84%B7" title="클라리넷">클라리넷</a>을 능숙하게 연주하여 오케스트라와 협연하기도 하였다. 루트비히 요제프 요한 비트겐슈타인은 교향곡 전체를 외워 휘파람으로 불렀다.<sup class="reference" id="cite_ref-.EC.95.88.EA.B4.91.EB.B3.B5_18-0"><a href="#cite_note-.EC.95.88.EA.B4.91.EB.B3.B5-18">[18]</a></sup></p>
<p>비트겐슈타인의 가족은 <a href="/wiki/%EC%9A%B0%EC%9A%B8%EC%A6%9D" title="우울증">우울증</a>에 따른 <a href="/wiki/%EC%9E%90%EC%82%B4" title="자살">자살</a> 경향을 보였다. 장남인 한스는 네 살 때 작곡할 정도로 신동이었으나 1902년 <a href="/wiki/%EC%BF%A0%EB%B0%94" title="쿠바">쿠바</a>의 <a href="/wiki/%EC%95%84%EB%B0%94%EB%82%98" title="아바나">아바나</a>에서 자살했다. 뒤를 이어 삼남 루돌프가 1904년 베를린에서 자살하였고 차남 쿠르트는 제1차 세계대전이 끝나가던 1918년 10월 오스트리아군의 총 퇴각이 결정되자 자신이 지휘하던 진중에서 총을 이용해 자살하였다.<sup class="reference" id="cite_ref-19"><a href="#cite_note-19">[19]</a></sup> 루트비히도 청년 시절 줄곧 자살 충동에 시달렸고, 러셀로부터 천재로 인정받은 후에야 마음의 안정을 찾을 수 있었다.<sup class="reference" id="cite_ref-20"><a href="#cite_note-20">[20]</a></sup></p>
<blockquote class="toccolours" style="float:none; padding: 10px 15px 10px 15px; display:table;">
<p>최근 내 상황은 완전히 비참함 자체입니다. 나는 생명을 끊는 것을 계속 생각해왔습니다. 지금도 이 생각은 여전히 나를 괴롭힙니다. 나는 내려갈 수 있는 가장 마지막 지점까지 가라앉았습니다. 당신은 절대로 이 지점까지 오지 않기를!<sup class="reference" id="cite_ref-21"><a href="#cite_note-21">[21]</a></sup></p>
<p style="text-align: right;">— <cite>파울 엥겔만에게 보낸 편지, 1920.5.30.</cite></p>

<p>1903년까지 비트겐슈타인은 집에서 가정교육받은 후 <a href="/wiki/%EB%A6%B0%EC%B8%A0" title="린츠">린츠</a>에 있는 린츠 국립실업고등학교에 3학년으로 편입하였다. 비트겐슈타인보다 6개월 생일이 빠른 <a href="/wiki/%EC%95%84%EB%8F%8C%ED%94%84_%ED%9E%88%ED%8B%80%EB%9F%AC" title="아돌프 히틀러">아돌프 히틀러</a>가 1년간 비트겐슈타인과 같은 학교에 다녔다.<sup class="reference" id="cite_ref-22"><a href="#cite_note-22">[22]</a></sup> 아돌프 히틀러와 비트겐슈타인이 서로 알던 사이였는지는 알려진 바가 없으나 양측 모두 이와 관련하여 어떠한 회고도 없는 것으로 보아 일면식도 없는 사이였으리라 추정된다.</p>
<p>학교 생활 동안 비트겐슈타인은 높은 톤의 어투에 약간 말더듬이가 있었고 우아한 옷을 입은 민감하고 겉으로 드러내지 않고 심중으로만 생각하는 학생이었다. 동급생 거의 대부분이 비트겐슈타인의 이런 점을 특징으로 꼽았다. 비트겐슈타인은 "선생", "헤어 루트비히"(<span style="font-size: smaller;"><a href="/wiki/%EB%8F%85%EC%9D%BC%EC%96%B4" title="독일어">독일어</a>:</span> <span lang="de" xml:lang="de">Herr Ludwig</span><font color="gray"><small>→루트비히씨<sup class="reference" id="cite_ref-23"><a href="#cite_note-23">[주해 1]</a></sup></small></font>)와 같은 별명으로 불렸다.<sup class="reference" id="cite_ref-24"><a href="#cite_note-24">[23]</a></sup></p>
<p>비트겐슈타인은 1905년 하늘을 나는 문제를 해결한 영웅과 천재들의 이야기가 들어 있는 <a href="/wiki/%EB%A3%A8%ED%8A%B8%EB%B9%84%ED%9E%88_%EB%B3%BC%EC%B8%A0%EB%A7%8C" title="루트비히 볼츠만">루트비히 볼츠만</a>의 선집<sup class="reference" id="cite_ref-Ste_25-0"><a href="#cite_note-Ste-25">[24]</a></sup> 을 읽고 볼츠만에게서 물리학을 배우려고 했으나 볼츠만은 1906년 생을 자살로 마감하였다.<sup class="reference" id="cite_ref-26"><a href="#cite_note-26">[25]</a></sup></p>
<p>1906년 베를린에서 <a href="/wiki/%EA%B8%B0%EA%B3%84%EA%B3%B5%ED%95%99" title="기계공학">기계공학</a>을 배우기 시작한 비트겐슈타인은 <a href="/wiki/%EA%B0%88%EB%A6%B4%EB%A0%88%EC%98%A4_%EA%B0%88%EB%A6%B4%EB%A0%88%EC%9D%B4" title="갈릴레오 갈릴레이">갈릴레오</a>, <a href="/wiki/%EB%A0%88%EC%98%A4%EB%82%98%EB%A5%B4%EB%8F%84_%EB%8B%A4_%EB%B9%88%EC%B9%98" title="레오나르도 다 빈치">레오나르도 다 빈치</a>, <a href="/wiki/%ED%8C%8C%EC%9A%B0%EC%8A%A4%ED%86%A0_%EB%B2%A0%EB%9E%80%EC%B9%98%EC%98%A4" title="파우스토 베란치오">파우스토 베란치오</a> 같은 사람들이 남긴 <a href="/wiki/%EB%A5%B4%EB%84%A4%EC%83%81%EC%8A%A4" title="르네상스">르네상스</a> 시기 작업에 흥미를 느꼈다.<sup class="reference" id="cite_ref-27"><a href="#cite_note-27">[26]</a></sup> 1908년 <a href="/wiki/%EB%A7%A8%EC%B2%B4%EC%8A%A4%ED%84%B0_%EB%B9%85%ED%86%A0%EB%A6%AC%EC%95%84_%EB%8C%80%ED%95%99%EA%B5%90" title="맨체스터 빅토리아 대학교">맨체스터 빅토리아 대학교</a>에서 <a href="/wiki/%ED%95%AD%EA%B3%B5" title="항공">항공</a>과 관련한 논문으로 <a href="/wiki/%EA%B3%B5%ED%95%99" title="공학">공학</a> <a href="/wiki/%EB%B0%95%EC%82%AC" title="박사">박사</a> 학위를 받은 비트겐슈타인은 공학 연구소의 연구생으로 등록되어 <a class="mw-redirect" href="/wiki/%EB%8C%80%EA%B8%B0%EA%B6%8C" title="대기권">대기권</a> 상층에 연을 띄우는 연구하였고 작은 제트 엔진에 사용될 프로펠러를 제작하였다.<sup class="reference" id="cite_ref-.EC.95.88.EA.B4.91.EB.B3.B5_18-1"><a href="#cite_note-.EC.95.88.EA.B4.91.EB.B3.B5-18">[18]</a></sup> 맨체스터에서 연구 생활하는 동안 버트런드 러셀과 <a class="mw-redirect" href="/wiki/%EC%95%A8%ED%94%84%EB%A6%AC%EB%93%9C_%EB%85%B8%EC%8A%A4_%ED%99%94%EC%9D%B4%ED%8A%B8%ED%97%A4%EB%93%9C" title="앨프리드 노스 화이트헤드">앨프리드 노스 화이트헤드</a>가 공저한 《<a href="/wiki/%EC%88%98%ED%95%99_%EC%9B%90%EB%A6%AC" title="수학 원리">수학 원리</a>》<sup class="reference" id="cite_ref-28"><a href="#cite_note-28">[27]</a></sup>, <a href="/wiki/%EA%B3%A0%ED%8B%80%EB%A1%9C%ED%94%84_%ED%94%84%EB%A0%88%EA%B2%8C" title="고틀로프 프레게">고틀로프 프레게</a>의 《산수의 근본 법칙》를 읽고 <a href="/wiki/%EC%88%98%ED%95%99%EA%B8%B0%EC%B4%88%EB%A1%A0" title="수학기초론">수학기초론</a>에 흥미를 느꼈다.<sup class="reference" id="cite_ref-29"><a href="#cite_note-29">[28]</a></sup> 1911년 여름 비트겐슈타인은 <a class="mw-redirect" href="/wiki/%ED%94%84%EB%A0%88%EA%B2%8C" title="프레게">프레게</a>를 방문한 후부터 연락을 자주 교환했고 <a href="/wiki/%EC%BC%80%EC%9E%84%EB%B8%8C%EB%A6%AC%EC%A7%80_%EB%8C%80%ED%95%99%EA%B5%90" title="케임브리지 대학교">케임브리지 대학교</a>의 버트런드 러셀에게서 배우기로 하였다.<sup class="reference" id="cite_ref-Connor-Robertson_30-0"><a href="#cite_note-Connor-Robertson-30">[29]</a></sup></p>
<h3><span class="mw-headline" id=".EC.BC.80.EC.9E.84.EB.B8.8C.EB.A6.AC.EC.A7.80_.EB.8C.80.ED.95.99_.EC.8B.9C.EC.A0.88">케임브리지 대학 시절</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%A3%A8%ED%8A%B8%EB%B9%84%ED%9E%88_%EB%B9%84%ED%8A%B8%EA%B2%90%EC%8A%88%ED%83%80%EC%9D%B8&amp;action=edit&amp;section=3" title="부분 편집: 케임브리지 대학 시절">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<div class="thumb tright">
<div class="thumbinner" style="width:222px;"><a class="image" href="/wiki/%ED%8C%8C%EC%9D%BC:Honourable_Bertrand_Russell.jpg"><img alt="" class="thumbimage" data-file-height="1859" data-file-width="1255" height="326" src="//upload.wikimedia.org/wikipedia/commons/thumb/9/9b/Honourable_Bertrand_Russell.jpg/220px-Honourable_Bertrand_Russell.jpg" srcset="//upload.wikimedia.org/wikipedia/commons/thumb/9/9b/Honourable_Bertrand_Russell.jpg/330px-Honourable_Bertrand_Russell.jpg 1.5x, //upload.wikimedia.org/wikipedia/commons/thumb/9/9b/Honourable_Bertrand_Russell.jpg/440px-Honourable_Bertrand_Russell.jpg 2x" width="220"/></a>
<div class="thumbcaption">
<div class="magnify"><a class="internal" href="/wiki/%ED%8C%8C%EC%9D%BC:Honourable_Bertrand_Russell.jpg" title="실제 크기로"></a></div>
버트런트 러셀<br/>
1916년 촬영</div>
</div>
</div>
<p>비트겐슈타인은 1911년 8월부터 <a href="/wiki/%EB%B2%84%ED%8A%B8%EB%9F%B0%EB%93%9C_%EB%9F%AC%EC%85%80" title="버트런드 러셀">버트런드 러셀</a>의 강의실에 출석하였고 얼마 지나지 않아 러셀과 웃음꽃을 피우면서 철학을 주제로 토론하는 사이가 되었다. 러셀은 비트겐슈타인에게 깊은 감명을 받았고, <a class="mw-redirect" href="/wiki/%EC%A1%B0%EC%A7%80_%EC%97%90%EB%93%9C%EC%9B%8C%EB%93%9C_%EB%AC%B4%EC%96%B4" title="조지 에드워드 무어">조지 에드워드 무어</a>를 알게 되어 수리 논리학의 원리를 대상으로 한 작업을 시작하였다.<sup class="reference" id="cite_ref-31"><a href="#cite_note-31">[30]</a></sup></p>
<p>이 시기 러셀은 수학 기초론과 철학을 확장하고자 연구하고 있었고 비트겐슈타인을 자신의 후계자로 여겼다.<sup class="reference" id="cite_ref-32"><a href="#cite_note-32">[31]</a></sup> 러셀은 비트겐슈타인에게서 자주 거침없이 비판받기도 하였다. 러셀은 비트겐슈타인이 천재다운 능력을 이용해 논리학상 난제를 해결하는 것에 매료되었고 영국 철학계에 비트겐슈타인의 능력을 소개하는 데 주력하였다. 그러나 비트겐슈타인은 러셀에 의해 자신이 논리실증주의 철학자로만 비추어지는 것을 못마땅해 하였다.<sup class="reference" id="cite_ref-33"><a href="#cite_note-33">[32]</a></sup></p>
<blockquote class="toccolours" style="float:none; padding: 10px 15px 10px 15px; display:table;">
<p>그 사람은 천재에게서 나타나는 전통 특징을 전형이 될 만하게 현로했다. 재언하면, 그 사람은 열정 있고 해박했으며, 격렬하고도 좌중을 휘어잡는 인재였다.<sup class="reference" id="cite_ref-34"><a href="#cite_note-34">[33]</a></sup><br/>
그 사람이 옳다는 것을 알 수 있었으며 나는 더는 철학의 기초 작업을 진행할 수 있으리라 기대할 수 없었다.<sup class="reference" id="cite_ref-35"><a href="#cite_note-35">[34]</a></sup></p>
<p style="text-align: right;">— <cite>러셀의 회고</cite></p>
</blockquote>
<p>이 시기 동안, 비트겐슈타인의 또다른 주요 관심사는 고전음악과 여행이었다. 1912년 <a href="/wiki/%EC%95%84%EC%9D%B4%EC%8A%AC%EB%9E%80%EB%93%9C" title="아이슬란드">아이슬란드</a>를 여행한 비트겐슈타인은 여로에 종종 동성 애인이었던 데이비드 핀센트와 동행하였으며 비트겐슈타인은 버트런드 러셀과 무어가 이끌던 비밀 모임인 <a class="mw-redirect" href="/wiki/%EC%BC%80%EC%9E%84%EB%B8%8C%EB%A6%AC%EC%A7%80_%EC%96%B4%ED%8F%AC%EC%8A%AC%EC%8A%A4" title="케임브리지 어포슬스">케임브리지 어포슬스</a>에 가입하였다.</p>
<p><a href="/wiki/1913%EB%85%84" title="1913년">1913년</a>에 비트겐슈타인의 아버지가 죽었다. 비트겐슈타인은 유산을 상속받아 유럽에서 부유한 사람 가운데 한 명이 되었다.<sup class="reference" id="cite_ref-36"><a href="#cite_note-36">[35]</a></sup> 비트겐슈타인은 상속받은 유산의 상당량을 오스트리아의 예술가와 작가들을 도우려고 기부하였다. 비트겐슈타인이 지원한 예술가 가운데에는 <a href="/wiki/%EB%9D%BC%EC%9D%B4%EB%84%88_%EB%A7%88%EB%A6%AC%EC%95%84_%EB%A6%B4%EC%BC%80" title="라이너 마리아 릴케">라이너 마리아 릴케</a>, <a href="/wiki/%EA%B2%8C%EC%98%A4%EB%A5%B4%ED%81%AC_%ED%8A%B8%EB%9D%BC%ED%81%B4" title="게오르크 트라클">게오르크 트라클</a> 같은 사람들이 있었다.<sup class="reference" id="cite_ref-37"><a href="#cite_note-37">[36]</a></sup> <a href="/wiki/1914%EB%85%84" title="1914년">1914년</a>에 비트겐슈타인은 트라클을 방문하고자 하였으나 트라클이 자살로 급작스럽게 생을 마감하여 뜻을 이루지 못하였다.<sup class="reference" id="cite_ref-38"><a href="#cite_note-38">[주해 2]</a></sup> 비트겐슈타인은 볼츠만에 이어 트라클까지 꼭 만나고 싶은 사람이 자살하는 일을 겪게 되었다.</p>
<p>비트겐슈타인은 케임브리지 대학교이 자신이 일하기에 이상적이지 못하다고 생각했는데, 그에겐 대학연구자들이 심오한 생각도 없으면서 그저 자신들이 영리하다는 것을 보여주려고 애쓰고 있다고 느껴졌기 때문이다. <a href="/wiki/1913%EB%85%84" title="1913년">1913년</a>에 비트겐슈타인은 <a href="/wiki/%EB%85%B8%EB%A5%B4%EC%9B%A8%EC%9D%B4" title="노르웨이">노르웨이</a>의 <a class="mw-redirect" href="/wiki/%EC%86%A1%EB%84%A4_%ED%94%BC%EC%9A%94%EB%A5%B4%EB%93%9C" title="송네 피요르드">송네 피요르드</a>의 끝에 있는 <a class="new" href="/w/index.php?title=%EC%8A%A4%ED%81%AC%EC%9A%9C%EB%8D%B4&amp;action=edit&amp;redlink=1" title="스크욜덴 (없는 문서)">스크욜덴</a> 인근의 오지에서 독거에 들어갔다.<sup class="reference" id="cite_ref-Connor-Robertson_30-1"><a href="#cite_note-Connor-Robertson-30">[29]</a></sup> 비트겐슈타인은 이곳의 한 집 이층을 빌려 겨우내 지내면서 자신의 작업에 몰두하였다. 이 시기는 비트겐슈타인에게서 가장 정열 있고 새로운 것이 생겨나는 시간이었다. 이 시기에 이루어진 비트겐슈타인의 작업은 흔히 "논고"라고 불리는 《<a class="mw-redirect" href="/wiki/%EB%85%BC%EB%A6%AC_%EC%B2%A0%ED%95%99_%EB%85%BC%EA%B3%A0" title="논리 철학 논고">논리 철학 논고</a>》로 정리되었다.</p>
<h3><span class="mw-headline" id=".EC.A0.9C1.EC.B0.A8_.EC.84.B8.EA.B3.84.EB.8C.80.EC.A0.84">제1차 세계대전</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%A3%A8%ED%8A%B8%EB%B9%84%ED%9E%88_%EB%B9%84%ED%8A%B8%EA%B2%90%EC%8A%88%ED%83%80%EC%9D%B8&amp;action=edit&amp;section=4" title="부분 편집: 제1차 세계대전">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<div class="thumb tright">
<div class="thumbinner" style="width:222px;"><a class="image" href="/wiki/%ED%8C%8C%EC%9D%BC:Wittgenstein_notes_1914.jpg"><img alt="" class="thumbimage" data-file-height="480" data-file-width="640" height="165" src="//upload.wikimedia.org/wikipedia/commons/thumb/4/4e/Wittgenstein_notes_1914.jpg/220px-Wittgenstein_notes_1914.jpg" srcset="//upload.wikimedia.org/wikipedia/commons/thumb/4/4e/Wittgenstein_notes_1914.jpg/330px-Wittgenstein_notes_1914.jpg 1.5x, //upload.wikimedia.org/wikipedia/commons/thumb/4/4e/Wittgenstein_notes_1914.jpg/440px-Wittgenstein_notes_1914.jpg 2x" width="220"/></a>
<div class="thumbcaption">
<div class="magnify"><a class="internal" href="/wiki/%ED%8C%8C%EC%9D%BC:Wittgenstein_notes_1914.jpg" title="실제 크기로"></a></div>
케임브리지 트리니티 컬리지에 전시된 1914년의 노트</div>
</div>
</div>
<p><a class="mw-redirect" href="/wiki/%EC%A0%9C1%EC%B0%A8_%EC%84%B8%EA%B3%84%EB%8C%80%EC%A0%84" title="제1차 세계대전">제1차 세계대전</a>이 일어난 <a href="/wiki/1914%EB%85%84" title="1914년">1914년</a>에 비트겐슈타인은 1년째 은둔 생활하고 있었다. 비트겐슈타인은 전쟁이 일어났다는 소식에 크게 충격받았고 <a href="/wiki/%EC%98%A4%EC%8A%A4%ED%8A%B8%EB%A6%AC%EC%95%84-%ED%97%9D%EA%B0%80%EB%A6%AC_%EC%A0%9C%EA%B5%AD_%EC%9C%A1%EA%B5%B0" title="오스트리아-헝가리 제국 육군">오스트리아 헝가리 제국 육군</a>에 자원하였다. 입대 초기에는 함선에서 근무하였으나 얼마 후 포병대로 전근하였다. <a href="/wiki/1916%EB%85%84" title="1916년">1916년</a> <a href="/wiki/3%EC%9B%94" title="3월">3월</a>에 비트겐슈타인은 오스트리아 7군에 자원하여 <a href="/wiki/%EB%B8%8C%EB%A3%A8%EC%8B%A4%EB%A1%9C%ED%94%84_%EA%B3%B5%EC%84%B8" title="브루실로프 공세">브루실로프 공세</a>를 방어하는 최전방 조종사로 참전하였다.<sup class="reference" id="cite_ref-Monk.2C_p.137-142_39-0"><a href="#cite_note-Monk.2C_p.137-142-39">[37]</a></sup> 영국군과의 전투 후에 비트겐슈타인은 "용감한 행동과 침착하고 냉정한 영웅다운 활약"으로 말미암아 수훈했다.<sup class="reference" id="cite_ref-40"><a href="#cite_note-40">[38]</a></sup> 1917년 1월 비트겐슈타인은 러시아 전선의 곡사포 연대로 전근되었고 여기서 다시 용감한 행동으로 은장무공훈장을 비롯해 훈장 여러 개를 받았다.<sup class="reference" id="cite_ref-Monk.2C_p.137-142_39-1"><a href="#cite_note-Monk.2C_p.137-142-39">[37]</a></sup> 1918년 비트겐슈타인은 대위로 승진하였고 오늘날 <a href="/wiki/%ED%8A%B8%EB%A0%8C%ED%8B%B0%EB%85%B8%EC%95%8C%ED%86%A0%EC%95%84%EB%94%94%EC%A0%9C_%EC%A3%BC" title="트렌티노알토아디제 주">트렌티노</a>로 불리는 이탈리아 티롤 남부 전역에서 산악 포병 연대에 배속되었다. 1918년 6월 오스트리아의 공세에서 전훈을 세운 비트겐슈타인은 최고 훈장인 금장무공훈장에 추천되었고 그 아래 단계인 검의 무공훈장을 받았다.<sup class="reference" id="cite_ref-41"><a href="#cite_note-41">[39]</a></sup> 비트겐슈타인은 1918년 11월 <a href="/wiki/%ED%8A%B8%EB%A0%8C%ED%86%A0" title="트렌토">트렌토</a> 근교에서 이탈리아 군의 포로가 되었다.<sup class="reference" id="cite_ref-Connor-Robertson_30-2"><a href="#cite_note-Connor-Robertson-30">[29]</a></sup></p>
<p>전쟁 기간 비트겐슈타인은 군인의 눈으로 목격한 천박을 대상으로 한 경멸을 노트에 기록하였다. 종전 뒤에도 비트겐슈타인은 자신의 노트에 철학과 종교를 대상으로 한 단상을 적어나갔다. 노트에 적힌 단상을 보면 비트겐슈타인의 종교를 향한 태도에 변화가 있었다는 사실을 알 수 있다. 케임브리지 대학교 시절 비트겐슈타인은 <a href="/wiki/%EC%A2%85%EA%B5%90" title="종교">종교</a>를 대상으로 해 <a href="/wiki/%EB%B6%88%EA%B0%80%EC%A7%80%EB%A1%A0" title="불가지론">불가지론</a>다운 태도를 견지했는데 전쟁 중에 비트겐슈타인은 <a href="/wiki/%EB%A0%88%ED%94%84_%ED%86%A8%EC%8A%A4%ED%86%A0%EC%9D%B4" title="레프 톨스토이">레프 톨스토이</a>의 《믿음 안의 찬송》을 갈리시아 루마니아 왕국에 있던 한 서점에서 구입한 후 어디나 가지고 다니면서 읽었으므로 후일 비트겐슈타인의 지휘에 있던 부대원들은 “찬송과 함께하는 사람들”이란 별명을 얻었으며,<sup class="reference" id="cite_ref-Monk90_42-0"><a href="#cite_note-Monk90-42">[40]</a></sup> 루트비히 요제프 요한 비트겐슈타인은 <a class="mw-redirect" href="/wiki/%EC%84%B1_%EC%95%84%EC%9A%B0%EA%B5%AC%EC%8A%A4%ED%8B%B0%EB%88%84%EC%8A%A4" title="성 아우구스티누스">성 아우구스티누스</a>의 여러 저작에서 종교상 영향받았다.<sup class="reference" id="cite_ref-Creegan_43-0"><a href="#cite_note-Creegan-43">[41]</a></sup></p>
<h4><span class="mw-headline" id=".EB.85.BC.EA.B3.A0">논고</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%A3%A8%ED%8A%B8%EB%B9%84%ED%9E%88_%EB%B9%84%ED%8A%B8%EA%B2%90%EC%8A%88%ED%83%80%EC%9D%B8&amp;action=edit&amp;section=5" title="부분 편집: 논고">편집</a><span class="mw-editsection-bracket">]</span></span></h4>
<div class="thumb tright">
<div class="thumbinner" style="width:222px;"><a class="image" href="/wiki/%ED%8C%8C%EC%9D%BC:Wittgenstein1920.jpg"><img alt="" class="thumbimage" data-file-height="598" data-file-width="641" height="205" src="//upload.wikimedia.org/wikipedia/commons/thumb/b/b2/Wittgenstein1920.jpg/220px-Wittgenstein1920.jpg" srcset="//upload.wikimedia.org/wikipedia/commons/thumb/b/b2/Wittgenstein1920.jpg/330px-Wittgenstein1920.jpg 1.5x, //upload.wikimedia.org/wikipedia/commons/thumb/b/b2/Wittgenstein1920.jpg/440px-Wittgenstein1920.jpg 2x" width="220"/></a>

<div class="magnify"><a class="internal" href="/wiki/%ED%8C%8C%EC%9D%BC:Wittgenstein1920.jpg" title="실제 크기로"></a></div>
1920년 가족 저택에서 촬영한 사진. 비트겐슈타인은 누나인 헤레네와 스웨덴 친구인 아르비드 시요그렌 사이에 앉아있다.</div>
</div>
</div>
<p>윤리, 종교에 딸린 기호를 가지고 "논리"에 부합한 연구를 시작한 비트겐슈타인은 초기에 관심했던 논리 분석을 윤리 개념에 접합하면서 전쟁 기간에 발전하게 한 소위 "그림 이론"을 적용하였다. 여기에 케임브리지 대학교 시절과 노르웨이에서 은둔 생활한 시기에 한 작업이 "논고"의 바탕이 되었다. 1918년 전쟁이 막바지로 치다를 무렵 비트겐슈타인이 이탈리아 북부의 포병 연대에서 중위로 복무할 때 비트겐슈타인은 데이비드 핀센트의 어머니에게서 편지를 한 통 받았다. 거기에는 데이비드가 비행기 사고로 죽었다는 소식이 들어있었다. 비트겐슈타인은 데이비드의 삼촌 폴에게 찾아가 완성된 논고를 핀센트에게 헌정하였다. 논고는 출판회사에 보내졌으나 출판되지 않았다.</p>
<p>1918년 10월 비트겐슈타인은 이탈리아 전선에 복귀하였으나 얼마 지나지 않아 이탈리아군의 포로가 되었다. 이탈리아 중부의 카치노에 수감된 비트겐슈타인은 <a href="/wiki/%EB%B2%84%ED%8A%B8%EB%9F%B0%EB%93%9C_%EB%9F%AC%EC%85%80" title="버트런드 러셀">버트런드 러셀</a>과 키네스 같은 영국 친구의 도움으로 책의 반입이 허락되었고 비트겐슈타인의 여러 원고가 영국으로 보내졌다. 러셀은 비트겐슈타인의 원고가 철학상 뛰어난 성과라는 사실을 간파했고 1919년 비트겐슈타인의 이름으로 출간하였다. <a href="/wiki/%ED%94%84%EB%9E%AD%ED%81%AC_%EB%9E%A8%EC%A7%80" title="프랭크 램지">프랭크 램지</a>와 <a href="/wiki/%EC%B0%B0%EC%8A%A4_%EC%BC%80%EC%9D%B4_%EC%98%A4%EA%B7%B8%EB%8D%98" title="찰스 케이 오그던">찰스 케이 오그던</a>이 비트겐슈타인의 원고를 영역하였고 비트겐슈타인이 영역물을 검토하였다. 책의 제목 《논리 철학 논고》는 <a href="/wiki/%EB%B0%94%EB%A4%BC%ED%9D%90_%EC%8A%A4%ED%94%BC%EB%85%B8%EC%9E%90" title="바뤼흐 스피노자">바뤼흐 스피노자</a>의 《정치학 논고》를 기려 <a class="mw-redirect" href="/wiki/%EC%A1%B0%EC%A7%80_%EC%97%90%EB%93%9C%EC%9B%8C%EB%93%9C_%EB%AC%B4%EC%96%B4" title="조지 에드워드 무어">조지 에드워드 무어</a>가 지은 것이다. 서문은 버트런드 러셀이 썼다.<sup class="reference" id="cite_ref-44"><a href="#cite_note-44">[42]</a></sup> 《<a class="mw-redirect" href="/wiki/%EB%85%BC%EB%A6%AC_%EC%B2%A0%ED%95%99_%EB%85%BC%EA%B3%A0" title="논리 철학 논고">논리 철학 논고</a>》는 중요한 철학 서적 가운데 하나로 꼽힌다. 비트겐슈타인은 전쟁포로로서 수감된 가운데 버트런드 러셀의 서문을 받아 보았다. 비트겐슈타인은 논고의 기본 사항을 러셀이 오해한다고 판단하고, 자신의 작업 출판을 대상으로 한 흥미를 완전히 잃고 말았다. 설상가상으로 출판사는 서문을 쓴 러셀만을 부각하게 할 뿐이었다. 논고는 1921년 <a href="/wiki/%EB%B9%8C%ED%97%AC%EB%A6%84_%EC%98%A4%EC%8A%A4%ED%8A%B8%EB%B0%9C%ED%8A%B8" title="빌헬름 오스트발트">빌헬름 오스트발트</a>가 운영하던 독일의 출판사인 《자연철학 분석(<span style="font-size: smaller;"><a href="/wiki/%EB%8F%85%EC%9D%BC%EC%96%B4" title="독일어">독일어</a>:</span> <span lang="de" xml:lang="de">Annalen der Naturphilosophie</span>)》에서 독일어로서 마침내 출간되었다. 러셀의 서문과 램지와 오그던이 영역한 판본은 1922년에 출간되었다.</p>
<h3><span class="mw-headline" id="1920.EB.85.84.EB.8C.80:_.EC.98.A4.EC.8A.A4.ED.8A.B8.EB.A6.AC.EC.95.84_.EA.B5.90.EC.A7.81_.EC.83.9D.ED.99.9C">1920년대: 오스트리아 교직 생활</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%A3%A8%ED%8A%B8%EB%B9%84%ED%9E%88_%EB%B9%84%ED%8A%B8%EA%B2%90%EC%8A%88%ED%83%80%EC%9D%B8&amp;action=edit&amp;section=6" title="부분 편집: 1920년대: 오스트리아 교직 생활">편집</a><span class="mw-editsection-bracket">]</span></span></h3>

<p>1920년이 되자 비트겐슈타인은 다른 사람이 되기를 간절히 원했다. 끔찍한 전투를 경험한 제1차 세계대전 이후 견고해진 비트겐슈타인의 지성과 감성은 "논고"로 집약되었다. 비트겐슈타인은 이 작업으로 철학의 모든 문제에 대답을 구했다고 생각했기에 철학을 떠나 <a href="/wiki/%EC%98%A4%EC%8A%A4%ED%8A%B8%EB%A6%AC%EC%95%84" title="오스트리아">오스트리아</a>의 초등학교에서 교직 생활을 시작하였다. 그는 톨스토이의 《요약복음서》를 읽고 깊은 감명을 받아 아버지에게서 물려받은 막대한 유산을 형제들에게 나눠주고는 단순하고 소박한 삶을 살기 위해 하스바흐, 오테르탈, 트라텐바흐와 같은 곳의 산골 초등학교를 돌면서 학생들을 가르쳤다.<sup class="reference" id="cite_ref-Jeffries_45-0"><a href="#cite_note-Jeffries-45">[43]</a></sup> 그러나 보수적인 농촌 학부모들 및 동료 교사들과 불화를 일으켜 교사생활은 순탄치 못했다.<sup class="reference" id="cite_ref-46"><a href="#cite_note-46">[44]</a></sup></p>

<p>논고를 번역한 <a href="/wiki/%ED%94%84%EB%9E%AD%ED%81%AC_%EB%9E%A8%EC%A7%80" title="프랭크 램지">프랭크 램지</a>가 1923년 가을 비트겐슈타인의 편지에 적힌 주소를 찾아 갔을 때 비트겐슈타인은 매우 검소하게 생활하고 있었다. 좁다란 텅 빈 방에 침대가 하나 덩그러니 놓여 있었고 작은 테이블과 난로가 고작이었다. 프랭크 램지는 저녁 식사로 질긴 빵 하나와 버터와 코코아 한 잔을 건네받았다. 비트겐슈타인은 아침 여덟 시에서 정오나 오후 한 시까지 수업하였고 오후 시간은 자유롭게 보냈다.<sup class="reference" id="cite_ref-48"><a href="#cite_note-48">[46]</a></sup></p>
<p>비트겐슈타인은 교직 생활하면서 아이들을 이롭게 하려고 42쪽으로 된 철자와 발음이 표기된 사전을 만들었다. 《어린이를 위한 사전》(<span style="font-size: smaller;"><a href="/wiki/%EB%8F%85%EC%9D%BC%EC%96%B4" title="독일어">독일어</a>:</span> <span lang="de" xml:lang="de">Wörterbuch für Volksschulen</span>)은 《논고》를 제외하면 비트겐슈타인 생전에 그의 이름으로 출판된 유일한 책이다. 이 사전은 1926년 빈에서 출판되었고 횔데르 피흘러 템프스키의 간단한 서문이 들어 있다.<sup class="reference" id="cite_ref-archiveTractatus_49-0"><a href="#cite_note-archiveTractatus-49">[47]</a></sup> 이 사전의 초판은 2005년 2월 경매에서 75,000 <a href="/wiki/%ED%8C%8C%EC%9A%B4%EB%93%9C_(%ED%86%B5%ED%99%94)" title="파운드 (통화)">£</a>에 판매되었다.<sup class="reference" id="cite_ref-50"><a href="#cite_note-50">[48]</a></sup> 교직 생활하면서 체벌 문제에 휩싸인 비트겐슈타인은 1926년 4월 11살 소년을 심하게 때렸다는 이유로 고발당하였고 결국 교직 생활을 그만두게 되었다. 교직을 그만 둔 후 비트겐슈타인은 빈에서 잠시 머문 후 <a class="new" href="/w/index.php?title=%ED%9C%98%ED%85%94%EB%8F%84%EB%A5%B4%ED%94%84&amp;action=edit&amp;redlink=1" title="휘텔도르프 (없는 문서)">휘텔도르프</a> 자비의 형제 수도회에서 정원사로서 일하였다.<sup class="reference" id="cite_ref-Jeffries_45-1"><a href="#cite_note-Jeffries-45">[43]</a></sup></p>
<h3><span class="mw-headline" id="1926.EB.85.84:_.EC.8A.A4.ED.86.A4.EB.B3.B4.EB.A1.9C.EC.9A.B0_.ED.95.98.EC.9A.B0.EC.8A.A4">1926년: 스톤보로우 하우스</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%A3%A8%ED%8A%B8%EB%B9%84%ED%9E%88_%EB%B9%84%ED%8A%B8%EA%B2%90%EC%8A%88%ED%83%80%EC%9D%B8&amp;action=edit&amp;section=7" title="부분 편집: 1926년: 스톤보로우 하우스">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<div class="thumb tleft">

비트겐슈타인이 설계한 <a class="new" href="/w/index.php?title=%EC%8A%A4%ED%86%A4%EB%B3%B4%EB%A1%9C%EC%9A%B0_%ED%95%98%EC%9A%B0%EC%8A%A4&amp;action=edit&amp;redlink=1" title="스톤보로우 하우스 (없는 문서)">스톤보로우 하우스</a></div>

<td style="text-align: left;">나는 수직으로 선 건물에 끌리지 않지만, …… 이것은 건축의 모든 가능성을 대상으로 한 기반을 내 스스로 보이는 것이다.</td>

<p>1926년 비트겐슈타인은 누나인 그레틀을 위해 새 집을 구상하였다. 건축은 전쟁 중에 친구가 된 <a class="new" href="/w/index.php?title=%ED%8C%8C%EC%9A%B8_%EC%97%A5%EA%B2%94%EB%A7%8C&amp;action=edit&amp;redlink=1" title="파울 엥겔만 (없는 문서)">파울 엥겔만</a>이 진행하였다. 비트겐슈타인과 파울 엥겔만은 참호에서 많은 시간을 함께 보냈었다. 파울 엥겔만은 새 집이 <a class="new" href="/w/index.php?title=%EC%95%84%EB%8F%8C%ED%94%84_%EB%A3%A8%EC%8A%A4&amp;action=edit&amp;redlink=1" title="아돌프 루스 (없는 문서)">아돌프 루스</a> 이후 현대 건축을 나타내는 공간으로 자리잡기를 바랐다. 비트겐슈타인은 이 집을 짓는 일에 온통 매달렸고 창틀, 문, 난방기 같은 것까지 일일이 살펴보았다. 문 손잡이와 난방기는 직접 설계하기도 하였다. 비트겐슈타인의 요구로 모든 창문에 150Kg에 이르는 금속 차양막을 달았다. 《비트겐슈타인의 건축》을 쓴 버나드 라이트너는 이 건축물이 건축사에서 다른 예를 찾기 힘든 독특한 것이라 평가하면서 "이 집은 비트겐슈타인의 의도로 매우 정교하게 지어졌다. 모든 철제는 바닥에 숨겨져 있도록 하였다"고 기술하였다. 집이 거의 완공되었으나 비트겐슈타인은 천정의 높이를 30mm 더 높이라고 하였고, 그가 원한대로 천정을 높이고서야 마침내 완공되었다.<sup class="reference" id="cite_ref-Jeffries_45-2"><a href="#cite_note-Jeffries-45">[43]</a></sup></p>
<p>비트겐슈타인의 누나 헤르미네는 이 집을 두고 "나는 언제나 내가 그 집 안에서 살아도 되는 걸까 자문하곤 하였다. 그 집은 신의 거처처럼 만들어져 있었다"라고 기록하였다. 그 집을 좋게 말하면 너무 소박하게 지었다고 생각한 비트겐슈타인은 이 집이 건축의 기본이 되는 생활이나 건강에 대한 고려가 없었다고 여겼다.<sup class="reference" id="cite_ref-Lewis_51-1"><a href="#cite_note-Lewis-51">[49]</a></sup> <a href="/wiki/%EC%A0%9C2%EC%B0%A8_%EC%84%B8%EA%B3%84_%EB%8C%80%EC%A0%84" title="제2차 세계 대전">세계대전</a>이 끝난 후 이 집은 러시아 군인들의 병영으로 사용되었다. 1950년대 그레틀의 아들이 개발업자에게 집을 팔아 없어질 위기에 놓였으나 빈 랜드마크 협회가 1971년 이 집을 구매해 보존한다. 이 집은 현재 불가리아 대사관으로 사용된다.<sup class="reference" id="cite_ref-Jeffries_45-3"><a href="#cite_note-Jeffries-45">[43]</a></sup></p>
<h3><span class="mw-headline" id=".EB.B9.88_.ED.95.99.ED.8C.8C">빈 학파</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%A3%A8%ED%8A%B8%EB%B9%84%ED%9E%88_%EB%B9%84%ED%8A%B8%EA%B2%90%EC%8A%88%ED%83%80%EC%9D%B8&amp;action=edit&amp;section=8" title="부분 편집: 빈 학파">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>비트겐슈타인은 건축 작업을 마치고 <a href="/wiki/%EB%AA%A8%EB%A6%AC%EC%B8%A0_%EC%8A%90%EB%A6%AC%ED%81%AC" title="모리츠 슐리크">모리츠 슐리크</a>와 만났다. 슐리크는 빈 학파를 이끌고 있었다. 비트겐슈타인의 "논고"는 빈 학파의 <a href="/wiki/%EB%85%BC%EB%A6%AC%EC%8B%A4%EC%A6%9D%EC%A3%BC%EC%9D%98" title="논리실증주의">논리실증주의</a> 발전에 큰 영향을 주었다. 슐리크 자신은 빈 학파의 논의에서 비트겐슈타인의 사상을 계승하였다고 한 적이 없으나 <a class="new" href="/w/index.php?title=%ED%94%84%EB%A6%AC%EB%93%9C%EB%A6%AC%ED%9E%88_%EB%B0%94%EC%9D%B4%EC%8A%A4%EB%A7%8C&amp;action=edit&amp;redlink=1" title="프리드리히 바이스만 (없는 문서)">프리드리히 바이스만</a>과 같은 빈 학파의 학자들은 비트겐슈타인과 철학에 기초한 주제를 화제로 하여 자주 대화했다.<sup class="reference" id="cite_ref-52"><a href="#cite_note-52">[50]</a></sup> 슐리크와 빈 학파의 회원들이 자신의 《논고》를 오해한다고 생각하여 자주 좌절한 비트겐슈타인은 종교에 딸린 생활과 신비에는 말로는 설명할 수 없는 신념이 있다고 생각하였고 논리실증주의는 이러한 것을 쓸모 없는 것으로 비하한다고 간주했다. 한 모임에서 비트겐슈타인은 <a href="/wiki/%EB%9D%BC%EB%B9%88%EB%93%9C%EB%9D%BC%EB%82%98%ED%8A%B8_%ED%83%80%EA%B3%A0%EB%A5%B4" title="라빈드라나트 타고르">라빈드라나트 타고르</a>의 시를 돌아가면서 낭송하는 사이 자신의 책 언급을 거부하여 좌중을 짜증나게 하였다. 이런 일이 있었어도 비트겐슈타인은 모임에 참석하였으며 철학에 다시 관심을 가졌다. 모임이 이어지는 동안 프랭크 램지가 《논고》와 관련해 토론하려고 빈을 방문하기도 하였다. 비트겐슈타인은 이 토론을 통해 프랭크 램지가 자신의 《논고》를 오해한다고 다시 한 번 생각하게 되었다..</p>
<h3><span class="mw-headline" id=".EC.BC.80.EC.9E.84.EB.B8.8C.EB.A6.AC.EC.A7.80_.EB.8C.80.ED.95.99.EA.B5.90.EC.97.90_.EB.B3.B5.EA.B7.80.ED.95.98.EB.8B.A4">케임브리지 대학교에 복귀하다</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%A3%A8%ED%8A%B8%EB%B9%84%ED%9E%88_%EB%B9%84%ED%8A%B8%EA%B2%90%EC%8A%88%ED%83%80%EC%9D%B8&amp;action=edit&amp;section=9" title="부분 편집: 케임브리지 대학교에 복귀하다">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>1929년 비트겐슈타인은 프랭크 램지와 다른 사람들의 간곡한 권유로 케임브리지 대학교에 복귀하였다. 비트겐슈타인이 도착한 기차역에는 잉글랜드의 지식인들이 세계에서 가장 유명한 철학자를 보려고 몰려들었다.</p>

<p>이런 환대에도 비트겐슈타인은 케임브리지 대학교의 학위를 받지 못했다. 버트런드 러셀은 비트겐슈타인이 《논고》를 집필하여 이미 철학 박사의 충분한 자격이 있다고 기록하였고 무어와 함께 박사학위 심사를 진행하였다. 비트겐슈타인은 심사관이 된 두 오랜 친구의 어깨에 손을 얻으며 말했다. "너무 걱정하지 마시죠. (논고를) 절대로 이해 못한다는 거 알고 있어요."<sup class="reference" id="cite_ref-53"><a href="#cite_note-53">[51]</a></sup></p>

<p>내 스스로 이 천재다운 연구를 검토하였고 내가 이것을 완전히 이해하지 못한다고 하더라도 논고가 철학 박사학위를 받기에 충분하다는 점은 명백하다.</p>

<p>박사학위를 받고 <a class="mw-redirect mw-disambig" href="/wiki/%ED%8A%B8%EB%A6%AC%EB%8B%88%ED%8B%B0_%EC%BB%AC%EB%A6%AC%EC%A7%80" title="트리니티 컬리지">트리니티 컬리지</a>에 부임한 비트겐슈타인은 친구의 집에서 만나게 된 <a class="new" href="/w/index.php?title=%EB%A7%88%EB%A5%B4%EA%B2%8C%EB%A6%AC%ED%85%8C_%EB%A0%88%EC%8A%A4%ED%95%91%EA%B1%B0&amp;action=edit&amp;redlink=1" title="마르게리테 레스핑거 (없는 문서)">마르게리테 레스핑거</a>와 혼인을 고려하였으나 1931년 파혼한 후 독신으로 지냈다. 비트겐슈타인은 <a href="/wiki/%EB%8F%99%EC%84%B1%EC%95%A0" title="동성애">동성애</a> 성향이 있었고 빈의 <a class="new" href="/w/index.php?title=%ED%94%84%EB%9D%BC%ED%85%8C%EB%A5%B4&amp;action=edit&amp;redlink=1" title="프라테르 (없는 문서)">프라테르</a> 공원에서 젊은 남자들과 관계를 맺었다. <a class="new" href="/w/index.php?title=%EC%9C%8C%EB%A6%AC%EC%97%84_%EC%9B%8C%EB%A0%8C_%EB%B0%94%ED%8B%80%EB%A6%AC&amp;action=edit&amp;redlink=1" title="윌리엄 워렌 바틀리 (없는 문서)">윌리엄 워렌 바틀리</a>는 비트겐슈타인의 전기를 쓰면서 1973년 발견된 비트겐슈타인의 노트에는 프라테르에서의 만남을 "친구의 격려"와 같이 표기하였다고 밝혔다.<sup class="reference" id="cite_ref-55"><a href="#cite_note-55">[53]</a></sup> 뒷날 바틀리의 주장은 여러 면에서 의심받았으나 비트겐슈타인이 <a class="new" href="/w/index.php?title=%EB%8D%B0%EC%9D%B4%EB%B9%84%EB%93%9C_%ED%95%80%EC%84%BC%ED%8A%B8&amp;action=edit&amp;redlink=1" title="데이비드 핀센트 (없는 문서)">데이비드 핀센트</a>, <a class="new" href="/w/index.php?title=%ED%94%84%EB%9E%9C%EC%8B%9C%EC%8A%A4_%EC%8A%A4%ED%82%A4%EB%84%88&amp;action=edit&amp;redlink=1" title="프랜시스 스키너 (없는 문서)">프랜시스 스키너</a>, <a class="new" href="/w/index.php?title=%EB%B2%A4_%EB%A6%AC%EC%B2%98%EB%93%9C&amp;action=edit&amp;redlink=1" title="벤 리처드 (없는 문서)">벤 리처드</a>와 같은 사람들과 동성애 관계에 있었다는 풍문은 거의 정설이 되었다.<sup class="reference" id="cite_ref-56"><a href="#cite_note-56">[54]</a></sup></p>

<p>일각에서는 비트겐슈타인이 정치적으로 <a class="mw-redirect" href="/wiki/%EC%A2%8C%ED%8C%8C" title="좌파">좌파적</a> 입장을 지녔다고 평가한다. 비트겐슈타인 스스로가 자신이 "심장에서부터 공산주의자"라고 말한 적이 있고 많은 방면에서 노동자에 대한 낭만주의적인 반응을 보이기도 하였다.<sup class="reference" id="cite_ref-58"><a href="#cite_note-58">[56]</a></sup> 비트겐슈타인은 도덕적 성찰이 없다는 이유로 과학적 방법을 혐오하였으며, 음악취향은 보수적이었고, 핵무기의 사용에 반대하였다.<sup class="reference" id="cite_ref-59"><a href="#cite_note-59">[57]</a></sup> 오스트리아 철학자 <a href="/wiki/%EC%98%A4%ED%86%A0_%EB%B0%94%EC%9D%B4%EB%8B%9D%EA%B1%B0" title="오토 바이닝거">오토 바이닝거</a>를 존경한 비트겐슈타인은 바이닝거의 이론을 복사하여 케임브리지 대학교에 배포하기도 하였다.<sup class="reference" id="cite_ref-60"><a href="#cite_note-60">[58]</a></sup> 오토 바이닝거처럼 비트겐슈타인도 민족성과 성 정체성으로 말미암아 대인관계에 문제가 있었다.<sup class="reference" id="cite_ref-61"><a href="#cite_note-61">[59]</a></sup> 1930년대 초반 기록된 MS 154와 같은 노트에는 자신의 정체성으로 말미암아 자책하는 여러 문구가 적혀 있다.</p>
<blockquote class="toccolours" style="float:none; padding: 10px 15px 10px 15px; display:table;">
<p>성자는 단지 <a href="/wiki/%EC%9C%A0%EB%8C%80%EC%9D%B8" title="유대인">유대인</a> 천재일 뿐이다. 가장 뛰어난 유대인 사상가라 할지라도 더는 재능이 없다. (내 자신을 대상으로 해)<sup class="reference" id="cite_ref-62"><a href="#cite_note-62">[60]</a></sup></p>
</blockquote>
<p>후일 비트겐슈타인은 "내 생각은 100% 헤브라이즘답다"라고 스스로 비판하였다.<sup class="reference" id="cite_ref-63"><a href="#cite_note-63">[61]</a></sup></p>
<blockquote class="toccolours" style="float:none; padding: 10px 15px 10px 15px; display:table;">
<p>비트겐슈타인이 자신이 유대인이라는 것에 대한 자기부정은 스스로를 파멸로 몰았다. 그러나 유대인이라는 것 때문에 그는 자신의 참신함과 천재성을 확신하였다.[<sup class="reference" id="cite_ref-64"><a href="#cite_note-64">[62]</a></sup></p>
<p style="text-align: right;">— <cite><a class="new" href="/w/index.php?title=%ED%95%9C%EC%8A%A4_%EC%8A%AC%EB%A3%A8%EA%B1%B0&amp;action=edit&amp;redlink=1" title="한스 슬루거 (없는 문서)">한스 슬루거</a></cite></p>
</blockquote>
<p>1934년 비트겐슈타인은 친구인 케인스가 쓴 《러시아에 대한 간략한 소개》에서 서술된 소비에트 생활에 매료되어 스키너와 <a href="/wiki/%EC%86%8C%EB%A0%A8" title="소련">소비에트사회주의공화국련방</a>으로 이민을 고려하였다. 그들은 러시아어를 배웠고 1935년에는 비트겐슈타인이 안전 요원이 지켜보는 가운데 <a class="mw-redirect" href="/wiki/%EB%A0%88%EB%8B%8C%EA%B7%B8%EB%9D%BC%EB%93%9C" title="레닌그라드">레닌그라드</a>와 <a class="mw-redirect" href="/wiki/%EB%AA%A8%EC%8A%A4%EC%BD%94%EB%B0%94" title="모스코바">모스코바</a>를 여행하였다. 비트겐슈타인은 러시아 당국에 교사 자리가 있는지 문의하였으나 이에 대답은 3주가 지나서야 돌아왔다.</p>
<p>1936년부터 37년 사이에 비트겐슈타인은 스키너와 함께 노르웨이에 다시 머물렀다.<sup class="reference" id="cite_ref-65"><a href="#cite_note-65">[63]</a></sup> 비트겐슈타인은 노르웨이에서 연구를 계속하였고 이것을 철학적 “조사”라고 불렀다. 1936년과 37년 사이의 겨울에 가까운 친구에게 보내는 “고백”을 썼다. 이 글들은 대부분 선의의 거짓말과 같은 것을 밝히는 소소한 내용이었으나 비트겐슈타인은 이를 통하여 자신을 정화하고자 하였다. 1938년 비트겐슈타인은 모리스 드루어리를 만나기 위해 아일랜드를 여행하였다. 드루어리는 비트겐슈타인에게 정신 의학을 소개한 제자이자 의사였다. 드루어리를 만난 자리에서 비트겐슈타인은 마침 드루어리를 찾아온 아일랜드 대통령 이몬 데 발레라를 만났다. 자신도 수학 교사 출신이었던 발레라는 비트겐슈타인의 업적이 수학의 발전에 기여할 수 있기를 희망하였다.</p>
<h3><span class="mw-headline" id=".EB.82.98.EC.B9.98.EC.9D.98_.EB.B6.80.EC.83.81.EA.B3.BC_.EA.B0.80.EC.A1.B1.EC.9D.98_.EC.9C.84.ED.97.98">나치의 부상과 가족의 위험</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%A3%A8%ED%8A%B8%EB%B9%84%ED%9E%88_%EB%B9%84%ED%8A%B8%EA%B2%90%EC%8A%88%ED%83%80%EC%9D%B8&amp;action=edit&amp;section=10" title="부분 편집: 나치의 부상과 가족의 위험">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>비트겐슈타인은 아일랜드에 있는 동안 <a href="/wiki/%EB%82%98%EC%B9%98_%EB%8F%85%EC%9D%BC" title="나치 독일">나치 독일</a>이 <a href="/wiki/%EC%98%A4%EC%8A%A4%ED%8A%B8%EB%A6%AC%EC%95%84" title="오스트리아">오스트리아</a>를 <a href="/wiki/%EC%98%A4%EC%8A%A4%ED%8A%B8%EB%A6%AC%EC%95%84_%EB%B3%91%ED%95%A9" title="오스트리아 병합">병합</a>하였다는 소식을 들었다. 이로 말미암아 비트겐슈타인의 가족은 독일 국적을 갖게 되었고 나치의 인종법에 따라 유대인으로 취급되게 되었다. 비트겐슈타인은 이러한 상황을 견딜 수 없었고 케인스와 같은 친구들의 힘을 빌어 영국이나 아일랜드 국적을 획득하고자 하였다. 그러나, 그의 형제자매들은 모두 오스트리아에 남아있었기 때문에 이들의 처지가 매우 위험하게 되었다. 비트겐슈타인은 처음에는 곧바로 오스트리아로 가고자 하였으나 친구들의 만류로 그만두었다.<sup class="reference" id="cite_ref-66"><a href="#cite_note-66">[64]</a></sup> 왼손 피아니스트로 명성을 떨치던 파울은 공연과 교습을 금지당하자 스위스를 거쳐 미국으로 탈출하였고, 여자형제들은 위조여권으로 탈출하려다가 발각되어 재판을 받았다. 유일한 희망은 나치 독일의 혼혈법에 따라 비트겐슈타인의 가족이 유대인 조상이 있는 혼혈로 인정받는 것이었다. 비트겐슈타인 집안 사람들은 조부인 헤르만이 독일 귀족 자인-비트겐슈타인 가문의 사생아였다는 증언을 모아 제출하였으나 거부당했다. 그러나 비트겐슈타인 가문의 해외 자산에 눈독을 들인 나치 당국과 재산의 상당 부분을 헌납하는 힘겨운 협상 끝에 비트겐슈타인의 가족들은 혼혈로 인정되어 전쟁이 끝날 때까지 살아남을 수 있었다. 이 긴박한 협상 과정은 알렉산더 워의 《비트겐슈타인 가문》에 상세히 묘사되어 있는데, 재산을 지키려는 파울과 목숨을 지키려는 누이들 사이의 갈등이 극에 달해 이후로 파울은 죽을 때까지 형제자매들과 다시는 얼굴을 보려 하지 않았고 연락도 하지 않았다. 폴란드 침공 하루 전날 비트겐슈타인 가문을 혼혈로 승인하는 문서에 직접 사인을 한 아돌프 히틀러가 언급한 바에 따르면 혼혈로 인정받기를 원하는 2,100 건의 신청가운데 인정된 건 수는 12건에 불과할 만큼 이러한 조치는 예외적인 것이었다.<sup class="reference" id="cite_ref-DavidE_67-0"><a href="#cite_note-DavidE-67">[65]</a></sup></p>
<h3><span class="mw-headline" id=".EC.B2.A0.ED.95.99_.EA.B5.90.EC.88.98.EA.B0.80_.EB.90.98.EB.8B.A4">철학 교수가 되다</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%A3%A8%ED%8A%B8%EB%B9%84%ED%9E%88_%EB%B9%84%ED%8A%B8%EA%B2%90%EC%8A%88%ED%83%80%EC%9D%B8&amp;action=edit&amp;section=11" title="부분 편집: 철학 교수가 되다">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>1939년 무어가 케임브리지 대학교의 철학 교수직을 사임하자 비트겐슈타인이 후임자로 임명되었으며, 교수직을 갖게 된 얼마 후 영국 시민권을 획득할 수 있었다. 1939년 7월 비트겐슈타인은 빈을 방문하여 그레틀과 다른 누나들을 만났고, 하루는 라이슈방크의 직원을 만나기 위해 베를린에 들렀다. 이후, 그는 가족이 처한 위험을 모면하기 위해 계획을 짜고 있던 형 파울을 설득하기 위해 뉴욕으로 갔다. 1939년 8월 가족에 대한 해방 요청이 접수되었는데, 전쟁이 시작되기 1주일 전에 알려지지 않은 사람의 승인으로 비트겐슈타인 가족의 재산 금 1.7톤이 나치에게 인도되었다.<sup class="reference" id="cite_ref-68"><a href="#cite_note-68">[66]</a></sup> 이는 2009년 가치로 환산하면 약 60억 달러에 해당하는 금액이다. 또한 비트겐슈타인은 같은 해에 모스크바를 두 번째로 방문하여 철학자인 소피야 야노프스카야를 만났다.<sup class="reference" id="cite_ref-69"><a href="#cite_note-69">[67]</a></sup></p>
<p>일을 마무리 지은 다음 비트겐슈타인은 영화관 제일 앞자리에서 영화를 보는가하면 추리소설을 읽으면서 편안히 서방세계를 관망하였다.<sup class="reference" id="cite_ref-70"><a href="#cite_note-70">[68]</a></sup> <a class="new" href="/w/index.php?title=%EB%85%B8%EB%A8%BC_%EB%A7%90%EC%BD%A4&amp;action=edit&amp;redlink=1" title="노먼 말콤 (없는 문서)">노먼 말콤</a>은 비트겐슈타인이 강의를 마치자마자 영화관으로 달려가곤 했다고 회고하였다.</p>
<blockquote class="toccolours" style="float:none; padding: 10px 15px 10px 15px; display:table;">
<p>강의를 마친 학생들이 교실을 나가려고 의자에서 몸을 빼는 순간 비트겐슈타인은 친구를 향해 "갈 수 있겠어?"하고 낮은 목소리로 묻곤 하였다. 영화관에 갈 때면 비트겐슈타인은 빵이나 파이 따위를 사들고서는 영화를 보면서 먹었다.<sup class="reference" id="cite_ref-71"><a href="#cite_note-71">[69]</a></sup></p>
<p style="text-align: right;">— <cite><a class="new" href="/w/index.php?title=%EB%85%B8%EB%A8%BC_%EB%A7%90%EC%BD%A4&amp;action=edit&amp;redlink=1" title="노먼 말콤 (없는 문서)">노먼 말콤</a></cite></p>
</blockquote>
<p>이 무렵 비트겐슈타인은 <a href="/wiki/%EC%88%98%ED%95%99%EA%B8%B0%EC%B4%88%EB%A1%A0" title="수학기초론">수학기초론</a>에 대한 자신의 기존 견해에 회의적인 태도를 보였다. 비트겐슈타인의 초기 사상에서 논리는 단단한 기호였으며, 그는 러셀과 화이트헤드의 《수학 개론》에 대해서도 이러한 사고를 기초로 비평하였다. 하지만 이 무렵에 이르러서 비트겐슈타인은 자신의 글과 강의를 통해서 과거 자신의 사상을 부정하였다.<sup class="reference" id="cite_ref-72"><a href="#cite_note-72">[70]</a></sup></p>
<h3><span class="mw-headline" id="2.EC.B0.A8_.EC.84.B8.EA.B3.84.EB.8C.80.EC.A0.84">2차 세계대전</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%A3%A8%ED%8A%B8%EB%B9%84%ED%9E%88_%EB%B9%84%ED%8A%B8%EA%B2%90%EC%8A%88%ED%83%80%EC%9D%B8&amp;action=edit&amp;section=12" title="부분 편집: 2차 세계대전">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>몽크의 회고에 따르면 비트겐슈타인은 전쟁이 시작되었는데도 철학 강의나 하고 있는 스스로의 처지를 견디기 힘들어하였다고 한다. 1941년 9월 비트겐슈타인은 철학자 길버트 라일의 형제였던 존 라일 교수에게 요청하여 런던에 있는 가이스 병원에서 봉사하겠노라고 요청하였다. 존 라일은 케임브리지 대학교의 의학 교수였다. 그는 비트겐슈타인의 요청을 받아들였고 이에 따라 비트겐슈타인은 가이스 병원에서 약품을 관리하는 일을 하였다.<sup class="reference" id="cite_ref-Monk431_73-0"><a href="#cite_note-Monk431-73">[71]</a></sup></p>
<p>병원의 직원들은 세상에서 가장 유명한 철학자가 자신들과 함께 일하고 있다는 사실을 알지 못하였다. 간혹 비트겐슈타인을 알아보는 사람들에게 비트겐슈타인은 "아무에게도 말하지 말아주세요."하고 부탁하였고, 자신을 비트겐슈타인 교수님이라 부르지 말고 여느 의사들과 같이 박사님이라 불러주기를 바랐다.<sup class="reference" id="cite_ref-Monk431_73-1"><a href="#cite_note-Monk431-73">[71]</a></sup></p>
<p>이 무렵 비트겐슈타인은 몹시 외로웠다.</p>
<blockquote class="toccolours" style="float:none; padding: 10px 15px 10px 15px; display:table;">
<p>내 생활의 미래를 대상으로 해 나는 더는 어떤 희망도 없다. 살아있는 사망이 길게 늘어진 것과 같은 이런 것은 예전에는 없었다. 최악 상황 말고는 내 미래를 상상할 수 없다. 친구도 없고 기쁨도 없고.</p>
<p style="text-align: right;">— <cite>비트겐슈타인, <i>1942년 4월 1일</i></cite></p>
</blockquote>
<p>비트겐슈타인은 <a class="new" href="/w/index.php?title=%ED%94%84%EB%9E%80%EC%8B%9C%EC%8A%A4_%EC%8A%A4%ED%82%A4%EB%84%88&amp;action=edit&amp;redlink=1" title="프란시스 스키너 (없는 문서)">프란시스 스키너</a>의 연구 수업에 참여한 십대였던 <a class="new" href="/w/index.php?title=%EC%BC%80%EC%9D%B4%EC%8A%A4_%EC%BB%A4%ED%81%AC&amp;action=edit&amp;redlink=1" title="케이스 커크 (없는 문서)">케이스 커크</a>와 우정을 발전시켜 나갔다. 이 관계는 스키너가 죽은 1941년까지 지속되었다. 스키너는 비트겐슈타인과 커크의 관계에 대해 별다른 관심을 두지 않았다. 그는 자신의 학문연구에 비트겐슈타인이 영향을 미치는 것이 최소화된 점을 고마워했다. 비트겐슈타인은 커크에게 우정 이상의 것을 표현하지 않았지만 스스로는 사랑에 빠져 있었다. 이 기간동안 비트겐슈타인은 일기에 종종 커크에 대한 자신의 감정을 적곤 하였다.</p>
<blockquote class="toccolours" style="float:none; padding: 10px 15px 10px 15px; display:table;">
<p>열흘 동안이나 K에게서 아무것도 듣지 못했다. 일주일 전에 그 사람을 언급했는데도. 아무래도 그 사람이 날 찬 듯하다. 비참한 생각.</p>
</blockquote>
<p>비트겐슈타인이 이 편지를 쓸 무렵 커크는 혼인하였고 둘은 다시는 만나지 않았다.<sup class="reference" id="cite_ref-74"><a href="#cite_note-74">[72]</a></sup></p>
<h3><span class="mw-headline" id=".EB.A7.8C.EB.85.84.EA.B8.B0">만년기</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%A3%A8%ED%8A%B8%EB%B9%84%ED%9E%88_%EB%B9%84%ED%8A%B8%EA%B2%90%EC%8A%88%ED%83%80%EC%9D%B8&amp;action=edit&amp;section=13" title="부분 편집: 만년기">편집</a><span class="mw-editsection-bracket">]</span></span></h3>

<td style="text-align: left;">죽음은 삶에서 일어나는 그저 그런 일이 아니다. 우리는 죽음을 경험하지 못한다. 만일 우리가 영원히 계속되는 시간이 아니라 시간이 존재하지 않는 곳에 들어선다면 삶은 현재를 살아가는 사람들에게만 속한 것이다. 제한없이 펼쳐진 이 길에서 우리의 삶에 끝은 없다.</td>

<p>1947년 비트겐슈타인은 교수직을 사임하고 집필에 전념하였다. 그의 교수직은 친구인 <a class="new" href="/w/index.php?title=%EA%B2%8C%EC%98%A4%EB%A5%B4%EA%B7%B8_%ED%97%A8%EB%A6%AC%ED%81%AC_%ED%8F%B0_%EB%B8%8C%EB%A6%AC%ED%9D%90%ED%8A%B8&amp;action=edit&amp;redlink=1" title="게오르그 헨리크 폰 브리흐트 (없는 문서)">게오르그 헨리크 폰 브리흐트</a>가 물려받았다. 1947년에서 48년까지 비트겐슈타인은 <a href="/wiki/%EB%8D%94%EB%B8%94%EB%A6%B0" title="더블린">더블린</a>의 유명한 관광명소인 로즈 호텔에서 머물렀다.<sup class="reference" id="cite_ref-archivefinalyears_76-0"><a href="#cite_note-archivefinalyears-76">[74]</a></sup></p>
<p>1949년 비트겐슈타인의 학생이었던 <a class="new" href="/w/index.php?title=%EB%85%B8%EB%A8%BC_%EB%A7%90%EC%BD%A4&amp;action=edit&amp;redlink=1" title="노먼 말콤 (없는 문서)">노먼 말콤</a>이 더블린으로 찾아왔을 때 비트겐슈타인은 병들어 있었다. 말콤은 보다 좋은 환경에서 요양하는 것이 좋겠다고 설득하여 비트겐슈타인을 미국으로 데려갔으나 그의 증세는 악화되었다. 비트겐슈타인은 말콤에게 "난 미국에서 죽고 싶지 않아. 난 유럽인이라고. 내가 있던 곳으로 돌려보내 주게"라고 요청하였다.<sup class="reference" id="cite_ref-Waugh273_77-0"><a href="#cite_note-Waugh273-77">[75]</a></sup></p>
<p>비트겐슈타인은 런던으로 돌아왔으며 암이 골수에까지 옮게 되었다. 비트겐슈타인은 1950년 2월 11일 빈을 방문하여 죽음이 임박한 누나 헤르미네를 만났다. 그녀는 비트겐슈타인을 간신히 알아보는 처지였다. 비트겐슈타인은 "나와 우리 모두에게 크나 큰 손실. 내가 생각했던 것 보다 더 심하다."라고 기록하였다. 누나의 임종을 지켜본 뒤 비트겐슈타인은 1950년 4월 케임브리지로 돌아갔다. 비트겐슈타인은 노르웨이를 한 번 더 가보고 싶어하였으나 뜻을 이루지 못하였다.<sup class="reference" id="cite_ref-archivefinalyears_76-1"><a href="#cite_note-archivefinalyears-76">[74]</a></sup></p>
<p>1951년 초 비트겐슈타인은 유언장을 다시 작성하였다. 그는 노먼 말콤에게 "내 마음은 이미 죽었어. 더이상은 고통일 뿐이지. 내 알기로 삶은 언젠가 끝나기 마련이고 그 다음엔 정신적인 삶만이 남게 되는거야"라고 말하였다.<sup class="reference" id="cite_ref-78"><a href="#cite_note-78">[76]</a></sup></p>
<p>62번째 생일이던 1951년 4월 26일 의사가 생이 이제 몇일 남지 않았다고 하자 비트겐슈타인은 "좋군요, 사람들에게 내 삶이 참 멋있었다고 전해주시오"라고 말하였다. 비트겐슈타인은 4월 29일 사망하였고 <a class="mw-redirect" href="/wiki/%EB%A1%9C%EB%A7%88_%EA%B0%80%ED%86%A8%EB%A6%AD" title="로마 가톨릭">로마 가톨릭</a> 식 장례의식을 거쳐 묻혔다.<sup class="reference" id="cite_ref-Monk576_79-0"><a href="#cite_note-Monk576-79">[77]</a></sup><sup class="reference" id="cite_ref-Monk576_79-1"><a href="#cite_note-Monk576-79">[77]</a></sup></p>
<h2><span class="mw-headline" id=".EC.B2.A0.ED.95.99">철학</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%A3%A8%ED%8A%B8%EB%B9%84%ED%9E%88_%EB%B9%84%ED%8A%B8%EA%B2%90%EC%8A%88%ED%83%80%EC%9D%B8&amp;action=edit&amp;section=14" title="부분 편집: 철학">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<h3><span class="mw-headline" id=".E3.80.8A.EB.85.BC.EB.A6.AC-.EC.B2.A0.ED.95.99_.EB.85.BC.EA.B3.A0.E3.80.8B">《논리-철학 논고》</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%A3%A8%ED%8A%B8%EB%B9%84%ED%9E%88_%EB%B9%84%ED%8A%B8%EA%B2%90%EC%8A%88%ED%83%80%EC%9D%B8&amp;action=edit&amp;section=15" title="부분 편집: 《논리-철학 논고》">편집</a><span class="mw-editsection-bracket">]</span></span></h3>

《논리-철학 논고》의 초판 표지</div>

<div class="rellink relarticle mainarticle"><img alt="" data-file-height="128" data-file-width="128" height="16" src="//upload.wikimedia.org/wikipedia/commons/thumb/e/ec/Crystal_Clear_app_xmag.svg/16px-Crystal_Clear_app_xmag.svg.png" srcset="//upload.wikimedia.org/wikipedia/commons/thumb/e/ec/Crystal_Clear_app_xmag.svg/24px-Crystal_Clear_app_xmag.svg.png 1.5x, //upload.wikimedia.org/wikipedia/commons/thumb/e/ec/Crystal_Clear_app_xmag.svg/32px-Crystal_Clear_app_xmag.svg.png 2x" width="16"/> 이 부분의 본문은 <a class="mw-redirect" href="/wiki/%EB%85%BC%EB%A6%AC_%EC%B2%A0%ED%95%99_%EB%85%BC%EA%B3%A0" title="논리 철학 논고">논리 철학 논고</a>입니다.</div>
<p>《<a class="mw-redirect" href="/wiki/%EB%85%BC%EB%A6%AC_%EC%B2%A0%ED%95%99_%EB%85%BC%EA%B3%A0" title="논리 철학 논고">논리-철학 논고</a>》(<span style="font-size: smaller;"><a href="/wiki/%EB%9D%BC%ED%8B%B4%EC%96%B4" title="라틴어">라틴어</a>:</span> <span lang="la" xml:lang="la">Tractatus Logico-Philosophicus</span>)는 비트겐슈타인의 전기 철학을 대표하는 저서이다. 초판의 서문은 비트겐슈타인의 스승이자 동료였던 <a href="/wiki/%EB%B2%84%ED%8A%B8%EB%9F%B0%EB%93%9C_%EB%9F%AC%EC%85%80" title="버트런드 러셀">버트런드 러셀</a>이 썼다. 비트겐슈타인은 <a class="mw-redirect" href="/wiki/1%EC%B0%A8_%EC%84%B8%EA%B3%84_%EB%8C%80%EC%A0%84" title="1차 세계 대전">1차 세계 대전</a> 기간 동안 본문을 썼으며 일정량이 모이면 러셀과 무어에게 보냈고 1918년 완성하였다.<sup class="reference" id="cite_ref-ReferenceA_17-1"><a href="#cite_note-ReferenceA-17">[17]</a></sup> 초판의 출간은 전쟁이 끝난 후인 1922년에 이루어졌다. 흔히 《논고》로 줄여부른다.</p>
<p>비트겐슈타인은 《논고》에서 기존의 철학에서 적용하는 철학적 문제란 언어의 논리를 잘못 적용한 것일 뿐이라고 주장한다.</p>

<td style="padding:10px; vertical-align: top;">이 책은 철학적 문제들을 다루고 있으며, 내가 믿기에는, 이러한 문제들의 문제 제기가 우리의 언어 논리에 대한 오해에 기인한다는 것을 보여 주고 있다. 이 책의 전체적인 뜻은 대략 다음의 말로 요약될 수 있을 것이다. 말해질 수 있는 것은 명료하게 말해질 수 있다. 그리고 이야기할 수 없는 것에 관해서는 우리는 침묵해야 한다. …… 나에겐 여기서 전달된 사고들의 진리성은 불가침적이며 결정적이라고 보인다. 따라서 나는 본질적인 점에서 문제들을 최종적으로 해결했다고 생각한다.<sup class="reference" id="cite_ref-80"><a href="#cite_note-80">[78]</a></sup></td>

<p>《논고》를 집필하던 시점의 비트겐슈타인이 보기에 기존의 철학은 말할 수 없는 것을 말하려고 함으로써 문제를 일으키고 있다는 것이다.<sup class="reference" id="cite_ref-81"><a href="#cite_note-81">[79]</a></sup> 비트겐 슈타인은 이러한 자신의 이론을 설명하기 위해 <b>그림 이론</b>(picture theory)을 제시한다.<sup class="reference" id="cite_ref-82"><a href="#cite_note-82">[80]</a></sup> 그림 이론을 구상하게된 까닭은 프랑스에서 일어난 교통사고 재판에서 모형들이 사용된 것을 본 것 때문이었다. 그림 이론이란 언어는 세계를, 명제는 사실을, 이름은 대상을 지칭한다는 것으로, 이러한 것들이 실제 대응 관계에 있다는 것이다. 비트겐슈타인은 자신의 일기장에 “한 문장에는 하나의 세계가 연습삼아 조립되어 있다”고 기록하였다.<sup class="reference" id="cite_ref-83"><a href="#cite_note-83">[81]</a></sup> 이러한 그림 이론은 기존의 철학, 특히 <a href="/wiki/%ED%98%95%EC%9D%B4%EC%83%81%ED%95%99" title="형이상학">형이상학</a>이나 <a href="/wiki/%EC%9C%A4%EB%A6%AC%ED%95%99" title="윤리학">도덕학</a>에서 신이나 자아, 도덕과 같은 것들은 실제 그것이 나타내고자 하는 것이 없어서 뜻(Sinn)이 없다고 본다. 따라서 이러한 개념에 대해 논의하는 것은 무의미하다고 주장한다. 그러나 <a href="/wiki/%EC%9E%90%EC%97%B0%EA%B3%BC%ED%95%99" title="자연과학">자연과학</a>과 같은 것은 실제 세계를 설명하는 것이기 때문에 그 의미가 있다고 본다. 이러한 점에서 비트겐슈타인은 《논리 철학 논고》를 "말할 수 없는 것에 관해서는 침묵해야 한다.<sup class="reference" id="cite_ref-84"><a href="#cite_note-84">[82]</a></sup>" 라고 끝맺었다. <a class="mw-redirect" href="/wiki/%EB%85%BC%EB%A6%AC_%EC%8B%A4%EC%A6%9D%EC%A3%BC%EC%9D%98" title="논리 실증주의">논리 실증주의자</a>들은 이 명제를 중요하게 생각했다.</p>
<p>하지만, 비트겐슈타인은 편집자에게 보내는 편지에서 오히려 말할 수 없는 것을 더 중요하게 생각한다고 고백했다. 말할 수 없는 것이 증명할 수 없어서 무의미한 것이 아니라, 구태여 증명하려 하여 무가치하게 만들지 말라는 것이었다<sup class="reference" id="cite_ref-85"><a href="#cite_note-85">[83]</a></sup>.</p>
<h3><span class="mw-headline" id=".E3.80.8A.EC.B2.A0.ED.95.99.EC.A0.81_.ED.83.90.EA.B5.AC.E3.80.8B">《철학적 탐구》</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%A3%A8%ED%8A%B8%EB%B9%84%ED%9E%88_%EB%B9%84%ED%8A%B8%EA%B2%90%EC%8A%88%ED%83%80%EC%9D%B8&amp;action=edit&amp;section=16" title="부분 편집: 《철학적 탐구》">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<div class="rellink relarticle mainarticle"><img alt="" data-file-height="128" data-file-width="128" height="16" src="//upload.wikimedia.org/wikipedia/commons/thumb/e/ec/Crystal_Clear_app_xmag.svg/16px-Crystal_Clear_app_xmag.svg.png" srcset="//upload.wikimedia.org/wikipedia/commons/thumb/e/ec/Crystal_Clear_app_xmag.svg/24px-Crystal_Clear_app_xmag.svg.png 1.5x, //upload.wikimedia.org/wikipedia/commons/thumb/e/ec/Crystal_Clear_app_xmag.svg/32px-Crystal_Clear_app_xmag.svg.png 2x" width="16"/> 이 부분의 본문은 <a class="mw-redirect" href="/wiki/%EC%B2%A0%ED%95%99%EC%A0%81_%ED%83%90%EA%B5%AC" title="철학적 탐구">철학적 탐구</a>입니다.</div>
<p>비트겐슈타인의 회고에 따르면, 그는 <a href="/wiki/%EC%BC%80%EC%9E%84%EB%B8%8C%EB%A6%AC%EC%A7%80_%EB%8C%80%ED%95%99%EA%B5%90" title="케임브리지 대학교">케임브리지 대학교</a>에서 함께 재직하던 이탈리아 출신의 경제학자 <a href="/wiki/%ED%94%BC%EC%97%90%EB%A1%9C_%EC%8A%A4%EB%9D%BC%ED%8C%8C" title="피에로 스라파">피에로 스라파</a>와 자신의 그림 이론에 대해 토론을 하던 가운데 잘못을 깨달았다고 한다. 비트겐슈타인이 이론에 스라파가 반론하면서 손가락 끝으로 목부분을 밀어 올렸다. 스라파의 행동은 이탈리아에서 의문이나 조소를 뜻하는 것으로 사용되는 <a href="/wiki/%EC%A0%9C%EC%8A%A4%EC%B2%98" title="제스처">제스처</a>였다. 순간 비트겐슈타인은 《논고》에서 주장했던 언어의 논리학과는 달리 일상 생활에서 쓰이는 언어의 의미는 결코 한 가지로 고착되지 않는다는 점을 깨달았다고 한다.<sup class="reference" id="cite_ref-86"><a href="#cite_note-86">[84]</a></sup> 비트겐슈타인은 이러한 생각의 전환을 바탕으로 《철학적 탐구》를 집필하였으나 출판하지는 않았다. 《철학적 탐구》는 비트겐슈타인의 사후에 남겨진 초고를 합하여 출판되었다.<sup class="reference" id="cite_ref-87"><a href="#cite_note-87">[85]</a></sup></p>
<p>《철학적 탐구》에 이르러 비트겐슈타인은 자신의 철학을 상당부분 수정하게 된다. 초기의 그림 이론과는 달리 일상적으로 사용되는 언어를 중요하게 여긴 것이다. 비트겐슈타인은 자신의 그림 이론을 포함한 기존에 있었던 사물과 언어가 일치한다는 주장을 반대하였다. 비트겐슈타인에 따르면, 언어가 있기 전에 생활 양식이 있다. 또한, 언어는 그 '뜻'<sup class="reference" id="cite_ref-88"><a href="#cite_note-88">[주해 3]</a></sup> 이 아니라 '사용'<sup class="reference" id="cite_ref-89"><a href="#cite_note-89">[주해 4]</a></sup> 에 본질이 있으며, 같은 언어를 사용한다는 것은 삶의 형식을 공유한다고 생각했다. 그리고 비트겐슈타인은 언어에는 하나의 공통된 본질이 있는 것이 아니라 그 쓰임에서 나타나는 여러 유사성이 있다고 생각했는데, 그는 이것을 '가족 유사성'(family resemblances)<sup class="reference" id="cite_ref-90"><a href="#cite_note-90">[주해 5]</a></sup> 이라고 불렀다. 비트겐슈타인은 언어를 놀이에 비유했는데, 줄넘기 놀이, 술래잡기, 가위바위보 등의 '놀이'에서도 어떤 본질이 있는 것이 아니라 마치 가족처럼 서로 유사한 점이 있다는 뜻이다. <a class="new" href="/w/index.php?title=%EB%8C%80%EB%8B%88%EC%96%BC_%EC%86%94%EB%A1%9C%EB%B8%8C&amp;action=edit&amp;redlink=1" title="대니얼 솔로브 (없는 문서)">대니얼 솔로브</a>는 이러한 비트겐슈타인의 가족유사성 접근을 현대형 프라이버시 개념을 정리하는 방법으로 활용하자고 주장하기도 하였다.<sup class="reference" id="cite_ref-91"><a href="#cite_note-91">[86]</a></sup></p>
<p>비트겐슈타인은 철학이라는 학문이 어떠한 사물이나 현상에 대한 학문이 아니라, 그러한 것들을 연구하는 학문에서 사용되는 언어를 연구하는 학문이라고 생각했다. 비트겐슈타인은 <a class="mw-redirect" href="/wiki/%EB%85%BC%EB%A6%AC_%EC%8B%A4%EC%A6%9D%EC%A3%BC%EC%9D%98" title="논리 실증주의">논리 실증주의자</a>들이 명료하고 논리적인 이상적인 상태의 언어를 추구하는 것을 비판했다. 이러한 철학은 옥스퍼드학파라고도 불리는 <a class="mw-redirect" href="/wiki/%EC%9D%BC%EC%83%81%EC%96%B8%EC%96%B4%ED%95%99%ED%8C%8C" title="일상언어학파">일상언어학파</a>가 잇게 된다.<sup class="reference" id="cite_ref-92"><a href="#cite_note-92">[87]</a></sup></p>
