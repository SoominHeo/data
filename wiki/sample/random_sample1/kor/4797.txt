<p><b>루한스크 주</b>(<span style="font-size: smaller;"><a href="/wiki/%EC%9A%B0%ED%81%AC%EB%9D%BC%EC%9D%B4%EB%82%98%EC%96%B4" title="우크라이나어">우크라이나어</a>:</span> <span lang="uk" xml:lang="uk">Луганська область</span>)는 <a href="/wiki/%EC%9A%B0%ED%81%AC%EB%9D%BC%EC%9D%B4%EB%82%98" title="우크라이나">우크라이나</a> 최동단에 위치한 주로 주도는 <a href="/wiki/%EB%A3%A8%ED%95%9C%EC%8A%A4%ED%81%AC" title="루한스크">루한스크</a>이며 면적은 26,684㎢, 인구는 2,409,000명(2006년 기준)이다. 북쪽으로는 <a href="/wiki/%EB%9F%AC%EC%8B%9C%EC%95%84" title="러시아">러시아</a> <a href="/wiki/%EB%B2%A8%EA%B3%A0%EB%A1%9C%EB%93%9C_%EC%A3%BC" title="벨고로드 주">벨고로드 주</a>와 <a href="/wiki/%EB%B3%B4%EB%A1%9C%EB%84%A4%EC%8B%9C_%EC%A3%BC" title="보로네시 주">보로네시 주</a>, 동쪽과 남쪽으로는 러시아 <a href="/wiki/%EB%A1%9C%EC%8A%A4%ED%86%A0%ED%94%84_%EC%A3%BC" title="로스토프 주">로스토프 주</a>, 서쪽으로는 <a href="/wiki/%ED%95%98%EB%A5%B4%ED%82%A4%EC%9A%B0_%EC%A3%BC" title="하르키우 주">하르키우 주</a>, <a href="/wiki/%EB%8F%84%EB%84%A4%EC%B8%A0%ED%81%AC_%EC%A3%BC" title="도네츠크 주">도네츠크 주</a>와 접한다.</p>
<p><a href="/wiki/1938%EB%85%84" title="1938년">1938년</a> <a href="/wiki/%ED%81%B4%EB%A6%AC%EB%A9%98%ED%8A%B8_%EB%B3%B4%EB%A1%9C%EC%8B%A4%EB%A1%9C%ED%94%84" title="클리멘트 보로실로프">클리멘트 보로실로프</a>의 이름을 딴 <b>보로실로프그라드 주</b>라는 이름으로 성립되었다. 보로실로프그라드라는 이름은 <a href="/wiki/1958%EB%85%84" title="1958년">1958년</a>까지, 그리고 다시 <a href="/wiki/1970%EB%85%84" title="1970년">1970년</a>부터 <a href="/wiki/1990%EB%85%84" title="1990년">1990년</a> 사이에 쓰였다.</p>
<p><a href="/wiki/2014%EB%85%84" title="2014년">2014년</a> 친(親)러시아 무장 세력이 <a href="/wiki/%EB%A3%A8%EA%B0%84%EC%8A%A4%ED%81%AC_%EC%9D%B8%EB%AF%BC%EA%B3%B5%ED%99%94%EA%B5%AD" title="루간스크 인민공화국">루간스크 인민공화국</a>이라는 이름으로 우크라이나로부터의 분리 독립을 선언하였다. 이 때문에 우크라이나 당국은 임시 주도를 <a href="/wiki/%EC%84%B8%EB%B2%A0%EB%A1%9C%EB%8F%84%EB%84%A4%EC%B8%A0%ED%81%AC" title="세베로도네츠크">세베로도네츠크</a>에 두었다.</p>
<h2><span class="mw-headline" id=".EC.A3.BC.EB.AF.BC">주민</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%A3%A8%ED%95%9C%EC%8A%A4%ED%81%AC_%EC%A3%BC&amp;action=edit&amp;section=1" title="부분 편집: 주민">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>2001년 인구조사 결과 <a href="/wiki/%EC%9A%B0%ED%81%AC%EB%9D%BC%EC%9D%B4%EB%82%98%EC%9D%B8" title="우크라이나인">우크라이나인</a>이 주민의 58%를 차지하며 주요 소수 민족은 <a href="/wiki/%EB%9F%AC%EC%8B%9C%EC%95%84%EC%9D%B8" title="러시아인">러시아인</a>(39.1%), <a href="/wiki/%EB%B2%A8%EB%9D%BC%EB%A3%A8%EC%8A%A4%EC%9D%B8" title="벨라루스인">벨라루스인</a>(0.8%) 등이다. 그러나 민족적 조성(造成)과는 달리 이 곳 주민들의 68.8%는 <a href="/wiki/%EB%9F%AC%EC%8B%9C%EC%95%84%EC%96%B4" title="러시아어">러시아어</a> 사용자들이다. <a href="/wiki/%EC%9A%B0%ED%81%AC%EB%9D%BC%EC%9D%B4%EB%82%98%EC%96%B4" title="우크라이나어">우크라이나어</a> 사용자는 30%에 불과하다.</p>
