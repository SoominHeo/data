<p>《<b>루흐나마</b>》(Ruhnama)는 <a href="/wiki/%ED%88%AC%EB%A5%B4%ED%81%AC%EB%A9%94%EB%8B%88%EC%8A%A4%ED%83%84" title="투르크메니스탄">투르크메니스탄</a>의 종신 대통령을 역임했던 <a href="/wiki/%EC%82%AC%ED%8C%8C%EB%A5%B4%EB%AC%B4%EB%9D%BC%ED%8A%B8_%EB%8B%88%EC%95%BC%EC%A1%B0%ED%94%84" title="사파르무라트 니야조프">사파르무라트 니야조프</a>가 <a href="/wiki/%EC%9E%90%EC%84%9C%EC%A0%84" title="자서전">자서전</a> 형식으로 쓴 책으로 <a href="/wiki/2001%EB%85%84" title="2001년">2001년</a>에 처음 출간되었다. 루흐나마는 "영혼의 책"을 뜻하는데 <a href="/wiki/%EC%95%84%EB%9E%8D%EC%96%B4" title="아랍어">아랍어</a>로 "영혼"을 뜻하는 '루흐'(روح, rūḥ)와 <a href="/wiki/%ED%8E%98%EB%A5%B4%EC%8B%9C%EC%95%84%EC%96%B4" title="페르시아어">페르시아어</a>로 "편지" 또는 "책"을 뜻하는 '나메'(نامه, nāmeh)의 합성어다.</p>
<p>이 책은 <a href="/wiki/%EB%9F%AC%EC%8B%9C%EC%95%84%EC%96%B4" title="러시아어">러시아어</a>, <a href="/wiki/%EC%A4%91%EA%B5%AD%EC%96%B4" title="중국어">중국어</a>, <a href="/wiki/%EC%98%81%EC%96%B4" title="영어">영어</a>, <a href="/wiki/%ED%84%B0%ED%82%A4%EC%96%B4" title="터키어">터키어</a>, <a href="/wiki/%EC%9D%BC%EB%B3%B8%EC%96%B4" title="일본어">일본어</a>, <a href="/wiki/%ED%8E%98%EB%A5%B4%EC%8B%9C%EC%95%84%EC%96%B4" title="페르시아어">페르시아어</a>, <a href="/wiki/%ED%95%9C%EA%B5%AD%EC%96%B4" title="한국어">한국어</a> 등 40개 이상의 언어로 번역되어 출판되었으며 시각 장애인을 위해 음성판, 점자판도 출판되었다. <a href="/wiki/2001%EB%85%84" title="2001년">2001년</a>에 1권이 출간되었고 <a href="/wiki/2004%EB%85%84" title="2004년">2004년</a>에 2권이 출간되었다. <a href="/wiki/2006%EB%85%84" title="2006년">2006년</a>에는 100만 부를 돌파하였다.</p>
<p>니야조프는 2006년 3월에 방송된 텔레비전 프로그램에서 이 책을 하루에 세 번 읽으면 천국에 간다고 선전했으며 투르크메니스탄에서는 이 책이 <a class="mw-redirect" href="/wiki/%EC%BF%A0%EB%9E%80" title="쿠란">쿠란</a>, <a href="/wiki/%EC%84%B1%EA%B2%BD" title="성경">성경</a>과 같은 책으로 취급받기도 하였다. 니야조프는 가장 먼저 이 책을 전국의 모든 학교와 도서관에 비치하도록 지시했으며 학교 및 대학 입학 시험, 공무원 자격 시험, 운전 면허 시험 등에서 이 책에 관한 시험 문제를 출제하도록 지시했다.</p>
<p>투르크메니스탄의 모든 학교에서는 이 책에 대한 교육이 의무적으로 진행되었고 학교와 도서관, 정부 기관에서는 이 책을 읽는 것이 의무화되었다. 또한 새로 임명된 투르크메니스탄의 공무원들은 의무적으로 이 책의 내용에 관한 면접을 받았다.</p>
