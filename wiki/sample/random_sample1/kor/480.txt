<p><b>2014년 동계 패럴림픽</b>(<span style="font-size: smaller;"><a href="/wiki/%EC%98%81%EC%96%B4" title="영어">영어</a>:</span> <span lang="en" xml:lang="en">2014 Winter Paralympics</span>, <span lang="en" xml:lang="en">XI Paralympic Winter Games</span>, <span style="font-size: smaller;"><a href="/wiki/%EB%9F%AC%EC%8B%9C%EC%95%84%EC%96%B4" title="러시아어">러시아어</a>:</span> <span lang="ru" xml:lang="ru">Зимние Паралимпийские игры 2014</span>)은 <a href="/wiki/%EB%9F%AC%EC%8B%9C%EC%95%84" title="러시아">러시아</a> <a href="/wiki/%EC%86%8C%EC%B9%98" title="소치">소치</a>에서 <a href="/wiki/2014%EB%85%84" title="2014년">2014년</a> <a href="/wiki/3%EC%9B%94_7%EC%9D%BC" title="3월 7일">3월 7일</a>부터 <a href="/wiki/3%EC%9B%94_16%EC%9D%BC" title="3월 16일">3월 16일</a>까지 개최된 <a href="/wiki/%EB%8F%99%EA%B3%84_%ED%8C%A8%EB%9F%B4%EB%A6%BC%ED%94%BD" title="동계 패럴림픽">동계 패럴림픽</a>이다. 2007년 7월 4일, <a href="/wiki/%EA%B5%AD%EC%A0%9C_%EC%98%AC%EB%A6%BC%ED%94%BD_%EC%9C%84%EC%9B%90%ED%9A%8C" title="국제 올림픽 위원회">국제 올림픽 위원회</a>는 <a href="/wiki/%EB%9F%AC%EC%8B%9C%EC%95%84" title="러시아">러시아</a> <a href="/wiki/%EC%86%8C%EC%B9%98" title="소치">소치</a>를 개최지로 선정하였다. 이는 러시아에서 열리는 첫 <a href="/wiki/%ED%8C%A8%EB%9F%B4%EB%A6%BC%ED%94%BD" title="패럴림픽">패럴림픽</a>이며, 처음으로 <a class="mw-redirect" href="/wiki/%EC%8A%A4%EB%85%B8%EC%9A%B0%EB%B3%B4%EB%93%9C" title="스노우보드">스노우보드</a> 종목이 대회 종목 내에 포함되었다.<sup class="reference" id="cite_ref-autogenerated1_1-0"><a href="#cite_note-autogenerated1-1">[1]</a></sup></p>

<h2><span class="mw-headline" id=".EA.B0.9C.EC.B5.9C.EC.A7.80_.EC.84.A0.EC.A0.95">개최지 선정</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=2014%EB%85%84_%EB%8F%99%EA%B3%84_%ED%8C%A8%EB%9F%B4%EB%A6%BC%ED%94%BD&amp;action=edit&amp;section=1" title="부분 편집: 개최지 선정">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p><a href="/wiki/%EA%B5%AD%EC%A0%9C_%EC%98%AC%EB%A6%BC%ED%94%BD_%EC%9C%84%EC%9B%90%ED%9A%8C" title="국제 올림픽 위원회">국제 올림픽 위원회</a>와 <a href="/wiki/%EA%B5%AD%EC%A0%9C_%ED%8C%A8%EB%9F%B4%EB%A6%BC%ED%94%BD_%EC%9C%84%EC%9B%90%ED%9A%8C" title="국제 패럴림픽 위원회">국제 패럴림픽 위원회</a>는 2001년 협의를 통해 2014년 동계 올림픽의 개최지에서 2014년 동계 패럴림픽도 함께 개최하기로 협약하였다. 2007년 7월 4일 <a href="/wiki/%EA%B3%BC%ED%85%8C%EB%A7%90%EB%9D%BC" title="과테말라">과테말라</a>의 <a class="mw-redirect" href="/wiki/%EA%B3%BC%ED%85%8C%EB%A7%90%EB%9D%BC%EC%8B%9C%ED%8B%B0" title="과테말라시티">과테말라시티</a>에서 열린 119차 IOC 총회에서 투표를 통해 <a href="/wiki/%EB%9F%AC%EC%8B%9C%EC%95%84" title="러시아">러시아</a> <a href="/wiki/%EC%86%8C%EC%B9%98" title="소치">소치</a>가 <a href="/wiki/2014%EB%85%84_%EB%8F%99%EA%B3%84_%EC%98%AC%EB%A6%BC%ED%94%BD" title="2014년 동계 올림픽">2014년 동계 올림픽</a> 및 동계 패럴림픽 개최지로 선정되었다.</p>

<h3><span class="mw-headline" id=".EB.A7.88.EC.8A.A4.EC.BD.94.ED.8A.B8">마스코트</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=2014%EB%85%84_%EB%8F%99%EA%B3%84_%ED%8C%A8%EB%9F%B4%EB%A6%BC%ED%94%BD&amp;action=edit&amp;section=4" title="부분 편집: 마스코트">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>2011년 2월 7일, 10종의 올림픽 및 3종의 패럴림픽 후보 디자인이 대중에 공개되었으며, 2011년 2월 26일 최종 결정되었다.<sup class="reference" id="cite_ref-7"><a href="#cite_note-7">[7]</a></sup> 밴쿠버 올림픽 이후 세 번째로 올림픽과 패럴림픽 마스코트가 동시에 공개된 사례이다. 2014년 동계 패럴림픽의 공식 마스코트로 선정된 한줄기빛과 눈꽃송이는 외계에서 왔다. <sup class="reference" id="cite_ref-8"><a href="#cite_note-8">[8]</a></sup> 불의 소년인 ‘한줄기빛’은 불꽃 모양의 머리카락을 가지고 있으며 열기가 가득한 행성에서 왔다. 반면, 눈의 소녀인 ‘눈꽃송이’는 눈의 결정을 닮았으며 얼음으로 뒤덮인 행성에서 왔다고 한다.<sup class="reference" id="cite_ref-9"><a href="#cite_note-9">[9]</a></sup></p>
<h2><span class="mw-headline" id=".EB.8C.80.ED.9A.8C_.EC.A7.84.ED.96.89">대회 진행</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=2014%EB%85%84_%EB%8F%99%EA%B3%84_%ED%8C%A8%EB%9F%B4%EB%A6%BC%ED%94%BD&amp;action=edit&amp;section=5" title="부분 편집: 대회 진행">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<h3><span class="mw-headline" id=".EC.A2.85.EB.AA.A9">종목</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=2014%EB%85%84_%EB%8F%99%EA%B3%84_%ED%8C%A8%EB%9F%B4%EB%A6%BC%ED%94%BD&amp;action=edit&amp;section=6" title="부분 편집: 종목">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>2014년 동계 패럴림픽은 다음과 같은 세부 종목으로 구성된다.</p>

<h3><span class="mw-headline" id=".EC.B0.B8.EA.B0.80.EA.B5.AD">참가국</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=2014%EB%85%84_%EB%8F%99%EA%B3%84_%ED%8C%A8%EB%9F%B4%EB%A6%BC%ED%94%BD&amp;action=edit&amp;section=7" title="부분 편집: 참가국">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>2014년 동계 패럴림픽 참가국은 다음과 같다. <a href="/wiki/2010%EB%85%84_%EB%8F%99%EA%B3%84_%ED%8C%A8%EB%9F%B4%EB%A6%BC%ED%94%BD" title="2010년 동계 패럴림픽">2010년 동계 패럴림픽</a> 참가국 가운데 <a href="/wiki/%ED%97%9D%EA%B0%80%EB%A6%AC" title="헝가리">헝가리</a>와 <a href="/wiki/%EB%82%A8%EC%95%84%ED%94%84%EB%A6%AC%EC%B9%B4_%EA%B3%B5%ED%99%94%EA%B5%AD" title="남아프리카 공화국">남아프리카 공화국</a>은 참가하지 않았다. <a href="/wiki/%EB%B8%8C%EB%9D%BC%EC%A7%88" title="브라질">브라질</a>, <a href="/wiki/%ED%84%B0%ED%82%A4" title="터키">터키</a>, <a href="/wiki/%EC%9A%B0%EC%A6%88%EB%B2%A0%ED%82%A4%EC%8A%A4%ED%83%84" title="우즈베키스탄">우즈베키스탄</a>이 처음으로 동계 패럴림픽에 참가했다.</p>

<h2><span class="mw-headline" id=".EC.82.AC.EA.B1.B4_.EB.B0.8F_.EB.85.BC.EB.9E.80">사건 및 논란</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=2014%EB%85%84_%EB%8F%99%EA%B3%84_%ED%8C%A8%EB%9F%B4%EB%A6%BC%ED%94%BD&amp;action=edit&amp;section=9" title="부분 편집: 사건 및 논란">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>2014년 동계 패럴림픽을 앞둔 상황에서 일어난 <a href="/wiki/2014%EB%85%84_%EC%9A%B0%ED%81%AC%EB%9D%BC%EC%9D%B4%EB%82%98_%ED%98%81%EB%AA%85" title="2014년 우크라이나 혁명">2014년 우크라이나 혁명</a> 이후에 러시아 군인들이 <a href="/wiki/%EC%9A%B0%ED%81%AC%EB%9D%BC%EC%9D%B4%EB%82%98" title="우크라이나">우크라이나</a>의 영토로 있던 <a href="/wiki/%ED%81%AC%EB%A6%BC_%EB%B0%98%EB%8F%84" title="크림 반도">크림 반도</a>를 <a href="/wiki/%ED%81%AC%EB%A6%BC_%EC%9C%84%EA%B8%B0" title="크림 위기">장악</a>했다. 이 소식을 접한 <a href="/wiki/%EB%AF%B8%EA%B5%AD" title="미국">미국</a>, <a href="/wiki/%ED%94%84%EB%9E%91%EC%8A%A4" title="프랑스">프랑스</a>, <a href="/wiki/%EB%8F%85%EC%9D%BC" title="독일">독일</a>, <a href="/wiki/%EC%98%81%EA%B5%AD" title="영국">영국</a>, <a href="/wiki/%EC%BA%90%EB%82%98%EB%8B%A4" title="캐나다">캐나다</a>는 2014년 동계 패럴림픽 개막식에 정부 대표단을 파견하려던 계획을 취소했다. 한편 우크라이나는 패럴림픽 개막식에 있었던 선수단 입장 행사에서 러시아의 크림 반도 장악에 항의하는 표시로 기수 1명만 입장했다.</p>
