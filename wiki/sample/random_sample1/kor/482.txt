<h2><span class="mw-headline" id=".EB.94.94.EB.B9.84.EC.A0.84_1">디비전 1</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=2014%EB%85%84_%EC%84%B8%EA%B3%84_%EB%82%A8%EC%9E%90_%EC%95%84%EC%9D%B4%EC%8A%A4%ED%95%98%ED%82%A4_%EC%84%A0%EC%88%98%EA%B6%8C_%EB%8C%80%ED%9A%8C&amp;action=edit&amp;section=2" title="부분 편집: 디비전 1">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<div class="rellink relarticle mainarticle"><img alt="" data-file-height="128" data-file-width="128" height="16" src="//upload.wikimedia.org/wikipedia/commons/thumb/e/ec/Crystal_Clear_app_xmag.svg/16px-Crystal_Clear_app_xmag.svg.png" srcset="//upload.wikimedia.org/wikipedia/commons/thumb/e/ec/Crystal_Clear_app_xmag.svg/24px-Crystal_Clear_app_xmag.svg.png 1.5x, //upload.wikimedia.org/wikipedia/commons/thumb/e/ec/Crystal_Clear_app_xmag.svg/32px-Crystal_Clear_app_xmag.svg.png 2x" width="16"/> 이 부분의 본문은 <a href="/wiki/2014%EB%85%84_%EB%94%94%EB%B9%84%EC%A0%84_1_IIHF_%EC%B1%94%ED%94%BC%EC%96%B8%EC%8B%AD" title="2014년 디비전 1 IIHF 챔피언십">2014년 디비전 1 IIHF 챔피언십</a>입니다.</div>
<p>디비전 1에 속하는 12개 국가들은 2개 그룹으로 나누어 별도로 경기를 치렀다. 상위권의 A그룹 국가들은 <a href="/wiki/%EB%8C%80%ED%95%9C%EB%AF%BC%EA%B5%AD" title="대한민국">대한민국</a> <a class="mw-redirect" href="/wiki/%EA%B3%A0%EC%96%91" title="고양">고양</a>에서, 하위권의 B그룹 국가들은 <a href="/wiki/%EB%A6%AC%ED%88%AC%EC%95%84%EB%8B%88%EC%95%84" title="리투아니아">리투아니아</a> <a href="/wiki/%EB%B9%8C%EB%89%B4%EC%8A%A4" title="빌뉴스">빌뉴스</a>에서 <a href="/wiki/4%EC%9B%94_20%EC%9D%BC" title="4월 20일">4월 20일</a> ~ <a href="/wiki/4%EC%9B%94_26%EC%9D%BC" title="4월 26일">4월 26일</a>에 경기를 치렀다.</p>

<h2><span class="mw-headline" id=".EB.94.94.EB.B9.84.EC.A0.84_2">디비전 2</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=2014%EB%85%84_%EC%84%B8%EA%B3%84_%EB%82%A8%EC%9E%90_%EC%95%84%EC%9D%B4%EC%8A%A4%ED%95%98%ED%82%A4_%EC%84%A0%EC%88%98%EA%B6%8C_%EB%8C%80%ED%9A%8C&amp;action=edit&amp;section=3" title="부분 편집: 디비전 2">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<div class="rellink relarticle mainarticle"><img alt="" data-file-height="128" data-file-width="128" height="16" src="//upload.wikimedia.org/wikipedia/commons/thumb/e/ec/Crystal_Clear_app_xmag.svg/16px-Crystal_Clear_app_xmag.svg.png" srcset="//upload.wikimedia.org/wikipedia/commons/thumb/e/ec/Crystal_Clear_app_xmag.svg/24px-Crystal_Clear_app_xmag.svg.png 1.5x, //upload.wikimedia.org/wikipedia/commons/thumb/e/ec/Crystal_Clear_app_xmag.svg/32px-Crystal_Clear_app_xmag.svg.png 2x" width="16"/> 이 부분의 본문은 <a class="new" href="/w/index.php?title=2014%EB%85%84_%EB%94%94%EB%B9%84%EC%A0%84_2_IIHF_%EC%B1%94%ED%94%BC%EC%96%B8%EC%8B%AD&amp;action=edit&amp;redlink=1" title="2014년 디비전 2 IIHF 챔피언십 (없는 문서)">2014년 디비전 2 IIHF 챔피언십</a>입니다.</div>
<p>디비전 2에 속한 12개 국가들은 2개 그룹으로 나누어 별도로 경기를 치렀다. 상위권의 A그룹 국가들은 <a href="/wiki/%EC%84%B8%EB%A5%B4%EB%B9%84%EC%95%84" title="세르비아">세르비아</a> <a href="/wiki/%EB%B2%A0%EC%98%A4%EA%B7%B8%EB%9D%BC%EB%93%9C" title="베오그라드">베오그라드</a>에서 <a href="/wiki/4%EC%9B%94_9%EC%9D%BC" title="4월 9일">4월 9일</a> ~ <a href="/wiki/4%EC%9B%94_15%EC%9D%BC" title="4월 15일">4월 15일</a>, 하위권의 B그룹 국가들은 <a href="/wiki/%EC%8A%A4%ED%8E%98%EC%9D%B8" title="스페인">스페인</a> <a class="mw-disambig" href="/wiki/%ED%95%98%EC%B9%B4" title="하카">하카</a>에서 <a href="/wiki/4%EC%9B%94_5%EC%9D%BC" title="4월 5일">4월 5일</a> ~ <a href="/wiki/4%EC%9B%94_11%EC%9D%BC" title="4월 11일">4월 11일</a> 경기를 치렀다.</p>
