<p><b>뤼크 베송</b>(<span style="font-size: smaller;"><a href="/wiki/%ED%94%84%EB%9E%91%EC%8A%A4%EC%96%B4" title="프랑스어">프랑스어</a>:</span> <span lang="fr" xml:lang="fr">Luc Besson</span>, <a href="/wiki/1959%EB%85%84" title="1959년">1959년</a> <a href="/wiki/3%EC%9B%94_18%EC%9D%BC" title="3월 18일">3월 18일</a>~ )은 <a href="/wiki/%ED%94%84%EB%9E%91%EC%8A%A4" title="프랑스">프랑스</a>의 <a class="mw-redirect" href="/wiki/%EC%98%81%ED%99%94%EA%B0%90%EB%8F%85" title="영화감독">영화감독</a>, <a class="mw-redirect mw-disambig" href="/wiki/%EC%A0%9C%EC%9E%91%EC%9E%90" title="제작자">제작자</a>, <a href="/wiki/%EA%B0%81%EB%B3%B8%EA%B0%80" title="각본가">각본가</a>이다.</p>
<h2><span class="mw-headline" id=".EC.83.9D.EC.95.A0">생애</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%A4%BC%ED%81%AC_%EB%B2%A0%EC%86%A1&amp;action=edit&amp;section=1" title="부분 편집: 생애">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p><a href="/wiki/%EA%B7%B8%EB%A6%AC%EC%8A%A4" title="그리스">그리스</a>와 <a href="/wiki/%EC%9C%A0%EA%B3%A0%EC%8A%AC%EB%9D%BC%EB%B9%84%EC%95%84" title="유고슬라비아">유고슬라비아</a>의 <a class="mw-redirect" href="/wiki/%ED%81%B4%EB%9F%BD_%EB%A9%94%EB%93%9C" title="클럽 메드">클럽 메드</a>에서 <a href="/wiki/%EC%8A%A4%EC%BF%A0%EB%B2%84_%EB%8B%A4%EC%9D%B4%EB%B9%99" title="스쿠버 다이빙">스쿠버 다이빙</a> 교사로 일하던 부모 밑에서 자랐다. 프랑스와 <a href="/wiki/%EB%AF%B8%EA%B5%AD" title="미국">미국</a>에서 <a class="new" href="/w/index.php?title=%EC%A1%B0%EA%B0%90%EB%8F%85&amp;action=edit&amp;redlink=1" title="조감독 (없는 문서)">조감독</a>으로 일하면서 영화계에 입문했다.</p>
