<p><b>류머티즘</b>(Rheumatism)은 <a href="/wiki/%EA%B2%B0%ED%95%A9_%EC%A1%B0%EC%A7%81" title="결합 조직">결합 조직</a>이나 <a href="/wiki/%EA%B4%80%EC%A0%88" title="관절">관절</a>에 통증을 유발하는 의학 문제를 일컫는 용어이며, <b>풍습</b>(風濕)이라고도 한다. 이러한 질환과 치료법을 연구하는 학문을 <a class="new" href="/w/index.php?title=%EB%A5%98%EB%A8%B8%ED%8B%B0%EC%8A%A4%ED%95%99&amp;action=edit&amp;redlink=1" title="류머티스학 (없는 문서)">류머티스학</a>(Rheumatology)이라고 한다.</p>

<h2><span class="mw-headline" id=".EC.A2.85.EB.A5.98">종류</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%A5%98%EB%A8%B8%ED%8B%B0%EC%A6%98&amp;action=edit&amp;section=1" title="부분 편집: 종류">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>현재 류머티즘으로 여겨지는 질환은 다음과 같다:</p>
<ul>
<li><a class="mw-redirect" href="/wiki/%EA%B0%95%EC%A7%81%EC%84%B1_%EC%B2%99%EC%B6%94%EC%97%BC" title="강직성 척추염">강직성 척추염</a> (<a class="extiw" href="https://en.wikipedia.org/wiki/Ankylosing_spondylitis" title="en:Ankylosing spondylitis">en</a>)</li>
<li><a class="new" href="/w/index.php?title=%EA%B1%B4%EC%84%A0_%EA%B4%80%EC%A0%88%EC%97%BC&amp;action=edit&amp;redlink=1" title="건선 관절염 (없는 문서)">건선 관절염</a> (<a class="extiw" href="https://en.wikipedia.org/wiki/Psoriatic_arthritis" title="en:Psoriatic arthritis">en</a>)</li>
<li><a href="/wiki/%EA%B1%B4%EC%B4%88%EC%97%BC" title="건초염">건초염</a></li>
<li><a class="new" href="/w/index.php?title=%EA%B2%BD%EB%B6%80%ED%86%B5&amp;action=edit&amp;redlink=1" title="경부통 (없는 문서)">경부통</a> (<a class="extiw" href="https://en.wikipedia.org/wiki/Neck_pain" title="en:Neck pain">en</a>)</li>
<li>골 <a href="/wiki/%EA%B4%80%EC%A0%88%EC%97%BC" title="관절염">관절염</a></li>
<li><a class="new" href="/w/index.php?title=%EA%B7%BC%EC%97%BC&amp;action=edit&amp;redlink=1" title="근염 (없는 문서)">근염</a> (<a class="extiw" href="https://en.wikipedia.org/wiki/Myositis" title="en:Myositis">en</a>)</li>
<li><a class="new" href="/w/index.php?title=%EB%82%AD%EC%97%BC&amp;action=edit&amp;redlink=1" title="낭염 (없는 문서)">낭염</a> (<a class="extiw" href="https://en.wikipedia.org/wiki/Capsulitis" title="en:Capsulitis">en</a>)</li>
<li><a href="/wiki/%EB%A5%98%EB%A7%88%ED%8B%B0%EC%8A%A4_%EA%B4%80%EC%A0%88%EC%97%BC" title="류마티스 관절염">류마티스 관절염</a></li>
<li><a class="new" href="/w/index.php?title=%EB%A5%98%EB%A8%B8%ED%8B%B0%EC%8A%A4_%EB%B0%9C%EC%97%B4&amp;action=edit&amp;redlink=1" title="류머티스 발열 (없는 문서)">류머티스 발열</a> (<a class="extiw" href="https://en.wikipedia.org/wiki/Rheumatic_fever" title="en:Rheumatic fever">en</a>)</li>

<li><a href="/wiki/%EC%A0%84%EC%8B%A0_%ED%99%8D%EB%B0%98%EC%84%B1_%EB%A3%A8%ED%91%B8%EC%8A%A4" title="전신 홍반성 루푸스">전신 홍반성 루푸스</a></li>

</ul>
<p>이러한 질환들은 <a href="/wiki/%EC%97%AD%ED%95%99_(%EC%9D%98%ED%95%99)" title="역학 (의학)">역학</a>적으로 관련성이 거의 없지만, 두 가지의 공통적인 특징을 갖고 있다. 이러한 질환들은 만성적이며, 종종 간헐적인 <a class="mw-disambig" href="/wiki/%EA%B3%A0%ED%86%B5" title="고통">고통</a>을 동반한다. 또한, 이러한 질환들은 매우 흔히 발생하며, 치료하기가 어렵다.</p>
<h2><span class="mw-headline" id=".EC.B9.98.EB.A3.8C.EB.B2.95">치료법</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%A5%98%EB%A8%B8%ED%8B%B0%EC%A6%98&amp;action=edit&amp;section=2" title="부분 편집: 치료법">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>과거에는 류머티즘을 치료하는 데 <a class="mw-redirect" href="/wiki/%EC%95%BD%EC%B4%88" title="약초">약초</a>를 이용한 여러 가지 치료법이 권장되었고, 현대 의학에서는 각기 다른 류머티즘 질환은 각기 다른 원인이나 복합적인 원인으로 인해 발생한다고 인식하며, 이에 따라 각기 다른 치료법을 요하고 있다.</p>
<p>그럼에도 불구하고, 주요 류머티즘 질환의 초기 치료에는 <a href="/wiki/%EC%95%84%EC%84%B8%ED%8A%B8%EC%95%84%EB%AF%B8%EB%85%B8%ED%8E%9C" title="아세트아미노펜">아세트아미노펜</a>이나 <a class="mw-redirect" href="/wiki/NSAIDs" title="NSAIDs">NSAIDs</a>이 포함된 <a href="/wiki/%EC%9D%B4%EB%B6%80%ED%94%84%EB%A1%9C%ED%8E%9C" title="이부프로펜">이부프로펜</a>이나 <a class="new" href="/w/index.php?title=%EB%94%94%ED%81%B4%EB%A1%9C%ED%8E%98%EB%82%99&amp;action=edit&amp;redlink=1" title="디클로페낙 (없는 문서)">디클로페낙</a> 등의 <a href="/wiki/%EC%A7%84%ED%86%B5%EC%A0%9C" title="진통제">진통제</a>가 동반되며, 때때로 더 강한 진통제를 필요로 한다.</p>
<h2><span class="mw-headline" id=".EB.A5.98.EB.A8.B8.ED.8B.B0.EC.A6.98.EA.B3.BC_.EB.82.A0.EC.94.A8.EC.99.80.EC.9D.98_.EC.83.81.EA.B4.80.EC.84.B1">류머티즘과 날씨와의 상관성</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%A5%98%EB%A8%B8%ED%8B%B0%EC%A6%98&amp;action=edit&amp;section=3" title="부분 편집: 류머티즘과 날씨와의 상관성">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>과거부터 류머티즘 질환의 <a class="mw-disambig" href="/wiki/%EA%B3%A0%ED%86%B5" title="고통">고통</a>과 날씨와의 관련성에 대한 많은 가설이 있었으나, 그러한 가설을 뒷밭침하거나 가설에 반대되는 확실한 증거가 나오지 않았다. <a href="/wiki/1995%EB%85%84" title="1995년">1995년</a> <a class="new" href="/w/index.php?title=%EB%B8%8C%EB%A6%AC%EA%B2%80_%EC%97%AC%EC%84%B1%EB%B3%91%EC%9B%90&amp;action=edit&amp;redlink=1" title="브리검 여성병원 (없는 문서)">브리검 여성병원</a>의 통증 관리 센터의 A. Naser와 동료들이 557명의 사람들을 대상으로 <a class="mw-redirect" href="/wiki/%EC%84%A4%EB%AC%B8_%EC%A1%B0%EC%82%AC" title="설문 조사">설문 조사</a>를 하였고, 다음과 같은 결론을 내리게 되었다.<sup class="reference" id="cite_ref-titlePressure_as_main_link_between_weather_and_pain._1-0"><a href="#cite_note-titlePressure_as_main_link_between_weather_and_pain.-1">[1]</a></sup></p>
<table align="center" style="border-collapse:collapse; border-style:none; background-color:transparent;">
<tr>
<td style="color:#90B5F0; width: 20px; vertical-align: top; font-size:40px;font-family:Georgia;font-weight:bold;text-align:left;">“</td>
<td style="padding:10px; vertical-align: top;"><span lang="en" xml:lang="en">"류머티즘 질환의 고통과 날씨와의 관련성에서의 주 문제는 <a href="/wiki/%EB%8C%80%EA%B8%B0%EC%95%95" title="대기압">대기압</a>의 변화이다. <a href="/wiki/%EC%A0%80%EA%B8%B0%EC%95%95" title="저기압">저기압</a>은 일반적으로 춥고 비오는 날씨와 관련이 있으며, 류머티즘 질환의 고통을 증대시킨다. <a href="/wiki/%EA%B3%A0%EA%B8%B0%EC%95%95" title="고기압">고기압</a>은 일반적으로 맑고 건조한 날씨와 관련이 있으며, 류머티즘 질환의 고통을 약화시킨다."</span></td>
<td style="color:#A0B7F2; width: 20px; vertical-align: bottom; font-size:40px;font-family:Georgia;font-weight:bold;text-align:right;">”</td>
</tr>
</table>
