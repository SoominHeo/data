<p><b>류큐 대학</b>(<span style="font-size: smaller;"><a href="/wiki/%EC%9D%BC%EB%B3%B8%EC%96%B4" title="일본어">일본어</a>:</span> <span lang="ja" xml:lang="ja">琉球大学</span>, <span lang="en" xml:lang="en">University of the Ryukyus</span>)은 <a href="/wiki/%EC%9D%BC%EB%B3%B8" title="일본">일본</a>의 국립 대학이다. 대학 본부는 <a href="/wiki/%EC%98%A4%ED%82%A4%EB%82%98%EC%99%80_%ED%98%84" title="오키나와 현">오키나와 현</a> 나카가미 군(中頭郡) <a href="/wiki/%EB%8B%88%EC%8B%9C%ED%95%98%EB%9D%BC_%EC%A0%95" title="니시하라 정">니시하라 정</a>(西原町)에 있다.</p>

<h2><span class="mw-headline" id=".EC.97.B0.ED.98.81">연혁</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%A5%98%ED%81%90_%EB%8C%80%ED%95%99&amp;action=edit&amp;section=1" title="부분 편집: 연혁">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>류큐 대학은 <a href="/wiki/1950%EB%85%84" title="1950년">1950년</a>에 <a class="mw-redirect" href="/wiki/%EB%AF%B8%EA%B5%AD%EC%9D%98_%EA%B5%B0%EC%82%AC" title="미국의 군사">미국군</a> 정부에 의해 설립되었다. <a href="/wiki/1966%EB%85%84" title="1966년">1966년</a>에 류큐 정부립(琉球政府立)이 되어 <a href="/wiki/1972%EB%85%84" title="1972년">1972년</a>에 일본 국립 대학이 되었다. 최초의 캠퍼스는 <a href="/wiki/%EC%8A%88%EB%A6%AC_%EC%84%B1" title="슈리 성">슈리 성</a>내에 있었지만, <a href="/wiki/1984%EB%85%84" title="1984년">1984년</a>에 현 캠퍼스로 이전되었다.</p>
<ul>
<li><a href="/wiki/1946%EB%85%84" title="1946년">1946년</a> 1월: 현 <a href="/wiki/%EC%9A%B0%EB%A3%A8%EB%A7%88_%EC%8B%9C" title="우루마 시">우루마 시</a>에 오키나와 문교학교(沖繩文敎學校) 설립.
<dl>
<dd>학과: 사범부, 외어부, 농림부.</dd>
</dl>
</li>
<li>1946년 9월 : 외어부가 오키나와 외국어학교(沖繩外國語學校)로서 독립.</li>
<li>1950년 5월 : <b>류큐 대학</b> 설립. 오키나와 문교학교, 오키나와 외국어학교를 통합.
<dl>
<dd>학부: 영어학부, 교육학부, 사회과학부, 농학부, 이학부, 응용학예학부.</dd>
</dl>
</li>
<li><a href="/wiki/1951%EB%85%84" title="1951년">1951년</a> 9월 : 임학부 설치.</li>
<li><a href="/wiki/1952%EB%85%84" title="1952년">1952년</a> 4월 : 영어학부를 어학부로 개명. 응용학예학부를 분리: 상학부, 가정학부.</li>
<li>1952년 5월 : <a href="/wiki/%EC%95%84%EB%A7%88%EB%AF%B8%EC%98%A4_%EC%84%AC" title="아마미오 섬">아마미오 섬</a>에 분교를 설치 (<a href="/wiki/1953%EB%85%84" title="1953년">1953년</a> 12월 폐지).</li>
<li>1966년 7월 : 류큐 정부립이 됨.</li>
<li><a href="/wiki/1967%EB%85%84" title="1967년">1967년</a> 4월 : 학부 개편: 법문하구, 교육학부, 이공학부, 농학부.</li>
<li><a href="/wiki/1968%EB%85%84" title="1968년">1968년</a> 5월 : 보건학부 설치.</li>
<li>1972년 5월 : 일본 국립 대학이 됨.</li>
<li><a href="/wiki/1979%EB%85%84" title="1979년">1979년</a> 4월 : 이공학부를 분리: 이학부, 공학부.</li>
<li>1979년 10월 : 의학부 설치.</li>
<li><a href="/wiki/2008%EB%85%84" title="2008년">2008년</a> 4월 : 관광산업과학부 설치.</li>
</ul>
