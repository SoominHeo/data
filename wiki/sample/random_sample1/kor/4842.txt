<p><b>르네 프레데리크 통</b>(<span style="font-size: smaller;"><a href="/wiki/%ED%94%84%EB%9E%91%EC%8A%A4%EC%96%B4" title="프랑스어">프랑스어</a>:</span> <span lang="fr" xml:lang="fr">René Frédéric Thom</span> <small><a href="/wiki/%EC%9C%84%ED%82%A4%EB%B0%B1%EA%B3%BC:IPA" title="위키백과:IPA">IPA</a>: </small><span class="IPA" title="IPA 발음">[ʁəne fʁedeʁik tõ]</span>, <a href="/wiki/1923%EB%85%84" title="1923년">1923년</a> <a href="/wiki/9%EC%9B%94_2%EC%9D%BC" title="9월 2일">9월 2일</a> - <a href="/wiki/2002%EB%85%84" title="2002년">2002년</a> <a href="/wiki/10%EC%9B%94_25%EC%9D%BC" title="10월 25일">10월 25일</a>)은 <a href="/wiki/%ED%94%84%EB%9E%91%EC%8A%A4" title="프랑스">프랑스</a>의 <a href="/wiki/%EC%88%98%ED%95%99%EC%9E%90" title="수학자">수학자</a>이다. <a href="/wiki/%EB%B3%B4%EC%B6%A9_%EA%B2%BD%EA%B3%84" title="보충 경계">보충 경계</a> 이론을 발견한 공로로 <a href="/wiki/1958%EB%85%84" title="1958년">1958년</a>에 <a href="/wiki/%ED%95%84%EC%A6%88%EC%83%81" title="필즈상">필즈상</a>을 수상하였다. <a class="new" href="/w/index.php?title=%EA%B8%89%EB%B3%80_%EC%9D%B4%EB%A1%A0&amp;action=edit&amp;redlink=1" title="급변 이론 (없는 문서)">급변 이론</a>(<span style="font-size: smaller;"><a href="/wiki/%EC%98%81%EC%96%B4" title="영어">영어</a>:</span> <span lang="en" xml:lang="en">catastrophy theory</span>)이라는 독창적인 방법의 기초를 닦은 수학자로 잘 알려져 있다.</p>

<h2><span class="mw-headline" id=".EC.83.9D.EC.95.A0">생애</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%A5%B4%EB%84%A4_%ED%86%B5&amp;action=edit&amp;section=1" title="부분 편집: 생애">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>통은 <a href="/wiki/1923%EB%85%84" title="1923년">1923년</a> 프랑스의 작은 도시 <a href="/wiki/%EB%AA%BD%EB%B2%A8%EB%A6%AC%EC%95%84%EB%A5%B4" title="몽벨리아르">몽벨리아르</a>에서 태어났다. 생-루이 고교(Lycée Saint-Louis)와 <a class="mw-redirect" href="/wiki/%ED%8C%8C%EB%A6%AC_%EC%8B%9C" title="파리 시">파리</a>의 <a href="/wiki/%EA%B3%A0%EB%93%B1%EC%82%AC%EB%B2%94%ED%95%99%EA%B5%90" title="고등사범학교">고등사범학교</a>에서 수학했다. 그는 <a href="/wiki/%EB%8B%88%EC%BD%9C%EB%9D%BC_%EB%B6%80%EB%A5%B4%EB%B0%94%ED%82%A4" title="니콜라 부르바키">니콜라 부르바키</a>의 창시자 중 하나인 <a href="/wiki/%EC%95%99%EB%A6%AC_%EC%B9%B4%EB%A5%B4%ED%83%95" title="앙리 카르탕">앙리 카르탕</a>을 통해 <a href="/wiki/%EC%9C%84%EC%83%81%EC%88%98%ED%95%99" title="위상수학">위상수학</a>에 입문했는데,<sup class="reference" id="cite_ref-.EA.B9.80.EC.9B.90.EA.B8.B0_1-0"><a href="#cite_note-.EA.B9.80.EC.9B.90.EA.B8.B0-1">[1]</a></sup> 그를 지도교수로 하여 1951년 <a href="/wiki/%ED%8C%8C%EB%A6%AC_%EB%8C%80%ED%95%99%EA%B5%90" title="파리 대학교">파리 대학교</a>에서 박사학위를 받았다. 그때의 박사학위 논문에는 훗날 그에게 <a href="/wiki/%ED%95%84%EC%A6%88%EC%83%81" title="필즈상">필즈상</a>을 선사하는 <a href="/wiki/%EB%B3%B4%EC%B6%A9_%EA%B2%BD%EA%B3%84" title="보충 경계">보충 경계</a> 이론의 기본 아이디어가 이미 제시되어 있었다. 르네는 지도교 수 <a href="/wiki/%EC%95%99%EB%A6%AC_%EC%B9%B4%EB%A5%B4%ED%83%95" title="앙리 카르탕">앙리 카르탕</a>을 비롯한 엄격하고 형식주의적인 <a href="/wiki/%EB%8B%88%EC%BD%9C%EB%9D%BC_%EB%B6%80%EB%A5%B4%EB%B0%94%ED%82%A4" title="니콜라 부르바키">니콜라 부르바키</a>로부터 많은 것을 배웠다. 그러나 자유분방하며 직관적인 성향의 그는 이러한 경향에 완전히 동조하지 않았는데, 이는 나중에 그가 수학을 떠나게 된 원인이 되었을 것이다. 한편 그는 <a href="/wiki/%EA%B7%B8%EB%A5%B4%EB%85%B8%EB%B8%94" title="그르노블">그르노블</a>(1953년~1954년)과 <a href="/wiki/%EC%8A%A4%ED%8A%B8%EB%9D%BC%EC%8A%A4%EB%B6%80%EB%A5%B4_%EB%8C%80%ED%95%99%EA%B5%90" title="스트라스부르 대학교">스트라스부르 대학교</a>(1954년~1963년)에서 교수직에 임용되어 수학을 가르쳤다. 1964년 파리 근교 뷔르쉬르이베트에 위치한 연구소 <a href="/wiki/IH%C3%89S" title="IHÉS">IHÉS</a>로 옮겨 번거로운 의무에 시달리지 않고 죽을 때까지 자유롭게 연구할 수 있게 되었다. 1976년에 그는 <a href="/wiki/%ED%94%84%EB%9E%91%EC%8A%A4_%EA%B3%BC%ED%95%99_%EC%95%84%EC%B9%B4%EB%8D%B0%EB%AF%B8" title="프랑스 과학 아카데미">프랑스 과학 아카데미</a> 회원이 되었다.</p>
<p>그의 초창기 연구 분야는 <a class="new" href="/w/index.php?title=%EB%AF%B8%EB%B6%84_%EC%9C%84%EC%83%81%EC%88%98%ED%95%99&amp;action=edit&amp;redlink=1" title="미분 위상수학 (없는 문서)">미분 위상수학</a>이었다. 이 분야에서 그는 나중에 <a href="/wiki/%ED%86%B5_%EA%B3%B5%EA%B0%84" title="통 공간">통 공간</a>(Thom space), <a class="mw-redirect" href="/wiki/%ED%86%B5_%ED%8A%B9%EC%84%B1%EB%A5%98" title="통 특성류">통 특성류</a>(Thom characteristic class)로 불리는 수학적 대상들을 연구했고, <a class="new" href="/w/index.php?title=%ED%86%B5_%ED%9A%A1%EB%8B%A8_%EC%A0%95%EB%A6%AC&amp;action=edit&amp;redlink=1" title="통 횡단 정리 (없는 문서)">통 횡단 정리</a>(Thom transversality theorem) 같은 중요한 결과들을 남겼다. <a href="/wiki/%EA%B2%8C%EC%9D%B4%EC%A7%80_%EC%9D%B4%EB%A1%A0" title="게이지 이론">게이지 이론</a>을 사용하여 조사된 <a class="new" href="/w/index.php?title=%ED%86%B5_%EC%B6%94%EC%B8%A1&amp;action=edit&amp;redlink=1" title="통 추측 (없는 문서)">통 추측</a>(Thom conjecture)은 그의 업적의 또다른 예이다. 특히 1954년 그가 밝힌 <a href="/wiki/%EB%B3%B4%EC%B6%A9_%EA%B2%BD%EA%B3%84" title="보충 경계">보충 경계</a> 이론으로 1958년에 <a href="/wiki/%ED%95%84%EC%A6%88%EC%83%81" title="필즈상">필즈상</a>을 수상한다. 러시아 수학자인 <a href="/wiki/%EB%A0%88%ED%94%84_%ED%8F%B0%ED%8A%B8%EB%9E%B4%EA%B8%B4" title="레프 폰트랴긴">레프 폰트랴긴</a>은 어떤 조건하에서 콤팩트한 n차원 표면이 어떤 n+1차원 표면의 경계가 될 수 있는지를 결정해야 했다. 그가 발견한 이 조건이 충분하다는 것을 입증한 것이 바로 <a href="/wiki/%EB%B3%B4%EC%B6%A9_%EA%B2%BD%EA%B3%84" title="보충 경계">보충 경계</a> 이론이다. 필즈상 위원회가 공식적으로 언급한 공헌은 다음과 같다.</p>

<p><a href="/wiki/%ED%95%98%EC%9D%B8%EC%B8%A0_%ED%98%B8%ED%94%84" title="하인츠 호프">하인츠 호프</a>는 통에게 <a href="/wiki/%ED%95%84%EC%A6%88%EC%83%81" title="필즈상">필즈상</a>을 직접 시상하였으며, 통에 대하여 "그의 업적이 매우 직관적이면서도 간결하고 풍성한 것"이라고 말했다.<sup class="reference" id="cite_ref-.EA.B9.80.EC.9B.90.EA.B8.B0_1-1"><a href="#cite_note-.EA.B9.80.EC.9B.90.EA.B8.B0-1">[1]</a></sup></p>
<p>차후 그는 <a class="new" href="/w/index.php?title=%ED%8A%B9%EC%9D%B4%EC%A0%90_%EC%9D%B4%EB%A1%A0&amp;action=edit&amp;redlink=1" title="특이점 이론 (없는 문서)">특이점 이론</a>(singularity theory)으로 전향하는데, 사실 <a class="new" href="/w/index.php?title=%EA%B8%89%EB%B3%80_%EC%9D%B4%EB%A1%A0&amp;action=edit&amp;redlink=1" title="급변 이론 (없는 문서)">급변 이론</a>(catastrophe theory)도 이 특이점 이론의 특수한 경우라고 할 수 있다. 1960년에서 1969년 사이 발표했던 심층적인 논문에서 계층화된 동위체 이론을 발전시켰다. 1960년에 통이 독일 <a class="mw-disambig" href="/wiki/%EB%B3%B8" title="본">본</a>에서 미분가능한 사상의 안정성에 관해 강의한 내용은 1969년에서 1970년 1년간 리버폴 대학에서 열린 특이점에 관한 심포지엄 와중에 출판되기도 했으며, 그의 아이디어를 이용하여 훗날 다른 수학자가 새로운 정리를 증명하기도 했다.</p>
<p><a href="/wiki/1968%EB%85%84" title="1968년">1968년</a>부터 <a href="/wiki/1972%EB%85%84" title="1972년">1972년</a> 사이에 통은 생물학과 사회학에 자주 등장하는 형태 변화를 수학적으로 나타낼 수 있는 방법을 연구하여, 1972년 발표한 책 《구조적 안정성과 <a class="new" href="/w/index.php?title=%ED%98%95%ED%83%9C%ED%98%95%EC%84%B1&amp;action=edit&amp;redlink=1" title="형태형성 (없는 문서)">형태형성</a>》(<span style="font-size: smaller;"><a href="/wiki/%EC%98%81%EC%96%B4" title="영어">영어</a>:</span> <span lang="en" xml:lang="en">Structural Stability and Morphogenesis</span>)<sup class="reference" id="cite_ref-2"><a href="#cite_note-2">[2]</a></sup> 에서 <a class="new" href="/w/index.php?title=%EA%B8%89%EB%B3%80_%EC%9D%B4%EB%A1%A0&amp;action=edit&amp;redlink=1" title="급변 이론 (없는 문서)">급변 이론</a>을 주장하였다. "급변" 이론이란 이름에서 알 수 있듯 이는 급격한 변화를 기술하는 새로운 관점을 제시한 이론으로서 학계에 큰 충격을 주었다. 그러나 정량적 예측을 이끌어내기 어려운 이론이라 다른 수학자들의 많은 비판을 받았다. 그는 말년에 전문적인 수학 연구에서 멀어져 보다 더 일반적인 철학적 담론에 치중하였다. 급변(catastrophe)의 구조를 통해 과학의 <a href="/wiki/%EC%9D%B8%EC%8B%9D%EB%A1%A0" title="인식론">인식론</a>적, 철학적 문제에 대해 많은 저술을 남겼으며, <a href="/wiki/%EC%95%84%EB%A6%AC%EC%8A%A4%ED%86%A0%ED%85%94%EB%A0%88%EC%8A%A4" title="아리스토텔레스">아리스토텔레스</a>의 저술을 재평가하기도 했다. 그가 수학에서 멀어진 하나의 요인은 <a href="/wiki/%EC%95%8C%EB%A0%89%EC%82%B0%EB%8D%94_%EA%B7%B8%EB%A1%9C%ED%85%90%EB%94%94%ED%81%AC" title="알렉산더 그로텐디크">알렉산더 그로텐디크</a>의 존재였다. 그로텐디크의 현란한 수학적 재능과 업적은 자신으로 하여금 다른 분야에 더 많은 관심을 갖게 했다고 통은 여러 번 말하곤 했다.</p>
