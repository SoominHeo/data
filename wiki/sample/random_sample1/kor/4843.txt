<p><b>르네-로베르 카블리에 드 라 살</b>(Rene-Robert Cavelier de La Salle, <a href="/wiki/1643%EB%85%84" title="1643년">1643년</a> <a href="/wiki/11%EC%9B%94_21%EC%9D%BC" title="11월 21일">11월 21일</a> ~ <a href="/wiki/1687%EB%85%84" title="1687년">1687년</a> <a href="/wiki/3%EC%9B%94_19%EC%9D%BC" title="3월 19일">3월 19일</a>)은 <a href="/wiki/%ED%94%84%EB%9E%91%EC%8A%A4" title="프랑스">프랑스</a>의 탐험가이다. 그는 유럽인 처음으로 <a href="/wiki/%EB%AF%B8%EC%8B%9C%EC%8B%9C%ED%94%BC_%EA%B0%95" title="미시시피 강">미시시피 강</a>을 통하여 <a href="/wiki/%EB%A9%95%EC%8B%9C%EC%BD%94_%EB%A7%8C" title="멕시코 만">멕시코 만</a>을 여행하였고, 미시시피 강 유역의 전체를 프랑스의 영토로 삼았다. 그의 많은 탐험은 프랑스의 신대륙에 큰 제국을 건설하는 희망에 힘을 주었다.</p>

<h2><span class="mw-headline" id=".EC.B4.88.EA.B8.B0_.EC.83.9D.EC.95.A0">초기 생애</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%A5%B4%EB%84%A4%EB%A1%9C%EB%B2%A0%EB%A5%B4_%EC%B9%B4%EB%B8%94%EB%A6%AC%EC%97%90_%EB%93%9C_%EB%9D%BC_%EC%82%B4&amp;action=edit&amp;section=1" title="부분 편집: 초기 생애">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>르네-로베르 카블리에는 <a href="/wiki/%EB%A3%A8%EC%95%99" title="루앙">루앙</a>에서 출생하여 자기 가족의 사유지의 이름을 따 라살이라고 이름을 지었다. 어린 시절에 <a href="/wiki/%EC%98%88%EC%88%98%ED%9A%8C" title="예수회">예수회</a>의 성직자들이 운영하는 <a href="/wiki/%ED%95%99%EA%B5%90" title="학교">학교</a>에 다니며 예수회 교인이 되는 교육을 받았다. <a href="/wiki/1655%EB%85%84" title="1655년">1655년</a> 종교적 훈련을 떠나 모험의 길을 찾았다. 다음해에 프랑스가 <a href="/wiki/%EC%8B%9D%EB%AF%BC%EC%A7%80" title="식민지">식민지</a>를 설립한 <a href="/wiki/%EC%BA%90%EB%82%98%EB%8B%A4" title="캐나다">캐나다</a>에 간다.</p>
<p><a href="/wiki/%EB%AA%AC%ED%8A%B8%EB%A6%AC%EC%98%AC" title="몬트리올">몬트리올</a> 근처에 약간의 땅을 회득하여 번영하는 <a href="/wiki/%EB%AA%A8%ED%94%BC" title="모피">모피</a> <a href="/wiki/%EC%83%81%EC%9D%B8" title="상인">상인</a>이 되었다. <a href="/wiki/%EB%AF%B8%EC%8B%9C%EC%8B%9C%ED%94%BC_%EA%B0%95" title="미시시피 강">미시시피 강</a>과 <a href="/wiki/%EC%98%A4%ED%95%98%EC%9D%B4%EC%98%A4_%EA%B0%95" title="오하이오 강">오하이오 강</a>의 위치를 알려준 <a class="mw-redirect" href="/wiki/%EC%9D%B8%EB%94%94%EC%96%B8" title="인디언">인디언</a>들과 많은 교역을 하였다. 인디언들은 이 두개의 강이 바다로 흘러간다는 걸 믿었고, 라살은 이 강들이 <a href="/wiki/%EB%B6%81%EC%95%84%EB%A9%94%EB%A6%AC%EC%B9%B4" title="북아메리카">북아메리카</a> 대륙을 거쳐 <a href="/wiki/%ED%83%9C%ED%8F%89%EC%96%91" title="태평양">태평양</a>으로 간다는 생각이 들었다. <a href="/wiki/1669%EB%85%84" title="1669년">1669년</a> 그의 땅을 팔고 강들을 찾으러 나섰다.</p>
<h2><span class="mw-headline" id=".EC.B2.AB_.ED.83.90.ED.97.98">첫 탐험</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%A5%B4%EB%84%A4%EB%A1%9C%EB%B2%A0%EB%A5%B4_%EC%B9%B4%EB%B8%94%EB%A6%AC%EC%97%90_%EB%93%9C_%EB%9D%BC_%EC%82%B4&amp;action=edit&amp;section=2" title="부분 편집: 첫 탐험">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p><a href="/wiki/1669%EB%85%84" title="1669년">1669년</a>부터 <a href="/wiki/1673%EB%85%84" title="1673년">1673년</a> 라살은 북아메리카의 막대한 내부를 돌아다녔다. 역사가들은 그가 현재의 <a class="mw-redirect" href="/wiki/%EC%98%A4%ED%95%98%EC%9D%B4%EC%98%A4" title="오하이오">오하이오</a> 지역만큼의 먼 여행을 했다고 믿는다. 이 여행이 끝날 무렵, 미시시피 강이 멕시코 만으로 흘러간다고 확신시켰다.</p>
<p><a href="/wiki/1674%EB%85%84" title="1674년">1674년</a> 프랑스로 돌아갔고, <a href="/wiki/%EB%A3%A8%EC%9D%B4_14%EC%84%B8" title="루이 14세">루이 14세</a>는 현재 <a href="/wiki/%EC%98%A8%ED%83%80%EB%A6%AC%EC%98%A4_%EC%A3%BC" title="온타리오 주">온타리오 주</a> <a href="/wiki/%ED%82%B9%EC%8A%A4%ED%84%B4_(%EC%98%A8%ED%83%80%EB%A6%AC%EC%98%A4_%EC%A3%BC)" title="킹스턴 (온타리오 주)">킹스턴</a>의 대지인 프론테나크 <a href="/wiki/%EC%9A%94%EC%83%88" title="요새">요새</a>(Fort Frontenac)를 포함한 땅을 라살에게 주었다. 라살은 요새에 모피 교역지를 설립하였고, 곧 캐나다에서 가장 힘있는 사람들 중의 하나가 된다. <a href="/wiki/1677%EB%85%84" title="1677년">1677년</a> 다시 프랑스로 돌아가 루이 14세에게 미시시피 강 탐험을 허락받았다.</p>
<p><a href="/wiki/1679%EB%85%84" title="1679년">1679년</a> 캐나다로 돌아가 프랑스의 <a href="/wiki/%EC%98%A4%EB%8C%80%ED%98%B8" title="오대호">오대호</a> 지방을 지배하는 여행에 진출하였다. <a href="/wiki/1680%EB%85%84" title="1680년">1680년</a> 그는 현재의 <a class="mw-redirect" href="/wiki/%EC%9D%BC%EB%A6%AC%EB%85%B8%EC%9D%B4" title="일리노이">일리노이</a> 지역에 유럽인들의 첫 정착지를 설립하였다. <a href="/wiki/%EC%9D%BC%EB%A6%AC%EB%85%B8%EC%9D%B4_%EA%B0%95" title="일리노이 강">일리노이 강</a>에 서있고, 현재의 페오리아(Peoria) 근처에 크레브쿠외르 요새(Fort Crevecouer)라고 이름을 지었다. 이 요새를 지은 후, 많은 공급을 위해 캐나다로 돌아갔다.</p>
<h2><span class="mw-headline" id=".EB.A9.95.EC.8B.9C.EC.BD.94_.EB.A7.8C.EC.9C.BC.EB.A1.9C_.EC.A7.84.EC.B6.9C">멕시코 만으로 진출</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%A5%B4%EB%84%A4%EB%A1%9C%EB%B2%A0%EB%A5%B4_%EC%B9%B4%EB%B8%94%EB%A6%AC%EC%97%90_%EB%93%9C_%EB%9D%BC_%EC%82%B4&amp;action=edit&amp;section=3" title="부분 편집: 멕시코 만으로 진출">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p><a href="/wiki/1681%EB%85%84" title="1681년">1681년</a> 후순에 일리노이 지방으로 돌아갔는 데, 크레브쿠외르 요새는 그곳의 거주민들의 반란에 의하여 파괴되었다. 그러나 라살은 그 일을 밀고 나갔고 그는 일리노이 강에서 20명의 프랑스인과 약 30명의 인디언을 이끌며 <a href="/wiki/%EC%B9%B4%EB%88%84" title="카누">카누</a>를 타고 미시시피 강으로 내려갔다.</p>
<p><a href="/wiki/1682%EB%85%84" title="1682년">1682년</a> <a href="/wiki/2%EC%9B%94_13%EC%9D%BC" title="2월 13일">2월 13일</a> 미시시피 강에서 진출이 시작되어 <a href="/wiki/4%EC%9B%94_9%EC%9D%BC" title="4월 9일">4월 9일</a> <a href="/wiki/%EB%A9%95%EC%8B%9C%EC%BD%94_%EB%A7%8C" title="멕시코 만">멕시코 만</a>에 도달하였다. 미시시피 강의 입구 근처에 십자가와 프랑스의 문장이 든 덧옷을 운반한 기둥을 세웠다. 그는 모든 대지가 미시시피 강에 의하여 배수되었고, 프랑스를 위해 그 공물을 바치는 요구하였다. 이 지방은 <a class="mw-redirect" href="/wiki/%EC%95%A0%ED%8E%84%EB%A0%88%EC%9D%B4%EC%B9%98%EC%96%B8_%EC%82%B0%EC%A4%84%EA%B8%B0" title="애펄레이치언 산줄기">애펄레이치언 산줄기</a>로부터 서부의 <a href="/wiki/%EB%A1%9C%ED%82%A4_%EC%82%B0%EB%A7%A5" title="로키 산맥">로키 산맥</a>의 동부에 펼쳐져 있으며, 북쪽의 오대호에서 남쪽의 멕시코 만까지 뻗어있다. 라살은 루이 14세의 명예를 가려 이 지방을 <a class="mw-redirect" href="/wiki/%EB%A3%A8%EC%9D%B4%EC%A7%80%EC%95%A0%EB%82%98" title="루이지애나">루이지애나</a>라고 명하였다.</p>
<p>그해의 후반에 일리노이 강에 걸쳐있는 <a href="/wiki/%EC%A0%88%EB%B2%BD" title="절벽">절벽</a>에 생루이 요새(Fort Saint Louis)를 건설하였다. 그 절벽은 지금의 굶주린 바위 주립 공원(Starved Rock State Park)이다. <a href="/wiki/1683%EB%85%84" title="1683년">1683년</a> 후반에 식민지를 위한 공급원과 정착자들을 데리러 프랑스로 떠났다.</p>
<p><a href="/wiki/1684%EB%85%84" title="1684년">1684년</a> 라살은 프랑스에서 300명 이상의 식민자들을 태운 4척의 배를 거느리고 멕시코 만으로 항해하였으나 이 진출은 라살이 실수로 시간이 지나도록 항해를 한 이유로 미시시피 강에 도달하지 못했다. <a href="/wiki/1685%EB%85%84" title="1685년">1685년</a> 그는 강의 서부 유역에 식민지를 세웠는데 이 대지는 현재 <a href="/wiki/%ED%85%8D%EC%82%AC%EC%8A%A4_%EC%A3%BC" title="텍사스 주">텍사스 주</a> 코퍼스 크리스티 지역에서 동부로 약 80마일 떨어져있는 마타고르다 만(Matagorda Bay) 근처이다.</p>
<p>하지만 인디언들은 새 정착지들을 위협하였고, 많은 식민자들이 질병으로 사망하였고 <a href="/wiki/1687%EB%85%84" title="1687년">1687년</a> 식민지는 절망적으로 도움이 필요하였다. 라살과 그의 선원들 몇명은 미시시피 강을 향해 행진하였으나 강을 찾지 못하였고 결국 자객들이 라살의 조카를 죽였으며 자신도 <a href="/wiki/3%EC%9B%94_19%EC%9D%BC" title="3월 19일">3월 19일</a>에 살해당하였다.</p>
