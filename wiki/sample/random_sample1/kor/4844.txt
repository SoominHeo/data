<p><b>르네사스 테크놀로지 사</b> (<span style="font-size: smaller;"><a href="/wiki/%EC%9D%BC%EB%B3%B8%EC%96%B4" title="일본어">일본어</a>:</span> <span lang="ja" xml:lang="ja">ルネサス テクノロジ</span>)는 <a href="/wiki/%EC%9D%BC%EB%B3%B8" title="일본">일본</a> 정부와 함께, NEC,히타치,미쓰비시 등 19개 기업이 공동 출자하여 설립한 <a href="/wiki/%EB%B0%98%EB%8F%84%EC%B2%B4" title="반도체">반도체</a> 제조사이다. 르네사스 테크놀로지는 본사가 토쿄에 있으며 20여개 국가에서 제조, 설계 및 영업활동을 한다. 르네사스 테크놀로지는 2008년 당시 휴대전화, 자동차용 시스템 반도체의 제조사중 가장 큰 기업이며 마이크로컨트롤러 제조사중에도 가장 큰 기업이었다. 게다가 르네사스는 <a class="mw-disambig" href="/wiki/LCD" title="LCD">LCD</a> 드라이버, <a class="new" href="/w/index.php?title=%EB%AC%B4%EC%84%A0_%EC%A3%BC%ED%8C%8C%EC%88%98&amp;action=edit&amp;redlink=1" title="무선 주파수 (없는 문서)">RF</a> <a class="mw-redirect" href="/wiki/%EC%A7%91%EC%A0%81%ED%9A%8C%EB%A1%9C" title="집적회로">칩</a>, <a class="new" href="/w/index.php?title=%ED%98%BC%EC%84%B1%EC%8B%A0%ED%98%B8_%EC%A7%91%EC%A0%81%ED%9A%8C%EB%A1%9C&amp;action=edit&amp;redlink=1" title="혼성신호 집적회로 (없는 문서)">혼성신호 집적회로</a> 및 <a class="new" href="/w/index.php?title=%EC%B9%A9%EA%B8%B0%EB%B0%98_%EC%8B%9C%EC%8A%A4%ED%85%9C&amp;action=edit&amp;redlink=1" title="칩기반 시스템 (없는 문서)">칩기반 시스템</a>으로도 유명했다.</p>
<p>생산되는 반도체 소자의 제품군은 다음과 같다:</p>
<ul>
<li>시스템 <a class="mw-redirect" href="/wiki/LSI" title="LSI">LSI</a>
<ul>
<li>마이크로컴퓨터/(<a class="mw-redirect" href="/wiki/%EC%8A%A4%EB%A7%88%ED%8A%B8_%EC%B9%B4%EB%93%9C" title="스마트 카드">스마트 카드</a>가 내장된) <a href="/wiki/%EB%A7%88%EC%9D%B4%ED%81%AC%EB%A1%9C%EC%BB%A8%ED%8A%B8%EB%A1%A4%EB%9F%AC" title="마이크로컨트롤러">마이크로컨트롤러</a></li>
<li><a href="/wiki/%EC%A3%BC%EB%AC%B8%ED%98%95_%EB%B0%98%EB%8F%84%EC%B2%B4" title="주문형 반도체">주문형 반도체</a></li>
<li>논리칩</li>
<li>아날로그 반도체</li>
</ul>
</li>
<li>부품 소자 (<a href="/wiki/%EB%8B%A4%EC%9D%B4%EC%98%A4%EB%93%9C" title="다이오드">다이오드</a>, <a href="/wiki/%EC%A0%84%EB%A0%A5_MOSFET" title="전력 MOSFET">전력 MOSFET</a>, <a href="/wiki/%EC%82%AC%EC%9D%B4%EB%A6%AC%EC%8A%A4%ED%84%B0" title="사이리스터">사이리스터</a>, <a class="new" href="/w/index.php?title=%ED%8A%B8%EB%9D%BC%EC%9D%B4%EC%95%A1&amp;action=edit&amp;redlink=1" title="트라이액 (없는 문서)">트라이액</a>, <a href="/wiki/%ED%8A%B8%EB%9E%9C%EC%A7%80%EC%8A%A4%ED%84%B0" title="트랜지스터">트랜지스터</a>)</li>
<li>메모리 (<a href="/wiki/%ED%94%8C%EB%9E%98%EC%8B%9C_%EB%A9%94%EB%AA%A8%EB%A6%AC" title="플래시 메모리">플래시</a>, <a class="mw-redirect" href="/wiki/%EC%97%90%EC%8A%A4%EB%9E%A8" title="에스램">에스램</a>, <a class="new" href="/w/index.php?title=TTRAM&amp;action=edit&amp;redlink=1" title="TTRAM (없는 문서)">TTRAM</a>)</li>
</ul>
<p>르네사스 테크놀로지는 <a class="mw-redirect" href="/wiki/%EB%85%84%EB%8F%84%EB%B3%84_%EB%B0%98%EB%8F%84%EC%B2%B4_%ED%8C%90%EB%A7%A4_%EC%88%9C%EC%9C%84" title="년도별 반도체 판매 순위">세계 최대 반도체 기업중 20위</a>에 포함되는 기업다.</p>
<p>르네사스 테크놀로지는 <a href="/wiki/2003%EB%85%84" title="2003년">2003년</a> <a href="/wiki/4%EC%9B%94_1%EC%9D%BC" title="4월 1일">4월 1일</a>에 <a class="mw-redirect" href="/wiki/NEC" title="NEC">NEC</a> (5%), <a href="/wiki/%ED%9E%88%ED%83%80%EC%B9%98_%EC%A0%9C%EC%9E%91%EC%86%8C" title="히타치 제작소">히타치 제작소</a> (55%), <a class="new" href="/w/index.php?title=%EB%AF%B8%EC%93%B0%EB%B9%84%EC%8B%9C_%EC%A0%84%EC%9E%90&amp;action=edit&amp;redlink=1" title="미쓰비시 전자 (없는 문서)">미쓰비시 전자</a> (45%)의 합작으로 설립되었다. 2014년 4월 5일 현재, Innovation Network Corporation of Japan(INCJ)가 69.15%의 지분을 차지하고 있으며, 히타치(7.66%), 미쓰비시 전자(6.26%) 등이 지분을 가지고 있다.</p>
