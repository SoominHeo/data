<p><b>르완다 내전</b>은 <a href="/wiki/%EC%A4%91%EC%95%99%EC%95%84%ED%94%84%EB%A6%AC%EC%B9%B4" title="중앙아프리카">중앙아프리카</a>의 <a href="/wiki/%EB%A5%B4%EC%99%84%EB%8B%A4" title="르완다">르완다</a>에서 일어난 대통령 <a href="/wiki/%EC%A5%90%EB%B2%A0%EB%82%A0_%ED%95%98%EB%B8%8C%EC%9E%90%EB%A6%AC%EB%A7%88%EB%82%98" title="쥐베날 하브자리마나">쥐베날 하브자리마나</a> 정부군과 반정부군 <a class="new" href="/w/index.php?title=%EB%A5%B4%EC%99%84%EB%8B%A4_%EC%95%A0%EA%B5%AD_%EC%A0%84%EC%84%A0&amp;action=edit&amp;redlink=1" title="르완다 애국 전선 (없는 문서)">르완다 애국 전선</a> 사이의 내전이다. 르완다 애국 전선과의 첫 충돌은 1990년 10월 1일 시작되었으며, 1993년 8월 4일 과도 정부를 세우는 <a class="new" href="/w/index.php?title=%EC%95%84%EB%A3%A8%EC%83%A4_%ED%8F%89%ED%99%94_%ED%98%91%EC%A0%95&amp;action=edit&amp;redlink=1" title="아루샤 평화 협정 (없는 문서)">아루샤 평화 협정</a>의 서명으로 표면적으로 종전되었다.<sup class="reference" id="cite_ref-BBC_timeline_2-0"><a href="#cite_note-BBC_timeline-2">[2]</a></sup></p>
<p>그러나, 1994년 4월 <a class="new" href="/w/index.php?title=%EC%A5%90%EB%B2%A0%EB%82%A0_%ED%95%98%EB%B8%8C%EC%9E%90%EB%A6%AC%EB%A7%88%EB%82%98%EC%99%80_%EC%8B%9C%ED%94%84%EB%A6%AC%EC%95%B5_%EC%9D%80%ED%83%80%EB%9E%B4%EB%AF%B8%EB%9D%BC%EC%9D%98_%EC%95%94%EC%82%B4&amp;action=edit&amp;redlink=1" title="쥐베날 하브자리마나와 시프리앵 은타랴미라의 암살 (없는 문서)">쥐베날 하브자리마나의 암살</a>이 촉매가 되어 <a class="mw-redirect" href="/wiki/%EB%A5%B4%EC%99%84%EB%8B%A4_%EC%A7%91%EB%8B%A8_%ED%95%99%EC%82%B4" title="르완다 집단 학살">르완다 집단 학살</a>이 일어났으며, 이로 인해 약 80만명이 사망했다. 전쟁과 집단 학살의 밀접한 원인으로 인해, 일부에서는 대량 학살이 서로 다른 사건이기보다는 전쟁의 새로운 폭발이라고도 보고 있다. 결국 르완다 애국 전선이 공세에 나서 르완다를 점령했다. <a href="/wiki/%ED%9B%84%ED%88%AC%EC%A1%B1" title="후투족">후투족</a> 망명 정부는 주변 국가에서 <a class="new" href="/w/index.php?title=%EB%8C%80%ED%98%B8%EC%88%98_%EB%82%9C%EB%AF%BC_%EC%9C%84%EA%B8%B0&amp;action=edit&amp;redlink=1" title="대호수 난민 위기 (없는 문서)">난민 캠프</a>를 설치하면서 새로운 르완다 애국 전선의 정부를 불안하게 만들었다. 르완다 애국 전선 및 여러 반군들은 <a href="/wiki/%EC%A0%9C1%EC%B0%A8_%EC%BD%A9%EA%B3%A0_%EC%A0%84%EC%9F%81" title="제1차 콩고 전쟁">제1차 콩고 전쟁</a>(1996~1997)을 추진했으며, 이 전쟁은 이후 <a href="/wiki/%EC%A0%9C2%EC%B0%A8_%EC%BD%A9%EA%B3%A0_%EC%A0%84%EC%9F%81" title="제2차 콩고 전쟁">제2차 콩고 전쟁</a>(1998~2003)으로 번지면서 후투군이 르완다의 재점령을 목표로 이 전쟁에 참전하였다. 따라서, 내전은 공식적으로 1993년에 종료했지만, 일부 문헌에서는 르완다 애국 전선이 키갈리를 점령한 1994년에 종전했다고 생각하거나, 1996년 <a href="/wiki/%EB%82%9C%EB%AF%BC" title="난민">난민</a> 캠프 해산, 더욱으로는 르완다 국경을 따라 존재하는 소규모 게릴라전 등을 통해 계속해서 내전이 지속되어 있다고 말하기도 한다.</p>

<h2><span class="mw-headline" id=".EB.B0.B0.EA.B2.BD">배경</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%A5%B4%EC%99%84%EB%8B%A4_%EB%82%B4%EC%A0%84&amp;action=edit&amp;section=1" title="부분 편집: 배경">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>당시 <a href="/wiki/%EB%A5%B4%EC%99%84%EB%8B%A4" title="르완다">르완다</a>는 이 지역에 진출한 소수 14%의 <a href="/wiki/%ED%88%AC%EC%B9%98%EC%A1%B1" title="투치족">투치족</a>이 왕국을 세워 토착부족인 85%의 <a href="/wiki/%ED%9B%84%ED%88%AC%EC%A1%B1" title="후투족">후투족</a>을 지배했다. 그 후 <a href="/wiki/1919%EB%85%84" title="1919년">1919년</a> <a href="/wiki/%EB%B2%A8%EA%B8%B0%EC%97%90" title="벨기에">벨기에</a>가 <a href="/wiki/%EB%A5%B4%EC%99%84%EB%8B%A4" title="르완다">르완다</a>를 지배하고 <a href="/wiki/%ED%88%AC%EC%B9%98%EC%A1%B1" title="투치족">투치족</a>에 대한 차별정책으로 종족간의 갈등이 시작되었다. <a href="/wiki/%EB%B2%A8%EA%B8%B0%EC%97%90" title="벨기에">벨기에</a>는 투치족 출신 왕들을 내세워 다스렸으나 <a href="/wiki/%EB%B2%A8%EA%B8%B0%EC%97%90" title="벨기에">벨기에</a>가 물러간 후 후투족과 투치족의 갈등으로 <a href="/wiki/%EB%82%B4%EC%A0%84" title="내전">내전</a>이 시작되었다.</p>
<h2><span class="mw-headline" id=".EC.A0.84.EC.9F.81_.EA.B3.BC.EC.A0.95">전쟁 과정</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%A5%B4%EC%99%84%EB%8B%A4_%EB%82%B4%EC%A0%84&amp;action=edit&amp;section=2" title="부분 편집: 전쟁 과정">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p><a href="/wiki/1959%EB%85%84" title="1959년">1959년</a> 투치족과 후투족 사이의 전쟁이 일어나자 국왕 <a class="new" href="/w/index.php?title=%EB%AF%80%EC%99%80%EB%AF%B8_%ED%82%A4%EA%B2%8C%EB%A6%AC_5%EC%84%B8&amp;action=edit&amp;redlink=1" title="므와미 키게리 5세 (없는 문서)">므와미 키게리 5세</a>는 망명을 하고 <a href="/wiki/1961%EB%85%84" title="1961년">1961년</a> 1월 <a href="/wiki/%EB%A5%B4%EC%99%84%EB%8B%A4" title="르완다">르완다</a>는 공화국을 선포했다.</p>
<p>독립 후에 많은 투치족이 쫓겨나자 <a href="/wiki/1963%EB%85%84" title="1963년">1963년</a> 이웃나라 <a href="/wiki/%EB%B6%80%EB%A3%AC%EB%94%94" title="부룬디">부룬디</a>의 투치족이 르완다를 기습 공격하여 이에 대한 복수로 르완다의 투치족이 피해를 입기도 했다. 결국 <a href="/wiki/1973%EB%85%84" title="1973년">1973년</a> 투치족이 <a href="/wiki/%EC%BF%A0%EB%8D%B0%ED%83%80" title="쿠데타">쿠데타</a>를 일으켜 정권을 잡고 후투족 지도자들을 살해했다. 이에 후투족 지도자들이 반란을 일으켰고 살아남은 투치족들이 이웃 나라 <a href="/wiki/%EC%9A%B0%EA%B0%84%EB%8B%A4" title="우간다">우간다</a>로 도망쳐 <a href="/wiki/1990%EB%85%84" title="1990년">1990년</a> 우간다로 망명한 사람들 5,000명이 <a class="new" href="/w/index.php?title=%EB%A5%B4%EC%99%84%EB%8B%A4_%EC%95%A0%EA%B5%AD_%EC%A0%84%EC%84%A0&amp;action=edit&amp;redlink=1" title="르완다 애국 전선 (없는 문서)">르완다 애국 전선</a>(RPF)를 결성해 <a href="/wiki/%EB%A5%B4%EC%99%84%EB%8B%A4" title="르완다">르완다</a>를 침략했다. <a href="/wiki/1992%EB%85%84" title="1992년">1992년</a> 후투족 정부와 RPF와의 협상으로 <a href="/wiki/1993%EB%85%84" title="1993년">1993년</a> 8월 <a href="/wiki/%ED%83%84%EC%9E%90%EB%8B%88%EC%95%84" title="탄자니아">탄자니아</a> <a href="/wiki/%EC%95%84%EB%A3%A8%EC%83%A4" title="아루샤">아루샤</a>에서 <a class="new" href="/w/index.php?title=%EC%95%84%EB%A3%A8%EC%83%A4_%EC%A1%B0%EC%95%BD&amp;action=edit&amp;redlink=1" title="아루샤 조약 (없는 문서)">아루샤 조약</a>이 체결되었다.</p>
<p><a class="new" href="/w/index.php?title=%EC%95%84%EB%A3%A8%EC%83%A4_%EC%A1%B0%EC%95%BD&amp;action=edit&amp;redlink=1" title="아루샤 조약 (없는 문서)">아루샤 조약</a>은 범과도정부를 요구했으나 과도 정부는 성립되지 않았고 <a href="/wiki/1994%EB%85%84" title="1994년">1994년</a> <a href="/wiki/4%EC%9B%94_6%EC%9D%BC" title="4월 6일">4월 6일</a> 후투족 출신 <a href="/wiki/%EB%A5%B4%EC%99%84%EB%8B%A4" title="르완다">르완다</a> 대통령 <a href="/wiki/%EC%A5%90%EB%B2%A0%EB%82%A0_%ED%95%98%EB%B8%8C%EC%9E%90%EB%A6%AC%EB%A7%88%EB%82%98" title="쥐베날 하브자리마나">쥐베날 하브자리마나</a>와 <a href="/wiki/%EB%B6%80%EB%A3%AC%EB%94%94" title="부룬디">부룬디</a> 대통령 <a class="mw-redirect" href="/wiki/%EC%8B%9C%ED%94%84%EB%A6%AC%EC%95%99_%EC%9D%80%ED%83%80%EB%9E%B4%EB%AF%B8%EB%9D%BC" title="시프리앙 은타랴미라">시프리앙 은타랴미라</a>가 비행기 요격 사고로 수도 <a href="/wiki/%ED%82%A4%EA%B0%88%EB%A6%AC" title="키갈리">키갈리</a> 근처에서 살해당했다.</p>
<p>이 사고는 르완다 대통령 경호대의 소행으로 밝혀져 이튿날 <a href="/wiki/1994%EB%85%84" title="1994년">1994년</a> <a href="/wiki/4%EC%9B%94_7%EC%9D%BC" title="4월 7일">4월 7일</a> 대통령직을 대행하던 <a class="new" href="/w/index.php?title=%EC%95%84%EA%B0%80%ED%8A%B8_%EC%9A%B0%EC%9C%8C%EB%A7%81%EA%B8%B0%EC%9D%B4%EB%A7%88%EB%82%98&amp;action=edit&amp;redlink=1" title="아가트 우윌링기이마나 (없는 문서)">아가트 우윌링기이마나</a> 수상과 많은 온건파 후투족 지도자들이 RPF에게 살해당하고 그 후 수개월 동안 군대와 대통령 경호대, 극단적 후투족 민명대가 20~50만 명에 달하는 투치족 민간인을 살해했다.</p>
<p><a href="/wiki/1994%EB%85%84" title="1994년">1994년</a> <a href="/wiki/7%EC%9B%94_4%EC%9D%BC" title="7월 4일">7월 4일</a>부터 투치족이 반격을 하여 수도 <a href="/wiki/%ED%82%A4%EA%B0%88%EB%A6%AC" title="키갈리">키갈리</a>를 함락시키고 RPF는 <a href="/wiki/%ED%82%A4%EA%B0%88%EB%A6%AC" title="키갈리">키갈리</a>에서 외부로 통하는 유일한 통로를 차단했다. 이에 수도 키갈리에 갇힌 6만 명의 후투족 민간인들이 반군의 보복이 두려워 탈출하기 시작했고 RPF가 르완다 제2의 도시인 <a href="/wiki/%EB%B6%80%ED%83%80%EB%A0%88" title="부타레">부타레</a>와 후투족 최후 거점인 <a href="/wiki/%EA%B8%B0%EC%84%B8%EB%8B%88" title="기세니">기세니</a>까지 진격하자 후투족들이 피란을 떠났다.</p>
<p>후투족들은 <a href="/wiki/%ED%82%A4%EA%B0%88%EB%A6%AC" title="키갈리">키갈리</a>에서 서남쪽으로 30 km 떨어진 임시정부의 잠정 수도 <a class="new" href="/w/index.php?title=%EA%B8%B0%ED%83%80%EB%9D%BC%EB%A7%88%EC%8B%9C&amp;action=edit&amp;redlink=1" title="기타라마시 (없는 문서)">기타라마시</a>로 가거나 서북부의 국경을 넘어 <a class="mw-redirect" href="/wiki/%EC%BD%A9%EA%B3%A0%EB%AF%BC%EC%A3%BC%EA%B3%B5%ED%99%94%EA%B5%AD" title="콩고민주공화국">콩고민주공화국</a> 등지로 피란을 갔다. 피란 중 300만 명의 후투족 난민이 발생했고 식량 부족과 <a href="/wiki/%EC%BD%9C%EB%A0%88%EB%9D%BC" title="콜레라">콜레라</a> 등의 전염병으로 많은 피란민이 사망했다.</p>
<p>한편 RPF는 7월 말에 전투 중지를 선언했고 후투족 난민은 <a href="/wiki/1996%EB%85%84" title="1996년">1996년</a> 말과 <a href="/wiki/1997%EB%85%84" title="1997년">1997년</a> 초에 다시 르완다로 돌아왔다.</p>
