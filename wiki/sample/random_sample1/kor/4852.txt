<p><b>르하브암</b>(재위 기원전 <a class="mw-redirect" href="/wiki/%EA%B8%B0%EC%9B%90%EC%A0%84_931%EB%85%84" title="기원전 931년">931년</a>경-<a class="mw-redirect" href="/wiki/%EA%B8%B0%EC%9B%90%EC%A0%84_913%EB%85%84" title="기원전 913년">913년</a>경)은 분열 <a href="/wiki/%EC%9C%A0%EB%8B%A4_%EC%99%95%EA%B5%AD" title="유다 왕국">유다 왕국</a>의 첫 왕이다. <a class="mw-redirect" href="/wiki/%EC%86%94%EB%A1%9C%EB%AA%AC_%EC%99%95" title="솔로몬 왕">솔로몬 왕</a>의 아들로 <a href="/wiki/%EC%97%AC%ED%98%B8%EC%99%80" title="여호와">여호와</a>를 저버리고 우상 숭배를 시작했다.</p>
<p><a href="/wiki/%EC%97%B0%ED%95%A9_%EC%9D%B4%EC%8A%A4%EB%9D%BC%EC%97%98_%EC%99%95%EA%B5%AD" title="연합 이스라엘 왕국">연합 이스라엘 왕국</a>의 전성기를 이루었던 <a class="mw-redirect" href="/wiki/%EC%86%94%EB%A1%9C%EB%AA%AC_%EC%99%95" title="솔로몬 왕">솔로몬 왕</a>이 죽자 르호보암이 분열 전 스켐으로 갔을 때 느밧의 아들 <a class="mw-redirect" href="/wiki/%EC%97%AC%EB%A1%9C%EB%B3%B4%EC%95%94_1%EC%84%B8" title="여로보암 1세">여로보암</a>이 <a href="/wiki/%EC%9D%B4%EC%A7%91%ED%8A%B8" title="이집트">이집트</a>에서 돌아오고 북쪽의 10개 지파들이 반기를 들어 <a class="mw-redirect" href="/wiki/%EC%97%AC%EB%A1%9C%EB%B3%B4%EC%95%94_1%EC%84%B8" title="여로보암 1세">여로보암 1세</a>를 새 왕으로 <a href="/wiki/%EC%9D%B4%EC%8A%A4%EB%9D%BC%EC%97%98_%EC%99%95%EA%B5%AD" title="이스라엘 왕국">이스라엘 왕국</a>을 세웠고 남쪽의 2지파는 솔로몬 왕의 아들 르호보암을 왕으로 <a href="/wiki/%EC%9C%A0%EB%8B%A4_%EC%99%95%EA%B5%AD" title="유다 왕국">유다 왕국</a>을 세워 <a href="/wiki/%EC%9D%B4%EC%8A%A4%EB%9D%BC%EC%97%98" title="이스라엘">이스라엘</a>이 남과 북으로 갈라졌다.</p>
<p>르호보암은 18만 대군을 이끌고 맞서려 했으나 선지자 <a class="new" href="/w/index.php?title=%EC%8A%A4%EB%A7%88%EC%95%BC&amp;action=edit&amp;redlink=1" title="스마야 (없는 문서)">스마야</a>의 경고로 그만 두었다. 그의 재위기 동안 유다 역시 큰 죄를 저질렀고 높은 언덕과 푸른 나무 아래마다 산당과 기념 기둥, 아세라 목상들을 세웠다. 제오년에는 <a href="/wiki/%EC%9D%B4%EC%A7%91%ED%8A%B8" title="이집트">이집트</a> 임금 <a class="new" href="/w/index.php?title=%EC%85%B0%EC%88%91%ED%81%AC_1%EC%84%B8&amp;action=edit&amp;redlink=1" title="셰숑크 1세 (없는 문서)">사삭</a>이 <a href="/wiki/%EC%98%88%EB%A3%A8%EC%82%B4%EB%A0%98" title="예루살렘">예루살렘</a>으로 쳐들어왔는데 병거 1200대와 기병 6만, 수없이 많은 리비아군과 수키군, 에티오피아군을 이끌고 올라왔다. 그리고 <a href="/wiki/%EC%98%88%EB%A3%A8%EC%82%B4%EB%A0%98" title="예루살렘">예루살렘</a>으로 와 성전과 왕궁의 보물들을 모조리 가져가 버렸다.</p>
