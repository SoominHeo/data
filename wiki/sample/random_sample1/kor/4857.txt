<td class="mbox-text"><span class="mbox-text-span">이 문서의 내용은 <b><a href="/wiki/%EC%9C%84%ED%82%A4%EB%B0%B1%EA%B3%BC:%EC%B6%9C%EC%B2%98_%EB%B0%9D%ED%9E%88%EA%B8%B0" title="위키백과:출처 밝히기">출처</a>가 분명하지 않습니다.</b><span class="hide-when-compact"><br/>
지금 바로 이 <span class="plainlinks"><a class="external text" href="//ko.wikipedia.org/w/index.php?title=%EB%A6%AC_%EB%B9%84%EC%B9%A0&amp;action=edit">문서를 편집</a></span>하여, 참고하신 문헌이나 <a href="/wiki/%EC%9C%84%ED%82%A4%EB%B0%B1%EA%B3%BC:%EC%8B%A0%EB%A2%B0%ED%95%A0_%EC%88%98_%EC%9E%88%EB%8A%94_%EC%B6%9C%EC%B2%98" title="위키백과:신뢰할 수 있는 출처">신뢰할 수 있는 출처</a>를 각주 등으로 표기해 주세요. <a href="/wiki/%EC%9C%84%ED%82%A4%EB%B0%B1%EA%B3%BC:%ED%99%95%EC%9D%B8_%EA%B0%80%EB%8A%A5" title="위키백과:확인 가능">검증</a>되지 않은 내용은 삭제될 수도 있습니다. 내용에 대한 의견은 <a class="new" href="/w/index.php?title=%ED%86%A0%EB%A1%A0:%EB%A6%AC_%EB%B9%84%EC%B9%A0&amp;action=edit&amp;redlink=1" title="토론:리 비칠 (없는 문서)">토론 문서</a>에서 나누어 주세요.</span> <small>(2015년 9월)</small></span></td>

<p><b>리 비칠</b> (Lee Beachill, <a href="/wiki/%EC%98%81%EA%B5%AD" title="영국">영국</a> <a class="mw-redirect" href="/wiki/%ED%97%88%EB%8D%94%EC%8A%A4%ED%95%84%EB%93%9C" title="허더스필드">허더스필드</a>(<a class="extiw" href="https://en.wikipedia.org/wiki/Huddersfield" title="en:Huddersfield">en:Huddersfield</a>), <a href="/wiki/1977%EB%85%84" title="1977년">1977년</a> <a href="/wiki/11%EC%9B%94_28%EC%9D%BC" title="11월 28일">11월 28일</a>생)은 <a href="/wiki/%EC%98%81%EA%B5%AD" title="영국">영국</a> 출신의 프로 <a class="mw-redirect" href="/wiki/%EC%8A%A4%EC%BF%BC%EC%8B%9C_(%EC%8A%A4%ED%8F%AC%EC%B8%A0)" title="스쿼시 (스포츠)">스쿼시</a> 선수이다.</p>
<p>비칠은 <a href="/wiki/2004%EB%85%84" title="2004년">2004년</a> 10월에 세계 1위 랭킹에 올라갔다. 그는 그해 <a href="/wiki/%EC%9B%94%EB%93%9C_%EC%98%A4%ED%94%88" title="월드 오픈">월드 오픈</a>에서 준우승을 차지하기도 했다.</p>
<p>비칠은 <a href="/wiki/2005%EB%85%84" title="2005년">2005년</a> <a href="/wiki/%EC%84%B8%EA%B3%84_%ED%8C%80_%EC%8A%A4%EC%BF%BC%EC%8B%9C_%EC%84%A0%EC%88%98%EA%B6%8C_%EB%8C%80%ED%9A%8C" title="세계 팀 스쿼시 선수권 대회">세계 팀 스쿼시 선수권 대회</a>에서 우승을 차지한 영국 대표팀 선수였다. 그는 또한 <a href="/wiki/2002%EB%85%84" title="2002년">2002년</a>과 <a href="/wiki/2006%EB%85%84" title="2006년">2006년</a> <a class="mw-redirect" href="/wiki/%EC%BB%A4%EB%A8%BC%EC%9B%B0%EC%8A%A4_%EA%B2%8C%EC%9E%84%EC%A6%88" title="커먼웰스 게임즈">커먼웰스 게임즈</a> 남자 복식에서도 <a href="/wiki/%ED%94%BC%ED%84%B0_%EB%8B%88%EC%BD%9C" title="피터 니콜">피터 니콜</a>과 함께 영국에 금메달을 안겨주었다.</p>
<p>비칠은 <a class="new" href="/w/index.php?title=%EB%B8%8C%EB%A6%AC%ED%8B%B0%EC%8B%9C_%EB%82%B4%EC%85%94%EB%84%90_%EC%8A%A4%EC%BF%BC%EC%8B%9C_%EC%84%A0%EC%88%98%EA%B6%8C_%EB%8C%80%ED%9A%8C&amp;action=edit&amp;redlink=1" title="브리티시 내셔널 스쿼시 선수권 대회 (없는 문서)">브리티시 내셔널 스쿼시 선수권 대회</a>에서 <a href="/wiki/2001%EB%85%84" title="2001년">2001년</a>, 2002년, 그리고 2005년 등 세 번 우승하였다.</p>
<p>주니어 선수로서, 비칠은 영국이 <a href="/wiki/1997%EB%85%84" title="1997년">1997년</a> 세계 주니어 팀 챔피언쉽의 우승에 일조를 하였고 12세 미만, 14세 미만, 17세 미만, 19세 미만 레벨에서도 우승 하였다.</p>
<p>리는 요크셔(Yorkshire)의 스켈맨도프(Skelmanthorpe) 스쿼시 클럽에서 크리스 백 코치의 지도를 받으며 처음 스쿼시를 시작하였다.</p>
<p>그는 <a class="new" href="/w/index.php?title=%ED%98%B8%EB%B2%84%EB%A6%AC&amp;action=edit&amp;redlink=1" title="호버리 (없는 문서)">호버리</a>(Horbury)의 호버리 스쿨을 다니며 언어학을 전공하였다.</p>
