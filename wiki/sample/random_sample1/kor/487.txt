<p><b>2014년 타이 쿠데타</b>(<span style="font-size: smaller;"><a href="/wiki/%ED%83%9C%EA%B5%AD%EC%96%B4" title="태국어">태국어</a>:</span> <span lang="th" xml:lang="th">รัฐประหารในประเทศไทย พ.ศ. 2557</span>)는 <a href="/wiki/2014%EB%85%84" title="2014년">2014년</a> <a href="/wiki/5%EC%9B%94_22%EC%9D%BC" title="5월 22일">5월 22일</a>, <a href="/wiki/%ED%83%80%EC%9D%B4" title="타이">타이</a>에서 육군 총사령관 <a href="/wiki/%EC%81%98%EB%9D%BC%EC%9C%B3_%EC%A7%A0%EC%98%A4%EC%B0%A8" title="쁘라윳 짠오차">쁘라윳 짠오차</a>의 지휘 하에 타이 군이 <a href="/wiki/%EC%9E%89%EB%9D%BD_%EC%B9%9C%EB%82%98%EC%99%93" title="잉락 친나왓">잉락 친나왓</a> 하의 <a href="/wiki/%EC%A0%9C60%EC%B0%A8_%ED%83%80%EC%9D%B4_%EB%82%B4%EA%B0%81" title="제60차 타이 내각">제60차 타이 내각</a>에 대해 군사 <a href="/wiki/%EC%BF%A0%EB%8D%B0%ED%83%80" title="쿠데타">쿠데타</a>를 일으킨 사건이다. 이 쿠데타는 <a href="/wiki/2013~2014%EB%85%84_%ED%83%80%EC%9D%B4_%EC%A0%95%EC%B9%98_%EC%9C%84%EA%B8%B0" title="2013~2014년 타이 정치 위기">2013~2014년 타이 정치 위기</a>의 6개월이 지난 이후 일어났다. 쿠데타 이후 군과 경찰로 구성된 <a href="/wiki/%EA%B5%AD%EA%B0%80%ED%8F%89%ED%99%94%EC%9C%A0%EC%A7%80%EC%9C%84%EC%9B%90%ED%9A%8C" title="국가평화유지위원회">국가평화유지위원회</a>(NPOMC)가 정부를 운영하여 <a class="mw-redirect" href="/wiki/%EC%9E%85%ED%97%8C_%EA%B5%B0%EC%A3%BC%EC%A0%9C" title="입헌 군주제">입헌 군주제</a> 형태에 <a href="/wiki/%EA%B5%B0%EC%82%AC_%EC%A0%95%EB%B6%80" title="군사 정부">군정하의</a> <a href="/wiki/%EC%9E%84%EC%8B%9C_%EC%A0%95%EB%B6%80" title="임시 정부">과도</a> <a href="/wiki/%EC%9D%98%EC%9B%90_%EB%82%B4%EA%B0%81%EC%A0%9C" title="의원 내각제">의원 내각제</a> <a href="/wiki/%EA%B5%B0%EC%A3%BC%EC%A0%9C" title="군주제">군주국</a>으로 정부형태가 전환되었다. 국가평화유지위원회의 위원장은 쿠데타 주도자인 <a href="/wiki/%EC%81%98%EB%9D%BC%EC%9C%B3_%EC%A7%A0%EC%98%A4%EC%B0%A8" title="쁘라윳 짠오차">쁘라윳 짠오차</a> 육군 총사령관이 맡았다. 이후 <a href="/wiki/5%EC%9B%94_26%EC%9D%BC" title="5월 26일">5월 26일</a>, <a href="/wiki/%ED%83%80%EC%9D%B4%EC%9D%98_%EA%B5%B0%EC%A3%BC_%EB%AA%A9%EB%A1%9D" title="타이의 군주 목록">국왕</a> <a href="/wiki/%ED%91%B8%EB%AF%B8%ED%8F%B0_%EC%95%84%EB%91%94%EC%95%BC%EB%8E%83" title="푸미폰 아둔야뎃">푸미폰 아둔야뎃</a>(라마 9세)은 쿠데타를 공식 승인했다.</p>
