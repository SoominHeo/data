<p><b>2015년 네팔 지진</b>은 <a href="/wiki/2015%EB%85%84" title="2015년">2015년</a> <a href="/wiki/4%EC%9B%94_25%EC%9D%BC" title="4월 25일">4월 25일</a> <a class="mw-redirect" href="/wiki/UTC" title="UTC">UTC</a> 6시 11분 26초(<a href="/wiki/%EB%84%A4%ED%8C%94_%ED%91%9C%EC%A4%80%EC%8B%9C" title="네팔 표준시">네팔 표준시</a> 11시 56분 26초)에 발생한 <a href="/wiki/%EB%AA%A8%EB%A9%98%ED%8A%B8_%EA%B7%9C%EB%AA%A8" title="모멘트 규모">모멘트 규모</a> 7.8, <a class="new" href="/w/index.php?title=%EB%A9%94%EB%A5%B4%EC%B9%BC%EB%A6%AC_%EC%A7%84%EB%8F%84&amp;action=edit&amp;redlink=1" title="메르칼리 진도 (없는 문서)">메르칼리 진도</a> IX(격렬)의 대지진이다. <a href="/wiki/%EC%A7%84%EC%95%99" title="진앙">진앙</a>은 <a href="/wiki/%EB%84%A4%ED%8C%94" title="네팔">네팔</a> <a href="/wiki/%EA%B0%84%EB%8B%A4%ED%82%A4_%EA%B5%AC" title="간다키 구">간다키 구</a> <a class="new" href="/w/index.php?title=%EA%B3%A0%EB%A5%B4%EC%B9%B4_%ED%98%84&amp;action=edit&amp;redlink=1" title="고르카 현 (없는 문서)">고르카 현</a>이며, <a href="/wiki/%EC%A7%84%EC%9B%90" title="진원">진원</a>까지의 깊이는 대략 15km로 매우 얕은 편이다.<sup class="reference" id="cite_ref-USGS_1-4"><a href="#cite_note-USGS-1">[1]</a></sup>이 지진은 <a href="/wiki/1934%EB%85%84_%EB%84%A4%ED%8C%94-%EB%B9%84%ED%95%98%EB%A5%B4_%EC%A7%80%EC%A7%84" title="1934년 네팔-비하르 지진">1934년 네팔-비하르 지진</a> 이후 네팔에서 발생한 가장 강력한 지진이다. 이 지진으로 인해 네팔, 중국, 인도, 파키스탄, 방글라데시 등지에서 8천400명 이상이 사망한 것으로 추정된다. 정확한 사망인의 수는 아직 잘 모르는 것으로 추정된다.<sup class="reference" id="cite_ref-4"><a href="#cite_note-4">[4]</a></sup><sup class="reference" id="cite_ref-5"><a href="#cite_note-5">[5]</a></sup>이 지진으로 인해 <a href="/wiki/%EC%B9%B4%ED%8A%B8%EB%A7%8C%EB%91%90_%EA%B3%84%EA%B3%A1" title="카트만두 계곡">카트만두 계곡</a>의 카트만두 <a href="/wiki/%EB%8D%94%EB%A5%B4%EB%B0%94%EB%A5%B4_%EA%B4%91%EC%9E%A5" title="더르바르 광장">더르바르 광장</a>과 같은 여러 <a class="mw-redirect" href="/wiki/%EC%9C%A0%EB%84%A4%EC%8A%A4%EC%BD%94" title="유네스코">유네스코</a> <a href="/wiki/%EC%84%B8%EA%B3%84%EC%9C%A0%EC%82%B0" title="세계유산">세계유산</a>이 파괴되었다. <a href="/wiki/%EC%97%90%EB%B2%A0%EB%A0%88%EC%8A%A4%ED%8A%B8_%EC%82%B0" title="에베레스트 산">에베레스트 산</a>에도 눈사태가 발생해 <a href="/wiki/2014%EB%85%84_%EC%97%90%EB%B2%A0%EB%A0%88%EC%8A%A4%ED%8A%B8_%EB%88%88%EC%82%AC%ED%83%9C" title="2014년 에베레스트 눈사태">2014년 에베레스트 눈사태</a> 이후 최대 사망자가 발생했다.</p>
<p>4월 26일 UTC 7:09:08에는 규모 6.7의 여진이 발생했다.<sup class="reference" id="cite_ref-6"><a href="#cite_note-6">[6]</a></sup> 5월 12일 UTC 16:05:19에는 카트만두 동쪽 83km지역에서 규모 7.4의 <a class="new" href="/w/index.php?title=2015%EB%85%84_5%EC%9B%94_%EB%84%A4%ED%8C%94_%EC%A7%80%EC%A7%84&amp;action=edit&amp;redlink=1" title="2015년 5월 네팔 지진 (없는 문서)">본진과 가까운 여진</a>이 발생했다.</p>

<h3><span class="mw-headline" id=".EB.88.88.EC.82.AC.ED.83.9C_.ED.94.BC.ED.95.B4">눈사태 피해</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=2015%EB%85%84_4%EC%9B%94_%EB%84%A4%ED%8C%94_%EC%A7%80%EC%A7%84&amp;action=edit&amp;section=3" title="부분 편집: 눈사태 피해">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<div class="rellink relarticle mainarticle"><img alt="" data-file-height="128" data-file-width="128" height="16" src="//upload.wikimedia.org/wikipedia/commons/thumb/e/ec/Crystal_Clear_app_xmag.svg/16px-Crystal_Clear_app_xmag.svg.png" srcset="//upload.wikimedia.org/wikipedia/commons/thumb/e/ec/Crystal_Clear_app_xmag.svg/24px-Crystal_Clear_app_xmag.svg.png 1.5x, //upload.wikimedia.org/wikipedia/commons/thumb/e/ec/Crystal_Clear_app_xmag.svg/32px-Crystal_Clear_app_xmag.svg.png 2x" width="16"/> 이 부분의 본문은 <a class="new" href="/w/index.php?title=2015%EB%85%84_%EC%97%90%EB%B2%A0%EB%A0%88%EC%8A%A4%ED%8A%B8_%EC%82%B0%EC%82%AC%ED%83%9C&amp;action=edit&amp;redlink=1" title="2015년 에베레스트 산사태 (없는 문서)">2015년 에베레스트 산사태</a>입니다.</div>
<p>지진으로 인해 <a href="/wiki/%EC%97%90%EB%B2%A0%EB%A0%88%EC%8A%A4%ED%8A%B8_%EC%82%B0" title="에베레스트 산">에베레스트 산</a>에서 <a href="/wiki/%EB%88%88%EC%82%AC%ED%83%9C" title="눈사태">눈사태</a>가 발생하면서 <a class="new" href="/w/index.php?title=%EC%97%90%EB%B2%A0%EB%A0%88%EC%8A%A4%ED%8A%B8_%EB%B2%A0%EC%9D%B4%EC%8A%A4%EC%BA%A0%ED%94%84&amp;action=edit&amp;redlink=1" title="에베레스트 베이스캠프 (없는 문서)">남부 베이스캠프</a>에서 17명이 사망했다.<sup class="reference" id="cite_ref-Sharma_25-0"><a href="#cite_note-Sharma-25">[25]</a></sup><sup class="reference" id="cite_ref-Graun_26-0"><a href="#cite_note-Graun-26">[26]</a></sup> <a href="/wiki/%EC%9D%B8%EB%8F%84%EA%B5%B0" title="인도군">인도군</a> 등산대는 에베레스트 베이스캠프에서 등산객 19명의 시신을 찾아냈고 에베레스트 산에서 고립되어 있던 등산객 최소 61명을 구조했다.<sup class="reference" id="cite_ref-auto2_27-0"><a href="#cite_note-auto2-27">[27]</a></sup><sup class="reference" id="cite_ref-auto3_28-0"><a href="#cite_note-auto3-28">[28]</a></sup><sup class="reference" id="cite_ref-auto4_29-0"><a href="#cite_note-auto4-29">[29]</a></sup> 차후 <a href="/wiki/%EA%B5%AC%EA%B8%80_%EC%96%B4%EC%8A%A4" title="구글 어스">구글 어스</a>형 프로젝트를 위해서 타사 고용인 세 명과 함께 해당 지역을 조사하던 <a href="/wiki/%EA%B5%AC%EA%B8%80" title="구글">구글</a>의 고위 임직원 <a class="new" href="/w/index.php?title=%EB%8C%84_%ED%94%84%EB%A0%88%EB%94%98%EB%B2%84%EA%B7%B8&amp;action=edit&amp;redlink=1" title="댄 프레딘버그 (없는 문서)">댄 프레딘버그</a>도 사상자에 포함됐다 .<sup class="reference" id="cite_ref-NYTFredinburg_15-1"><a href="#cite_note-NYTFredinburg-15">[15]</a></sup> 지진이 일어날 당시 부상자 61명에 실종자 '수십 여명'을 포함해 700명에서 1,000명 이상이 에베레스트 산에 있었던 것으로 보이며, 이보다 더 많은 등반가들이 높은 고도의 캠프에서 안전한 하산 경로를 잃고 초반에 고립되기도 했다.<sup class="reference" id="cite_ref-Graun_26-1"><a href="#cite_note-Graun-26">[26]</a></sup><sup class="reference" id="cite_ref-Sharma_25-1"><a href="#cite_note-Sharma-25">[25]</a></sup><sup class="reference" id="cite_ref-USAtoday1_30-0"><a href="#cite_note-USAtoday1-30">[30]</a></sup><sup class="reference" id="cite_ref-31"><a href="#cite_note-31">[31]</a></sup><sup class="reference" id="cite_ref-PostEverest_32-0"><a href="#cite_note-PostEverest-32">[32]</a></sup><sup class="reference" id="cite_ref-USATodayEverest_33-0"><a href="#cite_note-USATodayEverest-33">[33]</a></sup> 에베레스트 산은 진원지에서부터 동쪽으로 220km 가량 떨어진 곳에 위치해 있다.</p>
<p>4월 26일 아침 <a href="/wiki/%EC%9D%B8%EB%8F%84_%EA%B3%B5%EA%B5%B0" title="인도 공군">인도 공군</a> 소속 헬리콥터 몇 대가 구조작전을 위해 에베레스트 산에 도착했고,<sup class="reference" id="cite_ref-auto5_34-0"><a href="#cite_note-auto5-34">[34]</a></sup> 궂은 날씨로 작전이 중단되기 전까지 22명의 중상자들을 <a class="new" href="/w/index.php?title=%ED%8E%98%EB%A6%AC%EC%B2%B4&amp;action=edit&amp;redlink=1" title="페리체 (없는 문서)">페리체</a> 마을로 수송했다.<sup class="reference" id="cite_ref-NYTAvalanche_35-0"><a href="#cite_note-NYTAvalanche-35">[35]</a></sup> 페리체는 등반가들에게 중요한 단기 체류지로, 현지인들과 외국 관광객들이 주로 일하는 기초 병원 한 곳이 있다. 보도에 따르면 그날 이후 헬리콥터 한 대가 베이스캠프 다음으로 첫번째 여정인 에베레스트 제1캠프에 있던 등산객 여러 명을 대피시켰으며, 100명 가량의 일부 등반가들이 아직도 제1캠프와 제2캠프에서 안전하게 하산하지 못하고 있다.<sup class="reference" id="cite_ref-ReutersEvacuation_36-0"><a href="#cite_note-ReutersEvacuation-36">[36]</a></sup> 베이스캠프에 있던 등반가들은 당일 지진이 발생한 후 <a href="/wiki/%ED%8A%B8%EC%9C%84%ED%84%B0" title="트위터">트위터</a>로 코멘트를 보냈는데, 캠프에 남겨진 이들 중에는 '엄청난 폐허'와 '매우 불확실한 상황'이라 말하는 한편 그 지역이 <a class="mw-redirect" href="/wiki/%ED%95%B5%ED%8F%AD%ED%83%84" title="핵폭탄">핵폭탄</a> 투하라도 당한 것처럼 보였다고 언급했다. 한 등산객은 산에서 더 높은 고도에 갇힌 사람들은 '자포자기했다'고 <a href="/wiki/%ED%8E%98%EC%9D%B4%EC%8A%A4%EB%B6%81" title="페이스북">페이스북</a>을 통해 말하기도 했다.<sup class="reference" id="cite_ref-37"><a href="#cite_note-37">[37]</a></sup></p>
