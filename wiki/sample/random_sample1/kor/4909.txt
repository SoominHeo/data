<h2><span class="mw-headline" id=".EA.B0.9C.EC.9A.94">개요</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%A6%AC%EB%B3%B4%EC%86%9C&amp;action=edit&amp;section=1" title="부분 편집: 개요">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>리보솜은 <a href="/wiki/%EC%A7%80%EB%A6%84" title="지름">지름</a>이 약 20 <a href="/wiki/%EB%82%98%EB%85%B8%EB%AF%B8%ED%84%B0" title="나노미터">nm</a>(200 <a href="/wiki/%EC%98%B9%EC%8A%A4%ED%8A%B8%EB%A1%AC" title="옹스트롬">Å</a>)이고 65<a class="mw-redirect" href="/wiki/%ED%8D%BC%EC%84%BC%ED%8A%B8" title="퍼센트">%</a>의 <a href="/wiki/%EB%A6%AC%EB%B3%B4%EC%86%9C_RNA" title="리보솜 RNA">리보솜 RNA</a>와 35%의 <a class="new" href="/w/index.php?title=%EB%A6%AC%EB%B3%B4%EC%86%9C_%EB%8B%A8%EB%B0%B1%EC%A7%88&amp;action=edit&amp;redlink=1" title="리보솜 단백질 (없는 문서)">리보솜 단백질</a>로 구성되어 있다. <a href="/wiki/%EA%B3%A0%EC%84%B8%EA%B7%A0" title="고세균">고세균</a>, <a class="mw-redirect" href="/wiki/%EC%A7%84%EC%A0%95%EC%84%B8%EA%B7%A0" title="진정세균">진정세균</a> 및 <a href="/wiki/%EC%A7%84%ED%95%B5%EC%83%9D%EB%AC%BC" title="진핵생물">진핵생물</a>의 리보솜은 서로 다른 다양한 단백질과 <a href="/wiki/RNA" title="RNA">RNA</a>로 구성되어 있다. 리보솜은 <a href="/wiki/%EC%A0%84%EB%A0%B9_RNA" title="전령 RNA">전령 RNA</a>의 <a href="/wiki/%EC%BD%94%EB%8F%88" title="코돈">코돈</a>을 <a href="/wiki/%EB%B2%88%EC%97%AD_(%EC%83%9D%EB%AC%BC%ED%95%99)" title="번역 (생물학)">번역</a>하여 <a href="/wiki/%EC%9A%B4%EB%B0%98_RNA" title="운반 RNA">운반 RNA</a>에 연결된 <a href="/wiki/%EC%95%84%EB%AF%B8%EB%85%B8%EC%82%B0" title="아미노산">아미노산</a>을 배열하여 단백질을 형성한다. 리보솜은 또한 <a href="/wiki/%ED%9A%A8%EC%86%8C" title="효소">효소</a>로서 작용하기도 하는데 이때문에 <b>리보자임</b>이라 불리기도 한다.<sup class="reference" id="cite_ref-2"><a href="#cite_note-2">[2]</a></sup> 리보솜은 별도의 막이 없는 세포소기관이다.</p>
<p>단백질을 형성하지 않는 동안 RNA는 내부 원형질인 <a class="new" href="/w/index.php?title=%EC%8B%9C%ED%86%A0%EC%A1%B8&amp;action=edit&amp;redlink=1" title="시토졸 (없는 문서)">시토졸</a>에 존재한다. 리보자임으로 작용할 때에는 RNA로부터 구성되는데, 이는 <a class="new" href="/w/index.php?title=RNA_%EC%84%B8%EA%B3%84&amp;action=edit&amp;redlink=1" title="RNA 세계 (없는 문서)">RNA 세계</a>의 흔적으로 추정된다.<sup class="reference" id="cite_ref-3"><a href="#cite_note-3">[3]</a></sup></p>
<p>리보좀은 1950년대 중반 <a href="/wiki/%EB%A3%A8%EB%A7%88%EB%8B%88%EC%95%84" title="루마니아">루마니아</a> 생물학자인 <a class="new" href="/w/index.php?title=%EA%B2%8C%EC%98%A4%EB%A5%B4%EA%B7%B8_%EC%97%90%EB%B0%80_%ED%8C%94%EB%9D%BC%EB%8D%B0&amp;action=edit&amp;redlink=1" title="게오르그 에밀 팔라데 (없는 문서)">게오르그 에밀 팔라데</a>에 의해 최초로 관찰되었다. 팔라데는 <a class="mw-redirect" href="/wiki/%EC%A0%84%EC%9E%90%ED%98%84%EB%AF%B8%EA%B2%BD" title="전자현미경">전자현미경</a>을 이용하여 세포를 관찰하던 중 리보솜을 발견하였고, 이 공로로 1974년 <a href="/wiki/%EB%85%B8%EB%B2%A8_%EC%83%9D%EB%A6%AC%ED%95%99%C2%B7%EC%9D%98%ED%95%99%EC%83%81" title="노벨 생리학·의학상">노벨 생리학·의학상</a>을 수상하였다.<sup class="reference" id="cite_ref-4"><a href="#cite_note-4">[4]</a></sup> 리보솜이라는 이름은 1958년 리처드 B. 로버트가 명명하였다.</p>

<p>20세기 중반에 리보솜의 분자 구조와 기능이 발견된 이후 현재까지 활발한 연구가 계속되고 있다.</p>
<h2><span class="mw-headline" id=".EC.83.9D.EC.84.B1">생성</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%A6%AC%EB%B3%B4%EC%86%9C&amp;action=edit&amp;section=2" title="부분 편집: 생성">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p><a href="/wiki/%EC%84%B8%EA%B7%A0" title="세균">세균</a>의 리보솜은 <a href="/wiki/%EC%84%B8%ED%8F%AC%EC%A7%88" title="세포질">세포질</a>에 있는 <a href="/wiki/%EC%9C%A0%EC%A0%84%EC%9E%90" title="유전자">유전자</a>인 <a href="/wiki/%EC%98%A4%ED%8E%98%EB%A1%A0" title="오페론">오페론</a>을 <a href="/wiki/%EC%A0%84%EC%82%AC_(%EC%83%9D%EB%AC%BC%ED%95%99)" title="전사 (생물학)">전사</a>하여 생성된다. <a href="/wiki/%EC%A7%84%ED%95%B5%EC%83%9D%EB%AC%BC" title="진핵생물">진핵생물</a>의 경우 <a class="mw-redirect" href="/wiki/%EC%84%B8%ED%8F%AC%ED%95%B5" title="세포핵">세포핵</a> 안에 있는 <a href="/wiki/%ED%95%B5%EC%86%8C%EC%B2%B4" title="핵소체">핵소체</a>에서 4 종류의 <a href="/wiki/%EB%A6%AC%EB%B3%B4%EC%86%9C_RNA" title="리보솜 RNA">리보솜 RNA</a>가 만들어지며 <a class="new" href="/w/index.php?title=%EB%A6%AC%EB%B3%B4%EC%86%9C_%EB%8B%A8%EB%B0%B1%EC%A7%88&amp;action=edit&amp;redlink=1" title="리보솜 단백질 (없는 문서)">리보솜 단백질</a>과 결합되어 리보솜이 생성된다.</p>
<h2><span class="mw-headline" id=".EB.8B.A8.EB.B0.B1.EC.A7.88_.EC.83.9D.EC.84.B1.EA.B3.BC.EC.A0.95">단백질 생성과정</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%A6%AC%EB%B3%B4%EC%86%9C&amp;action=edit&amp;section=3" title="부분 편집: 단백질 생성과정">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<div class="thumb tright">
<div class="thumbinner" style="width:252px;"><a class="image" href="/wiki/%ED%8C%8C%EC%9D%BC:TRNA_ribosomes_diagram_en.svg"><img alt="" class="thumbimage" data-file-height="500" data-file-width="311" height="402" src="//upload.wikimedia.org/wikipedia/commons/thumb/1/10/TRNA_ribosomes_diagram_en.svg/250px-TRNA_ribosomes_diagram_en.svg.png" srcset="//upload.wikimedia.org/wikipedia/commons/thumb/1/10/TRNA_ribosomes_diagram_en.svg/375px-TRNA_ribosomes_diagram_en.svg.png 1.5x, //upload.wikimedia.org/wikipedia/commons/thumb/1/10/TRNA_ribosomes_diagram_en.svg/500px-TRNA_ribosomes_diagram_en.svg.png 2x" width="250"/></a>
<div class="thumbcaption">

리보솜에서 이루어지는 단백질 형성과정</div>
</div>
</div>
<p>리보솜이 mRNA의 <a href="/wiki/%EC%BD%94%EB%8F%88" title="코돈">코돈</a>을 인식하여 단백질을 합성하는 과정을 <a href="/wiki/%EB%B2%88%EC%97%AD_(%EC%83%9D%EB%AC%BC%ED%95%99)" title="번역 (생물학)">번역</a>이라 한다. 리보솜에서 이루어지는 단백질 생성과정은 다음과 같다.<sup class="reference" id="cite_ref-6"><a href="#cite_note-6">[6]</a></sup><sup class="reference" id="cite_ref-7"><a href="#cite_note-7">[7]</a></sup></p>
<ul>
<li>준비
<ul>
<li>리보솜의 소단위체와 대단위체가 결합한다. 리보솜에는 <a href="/wiki/%EC%9A%B4%EB%B0%98_RNA" title="운반 RNA">운반 RNA</a>가 들어설 수 있는 부위가 두 곳이 있는데 운반 RNA가 들어오는 쪽을 A 자리, 나가는 쪽을 P 자리라 한다.</li>
</ul>
</li>
</ul>
<ul>
<li>초기화
<ul>
<li>소단위체에 <a href="/wiki/%EC%A0%84%EB%A0%B9_RNA" title="전령 RNA">전령 RNA</a>가 결합하고, 시작 <a href="/wiki/%EC%BD%94%EB%8F%88" title="코돈">코돈</a>이 확인되면 대단위체의 A 자리에 메티오닌과 결합한 <a href="/wiki/%EC%9A%B4%EB%B0%98_RNA" title="운반 RNA">운반 RNA</a>가 도입되면서 <a href="/wiki/%EB%8B%A8%EB%B0%B1%EC%A7%88" title="단백질">단백질</a>생성이 시작된다.</li>
</ul>
</li>
</ul>
<ul>
<li>연장 과정
<ul>
<li><a href="/wiki/%EC%BD%94%EB%8F%88" title="코돈">코돈</a> 인식 : <a class="mw-redirect" href="/wiki/%ED%8F%B4%EB%A6%AC%ED%8E%A9%ED%83%80%EC%9D%B4%EB%93%9C" title="폴리펩타이드">폴리펩타이드</a>를 달고 있는 P 자리의 <a class="mw-redirect" href="/wiki/TRNA" title="TRNA">tRNA</a>가 코돈에 붙어있다. A 자리에 있는 전령 RNA의 <a href="/wiki/%EC%BD%94%EB%8F%88" title="코돈">코돈</a>과 상보성을 갖는 <a class="mw-redirect" href="/wiki/%EC%95%88%ED%8B%B0%EC%BD%94%EB%8F%88" title="안티코돈">안티코돈</a>을 가진 <a href="/wiki/%EC%9A%B4%EB%B0%98_RNA" title="운반 RNA">운반 RNA</a>가 아미노산을 가져와 결합한다.</li>
<li><a href="/wiki/%ED%8E%A9%ED%83%80%EC%9D%B4%EB%93%9C_%EA%B2%B0%ED%95%A9" title="펩타이드 결합">펩타이드 결합</a> 형성 : P 자리에 있는 기존의 <a class="mw-redirect" href="/wiki/%ED%8F%B4%EB%A6%AC%ED%8E%A9%ED%83%80%EC%9D%B4%EB%93%9C" title="폴리펩타이드">폴리펩타이드</a>와 A 자리에 있는 새로운 <a href="/wiki/%EC%9A%B4%EB%B0%98_RNA" title="운반 RNA">운반 RNA</a>의 <a href="/wiki/%EC%95%84%EB%AF%B8%EB%85%B8%EC%82%B0" title="아미노산">아미노산</a> 사이에 펩타이드 결합이 형성된다. 이때 아미노아실 결합이 P 자리의 운반 RNA에서 새로운 폴리펩타이드 쪽으로 옮겨간다.</li>
<li>변환 : P 자리에서 자유로워진 <a href="/wiki/%EC%9A%B4%EB%B0%98_RNA" title="운반 RNA">운반 RNA</a>가 빠져나간다. 이제 <a href="/wiki/%EC%A0%84%EB%A0%B9_RNA" title="전령 RNA">전령 RNA</a>가 한칸 움직일 수 있게 되고 코돈이 바뀐다.</li>
<li>반복 : A 자리의 <a class="mw-redirect" href="/wiki/%ED%8F%B4%EB%A6%AC%ED%8E%A9%ED%83%80%EC%9D%B4%EB%93%9C" title="폴리펩타이드">폴리펩타이드</a>-운반 RNA 중합분자가 P 자리로 옮겨가면서 A 자리가 비워진다. 비워진 A 자리에는 바뀐 코돈과 상보적인 운반 RNA가 새롭게 결합된다. 위 과정이 계속 반복되면서 폴리펩타이드 사슬이 길어진다.</li>
</ul>
</li>
</ul>
<ul>
<li>종료
<ul>
<li>종료 코돈에 종결 운반 RNA가 결합되면 합성되던 <a class="mw-redirect" href="/wiki/%ED%8F%B4%EB%A6%AC%ED%8E%A9%ED%83%80%EC%9D%B4%EB%93%9C" title="폴리펩타이드">폴리펩타이드</a>가 떨어져 나가면서 단백질 합성이 끝난다.</li>
</ul>
</li>
</ul>
<p>리보솜에서 일어나는 번역 과정의 에너지원은 <a class="new" href="/w/index.php?title=%EA%B5%AC%EC%95%84%EB%85%B8%EC%8B%A0_%EC%82%BC%EC%9D%B8%EC%82%B0&amp;action=edit&amp;redlink=1" title="구아노신 삼인산 (없는 문서)">GTP</a>의 가수분해이다.</p>
