<p><b>리본이끼목</b>(Metzgeriales)은 <a href="/wiki/%EB%A7%9D%EC%9A%B8%EC%9D%B4%EB%81%BC%EA%B0%95" title="망울이끼강">망울이끼강</a>에 속하는 <a href="/wiki/%EC%9A%B0%EC%82%B0%EC%9D%B4%EB%81%BC%EB%AC%B8" title="우산이끼문">우산이끼류</a> 목의 하나이다.<sup class="reference" id="cite_ref-ezmeta_2-0"><a href="#cite_note-ezmeta-2">[2]</a></sup></p>
<h2><span class="mw-headline" id=".ED.95.98.EC.9C.84_.EA.B3.BC">하위 과</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%A6%AC%EB%B3%B8%EC%9D%B4%EB%81%BC%EB%AA%A9&amp;action=edit&amp;section=1" title="부분 편집: 하위 과">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<ul>
<li>비늘물우산대이끼목 (Fossombroniales)으로 분류하기도 함.
<ul>
<li><a class="mw-redirect" href="/wiki/%EC%95%8C%EB%A6%AC%EC%86%8C%EB%8B%88%EC%95%84%EA%B3%BC" title="알리소니아과">알리소니아과</a> (Allisoniaceae)</li>
<li><a class="mw-redirect" href="/wiki/%EB%B0%94%EC%9C%84%EB%AC%BC%EC%9A%B0%EC%82%B0%EB%8C%80%EC%9D%B4%EB%81%BC%EA%B3%BC" title="바위물우산대이끼과">바위물우산대이끼과</a> (Calyculariaceae)</li>
<li><a href="/wiki/%EB%B9%84%EB%8A%98%EC%9A%B0%EC%82%B0%EB%8C%80%EC%9D%B4%EB%81%BC%EA%B3%BC" title="비늘우산대이끼과">비늘우산대이끼과</a> (Fossombroniaceae)<sup class="reference" id="cite_ref-ezmeta_2-1"><a href="#cite_note-ezmeta-2">[2]</a></sup></li>
<li><a class="mw-redirect" href="/wiki/%EB%A7%88%ED%82%A4%EB%85%B8%EC%9A%B0%EC%82%B0%EB%8C%80%EC%9D%B4%EB%81%BC%EA%B3%BC" title="마키노우산대이끼과">마키노우산대이끼과</a> (Makinoaceae)</li>
<li><a href="/wiki/%ED%8E%98%ED%83%88%EB%A1%9C%ED%95%84%EB%A3%B8%EA%B3%BC" title="페탈로필룸과">페탈로필룸과</a> (Petalophyllaceae)</li>
</ul>
</li>
<li>리본이끼목 (Metzgeriales)으로 좁게 분류하기도 함.
<ul>
<li><a href="/wiki/%EC%82%B0%ED%98%B8%EC%9A%B0%EC%82%B0%EB%8C%80%EC%9D%B4%EB%81%BC%EA%B3%BC" title="산호우산대이끼과">산호우산대이끼과</a> (Aneuraceae)<sup class="reference" id="cite_ref-ezmeta_2-2"><a href="#cite_note-ezmeta-2">[2]</a></sup></li>
<li><a href="/wiki/%EB%A6%AC%EB%B3%B8%EC%9D%B4%EB%81%BC%EA%B3%BC" title="리본이끼과">리본이끼과</a> (Metzgeriaceae)<sup class="reference" id="cite_ref-ezmeta_2-3"><a href="#cite_note-ezmeta-2">[2]</a></sup></li>
<li><a class="mw-redirect" href="/wiki/%EB%AF%B8%EC%A3%BC%ED%83%80%EB%8B%88%EC%95%84%EA%B3%BC" title="미주타니아과">미주타니아과</a> (Mizutaniaceae)</li>
</ul>
</li>
<li>거미집이끼목 (Pallaviciniales)으로 분류하기도 함.
<ul>
<li><a class="mw-redirect" href="/wiki/%ED%9E%88%EB%A9%94%EB%85%B8%ED%94%BC%ED%86%A4%EA%B3%BC" title="히메노피톤과">히메노피톤과</a> (Hymenophytaceae)</li>
<li><a href="/wiki/%EB%AF%B8%EC%97%AD%EC%9D%B4%EB%81%BC%EA%B3%BC" title="미역이끼과">미역이끼과</a> (Moerckiaceae)</li>
<li><a href="/wiki/%EA%B1%B0%EB%AF%B8%EC%A7%91%EC%9D%B4%EB%81%BC%EA%B3%BC" title="거미집이끼과">거미집이끼과</a> (Pallaviciniaceae)</li>
<li><a class="mw-redirect" href="/wiki/%ED%95%84%EB%A1%9C%ED%83%88%EB%A6%AC%EC%95%84%EA%B3%BC" title="필로탈리아과">필로탈리아과</a> (Phyllothalliaceae)</li>
<li><a class="mw-redirect" href="/wiki/%EC%82%B0%EB%8D%B0%EC%98%A4%ED%83%88%EB%A3%A8%EC%8A%A4%EA%B3%BC" title="산데오탈루스과">산데오탈루스과</a> (Sandeothallaceae)</li>
</ul>
</li>
<li>물우산대이끼목 (Pelliales)으로 분류하기도 함.
<ul>
<li><a href="/wiki/%EB%AC%BC%EC%9A%B0%EC%82%B0%EB%8C%80%EC%9D%B4%EB%81%BC%EA%B3%BC" title="물우산대이끼과">물우산대이끼과</a> (Pelliaceae)<sup class="reference" id="cite_ref-ezmeta_2-4"><a href="#cite_note-ezmeta-2">[2]</a></sup></li>
</ul>
</li>
<li><a class="mw-redirect" href="/wiki/%EB%A9%B8%EC%A2%85" title="멸종">†</a> Metzgeriites</li>
</ul>
