<p><b>리샤오펑</b>(<small><a href="/wiki/%EC%A4%91%EA%B5%AD%EC%96%B4" title="중국어">중국어</a></small> <small><a href="/wiki/%EA%B0%84%EC%B2%B4%EC%9E%90" title="간체자">간체</a>:</small> <span lang="zh-hans" xml:lang="zh-hans">李小鹏</span>, <small><a href="/wiki/%EC%A0%95%EC%B2%B4%EC%9E%90" title="정체자">정체</a>:</small> <span lang="zh-hant" xml:lang="zh-hant">李小鵬</span>, <small><a href="/wiki/%ED%95%9C%EC%96%B4_%EB%B3%91%EC%9D%8C" title="한어 병음">병음</a>:</small> <font face="Lucida Sans Unicode">Lǐ Xiǎopéng</font>, <a href="/wiki/1981%EB%85%84" title="1981년">1981년</a> <a href="/wiki/7%EC%9B%94_27%EC%9D%BC" title="7월 27일">7월 27일</a> <a href="/wiki/%ED%9B%84%EB%82%9C_%EC%84%B1" title="후난 성">후난 성</a> <a href="/wiki/%EC%B0%BD%EC%82%AC_%EC%8B%9C" title="창사 시">창사 시</a>)은 <a href="/wiki/%EC%A4%91%ED%99%94%EC%9D%B8%EB%AF%BC%EA%B3%B5%ED%99%94%EA%B5%AD" title="중화인민공화국">중화인민공화국</a>의 <a href="/wiki/%EC%B2%B4%EC%A1%B0" title="체조">체조</a> 선수로, 주요 종목은 <a href="/wiki/%ED%8F%89%ED%96%89%EB%B4%89" title="평행봉">평행봉</a>과 <a href="/wiki/%EB%8F%84%EB%A7%88_(%EC%B2%B4%EC%A1%B0)" title="도마 (체조)">도마</a>이다.</p>
<p>6세 때 체조를 시작했으며 12세 때 <a href="/wiki/%ED%9B%84%EB%82%9C_%EC%84%B1" title="후난 성">후난 성</a> 체조 대표팀 선수로 발탁되었다. 15세 때 중국 체조 국가대표로 발탁되었으며 <a href="/wiki/1997%EB%85%84" title="1997년">1997년</a> <a href="/wiki/%EB%A1%9C%EC%9E%94" title="로잔">로잔</a>에서 열린 세계 체조 선수권 대회에서 단체전 종합 금메달, 평행봉 은메달, 마루 동메달을 획득했다.</p>
<p><a href="/wiki/2000%EB%85%84_%ED%95%98%EA%B3%84_%EC%98%AC%EB%A6%BC%ED%94%BD" title="2000년 하계 올림픽">2000년 시드니 올림픽</a>에서 단체전 종합 금메달과 평행봉 금메달을 획득했으며 <a href="/wiki/2003%EB%85%84" title="2003년">2003년</a> <a href="/wiki/%EC%95%A0%EB%84%88%ED%95%98%EC%9E%84" title="애너하임">애너하임</a>에서 열린 세계 체조 선수권 대회에서 평행봉과 도마, 단체전 종합 금메달을 획득했지만 <a href="/wiki/2004%EB%85%84_%ED%95%98%EA%B3%84_%EC%98%AC%EB%A6%BC%ED%94%BD" title="2004년 하계 올림픽">2004년 아테네 올림픽</a>에서 단체전 종합 5위, 도마 7위, 평행봉 동메달이라는 최악의 성적을 받았다.</p>
<p><a href="/wiki/2006%EB%85%84" title="2006년">2006년</a> 재활 수술을 받았으며 <a href="/wiki/2008%EB%85%84_%ED%95%98%EA%B3%84_%EC%98%AC%EB%A6%BC%ED%94%BD" title="2008년 하계 올림픽">2008년 베이징 올림픽</a>에서 평행봉과 단체전 종합 금메달을 획득했다. 세계 체조 선수권 대회와 체조 월드컵, 하계 올림픽에서 금메달 16개를 획득했으며 <a href="/wiki/%EB%A6%AC%EB%8B%9D" title="리닝">리닝</a>과 함께 중국 체조 역사에서 금메달을 가장 많이 획득한 선수로 기록되어 있다.</p>
