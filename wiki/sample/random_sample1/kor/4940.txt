title="스페인어">스페인어</a>:</span> <span lang="es" xml:lang="es">El Himno de Riego</span> <small>엘 임노 데 리에고<sup>[<a href="/wiki/%EC%9C%84%ED%82%A4%EB%B0%B1%EA%B3%BC:%EC%8A%A4%ED%8E%98%EC%9D%B8%EC%96%B4%EC%9D%98_%ED%95%9C%EA%B8%80_%ED%91%9C%EA%B8%B0" title="위키백과:스페인어의 한글 표기">*</a>]</sup></small>)는 <a href="/wiki/1820%EB%85%84" title="1820년">1820년</a> <a class="new" href="/w/index.php?title=%EC%97%90%EB%B0%94%EB%A6%AC%EC%8A%A4%ED%86%A0_%EC%82%B0_%EB%AF%B8%EA%B5%AC%EC%97%98&amp;action=edit&amp;redlink=1" title="에바리스토 산 미구엘 (없는 문서)">에바리스토 산 미구엘</a>(<span style="font-size: smaller;"><a href="/wiki/%EC%8A%A4%ED%8E%98%EC%9D%B8%EC%96%B4" title="스페인어">스페인어</a>:</span> <span lang="es" xml:lang="es">Evaristo San Miguel</span>)이 작사, 동년에 <a class="new" href="/w/index.php?title=%ED%98%B8%EC%84%B8_%EB%A9%9C%EC%B4%88%EB%A5%B4_%EA%B3%A0%EB%AF%B8%EC%8A%A4&amp;action=edit&amp;redlink=1" title="호세 멜초르 고미스 (없는 문서)">호세 멜초르 고미스</a>(<span style="font-size: smaller;"><a href="/wiki/%EC%8A%A4%ED%8E%98%EC%9D%B8%EC%96%B4" title="스페인어">스페인어</a>:</span> <span lang="es" xml:lang="es">José Melchor Gomis</span>)가 작곡한 <a href="/wiki/%EC%8A%A4%ED%8E%98%EC%9D%B8" title="스페인">스페인</a>의 노래이자 옛 <a href="/wiki/%EA%B5%AD%EA%B0%80_(%EB%85%B8%EB%9E%98)" title="국가 (노래)">국가</a>이기도 하다. 전제군주였던 <a href="/wiki/%ED%8E%98%EB%A5%B4%EB%82%9C%EB%8F%84_7%EC%84%B8" title="페르난도 7세">페르난도 7세</a>에 대항하여 발생했던 자유주의적 반란인 <a href="/wiki/%EC%8A%A4%ED%8E%98%EC%9D%B8_%EB%B0%98%EB%9E%80" title="스페인 반란">스페인 반란</a>이 있었던 <a href="/wiki/1822%EB%85%84" title="1822년">1822년</a>부터 <a href="/wiki/1823%EB%85%84" title="1823년">1823년</a>까지, <a href="/wiki/%EC%95%84%EB%A7%88%EB%8D%B0%EC%98%A4_1%EC%84%B8" title="아마데오 1세">아마데오 1세</a>가 왕으로 즉위했던 <a href="/wiki/1870%EB%85%84" title="1870년">1870년</a>부터 <a href="/wiki/%EC%8A%A4%ED%8E%98%EC%9D%B8_%EC%A0%9C1%EA%B3%B5%ED%99%94%EA%B5%AD" title="스페인 제1공화국">스페인 제1공화국</a>이 <a href="/wiki/%EC%95%8C%ED%8F%B0%EC%86%8C_12%EC%84%B8" title="알폰소 12세">왕정 복고</a>로 전복된 <a href="/wiki/1874%EB%85%84" title="1874년">1874년</a>까지, 마지막으로 <a href="/wiki/%EC%95%8C%ED%8F%B0%EC%86%8C_13%EC%84%B8" title="알폰소 13세">왕정이 전복</a>된 <a href="/wiki/1931%EB%85%84" title="1931년">1931년</a>부터 <a href="/wiki/%EC%8A%A4%ED%8E%98%EC%9D%B8_%EB%82%B4%EC%A0%84" title="스페인 내전">스페인 내전</a>으로 인한 <a href="/wiki/%EC%8A%A4%ED%8E%98%EC%9D%B8_%EC%A0%9C2%EA%B3%B5%ED%99%94%EA%B5%AD" title="스페인 제2공화국">스페인 제2공화국</a>이 <a href="/wiki/%ED%94%84%EB%9E%80%EC%8B%9C%EC%8A%A4%EC%BD%94_%ED%94%84%EB%9E%91%EC%BD%94" title="프란시스코 프랑코">프란시스코 프랑코</a>에 전복된 <a href="/wiki/1939%EB%85%84" title="1939년">1939년</a>까지는 국가였다. 오늘날 스페인의 국가는 <a href="/wiki/1770%EB%85%84" title="1770년">1770년</a>에 제정된 <a href="/wiki/%EC%8A%A4%ED%8E%98%EC%9D%B8%EC%9D%98_%EA%B5%AD%EA%B0%80" title="스페인의 국가">왕의 행진(Marcha Real)</a>이다.</p>
<p>이 노래의 제목에 언급된 사람은 <a class="new" href="/w/index.php?title=%EC%8A%A4%ED%8E%98%EC%9D%B8_%EC%9C%A1%EA%B5%B0&amp;action=edit&amp;redlink=1" title="스페인 육군 (없는 문서)">스페인 육군</a>의 <a href="/wiki/%EC%9E%A5%EA%B5%B0" title="장군">장군</a>인 <a href="/wiki/%EB%9D%BC%ED%8C%8C%EC%97%98_%EB%8D%B8_%EB%A6%AC%EC%97%90%EA%B3%A0" title="라파엘 델 리에고">라파엘 델 리에고</a>(<span style="font-size: smaller;"><a href="/wiki/%EC%8A%A4%ED%8E%98%EC%9D%B8%EC%96%B4" title="스페인어">스페인어</a>:</span> <span lang="es" xml:lang="es">Rafael del Riego y Nuñez</span> <small>라파엘 델 리에고 이 누녜스<sup>[<a href="/wiki/%EC%9C%84%ED%82%A4%EB%B0%B1%EA%B3%BC:%EC%8A%A4%ED%8E%98%EC%9D%B8%EC%96%B4%EC%9D%98_%ED%95%9C%EA%B8%80_%ED%91%9C%EA%B8%B0" title="위키백과:스페인어의 한글 표기">*</a>]</sup></small>)라는 실존 인물이다. 그는 <a href="/wiki/1823%EB%85%84" title="1823년">1823년</a> 스페인 반란이 끝난 후 반란죄로 고발되었고 끝내 <a href="/wiki/%EA%B5%90%EC%88%98%ED%98%95" title="교수형">교수형</a>을 당했다.</p>
