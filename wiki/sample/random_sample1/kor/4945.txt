<p><b>리오 퍼디낸드</b> (<span style="font-size: smaller;"><a href="/wiki/%EC%98%81%EC%96%B4" title="영어">영어</a>:</span> <span lang="en" xml:lang="en">Rio Gavin Ferdinand</span>, <a href="/wiki/1978%EB%85%84" title="1978년">1978년</a> <a href="/wiki/11%EC%9B%94_7%EC%9D%BC" title="11월 7일">11월 7일</a>, <a href="/wiki/%EC%9E%89%EA%B8%80%EB%9E%9C%EB%93%9C" title="잉글랜드">잉글랜드</a> 페컴 ~ )는 <a href="/wiki/%EC%9E%89%EA%B8%80%EB%9E%9C%EB%93%9C" title="잉글랜드">잉글랜드</a>의 은퇴한 <a href="/wiki/%EC%B6%95%EA%B5%AC_%EC%84%A0%EC%88%98" title="축구 선수">축구 선수</a>로, 포지션은 센터백이며 비디치와의 호흡으로 최고의 센터백 중 한명으로 평가 받아오고 있다. <a href="/wiki/%EC%84%B8%EC%9D%B8%ED%8A%B8%EB%A3%A8%EC%8B%9C%EC%95%84" title="세인트루시아">세인트루시아</a>-<a href="/wiki/%EB%8F%84%EB%AF%B8%EB%8B%88%EC%B9%B4_%EC%97%B0%EB%B0%A9" title="도미니카 연방">도미니카</a>인과 <a href="/wiki/%EC%9E%89%EA%B8%80%EB%9E%9C%EB%93%9C" title="잉글랜드">잉글랜드</a>-<a href="/wiki/%EC%95%84%EC%9D%BC%EB%9E%9C%EB%93%9C" title="아일랜드">아일랜드</a>인의 혼혈이다. 그는 <a href="/wiki/%EC%9E%89%EA%B8%80%EB%9E%9C%EB%93%9C_%EC%B6%95%EA%B5%AC_%EA%B5%AD%EA%B0%80%EB%8C%80%ED%91%9C%ED%8C%80" title="잉글랜드 축구 국가대표팀">잉글랜드</a> 대표로, 3회 연속 <a href="/wiki/FIFA_%EC%9B%94%EB%93%9C%EC%BB%B5" title="FIFA 월드컵">월드컵</a>에 참가하였다.</p>
<p>그는 퀸즈 파크 레인저스 유스팀에서 공격수로 축구 선수 생활을 시작했으며, 그 후 <a class="mw-redirect" href="/wiki/%EC%9B%A8%EC%8A%A4%ED%8A%B8_%ED%96%84_%EC%9C%A0%EB%82%98%EC%9D%B4%ED%8B%B0%EB%93%9C_FC" title="웨스트 햄 유나이티드 FC">웨스트 햄 유나이티드</a>에 훈련생 신분으로 들어갔다. 이후 <a href="/wiki/%EB%A6%AC%EC%A6%88_%EC%9C%A0%EB%82%98%EC%9D%B4%ED%8B%B0%EB%93%9C_AFC" title="리즈 유나이티드 AFC">리즈 유나이티드</a>에서 20개월을 뛴 뒤, <a href="/wiki/%EB%A7%A8%EC%B2%B4%EC%8A%A4%ED%84%B0_%EC%9C%A0%EB%82%98%EC%9D%B4%ED%8B%B0%EB%93%9C_FC" title="맨체스터 유나이티드 FC">맨체스터 유나이티드</a>로 이적하였다.</p>
<p>그의 동생 중의 한명인 <a class="mw-redirect" href="/wiki/%EC%95%88%ED%86%A4_%ED%8D%BC%EB%94%94%EB%82%B8%EB%93%9C" title="안톤 퍼디낸드">안톤 퍼디낸드</a>는 그와 같은 웨스트햄 유스 출신으로 현재는 레딩의 수비수이다. 예전에 <a href="/wiki/%EC%9E%89%EA%B8%80%EB%9E%9C%EB%93%9C" title="잉글랜드">잉글랜드</a>의 스트라이커로 뛰었던 <a href="/wiki/%EB%A0%88%EC%8A%A4_%ED%8D%BC%EB%94%94%EB%82%B8%EB%93%9C" title="레스 퍼디낸드">레스 퍼디낸드</a> 와는 사촌지간이다. 퀸즈 파크 레인져스와 계약이 종료된후 <a href="/wiki/2015%EB%85%84" title="2015년">2015년</a> <a href="/wiki/5%EC%9B%94_30%EC%9D%BC" title="5월 30일">5월 30일</a> 현역 은퇴 하였다.</p>

<h2><span class="mw-headline" id=".ED.81.B4.EB.9F.BD_.EA.B2.BD.EB.A0.A5">클럽 경력</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%A6%AC%EC%98%A4_%ED%8D%BC%EB%94%94%EB%82%B8%EB%93%9C&amp;action=edit&amp;section=1" title="부분 편집: 클럽 경력">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<h3><span class="mw-headline" id=".EC.9B.A8.EC.8A.A4.ED.8A.B8_.ED.96.84_.EC.9C.A0.EB.82.98.EC.9D.B4.ED.8B.B0.EB.93.9C">웨스트 햄 유나이티드</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%A6%AC%EC%98%A4_%ED%8D%BC%EB%94%94%EB%82%B8%EB%93%9C&amp;action=edit&amp;section=2" title="부분 편집: 웨스트 햄 유나이티드">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>퀸스 파크 레인져스에서 뛰어난 두각을 보인 그는 처음에 스트라이커로 시작했다가 나중에 센터백 포지션으로 옮겨갔다. 퍼디낸드는 <a href="/wiki/1992%EB%85%84" title="1992년">1992년</a> 그가 Blackheath Bluecoat school의 학생 신분일 때, 웨스트햄 유나이티드와 계약하였고 그곳의 유스 시스템을 거쳐 프로 생활을 시작하였다. <a href="/wiki/1996%EB%85%84" title="1996년">1996년</a> <a href="/wiki/5%EC%9B%94_5%EC%9D%BC" title="5월 5일">5월 5일</a>, 퍼디낸드는 셰필드 웬즈데이와 1:1로 비긴 홈경기에서 Tony Cotee와 교체되어 출전함으로써 성인 클럽에 데뷔하였다. 97-98시즌에, 퍼디낸드는 겨우 20세의 나이로 Hammer of the Year 상을 받았다.</p>
<h3><span class="mw-headline" id=".EB.A6.AC.EC.A6.88_.EC.9C.A0.EB.82.98.EC.9D.B4.ED.8B.B0.EB.93.9C">리즈 유나이티드</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%A6%AC%EC%98%A4_%ED%8D%BC%EB%94%94%EB%82%B8%EB%93%9C&amp;action=edit&amp;section=3" title="부분 편집: 리즈 유나이티드">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>퍼디낸드는 <a href="/wiki/%ED%94%84%EB%A6%AC%EB%AF%B8%EC%96%B4%EB%A6%AC%EA%B7%B8" title="프리미어리그">프리미어리그</a>의 <a href="/wiki/%EB%A6%AC%EC%A6%88_%EC%9C%A0%EB%82%98%EC%9D%B4%ED%8B%B0%EB%93%9C_AFC" title="리즈 유나이티드 AFC">리즈 유나이티드</a>로 <a href="/wiki/2000%EB%85%84" title="2000년">2000년</a> 11월 1800만 파운드로 이적하였고 이것으로 세계에서 가장 비싼 수비수로 기록되게 되었으며 <a href="/wiki/2001%EB%85%84" title="2001년">2001년</a> 8월에는 <a href="/wiki/%EB%A6%AC%EC%A6%88_%EC%9C%A0%EB%82%98%EC%9D%B4%ED%8B%B0%EB%93%9C_AFC" title="리즈 유나이티드 AFC">리즈 유나이티드</a>의 주장이 되었다.</p>
<h3><span class="mw-headline" id=".EB.A7.A8.EC.B2.B4.EC.8A.A4.ED.84.B0_.EC.9C.A0.EB.82.98.EC.9D.B4.ED.8B.B0.EB.93.9C">맨체스터 유나이티드</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%A6%AC%EC%98%A4_%ED%8D%BC%EB%94%94%EB%82%B8%EB%93%9C&amp;action=edit&amp;section=4" title="부분 편집: 맨체스터 유나이티드">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p><a href="/wiki/2002%EB%85%84" title="2002년">2002년</a> <a href="/wiki/7%EC%9B%94_22%EC%9D%BC" title="7월 22일">7월 22일</a>, 퍼디낸드는 또 다른 프리미어리그의 클럽인 <a href="/wiki/%EB%A7%A8%EC%B2%B4%EC%8A%A4%ED%84%B0_%EC%9C%A0%EB%82%98%EC%9D%B4%ED%8B%B0%EB%93%9C_FC" title="맨체스터 유나이티드 FC">맨체스터 유나이티드 FC</a>와의 5년 계약에 사인하였으며 역사상 가장 비싼 <a href="/wiki/%EC%98%81%EA%B5%AD" title="영국">영국</a> 출신 축구 선수가 되었으며, 세계에서 가장 비싼 수비수 기록을 갱신했다. (그가 이전에 세운 기록은 <a href="/wiki/2001%EB%85%84" title="2001년">2001년</a> 투랑에 의해 깨졌었다) 이적료는 2000만 <a class="mw-redirect" href="/wiki/%EC%98%81%EA%B5%AD_%ED%8C%8C%EC%9A%B4%EB%93%9C" title="영국 파운드">파운드</a> 수준이었으며 상황에 따라, 리즈가 그들의 팬들에게 밝힌바에 따르면, 3000만 <a class="mw-redirect" href="/wiki/%EC%98%81%EA%B5%AD_%ED%8C%8C%EC%9A%B4%EB%93%9C" title="영국 파운드">파운드</a> 이상까지 될 수 있다고 한다. 최종적으로 퍼디낸드의 맨체스터 유나이티드로의 이적료는 3300만 <a class="mw-redirect" href="/wiki/%EC%98%81%EA%B5%AD_%ED%8C%8C%EC%9A%B4%EB%93%9C" title="영국 파운드">파운드</a>로 결정이 됐다. 이것은 에이전트의 수입까지 포함한 금액으로 리즈는 2900만 <a class="mw-redirect" href="/wiki/%EC%98%81%EA%B5%AD_%ED%8C%8C%EC%9A%B4%EB%93%9C" title="영국 파운드">파운드</a>를 갖게 되었다.</p>
<p>퍼디낸드는 맨체스터 Utd로 온 첫 해에 프리미어 리그 타이틀을 획득했다. 그는 또한 <a href="/wiki/2006%EB%85%84" title="2006년">2006년</a> 칼링컵에서 또 한번의 우승 메달을 얻었으며 <a href="/wiki/2003%EB%85%84" title="2003년">2003년</a> 칼링컵 준우승, <a href="/wiki/2005%EB%85%84" title="2005년">2005년</a> FA컵 준우승을 경험했다.</p>
<p><a href="/wiki/2005%EB%85%84" title="2005년">2005년</a> <a href="/wiki/12%EC%9B%94_14%EC%9D%BC" title="12월 14일">12월 14일</a>, 위건과의 경기에서 퍼디낸드는 유나이티드에서의 첫 골을 기록했고, 팀은 4-0으로 승리했다. 이것은 그가 올드 트래포드에서 보낸 3번의 시즌 뒤에 첫 골이었다. 그는 계속해서 다음 경기인 웨스트 브롬과의 경기에서도 헤딩골을 기록했다. 또한 그는 리버풀과의 올드 트래포드 홈경기에서 경기 막바지에 골을 넣기도 했으며, 이건 아마도 그가 맨체스터 Utd에서 기록한 골 중에서 가장 중요한 골로 남을 것이다. 다음 시즌인 2006년 10월 22일 리오는 리버풀과의 경기에서 또 한번 2-0의 승리를 거두었다. 퍼디낸드와 그의 동생은 05-06 시즌에 누가 얼마나 많은 골을 넣는가로 비공식적으로 내기를 했는데, 리오가 3-2로 이겼다고 한다.</p>
<p>그는 계속해서 인상적이고 꾸준한 활약을 보여주었고, 리오 퍼디낸드는 7명의 팀동료들과 함께 06-07 시즌 프리미어리그 올해의 팀에 뽑히게 되었다.퍼디낸드는 07-08 시즌도 괜찮은 출발을 보였다. 그는 유나이티드 수비의 일원으로서 2007년 10월 20일에 빌라파크에서 애스턴 빌라에게 골을 허용하기까지 6경기 연속 클린시트를 이끌어냈다. 또한 이 게임에서 그는 이번 시즌 첫 골을 기록하기도 했으며 그것은 게임 중 유나이티드가 기록한 골 중 3번째 골이기도 했다. 3일 뒤에 퍼디낸드는 유럽 대회에서의 첫골을 기록하였는데 디나모 키에프와의 경기에서의 선제골이 바로 그것이다. 유나이티드는 4-2로 승리했다.</p>

<p>2008년 1월 12일 퍼디낸드는 홈에서 뉴캐슬을 6-0으로 대파한 경기에서 또 한번의 골을 기록했다. 2008년 3월 8일 포츠머스와의 FA컵 준결승 경기에서 맨체스터 Utd가 밀리던 때, 그는 잠시 골키퍼를 맡기도 했다. 에드윈 반 데사르가 부상으로 물러나고, 교체된 토마스 쿠쉬착이 페널티킥을 허용하였을 때였다. 하지만 그는 문타리의 페널티킥을 막아내지는 못했고, 이 경기에서 유나이티드는 FA컵에서 탈락했다.</p>
<p>2008년 4월 6일 미들스보로와의 경기에서 퍼디낸드는 부상을 당했다. 그는 2008년 4월 9일 UEFA 챔피언스리그 8강 2차전 경기에서 모습을 드러냈긴 했음에도 그가 부상을 당했을 것이란 추측은 계속되었다. 그는 90분 풀타임을 소화했지만, 하프타임때 치료를 받아야만 했다. <a href="/wiki/FC_%EB%B0%94%EB%A5%B4%EC%85%80%EB%A1%9C%EB%82%98" title="FC 바르셀로나">FC 바르셀로나</a>와 링크가 나기도 했지만, 2008년 4월 16일 그는 마이클 캐릭, 웨스 브라운과 함께 새로운 5년 계약에 합의하였으며 새로운 주급은 13만 파운드로 결정이 되었다. 이것으로 그는 2013년까지 유나이티드에서 뛰게 되었으며 2008년 5월 15일 정식으로 사인했다.</p>
<p>2008년 5월 21일 퍼디낸드는 맨체스터 Utd의 주장으로서 첼시를 상대로 한 챔피언스 리그 결승전을 승리로 이끌었다. 그는 라이언 긱스와 함께 트로피를 들어올리게 되었으며, 긱스는 캡틴 게리 네빌이 부상으로 시즌 대부분을 결장하였을 때, 경기장 위에서 대부분의 경기에서 주장의 임무를 대신했기 때문이다.</p>
<p>2013년 5월13일 스완지와의 13-14시즌 그리고 알렉스 퍼거슨 감독의 마지막 홈경기에서 후반 41분 스코어가 1-1상황에서 반페르시의 코너킥이 퍼디낸드에게 연결됐고 환상적인 논스톱 슈팅으로 골망을 갈랐다. 맨유는 이후 안정적인 경기를 펼쳤고 결국 승리를 따냈고 퍼디낸드는 퍼거슨의 마지막 홈경기에서 승리를 안겨주는 선물을 주었다. 이날 맨유는 홈구장인 올드트래포드에서 퍼거슨과 함께 프리미어리그 우승컵을 들어올렸다.</p>
<p>13-14 시즌이 끝난 후, 퍼디난드는 맨유와의 재계약을 포기하고, 11년 간 함께한 팀과 결별하여 새로운 도전에 임할 준비를 마쳤다.</p>
<h3><span class="mw-headline" id=".ED.80.B8.EC.A6.88_.ED.8C.8C.ED.81.AC_.EB.A0.88.EC.9D.B8.EC.A0.80.EC.8A.A4">퀸즈 파크 레인저스</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%A6%AC%EC%98%A4_%ED%8D%BC%EB%94%94%EB%82%B8%EB%93%9C&amp;action=edit&amp;section=5" title="부분 편집: 퀸즈 파크 레인저스">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>맨유와 결별한 퍼디난드는 14-15 시즌을 앞두고, 퀸즈 파크 레인저스와 1년 계약을 맺고 이적하였지만, 부상과 노쇠화로 인하여 11경기 출전하였다. 그 후, 2015년 5월 30일에 현역 은퇴를 선언하였다.</p>
<h2><span class="mw-headline" id=".EA.B5.AD.EA.B0.80.EB.8C.80.ED.91.9C_.EA.B2.BD.EB.A0.A5">국가대표 경력</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%A6%AC%EC%98%A4_%ED%8D%BC%EB%94%94%EB%82%B8%EB%93%9C&amp;action=edit&amp;section=6" title="부분 편집: 국가대표 경력">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>19세 하고도 8일째, 퍼디낸드는 1997년 <a href="/wiki/11%EC%9B%94_15%EC%9D%BC" title="11월 15일">11월 15일</a> 카메룬과의 친선 경기에서 교체 출전하면서 첫 번째 국대 출전기록을 세웠다. 당시 잉글랜드 국대에서 가장 어린 수비수가 되었는데 이 기록은 2006년 미카 리차즈에 의해 깨어졌다. 20세때, 그는 1998년 FIFA 월드컵 대표팀에 백업 수비수로 뽑혔지만 음주운전으로 인해 글렌 호들 감독은 최종 스쿼드에서 그를 제외하였다. 퍼디낸드는 2002년과 2006년 월드컵 토너먼트에서는 수비수로선 제1의 선택으로 뽑혔다. 퍼디낸드는 잉글랜드 대표팀으로 10번의 월드컵 경기를 치렀는데 잉글랜드 대표팀은 그 중 7번의 클린 시트를 기록했으며 그가 피치 위에 있는 동안 겨우 4골만을 허용했다. (그는 2006년 월드컵에서 스웨덴의 헨릭 라르손이 골을 기록하기 전에 교체되었다) 2008년 3월 25일에는 퍼디낸드가 2008년 5월 26일에 있을 프랑스와의 친선경기에서 잉글랜드의 주장으로 뽑히기도 했다.</p>
<p><a href="/wiki/%EB%93%80%EB%9E%80_%EB%93%80%EB%9E%80" title="듀란 듀란">듀란 듀란</a>의 노래인 "Rio"는 축구 응원가이기도 하지만 사실 퍼디낸드를 위한 것이기도 하다. 2002년 그의 팬인 Simon le Bon (듀란 듀란의 리드 싱어) 는 만약 잉글랜드 대표팀이 승리하면 축구 응원가 중의 하나를 재녹음하기로 약속했다. 그러나 잉글랜드가 져버렸기 때문에 그의 약속이 열매를 맺지는 못했다.</p>
<p>퍼디낸드는 잉글랜드 대표팀으로서 3골을 넣었는데 첫 번째 골은 2002년 월드컵 2차 토너먼트에서 덴마크를 상대로 한 경기에서였다. 두 번째 골은 유로 2008 예선 중 웸블리에서 러시아를 상대로한 2007년 9월 12일에 나왔다. 세 번째 골은 2008년 10월 12일 카자흐스탄과의 2010년 월드컵 예선에서이다. 2008년 3월 25일 리오는 파비오 카펠로 감독이 부임한 두 번째 국대 경기에서 주장 완장을 찰 것이라는 소식을 들었다. FA의 발표에 따르면 리오가 주장으로 호명된 것은 월드컵에서 정식 주장을 결정하기까지 돌려가며 주장을 시켜보겠다는 카펠로의 계획에 따른 것이라고 한다. 그 후 잉글랜드 주장은 존 테리로 결정되었고 퍼디낸드는 부주장에 선임되었다. 그러나 존 테리의 사생활 문제로 인해 주장직을 박탈당한 후 국가대표팀의 주장은 퍼디낸드로 선임되었다.</p>
<h2><span class="mw-headline" id=".EC.82.AC.EC.83.9D.ED.99.9C">사생활</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%A6%AC%EC%98%A4_%ED%8D%BC%EB%94%94%EB%82%B8%EB%93%9C&amp;action=edit&amp;section=7" title="부분 편집: 사생활">편집</a><span class="mw-editsection-bracket">]</span></span></h2>

<div class="thumbinner" style="width:172px;"><a class="image" href="/wiki/%ED%8C%8C%EC%9D%BC:Rio_Ferdinand,_2004.jpg"><img alt="" class="thumbimage" data-file-height="358" data-file-width="257" height="237" src="//upload.wikimedia.org/wikipedia/commons/thumb/3/32/Rio_Ferdinand%2C_2004.jpg/170px-Rio_Ferdinand%2C_2004.jpg" srcset="//upload.wikimedia.org/wikipedia/commons/thumb/3/32/Rio_Ferdinand%2C_2004.jpg/255px-Rio_Ferdinand%2C_2004.jpg 1.5x, //upload.wikimedia.org/wikipedia/commons/3/32/Rio_Ferdinand%2C_2004.jpg 2x" width="170"/></a>

<p>2006년 맨체스터에서 퍼디낸드의 여자친구인 레베카 엘리슨은 그의 로렌츠라는 아들을 낳았다. 2007년 7월, <a href="/wiki/%EB%9D%BC%EC%8A%A4%EB%B2%A0%EC%9D%B4%EA%B1%B0%EC%8A%A4" title="라스베이거스">라스베이거스</a>에서 리오는 엘리슨에게 청혼, 그들은 현재 두 번째 아이를 계획 중이며, 리오는 현재 Altrincham의 Tatton에 살고 있다.</p>
<p>2008년 3월 15일, 리오의 배다른 형제인 제레미야가 맨체스터 유나이티드의 1군과 함께 알렉스 퍼거슨의 지도 아래 훈련을 받기도 했다.</p>
<p>2006년 FIFA 월드컵이 있기 전에 퍼디낸드는 팀 동료인 <a href="/wiki/%EB%8D%B0%EC%9D%B4%EB%B9%84%EB%93%9C_%EC%A0%9C%EC%9E%84%EC%8A%A4_(%EC%B6%95%EA%B5%AC_%EC%84%A0%EC%88%98)" title="데이비드 제임스 (축구 선수)">데이비드 제임스</a>, <a href="/wiki/%ED%94%BC%ED%84%B0_%ED%81%AC%EB%9D%BC%EC%9A%B0%EC%B9%98" title="피터 크라우치">피터 크라우치</a>, <a href="/wiki/%EC%95%A0%EC%8A%90%EB%A6%AC_%EC%BD%9C" title="애슐리 콜">애슐리 콜</a>, <a href="/wiki/%EC%88%80_%EB%9D%BC%EC%9D%B4%ED%8A%B8%ED%95%84%EB%A6%BD%EC%8A%A4" title="숀 라이트필립스">숀 라이트필립스</a>, <a href="/wiki/%EB%8D%B0%EC%9D%B4%EB%B9%84%EB%93%9C_%EB%B2%A0%EC%BB%B4" title="데이비드 베컴">데이비드 베컴</a>, <a href="/wiki/%EC%9B%A8%EC%9D%B8_%EB%A3%A8%EB%8B%88" title="웨인 루니">웨인 루니</a>, <a href="/wiki/%EA%B2%8C%EB%A6%AC_%EB%84%A4%EB%B9%8C" title="게리 네빌">게리 네빌</a>에게 몰래 카메라의 형식으로 장난을 쳤다. (존 테리에게도 장난을 칠 계획이었는데 이건 들통났다.) 퍼디낸드는 그들 몰래 웃기는 상황에 몰아넣었는데 그건 퍼디낸드가 처음에 그들을 유도하는 것으로 시작해서 마지막에 "당했지!(You got merked!)"하고 알려주는 것으로 끝났다.</p>
