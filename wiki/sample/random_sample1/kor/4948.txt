<p><b>리오넬 조스팽</b> (<span style="font-size: smaller;"><a href="/wiki/%ED%94%84%EB%9E%91%EC%8A%A4%EC%96%B4" title="프랑스어">프랑스어</a>:</span> <span lang="fr" xml:lang="fr">Lionel Jospin</span>, <a href="/wiki/1937%EB%85%84" title="1937년">1937년</a> <a href="/wiki/7%EC%9B%94_24%EC%9D%BC" title="7월 24일">7월 24일</a>~)은 <a href="/wiki/%ED%94%84%EB%9E%91%EC%8A%A4" title="프랑스">프랑스</a>의 <a href="/wiki/%EC%A0%95%EC%B9%98%EA%B0%80" title="정치가">정치가</a>로 <a href="/wiki/1997%EB%85%84" title="1997년">1997년</a>부터 <a href="/wiki/2002%EB%85%84" title="2002년">2002년</a>까지 <a href="/wiki/%ED%94%84%EB%9E%91%EC%8A%A4" title="프랑스">프랑스</a>의 <a href="/wiki/%ED%94%84%EB%9E%91%EC%8A%A4%EC%9D%98_%EC%B4%9D%EB%A6%AC" title="프랑스의 총리">총리</a>를 역임하였다. <a href="/wiki/%ED%94%84%EB%9E%91%EC%88%98%EC%95%84_%EB%AF%B8%ED%85%8C%EB%9E%91" title="프랑수아 미테랑">미테랑</a> 정부 하에서 교육부장관과 <a href="/wiki/%EC%82%AC%ED%9A%8C%EB%8B%B9_(%ED%94%84%EB%9E%91%EC%8A%A4)" title="사회당 (프랑스)">사회당</a> 제1서기를 역임하였으며, <a class="new" href="/w/index.php?title=1995%EB%85%84_%ED%94%84%EB%9E%91%EC%8A%A4_%EB%8C%80%ED%86%B5%EB%A0%B9_%EC%84%A0%EA%B1%B0&amp;action=edit&amp;redlink=1" title="1995년 프랑스 대통령 선거 (없는 문서)">1995년 대선</a>에서 <a href="/wiki/%EA%B3%B5%ED%99%94%EA%B5%AD%EC%97%B0%ED%95%A9" title="공화국연합">공화국연합</a>의 <a href="/wiki/%EC%9E%90%ED%81%AC_%EC%8B%9C%EB%9D%BC%ED%81%AC" title="자크 시라크">자크 시라크</a>와 맞붙어 석패를 해 정계의 주목을 받았다. 이를 바탕으로 사회당 제1서기에 재취임한 후 <a href="/wiki/1997%EB%85%84_%ED%94%84%EB%9E%91%EC%8A%A4_%EC%B4%9D%EC%84%A0" title="1997년 프랑스 총선">1997년 총선</a>에서는 1당을 차지하여 <a href="/wiki/%EB%8F%99%EA%B1%B0_%EC%A0%95%EB%B6%80" title="동거 정부">동거 정부</a>를 구성, 주 35시간 노동제 등 여러 정책을 시행하였다. <a class="new" href="/w/index.php?title=2002%EB%85%84_%ED%94%84%EB%9E%91%EC%8A%A4_%EB%8C%80%ED%86%B5%EB%A0%B9_%EC%84%A0%EA%B1%B0&amp;action=edit&amp;redlink=1" title="2002년 프랑스 대통령 선거 (없는 문서)">2002년 대선</a>에서는 <a href="/wiki/%EA%B7%B9%EC%9A%B0" title="극우">극우</a> 성향의 <a class="mw-redirect" href="/wiki/%EC%9E%A5%EB%A7%88%EB%A6%AC_%EB%A5%B4%ED%8E%9C" title="장마리 르펜">장마리 르펜</a>에 밀려 2차 투표 진출에 실패하자 정계 은퇴를 선언하였다.</p>

<h3><span class="mw-headline" id=".EC.B4.88.EA.B8.B0_.EC.83.9D.EC.95.A0">초기 생애</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%A6%AC%EC%98%A4%EB%84%AC_%EC%A1%B0%EC%8A%A4%ED%8C%BD&amp;action=edit&amp;section=2" title="부분 편집: 초기 생애">편집</a><span class="mw-editsection-bracket">]</span></span></h3>

<p>리오넬 조스팽은 <a href="/wiki/1937%EB%85%84" title="1937년">1937년</a> <a href="/wiki/7%EC%9B%94_24%EC%9D%BC" title="7월 24일">7월 24일</a>에 <a href="/wiki/%ED%94%84%EB%9E%91%EC%8A%A4" title="프랑스">프랑스</a> <a class="mw-redirect" href="/wiki/%ED%8C%8C%EB%A6%AC_(%EB%8F%84%EC%8B%9C)" title="파리 (도시)">파리</a> 부근의 <a href="/wiki/%EB%AB%BC%EB%8F%99" title="뫼동">뫼동</a>에서 <a href="/wiki/%EA%B5%90%EC%82%AC" title="교사">교사</a>인 아버지 밑에서 태어났다. 아버지는 독실한 <a href="/wiki/%EA%B0%9C%EC%8B%A0%EA%B5%90" title="개신교">개신교</a> 신자인 동시에 <a class="mw-redirect" href="/wiki/%EC%82%AC%ED%9A%8C%EC%A3%BC%EC%9D%98%EC%9E%90" title="사회주의자">사회주의자</a>였기 때문에 조스팽은 이러한 배경 속에서 <a href="/wiki/%EC%82%AC%ED%9A%8C%EC%A3%BC%EC%9D%98" title="사회주의">사회주의</a>를 지지하게 되었다.<sup class="reference" id="cite_ref-1"><a href="#cite_note-1">[1]</a></sup> <a href="/wiki/%EA%B5%AD%EB%A6%BD%ED%96%89%EC%A0%95%ED%95%99%EA%B5%90" title="국립행정학교">국립행정학교</a>를 입학한 그는 학창 시절 <a href="/wiki/%ED%95%99%EC%83%9D_%EC%9A%B4%EB%8F%99" title="학생 운동">학생 운동</a>에 열렬히 참여하였다. 우수한 성적으로 졸업한 후 그는 외무부에 들어가나 당시 정권을 잡고 있었던 <a class="new" href="/w/index.php?title=%EB%93%9C%EA%B3%A8%EC%A3%BC%EC%9D%98%EC%9E%90&amp;action=edit&amp;redlink=1" title="드골주의자 (없는 문서)">드골주의자</a>들에 대한 비토(거부)로 사직하게 된다. 이후 <a href="/wiki/%EB%AF%B8%EA%B5%AD" title="미국">미국</a>으로 유학을 간 그는 돌아와 <a class="new" href="/w/index.php?title=%ED%8C%8C%EB%A6%AC%EC%A0%9C6%EB%8C%80%ED%95%99%EA%B5%90&amp;action=edit&amp;redlink=1" title="파리제6대학교 (없는 문서)">파리제6대학교</a>에서 <a href="/wiki/%EA%B2%BD%EC%A0%9C%ED%95%99" title="경제학">경제학</a>을 가르쳤다.</p>
<h3><span class="mw-headline" id=".EC.A0.95.EA.B3.84_.EC.9E.85.EB.AC.B8">정계 입문</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%A6%AC%EC%98%A4%EB%84%AC_%EC%A1%B0%EC%8A%A4%ED%8C%BD&amp;action=edit&amp;section=3" title="부분 편집: 정계 입문">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p><a href="/wiki/1971%EB%85%84" title="1971년">1971년</a>에 그는 당시 새롭게 개편된 <a href="/wiki/%EC%82%AC%ED%9A%8C%EB%8B%B9_(%ED%94%84%EB%9E%91%EC%8A%A4)" title="사회당 (프랑스)">사회당</a>에 입당하였다. 그는 당시 제1서기였던 <a href="/wiki/%ED%94%84%EB%9E%91%EC%88%98%EC%95%84_%EB%AF%B8%ED%85%8C%EB%9E%91" title="프랑수아 미테랑">프랑수아 미테랑</a>의 총애를 받으며 주요 보직에 임명되었다. <a href="/wiki/1981%EB%85%84_%ED%94%84%EB%9E%91%EC%8A%A4_%EB%8C%80%ED%86%B5%EB%A0%B9_%EC%84%A0%EA%B1%B0" title="1981년 프랑스 대통령 선거">1981년 대선</a>에서 미테랑이 대통령에 당선되자 그는 뒤이어 제1서기에 취임하게 된다. 그는 <a href="/wiki/1988%EB%85%84" title="1988년">1988년</a>까지 직을 맡으면서 사회당을 전두지휘하였다.</p>
<p><a href="/wiki/1988%EB%85%84" title="1988년">1988년</a>에 미테랑이 <a class="new" href="/w/index.php?title=1988%EB%85%84_%ED%94%84%EB%9E%91%EC%8A%A4_%EB%8C%80%ED%86%B5%EB%A0%B9_%EC%84%A0%EA%B1%B0&amp;action=edit&amp;redlink=1" title="1988년 프랑스 대통령 선거 (없는 문서)">재선</a>을 한 후 뒤이어 <a class="new" href="/w/index.php?title=1988%EB%85%84_%ED%94%84%EB%9E%91%EC%8A%A4_%EC%B4%9D%EC%84%A0&amp;action=edit&amp;redlink=1" title="1988년 프랑스 총선 (없는 문서)">총선</a>에서도 사회당이 승리하자 그는 곧 입각하여 교육스포츠장관에 임명된다. 그는 여러 개혁을 추진하였지만 난관에 부딪히게 되며<sup class="reference" id="cite_ref-2"><a href="#cite_note-2">[2]</a></sup>, <a class="mw-redirect" href="/wiki/%EC%9D%B4%EC%8A%AC%EB%9E%8C%EA%B5%90%EB%8F%84" title="이슬람교도">이슬람교도</a>의 학교 내 <a href="/wiki/%EB%B2%A0%EC%9D%BC" title="베일">베일</a> 착용과 관련해 논란을 일으켰다. 당내 경쟁자들과의 갈등과 미테랑과의 정책 충돌로 그는 <a href="/wiki/1992%EB%85%84" title="1992년">1992년</a>에 새롭게 출범한 내각에서 쫓겨나며 뒤이어 <a href="/wiki/1993%EB%85%84_%ED%94%84%EB%9E%91%EC%8A%A4_%EC%B4%9D%EC%84%A0" title="1993년 프랑스 총선">1993년 총선</a>에서도 자신의 지역구에서 낙선한다.</p>
<h4><span class="mw-headline" id=".EB.8C.80.EC.84.A0_.EC.B6.9C.EB.A7.88">대선 출마</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%A6%AC%EC%98%A4%EB%84%AC_%EC%A1%B0%EC%8A%A4%ED%8C%BD&amp;action=edit&amp;section=4" title="부분 편집: 대선 출마">편집</a><span class="mw-editsection-bracket">]</span></span></h4>
<p><a class="new" href="/w/index.php?title=1995%EB%85%84_%ED%94%84%EB%9E%91%EC%8A%A4_%EB%8C%80%ED%86%B5%EB%A0%B9_%EC%84%A0%EA%B1%B0&amp;action=edit&amp;redlink=1" title="1995년 프랑스 대통령 선거 (없는 문서)">1995년 대선</a>에서 사회당 내 부패 스캔들로 경쟁자들이 휘청거리면서 그는 대선 후보로 선출된다. 학자풍의 이미지로 대중에게 생소하였던 그는 그러나 <a href="/wiki/%EC%9E%90%ED%81%AC_%EC%8B%9C%EB%9D%BC%ED%81%AC" title="자크 시라크">자크 시라크</a>와 <a href="/wiki/%EC%97%90%EB%91%90%EC%95%84%EB%A5%B4_%EB%B0%9C%EB%9D%BC%EB%92%A4%EB%A5%B4" title="에두아르 발라뒤르">에두아르 발라뒤르</a>를 누르고 1차 투표에서 1위를 차지하면서 돌풍을 일으킨다.<sup class="reference" id="cite_ref-3"><a href="#cite_note-3">[3]</a></sup><sup class="reference" id="cite_ref-4"><a href="#cite_note-4">[4]</a></sup> 그러나 2차 투표에서는 시라크에게 53%대 47%로 져 낙선하게 된다.<sup class="reference" id="cite_ref-5"><a href="#cite_note-5">[5]</a></sup></p>
<h3><span class="mw-headline" id=".EC.B4.9D.EB.A6.AC_.EC.9E.AC.EC.9E.84">총리 재임</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%A6%AC%EC%98%A4%EB%84%AC_%EC%A1%B0%EC%8A%A4%ED%8C%BD&amp;action=edit&amp;section=5" title="부분 편집: 총리 재임">편집</a><span class="mw-editsection-bracket">]</span></span></h3>

<a href="/wiki/2000%EB%85%84" title="2000년">2000년</a>에 <a href="/wiki/%EB%B8%94%EB%9D%BC%EB%94%94%EB%AF%B8%EB%A5%B4_%ED%91%B8%ED%8B%B4" title="블라디미르 푸틴">블라디미르 푸틴</a>과 함께</div>

<p>대선에서 돌풍을 바탕으로 그는 사회당 제1서기에 재취임하게 된다. <a href="/wiki/1997%EB%85%84" title="1997년">1997년</a> 시라크는 정국 혼란에 따른 신임을 묻기 위해 <a href="/wiki/1997%EB%85%84_%ED%94%84%EB%9E%91%EC%8A%A4_%EC%B4%9D%EC%84%A0" title="1997년 프랑스 총선">조기 총선</a>을 실시한다. 선거 결과, 사회당이 예상 외로 제1당을 차지하면서 우파 연합을 따돌리게 된다.<sup class="reference" id="cite_ref-6"><a href="#cite_note-6">[6]</a></sup><sup class="reference" id="cite_ref-7"><a href="#cite_note-7">[7]</a></sup> 조스팽은 즉시 <a href="/wiki/%ED%94%84%EB%9E%91%EC%8A%A4_%EA%B3%B5%EC%82%B0%EB%8B%B9" title="프랑스 공산당">공산당</a>과 <a class="new" href="/w/index.php?title=%EB%85%B9%EC%83%89%EB%8B%B9_(%ED%94%84%EB%9E%91%EC%8A%A4)&amp;action=edit&amp;redlink=1" title="녹색당 (프랑스) (없는 문서)">녹색당</a>과 연정 구성을 하여 제3차 <a href="/wiki/%EB%8F%99%EA%B1%B0_%EC%A0%95%EB%B6%80" title="동거 정부">동거 정부</a>를 개각하였다.</p>
<p>총리 재임 동안, 그는 시라크와 의견 대립을 하였으나 기존의 <a class="mw-redirect" href="/wiki/%EC%A2%8C%ED%8C%8C" title="좌파">좌파</a> 이념 대신 <a href="/wiki/%EC%8B%A0%EC%9E%90%EC%9C%A0%EC%A3%BC%EC%9D%98" title="신자유주의">신자유주의</a> 요소를 도입한 정책 시행으로 유연하게 이끌었다. 그의 내각은 노동 문제에 많은 부분을 기울려 주 35시간 노동제를 총선 공약대로 실시하였다. 그의 이런 정책들은 때로 사회당의 성향과 맞지 않다는 지적도 받았으나 재임 동안 높은 경제 성장률과 낮은 실업률을 바탕으로 높은 인기를 누렸다.<sup class="reference" id="cite_ref-8"><a href="#cite_note-8">[8]</a></sup></p>
<h3><span class="mw-headline" id=".EB.8C.80.EC.84.A0_.EB.82.99.EC.84.A0_.EB.B0.8F_.EC.A0.95.EA.B3.84_.EC.9D.80.ED.87.B4">대선 낙선 및 정계 은퇴</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%A6%AC%EC%98%A4%EB%84%AC_%EC%A1%B0%EC%8A%A4%ED%8C%BD&amp;action=edit&amp;section=6" title="부분 편집: 대선 낙선 및 정계 은퇴">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p><a href="/wiki/2001%EB%85%84" title="2001년">2001년</a> 불경기로 인하여 지방 선거에서 패배하는 등 그의 인기도 하락하게 되었다. <a class="new" href="/w/index.php?title=2002%EB%85%84_%ED%94%84%EB%9E%91%EC%8A%A4_%EB%8C%80%ED%86%B5%EB%A0%B9_%EC%84%A0%EA%B1%B0&amp;action=edit&amp;redlink=1" title="2002년 프랑스 대통령 선거 (없는 문서)">2002년 대선</a>에서 그는 이 점을 받아들여 우파적 요소를 공약에 넣은 후 출마하였다. 그의 태도에 많은 논란이 있은 후 여러 좌파 후보가 대선에 출마하였다. 1차 투표 결과, 좌파 후보의 난립속에 그는 <a href="/wiki/%EA%B7%B9%EC%9A%B0" title="극우">극우</a> 성향의 <a href="/wiki/%EC%9E%A5%EB%A7%88%EB%A6%AC_%EB%A5%B4_%ED%8E%9C" title="장마리 르 펜">장마리 르 펜</a>에 밀려 3위를 기록, 2차 투표 진출에 실패하였다. 그는 시라크 지지 선언을 하였으며 2차 투표 당일날 총리직 사퇴는 물론 정계 은퇴를 선언하였다. <a href="/wiki/2005%EB%85%84" title="2005년">2005년</a>에 그는 <a class="new" href="/w/index.php?title=2007%EB%85%84_%EB%8C%80%ED%86%B5%EB%A0%B9_%EC%84%A0%EA%B1%B0&amp;action=edit&amp;redlink=1" title="2007년 대통령 선거 (없는 문서)">2007년 대선</a> 출마를 위해 은퇴 번복을 선언하였다. 그는 곧이어 경선에 출마하였지만 <a href="/wiki/%EC%84%B8%EA%B3%A8%EB%A0%8C_%EB%A3%A8%EC%95%84%EC%96%84" title="세골렌 루아얄">세골렌 루아얄</a>에게 밀리자 경선 도중 사퇴하였다. <a href="/wiki/2014%EB%85%84" title="2014년">2014년</a>에는 <a class="mw-redirect" href="/wiki/%EB%82%98%ED%8F%B4%EB%A0%88%EC%98%B9" title="나폴레옹">나폴레옹</a>에 대한 저서 《나폴레옹의 폐해》를 발표해 베스트셀러 1위를 기록하기도 하였다.<sup class="reference" id="cite_ref-9"><a href="#cite_note-9">[9]</a></sup></p>
