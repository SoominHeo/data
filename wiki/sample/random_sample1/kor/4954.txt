<h3><span class="mw-headline" id=".ED.81.B4.EB.9F.BD_.EA.B2.BD.EB.A0.A5">클럽 경력</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%A6%AC%EC%9B%A8%EC%9D%B4%ED%8E%91&amp;action=edit&amp;section=2" title="부분 편집: 클럽 경력">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p><a href="/wiki/1998%EB%85%84" title="1998년">1998년</a> <a href="/wiki/%EC%A4%91%EA%B5%AD_%EC%8A%88%ED%8D%BC%EB%A6%AC%EA%B7%B8" title="중국 슈퍼리그">중국 슈퍼리그</a>의 <a class="mw-redirect" href="/wiki/%EC%84%A0%EC%A0%84_%ED%9B%99%EC%AB%9C" title="선전 훙쫜">선전 핑안</a>에서 프로 생활을 시작하였고, <a href="/wiki/2002%EB%85%84_FIFA_%EC%9B%94%EB%93%9C%EC%BB%B5" title="2002년 FIFA 월드컵">2002년 FIFA 월드컵</a>에서 좋은 활약을 보인 뒤 <a class="mw-redirect" href="/wiki/%EC%9E%89%EA%B8%80%EB%9E%9C%EB%93%9C_%ED%94%84%EB%A6%AC%EB%AF%B8%EC%96%B4%EB%A6%AC%EA%B7%B8" title="잉글랜드 프리미어리그">잉글랜드 프리미어리그</a>의 <a href="/wiki/%EC%97%90%EB%B2%84%ED%84%B4_FC" title="에버턴 FC">에버턴 FC</a>에서 입단 테스트를 받았다. <a href="/wiki/2002%EB%85%84" title="2002년">2002년</a> <a href="/wiki/%EC%A4%91%ED%99%94%EC%9D%B8%EB%AF%BC%EA%B3%B5%ED%99%94%EA%B5%AD" title="중화인민공화국">중화인민공화국</a>의 <a class="mw-redirect" href="/wiki/%ED%95%B8%EB%93%9C%ED%8F%B0" title="핸드폰">핸드폰</a> 메이커 회사인 <a class="new" href="/w/index.php?title=%EC%BB%A4%EC%A6%88%EC%95%88&amp;action=edit&amp;redlink=1" title="커즈안 (없는 문서)">커즈안</a>이 2년 동안 팀의 <a href="/wiki/%EC%8A%A4%ED%8F%B0%EC%84%9C" title="스폰서">스폰서</a>를 맡는다는 계약의 일환으로 <a href="/wiki/%EC%97%90%EB%B2%84%ED%84%B4_FC" title="에버턴 FC">에버턴 FC</a>에 임대 이적하였으나, 코칭스태프에 별다른 인상을 남기지 못하며 단 1경기 출전에 그쳤고, 시즌이 끝난 뒤 <a class="mw-redirect" href="/wiki/%EC%84%A0%EC%A0%84_%ED%9B%99%EC%AB%9C" title="선전 훙쫜">선전 젠리바오</a>로 복귀하였다.<sup class="reference" id="cite_ref-1"><a href="#cite_note-1">[1]</a></sup> <a href="/wiki/2006%EB%85%84" title="2006년">2006년</a>이 시작되자, 그는 소속팀인 <a class="mw-redirect" href="/wiki/%EC%84%A0%EC%A0%84_%ED%9B%99%EC%AB%9C" title="선전 훙쫜">선전 젠리바오</a>의 재정 문제로 인하여 팀을 떠나게 되었고, 많은 클럽이 그를 주시하였다. <a class="mw-redirect" href="/wiki/%EC%9D%B4%ED%83%88%EB%A6%AC%EC%95%84_%EC%84%B8%EB%A6%AC%EC%97%90_A" title="이탈리아 세리에 A">이탈리아 세리에 A</a>의 <a href="/wiki/ACF_%ED%94%BC%EC%98%A4%EB%A0%8C%ED%8B%B0%EB%82%98" title="ACF 피오렌티나">ACF 피오렌티나</a>의 관심을 받았지만, 600만 <a class="mw-redirect" href="/wiki/%EC%9C%84%EC%95%88" title="위안">위안</a>에 <a class="mw-redirect" href="/wiki/%EC%83%81%ED%95%98%EC%9D%B4_%EC%84%A0%ED%99%94" title="상하이 선화">상하이 선화</a>로 이적하였다.</p>
<p>그는 <a href="/wiki/2005%EB%85%84" title="2005년">2005년</a> <a href="/wiki/5%EC%9B%94_20%EC%9D%BC" title="5월 20일">5월 20일</a> <a class="mw-redirect" href="/wiki/%EC%84%A0%EC%A0%84_%ED%9B%99%EC%AB%9C" href="/wiki/AFC_%EC%B1%94%ED%94%BC%EC%96%B8%EC%8A%A4%EB%A6%AC%EA%B7%B8" title="AFC 챔피언스리그">AFC 챔피언스리그</a> 조별 예선 마지막 경기에서 <a class="new" href="/w/index.php?title=%EC%8B%A0%ED%8E%91&amp;action=edit&amp;redlink=1" title="신펑 (없는 문서)">신펑</a>의 결승골을 어시스트하여 팀의 1-0 승리에 공헌하였고, 팀은 1, 2차전 합계 결과 <a href="/wiki/%EC%88%98%EC%9B%90_%EC%82%BC%EC%84%B1_%EB%B8%94%EB%A3%A8%EC%9C%99%EC%A6%88" title="수원 삼성 블루윙즈">수원 삼성 블루윙즈</a>와 승점에서 동률을 이룬 뒤 상대전적에서 1승 1무로 앞서, 팀의 8강 진출에 크게 기여하였다.<sup class="reference" id="cite_ref-2"><a href="#cite_note-2">[2]</a></sup> 하지만 <a href="/wiki/2006%EB%85%84" title="2006년">2006년</a> <a href="/wiki/9%EC%9B%94_20%EC%9D%BC" title="9월 20일">9월 20일</a> <a class="new" href="/w/index.php?title=%EC%83%81%ED%99%94%EC%9D%B4_%EC%84%A0%ED%99%94&amp;action=edit&amp;redlink=1" title="상화이 선화 (없는 문서)">상화이 선화</a> 소속으로 참가한 <a href="/wiki/%EC%A0%84%EB%B6%81_%ED%98%84%EB%8C%80_%EB%AA%A8%ED%84%B0%EC%8A%A4" title="전북 현대 모터스">전북 현대 모터스</a>와의 <a href="/wiki/AFC_%EC%B1%94%ED%94%BC%EC%96%B8%EC%8A%A4%EB%A6%AC%EA%B7%B8" title="AFC 챔피언스리그">AFC 챔피언스리그</a> 8강 2차전에서 <a href="/wiki/%EC%A1%B0%EC%A0%9C_%EC%B9%B4%EB%A5%BC%EB%A3%A8%EC%8A%A4_%ED%8E%98%ED%97%A4%EC%9D%B4%EB%9D%BC_%ED%95%84%EB%A5%98" title="조제 카를루스 페헤이라 필류">제칼로</a>의 복부를 발로 걷어차 <a href="/wiki/%EB%A0%88%EB%93%9C%EC%B9%B4%EB%93%9C" title="레드카드">레드카드</a>를 받았고, 결국 팀은 1, 2차전 합계 결과 8강에서 탈락하게 되었다.<sup class="reference" id="cite_ref-3"><a href="#cite_note-3">[3]</a></sup> 경기 후 그는 팀의 감독인 <a class="new" href="/w/index.php?title=%EC%9A%B0%EC%A7%84%EC%BF%A0%EC%9D%B4&amp;action=edit&amp;redlink=1" title="우진쿠이 (없는 문서)">우진쿠이</a>로부터 팀의 8강 탈락의 원인이라는 지적을 받게 되었으며,<sup 벌금 3천 <a href="/wiki/%EB%8B%AC%EB%9F%AC" title="달러">달러</a>의 벌금을 부과받는 징계를 받았다.<sup class="reference" id="cite_ref-5"><a href="#cite_note-5">[5]</a></sup></p>
<p><a href="/wiki/2008%EB%85%84" title="2008년">2008년</a> 소속팀에서 출전 기회를 잡지 못하자, 그는 리그 준우승 팀인 <a class="mw-redirect" href="/wiki/%EC%83%81%ED%95%98%EC%9D%B4_%EC%84%A0%ED%99%94" title="상하이 선화">상하이 선화</a>를 떠나 350만 <a class="mw-redirect" href="/wiki/%EC%9C%84%EC%95%88" title="위안">위안</a>에 예전 <a class="mw-redirect" href="/wiki/%EC%84%A0%EC%A0%84_%ED%9B%99%EC%AB%9C" title="선전 훙쫜">선전 핑안</a>과 <a href="/wiki/%EC%A4%91%ED%99%94%EC%9D%B8%EB%AF%BC%EA%B3%B5%ED%99%94%EA%B5%AD_%EC%B6%95%EA%B5%AC_%EA%B5%AD%EA%B0%80%EB%8C%80%ED%91%9C%ED%8C%80" title="중화인민공화국 축구 국가대표팀">중화인민공화국 국가대표팀</a>의 감독이었던 <a class="new" href="/w/index.php?title=%EC%A3%BC%EA%B4%91%ED%9B%84&amp;action=edit&amp;redlink=1" title="주광후 (없는 문서)">주광후</a>가 지휘봉을 잡고 있는 <a href="/wiki/%EC%9A%B0%ED%95%9C_%EA%B4%91%EA%B5%AC" title="우한 광구">우한 광구</a>로 이적하였다. 그러나 <a href="/wiki/2008%EB%85%84" title="2008년">2008년</a> <a href="/wiki/9%EC%9B%94_28%EC%9D%BC" title="9월 28일">9월 28일</a> <a href="/wiki/%EB%B2%A0%EC%9D%B4%EC%A7%95_%EA%B6%88%EC%95%88" title="베이징 궈안">베이징 궈안</a>과의 리그 경기에서 난투극에 휘말렸고, <a class="new" href="/w/index.php?title=%EC%A4%91%ED%99%94%EC%9D%B8%EB%AF%BC%EA%B3%B5%ED%99%94%EA%B5%AD_%EC%B6%95%EA%B5%AC%ED%98%91%ED%9A%8C&amp;action=edit&amp;redlink=1" title="중화인민공화국 축구협회 (없는 문서)">중화인민공화국 축구협회</a>로부터 6경기 출장 정지와 벌금 8천 <a class="mw-redirect" href="/wiki/%EC%9C%84%EC%95%88" title="위안">위안</a>의 벌금을 부과받는 중징계를 받았다. 이 징계로 <a href="/wiki/%EC%A4%91%EA%B5%AD_%EC%8A%88%ED%8D%BC%EB%A6%AC%EA%B7%B8" title="중국 슈퍼리그">1부리그</a> 잔류에 매우 큰 타격을 입게 된 소속팀 <a href="/wiki/%EC%9A%B0%ED%95%9C_%EA%B4%91%EA%B5%AC" title="우한 광구">우한 광구</a>는 격렬히 항의하였지만 소용이 없자, 징계에 불복하여 심판을 폭행하고 잔여 경기를 거부하여 2008 시즌 전 경기가 몰수패 처리되고 2부리그로 강등되었다.</p>
<p>2부리그로의 강등이 결정되자, <a href="/wiki/%EC%9A%B0%ED%95%9C_%EA%B4%91%EA%B5%AC" title="우한 광구">우한 광구</a>에 소속된 비연고 출신의 많은 연봉을 받는 선수는 이적 명단에 이름을 올리게 되었고, 리웨이펑도 이에 포함되었다. 하지만 높은 연봉과 많은 이적료, 6경기 출장 정지의 징계 때문에, 리그 내의 다른 수비수들보다 뛰어난 능력을 갖고 있다는 평가에도 불구하고 이적할 팀을 구하기 힘들게 되었다. 하지만 <a href="/wiki/%EC%95%84%EC%8B%9C%EC%95%84_%EC%B6%95%EA%B5%AC_%EC%97%B0%EB%A7%B9" title="아시아 축구 연맹">아시아 축구 연맹</a>에서 '아시아쿼터제'를 도입하자, <a class="mw-redirect" href="/wiki/K-%EB%A6%AC%EA%B7%B8" title="K-리그">K-리그</a>와 <a href="/wiki/J%EB%A6%AC%EA%B7%B8" title="J리그">J리그</a>의 여러 클럽들이 그의 영입을 원하였고, <a href="/wiki/2009%EB%85%84" title="2009년">2009년</a> <a href="/wiki/1%EC%9B%94_9%EC%9D%BC" title="1월 9일">1월 9일</a> 2년 계약을 맺고 프로로 데뷔할 당시 <a class="mw-redirect" href="/wiki/%EC%84%A0%EC%A0%84_%ED%9B%99%EC%AB%9C" title="선전 훙쫜">선전 핑안</a>의 감독이었던 <a href="/wiki/%EC%B0%A8%EB%B2%94%EA%B7%BC" title="차범근">차범근</a>이 지휘봉을 잡고 있는 <a href="/wiki/%EC%88%98%EC%9B%90_%EC%82%BC%EC%84%B1_%EB%B8%94%EB%A3%A8%EC%9C%99%EC%A6%88" title="수원 삼성 블루윙즈">수원 삼성 블루윙즈</a>로 이적하면서 <a class="mw-redirect" href="/wiki/K-%EB%A6%AC%EA%B7%B8" title="K-리그">K-리그</a>에서 뛴 최초의 <a href="/wiki/%EC%A4%91%ED%99%94%EC%9D%B8%EB%AF%BC%EA%B3%B5%ED%99%94%EA%B5%AD" title="중화인민공화국">중화인민공화국</a> 선수가 되었다.<sup class="reference" id="cite_ref-6"><a href="#cite_note-6">[6]</a></sup><sup class="reference" id="cite_ref-7"><a href="#cite_note-7">[7]</a></sup></p>
<p><a href="/wiki/%EC%88%98%EC%9B%90_%EC%82%BC%EC%84%B1_%EB%B8%94%EB%A3%A8%EC%9C%99%EC%A6%88" title="수원 삼성 블루윙즈">수원 삼성 블루윙즈</a>으로 이적한 후에 가진 첫 경기인 시즌 전 <a class="mw-redirect" href="/wiki/4%EC%9B%94_7%EC%9D%BC" title="4월 7일">4월 7일</a> <a class="mw-redirect" href="/wiki/%EC%83%81%ED%95%98%EC%9D%B4_%EC%84%A0%ED%99%94" title="상하이 선화">상하이 선화</a>와의 <a href="/wiki/%EC%83%81%ED%95%98%EC%9D%B4_%EC%84%A0%ED%99%94" title="상하이 선화">상하이 선화</a>와의 홈 경기에서 석연치 않은 판정으로 <a href="/wiki/%EC%88%98%EC%9B%90_%EC%82%BC%EC%84%B1_%EB%B8%94%EB%A3%A8%EC%9C%99%EC%A6%88" href="/wiki/10%EC%9B%94_18%EC%9D%BC" title="10월 18일">10월 18일</a> <a class="mw-redirect" href="/wiki/%EC%84%B1%EB%82%A8_%EC%9D%BC%ED%99%94_%EC%B2%9C%EB%A7%88" title="성남 일화 천마">성남 일화 천마</a>와의 리그 28라운드 경기에서 팀이 0-1로 뒤진 상황에서 동점골을 성공시켰으나, 결국 팀은 2-3으로 패하여 6강 <a href="/wiki/%ED%94%8C%EB%A0%88%EC%9D%B4%EC%98%A4%ED%94%84" title="플레이오프">플레이오프</a> 경쟁에서 탈락하였다.<sup class="reference" id="cite_ref-11"><a href="#cite_note-11">[11]</a></sup></p>
<p>좋은 활약으로 <a href="/wiki/K%EB%A6%AC%EA%B7%B8_2009" title="K리그 2009">2009년</a>을 마무리한 이후 <a href="/wiki/%EC%A0%84%EB%B6%81_%ED%98%84%EB%8C%80_%EB%AA%A8%ED%84%B0%EC%8A%A4" title="전북 현대 모터스">전북 현대 모터스</a>로부터 높은 연봉의 러브콜을 받았으나 <a href="/wiki/%EC%88%98%EC%9B%90_%EC%82%BC%EC%84%B1_%EB%B8%94%EB%A3%A8%EC%9C%99%EC%A6%88" title="수원 삼성 블루윙즈">수원 삼성 블루윙즈</a>와 <a href="/wiki/%EC%B0%A8%EB%B2%94%EA%B7%BC" title="차범근">차범근</a> 감독에 대한 의리를 지키기 위해 거절하였다.</p>
<p><a href="/wiki/K%EB%A6%AC%EA%B7%B8_2010" title="K리그 2010">2010 시즌</a>에서도 좋은 활약을 보이며 팀 수비의 중추적인 역할을 해왔으나 결국 자신을 데려온 <a href="/wiki/%EC%B0%A8%EB%B2%94%EA%B7%BC" title="차범근">차범근</a> 감독이 시즌 도중 성적 부진으로 인해 해임되고 새롭게 부임한 <a href="/wiki/%EC%9C%A4%EC%84%B1%ED%9A%A8" title="윤성효">윤성효</a> 감독의 리빌딩에 의해 중국의 <a href="/wiki/%ED%86%88%EC%A7%84_%ED%84%B0%EB%8B%A4" title="톈진 터다">톈진 터다</a>로 이적하였다.</p>
<h3><span class="mw-headline" id=".EA.B5.AD.EA.B0.80.EB.8C.80.ED.91.9C_.EA.B2.BD.EB.A0.A5">국가대표 경력</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%A6%AC%EC%9B%A8%EC%9D%B4%ED%8E%91&amp;action=edit&amp;section=3" title="부분 편집: 국가대표 경력">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p><a href="/wiki/1998%EB%85%84" title="1998년">1998년</a> <a href="/wiki/11%EC%9B%94_22%EC%9D%BC" title="11월 22일">11월 22일</a> <a class="mw-redirect" href="/wiki/%EC%83%81%ED%95%98%EC%9D%B4" title="상하이">상하이</a>에서 열린 <a href="/wiki/%EB%8C%80%ED%95%9C%EB%AF%BC%EA%B5%AD_%EC%B6%95%EA%B5%AC_%EA%B5%AD%EA%B0%80%EB%8C%80%ED%91%9C%ED%8C%80" title="대한민국 축구 국가대표팀">대한민국</a>과의 친선경기에서 <a href="/wiki/%EC%A4%91%ED%99%94%EC%9D%B8%EB%AF%BC%EA%B3%B5%ED%99%94%EA%B5%AD_%EC%B6%95%EA%B5%AC_%EA%B5%AD%EA%B0%80%EB%8C%80%ED%91%9C%ED%8C%80" title="중화인민공화국 축구 국가대표팀">중화인민공화국 국가대표팀</a>에 데뷔하였고, <a href="/wiki/1998%EB%85%84" title="1998년">1998년</a> <a href="/wiki/12%EC%9B%94_12%EC%9D%BC" title="12월 12일">12월 12일</a> <a href="/wiki/%EB%B0%A9%EC%BD%95" title="방콕">방콕</a>에서 열린 <a href="/wiki/%EC%98%A4%EB%A7%8C_%EC%B6%95%EA%B5%AC_%EA%B5%AD%EA%B0%80%EB%8C%80%ED%91%9C%ED%8C%80" title="오만 축구 국가대표팀">오만</a>과의 <a href="/wiki/1998%EB%85%84_%EC%95%84%EC%8B%9C%EC%95%88_%EA%B2%8C%EC%9E%84" title="1998년 아시안 게임">1998년 아시안 게임</a> 경기에서 <a href="/wiki/A%EB%A7%A4%EC%B9%98" title="A매치">A매치</a> 첫 득점을 올렸다.</p>
<p>그는 <a href="/wiki/2000%EB%85%84_AFC_%EC%95%84%EC%8B%9C%EC%95%88%EC%BB%B5" title="2000년 AFC 아시안컵">2000년 AFC 아시안컵</a>에서 조별 예선 첫 경기인 <a href="/wiki/%EB%8C%80%ED%95%9C%EB%AF%BC%EA%B5%AD_%EC%B6%95%EA%B5%AC_%EA%B5%AD%EA%B0%80%EB%8C%80%ED%91%9C%ED%8C%80" title="대한민국 축구 국가대표팀">대한민국</a>전을 제외한 전 경기에서 선발 출전하여, 팀이 4위를 차지하는 데 크게 기여하였고, <a href="/wiki/2002%EB%85%84_FIFA_%EC%9B%94%EB%93%9C%EC%BB%B5" title="2002년 FIFA 월드컵">2002년 FIFA 월드컵</a>에서 전 경기에 선발 출전하였으나, 팀은 16강 진출에 실패하였다. 이후 <a href="/wiki/2004%EB%85%84_AFC_%EC%95%84%EC%8B%9C%EC%95%88%EC%BB%B5" title="2004년 AFC 아시안컵">2004년 AFC 아시안컵</a>에서 결승전인 <a href="/wiki/%EC%9D%BC%EB%B3%B8_%EC%B6%95%EA%B5%AC_%EA%B5%AD%EA%B0%80%EB%8C%80%ED%91%9C%ED%8C%80" title="일본 축구 국가대표팀">일본</a>전을 제외한 전 경기에서 선발 출전하여, 팀이 준우승을 차지하는 데 크게 기여하였고, <a href="/wiki/2007%EB%85%84_AFC_%EC%95%84%EC%8B%9C%EC%95%88%EC%BB%B5" title="2007년 AFC 아시안컵">2007년 AFC 아시안컵</a>에도 참가하였으나, 팀은 8강 진출에 실패하였다.</p>
<p>그는 <a href="/wiki/2003%EB%85%84" title="2003년">2003년</a> <a href="/wiki/%EC%95%84%EB%A6%AC_%ED%95%9C" title="아리 한">아리 한</a>이 국가대표팀 감독으로 취임한 뒤 팀의 주장을 맡았다. 하지만 14개월이라는 기간동안 6번의 퇴장을 당하여 <a href="/wiki/2006%EB%85%84" title="2006년">2006년</a> 국가대표팀 주장 자리를 반납하였고, 후임은 <a href="/wiki/%EC%A0%95%EC%A6%88" title="정즈">정즈</a>가 맡게 되었다.</p>
<p>그는 <a href="/wiki/2008%EB%85%84" title="2008년">2008년</a> <a href="/wiki/2%EC%9B%94_20%EC%9D%BC" title="2월 20일">2월 20일</a> <a class="mw-redirect" href="/wiki/%EC%B6%A9%EC%B9%AD" title="충칭">충칭</a>에서 열린 <a href="/wiki/%EC%9D%BC%EB%B3%B8_%EC%B6%95%EA%B5%AC_%EA%B5%AD%EA%B0%80%EB%8C%80%ED%91%9C%ED%8C%80" title="일본 축구 국가대표팀">일본</a>과의 <a class="mw-redirect" href="/wiki/2008%EB%85%84_%EB%8F%99%EC%95%84%EC%8B%9C%EC%95%84%EC%BB%B5" title="2008년 동아시아컵">2008년 동아시아컵</a> 경기에서 <a href="/wiki/%ED%8C%90%EC%A6%88%EC%9D%B4" title="판즈이">판즈이</a>, <a class="mw-redirect" href="/wiki/%ED%95%98%EC%98%A4%ED%95%98%EC%9D%B4%EB%8F%99" title="하오하이동">하오하이동</a>에 이어 3번째로 <a class="mw-redirect" href="/wiki/%EC%84%BC%EC%B6%94%EB%A6%AC_%ED%81%B4%EB%9F%BD" title="센추리 클럽">센추리 클럽</a>에 가입한 <a href="/wiki/%EC%A4%91%ED%99%94%EC%9D%B8%EB%AF%BC%EA%B3%B5%ED%99%94%EA%B5%AD" title="중화인민공화국">중화인민공화국</a>선수가 되었다.</p>
<p>이후 자국에서 열린 <a href="/wiki/2008%EB%85%84_%ED%95%98%EA%B3%84_%EC%98%AC%EB%A6%BC%ED%94%BD" title="2008년 하계 올림픽">2008년 하계 올림픽</a>에서 <a href="/wiki/%EC%A0%95%EC%A6%88" title="정즈">정즈</a>, <a href="/wiki/%ED%95%9C%ED%8E%91" title="한펑">한펑</a>과 함께 <a href="/wiki/%EC%99%80%EC%9D%BC%EB%93%9C%EC%B9%B4%EB%93%9C_(%EC%8A%A4%ED%8F%AC%EC%B8%A0)" title="와일드카드 (스포츠)">와일드카드</a>로 선발되어 전 경기에 선발 출전하였고,<sup class="reference" id="cite_ref-12"><a href="#cite_note-12">[12]</a></sup> 특히 <a href="/wiki/2008%EB%85%84" title="2008년">2008년</a> <a href="/wiki/8%EC%9B%94_13%EC%9D%BC" title="8월 13일">8월 13일</a> <a href="/wiki/%EB%B8%8C%EB%9D%BC%EC%A7%88_%EC%B6%95%EA%B5%AC_%EA%B5%AD%EA%B0%80%EB%8C%80%ED%91%9C%ED%8C%80" title="브라질 축구 국가대표팀">브라질</a>과의 경기에서 결장한 <a href="/wiki/%EC%A0%95%EC%A6%88" title="정즈">정즈</a>를 대신하여 팀의 주장을 맡기도 하였으나, 팀은 8강 진출에 실패하였다.</p>
<h2><span class="mw-headline" id=".ED.94.8C.EB.A0.88.EC.9D.B4_.EC.8A.A4.ED.83.80.EC.9D.BC">플레이 스타일</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%A6%AC%EC%9B%A8%EC%9D%B4%ED%8E%91&amp;action=edit&amp;section=4" title="부분 편집: 플레이 스타일">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p><a href="/wiki/%EC%A4%91%ED%99%94%EC%9D%B8%EB%AF%BC%EA%B3%B5%ED%99%94%EA%B5%AD_%EC%B6%95%EA%B5%AC_%EA%B5%AD%EA%B0%80%EB%8C%80%ED%91%9C%ED%8C%80" title="중화인민공화국 축구 국가대표팀">국가대표팀</a>에서 <a class="mw-redirect" href="/wiki/%EC%84%BC%EC%B6%94%EB%A6%AC%ED%81%B4%EB%9F%BD" title="센추리클럽">센추리클럽</a> 에 가입했을 정도로 풍부한 경험을 바탕으로 한 노련한 수비를 자랑한다.</p>
<p>하지만 쉽게 흥분을 하는 성격과 거친 말투로 인하여 종종 <a href="/wiki/%EB%A0%88%EB%93%9C%EC%B9%B4%EB%93%9C" title="레드카드">레드카드</a>를 받기도 하였으나, <a href="/wiki/K%EB%A6%AC%EA%B7%B8" title="K리그">K리그</a> 진출 이후에는 그런 점이 많이 사라졌다.</p>

국가대표팀">조선민주주의인민공화국</a>전에 출전할 수 없게 되었다.<sup class="reference" id="cite_ref-13"><a href="#cite_note-13">[13]</a></sup><sup class="reference" id="cite_ref-14"><a href="#cite_note-14">[14]</a></sup> <a <p><a href="/wiki/2005%EB%85%84" title="2005년">2005년</a> <a href="/wiki/9%EC%9B%94_22%EC%9D%BC" title="9월 22일">9월 22일</a> <a href="/wiki/%EC%A4%91%ED%99%94%EC%9D%B8%EB%AF%BC%EA%B3%B5%ED%99%94%EA%B5%AD" title="중화인민공화국">중화인민공화국</a>의 한 언론과의 인터뷰에서 자신이 한류드라마 <a href="/wiki/%EB%8C%80%EC%9E%A5%EA%B8%88" title="대장금">대장금</a>의 팬임을 밝히고, 최근 자신의 어려운 상황을 '대장금 정신'으로 이겨내겠다는 의지를 나타내기도 하였다.<sup class="reference" id="cite_ref-17"><a href="#cite_note-17">[17]</a></sup></p>
