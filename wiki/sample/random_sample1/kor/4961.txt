<p><b>리처드 유언 보처즈</b>(<span style="font-size: smaller;"><a href="/wiki/%EC%98%81%EC%96%B4" title="영어">영어</a>:</span> <span lang="en" xml:lang="en">Richard Ewen Borcherds</span> <small><a href="/wiki/%EC%9C%84%ED%82%A4%EB%B0%B1%EA%B3%BC:IPA" title="위키백과:IPA">IPA</a>: </small><span class="IPA" title="IPA 발음">[ˈrɪtʃə(r)d ˈjuːən ˈbɔ(r)tʃə(r)dz]</span>, <a href="/wiki/1959%EB%85%84" title="1959년">1959년</a> <a href="/wiki/11%EC%9B%94_29%EC%9D%BC" title="11월 29일">11월 29일</a> ~ )는 <a href="/wiki/%EA%B5%B0%EB%A1%A0" title="군론">군론</a>과 <a href="/wiki/%EB%A6%AC_%EB%8C%80%EC%88%98" title="리 대수">리 대수</a>, <a href="/wiki/%EC%88%98%EB%A6%AC%EB%AC%BC%EB%A6%AC%ED%95%99" title="수리물리학">수리물리학</a>을 연구하는 영국의 <a href="/wiki/%EC%88%98%ED%95%99%EC%9E%90" title="수학자">수학자</a>이다.</p>

<h2><span class="mw-headline" id=".EC.83.9D.EC.95.A0">생애</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%A6%AC%EC%B2%98%EB%93%9C_%EB%B3%B4%EC%B2%98%EC%A6%88&amp;action=edit&amp;section=1" title="부분 편집: 생애">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p><a href="/wiki/%EB%82%A8%EC%95%84%ED%94%84%EB%A6%AC%EC%B9%B4_%EA%B3%B5%ED%99%94%EA%B5%AD" title="남아프리카 공화국">남아프리카 공화국</a>의 <a href="/wiki/%EC%BC%80%EC%9D%B4%ED%94%84%ED%83%80%EC%9A%B4" title="케이프타운">케이프타운</a>에서 태어났으며, <a href="/wiki/%EB%B2%84%EB%B0%8D%EC%97%84" title="버밍엄">버밍엄</a>의 킹에드워드 학교(<span style="font-size: smaller;"><a href="/wiki/%EC%98%81%EC%96%B4" title="영어">영어</a>:</span> <span lang="en" xml:lang="en">King Edward’s School</span>)에서 교육 받은 후, <a href="/wiki/%EC%BC%80%EC%9E%84%EB%B8%8C%EB%A6%AC%EC%A7%80_%EB%8C%80%ED%95%99%EA%B5%90" title="케임브리지 대학교">케임브리지 대학교</a>에서 <a href="/wiki/%EC%A1%B4_%ED%98%B8%ED%84%B4_%EC%BD%98%EC%9B%A8%EC%9D%B4" title="존 호턴 콘웨이">존 호턴 콘웨이</a>의 지도 아래에 박사학위를 받았다. 박사학위 후 케임브리지 대학을 비롯한 여러 곳에서 수학을 가르쳤고, 현재는 <a class="mw-redirect" href="/wiki/%EC%BA%98%EB%A6%AC%ED%8F%AC%EB%8B%88%EC%95%84_%EB%8C%80%ED%95%99%EA%B5%90_%EB%B2%84%ED%81%B4%EB%A6%AC_%EB%B6%84%EA%B5%90" title="캘리포니아 대학교 버클리 분교">캘리포니아 대학교 버클리 분교</a>에서 수학과 교수로 재직 중이다. <a href="/wiki/1998%EB%85%84" title="1998년">1998년</a> <a href="/wiki/%EB%8F%85%EC%9D%BC" title="독일">독일</a>의 <a href="/wiki/%EB%B2%A0%EB%A5%BC%EB%A6%B0" title="베를린">베를린</a>에서 열린 제23차 <a href="/wiki/%EC%84%B8%EA%B3%84_%EC%88%98%ED%95%99%EC%9E%90_%EB%8C%80%ED%9A%8C" title="세계 수학자 대회">세계 수학자 대회</a>에서, <a href="/wiki/%EB%A7%89%EC%8B%AC_%EC%BD%98%EC%B2%B4%EB%B9%84%EC%B9%98" title="막심 콘체비치">막심 콘체비치</a>, <a href="/wiki/%EC%9C%8C%EB%A6%AC%EC%97%84_%ED%8B%B0%EB%A8%B8%EC%8B%9C_%EA%B0%80%EC%9B%8C%EC%8A%A4" title="윌리엄 티머시 가워스">윌리엄 티머시 가워스</a>, <a href="/wiki/%EC%BB%A4%ED%8B%B0%EC%8A%A4_%EB%A7%A5%EB%A9%80%EB%9F%B0" title="커티스 맥멀런">커티스 맥멀런</a>과 함께 <a href="/wiki/%ED%95%84%EC%A6%88%EC%83%81" title="필즈상">필즈상</a>을 수상하였다.</p>
<h2><span class="mw-headline" id=".EC.97.85.EC.A0.81">업적</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%A6%AC%EC%B2%98%EB%93%9C_%EB%B3%B4%EC%B2%98%EC%A6%88&amp;action=edit&amp;section=2" title="부분 편집: 업적">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>보처즈의 많은 업적들 중 가장 유명한 것은 <a href="/wiki/%EC%9C%A0%ED%95%9C%EA%B5%B0" title="유한군">유한군</a>에 대한 업적이다. 보처즈는 <a href="/wiki/%EB%93%B1%EA%B0%81_%EC%9E%A5%EB%A1%A0" title="등각 장론">등각 장론</a>의 공리화인 <a href="/wiki/%EA%BC%AD%EC%A7%93%EC%A0%90_%EC%97%B0%EC%82%B0%EC%9E%90_%EB%8C%80%EC%88%98" title="꼭짓점 연산자 대수">꼭짓점 연산자 대수</a>의 개념을 정의하였고, 이것을 이용하여 <a href="/wiki/%EC%A1%B4_%ED%98%B8%ED%84%B4_%EC%BD%98%EC%9B%A8%EC%9D%B4" title="존 호턴 콘웨이">존 호턴 콘웨이</a>와 사이먼 노턴(Simon Norton)이 제안한 가공할 헛소리(<span style="font-size: smaller;"><a href="/wiki/%EC%98%81%EC%96%B4" title="영어">영어</a>:</span> <span lang="en" xml:lang="en">monstrous moonshine</span>) 추측을 증명하였다. 헛소리는 <a href="/wiki/%EA%B4%B4%EB%AC%BC%EA%B5%B0_(%EC%88%98%ED%95%99)" title="괴물군 (수학)">괴물군</a>의 <a class="mw-redirect" href="/wiki/%EA%B8%B0%EC%95%BD_%ED%91%9C%ED%98%84" title="기약 표현">기약 표현</a>과 <a href="/wiki/J-%EB%B6%88%EB%B3%80%EB%9F%89" title="J-불변량">j-불변량</a>의 <a href="/wiki/%ED%91%B8%EB%A6%AC%EC%97%90_%EA%B8%89%EC%88%98" title="푸리에 급수">푸리에 급수</a>의 계수 사이의 관계이며, 보처즈는 이를 <a href="/wiki/%EB%B3%B4%EC%86%90_%EB%81%88_%EC%9D%B4%EB%A1%A0" title="보손 끈 이론">보손 끈 이론</a>을 <a class="new" href="/w/index.php?title=%EB%A6%AC%EC%B9%98_%EA%B2%A9%EC%9E%90&amp;action=edit&amp;redlink=1" title="리치 격자 (없는 문서)">리치 격자</a>(24차원의 <a href="/wiki/%EC%9C%A0%EB%8B%88%EB%AA%A8%EB%93%88%EB%9F%AC_%EA%B2%A9%EC%9E%90" title="유니모듈러 격자">유니모듈러 격자</a>)이 축소화한 <a href="/wiki/%EA%BC%AD%EC%A7%93%EC%A0%90_%EC%97%B0%EC%82%B0%EC%9E%90_%EB%8C%80%EC%88%98" title="꼭짓점 연산자 대수">꼭짓점 연산자 대수</a>를 사용하여 증명하였다. 최근에는, 보처즈는 <a href="/wiki/%EC%96%91%EC%9E%90%EC%9E%A5%EB%A1%A0" title="양자장론">양자장론</a>을 수학적으로 엄밀하게 정의하는 것을 연구하고 있다.</p>
