<p><b>리처드 웨슬리 해밍</b>(Richard Wesley Hamming, <a href="/wiki/1915%EB%85%84" title="1915년">1915년</a> <a href="/wiki/2%EC%9B%94_11%EC%9D%BC" title="2월 11일">2월 11일</a> ~ <a href="/wiki/1998%EB%85%84" title="1998년">1998년</a> <a href="/wiki/1%EC%9B%94_7%EC%9D%BC" title="1월 7일">1월 7일</a>)은 미국의 <a href="/wiki/%EC%88%98%ED%95%99%EC%9E%90" title="수학자">수학자</a>이며 <a class="mw-redirect" href="/wiki/%EC%A0%84%EC%82%B0%ED%95%99" title="전산학">전산학</a>과 <a href="/wiki/%ED%86%B5%EC%8B%A0%EC%9D%B4%EB%A1%A0" title="통신이론">통신이론</a>에 지대한 공헌을 했다. <a href="/wiki/%ED%95%B4%EB%B0%8D_%EB%B6%80%ED%98%B8" title="해밍 부호">해밍 부호</a>, <a class="new" href="/w/index.php?title=%EC%B0%BD%EB%AC%B8_%ED%95%A8%EC%88%98&amp;action=edit&amp;redlink=1" title="창문 함수 (없는 문서)">창문 함수</a>(디지털 필터에 사용함), <a class="new" href="/w/index.php?title=%ED%95%B4%EB%B0%8D_%EC%88%98&amp;action=edit&amp;redlink=1" title="해밍 수 (없는 문서)">해밍 수</a>, <a href="/wiki/%ED%95%B4%EB%B0%8D_%EA%B1%B0%EB%A6%AC" title="해밍 거리">해밍 거리</a> 등을 만들었다.</p>
<p>1915년 <a class="mw-redirect" href="/wiki/%EC%9D%BC%EB%A6%AC%EB%85%B8%EC%9D%B4%EC%A3%BC" title="일리노이주">일리노이주</a> <a href="/wiki/%EC%8B%9C%EC%B9%B4%EA%B3%A0" title="시카고">시카고</a>에서 태어났다. <a href="/wiki/1937%EB%85%84" title="1937년">1937년</a>에 <a class="mw-redirect" href="/wiki/%EC%8B%9C%EC%B9%B4%EA%B3%A0_%EB%8C%80%ED%95%99" title="시카고 대학">시카고 대학</a>에서 학사 학위를 받았고, 1939년에 네브라스카 대학에서 석사 학위를 받았다. <a href="/wiki/1942%EB%85%84" title="1942년">1942년</a>에는 <a class="mw-redirect" href="/wiki/%EC%9D%BC%EB%A6%AC%EB%85%B8%EC%9D%B4_%EC%A3%BC%EB%A6%BD_%EB%8C%80%ED%95%99%EA%B5%90_%EC%96%B4%EB%B0%94%EB%82%98-%EC%83%B4%ED%8E%98%EC%9D%B8" title="일리노이 주립 대학교 어바나-샴페인">일리노이 대학교</a>(University of Illinois at Urbana-Champaign, UIUC)에서 박사 학위를 받았다. 2차 대전 당시 <a href="/wiki/%EB%A7%A8%ED%95%B4%ED%8A%BC_%EA%B3%84%ED%9A%8D" title="맨해튼 계획">맨해튼 계획</a>에 참가하여, 컴퓨터 프로그램을 이용하여 <a href="/wiki/%EB%AC%BC%EB%A6%AC%ED%95%99%EC%9E%90" title="물리학자">물리학자</a>들이 만든 <a href="/wiki/%EB%B0%A9%EC%A0%95%EC%8B%9D" title="방정식">방정식</a>의 해를 구하는 일을 맡았다. 이 방정식은 <a class="mw-redirect" href="/wiki/%EC%9B%90%EC%9E%90%ED%8F%AD%ED%83%84" title="원자폭탄">원자폭탄</a>을 터뜨릴 경우, 지구의 대기권에 어떤 영향을 미치는지 알아내는 것이었다. 계산 결과 원자폭탄이 대기권까지 태워버리지 않는다는 것을 확인하였으며, 미국 정부는 <a class="mw-redirect" href="/wiki/%EB%89%B4%EB%A9%95%EC%8B%9C%EC%BD%94%EC%A3%BC" title="뉴멕시코주">뉴멕시코주</a>에서 원자폭탄을 시험했다. 그리고 <a href="/wiki/%EC%9D%BC%EB%B3%B8" title="일본">일본</a>에 두번 투하했다.</p>
