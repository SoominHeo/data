<h3><span class="mw-headline" id=".ED.81.B4.EB.9F.BD_.EA.B2.BD.EB.A0.A5">클럽 경력</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%A6%AC%EC%B9%B4%EB%A5%B4%EB%8F%84_%EB%AA%AC%ED%86%A8%EB%A6%AC%EB%B3%B4&amp;action=edit&amp;section=2" title="부분 편집: 클럽 경력">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>몬톨리보는 <a href="/wiki/%EC%84%B8%EB%A6%AC%EC%97%90_B" title="세리에 B">세리에 B</a>에서 2003-2004 시즌에 데뷔했다. <a href="/wiki/%EC%95%84%ED%83%88%EB%9E%80%ED%83%80_BC" title="아탈란타 BC">아탈란타</a>에서 데뷔시즌에 40경기 4골을 뽑아내는 환상적인 모습을 선보였고 그의 팀인 <a href="/wiki/%EC%95%84%ED%83%88%EB%9E%80%ED%83%80_BC" title="아탈란타 BC">아탈란타</a>는 <a href="/wiki/%EC%84%B8%EB%A6%AC%EC%97%90_A" title="세리에 A">세리에 A</a>로 승격을 확정지었다. <a href="/wiki/%EC%84%B8%EB%A6%AC%EC%97%90_A" title="세리에 A">세리에 A</a>에서 데뷔한 몬톨리보는 <a href="/wiki/%EC%95%84%ED%83%88%EB%9E%80%ED%83%80_BC" title="아탈란타 BC">아탈란타</a>의 신성답게 32경기 3골을 넣는 활약을 보여주었다. 하지만 <a href="/wiki/%EC%95%84%ED%83%88%EB%9E%80%ED%83%80_BC" title="아탈란타 BC">아탈란타</a>는 좋지 않은 성적을 거두면서 강등되었고 그는 2005년 여름 <a href="/wiki/ACF_%ED%94%BC%EC%98%A4%EB%A0%8C%ED%8B%B0%EB%82%98" title="ACF 피오렌티나">피오렌티나</a>로 이적하게 되었다.</p>
<p><a href="/wiki/ACF_%ED%94%BC%EC%98%A4%EB%A0%8C%ED%8B%B0%EB%82%98" title="ACF 피오렌티나">피오렌티나</a>로 이적한 몬톨리보는 첫시즌 17경기를 뛰면서 좋은 활약을 펼쳤다. 그리고 다음 시즌인 2006-2007시즌 이후로 그는 <a href="/wiki/ACF_%ED%94%BC%EC%98%A4%EB%A0%8C%ED%8B%B0%EB%82%98" title="ACF 피오렌티나">피오렌티나</a>의 핵심 선수로 자리잡았다. 이전 시즌인 2005-2006시즌 뛰었던 17경기와 넣었던 1골을 포함, 그는 6년동안 <a href="/wiki/ACF_%ED%94%BC%EC%98%A4%EB%A0%8C%ED%8B%B0%EB%82%98" title="ACF 피오렌티나">ACF 피오렌티나</a>에서 189경기를 뛰고 13골을 넣는 활약을 펼쳤다. 하지만 그는 2010-2011시즌 이후에 재계약을 하지 않았고, 2012년 계약이 만료되기 때문에, 피오렌티나는 이적료를 받기 위해서는 올 시즌 그를 팔아야만하는 좋지 않은 상황에 놓이게 되었다.</p>
<h3><span class="mw-headline" id=".EA.B5.AD.EA.B0.80.EB.8C.80.ED.91.9C.ED.8C.80_.EA.B2.BD.EB.A0.A5">국가대표팀 경력</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%A6%AC%EC%B9%B4%EB%A5%B4%EB%8F%84_%EB%AA%AC%ED%86%A8%EB%A6%AC%EB%B3%B4&amp;action=edit&amp;section=3" title="부분 편집: 국가대표팀 경력">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>2007년 10월 17일 그는 <a href="/wiki/%EB%82%A8%EC%95%84%ED%94%84%EB%A6%AC%EC%B9%B4_%EA%B3%B5%ED%99%94%EA%B5%AD_%EC%B6%95%EA%B5%AC_%EA%B5%AD%EA%B0%80%EB%8C%80%ED%91%9C%ED%8C%80" title="남아프리카 공화국 축구 국가대표팀">남아프리카 공화국</a>과의 경기에서 국가대표팀에 데뷔했다. 그는 또한 <a class="mw-redirect" href="/wiki/%EC%9C%A0%EB%A1%9C_2008" title="유로 2008">유로 2008</a>에서 처음 명단을 발표했을 때는 들어갔으나, 최종 명단에는 들지 못했다. <a class="mw-redirect" href="/wiki/%EC%9C%A0%EB%A1%9C_2008" title="유로 2008">유로 2008</a>에서 제외된 몬톨리보는 같은 해 열린 <a href="/wiki/2008%EB%85%84_%ED%95%98%EA%B3%84_%EC%98%AC%EB%A6%BC%ED%94%BD" title="2008년 하계 올림픽">2008년 하계 올림픽</a>에 올림픽 축구 대표로 출전하게 됐고, <a href="/wiki/%EB%8C%80%ED%95%9C%EB%AF%BC%EA%B5%AD_%EC%B6%95%EA%B5%AC_%EA%B5%AD%EA%B0%80%EB%8C%80%ED%91%9C%ED%8C%80" title="대한민국 축구 국가대표팀">대한민국</a>과의 2차전에서 득점을 기록했다. 이듬해 <a href="/wiki/2009%EB%85%84_FIFA_%EC%BB%A8%ED%8E%98%EB%8D%94%EB%A0%88%EC%9D%B4%EC%85%98%EC%8A%A4%EC%BB%B5" title="2009년 FIFA 컨페더레이션스컵">2009년 FIFA 컨페더레이션스컵</a>에서 엔트리에 발탁되면서 로테이션 선수로 자리잡았고 <a class="mw-redirect" href="/wiki/2010_FIFA_%EC%9B%94%EB%93%9C%EC%BB%B5" title="2010 FIFA 월드컵">2010 FIFA 월드컵</a>에서는 대회 직전 부상당한 <a href="/wiki/%EC%95%88%EB%93%9C%EB%A0%88%EC%95%84_%ED%94%BC%EB%A5%BC%EB%A1%9C" title="안드레아 피를로">안드레아 피를로</a>의 대체자로 3경기를 모두 뛰면서 이탈리아 대표팀의 주전 미드필더가 됐다.</p>
