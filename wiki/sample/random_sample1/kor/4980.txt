<p><b>리커창</b>(<small><a href="/wiki/%EC%A4%91%EA%B5%AD%EC%96%B4" title="중국어">중국어</a></small><small>:</small> <span lang="zh" xml:lang="zh">李克强</span>, <small><a href="/wiki/%ED%95%9C%EC%96%B4_%EB%B3%91%EC%9D%8C" title="한어 병음">병음</a>:</small> <font face="Lucida Sans Unicode">Lǐ Kèqíang</font>, <a href="/wiki/1955%EB%85%84" title="1955년">1955년</a> <a href="/wiki/7%EC%9B%94_1%EC%9D%BC" title="7월 1일">7월 1일</a> ~ )은 <a href="/wiki/%EC%A4%91%ED%99%94%EC%9D%B8%EB%AF%BC%EA%B3%B5%ED%99%94%EA%B5%AD" title="중화인민공화국">중화인민공화국</a>의 정치인이다. <a href="/wiki/2007%EB%85%84" title="2007년">2007년</a> 10월 <a href="/wiki/%EC%A4%91%EA%B5%AD%EA%B3%B5%EC%82%B0%EB%8B%B9_%EC%A4%91%EC%95%99%EC%A0%95%EC%B9%98%EA%B5%AD_%EC%83%81%EB%AC%B4%EC%9C%84%EC%9B%90%ED%9A%8C" title="중국공산당 중앙정치국 상무위원회">중국공산당 중앙정치국 상무위원회</a>의 서열 7위의 상무위원으로 선출되었으며 <a href="/wiki/2013%EB%85%84" title="2013년">2013년</a> 3월 중국의 <a href="/wiki/%EC%A4%91%ED%99%94%EC%9D%B8%EB%AF%BC%EA%B3%B5%ED%99%94%EA%B5%AD%EC%9D%98_%EA%B5%AD%EB%AC%B4%EC%9B%90%EC%B4%9D%EB%A6%AC" title="중화인민공화국의 국무원총리">국무원총리</a>에 올랐다.</p>

<h2><span class="mw-headline" id=".EA.B2.BD.EB.A0.A5">경력</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%A6%AC%EC%BB%A4%EC%B0%BD&amp;action=edit&amp;section=1" title="부분 편집: 경력">원본 편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p><a href="/wiki/1955%EB%85%84" title="1955년">1955년</a> <a href="/wiki/%EC%95%88%ED%9B%84%EC%9D%B4_%EC%84%B1" title="안후이 성">안후이 성</a>에서 태어나 베이징 정계에 대학 진학과 함께 진출. <a href="/wiki/1976%EB%85%84" title="1976년">1976년</a> 10월 <a class="mw-redirect" href="/wiki/%EC%A4%91%EA%B5%AD_%EA%B3%B5%EC%82%B0%EB%8B%B9" title="중국 공산당">중국 공산당</a>에 입당했으며 <a href="/wiki/%EC%95%88%ED%9B%84%EC%9D%B4_%EC%84%B1" title="안후이 성">안후이 성</a>의 봉양 현 공사 당지부 서기를 출발로 정치에 입문하였다. <a href="/wiki/1977%EB%85%84" title="1977년">1977년</a> 학번으로 <a href="/wiki/%ED%99%94%EA%B6%88%ED%8E%91" title="화궈펑">화궈펑</a> 시대에 <a class="mw-redirect" href="/wiki/%EB%B6%81%EA%B2%BD_%EB%8C%80%ED%95%99" title="북경 대학">북경 대학</a>에 법률학과에 입학하였으며, 재학 기간 북경 대학 공산주의청년단 서기를 맡았다. 뒤에 1994년 <a class="mw-redirect" href="/wiki/%EB%B6%81%EA%B2%BD_%EB%8C%80%ED%95%99" title="북경 대학">북경 대학</a>에서 <a href="/wiki/%EA%B2%BD%EC%A0%9C%ED%95%99" title="경제학">경제학</a> 박사 학위를 받았다. 1985년-1993년 공청단 중앙서기처 서기 겸 전국청년연합 부주석을 지냈고, <a href="/wiki/1993%EB%85%84" title="1993년">1993년</a>에 <a class="mw-redirect" href="/wiki/%EC%A4%91%EA%B5%AD%EA%B3%B5%EC%82%B0%EC%A3%BC%EC%9D%98%EC%B2%AD%EB%85%84%EB%8B%A8" title="중국공산주의청년단">중국공산주의청년단</a> 제1서기가 되었으며, <a href="/wiki/%EB%A7%81%EC%A7%80%ED%99%94" title="링지화">링지화</a>와 <a href="/wiki/%EB%A6%AC%EC%9C%84%EC%95%88%EC%B0%A8%EC%98%A4" title="리위안차오">리위안차오</a> 등 <a class="mw-redirect" href="/wiki/%EA%B3%B5%EC%B2%AD%EB%8B%A8" title="공청단">공청단</a> 지도부의 후원을 받아 빠른 속도로 성장하는 유망 정치인이 되었다. 리커창은 <a href="/wiki/1999%EB%85%84" title="1999년">1999년</a>에 43세의 나이로 <a href="/wiki/%ED%97%88%EB%82%9C_%EC%84%B1" title="허난 성">허난 성</a> 성장이 되었는데, 당시 <a href="/wiki/%EC%A4%91%ED%99%94%EC%9D%B8%EB%AF%BC%EA%B3%B5%ED%99%94%EA%B5%AD" title="중화인민공화국">중화인민공화국</a> 최연소 성장이었다. 허난 성에서 뒤떨어진 내륙 지역을 투자 지역으로 변화시키려고 노력하였다. <a href="/wiki/2004%EB%85%84" title="2004년">2004년</a> 말, <a href="/wiki/%EB%9E%B4%EC%98%A4%EB%8B%9D_%EC%84%B1" title="랴오닝 성">랴오닝 성</a>의 당위원회 서기로 옮긴 뒤, <a href="/wiki/2007%EB%85%84" title="2007년">2007년</a> 중반까지 <a href="/wiki/%EB%9E%B4%EC%98%A4%EB%8B%9D_%EC%84%B1" title="랴오닝 성">랴오닝 성</a>의 당위원회 서기로 재임하면서 동북 진흥 공정을 주도하였다. 2007년 10월 중국공산당 중앙정치국 상무위원. 2008년부터 국무원 부총리를 지냈다. 2012년 11월 15일 중국공산당 제18기 1중 전회에서 상무위원에 연임되어, 총리가 되었다.</p>
<h2><span class="mw-headline" id=".EA.B0.80.EC.A1.B1_.EA.B4.80.EA.B3.84">가족 관계</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%A6%AC%EC%BB%A4%EC%B0%BD&amp;action=edit&amp;section=2" title="부분 편집: 가족 관계">원본 편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>리커창의 부인인 청훙(程虹)은 베이징 수도 경제 무역대 외국어학 교수를 역임했다.</p>
<h2><span class="mw-headline" id=".EA.B8.B0.ED.83.80">기타</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%A6%AC%EC%BB%A4%EC%B0%BD&amp;action=edit&amp;section=3" title="부분 편집: 기타">원본 편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<ul>
<li>2010년 10월 언론 보도에 따르면 <a href="/wiki/%EC%9A%B0%EB%B0%A9%EA%B6%88" title="우방궈">우방궈</a>의 후임으로 2013년에 전국인민대표대회 상무위원장이 될 가능성이 있었고<sup class="reference" id="cite_ref-1"><a href="#cite_note-1">[1]</a></sup>, <a href="/wiki/%EC%9B%90%EC%9E%90%EB%B0%94%EC%98%A4" title="원자바오">원자바오</a> 현 국무원 총리의 후임으로서 <a href="/wiki/%EC%8B%9C%EC%A7%84%ED%95%91" title="시진핑">시진핑</a> 차기 주석과 함께 중국 정부를 이끌 가능성도 컸으며<sup class="reference" id="cite_ref-2"><a href="#cite_note-2">[2]</a></sup> 이때 총리 후보로 거론돼온 <a href="/wiki/%EC%99%95%EC%B9%98%EC%82%B0" title="왕치산">왕치산</a>은 경제·금융을 총괄할 가능성과, 전인대 상무위원장 직을 맡을 가능성 등이 제기됐었다.</li>
</ul>
