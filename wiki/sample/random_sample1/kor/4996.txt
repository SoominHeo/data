<p><b>리하르트 빌슈테터</b>(Richard Willstätter, <a href="/wiki/1872%EB%85%84" title="1872년">1872년</a> <a href="/wiki/8%EC%9B%94_13%EC%9D%BC" title="8월 13일">8월 13일</a> ~ <a href="/wiki/1942%EB%85%84" title="1942년">1942년</a> <a href="/wiki/8%EC%9B%94_3%EC%9D%BC" title="8월 3일">8월 3일</a>)는 <a href="/wiki/%EB%8F%85%EC%9D%BC" title="독일">독일</a>의 화학자이다. <a href="/wiki/1902%EB%85%84" title="1902년">1902년</a> <a href="/wiki/%EB%AE%8C%ED%97%A8_%EB%8C%80%ED%95%99%EA%B5%90" title="뮌헨 대학교">뮌헨 대학교</a> 교수, <a href="/wiki/1905%EB%85%84" title="1905년">1905년</a> <a class="mw-redirect" href="/wiki/ETH_%EC%B7%A8%EB%A6%AC%ED%9E%88" title="ETH 취리히">ETH 취리히</a> 교수를 거쳐, <a href="/wiki/1911%EB%85%84" title="1911년">1911년</a> <a class="new" href="/w/index.php?title=%EC%B9%B4%EC%9D%B4%EC%A0%80_%EB%B9%8C%ED%97%AC%EB%A6%84_%EC%9C%A0%EA%B8%B0_%ED%99%94%ED%95%99_%EC%97%B0%EA%B5%AC%EC%86%8C&amp;action=edit&amp;redlink=1" title="카이저 빌헬름 유기 화학 연구소 (없는 문서)">카이저 빌헬름 유기 화학 연구소</a> 소장이 되었다. <a href="/wiki/%EC%95%8C%EC%B9%BC%EB%A1%9C%EC%9D%B4%EB%93%9C" title="알칼로이드">알칼로이드</a>·<a href="/wiki/%EC%97%BD%EB%A1%9D%EC%86%8C" title="엽록소">엽록소</a>의 구조·<a href="/wiki/%ED%9A%A8%EC%86%8C" title="효소">효소</a>의 정체·혈액 색소의 연구 등 화학계에 큰 공헌을 하였으므로, <a href="/wiki/1915%EB%85%84" title="1915년">1915년</a> <a href="/wiki/%EB%85%B8%EB%B2%A8_%ED%99%94%ED%95%99%EC%83%81" title="노벨 화학상">노벨 화학상</a>을 받았다. 저서로 《엽록소 연구》, 《탄소 동화 작용의 연구》, 《효소 연구》 등이 있다.</p>
