<p><b>리하르트 조르게</b>(<span style="font-size: smaller;"><a href="/wiki/%EB%8F%85%EC%9D%BC%EC%96%B4" title="독일어">독일어</a>:</span> <span lang="de" xml:lang="de">Richard Sorge</span>, <a href="/wiki/1895%EB%85%84" title="1895년">1895년</a> <a href="/wiki/10%EC%9B%94_4%EC%9D%BC" title="10월 4일">10월 4일</a> ~ <a href="/wiki/1944%EB%85%84" title="1944년">1944년</a> <a href="/wiki/11%EC%9B%94_7%EC%9D%BC" title="11월 7일">11월 7일</a>) 혹은 <b>리하르트 빌겔모비치 조르게</b>(<span style="font-size: smaller;"><a href="/wiki/%EB%9F%AC%EC%8B%9C%EC%95%84%EC%96%B4" title="러시아어">러시아어</a>:</span> <span lang="ru" xml:lang="ru">Ри́хард Вильге́льмович Зо́рге</span><sup class="reference" id="cite_ref-1"><a href="#cite_note-1">[1]</a></sup>)는 <a href="/wiki/%EC%9D%BC%EB%B3%B8" title="일본">일본</a>을 주무대로 활약한 <a href="/wiki/%EC%86%8C%EB%A0%A8" title="소련">소련</a>의 간첩이다. 그는 신문 기자로 가장하여 독일 신문의 일본 특파원으로 일하면서 <a class="mw-redirect" href="/wiki/%EC%A0%9C2%EC%B0%A8_%EC%84%B8%EA%B3%84%EB%8C%80%EC%A0%84" title="제2차 세계대전">제2차 세계대전</a> 중 <a href="/wiki/%EC%B6%94%EC%B6%95%EA%B5%AD" title="추축국">추축국</a>의 주요 정보를 빼내 소련에 넘겨 연합국의 승리에 큰 기여를 하였다. 1944년 <a href="/wiki/%EC%9D%BC%EB%B3%B8" title="일본">일본</a>에 체포되어 처형되었다.</p>

<h2><span class="mw-headline" id=".EC.B4.88.EA.B8.B0_.EC.83.9D.EC.95.A0">초기 생애</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%A6%AC%ED%95%98%EB%A5%B4%ED%8A%B8_%EC%A1%B0%EB%A5%B4%EA%B2%8C&amp;action=edit&amp;section=1" title="부분 편집: 초기 생애">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>리하르트 조르게는 당시 <a href="/wiki/%EB%9F%AC%EC%8B%9C%EC%95%84_%EC%A0%9C%EA%B5%AD" title="러시아 제국">러시아 제국</a>의 영토였고, 현재 <a href="/wiki/%EC%95%84%EC%A0%9C%EB%A5%B4%EB%B0%94%EC%9D%B4%EC%9E%94" title="아제르바이잔">아제르바이잔</a>의 <a href="/wiki/%EB%B0%94%EC%BF%A0" title="바쿠">바쿠</a> 근처인 사분치에서 태어났다. 아버지는 독일인 유전 기술자였고, 어머니는 러시아인이었다. 아버지는 코카서스 석유회사에 고용된 기술자였으나, 계약이 해지되어 독일의 <a href="/wiki/%EB%B2%A0%EB%A5%BC%EB%A6%B0" title="베를린">베를린</a>으로 돌아왔다. 리하르트의 친조부의 형제였던 프리드리히 조르게는 <a href="/wiki/%EA%B3%B5%EC%82%B0%EC%A3%BC%EC%9D%98" title="공산주의">공산주의</a> 이론가인 <a class="mw-redirect" href="/wiki/%EB%A7%88%EB%A5%B4%ED%81%AC%EC%8A%A4" title="마르크스">마르크스</a>의 동지였다.</p>
<p>1914년 10월 조르게는 <a class="mw-redirect" href="/wiki/%EC%A0%9C1%EC%B0%A8_%EC%84%B8%EA%B3%84%EB%8C%80%EC%A0%84" title="제1차 세계대전">제1차 세계대전</a>에 학도자원병으로 참전하였다. 그는 서부전선에 파견되어 크게 부상당했는데, 손가락 세 개를 잃고 평생 다리를 절게 되었다. 그는 상병으로 승진해 철십자 훈장을 받았지만, 이러한 부상 때문에 제대를 했다. 그는 부상회복 기간 동안 마르크스의 서적을 읽고 공산주의자가 되었다. 그는 나머지 전쟁기간중에 베를린, <a class="mw-redirect" href="/wiki/%ED%82%AC" title="킬">킬</a>, <a href="/wiki/%ED%95%A8%EB%B6%80%EB%A5%B4%ED%81%AC" title="함부르크">함부르크</a>에 있는 대학교에서 경제학을 공부하였다. 뿐만 아니라 그는 <a href="/wiki/%EB%86%8D%EC%97%85" title="농업">농업</a>과 <a class="new" href="/w/index.php?title=%ED%8F%AC%EB%B3%91%EC%88%A0&amp;action=edit&amp;redlink=1" title="포병술 (없는 문서)">포병술</a>에 해박한 지식을 가지고 있었다. <a href="/wiki/1919%EB%85%84" title="1919년">1919년</a> 조르게는 정치학 박사학위를 받았고, <a href="/wiki/%EA%B5%90%EC%82%AC" title="교사">교사</a>로 일하면서 <a class="mw-disambig" href="/wiki/%EB%8F%85%EC%9D%BC_%EA%B3%B5%EC%82%B0%EB%8B%B9" title="독일 공산당">독일 공산당</a>에 입당하였다. 그러나 이러한 정치 활동 때문에 직업을 잃고, <a href="/wiki/%EB%AA%A8%EC%8A%A4%ED%81%AC%EB%B0%94" title="모스크바">모스크바</a>로 가서 <a href="/wiki/%EC%BD%94%EB%AF%BC%ED%85%8C%EB%A5%B8" title="코민테른">코민테른</a>의 요원이 되었다.</p>
<h2><span class="mw-headline" id=".EA.B0.84.EC.B2.A9.EC.9D.B4_.EB.90.98.EB.8B.A4">간첩이 되다</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%A6%AC%ED%95%98%EB%A5%B4%ED%8A%B8_%EC%A1%B0%EB%A5%B4%EA%B2%8C&amp;action=edit&amp;section=2" title="부분 편집: 간첩이 되다">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>조르게는 소련에 의해 첩보요원으로 선발되어 신문기자로 신분을 위장하고 유럽의 여러나라에 파견되었다. 그는 <a href="/wiki/1920%EB%85%84" title="1920년">1920년</a>부터 <a href="/wiki/1924%EB%85%84" title="1924년">1924년</a>까지 독일에서 머물렀는데 1921년 그는 옛 스승 알베르트 게어라흐의 아내였던 크리스티네와 결혼을 했다. 이후 여러 공작에 가담하였고, 1924년 <a href="/wiki/%EB%AA%A8%EC%8A%A4%ED%81%AC%EB%B0%94" title="모스크바">모스크바</a>로 돌아와서 소련의 첩보기관인 <a href="/wiki/%EA%B5%AD%EA%B0%80%EC%A0%95%EC%B9%98%EB%B6%80" title="국가정치부">국가정치부</a>에서 정보분석자로 일했다. 그는 일 때문에 가정을 돌보지 않아서 이혼하고 말았다.</p>
<p><a href="/wiki/1929%EB%85%84" title="1929년">1929년</a> 조르게는 <a href="/wiki/%EC%86%8C%EB%A0%A8%EA%B5%B0" title="소련군">소련군</a>의 제4국 (GRU, 군사정보국)으로 자리를 옮겼다. 그는 이후 죽을때까지 이 부서의 요원으로 일했다. 그는 이해 <a href="/wiki/%EC%98%81%EA%B5%AD" title="영국">영국</a>에 파견되어 정보를 수집하였다. 다시 독일로 가서 <a href="/wiki/%EC%A2%8C%EC%9D%B5" title="좌익">좌익</a>활동에 연루되지 말고 <a class="mw-redirect" href="/wiki/%EB%82%98%EC%B9%98%EB%8B%B9" title="나치당">나치당</a>에 가입하라는 지령을 받고 이에 따랐다. 그는 신분을 위장하기 위해 농업신문인 <b>도이체 게트라이드 차이퉁</b>(Deutsche Getreide-Zeitung)에 기자로 취직하였다.</p>
<p><a href="/wiki/1930%EB%85%84" title="1930년">1930년</a> 조르게는 중국의 <a class="mw-redirect" href="/wiki/%EC%83%81%ED%95%98%EC%9D%B4" title="상하이">상하이</a>로 가서 정보수집과 혁명공작을 하였다. 공식적으로 그는 한 독일의 통신사의 편집인과 <b>프랑크푸르터 차이퉁</b>의 특파원의 직함을 가지고 있었다. 여기서 그는 저명한 좌익 저널리스트인 <a class="new" href="/w/index.php?title=%EC%95%84%EA%B7%B8%EB%84%A4%EC%8A%A4_%EC%8A%A4%EB%A9%94%EB%93%A4%EB%A6%AC&amp;action=edit&amp;redlink=1" title="아그네스 스메들리 (없는 문서)">아그네스 스메들리</a>를 만났고, 그녀와 잠시 사귀었으며, 그녀는 조르게에게 일본기자들을 소개하였다. 그는 일본기자 <a class="mw-redirect" href="/wiki/%EC%98%A4%EC%9E%90%ED%82%A4_%ED%98%B8%EC%B8%A0%EB%AF%B8" title="오자키 호츠미">오자키 호츠미</a>를 포섭하여 정보원으로 삼았다. 그는 농업전문가로 행세하여 중국각지를 여행하면서 당시 <a href="/wiki/%EC%9E%A5%EC%A0%9C%EC%8A%A4" title="장제스">장제스</a>의 대대적인 탄압을 피해서 지하로 숨어 있던 <a href="/wiki/%EC%A4%91%EA%B5%AD%EA%B3%B5%EC%82%B0%EB%8B%B9" title="중국공산당">중국공산당</a>의 당원들과 접촉할 수 있었다. 1932년 그는 일본군과 중국군이 싸운 <a href="/wiki/%EC%A0%9C1%EC%B0%A8_%EC%83%81%ED%95%98%EC%9D%B4_%EC%82%AC%EB%B3%80" title="제1차 상하이 사변">제1차 상하이 사변</a>을 취재하였고, 이해 12월 모스크바로 소환되었다.</p>
<h2><span class="mw-headline" id=".EB.8C.80.EC.9D.BC.EB.B3.B8_.EC.B2.A9.EB.B3.B4_.ED.99.9C.EB.8F.99">대일본 첩보 활동</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%A6%AC%ED%95%98%EB%A5%B4%ED%8A%B8_%EC%A1%B0%EB%A5%B4%EA%B2%8C&amp;action=edit&amp;section=3" title="부분 편집: 대일본 첩보 활동">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>소련 정보당국은 조르게에게 일본으로 가서 첩보망을 구성하도록 명령했다. 그리하여 조르게는 일단 독일로 돌아가서 철저한 나치스로 위장하고, 새로운 신문사에 들어갔다. 특히 취중에 신분을 노출시키지 않기 위해서 좋아하던 <a href="/wiki/%EC%88%A0" title="술">술</a>까지 끊었다.</p>
<p><a href="/wiki/1933%EB%85%84" title="1933년">1933년</a> 9월 6일 조르게는 <a class="mw-redirect" href="/wiki/%EC%9A%94%EC%BD%94%ED%95%98%EB%A7%88" title="요코하마">요코하마</a>에 도착했다. 그곳에서 그는 일본인과 외국인, 그리고 사업가와 신문기자등으로 구성된 첩보망을 조직하였다. 그의 요원들은 일본의 저명한 정치가들과 접촉하여 일본의 대외정책에 관한 정보를 수집하였다. 특히 <a class="mw-redirect" href="/wiki/%EC%98%A4%EC%9E%90%ED%82%A4_%ED%98%B8%EC%B8%A0%EB%AF%B8" title="오자키 호츠미">오자키 호츠미</a>는 <a href="/wiki/%EC%B4%9D%EB%A6%AC" title="총리">총리</a>였던 <a href="/wiki/%EA%B3%A0%EB%85%B8%EC%97%90_%ED%9B%84%EB%AF%B8%EB%A7%88%EB%A1%9C" title="고노에 후미마로">고노에 후미마로</a>의 정책보좌역이 되어 일본정부의 기밀문서를 빼내 조르게에게 넘겼다. 그는 공식적으로 나치의 열렬한 지지자로 위장했기 때문에 주일 독일대사관에 쉽게 접근할 수 있었고, 그곳에서도 고급 정보를 수집할 수 있었다. 그는 독일 대사였던 <a class="new" href="/w/index.php?title=%EC%98%A4%EC%9D%B4%EA%B2%90_%EC%98%A4%ED%86%A0&amp;action=edit&amp;redlink=1" title="오이겐 오토 (없는 문서)">오이겐 오토</a>(Eugen Otto)와 매우 친밀했고, 오토의 아내와도 내연의 관계를 맺고 있었다. 그러나 첩보활동에 따른 스트레스 때문에 그는 폭음하기 시작했다.</p>
<p>조르게는 이러한 첩보망을 통해 독일-이탈리아-일본의 <a href="/wiki/%EB%B0%A9%EA%B3%B5_%ED%98%91%EC%A0%95" title="방공 협정">방공 협정</a>, 독일-일본 협약, <a href="/wiki/%EC%A7%84%EC%A3%BC%EB%A7%8C_%EA%B3%B5%EA%B2%A9" title="진주만 공격">진주만 공격</a>의 정보를 빼내 소련에 전달했다. 특히 <a href="/wiki/1941%EB%85%84" title="1941년">1941년</a> 일본주재 독일 무관에게서 정보를 빼내, 독일의 소련 침공계획인 <a href="/wiki/%EB%B0%94%EB%A5%B4%EB%B0%94%EB%A1%9C%EC%82%AC_%EC%9E%91%EC%A0%84" title="바르바로사 작전">바르바로사 작전</a>의 정확한 개시 일자 (6월 22일, 혹은 그보다 앞선 6월 20일로 전달했다는 설도 있다) 를 전달했으나, <a href="/wiki/%EC%9D%B4%EC%98%A4%EC%8B%9C%ED%94%84_%EC%8A%A4%ED%83%88%EB%A6%B0" title="이오시프 스탈린">스탈린</a>은 이를 무시하여 소련군의 대패를 초래하였다.</p>
<p>소련은 서쪽을 침공한 독일에 발맞추어 동맹국 일본도 아시아쪽에서 협공할 것을 우려하였는데, 1941년 9월 14일 조르게는 이에 관련한 일본군의 동향을 전달하였다. 그것은 독일군이 모스크바를 함락시키기 전까지는 일본군은 소련을 공격하지 않는다는 것이었다. 대신에 일본군은 자원을 확보하기 위해 남방으로 향한다는 정보를 전달하였다. 이 첩보에 근거하여 소련군은 일본 <a href="/wiki/%EA%B4%80%EB%8F%99%EA%B5%B0" title="관동군">관동군</a>에 맞서 극동에 배치된 정예 병력을 빼내 서부로 돌렸고, 이 병력은 <a href="/wiki/%EB%AA%A8%EC%8A%A4%ED%81%AC%EB%B0%94_%EA%B3%B5%EB%B0%A9%EC%A0%84" title="모스크바 공방전">모스크바 공방전</a>에 투입되어 동부전선에서 소련군이 독일군을 처음으로 패퇴시키는데 공헌하였다.</p>
<h2><span class="mw-headline" id=".EC.B2.B4.ED.8F.AC.EC.99.80_.EC.B2.98.ED.98.95">체포와 처형</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%A6%AC%ED%95%98%EB%A5%B4%ED%8A%B8_%EC%A1%B0%EB%A5%B4%EA%B2%8C&amp;action=edit&amp;section=4" title="부분 편집: 체포와 처형">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>일본이 전시국면으로 치달을수록 조르게가 첩보활동을 벌이는 것은 매우 위험해졌다. 그러나 전쟁은 중대국면으로 치닫고 있었기 때문에 조르게는 첩보활동을 계속했다. 소련에서 쓰이던 1회용 암호표에 의한 무선량이 대폭 늘어났기 때문에 일본의 방첩기관(<a href="/wiki/%ED%8A%B9%EB%B3%84%EA%B3%A0%EB%93%B1%EA%B2%BD%EC%B0%B0" title="특별고등경찰">특별고등경찰</a>)은 이를 인지했고 엄중한 감시를 하고 있었다. 조르게의 요원이었던 오자키는 <a href="/wiki/1941%EB%85%84" title="1941년">1941년</a> 10월 14일 먼저 체포되었고 조르게는 10월 18일 <a href="/wiki/%EB%8F%84%EC%BF%84" title="도쿄">도쿄</a>에서 체포되었다.</p>
<p>독일대사인 오이겐 오토는 이를 듣고 놀랐다. 사실 조르게는 소련을 위한 첩보활동을 하면서 일부를 독일대사관 측에도 제공하고 있었기 때문에 오토는 조르게의 체포를 일본의 과민반응 쯤으로 생각하고 있었다. 처음에 일본은 조르게를 <a href="/wiki/%EB%8F%85%EC%9D%BC_%EA%B5%AD%EB%B0%A9%EA%B5%B0" title="독일 국방군">독일 국방군</a>의 첩보기관인 <a class="mw-redirect" href="/wiki/%EC%95%84%ED%94%84%EB%B2%A0%EC%96%B4" title="아프베어">아프베어</a>(Abwehr)의 요원으로 생각하고 있었으나 아프베어는 이를 부인했다. 뿐만 아니라 조르게는 <a href="/wiki/%EA%B3%A0%EB%AC%B8" title="고문">고문</a>에도 불구하고 소련과의 연관을 부정했고, 소련도 조르게를 자국의 요원이라고 인정하지 않았다.</p>
<p>일본은 조르게와 소련에서 붙잡힌 일본간첩과 교환하려고 하였으나, 소련은 조르게의 존재를 인정하지 않아 성사되지 않았다. 조르게는 <a class="mw-redirect" href="/wiki/%EC%8A%A4%EA%B0%80%EB%AA%A8_%ED%98%95%EB%AC%B4%EC%86%8C" title="스가모 형무소">스가모 형무소</a>에 수감되어 있었다. 조르게는 <a href="/wiki/1944%EB%85%84" title="1944년">1944년</a> 11월 7일에 교수형으로 스가모에서 처형되었다. 그의 요원이었던 오자키는 그보다 조금 이른시간에 처형되었다. 조르게는 일단 스가모 형무소내의 묘원에 묻혔다가 도쿄의 <a class="new" href="/w/index.php?title=%ED%83%80%EB%A7%88_%EA%B3%B5%EB%8F%99%EB%AC%98%EC%A7%80&amp;action=edit&amp;redlink=1" title="타마 공동묘지 (없는 문서)">타마 공동묘지</a>로 이장되었다. 그의 일본인 연인이었던 이시이 하나코는 그녀가 살아있던 <a href="/wiki/2000%EB%85%84" title="2000년">2000년</a>까지 계속 조르게의 묘를 찾았다.</p>
<h2><span class="mw-headline" id=".EC.82.AC.ED.9B.84">사후</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%A6%AC%ED%95%98%EB%A5%B4%ED%8A%B8_%EC%A1%B0%EB%A5%B4%EA%B2%8C&amp;action=edit&amp;section=5" title="부분 편집: 사후">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<div class="thumb tright">
<div class="thumbinner" style="width:212px;"><a class="image" href="/wiki/%ED%8C%8C%EC%9D%BC:Dr_Richard_Sorge_spy.jpg"><img alt="" class="thumbimage" data-file-height="512" data-file-width="366" height="294" src="//upload.wikimedia.org/wikipedia/commons/thumb/d/d1/Dr_Richard_Sorge_spy.jpg/210px-Dr_Richard_Sorge_spy.jpg" srcset="//upload.wikimedia.org/wikipedia/commons/thumb/d/d1/Dr_Richard_Sorge_spy.jpg/315px-Dr_Richard_Sorge_spy.jpg 1.5x, //upload.wikimedia.org/wikipedia/commons/d/d1/Dr_Richard_Sorge_spy.jpg 2x" width="210"/></a>
<div class="thumbcaption">
<div class="magnify"><a class="internal" href="/wiki/%ED%8C%8C%EC%9D%BC:Dr_Richard_Sorge_spy.jpg" title="실제 크기로"></a></div>
조르게가 그려진 우표</div>
</div>
</div>
<p><a href="/wiki/1961%EB%85%84" title="1961년">1961년</a> 조르게의 활약상을 각색한 영화 《Qui êtes-vous, Monsieur Sorge?》 (조르게씨, 당신은 누구요?)가 <a href="/wiki/%ED%94%84%EB%9E%91%EC%8A%A4" title="프랑스">프랑스</a>, 서독, 이탈리아, 일본 합작으로 만들어졌다. 이 영화는 소련에서도 개봉되어 인기를 끌었고, 영화를 관람한 소련의 서기장 <a href="/wiki/%EB%8B%88%ED%82%A4%ED%83%80_%ED%9D%90%EB%A3%A8%EC%87%BC%ED%94%84" title="니키타 흐루쇼프">흐루쇼프</a>는 첩보기관이었던 <a href="/wiki/%EA%B5%AD%EA%B0%80%EB%B3%B4%EC%95%88%EC%9C%84%EC%9B%90%ED%9A%8C" title="국가보안위원회">국가보안위원회</a>(KGB)에게 이 영화가 사실에 근거한 것이냐고 물었다. 조르게가 실존 인물임을 확인한 흐루쇼프는 1964년 조르게에게 <a class="mw-redirect" href="/wiki/%EC%86%8C%EB%B9%84%EC%97%90%ED%8A%B8_%EC%97%B0%EB%B0%A9_%EC%98%81%EC%9B%85" title="소비에트 연방 영웅">소비에트 연방 영웅</a> 칭호를 수여했고, 조르게가 공식적으로 자국요원임을 인정하였다. 또한 조르게의 일본인 연인 이시이 하나코에게 <a href="/wiki/%EC%97%B0%EA%B8%88" title="연금">연금</a>을 주었다. 한편 주일본 소련 대사는 부임후 그의 묘소를 찾는 것이 관례였고, 현재는 러시아 대사도 마찬가지로 그의 묘소를 찾는다고 한다.</p>
<p><a href="/wiki/%EB%8F%99%EB%8F%85" title="동독">동독</a>에서도 파시스트 체제에 대항했던 조르게를 높이 평가하여 그의 전기가 발표되고 기념우표를 발행하는 등의 기념사업을 하였다. 뿐만 아니라 <a href="/wiki/%EC%9D%BC%EB%B3%B8" title="일본">일본</a>과 <a href="/wiki/%ED%8F%B4%EB%9E%80%EB%93%9C" title="폴란드">폴란드</a>에서는 그를 다룬 만화가 출판되었다.</p>
<p>2003년 일본에서도 '스파이 조르게'라는 영화가 만들어졌다.</p>
<p>이외에도 여러 편의 책이 그를 주제로 씌여졌다.</p>
