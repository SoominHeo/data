<h2><span class="mw-headline" id=".EB.AF.B8.EA.B5.AD_.ED.94.84.EB.A1.9C_.EC.95.BC.EA.B5.AC_.EC.8B.9C.EC.A0.88">미국 프로 야구 시절</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%A6%AD_%EB%B0%94%EC%9B%8C&amp;action=edit&amp;section=1" title="부분 편집: 미국 프로 야구 시절">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p><a href="/wiki/1997%EB%85%84" title="1997년">1997년</a> 아마추어 신인 드래프트에서 <a href="/wiki/%EB%B3%BC%ED%8B%B0%EB%AA%A8%EC%96%B4_%EC%98%A4%EB%A6%AC%EC%98%AC%EC%8A%A4" title="볼티모어 오리올스">볼티모어 오리올스</a>의 5라운드 지명을 받았다. 볼티모어 오리올스 산하 마이너 리그 팀에서 1997년부터 <a href="/wiki/2001%EB%85%84" title="2001년">2001년</a>까지 주로 선발 투수로 활동하면서 41승 39패, 평균 자책 4.39를 기록하였다. <a href="/wiki/2001%EB%85%84" title="2001년">2001년</a> <a href="/wiki/9%EC%9B%94_2%EC%9D%BC" title="9월 2일">9월 2일</a> <a href="/wiki/%EC%8B%9C%EC%95%A0%ED%8B%80_%EB%A7%A4%EB%A6%AC%EB%84%88%EC%8A%A4" title="시애틀 매리너스">시애틀 매리너스</a>를 상대로 한 경기에서 처음으로 메이저 리그 경기를 가졌는데, 6.1이닝 동안 3피안타 1실점의 호투를 했음에도 팀이 1-0으로 져서 패전투수가 되었다. 2001년부터 2005년까지는 주로 볼티모어의 구원투수로 활약하였다.</p>
<p>2005년 시즌 종료 후 오리올스와 재계약에 실패했고, 11월에 <a href="/wiki/%ED%85%8D%EC%82%AC%EC%8A%A4_%EB%A0%88%EC%9D%B8%EC%A0%80%EC%8A%A4" title="텍사스 레인저스">텍사스 레인저스</a>와 마이너 리그 계약을 체결한다. <a href="/wiki/2006%EB%85%84" title="2006년">2006년</a> <a href="/wiki/4%EC%9B%94_7%EC%9D%BC" title="4월 7일">4월 7일</a> 메이저 리그로 승격되어 시즌 종료 후까지 메이저 리그에 남게 된다. 한번의 선발 등판과 57번의 구원등판에서 3승 1패 2세이브, 평균 자책 3.55를 기록한다. 시즌 마지막 주에는 어깨부상으로 경기에 나오지 못했다. <a href="/wiki/2007%EB%85%84" title="2007년">2007년</a> 1월 텍사스 구단은 연봉조정신청을 피하기 위해 바워와 1년 73만 달러의 조건으로 계약을 맺는다. 하지만 스프링캠프에서의 부진으로 팀은 지명할당으로 웨이버 공시한다. <a href="/wiki/4%EC%9B%94_6%EC%9D%BC" title="4월 6일">4월 6일</a> <a href="/wiki/%ED%95%84%EB%9D%BC%EB%8D%B8%ED%94%BC%EC%95%84_%ED%95%84%EB%A6%AC%EC%8A%A4" title="필라델피아 필리스">필라델피아 필리스</a> 구단이 바워와 마이너 리그 계약을 체결했지만 6월 20일 방출된다. <a href="/wiki/7%EC%9B%94_1%EC%9D%BC" title="7월 1일">7월 1일</a> 다저스 산하 마이너 리그 팀과 다시 계약을 체결한다.</p>
<p>2007년 <a href="/wiki/11%EC%9B%94_30%EC%9D%BC" title="11월 30일">11월 30일</a> 스프링캠프 초청자격으로 <a href="/wiki/%ED%81%B4%EB%A6%AC%EB%B8%94%EB%9E%9C%EB%93%9C_%EC%9D%B8%EB%94%94%EC%96%B8%EC%8A%A4" title="클리블랜드 인디언스">클리블랜드 인디언스</a>와 마이너 리그 계약을 체결하지만, <a href="/wiki/2008%EB%85%84" title="2008년">2008년</a> <a href="/wiki/7%EC%9B%94_4%EC%9D%BC" title="7월 4일">7월 4일</a> 인디언스는 바워를 지명할당처분을 했고, 바워가 마이너 리그 계약을 거부하자 곧바로 방출한다. <a href="/wiki/7%EC%9B%94_11%EC%9D%BC" title="7월 11일">7월 11일</a> <a href="/wiki/%ED%86%A0%EB%A1%A0%ED%86%A0_%EB%B8%94%EB%A3%A8%EC%A0%9C%EC%9D%B4%EC%8A%A4" title="토론토 블루제이스">토론토 블루제이스</a>와 마이너 리그 계약을 맺었고 시즌 종료 후에는 자유 계약 선수가 된다. 2008년 11월 토론토와 재계약을 했지만 이듬해 스프링캠프에서 방출되었다. <a href="/wiki/2009%EB%85%84" title="2009년">2009년</a> 미국 프로 야구 독립리그인 아틀랜틱 리그 요크 레볼루션 팀에서 활동하다가, <a href="/wiki/5%EC%9B%94_13%EC%9D%BC" title="5월 13일">5월 13일</a> 대한민국의 <a href="/wiki/LG_%ED%8A%B8%EC%9C%88%EC%8A%A4" title="LG 트윈스">LG 트윈스</a>와 입단 계약을 체결한다.</p>
<h2><span class="mw-headline" id=".ED.95.9C.EA.B5.AD_.ED.94.84.EB.A1.9C_.EC.95.BC.EA.B5.AC_.EC.8B.9C.EC.A0.88">한국 프로 야구 시절</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%A6%AD_%EB%B0%94%EC%9B%8C&amp;action=edit&amp;section=2" title="부분 편집: 한국 프로 야구 시절">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p><a href="/wiki/2009%EB%85%84" title="2009년">2009년</a> <a href="/wiki/5%EC%9B%94_13%EC%9D%BC" title="5월 13일">5월 13일</a> 기존 용병이었던 <a href="/wiki/%ED%81%AC%EB%A6%AC%EC%8A%A4_%EC%98%A5%EC%8A%A4%ED%94%84%EB%A7%81" title="크리스 옥스프링">크리스 옥스프링</a> 선수가 팔꿈치 부상에 따른 방출로 인해, 대체 외국인 선수로 <a href="/wiki/LG_%ED%8A%B8%EC%9C%88%EC%8A%A4" title="LG 트윈스">LG 트윈스</a>와 입단 계약을 체결하였다.<sup class="reference" id="cite_ref-1"><a href="#cite_note-1">[1]</a></sup> 한국에서의 첫 경기인 <a href="/wiki/2009%EB%85%84" title="2009년">2009년</a> <a href="/wiki/5%EC%9B%94_21%EC%9D%BC" title="5월 21일">5월 21일</a> 광주구장 <a href="/wiki/KIA_%ED%83%80%EC%9D%B4%EA%B1%B0%EC%A6%88" title="KIA 타이거즈">KIA 타이거즈</a>와의 경기에 선발 투수로 나와 1.1 이닝동안 4피안타 4볼넷 7실점을 허용하였다.<sup class="reference" id="cite_ref-2"><a href="#cite_note-2">[2]</a></sup> 그러나 시즌 도중 손가락 부상을 입어 결국 4강 진출이 급급했던 <a href="/wiki/LG_%ED%8A%B8%EC%9C%88%EC%8A%A4" title="LG 트윈스">LG 트윈스</a>로부터 웨이버 공시를 통한 수순으로 퇴출되었다.</p>
