<h2><span class="mw-headline" id=".EC.83.9D.EC.95.A0">생애</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%A6%B0%ED%8E%91&amp;action=edit&amp;section=1" title="부분 편집: 생애">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p><a href="/wiki/%EC%A4%91%ED%99%94%EC%9D%B8%EB%AF%BC%EA%B3%B5%ED%99%94%EA%B5%AD" title="중화인민공화국">중화인민공화국</a> <a href="/wiki/%ED%91%B8%EC%A0%A0_%EC%84%B1" title="푸젠 성">푸젠 성</a> <a href="/wiki/%ED%91%B8%EC%A0%80%EC%9A%B0_%EC%8B%9C" title="푸저우 시">푸저우</a>이고 <a href="/wiki/%EC%A4%91%ED%99%94%EC%9D%B8%EB%AF%BC%EA%B3%B5%ED%99%94%EA%B5%AD" title="중화인민공화국">중화인민공화국</a> <a href="/wiki/%ED%91%B8%EC%A0%A0_%EC%84%B1" title="푸젠 성">푸젠 성</a> <a href="/wiki/%EC%83%A4%EB%A8%BC_%EC%8B%9C" title="샤먼 시">샤먼</a>에서 출생하였으며 <a href="/wiki/%EC%A4%91%ED%99%94%EC%9D%B8%EB%AF%BC%EA%B3%B5%ED%99%94%EA%B5%AD" title="중화인민공화국">중화인민공화국</a> <a href="/wiki/%ED%91%B8%EC%A0%A0_%EC%84%B1" title="푸젠 성">푸젠 성</a> <a href="/wiki/%ED%91%B8%EC%A0%80%EC%9A%B0_%EC%8B%9C" title="푸저우 시">푸저우</a>와 <a href="/wiki/%EA%B4%91%EB%91%A5_%EC%84%B1" title="광둥 성">중화인민공화국 광둥 성</a> <a href="/wiki/%EA%B4%91%EC%A0%80%EC%9A%B0_%EC%8B%9C" title="광저우 시">광저우</a>에서 잠시 유년기를 잠시 유년기를 보낸 적이 있는 그는 이후 <a href="/wiki/%ED%99%8D%EC%BD%A9" title="홍콩">홍콩</a>에서 신장은 180cm으로 체중은 67kg으로 성장하였다.</p>
<p><a href="/wiki/1999%EB%85%84" title="1999년">1999년</a> <a href="/wiki/%ED%99%8D%EC%BD%A9" title="홍콩">홍콩</a>의 <a href="/wiki/%ED%85%94%EB%A0%88%EB%B9%84%EC%A0%84_%EB%93%9C%EB%9D%BC%EB%A7%88" title="텔레비전 드라마">텔레비전 드라마</a>에서 <a href="/wiki/%EB%B0%B0%EC%9A%B0" title="배우">연기자</a>로 첫 데뷔하였고 <a href="/wiki/2001%EB%85%84" title="2001년">2001년</a> <a href="/wiki/%EC%98%81%ED%99%94" title="영화">영화</a> 《별련(別戀)》의 주연으로 <a class="mw-redirect" href="/wiki/%EC%98%81%ED%99%94_%EB%B0%B0%EC%9A%B0" title="영화 배우">영화 배우</a> 데뷔하였으며 <a href="/wiki/2002%EB%85%84" title="2002년">2002년</a> <a href="/wiki/%EA%B0%80%EC%88%98" title="가수">가수</a> 데뷔하였다.</p>
<p><a href="/wiki/%ED%99%8D%EC%BD%A9" title="홍콩">홍콩</a>의 <a class="mw-redirect" href="/wiki/%EC%98%81%ED%99%94_%EB%B0%B0%EC%9A%B0" title="영화 배우">영화 배우</a> 겸 <a href="/wiki/%EA%B0%80%EC%88%98" title="가수">가수</a> <a href="/wiki/%EB%A6%B0%EC%83%A4%EC%9B%A8%EC%9D%B4" title="린샤웨이">린샤웨이</a>(林夏薇)가 린펑의 사촌동생이다.</p>
