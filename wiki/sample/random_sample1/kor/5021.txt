<p><b>릴리 프랭키</b> (<span lang="ja" xml:lang="ja">リリー・フランキー</span>, Lily Franky, <a href="/wiki/1963%EB%85%84" title="1963년">1963년</a> <a href="/wiki/11%EC%9B%94_4%EC%9D%BC" title="11월 4일">11월 4일</a> ~)는 <a href="/wiki/%EC%9D%BC%EB%B3%B8" title="일본">일본</a>의 <a href="/wiki/%EC%82%BD%ED%99%94%EA%B0%80" title="삽화가">삽화가</a>이자 <a href="/wiki/%EC%9E%91%EA%B0%80" title="작가">작가</a>, <a class="mw-redirect" href="/wiki/%EC%88%98%ED%95%84%EA%B0%80" title="수필가">수필가</a>, <a href="/wiki/%EC%86%8C%EC%84%A4%EA%B0%80" title="소설가">소설가</a>, 그림책 작가, <a href="/wiki/%EB%94%94%EC%9E%90%EC%9D%B4%EB%84%88" title="디자이너">디자이너</a>, <a href="/wiki/%EC%9D%8C%EC%95%85%EA%B0%80" title="음악가">음악가</a>, <a href="/wiki/%EC%9E%91%EC%82%AC%EA%B0%80" title="작사가">작사가</a>, <a href="/wiki/%EC%9E%91%EA%B3%A1%EA%B0%80" title="작곡가">작곡가</a>, 방송 작가, 연출가, 라디오 DJ <a href="/wiki/%EC%82%AC%EC%A7%84%EA%B0%80" title="사진가">사진가</a>, <a href="/wiki/%EB%B0%B0%EC%9A%B0" title="배우">배우</a>이다. 본명은 <b>나카가와 마사야</b> (<span lang="ja" xml:lang="ja">中川 雅也</span>)이며, <a href="/wiki/%EB%AC%B4%EC%82%AC%EC%8B%9C%EB%85%B8_%EB%AF%B8%EC%88%A0%EB%8C%80%ED%95%99" title="무사시노 미술대학">무사시노 미술대학</a>을 졸업했다.</p>
