<h1 class="firstHeading" id="firstHeading" lang="ko">링단 칸</h1>
<div class="mw-body-content" id="bodyContent">

</div>
<div class="mw-content-ltr" dir="ltr" id="mw-content-text" lang="ko"><script>function mfTempOpenSection(id){var block=document.getElementById("mf-section-"+id);block.className+=" open-block";block.previousSibling.className+=" open-block";}</script><p><b>링단 칸</b>(<span style="font-size: smaller;"><a href="/wiki/%EC%A4%91%EA%B5%AD%EC%96%B4" title="중국어">중국어</a>:</span> <span lang="zh" xml:lang="zh">林丹汗</span>, <span style="font-size: smaller;"><a href="/wiki/%EC%98%81%EC%96%B4" title="영어">영어</a>:</span> <span lang="en" xml:lang="en">Ligdan Khan</span>, <a class="extiw" href="https://en.wikipedia.org/wiki/Mongolian_Cyrillic_script" title="en:Mongolian Cyrillic script">Mongolian Cyrillic</a>: Лигдэн Хутугт хаан <a href="/wiki/1592%EB%85%84" title="1592년">1592년</a> ~ <a href="/wiki/1634%EB%85%84" title="1634년">1634년</a>, 재위: <a href="/wiki/1604%EB%85%84" title="1604년">1604년</a> ~ <a href="/wiki/1634%EB%85%84" title="1634년">1634년</a>)은 몽골 <a href="/wiki/%EB%B6%81%EC%9B%90" title="북원">북원</a>의 <a href="/wiki/%EB%B3%B4%EB%A5%B4%EC%A7%80%EA%B8%B4" title="보르지긴">보르지긴</a> 대칸이다.</p>
<h2><span class="mw-headline" id=".EC.83.9D.EC.95.A0">생애</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%A7%81%EB%8B%A8_%EC%B9%B8&amp;action=edit&amp;section=1" title="부분 편집: 생애">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p><a href="/wiki/%EB%B3%B4%EC%96%80_%EC%B2%B4%EC%B2%B8_%EC%B9%B8" title="보얀 체첸 칸">보얀 체첸 칸</a>의 손자이며, 아버지가 일찍 죽어서 <a href="/wiki/1603%EB%85%84" title="1603년">1603년</a> 칸의 후계자로 지명되었다. <a href="/wiki/1604%EB%85%84" title="1604년">1604년</a> <a href="/wiki/%EB%B6%81%EC%9B%90" title="북원">북원</a>의 칸에 즉위하여 <a href="/wiki/%EC%B0%A8%ED%95%98%EB%A5%B4" title="차하르">차하르</a>를 정비하고 다른 몽골 부족들과 연합하여 명에 원정하는 등 그의 재위 초기에 <a href="/wiki/%EC%B0%A8%ED%95%98%EB%A5%B4" title="차하르">차하르</a>는 많은 발전을 이루었다. 또한 그는 <a href="/wiki/%EB%B6%88%EA%B5%90" title="불교">불교</a>를 장려하여 <a href="/wiki/%ED%8B%B0%EB%B2%A0%ED%8A%B8" title="티베트">티베트</a>에서 불교 지도자들을 초빙하고 차하르를 몽골의 불교 중심지로, 자신을 종교적 지도자인 활불(호토쿠트)로 선언했다. <a href="/wiki/%EC%BF%A0%EB%B9%8C%EB%9D%BC%EC%9D%B4_%EC%B9%B8" title="쿠빌라이 칸">쿠빌라이 칸</a> 시대의 사키아파 <a class="mw-redirect" href="/wiki/%EB%9D%BC%EB%A7%88%EA%B5%90" title="라마교">라마교</a>를 부활시켰고, 수도 차가안에 대흑천의 탱화를 설치하고 쿠리예에 사찰을 건립했다.</p>
<p><a href="/wiki/1612%EB%85%84" title="1612년">1612년</a>, 동부 몽골 부족들이 만주족의 <a href="/wiki/%ED%9B%84%EA%B8%88" title="후금">후금</a>과 혼인관계를 맺으면서 몽골의 단합은 흔들리기 시작한다. 링단은 <a href="/wiki/1618%EB%85%84" title="1618년">1618년</a> <a href="/wiki/%EB%AA%85%EB%82%98%EB%9D%BC" title="명나라">명나라</a>와 조약을 체결하고, <a href="/wiki/%ED%9B%84%EA%B8%88" title="후금">후금</a>을 견제하는 대가로 매년 은화 수만냥을 받기로 한다. <a href="/wiki/1619%EB%85%84" title="1619년">1619년</a>, 동맹부족들과 함께 <a href="/wiki/%ED%9B%84%EA%B8%88" title="후금">후금</a>을 공격하지만 대패하고, 동부의 많은 부족들이 연맹에서 이탈하기 시작한다. 이후 관계가 악화된 후금에 대항해 몽골 부족들에 대해서 가혹한 정치를 펴는데, 이로 인해 더 많은 부족들이 연맹을 이탈하고 후금과의 동맹에 가담한다.</p>
<p><a href="/wiki/1624%EB%85%84" title="1624년">1624년</a> 호르친을 비롯한 몇몇 부족들이 <a href="/wiki/%ED%9B%84%EA%B8%88" title="후금">후금</a>과 동맹을 맺는다. 링단은 보복을 위해 원정에 착수하지만, <a href="/wiki/%ED%9B%84%EA%B8%88" title="후금">후금</a>의 구원군이 도착하여 퇴각한다.</p>
<p><a href="/wiki/1627%EB%85%84" title="1627년">1627년</a> 많은 부족들이 링단에 대항하여 반란을 일으켰으며, 링단은 이를 진압하지만 차하르에서도 만명 이상의 사망자가 나온다. <a href="/wiki/1631%EB%85%84" title="1631년">1631년</a> 링단은 다싱안링 산맥을 넘어서 호르친과 투마드를 공격하고, 몽골 부족들은 이에 맞서 강력한 동맹을 형성한다.</p>
<p><a href="/wiki/1632%EB%85%84" title="1632년">1632년</a> 후금과 몽골 연맹이 링단 칸에 대해 원정하였다. 링단은 차하르의 백성들 10만명을 데리고 청해로 피신하여 재기를 노린다.</p>
<p><a href="/wiki/1634%EB%85%84" title="1634년">1634년</a> 티베트 겔룩파로 원정을 떠나던 도중 천연두로 죽고 그의 아들 <a href="/wiki/%EC%97%90%EC%A0%9C%EC%9D%B4_%EC%B9%B8" title="에제이 칸">에제이</a>가 돌아왔다. 그리고 몽골은 결국 <a href="/wiki/%EC%B2%AD%EB%82%98%EB%9D%BC" title="청나라">청나라</a>에 병합되었다.</p>
