<p><b>2016년 미국 대통령 선거 공화당 예비경선</b>은 <a href="/wiki/2016%EB%85%84" title="2016년">2016년</a> <a href="/wiki/11%EC%9B%94_8%EC%9D%BC" title="11월 8일">11월 8일</a>에 실시될 <a href="/wiki/2016%EB%85%84_%EB%AF%B8%EA%B5%AD_%EB%8C%80%ED%86%B5%EB%A0%B9_%EC%84%A0%EA%B1%B0" title="2016년 미국 대통령 선거">2016년 미국 대통령 선거</a>에서 <a href="/wiki/%EA%B3%B5%ED%99%94%EB%8B%B9_(%EB%AF%B8%EA%B5%AD)" title="공화당 (미국)">공화당</a> 대통령 후보를 선출하기 위해 2016년 <a href="/wiki/2%EC%9B%94_1%EC%9D%BC" title="2월 1일">2월 1일</a>부터 <a href="/wiki/7%EC%9B%94_21%EC%9D%BC" title="7월 21일">7월 21일</a>까지 실시될 예정이다. 현재 14명의 후보가 경선에 참여하고 있는 가운데, <a class="mw-redirect" href="/wiki/%EC%97%AC%EB%A1%A0%EC%A1%B0%EC%82%AC" title="여론조사">여론조사</a>에서는 <a href="/wiki/%EB%89%B4%EC%9A%95_%EC%A3%BC" title="뉴욕 주">뉴욕 주</a>의 <a href="/wiki/%EC%82%AC%EC%97%85%EA%B0%80" title="사업가">사업가</a>인 <a href="/wiki/%EB%8F%84%EB%84%90%EB%93%9C_%ED%8A%B8%EB%9F%BC%ED%94%84" title="도널드 트럼프">도널드 트럼프</a>가 현재 1위이다.</p>

<h2><span class="mw-headline" id=".EA.B2.BD.EC.84.A0_.EA.B3.BC.EC.A0.95">경선 과정</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=2016%EB%85%84_%EB%AF%B8%EA%B5%AD_%EB%8C%80%ED%86%B5%EB%A0%B9_%EC%84%A0%EA%B1%B0_%EA%B3%B5%ED%99%94%EB%8B%B9_%EC%98%88%EB%B9%84%EA%B2%BD%EC%84%A0&amp;action=edit&amp;section=3" title="부분 편집: 경선 과정">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<div class="thumb tleft">
<div class="thumbinner" style="width:252px;"><a class="image" href="/wiki/%ED%8C%8C%EC%9D%BC:Governor_of_Florida_Jeb_Bush_at_VFW_in_Hudson,_New_Hampshire,_July_8th,_2015_by_Michael_Vadon_a_03.jpg"><img alt="" class="thumbimage" data-file-height="3840" data-file-width="5760" height="167" src="//upload.wikimedia.org/wikipedia/commons/thumb/d/d8/Governor_of_Florida_Jeb_Bush_at_VFW_in_Hudson%2C_New_Hampshire%2C_July_8th%2C_2015_by_Michael_Vadon_a_03.jpg/250px-Governor_of_Florida_Jeb_Bush_at_VFW_in_Hudson%2C_New_Hampshire%2C_July_8th%2C_2015_by_Michael_Vadon_a_03.jpg" srcset="//upload.wikimedia.org/wikipedia/commons/thumb/d/d8/Governor_of_Florida_Jeb_Bush_at_VFW_in_Hudson%2C_New_Hampshire%2C_July_8th%2C_2015_by_Michael_Vadon_a_03.jpg/375px-Governor_of_Florida_Jeb_Bush_at_VFW_in_Hudson%2C_New_Hampshire%2C_July_8th%2C_2015_by_Michael_Vadon_a_03.jpg 1.5x, //upload.wikimedia.org/wikipedia/commons/thumb/d/d8/Governor_of_Florida_Jeb_Bush_at_VFW_in_Hudson%2C_New_Hampshire%2C_July_8th%2C_2015_by_Michael_Vadon_a_03.jpg/500px-Governor_of_Florida_Jeb_Bush_at_VFW_in_Hudson%2C_New_Hampshire%2C_July_8th%2C_2015_by_Michael_Vadon_a_03.jpg 2x" width="250"/></a>
<div class="thumbcaption">
<div class="magnify"><a class="internal" href="/wiki/%ED%8C%8C%EC%9D%BC:Governor_of_Florida_Jeb_Bush_at_VFW_in_Hudson,_New_Hampshire,_July_8th,_2015_by_Michael_Vadon_a_03.jpg" title="실제 크기로"></a></div>
<a href="/wiki/%EC%A0%AD_%EB%B6%80%EC%8B%9C" title="젭 부시">젭 부시</a>는 2014년부터 2015년까지 여론조사 1위를 기록하였으나, 중반으로 넘어오면서 지지율이 추락하였다.</div>
</div>
</div>
<p><a href="/wiki/2012%EB%85%84_%EB%AF%B8%EA%B5%AD_%EB%8C%80%ED%86%B5%EB%A0%B9_%EC%84%A0%EA%B1%B0" title="2012년 미국 대통령 선거">2012년 대선</a>이 끝난 후, 기존 경선에 출마하였거나 경선 참여를 고려하였던 후보들이 거론되기 시작하였다. 대표적으로는 <a href="/wiki/%ED%8B%B0%ED%8C%8C%ED%8B%B0_%EC%9A%B4%EB%8F%99" title="티파티 운동">티파티 운동</a>의 후원을 받은 <a href="/wiki/%ED%85%8C%EB%93%9C_%ED%81%AC%EB%A3%A8%EC%A6%88" title="테드 크루즈">테드 크루즈</a>, <a href="/wiki/%EB%A7%88%EC%BD%94_%EB%A3%A8%EB%B9%84%EC%98%A4" title="마코 루비오">마코 루비오</a>, <a href="/wiki/%EB%9E%9C%EB%93%9C_%ED%8F%B4" title="랜드 폴">랜드 폴</a> 등의 젊은 연방 상원의원들이었다. 이외에 <a href="/wiki/%EB%AF%BC%EC%A3%BC%EB%8B%B9_(%EB%AF%B8%EA%B5%AD)" title="민주당 (미국)">민주당</a>이 정치적 우세를 보이고 있는 주 출신의 정치인들도 거론되었는데, 대표적으로 <a href="/wiki/%ED%81%AC%EB%A6%AC%EC%8A%A4_%ED%81%AC%EB%A6%AC%EC%8A%A4%ED%8B%B0" title="크리스 크리스티">크리스 크리스티</a>, <a href="/wiki/%EC%A1%B0%EC%A7%80_%ED%8D%BC%ED%83%80%ED%82%A4" title="조지 퍼타키">조지 퍼타키</a> 등이었다. 특히 크리스티는 <a href="/wiki/2013%EB%85%84" title="2013년">2013년</a>에 앞도적으로 <a class="new" href="/w/index.php?title=2013%EB%85%84_%EB%89%B4%EC%A0%80%EC%A7%80_%EC%A3%BC%EC%A7%80%EC%82%AC_%EC%84%A0%EA%B1%B0&amp;action=edit&amp;redlink=1" title="2013년 뉴저지 주지사 선거 (없는 문서)">재선에 성공</a>하자 차기 후보 여론조사에서 1위를 기록하였다. 그러나 <a class="new" href="/w/index.php?title=%EB%B8%8C%EB%A6%AC%EC%A7%80%EA%B2%8C%EC%9D%B4%ED%8A%B8&amp;action=edit&amp;redlink=1" title="브리지게이트 (없는 문서)">브리지게이트</a> 사건으로 크리스티가 추락한 가운데<sup class="reference" id="cite_ref-1"><a href="#cite_note-1">[1]</a></sup>, <a href="/wiki/%EC%A1%B0%EC%A7%80_W._%EB%B6%80%EC%8B%9C" title="조지 W. 부시">조지 W. 부시</a>의 동생이자 전 <a href="/wiki/%ED%94%8C%EB%A1%9C%EB%A6%AC%EB%8B%A4_%EC%A3%BC" title="플로리다 주">플로리다</a> 주지사 출신의 <a href="/wiki/%EC%A0%AD_%EB%B6%80%EC%8B%9C" title="젭 부시">젭 부시</a>가 부상하였다. 이런 가운데 기존 경선에 출마하였던 <a href="/wiki/%EB%A7%88%EC%9D%B4%ED%81%AC_%ED%97%88%EC%BB%A4%EB%B9%84" title="마이크 허커비">마이크 허커비</a>, <a href="/wiki/%EB%A6%AD_%EC%83%8C%ED%86%A0%EB%9F%BC" title="릭 샌토럼">릭 샌토럼</a> 등이 출마하면서 공화당 경선은 다량의 후보들이 참여하게 되었다.</p>
<p><br/>
한편 사업가이자 TV 호스트를 맡고 있는 <a href="/wiki/%EB%8F%84%EB%84%90%EB%93%9C_%ED%8A%B8%EB%9F%BC%ED%94%84" title="도널드 트럼프">도널드 트럼프</a>가 대선 출마를 선언하였다. 대체로의 반응은 회의적이었으나 곧이어 실시된 여론조사에서 트럼프는 1위를 차지해 정계를 충격에 빠뜨렸다. 기존 1위를 차지하던 부시가 트럼프의 부상을 제대로 견제하지 못하던 사이, 외부 인사였던 <a href="/wiki/%EB%B2%A4_%EC%B9%B4%EC%8A%A8" title="벤 카슨">벤 카슨</a>, <a href="/wiki/%EC%B9%BC%EB%A6%AC_%ED%94%BC%EC%98%A4%EB%A6%AC%EB%82%98" title="칼리 피오리나">칼리 피오리나</a>가 급부상하기 시작하였다. 이들은 이전에 정치적 경험이 전무하거나 선거에 출마하지 않았던 사람들이어서 이들이 선두권을 달리자 기존 주류에서는 위기감을 느끼기 시작하였다. <a href="/wiki/%EC%9C%84%EC%8A%A4%EC%BD%98%EC%8B%A0_%EC%A3%BC" title="위스콘신 주">위스콘신</a> 주지사였던 <a class="new" href="/w/index.php?title=%EC%8A%A4%EC%BD%A7_%EC%9B%8C%EC%BB%A4&amp;action=edit&amp;redlink=1" title="스콧 워커 (없는 문서)">스콧 워커</a>는 트럼프의 지명을 막아야 된다며 경선을 포기하였다.</p>
<p>트럼프의 1위가 시간이 지날수록 고착화되자, 공화당에서는 트럼프의 지명을 막기 위해 새로운 후보가 떠오르기 시작했다. 특히 기존 주류에서는 현 상원의원을 맡고 있는 루비오와 크루즈가 부상하기 시작하였다. 또한 외부 인사이면서 지지 범위가 넓은 카슨이 2위를 기록하기 시작하는 등 공화당 경선은 장기화 흐름을 보이게 된다.</p>
<p><a href="/wiki/2015%EB%85%84_11%EC%9B%94_%ED%8C%8C%EB%A6%AC_%ED%85%8C%EB%9F%AC" title="2015년 11월 파리 테러">2015년 11월 파리 테러</a>가 벌어진 후, 트럼프는 "<a href="/wiki/%EB%AC%B4%EC%8A%AC%EB%A6%BC" title="무슬림">무슬림</a>의 입국을 제한하여야 한다"는 발언을 하여 논란을 일으켰다. <a href="/wiki/%EB%B0%B1%EC%95%85%EA%B4%80" title="백악관">백악관</a>과 <a href="/wiki/%EC%A4%91%EB%8F%99" title="중동">중동</a> 여러 국가들은 트럼프를 비난하였지만, 발언 이후 여론조사에서 트럼프의 지지율이 오히려 더 높아지자 공화당 내에서는 전당대회에서 트럼프를 막아낼 후보 단일화를 진행해야 된다는 의견이 나오게 되었다. 그러나 이에 대해 트럼프는 불공정한 경선 시 탈당 후 무소속 출마를 경고하였고, 카슨 등도 비판하였다.</p>
