<p><b>링크벨트 건설기계</b>(<span style="font-size: smaller;"><a href="/wiki/%EC%98%81%EC%96%B4" title="영어">영어</a>:</span> <span lang="en" xml:lang="en">Link-Belt Construction Equipment</span>)는 <a href="/wiki/%EB%AF%B8%EA%B5%AD" title="미국">미국</a>의 건설 장비 제조 업체로 <a href="/wiki/%EC%9D%BC%EB%B3%B8" title="일본">일본</a>의 기업인 <b><a class="new" href="/w/index.php?title=%EC%8A%A4%EB%AF%B8%ED%86%A0%EB%AA%A8_%EC%A4%91%EA%B3%B5%EC%97%85&amp;action=edit&amp;redlink=1" title="스미토모 중공업 (없는 문서)">스미토모 중공업</a></b>(<span style="font-size: smaller;"><a href="/wiki/%EC%98%81%EC%96%B4" title="영어">영어</a>:</span> <span lang="en" xml:lang="en">Sumitomo Heavy Industries</span>, <span style="font-size: smaller;"><a href="/wiki/%EC%9D%BC%EB%B3%B8%EC%96%B4" title="일본어">일본어</a>:</span> <span lang="ja" xml:lang="ja">住友重機械工業株式会社</span>)의 자회사로 본사는 <a href="/wiki/%EB%AF%B8%EA%B5%AD" title="미국">미국</a> <a href="/wiki/%EC%BC%84%ED%84%B0%ED%82%A4_%EC%A3%BC" title="켄터키 주">켄터키 주</a> <a class="mw-disambig" href="/wiki/%EB%A0%89%EC%8B%B1%ED%84%B4" title="렉싱턴">렉싱턴</a>에 위치해 있다.</p>

<h2><span class="mw-headline" id=".EC.97.AD.EC.82.AC">역사</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%A7%81%ED%81%AC%EB%B2%A8%ED%8A%B8_%EA%B1%B4%EC%84%A4%EA%B8%B0%EA%B3%84&amp;action=edit&amp;section=1" title="부분 편집: 역사">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p><a href="/wiki/1880%EB%85%84" title="1880년">1880년</a> <b>윌리엄 다나이 워트</b>(<span style="font-size: smaller;"><a href="/wiki/%EC%98%81%EC%96%B4" title="영어">영어</a>:</span> <span lang="en" xml:lang="en">William Dana Ewart</span>)가 <b>링크벨트 머시너리 컴파니</b>(<span style="font-size: smaller;"><a href="/wiki/%EC%98%81%EC%96%B4" title="영어">영어</a>:</span> <span lang="en" xml:lang="en">Link-Belt Machinery Company</span>)로 설립했다. <a href="/wiki/1874%EB%85%84" title="1874년">1874년</a>에 수확기에 사용하기 위해 최초로 <a class="new" href="/w/index.php?title=%EC%B2%B4%EC%9D%B8_%EB%93%9C%EB%9D%BC%EC%9D%B4%EB%B8%8C&amp;action=edit&amp;redlink=1" title="체인 드라이브 (없는 문서)">체인 드라이브</a>를 개발했다. <a href="/wiki/1922%EB%85%84" title="1922년">1922년</a>에 당시 <a href="/wiki/%EA%B8%B0%EC%A4%91%EA%B8%B0" title="기중기">기중기</a>와 <a class="mw-redirect" href="/wiki/%EA%B5%B4%EC%82%AD%EA%B8%B0" title="굴삭기">굴삭기</a> 제작사인 <b>스피더 컴포테이션</b>(<span style="font-size: smaller;"><a href="/wiki/%EC%98%81%EC%96%B4" title="영어">영어</a>:</span> <span lang="en" xml:lang="en">Link-Belt Speeder Corporation</span>)을 인수했다. 이후 <a href="/wiki/1939%EB%85%84" title="1939년">1939년</a>에 <b>스피더 컴포테이션</b>(<span style="font-size: smaller;"><a href="/wiki/%EC%98%81%EC%96%B4" title="영어">영어</a>:</span> <span lang="en" xml:lang="en">Link-Belt Speeder Corporation</span>)과 합병했다. <a href="/wiki/1967%EB%85%84" title="1967년">1967년</a>에 <a href="/wiki/%EB%AF%B8%EA%B5%AD" title="미국">미국</a>의 기업이였던 <b>FMC 컴포테이션</b>(<span style="font-size: smaller;"><a href="/wiki/%EC%98%81%EC%96%B4" title="영어">영어</a>:</span> <span lang="en" xml:lang="en">FMC Corporation</span>)이 인수했다. <a href="/wiki/1986%EB%85%84" title="1986년">1986년</a>에 <a href="/wiki/%EC%9D%BC%EB%B3%B8" title="일본">일본</a>의 대기업인 <a class="new" href="/w/index.php?title=%EC%8A%A4%EB%AF%B8%ED%86%A0%EB%AA%A8_%EC%A4%91%EA%B3%B5%EC%97%85&amp;action=edit&amp;redlink=1" title="스미토모 중공업 (없는 문서)">스미토모 중공업</a>과 합작 투자를 형성했다. <a href="/wiki/1980%EB%85%84" title="1980년">1980년</a>에 <a href="/wiki/%EC%86%8C%EB%B0%A9%EC%B0%A8" title="소방차">소방차</a> 사업까지 진출 했으나 부진으로 <a href="/wiki/1990%EB%85%84" title="1990년">1990년</a>에 생산이 중지되었다.<sup class="reference" id="cite_ref-1"><a href="#cite_note-1">[1]</a></sup> <a href="/wiki/1998%EB%85%84" title="1998년">1998년</a>에 <b>LBX 컴파니</b>(<span style="font-size: smaller;"><a href="/wiki/%EC%98%81%EC%96%B4" title="영어">영어</a>:</span> <span lang="en" xml:lang="en">LBX Company</span>)가 설립되면서 <a class="mw-redirect" href="/wiki/%EA%B5%B4%EC%82%AD%EA%B8%B0" title="굴삭기">굴삭기</a> 제작 사업 부문이 분사되면서 현재는 <a href="/wiki/%EA%B8%B0%EC%A4%91%EA%B8%B0" title="기중기">기중기</a>만 생산하고 있다.</p>
