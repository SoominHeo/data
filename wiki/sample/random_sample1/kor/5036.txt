<p><b>마거릿 힐다 대처</b> 여남작(<span style="font-size: smaller;"><a href="/wiki/%EC%98%81%EC%96%B4" title="영어">영어</a>:</span> <span lang="en" xml:lang="en">Margaret Hilda Thatcher</span> <small>마거릿 힐더 새처<sup>[<a href="/wiki/%EC%9C%84%ED%82%A4%EB%B0%B1%EA%B3%BC:%EC%98%81%EC%96%B4%EC%9D%98_%ED%95%9C%EA%B8%80_%ED%91%9C%EA%B8%B0" title="위키백과:영어의 한글 표기">*</a>]</sup></small>, Baroness Thatcher, <a href="/wiki/%EA%B0%80%ED%84%B0_%ED%9B%88%EC%9E%A5" title="가터 훈장">LG</a>, <a class="extiw" href="https://en.wikipedia.org/wiki/Order_of_Merit" title="en:Order of Merit">OM</a>, <a href="/wiki/1925%EB%85%84" title="1925년">1925년</a> <a href="/wiki/10%EC%9B%94_13%EC%9D%BC" title="10월 13일">10월 13일</a>~<a href="/wiki/2013%EB%85%84" title="2013년">2013년</a> <a href="/wiki/4%EC%9B%94_8%EC%9D%BC" title="4월 8일">4월 8일</a>)은 <a href="/wiki/1979%EB%85%84" title="1979년">1979년</a>부터 <a href="/wiki/1990%EB%85%84" title="1990년">1990년</a>까지 <a href="/wiki/%EC%98%81%EA%B5%AD" title="영국">영국</a>의 <a href="/wiki/%EC%98%81%EA%B5%AD%EC%9D%98_%EC%B4%9D%EB%A6%AC" title="영국의 총리">총리</a>를 지낸 <a href="/wiki/%EC%A0%95%EC%B9%98%EA%B0%80" title="정치가">정치가</a>이다. 영국 최초의 여성 <a href="/wiki/%EB%B3%B4%EC%88%98%EB%8B%B9_(%EC%98%81%EA%B5%AD)" title="보수당 (영국)">보수당</a> 당수이기도 하다. 대처는 영국병을 극복한 인물로 비정된다. 집권 후 긴축재정을 실시하여 물가 인상을 억제하였으며, 이후 소득세 감면, 소비세와 간접세 증가, 은행 금리와 이자율 증가, 정부 규모 축소, 실력 성과제도 도입, 민간 기업의 업무 간소화 추진 등을 통해 영국의 전체적 경제 성장률을 플러스로 돌려놓았으며, 인플레이션을 잡는데도 기여했다. <a href="/wiki/1982%EB%85%84" title="1982년">1982년</a> <a href="/wiki/%EC%95%84%EB%A5%B4%ED%97%A8%ED%8B%B0%EB%82%98" title="아르헨티나">아르헨티나</a>와의 <a href="/wiki/%ED%8F%AC%ED%81%B4%EB%9E%9C%EB%93%9C_%EC%A0%84%EC%9F%81" title="포클랜드 전쟁">포클랜드 전쟁</a>에서 승리하였다.</p>
<p>영국 옥스퍼드대학의 서머빌 칼리지에서 법학과 화학을 전공하고, <a href="/wiki/1959%EB%85%84" title="1959년">1959년</a> <a href="/wiki/%EB%B3%B4%EC%88%98%EB%8B%B9_(%EC%98%81%EA%B5%AD)" title="보수당 (영국)">보수당</a> <a class="mw-redirect" href="/wiki/%EC%98%81%EA%B5%AD_%ED%95%98%EC%9B%90" title="영국 하원">영국 하원</a> 의원에 당선되어 정계에 입문했다. 이후 1961년∼1964년 연금ㆍ국가보험 정무담당 차관, 1970년부터 74년까지는 보수당 히스 내각의 교육부장관, 과학부장관을 지냈다. <a href="/wiki/1975%EB%85%84" title="1975년">1975년</a>에는 보수당의 당수로 선발되어 <a href="/wiki/%EC%98%81%EA%B5%AD" title="영국">영국</a> 최초의 여성 당수가 되었다. <a href="/wiki/1979%EB%85%84" title="1979년">1979년</a> 총선거에서는 노동당의 <a href="/wiki/%EC%A0%9C%EC%9E%84%EC%8A%A4_%EC%BA%98%EB%9F%AC%ED%97%8C" title="제임스 캘러헌">제임스 캘러헌</a> 전 수상을 누르고 승리하여 첫 여성 총리가 됐다. 지속적으로 국민의 지지를 얻어 <a href="/wiki/1987%EB%85%84" title="1987년">1987년</a> 총선거에 의해 3선에 성공하였으며, 이를 통해 20세기 영국 총리 중 가장 긴 11년 209일의 재임기간을 지냈다.<sup class="reference" id="cite_ref-2"><a href="#cite_note-2">[2]</a></sup></p>
<p><a href="/wiki/%EC%9C%A0%EB%9F%BD" title="유럽">유럽</a>에서 혈통이나 재산, 결혼의 도움을 받지 않고 강대국의 지도자가 된 역사상 최초의 여성 정치인이기도 하다. 보수적이며 온화하면서도 강경한 성품으로 <b><a href="/wiki/%EC%B2%A0%EC%9D%98_%EC%97%AC%EC%9D%B8" title="철의 여인">철의 여인</a></b>(Iron Lady)이라는 별명으로 불려서, 전기 영화 《<a href="/wiki/%EC%B2%A0%EC%9D%98_%EC%97%AC%EC%9D%B8_(%EC%98%81%ED%99%94)" title="철의 여인 (영화)">철의 여인</a>》도 만들어졌다.</p>

<h2><span class="mw-headline" id=".EC.83.9D.EC.95.A0">생애</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%A7%88%EA%B1%B0%EB%A6%BF_%EB%8C%80%EC%B2%98&amp;action=edit&amp;section=1" title="부분 편집: 생애">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<h3><span class="mw-headline" id=".EC.B4.88.EA.B8.B0_.EC.83.9D.EC.95.A0">초기 생애</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%A7%88%EA%B1%B0%EB%A6%BF_%EB%8C%80%EC%B2%98&amp;action=edit&amp;section=2" title="부분 편집: 초기 생애">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<div class="thumb tleft">
<div class="thumbinner" style="width:222px;"><a class="image" href="/wiki/%ED%8C%8C%EC%9D%BC:Maison_natale_de_Margaret_Thatcher,_Grantham.JPG"><img alt="" class="thumbimage" data-file-height="1214" data-file-width="2592" height="103" src="//upload.wikimedia.org/wikipedia/commons/thumb/a/ae/Maison_natale_de_Margaret_Thatcher%2C_Grantham.JPG/220px-Maison_natale_de_Margaret_Thatcher%2C_Grantham.JPG" srcset="//upload.wikimedia.org/wikipedia/commons/thumb/a/ae/Maison_natale_de_Margaret_Thatcher%2C_Grantham.JPG/330px-Maison_natale_de_Margaret_Thatcher%2C_Grantham.JPG 1.5x, //upload.wikimedia.org/wikipedia/commons/thumb/a/ae/Maison_natale_de_Margaret_Thatcher%2C_Grantham.JPG/440px-Maison_natale_de_Margaret_Thatcher%2C_Grantham.JPG 2x" width="220"/></a>
<div class="thumbcaption">
<div class="magnify"><a class="internal" href="/wiki/%ED%8C%8C%EC%9D%BC:Maison_natale_de_Margaret_Thatcher,_Grantham.JPG" title="실제 크기로"></a></div>
대처의 생가</div>
</div>
</div>
<p><a href="/wiki/1925%EB%85%84" title="1925년">1925년</a> <a href="/wiki/10%EC%9B%94_13%EC%9D%BC" title="10월 13일">10월 13일</a> <a href="/wiki/%EB%A7%81%EC%BB%A8%EC%85%94_%EC%A3%BC" title="링컨셔 주">링컨셔 주</a> <a class="extiw" href="https://en.wikipedia.org/wiki/Grantham" title="en:Grantham">그랜샘</a>에서 식료품집 둘째 딸로 태어났다. 결혼 전 성은 <b>로버츠</b>(Roberts)이다. 어린 시절을 그랜샘에서 보냈는데, 그곳에 아버지가 2개의 식료품점을 운영하고 있었다.<sup class="reference" id="cite_ref-Beckett-p3_3-0"><a href="#cite_note-Beckett-p3-3">[3]</a></sup> 일각에서는 그녀의 집안이 생계를 이어가기 어려울 만큼 극도로 가난한 것처럼 묘사되지만 생각처럼 가난한 것은 아니라 한다. 아버지 알프레드 로버츠는 현지의 명사로, 시장을 지낸 경험도 있었다. 대처의 생가는 대대로 <a href="/wiki/%EA%B0%90%EB%A6%AC%EA%B5%90" title="감리교">감리교</a>의 독실한 신자이며, 생가의 가훈이었던 "검소 검약", “자기책임”, “자조노력”의 정신은 대처에게도 현저하게 계승되었다. 대처는 아버지 알프레드를 매우 존경했고, “인간으로서 필요한 것은 모두 아버지에게서 배웠다”는 말을 자주 입에 담았다.</p>
<h3><span class="mw-headline" id=".EB.8C.80.ED.95.99_.EC.8B.9C.EC.A0.88">대학 시절</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%A7%88%EA%B1%B0%EB%A6%BF_%EB%8C%80%EC%B2%98&amp;action=edit&amp;section=3" title="부분 편집: 대학 시절">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p><a href="/wiki/%EC%98%A5%EC%8A%A4%ED%8D%BC%EB%93%9C_%EB%8C%80%ED%95%99%EA%B5%90" title="옥스퍼드 대학교">옥스퍼드 대학교</a> 시절에는 <a href="/wiki/%ED%94%84%EB%A6%AC%EB%93%9C%EB%A6%AC%ED%9E%88_%ED%95%98%EC%9D%B4%EC%97%90%ED%81%AC" title="프리드리히 하이에크">프리드리히 하이에크</a>의 경제학에도 심취해 있었다. 그때 품은 경제에 대한 생각이 이후의 <a href="/wiki/%EC%8B%A0%EC%9E%90%EC%9C%A0%EC%A3%BC%EC%9D%98" title="신자유주의">신자유주의</a> 경제 개혁(소위 <a class="mw-redirect" href="/wiki/%EB%8C%80%EC%B2%98%EB%A6%AC%EC%A6%98" title="대처리즘">대처리즘</a>)의 원류가 되었다. <a href="/wiki/1947%EB%85%84" title="1947년">1947년</a> <a href="/wiki/%EC%98%A5%EC%8A%A4%ED%8D%BC%EB%93%9C_%EB%8C%80%ED%95%99%EA%B5%90" title="옥스퍼드 대학교">옥스퍼드 대학교</a> 화학과를 졸업했다. 독학으로 법률을 공부해, 29세 때 변호사 시험에 합격했다.</p>
<h2><span class="mw-headline" id=".EC.A0.95.EC.B9.98_.ED.99.9C.EB.8F.99">정치 활동</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%A7%88%EA%B1%B0%EB%A6%BF_%EB%8C%80%EC%B2%98&amp;action=edit&amp;section=4" title="부분 편집: 정치 활동">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<h3><span class="mw-headline" id=".ED.95.98.EC.9B.90_.EC.9D.98.EC.9B.90">하원 의원</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%A7%88%EA%B1%B0%EB%A6%BF_%EB%8C%80%EC%B2%98&amp;action=edit&amp;section=5" title="부분 편집: 하원 의원">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p><a href="/wiki/1959%EB%85%84" title="1959년">1959년</a> <a href="/wiki/%EB%B3%B4%EC%88%98%EB%8B%B9_(%EC%98%81%EA%B5%AD)" title="보수당 (영국)">보수당</a> 공천으로 출마하여 <a class="mw-redirect" href="/wiki/%EC%98%81%EA%B5%AD_%ED%95%98%EC%9B%90" title="영국 하원">영국 하원</a>의원에 당선되어 정계에 뛰어들었고, <a href="/wiki/1970%EB%85%84" title="1970년">1970년</a> 당시의 <a href="/wiki/%EC%97%90%EB%93%9C%EC%9B%8C%EB%93%9C_%ED%9E%88%EC%8A%A4" title="에드워드 히스">에드워드 히스</a> 총리에 의해 교육부 장관에 임명되었다. <a href="/wiki/1975%EB%85%84" title="1975년">1975년</a> 영국에서는 처음으로 보수당 여성당수가 되었다.</p>
<p><a href="/wiki/1950%EB%85%84" title="1950년">1950년</a>, <a href="/wiki/%EB%B3%B4%EC%88%98%EB%8B%B9_(%EC%98%81%EA%B5%AD)" title="보수당 (영국)">보수당</a>에서 <a class="mw-redirect" href="/wiki/%ED%95%98%EC%9B%90%EC%9D%98%EC%9B%90" title="하원의원">하원의원</a> 선거에 출마했지만 낙선했다. 이듬해 <a href="/wiki/1951%EB%85%84" title="1951년">1951년</a>에는 10세 연상의 <a class="extiw" href="https://en.wikipedia.org/wiki/Denis_Thatcher" title="w:Denis Thatcher">데니스 대처</a>와 결혼하고 법률 공부를 시작한다. <a href="/wiki/1954%EB%85%84" title="1954년">1954년</a>에 변호사 자격을 취득했다. 또한 이 당시는 여성의 권리 확대를 강하게 호소했다. <a href="/wiki/1959%EB%85%84" title="1959년">1959년</a>에 하원의원에 첫 당선, <a href="/wiki/1970%EB%85%84" title="1970년">1970년</a>에 히스 내각에서 교육과학부 장관을 맡는다. 이때 교육 관련 예산을 삭감할 필요에 직면한 대처는 학교에서 우유 무상급식 폐지를 결정하고 “마거릿 대처, 우유 강탈자”(<i>Margaret Thatcher, Milk Snatcher</i>)로 비난을 받으며, 맹렬한 항의의 폭풍을 일으켰다.<sup class="reference" id="cite_ref-4"><a href="#cite_note-4">[4]</a></sup> 그는 교육시설물 개보수와 교원 인건비를 절감할 수는 없다고 맞대응했다.</p>
<h3><span class="mw-headline" id=".EB.8B.B9.EB.8C.80.ED.91.9C">당대표</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%A7%88%EA%B1%B0%EB%A6%BF_%EB%8C%80%EC%B2%98&amp;action=edit&amp;section=6" title="부분 편집: 당대표">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<div class="thumb tleft">

야당 대표 시절의 대처</div>
</div>

<p><a href="/wiki/1974%EB%85%84" title="1974년">1974년</a> 선거에서 보수당은 패배를 당했고, 이듬해 <a href="/wiki/1975%EB%85%84" title="1975년">1975년</a> 2월 보수당 당대표 선거가 실시된다. 원래 대처는 당내 우파의 키스 요셉을 지지했지만, 조셉은 수많은 문제성 발언을 야기해 당 안팎에서 반발을 받자 출마를 포기했다. 따라서 우파에서 대처가 출마한다. 교육과학부 장관의 경험 밖에 없는 대처의 당대표 선거 출마를 걱정하는 목소리도 많았지만, <a href="/wiki/%EC%97%90%EB%93%9C%EC%9B%8C%EB%93%9C_%ED%9E%88%EC%8A%A4" title="에드워드 히스">에드워드 히스</a>를 물리치고 보수당 대표로 취임한다. 같은 해 영국을 포함한 전 35개국에서 체결하여 채택된 〈<a href="/wiki/%ED%97%AC%EC%8B%B1%ED%82%A4_%EC%84%A0%EC%96%B8" title="헬싱키 선언">헬싱키 선언</a>〉을 통렬하게 비판했다.</p>
<p>반면 <a href="/wiki/%EC%86%8C%EB%A0%A8" title="소련">소련</a>의 국방부 기관지 〈크라스나야 즈붸즈다〉(빨간별, Красная звезда (газета) : 현재에도 러시아 국방부 기관지로 발행)"는 <a href="/wiki/1976%EB%85%84" title="1976년">1976년</a> <a href="/wiki/1%EC%9B%94_24%EC%9D%BC" title="1월 24일">1월 24일</a> 문제의 기사 속에서 대처를 ‘철의 여인’이라고 비난했다.<sup class="reference" id="cite_ref-5"><a href="#cite_note-5">[5]</a></sup> 아이러니하게도 이 ‘철의 여인’의 통칭을 대처 자신도 마음에 들어했으며, 모든 미디어에서 다루어졌기 때문에 대처의 대명사로 자리 잡았다.</p>
<h3><span class="mw-headline" id=".EC.B4.9D.EB.A6.AC_.EC.9E.AC.EC.A7.81_.EC.8B.9C.EC.A0.88">총리 재직 시절</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%A7%88%EA%B1%B0%EB%A6%BF_%EB%8C%80%EC%B2%98&amp;action=edit&amp;section=7" title="부분 편집: 총리 재직 시절">편집</a><span class="mw-editsection-bracket">]</span></span></h3>

<div class="thumbcaption">

<a class="mw-redirect" href="/wiki/%EB%A0%88%EC%9D%B4%EA%B1%B4" title="레이건">레이건</a>과 대처</div>

<p><a href="/wiki/1979%EB%85%84" title="1979년">1979년</a>부터 11년간 영국 최초의 여성 총리로 재임했다. 집권 후 긴축재정을 실시하여 물가 인상을 억제하였다. 이후 대처는 직후 소득세 감면, 제품에 부과된 특별소비세를 비롯한 소비세와 간접세 증가, 은행 금리와 이자율 증가, 정부 규모 축소 등을 추진했다. 또한 근무기간에 따른 연공서열제도를 폐지하고 실력 성과제도를 도입하였으며 무능력자 및 부패혐의자는 무조건 해임, 파면 등의 정책을 실시하였다. 또한 정부의 규모를 축소시켜 각종 세금의 낭비를 줄였고, 전산화와 기계 생산 설비의 보급 등 민간 기업의 업무 간소화 추진 등을 추진하였다. 이로서 세금과 각종 비용의 감소 효과를 불러와 영국의 전체적 경제 성장률을 마이너스에서 플러스로 회복시키고, <a href="/wiki/%EC%9D%B8%ED%94%8C%EB%A0%88%EC%9D%B4%EC%85%98" title="인플레이션">인플레이션</a>을 잡게 되었다. 그녀가 펼친 경제 정책은 <a href="/wiki/%EC%8B%A0%EC%9E%90%EC%9C%A0%EC%A3%BC%EC%9D%98" title="신자유주의">신자유주의</a>에 기반을 두었다. 또한 인플레와 노사분규로 인해 침체된 영국 경제를 회복하기 위한 방안 즉, <a class="mw-redirect" href="/wiki/%EB%8C%80%EC%B2%98%EB%A6%AC%EC%A6%98" title="대처리즘">대처리즘</a>을 내세웠다. 이 정책은 동시대의 정치 외교적 동지였던 <a href="/wiki/%EB%AF%B8%EA%B5%AD" title="미국">미국</a> <a href="/wiki/%EB%A1%9C%EB%84%90%EB%93%9C_%EB%A0%88%EC%9D%B4%EA%B1%B4" title="로널드 레이건">레이건</a> 대통령의 정책인 <a href="/wiki/%EB%A0%88%EC%9D%B4%EA%B1%B0%EB%85%B8%EB%AF%B9%EC%8A%A4" title="레이거노믹스">레이거노믹스</a>와 많은 점에서 공통점을 가지고 있다.</p>
<p>정치적으로는 철저히 <a href="/wiki/%EB%B0%98%EA%B3%B5%EC%A3%BC%EC%9D%98" title="반공주의">반공주의</a>를 추구했다. 하지만 맹목적인 보수주의는 아니라 사상, 출판, 신념의 자유를 보장했다. 그의 정책은 영국 국내, 사회면에서는 <a href="/wiki/%EB%B3%B4%EC%88%98%EC%A3%BC%EC%9D%98" title="보수주의">보수주의</a>와 <a href="/wiki/%EC%9E%90%EC%9C%A0%EC%A3%BC%EC%9D%98" title="자유주의">자유주의</a>가 혼재되었지만, 경제분야에서는 자유주의를 지향하였다. 하지만 공산주의나 사회주의에는 강하게 적대적이었다. 그래서 <a href="/wiki/%EC%86%8C%EB%A0%A8" title="소련">소련</a>에 의해 '<a href="/wiki/%EC%B2%A0%EC%9D%98_%EC%97%AC%EC%9D%B8" title="철의 여인">철의 여인</a>'이라는 별명이 지어졌다. <a href="/wiki/%EC%95%A8%EB%9F%B0_%EB%AC%B4%EC%96%B4" title="앨런 무어">앨런 무어</a>는 《<a href="/wiki/%EB%B8%8C%EC%9D%B4_%ED%8F%AC_%EB%B2%A4%EB%8D%B0%ED%83%80" title="브이 포 벤데타">브이 포 벤데타</a>》에서 대처리즘의 극단적인 자유주의와 보수주의적 면모를 비꼬았다.</p>
<p>경제적으로는 과감한 시장주의 경제를 도입하여 영국을 <a href="/wiki/%EC%98%81%EA%B5%AD%EB%B3%91" title="영국병">영국병</a>으로부터 구하고자 했다. 장기간 이어진 석탄 노동자와 철강 노동자 파업을 진압하고, 주요 국영 기업을 민영화했으며 사회 복지 대상자의 심사를 엄격히하고, 복지 혜택을 감축했다. 외교적으로는 미국과 우호관계를 심화하면서 <a href="/wiki/%EC%9C%A0%EB%9F%BD_%EC%97%B0%ED%95%A9" title="유럽 연합">유럽 공동체</a>(European Community)의 경제 공동체 이상의 발전에는 반대하였고, <a href="/wiki/1982%EB%85%84" title="1982년">1982년</a> <a href="/wiki/%EC%95%84%EB%A5%B4%ED%97%A8%ED%8B%B0%EB%82%98" title="아르헨티나">아르헨티나</a>의 침공으로 발발한 <a href="/wiki/%ED%8F%AC%ED%81%B4%EB%9E%9C%EB%93%9C_%EC%A0%84%EC%9F%81" title="포클랜드 전쟁">포클랜드 전쟁</a>을 승리로 이끌었다.</p>
<p>그녀의 정책은 <a href="/wiki/%ED%86%A0%EB%8B%88_%EB%B8%94%EB%A0%88%EC%96%B4" title="토니 블레어">토니 블레어</a> 총리 시절의 영국의 경제 호황의 밑바탕이 됐다고 인정받고 있으나, 반대측에서는 빈부 및 지역 격차, 영국 제조업의 붕괴를 초래했다고 비판한다. 또한 대처의 경제 개혁 정책은 당시 영국의 경제 문제를 뿌리부터 해소하지 못했다는 비판 역시 받고 있다.</p>
<p>한편, <a href="/wiki/1982%EB%85%84" title="1982년">1982년</a> 아들 마크가 <a class="new" href="/w/index.php?title=%ED%8C%8C%EB%A6%AC-%EB%8B%A4%EC%B9%B4%EB%A5%B4_%EB%9E%A0%EB%A6%AC&amp;action=edit&amp;redlink=1" title="파리-다카르 랠리 (없는 문서)">파리-다카르 랠리</a>에서 실종된 뒤 구조됐는데,<sup class="reference" id="cite_ref-.EB.A7.88.EC.95.BD1_6-0"><a href="#cite_note-.EB.A7.88.EC.95.BD1-6">[6]</a></sup><sup class="reference" id="cite_ref-.EA.B3.A0.EC.B6.A9_7-0"><a href="#cite_note-.EA.B3.A0.EC.B6.A9-7">[7]</a></sup> 이로 인해 철의 여인으로 불리던 대처가 눈물을 보이는 모습을 보이기도 하였다.<sup class="reference" id="cite_ref-.EB.A7.88.EC.95.BD1_6-1"><a href="#cite_note-.EB.A7.88.EC.95.BD1-6">[6]</a></sup><sup class="reference" id="cite_ref-.EA.B3.A0.EC.B6.A9_7-1"><a href="#cite_note-.EA.B3.A0.EC.B6.A9-7">[7]</a></sup></p>
<h2><span class="mw-headline" id=".EC.A0.95.EA.B3.84_.EC.9D.80.ED.87.B4_.EC.9D.B4.ED.9B.84">정계 은퇴 이후</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%A7%88%EA%B1%B0%EB%A6%BF_%EB%8C%80%EC%B2%98&amp;action=edit&amp;section=8" title="부분 편집: 정계 은퇴 이후">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<h3><span class="mw-headline" id=".EC.A0.95.EA.B3.84_.EC.9D.80.ED.87.B4">정계 은퇴</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%A7%88%EA%B1%B0%EB%A6%BF_%EB%8C%80%EC%B2%98&amp;action=edit&amp;section=9" title="부분 편집: 정계 은퇴">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p><a href="/wiki/1990%EB%85%84" title="1990년">1990년</a> 유럽 통합 여론이 나오자 마거릿 대처는 이를 반대하였다. 이후 대처는 줄곧 유럽 연합 참여 반대 입장을 고수하던 중 <a class="mw-disambig" href="/wiki/%EB%B3%B4%EC%88%98%EB%8B%B9" title="보수당">보수당</a> 지도부의 반발을 사게 되었다. 그해 <a href="/wiki/11%EC%9B%94" title="11월">11월</a> 보수당 당수 경선에 나섰다가 1차 투표에서 당선에 실패하자, <a href="/wiki/1991%EB%85%84" title="1991년">1991년</a> <a href="/wiki/5%EC%9B%94" title="5월">5월</a> 탈당하고 정계에서 은퇴하였다. 이후 필립 모리스사 고문이 되었고, <a href="/wiki/1993%EB%85%84" title="1993년">1993년</a> <a href="/wiki/%EB%AF%B8%EA%B5%AD" title="미국">미국</a> <a href="/wiki/%EB%B2%84%EC%A7%80%EB%8B%88%EC%95%84_%EC%A3%BC" title="버지니아 주">버지니아 주</a>의 <a class="mw-redirect" href="/wiki/%EC%9C%8C%EB%A6%AC%EC%97%84_%EC%95%A4_%EB%A9%94%EB%A6%AC_%EB%8C%80%ED%95%99%EA%B5%90" title="윌리엄 앤 메리 대학교">윌리엄 앤 메리 대학교</a> 총장으로 초빙되어 <a href="/wiki/2000%EB%85%84" title="2000년">2000년</a>까지 재직하였다.<sup class="reference" id="cite_ref-8"><a href="#cite_note-8">[8]</a></sup></p>
<p>한편 <a href="/wiki/%EC%95%84%EB%A5%B4%ED%97%A8%ED%8B%B0%EB%82%98" title="아르헨티나">아르헨티나</a>의 <a href="/wiki/%ED%8F%AC%ED%81%B4%EB%9E%9C%EB%93%9C_%EC%A0%9C%EB%8F%84" title="포클랜드 제도">포클랜드 제도</a> 탈환에 영국이 맞대응하도록 한 장본인으로 이 맞대응으로 인하여 <a href="/wiki/%ED%8F%AC%ED%81%B4%EB%9E%9C%EB%93%9C_%EC%A0%84%EC%9F%81" title="포클랜드 전쟁">포클랜드 전쟁</a>이 발발했다. 이 때문에 그가 퇴진한 직후부터 포클랜드 제도를 반환하자는 여론이 일부 제기되었고, 마거릿 대처가 사망하자 영국이 아르헨티나에 포클랜드 제도를 반환할 가능성이 높아졌다.</p>

<p>대처는 <a href="/wiki/2002%EB%85%84" title="2002년">2002년</a> 경미한 <a href="/wiki/%EB%87%8C%EC%A1%B8%EC%A4%91" title="뇌졸중">뇌졸중</a>을 겪은 이후 대외 활동을 자제하고 자택에 칩거하였다. <a href="/wiki/2003%EB%85%84" title="2003년">2003년</a> 남편 데니스가 사망하였다. <a href="/wiki/2012%EB%85%84" title="2012년">2012년</a> <a href="/wiki/%EB%B0%A9%EA%B4%91" title="방광">방광</a>에 생긴 <a href="/wiki/%EC%A2%85%EC%96%91" title="종양">종양</a>을 제거하는 수술을 받았고, <a href="/wiki/2013%EB%85%84" title="2013년">2013년</a> <a href="/wiki/4%EC%9B%94_8%EC%9D%BC" title="4월 8일">4월 8일</a> 사망했다.<sup class="reference" id="cite_ref-9"><a href="#cite_note-9">[9]</a></sup> 향년 87세</p>
<p>마거릿 대처가 사망하자 영국의 반응은 극명하게 갈렸는데 마거릿 대처의 죽음을 '쌤통'<sup class="reference" id="cite_ref-10"><a href="#cite_note-10">[10]</a></sup>이라며 자축하고 마거릿 대처가 영국의 수상임에도 불구하고 대처의 민영화 남발에 불만을 크게 품은 일부 사람들은 "마거릿 대처의 장례식도 민영화해야 한다"고 주장하기도 했다.<sup class="reference" id="cite_ref-11"><a href="#cite_note-11">[11]</a></sup> 이와 반대로 영국의 호황을 이끌어낸, 영국을 대표하는 수상으로 평가하며 그녀의 죽음을 애도하는 사람들 또한 있었다. 저서로는 《권력에로의 길》이 있다.<sup class="reference" id="cite_ref-12"><a href="#cite_note-12">[12]</a></sup></p>
<h2><span class="mw-headline" id=".EA.B0.80.EC.A1.B1">가족</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%A7%88%EA%B1%B0%EB%A6%BF_%EB%8C%80%EC%B2%98&amp;action=edit&amp;section=11" title="부분 편집: 가족">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>배우자 데니스 대처 사이에서 이란성 쌍둥이 1남 1녀를 두었는데 각각 캐롤 대처(1953년 8월 15일~)와 마크 대처(1953년 8월 15일~)이다.</p>
<p>마거릿 대처의 아들 마크 대처는 마거릿 대처가 총리직에 재직해 있는 동안 자신의 어머니가 총리인 것을 이용해서 <a href="/wiki/%EC%98%81%EA%B5%AD%EA%B5%B0" title="영국군">영국군</a> 제식 무기를 밀반출해 중동의 암시장에 내다 파는 무기밀매를 하면서 <a href="/wiki/%EB%82%A8%EC%95%84%ED%94%84%EB%A6%AC%EC%B9%B4_%EA%B3%B5%ED%99%94%EA%B5%AD" title="남아프리카 공화국">남아프리카 공화국</a>에 근거지를 만든 후 불법 용병단을 결성해서 <a href="/wiki/1996%EB%85%84" title="1996년">1996년</a>에 <a class="mw-redirect" href="/wiki/%EC%A0%81%EB%8F%84%EA%B8%B0%EB%8B%88" title="적도기니">적도기니</a>의 <a href="/wiki/%EC%BF%A0%EB%8D%B0%ED%83%80" title="쿠데타">쿠데타</a> 가담했는데 남아프리카공화국에서 적도기니로 무장병력 70명을 수송하던 도중 <a href="/wiki/%EC%A7%90%EB%B0%94%EB%B8%8C%EC%9B%A8" title="짐바브웨">짐바브웨</a>에서 붙잡히는 바람에 쿠데타가 실패했다. 결국 이 혐의로 도주하다가 <a href="/wiki/2004%EB%85%84" title="2004년">2004년</a>에 <a class="mw-redirect" href="/wiki/%EB%82%A8%EC%95%84%ED%94%84%EB%A6%AC%EC%B9%B4%EA%B3%B5%ED%99%94%EA%B5%AD" title="남아프리카공화국">남아프리카공화국</a>에서 체포되었다. 하지만 어머니인 마거릿 대처가 엄청난 보석금을 물어줘서야 겨우 집행유예로 석방 후 영국으로 추방되었다. 이 일로 인해 마크 대처는 <a href="/wiki/%EB%AF%B8%EA%B5%AD" title="미국">미국</a>에서 영구 입국불허 인물로 등록되었다. 하지만 마크의 가족들이 미국에서 살고 있기 때문에 가족을 만나러 갈 수가 없었다. 그 이후 모친상을 당하자 장례식에서 오래간만에 가족들과 재회했다. 마크 대처의 행보로 미루어본다면 마거릿 대처의 자녀교육은 확실히 실패했다. 일각에서는 마크 대처를 <a href="/wiki/%EA%B9%80%EC%A0%95%EB%82%A8_(1971%EB%85%84)" title="김정남 (1971년)">김정남 (1971년)</a>과 더불어 유명인사의 최악의 아들로 선정했다.</p>
