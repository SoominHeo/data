<p><b>마건충</b>(<small><a href="/wiki/%EC%A4%91%EA%B5%AD%EC%96%B4" title="중국어">중국어</a></small> <small><a href="/wiki/%EC%A0%95%EC%B2%B4%EC%9E%90" title="정체자">정체</a>:</small> <span lang="zh-hant" xml:lang="zh-hant">馬建忠</span>, <small><a href="/wiki/%ED%95%9C%EC%96%B4_%EB%B3%91%EC%9D%8C" title="한어 병음">병음</a>:</small> <font face="Lucida Sans Unicode">Ma Jianzhong</font>, <a href="/wiki/1845%EB%85%84" title="1845년">1845년</a> <a href="/wiki/2%EC%9B%94_9%EC%9D%BC" title="2월 9일">2월 9일</a> - <a href="/wiki/1900%EB%85%84" title="1900년">1900년</a> <a href="/wiki/8%EC%9B%94_14%EC%9D%BC" title="8월 14일">8월 14일</a>)은 <a href="/wiki/%EC%B2%AD%EB%82%98%EB%9D%BC" title="청나라">청나라</a> 말의 사상가, 외교관, 정치가, 언어학자이다.</p>
<p>이홍장의 관리를 지내며 청일전쟁 강화조약에도 참관한 그의 자는 미숙(眉叔)이며, <a href="/wiki/%EC%9E%A5%EC%91%A4_%EC%84%B1" title="장쑤 성">장쑤 성</a> <a href="/wiki/%EC%A0%84%EC%9E%A5_%EC%8B%9C" title="전장 시">전장</a> 출신으로 마건훈과 마상백의 동생이다.</p>

<h2><span class="mw-headline" id=".EC.83.9D.EC.95.A0">생애</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%A7%88%EA%B1%B4%EC%B6%A9&amp;action=edit&amp;section=1" title="부분 편집: 생애">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p><a href="/wiki/%EA%B0%80%ED%86%A8%EB%A6%AD" title="가톨릭">가톨릭</a>교도의 가정에서 태어났다. 둘째 형 마건훈은 증국 관리에게 발탁되어 <a href="/wiki/%EC%9D%B4%ED%99%8D%EC%9E%A5" title="이홍장">이홍장</a>의 막료가 되어 <a href="/wiki/%ED%9A%8C%EA%B5%B0" title="회군">회군</a>의 병량대를 맡았다. 넷째 형 마상백은 교육자로 전단 대학(震旦大學)과 <a href="/wiki/%ED%91%B8%EB%8B%A8_%EB%8C%80%ED%95%99" title="푸단 대학">푸단 대학</a>의 설립자이다.</p>
<p>마건충은 어려서부터 <a href="/wiki/%EA%B3%BC%EA%B1%B0" title="과거">과거</a>를 목표로 <a href="/wiki/%EC%82%AC%EC%84%9C%EC%98%A4%EA%B2%BD" title="사서오경">사서오경</a>을 배웠지만, <a href="/wiki/1853%EB%85%84" title="1853년">1853년</a> <a href="/wiki/%ED%95%A8%ED%92%8D%EC%A0%9C" title="함풍제">함풍제</a> 3년에 <a href="/wiki/%ED%83%9C%ED%8F%89%EC%B2%9C%EA%B5%AD" title="태평천국">태평천국</a> 군이 <a class="mw-redirect" href="/wiki/%EB%82%9C%EC%A7%95" title="난징">난징</a>에 입성하면서 가족은 <a class="mw-redirect" href="/wiki/%EC%83%81%ED%95%98%EC%9D%B4" title="상하이">상하이</a>로 피난했다. 마건충과 형제들은 <a href="/wiki/%EC%98%88%EC%88%98%ED%9A%8C" title="예수회">예수회</a>가 설립한 서회공학(徐匯公学)에 입학하여 <a href="/wiki/%ED%94%84%EB%9E%91%EC%8A%A4%EC%96%B4" title="프랑스어">프랑스어</a>와 <a href="/wiki/%EB%9D%BC%ED%8B%B4%EC%96%B4" title="라틴어">라틴어</a>를 배웠다. <a href="/wiki/%EC%A0%9C2%EC%B0%A8_%EC%95%84%ED%8E%B8_%EC%A0%84%EC%9F%81" title="제2차 아편 전쟁">제2차 아편 전쟁</a>을 통해 <a href="/wiki/%EC%84%9C%EC%96%91" title="서양">서양</a>의 학문을 배울 필요성을 절감하고, <a href="/wiki/%EB%9D%BC%ED%8B%B4%EC%96%B4" title="라틴어">라틴어</a>, <a href="/wiki/%ED%94%84%EB%9E%91%EC%8A%A4%EC%96%B4" title="프랑스어">프랑스어</a>, <a href="/wiki/%EC%98%81%EC%96%B4" title="영어">영어</a>, <a href="/wiki/%EA%B7%B8%EB%A6%AC%EC%8A%A4%EC%96%B4" title="그리스어">그리스어</a>를 계속 공부했다.</p>
<p><a href="/wiki/1870%EB%85%84" title="1870년">1870년</a> <a href="/wiki/%EB%8F%99%EC%B9%98" title="동치">동치</a> 9년, 둘째 형 마건훈의 인도로 이홍장의 막료가 되어 문화 지식과 실력을 인정받았다. <a href="/wiki/1878%EB%85%84" title="1878년">1878년</a> <a href="/wiki/%EA%B4%91%EC%84%9C_(%EC%97%B0%ED%98%B8)" title="광서 (연호)">광서</a> 4년 낭중의 자격으로 이홍장 의해 <a href="/wiki/%EA%B5%AD%EC%A0%9C%EB%B2%95" title="국제법">국제법</a>을 배우기 위하여 <a href="/wiki/%ED%94%84%EB%9E%91%EC%8A%A4" title="프랑스">프랑스</a>로 파견되었다. 동시에 프랑스 주재 공사 곽숭도(郭嵩燾)의 통역이 되었다. <a class="mw-disambig" href="/wiki/%ED%8C%8C%EB%A6%AC" title="파리">파리</a>에서 중국인으로서는 처음으로 학사 학위를 받고, <a href="/wiki/%ED%8C%8C%EB%A6%AC_%EC%A0%95%EC%B9%98_%EB%8C%80%ED%95%99" title="파리 정치 대학">파리 정치 대학</a>에서 법학 학위를 얻었다.</p>
<p><a href="/wiki/1880%EB%85%84" title="1880년">1880년</a> 광서 6년, <a class="mw-redirect" href="/wiki/%ED%86%88%EC%A7%84" title="톈진">톈진</a>으로 돌아와 <a href="/wiki/%EC%9D%B4%ED%99%8D%EC%9E%A5" title="이홍장">이홍장</a> 휘하에서 <a href="/wiki/%EC%96%91%EB%AC%B4%EC%9A%B4%EB%8F%99" title="양무운동">양무운동</a>을 추진한다. 다음 해 <a href="/wiki/1881%EB%85%84" title="1881년">1881년</a>에는 <a href="/wiki/%EC%95%84%ED%8E%B8" title="아편">아편</a> 전매와 세수 문제의 해결을 위해 <a href="/wiki/%EC%9D%B8%EB%8F%84_%EC%A0%9C%EA%B5%AD" title="인도 제국">인도 제국</a>에 파견되었다. <a href="/wiki/1882%EB%85%84" title="1882년">1882년</a>에는 이홍장 의해 <a href="/wiki/%EC%A1%B0%EC%84%A0" title="조선">조선</a>에 파견되어 조선과 <a href="/wiki/%EC%98%81%EA%B5%AD" title="영국">영국</a>, <a href="/wiki/%EB%AF%B8%EA%B5%AD" title="미국">미국</a>, <a href="/wiki/%EB%8F%85%EC%9D%BC" title="독일">독일</a>과의 <a class="mw-redirect" href="/wiki/%EC%A1%B0%EB%AF%B8%EC%88%98%ED%98%B8%ED%86%B5%EC%83%81%EC%A1%B0%EC%95%BD" title="조미수호통상조약">조미수호통상조약</a> 등 통상조약 체결을 추진한다. <a href="/wiki/%EC%9E%84%EC%98%A4%EA%B5%B0%EB%9E%80" title="임오군란">임오군란</a>이 발생하자 <a href="/wiki/%ED%9D%A5%EC%84%A0%EB%8C%80%EC%9B%90%EA%B5%B0" title="흥선대원군">흥선대원군</a>을 납치하여 연행했다. 같은 해 현재도 대한민국에서 사용되고 있는 태극기의 도안을 제안했다고 전해진다.<sup class="reference" id="cite_ref-1"><a href="#cite_note-1">[1]</a></sup></p>
<p><a href="/wiki/1884%EB%85%84" title="1884년">1884년</a>, <a class="new" href="/w/index.php?title=%EB%8B%B9%EC%A0%95%EC%B6%94&amp;action=edit&amp;redlink=1" title="당정추 (없는 문서)">당정추</a>(唐廷樞)가 관리하는 ‘윤선상초국’(輪船商招局)에 들어 가 <a href="/wiki/1890%EB%85%84" title="1890년">1890년</a>에는 대외 무역의 발전과 민간 산업의 진흥을 주장한 《부민설》(富民說)을 저술하여 이홍장에 제출했다. 이홍장으로부터 상하이 기기직포국(上海機器職布局) 총재에 임명되기도 했지만, 경영난으로 사직하여 고향에 은퇴했지만, <a href="/wiki/1895%EB%85%84" title="1895년">1895년</a>에 이홍장으로부터 <a class="mw-redirect" href="/wiki/%EB%B2%A0%EC%9D%B4%EC%A7%95" title="베이징">베이징</a>으로 호출되어 <a class="mw-redirect" href="/wiki/%EC%B2%AD%EC%9D%BC%EC%A0%84%EC%9F%81" title="청일전쟁">청일전쟁</a>의 강화 조약을 맺기 위해에 일본으로 향한 이홍장을 보좌했다. <a href="/wiki/1896%EB%85%84" title="1896년">1896년</a> 광서 22년, 상하이에서 〈시무보〉(時務報)의 주필인 <a href="/wiki/%EB%9F%89%EC%B9%98%EC%B0%A8%EC%98%A4" title="량치차오">량치차오</a>(梁啓超)를 알게되어 《적가재기언기행》(適可斎記言記行)을 저술했다.</p>
<p><a href="/wiki/1898%EB%85%84" title="1898년">1898년</a> 광서 26년, 중국인에 의해 저술된 최초의 중국의 문법서인 《문통(마씨문통)》(文通 또는 馬氏文通)을 저술 간행했다. (이미 서양인들에 의해 쓰여진 문법서는 여러 권 존재하고 있었다.) <sup class="reference" id="cite_ref-2"><a href="#cite_note-2">[2]</a></sup><sup class="reference" id="cite_ref-3"><a href="#cite_note-3">[3]</a></sup> 오늘 날 대부분의 학자들의 형인 마상백이 이 저서의 저술에 관여했다고 믿고 있다.<sup class="reference" id="cite_ref-4"><a href="#cite_note-4">[4]</a></sup></p>
<p><a href="/wiki/1900%EB%85%84" title="1900년">1900년</a> 광서 28년 다시 이홍장에게 소환되었다가 같은 해에 사망했다.</p>
<h2><span class="mw-headline" id=".EC.82.AC.EC.83.81">사상</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%A7%88%EA%B1%B4%EC%B6%A9&amp;action=edit&amp;section=2" title="부분 편집: 사상">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>마건충은 통행료의 폐지, <a href="/wiki/%EA%B4%80%EC%84%B8" title="관세">관세</a> 자주권의 회복, 무역 확대, 민간 산업의 진흥, 의회 제도의 채택을 주장했다. 또한 많은 사람들이 외국의 과학과 문화 지식을 흡수할 수 있도록 번역 서원을 개설하자고 제안했다.</p>
