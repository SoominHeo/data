<p><b>마그레브</b> (المغرب العربي)는 <a href="/wiki/%EC%95%84%EB%9E%8D%EC%96%B4" title="아랍어">아랍어</a>로 "해가 지는 지역" 또는 "서쪽" 이란 뜻으로 대체로 오늘날의 <a href="/wiki/%EB%B6%81%EC%95%84%ED%94%84%EB%A6%AC%EC%B9%B4" title="북아프리카">북아프리카</a> 지역, 즉, <a href="/wiki/%EB%AA%A8%EB%A1%9C%EC%BD%94" title="모로코">모로코</a>, <a href="/wiki/%EC%95%8C%EC%A0%9C%EB%A6%AC" title="알제리">알제리</a>, <a href="/wiki/%ED%8A%80%EB%8B%88%EC%A7%80" title="튀니지">튀니지</a>를 아우르는 지역을 말한다. 역사적으로는 때로 <a class="mw-redirect" href="/wiki/%EC%9D%B4%EC%8A%AC%EB%9E%8C" title="이슬람">이슬람</a>이 지배하는 <a href="/wiki/%EC%9D%B4%EB%B2%A0%EB%A6%AC%EC%95%84_%EB%B0%98%EB%8F%84" title="이베리아 반도">이베리아 반도</a>와 <a href="/wiki/%EC%8B%9C%EC%B9%A0%EB%A6%AC%EC%95%84" title="시칠리아">시칠리아</a>, <a href="/wiki/%EB%AA%B0%ED%83%80" title="몰타">몰타</a>를 포괄하여 지칭하기도 했다.</p>
