<p><b>마누엘 "마놀로" 산치스 온티유엘로</b>(<span style="font-size: smaller;"><a href="/wiki/%EC%8A%A4%ED%8E%98%EC%9D%B8%EC%96%B4" title="스페인어">스페인어</a>:</span> <span lang="es" xml:lang="es">Manuel "Manolo" Sanchís Hontiyuelo</span>, <a href="/wiki/1965%EB%85%84" title="1965년">1965년</a> <a href="/wiki/5%EC%9B%94_23%EC%9D%BC" title="5월 23일">5월 23일</a>, <a href="/wiki/%EB%A7%88%EB%93%9C%EB%A6%AC%EB%93%9C" title="마드리드">마드리드</a> ~ )는 은퇴한 <a href="/wiki/%EC%8A%A4%ED%8E%98%EC%9D%B8" title="스페인">스페인</a>의 축구 선수로, 현역 시절 <a href="/wiki/%EC%88%98%EB%B9%84%EC%88%98" title="수비수">스위퍼</a>로 활약하였다.</p>
<p><a href="/wiki/%EB%A0%88%EC%95%8C_%EB%A7%88%EB%93%9C%EB%A6%AC%EB%93%9C_CF" title="레알 마드리드 CF">레알 마드리드 CF</a>의 유소년팀을 주축으로 한 <a class="new" href="/w/index.php?title=%ED%80%B8%ED%83%80_%EB%8D%B8_%EB%B6%80%EC%9D%B4%ED%8A%B8%EB%A0%88&amp;action=edit&amp;redlink=1" title="퀸타 델 부이트레 (없는 문서)">퀸타 델 부이트레</a>의 주축 멤버들 중 하나로, <a href="/wiki/%EB%A0%88%EC%95%8C_%EB%A7%88%EB%93%9C%EB%A6%AC%EB%93%9C_CF" title="레알 마드리드 CF">레알 마드리드 CF</a> 역사상 5명의 원클럽 맨들 중 하나이다. 그의 아버지인 마누엘 산치스 마르티네스(Manuel Sanchís Martínez)도 또한 축구선수로, 그와 마찬가지로 <a href="/wiki/%EB%A0%88%EC%95%8C_%EB%A7%88%EB%93%9C%EB%A6%AC%EB%93%9C_CF" title="레알 마드리드 CF">레알 마드리드 CF</a>와 <a href="/wiki/%EC%8A%A4%ED%8E%98%EC%9D%B8_%EC%B6%95%EA%B5%AC_%EA%B5%AD%EA%B0%80%EB%8C%80%ED%91%9C%ED%8C%80" title="스페인 축구 국가대표팀">스페인 축구 국가대표팀</a> 일원이었다.</p>
<p>산치스는 그의 700번이 넘는 클럽의 공식 경기에 출장하였으며, 국가대표팀 일원으로 한차례의 <a href="/wiki/FIFA_%EC%9B%94%EB%93%9C%EC%BB%B5" title="FIFA 월드컵">FIFA 월드컵</a>과 한차례의 <a href="/wiki/UEFA_%EC%9C%A0%EB%9F%BD_%EC%B6%95%EA%B5%AC_%EC%84%A0%EC%88%98%EA%B6%8C_%EB%8C%80%ED%9A%8C" title="UEFA 유럽 축구 선수권 대회">UEFA 유럽 축구 선수권 대회</a>에 참여하였다.</p>

<h2><span class="mw-headline" id=".ED.81.B4.EB.9F.BD_.EA.B2.BD.EB.A0.A5">클럽 경력</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%A7%88%EB%88%84%EC%97%98_%EC%82%B0%EC%B9%98%EC%8A%A4_%EC%98%A8%ED%8B%B0%EC%9C%A0%EC%97%98%EB%A1%9C&amp;action=edit&amp;section=1" title="부분 편집: 클럽 경력">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p><a href="/wiki/%EB%A0%88%EC%95%8C_%EB%A7%88%EB%93%9C%EB%A6%AC%EB%93%9C_CF" title="레알 마드리드 CF">레알 마드리드 CF</a>의 유소년팀 출신으로, <a href="/wiki/%EB%A7%88%EB%93%9C%EB%A6%AC%EB%93%9C" title="마드리드">마드리드</a> 출신의 산치스는 1983년 12월 4일, <a href="/wiki/%EB%A0%88%EC%95%8C_%EB%AC%B4%EB%A5%B4%EC%8B%9C%EC%95%84" title="레알 무르시아">레알 무르시아</a>전에서 데뷔하였으며, 이후 그의 데뷔 시즌에서 17골 2득점의 1군 출전 기록을 세웠다.</p>
<p>이어지는 15시즌 동안, 산치스는 2시즌을 빼놓고 모두 30경기 이상 출전하였다. 그는 머랭 군단의 주축 선수로, 6차례의 <a class="mw-redirect" href="/wiki/%ED%94%84%EB%A6%AC%EB%A9%94%EB%9D%BC%EB%A6%AC%EA%B0%80" title="프리메라리가">라 리가</a> 우승을 거두고, 2차례의 <a href="/wiki/UEFA_%EC%B1%94%ED%94%BC%EC%96%B8%EC%8A%A4%EB%A6%AC%EA%B7%B8" title="UEFA 챔피언스리그">UEFA 챔피언스리그</a>, 2차례의 <a href="/wiki/UEFA_%EC%9C%A0%EB%A1%9C%ED%8C%8C%EB%A6%AC%EA%B7%B8" title="UEFA 유로파리그">UEFA 컵</a> 우승을 거두었다. 비록 <a href="/wiki/%ED%94%84%EB%A6%AC%EB%A9%94%EB%9D%BC%EB%A6%AC%EA%B0%80_1999-2000" title="프리메라리가 1999-2000">1999-2000</a> 시즌에는 출장수가 급격하게 줄었으나, <a href="/wiki/%EB%B0%9C%EB%A0%8C%EC%8B%9C%EC%95%84_CF" title="발렌시아 CF">발렌시아 CF</a>와의 <a href="/wiki/2000_UEFA_%EC%B1%94%ED%94%BC%EC%96%B8%EC%8A%A4%EB%A6%AC%EA%B7%B8_%EA%B2%B0%EC%8A%B9%EC%A0%84" title="2000 UEFA 챔피언스리그 결승전">UEFA 챔피언스리그 결승전</a>에서 11분 출장하였고, 팀의 3-0 승리를 도왔다.</p>
<p>2001년, 산치스는 36세의 나이로 현역에서 은퇴하였고, <a href="/wiki/%EB%A0%88%EC%95%8C_%EB%A7%88%EB%93%9C%EB%A6%AC%EB%93%9C_CF" title="레알 마드리드 CF">레알 마드리드 CF</a> 소속으로 710경기에 출장하였다. (이 중 524 경기를 리그에서 출전하였다.) 그는 13년간 마드리드의 주장직을 맡기도 하였다.</p>
<h2><span class="mw-headline" id=".EA.B5.AD.EA.B0.80.EB.8C.80.ED.91.9C.ED.8C.80_.EA.B2.BD.EB.A0.A5">국가대표팀 경력</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%A7%88%EB%88%84%EC%97%98_%EC%82%B0%EC%B9%98%EC%8A%A4_%EC%98%A8%ED%8B%B0%EC%9C%A0%EC%97%98%EB%A1%9C&amp;action=edit&amp;section=2" title="부분 편집: 국가대표팀 경력">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>U-21 국가대표팀에서 두각을 나타낸 그는 UEFA U-21 챔피언쉽에서 우승을 거두었고, 같은 해에는 <a href="/wiki/%EC%8A%A4%ED%8E%98%EC%9D%B8_%EC%B6%95%EA%B5%AC_%EA%B5%AD%EA%B0%80%EB%8C%80%ED%91%9C%ED%8C%80" title="스페인 축구 국가대표팀">성인 국가대표팀</a>에 데뷔하기도 하였다. 11월 12일, <a href="/wiki/UEFA_%EC%9C%A0%EB%A1%9C_1988" title="UEFA 유로 1988">UEFA 유로 1988</a> 예선에서 그는 <a href="/wiki/%EB%A3%A8%EB%A7%88%EB%8B%88%EC%95%84_%EC%B6%95%EA%B5%AC_%EA%B5%AD%EA%B0%80%EB%8C%80%ED%91%9C%ED%8C%80" title="루마니아 축구 국가대표팀">루마니아</a>를 상대로 경기의 유일한 골을 뽑아내기도 하였다.</p>
<p>그는 국가대표팀 경기에 48번 풀타임 출장하였고, <a href="/wiki/UEFA_%EC%9C%A0%EB%A1%9C_1988" title="UEFA 유로 1988">UEFA 유로 1988</a>과 <a href="/wiki/1990%EB%85%84_FIFA_%EC%9B%94%EB%93%9C%EC%BB%B5" title="1990년 FIFA 월드컵">1990년 FIFA 월드컵</a>에 참가하였다. 그의 마지막 국가대표팀 경기는 1992년 3월 11일, <a href="/wiki/%EB%AF%B8%EA%B5%AD_%EC%B6%95%EA%B5%AC_%EA%B5%AD%EA%B0%80%EB%8C%80%ED%91%9C%ED%8C%80" title="미국 축구 국가대표팀">미국</a>전이었다.</p>
