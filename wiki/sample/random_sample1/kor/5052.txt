<p><b>마뉴엘 만프레트 프리드리히</b> (Manuel Manfred Friedrich, <a href="/wiki/1979%EB%85%84" title="1979년">1979년</a> <a href="/wiki/9%EC%9B%94_13%EC%9D%BC" title="9월 13일">9월 13일</a> <a class="new" href="/w/index.php?title=%EB%B0%94%ED%8A%B8_%ED%81%AC%EB%A1%9C%EC%9D%B4%EC%B8%A0%EB%82%98%ED%9D%90&amp;action=edit&amp;redlink=1" title="바트 크로이츠나흐 (없는 문서)">바트 크로이츠나흐</a> ~)는 <a href="/wiki/%EB%8F%85%EC%9D%BC" title="독일">독일</a>의 <a href="/wiki/%EC%B6%95%EA%B5%AC_%EC%84%A0%EC%88%98" title="축구 선수">축구 선수</a>로, <a href="/wiki/%EC%88%98%EB%B9%84%EC%88%98" title="수비수">센터백</a>이다. 현재 <a href="/wiki/%EC%9D%B8%EB%8F%84_%EC%8A%88%ED%8D%BC%EB%A6%AC%EA%B7%B8" title="인도 슈퍼리그">인도 슈퍼리그</a>의 <a href="/wiki/%EB%AD%84%EB%B0%94%EC%9D%B4_%EC%8B%9C%ED%8B%B0_FC" title="뭄바이 시티 FC">뭄바이 시티 FC</a>에서 뛰고 있다.</p>
<h2><span class="mw-headline" id=".ED.81.B4.EB.9F.BD_.EA.B2.BD.EB.A0.A5">클럽 경력</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%A7%88%EB%89%B4%EC%97%98_%ED%94%84%EB%A6%AC%EB%93%9C%EB%A6%AC%ED%9E%88&amp;action=edit&amp;section=1" title="부분 편집: 클럽 경력">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>프리드리히는 <a href="/wiki/1._FSV_%EB%A7%88%EC%9D%B8%EC%B8%A0_05" title="1. FSV 마인츠 05">1. FSV 마인츠 05</a>에서 프로 데뷔를 하기 전 SG 굴덴탈 07에서 축구를 시작하였다. <a href="/wiki/1995%EB%85%84" title="1995년">1995년</a> 프리드리히는 당시 2부리그에 위치한 <a href="/wiki/1._FSV_%EB%A7%88%EC%9D%B8%EC%B8%A0_05" title="1. FSV 마인츠 05">1. FSV 마인츠 05</a>에 합류하였다. 하지만, 팀 데뷔를 하기까지 <a href="/wiki/2000%EB%85%84" title="2000년">2000년</a> <a href="/wiki/2%EC%9B%94_26%EC%9D%BC" title="2월 26일">2월 26일</a> <a href="/wiki/FC_%EC%97%90%EB%84%A4%EB%A5%B4%EA%B8%B0_%EC%BD%A7%EB%B6%80%EC%8A%A4" title="FC 에네르기 콧부스">FC 에네르기 콧부스</a>전까지 기다려야 하였다.</p>
<p>다음 시즌, 프리드리히는 강력한 수비 옵션으로, 8골을 넣었다 (페널티는 없다). 후에 마인츠는 4위로 마감하였고, 1부리그 승격에 1점이 부족하였다.</p>
<p>이러한 활약은 1부리그의 팀들로부터 관심을 모았고, 프리드리히는 <a href="/wiki/SV_%EB%B2%A0%EB%A5%B4%EB%8D%94_%EB%B8%8C%EB%A0%88%EB%A9%98" title="SV 베르더 브레멘">SV 베르더 브레멘</a>과 계약하였다. 그러나, 2년간 단 1번밖에 출장하는 부진한 모습을 보였다. 또한 이 한번의 출장은 <a href="/wiki/2003%EB%85%84" title="2003년">2003년</a> <a href="/wiki/8%EC%9B%94_23%EC%9D%BC" title="8월 23일">8월 23일</a>, 4-1로 승리한 <a href="/wiki/FC_%EC%83%AC%EC%BC%80_04" title="FC 샬케 04">FC 샬케 04</a>와의 경기에서 최종 7분이었다. 그는 리저날리가 노르트에 속한 2군에서 13경기에 출장해 2골을 넣었지만, 2002-03 시즌을 인대 파열로 거의 결장하였다.</p>
<p><a href="/wiki/2004%EB%85%84" title="2004년">2004년</a> <a href="/wiki/1%EC%9B%94" title="1월">1월</a> 말, 프리드리히는 친정팀 <a href="/wiki/1._FSV_%EB%A7%88%EC%9D%B8%EC%B8%A0_05" title="1. FSV 마인츠 05">1. FSV 마인츠 05</a>로 복귀하였고, 승격을 위한 17점을 얻는데 공헌하였다. <a href="/wiki/5%EC%9B%94_23%EC%9D%BC" title="5월 23일">5월 23일</a>그는 SV 아인트라흐트 트리어 05와의 원정경기에서 골을 넣었고, 팀은 3-0으로 승리하여 시즌 최종 라운드에서 승격을 확정하였다 (골득실 차이로 3위). 다음 3시즌동안 대부분의 경기에 출장하여 4골을 넣었지만, 마인츠는 2007년에 강등되었다.</p>
<p><a href="/wiki/2007%EB%85%84" title="2007년">2007년</a> <a href="/wiki/7%EC%9B%94" title="7월">7월</a>, 그는 <a href="/wiki/%EB%B0%94%EC%9D%B4%EC%96%B4_04_%EB%A0%88%EB%B2%84%EC%BF%A0%EC%A0%A0" title="바이어 04 레버쿠젠">바이어 04 레버쿠젠</a>으로 이적, 수비로서 인상적인 모습을 보임과 동시에, 시즌 초반 <a href="/wiki/%EC%B9%B4%EB%A5%BC%EC%8A%A4%EB%A3%A8%EC%97%90_SC" title="카를스루에 SC">카를스루에 SC</a> (3-0) 과 <a href="/wiki/VfL_%EB%B3%B4%ED%9B%94" title="VfL 보훔">VfL 보훔</a> (2-0) 홈 경기 승리에 공헌하였다.</p>
<p><a href="/wiki/2013%EB%85%84" title="2013년">2013년</a> <a href="/wiki/11%EC%9B%94" title="11월">11월</a>, 그는 <a href="/wiki/%EB%B3%B4%EB%A3%A8%EC%8B%9C%EC%95%84_%EB%8F%84%EB%A5%B4%ED%8A%B8%EB%AC%B8%ED%8A%B8" title="보루시아 도르트문트">보루시아 도르트문트</a>으로 자유이적을 하였다.</p>
<p>그리고 1년 후에 새롭게 창설된 <a href="/wiki/%EC%9D%B8%EB%8F%84_%EC%8A%88%ED%8D%BC%EB%A6%AC%EA%B7%B8" title="인도 슈퍼리그">인도 슈퍼리그</a>의 <a href="/wiki/%EB%AD%84%EB%B0%94%EC%9D%B4_%EC%8B%9C%ED%8B%B0_FC" title="뭄바이 시티 FC">뭄바이 시티 FC</a>로 이적했다.</p>
<h2><span class="mw-headline" id=".EA.B5.AD.EA.B0.80.EB.8C.80.ED.91.9C_.EA.B2.BD.EB.A0.A5">국가대표 경력</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%A7%88%EB%89%B4%EC%97%98_%ED%94%84%EB%A6%AC%EB%93%9C%EB%A6%AC%ED%9E%88&amp;action=edit&amp;section=2" title="부분 편집: 국가대표 경력">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p><a href="/wiki/2006%EB%85%84" title="2006년">2006년</a> <a href="/wiki/3%EC%9B%94" title="3월">3월</a>, 프리드리히는 <a href="/wiki/%EB%AF%B8%EA%B5%AD_%EC%B6%95%EA%B5%AC_%EA%B5%AD%EA%B0%80%EB%8C%80%ED%91%9C%ED%8C%80" title="미국 축구 국가대표팀">미국</a>과의 친선경기에서 차출되었고, <a href="/wiki/1._FSV_%EB%A7%88%EC%9D%B8%EC%B8%A0_05" title="1. FSV 마인츠 05">1. FSV 마인츠 05</a> 선수로는 최초로 <a href="/wiki/%EB%8F%85%EC%9D%BC_%EC%B6%95%EA%B5%AC_%EA%B5%AD%EA%B0%80%EB%8C%80%ED%91%9C%ED%8C%80" title="독일 축구 국가대표팀">독일 국가대표</a>에 뛴 선수가 되었다. 하지만, 그는 경기에서 결장하였고 <a href="/wiki/2006%EB%85%84_FIFA_%EC%9B%94%EB%93%9C%EC%BB%B5" title="2006년 FIFA 월드컵">2006년 FIFA 월드컵</a> 23인 최종 엔트리에 들지 못하였다.</p>
<p><a href="/wiki/2006%EB%85%84" title="2006년">2006년</a> <a href="/wiki/8%EC%9B%94_16%EC%9D%BC" title="8월 16일">8월 16일</a>, 프리드리히는 3-0으로 승리한 <a href="/wiki/%EC%8A%A4%EC%9B%A8%EB%8D%B4_%EC%B6%95%EA%B5%AC_%EA%B5%AD%EA%B0%80%EB%8C%80%ED%91%9C%ED%8C%80" title="스웨덴 축구 국가대표팀">스웨덴</a>과의 친선경기에서 데뷔전을 치렀다. 그는 이 경기에서 후반전 45분을 뛰었다. 그는 <a href="/wiki/%EC%95%84%EC%9D%BC%EB%9E%9C%EB%93%9C_%EC%B6%95%EA%B5%AC_%EA%B5%AD%EA%B0%80%EB%8C%80%ED%91%9C%ED%8C%80" title="아일랜드 축구 국가대표팀">아일랜드</a>와 <a href="/wiki/%EC%82%B0%EB%A7%88%EB%A6%AC%EB%85%B8_%EC%B6%95%EA%B5%AC_%EA%B5%AD%EA%B0%80%EB%8C%80%ED%91%9C%ED%8C%80" title="산마리노 축구 국가대표팀">산마리노</a>와의 <a href="/wiki/UEFA_%EC%9C%A0%EB%A1%9C_2008" title="UEFA 유로 2008">UEFA 유로 2008</a>예선에 풀타임 출장하였고, 산마리노전에서는 팀의 12번째 골이 될 한골을 넣었다. 팀은 이 약체에 13-0으로 승리하였다.</p>
