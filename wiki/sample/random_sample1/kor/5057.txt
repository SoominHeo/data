<p><b>마다가스카르 공화국</b>(<span style="font-size: smaller;"><a href="/wiki/%EB%A7%90%EB%9D%BC%EA%B0%80%EC%8B%9C%EC%96%B4" title="말라가시어">말라가시어</a>:</span> <span lang="mg" xml:lang="mg">Repoblikan'i Madagasikara</span> <small>레푸블리캰 마다가스캬러</small>, <span style="font-size: smaller;"><a href="/wiki/%ED%94%84%EB%9E%91%EC%8A%A4%EC%96%B4" title="프랑스어">프랑스어</a>:</span> <span lang="fr" xml:lang="fr">République de Madagascar</span> <small>레퓌블리크 드 마다가스카르<sup>[<a href="/wiki/%EC%9C%84%ED%82%A4%EB%B0%B1%EA%B3%BC:%ED%94%84%EB%9E%91%EC%8A%A4%EC%96%B4%EC%9D%98_%ED%95%9C%EA%B8%80_%ED%91%9C%EA%B8%B0" title="위키백과:프랑스어의 한글 표기">*</a>]</sup></small>), 약칭 <b>마다가스카르</b>(<span style="font-size: smaller;"><a href="/wiki/%EB%AC%B8%ED%99%94%EC%96%B4" title="문화어">문화어</a>:</span> <b>마다가스까르</b>)는 <a href="/wiki/%EC%95%84%ED%94%84%EB%A6%AC%EC%B9%B4" title="아프리카">아프리카</a> 동쪽에 있는 <a class="mw-redirect" href="/wiki/%EA%B3%B5%ED%99%94%EA%B5%AD" title="공화국">공화국</a>이며 <a href="/wiki/%EC%84%AC%EB%82%98%EB%9D%BC" title="섬나라">섬나라</a>이다. 수도는 <a href="/wiki/%EC%95%88%ED%83%80%EB%82%98%EB%82%98%EB%A6%AC%EB%B3%B4" title="안타나나리보">안타나나리보</a>이며, 공용어는 <a href="/wiki/%EB%A7%90%EB%9D%BC%EA%B0%80%EC%8B%9C%EC%96%B4" title="말라가시어">말라가시어</a>와 <a href="/wiki/%ED%94%84%EB%9E%91%EC%8A%A4%EC%96%B4" title="프랑스어">프랑스어</a>이다. 인구는 약 2,200만 명이고, 주민은 <a href="/wiki/%EB%8F%99%EB%82%A8%EC%95%84%EC%8B%9C%EC%95%84" title="동남아시아">동남아시아</a>의 <a class="mw-redirect" href="/wiki/%EB%A7%90%EB%A0%88%EC%9D%B4%EC%A1%B1" title="말레이족">말레이족</a>과 가까운 말라가시인이 다수를 차지한다.</p>

<h2><span class="mw-headline" id=".EC.97.AD.EC.82.AC">역사</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%A7%88%EB%8B%A4%EA%B0%80%EC%8A%A4%EC%B9%B4%EB%A5%B4&amp;action=edit&amp;section=1" title="부분 편집: 역사">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>소규모 부족국가들이 난립하다가 18세기경 <a href="/wiki/%EB%A9%94%EB%A6%AC%EB%82%98_%EC%99%95%EA%B5%AD" title="메리나 왕국">메리나 왕국</a>을 중심으로 섬의 대부분이 통일 되었다. 그러나 프랑스의 침략을 받아 1897년 <a href="/wiki/%ED%94%84%EB%9E%91%EC%8A%A4" title="프랑스">프랑스</a>의 식민지로 전락하였다. <a href="/wiki/1957%EB%85%84" title="1957년">1957년</a>에 자치령 말라가시 공화국이 되었다가, <a href="/wiki/1960%EB%85%84" title="1960년">1960년</a> <a href="/wiki/6%EC%9B%94_26%EC%9D%BC" title="6월 26일">6월 26일</a>에 <a href="/wiki/%ED%94%84%EB%9E%91%EC%8A%A4" title="프랑스">프랑스</a>로부터 독립하였다. 1972년에 <a href="/wiki/%EC%82%AC%ED%9A%8C%EC%A3%BC%EC%9D%98" title="사회주의">사회주의</a> 정권이 들어선 후 <a href="/wiki/1975%EB%85%84" title="1975년">1975년</a>에 국명을 마다가스카르 민주 공화국으로 바꾸었으나, <a href="/wiki/1991%EB%85%84" title="1991년">1991년</a>에 경제 악화 등으로 민주화 요구가 일어나 <a href="/wiki/1992%EB%85%84" title="1992년">1992년</a>에 신헌법을 제정하고 국명을 마다가스카르 공화국으로 바꾸었다. <a href="/wiki/2007%EB%85%84" title="2007년">2007년</a>에 신헌법이 다시 제정되었다. <a href="/wiki/2014%EB%85%84" title="2014년">2014년</a> <a href="/wiki/1%EC%9B%94_25%EC%9D%BC" title="1월 25일">1월 25일</a>에 <a href="/wiki/%EC%95%88%EB%93%9C%EB%A6%AC_%EB%9D%BC%EC%A1%B0%EC%97%98%EB%A6%AC%EB%82%98" title="안드리 라조엘리나">안드리 라조엘리나</a>가 물러나고 <a class="new" href="/w/index.php?title=%ED%97%A4%EB%A6%AC_%EB%9D%BC%EC%9E%90%EC%98%A4%EB%82%98%EB%A6%AC%EB%A7%98%ED%94%BC%EC%95%84%EB%8B%88%EB%82%98&amp;action=edit&amp;redlink=1" title="헤리 라자오나리맘피아니나 (없는 문서)">헤리 라자오나리맘피아니나</a>가 취임했다.<sup class="reference" id="cite_ref-navernews_1-0"><a href="#cite_note-navernews-1">[1]</a></sup></p>
<h2><span class="mw-headline" id=".EC.A7.80.EB.A6.AC">지리</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%A7%88%EB%8B%A4%EA%B0%80%EC%8A%A4%EC%B9%B4%EB%A5%B4&amp;action=edit&amp;section=2" title="부분 편집: 지리">편집</a><span class="mw-editsection-bracket">]</span></span></h2>

<p>마다가스카르 섬은 <a href="/wiki/%EB%84%93%EC%9D%B4%EC%88%9C_%EC%84%AC_%EB%AA%A9%EB%A1%9D" title="넓이순 섬 목록">세계에서 4번째로 큰 섬</a>이다. 마다가스카르의 전체 면적은 587,041km²로 세계에서 46번째로 큰 나라이다. 동쪽으로는 가파른 절벽이 있어 중앙 고원이 있고 열대우림과 마주한다.</p>
<p>이 섬은 <a class="mw-redirect" href="/wiki/%EC%95%84%ED%94%84%EB%A6%AC%EC%B9%B4_%EB%8C%80%EB%A5%99" title="아프리카 대륙">아프리카 대륙</a>과 멀리 떨어져 있어 <a href="/wiki/%ED%98%B8%EB%9E%91%EC%9D%B4%EA%BC%AC%EB%A6%AC%EC%97%AC%EC%9A%B0%EC%9B%90%EC%88%AD%EC%9D%B4" title="호랑이꼬리여우원숭이">호랑이꼬리여우원숭이</a> 등 이 섬에서만 발견되는 독특한 동식물종이 많다.</p>
<h2><span class="mw-headline" id=".ED.96.89.EC.A0.95_.EA.B5.AC.EC.97.AD">행정 구역</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%A7%88%EB%8B%A4%EA%B0%80%EC%8A%A4%EC%B9%B4%EB%A5%B4&amp;action=edit&amp;section=3" title="부분 편집: 행정 구역">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<div class="rellink relarticle mainarticle"><img alt="" data-file-height="128" data-file-width="128" height="16" src="//upload.wikimedia.org/wikipedia/commons/thumb/e/ec/Crystal_Clear_app_xmag.svg/16px-Crystal_Clear_app_xmag.svg.png" srcset="//upload.wikimedia.org/wikipedia/commons/thumb/e/ec/Crystal_Clear_app_xmag.svg/24px-Crystal_Clear_app_xmag.svg.png 1.5x, //upload.wikimedia.org/wikipedia/commons/thumb/e/ec/Crystal_Clear_app_xmag.svg/32px-Crystal_Clear_app_xmag.svg.png 2x" width="16"/> 이 부분의 본문은 <a href="/wiki/%EB%A7%88%EB%8B%A4%EA%B0%80%EC%8A%A4%EC%B9%B4%EB%A5%B4%EC%9D%98_%ED%96%89%EC%A0%95_%EA%B5%AC%EC%97%AD" title="마다가스카르의 행정 구역">마다가스카르의 행정 구역</a>입니다.</div>
<p><a href="/wiki/%EC%88%98%EB%8F%84" title="수도">수도</a>는 <a href="/wiki/%EC%95%88%ED%83%80%EB%82%98%EB%82%98%EB%A6%AC%EB%B3%B4" title="안타나나리보">안타나나리보</a>이다. 줄여서 <b>타나</b>라고 부르기도 한다.</p>
<h2><span class="mw-headline" id=".EA.B8.B0.ED.9B.84">기후</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%A7%88%EB%8B%A4%EA%B0%80%EC%8A%A4%EC%B9%B4%EB%A5%B4&amp;action=edit&amp;section=4" title="부분 편집: 기후">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>마다가스카르의 북부는 <a href="/wiki/%EC%97%B4%EB%8C%80" title="열대">열대</a>, 중부 고원 지대는 <a href="/wiki/%EC%95%84%EC%97%B4%EB%8C%80" title="아열대">아열대</a> 및 <a href="/wiki/%EC%98%A8%EB%8C%80" title="온대">온대</a>, 남부는 약간의 사막성 기후와 온대로 이루어진다. 이 나라는 남반구에 있기 때문에 북반구와는 계절이 반대이다.</p>
<h2><span class="mw-headline" id=".EC.A3.BC.EB.AF.BC">주민</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%A7%88%EB%8B%A4%EA%B0%80%EC%8A%A4%EC%B9%B4%EB%A5%B4&amp;action=edit&amp;section=5" title="부분 편집: 주민">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>이 나라는 다른 <a href="/wiki/%EC%95%84%ED%94%84%EB%A6%AC%EC%B9%B4" title="아프리카">아프리카</a> 국가들과 달리 주민의 대다수가 <a href="/wiki/%EC%98%A4%EC%8A%A4%ED%8A%B8%EB%A1%9C%EB%84%A4%EC%8B%9C%EC%95%84%EC%96%B4%EC%A1%B1" title="오스트로네시아어족">오스트로네시아어족</a>의 언어를 쓰는 오스트로네시아어계이다. <a class="new" href="/w/index.php?title=%EB%A9%94%EB%A6%AC%EB%82%98%EC%A1%B1&amp;action=edit&amp;redlink=1" title="메리나족 (없는 문서)">메리나족</a>이 26%으로 다수를 차지하며, 그 밖에 <a class="new" href="/w/index.php?title=%EB%B2%A0%ED%8B%B0%EB%AF%B8%EC%82%AC%EB%9D%BC%EC%B9%B4%EC%A1%B1&amp;action=edit&amp;redlink=1" title="베티미사라카족 (없는 문서)">베티미사라카족</a>이 15%, <a class="new" href="/w/index.php?title=%EB%B2%A0%ED%8B%B8%EB%A0%88%EC%9A%B0%EC%A1%B1&amp;action=edit&amp;redlink=1" title="베틸레우족 (없는 문서)">베틸레우족</a>이 12%, <a class="new" href="/w/index.php?title=%ED%88%AC%EB%AF%B8%ED%97%A4%ED%8B%B0%EC%A1%B1&amp;action=edit&amp;redlink=1" title="투미헤티족 (없는 문서)">투미헤티족</a>이 7%를 차지한다.</p>
<h2><span class="mw-headline" id=".EC.96.B8.EC.96.B4">언어</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%A7%88%EB%8B%A4%EA%B0%80%EC%8A%A4%EC%B9%B4%EB%A5%B4&amp;action=edit&amp;section=6" title="부분 편집: 언어">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p><a href="/wiki/2010%EB%85%84" title="2010년">2010년</a> <a href="/wiki/11%EC%9B%94_17%EC%9D%BC" title="11월 17일">11월 17일</a>에 국민투표를 실시하여 확정된 새 헌법을 <a href="/wiki/12%EC%9B%94_11%EC%9D%BC" title="12월 11일">12월 11일</a>에 공포하였는데, 새 헌법은 제4조제1항에 "말라가시어는 국어이다."라고 규정하고 같은 조 제5항에 "공식어는 <a href="/wiki/%EB%A7%90%EB%9D%BC%EA%B0%80%EC%8B%9C%EC%96%B4" title="말라가시어">말라가시어</a>와 <a href="/wiki/%ED%94%84%EB%9E%91%EC%8A%A4%EC%96%B4" title="프랑스어">프랑스어</a>이다."라고 규정하였다. 이로써 <a href="/wiki/2007%EB%85%84" title="2007년">2007년</a> 헌법에서 도입한 <a href="/wiki/%EC%98%81%EC%96%B4" title="영어">영어</a>의 지위는 박탈되었다. 마다가스카르에서 영어는 실제 거의 쓰이지 않는 언어이다.</p>
<p>대다수 주민들은 다소간 방언의 차이가 있긴 하지만 공통의 <a href="/wiki/%EB%A7%90%EB%9D%BC%EA%B0%80%EC%8B%9C%EC%96%B4" title="말라가시어">말라가시어</a>를 사용한다. 마다가스카르는 <a href="/wiki/%EB%B6%80%EB%A3%AC%EB%94%94" title="부룬디">부룬디</a>, <a href="/wiki/%EB%A5%B4%EC%99%84%EB%8B%A4" title="르완다">르완다</a>, <a href="/wiki/%EC%86%8C%EB%A7%90%EB%A6%AC%EC%95%84" title="소말리아">소말리아</a> 등과 같이 사실상 단일어를 사용하는 아프리카 국가이다. <a href="/wiki/%ED%94%84%EB%9E%91%EC%8A%A4%EC%96%B4" title="프랑스어">프랑스어</a>는 국가의 준공용어이며, 이는 <a href="/wiki/%ED%94%84%EB%9E%91%EC%8A%A4" title="프랑스">프랑스</a>의 식민통치의 영향 때문이다. 말라가시어는 아프리카에서 유일하게 <a href="/wiki/%EC%98%A4%EC%8A%A4%ED%8A%B8%EB%A1%9C%EB%84%A4%EC%8B%9C%EC%95%84%EC%96%B4%EC%A1%B1" title="오스트로네시아어족">오스트로네시아어족</a>에 속하는 언어이다. <a href="/wiki/%EC%98%81%EC%96%B4" title="영어">영어</a>, <a href="/wiki/%ED%8F%AC%EB%A5%B4%ED%88%AC%EA%B0%88%EC%96%B4" title="포르투갈어">포르투갈어</a>, <a href="/wiki/%EC%BD%94%EB%AA%A8%EB%A1%9C%EC%96%B4" title="코모로어">코모로어</a>, <a href="/wiki/%EC%95%84%EB%9E%8D%EC%96%B4" title="아랍어">아랍어</a>를 사용하는 주민들도 극소수있다.</p>
<h2><span class="mw-headline" id=".EC.A2.85.EA.B5.90">종교</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%A7%88%EB%8B%A4%EA%B0%80%EC%8A%A4%EC%B9%B4%EB%A5%B4&amp;action=edit&amp;section=7" title="부분 편집: 종교">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p><a href="/wiki/%EC%95%A0%EB%8B%88%EB%AF%B8%EC%A6%98" title="애니미즘">애니미즘</a> 등의 토착 종교가 절반을 차지하고, <a href="/wiki/%EA%B8%B0%EB%8F%85%EA%B5%90" title="기독교">기독교</a>가 41%, <a href="/wiki/%EC%9D%B4%EC%8A%AC%EB%9E%8C%EA%B5%90" title="이슬람교">이슬람교</a>가 7%를 점하고 있다.</p>
<h2><span class="mw-headline" id=".EB.AC.B8.ED.99.94">문화</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%A7%88%EB%8B%A4%EA%B0%80%EC%8A%A4%EC%B9%B4%EB%A5%B4&amp;action=edit&amp;section=8" title="부분 편집: 문화">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>이 나라는 문화적으로 아프리카 대륙의 다른 국가들과 이질적인 면이 많다. 특히, 언어나 생활풍습 등 여러 면에서 <a href="/wiki/%EC%9D%B8%EB%8F%84%EB%84%A4%EC%8B%9C%EC%95%84" title="인도네시아">인도네시아</a>, <a href="/wiki/%EB%A7%90%EB%A0%88%EC%9D%B4%EC%8B%9C%EC%95%84" title="말레이시아">말레이시아</a> 등 <a href="/wiki/%EB%8F%99%EB%82%A8%EC%95%84%EC%8B%9C%EC%95%84" title="동남아시아">동남아시아</a> 국가들과 비슷해 이들의 조상은 <a href="/wiki/%EC%9D%B8%EB%8F%84%EB%84%A4%EC%8B%9C%EC%95%84" title="인도네시아">인도네시아</a>에서 넘어온 <a class="mw-redirect" href="/wiki/%EB%A7%90%EB%A0%88%EC%9D%B4%EC%A1%B1" title="말레이족">말레이족</a>으로 추정되고 있다.</p>
<h2><span class="mw-headline" id=".EC.82.B0.EC.97.85">산업</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%A7%88%EB%8B%A4%EA%B0%80%EC%8A%A4%EC%B9%B4%EB%A5%B4&amp;action=edit&amp;section=9" title="부분 편집: 산업">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>농업이 중심 산업이고, 수출품으로는 바닐라, 쌀, 설탕이 있다. 전 세계 바닐라 생산의 60%를 생산하고 있다. 광물 자원은 적고 개발도 늦은 편이다.</p>
<h2><span class="mw-headline" id=".EB.8C.80.EC.99.B8_.EA.B4.80.EA.B3.84">대외 관계</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%A7%88%EB%8B%A4%EA%B0%80%EC%8A%A4%EC%B9%B4%EB%A5%B4&amp;action=edit&amp;section=10" title="부분 편집: 대외 관계">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<h3><span class="mw-headline" id=".EB.8C.80.ED.95.9C.EB.AF.BC.EA.B5.AD.EA.B3.BC.EC.9D.98_.EA.B4.80.EA.B3.84">대한민국과의 관계</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%A7%88%EB%8B%A4%EA%B0%80%EC%8A%A4%EC%B9%B4%EB%A5%B4&amp;action=edit&amp;section=11" title="부분 편집: 대한민국과의 관계">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>마다가스카르는 <a href="/wiki/1962%EB%85%84" title="1962년">1962년</a>에 <a href="/wiki/%EB%8C%80%ED%95%9C%EB%AF%BC%EA%B5%AD" title="대한민국">대한민국</a>과 수교하였으나, <a href="/wiki/1972%EB%85%84" title="1972년">1972년</a>에 10월 유신 정권이 수립되면서 같은 해에 <a href="/wiki/%EC%A1%B0%EC%84%A0%EB%AF%BC%EC%A3%BC%EC%A3%BC%EC%9D%98%EC%9D%B8%EB%AF%BC%EA%B3%B5%ED%99%94%EA%B5%AD" title="조선민주주의인민공화국">조선민주주의인민공화국</a>과 외교관계를 수립하고 당시 마다가스카르 외무장관이 평양을 방문하면서 단교하였다. 사회주의 정권 아래의 마다가스카르는 <a class="mw-redirect" href="/wiki/%EC%84%9C%EC%9A%B8" title="서울">서울</a>에서 열린 <a href="/wiki/1988%EB%85%84_%ED%95%98%EA%B3%84_%EC%98%AC%EB%A6%BC%ED%94%BD" title="1988년 하계 올림픽">1988년 하계 올림픽</a>에 불참하였다. 그러나, 소련 붕괴와 냉전 종식의 영향으로 마다가스카르에도 사회주의 독재 정권이 끝나면서 <a href="/wiki/1993%EB%85%84" title="1993년">1993년</a>에 <a href="/wiki/%EB%8C%80%ED%95%9C%EB%AF%BC%EA%B5%AD" title="대한민국">대한민국</a>과 다시 수교하였다. 마다가스카르와 한국 모두 양국에는 대사관이 없으므로 한국 측은 <a href="/wiki/%EB%82%A8%EC%95%84%ED%94%84%EB%A6%AC%EC%B9%B4_%EA%B3%B5%ED%99%94%EA%B5%AD" title="남아프리카 공화국">남아프리카 공화국</a>의 <a href="/wiki/%ED%94%84%EB%A6%AC%ED%86%A0%EB%A6%AC%EC%95%84" title="프리토리아">프리토리아</a>에서, 마다가스카르 측은 <a href="/wiki/%EC%9D%BC%EB%B3%B8" title="일본">일본</a>의 <a href="/wiki/%EB%8F%84%EC%BF%84_%EB%8F%84" title="도쿄 도">도쿄 도</a>에서 각각 상주공관을 겸임하고 있으나, 마다가스카르에는 조만간 한국 대사관이 개관하게 된다. 신설시 <a href="/wiki/%EC%BD%94%EB%AA%A8%EB%A1%9C" title="코모로">코모로</a>, <a href="/wiki/%EC%84%B8%EC%9D%B4%EC%85%B8" title="세이셸">세이셸</a>, <a href="/wiki/%EB%AA%A8%EB%A6%AC%EC%85%94%EC%8A%A4" title="모리셔스">모리셔스</a>를 겸임국으로 배정시킬 것으로 예상된다.</p>
<h3><span class="mw-headline" id=".ED.94.84.EB.9E.91.EC.8A.A4.EC.99.80.EC.9D.98_.EA.B4.80.EA.B3.84">프랑스와의 관계</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%A7%88%EB%8B%A4%EA%B0%80%EC%8A%A4%EC%B9%B4%EB%A5%B4&amp;action=edit&amp;section=12" title="부분 편집: 프랑스와의 관계">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p><a href="/wiki/%ED%94%84%EB%9E%91%EC%8A%A4" title="프랑스">프랑스</a>는 마다가스카르를 식민 통치했기 때문에 마다가스카르 국민들은 프랑스에 대한 감정이 좋지 않으나, 최근에는 조금씩 관계를 회복하고 있다.</p>
