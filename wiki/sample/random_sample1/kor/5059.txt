<p><b>마다바</b>(<span style="font-size: smaller;"><a href="/wiki/%EC%95%84%EB%9E%8D%EC%96%B4" title="아랍어">아랍어</a>:</span> <span lang="ar" xml:lang="ar">مادبا‎</span>)는 <a href="/wiki/%EC%9A%94%EB%A5%B4%EB%8B%A8" title="요르단">요르단</a> <a href="/wiki/%EB%A7%88%EB%8B%A4%EB%B0%94_%EC%A3%BC" title="마다바 주">마다바 주</a>의 주도로, 인구는 약 60,000명이며 요르단에서 다섯 번째로 인구가 많은 도시이다. <a href="/wiki/%EC%95%94%EB%A7%8C" title="암만">암만</a>(요르단의 수도)에서 남서쪽으로 30km 정도 떨어진 곳에 위치한다. <a class="mw-redirect" href="/wiki/%EB%B9%84%EC%9E%94%ED%8B%B4_%EC%A0%9C%EA%B5%AD" title="비잔틴 제국">비잔틴 제국</a>과 <a href="/wiki/%EC%9A%B0%EB%A7%88%EC%9D%B4%EC%95%BC_%EC%99%95%EC%A1%B0" title="우마이야 왕조">우마이야 왕조</a> 시대에 제작된 <a href="/wiki/%EB%AA%A8%EC%9E%90%EC%9D%B4%ED%81%AC" title="모자이크">모자이크</a> 장식으로 유명한 도시이며 특히 모자이크로 만든 성지 지도가 유명하다.</p>
<p>마다바는 요르단의 도시 들 중에서는 <a href="/wiki/%EA%B8%B0%EB%8F%85%EA%B5%90" title="기독교">기독교</a> 인구가 가장 많은 도시로, 시가지 곳곳에 <a href="/wiki/%EA%B8%B0%EB%8F%85%EA%B5%90" title="기독교">기독교</a> 각 종파의 <a href="/wiki/%EA%B5%90%ED%9A%8C" title="교회">교회</a>나 <a href="/wiki/%EC%84%B1%EB%8B%B9" title="성당">성당</a>이 있다. 유명한 여행지로는 <a class="mw-redirect" href="/wiki/%EC%84%B1_%EC%A1%B0%EC%A7%80" title="성 조지">성 조지</a> <a href="/wiki/%EA%B7%B8%EB%A6%AC%EC%8A%A4_%EC%A0%95%EA%B5%90%ED%9A%8C" title="그리스 정교회">그리스 정교회</a> 성당, <a href="/wiki/%EC%84%B8%EB%A1%80%EC%9E%90_%EC%9A%94%ED%95%9C" title="세례자 요한">세례자 요한</a> <a class="mw-redirect" href="/wiki/%EB%A1%9C%EB%A7%88_%EA%B0%80%ED%86%A8%EB%A6%AD" title="로마 가톨릭">로마 가톨릭</a> 성당, 마다바 고고학 공원, 마다바 고고학 박물관 등이 있다.</p>
