<p><b><a class="mw-redirect" href="/wiki/%EC%84%B1_%EC%82%BD%EB%B0%94%EC%8A%A4" title="성 삽바스">의로운 사람 성 삽바스</a>의 대 라브라</b>,<sup class="reference" id="cite_ref-1"><a href="#cite_note-1">[1]</a></sup> <a href="/wiki/%EC%95%84%EB%9E%8D%EC%96%B4" title="아랍어">아랍어</a>로 <b>마르 사바</b>(<span style="font-size: smaller;"><a href="/wiki/%EC%95%84%EB%9E%8D%EC%96%B4" title="아랍어">아랍어</a>:</span> <span lang="ar" xml:lang="ar">دير مار سابا</span>; <span style="font-size: smaller;"><a href="/wiki/%ED%9E%88%EB%B8%8C%EB%A6%AC%EC%96%B4" title="히브리어">히브리어</a>:</span> <span lang="he" xml:lang="he">מנזר מר סבא</span>; <span style="font-size: smaller;"><a href="/wiki/%EA%B7%B8%EB%A6%AC%EC%8A%A4%EC%96%B4" title="그리스어">그리스어</a>:</span> <span lang="el" xml:lang="el">Λαύρα Σάββα τοῦ Ἡγιασμένου</span>)는 <a href="/wiki/%EC%9A%94%EB%A5%B4%EB%8B%A8_%EA%B0%95_%EC%84%9C%EC%95%88_%EC%A7%80%EA%B5%AC" title="요르단 강 서안 지구">요르단 강 서안 지구</a> <a href="/wiki/%EB%B2%A0%EB%93%A4%EB%A0%88%ED%97%B4" title="베들레헴">베들레헴</a> 동부에 있는 <a href="/wiki/%ED%82%A4%EB%93%9C%EB%A1%A0_%EA%B3%84%EA%B3%A1" title="키드론 계곡">키드론 계곡</a>에서 내려다 보이는<sup class="reference" id="cite_ref-2"><a href="#cite_note-2">[2]</a></sup> <a href="/wiki/%EB%8F%99%EB%B0%A9_%EC%A0%95%EA%B5%90%ED%9A%8C" title="동방 정교회">동방 정교회</a> <a href="/wiki/%EC%88%98%EB%8F%84%EC%9B%90" title="수도원">수도원</a>이다. <a href="/wiki/%EC%B9%B4%ED%8C%8C%EB%8F%84%ED%82%A4%EC%95%84" title="카파도키아">카파도키아</a>의 <a class="mw-redirect" href="/wiki/%EC%84%B1_%EC%82%BD%EB%B0%94%EC%8A%A4" title="성 삽바스">성 삽바스</a>에 의한 이 수도원의 설립연도는 <a href="/wiki/483" title="483">483</a>년<sup class="reference" id="cite_ref-3"><a href="#cite_note-3">[3]</a></sup> 이며 오늘날 약 20 명의 수사들이 거주하고 있다. 이 수도원은 세계에서 가장 오래 됐으면서도 사람이 살고 있는 수도원 가운데 하나로 여겨지며, 여전히 고대 전통의 많은 부분이 유지되고 있다. 그 특징 중에 하나는 여성의 주요 구내 입장의 금지이다. 여성이 출입할 수 있는 유일한 건물은 정문 근처에 있는 여성의 탑이다.</p>

<p>이 수도원은 <a class="mw-redirect" href="/wiki/%EC%84%B1_%EC%82%BD%EB%B0%94%EC%8A%A4" title="성 삽바스">의인 성 삽바스</a>의 <a class="new" href="/w/index.php?title=%EC%84%B1%EC%9C%A0%EB%AC%BC&amp;action=edit&amp;redlink=1" title="성유물 (없는 문서)">성유물</a>을 보유하고 있다. 마르 사바는 때때로 산타 삽바 <a href="/wiki/%EC%88%98%EB%8F%84%EC%9B%90" title="수도원">수도원</a>으로 불린다.<sup class="reference" id="cite_ref-4"><a href="#cite_note-4">[4]</a></sup></p>
<p>마르 사바는 726년에 <a class="mw-redirect" href="/wiki/%EC%84%B1%ED%99%94%EC%83%81" title="성화상">성화상</a>의 숭상을 금지한 비잔티움 황제 <a href="/wiki/%EB%A0%88%EC%98%A4_3%EC%84%B8_(%EB%B9%84%EC%9E%94%ED%8B%B0%EC%9B%80_%EC%A0%9C%EA%B5%AD)" title="레오 3세 (비잔티움 제국)">레온 3세 이사우리아인</a>의 칙령에 대하여 그에게 서신들을 보내어 <a class="new" href="/w/index.php?title=%EC%84%B1%ED%99%94%EC%83%81%EB%85%BC%EC%9F%81&amp;action=edit&amp;redlink=1" title="성화상논쟁 (없는 문서)">성화상논쟁</a>에서 핵심 역할을 했던 수사 <a href="/wiki/%EC%9A%94%ED%95%9C_(%EB%8B%A4%EB%A7%88%EC%8A%A4%EC%BF%A0%EC%8A%A4)" title="요한 (다마스쿠스)">다마스쿠스의 성 요한</a>의 본거지이기도 했다. 요한은 <a href="/wiki/%EC%B9%BC%EB%A6%AC%ED%8C%8C" title="칼리파">칼리파</a> 무슬림 <a class="new" href="/w/index.php?title=%EC%95%95%EB%93%9C_%EC%95%8C-%EB%A7%90%EB%A6%AC%ED%81%AC_%EC%9D%B4%EB%B8%90_%EB%A7%88%EB%A5%B4%EC%99%84&amp;action=edit&amp;redlink=1" title="압드 알-말리크 이븐 마르완 (없는 문서)">압드 알-말리크 이븐 마르완</a>의 고위 재무 관리자로 일했다. 결국 그는 숭고한 부름을 들어 <a class="new" href="/w/index.php?title=%EC%9C%A0%EB%8C%80_%EC%82%AC%EB%A7%89&amp;action=edit&amp;redlink=1" title="유대 사막 (없는 문서)">유대 사막</a>으로 이주했으며 마르 사바에서 <a class="new" href="/w/index.php?title=%EC%88%98%EB%8F%84_%EC%82%AC%EC%A0%9C&amp;action=edit&amp;redlink=1" title="수도 사제 (없는 문서)">수도 사제</a>로 <a href="/wiki/%EC%95%88%EC%88%98%EB%A1%80" title="안수례">서품</a> 받았다. 성 요한의 무덤이 이 수도원 아래의 한 동굴에 있다.</p>
<p>이 수도원은 <a class="new" href="/w/index.php?title=%EB%B9%84%EC%9E%94%ED%8B%B0%EC%9B%80_%EC%A0%84%EB%A1%80&amp;action=edit&amp;redlink=1" title="비잔티움 전례 (없는 문서)">비잔티움 전례</a>를 따르는 <a href="/wiki/%EB%8F%99%EB%B0%A9_%EC%A0%95%EA%B5%90%ED%9A%8C" title="동방 정교회">동방 정교회</a>와 <a href="/wiki/%EB%8F%99%EB%B0%A9_%EA%B0%80%ED%86%A8%EB%A6%AD%EA%B5%90%ED%9A%8C" title="동방 가톨릭교회">동방 가톨릭교회</a> 전역에서 표준이 된 성 삽바스의 수도원 <a class="new" href="/w/index.php?title=%ED%8B%B0%ED%94%BC%EC%BD%98&amp;action=edit&amp;redlink=1" title="티피콘 (없는 문서)">티피콘</a>(성무 의식 예절)에서 <a class="mw-redirect" href="/wiki/%EC%A0%95%EA%B5%90%ED%9A%8C" title="정교회">정교회</a> <a class="mw-disambig" href="/wiki/%EC%98%88%EC%A0%84" title="예전">전례</a>의 역사적인 발전에 있어서 중요한 곳이다. 그 <a class="new" href="/w/index.php?title=%EB%B4%89%EC%82%AC%EB%A1%80&amp;action=edit&amp;redlink=1" title="봉사례 (없는 문서)">봉사례</a>는 <a href="/wiki/%EC%98%88%EB%A3%A8%EC%82%B4%EB%A0%98_%EC%A0%95%EA%B5%90%ED%9A%8C_%EC%B4%9D%EB%8C%80%EC%A3%BC%EA%B5%90" title="예루살렘 정교회 총대주교">예루살렘 총대주교</a>에 의해 행해지는 성무의 표준 양식으로 채택되었고 성 삽바스의 봉사례에 지역적인 전통의 몇몇 특정 수도 예절들이 덧붙혀졌다. 그 봉사례는 그 곳에서부터 <a href="/wiki/%EC%BD%98%EC%8A%A4%ED%83%84%ED%8B%B0%EB%85%B8%ED%8F%B4%EB%A6%AC%EC%8A%A4" title="콘스탄티노폴리스">콘스탄티노폴리스</a>로 전파되었고 거기서 <a href="/wiki/%EB%B9%84%EC%9E%94%ED%8B%B0%EC%9B%80_%EC%A0%9C%EA%B5%AD" title="비잔티움 제국">비잔티움</a> 세계의 전역으로 퍼져나갔다. 그 티피콘은 특히 콘스탄티노폴리스에 있는 <a class="new" href="/w/index.php?title=%EC%8A%A4%ED%88%AC%EB%94%94%EC%98%A4%EC%8A%A4_%EC%88%98%EB%8F%84%EC%9B%90&amp;action=edit&amp;redlink=1" title="스투디오스 수도원 (없는 문서)">스투디온</a> 수도원에서 추가적인 발달이 있었고, 여전히 <i>성 삽바스의 티피콘</i>이라고 일컬어진다.</p>
<p>마르 사바는 <a class="new" href="/w/index.php?title=%EB%AA%A8%ED%8A%BC_%EC%8A%A4%EB%AF%B8%EC%8A%A4&amp;action=edit&amp;redlink=1" title="모튼 스미스 (없는 문서)">모튼 스미스</a>가 이른바 <a class="new" href="/w/index.php?title=%EB%A7%88%EB%A5%B4%EC%BD%94%EC%9D%98_%EB%B9%84%EB%B0%80_%EB%B3%B5%EC%9D%8C%EC%84%9C&amp;action=edit&amp;redlink=1" title="마르코의 비밀 복음서 (없는 문서)">마르코의 비밀 복음서</a>의 발췌 부분들이 포함된 <a href="/wiki/%EC%95%8C%EB%A0%89%EC%82%B0%EB%93%9C%EB%A6%AC%EC%95%84%EC%9D%98_%ED%81%B4%EB%A0%88%EB%A9%98%EC%8A%A4" title="알렉산드리아의 클레멘스">알렉산드리아의 클레멘스</a>의 한 <a class="new" href="/w/index.php?title=%EB%A7%88%EB%A5%B4_%EC%82%AC%EB%B0%94_%ED%8E%B8%EC%A7%80&amp;action=edit&amp;redlink=1" title="마르 사바 편지 (없는 문서)">서신</a>의 사본을 발견했다고 주장했던 곳이다.</p>
