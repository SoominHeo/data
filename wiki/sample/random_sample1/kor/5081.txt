<p><b>마르시우 아모호주 두스 산투스</b> (Márcio Amoroso dos Santos, <a href="/wiki/1974%EB%85%84" title="1974년">1974년</a> <a href="/wiki/7%EC%9B%94_5%EC%9D%BC" title="7월 5일">7월 5일</a>, <a href="/wiki/%EB%B8%8C%EB%9D%BC%EC%A7%88%EB%A6%AC%EC%95%84" title="브라질리아">브라질리아</a> ~) 는 은퇴한 <a href="/wiki/%EB%B8%8C%EB%9D%BC%EC%A7%88" title="브라질">브라질</a>의 <a href="/wiki/%EC%B6%95%EA%B5%AC" title="축구">축구</a> 선수이다. 그는 일본, 이탈리아, 스페인, 그리스 등의 축구 리그에서 활동하였고, <a href="/wiki/%EB%B8%8C%EB%9D%BC%EC%A7%88_%EC%B6%95%EA%B5%AC_%EA%B5%AD%EA%B0%80%EB%8C%80%ED%91%9C%ED%8C%80" title="브라질 축구 국가대표팀">브라질 국가대표팀</a> 차출경험이 있다. 그는 전성기 시절, 탁월한 드리블링 능력과 골 결정력을 가지고 있었다.<sup class="reference" id="cite_ref-1"><a href="#cite_note-1">[1]</a></sup></p>

<h2><span class="mw-headline" id=".ED.81.B4.EB.9F.BD_.EA.B2.BD.EB.A0.A5">클럽 경력</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%A7%88%EB%A5%B4%EC%8B%9C%EC%9A%B0_%EC%95%84%EB%AA%A8%ED%98%B8%EC%A3%BC&amp;action=edit&amp;section=1" title="부분 편집: 클럽 경력">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>아모호주는 고향 클럽인 <a href="/wiki/%EA%B5%AC%EC%95%84%EB%9D%BC%EB%8B%88_FC" title="구아라니 FC">구아라니 FC</a>에서 1992년에 데뷔하였다. 1992년 7월, 그는 일본 <a class="mw-redirect" href="/wiki/J%EB%A6%AC%EA%B7%B8_%EB%94%94%EB%B9%84%EC%A0%84_1" title="J리그 디비전 1">J리그 디비전 1</a> 클럽 <a href="/wiki/%EB%8F%84%EC%BF%84_%EB%B2%A0%EB%A5%B4%EB%94%94" title="도쿄 베르디">베르디 가와사키</a>로 임대되었다. 2년 뒤 구아라니 FC로 복귀하였다. 1996년, 그는 <a href="/wiki/CR_%ED%94%8C%EB%9D%BC%EB%A9%A9%EA%B5%AC" title="CR 플라멩구">CR 플라멩구</a>로 이적하였으나, 이후 <a href="/wiki/%EC%9D%B4%ED%83%88%EB%A6%AC%EC%95%84" title="이탈리아">이탈리아</a> <a href="/wiki/%EC%84%B8%EB%A6%AC%EC%97%90_A" title="세리에 A">세리에 A</a>에서 1990년대 당시 평범했던 <a href="/wiki/%EC%9A%B0%EB%94%94%EB%84%A4%EC%84%B8_%EC%B9%BC%EC%B4%88" title="우디네세 칼초">우디네세 칼초</a>로 이적하였다. 그곳에서 그는 <a href="/wiki/%EC%98%AC%EB%A6%AC%EB%B2%84_%EB%B9%84%EC%96%B4%ED%98%B8%ED%94%84" title="올리버 비어호프">올리버 비어호프</a>와 주전으로 모험적인 3-4-3 포메이션의 쓰리톱들 중 한명으로 출전하였다. 비어호프가 <a href="/wiki/AC_%EB%B0%80%EB%9E%80" title="AC 밀란">AC 밀란</a>으로 이적한 후, 다수는 우디네세 칼초가 성공을 되풀이 할 수 없을 것이라고 생각하였으나, 바로 다음 시즌, 아모호주는 팀의 핵심으로 활동하여 세리에 A 득점왕 타이틀을 가져갔다. 이후 그는 <a class="mw-redirect" href="/wiki/%ED%8C%8C%EB%A5%B4%EB%A7%88_FC" title="파르마 FC">파르마 FC</a>로 거액에 이적하였다. 파르마 FC는 리그 타이틀을 획득할 가능성을 전혀 주지 못하였고, 그에 따라 아모호주는 또다시 이적하였다. 이번에 그는 독일의 <a href="/wiki/%EB%B3%B4%EB%A3%A8%EC%8B%9C%EC%95%84_%EB%8F%84%EB%A5%B4%ED%8A%B8%EB%AC%B8%ED%8A%B8" title="보루시아 도르트문트">보루시아 도르트문트</a>로 이적하였고, 이 팀은 2001-02 시즌에 <a href="/wiki/%ED%91%B8%EC%8A%A4%EB%B0%9C-%EB%B6%84%EB%8D%B0%EC%8A%A4%EB%A6%AC%EA%B0%80" title="푸스발-분데스리가">분데스리가</a>를 우승함과 동시에 자신은 득점왕에 이름을 올렸다. 그는 팀을 <a href="/wiki/UEFA%EC%BB%B5_2001-02" title="UEFA컵 2001-02">UEFA컵 결승전</a>에 올리는데 공을 세웠고, 그는 <a href="/wiki/%ED%8E%98%EC%98%88%EB%85%B8%EB%A5%B4%ED%8A%B8" title="페예노르트">페예노르트</a>에게 페널티골로 2-3으로 지는 것을 막지 못하였다. 아모호주는 2004-05 시즌에 <a href="/wiki/%EB%A7%90%EB%9D%BC%EA%B0%80_CF" title="말라가 CF">말라가 CF</a>에서 활동하였다.</p>
<p>2005년 여름, 아모호주는 <a href="/wiki/%EC%83%81%ED%8C%8C%EC%9A%B8%EB%A3%A8_FC" title="상파울루 FC">상파울루 FC</a>로 이적하여 즉시 팀을 남미 최고 권위 대회인 <a href="/wiki/%EC%BD%94%ED%8C%8C_%EB%A6%AC%EB%B2%A0%EB%A5%B4%ED%83%80%EB%8F%84%EB%A0%88%EC%8A%A4" title="코파 리베르타도레스">코파 리베르타도레스</a>에서 우승하는데 큰 공을 세웠다. 2006년 1월, <a href="/wiki/FIFA_%ED%81%B4%EB%9F%BD_%EC%9B%94%EB%93%9C%EC%BB%B5" title="FIFA 클럽 월드컵">FIFA 클럽 월드컵</a>에서 우승이라는 팀 타이틀과 대회 득점왕이라는 개인 타이틀을 획득한 뒤, 이탈리아로 복귀하였다. 그는 <a href="/wiki/AC_%EB%B0%80%EB%9E%80" title="AC 밀란">AC 밀란</a>과 18개월의 계약을 체결하였고, <a href="/wiki/AS_%EB%AA%A8%EB%82%98%EC%BD%94_FC" title="AS 모나코 FC">AS 모나코 FC</a>로 이적한 <a href="/wiki/%ED%81%AC%EB%A6%AC%EC%8A%A4%ED%8B%B0%EC%95%88_%EB%B9%84%EC%97%90%EB%A6%AC" title="크리스티안 비에리">크리스티안 비에리</a>를 대체하게 되었다.</p>
<p>그러나 아모호주는 2006년 9월 1일에 AC 밀란과의 계약을 해지하였고, 즉시 <a href="/wiki/SC_%EC%BD%94%EB%A6%B0%EC%B9%98%EC%95%99%EC%8A%A4_%ED%8C%8C%EC%9A%B8%EB%A6%AC%EC%8A%A4%ED%83%80" title="SC 코린치앙스 파울리스타">SC 코린치앙스 파울리스타</a>와 계약을 체결하였다. 아모호주는 등번호 10번을 부여 받아 <a class="mw-redirect" href="/wiki/%EC%9B%A8%EC%8A%A4%ED%8A%B8_%ED%96%84_%EC%9C%A0%EB%82%98%EC%9D%B4%ED%8B%B0%EB%93%9C_FC" title="웨스트 햄 유나이티드 FC">웨스트 햄 유나이티드 FC</a>로 떠난 <a href="/wiki/%EC%B9%B4%EB%A5%BC%EB%A1%9C%EC%8A%A4_%ED%85%8C%EB%B2%A0%EC%8A%A4" title="카를로스 테베스">카를로스 테베스</a>를 대체하였다. 그러나 그곳에서 그는 기량을 만개하지 못하였고, 2007년 4월에 계약을 해지한 뒤, <a href="/wiki/%EA%B7%B8%EB%A0%88%EB%AF%B8%EC%9A%B0" title="그레미우">그레미우</a>와 계약하였다. 8월을 기점으로, 아모호주는 기량 하락으로 출전이 불가능하게 되자 그레미우 계약을 해지하였다. 2008년 1월, 그는 <a href="/wiki/%EC%95%84%EB%A6%AC%EC%8A%A4_FC" title="아리스 FC">아리스 FC</a>와 1년 반 계약을 체결하였다. 그러나 그는 6개월밖에 채우지 못하였다. 2008년 12월 19일, 아모호주는 2008-09 시즌을 위해 과라니로 복귀하였다.<sup class="reference" id="cite_ref-guarani2009_2-0"><a href="#cite_note-guarani2009-2">[2]</a></sup></p>
<h2><span class="mw-headline" id=".EA.B0.9C.EC.9D.B8">개인</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%A7%88%EB%A5%B4%EC%8B%9C%EC%9A%B0_%EC%95%84%EB%AA%A8%ED%98%B8%EC%A3%BC&amp;action=edit&amp;section=2" title="부분 편집: 개인">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>아리스 테살로니키는 아모호주가 6개국에서 거친 12번째 클럽이었다. 그는 20개의 트로피와 개인상을 받았는데, 이중 하나는 브라질 국가대표에서 활약하며 획득한 <a href="/wiki/%EC%BD%94%ED%8C%8C_%EC%95%84%EB%A9%94%EB%A6%AC%EC%B9%B4" title="코파 아메리카">코파 아메리카</a> 타이틀이고, 둘은 상 파울루에서 활약하며 획득한 코파 리베르타도레스와 FIFA 클럽 월드컵 타이틀이다. 그는 또한 베르디 가와사키, 플라멩구, 우디네세, 파르마, 보루시아 도르트문트, 말라가, 밀란, 코린치앙스, 그레미우, 그리고 과라니에서 활동하였다.</p>
<p>아모호주는 3개국에서 득점왕을 하였고, 50M DM으로 보루시아 도르트문트에 2001년 여름 이적하여 분데스리가 최대 이적료를 기록하기도 한다. 파르마는 이보다 2년 전에 실제로 €2M 이상의 비용을 들여 아모호주와 계약을 체결하였다.</p>
