<p><b>마르코 마테라치</b> (<span style="font-size: smaller;"><a href="/wiki/%EC%9D%B4%ED%83%88%EB%A6%AC%EC%95%84%EC%96%B4" title="이탈리아어">이탈리아어</a>:</span> <span lang="it" xml:lang="it">Marco Materazzi</span>, <a href="/wiki/%EC%9D%B4%ED%83%88%EB%A6%AC%EC%95%84_%EA%B3%B5%ED%99%94%EA%B5%AD_%EA%B3%B5%EB%A1%9C_%ED%9B%88%EC%9E%A5" title="이탈리아 공화국 공로 훈장">Ufficiale OMRI</a>, <a href="/wiki/1973%EB%85%84" title="1973년">1973년</a> <a href="/wiki/8%EC%9B%94_19%EC%9D%BC" title="8월 19일">8월 19일</a>, <a href="/wiki/%EC%9D%B4%ED%83%88%EB%A6%AC%EC%95%84" title="이탈리아">이탈리아</a> <a href="/wiki/%EB%A0%88%EC%B2%B4" title="레체">레체</a> ~ )는 <a href="/wiki/%EC%9D%B4%ED%83%88%EB%A6%AC%EC%95%84" title="이탈리아">이탈리아</a>의 <a href="/wiki/%EC%B6%95%EA%B5%AC_%EC%84%A0%EC%88%98" title="축구 선수">축구 선수</a>다. 현재 인도의 <a href="/wiki/%EC%B2%B8%EB%82%98%EC%9D%B4_FC" title="첸나이 FC">첸나이 FC</a>의 플레잉 코치로 활약하고 있다. 장신을 바탕으로 거친 수비를 펼치는 것으로 유명하다.</p>

<h3><span class="mw-headline" id=".ED.81.B4.EB.A0.B5_.EA.B2.BD.EB.A0.A5">클렵 경력</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%A7%88%EB%A5%B4%EC%BD%94_%EB%A7%88%ED%85%8C%EB%9D%BC%EC%B9%98&amp;action=edit&amp;section=2" title="부분 편집: 클렵 경력">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>마테라치는 <a href="/wiki/1990%EB%85%84" title="1990년">1990년</a> <a href="/wiki/ACR_%EB%A9%94%EC%8B%9C%EB%82%98" title="ACR 메시나">메시나</a>와 아마추어 팀인 토르 디 퀸토에서 유스 생활을 경험하였고, <a class="new" href="/w/index.php?title=%EC%84%B8%EB%A6%AC%EC%97%90_C2&amp;action=edit&amp;redlink=1" title="세리에 C2 (없는 문서)">세리에 C2</a>의 마르살라와 <a class="mw-redirect" href="/wiki/%EC%84%B8%EB%A6%AC%EC%97%90_C1" title="세리에 C1">세리에 C1</a>인 <a class="new" href="/w/index.php?title=ASD_%ED%8A%B8%EB%9D%BC%ED%8C%8C%EB%8B%88_%EC%B9%BC%EC%B4%88&amp;action=edit&amp;redlink=1" title="ASD 트라파니 칼초 (없는 문서)">트라파니</a>에서 선수 생활을 보내는 등 오랫동안 무명 선수로 지냈다. 이후 <a href="/wiki/1995%EB%85%84" title="1995년">1995년</a> <a href="/wiki/%EC%84%B8%EB%A6%AC%EC%97%90_B" title="세리에 B">세리에 B</a>의 <a class="mw-redirect" href="/wiki/%ED%8E%98%EB%A3%A8%EC%9E%90_%EC%B9%BC%EC%B4%88" title="페루자 칼초">페루자</a>에 입단하였으나, <a href="/wiki/1996%EB%85%84" title="1996년">1996년</a> 세리에 C의 <a href="/wiki/%EC%B9%B4%EB%A5%B4%ED%94%BC_FC_1909" title="카르피 FC 1909">카르피</a>로 임대를 떠나기도 하였다. 1998-99 시즌을 앞두고 그는 <a class="mw-redirect" href="/wiki/%EC%9E%89%EA%B8%80%EB%9E%9C%EB%93%9C_%ED%94%84%EB%A6%AC%EB%AF%B8%EC%96%B4%EB%A6%AC%EA%B7%B8" title="잉글랜드 프리미어리그">잉글랜드 프리미어리그</a>의 <a href="/wiki/%EC%97%90%EB%B2%84%ED%84%B4_FC" title="에버턴 FC">에버턴</a>으로 이적하여 <a href="/wiki/%EB%AF%B8%EB%93%A4%EC%A6%88%EB%B8%8C%EB%9F%AC_FC" title="미들즈브러 FC">미들즈브러</a>를 상대로 골을 터뜨리기도 하였으나, 불과 27경기에서 4번의 퇴장을 당하기도 하였다. 그는 1999-00 시즌 친정팀인 <a class="mw-redirect" href="/wiki/%ED%8E%98%EB%A3%A8%EC%9E%90_%EC%B9%BC%EC%B4%88" title="페루자 칼초">페루자</a>로 복귀하였고, 2000-01 시즌 12골을 기록하여 <a href="/wiki/%EB%8B%A4%EB%8B%88%EC%97%98_%ED%8C%8C%EC%82%AC%EB%A0%88%EC%95%BC" title="다니엘 파사레야">다니엘 파사레야</a>를 제치고 <a href="/wiki/%EC%84%B8%EB%A6%AC%EC%97%90_A" title="세리에 A">세리에 A</a> 역사상 수비수가 한 시즌당 기록한 최다골을 경신하였다. <a href="/wiki/2001%EB%85%84" title="2001년">2001년</a> 그는 1000만 유로에 <a href="/wiki/FC_%EC%9D%B8%ED%85%8C%EB%A5%B4%EB%82%98%EC%B9%98%EC%98%A4%EB%82%A0%EB%A0%88_%EB%B0%80%EB%9D%BC%EB%85%B8" title="FC 인테르나치오날레 밀라노">인테르나치오날레</a>으로 이적하였고, 2006-07 시즌 10골을 기록하여 그 시즌에 활약한 수비수 중 최다골을 기록하였다. 그리고 2011년에 그는 더 이상 인테르와 계약을 갱신하지 않아 자유 계약 선수가 되었다. 그 뒤로 다른 팀과의 계약설이 돌았으나 그는 결국 다른 팀과 계약하지 않으면서 은퇴하였다.</p>
<h3><span class="mw-headline" id=".EA.B5.AD.EA.B0.80.EB.8C.80.ED.91.9C.ED.8C.80_.EA.B2.BD.EB.A0.A5">국가대표팀 경력</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%A7%88%EB%A5%B4%EC%BD%94_%EB%A7%88%ED%85%8C%EB%9D%BC%EC%B9%98&amp;action=edit&amp;section=3" title="부분 편집: 국가대표팀 경력">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>마테라치는 <a href="/wiki/2006%EB%85%84_FIFA_%EC%9B%94%EB%93%9C%EC%BB%B5" title="2006년 FIFA 월드컵">2006년 FIFA 월드컵</a>에 <a href="/wiki/%EC%9D%B4%ED%83%88%EB%A6%AC%EC%95%84_%EC%B6%95%EA%B5%AC_%EA%B5%AD%EA%B0%80%EB%8C%80%ED%91%9C%ED%8C%80" title="이탈리아 축구 국가대표팀">이탈리아 축구 국가대표팀</a>에 선발되었고, 팀이 우승하는 데 크게 기여하였다. 그 공헌으로 2006년 12월에 이탈리아 공화국 훈장 4등급 (Ufficiale OMRI)을 받았다. 그는 원래 주전 선수는 아니었으나 <a href="/wiki/%EC%95%8C%EB%A0%88%EC%82%B0%EB%93%9C%EB%A1%9C_%EB%84%A4%EC%8A%A4%ED%83%80" title="알레산드로 네스타">알레산드로 네스타</a>가 <a href="/wiki/2006%EB%85%84" title="2006년">2006년</a> <a href="/wiki/6%EC%9B%94_22%EC%9D%BC" title="6월 22일">6월 22일</a> <a href="/wiki/%EC%B2%B4%EC%BD%94_%EC%B6%95%EA%B5%AC_%EA%B5%AD%EA%B0%80%EB%8C%80%ED%91%9C%ED%8C%80" title="체코 축구 국가대표팀">체코</a>와의 경기에서 부상을 당하면서 대신 투입되어 그 경기에서 결승골을 넣으면서 공식 맨 오브 더 매치에 선정되는 등 맹활약을 하였다.</p>
<p><a href="/wiki/%EC%98%A4%EC%8A%A4%ED%8A%B8%EB%A0%88%EC%9D%BC%EB%A6%AC%EC%95%84_%EC%B6%95%EA%B5%AC_%EA%B5%AD%EA%B0%80%EB%8C%80%ED%91%9C%ED%8C%80" title="오스트레일리아 축구 국가대표팀">오스트레일리아</a>와의 16강전에서 퇴장을 당하여 <a href="/wiki/%EC%9A%B0%ED%81%AC%EB%9D%BC%EC%9D%B4%EB%82%98_%EC%B6%95%EA%B5%AC_%EA%B5%AD%EA%B0%80%EB%8C%80%ED%91%9C%ED%8C%80" title="우크라이나 축구 국가대표팀">우크라이나</a>와의 8강전에 출장하지 못하였으나, <a href="/wiki/%EB%8F%85%EC%9D%BC_%EC%B6%95%EA%B5%AC_%EA%B5%AD%EA%B0%80%EB%8C%80%ED%91%9C%ED%8C%80" title="독일 축구 국가대표팀">독일</a>과의 준결승전부터 다시 중용되었다. 결승전에서는 <a href="/wiki/%ED%94%84%EB%9E%91%EC%8A%A4_%EC%B6%95%EA%B5%AC_%EA%B5%AD%EA%B0%80%EB%8C%80%ED%91%9C%ED%8C%80" title="프랑스 축구 국가대표팀">프랑스</a>를 상대로 동점골을 터뜨렸다. 연장전에서는 프랑스의 <a href="/wiki/%EC%A7%80%EB%84%A4%EB%94%98_%EC%A7%80%EB%8B%A8" title="지네딘 지단">지네딘 지단</a>이 그의 가슴을 머리로 들이받는 행동으로 인해 퇴장당하기도 했다. 득점 없이 연장전이 끝난 후에는 <a href="/wiki/%EC%8A%B9%EB%B6%80%EC%B0%A8%EA%B8%B0" title="승부차기">승부차기</a>도 성공하였다.</p>
<p>그는 <a href="/wiki/UEFA_%EC%9C%A0%EB%A1%9C_2008" title="UEFA 유로 2008">UEFA 유로 2008</a> 당시 <a href="/wiki/%EB%84%A4%EB%8D%9C%EB%9E%80%EB%93%9C_%EC%B6%95%EA%B5%AC_%EA%B5%AD%EA%B0%80%EB%8C%80%ED%91%9C%ED%8C%80" title="네덜란드 축구 국가대표팀">네덜란드</a>와의 첫 경기에 선발 출전하였지만 54분 교체되었고, 이후 다시 출전하지 못하였다.</p>
