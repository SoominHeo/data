<h1 class="firstHeading" id="firstHeading" lang="ko">마르크스주의 사회학</h1>
<div class="mw-body-content" id="bodyContent">
<div id="siteSub">위키백과, 우리 모두의 백과사전.</div>
<div id="contentSub"></div>
<div class="mw-jump" id="jump-to-nav">
					이동:					<a href="#mw-head">둘러보기</a>, 					<a href="#p-search">검색</a>
</div>
<div class="mw-content-ltr" dir="ltr" id="mw-content-text" lang="ko"><script>function mfTempOpenSection(id){var block=document.getElementById("mf-section-"+id);block.className+=" open-block";block.previousSibling.className+=" open-block";}</script><p><b>마르크스주의 사회학</b>은 <a href="/wiki/%EB%A7%88%EB%A5%B4%ED%81%AC%EC%8A%A4%EC%A3%BC%EC%9D%98" title="마르크스주의">마르크스주의</a>의 관점을 기반으로 하는 <a href="/wiki/%EC%82%AC%ED%9A%8C%ED%95%99" title="사회학">사회학</a>의 한 분파이다.<sup class="reference" id="cite_ref-Allan_1-0"><a href="#cite_note-Allan-1">[1]</a></sup> 마르크스주의 자체도 <a class="mw-redirect" href="/wiki/%EC%A0%95%EC%B9%98_%EC%B2%A0%ED%95%99" title="정치 철학">정치 철학</a>이면서 동시에 사회학으로도 인정되고 있다. 사회학으로서 마르크스 주의는 다른 표준적, 규범적 분파에 비해 보다 과학적, 구조적, 객관적인 연구를 하고자 한다. 마르크스주의 사회학은 “자본주의 사회에서 혁명적 <a href="/wiki/%EB%85%B8%EB%8F%99%EC%9E%90" title="노동자">노동</a> <a href="/wiki/%EC%82%AC%ED%9A%8C_%EA%B3%84%EA%B8%89" title="사회 계급">계급</a>을 동원하기 위해 마르크스주의의 <a class="mw-redirect" href="/wiki/%EA%B2%BD%ED%97%98%EC%A3%BC%EC%9D%98" title="경험주의">경험주의</a>와 <a class="new" href="/w/index.php?title=%EC%8B%A4%EC%A6%9D%EC%A3%BC%EC%9D%98_(%EC%82%AC%ED%9A%8C%ED%95%99)&amp;action=edit&amp;redlink=1" title="실증주의 (사회학) (없는 문서)">실증주의</a>적인 방법을 발전시킨 <a href="/wiki/%EA%B0%88%EB%93%B1_%EC%9D%B4%EB%A1%A0" title="갈등 이론">갈등 이론</a>”으로 정의할 수 있다.<sup class="reference" id="cite_ref-EOS_2-0"><a href="#cite_note-EOS-2">[2]</a></sup> <a class="new" href="/w/index.php?title=%EB%AF%B8%EA%B5%AD_%EC%82%AC%ED%9A%8C%ED%95%99_%ED%95%99%ED%9A%8C&amp;action=edit&amp;redlink=1" title="미국 사회학 학회 (없는 문서)">미국 사회학 학회</a>의 마르크스주의 사회학 분과는 자신들의 연구 분야에 대해 "마르크스주의 방법론과 분석법을 활용하여 복잡하고 역동적인 현대 사회를 설명하고자 한다"고 밝히고 있다.<sup class="reference" id="cite_ref-ASA_3-0"><a href="#cite_note-ASA-3">[3]</a></sup> 마르크스주의 사회학은 <a href="/wiki/%EB%B9%84%ED%8C%90_%EC%9D%B4%EB%A1%A0" title="비판 이론">비판 이론</a>과 <a class="new" href="/w/index.php?title=%EB%AC%B8%ED%99%94%ED%95%99&amp;action=edit&amp;redlink=1" title="문화학 (없는 문서)">문화학</a>의 발전을 촉진하였다.</p>

<h2><span class="mw-headline" id=".EA.B0.9C.EB.85.90.EA.B3.BC_.EC.A3.BC.EC.A0.9C">개념과 주제</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%A7%88%EB%A5%B4%ED%81%AC%EC%8A%A4%EC%A3%BC%EC%9D%98_%EC%82%AC%ED%9A%8C%ED%95%99&amp;action=edit&amp;section=1" title="부분 편집: 개념과 주제">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>마르크스주의 사회학의 핵심 <a href="/wiki/%EA%B0%9C%EB%85%90" title="개념">개념</a>은 <a href="/wiki/%EC%9C%A0%EB%AC%BC%EC%82%AC%EA%B4%80" title="유물사관">역사적 유물론</a>, <a class="new" href="/w/index.php?title=%EC%83%9D%EC%82%B0_%EC%96%91%EC%8B%9D&amp;action=edit&amp;redlink=1" title="생산 양식 (없는 문서)">생산 양식</a>, <a href="/wiki/%EC%9E%90%EB%B3%B8" title="자본">자본</a>과 <a href="/wiki/%EB%85%B8%EB%8F%99" title="노동">노동</a>의 관계와 같은 것이 있다. 마르크스주의 사회학은 경제와 사회의 관계를 중요하게 생각하지만, 그것만 다루는 것은 아니다.<sup class="reference" id="cite_ref-ASA_3-1"><a href="#cite_note-ASA-3">[3]</a></sup> 마르크스주의 사회학의 핵심 질문은 다음과 같다.<sup class="reference" id="cite_ref-Allan_1-1"><a href="#cite_note-Allan-1">[1]</a></sup></p>
<ul>
<li>자본은 노동자를 어떻게 통제하는가?</li>
<li>생산양식은 사회 계급에 어떻게 영향을 주는가?</li>
<li>노동자, 자본, <a href="/wiki/%EA%B5%AD%EA%B0%80" title="국가">국가</a>, <a href="/wiki/%EB%AC%B8%ED%99%94" title="문화">문화</a>는 어떠한 관계에 놓여 있는가?</li>
<li>경제 지표는 <a class="mw-redirect" href="/wiki/%EC%A0%A0%EB%8D%94" title="젠더">사회적 성별</a>과 <a href="/wiki/%EC%9D%B8%EC%A2%85" title="인종">인종</a>과 같은 하위 문화를 포함하여 사회에 어떻게 불평등을 초래하는가?</li>
</ul>
<p>마르크스주의 사회학은 <a class="new" href="/w/index.php?title=%EC%82%AC%ED%9A%8C%ED%95%99_%EC%9D%B4%EB%A1%A0&amp;action=edit&amp;redlink=1" title="사회학 이론 (없는 문서)">사회학 이론</a> 분야에서 <a href="/wiki/%EA%B0%88%EB%93%B1_%EC%9D%B4%EB%A1%A0" title="갈등 이론">갈등 이론</a>과 <a href="/wiki/%EB%B9%84%ED%8C%90_%EC%9D%B4%EB%A1%A0" title="비판 이론">비판 이론</a>이 결합된 주요 <a class="new" href="/w/index.php?title=%EC%82%AC%ED%9A%8C%ED%95%99_%ED%8C%A8%EB%9F%AC%EB%8B%A4%EC%9E%84&amp;action=edit&amp;redlink=1" title="사회학 패러다임 (없는 문서)">사회학 패러다임</a>의 하나로 여겨지고 있다. 또한, 마르크스주의와 연관되어 있기 때문에 <a class="new" href="/w/index.php?title=%EA%B3%84%EA%B8%89_%ED%98%81%EB%AA%85&amp;action=edit&amp;redlink=1" title="계급 혁명 (없는 문서)">계급 혁명</a>의 달성과 일정 정도 관련을 갖고 있기도 하다.<sup class="reference" id="cite_ref-Allan_1-2"><a href="#cite_note-Allan-1">[1]</a></sup> 한편, <a class="new" href="/w/index.php?title=%EB%A7%88%EB%A5%B4%ED%81%AC%EC%8A%A4%EC%A3%BC%EC%9D%98_%EC%B2%A0%ED%95%99&amp;action=edit&amp;redlink=1" title="마르크스주의 철학 (없는 문서)">마르크스주의 철학</a>이 주로 관념 철학에 대해 비판하는 것을 목표로 하는 것에 비해, 마르크스주의 사회학은 <a href="/wiki/%EC%A0%95%EC%B9%98%EA%B2%BD%EC%A0%9C%ED%95%99" title="정치경제학">정치경제학</a>적 현상의 객관적 기술을 목표로 한다.<sup class="reference" id="cite_ref-EOS_2-1"><a href="#cite_note-EOS-2">[2]</a></sup></p>
<p><a class="new" href="/w/index.php?title=%EA%B2%BD%EC%A0%9C%EC%82%AC%ED%9A%8C%ED%95%99&amp;action=edit&amp;redlink=1" title="경제사회학 (없는 문서)">경제사회학</a> 역시 마르크스주의 사회학과 같이 경제 현상과 사회의 관계를 탐구하지만, 마르크스주의 사회학은 <a href="/wiki/%EB%A7%88%EB%A5%B4%ED%81%AC%EC%8A%A4%EC%A3%BC%EC%9D%98" title="마르크스주의">마르크스주의</a>의 관점을 기반으로 한다는 차이점이 있다.</p>
<h2><span class="mw-headline" id=".EC.97.AD.EC.82.AC">역사</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%A7%88%EB%A5%B4%ED%81%AC%EC%8A%A4%EC%A3%BC%EC%9D%98_%EC%82%AC%ED%9A%8C%ED%95%99&amp;action=edit&amp;section=2" title="부분 편집: 역사">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>마르크스주의 경제학은 19 세기 말부터 20 세기 초 사이에 <a href="/wiki/%EC%B9%B4%EB%A5%BC_%EB%A7%88%EB%A5%B4%ED%81%AC%EC%8A%A4" title="카를 마르크스">카를 마르크스</a>의 영향을 받아 형성되었다.<sup class="reference" id="cite_ref-Allan_1-3"><a href="#cite_note-Allan-1">[1]</a></sup><sup class="reference" id="cite_ref-EOS_2-2"><a href="#cite_note-EOS-2">[2]</a></sup><sup class="reference" id="cite_ref-Bottomore_4-0"><a href="#cite_note-Bottomore-4">[4]</a></sup> 마르크스는 <a href="/wiki/%EB%A7%89%EC%8A%A4_%EB%B2%A0%EB%B2%84" title="막스 베버">막스 베버</a>, <a href="/wiki/%EC%97%90%EB%B0%80_%EB%92%A4%EB%A5%B4%EC%BC%90" title="에밀 뒤르켐">에밀 뒤르켐</a> 등과 함께 <a class="new" href="/w/index.php?title=%EC%82%AC%ED%9A%8C%ED%95%99%EC%9D%98_%EC%97%AD%EC%82%AC&amp;action=edit&amp;redlink=1" title="사회학의 역사 (없는 문서)">초기 사회학</a>의 각 분야에 두루 영향을 준 사상가이다.<sup class="reference" id="cite_ref-Bottomore_4-1"><a href="#cite_note-Bottomore-4">[4]</a></sup> 초기 마르크스주의 학파는 <a class="new" href="/w/index.php?title=%EC%98%A4%EC%8A%A4%ED%8A%B8%EB%A6%AC%EC%95%84_%EB%A7%88%EB%A5%B4%ED%81%AC%EC%8A%A4%EC%A3%BC%EC%9D%98&amp;action=edit&amp;redlink=1" title="오스트리아 마르크스주의 (없는 문서)">오스트리아 마르크스주의</a>로 불렸는데 <a class="new" href="/w/index.php?title=%EC%B9%B4%EB%A5%BC_%EA%B7%B8%EB%A4%BC%EB%B2%A0%EB%A5%B4%ED%81%AC&amp;action=edit&amp;redlink=1" title="카를 그뤼베르크 (없는 문서)">카를 그뤼베르크</a>, <a class="new" href="/w/index.php?title=%EC%95%88%ED%86%A0%EB%8B%88%EC%98%A4_%EB%9D%BC%EB%B8%8C%EB%A6%AC%EC%98%AC%EB%9D%BC&amp;action=edit&amp;redlink=1" title="안토니오 라브리올라 (없는 문서)">안토니오 라브리올라</a>와 같은 학자들이 대표적이다.<sup class="reference" id="cite_ref-Bottomore_4-2"><a href="#cite_note-Bottomore-4">[4]</a></sup> 초기 마르크스주의 사회학은 학계 밖에서 발전되었고, 종종 “<a class="mw-redirect" href="/wiki/%EB%B6%80%EB%A5%B4%EC%A3%BC%EC%95%84" title="부르주아">부르주아</a> 학파”에 대해 대적한다고 표방하기도 하였다.<sup class="reference" id="cite_ref-Bottomore_4-3"><a href="#cite_note-Bottomore-4">[4]</a></sup></p>
<p>그러나, <a href="/wiki/%EB%9F%AC%EC%8B%9C%EC%95%84_%ED%98%81%EB%AA%85" title="러시아 혁명">러시아 혁명</a> 이후 <a href="/wiki/%EC%86%8C%EB%A0%A8" title="소련">소련</a>은 사회학 자체를 부르주아의 학문으로 여겨 연구를 억압하였고 마르크스주의 사회학 역시 이러한 이유로 공산주의 국가 내에서 탄압을 받았다.<sup class="reference" id="cite_ref-Bottomore_4-4"><a href="#cite_note-Bottomore-4">[4]</a></sup> 이러한 탄압이 수 십년 지속된 뒤 마르크스주의 사회학이 다시 시작되었지만, 소련을 중심으로한 역사적 유물론의 <a class="mw-disambig" href="/wiki/%EB%8F%84%EA%B7%B8%EB%A7%88" title="도그마">도그마</a>를 옹호하는 <a class="new" href="/w/index.php?title=%EC%86%8C%EB%B9%84%EC%97%90%ED%8A%B8_%EB%A7%88%EB%A5%B4%ED%81%AC%EC%8A%A4%EC%A3%BC%EC%9D%98&amp;action=edit&amp;redlink=1" title="소비에트 마르크스주의 (없는 문서)">소비에트 마르크스주의</a> 학파와, 소련에 대해 독립적인 <a href="/wiki/%EC%84%9C%EC%96%91" title="서양">서방 세계</a>의 마르크스주의 연구자들을 중심으로 한 학파로 양분되었다.<sup class="reference" id="cite_ref-Allan_1-4"><a href="#cite_note-Allan-1">[1]</a></sup><sup class="reference" id="cite_ref-EOS_2-3"><a href="#cite_note-EOS-2">[2]</a></sup> 1940년대부터 시작된 <a class="new" href="/w/index.php?title=%EC%84%9C%EB%B0%A9_%EB%A7%88%EB%A5%B4%ED%81%AC%EC%8A%A4%EC%A3%BC%EC%9D%98&amp;action=edit&amp;redlink=1" title="서방 마르크스주의 (없는 문서)">서방 마르크스주의</a>는 점차 서구 학계의 일원이 되어 <a href="/wiki/%ED%94%84%EB%9E%91%ED%81%AC%ED%91%B8%EB%A5%B4%ED%8A%B8_%ED%95%99%ED%8C%8C" title="프랑크푸르트 학파">프랑크푸르트 학파</a>의 <a href="/wiki/%EB%B9%84%ED%8C%90_%EC%9D%B4%EB%A1%A0" title="비판 이론">비판 이론</a>과 같이 사회학의 분파로서 인정받게 되었다.<sup class="reference" id="cite_ref-Allan_1-5"><a href="#cite_note-Allan-1">[1]</a></sup></p>
<p><a href="/wiki/%ED%83%88%EA%B3%B5%EC%82%B0%EC%A3%BC%EC%9D%98" title="탈공산주의">탈공산주의</a> 이후 옛 공산주의 국가에서는 마르크스주의 사회학을 배격하는 성향이 두드러지게 나타나고 있는데, 대표적인 사례로는 <a class="new" href="/w/index.php?title=%ED%8F%B4%EB%9E%80%EB%93%9C%EC%9D%98_%EC%82%AC%ED%9A%8C%ED%95%99&amp;action=edit&amp;redlink=1" title="폴란드의 사회학 (없는 문서)">폴란드의 사회학</a> 동향을 들 수 있다. 그러나, <a class="new" href="/w/index.php?title=%EC%A4%91%EA%B5%AD%EC%9D%98_%EC%82%AC%ED%9A%8C%ED%95%99&amp;action=edit&amp;redlink=1" title="중국의 사회학 (없는 문서)">중국의 사회학</a>과 같이 공산주의가 유지되고 있는 나라에서 마르크스주의 사회학은 여전히 주류 학파로서 유지되고 있다.</p>
