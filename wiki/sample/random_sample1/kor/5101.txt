<p><b>마르틴 막스</b> (Martin Max, <a href="/wiki/1968%EB%85%84" title="1968년">1968년</a> <a href="/wiki/8%EC%9B%94_7%EC%9D%BC" title="8월 7일">8월 7일</a>, <a href="/wiki/%ED%83%80%EB%A5%B4%EB%85%B8%ED%94%84%EC%8A%A4%ED%82%A4%EC%97%90%EA%B5%AC%EB%A6%AC" title="타르노프스키에구리">타르노프스키에구리</a> ~) 는 은퇴한 <a href="/wiki/%ED%8F%B4%EB%9E%80%EB%93%9C" title="폴란드">폴란드</a> 태생의 <a href="/wiki/%EB%8F%85%EC%9D%BC" title="독일">독일</a>인 <a href="/wiki/%EC%B6%95%EA%B5%AC" title="축구">축구</a> 선수로, 현역 시절 <a href="/wiki/%EA%B3%B5%EA%B2%A9%EC%88%98" title="공격수">스트라이커</a>로 활동하였다.</p>
<p><a href="/wiki/%ED%91%B8%EC%8A%A4%EB%B0%9C-%EB%B6%84%EB%8D%B0%EC%8A%A4%EB%A6%AC%EA%B0%80" title="푸스발-분데스리가">푸스발-분데스리가</a>의 최고령 득점왕들 중 하나로, 이 업적을 각각 32세와 34세, 두 차례 달성하였고, 독일의 4개 클럽에서 활동하였다.</p>

<h2><span class="mw-headline" id=".ED.81.B4.EB.9F.BD_.EA.B2.BD.EB.A0.A5">클럽 경력</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%A7%88%EB%A5%B4%ED%8B%B4_%EB%A7%89%EC%8A%A4&amp;action=edit&amp;section=1" title="부분 편집: 클럽 경력">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p><a class="mw-redirect" href="/wiki/%EC%8B%A4%EB%A1%B1%EC%8A%A4%ED%81%AC" title="실롱스크">실롱스크</a>, <a href="/wiki/%ED%83%80%EB%A5%B4%EB%85%B8%ED%94%84%EC%8A%A4%ED%82%A4%EC%97%90%EA%B5%AC%EB%A6%AC" title="타르노프스키에구리">타르노프스키에구리</a> 출신으로, 막스는 로드워 구르니키 비톰에서 축구계에 입문하였고, 이후 청소년기에 <a href="/wiki/%EB%8F%85%EC%9D%BC" title="독일">독일</a>로 이주하였다. <a href="/wiki/%ED%91%B8%EC%8A%A4%EB%B0%9C-%EB%B6%84%EB%8D%B0%EC%8A%A4%EB%A6%AC%EA%B0%80" title="푸스발-분데스리가">분데스리가</a>에서, 그는 <a href="/wiki/%EB%B3%B4%EB%A3%A8%EC%8B%9C%EC%95%84_%EB%AC%80%ED%97%A8%EA%B8%80%EB%9D%BC%ED%8A%B8%EB%B0%94%ED%9D%90" title="보루시아 묀헨글라트바흐">보루시아 묀헨글라트바흐</a> 선수로 데뷔하였고, <a href="/wiki/%ED%91%B8%EC%8A%A4%EB%B0%9C-%EB%B6%84%EB%8D%B0%EC%8A%A4%EB%A6%AC%EA%B0%80_1994-95" title="푸스발-분데스리가 1994-95">이 곳에서의 마지막 시즌</a>에 20경기 출전 무득점을 기록하였다. 같은 해, <a href="/wiki/FC_%EC%83%AC%EC%BC%80_04" title="FC 샬케 04">FC 샬케 04</a>로 이적하였다.</p>
<p>샬케에서, 막스는 기량이 향상되었다. 처음 두 시즌간 그는 23골을 득점하였고, <a href="/wiki/UEFA_%EC%9C%A0%EB%A1%9C%ED%8C%8C%EB%A6%AC%EA%B7%B8" title="UEFA 유로파리그">UEFA컵</a> <a href="/wiki/UEFA%EC%BB%B5_1996-97" title="UEFA컵 1996-97">결승전</a>에서 <a href="/wiki/%EC%8A%B9%EB%B6%80%EC%B0%A8%EA%B8%B0" title="승부차기">승부차기</a> 끝에 <a href="/wiki/FC_%EC%9D%B8%ED%85%8C%EB%A5%B4%EB%82%98%EC%B9%98%EC%98%A4%EB%82%A0%EB%A0%88_%EB%B0%80%EB%9D%BC%EB%85%B8" title="FC 인테르나치오날레 밀라노">FC 인테르나치오날레 밀라노</a>를 꺾고 우승하였다.</p>
<p>그는 1999년에 <a href="/wiki/TSV_1860_%EB%AE%8C%ED%97%A8" title="TSV 1860 뮌헨">TSV 1860 뮌헨</a>으로 이적하였고, 19골로 <a href="/wiki/%ED%91%B8%EC%8A%A4%EB%B0%9C-%EB%B6%84%EB%8D%B0%EC%8A%A4%EB%A6%AC%EA%B0%80_1999-2000" title="푸스발-분데스리가 1999-2000">1999-2000 시즌</a> 득점왕이 되었다. <a href="/wiki/%ED%91%B8%EC%8A%A4%EB%B0%9C-%EB%B6%84%EB%8D%B0%EC%8A%A4%EB%A6%AC%EA%B0%80_2001-02" title="푸스발-분데스리가 2001-02">2001-02 시즌</a>에는 18골을 득점하여 <a href="/wiki/%EB%A7%88%EB%A5%B4%EC%8B%9C%EC%9A%B0_%EC%95%84%EB%AA%A8%ED%98%B8%EC%A3%BC" title="마르시우 아모호주">마르시우 아모호주</a>와 공동 득점왕이 되었다.</p>
<p>35세가 된 막스는 <a href="/wiki/FC_%ED%95%9C%EC%9E%90_%EB%A1%9C%EC%8A%A4%ED%86%A0%ED%81%AC" title="FC 한자 로스토크">FC 한자 로스토크</a>로 이적하였고, 구 <a class="mw-redirect" href="/wiki/%EB%8F%85%EC%9D%BC_%EB%AF%BC%EC%A3%BC_%EA%B3%B5%ED%99%94%EA%B5%AD" title="독일 민주 공화국">동독</a> 클럽에 지대한 양의 힘을 불어넣어 여유롭게 1부리그 잔류를 확정시켰다. 그는 20골을 득점하여 득점 3위를 기록하였고, <a href="/wiki/%ED%91%B8%EC%8A%A4%EB%B0%9C-%EB%B6%84%EB%8D%B0%EC%8A%A4%EB%A6%AC%EA%B0%80_2003-04" title="푸스발-분데스리가 2003-04">2003-04 시즌</a>을 끝으로 은퇴하였는데, 당시 그는 396경기에 출전하여 126번의 1부리그 득점을 기록하였다. 은퇴 후, 막스는 유소년 축구 캠프를 운영하기 시작하였다.</p>
<h2><span class="mw-headline" id=".EA.B5.AD.EA.B0.80.EB.8C.80.ED.91.9C.ED.8C.80_.EA.B2.BD.EB.A0.A5">국가대표팀 경력</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%A7%88%EB%A5%B4%ED%8B%B4_%EB%A7%89%EC%8A%A4&amp;action=edit&amp;section=2" title="부분 편집: 국가대표팀 경력">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>1860 뮌헨에서 우수한 성과를 보인 그는 <a href="/wiki/%EB%8F%85%EC%9D%BC_%EC%B6%95%EA%B5%AC_%EA%B5%AD%EA%B0%80%EB%8C%80%ED%91%9C%ED%8C%80" title="독일 축구 국가대표팀">독일 축구 국가대표팀</a>에 차출되었고, 2002년 4월 17일 (당시 나이가 거의 34세), 0-1로 패한 <a href="/wiki/%EC%95%84%EB%A5%B4%ED%97%A8%ED%8B%B0%EB%82%98_%EC%B6%95%EA%B5%AC_%EA%B5%AD%EA%B0%80%EB%8C%80%ED%91%9C%ED%8C%80" title="아르헨티나 축구 국가대표팀">아르헨티나</a>전에서 84분에 교체 투입되어 처음이자 마지막으로 국가대표팀 경기에 참여하였다.</p>
