<p><b>마리니드 왕조</b>는 <a href="/wiki/%EB%B2%A0%EB%A5%B4%EB%B2%A0%EB%A5%B4%EC%9D%B8" title="베르베르인">베르베르인</a>의 일파인 <a class="new" href="/w/index.php?title=%EC%A0%9C%EB%82%98%ED%83%80&amp;action=edit&amp;redlink=1" title="제나타 (없는 문서)">제나타</a>가 세운 옛 국가이다. 지금의 <a href="/wiki/%EB%AA%A8%EB%A1%9C%EC%BD%94" title="모로코">모로코</a> 지역을 중심으로 13세기에서 15세기까지 존속하였다.<sup class="reference" id="cite_ref-1"><a href="#cite_note-1">[1]</a></sup><sup class="reference" id="cite_ref-2"><a href="#cite_note-2">[2]</a></sup></p>
<p>1244년 마리니드 왕조는 <a class="mw-redirect" href="/wiki/%EB%AC%B4%EC%99%80%ED%9E%88%EB%93%9C_%EC%99%95%EC%A1%B0" title="무와히드 왕조">무와히드 왕조</a>에게서 모로코 지역의 통치권을 가져왔고<sup class="reference" id="cite_ref-3"><a href="#cite_note-3">[3]</a></sup>, 14세기 중반 무렵 <a href="/wiki/%EB%A7%88%EA%B7%B8%EB%A0%88%EB%B8%8C" title="마그레브">마그레브</a> 전체를 통치하였다. 마리니드 왕조는 <a href="/wiki/%EC%95%8C%EC%95%88%EB%8B%AC%EB%A3%A8%EC%8A%A4" title="알안달루스">알안달루스</a>의 <a href="/wiki/%EA%B7%B8%EB%9D%BC%EB%82%98%EB%8B%A4_%ED%86%A0%ED%9B%84%EA%B5%AD" title="그라나다 토후국">그라나다 토후국</a>을 지원하여 이베리아 반도의 유럽인 국가들과 수 차레 전쟁을 치렀다.그러나, 1340년 <a href="/wiki/%ED%8F%AC%EB%A5%B4%ED%88%AC%EA%B0%88_%EC%99%95%EA%B5%AD" title="포르투갈 왕국">포르투갈 왕국</a>과 싸운 <a class="new" href="/w/index.php?title=%ED%9E%88%EC%9A%B0%EC%82%B4%EB%9D%BC%EB%91%90_%EC%A0%84%ED%88%AC&amp;action=edit&amp;redlink=1" title="히우살라두 전투 (없는 문서)">히우살라두 전투</a>에서 패하여 <a href="/wiki/%EC%A7%80%EB%B8%8C%EB%A1%A4%ED%84%B0_%ED%95%B4%ED%98%91" title="지브롤터 해협">지브롤터 해협</a>에 대한 통제권을 잃었고, 1344년에는 <a href="/wiki/%EC%B9%B4%EC%8A%A4%ED%8B%B0%EC%95%BC_%EC%99%95%EA%B5%AD" title="카스티야 왕국">카스티야 왕국</a>에게 <a href="/wiki/%EC%95%8C%ED%97%A4%EC%8B%9C%EB%9D%BC%EC%8A%A4" title="알헤시라스">알헤시라스</a>를 빼앗겼다.<sup class="reference" id="cite_ref-4"><a href="#cite_note-4">[4]</a></sup></p>
<p>마리니드 왕조는 <a href="/wiki/1465%EB%85%84" title="1465년">1465년</a> <a class="new" href="/w/index.php?title=%EB%AA%A8%EB%A1%9C%EC%BD%94_%EB%B0%98%EB%9E%80_(1465%EB%85%84)&amp;action=edit&amp;redlink=1" title="모로코 반란 (1465년) (없는 문서)">모로코 반란</a>으로 멸망하였다.<sup class="reference" id="cite_ref-5"><a href="#cite_note-5">[5]</a></sup> 마리니드 왕조가 사라진 뒤 1472년 <a class="new" href="/w/index.php?title=%EC%99%80%ED%83%80%EC%8B%9C%EB%93%9C_%EC%99%95%EC%A1%B0&amp;action=edit&amp;redlink=1" title="와타시드 왕조 (없는 문서)">와타시드 왕조</a>가 들어섰다.<sup class="reference" id="cite_ref-6"><a href="#cite_note-6">[6]</a></sup></p>
