<p><b>스페인의 마리아 루도비카</b> (<span style="font-size: smaller;"><a href="/wiki/%EB%8F%85%EC%9D%BC%EC%96%B4" title="독일어">독일어</a>:</span> <span lang="de" xml:lang="de">Maria Ludovika von Spanien</span>, <a href="/wiki/1745%EB%85%84" title="1745년">1745년</a> <a href="/wiki/11%EC%9B%94_24%EC%9D%BC" title="11월 24일">11월 24일</a> ~ <a href="/wiki/1792%EB%85%84" title="1792년">1792년</a> <a href="/wiki/3%EC%9B%94_15%EC%9D%BC" title="3월 15일">3월 15일</a>)은 <a class="mw-redirect" href="/wiki/%EC%8B%A0%EC%84%B1%EB%A1%9C%EB%A7%88%EC%A0%9C%EA%B5%AD%EC%9D%98_%EB%A0%88%EC%98%A4%ED%8F%B4%ED%8A%B8_2%EC%84%B8" title="신성로마제국의 레오폴트 2세">레오폴트 2세</a>의 배우자이다. 본래 그녀의 남편의 지위는 <a 그러나 시아주버니인 <a href="/wiki/%EC%9A%94%EC%A0%9C%ED%94%84_2%EC%84%B8" title="요제프 2세">요제프 2세</a>가 <a class="mw-redirect" class="mw-redirect" href="/wiki/%EC%8B%A0%EC%84%B1_%EB%A1%9C%EB%A7%88_%EC%A0%9C%EA%B5%AD_%ED%99%A9%EC%A0%9C" title="신성 로마 제국 황제">신성 로마 제국 황제</a>의 제위를 이어받게 되었다. 그에 따라 마리아 루도비카 역시 <a class="new" href="/w/index.php?title=%EC%8B%A0%EC%84%B1_%EB%A1%9C%EB%A7%88_%ED%99%A9%ED%9B%84&amp;action=edit&amp;redlink=1" title="신성 로마 황후 (없는 문서)">신성 로마 황후</a>, <a href="/wiki/%EC%98%A4%EC%8A%A4%ED%8A%B8%EB%A6%AC%EC%95%84%EC%9D%98_%EA%B5%B0%EC%A3%BC_%EB%B0%B0%EC%9A%B0%EC%9E%90" title="오스트리아의 군주 배우자">오스트리아 대공비</a>, <a class="new"
