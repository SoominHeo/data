<p><b>마리안 루푸</b>(<span style="font-size: smaller;"><a href="/wiki/%EB%A3%A8%EB%A7%88%EB%8B%88%EC%95%84%EC%96%B4" title="루마니아어">루마니아어</a>:</span> <span lang="ro" xml:lang="ro">Marian Lupu</span>, <a href="/wiki/1966%EB%85%84" title="1966년">1966년</a> <a href="/wiki/6%EC%9B%94_20%EC%9D%BC" title="6월 20일">6월 20일</a> ~ )는 <a href="/wiki/%EB%AA%B0%EB%8F%84%EB%B0%94" title="몰도바">몰도바</a>의 정치인으로, <a href="/wiki/2005%EB%85%84" title="2005년">2005년</a> <a href="/wiki/3%EC%9B%94" title="3월">3월</a>부터 <a href="/wiki/2009%EB%85%84" title="2009년">2009년</a> <a href="/wiki/5%EC%9B%94" title="5월">5월</a>까지 의회의 의장직을 맡았다. 2010년부터 2012년까지 몰도바의 임시 대통령으로 재직했다.</p>
<p>한때 <a class="new" href="/w/index.php?title=%EB%AA%B0%EB%8F%84%EB%B0%94_%EA%B3%B5%EC%82%B0%EB%8B%B9&amp;action=edit&amp;redlink=1" title="몰도바 공산당 (없는 문서)">몰도바 공산당</a>원이었으며 <a class="new" href="/w/index.php?title=%EC%A7%80%EB%82%98%EC%9D%B4%EB%8B%A4_%EA%B7%B8%EB%A0%88%EC%B2%B4%EC%95%84%EB%88%84%EC%9D%B4&amp;action=edit&amp;redlink=1" title="지나이다 그레체아누이 (없는 문서)">지나이다 그레체아누이</a>의 자리를 이어 몰도바의 총리직을 맡을 뻔 하였으나, <a href="/wiki/2009%EB%85%84" title="2009년">2009년</a> <a href="/wiki/6%EC%9B%94_3%EC%9D%BC" title="6월 3일">6월 3일</a> 몰도바 공산당측에서 개혁이 불가능하다는 이유 때문에 결렬되고 말았다. 이후 <a class="new" href="/w/index.php?title=%EB%AF%BC%EC%A3%BC%EB%8B%B9_(%EB%AA%B0%EB%8F%84%EB%B0%94)&amp;action=edit&amp;redlink=1" title="민주당 (몰도바) (없는 문서)">민주당</a>으로 이전하였으나, 총선에서 의석을 확보할 수 없었다<sup class="reference" id="cite_ref-1"><a href="#cite_note-1">[1]</a></sup>. 2개월 후 다시 열린 총선에서는 민주당이 13석을 확보할 수 있게 되었다.</p>
<p><a class="new" href="/w/index.php?title=%EB%AA%B0%EB%8F%84%EB%B0%94_%EA%B5%AD%EB%A6%BD_%EB%8C%80%ED%95%99&amp;action=edit&amp;redlink=1" title="몰도바 국립 대학 (없는 문서)">몰도바 국립 대학</a>에서 경제학을 배웠고, <a class="mw-disambig" href="/wiki/%EC%9B%8C%EC%8B%B1%ED%84%B4" title="워싱턴">워싱턴</a>에 위치한 <a href="/wiki/%EA%B5%AD%EC%A0%9C_%ED%86%B5%ED%99%94_%EA%B8%B0%EA%B8%88" title="국제 통화 기금">국제 통화 기금</a>과 <a href="/wiki/%EC%A0%9C%EB%84%A4%EB%B0%94" title="제네바">제네바</a>에 위치한 <a class="new" href="/w/index.php?title=%EC%84%B8%EA%B3%84_%EB%AC%B4%EC%97%AD%EA%B8%B0%EA%B5%AC&amp;action=edit&amp;redlink=1" title="세계 무역기구 (없는 문서)">세계 무역기구</a>에서 일하였다.</p>
<p><a href="/wiki/2001%EB%85%84" title="2001년">2001년</a> <a href="/wiki/6%EC%9B%94" title="6월">6월</a> 경제부 차관으로 임명되었고, <a href="/wiki/2003%EB%85%84" title="2003년">2003년</a> 경제부 장관으로 승진했다. <a href="/wiki/2005%EB%85%84" title="2005년">2005년</a> 몰도바 의회 의장이 되었다. <a href="/wiki/2010%EB%85%84" title="2010년">2010년</a> 몰도바의 6대 대통령이 되었다.</p>
<p>그는 모국어인 <a href="/wiki/%EB%A3%A8%EB%A7%88%EB%8B%88%EC%95%84%EC%96%B4" title="루마니아어">루마니아어</a> 말고도 <a href="/wiki/%EC%98%81%EC%96%B4" title="영어">영어</a>, <a class="mw-redirect" href="/wiki/%EB%B6%88%EC%96%B4" title="불어">불어</a>, <a href="/wiki/%EB%9F%AC%EC%8B%9C%EC%95%84%EC%96%B4" title="러시아어">러시아어</a> 등을 말할 수 있다.</p>
