<h1 class="firstHeading" id="firstHeading" lang="ko">마리앵 은구아비</h1>
<div class="mw-body-content" id="bodyContent">

은구아비에 의해 통제되던 콩고 인민 공화국군(1970 -1992)의 깃발</div>

<p><b>마리앵 은구아비</b>(<span style="font-size: smaller;"><a href="/wiki/%ED%94%84%EB%9E%91%EC%8A%A4%EC%96%B4" title="프랑스어">프랑스어</a>:</span> <span lang="fr" xml:lang="fr">Marien Ngouabi</span>, <a href="/wiki/1938%EB%85%84" title="1938년">1938년</a> <a href="/wiki/12%EC%9B%94_31%EC%9D%BC" title="12월 31일">12월 31일</a> ~ <a href="/wiki/1977%EB%85%84" title="1977년">1977년</a> <a href="/wiki/3%EC%9B%94_18%EC%9D%BC" title="3월 18일">3월 18일</a>)는 <a class="mw-redirect" href="/wiki/%EC%BD%A9%EA%B3%A0_%EC%9D%B8%EB%AF%BC_%EA%B3%B5%ED%99%94%EA%B5%AD" title="콩고 인민 공화국">콩고 인민 공화국</a>의 대통령으로, <a href="/wiki/1968%EB%85%84" title="1968년">1968년</a>부터 <a href="/wiki/1977%EB%85%84" title="1977년">1977년</a>까지 그 권한을 맡았다.</p>
<p>콩고 이북에서 태어나 브리자빌, 부아르에서 공부하고 1961년 <a href="/wiki/%ED%94%84%EB%9E%91%EC%8A%A4" title="프랑스">프랑스</a>에 정착해 스트라스부르에서 교육을 받고 군대에 입대하였다. 1962년 콩고로 돌아와 부앵트누아르 수비대 중위로 주둔했하다가 1963년 대위로 진급했다.</p>
<p>1965년 콩고 공화국 최초의 낙하산 대대를 창설했으나 정부와의 갈등으로 1966년 강등되었고 이후 쿠데타 모의를 이유로 대통령 <a class="new" href="/w/index.php?title=%EC%95%8C%ED%90%81%EC%8A%A4_%EB%A7%88%EC%85%88%EB%B0%94%EB%8D%B0%EB%B0%94&amp;action=edit&amp;redlink=1" title="알퐁스 마셈바데바 (없는 문서)">알퐁스 마셈바데바</a>에 의해 1968년 7월 29일 중위 <a class="new" href="/w/index.php?title=%EC%95%8C%ED%94%84%EB%A0%88%EB%93%9C_%EB%9D%BC%EC%9A%B8&amp;action=edit&amp;redlink=1" title="알프레드 라울 (없는 문서)">알프레드 라울</a>과 함께 체포되었다. 그러나 은구아비의 체포는 군인들의 불만을 폭발시켜 7월 31일 군인들에 의해 구출되어 8월 5일 국가 혁명 위원회(CNR)을 조직하고 수장이 되었다.</p>
<p>9월 4일 마셈바데바가 사임하자 총리 알프레드 라울이 12월 31일까지 대통령을 맡다가 은구아비에게 권한을 넘기면서 콩고를 아프리카 최초의 마르크스주의 국가로 선포했다. 콩고의 유일 합법 정당인 <a class="new" href="/w/index.php?title=%EC%BD%A9%EA%B3%A0_%EB%85%B8%EB%8F%99%EB%8B%B9&amp;action=edit&amp;redlink=1" title="콩고 노동당 (없는 문서)">콩고 노동당</a>(PCT)을 결성하고 정치적으로는 중도였으나, 실제로는 <a class="mw-redirect" href="/wiki/%EB%A7%88%EB%A5%B4%ED%81%AC%EC%8A%A4_%EC%9D%B4%EB%8D%B0%EC%98%AC%EB%A1%9C%EA%B8%B0" title="마르크스 이데올로기">마르크스 이데올로기</a>를 도입하여 엄격한 사회주의를 표하였다.</p>
<p>또한 북부에서 남부까지 국가 전체를 통제하면서 여러 반발을 불러일으켜 1972년 2월 브리자빌에서 야당에 의한 군사 쿠데타가 일어났는데 은구아비가 석유가 풍부한 앙골라의 영토인 <a href="/wiki/%EC%B9%B4%EB%B9%88%EB%8B%A4" title="카빈다">카빈다</a>를 위협하면서 프랑스의 압력에 의해 일어났다고 추측된다. 1973년 은구아비는 <a href="/wiki/%EC%A4%91%ED%99%94%EC%9D%B8%EB%AF%BC%EA%B3%B5%ED%99%94%EA%B5%AD" title="중화인민공화국">중화인민공화국</a>을 방문했고 <a href="/wiki/1974%EB%85%84" title="1974년">1974년</a> 12월 30일 PCT 후보로 나서 재선되었고 PCT의 영구 의장으로도 선출되었다.</p>
<p>1975년 <a href="/wiki/1%EC%9B%94_9%EC%9D%BC" title="1월 9일">1월 9일</a> 대통령 선서 후 취임하였고 <a href="/wiki/%EC%86%8C%EB%A0%A8" title="소련">소련</a>과 경제 원조 협정을 체결하기도 했으나 <a href="/wiki/1977%EB%85%84" title="1977년">1977년</a> <a href="/wiki/3%EC%9B%94_18%EC%9D%BC" title="3월 18일">3월 18일</a> 전직 대통령 마셈바데바와 함께 자살 폭탄 테러로 암살당했다. 사후 당 군사위원회(CMP)의 주도로 요야킴 욤비 오판고가 임시 정부를 조직하고 후임 대통령이 되었다. 브리자빌에 묻혔으며 해마다 3월 18일은 마리앵 은구아비의 날로 정해졌으며 브리자빌에 위치한 콩고 유일의 대학도 <a class="new" href="/w/index.php?title=%EB%A7%88%EB%A6%AC%EC%95%B5_%EC%9D%80%EA%B5%AC%EC%95%84%EB%B9%84_%EB%8C%80%ED%95%99%EA%B5%90&amp;action=edit&amp;redlink=1" title="마리앵 은구아비 대학교 (없는 문서)">마리앵 은구아비 대학교</a>로 명명되었다.</p>
