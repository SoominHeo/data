<p><a href="/wiki/1999%EB%85%84" title="1999년">1999년</a> 18세의 나이로 <a href="/wiki/%ED%8A%B8%EB%A3%A8%EC%95%84_AC" title="트루아 AC">트루아 AC</a>에 입단하였으며 <a class="new" href="/w/index.php?title=%EB%A6%AC%EA%B7%B8_1_2000-01&amp;action=edit&amp;redlink=1" title="리그 1 2000-01 (없는 문서)">2000-01 시즌</a>부터 A팀에서 활약하기 시작하였으나 큰 활약을 보이지 못하고 <a href="/wiki/2003%EB%85%84" title="2003년">2003년</a> <a href="/wiki/FC_%EB%A9%94%EC%8A%A4" title="FC 메스">FC 메스</a>로 임대되었다. 메스에서 12경기에 출전하여 5골을 넣는 준수한 활약을 펼치자 당시 메스의 <a class="new" href="/w/index.php?title=%EC%9E%A5_%ED%8E%98%EB%A5%B4%EB%82%9C%EB%8D%B0%EC%8A%A4&amp;action=edit&amp;redlink=1" title="장 페르난데스 (없는 문서)">장 페르난데스</a> 감독이 완전 영입을 추진하였으나, 이전에 이미 <a href="/wiki/RC_%EC%8A%A4%ED%8A%B8%EB%9D%BC%EC%8A%A4%EB%B6%80%EB%A5%B4" title="RC 스트라스부르">RC 스트라스부르</a>로의 이적이 확정되면서 영입에 실패하였다.</p>
<p>스트라스부르에서 <a href="/wiki/2005%EB%85%84" title="2005년">2005년</a> <a href="/wiki/%EC%BF%A0%ED%94%84_%EB%93%9C_%EB%9D%BC_%EB%A6%AC%EA%B7%B8" title="쿠프 드 라 리그">쿠프 드 라 리그</a> 우승을 이끌고 <a href="/wiki/%EC%98%AC%EB%9E%AD%ED%94%BC%ED%81%AC_%EB%93%9C_%EB%A7%88%EB%A5%B4%EC%84%B8%EC%9C%A0" title="올랭피크 드 마르세유">올랭피크 드 마르세유</a>로 이적하였다. <a href="/wiki/2010%EB%85%84" title="2010년">2010년</a> 8백만 유로의 이적료로 <a class="mw-redirect" href="/wiki/%ED%8E%98%EB%84%A4%EB%A5%B4%EB%B0%94%EC%B2%B4_SK" title="페네르바체 SK">페네르바체 SK</a>로 이적할 때까지 마르세유의 주전 공격수로 활약하였으며 <a href="/wiki/%EB%A6%AC%EA%B7%B8_1_2009-10" title="리그 1 2009-10">2009-10 시즌</a> 득점왕에 오르기도 하였다. <a href="/wiki/2011%EB%85%84" title="2011년">2011년</a> <a href="/wiki/9%EC%9B%94" title="9월">9월</a>, 7백 50만 유로의 이적료로 <a class="mw-redirect" href="/wiki/%EC%95%8C%EC%82%AC%EB%93%9C_%EC%8A%A4%ED%8F%AC%EC%B8%A0_%ED%81%B4%EB%9F%BD" title="알사드 스포츠 클럽">알사드 스포츠 클럽</a>으로 이적하였다. 알사드로 이적한 뒤 한 달 후에 열린 <a href="/wiki/AFC_%EC%B1%94%ED%94%BC%EC%96%B8%EC%8A%A4%EB%A6%AC%EA%B7%B8_2011" title="AFC 챔피언스리그 2011">2011 AFC 챔피언스리그</a> 4강 1차전에서 <a href="/wiki/%EC%88%98%EC%9B%90_%EC%82%BC%EC%84%B1_%EB%B8%94%EB%A3%A8%EC%9C%99%EC%A6%88" title="수원 삼성 블루윙즈">수원 삼성 블루윙즈</a>를 상대로 넣은 두 번째 골이 논란의 대상이 되기도 하였다.<sup class="reference" id="cite_ref-1"><a href="#cite_note-1">[1]</a></sup></p>
