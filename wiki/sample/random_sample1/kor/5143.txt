<p><b>마빈 리 민스키</b>(Marvin Lee Minsky, <a href="/wiki/1927%EB%85%84" title="1927년">1927년</a> <a href="/wiki/8%EC%9B%94_9%EC%9D%BC" title="8월 9일">8월 9일</a> ~ <a href="/wiki/2016%EB%85%84" title="2016년">2016년</a> <a href="/wiki/1%EC%9B%94_24%EC%9D%BC" title="1월 24일">1월 24일</a>)는 <a href="/wiki/%EC%9D%B8%EA%B3%B5%EC%A7%80%EB%8A%A5" title="인공지능">인공지능</a>(AI) 분야를 개척한 미국인 과학자이다. <a class="mw-redirect" href="/wiki/%EB%A7%A4%EC%82%AC%EC%B6%94%EC%84%B8%EC%B8%A0_%EA%B3%B5%EA%B3%BC%EB%8C%80%ED%95%99" title="매사추세츠 공과대학">MIT</a>의 인공지능 연구소의 공동 설립자이며, AI와 관련된 책들을 저술했다.</p>
<p>뉴욕에서 태어나 브롱크스 과학 고등학교를 나왔다. <a href="/wiki/%EB%A7%A4%EC%82%AC%EC%B6%94%EC%84%B8%EC%B8%A0_%EC%A3%BC" title="매사추세츠 주">매사추세츠 주</a>의 필립스 아카데미를 다녔고, 1944년부터 1945년까지 미국 해군으로 복무했다. 1950년 <a href="/wiki/%ED%95%98%EB%B2%84%EB%93%9C_%EB%8C%80%ED%95%99%EA%B5%90" title="하버드 대학교">하버드 대학교</a>에서 수학 학사학위를 받고, <a href="/wiki/1954%EB%85%84" title="1954년">1954년</a>에 <a href="/wiki/%ED%94%84%EB%A6%B0%EC%8A%A4%ED%84%B4_%EB%8C%80%ED%95%99%EA%B5%90" title="프린스턴 대학교">프린스턴 대학교</a>에서 수학 박사 학위를 받았다. <a href="/wiki/1958%EB%85%84" title="1958년">1958년</a>부터 MIT 교수직으로 재직했다.</p>
<p>1953년에 <a href="/wiki/%EA%B3%B5%EC%B4%88%EC%A0%90_%EB%A0%88%EC%9D%B4%EC%A0%80_%EC%A3%BC%EC%82%AC_%ED%98%84%EB%AF%B8%EA%B2%BD" title="공초점 레이저 주사 현미경">공초점 레이저 주사 현미경</a>에 관한 이론을 개발해 냈다.</p>
<p><a href="/wiki/1970%EB%85%84" title="1970년">1970년</a>에 <a href="/wiki/%ED%8A%9C%EB%A7%81%EC%83%81" title="튜링상">튜링상</a>, <a href="/wiki/1990%EB%85%84" title="1990년">1990년</a>에 <a class="new" href="/w/index.php?title=%EC%9D%BC%EB%B3%B8%EA%B5%AD%EC%A0%9C%EC%83%81&amp;action=edit&amp;redlink=1" title="일본국제상 (없는 문서)">일본국제상</a>, 2001년 벤자민 플랭클린 메달을 비롯한 다양한 상을 수상했으며, 미국 국립 공학 학술원(National Academy of Engineering)과 국립 과학 학술원(National Academy of Sciences)의 회원이다.</p>
