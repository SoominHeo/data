<h2><span class="mw-headline" id=".EC.9D.B8.EB.AC.BC.EA.B3.BC_.EA.B2.BD.EB.A0.A5">인물과 경력</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%A7%88%EC%82%AC%ED%82%A4_%EC%A7%84%EC%9E%90%EB%B6%80%EB%A1%9C&amp;action=edit&amp;section=1" title="부분 편집: 인물과 경력">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>마사키 진자부로는 <a href="/wiki/1876%EB%85%84" title="1876년">1876년</a> <a href="/wiki/%EC%82%AC%EA%B0%80_%ED%98%84" title="사가 현">사가 현</a>에서 태어났다. <a href="/wiki/1896%EB%85%84" title="1896년">1896년</a> <a href="/wiki/%EC%9D%BC%EB%B3%B8_%EC%9C%A1%EA%B5%B0%EC%82%AC%EA%B4%80%ED%95%99%EA%B5%90" title="일본 육군사관학교">일본 육군사관학교</a>를 졸업하고 <a href="/wiki/%EB%9F%AC%EC%9D%BC_%EC%A0%84%EC%9F%81" title="러일 전쟁">러일 전쟁</a>에 종군하였으며, 귀국 후에는 일본 육군대학교 제19기를 졸업하였다. 이때 동기 중에선 후에 일본 육군 장성이 되는 <a href="/wiki/%EC%95%84%EB%9D%BC%ED%82%A4_%EC%82%AC%EB%8B%A4%EC%98%A4" title="아라키 사다오">아라키 사다오</a>, <a href="/wiki/%EC%95%84%EB%B2%A0_%EB%85%B8%EB%B6%80%EC%9C%A0%ED%82%A4" title="아베 노부유키">아베 노부유키</a>, <a href="/wiki/%ED%98%BC%EC%A1%B0_%EC%8B%9C%EA%B2%8C%EB%A3%A8" title="혼조 시게루">혼조 시게루</a> 등이 포함돼 있었다.</p>
<p>그가 군에서 맡았던 주요 직책으로는 군무국 군사 과장, 근위 보병 제1연대장, 보병 제1여단장, <a href="/wiki/%EC%9D%BC%EB%B3%B8_%EC%9C%A1%EA%B5%B0%EC%82%AC%EA%B4%80%ED%95%99%EA%B5%90" title="일본 육군사관학교">일본 육군사관학교</a>장, 일본제국 육군 제8사단장, 일본제국 육군 제1사단장, 타이완 주둔군 사령관, 참모 차장 겸 군사 참의관 등이 있으며,</p>
<p>특히 <a href="/wiki/%EC%9D%BC%EB%B3%B8_%EC%9C%A1%EA%B5%B0%EC%82%AC%EA%B4%80%ED%95%99%EA%B5%90" title="일본 육군사관학교">일본 육사</a> 교장 시절에는 존황절대주의(尊皇絶對主義)의 육성에 힘쓰며 후에 <a href="/wiki/2%C2%B726_%EC%82%AC%EA%B1%B4" title="2·26 사건">2·26 사건</a>의 주모자 안도 데루조(安藤 輝三), 이소베 아사이치(磯部 浅一) 등을 배출해냈다. <a href="/wiki/1932%EB%85%84" title="1932년">1932년</a> 참모 차장이 된 후에는 <a href="/wiki/%EC%95%84%EB%9D%BC%ED%82%A4_%EC%82%AC%EB%8B%A4%EC%98%A4" title="아라키 사다오">아라키 사다오</a> 일본 육군 대신과 함께 국가 혁신을 도모하는 <a href="/wiki/%ED%99%A9%EB%8F%84%ED%8C%8C" title="황도파">황도파</a>를 형성하였다.<sup class="reference" id="cite_ref-1"><a href="#cite_note-1">[1]</a></sup> 세력의 확대를 도모해 중견 장교들의 신망을 책임졌지만, 후에 당파적인 행동이 반발을 사 <a href="/wiki/%ED%86%B5%EC%A0%9C%ED%8C%8C" title="통제파">통제파</a>의 등장을 야기시켰다. 직위는 참모 차장이었지만, 당시 일본의 참모 총장이었던 <a class="mw-redirect" href="/wiki/%EA%B0%84%EC%9D%B8%EB%85%B8%EB%AF%B8%EC%95%BC_%EA%B3%A0%ED%86%A0%ED%9E%88%ED%86%A0" title="간인노미야 고토히토">간인노미야 고토히토</a> 아래에서 사실상 참모 총장으로서 참모본부를 움직였고, <a class="mw-redirect" href="/wiki/%EB%A7%8C%EC%A3%BC_%EC%82%AC%EB%B3%80" title="만주 사변">만주 사변</a>에서는 세력 확대를 주장했다.</p>
<p><a href="/wiki/1934%EB%85%84" title="1934년">1934년</a>에는 <a class="extiw" href="https://ja.wikipedia.org/wiki/%E6%95%99%E8%82%B2%E7%B7%8F%E7%9B%A3" title="ja:教育総監">교육 총감</a>으로 취임해 <a href="/wiki/%EC%B2%9C%ED%99%A9%EA%B8%B0%EA%B4%80%EC%84%A4" title="천황기관설">천황기관설</a> 문제에선 국체명징운동을 적극적으로 추진해 천황기관설을 공격하였다. 이 흐름을 염려해 일본 육군의 개혁을 단행하려고 한 <a href="/wiki/%EC%98%A4%EC%B9%B4%EB%8B%A4_%EA%B2%8C%EC%9D%B4%EC%8A%A4%EC%BC%80" title="오카다 게이스케">오카다</a> 내각의 <a href="/wiki/%ED%95%98%EC%95%BC%EC%8B%9C_%EC%84%BC%EC%A3%BC%EB%A1%9C" title="하야시 센주로">하야시 센주로</a>(林銑十郞) 일본 육군 대신과 일본 육군성 군무국장 <a href="/wiki/%EB%82%98%EA%B0%80%ED%83%80_%EB%8D%B0%EC%93%B0%EC%9E%94" title="나가타 데쓰잔">나가타 데쓰잔</a>(永田鉄山) 소장이 <a href="/wiki/1935%EB%85%84" title="1935년">1935년</a> <a href="/wiki/7%EC%9B%94" title="7월">7월</a>, 그를 육군 대장으로 승진시킴과 동시에, 육군의 세 장관 가운데 하나인 교육 총감 직에서 물러나게 하였고, 후임에 <a class="extiw" href="https://ja.wikipedia.org/wiki/%E6%B8%A1%E8%BE%BA%E9%8C%A0%E5%A4%AA%E9%83%8E" title="ja:渡辺錠太郎">와타나베 조타로</a>(渡辺錠太郎)를 앉혔다. 이 인사에 반발한 <a href="/wiki/%ED%99%A9%EB%8F%84%ED%8C%8C" title="황도파">황도파</a>의 <a class="extiw" href="https://ja.wikipedia.org/wiki/%E7%9B%B8%E6%B2%A2%E4%B8%89%E9%83%8E" title="ja:相沢三郎">아이자와 사부로</a>(相沢三郎) 일본 육군 중령이 <a href="/wiki/8%EC%9B%94" title="8월">8월</a> <a class="extiw" href="https://ja.wikipedia.org/wiki/%E7%9B%B8%E6%B2%A2%E4%BA%8B%E4%BB%B6" title="ja:相沢事件">아이자와 사건</a>을 일으켜 <a href="/wiki/%EB%82%98%EA%B0%80%ED%83%80_%EB%8D%B0%EC%93%B0%EC%9E%94" title="나가타 데쓰잔">나가타 데쓰잔</a>을 살해하였다.<sup class="reference" id="cite_ref-2"><a href="#cite_note-2">[2]</a></sup></p>
<p>이후 육군의 개혁에 반발한 <a href="/wiki/%ED%99%A9%EB%8F%84%ED%8C%8C" title="황도파">황도파</a>의 청년 장교들에 의해 <a href="/wiki/2%C2%B726_%EC%82%AC%EA%B1%B4" title="2·26 사건">2·26 사건</a><sup class="reference" id="cite_ref-3"><a href="#cite_note-3">[3]</a></sup> 이 일어났고<sup class="reference" id="cite_ref-4"><a href="#cite_note-4">[4]</a></sup>, 마사키 자신은 이 사건에 직접적으로 가담하지 않았다고 하나, 진위는 분명하지 않다. 그는 반란의 주모자인 안도 아키라와 아소베 아사이치 등의 실행 지지와 받아들이는 태도를 사건 전부터 나타내었고, 내각 조성 명령을 기대하면서 행동하였다.</p>
<p>이것이 원인이 돼 <a href="/wiki/1936%EB%85%84" title="1936년">1936년</a> <a href="/wiki/7%EC%9B%94" title="7월">7월</a> 구류되어 헌병대 본부의 조사를 받은 후 반란 방조로 군법 회의에 기소되었지만, 사건 관여를 부인하였다. 논고 구형을 통해 금고 13년형을 받았지만, <a href="/wiki/1937%EB%85%84" title="1937년">1937년</a> <a href="/wiki/9%EC%9B%94" title="9월">9월</a>의 판결로 무죄가 되었다. 그러나, 이 결과로 인해 <a href="/wiki/%ED%99%A9%EB%8F%84%ED%8C%8C" title="황도파">황도파</a>의 힘이 쇠약해지고 <a href="/wiki/%ED%86%B5%EC%A0%9C%ED%8C%8C" title="통제파">통제파</a>의 힘이 강화되었다.</p>
<p>항복 후인 <a href="/wiki/1945%EB%85%84" title="1945년">1945년</a> <a href="/wiki/11%EC%9B%94_19%EC%9D%BC" title="11월 19일">11월 19일</a>, A급 전범으로서 체포 명령이 발령돼 <a class="mw-redirect" href="/wiki/%EC%8A%A4%EA%B0%80%EB%AA%A8_%ED%98%95%EB%AC%B4%EC%86%8C" title="스가모 형무소">스가모 형무소</a>에 입소하였다. 그러나 <a class="mw-redirect" href="/wiki/%EA%B7%B9%EB%8F%99_%EA%B5%AD%EC%A0%9C_%EA%B5%B0%EC%82%AC_%EC%9E%AC%ED%8C%90" title="극동 국제 군사 재판">극동 국제 군사 재판</a>에선 불기소 처분되었고, <a href="/wiki/1956%EB%85%84" title="1956년">1956년</a> 사망하였다.</p>
