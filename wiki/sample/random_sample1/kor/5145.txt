<p><b>마사히토 친왕비 하나코</b>(正仁親王妃華子)는 <a href="/wiki/%EC%9D%BC%EB%B3%B8" title="일본">일본</a>의 <a href="/wiki/%ED%99%A9%EC%A1%B1" title="황족">황족</a>으로 <a href="/wiki/%ED%9E%88%ED%83%80%EC%B9%98%EB%85%B8%EB%AF%B8%EC%95%BC_%EB%A7%88%EC%82%AC%ED%9E%88%ED%86%A0_%EC%B9%9C%EC%99%95" title="히타치노미야 마사히토 친왕">히타치노미야 마사히토 친왕</a>(常陸宮正仁親王)의 비이다. 혼인 전의 이름은 <b>쓰가루 하나코</b>(津軽華子). <a href="/wiki/%EC%8B%A0%EC%9C%84" title="신위">신위</a>(身位)는 <a href="/wiki/%EC%B9%9C%EC%99%95%EB%B9%84" title="친왕비">친왕비</a>(親王妃, 히타치노미야 비(常陸宮妃)). 전후 유일한 <a href="/wiki/%ED%99%94%EC%A1%B1" title="화족">화족가</a> 출신 친왕비에 해당한다.</p>
<h2><span class="mw-headline" id=".EA.B0.80.EA.B3.84">가계</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%A7%88%EC%82%AC%ED%9E%88%ED%86%A0_%EC%B9%9C%EC%99%95%EB%B9%84_%ED%95%98%EB%82%98%EC%BD%94&amp;action=edit&amp;section=1" title="부분 편집: 가계">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>친조부는 <a class="new" href="/w/index.php?title=%EC%98%A4%EC%99%80%EB%A6%AC_%EB%8F%84%EC%BF%A0%EA%B0%80%EC%99%80_%EA%B0%80&amp;action=edit&amp;redlink=1" title="오와리 도쿠가와 가 (없는 문서)">오와리 도쿠가와 남작가</a>(오와리 분가) 초대 당주 <a class="new" href="/w/index.php?title=%EB%8F%84%EC%BF%A0%EA%B0%80%EC%99%80_%EC%9A%94%EC%8B%9C%EC%BF%A0%EB%AF%B8&amp;action=edit&amp;redlink=1" title="도쿠가와 요시쿠미 (없는 문서)">도쿠가와 요시쿠미</a>(徳川義恕)이며 양조부는 <a href="/wiki/%ED%9E%88%EB%A1%9C%EC%82%AC%ED%82%A4_%EB%B2%88" title="히로사키 번">히로사키</a> <a class="new" href="/w/index.php?title=%EC%93%B0%EA%B0%80%EB%A3%A8_%EC%94%A8&amp;action=edit&amp;redlink=1" title="쓰가루 씨 (없는 문서)">쓰가루 백작가</a> 제2대 당주 <a class="new" href="/w/index.php?title=%EC%93%B0%EA%B0%80%EB%A3%A8_%ED%9B%84%EC%82%AC%EB%A7%88%EB%A1%9C&amp;action=edit&amp;redlink=1" title="쓰가루 후사마로 (없는 문서)">쓰가루 후사마로</a>(津軽英麿)이다. 친외조부는 <a href="/wiki/%EA%B3%A0%EC%BF%A0%EB%9D%BC_%EB%B2%88" title="고쿠라 번">고쿠라</a> <a href="/wiki/%EC%98%A4%EA%B0%80%EC%82%AC%EC%99%80%EB%9D%BC_%EC%94%A8" title="오가사와라 씨">오가사와라 백작가</a> 초대 당주 <a class="new" href="/w/index.php?title=%EC%98%A4%EA%B0%80%EC%82%AC%EC%99%80%EB%9D%BC_%EB%8B%A4%EB%8B%A4%EB%85%B8%EB%B6%80&amp;action=edit&amp;redlink=1" title="오가사와라 다다노부 (없는 문서)">오가사와라 다다노부</a>(小笠原忠忱)이며 양외조부는 <a href="/wiki/%EB%AC%B4%EC%93%B0_%EA%B5%AD" title="무쓰 국">무쓰</a> 히로사키 번 마지막 번주 <a href="/wiki/%EC%93%B0%EA%B0%80%EB%A3%A8_%EC%93%B0%EA%B5%AC%EC%95%84%ED%82%A4%EB%9D%BC" title="쓰가루 쓰구아키라">쓰가루 쓰구아키라</a>(津軽承昭)이다. 아버지는 쓰가루 백작가 제3대 당주 <a class="new" href="/w/index.php?title=%EC%93%B0%EA%B0%80%EB%A3%A8_%EC%9A%94%EC%8B%9C%ED%83%80%EC%B9%B4&amp;action=edit&amp;redlink=1" title="쓰가루 요시타카 (없는 문서)">쓰가루 요시타카</a>(津軽義孝)이다.</p>
