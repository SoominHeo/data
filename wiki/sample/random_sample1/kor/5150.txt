<h1 class="firstHeading" id="firstHeading" lang="ko">마스 1호</h1>
<div class="mw-body-content" id="bodyContent">
<div id="siteSub">위키백과, 우리 모두의 백과사전.</div>
<div id="contentSub"></div>
<div class="mw-jump" id="jump-to-nav">
					이동:					<a href="#mw-head">둘러보기</a>, 					<a href="#p-search">검색</a>
</div>
<div class="mw-content-ltr" dir="ltr" id="mw-content-text" lang="ko"><script>function mfTempOpenSection(id){var block=document.getElementById("mf-section-"+id);block.className+=" open-block";block.previousSibling.className+=" open-block";}</script><p><b>마스 1호</b>는 <a href="/wiki/%EC%86%8C%EB%A0%A8" title="소련">소련</a>의 <a href="/wiki/%ED%99%94%EC%84%B1" title="화성">화성</a> 탐사선으로 <a href="/wiki/1962%EB%85%84" title="1962년">1962년</a> <a href="/wiki/11%EC%9B%94_1%EC%9D%BC" title="11월 1일">11월 1일</a>에 발사되었다. <a href="/wiki/%EC%86%8C%EB%A0%A8" title="소련">소련</a>은 당시 <a href="/wiki/%EB%A3%A8%EB%82%98_1%ED%98%B8" title="루나 1호">루나 1호</a>를 발사하고 나서, <a href="/wiki/%EB%AF%B8%EA%B5%AD" title="미국">미국</a>에게 자신감을 보이던 <a href="/wiki/%EC%86%8C%EB%A0%A8" title="소련">소련</a>이 <a href="/wiki/%ED%99%94%EC%84%B1" title="화성">화성</a> 탐사선을 발사시키겠다는 것을 발표하였다. 그로써, <a href="/wiki/%EC%86%8C%EB%A0%A8" title="소련">소련</a>은 <a class="new" href="/w/index.php?title=%EB%A7%88%EC%8A%A4_%EA%B3%84%ED%9A%8D&amp;action=edit&amp;redlink=1" title="마스 계획 (없는 문서)">마스 계획</a>을 세웠으며, 마스 1960A, 마스1960B, 마스 1962A 를 발사했으나 모두 실패했다. 마스1962A 실패후 소련은 마스 1호를 발사시켰지만. <a href="/wiki/%ED%99%94%EC%84%B1" title="화성">화성</a>에 19만 5천 km까지 접근한 뒤 교신이 두절되었다.</p>
