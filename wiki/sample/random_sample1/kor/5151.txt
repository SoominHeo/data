<p><b>마스 패스파인더</b> (<span style="font-size: smaller;"><a href="/wiki/%EC%98%81%EC%96%B4" title="영어">영어</a>:</span> <span lang="en" xml:lang="en">Mars Pathfinder</span>) 는 1997년에 <a href="/wiki/%ED%99%94%EC%84%B1" title="화성">화성</a>에 착륙한 <a class="mw-disambig" href="/wiki/%EB%AC%B4%EC%9D%B8" title="무인">무인</a> 착륙선과 이동식 로버를 지칭하는 말로 <a class="mw-redirect" href="/wiki/NASA" title="NASA">NASA</a>의 '디스커버리 계획' 의 일환이다. 이동식 로버의 이름은 <b>소저너</b> 이며 본체 착륙선은 1996년 12월 20일에 사망한 <a href="/wiki/%EC%B9%BC_%EC%84%B8%EC%9D%B4%EA%B1%B4" title="칼 세이건">칼 세이건</a> 을 기리기 위해 <b>칼 세이건 추모기지</b> 로 개명되었다.<sup class="reference" id="cite_ref-3"><a href="#cite_note-3">[3]</a></sup></p>
<p><a href="/wiki/%EB%A7%88%EC%8A%A4_%EA%B8%80%EB%A1%9C%EB%B2%8C_%EC%84%9C%EB%B2%A0%EC%9D%B4%EC%96%B4" title="마스 글로벌 서베이어">마스 글로벌 서베이어</a>가 발사된지 약 한달 뒤인 1996년 12월 4일에 <a href="/wiki/%EB%8D%B8%ED%83%80_II" title="델타 II">델타 II</a> 로켓에 탑재되어 발사되었고 미국의 독립기념일인 1997년 7월 4일에 <a href="/wiki/%ED%99%94%EC%84%B1" title="화성">화성</a>의 <a class="new" href="/w/index.php?title=%EC%95%84%EB%A0%88%EC%8A%A4_%EB%B0%9C%EB%A6%AC%EC%8A%A4&amp;action=edit&amp;redlink=1" title="아레스 발리스 (없는 문서)">아레스 발리스</a>에 성공적으로 착륙하였다. 착륙 이후에 착륙선이 활동을 위한 날개 모양의 <a href="/wiki/%ED%83%9C%EC%96%91_%EC%A0%84%EC%A7%80" title="태양 전지">태양 전지</a>판 3개를 전개하였고 소저너 로버도 활동을 개시하면서 임무가 시작되었다. 주 목표는 <a href="/wiki/%ED%99%94%EC%84%B1%EC%9D%98_%EB%8C%80%EA%B8%B0" title="화성의 대기">화성의 대기</a>와 <a href="/wiki/%EA%B8%B0%ED%9B%84" title="기후">기후</a>, <a class="mw-redirect" href="/wiki/%ED%86%A0%EC%96%91" title="토양">토양</a>과 <a href="/wiki/%EC%95%94%EC%84%9D" title="암석">암석</a>의 연구였다. 마스 패스파인더는 1976년에 <a href="/wiki/%EB%B0%94%EC%9D%B4%ED%82%B9_%EA%B3%84%ED%9A%8D" title="바이킹 계획">바이킹 착륙선</a>이 착륙한 이후 20년만에 착륙하는 탐사선이였으며 <a class="mw-redirect" href="/wiki/NASA" title="NASA">NASA</a>의 화성 탐사 재개가 목적이었다. 또한 <a href="/wiki/%EB%A7%88%EC%8A%A4_%EC%98%B5%EC%A0%80%EB%B2%84" title="마스 옵저버">마스 옵저버</a> 같은 너무 비싸고 복잡한 탐사선들이 여럿 실패하면서 얻은 교훈을 통해 "더 빠르고 더 뛰어나며 더 값싼" 이라는 구호 아래 시행되었으며 임무에 사용된 총 비용은 약 1억 5천만 달러였다.</p>
<p>마스 패스파인더에는 지금까지 사용되지 않은 다양한 종류의 신기술들이 사용됐는데 이때 패스파인더를 통해 검증된 기술들은 현재까지도 작동하는 <a class="mw-redirect" href="/wiki/%EC%98%A4%ED%8D%BC%ED%8A%9C%EB%8B%88%ED%8B%B0_%EB%A1%9C%EB%B2%84" title="오퍼튜니티 로버">오퍼튜니티 로버</a>와 작동은 멈추었지만 쌍둥이 로버인 <a class="mw-redirect" href="/wiki/%EC%8A%A4%ED%94%BC%EB%A6%BF_%EB%A1%9C%EB%B2%84" title="스피릿 로버">스피릿 로버</a>에도 사용되었다.</p>
<p>임무 총괄과 명령 조달은 <a href="/wiki/%EC%A0%9C%ED%8A%B8%EC%B6%94%EC%A7%84%EC%97%B0%EA%B5%AC%EC%86%8C" title="제트추진연구소">JPL</a>(제트추진연구소)가 맡았다.</p>

<h2><span class="mw-headline" id=".EC.9E.84.EB.AC.B4_.EB.AA.A9.ED.91.9C">임무 목표</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%A7%88%EC%8A%A4_%ED%8C%A8%EC%8A%A4%ED%8C%8C%EC%9D%B8%EB%8D%94&amp;action=edit&amp;section=1" title="부분 편집: 임무 목표">편집</a><span class="mw-editsection-bracket">]</span></span></h2>

<h2><span class="mw-headline" id=".ED.83.90.EC.82.AC_.EA.B8.B0.EA.B5.AC">탐사 기구</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%A7%88%EC%8A%A4_%ED%8C%A8%EC%8A%A4%ED%8C%8C%EC%9D%B8%EB%8D%94&amp;action=edit&amp;section=2" title="부분 편집: 탐사 기구">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<div class="thumb tleft">
<div class="thumbinner" style="width:252px;"><a class="image" href="/wiki/%ED%8C%8C%EC%9D%BC:Imager_For_Mars_Pathfinder.gif"><img alt="" class="thumbimage" data-file-height="224" data-file-width="288" height="194" src="//upload.wikimedia.org/wikipedia/commons/thumb/5/5e/Imager_For_Mars_Pathfinder.gif/250px-Imager_For_Mars_Pathfinder.gif" srcset="//upload.wikimedia.org/wikipedia/commons/5/5e/Imager_For_Mars_Pathfinder.gif 1.5x" width="250"/></a>
<div class="thumbcaption">
<div class="magnify"><a class="internal" href="/wiki/%ED%8C%8C%EC%9D%BC:Imager_For_Mars_Pathfinder.gif" title="실제 크기로"></a></div>
마스 패스파인더에 탑재된 카메라인 IMP의 모습</div>
</div>
</div>
<p>마스 패스파인더에는 <a href="/wiki/%ED%99%94%EC%84%B1" title="화성">화성</a> 토양을 분석하기 위한 여러 종류의 기구들이 착륙선과 로버에 탑재되었다. 본체 착륙선에는 높낮이 조절이 가능한 Imager for Mars Pathfinder (IMP) 라는 360도 회전이 가능한 특수 스테레오 카메라로 2개의 렌즈를 통해 화성의 풍경을 촬영할 수 있는 컬러 카메라였다. 미국의 <a href="/wiki/%EC%95%A0%EB%A6%AC%EC%A1%B0%EB%82%98_%EB%8C%80%ED%95%99%EA%B5%90" title="애리조나 대학교">애리조나 대학교</a>에서 제작하였으며 <a href="/wiki/%EB%A1%9D%ED%9E%88%EB%93%9C_%EB%A7%88%ED%8B%B4" title="록히드 마틴">록히드 마틴</a> 외 여러 기업에서 지원해주었다.<sup class="reference" id="cite_ref-Pathfinder_Instrument_5-0"><a href="#cite_note-Pathfinder_Instrument-5">[5]</a></sup> 또한 <a href="/wiki/%ED%99%94%EC%84%B1%EC%9D%98_%EB%8C%80%EA%B8%B0" title="화성의 대기">화성의 대기</a>와 <a href="/wiki/%EA%B8%B0%ED%9B%84" title="기후">기후</a>를 분석하는 Atmospheric Structure Instrument / Meteorology Package (ASI/MET) 라는 기구도 탑재되었는데 ASI는 <a href="/wiki/%EC%98%A8%EB%8F%84" title="온도">온도</a>와 <a href="/wiki/%EA%B8%B0%EC%95%95" title="기압">기압</a>, <a href="/wiki/%EB%B0%94%EB%9E%8C" title="바람">바람</a> 에 관한 정보를 수집할 수 있었고 MET은 3개의 <a href="/wiki/%EB%B0%94%EB%9E%8C%EC%9E%90%EB%A3%A8" title="바람자루">바람자루</a>가 탑재되어 <a href="/wiki/%ED%99%94%EC%84%B1" title="화성">화성</a>에서 부는 <a href="/wiki/%EB%B0%94%EB%9E%8C" title="바람">바람</a>의 풍속을 측정할 수 있었다.<sup class="reference" id="cite_ref-6"><a href="#cite_note-6">[6]</a></sup></p>
<p>소저너 로버는 화성 표면에서 운행하는 최초의 로버였다. 러시아에서 <a href="/wiki/%EB%8B%AC" title="달">달</a>에 로버를 먼저 보내긴 했으나 <a href="/wiki/%ED%99%94%EC%84%B1" title="화성">화성</a>에서 운행하는 로버는 소저너가 최초였다. 소저너는 6개의 바퀴가 부착되어 있었으며 평균 주행 속도는 초속 1 cm 정도였고, Alpha Proton X-ray Spectrometer (APXS) 라는 X-레이 분광기가 탑재되어 있었는데 이는 러시아의 화성 탐사선이였던 <a href="/wiki/%EB%A7%88%EC%8A%A4_96" title="마스 96">마스 96</a> 호에도 쓰인 기구였지만 소저너 로버에 탑재된 분광기는 <a href="/wiki/%EB%A7%88%EC%8A%A4_96" title="마스 96">마스 96</a> 호의 것과는 달리 약간의 개량을 거친 버전이었다.<sup class="reference" id="cite_ref-Pathfinder_Instrument_5-1"><a href="#cite_note-Pathfinder_Instrument-5">[5]</a></sup> APXS는 <a href="/wiki/%ED%99%94%EC%84%B1" title="화성">화성</a> 표면에서 <a class="mw-redirect" href="/wiki/%ED%86%A0%EC%96%91" title="토양">토양</a>과 <a href="/wiki/%EC%95%94%EC%84%9D" title="암석">암석</a>의 성분을 검사하는 역할을 했고 흑백 카메라와 컬러 카메라가 각각 한 개씩 탑재되었는데, 몇 밀리미터에서 몇백 미터 거리에서 <a href="/wiki/%ED%99%94%EC%84%B1" title="화성">화성</a>의 표면을 촬영할 수 있었고 이를 통해 <a href="/wiki/%ED%99%94%EC%84%B1" title="화성">화성</a> 지질학과 <a href="/wiki/%EC%95%94%EC%84%9D" title="암석">암석</a>들에 관한 정보 수집과 연구가 가능했다.</p>
<p>또한 0.3 <a class="mw-redirect" href="/wiki/%EB%A9%94%EA%B0%80%ED%94%BD%EC%85%80" title="메가픽셀">메가픽셀</a>의 해상도를 가진 주행용 네비게이션 흑백 카메라가 탑재되었고 장애물이나 위험을 탐지하는 레이저 기구가 5개 탑재되어 있었다. 착륙선과 소저너 로버 모두 전력원은 태양 전지판에서 공급받았으며 착륙선은 날개 모양의 태양 전지판 3개에서 35 W 를, 소저너 로버는 로버 상단에 부착된 태양 전지판에서 13 W 를 공급받으며 활동했다. 정보 송수신에는 X-밴드 고이득 안테나가 사용되었고 대역폭은 6kb/s <a href="/wiki/%EC%8B%AC%EC%9A%B0%EC%A3%BC_%ED%86%B5%EC%8B%A0%EB%A7%9D" title="심우주 통신망">심우주 통신망</a> 이었다.</p>

<h3><span class="mw-headline" id=".EC.B0.A9.EB.A5.99_.EA.B3.BC.EC.A0.95">착륙 과정</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%A7%88%EC%8A%A4_%ED%8C%A8%EC%8A%A4%ED%8C%8C%EC%9D%B8%EB%8D%94&amp;action=edit&amp;section=6" title="부분 편집: 착륙 과정">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<ul>
<li><a href="/wiki/1997%EB%85%84" title="1997년">1997년</a> <a href="/wiki/7%EC%9B%94_4%EC%9D%BC" title="7월 4일">7월 4일</a>, 6.1 km/s 의 속도로 <a href="/wiki/%ED%99%94%EC%84%B1" title="화성">화성</a> 대기권 돌입</li>
<li>이 이후, 탐사선은 반송파만 송신하고, 각 사건시에는 신호를 송신. <a href="/wiki/%EC%A7%80%EA%B5%AC" title="지구">지구</a>에서는 <a href="/wiki/%EB%8F%84%ED%94%8C%EB%9F%AC_%ED%9A%A8%EA%B3%BC" title="도플러 효과">도플러 효과</a>로 탐사선의 속도를 계측.</li>
<li>내열 셀에 의한 에어브레이킹으로 370 m/s 까지 감속</li>
<li>직경 약 11 m 의 낙하산을 전개해 68 m/s 까지 감속</li>
<li>20초 후, 내열 셀을 분리</li>
<li>20초 후, 착륙선이 에어로 셀에서 분리되어 20 m 의 줄에 매달림</li>

<p>2 화성일에(솔 2), 사진에서 에어백의 하나가 완전히 사그러들지 않은 것이 확인되어 로버가 <a href="/wiki/%ED%99%94%EC%84%B1" title="화성">화성</a> 표면에서 이동하는 데 장애가 될 것으로 예상되었다. <a href="/wiki/%EC%A7%80%EA%B5%AC" title="지구">지구</a>에서의 지시에 의해 착륙선의 패널을 하나 움직여 무사히 로버를 <a href="/wiki/%ED%99%94%EC%84%B1" title="화성">화성</a> 표면으로 이동시켰다.</p>
<h2><span class="mw-headline" id=".EC.9E.84.EB.AC.B4.EC.9D.98_.EC.A2.85.EB.A3.8C">임무의 종료</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%A7%88%EC%8A%A4_%ED%8C%A8%EC%8A%A4%ED%8C%8C%EC%9D%B8%EB%8D%94&amp;action=edit&amp;section=7" title="부분 편집: 임무의 종료">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>당초 약 1주일에서 1개월 정도 활동할 것이라고 생각되었던 로버와 착륙선이지만, 착륙으로부터 83 화성일 후인 <a href="/wiki/1997%EB%85%84" title="1997년">1997년</a> <a href="/wiki/9%EC%9B%94_27%EC%9D%BC" title="9월 27일">9월 27일</a> 10:23 <a href="/wiki/%ED%98%91%EC%A0%95_%EC%84%B8%EA%B3%84%EC%8B%9C" title="협정 세계시">UTC</a>에 패스파인더와의 통신이 두절될 때까지 약 3개월간 작동했다. <a href="/wiki/%EB%AF%B8%EA%B5%AD_%ED%95%AD%EA%B3%B5%EC%9A%B0%EC%A3%BC%EA%B5%AD" title="미국 항공우주국">NASA</a>는 <a href="/wiki/1998%EB%85%84" title="1998년">1998년</a> <a href="/wiki/3%EC%9B%94_10%EC%9D%BC" title="3월 10일">3월 10일</a>까지 통신 재개를 시도했지만, <a href="/wiki/10%EC%9B%94_7%EC%9D%BC" title="10월 7일">10월 7일</a>에 짧은 신호를 수신한 것 외에는 통신은 재개되지 않았다.</p>
<p>통신 두절의 원인은 불분명하지만, 로버는 문제가 없는 상태에서 착륙선이 저온에 의해 고장났을 가능성이 있다. 이 경우, 로버는 미리 <a href="/wiki/%EC%A7%80%EA%B5%AC" title="지구">지구</a>와의 통신이 두절되었을 경우 착륙선 근방으로 이동하지만, 다만 착륙선에 탑승해선 안 된다고 프로그램되어 있었는데, 이 때문에 착륙선 주위를 고장날 때까지 회전했을 것이라고 추측되었다.</p>
<h3><span class="mw-headline" id=".EC.B0.A9.EB.A5.99_.EC.9E.A5.EC.86.8C">착륙 장소</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%A7%88%EC%8A%A4_%ED%8C%A8%EC%8A%A4%ED%8C%8C%EC%9D%B8%EB%8D%94&amp;action=edit&amp;section=8" title="부분 편집: 착륙 장소">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
