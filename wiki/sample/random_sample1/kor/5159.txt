<p><b>마쓰다이라 다다나오</b>(<span style="font-size: smaller;"><a href="/wiki/%EC%9D%BC%EB%B3%B8%EC%96%B4" title="일본어">일본어</a>:</span> <span lang="ja"><span lang="ja"><ruby><rb>松平忠直</rb> <rp>(</rp><rt>まつだいら ただなお</rt><rp>)</rp></ruby></span></span>,<a href="/wiki/1595%EB%85%84" title="1595년">1595년</a> <a href="/wiki/7%EC%9B%94_16%EC%9D%BC" title="7월 16일">7월 16일</a>(<a href="/wiki/%EB%B6%84%EB%A1%9C%EC%BF%A0" title="분로쿠">분로쿠</a> 4년 <a href="/wiki/6%EC%9B%94_10%EC%9D%BC" title="6월 10일">6월 10일</a>) - <a href="/wiki/1650%EB%85%84" title="1650년">1650년</a> <a href="/wiki/10%EC%9B%94_5%EC%9D%BC" title="10월 5일">10월 5일</a>(<a href="/wiki/%EA%B2%8C%EC%9D%B4%EC%95%88" title="게이안">게이안</a> 3년 <a href="/wiki/9%EC%9B%94_10%EC%9D%BC" title="9월 10일">9월 10일</a>))는 일본 <a href="/wiki/%EC%97%90%EB%8F%84_%EC%8B%9C%EB%8C%80" title="에도 시대">에도 시대</a> 초기의 <a href="/wiki/%EB%8B%A4%EC%9D%B4%EB%AC%98" title="다이묘">다이묘</a>이다. <a href="/wiki/%EB%8F%84%EC%BF%A0%EA%B0%80%EC%99%80_%EC%9D%B4%EC%97%90%EC%95%BC%EC%8A%A4" title="도쿠가와 이에야스">도쿠가와 이에야스</a>(德川家康)의 차남 <a href="/wiki/%EC%9C%A0%ED%82%A4_%ED%9E%88%EB%8D%B0%EC%95%BC%EC%8A%A4" title="유키 히데야스">유키 히데야스</a>(結城秀康)의 장남으로, <a href="/wiki/%ED%9B%84%EC%BF%A0%EC%9D%B4_%EB%B2%88" title="후쿠이 번">후쿠이 번</a>(福井藩) 2대 번주이다.</p>
<h2><span class="mw-headline" id=".EC.83.9D.EC.95.A0">생애</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%A7%88%EC%93%B0%EB%8B%A4%EC%9D%B4%EB%9D%BC_%EB%8B%A4%EB%8B%A4%EB%82%98%EC%98%A4_(1595%EB%85%84)&amp;action=edit&amp;section=1" title="부분 편집: 생애">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p><a href="/wiki/1595%EB%85%84" title="1595년">1595년</a>(<a href="/wiki/%EB%B6%84%EB%A1%9C%EC%BF%A0" title="분로쿠">분로쿠</a> 4년) 출생하였다. 아버지는 <a href="/wiki/%EC%9C%A0%ED%82%A4_%ED%9E%88%EB%8D%B0%EC%95%BC%EC%8A%A4" title="유키 히데야스">유키 히데야스</a>, 어머니는 유키 히데야스의 측실 나카가와 씨(中川氏)이다. <a href="/wiki/1607%EB%85%84" title="1607년">1607년</a>(<a href="/wiki/%EA%B2%8C%EC%9D%B4%EC%B4%88" title="게이초">게이초</a> 12년) 아버지 유키 히데야스가 사망하자 12세의 나이에 <a href="/wiki/%EC%97%90%EC%B9%98%EC%A0%A0_%EA%B5%AD" title="에치젠 국">에치젠</a>(越前) <a href="/wiki/%ED%9B%84%EC%BF%A0%EC%9D%B4_%EB%B2%88" title="후쿠이 번">후쿠이 번</a> 75만 석의 다이묘가 되었다. <a href="/wiki/1611%EB%85%84" title="1611년">1611년</a>(<a href="/wiki/%EA%B2%8C%EC%9D%B4%EC%B4%88" title="게이초">게이초</a> 16년) 16세 때 숙부이자 2대 쇼군 <a href="/wiki/%EB%8F%84%EC%BF%A0%EA%B0%80%EC%99%80_%ED%9E%88%EB%8D%B0%ED%83%80%EB%8B%A4" title="도쿠가와 히데타다">도쿠가와 히데타다</a>(徳川秀忠)의 3녀 가쓰히메(勝姫)를 정실로 맞아들였다.</p>
<p><a href="/wiki/1614%EB%85%84" title="1614년">1614년</a>(<a href="/wiki/%EA%B2%8C%EC%9D%B4%EC%B4%88" title="게이초">게이초</a> 19년) <a href="/wiki/%EC%98%A4%EC%82%AC%EC%B9%B4_%EC%A0%84%ED%88%AC" title="오사카 전투">오사카 전투</a>(大坂の役)에 참여하였다. 첫 전쟁인 오사카 겨울 전투(大坂冬の陣)에서는 군졸들을 제대로 다루지 못하고 장수로서 미숙한 모습을 보여 할아버지인 <a href="/wiki/%EB%8F%84%EC%BF%A0%EA%B0%80%EC%99%80_%EC%9D%B4%EC%97%90%EC%95%BC%EC%8A%A4" title="도쿠가와 이에야스">도쿠가와 이에야스</a>에게 꾸지람을 들었으나, 이듬해 오사카 여름 전투(大坂夏の陣) 때에는 도요토미 측의 장수인 <a href="/wiki/%EC%82%AC%EB%82%98%EB%8B%A4_%EB%85%B8%EB%B6%80%EC%8B%9C%EA%B2%8C" title="사나다 노부시게">사나다 노부시게</a>(真田信繁) 등을 죽이고 <a href="/wiki/%EC%98%A4%EC%82%AC%EC%B9%B4_%EC%84%B1" title="오사카 성">오사카 성</a>(大坂城)에 제일 먼저 입성하는 등 공을 세웠다.</p>
<p>그러나 전쟁이 끝난 뒤 논공행상에 불만을 품으면서 점차 막부에 대해 반항적인 태도를 보였다. <a href="/wiki/1621%EB%85%84" title="1621년">1621년</a>(<a href="/wiki/%EA%B2%90%EB%82%98" title="겐나">겐나</a> 7년)에는 병을 핑계로 <a href="/wiki/%EC%97%90%EB%8F%84" title="에도">에도</a>로의 <a href="/wiki/%EC%B0%B8%EA%B7%BC%EA%B5%90%EB%8C%80" title="참근교대">참근교대</a> 의무를 게을리하였고<sup class="reference" id="cite_ref-1"><a href="#cite_note-1">[1]</a></sup>, <a href="/wiki/1622%EB%85%84" title="1622년">1622년</a>(<a href="/wiki/%EA%B2%90%EB%82%98" title="겐나">겐나</a> 8년)에는 쇼군의 딸이자 자신의 아내인 가쓰히메의 살해를 기획하였다 적발되었으며, 군사를 조련하고 가신들을 이유 없이 벌주는 등 기행을 거듭하였다. 결국 <a href="/wiki/1623%EB%85%84" title="1623년">1623년</a>(<a href="/wiki/%EA%B2%90%EB%82%98" title="겐나">겐나</a> 9년) 쇼군 히데타다에 의해 은거를 명령받고 동생 <a href="/wiki/%EB%A7%88%EC%93%B0%EB%8B%A4%EC%9D%B4%EB%9D%BC_%EB%8B%A4%EB%8B%A4%EB%A7%88%EC%82%AC_(1598%EB%85%84)" title="마쓰다이라 다다마사 (1598년)">다다마사</a>(忠昌)에게 다이묘 직을 물려준 뒤 출가하였다.</p>
<p><a href="/wiki/1650%EB%85%84" title="1650년">1650년</a> 향년 56세로 사망하였다.</p>
