<p><b>마쓰다이라 이에키요</b>(<span style="font-size: smaller;"><a href="/wiki/%EC%9D%BC%EB%B3%B8%EC%96%B4" title="일본어">일본어</a>:</span> <span lang="ja" xml:lang="ja">松平家清</span>, <a href="/wiki/1566%EB%85%84" title="1566년">1566년</a> ~ <a href="/wiki/1611%EB%85%84" title="1611년">1611년</a> <a href="/wiki/2%EC%9B%94_3%EC%9D%BC" title="2월 3일">2월 3일</a>)는 <a class="mw-redirect" href="/wiki/%EC%95%84%EC%A6%88%EC%B9%98%EB%AA%A8%EB%AA%A8%EC%95%BC%EB%A7%88_%EC%8B%9C%EB%8C%80" title="아즈치모모야마 시대">아즈치모모야마 시대</a>에서 <a href="/wiki/%EC%97%90%EB%8F%84_%EC%8B%9C%EB%8C%80" title="에도 시대">에도 시대</a> 전기에 걸쳐 <a href="/wiki/%EB%8F%84%EC%BF%A0%EA%B0%80%EC%99%80_%EC%94%A8" title="도쿠가와 씨">도쿠가와 씨</a>(徳川氏)의 가신으로 활약한 무장으로, <a href="/wiki/%EB%8B%A4%EC%BC%80%EB%85%B8%EC%95%BC_%EB%A7%88%EC%93%B0%EB%8B%A4%EC%9D%B4%EB%9D%BC_%EA%B0%80" title="다케노야 마쓰다이라 가">다케노야 마쓰다이라 가</a>의 6대 당주이다. 통칭은 요지로(与二郎).</p>
<h2><span class="mw-headline" id=".EC.83.9D.EC.95.A0">생애</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%A7%88%EC%93%B0%EB%8B%A4%EC%9D%B4%EB%9D%BC_%EC%9D%B4%EC%97%90%ED%82%A4%EC%9A%94&amp;action=edit&amp;section=1" title="부분 편집: 생애">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>다케노야 마쓰다이라 가의 무장으로서 <a href="/wiki/%EB%8F%84%EC%BF%A0%EA%B0%80%EC%99%80_%EC%9D%B4%EC%97%90%EC%95%BC%EC%8A%A4" title="도쿠가와 이에야스">도쿠가와 이에야스</a>(德川家康)에게 사관하여, 성인식 때에는 조부 이래의 돌림자인 「기요(清)」의 앞에, 이에야스로부터 「이에(家)」 한 자를 받아서 이에키요라고 칭했다. <a href="/wiki/1590%EB%85%84" title="1590년">1590년</a>(<a class="mw-redirect" href="/wiki/%EB%8D%B4%EC%87%BC_(1573%EB%85%84)" title="덴쇼 (1573년)">덴쇼</a> 18년), 이에야스가 <a class="mw-redirect" href="/wiki/%EA%B0%84%ED%86%A0" title="간토">간토</a>로 이봉되었기 때문에, 그때까지의 공을 인정받아 <a href="/wiki/%EB%AC%B4%EC%82%AC%EC%8B%9C_%EA%B5%AD" title="무사시 국">무사시 국</a> <a class="new" href="/w/index.php?title=%ED%95%98%EC%B9%98%EB%A7%8C%EC%95%BC%EB%A7%88_%EB%B2%88&amp;action=edit&amp;redlink=1" title="하치만야마 번 (없는 문서)">하치만야마</a>(八幡山)에 1만 석의 영지를 받았다. 이 구니가에(国替え)의 도중에 정실 덴케이인이 여아를 출산했지만, 결국 죽는다.</p>
<p><a href="/wiki/1600%EB%85%84" title="1600년">1600년</a>(<a href="/wiki/%EA%B2%8C%EC%9D%B4%EC%B4%88" title="게이초">게이초</a> 5년)의 <a href="/wiki/%EC%84%B8%ED%82%A4%EA%B0%80%ED%95%98%EB%9D%BC_%EC%A0%84%ED%88%AC" title="세키가하라 전투">세키가하라 전투</a> 때에는 <a href="/wiki/%EC%98%A4%EC%99%80%EB%A6%AC_%EA%B5%AD" title="오와리 국">오와리 국</a> <a href="/wiki/%EA%B8%B0%EC%9A%94%EC%8A%A4_%EC%84%B1" title="기요스 성">기요스 성</a>(清洲城)의 수비를 책임졌다. 전후, 이에야스로부터 그 공적을 인정받아, <a href="/wiki/%EB%AF%B8%EC%B9%B4%EC%99%80_%EA%B5%AD" title="미카와 국">미카와 국</a> <a class="mw-disambig" href="/wiki/%EC%9A%94%EC%8B%9C%EB%8B%A4_%EC%84%B1" title="요시다 성">요시다</a> 3만 석의 영지를 받아서, <a href="/wiki/%EC%9A%94%EC%8B%9C%EB%8B%A4_%EB%B2%88_(%EB%AF%B8%EC%B9%B4%EC%99%80_%EA%B5%AD)" title="요시다 번 (미카와 국)">미카와 요시다 번</a>의 초대 번주가 되었다.</p>
<p>1611년 2월 3일(게이초 15년 12월 21일)에 45세로 죽는다. 뒤는 적남 <a href="/wiki/%EB%A7%88%EC%93%B0%EB%8B%A4%EC%9D%B4%EB%9D%BC_%EB%8B%A4%EB%8B%A4%ED%82%A4%EC%9A%94" title="마쓰다이라 다다키요">다다키요</a>가 이었다.</p>
<p>법명은 葉雲全霜清宝院이며, 묘소는 <a href="/wiki/%EC%95%84%EC%9D%B4%EC%B9%98_%ED%98%84" title="아이치 현">아이치 현</a> <a href="/wiki/%EA%B0%80%EB%A7%88%EA%B3%A0%EB%A6%AC_%EC%8B%9C" title="가마고리 시">가마고리 시</a>의 덴에이지(全栄寺, 후에 류타이잔龍台山 덴케이인으로 개칭)에 있다.</p>
<h2><span class="mw-headline" id=".EA.B0.80.EA.B3.84">가계</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%A7%88%EC%93%B0%EB%8B%A4%EC%9D%B4%EB%9D%BC_%EC%9D%B4%EC%97%90%ED%82%A4%EC%9A%94&amp;action=edit&amp;section=2" title="부분 편집: 가계">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>이에키요는 5대 당주 <a href="/wiki/%EB%A7%88%EC%93%B0%EB%8B%A4%EC%9D%B4%EB%9D%BC_%EA%B8%B0%EC%9A%94%EB%AC%B4%EB%84%A4" title="마쓰다이라 기요무네">마쓰다이라 기요무네</a>(松平清宗)의 아들이며, 어머니는 <a href="/wiki/%ED%9B%84%EC%BD%94%EC%A6%88_%EB%A7%88%EC%93%B0%EB%8B%A4%EC%9D%B4%EB%9D%BC_%EA%B0%80" title="후코즈 마쓰다이라 가">후코즈 마쓰다이라 가</a> <a class="new" href="/w/index.php?title=%EB%A7%88%EC%93%B0%EB%8B%A4%EC%9D%B4%EB%9D%BC_%EC%9A%94%EC%8B%9C%EC%B9%B4%EA%B2%8C&amp;action=edit&amp;redlink=1" title="마쓰다이라 요시카게 (없는 문서)">요시카게</a>(好景)의 딸이다.</p>
<p>정실은 <a href="/wiki/%ED%9E%88%EC%82%AC%EB%A7%88%EC%93%B0_%EB%8F%84%EC%8B%9C%EC%B9%B4%EC%93%B0" title="히사마쓰 도시카쓰">히사마쓰 도시카쓰</a>(久松俊勝)의 딸로, <a href="/wiki/%EB%8F%84%EC%BF%A0%EA%B0%80%EC%99%80_%EC%9D%B4%EC%97%90%EC%95%BC%EC%8A%A4" title="도쿠가와 이에야스">이에야스</a>와는 아버지가 다른 남매간인 <a class="new" href="/w/index.php?title=%EB%8D%B4%EC%BC%80%EC%9D%B4%EC%9D%B8&amp;action=edit&amp;redlink=1" title="덴케이인 (없는 문서)">덴케이인</a>(天桂院)이다.</p>
<p>이에키요의 딸들은, <a class="mw-redirect" href="/wiki/%EA%B2%90%EB%A1%9C%EC%BF%A0_%EC%95%84%EC%BD%94_%EC%82%AC%EA%B1%B4" title="겐로쿠 아코 사건">겐로쿠 아코 사건</a>(元禄赤穂事件)으로 유명한 <a href="/wiki/%EC%95%84%EC%82%AC%EB%85%B8_%EB%82%98%EA%B0%80%EB%85%B8%EB%A6%AC" title="아사노 나가노리">아사노 나가노리</a>(浅野長矩)의 증조부인 <a class="new" href="/w/index.php?title=%EC%95%84%EC%82%AC%EB%85%B8_%EB%82%98%EA%B0%80%EC%8B%9C%EA%B2%8C&amp;action=edit&amp;redlink=1" title="아사노 나가시게 (없는 문서)">아사노 나가시게</a>(浅野長重), <a class="new" href="/w/index.php?title=%ED%98%BC%EB%8B%A4_%EC%95%BC%EC%8A%A4%EB%85%B8%EB%A6%AC&amp;action=edit&amp;redlink=1" title="혼다 야스노리 (없는 문서)">혼다 야스노리</a>(本多康紀), 후코즈의 <a class="new" href="/w/index.php?title=%EB%A7%88%EC%93%B0%EB%8B%A4%EC%9D%B4%EB%9D%BC_%EB%8B%A4%EB%8B%A4%ED%86%A0%EC%8B%9C_(%ED%9B%84%EC%BD%94%EC%A6%88_%EB%A7%88%EC%93%B0%EB%8B%A4%EC%9D%B4%EB%9D%BC_%EA%B0%80)&amp;action=edit&amp;redlink=1" title="마쓰다이라 다다토시 (후코즈 마쓰다이라 가) (없는 문서)">마쓰다이라 다다토시</a>(松平忠利)에게 시집갔다.</p>
