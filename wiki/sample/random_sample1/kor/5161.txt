<p><b>마쓰마에 다카히로</b>(<span style="font-size: smaller;"><a href="/wiki/%EC%9D%BC%EB%B3%B8%EC%96%B4" title="일본어">일본어</a>:</span> <span lang="ja"><span lang="ja"><ruby><rb>松前崇廣</rb> <rp>(</rp><rt>まつまえ たかひろ</rt><rp>)</rp></ruby></span></span> <small>마츠마에 타카히로<sup>[<a href="/wiki/%EC%9C%84%ED%82%A4%EB%B0%B1%EA%B3%BC:%EC%9D%BC%EB%B3%B8%EC%96%B4%EC%9D%98_%ED%95%9C%EA%B8%80_%ED%91%9C%EA%B8%B0" title="위키백과:일본어의 한글 표기">*</a>]</sup></small>, <a href="/wiki/1829%EB%85%84" title="1829년">1829년</a> <a href="/wiki/12%EC%9B%94_10%EC%9D%BC" title="12월 10일">12월 10일</a> ~ <a href="/wiki/1866%EB%85%84" title="1866년">1866년</a> <a href="/wiki/6%EC%9B%94_9%EC%9D%BC" title="6월 9일">6월 9일</a>)는 <a href="/wiki/%EB%A7%88%EC%93%B0%EB%A7%88%EC%97%90_%EB%B2%88" title="마쓰마에 번">마쓰마에 번</a>의 12대 번주로, 훗날 <a href="/wiki/%EB%A1%9C%EC%A3%BC" title="로주">로주</a>(老中)의 자리에도 올랐다.</p>
<p><a href="/wiki/%EB%A7%88%EC%93%B0%EB%A7%88%EC%97%90_%EB%B2%88" title="마쓰마에 번">마쓰마에 번</a>의 9대 번주 <a href="/wiki/%EB%A7%88%EC%93%B0%EB%A7%88%EC%97%90_%EC%95%84%ED%82%A4%ED%9E%88%EB%A1%9C" title="마쓰마에 아키히로">마쓰마에 아키히로</a>(松前章廣)의 여섯 번째 아들로, <a href="/wiki/%EB%A7%88%EC%93%B0%EB%A7%88%EC%97%90_%EC%84%B1" title="마쓰마에 성">후쿠야마 성</a>에서 태어났다. 어릴 적에 승마에 능했고, <a href="/wiki/%EB%82%9C%ED%95%99" title="난학">난학</a>, 영어, 병학 등을 배웠으며, 서양 문물에 관심이 많아 전자기기, 사진, 화학과 관련된 기기를 사용하는 등 서양통이었다. 사촌인 11대 번주 <a href="/wiki/%EB%A7%88%EC%93%B0%EB%A7%88%EC%97%90_%EB%A7%88%EC%82%AC%ED%9E%88%EB%A1%9C" title="마쓰마에 마사히로">마쓰마에 마사히로</a>(松前昌廣)가 병중일 때 그 아들 <a href="/wiki/%EB%A7%88%EC%93%B0%EB%A7%88%EC%97%90_%EB%85%B8%EB%A6%AC%ED%9E%88%EB%A1%9C_(13%EB%8C%80)" title="마쓰마에 노리히로 (13대)">노리히로</a>(德廣)가 아직 나이가 어리므로, <a href="/wiki/1849%EB%85%84" title="1849년">1849년</a> 아키히로는 은거하고 다카히로는 그 양자가 되어 12대 번주에 취임했다. 그해 <a href="/wiki/%EC%97%90%EB%8F%84" title="에도">에도</a>에 상경하여 <a href="/wiki/%EB%8F%84%EC%BF%A0%EA%B0%80%EC%99%80_%EC%9D%B4%EC%97%90%EC%9A%94%EC%8B%9C" title="도쿠가와 이에요시">도쿠가와 이에요시</a>를 알현하고 종5위 하 이즈노카미(伊豆守)에 임관하였다. 이때 북방 경비의 강화를 위해 축성할 것을 명령받아 <a href="/wiki/%EB%A7%88%EC%93%B0%EB%A7%88%EC%97%90_%EC%84%B1" title="마쓰마에 성">마쓰마에 성</a>을 지었다. <a href="/wiki/1853%EB%85%84" title="1853년">1853년</a> 완공된 이 성은 전통 방식으로 지어진 <a href="/wiki/%EC%97%90%EB%8F%84_%EC%8B%9C%EB%8C%80" title="에도 시대">에도 시대</a> 최후의 성이었다.</p>
<p><a href="/wiki/1854%EB%85%84" title="1854년">1854년</a>, 막부가 러시아에 대한 경비 강화를 위해 <a class="new" href="/w/index.php?title=%ED%95%98%EC%BD%94%EB%8B%A4%ED%85%8C_%EB%B6%80%EA%B5%90&amp;action=edit&amp;redlink=1" title="하코다테 부교 (없는 문서)">하코다테 부교</a>(箱館奉行)를 재설치하고 마쓰마에 번의 영지였던 하코다테 주변 마을과 <a href="/wiki/%EC%97%90%EC%A1%B0%EC%B9%98" title="에조치">에조치</a> 전역을 막부 직할령으로 삼았다. 대신 <a href="/wiki/%EB%A7%88%EC%93%B0%EB%A7%88%EC%97%90_%EB%B2%88" title="마쓰마에 번">마쓰마에 번</a>은 <a class="mw-redirect" href="/wiki/%EB%AC%B4%EC%93%B0%EB%85%B8%EC%BF%A0%EB%8B%88" title="무쓰노쿠니">무쓰</a> 야나가와(梁川)와 <a class="mw-redirect" href="/wiki/%EB%8D%B0%EC%99%80%EB%85%B8%EC%BF%A0%EB%8B%88" title="데와노쿠니">데와</a> 히가시네(東根), 오하나사와(尾花澤) 등 총합 4만 석의 영지를 지정받고, 매년 1만 8천 냥을 하사받게 되었다. 하지만 기존의 대(對) <a href="/wiki/%EC%97%90%EC%A1%B0" title="에조">에조</a> 교역권은 이후 상실되었고, 번의 재정은 궁핍해지게 되었다.</p>
<p><a href="/wiki/1863%EB%85%84" title="1863년">1863년</a>, 막부는 서양통인 다카히로를 <a class="new" href="/w/index.php?title=%EC%A7%80%EC%83%A4_%EB%B6%80%EA%B5%90&amp;action=edit&amp;redlink=1" title="지샤 부교 (없는 문서)">지샤 부교</a>(寺社奉行)에 임명하였고, 이듬해에는 로주 격 겸 육해군 총 부교(老中格兼陸海軍総奉行)를 거쳐 <a href="/wiki/%EB%A1%9C%EC%A3%BC" title="로주">로주</a>(老中)에 발탁되었다. 이 해에 마쓰마에 일대의 8개 마을을 영지로 돌려받았으며, 지원금 700냥이 삭감되었다. <a href="/wiki/1865%EB%85%84" title="1865년">1865년</a>에는 육군 겸 해군 총재가 되었다. 이 무렵 영국, 미국, 프랑스, 네덜란드 4개국의 <a href="/wiki/%ED%9A%A8%EA%B3%A0_%ED%98%84" title="효고 현">효고</a>(兵庫) 개항 요구에 대하여, 로주 <a href="/wiki/%EC%95%84%EB%B2%A0_%EB%A7%88%EC%82%AC%ED%86%A0" title="아베 마사토">아베 마사토</a>와 함께 조정의 칙허를 받지 않고 효고 개항을 결정하였다. 이로 인해 조정으로부터 관위 박탈, 근신의 칙령이 내려졌고, 막부는 두 로주를 면직하고 근신을 명했다. 다카히로는 <a href="/wiki/1866%EB%85%84" title="1866년">1866년</a> 마쓰마에로 돌아왔으나 열병으로 38세의 나이로 사망했다. 양자 <a href="/wiki/%EB%A7%88%EC%93%B0%EB%A7%88%EC%97%90_%EB%85%B8%EB%A6%AC%ED%9E%88%EB%A1%9C_(13%EB%8C%80)" title="마쓰마에 노리히로 (13대)">노리히로</a>가 후사를 이었다.</p>
