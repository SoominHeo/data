<p><b>마쓰오 바쇼</b>(<span style="font-size: smaller;"><a href="/wiki/%EC%9D%BC%EB%B3%B8%EC%96%B4" title="일본어">일본어</a>:</span> <span lang="ja"><span lang="ja"><ruby><rb>松尾芭蕉</rb> <rp>(</rp><rt>まつお ばしょう</rt><rp>)</rp></ruby></span></span> <small>마츠오 바쇼<sup>[<a href="/wiki/%EC%9C%84%ED%82%A4%EB%B0%B1%EA%B3%BC:%EC%9D%BC%EB%B3%B8%EC%96%B4%EC%9D%98_%ED%95%9C%EA%B8%80_%ED%91%9C%EA%B8%B0" title="위키백과:일본어의 한글 표기">*</a>]</sup></small>, <a href="/wiki/1644%EB%85%84" title="1644년">1644년</a> ~ <a href="/wiki/1694%EB%85%84" title="1694년">1694년</a> <a href="/wiki/11%EC%9B%94_28%EC%9D%BC" title="11월 28일">11월 28일</a>)는, 일본 <a href="/wiki/%EC%97%90%EB%8F%84_%EC%8B%9C%EB%8C%80" title="에도 시대">에도 시대</a>(江戶時代) 전기의 <a href="/wiki/%ED%95%98%EC%9D%B4%EC%BF%A0" title="하이쿠">하이쿠</a> <a href="/wiki/%EC%8B%9C%EC%9D%B8" title="시인">시인</a>으로 에도 시대 전기 하이카이(俳諧)<sup class="reference" id="cite_ref-.EA.B0.81.EC.A3.BC_1-0"><a href="#cite_note-.EA.B0.81.EC.A3.BC-1">[1]</a></sup>로 널리 알려진 사람이다. 어렸을 때의 이름은 긴자쿠(金作)이며, 통칭은 진시치로(甚七郎) 또는 진시로(甚四郎)이다. 본명은 주고에몬 무네후사(忠右衛門宗房)로 처음에는 이 「무네후사」를 필명으로 사용하다가 그 뒤에 도세이(桃青), 마지막으로 바쇼라는 필명을 썼다.<sup class="reference" id="cite_ref-2"><a href="#cite_note-2">[2]</a></sup></p>
<p>에도 전기의 가인(歌人)이자 학자였던 기타무라 기긴(北村季吟)의 문하에서 하이쿠를 배웠다. 「쇼후(蕉風)」라 불리는 예술성 높은 문장 경향을 확립하여 후세에 「하이쿠의 성인(俳聖)」으로서 세계적으로 그 이름이 알려진 일본 역사상 최고의 하이쿠 시인의 한 사람이다.</p>
<p>제자 가와이 소라(河合曾良)와 함께 <a href="/wiki/%EA%B2%90%EB%A1%9C%EC%BF%A0" title="겐로쿠">겐로쿠</a>(元禄) 2년 3월 27일(<a href="/wiki/1689%EB%85%84" title="1689년">1689년</a> 5월 16일)에 <a href="/wiki/%EC%97%90%EB%8F%84" title="에도">에도</a>(江戶)를 비롯한 도호쿠(東北), 호쿠리쿠(北陸) 지방을 돌며 <a href="/wiki/%EA%B8%B0%ED%9B%84_%EC%8B%9C" title="기후 시">기후</a>(岐阜)의 <a href="/wiki/%EC%98%A4%EA%B0%80%ED%82%A4_%EC%8B%9C" title="오가키 시">오가키</a>(大垣)까지 여행한 기행문 『오쿠노 호소미치(おくのほそ道)』를 남겼다.</p>

<h2><span class="mw-headline" id=".EC.83.9D.EC.95.A0">생애</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%A7%88%EC%93%B0%EC%98%A4_%EB%B0%94%EC%87%BC&amp;action=edit&amp;section=1" title="부분 편집: 생애">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p><a href="/wiki/1644%EB%85%84" title="1644년">1644년</a> <a href="/wiki/%EC%9D%B4%EA%B0%80_%EA%B5%AD" title="이가 국">이가 국</a>(지금의 <a href="/wiki/%EB%AF%B8%EC%97%90_%ED%98%84" title="미에 현">미에 현</a>(三重県) <a href="/wiki/%EC%9D%B4%EA%B0%80_%EC%8B%9C" title="이가 시">이가 시</a>(伊賀市)) 농민 집안의 아들로 태어났다. 이름은 무네후사(宗房)이다. 처음에는 무네후사를 배호로 사용하다가 그 후 모모아오, 마지막으로 바쇼라고 고쳤다. <a href="/wiki/1662%EB%85%84" title="1662년">1662년</a> <a href="/wiki/%EC%93%B0_%EB%B2%88" title="쓰 번">쓰 번</a>의 번주 <a class="mw-disambig" href="/wiki/%EB%8F%84%EB%8F%84_%EB%8B%A4%EC%B9%B4%EC%93%B0%EA%B5%AC" title="도도 다카쓰구">도도 다카쓰구</a>의 조카 <a class="new" href="/w/index.php?title=%EB%8F%84%EB%8F%84_%EC%9A%94%EC%8B%9C%ED%83%80%EB%8B%A4&amp;action=edit&amp;redlink=1" title="도도 요시타다 (없는 문서)">요시타다</a>에게 봉사했고, 요시타다가 세상을 떠난 <a href="/wiki/1666%EB%85%84" title="1666년">1666년</a>까지 섬기는 한편 하이쿠를 배웠다.</p>
<p>쇼후로 불리는 예술성 높은 문장 경향을 확립하여 하이쿠의 명인으로 불린다.</p>
<h2><span class="mw-headline" id=".EC.9E.91.ED.92.88">작품</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%A7%88%EC%93%B0%EC%98%A4_%EB%B0%94%EC%87%BC&amp;action=edit&amp;section=2" title="부분 편집: 작품">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>바쇼가 제자인 가와이 소라를 동반해 <a href="/wiki/%EA%B2%90%EB%A1%9C%EC%BF%A0" title="겐로쿠">겐로쿠</a> 2년 3월 27일(<a href="/wiki/1689%EB%85%84" title="1689년">1689년</a> 5월 16일)에 <a href="/wiki/%EC%97%90%EB%8F%84" title="에도">에도</a>를 출발하여 <a class="mw-redirect" href="/wiki/%EB%8F%84%ED%98%B8%EC%BF%A0" title="도호쿠">도호쿠</a>, <a class="mw-redirect" href="/wiki/%ED%98%B8%EC%BF%A0%EB%A6%AC%EC%BF%A0" title="호쿠리쿠">호쿠리쿠</a>를 돌아 다녀 <a href="/wiki/%EA%B8%B0%ED%9B%84_%ED%98%84" title="기후 현">기후 현</a> <a class="new" href="/w/index.php?title=%EC%98%A4%EC%B9%B4%EC%9D%B4&amp;action=edit&amp;redlink=1" title="오카이 (없는 문서)">오카이</a>까지 여행하면서 쓴 기행문 『오쿠노 호소미치』가 있다.</p>
