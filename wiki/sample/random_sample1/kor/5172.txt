<p><b>마에다 도시마스</b>(前田利益 <a href="/wiki/1543%EB%85%84" title="1543년">1543년</a> ~ <a href="/wiki/1612%EB%85%84" title="1612년">1612년</a>)는 <a href="/wiki/%EC%84%BC%EA%B3%A0%EC%BF%A0_%EC%8B%9C%EB%8C%80" title="센고쿠 시대">센고쿠 시대</a> 말기 ~ <a href="/wiki/%EC%97%90%EB%8F%84_%EC%8B%9C%EB%8C%80" title="에도 시대">에도 시대</a> 초기의 <a href="/wiki/%EB%AC%B4%EC%9E%A5" title="무장">무장</a>이다. <a href="/wiki/%EB%A7%88%EC%97%90%EB%8B%A4_%EB%8F%84%EC%8B%9C%EC%9D%B4%EC%97%90" title="마에다 도시이에">마에다 도시이에</a>(前田利家)의 의붓조카. 어린 시절 <a class="new" href="/w/index.php?title=%EB%8B%A4%ED%82%A4%EA%B0%80%EC%99%80_%EC%94%A8&amp;action=edit&amp;redlink=1" title="다키가와 씨 (없는 문서)">다키가와 씨</a>에서 <a class="new" href="/w/index.php?title=%EB%A7%88%EC%97%90%EB%8B%A4_%EC%94%A8&amp;action=edit&amp;redlink=1" title="마에다 씨 (없는 문서)">마에다 씨</a>의 양자로 들어갔고, 생부는 <a href="/wiki/%EB%8B%A4%ED%82%A4%EA%B0%80%EC%99%80_%EA%B0%80%EC%A6%88%EB%A7%88%EC%8A%A4" title="다키가와 가즈마스">다키가와 가즈마스</a>라는 설이 유력하다. 그는 <b>마에다 게이지</b>(前田慶次)/<b>게이지로</b>(慶次郎)라는 다른 이름으로 불렸고 호탕하고 박력있는 반면 언제나 제멋대로이면서도 그릇된 행동을 하지 않았다.</p>
<p>시(노래), 악기연주, 검무, 검술을 두루두루 잘 했고 창술의 달인이기도 했다. 그리하여 마에다 도시마스에게 '꽃피는 경차', '천하제일창' 등의 별명이 붙었다. 그 용맹이 일본 내에서는 삼국지의 <a href="/wiki/%EC%97%AC%ED%8F%AC" title="여포">여포</a>(呂布)와 비견되기도 한다.</p>
<h2><span class="mw-headline" id=".EC.83.9D.EC.95.A0">생애</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%A7%88%EC%97%90%EB%8B%A4_%EB%8F%84%EC%8B%9C%EB%A7%88%EC%8A%A4&amp;action=edit&amp;section=1" title="부분 편집: 생애">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>어렸을 때 <a href="/wiki/%EB%A7%88%EC%97%90%EB%8B%A4_%EB%8F%84%EC%8B%9C%EC%9D%B4%EC%97%90" title="마에다 도시이에">마에다 도시이에</a>의 형인 <a class="new" href="/w/index.php?title=%EB%A7%88%EC%97%90%EB%8B%A4_%EB%8F%84%EC%8B%9C%ED%9E%88%EC%82%AC&amp;action=edit&amp;redlink=1" title="마에다 도시히사 (없는 문서)">마에다 도시히사</a>의 양자로 들어갔다. 어떤 사건을 계기로 <a href="/wiki/%EB%8B%A4%ED%85%8C_%EB%A7%88%EC%82%AC%EB%AC%B4%EB%84%A4" title="다테 마사무네">다테 마사무네</a>(伊達政宗)를 혼내주고 그와 교우관계를 맺었고 항상 형을 구박하고 무시하는 <a href="/wiki/%EB%8B%A4%ED%85%8C_%EB%A7%88%EC%82%AC%EB%AF%B8%EC%B9%98" title="다테 마사미치">고지로</a>(小次郎)를 다른 이유를 잡아서 참수하기도 했다. 만나는 어떠한 이들과도 교분을 쌓으며 풍류를 즐겼다.</p>
<p>그의 천부적 재능을 눈여겨 본 <a href="/wiki/%EB%8F%84%EC%9A%94%ED%86%A0%EB%AF%B8_%ED%9E%88%EB%8D%B0%EC%9A%94%EC%8B%9C" title="도요토미 히데요시">도요토미 히데요시</a>(豐臣秀吉)는 수단 방법 가리지 않고 마에다 도시마스를 영입하려 했으나 마에다 도시마스는 끝내 이 영입을 거절했다. 마에다 문중의 번주가 마에다 도시이에로 결정되자 마에다 도시마스의 고쿠다카는 <b>0석</b>이 되어버렸고 이 때문에 마에다 도시마스는 <a href="/wiki/%EC%9E%84%EC%A7%84%EC%99%9C%EB%9E%80" title="임진왜란">임진왜란</a>을 전후하여 막역한 친우인 <a href="/wiki/%EB%82%98%EC%98%A4%EC%97%90_%EA%B0%80%EB%84%A4%EC%93%B0%EA%B5%AC" title="나오에 가네쓰구">나오에 가네쓰구</a>를 따라 '고쿠조인 소노사이'라는 가명으로 <a href="/wiki/%EC%9A%B0%EC%97%90%EC%8A%A4%EA%B8%B0_%EA%B0%80%EA%B2%8C%EC%B9%B4%EC%93%B0" title="우에스기 가게카쓰">우에스기 가게카쓰</a>(上杉景勝)의 가신이 되었다.</p>
<p><a href="/wiki/%EC%84%B8%ED%82%A4%EA%B0%80%ED%95%98%EB%9D%BC_%EC%A0%84%ED%88%AC" title="세키가하라 전투">세키가하라 전투</a>가 발발하자 우에스기 가게카쓰는 서군에 가담했는데 그 서군이 패배하자 우에스기 가게카쓰는 항복하여 동군의 맹주인 <a href="/wiki/%EB%8F%84%EC%BF%A0%EA%B0%80%EC%99%80_%EC%9D%B4%EC%97%90%EC%95%BC%EC%8A%A4" title="도쿠가와 이에야스">도쿠가와 이에야스</a>(德川家康)에 의해 봉지(封地)를 삭감당하게 되었다. 그 사건 이후 도시마스는 자신의 이름을 버리고 죽을 때까지 여기저기를 떠돌며 낭인으로서 은둔생활을 하게 된다.</p>
<h2><span class="mw-headline" id=".EC.9D.B8.EB.AC.BC.ED.8F.89">인물평</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%A7%88%EC%97%90%EB%8B%A4_%EB%8F%84%EC%8B%9C%EB%A7%88%EC%8A%A4&amp;action=edit&amp;section=2" title="부분 편집: 인물평">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>마에다 도시마스는 천하를 뒤엎을 만큼의 천부적 재능을 지녔음에도 불구하고 권력에 대한 욕심은 전혀 없었다. 그래서 마에다 도시마스는 다이묘라면 누구나 탐내는 인물이었다. 누구를 만나든 그냥 '친구'일 뿐이고 그 성격탓에 그 당시의 일본인이었다면 누구나 어렵게 생각하는 거대번주 <a href="/wiki/%EB%8B%A4%ED%85%8C_%EB%A7%88%EC%82%AC%EB%AC%B4%EB%84%A4" title="다테 마사무네">다테 마사무네</a>를 면전에서 꾸짖는가 하면 <a href="/wiki/%EB%8F%84%EC%9A%94%ED%86%A0%EB%AF%B8_%ED%9E%88%EB%8D%B0%EC%9A%94%EC%8B%9C" title="도요토미 히데요시">도요토미 히데요시</a>의 생일날 <a href="/wiki/%EB%8F%84%EC%9A%94%ED%86%A0%EB%AF%B8_%ED%9E%88%EB%8D%B0%EC%9A%94%EC%8B%9C" title="도요토미 히데요시">도요토미 히데요시</a>를 찾아가서 원숭이춤을 추며 조롱하는 한편 자신을 죽이러 온 <a href="/wiki/%EA%B5%AC%EB%85%B8%EC%9D%B4%EC%B9%98" title="구노이치">구노이치</a>를 꾀어 사랑에 빠지기도 했다.</p>
