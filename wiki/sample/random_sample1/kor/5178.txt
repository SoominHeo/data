<p><b>마에조라 역</b>(<span style="font-size: smaller;"><a href="/wiki/%EC%9D%BC%EB%B3%B8%EC%96%B4" title="일본어">일본어</a>:</span> <span lang="ja"><span lang="ja"><ruby><rb>前空駅</rb> <rp>(</rp><rt>まえぞらえき</rt><rp>)</rp></ruby></span></span> <small>마에조라에키<sup>[<a href="/wiki/%EC%9C%84%ED%82%A4%EB%B0%B1%EA%B3%BC:%EC%9D%BC%EB%B3%B8%EC%96%B4%EC%9D%98_%ED%95%9C%EA%B8%80_%ED%91%9C%EA%B8%B0" title="위키백과:일본어의 한글 표기">*</a>]</sup></small>)은 <a href="/wiki/%EC%9D%BC%EB%B3%B8" title="일본">일본</a> <a href="/wiki/%ED%9E%88%EB%A1%9C%EC%8B%9C%EB%A7%88_%ED%98%84" title="히로시마 현">히로시마 현</a> <a href="/wiki/%ED%95%98%EC%93%B0%EC%B9%B4%EC%9D%B4%EC%B9%98_%EC%8B%9C" title="하쓰카이치 시">하쓰카이치 시</a>에 있는 <a class="mw-redirect" href="/wiki/%EC%84%9C%EC%9D%BC%EB%B3%B8_%EC%97%AC%EA%B0%9D%EC%B2%A0%EB%8F%84" title="서일본 여객철도">서일본 여객철도</a> <a href="/wiki/%EC%82%B0%EC%9A%94_%EB%B3%B8%EC%84%A0" title="산요 본선">산요 본선</a>의 <a href="/wiki/%EC%B2%A0%EB%8F%84%EC%97%AD" title="철도역">역</a>이다. <a class="mw-redirect" href="/wiki/%EC%83%81%EB%8C%80%EC%8B%9D_%EC%8A%B9%EA%B0%95%EC%9E%A5" title="상대식 승강장">상대식 승강장</a> 2면 2선 구조를 지닌 <a href="/wiki/%EC%B2%A0%EB%8F%84%EC%97%AD" title="철도역">지상역</a>이자 <a href="/wiki/%EA%B5%90%EC%83%81%EC%97%AD" title="교상역">교상역</a>이며, <a href="/wiki/%EB%8B%88%EC%8B%9C%ED%9E%88%EB%A1%9C%EC%8B%9C%EB%A7%88_%EC%97%AD" title="니시히로시마 역">니시히로시마 역</a>이 관리하고 있다.</p>
