<p><b>2018년 동계 올림픽의 경기장</b>은 <a href="/wiki/%EB%8C%80%ED%95%9C%EB%AF%BC%EA%B5%AD" title="대한민국">대한민국</a> <a href="/wiki/%ED%8F%89%EC%B0%BD%EA%B5%B0" title="평창군">평창군</a>의 올림픽 주 경기장인 <a href="/wiki/%ED%8F%89%EC%B0%BD_%EC%98%AC%EB%A6%BC%ED%94%BD_%EC%8A%A4%ED%83%80%EB%94%94%EC%9B%80" title="평창 올림픽 스타디움">평창 올림픽 스타디움</a>에서 30분대로 이동이 가능하도록 배치해 선수와 경기 중심 올림픽 대회를 개최할 예정이다. 경기 지역은 설상 경기가 개최되는 <a href="/wiki/%EB%8C%80%ED%95%9C%EB%AF%BC%EA%B5%AD" title="대한민국">대한민국</a> <a href="/wiki/%EA%B0%95%EC%9B%90%EB%8F%84_(%EB%82%A8)" title="강원도 (남)">강원도</a> <a href="/wiki/%ED%8F%89%EC%B0%BD%EA%B5%B0" title="평창군">평창군</a>과 강원도 <a href="/wiki/%EC%A0%95%EC%84%A0%EA%B5%B0" title="정선군">정선군</a> 일대의 <b>마운틴 클러스터</b>와 빙상 경기가 개최되는 강원도 <a href="/wiki/%EA%B0%95%EB%A6%89%EC%8B%9C" title="강릉시">강릉시</a>의 <b>코스탈 클러스터</b>로 나뉜다. <a class="mw-redirect" href="/wiki/%ED%8F%89%EC%B0%BD_%EB%8F%99%EA%B3%84_%EC%98%AC%EB%A6%BC%ED%94%BD" title="평창 동계 올림픽">평창 동계 올림픽</a>의 모든 경기는 총 12개의 경기장에서 열리게 된다.</p>
