<h2><span class="mw-headline" id=".ED.81.B4.EB.9F.BD_.EC.BB.A4.EB.A6.AC.EC.96.B4">클럽 커리어</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%A7%88%EC%9A%B0%EB%A1%9C_%EC%9D%B4%EC%B9%B4%EB%A5%B4%EB%94%94&amp;action=edit&amp;section=1" title="부분 편집: 클럽 커리어">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<h3><span class="mw-headline" id=".EC.B4.88.EA.B8.B0_.EC.BB.A4.EB.A6.AC.EC.96.B4">초기 커리어</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%A7%88%EC%9A%B0%EB%A1%9C_%EC%9D%B4%EC%B9%B4%EB%A5%B4%EB%94%94&amp;action=edit&amp;section=2" title="부분 편집: 초기 커리어">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>이카르디는 아르헨티나에서 태어났지만 그의 가족들과 함께 6살에 스페인으로 이민을 갔다. 이카르디는 <a class="new" href="/w/index.php?title=UD_%EB%B2%A0%EC%B9%9C%EB%8B%A4%EB%A6%AC%EC%98%A4&amp;action=edit&amp;redlink=1" title="UD 베친다리오 (없는 문서)">UD 베친다리오</a>에서 축구를 시작했으며 유스 생활을 하는동안 500골이 넘는 득점을 해냈다.</p>
<h3><span class="mw-headline" id=".EB.B0.94.EB.A5.B4.EC.85.80.EB.A1.9C.EB.82.98">바르셀로나</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%A7%88%EC%9A%B0%EB%A1%9C_%EC%9D%B4%EC%B9%B4%EB%A5%B4%EB%94%94&amp;action=edit&amp;section=3" title="부분 편집: 바르셀로나">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>2008년, <a href="/wiki/FC_%EB%B0%94%EB%A5%B4%EC%85%80%EB%A1%9C%EB%82%98" title="FC 바르셀로나">바르셀로나</a>와 <a class="mw-redirect" href="/wiki/%EB%A0%88%EC%95%8C_%EB%A7%88%EB%93%9C%EB%A6%AC%EB%93%9C" title="레알 마드리드">레알 마드리드</a>, <a href="/wiki/%EB%B0%9C%EB%A0%8C%EC%8B%9C%EC%95%84_CF" title="발렌시아 CF">발렌시아</a>, <a href="/wiki/%EC%84%B8%EB%B9%84%EC%95%BC_FC" title="세비야 FC">세비야</a>, <a class="mw-redirect" href="/wiki/%EC%97%90%EC%8A%A4%ED%8C%8C%EB%87%B0" title="에스파뇰">에스파뇰</a>, <a class="mw-redirect" href="/wiki/%EB%8D%B0%ED%8F%AC%EB%A5%B4%ED%8B%B0%EB%B3%B4_%EB%9D%BC_%EC%BD%94%EB%A3%A8%EB%83%90" title="데포르티보 라 코루냐">데포르티보 라 코루냐</a>, <a class="mw-redirect" href="/wiki/%EC%95%84%EC%8A%A4%EB%82%A0" title="아스날">아스날</a>, <a href="/wiki/%EB%A6%AC%EB%B2%84%ED%92%80_FC" title="리버풀 FC">리버풀 FC</a> 등등의 많은 클럽들이 이카르디의 영입전쟁에 뛰어들었다. 바르셀로나가 영입전쟁에서 최종승리하여 2013년까지 이카르디와 계약을 성공하였고, 2008-09시즌에 U-17팀으로 승격하였다.</p>
<h3><span class="mw-headline" id=".EC.82.BC.ED.94.84.EB.8F.84.EB.A6.AC.EC.95.84">삼프도리아</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%A7%88%EC%9A%B0%EB%A1%9C_%EC%9D%B4%EC%B9%B4%EB%A5%B4%EB%94%94&amp;action=edit&amp;section=4" title="부분 편집: 삼프도리아">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>2011년 1월 11일, <a href="/wiki/UC_%EC%82%BC%ED%94%84%EB%8F%84%EB%A6%AC%EC%95%84" title="UC 삼프도리아">UC 삼프도리아</a>는 시즌이 끝날때까지만 이카르디를 임대하기로 계약했다.<sup class="reference" id="cite_ref-1"><a href="#cite_note-1">[1]</a></sup><sup class="reference" id="cite_ref-a_2-0"><a href="#cite_note-a-2">[2]</a></sup> 프리마베라 팀에서 19경기 13골을 기록하여 6개월간의 임대는 대성공적이었다. 삼프도리아는 40만 유로의 옵션을 더하여 이카르디를 2011년 7월에 3년 계약을 성공시켰다.<sup class="reference" id="cite_ref-a_2-1"><a href="#cite_note-a-2">[2]</a></sup><sup class="reference" id="cite_ref-3"><a href="#cite_note-3">[3]</a></sup> 2012년 5월 12일에 그는 <a href="/wiki/SS_%EC%9C%A0%EB%B2%A0_%EC%8A%A4%ED%83%80%EB%B9%84%EC%95%84" title="SS 유베 스타비아">SS 유베 스타비아</a>를 상대로 첫 대뷔전을 치뤘고 75분에 교체 출전하였다. 10분뒤 이카르디는 그의 첫 1군 무대와 삼프도리아에서의 첫 대뷔골을 터트려 2:1로 승리하였다.<sup class="reference" id="cite_ref-4"><a href="#cite_note-4">[4]</a></sup> 2013년 1월 6일에는 <a href="/wiki/%EC%9C%A0%EB%B2%A4%ED%88%AC%EC%8A%A4_%EC%8A%A4%ED%83%80%EB%94%94%EC%9B%80" title="유벤투스 스타디움">유벤투스 스타디움</a>에서 <a class="mw-redirect" href="/wiki/%EC%A7%80%EC%95%88%EB%A3%A8%EC%9D%B4%EC%A7%80_%EB%B6%80%ED%8F%B0" title="지안루이지 부폰">지안루이지 부폰</a>이 지키고 있는 <a href="/wiki/%EC%9C%A0%EB%B2%A4%ED%88%AC%EC%8A%A4_FC" title="유벤투스 FC">유벤투스 FC</a>를 상대로 2골을 넣어 팀을 승리로 이끌었고 1월 27일에는 <a class="mw-redirect" href="/wiki/%ED%8E%98%EC%8A%A4%EC%B9%B4%EB%9D%BC_%EC%B9%BC%EC%B4%88" title="페스카라 칼초">페스카라 칼초</a>를 상대로 혼자 4골을 득점했다. 소속팀 <a href="/wiki/UC_%EC%82%BC%ED%94%84%EB%8F%84%EB%A6%AC%EC%95%84" title="UC 삼프도리아">UC 삼프도리아</a>는 50년만에 해당 소속선수가 1군 무대에서 4골을 넣는 대기록을 세우기도 했다.<sup class="reference" id="cite_ref-5"><a href="#cite_note-5">[5]</a></sup> 2013년 5월 19일 삼프도리아의 홈 경기에서도 1골을 넣으며 팀을 3대2 승리로 이끌었다. <a href="/wiki/%EC%9C%A0%EB%B2%A4%ED%88%AC%EC%8A%A4_FC" title="유벤투스 FC">유벤투스 FC</a>는 이 시즌에 27승 6무 5패로 우승을 달성했지만 <a href="/wiki/UC_%EC%82%BC%ED%94%84%EB%8F%84%EB%A6%AC%EC%95%84" title="UC 삼프도리아">UC 삼프도리아</a>한테는 이카르디의 맹활약 때문에 한번도 이기지 못한 것이다.</p>
<h3><span class="mw-headline" id=".EC.9D.B8.ED.85.8C.EB.A5.B4.EB.82.98.EC.B9.98.EC.98.A4.EB.82.A0.EB.A0.88">인테르나치오날레</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%A7%88%EC%9A%B0%EB%A1%9C_%EC%9D%B4%EC%B9%B4%EB%A5%B4%EB%94%94&amp;action=edit&amp;section=5" title="부분 편집: 인테르나치오날레">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>2013년 7월 9일, 이카르디는 <a href="/wiki/FC_%EC%9D%B8%ED%85%8C%EB%A5%B4%EB%82%98%EC%B9%98%EC%98%A4%EB%82%A0%EB%A0%88_%EB%B0%80%EB%9D%BC%EB%85%B8" title="FC 인테르나치오날레 밀라노">FC 인테르나치오날레 밀라노</a>로 이적했다. 인테르 이적후 2013년 9월 15일 첫 번째 <a class="new" href="/w/index.php?title=%EC%9D%B4%ED%83%88%EB%A6%AC%EC%95%84_%EB%8D%94%EB%B9%84&amp;action=edit&amp;redlink=1" title="이탈리아 더비 (없는 문서)">이탈리아 더비</a>에서 후반 68분 교체 출전 후 5분만에 선취 골을 넣었다. 그러나 불과 2분뒤에 바로 동점골을 허용하여 1-1로 비기긴 했지만, 이 경기 후 이카르디는 진정한 유벤투스 킬러임을 증명하였다. 2014/2015 시즌 22골로 세리에A 득점왕</p>
<h2><span class="mw-headline" id=".EA.B5.AD.EA.B0.80.EB.8C.80.ED.91.9C.ED.8C.80_.EA.B2.BD.EB.A0.A5">국가대표팀 경력</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%A7%88%EC%9A%B0%EB%A1%9C_%EC%9D%B4%EC%B9%B4%EB%A5%B4%EB%94%94&amp;action=edit&amp;section=6" title="부분 편집: 국가대표팀 경력">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>8월 12일, 이카르디는 <a class="new" href="/w/index.php?title=%EB%8F%85%EC%9D%BC_U-20_%EC%B6%95%EA%B5%AC_%EA%B5%AD%EA%B0%80%EB%8C%80%ED%91%9C%ED%8C%80&amp;action=edit&amp;redlink=1" title="독일 U-20 축구 국가대표팀 (없는 문서)">독일</a>을 상대로 <a href="/wiki/%EC%95%84%EB%A5%B4%ED%97%A8%ED%8B%B0%EB%82%98_U-20_%EC%B6%95%EA%B5%AC_%EA%B5%AD%EA%B0%80%EB%8C%80%ED%91%9C%ED%8C%80" title="아르헨티나 U-20 축구 국가대표팀">아르헨티나 U-20 축구 국가대표팀</a> 데뷔전을 치렀다.</p>
<p>2013년 2월, 이카르디는 <a href="/wiki/%EC%95%84%EB%A5%B4%ED%97%A8%ED%8B%B0%EB%82%98_%EC%B6%95%EA%B5%AC_%EA%B5%AD%EA%B0%80%EB%8C%80%ED%91%9C%ED%8C%80" title="아르헨티나 축구 국가대표팀">아르헨티나 성인팀</a>에 뛰기를 원한다는것을 밝히며, "나는 아르헨티나인이며, 나의 조국의 유니폼을 입는걸 항상 꿈꿔왔다. 사실 그일을 하는건 나에게 최대의 일 일 것이다."<sup class="reference" id="cite_ref-6"><a href="#cite_note-6">[6]</a></sup></p>
<p>2013년 10월 15일, <a href="/wiki/%EC%9A%B0%EB%A3%A8%EA%B3%BC%EC%9D%B4_%EC%B6%95%EA%B5%AC_%EA%B5%AD%EA%B0%80%EB%8C%80%ED%91%9C%ED%8C%80" title="우루과이 축구 국가대표팀">우루과이</a>에게 2-3으로 패한 <a href="/wiki/FIFA_%EC%9B%94%EB%93%9C%EC%BB%B5" title="FIFA 월드컵">FIFA 월드컵</a> 예선전에서 첫 성인 대표팀 데뷔전을 치렀다.<sup class="reference" id="cite_ref-7"><a href="#cite_note-7">[7]</a></sup></p>
<h2><span class="mw-headline" id=".EA.B0.9C.EC.9D.B8_.EC.82.B6">개인 삶</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%A7%88%EC%9A%B0%EB%A1%9C_%EC%9D%B4%EC%B9%B4%EB%A5%B4%EB%94%94&amp;action=edit&amp;section=7" title="부분 편집: 개인 삶">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>이카르디 유명 미디어 스타 <a class="new" href="/w/index.php?title=%EC%99%84%EB%8B%A4_%EB%82%98%EB%9D%BC&amp;action=edit&amp;redlink=1" title="완다 나라 (없는 문서)">완다 나라</a>와 결혼을 했다. 나라는 이카르디의 전 <a href="/wiki/UC_%EC%82%BC%ED%94%84%EB%8F%84%EB%A6%AC%EC%95%84" title="UC 삼프도리아">삼프도리아</a> 동료 <a class="mw-redirect" href="/wiki/%EB%A7%89%EC%8B%9C_%EB%A1%9C%ED%8E%98%EC%8A%A4" title="막시 로페스">막시 로페스</a>와 결혼을 했었지만 이카르디와의 관계가 밝혀진후 헤어졌다. 그녀와 로페스는 2013년 12월에 이혼 소송 절차에 들어갔다. 나라와 이카르디는 이혼 소송이 끝난지가 얼마 안되어, 2014년 5월에 <a href="/wiki/%EC%95%84%EB%A5%B4%ED%97%A8%ED%8B%B0%EB%82%98" title="아르헨티나">아르헨티나</a> <a href="/wiki/%EB%B6%80%EC%97%90%EB%85%B8%EC%8A%A4%EC%95%84%EC%9D%B4%EB%A0%88%EC%8A%A4" title="부에노스아이레스">부에노스아이레스</a>에서 작은 결혼식을 올렸다.<sup class="reference" id="cite_ref-8"><a href="#cite_note-8">[8]</a></sup> 2014년 4월에 열린 <a href="/wiki/FC_%EC%9D%B8%ED%85%8C%EB%A5%B4%EB%82%98%EC%B9%98%EC%98%A4%EB%82%A0%EB%A0%88_%EB%B0%80%EB%9D%BC%EB%85%B8" title="FC 인테르나치오날레 밀라노">인테르나치오날레</a>와 삼프도리아의 <a href="/wiki/%EC%84%B8%EB%A6%AC%EC%97%90_A" title="세리에 A">세리에 A</a> 경기에서 로페스는 이카르디와의 악수를 거절하며, 경기는 "완다 더비"라고 불리기도 하였다.<sup class="reference" id="cite_ref-9"><a href="#cite_note-9">[9]</a></sup></p>
