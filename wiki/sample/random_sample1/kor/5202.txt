<p><b>마이창칭</b>(麥長靑, <b>맥장청</b>, 영문명 <b>Evergreen Mak</b>, <a href="/wiki/1967%EB%85%84" title="1967년">1967년</a> <a href="/wiki/12%EC%9B%94_18%EC%9D%BC" title="12월 18일">12월 18일</a> ~ )은 <a href="/wiki/%ED%99%8D%EC%BD%A9" title="홍콩">중화인민공화국 홍콩 특별행정구</a>의 <a class="mw-redirect" href="/wiki/%EC%98%81%ED%99%94_%EB%B0%B0%EC%9A%B0" title="영화 배우">영화배우</a>, <a href="/wiki/%EB%B0%B0%EC%9A%B0" title="배우">텔레비전 연기자</a>이다.</p>
<h2><span class="mw-headline" id=".EC.83.9D.EC.95.A0">생애</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%A7%88%EC%9D%B4%EC%B0%BD%EC%B9%AD&amp;action=edit&amp;section=1" title="부분 편집: 생애">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p><a href="/wiki/%EC%A4%91%ED%99%94%EC%9D%B8%EB%AF%BC%EA%B3%B5%ED%99%94%EA%B5%AD" title="중화인민공화국">중화인민공화국</a> <a href="/wiki/%EA%B4%91%EB%91%A5_%EC%84%B1" title="광둥 성">광둥 성</a> <a href="/wiki/%ED%8F%AC%EC%82%B0_%EC%8B%9C" title="포산 시">포산 시</a> <a href="/wiki/%EC%88%9C%EB%8D%94_%EA%B5%AC" title="순더 구">순더 구</a> 출생이고 친가 원적지는 <a href="/wiki/%EC%A4%91%ED%99%94%EC%9D%B8%EB%AF%BC%EA%B3%B5%ED%99%94%EA%B5%AD" title="중화인민공화국">중화인민공화국</a> <a href="/wiki/%EA%B4%91%EB%91%A5_%EC%84%B1" title="광둥 성">광둥 성</a> <a href="/wiki/%EC%B9%AD%EC%9C%84%EC%95%88_%EC%8B%9C" title="칭위안 시">칭위안</a>이며 외가 본적지는 <a href="/wiki/%EC%A4%91%ED%99%94%EC%9D%B8%EB%AF%BC%EA%B3%B5%ED%99%94%EA%B5%AD" title="중화인민공화국">중화인민공화국</a> <a href="/wiki/%EA%B4%91%EB%91%A5_%EC%84%B1" title="광둥 성">광둥 성</a> <a href="/wiki/%ED%8F%AC%EC%82%B0_%EC%8B%9C" title="포산 시">포산 시</a> <a href="/wiki/%EB%82%9C%ED%95%98%EC%9D%B4_%EA%B5%AC" title="난하이 구">난하이 구</a>이다.</p>
<p><a href="/wiki/1987%EB%85%84" title="1987년">1987년</a> <a href="/wiki/%ED%85%94%EB%A0%88%EB%B9%84%EC%A0%84_%EB%93%9C%EB%9D%BC%EB%A7%88" title="텔레비전 드라마">텔레비전 드라마</a>에서 <a href="/wiki/%EB%B0%B0%EC%9A%B0" title="배우">연기자</a>로 첫 데뷔하였고 이듬해 <a href="/wiki/1988%EB%85%84" title="1988년">1988년</a> <a href="/wiki/%EC%98%81%ED%99%94" title="영화">영화</a> 《형경본색(刑警本色)》으로 <a class="mw-redirect" href="/wiki/%EC%98%81%ED%99%94_%EB%B0%B0%EC%9A%B0" title="영화 배우">영화배우</a> 데뷔하였으며 <a href="/wiki/2011%EB%85%84" title="2011년">2011년</a>에는 <a href="/wiki/%EA%B0%80%EC%88%98" title="가수">가수</a>로도 데뷔하였다.</p>
