<p><b>마이클 존 마이어스</b>(Michael John Myers, <a href="/wiki/1963%EB%85%84" title="1963년">1963년</a> <a href="/wiki/5%EC%9B%94_25%EC%9D%BC" title="5월 25일">5월 25일</a> ~ )는 영화 《<a class="mw-redirect" href="/wiki/%EC%98%A4%EC%8A%A4%ED%8B%B4_%ED%8C%8C%EC%9B%8C%EC%8A%A4" title="오스틴 파워스">오스틴 파워스</a>》로 유명한 <a href="/wiki/%EC%BA%90%EB%82%98%EB%8B%A4" title="캐나다">캐나다</a>인 배우이자 코미디언이다. 영화 시나리오를 쓰는 작가이자, 영화 제작자이기도 하다.</p>
<p>마이어스의 부모는 모두 <a href="/wiki/%EC%98%81%EA%B5%AD" title="영국">영국</a> <a href="/wiki/%EB%A6%AC%EB%B2%84%ED%92%80" title="리버풀">리버풀</a> 출신이며, 마이어스 자신은 미국, 영국과 캐나다의 국적을 갖고 있다. 혈통은 주로 <a href="/wiki/%EC%9E%89%EA%B8%80%EB%9E%9C%EB%93%9C" title="잉글랜드">잉글랜드</a>, <a href="/wiki/%EC%8A%A4%EC%BD%94%ED%8B%80%EB%9E%9C%EB%93%9C" title="스코틀랜드">스코틀랜드</a>, <a href="/wiki/%EC%95%84%EC%9D%BC%EB%9E%9C%EB%93%9C" title="아일랜드">아일랜드</a>계가 섞여 있다.</p>
<p>마이어스는 <a href="/wiki/1989%EB%85%84" title="1989년">1989년</a>부터 <a href="/wiki/1995%EB%85%84" title="1995년">1995년</a>까지 미국 <a href="/wiki/NBC" title="NBC">NBC</a>의 인기 코미디 프로그램인 《<a href="/wiki/%EC%83%88%ED%84%B0%EB%8D%B0%EC%9D%B4_%EB%82%98%EC%9D%B4%ED%8A%B8_%EB%9D%BC%EC%9D%B4%EB%B8%8C" title="새터데이 나이트 라이브">새터데이 나이트 라이브</a>》 멤버로 활약하면서, 디터, 린다 리치맨, 웨인 캠벨 등의 캐릭터를 연기했다. 이 중 웨이 캠벨 캐릭터는 <a href="/wiki/1992%EB%85%84" title="1992년">1992년</a> 《<a class="new" href="/w/index.php?title=%EC%9B%A8%EC%9D%B8%EC%8A%A4_%EC%9B%94%EB%93%9C&amp;action=edit&amp;redlink=1" title="웨인스 월드 (없는 문서)">웨인스 월드</a>》라는 영화로 만들어져 크게 흥행에 성공하고, 1993년에는 《<a class="new" href="/w/index.php?title=%EC%9B%A8%EC%9D%B8%EC%8A%A4_%EC%9B%94%EB%93%9C2&amp;action=edit&amp;redlink=1" title="웨인스 월드2 (없는 문서)">웨인스 월드2</a>》라는 속편이 제작된다.</p>
<p><a href="/wiki/1997%EB%85%84" title="1997년">1997년</a> 마이어스는 영화 《<a class="mw-redirect" href="/wiki/%EC%98%A4%EC%8A%A4%ED%8B%B4_%ED%8C%8C%EC%9B%8C%EC%8A%A4" title="오스틴 파워스">오스틴 파워스</a>》를 제작하고, 자신이 주인공을 비롯한 역을 맡아 연기한다. 이 영화는 대한민국에서도 개봉하여 인기를 끌었다. 1999년 마이어스는 영화 《54》에 출연하여 처음으로 코미디가 아닌 정통 연기를 선보인다. 이 영화도 나름대로 성공을 거두었으며, 마이어스의 정통 연기도 호평을 받았다.</p>
<p>마이어스는 <a href="/wiki/%EB%B1%85%EA%B8%80%EC%8A%A4" title="뱅글스">뱅글스</a>의 기타리스트이자 싱어였던 <a class="mw-redirect" href="/wiki/%EC%88%98%EC%9E%94%EB%82%98_%ED%98%B8%ED%94%84%EC%8A%A4" title="수잔나 호프스">수잔나 호프스</a>와 밴드 "밍 티"의 멤버이기도 하다. 밴드는 《오스틴 파워스》에서 〈BBC〉와 〈Daddy Wasn't There〉를 불렀다.</p>
<p>그가 태어난 캐나다 온타리오 주 스카버러(Scarborough) 에는 마이크 마이어스 드라이브라고 그의 이름을 딴 거리가 있다. 이 거리의 표지판을 종종 떼어간다고 한다.</p>
<p>그의 형제인 <a class="new" href="/w/index.php?title=%ED%8F%B4_%EB%A7%88%EC%9D%B4%EC%96%B4%EC%8A%A4&amp;action=edit&amp;redlink=1" title="폴 마이어스 (없는 문서)">폴 마이어스</a>는 <a href="/wiki/%EC%9D%8C%EC%95%85%EA%B0%80" title="음악가">음악가</a>이자 <a href="/wiki/%EC%9E%91%EA%B0%80" title="작가">작가</a>이다. <a href="/wiki/2005%EB%85%84" title="2005년">2005년</a> 마이어스는 12년간 결혼생활을 유지했던 로빈 루잔과 이혼했다.</p>
