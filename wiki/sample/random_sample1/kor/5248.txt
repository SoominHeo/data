<p><b>마일스 파우어</b>(<span style="font-size: smaller;"><a href="/wiki/%EC%9D%BC%EB%B3%B8%EC%96%B4" title="일본어">일본어</a>:</span> <span lang="ja" xml:lang="ja">マイルス・パウアー</span>)는 그의 별명(애칭)인 <b>테일즈</b>(<span style="font-size: smaller;"><a href="/wiki/%EC%9D%BC%EB%B3%B8%EC%96%B4" title="일본어">일본어</a>:</span> <span lang="ja" xml:lang="ja">テイルス</span>)로 더 잘 알려져 있으며 <a class="mw-redirect" href="/wiki/%EC%86%8C%EB%8B%89_%EB%8D%94_%ED%97%A4%EC%A7%80%ED%98%B8%EA%B7%B8_%EC%8B%9C%EB%A6%AC%EC%A6%88" title="소닉 더 헤지호그 시리즈">소닉 더 헤지호그 시리즈</a>에 나오는 <a href="/wiki/%EB%B9%84%EB%94%94%EC%98%A4_%EA%B2%8C%EC%9E%84" title="비디오 게임">비디오 게임</a>, <a href="/wiki/%EB%A7%8C%ED%99%94" title="만화">만화(코믹)</a> 그리고 <a href="/wiki/%EC%95%A0%EB%8B%88%EB%A9%94%EC%9D%B4%EC%85%98" title="애니메이션">애니메이션</a>에 등장하는 가상의 캐릭터이다.</p>

<h2><span class="mw-headline" id=".EC.84.B1.28.E5.A7.93.29_.ED.91.9C.EA.B8.B0">성(姓) 표기</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%A7%88%EC%9D%BC%EC%A6%88_%ED%94%84%EB%9D%BC%EC%9A%B0%EC%96%B4&amp;action=edit&amp;section=1" title="부분 편집: 성(姓) 표기">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>테일즈의 본명 중 'Prower' 는 <b>파우어</b>라고 발음하는 것이 맞다. 세가 <a class="mw-redirect" href="/wiki/%EA%B2%8C%EC%9E%84%EA%B8%B0%EC%96%B4" title="게임기어">게임기어</a>로 발매된 <a class="mw-redirect" href="/wiki/%EC%86%8C%EB%8B%89%26%ED%85%8C%EC%9D%BC%EC%A6%88_2" title="소닉&amp;테일즈 2">소닉 카오스</a>의 캐스팅 영상을 보면 개발자들이 파우어(Prower)의 발음을 헷갈려서 <b>마일스 파워</b>(Miles Power)라고 오역해놓은 것을 볼 수 있다. 한국에서는 정발 명칭인 <b>마일즈 프로워</b>로 표기되어 있다.</p>
<p>참고로 파우어(Prower)의 일본식 발음은 <b>파우아</b>(パウアー)이다.</p>
<h2><span class="mw-headline" id=".EC.84.A4.EC.A0.95">설정</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%A7%88%EC%9D%BC%EC%A6%88_%ED%94%84%EB%9D%BC%EC%9A%B0%EC%96%B4&amp;action=edit&amp;section=2" title="부분 편집: 설정">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>마일스 파우어라는 이름은 마일스 퍼 아워(Miles per hour)의 말장난 형태이며<sup class="reference" id="cite_ref-1"><a href="#cite_note-1">[1]</a></sup>, 이것은 그의 <a href="/wiki/%EC%86%8C%EB%8B%89_%EB%8D%94_%ED%97%A4%EC%A7%80%ED%98%B8%EA%B7%B8_(%EC%BA%90%EB%A6%AD%ED%84%B0)" title="소닉 더 헤지호그 (캐릭터)">소닉 더 헤지호그 시리즈</a>의 엄청난 속도를 보여주는 것이다. 그는 두 개의 꼬리를 가진 어린 <a href="/wiki/%EC%97%AC%EC%9A%B0" title="여우">여우</a>이며, 그 특징 때문에 또래에게 <a class="mw-redirect" href="/wiki/%EC%9D%B4%EC%A7%80%EB%A9%94" title="이지메">이지메</a>를 당했다는 설정이 존재한다.<sup class="reference" id="cite_ref-2"><a href="#cite_note-2">[2]</a></sup> 어느 날부터 소닉에게 동경을 느껴 동행함에 따라, 앞으로 소닉에게 신뢰받는 동료가 되기 위해 소닉의 사이드킥으로 활약한다.</p>
<h2><span class="mw-headline" id=".EA.B8.B0.EC.88.A0">기술</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%A7%88%EC%9D%BC%EC%A6%88_%ED%94%84%EB%9D%BC%EC%9A%B0%EC%96%B4&amp;action=edit&amp;section=3" title="부분 편집: 기술">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>테일즈가 가장 주무기로 내세우는 기술은 두 개의 꼬리를 이용한 액션과 기계 다루기이다. 소닉&amp;테일즈 2에서부터 꼬리를 회전시켜 비행하는 액션을 선보인 적 있다. 또한 지상에서 꼬리치기를 해 적을 공격하는 경우로도 사용한 적 있다.<sup class="reference" id="cite_ref-3"><a href="#cite_note-3">[3]</a></sup> 기계를 다루는 쪽으로는 <a href="/wiki/%EC%86%8C%EB%8B%89_%26_%ED%85%8C%EC%9D%BC%EC%A6%88_2" title="소닉 &amp; 테일즈 2">소닉 &amp; 테일즈 2</a>부터 나온 소닉과 테일즈의 전용 비행기 <b>토네이도</b>, <a href="/wiki/%EC%86%8C%EB%8B%89_%EC%96%B8%EB%A6%AC%EC%89%AC%EB%93%9C" title="소닉 언리쉬드">소닉 언리쉬드</a>부터 선보인 테일즈의 단말기 <b>마일즈 일렉트릭</b>, <a href="/wiki/%EC%86%8C%EB%8B%89_%EB%9D%BC%EC%9D%B4%EB%8D%94%EC%A6%88" title="소닉 라이더즈">소닉 라이더즈</a>시리즈의 팀 소닉의 익스트림 기어인 <b>블루 스타</b>, <b>옐로 테일</b>, <b>레드 록</b>등이 있다. 현재 소닉 게임 시리즈가 여러 캐릭터를 다루지 않고 소닉 1인만을 조작하는 형태로 나옴에 따라 테일즈는 기계를 다루는 쪽으로 많이 연출되고 있다.</p>
<h2><span class="mw-headline" id=".ED.86.A0.EB.84.A4.EC.9D.B4.EB.8F.84">토네이도</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%A7%88%EC%9D%BC%EC%A6%88_%ED%94%84%EB%9D%BC%EC%9A%B0%EC%96%B4&amp;action=edit&amp;section=4" title="부분 편집: 토네이도">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>원래 소닉의 비행기였으나, 테일즈가 허락을 맡고 개조하고 있는 비행기. 토네이도는 설정상 계속 개조되고 있는 것으로 보이며, 많은 작품에서 모습이 다른 기체를 들고 온다. 우주를 비롯한 기권 내에서 정상 속도로 운행이 가능하며, 커버가 없어도 호흡이 가능하다.</p>
<h2><span class="mw-headline" id=".EA.B4.80.EB.A0.A8_.EC.A0.95.EB.B3.B4">관련 정보</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%A7%88%EC%9D%BC%EC%A6%88_%ED%94%84%EB%9D%BC%EC%9A%B0%EC%96%B4&amp;action=edit&amp;section=5" title="부분 편집: 관련 정보">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>일본 세가 홈페이지에 의하면 그는 8살이며, 키는 80 cm이고 몸무게는 20 kg이다. 그의 생일은 <a href="/wiki/10%EC%9B%94_16%EC%9D%BC" title="10월 16일">10월 16일</a>로 알려져 있는데, 그 날은 그가 <a href="/wiki/1992%EB%85%84" title="1992년">1992년</a>에 처음으로 <a class="mw-redirect" href="/wiki/%EB%A7%88%EC%8A%A4%ED%84%B0_%EC%8B%9C%EC%8A%A4%ED%85%9C" title="마스터 시스템">마스터 시스템</a>과 <a href="/wiki/%EA%B2%8C%EC%9E%84_%EA%B8%B0%EC%96%B4" title="게임 기어">게임 기어</a>용 게임인 <a href="/wiki/%EC%86%8C%EB%8B%89_%EB%8D%94_%ED%97%A4%EC%A7%80%ED%98%B8%EA%B7%B8_2_(8%EB%B9%84%ED%8A%B8)" title="소닉 더 헤지호그 2 (8비트)">소닉 더 헤지호그 2</a>에 처음 나온 날이다. 대부분의 플레이어에게는 이 캐릭터의 등장은 <a href="/wiki/%EB%A9%94%EA%B0%80_%EB%93%9C%EB%9D%BC%EC%9D%B4%EB%B8%8C" title="메가 드라이브">세가 제네시스/메가 드라이브</a>용 <a href="/wiki/%EC%86%8C%EB%8B%89_%EB%8D%94_%ED%97%A4%EC%A7%80%ED%98%B8%EA%B7%B8_2" title="소닉 더 헤지호그 2">소닉 더 헤지호그 2</a>가 될 것이다.</p>
<p>그의 첫 출현에서는 그의 털은 오랜지 색이었지만, 미국의 바람돌이 소닉, 고슴도치 소닉에서는 갈색으로 나왔다. <a href="/wiki/1998%EB%85%84" title="1998년">1998년</a> <a href="/wiki/%EB%93%9C%EB%A6%BC%EC%BA%90%EC%8A%A4%ED%8A%B8" title="드림캐스트">드림캐스트</a>로 나온<a href="/wiki/%EC%86%8C%EB%8B%89_%EC%96%B4%EB%93%9C%EB%B2%A4%EC%B2%98" title="소닉 어드벤처">소닉 어드벤처</a>에서는 그의 털은 노란 오랜지 색으로 바뀌었으며, <a href="/wiki/2003%EB%85%84" title="2003년">2003년</a>에 나온 <a href="/wiki/%EC%86%8C%EB%8B%89_%ED%9E%88%EC%96%B4%EB%A1%9C%EC%A6%88" title="소닉 히어로즈">소닉 히어로즈</a>에서는 밝은 노란색이 되었다. <a href="/wiki/1993%EB%85%84" title="1993년">1993년</a>과 <a href="/wiki/1995%EB%85%84" title="1995년">1995년</a>에는 그가 주인공인 게임에 출현했는데, <a class="mw-redirect" href="/wiki/%EC%84%B8%EA%B0%80_%ED%94%BC%EC%BD%94" title="세가 피코">피코</a>용 <a href="/wiki/%ED%85%8C%EC%9D%BC%EC%A6%88_%EC%95%A4_%EB%AE%A4%EC%A7%81_%EB%A9%94%EC%9D%B4%EC%BB%A4" title="테일즈 앤 뮤직 메이커">테일즈 &amp; 뮤직 메이커</a>와 세가 게임 기어용 <a href="/wiki/%ED%85%8C%EC%9D%BC%EC%A6%88_%EC%96%B4%EB%93%9C%EB%B2%A4%EC%B2%98" title="테일즈 어드벤처">테일즈 어드벤처</a>와 <a href="/wiki/%ED%85%8C%EC%9D%BC%EC%A6%88_%EC%8A%A4%EC%B9%B4%EC%9D%B4%ED%8C%A8%ED%8A%B8%EB%A1%A4" title="테일즈 스카이패트롤">테일즈 스카이 패트롤</a>이 그것이다.</p>
<h2><span class="mw-headline" id=".EC.84.B1.EC.9A.B0">성우</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%A7%88%EC%9D%BC%EC%A6%88_%ED%94%84%EB%9D%BC%EC%9A%B0%EC%96%B4&amp;action=edit&amp;section=6" title="부분 편집: 성우">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>메가 드라이브판에 대하고는 소리의 설정은 존재하지 않았지만, 소닉 어드벤쳐 시리즈에서 성우에 의한 소리의 연출이 시작된다. 소닉 어드벤처:(林一樹) 소닉 셔플·소닉 어드벤쳐2:무라타 아츠키(村田あつき), 소닉 vs 메탈소닉 OVA판: <a href="/wiki/%EC%8B%9C%EC%9D%B4%EB%82%98_%ED%97%A4%ED%82%A4%EB%A3%A8" title="시이나 헤키루">시이나 헤키루</a>, 소닉X: <a href="/wiki/%ED%9E%88%EB%A1%9C%ED%95%98%EC%8B%9C_%EB%A3%8C" title="히로하시 료">히로하시 료</a>와 4도의 변경이 이루어지고 있다. 이미지의 정합성을 취하기 위해, 소닉X 이후는 게임 작품에 대해도 <a href="/wiki/%ED%9E%88%EB%A1%9C%ED%95%98%EC%8B%9C_%EB%A3%8C" title="히로하시 료">히로하시 료</a>가 연기하고 있다.<a class="mw-redirect" href="/wiki/%EC%86%8C%EB%8B%89_%EC%A0%9C%EB%84%A4%EB%A0%88%EC%9D%B4%EC%85%98%EC%A6%88" title="소닉 제네레이션즈">소닉 제네레이션즈</a>의 과거의 테일즈는 <a class="new" href="/w/index.php?title=%EC%9A%94%EC%8B%9C%EB%82%98%EA%B0%80_%ED%83%80%EC%BF%A0%ED%86%A0&amp;action=edit&amp;redlink=1" title="요시나가 타쿠토 (없는 문서)">요시나가 타쿠토</a>가 연기했다.</p>
