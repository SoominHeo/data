<p><b>마제야</b>(Mageia)는 <a href="/wiki/%EB%A7%A8%EB%93%9C%EB%A6%AC%EB%B0%94_%EB%A6%AC%EB%88%85%EC%8A%A4" title="맨드리바 리눅스">맨드리바 리눅스</a>에서 <a href="/wiki/%ED%8F%AC%ED%81%AC_(%EC%86%8C%ED%94%84%ED%8A%B8%EC%9B%A8%EC%96%B4_%EA%B0%9C%EB%B0%9C)" title="포크 (소프트웨어 개발)">포크</a>되어 개발되는 <a href="/wiki/%EB%A6%AC%EB%88%85%EC%8A%A4" title="리눅스">리눅스</a> 배포판의 일종이다. <a href="/wiki/2011%EB%85%84" title="2011년">2011년</a> 6월 1일에 첫 번째 버전이 출시되었다. 마제야는 맨드리바 리눅스의 전직 직원들과 개발자들, 커뮤니티 구성원들에 의해 2010년부터 개발이 진행되었다. 마제야라는 이름은 <a href="/wiki/%EA%B7%B8%EB%A6%AC%EC%8A%A4%EC%96%B4" title="그리스어">그리스어</a>에서 유래하였으며, 마법이라는 뜻이다."<sup class="reference" id="cite_ref-3"><a href="#cite_note-3">[3]</a></sup></p>

<h2><span class="mw-headline" id=".EB.8D.B0.EC.8A.A4.ED.81.AC.ED.83.91_.ED.99.98.EA.B2.BD">데스크탑 환경</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%A7%88%EC%A0%9C%EC%95%BC&amp;action=edit&amp;section=1" title="부분 편집: 데스크탑 환경">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>다른 리눅스 배포판들과 마찬가지로 마제야는 리눅스에서 쓰이는 주요 데스크탑 환경을 지원한다. <a href="/wiki/DVD" title="DVD">DVD</a> 버전의 경우 <a href="/wiki/KDE" title="KDE">KDE</a>와 <a href="/wiki/%EA%B7%B8%EB%86%88" title="그놈">그놈</a>을 사용하며, CD 버전의 경우 <a href="/wiki/LXDE" title="LXDE">LXDE</a>를 사용한다.</p>
<h2><span class="mw-headline" id=".EA.B0.9C.EB.B0.9C">개발</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%A7%88%EC%A0%9C%EC%95%BC&amp;action=edit&amp;section=2" title="부분 편집: 개발">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>마제야는 9개월 단위로 발표되며, 각 배포판의 지원기간은 18개월이다. 현재 최신 버전은 2015년 6월에 발표한 <b>5</b>이다.</p>
