<p><b>마쓰다 류헤이</b>(<span style="font-size: smaller;"><a href="/wiki/%EC%9D%BC%EB%B3%B8%EC%96%B4" title="일본어">일본어</a>:</span> <span lang="ja"><span lang="ja"><ruby><rb>松田龍平</rb> <rp>(</rp><rt>まつだ りゅうへい</rt><rp>)</rp></ruby></span></span>, <a href="/wiki/1983%EB%85%84" title="1983년">1983년</a> <a href="/wiki/5%EC%9B%94_9%EC%9D%BC" title="5월 9일">5월 9일</a> ~ )는 <a href="/wiki/%EC%9D%BC%EB%B3%B8" title="일본">일본</a>의 <a href="/wiki/%EB%B0%B0%EC%9A%B0" title="배우">배우</a>이다. 호리코시 고등학교를 중퇴하였다. 일본인 아버지와 재일 한국인 3세 어머니의 아들인 아버지 <a class="mw-redirect" href="/wiki/%EB%A7%88%EC%93%B0%EB%8B%A4_%EC%9C%A0%EC%82%AC%EC%BF%A0" title="마쓰다 유사쿠">마쓰다 유사쿠</a>와 일본인 어머니 <a class="mw-redirect" href="/wiki/%EB%A7%88%EC%93%B0%EB%8B%A4_%EB%AF%B8%EC%9C%A0%ED%82%A4" title="마쓰다 미유키">마쓰다 미유키</a>의 장남으로 태어났다.</p>

<h2><span class="mw-headline" id=".EC.9D.B8.EB.AC.BC">인물</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%A7%88%EC%B8%A0%EB%8B%A4_%EB%A5%98%ED%97%A4%EC%9D%B4&amp;action=edit&amp;section=1" title="부분 편집: 인물">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<ul>
<li><a href="/wiki/%EB%B0%B0%EC%9A%B0" title="배우">배우</a> <a href="/wiki/%EB%A7%88%EC%B8%A0%EB%8B%A4_%EC%9C%A0%EC%82%AC%EC%BF%A0" title="마츠다 유사쿠">마쓰다 유사쿠</a>와 <a class="mw-redirect" href="/wiki/%EB%A7%88%EC%93%B0%EB%8B%A4_%EB%AF%B8%EC%9C%A0%ED%82%A4" title="마쓰다 미유키">마쓰다 미유키</a> 사이에서 장남으로 <a href="/wiki/%EC%97%B0%EC%98%88%EC%9D%B8" title="연예인">연예인</a> 일가에서 태어났다. 남동생은 배우 <a class="mw-redirect" href="/wiki/%EB%A7%88%EC%93%B0%EB%8B%A4_%EC%87%BC%ED%83%80" title="마쓰다 쇼타">마쓰다 쇼타</a>, 이모도 배우인 <a class="new" href="/w/index.php?title=%EA%B5%AC%EB%A7%88%EA%B0%80%EC%9D%B4_%EB%A7%88%EB%AF%B8&amp;action=edit&amp;redlink=1" title="구마가이 마미 (없는 문서)">구마가이 마미</a>.</li>
<li>교제하던 <a href="/wiki/%EB%AA%A8%EB%8D%B8_(%EC%A7%81%EC%97%85)" title="모델 (직업)">패션 모델</a> <a href="/wiki/%EC%98%A4%ED%83%80_%EB%A6%AC%EB%82%98" title="오타 리나">오타 리나</a>의 생일인 <a href="/wiki/2009%EB%85%84" title="2009년">2009년</a> <a href="/wiki/1%EC%9B%94_11%EC%9D%BC" title="1월 11일">1월 11일</a>에 결혼. 같은 해 <a href="/wiki/7%EC%9B%94_4%EC%9D%BC" title="7월 4일">7월 4일</a>에 <a href="/wiki/%EB%94%B8" title="딸">장녀</a>를 출산하였다.</li>
<li>취미는 <a href="/wiki/%EC%B6%95%EA%B5%AC" title="축구">축구</a>와 TV 게임, 모터크로스.</li>
</ul>
<h2><span class="mw-headline" id=".EC.95.BD.EB.A0.A5">약력</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%A7%88%EC%B8%A0%EB%8B%A4_%EB%A5%98%ED%97%A4%EC%9D%B4&amp;action=edit&amp;section=2" title="부분 편집: 약력">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<ul>
<li>6살 때 아버지 마츠다 유사쿠를 여읜다.</li>
<li>스기나미 구 이구사 중학교 재학 중, 《<a href="/wiki/%EA%B3%A0%ED%95%98%ED%86%A0" title="고하토">고하토</a>》의 주연을 찾고있던 <a href="/wiki/%EC%98%A4%EC%8B%9C%EB%A7%88_%EB%82%98%EA%B8%B0%EC%82%AC" title="오시마 나기사">오시마 나기사</a>의 눈에 띄어, 직접 출연 요청을 한 것을 계기로 배우의 길에 들어선다.</li>
<li><a href="/wiki/1999%EB%85%84" title="1999년">1999년</a> 영화 《고하토》(오시마 나기사 작품)으로 데뷔. 일본 아카데미상, 키네마순보, 마이니치 영화 콩쿨, 블루리본상을 시작으로, 그 해 신인상을 휩쓴다.</li>
<li><a href="/wiki/2002%EB%85%84" title="2002년">2002년</a>, 《<a href="/wiki/%EC%9A%B0%EC%9A%B8%ED%95%9C_%EC%B2%AD%EC%B6%98" title="우울한 청춘">우울한 청춘</a>》(도요타 토시아키 작품)에 주연, 압도적인 존재감으로 주목을 받게 된다.</li>
<li><a href="/wiki/2004%EB%85%84" title="2004년">2004년</a>, 《야차연못》으로 연극 첫 공연. 또 영화 《사랑의 문》을 통해 로맨틱 코미디에 처음으로 도전해 연기의 폭을 넓혔다.</li>
<li><a href="/wiki/2007%EB%85%84" title="2007년">2007년</a>, <a href="/wiki/%ED%85%94%EB%A0%88%EB%B9%84%EC%A0%84_%EB%93%9C%EB%9D%BC%EB%A7%88" title="텔레비전 드라마">드라마</a> 《<a href="/wiki/%ED%95%98%EA%B2%8C%ED%83%80%EC%B9%B4" title="하게타카">하게타카</a>》(<a href="/wiki/%EC%9D%BC%EB%B3%B8%EB%B0%A9%EC%86%A1%ED%98%91%ED%9A%8C" title="일본방송협회">NHK</a>)에서 연속드라마 첫 출연.</li>
<li><a href="/wiki/2008%EB%85%84" title="2008년">2008년</a>, 《내일의 키타 요시오~세상에서 제일 불운한 남자의, 기적의 11일간~》(<a href="/wiki/%EA%B0%84%EC%82%AC%EC%9D%B4_TV_%EB%B0%A9%EC%86%A1" title="간사이 TV 방송">칸사이 TV</a>)으로 <a href="/wiki/%EB%AF%BC%EC%98%81_%EB%B0%A9%EC%86%A1" title="민영 방송">민영 방송</a> 연속 드라마에 처음으로 출연하였다.</li>
</ul>
