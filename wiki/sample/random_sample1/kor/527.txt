<p><b>2026년 FIFA 월드컵</b>(<span style="font-size: smaller;"><a href="/wiki/%EC%98%81%EC%96%B4" title="영어">영어</a>:</span> <span lang="en" xml:lang="en">2026 FIFA World Cup</span>)은 <a href="/wiki/2026%EB%85%84" title="2026년">2026년</a>에 열릴 예정인 23번째 <a href="/wiki/FIFA_%EC%9B%94%EB%93%9C%EC%BB%B5" title="FIFA 월드컵">FIFA 월드컵</a>이다. <a href="/wiki/2017%EB%85%84" title="2017년">2017년</a> <a href="/wiki/5%EC%9B%94_10%EC%9D%BC" title="5월 10일">5월 10일</a> <a href="/wiki/%EB%A7%90%EB%A0%88%EC%9D%B4%EC%8B%9C%EC%95%84" title="말레이시아">말레이시아</a> <a href="/wiki/%EC%BF%A0%EC%95%8C%EB%9D%BC%EB%A3%B8%ED%91%B8%EB%A5%B4" title="쿠알라룸푸르">쿠알라룸푸르</a>에서 개최지가 결정될 예정이었지만 <a href="/wiki/2018%EB%85%84_FIFA_%EC%9B%94%EB%93%9C%EC%BB%B5" title="2018년 FIFA 월드컵">2018년 FIFA 월드컵</a>과 <a href="/wiki/2022%EB%85%84_FIFA_%EC%9B%94%EB%93%9C%EC%BB%B5" title="2022년 FIFA 월드컵">2022년 FIFA 월드컵</a> 유치 과정에서 일어난 비리 사건(<a href="/wiki/2015%EB%85%84_%EA%B5%AD%EC%A0%9C_%EC%B6%95%EA%B5%AC_%EC%97%B0%EB%A7%B9_%EB%B6%80%ED%8C%A8_%EC%82%AC%EA%B1%B4" title="2015년 국제 축구 연맹 부패 사건">2015년 국제 축구 연맹 부패 사건</a>) 수사에 관한 여파로 인해 연기된 상태이다. 현 FIFA 회장 <a href="/wiki/%EC%9E%94%EB%8B%88_%EC%9D%B8%ED%8C%90%ED%8B%B0%EB%85%B8" title="잔니 인판티노">잔니 인판티노</a>가 FIFA 회장 선거에서 2026년 FIFA 월드컵 본선 참가국 수를 40개국으로 늘리겠다고 공약하여 당선되었고 <a href="/wiki/2017%EB%85%84" title="2017년">2017년</a> <a href="/wiki/1%EC%9B%94_10%EC%9D%BC" title="1월 10일">1월 10일</a> 스위스 취리히에서 열린 FIFA 이사회에서 투표를 통해 만장일치로 참가국을 48개국으로 확대하게 되었다. <sup class="reference" id="cite_ref-1"><a href="#cite_note-1">[1]</a></sup><sup class="reference" id="cite_ref-2"><a href="#cite_note-2">[2]</a></sup></p>
<p>2026년 FIFA 월드컵 본선 개최 유치 후보국의 자격은 <a href="/wiki/2022%EB%85%84_FIFA_%EC%9B%94%EB%93%9C%EC%BB%B5" title="2022년 FIFA 월드컵">2022년 FIFA 월드컵</a> 개최 대륙 연맹인 <a href="/wiki/%EC%95%84%EC%8B%9C%EC%95%84_%EC%B6%95%EA%B5%AC_%EC%97%B0%EB%A7%B9" title="아시아 축구 연맹">아시아 축구 연맹</a> 소속 국가 축구 협회를 제외하고 모두 입후보 자격이 있다.<sup class="reference" id="cite_ref-press_conference_3-0"><a href="#cite_note-press_conference-3">[3]</a></sup><sup class="reference" id="cite_ref-bbc_fifa_rotation_4-0"><a href="#cite_note-bbc_fifa_rotation-4">[4]</a></sup> 그러므로 <a class="mw-redirect" href="/wiki/%EC%9C%A0%EB%9F%BD_%EC%B6%95%EA%B5%AC_%EC%97%B0%EB%A7%B9" title="유럽 축구 연맹">유럽 축구 연맹</a>, <a class="mw-redirect" href="/wiki/%EB%B6%81%EC%A4%91%EB%AF%B8_%EC%B6%95%EA%B5%AC_%EC%97%B0%EB%A7%B9" title="북중미 축구 연맹">북중미 축구 연맹</a>, <a href="/wiki/%EB%82%A8%EB%AF%B8_%EC%B6%95%EA%B5%AC_%EC%97%B0%EB%A7%B9" title="남미 축구 연맹">남미 축구 연맹</a>, <a href="/wiki/%EC%95%84%ED%94%84%EB%A6%AC%EC%B9%B4_%EC%B6%95%EA%B5%AC_%EC%97%B0%EB%A7%B9" title="아프리카 축구 연맹">아프리카 축구 연맹</a>, <a href="/wiki/%EC%98%A4%EC%84%B8%EC%95%84%EB%8B%88%EC%95%84_%EC%B6%95%EA%B5%AC_%EC%97%B0%EB%A7%B9" title="오세아니아 축구 연맹">오세아니아 축구 연맹</a> 소속 국가 축구 협회들은 입후보 자격이 있다. 2개국 이상 복수 국가 축구 협회들이 <a href="/wiki/FIFA_%EC%9B%94%EB%93%9C%EC%BB%B5" title="FIFA 월드컵">FIFA 월드컵</a> 본선 대회를 공동 개최국 후보에 출마할 수 있다.</p>
