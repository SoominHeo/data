<p><b>마카롱</b>(<span style="font-size: smaller;"><a href="/wiki/%ED%94%84%EB%9E%91%EC%8A%A4%EC%96%B4" title="프랑스어">프랑스어</a>:</span> <span lang="fr" xml:lang="fr">Macaron</span>, <small><a href="/wiki/%EC%9C%84%ED%82%A4%EB%B0%B1%EA%B3%BC:IPA" title="위키백과:IPA">IPA</a>: </small><span class="IPA" title="IPA 발음">[​makaˈʁɔ̃]</span><sup class="reference" id="cite_ref-1"><a href="#cite_note-1">[1]</a></sup>)은 <a href="/wiki/%ED%94%84%EB%9E%91%EC%8A%A4" title="프랑스">프랑스</a>의 대표적인 당과 제품이다. <a href="/wiki/%EB%A8%B8%EB%9E%AD" title="머랭">머랭</a>을 주재료로 하고 <a href="/wiki/%ED%9D%B0%EC%9E%90" title="흰자">달걀 흰자위</a>, <a href="/wiki/%EC%84%A4%ED%83%95" title="설탕">백설탕</a>, <a href="/wiki/%EC%95%84%EB%AA%AC%EB%93%9C" title="아몬드">아몬드</a> 가루와 아몬드 밀가루 등으로 만든 과자류이며, 모양은 동그랗고 손에 들어갈 수 있을 만큼 조그맣다. 마카롱의 구조는 잘 부서지지만 약간 딱딱한 껍질로 된 위아래 부위와, 그 중간에 <a href="/wiki/%EB%A8%B8%EB%9E%AD" title="머랭">머랭</a>이나 <a href="/wiki/%EC%9E%BC" title="잼">잼</a>, <a href="/wiki/%EB%A7%88%EC%A7%80%ED%8C%AC" title="마지팬">마지팬</a>, <a class="mw-disambig" href="/wiki/%ED%81%AC%EB%A6%BC" title="크림">크림</a>등을 넣어 만든 부드러운 중간 부위로 나뉜다. 지역마다 별명도 있는데, 프랑스의 수도인 <a href="/wiki/%ED%8C%8C%EB%A6%AC_(%ED%94%84%EB%9E%91%EC%8A%A4)" title="파리 (프랑스)">파리</a>에서는 <b>제르베르</b>(Gerber)<sup class="reference" id="cite_ref-LPM_2-0"><a href="#cite_note-LPM-2">[2]</a></sup>, <a href="/wiki/%EB%A3%A9%EC%85%88%EB%B6%80%EB%A5%B4%ED%81%AC" title="룩셈부르크">룩셈부르크</a>와 <a href="/wiki/%EB%8F%85%EC%9D%BC%EC%96%B4" title="독일어">독일어</a>를 이용하는 일부 지역에서는 <b>룩셈부르게를리</b>(Luxemburgerli)<sup class="reference" id="cite_ref-3"><a href="#cite_note-3">[3]</a></sup>라고도 하며, <a href="/wiki/%EB%A7%88%EC%B9%B4%EB%A3%AC" title="마카룬">마카룬</a>과 혼동되지 않도록 <b>프랑스 마카롱</b>(French Macaron)이라고도 한다<sup class="reference" id="cite_ref-PopSugar_4-0"><a href="#cite_note-PopSugar-4">[4]</a></sup>. <a href="/wiki/%ED%94%84%EB%9E%91%EC%8A%A4" title="프랑스">프랑스</a>와 <a href="/wiki/%EC%9C%A0%EB%9F%BD" title="유럽">유럽</a> 국가에서 가장 인기를 많이 얻고 있다.<sup class="reference" id="cite_ref-MadMac_5-0"><a href="#cite_note-MadMac-5">[5]</a></sup></p>

<p>오늘날 마카롱은 <a href="/wiki/%ED%94%84%EB%9E%91%EC%8A%A4" title="프랑스">프랑스</a>를 대표하는 과자로 여겨지지만, 본래 마카롱은 <a href="/wiki/%EC%9D%B4%ED%83%88%EB%A6%AC%EC%95%84" title="이탈리아">이탈리아</a>에서 유래된 과자다. 그러나 마카롱이 처음 만들어진 곳은 확실하게 알려져 있지 않다. 대부분의 문헌은 1500년에 시작되었음을 알리지만, 일부 자료들은 <a href="/wiki/%EC%A4%91%EC%84%B8" title="중세">중세</a> <a href="/wiki/%EC%9C%A0%EB%9F%BD" title="유럽">유럽</a>에서 마카롱이 처음 만들어졌다고 한다. 더불어 이탈리아의 <a href="/wiki/%EA%B8%B0%EB%8F%85%EA%B5%90" title="기독교">가톨릭</a> <a href="/wiki/%EC%88%98%EB%8F%84%EC%9B%90" title="수도원">수도원</a>에서 처음 만들어졌다고도 하는데, 또한 어떤 자료들은 791년, 코르메리 (Cormery)라는 프랑스 중부에 자리한 마을에서 만들기 시작했다고도 한다. 그러나 이 자료들은 공식적으로 확인되지 않았으며, 마카롱에 대한 공식적인 자료들로 인정을 받지 못하고 있다.</p>
<p>마카롱의 기원에 대해서 공식적으로 밝혀진 것은 다음과 같다. 1453년부터 이탈리아 요리에 본격적으로 아몬드가 사용되기 시작하면서, 1500년 마카롱의 주재료인 아몬드 페이스트가 이탈리아에서 개발되었다. 1533년 이탈리아의 귀족 <a href="/wiki/%EC%B9%B4%ED%8A%B8%EB%A6%B0_%EB%93%9C_%EB%A9%94%EB%94%94%EC%8B%9C%EC%8A%A4" title="카트린 드 메디시스">카트린 드 메디시스</a>가 프랑스의 왕이었던 <a href="/wiki/%EC%95%99%EB%A6%AC_2%EC%84%B8" title="앙리 2세">앙리 2세</a>랑 결혼할 무렵, 카트린의 요리사들이 카트린의 아버지가 소개한 아몬드 페이스트로 새로운 요리를 개발하였는데, 이것이 마카롱의 탄생으로 알려져 있다. 이러한 프랑스와 이탈리아 귀족들의 통혼으로 인하여 때문에 마카롱이 이탈리아에서 프랑스로 전파되었던 것이다.(오늘날 프랑스를 대표하는 요리 문화는 본래 이탈리아를 비롯한 해외에서 수입해와 이를 개량하여 성립되었다.) 마카롱이라는 단어가 최초로 등장한 것은 프랑스 <a class="mw-redirect" href="/wiki/%EB%A5%B4%EB%84%A4%EC%83%81%EC%8A%A4_%EC%8B%9C%EB%8C%80" title="르네상스 시대">르네상스 시대</a>의 유명한 작가였던 <a href="/wiki/%ED%94%84%EB%9E%91%EC%88%98%EC%95%84_%EB%9D%BC%EB%B8%94%EB%A0%88" title="프랑수아 라블레">프랑수아 라블레</a>가 1558년에 쓴 기록이다.</p>

<p>마카롱이 다른 귀족들에게 알려져 인기를 얻게된 시기는 서술한 <a href="/wiki/%EC%B9%B4%ED%8A%B8%EB%A6%B0_%EB%93%9C_%EB%A9%94%EB%94%94%EC%8B%9C%EC%8A%A4" title="카트린 드 메디시스">카트린 드 메디시스</a>와 <a href="/wiki/%EC%95%99%EB%A6%AC_2%EC%84%B8" title="앙리 2세">앙리 2세</a>의 결혼으로부터 약 한세기가 지난 1660년 즈음이었다. 프랑스 중부의 <a class="new" href="/w/index.php?title=%EB%AA%AC%EB%AA%A8%EB%A6%AC%EC%98%A8&amp;action=edit&amp;redlink=1" title="몬모리온 (없는 문서)">몬모리온</a>에서 마카롱을 본격적으로 만들기 시작하여 여러 행사에서 소비되었다. 마카롱은 <a href="/wiki/%EB%A3%A8%EC%9D%B4_14%EC%84%B8" title="루이 14세">루이 14세</a>랑 <a href="/wiki/%EC%98%A4%EC%8A%A4%ED%8A%B8%EB%A6%AC%EC%95%84%EC%9D%98_%EB%A7%88%EB%A6%AC_%ED%85%8C%EB%A0%88%EC%A6%88" title="오스트리아의 마리 테레즈">오스트리아의 마리 테레즈</a>의 결혼식 축제에서도 제공된 음식중 하나였으며, <a href="/wiki/%EC%B9%B4%ED%8A%B8%EB%A6%B0_%EB%93%9C_%EB%A9%94%EB%94%94%EC%8B%9C%EC%8A%A4" title="카트린 드 메디시스">카트린 드 메디시스</a>가 한때 마카롱을 섭취하여 굶주림을 피했다는 유사도 있다. 마카롱은 <a href="/wiki/%ED%94%84%EB%9E%91%EC%8A%A4" title="프랑스">프랑스</a> 귀족들과 왕들도 즐겨 먹은 음식이였다고 하며, 1802년에 루이 14세가 <a href="/wiki/%EB%B2%A0%EB%A5%B4%EC%82%AC%EC%9C%A0_%EA%B6%81%EC%A0%84" title="베르사유 궁전">베르사유 궁전</a>에서 왕의 되었을 때부터 <a href="/wiki/%EB%A3%A8%EC%9D%B4_16%EC%84%B8" title="루이 16세">루이 16세</a>가 왕이 되었을 때까지 프랑스 왕들에게 마카롱이 간식으로 제공되었다.<sup class="reference" id="cite_ref-Mr._Macaron_6-0"><a href="#cite_note-Mr._Macaron-6">[6]</a></sup> 마카롱이 처음 요리책에 등장했을 때에는 1653년, 《프랑수아 피에르 데 바렝의 프랑스 음식 요리법》(François Pierre de La Varenne’s Le Pâtissier François)에 처음 나왔지만 마카롱을 요리하는 법을 처음으로 가르치는 요리책은 1692년의 《과자류, 술과 과일 요리하는 특별방법》(Nouvelle Instruction pour les Confitures, les Liqueurs, et les Fruits)에서 나온다. 이 이후로 대부분 프랑스의 요리책들에는 마카롱을 요리하는 법이 나온다.</p>

<p>오늘날 마카롱은 보통 <a href="/wiki/%ED%9B%84%EC%8B%9D" title="후식">후식</a>으로 여겨지지만, 과거에는 본래 음식의 한 종류로 여겨졌다. 처음에는 매우 간단하게 생긴 과자였으며, 다양하지 않은 색깔에 두 겹보다 한 겹으로 만들어지던 과자였다. 나중에 20세기 초반때 프랑스의 귀족 요리사 루이 에르네 라뒤레(Louis Ernest Ladurée)의 손자였던 피에르 데퐁탠(Pierre Desfontaines)이 초콜릿 파나체로 마카롱 두 개를 붙여 만들었다. 이것이 현재 흔히 알려진 두 겹으로 이루어진 마카롱의 시초였다.<sup class="reference" id="cite_ref-MadMac_5-1"><a href="#cite_note-MadMac-5">[5]</a></sup> 또한, 마카롱의 종류가 다양해지면서 색깔과 맛 종류가 증가하기 시작했을 때인 1790년에는 <a href="/wiki/%EB%82%AD%EC%8B%9C" title="낭시">낭시</a>, <a href="/wiki/%EB%9E%AD%EC%8A%A4" title="랭스">랭스</a>, <a href="/wiki/%EC%95%84%EB%AF%B8%EC%95%B5" title="아미앵">아미앵</a> 등의 프랑스의 여러 도시들에서 각 지역별 변종을 만들기 시작했다. 1800년 당시에는 프랑스의 수도원에서 일하던 <a href="/wiki/%EC%88%98%EB%85%80" title="수녀">수녀</a>들이 고기를 먹을 수 없게 되자, 건강에 매우 좋다는 이유로 마카롱이 인기를 얻기 시작했다. <a href="/wiki/%EB%82%AD%EC%8B%9C" title="낭시">낭시</a>에 있었던 어느 신녀 두명이 마카롱 전문으로 만들었지만, 나중에 <a class="mw-redirect" href="/wiki/%ED%94%84%EB%9E%91%EC%8A%A4_%EB%8C%80%ED%98%81%EB%AA%85" title="프랑스 대혁명">프랑스 대혁명</a>이 일어나 교회의 체계가 붕괴되자, 마카롱을 구워 삶을 연명했다. 이들이 나중에 매우 유명하게 되면서 'les Soeurs Macarons' (마카롱 자매)라는 이름으로 전설으로 남는다. 오늘날, 그들의 가게도 전설이 되어 가게가 위치하는 길의 이름은 그들을 기원하기 위하여 개명되었다.<sup class="reference" id="cite_ref-LPM_2-1"><a href="#cite_note-LPM-2">[2]</a></sup></p>
<h3><span class="mw-headline" id=".EC.96.B4.EC.9B.90">어원</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%A7%88%EC%B9%B4%EB%A1%B1&amp;action=edit&amp;section=2" title="부분 편집: 어원">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>마카롱의 이름은 <a href="/wiki/%EA%B7%B8%EB%A6%AC%EC%8A%A4%EC%96%B4" title="그리스어">그리스어</a>에서 기인하였다. 이 단어가 <a href="/wiki/%EC%9D%B4%ED%83%88%EB%A6%AC%EC%95%84%EC%96%B4" title="이탈리아어">이탈리아어</a>로 옮겨지면서 'maccherone(마카로네)', 'maccaroni(마케로니)'에서 온 것이다. 'Macaron(마카롱)'과 'maccaroni(마케로니)'는 1650년에 제작된 프랑스어 원본 사전(Les Origines de la Langue Française)에 의하면 '<a href="/wiki/%EC%B9%98%EC%A6%88" title="치즈">치즈</a>로 뒤덮인 <a href="/wiki/%ED%8C%8C%EC%8A%A4%ED%83%80" title="파스타">파스타</a> 요리'를 지칭하지만, 이 단어들은 과자나 밀가루로 만든 음식으로도 쓰였다. 1675년에 제작된 어느 <a href="/wiki/%ED%94%84%EB%9E%91%EC%8A%A4%EC%96%B4" title="프랑스어">프랑스어</a>-<a href="/wiki/%EC%98%81%EC%96%B4" title="영어">영어</a> 사전에서는 마카롱을 이런 식으로 표현하였다:</p>

<p>또 다른 이탈리아어 단어인 'Maccare (마카레)', 또는 'Amaccare (아마카레)'<sup class="reference" id="cite_ref-PopSugar_4-1"><a href="#cite_note-PopSugar-4">[4]</a></sup>에서도 마카롱이라는 단어가 유래했다라고 하는데, 이는 '섞다', '갈다', 또는 '주무르다'라는 뜻이다. 아몬드를 갈아서 마카롱에 들어가는 아몬드 페이스트가 만들어진 것을 본따 만든 이름인 것이다.<sup class="reference" id="cite_ref-LPM_2-2"><a href="#cite_note-LPM-2">[2]</a></sup><sup class="reference" id="cite_ref-Nibble_7-0"><a href="#cite_note-Nibble-7">[7]</a></sup></p>
<h2><span class="mw-headline" id=".EC.B0.A8.EC.9D.B4.EC.A0.90">차이점</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%A7%88%EC%B9%B4%EB%A1%B1&amp;action=edit&amp;section=3" title="부분 편집: 차이점">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<h3><span class="mw-headline" id=".EB.A7.88.EC.B9.B4.EB.A3.AC">마카룬</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%A7%88%EC%B9%B4%EB%A1%B1&amp;action=edit&amp;section=4" title="부분 편집: 마카룬">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<div class="thumb tleft">
<div class="thumbinner" style="width:252px;"><a class="image" href="/wiki/%ED%8C%8C%EC%9D%BC:Macaroons_on_a_plate,_April_2010.jpg"><img alt="" class="thumbimage" data-file-height="1245" data-file-width="1349" height="231" src="//upload.wikimedia.org/wikipedia/commons/thumb/e/e7/Macaroons_on_a_plate%2C_April_2010.jpg/250px-Macaroons_on_a_plate%2C_April_2010.jpg" srcset="//upload.wikimedia.org/wikipedia/commons/thumb/e/e7/Macaroons_on_a_plate%2C_April_2010.jpg/375px-Macaroons_on_a_plate%2C_April_2010.jpg 1.5x, //upload.wikimedia.org/wikipedia/commons/thumb/e/e7/Macaroons_on_a_plate%2C_April_2010.jpg/500px-Macaroons_on_a_plate%2C_April_2010.jpg 2x" width="250"/></a>
<div class="thumbcaption">
<div class="magnify"><a class="internal" href="/wiki/%ED%8C%8C%EC%9D%BC:Macaroons_on_a_plate,_April_2010.jpg" title="실제 크기로"></a></div>
마카룬 (Macaroon)은 마카롱과 같은 역사를 나누지만 실제로는 다른 음식으로 알려져 있다.</div>
</div>
</div>
<div class="rellink relarticle mainarticle"><img alt="" data-file-height="128" data-file-width="128" height="16" src="//upload.wikimedia.org/wikipedia/commons/thumb/e/ec/Crystal_Clear_app_xmag.svg/16px-Crystal_Clear_app_xmag.svg.png" srcset="//upload.wikimedia.org/wikipedia/commons/thumb/e/ec/Crystal_Clear_app_xmag.svg/24px-Crystal_Clear_app_xmag.svg.png 1.5x, //upload.wikimedia.org/wikipedia/commons/thumb/e/ec/Crystal_Clear_app_xmag.svg/32px-Crystal_Clear_app_xmag.svg.png 2x" width="16"/> 이 부분의 본문은 <a href="/wiki/%EB%A7%88%EC%B9%B4%EB%A3%AC" title="마카룬">마카룬</a>입니다.</div>
<p>마카롱(Macaron)과 <a href="/wiki/%EB%A7%88%EC%B9%B4%EB%A3%AC" title="마카룬">마카룬</a>(Macaroon)은 생김새부터 상당히 다른 과자지만, 대부분 사람들은 마카롱과 마카룬을 상당히 헷갈려한다. 마카룬은 <a href="/wiki/%EC%BD%94%EC%BD%94%EB%84%9B" title="코코넛">코코넛</a>을 이용하여 만드는 과자이며, 다양한 색깔의 마카롱과 달리 마카룬은 하나의 색으로 알려져 있다. 두 과자의 이름은 모두 마카로네(maccherone)에서 기인되었으며, 요리시 이용되는 재료도 <a href="/wiki/%ED%9D%B0%EC%9E%90" title="흰자">흰자</a>, <a href="/wiki/%EC%84%A4%ED%83%95" title="설탕">설탕</a>, 그리고 <a href="/wiki/%EC%95%84%EB%AA%AC%EB%93%9C" title="아몬드">아몬드</a> 또는 <a href="/wiki/%EC%BD%94%EC%BD%94%EB%84%9B" title="코코넛">코코넛</a> 등으로 유사하다.<sup class="reference" id="cite_ref-8"><a href="#cite_note-8">[8]</a></sup> 마카롱과 마카룬이 어떻게 나뉘었는가에 대해서는 다음과 같이 여러 설이 있다.<sup class="reference" id="cite_ref-Ruche_9-0"><a href="#cite_note-Ruche-9">[9]</a></sup></p>
<ul>
<li>마카롱은 프랑스에서 대인기를 얻는 반면, <a href="/wiki/%EB%B0%80%EA%B0%80%EB%A3%A8" title="밀가루">밀가루</a>를 많이 이용하지 않는 마카룬은 이탈리아계 <a href="/wiki/%EC%9C%A0%EB%8C%80%EC%9D%B8" title="유대인">유대인</a>들에게 <a href="/wiki/%EC%9C%A0%EC%9B%94%EC%A0%88" title="유월절">유월절</a> 음식으로 많은 인기를 얻었다. 이러하여 마카롱은 그 발전 과정에서 프랑스풍의 영향을 많이 받았던 반면, 마카룬은 유대교 전통 과자들의 영향을 받아 발전되었다.</li>
<li>1800년대 <a href="/wiki/%EB%AF%B8%EA%B5%AD" title="미국">미국</a>에서는 <a href="/wiki/%EC%9D%B8%EB%8F%84" title="인도">인도</a>에서 수출해온 <a href="/wiki/%EC%BD%94%EC%BD%94%EB%84%9B" title="코코넛">코코넛</a>을 이용한 후식거리가 유행이었다. 따라서, 마카롱에다 아몬드 페이스트를 넣는 대신 코코넛을 이용하여 마카룬의 모습으로 변화한 것이다.</li>
<li>마카룬은 마카롱과는 관련 없이 별도로 <a href="/wiki/%EC%98%81%EA%B5%AD" title="영국">영국</a>의 <a href="/wiki/%EC%8A%A4%EC%BD%94%ED%8B%80%EB%9E%9C%EB%93%9C" title="스코틀랜드">스코틀랜드</a>에서 따로 시작되었다.</li>
</ul>
<p>이렇듯 불분명한 기원과 대채로 재료에도 불구하고, 마라롱과 마카룬의 기원과 이름에 대한 논쟁은 계속 진행 중이다.</p>
<h3><span class="mw-headline" id=".EB.A3.A9.EC.85.88.EB.B6.80.EB.A5.B4.EA.B2.8C.EB.A5.BC.EB.A6.AC">룩셈부르게를리</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%A7%88%EC%B9%B4%EB%A1%B1&amp;action=edit&amp;section=5" title="부분 편집: 룩셈부르게를리">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<div class="thumb tleft">
<div class="thumbinner" style="width:252px;"><a class="image" href="/wiki/%ED%8C%8C%EC%9D%BC:Luxemburgerli_Himbeere_Zitrone.jpg"><img alt="" class="thumbimage" data-file-height="2722" data-file-width="4083" height="167" src="//upload.wikimedia.org/wikipedia/commons/thumb/e/e9/Luxemburgerli_Himbeere_Zitrone.jpg/250px-Luxemburgerli_Himbeere_Zitrone.jpg" srcset="//upload.wikimedia.org/wikipedia/commons/thumb/e/e9/Luxemburgerli_Himbeere_Zitrone.jpg/375px-Luxemburgerli_Himbeere_Zitrone.jpg 1.5x, //upload.wikimedia.org/wikipedia/commons/thumb/e/e9/Luxemburgerli_Himbeere_Zitrone.jpg/500px-Luxemburgerli_Himbeere_Zitrone.jpg 2x" width="250"/></a>
<div class="thumbcaption">
<div class="magnify"><a class="internal" href="/wiki/%ED%8C%8C%EC%9D%BC:Luxemburgerli_Himbeere_Zitrone.jpg" title="실제 크기로"></a></div>
룩셈부르게를리 (Luxemburgerli)는 마카롱보다 크기가 작으며, 더욱 동그랗게 생겼다.</div>
</div>
</div>
<p><a href="/wiki/%EB%8F%85%EC%9D%BC" title="독일">독일</a>, <a href="/wiki/%EC%8A%A4%EC%9C%84%EC%8A%A4" title="스위스">스위스</a>, <a href="/wiki/%EB%A3%A9%EC%85%88%EB%B6%80%EB%A5%B4%ED%81%AC" title="룩셈부르크">룩셈부르크</a>와 같이 <a href="/wiki/%EB%8F%85%EC%9D%BC%EC%96%B4" title="독일어">독일어</a>를 이용하는 국가들에서는 주로 마카롱을 룩셈부르게를리(Luxemburgerli)라고 대신 부른다. 마카롱과 룩셈부르게를리는 매우 비슷하지만 룩셈부르게를리는 중간에 <a class="mw-disambig" href="/wiki/%ED%81%AC%EB%A6%BC" title="크림">크림</a>만 들어가는 방면, 마카롱은 크림 뿐만 아니라 <a href="/wiki/%EC%9E%BC" title="잼">잼</a>, <a href="/wiki/%EB%A7%88%EC%A7%80%ED%8C%AC" title="마지팬">마지팬</a>, 초콜릿 파나체 등을 넣어도 된다. 또한 룩셈부르게를리는 윗부분의 마카롱보다 더 둥글며 크기는 작고, 더욱 가격이 싸다고 한다(평균 유럽에서는 마카롱 하나가 2유로, 룩셈부르게를리는 0.4유로).<sup class="reference" id="cite_ref-10"><a href="#cite_note-10">[10]</a></sup></p>
<p>룩셈부르겔리는 1950년때에 그의 이름같이 룩셈부르크라는 서유럽 나라에서 온 사람이 발명하였다. 스프렁리(Sprüngli)라는 요리사가 <a href="/wiki/%EC%8A%A4%EC%9C%84%EC%8A%A4" title="스위스">스위스</a>의 <a href="/wiki/%EC%B7%A8%EB%A6%AC%ED%9E%88" title="취리히">취리히</a>에서 무역을 공부하다 여기서 룩셈부르겔리를 만들기 시작했다. 계속 만들다 보니 나중에 유명해지기 시작했고, 사람들한테 이 과자를 뭐라고 부를까라고 묻더니 아무도 답변해주지 않아, 스프렁리가 출신한 국가인 룩셈부르크에서 이름을 따왔다.<sup class="reference" id="cite_ref-11"><a href="#cite_note-11">[11]</a></sup></p>

<p>마카롱은 대부분 과자류들과는 다르게 종류가 매우 다양하며, 정확한 원맛이 없다.</p>
<p>마카롱은 제작하는 것이 상당히 까다로운 음식이다. 아몬드 페이스트를 만들 때 아몬드 가루 크기에도 영향을 받을 만큼 조금이라도 실수를 하는 경우 그 결과의 차이가 크다. 완벽한 마카롱은 위가 살짝 둥글어 균형이 잡혀 있고, 밑에는 약간 구겨진 받침이 있다.<sup class="reference" id="cite_ref-12"><a href="#cite_note-12">[12]</a></sup><sup class="reference" id="cite_ref-13"><a href="#cite_note-13">[13]</a></sup></p>

<p><a href="/wiki/%EB%8C%80%ED%95%9C%EB%AF%BC%EA%B5%AD" title="대한민국">한국</a>에서는 마카롱의 인기가 매우 높은 편이며<sup class="reference" id="cite_ref-14"><a href="#cite_note-14">[14]</a></sup>, 최근에 마카롱을 판매하는 매점들이 증가하면서 자주 볼 수 있는 음식이 되었다. 특히 수도권을 중심으로 마카롱이 한국인 입맛에 알맞게 더욱 변화되고 있으며, 마카롱 아이스크림과 같은 이전에 없던 새로운 마카롱들도 만들어지고 있다.<sup class="reference" id="cite_ref-15"><a href="#cite_note-15">[15]</a></sup> 2012년 7월, <a class="mw-redirect" href="/wiki/%EC%84%9C%EC%9A%B8" title="서울">서울</a>에 프랑스의 <a href="/wiki/%EB%9D%BC%EB%92%A4%EB%A0%88" title="라뒤레">라뒤레</a> 마카롱 가게가 생겼다.</p>

<p><a href="/wiki/%ED%94%84%EB%9E%91%EC%8A%A4" title="프랑스">프랑스</a>와 <a href="/wiki/%EC%8A%A4%EC%B9%B8%EB%94%94%EB%82%98%EB%B9%84%EC%95%84" title="스칸디나비아">스칸디나비아</a>에는 마카롱을 기념하는 박물관이 있다. 또한 세계에서 최초로 3월 20일에 마카롱의 날을 제정하였다.</p>
<h3><span class="mw-headline" id=".EB.AF.B8.EA.B5.AD">미국</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%A7%88%EC%B9%B4%EB%A1%B1&amp;action=edit&amp;section=10" title="부분 편집: 미국">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p><a href="/wiki/%EB%AF%B8%EA%B5%AD" title="미국">미국</a>에서는 20세기까지 아무도 마카롱을 모른다고 할 만큼 잘 알려지지 않은 음식이었다. 1993년에 <a href="/wiki/%EB%89%B4%EC%9A%95" title="뉴욕">뉴욕</a>에서 다니엘 볼루드 (Daniel Boulud)라는 요리사가 미국에서 최초로 마카롱 사업을 열었다. 그 이후 마카롱의 인기가 서서히 증가하여 오늘날에 이르러서는 흔한 음식이 되었다. 하지만 그 인기가 <a href="/wiki/%ED%94%84%EB%9E%91%EC%8A%A4" title="프랑스">프랑스</a>나 <a href="/wiki/%EC%9C%A0%EB%9F%BD" title="유럽">유럽</a>에 미치지 못하며, 요리 전문가들은 미국의 마카롱은 매우 비싸거나 제대로 만들어지지 않는다고 비판하기도 한다.<sup class="reference" id="cite_ref-MadMac_5-2"><a href="#cite_note-MadMac-5">[5]</a></sup></p>
<p>2010년, 뉴욕에서는 뉴욕 마카롱의 날(Macaron Day NYC)이 생기면서 <a href="/wiki/%ED%8C%8C%EB%A6%AC_(%ED%94%84%EB%9E%91%EC%8A%A4)" title="파리 (프랑스)">파리</a>에 이어 세계 두번째로 마카롱을 기념하는 날이 생겼다.<sup class="reference" id="cite_ref-Mr._Macaron_6-1"><a href="#cite_note-Mr._Macaron-6">[6]</a></sup></p>
