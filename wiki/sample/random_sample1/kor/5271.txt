<p><b>마카베오기 상권</b> 또는 <b>마카베오기 제1권</b>은 <a href="/wiki/%EA%B8%B0%EB%8F%85%EA%B5%90" title="기독교">기독교</a>의 가톨릭과 정교회 및 개신교 일부 종파에서 <a class="mw-disambig" href="/wiki/%EC%A0%95%EA%B2%BD" title="정경">정경</a>으로 취급하는 <a href="/wiki/%EA%B5%AC%EC%95%BD%EC%84%B1%EA%B2%BD" title="구약성경">구약성경</a>이다. 요야립 가문의 사제이자 <a class="mw-disambig" href="/wiki/%EC%8B%9C%EB%A9%94%EC%98%A8" title="시메온">시메온</a>의 손자 <a class="new" href="/w/index.php?title=%EB%A7%88%ED%83%80%ED%8B%B0%EC%95%84%EC%8A%A4&amp;action=edit&amp;redlink=1" title="마타티아스 (없는 문서)">마타티아스</a>의 아들 <a href="/wiki/%EC%9C%A0%EB%8B%A4_%EB%A7%88%EC%B9%B4%EB%B2%A0%EC%98%A4" title="유다 마카베오">유다 마카베오</a>에 대해 쓴 역사서이다.</p>
<p><a class="mw-redirect" href="/wiki/%EC%97%90%EC%8A%A4%EB%9D%BC" title="에스라">에스라기</a> 이후 <a href="/wiki/%ED%8E%98%EB%A5%B4%EC%8B%9C%EC%95%84_%EC%A0%9C%EA%B5%AD" title="페르시아 제국">페르시아 제국</a>의 멸망과 유대인들의 저항에 대해 쓰고 있으며 상권에서는 <a href="/wiki/%ED%8E%98%EB%A5%B4%EC%8B%9C%EC%95%84_%EC%A0%9C%EA%B5%AD" title="페르시아 제국">페르시아 제국</a>의 멸망에서부터 유대인들의 저항 지도자 <a href="/wiki/%EC%8B%9C%EB%AA%AC_%EB%A7%88%EC%B9%B4%EB%B2%A0%EC%98%A4" title="시몬 마카베오">시몬</a>이 살해당하기까지를 기록하며 <a href="/wiki/%EC%85%80%EB%A0%88%EC%9A%B0%EC%BD%94%EC%8A%A4_%EC%A0%9C%EA%B5%AD" title="셀레우코스 제국">셀레우코스 제국</a>의 역사도 보여준다.</p>

<p><b>1장</b> <a href="/wiki/%EB%A7%88%EC%BC%80%EB%8F%84%EB%8B%88%EC%95%84_%EC%99%95%EA%B5%AD" title="마케도니아 왕국">마케도니아</a>의 임금 <a class="mw-redirect" href="/wiki/%EC%95%8C%EB%A0%89%EC%82%B0%EB%93%9C%EB%A1%9C%EC%8A%A4_%EB%8C%80%EC%99%95" title="알렉산드로스 대왕">알렉산드로스 대왕</a>은 정복 활동을 펼쳐 <a href="/wiki/%ED%8E%98%EB%A5%B4%EC%8B%9C%EC%95%84_%EC%A0%9C%EA%B5%AD" title="페르시아 제국">페르시아 제국</a>을 정벌하고 온 세상의 임금들을 죽여 넓은 영토와 제국을 확보한다. 그러나 33세의 젊은 나이에 죽고 제국은 그의 부하 장수들에 의해 나뉘고 그들을 세상을 온 악으로 가득 채웠다.</p>
<p><a href="/wiki/%EC%85%80%EB%A0%88%EC%9A%B0%EC%BD%94%EC%8A%A4_%EC%A0%9C%EA%B5%AD" title="셀레우코스 제국">그리스 왕국</a>의 왕으로 즉위한 <a class="mw-redirect" href="/wiki/%EC%95%88%ED%8B%B0%EC%98%A4%EC%BD%94%EC%8A%A4_4%EC%84%B8" title="안티오코스 4세">안티오코스 에피파네스</a>는 이민족의 풍습을 따르며 <a href="/wiki/%EC%9D%B4%EC%A7%91%ED%8A%B8" title="이집트">이집트</a>와 <a href="/wiki/%EC%9D%B4%EC%8A%A4%EB%9D%BC%EC%97%98" title="이스라엘">이스라엘</a>을 쳐 정복했다. 또한 <a href="/wiki/%EC%9C%A0%EB%8C%80%EC%9D%B8" title="유대인">유대인</a>들과 <a href="/wiki/%EC%9C%A0%EB%8C%80%EA%B5%90" title="유대교">유대교</a>를 박해했다.</p>
<p><b>2장</b> 이때 <a href="/wiki/%EC%9C%A0%EB%8B%A4_%EB%A7%88%EC%B9%B4%EB%B2%A0%EC%98%A4" title="유다 마카베오">마카베오</a>의 아버지이자 사제인 <a class="new" href="/w/index.php?title=%EB%A7%88%ED%83%80%ED%8B%B0%EC%95%84%EC%8A%A4&amp;action=edit&amp;redlink=1" title="마타티아스 (없는 문서)">마타티아스</a>는 그의 다섯 아들과 함께 이민족들의 제사를 거부하고 임금의 신하를 죽이고 제단을 헐어버린 뒤 아들들과 함께 산으로 도망쳤다. <a class="new" href="/w/index.php?title=%EB%A7%88%ED%8B%B0%ED%83%80%EC%95%84%EC%8A%A4&amp;action=edit&amp;redlink=1" title="마티타아스 (없는 문서)">마티타아스</a>는 산에서 그를 쫓아오는 군대에 맞서 투쟁을 하며 여러 곳을 돌아다니며 이민족들의 제단을 헐고 유대인들에게 <a href="/wiki/%ED%95%A0%EB%A1%80" title="할례">할례</a>를 베풀었다.</p>
<p><b>3장</b> <a class="new" href="/w/index.php?title=%EB%A7%88%ED%83%80%ED%8B%B0%EC%95%84%EC%8A%A4&amp;action=edit&amp;redlink=1" title="마타티아스 (없는 문서)">마타티아스</a>는 얼마 후 유언을 남기고 죽고 그의 아들 <a href="/wiki/%EC%9C%A0%EB%8B%A4_%EB%A7%88%EC%B9%B4%EB%B2%A0%EC%98%A4" title="유다 마카베오">유다 마카베오</a>가 그의 뒤를 잇는다. 유다 마카베오는 벳 호른에서 <a class="new" href="/w/index.php?title=%EC%95%84%ED%8F%B4%EB%A1%9C%EB%8B%88%EC%9A%B0%EC%8A%A4_(%EC%85%80%EB%A0%88%EC%9A%B0%EC%BD%94%EC%8A%A4_%EC%A0%9C%EA%B5%AD%EC%9D%98_%EC%9E%A5%EC%88%98)&amp;action=edit&amp;redlink=1" title="아폴로니우스 (셀레우코스 제국의 장수) (없는 문서)">아폴로니우스</a>가 이끄는 이민족들과 <a class="mw-redirect" href="/wiki/%EC%95%84%EC%8B%9C%EB%A6%AC%EC%95%84_%EC%A0%9C%EA%B5%AD" title="아시리아 제국">아시리아인</a>들에 맞서 승리를 거두었다. 이후 안티오코스는 페르시아 지역을 떠나며 왕족인 <a class="new" href="/w/index.php?title=%EB%A6%AC%EC%8B%9C%EC%95%84%EC%8A%A4&amp;action=edit&amp;redlink=1" title="리시아스 (없는 문서)">리시아스</a>를 섭정으로 임명한다.</p>
<p><a class="new" href="/w/index.php?title=%EB%A6%AC%EC%8B%9C%EC%95%84%EC%8A%A4&amp;action=edit&amp;redlink=1" title="리시아스 (없는 문서)">리시아스</a>는 <a class="new" href="/w/index.php?title=%EB%8F%84%EB%A6%AC%EB%A9%94%EB%84%A4%EC%8A%A4&amp;action=edit&amp;redlink=1" title="도리메네스 (없는 문서)">도리메네스</a>의 아들 <a class="new" href="/w/index.php?title=%ED%94%84%ED%86%A8%EB%A0%88%EB%A7%88%EC%9D%B4%EC%98%A4%EC%8A%A4_(%EC%85%80%EB%A0%88%EC%9A%B0%EC%BD%94%EC%8A%A4_%EC%A0%9C%EA%B5%AD%EC%9D%98_%EC%9E%A5%EC%88%98)&amp;action=edit&amp;redlink=1" title="프톨레마이오스 (셀레우코스 제국의 장수) (없는 문서)">프톨레마이오스</a>와 <a class="new" href="/w/index.php?title=%EB%8B%88%EC%B9%B4%EB%85%B8%EB%A5%B4&amp;action=edit&amp;redlink=1" title="니카노르 (없는 문서)">니카노르</a>, <a class="new" href="/w/index.php?title=%EA%B3%A0%EB%A5%B4%EA%B8%B0%EC%95%84%EC%8A%A4_(%EC%85%80%EB%A0%88%EC%9A%B0%EC%BD%94%EC%8A%A4_%EC%A0%9C%EA%B5%AD%EC%9D%98_%EC%9E%A5%EC%88%98)&amp;action=edit&amp;redlink=1" title="고르기아스 (셀레우코스 제국의 장수) (없는 문서)">고르기아스</a>를 부하로 뽑아 유대인들을 정벌하기 위해 군대를 파견한다. 그러나 온 <a href="/wiki/%EC%9C%A0%EB%8C%80%EC%9D%B8" title="유대인">유대인</a>들이 단결하고 <a href="/wiki/%EC%8B%9C%EB%A6%AC%EC%95%84" title="시리아">시리아</a> 군대와 <a href="/wiki/%ED%95%84%EB%A6%AC%EC%8B%9C%ED%85%8C%EC%9D%B8" title="필리시테인">필리시테인</a> 지방의 있던 군대도 유대인들과 힘을 합쳐 미츠파에 모인다.</p>
<p><b>4장</b> 미츠파에 모인 유대인들은 엠마오에서 <a class="new" href="/w/index.php?title=%EA%B3%A0%EB%A5%B4%EA%B8%B0%EC%95%84%EC%8A%A4_(%EC%85%80%EB%9D%BC%EC%9A%B0%EC%BD%94%EC%8A%A4_%EC%A0%9C%EA%B5%AD%EC%9D%98_%EC%9E%A5%EC%88%98)&amp;action=edit&amp;redlink=1" title="고르기아스 (셀라우코스 제국의 장수) (없는 문서)">고르기아스</a>를 무찌르고 대승을 거둔다. 이 소식을 들은 <a class="new" href="/w/index.php?title=%EB%A6%AC%EC%8B%9C%EC%95%84%EC%8A%A4&amp;action=edit&amp;redlink=1" title="리시아스 (없는 문서)">리시아스</a>는 크게 낙담하고 이듬해 유대인들을 제압하기 위해 보병 6만과 기병 5000을 소집해 <a class="new" href="/w/index.php?title=%EC%9D%B4%EB%91%90%EB%A7%A4%EC%95%84&amp;action=edit&amp;redlink=1" title="이두매아 (없는 문서)">이두매아</a> 지역으로 진격했으나 크게 패배하여 <a class="mw-redirect" href="/wiki/%EC%95%88%ED%8B%B0%EC%98%A4%ED%82%A4%EC%95%84" title="안티오키아">안티오키아</a>로 퇴각하고 만다.</p>
<p>그리고 마카베오와 그의 형제들은 성전을 정화하고 다시 봉헌했다.</p>
<p><b>5장</b> 제단이 복구되고 성소가 봉헌되었다는 소식을 듣은 이민족들은 유대인들을 공격했으나 아크라바테네에서 이두매인들은 크게 패하고 <a class="mw-redirect" href="/wiki/%EC%95%94%EB%AA%AC_(%EB%AF%BC%EC%A1%B1)" title="암몬 (민족)">암몬인</a>들 역시 정벌당한다. 그러자 이민족들은 자기 영토 안의 유대인들을 박해하기 시작했고 길앗과 <a href="/wiki/%EA%B0%88%EB%A6%B4%EB%9E%98%EC%95%84" title="갈릴래아">갈릴래아</a> 지역의 유대인들이 크게 핍박을 받았다.</p>
<p>그러자 <a href="/wiki/%EC%8B%9C%EB%AA%AC_%EB%A7%88%EC%B9%B4%EB%B2%A0%EC%98%A4" title="시몬 마카베오">시몬</a>이 나서 함께 <a href="/wiki/%EA%B0%88%EB%A6%B4%EB%9E%98%EC%95%84" title="갈릴래아">갈릴래아</a>로 가서 이민족들과 맞서 싸웠고 병사 3000명을 쓰러뜨렸으며 많은 전리품을 거두었다. 마카베오와 그의 동생 <a class="mw-redirect" href="/wiki/%EC%9A%94%EB%82%98%EB%8B%A8_(%EB%A7%88%ED%83%80%ED%8B%B0%EC%95%84%EC%8A%A4%EC%9D%98_%EC%95%84%EB%93%A4)" title="요나단 (마타티아스의 아들)">요나단</a>은 <a href="/wiki/%EC%9A%94%EB%A5%B4%EB%8B%A8_%EA%B0%95" title="요르단 강">요르단 강</a>을 건너 진군해 여러 이민족들과 <a class="new" href="/w/index.php?title=%ED%8B%B0%EB%AA%A8%ED%85%8C%EC%98%A4%EC%8A%A4&amp;action=edit&amp;redlink=1" title="티모테오스 (없는 문서)">티모테오스</a>를 무찔렀다. 하지만 <a class="mw-redirect" href="/wiki/%EC%A6%88%EC%B9%B4%EB%A5%B4%EC%95%BC" title="즈카르야">즈카르야</a>의 아들 <a class="new" href="/w/index.php?title=%EC%9A%94%EC%85%89_(%EC%A6%88%EC%B9%B4%EB%A5%B4%EC%95%BC%EC%9D%98_%EC%95%84%EB%93%A4)&amp;action=edit&amp;redlink=1" title="요셉 (즈카르야의 아들) (없는 문서)">요셉</a>과 <a href="/wiki/%EC%95%84%EC%9E%90%EB%A5%B4%EC%95%BC" title="아자르야">아자르야</a>는 과신을 하며 얌니아로 진군하다가 <a class="new" href="/w/index.php?title=%EA%B3%A0%EB%A5%B4%EA%B8%B0%EC%95%84%EC%8A%A4_(%EC%85%80%EB%A0%88%EC%9A%B0%EC%BD%94%EC%8A%A4_%EC%A0%9C%EA%B5%AD%EC%9D%98_%EC%9E%A5%EC%88%98)&amp;action=edit&amp;redlink=1" title="고르기아스 (셀레우코스 제국의 장수) (없는 문서)">고르기아스</a>에게 크게 패배했다.</p>
<p>유대인들은 <a class="new" href="/w/index.php?title=%EC%9D%B4%EB%91%90%EB%A7%A4%EC%95%84&amp;action=edit&amp;redlink=1" title="이두매아 (없는 문서)">이두매아</a>와 <a href="/wiki/%ED%95%84%EB%A6%AC%EC%8B%9C%ED%85%8C%EC%9D%B8" title="필리시테인">필리시테인</a> 지역에서 또 다시 대승을 거두고 많은 전리품들을 얻었다.</p>
<p><b>6장</b> 내륙 여러 지방을 돌아다니던 <a class="mw-redirect" href="/wiki/%EC%95%88%ED%8B%B0%EC%98%A4%EC%BD%94%EC%8A%A4_4%EC%84%B8" title="안티오코스 4세">안티오코스 에피파네스</a>는 <a class="mw-disambig" href="/wiki/%ED%8E%98%EB%A5%B4%EC%8B%9C%EC%95%84" title="페르시아">페르시아</a> 지역의 <a class="new" href="/w/index.php?title=%EC%97%98%EB%A6%AC%EB%A7%88%EC%9D%B4%EC%8A%A4&amp;action=edit&amp;redlink=1" title="엘리마이스 (없는 문서)">엘리마이스</a>를 공격했으나 오히려 패배했고 <a href="/wiki/%EB%B0%94%EB%B9%8C%EB%A1%A0" title="바빌론">바빌론</a>으로 향한다. 그곳에서 유대인들에게 자신의 군대가 크게 패했다는 소식을 들은 안티오코스는 큰 충격을 받았고 결국 화병이 나 죽고 만다. 안티오코스가 죽자 그의 아들 <a class="mw-redirect" href="/wiki/%EC%95%88%ED%8B%B0%EC%98%A4%EC%BD%94%EC%8A%A4_5%EC%84%B8" title="안티오코스 5세">안티오코스 에우파토르</a>가 뒤를 잇는다.</p>
<p><a href="/wiki/%EC%98%88%EB%A3%A8%EC%82%B4%EB%A0%98" title="예루살렘">예루살렘</a>에서는 성채에 있는 자들이 이민족들을 지원하며 못된 짓들을 하자 유대인들은 예루살렘 성채를 포위했다. 그러자 안티오코스가 나섰으나 벳 즈카르야 전투에서 유대인들에게 크게 패하고 벳 추르가 점령당했으며 시온이 포위되었다.</p>
<p>결국 안티오코스는 화친을 제의했고 <a class="new" href="/w/index.php?title=%EB%A6%AC%EC%8B%9C%EC%95%84%EC%8A%A4&amp;action=edit&amp;redlink=1" title="리시아스 (없는 문서)">리시아스</a>는 급히 철군하였다.</p>
<p><b>7장</b> 한편 <a class="mw-disambig" href="/wiki/%EC%85%80%EB%A0%88%EC%9A%B0%EC%BD%94%EC%8A%A4" title="셀레우코스">셀레우코스</a>의 아들 데메트리오스가 <a href="/wiki/%EB%A1%9C%EB%A7%88" title="로마">로마</a>에서 빠져나와 <a class="new" href="/w/index.php?title=%EB%A6%AC%EC%8B%9C%EC%95%84%EC%8A%A4&amp;action=edit&amp;redlink=1" title="리시아스 (없는 문서)">리시아스</a>와 <a class="mw-redirect" href="/wiki/%EC%95%88%ED%8B%B0%EC%98%A4%EC%BD%94%EC%8A%A4_5%EC%84%B8" title="안티오코스 5세">안티오코스 에우파토르</a>를 살해하고 <a class="mw-redirect" href="/wiki/%EB%8D%B0%EB%A9%94%ED%8A%B8%EB%A6%AC%EC%98%A4%EC%8A%A4_1%EC%84%B8_(%EC%85%80%EB%A0%88%EC%9A%B0%EC%BD%94%EC%8A%A4_%EC%A0%9C%EA%B5%AD)" title="데메트리오스 1세 (셀레우코스 제국)">데메트리오스 1세</a>로 왕위에 올랐다. 데메트리오스는 자신의 벗 <a class="new" href="/w/index.php?title=%EB%B0%94%ED%82%A4%EB%8D%B0%EC%8A%A4&amp;action=edit&amp;redlink=1" title="바키데스 (없는 문서)">바키데스</a>를 부하로 뽑고 <a href="/wiki/%EC%9C%A0%ED%94%84%EB%9D%BC%ED%85%8C%EC%8A%A4_%EA%B0%95" title="유프라테스 강">유프라테스 강</a> 서쪽 지방의 총독으로 임명하고 사악한 대사제 <a class="new" href="/w/index.php?title=%EC%95%8C%ED%82%A4%EB%AA%A8%EC%8A%A4&amp;action=edit&amp;redlink=1" title="알키모스 (없는 문서)">알키모스</a>를 보내 유대 지역을 정벌하도록 명령한다.</p>
<p>도중 <a class="new" href="/w/index.php?title=%EC%95%8C%ED%82%A4%EB%AA%A8%EC%8A%A4&amp;action=edit&amp;redlink=1" title="알키모스 (없는 문서)">알키모스</a>가 대사제직에서 해임당하려는 위기를 맞자 그 사이에 그의 군대가 이스라엘 전역을 약탈하고 큰 피해를 입힌다. 한편 옛 <a class="new" href="/w/index.php?title=%EB%A6%AC%EC%8B%9C%EC%95%84%EC%8A%A4&amp;action=edit&amp;redlink=1" title="리시아스 (없는 문서)">리시아스</a>의 부하였던 <a class="new" href="/w/index.php?title=%EB%8B%88%EC%B9%B4%EB%85%B8%EB%A5%B4&amp;action=edit&amp;redlink=1" title="니카노르 (없는 문서)">니카노르</a>는 데메트리오스에게로 귀화해 <a href="/wiki/%EC%8B%9C%EB%A6%AC%EC%95%84" title="시리아">시리아</a> 군대와 함께 카파르살라마 부근에서 유대인들과 맞서 싸웠다. 그러나 벳 호론 지역에서 크게 패하고 <a class="new" href="/w/index.php?title=%EB%8B%88%EC%B9%B4%EB%85%B8%EB%A5%B4&amp;action=edit&amp;redlink=1" title="니카노르 (없는 문서)">니카노르</a>는 전사했다.</p>
<p><b>8장</b> <a href="/wiki/%EC%9C%A0%EB%8C%80%EC%9D%B8" title="유대인">유대인</a>들은 <a class="mw-redirect" href="/wiki/%EB%A1%9C%EB%A7%88%EC%9D%B8" title="로마인">로마인</a>들의 명성을 듣고 <a class="new" href="/w/index.php?title=%EC%95%84%EC%BD%94%EC%B8%A0&amp;action=edit&amp;redlink=1" title="아코츠 (없는 문서)">아코츠</a>의 손자이자 <a class="new" href="/w/index.php?title=%EC%9A%94%ED%95%9C_(%EC%95%84%EC%BD%94%EC%B8%A0%EC%9D%98_%EC%95%84%EB%93%A4)&amp;action=edit&amp;redlink=1" title="요한 (아코츠의 아들) (없는 문서)">요한</a>의 아들 <a class="new" href="/w/index.php?title=%EC%97%90%EC%9A%B0%ED%8F%B4%EB%A0%88%EB%AA%A8%EC%8A%A4&amp;action=edit&amp;redlink=1" title="에우폴레모스 (없는 문서)">에우폴레모스</a>와 <a href="/wiki/%EC%97%98%EC%95%84%EC%9E%90%EB%A5%B4" title="엘아자르">엘아자르</a>의 아들 <a class="new" href="/w/index.php?title=%EC%95%BC%EC%86%90&amp;action=edit&amp;redlink=1" title="야손 (없는 문서)">야손</a>을 뽑아 <a href="/wiki/%EB%A1%9C%EB%A7%88" title="로마">로마</a>로 보내 우호 동맹을 맺게 하는데 유대와 로마가 위험에 처하면 언제든지 도와주겠다고 서로 약속했다.</p>
<p><b>9장</b> <a href="/wiki/%EC%85%80%EB%A0%88%EC%9A%B0%EC%BD%94%EC%8A%A4_%EC%A0%9C%EA%B5%AD" title="셀레우코스 제국">셀레우코스 제국</a>에서는 <a class="mw-redirect" href="/wiki/%EB%8D%B0%EB%A9%94%ED%8A%B8%EB%A6%AC%EC%98%A4%EC%8A%A4_1%EC%84%B8_(%EC%85%80%EB%A0%88%EC%9A%B0%EC%BD%94%EC%8A%A4_%EC%A0%9C%EA%B5%AD)" title="데메트리오스 1세 (셀레우코스 제국)">데메트리오스 1세</a>가 <a class="new" href="/w/index.php?title=%EB%8B%88%EC%B9%B4%EB%85%B8%EB%A5%B4&amp;action=edit&amp;redlink=1" title="니카노르 (없는 문서)">니카노르</a>가 전사했다는 소식을 듣고 직접 군대를 이끌고 <a class="new" href="/w/index.php?title=%EB%B0%94%ED%82%A4%EB%8D%B0%EC%8A%A4&amp;action=edit&amp;redlink=1" title="바키데스 (없는 문서)">바키데스</a>, <a class="new" href="/w/index.php?title=%EC%95%8C%ED%82%A4%EB%AA%A8%EC%8A%A4&amp;action=edit&amp;redlink=1" title="알키모스 (없는 문서)">알키모스</a>와 함께 유대 지방으로 쳐들어갔다. 그들은 행군을 하며 아르벨라의 매살롯에 진을 치고 그곳을 페허로 만들어 버렸고 군대 수에 놀란 유대군들을 전투를 치르기도 전에 도망쳐 버리고 말았다.</p>
<p>마지막 800명의 군사만 남자 <a href="/wiki/%EC%9C%A0%EB%8B%A4_%EB%A7%88%EC%B9%B4%EB%B2%A0%EC%98%A4" title="유다 마카베오">유다 마카베오</a>는 엘리사에 진을 치고 최후의 전투를 감행해 싸우다가 결국 전사했다. 이로써 유대군은 다 흩어지고 동생 <a class="mw-redirect" href="/wiki/%EC%9A%94%EB%82%98%EB%8B%A8_(%EB%A7%88%ED%83%80%ED%8B%B0%EC%95%84%EC%8A%A4%EC%9D%98_%EC%95%84%EB%93%A4)" title="요나단 (마타티아스의 아들)">요나단</a>과 저항 지도자 <a href="/wiki/%EC%8B%9C%EB%AA%AC_%EB%A7%88%EC%B9%B4%EB%B2%A0%EC%98%A4" title="시몬 마카베오">시몬</a>만이 살아남아 마카베오의 장례를 치렀다.</p>
<p>마카베오가 죽자 동생 요나단이 뒤를 잇게 되었다. 요나단은 군대를 재정비하여 아스파르 못 가에 진을 치고 싸워 바키데스를 무찔렀다. 바키데스는 도망쳐서 <a href="/wiki/%EC%98%88%EB%A6%AC%EC%BD%94" title="예리코">예리코</a>와 <a href="/wiki/%EC%98%88%EB%A3%A8%EC%82%B4%EB%A0%98" title="예루살렘">예루살렘</a>, 벳 호론, 베텔, 팀나 등 유대의 주요 성읍 요새들을 튼튼히 하고 수비했다.</p>
<p>이후 <a class="new" href="/w/index.php?title=%EC%95%8C%ED%82%A4%EB%AA%A8%EC%8A%A4&amp;action=edit&amp;redlink=1" title="알키모스 (없는 문서)">알키모스</a>는 죽고 바키데스는 벳 바시에서 요나단과 시몬에게 다시 크게 패해 유대를 떠나고 말았다.</p>
<p><b>10장</b> 쫓겨났던 <a href="/wiki/%EC%85%80%EB%A0%88%EC%9A%B0%EC%BD%94%EC%8A%A4_%EC%A0%9C%EA%B5%AD" title="셀레우코스 제국">셀레우코스 제국</a>의 전 왕 <a class="mw-redirect" href="/wiki/%EC%95%88%ED%8B%B0%EC%98%A4%EC%BD%94%EC%8A%A4_5%EC%84%B8" title="안티오코스 5세">안티오코스 에우파토르</a>의 아들 <a class="new" href="/w/index.php?title=%EC%95%8C%EB%A0%89%EC%82%B0%EB%93%9C%EB%A1%9C%EC%8A%A4_%EB%B0%9C%EB%9D%BC%EC%8A%A4&amp;action=edit&amp;redlink=1" title="알렉산드로스 발라스 (없는 문서)">알렉산드로스 에피파네스</a>는 반란을 일으켜 프톨레마이스에 상륙해 그곳을 점령한다. 그러자 데메트리오스는 군대를 보내고 요나단이 알렉산드로스와 화친을 맺기 전에 먼저 화친을 맺자는 작전을 세웠다.</p>
<p>또한 데메트리오스는 유대 민족에게 새로운 제안을 하기까지 했으나 알렉산드로스와 전투 중 전사하고 알렉산드로스가 왕위에 올라 이집트의 파라오 프톨레미오스에게 서신을 보내 딸 클레오파트라와 혼인을 한다. 그리고 알렉산드로스는 요나단과 맹약을 맺었다.</p>
<p>한편 데메트리오스의 잔당이자 아들인 데메트리오스 니카토르는 <a class="mw-redirect" href="/wiki/%ED%81%AC%EB%A0%88%ED%83%80_%EC%84%AC" title="크레타 섬">크레타 섬</a>에서 나와 <a class="new" href="/w/index.php?title=%EC%95%84%ED%8F%B4%EB%A1%9C%EB%8B%88%EC%9A%B0%EC%8A%A4_(%EC%85%80%EB%A0%88%EC%9A%B0%EC%BD%94%EC%8A%A4_%EC%A0%9C%EA%B5%AD%EC%9D%98_%EC%9E%A5%EC%88%98)&amp;action=edit&amp;redlink=1" title="아폴로니우스 (셀레우코스 제국의 장수) (없는 문서)">아폴로니우스</a>를 보낸다. 자신의 영토를 지나게 해달라는 <a class="new" href="/w/index.php?title=%EC%95%84%ED%8F%B4%EB%A1%9C%EB%8B%88%EC%9A%B0%EC%8A%A4_(%EC%85%80%EB%A0%88%EC%9A%B0%EC%BD%94%EC%8A%A4_%EC%A0%9C%EA%B5%AD%EC%9D%98_%EC%9E%A5%EC%88%98)&amp;action=edit&amp;redlink=1" title="아폴로니우스 (셀레우코스 제국의 장수) (없는 문서)">아폴로니우스</a>의 서신을 받은 요나단은 분노해 시몬과 함께 군사 1만 명을 뽑아 예루살렘에서 나와 아스돗에서 결전을 벌였고 아폴로니우스는 크게 패했으며 알렉산드로스 왕은 그 소식을 듣고 요나단을 더욱 영예롭게 하고 많은 선물을 보낸다.</p>
<p><b>11장</b> 아폴로니우스가 패배하자 데메트리오스 니카토르는 이집트 파라오 <a class="mw-disambig" href="/wiki/%ED%94%84%ED%86%A8%EB%A0%88%EB%A7%88%EC%9D%B4%EC%98%A4%EC%8A%A4" title="프톨레마이오스">프톨레마이오스</a>와 동맹을 맺었다. 그러자 <a class="new" href="/w/index.php?title=%EC%95%8C%EB%A0%89%EC%82%B0%EB%93%9C%EB%A1%9C%EC%8A%A4_%EB%B0%9C%EB%9D%BC%EC%8A%A4&amp;action=edit&amp;redlink=1" title="알렉산드로스 발라스 (없는 문서)">알렉산드로스 발라스</a>와 <a class="mw-disambig" href="/wiki/%ED%94%84%ED%86%A8%EB%A0%88%EB%A7%88%EC%9D%B4%EC%98%A4%EC%8A%A4" title="프톨레마이오스">프톨레마이오스</a>는 킬리키아 지역에서 전투를 벌였고 알렉산드로스는 전사했으며 프톨레마이오스도 3일 뒤 요새에서 사망한다. 그의 부하 장수들도 그곳 주민들에게 살해당하면서 데메트리오스 니카토르가 <a class="mw-disambig" href="/wiki/%EB%8D%B0%EB%A9%94%ED%8A%B8%EB%A6%AC%EC%98%A4%EC%8A%A4_2%EC%84%B8" title="데메트리오스 2세">데메트리오스 2세</a>로 즉위한다.</p>
<p>요나단은 이를 경계해 <a class="mw-disambig" href="/wiki/%EB%8D%B0%EB%A9%94%ED%8A%B8%EB%A6%AC%EC%98%A4%EC%8A%A4_2%EC%84%B8" title="데메트리오스 2세">데메트리오스 2세</a>와 동맹을 맺고 영토를 세 구역으로 나눈다. 한편 데메트리오스의 휘하 장수 <a href="/wiki/%EB%94%94%EC%98%A4%EB%8F%84%ED%86%A0%EC%8A%A4_%ED%8A%B8%EB%A6%AC%ED%8F%B0" title="디오도토스 트리폰">디오도토스 트리폰</a>은 흉계를 꾸며 데메트리오스의 아들 안티오코스 디오니수스를 아라비아인 <a class="new" href="/w/index.php?title=%EC%9D%B4%EB%A7%90%EC%BF%A0%EC%97%90&amp;action=edit&amp;redlink=1" title="이말쿠에 (없는 문서)">이말쿠에</a>에게로 보낸다. 요나단은 셀레우코스 제국에 반란이 일어나자 데메트리오스를 도와 <a class="mw-redirect" href="/wiki/%EC%95%88%ED%8B%B0%EC%98%A4%ED%82%A4%EC%95%84" title="안티오키아">안티오키아</a>로 용사 3000명을 보내 승리하지만 데메트리오스는 요나단이 자신과 한 약속을 어겨 사이가 나빠졌다.</p>
<p>그러자 요나단은 <a href="/wiki/%EB%94%94%EC%98%A4%EB%8F%84%ED%86%A0%EC%8A%A4_%ED%8A%B8%EB%A6%AC%ED%8F%B0" title="디오도토스 트리폰">디오도토스 트리폰</a>과 동맹을 맺고 겐네사렛 호숫가에서 데메트리오스와 전투를 벌여 승리한다.</p>
<p><b>12장</b> 요나단은 <a href="/wiki/%EB%A1%9C%EB%A7%88" title="로마">로마</a>와 맺은 동맹을 강화하고 <a href="/wiki/%EC%8A%A4%ED%8C%8C%EB%A5%B4%ED%83%80" title="스파르타">스파르타인</a>들에게도 편지를 보낸다. 요나단과 시몬은 하맛 지방에서 다시 한번 데메트리오스를 무찌르고 승리한다. 이때 트리폰이 데메트리오스의 아들 안티오코스를 죽이고 왕이 되려 하자 요나단은 이를 용납하지 않고 트리폰을 공격한다.</p>
<p>그러자 트리폰은 요나단을 후히 대접하고 군사를 돌려보내게 한다. 그러나 이것은 트리폰의 흉계였고 요나단은 그곳 주민들에게 붙잡혔고 그의 부하 장수들과 군사들이 살해당한다. 트리폰은 군대를 보내 남은 요나단의 군사들을 전멸시키려 했으나 오히려 패했다.</p>
<p><b>13장</b> 요나단이 사로잡히자 그의 형 시몬이 뒤를 이었고 마침내 요나단은 트리폰에게 살해당했다. <a href="/wiki/%EC%85%80%EB%A0%88%EC%9A%B0%EC%BD%94%EC%8A%A4_%EC%A0%9C%EA%B5%AD" title="셀레우코스 제국">셀레우코스 제국</a>에서는 <a href="/wiki/%EB%94%94%EC%98%A4%EB%8F%84%ED%86%A0%EC%8A%A4_%ED%8A%B8%EB%A6%AC%ED%8F%B0" title="디오도토스 트리폰">디오도토스 트리폰</a>이 <a class="mw-disambig" href="/wiki/%EB%8D%B0%EB%A9%94%ED%8A%B8%EB%A6%AC%EC%98%A4%EC%8A%A4_2%EC%84%B8" title="데메트리오스 2세">데메트리오스 2세</a>를 폐위시키고 안티오코스 디오니수스를 <a class="mw-redirect" href="/wiki/%EC%95%88%ED%8B%B0%EC%98%A4%EC%BD%94%EC%8A%A4_6%EC%84%B8" title="안티오코스 6세">안티오코스 6세</a>로 왕위에 올려 모시다가 도리어 그를 살해하고 왕위에 올랐다.</p>
<p><b>14장</b> 시몬은 쫓겨난 <a class="mw-disambig" href="/wiki/%EB%8D%B0%EB%A9%94%ED%8A%B8%EB%A6%AC%EC%98%A4%EC%8A%A4_2%EC%84%B8" title="데메트리오스 2세">데메트리오스 2세</a>와 동맹을 맺고 트리폰에 맞서 가자라에서 진을 치고 전투 끝에 점령하였고 <a href="/wiki/%EC%98%88%EB%A3%A8%EC%82%B4%EB%A0%98" title="예루살렘">예루살렘</a> 성채에서는 악한 무리들을 쫓아냈다. 그러다 데메트리오스가 트리폰과 싸우다가 <a class="mw-disambig" href="/wiki/%ED%8E%98%EB%A5%B4%EC%8B%9C%EC%95%84" title="페르시아">페르시아</a>와 <a href="/wiki/%EB%A9%94%EB%94%94%EC%95%84" title="메디아">메디아</a>의 왕 <a class="new" href="/w/index.php?title=%EC%95%84%EB%A5%B4%EC%82%AC%EC%BC%80%EC%8A%A4&amp;action=edit&amp;redlink=1" title="아르사케스 (없는 문서)">아르사케스</a>에게 사로잡히고 만다.</p>
<p>시몬은 <a href="/wiki/%EB%A1%9C%EB%A7%88" title="로마">로마</a>와 <a href="/wiki/%EC%8A%A4%ED%8C%8C%EB%A5%B4%ED%83%80" title="스파르타">스파르타</a>와 맺은 동맹을 재확인하고 부흥을 꾀한다.</p>
<p><b>15장</b> <a class="mw-redirect" href="/wiki/%EC%95%88%ED%8B%B0%EC%98%A4%EC%BD%94%EC%8A%A4_6%EC%84%B8" title="안티오코스 6세">안티오코스 6세</a>의 아들 안티오코스 시데테스는 유대인들에게 편지를 보내고 트리폰을 공격한다. 시몬은 안티오코스와 동맹을 맺고 트리폰과 싸우려 했으나 안티오코스는 시몬을 배반하고 말았다. 트리폰은 배를 타고 오르토시아로 달아나고 안티오코스 시데테스가 <a class="mw-redirect" href="/wiki/%EC%95%88%ED%8B%B0%EC%98%A4%EC%BD%94%EC%8A%A4_7%EC%84%B8" title="안티오코스 7세">안티오코스 7세</a>로 왕위에 올랐다.</p>
<p>안티오코스는 <a class="new" href="/w/index.php?title=%EC%BC%84%EB%8D%B0%EB%B0%B0%EC%98%A4%EC%8A%A4&amp;action=edit&amp;redlink=1" title="켄데배오스 (없는 문서)">켄데배오스</a>를 해안 지역 수장으로 임명하고 유대를 침공하도록 한다. <a class="new" href="/w/index.php?title=%EC%BC%84%EB%8D%B0%EB%B0%B0%EC%98%A4%EC%8A%A4&amp;action=edit&amp;redlink=1" title="켄데배오스 (없는 문서)">켄데배오스</a>는 <a class="mw-redirect" href="/wiki/%EC%96%8C%EB%8B%88%EC%95%84" title="얌니아">얌니아</a>로 가 물건을 약탈하고 백성들을 괴롭혔다. 시몬은 아들 <a class="new" href="/w/index.php?title=%EC%9C%A0%EB%8B%A4_(%EC%8B%9C%EB%AA%AC%EC%9D%98_%EC%95%84%EB%93%A4)&amp;action=edit&amp;redlink=1" title="유다 (시몬의 아들) (없는 문서)">유다</a>, <a class="new" href="/w/index.php?title=%EC%9A%94%ED%95%9C_(%EC%8B%9C%EB%AA%AC%EC%9D%98_%EC%95%84%EB%93%A4)&amp;action=edit&amp;redlink=1" title="요한 (시몬의 아들) (없는 문서)">요한</a>과 <a class="new" href="/w/index.php?title=%EC%BC%84%EB%8D%B0%EB%B0%B0%EC%98%A4%EC%8A%A4&amp;action=edit&amp;redlink=1" title="켄데배오스 (없는 문서)">켄데배오스</a>를 공격해 대승을 거두었다.</p>
<p><b>16장</b> 그러나 그 무렵 <a class="new" href="/w/index.php?title=%EC%95%84%EB%B6%80%EB%B3%B4%EC%8A%A4&amp;action=edit&amp;redlink=1" title="아부보스 (없는 문서)">아부보스</a>의 아들 <a class="mw-disambig" href="/wiki/%ED%94%84%ED%86%A8%EB%A0%88%EB%A7%88%EC%9D%B4%EC%98%A4%EC%8A%A4" title="프톨레마이오스">프톨레마이오스</a>가 <a href="/wiki/%EC%98%88%EB%A6%AC%EC%BD%94" title="예리코">예리코</a> 평야에서 장수로 임명되었다. 그런데 그는 교만해져 시몬과 아들들을 없앨 흉계를 꾸몄다. 시몬은 자기 아들 <a class="new" href="/w/index.php?title=%EB%A7%88%ED%83%80%ED%8B%B0%EC%95%84%EC%8A%A4_(%EC%8B%9C%EB%AA%AC%EC%9D%98_%EC%95%84%EB%93%A4)&amp;action=edit&amp;redlink=1" title="마타티아스 (시몬의 아들) (없는 문서)">마타티아스</a>와 <a class="new" href="/w/index.php?title=%EC%9C%A0%EB%8B%A4_(%EC%8B%9C%EB%AA%AC%EC%9D%98_%EC%95%84%EB%93%A4)&amp;action=edit&amp;redlink=1" title="유다 (시몬의 아들) (없는 문서)">유다</a>를 데리고 <a href="/wiki/%EC%98%88%EB%A6%AC%EC%BD%94" title="예리코">예리코</a>로 내렸갔다. 기회를 였보던 <a class="mw-disambig" href="/wiki/%ED%94%84%ED%86%A8%EB%A0%88%EB%A7%88%EC%9D%B4%EC%98%A4%EC%8A%A4" title="프톨레마이오스">프톨레마이오스</a>는 흉계를 써서 시몬은 술에 취하게 만든 뒤 덮쳐 시몬과 그의 아들들을 살해하였다.</p>
<p>그 뒤 <a class="mw-disambig" href="/wiki/%ED%94%84%ED%86%A8%EB%A0%88%EB%A7%88%EC%9D%B4%EC%98%A4%EC%8A%A4" title="프톨레마이오스">프톨레마이오스</a>는 이 사실을 <a href="/wiki/%EC%85%80%EB%A0%88%EC%9A%B0%EC%BD%94%EC%8A%A4_%EC%A0%9C%EA%B5%AD" title="셀레우코스 제국">셀레우코스 제국</a>의 왕 <a class="mw-redirect" href="/wiki/%EC%95%88%ED%8B%B0%EC%98%A4%EC%BD%94%EC%8A%A4_7%EC%84%B8" title="안티오코스 7세">안티오코스 7세</a>에게 알려 군사를 보내달라고 했다. 그리고 시몬의 남은 아들 <a class="new" href="/w/index.php?title=%EC%9A%94%ED%95%9C_(%EC%8B%9C%EB%AA%AC%EC%9D%98_%EC%95%84%EB%93%A4)&amp;action=edit&amp;redlink=1" title="요한 (시몬의 아들) (없는 문서)">요한</a>을 죽이려고 가자라로 군대를 보냈다.</p>
<p>요한은 이에 깜짝 놀라 자기들을 죽이러 온 병사들을 미리 알고 그들을 죽였다. 이를 끝으로 마카베오기 상권의 기록은 마무리된다.</p>
