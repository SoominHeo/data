<b>마케도니아 지도와 현대의 국경선</b> - 붉은색은 <a href="/wiki/%EB%A7%88%EC%BC%80%EB%8F%84%EB%8B%88%EC%95%84_%EA%B3%B5%ED%99%94%EA%B5%AD" title="마케도니아 공화국">마케도니아 공화국</a>, 그 남서쪽의 붉은색은 <a href="/wiki/%EC%95%8C%EB%B0%94%EB%8B%88%EC%95%84" title="알바니아">알바니아</a>령, 남쪽의 밝은색은 <a href="/wiki/%EB%A7%88%EC%BC%80%EB%8F%84%EB%8B%88%EC%95%84_(%EA%B7%B8%EB%A6%AC%EC%8A%A4)" title="마케도니아 (그리스)">그리스령</a>, 그 동북쪽의 녹색은 <a href="/wiki/%EB%B6%88%EA%B0%80%EB%A6%AC%EC%95%84" title="불가리아">불가리아</a>령</div>
</div>
</div>
<p><b>마케도니아</b>는 <a href="/wiki/%EB%82%A8%EC%9C%A0%EB%9F%BD" title="남유럽">남유럽</a> <a href="/wiki/%EB%B0%9C%EC%B9%B8_%EB%B0%98%EB%8F%84" title="발칸 반도">발칸 반도</a>에 있는 지리ㆍ역사상의 지역을 일컫는다. 그 경계는 시대에 따라 크게 바뀌었으나, 마케도니아는 보통 <a href="/wiki/%EA%B7%B8%EB%A6%AC%EC%8A%A4" title="그리스">그리스</a>, <a href="/wiki/%EB%A7%88%EC%BC%80%EB%8F%84%EB%8B%88%EC%95%84_%EA%B3%B5%ED%99%94%EA%B5%AD" title="마케도니아 공화국">마케도니아 공화국</a>, <a href="/wiki/%EB%B6%88%EA%B0%80%EB%A6%AC%EC%95%84" title="불가리아">불가리아</a>, <a href="/wiki/%EC%95%8C%EB%B0%94%EB%8B%88%EC%95%84" title="알바니아">알바니아</a>, <a href="/wiki/%EC%84%B8%EB%A5%B4%EB%B9%84%EC%95%84" title="세르비아">세르비아</a> 발칸 반도 5개국의 각 영토를 이룬다. 면적은 대략 67,000km²에 이르며, 인구는 약 476만 명이다.</p>
<p>마케도니아는 <a class="new" href="/w/index.php?title=%EC%95%8C%EB%A6%AC%EC%95%84%ED%81%AC%EB%AA%AC_%EA%B0%95&amp;action=edit&amp;redlink=1" title="알리아크몬 강 (없는 문서)">알리아크몬 강</a>, <a href="/wiki/%EB%B0%94%EB%A5%B4%EB%8B%A4%EB%A5%B4_%EA%B0%95" title="바르다르 강">바르다르 강</a>, <a href="/wiki/%EC%8A%A4%ED%8A%B8%EB%A3%A8%EB%A7%88_%EA%B0%95" title="스트루마 강">스트루마 강</a>, <a class="new" href="/w/index.php?title=%EB%A9%94%EC%8A%A4%ED%83%80_%EA%B0%95&amp;action=edit&amp;redlink=1" title="메스타 강 (없는 문서)">메스타 강</a>이 흐르는 분지와 <a href="/wiki/%ED%85%8C%EC%82%B4%EB%A1%9C%EB%8B%88%ED%82%A4" title="테살로니키">테살로니키</a> 및 <a href="/wiki/%EC%84%B8%EB%A0%88%EC%8A%A4_(%EA%B7%B8%EB%A6%AC%EC%8A%A4)" title="세레스 (그리스)">세레스</a> 주변 평원을 아우른다.</p>
