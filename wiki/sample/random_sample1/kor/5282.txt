<h1 class="firstHeading" id="firstHeading" lang="ko">마크 러시노비치</h1>
<div class="mw-body-content" id="bodyContent">
<div id="siteSub">위키백과, 우리 모두의 백과사전.</div>
<div id="contentSub"></div>
<div class="mw-jump" id="jump-to-nav">
					이동:					<a href="#mw-head">둘러보기</a>, 					<a href="#p-search">검색</a>
</div>
<div class="mw-content-ltr" dir="ltr" id="mw-content-text" lang="ko"><script>function mfTempOpenSection(id){var block=document.getElementById("mf-section-"+id);block.className+=" open-block";block.previousSibling.className+=" open-block";}</script><p><b>마크 러시노비치</b>(Mark Russinovich, 1966년 경으로 태어난 것으로 추측~ )는 <a href="/wiki/%EB%A7%88%EC%9D%B4%ED%81%AC%EB%A1%9C%EC%86%8C%ED%94%84%ED%8A%B8" title="마이크로소프트">마이크로소프트</a>를 위하여 기술 분야에서 일하는 <a class="new" href="/w/index.php?title=%EC%86%8C%ED%94%84%ED%8A%B8%EC%9B%A8%EC%96%B4_%EA%B3%B5%ED%95%99%EC%9E%90&amp;action=edit&amp;redlink=1" title="소프트웨어 공학자 (없는 문서)">소프트웨어 공학자</a>이자 <a href="/wiki/%EC%9E%91%EA%B0%80" title="작가">저자</a>이다. 그는 "윈도 2000 아키텍처"(Architecture of Windows 2000)라는 주제로 테크넷 매거진(<i>Technet Magazine</i>)과 윈도 IT 프로(<i>Windows IT Pro</i>) 잡지(이전의 <i>Windows NT Magazine</i>)에 공헌을 하였으며 <i>Inside Windows 2000</i> 제3판의 공저자이기도 하다. 러시노비치는 <a class="mw-redirect" href="/wiki/%EC%9C%88%EB%8F%84_NT" title="윈도 NT">윈도 NT</a>, <a class="mw-redirect" href="/wiki/%EC%9C%88%EB%8F%84_2000" title="윈도 2000">윈도 2000</a> 커널 모드 <a href="/wiki/%ED%94%84%EB%A1%9C%EA%B7%B8%EB%9E%98%EB%A8%B8" title="프로그래머">프로그래머</a>에 쓰이는 도구들과 <a href="/wiki/%EB%8F%84%EC%8A%A4" title="도스">도스</a>용 <a href="/wiki/NTFS" title="NTFS">NTFS</a> 파일 시스템 드라이버를 위한 도구들을 많이 만들어내었다. 그는 윈도 전문가로도 널리 알려져 있다.</p>
