<p><b>마키노 도미타로</b>(牧野富太郎 ; <a href="/wiki/1862%EB%85%84" title="1862년">1862년</a> <a href="/wiki/5%EC%9B%94_22%EC%9D%BC" title="5월 22일">5월 22일</a> ~ <a href="/wiki/1957%EB%85%84" title="1957년">1957년</a> <a href="/wiki/1%EC%9B%94_18%EC%9D%BC" title="1월 18일">1월 18일</a>)는 <a href="/wiki/%EC%9D%BC%EB%B3%B8" title="일본">일본</a>의 <a href="/wiki/%EC%8B%9D%EB%AC%BC" title="식물">식물</a> 분류<a class="mw-redirect" href="/wiki/%ED%95%99%EC%9E%90" title="학자">학자</a>이다. <a href="/wiki/%EA%B3%A0%EC%B9%98_%ED%98%84" title="고치 현">고치</a>에서 출생하여 <a class="mw-redirect" href="/wiki/%EB%8F%84%EC%BF%84%EB%8C%80%ED%95%99" title="도쿄대학">도쿄대학</a> 강사를 지냈다. 일본 식물 <a href="/wiki/%EC%97%B0%EA%B5%AC" title="연구">연구</a>에 종사하여 <a href="/wiki/1927%EB%85%84" title="1927년">1927년</a>에 <a href="/wiki/%ED%95%99%EC%9C%84" title="학위">학위</a>를 받았다. <a href="/wiki/%EC%9D%BC%EB%B3%B8%EC%9D%B8" title="일본인">일본인</a>으로 일본 식물에 <a href="/wiki/%ED%95%99%EB%AA%85" title="학명">학명</a>을 붙인 최초의 사람으로 그가 <a href="/wiki/%EC%9D%B4%EB%A6%84" title="이름">이름</a>을 붙인 식물은 1000여 종, 1500여 변종에 이른다. <a href="/wiki/1936%EB%85%84" title="1936년">1936년</a>에 과거 50년 동안의 연구를 집대성한 《마키노 식물학 전집》을 완성하였다. <a href="/wiki/%ED%95%99%EB%AA%85" title="학명">학명</a> 명명자 부분의 ‘Makino’가 그를 가리킨다.</p>
