<p><b>마테야 케주만</b>(<span style="font-size: smaller;"><a href="/wiki/%EC%84%B8%EB%A5%B4%EB%B9%84%EC%95%84%EC%96%B4" title="세르비아어">세르비아어</a>:</span> <span lang="sr" xml:lang="sr">Матеја Кежман, Mateja Kežman</span>, <a href="/wiki/1979%EB%85%84" title="1979년">1979년</a> <a href="/wiki/4%EC%9B%94_12%EC%9D%BC" title="4월 12일">4월 12일</a>, <a href="/wiki/%EC%9C%A0%EA%B3%A0%EC%8A%AC%EB%9D%BC%EB%B9%84%EC%95%84_%EC%82%AC%ED%9A%8C%EC%A3%BC%EC%9D%98_%EC%97%B0%EB%B0%A9%EA%B3%B5%ED%99%94%EA%B5%AD" title="유고슬라비아 사회주의 연방공화국">유고슬라비아 사회주의 연방공화국</a> <a href="/wiki/%EC%A0%9C%EB%AC%B8_(%EB%B2%A0%EC%98%A4%EA%B7%B8%EB%9D%BC%EB%93%9C)" title="제문 (베오그라드)">제문</a> ~ )은 은퇴한 <a href="/wiki/%EC%84%B8%EB%A5%B4%EB%B9%84%EC%95%84" title="세르비아">세르비아</a>의 축구 선수이다. 포지션은 <a href="/wiki/%EA%B3%B5%EA%B2%A9%EC%88%98" title="공격수">공격수</a>였으며, 최전방에서 골을 넣는 스트라이커의 역할을 수행했다. 현재는 <a href="/wiki/%EC%8A%A4%ED%8F%AC%EC%B8%A0_%EC%97%90%EC%9D%B4%EC%A0%84%ED%8A%B8" title="스포츠 에이전트">스포츠 에이전트</a>로 활동 중이다.</p>

<h2><span class="mw-headline" id=".EC.84.A0.EC.88.98_.EA.B2.BD.EB.A0.A5">선수 경력</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%A7%88%ED%85%8C%EC%95%BC_%EC%BC%80%EC%A3%BC%EB%A7%8C&amp;action=edit&amp;section=1" title="부분 편집: 선수 경력">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<h3><span class="mw-headline" id=".ED.81.B4.EB.9F.BD_.EA.B2.BD.EB.A0.A5">클럽 경력</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%A7%88%ED%85%8C%EC%95%BC_%EC%BC%80%EC%A3%BC%EB%A7%8C&amp;action=edit&amp;section=2" title="부분 편집: 클럽 경력">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>케주만은 <a href="/wiki/1998%EB%85%84" title="1998년">1998년</a> <a href="/wiki/FK_%ED%8C%8C%EB%A5%B4%ED%8B%B0%EC%9E%94" title="FK 파르티잔">FK 파르티잔</a>으로 이적하여 좋은 모습을 보인 뒤, <a href="/wiki/2000%EB%85%84" title="2000년">2000년</a> 1050만 유로의 이적료로 <a class="mw-redirect" href="/wiki/%EB%84%A4%EB%8D%9C%EB%9E%80%EB%93%9C_%EC%97%90%EB%A0%88%EB%94%94%EB%B9%84%EC%8B%9C" title="네덜란드 에레디비시">네덜란드 에레디비시</a>의 <a href="/wiki/PSV_%EC%97%90%EC%9D%B8%ED%8A%B8%ED%98%B8%EB%B2%88" title="PSV 에인트호번">PSV 에인트호번</a>으로 이적하였다. 그는 <a href="/wiki/2001%EB%85%84" title="2001년">2001년</a>과 <a href="/wiki/2003%EB%85%84" title="2003년">2003년</a>, <a href="/wiki/2004%EB%85%84" title="2004년">2004년</a>에 걸쳐서 세 번의 리그 득점왕을 차지하였고, 2000-01 시즌과 2002-03 시즌 2번의 리그 우승에 공헌하였다. 그는 자신의 최고 전성기라고 불러도 손색이 없을 정도로 대단한 활약을 펼쳤고, 이러한 활약으로 인하여 <a href="/wiki/%ED%94%84%EB%A6%AC%EB%AF%B8%EC%96%B4%EB%A6%AC%EA%B7%B8" title="프리미어리그">프리미어리그</a>, <a href="/wiki/%EC%84%B8%EB%A6%AC%EC%97%90_A" title="세리에 A">세리에 A</a> 등의 거대 클럽의 관심을 받게 되었다.</p>
<p>그는 <a href="/wiki/2004%EB%85%84" title="2004년">2004년</a> 여름 이적시장에서 530만 파운드의 이적료로 <a class="mw-redirect" href="/wiki/%EC%9E%89%EA%B8%80%EB%9E%9C%EB%93%9C_%ED%94%84%EB%A6%AC%EB%AF%B8%EC%96%B4%EB%A6%AC%EA%B7%B8" title="잉글랜드 프리미어리그">잉글랜드 프리미어리그</a> <a href="/wiki/%EC%B2%BC%EC%8B%9C_FC" title="첼시 FC">첼시 FC</a>로 이적하였다. 그는 2004-05 시즌 <a href="/wiki/%EB%A6%AC%EB%B2%84%ED%92%80_FC" title="리버풀 FC">리버풀 FC</a>과의 <a class="mw-redirect" href="/wiki/%ED%92%8B%EB%B3%BC_%EB%A6%AC%EA%B7%B8_%EC%BB%B5" title="풋볼 리그 컵">풋볼 리그 컵</a> 결승전에서 결승골을 기록하기도 하였지만, 큰 인상을 심어주지 못하며 결국 한 시즌 뒤인 <a href="/wiki/2005%EB%85%84" title="2005년">2005년</a> 여름 <a class="mw-redirect" href="/wiki/%EC%8A%A4%ED%8E%98%EC%9D%B8_%ED%94%84%EB%A6%AC%EB%A9%94%EB%9D%BC%EB%A6%AC%EA%B0%80" title="스페인 프리메라리가">스페인 프리메라리가</a>의 <a href="/wiki/%EC%95%84%ED%8B%80%EB%A0%88%ED%8B%B0%EC%BD%94_%EB%A7%88%EB%93%9C%EB%A6%AC%EB%93%9C" title="아틀레티코 마드리드">아틀레티코 마드리드</a>로 이적하였다. 그는 <a href="/wiki/%ED%8E%98%EB%A5%B4%EB%82%9C%EB%8F%84_%ED%86%A0%EB%A0%88%EC%8A%A4" title="페르난도 토레스">페르난도 토레스</a>와 투톱을 이루며 경기에 출장하였지만, 이곳에서도 많은 골을 넣지는 못하며 실망스러운 모습을 보였다. 다시 한 시즌 뒤 그는 <a class="mw-redirect" href="/wiki/%ED%84%B0%ED%82%A4_%EC%88%98%ED%8E%98%EB%A5%B4%EB%A6%AC%EA%B7%B8" title="터키 수페르리그">터키 수페르리그</a>의 <a class="mw-redirect" href="/wiki/%ED%8E%98%EB%84%A4%EB%A5%B4%EB%B0%94%EC%B2%B4_SK" title="페네르바체 SK">페네르바체 SK</a>로 이적하였고, 그 곳에서 두 시즌 동안 준수한 활약을 펼치며 자신의 기량을 재회복하는 발판으로 삼았다.</p>
<p>그는 <a href="/wiki/2008%EB%85%84" title="2008년">2008년</a> <a href="/wiki/8%EC%9B%94_19%EC%9D%BC" title="8월 19일">8월 19일</a> 여름 이적시장에서 400만 유로의 이적료로 <a href="/wiki/%EB%A6%AC%EA%B7%B8_1" title="리그 1">리그 1</a>의 <a href="/wiki/%ED%8C%8C%EB%A6%AC_%EC%83%9D%EC%A0%9C%EB%A5%B4%EB%A7%B9_FC" title="파리 생제르맹 FC">파리 생제르맹 FC</a>으로 1년 임대를 떠나게 되었고, 시즌이 끝난 뒤 파리 생제르맹 FC에서 100만 유로의 이적료를 추가로 지불하여 완전 이적에 성공하였다. 그는 <a href="/wiki/2009%EB%85%84" title="2009년">2009년</a> <a href="/wiki/8%EC%9B%94_30%EC%9D%BC" title="8월 30일">8월 30일</a> <a href="/wiki/%EB%9F%AC%EC%8B%9C%EC%95%84_%ED%94%84%EB%A6%AC%EB%AF%B8%EC%96%B4%EB%A6%AC%EA%B7%B8" title="러시아 프리미어리그">러시아 프리미어리그</a>의 <a href="/wiki/FC_%EC%A0%9C%EB%8B%88%ED%8A%B8_%EC%83%81%ED%8A%B8%ED%8E%98%ED%85%8C%EB%A5%B4%EB%B6%80%EB%A5%B4%ED%81%AC" title="FC 제니트 상트페테르부르크">FC 제니트 상트페테르부르크</a>로 임대 이적하였고, 시즌이 끝난 뒤 복귀하였다. <a href="/wiki/2011%EB%85%84" title="2011년">2011년</a> <a href="/wiki/1%EC%9B%94_20%EC%9D%BC" title="1월 20일">1월 20일</a> <a class="mw-redirect" href="/wiki/%EC%82%AC%EC%9A%B0%EC%8A%A4_%EC%B0%A8%EC%9D%B4%EB%82%98" title="사우스 차이나">사우스 차이나</a>에 입단했고, 계약 기간이 끝나자 <a href="/wiki/8%EC%9B%94_31%EC%9D%BC" title="8월 31일">8월 31일</a> <a class="mw-redirect" href="/wiki/FC_BATE" title="FC BATE">FC BATE</a>에 입단해 활약했다. <a href="/wiki/2012%EB%85%84" title="2012년">2012년</a> <a class="mw-redirect" href="/wiki/%EC%82%AC%EC%9A%B0%EC%8A%A4_%EC%B0%A8%EC%9D%B4%EB%82%98" title="사우스 차이나">사우스 차이나</a>로 돌아와 그곳에서 은퇴했다.</p>
<h3><span class="mw-headline" id=".EA.B5.AD.EA.B0.80.EB.8C.80.ED.91.9C.ED.8C.80_.EA.B2.BD.EB.A0.A5">국가대표팀 경력</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%A7%88%ED%85%8C%EC%95%BC_%EC%BC%80%EC%A3%BC%EB%A7%8C&amp;action=edit&amp;section=3" title="부분 편집: 국가대표팀 경력">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>케주만은 <a href="/wiki/2000%EB%85%84_3%EC%9B%94" title="2000년 3월">2000년 3월</a> <a href="/wiki/%EC%A4%91%ED%99%94%EC%9D%B8%EB%AF%BC%EA%B3%B5%ED%99%94%EA%B5%AD_%EC%B6%95%EA%B5%AC_%EA%B5%AD%EA%B0%80%EB%8C%80%ED%91%9C%ED%8C%80" title="중화인민공화국 축구 국가대표팀">중국</a>전에서 국가대표팀 데뷔전을 가졌다.</p>
<p>케주만은 <a href="/wiki/2006%EB%85%84_FIFA_%EC%9B%94%EB%93%9C%EC%BB%B5" title="2006년 FIFA 월드컵">2006년 FIFA 월드컵</a>에서 <a href="/wiki/%EB%84%A4%EB%8D%9C%EB%9E%80%EB%93%9C_%EC%B6%95%EA%B5%AC_%EA%B5%AD%EA%B0%80%EB%8C%80%ED%91%9C%ED%8C%80" title="네덜란드 축구 국가대표팀">네덜란드</a>전에 출전하여 활약을 보였으나 <a href="/wiki/%EC%95%84%EB%A5%B4%ED%97%A8%ED%8B%B0%EB%82%98_%EC%B6%95%EA%B5%AC_%EA%B5%AD%EA%B0%80%EB%8C%80%ED%91%9C%ED%8C%80" title="아르헨티나 축구 국가대표팀">아르헨티나</a>전에서 <a href="/wiki/%EC%84%B8%EB%A5%B4%EB%B9%84%EC%95%84_%EB%AA%AC%ED%85%8C%EB%84%A4%EA%B7%B8%EB%A1%9C_%EC%B6%95%EA%B5%AC_%EA%B5%AD%EA%B0%80%EB%8C%80%ED%91%9C%ED%8C%80" title="세르비아 몬테네그로 축구 국가대표팀">세르비아 몬테네그로</a>가 일방적인 0-6 패배를 당할 때 퇴장당했다.<sup class="reference" id="cite_ref-1"><a href="#cite_note-1">[1]</a></sup> 그 뒤로 케주만은 국가대표팀에 선발되지 않았다.</p>
