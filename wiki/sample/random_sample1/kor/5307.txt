<p><b>마티아스 긴터</b> (Matthias Ginter, <a href="/wiki/1994%EB%85%84" title="1994년">1994년</a> <a href="/wiki/1%EC%9B%94_19%EC%9D%BC" title="1월 19일">1월 19일</a>, <a href="/wiki/%EB%B0%94%EB%8D%B4%EB%B7%94%EB%A5%B4%ED%85%9C%EB%B2%A0%EB%A5%B4%ED%81%AC_%EC%A3%BC" title="바덴뷔르템베르크 주">바덴뷔르템베르크 주</a> <a href="/wiki/%ED%94%84%EB%9D%BC%EC%9D%B4%EB%B6%80%EB%A5%B4%ED%81%AC" title="프라이부르크">프라이부르크 임 브라이스가우</a> ~) 는 <a href="/wiki/%EB%8F%85%EC%9D%BC" title="독일">독일</a>의 <a href="/wiki/%EC%B6%95%EA%B5%AC" title="축구">축구</a> 선수로, 현재 <a href="/wiki/%ED%91%B8%EC%8A%A4%EB%B0%9C-%EB%B6%84%EB%8D%B0%EC%8A%A4%EB%A6%AC%EA%B0%80" title="푸스발-분데스리가">분데스리가</a>의 <a href="/wiki/%EB%B3%B4%EB%A3%A8%EC%8B%9C%EC%95%84_%EB%8F%84%EB%A5%B4%ED%8A%B8%EB%AC%B8%ED%8A%B8" title="보루시아 도르트문트">도르트문트</a> 소속으로 주 포지션은 <a class="mw-redirect" href="/wiki/%ED%92%80%EB%B0%B1" title="풀백">풀백</a>이다. <a class="mw-redirect" href="/wiki/%EC%84%BC%ED%84%B0%EB%B0%B1" title="센터백">센터백</a>과 <a class="mw-redirect" href="/wiki/%EC%88%98%EB%B9%84%ED%98%95_%EB%AF%B8%EB%93%9C%ED%95%84%EB%8D%94" title="수비형 미드필더">수비형 미드필더</a>로도 뛸 수 있다.</p>

<h3><span class="mw-headline" id=".ED.81.B4.EB.9F.BD_.EA.B2.BD.EB.A0.A5">클럽 경력</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%A7%88%ED%8B%B0%EC%95%84%EC%8A%A4_%EA%B8%B4%ED%84%B0&amp;action=edit&amp;section=2" title="부분 편집: 클럽 경력">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>긴터는 마어히에서 경력을 시작하였으며, 2005-06 시즌에 <a href="/wiki/SC_%ED%94%84%EB%9D%BC%EC%9D%B4%EB%B6%80%EB%A5%B4%ED%81%AC" title="SC 프라이부르크">프라이부르크</a>의 유소년팀으로 옮겼다. 유소년 A팀 일원으로 그는 유소년 <a href="/wiki/DFB-%ED%8F%AC%EC%B9%BC" title="DFB-포칼">DFB-포칼</a> 우승을 거두었다.</p>
<p>2012년 1월, 긴터는 몇몇 선수가 겨울 이적 시장을 통해 나감에 따라 1군과 훈련하였다. 2012년 1월 21일, 긴터는 <a href="/wiki/FC_%EC%95%84%EC%9A%B0%ED%81%AC%EC%8A%A4%EB%B6%80%EB%A5%B4%ED%81%AC" title="FC 아우크스부르크">아우크스부르크</a>와의 강등권 대결 경기에서 70분에 <a class="new" href="/w/index.php?title=%EC%95%88%ED%86%A4_%ED%91%B8%ED%8A%B8%EC%8B%A4%EB%9D%BC&amp;action=edit&amp;redlink=1" title="안톤 푸트실라 (없는 문서)">안톤 푸트실라</a>와 교체 투입되며 프로 데뷔를 하였다. 경기의 88분에, <a class="new" href="/w/index.php?title=%EB%AF%B8%ED%95%98%EC%97%98_%EB%A3%B8&amp;action=edit&amp;redlink=1" title="미하엘 룸 (없는 문서)">미하엘 룸</a>의 프리킥을 팀의 1-0 결승골로 이끌었다.<sup class="reference" id="cite_ref-1"><a href="#cite_note-1">[1]</a></sup> 긴터의 골은 그의 18번째 생일 이틀 후에 터졌는데, 그에 따라 긴터는 SCF 클럽 역사상 최연소 분데스리가 득점자로 이름을 올렸다. 이 기록의 이전 보유자는 <a href="/wiki/%EB%8D%B0%EB%8B%88%EC%8A%A4_%EC%95%84%EC%98%A4%EA%B3%A0" title="데니스 아오고">데니스 아오고</a>였다.<sup class="reference" id="cite_ref-2"><a href="#cite_note-2">[2]</a></sup> 이틀 후인 1월 23일, 긴터는 프라이부르크의 1군에 합류하기 위한 계약서에 서명하였다. 그는 2014년 3월 5일, 칠레와의 경기에서 국가대표팀 데뷔전을 가졌고, 88분에 외질과 교체 투입된 그는 독일 국가대표팀 경기에 출전한 900번째 선수라는 기록을 세웠다.<sup class="reference" id="cite_ref-3"><a href="#cite_note-3">[3]</a></sup></p>
<h3><span class="mw-headline" id=".EA.B5.AD.EA.B0.80.EB.8C.80.ED.91.9C.ED.8C.80">국가대표팀</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%A7%88%ED%8B%B0%EC%95%84%EC%8A%A4_%EA%B8%B4%ED%84%B0&amp;action=edit&amp;section=3" title="부분 편집: 국가대표팀">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>긴터는 독일 청소년 국가대표팀 출신으로, 자국의 U-18, U-19 국가대표팀을 대표하였었고, 2013년부터 <a href="/wiki/%EB%8F%85%EC%9D%BC_U-21_%EC%B6%95%EA%B5%AC_%EA%B5%AD%EA%B0%80%EB%8C%80%ED%91%9C%ED%8C%80" title="독일 U-21 축구 국가대표팀">독일 U-21 국가대표팀</a>에 소속되어 있다. 2014년 3월 5일, 그는 1-0으로 이긴 <a href="/wiki/%EC%B9%A0%EB%A0%88_%EC%B6%95%EA%B5%AC_%EA%B5%AD%EA%B0%80%EB%8C%80%ED%91%9C%ED%8C%80" title="칠레 축구 국가대표팀">칠레</a>와의 <a class="mw-redirect" href="/wiki/%EC%B9%9C%EC%84%A0%EA%B2%BD%EA%B8%B0" title="친선경기">친선경기</a>에서 막판 교체 투입되며 <a href="/wiki/%EB%8F%85%EC%9D%BC_%EC%B6%95%EA%B5%AC_%EA%B5%AD%EA%B0%80%EB%8C%80%ED%91%9C%ED%8C%80" title="독일 축구 국가대표팀">독일 성인 국가대표팀</a> 데뷔전을 가졌다.<sup class="reference" id="cite_ref-4"><a href="#cite_note-4">[4]</a></sup> 그는 2014년 5월에 독일의 <a href="/wiki/2014%EB%85%84_FIFA_%EC%9B%94%EB%93%9C%EC%BB%B5" title="2014년 FIFA 월드컵">2014년 FIFA 월드컵</a> 30인 예비 엔트리에 포함되었다.<sup class="reference" id="cite_ref-World_Cup_2014_5-0"><a href="#cite_note-World_Cup_2014-5">[5]</a></sup> <a class="new" href="/w/index.php?title=Taruhan_Bola&amp;action=edit&amp;redlink=1" title="Taruhan Bola (없는 문서)">Taruhan Bola</a> | <a class="new" href="/w/index.php?title=Judi_Bola&amp;action=edit&amp;redlink=1" title="Judi Bola (없는 문서)">Judi Bola</a> | <a class="new" href="/w/index.php?title=Vipbet88&amp;action=edit&amp;redlink=1" title="Vipbet88 (없는 문서)">Vipbet88</a> | <a class="new" href="/w/index.php?title=Mansion88&amp;action=edit&amp;redlink=1" title="Mansion88 (없는 문서)">Mansion88</a></p>
