<p><b>마틴 라니히</b> (Martin Lanig, <a href="/wiki/1984%EB%85%84" title="1984년">1984년</a> <a href="/wiki/7%EC%9B%94_11%EC%9D%BC" title="7월 11일">7월 11일</a>, <a href="/wiki/%EB%B0%94%EB%8D%B4%EB%B7%94%EB%A5%B4%ED%85%9C%EB%B2%A0%EB%A5%B4%ED%81%AC_%EC%A3%BC" title="바덴뷔르템베르크 주">바덴뷔르템베르크 주</a> <a class="new" href="/w/index.php?title=%EB%B0%94%ED%8A%B8_%EB%A9%94%EB%A5%B4%EA%B2%90%ED%83%80%EC%9E%84&amp;action=edit&amp;redlink=1" title="바트 메르겐타임 (없는 문서)">바트 메르겐타임</a> ~) 은 <a href="/wiki/%EB%8F%85%EC%9D%BC" title="독일">독일</a>의 <a href="/wiki/%EC%B6%95%EA%B5%AC" title="축구">축구</a> 선수로, 현재 <a href="/wiki/%ED%91%B8%EC%8A%A4%EB%B0%9C-%EB%B6%84%EB%8D%B0%EC%8A%A4%EB%A6%AC%EA%B0%80" title="푸스발-분데스리가">분데스리가</a>의 <a href="/wiki/%EC%95%84%EC%9D%B8%ED%8A%B8%EB%9D%BC%ED%9D%90%ED%8A%B8_%ED%94%84%EB%9E%91%ED%81%AC%ED%91%B8%EB%A5%B4%ED%8A%B8" title="아인트라흐트 프랑크푸르트">아인트라흐트 프랑크푸르트</a>에서 <a href="/wiki/%EB%AF%B8%EB%93%9C%ED%95%84%EB%8D%94" title="미드필더">미드필더</a>로 활약하고 있다.<sup class="reference" id="cite_ref-profile1_1-0"><a href="#cite_note-profile1-1">[1]</a></sup></p>
<p><a href="/wiki/2008%EB%85%84" title="2008년">2008년</a> <a href="/wiki/7%EC%9B%94" title="7월">7월</a>, 그는 <a href="/wiki/SpVgg_%EA%B7%B8%EB%A1%9C%EC%9D%B4%ED%84%B0_%ED%93%8C%EB%A5%B4%ED%8A%B8" title="SpVgg 그로이터 퓌르트">SpVgg 그로이터 퓌르트</a>에서 <a href="/wiki/VfB_%EC%8A%88%ED%88%AC%ED%8A%B8%EA%B0%80%EB%A5%B4%ED%8A%B8" title="VfB 슈투트가르트">VfB 슈투트가르트</a>로 이적하였다.<sup class="reference" id="cite_ref-profile2_2-0"><a href="#cite_note-profile2-2">[2]</a></sup> 라니히는 <a href="/wiki/2008%EB%85%84" title="2008년">2008년</a> <a href="/wiki/10%EC%9B%94_5%EC%9D%BC" title="10월 5일">10월 5일</a>, <a href="/wiki/SV_%EB%B2%A0%EB%A5%B4%EB%8D%94_%EB%B8%8C%EB%A0%88%EB%A9%98" title="SV 베르더 브레멘">SV 베르더 브레멘</a>전에서 VfB 소속 첫 골을 기록하였다.</p>
<p><a href="/wiki/2010%EB%85%84" title="2010년">2010년</a> <a href="/wiki/7%EC%9B%94" title="7월">7월</a>, 그는 <a href="/wiki/1._FC_%EC%BE%B0%EB%A5%B8" title="1. FC 쾰른">1. FC 쾰른</a>으로 이적하였다.<sup class="reference" id="cite_ref-profile3_3-0"><a href="#cite_note-profile3-3">[3]</a></sup></p>
