<p><b>마하출라롱컨라자위달라야 대학교</b>(<span style="font-size: smaller;"><a href="/wiki/%ED%83%9C%EA%B5%AD%EC%96%B4" title="태국어">태국어</a>:</span> <span lang="th" xml:lang="th">มหาวิทยาลัยมหาจุฬาลงกรณราชวิทยาลัย</span> <small>마하출라롱컨라자위달라야<sup>[<a class="mw-redirect" href="/wiki/%EC%9C%84%ED%82%A4%EB%B0%B1%EA%B3%BC:%ED%83%80%EC%9D%B4%EC%96%B4%EC%9D%98_%ED%95%9C%EA%B8%80_%ED%91%9C%EA%B8%B0" title="위키백과:타이어의 한글 표기">*</a>]</sup></small>)는 <a href="/wiki/%ED%83%80%EC%9D%B4" title="타이">타이</a>의 <a href="/wiki/%EA%B5%AD%EB%A6%BD_%EB%8C%80%ED%95%99" title="국립 대학">국립</a> <a class="new" href="/w/index.php?title=%EB%B6%88%EA%B5%90_%EB%8C%80%ED%95%99&amp;action=edit&amp;redlink=1" title="불교 대학 (없는 문서)">불교 대학</a>이다. 주 캠퍼스는 원래 <a href="/wiki/%EB%B0%A9%EC%BD%95" title="방콕">방콕</a>의 <a class="new" href="/w/index.php?title=%EC%99%93_%EB%A7%88%ED%95%98%ED%83%93&amp;action=edit&amp;redlink=1" title="왓 마하탓 (없는 문서)">왓 마하탓</a>에 위치해 있으며, 타이 내 여러 지역에 캠퍼스가 있다.</p>
<p>이 대학교는 <a class="mw-redirect" href="/wiki/%EB%9D%BC%EB%A7%88_5%EC%84%B8" title="라마 5세">라마 5세</a> 쭐랄롱꼰 대왕에 의해 <a href="/wiki/1887%EB%85%84" title="1887년">1887년</a> <a href="/wiki/%EB%B6%88%EA%B5%90" title="불교">불교</a> <a href="/wiki/%EC%8A%B9%EB%A0%A4" title="승려">승려</a>, <a class="new" href="/w/index.php?title=%EC%82%AC%EB%AF%B8%EC%8A%B9&amp;action=edit&amp;redlink=1" title="사미승 (없는 문서)">사미승</a>, 평신도 등을 대상으로 불교 교육에 중점을 둔 고등 교육을 제공하기 위해 개교하였다. 강좌는 <a href="/wiki/1889%EB%85%84" title="1889년">1889년</a>부터 개설되었으며, 현재와 같은 '마하쭐랄롱꼰랏위타얄라이 대학교'라는 이름을 획득한 것은 <a href="/wiki/1896%EB%85%84" title="1896년">1896년</a>이다.</p>
<p>또 다른 타이의 불교대학인 <a href="/wiki/%EB%A7%88%ED%95%A8%EA%BF%8B%EB%9E%8F%EC%9C%84%ED%83%80%EC%96%84%EB%9D%BC%EC%9D%B4_%EB%8C%80%ED%95%99%EA%B5%90" title="마함꿋랏위타얄라이 대학교">마함꿋랏위타얄라이 대학교</a>와 함께 이 대학교는 <a href="/wiki/1997%EB%85%84" title="1997년">1997년</a>부터 국립대학이 되었다.</p>

<h2><span class="mw-headline" id=".EC.86.8C.EC.86.8D_.EA.B8.B0.EA.B4.80">소속 기관</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%A7%88%ED%95%98%EC%AD%90%EB%9E%84%EB%A1%B1%EA%BC%B0%EB%9E%8F%EC%9C%84%ED%83%80%EC%96%84%EB%9D%BC%EC%9D%B4_%EB%8C%80%ED%95%99%EA%B5%90&amp;action=edit&amp;section=1" title="부분 편집: 소속 기관">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>마하쭐랄롱꼰랏위타얄라이 대학교는 다음과 같은 여러 학부로 구성되어 있다.</p>
<ul>
<li>불교학부</li>
<li>교육학부</li>
<li>인문학부</li>
<li>사회과학부</li>
</ul>
<p>이 외에도 이 대학교에는 <a href="/wiki/%EB%8C%80%ED%95%99%EC%9B%90" title="대학원">대학원</a> 과정이 개설되어 있으며, 국제 과정도 있다. 현대에는 교육학부 소속으로 <a href="/wiki/%EC%88%98%ED%95%99" title="수학">수학</a> 과정, 인문학부 소속으로 <a href="/wiki/%EC%8B%AC%EB%A6%AC%ED%95%99" title="심리학">심리학</a> 과정, 사회과학부 소속으로 <a href="/wiki/%EA%B2%BD%EC%A0%9C%ED%95%99" title="경제학">경제학</a> 과정 등을 개설하여, 차츰 종합대학화가 진행되는 추세이다.<sup class="reference" id="cite_ref-1"><a href="#cite_note-1">[1]</a></sup><sup class="reference" id="cite_ref-2"><a href="#cite_note-2">[2]</a></sup><sup class="reference" id="cite_ref-3"><a href="#cite_note-3">[3]</a></sup></p>
<h2><span class="mw-headline" id=".EC.BA.A0.ED.8D.BC.EC.8A.A4">캠퍼스</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%A7%88%ED%95%98%EC%AD%90%EB%9E%84%EB%A1%B1%EA%BC%B0%EB%9E%8F%EC%9C%84%ED%83%80%EC%96%84%EB%9D%BC%EC%9D%B4_%EB%8C%80%ED%95%99%EA%B5%90&amp;action=edit&amp;section=2" title="부분 편집: 캠퍼스">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
