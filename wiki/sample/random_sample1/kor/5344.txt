<p><b>만수르 빈 자이드 빈 술탄 알나하얀</b>(<span style="font-size: smaller;"><a href="/wiki/%EC%95%84%EB%9E%8D%EC%96%B4" title="아랍어">아랍어</a>:</span> <span lang="ar" xml:lang="ar">منصور بن زايد آل نهيان</span>, <span style="font-size: smaller;"><a href="/wiki/%EC%98%81%EC%96%B4" title="영어">영어</a>:</span> <span lang="en" xml:lang="en">Mansour bin Zayed bin Sultan Al Nahyan</span>, <a href="/wiki/1970%EB%85%84" title="1970년">1970년</a> <a href="/wiki/11%EC%9B%94_20%EC%9D%BC" title="11월 20일">11월 20일</a> ~ ) 혹은 <b>셰이흐 만수르</b>(Sheikh Mansour)는 <a href="/wiki/%EC%95%84%EB%9E%8D%EC%97%90%EB%AF%B8%EB%A6%AC%ED%8A%B8" title="아랍에미리트">아랍에미리트</a>의 정치인이자 <a href="/wiki/%EC%95%84%EB%9E%8D%EC%97%90%EB%AF%B8%EB%A6%AC%ED%8A%B8%EC%9D%98_%EB%8C%80%ED%86%B5%EB%A0%B9" title="아랍에미리트의 대통령">대통령</a>인 <a class="mw-redirect" href="/wiki/%ED%95%A0%EB%A6%AC%ED%8C%8C_%EB%B9%88_%EC%9E%90%EC%9D%B4%EB%93%9C_%EC%95%8C%EB%82%98%ED%95%98%EC%96%80" title="할리파 빈 자이드 알나하얀">할리파 빈 자이드 알나하얀</a>의 동생으로, <a href="/wiki/%ED%94%84%EB%A6%AC%EB%AF%B8%EC%96%B4%EB%A6%AC%EA%B7%B8" title="프리미어리그">프리미어리그</a>의 <a href="/wiki/%EB%A7%A8%EC%B2%B4%EC%8A%A4%ED%84%B0_%EC%8B%9C%ED%8B%B0_FC" title="맨체스터 시티 FC">맨체스터 시티 FC</a>의 구단주이기도 하며 재산은 가족 재산들까지 포함해 약25조원(약 150억 파운드)로 추정된다. 만수르는 현재 알나하얀 가문의 2인자이다.</p>

<h2><span class="mw-headline" id=".EC.9D.BC.EC.83.9D.EA.B3.BC_.EA.B2.BD.EB.A0.A5">일생과 경력</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%A7%8C%EC%88%98%EB%A5%B4_%EB%B9%88_%EC%9E%90%EC%9D%B4%EB%93%9C_%EC%95%8C%EB%82%98%ED%95%98%EC%96%80&amp;action=edit&amp;section=1" title="부분 편집: 일생과 경력">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p><a href="/wiki/%EC%95%84%EB%9E%8D%EC%97%90%EB%AF%B8%EB%A6%AC%ED%8A%B8" title="아랍에미리트">아랍에미리트</a> (독립 이전에는 톨샤르 오만) <a href="/wiki/%EC%95%84%EB%B6%80%EB%8B%A4%EB%B9%84" title="아부다비">아부다비</a> 출생으로 1989년 미국 산타바바라 커뮤니티 컬리지 영문학과에 입학했고 이후 <a href="/wiki/1993%EB%85%84" title="1993년">1993년</a> 아랍에미리트연합대학교에서 국제학 학사 학위를 취득하였다. <a href="/wiki/1997%EB%85%84" title="1997년">1997년</a> 아랍에미리트의 초대 대통령이던 아버지 자이드 빈 술탄 알나하얀에 의해 사무실을 담당하였고, 형 할리파 빈 자이드 알나하얀이 대통령이 되면서 본격적으로 정치에 참여하였다.</p>
<p><a href="/wiki/2000%EB%85%84" title="2000년">2000년</a>부터 문서 및 연구 국립센터 위원장, <a href="/wiki/2005%EB%85%84" title="2005년">2005년</a> 아부다비 교육위원회(ADEC), 에미리트 재단, 아부다비 식품, 아부다비 발전을 위한 기금의 회장을 지냈고, <a href="/wiki/2005%EB%85%84" title="2005년">2003년</a> ~ <a href="/wiki/2006%EB%85%84" title="2006년">20012년</a>에는 아부다비 사법학과 반장으로 재직하였다.</p>
<p><a href="/wiki/2005%EB%85%84" title="2005년">2012년</a> 석유 최고위원회의 회원으로 임명되었고 국제 석유 투자 회사위원회와 아부다비 투자위원회(ADIA)의 위원장을 지냈고, <a href="/wiki/2007%EB%85%84" title="2007년">2001년</a> 에미리트 투자 기관의 자금 담당관으로 임명되었다.</p>
<p><a href="/wiki/2007%EB%85%84" title="2007년">2007년</a> 칼리프 빈 자이드 자선 재단의 회장으로 임명되었고, <a href="/wiki/2006%EB%85%84" title="2006년">2006년</a>까지는 걸프 은행에서 아랍에미리트의 학생들을 위한 유학 장학금 제도를 설립하였다.</p>
<p><a href="/wiki/2008%EB%85%84_9%EC%9B%94" title="2008년 9월">2008년 9월</a> <a href="/wiki/%EC%88%A0%EB%9D%BC%EC%9D%B4%EB%A7%8C_%EC%95%8C%ED%8C%8C%ED%9E%98" title="술라이만 알파힘">술라이만 알파힘</a>과 함께 잉글랜드 프리미어리그의 <a href="/wiki/%EB%A7%A8%EC%B2%B4%EC%8A%A4%ED%84%B0_%EC%8B%9C%ED%8B%B0_FC" title="맨체스터 시티 FC">맨체스터 시티 FC</a>를 <a href="/wiki/%ED%83%80%EC%9D%B4" title="타이">타이</a>의 전 총리 <a href="/wiki/%ED%83%81%EC%8B%A0_%EC%B9%9C%EB%82%98%EC%99%93" title="탁신 친나왓">탁신 친나왓</a>으로부터 인수해 구단주가 되어 <a href="/wiki/%EB%A7%A8%EC%B2%B4%EC%8A%A4%ED%84%B0_%EC%8B%9C%ED%8B%B0_FC" title="맨체스터 시티 FC">맨체스터 시티 FC</a>를 통하여 2011-2012, 2013-2014년 프리미어리그를 우승하였다.</p>
<h2><span class="mw-headline" id=".EB.B3.B8.EB.AA.85.EC.9D.98_.EC.9D.98.EB.AF.B8">본명의 의미</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%A7%8C%EC%88%98%EB%A5%B4_%EB%B9%88_%EC%9E%90%EC%9D%B4%EB%93%9C_%EC%95%8C%EB%82%98%ED%95%98%EC%96%80&amp;action=edit&amp;section=2" title="부분 편집: 본명의 의미">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p><b>만수르 빈 자이드 빈 술탄 알나하얀</b>이라는 이름 자체가 그가 왕족 가문의 고귀한 혈통을 지니고 태어났음을 의미한다.<sup class="reference" id="cite_ref-1"><a href="#cite_note-1">[1]</a></sup> 일단 알나하얀 가문은 1700년대부터 아부다비를 통치해왔다. 게다가 그의 이름에 들어가있는 <a href="/wiki/%EC%88%A0%ED%83%84" title="술탄">술탄</a>이라는 이름의 의미 자체가 중동에서는 왕 또는 국가지도자를 의미하며 독일어문화권 국가의 <a href="/wiki/%EC%B9%B4%EC%9D%B4%EC%A0%80" title="카이저">카이저</a>, 러시아의 <a href="/wiki/%EC%B0%A8%EB%A5%B4" title="차르">차르</a>와 같은 의미의 호칭이다. 또한 만수르라는 이름의 어원은 아랍어 단어 '나스르(nasr)'에서 온 것으로 '승리를 거둔'(victorious)이란 뜻을 갖고 있다. 이는 영어 문화권의 여성 이름인 '빅토리아(victoria)'나 스페인어 문화권의 남성 이름인 '헤라르도(gerardo)'와 뜻이 동일하다.</p>
<h2><span class="mw-headline" id=".EC.82.AC.EC.97.85_.EA.B2.BD.EB.A0.A5">사업 경력</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%A7%8C%EC%88%98%EB%A5%B4_%EB%B9%88_%EC%9E%90%EC%9D%B4%EB%93%9C_%EC%95%8C%EB%82%98%ED%95%98%EC%96%80&amp;action=edit&amp;section=3" title="부분 편집: 사업 경력">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>국제석유투자회사(International Petroleum Investment Company)의 소유주이다. 이 회사는 아바르 인베스트먼트(Aabar Investments)의 71% 지분을 소유하고 있으며 만수르의 투자 대상 중의 하나이다.</p>
