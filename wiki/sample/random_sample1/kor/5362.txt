<p><b>말론 브란도 주니어</b>(<span style="font-size: smaller;"><a href="/wiki/%EC%98%81%EC%96%B4" title="영어">영어</a>:</span> <span lang="en" xml:lang="en">Marlon Brando, Jr.</span> <small>말런 브랜도<sup>[<a href="/wiki/%EC%9C%84%ED%82%A4%EB%B0%B1%EA%B3%BC:%EC%98%81%EC%96%B4%EC%9D%98_%ED%95%9C%EA%B8%80_%ED%91%9C%EA%B8%B0" title="위키백과:영어의 한글 표기">*</a>]</sup></small>, <a href="/wiki/1924%EB%85%84" title="1924년">1924년</a> <a href="/wiki/4%EC%9B%94_3%EC%9D%BC" title="4월 3일">4월 3일</a> ~ <a href="/wiki/2004%EB%85%84" title="2004년">2004년</a> <a href="/wiki/7%EC%9B%94_1%EC%9D%BC" title="7월 1일">7월 1일</a>)는 반세기 동안 활동한 <a href="/wiki/%EB%AF%B8%EA%B5%AD" title="미국">미국</a>의 <a class="mw-redirect" href="/wiki/%EC%98%81%ED%99%94_%EB%B0%B0%EC%9A%B0" title="영화 배우">영화 배우</a>이다. 영화사에서 가장 중요한 배우 중의 하나이며, 1950년대에 강한 정신적 기술을 강조한 조직적 스타일의 연기로 인기를 끌면서 《전선》(On the Waterfront, 1953)에서 테리 맬로이 역할과 《<a href="/wiki/%EB%8C%80%EB%B6%80_(%EC%98%81%ED%99%94)" title="대부 (영화)">대부</a>》(The Godfather, 1971)에서 <a href="/wiki/%EB%B9%84%ED%86%A0_%EC%BD%94%EB%A5%BC%EB%A0%88%EC%98%A4%EB%84%A4" title="비토 코를레오네">비토 코를레오네</a> 역할로 두번의 <a href="/wiki/%EC%95%84%EC%B9%B4%EB%8D%B0%EB%AF%B8%EC%83%81" title="아카데미상">아카데미</a> 남우주연상을 수상하였다.</p>

<h2><span class="mw-headline" id=".EC.83.9D.EC.95.A0">생애</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%A7%90%EB%A1%A0_%EB%B8%8C%EB%9E%80%EB%8F%84&amp;action=edit&amp;section=1" title="부분 편집: 생애">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>그의 원래 성은 브란다우(Brandau)로 <a href="/wiki/%EB%84%A4%EB%B8%8C%EB%9E%98%EC%8A%A4%EC%B9%B4_%EC%A3%BC" title="네브래스카 주">네브래스카 주</a> <a href="/wiki/%EC%98%A4%EB%A7%88%ED%95%98" title="오마하">오마하</a>에서 출생하여 <a href="/wiki/1944%EB%85%84" title="1944년">1944년</a> <a href="/wiki/%EB%B8%8C%EB%A1%9C%EB%93%9C%EC%9B%A8%EC%9D%B4_%EC%97%B0%EA%B7%B9" title="브로드웨이 연극">브로드웨이</a>에 데뷔하였다. <a href="/wiki/1947%EB%85%84" title="1947년">1947년</a> 《<a href="/wiki/%EC%9A%95%EB%A7%9D%EC%9D%B4%EB%9D%BC%EB%8A%94_%EC%9D%B4%EB%A6%84%EC%9D%98_%EC%A0%84%EC%B0%A8" title="욕망이라는 이름의 전차">욕망이라는 이름의 전차</a>》(A Streetcar Named Desire)에서 스탠리 코왈스키 역을 맡으면서 명성을 얻었으며, <a href="/wiki/1951%EB%85%84" title="1951년">1951년</a> 동명의 영화판(<a href="/wiki/%EC%97%98%EB%A6%AC%EC%95%84_%EC%B9%B4%EC%9E%94" title="엘리아 카잔">엘리아 카잔</a> <a href="/wiki/%EC%98%81%ED%99%94_%EA%B0%90%EB%8F%85" title="영화 감독">감독</a>)에서도 같은 역을 맡았다.</p>
<p>영화 배우로서 첫 데뷔작은 《사나이》(The Man, 1950)이다. 그의 초기 출연작으로는 《사나운 인간》(The Wild One, 1953), 《8월 달의 찻집》(The Teahouse of the August Moon, 1956)과 《어린 사자들》(1958)을 포함한다.</p>
<p><a href="/wiki/%ED%94%84%EB%9E%9C%EC%8B%9C%EC%8A%A4_%ED%8F%AC%EB%93%9C_%EC%BD%94%ED%8F%B4%EB%9D%BC" title="프랜시스 포드 코폴라">프랜시스 포드 코폴라</a> 감독의 작품 《<a href="/wiki/%EB%8C%80%EB%B6%80_(%EC%98%81%ED%99%94)" title="대부 (영화)">대부</a>》의 <a href="/wiki/%EB%B9%84%ED%86%A0_%EC%BD%94%EB%A5%BC%EB%A0%88%EC%98%A4%EB%84%A4" title="비토 코를레오네">비토 코를레오네</a>, 《<a href="/wiki/%EC%A7%80%EC%98%A5%EC%9D%98_%EB%AC%B5%EC%8B%9C%EB%A1%9D" title="지옥의 묵시록">지옥의 묵시록</a>》(Apocalypse Now, 1979)의 월터 E. 커츠 대령 역으로 유명하며, <a href="/wiki/1978%EB%85%84" title="1978년">1978년</a> <a href="/wiki/%EB%A6%AC%EC%B2%98%EB%93%9C_%EB%8F%84%EB%84%88" title="리처드 도너">리처드 도너</a> 감독의 《<a href="/wiki/%EC%8A%88%ED%8D%BC%EB%A7%A8" title="슈퍼맨">슈퍼맨</a>》(Superman: The Movie)에서 조르 엘 역을 맡아 세계의 주목을 받게 되었다.</p>
<p>1960년대와 1970년대에 브란도는 <a href="/wiki/%EC%95%84%ED%94%84%EB%A6%AC%EC%B9%B4%EA%B3%84_%EB%AF%B8%EA%B5%AD%EC%9D%B8" title="아프리카계 미국인">흑인</a>과 <a class="mw-redirect" href="/wiki/%EC%95%84%EB%A9%94%EB%A6%AC%EC%B9%B4_%EC%9D%B8%EB%94%94%EC%96%B8" title="아메리카 인디언">인디언</a>들을 위한 양쪽의 인권 운동에 후원한 자유주의 정치 활동을 펼쳐오기도 하였다. 뿐만 아니라 자신의 <a href="/wiki/%EC%96%91%EC%84%B1%EC%95%A0" title="양성애">양성애</a> 성향도 숨기지 않았으며 <a href="/wiki/%EC%84%B1%EC%86%8C%EC%88%98%EC%9E%90" title="성소수자">성소수자</a> 인권운동을 옹호하는 발언을 하기도 하였다.</p>
<p>그의 다른 출연작으로는 《파리의 마지막 탱고》(Last Tango in Paris, 1973), 《돈 후안 데마르코》(Don Juan DeMarco, 1995), 《모로 박사의 섬》(The Island of Dr. Moreau, 1996), 《더 스코어》(The Score, 2001) 등이 있다.</p>
<p>그의 저서로는 자서전〈나의 어머니가 나에게 가르쳐 준 노래들(Songs My Mother Taught Me)〉(1994)이 있다. <a href="/wiki/2004%EB%85%84" title="2004년">2004년</a> <a href="/wiki/7%EC%9B%94_1%EC%9D%BC" title="7월 1일">7월 1일</a> <a href="/wiki/%EB%A1%9C%EC%8A%A4%EC%95%A4%EC%A0%A4%EB%A0%88%EC%8A%A4" title="로스앤젤레스">로스앤젤레스</a>에서 사망하였다.</p>
