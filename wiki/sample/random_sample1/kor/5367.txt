<p><b>말린체</b>(Malinche, <a href="/wiki/1496%EB%85%84" title="1496년">1496년</a> - <a href="/wiki/1529%EB%85%84" title="1529년">1529년</a>) 또는 <b>도냐 마리나</b>(doña Marina)는 <a class="mw-redirect" href="/wiki/%EC%95%84%EC%A6%88%ED%85%8D" title="아즈텍">아즈텍</a> 출신의 인디오였다. <b>라 말린체</b>(La Malinche), <b>말린친</b>(Malintzin) 등으로도 부른다. 노예생활을 전전하다 <a href="/wiki/%EC%97%90%EB%A5%B4%EB%82%9C_%EC%BD%94%EB%A5%B4%ED%85%8C%EC%8A%A4" title="에르난 코르테스">에르난 코르테스</a>의 통역이 되었으며, <a class="mw-redirect" href="/wiki/%EC%97%90%EC%8A%A4%ED%8C%8C%EB%83%90" title="에스파냐">에스파냐</a> 침략군의 통역 겸 길안내인을 자처하였으므로 <a href="/wiki/%EB%A9%95%EC%8B%9C%EC%BD%94" title="멕시코">멕시코</a> 및 <a href="/wiki/%EB%82%A8%EC%95%84%EB%A9%94%EB%A6%AC%EC%B9%B4" title="남아메리카">남아메리카</a>에서는 배신자, 반역자의 대명사로 알려져 있다.</p>
<h2><span class="mw-headline" id=".EC.83.9D.EC.95.A0">생애</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%A7%90%EB%A6%B0%EC%B2%B4&amp;action=edit&amp;section=1" title="부분 편집: 생애">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>말린체의 원래 이름은 말리날리 테네팔(Malinali Tenepal)이다. 멕시코만의 소읍에서 <a class="new" href="/w/index.php?title=%EB%82%98%EC%9A%B0%EC%95%84%EC%A1%B1&amp;action=edit&amp;redlink=1" title="나우아족 (없는 문서)">나우아족</a> 귀족 가문의 딸로 태어나 귀족으로서 교육을 받았다. 당시 <a class="mw-redirect" href="/wiki/%EC%95%84%EC%A6%88%ED%85%8C%EC%B9%B4" title="아즈테카">아즈테카</a> 사회에서는 귀족은 남녀 모두 기숙학교에서 다양한 교양을 쌓았다. 아버지가 죽자 어머니가 재혼하고 새로 아들을 낳았다. 아즈테카 사회에서는 여성도 재산을 상속하고 상속받을 권리가 있었기 때문에 어머니는 자신의 재산을 새로 낳은 아들에게만 물려주기 위해 딸 말린체를 <a href="/wiki/%EC%9C%A0%EC%B9%B4%ED%83%84_%EB%B0%98%EB%8F%84" title="유카탄 반도">유카탄 반도</a>의 <a class="mw-redirect" href="/wiki/%EB%A7%88%EC%95%BC%EC%A1%B1" title="마야족">마야족</a> 귀족에게 노예로 팔았다. 코르테스가 유카탄 지역에에 도착하자 이미 한차례 스페인 인을 만난 적이 있던 마야족 사람들은 그들에게 우호의 표시로 선물과 여자 노예 20명을 바쳤다. 말린체는 그 20명 중 하나였다. 팔려온 다음날 기독교식 세례를 받고 그날 이후 말린체는 도냐 마리나라고 불렸다.(출처: Revista Iberoamericana 21.2 (2010): 57-82)</p>
<p>나우아족 출신이며 마야족의 노예로 살면서 마야어를 배운 말린체는 <a href="/wiki/%EC%97%90%EB%A5%B4%EB%82%9C_%EC%BD%94%EB%A5%B4%ED%85%8C%EC%8A%A4" title="에르난 코르테스">에르난 코르테스</a>의 통역으로 코르테스의 신뢰를 얻게 되었으며, <a href="/wiki/%EC%97%90%EB%A5%B4%EB%82%9C_%EC%BD%94%EB%A5%B4%ED%85%8C%EC%8A%A4" title="에르난 코르테스">에르난 코르테스</a>의 <a class="mw-redirect" href="/wiki/%EC%95%84%EC%A6%88%ED%85%8D" title="아즈텍">아즈텍</a> 정복에 협력하였다. 이때문에 <a href="/wiki/%EB%82%A8%EC%95%84%EB%A9%94%EB%A6%AC%EC%B9%B4" title="남아메리카">남아메리카</a>에서는 현재도 말린체는 배신자의 대명사로 통하게 되었다.</p>
<p><a href="/wiki/%EC%97%90%EB%A5%B4%EB%82%9C_%EC%BD%94%EB%A5%B4%ED%85%8C%EC%8A%A4" title="에르난 코르테스">에르난 코르테스</a>와의 사이에서 아들 <a class="new" href="/w/index.php?title=%EB%8F%88_%EB%A7%88%EB%A5%B4%ED%8B%B4_%EC%BD%94%EB%A5%B4%ED%85%8C%EC%8A%A4&amp;action=edit&amp;redlink=1" title="돈 마르틴 코르테스 (없는 문서)">돈 마르틴 코르테스</a>(Martin Cortes)를 두었다. 코르테스와 코르테스 본처의 주선으로 <a href="/wiki/1526%EB%85%84" title="1526년">1526년</a>과 <a href="/wiki/1527%EB%85%84" title="1527년">1527년</a> 무렵에 <a class="new" href="/w/index.php?title=%ED%9B%84%EC%95%88_%ED%95%98%EB%9D%BC%EB%AF%B8%EC%9A%94&amp;action=edit&amp;redlink=1" title="후안 하라미요 (없는 문서)">후안 하라미요</a>(Juan Jaramillo)라는 콩키스타도르와 결혼하였다. 후안 하라미요와의 사이에서는 마리아 하라미요(Maria Jaramillo)라는 딸이 태어났다.</p>
