<p><b>망우동</b>(忘憂洞)은 <a href="/wiki/%EC%84%9C%EC%9A%B8%ED%8A%B9%EB%B3%84%EC%8B%9C" title="서울특별시">서울특별시</a> <a href="/wiki/%EC%A4%91%EB%9E%91%EA%B5%AC" title="중랑구">중랑구</a>의 법정동이다. 망우동은 <a href="/wiki/%EA%B2%BD%EA%B8%B0%EB%8F%84" title="경기도">경기도</a>, <a href="/wiki/%EA%B0%95%EC%9B%90%EB%8F%84" title="강원도">강원도</a>로부터 진입하는 서울의 동쪽 관문으로, 유통시설, 금융기관, 업무용 빌딩이 밀집하여 경제활동이 활발한 지역이다.<sup class="reference" id="cite_ref-1"><a href="#cite_note-1">[1]</a></sup></p>

<h2><span class="mw-headline" id=".EA.B0.9C.EC.9A.94">개요</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%A7%9D%EC%9A%B0%EB%8F%99&amp;action=edit&amp;section=1" title="부분 편집: 개요">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p><a href="/wiki/1913%EB%85%84" title="1913년">1913년</a> <a href="/wiki/12%EC%9B%94_29%EC%9D%BC" title="12월 29일">12월 29일</a> 중랑 지역은 <a href="/wiki/%EA%B2%BD%EA%B8%B0%EB%8F%84" title="경기도">경기도</a> <a href="/wiki/%EC%96%91%EC%A3%BC%EA%B5%B0" title="양주군">양주군</a> 구리면(현재의 상봉·중화·묵·망우·신내동)과 <a href="/wiki/%EA%B3%A0%EC%96%91%EA%B5%B0" title="고양군">고양군</a> 뚝도면(현재의 면목·중곡동)의 일부가 되었다.<sup class="reference" id="cite_ref-2"><a href="#cite_note-2">[2]</a></sup> <a href="/wiki/1949%EB%85%84" title="1949년">1949년</a> <a href="/wiki/8%EC%9B%94_15%EC%9D%BC" title="8월 15일">8월 15일</a>에 서울특별시 구역 확장으로 <a href="/wiki/%EA%B3%A0%EC%96%91%EA%B5%B0" title="고양군">고양군</a> 뚝도면 면목리가 서울시로 편입되었고, <a href="/wiki/1963%EB%85%84" title="1963년">1963년</a> <a href="/wiki/1%EC%9B%94_1%EC%9D%BC" title="1월 1일">1월 1일</a> 다시 <a href="/wiki/%EC%96%91%EC%A3%BC%EA%B5%B0" title="양주군">양주군</a> 구리면의 상봉·중화·묵·망우·신내리가 편입되어 현재의 상봉·중화·묵·망우·신내동이 되었다. <a href="/wiki/1968%EB%85%84" title="1968년">1968년</a> <a href="/wiki/1%EC%9B%94_1%EC%9D%BC" title="1월 1일">1월 1일</a>에 <a href="/wiki/%EB%8F%99%EB%8C%80%EB%AC%B8%EA%B5%AC" title="동대문구">동대문구</a>에 편입되어 행정동간 조정으로 면목동을 면목1동·2동으로 분동하였으며, <a href="/wiki/1973%EB%85%84" title="1973년">1973년</a> 7월에는 <a href="/wiki/%EA%B4%80%EC%95%85%EA%B5%AC" title="관악구">관악구</a>, <a href="/wiki/%EB%8F%84%EB%B4%89%EA%B5%AC" title="도봉구">도봉구</a>의 2개 구가 증설됨과 동시에 구조정으로 <a href="/wiki/%EB%A9%B4%EB%AA%A9%EB%8F%99" title="면목동">면목동</a>의 일부가 <a href="/wiki/%EC%84%B1%EB%8F%99%EA%B5%AC" title="성동구">성동구</a>로 편입되면서 면목1동을 나누어 면목3·4동을 증설하였으며, <a href="/wiki/1977%EB%85%84" title="1977년">1977년</a> <a href="/wiki/9%EC%9B%94_1%EC%9D%BC" title="9월 1일">9월 1일</a>에는 면목1동 일부를 면목6동으로 분동하였다. 그 후 <a href="/wiki/1988%EB%85%84" title="1988년">1988년</a> <a href="/wiki/1%EC%9B%94_1%EC%9D%BC" title="1월 1일">1월 1일</a> 인구가 많은 구를 분구하여 <a href="/wiki/%EC%A4%91%EB%9E%91%EA%B5%AC" title="중랑구">중랑구</a>·<a href="/wiki/%EB%85%B8%EC%9B%90%EA%B5%AC" title="노원구">노원구</a>·<a href="/wiki/%EC%84%9C%EC%B4%88%EA%B5%AC" title="서초구">서초구</a>·<a href="/wiki/%EC%96%91%EC%B2%9C%EA%B5%AC" title="양천구">양천구</a>·<a href="/wiki/%EC%86%A1%ED%8C%8C%EA%B5%AC"
