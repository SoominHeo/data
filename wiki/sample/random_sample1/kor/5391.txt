<h1 class="firstHeading" id="firstHeading" lang="ko">매사추세츠 만 식민지</h1>

<p><b>매사추세츠만 식민지</b>(Massachusetts Bay Colony)는 17세기에 <a class="mw-redirect" href="/wiki/%EB%B6%81%EB%AF%B8" title="북미">북미</a> 동해안에 <a href="/wiki/%EC%98%81%EA%B5%AD" title="영국">영국</a>이 세운 식민지이다. <a href="/wiki/%EB%89%B4%EC%9E%89%EA%B8%80%EB%9E%9C%EB%93%9C" title="뉴잉글랜드">뉴잉글랜드</a>에 속하며, 오늘날의 <a class="mw-disambig" href="/wiki/%EC%84%B8%EC%9D%BC%EB%9F%BC" title="세일럼">세일럼</a>과 <a href="/wiki/%EB%B3%B4%EC%8A%A4%ED%84%B4" title="보스턴">보스턴</a>을 중심으로 하고 있었다. 그 지역은 현재 미국 50개 주 중 하나인 <a href="/wiki/%EB%A7%A4%EC%82%AC%EC%B6%94%EC%84%B8%EC%B8%A0_%EC%A3%BC" title="매사추세츠 주">매사추세츠 주</a>가 되었다.</p>
<h2><span class="mw-headline" id=".EC.B4.88.EA.B8.B0_.EA.B0.9C.EC.B2.99">초기 개척</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%A7%A4%EC%82%AC%EC%B6%94%EC%84%B8%EC%B8%A0_%EB%A7%8C_%EC%8B%9D%EB%AF%BC%EC%A7%80&amp;action=edit&amp;section=1" title="부분 편집: 초기 개척">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>북미 동해안에 최초의 영구 잉글랜드 거주지를 개척하려는 계획은 잉글랜드의 국왕 <a class="mw-disambig" href="/wiki/%EC%A0%9C%EC%9E%84%EC%8A%A4_1%EC%84%B8" title="제임스 1세">제임스 1세</a>가 두 개의 합자회사를 만든 1606년 말부터 시작되었다. <a href="/wiki/%EB%9F%B0%EB%8D%98_%ED%9A%8C%EC%82%AC" title="런던 회사">런던 회사</a>는 남부 지역을 담당하였고, <a class="mw-redirect" href="/wiki/%EC%A0%9C%EC%9E%84%EC%8A%A4%ED%83%80%EC%9A%B4_%EA%B1%B0%EC%A3%BC%EC%A7%80" title="제임스타운 거주지">제임스타운 거주지</a>를 개척하였다. <a class="new" href="/w/index.php?title=%ED%94%8C%EB%A6%AC%EB%A8%B8%EC%8A%A4_%ED%9A%8C%EC%82%AC&amp;action=edit&amp;redlink=1" title="플리머스 회사 (없는 문서)">플리머스 회사</a>는 페르디난드 조지 경의 지도 하에 오늘 날의 <a href="/wiki/%EB%89%B4%EC%9E%89%EA%B8%80%EB%9E%9C%EB%93%9C" title="뉴잉글랜드">뉴잉글랜드</a>를 포함하여 북부 지역을 담당하여 <a href="/wiki/1607%EB%85%84" title="1607년">1607년</a> 오늘 날의 <a href="/wiki/%EB%A9%94%EC%9D%B8_%EC%A3%BC" title="메인 주">메인 주</a>에 <a class="new" href="/w/index.php?title=%ED%8C%9D%ED%96%84_%EC%8B%9D%EB%AF%BC%EC%A7%80&amp;action=edit&amp;redlink=1" title="팝햄 식민지 (없는 문서)">팝햄 식민지</a>를 개척하였다. 이러한 개척은 120명 규모로는 극한의 어려움이라는 것이 증명되었고, 개척민들은 1년 후에 그곳을 포기해야 했다.</p>
<p><a href="/wiki/1620%EB%85%84" title="1620년">1620년</a> 11월에 <a class="mw-redirect" href="/wiki/%ED%95%84%EA%B7%B8%EB%A6%BC%EC%A6%88" title="필그림즈">필그림즈</a> 무리가 <a href="/wiki/%ED%94%8C%EB%A6%AC%EB%A8%B8%EC%8A%A4_%EC%8B%9D%EB%AF%BC%EC%A7%80" title="플리머스 식민지">플리머스 식민지</a>를 개척하였다. 비록 이 거주지는 엄청난 역경을 겪었지만, 약간의 이익을 올릴 수 있었다. 이것은 잉글랜드에서 긍정적인 평판을 가져왔다.</p>
<p>매사추세츠만 식민지는 영국 왕이나 회사에서 다양한 식민지 개척을 위해 칙허를 한 지역으로, 일부는 영역이 중복되어 개척되었고, 나중에 13개 식민지 영토가 구성된 적도 있었고, 식민지 집단의 일부는 매사추세츠만 식민지 역사와 직접적인 연관을 맺었다.</p>
<p>영국 <a href="/wiki/%ED%94%8C%EB%A6%AC%EB%A8%B8%EC%8A%A4" title="플리머스">플리머스</a>에 있던 <a href="/wiki/%EB%B2%84%EC%A7%80%EB%8B%88%EC%95%84_%ED%9A%8C%EC%82%AC" title="버지니아 회사">버지니아 회사</a>는 <a href="/wiki/1607%EB%85%84" title="1607년">1607년</a> 버지니아 칙허의 일부로 북위 38도선에서 45도선까지 토지를 인정했다. 현재 <a href="/wiki/%EB%A9%94%EC%9D%B8_%EC%A3%BC" title="메인 주">메인 주</a> 케네벡 강 하구에 만들어진 유일한 개척지 <a class="new" href="/w/index.php?title=%ED%8C%9D%ED%96%84_%EC%8B%9D%EB%AF%BC%EC%A7%80&amp;action=edit&amp;redlink=1" title="팝햄 식민지 (없는 문서)">팝햄 식민지</a>는 <a href="/wiki/1608%EB%85%84" title="1608년">1608년</a> 버려졌다. 북위 41도선 (롱 아일랜드 사운드의 남쪽)에서 남쪽의 토지는 자매 회사인 런던의 <a href="/wiki/%EB%B2%84%EC%A7%80%EB%8B%88%EC%95%84_%ED%9A%8C%EC%82%AC" title="버지니아 회사">버지니아 회사</a>에 부여되며, 이전부터 공동으로 영토의 영유권을 주장하고 있었다.</p>
<p><a href="/wiki/1607%EB%85%84" title="1607년">1607년</a> "플리머스 회사"의 영토가 뉴잉글랜드 플리머스 위원회의 지도로 개편되었다. 잉글랜드의 왕 제임스 1세는 북위 40도선에서 48도선까지 "전체 본토를 바다에서 바다까지" 칙허를 인정했다. 여기에는 남쪽은 현재 뉴저지 중앙에서 북쪽은 <a href="/wiki/%EB%89%B4%EB%B8%8C%EB%9F%B0%EC%A6%88%EC%9C%85_%EC%A3%BC" title="뉴브런즈윅 주">뉴브런즈윅 주</a>와 <a href="/wiki/%EB%85%B8%EB%B0%94%EC%8A%A4%EC%BD%94%EC%83%A4_%EC%A3%BC" title="노바스코샤 주">노바스코샤 주</a>까지 포함되어 있었다.</p>
<p>이후 <a href="/wiki/1620%EB%85%84" title="1620년">1620년</a>, <a class="mw-disambig" href="/wiki/%EC%98%81%EA%B5%AD_%EC%84%B1%EA%B3%B5%ED%9A%8C" title="영국 성공회">영국 성공회</a>에서 분리를 원했던 개척민 집단인 현재 <a href="/wiki/%ED%95%84%EA%B7%B8%EB%A6%BC_%ED%8C%8C%EB%8D%94%EC%8A%A4" title="필그림 파더스">필그림 파더스</a>라는 사람들이 《<a href="/wiki/%EB%A9%94%EC%9D%B4%ED%94%8C%EB%9D%BC%EC%9B%8C_%ED%98%B8" title="메이플라워 호">메이플라워 호</a>》를 타고 영국에서 건너와, 플리머스 위원회가 소유한 토지에 직접 플리머스 식민지를 만들었다. 필그림은 원래 버지니아 주 <a href="/wiki/%ED%97%88%EB%93%9C%EC%8A%A8_%EA%B0%95" title="허드슨 강">허드슨 강</a> 유역에 갈 예정이었지만, 강력한 폭풍우 등의 이유로 <a href="/wiki/%ED%94%8C%EB%A6%AC%EB%A8%B8%EC%8A%A4_(%EB%A7%A4%EC%82%AC%EC%B6%94%EC%84%B8%EC%B8%A0_%EC%A3%BC)" title="플리머스 (매사추세츠 주)">플리머스</a>로 머물게 되었다. 처음 정착한 곳이 현재의 플리머스 시가 되었다.</p>
<p><a href="/wiki/%EB%A9%94%EC%9D%B4%ED%94%8C%EB%9D%BC%EC%9B%8C_%EC%84%9C%EC%95%BD" title="메이플라워 서약">메이플라워 서약</a>을 만들어 <a href="/wiki/%ED%94%8C%EB%A6%AC%EB%A8%B8%EC%8A%A4_%EC%8B%9D%EB%AF%BC%EC%A7%80" title="플리머스 식민지">플리머스 식민지</a>를 세운 것은 미국의 역사에서 독창적인 사건이라고 할 수 있다. 하지만 뉴잉글랜드 지역에서 인구도, 경제력도 지배적이 된 것은 매사추세츠만 식민지이고 세일럼과 보스턴의 후계자였다. 또한 <a href="/wiki/1691%EB%85%84" title="1691년">1691년</a>부터 <a href="/wiki/1692%EB%85%84" title="1692년">1692년</a> 플리머스 식민지 및 기타 인근 식민지를 합병하여, 매사추세츠만 식민지로 통합되었다.</p>
