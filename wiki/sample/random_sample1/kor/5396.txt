<h1 class="firstHeading" id="firstHeading" lang="ko">매체 접근 제어</h1>
<div class="mw-body-content" id="bodyContent">
<div id="siteSub">위키백과, 우리 모두의 백과사전.</div>
<div id="contentSub"></div>
<div class="mw-jump" id="jump-to-nav">
					이동:					<a href="#mw-head">둘러보기</a>, 					<a href="#p-search">검색</a>
</div>
<div class="mw-content-ltr" dir="ltr" id="mw-content-text" lang="ko"><script>function mfTempOpenSection(id){var block=document.getElementById("mf-section-"+id);block.className+=" open-block";block.previousSibling.className+=" open-block";}</script><p><b>매체 접근 제어</b>(<span lang="en" xml:lang="en">Media Access Control</span>, <b>MAC</b>)는 <a href="/wiki/%ED%86%B5%EC%8B%A0_%ED%94%84%EB%A1%9C%ED%86%A0%EC%BD%9C" title="통신 프로토콜">자료 전송 프로토콜</a>의 하부 계층이며 <a class="mw-redirect" href="/wiki/OSI_%EB%AA%A8%EB%8D%B8" title="OSI 모델">일곱 계층의 OSI 모델</a>에 규정된 <a href="/wiki/%EB%8D%B0%EC%9D%B4%ED%84%B0_%EB%A7%81%ED%81%AC_%EA%B3%84%EC%B8%B5" title="데이터 링크 계층">데이터 링크 계층</a>의 일부이다.</p>
<p>유선 멀티드롭 네트워크를 위한 <a href="/wiki/%ED%8C%A8%ED%82%B7_%EA%B5%90%ED%99%98" title="패킷 교환">패킷 모드</a> 다중 접근 프로토콜의 예는 다음과 같다.</p>
<ul>
<li><a class="mw-redirect" href="/wiki/CSMA/CD" title="CSMA/CD">CSMA/CD</a> (<a href="/wiki/%EC%9D%B4%EB%8D%94%EB%84%B7" title="이더넷">이더넷</a>, <a href="/wiki/IEEE_802.3" title="IEEE 802.3">IEEE 802.3</a>에서 쓰임)</li>
<li><a class="new" href="/w/index.php?title=%ED%86%A0%ED%81%B0_%EB%B2%84%EC%8A%A4&amp;action=edit&amp;redlink=1" title="토큰 버스 (없는 문서)">토큰 버스</a> (<a class="mw-redirect" href="/wiki/IEEE_802.4" title="IEEE 802.4">IEEE 802.4</a>)</li>
<li><a class="mw-redirect" href="/wiki/%ED%86%A0%ED%81%B0_%EB%A7%81" title="토큰 링">토큰 링</a> (<a class="mw-redirect" href="/wiki/IBM_%ED%86%A0%ED%81%B0_%EB%A7%81" title="IBM 토큰 링">IEEE 802.5</a>)</li>
<li><a class="new" href="/w/index.php?title=%ED%86%A0%ED%81%B0_%ED%8C%A8%EC%8B%B1&amp;action=edit&amp;redlink=1" title="토큰 패싱 (없는 문서)">토큰 패싱</a> (<a class="new" href="/w/index.php?title=FDDI&amp;action=edit&amp;redlink=1" title="FDDI (없는 문서)">FDDI</a>에서 쓰임)</li>
</ul>
<p>패킷 라디오 무선 네트워크에 쓰이는 다중 접속 프로토콜의 예는 다음과 같다:</p>
<ul>
<li><a class="mw-redirect" href="/wiki/CSMA/CA" title="CSMA/CA">CSMA/CA</a> (<a href="/wiki/IEEE_802.11" title="IEEE 802.11">IEEE 802.11</a>/<a class="mw-redirect" href="/wiki/WiFi" title="WiFi">WiFi</a> <a class="mw-redirect" href="/wiki/%EB%AC%B4%EC%84%A0_LAN" title="무선 LAN">WLAN</a>에서 쓰임)</li>
<li><a class="mw-redirect" href="/wiki/ALOHAnet" title="ALOHAnet">슬롯형 ALOHA</a></li>
<li><a class="new" href="/w/index.php?title=%EB%8B%A4%EC%9D%B4%EB%82%98%EB%AF%B9_TDMA&amp;action=edit&amp;redlink=1" title="다이나믹 TDMA (없는 문서)">다이나믹 TDMA</a></li>
<li><a class="new" href="/w/index.php?title=R-ALOHA&amp;action=edit&amp;redlink=1" title="R-ALOHA (없는 문서)">R-ALOHA</a></li>
<li><a class="mw-redirect" href="/wiki/CDMA" title="CDMA">CDMA</a></li>
<li><a class="new" href="/w/index.php?title=OFDMA&amp;action=edit&amp;redlink=1" title="OFDMA (없는 문서)">OFDMA</a></li>
</ul>
