<p><b>매크로블록</b>(macroblock)은 <a href="/wiki/%EC%9D%B4%EC%82%B0_%EC%BD%94%EC%82%AC%EC%9D%B8_%EB%B3%80%ED%99%98" title="이산 코사인 변환">이산 코사인 변환</a>과 같은 선형 블록 변환 기반 <a href="/wiki/%EA%B7%B8%EB%A6%BC_%EC%95%95%EC%B6%95" title="그림 압축">그림</a> 및 <a href="/wiki/%EC%98%81%EC%83%81_%EC%95%95%EC%B6%95" title="영상 압축">영상 압축</a> 포맷의 처리 단위이다. 매크로블록은 보통 16x16 샘플로 이루어지며 변환 블록들로 다시 나뉜다. 그리고 예측블록으로 또 다시 나뉜다. 매크로블록에 기반한 포맷에는 <a href="/wiki/JPEG" title="JPEG">JPEG</a>, <a href="/wiki/H.261" title="H.261">H.261</a>, <a href="/wiki/MPEG-1#.ED.8C.8C.ED.8A.B8_2:.EC.98.81.EC.83.81" title="MPEG-1">MPEG-1 파트 2</a>, <a href="/wiki/H.262/MPEG-2_%ED%8C%8C%ED%8A%B8_2" title="H.262/MPEG-2 파트 2">H.262/MPEG-2 파트 2</a>, <a href="/wiki/H.263" title="H.263">H.263</a>, <a href="/wiki/MPEG-4_%ED%8C%8C%ED%8A%B8_2" title="MPEG-4 파트 2">MPEG-4 파트 2</a>, <a href="/wiki/H.264/MPEG-4_AVC" title="H.264/MPEG-4 AVC">H.264/MPEG-4 AVC</a>이 있다.<sup class="reference" id="cite_ref-H261_1-0"><a href="#cite_note-H261-1">[1]</a></sup><sup class="reference" id="cite_ref-H262_2-0"><a href="#cite_note-H262-2">[2]</a></sup><sup class="reference" id="cite_ref-H263_3-0"><a href="#cite_note-H263-3">[3]</a></sup><sup class="reference" id="cite_ref-H264_4-0"><a href="#cite_note-H264-4">[4]</a></sup></p>
