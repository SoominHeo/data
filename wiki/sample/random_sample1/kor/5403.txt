<p><b>맥 타이탄</b>(<span style="font-size: smaller;"><a href="/wiki/%EC%98%81%EC%96%B4" title="영어">영어</a>:</span> <span lang="en" xml:lang="en">Mack Titan</span>)은 <a href="/wiki/%EB%AF%B8%EA%B5%AD" title="미국">미국</a> <a class="mw-redirect" href="/wiki/%EB%A7%A5_%ED%8A%B8%EB%9F%AD" title="맥 트럭">맥 트럭</a>사가 생산하는 대형 트럭으로 <a href="/wiki/1995%EB%85%84" title="1995년">1995년</a>에 <a href="/wiki/%EB%A1%9C%EB%93%9C_%ED%8A%B8%EB%A0%88%EC%9D%B8" title="로드 트레인">로드 트레인</a>과 <a href="/wiki/%EC%98%A4%EB%B2%84%EC%82%AC%EC%9D%B4%EC%A6%88_%EB%A1%9C%EB%93%9C" title="오버사이즈 로드">오버사이즈 로드</a>에 대응하기 위해 <a href="/wiki/%EC%98%A4%EC%8A%A4%ED%8A%B8%EB%A0%88%EC%9D%BC%EB%A6%AC%EC%95%84" title="오스트레일리아">오스트레일리아</a>에서 최초로 출시 되었다가 <a href="/wiki/2008%EB%85%84" title="2008년">2008년</a>에 <a href="/wiki/%EB%B6%81%EC%95%84%EB%A9%94%EB%A6%AC%EC%B9%B4" title="북아메리카">북아메리카</a> 사양에 맞춰 <a href="/wiki/%EB%AF%B8%EA%B5%AD" title="미국">미국</a>에서 출시하기 시작했다.</p>

<h2><span class="mw-headline" id=".EA.B0.9C.EC.9A.94">개요</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%A7%A5_%ED%83%80%EC%9D%B4%ED%83%84&amp;action=edit&amp;section=1" title="부분 편집: 개요">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>이 트럭은 <a href="/wiki/%EC%98%A4%EC%8A%A4%ED%8A%B8%EB%A0%88%EC%9D%BC%EB%A6%AC%EC%95%84" title="오스트레일리아">오스트레일리아</a>의 장거리 중량 수송 시장을 공략하기 위해 개발이 되었고 큰 규모와 출력이 강한 편이다. 최대 추력은 685마력, 최대 토크는 317kg/m로 최대 견인 총중량(<span style="font-size: smaller;"><a href="/wiki/%EC%98%81%EC%96%B4" title="영어">영어</a>:</span> <span lang="en" xml:lang="en">GCWR</span>)<sup class="reference" id="cite_ref-1"><a href="#cite_note-1">[1]</a></sup>은 최대 200톤을 자랑하는 로드 <a href="/wiki/%ED%8A%B8%EB%A0%88%EC%9D%BC%EB%9F%AC_(%EC%9E%90%EB%8F%99%EC%B0%A8)" title="트레일러 (자동차)">트레일러</a> 중에서 가장 강력한 편이다. 다른 로드 <a href="/wiki/%ED%8A%B8%EB%A0%88%EC%9D%BC%EB%9F%AC_(%EC%9E%90%EB%8F%99%EC%B0%A8)" title="트레일러 (자동차)">트레일러</a>과 달리 중량 화물 수송에 특회된 옵션을 갖추고 있다.</p>
<h2><span class="mw-headline" id=".EC.98.A4.EC.8A.A4.ED.8A.B8.EB.A0.88.EC.9D.BC.EB.A6.AC.EC.95.84.EC.9D.98_.EC.82.AC.EC.96.91"><a href="/wiki/%EC%98%A4%EC%8A%A4%ED%8A%B8%EB%A0%88%EC%9D%BC%EB%A6%AC%EC%95%84" title="오스트레일리아">오스트레일리아</a>의 사양</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%A7%A5_%ED%83%80%EC%9D%B4%ED%83%84&amp;action=edit&amp;section=2" title="부분 편집: 오스트레일리아의 사양">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<ul>
<li>더운 기후에 대응하긱 위해 <a href="/wiki/%EB%9D%BC%EB%94%94%EC%97%90%EC%9D%B4%ED%84%B0" title="라디에이터">라디에이터</a>를 장착했다.</li>
<li><a href="/wiki/%EB%A1%9C%EB%93%9C_%ED%8A%B8%EB%A0%88%EC%9D%B8" title="로드 트레인">로드 트레인</a> 견인에 대응하기 위해 대용량 에어 탱크를 장착했다.</li>
<li>중량 화물과 비포장 도로에 대응하기 위해 이중, 삼중 강화 프레임 구조로 설계했다.</li>
<li>로드킬에 대응하기 위해 대형 전투범퍼를 사용했다.</li>
<li>비포장 도로 주행 시 먼지 유입을 방지하기 위해 운전실 좌우에 수직으로 높이 뻗는 공기 흡입구가 있다.</li>
<li>후륜 가변축 옵션이 있으며 <a class="mw-redirect" href="/wiki/%EB%A7%A5_%ED%8A%B8%EB%9F%AD" title="맥 트럭">맥 트럭</a>에서 제작한 16단 변속기와 <a class="new" href="/w/index.php?title=%EC%9D%B4%ED%8A%BC_%EC%BB%B4%ED%8F%AC%EB%A0%88%EC%9D%B4%EC%85%98&amp;action=edit&amp;redlink=1" title="이튼 컴포레이션 (없는 문서)">이튼 컴포레이션</a>에서 제작한 18단 변속기 옵션이 있다.</li>
<li><a href="/wiki/%EC%98%81%EA%B5%AD" title="영국">영국</a>, <a href="/wiki/%EC%9D%BC%EB%B3%B8" title="일본">일본</a>, <a href="/wiki/%EB%89%B4%EC%A7%88%EB%9E%9C%EB%93%9C" title="뉴질랜드">뉴질랜드</a>와 마찬가지로 좌측 통행을 하기 때문에 우핸들 옵션을 적용했다.</li>
</ul>
