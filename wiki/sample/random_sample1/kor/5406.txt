<p><b>맥스 시번 보커스</b>(<span style="font-size: smaller;"><a href="/wiki/%EC%98%81%EC%96%B4" title="영어">영어</a>:</span> <span lang="en" xml:lang="en">Max Sieben Baucus</span>, <a href="/wiki/1941%EB%85%84" title="1941년">1941년</a> <a href="/wiki/12%EC%9B%94_11%EC%9D%BC" title="12월 11일">12월 11일</a> ~ )는 <a href="/wiki/%EB%AF%B8%EA%B5%AD" title="미국">미국</a>의 정치인이다.</p>
<p><a href="/wiki/%EB%AA%AC%ED%83%9C%EB%82%98_%EC%A3%BC" title="몬태나 주">몬태나 주</a> <a href="/wiki/%ED%97%AC%EB%A0%88%EB%82%98_(%EB%AA%AC%ED%83%9C%EB%82%98_%EC%A3%BC)" title="헬레나 (몬태나 주)">헬레나</a>에서 태어났다. 1964년 <a href="/wiki/%EB%AF%B8%EB%84%A4%EC%86%8C%ED%83%80_%EC%A3%BC" title="미네소타 주">미네소타 주</a> 칼튼 대학교를 다니다 <a href="/wiki/%EC%BA%98%EB%A6%AC%ED%8F%AC%EB%8B%88%EC%95%84_%EC%A3%BC" title="캘리포니아 주">캘리포니아 주</a> <a class="mw-redirect" href="/wiki/%EC%8A%A4%ED%83%A0%ED%8F%AC%EB%93%9C_%EB%8C%80%ED%95%99%EA%B5%90" title="스탠포드 대학교">스탠포드 대학교</a>로 편입하여 <a href="/wiki/%EA%B2%BD%EC%A0%9C%ED%95%99" title="경제학">경제학</a>을 전공으로 학사 학위를 취득했다. 그 후 <a href="/wiki/%EC%8A%A4%ED%83%A0%ED%8D%BC%EB%93%9C_%EB%B2%95%ED%95%99%EB%8C%80%ED%95%99%EC%9B%90" title="스탠퍼드 법학대학원">스탠포드 로스쿨</a>로 진학하여 1967년 법무박사(J.D.) 학위를 취득한 후 <a href="/wiki/%EB%B3%80%ED%98%B8%EC%82%AC" title="변호사">변호사</a> 자격을 얻었다. <a class="mw-redirect" href="/wiki/%EC%9B%8C%EC%8B%B1%ED%84%B4_D._C." title="워싱턴 D. C.">워싱턴 D. C.</a>의 <a class="new" href="/w/index.php?title=%EC%A6%9D%EA%B6%8C%EA%B1%B0%EB%9E%98%EC%9C%84%EC%9B%90%ED%9A%8C&amp;action=edit&amp;redlink=1" title="증권거래위원회 (없는 문서)">증권거래위원회</a>(Securities and Exchange Commission)에서 3년 간 일한 후 몬태나 주로 돌아가 변호사로 활동하다 <a href="/wiki/1973%EB%85%84" title="1973년">1973년</a> 주의회 의원으로 뽑혔다. <a href="/wiki/1974%EB%85%84" title="1974년">1974년</a> <a href="/wiki/%EB%AF%BC%EC%A3%BC%EB%8B%B9_(%EB%AF%B8%EA%B5%AD)" title="민주당 (미국)">민주당</a> 소속으로 <a href="/wiki/%EB%AF%B8%EA%B5%AD_%ED%95%98%EC%9B%90" title="미국 하원">연방 하원의원</a>에 선출되어 <a href="/wiki/1975%EB%85%84" title="1975년">1975년</a>부터 하원의원이 되었고, <a href="/wiki/1978%EB%85%84" title="1978년">1978년</a> <a href="/wiki/%EB%AF%B8%EA%B5%AD_%EC%83%81%EC%9B%90" title="미국 상원">상원의원</a>에 당선되어 상원으로 자리를 옮겨 2014년까지 39년간 국회의원으로 재직하였다. 그는 주로 <a class="mw-redirect" href="/wiki/%EC%A1%B0%EC%84%B8" title="조세">조세</a>·<a class="new" href="/w/index.php?title=%EC%9E%AC%EB%AC%B4&amp;action=edit&amp;redlink=1" title="재무 (없는 문서)">재무</a>·<a href="/wiki/%EB%86%8D%EC%97%85" title="농업">농업</a> 분야에서 활동하며, 상원 내에서 대체적으로 중도적인 입장을 보였다. 농업 지역인 몬태나 주의 이익을 대변하여 외국에 <a href="/wiki/%EC%87%A0%EA%B3%A0%EA%B8%B0" title="쇠고기">쇠고기</a> 수출을 늘리기 위한 노력을 했으며, <a href="/wiki/2006%EB%85%84_12%EC%9B%94" title="2006년 12월">2006년 12월</a> <a href="/wiki/%EB%8C%80%ED%95%9C%EB%AF%BC%EA%B5%AD" title="대한민국">대한민국</a>과의 <a class="mw-redirect" href="/wiki/%ED%95%9C%EB%AF%B8%EC%9E%90%EC%9C%A0%EB%AC%B4%EC%97%AD%ED%98%91%EC%A0%95" title="한미자유무역협정">FTA</a> 협상 회담을 몬태나 주로 유치하는 데 영향력을 행사하였고, 회담장에서는 쇠고기 스테이크 요리를 먹으며 미리 익혀둔 <a href="/wiki/%ED%95%9C%EA%B5%AD%EC%96%B4" title="한국어">한국어</a>로 '맛있습니다'를 연발하는 퍼포먼스를 펼쳐 눈길을 끌었다<sup class="reference" id="cite_ref-1"><a href="#cite_note-1">[1]</a></sup>. <a href="/wiki/2006%EB%85%84" title="2006년">2006년</a> 말 중간 선거로 민주당이 다수당이 되면서 <a href="/wiki/2007%EB%85%84" title="2007년">2007년</a>부터 상원 재무위원회 위원장을 맡아 최근에도 대한민국에 대해 쇠고기 시장 개방을 강하게 요구하였다<sup class="reference" id="cite_ref-2"><a href="#cite_note-2">[2]</a></sup>. <a href="/wiki/2013%EB%85%84" title="2013년">2013년</a> 말, <a href="/wiki/%EC%A4%91%ED%99%94%EC%9D%B8%EB%AF%BC%EA%B3%B5%ED%99%94%EA%B5%AD" title="중화인민공화국">중화인민공화국</a> 주재 대사로 지명되어, 상원 인준을 통과한 후 <a href="/wiki/2014%EB%85%84" title="2014년">2014년</a> 상원의원직에서 사퇴하고 중국에 부임하였다.<sup class="reference" id="cite_ref-3"><a href="#cite_note-3">[3]</a></sup></p>
