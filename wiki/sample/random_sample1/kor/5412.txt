<p><b>맨틀</b>(mantle)은 <a href="/wiki/%EC%BB%B4%ED%93%A8%ED%8C%85" title="컴퓨팅">컴퓨팅</a>에서 3차원 비디오 게임을 대상으로 하는 저급 <a href="/wiki/%EB%A0%8C%EB%8D%94%EB%A7%81" title="렌더링">렌더링</a> <a href="/wiki/API" title="API">API</a>이다.<sup class="reference" id="cite_ref-whitepaper_4-0"><a href="#cite_note-whitepaper-4">[4]</a></sup> <a href="/wiki/%EC%96%B4%EB%93%9C%EB%B0%B4%EC%8A%A4%ED%8A%B8_%EB%A7%88%EC%9D%B4%ED%81%AC%EB%A1%9C_%EB%94%94%EB%B0%94%EC%9D%B4%EC%8B%9C%EC%8A%A4" title="어드밴스트 마이크로 디바이시스">AMD</a>가 2013년을 기점으로 <a class="mw-redirect" href="/wiki/EA_DICE" title="EA DICE">DICE</a>와 협력하여 개발하였다.<sup class="reference" id="cite_ref-Dave_Altavilla_1-1"><a href="#cite_note-Dave_Altavilla-1">[1]</a></sup> 맨틀은 <a href="/wiki/Direct3D" title="Direct3D">Direct3D</a>와 <a href="/wiki/OpenGL" title="OpenGL">OpenGL</a>을 대체할 목적으로 설계되었으며 주로 <a href="/wiki/%EA%B0%9C%EC%9D%B8%EC%9A%A9_%EC%BB%B4%ED%93%A8%ED%84%B0" title="개인용 컴퓨터">개인용 컴퓨터</a>를 대상으로 하지만, <a href="/wiki/%ED%94%8C%EB%A0%88%EC%9D%B4%EC%8A%A4%ED%85%8C%EC%9D%B4%EC%85%98_4" title="플레이스테이션 4">플레이스테이션 4</a>, <a href="/wiki/%EC%97%91%EC%8A%A4%EB%B0%95%EC%8A%A4_%EC%9B%90" title="엑스박스 원">엑스박스 원</a>에 장착된 GPU도 지원한다.<sup class="reference" id="cite_ref-Dave_Altavilla_1-2"><a href="#cite_note-Dave_Altavilla-1">[1]</a></sup><sup class="reference" id="cite_ref-tomshardware_5-0"><a href="#cite_note-tomshardware-5">[5]</a></sup> AMD에 따르면 <a href="/wiki/%ED%81%AC%EB%A1%9C%EB%85%B8%EC%8A%A4_%EA%B7%B8%EB%A3%B9" title="크로노스 그룹">크로노스 그룹</a>의 맨틀 태생의 <a href="/wiki/%EB%B2%8C%EC%BB%A8_(API)" title="벌컨 (API)">벌컨 API</a>이 <a href="/wiki/%EB%B9%84%EB%94%94%EC%98%A4_%EA%B2%8C%EC%9E%84" title="비디오 게임">게이밍</a> 산업에서 맨틀을 큰 폭으로 대체하고 있기 때문에 맨틀은 2015년 3월 이후로 포커스에 변화가 있을 예정이다.<sup class="reference" id="cite_ref-6"><a href="#cite_note-6">[6]</a></sup> 2년 동안 드라이버를 통해 지원되었으나 2015년 3월까지 맨틀 API 그 자체는 대중들에게 공표되지 못했다.<sup class="reference" id="cite_ref-7"><a href="#cite_note-7">[7]</a></sup></p>
