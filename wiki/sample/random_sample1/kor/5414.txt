<p><b>맵시 쿼크</b>(<span lang="en" xml:lang="en">charm quark</span> <small>참 쿼크<sup>[<a href="/wiki/%EC%9C%84%ED%82%A4%EB%B0%B1%EA%B3%BC:%EC%98%81%EC%96%B4%EC%9D%98_%ED%95%9C%EA%B8%80_%ED%91%9C%EA%B8%B0" title="위키백과:영어의 한글 표기">*</a>]</sup></small>, 기호 c)는 제2<a href="/wiki/%EC%84%B8%EB%8C%80_(%EB%AC%BC%EB%A6%AC%ED%95%99)" title="세대 (물리학)">세대</a>의 <a href="/wiki/%EC%BF%BC%ED%81%AC" title="쿼크">쿼크</a>로, +(2/3) <a href="/wiki/%EA%B8%B0%EB%B3%B8_%EC%A0%84%ED%95%98" title="기본 전하">e</a>의 전하량을 지니고 있다. 질량은 약1.3<a href="/wiki/%EC%A0%84%EC%9E%90%EB%B3%BC%ED%8A%B8" title="전자볼트">GeV</a>/<a class="mw-redirect" href="/wiki/%EB%B9%9B%EC%9D%98_%EC%86%8D%EB%8F%84" title="빛의 속도"><i>c</i></a><sup>2</sup>이다. 이는 네 번째로 가장 무거운 쿼크이며, <a href="/wiki/%EC%96%91%EC%84%B1%EC%9E%90" title="양성자">양성자</a>보다 조금 더 무겁다.</p>
<h2><span class="mw-headline" id=".EC.97.AD.EC.82.AC">역사</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%A7%B5%EC%8B%9C_%EC%BF%BC%ED%81%AC&amp;action=edit&amp;section=1" title="부분 편집: 역사">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p><a href="/wiki/1970%EB%85%84" title="1970년">1970년</a> <a class="mw-redirect" href="/wiki/%EC%85%B8%EB%8D%98_%EA%B8%80%EB%9E%98%EC%87%BC" title="셸던 글래쇼">셸던 글래쇼</a>와 <a href="/wiki/%EC%9D%B4%EC%98%A4%EC%95%84%EB%8B%88%EC%8A%A4_%EC%9D%BC%EB%A6%AC%EC%98%A4%ED%92%80%EB%A1%9C%EC%8A%A4" title="이오아니스 일리오풀로스">이오아니스 일리오풀로스</a>, <a class="new" href="/w/index.php?title=%EB%A3%A8%EC%B9%98%EC%95%84%EB%85%B8_%EB%A7%88%EC%9D%B4%EC%95%84%EB%8B%88&amp;action=edit&amp;redlink=1" title="루치아노 마이아니 (없는 문서)">루치아노 마이아니</a>(<span style="font-size: smaller;"><a href="/wiki/%EC%9D%B4%ED%83%88%EB%A6%AC%EC%95%84%EC%96%B4" title="이탈리아어">이탈리아어</a>:</span> <span lang="it" xml:lang="it">Luciano Maiani</span>)가 네 번째 맵시 쿼크의 존재를 처음으로 예측하였다. <a href="/wiki/1974%EB%85%84" title="1974년">1974년</a> <a href="/wiki/%EC%8A%A4%ED%83%A0%ED%8D%BC%EB%93%9C_%EC%84%A0%ED%98%95_%EA%B0%80%EC%86%8D%EA%B8%B0_%EC%84%BC%ED%84%B0" title="스탠퍼드 선형 가속기 센터">스탠퍼드 선형 가속기 센터</a>의 <a href="/wiki/%EB%B2%84%ED%84%B4_%EB%A6%AD%ED%84%B0" title="버턴 릭터">버턴 릭터</a> 연구진과 <a class="mw-redirect" href="/wiki/%EB%B8%8C%EB%A3%A9%ED%97%A4%EC%9D%B4%EB%B8%90_%EA%B5%AD%EB%A6%BD_%EC%97%B0%EA%B5%AC%EC%86%8C" title="브룩헤이븐 국립 연구소">브룩헤이븐 국립 연구소</a>의 <a href="/wiki/%EC%83%88%EB%AE%A4%EC%96%BC_%EC%B0%A8%EC%98%A4_%EC%B6%A9_%ED%8C%85" title="새뮤얼 차오 충 팅">새뮤얼 차오 충 팅</a> 연구진이 각각 독립적으로 <a href="/wiki/%EC%A0%9C%EC%9D%B4/%ED%94%84%EC%8B%9C_%EC%A4%91%EA%B0%84%EC%9E%90" title="제이/프시 중간자">J/ψ</a> 중간자(c<span style="text-decoration:overline;">c</span>)와 함께 발견하였다. 그 당시 발견한 맵시 입자를 <a class="mw-redirect" href="/wiki/%EB%B8%8C%EB%A3%A9%ED%97%A4%EC%9D%B4%EB%B8%90_%EA%B5%AD%EB%A6%BD_%EC%97%B0%EA%B5%AC%EC%86%8C" title="브룩헤이븐 국립 연구소">브룩헤이븐 국립 연구소</a> 연구진은 "J입자"로, <a href="/wiki/%EC%8A%A4%ED%83%A0%ED%8D%BC%EB%93%9C_%EC%84%A0%ED%98%95_%EA%B0%80%EC%86%8D%EA%B8%B0_%EC%84%BC%ED%84%B0" title="스탠퍼드 선형 가속기 센터">스탠퍼드 선형 가속기 센터</a> 연구진은 "ψ입자"로 이름 지었으며, 이후 이 입자는 "J/ψ 입자"로 불리게 되었다.</p>
<h2><span class="mw-headline" id=".EB.A7.B5.EC.8B.9C_.EC.BF.BC.ED.81.AC.EB.A5.BC_.EC.A7.80.EB.8B.88.EB.8A.94_.EA.B0.95.EC.9E.85.EC.9E.90">맵시 쿼크를 지니는 강입자</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%A7%B5%EC%8B%9C_%EC%BF%BC%ED%81%AC&amp;action=edit&amp;section=2" title="부분 편집: 맵시 쿼크를 지니는 강입자">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>다음과 같은 일부 <a href="/wiki/%EA%B0%95%EC%9E%85%EC%9E%90" title="강입자">강입자</a>는 맵시 쿼크를 지니고 있다.</p>
<ul>
<li><a href="/wiki/%EB%94%94_%EC%A4%91%EA%B0%84%EC%9E%90" title="디 중간자">D 중간자</a>는 하나의 맵시 쿼크(혹은 그 <a href="/wiki/%EB%B0%98%EC%9E%85%EC%9E%90" title="반입자">반입자</a>)와 하나의 <a href="/wiki/%EC%9C%84_%EC%BF%BC%ED%81%AC" title="위 쿼크">위 쿼크</a> 또는 <a href="/wiki/%EC%95%84%EB%9E%98_%EC%BF%BC%ED%81%AC" title="아래 쿼크">아래 쿼크</a>를 지니는 <a href="/wiki/%EC%A4%91%EA%B0%84%EC%9E%90" title="중간자">중간자</a>이다.</li>
<li><a href="/wiki/%EB%94%94_%EC%A4%91%EA%B0%84%EC%9E%90" title="디 중간자">D<sub>s</sub> 중간자</a>는 하나의 맵시 쿼크와 하나의 <a href="/wiki/%EA%B8%B0%EB%AC%98_%EC%BF%BC%ED%81%AC" title="기묘 쿼크">기묘 쿼크</a>를 지닌다.</li>
<li><a href="/wiki/%EC%A0%9C%EC%9D%B4/%ED%94%84%EC%8B%9C_%EC%A4%91%EA%B0%84%EC%9E%90" title="제이/프시 중간자">제이/프시 중간자</a>는 "맵시 쿼크"와 "맵시 반쿼크"로 이루어진다.</li>
<li>맵시 <a href="/wiki/%EC%A4%91%EC%9E%85%EC%9E%90" title="중입자">중입자</a>도 존재하며, 기묘한 중입자와 대응되도록 명명한다.</li>
</ul>
