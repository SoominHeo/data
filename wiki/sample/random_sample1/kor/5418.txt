<p><b>후촉 고조 맹지상</b>(後蜀高祖 孟知祥, <a href="/wiki/874%EB%85%84" title="874년">874년</a> <a href="/wiki/5%EC%9B%94_10%EC%9D%BC" title="5월 10일">5월 10일</a><sup class="reference" id="cite_ref-AS_1-6"><a href="#cite_note-AS-1">[1]</a></sup><sup class="reference" id="cite_ref-SGCQ48_4-1"><a href="#cite_note-SGCQ48-4">[4]</a></sup> ~ <a href="/wiki/934%EB%85%84" title="934년">934년</a> <a href="/wiki/5%EC%9B%94_24%EC%9D%BC" title="5월 24일">5월 24일</a><sup class="reference" id="cite_ref-AS_1-7"><a href="#cite_note-AS-1">[1]</a></sup><sup class="reference" id="cite_ref-ZZTJ279_3-3"><a href="#cite_note-ZZTJ279-3">[3]</a></sup>)은 <a class="mw-redirect" href="/wiki/%EC%98%A4%EB%8C%80%EC%8B%AD%EA%B5%AD_%EC%8B%9C%EB%8C%80" title="오대십국 시대">오대십국 시대</a> <a href="/wiki/%ED%9B%84%EC%B4%89" title="후촉">후촉</a>의 창건자이자 초대 <a href="/wiki/%ED%99%A9%EC%A0%9C" title="황제">황제</a>였다. 그는 당초 <a href="/wiki/%ED%9B%84%EB%8B%B9" title="후당">후당</a>의 <a href="/wiki/%EC%9E%A5%EA%B5%B0" title="장군">장군</a>이었으며, 후당의 개국 황제 <a class="mw-redirect" href="/wiki/%ED%9B%84%EB%8B%B9_%EC%9E%A5%EC%A2%85" title="후당 장종">장종</a> <a href="/wiki/%EC%9D%B4%EC%A1%B4%EC%9A%B1" title="이존욱">이존욱</a>의 큰누나 또는 사촌<sup class="reference" id="cite_ref-GPFQ_6-0"><a href="#cite_note-GPFQ-6">[6]</a></sup>과 결혼하였고, 장종의 <a href="/wiki/%EC%A0%84%EC%B4%89" title="전촉">전촉</a> 원정 이후에는 서천절도사(西川節度使, <a class="mw-redirect" href="/wiki/%EB%B3%B8%EB%B6%80" title="본부">본부</a>는 지금의 <a href="/wiki/%EC%93%B0%EC%B4%A8_%EC%84%B1" title="쓰촨 성">쓰촨 성</a> <a href="/wiki/%EC%B2%AD%EB%91%90_%EC%8B%9C" title="청두 시">청두 시</a>)를 지냈으나, 장종 사후 장종의 의형이자 후계자인 <a class="mw-redirect" href="/wiki/%ED%9B%84%EB%8B%B9_%EB%AA%85%EC%A2%85" title="후당 명종">명종</a> <a href="/wiki/%EC%9D%B4%EC%82%AC%EC%9B%90" title="이사원">이사원</a>과 소원해졌다. 나중에는 명종의 추밀사(樞密使) <a href="/wiki/%EC%95%88%EC%A4%91%ED%9A%8C" title="안중회">안중회</a>의 비난이 두려워서, 동천절도사(東川節度使, <a class="mw-redirect" href="/wiki/%EB%B3%B8%EB%B6%80" title="본부">본부</a>는 지금의 <a href="/wiki/%EC%93%B0%EC%B4%A8_%EC%84%B1" title="쓰촨 성">쓰촨 성</a> <a href="/wiki/%EB%AA%90%EC%96%91_%EC%8B%9C" title="몐양 시">몐양 시</a>) <a href="/wiki/%EB%8F%99%EC%9E%A5" title="동장">동장</a>과 결탁하여 <a href="/wiki/%EB%B0%98%EB%9E%80" title="반란">반란</a>을 일으켰다. 이후 맹・동 <a class="mw-redirect" href="/wiki/%EC%97%B0%ED%95%A9%EA%B5%B0" title="연합군">연합군</a>은 그들을 진압하려는 후당의 시도를 물리칠 수 있었고, 맹지상은 나중에 동장을 격파하고 동장의 <a href="/wiki/%EC%98%81%ED%86%A0" title="영토">영토</a>까지 자신의 지배하에 편입시켰다. 명종 연간의 남은 기간에 명종의 명목상의 신하가 된 후, 그는 <a href="/wiki/934%EB%85%84" title="934년">934년</a> 후촉을 세우고 자신의 <a href="/wiki/%EC%98%81%EC%A7%80" title="영지">영지</a>를 <a class="mw-redirect" href="/wiki/%EB%8F%85%EB%A6%BD%EA%B5%AD" title="독립국">독립국</a>으로 선포하였다. <a href="/wiki/%EC%9E%90_(%EC%9D%B4%EB%A6%84)" title="자 (이름)">자</a>는 <b>보윤</b>(保胤).</p>

<h2><span class="mw-headline" id=".EC.83.9D.EC.95.A0">생애</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%A7%B9%EC%A7%80%EC%83%81&amp;action=edit&amp;section=1" title="부분 편집: 생애">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p><a href="/wiki/%ED%9B%84%EC%B4%89" title="후촉">후촉</a>이 건국되기 이전 촉지역은 <a href="/wiki/%EC%A0%84%EC%B4%89" title="전촉">전촉</a>이 지배하고 있었으나, 내부의 부패가 심해 <a href="/wiki/925%EB%85%84" title="925년">925년</a> 후당의 장종(莊宗:<a href="/wiki/%EC%9D%B4%EC%A1%B4%EC%9A%B1" title="이존욱">이존욱</a>)에 의해 멸망당했다. 그러고 나서 이 땅의 통치를 위임받은 이가 후촉의 시조인 맹지상이었다.</p>
<p>그 후 <a href="/wiki/%ED%9B%84%EB%8B%B9" title="후당">후당</a>에서는 장종이 살해당하고 명종(明宗:이사원)이 옹립되었다. 명종은 촉에 있던 맹지상에 대해 경계심을 품고 그를 억제하려는 움직임을 보였다. 이에 반발한 맹지상은 <a href="/wiki/930%EB%85%84" title="930년">930년</a> 거병하여 후당군을 촉에서 몰아내고, <a href="/wiki/932%EB%85%84" title="932년">932년</a>까지 촉의 전역을 장악했다. 이쯤되자 명종도 맹지상을 완전히 지배하에 두는 것이 어렵다는 것을 알고 회유책으로 전환하여 <a href="/wiki/933%EB%85%84" title="933년">933년</a> 맹지상을 촉왕에 봉하였다. 다음 해 명종이 죽자 맹지상도 완전히 자립하여 황제의 자리에 오르나 같은 해 사망했다.</p>
<p>그 후 맹지상의 3남 <a href="/wiki/%EB%A7%B9%EC%B0%BD" title="맹창">맹창</a>(孟昶)이 제위를 이었다.</p>
<h3><span class="mw-headline" id=".EC.83.9D.EC.95.A0_.EC.B4.88.EB.B0.98">생애 초반</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%A7%B9%EC%A7%80%EC%83%81&amp;action=edit&amp;section=2" title="부분 편집: 생애 초반">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<h4><span class="mw-headline" id=".EC.B6.9C.EC.8B.A0_.EB.B0.B0.EA.B2.BD">출신 배경</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%A7%B9%EC%A7%80%EC%83%81&amp;action=edit&amp;section=3" title="부분 편집: 출신 배경">편집</a><span class="mw-editsection-bracket">]</span></span></h4>
<p>맹지상은 <a href="/wiki/%EB%8B%B9_%EC%9D%98%EC%A2%85" title="당 의종">당 의종</a> 연간의 <a href="/wiki/874%EB%85%84" title="874년">874년</a>에 태어났다. 그의 집안은 형주(邢州, 지금의 <a href="/wiki/%ED%97%88%EB%B2%A0%EC%9D%B4_%EC%84%B1" title="허베이 성">허베이 성</a> <a href="/wiki/%EC%8B%B1%ED%83%80%EC%9D%B4_%EC%8B%9C" title="싱타이 시">싱타이 시</a>) 출신으로, 그의 할아버지 맹찰(孟察)과 아버지 맹도(孟道)를 비롯한 그의 조상들은 수대에 걸쳐 형주에서 <a href="/wiki/%EC%9E%A5%EA%B5%90" title="장교">장교</a>를 지냈다. 그의 백부 <a href="/wiki/%EB%A7%B9%EB%B0%A9%EB%A6%BD" title="맹방립">맹방립</a>은 <a href="/wiki/%EB%8B%B9%EB%82%98%EB%9D%BC" title="당나라">당나라</a> 말에 형주가 속해 있던 소의군 절도사(昭義軍節度使, <a class="mw-redirect" href="/wiki/%EB%B3%B8%EB%B6%80" title="본부">본부</a>는 지금의 <a href="/wiki/%EC%82%B0%EC%8B%9C_%EC%84%B1_(%EC%82%B0%EC%84%9C%EC%84%B1)" title="산시 성 (산서성)">산시 성</a> <a href="/wiki/%EC%B0%BD%EC%A6%88_%EC%8B%9C" title="창즈 시">창즈 시</a>)가 되었고,<sup class="reference" id="cite_ref-HFD136_5-1"><a href="#cite_note-HFD136-5">[5]</a></sup> 나중에는 <a href="/wiki/882%EB%85%84" title="882년">882년</a>에 번진(藩鎭)의 <a href="/wiki/%EC%88%98%EB%8F%84" title="수도">수도</a>를 노주(潞州)에서 형주로 이전하여, <a href="/wiki/883%EB%85%84" title="883년">883년</a> 두쪽으로 번진의 분열을 이끈 <a href="/wiki/%EB%B0%98%EB%9E%80" title="반란">반란</a>을 촉발시켰다.<sup class="reference" id="cite_ref-ZZTJ255_7-0"><a href="#cite_note-ZZTJ255-7">[7]</a></sup></p>
<p>이후 <a href="/wiki/889%EB%85%84" title="889년">889년</a> 맹방립이 <a href="/wiki/%EC%9E%90%EC%82%B4" title="자살">자살</a>하자, 그 후사는 그의 동생(맹지상의 삼촌) 맹천(孟遷)에게로 이어졌고, 맹천은 <a href="/wiki/890%EB%85%84" title="890년">890년</a> 자신이 지배하던 소의군 번진의 일부를 대<a href="/wiki/%EA%B5%B0%EB%B2%8C" title="군벌">군벌</a> 하동절도사(河東節度使, <a class="mw-redirect" href="/wiki/%EB%B3%B8%EB%B6%80" title="본부">본부</a>는 지금의 <a href="/wiki/%EC%82%B0%EC%8B%9C_%EC%84%B1_(%EC%82%B0%EC%84%9C%EC%84%B1)" title="산시 성 (산서성)">산시 성</a> <a href="/wiki/%ED%83%80%EC%9D%B4%EC%9C%84%EC%95%88_%EC%8B%9C" title="타이위안 시">타이위안 시</a>) <a href="/wiki/%EC%9D%B4%EA%B7%B9%EC%9A%A9" title="이극용">이극용</a>에게 바치고 투항하였다.<sup class="reference" id="cite_ref-ZZTJ258_8-0"><a href="#cite_note-ZZTJ258-8">[8]</a></sup> 후에 <a href="/wiki/%EC%9D%B4%EA%B7%B9%EC%9A%A9" title="이극용">이극용</a>은 <a href="/wiki/899%EB%85%84" title="899년">899년</a> 맹천에게 (재통합된) 소의군 번진의 지휘권을 되돌려줬다.<sup class="reference" id="cite_ref-ZZTJ261_9-0"><a href="#cite_note-ZZTJ261-9">[9]</a></sup> 하지만 <a href="/wiki/901%EB%85%84" title="901년">901년</a> 소의군이 이극용의 <a class="mw-redirect" href="/wiki/%EC%88%99%EC%A0%81" title="숙적">숙적</a> 선무군 절도사(宣武軍節度使, <a class="mw-redirect" href="/wiki/%EB%B3%B8%EB%B6%80" title="본부">본부</a>는 지금의 <a href="/wiki/%ED%97%88%EB%82%9C_%EC%84%B1" title="허난 성">허난 성</a> <a href="/wiki/%EC%B9%B4%EC%9D%B4%ED%8E%91_%EC%8B%9C" title="카이펑 시">카이펑 시</a>) <a href="/wiki/%EC%A3%BC%EC%A0%84%EC%B6%A9" title="주전충">주전충</a>에게 <a href="/wiki/%EA%B3%B5%EA%B2%A9" title="공격">공격</a>받자, 맹천은 주전충에게 번진을 바치고 투항하였다.<sup class="reference" id="cite_ref-ZZTJ261_9-1"><a href="#cite_note-ZZTJ261-9">[9]</a></sup></p>
<h4><span class="mw-headline" id=".EC.B4.88.EC.B0.BD.EA.B8.B0.EC.9D.98_.EA.B2.BD.EB.A0.A5">초창기의 경력</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%A7%B9%EC%A7%80%EC%83%81&amp;action=edit&amp;section=4" title="부분 편집: 초창기의 경력">편집</a><span class="mw-editsection-bracket">]</span></span></h4>
<p>맹지상의 아버지 맹도는 이극용을 계속 섬겼으나, 하동군 내에서 두각을 드러내지는 못했다. 하지만, 맹지상의 재능에 감동한 이극용은 자신의 큰딸을 맹지상과 결혼시켜 주었다.<sup class="reference" id="cite_ref-SGCQ48_4-2"><a href="#cite_note-SGCQ48-4">[4]</a></sup><sup class="reference" id="cite_ref-NHFD64_10-0"><a href="#cite_note-NHFD64-10">[10]</a></sup><sup class="reference" id="cite_ref-GPFQ_6-1"><a href="#cite_note-GPFQ-6">[6]</a></sup> (맹지상의 여동생은 나중에 이극용의 다른 동생 <a class="new" href="/w/index.php?title=%EC%9D%B4%EA%B7%B9%EB%85%95&amp;action=edit&amp;redlink=1" title="이극녕 (없는 문서)">이극녕</a>과 결혼하였다.<sup class="reference" id="cite_ref-NHFD64_10-1"><a href="#cite_note-NHFD64-10">[10]</a></sup>) 그 후, 맹지상은 하동군 교련사(敎練使)가 되었다.<sup class="reference" id="cite_ref-NHFD64_10-2"><a href="#cite_note-NHFD64-10">[10]</a></sup></p>
