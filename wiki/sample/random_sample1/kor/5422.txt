<p><b>머리 강</b>(<span style="font-size: smaller;"><a href="/wiki/%EC%98%81%EC%96%B4" title="영어">영어</a>:</span> <span lang="en" xml:lang="en">Murray River</span>, <a href="/wiki/%EC%82%AC%EC%9A%B0%EC%8A%A4%EC%98%A4%EC%8A%A4%ED%8A%B8%EB%A0%88%EC%9D%BC%EB%A6%AC%EC%95%84_%EC%A3%BC" title="사우스오스트레일리아 주">사우스오스트레일리아 주</a>에서는 "River Murray")은 <a href="/wiki/%EC%98%A4%EC%8A%A4%ED%8A%B8%EB%A0%88%EC%9D%BC%EB%A6%AC%EC%95%84" title="오스트레일리아">오스트레일리아</a>에서 <a href="/wiki/%EA%B8%B8%EC%9D%B4%EC%88%9C_%EA%B0%95_%EB%AA%A9%EB%A1%9D" title="길이순 강 목록">가장 긴 강</a>으로 길이는 2,375 km(1,476 mi)이다.<sup class="reference" id="cite_ref-1"><a href="#cite_note-1">[1]</a></sup> 머리 강은 가장 큰 지류인 <a href="/wiki/%EB%8B%AC%EB%A7%81_%EA%B0%95" title="달링 강">달링 강</a>과 함께 <a href="/wiki/%EC%98%A4%EC%8A%A4%ED%8A%B8%EB%A0%88%EC%9D%BC%EB%A6%AC%EC%95%84" title="오스트레일리아">오스트레일리아</a> 대륙의 유일한 주요 하계망이라 할 수 있다.</p>
<p>호주 토착종인 <a class="mw-redirect" href="/wiki/%EB%A8%B8%EB%A0%88%EC%9D%B4_%EB%8C%80%EA%B5%AC" title="머레이 대구">머레이 대구</a>가 살고 있는 서식지이다.</p>
<h2><span class="mw-headline" id=".EC.9C.A0.EB.A1.9C">유로</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%A8%B8%EB%A6%AC_%EA%B0%95&amp;action=edit&amp;section=1" title="부분 편집: 유로">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>머리 강은 <a class="new" href="/w/index.php?title=%EC%98%A4%EC%8A%A4%ED%8A%B8%EB%A0%88%EC%9D%BC%EB%A6%AC%EC%95%84_%EC%95%8C%ED%94%84%EC%8A%A4_%EC%82%B0%EB%A7%A5&amp;action=edit&amp;redlink=1" title="오스트레일리아 알프스 산맥 (없는 문서)">오스트레일리아 알프스 산맥</a>에서 발원하여 <a href="/wiki/%EC%98%A4%EC%8A%A4%ED%8A%B8%EB%A0%88%EC%9D%BC%EB%A6%AC%EC%95%84" title="오스트레일리아">오스트레일리아</a>의 가장 높은 산들의 서쪽면을 따라 흘러간다. 머리 강의 대부분은 <a href="/wiki/%EC%98%A4%EC%8A%A4%ED%8A%B8%EB%A0%88%EC%9D%BC%EB%A6%AC%EC%95%84" title="오스트레일리아">오스트레일리아</a>의 내륙 평야에 위치한다. 내륙 평야를 구불구불 가로질러 북서쪽으로 흘러가서 <a href="/wiki/%EB%89%B4%EC%82%AC%EC%9A%B0%EC%8A%A4%EC%9B%A8%EC%9D%BC%EC%8A%A4_%EC%A3%BC" title="뉴사우스웨일스 주">뉴사우스웨일스 주</a>와 <a href="/wiki/%EB%B9%85%ED%86%A0%EB%A6%AC%EC%95%84_%EC%A3%BC" title="빅토리아 주">빅토리아 주</a>의 경계를 이룬다. 그런 후 하구 근처의 <a class="new" href="/w/index.php?title=%EC%95%8C%EB%A0%89%EC%82%B0%EB%93%9C%EB%A6%AC%EB%82%98_%ED%98%B8&amp;action=edit&amp;redlink=1" title="알렉산드리나 호 (없는 문서)">알렉산드리나 호</a>에 도달하기 전 약 500 km (310 mi) 지점에서 남쪽으로 방향을 바꾸어 <a href="/wiki/%EC%82%AC%EC%9A%B0%EC%8A%A4%EC%98%A4%EC%8A%A4%ED%8A%B8%EB%A0%88%EC%9D%BC%EB%A6%AC%EC%95%84_%EC%A3%BC" title="사우스오스트레일리아 주">사우스오스트레일리아 주</a>로 흘러들어간다. 중류에 이르러 북쪽에서 내려오는 주요 지류들과 합류한다. 하류의 650 km 구간은 <a href="/wiki/%EC%82%AC%EC%9A%B0%EC%8A%A4%EC%98%A4%EC%8A%A4%ED%8A%B8%EB%A0%88%EC%9D%BC%EB%A6%AC%EC%95%84_%EC%A3%BC" title="사우스오스트레일리아 주">사우스오스트레일리아 주</a>를 흐르면서 여러 개의 얕은 호수로 군데군데 단절된 협곡을 지난다. 애들레이드의 남쪽에 바다를 따라 길게 형성된 쿠롱 석호 지대를 지나 <a href="/wiki/%EA%B7%B8%EB%A0%88%EC%9D%B4%ED%8A%B8%EC%98%A4%EC%8A%A4%ED%8A%B8%EB%A0%88%EC%9D%BC%EB%A6%AC%EC%95%84_%EB%A7%8C" title="그레이트오스트레일리아 만">그레이트오스트레일리아 만</a>으로 흘러 나가 바다와 만난다.</p>
<p>머리 강은 <a class="new" href="/w/index.php?title=%EC%95%8C%EB%A0%89%EC%82%B0%EB%93%9C%EB%A6%AC%EB%82%98_%ED%98%B8&amp;action=edit&amp;redlink=1" title="알렉산드리나 호 (없는 문서)">알렉산드리나 호</a>를 비롯한 <a class="mw-redirect" href="/wiki/%EC%97%BC%EB%8F%84" title="염도">염도</a>가 변동하는 대여섯 개의 호수를 지나가며 <a class="new" href="/w/index.php?title=%EB%A8%B8%EB%A6%AC_%ED%95%98%EA%B5%AC&amp;action=edit&amp;redlink=1" title="머리 하구 (없는 문서)">머리 하구</a>를 통해 <a href="/wiki/%EC%9D%B8%EB%8F%84%EC%96%91" title="인도양">인도양</a>의 동남부로 빠져나가기 전에 <a class="new" href="/w/index.php?title=%EC%BF%A0%EB%A1%B1_%EA%B5%AD%EB%A6%BD%EA%B3%B5%EC%9B%90&amp;action=edit&amp;redlink=1" title="쿠롱 국립공원 (없는 문서)">쿠롱 국립공원</a>을 지나간다. 머리 강의 하구인 <a class="new" href="/w/index.php?title=%EB%A8%B8%EB%A6%AC_%ED%95%98%EA%B5%AC&amp;action=edit&amp;redlink=1" title="머리 하구 (없는 문서)">머리 하구</a>가 위치한 <a href="/wiki/%EC%9D%B8%EB%8F%84%EC%96%91" title="인도양">인도양</a> 동남부는 종종 오스트레일리아를 표시하는 지도에서 <a class="new" href="/w/index.php?title=%EA%B5%B4%EC%99%80_(%EC%82%AC%EC%9A%B0%EC%8A%A4%EC%98%A4%EC%8A%A4%ED%8A%B8%EB%A0%88%EC%9D%BC%EB%A6%AC%EC%95%84_%EC%A3%BC)&amp;action=edit&amp;redlink=1" title="굴와 (사우스오스트레일리아 주) (없는 문서)">굴와</a> 근처에서 <a href="/wiki/%EB%82%A8%EA%B7%B9%ED%95%B4" title="남극해">남극해</a>로 표시되기도 한다.<sup class="reference" id="cite_ref-MDBC_mouth_2-0"><a href="#cite_note-MDBC_mouth-2">[2]</a></sup></p>
<h2><span class="mw-headline" id=".EC.B9.98.EC.88.98">치수</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%A8%B8%EB%A6%AC_%EA%B0%95&amp;action=edit&amp;section=2" title="부분 편집: 치수">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>머리 강의 유량 변동은 매우 심한 편이어서 건기에는 완전히 말라 바닥을 드러내기도 한다. 그러나 최근에는 <a class="new" href="/w/index.php?title=%EC%8A%A4%EB%85%B8%EC%9D%B4_%EC%82%B0%EB%A7%A5&amp;action=edit&amp;redlink=1" title="스노이 산맥 (없는 문서)">스노이 산맥</a>의 수력발전 계획의 일환으로 <a class="new" href="/w/index.php?title=%EC%8A%A4%EB%85%B8%EC%9D%B4_%EA%B0%95&amp;action=edit&amp;redlink=1" title="스노이 강 (없는 문서)">스노이 강</a>에서 머리 강 상류로 유로가 바뀌어 흐르게 하는 유역 변경식 발전소를 건설하여 머리 강의 유량이 다소 증가하였다. 2008년의 측정값을 기준으로 할 때, 머리 강에는 자연 흐름의 상태일 때의 38% 정도의 유량만이 흐르고 있다. 머리 강은 <a href="/wiki/%EC%98%A4%EC%8A%A4%ED%8A%B8%EB%A0%88%EC%9D%BC%EB%A6%AC%EC%95%84" title="오스트레일리아">오스트레일리아</a>에서 관개 및 농업화가 가장 많이 이루어진 강들 중의 하나이다.</p>
