<p><b>멍</b>(<span lang="en" xml:lang="en">bruise, contusion</span>)은 <a href="/wiki/%EC%A1%B0%EC%A7%81_(%EC%83%9D%EB%AC%BC%ED%95%99)" title="조직 (생물학)">조직</a>의 <a class="new" href="/w/index.php?title=%ED%98%88%EC%A2%85&amp;action=edit&amp;redlink=1" title="혈종 (없는 문서)">혈종</a>의 일종이다.<sup class="reference" id="cite_ref-1"><a href="#cite_note-1">[1]</a></sup> 여기서 <a href="/wiki/%EB%AA%A8%EC%84%B8%ED%98%88%EA%B4%80" title="모세혈관">모세혈관</a>과 간혹 <a class="new" href="/w/index.php?title=%EC%84%B8%EC%A0%95%EB%A7%A5&amp;action=edit&amp;redlink=1" title="세정맥 (없는 문서)">세정맥</a>은 <a href="/wiki/%EC%99%B8%EC%83%81" title="외상">외상</a>에 의해 상해를 입게 되면, 피가 스며들어 <a href="/wiki/%EC%B6%9C%ED%98%88" title="출혈">출혈</a>이 일어나거나 주변 <a class="new" href="/w/index.php?title=%EC%82%AC%EC%9D%B4_%EC%84%B8%ED%8F%AC&amp;action=edit&amp;redlink=1" title="사이 세포 (없는 문서)">사이 세포</a> 조직으로 내출혈이 일어날 수 있다. 압박에 의하지 않더라도, <a href="/wiki/%ED%94%BC%EB%B6%80" title="피부">피부</a>, <a class="new" href="/w/index.php?title=%ED%94%BC%ED%95%98_%EC%A1%B0%EC%A7%81&amp;action=edit&amp;redlink=1" title="피하 조직 (없는 문서)">피하 조직</a>, <a href="/wiki/%EA%B7%BC%EC%9C%A1" title="근육">근육</a>, <a href="/wiki/%EB%BC%88" title="뼈">뼈</a> 수준에서 모세관을 통해 멍이 발생할 수 있다.<sup class="reference" id="cite_ref-2"><a href="#cite_note-2">[2]</a></sup><sup class="reference" id="cite_ref-3"><a href="#cite_note-3">[3]</a></sup></p>

<h2><span class="mw-headline" id=".EB.B3.91.EC.9D.B8">병인</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%A9%8D&amp;action=edit&amp;section=1" title="부분 편집: 병인">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>멍은 <a href="/wiki/%ED%98%88%EC%86%8C%ED%8C%90" title="혈소판">혈소판</a>이나 <a class="new" href="/w/index.php?title=%ED%98%88%EC%95%B5%EC%9D%91%EA%B3%A0%EC%9D%B8%EC%9E%90&amp;action=edit&amp;redlink=1" title="혈앵응고인자 (없는 문서)">혈앵응고인자</a> 관련 질병이 있는 환자에게서 관찰될 수 있다. 설명이 되지 않는 멍의 경우 <a href="/wiki/%EC%95%84%EB%8F%99_%ED%95%99%EB%8C%80" title="아동 학대">아동 학대</a>, <a href="/wiki/%EA%B0%80%EC%A0%95%ED%8F%AD%EB%A0%A5" title="가정폭력">가정폭력</a>, 또는 <a href="/wiki/%EB%B0%B1%ED%98%88%EB%B3%91" title="백혈병">백혈병</a>이나 <a class="new" href="/w/index.php?title=%EC%88%98%EB%A7%89%EC%97%BC%EA%B7%A0&amp;action=edit&amp;redlink=1" title="수막염균 (없는 문서)">수막염균</a> 감염 문제의 증상일 수 있다.</p>
