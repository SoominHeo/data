<div class="mw-body-content" id="bodyContent">

<p><b>메달</b>(medal)은 <a class="mw-redirect" href="/wiki/%EB%8F%99%EC%A0%84" title="동전">동전</a> 모양의 조각품으로 <a href="/wiki/%ED%9B%88%EC%9E%A5" title="훈장">훈장</a>, <a class="mw-redirect" href="/wiki/%EC%B4%88%EC%83%81%ED%99%94" title="초상화">초상화</a>, 예술적 표현 등이 금속 등의 물질에 새겨져 있다. 메달은 운동, 군사, 과학, 학업 등의 성취를 인식시키는 한 형태로 개인이나 단체에게 수여할 수 있다. 메달은 또한 개인이나 사건, 예술 표현 작품을 기념하기 위해 만들기도 한다. 종교의 믿음으로 제공되는 <a class="new" href="/w/index.php?title=%EA%B8%B0%EC%97%AC_%EB%A9%94%EB%8B%AC&amp;action=edit&amp;redlink=1" title="기여 메달 (없는 문서)">기여 메달</a>도 존재한다.</p>
<h2><span class="mw-headline" id=".EA.B2.BD.EC.9F.81_.EB.A9.94.EB.8B.AC">경쟁 메달</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%A9%94%EB%8B%AC&amp;action=edit&amp;section=1" title="부분 편집: 경쟁 메달">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>메달은 역사적으로 다양한 종류의 경쟁 활동을 통해, 특히 운동 부문에서 상으로 수여한다.</p>
<p>전통적으로 메달은 다음과 같은 금속으로 만든다:</p>
<ol>
<li><a href="/wiki/%EA%B8%88%EB%A9%94%EB%8B%AC" title="금메달">금메달</a>: <a href="/wiki/%EA%B8%88" title="금">금</a> (또는 다른 노란 메달, 이를테면 <a href="/wiki/%ED%99%A9%EB%8F%99" title="황동">황동</a>)</li>
<li><a href="/wiki/%EC%9D%80%EB%A9%94%EB%8B%AC" title="은메달">은메달</a>: <a href="/wiki/%EC%9D%80" title="은">은</a> (또는 다른 회색 메달, 이를테면 <a href="/wiki/%EA%B0%95%EC%B2%A0" title="강철">강철</a>)</li>
<li><a href="/wiki/%EB%8F%99%EB%A9%94%EB%8B%AC" title="동메달">동메달</a>: <a href="/wiki/%EC%B2%AD%EB%8F%99" title="청동">동</a></li>
</ol>
<p>이 금속들은 <a href="/wiki/%EA%B7%B8%EB%A6%AC%EC%8A%A4_%EC%8B%A0%ED%99%94" title="그리스 신화">그리스 신화</a>에서 <a href="/wiki/%ED%97%A4%EC%8B%9C%EC%98%A4%EB%8F%84%EC%8A%A4" title="헤시오도스">헤시오도스</a>가 설명한 세 가지 <a href="/wiki/%EC%9D%B8%EA%B0%84%EC%9D%98_%EC%8B%9C%EB%8C%80" title="인간의 시대">인간의 시대</a>를 나타내는 것인데, 이 세 가지 시대는 인간이 신과 함께 살았던 <a class="new" href="/w/index.php?title=%EA%B8%88%EC%9D%98_%EC%8B%9C%EB%8C%80&amp;action=edit&amp;redlink=1" title="금의 시대 (없는 문서)">금의 시대</a>, 젊음이 천년까지 지속되는 <a class="new" href="/w/index.php?title=%EC%9D%80%EC%9D%98_%EC%8B%9C%EB%8C%80&amp;action=edit&amp;redlink=1" title="은의 시대 (없는 문서)">은의 시대</a>, 그리고 영웅의 시대인 <a class="new" href="/w/index.php?title=%EC%B2%AD%EB%8F%99_%EC%8B%9C%EB%8C%80&amp;action=edit&amp;redlink=1" title="청동 시대 (없는 문서)">청동 시대</a>이다. (현재 시대는 <a class="new" href="/w/index.php?title=%EC%B2%A0%EC%9D%98_%EC%8B%9C%EB%8C%80&amp;action=edit&amp;redlink=1" title="철의 시대 (없는 문서)">철의 시대</a>) 금에서 청동으로 갈수록 <a href="/wiki/%EB%B6%80%EC%8B%9D" title="부식">부식</a>하기 쉬운 경향을 보인다는 것이 특징이다.</p>
<p>이 메달의 기준은 <a href="/wiki/1904%EB%85%84_%ED%95%98%EA%B3%84_%EC%98%AC%EB%A6%BC%ED%94%BD" title="1904년 하계 올림픽">1904년 하계 올림픽</a>에 채택되었다. 1896년 경기에서는 승자에게 은메달을 수여하였고, 2위에게는 동메달을 주었다. 1900년에는 메달 대신에 다른 상을 수여하였다.</p>
