<p><b>메달 오브 아너: 언더그라운드</b>(<span lang="en" xml:lang="en">Medal of Honor: Underground</span>)는 <a class="mw-redirect" href="/wiki/EA_%EB%A1%9C%EC%8A%A4%EC%95%A4%EC%A0%A4%EB%A0%88%EC%8A%A4" title="EA 로스앤젤레스">드림웍스 인터렉티브</a>가 제작한 <a href="/wiki/%EB%A9%94%EB%8B%AC_%EC%98%A4%EB%B8%8C_%EC%95%84%EB%84%88_%EC%8B%9C%EB%A6%AC%EC%A6%88" title="메달 오브 아너 시리즈">메달 오브 아너 시리즈</a>의 두 번째 게임이다. 프랑스판의 명칭은 "<span lang="fr" xml:lang="fr">Medal of Honor : Résistance</span>"이다.</p>
<h2><span class="mw-headline" id=".EA.B0.9C.EC.9A.94">개요</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%A9%94%EB%8B%AC_%EC%98%A4%EB%B8%8C_%EC%95%84%EB%84%88:_%EC%96%B8%EB%8D%94%EA%B7%B8%EB%9D%BC%EC%9A%B4%EB%93%9C&amp;action=edit&amp;section=1" title="부분 편집: 개요">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>플레이어는 전작에 등장하였던 독일 <a href="/wiki/%EB%82%98%EC%B9%98%EC%A6%98" title="나치즘">나치즘</a>에 저항하는 <a href="/wiki/%ED%94%84%EB%9E%91%EC%8A%A4" title="프랑스">프랑스</a> <a href="/wiki/%EB%A0%88%EC%A7%80%EC%8A%A4%ED%83%95%EC%8A%A4" title="레지스탕스">레지스탕스</a>의 일원인 <i>마농 베티스테</i>의 역할을 맡아 <a href="/wiki/%EC%A0%84%EB%9E%B5%EC%82%AC%EB%AC%B4%EA%B5%AD" title="전략사무국">전략사무국</a>에 소속되어 프랑스에서의 저항 운동, <a href="/wiki/%EB%B6%81%EC%95%84%ED%94%84%EB%A6%AC%EC%B9%B4" title="북아프리카">북아프리카</a>의 <a href="/wiki/%EC%B9%B4%EC%82%AC%EB%B8%94%EB%9E%91%EC%B9%B4" title="카사블랑카">카사블랑카</a>, <a href="/wiki/%EA%B7%B8%EB%A6%AC%EC%8A%A4" title="그리스">그리스</a>, <a href="/wiki/%EC%9D%B4%ED%83%88%EB%A6%AC%EC%95%84" title="이탈리아">이탈리아</a>의 <a class="new" href="/w/index.php?title=%EB%AA%AC%ED%85%8C_%EC%B9%B4%EC%A7%80%EB%85%B8&amp;action=edit&amp;redlink=1" title="몬테 카지노 (없는 문서)">몬테 카지노</a>, <a href="/wiki/%EB%8F%85%EC%9D%BC" title="독일">독일</a>에서 잠입하게 된다.</p>
