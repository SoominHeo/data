<p><b>메르세데스-벤츠 GLA 클래스</b>(Mercedes-Benz GLA-Class)는 <a href="/wiki/%EB%8F%85%EC%9D%BC" title="독일">독일</a>의 <a href="/wiki/%EC%9E%90%EB%8F%99%EC%B0%A8" title="자동차">자동차</a> 회사인 <a href="/wiki/%EB%A9%94%EB%A5%B4%EC%84%B8%EB%8D%B0%EC%8A%A4-%EB%B2%A4%EC%B8%A0" title="메르세데스-벤츠">메르세데스-벤츠</a>가 생산, 판매하는 소형 크로스오버 <a href="/wiki/%EC%8A%A4%ED%8F%AC%EC%B8%A0_%EC%9C%A0%ED%8B%B8%EB%A6%AC%ED%8B%B0_%EC%9E%90%EB%8F%99%EC%B0%A8" title="스포츠 유틸리티 자동차">스포츠 유틸리티 자동차</a>이다. <a href="/wiki/%EB%A9%94%EB%A5%B4%EC%84%B8%EB%8D%B0%EC%8A%A4-%EB%B2%A4%EC%B8%A0_A_%ED%81%B4%EB%9E%98%EC%8A%A4" title="메르세데스-벤츠 A 클래스">A 클래스</a>의 플랫폼을 기반으로 만들어졌다. <a href="/wiki/2013%EB%85%84_4%EC%9B%94" title="2013년 4월">2013년 4월</a>에 <a class="new" href="/w/index.php?title=%EC%98%A4%ED%86%A0_%EC%83%81%ED%95%98%EC%9D%B4&amp;action=edit&amp;redlink=1" title="오토 상하이 (없는 문서)">오토 상하이</a>를 통해 <a href="/wiki/%EC%BD%98%EC%85%89%ED%8A%B8_%EC%B9%B4" title="콘셉트 카">컨셉트 카</a>가 공개되었고, 동년 <a href="/wiki/9%EC%9B%94" title="9월">9월</a>에 <a href="/wiki/%ED%94%84%EB%9E%91%ED%81%AC%ED%91%B8%EB%A5%B4%ED%8A%B8_%EA%B5%AD%EC%A0%9C_%EB%AA%A8%ED%84%B0%EC%87%BC" title="프랑크푸르트 국제 모터쇼">프랑크푸르트 국제 모터쇼</a>에서 양산형이 공개되었다. 이후 <a href="/wiki/12%EC%9B%94" title="12월">12월</a>에 정식 출시되었다. 고성능 사양인 AMG GLA 45는 <a href="/wiki/2014%EB%85%84_1%EC%9B%94" title="2014년 1월">2014년 1월</a>에 <a href="/wiki/%EB%B6%81%EB%AF%B8_%EA%B5%AD%EC%A0%9C_%EC%98%A4%ED%86%A0%EC%87%BC" title="북미 국제 오토쇼">북미 국제 오토쇼</a>에서 공개되었고, 같은 해 <a href="/wiki/%EC%97%AC%EB%A6%84" title="여름">여름</a>에 출시되었다. <a href="/wiki/9%EC%9B%94" title="9월">9월</a>에 <a href="/wiki/%EB%8C%80%ED%95%9C%EB%AF%BC%EA%B5%AD" title="대한민국">대한민국</a>에서도 수입·판매를 개시하였다. <a href="/wiki/%EC%9D%BC%EB%B3%B8" title="일본">일본</a>에서는 <a href="/wiki/%EB%8B%8C%ED%85%90%EB%8F%84" title="닌텐도">닌텐도</a>와 협력한 적이 있으며, <a href="/wiki/%EB%A7%88%EB%A6%AC%EC%98%A4" title="마리오">슈퍼 마리오</a>를 모델로 독특한 광고가 방영되어 화제가 되기도 했다. <a href="/wiki/2017%EB%85%84" title="2017년">2017년</a>에 페이스 리프트를 거쳐 범퍼에 안개등이 추가되었다.</p>
