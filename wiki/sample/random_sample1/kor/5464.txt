<p><b>메리 터스티그 마허 플랜트</b>(Mary Tustegge Meagher Plant, <a href="/wiki/1964%EB%85%84" title="1964년">1964년</a> <a href="/wiki/10%EC%9B%94_27%EC%9D%BC" title="10월 27일">10월 27일</a> ~ )는 <a href="/wiki/%EB%AF%B8%EA%B5%AD" title="미국">미국</a>의 <a href="/wiki/%EC%88%98%EC%98%81" title="수영">수영</a> 선수이자, 전 세계 기록 보유자이다. <a href="/wiki/1981%EB%85%84" title="1981년">1981년</a> 자신의 생존 세계 기록들 - 100m(57.93초)와 200m <a href="/wiki/%EC%A0%91%EC%98%81" title="접영">접영</a>(2분 05.96초)를 능가하여, 이 기록들이 18년과 19년 동안 존경할 만한 기록들로 서있었다.</p>
<p><a href="/wiki/%EC%BC%84%ED%84%B0%ED%82%A4_%EC%A3%BC" title="켄터키 주">켄터키 주</a> <a href="/wiki/%EB%A3%A8%EC%9D%B4%EB%B9%8C" title="루이빌">루이빌</a>에서 태어난 마허는 어린 나이에 수영을 시작하여, 14세의 나이로 <a href="/wiki/1979%EB%85%84" title="1979년">1979년</a> <a href="/wiki/%ED%8C%AC%EC%95%84%EB%A9%94%EB%A6%AC%EC%B9%B8_%EA%B2%8C%EC%9E%84" title="팬아메리칸 게임">팬아메리칸 게임</a>에서 자신의 첫 접영 세계 기록들을 세웠다.</p>
<p><a class="mw-redirect" href="/wiki/1980%EB%85%84_%EB%AA%A8%EC%8A%A4%ED%81%AC%EB%B0%94_%EC%98%AC%EB%A6%BC%ED%94%BD" title="1980년 모스크바 올림픽">1980년 모스크바 올림픽</a>에서 메달 획득을 기대하였지만, 미국의 불참으로 기회를 놓쳤다.</p>
<p><a href="/wiki/1981%EB%85%84" title="1981년">1981년</a> <a href="/wiki/%EC%9C%84%EC%8A%A4%EC%BD%98%EC%8B%A0_%EC%A3%BC" title="위스콘신 주">위스콘신 주</a> 브라운 디어에서 열린 국내 선수권 대회에서 배영의 양종목에서 세계 기록들을 세웠다. <a href="/wiki/1999%EB%85%84" title="1999년">1999년</a> <a href="/wiki/%EC%A0%9C%EB%8B%88_%ED%86%B0%EC%8A%A8" title="제니 톰슨">제니 톰슨</a>이 100m 기록을 낮추었고, 이듬해 <a href="/wiki/%EC%88%98%EC%A7%80_%EC%98%A4%EB%8B%90" title="수지 오닐">수지 오닐</a>이 200m 기록을 세우면서 위의 기록들이 갱신되었다.</p>
<p>그해와 <a href="/wiki/1985%EB%85%84" title="1985년">1985년</a> 스위밍 월드 잡지에 의하여 "올해의 여성 수영 선수"로 선정되었다.</p>
<p><a href="/wiki/%EC%BA%98%EB%A6%AC%ED%8F%AC%EB%8B%88%EC%95%84_%EB%8C%80%ED%95%99%EA%B5%90_%EB%B2%84%ED%81%B4%EB%A6%AC" title="캘리포니아 대학교 버클리">캘리포니아 대학교 버클리</a>에서 선수로 활약하였으며, <a href="/wiki/1987%EB%85%84" title="1987년">1987년</a> <a class="mw-redirect" href="/wiki/%EC%82%AC%ED%9A%8C_%EA%B3%BC%ED%95%99" title="사회 과학">사회 과학</a>에서 문학사를 취득하면서 졸업하였다.</p>
<p><a class="mw-redirect" href="/wiki/1984%EB%85%84_%EB%A1%9C%EC%8A%A4%EC%95%A4%EC%A0%A4%EB%A0%88%EC%8A%A4_%EC%98%AC%EB%A6%BC%ED%94%BD" title="1984년 로스앤젤레스 올림픽">1984년 로스앤젤레스 올림픽</a>에서 접영의 양종목과 400m 혼계영에서 3개의 <a href="/wiki/%EA%B8%88%EB%A9%94%EB%8B%AC" title="금메달">금메달</a>을 획득하였다. 4년 후, <a href="/wiki/1988%EB%85%84_%ED%95%98%EA%B3%84_%EC%98%AC%EB%A6%BC%ED%94%BD" title="1988년 하계 올림픽">서울 올림픽</a>에서 200m 접영 <a href="/wiki/%EB%8F%99%EB%A9%94%EB%8B%AC" title="동메달">동메달</a> 획득을 마지막으로 수영에서 은퇴하였다. 그녀는 24개의 국내 타이틀을 우승하였다.</p>
<p>루이빌에는 그녀의 이름을 딴 수영장이 있다.</p>
