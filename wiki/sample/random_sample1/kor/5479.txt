<p><b>메소포타미아 속주</b>(<span style="font-size: smaller;"><a href="/wiki/%EB%9D%BC%ED%8B%B4%EC%96%B4" title="라틴어">라틴어</a>:</span> <span lang="la" xml:lang="la">Provincia Macedoniae</span>)는 현재의 <a href="/wiki/%EC%9D%B4%EB%9D%BC%ED%81%AC" title="이라크">이라크</a>와 <a href="/wiki/%ED%84%B0%ED%82%A4" title="터키">터키</a>, <a href="/wiki/%EC%8B%9C%EB%A6%AC%EC%95%84" title="시리아">시리아</a>, <a href="/wiki/%EC%BF%A0%EC%9B%A8%EC%9D%B4%ED%8A%B8" title="쿠웨이트">쿠웨이트</a> 지방을 일컫는 고대의 지방 이름이다. 주도는 <a class="new" href="/w/index.php?title=%EC%95%84%EB%AF%B8%EB%8B%A4&amp;action=edit&amp;redlink=1" title="아미다 (없는 문서)">아미다</a>(현 터키 디야르바크르), <a class="new" href="/w/index.php?title=%EB%8B%88%EC%8B%9C%EB%B9%84%EC%8A%A4&amp;action=edit&amp;redlink=1" title="니시비스 (없는 문서)">니시비스</a>(현 터키 누사이빈)이다.</p>
<p>북쪽으로 <a href="/wiki/%EC%95%84%EB%A5%B4%EB%A9%94%EB%8B%88%EC%95%84" title="아르메니아">아르메니아</a>, 서쪽으로 <a href="/wiki/%EC%8B%9C%EB%A6%AC%EC%95%84_%EC%86%8D%EC%A3%BC" title="시리아 속주">시리아 속주</a>, 남쪽으로 아라비아 반도, 동쪽으로 <a class="new" href="/w/index.php?title=%EC%95%84%EC%8B%9C%EB%A6%AC%EC%95%84_%EC%86%8D%EC%A3%BC&amp;action=edit&amp;redlink=1" title="아시리아 속주 (없는 문서)">아시리아 속주</a>, 페르시아에 접해 있다. <a class="new" href="/w/index.php?title=%EA%B8%B0%EC%9B%90%ED%9B%84_116%EB%85%84&amp;action=edit&amp;redlink=1" title="기원후 116년 (없는 문서)">기원후 116년</a>경 <a href="/wiki/%EB%A1%9C%EB%A7%88-%ED%8C%8C%EB%A5%B4%ED%8B%B0%EC%95%84_%EC%A0%84%EC%9F%81" title="로마-파르티아 전쟁">로마-파르티아 전쟁</a> 이후 <a href="/wiki/%ED%8A%B8%EB%9D%BC%EC%95%BC%EB%88%84%EC%8A%A4" title="트라야누스">트라야누스</a>가 정복했고, 로마의 <a class="mw-redirect" href="/wiki/%EB%A1%9C%EB%A7%88_%EC%86%8D%EC%A3%BC" title="로마 속주">속주</a>가 되었다. 이곳은 118년 <a href="/wiki/%ED%95%98%EB%93%9C%EB%A6%AC%EC%95%84%EB%88%84%EC%8A%A4" title="하드리아누스">하드리아누스</a> 때 폐지된다. 이후 메소포타미아 북부 속주의 형태로 재설립되었다.</p>
<p>후기 로마 제국의 지정학적 상황에 있어 북부 메소포타미아는 페르시아에 대한 영향력을 끼칠 수 있는 귀중한 발판이 될 수 있었다. 메소포타미아 속주는 <a href="/wiki/637%EB%85%84" title="637년">637년</a>까지 동로마 제국의 일부로 남아 있었다가, 637년 <a href="/wiki/%EC%A0%95%ED%86%B5_%EC%B9%BC%EB%A6%AC%ED%8C%8C%EC%A1%B0" title="정통 칼리파조">정통 칼리파조</a>와의 전쟁으로 상실하게 된다.</p>
