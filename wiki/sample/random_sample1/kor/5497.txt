<p><b>메이신 고속도로</b>(<span style="font-size: smaller;"><a href="/wiki/%EC%9D%BC%EB%B3%B8%EC%96%B4" title="일본어">일본어</a>:</span> <span lang="ja" xml:lang="ja">名神高速道路(めいしんこうそくどうろ)</span> <small>메이신코오소쿠도오로<sup>[<a href="/wiki/%EC%9C%84%ED%82%A4%EB%B0%B1%EA%B3%BC:%EC%9D%BC%EB%B3%B8%EC%96%B4%EC%9D%98_%ED%95%9C%EA%B8%80_%ED%91%9C%EA%B8%B0" title="위키백과:일본어의 한글 표기">*</a>]</sup></small>, 명신고속도로)는 <a href="/wiki/%EC%95%84%EC%9D%B4%EC%B9%98_%ED%98%84" title="아이치 현">아이치 현</a> <a href="/wiki/%EA%B3%A0%EB%A7%88%ED%82%A4_%EC%8B%9C" title="고마키 시">고마키 시</a> <a class="new" href="/w/index.php?title=%EA%B3%A0%EB%A7%88%ED%82%A4_%EB%82%98%EB%93%A4%EB%AA%A9&amp;action=edit&amp;redlink=1" title="고마키 나들목 (없는 문서)">고마키 나들목</a>과 <a href="/wiki/%ED%9A%A8%EA%B3%A0_%ED%98%84" title="효고 현">효고 현</a> <a href="/wiki/%EB%8B%88%EC%8B%9C%EB%85%B8%EB%AF%B8%EC%95%BC_%EC%8B%9C" title="니시노미야 시">니시노미야 시</a> <a class="new" href="/w/index.php?title=%EB%8B%88%EC%8B%9C%EB%85%B8%EB%AF%B8%EC%95%BC_%EB%82%98%EB%93%A4%EB%AA%A9&amp;action=edit&amp;redlink=1" title="니시노미야 나들목 (없는 문서)">니시노미야 나들목</a>을 잇는 <a href="/wiki/%EA%B3%A0%EC%86%8D%EB%8F%84%EB%A1%9C" title="고속도로">고속도로</a> (<a href="/wiki/%EC%9D%BC%EB%B3%B8%EC%9D%98_%EA%B3%A0%EC%86%8D%EB%8F%84%EB%A1%9C" title="일본의 고속도로">고속 자동차 국도</a>)이다. 일본에서 최초로 건설된 고속도로이며, 고마키 나들목 ~ <a class="new" href="/w/index.php?title=%EC%8A%A4%EC%9D%B4%ED%83%80_%EB%B6%84%EA%B8%B0%EC%A0%90&amp;action=edit&amp;redlink=1" title="스이타 분기점 (없는 문서)">스이타 분기점</a> 구간은 <a href="/wiki/%EC%95%84%EC%8B%9C%EC%95%88_%ED%95%98%EC%9D%B4%EC%9B%A8%EC%9D%B4_1%ED%98%B8%EC%84%A0" title="아시안 하이웨이 1호선">아시아 고속도로 1호선</a>의 일부이다.</p>
<p><a class="new" href="/w/index.php?title=%EC%9A%94%EC%B9%B4%EC%9D%B4%EC%B9%98_%EB%82%98%EB%93%A4%EB%AA%A9&amp;action=edit&amp;redlink=1" title="요카이치 나들목 (없는 문서)">요카이치 나들목</a>에서부터 동쪽은 <a href="/wiki/%EC%A4%91%EC%9D%BC%EB%B3%B8_%EA%B3%A0%EC%86%8D%EB%8F%84%EB%A1%9C" title="중일본 고속도로">중일본 고속도로</a>가, 서쪽을 <a href="/wiki/%EC%84%9C%EC%9D%BC%EB%B3%B8_%EA%B3%A0%EC%86%8D%EB%8F%84%EB%A1%9C" title="서일본 고속도로">서일본 고속도로</a>가 관리하고 있다.</p>
