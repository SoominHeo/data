<p><b>메이저 리그 게이밍</b>(Major League Gaming, <b>MLG</b>)은 2003년 부터 열리고있는 <a class="mw-redirect" href="/wiki/%EB%B6%81%EB%AF%B8" title="북미">북미</a> <a href="/wiki/E%EC%8A%A4%ED%8F%AC%EC%B8%A0" title="E스포츠">e스포츠</a> 게임 대회이다.</p>
<p>초창기인 2004년부터 2009년까지는 <a class="mw-redirect" href="/wiki/%EC%BD%98%EC%86%94_%EA%B2%8C%EC%9E%84%EA%B8%B0" title="콘솔 게임기">콘솔 게임기</a> 종목으로 운영되었으며 <a class="mw-disambig" href="/wiki/FPS" title="FPS">FPS</a>와 <a href="/wiki/%EB%8C%80%EC%A0%84_%EA%B2%A9%ED%88%AC_%EA%B2%8C%EC%9E%84" title="대전 격투 게임">대전 격투 게임</a> 중심으로 리그를 열었으나, 2010년 부터는 <a href="/wiki/%EC%8A%A4%ED%83%80%ED%81%AC%EB%9E%98%ED%94%84%ED%8A%B8_II" title="스타크래프트 II">스타크래프트 II</a>, <a href="/wiki/%EB%A6%AC%EA%B7%B8_%EC%98%A4%EB%B8%8C_%EB%A0%88%EC%A0%84%EB%93%9C" title="리그 오브 레전드">리그 오브 레전드</a>, <a href="/wiki/%EC%B2%A0%EA%B6%8C_6" title="철권 6">철권 6</a> 등을 새로 포함시키면서 규모가 커지게 되었다.</p>
<p>선수 선발은 국가 제한 없이 온오프라인 예선을 통하거나 초청으로 이루어지며, 장소는 시즌별로 미국 각지를 돌아가며 열린다. 방송은 MLG 홈페이지와 <a class="mw-disambig" href="/wiki/CBS" title="CBS">CBS</a>, <a class="mw-redirect" href="/wiki/%ED%8A%B8%EC%9C%84%EC%B9%98TV" title="트위치TV">트위치TV</a>와의 협력 관계를 통해 이루어진다.<sup class="reference" id="cite_ref-1"><a href="#cite_note-1">[1]</a></sup></p>
