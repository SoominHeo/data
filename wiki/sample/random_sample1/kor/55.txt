<p><b>마케마케</b>(<a class="mw-redirect" href="/wiki/%EC%86%8C%ED%96%89%EC%84%B1%EB%AA%85" title="소행성명">소행성명</a> <b>136472 Makemake</b>)는 <a href="/wiki/%ED%83%9C%EC%96%91%EA%B3%84" title="태양계">태양계</a>의 <a href="/wiki/%EC%99%9C%ED%96%89%EC%84%B1" title="왜행성">왜행성</a>들 중 세 번째로 크며 <a class="mw-redirect" href="/wiki/%ED%81%90%EB%B9%84%EC%9B%90%EC%A1%B1" title="큐비원족">큐비원족</a> 중 가장 큰 두 개 천체 중 하나이다. 지름은 대략 <a href="/wiki/%EB%AA%85%EC%99%95%EC%84%B1" title="명왕성">명왕성</a>의 3분의 2 수준이다.<sup class="reference" id="cite_ref-1"><a href="#cite_note-1">[1]</a></sup> 현재 마케마케는 태양에서 52 <a href="/wiki/%EC%B2%9C%EB%AC%B8%EB%8B%A8%EC%9C%84" title="천문단위">천문단위</a> 떨어져 있는데 이는 <a class="mw-redirect" href="/wiki/%EC%B9%B4%EC%9D%B4%ED%8D%BC%EB%8C%80" title="카이퍼대">카이퍼대</a> 외곽지대보다 약간 먼 정도 거리이다. 마케마케는 천천히 태양에 가까워지고 있으며, <a class="mw-redirect" href="/wiki/%EA%B7%BC%EC%9D%BC%EC%A0%90" title="근일점">근일점</a>에서의 <a class="mw-redirect" href="/wiki/%EA%B6%A4%EB%8F%84%EC%9E%A5%EB%B0%98%EA%B2%BD" title="궤도장반경">궤도장반경</a>은 약 46 천문단위로 큐비원족 영역 내로 진입한다.</p>
<p>이 천체의 원래 이름은 <b>2005 FY<sub>9</sub></b>로 이후 <b>136472</b>가 붙었다. <a href="/wiki/2005%EB%85%84" title="2005년">2005년</a> <a href="/wiki/3%EC%9B%94_31%EC%9D%BC" title="3월 31일">3월 31일</a> 마이클 브라운 탐사팀에 의해 발견되었다. 발견 사실은 <a href="/wiki/2005%EB%85%84" title="2005년">2005년</a> <a href="/wiki/7%EC%9B%94_29%EC%9D%BC" title="7월 29일">7월 29일</a> 공표되었다. <a href="/wiki/2008%EB%85%84" title="2008년">2008년</a> <a href="/wiki/6%EC%9B%94_11%EC%9D%BC" title="6월 11일">6월 11일</a> <a class="mw-redirect" href="/wiki/%EA%B5%AD%EC%A0%9C_%EC%B2%9C%EB%AC%B8_%EC%97%B0%EB%A7%B9" title="국제 천문 연맹">국제 천문 연맹</a>은 마케마케를 잠재적 <a href="/wiki/%ED%94%8C%EB%A3%A8%ED%86%A0%EC%9D%B4%EB%93%9C" title="플루토이드">플루토이드</a>(<a href="/wiki/%ED%95%B4%EC%99%95%EC%84%B1" title="해왕성">해왕성</a> 너머에 있는 왜행성들로, 마케마케는 <a href="/wiki/%EB%AA%85%EC%99%95%EC%84%B1" title="명왕성">명왕성</a> 및 <a href="/wiki/%EC%97%90%EB%A6%AC%EC%8A%A4_(%EC%99%9C%ED%96%89%EC%84%B1)" title="에리스 (왜행성)">에리스</a>와 함께 여기에 속한다) 목록에 포함시켰다. 마케마케는 <a href="/wiki/2008%EB%85%84" title="2008년">2008년</a> <a href="/wiki/7%EC%9B%94_11%EC%9D%BC" title="7월 11일">7월 11일</a> 공식적으로 명왕성형 천체의 지위를 얻게 되었다.<sup class="reference" id="cite_ref-name_2-0"><a href="#cite_note-name-2">[2]</a></sup><sup class="reference" id="cite_ref-Tancredi_3-0"><a href="#cite_note-Tancredi-3">[3]</a></sup><sup class="reference" id="cite_ref-Brown_4-0"><a href="#cite_note-Brown-4">[4]</a></sup></p>

<h2><span class="mw-headline" id=".EB.B0.9C.EA.B2.AC">발견</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%A7%88%EC%BC%80%EB%A7%88%EC%BC%80_(%EC%99%9C%ED%96%89%EC%84%B1)&amp;action=edit&amp;section=1" title="부분 편집: 발견">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>마케마케는 마이클 브라운 탐사팀에 의해 2005년 3월 31일 발견되었다.<sup class="reference" id="cite_ref-jpldata_5-0"><a href="#cite_note-jpldata-5">[5]</a></sup> 이 발견은 에리스와 같은 날, 그리고 <a href="/wiki/%ED%95%98%EC%9A%B0%EB%A9%94%EC%95%84_(%EC%99%9C%ED%96%89%EC%84%B1)" title="하우메아 (왜행성)">하우메아</a>의 발표일 이틀 후인 2005년 7월 29일에 공표되었다.<sup class="reference" id="cite_ref-6"><a href="#cite_note-6">[6]</a></sup></p>
<p>상대적으로 밝은(명왕성의 15등급보다 약간 어두운 16.7등급 정도) 밝기에도 불구하고, 마케마케는 최근까지 많은 다른 카이퍼대 천체들이 발견될 때까지도 그 존재를 드러내지 않았다. 이는 상대적으로 <a class="mw-redirect" href="/wiki/%EA%B6%A4%EB%8F%84%EA%B2%BD%EC%82%AC%EA%B0%81" title="궤도경사각">궤도경사각</a>이 큰데다, 발견 당시 <a href="/wiki/%ED%99%A9%EB%8F%84" title="황도">황도</a>에서 가장 멀리 떨어진 지점에 자리잡고 있었기 때문이다(<a href="/wiki/%EB%A8%B8%EB%A6%AC%ED%84%B8%EC%9E%90%EB%A6%AC" title="머리털자리">머리털자리</a>에 있었다).<sup class="reference" id="cite_ref-Horizons_7-0"><a href="#cite_note-Horizons-7">[7]</a></sup> 이미 발견된 많은 왜행성들은 대체로 황도 근처에 자리잡고 있었다.</p>
<p><a href="/wiki/%ED%81%B4%EB%9D%BC%EC%9D%B4%EB%93%9C_%ED%86%B0%EB%B3%B4" title="클라이드 톰보">클라이드 톰보</a>가 <a href="/wiki/1930%EB%85%84" title="1930년">1930년</a> 명왕성을 발견했을 당시 마케마케의 위치는 황도에서 불과 몇 도 되지 않은 곳이었고(<a href="/wiki/%ED%99%A9%EC%86%8C%EC%9E%90%EB%A6%AC" title="황소자리">황소자리</a>와 <a href="/wiki/%EB%A7%88%EC%B0%A8%EB%B6%80%EC%9E%90%EB%A6%AC" title="마차부자리">마차부자리</a>의 경계선), <a class="mw-redirect" href="/wiki/%EA%B2%89%EB%B3%B4%EA%B8%B0_%EB%93%B1%EA%B8%89" title="겉보기 등급">겉보기 등급</a>은 16.0이었다.<sup class="reference" id="cite_ref-Horizons_7-1"><a href="#cite_note-Horizons-7">[7]</a></sup> 불운하게도 마케마케가 있는 곳은 은하수 부근으로 뒷배경의 별들이 많아서 어두운 천체를 발견하기가 힘들었다. 톰보는 명왕성 발견 후에도 여러 해에 걸쳐 미지의 천체를 계속 찾았으나<sup class="reference" id="cite_ref-8"><a href="#cite_note-8">[8]</a></sup> 마케마케 혹은 다른 <a class="mw-redirect" href="/wiki/%ED%81%90%EB%B9%84%EC%9B%90%EC%A1%B1" title="큐비원족">큐비원족</a> 천체의 존재를 감지하는 데에는 실패했다.</p>
<h3><span class="mw-headline" id=".EB.AA.85.EC.B9.AD">명칭</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%A7%88%EC%BC%80%EB%A7%88%EC%BC%80_(%EC%99%9C%ED%96%89%EC%84%B1)&amp;action=edit&amp;section=2" title="부분 편집: 명칭">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>2005 FY<sub>9</sub>는 발표 사실이 대중에 공개됨과 함께 <b>마케마케</b><sup class="reference" id="cite_ref-Craig2004_9-0"><a href="#cite_note-Craig2004-9">[9]</a></sup> 라는 이름을 얻었다. 마케마케 이름을 붙이기 전에 이 천체의 별칭은 <b>부활절 토끼</b>였다(이런 이름이 붙은 이유는 <a href="/wiki/%EB%B6%80%ED%99%9C%EC%A0%88" title="부활절">부활절</a> 직후 발견되었기 때문이다).<sup class="reference" id="cite_ref-brownblog_10-0"><a href="#cite_note-brownblog-10">[10]</a></sup></p>
<p>2008년 국제 천문 연맹의 규칙에 따라 2005 FY<sub>9</sub>에는 <a class="mw-redirect" href="/wiki/%EC%A1%B0%EB%AC%BC%EC%A3%BC" title="조물주">조물주</a>의 이름이 붙었다. <a href="/wiki/%EB%A7%88%EC%BC%80%EB%A7%88%EC%BC%80_(%EC%8B%A0%ED%99%94)" title="마케마케 (신화)">마케마케</a>는 <a href="/wiki/%EC%9D%B4%EC%8A%A4%ED%84%B0_%EC%84%AC" title="이스터 섬">이스터 섬</a>의 신화에 등장하는, 세상을 창조한 신의 이름이며,<sup class="reference" id="cite_ref-name_2-1"><a href="#cite_note-name-2">[2]</a></sup> 이 이름을 선택한 이유는 부활절의 의미를 간직하기 위한 의도가 있었기 때문이다.</p>
<h2><span class="mw-headline" id=".EB.AC.BC.EB.A6.AC.EC.A0.81_.ED.8A.B9.EC.A7.95">물리적 특징</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%A7%88%EC%BC%80%EB%A7%88%EC%BC%80_(%EC%99%9C%ED%96%89%EC%84%B1)&amp;action=edit&amp;section=3" title="부분 편집: 물리적 특징">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<h3><span class="mw-headline" id=".ED.81.AC.EA.B8.B0_.EB.B0.8F_.EB.B0.9D.EA.B8.B0">크기 및 밝기</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%A7%88%EC%BC%80%EB%A7%88%EC%BC%80_(%EC%99%9C%ED%96%89%EC%84%B1)&amp;action=edit&amp;section=4" title="부분 편집: 크기 및 밝기">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>마케마케는 현재 시각적으로 볼 때 명왕성 다음으로 카이퍼대 천체들 중 밝은 존재로,<sup class="reference" id="cite_ref-brown2006_11-0"><a href="#cite_note-brown2006-11">[11]</a></sup> 3월에 지구와 <a href="/wiki/%EC%B6%A9" title="충">충</a> 위치인 <a href="/wiki/%EB%A8%B8%EB%A6%AC%ED%84%B8%EC%9E%90%EB%A6%AC" title="머리털자리">머리털자리</a> 부근에 있었고 이때 밝기는 겉보기 등급 기준 16.7이었다.<sup class="reference" id="cite_ref-Horizons_7-2"><a href="#cite_note-Horizons-7">[7]</a></sup> 이는 아마추어 천문가용 <a href="/wiki/%EB%A7%9D%EC%9B%90%EA%B2%BD" title="망원경">망원경</a>으로 충분히 볼 수 있는 밝기이다. 마케마케의 크기는 정확히 알려져 있지는 않으나 <a class="mw-redirect" href="/wiki/%EC%8A%A4%ED%94%BC%EC%B2%98_%EC%9A%B0%EC%A3%BC_%EB%A7%9D%EC%9B%90%EA%B2%BD" title="스피처 우주 망원경">스피처 우주 망원경</a>이 <a href="/wiki/%EC%A0%81%EC%99%B8%EC%84%A0" title="적외선">적외선</a> 영역으로 관측한 결과, 명왕성의 스펙트럼과 유사하다는 점에 착안하여 약 1,500 킬로미터 반지름을 지녔다는 결론을 얻게 되었다.<sup class="reference" id="cite_ref-Stansberry_2007_12-0"><a href="#cite_note-Stansberry_2007-12">[12]</a></sup> 이는 하우메아보다 약간 더 큰 수준으로, 마케마케는 에리스와 명왕성 다음으로 카이퍼대 천체들 중 세 번째로 큰 <a href="/wiki/%EC%99%9C%ED%96%89%EC%84%B1" title="왜행성">왜행성</a>이라는 지위를 갖게 되었다.<sup class="reference" id="cite_ref-Tegler_13-0"><a href="#cite_note-Tegler-13">[13]</a></sup><sup class="reference" id="cite_ref-14"><a href="#cite_note-14">[14]</a></sup> 마케마케는 현재 태양계의 네 번째 왜행성으로 등록되어 있다.</p>
<h3><span class="mw-headline" id=".EC.8A.A4.ED.8E.99.ED.8A.B8.EB.9F.BC">스펙트럼</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%A7%88%EC%BC%80%EB%A7%88%EC%BC%80_(%EC%99%9C%ED%96%89%EC%84%B1)&amp;action=edit&amp;section=5" title="부분 편집: 스펙트럼">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>2006년 <a class="new" href="/w/index.php?title=%EC%B2%9C%EB%AC%B8%ED%95%99%EA%B3%BC_%EC%B2%9C%EC%B2%B4%EB%AC%BC%EB%A6%AC%ED%95%99_%EC%A0%80%EB%84%90&amp;action=edit&amp;redlink=1" title="천문학과 천체물리학 저널 (없는 문서)">천문학과 천체물리학 저널</a>에 도착한 편지에 따르면, 리칸드로 연구진은 <a href="/wiki/%EC%9C%8C%EB%A6%AC%EC%97%84_%ED%97%88%EC%85%9C_%EB%A7%9D%EC%9B%90%EA%B2%BD" title="윌리엄 허셜 망원경">윌리엄 허셜 망원경</a>과 <a class="new" href="/w/index.php?title=TNG_%EB%A7%9D%EC%9B%90%EA%B2%BD&amp;action=edit&amp;redlink=1" title="TNG 망원경 (없는 문서)">TNG 망원경</a>을 이용한 <a class="mw-redirect" href="/wiki/%EA%B0%80%EC%8B%9C%EA%B4%91" title="가시광">가시광</a> 및 <a class="mw-redirect" href="/wiki/%EA%B7%BC%EC%A0%81%EC%99%B8%EC%84%A0" title="근적외선">근적외선</a> 영역 관측을 통해, 마케마케 표면이 명왕성과 닮았음을 보여주었다.<sup class="reference" id="cite_ref-atmosphere_15-0"><a href="#cite_note-atmosphere-15">[15]</a></sup> 가시광선으로 보았을 때 마케마케는 붉게 보였는데 이는 에리스가 중성 스펙트럼을 보여준 것과는 반대였다. 적외선 영역으로 관찰한 결과 명왕성과 에리스처럼 <a href="/wiki/%EB%A9%94%ED%85%8C%EC%9D%B8" title="메테인">메테인</a>(CH<sub>4</sub>)이 검출되었는데(명왕성보다도 많은 양이었다), 이는 마케마케 표면에 일시적인 대기가 존재함을 의미하는 것일 가능성을 보여주는 것이었다.</p>
<p>마케마케 표면을 <a href="/wiki/%EB%B6%84%EA%B4%91%ED%95%99" title="분광학">분광학</a>적으로 분석한 결과 지름 1센티미터 정도의, 상당량의 에탄이 섞인, 메탄 얼음 조각들이 있다는 증거가 포착되었다. 이들 얼음 조각들은 대부분이 태양 복사로 인한 메탄의 <a class="mw-redirect" href="/wiki/%EA%B4%91%ED%95%B4%EB%A6%AC" title="광해리">광해리</a>로 인해 생겨난 것으로 보인다. 에리스나 명왕성과는 달리 마케마케 표면에는 <a href="/wiki/%EC%A7%88%EC%86%8C" title="질소">질소</a> 얼음이 없었으며 이는 태양계의 역사를 지나오면서 표면에 있던 질소가 어떠한 이유 때문에 고갈되었음을 뜻한다.<sup class="reference" id="cite_ref-brown_16-0"><a href="#cite_note-brown-16">[16]</a></sup></p>
<h2><span class="mw-headline" id=".EA.B6.A4.EB.8F.84">궤도</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%A7%88%EC%BC%80%EB%A7%88%EC%BC%80_(%EC%99%9C%ED%96%89%EC%84%B1)&amp;action=edit&amp;section=6" title="부분 편집: 궤도">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>마케마케는 현재 태양에서 52 천문단위 떨어져 있으며<sup class="reference" id="cite_ref-Horizons_7-3"><a href="#cite_note-Horizons-7">[7]</a></sup> 이는 마케마케의 자기 궤도에서, 태양에서 가장 멀리 떨어져 있는 위치이다.<sup class="reference" id="cite_ref-brown_16-1"><a href="#cite_note-brown-16">[16]</a></sup> 마케마케의 궤도는 궤도경사각이 크고(29도) 공전궤도 <a href="/wiki/%EC%9D%B4%EC%8B%AC%EB%A5%A0" title="이심률">이심률</a>도 어느 정도 크다는(0.15) 점에서 하우메아와 매우 비슷하다.<sup class="reference" id="cite_ref-17"><a href="#cite_note-17">[17]</a></sup> 그럼에도 불구하고 마케마케의 궤도는 이들보다 좀 더 태양에서 먼 곳에 형성되어 있으며 공전주기는 310년 정도로 명왕성의 248년보다 크다.<sup class="reference" id="cite_ref-Buie_18-0"><a href="#cite_note-Buie-18">[18]</a></sup></p>

큐비원족 천체들의 궤도를 보여주는 그림. 각각의 궤도는 마케마케(파랑색), 하우메아(초록색), 명왕성(붉은색)이다.</div>

<p>위 그림에는 명왕성과 대비하여 하우메아와 마케마케의 궤도를 비교하여 보여주고 있다. 그림에서 <a class="mw-redirect" href="/wiki/%EA%B7%BC%EC%9D%BC%EC%A0%90" title="근일점">근일점</a>은 소문자 (q)로 표시했으며<sup class="reference" id="cite_ref-jpldata_5-1"><a href="#cite_note-jpldata-5">[5]</a></sup> <a class="mw-redirect" href="/wiki/%EC%9B%90%EC%9D%BC%EC%A0%90" title="원일점">원일점</a>은 대문자 (Q)이다. 근일점과 원일점에는 각각 해당 일자가 표시되어 있다. 그림 내 왜행성들의 위치는 2006년 4월 기준이며 각 천체의 이미지는 상대적 크기 및 반사율, 색을 고려하여 작성된 것이다. 두 왜행성 모두 황도보다 훨씬 위에 자리잡고 있다(해왕성 궤도는 흰색이다). 마케마케는 2034년 원일점을 통과할 것이다. 반면 하우메아는 1991년 이미 원일점을 통과했다.<sup class="reference" id="cite_ref-19"><a href="#cite_note-19">[19]</a></sup></p>
<h2><span class="mw-headline" id=".EC.9C.84.EC.84.B1">위성</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%A7%88%EC%BC%80%EB%A7%88%EC%BC%80_(%EC%99%9C%ED%96%89%EC%84%B1)&amp;action=edit&amp;section=7" title="부분 편집: 위성">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>2016년 4월 26일 최소 21,000km 떨어진 곳에서 12일 주기로 공전하는 것으로 추정되는 위성이 발견됐다. 임시이름은 MK2 이며 지름은 약 161km 으로 추정된다.</p>
<h2><span class="mw-headline" id=".EB.B6.84.EB.A5.98">분류</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%A7%88%EC%BC%80%EB%A7%88%EC%BC%80_(%EC%99%9C%ED%96%89%EC%84%B1)&amp;action=edit&amp;section=8" title="부분 편집: 분류">편집</a><span class="mw-editsection-bracket">]</span></span></h2>

<p>마케마케는 <a class="mw-redirect" href="/wiki/%ED%81%90%EB%B9%84%EC%9B%90%EC%A1%B1" title="큐비원족">큐비원족</a>에 속하며, 따라서 궤도도 큐비원족 영역 내에 있고 중력적으로 해왕성의 영향을 받지 않는다.<sup class="reference" id="cite_ref-20"><a href="#cite_note-20">[20]</a></sup><sup class="reference" id="cite_ref-jewitt_21-0"><a href="#cite_note-jewitt-21">[21]</a></sup> 해왕성과 3:2 궤도공명을 보이는 명왕성족들과는 달리, 큐비원족 천체들은 <a href="/wiki/%ED%95%B4%EC%99%95%EC%84%B1" title="해왕성">해왕성</a>의 중력 <a class="new" href="/w/index.php?title=%EC%84%AD%EB%8F%99&amp;action=edit&amp;redlink=1" title="섭동 (없는 문서)">섭동</a>에 관계없이, 태양에서 보다 먼 곳에 근일점을 형성하고 있다.<sup class="reference" id="cite_ref-jewitt_21-1"><a href="#cite_note-jewitt-21">[21]</a></sup> 큐비원족의 궤도는 이심률이 작고 행성들과 비슷한 모양으로 태양을 공전하고 있다. 그러나 마케마케는 다른 큐비원족들과는 달리 궤도경사각이 크다.<sup class="reference" id="cite_ref-22"><a href="#cite_note-22">[22]</a></sup></p>
<p><a href="/wiki/2006%EB%85%84" title="2006년">2006년</a> <a href="/wiki/8%EC%9B%94_24%EC%9D%BC" title="8월 24일">8월 24일</a> <a class="mw-redirect" href="/wiki/%EA%B5%AD%EC%A0%9C_%EC%B2%9C%EB%AC%B8_%EC%97%B0%EB%A7%B9" title="국제 천문 연맹">국제 천문 연맹</a>은 공식적으로 <a href="/wiki/%ED%96%89%EC%84%B1%EC%9D%98_%EC%A0%95%EC%9D%98" title="행성의 정의">행성의 새로운 정의</a>를 발표하였다. 이 정의에 의하면 <a href="/wiki/%ED%83%9C%EC%96%91%EA%B3%84_%EC%86%8C%EC%B2%9C%EC%B2%B4" title="태양계 소천체">태양계 소천체</a>는 ‘자신의 중력으로 둥근 구체를 형성할 수 없는 천체’들을 말한다. 여기에 반해 <a href="/wiki/%EC%99%9C%ED%96%89%EC%84%B1" title="왜행성">왜행성</a>은 ‘중력으로 구체를 형성할 수는 있으나, 궤도상에 있던 물질들을 청소하지는 못한 천체’이고, <a href="/wiki/%ED%96%89%EC%84%B1" title="행성">행성</a>은 ‘자체 중력으로 둥근 모양을 형성하며 궤도상에 있던 물질들을 청소한 천체’이다.<sup class="reference" id="cite_ref-IAU0603_23-0"><a href="#cite_note-IAU0603-23">[23]</a></sup> 이 분류에 의하면 명왕성, 에리스, <a href="/wiki/%EC%84%B8%EB%A0%88%EC%8A%A4_(%EC%99%9C%ED%96%89%EC%84%B1)" title="세레스 (왜행성)">세레스</a>는 왜행성이 된다.<sup class="reference" id="cite_ref-IAU0603_23-1"><a href="#cite_note-IAU0603-23">[23]</a></sup></p>
<p>2008년 6월 11일 국제 천문 연맹은 왜행성의 하위 개념이라고 할 수 있는 <a href="/wiki/%ED%94%8C%EB%A3%A8%ED%86%A0%EC%9D%B4%EB%93%9C" title="플루토이드">플루토이드</a>(명왕성형 천체)를 만듦으로써 2006년 설정한 분류 체계를 다듬었다. 플루토이드는 해왕성 궤도 너머에 있는 왜행성들을 가리킨다. 따라서 에리스와 명왕성은 플루토이드인 반면, 세레스는 아니다. 플루토이드로 인정받기 위해서는 절대 등급이 1보다 낮아야 한다.<sup class="reference" id="cite_ref-24"><a href="#cite_note-24">[24]</a></sup> 이 기준에 의하면 플루토이드의 자격이 있는 것은 마케마케와 하우메아 뿐이다. 2008년 7월 11일 국제 천문 연맹은 공식적으로 마케마케를 명왕성과 에리스 다음으로, 왜행성이자 플루토이드로 분류하였다.<sup class="reference" id="cite_ref-name_2-2"><a href="#cite_note-name-2">[2]</a></sup></p>
