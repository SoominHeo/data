<p><b>메이지 유신</b>(<span style="font-size: smaller;"><a href="/wiki/%EC%9D%BC%EB%B3%B8%EC%96%B4" title="일본어">일본어</a>:</span> <span lang="ja"><span lang="ja"><ruby><rb>明治維新</rb> <rp>(</rp><rt>めいじ いしん</rt><rp>)</rp></ruby></span></span> <small>메이지 이신<sup>[<a href="/wiki/%EC%9C%84%ED%82%A4%EB%B0%B1%EA%B3%BC:%EC%9D%BC%EB%B3%B8%EC%96%B4%EC%9D%98_%ED%95%9C%EA%B8%80_%ED%91%9C%EA%B8%B0" title="위키백과:일본어의 한글 표기">*</a>]</sup></small>)은 막번 체제를 해체하고 <a class="mw-redirect" href="/wiki/%EC%99%95%EC%A0%95_%EB%B3%B5%EA%B3%A0" title="왕정 복고">왕정 복고</a>를 통한 중앙 통일 권력의 확립에 이르는 광범위한 변혁 과정을 총칭한다.<sup class="reference" id="cite_ref-1"><a href="#cite_note-1">[1]</a></sup> 메이지 유신이라는 것은 학문상의 명칭으로 당시에는 어일신(<span style="font-size: smaller;"><a href="/wiki/%EC%9D%BC%EB%B3%B8%EC%96%B4" title="일본어">일본어</a>:</span> <span lang="ja"><span lang="ja"><ruby><rb>御一新</rb> <rp>(</rp><rt>ごいっしん</rt><rp>)</rp></ruby></span></span> <small>고잇신<sup>[<a href="/wiki/%EC%9C%84%ED%82%A4%EB%B0%B1%EA%B3%BC:%EC%9D%BC%EB%B3%B8%EC%96%B4%EC%9D%98_%ED%95%9C%EA%B8%80_%ED%91%9C%EA%B8%B0" title="위키백과:일본어의 한글 표기">*</a>]</sup></small>) 등으로 불리었다.<sup class="reference" id="cite_ref-2"><a href="#cite_note-2">[2]</a></sup> 메이지 유신의 개시 시기에 대해 대체로 <a href="/wiki/%EB%8D%B4%ED%8F%AC" title="덴포">덴포</a>(天保) 시기로 일치하고 있지만 종료 시기에 대해서는 <a href="/wiki/1871%EB%85%84" title="1871년">1871년</a> 폐번치현(廢藩置縣), <a href="/wiki/1873%EB%85%84" title="1873년">1873년</a> 지조개정(地租改正), <a href="/wiki/1877%EB%85%84" title="1877년">1877년</a> <a href="/wiki/%EC%84%B8%EC%9D%B4%EB%82%9C_%EC%A0%84%EC%9F%81" title="세이난 전쟁">세이난 전쟁</a>(西南戰爭), <a href="/wiki/1889%EB%85%84" title="1889년">1889년</a> <a class="mw-redirect" href="/wiki/%EC%9D%BC%EB%B3%B8_%EC%A0%9C%EA%B5%AD_%ED%97%8C%EB%B2%95" title="일본 제국 헌법">헌법</a> 발표 등으로 여러 설이 있으며 정설은 확립되어 있지 않다.<sup class="reference" id="cite_ref-3"><a href="#cite_note-3">[3]</a></sup> 그러나, 중국 학계에서는 1889년 헌법 제정으로 메이지 유신이 종료되었다고 본다.</p>

<h2><span class="mw-headline" id=".EB.B0.B0.EA.B2.BD">배경</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%A9%94%EC%9D%B4%EC%A7%80_%EC%9C%A0%EC%8B%A0&amp;action=edit&amp;section=1" title="부분 편집: 배경">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<div class="thumb tleft">

<div class="magnify"><a class="internal" href="/wiki/%ED%8C%8C%EC%9D%BC:Taisehokan.jpg" title="실제 크기로"></a></div>
<a href="/wiki/%EB%8C%80%EC%A0%95%EB%B4%89%ED%99%98" title="대정봉환">대정봉환</a></div>

<p><a href="/wiki/%EC%97%90%EB%8F%84_%EB%A7%89%EB%B6%80" title="에도 막부">에도 막부</a>는 외교에 관한 권리를 독점하고, 일본인의 출입국과 무역을 관리, 통제, 제한하기 위해서 오랫동안 <a class="mw-redirect" href="/wiki/%EC%87%84%EA%B5%AD" title="쇄국">쇄국</a> 체제를 유지해 왔다. 그러나 <a href="/wiki/1856%EB%85%84" title="1856년">1856년</a>부터 <a href="/wiki/1860%EB%85%84" title="1860년">1860년</a>에 걸친 <a href="/wiki/%EC%95%84%ED%8E%B8_%EC%A0%84%EC%9F%81" title="아편 전쟁">아편 전쟁</a> 이후 <a href="/wiki/%EB%8F%99%EC%95%84%EC%8B%9C%EC%95%84" title="동아시아">동아시아</a>로 진출하려는 서구 <a href="/wiki/%EC%A0%9C%EA%B5%AD%EC%A3%BC%EC%9D%98" title="제국주의">제국주의</a>의 물결은 더욱 거세지고 있었다. <a href="/wiki/1853%EB%85%84" title="1853년">1853년</a> <a href="/wiki/%EB%AF%B8%EA%B5%AD" title="미국">미국</a>의 동인도함대 사령관 <a class="mw-redirect" href="/wiki/%EB%A7%A4%EC%8A%88_%EC%BA%98%EB%B8%8C%EB%A0%88%EC%9D%B4%EC%8A%A4_%ED%8E%98%EB%A6%AC" title="매슈 캘브레이스 페리">매슈 페리</a> 제독이 <a href="/wiki/%EB%B0%80%EB%9F%AC%EB%93%9C_%ED%95%84%EB%AA%A8%EC%96%B4" title="밀러드 필모어">밀러드 필모어</a> 미국 대통령의 개국 요구 국서를 가지고 일본에 왔다.(<a href="/wiki/%ED%9D%91%EC%84%A0_%EB%82%B4%ED%95%AD" title="흑선 내항">흑선 내항</a>) 이에 막부는 <a href="/wiki/1854%EB%85%84" title="1854년">1854년</a> <a class="mw-redirect" href="/wiki/%EB%AF%B8%EC%9D%BC%ED%99%94%EC%B9%9C%EC%A1%B0%EC%95%BD" title="미일화친조약">미일화친조약</a>에 이어 <a href="/wiki/1858%EB%85%84" title="1858년">1858년</a>에는 <a href="/wiki/%EB%AF%B8%EA%B5%AD" title="미국">미국</a>을 비롯하여 <a href="/wiki/%EC%98%81%EA%B5%AD" title="영국">영국</a>, <a href="/wiki/%EB%9F%AC%EC%8B%9C%EC%95%84" title="러시아">러시아</a>, <a href="/wiki/%EB%84%A4%EB%8D%9C%EB%9E%80%EB%93%9C" title="네덜란드">네덜란드</a>, <a href="/wiki/%ED%94%84%EB%9E%91%EC%8A%A4" title="프랑스">프랑스</a>와 굴욕적인 통상조약(<a href="/wiki/%EC%95%88%EC%84%B8%EC%9D%B4_5%EA%B0%9C%EA%B5%AD_%EC%A1%B0%EC%95%BD" title="안세이 5개국 조약">안세이 5개국 조약</a>)을 체결하였다.</p>
<p>그러나 이 조약은 막부 정부에서 칙허 없이 처리했다는 점 때문에 이에 반발한 반막부 세력이 일어나 막부 정부와 대립하는 혼란기를 겪었다. 그러다가 260여 년이나 내려오던 도쿠가와 막부가 <a href="/wiki/1866%EB%85%84" title="1866년">1866년</a> 사카모토 료마를 내세우는 <a href="/wiki/%EC%82%BF%EC%B4%88_%EB%8F%99%EB%A7%B9" title="삿초 동맹">삿초 동맹</a>에 패배하였고, <a href="/wiki/1867%EB%85%84" title="1867년">1867년</a>에는 <a href="/wiki/%EB%8C%80%EC%A0%95%EB%B4%89%ED%99%98" title="대정봉환">대정봉환</a>과 <a href="/wiki/%EC%99%95%EC%A0%95%EB%B3%B5%EA%B3%A0_(%EC%9D%BC%EB%B3%B8)" title="왕정복고 (일본)">왕정복고</a>가 이루어졌다.</p>
<h2><span class="mw-headline" id=".EB.8F.99.EB.A7.B9">동맹</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%A9%94%EC%9D%B4%EC%A7%80_%EC%9C%A0%EC%8B%A0&amp;action=edit&amp;section=2" title="부분 편집: 동맹">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p><a href="/wiki/1866%EB%85%84" title="1866년">1866년</a> <a href="/wiki/%EC%82%AC%EC%93%B0%EB%A7%88_%EB%B2%88" title="사쓰마 번">사쓰마 번</a>의 지도자 <a href="/wiki/%EC%82%AC%EC%9D%B4%EA%B3%A0_%EB%8B%A4%EC%B9%B4%EB%AA%A8%EB%A6%AC" title="사이고 다카모리">사이고 다카모리</a>와 <a href="/wiki/%EC%A1%B0%EC%8A%88_%EB%B2%88" title="조슈 번">조슈 번</a>의 <a href="/wiki/%EA%B8%B0%EB%8F%84_%EB%8B%A4%EC%B9%B4%EC%9A%94%EC%8B%9C" title="기도 다카요시">기도 다카요시</a> 사이의 <a href="/wiki/%EC%82%BF%EC%B4%88_%EB%8F%99%EB%A7%B9" title="삿초 동맹">삿초 동맹</a>으로 메이지 유신이 이뤄졌다. 이 두 지도자는 <a href="/wiki/%EA%B3%A0%EB%A9%94%EC%9D%B4_%EC%B2%9C%ED%99%A9" title="고메이 천황">고메이 천황</a>을 지지하였다. 이들은 <a href="/wiki/%EC%82%AC%EC%B9%B4%EB%AA%A8%ED%86%A0_%EB%A3%8C%EB%A7%88" title="사카모토 료마">사카모토 료마</a>에 의해 천거되었는데 도쿠가와 쇼군의 지배에 도전하여 천황의 권력을 회복하기 위해서였다. <a href="/wiki/1866%EB%85%84" title="1866년">1866년</a> <a href="/wiki/12%EC%9B%94_25%EC%9D%BC" title="12월 25일">12월 25일</a> 고메이 천황이 세상을 떠나자 1867년 1월 9일 <a href="/wiki/%EB%A9%94%EC%9D%B4%EC%A7%80_%EC%B2%9C%ED%99%A9" title="메이지 천황">메이지 천황</a>이 뒤를 이었다.</p>
<h2><span class="mw-headline" id=".EA.B0.9C.ED.98.81.EC.9D.98_.EB.82.B4.EC.9A.A9">개혁의 내용</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%A9%94%EC%9D%B4%EC%A7%80_%EC%9C%A0%EC%8B%A0&amp;action=edit&amp;section=3" title="부분 편집: 개혁의 내용">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>메이지 정부는 주로 구미 열강 국을 따라 잡기 위해, 개혁을 모색하였다. 학제, 징병령, 지조개정(地租改正) 등 일련의 개혁을 추진하고, <a href="/wiki/%EB%B6%80%EA%B5%AD%EA%B0%95%EB%B3%91" title="부국강병">부국강병</a>의 기치 하에 유럽과 미국의 근대 국가를 모델로 하여, 민주화와 인권 운동을 탄압하고 천황이 주도하여 일방적 <a href="/wiki/%EC%9E%90%EB%B3%B8%EC%A3%BC%EC%9D%98" title="자본주의">자본주의</a> 육성과 군사적 강화에 노력하였다.</p>
<h3><span class="mw-headline" id=".EC.A4.91.EC.95.99_.ED.96.89.EC.A0.95">중앙 행정</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%A9%94%EC%9D%B4%EC%A7%80_%EC%9C%A0%EC%8B%A0&amp;action=edit&amp;section=4" title="부분 편집: 중앙 행정">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<div class="thumb tright">

<a href="/wiki/%EB%A9%94%EC%9D%B4%EC%A7%80_%EC%B2%9C%ED%99%A9" title="메이지 천황">메이지 천황</a>의 <a href="/wiki/%EB%8F%84%EC%BF%84_%ED%96%89%ED%96%89" title="도쿄 행행">도쿄 행행</a><br/>

<p>막번체제의 붕괴로, <a href="/wiki/%EC%A4%91%EC%95%99%EC%A7%91%EA%B6%8C" title="중앙집권">중앙집권</a> 국가의 확립을 서둘러야 했던 신정부는, <a class="mw-redirect" href="/wiki/%EC%9C%A8%EB%A0%B9%EC%A0%9C" title="율령제">율령제</a>를 본보기로 한 명칭을 부활시켰다.</p>
<p><a class="mw-redirect" href="/wiki/%EC%99%95%EC%A0%95_%EB%B3%B5%EA%B3%A0%EC%9D%98_%EB%8C%80%ED%98%B8%EB%A0%B9" title="왕정 복고의 대호령">왕정 복고의 대호령</a>으로 인해, <a href="/wiki/%EB%A7%89%EB%B6%80" title="막부">막부</a>나 <a href="/wiki/%EC%84%AD%EC%A0%95" title="섭정">섭정</a>, <a class="mw-redirect" href="/wiki/%EA%B0%84%ED%8C%8C%EC%BF%A0" title="간파쿠">간파쿠</a>의 폐지와 일본 천황의 친정이 결정되어, 일본 천황 아래로 총재(総裁), 의정(議定), <a class="mw-disambig" href="/wiki/%EC%82%B0%EC%9A%94" title="산요">산요</a>(参与)로 나뉘게 되는 관제가 시행되었지만, <a href="/wiki/%EB%A9%94%EC%9D%B4%EC%A7%80_%EC%B2%9C%ED%99%A9" title="메이지 천황">메이지 천황</a>은 아직 어렸기 때문에 그것을 보좌하는 체제가 필요했다. 거기서, 메이지 원년 4월 21일, 정체서(政体書)가 공포되었고(정체서 체제), 다음 해 율령제의 2관 8성을 본뜬 2관 6성제가 발족한다. 구체적인 행정 기구로써는, 태정관(太政官)과 신기관(神祇官)을 두어, 태정관 아래에 각 성(省)을 두는 율령제가 모사되었지만, 그 후 <a class="new" href="/w/index.php?title=%EC%9D%BC%EB%B3%B8_%EB%AF%BC%EB%B6%80%EC%84%B1&amp;action=edit&amp;redlink=1" title="일본 민부성 (없는 문서)">민부성</a>에서 <a class="new" href="/w/index.php?title=%EC%9D%BC%EB%B3%B8_%EA%B3%B5%EB%B6%80%EC%84%B1&amp;action=edit&amp;redlink=1" title="일본 공부성 (없는 문서)">공부성</a>으로 분리되거나, <a class="new" href="/w/index.php?title=%EC%9D%BC%EB%B3%B8_%ED%98%95%EB%B6%80%EC%84%B1&amp;action=edit&amp;redlink=1" title="일본 형부성 (없는 문서)">형부성</a>으로부터 <a class="mw-redirect" href="/wiki/%EC%9D%BC%EB%B3%B8_%EC%82%AC%EB%B2%95%EC%84%B1" title="일본 사법성">사법성</a>으로의 개편 등 무수한 개변이 필요했기 때문에, 제도가 안정되지 않았다. 또한 입법부인 좌원(左院, 후 원로원(元老院)), 우원이나 지방관 회의등도 설치, 폐지가 반복되었다. 메이지 중앙의 개혁은 <a href="/wiki/1885%EB%85%84" title="1885년">1885년</a> <a class="new" href="/w/index.php?title=%EB%82%B4%EA%B0%81_%EC%A0%9C%EB%8F%84&amp;action=edit&amp;redlink=1" title="내각 제도 (없는 문서)">내각 제도</a>의 발족으로 간신히 안정된다.</p>
<p>또한, 입법부에 관해서 <a href="/wiki/%EA%B8%B0%EB%8F%84_%EB%8B%A4%EC%B9%B4%EC%9A%94%EC%8B%9C" title="기도 다카요시">기도 다카요시</a>등이 메이지 초부터 의회개설을 주장하였지만, 의회제도를 발족시키기 위해선, 관제 개혁, 민도(民度), 국민 교육등의 부분에서 미성숙하여 시기 상조였기 때문에, <a href="/wiki/%EC%98%A4%EC%BF%A0%EB%B3%B4_%EB%8F%84%EC%8B%9C%EB%AF%B8%EC%B9%98" title="오쿠보 도시미치">오쿠보 도시미치</a>를 중심으로 '<b>유사 전제(有司専制)<sup class="reference" id="cite_ref-4"><a href="#cite_note-4">[4]</a></sup>'</b>라고 불리는 사쓰마, 조슈 번의 파벌에 의한 관료를 중심으로 한 개혁 체제가 유지되었다. 그러나, <a href="/wiki/%EC%9E%90%EC%9C%A0%EB%AF%BC%EA%B6%8C%EC%9A%B4%EB%8F%99" title="자유민권운동">자유민권운동</a>의 고조나, 여러 제도의 정비에 의한 개혁 성숙 등도 있어, <a href="/wiki/1881%EB%85%84" title="1881년">1881년</a>에 《국회 개설 조서》가 나와, 동시에 <a href="/wiki/%EC%9D%B4%ED%86%A0_%ED%9E%88%EB%A1%9C%EB%B6%80%EB%AF%B8" title="이토 히로부미">이토 히로부미</a>등에 의해 의회 제도를 전제로 한 <a href="/wiki/%ED%97%8C%EB%B2%95" title="헌법">헌법</a>제정의 움직임이 본격화되어, 헌법 심의를 위한 <a class="mw-redirect" href="/wiki/%EC%9D%BC%EB%B3%B8_%EC%B6%94%EB%B0%80%EC%9B%90" title="일본 추밀원">추밀원</a>이 설치되었다. 1889년 <a class="mw-redirect" href="/wiki/%EC%9D%BC%EB%B3%B8_%EC%A0%9C%EA%B5%AD_%ED%97%8C%EB%B2%95" title="일본 제국 헌법">일본 제국 헌법</a> 공포, 다음 해 제국 의회가 발족하여, 아시아에서 처음으로 본격적인 <a class="mw-redirect" href="/wiki/%EC%9E%85%ED%97%8C_%EA%B5%B0%EC%A3%BC%EC%A0%9C" title="입헌 군주제">입헌 군주제</a>·의회제 <a href="/wiki/%EB%AF%BC%EC%A3%BC%EC%A3%BC%EC%9D%98" title="민주주의">민주주의</a> 국가가 완성되었다.<sup class="reference" id="cite_ref-5"><a href="#cite_note-5">[5]</a></sup></p>
<p>수도에 대해선, 당초 <a href="/wiki/%EA%B5%90%ED%86%A0" title="교토">교토</a>에서는 구폐(舊弊)가 많다고 하여, <a href="/wiki/%EC%98%A4%EC%82%AC%EC%B9%B4" title="오사카">오사카</a> 천도론이 <a href="/wiki/%EC%98%A4%EC%BF%A0%EB%B3%B4_%EB%8F%84%EC%8B%9C%EB%AF%B8%EC%B9%98" title="오쿠보 도시미치">오쿠보 도시미치</a>를 중심으로 주창되었다. 그러나 교토에서 옮기는 것에 대해 반대가 많았고, <a href="/wiki/%EC%97%90%EB%8F%84_%EA%B0%9C%EC%84%B1" title="에도 개성">에도 성의 개성</a>도 있어, 에도를 <a href="/wiki/%EB%8F%84%EC%BF%84" title="도쿄">도쿄</a>로 하는 것으로 결정되었다. <a href="/wiki/%EB%A9%94%EC%9D%B4%EC%A7%80_%EC%B2%9C%ED%99%A9" title="메이지 천황">메이지 천황</a>의 2번 도쿄 행차로 의해 <a href="/wiki/%ED%83%9C%EC%A0%95%EA%B4%80" title="태정관">태정관</a>도 도쿄에 옮겨져, 도쿄가 사실상의 수도가 되었다.</p>
<h3><span class="mw-headline" id=".EC.A7.80.EC.97.AD_.ED.96.89.EC.A0.95">지역 행정</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%A9%94%EC%9D%B4%EC%A7%80_%EC%9C%A0%EC%8B%A0&amp;action=edit&amp;section=5" title="부분 편집: 지역 행정">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>메이지 정부의 지방 행정으로, 도쿠가와 가를 <a href="/wiki/%EC%8A%A8%ED%91%B8_%EB%B2%88" title="슨푸 번">슨푸 번</a>으로 이동, 교토, <a href="/wiki/%EB%82%98%EA%B0%80%EC%82%AC%ED%82%A4_%ED%98%84" title="나가사키 현">나가사키</a>, <a class="mw-redirect" href="/wiki/%ED%95%98%EC%BD%94%EB%8B%A4%ED%85%8C" title="하코다테">하코다테</a>를 정부 직할 '부(府)'로 한 것 이외는, 원칙으로서 이전의 번체제가 유지되고 있었다. 그러나, 부국강병 목적의 근대국가 건설을 추진하기 위해서는, 중앙집권화에 의한 정부의 지방 지배 강화가 필요했다.</p>
<p>우선, 메이지 2년의 <b><a href="/wiki/%ED%8C%90%EC%A0%81%EB%B4%89%ED%99%98" title="판적봉환">판적봉환</a></b>으로 구 번의 영주가 자발적으로 판(토지), 적(인민)을 천황에게 반납하여, 다시 지번사(知藩事,지한지)로 임명되어, 번의 영지와 영주의 분리를 도모할 수 있어, 중요 지역이나 옛 막부부 직할지였던 부, 현과 함께 '부번현 체제(府藩県体制)'가 된다. 그러나, 중앙집권화를 진행시켜, 개혁을 전국적으로 망라하려는 데에 있어 번의 존재는 방해가 되었고, 번 측에서도 재정의 핍박이 계속 되어 자발적으로 폐번을 신청하는 번이 잇따랐다. 1871년 8월 29일, 사쓰마, 조슈 번 출신의 지도자에 의해 <a href="/wiki/%ED%8F%90%EB%B2%88%EC%B9%98%ED%98%84" title="폐번치현">폐번치현</a>이 실시되어 <a href="/wiki/%EB%8F%84%EB%8F%84%EB%B6%80%ED%98%84" title="도도부현">도도부현</a>제도 설치(당초에는 3부 302현, 직후 정리되어 3부 72현), 중앙정부로부터 지사를 파견하는 제도가 실시되었다. 이것에 대해선, 령제국의 지명을 이용하지 않았기 때문에, 도시명이 부현명이 된 곳도 적지 않다. 그러나 <a href="/wiki/%EC%82%AC%EC%93%B0%EB%A7%88_%EB%B2%88" title="사쓰마 번">사쓰마 번</a>의 <a href="/wiki/%EC%8B%9C%EB%A7%88%EC%A6%88_%ED%9E%88%EC%82%AC%EB%AF%B8%EC%93%B0" title="시마즈 히사미쓰">시마즈 히사미쓰</a>이외는 눈에 띈 반발은 없었다.<sup class="reference" id="cite_ref-6"><a href="#cite_note-6">[6]</a></sup></p>
<h3><span class="mw-headline" id=".EC.97.AC.EB.9F.AC_.EC.A0.9C.EB.8F.84.EC.9D.98_.EA.B0.9C.ED.98.81">여러 제도의 개혁</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%A9%94%EC%9D%B4%EC%A7%80_%EC%9C%A0%EC%8B%A0&amp;action=edit&amp;section=6" title="부분 편집: 여러 제도의 개혁">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p><a href="/wiki/%ED%8F%90%EB%B2%88%EC%B9%98%ED%98%84" title="폐번치현">폐번치현</a>(번을 폐지하고 모두 현으로 바꾸는 것)과 태정관 제도의 개혁을 거쳐 중앙집권 체제가 갖추어짐으로써, 간신히 옛 막부 시대의 제도를 개혁할 준비가 갖추어졌다. 거의 동시에 궁중 개혁도 이루어져, 구래의 궁중직이나 궁녀 폐지, <a class="mw-redirect" href="/wiki/%EC%82%AC%EC%A1%B1" title="사족">사족</a>을 중심으로 한 시종 등이 <a href="/wiki/%EB%A9%94%EC%9D%B4%EC%A7%80_%EC%B2%9C%ED%99%A9" title="메이지 천황">메이지 천황</a>을 개혁 군주에 어울리는 천황으로 양육하게 된다. 막부 말기에는 병약했던 메이지 천황도, 사족에 의한 양육 때문인지 건강도 회복하여, 서양식 입헌 군주로서의 마음가짐도 배워, '메이지 국가' 원수로 어울리는 존재가 되어 갔다. 특히 헌법 제정 과정에서의 추밀원 심의에 모두 참석하였고, 또 국회 개설 전후 입헌 정치 미성숙기에 수상이 빈번하게 사임, 교대했을 때에도 정국의 조정자로서 중요한 역할을 담당하였다.</p>
<p>신분 제도에서는, 에도 막부 아래의 '<a href="/wiki/%EC%82%AC%EB%86%8D%EA%B3%B5%EC%83%81" title="사농공상">사농공상</a>'의 구별을 폐지, 구 무사 계급을 <a class="mw-redirect" href="/wiki/%EC%82%AC%EC%A1%B1" title="사족">사족</a>, 그 이외를 <a href="/wiki/%ED%8F%89%EB%AF%BC" title="평민">평민</a>으로 하여, '사민평등'을 구가하는 한편, 구 <a class="mw-redirect" href="/wiki/%EA%B5%AC%EA%B2%8C" title="구게">구게</a>, <a href="/wiki/%EB%8B%A4%EC%9D%B4%EB%AC%98" title="다이묘">다이묘</a>나 일부 승려 등을 새롭게 <a href="/wiki/%ED%99%94%EC%A1%B1" title="화족">화족</a>으로서 특권 계급으로 함과 동시에, <a class="mw-redirect" href="/wiki/%EC%9D%BC%EB%B3%B8_%EA%B6%81%EB%82%B4%EC%84%B1" title="일본 궁내성">궁내성</a> 지배 아래로 두게 되었다.</p>
<p>또한, 유신 정부는 서양의 여러 제도를 연구하기 위해 <a href="/wiki/%EC%9D%B4%EC%99%80%EC%BF%A0%EB%9D%BC_%EB%8F%84%EB%AA%A8%EB%AF%B8" title="이와쿠라 도모미">이와쿠라 도모미</a>를 정사, <a href="/wiki/%EC%98%A4%EC%BF%A0%EB%B3%B4_%EB%8F%84%EC%8B%9C%EB%AF%B8%EC%B9%98" title="오쿠보 도시미치">오쿠보 도시미치</a>, <a href="/wiki/%EA%B8%B0%EB%8F%84_%EB%8B%A4%EC%B9%B4%EC%9A%94%EC%8B%9C" title="기도 다카요시">기도 다카요시</a> <a href="/wiki/%EC%9D%B4%ED%86%A0_%ED%9E%88%EB%A1%9C%EB%B6%80%EB%AF%B8" title="이토 히로부미">이토 히로부미</a> 등을 부사로 하는 <a href="/wiki/%EC%9D%B4%EC%99%80%EC%BF%A0%EB%9D%BC_%EC%82%AC%EC%A0%88%EB%8B%A8" title="이와쿠라 사절단">이와쿠라 사절단</a>을 구미에 파견하지만, '잔류 정부'로 불리며 일본에 남은 <a href="/wiki/%EC%82%AC%EC%9D%B4%EA%B3%A0_%EB%8B%A4%EC%B9%B4%EB%AA%A8%EB%A6%AC" title="사이고 다카모리">사이고 다카모리</a>, <a href="/wiki/%EC%9D%B4%EB%85%B8%EC%9A%B0%EC%97%90_%EA%B0%80%EC%98%A4%EB%A3%A8" title="이노우에 가오루">이노우에 가오루</a>, <a href="/wiki/%EC%98%A4%EC%BF%A0%EB%A7%88_%EC%8B%9C%EA%B2%8C%EB%85%B8%EB%B6%80" title="오쿠마 시게노부">오쿠마 시게노부</a>, <a href="/wiki/%EC%9D%B4%ED%83%80%EA%B0%80%ED%82%A4_%EB%8B%A4%EC%9D%B4%EC%8A%A4%EC%BC%80" title="이타가키 다이스케">이타가키 다이스케</a>, <a href="/wiki/%EC%97%90%ED%86%A0_%EC%8B%A0%ED%8E%98%EC%9D%B4" title="에토 신페이">에토 신페이</a>, <a class="new" href="/w/index.php?title=%EC%98%A4%ED%82%A4_%EB%8B%A4%EC%B9%B4%ED%86%A0&amp;action=edit&amp;redlink=1" title="오키 다카토 (없는 문서)">오키 다카토</a> 등에 의해서, 차례차례로 개혁은 진행되어 갔다.</p>
<p>주 개혁으로는 학제(学制) 개혁, 지조(地租) 개정, 징병령, <a href="/wiki/%EA%B7%B8%EB%A0%88%EA%B3%A0%EB%A6%AC%EB%A0%A5" title="그레고리력">그레고리력</a> 채용, 사법 제도 정비, <a href="/wiki/%EB%8B%A8%EB%B0%9C%EB%A0%B9" title="단발령">단발령</a> 등이 있다. 다만, 이런 개혁은 급격하게 진행되었기 때문에 모순도 적지 않았고, 사족이나 농민의 불만을 사서, 후에 <a href="/wiki/%EC%A0%95%ED%95%9C%EB%A1%A0" title="정한론">정한론</a>으로 이어졌다 라고도 말해진다. 구미 사절에서 귀국한 이와쿠라나 오쿠보가 <a href="/wiki/%EC%A0%95%ED%95%9C%EB%A1%A0" title="정한론">정한론</a>을 지지하지 않았고, 더욱 더 오쿠보 밑으로 <a href="/wiki/%EC%9D%BC%EB%B3%B8_%EB%82%B4%EB%AC%B4%EC%84%B1" title="일본 내무성">내무성</a>이 설립됨으로써 여러 개혁의 정리가 이루어지게 된다.</p>
<p>또한, 같은 시기에 민간에서 이루어진 문명 개화의 움직임, 육식 보급과 철도 개통 등과 함께, 신시대 '메이지'의 분위기가 양성되었다.</p>
<p>경제 산업분야에선, <a href="/wiki/%EB%B6%80%EA%B5%AD%EA%B0%95%EB%B3%91" title="부국강병">부국강병</a>, <a href="/wiki/%EC%8B%9D%EC%82%B0%ED%9D%A5%EC%97%85" title="식산흥업">식산흥업</a>의 슬로건 아래, <a class="new" href="/w/index.php?title=%EB%8F%84%EB%AF%B8%EC%98%A4%EC%B9%B4_%EC%A0%9C%EC%82%AC%EC%9E%A5&amp;action=edit&amp;redlink=1" title="도미오카 제사장 (없는 문서)">도미오카 제사장</a>을 비롯한 관영공장이 만들어지는 등, 정부 주도의 산업 육성이 시작되어, 서양식 공업 기술이 도입되었다. 금융제도에서도 통화단위로 '<a href="/wiki/%EC%9D%BC%EB%B3%B8_%EC%97%94" title="일본 엔">엔</a>'을 도입(1871년), 국립은행 조례에 의한 국립은행(내셔널 뱅크)을 거쳐, 통화 발행권을 독점하는 <a href="/wiki/%EC%A4%91%EC%95%99%EC%9D%80%ED%96%89" title="중앙은행">중앙은행</a>으로써 <a class="mw-redirect" href="/wiki/%EC%9D%BC%EB%B3%B8_%EC%9D%80%ED%96%89" title="일본 은행">일본 은행</a>을 설립(1882년) 하는 등, 자본주의적 금융제도의 정비도 이루어졌다. 유통 분야에서는, 우편제도, 전신망의 정비, 철도 및 선박 운수(민간 우편 기선 미쓰비시 회사와 국책 회사인 공동 운수 회사의 경합을 거쳐 <a class="new" href="/w/index.php?title=%EC%9D%BC%EB%B3%B8_%EC%9C%A0%EC%84%BC&amp;action=edit&amp;redlink=1" title="일본 유센 (없는 문서)">일본 유센</a> 회사가 되었다.) 등의 정비가 이루어졌다. 이러한 자본 활동에는, 실직한 대신 녹봉을 얻은 <a href="/wiki/%ED%99%94%EC%A1%B1" title="화족">화족</a>의 자산에 의한 투자 활동도 배경에 있었다.</p>
<p>이러한 개혁에는 적극적으로 서양 문명의 선진 제도가 받아들여지는 과정에서, '<a href="/wiki/%EC%99%B8%EA%B5%AD%EC%9D%B8_%EC%B4%88%EB%B9%99%EC%82%AC" title="외국인 초빙사">오야토이 외국인</a>'이라고 불리는 외국인이, 기술 지도, 교육 분야, 관제, 군제 정비 등 여러가지 분야에서 고용되어 근대국가 건설을 도왔다.</p>
<h3><span class="mw-headline" id=".EC.A2.85.EA.B5.90.EC.A0.95.EC.B1.85">종교정책</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%A9%94%EC%9D%B4%EC%A7%80_%EC%9C%A0%EC%8B%A0&amp;action=edit&amp;section=7" title="부분 편집: 종교정책">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<h4><span class="mw-headline" id=".EB.B6.88.EA.B5.90">불교</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%A9%94%EC%9D%B4%EC%A7%80_%EC%9C%A0%EC%8B%A0&amp;action=edit&amp;section=8" title="부분 편집: 불교">편집</a><span class="mw-editsection-bracket">]</span></span></h4>
<p>종교적으로는, 고대의 제정일치로 돌아가는 개혁인 점에서, <a href="/wiki/1867%EB%85%84" title="1867년">1867년</a> 음력 1월 17일에 제정된 직제에는 <a href="/wiki/%ED%95%98%EB%8A%90%EB%8B%98" title="하느님">하느님</a>을 7가지 필두에 두어, <a href="/wiki/%EC%9D%8C%EB%A0%A5_3%EC%9B%94" title="음력 3월">음력 3월</a>에는 <a class="new" href="/w/index.php?title=%EC%8B%A0%EB%B6%88%EB%B6%84%EB%A6%AC&amp;action=edit&amp;redlink=1" title="신불분리 (없는 문서)">신불분리령</a>이 내려졌다. 신불분리령의 주 의미는 불교 배척이 아닌, 에도 시대까지의 <a class="new" href="/w/index.php?title=%EC%8B%A0%EB%B6%88%EC%A0%88%EC%B6%A9&amp;action=edit&amp;redlink=1" title="신불절충 (없는 문서)">신불절충</a>에서 불교와 신도의 혼합에서 양자를 분리하는 것이었지만, 당시의 복고적 시운은 <a href="/wiki/%EB%B6%88%EA%B5%90" title="불교">불교</a>조차도 외래의 종교로서 강하게 배척하는 <a href="/wiki/%ED%8F%90%EB%B6%88%ED%9B%BC%EC%84%9D" title="폐불훼석">폐불훼석</a>으로 향했다.</p>
<h4><span class="mw-headline" id=".EA.B8.B0.EB.8F.85.EA.B5.90">기독교</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%A9%94%EC%9D%B4%EC%A7%80_%EC%9C%A0%EC%8B%A0&amp;action=edit&amp;section=9" title="부분 편집: 기독교">편집</a><span class="mw-editsection-bracket">]</span></span></h4>
<p>1612년 기독교 금령에 의해 막부의 질서에 반대하는 반국가적 종교로 규정되어 탄압받던 <a href="/wiki/%EA%B8%B0%EB%8F%85%EA%B5%90" title="기독교">기독교</a>는<sup class="reference" id="cite_ref-7"><a href="#cite_note-7">[7]</a></sup> 메이지 정부에 의해서도 계속 금지되어 기독교 지도자 총 140명은 <a href="/wiki/%ED%95%98%EA%B8%B0_%EC%8B%9C" title="하기 시">하기</a>（66명）, <a href="/wiki/%EC%93%B0%EC%99%80%EB%85%B8_%EC%A0%95" title="쓰와노 정">쓰와노</a>（28명）, <a href="/wiki/%ED%9B%84%EC%BF%A0%EC%95%BC%EB%A7%88_%EC%8B%9C" title="후쿠야마 시">후쿠야마</a>（20명）로 나뉘어 강제로 이주되었다. 그 후, 메이지 2년（<a href="/wiki/1869%EB%85%84" title="1869년">1869년</a>）12월 7일에는, 기독교인 약 3,000명을 <a class="mw-redirect" href="/wiki/%EA%B0%80%EB%82%98%EC%9E%90%EC%99%80_%EB%B2%88" title="가나자와 번">가나자와</a> 및 10개의 번으로 분산 이주시켰다. 하지만 메이지 4년（<a href="/wiki/1871%EB%85%84" title="1871년">1871년</a>）11월, <a href="/wiki/%EC%9D%B4%EC%99%80%EC%BF%A0%EB%9D%BC_%EB%8F%84%EB%AA%A8%EB%AF%B8" title="이와쿠라 도모미">이와쿠라 도모미</a>특명 전권 신도 국교화 정책과 관련해서, 기독교 금지를 해제해도 즉시 구미가 조약개정에는 응하지 않을 것이라는 회의적인 생각에 근거한 정부내의 보호파의 반대뿐만 아니라, 종교계나 일반 민중에서도 '사종문(邪宗門)<sup class="reference" id="cite_ref-8"><a href="#cite_note-8">[8]</a></sup>' 금지 해제에는 반대하는 목소리가 강해 분규(紛糾)하였으나, <a href="/wiki/1873%EB%85%84" title="1873년">1873년</a> 2월 24일 금지 고찰(高札)<sup class="reference" id="cite_ref-9"><a href="#cite_note-9">[9]</a></sup> 을 제거, 그 뜻을 각 구니에 통고, 이주된 기독교도들이 마을로 돌아오자 드디어 종결되었다.</p>

<p>신정부의, 최대 목표는 구미 열강을 따라 잡는 것이기 때문에, 구 막부 시대에 체결된 <b>불평등 조약</b> 개정이 급선무로 여겨졌다. <a href="/wiki/%EC%9D%B4%EC%99%80%EC%BF%A0%EB%9D%BC_%EC%82%AC%EC%A0%88%EB%8B%A8" title="이와쿠라 사절단">이와쿠라 사절단</a>은 서구 여러 제도 조사도 목적이었지만, 조약 개정을 위한 사전 준비라는 면도 있었다. 실제로 교섭도 준비되었지만, 일본을 근대국가라고 보지 않는 구미제국은 상대가 되지 않았고, 아직 시기 상조였다. 그 때문에, 우선 국내정책의 측면에서 국민들이 가진 외국인에 대한 반감을 제거하기 위해 외국인에 대한 공격을 법령으로 금지시키고, 만국공법을 수용하였으며 외국인이 국내여행을 자유롭게 할 수 있도록 조치<sup class="reference" id="cite_ref-park_10-0"><a href="#cite_note-park-10">[10]</a></sup>, 유럽화 정책 등여러 정책을 시도했지만, 조약 개정 자체는 반세기에 미치는 부단한 노력을 필요로 했다.</p>
<h2><span class="mw-headline" id=".EA.B0.9C.ED.98.81.EC.9D.98_.EC.98.81.ED.96.A5.EA.B3.BC_.EB.B9.84.ED.8C.90">개혁의 영향과 비판</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%A9%94%EC%9D%B4%EC%A7%80_%EC%9C%A0%EC%8B%A0&amp;action=edit&amp;section=11" title="부분 편집: 개혁의 영향과 비판">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<ul>
<li>주로 구 사쓰마 번, 조슈 번 및 일부 <a class="mw-redirect" href="/wiki/%EA%B5%AC%EA%B2%8C" title="구게">구게</a>에 의한 전제 정치였기 때문에 '번 파벌 정부'라고 야유를 받았다. 하지만 중급 관료 이상에 구 <a href="/wiki/%EC%8B%A0%ED%8C%90_%EB%8B%A4%EC%9D%B4%EB%AC%98" title="신판 다이묘">신판</a> 번, 구 막부 신하 였던 자도 적지 않았기 때문에, 일부 세력이 주도했다고는 하기 어렵다.</li>
<li>메이지 유신의 개혁은 새로운 제도로 인한 모순을 낳기도 했지만, 단기간에 입헌제도 달성에 성공하였기 때문에 아시아에서는 메이지 유신을 모범으로 개혁, 독립 운동을 실시하려고 하였다. 다만 대부분이 형태만의 개혁이나 일면만 파악한 '위에서 부터의 개혁'으로 시종하여, 성공한 예는 많지 않다.</li>
<li>전통주의자들은 신성한 <a href="/wiki/%EC%9D%BC%EB%B3%B8_%EC%B2%9C%ED%99%A9" title="일본 천황">천황제</a>가 현대 문명에 양보를 강요당하는 문제에 대해 분개하고 있었다.</li>
</ul>
