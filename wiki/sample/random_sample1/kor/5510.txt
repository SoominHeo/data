<p><b>메클렌부르크-슈트렐리츠 대공국</b>(Großherzogtum Mecklenburg-Strelitz)은 <a href="/wiki/1815%EB%85%84" title="1815년">1815년</a>부터 <a href="/wiki/1918%EB%85%84" title="1918년">1918년</a>까지 독일 북부에 있었던 대공국이었다. <a href="/wiki/1918%EB%85%84" title="1918년">1918년</a>부터 <a href="/wiki/1934%EB%85%84" title="1934년">1934년</a>까지는 주(州)로 기능하다가 <a href="/wiki/1934%EB%85%84" title="1934년">1934년</a> <a class="mw-redirect" href="/wiki/%EB%82%98%EC%B9%98" title="나치">나치</a>에 의해 <a class="mw-redirect" href="/wiki/%EB%A9%94%ED%81%B4%EB%A0%8C%EB%B6%80%EB%A5%B4%ED%81%AC-%EC%8A%88%EB%B2%A0%EB%A6%B0" title="메클렌부르크-슈베린">메클렌부르크-슈베린</a>와 통합되어 메클렌부르크주(州)가 되었다. 현재는 <a href="/wiki/%EB%A9%94%ED%81%B4%EB%A0%8C%EB%B6%80%EB%A5%B4%ED%81%AC%ED%8F%AC%EC%96%B4%ED%8F%AC%EB%A9%94%EB%A5%B8_%EC%A3%BC" title="메클렌부르크포어포메른 주">메클렌부르크포어포메른 주</a>(州)의 일부가 되어 있다. 수도는 <a class="new" href="/w/index.php?title=%EB%85%B8%EC%9D%B4%EC%8A%88%ED%8A%B8%EB%A0%90%EB%A6%AC%EC%B8%A0&amp;action=edit&amp;redlink=1" title="노이슈트렐리츠 (없는 문서)">노이슈트렐리츠</a>에 두고 있었으며, 국교는 루터파 <a class="mw-redirect" href="/wiki/%ED%94%84%EB%A1%9C%ED%85%8C%EC%8A%A4%ED%83%84%ED%8A%B8" title="프로테스탄트">프로테스탄트</a>였다.</p>

<h2><span class="mw-headline" id=".EC.97.AD.EB.8C.80_.EA.B5.B0.EC.A3.BC">역대 군주</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%A9%94%ED%81%B4%EB%A0%8C%EB%B6%80%EB%A5%B4%ED%81%AC%EC%8A%88%ED%8A%B8%EB%A0%90%EB%A6%AC%EC%B8%A0_%EB%8C%80%EA%B3%B5%EA%B5%AD&amp;action=edit&amp;section=1" title="부분 편집: 역대 군주">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>두 메클렌부르크 대공국은 <a href="/wiki/%EC%84%9C%EC%8A%AC%EB%9D%BC%EB%B8%8C%EC%A1%B1" title="서슬라브족">서슬라브족</a>의 혈통이나 게르만화(化)된 오보트리텐(Obodriten) 대공가의 대공국들이었다.</p>
<h3><span class="mw-headline" id=".EB.A9.94.ED.81.B4.EB.A0.8C.EB.B6.80.EB.A5.B4.ED.81.AC.EC.8A.88.ED.8A.B8.EB.A0.90.EB.A6.AC.EC.B8.A0_.EA.B3.B5">메클렌부르크슈트렐리츠 공</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%A9%94%ED%81%B4%EB%A0%8C%EB%B6%80%EB%A5%B4%ED%81%AC%EC%8A%88%ED%8A%B8%EB%A0%90%EB%A6%AC%EC%B8%A0_%EB%8C%80%EA%B3%B5%EA%B5%AD&amp;action=edit&amp;section=2" title="부분 편집: 메클렌부르크슈트렐리츠 공">편집</a><span class="mw-editsection-bracket">]</span></span></h3>

<li><a href="/wiki/1708%EB%85%84" title="1708년">1708년</a> - <a href="/wiki/1752%EB%85%84" title="1752년">1752년</a> 아돌프 프리드리히 3세 (1686-1752)</li>

<h3><span class="mw-headline" id=".EB.A9.94.ED.81.B4.EB.A0.8C.EB.B6.80.EB.A5.B4.ED.81.AC.EC.8A.88.ED.8A.B8.EB.A0.90.EB.A6.AC.EC.B8.A0_.EB.8C.80.EA.B3.B5">메클렌부르크슈트렐리츠 대공</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%A9%94%ED%81%B4%EB%A0%8C%EB%B6%80%EB%A5%B4%ED%81%AC%EC%8A%88%ED%8A%B8%EB%A0%90%EB%A6%AC%EC%B8%A0_%EB%8C%80%EA%B3%B5%EA%B5%AD&amp;action=edit&amp;section=3" title="부분 편집: 메클렌부르크슈트렐리츠 대공">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<ul>
<li><a href="/wiki/1815%EB%85%84" title="1815년">1815년</a> - <a href="/wiki/1816%EB%85%84" title="1816년">1816년</a> 카를 2세 (1741-1816)</li>
<li><a href="/wiki/1816%EB%85%84" title="1816년">1816년</a> - <a href="/wiki/1860%EB%85%84" title="1860년">1860년</a> 게오르크 빌헬름 (1779-1860)</li>
<li><a href="/wiki/1860%EB%85%84" title="1860년">1860년</a> - <a href="/wiki/1904%EB%85%84" title="1904년">1904년</a> 프리드리히 빌헬름 (1819-1904)</li>
<li><a href="/wiki/1904%EB%85%84" title="1904년">1904년</a> - <a href="/wiki/1914%EB%85%84" title="1914년">1914년</a> 아돌프 프리드리히 5세 (1848-1914)</li>
<li><a href="/wiki/1914%EB%85%84" title="1914년">1914년</a> - <a href="/wiki/1918%EB%85%84" title="1918년">1918년</a> 아돌프 프리드리히 6세 (1882-1918)</li>
</ul>
<p><a href="/wiki/1918%EB%85%84" title="1918년">1918년</a> <a href="/wiki/2%EC%9B%94_23%EC%9D%BC" title="2월 23일">2월 23일</a> 대공 아돌프 프리드리히 6세가 자살하여 대공위 계승에 위기가 닥쳤다. 따라서 비어 있는 대공위를 이웃 <a href="/wiki/%EB%A9%94%ED%81%B4%EB%A0%8C%EB%B6%80%EB%A5%B4%ED%81%AC%EC%8A%88%EB%B2%A0%EB%A6%B0_%EB%8C%80%EA%B3%B5%EA%B5%AD" title="메클렌부르크슈베린 대공국">메클렌부르크슈베린 대공국</a>의 <a class="new" href="/w/index.php?title=%EB%A9%94%ED%81%B4%EB%A0%8C%EB%B6%80%EB%A5%B4%ED%81%AC%EC%8A%88%EB%B2%A0%EB%A6%B0%EC%9D%98_%ED%94%84%EB%A6%AC%EB%93%9C%EB%A6%AC%ED%9E%88_%ED%94%84%EB%9E%80%EC%B8%A0_4%EC%84%B8&amp;action=edit&amp;redlink=1" title="메클렌부르크슈베린의 프리드리히 프란츠 4세 (없는 문서)">프리드리히 프란츠 4세</a>가 독일 제국이 붕괴할 때까지 섭정이 되어 다스렸다.</p>
<p>한편 아돌프 프리드리히 6세의 유일한 남자 혈육인 사촌 <b>카를 미하엘</b> 공은 어머니 예카테리나가 러시아 황족이었기 때문에 러시아 제국군에 복무하였고, <a href="/wiki/1914%EB%85%84" title="1914년">1914년</a> 메클렌부르크 시민권을 포기하고 러시아에 귀화하였다. 그러나 그가 유일한 메클렌부르크-슈트렐리츠 대공위의 유일한 상속자가 되었다. 카를 미하엘 공은 <a href="/wiki/1928%EB%85%84" title="1928년">1928년</a>에 형인 게오르크 알렉산더 공의 귀천상혼으로 태어난 조카 게오르크(카를로프Carlow 백작)을 입양했으며, 이에 따라 게오르크는 전하(Serene Highness)의 경칭을 받았다. 한편 이 입양은 이듬해에 <a class="mw-redirect" href="/wiki/%EB%A1%9C%EB%A7%88%EB%85%B8%ED%94%84_%EC%99%95%EA%B0%80" title="로마노프 왕가">로마노프 왕가</a>의 수장 <a href="/wiki/%ED%82%A4%EB%A6%B4_%EB%B8%94%EB%9D%BC%EB%94%94%EB%AF%B8%EB%A1%9C%EB%B9%84%EC%B9%98_%EB%A1%9C%EB%A7%88%EB%85%B8%ED%94%84" title="키릴 블라디미로비치 로마노프">키릴 블라디미로비치 로마노프</a>와 메클렌부르크슈베린 대공가의 수장 프리드리히 프란츠 4세의 승인을 받았다.</p>
<h3><span class="mw-headline" id=".EB.A9.94.ED.81.B4.EB.A0.8C.EB.B6.80.EB.A5.B4.ED.81.AC.EC.8A.88.ED.8A.B8.EB.A0.90.EB.A6.AC.EC.B8.A0_.EB.8C.80.EA.B3.B5.EA.B0.80_.EC.88.98.EC.9E.A5">메클렌부르크슈트렐리츠 대공가 수장</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%A9%94%ED%81%B4%EB%A0%8C%EB%B6%80%EB%A5%B4%ED%81%AC%EC%8A%88%ED%8A%B8%EB%A0%90%EB%A6%AC%EC%B8%A0_%EB%8C%80%EA%B3%B5%EA%B5%AD&amp;action=edit&amp;section=4" title="부분 편집: 메클렌부르크슈트렐리츠 대공가 수장">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<ul>
<li><a href="/wiki/1934%EB%85%84" title="1934년">1934년</a>(1950년) - <a href="/wiki/1963%EB%85%84" title="1963년">1963년</a> 게오르크</li>
</ul>
<p><a href="/wiki/1934%EB%85%84" title="1934년">1934년</a> 게오르크 공은 메클렌부르크슈트렐리츠 대공가의 수장이 되었고, <a href="/wiki/1950%EB%85%84" title="1950년">1950년</a> 메클렌부르크슈베린 대공가도 그것을 완전히 승인하였다. 아울러 카를로츠 백작 작위는 폐기되었다. 한편 게오르크는 프로테스탄트에서 가톨릭으로 개종하였다.</p>
<ul>
<li><a href="/wiki/1963%EB%85%84" title="1963년">1963년</a> - <a href="/wiki/1996%EB%85%84" title="1996년">1996년</a> 게오르크 알렉산더</li>
<li><a href="/wiki/1996%EB%85%84" title="1996년">1996년</a> - 현재 보르빈</li>
</ul>
<p><a href="/wiki/2001%EB%85%84" title="2001년">2001년</a> 메클렌부르크슈베린 대공가의 남계 자손이 단절됨에 따라, 메클렌부르크슈트렐리츠 대공가는 메클렌부르크 가문의 유일한 계통이 되었다. 보르빈은 2남 1녀를 두고 있다.</p>
