<p><b>메토디우스</b> 또는 <b>메토디오스</b> (<a href="/wiki/%EA%B7%B8%EB%A6%AC%EC%8A%A4%EC%96%B4" title="그리스어">그리스어</a>: Μεθόδιος , <a href="/wiki/%EA%B5%90%ED%9A%8C_%EC%8A%AC%EB%9D%BC%EB%B8%8C%EC%96%B4" title="교회 슬라브어">교회 슬라브어</a>: Мефодии, <a href="/wiki/826%EB%85%84" title="826년">826년</a> - <a href="/wiki/885%EB%85%84" title="885년">885년</a> <a href="/wiki/4%EC%9B%94_6%EC%9D%BC" title="4월 6일">4월 6일</a>)는 <a href="/wiki/%EB%B9%84%EC%9E%94%ED%8B%B0%EC%9B%80_%EC%A0%9C%EA%B5%AD" title="비잔티움 제국">비잔티움 제국</a>의 <a href="/wiki/%EA%B8%B0%EB%8F%85%EA%B5%90" title="기독교">기독교</a> 신학자이자 선교사이며 <a class="new" href="/w/index.php?title=%EB%8C%80_%EB%AA%A8%EB%9D%BC%EB%B9%84%EC%95%84&amp;action=edit&amp;redlink=1" title="대 모라비아 (없는 문서)">대 모라비아</a>의 <a href="/wiki/%EB%8C%80%EC%A3%BC%EA%B5%90" title="대주교">대주교</a>였다. 동생인 선교사 <a class="mw-disambig" href="/wiki/%ED%82%A4%EB%A6%B4%EB%A1%9C%EC%8A%A4" title="키릴로스">키릴루스</a>와 함께 <a href="/wiki/%EC%8A%AC%EB%9D%BC%EB%B8%8C%EC%A1%B1" title="슬라브족">슬라브족</a>에 대한 선교 사업에 나서 슬라브족을 <a href="/wiki/%EA%B8%B0%EB%8F%85%EA%B5%90" title="기독교">기독교화</a>하는데 큰 역할을 한 것으로 유명하며, 특히 슬라브족의 선교를 위해 <a href="/wiki/%ED%82%A4%EB%A6%B4_%EB%AC%B8%EC%9E%90" title="키릴 문자">키릴 문자</a>를 발명했다고 전해진다. 동생과 함께 <a href="/wiki/%EB%A1%9C%EB%A7%88_%EA%B0%80%ED%86%A8%EB%A6%AD%EA%B5%90%ED%9A%8C" title="로마 가톨릭교회">로마 가톨릭교회</a>와 <a href="/wiki/%EB%8F%99%EB%B0%A9_%EC%A0%95%EA%B5%90%ED%9A%8C" title="동방 정교회">동방 정교회</a>에서 모두 <a href="/wiki/%EC%84%B1%EC%9D%B8_(%EC%A2%85%EA%B5%90)" title="성인 (종교)">성인</a>으로 추대되었기 때문에 <b>성 메토디우스</b>라고도 한다. 동방 정교회에서의 축일은 <a href="/wiki/2%EC%9B%94_14%EC%9D%BC" title="2월 14일">2월 14일</a>이며, 로마 가톨릭교회에서의 축일은 <a href="/wiki/5%EC%9B%94_11%EC%9D%BC" title="5월 11일">5월 11일</a>이다. <a href="/wiki/%EC%84%B1%EA%B3%B5%ED%9A%8C" title="성공회">성공회</a>에서도 기존에 기념하던 성인들을 <a href="/wiki/%EC%A2%85%EA%B5%90_%EA%B0%9C%ED%98%81" title="종교 개혁">종교 개혁</a> 이후에도 줄곧 기념하기 때문에, 성 키릴루스와 메토디우스 형제도 <a href="/wiki/%EC%84%B1%EC%9D%B8_(%EC%A2%85%EA%B5%90)" title="성인 (종교)">성인</a>으로 인정받는다.</p>

<h2><span class="mw-headline" id=".EC.83.9D.EC.95.A0">생애</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%A9%94%ED%86%A0%EB%94%94%EC%9A%B0%EC%8A%A4&amp;action=edit&amp;section=1" title="부분 편집: 생애">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<h3><span class="mw-headline" id=".EC.8A.AC.EB.9D.BC.EB.B8.8C_.EC.84.A0.EA.B5.90_.EC.82.AC.EC.97.85">슬라브 선교 사업</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%A9%94%ED%86%A0%EB%94%94%EC%9A%B0%EC%8A%A4&amp;action=edit&amp;section=2" title="부분 편집: 슬라브 선교 사업">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>키릴루스와 메토디우스는 <a href="/wiki/%EA%B7%B8%EB%A6%AC%EC%8A%A4" title="그리스">그리스</a> <a href="/wiki/%ED%85%8C%EC%82%B4%EB%A1%9C%EB%8B%88%ED%82%A4" title="테살로니키">테살로니키</a>의 <a href="/wiki/%EB%B9%84%EC%9E%94%ED%8B%B0%EC%9B%80_%EC%A0%9C%EA%B5%AD" title="비잔티움 제국">비잔티움 제국</a>의 해군 제독의 집안에서 태어났다. 15살에 부모를 여의고 고아가 되었으나 일찍부터 관직에 나갔다. 그는 슬라브족이 많은 지역에서 근무하였기 때문에 슬라브어에 능했는데 관직에서 물러나 명상의 삶을 살기로 결심하고 <a href="/wiki/%EB%B9%84%ED%8B%B0%EB%8B%88%EC%95%84" title="비티니아">비티니아</a>의 올림푸스 산에 있는 수도원에서 은거하면서 살았다.</p>
<p><a href="/wiki/862%EB%85%84" title="862년">862년</a> <a class="new" href="/w/index.php?title=%EB%8C%80_%EB%AA%A8%EB%9D%BC%EB%B9%84%EC%95%84&amp;action=edit&amp;redlink=1" title="대 모라비아 (없는 문서)">대 모라비아</a>의 군주 <a class="new" href="/w/index.php?title=%EB%A1%9C%EC%8A%A4%ED%8B%B0%EC%8A%AC%EB%9D%BC%ED%94%84&amp;action=edit&amp;redlink=1" title="로스티슬라프 (없는 문서)">로스티슬라프</a>가 황제 <a href="/wiki/%EB%AF%B8%EC%B9%B4%EC%97%98_3%EC%84%B8" title="미카엘 3세">미카일 3세</a>와 <a class="mw-redirect" href="/wiki/%EC%BD%98%EC%8A%A4%ED%83%84%ED%8B%B0%EB%85%B8%ED%8F%B4%EB%A6%AC%EC%8A%A4_%EC%84%B8%EA%B3%84%EC%B4%9D%EB%8C%80%EC%A3%BC%EA%B5%90" title="콘스탄티노폴리스 세계총대주교">콘스탄티노폴리스 총대주교</a> <a class="mw-redirect" href="/wiki/%ED%8F%AC%ED%8B%B0%EC%9A%B0%EC%8A%A4" title="포티우스">포티우스</a>에게 사절을 보내 기독교로 개종을 요청하였다. 사실 로스티슬라프의 요청은 종교적이라기보다는 정치적인 것이었다. 그는 북서부의 국경에서 프랑크족의 압박을 받았는데 <a href="/wiki/%EB%A3%A8%ED%8A%B8%EB%B9%84%ED%9E%88_2%EC%84%B8_(%EB%8F%99%ED%94%84%EB%9E%91%ED%81%AC)" title="루트비히 2세 (동프랑크)">프랑크의 루트비히 2세</a>가 <a href="/wiki/%EB%B3%B4%EB%A6%AC%EC%8A%A4_1%EC%84%B8" title="보리스 1세">불가리아의 보리스 1세</a>와 동맹을 맺자 자신도 동맹 세력이 필요했던 것이다. 이는 항상 콘스탄티노폴리스 교회의 영향력 확대를 바라는 황제와 포티우스에게 절호의 기회였기 때문에 언어에 뛰어난 키릴루스가 선정되었고 <a href="/wiki/864%EB%85%84" title="864년">864년</a> 초여름 키릴루스는 수도원에 은거하던 형 메토디우스를 불러내어 형제는 모라비아로 떠났다. 그곳에서 형제는 슬라브어로 <a href="/wiki/%EC%84%B1%EA%B2%BD" title="성경">성서</a>를 번역하며 3년 반 동안 슬라브족의 선교에 헌신했다.</p>
<p>일설에 따르면 키릴루스는 그 때까지 문자가 없었던 <a href="/wiki/%EC%8A%AC%EB%9D%BC%EB%B8%8C%EC%96%B4%ED%8C%8C" title="슬라브어파">슬라브어</a>를 표현하기 위해 새 문자를 발명해서 성서와 일부 기도문을 번역했는데 그가 사용한 언어는 마케도니아식 슬라브어였다고 한다. 모라비아인들이 쓰는 언어와는 달랐기 때문에 모라비아 선교는 실패했지만 나중에 이 문자들은 슬라브족의 언어발달에 지대한 공헌을 했다. 예전에는 키릴루스가 고안한 문자는 현대의 키릴 문자가 아니라 더 조악하고 오래전에 사멸한 <a href="/wiki/%EA%B8%80%EB%9D%BC%EA%B3%A8_%EB%AC%B8%EC%9E%90" title="글라골 문자">글라골 문자</a>라고 생각했으나 최근의 연구에 의해 부정되었다.<sup class="reference" id="cite_ref-1"><a href="#cite_note-1">[1]</a></sup></p>
<h3><span class="mw-headline" id=".EB.A1.9C.EB.A7.88.EC.97.90.EC.84.9C_.EC.A3.BC.EA.B5.90.EA.B0.80_.EB.90.98.EB.8B.A4">로마에서 주교가 되다</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%A9%94%ED%86%A0%EB%94%94%EC%9A%B0%EC%8A%A4&amp;action=edit&amp;section=3" title="부분 편집: 로마에서 주교가 되다">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<div class="thumb tleft">
<div class="thumbinner" style="width:202px;"><a class="image" href="/wiki/%ED%8C%8C%EC%9D%BC:Holy_Trinity_Column-Saint_Methodius.jpg"><img alt="" class="thumbimage" data-file-height="2304" data-file-width="1728" height="267" src="//upload.wikimedia.org/wikipedia/commons/thumb/b/b6/Holy_Trinity_Column-Saint_Methodius.jpg/200px-Holy_Trinity_Column-Saint_Methodius.jpg" srcset="//upload.wikimedia.org/wikipedia/commons/thumb/b/b6/Holy_Trinity_Column-Saint_Methodius.jpg/300px-Holy_Trinity_Column-Saint_Methodius.jpg 1.5x, //upload.wikimedia.org/wikipedia/commons/thumb/b/b6/Holy_Trinity_Column-Saint_Methodius.jpg/400px-Holy_Trinity_Column-Saint_Methodius.jpg 2x" width="200"/></a>
<div class="thumbcaption">
<div class="magnify"><a class="internal" href="/wiki/%ED%8C%8C%EC%9D%BC:Holy_Trinity_Column-Saint_Methodius.jpg" title="실제 크기로"></a></div>
메토디우스의 동상(<a class="mw-redirect" href="/wiki/%EC%98%AC%EB%A1%9C%EB%AA%A8%EC%B8%A0" title="올로모츠">올로모츠</a>의 성 <a href="/wiki/%EC%82%BC%EC%9C%84%EC%9D%BC%EC%B2%B4" title="삼위일체">삼위일체</a> 기둥, <a href="/wiki/%EC%B2%B4%EC%BD%94" title="체코">체코</a>)</div>
</div>
</div>
<p>당시 <a href="/wiki/%EC%9E%98%EC%B8%A0%EB%B6%80%EB%A5%B4%ED%81%AC" title="잘츠부르크">잘츠부르크</a> <a href="/wiki/%EB%8C%80%EC%A3%BC%EA%B5%90" title="대주교">대주교</a> 테오트마르와 <a href="/wiki/%ED%8C%8C%EC%82%AC%EC%9A%B0" title="파사우">파사우</a> <a href="/wiki/%EC%A3%BC%EA%B5%90" title="주교">주교</a>는 메토디우스 형제의 모라비아 선교사업에 대해 불만을 품었고 그들의 슬라브어 전례를 베풀고 관할하는 것에 대해 교황에게 항의 하였다. <a href="/wiki/867%EB%85%84" title="867년">867년</a> <a href="/wiki/%EA%B5%90%ED%99%A9_%EB%8B%88%EC%BD%9C%EB%9D%BC%EC%98%A4_1%EC%84%B8" title="교황 니콜라오 1세">교황 니콜라오 1세</a>는 키릴루스와 메토디우스 형제를 로마로 초청하여 분쟁을 해결하려고 하였다. 그러나 니콜라오 교황은 11월 13일에 갑작스럽게 선종하였고 이듬해 키릴루스와 메토디우스는 <a href="/wiki/%EB%A1%9C%EB%A7%88" title="로마">로마</a>에 도착하여 당시 로마의 새로운 교황인 <a href="/wiki/%EA%B5%90%ED%99%A9_%ED%95%98%EB%93%9C%EB%A6%AC%EC%95%84%EB%85%B8_2%EC%84%B8" title="교황 하드리아노 2세">교황 하드리아노 2세</a>에게 자신들의 입장을 설명하였다. 새로운 교황은 그들의 입장에 동조해 슬라브식 전례를 사용하는 것을 공식적으로 승인했다.</p>
<p><a href="/wiki/869%EB%85%84" title="869년">869년</a> 동생 키릴루스는 병으로 로마에서 죽었고 교황 하드리아노는 메토디우스를 로마 교황 사절 겸 <a class="new" href="/w/index.php?title=%EC%8B%9C%EB%A5%B4%EB%AF%B8%EC%9B%80&amp;action=edit&amp;redlink=1" title="시르미움 (없는 문서)">시르미움</a>의 대주교로 임명하여 슬라브족에게 파송했다. 모라비아와 판노니아는 독일계 교구의 관할에서 벗어나 메토디우스의 관할로 넘어가게 되었고 계속 슬라브족에 대한 선교사업을 벌였다. 그러나 대 모라비아의 <a class="new" href="/w/index.php?title=%EB%A1%9C%EC%8A%A4%ED%8B%B0%EC%8A%AC%EB%9D%BC%ED%94%84&amp;action=edit&amp;redlink=1" title="로스티슬라프 (없는 문서)">로스티슬라프</a>의 조카이며 후계자인 스바토플루크가 더 이상 메토디우스를 후원하지 않게 되었다.</p>
<h3><span class="mw-headline" id=".EB.8F.85.EC.9D.BC_.EA.B5.90.EA.B5.AC.EC.99.80.EC.9D.98_.EB.B6.84.EC.9F.81">독일 교구와의 분쟁</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%A9%94%ED%86%A0%EB%94%94%EC%9A%B0%EC%8A%A4&amp;action=edit&amp;section=4" title="부분 편집: 독일 교구와의 분쟁">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>그는 <a href="/wiki/870%EB%85%84" title="870년">870년</a> 다시 한번 교회의 관할권을 놓고 독일의 <a href="/wiki/%EC%84%B1%EC%A7%81%EC%9E%90" title="성직자">성직자</a>들과 분쟁을 벌였는데 <a href="/wiki/%EB%A3%A8%ED%8A%B8%EB%B9%84%ED%9E%88_2%EC%84%B8_(%EB%8F%99%ED%94%84%EB%9E%91%ED%81%AC)" title="루트비히 2세 (동프랑크)">독일왕 루트비히</a>와 독일인 성직자들은 모라비아에 대한 영향력을 강화하려고 메토디우스를 재판에 회부하여 가혹행위를 가했으며 <a href="/wiki/%EC%8A%88%EB%B0%94%EB%B2%A4" title="슈바벤">슈바벤</a>에 있는 감옥에 가두었다.</p>
<p>메토디우스는 슈바벤에서 유배되어 있다가 <a href="/wiki/%EA%B5%90%ED%99%A9_%EC%9A%94%ED%95%9C_8%EC%84%B8" title="교황 요한 8세">교황 요한 8세</a>의 중재로 석방되어 다시 교구로 돌아갔다. 그러나 독일의 성직자들은 계속 그를 공격했고 급기야 <a href="/wiki/880%EB%85%84" title="880년">880년</a> 메토디우스는 슬라브어로 전례를 베푸는 문제로 <a href="/wiki/%EC%9D%B4%EB%8B%A8" title="이단">이단</a>으로 고발되어 <a href="/wiki/%EB%A1%9C%EB%A7%88" title="로마">로마</a>로 소환되었다. 그러나, 요한 8세에게 뛰어난 변론과 열의로 그의 호의를 받아 슬라브어 전례를 사용해도 된다는 승인을 얻었다. 그러나 자신의 보좌 주교인 비킹이 계속 자신의 권위에 도전하자 메토디우스는 882년 <a href="/wiki/%EC%BD%98%EC%8A%A4%ED%83%84%ED%8B%B0%EB%85%B8%ED%8F%B4%EB%A6%AC%EC%8A%A4" title="콘스탄티노폴리스">콘스탄티노폴리스</a>를 방문하여 콘스탄티노폴리스 교회 내에서 자신의 입장을 강화하려고 시도했다.</p>
<p>메토디우스는 <a href="/wiki/885%EB%85%84" title="885년">885년</a> <a href="/wiki/4%EC%9B%94_8%EC%9D%BC" title="4월 8일">4월 8일</a> 모라비아에서 죽었고 그의 사후 <a href="/wiki/%EA%B5%90%ED%99%A9_%EC%8A%A4%ED%85%8C%ED%8C%8C%EB%85%B8_5%EC%84%B8" title="교황 스테파노 5세">교황 스테파노 5세</a>는 예배에서 슬라브어 사용을 금지했고, 메토디우스의 제자로 그의 뒤를 이은 비킹은 키릴루스와 메토디우스의 제자들을 추방했다.</p>
<p>메토디우스와 키릴루스 형제는 슬라브 선교의 공을 인정받아 동방정교회로부터 즉각 <a href="/wiki/%EC%84%B1%EC%9D%B8_(%EC%A2%85%EA%B5%90)" title="성인 (종교)">성인</a>으로 시성되었다. 로마 가톨릭에서는 <a href="/wiki/1880%EB%85%84" title="1880년">1880년</a>에 이르러서야 그들을 성인으로 추대하였고 <a href="/wiki/1985%EB%85%84" title="1985년">1985년</a> <a href="/wiki/%EA%B5%90%ED%99%A9_%EC%9A%94%ED%95%9C_%EB%B0%94%EC%98%A4%EB%A1%9C_2%EC%84%B8" title="교황 요한 바오로 2세">교황 요한 바오로 2세</a>는 그들을 <a href="/wiki/%EC%9C%A0%EB%9F%BD" title="유럽">유럽</a>의 수호성인으로 선포하였다.</p>
