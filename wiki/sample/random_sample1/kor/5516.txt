<h1 class="firstHeading" id="firstHeading" lang="ko">메톤 주기</h1>
<div class="mw-body-content" id="bodyContent">
<div id="siteSub">위키백과, 우리 모두의 백과사전.</div>

<p><b>메톤 주기</b>(Metonic cycle)는 <a href="/wiki/%ED%83%9C%EC%96%91%EB%A0%A5" title="태양력">태양력</a>(solar calendar)과 <a href="/wiki/%ED%83%9C%EC%9D%8C%EB%A0%A5" title="태음력">태음력</a>(lunar calendar)이 거의 완전히 일치하는 주기로 <a href="/wiki/19" title="19">19</a> <a href="/wiki/%ED%83%9C%EC%96%91%EB%85%84" title="태양년">태양년</a>(tropical years)에 해당한다. 이것은 <a href="/wiki/%ED%83%9C%EC%96%91%EB%85%84" title="태양년">태양년</a>(tropical year)과 <a href="/wiki/%EB%8B%AC" title="달">달</a>의 <a class="new" href="/w/index.php?title=%ED%9A%8C%ED%95%A9%EC%9B%94&amp;action=edit&amp;redlink=1" title="회합월 (없는 문서)">회합월</a>(synodic month)의 <a class="mw-redirect" href="/wiki/%EC%B5%9C%EC%86%8C_%EA%B3%B5%EB%B0%B0%EC%88%98" title="최소 공배수">최소 공배수</a>와 거의 일치한다. <b>엔네아데카에테리스</b>(Enneadecaeteris)라고도 하는데 이 낱말은 19년을 뜻하는 <a href="/wiki/%EA%B7%B8%EB%A6%AC%EC%8A%A4%EC%96%B4" title="그리스어">그리스어</a> 낱말로부터 유래한 것이다.</p>
<p><a href="/wiki/%EA%B8%B0%EC%9B%90%EC%A0%84_433%EB%85%84" title="기원전 433년">기원전 433년</a>에 <a href="/wiki/%EA%B7%B8%EB%A6%AC%EC%8A%A4" title="그리스">그리스</a>의 천문학자 <a class="new" href="/w/index.php?title=%EB%A9%94%ED%86%A4&amp;action=edit&amp;redlink=1" title="메톤 (없는 문서)">메톤</a>이 메톤 주기를 발견하였다.<sup class="reference" id="cite_ref-.EB.84.A4.EC.9D.B4.EB.B2.84-.EB.A9.94.ED.86.A4.EC.A3.BC.EA.B8.B0_1-0"><a href="#cite_note-.EB.84.A4.EC.9D.B4.EB.B2.84-.EB.A9.94.ED.86.A4.EC.A3.BC.EA.B8.B0-1">[1]</a></sup> <a href="/wiki/%EC%A4%91%EA%B5%AD" title="중국">중국</a>에서는 이보다 훨씬 앞선, <a class="mw-redirect" href="/wiki/%EC%B6%98%EC%B6%94%EC%8B%9C%EB%8C%80" title="춘추시대">춘추시대</a>(기원전 770~403)에 속한 <a class="mw-redirect" href="/wiki/%EA%B8%B0%EC%9B%90%EC%A0%84_600%EB%85%84" title="기원전 600년">기원전 600년</a>경에 이 19년 순환주기를 발견하였는데, 이 순환주기를 <b>장법</b>(章法)이라 부르고 <a href="/wiki/%EC%97%AD%EB%B2%95" title="역법">역법</a>에 사용하였다.<sup class="reference" id="cite_ref-.EB.84.A4.EC.9D.B4.EB.B2.84-.EB.A9.94.ED.86.A4.EC.A3.BC.EA.B8.B0_1-1"><a href="#cite_note-.EB.84.A4.EC.9D.B4.EB.B2.84-.EB.A9.94.ED.86.A4.EC.A3.BC.EA.B8.B0-1">[1]</a></sup> <b>19년 7윤법</b>이라고도 하였다.<sup class="reference" id="cite_ref-.EB.84.A4.EC.9D.B4.EB.B2.84-.EB.A9.94.ED.86.A4.EC.A3.BC.EA.B8.B0_1-2"><a href="#cite_note-.EB.84.A4.EC.9D.B4.EB.B2.84-.EB.A9.94.ED.86.A4.EC.A3.BC.EA.B8.B0-1">[1]</a></sup></p>

<h2><span class="mw-headline" id=".EA.B4.80.EB.A0.A8_.EC.9A.A9.EC.96.B4">관련 용어</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%A9%94%ED%86%A4_%EC%A3%BC%EA%B8%B0&amp;action=edit&amp;section=1" title="부분 편집: 관련 용어">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<h4><span class="mw-headline" id=".ED.83.9C.EC.96.91.EB.85.84">태양년</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%A9%94%ED%86%A4_%EC%A3%BC%EA%B8%B0&amp;action=edit&amp;section=2" title="부분 편집: 태양년">편집</a><span class="mw-editsection-bracket">]</span></span></h4>
<p><a href="/wiki/%ED%83%9C%EC%96%91%EB%85%84" title="태양년">태양년</a>(tropical year)은 <a href="/wiki/%EC%A7%80%EA%B5%AC" title="지구">지구</a>가 <a href="/wiki/%ED%83%9C%EC%96%91" title="태양">태양</a> 둘레로 자신의 궤도를 한 바퀴 도는 데 걸리는 시간으로 <a class="mw-redirect" href="/wiki/%ED%9A%8C%EA%B7%80%EB%85%84" title="회귀년">회귀년</a>이라고도 한다. 다른 말로는 <a href="/wiki/%EC%A7%80%EA%B5%AC" title="지구">지구</a>의 <a class="new" href="/w/index.php?title=%EA%B6%A4%EB%8F%84%EC%A3%BC%EA%B8%B0&amp;action=edit&amp;redlink=1" title="궤도주기 (없는 문서)">궤도주기</a>(orbital period) 또는 <a class="new" href="/w/index.php?title=%ED%95%AD%EC%84%B1%EC%A3%BC%EA%B8%B0&amp;action=edit&amp;redlink=1" title="항성주기 (없는 문서)">항성주기</a>(sidereal period)라고도 한다. 예를 들어, <a href="/wiki/%EB%B4%84" title="봄">봄</a>에서 다음 <a href="/wiki/%EB%B4%84" title="봄">봄</a>까지 걸리는 시간이라고 할 수 있다. <a href="/wiki/%ED%83%9C%EC%96%91%EB%85%84" title="태양년">태양년</a>은 365일 5시간 48분 45.5초(365.24219일)이다.</p>
<h4><span class="mw-headline" id=".ED.9A.8C.ED.95.A9.EC.9B.94">회합월</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%A9%94%ED%86%A4_%EC%A3%BC%EA%B8%B0&amp;action=edit&amp;section=3" title="부분 편집: 회합월">편집</a><span class="mw-editsection-bracket">]</span></span></h4>
<p><a href="/wiki/%EB%8B%AC" title="달">달</a>의 <a class="new" href="/w/index.php?title=%ED%9A%8C%ED%95%A9%EC%9B%94&amp;action=edit&amp;redlink=1" title="회합월 (없는 문서)">회합월</a>(synodic month 또는 synodic lunar month)은 달의 <a class="mw-redirect" href="/wiki/%ED%9A%8C%ED%95%A9%EC%A3%BC%EA%B8%B0" title="회합주기">회합주기</a>(synodic period)라고도 한다. <a class="mw-redirect" href="/wiki/%ED%9A%8C%ED%95%A9%EC%A3%BC%EA%B8%B0" title="회합주기">회합주기</a>는 <a href="/wiki/%EC%A7%80%EA%B5%AC" title="지구">지구</a>에서 볼 때 <a href="/wiki/%EB%8B%AC" title="달">달</a>이 <a href="/wiki/%ED%83%9C%EC%96%91" title="태양">태양</a>에 대해 상대적으로 같은 위치에 도달하는 데 걸리는 시간을 말한다. 예를 들어, <a href="/wiki/%EB%B3%B4%EB%A6%84%EB%8B%AC" title="보름달">보름달</a>에서 다음 <a href="/wiki/%EB%B3%B4%EB%A6%84%EB%8B%AC" title="보름달">보름달</a>까지 걸리는 시간이라고 할 수 있다. 달의 <a class="mw-redirect" href="/wiki/%ED%9A%8C%ED%95%A9%EC%A3%BC%EA%B8%B0" title="회합주기">회합주기</a>를 <a class="new" href="/w/index.php?title=%EC%82%AD%EB%A7%9D%EC%A3%BC%EA%B8%B0&amp;action=edit&amp;redlink=1" title="삭망주기 (없는 문서)">삭망주기</a>라고도 한다. <a href="/wiki/%EB%8B%AC" title="달">달</a>의 <a class="mw-redirect" href="/wiki/%ED%9A%8C%ED%95%A9%EC%A3%BC%EA%B8%B0" title="회합주기">회합주기</a>는 29일 12시간 44.0분(29.530588일)이다.</p>
<p>한편, <a href="/wiki/%EB%8B%AC" title="달">달</a>이 <a href="/wiki/%EC%A7%80%EA%B5%AC" title="지구">지구</a> 둘레로 자신의 궤도를 한 바퀴 도는데 걸리는 시간을 <a href="/wiki/%EB%8B%AC" title="달">달</a>의 <a class="new" href="/w/index.php?title=%ED%95%AD%EC%84%B1%EC%A3%BC%EA%B8%B0&amp;action=edit&amp;redlink=1" title="항성주기 (없는 문서)">항성주기</a>(sidereal period)라고 하는데, 이 기간은 27일 7시간 43.1분(27.321582일)이다. <a href="/wiki/%EB%8B%AC" title="달">달</a>의 <a class="mw-redirect" href="/wiki/%ED%9A%8C%ED%95%A9%EC%A3%BC%EA%B8%B0" title="회합주기">회합주기</a>와 <a href="/wiki/%EB%8B%AC" title="달">달</a>의 <a class="new" href="/w/index.php?title=%ED%95%AD%EC%84%B1%EC%A3%BC%EA%B8%B0&amp;action=edit&amp;redlink=1" title="항성주기 (없는 문서)">항성주기</a>가 다른 이유는 <a href="/wiki/%EB%8B%AC" title="달">달</a>이 <a href="/wiki/%EC%A7%80%EA%B5%AC" title="지구">지구</a> 둘레로 도는 동안 <a href="/wiki/%EC%A7%80%EA%B5%AC" title="지구">지구</a>도 <a href="/wiki/%ED%83%9C%EC%96%91" title="태양">태양</a>의 둘레를 돌기 때문이다. <a href="/wiki/%EB%8B%AC" title="달">달</a>과 <a href="/wiki/%EC%A7%80%EA%B5%AC" title="지구">지구</a>가 같은 방향으로 돌기 때문에, 예를 들어 <a href="/wiki/%EB%B3%B4%EB%A6%84%EB%8B%AC" title="보름달">보름달</a>이 보인 후 다시 다음 <a href="/wiki/%EB%B3%B4%EB%A6%84%EB%8B%AC" title="보름달">보름달</a>이 보이기 위해서는 <a href="/wiki/%EB%8B%AC" title="달">달</a>은 자신의 <a class="mw-redirect" href="/wiki/%ED%9A%8C%ED%95%A9%EC%A3%BC%EA%B8%B0" title="회합주기">회합주기</a>(29.530588일)와 <a class="new" href="/w/index.php?title=%ED%95%AD%EC%84%B1%EC%A3%BC%EA%B8%B0&amp;action=edit&amp;redlink=1" title="항성주기 (없는 문서)">항성주기</a>(27.321582일)의 차이인 약 2일간을 더 돌아야 한다.</p>
<h2><span class="mw-headline" id=".EC.88.98.ED.95.99.EC.A0.81_.EA.B8.B0.EC.B4.88">수학적 기초</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%A9%94%ED%86%A4_%EC%A3%BC%EA%B8%B0&amp;action=edit&amp;section=4" title="부분 편집: 수학적 기초">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p><a href="/wiki/%EA%B7%B8%EB%A6%AC%EC%8A%A4" title="그리스">그리스</a>의 천문학자인 <a class="new" href="/w/index.php?title=%EB%A9%94%ED%86%A4&amp;action=edit&amp;redlink=1" title="메톤 (없는 문서)">메톤</a>(Meton of Athens)은 19<a href="/wiki/%ED%83%9C%EC%96%91%EB%85%84" title="태양년">태양년</a>(tropical years)이 <a href="/wiki/%EB%8B%AC" title="달">달</a>의 235<a class="new" href="/w/index.php?title=%ED%9A%8C%ED%95%A9%EC%9B%94&amp;action=edit&amp;redlink=1" title="회합월 (없는 문서)">회합월</a>(synodic lunar months)과 2시간의 차이가 있을 뿐 거의 완전히 일치한다는 것을 발견하였다. 이를 계산으로 검증해 보면 다음과 같다:</p>
<ol>
<li>19<a href="/wiki/%ED%83%9C%EC%96%91%EB%85%84" title="태양년">태양년</a>을 일수로 계산하면 19태양년 x 365.24219일 = 6939.60161일이다.</li>
<li>235<a class="new" href="/w/index.php?title=%ED%9A%8C%ED%95%A9%EC%9B%94&amp;action=edit&amp;redlink=1" title="회합월 (없는 문서)">회합월</a>을 일수로 계산하면 235회합월 x 29.530588일 = 6939.68818일이다.</li>
<li>이 둘의 차이는 6939.68818일 - 6939.60161일 = 0.08657일이다.</li>
<li>1일은 24시간이므로 0.08657일은 0.08657 x 24시간 = 2.07768시간이다.</li>
<li>따라서, 19<a href="/wiki/%ED%83%9C%EC%96%91%EB%85%84" title="태양년">태양년</a>마다 <a href="/wiki/%ED%83%9C%EC%96%91%EB%A0%A5" title="태양력">태양력</a>과 <a href="/wiki/%ED%83%9C%EC%9D%8C%EB%A0%A5" title="태음력">태음력</a> 간에는 약 2시간의 차이가 발생한다.</li>
</ol>
<h2><span class="mw-headline" id=".ED.83.9C.EC.9D.8C.EB.A0.A5.EC.9D.98_.EC.A1.B0.EC.A0.95">태음력의 조정</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%A9%94%ED%86%A4_%EC%A3%BC%EA%B8%B0&amp;action=edit&amp;section=5" title="부분 편집: 태음력의 조정">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>19<a href="/wiki/%ED%83%9C%EC%96%91%EB%85%84" title="태양년">태양년</a>마다 <a href="/wiki/%ED%83%9C%EC%96%91%EB%A0%A5" title="태양력">태양력</a>과 <a href="/wiki/%ED%83%9C%EC%9D%8C%EB%A0%A5" title="태음력">태음력</a> 간에 주기가 거의 완전히 일치한다는 사실을 이용하여 <a href="/wiki/%ED%83%9C%EC%9D%8C%EB%A0%A5" title="태음력">태음력</a>이 <a href="/wiki/%ED%83%9C%EC%96%91%EB%A0%A5" title="태양력">태양력</a>과 비슷해지도록 조정할 수 있다. <a href="/wiki/%ED%83%9C%EC%9D%8C%EB%A0%A5" title="태음력">태음력</a>을 <a href="/wiki/%ED%83%9C%EC%96%91%EB%A0%A5" title="태양력">태양력</a>과 비슷하게 맞춘다는 것은 <a href="/wiki/%ED%83%9C%EC%9D%8C%EB%A0%A5" title="태음력">태음력</a> 달력이 <a href="/wiki/%EA%B3%84%EC%A0%88" title="계절">계절</a>과 거의 일치하게 된다는 것을 의미한다. 이렇게 하는 한 가지 방법은 다음과 같다.</p>
<ol>
<li>235<a class="new" href="/w/index.php?title=%ED%9A%8C%ED%95%A9%EC%9B%94&amp;action=edit&amp;redlink=1" title="회합월 (없는 문서)">회합월</a>은 일수로는 6940일(days)이다(235 x 29.530588일 = 6939.68818일).</li>
<li>6940일의 1/19은 6940일 / 19 = 365.263157894737일(365 + 1/4 + 1/76일)이다.</li>
<li>365.263157894737일은 12<a class="new" href="/w/index.php?title=%ED%9A%8C%ED%95%A9%EC%9B%94&amp;action=edit&amp;redlink=1" title="회합월 (없는 문서)">회합월</a>(synodic months)인 12 x 29.530588일 = 354.367056일보다 10.896101894737일(365.263157894737 - 354.367056) 더 크다.</li>
<li>이것이 뜻하는 바는 <a class="mw-redirect" href="/wiki/%EC%9D%8C%EB%A0%A5" title="음력">음력</a>으로 1년이 지나면 <a href="/wiki/%ED%83%9C%EC%96%91%EB%A0%A5" title="태양력">태양력</a>과는 약 11일의 차이가 난다는 것을 의미한다.</li>
<li>따라서 <a class="mw-redirect" href="/wiki/%EC%9D%8C%EB%A0%A5" title="음력">음력</a>으로 3년이 지나면 <a href="/wiki/%ED%83%9C%EC%96%91%EB%A0%A5" title="태양력">태양력</a>과는 약 33일의 차이가 나게 된다.</li>
<li>따라서 3년마다 <a class="mw-redirect" href="/wiki/%EC%9D%8C%EB%A0%A5" title="음력">음력</a>에 1달을 추가하며 13개의 음력 달들이 있게 하면, <a href="/wiki/%ED%83%9C%EC%9D%8C%EB%A0%A5" title="태음력">태음력</a>은 어느 정도 근사하게 <a href="/wiki/%ED%83%9C%EC%96%91%EB%A0%A5" title="태양력">태양력</a>과 일치하게 된다.</li>
<li><a href="/wiki/%ED%83%9C%EC%96%91%EB%A0%A5" title="태양력">태양력</a> 19년 동안에 이런 작업을 총 7번(19년/3년 = 6.33) 행하면 된다.</li>
<li>이런 이유로, 메톤 주기를 <b>19년 7윤법</b>이라고도 하였다.</li>
</ol>
<p><a class="new" href="/w/index.php?title=%EB%A9%94%ED%86%A4&amp;action=edit&amp;redlink=1" title="메톤 (없는 문서)">메톤</a>은 19<a href="/wiki/%ED%83%9C%EC%96%91%EB%85%84" title="태양년">태양년</a>을 6,940일로 보고, 이것을 큰달(30일) 125개월과 작은달(29일) 110개월의 총 235개월 6,940일로 배분하였다.<sup class="reference" id="cite_ref-.EB.84.A4.EC.9D.B4.EB.B2.84-.EB.A9.94.ED.86.A4.EC.A3.BC.EA.B8.B0_1-3"><a href="#cite_note-.EB.84.A4.EC.9D.B4.EB.B2.84-.EB.A9.94.ED.86.A4.EC.A3.BC.EA.B8.B0-1">[1]</a></sup> 이 방법은 <a href="/wiki/%ED%8F%89%EB%85%84" title="평년">평년</a>(12개월) 12년과 <a href="/wiki/%EC%9C%A4%EB%85%84" title="윤년">윤년</a>(13개월) 7년의 총 235개월로 <a href="/wiki/%ED%83%9C%EC%9D%8C%EB%A0%A5" title="태음력">태음력</a> 달력을 구성하는 것과 동일하므로 <b>19년 7윤법</b>이라고도 하였다.<sup class="reference" id="cite_ref-.EB.84.A4.EC.9D.B4.EB.B2.84-.EB.A9.94.ED.86.A4.EC.A3.BC.EA.B8.B0_1-4"><a href="#cite_note-.EB.84.A4.EC.9D.B4.EB.B2.84-.EB.A9.94.ED.86.A4.EC.A3.BC.EA.B8.B0-1">[1]</a></sup></p>
<p>메톤주기의 존재를 알 수 있는 예로, 특정한 양력 날짜와 그에 대응하는 음력 날짜는 19년이 지나면 다시 일치한다는 사실을 찾을 수 있다. 하지만 정확히 맞물리는 건 아니라서 하루 정도 어긋나기도 한다.</p>
