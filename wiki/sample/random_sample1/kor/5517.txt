<p><b>메트로젯 항공 9268편</b>(7K9268/KGL9268)은 <a href="/wiki/%EB%9F%AC%EC%8B%9C%EC%95%84" title="러시아">러시아</a> <a href="/wiki/%EC%BD%94%EA%B0%88%EB%A6%AC%EB%A7%88%EB%B9%84%EC%95%84_%ED%95%AD%EA%B3%B5" title="코갈리마비아 항공">코갈리마비아 항공</a>의 전세기가, <a href="/wiki/%EC%9D%B4%EC%A7%91%ED%8A%B8" title="이집트">이집트</a> <a href="/wiki/%EC%83%A4%EB%A6%84%EC%97%98%EC%85%B0%EC%9D%B4%ED%81%AC_%EA%B5%AD%EC%A0%9C%EA%B3%B5%ED%95%AD" title="샤름엘셰이크 국제공항">샤름엘셰이크 국제공항</a> 을 출발하여 <a href="/wiki/%EB%9F%AC%EC%8B%9C%EC%95%84" title="러시아">러시아</a> <a href="/wiki/%EC%83%81%ED%8A%B8%ED%8E%98%ED%85%8C%EB%A5%B4%EB%B6%80%EB%A5%B4%ED%81%AC" title="상트페테르부르크">상트페테르부르크</a> <a href="/wiki/%ED%92%80%EC%BD%94%EB%B3%B4_%EA%B5%AD%EC%A0%9C%EA%B3%B5%ED%95%AD" title="풀코보 국제공항">풀코보 국제공항</a> 로 향하던 2015년 10월 31일 4시 13분(UTC), 이집트 <a href="/wiki/%EC%8B%9C%EB%82%98%EC%9D%B4_%EB%B0%98%EB%8F%84" title="시나이 반도">시나이 반도</a> 북부에 추락한 항공 사고를 말한다.</p>
<p>사고기는 <a class="mw-redirect" href="/wiki/%EC%97%90%EC%96%B4%EB%B2%84%EC%8A%A4_A321" title="에어버스 A321">에어버스 A321-231</a> 기종으로, 7명의 승무원과 217명의 승객을 태우고 있었다. 대부분은 관광객으로, 우크라이나인 3명 이상, 벨라루스인 1명 외에 모두 <a href="/wiki/%EB%9F%AC%EC%8B%9C%EC%95%84%EC%9D%B8" title="러시아인">러시아인</a>이었다.</p>
<p>사고의 원인에 대해서는 여러 설이 존재했는데, 사고 직후 당시에는 기체결함으로 추정, <a href="/wiki/%EC%9D%B4%EC%8A%AC%EB%9E%8C_%EA%B5%AD%EA%B0%80_(%EB%8B%A8%EC%B2%B4)" title="이슬람 국가 (단체)">IS</a>의 소행 가능성은 낮다고 여겼으나, <a href="/wiki/%EC%9D%B4%EC%8A%AC%EB%9E%8C_%EA%B5%AD%EA%B0%80_(%EB%8B%A8%EC%B2%B4)" title="이슬람 국가 (단체)">IS</a>가 자신들의 소행이라고 주장한 것과, 9268편 블랙박스에서 기체결함에 의한 폭발음이 아닌 폭발물에 의한 폭발음이 들린다는 점, <a href="/wiki/%EC%9D%B4%EC%8A%AC%EB%9E%8C_%EA%B5%AD%EA%B0%80_(%EB%8B%A8%EC%B2%B4)" title="이슬람 국가 (단체)">IS</a> 내부 교신 감청 결과를 토대로 항공기의 추락 원인이 <a href="/wiki/%EC%9D%B4%EC%8A%AC%EB%9E%8C_%EA%B5%AD%EA%B0%80_(%EB%8B%A8%EC%B2%B4)" title="이슬람 국가 (단체)">IS</a>의 격추때문이라는 주장에 힘이 실리다가, 이 외의 <a href="/wiki/%ED%85%8C%EC%9D%BC%EC%8A%A4%ED%8A%B8%EB%9D%BC%EC%9D%B4%ED%81%AC" title="테일스트라이크">테일스트라이크</a>, 유지보수 문제설 등이 존재 하여 의견이 분분하였으나, <a href="/wiki/2015%EB%85%84" title="2015년">2015년</a> <a href="/wiki/11%EC%9B%94_17%EC%9D%BC" title="11월 17일">11월 17일</a>, <a href="/wiki/%EB%9F%AC%EC%8B%9C%EC%95%84_%EC%97%B0%EB%B0%A9%EB%B3%B4%EC%95%88%EA%B5%AD" title="러시아 연방보안국">러시아 연방보안국</a>이 메트로젯 항공 9268편의 추락 원인이 폭탄 폭발 때문이라고 공식 발표하였다.</p>
<p>이 사고 이후 <a href="/wiki/%EB%9F%AC%EC%8B%9C%EC%95%84" title="러시아">러시아</a>는 항공 안보 기준이 마련될 때까지는 <a href="/wiki/%EC%9D%B4%EC%A7%91%ED%8A%B8" title="이집트">이집트</a>로 향하는 모든 항공기의 운항을 금지시키게 되었다.</p>
