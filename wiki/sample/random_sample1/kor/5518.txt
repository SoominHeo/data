<p><b>메트로폴리탄 선</b>(Metropolitan line)은 <a href="/wiki/%EB%9F%B0%EB%8D%98_%EC%A7%80%ED%95%98%EC%B2%A0" title="런던 지하철">런던 지하철</a>의 노선이다. 노선색은 <a class="mw-redirect" href="/wiki/%EC%9E%90%EC%A3%BC%EC%83%89" title="자주색">자주색</a>이다. 세계에서 가장 오래된 <a href="/wiki/%EC%A7%80%ED%95%98%EC%B2%A0" title="지하철">지하철</a> 노선이기도 하다. <a href="/wiki/%EC%8B%9C%ED%8B%B0%EC%98%A4%EB%B8%8C%EB%9F%B0%EB%8D%98" title="시티오브런던">시티</a>의 <a class="new" href="/w/index.php?title=%EC%95%8C%EB%93%9C%EA%B2%8C%EC%9D%B4%ED%8A%B8_%EC%97%AD&amp;action=edit&amp;redlink=1" title="알드게이트 역 (없는 문서)">알드게이트</a>에서 시작하여, 북서부의 <a class="new" href="/w/index.php?title=%EC%98%A5%EC%8A%A4%EB%B8%8C%EB%A6%AC%EC%A7%80_%EC%97%AD&amp;action=edit&amp;redlink=1" title="옥스브리지 역 (없는 문서)">옥스브리지</a>,<a class="new" href="/w/index.php?title=%EC%95%84%EB%A8%B8%EC%85%A4_%EC%97%AD&amp;action=edit&amp;redlink=1" title="아머셤 역 (없는 문서)">아머셤</a>,<a class="new" href="/w/index.php?title=%EC%B2%B4%EC%85%A4_%EC%97%AD&amp;action=edit&amp;redlink=1" title="체셤 역 (없는 문서)">체셤</a>,<a class="new" href="/w/index.php?title=%EC%99%93%ED%8D%BC%EB%93%9C_%EC%97%AD&amp;action=edit&amp;redlink=1" title="왓퍼드 역 (없는 문서)">왓퍼드</a> 방면으로 향하는 노선이다. 길이는 41.6 마일이며, 34개의 역이 있다. 2011년 12월 기준으로 약 6700만 여명의 승객이 이용하였다. 노선색은 <a href="/wiki/%EB%A7%88%EC%A0%A0%ED%83%80" title="마젠타">마젠타</a>색(Pantone 235)이다.</p>

<p><a href="/wiki/1863%EB%85%84" title="1863년">1863년</a> <a href="/wiki/1%EC%9B%94_10%EC%9D%BC" title="1월 10일">1월 10일</a>에 <a href="/wiki/%EB%9F%B0%EB%8D%98_%ED%8C%A8%EB%94%A9%ED%84%B4_%EC%97%AD" title="런던 패딩턴 역">런던 패딩턴 역</a>과 <a class="new" href="/w/index.php?title=%ED%8C%A8%EB%A7%81%EB%8D%98_%EC%97%AD&amp;action=edit&amp;redlink=1" title="패링던 역 (없는 문서)">패링던 역</a>을 연결하는 노선으로 처음 개통하였으며, 당시의 이름은 <a class="new" href="/w/index.php?title=%EB%A9%94%ED%8A%B8%EB%A1%9C%ED%8F%B4%EB%A6%AC%ED%83%84_%EC%B2%A0%EB%8F%84&amp;action=edit&amp;redlink=1" title="메트로폴리탄 철도 (없는 문서)">메트로폴리탄 철도</a> (Metropolitan Railway)였다. 개통 초창기에는 증기 기관차에 목조 객차를 달아 운행한 형태였다. 이후 여러 차례의 연장을 거쳤는데, 해로 구간은 1880년에 연장되었고, 이어서 베이커 스트리트 역에서 50 마일 떨어진 <a class="mw-redirect" href="/wiki/%EB%B2%84%ED%82%B9%EC%97%84%EC%85%94" title="버킹엄셔">버킹엄셔</a>의 <a class="new" href="/w/index.php?title=%EB%B2%84%EB%8B%88_%EC%A0%95%EC%85%98&amp;action=edit&amp;redlink=1" title="버니 정션 (없는 문서)">버니 정션</a>(<a class="extiw" href="https://en.wikipedia.org/wiki/Verney_Junction_railway_station" title="en:Verney Junction railway station">Verney Junction</a>)까지 확장되었다. 19세기 말 부터 철도는 <a class="new" href="/w/index.php?title=%EB%A9%94%EB%A6%B4%EB%B2%88_%EC%97%AD&amp;action=edit&amp;redlink=1" title="메릴번 역 (없는 문서)">메릴번 역</a>에서 <a class="new" href="/w/index.php?title=%EA%B7%B8%EB%A0%88%EC%9D%B4%ED%8A%B8_%EC%84%BC%ED%8A%B8%EB%9F%B4_%EC%B2%A0%EB%8F%84&amp;action=edit&amp;redlink=1" title="그레이트 센트럴 철도 (없는 문서)">그레이트 센트럴 철도</a>(<a class="extiw" href="https://en.wikipedia.org/wiki/Great_Central_Railway" title="en:Great Central Railway">Great Central Railway</a>)와 선로를 공유하였다.</p>
<p>센트럴 런던 선은 1907년 부터 전철화 되었으나, 전기기관차는 해로의 북부로 향하는 열차의 증기기관차와 교환되었다. 철도가 1933년에 런던 여객 수송위원회(London Passenger Transport Board)에 의해 흡수된 이후 노선은 <a class="new" href="/w/index.php?title=%EC%97%90%EC%9D%BC%EC%A6%88%EB%B2%84%EB%A6%AC_%EC%97%AD&amp;action=edit&amp;redlink=1" title="에일즈버리 역 (없는 문서)">에일즈버리</a>(Aylesbury)에서 끊기게 되었다. 증기기관차는 1961년까지 가동되었고, 이후 노선은 아머셤에서 끊기고 동시에 전구간 전철화 되었다.</p>
<p><a href="/wiki/1988%EB%85%84" title="1988년">1988년</a>에 일부 구간(<a class="mw-redirect" href="/wiki/%ED%8C%A8%EB%94%A9%ED%84%B4_%EC%97%AD" title="패딩턴 역">패딩턴</a>(<a class="new" href="/w/index.php?title=%EB%B9%84%EC%88%8D_%EB%A1%9C%EB%93%9C_%EC%97%AD&amp;action=edit&amp;redlink=1" title="비숍 로드 역 (없는 문서)">비숍 로드</a>~<a class="new" href="/w/index.php?title=%ED%8C%A8%EB%A7%81%EB%8D%98_%EC%8A%A4%ED%8A%B8%EB%A6%AC%ED%8A%B8_%EC%97%AD&amp;action=edit&amp;redlink=1" title="패링던 스트리트 역 (없는 문서)">패링던 스트리트</a>)이 <a href="/wiki/%ED%95%B4%EB%A8%B8%EC%8A%A4%EB%AF%B8%EC%8A%A4_%26_%EC%8B%9C%ED%8B%B0_%EC%84%A0" title="해머스미스 &amp; 시티 선">해머스미스 &amp; 시티 선</a>과 <a href="/wiki/%EB%94%94%EC%8A%A4%ED%8A%B8%EB%A6%AD%ED%8A%B8_%EC%84%A0" title="디스트릭트 선">디스트릭트 선</a>,<a href="/wiki/%EC%84%9C%ED%81%B4_%EC%84%A0" title="서클 선">서클 선</a>으로 계통 분리되었다.</p>
<p>왓퍼드 분기는 <a class="new" href="/w/index.php?title=%EC%99%93%ED%8D%BC%EB%93%9C_%EC%A0%95%EC%85%98_%EC%97%AD&amp;action=edit&amp;redlink=1" title="왓퍼드 정션 역 (없는 문서)">왓퍼드 정션 역</a>(<a class="extiw" href="https://en.wikipedia.org/wiki/Watford_Junction_railway_station" title="en:Watford Junction railway station">Watford Junction</a>) 방면으로 전환하고, <a class="new" href="/w/index.php?title=%EC%99%93%ED%8D%BC%EB%93%9C_%EC%97%AD&amp;action=edit&amp;redlink=1" title="왓퍼드 역 (없는 문서)">왓퍼드 역</a>이 폐역된 후 새로운 역이 개업할 예정이다. 그리고 연장 노선은 <a href="/wiki/%EB%9F%B0%EB%8D%98_%EC%98%A4%EB%B2%84%EA%B7%B8%EB%9D%BC%EC%9A%B4%EB%93%9C" title="런던 오버그라운드">런던 오버그라운드</a> <a class="new" href="/w/index.php?title=%EC%99%93%ED%8D%BC%EB%93%9C_DC_%EC%84%A0&amp;action=edit&amp;redlink=1" title="왓퍼드 DC 선 (없는 문서)">왓퍼드 DC 선</a>(<a class="extiw" href="https://en.wikipedia.org/wiki/Watford_DC_Line" title="en:Watford DC Line">Watford DC Line|</a>)과 연결된다. 메트로폴리탄 선은 오버그라운드 역인 <a class="new" href="/w/index.php?title=%EC%99%93%ED%8D%BC%EB%93%9C_%ED%95%98%EC%9D%B4_%EC%8A%A4%ED%8A%B8%EB%A6%AC%ED%8A%B8_%EC%97%AD&amp;action=edit&amp;redlink=1" title="왓퍼드 하이 스트리트 역 (없는 문서)">왓퍼드 하이 스트리트 역</a>(<a class="extiw" href="https://en.wikipedia.org/wiki/Watford_High_Street_railway_station" title="en:Watford High Street railway station">Watford High Street</a>)와 연결하여 <a class="new" href="/w/index.php?title=%EC%9B%A8%EC%8A%A4%ED%8A%B8_%EC%BD%94%EC%8A%A4%ED%8A%B8_%EB%B3%B8%EC%84%A0&amp;action=edit&amp;redlink=1" title="웨스트 코스트 본선 (없는 문서)">웨스트 코스트 본선</a>(<a class="extiw" href="https://en.wikipedia.org/wiki/West_Coast_Main_Line" title="en:West Coast Main Line">West Coast Main Line</a>)과 연결되는 <a class="new" href="/w/index.php?title=%EC%99%93%ED%8D%BC%EB%93%9C_%EC%A0%95%EC%85%98_%EC%97%AD&amp;action=edit&amp;redlink=1" title="왓퍼드 정션 역 (없는 문서)">왓퍼드 정션 역</a>에서 종착하게 된다. 2016년에 공사를 시작하여 2020년까지 완공될 예정이다.</p>
<h3><span class="mw-headline" id=".EC.97.B0.ED.98.81">연혁</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%A9%94%ED%8A%B8%EB%A1%9C%ED%8F%B4%EB%A6%AC%ED%83%84_%EC%84%A0&amp;action=edit&amp;section=2" title="부분 편집: 연혁">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<ul>
<li>1863년 1월 10일, 세계 최초의 지하철이 문을 열었다. 바로 <a class="new" href="/w/index.php?title=%EB%A9%94%ED%8A%B8%EB%A1%9C%ED%8F%B4%EB%A6%AC%ED%83%84_%EB%A0%88%EC%9D%BC%EC%9B%A8%EC%9D%B4&amp;action=edit&amp;redlink=1" title="메트로폴리탄 레일웨이 (없는 문서)">메트로폴리탄 레일웨이</a>(Metropolitan Railway) 였다. 역 수는 7개였고 역 명은 <a class="new" href="/w/index.php?title=%EB%9F%B0%EB%8D%98_%ED%8E%98%EB%94%A9%ED%84%B4_%EC%97%AD&amp;action=edit&amp;redlink=1" title="런던 페딩턴 역 (없는 문서)">페딩턴</a>(비숍스 로드),<a class="mw-redirect mw-disambig" href="/wiki/%EC%97%90%EC%A7%80%EC%9B%A8%EC%96%B4_%EB%A1%9C%EB%93%9C_%EC%97%AD" title="에지웨어 로드 역">에지웨어 로드</a>,<a class="new" href="/w/index.php?title=%EB%B2%A0%EC%9D%B4%EC%BB%A4_%EC%8A%A4%ED%8A%B8%EB%A6%AC%ED%8A%B8_%EC%97%AD&amp;action=edit&amp;redlink=1" title="베이커 스트리트 역 (없는 문서)">베이커 스트리트</a>,포트랜드 로드(현재 <a class="new" href="/w/index.php?title=%EA%B7%B8%EB%A0%88%EC%9D%B4%ED%8A%B8%ED%8F%AC%ED%8B%80%EB%9E%9C%EB%93%9C_%EC%8A%A4%ED%8A%B8%EB%A6%AC%ED%8A%B8_%EC%97%AD&amp;action=edit&amp;redlink=1" title="그레이트포틀랜드 스트리트 역 (없는 문서)">그레이트 포틀랜드 스트리트</a>),고어 스트리트(현재 <a class="new" href="/w/index.php?title=%EC%9C%A0%EC%8A%A4%ED%84%B4_%EC%8A%A4%ED%80%98%EC%96%B4_%EC%97%AD&amp;action=edit&amp;redlink=1" title="유스턴 스퀘어 역 (없는 문서)">유스턴 스퀘어</a>), <a href="/wiki/%ED%82%B9%EC%8A%A4%ED%81%AC%EB%A1%9C%EC%8A%A4_%EC%84%B8%EC%9D%B8%ED%8A%B8%ED%8C%90%ED%81%AC%EB%9D%BC%EC%8A%A4_%EC%97%AD" title="킹스크로스 세인트판크라스 역">킹스 크로스</a>, 그리고 <a class="new" href="/w/index.php?title=%ED%8C%A8%EB%A7%81%EB%8D%98_%EC%97%AD&amp;action=edit&amp;redlink=1" title="패링던 역 (없는 문서)">패링던 스트리트</a>였다.</li>
<li>1864년 6월 13일에 패딩턴(비숍스 로드)에서 <a class="new" href="/w/index.php?title=%ED%95%B4%EB%A8%B8%EC%8A%A4%EB%AF%B8%EC%8A%A4_%EC%97%AD&amp;action=edit&amp;redlink=1" title="해머스미스 역 (없는 문서)">해머스미스</a>(Hammersmith)매트로폴리탄 연장이 되었다. 1865년 패링던 스트리트에서 <a class="new" href="/w/index.php?title=%EB%AC%B4%EC%96%B4%EA%B2%8C%EC%9D%B4%ED%8A%B8_%EC%97%AD&amp;action=edit&amp;redlink=1" title="무어게이트 역 (없는 문서)">무어게이트</a>(Moorgate)가 개통되었다.</li>
<li>1868년, 베이커 스트리트에서 스위스 코테지 지선이 개통되었다.</li>
<li>1868년 4월, 메트로폴리탄 레일웨이가 <a class="mw-redirect mw-disambig" href="/wiki/%EC%97%90%EC%A7%80%EC%9B%A8%EC%96%B4_%EB%A1%9C%EB%93%9C_%EC%97%AD" title="에지웨어 로드 역">에지웨어 로드</a>에서 사우스 캔싱턴까지 연장하였다.</li>
<li>1876년, 메트로폴리탄 레일웨이가 비숍게이트까지 연장하였다.</li>
<li>1879~1880년, 메트로폴리탄 레일웨이가 스위스 코디지에서 <a class="new" href="/w/index.php?title=%ED%95%B4%EB%A1%9C%EC%98%A8%EB%8D%94%ED%9E%90_%EC%97%AD&amp;action=edit&amp;redlink=1" title="해로온더힐 역 (없는 문서)">해로 언 더 힐</a>까지 개통되었다.</li>
<li>1882년, <a class="new" href="/w/index.php?title=%EC%95%8C%EB%93%9C%EA%B2%8C%EC%9D%B4%ED%8A%B8_%EC%97%AD&amp;action=edit&amp;redlink=1" title="알드게이트 역 (없는 문서)">알드게이트</a> 에서 타워 힐 역 연장이 개통되었다.</li>
<li>1884년, 비숍게이트 동쪽으로 <a href="/wiki/%EB%94%94%EC%8A%A4%ED%8A%B8%EB%A6%AD%ED%8A%B8_%EC%84%A0" title="디스트릭트 선">디스트릭트 노선</a>과 연결해 뉴 크로스까지 개통되었다.</li>
<li>1884년, 10월 <a href="/wiki/%EC%84%9C%ED%81%B4_%EC%84%A0" title="서클 선">서클 노선</a>이 순환선을 완성하면서 알드게이트에서 타워 힐 연장이 문을 닫았다.</li>
<li>1885~1889년 동안 3번의 연장을 거쳐 해로에서 <a class="new" href="/w/index.php?title=%EC%B2%B4%EC%85%A4_%EC%97%AD&amp;action=edit&amp;redlink=1" title="체셤 역 (없는 문서)">체섬</a>까지의 연장이 개통되었다.(1단계 해로~피너,2단계 피너~리치만워스,3단계 리치만워스~체섬)</li>
<li>1891~1899년 동안 체섬 지선 하나 전인 챌폰트 앤드 렛마이어에서 버니 정션과 브릴까지 연장이 개통되었다.</li>
<li>1904년, 해로에서 <a class="new" href="/w/index.php?title=%EC%98%A5%EC%8A%A4%EB%B8%8C%EB%A6%AC%EC%A7%80_%EC%97%AD&amp;action=edit&amp;redlink=1" title="옥스브리지 역 (없는 문서)">억스브리지</a>까지 지선이 개통되었다.</li>
<li>1900년경, 그레이트 노던 엔드 시티 레일웨이를 메트로폴리탄 레일웨이에서 인수하였다.</li>
<li>1925년, <a class="new" href="/w/index.php?title=%EB%AC%B4%EC%96%B4_%ED%8C%8C%ED%81%AC_%EC%97%AD&amp;action=edit&amp;redlink=1" title="무어 파크 역 (없는 문서)">무어 파크</a>에서 왓포드 지선이 개통되었다.</li>
<li>1932년, 웸블리 파크에서 스탠모어 지선이 개통되었다.</li>
<li>1939년, 베이컬루 노선이 스탄모어까지 지선을 만들면서 매트로폴리탄 노선 핀칠리 로드~스탠모어 지선의 모든 역들을 <a href="/wiki/%EB%B2%A0%EC%9D%B4%EC%BB%AC%EB%A3%A8_%EC%84%A0" title="베이컬루 선">베이컬루 선</a>이 인수하고 메트로폴리탄 노선은 <a class="new" href="/w/index.php?title=%EC%9B%B8%EB%B8%94%EB%A6%AC_%ED%8C%8C%ED%81%AC_%EC%97%AD&amp;action=edit&amp;redlink=1" title="웸블리 파크 역 (없는 문서)">웸블리 파크</a>와 <a class="new" href="/w/index.php?title=%ED%95%80%EC%B9%A0%EB%A6%AC_%EB%A1%9C%EB%93%9C_%EC%97%AD&amp;action=edit&amp;redlink=1" title="핀칠리 로드 역 (없는 문서)">핀칠리 로드</a>에 새로운 승강장을 만들어 정차한다.</li>
<li>1940년경, 비숍게이트-뉴 크로스 지선의 문을 닫았다.</li>
<li>1961년, 아머샴 북쪽의 역들을 <a class="mw-redirect" href="/wiki/%EB%B8%8C%EB%A6%AC%ED%8B%B0%EC%8B%9C_%EB%A0%88%EC%9D%BC" title="브리티시 레일">브리티시 레일</a>역으로 바꾸고 <a class="new" href="/w/index.php?title=%EC%95%84%EB%A8%B8%EC%85%A4_%EC%97%AD&amp;action=edit&amp;redlink=1" title="아머셤 역 (없는 문서)">아며샴</a>이 종점이 되었다.</li>
<li>1980년경, 베이커 스트리트-해머스미스 지선을 <a href="/wiki/%ED%95%B4%EB%A8%B8%EC%8A%A4%EB%AF%B8%EC%8A%A4_%26_%EC%8B%9C%ED%8B%B0_%EC%84%A0" title="해머스미스 &amp; 시티 선">해머스미스 앤드 시티 노선</a>으로 계통 분리되었다.</li>
</ul>
<p>이후로 오늘날의 모습을 가지게 되었다</p>
<h2><span class="mw-headline" id=".EB.85.B8.EC.84.A0">노선</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%A9%94%ED%8A%B8%EB%A1%9C%ED%8F%B4%EB%A6%AC%ED%83%84_%EC%84%A0&amp;action=edit&amp;section=3" title="부분 편집: 노선">편집</a><span class="mw-editsection-bracket">]</span></span></h2>

<p><a class="new" href="/w/index.php?title=%EC%95%8C%EB%93%9C%EA%B2%8C%EC%9D%B4%ED%8A%B8_%EC%97%AD&amp;action=edit&amp;redlink=1" title="알드게이트 역 (없는 문서)">알드게이트 역</a>과 <a class="new" href="/w/index.php?title=%ED%95%80%EC%B9%A0%EB%A6%AC_%EB%A1%9C%EB%93%9C_%EC%97%AD&amp;action=edit&amp;redlink=1" title="핀칠리 로드 역 (없는 문서)">핀칠리 로드 역</a> 사이의 구간은 지하로 운영한다. 지하 구간은 다른 튜브(Tube) 노선과 달리 <a class="new" href="/w/index.php?title=%EA%B0%9C%EC%B0%A9%EC%8B%9D&amp;action=edit&amp;redlink=1" title="개착식 (없는 문서)">개착식</a>으로 지어졌다. <a class="new" href="/w/index.php?title=%EB%B2%A0%EC%9D%B4%EC%BB%A4_%EC%8A%A4%ED%8A%B8%EB%A6%AC%ED%8A%B8_%EC%97%AD&amp;action=edit&amp;redlink=1" title="베이커 스트리트 역 (없는 문서)">베이커 스트리트 역</a> 이후 서부 구간은 지상으로 운용하며, <a href="/wiki/%EB%82%B4%EC%85%94%EB%84%90_%EB%A0%88%EC%9D%BC" title="내셔널 레일">내셔널 레일</a>과 선로를 공유한다. <a class="new" href="/w/index.php?title=%EB%85%B8%EC%8A%A4%EC%9A%B0%EB%93%9C_%EC%97%AD&amp;action=edit&amp;redlink=1" title="노스우드 역 (없는 문서)">노스우드 역</a>을 벗어난 이후 <a href="/wiki/%EA%B7%B8%EB%A0%88%EC%9D%B4%ED%84%B0%EB%9F%B0%EB%8D%98" title="그레이터런던">그레이터런던</a>을 벗어나 <a class="mw-redirect" href="/wiki/%ED%97%88%ED%8A%B8%ED%8F%AC%EB%93%9C%EC%85%94" title="허트포드셔">허트포드셔</a> 지역으로 가게 된다. <a href="/wiki/%EB%9F%B0%EB%8D%98_%EC%A7%80%ED%95%98%EC%B2%A0" title="런던 지하철">런던 지하철</a>에서 그레이터런던을 벗어나는 노선은 이 노선과 <a href="/wiki/%EC%84%BC%ED%8A%B8%EB%9F%B4_%EC%84%A0" title="센트럴 선">센트럴 선</a> 뿐이다.</p>
<p><a class="new" href="/w/index.php?title=%EC%95%8C%EB%93%9C%EA%B2%8C%EC%9D%B4%ED%8A%B8_%EC%97%AD&amp;action=edit&amp;redlink=1" title="알드게이트 역 (없는 문서)">알드게이트 역</a>과 <a class="new" href="/w/index.php?title=%EB%B2%A0%EC%9D%B4%EC%BB%A4_%EC%8A%A4%ED%8A%B8%EB%A6%AC%ED%8A%B8_%EC%97%AD&amp;action=edit&amp;redlink=1" title="베이커 스트리트 역 (없는 문서)">베이커 스트리트 역</a> 사이의 구간은 <a href="/wiki/%EC%84%9C%ED%81%B4_%EC%84%A0" title="서클 선">서클 선</a>과 <a href="/wiki/%ED%95%B4%EB%A8%B8%EC%8A%A4%EB%AF%B8%EC%8A%A4_%26_%EC%8B%9C%ED%8B%B0_%EC%84%A0" title="해머스미스 &amp; 시티 선">해머스미스 &amp; 시티 선</a>과 선로를 공유하고, <a class="new" href="/w/index.php?title=%EB%A0%88%EC%9D%B4%EB%84%88%EC%8A%A4_%EB%A0%88%EC%9D%B8_%EC%97%AD&amp;action=edit&amp;redlink=1" title="레이너스 레인 역 (없는 문서)">레이너스 레인 역</a>과 <a class="new" href="/w/index.php?title=%EC%98%A5%EC%8A%A4%EB%B8%8C%EB%A6%AC%EC%A7%80_%EC%97%AD&amp;action=edit&amp;redlink=1" title="옥스브리지 역 (없는 문서)">옥스브리지 역</a> 사이의 구간은 <a href="/wiki/%ED%94%BC%EC%B9%B4%EB%94%9C%EB%A6%AC_%EC%84%A0" title="피카딜리 선">피카딜리 선</a>과 선로를 공유하고, <a class="new" href="/w/index.php?title=%ED%95%B4%EB%A1%9C%EC%98%A8%EB%8D%94%ED%9E%90_%EC%97%AD&amp;action=edit&amp;redlink=1" title="해로온더힐 역 (없는 문서)">해로온더힐 역</a>과 <a class="new" href="/w/index.php?title=%EC%95%84%EB%A8%B8%EC%85%A4_%EC%97%AD&amp;action=edit&amp;redlink=1" title="아머셤 역 (없는 문서)">아머셤 역</a> 사이의 구간은 <a class="new" href="/w/index.php?title=%EC%B9%A0%ED%84%B4_%EB%A0%88%EC%9D%BC%EC%9B%A8%EC%9D%B4&amp;action=edit&amp;redlink=1" title="칠턴 레일웨이 (없는 문서)">칠턴 레일웨이</a>와 선로를 공유한다.</p>
<p>대부분의 선로는 2개의 복선 형태이나, 웸블리 파크와 무어 파크 사이의 구간은 4개의 복복선 형태이다. 이 구간에서 빠른 열차가 느린 열차를 추월할 수 있다. 웸블리파크와 핀칠리로드 사이에도 4개의 선로가 있었으나, 이 중 2개 선로는 1939년에 <a href="/wiki/%EB%B2%A0%EC%9D%B4%EC%BB%AC%EB%A3%A8_%EC%84%A0" title="베이컬루 선">베이컬루 선</a>으로 옮겼고, 1979년에 <a href="/wiki/%EC%A3%BC%EB%B9%8C%EB%A6%AC_%EC%84%A0" title="주빌리 선">주빌리 선</a>으로 옮겼다.</p>
