<h2><span class="mw-headline" id=".EC.83.9D.EC.95.A0">생애</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%A9%94%ED%9D%90%EB%A9%94%ED%8A%B8_%EC%98%A4%EB%A5%B4%ED%95%9C&amp;action=edit&amp;section=1" title="부분 편집: 생애">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>메흐메트 오르한은 압둘 카디르 왕자의 두 번째 부인이 낳은 첫째 아들이었다. 그의 아버지 압둘 카디르 왕자는 1876~1909년에 재임한 34대 <a href="/wiki/%EC%88%A0%ED%83%84" title="술탄">술탄</a>인 <a href="/wiki/%EC%95%95%EB%92%AC%ED%95%98%EB%AF%B8%ED%8A%B8_2%EC%84%B8" title="압뒬하미트 2세">압뒬하미트 2세</a>의 두 번째 부인 소생의 아들이었으며, 오스만 제국 육군 총사령관을 역임하였다. 그러나 카디르 왕자는 첫 번째 부인과의 결혼에서는 자식이 없던 상태에서, 정식으로 결혼하지 않은 두 번째 부인에게서 메흐메트 오르한을 얻었다. 이러한 문제 때문에 두 번째 부인과의 결혼은 왕실로부터 한동안 공식적으로 인정받지 못했다. 한편 <a href="/wiki/%EC%95%95%EB%92%AC%ED%95%98%EB%AF%B8%ED%8A%B8_2%EC%84%B8" title="압뒬하미트 2세">압뒬하미트 2세</a>가 <a class="mw-redirect" href="/wiki/%EC%B2%AD%EB%85%84_%ED%88%AC%EB%A5%B4%ED%81%AC%EB%8B%B9" title="청년 투르크당">청년 투르크당</a>에 의해 강제 퇴위된 이후, <a href="/wiki/%EC%88%A0%ED%83%84" title="술탄">술탄</a> 지위는 연장자 상속 원칙에 따라 <a href="/wiki/%EC%95%95%EB%92%AC%ED%95%98%EB%AF%B8%ED%8A%B8_2%EC%84%B8" title="압뒬하미트 2세">압뒬하미트 2세</a>의 이복동생인 <a href="/wiki/%EB%A9%94%ED%9D%90%EB%A9%94%ED%8A%B8_6%EC%84%B8" title="메흐메트 6세">메흐메트 6세</a>와 <a class="mw-redirect" href="/wiki/%EB%A9%94%ED%9D%90%EB%A9%94%ED%8A%B8_7%EC%84%B8" title="메흐메트 7세">메흐메트 7세</a>로 계승되었다.</p>
<p><a href="/wiki/1922%EB%85%84" title="1922년">1922년</a> <a href="/wiki/11%EC%9B%94_1%EC%9D%BC" title="11월 1일">11월 1일</a>에는 <a href="/wiki/%EB%AC%B4%EC%8A%A4%ED%83%80%ED%8C%8C_%EC%BC%80%EB%A7%90_%EC%95%84%ED%83%80%ED%8A%80%EB%A5%B4%ED%81%AC" title="무스타파 케말 아타튀르크">무스타파 케말 아타튀르크</a>에 의해 술탄제가 폐지되었다. 새로 출범한 <a href="/wiki/%ED%84%B0%ED%82%A4" title="터키">터키</a> 공화국은 이어 <a href="/wiki/1924%EB%85%84" title="1924년">1924년</a> <a href="/wiki/3%EC%9B%94_3%EC%9D%BC" title="3월 3일">3월 3일</a>에 오스만 가문이 갖고 있던 <a href="/wiki/%EC%B9%BC%EB%A6%AC%ED%8C%8C" title="칼리파">칼리파</a> 권한마저 폐지해버렸다. 이로 인해 주요 황족들은 모두 강제 추방 조치가 내려졌으며, 메흐메트 오르한도 가족과 함께 수중에 단 2,000파운드만을 챙긴 채 외국으로 떠났다. 이어 <a href="/wiki/1924%EB%85%84" title="1924년">1924년</a> <a href="/wiki/4%EC%9B%94_23%EC%9D%BC" title="4월 23일">4월 23일</a>을 기해 <a href="/wiki/%ED%84%B0%ED%82%A4_%EB%8C%80%EA%B5%AD%EB%AF%BC%EC%9D%98%ED%9A%8C" title="터키 대국민의회">터키 대국민의회</a>는 <a href="/wiki/%EC%98%A4%EC%8A%A4%EB%A7%8C_%EC%A0%9C%EA%B5%AD" title="오스만 제국">오스만 제국</a>의 주요 인사 150명을 비롯한 황족들을 기피인물로 지정하고 이들의 재입국을 법적으로 봉쇄해버렸다. 새로운 법령에 의해 오스만 가문의 남자 인사들은 50년간, 여자 인사들은 28년간 입국이 불허되었다. 이후 메흐메트 오르한은 한때 <a href="/wiki/%EC%95%8C%EB%B0%94%EB%8B%88%EC%95%84" title="알바니아">알바니아</a> 국왕 <a href="/wiki/%EC%A1%B0%EA%B5%AC_1%EC%84%B8" title="조구 1세">조구 1세</a>의 후의를 입으며 지내기도 했으나, 결국 <a href="/wiki/%EC%95%84%EB%A5%B4%ED%97%A8%ED%8B%B0%EB%82%98" title="아르헨티나">아르헨티나</a>에서 조선 노동자 일을, <a href="/wiki/%EB%B2%A0%EC%9D%B4%EB%A3%A8%ED%8A%B8" title="베이루트">베이루트</a>와 <a href="/wiki/%EB%8B%A4%EB%A7%88%EC%8A%A4%EC%BF%A0%EC%8A%A4" title="다마스쿠스">다마스쿠스</a>를 오가는 장거리 택시 운전기사 일을 하다가 <a href="/wiki/%ED%94%84%EB%9E%91%EC%8A%A4" title="프랑스">프랑스</a>로 넘어가 미군 묘지 안내원을 하는 등 궁벽한 삶을 이어갔다.<sup class="reference" id="cite_ref-1"><a href="#cite_note-1">[1]</a></sup></p>
<p><a href="/wiki/1983%EB%85%84" title="1983년">1983년</a>에는 33대 <a href="/wiki/%EC%88%A0%ED%83%84" title="술탄">술탄</a> 무라트 5세의 증손자이자 오스만 가문의 41대 수장이던 알리 바십이 사망하면서, 남은 가문 인사 가운데 최연장자로서 가문 수장 직위를 이어받았다. 그는 터키 정부의 입국 금지 조치가 풀린 이후에도 터키 방문을 거부하다가 <a href="/wiki/1992%EB%85%84" title="1992년">1992년</a>에야 잠시 방문하였다. 이후에는 <a href="/wiki/%ED%94%84%EB%9E%91%EC%8A%A4" title="프랑스">프랑스</a>의 <a href="/wiki/%EB%8B%88%EC%8A%A4_(%ED%94%84%EB%9E%91%EC%8A%A4)" title="니스 (프랑스)">니스</a>로 돌아가 단촐한 아파트에서 홀로 여생을 보내다가 <a href="/wiki/1994%EB%85%84" title="1994년">1994년</a> <a href="/wiki/3%EC%9B%94_12%EC%9D%BC" title="3월 12일">3월 12일</a>에 84세를 일기로 사망하였다.</p>
<p><a href="/wiki/1933%EB%85%84" title="1933년">1933년</a>, <a href="/wiki/1943%EB%85%84" title="1943년">1943년</a>에 두 차례 결혼하였으며, 첫 부인과의 사이에서 딸 하나를, 둘째 부인과의 사이에서 아들 하나를 얻었다.</p>
<h2><span class="mw-headline" id=".EC.9E.98.EB.AA.BB.EB.90.9C_.EC.86.8D.EC.84.A4">잘못된 속설</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%A9%94%ED%9D%90%EB%A9%94%ED%8A%B8_%EC%98%A4%EB%A5%B4%ED%95%9C&amp;action=edit&amp;section=2" title="부분 편집: 잘못된 속설">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>신경정신과 의사이자 에세이 저술가로 널리 알려진 <a href="/wiki/%EC%9D%B4%EC%8B%9C%ED%98%95_(1934%EB%85%84)" title="이시형 (1934년)">이시형</a> 교수는, 그의 2001년 작 &lt;이시형의 세상 바꿔보기&gt;에 수록된 '조국 땅에 살 수 있는 자격'이라는 글에서 오스만 제국의 마지막 <a href="/wiki/%ED%99%A9%ED%83%9C%EC%9E%90" title="황태자">황태자</a>이자 평생 독신으로 살아온 메흐메트 오르한이 해외를 떠돌다 <a href="/wiki/1992%EB%85%84" title="1992년">1992년</a>에 일시 귀국했을 때의 일화를 적었다. 이 글에서는 메흐메트 오르한이 <a href="/wiki/%EB%8F%8C%EB%A7%88%EB%B0%94%ED%9D%90%EC%B2%B4_%EA%B6%81%EC%A0%84" title="돌마바흐체 궁전">돌마바흐체 궁전</a>에서 감격에 겨워하다가 영국으로 돌아갔다는 이야기를 적고 있다. 그 영향을 받아 '오스만 제국의 마지막 <a href="/wiki/%ED%99%A9%ED%83%9C%EC%9E%90" title="황태자">황태자</a>'라는 주제로 여러 변형된 이야기가 세간에 떠돌기도 했다.</p>
<p>그러나 이러한 속설 대부분은 <a href="/wiki/%EC%98%A4%EC%8A%A4%EB%A7%8C_%EC%A0%9C%EA%B5%AD" title="오스만 제국">오스만 제국</a>의 왕위 계승권에 대한 잘못된 이해와, 메흐메트 오르한의 이복동생이자 그의 사후 오스만 가문 수장을 이어받은 <a class="new" href="/w/index.php?title=%EC%97%90%EB%A5%B4%ED%88%AC%EB%A3%B0_%EC%98%A4%EC%8A%A4%EB%A7%8C&amp;action=edit&amp;redlink=1" title="에르투룰 오스만 (없는 문서)">에르투룰 오스만</a>(Ertuğrul Osman)과의 착각으로 빚어진 것이다. <a href="/wiki/%EC%98%A4%EC%8A%A4%EB%A7%8C_%EC%A0%9C%EA%B5%AD" title="오스만 제국">오스만 제국</a> 후기의 <a href="/wiki/%EC%88%A0%ED%83%84" title="술탄">술탄</a>은 장자에게 상속되는 것이 아니라 가문 최연장자에게 상속되었으므로 일반적인 의미의 <a href="/wiki/%ED%99%A9%ED%83%9C%EC%9E%90" title="황태자">황태자</a>는 존재하지 않았다. <a href="/wiki/%EC%98%81%EA%B5%AD" title="영국">영국</a>에서 <a href="/wiki/%EC%99%95%EC%84%B8%EC%9E%90" title="왕세자">왕세자</a>에게 '<a href="/wiki/%EC%9B%A8%EC%9D%BC%EC%8A%A4_%EA%B3%B5" title="웨일스 공">웨일스 공</a>(Prince of Wales)' 작위를 주듯이, 현직 <a href="/wiki/%EC%88%A0%ED%83%84" title="술탄">술탄</a>의 아들 가운데 대표 왕자가 '알바니아 국왕' 칭호를 받고 활동할 뿐이었다. 메흐메드 압뒬 카디르 에펜디(Mehmed Abdülkadir Efendi) 왕자나 메흐메트 오르한은 <a href="/wiki/%EC%98%A4%EC%8A%A4%EB%A7%8C_%EC%A0%9C%EA%B5%AD" title="오스만 제국">오스만 제국</a> 말기에는 다른 가문 연장자들이 많았기에 술탄 계승 서열도 매우 낮았고, 혼사를 둘러싼 갈등으로 인해 '알바니아 국왕' 직위에서도 밀려난 상태였다. 때문에 메흐메트 오르한은 <a href="/wiki/%EC%98%A4%EC%8A%A4%EB%A7%8C_%EC%A0%9C%EA%B5%AD" title="오스만 제국">오스만 제국</a>의 심장부인 <a href="/wiki/%EB%8F%8C%EB%A7%88%EB%B0%94%ED%9D%90%EC%B2%B4_%EA%B6%81%EC%A0%84" title="돌마바흐체 궁전">돌마바흐체 궁전</a>에 특별한 추억이 있지도 않았다. 메흐메트 오르한에게 가문 수장직이 돌아온 것은 그의 나이가 74세를 넘고 나서인 <a href="/wiki/1983%EB%85%84" title="1983년">1983년</a>의 일이었다. 그리고 두 번의 결혼으로 1남 1녀를 두고 있었으므로 독신을 고수했다는 이야기도 사실과 다르다. 따라서 '독신을 지킨 <a href="/wiki/%EC%98%A4%EC%8A%A4%EB%A7%8C_%EC%A0%9C%EA%B5%AD" title="오스만 제국">오스만 제국</a>의 마지막 <a href="/wiki/%ED%99%A9%ED%83%9C%EC%9E%90" title="황태자">황태자</a>'라는 표현은 틀린 것이다.</p>
<p>마지막 <a href="/wiki/%ED%99%A9%ED%83%9C%EC%9E%90" title="황태자">황태자</a>로 불릴 만한 인물은 메흐메트 오르한의 사촌동생인 에르투룰 오스만이었다. 그의 아버지 메흐메트 부르하네딘 왕자는 <a href="/wiki/%EC%95%95%EB%92%AC%ED%95%98%EB%AF%B8%ED%8A%B8_2%EC%84%B8" title="압뒬하미트 2세">압뒬하미트 2세</a>의 네 번째 부인 소생이었으나, 메흐메드 압뒬 카디르 에펜디 왕자가 결혼 문제로 공백기에 있던 1914~19년에 공식적인 '알바니아 국왕' 직을 맡았다. 에르투룰 오스만은 터키 공화국의 입국 금지 조치 이후 <a href="/wiki/%EB%AF%B8%EA%B5%AD" title="미국">미국</a>에서 거주하다가 메흐메트 오르한과 마찬가지로 <a href="/wiki/1992%EB%85%84" title="1992년">1992년</a>에 터키 정부의 초청으로 고국을 방문했다. 국내에 극적인 형태로 소개된 <a href="/wiki/%EB%8F%8C%EB%A7%88%EB%B0%94%ED%9D%90%EC%B2%B4_%EA%B6%81%EC%A0%84" title="돌마바흐체 궁전">돌마바흐체 궁전</a> 방문 일화 등은 대부분 이 에르투룰 오스만이 방문했을 때의 이야기였다. 그는 메흐메트 오르한 사망 이후 오스만 가문의 수장직을 이어받았으며, 술탄제 폐지 이전의 <a href="/wiki/%EC%98%A4%EC%8A%A4%EB%A7%8C_%EC%A0%9C%EA%B5%AD" title="오스만 제국">오스만 제국</a>에서 출생한 마지막 수장이 되었다.</p>
