<p><b>멕시코-미국 전쟁</b>(Mexican-American War)은 <a href="/wiki/1846%EB%85%84" title="1846년">1846년</a>과 <a href="/wiki/1848%EB%85%84" title="1848년">1848년</a> 사이 <a href="/wiki/%EB%A9%95%EC%8B%9C%EC%BD%94" title="멕시코">멕시코</a>와 <a href="/wiki/%EB%AF%B8%EA%B5%AD" title="미국">미국</a> 사이에 발생한 군사 분쟁이다. 맥락에 따라서 <b>멕시코 전쟁</b>, 그리고 당시 미국 대통령인 <a class="mw-redirect" href="/wiki/%EC%A0%9C%EC%9E%84%EC%8A%A4_%ED%8F%AC%ED%81%AC" title="제임스 포크">제임스 포크</a>에서 따서 <b>포크의 전쟁</b>로 불리기도 한다.</p>
<p>이 전쟁은 <a href="/wiki/1836%EB%85%84" title="1836년">1836년</a> <a href="/wiki/%ED%85%8D%EC%82%AC%EC%8A%A4_%ED%98%81%EB%AA%85" title="텍사스 혁명">텍사스 혁명</a>으로 <a href="/wiki/%ED%85%8D%EC%82%AC%EC%8A%A4_%EA%B3%B5%ED%99%94%EA%B5%AD" title="텍사스 공화국">텍사스 공화국</a>이 성립되었지만, 멕시코는 이 땅을 자국의 영토로 여겨 <a href="/wiki/1845%EB%85%84" title="1845년">1845년</a> <a href="/wiki/%ED%85%8D%EC%82%AC%EC%8A%A4_%ED%95%A9%EB%B3%91" title="텍사스 합병">텍사스 합병</a>으로 텍사스가 미국의 주가 되자 일어났다.</p>

<h2><span class="mw-headline" id=".EB.B0.B0.EA.B2.BD">배경</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%A9%95%EC%8B%9C%EC%BD%94-%EB%AF%B8%EA%B5%AD_%EC%A0%84%EC%9F%81&amp;action=edit&amp;section=1" title="부분 편집: 배경">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p><a href="/wiki/1821%EB%85%84" title="1821년">1821년</a> <a href="/wiki/%EC%8A%A4%ED%8E%98%EC%9D%B8" title="스페인">스페인</a>으로부터 독립한 멕시코는 내전으로 번질 수도 있는 수 많은 갈등을 겪고 있었다. <a href="/wiki/1836%EB%85%84" title="1836년">1836년</a> <a href="/wiki/%ED%85%8D%EC%82%AC%EC%8A%A4_%ED%98%81%EB%AA%85" title="텍사스 혁명">텍사스 혁명</a>으로 발발된 <a href="/wiki/%ED%85%8D%EC%82%AC%EC%8A%A4_%EA%B3%B5%ED%99%94%EA%B5%AD" title="텍사스 공화국">텍사스 공화국</a>의 독립 문제에 대해서는 반대하는 입장으로 비교적 단합을 하였다. 멕시코는 미국이 텍사스를 병합하면 전쟁이 시작될 수도 있다고 위협을 했다.<sup class="reference" id="cite_ref-1"><a href="#cite_note-1">[1]</a></sup> 반면, <a class="mw-redirect" href="/wiki/%EC%A0%9C%EC%9E%84%EC%8A%A4_%ED%8F%AC%ED%81%AC" title="제임스 포크">제임스 포크</a> 대통령의 ‘<a href="/wiki/%EB%AA%85%EB%B0%B1%ED%95%9C_%EC%9A%B4%EB%AA%85" title="명백한 운명">명백한 운명</a>’에 대한 정신은 서부 확장에 대한 미국의 이익에 집중되어 있었다.</p>
<p>멕시코의 군사력과 외교 역량은 독립 이후 쇠퇴되어 있었고, 국가의 북부 절반을 <a href="/wiki/%EC%BD%94%EB%A7%8C%EC%B9%98_%EC%A1%B1" title="코만치 족">코만치 족</a>, <a href="/wiki/%EC%95%84%ED%8C%8C%EC%B9%98_%EC%A1%B1" title="아파치 족">아파치 족</a>, <a href="/wiki/%EB%82%98%EB%B0%94%ED%98%B8_%EC%A1%B1" title="나바호 족">나바호 족</a> 등의 인디언 부족에게 취약점을 노출했다. 이들 인디언들 중 특히 코만치족은 멕시코의 이러한 취약점을 공략하여 수 백마일에 이르는 대규모의 습격을 단행하여 가축을 약탈하였고, 이것들을 텍사스와 미국에 공급하였다.<sup class="reference" id="cite_ref-2"><a href="#cite_note-2">[2]</a></sup></p>
<p>인디언의 습격은 수천 명의 사상자를 낳았고, 북부 멕시코를 황폐화시켰다. <a href="/wiki/1846%EB%85%84" title="1846년">1846년</a> 미합중국 군대가 멕시코 북부에 진입했을 때 그들은 실의에 빠진 사람들만 볼 수 있었다. 그리하여 미국인들에게 저항하는 민병대는 거의 존재하지 않았다.<sup class="reference" id="cite_ref-3"><a href="#cite_note-3">[3]</a></sup></p>
<p>이들 인디언들의 적대적인 활동으로 인해 멕시코 내부와 알타캘리포니아와 뉴멕시코 사이의 교역과 소통은 어렵게 되었다. 결과적으로 멕시코 미국 전쟁이 발발할 시기에 뉴멕시코는 <a href="/wiki/%EC%82%B0%ED%83%80%ED%8E%98_%EA%B0%80%EB%8F%84" title="산타페 가도">산타페 가도</a>를 통한 미국과의 무역에 의존적일 수 밖에 없었다.<sup class="reference" id="cite_ref-4"><a href="#cite_note-4">[4]</a></sup></p>
<p>텍사스 공화국이 <a href="/wiki/1845%EB%85%84" title="1845년">1845년</a> <a href="/wiki/%ED%85%8D%EC%82%AC%EC%8A%A4_%ED%95%A9%EB%B3%91" title="텍사스 합병">텍사스 합병</a>을 통해 미국의 28번째 주인 <a href="/wiki/%ED%85%8D%EC%82%AC%EC%8A%A4_%EC%A3%BC" title="텍사스 주">텍사스 주</a>가 되자 종주국이라고 생각한 멕시코는 이 합병을 인정하지 않았다. 결국 이듬해 <a href="/wiki/1846%EB%85%84" title="1846년">1846년</a> 발발한 전쟁이 멕시코-미국 전쟁이다.</p>
<p>멕시코와 미국은 각각 텍사스에 대한 종주권을 주장하였다. 텍사스는 멕시코와 싸워 독립을 쟁취한 지 오래지 않았으며 멕시코는 텍사스를 반란 지방으로 인식해 독립을 인정치 않고 있었다. 전쟁의 근본 이유는 미국의 서진 확장 경향과 최근 <a href="/wiki/%EB%A9%95%EC%8B%9C%EC%BD%94_%EB%8F%85%EB%A6%BD_%EC%A0%84%EC%9F%81" title="멕시코 독립 전쟁">멕시코 독립 전쟁</a>을 통해 독립한 멕시코의 정치적 불안정이었다. 미국은 멕시코와 협상을 하는 것이 어려운 시점이었으며 멕시코는 북부 지방을 관리하는 것이 어려운 시기였다. 텍사스는 전쟁을 지지했다. 미국 내부에서 <a href="/wiki/%ED%9C%98%EA%B7%B8%EB%8B%B9_(%EB%AF%B8%EA%B5%AD)" title="휘그당 (미국)">휘그당</a>이 거세게 전쟁에 반대했다. 멕시코에서 전쟁은 방어 전략상 필요 수준으로 여겨졌다.</p>
<p>전쟁의 가장 중요한 결과는 멕시코가 <a class="mw-redirect" href="/wiki/%EC%BA%98%EB%A6%AC%ED%8F%AC%EB%8B%88%EC%95%84" title="캘리포니아">캘리포니아</a>와 <a class="mw-redirect" href="/wiki/%EB%89%B4%EB%A9%95%EC%8B%9C%EC%BD%94" title="뉴멕시코">뉴멕시코</a>를 미국에 팔아넘기게 된 것이었다. 미국 정계는 새로 획득한 주에서 <a class="mw-redirect" href="/wiki/%EB%85%B8%EC%98%88_%EC%A0%9C%EB%8F%84" title="노예 제도">노예 제도</a>와 관련하여 격렬한 논쟁을 벌여 ‘<a href="/wiki/1850%EB%85%84_%ED%83%80%ED%98%91" title="1850년 타협">1850년의 타협</a>’을 보았다. 멕시코에서는 정치적 혼란기가 계속되어 전쟁은 큰 영향을 끼치지 못했다. 또한 이 전쟁은 유럽의 1848년의 혁명과 거의 때를 같이하고 있어 더욱 중요한 의미가 있다.</p>

<p>멕시코-미국 전쟁 이전 이미 미국의 산악인들과 무역상들은 ‘<a href="/wiki/%EC%82%B0%ED%83%80%ED%8E%98_%EA%B0%80%EB%8F%84" title="산타페 가도">산타페 가도</a>’와 ‘<a href="/wiki/%EC%BA%98%EB%A6%AC%ED%8F%AC%EB%8B%88%EC%95%84_%EA%B0%80%EB%8F%84" title="캘리포니아 가도">캘리포니아 가도</a>’(California Trail)를 통해 이 땅에 자주 접촉하고 있었다. 미국 상선들은 활발하게 무역을 하고 있었다. 이 지역은 30년간 <a href="/wiki/%EB%A9%95%EC%8B%9C%EC%BD%94_%EC%A0%9C1%EA%B3%B5%ED%99%94%EA%B5%AD" title="멕시코 제1공화국">멕시코 공화국</a>의 영토였으며 그 이전에는 <a class="mw-redirect" href="/wiki/%EC%97%90%EC%8A%A4%ED%8C%8C%EB%83%90" title="에스파냐">에스파냐</a> 제국의 영역이었다.</p>
<p>미국이 <a href="/wiki/%EB%A3%A8%EC%9D%B4%EC%A7%80%EC%95%A0%EB%82%98_%EB%A7%A4%EC%9E%85" title="루이지애나 매입">루이지애나 매입</a> 이후 <a href="/wiki/%EC%8A%A4%ED%8E%98%EC%9D%B8_%EC%A0%9C%EA%B5%AD" title="스페인 제국">스페인 제국</a>의 토지 제공에 힘입어 미국의 백인 정착자들이 서부로 움직이기 시작했다. <a href="/wiki/%EB%A9%95%EC%8B%9C%EC%BD%94_%EB%8F%85%EB%A6%BD_%EC%A0%84%EC%9F%81" title="멕시코 독립 전쟁">멕시코 독립 전쟁</a> 이후 이 영토가 멕시코 공화국의 땅이 된 이후에도 서부로 향하는 이민은 계속되었다. 멕시코 정부는 자국의 국교인 <a class="mw-redirect" href="/wiki/%EB%A1%9C%EB%A7%88_%EA%B0%80%ED%86%A8%EB%A6%AD" title="로마 가톨릭">로마 가톨릭</a>이 아니라 <a href="/wiki/%EA%B0%9C%EC%8B%A0%EA%B5%90" title="개신교">개신교</a>를 믿는 정착자들과 불편한 관계를 지속했다. 멕시코가 전국적으로 노예 제도를 폐지했을 때에도 일부 백인 이민자들은 법의 이행을 거부했다. 멕시코 중앙 정부의 간섭에 대한 반발로 <a href="/wiki/%ED%85%8D%EC%82%AC%EC%8A%A4_%ED%98%81%EB%AA%85" title="텍사스 혁명">텍사스 혁명</a>이 일어나게 된다.</p>
<p><a href="/wiki/%EB%A9%95%EC%8B%9C%EC%BD%94_%EB%8F%85%EB%A6%BD_%EC%A0%84%EC%9F%81" title="멕시코 독립 전쟁">멕시코 독립 전쟁</a>으로 인하여 약화되고, 재정 능력을 상실한 멕시코 정부는 수도에서 몇천 킬로미터씩이나 떨어진 북방 지방을 제어하기에는 역부족이었다. 결국 텍사스는 <a class="mw-redirect" href="/wiki/%EC%82%B0%ED%83%80_%EC%95%88%EB%82%98" title="산타 안나">산타 안나</a>가 지도하는 멕시코 군대를 물리치고 <a href="/wiki/1836%EB%85%84" title="1836년">1836년</a> 독립을 달성하였다.</p>
<h2><span class="mw-headline" id=".EC.A0.84.EC.9F.81_.EA.B3.BC.EC.A0.95">전쟁 과정</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%A9%95%EC%8B%9C%EC%BD%94-%EB%AF%B8%EA%B5%AD_%EC%A0%84%EC%9F%81&amp;action=edit&amp;section=3" title="부분 편집: 전쟁 과정">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p><a href="/wiki/%ED%85%8D%EC%82%AC%EC%8A%A4_%EA%B3%B5%ED%99%94%EA%B5%AD" title="텍사스 공화국">텍사스 공화국</a>은 연방에 가입해 미국의 한 주가 되고 미국과 멕시코의 <a class="mw-redirect" href="/wiki/%EC%BA%98%EB%A6%AC%ED%8F%AC%EB%8B%88%EC%95%84" title="캘리포니아">캘리포니아</a>와 <a class="mw-redirect" href="/wiki/%EC%95%A0%EB%A6%AC%EC%A1%B0%EB%82%98" title="애리조나">애리조나</a> 등지의 문제로 인해 갈등이 생겼다. 텍사스를 합병한 미국은 <a href="/wiki/%EB%A6%AC%EC%98%A4%EA%B7%B8%EB%9E%80%EB%8D%B0_%EA%B0%95" title="리오그란데 강">리오그란데 강</a>을 멕시코와의 국경으로 주장했다. 그러나 멕시코는 마찬가지로 리오그란데 강 북쪽을 흐르는 <a href="/wiki/%EB%88%84%EC%97%90%EC%8B%9C%EC%8A%A4_%EA%B0%95" title="누에시스 강">누에시스 강</a>(<i>Nueses River</i>)을 국경으로 삼아야 한다고 주장하여 양국의 주장을 좁히지 못했다. 당시 미국 대통령 <a class="mw-redirect" href="/wiki/%EC%A0%9C%EC%9E%84%EC%8A%A4_%ED%8F%AC%ED%81%AC" title="제임스 포크">제임스 포크</a>는 미국이 주장하는 텍사스 땅을 확보하도록 군에 지시했다. 이 지시를 받은 <a href="/wiki/%EC%9E%AC%EC%BB%A4%EB%A6%AC_%ED%85%8C%EC%9D%BC%EB%9F%AC" title="재커리 테일러">재커리 테일러</a> 장군이 이끄는 군대는 누에세스 강을 남하하여 멕시코의 비난에도 불구하고 <a href="/wiki/%EB%B8%8C%EB%9D%BC%EC%9A%B4_%EC%9A%94%EC%83%88" title="브라운 요새">브라운 요새</a>(Fort Brown)를 세웠다.</p>
<p><a href="/wiki/1846%EB%85%84" title="1846년">1846년</a> <a href="/wiki/4%EC%9B%94_24%EC%9D%BC" title="4월 24일">4월 24일</a> 멕시코의 기병 부대가 리오그란데에서 미국의 분견대를 사로잡으며 교전을 벌였다. 〈<a href="/wiki/%ED%8C%94%EB%A1%9C_%EC%95%8C%ED%86%A0_%EC%A0%84%ED%88%AC" title="팔로 알토 전투">팔로 알토</a>〉(<i>Palo Alto</i>)와 〈<a class="mw-redirect" href="/wiki/%EB%A0%88%EC%82%AC%EC%B9%B4_%EB%93%9C_%EB%9D%BC_%ED%8C%94%EB%A7%88_%EC%A0%84%ED%88%AC" title="레사카 드 라 팔마 전투">레사카 드 라 팔마</a>〉(<i>Resaca De La Palma</i>)에서 국경 충돌과 교전 이후 미합중국군의 인명 피해가 발생하자, 의회는 <a href="/wiki/5%EC%9B%94_13%EC%9D%BC" title="5월 13일">5월 13일</a>에 <a class="mw-redirect" href="/wiki/%EC%84%A0%EC%A0%84%ED%8F%AC%EA%B3%A0" title="선전포고">선전포고</a>를 했다. 남부 출신과 <a href="/wiki/%EB%AF%BC%EC%A3%BC%EB%8B%B9_(%EB%AF%B8%EA%B5%AD)" title="민주당 (미국)">민주당</a>이 이 조치를 지지했다. 한편 북부 출신과 <a href="/wiki/%ED%9C%98%EA%B7%B8%EB%8B%B9_(%EB%AF%B8%EA%B5%AD)" title="휘그당 (미국)">휘그당</a>은 일방적인 전쟁 선포를 반대했다. 멕시코는 <a href="/wiki/5%EC%9B%94_23%EC%9D%BC" title="5월 23일">5월 23일</a>에 선전포고를 했다.</p>
<p>미합중국의 선전포고 후 미합중군은 <a href="/wiki/%EB%A1%9C%EC%8A%A4%EC%95%A4%EC%A0%A4%EB%A0%88%EC%8A%A4" title="로스앤젤레스">로스앤젤레스</a>를 포함한 캘리포니아의 여러 도시를 점령했다. 〈<a href="/wiki/%EB%AA%AC%ED%85%8C%EB%A0%88%EC%9D%B4_%EC%A0%84%ED%88%AC" title="몬테레이 전투">몬테레이 전투</a>〉는 <a href="/wiki/1846%EB%85%84" title="1846년">1846년</a> 9월에 일어났다. <a href="/wiki/1847%EB%85%84" title="1847년">1847년</a> <a href="/wiki/2%EC%9B%94_22%EC%9D%BC" title="2월 22일">2월 22일</a>, 〈<a href="/wiki/%EB%B6%80%EC%97%90%EB%82%98_%EB%B9%84%EC%8A%A4%ED%83%80_%EC%A0%84%ED%88%AC" title="부에나 비스타 전투">부에나 비스타 전투</a>〉에서 테일러 장군이 <a href="/wiki/%EC%95%88%ED%86%A0%EB%8B%88%EC%98%A4_%EB%A1%9C%ED%8E%98%EC%8A%A4_%EB%8D%B0_%EC%82%B0%ED%83%80_%EC%95%88%EB%82%98" title="안토니오 로페스 데 산타 안나">안토니오 로페스 데 산타 안나</a> 장군 휘하의 멕시코 군을 물리치고 <a class="mw-redirect" href="/wiki/%EC%95%8C%ED%83%80%EC%BA%98%EB%A6%AC%ED%8F%AC%EB%8B%88%EC%95%84" title="알타캘리포니아">알타캘리포니아</a>와 <a class="mw-redirect" href="/wiki/%EB%89%B4%EB%A9%95%EC%8B%9C%EC%BD%94" title="뉴멕시코">뉴멕시코</a>의 점령을 확고히 했다. 미합중국군은 멕시코군의 4배나 되는 대군으로 멕시코군을 격파하여 <a class="mw-redirect" href="/wiki/%EB%89%B4%EB%A9%95%EC%8B%9C%EC%BD%94" title="뉴멕시코">뉴멕시코</a>의 수도 <a class="mw-redirect" href="/wiki/%EC%83%8C%ED%83%80%ED%8E%98%EC%9D%B4" title="샌타페이">샌타페이</a>를 점령했다. <a href="/wiki/1847%EB%85%84" title="1847년">1847년</a> 3월에는 <a href="/wiki/%EC%9C%88%ED%95%84%EB%93%9C_%EC%8A%A4%EC%BD%A7" title="윈필드 스콧">윈필드 스콧</a> 장군이 이끄는 미합중군이 역사상 최초의 수륙양용 군사작전을 수행해 <a href="/wiki/%EB%A9%95%EC%8B%9C%EC%BD%94" title="멕시코">멕시코</a> <a href="/wiki/%EB%B2%A0%EB%9D%BC%ED%81%AC%EB%A3%A8%EC%8A%A4" title="베라크루스">베라크루스</a>에 상륙했다. <a href="/wiki/%EB%A9%95%EC%8B%9C%EC%BD%94" title="멕시코">멕시코</a> 사령관은 항복하고 <a href="/wiki/1847%EB%85%84" title="1847년">1847년</a> <a href="/wiki/9%EC%9B%94_14%EC%9D%BC" title="9월 14일">9월 14일</a> <a href="/wiki/%EB%A9%95%EC%8B%9C%EC%BD%94" title="멕시코">멕시코</a>의 수도 <a href="/wiki/%EB%A9%95%EC%8B%9C%EC%BD%94%EC%8B%9C%ED%8B%B0" title="멕시코시티">멕시코시티</a>는 미군에게 함락되었다.</p>
<p>결국 <a href="/wiki/1848%EB%85%84" title="1848년">1848년</a> 1월 멕시코는 평화 협정을 요청해 <a href="/wiki/1848%EB%85%84" title="1848년">1848년</a> <a href="/wiki/2%EC%9B%94_2%EC%9D%BC" title="2월 2일">2월 2일</a> <a href="/wiki/%EB%A9%95%EC%8B%9C%EC%BD%94" title="멕시코">멕시코</a> <a class="new" href="/w/index.php?title=%EA%B3%BC%EB%8B%AC%EB%A3%A8%ED%8E%98_%EC%9D%B4%EB%8B%AC%EA%B3%A0&amp;action=edit&amp;redlink=1" title="과달루페 이달고 (없는 문서)">과달루페 이달고</a>에서 《<a class="mw-redirect" href="/wiki/%EA%B3%BC%EB%8B%AC%EB%A3%A8%ED%8E%98-%EC%9D%B4%EB%8B%AC%EA%B3%A0_%EC%A1%B0%EC%95%BD" title="과달루페-이달고 조약">과달루페-이달고 조약</a>》을 맺어 전쟁은 종결되었다.</p>
<h2><span class="mw-headline" id=".EA.B2.B0.EA.B3.BC_.EB.B0.8F_.EC.98.81.ED.96.A5">결과 및 영향</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%A9%95%EC%8B%9C%EC%BD%94-%EB%AF%B8%EA%B5%AD_%EC%A0%84%EC%9F%81&amp;action=edit&amp;section=4" title="부분 편집: 결과 및 영향">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p><a href="/wiki/%EA%B3%BC%EB%8B%AC%EB%A3%A8%ED%8E%98_%EC%9D%B4%EB%8B%AC%EA%B3%A0_%EC%A1%B0%EC%95%BD" title="과달루페 이달고 조약">과달루페 이달고 조약</a>으로 미국은 겨우 1,825만 달러를 지급하고 멕시코로부터 <a class="mw-redirect" href="/wiki/%EB%89%B4%EB%A9%95%EC%8B%9C%EC%BD%94" title="뉴멕시코">뉴멕시코</a>, <a class="mw-redirect" href="/wiki/%EC%BA%98%EB%A6%AC%ED%8F%AC%EB%8B%88%EC%95%84" title="캘리포니아">캘리포니아</a>, <a class="mw-redirect" href="/wiki/%EC%BD%9C%EB%A1%9C%EB%9D%BC%EB%8F%84" title="콜로라도">콜로라도</a>, <a class="mw-redirect" href="/wiki/%EC%95%A0%EB%A6%AC%EC%A1%B0%EB%82%98" title="애리조나">애리조나</a>, <a class="mw-redirect" href="/wiki/%EB%84%A4%EB%B0%94%EB%8B%A4" title="네바다">네바다</a>, <a class="mw-redirect" href="/wiki/%EC%9C%A0%ED%83%80" title="유타">유타</a> 주 등을 할양 받아 한반도 넓이의 15배에 달하는 300만 평방킬로미터의 영토를 넓혔다. 미국은 겨우 멕시코인들에게 진 빚 325만 달러를 떠맡도록 하였고 군사비 지출도 9,800만 달러밖에 되지 않았다고 한다. 하지만 전쟁 직전 일부 미국 여론들은 전쟁을 반대했는데 지식인 <a href="/wiki/%ED%97%A8%EB%A6%AC_%EB%8D%B0%EC%9D%B4%EB%B9%84%EB%93%9C_%EC%86%8C%EB%A1%9C" title="헨리 데이비드 소로">헨리 데이비드 소로</a>는 전쟁세를 내는 대신 감옥에 가는 것으로 항의했고 <a class="mw-redirect" href="/wiki/%EB%9E%84%ED%94%84_%EC%99%88%EB%8F%84_%EC%97%90%EB%A8%B8%EC%8A%A8" title="랄프 왈도 에머슨">랄프 왈도 에머슨</a>은 미국의 폭력을 부끄러워하며 멕시코가 미국에게 독이 될 것이라고 했다. 또한 <a class="mw-redirect mw-disambig" href="/wiki/%EB%82%A8%EB%B6%81%EC%A0%84%EC%9F%81" title="남북전쟁">남북전쟁</a>의 전쟁 영웅 <a class="mw-redirect" href="/wiki/%EC%9C%A8%EB%A6%AC%EC%8B%9C%EC%8A%A4_%EA%B7%B8%EB%9E%9C%ED%8A%B8" title="율리시스 그랜트">율리시스 그랜트</a> 역시 이를 두고두고 한탄했다고 전해진다.</p>
