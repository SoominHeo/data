<p><b>멜빈 에머리 "멜" 패턴</b>(Melvin Emery "Mel" Patton, <a href="/wiki/1924%EB%85%84" title="1924년">1924년</a> <a href="/wiki/11%EC%9B%94_16%EC%9D%BC" title="11월 16일">11월 16일</a> ~ <a href="/wiki/2014%EB%85%84" title="2014년">2014년</a> <a href="/wiki/5%EC%9B%94_9%EC%9D%BC" title="5월 9일">5월 9일</a>)은 <a href="/wiki/%EB%AF%B8%EA%B5%AD" title="미국">미국</a>의 <a class="mw-redirect" href="/wiki/%EB%8B%A8%EA%B1%B0%EB%A6%AC%EB%8B%AC%EB%A6%AC%EA%B8%B0" title="단거리달리기">단거리달리기</a> 선수로 <a href="/wiki/1948%EB%85%84_%ED%95%98%EA%B3%84_%EC%98%AC%EB%A6%BC%ED%94%BD" title="1948년 하계 올림픽">1948년 하계 올림픽</a> 2관왕이었다.</p>
<p><a href="/wiki/%EB%A1%9C%EC%8A%A4%EC%95%A4%EC%A0%A4%EB%A0%88%EC%8A%A4" title="로스앤젤레스">로스앤젤레스</a>에서 태어나 1940년대 후반에 "펠 멜"이라는 별명이 붙으면서 <a href="/wiki/%EC%84%9C%EB%8D%98%EC%BA%98%EB%A6%AC%ED%8F%AC%EB%8B%88%EC%95%84_%EB%8C%80%ED%95%99%EA%B5%90" title="서던캘리포니아 대학교">서던캘리포니아 대학교</a>에서 딘 크롬웰의 코치를 받아 육상에서 자신의 주목을 만들었다.</p>
<p><a href="/wiki/1947%EB%85%84" title="1947년">1947년</a> NCAA 100 야드 경주를 우승하고 <a href="/wiki/1948%EB%85%84" title="1948년">1948년</a>과 <a href="/wiki/1949%EB%85%84" title="1949년">1949년</a>에는 100 야드와 220 야드 2종목을 같은 경기에서 완료하였다. <a href="/wiki/1947%EB%85%84" title="1947년">1947년</a>에는 자신이 이듬해 9.3초로 낮춘 100 야드 경주 세계 기록 9.4초의 동점을 이루었다. <a href="/wiki/1949%EB%85%84" title="1949년">1949년</a>에는 220 야드의 세계 기록 20.2초를 세워 <a href="/wiki/%EC%A0%9C%EC%8B%9C_%EC%98%A4%EC%96%B8%EC%8A%A4" title="제시 오언스">제시 오언스</a>의 기록을 깼다.</p>
<p>올림픽 선발 시합의 100m 결승전에서 <a class="new" href="/w/index.php?title=%EB%B0%94%EB%8B%88_%EC%9C%A0%EC%96%BC&amp;action=edit&amp;redlink=1" title="바니 유얼 (없는 문서)">바니 유얼</a>에게 패하고나서 <a href="/wiki/1948%EB%85%84_%ED%95%98%EA%B3%84_%EC%98%AC%EB%A6%BC%ED%94%BD" title="1948년 하계 올림픽">런던 올림픽</a>에서는 5위에 그쳤다. 200m 경주와 400m 릴레이에서 2개의 <a href="/wiki/%EA%B8%88%EB%A9%94%EB%8B%AC" title="금메달">금메달</a>을 획득하면서 자신의 실망에 보상을 받았다.</p>
<p>육상에서 은퇴한 후, 전자 산업에 들어가기 전에 코치를 지냈다. 89세의 나이로 <a href="/wiki/%EC%BA%98%EB%A6%AC%ED%8F%AC%EB%8B%88%EC%95%84_%EC%A3%BC" title="캘리포니아 주">캘리포니아 주</a> 폴브룩에서 사망하였다.</p>
