<p><b>멜버른 빅토리 FC</b> (Melbourne Victory Football Club)은 <a href="/wiki/A%EB%A6%AC%EA%B7%B8" title="A리그">A리그</a>에 소속된 <a href="/wiki/%EC%98%A4%EC%8A%A4%ED%8A%B8%EB%A0%88%EC%9D%BC%EB%A6%AC%EC%95%84" title="오스트레일리아">오스트레일리아</a>의 프로축구단으로 <a href="/wiki/%EB%B9%85%ED%86%A0%EB%A6%AC%EC%95%84_%EC%A3%BC" title="빅토리아 주">빅토리아 주</a>의 <a href="/wiki/%EB%A9%9C%EB%B2%84%EB%A5%B8" title="멜버른">멜버른</a>을 연고로 <a href="/wiki/2004%EB%85%84" title="2004년">2004년</a>에 창단되었다. 2008년 현재, 멜버른 빅토리는 2006-07시즌에 프리미어십과 챔피언십을 동시에 우승하면서 더블을 기록한 유일한 팀이다. 또한 <a href="/wiki/2008%EB%85%84" title="2008년">2008년</a>의 프리시즌 챌린지컵을 우승하면서 세 가지 대회를 모두 우승한 유일한 팀이기도 하다.</p>

<h2><span class="mw-headline" id=".EC.97.AD.EC.82.AC">역사</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%A9%9C%EB%B2%84%EB%A5%B8_%EB%B9%85%ED%86%A0%EB%A6%AC_FC&amp;action=edit&amp;section=1" title="부분 편집: 역사">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>멜버른 빅토리는 <a href="/wiki/2004%EB%85%84" title="2004년">2004년</a>에 오스트레일리아의 국내 리그인 <a class="new" href="/w/index.php?title=%EB%82%B4%EC%85%94%EB%84%90_%EC%82%AC%EC%BB%A4_%EB%A6%AC%EA%B7%B8&amp;action=edit&amp;redlink=1" title="내셔널 사커 리그 (없는 문서)">내셔널 사커 리그</a>가 현재의 <a href="/wiki/A%EB%A6%AC%EA%B7%B8" title="A리그">A리그</a>로 바뀐다는 발표가 있고 나서 창단되었다. 멜버른 빅토리는 2부 리그인 <a class="new" href="/w/index.php?title=%EB%B9%85%ED%86%A0%EB%A6%AC%EC%95%84_%ED%94%84%EB%A6%AC%EB%AF%B8%EC%96%B4%EB%A6%AC%EA%B7%B8&amp;action=edit&amp;redlink=1" title="빅토리아 프리미어리그 (없는 문서)">빅토리아 프리미어리그</a>에 참가하고 있는 <a class="new" href="/w/index.php?title=%EC%82%AC%EC%9A%B0%EC%8A%A4_%EB%A9%9C%EB%B2%84%EB%A5%B8_FC&amp;action=edit&amp;redlink=1" title="사우스 멜버른 FC (없는 문서)">사우스 멜버른 FC</a>와 <a class="new" href="/w/index.php?title=%EB%A9%9C%EB%B2%84%EB%A5%B8_%EB%82%98%EC%9D%B4%EC%B8%A0&amp;action=edit&amp;redlink=1" title="멜버른 나이츠 (없는 문서)">멜버른 나이츠</a>를 대신하여 A리그에 참가하였다.</p>
<p>2004-05시즌엔 리그가 없었기 때문에 멜버른 빅토리는 팀의 구성에 나섰다. 이 기간 동안 <a href="/wiki/%EC%98%A4%EC%8A%A4%ED%8A%B8%EB%A0%88%EC%9D%BC%EB%A6%AC%EC%95%84_%EC%B6%95%EA%B5%AC_%EA%B5%AD%EA%B0%80%EB%8C%80%ED%91%9C%ED%8C%80" title="오스트레일리아 축구 국가대표팀">오스트레일리아 축구 국가대표팀</a> 출신 선수들을 영입하는 등 팀의 실력 향상에 힘썼다.</p>
<p>클럽의 첫 시즌이었던 2005-06시즌은 8개 팀 가운데 7위로 마감하면서 좋지 않은 시즌을 보냈다.</p>
<p>다음 시즌인 2006-07시즌에는 정규리그에서 1위를 하면서 프리미어를 차지하였다. 또한 상위 네 팀이 참가하는 챔피언십에서도 <a href="/wiki/%EC%95%A0%EB%93%A4%EB%A0%88%EC%9D%B4%EB%93%9C_%EC%9C%A0%EB%82%98%EC%9D%B4%ED%8B%B0%EB%93%9C_FC" title="애들레이드 유나이티드 FC">애들레이드 유나이티드</a>를 누르고 우승을 차지하며 프리미어-챔피언을 동시에 차지하는 더블을 기록하였다.</p>
<p>그 후 2007-08시즌을 5위로 마감하고, 2008-09시즌엔 시즌 전 펼쳐는 프리시즌 챌린지컵에서 우승을 차지하였고 정규 리그에서도 좋은 모습을 보이며 그랜드 파이널에 진출하였고 그랜드 파이널에서 승리하며 리그 우승을 거머쥐었다. 이후 출전한 <a href="/wiki/AFC_%EC%B1%94%ED%94%BC%EC%96%B8%EC%8A%A4%EB%A6%AC%EA%B7%B8_2010" title="AFC 챔피언스리그 2010">2010 AFC 챔피언스리그</a>에선 조별 예선에서 <a href="/wiki/%EC%84%B1%EB%82%A8_FC" title="성남 FC">성남 FC</a>와 <a href="/wiki/%EB%B2%A0%EC%9D%B4%EC%A7%95_%EA%B6%88%EC%95%88" title="베이징 궈안">베이징 궈안</a>에 밀려 탈락하였다.</p>
