<p><b>멤피스 국제공항</b>(<span style="font-size: smaller;"><a href="/wiki/%EC%98%81%EC%96%B4" title="영어">영어</a>:</span> <span lang="en">Memphis International Airport</span>, <a href="/wiki/%EA%B5%AD%EC%A0%9C_%ED%95%AD%EA%B3%B5_%EC%9A%B4%EC%86%A1_%ED%98%91%ED%9A%8C" title="국제 항공 운송 협회">IATA</a>: <b>MEM</b>, <a href="/wiki/%EA%B5%AD%EC%A0%9C_%EB%AF%BC%EA%B0%84_%ED%95%AD%EA%B3%B5_%EA%B8%B0%EA%B5%AC" title="국제 민간 항공 기구">ICAO</a>: <b>KMEM</b>)은 <a href="/wiki/%EB%AF%B8%EA%B5%AD" title="미국">미국</a> <a href="/wiki/%ED%85%8C%EB%84%A4%EC%8B%9C_%EC%A3%BC" title="테네시 주">테네시 주</a> <a href="/wiki/%EB%A9%A4%ED%94%BC%EC%8A%A4_(%ED%85%8C%EB%84%A4%EC%8B%9C_%EC%A3%BC)" title="멤피스 (테네시 주)">멤피스</a>에 위치한 <a href="/wiki/%EA%B5%AD%EC%A0%9C%EA%B3%B5%ED%95%AD" title="국제공항">국제공항</a>으로 <a href="/wiki/%ED%8E%98%EB%8D%B1%EC%8A%A4_%EC%9D%B5%EC%8A%A4%ED%94%84%EB%A0%88%EC%8A%A4" title="페덱스 익스프레스">페덱스 익스프레스</a>의 세계적인 슈퍼허브로 항공 화물 운송에 있어 큰 비중을 차지하고 있다.</p>

<h2><span class="mw-headline" id=".EA.B0.9C.EC.9A.94">개요</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%A9%A4%ED%94%BC%EC%8A%A4_%EA%B5%AD%EC%A0%9C%EA%B3%B5%ED%95%AD&amp;action=edit&amp;section=1" title="부분 편집: 개요">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>멤피스 국제공항은 <a href="/wiki/%ED%8E%98%EB%8D%B1%EC%8A%A4_%EC%9D%B5%EC%8A%A4%ED%94%84%EB%A0%88%EC%8A%A4" title="페덱스 익스프레스">페덱스 익스프레스</a>의 화물 직항편이 <a href="/wiki/%EB%AF%B8%EA%B5%AD" title="미국">미국</a>의 여러 도시를 비롯하여 <a href="/wiki/%EC%9C%A0%EB%9F%BD" title="유럽">유럽</a>, <a href="/wiki/%EB%82%A8%EC%95%84%EB%A9%94%EB%A6%AC%EC%B9%B4" title="남아메리카">남아메리카</a>, <a href="/wiki/%EC%9D%BC%EB%B3%B8" title="일본">일본</a>까지 운항되고 있을 정도로 이와 같은 화물기 운영에 힘입어 멤피스 국제공항은 <a href="/wiki/1993%EB%85%84" title="1993년">1993년</a>부터 현재까지 세계 최대의 화물 운송량을 기록하고 있다. 또한 <a href="/wiki/%EB%8D%B8%ED%83%80_%ED%95%AD%EA%B3%B5" title="델타 항공">델타 항공</a>의 <a href="/wiki/%ED%97%88%EB%B8%8C_%EA%B3%B5%ED%95%AD" title="허브 공항">허브 공항</a>으로 <a class="mw-redirect" href="/wiki/%EC%8A%A4%ED%9E%88%ED%8F%B4_%EA%B5%AD%EC%A0%9C%EA%B3%B5%ED%95%AD" title="스히폴 국제공항">암스테르담</a>행 직항편이 매일 운영되고 있다. <a href="/wiki/%EB%8D%B8%ED%83%80_%ED%95%AD%EA%B3%B5" title="델타 항공">델타 항공</a>에 합병되기 전에 <a href="/wiki/%EB%85%B8%EC%8A%A4%EC%9B%A8%EC%8A%A4%ED%8A%B8_%ED%95%AD%EA%B3%B5" title="노스웨스트 항공">노스웨스트 항공</a>은 <a href="/wiki/%EB%82%98%EB%A6%AC%ED%83%80_%EA%B5%AD%EC%A0%9C%EA%B3%B5%ED%95%AD" title="나리타 국제공항">도쿄</a>를 오가는 직항편을 만들 것을 검토하기도 했다.</p>
<h2><span class="mw-headline" id=".EC.97.AD.EC.82.AC">역사</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%A9%A4%ED%94%BC%EC%8A%A4_%EA%B5%AD%EC%A0%9C%EA%B3%B5%ED%95%AD&amp;action=edit&amp;section=2" title="부분 편집: 역사">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>멤피스 국제공항은 멤피스 시가지로부터 10km 떨어진 200에이커(0.8km²)의 농경지에 처음 만들어졌다. 설립 초기의 멤피스 국제공항은 3개의 격납고와 비포장된 활주로로 이루어져 있었다. 이때 당시의 승객 및 항공 우편 운송은 <a href="/wiki/%EC%95%84%EB%A9%94%EB%A6%AC%EC%B9%B8_%ED%95%AD%EA%B3%B5" title="아메리칸 항공">아메리칸 항공</a>과 시카고, 남부 항공의 항공편이 운영되었다. <a class="mw-redirect" href="/wiki/%EC%A0%9C2%EC%B0%A8_%EC%84%B8%EA%B3%84%EB%8C%80%EC%A0%84" title="제2차 세계대전">제2차 세계대전</a> 당시 <a href="/wiki/%EB%AF%B8%EA%B5%AD_%EA%B3%B5%EA%B5%B0" title="미국 공군">미국 공군</a> 항공 수송 사령부에 의해 새로 제작된 항공기를 수송하는 거점 역할을 맡기도 했으며 현재 사용하고 있는 터미널 건물은 <a href="/wiki/1963%EB%85%84" title="1963년">1963년</a> 지어졌으며, <a href="/wiki/1969%EB%85%84" title="1969년">1969년</a>에 공항의 이름을 멤피스 지역공항에서 멤피스 국제공항으로 바꾼다. 그러나 <a href="/wiki/1995%EB%85%84" title="1995년">1995년</a> 최초의 국제선 직항편인 암스테르담행 항공편이 운영되기 전까지는 국제선 직항편을 갖지 못했다. <a href="/wiki/1973%EB%85%84" title="1973년">1973년</a>에 <a href="/wiki/%ED%8E%98%EB%8D%B1%EC%8A%A4_%EC%9D%B5%EC%8A%A4%ED%94%84%EB%A0%88%EC%8A%A4" title="페덱스 익스프레스">페덱스 익스프레스</a>의 화물 운송 허브가 설립되었고 <a href="/wiki/1985%EB%85%84" title="1985년">1985년</a>에 <a class="new" href="/w/index.php?title=%EB%A6%AC%ED%8D%BC%EB%B8%94%EB%A6%AD_%ED%95%AD%EA%B3%B5&amp;action=edit&amp;redlink=1" title="리퍼블릭 항공 (없는 문서)">리퍼블릭 항공</a>의 승객 허브가 만들어졌다. 이는 다음해인 <a href="/wiki/1986%EB%85%84" title="1986년">1986년</a> <a href="/wiki/%EB%85%B8%EC%8A%A4%EC%9B%A8%EC%8A%A4%ED%8A%B8_%ED%95%AD%EA%B3%B5" title="노스웨스트 항공">노스웨스트 항공</a>에 의해 인수되었고 다시 <a href="/wiki/2008%EB%85%84" title="2008년">2008년</a> <a href="/wiki/%EB%8D%B8%ED%83%80_%ED%95%AD%EA%B3%B5" title="델타 항공">델타 항공</a>에 의해 인수되면서 현재는 <a class="mw-redirect" href="/wiki/%ED%97%88%EB%B8%8C%EA%B3%B5%ED%95%AD" title="허브공항">허브공항</a>이 되었다.</p>

<h3><span class="mw-headline" id=".EB.8F.84.EB.A1.9C_.EA.B5.90.ED.86.B5">도로 교통</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%A9%A4%ED%94%BC%EC%8A%A4_%EA%B5%AD%EC%A0%9C%EA%B3%B5%ED%95%AD&amp;action=edit&amp;section=6" title="부분 편집: 도로 교통">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<ul>
<li>호텔 셔틀버스의 경우 멤피스 지역의 호텔과 모텔을 연계한다.</li>

<li>렌탈 카의 경우 모든 수하물 수취 지역에서 출구 근처에서 승차한다.</li>

<h2><span class="mw-headline" id=".EC.82.AC.EA.B1.B4_.EB.B0.8F_.EC.82.AC.EA.B3.A0">사건 및 사고</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%A9%A4%ED%94%BC%EC%8A%A4_%EA%B5%AD%EC%A0%9C%EA%B3%B5%ED%95%AD&amp;action=edit&amp;section=13" title="부분 편집: 사건 및 사고">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<div class="rellink relarticle mainarticle"><img alt="" data-file-height="128" data-file-width="128" height="16" src="//upload.wikimedia.org/wikipedia/commons/thumb/e/ec/Crystal_Clear_app_xmag.svg/16px-Crystal_Clear_app_xmag.svg.png" srcset="//upload.wikimedia.org/wikipedia/commons/thumb/e/ec/Crystal_Clear_app_xmag.svg/24px-Crystal_Clear_app_xmag.svg.png 1.5x, //upload.wikimedia.org/wikipedia/commons/thumb/e/ec/Crystal_Clear_app_xmag.svg/32px-Crystal_Clear_app_xmag.svg.png 2x" width="16"/> 이 부분의 본문은 <a href="/wiki/%ED%8E%98%EB%8D%B1%EC%8A%A4_%EC%9D%B5%EC%8A%A4%ED%94%84%EB%A0%88%EC%8A%A4_647%ED%8E%B8" title="페덱스 익스프레스 647편">페덱스 익스프레스 647편</a>입니다.</div>
<ul>
<li><a href="/wiki/2003%EB%85%84" title="2003년">2003년</a> <a href="/wiki/12%EC%9B%94_18%EC%9D%BC" title="12월 18일">12월 18일</a> <a href="/wiki/%EB%AF%B8%EA%B5%AD" title="미국">미국</a> <a class="new" href="/w/index.php?title=%EC%98%A4%ED%81%B4%EB%9E%9C%EB%93%9C_%EA%B5%AD%EC%A0%9C%EA%B3%B5%ED%95%AD_(%EB%AF%B8%EA%B5%AD)&amp;action=edit&amp;redlink=1" title="오클랜드 국제공항 (미국) (없는 문서)">오클랜드 국제공항</a>을 출발하여 멤피스 국제공항에 착률할 예정이었던 <a href="/wiki/%ED%8E%98%EB%8D%B1%EC%8A%A4_%EC%9D%B5%EC%8A%A4%ED%94%84%EB%A0%88%EC%8A%A4_647%ED%8E%B8" title="페덱스 익스프레스 647편">페덱스 익스프레스 647편</a>이 멤피스 국제공항에 착륙도중 항공기가 추락한 사고로 비행기 앞부분이 전소되었다. 사고 원인은 조종사의 실수로 일어난 사고로 <a href="/wiki/1978%EB%85%84" title="1978년">1978년</a> 설립 이후 최초의 사고로 기록됐다.</li>
</ul>
