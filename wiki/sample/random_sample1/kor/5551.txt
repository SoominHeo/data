<h1 class="firstHeading" id="firstHeading" lang="ko">면역학</h1>
<div class="mw-body-content" id="bodyContent">
<div id="siteSub">위키백과, 우리 모두의 백과사전.</div>
<div id="contentSub"></div>

<div class="mw-content-ltr" dir="ltr" id="mw-content-text" lang="ko"><script>function mfTempOpenSection(id){var block=document.getElementById("mf-section-"+id);block.className+=" open-block";block.previousSibling.className+=" open-block";}</script><p><b>면역학</b>(免疫學, <span lang="en" xml:lang="en">immunology</span>)은 여러 <a href="/wiki/%EC%83%9D%EB%AC%BC%EC%B2%B4" title="생물체">생물체</a>의 <a href="/wiki/%EB%A9%B4%EC%97%AD%EA%B3%84" title="면역계">면역계</a>에 대해 연구하는 <a class="new" href="/w/index.php?title=%EC%83%9D%EB%AC%BC%EC%9D%98%ED%95%99&amp;action=edit&amp;redlink=1" title="생물의학 (없는 문서)">생물의학</a>(biomedical)의 한 분야이다.<sup class="reference" id="cite_ref-1"><a href="#cite_note-1">[1]</a></sup> 건강한 상태 및 질병에 걸린 상태에서의 면역계의 <a href="/wiki/%EC%83%9D%EB%A6%AC%ED%95%99" title="생리학">생리적</a> 작용에 대해서 다루며, 특히 면역계의 오류로 인한 질환(<a class="mw-redirect" href="/wiki/%EC%9E%90%EA%B0%80_%EB%A9%B4%EC%97%AD_%EC%A7%88%ED%99%98" title="자가 면역 질환">자가 면역 질환</a>, <a href="/wiki/%EA%B3%BC%EB%AF%BC%EC%A6%9D" title="과민증">과민증</a>, <a class="new" href="/w/index.php?title=%EB%A9%B4%EC%97%AD_%EA%B2%B0%ED%95%8D%EC%A6%9D&amp;action=edit&amp;redlink=1" title="면역 결핍증 (없는 문서)">면역 결핍증</a>, <a class="new" href="/w/index.php?title=%EC%9D%B4%EC%8B%9D_%EA%B1%B0%EB%B6%80&amp;action=edit&amp;redlink=1" title="이식 거부 (없는 문서)">이식 거부</a>) 및 면역계의 생리학적 특성에 대해서 관심있게 다루고 있다.</p>

<h2><span class="mw-headline" id=".EC.97.AD.EC.82.AC">역사</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%A9%B4%EC%97%AD%ED%95%99&amp;action=edit&amp;section=1" title="부분 편집: 역사">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>면역 반응은 과거부터 '한번 병에 걸리면 다음에는 그 병에 잘 걸리지 않는다'라고 알려져 왔다. 이를 실제로 적용한 사람은 18세기 말의 <a href="/wiki/%EC%97%90%EB%93%9C%EC%9B%8C%EB%93%9C_%EC%A0%9C%EB%84%88" title="에드워드 제너">제너</a>로, 소에게 걸리는 <a class="mw-redirect" href="/wiki/%EC%9A%B0%EB%91%90_(%EB%B0%94%EC%9D%B4%EB%9F%AC%EC%8A%A4)" title="우두 (바이러스)">우두</a>(cowpox)를 이용하여 사람의 치명적인 병이었던 <a href="/wiki/%EC%B2%9C%EC%97%B0%EB%91%90" title="천연두">천연두</a>(smallpox)의 예방 방법을 적용한 것을 면역학의 시초로 볼 수 있다. 이후 각종 질병의 원인균들이 발견되고, 프랑스의 <a href="/wiki/%EB%A3%A8%EC%9D%B4_%ED%8C%8C%EC%8A%A4%ED%87%B4%EB%A5%B4" title="루이 파스퇴르">루이 파스퇴르</a>(Louis Pasteur)가 최초로 근대화된 <a class="new" href="/w/index.php?title=%EC%98%88%EB%B0%A9%EC%A0%91%EC%A2%85%EB%B2%95&amp;action=edit&amp;redlink=1" title="예방접종법 (없는 문서)">예방접종법</a>(vaccination)을 개발하면서 본격적으로 의학과 함께 발전하기 시작했다. 현재 인간의 각종 질병들, 특히 <a class="mw-redirect" href="/wiki/%ED%9B%84%EC%B2%9C%EC%84%B1_%EB%A9%B4%EC%97%AD_%EA%B2%B0%ED%95%8D_%EC%A6%9D%ED%9B%84%EA%B5%B0" title="후천성 면역 결핍 증후군">AIDS</a>, <a href="/wiki/%EC%95%94" title="암">암</a>과 같은 난치성 질환에 대한 면역치료는 큰 과제로 남아있다.</p>
<h2><span class="mw-headline" id=".EA.B0.9C.EC.9A.94">개요</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%A9%B4%EC%97%AD%ED%95%99&amp;action=edit&amp;section=2" title="부분 편집: 개요">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>인간(또는 분류 상 파충류, 조류, 포유류에 속하는 동물들)의 면역 반응은 혈액 속의 백혈구에 의해 일어나며, 크게 <a class="mw-redirect" href="/wiki/%EC%B2%B4%EC%95%A1%EC%84%B1_%EB%A9%B4%EC%97%AD" title="체액성 면역">체액성 면역</a> (humoral immunity)과 <a class="mw-redirect" href="/wiki/%EC%84%B8%ED%8F%AC%EC%84%B1_%EB%A9%B4%EC%97%AD" title="세포성 면역">세포성 면역</a> (T-cell mediated immunity)으로 나눌 수 있다. 이러한 면역반응은 내게 속한 것들에 대해서는 관용(tolerance)을 보이고 그 외에는 모두 공격하는 반응인데, 이러한 반응을 잘 이용하면 질병을 치료하는 데 크게 도움이 된다. 그러나 면역반응이 잘못 일어나면 나 자신을 공격하게 될 수도 있으며 이로 인해 다양한 면역 관련 질환이 발생하게 된다. <a class="mw-redirect" href="/wiki/%EC%9D%B8%EC%8A%90%EB%A6%B0_%EC%9D%98%EC%A1%B4%ED%98%95_%EB%8B%B9%EB%87%A8%EB%B3%91" title="인슐린 의존형 당뇨병">인슐린 의존형 당뇨병</a> (insulin-dependent diabetes mellitus), <a class="mw-redirect" href="/wiki/%EB%A5%98%EB%A7%88%ED%8B%B0%EC%A6%98" title="류마티즘">류마티즘</a> (Rheumatoid arthritis), <a class="mw-redirect" href="/wiki/%ED%99%8D%EB%B0%98%EC%84%B1_%EB%82%AD%EC%B0%BD" title="홍반성 낭창">홍반성 낭창</a> (Systemic lupus erythematosus) 이 대표적인 면역 관련 질환이다. 또한 조직 이식의 경우 이러한 면역반응으로 인해 이식된 조직이 파괴되기도 하며, 따라서 면역기능의 적절한 억제 또한 면역학의 중요한 과제이다.</p>
<h2><span class="mw-headline" id=".EB.A9.B4.EC.97.AD_.EB.B0.98.EC.9D.91_.EC.A2.85.EB.A5.98">면역 반응 종류</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%A9%B4%EC%97%AD%ED%95%99&amp;action=edit&amp;section=3" title="부분 편집: 면역 반응 종류">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>면역 반응은 크게 2가지로 나누어 이해할 수 있다. 이 때, 2가지로 구분하는 방법 자체가 2가지가 있다.</p>
<ul>
<li><a class="mw-redirect" href="/wiki/%EC%B2%B4%EC%95%A1%EC%84%B1_%EB%A9%B4%EC%97%AD" title="체액성 면역">체액성 면역</a>(humoral immunity) vs. <a class="mw-redirect" href="/wiki/%EC%84%B8%ED%8F%AC%EC%84%B1_%EB%A9%B4%EC%97%AD" title="세포성 면역">세포성 면역</a>(cell-mediated immunity)</li>
<li><a class="mw-redirect" href="/wiki/%EC%84%A0%EC%B2%9C%EC%84%B1_%EB%A9%B4%EC%97%AD" title="선천성 면역">선천성 면역</a>(innate immunity) vs. <a class="new" href="/w/index.php?title=%EC%A0%81%EC%9D%91%EC%84%B1_%EB%A9%B4%EC%97%AD&amp;action=edit&amp;redlink=1" title="적응성 면역 (없는 문서)">적응성 면역</a>(adaptive immunity)</li>
</ul>
<h2><span class="mw-headline" id=".EA.B4.80.EB.A0.A8_.EC.84.B8.ED.8F.AC">관련 세포</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%A9%B4%EC%97%AD%ED%95%99&amp;action=edit&amp;section=4" title="부분 편집: 관련 세포">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>혈구중 하나인 백혈구는 크게 식세포와 림프구로 나뉜다.</p>
<p><br/>
<b>식세포</b></p>
<ul>
<li><a class="new" href="/w/index.php?title=%ED%98%B8%EC%97%BC%EA%B8%B0%EC%84%B1%EB%B0%B1%ED%98%88%EA%B5%AC&amp;action=edit&amp;redlink=1" title="호염기성백혈구 (없는 문서)">호염기성백혈구</a> : <a class="mw-redirect" href="/wiki/%EC%95%8C%EB%9F%AC%EC%A7%80" title="알러지">알러지</a> 반응에 관여한다. <a href="/wiki/%ED%9E%88%EC%8A%A4%ED%83%80%EB%AF%BC" title="히스타민">히스타민</a>을 분비하며 <a href="/wiki/T%EC%84%B8%ED%8F%AC" title="T세포">T세포</a>의 분화를 촉진한다.</li>
<li><a class="new" href="/w/index.php?title=%ED%98%B8%EC%82%B0%EC%84%B1%EB%B0%B1%ED%98%88%EA%B5%AC&amp;action=edit&amp;redlink=1" title="호산성백혈구 (없는 문서)">호산성백혈구</a> : <a class="mw-redirect" href="/wiki/%EA%B8%B0%EC%83%9D%EC%B6%A9" title="기생충">기생충</a>을 죽인다.</li>
<li><a class="new" href="/w/index.php?title=%ED%98%B8%EC%A4%91%EC%84%B1%EB%B0%B1%ED%98%88%EA%B5%AC&amp;action=edit&amp;redlink=1" title="호중성백혈구 (없는 문서)">호중성백혈구</a> : <a href="/wiki/%EB%B3%91%EC%9B%90%EC%B2%B4" title="병원체">병원체</a>를 먹는다. (비특이적 면역)</li>
<li><a href="/wiki/%EB%B9%84%EB%A7%8C_%EC%84%B8%ED%8F%AC" title="비만 세포">비만 세포</a> : <a href="/wiki/%ED%9E%88%EC%8A%A4%ED%83%80%EB%AF%BC" title="히스타민">히스타민</a>을 분비한다.</li>
<li><a href="/wiki/%EB%8B%A8%ED%95%B5%EA%B5%AC" title="단핵구">단핵구</a> : <a href="/wiki/%EB%8C%80%EC%8B%9D%EC%84%B8%ED%8F%AC" title="대식세포">대식세포</a>로 분화한다. <a class="new" href="/w/index.php?title=%EB%AA%A8%EB%85%B8%EC%B9%B4%EC%9D%B8&amp;action=edit&amp;redlink=1" title="모노카인 (없는 문서)">모노카인</a>을 분비한다.</li>
<li><a href="/wiki/%EB%8C%80%EC%8B%9D%EC%84%B8%ED%8F%AC" title="대식세포">대식세포</a> : 미생물을 소화사며, <a href="/wiki/T%EC%84%B8%ED%8F%AC" title="T세포">T세포</a>에 항원결정부위(epitope)를 제시하여 활성화시킨다. <a href="/wiki/%EB%A6%BC%ED%8F%AC%EC%B9%B4%EC%9D%B8" title="림포카인">림포카인</a>을 분비한다.</li>
<li><a href="/wiki/%EC%88%98%EC%A7%80%EC%83%81_%EC%84%B8%ED%8F%AC" title="수지상 세포">수지상 세포</a> : <a href="/wiki/%ED%95%AD%EC%9B%90" title="항원">항원</a>을 제시한다.</li>
</ul>
<p><b>림프구</b></p>
<ul>
<li><a href="/wiki/B%EC%84%B8%ED%8F%AC" title="B세포">B세포</a> : 형질세포로 항체를 분비하며, 기억세포가 된다.</li>
<li><a href="/wiki/T%EC%84%B8%ED%8F%AC" title="T세포">T세포</a> : 독성 T 세포 는 <a class="mw-redirect" href="/wiki/%EC%84%B8%ED%8F%AC%EC%84%B1_%EB%A9%B4%EC%97%AD" title="세포성 면역">세포성 면역</a>에 관계하며 도우미 T 세포는 B 세포를 분화시킨다.</li>
<li><a class="mw-redirect" href="/wiki/%EC%9E%90%EC%97%B0_%EC%82%B4%EC%84%B8%ED%8F%AC" title="자연 살세포">자연 살세포</a> : 비특이적 방어, 세포자살에 관여한다.</li>
</ul>
