<h1 class="firstHeading" id="firstHeading" lang="ko">명량해협</h1>
<div class="mw-body-content" id="bodyContent">
<div id="siteSub">위키백과, 우리 모두의 백과사전.</div>

<div class="mw-content-ltr" dir="ltr" id="mw-content-text" lang="ko"><script>function mfTempOpenSection(id){var block=document.getElementById("mf-section-"+id);block.className+=" open-block";block.previousSibling.className+=" open-block";}</script><p><b>명량해협</b>(鳴梁海峽) 또는 <b>울돌목</b>은 <a href="/wiki/%EC%A0%84%EB%9D%BC%EB%82%A8%EB%8F%84" title="전라남도">전라남도</a> <a href="/wiki/%ED%95%B4%EB%82%A8%EA%B5%B0" title="해남군">해남군</a> 문내면 학동리의 <a class="new" href="/w/index.php?title=%ED%99%94%EC%9B%90%EB%B0%98%EB%8F%84&amp;action=edit&amp;redlink=1" title="화원반도 (없는 문서)">화원반도</a>와 <a href="/wiki/%EC%A7%84%EB%8F%84%EA%B5%B0" title="진도군">진도군</a> 군내면 녹진리 사이의 있는 <a href="/wiki/%ED%95%B4%ED%98%91" title="해협">해협</a>이다. 길이 약 1.5km이며, 폭이 가장 짧은 곳은 약 300m 정도가 된다. <a class="mw-redirect" href="/wiki/%EB%B0%80%EB%AC%BC" title="밀물">밀물</a> 때에는 넓은 <a class="mw-redirect" href="/wiki/%EB%82%A8%ED%95%B4_(%ED%95%9C%EB%B0%98%EB%8F%84)" title="남해 (한반도)">남해</a>의 바닷물이 한꺼번에 명량해협을 통과하여 <a class="mw-redirect" href="/wiki/%EC%84%9C%ED%95%B4" title="서해">서해</a>로 빠져 나가 조류가 5<a class="mw-redirect" href="/wiki/M/s" title="M/s">m/s</a> 이상으로 매우 빠르다. 이를 이용하여 <a href="/wiki/%EC%A0%95%EC%9C%A0%EC%9E%AC%EB%9E%80" title="정유재란">정유재란</a> 당시 <a href="/wiki/%EB%AA%85%EB%9F%89_%ED%95%B4%EC%A0%84" title="명량 해전">명량 해전</a>에서 <a href="/wiki/%EC%9D%B4%EC%88%9C%EC%8B%A0" title="이순신">이순신</a>이 이끄는 <a href="/wiki/%EC%A1%B0%EC%84%A0" title="조선">조선</a>군이 승리하였다. 물길이 <a href="/wiki/%EC%95%94%EC%B4%88" title="암초">암초</a>에 부딪혀 튕겨 나오는 소리가 매우 커 바다가 우는 것 같다고 하여 울돌목이라고도 불린다. 유속은 약 10노트(10knot, 시속 20km 정도)라고 한다.</p>

<h2><span class="mw-headline" id=".EA.B0.9C.EC.9A.94">개요</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%AA%85%EB%9F%89%ED%95%B4%ED%98%91&amp;action=edit&amp;section=1" title="부분 편집: 개요">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>1984년 완공된 명량을 가로지르는 길이 484m, 너비 11.7m의 <a href="/wiki/%EC%A7%84%EB%8F%84%EB%8C%80%EA%B5%90" title="진도대교">진도대교</a>가 건설되어 <a class="mw-redirect" href="/wiki/%EC%A7%84%EB%8F%84_(%EC%A7%84%EB%8F%84%EA%B5%B0)" title="진도 (진도군)">진도</a>와 <a href="/wiki/%ED%95%B4%EB%82%A8%EA%B5%B0" title="해남군">해남군</a>이 연결되어 있다. 2005년 12월 15일 제2진도대교가 개통되었고, <a href="/wiki/2008%EB%85%84" title="2008년">2008년</a> 부터, 빠른 <a class="new" href="/w/index.php?title=%EC%A1%B0%EB%A5%98_(%EB%B0%94%EB%8B%A4)&amp;action=edit&amp;redlink=1" title="조류 (바다) (없는 문서)">조류</a>를 이용한 <a class="new" href="/w/index.php?title=%EC%A1%B0%EB%A5%98%EB%B0%9C%EC%A0%84%EC%86%8C&amp;action=edit&amp;redlink=1" title="조류발전소 (없는 문서)">조류발전소</a>의 건설이 진행되고 있다.<sup class="reference" id="cite_ref-1"><a href="#cite_note-1">[1]</a></sup></p>
<p>울돌목은 서해의 길목으로 폭은 294m로, 물살이 세다. 1597년 정유재란 때 이순신 장군은 빠른 물살을 이용해 배 12척으로 10배 이상의 적함 130여 척을 크게 격파하였다.</p>
