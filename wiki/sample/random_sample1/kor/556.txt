<p><b>20세기</b>는 <a href="/wiki/1901%EB%85%84" title="1901년">1901년</a> <a href="/wiki/1%EC%9B%94_1%EC%9D%BC" title="1월 1일">1월 1일</a>부터 <a href="/wiki/2000%EB%85%84" title="2000년">2000년</a> <a href="/wiki/12%EC%9B%94_31%EC%9D%BC" title="12월 31일">12월 31일</a>까지의 기간이다.</p>
<p>20세기는 전쟁의 세기라 불릴만큼 크고 작은 전쟁이 계속되었다.<sup class="reference" id="cite_ref-1"><a href="#cite_note-1">[1]</a></sup><sup class="reference" id="cite_ref-2"><a href="#cite_note-2">[2]</a></sup><sup class="reference" id="cite_ref-3"><a href="#cite_note-3">[3]</a></sup> <a href="/wiki/%EC%A0%9C1%EC%B0%A8_%EC%84%B8%EA%B3%84_%EB%8C%80%EC%A0%84" title="제1차 세계 대전">제1차 세계 대전</a>과 <a href="/wiki/%EC%A0%9C2%EC%B0%A8_%EC%84%B8%EA%B3%84_%EB%8C%80%EC%A0%84" title="제2차 세계 대전">제2차 세계 대전</a>은 세계의 거의 모든 국가가 두 진영으로 갈려 싸운 전쟁이었다. 두 번의 세계 대전 이외에도 <a href="/wiki/%EC%8A%A4%ED%8E%98%EC%9D%B8_%EB%82%B4%EC%A0%84" title="스페인 내전">스페인 내전</a>, <a href="/wiki/%ED%95%9C%EA%B5%AD_%EC%A0%84%EC%9F%81" title="한국 전쟁">한국 전쟁</a>, <a href="/wiki/%EB%B2%A0%ED%8A%B8%EB%82%A8_%EC%A0%84%EC%9F%81" title="베트남 전쟁">베트남 전쟁</a>, <a href="/wiki/%EB%B3%B4%EC%8A%A4%EB%8B%88%EC%95%84_%EC%A0%84%EC%9F%81" title="보스니아 전쟁">보스니아 전쟁</a>과 같이 이념이나 종교, 인종의 차이를 앞세운 전쟁이 끊이지 않았다. 국제 사회는 <a href="/wiki/%EA%B5%AD%EC%A0%9C%EB%B2%95" title="국제법">국제법</a>을 확립하고 상호 안보 공조와 경제 협력을 통해 세계의 평화를 도모하고자 <a href="/wiki/%EC%9C%A0%EC%97%94" title="유엔">유엔</a>을 만들었지만<sup class="reference" id="cite_ref-4"><a href="#cite_note-4">[4]</a></sup>, 20세기의 대부분 동안 세계는 이념에 따라 양분되어 <a href="/wiki/%EB%83%89%EC%A0%84" title="냉전">냉전</a> 체제를 유지하였고 두 진영의 <a href="/wiki/%EB%8C%80%EB%A6%AC%EC%A0%84" title="대리전">대리전</a> 성격을 띈 크고 작은 분쟁이 끊이지 않았다.</p>
<p>19세기 동안 맹위를 떨쳤던 서구 열강에 의한 제국주의적 식민지 확대는 두 차례의 세계 대전을 고비로 쇠락하였다. <a href="/wiki/%EC%95%84%EC%8B%9C%EC%95%84" title="아시아">아시아</a>와 <a href="/wiki/%EC%95%84%ED%94%84%EB%A6%AC%EC%B9%B4" title="아프리카">아프리카</a>의 각지에서 수 많은 식민지들이 독립하여 신생독립국들이 세워졌다. 식민지의 독립은 <a href="/wiki/%EC%9D%B8%EB%8F%84" title="인도">인도</a>와 같이 평화적으로 진행된 경우도 있었으나, <a href="/wiki/%EC%9D%B8%EB%8F%84%EB%84%A4%EC%8B%9C%EC%95%84" title="인도네시아">인도네시아</a>, <a href="/wiki/%EB%B2%A0%ED%8A%B8%EB%82%A8" title="베트남">베트남</a>, <a href="/wiki/%EB%A6%AC%EB%B9%84%EC%95%84" title="리비아">리비아</a> 같은 국가들 처럼 독립전쟁을 치르고서야 이루어진 경우가 많았다. 새롭게 독립한 국가들 사이에서는 어느 진영에도 가입하지 않는 <a href="/wiki/%EB%B9%84%EB%8F%99%EB%A7%B9_%EC%9A%B4%EB%8F%99" title="비동맹 운동">비동맹 운동</a>이 일어났다. 이들은 냉전의 양 진영에 대비되어 흔히 <a href="/wiki/%EC%A0%9C3%EC%84%B8%EA%B3%84" title="제3세계">제3세계</a>로 불렸다.<sup class="reference" id="cite_ref-5"><a href="#cite_note-5">[5]</a></sup></p>
<p><a href="/wiki/%ED%95%9C%EA%B5%AD%EC%9D%98_%EC%97%AD%EC%82%AC" title="한국의 역사">한국의 역사</a>에서 20세기는 <a href="/wiki/%EB%8C%80%ED%95%9C%EC%A0%9C%EA%B5%AD" title="대한제국">대한제국</a> 시기와, <a class="mw-redirect" href="/wiki/%EC%9D%BC%EC%A0%9C%EA%B0%95%EC%A0%90%EA%B8%B0" title="일제강점기">일제강점기</a>, 그리고 해방이후 <a href="/wiki/%EB%8C%80%ED%95%9C%EB%AF%BC%EA%B5%AD" title="대한민국">대한민국</a>과 <a href="/wiki/%EC%A1%B0%EC%84%A0%EB%AF%BC%EC%A3%BC%EC%A3%BC%EC%9D%98%EC%9D%B8%EB%AF%BC%EA%B3%B5%ED%99%94%EA%B5%AD" title="조선민주주의인민공화국">조선민주주의인민공화국</a>의 분단 시기로 구분될 수 있다. <a href="/wiki/1905%EB%85%84" title="1905년">1905년</a> <a href="/wiki/%EC%9D%BC%EB%B3%B8_%EC%A0%9C%EA%B5%AD" title="일본 제국">일본 제국</a>은 이른바 <a href="/wiki/%EC%9D%84%EC%82%AC%EC%A1%B0%EC%95%BD" title="을사조약">을사조약</a>을 강제하여 대한제국의 외교권을 박탈하고 <a href="/wiki/%ED%86%B5%EA%B0%90%EB%B6%80" title="통감부">통감부</a>를 설치하여 내정을 간섭하였다.<sup class="reference" id="cite_ref-6"><a href="#cite_note-6">[6]</a></sup> 1907년 <a class="mw-redirect" href="/wiki/%EC%A0%95%EB%AF%B8%EC%A1%B0%EC%95%BD" title="정미조약">정미조약</a>으로 8,800명 밖에 남지 않았던 대한제국의 군대마저 해산한 일제는<sup class="reference" id="cite_ref-7"><a href="#cite_note-7">[7]</a></sup>, 1910년 대한제국을 강점하여 식민지로 삼았다. 일제강점기 동안 <a href="/wiki/%ED%95%9C%EA%B5%AD%EC%9D%B8" title="한국인">한국인</a>들은 지속적으로 독립운동을 이어갔다. 1919년 <a href="/wiki/3%C2%B71_%EC%9A%B4%EB%8F%99" title="3·1 운동">3·1 운동</a>은 일제의 강점에 맞서는 전민족적인 저항 운동이었다.<sup class="reference" id="cite_ref-8"><a href="#cite_note-8">[8]</a></sup> 이후 <a class="mw-redirect" href="/wiki/%EB%8C%80%ED%95%9C%EB%AF%BC%EA%B5%AD_%EC%9E%84%EC%8B%9C%EC%A0%95%EB%B6%80" title="대한민국 임시정부">대한민국 임시정부</a>를 비롯한 다양한 독립운동이 있었다.<sup class="reference" id="cite_ref-9"><a href="#cite_note-9">[9]</a></sup> 해외에서는 <a href="/wiki/%EB%8F%85%EB%A6%BD%EA%B5%B0_(%EC%A1%B0%EC%84%A0)" title="독립군 (조선)">독립군</a>을 조직하여 무장 독립 투쟁을 하는 한편<sup class="reference" id="cite_ref-10"><a href="#cite_note-10">[10]</a></sup> 각국을 상대로한 외교적 노력이 있었고, 국내에서는 일제의 수탈에 항거하여 각종 파업과 쟁의가 끊이지 않았다.<sup class="reference" id="cite_ref-11"><a href="#cite_note-11">[11]</a></sup> <a href="/wiki/1945%EB%85%84" title="1945년">1945년</a> 제2차 세계대전에서 <a href="/wiki/%EC%B6%94%EC%B6%95%EA%B5%AD" title="추축국">추축국</a>의 일원이었던 일본 제국이 패망하여 해방을 맞이하였으나, 거세지던 <a href="/wiki/%EB%83%89%EC%A0%84" title="냉전">냉전</a>의 영향력 아래 단일 국가를 수립하지 못하고 남북으로 분단되었다. 남북의 두 정권은 서로에게 적대적이었으며 결국 1950년 6월 25일 조선민주주의인민공화국의 남침으로 <a class="mw-redirect" href="/wiki/%ED%95%9C%EA%B5%AD%EC%A0%84%EC%9F%81" title="한국전쟁">한국전쟁</a>이 일어났다.<sup class="reference" id="cite_ref-12"><a href="#cite_note-12">[12]</a></sup> 한국전쟁은 큰 피해를 남기고 휴전을 맞았고, 이후 두 국가는 체제 경쟁을 지속하였다. 조선민주주의인민공화국은 전쟁이후 <a class="mw-redirect" href="/wiki/%EA%B9%80%EC%9D%BC%EC%84%B1%EC%A3%BC%EC%9D%98" title="김일성주의">김일성주의</a>를 표방하고 이후 유일체제라고 불리는 <a href="/wiki/%EC%A3%BC%EC%B2%B4%EC%82%AC%EC%83%81" title="주체사상">주체사상</a>을 바탕으로한 일당 독재 국가를 이루었다. 대한민국은 <a href="/wiki/%EC%9D%B4%EC%8A%B9%EB%A7%8C" title="이승만">이승만</a>의 장기 독재에 저항한 <a href="/wiki/4%C2%B719_%ED%98%81%EB%AA%85" title="4·19 혁명">4·19 혁명</a>이 일어났으나, <a class="mw-redirect" href="/wiki/5%C2%B716_%EA%B5%B0%EC%82%AC%EC%A0%95%EB%B3%80" title="5·16 군사정변">5·16 군사정변</a>으로 권력을 잡은 <a href="/wiki/%EB%B0%95%EC%A0%95%ED%9D%AC" title="박정희">박정희</a>에 의한 <a class="mw-redirect" href="/wiki/%EA%B5%B0%EC%82%AC_%EB%8F%85%EC%9E%AC" title="군사 독재">군사 독재</a>가 이루어졌고, 박정희의 사망 이후에도 <a href="/wiki/5%C2%B718_%EA%B4%91%EC%A3%BC_%EB%AF%BC%EC%A3%BC%ED%99%94_%EC%9A%B4%EB%8F%99" title="5·18 광주 민주화 운동">5·18 광주 민주화 운동</a>을 유혈 진압한 <a href="/wiki/%EC%A0%84%EB%91%90%ED%99%98" title="전두환">전두환</a>에 의해 군사 독재가 지속되었다. 1987년 <a href="/wiki/6%EC%9B%94_%ED%95%AD%EC%9F%81" title="6월 항쟁">6월 항쟁</a>의 결과 <a href="/wiki/%EB%8C%80%ED%86%B5%EB%A0%B9" title="대통령">대통령</a> 직선제를 골자로 하는 <a href="/wiki/%EB%8C%80%ED%95%9C%EB%AF%BC%EA%B5%AD_%ED%97%8C%EB%B2%95" title="대한민국 헌법">대한민국 헌법</a> 개정이 이루어졌다. 1987년 개정 헌법은 오늘날에도 유지되고 있는 현행 헌법이기도 한다. 이후 <a href="/wiki/%EB%85%B8%ED%83%9C%EC%9A%B0" title="노태우">노태우</a>, <a href="/wiki/%EA%B9%80%EC%98%81%EC%82%BC" title="김영삼">김영삼</a>, <a href="/wiki/%EA%B9%80%EB%8C%80%EC%A4%91" title="김대중">김대중</a> 등이 대통령을 역임하였다.</p>
<p>한편, 20세기 동안 <a href="/wiki/%EA%B3%BC%ED%95%99" title="과학">과학</a>, <a href="/wiki/%EA%B8%B0%EC%88%A0" title="기술">기술</a>은 전분야에 걸쳐 급격한 발전을 이루었다. 19세기에 <a href="/wiki/%EA%B7%B8%EB%A0%88%EA%B3%A0%EC%96%B4_%EB%A9%98%EB%8D%B8" title="그레고어 멘델">그레고어 멘델</a>에 의해 발견된 <a href="/wiki/%EB%A9%98%EB%8D%B8%EC%9D%98_%EC%9C%A0%EC%A0%84%EB%B2%95%EC%B9%99" title="멘델의 유전법칙">멘델의 유전법칙</a>은 20세기 초 여러 학자들에 의해 다시 발견되어 현대 <a href="/wiki/%EC%9C%A0%EC%A0%84%ED%95%99" title="유전학">유전학</a>의 기반이 되었다.<sup class="reference" id="cite_ref-13"><a href="#cite_note-13">[13]</a></sup> 이후 유전 물질에 대한 탐구가 계속되어 1928년 <a href="/wiki/%ED%94%84%EB%A0%88%EB%8D%94%EB%A6%AD_%EA%B7%B8%EB%A6%AC%ED%94%BC%EC%8A%A4" title="프레더릭 그리피스">프레더릭 그리피스</a>는 <a href="/wiki/%EA%B7%B8%EB%A6%AC%ED%94%BC%EC%8A%A4_%EC%8B%A4%ED%97%98" title="그리피스 실험">그리피스 실험</a>을 통해 <a href="/wiki/%ED%98%95%EC%A7%88%EC%A0%84%ED%99%98" title="형질전환">형질전환</a>을 발견하였고<sup class="reference" id="cite_ref-14"><a href="#cite_note-14">[14]</a></sup>, 1952년 앨프리드 허시와 마사 체이스가 박테리오파지를 이용하여 DNA가 유전물질임을 증명하였으며<sup class="reference" id="cite_ref-15"><a href="#cite_note-15">[15]</a></sup>, 1953년 <a class="mw-redirect" href="/wiki/%EC%A0%9C%EC%9E%84%EC%8A%A4_%EC%99%93%EC%8A%A8" title="제임스 왓슨">제임스 왓슨</a>과 <a href="/wiki/%ED%94%84%EB%9E%9C%EC%8B%9C%EC%8A%A4_%ED%81%AC%EB%A6%AD" title="프랜시스 크릭">프랜시스 크릭</a>이 <a href="/wiki/DNA" title="DNA">DNA</a>의 구조를 밝혔다.<sup class="reference" id="cite_ref-16"><a href="#cite_note-16">[16]</a></sup></p>
<p><a href="/wiki/%EB%AC%BC%EB%A6%AC%ED%95%99" title="물리학">물리학</a> 분야에서는 <a href="/wiki/%EC%95%8C%EB%B2%A0%EB%A5%B4%ED%8A%B8_%EC%95%84%EC%9D%B8%EC%8A%88%ED%83%80%EC%9D%B8" title="알베르트 아인슈타인">알베르트 아인슈타인</a>이 <a class="mw-redirect" href="/wiki/%EC%83%81%EB%8C%80%EC%84%B1_%EC%9D%B4%EB%A1%A0" title="상대성 이론">상대성 이론</a>을 발표하여 물리학의 <a href="/wiki/%ED%8C%A8%EB%9F%AC%EB%8B%A4%EC%9E%84" title="패러다임">패러다임</a>을 송두리째 바꾸었다. 아인슈타인인 자신의 발견에 대해 “상대성 이론은 돌파구가 있을 것 같지 않은 심각하고 깊은 옛 이론의 모순을 해결하기 위해 생겨났다. 이 새로운 이론은 일관성과 간결함을 유지하면서 옛 이론의 모순을 강력히 해결한다.”고 자평하였다.<sup class="reference" id="cite_ref-17"><a href="#cite_note-17">[17]</a></sup> <a href="/wiki/%EB%A1%9C%EC%BC%93_%EA%B3%B5%ED%95%99" title="로켓 공학">로켓 공학</a>과 <a href="/wiki/%EB%AC%B4%EC%84%A0_%ED%86%B5%EC%8B%A0" title="무선 통신">무선 통신</a>의 발달, 그리고 <a href="/wiki/%EC%BB%B4%ED%93%A8%ED%84%B0" title="컴퓨터">컴퓨터</a>의 출현에 힘입어 인류는 처음으로 <a href="/wiki/%EC%9A%B0%EC%A3%BC_%EA%B3%B5%EA%B0%84" title="우주 공간">우주 공간</a>으로 나갈 수 있었다. 1961년 <a href="/wiki/%EB%B3%B4%EC%8A%A4%ED%86%A0%ED%81%AC_1%ED%98%B8" title="보스토크 1호">보스토크 1호</a>에 탑승한 <a href="/wiki/%EC%9C%A0%EB%A6%AC_%EA%B0%80%EA%B0%80%EB%A6%B0" title="유리 가가린">유리 가가린</a>은 최초의 <a href="/wiki/%EC%9A%B0%EC%A3%BC%EB%B9%84%ED%96%89%EC%82%AC" title="우주비행사">우주비행사</a>가 되었다.<sup class="reference" id="cite_ref-18"><a href="#cite_note-18">[18]</a></sup> 1969년에는 <a href="/wiki/%EC%95%84%ED%8F%B4%EB%A1%9C_11%ED%98%B8" title="아폴로 11호">아폴로 11호</a>의 승무원들이 <a href="/wiki/%EB%8B%AC" title="달">달</a>에 착륙하였다.<sup class="reference" id="cite_ref-19"><a href="#cite_note-19">[19]</a></sup> 처음 달에 발을 딛은 <a href="/wiki/%EB%8B%90_%EC%95%94%EC%8A%A4%ED%8A%B8%EB%A1%B1" title="닐 암스트롱">닐 암스트롱</a>은 “이것은 한 명의 인간에게 있어서는 작은 한 걸음이지만, 인류에게 있어서는 위대한 도약이다.”라는 말을 남겼다. <a href="/wiki/%EC%B2%9C%EC%B2%B4%EB%AC%BC%EB%A6%AC%ED%95%99" title="천체물리학">천체물리학</a>에서는 1964년 미국의 전파 천문학자 <a href="/wiki/%EC%95%84%EB%85%B8_%EC%95%A8%EB%9F%B0_%ED%8E%9C%EC%A7%80%EC%96%B4%EC%8A%A4" title="아노 앨런 펜지어스">아노 앨런 펜지어스</a>와 <a href="/wiki/%EB%A1%9C%EB%B2%84%ED%8A%B8_%EC%9A%B0%EB%93%9C%EB%A1%9C_%EC%9C%8C%EC%8A%A8" title="로버트 우드로 윌슨">로버트 우드로 윌슨</a>이 <a class="mw-redirect" href="/wiki/%EC%9A%B0%EC%A3%BC_%EB%B0%B0%EA%B2%BD_%EB%B3%B5%EC%82%AC" title="우주 배경 복사">우주 배경 복사</a>를 발견하여 <a class="mw-redirect" href="/wiki/%EB%B9%85%EB%B1%85_%EC%9D%B4%EB%A1%A0" title="빅뱅 이론">빅뱅 이론</a>을 실증하였다.<sup class="reference" id="cite_ref-20"><a href="#cite_note-20">[20]</a></sup></p>
<p>20세기 <a href="/wiki/%ED%99%94%ED%95%99" title="화학">화학</a> 분야에서의 가장 큰 성과는 인공적인 <a href="/wiki/%EA%B3%A0%EB%B6%84%EC%9E%90%ED%99%94%ED%95%A9%EB%AC%BC" title="고분자화합물">고분자화합물</a>의 대량생산에 성공한 것을 꼽을 수 있다. 20세기 초 <a class="mw-redirect" href="/wiki/%EB%A0%88%EC%98%A4_%EB%B2%A0%EC%9D%B4%ED%81%B4%EB%9E%9C%EB%93%9C" title="레오 베이클랜드">레오 베이클랜드</a>가 <a href="/wiki/%ED%8E%98%EB%86%80" title="페놀">페놀</a>과 <a class="mw-redirect" href="/wiki/%ED%8F%AC%EB%A6%84%EC%95%8C%EB%8D%B0%ED%9E%88%EB%93%9C" title="포름알데히드">포름알데히드</a>를 반응시켜 <a href="/wiki/%EB%B2%A0%EC%9D%B4%ED%81%B4%EB%9D%BC%EC%9D%B4%ED%8A%B8" title="베이클라이트">베이클라이트</a>를 합성한 이후 다양한 <a href="/wiki/%ED%94%8C%EB%9D%BC%EC%8A%A4%ED%8B%B1" title="플라스틱">플라스틱</a>이 개발되어 생활 전반에 사용되게 되었다.<sup class="reference" id="cite_ref-.EB.B0.95.EC.98.81.EC.88.98_21-0"><a href="#cite_note-.EB.B0.95.EC.98.81.EC.88.98-21">[21]</a></sup> 1933년 영국에서sms <a href="/wiki/%ED%8F%B4%EB%A6%AC%EC%97%90%ED%8B%B8%EB%A0%8C" title="폴리에틸렌">폴리에틸렌</a>을 합성하여 다양한 형태의 물건을 만들었고<sup class="reference" id="cite_ref-.EB.B0.95.EC.98.81.EC.88.98_21-1"><a href="#cite_note-.EB.B0.95.EC.98.81.EC.88.98-21">[21]</a></sup>, 1935년 <a class="new" href="/w/index.php?title=%EC%9C%8C%EB%A6%AC%EC%8A%A4_%EC%BA%90%EB%9F%AC%EB%8D%94%EC%8A%A4&amp;action=edit&amp;redlink=1" title="윌리스 캐러더스 (없는 문서)">윌리스 캐러더스</a>가 <a href="/wiki/%EB%82%98%EC%9D%BC%EB%A1%A0" title="나일론">나일론</a>을 발명하였다.<sup class="reference" id="cite_ref-22"><a href="#cite_note-22">[22]</a></sup></p>
<p>20세기에는 여러 가지 새로운 기술을 바탕으로 한 도구들이 등장하였다. <a href="/wiki/%EB%B9%84%ED%96%89%EA%B8%B0" title="비행기">비행기</a>, <a href="/wiki/%EB%83%89%EC%9E%A5%EA%B3%A0" title="냉장고">냉장고</a>, <a href="/wiki/%EB%9D%BC%EB%94%94%EC%98%A4" title="라디오">라디오</a>, <a href="/wiki/%ED%85%94%EB%A0%88%EB%B9%84%EC%A0%84" title="텔레비전">텔레비전</a>, <a href="/wiki/%EC%9D%B8%EA%B3%B5%EC%9C%84%EC%84%B1" title="인공위성">인공위성</a>, <a class="mw-redirect" href="/wiki/%EC%9D%B4%EB%8F%99%ED%86%B5%EC%8B%A0" title="이동통신">이동통신</a>, <a href="/wiki/%EC%BB%B4%ED%93%A8%ED%84%B0" title="컴퓨터">컴퓨터</a> 등 20세기의 발명품들은 오늘날에도 일상 생활에 큰 영향을 주고 있다. 또한 방송과 통신의 발달로 <a href="/wiki/%EB%8C%80%EC%A4%91_%EB%A7%A4%EC%B2%B4" title="대중 매체">대중 매체</a>가 크게 성장하였고, 이를 통해 <a href="/wiki/%EB%8C%80%EC%A4%91%EC%9D%8C%EC%95%85" title="대중음악">대중음악</a>이 세계적으로 동시에 유행하기도 하였다. 2008년 《<a href="/wiki/%EB%B9%8C%EB%B3%B4%EB%93%9C" title="빌보드">빌보드</a>》지는 빌보드 핫 100 차트 50주년을 기념하여 그 동안의 차트 기록을 분석해 순위를 발표했는데, 가장 성공적인 아티스트로 <a href="/wiki/%EB%B9%84%ED%8B%80%EC%A6%88" title="비틀즈">비틀즈</a>를 꼽았다.<sup class="reference" id="cite_ref-23"><a href="#cite_note-23">[23]</a></sup></p>
