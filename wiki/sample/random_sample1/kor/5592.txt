<h1 class="firstHeading" id="firstHeading" lang="ko">모래폭풍</h1>
<div class="mw-body-content" id="bodyContent">
<div id="siteSub">위키백과, 우리 모두의 백과사전.</div>
<div id="contentSub"></div>
<div class="mw-jump" id="jump-to-nav">
					이동:					<a href="#mw-head">둘러보기</a>, 					<a href="#p-search">검색</a>
</div>
<div class="mw-content-ltr" dir="ltr" id="mw-content-text" lang="ko"><script>function mfTempOpenSection(id){var block=document.getElementById("mf-section-"+id);block.className+=" open-block";block.previousSibling.className+=" open-block";}</script><p><b>모래폭풍</b>(Dust storm)은 <a class="new" href="/w/index.php?title=%EA%B1%B4%EC%A1%B0%EC%A7%80%EC%97%AD&amp;action=edit&amp;redlink=1" title="건조지역 (없는 문서)">건조지역</a>에서 발생하는 <a href="/wiki/%EB%AA%A8%EB%9E%98" title="모래">모래</a>를 동반한 강력한 바람을 말한다. 지역이나 국가에 따라서 부르는 명칭이 각각 다른데, 일반적으로는 <b><a class="new" href="/w/index.php?title=%EC%BA%84%EC%8B%A0&amp;action=edit&amp;redlink=1" title="캄신 (없는 문서)">캄신</a></b>을 가리키는 경우가 많다. 아시아의 <b><a href="/wiki/%ED%99%A9%EC%82%AC" title="황사">황사</a></b>는 미국까지 영향을 미친다. <a href="/wiki/%ED%99%94%EC%84%B1" title="화성">화성</a>에도 모래폭풍이 발생하는 것이 확인되었다.</p>
<h2><span class="mw-headline" id=".EC.9B.90.EC.9D.B8">원인</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%AA%A8%EB%9E%98%ED%8F%AD%ED%92%8D&amp;action=edit&amp;section=1" title="부분 편집: 원인">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<ul>
<li>건조기후 지역은 <a href="/wiki/%EB%82%AE" title="낮">낮</a>과 <a href="/wiki/%EB%B0%A4_(%EC%8B%9C%EA%B0%84)" title="밤 (시간)">밤</a>의 기온차가 크기 때문에 <a href="/wiki/%EB%B0%A4_(%EC%8B%9C%EA%B0%84)" title="밤 (시간)">밤</a>이 되면 <a href="/wiki/%EB%82%AE" title="낮">낮</a> 동안 뜨거워졌던 <a class="mw-redirect" href="/wiki/%EA%B3%B5%EA%B8%B0_(%EA%B8%B0%EC%B2%B4)" title="공기 (기체)">공기</a>가 <a class="new" href="/w/index.php?title=%EB%8C%80%EB%A5%98_%ED%98%84%EC%83%81&amp;action=edit&amp;redlink=1" title="대류 현상 (없는 문서)">대류 현상</a>으로 인해 위로 올라간다. 이때 <a href="/wiki/%EB%A8%BC%EC%A7%80" title="먼지">먼지</a>와 <a href="/wiki/%EB%AA%A8%EB%9E%98" title="모래">모래</a> 역시 함께 하늘로 올라가는데, 이들이 <a href="/wiki/%EB%B0%94%EB%9E%8C" title="바람">바람</a>과 만나면 모래폭풍이 된다.</li>
<li>서로 다른 두 개의 <a class="mw-redirect" href="/wiki/%EA%B3%B5%EA%B8%B0_(%EA%B8%B0%EC%B2%B4)" title="공기 (기체)">공기</a> 덩어리가 한 지역에서 부딪치면 강력한 <a href="/wiki/%EB%B0%94%EB%9E%8C" title="바람">바람</a>이 이는데, 이때 <a href="/wiki/%EB%95%85" title="땅">땅</a>에 모래가 있으면 <a href="/wiki/%EB%B0%94%EB%9E%8C" title="바람">바람</a>과 함께 모래폭풍이 일어나게 된다.</li>
</ul>
