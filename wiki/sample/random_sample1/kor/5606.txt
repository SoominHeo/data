<h1 class="firstHeading" id="firstHeading" lang="ko">모리 에이스케</h1>
<div class="mw-body-content" id="bodyContent">
<div id="siteSub">위키백과, 우리 모두의 백과사전.</div>
<div id="contentSub"></div>
<div class="mw-jump" id="jump-to-nav">
					이동:					<a href="#mw-head">둘러보기</a>, 					<a href="#p-search">검색</a>
</div>
<div class="mw-content-ltr" dir="ltr" id="mw-content-text" lang="ko"><script>function mfTempOpenSection(id){var block=document.getElementById("mf-section-"+id);block.className+=" open-block";block.previousSibling.className+=" open-block";}</script><p><b>모리 에이스케</b>(森 英介, もり えいすけ, 1948년 8월 31일 ~ )는 <a href="/wiki/%EC%9D%BC%EB%B3%B8" title="일본">일본</a>의 정치인으로, <a href="/wiki/%EC%9E%90%EC%9C%A0%EB%AF%BC%EC%A3%BC%EB%8B%B9_(%EC%9D%BC%EB%B3%B8)" title="자유민주당 (일본)">자유민주당</a> 소속의 중의원 의원이다. <a class="new" href="/w/index.php?title=%EC%95%84%EC%86%8C_%EB%8B%A4%EB%A1%9C_%EB%82%B4%EA%B0%81&amp;action=edit&amp;redlink=1" title="아소 다로 내각 (없는 문서)">아소 다로 내각</a>에서 법무대신을 지냈으며, 전 <a class="mw-redirect" href="/wiki/%EC%9D%BC%EB%B3%B8_%EB%82%B4%EA%B0%81%EC%B4%9D%EB%A6%AC%EB%8C%80%EC%8B%A0" title="일본 내각총리대신">일본 내각총리대신</a> <a href="/wiki/%EB%AF%B8%ED%82%A4_%EB%8B%A4%EC%BC%80%EC%98%A4" title="미키 다케오">미키 다케오</a>의 부인 미키 무쓰코 (三木睦子) 는 그의 큰어머니이다.</p>
<h2><span class="mw-headline" id=".EC.95.BD.EB.A0.A5">약력</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%AA%A8%EB%A6%AC_%EC%97%90%EC%9D%B4%EC%8A%A4%EC%BC%80&amp;action=edit&amp;section=1" title="부분 편집: 약력">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p><a href="/wiki/%EC%A7%80%EB%B0%94_%ED%98%84" title="지바 현">지바 현</a> <a href="/wiki/%EA%B0%80%EC%93%B0%EC%9A%B0%EB%9D%BC_%EC%8B%9C" title="가쓰우라 시">가쓰우라 시</a>에서 태어나, 1974년 <a href="/wiki/%EB%8F%84%ED%98%B8%EC%BF%A0_%EB%8C%80%ED%95%99" title="도호쿠 대학">도호쿠 대학</a> 공학부를 졸업하였다. 졸업 후 <a href="/wiki/%EA%B0%80%EC%99%80%EC%82%AC%ED%82%A4_%EC%A4%91%EA%B3%B5%EC%97%85" title="가와사키 중공업">가와사키 중공업</a>에 들어갔으며, 1984년 원자력 플랜트의 용접 연구 성과로 공학 박사 학위를 수여받았다. 1990년 제39회 일본 중의원 의원 총선거에 출마해 첫 당선되었으며, 1994년 노동 정무 차관을 지낸 후, 중의원과 자민당에서 여러 직위를 거쳤다. 2003년 후생노동성 부대신이 되었으며, 2008년 <a class="new" href="/w/index.php?title=%EC%95%84%EC%86%8C_%EB%8B%A4%EB%A1%9C_%EB%82%B4%EA%B0%81&amp;action=edit&amp;redlink=1" title="아소 다로 내각 (없는 문서)">아소 다로 내각</a>에서 법무대신이 되었다. 2009년 제45회 일본 중의원 의원 총선거에서 당선되며 통산 일곱 번째 당선을 기록하였다.</p>
<h2><span class="mw-headline" id=".EA.B0.80.EC.A1.B1">가족</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%AA%A8%EB%A6%AC_%EC%97%90%EC%9D%B4%EC%8A%A4%EC%BC%80&amp;action=edit&amp;section=2" title="부분 편집: 가족">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<ul>
<li>할아버지 모리 노부테루 (森矗昶, 1884년~1941년, 기업인, 모리 콘체른 창립자, 전 중의원 의원)</li>
<li>첫째 큰아버지 모리 기요시 (森清, 1915년~1968년, 정치인, 기업인, <a class="mw-redirect" href="/wiki/%EC%A0%9C1%EC%B0%A8_%EC%82%AC%ED%86%A0_%EB%82%B4%EA%B0%81_%EC%A0%9C2%EC%B0%A8_%EA%B0%9C%EC%A1%B0%EB%82%B4%EA%B0%81" title="제1차 사토 내각 제2차 개조내각">제1차 사토 내각 제2차 개조내각</a>에서 <a href="/wiki/%EC%9D%BC%EB%B3%B8_%EC%B4%9D%EB%A6%AC%EB%B6%80" title="일본 총리부">총리부 총무장관</a> 역임.)</li>
<li>둘째 큰아버지 모리 사토루 (森曉, 1907년~1982년, 경영자, 정치인, 전 중의원 의원, <a class="new" href="/w/index.php?title=%EC%87%BC%EC%99%80_%EB%8D%B4%EC%BD%94&amp;action=edit&amp;redlink=1" title="쇼와 덴코 (없는 문서)">쇼와 덴코</a> 사장, <a href="/wiki/%EC%A7%80%EB%B0%94_%EA%B3%B5%EC%97%85%EB%8C%80%ED%95%99" title="지바 공업대학">지바 공업대학</a> 이사장 등을 역임.)</li>
<li>아버지 모리 요시히데 (森美秀, 1919년~1988년, 정치인, 기업인, 전 중의원 의원, <a class="mw-redirect" href="/wiki/%EC%A0%9C2%EC%B0%A8_%EB%82%98%EC%B9%B4%EC%86%8C%EB%84%A4_%EB%82%B4%EA%B0%81_%EC%A0%9C2%EC%B0%A8_%EA%B0%9C%EC%A1%B0%EB%82%B4%EA%B0%81" title="제2차 나카소네 내각 제2차 개조내각">제2차 나카소네 내각 제2차 개조내각</a>에서 환경대신 역임.)</li>
</ul>
