<p><b>모모</b>(Momo)는 <a href="/wiki/%EB%8F%85%EC%9D%BC" title="독일">독일</a>의 아동문학가 <a href="/wiki/%EB%AF%B8%ED%95%98%EC%97%98_%EC%97%94%EB%8D%B0" title="미하엘 엔데">미하엘 엔데</a>가 <a href="/wiki/1973%EB%85%84" title="1973년">1973년</a> 발표한 책의 이름이자 주인공의 이름이다. 발간 당시의 정식이름은 《모모, 시간도둑과 사람들에게 빼앗긴 시간을 돌려준 한 아이의 이상한 이야기(MOMO oder Die seltsame Geschichte von den Zeit-Dieben und von dem Kind, das den Menschen die gestohlene Zeit zurückbrachte)》이다.</p>

<h2><span class="mw-headline" id=".EC.9E.91.EA.B0.80">작가</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%AA%A8%EB%AA%A8_(%EC%86%8C%EC%84%A4)&amp;action=edit&amp;section=1" title="부분 편집: 작가">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<div class="rellink relarticle mainarticle"><img alt="" data-file-height="128" data-file-width="128" height="16" src="//upload.wikimedia.org/wikipedia/commons/thumb/e/ec/Crystal_Clear_app_xmag.svg/16px-Crystal_Clear_app_xmag.svg.png" srcset="//upload.wikimedia.org/wikipedia/commons/thumb/e/ec/Crystal_Clear_app_xmag.svg/24px-Crystal_Clear_app_xmag.svg.png 1.5x, //upload.wikimedia.org/wikipedia/commons/thumb/e/ec/Crystal_Clear_app_xmag.svg/32px-Crystal_Clear_app_xmag.svg.png 2x" width="16"/> 이 부분의 본문은 <a href="/wiki/%EB%AF%B8%ED%95%98%EC%97%98_%EC%97%94%EB%8D%B0" title="미하엘 엔데">미하엘 엔데</a>입니다.</div>
<p>미하엘 엔데는 독일 <a href="/wiki/%EB%B0%94%EC%9D%B4%EC%97%90%EB%A5%B8_%EC%A3%BC" title="바이에른 주">바이에른 주</a> 출신의 아동문학가로 이 책 이외에 《짐 크노프와 기관사 루카스》, 《<a class="mw-redirect" href="/wiki/%EB%81%9D_%EC%97%86%EB%8A%94_%EC%9D%B4%EC%95%BC%EA%B8%B0" title="끝 없는 이야기">끝 없는 이야기</a>》 등으로도 알려져 있다.</p>

<h2><span class="mw-headline" id=".EC.A3.BC.EC.A0.9C.EC.99.80_.EB.B9.84.ED.8F.89">주제와 비평</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%AA%A8%EB%AA%A8_(%EC%86%8C%EC%84%A4)&amp;action=edit&amp;section=2" title="부분 편집: 주제와 비평">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>미하엘 엔데는 <a class="mw-redirect" href="/wiki/%ED%8C%90%ED%83%80%EC%A7%80_%EC%86%8C%EC%84%A4" title="판타지 소설">판타지 소설</a>을 통해 <a href="/wiki/%EB%AC%B8%EB%AA%85" title="문명">문명</a>에 의한 <a href="/wiki/%EC%9E%90%EC%97%B0" title="자연">자연</a>의 파괴(끝 없는 이야기)와 소비중심의 문명(모모)을 비판하였다. 현대 철학자인 <a class="new" href="/w/index.php?title=%EB%8D%B0%EC%9D%B4%EB%B9%84%EB%93%9C_%EB%A1%9C%EC%9D%B4&amp;action=edit&amp;redlink=1" title="데이비드 로이 (없는 문서)">데이비드 로이</a>와 <a class="new" href="/w/index.php?title=%EB%A6%B0%EB%8B%A4_%EA%B5%BF%ED%9C%B4&amp;action=edit&amp;redlink=1" title="린다 굿휴 (없는 문서)">린다 굿휴</a>는 <a href="/wiki/2002%EB%85%84" title="2002년">2002년</a> 《모모, 도건, 시간의 일반화》라는 책에서, "이 책은 1973년에 쓰였지만 시간이 지나면서 (시간도둑의) 악몽이 현실이 되고 있다"며 《모모》를 <a href="/wiki/20%EC%84%B8%EA%B8%B0" title="20세기">20세기</a>의 가장 중요한 소설 중의 하나로 선정하였다.<sup class="reference" id="cite_ref-1"><a href="#cite_note-1">[1]</a></sup></p>
<h2><span class="mw-headline" id=".EC.98.81.ED.99.94">영화</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%AA%A8%EB%AA%A8_(%EC%86%8C%EC%84%A4)&amp;action=edit&amp;section=3" title="부분 편집: 영화">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p><a href="/wiki/1986%EB%85%84" title="1986년">1986년</a> <a href="/wiki/%EB%8F%85%EC%9D%BC" title="독일">독일</a>과 <a href="/wiki/%EC%9D%B4%ED%83%88%EB%A6%AC%EC%95%84" title="이탈리아">이탈리아</a>의 <a class="mw-redirect" href="/wiki/%EC%98%81%ED%99%94%EC%82%AC" title="영화사">영화사</a>에 의해 <a href="/wiki/%EC%98%81%ED%99%94" title="영화">영화</a>로 만들어졌다.</p>
<h2><span class="mw-headline" id=".EB.8C.80.ED.95.9C.EB.AF.BC.EA.B5.AD.EA.B3.BC_.E3.80.8A.EB.AA.A8.EB.AA.A8.E3.80.8B"><a href="/wiki/%EB%8C%80%ED%95%9C%EB%AF%BC%EA%B5%AD" title="대한민국">대한민국</a>과 《모모》</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%AA%A8%EB%AA%A8_(%EC%86%8C%EC%84%A4)&amp;action=edit&amp;section=4" title="부분 편집: 대한민국과 《모모》">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p><a href="/wiki/1977%EB%85%84" title="1977년">1977년</a> 9월, 차경아가 번역한 최초의 한국어 번역본이 청람문화사에서 출간되었다. 이 번역본에는 작가인 미하엘 엔데의 〈한국 어린이에게 부치는 말〉이 포함되었다.<sup class="reference" id="cite_ref-2"><a href="#cite_note-2">[2]</a></sup></p>
<p><a href="/wiki/1978%EB%85%84" title="1978년">1978년</a>, 가수 <a href="/wiki/%EA%B9%80%EB%A7%8C%EC%A4%80" title="김만준">김만준</a>이 부른 〈모모〉가 인기를 끌었는데, 여기에서의 ‘모모’는 에밀 아자르의 《자기앞의 생》에서 따온 것이었다.<sup class="reference" id="cite_ref-3"><a href="#cite_note-3">[3]</a></sup> ‘모모’가 등장하는 에밀 아자르와 미하엘 엔데의 두 작품은 ‘모모 선풍’을 일으켰다. 이들을 소재로 한 연극이 공연되었고, 김만준의 ‘모모’로부터 《모모는 철부지》라는 제목의 영화가 제작되었으며, 미하엘 엔데의 ‘모모’는 영화화가 계획되기도 하였다.<sup class="reference" id="cite_ref-4"><a href="#cite_note-4">[4]</a></sup></p>
