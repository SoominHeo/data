<h1 class="firstHeading" id="firstHeading" lang="ko">모센 아라키</h1>
<div class="mw-body-content" id="bodyContent">
<div id="siteSub">위키백과, 우리 모두의 백과사전.</div>
<div id="contentSub"></div>
<div class="mw-jump" id="jump-to-nav">
					이동:					<a href="#mw-head">둘러보기</a>, 					<a href="#p-search">검색</a>
</div>
<div class="mw-content-ltr" dir="ltr" id="mw-content-text" lang="ko"><script>function mfTempOpenSection(id){var block=document.getElementById("mf-section-"+id);block.className+=" open-block";block.previousSibling.className+=" open-block";}</script><p><a href="/wiki/%EC%95%84%EC%95%BC%ED%86%A8%EB%9D%BC" title="아야톨라">아야톨라</a> 셰이크 <b>모센 아라키</b>(<span style="font-size: smaller;"><a href="/wiki/%EC%98%81%EC%96%B4" title="영어">영어</a>:</span> <span lang="en" xml:lang="en">Ayatollah Sheikh Mohsen Araki</span>, <span style="font-size: smaller;"><a href="/wiki/%ED%8E%98%EB%A5%B4%EC%8B%9C%EC%95%84%EC%96%B4" title="페르시아어">페르시아어</a>:</span> <span lang="fa" xml:lang="fa">محسن اراکی</span>, <span style="font-size: smaller;"><a href="/wiki/%EC%95%84%EB%9E%8D%EC%96%B4" title="아랍어">아랍어</a>:</span> <span lang="ar" xml:lang="ar">محسن الأراكي</span>)는 <a href="/wiki/%EC%9D%B4%EB%9E%80" title="이란">이란</a>의 학자, 성직자, 대학 강사, 정치인이다. 그는 <a href="/wiki/%EC%9D%B4%EB%9E%80" title="이란">이란</a>에 있는 <a class="new" href="/w/index.php?title=%EC%A0%84%EB%AC%B8%EA%B0%80_%ED%9A%8C%EC%9D%98&amp;action=edit&amp;redlink=1" title="전문가 회의 (없는 문서)">전문가 회의</a>의 예전 멤버였으며 중요한 이란의 성직자이자 <a href="/wiki/%EC%95%84%EC%95%BC%ED%86%A8%EB%9D%BC" title="아야톨라">대 아야톨라</a> <a href="/wiki/%EB%AA%A8%ED%95%98%EB%A7%88%EB%93%9C_%EB%B0%94%ED%82%A4%EB%A5%B4_%EC%95%8C%EC%82%AC%EB%93%9C%EB%A5%B4" title="모하마드 바키르 알사드르">모하마드 바키르 알사드르</a>의 학생들 중 한 명이다.</p>
<h2><span class="mw-headline" id=".EC.82.B6">삶</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%AA%A8%EC%84%BC_%EC%95%84%EB%9D%BC%ED%82%A4&amp;action=edit&amp;section=1" title="부분 편집: 삶">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>그는 <a href="/wiki/%EC%9D%B4%EB%9D%BC%ED%81%AC" title="이라크">이라크</a>, <a href="/wiki/%EB%82%98%EC%9E%90%ED%94%84" title="나자프">나자프</a>에서 태어났다. <a href="/wiki/%EB%82%98%EC%9E%90%ED%94%84" title="나자프">나자프</a>와 <a class="mw-redirect" href="/wiki/%EC%BD%A4" title="콤">콤</a>에 있는 <a href="/wiki/%EB%A7%88%EB%93%9C%EB%9D%BC%EC%82%AC" title="마드라사">이슬람교 학교</a>로부터 이익을 얻었다. 그는 <a href="/wiki/%EC%95%84%EB%9E%8D%EC%96%B4" title="아랍어">아랍어</a>와 <a href="/wiki/%EC%98%81%EC%96%B4" title="영어">영어</a>를 유창하게 말하며 페르시아어, 아랍어, 영어의 세 언어로 된 12권의 책을 오늘 날까지 저술한 상태이다. 그는 런던에 있는 <a href="/wiki/%EC%95%84%EC%95%BC%ED%86%A8%EB%9D%BC" title="아야톨라">대 아야톨라</a> <a href="/wiki/%EC%95%8C%EB%A6%AC_%ED%95%98%EB%A9%94%EB%84%A4%EC%9D%B4" title="알리 하메네이">알리 하메네이</a>의 개인 대변인인이었으며 또한 2004년까지 <a class="new" href="/w/index.php?title=%EC%98%81%EA%B5%AD%EC%9D%98_%EC%9D%B4%EC%8A%AC%EB%9E%8C%EA%B5%90_%EC%84%BC%ED%84%B0&amp;action=edit&amp;redlink=1" title="영국의 이슬람교 센터 (없는 문서)">영국의 이슬람교 센터</a>의 장이었다.</p>
