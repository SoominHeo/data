<p><b>모스크</b>(<span style="font-size: smaller;"><a href="/wiki/%EC%95%84%EB%9E%8D%EC%96%B4" title="아랍어">아랍어</a>:</span> <span lang="ar" xml:lang="ar">مسجد</span>, 마스지드)는 <a href="/wiki/%EC%9D%B4%EC%8A%AC%EB%9E%8C%EA%B5%90" title="이슬람교">이슬람교</a>의 예배 및 집회 장소로 <a href="/wiki/%ED%95%9C%EA%B5%AD%EC%96%B4" title="한국어">한국어</a>로는 <b>성원</b>(聖院), <a href="/wiki/%EC%A4%91%EA%B5%AD%EC%96%B4" title="중국어">중국어</a>로는 <b>청진사</b>(淸眞寺)라고 번역한다. 모스크는 특유의 둥근 지붕과 건물을 둘러싼 <a class="mw-redirect" href="/wiki/%EB%AF%B8%EB%84%88%EB%A0%9B" title="미너렛">미너렛</a>이라 불리는 첨탑이 특징적이다. 벽면은 <a href="/wiki/%EB%B6%88%EA%B5%90" title="불교">불교</a>의 <a href="/wiki/%ED%83%B1%ED%99%94" title="탱화">탱화</a>나 <a href="/wiki/%EA%B8%B0%EB%8F%85%EA%B5%90" title="기독교">기독교</a>의 <a class="mw-redirect" href="/wiki/%EC%84%B1%ED%99%94" title="성화">성화</a>, <a class="mw-redirect" href="/wiki/%EC%8A%A4%ED%85%8C%EC%9D%B8%EB%93%9C%EA%B8%80%EB%9D%BC%EC%8A%A4" title="스테인드글라스">스테인드글라스</a>와 달리 <a class="mw-redirect" href="/wiki/%EC%BF%A0%EB%9E%80" title="쿠란">쿠란</a>의 구절이나 <a href="/wiki/%EC%95%84%EB%9D%BC%EB%B2%A0%EC%8A%A4%ED%81%AC" title="아라베스크">아라베스크</a> 무늬로 장식한다. 이는 <a href="/wiki/%EC%9D%B4%EC%8A%AC%EB%9E%8C%EA%B5%90" title="이슬람교">이슬람교</a>가 <a href="/wiki/%EB%AC%B4%ED%95%A8%EB%A7%88%EB%93%9C" title="무함마드">무함마드</a>의 초상화 등 종교에 관련된 그림을 금지했기 때문이다.</p>

<h2><span class="mw-headline" id=".EC.97.AD.EC.82.AC">역사</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%AA%A8%EC%8A%A4%ED%81%AC&amp;action=edit&amp;section=1" title="부분 편집: 역사">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>오늘날의 모스크는 웅장한 입구와 높은 첨탑을 자랑한다. 그러나 <a href="/wiki/%EC%95%84%EB%9D%BC%EB%B9%84%EC%95%84_%EB%B0%98%EB%8F%84" title="아라비아 반도">아라비아 반도</a>에 최초로 지어진 세 곳의 모스크는 보다 단순한 형태를 취하고 있었다. 모스크의 모습과 건축 기술은 이슬람교와 함께 지난 1천 년 동안 진화하여 왔다.</p>
<h3><span class="mw-headline" id=".ED.99.95.EC.82.B0.EA.B3.BC_.EC.A7.84.ED.99.94">확산과 진화</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%AA%A8%EC%8A%A4%ED%81%AC&amp;action=edit&amp;section=2" title="부분 편집: 확산과 진화">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p><a href="/wiki/%EB%AC%B4%EC%8A%AC%EB%A6%BC" title="무슬림">무슬림</a>이 아라비아 반도를 넘어 세계로 퍼져나가게 됨에 따라 모스크 역시 세계 곳곳에 지어졌다. 640년 <a href="/wiki/%EC%95%84%EB%9E%8D%EC%9D%B8" title="아랍인">아랍인</a>에 의해 <a href="/wiki/%EC%9D%B4%EC%A7%91%ED%8A%B8" title="이집트">이집트</a>에 이슬람교가 전파되면서 수도 <a href="/wiki/%EC%B9%B4%EC%9D%B4%EB%A1%9C" title="카이로">카이로</a>에는 수많은 모스크가 세워졌다. 이 때문에 카이로는 "천 개의 첨탑이 있는 도시"라는 별칭을 갖게 되었다.<sup class="reference" id="cite_ref-1"><a href="#cite_note-1">[1]</a></sup> 이집트의 모스크는 이슬람 학교인 <a href="/wiki/%EB%A7%88%EB%93%9C%EB%9D%BC%EC%82%AC" title="마드라사">마드라사</a>의 역할을 겸했는데 이는 다른 곳의 모스크가 병원이나 무덤의 역할을 하였던 것과 대비된다.<sup class="reference" id="cite_ref-2"><a href="#cite_note-2">[2]</a></sup></p>
<p><a href="/wiki/%EC%8B%9C%EC%B9%A0%EB%A6%AC%EC%95%84" title="시칠리아">시칠리아</a>와 <a class="mw-redirect" href="/wiki/%EC%97%90%EC%8A%A4%ED%8C%8C%EB%83%90" title="에스파냐">에스파냐</a>의 모스크는 <a class="mw-redirect" href="/wiki/%EC%84%9C%EA%B3%A0%ED%8A%B8%EC%A1%B1" title="서고트족">서고트족</a>의 건축술뿐만 아니라 <a href="/wiki/%EB%AC%B4%EC%96%B4%EC%9D%B8" title="무어인">무어인</a>의 기술이 반영되어 있다.<sup class="reference" id="cite_ref-3"><a href="#cite_note-3">[3]</a></sup></p>
<p><a href="/wiki/%EC%A4%91%EA%B5%AD" title="중국">중국</a>에 세워진 최초의 모스크는 8세기경 <a href="/wiki/%EC%8B%9C%EC%95%88_%EC%8B%9C" title="시안 시">시안 시</a>에 세워진 <a class="new" href="/w/index.php?title=%EC%8B%9C%EC%95%88_%EB%8C%80%EC%B2%AD%EC%A7%84%EC%82%AC&amp;action=edit&amp;redlink=1" title="시안 대청진사 (없는 문서)">시안 대청진사</a>이다. 중국의 모스크는 중국식 건축 기술을 받아들여 <a href="/wiki/%ED%83%91" title="탑">탑</a>과 같은 모습을 지녔으며 다른 곳의 모스크와 달리 첨탑이 없고 지붕에 녹색 기와를 얹혔다. 이는 돔으로 된 전형적인 모스크와는 사뭇 다른 모습이다.<sup class="reference" id="cite_ref-4"><a href="#cite_note-4">[4]</a></sup></p>
<p>15세기 이슬람은 <a href="/wiki/%EC%9D%B8%EB%8F%84%EB%84%A4%EC%8B%9C%EC%95%84" title="인도네시아">인도네시아</a>와 <a href="/wiki/%EC%9E%90%EC%99%80_%EC%84%AC" title="자와 섬">자와 섬</a>에 전파되었다. 이슬람교는 이 지역의 기존 종교인 <a href="/wiki/%ED%9E%8C%EB%91%90%EA%B5%90" title="힌두교">힌두교</a>, <a href="/wiki/%EB%B6%88%EA%B5%90" title="불교">불교</a>와 문화적 영향을 주고 받았으며 인도네시아 양식 또는 자와 양식의 독특한 모스크가 만들어지게 되었다. 이 시기 자와 섬의 모스크는 힌두교, 불교, 중국의 건축 양식으로부터 영향을 받았다. 인도네시아의 모스크는 19세기까지도 돔이 없었으며 힌두 사원과 같은 다층 구조의 지붕으로 설계되었다. <a href="/wiki/%EB%B0%9C%EB%A6%AC_%EC%84%AC" title="발리 섬">발리 섬</a>의 모스크는 오늘날에도 여전히 이러한 모양을 간직하고 있다. 1474년 지어진 인도네시아 데막의 <a class="new" href="/w/index.php?title=%EB%A7%88%EC%8A%A4%EC%A7%80%EB%93%9C_%EC%95%84%EA%B6%81_%EB%8D%B0%EB%A7%89&amp;action=edit&amp;redlink=1" title="마스지드 아궁 데막 (없는 문서)">마스지드 아궁 데막</a>과 같은 모스크가 대표적이다.</p>
<p><a href="/wiki/%EC%9D%B8%EB%8F%84" title="인도">인도</a>에서는 16세기부터 17세기에 걸쳐 <a href="/wiki/%EB%AC%B4%EA%B5%B4_%EC%A0%9C%EA%B5%AD" title="무굴 제국">무굴 제국</a>의 모스크가 세워졌다. <a href="/wiki/%EB%8D%B8%EB%A6%AC" title="델리">델리</a>의 <a href="/wiki/%EC%9E%90%EB%A7%88_%EB%A7%88%EC%8A%A4%EC%A7%80%EB%93%9C" title="자마 마스지드">자마 마스지드</a>가 유명하다.</p>
