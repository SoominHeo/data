모스크바 평화 조약에 따라 핀란드가 소련에 할양한 영토</div>
</div>
</div>
<p><b>모스크바 평화 조약</b>은 <a href="/wiki/1940%EB%85%84" title="1940년">1940년</a> <a href="/wiki/3%EC%9B%94_12%EC%9D%BC" title="3월 12일">3월 12일</a>에 <a href="/wiki/%ED%95%80%EB%9E%80%EB%93%9C" title="핀란드">핀란드</a>와 <a href="/wiki/%EC%86%8C%EB%A0%A8" title="소련">소련</a> 사이에 체결되어 <a href="/wiki/3%EC%9B%94_21%EC%9D%BC" title="3월 21일">3월 21일</a>에 비준한 평화 조약이다. 105일 동안 벌어진 <a href="/wiki/%EA%B2%A8%EC%9A%B8_%EC%A0%84%EC%9F%81" title="겨울 전쟁">겨울 전쟁</a>을 종식시킨 이 조약에 따라 핀란드는 상당수의 영토를 소련에 할양했지만 핀란드의 독립을 보증하면서 핀란드를 합병하려던 소련의 야망을 좌절시켰다. 소련 측에서는 <a href="/wiki/%EB%B1%8C%EC%B2%B4%EC%8A%AC%EB%9D%BC%ED%94%84_%EB%AA%B0%EB%A1%9C%ED%86%A0%ED%94%84" title="뱌체슬라프 몰로토프">뱌체슬라프 몰로토프</a>, <a class="new" href="/w/index.php?title=%EC%95%88%EB%93%9C%EB%A0%88%EC%9D%B4_%EC%A6%88%EB%8B%A4%EB%85%B8%ED%94%84&amp;action=edit&amp;redlink=1" title="안드레이 즈다노프 (없는 문서)">안드레이 즈다노프</a>, <a href="/wiki/%EC%95%8C%EB%A0%89%EC%82%B0%EB%93%9C%EB%A5%B4_%EB%B0%94%EC%8B%A4%EB%A0%99%EC%8A%A4%ED%82%A4" title="알렉산드르 바실렙스키">알렉산드르 바실렙스키</a>, 핀란드 측에서는 <a href="/wiki/%EB%A6%AC%EC%8A%A4%ED%86%A0_%EB%A4%BC%ED%8B%B0" title="리스토 뤼티">리스토 뤼티</a>, <a href="/wiki/%EC%9C%A0%ED%98%B8_%EC%BF%A0%EC%8A%A4%ED%8B%B0_%ED%8C%8C%EC%8B%9C%ED%82%A4%EB%B9%84" title="유호 쿠스티 파시키비">유호 쿠스티 파시키비</a>, <a class="new" href="/w/index.php?title=%EB%A3%A8%EB%8F%8C%ED%94%84_%EB%B0%9C%EB%8D%B4&amp;action=edit&amp;redlink=1" title="루돌프 발덴 (없는 문서)">루돌프 발덴</a>, <a class="new" href="/w/index.php?title=%EB%B0%B0%EC%9D%B4%EB%87%8C_%EB%B3%B4%EC%9D%B4%EC%98%A8%EB%A7%88&amp;action=edit&amp;redlink=1" title="배이뇌 보이온마 (없는 문서)">배이뇌 보이온마</a>가 서명했다.</p>
<p>이 조약에 따라 핀란드는 <a href="/wiki/%EB%B9%84%EB%B3%B4%EB%A5%B4%ED%81%AC" title="비보르크">비보르크</a>를 포함한 <a href="/wiki/%EC%B9%B4%EB%A0%90%EB%A6%AC%EC%95%BC_%EC%A7%80%ED%98%91" title="카렐리야 지협">카렐리야 지협</a>, <a href="/wiki/%EB%9D%BC%EB%8F%84%EA%B0%80_%ED%98%B8" title="라도가 호">라도가 호</a>의 핀란드 호반 지역을 소련에 할양했다. 또한 <a href="/wiki/%ED%95%AD%EC%BD%94" title="항코">항코</a> 반도를 30년 동안 소련에 임대했다.</p>
