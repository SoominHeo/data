<p><sup>1</sup>성인 클럽 출장수와 골은 정규 리그 기록만 세며,<br/>
<a href="/wiki/2010%EB%85%84" title="2010년">2010년</a> <a href="/wiki/9%EC%9B%94_30%EC%9D%BC" title="9월 30일">9월 30일</a> 기준이다.<br/>
<sup>2</sup>국가대표팀의 출장수와 골은<br/>
<a href="/wiki/2012%EB%85%84" title="2012년">2012년</a> <a href="/wiki/6%EC%9B%94_10%EC%9D%BC" title="6월 10일">6월 10일</a> 기준이다.</p>

<p><b>모하메드 칼론</b>(Mohamed Kallon, <a href="/wiki/1979%EB%85%84" title="1979년">1979년</a> <a href="/wiki/10%EC%9B%94_6%EC%9D%BC" title="10월 6일">10월 6일</a> ~ )은 <a href="/wiki/%EC%8B%9C%EC%97%90%EB%9D%BC%EB%A6%AC%EC%98%A8" title="시에라리온">시에라리온</a>의 축구 선수이다.</p>
<p>칼론은 <a href="/wiki/FC_%EC%9D%B8%ED%85%8C%EB%A5%B4%EB%82%98%EC%B9%98%EC%98%A4%EB%82%A0%EB%A0%88_%EB%B0%80%EB%9D%BC%EB%85%B8" title="FC 인테르나치오날레 밀라노">FC 인테르나치오날레 밀라노</a>에서 3시즌간 활약하였고 <a class="mw-redirect" href="/wiki/2005%EB%85%84_AFC_%EC%B1%94%ED%94%BC%EC%96%B8%EC%8A%A4%EB%A6%AC%EA%B7%B8" title="2005년 AFC 챔피언스리그">2005년 AFC 챔피언스리그</a>에서는 득점왕을 기록하기도 했다.</p>
<p>선수 생활 말년에는 자신의 이름을 딴 축구 클럽을 만든 뒤에 그 팀의 선수로 뛰고 있다.</p>
<p>그의 형인 <a class="new" href="/w/index.php?title=%EB%AC%B4%EC%82%AC_%EC%B9%BC%EB%A1%A0&amp;action=edit&amp;redlink=1" title="무사 칼론 (없는 문서)">무사 칼론</a>과 <a class="new" href="/w/index.php?title=%EC%BC%80%EB%AA%A8%EC%B9%B4%EC%9D%B4_%EC%B9%BC%EB%A1%A0&amp;action=edit&amp;redlink=1" title="케모카이 칼론 (없는 문서)">케모카이 칼론</a> 역시 축구 선수였으며 <a href="/wiki/%EC%8B%9C%EC%97%90%EB%9D%BC%EB%A6%AC%EC%98%A8_%EC%B6%95%EA%B5%AC_%EA%B5%AD%EA%B0%80%EB%8C%80%ED%91%9C%ED%8C%80" title="시에라리온 축구 국가대표팀">시에라리온 축구 국가대표팀</a>으로도 활약하였다.</p>
