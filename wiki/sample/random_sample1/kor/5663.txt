<p><b>목동운동장</b>(木洞運動場)은 <a href="/wiki/%EB%8C%80%ED%95%9C%EB%AF%BC%EA%B5%AD" title="대한민국">대한민국</a> <a href="/wiki/%EC%84%9C%EC%9A%B8%ED%8A%B9%EB%B3%84%EC%8B%9C" title="서울특별시">서울특별시</a>가 생활 체육의 저변 확대와 아마 야구 활성화를 목적으로 <a href="/wiki/1987%EB%85%84" title="1987년">1987년</a> <a href="/wiki/%EB%8C%80%ED%95%9C%EB%AF%BC%EA%B5%AD" title="대한민국">대한민국</a> <a href="/wiki/%EC%84%9C%EC%9A%B8%ED%8A%B9%EB%B3%84%EC%8B%9C" title="서울특별시">서울특별시</a> <a href="/wiki/%EC%96%91%EC%B2%9C%EA%B5%AC" title="양천구">양천구</a>에 건립한 <a href="/wiki/%EC%A2%85%ED%95%A9%EC%9A%B4%EB%8F%99%EC%9E%A5" title="종합운동장">스포츠 경기장 종합 단지</a>이다. <a href="/wiki/1987%EB%85%84" title="1987년">1987년</a> <a href="/wiki/12%EC%9B%94_9%EC%9D%BC" title="12월 9일">12월 9일</a>에 착공하여 <a href="/wiki/1989%EB%85%84" title="1989년">1989년</a> <a href="/wiki/11%EC%9B%94_14%EC%9D%BC" title="11월 14일">11월 14일</a>에 완공됐다.<sup class="reference" id="cite_ref-1"><a href="#cite_note-1">[1]</a></sup></p>
<p>원래는 목동운동장이 모든 시설을 포함하는 스포츠 경기장 종합 단지를 뜻하는 용어이지만 목동운동장 주경기장을 바로 지칭하는 용어로 동시에 사용되며 다른 시설들은 <a href="/wiki/%EB%AA%A9%EB%8F%99%EC%95%BC%EA%B5%AC%EC%9E%A5" title="목동야구장">목동야구장</a>, <a href="/wiki/%EB%AA%A9%EB%8F%99%EC%95%84%EC%9D%B4%EC%8A%A4%EB%A7%81%ED%81%AC" title="목동아이스링크">목동아이스링크</a> 이렇게 구분하여 불리기도 한다.</p>
<p>한편 이용자의 편의를 위해 운동장 주차장과는 별도로 <a href="/wiki/%EC%9C%A0%EC%88%98%EC%A7%80" title="유수지">유수지</a>를 복개하여 조성한 주차 공간을 제공하고 있다.</p>

<h3><span class="mw-headline" id=".EC.A3.BC.EA.B2.BD.EA.B8.B0.EC.9E.A5">주경기장</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%AA%A9%EB%8F%99%EC%9A%B4%EB%8F%99%EC%9E%A5&amp;action=edit&amp;section=2" title="부분 편집: 주경기장">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>축구와 육상경기를 할 수 있는 <a href="/wiki/%EB%8B%A4%EB%AA%A9%EC%A0%81_%EA%B2%BD%EA%B8%B0%EC%9E%A5" title="다목적 경기장">다목적 경기장</a>인 주경기장은 20,236석의 수용인원을 가지며 주로 축구 및 육상 대회와 각종 체육행사가 열리고 있다. 과거 <a href="/wiki/K%EB%A6%AC%EA%B7%B8_%ED%81%B4%EB%9E%98%EC%8B%9D" title="K리그 클래식">K리그</a> 소속 <a href="/wiki/%EC%A0%9C%EC%A3%BC_%EC%9C%A0%EB%82%98%EC%9D%B4%ED%8B%B0%EB%93%9C_FC" title="제주 유나이티드 FC">부천 SK</a>, <a href="/wiki/FC_%EC%84%9C%EC%9A%B8" title="FC 서울">안양 LG 치타스</a>, <a href="/wiki/%EB%82%B4%EC%85%94%EB%84%90%EB%A6%AC%EA%B7%B8_(%EB%8C%80%ED%95%9C%EB%AF%BC%EA%B5%AD)" title="내셔널리그 (대한민국)">내셔널리그</a> 소속 <a href="/wiki/%EC%84%9C%EC%9A%B8%EC%8B%9C%EC%B2%AD_%EC%B6%95%EA%B5%AC%EB%8B%A8" title="서울시청 축구단">서울시청 축구단</a>의 홈구장으로 사용되기도 했다. 주경기장 이외에 야구장, 실내빙상장의 체육 시설과 사격장, 체력단련장의 부대 시설로 구성되어 있으며, 인근에 목동 대단위 주택단지 거주민의 편의 시설로 많이 이용되고 있다.</p>
