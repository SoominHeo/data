<p><b>목샤어</b>은 <a href="/wiki/%EB%9F%AC%EC%8B%9C%EC%95%84%EC%96%B4" title="러시아어">러시아어</a>, <a href="/wiki/%EC%97%90%EB%A5%B4%EC%9E%90%EC%96%B4" title="에르자어">에르자어</a>와 함께 <a href="/wiki/%EB%AA%A8%EB%A5%B4%EB%8F%84%EB%B0%94_%EA%B3%B5%ED%99%94%EA%B5%AD" title="모르도바 공화국">모르도바 공화국</a>의 공용어로 지정된 언어이다. 에르자어와 함께 <a href="/wiki/%ED%95%80%EB%B3%BC%EA%B0%80%EC%96%B4%EA%B5%B0" title="핀볼가어군">핀볼가어군</a>에 속해 있다.</p>
<h2><span class="mw-headline" id=".EC.A7.80.EC.97.AD">지역</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%AA%A9%EC%83%A4%EC%96%B4&amp;action=edit&amp;section=1" title="부분 편집: 지역">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>현재 목샤어를 사용하는 사람들은 50만명이며 주로 <a href="/wiki/%EB%AA%A8%EB%A5%B4%EB%8F%84%EB%B0%94_%EA%B3%B5%ED%99%94%EA%B5%AD" title="모르도바 공화국">모르도바 공화국</a>에 분포되어 있다. 일부는 러시아의 다른 지역(<a href="/wiki/%ED%8E%9C%EC%9E%90_%EC%A3%BC" title="펜자 주">펜자 주</a>, <a href="/wiki/%EC%82%AC%EB%A7%88%EB%9D%BC_%EC%A3%BC" title="사마라 주">사마라 주</a>, <a href="/wiki/%EC%82%AC%EB%9D%BC%ED%86%A0%ED%94%84_%EC%A3%BC" title="사라토프 주">사라토프 주</a>, <a href="/wiki/%EC%98%A4%EB%A0%8C%EB%B6%80%EB%A5%B4%ED%81%AC_%EC%A3%BC" title="오렌부르크 주">오렌부르크 주</a>, <a href="/wiki/%EB%9E%B4%EC%9E%94_%EC%A3%BC" title="랴잔 주">랴잔 주</a>, <a href="/wiki/%ED%83%90%EB%B3%B4%ED%94%84_%EC%A3%BC" title="탐보프 주">탐보프 주</a>, <a href="/wiki/%ED%83%80%ED%83%80%EB%A5%B4_%EA%B3%B5%ED%99%94%EA%B5%AD" title="타타르 공화국">타타르 공화국</a>, <a href="/wiki/%EB%B0%94%EC%8B%9C%ED%82%A4%EB%A5%B4_%EA%B3%B5%ED%99%94%EA%B5%AD" title="바시키르 공화국">바시키르 공화국</a>, <a href="/wiki/%EC%8B%9C%EB%B2%A0%EB%A6%AC%EC%95%84" title="시베리아">시베리아</a>와 <a href="/wiki/%EB%9F%AC%EC%8B%9C%EC%95%84_%EA%B7%B9%EB%8F%99" title="러시아 극동">극동</a>)에 거주한다. 러시아 국외에는 <a href="/wiki/%EC%95%84%EB%A5%B4%EB%A9%94%EB%8B%88%EC%95%84" title="아르메니아">아르메니아</a>와 <a href="/wiki/%EB%AF%B8%EA%B5%AD" title="미국">미국</a>에 거주한다.</p>
