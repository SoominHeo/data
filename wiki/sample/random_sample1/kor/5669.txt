<p><b>목포공항</b>(木浦空港, <span lang="en" xml:lang="en">Mokpo Airport</span>, <a class="mw-redirect" href="/wiki/ICAO" title="ICAO">ICAO</a>: <b>RKJM</b>)은 <a href="/wiki/%EB%8C%80%ED%95%9C%EB%AF%BC%EA%B5%AD" title="대한민국">대한민국</a> <a href="/wiki/%EC%A0%84%EB%9D%BC%EB%82%A8%EB%8F%84" title="전라남도">전라남도</a> <a href="/wiki/%EC%98%81%EC%95%94%EA%B5%B0" title="영암군">영암군</a> <a href="/wiki/%EC%82%BC%ED%98%B8%EC%9D%8D" title="삼호읍">삼호읍</a><sup class="reference" id="cite_ref-1"><a href="#cite_note-1">[1]</a></sup> 용당리에 있는 군용(해군)공항이다. 인근에 대불산업단지 및 <a href="/wiki/%EB%8C%80%EB%B6%88%EC%97%AD" title="대불역">대불역</a>이 있으며, <a href="/wiki/%EC%82%BC%ED%98%B8%EB%8C%80%EA%B5%90" title="삼호대교">삼호대교</a>를 통해 <a href="/wiki/%EB%AA%A9%ED%8F%AC%EC%8B%9C" title="목포시">목포시</a>와 연결되었다. 해군 비행장을 겸해 <a href="/wiki/1969%EB%85%84" title="1969년">1969년</a> 서울~제주 간 여객기의 중간 경유지로 개항하였다가 여객 수요 감소로 <a href="/wiki/1972%EB%85%84" title="1972년">1972년</a> 폐쇄되었다. 이후 인근의 대불산업단지 등 서남 해안권 발전을 열망하는 지역 민의에 부응하여 활주로 연장 및 항행 안전시설 보강으로 <a href="/wiki/1992%EB%85%84" title="1992년">1992년</a> <a href="/wiki/7%EC%9B%94" title="7월">7월</a>에 다시 개항하여 한때 서울 6회, 제주 2회, 부산 1회씩을 운항하기도 하였다.</p>
<p>그러나 재개항한 지 불과 1년 후인 <a href="/wiki/1993%EB%85%84" title="1993년">1993년</a> <a href="/wiki/7%EC%9B%94_26%EC%9D%BC" title="7월 26일">7월 26일</a> <a href="/wiki/%EC%95%84%EC%8B%9C%EC%95%84%EB%82%98%ED%95%AD%EA%B3%B5_733%ED%8E%B8_%EC%B6%94%EB%9D%BD_%EC%82%AC%EA%B3%A0" title="아시아나항공 733편 추락 사고">아시아나항공 OZ733편</a>이 착륙을 시도하던 도중 <a href="/wiki/%ED%95%B4%EB%82%A8%EA%B5%B0" title="해남군">해남군</a> <a href="/wiki/%ED%99%94%EC%9B%90%EB%A9%B4" title="화원면">화원면</a> 마산리의 야산에 추락하는 사고가 발생하면서 안전성에 문제가 제시되었으나, 활주로만 100m 확장(1,500m→1,600m)한 채 운항을 계속하였으며, <a class="mw-redirect" href="/wiki/%EA%B3%84%EA%B8%B0%EC%B0%A9%EB%A5%99%EC%9E%A5%EC%B9%98" title="계기착륙장치">계기착륙장치</a>도 없었다. 이 사고로 탄력을 받은 <a href="/wiki/%EB%AC%B4%EC%95%88%EA%B5%AD%EC%A0%9C%EA%B3%B5%ED%95%AD" title="무안국제공항">무안국제공항</a> 공사 이후에는 시설 투자가 거의 없었다. 서해안에 인접해 있는 공항의 특성상 잦은 안개와 돌풍, 항행안전시설 부재로 결항률 전국 1위 공항이라는 불명예를 안고 있었다. <a href="/wiki/%EC%95%84%EC%8B%9C%EC%95%84%EB%82%98%ED%95%AD%EA%B3%B5_733%ED%8E%B8_%EC%B6%94%EB%9D%BD_%EC%82%AC%EA%B3%A0" title="아시아나항공 733편 추락 사고">아시아나항공 OZ733편</a> 추락 사고 이후 호남권 신 공항으로 예정되어 있었던 <a href="/wiki/%EB%AC%B4%EC%95%88%EA%B5%AD%EC%A0%9C%EA%B3%B5%ED%95%AD" title="무안국제공항">무안국제공항</a>의 건설이 탄력을 받게 되고, 빈약한 공항 규모 때문에 더 이상 활성화되지 못한 채 <a href="/wiki/%EC%84%9C%ED%95%B4%EC%95%88%EA%B3%A0%EC%86%8D%EB%8F%84%EB%A1%9C" title="서해안고속도로">서해안고속도로</a> 및 <a href="/wiki/%ED%95%9C%EA%B5%AD%EA%B3%A0%EC%86%8D%EC%B2%A0%EB%8F%84" title="한국고속철도">KTX</a> 개통으로 승객이 계속 감소하였다. <a href="/wiki/2007%EB%85%84" title="2007년">2007년</a> <a href="/wiki/11%EC%9B%94_8%EC%9D%BC" title="11월 8일">11월 8일</a> <a href="/wiki/%EB%AA%A9%ED%8F%AC%EC%8B%9C" title="목포시">목포시</a>의 북쪽에 <a href="/wiki/%EB%AC%B4%EC%95%88%EA%B5%AD%EC%A0%9C%EA%B3%B5%ED%95%AD" title="무안국제공항">무안국제공항</a>이 개항하면서 마지막으로 남아 있던 <a href="/wiki/%EA%B9%80%ED%8F%AC%EA%B5%AD%EC%A0%9C%EA%B3%B5%ED%95%AD" title="김포국제공항">김포</a> - 목포 간 항공편이 <a href="/wiki/2007%EB%85%84" title="2007년">2007년</a> <a href="/wiki/11%EC%9B%94_7%EC%9D%BC" title="11월 7일">11월 7일</a> 마지막 운항을 끝으로 <a href="/wiki/%EA%B9%80%ED%8F%AC%EA%B5%AD%EC%A0%9C%EA%B3%B5%ED%95%AD" title="김포국제공항">서울</a>행 여객기의 운항이 종료되었다. 개항 다음 날 <a href="/wiki/%EA%B9%80%ED%8F%AC%EA%B5%AD%EC%A0%9C%EA%B3%B5%ED%95%AD" title="김포국제공항">김포</a> 및 <a href="/wiki/%EC%A0%9C%EC%A3%BC%EA%B5%AD%EC%A0%9C%EA%B3%B5%ED%95%AD" title="제주국제공항">제주</a>발 여객기의 종착지를 <a href="/wiki/%EB%AC%B4%EC%95%88%EA%B5%AD%EC%A0%9C%EA%B3%B5%ED%95%AD" title="무안국제공항">무안</a>으로 이전함과 동시에 목포공항이 폐쇄되어 민간 공항으로는 더 이상 쓰이지 않게 되었다.<sup class="reference" id="cite_ref-2"><a href="#cite_note-2">[2]</a></sup></p>

<h2><span class="mw-headline" id=".EC.82.AC.EA.B1.B4_.EB.B0.8F_.EC.82.AC.EA.B3.A0">사건 및 사고</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%AA%A9%ED%8F%AC%EA%B3%B5%ED%95%AD&amp;action=edit&amp;section=5" title="부분 편집: 사건 및 사고">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<div class="rellink relarticle mainarticle"><img alt="" data-file-height="128" data-file-width="128" height="16" src="//upload.wikimedia.org/wikipedia/commons/thumb/e/ec/Crystal_Clear_app_xmag.svg/16px-Crystal_Clear_app_xmag.svg.png" srcset="//upload.wikimedia.org/wikipedia/commons/thumb/e/ec/Crystal_Clear_app_xmag.svg/24px-Crystal_Clear_app_xmag.svg.png 1.5x, //upload.wikimedia.org/wikipedia/commons/thumb/e/ec/Crystal_Clear_app_xmag.svg/32px-Crystal_Clear_app_xmag.svg.png 2x" width="16"/> 이 부분의 본문은 <a href="/wiki/%EC%95%84%EC%8B%9C%EC%95%84%EB%82%98%ED%95%AD%EA%B3%B5_733%ED%8E%B8_%EC%B6%94%EB%9D%BD_%EC%82%AC%EA%B3%A0" title="아시아나항공 733편 추락 사고">아시아나항공 733편 추락 사고</a>입니다.</div>
<p><a href="/wiki/1993%EB%85%84" title="1993년">1993년</a> <a href="/wiki/7%EC%9B%94_26%EC%9D%BC" title="7월 26일">7월 26일</a> 14시 20분에 <a href="/wiki/%EA%B9%80%ED%8F%AC%EA%B5%AD%EC%A0%9C%EA%B3%B5%ED%95%AD" title="김포국제공항">김포국제공항</a>을 떠나 목포공항으로 향하던 <a href="/wiki/%EC%95%84%EC%8B%9C%EC%95%84%EB%82%98%ED%95%AD%EA%B3%B5" title="아시아나항공">아시아나항공</a> OZ733편 (<a href="/wiki/%EB%B3%B4%EC%9E%89_737" title="보잉 737">B737-500</a>, 등록번호 HL7229)이 비가 내리는 궂은 날씨 속에 15시 15분 ~ 30분경 목포공항에 3번의 착륙 시도를 했다가 실패한 후 목포공항의 <a href="/wiki/%EB%A0%88%EC%9D%B4%EB%8D%94" title="레이더">레이더</a>에서 사라졌다. 15시 50분 경 해당 항공기는 목포공항에서 10여km 떨어진 <a href="/wiki/%EC%A0%84%EB%9D%BC%EB%82%A8%EB%8F%84" title="전라남도">전라남도</a> <a href="/wiki/%ED%95%B4%EB%82%A8%EA%B5%B0" title="해남군">해남군</a> <a href="/wiki/%ED%99%94%EC%9B%90%EB%A9%B4" title="화원면">화원면</a> 마산리 뒷산에서 추락한 채 발견됐고 승무원을 포함한 68명이 숨지는 사고가 발생했다. 생존자 중 일부가 사고 현장을 나와 마을지소에 신고하면서 사고 사실이 알려졌다.</p>
