<p><b>몬테네그로 공국</b>(<span style="font-size: smaller;"><a href="/wiki/%EC%84%B8%EB%A5%B4%EB%B9%84%EC%95%84%EC%96%B4" title="세르비아어">세르비아어</a>:</span> <span lang="sr" xml:lang="sr">Књажевина Црнa Горa / Knjaževina Crna Gora</span> <small>크냐제비나 츠르나고라</small>)은 <a href="/wiki/1852%EB%85%84" title="1852년">1852년</a> <a href="/wiki/3%EC%9B%94_13%EC%9D%BC" title="3월 13일">3월 13일</a>부터 <a href="/wiki/1910%EB%85%84" title="1910년">1910년</a> <a href="/wiki/8%EC%9B%94_28%EC%9D%BC" title="8월 28일">8월 28일</a>까지 현재의 <a href="/wiki/%EB%AA%AC%ED%85%8C%EB%84%A4%EA%B7%B8%EB%A1%9C" title="몬테네그로">몬테네그로</a>에 존재했던 공국이다.</p>
<p>1852년 3월 13일 몬테네그로의 공을 자처한 <a href="/wiki/%ED%8E%98%ED%8A%B8%EB%A1%9C%EB%B9%84%EC%B9%98%EB%84%A4%EA%B3%A0%EC%8B%9C_%EA%B0%80" title="페트로비치네고시 가">페트로비치네고시 가</a>의 <a class="new" href="/w/index.php?title=%EB%8B%A4%EB%8B%90%EB%A1%9C_1%EC%84%B8&amp;action=edit&amp;redlink=1" title="다닐로 1세 (없는 문서)">다닐로 1세</a>가 자신이 소유하고 있던 영지를 세속화하면서 몬테네그로 공국이 수립되었다. <a href="/wiki/1858%EB%85%84" title="1858년">1858년</a> <a href="/wiki/5%EC%9B%94_1%EC%9D%BC" title="5월 1일">5월 1일</a>에 몬테네그로 공국이 그라호바츠(Grahovac) 전투에서 <a href="/wiki/%EC%98%A4%EC%8A%A4%EB%A7%8C_%EC%A0%9C%EA%B5%AD" title="오스만 제국">오스만 제국</a>에 승리하면서 몬테네그로 공국과 오스만 제국 간의 경계선이 확립되었다.</p>
<p><a href="/wiki/1860%EB%85%84" title="1860년">1860년</a> <a href="/wiki/8%EC%9B%94_13%EC%9D%BC" title="8월 13일">8월 13일</a>에 다닐로 1세가 암살되었고 다닐로 1세의 조카였던 <a class="new" href="/w/index.php?title=%EB%8B%88%EC%BD%9C%EB%9D%BC_1%EC%84%B8&amp;action=edit&amp;redlink=1" title="니콜라 1세 (없는 문서)">니콜라 1세</a>가 몬테네그로의 공으로 즉위했다. <a href="/wiki/1876%EB%85%84" title="1876년">1876년</a>부터 1878년까지 일어난 몬테네그로-오스만 전쟁에서 몬테네그로 공국은 <a href="/wiki/%EB%9F%AC%EC%8B%9C%EC%95%84_%EC%A0%9C%EA%B5%AD" title="러시아 제국">러시아 제국</a>, <a href="/wiki/%EC%84%B8%EB%A5%B4%EB%B9%84%EC%95%84_%EA%B3%B5%EA%B5%AD" title="세르비아 공국">세르비아 공국</a>의 지원을 받으면서 오스만 제국을 격파했다.</p>
<p><a href="/wiki/1878%EB%85%84" title="1878년">1878년</a> <a href="/wiki/3%EC%9B%94_3%EC%9D%BC" title="3월 3일">3월 3일</a> 러시아 제국과 오스만 제국 사이에 체결된 <a href="/wiki/%EC%82%B0%EC%8A%A4%ED%85%8C%ED%8C%8C%EB%85%B8_%EC%A1%B0%EC%95%BD" title="산스테파노 조약">산스테파노 조약</a>에 따라 오스만 제국은 몬테네그로 공국의 독립을 승인했으며 몬테네그로 공국의 영토가 확대되었다. 1878년 7월 13일에 열린 <a href="/wiki/%EB%B2%A0%EB%A5%BC%EB%A6%B0_%ED%9A%8C%EC%9D%98_(1878%EB%85%84)" title="베를린 회의 (1878년)">베를린 회의</a>를 통해 몬테네그로 공국은 국제적인 승인을 받았다.</p>
<p><a href="/wiki/1905%EB%85%84" title="1905년">1905년</a>에 제정된 헌법에 따라 니콜라 1세의 칭호가 공(公)에서 국왕으로 격상되었다. <a href="/wiki/1910%EB%85%84" title="1910년">1910년</a> <a href="/wiki/8%EC%9B%94_28%EC%9D%BC" title="8월 28일">8월 28일</a>을 기해 <a href="/wiki/%EB%AA%AC%ED%85%8C%EB%84%A4%EA%B7%B8%EB%A1%9C_%EC%99%95%EA%B5%AD" title="몬테네그로 왕국">몬테네그로 왕국</a>으로 개편되었다.</p>
