<p><b>몬테네그로인</b>은 <a href="/wiki/%EB%82%A8%EC%8A%AC%EB%9D%BC%EB%B8%8C%EC%A1%B1" title="남슬라브족">남슬라브족</a>에 속하는 <a href="/wiki/%EB%AA%AC%ED%85%8C%EB%84%A4%EA%B7%B8%EB%A1%9C" title="몬테네그로">몬테네그로</a>의 주요 민족을 말하거나, 몬테네그로의 국민(<a href="/wiki/%EC%8B%9C%EB%AF%BC%EA%B6%8C" title="시민권">시민권</a>자)를 뜻한다. 보통은 남슬라브계의 몬테네그로 민족을 가리키는 표현으로 사용된다. 대부분의 <a href="/wiki/%EB%AA%AC%ED%85%8C%EB%84%A4%EA%B7%B8%EB%A1%9C" title="몬테네그로">몬테네그로</a>인이 <a href="/wiki/%EB%AA%AC%ED%85%8C%EB%84%A4%EA%B7%B8%EB%A1%9C" title="몬테네그로">몬테네그로</a>에 거주하지만, <a href="/wiki/%EC%84%B8%EB%A5%B4%EB%B9%84%EC%95%84" title="세르비아">세르비아</a>에도 일부 있다.</p>

<h2><span class="mw-headline" id=".EC.97.AD.EC.82.AC">역사</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%AA%AC%ED%85%8C%EB%84%A4%EA%B7%B8%EB%A1%9C%EC%9D%B8&amp;action=edit&amp;section=1" title="부분 편집: 역사">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<table class="notice section" style="text-align: left; margin:0 0 0.5em 0.5em; clear:none; font-size:smaller;">
<tr>
<td width="auto"><img alt="" data-file-height="400" data-file-width="400" height="25" src="//upload.wikimedia.org/wikipedia/commons/thumb/2/2c/Broom_icon.svg/25px-Broom_icon.svg.png" srcset="//upload.wikimedia.org/wikipedia/commons/thumb/2/2c/Broom_icon.svg/38px-Broom_icon.svg.png 1.5x, //upload.wikimedia.org/wikipedia/commons/thumb/2/2c/Broom_icon.svg/50px-Broom_icon.svg.png 2x" width="25"/> 이 문단은 <b><a href="/wiki/%EC%9C%84%ED%82%A4%EB%B0%B1%EA%B3%BC:%ED%8E%B8%EC%A7%91_%EC%A7%80%EC%B9%A8" title="위키백과:편집 지침">위키백과의 편집 지침</a></b>에 맞춰 다듬어야 합니다. 더 좋은 문단이 되도록 <span class="plainlinks"><a class="external text" href="//ko.wikipedia.org/w/index.php?title=%EB%AA%AC%ED%85%8C%EB%84%A4%EA%B7%B8%EB%A1%9C%EC%9D%B8&amp;action=edit">문단 수정을 도와주세요</a></span>. (2010년 2월 24일에 문단의 정리가 요청되었습니다.)</td>
</tr>

<p>초기에는 <a href="/wiki/%EB%A1%9C%EB%A7%88_%EC%A0%9C%EA%B5%AD" title="로마 제국">로마 제국</a>, <a class="mw-redirect" href="/wiki/%EB%B9%84%EC%9E%94%ED%8B%B4_%EC%A0%9C%EA%B5%AD" title="비잔틴 제국">비잔틴 제국</a>, <a href="/wiki/%EC%98%A4%EC%8A%A4%EB%A7%8C_%EC%A0%9C%EA%B5%AD" title="오스만 제국">오스만 제국</a>의 지배를 받아 왔다. 1852년에 <a href="/wiki/%EC%98%A4%EC%8A%A4%EB%A7%8C_%EC%A0%9C%EA%B5%AD" title="오스만 제국">오스만 제국</a>으로부터 <a href="/wiki/%EB%AA%AC%ED%85%8C%EB%84%A4%EA%B7%B8%EB%A1%9C_%EA%B3%B5%EA%B5%AD" title="몬테네그로 공국">몬테네그로 공국</a>이라는 이름으로 독립한 <a href="/wiki/%EB%AA%AC%ED%85%8C%EB%84%A4%EA%B7%B8%EB%A1%9C" title="몬테네그로">몬테네그로</a>는 사실상, <a href="/wiki/%EB%AA%AC%ED%85%8C%EB%84%A4%EA%B7%B8%EB%A1%9C" title="몬테네그로">몬테네그로</a>인의 역사가 시작된 것이다. 그러나, 1918년에 <a href="/wiki/%EC%84%B8%EB%A5%B4%EB%B9%84%EC%95%84" title="세르비아">세르비아</a>에 합병되어서 <a href="/wiki/%EB%AA%AC%ED%85%8C%EB%84%A4%EA%B7%B8%EB%A1%9C" title="몬테네그로">몬테네그로</a>의 역사는 세르비아의 역사하고 유사한 방식으로 몬테네그로인은 세르비아와 비슷해지도록 살게 한 것이다. <a href="/wiki/2006%EB%85%84" title="2006년">2006년</a> <a href="/wiki/6%EC%9B%94_5%EC%9D%BC" title="6월 5일">6월 5일</a> 완전 해방되었다.</p>

<h2><span class="mw-headline" id=".EB.AC.B8.ED.99.94">문화</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%AA%AC%ED%85%8C%EB%84%A4%EA%B7%B8%EB%A1%9C%EC%9D%B8&amp;action=edit&amp;section=3" title="부분 편집: 문화">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>88년 동안, <a href="/wiki/%EC%84%B8%EB%A5%B4%EB%B9%84%EC%95%84" title="세르비아">세르비아</a>의 지배를 받아 왔기 때문에, 지금의 <a href="/wiki/%EB%AA%AC%ED%85%8C%EB%84%A4%EA%B7%B8%EB%A1%9C" title="몬테네그로">몬테네그로</a>인의 생활은 <a href="/wiki/%EC%84%B8%EB%A5%B4%EB%B9%84%EC%95%84" title="세르비아">세르비아</a>와 유사하다. 종교는 <a class="new" href="/w/index.php?title=%EB%AA%AC%ED%85%8C%EB%84%A4%EA%B7%B8%EB%A1%9C_%EC%A0%95%EA%B5%90%ED%9A%8C&amp;action=edit&amp;redlink=1" title="몬테네그로 정교회 (없는 문서)">몬테네그로 정교회</a>를 믿고, 언어는 <a href="/wiki/%EB%AA%AC%ED%85%8C%EB%84%A4%EA%B7%B8%EB%A1%9C%EC%96%B4" title="몬테네그로어">몬테네그로어</a>를 쓴다.</p>
