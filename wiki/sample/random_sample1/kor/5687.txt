<p><b>몬페라토의 후작과 공작</b>은 <a href="/wiki/%EB%AA%AC%ED%8E%98%EB%9D%BC%ED%86%A0" title="몬페라토">몬페라토</a>가 불린 <a href="/wiki/%ED%86%A0%EB%A6%AC%EB%85%B8" title="토리노">토리노</a>의 동쪽과 <a href="/wiki/%ED%8F%AC_%EA%B0%95" title="포 강">포 강</a>의 남쪽에 있는 <a class="mw-redirect" href="/wiki/%ED%94%BC%EC%97%90%EB%AA%AC%ED%85%8C" title="피에몬테">피에몬테</a> 지역 일대의 통치자들이다. <a href="/wiki/%EB%AA%AC%ED%8E%98%EB%9D%BC%ED%86%A0_%EB%B3%80%EA%B2%BD%EB%B0%B1%EA%B5%AD" title="몬페라토 변경백국">몬페라토 변경백국</a>은 <a href="/wiki/%EB%B2%A0%EB%A0%8C%EA%B0%80%EB%A6%AC%EC%98%A4_2%EC%84%B8" title="베렌가리오 2세">베렌가리오 2세</a>가 이탈리아 왕국의 북서쪽에 권력을 재분배하던 기간인 950년에 세워졌다. <a class="new" href="/w/index.php?title=%EC%95%8C%EB%A0%88%EB%9D%BC%EB%AF%B8%EC%B9%98_%EA%B0%80%EB%AC%B8&amp;action=edit&amp;redlink=1" title="알레라미치 가문 (없는 문서)">알레라미치 가문</a>이 통치하게 되면서 몬페라토라는 이름이 붙게 되었다. 1574년, 몬페라토는 <a href="/wiki/%EB%A7%89%EC%8B%9C%EB%B0%80%EB%A6%AC%EC%95%88_2%EC%84%B8_(%EC%8B%A0%EC%84%B1_%EB%A1%9C%EB%A7%88_%EC%A0%9C%EA%B5%AD)" title="막시밀리안 2세 (신성 로마 제국)">막시밀리안 2세</a> 때 공국으로 승격하였다(<a class="new" href="/w/index.php?title=%EB%AA%AC%ED%8E%98%EB%9D%BC%ED%86%A0_%EA%B3%B5%EA%B5%AD&amp;action=edit&amp;redlink=1" title="몬페라토 공국 (없는 문서)">몬페라토 공국</a> 보기).</p>

<p>1536년 신성 로마 제국 황제 <a href="/wiki/%EC%B9%B4%EB%A5%BC_5%EC%84%B8" title="카를 5세">카를 5세</a>는 <a href="/wiki/%EC%82%B4%EB%A3%A8%EC%B4%88_%ED%9B%84%EA%B5%AD" title="살루초 후국">살루초 후국</a>과 사보이아의 주장에도 불구하고, 몬페라토 후국을 곤차가 가문에게 넘겨주기로 약속하였다. 이는 <a class="new" href="/w/index.php?title=%EC%B9%B4%ED%86%A0%EC%BA%89%EB%B8%8C%EB%A0%88%EC%8B%9C_%EC%A1%B0%EC%95%BD&amp;action=edit&amp;redlink=1" title="카토캉브레시 조약 (없는 문서)">카토캉브레시 조약</a>으로 승인되었다<sup class="reference" id="cite_ref-2"><a href="#cite_note-2">[2]</a></sup></p>

<li><a href="/wiki/%EA%B5%B4%EB%A6%AC%EC%97%90%EB%AA%A8_1%EC%84%B8_%EA%B3%A4%EC%B0%A8%EA%B0%80" title="굴리에모 1세 곤차가">굴리에모 10세</a> (1550–1587)</li>
