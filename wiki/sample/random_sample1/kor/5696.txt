<p><b>몰몬제비나비</b>(<a href="/wiki/%ED%95%99%EB%AA%85" title="학명">학명</a>: <i>Papilio polytes</i>)는 <a href="/wiki/%EB%82%98%EB%B9%84%EB%AA%A9" title="나비목">나비목</a> <a href="/wiki/%ED%98%B8%EB%9E%91%EB%82%98%EB%B9%84%EA%B3%BC" title="호랑나비과">호랑나비과</a>의 <a href="/wiki/%EB%82%98%EB%B9%84" title="나비">나비</a>이다.</p>
<p>성충은 앞날개 길이 50mm 정도, 날개 편 길이 7 - 8cm 정도이다. 다른 호랑나비과의 나비들에 비하면 소형으로, '몰몬제비나비'라는 이름은 영명 'Mormon butterfly'에서 유래한다.</p>
<p><a href="/wiki/%EC%9D%B8%EB%8F%84" title="인도">인도</a>부터 <a href="/wiki/%EB%8F%99%EB%82%A8%EC%95%84%EC%8B%9C%EC%95%84" title="동남아시아">동남아시아</a>의 <a href="/wiki/%EC%97%B4%EB%8C%80" title="열대">열대</a> 지역까지 넓게 분포하며, <a href="/wiki/%ED%95%9C%EB%B0%98%EB%8F%84" title="한반도">한반도</a>에는 분포하지 않으며 <a href="/wiki/%EB%AF%B8%EC%A0%91" title="미접">미접</a>으로 발견된 적이 없는 열대성 나비이다. <a href="/wiki/%EC%9D%BC%EB%B3%B8_%EC%97%B4%EB%8F%84" title="일본 열도">일본 열도</a>에서는 <a href="/wiki/%EB%8F%84%EC%B9%B4%EB%9D%BC_%EC%97%B4%EB%8F%84" title="도카라 열도">도카라 열도</a>, <a class="new" href="/w/index.php?title=%EB%82%98%EC%B9%B4%EB%85%B8_%EC%84%AC&amp;action=edit&amp;redlink=1" title="나카노 섬 (없는 문서)">나카노 섬</a> 이남의 <a href="/wiki/%EB%82%9C%EC%84%B8%EC%9D%B4_%EC%A0%9C%EB%8F%84" title="난세이 제도">난세이 제도</a>에 분포한다. 오키나와 지방에서는 흔하게 볼 수 있고, <a href="/wiki/%EC%95%BC%EC%97%90%EC%95%BC%EB%A7%88_%EC%A0%9C%EB%8F%84" title="야에야마 제도">야에야마 제도</a>에서는 연중, <a class="mw-redirect" href="/wiki/%EC%95%84%EB%A7%88%EB%AF%B8_%EA%B5%B0%EB%8F%84" title="아마미 군도">아마미 군도</a>에서는 2월 중순 - 11월 하순에 보인다. 1년에 5 - 6회 발생한다. 붉은 꽃에 잘 모이며, 황무지나 숲,<a class="mw-redirect" href="/wiki/%ED%95%B4%EC%95%88" title="해안">해안</a> 등에서 볼 수 있다.</p>
<p><a class="mw-redirect" href="/wiki/%EC%9C%A0%EC%B6%A9" title="유충">유충</a>은 <a class="mw-redirect" href="/wiki/%EA%B7%A4" title="귤">귤</a> 등 <a href="/wiki/%EC%9A%B4%ED%96%A5%EA%B3%BC" title="운향과">운향과</a> 각종을 식초로 하며 이 때문에 <a href="/wiki/%ED%95%B4%EC%B6%A9" title="해충">해충</a>으로 여겨진다. 유충은 다른 <a href="/wiki/%ED%98%B8%EB%9E%91%EB%82%98%EB%B9%84%EC%86%8D" title="호랑나비속">호랑나비속</a>처럼 1 - 4령까지 새똥과 같은 색이고, 5령이 되면 녹색이 된다. 위험을 느끼면 붉은 뿔을 내밀며, 이 뿔에서는 고약한 냄새가 난다.</p>

<p>몰몬제비나비의 암컷은 유전적으로 2가지 형태가 나타난다. 수컷과 같이 뒷날개에 흰 띠가 있는 「보통형」(흰띠형), 뒷날개에 흰 띠와 함께 적색의 반점이 있는 「붉은무늬형」(적문형)이다. 붉은무늬형은 <a class="new" href="/w/index.php?title=%EB%B6%89%EC%9D%80%EC%82%AC%ED%96%A5%EC%A0%9C%EB%B9%84%EB%82%98%EB%B9%84&amp;action=edit&amp;redlink=1" title="붉은사향제비나비 (없는 문서)">붉은사향제비나비</a>(<i>Pachliopta aristolochiae</i>)가 서식하는 장소에 나타나지만, 붉은사향제비나비가 서식하지 않는 곳에는 붉은무늬형은 나타나지 않는다. 같은 남방계 나비인 붉은사향제비나비는 유충 시대에 <a href="/wiki/%EC%A5%90%EB%B0%A9%EC%9A%B8%EB%8D%A9%EA%B5%B4%EA%B3%BC" title="쥐방울덩굴과">쥐방울덩굴과</a>의 식물을 먹이로 삼으며, 성충이 되어도 몸안에 섭식식물의 독소가 있다. 이 때문에 베니몬나비를 포식자(주로 조류)가 포식하면 <a href="/wiki/%EC%A4%91%EB%8F%85" title="중독">중독</a>을 일으켜 나비를 토해 버리며, 붉은사향제비나비를 먹지 않게 된다. 몰몬제비나비는 본래 독이 없지만, 붉은사향제비나비의 체색을 흉내내는 것으로, 적으로부터 몸을 지키고 있다고 생각할 수 있다. 이를 베이츠 <a href="/wiki/%EC%9D%98%ED%83%9C" title="의태">의태</a>의 한 예시로 볼 수 있다.</p>
