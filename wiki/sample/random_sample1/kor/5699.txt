<p><b>몰타어</b>(<span lang="mt" xml:lang="mt">Malti, Lingwa Maltija</span>)는 <a href="/wiki/%EB%AA%B0%ED%83%80" title="몰타">몰타 공화국</a>에서 <a href="/wiki/%EA%B3%B5%EC%9A%A9%EC%96%B4" title="공용어">공용어</a>로서 쓰이고 있는 <a href="/wiki/%EC%96%B8%EC%96%B4" title="언어">언어</a>이다. 몰타어는 구어체 <a href="/wiki/%EC%95%84%EB%9E%8D%EC%96%B4" title="아랍어">아랍어</a>의 변종의 하나이지만, <a href="/wiki/%EB%A1%9C%EB%A7%9D%EC%8A%A4%EC%96%B4%EA%B5%B0" title="로망스어군">로망스계</a> <a href="/wiki/%EC%96%B4%ED%9C%98" title="어휘">어휘</a>의 <a class="mw-redirect" href="/wiki/%EC%B0%A8%EC%9A%A9%EC%96%B4" title="차용어">차용어</a>가 많기 때문에, <a href="/wiki/%EC%95%84%EB%9E%8D%EC%96%B4" title="아랍어">아랍어</a>와는 다른 언어로 보는 화자도 있다. 유럽에서는 유일하게 <a class="mw-redirect" href="/wiki/%EC%95%84%ED%94%84%EB%A6%AC%EC%B9%B4%EC%95%84%EC%8B%9C%EC%95%84%EC%96%B4%EC%A1%B1" title="아프리카아시아어족">아프리카아시아어족</a> <a href="/wiki/%EC%85%88%EC%96%B4%ED%8C%8C" title="셈어파">셈어파</a>의 언어이다. 구어체 아랍어 중에서는 <a href="/wiki/%EB%A7%88%EA%B7%B8%EB%A0%88%EB%B8%8C" title="마그레브">마그레브</a> 방언, 특히 <a class="new" href="/w/index.php?title=%ED%8A%80%EB%8B%88%EC%A7%80_%EC%95%84%EB%9E%8D%EC%96%B4&amp;action=edit&amp;redlink=1" title="튀니지 아랍어 (없는 문서)">튀니지 아랍어</a>와 가장 가깝다. <a href="/wiki/%EC%8B%9C%EC%B9%A0%EB%A6%AC%EC%95%84%EC%96%B4" title="시칠리아어">시칠리아어</a>, <a href="/wiki/%EC%9D%B4%ED%83%88%EB%A6%AC%EC%95%84%EC%96%B4" title="이탈리아어">이탈리아어</a> 등, <a href="/wiki/%EB%A1%9C%EB%A7%9D%EC%8A%A4%EC%96%B4%EA%B5%B0" title="로망스어군">로망스계</a> 어휘를 많이 포함하며, 셈계 언어에서 유일한 <a href="/wiki/%EB%A1%9C%EB%A7%88%EC%9E%90" title="로마자">로마자</a>에 의한 <a class="new" href="/w/index.php?title=%EC%A0%95%EC%84%9C%EB%B2%95&amp;action=edit&amp;redlink=1" title="정서법 (없는 문서)">정서법</a>을 가진다.</p>
<p><a href="/wiki/1934%EB%85%84" title="1934년">1934년</a> <a href="/wiki/%EC%98%81%EC%96%B4" title="영어">영어</a>와 함께 몰타의 공용어로 채용되었다. 요즘은 약 60만 명의 화자가 있다. <a href="/wiki/%EC%98%A4%EC%8A%A4%ED%8A%B8%EB%A0%88%EC%9D%BC%EB%A6%AC%EC%95%84" title="오스트레일리아">오스트레일리아</a>나 <a href="/wiki/%EB%AF%B8%EA%B5%AD" title="미국">미국</a>, <a href="/wiki/%EC%BA%90%EB%82%98%EB%8B%A4" title="캐나다">캐나다</a>로 이민한 사람들 사이에서도 사용되고 있다.</p>

<h2><span class="mw-headline" id=".EB.AC.B8.EB.B2.95">문법</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%AA%B0%ED%83%80%EC%96%B4&amp;action=edit&amp;section=2" title="부분 편집: 문법">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>로망스어의 영향을 받고는 있지만, 몰타어의 문법은 여전히 <a href="/wiki/%EC%95%84%EB%9E%8D%EC%96%B4" title="아랍어">아랍어</a> 자체이다. <a href="/wiki/%ED%98%95%EC%9A%A9%EC%82%AC" title="형용사">형용사</a>는 <a class="mw-disambig" href="/wiki/%EB%AA%85%EC%82%AC" title="명사">명사</a>를 뒤에서 수식하며, <a class="new" href="/w/index.php?title=%EB%B3%B8%EB%9E%98%EB%B6%80%EC%82%AC&amp;action=edit&amp;redlink=1" title="본래부사 (없는 문서)">본래부사</a>는 적고, <a href="/wiki/%EC%96%B4%EC%88%9C" title="어순">어순</a>은 상당히 자유롭다. <a href="/wiki/%EC%95%84%EB%9E%8D%EC%96%B4" title="아랍어">아랍어</a>나 <a href="/wiki/%ED%9E%88%EB%B8%8C%EB%A6%AC%EC%96%B4" title="히브리어">히브리어</a>와 같이, 명사가 <a class="mw-redirect" href="/wiki/%EC%A0%95%EA%B4%80%EC%82%AC" title="정관사">정관사</a>를 취하는 경우, 그것을 수식하는 셈어 기원의 형용사도 정관사를 취한다(예: L-Art l-Imqaddsa(=&lt;정관사&gt;-토지 &lt;정관사&gt;-성스러운). 덧붙여서 아랍어에서는 'al-'arḍ 'al-muqaddasa, 히브리어에서는 ha'arets hakkedoša이다. 단, 형용사가 정관사를 취하지 않는 경우도 있다. 로망스어 기원의 형용사는 정관사를 취하지 않는다.</p>
<p>명사의 수에는 부분적으로 <a href="/wiki/%EC%88%98_(%EB%AC%B8%EB%B2%95)" title="수 (문법)">쌍수</a>가 잔존하고 있다.</p>
<p><a class="mw-disambig" href="/wiki/%EB%8F%99%EC%82%AC" title="동사">동사</a>는 아랍어의 3어근의 패턴을 유지하고 있다. 즉, 자음3개로 된 어근에 <a href="/wiki/%EC%A0%91%EB%91%90%EC%82%AC" title="접두사">접두사</a>, <a class="new" href="/w/index.php?title=%EC%A0%91%EC%A4%91%EC%82%AC&amp;action=edit&amp;redlink=1" title="접중사 (없는 문서)">접중사</a>, <a href="/wiki/%EC%A0%91%EB%AF%B8%EC%82%AC" title="접미사">접미사</a>를 덧붙이는 것으로 동사에 문법적인 의미를 부여하는 것이다. 예를 들면, "(글을) 쓰다"를 나타내는 3개의 자음 KTB에 접중사 -i-, 접미사 -na를 더해서 ktibna(우리들은 썼다)를 만든다(덧붙여서 말하면 아랍어에서는 katabna, 히브리어에서는 katavnu). <a href="/wiki/%EC%8B%9C%EC%A0%9C" title="시제">시제</a>는 <a class="new" href="/w/index.php?title=%EB%B9%84%EA%B3%BC%EA%B1%B0&amp;action=edit&amp;redlink=1" title="비과거 (없는 문서)">비과거</a>와 <a href="/wiki/%EA%B3%BC%EA%B1%B0%EC%8B%9C%EC%A0%9C" title="과거시제">과거</a>의 2개이다.</p>
<p>로망스어 기원의 동사에는 아랍어적인 동사파생법은 적용되지 않지만, 동사활용은 아랍어의 활용법을 그대로 사용하며 접두사와 접미사가 붙는다. 예를 들면, iddeċidejna(우리들은 정했다)는 로망스어의 동사 (i)ddeċida에 아랍어의 어미 -ejna(1인칭복수완료)가 붙은 것이다. <a class="new" href="/w/index.php?title=%ED%8A%80%EB%8B%88%EC%A7%80_%EC%95%84%EB%9E%8D%EC%96%B4&amp;action=edit&amp;redlink=1" title="튀니지 아랍어 (없는 문서)">튀니지 아랍어</a> 등 몇 개의 방언에서 이러한 동사활용을 볼 수 있다고는 하나, 아랍어에서는 이러한 동사활용은 실행하지 않는 것이 기본이다.</p>
<p>몰타어의 문법에는 아랍어적인 것과 로망스어적인 것이 공존하고 있으며, 어휘의 기원과 관습에 따라 가려 쓴다. 영어로부터의 차용어에 보이는 <a class="mw-redirect" href="/wiki/%EC%95%B5%EA%B8%80%EB%A1%9C%EC%83%89%EC%8A%A8%EC%96%B4" title="앵글로색슨어">앵글로색슨어</a>적인 요소는 최근의 현상이다.</p>
<p>로망스어적인 문법은 대체로 주변적이다. 로망스기원의 명사는 -i이나 -ijiet을 덧붙이는 것으로 복수가 된다. 예를 들면, lingwa(언어)는 lingwi로 복수화된다. 아랍어어휘의 복수는 조금 더 복잡하다. 어미복수는 -at/-iet/-ijiet(아랍어의 -at, 히브리어의 -ot), -(i)n(아랍어의 -iin, 히브리어의 -im), -an/-ien(아랍어의 -aan)을 붙이는 것으로 행해지며, 어내(語內)복수는 ktieb(책)/kotba; raġel(남자)/(i)rġiel같이 모음의 변화에 따라 나타내어진다. 이것들은 아랍어외에 셈어계에서는 일반적인 굴절이다(히브리어에서는 sefer(책)/ sfarim)</p>
<h2><span class="mw-headline" id=".EC.96.B4.ED.9C.98">어휘</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%AA%B0%ED%83%80%EC%96%B4&amp;action=edit&amp;section=3" title="부분 편집: 어휘">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>몰타어의 어휘는 아랍어 어휘를 기초로 하며(이탈리아어 <a class="new" href="/w/index.php?title=%ED%86%A0%EC%8A%A4%EC%B9%B4%EB%82%98_%EB%B0%A9%EC%96%B8&amp;action=edit&amp;redlink=1" title="토스카나 방언 (없는 문서)">토스카나 방언</a>보다 오히려) <a href="/wiki/%EC%8B%9C%EC%B9%A0%EB%A6%AC%EC%95%84%EC%96%B4" title="시칠리아어">시칠리아어</a>에서 방대한 차용어를 받아들인 하이브리드이다. 이 점에 있어 <a class="mw-redirect" href="/wiki/%EB%A1%9C%EB%A7%9D%EC%8A%A4%EC%96%B4" title="로망스어">로망스어</a>의 영향을 강하게 받은 <a class="mw-redirect" href="/wiki/%EA%B2%8C%EB%A5%B4%EB%A7%8C%EC%96%B4" title="게르만어">게르만어</a>인 <a href="/wiki/%EC%98%81%EC%96%B4" title="영어">영어</a>, 아랍어의 영향을 강하게 받은 <a href="/wiki/%EC%9D%B8%EB%8F%84%EC%9D%B4%EB%9E%80%EC%96%B4%ED%8C%8C" title="인도이란어파">인도이란어파</a>의 <a href="/wiki/%ED%8E%98%EB%A5%B4%EC%8B%9C%EC%95%84%EC%96%B4" title="페르시아어">페르시아어</a>와 닮아 있다.</p>
<p>어휘의 60%는 아랍어 어휘이며, 나머지는 로망스어라고 계산된다. Zammit (2000)에 의하면 <a href="/wiki/%ED%98%84%EB%8C%80_%ED%91%9C%EC%A4%80_%EC%95%84%EB%9E%8D%EC%96%B4" title="현대 표준 아랍어">쿠란 아랍어</a>로부터 샘플을 취한 1,820개 단어의 어근 중 40%가 몰타어에서 찾아낼 수 있다라고 하는 것이다. 이것은 아랍어의 <a class="new" href="/w/index.php?title=%EB%AA%A8%EB%A1%9C%EC%BD%94_%EC%95%84%EB%9E%8D%EC%96%B4&amp;action=edit&amp;redlink=1" title="모로코 아랍어 (없는 문서)">모로코 아랍어</a>(58%)나 <a class="new" href="/w/index.php?title=%EC%8B%9C%EB%A6%AC%EC%95%84_%EC%95%84%EB%9E%8D%EC%96%B4&amp;action=edit&amp;redlink=1" title="시리아 아랍어 (없는 문서)">시리아 아랍어</a>(72%)에 비해 적다. 일반적으로 기본적인 개념을 나타내는 어휘는 아랍어이며, 새로운 개념, 물건, 정부, 법률, 교육, 예술, 문학에 관한 "고상"한 어휘는 시칠리아어로부터 차용되어 있다. 따라서, raġel(남자), mara(여자), tifel(아이), dar(집), xemx(태양), sajf(여름)이라고 말한 어휘는 전부 아랍어가 기원이며, skola(학교), gvern(정부), repubblika(공화국), re(왕), natura(자연), pulizija(경찰), ċentru(중심), teatru(극장, 영화관), differenza(차이) 등은 시칠리아어로부터의 차용어이다. 따라서, 로망스어 화자는 <a class="external text" href="http://mt.wikipedia.org">몰타어판 위키백과의 메인 페이지</a> 나 신문 기사의 표제를 이해할 수는 있지만, "그 남자는 집에 있다"같은 기본적인 문장조차 이해할 수 없다. 이것은 외국어가 할 수 없는 영어화자가 공식적인 또는 학술적인 프랑스어의 의미를 추측할 수 있는데도, 간단한 문장은 오히려 이해할 수 없다라는 것을 닮았다.</p>
<p>예로서 <a href="/wiki/%EC%84%B8%EA%B3%84_%EC%9D%B8%EA%B6%8C_%EC%84%A0%EC%96%B8" title="세계 인권 선언">세계 인권 선언</a>의 첫 번째 부분을 든다.</p>
<p>Il-bnedmin kollha jitwieldu ħielsa u ugwali fid-dinjità u d-drittijiet. Huma mogħnija bir-raġuni u bil-kuxjenza u għandhom iġibu ruħhom ma' xulxin bi spirtu ta' aħwa.</p>
<p>로망스계 어휘는 이탈리아어 토스카나 방언뿐만이 아니고 시칠리어의 발음을 반영하고 있다. 따라서, 이탈리아어의 어말의 o는 u로, 어말의 e는 i가 되고 있다. 예를 들면, 이탈리아어의 Cristo(<a href="/wiki/%EA%B7%B8%EB%A6%AC%EC%8A%A4%EB%8F%84" title="그리스도">그리스도</a>), arte(예술)은 몰타어에서는 각각 Kristu(시칠리아어에서는 Cristu), arti(시칠리아어에서도 arti)이다.</p>
