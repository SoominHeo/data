<p><b>묠니르</b>(<span style="font-size: smaller;"><a href="/wiki/%EA%B3%A0%EB%8C%80_%EB%85%B8%EB%A5%B4%EB%93%9C%EC%96%B4" title="고대 노르드어">고대 노르드어</a>:</span> <span lang="non" xml:lang="non">Mjǫlnir</span> <small>[ˈmjɒlnɪər]</small><font color="gray"><small>→박살내는 것</small></font>)는 <a href="/wiki/%EB%85%B8%EB%A5%B4%EB%93%9C_%EC%8B%A0%ED%99%94" title="노르드 신화">노르드 신화</a>의 천둥신 <a href="/wiki/%ED%86%A0%EB%A5%B4" title="토르">토르</a>의 망치이다. 묠니르는 산을 평지로 만들 수 있는, 노르드 신화에서 가장 무시무시한 무기 중 하나이다.<sup class="reference" id="cite_ref-1"><a href="#cite_note-1">[1]</a></sup><sup class="reference" id="cite_ref-2"><a href="#cite_note-2">[2]</a></sup><sup class="reference" id="cite_ref-3"><a href="#cite_note-3">[3]</a></sup> <a class="mw-redirect" href="/wiki/%EC%8A%A4%EB%85%B8%EB%A6%AC_%EC%8A%A4%ED%88%B4%EB%A3%A8%EC%86%90" title="스노리 스툴루손">스노리 스툴루손</a>에 따르면 <a class="new" href="/w/index.php?title=%EC%8B%A0%EB%93%9C%EB%A6%AC&amp;action=edit&amp;redlink=1" title="신드리 (없는 문서)">신드리</a>와 <a class="new" href="/w/index.php?title=%EB%B8%8C%EB%A1%9D%ED%81%AC&amp;action=edit&amp;redlink=1" title="브록크 (없는 문서)">브록크</a>라는 <a href="/wiki/%EB%93%9C%EB%B2%A0%EB%A5%B4%EA%B7%B8" title="드베르그">드베르그</a> 형제가 이 망치를 만들었다고 한다.</p>

<h2><span class="mw-headline" id=".EC.96.B4.EC.9B.90">어원</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%AC%A0%EB%8B%88%EB%A5%B4&amp;action=edit&amp;section=1" title="부분 편집: 어원">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p><a class="mw-redirect" href="/wiki/%EB%85%B8%EB%A5%B4%EB%93%9C%EC%96%B4" title="노르드어">노르드어</a> 낱말 <span lang="non" xml:lang="non"><i>Mjǫllnir</i></span> <small>묠니르['mjɔlːnir]</small>는 13세기가 되면 <a class="new" href="/w/index.php?title=%EA%B3%A0%EB%8C%80_%EC%95%84%EC%9D%B4%EC%8A%AC%EB%9E%80%EB%93%9C%EC%96%B4&amp;action=edit&amp;redlink=1" title="고대 아이슬란드어 (없는 문서)">고대 아이슬란드어</a> 문헌에서 <span lang="is" xml:lang="is"><i>Mjøllnir</i></span> <small>묄니르['mjœlːnir]</small>로 변한다.<sup class="reference" id="cite_ref-ON_spelling_4-0"><a href="#cite_note-ON_spelling-4">[4]</a></sup> 현대 아이슬란드어 형태는 <span lang="is" xml:lang="is"><i>Mjölnir</i></span>이며, 노르웨이와 덴마크에서는 <span lang="no" xml:lang="no"><i>Mjølner</i></span>, 스웨덴에서는 <span lang="sv" xml:lang="sv"><i>Mjölner</i></span> 라고 한다.</p>
<p>"묠니르"의 어원은 게르만 조어 <i>*meldunjaz</i>이며, 그 어근은 "갈아 빻다" 라는 뜻의 <i><a class="extiw" href="https://ko.wiktionary.org/wiki/en:Appendix:Proto-Germanic/malan%C4%85" title="wikt:en:Appendix:Proto-Germanic/malaną">*malanan</a></i> 이다. <i><a class="extiw" href="https://ko.wiktionary.org/wiki/en:Appendix:Proto-Germanic/melw%C4%85" title="wikt:en:Appendix:Proto-Germanic/melwą">*melwan</a></i> 은 "밀가루"라는 뜻이며, 고대 아이슬란드어로는 <span lang="is" xml:lang="is"><i>meldr, mjǫll, mjǫl</i></span> 이라고 한다. 노르드어 <span lang="non" xml:lang="non"><i>mala</i></span> <small>말라</small>, 고트어·고대 고지독일어·고대 색슨어 <span lang="goh" xml:lang="goh"><i>malan</i></span>는 리투아니아어 <span lang="lt" xml:lang="lt"><i>malŭ, malti</i></span>, 라트비아어 <span lang="lv" xml:lang="lv"><i>maíu</i></span>, 고대 교회 슬라브어 <span lang="cu" xml:lang="cu"><i><a class="extiw" href="https://ko.wiktionary.org/wiki/en:%D0%BC%D0%BE%D0%BB%D0%BE%D1%82%D1%8C" title="wikt:en:молоть">meljǫ</a>, <a class="extiw" href="https://ko.wiktionary.org/wiki/en:%D0%BC%D0%BB%D1%A3%D1%82%D0%B8" title="wikt:en:млѣти">mlěti</a></i></span>, 고대 아일랜드어 <span lang="sga" xml:lang="sga"><i>melim</i></span>, 그리스어 <span lang="el" xml:lang="el">μύλλω</span>, 라틴어 <span lang="la" xml:lang="la"><i>molō</i></span>와 비교할 수 있으며, 이것들은 모두 "갈아 빻다"라는 뜻이다. <a href="/wiki/%EC%82%B0%EC%8A%A4%ED%81%AC%EB%A6%AC%ED%8A%B8%EC%96%B4" title="산스크리트어">산스크리트어</a> <span lang="sa" xml:lang="sa">मृणाति</span> <small>므르나티mr̥ṇā́ti</small>는 "박살내다, 으깨다, 쳐죽이다"라는 뜻이다.<sup class="reference" id="cite_ref-5"><a href="#cite_note-5">[5]</a></sup> 즉 "묠니르"는 "갈아 빻는 것, 박살내는 것"이라고 해석할 수 있다.</p>
<h2><span class="mw-headline" id=".EC.B6.9C.EC.A0.84">출전</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%AC%A0%EB%8B%88%EB%A5%B4&amp;action=edit&amp;section=2" title="부분 편집: 출전">편집</a><span class="mw-editsection-bracket">]</span></span></h2>

<h3><span class="mw-headline" id=".EA.B3.A0_.EC.97.90.EB.8B.A4">고 에다</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%AC%A0%EB%8B%88%EB%A5%B4&amp;action=edit&amp;section=3" title="부분 편집: 고 에다">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>토르는 두 마리 염소 <a href="/wiki/%ED%83%95%EA%B7%B8%EB%A6%AC%EC%8A%A4%EB%8B%88%EB%A5%B4%EC%99%80_%ED%83%95%EA%B7%B8%EB%87%A8%EC%8A%A4%ED%8A%B8" title="탕그리스니르와 탕그뇨스트">탕그리스니르와 탕그뇨스트</a>가 모는 <a href="/wiki/%EC%A0%84%EC%B0%A8_(%EA%B3%A0%EB%8C%80_%EB%AC%B4%EA%B8%B0)" title="전차 (고대 무기)">전차</a>, 혁대 <a href="/wiki/%EB%A9%94%EA%B9%85%EC%9A%94%EB%A5%B4%EB%93%9C" title="메깅요르드">메깅요르드</a>, 무쇠 장갑 <a href="/wiki/%EC%95%BC%EB%A6%89%EA%B7%B8%EB%A0%88%EC%9D%B4%ED%94%84" title="야릉그레이프">야릉그레이프</a>를 가지고 있는데, 묠니르를 들어올리기 위해서는 야릉그레이프를 껴야 한다. 묠니르는 토르의 모험담들 중 몇몇 경우의 중심 소재가 된다.</p>
<p>이러한 사실은 《<a href="/wiki/%EA%B3%A0_%EC%97%90%EB%8B%A4" title="고 에다">고 에다</a>》 중 〈<a class="mw-redirect" href="/wiki/%EC%8A%A4%EB%A5%8C%EC%9D%98_%EC%84%9C%EC%82%AC%EC%8B%9C" title="스륌의 서사시">스륌의 서사시</a>〉에 명확하게 언급된다. 거인 <a class="mw-redirect" href="/wiki/%EC%8A%A4%EB%A5%8C" title="스륌">스륌</a>이 묠니르를 훔쳐가서는 돌려받고 싶거들랑 여신 <a href="/wiki/%ED%94%84%EB%A0%88%EC%9D%B4%EC%95%BC" title="프레이야">프레이야</a>를 자기 신부로 내놓으라고 요구한다. 묠니르를 되찾아 오기 위해 <a href="/wiki/%EB%A1%9C%ED%82%A4" title="로키">로키</a>는 다른 <a class="mw-redirect" href="/wiki/%EC%97%90%EC%8B%9C%EB%A5%B4" title="에시르">에시르</a>와 공모하여 토르를 프레이야로 변장시키고 스륌에게 데려간다. 결혼식 피로연 자리에서 스륌은 부부의 결합을 상징하기 위해 "신부"의 무릎에 묠니르를 올려놓는다. 그러자 토르는 변장을 찢어발기고 묠니르를 집어들어 스륌을 비롯해 결혼식에 모인 거인들을 모조리 쳐죽여 버렸다.</p>
<h3><span class="mw-headline" id=".EC.8B.A0_.EC.97.90.EB.8B.A4">신 에다</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%AC%A0%EB%8B%88%EB%A5%B4&amp;action=edit&amp;section=4" title="부분 편집: 신 에다">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>묠니르의 기원에 대한 이야기는 《<a href="/wiki/%EC%8B%A0_%EC%97%90%EB%8B%A4" title="신 에다">신 에다</a>》 중 〈<a href="/wiki/%EC%8B%9C%EC%96%B4%EB%B2%95" title="시어법">시어법</a>〉에 나온다.<sup class="reference" id="cite_ref-6"><a href="#cite_note-6">[6]</a></sup></p>
<p><a href="/wiki/%EB%93%9C%EB%B2%A0%EB%A5%B4%EA%B7%B8" title="드베르그">드베르그</a> <a class="new" href="/w/index.php?title=%EC%9D%B4%EB%B0%9C%EB%94%94%EC%9D%98_%EC%95%84%EB%93%A4%EB%93%A4&amp;action=edit&amp;redlink=1" title="이발디의 아들들 (없는 문서)">이발디의 아들들</a>이 <a href="/wiki/%EC%98%A4%EB%94%98" title="오딘">오딘</a>의 창 <a href="/wiki/%EA%B6%81%EB%8B%88%EB%A5%B4" title="궁니르">궁니르</a>, <a href="/wiki/%ED%94%84%EB%A0%88%EC%9D%B4" title="프레이">프레이</a>의 배 <a href="/wiki/%EC%8A%A4%ED%82%A4%EB%93%9C%EB%B8%94%EB%9D%BC%EB%93%9C%EB%8B%88%EB%A5%B4" title="스키드블라드니르">스키드블라드니르</a>를 만들었는데, <a href="/wiki/%EB%A1%9C%ED%82%A4" title="로키">로키</a>가 다른 드베르그 <a class="new" href="/w/index.php?title=%EC%8B%A0%EB%93%9C%EB%A6%AC&amp;action=edit&amp;redlink=1" title="신드리 (없는 문서)">신드리</a>(또는 <a href="/wiki/%EC%97%90%EC%9D%B4%ED%8A%B8%EB%A6%AC" title="에이트리">에이트리</a>)와 <a class="new" href="/w/index.php?title=%EB%B8%8C%EB%A1%9D%ED%81%AC&amp;action=edit&amp;redlink=1" title="브록크 (없는 문서)">브록크</a> 형제를 만나 그들은 이발디의 아들들의 발명품보다 더 멋진 물건을 만들어내지 못할 것이라고 장담하며 자기 머리를 건다.</p>
<p>신드리와 브록크는 로키의 내기를 받아들여 작업을 개시한다. 신드리는 돼지 가죽을 로(爐)에 집어넣고 동생에게 자기가 그만하라고 할 때까지 계속 <a href="/wiki/%ED%92%80%EB%AC%B4" title="풀무">풀무</a>질을 하라고 지시한다. <a class="mw-disambig" href="/wiki/%ED%8C%8C%EB%A6%AC" title="파리">파리</a>로 둔갑한 로키가 브록크의 팔을 물어뜯으면서 방해하지만 브록크는 계속 풀무질을 했다.</p>
<p>그리하여 신드리는 빛나는 털을 가진 멧돼지 <a href="/wiki/%EA%B5%B4%EB%A6%B0%EB%B6%80%EB%A5%B4%EC%8A%A4%ED%8B%B0" title="굴린부르스티">굴린부르스티</a>를 뽑다냈다. 그 뒤 신드리는 로 속에 황금을 집어넣고 브록크에게 다시 같은 지시를 내린다. 파리로 둔갑한 로키가 다시 이번에는 브록크의 목을 물어뜯으며 방해한다. 이번에도 브록크는 방해를 뿌리치고 계속 풀무질을 했고, 돌아온 신드리는 9일마다 여덟 배로 불어나는 오딘의 반지 <a href="/wiki/%EB%93%9C%EB%9D%BC%EC%9A%B0%ED%94%84%EB%8B%88%EB%A5%B4" title="드라우프니르">드라우프니르</a>를 뽑아낸다.</p>
<p>마지막으로 신드리는 무쇠를 로 속에 집어넣고 브록크에게 풀무질을 절대 멈추지 말라고 지시한다. 세 번째로 찾아론 로키는 이번에는 브록크의 눈꺼풀을 물어뜯는다. 너무 세게 물어뜯은 나머지 피가 새어나왔고, 눈에 피가 들어간 브록크는 눈을 닦느라고 풀무질을 잠깐 멈추고 말았다. 잠시 뒤 돌아온 신드리는 로 속에서 묠니르를 꺼냈다. 그러나 브록크가 풀무질을 멈춘 바람에 묠니르의 손잡이가 너무 짧아져서 한 손으로밖에 들 수 없게 되었다.</p>
<p>망치 손잡이에 하자가 있음에도 불구하고 신드리 형제는 내기에서 이기고 로키의 머리를 가질 수 있게 되었다. 그러나 로키는 그들이 가진 것은 자신의 머리 뿐이지 목은 가지지 못했으므로 머리를 목에서 베어낼 수 없다고 우겼다. 그러자 브록크는 로키의 입을 꿰메어 그 입을 닥치게 만들었다.</p>

<h3><span class="mw-headline" id=".EA.B2.8C.EB.A5.B4.EB.A7.8C.EC.A1.B1_.EB.8C.80.EC.9D.B4.EB.8F.99_.EC.8B.9C.EB.8C.80">게르만족 대이동 시대</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%AC%A0%EB%8B%88%EB%A5%B4&amp;action=edit&amp;section=6" title="부분 편집: 게르만족 대이동 시대">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>바이킹 시대 이전에 나타나는 묠니르의 흔적은 <a href="/wiki/%EB%AF%BC%EC%A1%B1_%EB%8C%80%EC%9D%B4%EB%8F%99" title="민족 대이동">민족 대이동</a> 시대 <a href="/wiki/%EC%95%8C%EB%A0%88%EB%A7%8C%EB%8B%88" title="알레만니">알레만니</a>에 관한 기록에서 나타난다. 로마인들은 알레만니의 <a class="mw-redirect" href="/wiki/%EB%8F%84%EB%82%98%EB%A5%B4" title="도나르">도나르</a> 신의 상징을 <a class="new" href="/w/index.php?title=%ED%97%A4%EB%9D%BC%ED%81%B4%EB%A0%88%EC%8A%A4%EC%9D%98_%EA%B3%A4%EB%B4%89&amp;action=edit&amp;redlink=1" title="헤라클레스의 곤봉 (없는 문서)">헤라클레스의 곤봉</a>과 동일시했다.<sup class="reference" id="cite_ref-7"><a href="#cite_note-7">[7]</a></sup></p>
<h3><span class="mw-headline" id=".EB.B0.94.EC.9D.B4.ED.82.B9_.EC.8B.9C.EB.8C.80.EC.9D.98_.EB.B6.80.EC.A0.81">바이킹 시대의 부적</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%AC%A0%EB%8B%88%EB%A5%B4&amp;action=edit&amp;section=7" title="부분 편집: 바이킹 시대의 부적">편집</a><span class="mw-editsection-bracket">]</span></span></h3>

<p>스칸디나비아 전역에서 9세기에서 11세기 사이에 만드렁진 묠니르 부적 50여 개가 발견되었다. 이들이 가장 흔하게 발견된 곳은 <a href="/wiki/%EA%B8%B0%EB%8F%85%EA%B5%90" title="기독교">기독교</a>의 영향이 강하게 미친 노르웨이 남부, 스웨덴 동남부, 덴마크 등지이다.<sup class="reference" id="cite_ref-8"><a href="#cite_note-8">[8]</a></sup> 묠니르는 기독교의 <a href="/wiki/%EC%8B%AD%EC%9E%90%EA%B0%80" title="십자가">십자가</a>와 유사하게 생겼기 때문에, 당대 스칸디나비아를 잠식해 오던 기독교에 대한 반항적 행위로 묠니르 부적을 목에 매고 다니는 행위가 유행하게 된 것으로 보인다.<sup class="reference" id="cite_ref-GODSDAV_9-0"><a href="#cite_note-GODSDAV-9">[9]</a></sup></p>
<p>영국 요크셔에서 발굴된 기원후 1000년경의 묠니르 펜던트는 <a class="new" href="/w/index.php?title=%EC%96%B8%EC%85%9C_%EC%9E%90%EC%B2%B4&amp;action=edit&amp;redlink=1" title="언셜 자체 (없는 문서)">언셜 자체</a>와 십자가가 새겨져 있는데, 이는 이 펜던트의 기독교도 주인이 게르만 이교와 기독교의 상징물을 융합시켜 사용했음을 보여준다.<sup class="reference" id="cite_ref-10"><a href="#cite_note-10">[10]</a></sup></p>
<p>덴마크 <a class="mw-redirect" href="/wiki/%EC%9C%A0%ED%8B%80%EB%9E%80%EB%93%9C" title="유틀란드">유틀란드</a>에서 발견된 10세기의 동석 거푸집은 십자가를 만들 수도 있고 묠니르를 만들 수도 있다. 이 거푸집을 사용한 대장장이는 기독교도 고객과 이교도 고객을 모두 상대하면서 이문을 올린 것으로 보인다.<sup class="reference" id="cite_ref-11"><a href="#cite_note-11">[11]</a></sup> 아이슬란드 포시에서 발견되어 현재 <a href="/wiki/%EC%95%84%EC%9D%B4%EC%8A%AC%EB%9E%80%EB%93%9C_%EA%B5%AD%EB%A6%BD%EB%B0%95%EB%AC%BC%EA%B4%80" title="아이슬란드 국립박물관">아이슬란드 국립박물관</a>에 소장 중인 은으로 된 유물은 십자가처럼 보이기도 하고 묠니르처럼 보이기도 한다. 특이하게도 이 유물의 가늘고 긴 십자팔 끝은 짐승(아마 늑대로 추측됨)의 머리 모양으로 마감되어 있다.</p>
<p>2014년 덴마크의 롤란드 섬에서 발견된 유물에는 <span lang="non" xml:lang="non">Hmar x is</span> 라는 글자가 새겨져 있다. 이는 “이것은 망치다”라는 뜻으로, 이러한 종류의 펜던트가 망치를 표현한 것임을 정확히 알 수 있게 하는 정보가 담긴 최초의 유물이다.<sup class="reference" id="cite_ref-12"><a href="#cite_note-12">[12]</a></sup> 다만 "망치"의 정확한 철자는 <span lang="non" xml:lang="non">hmar</span> 가 아니라 <span lang="non" xml:lang="non">hamar</span> 이다. 이는 유물의 제작자가 글을 제대로 깨우친 사람이 아니었음을 알 수 있게 한다.<sup class="reference" id="cite_ref-13"><a href="#cite_note-13">[13]</a></sup></p>
<h3><span class="mw-headline" id=".EB.B0.94.EC.9D.B4.ED.82.B9_.EC.8B.9C.EB.8C.80.EC.9D.98_.EB.AA.85.EB.AC.B8">바이킹 시대의 명문</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%AC%A0%EB%8B%88%EB%A5%B4&amp;action=edit&amp;section=8" title="부분 편집: 바이킹 시대의 명문">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>덴마크와 스웨덴 남부에서 발견된 몇 점의 <a class="new" href="/w/index.php?title=%EC%95%94%EA%B0%81%ED%99%94&amp;action=edit&amp;redlink=1" title="암각화 (없는 문서)">암각화</a>들과 <a href="/wiki/%EB%A3%AC%EB%8F%8C" title="룬돌">룬돌</a>들 중에는 망치가 새겨진 것들이 있다. 그 예시로는 <a class="new" href="/w/index.php?title=%EC%95%8C%ED%88%AC%EB%82%98_%EB%A3%AC%EB%8F%8C&amp;action=edit&amp;redlink=1" title="알투나 룬돌 (없는 문서)">U 1161</a>, <a class="new" href="/w/index.php?title=%EC%87%A0%EB%8D%B0%EB%A5%B4%EB%A7%8C%EB%9E%80%EB%93%9C_86%EB%B2%88_%EB%A3%AC%EB%8F%8C&amp;action=edit&amp;redlink=1" title="쇠데르만란드 86번 룬돌 (없는 문서)">Sö 86</a>, <a class="new" href="/w/index.php?title=%EC%8A%A4%ED%85%90%ED%81%AC%EB%B9%84%EC%8A%A4%ED%83%80_%EB%A3%AC%EB%8F%8C&amp;action=edit&amp;redlink=1" title="스텐크비스타 룬돌 (없는 문서)">Sö 111</a>, Sö 140, <a class="new" href="/w/index.php?title=%EB%B1%8C%EB%A5%B4%EB%B9%84_%EB%A3%AC%EB%8F%8C&amp;action=edit&amp;redlink=1" title="뱌르비 룬돌 (없는 문서)">Vg 113</a>, <a class="new" href="/w/index.php?title=%EC%B9%B4%EB%A5%BC%EB%A0%88%EB%B9%84_%EB%A3%AC%EB%8F%8C&amp;action=edit&amp;redlink=1" title="카를레비 룬돌 (없는 문서)">Öl 1</a>, <a class="new" href="/w/index.php?title=%EB%9D%BC%EC%97%90%EB%B3%B4%EB%A5%B4%EA%B7%B8_%EB%A3%AC%EB%8F%8C&amp;action=edit&amp;redlink=1" title="라에보르그 룬돌 (없는 문서)">DR 26</a>, <a class="new" href="/w/index.php?title=%EB%8D%B4%EB%A7%88%ED%81%AC_48%EB%B2%88_%EB%A3%AC_%EB%AA%85%EB%AC%B8&amp;action=edit&amp;redlink=1" title="덴마크 48번 룬 명문 (없는 문서)">DR 48</a>, DR 120, DR 331 등이 있다.<sup class="reference" id="cite_ref-14"><a href="#cite_note-14">[14]</a></sup><sup class="reference" id="cite_ref-Handbook_15-0"><a href="#cite_note-Handbook-15">[15]</a></sup> 망치가 새겨져 있지 않아도 <a href="/wiki/%ED%86%A0%EB%A5%B4" title="토르">토르</a>가 이 돌을 수호해 주실 것을 기원하는 내용이 새겨진 룬돌들도 있다. 덴마크 비링에서 발견된 돌에는 <span lang="non" xml:lang="non"><i>þur uiki þisi kuml</i></span> <small>투르 여이키 티시 쿠믈</small>이라는 명문이 새겨져 있는데, 이는 토르에게 이 기념물을 바친다는 뜻이다. 유사한 내용의 명문이 새겨진 돌이 여러 개 있는데, 모두 토르에게 돌을 바친다거나 토르가 어떤 특정한 물건을 수호해 주시기를 기원하는 내용이 새겨져 있다. 이러한 명문들은 하나님이 망자를 보호해 주실 것을 기원하는 기독교도들에 대한 대응의 차원에서 제작된 것들로 보인다.<sup class="reference" id="cite_ref-16"><a href="#cite_note-16">[16]</a></sup></p>
