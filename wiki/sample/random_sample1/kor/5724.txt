<h1 class="firstHeading" id="firstHeading" lang="ko">무깟디마</h1>
<div class="mw-body-content" id="bodyContent">
<div id="siteSub">위키백과, 우리 모두의 백과사전.</div>
<div id="contentSub"></div>
<div class="mw-jump" id="jump-to-nav">
					이동:					<a href="#mw-head">둘러보기</a>, 					<a href="#p-search">검색</a>
</div>
<div class="mw-content-ltr" dir="ltr" id="mw-content-text" lang="ko"><script>function mfTempOpenSection(id){var block=document.getElementById("mf-section-"+id);block.className+=" open-block";block.previousSibling.className+=" open-block";}</script><p><b>무깟디마</b>(<span style="font-size: smaller;"><a href="/wiki/%EC%95%84%EB%9E%8D%EC%96%B4" title="아랍어">아랍어</a>:</span> <span lang="ar" xml:lang="ar">مقدّمة</span>)는 <a class="mw-disambig" href="/wiki/%EC%95%84%EB%9E%8D" title="아랍">아랍</a>, <a class="mw-redirect" href="/wiki/%EB%B2%A0%EB%A5%B4%EB%B2%A0%EB%A5%B4%EC%A1%B1" title="베르베르족">베르베르족</a> 출신 역사가 <a class="mw-redirect" href="/wiki/%EC%9D%B4%EB%B8%90_%EC%B9%BC%EB%91%94" title="이븐 칼둔">이븐 칼둔</a>이 1377년에 저술한 책이다. <b>이븐 칼둔의 서문</b>으로도 알려져있다. 이 책은 초기 인류사에 관해 기록하였다.</p>
<p>몇몇 현대 사상가들은 이 책을 <a href="/wiki/%EC%97%AD%EC%82%AC%EC%B2%A0%ED%95%99" title="역사철학">역사철학</a>, <a href="/wiki/%EC%82%AC%ED%9A%8C%EA%B3%BC%ED%95%99" title="사회과학">사회과학</a>,<sup class="reference" id="cite_ref-1"><a href="#cite_note-1">[1]</a></sup> <a href="/wiki/%EC%82%AC%ED%9A%8C%ED%95%99" title="사회학">사회학</a>,<sup class="reference" id="cite_ref-Mowlana_2-0"><a href="#cite_note-Mowlana-2">[2]</a></sup><sup class="reference" id="cite_ref-Gates_3-0"><a href="#cite_note-Gates-3">[3]</a></sup><sup class="reference" id="cite_ref-4"><a href="#cite_note-4">[4]</a></sup> <a href="/wiki/%EC%9D%B8%EA%B5%AC%ED%86%B5%EA%B3%84%ED%95%99" title="인구통계학">인구통계학</a>,<sup class="reference" id="cite_ref-Mowlana_2-1"><a href="#cite_note-Mowlana-2">[2]</a></sup> <a href="/wiki/%EC%97%AD%EC%82%AC%ED%95%99" title="역사학">역사학</a>,<sup class="reference" id="cite_ref-Gates_3-1"><a href="#cite_note-Gates-3">[3]</a></sup><sup class="reference" id="cite_ref-Salahuddin_Ahmed_1999_5-0"><a href="#cite_note-Salahuddin_Ahmed_1999-5">[5]</a></sup> <a href="/wiki/%EB%AC%B8%ED%99%94%EC%82%AC" title="문화사">문화사</a>,<sup class="reference" id="cite_ref-Mohamad_Abdalla_2007_p._61-70_6-0"><a href="#cite_note-Mohamad_Abdalla_2007_p._61-70-6">[6]</a></sup><sup class="reference" id="cite_ref-7"><a href="#cite_note-7">[7]</a></sup> 사회 다원설,<sup class="reference" id="cite_ref-8"><a href="#cite_note-8">[8]</a></sup> 생태학,<sup class="reference" id="cite_ref-9"><a href="#cite_note-9">[9]</a></sup> <a class="new" href="/w/index.php?title=%EB%8B%A4%EC%9B%90%EC%84%A4&amp;action=edit&amp;redlink=1" title="다원설 (없는 문서)">다원설</a>, <a href="/wiki/%EA%B2%BD%EC%A0%9C%ED%95%99" title="경제학">경제학</a><sup class="reference" id="cite_ref-Oweiss_10-0"><a href="#cite_note-Oweiss-10">[10]</a></sup><sup class="reference" id="cite_ref-Jean_David_C_1971_11-0"><a href="#cite_note-Jean_David_C_1971-11">[11]</a></sup>을 다루는 최초의 서적으로도 본다. 또 무깟디마는 이슬람 신학, <a href="/wiki/%EC%A0%95%EC%B9%98%EC%B2%A0%ED%95%99" title="정치철학">정치철학</a> 그리고 <a href="/wiki/%EC%83%9D%EB%AC%BC%ED%95%99" title="생물학">생물학</a>, <a href="/wiki/%ED%99%94%ED%95%99" title="화학">화학</a> 영역도 다루고 있다.</p>
<p>무깟디마는 사전적으로 '머리글', '서문'을 뜻한다. 1377년, 이븐 칼둔은 《충고의 서》라는 세계사에 관한 책 (총 7권)의 서문으로 무깟디마를 썼다.</p>
