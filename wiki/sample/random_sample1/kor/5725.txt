<p><b>무드로스 정전 협정</b>(<span style="font-size: smaller;"><a href="/wiki/%ED%84%B0%ED%82%A4%EC%96%B4" title="터키어">터키어</a>:</span> <span lang="tr" xml:lang="tr">Mondros Ateşkes Anlaşması</span>, 1918년 10월 30일)은 <a href="/wiki/%EC%A0%9C1%EC%B0%A8_%EC%84%B8%EA%B3%84_%EB%8C%80%EC%A0%84" title="제1차 세계 대전">제1차 세계 대전</a> 막바지에 <a class="mw-redirect" href="/wiki/%EC%97%B0%ED%95%A9%EA%B5%AD_(%EC%A0%9C1%EC%B0%A8_%EC%84%B8%EA%B3%84_%EB%8C%80%EC%A0%84)" title="연합국 (제1차 세계 대전)">연합군</a>과 <a href="/wiki/%EC%98%A4%EC%8A%A4%EB%A7%8C_%EC%A0%9C%EA%B5%AD" title="오스만 제국">오스만 제국</a>이 <a class="mw-redirect" href="/wiki/%EC%A4%91%EB%8F%99_%EC%A0%84%EC%84%A0_(%EC%A0%9C1%EC%B0%A8_%EC%84%B8%EA%B3%84_%EB%8C%80%EC%A0%84)" title="중동 전선 (제1차 세계 대전)">중동 전선</a>에서 양측의 적대 행위를 종식하도록 합의한 정전 협정이다. 오스만 제국 해상 장관 <a class="new" href="/w/index.php?title=%EB%9D%BC%EC%9A%B0%ED%94%84_%EB%B2%A0%EC%9D%B4&amp;action=edit&amp;redlink=1" title="라우프 베이 (없는 문서)">라우프 베이</a>와 영국 <a class="new" href="/w/index.php?title=%EC%84%9C%EB%A8%B8%EC%85%8B_%EC%95%84%EC%84%9C_%EA%B3%A0-%EC%B9%BC%ED%84%B0%ED%8E%98&amp;action=edit&amp;redlink=1" title="서머셋 아서 고-칼터페 (없는 문서)">서머셋 아서 고-칼터페</a> 제독이 그리스 <a href="/wiki/%EB%A6%BC%EB%85%B8%EC%8A%A4_%EC%84%AC" title="림노스 섬">림노스 섬</a>의 <a class="new" href="/w/index.php?title=%EB%AC%B4%EB%93%9C%EB%A1%9C%EC%8A%A4&amp;action=edit&amp;redlink=1" title="무드로스 (없는 문서)">무드로스</a> 항구에 정박한 <a class="new" href="/w/index.php?title=HMS_%EC%95%84%EA%B0%80%EB%A9%A4%EB%85%BC_%ED%98%B8_(1906%EB%85%84)&amp;action=edit&amp;redlink=1" title="HMS 아가멤논 호 (1906년) (없는 문서)">HMS 아가멤논 호</a> 선상에서 협정 문서에 서명하였다.<sup class="reference" id="cite_ref-1"><a href="#cite_note-1">[1]</a></sup></p>
<p>당시 <a href="/wiki/%EC%98%A4%EC%8A%A4%EB%A7%8C_%EC%A0%9C%EA%B5%AD" title="오스만 제국">오스만 제국</a>은 <a class="mw-redirect" href="/wiki/%EC%A0%9C1%EC%B0%A8_%EC%84%B8%EA%B3%84%EB%8C%80%EC%A0%84" title="제1차 세계대전">제1차 세계대전</a> 에서 크게 패하고 연합군이 진격해오면서 더 이상 버티지 못하고 정전 협정 체결을 결정하게 된 것이다. 이 협정으로 오스만 제국은 <a href="/wiki/%ED%97%A4%EC%9E%90%EC%A6%88" title="헤자즈">헤자즈</a>·<a href="/wiki/%EC%98%88%EB%A9%98" title="예멘">예멘</a>·<a href="/wiki/%EC%8B%9C%EB%A6%AC%EC%95%84" title="시리아">시리아</a>·<a href="/wiki/%EB%A9%94%EC%86%8C%ED%8F%AC%ED%83%80%EB%AF%B8%EC%95%84" title="메소포타미아">메소포타미아</a>·<a href="/wiki/%ED%8A%B8%EB%A6%AC%ED%8F%B4%EB%A6%AC%ED%83%80%EB%8B%88%EC%95%84" title="트리폴리타니아">트리폴리타니아</a>·<a href="/wiki/%ED%82%A4%EB%A0%88%EB%82%98%EC%9D%B4%EC%B9%B4" title="키레나이카">키레나이카</a>를 양도했고 연합국은 <a href="/wiki/%EB%8B%A4%EB%A5%B4%EB%8B%A4%EB%84%AC%EC%8A%A4_%ED%95%B4%ED%98%91" title="다르다넬스 해협">다르다넬스 해협</a>, <a class="mw-redirect" href="/wiki/%EB%B3%B4%EC%8A%A4%ED%8F%AC%EB%A1%9C%EC%8A%A4_%ED%95%B4%ED%98%91" title="보스포로스 해협">보스포로스 해협</a>, <a class="new" href="/w/index.php?title=%EB%B0%94%ED%88%BC&amp;action=edit&amp;redlink=1" title="바툼 (없는 문서)">바툼</a>(지금의 <a href="/wiki/%EC%A1%B0%EC%A7%80%EC%95%84_(%EA%B5%AD%EA%B0%80)" title="조지아 (국가)">조지아</a> 남서부에 있음)과 <a class="new" href="/w/index.php?title=%ED%86%A0%EB%A1%9C%EC%8A%A4&amp;action=edit&amp;redlink=1" title="토로스 (없는 문서)">토로스</a> 터널을 점령했다. 연합국은 <a href="/wiki/%EC%95%84%EB%82%98%ED%86%A8%EB%A6%AC%EC%95%84" title="아나톨리아">아나톨리아</a>에 있는 6개의 아르메니아인 지역이 '혼란할 때' 이 지역을 점령할 수 있는 권리와 연합국의 안보를 위협하는 경우 '전략적 지점은 어느 곳이나' 빼앗을 수 있는 권리도 얻었다. 오스만 군대는 해산되었고 항구·철도·천연자원은 연합국에 넘어갔다.</p>
<p>정전 협정 이후 <a class="new" href="/w/index.php?title=%EC%9D%B4%EC%8A%A4%ED%83%84%EB%B6%88_%EC%A0%90%EB%A0%B9&amp;action=edit&amp;redlink=1" title="이스탄불 점령 (없는 문서)">이스탄불이 점령</a>되고, <a class="new" href="/w/index.php?title=%EC%98%A4%EC%8A%A4%EB%A7%8C_%EC%A0%9C%EA%B5%AD%EC%9D%98_%EB%B6%84%ED%95%A0&amp;action=edit&amp;redlink=1" title="오스만 제국의 분할 (없는 문서)">오스만 제국이 분할</a>되었다. 1920년 8월 10일에 <a href="/wiki/%EC%84%B8%EB%B8%8C%EB%A5%B4_%EC%A1%B0%EC%95%BD" title="세브르 조약">세브르 조약</a>도 체결되었는데, 이 조약은 <a href="/wiki/%ED%84%B0%ED%82%A4_%EB%8F%85%EB%A6%BD_%EC%A0%84%EC%9F%81" title="터키 독립 전쟁">터키 독립 전쟁</a>이 터지면서 이행되지 못하였다.</p>
