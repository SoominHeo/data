<p><b>무라타 아리사</b> (<span style="font-size: smaller;"><a href="/wiki/%EC%9D%BC%EB%B3%B8%EC%96%B4" title="일본어">일본어</a>:</span> <span lang="Ja" xml:lang="Ja">村田愛里咲</span>, <a href="/wiki/1990%EB%85%84" title="1990년">1990년</a> <a href="/wiki/10%EC%9B%94_17%EC%9D%BC" title="10월 17일">10월 17일</a> ~ )는 <a href="/wiki/%EC%9D%BC%EB%B3%B8" title="일본">일본</a>의 <a href="/wiki/%ED%94%84%EB%A6%AC%EC%8A%A4%ED%83%80%EC%9D%BC_%EC%8A%A4%ED%82%A4" title="프리스타일 스키">프리스타일 스키</a> 선수로 주 종목은 모굴이다. <a class="mw-redirect" href="/wiki/2010_%EB%B0%B4%EC%BF%A0%EB%B2%84_%EC%98%AC%EB%A6%BC%ED%94%BD" title="2010 밴쿠버 올림픽">2010 밴쿠버 올림픽</a>과 <a class="mw-redirect" href="/wiki/2014_%EC%86%8C%EC%B9%98_%EC%98%AC%EB%A6%BC%ED%94%BD" title="2014 소치 올림픽">2014 소치 올림픽</a>에 일본 국가대표로 참가하였다</p>
<h2><span class="mw-headline" id=".EC.9D.BC.EC.83.9D">일생</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%AC%B4%EB%9D%BC%ED%83%80_%EC%95%84%EB%A6%AC%EC%82%AC&amp;action=edit&amp;section=1" title="부분 편집: 일생">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p><a href="/wiki/%ED%9B%84%EC%BF%A0%EC%98%A4%EC%B9%B4_%ED%98%84" title="후쿠오카 현">후쿠오카 현</a> <a href="/wiki/%EA%B8%B0%ED%83%80%ED%81%90%EC%8A%88_%EC%8B%9C" title="기타큐슈 시">기타큐슈 시</a>의 교가쿠 유치원(行学幼稚園)을 운영하는 집안에서 태어났다.<sup class="reference" id="cite_ref-1"><a href="#cite_note-1">[1]</a></sup> 삼촌의 권유로 스키에 입문했으며, 초등학교 6학년 때 <a class="new" href="/w/index.php?title=%EB%8D%A4%EB%B8%94%EB%A7%81_%ED%81%B4%EB%9F%BD_%EC%8A%A4%ED%8E%98%EC%9D%B4%EC%8A%A4_%EC%9B%8C%ED%81%AC&amp;action=edit&amp;redlink=1" title="덤블링 클럽 스페이스 워크 (없는 문서)">덤블링 클럽 스페이스 워크</a>에 입단하여 후쿠오카 현 주니어 알파인 스키 선수권대회에서 입상했으며, 당시 적수로 현재 권투 선수로 활동하고 있는 <a class="new" href="/w/index.php?title=%EA%B5%AC%EB%A1%9C%ED%82%A4_%EC%9C%A0%EC%BD%94&amp;action=edit&amp;redlink=1" title="구로키 유코 (없는 문서)">구로키 유코</a>가 있다. 오빠의 영향으로 중학교 1학년 때 <a href="/wiki/%EB%AA%A8%EA%B5%B4%EC%8A%A4%ED%82%A4" title="모굴스키">모굴스키</a>로 전향하여 덤블링 및 공중 동작을 훈련했다. 중학교 2학년 때인 2005년 12월에 모굴 스키 일본 국가대표로 발탁되어, <a href="/wiki/%EB%AF%B8%EA%B5%AD" title="미국">미국</a> <a class="mw-disambig" href="/wiki/%ED%8C%8C%ED%81%AC%EC%8B%9C%ED%8B%B0" title="파크시티">파크시티</a>에서 열린 <a class="new" href="/w/index.php?title=%EB%B6%81%EC%95%84%EB%A9%94%EB%A6%AC%EC%B9%B4_%EC%BB%B5&amp;action=edit&amp;redlink=1" title="북아메리카 컵 (없는 문서)">북아메리카 컵</a>에 참가하였으며, 모굴 종목에서 35위, 듀얼 모굴 종목에서 23위를 기록했다.</p>
<p>이후 2006년에 <a href="/wiki/%ED%9B%84%EC%BF%A0%EC%8B%9C%EB%A7%88_%ED%98%84" title="후쿠시마 현">후쿠시마 현</a> <a href="/wiki/%EC%95%BC%EB%A7%88_%EA%B5%B0" title="야마 군">야마 군</a>의 <a href="/wiki/%EC%9D%B4%EB%82%98%EC%99%80%EC%8B%9C%EB%A1%9C_%EC%A0%95" title="이나와시로 정">이나와시로 정</a>에서 열린 <a class="new" href="/w/index.php?title=%EC%A0%84%EC%9D%BC%EB%B3%B8_%EB%AA%A8%EA%B5%B4_%EC%8A%A4%ED%82%A4_%EC%84%A0%EC%88%98%EA%B6%8C%EB%8C%80%ED%9A%8C&amp;action=edit&amp;redlink=1" title="전일본 모굴 스키 선수권대회 (없는 문서)">일본 선수권대회</a>에 참가하였으며, 모굴 종목에서 10위, 듀얼 모굴 종목에서 6위를 기록했다. 같은 해에 기타큐슈 시립 도호쿠 중학교를 졸업하고 모굴 스키에 전념할 수 있는 환경을 위해 <a href="/wiki/%ED%99%8B%EC%B9%B4%EC%9D%B4%EB%8F%84" title="홋카이도">홋카이도</a> <a href="/wiki/%EC%82%BF%ED%8F%AC%EB%A1%9C_%EC%8B%9C" title="삿포로 시">삿포로 시</a>의 <a class="new" href="/w/index.php?title=%ED%99%8B%EC%B9%B4%EC%9D%B4%EB%8F%84_%EC%87%BC%EC%A7%80_%EA%B0%80%EC%BF%A0%EC%97%94_%EA%B3%A0%EB%93%B1%ED%95%99%EA%B5%90&amp;action=edit&amp;redlink=1" title="홋카이도 쇼지 가쿠엔 고등학교 (없는 문서)">홋카이도 쇼지 가쿠엔 고등학교</a>로 진학했다. 이 해 1월에 다시 국가대표로 발탁하여 <a href="/wiki/%EC%BA%90%EB%82%98%EB%8B%A4" title="캐나다">캐나다</a> <a href="/wiki/%ED%80%98%EB%B2%A1_%EC%A3%BC" title="퀘벡 주">퀘벡 주</a>의 <a class="new" href="/w/index.php?title=%EB%AA%BD%EA%B0%80%EB%B8%8C%EB%A6%AC%EC%97%98&amp;action=edit&amp;redlink=1" title="몽가브리엘 (없는 문서)">몽가브리엘</a>에서 열린 북아메리카 컵에 참가하여 모굴 부문 35위, 듀얼 모굴 부문 23위를 기록했다. 2월에는 캐나다 퀘벡 주 <a class="new" href="/w/index.php?title=%EB%B0%9C%EC%83%9D%EC%BD%A4&amp;action=edit&amp;redlink=1" title="발생콤 (없는 문서)">발생콤</a>에서 열린 북아메리카 컵에 참가하여 모굴 13위, 듀멀 모굴 17위를 기록했다. 이후 <a href="/wiki/%EB%8F%84%EC%95%BC%EB%A7%88_%EC%8B%9C" title="도야마 시">도야마 시</a> <a class="new" href="/w/index.php?title=%EC%95%BC%EC%8A%A4%EC%98%A4_%EC%A0%95&amp;action=edit&amp;redlink=1" title="야스오 정 (없는 문서)">야스오 정</a>의 <a class="new" href="/w/index.php?title=%EB%8B%A4%ED%85%8C%EC%95%BC%EB%B0%94%EC%82%B0%EB%A1%9C%EC%BF%A0&amp;action=edit&amp;redlink=1" title="다테야바산로쿠 (없는 문서)">다테야바산로쿠</a>에서 열린 2007 일본 선수권대회 모굴 종목에서 <a class="new" href="/w/index.php?title=%EC%9A%B0%EC%97%90%EB%AC%B4%EB%9D%BC_%EC%95%84%EC%9D%B4%EC%BD%94&amp;action=edit&amp;redlink=1" title="우에무라 아이코 (없는 문서)">우에무라 아이코</a>와 <a class="new" href="/w/index.php?title=%EC%9D%B4%ED%86%A0_%EB%AF%B8%ED%82%A4_(%EC%8A%A4%ED%82%A4_%EC%84%A0%EC%88%98)&amp;action=edit&amp;redlink=1" title="이토 미키 (스키 선수) (없는 문서)">이토 미키</a>의 뒤를 이어 동메달을 획득했다. 2008년 1월에는 캐나다 <a href="/wiki/%EC%BA%98%EA%B1%B0%EB%A6%AC" title="캘거리">캘거리</a>의 <a class="new" href="/w/index.php?title=%EC%BA%90%EB%82%98%EB%8B%A4_%EC%98%AC%EB%A6%BC%ED%94%BD_%ED%8C%8C%ED%81%AC&amp;action=edit&amp;redlink=1" title="캐나다 올림픽 파크 (없는 문서)">캐나다 올림픽 파크</a>에서 열린 북아메리카 컵에 일본 대표로 참가하여 모굴 부문 16위, 듀얼 모굴 부문 12위를 기록하였으며, 발생콤 북아메리카 컵에서는 모굴 9위, 듀얼 모굴 12위에 올랐다.</p>
<p>2월 16일에는 자신의 첫 월드컵 대회인 이나와시로 월드컵에 참가하여 모굴 종목 19위를 기록했다. 이후 8월에 <a href="/wiki/%EC%98%A4%EC%8A%A4%ED%8A%B8%EB%A0%88%EC%9D%BC%EB%A6%AC%EC%95%84" title="오스트레일리아">오스트레일리아</a> <a href="/wiki/%EB%89%B4%EC%82%AC%EC%9A%B0%EC%8A%A4%EC%9B%A8%EC%9D%BC%EC%8A%A4_%EC%A3%BC" title="뉴사우스웨일스 주">뉴사우스웨일스 주</a>의 <a class="new" href="/w/index.php?title=%ED%8E%98%EB%A6%AC%EC%85%94_%EC%8A%A4%ED%82%A4_%EB%A6%AC%EC%A1%B0%ED%8A%B8&amp;action=edit&amp;redlink=1" title="페리셔 스키 리조트 (없는 문서)">페리셔 스키 리조트</a>에서 열린 <a class="new" href="/w/index.php?title=%EC%98%A4%EC%8A%A4%ED%8A%B8%EB%A0%88%EC%9D%BC%EB%A6%AC%EC%95%84-%EB%89%B4%EC%A7%88%EB%9E%9C%EB%93%9C_%EC%BB%B5&amp;action=edit&amp;redlink=1" title="오스트레일리아-뉴질랜드 컵 (없는 문서)">오스트레일리아-뉴질랜드 컵</a>에 참가하여 21일에 열린 모굴 경기에서 일본의 우에무라와 오스트레일리아의 <a class="new" href="/w/index.php?title=%EC%97%90%EB%A7%88_%EC%B1%84%ED%94%84%EB%A8%BC&amp;action=edit&amp;redlink=1" title="에마 채프먼 (없는 문서)">에마 채프먼</a>, 체코의 <a class="new" href="/w/index.php?title=%EB%8B%88%EC%BD%9C%EB%9D%BC_%EC%88%98%EB%8F%84%EB%B0%94&amp;action=edit&amp;redlink=1" title="니콜라 수도바 (없는 문서)">니콜라 수도바</a>의 뒤를 이어 4위를 기록했고, 22일의 경기에서는 채프먼과 우에무라의 뒤를 이어 3위를 기록했다. 이후 <a href="/wiki/%ED%94%84%EB%9E%91%EC%8A%A4" title="프랑스">프랑스</a>, 캐나다, 미국, <a href="/wiki/%EC%8A%A4%EC%9B%A8%EB%8D%B4" title="스웨덴">스웨덴</a>, <a href="/wiki/%EB%85%B8%EB%A5%B4%EC%9B%A8%EC%9D%B4" title="노르웨이">노르웨이</a>에서 열린 월드컵에 참가하였으며, 2009년에 고등학교를 졸업하고 <a href="/wiki/%ED%98%B8%EC%BF%A0%EC%87%BC_%EB%8C%80%ED%95%99" title="호쿠쇼 대학">호쿠쇼 대학</a> 평생교육시스템 학부 교육지도학과에 입학했으며, 이 대학교의 스키부에 입단했다.</p>
<p>2009년 12월에는 핀란드 <a class="new" href="/w/index.php?title=%EC%BC%80%EB%AF%B8%EC%95%A0%EB%A5%B4%EB%B9%84&amp;action=edit&amp;redlink=1" title="케미애르비 (없는 문서)">케미애르비</a>의 <a class="new" href="/w/index.php?title=%EC%88%98%EC%98%A4%EB%AC%B4_%EC%8A%A4%ED%82%A4_%EB%A6%AC%EC%A1%B0%ED%8A%B8&amp;action=edit&amp;redlink=1" title="수오무 스키 리조트 (없는 문서)">수오무 스키 리조트</a>에서 열린 <a class="new" href="/w/index.php?title=%EC%9C%A0%EB%A1%9C%ED%8C%8C_%EC%BB%B5&amp;action=edit&amp;redlink=1" title="유로파 컵 (없는 문서)">유로파 컵</a>에 참가하였으며,2010년 2월에는 캐나다 <a href="/wiki/%EB%B0%B4%EC%BF%A0%EB%B2%84" title="밴쿠버">밴쿠버</a>에서 열린 <a class="mw-redirect" href="/wiki/2010_%EB%8F%99%EA%B3%84_%EC%98%AC%EB%A6%BC%ED%94%BD" title="2010 동계 올림픽">2010 동계 올림픽</a>에 일본 국가대표로 발탁되어 <a href="/wiki/%EA%B7%9C%EC%8A%88" title="규슈">규슈</a> 지방 출신 최초로 올림픽 스키 종목 국가대표가 되었다. 무라타는 여자 모굴 종목에서 공중 묘기에서 높은 평가를 받으며 8위를 기록했다. 3월에는 <a href="/wiki/%EC%8A%A4%ED%8E%98%EC%9D%B8" title="스페인">스페인</a> <a class="mw-redirect mw-disambig" href="/wiki/%EC%8B%9C%EC%97%90%EB%9D%BC_%EB%84%A4%EB%B0%94%EB%8B%A4" title="시에라 네바다">시에라 네바다</a>에서 열린 월드컵에서 7위를 기록했고, 2011년 2월에는 미국 <a class="new" href="/w/index.php?title=%EB%94%94%EC%96%B4%EB%B0%B8%EB%A6%AC&amp;action=edit&amp;redlink=1" title="디어밸리 (없는 문서)">디어밸리</a>에서 열린 <a class="new" href="/w/index.php?title=2011%EB%85%84_%EC%84%B8%EA%B3%84_%ED%94%84%EB%A6%AC%EC%8A%A4%ED%83%80%EC%9D%BC_%EC%8A%A4%ED%82%A4_%EC%84%A0%EC%88%98%EA%B6%8C%EB%8C%80%ED%9A%8C&amp;action=edit&amp;redlink=1" title="2011년 세계 프리스타일 스키 선수권대회 (없는 문서)">세계 프리스타일 스키 선수권대회</a>에 일본 국가대표로 참가하여 모굴 종목 12위, 듀얼 모굴 종목 13위를 기록했다. 이후 <a href="/wiki/%EC%B2%B4%EC%BD%94" title="체코">체코</a> <a href="/wiki/%EB%A7%88%EB%A6%AC%EC%95%88%EC%8A%A4%EC%BC%80%EB%9D%BC%EC%A6%88%EB%84%A4" title="마리안스케라즈네">마리안스케라즈네</a>에서 열린 월드컵에서 듀얼 모굴 종목 10위를 기록했다. 8월에는 페리셔 리조트에서 열린 오스트레일리아-뉴질랜드 컵 모굴 부문에서 오스트레일리아의 <a class="new" href="/w/index.php?title=%EB%B8%8C%EB%A6%AC%ED%8A%B8%EB%8B%88_%EC%BD%95%EC%8A%A4_(%ED%94%84%EB%A6%AC%EC%8A%A4%ED%83%80%EC%9D%BC_%EC%8A%A4%ED%82%A4_%EC%84%A0%EC%88%98)&amp;action=edit&amp;redlink=1" title="브리트니 콕스 (프리스타일 스키 선수) (없는 문서)">브리트니 콕스</a>와 일본의 이토 미키를 누르고 금메달을 획득했다.</p>
<p>2011년 12월에는 프랑스 <a class="mw-redirect" href="/wiki/%EB%A9%94%EB%A6%AC%EB%B2%A8" title="메리벨">메리벨</a>에서 열린 월드컵에서 듀얼 모굴 8위에 올랐다. 2012년 2월에는 <a href="/wiki/%EB%8B%88%EA%B0%80%ED%83%80_%ED%98%84" title="니가타 현">니가타 현</a> <a href="/wiki/%EB%AF%B8%EB%82%98%EB%AF%B8%EC%9A%B0%EC%98%A4%EB%88%84%EB%A7%88_%EA%B5%B0" title="미나미우오누마 군">미나미우오누마 군</a>의 <a href="/wiki/%EC%9C%A0%EC%9E%90%EC%99%80_%EC%A0%95" title="유자와 정">유자와 정</a>에서 열린 월드컵에서 모굴 6위, 듀얼 모굴 8위를 기록하였다. 이후 3월 5일에 <a href="/wiki/%EB%9F%AC%EC%8B%9C%EC%95%84" title="러시아">러시아</a> <a href="/wiki/%EC%86%8C%EC%B9%98" title="소치">소치</a>에서 열린 유로파 컵에서 이토 미키와 러시아의 <a class="new" href="/w/index.php?title=%EC%9D%B4%EB%A6%AC%EB%82%98_%ED%94%BC%EC%82%AC%EB%A0%99%EC%8A%A4%EC%B9%B4%EC%95%BC&amp;action=edit&amp;redlink=1" title="이리나 피사렙스카야 (없는 문서)">이리나 피사렙스카야</a>를 누르고 우승했으며, 3월 9일에 스웨덴 <a class="mw-disambig" href="/wiki/%EC%98%A4%EB%A0%88" title="오레">오레</a>에서 열린 월드컵에서 미국의 해나 키어니와 캐나다의 클로에 뒤푸르라푸앙트의 뒤를 이어 3위를 하여 동메달을 획득했다. 이는 무라타가 획득한 최초의 월드컵 메달이다. 이후 2013 <a class="mw-redirect mw-disambig" href="/wiki/%EB%A0%88%EC%9D%B4%ED%81%AC%ED%94%8C%EB%A0%88%EC%9D%B4%EC%8B%9C%EB%93%9C" title="레이크플레이시드">레이크플레이시드</a> 월드컵 9위, 디어밸리 월드컵에서 5위를 하는 등 월드컵에서 활동했고, 2013년 3월에는 노르웨이 <a class="mw-disambig" href="/wiki/%EB%B3%B4%EC%8A%A4" title="보스">보스</a>에서 열린 <a class="new" href="/w/index.php?title=2013%EB%85%84_%EC%84%B8%EA%B3%84_%ED%94%84%EB%A6%AC%EC%8A%A4%ED%83%80%EC%9D%BC_%EC%8A%A4%ED%82%A4_%EC%84%A0%EC%88%98%EA%B6%8C%EB%8C%80%ED%9A%8C&amp;action=edit&amp;redlink=1" title="2013년 세계 프리스타일 스키 선수권대회 (없는 문서)">세계 선수권대회</a>에 참가하여 모굴 종목 6위, 듀얼 모굴 종목 5위에 올랐다. 이후 러시아 소치에서 열린 <a class="mw-redirect" href="/wiki/2014_%EB%8F%99%EA%B3%84_%EC%98%AC%EB%A6%BC%ED%94%BD" title="2014 동계 올림픽">2014 동계 올림픽</a>에도 일본 국가대표로 발탁되어 1차 예선에서 22위를 기록하였고, 2차 예선에서 5위를 기록하여 결선에 진출했으나 기권했다.</p>
