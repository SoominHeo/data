<p><b>무라트 2세</b>(<span style="font-size: smaller;"><a href="/wiki/%EC%98%A4%EC%8A%A4%EB%A7%8C_%ED%84%B0%ED%82%A4%EC%96%B4" title="오스만 터키어">오스만 터키어</a>:</span> <span lang="ota" xml:lang="ota">مراد ثانى</span>, <span style="font-size: smaller;"><a href="/wiki/%ED%84%B0%ED%82%A4%EC%96%B4" title="터키어">터키어</a>:</span> <span lang="tr" xml:lang="tr">II. Murat</span>, <a href="/wiki/1404%EB%85%84" title="1404년">1404년</a> - <a href="/wiki/1451%EB%85%84" title="1451년">1451년</a> <a href="/wiki/2%EC%9B%94_3%EC%9D%BC" title="2월 3일">2월 3일</a>)는 <a href="/wiki/%EC%98%A4%EC%8A%A4%EB%A7%8C_%EC%A0%9C%EA%B5%AD" title="오스만 제국">오스만 제국</a>의 제6대 <a href="/wiki/%EC%88%A0%ED%83%84" title="술탄">술탄</a>이다(재위 <a href="/wiki/1421%EB%85%84" title="1421년">1421년</a> - <a href="/wiki/1444%EB%85%84" title="1444년">1444년</a>).(<a href="/wiki/1446%EB%85%84" title="1446년">1446년</a> - <a href="/wiki/1451%EB%85%84" title="1451년">1451년</a>)중간에 2년(<a href="/wiki/1444%EB%85%84" title="1444년">1444년</a> - <a href="/wiki/1446%EB%85%84" title="1446년">1446년</a>)은 그의 아들 <a class="mw-redirect" href="/wiki/%EB%A9%94%ED%9D%90%EB%A9%94%EB%93%9C_2%EC%84%B8" title="메흐메드 2세">메흐메드 2세</a>에게 권좌를 내주고 은퇴생활을 했었다. 무라트 2세의 통치기간은 <a href="/wiki/%EB%B0%9C%EC%B9%B8_%EB%B0%98%EB%8F%84" title="발칸 반도">발칸 반도</a>의 <a href="/wiki/%EA%B8%B0%EB%8F%85%EA%B5%90" title="기독교">기독교</a> 세력과 <a href="/wiki/%EC%95%84%EB%82%98%ED%86%A8%EB%A6%AC%EC%95%84" title="아나톨리아">아나톨리아</a>의 오랜 전쟁과 튀르크 <a href="/wiki/%EC%97%B0%ED%95%A9%EA%B5%AD" title="연합국">연합국</a>과의 25년에 걸친 투쟁으로 점철되어 있다.</p>
<h2><span class="mw-headline" id=".EC.83.9D.EC.95.A0">생애</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%AC%B4%EB%9D%BC%ED%8A%B8_2%EC%84%B8&amp;action=edit&amp;section=1" title="부분 편집: 생애">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p><a href="/wiki/1421%EB%85%84" title="1421년">1421년</a> 아버지 <a href="/wiki/%EB%A9%94%ED%9D%90%EB%A9%94%ED%8A%B8_1%EC%84%B8" title="메흐메트 1세">메흐메트 1세</a>가 사망하자 술탄에 즉위했다. 그의 아버지는 생전에 술탄의 지위를 놓고 다른 형제들과 다툼을 벌이고 있었을 때에 <a class="mw-redirect" href="/wiki/%EB%8F%99%EB%A1%9C%EB%A7%88_%EC%A0%9C%EA%B5%AD" title="동로마 제국">동로마 제국</a>의 원조를 얻었다. 이때 동로마 제국과 오스만 제국은 동맹관계가 되었다.</p>
<p>하지만 무라트 2세의 통치 초기에 <a class="mw-redirect" href="/wiki/%EB%B9%84%EC%9E%94%ED%8B%B0%EC%9B%80_%ED%99%A9%EC%A0%9C_%EC%97%B0%EB%8C%80%ED%91%9C" title="비잔티움 황제 연대표">비잔티움 황제</a> <a href="/wiki/%EB%A7%88%EB%88%84%EC%97%98_2%EC%84%B8_%ED%8C%94%EB%9D%BC%EC%9D%B4%EC%98%AC%EB%A1%9C%EA%B3%A0%EC%8A%A4" title="마누엘 2세 팔라이올로고스">마누엘 2세 팔라이올로고스</a>의 아들 <a href="/wiki/%EC%9A%94%ED%95%9C%EB%84%A4%EC%8A%A4_8%EC%84%B8_%ED%8C%94%EB%9D%BC%EC%9D%B4%EC%98%AC%EB%A1%9C%EA%B3%A0%EC%8A%A4" title="요한네스 8세 팔라이올로고스">요한네스 8세 팔라이올로고스</a>가 <a href="/wiki/%EC%95%99%EC%B9%B4%EB%9D%BC_%EC%A0%84%ED%88%AC" title="앙카라 전투">앙카라 전투</a>에서 죽었다고 생각했던 아버지의 형제 무스타파(이 인물이 진짜인지는 지금까지 불명이다. 오스만 측에서는 꾸며진 인물이라고 판단했다.)를 대립 술탄으로 옹립해 내정간섭을 해왔다.</p>
<p>이에 분노한 무라트 2세는 무스타파를 격파하고 그를 처형한 뒤 <a href="/wiki/%EB%B9%84%EC%9E%94%ED%8B%B0%EC%9B%80_%EC%A0%9C%EA%B5%AD" title="비잔티움 제국">비잔티움 제국</a>과의 동맹을 파기했다. 동로마 제국의 수도 <a href="/wiki/%EC%BD%98%EC%8A%A4%ED%83%84%ED%8B%B0%EB%85%B8%ED%8F%B4%EB%A6%AC%EC%8A%A4" title="콘스탄티노폴리스">콘스탄티노폴리스</a>를 포위한 오스만군에게 이미 약해진 비잔티움군은 적수가 되지 못하였으며, 수도는 함락 위기에 빠졌다. 하지만 마누엘 2세 팔라이올로고스가 아버지의 은인이란 점과 매우 뛰어난 문화인으로써 재능을 가지고 있었기에 오스만 제국에게 유리한 화평조약을 맺고 군을 돌려보냈다. 이 조약으로 이후 동로마 제국은 다시 오스만 제국에게 공물을 보내야만 했다. 일설에는 ‘본국에서 반란이 일어났기 때문에 물러났다’라는 얘기도 있다.</p>
<p><a href="/wiki/1425%EB%85%84" title="1425년">1425년</a>까지 정적들을 거의 제거하고 아나톨리아 서부의 <a class="new" href="/w/index.php?title=%ED%88%AC%EB%A5%B4%ED%81%AC%EB%A9%98_%EA%B3%B5%EA%B5%AD&amp;action=edit&amp;redlink=1" title="투르크멘 공국 (없는 문서)">투르크멘 공국</a>에 대한 오스만 제국의 지배권을 재확립했으며 1430년에는 발칸 반도로 관심을 돌려 5년간의 싸움 끝에 <a href="/wiki/%EB%B2%A0%EB%84%A4%EC%B9%98%EC%95%84_%EA%B3%B5%ED%99%94%EA%B5%AD" title="베네치아 공화국">베네치아 공화국</a>에게 양도되어있던 북부 <a href="/wiki/%EA%B7%B8%EB%A6%AC%EC%8A%A4" title="그리스">그리스</a>의 <a class="mw-redirect" href="/wiki/%ED%85%8C%EC%82%B4%EB%A1%9C%EB%8B%88%EC%B9%B4" title="테살로니카">테살로니카</a>(지금의 <a href="/wiki/%ED%85%8C%EC%82%B4%EB%A1%9C%EB%8B%88%ED%82%A4" title="테살로니키">테살로니키</a>)를 점령했다.</p>
<p>오스만군은 처음에는 <a href="/wiki/%ED%97%9D%EA%B0%80%EB%A6%AC_%EC%99%95%EA%B5%AD" title="헝가리 왕국">헝가리</a>·<a href="/wiki/%EC%84%B8%EB%A5%B4%EB%B9%84%EC%95%84_%EC%99%95%EA%B5%AD" title="세르비아 왕국">세르비아</a>·카라만 연합군에 잘 대항했지만 <a href="/wiki/1441%EB%85%84" title="1441년">1441년</a> 이후 <a class="mw-redirect" href="/wiki/%EC%8B%A0%EC%84%B1%EB%A1%9C%EB%A7%88%EC%A0%9C%EA%B5%AD" title="신성로마제국">신성로마제국</a>·<a href="/wiki/%ED%8F%B4%EB%9E%80%EB%93%9C" title="폴란드">폴란드</a>·<a href="/wiki/%EC%95%8C%EB%B0%94%EB%8B%88%EC%95%84" title="알바니아">알바니아</a>군까지 연합군에 가세하게 되자 오스만 제국은 니시와 소피아를 잃고 잘로와즈에서는 크게 패했다. <a href="/wiki/1444%EB%85%84" title="1444년">1444년</a> <a href="/wiki/6%EC%9B%94_12%EC%9D%BC" title="6월 12일">6월 12일</a> <a href="/wiki/%EC%97%90%EB%94%94%EB%A5%B4%EB%84%A4" title="에디르네">에디르네</a>에서 무라트는 발칸의 제국과 10년간 강화조약에 서명하고, 당시 12세 된 아들 <a class="mw-redirect" href="/wiki/%EB%A9%94%ED%9D%90%EB%A9%94%EB%93%9C_2%EC%84%B8" title="메흐메드 2세">메흐메드 2세</a>에게 술탄의 지위를 양위하고 자신은 아나톨리아로 남부로 돌아가 은둔생활을 하였다.</p>
<h2><span class="mw-headline" id=".EB.B3.B5.EA.B7.80">복귀</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%AC%B4%EB%9D%BC%ED%8A%B8_2%EC%84%B8&amp;action=edit&amp;section=2" title="부분 편집: 복귀">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p><a href="/wiki/%EC%84%9C%EC%9C%A0%EB%9F%BD" title="서유럽">서유럽</a>의 기독교 세력은 <a href="/wiki/%EA%B5%90%ED%99%A9_%EC%97%90%EC%9A%B0%EC%A0%9C%EB%8B%88%EC%98%A4_4%EC%84%B8" title="교황 에우제니오 4세">교황 에우제니오 4세</a>의 지원하에 곧 강화협정을 깨뜨리고 오스만을 침략했다. 아들 메흐메트는 젊고 경험이 미숙했기에 관료들이 따르지 않았고 원정에도 실패했기 때문에 아들의 간곡한 요청에 의해, 무라트는 다시 복귀하여 오스만 군대를 이끌고 <a href="/wiki/1444%EB%85%84" title="1444년">1444년</a> 11월 <a href="/wiki/%EB%B0%94%EB%A5%B4%EB%82%98_%EC%A0%84%ED%88%AC" title="바르나 전투">바르나 전투</a>에서 헝가리-폴란드 기독교 연합군을 무찔렀다.</p>
<h2><span class="mw-headline" id=".EB.91.90_.EB.B2.88.EC.A7.B8_.EC.9E.AC.EC.9C.84.EC.99.80_.EC.A3.BD.EC.9D.8C">두 번째 재위와 죽음</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%AC%B4%EB%9D%BC%ED%8A%B8_2%EC%84%B8&amp;action=edit&amp;section=3" title="부분 편집: 두 번째 재위와 죽음">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p><a href="/wiki/%EB%B0%94%EB%A5%B4%EB%82%98_%EC%A0%84%ED%88%AC" title="바르나 전투">바르나 전투</a> 이후 무라트는 <a href="/wiki/1446%EB%85%84" title="1446년">1446년</a> 다시 오스만 제국의 통치권을 쥐게 되었다.</p>
<p><a href="/wiki/1448%EB%85%84" title="1448년">1448년</a> 10월 17일 <a href="/wiki/%EC%BD%94%EC%86%8C%EB%B3%B4_%EC%A0%84%ED%88%AC_(1448%EB%85%84)" title="코소보 전투 (1448년)">제2차 코소보 전투</a>에서 세르비아를 포함한 기독교 나라들과 싸워 승리를 거두고, 비잔티움 제국을 계속 압박했다. 그러나 무라트 2세는 비잔티움 제국을 멸망시킬 생각은 없었기에 요한네스 8세의 뒤를 이은 <a href="/wiki/%EC%BD%98%EC%8A%A4%ED%83%84%ED%8B%B0%EB%88%84%EC%8A%A4_11%EC%84%B8" title="콘스탄티누스 11세">콘스탄티누스 11세</a>의 즉위를 승인하고 화평조약을 맺었다.</p>
<p><a href="/wiki/1450%EB%85%84" title="1450년">1450년</a>-<a href="/wiki/1451%EB%85%84" title="1451년">1451년</a> 겨울, 무라트는 병으로 쓰러졌고 <a href="/wiki/%EC%97%90%EB%94%94%EB%A5%B4%EB%84%A4" title="에디르네">에디르네</a>에서 죽었다. 그러자 그의 아들 메흐메드 2세가 다시 그의 뒤를 이었다.</p>
