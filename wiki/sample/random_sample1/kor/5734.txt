<h2><span class="mw-headline" id=".EC.83.9D.EC.95.A0">생애</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%AC%B4%EB%9D%BC%ED%8A%B8_3%EC%84%B8&amp;action=edit&amp;section=1" title="부분 편집: 생애">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>무라트 3세는 <a href="/wiki/%EC%88%A0%ED%83%84" title="술탄">술탄</a> <a href="/wiki/%EC%85%80%EB%A6%BC_2%EC%84%B8" title="셀림 2세">셀림 2세</a>와 <a class="new" href="/w/index.php?title=%EB%88%8C%EB%B0%94%EB%88%84_%EC%88%A0%ED%83%84&amp;action=edit&amp;redlink=1" title="눌바누 술탄 (없는 문서)">눌바누 술탄</a>의 사이에서 태어난 장자이다. 그의 어머니인 <a class="new" href="/w/index.php?title=%EB%88%8C%EB%B0%94%EB%88%84_%EC%88%A0%ED%83%84&amp;action=edit&amp;redlink=1" title="눌바누 술탄 (없는 문서)">눌바누 술탄</a>의 원래 이름은 라셀(Raşel)로 <a href="/wiki/%EC%9C%A0%EB%8C%80%EC%9D%B8" title="유대인">유대인</a>으로 알려져 있다. 그녀의 친부모에 관한 자세한 기록은 전해지지 않는다. 다만 <a href="/wiki/%EC%88%A0%ED%83%84" title="술탄">술탄</a>의 <a href="/wiki/%ED%95%98%EB%A0%98" title="하렘">하렘</a>으로 들어온 이후 <a href="/wiki/%EC%9D%B4%EC%8A%AC%EB%9E%8C%EA%B5%90" title="이슬람교">이슬람</a>으로 강제 개종된 것으로 보인다. 혹은 바다를 여행하던 도중에 <a href="/wiki/%ED%84%B0%ED%82%A4" title="터키">터키</a> <a href="/wiki/%ED%95%B4%EC%A0%81" title="해적">해적</a>들에게 납치되어 <a href="/wiki/%EC%88%A0%ED%83%84" title="술탄">술탄</a>의 <a href="/wiki/%ED%95%98%EB%A0%98" title="하렘">하렘</a>에 바쳐진 <a href="/wiki/%EB%B2%A0%EB%84%A4%EC%B9%98%EC%95%84_%EA%B3%B5%ED%99%94%EA%B5%AD" title="베네치아 공화국">베네치아 공화국</a> 여성이라는 이야기도 있다.<sup class="reference" id="cite_ref-1"><a href="#cite_note-1">[1]</a></sup></p>
<p>무라트 3세는 <a href="/wiki/%ED%94%84%EB%9E%91%EC%8A%A4%EC%96%B4" title="프랑스어">프랑스어</a>와 <a href="/wiki/%EC%95%84%EB%9E%8D%EC%96%B4" title="아랍어">아랍어</a>에 대한 교육을 받았다고 전해진다. 또한 그는 1558년에 아버지 <a href="/wiki/%EC%88%A0%ED%83%84" title="술탄">술탄</a> <a href="/wiki/%EC%85%80%EB%A6%BC_2%EC%84%B8" title="셀림 2세">셀림 2세</a>가 <a href="/wiki/%EB%A7%88%EB%8B%88%EC%82%AC" title="마니사">마니사</a> 총독에서 남쪽 <a href="/wiki/%EC%B9%B4%EB%9D%BC%EB%A7%8C" title="카라만">카라만</a>(Karaman)지역의 총독으로 임명되자, 할아버지인 <a href="/wiki/%EC%89%B4%EB%A0%88%EC%9D%B4%EB%A7%8C_1%EC%84%B8" title="쉴레이만 1세">쉴레이만 1세</a>의 곁으로 보내졌다. 이 후 <a href="/wiki/%EC%85%80%EB%A6%BC_2%EC%84%B8" title="셀림 2세">셀림 2세</a>가 <a href="/wiki/%EC%88%A0%ED%83%84" title="술탄">술탄</a>의 자리에 오르자 다시 <a href="/wiki/%EB%A7%88%EB%8B%88%EC%82%AC" title="마니사">마니사</a>로 보내졌다가 <a href="/wiki/%EC%85%80%EB%A6%BC_2%EC%84%B8" title="셀림 2세">셀림 2세</a>가 승하한 1574년 12월 22일에 <a href="/wiki/%EC%9D%B4%EC%8A%A4%ED%83%84%EB%B6%88" title="이스탄불">이스탄불</a>로 돌아와 재위에 올랐다.</p>

<h2><span class="mw-headline" id=".EA.B8.B0.EB.A1.9D">기록</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%AC%B4%EB%9D%BC%ED%8A%B8_3%EC%84%B8&amp;action=edit&amp;section=4" title="부분 편집: 기록">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>무라트 3세의 다음 후계자인 <a class="new" href="/w/index.php?title=%EB%A9%94%EB%A9%94%ED%8A%B8_3%EC%84%B8&amp;action=edit&amp;redlink=1" title="메메트 3세 (없는 문서)">메메트 3세</a>를 낳은 여인은 <a class="new" href="/w/index.php?title=%EC%82%AC%ED%94%BC%EC%98%88_%EC%88%A0%ED%83%84&amp;action=edit&amp;redlink=1" title="사피예 술탄 (없는 문서)">사피예 술탄</a>(Safiye Sultan)이다.<a href="/wiki/%EC%9D%B4%ED%83%88%EB%A6%AC%EC%95%84" title="이탈리아">이탈리아</a> <a href="/wiki/%EB%B2%A0%EB%84%A4%EC%B9%98%EC%95%84_%EA%B3%B5%ED%99%94%EA%B5%AD" title="베네치아 공화국">베네치아 공국</a> 출신으로 알려져있으나 <a class="new" href="/w/index.php?title=%EB%91%90%EC%B9%B4%EA%B8%B0%EB%8B%88_%EA%B3%B5%EA%B5%AD&amp;action=edit&amp;redlink=1" title="두카기니 공국 (없는 문서)">두카기니 공국</a>의 <a href="/wiki/%EC%95%8C%EB%B0%94%EB%8B%88%EC%95%84%EC%9D%B8" title="알바니아인">알바니아인</a>으로 추측된다.<sup class="reference" id="cite_ref-2"><a href="#cite_note-2">[2]</a></sup><sup class="reference" id="cite_ref-3"><a href="#cite_note-3">[3]</a></sup><sup class="reference" id="cite_ref-4"><a href="#cite_note-4">[4]</a></sup><sup class="reference" id="cite_ref-5"><a href="#cite_note-5">[5]</a></sup><sup class="reference" id="cite_ref-6"><a href="#cite_note-6">[6]</a></sup><sup class="reference" id="cite_ref-7"><a href="#cite_note-7">[7]</a></sup><sup class="reference" id="cite_ref-8"><a href="#cite_note-8">[8]</a></sup><sup class="reference" id="cite_ref-9"><a href="#cite_note-9">[9]</a></sup><sup class="reference" id="cite_ref-10"><a href="#cite_note-10">[10]</a></sup><sup class="reference" id="cite_ref-11"><a href="#cite_note-11">[11]</a></sup> 무라트 3세의 자녀는 백여명이 넘는 것으로 알려졌는데, 이 중 19명에서 20명이 아들로 알려졌다. 기록상으로 전해지는 자녀들의 이름은 대략 아래와 같다.</p>
<p><b>남자</b></p>
<p>1. 메메트 3세 2. 셀림 3. 베야짓 4. 무스타파 5. 오스만 6. 지한기르 7. 압둘라 8. 압둘라흐만 9. 압둘라 10. 하산 11. 아흐멧 12. 야쿱 13. 알렘샤 14. 유슙 15. 후세인 16. 콜쿳 17. 알리 18. 이삭 19. 오메르 20. 알라딘 21. 다붓</p>
<p><b>여자</b></p>
<p>1. 아이쉐 2. 파트마 3. 미흐리마 4. 암리예</p>
