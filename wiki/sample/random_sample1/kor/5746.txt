<p><b>무슈 베르두</b> (Monsieur Verdoux) 또는 <b>살인광 시대</b>는 <a href="/wiki/1947%EB%85%84" title="1947년">1947년</a> <a href="/wiki/%EC%B0%B0%EB%A6%AC_%EC%B1%84%ED%94%8C%EB%A6%B0" title="찰리 채플린">찰리 채플린</a>이 제작한 영화이다. 영화는 <a href="/wiki/1930%EB%85%84%EB%8C%80" title="1930년대">1930년대</a> <a href="/wiki/%ED%94%84%EB%9E%91%EC%8A%A4" title="프랑스">프랑스</a>를 배경으로 하고 있으며 채플린은 이미 <a href="/wiki/%EC%9C%84%EB%8C%80%ED%95%9C_%EB%8F%85%EC%9E%AC%EC%9E%90" title="위대한 독재자">위대한 독재자</a>로 <a href="/wiki/%EB%A7%A4%EC%B9%B4%EC%8B%9C%EC%A6%98" title="매카시즘">매카시즘</a>에 휘말리다가 이 영화로 <a href="/wiki/1952%EB%85%84" title="1952년">1952년</a> <a href="/wiki/%EB%AF%B8%EA%B5%AD" title="미국">미국</a>에서 쫓겨나게 된다.</p>
<h2><span class="mw-headline" id=".EB.92.B7.EC.9D.B4.EC.95.BC.EA.B8.B0">뒷이야기</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%AC%B4%EC%8A%88_%EB%B2%A0%EB%A5%B4%EB%91%90&amp;action=edit&amp;section=1" title="부분 편집: 뒷이야기">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<ul>
<li><a href="/wiki/1942%EB%85%84" title="1942년">1942년</a>, &lt;<a href="/wiki/%EC%8B%9C%EB%AF%BC_%EC%BC%80%EC%9D%B8" title="시민 케인">시민 케인</a>&gt;으로 유명한 <a href="/wiki/%EC%98%A4%EC%86%90_%EC%9B%B0%EC%8A%A4" title="오손 웰스">오손 웰스</a>는 <a href="/wiki/%ED%94%84%EB%9E%91%EC%8A%A4" title="프랑스">프랑스</a>의 살인마 랑드뤼에 관한 영화를 구상하였다. 웰스는 이 영화의 아이디어를 채플린에게 <b>5000 달러</b>와 <b>오프닝에서 오손 웰스의 구상</b>이라고 나오는 조건으로 주었다.</li>
<li><a href="/wiki/1910%EB%85%84%EB%8C%80" title="1910년대">1910년대</a>에서 <a href="/wiki/1920%EB%85%84%EB%8C%80" title="1920년대">1920년대</a>까지 채플린의 여자 주인공을 맡았던 <a href="/wiki/%EC%97%90%EB%93%9C%EB%82%98_%ED%8D%BC%EB%B0%94%EC%9D%B4%EC%96%B8%EC%8A%A4" title="에드나 퍼바이언스">에드나 퍼바이언스</a>는 이번 영화에서 베르두의 결혼 파티에서 잠깐 나왔다.</li>
<li>채플린은 이 영화를 80일 만에 완성하였다고 한다.</li>
<li>이 영화는 <a href="/wiki/%EC%95%84%EC%B9%B4%EB%8D%B0%EB%AF%B8%EC%83%81" title="아카데미상">아카데미</a>에서 각본상 후보로 올랐다.</li>
<li>재판 장면에서 벌어지는 채플린의 대사는 무척 무정부주의 적이어서 <a href="/wiki/%EB%A7%A4%EC%B9%B4%EC%8B%9C%EC%A6%98" title="매카시즘">매카시즘</a>에 휘말리고 있는 채플린을 위해 많은 대사가 전면 삭제되고 편집되었다. 그러나 다음 대사는 끝까지 이어졌다.</li>
</ul>
