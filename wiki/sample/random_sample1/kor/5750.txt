<p><b>무슬림</b>(<span style="font-size: smaller;"><a href="/wiki/%EC%95%84%EB%9E%8D%EC%96%B4" title="아랍어">아랍어</a>:</span> <span lang="ar" xml:lang="ar">مسلم</span> Muslim)은 <a class="mw-redirect" href="/wiki/%EC%9D%B4%EC%8A%AC%EB%9E%8C" title="이슬람">이슬람</a>의 제4 분사형으로 '복종의 행위를 취하는 사람'을 말한다. 즉 <a class="mw-redirect" href="/wiki/%EC%9D%B4%EC%8A%AC%EB%9E%8C" title="이슬람">이슬람</a>을 믿는 사람, <b>이슬람교도</b>를 뜻한다. <b>무슬리마</b>(<span style="font-size: smaller;"><a href="/wiki/%EC%95%84%EB%9E%8D%EC%96%B4" title="아랍어">아랍어</a>:</span> <span lang="ar" xml:lang="ar">مسلمة‎</span> Muslimah)라는 여성형 표현도 있으나, 보통 총칭할 때는 '무슬림'이라고 한다. <a href="/wiki/%ED%95%9C%EA%B5%AD%EC%96%B4" title="한국어">한국어</a>의 <a href="/wiki/%EC%99%B8%EB%9E%98%EC%96%B4_%ED%91%9C%EA%B8%B0%EB%B2%95" title="외래어 표기법">외래어 표기법</a>에서는 <a href="/wiki/%EC%98%81%EC%96%B4" title="영어">영어</a>식 표기인 <b>모슬렘</b>(moslem)이 <a href="/wiki/%ED%91%9C%EC%A4%80%EC%96%B4" title="표준어">표준어</a>로 등재되어 있으나<sup class="reference" id="cite_ref-1"><a href="#cite_note-1">[1]</a></sup> 모슬렘이라는 표기는 이슬람교도를 비하하는 의미가 있다는 지적이 있어<sup class="reference" id="cite_ref-2"><a href="#cite_note-2">[2]</a></sup> 현재는 <a href="/wiki/%EC%95%84%EB%9E%8D%EC%96%B4" title="아랍어">아랍어</a> 원발음에 가까운 <b>무슬림</b>이라는 표기가 한국어권에서도 더 많이 통용되고 있다.</p>
<h2><span class="mw-headline" id=".EB.AC.B4.EC.8A.AC.EB.A6.BC.EC.9D.98_.EC.9D.98.EB.AC.B4">무슬림의 의무</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%AC%B4%EC%8A%AC%EB%A6%BC&amp;action=edit&amp;section=1" title="부분 편집: 무슬림의 의무">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<ul>
<li>무슬림은 유일한 <a href="/wiki/%EC%95%8C%EB%9D%BC" title="알라">알라</a>를 믿고, 그 말에 따르는 사람들을 말한다. <a class="mw-redirect" href="/wiki/%EC%B0%BD%EC%A1%B0%EC%A3%BC" title="창조주">창조주</a> <a href="/wiki/%EC%95%8C%EB%9D%BC" title="알라">알라</a> 이외에 그 어떤 존재도(<a href="/wiki/%EB%AA%A8%EC%84%B8" title="모세">모세</a>, <a href="/wiki/%EC%98%88%EC%88%98" title="예수">예수</a>, <a href="/wiki/%EB%AC%B4%ED%95%A8%EB%A7%88%EB%93%9C" title="무함마드">무함마드</a>, <a href="/wiki/%EC%84%B1%EC%9D%B8_(%EC%A2%85%EA%B5%90)" title="성인 (종교)">성인</a>, <a class="mw-redirect" href="/wiki/%ED%95%99%EC%9E%90" title="학자">학자</a>, <a href="/wiki/%EC%84%B1%EC%A7%81%EC%9E%90" title="성직자">성직자</a>, ...) 숭배하지 않는다.</li>
<li>무슬림은 <a href="/wiki/%EA%BE%B8%EB%9E%80" title="꾸란">꾸란</a>을 믿지만, <a href="/wiki/%EA%B5%AC%EC%95%BD%EC%84%B1%EA%B2%BD" title="구약성경">구약성경</a>과 <a href="/wiki/%EC%8B%A0%EC%95%BD%EC%84%B1%EA%B2%BD" title="신약성경">신약성경</a>을 믿지는 않는다. 꾸란은 번역이 금지되어 있고, 성서와 유사하다.</li>
<li><a href="/wiki/%EB%A9%94%EC%B9%B4" title="메카">메카</a> 성지 순례, <a href="/wiki/%EB%9D%BC%EB%A7%88%EB%8B%A8" title="라마단">라마단</a> 참석</li>
<li>자선을 베풀 의무(물론 도움을 받는 사람이 부끄럽지 않도록 주의해야 한다.)</li>
<li><a href="/wiki/%EC%88%A0" title="술">술</a>, <a href="/wiki/%EB%8F%BC%EC%A7%80" title="돼지">돼지고기</a> 등 <a class="mw-redirect" href="/wiki/%EC%9D%B4%EC%8A%AC%EB%9E%8C" title="이슬람">이슬람</a>에서 금하는 음식을 멀리할 의무</li>
<li><a href="/wiki/%EB%82%A8%EC%84%B1" title="남성">남성</a>이 <a href="/wiki/%EC%97%AC%EC%84%B1" title="여성">여성</a>과, 여성이 남성과 자리를 같이하지 않을 의무</li>
<li>알라가 금지한 모든 것은 무슬림들에게 금지된 것이며 <a href="/wiki/%ED%95%98%EB%9E%8C" title="하람">하람</a>이라고 하고, 허용된 것은 <a href="/wiki/%ED%95%A0%EB%9E%84" title="할랄">할랄</a>이라고 한다.</li>
<li><a class="mw-redirect" href="/wiki/%EC%9D%B4%EC%8A%AC%EB%9E%8C" title="이슬람">이슬람</a>을 전파하기 위해 <a href="/wiki/%EC%A7%80%ED%95%98%EB%93%9C" title="지하드">지하드</a>라는 종교적 의무가 부과된다.</li>
<li>하루에 5번씩 기도한다.</li>
<li><a href="/wiki/%EC%A7%80%EB%8A%90%EB%9F%AC%EB%AF%B8" title="지느러미">지느러미</a>가 없는 <a href="/wiki/%EC%83%9D%EC%84%A0" title="생선">생선</a>은 못 먹는다.</li>
<li><a href="/wiki/%EC%9D%B4%EC%8A%AC%EB%9E%8C%EB%A0%A5" title="이슬람력">이슬람력</a> 11월에 알라에게 제물을 바친다. 이때 소, 양, 염소 등이 할랄의식에 따라 제물로 바쳐지고, 1/3은 가족, 1/3은 어려운 사람, 1/3은 이웃에게 나누어 준다.</li>
</ul>
