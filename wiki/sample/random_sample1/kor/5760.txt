<p><b>무하마드 이크발</b>(<a href="/wiki/1877%EB%85%84" title="1877년">1877년</a> <a href="/wiki/11%EC%9B%94_9%EC%9D%BC" title="11월 9일">11월 9일</a> ~ <a href="/wiki/1938%EB%85%84" title="1938년">1938년</a>, <span style="font-size: smaller;"><a href="/wiki/%EC%9A%B0%EB%A5%B4%EB%91%90%EC%96%B4" title="우르두어">우르두어</a>:</span> <span lang="ur" xml:lang="ur">علامہ سر محمد اقبال</span>)은 <a href="/wiki/%EC%9D%B8%EB%8F%84" title="인도">인도</a> 근대의 <a href="/wiki/%EC%9A%B0%EB%A5%B4%EB%91%90%EC%96%B4" title="우르두어">우르두어</a>, <a class="mw-disambig" href="/wiki/%ED%8E%98%EB%A5%B4%EC%8B%9C%EC%95%84" title="페르시아">페르시아</a>어의 <a href="/wiki/%EC%8B%9C%EC%9D%B8" title="시인">시인</a>·<a class="mw-redirect" href="/wiki/%EC%82%AC%EC%83%81%EA%B0%80" title="사상가">사상가</a>이다. <a href="/wiki/%ED%8E%80%EC%9E%90%EB%B8%8C_%EC%A3%BC_(%ED%8C%8C%ED%82%A4%EC%8A%A4%ED%83%84)" title="펀자브 주 (파키스탄)">펀자브 주</a> <a href="/wiki/%EC%8B%9C%EC%95%8C%EC%BD%94%ED%8A%B8" title="시알코트">시알코트</a>에서 부유한 재단사의 아들로 태어나 <a href="/wiki/%EB%9D%BC%ED%98%B8%EB%A5%B4" title="라호르">라호르</a>에서 생활하고 <a href="/wiki/%EC%9C%A0%EB%9F%BD" title="유럽">유럽</a>에 유학하여 <a href="/wiki/%EC%B2%A0%ED%95%99" title="철학">철학</a>·<a href="/wiki/%EB%B2%95%EB%A5%A0" title="법률">법률</a>을 전공, &lt;페르시아 형이상학의 발달&gt;로 학위를 받았다. 귀국 후 <a href="/wiki/%EB%B3%80%ED%98%B8%EC%82%AC" title="변호사">변호사</a>를 개업하는 한편, 교육·정치 분야에서 활약하였고, <a href="/wiki/1930%EB%85%84" title="1930년">1930년</a> &lt;파키스탄 구상(構想)&gt;을 발표하였다.</p>
<p>자연이나 애국심을 읊은 우르두 시는 &lt;방울 소리&gt; &lt;지브리르의 날개&gt; 등의 시집에 수록되어 있다. 페르시아 시는 철학·신학적 경향의 작품이 많으며, 특히 시집 &lt;자아(自我)의 비밀&gt; &lt;몰아(沒我)의 신비&gt;를 발표하여 '자아 철학'을 확립하고, 인도-<a href="/wiki/%EB%AC%B4%EC%8A%AC%EB%A6%BC" title="무슬림">무슬림</a> 지식층의 열광적인 지지를 받아 <a href="/wiki/%ED%8C%8C%ED%82%A4%EC%8A%A4%ED%83%84" title="파키스탄">파키스탄</a> 건국 운동의 사상적 지주로서의 역할을 하였다. 시집 &lt;동양의 탁선(託宣)&gt; &lt;페르시아 찬가&gt; 외에 만년의 작품 &lt;영원의 서(書)&gt;는 <a class="mw-redirect" href="/wiki/%EB%8B%A8%ED%85%8C" title="단테">단테</a>의 &lt;신곡&gt;의 본을 따서 꾸민 걸작이다. 영어로 된 강연집 &lt;이슬람에서의 종교사상의 재건&gt;은 사상사(思想史)에서 중요한 지위를 차지하고 있다.</p>
