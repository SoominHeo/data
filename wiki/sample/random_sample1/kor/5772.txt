<h2><span class="mw-headline" id=".ED.8A.B9.EC.A7.95">특징</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%AC%B8%EC%88%98%EB%B3%B4%EC%82%B4&amp;action=edit&amp;section=2" title="부분 편집: 특징">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p><a href="/wiki/%EC%A7%80%ED%98%9C" title="지혜">지혜</a>를 완전히 갖춘 보살로서 <a class="mw-redirect" href="/wiki/%EC%84%9D%EA%B0%80%EB%AA%A8%EB%8B%88%EB%B6%88" title="석가모니불">석가모니불</a>의 지덕과 체덕을 맡아서 석가모니불의 교화를 돕기 위해 나타난 보살이다. 여러 형태 중 <a href="/wiki/%EC%82%AC%EC%9E%90" title="사자">사자</a>, <a href="/wiki/%EA%B3%B5%EC%9E%91_(%EB%8F%99%EB%AC%BC)" title="공작 (동물)">공작</a>을 타는 것은 그의 <a href="/wiki/%EC%A7%80%ED%98%9C" title="지혜">지혜</a>가 용맹함을 나타내는 것이다. 오른손에 <a class="mw-disambig" href="/wiki/%EC%B9%BC" title="칼">칼</a>을 들고 있는 것은 <a class="new" href="/w/index.php?title=%EC%9D%BC%EC%B2%B4%EC%A4%91%EC%83%9D&amp;action=edit&amp;redlink=1" title="일체중생 (없는 문서)">일체중생</a>의 <a href="/wiki/%EB%B2%88%EB%87%8C" title="번뇌">번뇌</a>를 끊는다는 뜻이고, 왼손에 <a class="new" href="/w/index.php?title=%EC%B2%AD%EB%A0%A8%ED%99%94&amp;action=edit&amp;redlink=1" title="청련화 (없는 문서)">청련화</a>를 쥐고 있는 형상은 일체 여래의 지혜와 무상(無相)의 지덕(智德)을 맡아서 제법에 물들지 아니하여 마음이 머무르는 곳이 없다는 뜻이다. 머리에 <a href="/wiki/%EC%83%81%ED%88%AC" title="상투">상투</a>를 맺고 있는 것은 지혜를 상징하며, 다섯 개의 상투는 <a class="mw-redirect" href="/wiki/%EB%8C%80%EC%9D%BC%EC%97%AC%EB%9E%98" title="대일여래">대일여래</a>의 <a class="mw-disambig" href="/wiki/%EC%98%A4%EC%A7%80" title="오지">오지</a>(五智)를 표현한 것이다. 다섯 <a class="mw-redirect" href="/wiki/%EB%8F%99%EC%9E%90" title="동자">동자</a>로 문수의 분신을 나타내는 경우도 많다.</p>
<p>화현 보살, 석가의 교화를 돕고자 일시 보살의 자리를 보인 보처보살로 현학적이고 이지적인 보살이며 보살행을 실천하는 보살이라고도 믿어진다.</p>
<h2><span class="mw-headline" id=".ED.95.9C.EA.B5.AD">한국</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%AC%B8%EC%88%98%EB%B3%B4%EC%82%B4&amp;action=edit&amp;section=3" title="부분 편집: 한국">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p><a href="/wiki/%EA%B2%BD%EC%83%81%EB%8F%84" title="경상도">경상도</a> <a href="/wiki/%EC%B2%AD%EB%9F%89%EC%82%B0_(%EA%B2%BD%EC%83%81%EB%B6%81%EB%8F%84)" title="청량산 (경상북도)">청량산</a>, <a href="/wiki/%EA%B0%95%EC%9B%90%EB%8F%84" title="강원도">강원도</a> <a href="/wiki/%EC%98%A4%EB%8C%80%EC%82%B0" title="오대산">오대산</a>, <a class="mw-redirect" href="/wiki/%EC%9A%B8%EC%A7%84" title="울진">울진</a>의 <a href="/wiki/%EB%AC%B8%EC%88%98%EC%82%B0" title="문수산">문수산</a>, <a href="/wiki/%EA%B8%88%EA%B0%95%EC%82%B0" title="금강산">금강산</a> <a class="new" href="/w/index.php?title=%EB%AC%B8%EC%88%98%EB%B4%89&amp;action=edit&amp;redlink=1" title="문수봉 (없는 문서)">문수봉</a> 등에 문수신앙과 관련이 있다. <a href="/wiki/%EB%B6%88%EA%B5%90" title="불교">불교</a>의 우주관에 따르면 세상은 수미산 남쪽의 남섬부주며 오정산은 다섯 봉우리의 산으로 오대산과 같은 뜻으로 여겨 오대산 신앙을 낳았고, <a href="/wiki/%EA%B3%A0%EB%A0%A4" title="고려">고려</a> 말 <a class="mw-redirect" href="/wiki/%EA%B3%B5%EB%AF%BC%EC%99%95" title="공민왕">공민왕</a> 때 <a href="/wiki/%EC%8B%A0%EB%8F%88" title="신돈">신돈</a>이 <a class="new" href="/w/index.php?title=%EB%AC%B8%EC%88%98%ED%9A%8C&amp;action=edit&amp;redlink=1" title="문수회 (없는 문서)">문수회</a>를 만들어 문수법회를 여는 등 왕실의 후원을 입어 후대에까지 전승되었다.</p>
<p>문수보살은 <a href="/wiki/%EB%B3%B4%ED%98%84%EB%B3%B4%EC%82%B4" title="보현보살">보현보살</a>과 함께 <a href="/wiki/%EB%B6%80%EC%B2%98" title="부처">부처</a>의 협시불로서 <a class="new" href="/w/index.php?title=%EC%82%BC%EC%A1%B4%EB%B6%88&amp;action=edit&amp;redlink=1" title="삼존불 (없는 문서)">삼존불</a>을 모신 <a class="new" href="/w/index.php?title=%EB%B2%95%EB%8B%B9&amp;action=edit&amp;redlink=1" title="법당 (없는 문서)">법당</a>에 흔한 보살이나 <a class="mw-redirect" href="/wiki/%EA%B4%80%EC%9D%8C" title="관음">관음</a>, <a class="mw-redirect" href="/wiki/%EB%AF%B8%EB%A5%B5" title="미륵">미륵</a>, <a class="mw-redirect" href="/wiki/%EC%A7%80%EC%9E%A5%EC%8B%A0%EC%95%99" title="지장신앙">지장신앙</a> 만큼 독자적으로 민간에서 신앙되지는 못했다. 한국에는 <a href="/wiki/%EC%8B%A0%EB%9D%BC" title="신라">신라</a> 27대 <a href="/wiki/%EC%84%A0%EB%8D%95%EC%97%AC%EC%99%95" title="선덕여왕">선덕여왕</a>때 <a href="/wiki/%EC%A4%91%EA%B5%AD" title="중국">중국</a>을 통해 <a class="mw-redirect" href="/wiki/%EC%9E%90%EC%9E%A5%EC%9C%A8%EC%82%AC" title="자장율사">자장율사</a>가 소개했다.</p>
<p><a class="mw-redirect" href="/wiki/%EC%9A%B8%EC%82%B0" title="울산">울산</a> <a class="new" href="/w/index.php?title=%EB%AC%B8%EC%88%98%EC%82%AC_(%EC%9A%B8%EC%82%B0_%EC%9A%B8%EC%A3%BC%EA%B5%B0)&amp;action=edit&amp;redlink=1" title="문수사 (울산 울주군) (없는 문서)">문수사</a>는 <a href="/wiki/%EC%9B%90%EC%84%B1%EC%99%95" title="원성왕">원성왕</a>때 <a class="new" href="/w/index.php?title=%EC%97%B0%ED%9A%8C%EA%B5%AD%EC%82%AC&amp;action=edit&amp;redlink=1" title="연회국사 (없는 문서)">연회국사</a>가 창건했는데, 길을 걷던 연희대사가 문수대성이 현신한 농부를 처음 만난 자리에 문수암을 짓고, 변재천녀가 현신한 처녀를 만난 곳에 보현암을 지었다고 한다. 또다른 설은 <a href="/wiki/%EC%84%A0%EB%8D%95%EC%97%AC%EC%99%95" title="선덕여왕">선덕여왕</a>때 <a href="/wiki/%EC%9E%90%EC%9E%A5" title="자장">자장</a>이 창건했는데, 중국을 다녀온 자장율사가 울산을 지나던 중 문수산의 모습이 너무 깨끗하고 성스럽고 중국에 있던 <a class="mw-disambig" href="/wiki/%EC%B2%AD%EB%9F%89%EC%82%B0" title="청량산">청량산</a>과 닮아서 청량산이라 이름하고 문수암을 세웠다고 한다. 울산 태화동에 자장율사가 태화사지를 만들고, 강 이름을 <a href="/wiki/%ED%83%9C%ED%99%94%EA%B0%95" title="태화강">태화강</a>이라 불렀으며, 또한 울산에는 문수보살이 현신한 승려를 묘사한 &lt;헐수정과 무거동&gt; 전설도 전해진다. <a href="/wiki/%EC%A0%84%EB%9D%BC%EB%B6%81%EB%8F%84" title="전라북도">전라북도</a> <a class="mw-redirect" href="/wiki/%EA%B3%A0%EC%B0%BD" title="고창">고창</a> <a class="mw-disambig" href="/wiki/%EC%B2%AD%EB%9F%89%EC%82%B0" title="청량산">청량산</a> <a class="new" href="/w/index.php?title=%EB%AC%B8%EC%88%98%EC%82%AC_(%EC%A0%84%EB%B6%81_%EA%B3%A0%EC%B0%BD%EA%B5%B0)&amp;action=edit&amp;redlink=1" title="문수사 (전북 고창군) (없는 문서)">문수사</a>도 자장율사가 선덕왕 13년에 창건했는데, 대웅전 뒤 문수전에 높이 2.25미터의 문수보살 석상이 있다.<sup class="reference" id="cite_ref-1"><a href="#cite_note-1">[1]</a></sup></p>
<h2><span class="mw-headline" id=".EB.A7.8C.EC.A3.BC">만주</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%AC%B8%EC%88%98%EB%B3%B4%EC%82%B4&amp;action=edit&amp;section=4" title="부분 편집: 만주">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>본래 "여진"(여진어[만주어]: 주션)이란 이름을 갖고 있던 여진족은 청태조 아이신기로오 누르하치에 의해 "만주"라는 이름으로 바뀌는데 만주가 바로 문수에서 따온 이름이라고 한다.</p>
