<p><b>문수역</b>(文殊驛)은 <a href="/wiki/%EA%B2%BD%EC%83%81%EB%B6%81%EB%8F%84" title="경상북도">경상북도</a> <a href="/wiki/%EC%98%81%EC%A3%BC%EC%8B%9C" title="영주시">영주시</a> 문수면 적동리에 위치한 <a href="/wiki/%EC%A4%91%EC%95%99%EC%84%A0" title="중앙선">중앙선</a>의 역이다. <a href="/wiki/1941%EB%85%84" title="1941년">1941년</a> <a href="/wiki/7%EC%9B%94_1%EC%9D%BC" title="7월 1일">7월 1일</a>에 보통역으로 영업을 시작했다가, <a href="/wiki/2007%EB%85%84" title="2007년">2007년</a> <a href="/wiki/6%EC%9B%94_1%EC%9D%BC" title="6월 1일">6월 1일</a>부터 여객열차가 정차하지 않으며 현재는 열차가 서로 교행할 때에만 운전정차를 한다.</p>
<p><a href="/wiki/2013%EB%85%84" title="2013년">2013년</a>에 <a href="/wiki/%EC%98%81%EC%A3%BC%EB%8C%90" title="영주댐">영주댐</a> 건설로 인해 수몰되는 지역에서 이 역과 <a href="/wiki/%EB%A7%88%EC%82%AC%EC%97%AD" title="마사역">마사역</a> 간의 구간의 철도가 이설되었다.</p>
<h2><span class="mw-headline" id=".EC.97.B0.ED.98.81">연혁</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%AC%B8%EC%88%98%EC%97%AD&amp;action=edit&amp;section=1" title="부분 편집: 연혁">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<ul>
<li>1941년 7월 1일 : 보통역으로 영업 개시<sup class="reference" id="cite_ref-1"><a href="#cite_note-1">[1]</a></sup></li>
<li>1977년 5월 16일 : 수소화물취급 중지<sup class="reference" id="cite_ref-2"><a href="#cite_note-2">[2]</a></sup></li>
<li>1986년 10월 25일 : 제1종 기계연동장치 사용개시</li>
<li>1993년 4월 15일 : 차내취급역으로 지정</li>
<li>1995년 1월 10일 : 제1종 기계연동장치 피제어역지정(평은역 제어)</li>
<li>1997년 6월 1일 : 배치간이역으로 격하<sup class="reference" id="cite_ref-3"><a href="#cite_note-3">[3]</a></sup></li>
<li>1998년 3월 9일 : 구내 선로 연장 및 안전측선 설치</li>
<li>2000년 11월 11일 : <a href="/wiki/%EC%98%81%EC%A3%BC%EC%97%AD" title="영주역">영주</a>~<strong class="selflink">문수</strong>간 자동폐색장치 신설</li>
<li>2000년 12월 26일 : 중앙선 CTC 사용개시</li>
<li>2003년 9월 15일 : 보통역으로 승격 및 화물취급 개시<sup class="reference" id="cite_ref-4"><a href="#cite_note-4">[4]</a></sup></li>
<li>2007년 6월 1일 : 일반열차 통과</li>
<li>2007년 7월 1일 : 역광장 포장</li>
</ul>
