<p><b>물병자리 람다</b>(λ Aqr)는 <a class="mw-redirect" href="/wiki/%EC%B2%9C%EA%B5%AC_%EC%A0%81%EB%8F%84" title="천구 적도">천구 적도</a> 근처 <a href="/wiki/%EB%AC%BC%EB%B3%91%EC%9E%90%EB%A6%AC" title="물병자리">물병자리</a> 방향에 보이는 <a href="/wiki/%ED%95%AD%EC%84%B1" title="항성">항성</a>이다. 람다의 고유명칭으로는 <a href="/wiki/%EA%B3%A0%EB%8C%80_%EA%B7%B8%EB%A6%AC%EC%8A%A4" title="고대 그리스">고대 그리스</a> ὕδωρ(‘물’), ἔκχυσις(‘쏟아져 나옴’)에서 나온 ‘히도르’, ‘엑키시스’가 있지만 잘 알려져 있지는 않다. 람다별의 겉보기 등급은 3.722로<sup class="reference" id="cite_ref-sps1966_1-3"><a href="#cite_note-sps1966-1">[1]</a></sup> 맨눈으로 잘 보인다. 지구로부터의 거리는 약 390광년이다.<sup class="reference" id="cite_ref-aaa474_2_653_2-3"><a href="#cite_note-aaa474_2_653-2">[2]</a></sup></p>
<p>동양 문화권에서 이 별은 <a href="/wiki/%EC%8B%A4%EC%88%98_(%EB%B3%84%EC%9E%90%EB%A6%AC)" title="실수 (별자리)">실수</a>의 ‘누벽진’(<span lang="zh" xml:lang="zh">壘壁陣</span>, <span lang="zh-Latn" xml:lang="zh-Latn">Lěi Bì Zhèn</span>)에 속해 있다. 실수를 구성하는 별들은 물병자리 람다, <a class="new" href="/w/index.php?title=%EC%97%BC%EC%86%8C%EC%9E%90%EB%A6%AC_%EC%B9%B4%ED%8C%8C&amp;action=edit&amp;redlink=1" title="염소자리 카파 (없는 문서)">염소자리 카파</a>, <a class="new" href="/w/index.php?title=%EC%97%BC%EC%86%8C%EC%9E%90%EB%A6%AC_%EC%97%A1%EC%8B%A4%EB%A1%A0&amp;action=edit&amp;redlink=1" title="염소자리 엡실론 (없는 문서)">염소자리 엡실론</a>, <a class="new" href="/w/index.php?title=%EC%97%BC%EC%86%8C%EC%9E%90%EB%A6%AC_%EA%B0%90%EB%A7%88&amp;action=edit&amp;redlink=1" title="염소자리 감마 (없는 문서)">염소자리 감마</a>, <a <a class="new" href="/w/index.php?title=%EB%AC%BC%EB%B3%91%EC%9E%90%EB%A6%AC_%EC%8B%9C%EA%B7%B8%EB%A7%88&amp;action=edit&amp;redlink=1" title="물병자리 시그마 (없는 문서)">시그마</a>, <a href="/wiki/%EB%AC%BC%EB%B3%91%EC%9E%90%EB%A6%AC_%ED%94%BC" title="물병자리 피">피</a>, <a class="new" href="/w/index.php?title=%EB%AC%BC%EA%B3%A0%EA%B8%B0%EC%9E%90%EB%A6%AC_27&amp;action=edit&amp;redlink=1" title="물고기자리 27 (없는 문서)">물고기자리 27</a>, <a class="new" href="/w/index.php?title=%EB%AC%BC%EA%B3%A0%EA%B8%B0%EC%9E%90%EB%A6%AC_29&amp;action=edit&amp;redlink=1" title="물고기자리 29 (없는 문서)">물고기자리 29</a>, <a class="new" href="/w/index.php?title=%EB%AC%BC%EA%B3%A0%EA%B8%B0%EC%9E%90%EB%A6%AC_33&amp;action=edit&amp;redlink=1" title="물고기자리 33 (없는 문서)">물고기자리 33</a>, <a class="new" href="/w/index.php?title=%EB%AC%BC%EA%B3%A0%EA%B8%B0%EC%9E%90%EB%A6%AC_30&amp;action=edit&amp;redlink=1" title="물고기자리 30 (없는 문서)">물고기자리 30</a>이 있다.<sup class="reference" id="cite_ref-8"><a href="#cite_note-8">[8]</a></sup> 람다별의 이름은 <b>누벽진칠</b>(<span lang="zh" xml:lang="zh">壘壁陣七</span>, <span lang="zh-Latn" xml:lang="zh-Latn">Lěi Bì Zhèn qī</span>)로 ‘누벽진의 일곱 번째 별’이라는 뜻이다.<sup class="reference" id="cite_ref-9"><a href="#cite_note-9">[9]</a></sup></p>
<p>물병자리 람다는 분광형 M2.5 III의 <a class="mw-redirect" href="/wiki/%EC%A0%81%EC%83%89_%EA%B1%B0%EC%84%B1" title="적색 거성">적색 거성</a>이다.<sup class="reference" id="cite_ref-mnras389_2_869_3-1"><a href="#cite_note-mnras389_2_869-3">[3]</a></sup> 람다는 <a href="/wiki/%EB%B0%98%EA%B7%9C%EC%B9%99_%EB%B3%80%EA%B4%91%EC%84%B1" title="반규칙 변광성">반규칙 변광성</a>으로 24.5일, 32일, 49.5일의 변광 주기를 보인다.<sup class="reference" id="cite_ref-mnras400_4_1945_10-0"><a href="#cite_note-mnras400_4_1945-10">[10]</a></sup> 람다는 <a class="mw-redirect" href="/wiki/%EC%A0%90%EA%B7%BC%EA%B1%B0%EC%84%B1%EA%B0%80%EC%A7%80" title="점근거성가지">점근거성가지</a> 상태에 있으며 <a href="/wiki/%ED%83%84%EC%86%8C" title="탄소">탄소</a>, <a href="/wiki/%EC%82%B0%EC%86%8C" title="산소">산소</a> 성분의 중심핵 주위를 두른 수소와 헬륨을 태워 에너지를 생산하고 있다.<sup class="reference" id="cite_ref-cia152_178_11-0"><a href="#cite_note-cia152_178-11">[11]</a></sup></p>
