<p><b>루이스 미겔 브리투 가르시아 몬테이루</b> (Luís Miguel Brito Garcia Monteiro, <a href="/wiki/1980%EB%85%84" title="1980년">1980년</a> <a href="/wiki/1%EC%9B%94_4%EC%9D%BC" title="1월 4일">1월 4일</a> <a href="/wiki/%EB%A6%AC%EC%8A%A4%EB%B3%B8" title="리스본">리스본</a> ~), 혹은 <b>미겔</b>은 <a href="/wiki/%ED%8F%AC%EB%A5%B4%ED%88%AC%EA%B0%88" title="포르투갈">포르투갈</a>의 <a href="/wiki/%EC%B6%95%EA%B5%AC_%EC%84%A0%EC%88%98" title="축구 선수">축구 선수</a>로, 현재 <a class="mw-redirect" href="/wiki/%ED%94%84%EB%A6%AC%EB%A9%94%EB%9D%BC%EB%A6%AC%EA%B0%80" title="프리메라리가">스페인 라 리가</a>의 <a href="/wiki/%EB%B0%9C%EB%A0%8C%EC%8B%9C%EC%95%84_CF" title="발렌시아 CF">발렌시아 CF</a> 소속이다. 주로 <a href="/wiki/%EC%88%98%EB%B9%84%EC%88%98" title="수비수">라이트 백</a>으로 기용된다.</p>

<h3><span class="mw-headline" id=".EC.B4.88.EC.B0.BD.EA.B8.B0.EC.99.80_.EB.B2.A4.ED.94.BC.EC.B9.B4">초창기와 벤피카</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%AF%B8%EA%B2%94_%EB%AA%AC%ED%85%8C%EC%9D%B4%EB%A3%A8&amp;action=edit&amp;section=2" title="부분 편집: 초창기와 벤피카">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>미겔은 <a class="mw-redirect" href="/wiki/CF_%EC%97%90%EC%8A%A4%ED%8A%B8%EB%A0%90%EB%9D%BC_%EB%8B%A4_%EC%95%84%EB%A7%88%EB%8F%84%EB%9D%BC" title="CF 에스트렐라 다 아마도라">CF 에스트렐라 다 아마도라</a>에서 <a href="/wiki/%EB%AF%B8%EB%93%9C%ED%95%84%EB%8D%94" title="미드필더">윙어</a>로 데뷔하였다. <a href="/wiki/1999%EB%85%84" title="1999년">1999년</a> <a href="/wiki/4%EC%9B%94_30%EC%9D%BC" title="4월 30일">4월 30일</a>, 그는 1-2로 패한 <a href="/wiki/%EB%B3%B4%EC%95%84%EB%B9%84%EC%8A%A4%ED%83%80_FC" title="보아비스타 FC">보아비스타 FC</a> 전에서 데뷔하여 17분을 소화하였다. 그 후, 그는 소속팀이 1부리그에 승격하기 전까지 28경기를 더 출장하였다. 그 후, 그는 <a href="/wiki/SL_%EB%B2%A4%ED%94%BC%EC%B9%B4" title="SL 벤피카">SL 벤피카</a>로 이적하였다.</p>
<p><a href="/wiki/2000%EB%85%84" title="2000년">2000년</a> 여름, 미겔은 <a href="/wiki/SL_%EB%B2%A4%ED%94%BC%EC%B9%B4" title="SL 벤피카">SL 벤피카</a>로 이적하여 <a href="/wiki/%EB%AF%B8%EB%93%9C%ED%95%84%EB%8D%94" title="미드필더">오른쪽 윙어</a>에서 (코칭스태프 <a class="new" href="/w/index.php?title=%ED%8E%98%EB%A5%B4%EB%82%9C%EB%91%90_%EC%83%AC%EB%9D%BC%EB%82%98&amp;action=edit&amp;redlink=1" title="페르난두 샬라나 (없는 문서)">페르난두 샬라나</a>의 지적으로 인하여) <a href="/wiki/%EC%88%98%EB%B9%84%EC%88%98" title="수비수">라이트 백</a>으로 변신하였다. 이 포지션 전환은 그가 세계적인 선수로 뛰어오르게 하였다. 2004-05 시즌 그는 22경기에 출장하여 2골을 기록하였고, 벤피카는 11년 만에 포르투갈 리그 정상에 복귀하였다.</p>
<h3><span class="mw-headline" id=".EB.B0.9C.EB.A0.8C.EC.8B.9C.EC.95.84_CF">발렌시아 CF</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%AF%B8%EA%B2%94_%EB%AA%AC%ED%85%8C%EC%9D%B4%EB%A3%A8&amp;action=edit&amp;section=3" title="부분 편집: 발렌시아 CF">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>미겔은 <a href="/wiki/2005%EB%85%84" title="2005년">2005년</a> <a href="/wiki/8%EC%9B%94" title="8월">8월</a>에 €7.5M의 가격에 <a href="/wiki/%EB%B0%9C%EB%A0%8C%EC%8B%9C%EC%95%84_CF" title="발렌시아 CF">발렌시아 CF</a>로 이적하였다.<sup class="reference" id="cite_ref-VCFtransfer_1-0"><a href="#cite_note-VCFtransfer-1">[1]</a></sup> 주전 자리를 굳히고, 그는 <a href="/wiki/2007%EB%85%84" title="2007년">2007년</a> <a href="/wiki/9%EC%9B%94" title="9월">9월</a>에 5년 재계약을 하였고,<sup class="reference" id="cite_ref-ContractExtension_2-0"><a href="#cite_note-ContractExtension-2">[2]</a></sup> 이듬 해 <a href="/wiki/%EC%BD%94%ED%8C%8C_%EB%8D%B8_%EB%A0%88%EC%9D%B4" title="코파 델 레이">코파 델 레이</a> 결승전에서 <a href="/wiki/%ED%97%A4%ED%83%80%ED%8E%98_CF" title="헤타페 CF">헤타페 CF</a>를 3-1로 꺾어, 발렌시아가 컵 대회 우승을 하는 데 도왔다.</p>
<p>2009-10 시즌, 새로 영입된 <a class="new" href="/w/index.php?title=%EB%B8%8C%EB%A3%A8%EB%85%B8_%EC%82%B4%ED%86%A0%EB%A5%B4&amp;action=edit&amp;redlink=1" title="브루노 살토르 (없는 문서)">브루노 살토르</a>와 포지션 경쟁을 하였으나, 여전히 25경기를 출장하여, 소속팀이 리그 3위로 마감하고, <a href="/wiki/UEFA_%EC%B1%94%ED%94%BC%EC%96%B8%EC%8A%A4%EB%A6%AC%EA%B7%B8" title="UEFA 챔피언스리그">UEFA 챔피언스리그</a> 복귀를 도왔다.</p>
<h2><span class="mw-headline" id=".EA.B5.AD.EA.B0.80.EB.8C.80.ED.91.9C.ED.8C.80_.EA.B2.BD.EB.A0.A5">국가대표팀 경력</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%AF%B8%EA%B2%94_%EB%AA%AC%ED%85%8C%EC%9D%B4%EB%A3%A8&amp;action=edit&amp;section=4" title="부분 편집: 국가대표팀 경력">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p><a href="/wiki/2003%EB%85%84" title="2003년">2003년</a> <a href="/wiki/2%EC%9B%94_13%EC%9D%BC" title="2월 13일">2월 13일</a>, <a href="/wiki/%ED%8F%AC%EB%A5%B4%ED%88%AC%EA%B0%88_%EC%B6%95%EA%B5%AC_%EA%B5%AD%EA%B0%80%EB%8C%80%ED%91%9C%ED%8C%80" title="포르투갈 축구 국가대표팀">포르투갈 축구 국가대표팀</a>에 발탁되어 <a href="/wiki/%EC%9D%B4%ED%83%88%EB%A6%AC%EC%95%84_%EC%B6%95%EA%B5%AC_%EA%B5%AD%EA%B0%80%EB%8C%80%ED%91%9C%ED%8C%80" title="이탈리아 축구 국가대표팀">이탈리아</a>를 상대로 데뷔전을 치렀고, <a href="/wiki/UEFA_%EC%9C%A0%EB%A1%9C_2004" title="UEFA 유로 2004">UEFA 유로 2004</a>의 명단에 들었다. 포르투갈은 결승전에서 <a href="/wiki/%EA%B7%B8%EB%A6%AC%EC%8A%A4_%EC%B6%95%EA%B5%AC_%EA%B5%AD%EA%B0%80%EB%8C%80%ED%91%9C%ED%8C%80" title="그리스 축구 국가대표팀">그리스</a>에 개막전과 결승전에 패하여 준우승을 거두었고, 이후에 그는 <a href="/wiki/2006%EB%85%84_FIFA_%EC%9B%94%EB%93%9C%EC%BB%B5" title="2006년 FIFA 월드컵">2006년 FIFA 월드컵</a>에도 참여하였다.</p>
<p><a href="/wiki/UEFA_%EC%9C%A0%EB%A1%9C_2008" title="UEFA 유로 2008">UEFA 유로 2008</a>에서 그는 <a href="/wiki/%EC%A1%B0%EC%A0%9C_%EB%B3%B4%EC%8B%B1%EC%99%80" title="조제 보싱와">조제 보싱와</a>의 교체 요원으로 활약하였으며, 그는 0-2로 패한 <a href="/wiki/%EC%8A%A4%EC%9C%84%EC%8A%A4_%EC%B6%95%EA%B5%AC_%EA%B5%AD%EA%B0%80%EB%8C%80%ED%91%9C%ED%8C%80" title="스위스 축구 국가대표팀">스위스</a>전에만 출장하였다.</p>
<p>그는 <a href="/wiki/%EB%B0%9C%EB%A0%8C%EC%8B%9C%EC%95%84_CF" title="발렌시아 CF">발렌시아 CF</a>에서 기복이 심했음에도 불구하고, <a href="/wiki/%EB%82%A8%EC%95%84%ED%94%84%EB%A6%AC%EC%B9%B4_%EA%B3%B5%ED%99%94%EA%B5%AD" title="남아프리카 공화국">남아프리카 공화국</a>에서 열린 <a href="/wiki/2010%EB%85%84_FIFA_%EC%9B%94%EB%93%9C%EC%BB%B5" title="2010년 FIFA 월드컵">2010년 FIFA 월드컵</a>의 최종 엔트리에도 들었다. 미겔은 포르투갈의 네 경기에 쓰인 세명의 라이트 백 중 한명으로, 그는 <a class="new" href="/w/index.php?title=%EC%A1%B0%EC%84%A0%EC%9D%B8%EB%AF%BC%EA%B3%B5%ED%99%94%EA%B5%AD_%EC%B6%95%EA%B5%AC_%EA%B5%AD%EA%B0%80%EB%8C%80%ED%91%9C%ED%8C%80&amp;action=edit&amp;redlink=1" title="조선인민공화국 축구 국가대표팀 (없는 문서)">조선인민공화국</a>전에 출장하여 팀의 7-0 승리를 도왔다. <a href="/wiki/2010%EB%85%84" title="2010년">2010년</a> <a href="/wiki/9%EC%9B%94_10%EC%9D%BC" title="9월 10일">9월 10일</a>, 그는 30세의 나이로 국가대표 은퇴를 선언하였다.</p>
