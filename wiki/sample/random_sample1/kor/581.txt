<p><b>2순환로</b>(二循環路) 또는 <b>제2순환로</b>(第二循環路)는 <a href="/wiki/%EA%B4%91%EC%A3%BC%EA%B4%91%EC%97%AD%EC%8B%9C" title="광주광역시">광주광역시</a>의 외곽 지역을 순환하는 27.66km의 도시고속도로이다. 기존의 <a class="mw-redirect" href="/wiki/%EC%A0%9C1%EC%88%9C%ED%99%98%EB%A1%9C_(%EA%B4%91%EC%A3%BC)" title="제1순환로 (광주)">제1순환로</a> <sup class="reference" id="cite_ref-1"><a href="#cite_note-1">[1]</a></sup>가 시내에 위치한 반면 제2순환로는 시외에 위치하고 있다. 전 구간이 <a href="/wiki/%EC%9E%90%EB%8F%99%EC%B0%A8_%EC%A0%84%EC%9A%A9%EB%8F%84%EB%A1%9C" title="자동차 전용도로">자동차 전용도로</a> 구간으로 <sup class="reference" id="cite_ref-2"><a href="#cite_note-2">[2]</a></sup>, <a href="/wiki/1992%EB%85%84" title="1992년">1992년</a> <a href="/wiki/10%EC%9B%94_1%EC%9D%BC" title="10월 1일">10월 1일</a>에 착공하여 <a href="/wiki/1995%EB%85%84" title="1995년">1995년</a> <a href="/wiki/1%EC%9B%94_15%EC%9D%BC" title="1월 15일">1월 15일</a>을 시작으로 일부 구간이 순차적으로 개통됐다. <a href="/wiki/2004%EB%85%84" title="2004년">2004년</a> <a href="/wiki/12%EC%9B%94_10%EC%9D%BC" title="12월 10일">12월 10일</a> 각화 나들목 ~ <a href="/wiki/%EB%8F%99%EA%B4%91%EC%A3%BC_%EB%82%98%EB%93%A4%EB%AA%A9" title="동광주 나들목">동광주 나들목</a> 구간의 교통 혼잡으로 인해 <a href="/wiki/%ED%98%B8%EB%82%A8%EA%B3%A0%EC%86%8D%EB%8F%84%EB%A1%9C" title="호남고속도로">호남고속도로</a>와 접속 노선이 <a href="/wiki/%EB%AC%B8%ED%9D%A5_%EB%B6%84%EA%B8%B0%EC%A0%90" title="문흥 분기점">문흥 분기점</a>으로 변경됐으며, 문흥 나들목 신설 등의 변경 노선 구간을 착공하였다. <a href="/wiki/2009%EB%85%84" title="2009년">2009년</a> <a href="/wiki/12%EC%9B%94_17%EC%9D%BC" title="12월 17일">12월 17일</a> <a href="/wiki/%EB%AC%B8%ED%9D%A5_%EB%B6%84%EA%B8%B0%EC%A0%90" title="문흥 분기점">문흥 분기점</a>이 개통하면서 전 구간이 완공되었다. 순환도로 구간 중 일부 민간 자본에 의해 건설된 구간이 있어 각 3개 요금소에서 통행료를 징수하고 있다.</p>
<p>본래 광주광역시에서는 이 도로에 대해 <b>광주광역시도 제77호선</b>이라는 번호를 부여할 계획이었으나 이후 가로에 도로명칭을 표기하게 되면서 계획이 폐기되었다.</p>
