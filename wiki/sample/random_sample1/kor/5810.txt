<p><b>미겔 안젤루 레오나르두 비토르</b>(Miguel Ângelo Leonardo Vítor, <a href="/wiki/1989%EB%85%84" title="1989년">1989년</a> <a href="/wiki/6%EC%9B%94_30%EC%9D%BC" title="6월 30일">6월 30일</a>, <a class="new" href="/w/index.php?title=%ED%86%A0%ED%9D%90%EC%8A%A4%EB%B2%A0%EB%93%9C%EB%9D%BC%EC%8A%A4&amp;action=edit&amp;redlink=1" title="토흐스베드라스 (없는 문서)">토흐스베드라스</a> ~ )는 <a href="/wiki/%ED%8F%AC%EB%A5%B4%ED%88%AC%EA%B0%88" title="포르투갈">포르투갈</a>의 축구 선수로, 현재 <a href="/wiki/%EB%A6%AC%EA%B0%93_%ED%95%98%EC%95%8C" title="리갓 하알">리갓 하알</a>의 <a href="/wiki/%ED%95%98%ED%8F%AC%EC%97%98_%EB%B2%A0%EB%A5%B4%EC%85%B0%EB%B0%94_FC" title="하포엘 베르셰바 FC">하포엘 베르셰바</a>에서 <a href="/wiki/%EC%88%98%EB%B9%84%EC%88%98" title="수비수">센터백</a>으로 활약하고 있다.</p>

<h2><span class="mw-headline" id=".ED.81.B4.EB.9F.BD_.EA.B2.BD.EB.A0.A5">클럽 경력</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%AF%B8%EA%B2%94_%EB%B9%84%ED%86%A0%EB%A5%B4&amp;action=edit&amp;section=1" title="부분 편집: 클럽 경력">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>2007-08 시즌 <a href="/wiki/SL_%EB%B2%A4%ED%94%BC%EC%B9%B4" title="SL 벤피카">SL 벤피카</a>의 1군으로 승격된 비토르는, <a href="/wiki/2007%EB%85%84" title="2007년">2007년</a> <a href="/wiki/8%EC%9B%94" title="8월">8월</a>, <a href="/wiki/%EB%B9%84%ED%86%A0%EB%A6%AC%EC%95%84_SC" title="비토리아 SC">비토리아 SC</a>전에서 <a href="/wiki/%ED%94%84%EB%A6%AC%EB%A9%94%EC%9D%B4%EB%9D%BC%EB%A6%AC%EA%B0%80" title="프리메이라리가">포르투갈 리그</a> 데뷔전을 치루었다. 이후, <a class="mw-redirect" href="/wiki/FC_%EC%BE%A8%EB%B2%A4%ED%95%98%EC%9A%B4" title="FC 쾨벤하운">FC 쾨벤하운</a>과의 <a href="/wiki/UEFA_%EC%B1%94%ED%94%BC%EC%96%B8%EC%8A%A4%EB%A6%AC%EA%B7%B8" title="UEFA 챔피언스리그">UEFA 챔피언스리그</a> 3차예선전에서, 주전 <a href="/wiki/%EC%88%98%EB%B9%84%EC%88%98" title="수비수">센터백</a> 듀오였던 <a href="/wiki/%EB%A3%A8%EC%9D%B4%EC%9E%A5" title="루이장">루이장</a>과 <a href="/wiki/%EB%8B%A4%EB%B9%84%EB%93%9C_%EB%A3%A8%EC%9D%B4%EC%8A%A4" title="다비드 루이스">다비드 루이스</a>가 부상으로 결장함에 따라 출전하였다.</p>
<p><a href="/wiki/2008%EB%85%84" title="2008년">2008년</a> <a href="/wiki/1%EC%9B%94" title="1월">1월</a>부터 시즌 종료때까지, 그는 <a class="new" href="/w/index.php?title=%ED%98%B8%EB%A9%94%EC%9A%B0_%ED%9E%88%EB%B2%A0%EC%9D%B4%EB%A3%A8&amp;action=edit&amp;redlink=1" title="호메우 히베이루 (없는 문서)">호메우 히베이루</a>와 함께 <a class="new" href="/w/index.php?title=CD_%EC%95%84%EB%B8%8C&amp;action=edit&amp;redlink=1" title="CD 아브 (없는 문서)">CD 아브</a>로 임대되었다. 시즌 종료 후, 그는 <a href="/wiki/SL_%EB%B2%A4%ED%94%BC%EC%B9%B4" title="SL 벤피카">SL 벤피카</a>로 복귀하였고, 2008-09 시즌동안 자주 출장하였다.</p>
<p><a href="/wiki/2010%EB%85%84" title="2010년">2010년</a> <a href="/wiki/7%EC%9B%94" title="7월">7월</a> 초, 비토르는 <a class="mw-redirect" href="/wiki/%ED%92%8B%EB%B3%BC_%EB%A6%AC%EA%B7%B8_%EC%B1%94%ED%94%BC%EC%96%B8%EC%89%BD" title="풋볼 리그 챔피언쉽">잉글랜드 챔피언쉽</a>의 <a href="/wiki/%EB%A0%88%EC%8A%A4%ED%84%B0_%EC%8B%9C%ED%8B%B0_FC" title="레스터 시티 FC">레스터 시티 FC</a>에 한 시즌과 임대되었고, 같은 국적의 <a class="new" href="/w/index.php?title=%EC%A3%BC%EC%95%99_%EB%AF%B8%EA%B2%94_%EB%8B%A4_%EC%BF%A0%EB%83%90_%ED%85%8C%EC%9D%B4%EC%84%B8%EC%9D%B4%EB%9D%BC&amp;action=edit&amp;redlink=1" title="주앙 미겔 다 쿠냐 테이세이라 (없는 문서)">모레누</a>와 <a class="new" href="/w/index.php?title=%ED%8C%8C%EC%9A%B8%EB%A3%A8_%EC%88%98%EC%9E%90&amp;action=edit&amp;redlink=1" title="파울루 수자 (없는 문서)">파울루 수자</a>와 합류하였다.<sup class="reference" id="cite_ref-Leicester_1-0"><a href="#cite_note-Leicester-1">[1]</a></sup> 그는 <a href="/wiki/9%EC%9B%94_21%EC%9D%BC" title="9월 21일">9월 21일</a> <a class="new" href="/w/index.php?title=%ED%94%84%EB%9E%98%ED%8A%BC_%ED%8C%A8%ED%81%AC&amp;action=edit&amp;redlink=1" title="프래튼 패크 (없는 문서)">프래튼 패크</a>에서의 <a href="/wiki/%ED%8F%AC%EC%B8%A0%EB%A8%B8%EC%8A%A4_FC" title="포츠머스 FC">포츠머스 FC</a>전 <a class="mw-redirect" href="/wiki/%ED%92%8B%EB%B3%BC_%EB%A6%AC%EA%B7%B8_%EC%BB%B5" title="풋볼 리그 컵">리그컵</a> 원정 경기에서 레스터 데뷔를 하였고, 경기는 2-1 승으로 종료되었다.<sup class="reference" id="cite_ref-LeicesterDebut_2-0"><a href="#cite_note-LeicesterDebut-2">[2]</a></sup> <a href="/wiki/11%EC%9B%94_6%EC%9D%BC" title="11월 6일">11월 6일</a>, 프로 계약 이후 처음으로 골을 기록하였다. 그는 오크웰에서 열린 <a href="/wiki/%EB%B0%98%EC%A6%90%EB%A6%AC_FC" title="반즐리 FC">반즐리 FC</a>와의 경기에서 18분에 <a class="new" href="/w/index.php?title=%EB%A7%88%EB%A5%B4%ED%8B%B4_%EC%9B%A8%EA%B7%B8%ED%98%BC&amp;action=edit&amp;redlink=1" title="마르틴 웨그혼 (없는 문서)">마르틴 웨그혼</a>의 프리킥을 헤딩으로 득점하였고, 팀은 2-0으로 승리하였다.<sup class="reference" id="cite_ref-goal_3-0"><a href="#cite_note-goal-3">[3]</a></sup></p>
<p><a href="/wiki/2010%EB%85%84" title="2010년">2010년</a> <a href="/wiki/12%EC%9B%94_19%EC%9D%BC" title="12월 19일">12월 19일</a>, 비토르는 <a href="/wiki/%EC%9E%85%EC%8A%A4%EC%9C%84%EC%B9%98_%ED%83%80%EC%9A%B4_FC" title="입스위치 타운 FC">입스위치 타운 FC</a>와의 경기에서 햄스트링 부상을 당하였고, 팀은 이 원정 경기에서 0-3으로 완패하였다.<sup class="reference" id="cite_ref-injury_4-0"><a href="#cite_note-injury-4">[4]</a></sup> 이후, <a href="/wiki/2011%EB%85%84" title="2011년">2011년</a> <a href="/wiki/3%EC%9B%94_12%EC%9D%BC" title="3월 12일">3월 12일</a>, 3-0으로 승리한 <a href="/wiki/%EC%8A%A4%EC%BB%A8%EC%86%8C%ED%94%84_%EC%9C%A0%EB%82%98%EC%9D%B4%ED%8B%B0%EB%93%9C_FC" title="스컨소프 유나이티드 FC">스컨소프 유나이티드 FC</a>에서 복귀하였다.<sup class="reference" id="cite_ref-Scunthrope_5-0"><a href="#cite_note-Scunthrope-5">[5]</a></sup></p>
