<h1 class="firstHeading" id="firstHeading" lang="ko">미국 남부 문학</h1>
<div class="mw-body-content" id="bodyContent">
<div id="siteSub">위키백과, 우리 모두의 백과사전.</div>
<div id="contentSub"></div>
<div class="mw-jump" id="jump-to-nav">
					이동:					<a href="#mw-head">둘러보기</a>, 					<a href="#p-search">검색</a>
</div>
<div class="mw-content-ltr" dir="ltr" id="mw-content-text" lang="ko"><script>function mfTempOpenSection(id){var block=document.getElementById("mf-section-"+id);block.className+=" open-block";block.previousSibling.className+=" open-block";}</script><p><b>미국 남부 문학</b>은 미국의 남부 주에 관한 <a href="/wiki/%EB%AF%B8%EA%B5%AD_%EB%AC%B8%ED%95%99" title="미국 문학">미국 문학</a> 또는 그 지역 출신의 작가들이 쓴 글들이다. 미국 남부 문학의 특징은 다음과 같다. 문학 작품들이 <a href="/wiki/%EB%AF%B8%EA%B5%AD%EC%9D%98_%EC%97%AD%EC%82%AC" title="미국의 역사">미국 남부 역사</a>와 <a href="/wiki/%EA%B0%80%EC%A1%B1" title="가족">가족</a>의 중요성과 공동체 의식과 공동체 안에서 한 사람의 역할과 그 지역의 지배적인 <a href="/wiki/%EC%A2%85%EA%B5%90" title="종교">종교</a>(<a href="/wiki/%EA%B0%9C%EC%8B%A0%EA%B5%90" title="개신교">개신교</a>)와 종교가 자주 가져오는 마음의 부담이나 보상과 인종 간의 갈등과 땅과 그것이 가져오는 약속과 사회 계층 의식과 미국 남부 사투리의 사용에 초점을 맞추고 있다는 것이다.<sup class="reference" id="cite_ref-autogenerated1_1-0"><a href="#cite_note-autogenerated1-1">[1]</a></sup></p>
<div class="thumb tright">
<div class="thumbinner" style="width:302px;"><a class="image" href="/wiki/%ED%8C%8C%EC%9D%BC:US_map-South_Modern.png"><img alt="" class="thumbimage" data-file-height="365" data-file-width="566" height="193" src="//upload.wikimedia.org/wikipedia/commons/thumb/8/8d/US_map-South_Modern.png/300px-US_map-South_Modern.png" srcset="//upload.wikimedia.org/wikipedia/commons/thumb/8/8d/US_map-South_Modern.png/450px-US_map-South_Modern.png 1.5x, //upload.wikimedia.org/wikipedia/commons/8/8d/US_map-South_Modern.png 2x" width="300"/></a>
<div class="thumbcaption">
<div class="magnify"><a class="internal" href="/wiki/%ED%8C%8C%EC%9D%BC:US_map-South_Modern.png" title="실제 크기로"></a></div>
<b>현대 정의</b> 짙은 빨강색으로 칠해진 주들이 남부에 대한 현대적 정의에 항상 포함되는 지역이며, 반면에 그보다 옅은 빨강색으로 칠해진 주들은 일반적으로 포함되는 지역이다. 줄이 그어진 주들은 때때로/자주 남부로 간주되는 지역이다.<sup class="reference" id="cite_ref-2"><a href="#cite_note-2">[2]</a></sup><sup class="reference" id="cite_ref-3"><a href="#cite_note-3">[3]</a></sup></div>
</div>
</div>
<h2><span class="mw-headline" id=".EB.82.A8.EB.B6.80_.EB.AC.B8.ED.95.99.EC.9D.98_.EA.B0.9C.EC.9A.94">남부 문학의 개요</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%AF%B8%EA%B5%AD_%EB%82%A8%EB%B6%80_%EB%AC%B8%ED%95%99&amp;action=edit&amp;section=1" title="부분 편집: 남부 문학의 개요">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>단순하게 말해서 미국 남부 문학은 지리적인 이유 만큼이나 역사적인 이유로 미국 남부로 분류된 미국의 주들 즉 <a class="mw-redirect" href="/wiki/%EC%82%AC%EC%9A%B0%EC%8A%A4_%EC%BA%90%EB%A1%A4%EB%9D%BC%EC%9D%B4%EB%82%98_%EC%A3%BC" title="사우스 캐롤라이나 주">사우스 캐롤라이나 주</a>, <a href="/wiki/%EC%A1%B0%EC%A7%80%EC%95%84_%EC%A3%BC" title="조지아 주">조지아 주</a>, <a href="/wiki/%ED%94%8C%EB%A1%9C%EB%A6%AC%EB%8B%A4_%EC%A3%BC" title="플로리다 주">플로리다 주</a>, <a href="/wiki/%EC%95%A8%EB%9D%BC%EB%B0%B0%EB%A7%88_%EC%A3%BC" title="앨라배마 주">앨라배마 주</a>, <a class="mw-redirect" href="/wiki/%EB%85%B8%EC%8A%A4_%EC%BA%90%EB%A1%A4%EB%9D%BC%EC%9D%B4%EB%82%98_%EC%A3%BC" title="노스 캐롤라이나 주">노스 캐롤라이나 주</a>, <a href="/wiki/%EB%B2%84%EC%A7%80%EB%8B%88%EC%95%84_%EC%A3%BC" title="버지니아 주">버지니아 주</a>, <a href="/wiki/%ED%85%8C%EB%84%A4%EC%8B%9C_%EC%A3%BC" title="테네시 주">테네시 주</a>, <a href="/wiki/%EB%AF%B8%EC%8B%9C%EC%8B%9C%ED%94%BC_%EC%A3%BC" title="미시시피 주">미시시피 주</a>, <a href="/wiki/%EB%A3%A8%EC%9D%B4%EC%A7%80%EC%95%A0%EB%82%98_%EC%A3%BC" title="루이지애나 주">루이지애나 주</a>, <a href="/wiki/%ED%85%8D%EC%82%AC%EC%8A%A4_%EC%A3%BC" title="텍사스 주">텍사스 주</a>, <a href="/wiki/%EC%98%A4%ED%81%B4%EB%9D%BC%ED%98%B8%EB%A7%88_%EC%A3%BC" title="오클라호마 주">오클라호마 주</a>, <a href="/wiki/%EC%BC%84%ED%84%B0%ED%82%A4_%EC%A3%BC" title="켄터키 주">켄터키 주</a>, <a class="mw-redirect" href="/wiki/%EC%9B%A8%EC%8A%A4%ED%8A%B8_%EB%B2%84%EC%A7%80%EB%8B%88%EC%95%84_%EC%A3%BC" title="웨스트 버지니아 주">웨스트 버지니아 주</a>, <a href="/wiki/%EC%95%84%EC%B9%B8%EC%86%8C_%EC%A3%BC" title="아칸소 주">아칸소 주</a> 지역에 대한 글쓰기로 구성되어 있다.<sup class="reference" id="cite_ref-4"><a href="#cite_note-4">[4]</a></sup> 남북전쟁 이전에는 미국 남부에 <a href="/wiki/%EB%AF%B8%EC%A3%BC%EB%A6%AC_%EC%A3%BC" title="미주리 주">미주리 주</a>, <a href="/wiki/%EB%A9%94%EB%A6%B4%EB%9E%9C%EB%93%9C_%EC%A3%BC" title="메릴랜드 주">메릴랜드 주</a>, <a href="/wiki/%EB%8D%B8%EB%9D%BC%EC%9B%A8%EC%96%B4_%EC%A3%BC" title="델라웨어 주">델라웨어 주</a>가 포함되어 있었다.</p>
