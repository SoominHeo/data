<p><a href="/wiki/%EB%AF%B8%EA%B5%AD_%EB%82%A8%EB%B6%80" title="미국 남부">미국 남부</a>의 문화는 미국내에서 가장 지역색이 강하고 독특하다. 사회적으로 보수적이며 전통적으로 농경 문화와 함께 사회 계층이 뚜렷하고 교회가 지역사회의 중심체이다. 이 지역의 문화는 주로 <a href="/wiki/%EA%B4%80%EC%8A%B5" title="관습">관습</a>과 <a href="/wiki/%EC%9D%8C%EC%8B%9D" title="음식">음식</a> 그리고 <a href="/wiki/%EC%9D%8C%EC%95%85" title="음악">음악</a>으로 다른 지역의 문화와 구별된다. </p>

<h2><span class="mw-headline" id=".EC.A3.BC.EB.AF.BC">주민</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%AF%B8%EA%B5%AD_%EB%82%A8%EB%B6%80%EC%9D%98_%EB%AC%B8%ED%99%94&amp;action=edit&amp;section=1" title="부분 편집: 주민">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p><a href="/wiki/%EC%98%81%EA%B5%AD" title="영국">영국</a>의 식민지로 시작한 남부 지역은 남부 <a href="/wiki/%EC%98%81%EA%B5%AD" title="영국">영국</a>인에 의해 개척되어 <a href="/wiki/%EC%8A%A4%EC%BD%94%ED%8B%80%EB%9E%9C%EB%93%9C" title="스코틀랜드">스코틀랜드</a>, 북부 영국인, 스코트-아일랜드인이 산악지역(<a href="/wiki/%EC%95%A0%ED%8C%94%EB%9E%98%EC%B9%98%EC%95%84_%EC%82%B0%EB%A7%A5" title="애팔래치아 산맥">애팔래치아 산맥</a>)에 정착하게 된다. 후에 <a href="/wiki/%EB%8F%85%EC%9D%BC" title="독일">독일</a>인과 <a class="mw-redirect" href="/wiki/%EB%85%B8%EC%98%88" title="노예">노예</a>로써 들어온 <a href="/wiki/%EC%95%84%ED%94%84%EB%A6%AC%EC%B9%B4%EA%B3%84_%EB%AF%B8%EA%B5%AD%EC%9D%B8" title="아프리카계 미국인">흑인</a>이 <a class="mw-redirect" href="/wiki/%EC%9D%B4%EB%AF%BC" title="이민">이민</a>와 현재 <a href="/wiki/%EC%95%84%ED%94%84%EB%A6%AC%EC%B9%B4%EA%B3%84_%EB%AF%B8%EA%B5%AD%EC%9D%B8" title="아프리카계 미국인">아프리카계 미국인</a>과 영국계 미국인이 이 지역의 주류를 이룬다. 전통적으로 백인과 흑인 위주의 사회였으나, 현재 공업화와 도시화에 발맞추어 <a class="new" href="/w/index.php?title=%EC%95%84%EC%8B%9C%EC%95%84_%EC%9D%B8&amp;action=edit&amp;redlink=1" title="아시아 인 (없는 문서)">아시아 인</a>과 이민계의 인구 비율이 일부 도시에서 급증하고 있다. <a href="/wiki/%ED%9E%88%EC%8A%A4%ED%8C%A8%EB%8B%89" title="히스패닉">히스패닉</a> 인구또한 증가하는 추세이다. 특히 <a href="/wiki/%ED%94%8C%EB%A1%9C%EB%A6%AC%EB%8B%A4_%EC%A3%BC" title="플로리다 주">플로리다 주</a>의 히스패닉 인구는 매우 많고 정치적으로 중요하다.</p>
<h2><span class="mw-headline" id=".EC.A2.85.EA.B5.90">종교</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%AF%B8%EA%B5%AD_%EB%82%A8%EB%B6%80%EC%9D%98_%EB%AC%B8%ED%99%94&amp;action=edit&amp;section=2" title="부분 편집: 종교">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>이 지역은 미국의 <b>바이블 벨트</b>로 불리는 <a href="/wiki/%EA%B8%B0%EB%8F%85%EA%B5%90" title="기독교">기독교</a>의 중심지이다. <a href="/wiki/%EB%B3%B5%EC%9D%8C%EC%A3%BC%EC%9D%98" title="복음주의">복음주의</a> <a href="/wiki/%EA%B0%9C%EC%8B%A0%EA%B5%90" title="개신교">개신교</a>와 <a class="mw-disambig" href="/wiki/%EB%B3%B4%EC%88%98" title="보수">보수</a> <a href="/wiki/%EB%A1%9C%EB%A7%88_%EA%B0%80%ED%86%A8%EB%A6%AD%EA%B5%90%ED%9A%8C" title="로마 가톨릭교회">로마 가톨릭교회</a>가 지배적이며 이는 이 지역의 역사와 관련이 깊다. 대다수의 <a class="mw-disambig" href="/wiki/%EB%B3%B4%EC%88%98" title="보수">보수</a>적인 <a href="/wiki/%EB%B0%B1%EC%9D%B8" title="백인">백인</a>과 <a href="/wiki/%EC%95%84%ED%94%84%EB%A6%AC%EC%B9%B4%EA%B3%84_%EB%AF%B8%EA%B5%AD%EC%9D%B8" title="아프리카계 미국인">흑인</a>들은 <a href="/wiki/%EB%B3%B5%EC%9D%8C%EC%A3%BC%EC%9D%98" title="복음주의">복음주의</a> 기독교를 신봉하며, 자기 종교에 강한 믿음과 자신감을 가지고있다. 이는 아프리카계 미국인(흑인)들의 노예 역사와 가난과 관련있고 보수적인 정치 성향과도 밀접한 관계가 있다. 가톨릭교회는 <a href="/wiki/%ED%94%84%EB%9E%91%EC%8A%A4" title="프랑스">프랑스</a>와 <a href="/wiki/%EC%8A%A4%ED%8E%98%EC%9D%B8" title="스페인">스페인</a>의 식민지였던 해안 지방에 번성하며 <a class="mw-redirect" href="/wiki/%ED%94%8C%EB%A1%9C%EB%A6%AC%EB%8B%A4" title="플로리다">플로리다</a>나 <a class="mw-redirect" href="/wiki/%ED%85%8D%EC%82%AC%EC%8A%A4" title="텍사스">텍사스</a> 등 <a href="/wiki/%ED%9E%88%EC%8A%A4%ED%8C%A8%EB%8B%89" title="히스패닉">히스패닉</a>이 급증하고 있는 여러 지역에서 지배적인 종교이다.</p>
<h2><span class="mw-headline" id=".EB.B0.A9.EC.96.B8">방언</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%AF%B8%EA%B5%AD_%EB%82%A8%EB%B6%80%EC%9D%98_%EB%AC%B8%ED%99%94&amp;action=edit&amp;section=3" title="부분 편집: 방언">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>남부 방언은 모든 영어 방언 중에서 가장 큰 집단을 형성하며 억양의 차이도 표준어와 크다.</p>
<h2><span class="mw-headline" id=".EC.9D.8C.EC.8B.9D">음식</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%AF%B8%EA%B5%AD_%EB%82%A8%EB%B6%80%EC%9D%98_%EB%AC%B8%ED%99%94&amp;action=edit&amp;section=4" title="부분 편집: 음식">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p><a href="/wiki/%EB%A3%A8%EC%9D%B4%EC%A7%80%EC%95%A0%EB%82%98_%EC%A3%BC" title="루이지애나 주">루이지애나 주</a>의 <a href="/wiki/%EC%BC%80%EC%9D%B4%EC%A4%80" title="케이준">케이준</a>과 <a href="/wiki/%ED%81%AC%EB%A6%AC%EC%98%AC" title="크리올">크리올</a> 음식은 전세계적으로 유명하며 <a href="/wiki/%ED%8C%8C%ED%8C%8C%EC%9D%B4%EC%8A%A4" title="파파이스">파파이스</a>의 탄생지로 알려져 있다. <a href="/wiki/%EC%BC%84%ED%84%B0%ED%82%A4_%EC%A3%BC" title="켄터키 주">켄터키 주</a>는 <a href="/wiki/KFC" title="KFC">KFC</a>로 알려진 닭튀김, 켄터키 프라이드 치킨이 유명하다. <a href="/wiki/%ED%85%8C%EB%84%A4%EC%8B%9C_%EC%A3%BC" title="테네시 주">테네시 주</a> <a class="mw-disambig" href="/wiki/%EB%A9%A4%ED%94%BC%EC%8A%A4" title="멤피스">멤피스</a>의 <a class="mw-redirect" href="/wiki/%EB%B0%94%EB%B2%A0%ED%81%90" title="바베큐">바베큐</a>와 <a href="/wiki/%ED%85%8D%EC%82%AC%EC%8A%A4_%EC%A3%BC" title="텍사스 주">텍사스 주</a>는 다양한 스타일의 음식(<a class="mw-redirect" href="/wiki/%ED%85%8D%EC%8A%A4-%EB%A9%95%EC%8A%A4" title="텍스-멕스">텍스-멕스</a>, <a href="/wiki/%EB%A9%95%EC%8B%9C%EC%BD%94" title="멕시코">멕시코</a>식)이 유명하다. 대체로 튀긴 음식이 많으며 상대적으로 짜거나 단 편이다. 여러 인종과 문화의 결합으로 다양하고 독특한 음식 문화를 가지고 있다.</p>
<h2><span class="mw-headline" id=".EB.AC.B8.ED.95.99">문학</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%AF%B8%EA%B5%AD_%EB%82%A8%EB%B6%80%EC%9D%98_%EB%AC%B8%ED%99%94&amp;action=edit&amp;section=5" title="부분 편집: 문학">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<div class="rellink relarticle mainarticle"><img alt="" data-file-height="128" data-file-width="128" height="16" src="//upload.wikimedia.org/wikipedia/commons/thumb/e/ec/Crystal_Clear_app_xmag.svg/16px-Crystal_Clear_app_xmag.svg.png" srcset="//upload.wikimedia.org/wikipedia/commons/thumb/e/ec/Crystal_Clear_app_xmag.svg/24px-Crystal_Clear_app_xmag.svg.png 1.5x, //upload.wikimedia.org/wikipedia/commons/thumb/e/ec/Crystal_Clear_app_xmag.svg/32px-Crystal_Clear_app_xmag.svg.png 2x" width="16"/> 이 부분의 본문은 <a href="/wiki/%EB%AF%B8%EA%B5%AD_%EB%82%A8%EB%B6%80_%EB%AC%B8%ED%95%99" title="미국 남부 문학">미국 남부 문학</a>입니다.</div>
<p>미국 남부 문학의 특징은 문학 작품들이 미국 남부 역사와 <a href="/wiki/%EA%B0%80%EC%A1%B1" title="가족">가족</a>의 중요성, 공동체 의식, 공동체 내에서 개인의 역할과 <a href="/wiki/%EC%A2%85%EA%B5%90" title="종교">종교</a>(<a href="/wiki/%EA%B0%9C%EC%8B%A0%EA%B5%90" title="개신교">개신교</a>), 종교의 압력과 보상, 인종 간의 갈등, 토지와 그것의 산물, 사회 계층 의식, 미국 남부 사투리의 사용에 초점을 맞추고 있다.</p>
<h2><span class="mw-headline" id=".EC.9D.8C.EC.95.85">음악</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%AF%B8%EA%B5%AD_%EB%82%A8%EB%B6%80%EC%9D%98_%EB%AC%B8%ED%99%94&amp;action=edit&amp;section=6" title="부분 편집: 음악">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>이 지역은 <a class="mw-disambig" href="/wiki/%EA%B0%80%EC%8A%A4%ED%8E%A0" title="가스펠">가스펠</a>, <a href="/wiki/%EC%BB%A8%ED%8A%B8%EB%A6%AC_%EC%9D%8C%EC%95%85" title="컨트리 음악">컨트리 음악</a>, <a href="/wiki/%EB%A6%AC%EB%93%AC_%EC%95%A4_%EB%B8%94%EB%A3%A8%EC%8A%A4" title="리듬 앤 블루스">리듬 앤 블루스</a>, <a class="mw-disambig" href="/wiki/%EC%86%8C%EC%9A%B8" title="소울">소울</a>, <a href="/wiki/%EB%A1%9C%ED%81%B0%EB%A1%A4" title="로큰롤">로큰롤</a>, <a href="/wiki/%EB%B8%94%EB%A3%A8%EA%B7%B8%EB%9E%98%EC%8A%A4" title="블루그래스">블루그래스</a>, <a href="/wiki/%EC%9E%AC%EC%A6%88" title="재즈">재즈</a>의 탄생지이며 또한 <a href="/wiki/%EC%96%BC%ED%84%B0%EB%84%88%ED%8B%B0%EB%B8%8C_%EB%A1%9D" title="얼터너티브 록">얼터너티브 록</a>과 <a href="/wiki/%ED%9E%99%ED%95%A9" title="힙합">힙합</a>의 중심지이다. 많은 <a href="/wiki/%EC%9D%8C%EC%95%85_%EC%9E%A5%EB%A5%B4" title="음악 장르">음악 장르</a>들은 <a href="/wiki/%EC%95%84%ED%94%84%EB%A6%AC%EC%B9%B4%EA%B3%84_%EB%AF%B8%EA%B5%AD%EC%9D%B8" title="아프리카계 미국인">아프리카계 미국인</a>의 영향을 받았고 이민계의 참여가 컸다. <a href="/wiki/%EB%89%B4%EC%98%AC%EB%A6%AC%EC%96%B8%EC%8A%A4" title="뉴올리언스">뉴올리언스</a>는 <a href="/wiki/%EC%9E%AC%EC%A6%88" title="재즈">재즈</a>가, <a href="/wiki/%EB%82%B4%EC%8A%88%EB%B9%8C" title="내슈빌">내슈빌</a>은 <a href="/wiki/%EC%BB%A8%ED%8A%B8%EB%A6%AC_%EC%9D%8C%EC%95%85" title="컨트리 음악">컨트리 음악</a>이, <a class="mw-redirect" href="/wiki/%EC%95%A0%ED%8B%80%EB%9E%80%ED%83%80" title="애틀란타">애틀란타</a>는 <a href="/wiki/%ED%9E%99%ED%95%A9" title="힙합">힙합</a>과 <a class="mw-redirect" href="/wiki/%EC%9D%B8%EB%94%94_%EC%9D%8C%EC%95%85" title="인디 음악">인디 음악</a>이 유명하며, <a href="/wiki/%EB%A9%A4%ED%94%BC%EC%8A%A4_(%ED%85%8C%EB%84%A4%EC%8B%9C_%EC%A3%BC)" title="멤피스 (테네시 주)">멤피스 (테네시 주)</a>는 <a href="/wiki/%EC%97%98%EB%B9%84%EC%8A%A4_%ED%94%84%EB%A0%88%EC%8A%AC%EB%A6%AC" title="엘비스 프레슬리">엘비스 프레슬리</a>의 고향이자 <a href="/wiki/%EB%A1%9C%ED%81%B0%EB%A1%A4" title="로큰롤">로큰롤</a>의 중심지이다. 또한 <a href="/wiki/%EC%98%A4%EC%8A%A4%ED%8B%B4_(%ED%85%8D%EC%82%AC%EC%8A%A4_%EC%A3%BC)" title="오스틴 (텍사스 주)">오스틴 (텍사스 주)</a>은 전세계 음악 산업의 중심지이다.</p>
