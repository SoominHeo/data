<p><b>미국 수정 헌법 제1조</b>(The First Amendment 또는 Amendment I)는 종교의 설립을 주선하거나, 자유로운 종교 활동을 방해하거나, <a href="/wiki/%EC%96%B8%EB%A1%A0%EC%9D%98_%EC%9E%90%EC%9C%A0" title="언론의 자유">언론의 자유</a>를 막거나, <a class="mw-redirect" href="/wiki/%EC%B6%9C%ED%8C%90%EC%9D%98_%EC%9E%90%EC%9C%A0" title="출판의 자유">출판의 자유</a>를 침해하거나, 평화로운 <a href="/wiki/%EC%A7%91%ED%9A%8C%EC%9D%98_%EC%9E%90%EC%9C%A0" title="집회의 자유">집회의 자유</a>를 방해하거나, 정부에 대한 탄원의 권리를 막는 어떠한 법 제정도 금지하는 미국의 헌법 수정안이다. <a class="mw-redirect" href="/wiki/%EB%AF%B8%EA%B5%AD_%EA%B6%8C%EB%A6%AC_%EC%9E%A5%EC%A0%84" title="미국 권리 장전">권리 장전</a>을 구성하는 10개의 개정안 중 하나로 <a href="/wiki/1791%EB%85%84" title="1791년">1791년</a> <a href="/wiki/12%EC%9B%94_15%EC%9D%BC" title="12월 15일">12월 15일</a> 채택되었다.</p>
<p>권리 장전은 원래 헌법 비준에 대한 <a class="mw-redirect" href="/wiki/%EB%B0%98%EC%97%B0%EB%B0%A9%EC%A3%BC%EC%9D%98%EC%9E%90" title="반연방주의자">반연방주의자</a>들의 반대를 누그러뜨리는 조치로 제안되었다. 처음에는 수정 헌법 제1조가 의회에 의해 제정되는 법에만 한정되었으며, 그 많은 조항들이 오늘날보다 훨씬 좁게 해석되었다. 1925년 <a class="new" href="/w/index.php?title=%EA%B8%B0%ED%8A%B8%EB%A1%9C_%EB%8C%80_%EB%89%B4%EC%9A%95_%EC%A3%BC&amp;action=edit&amp;redlink=1" title="기트로 대 뉴욕 주 (없는 문서)">기트로 대 뉴욕 주</a>(<i>Gitlow v. New York</i>) 사건으로 시작하여, 연방 대법원은 수정 헌법 제1조를 <a class="mw-redirect" href="/wiki/%EB%AF%B8%EA%B5%AD_%ED%97%8C%EB%B2%95_%EC%88%98%EC%A0%95_%EC%A0%9C14%EC%A1%B0" title="미국 헌법 수정 제14조">수정 헌법 제14조</a>에 명시된 <a class="new" href="/w/index.php?title=%EC%A0%81%EB%B2%95%ED%95%9C_%EC%A0%88%EC%B0%A8&amp;action=edit&amp;redlink=1" title="적법한 절차 (없는 문서)">적법한 절차</a> 조항을 통해 주에 적용시켰다.</p>
<p><a href="/wiki/1947%EB%85%84" title="1947년">1947년</a> <a class="new" href="/w/index.php?title=%EC%97%90%EB%B2%84%EC%8A%A8_%EB%8C%80_%EA%B5%90%EC%9C%A1%EC%9C%84%EC%9B%90%ED%9A%8C&amp;action=edit&amp;redlink=1" title="에버슨 대 교육위원회 (없는 문서)">에버슨 대 교육위원회</a> 사건에서, 연방 대법원은 주와 교회 사이의 분리벽을 요청하기 위해서 <a href="/wiki/%EB%AF%B8%EA%B5%AD_%EA%B1%B4%EA%B5%AD%EC%9D%98_%EC%95%84%EB%B2%84%EC%A7%80" title="미국 건국의 아버지">미국 건국의 아버지</a> <a href="/wiki/%ED%86%A0%EB%A8%B8%EC%8A%A4_%EC%A0%9C%ED%8D%BC%EC%8A%A8" title="토머스 제퍼슨">토머스 제퍼슨</a>의 서신을 꺼내었다. 물론 분리의 경계는 여전히 논란으로 남아있다. 연설을 할 수 있는 권리는 다양한 형태의 정치적 발언, 익명 연설, 모금, <a class="mw-redirect" href="/wiki/%ED%8F%AC%EB%A5%B4%EB%85%B8" title="포르노">포르노</a>, 그리고 학교의 발언 등을 보장하는 20세기와 21세의 계속적인 법원 판결을 통해 급격히 확장되었다. 이러한 것들은 수정 헌법 제1조의 일련의 예외조항으로 규정된 것들이었다. 연방 대법원은 명예훼손을 부담을 막는 영국 연방법의 선례들을 뒤집었고, 가장 유명한 예가 <a href="/wiki/1964%EB%85%84" title="1964년">1964년</a> <a class="mw-redirect" href="/wiki/%EB%89%B4%EC%9A%95%ED%83%80%EC%9E%84%EC%A6%88" title="뉴욕타임즈">뉴욕타임즈</a> 대 설리반 사건이었다. 그러나 상업적인 발언은 수정 헌법 제1조에서 정치적 발언보다 보호를 덜 받으며, 따라서 더 큰 규제를 규제를 따른다.</p>
<p>출판의 자유 조항은 정보와 의견의 출판을 보장하며, 다양한 매체에 적용이 된다. <a href="/wiki/1931%EB%85%84" title="1931년">1931년</a> <a class="new" href="/w/index.php?title=%EB%8B%88%EC%96%B4_%EB%8C%80_%EB%AF%B8%EB%84%A4%EC%86%8C%ED%83%80_%EC%82%AC%EA%B1%B4&amp;action=edit&amp;redlink=1" title="니어 대 미네소타 사건 (없는 문서)">니어 대 미네소타 사건</a>과 <a href="/wiki/1971%EB%85%84" title="1971년">1971년</a> <a class="mw-redirect" href="/wiki/%EB%89%B4%EC%9A%95%ED%83%80%EC%9E%84%EC%A6%88" title="뉴욕타임즈">뉴욕타임즈</a> 대 미국 재판에서 연방 대법원은 수정 제1조가 거의 모든 사건에서 사전 제약(<a href="/wiki/%EA%B2%80%EC%97%B4" title="검열">검열</a>)로 부터의 보호를 보장하고 있다고 판결했다. 탄원 조항은 정부의 조치에 대한 모든 부서에 탄원의 권리를 보장한다. 연방 대법원은 또한 수정 제1조가 <a class="mw-redirect" href="/wiki/%EC%A7%91%ED%9A%8C%EC%99%80_%EA%B2%B0%EC%82%AC%EC%9D%98_%EC%9E%90%EC%9C%A0" title="집회와 결사의 자유">집회와 결사의 자유</a>를 암묵적으로 보장한다고 판결을 했다.</p>

<h2><span class="mw-headline" id=".EC.8B.A0.EC.95.99.EC.9D.98_.EC.A7.84.EC.A0.95.EC.84.B1">신앙의 진정성</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%AF%B8%EA%B5%AD_%EC%88%98%EC%A0%95_%ED%97%8C%EB%B2%95_%EC%A0%9C1%EC%A1%B0&amp;action=edit&amp;section=2" title="부분 편집: 신앙의 진정성">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
