<p><b>미국 육군보병학교</b>(United States Army Infantry School)는 <a href="/wiki/1918%EB%85%84" title="1918년">1918년</a> <a href="/wiki/%EB%AF%B8%EA%B5%AD" title="미국">미국</a> <a href="/wiki/%EC%A1%B0%EC%A7%80%EC%95%84_%EC%A3%BC" title="조지아 주">조지아 주</a> <a class="extiw" href="https://en.wikipedia.org/wiki/Fort_Benning" title="en:Fort Benning">포트 베닝</a>에 첫 설립되어 현재에 이르고 있는 미국 국립 육군 보병 군사교육기관이다.</p>

<h2><span class="mw-headline" id=".EC.97.AD.EC.82.AC">역사</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%AF%B8%EA%B5%AD_%EC%9C%A1%EA%B5%B0_%EB%B3%B4%EB%B3%91%ED%95%99%EA%B5%90&amp;action=edit&amp;section=1" title="부분 편집: 역사">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p><a href="/wiki/1918%EB%85%84" title="1918년">1918년</a> 첫 개교하여 <a href="/wiki/1918%EB%85%84" title="1918년">1918년</a>부터 <a href="/wiki/1920%EB%85%84" title="1920년">1920년</a>까지 <a class="extiw" href="https://en.wikipedia.org/wiki/Charles_S._Farnsworth" title="en:Charles S. Farnsworth">찰스 스튜어트 판즈워스</a>가 <a class="mw-redirect" href="/wiki/%EB%AF%B8%EA%B5%AD_%EC%9C%A1%EA%B5%B0%EB%B3%B4%EB%B3%91%ED%95%99%EA%B5%90" title="미국 육군보병학교">미국 육군보병학교</a> 초대 사령장으로 재임하였고 <a href="/wiki/1920%EB%85%84" title="1920년">1920년</a> <a href="/wiki/%EB%AF%B8%EA%B5%AD_%EC%9C%A1%EA%B5%B0" title="미국 육군">미국 육군</a> 군사 법률 일부 개정으로 인하여 미국 육군보병학교 사령장이었던 <a class="extiw" href="https://en.wikipedia.org/wiki/Charles_S._Farnsworth" title="en:Charles S. Farnsworth">찰스 스튜어트 판즈워스</a>가 <a href="/wiki/1920%EB%85%84" title="1920년">1920년</a>부터 <a href="/wiki/1925%EB%85%84" title="1925년">1925년</a>까지 <a class="mw-redirect" href="/wiki/%EB%AF%B8%EA%B5%AD_%EC%9C%A1%EA%B5%B0%EB%B3%B4%EB%B3%91%ED%95%99%EA%B5%90" title="미국 육군보병학교">미국 육군보병학교</a> 초대 교장으로 연이어 보직 변경 재임하였다.</p>
<h2><span class="mw-headline" id=".EB.B6.84.EA.B5.90">분교</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%AF%B8%EA%B5%AD_%EC%9C%A1%EA%B5%B0_%EB%B3%B4%EB%B3%91%ED%95%99%EA%B5%90&amp;action=edit&amp;section=2" title="부분 편집: 분교">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p><a href="/wiki/1924%EB%85%84" title="1924년">1924년</a> <a href="/wiki/%EB%AF%B8%EA%B5%AD" title="미국">미국</a> <a href="/wiki/%EC%A1%B0%EC%A7%80%EC%95%84_%EC%A3%BC" title="조지아 주">조지아 주</a> <a
