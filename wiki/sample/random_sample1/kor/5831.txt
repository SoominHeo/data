<p><b>미국 육군 정보 및 보안 사령부</b>(<span lang="en" xml:lang="en">United States Army Intelligence and Security Command</span>, 이하 INSCOM)는 <a class="new" href="/w/index.php?title=%EB%AF%B8%EA%B5%AD_%EC%9C%A1%EA%B5%B0_%EB%B3%B4%EC%95%88%EA%B5%AD&amp;action=edit&amp;redlink=1" title="미국 육군 보안국 (없는 문서)">미국 육군 보안국</a>의 후신으로 세워진 <a href="/wiki/%EB%AF%B8%EA%B5%AD_%EC%9C%A1%EA%B5%B0" title="미국 육군">미국 육군</a>의 <a href="/wiki/%EC%B2%A9%EB%B3%B4" title="첩보">첩보</a> 분석 및 <a href="/wiki/%EB%B0%A9%EC%B2%A9" title="방첩">방첩</a> 전담 기능 <a href="/wiki/%EC%82%AC%EB%A0%B9%EB%B6%80" title="사령부">사령부</a>이다. <a href="/wiki/%EB%B2%84%EC%A7%80%EB%8B%88%EC%95%84_%EC%A3%BC" title="버지니아 주">버지니아 주</a> <a class="new" href="/w/index.php?title=%ED%8F%AC%ED%8A%B8_%EB%B2%A8%EB%B3%B4%EC%96%B4&amp;action=edit&amp;redlink=1" title="포트 벨보어 (없는 문서)">포트 벨보어</a>에 주둔하고 있으며, 미국 육군의 사령관들과 정부의 정책 결정권자들에게 직접 보고하는 부대(<span style="font-size: smaller;"><a href="/wiki/%EC%98%81%EC%96%B4" title="영어">영어</a>:</span> <span lang="en" xml:lang="en">Direct Reporting Unit</span>)이다.</p>
<p>INSCOM은 <a href="/wiki/%EB%AF%B8%EA%B5%AD_%EA%B5%AD%EA%B0%80%EC%95%88%EB%B3%B4%EA%B5%AD" title="미국 국가안보국">국가안보국</a>과 더불어 미국의 통합된 <a href="/wiki/%EC%8B%A0%ED%98%B8_%EC%A0%95%EB%B3%B4" title="신호 정보">신호 정보</a> 조직이다. 이 곳에 대응하는 다른 군종의 조직은 <a href="/wiki/%EB%AF%B8%EA%B5%AD_%ED%95%B4%EA%B5%B0" title="미국 해군">해군</a> <a class="new" href="/w/index.php?title=%EB%AF%B8%EA%B5%AD_%ED%95%A8%EB%8C%80_%EC%82%AC%EC%9D%B4%EB%B2%84_%EC%82%AC%EB%A0%B9%EB%B6%80&amp;action=edit&amp;redlink=1" title="미국 함대 사이버 사령부 (없는 문서)">함대 사이버 사령부</a>, <a href="/wiki/%EB%AF%B8%EA%B5%AD_%EA%B3%B5%EA%B5%B0" title="미국 공군">공군</a> <a class="new" href="/w/index.php?title=%EC%A0%9C25%EA%B3%B5%EA%B5%B0_(%EB%AF%B8%EA%B5%AD)&amp;action=edit&amp;redlink=1" title="제25공군 (미국) (없는 문서)">정보, 감시 및 정찰국</a>(AFISRA), <a href="/wiki/%EB%AF%B8%EA%B5%AD_%ED%95%B4%EB%B3%91%EB%8C%80" title="미국 해병대">해병대</a> 암호화 지원 대대(MCSB), <a href="/wiki/%EB%AF%B8%EA%B5%AD_%ED%95%B4%EC%95%88%EA%B2%BD%EB%B9%84%EB%8C%80" title="미국 해안경비대">해안경비대</a> <a class="new" href="/w/index.php?title=%ED%95%B4%EC%95%88%EA%B2%BD%EB%B9%84%EB%8C%80_%EC%A0%95%EB%B3%B4%EB%B6%80&amp;action=edit&amp;redlink=1" title="해안경비대 정보부 (없는 문서)">정보부</a>이 있으며, 이들은 <a class="new" href="/w/index.php?title=%EC%A4%91%EC%95%99_%EC%95%88%EB%B3%B4%EA%B5%AD&amp;action=edit&amp;redlink=1" title="중앙 안보국 (없는 문서)">중앙 안보국</a>(CSS) 예하 기구로 편성되어 있다.</p>
<p>INSCOM은 1977년 1월 1일 앨링턴 회관에서 창설되었다. 전신국인 육군 보안국의 국장이었던 윌리엄 I. 롤야(<span lang="en" xml:lang="en">William I. Rolya</span>) 준장이 사령관직을 그대로 이었다. 같은해 10월 1일 <a href="/wiki/%EB%AF%B8%EA%B5%AD_%EC%9C%A1%EA%B5%B0%EB%B6%80" title="미국 육군부">육군부</a>의 육군 정보국을 통합하였다.</p>
