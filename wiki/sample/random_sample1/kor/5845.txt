<p><b><a href="/wiki/%EB%AF%B8%EA%B5%AD" title="미국">미국</a>의 행정 구역</b>은 50개의 <a href="/wiki/%EB%AF%B8%EA%B5%AD%EC%9D%98_%EC%A3%BC" title="미국의 주">주</a>(State)와 <a href="/wiki/%EC%9B%8C%EC%8B%B1%ED%84%B4_D.C." title="워싱턴 D.C.">워싱턴 D.C.</a>로 이루어져 있다.</p>
<p>각 주는 <a href="/wiki/%EA%B5%B0_(%EB%AF%B8%EA%B5%AD)" title="군 (미국)">군</a>(County, <a class="mw-disambig" href="/wiki/%EC%B9%B4%EC%9A%B4%ED%8B%B0" title="카운티">카운티</a>)로 나뉘는데, <a href="/wiki/%EB%A3%A8%EC%9D%B4%EC%A7%80%EC%95%A0%EB%82%98_%EC%A3%BC" title="루이지애나 주">루이지애나 주</a>에서는 카운티(County) 대신 <a href="/wiki/%EC%82%AC%EB%AA%A9%EA%B5%AC" title="사목구">사목구</a>(Parish)라고 하며, <a href="/wiki/%EC%95%8C%EB%9E%98%EC%8A%A4%EC%B9%B4_%EC%A3%BC" title="알래스카 주">알래스카 주</a>에서는 <a href="/wiki/%EA%B5%AC_(%ED%96%89%EC%A0%95_%EA%B5%AC%EC%97%AD)" title="구 (행정 구역)">구</a>(Borough 또는 Census Area)라고 한다. Census Area에는 구청이 없다.</p>
<p>각 <a class="mw-disambig" href="/wiki/%EC%B9%B4%EC%9A%B4%ED%8B%B0" title="카운티">카운티</a>는 <a class="new" href="/w/index.php?title=%EC%8B%9C_(%EB%AF%B8%EA%B5%AD)&amp;action=edit&amp;redlink=1" title="시 (미국) (없는 문서)">시</a>(City)·<a class="new" href="/w/index.php?title=%EC%9D%8D_(%EB%AF%B8%EA%B5%AD)&amp;action=edit&amp;redlink=1" title="읍 (미국) (없는 문서)">읍</a>(Town)·<a class="new" href="/w/index.php?title=%EB%A6%AC_(%EB%AF%B8%EA%B5%AD)&amp;action=edit&amp;redlink=1" title="리 (미국) (없는 문서)">리</a>(Village)로 나뉘어 있다.</p>
