<p><b>미나미센다이 역</b>(<span style="font-size: smaller;"><a href="/wiki/%EC%9D%BC%EB%B3%B8%EC%96%B4" title="일본어">일본어</a>:</span> <span lang="ja" xml:lang="ja">南仙台駅</span>)은 <a href="/wiki/%EC%9D%BC%EB%B3%B8" title="일본">일본</a> <a href="/wiki/%EB%AF%B8%EC%95%BC%EA%B8%B0_%ED%98%84" title="미야기 현">미야기 현</a> <a href="/wiki/%EC%84%BC%EB%8B%A4%EC%9D%B4_%EC%8B%9C" title="센다이 시">센다이 시</a> <a href="/wiki/%EB%8B%A4%EC%9D%B4%ED%95%98%EC%BF%A0_%EA%B5%AC" title="다이하쿠 구">다이하쿠 구</a>에 있는 <a class="mw-redirect" href="/wiki/%EB%8F%99%EC%9D%BC%EB%B3%B8_%EC%97%AC%EA%B0%9D%EC%B2%A0%EB%8F%84" title="동일본 여객철도">동일본 여객철도</a> <a href="/wiki/%EB%8F%84%ED%98%B8%EC%BF%A0_%EB%B3%B8%EC%84%A0" title="도호쿠 본선">도호쿠 본선</a>의 <a href="/wiki/%EC%B2%A0%EB%8F%84%EC%97%AD" title="철도역">역</a>이다. 여기는 <a href="/wiki/%EC%A1%B0%EB%B0%98_%EC%84%A0" title="조반 선">조반 선</a>과 <a href="/wiki/%EC%84%BC%EB%8B%A4%EC%9D%B4_%EA%B3%B5%ED%95%AD%EC%B2%A0%EB%8F%84_%EC%84%BC%EB%8B%A4%EC%9D%B4_%EA%B3%B5%ED%95%AD%EC%84%A0" title="센다이 공항철도 센다이 공항선">센다이 공항철도 센다이 공항선</a>의 열차도 상호 운행하며, 이용이 가능하다. <a class="mw-redirect" href="/wiki/%EB%8B%A8%EC%84%A0_%EC%8A%B9%EA%B0%95%EC%9E%A5" title="단선 승강장">단선</a> · <a class="mw-redirect" href="/wiki/%EC%84%AC%EC%8B%9D_%EC%8A%B9%EA%B0%95%EC%9E%A5" title="섬식 승강장">섬식</a>의 형태를 합친 복합 구조의 철도 역사이며, <a href="/wiki/%EB%82%98%ED%86%A0%EB%A6%AC_%EC%97%AD" title="나토리 역">나토리 역</a>이 이 역을 관리한다.</p>
