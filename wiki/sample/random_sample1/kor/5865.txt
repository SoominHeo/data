<p><b>미네소타 주</b>(<span style="font-size: smaller;"><a href="/wiki/%EC%98%81%EC%96%B4" title="영어">영어</a>:</span> <span lang="en" xml:lang="en">State of Minnesota</span>)는 <a href="/wiki/%EB%AF%B8%EA%B5%AD" title="미국">미국</a> 중서부에 위치한 <a href="/wiki/%EB%AF%B8%EA%B5%AD%EC%9D%98_%EC%A3%BC" title="미국의 주">주</a>이다. 북쪽으로는 <a href="/wiki/%EC%BA%90%EB%82%98%EB%8B%A4" title="캐나다">캐나다</a> (<a href="/wiki/%EB%A7%A4%EB%8B%88%ED%86%A0%EB%B0%94_%EC%A3%BC" title="매니토바 주">매니토바 주</a>, <a href="/wiki/%EC%98%A8%ED%83%80%EB%A6%AC%EC%98%A4_%EC%A3%BC" title="온타리오 주">온타리오 주</a>)와 맞닿으며, 동쪽으로는 <a href="/wiki/%EC%9C%84%EC%8A%A4%EC%BD%98%EC%8B%A0_%EC%A3%BC" title="위스콘신 주">위스콘신 주</a>와 <a href="/wiki/%EC%8A%88%ED%94%BC%EB%A6%AC%EC%96%B4_%ED%98%B8" title="슈피리어 호">슈피리어 호</a>, 남쪽으로는 <a href="/wiki/%EC%95%84%EC%9D%B4%EC%98%A4%EC%99%80_%EC%A3%BC" title="아이오와 주">아이오와 주</a>, 서쪽으로는 <a href="/wiki/%EB%85%B8%EC%8A%A4%EB%8B%A4%EC%BD%94%ED%83%80_%EC%A3%BC" title="노스다코타 주">노스다코타 주</a>와 <a href="/wiki/%EC%82%AC%EC%9A%B0%EC%8A%A4%EB%8B%A4%EC%BD%94%ED%83%80_%EC%A3%BC" title="사우스다코타 주">사우스다코타 주</a>와 접하며 슈피리어 호 건너로는 <a href="/wiki/%EB%AF%B8%EC%8B%9C%EA%B0%84_%EC%A3%BC" title="미시간 주">미시간 주</a>와 마주하고 있다. 이 주는 <a href="/wiki/%EC%95%8C%EB%9E%98%EC%8A%A4%EC%B9%B4_%EC%A3%BC" title="알래스카 주">알래스카 주</a>를 제외한 미국 본토 주중 가장 북쪽에 위치하고 있다. 미네소타라는 이름은 <a class="mw-redirect" href="/wiki/%EB%8B%A4%EC%BD%94%ED%83%80_%EC%A1%B1" title="다코타 족">다코타 족</a>이 <a href="/wiki/%EB%AF%B8%EB%84%A4%EC%86%8C%ED%83%80_%EA%B0%95" title="미네소타 강">미네소타 강</a>을 <i>mini sota</i> ("흰거품 물" 혹은 "하늘 빛을 띤 물"라는 뜻)라고 불렀던 데서 비롯되었다.</p>
<p>미네소타는 <a href="/wiki/%EB%AF%B8%EA%B5%AD_%EC%A4%91%EC%84%9C%EB%B6%80" title="미국 중서부">미국 중서부</a> 지역에서 가장 큰 주이며 가장 큰 <a href="/wiki/%EB%8C%80%EB%8F%84%EC%8B%9C" title="대도시">대도시</a>는 <a href="/wiki/%EC%8C%8D%EB%91%A5%EC%9D%B4_%EB%8F%84%EC%8B%9C" title="쌍둥이 도시">트윈 시티</a>인데 이 곳은 미네소타 주 인구의 절반 이상이 거주하는 <a href="/wiki/%EB%AF%B8%EB%8B%88%EC%95%A0%ED%8F%B4%EB%A6%AC%EC%8A%A4" title="미니애폴리스">미니애폴리스</a>와 주도인 <a href="/wiki/%EC%84%B8%EC%9D%B8%ED%8A%B8%ED%8F%B4_(%EB%AF%B8%EB%84%A4%EC%86%8C%ED%83%80_%EC%A3%BC)" title="세인트폴 (미네소타 주)">세인트폴</a>및 이를 둘러싼 지역을 일컫는다. 미네소타 주는 미국에서 <a href="/wiki/%EC%8A%A4%EC%B9%B8%EB%94%94%EB%82%98%EB%B9%84%EC%95%84" title="스칸디나비아">스칸디나비아</a> 계통의 주민이 많이 사는 주이기도 하다. 또한 이 주는 미국의 대규모 곡창 지대로서 근 20년간 사용되는 <a href="/wiki/%EC%9E%90%EC%9B%90" title="자원">자원</a>들을 가지고 있는 곳이기도 하다.</p>
<p><a class="mw-disambig" href="/wiki/USS_%EB%AF%B8%EB%84%A4%EC%86%8C%ED%83%80" title="USS 미네소타">USS 미네소타</a>(USS Minnesota)와 잠수함 고퍼(SS Gopher State)는 이 주를 기리는 의미에서 이름 붙여졌다. 미네소타 주의 별칭으로는 <i>만호의 땅 (Land of 10,000 Lakes)</i>과 <i>북극성 주 (North Star State)</i>, <i>땅다람쥐의 주 (The Gopher State)</i>등등이 있다.</p>

<h2><span class="mw-headline" id=".EC.97.AD.EC.82.AC">역사</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%AF%B8%EB%84%A4%EC%86%8C%ED%83%80_%EC%A3%BC&amp;action=edit&amp;section=1" title="부분 편집: 역사">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<h3><span class="mw-headline" id=".EC.B4.88.EA.B8.B0_.EC.97.AD.EC.82.AC">초기 역사</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%AF%B8%EB%84%A4%EC%86%8C%ED%83%80_%EC%A3%BC&amp;action=edit&amp;section=2" title="부분 편집: 초기 역사">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>1600년대의 마지막 절반 시기에 유럽인들이 처음으로 미네소타 지방에 들어왔다. 그들은 북부의 숲들에서 산티 족 혹은 <a class="mw-redirect" href="/wiki/%EB%8B%A4%EC%BD%94%ED%83%80_%EC%A1%B1" title="다코타 족">다코타 족</a>, <a href="/wiki/%EC%88%98_%EC%A1%B1" title="수 족">수 족</a> 인디언들을 찾았다. 수 족은 둥근 천장 모습의 오막집에서 살았고, 실력있는 <a class="mw-redirect" href="/wiki/%EC%82%AC%EB%83%A5%EA%BE%BC" title="사냥꾼">사냥꾼</a>들이었다. <a href="/wiki/1750%EB%85%84" title="1750년">1750년</a>으로 봐서 <a href="/wiki/%EC%98%A4%EC%A7%80%EB%B8%8C%EC%99%80_%EC%A1%B1" title="오지브와 족">오지브와 족</a>이라고도 불리는 <a class="mw-redirect" href="/wiki/%EC%B9%98%ED%8E%98%EC%99%80_%EC%A1%B1" title="치페와 족">치페와 족</a>들이 서부로 이동하면서 미네소타에 들어갔다. 치페와 족들은 지방의 북부 숲들을 차지하였고 수 족들을 남서부로 이동하도록 강요하였다. 수 족들은 방랑자가 되었으며, 2개의 종족들은 수많은 세월 동안에 원수들로 남아있었다.</p>
<h3><span class="mw-headline" id=".ED.83.90.ED.97.98">탐험</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%AF%B8%EB%84%A4%EC%86%8C%ED%83%80_%EC%A3%BC&amp;action=edit&amp;section=3" title="부분 편집: 탐험">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<div class="thumb tleft">
<div class="thumbinner" style="width:222px;"><a class="image" href="/wiki/%ED%8C%8C%EC%9D%BC:Father_Hennepin_at_the_Falls_of_St._Anthony.jpg"><img alt="" class="thumbimage" data-file-height="387" data-file-width="638" height="133" src="//upload.wikimedia.org/wikipedia/commons/thumb/c/c2/Father_Hennepin_at_the_Falls_of_St._Anthony.jpg/220px-Father_Hennepin_at_the_Falls_of_St._Anthony.jpg" srcset="//upload.wikimedia.org/wikipedia/commons/thumb/c/c2/Father_Hennepin_at_the_Falls_of_St._Anthony.jpg/330px-Father_Hennepin_at_the_Falls_of_St._Anthony.jpg 1.5x, //upload.wikimedia.org/wikipedia/commons/thumb/c/c2/Father_Hennepin_at_the_Falls_of_St._Anthony.jpg/440px-Father_Hennepin_at_the_Falls_of_St._Anthony.jpg 2x" width="220"/></a>
<div class="thumbcaption">
<div class="magnify"><a class="internal" href="/wiki/%ED%8C%8C%EC%9D%BC:Father_Hennepin_at_the_Falls_of_St._Anthony.jpg" title="실제 크기로"></a></div>
세인트앤서니 폭포를 발견한 <a class="new" href="/w/index.php?title=%EB%A3%A8%EC%9D%B4_%EC%97%94%EB%8A%90%ED%8C%BD&amp;action=edit&amp;redlink=1" title="루이 엔느팽 (없는 문서)">루이 엔느팽</a> 신부</div>
</div>
</div>
<p>2명의 <a href="/wiki/%ED%94%84%EB%9E%91%EC%8A%A4" title="프랑스">프랑스</a> <a href="/wiki/%EB%AA%A8%ED%94%BC" title="모피">모피</a> 교역인들인 <a class="new" href="/w/index.php?title=%ED%94%BC%EC%97%90%EB%A5%B4_%EC%97%90%EC%8A%A4%ED%94%84%EB%A6%AC_%EB%9D%BC%EB%94%94%EC%86%A1&amp;action=edit&amp;redlink=1" title="피에르 에스프리 라디송 (없는 문서)">피에르 에스프리 라디송</a>과 <a class="new" href="/w/index.php?title=%EB%A9%94%EB%8B%A4%EB%A5%B4_%EC%8A%88%EC%95%84%EB%A5%B4_%EB%93%9C_%EA%B7%B8%EB%A1%9C%EC%84%B8%EC%98%88%EB%A5%B4&amp;action=edit&amp;redlink=1" title="메다르 슈아르 드 그로세예르 (없는 문서)">메다르 슈아르 드 그로세예르</a>는 미네소타에 들어온 첫 유럽인들이었다. 그들은 <a href="/wiki/1660%EB%85%84" title="1660년">1660년</a> 경에 <a href="/wiki/%EC%8A%88%ED%94%BC%EB%A6%AC%EC%96%B4_%ED%98%B8" title="슈피리어 호">슈피리어 호</a>에 있는 현재의 투 하버스 근처에 도착하였다.</p>
<p>또 다른 프랑스인 <a class="new" href="/w/index.php?title=%EB%8B%A4%EB%8B%88%EC%97%98_%EA%B7%B8%EB%A0%88%EC%9D%B4%EC%86%94%EB%A1%B1_%EB%92%AC%EB%A4%BC%ED%8A%B8&amp;action=edit&amp;redlink=1" title="다니엘 그레이솔롱 뒬뤼트 (없는 문서)">다니엘 그레이솔롱 뒬뤼트</a>는 <a href="/wiki/1679%EB%85%84" title="1679년">1679년</a>에 도착하였다. 뒬뤼트는 <a href="/wiki/%ED%83%9C%ED%8F%89%EC%96%91" title="태평양">태평양</a>으로 향하는 수로를 찾는 데 희망을 둔 모험가였다. 그는 슈피리어 호의 서부 기슭에 상륙하고, 그러고나서 미네소타의 내부로 향하여 밀어나갔다. 그는 <a href="/wiki/%EB%A3%A8%EC%9D%B4_14%EC%84%B8" title="루이 14세">루이 14세</a>를 위하여 지방의 전체를 요구하였다.</p>
<p><a href="/wiki/1680%EB%85%84" title="1680년">1680년</a> <a href="/wiki/%EB%B2%A8%EA%B8%B0%EC%97%90" title="벨기에">벨기에</a>의 선교사 <a class="new" href="/w/index.php?title=%EB%A3%A8%EC%9D%B4_%EC%97%94%EB%8A%90%ED%8C%BD&amp;action=edit&amp;redlink=1" title="루이 엔느팽 (없는 문서)">루이 엔느팽</a> 신부가 <a href="/wiki/%EB%AF%B8%EC%8B%9C%EC%8B%9C%ED%94%BC_%EA%B0%95" title="미시시피 강">미시시피 강</a> 상류를 탐험하기 위하여 <a class="mw-redirect" href="/wiki/%EC%9D%BC%EB%A6%AC%EB%85%B8%EC%9D%B4" title="일리노이">일리노이</a> 지방을 출발하였다. 그러나 수 족들이 엔느팽 신부와 그의 동료들을 사로잡았다. 인디언들은 그들을 미네소타로 데려갔다. 포로에 불구하고, 엔느팽 신부는 지방의 거의를 보았다. 그는 자신이 발견하고 이름을 지은 세인트앤서니 폭포가 있는 현재의 <a href="/wiki/%EB%AF%B8%EB%8B%88%EC%95%A0%ED%8F%B4%EB%A6%AC%EC%8A%A4" title="미니애폴리스">미니애폴리스</a>에 속하는 대지를 방문한 첫 유럽인이 되었다. 그 동안에 뒬뤼트가 인디언들이 3명의 백인들을 사로잡았다는 소식을 들었다. 그는 인디언들을 찾아 성공적으로 포로들을 풀어놓기를 요청하였다.</p>
<h3><span class="mw-headline" id=".ED.86.B5.EC.B9.98.EB.A5.BC_.EC.9C.84.ED.95.9C_.EB.B6.84.ED.88.AC">통치를 위한 분투</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%AF%B8%EB%84%A4%EC%86%8C%ED%83%80_%EC%A3%BC&amp;action=edit&amp;section=4" title="부분 편집: 통치를 위한 분투">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p><a href="/wiki/1762%EB%85%84" title="1762년">1762년</a> 프랑스는 미네소타의 거의를 포함하는 미시시피 강의 서부에 있는 모든 대지를 <a href="/wiki/%EC%8A%A4%ED%8E%98%EC%9D%B8" title="스페인">스페인</a>에게 주었다. 그러나 스페인인들은 지방에 탐험 혹은 정착을 시도하지 않았고 프랑스의 덫 사냥꾼들은 지속적으로 거기에서 모피를 수집하였다. <a href="/wiki/1763%EB%85%84" title="1763년">1763년</a> <a href="/wiki/%ED%94%84%EB%A0%8C%EC%B9%98_%EC%9D%B8%EB%94%94%EC%96%B8_%EC%A0%84%EC%9F%81" title="프렌치 인디언 전쟁">프렌치 인디언 전쟁</a>이 벌어지자, 프랑스는 북아메리카에서 경쟁적 요구자인 <a href="/wiki/%EC%98%81%EA%B5%AD" title="영국">영국</a>에게 패하였다. 프랑스는 미네소타 동부를 포함한 미시시피 강의 동부에 있는 대지를 모두 영국에 주었다. 다음 50년 동안에 노스웨스트 회사와 다른 영국의 모피 교역 상사들이 지방에 교역지들을 설립하였다.</p>
<p><a href="/wiki/1783%EB%85%84" title="1783년">1783년</a> <a href="/wiki/%EB%AF%B8%EA%B5%AD_%EB%8F%85%EB%A6%BD_%EC%A0%84%EC%9F%81" title="미국 독립 전쟁">미국 독립 전쟁</a>이 끝나자, 영국은 <a href="/wiki/%EC%98%A4%EB%8C%80%ED%98%B8" title="오대호">오대호</a> 남부와 미시시피 강의 동부였던 대지를 미국에 주었다. 이 막대한 지역은 <a href="/wiki/1787%EB%85%84" title="1787년">1787년</a> <a href="/wiki/%EB%AF%B8%EA%B5%AD_%EC%9D%98%ED%9A%8C" title="미국 의회">미국 의회</a>가 창조한 <a href="/wiki/%EB%85%B8%EC%8A%A4%EC%9B%A8%EC%8A%A4%ED%8A%B8_%EC%A4%80%EC%A3%BC" title="노스웨스트 준주">노스웨스트 준주</a>의 일부가 되었다. 하지만, 영국의 모피 회사들은 지속적으로 지방에서 교역을 하였다. 미국은 <a href="/wiki/1812%EB%85%84" title="1812년">1812년</a> <a class="mw-redirect" href="/wiki/%EB%AF%B8%EC%98%81_%EC%A0%84%EC%9F%81" title="미영 전쟁">미영 전쟁</a> 이후까지 노스웨스트 준주의 완전한 통치를 얻지 않았다.</p>
<h3><span class="mw-headline" id=".EB.A3.A8.EC.9D.B4.EC.A7.80.EC.95.A0.EB.82.98_.EB.A7.A4.EC.9E.85">루이지애나 매입</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%AF%B8%EB%84%A4%EC%86%8C%ED%83%80_%EC%A3%BC&amp;action=edit&amp;section=5" title="부분 편집: 루이지애나 매입">편집</a><span class="mw-editsection-bracket">]</span></span></h3>

<p><a href="/wiki/1800%EB%85%84" title="1800년">1800년</a> <a href="/wiki/%EB%82%98%ED%8F%B4%EB%A0%88%EC%98%B9_%EB%B3%B4%EB%82%98%ED%8C%8C%EB%A5%B4%ED%8A%B8" title="나폴레옹 보나파르트">나폴레옹 보나파르트</a>는 스페인에게 미시시피 강의 서부 지방을 프랑스에게 돌려주도록 강요하였다. 프랑스는 루이지애나라 불리는 이 지방을 <a href="/wiki/1803%EB%85%84" title="1803년">1803년</a> 미국에게 팔았다. 2년 후에 <a class="new" href="/w/index.php?title=%EC%A0%9C%EB%B7%B8%EB%9F%B0_%ED%8C%8C%EC%9D%B4%ED%81%AC&amp;action=edit&amp;redlink=1" title="제뷸런 파이크 (없는 문서)">제뷸런 파이크</a>가 미시시피 강의 상류와 미네소타의 황야를 탐험하는 데 보내졌다.</p>
<p><a href="/wiki/1819%EB%85%84" title="1819년">1819년</a> 미국 육군은 <a href="/wiki/%EB%AF%B8%EB%84%A4%EC%86%8C%ED%83%80_%EA%B0%95" title="미네소타 강">미네소타 강</a>과 미시시피 강이 만나는 점에서 미네소타의 남동부에 일시적인 건물들에 요새를 설립하였다. <a href="/wiki/1820%EB%85%84" title="1820년">1820년</a> 미군들은 거기에 영구적으로 세인트앤서니 요새를 지었다. <a href="/wiki/1825%EB%85%84" title="1825년">1825년</a>에 완공된 이 요새는 스넬링 요새로 다시 이름을 지었다. 그 요새는 군사 요지로는 물론 산업과 문화의 중심지가 되었다. 탐험가들은 가끔 미네소타의 발견되지 않은 부분들을 위하여 세워진 기지로서 스넬링 요새를 이용하였다. 이 탐험가들은 <a class="new" href="/w/index.php?title=%EC%8A%A4%ED%8B%B0%EB%B8%90_%EB%A1%B1&amp;action=edit&amp;redlink=1" title="스티븐 롱 (없는 문서)">스티븐 롱</a>, <a class="new" href="/w/index.php?title=%EC%9C%8C%EB%A6%AC%EC%97%84_%ED%82%A4%ED%8C%85&amp;action=edit&amp;redlink=1" title="윌리엄 키팅 (없는 문서)">윌리엄 키팅</a>과 <a class="new" href="/w/index.php?title=%EC%A1%B0%EC%A7%80_%ED%94%BC%EB%8D%94%EC%8A%A4%ED%86%A0%EB%84%88&amp;action=edit&amp;redlink=1" title="조지 피더스토너 (없는 문서)">조지 피더스토너</a>를 포함한다. <a href="/wiki/1832%EB%85%84" title="1832년">1832년</a> <a class="new" href="/w/index.php?title=%ED%97%A8%EB%A6%AC_%EC%8A%A4%EC%BF%A8%ED%81%AC%EB%9E%98%ED%94%84%ED%8A%B8&amp;action=edit&amp;redlink=1" title="헨리 스쿨크래프트 (없는 문서)">헨리 스쿨크래프트</a>는 미시시피 강의 근원이 되는 <a href="/wiki/%EC%9D%B4%ED%83%80%EC%8A%A4%EC%B9%B4_%ED%98%B8" title="이타스카 호">이타스카 호</a>를 발견하여 이름을 지었다.</p>
<p>1830년대에 세인트크루아 협곡에서 벌목이 시작되었다. <a href="/wiki/1837%EB%85%84" title="1837년">1837년</a> 수 족과 치페와 족들은 자신들이 세인트크루아 강 주변의 벌목 지역이라고 주장한 지역을 미국 정부에 팔았다. 벌목공과 정착자들이 곧 그 지역으로 이주하였다. 정착자들은 후에 미니애폴리스와 병합된 세인트앤서니, 세인트폴과 스틸워터의 3개 타운들을 창립하였다.</p>
<h3><span class="mw-headline" id=".EC.A4.80.EC.A3.BC_.EC.8B.9C.EA.B8.B0">준주 시기</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%AF%B8%EB%84%A4%EC%86%8C%ED%83%80_%EC%A3%BC&amp;action=edit&amp;section=6" title="부분 편집: 준주 시기">편집</a><span class="mw-editsection-bracket">]</span></span></h3>

<p>세월 동안에 미네소타의 일부들은 일리노이, 인디애나, 아이오와, 미시간, 미주리와 위스콘신의 영토들에 속하였고, 루이지애나의 영토와 구역으로 속하였다. <a href="/wiki/1849%EB%85%84" title="1849년">1849년</a> <a href="/wiki/3%EC%9B%94_3%EC%9D%BC" title="3월 3일">3월 3일</a> 의회는 미네소타 영토를 창조하였다. 그 남부, 북부와 동부의 경계 지역들은 오늘날 주의 경계들과 같았다. 서부 경계지역은 <a href="/wiki/%EB%AF%B8%EC%A3%BC%EB%A6%AC_%EA%B0%95" title="미주리 강">미주리 강</a>과 <a class="new" href="/w/index.php?title=%ED%99%94%EC%9D%B4%ED%8A%B8%EC%96%B4%EC%8A%A4_%EA%B0%95&amp;action=edit&amp;redlink=1" title="화이트어스 강 (없는 문서)">화이트어스 강</a>으로 뻗었다. <a class="new" href="/w/index.php?title=%EC%95%8C%EB%A0%89%EC%82%B0%EB%8D%94_%EB%9E%A8%EC%A7%80&amp;action=edit&amp;redlink=1" title="알렉산더 램지 (없는 문서)">알렉산더 램지</a>는 영토의 초대 총독으로 임명되었다. 영토가 되었을 때에 미네소타에는 대략 4,000명의 백인들이 살았다.</p>
<p><a href="/wiki/1851%EB%85%84" title="1851년">1851년</a> 미국 정부로부터 압력에 놓였던 수 족들은 미시시피 강의 서부에 있는 막대한 지역에 자신들의 권리들을 포기하는 2개의 조약을 맺었다. 이 대지의 대부분은 미네소타의 남부였다. 이 새로운 부유한 영토는 백인 정착지와 새로 들어오는 사람들에게 열렸다.</p>

<p><a href="/wiki/1858%EB%85%84" title="1858년">1858년</a> <a href="/wiki/5%EC%9B%94_11%EC%9D%BC" title="5월 11일">5월 11일</a> 의회는 미네소타를 32번째 주로서 합중국에 가입하는 데 인정하였다. 미네소타의 투표인들은 자신들의 주의 초대 주지사로서 <a class="new" href="/w/index.php?title=%ED%97%A8%EB%A6%AC_%EC%8B%9C%EB%B8%94%EB%A6%AC&amp;action=edit&amp;redlink=1" title="헨리 시블리 (없는 문서)">헨리 시블리</a>를 선출하였다. 시블리는 미국 모피 회사의 대리를 지내왔고, 미네소타 영토의 창조를 위하여 일해왔다. 주로 승격될 당시에 미네소타의 인구는 대략 150,000명이었다.</p>
<p><a href="/wiki/1861%EB%85%84" title="1861년">1861년</a> <a class="mw-disambig" href="/wiki/%EB%82%A8%EB%B6%81_%EC%A0%84%EC%9F%81" title="남북 전쟁">남북 전쟁</a>이 시작되자, 미네소타 주는 북군을 위하여 군인들을 제공한 첫 주가 되었다. <a href="/wiki/1862%EB%85%84" title="1862년">1862년</a> <a href="/wiki/8%EC%9B%94" title="8월">8월</a>에는 다코타 분쟁이 일어났다. 보류와 거의 굶주리는 처지에 모인 인디언들이 정착자들을 공격하기 시작하였다. 주립 국민군이 싸움을 끝내기 전에 수백명의 정착자들이 살해되었다. 많은 인디언들은 주에서 달아나고, 38명의 인디언들이 교수형을 당하였다. 남아있던 수 족들은 미네소타로부터 다른 보호 구역으로 이주하였다.</p>
<h3><span class="mw-headline" id=".EC.82.B0.EC.97.85.EC.A0.81_.EB.B0.9C.EC.A0.84">산업적 발전</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%AF%B8%EB%84%A4%EC%86%8C%ED%83%80_%EC%A3%BC&amp;action=edit&amp;section=8" title="부분 편집: 산업적 발전">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<div class="thumb tleft">
<div class="thumbinner" style="width:222px;"><a class="image" href="/wiki/%ED%8C%8C%EC%9D%BC:Duluth_Ore_Docks.jpg"><img alt="" class="thumbimage" data-file-height="3497" data-file-width="4478" height="172" src="//upload.wikimedia.org/wikipedia/commons/thumb/9/97/Duluth_Ore_Docks.jpg/220px-Duluth_Ore_Docks.jpg" srcset="//upload.wikimedia.org/wikipedia/commons/thumb/9/97/Duluth_Ore_Docks.jpg/330px-Duluth_Ore_Docks.jpg 1.5x, //upload.wikimedia.org/wikipedia/commons/thumb/9/97/Duluth_Ore_Docks.jpg/440px-Duluth_Ore_Docks.jpg 2x" width="220"/></a>

덜러스 항구에서 <a href="/wiki/%EC%B2%A0%EA%B4%91%EC%84%9D" title="철광석">철광석</a>을 실어나르는 배</div>

<p><a href="/wiki/1865%EB%85%84" title="1865년">1865년</a> 남북 전쟁이 끝난 후, 산업적 발전이 미네소타 주에서 빠르게 일어났다. 철도들이 주를 가로지르면서 확장되었고, 낡은 수 족의 사냥지들은 <a href="/wiki/%EB%B0%80" title="밀">밀</a>을 심는 대지가 되었다. 제분소들이 밀 생산 지방을 통하여 생겨났으나 대부분은 미니애폴리스 지역에 있었다. 미니애폴리스는 제분업의 도시로 알려졌고, 1880년대부터 1930년대까지 국가에서 제1의 <a href="/wiki/%EB%B0%80%EA%B0%80%EB%A3%A8" title="밀가루">밀가루</a> 생산지가 되었다.</p>
<p>미네소타는 새로 들어오는 사람들을 끌어들이기 위하여 원기 왕성한 차도들을 수행하였다. 주 정부와 철도들은 미네소타에서 기회들을 묘사하는 팜플렛들을 유럽에 보냈다. 1870년대, 1880년대와 1890년대에 수천명의 이민들, 특히 <a href="/wiki/%EB%8F%85%EC%9D%BC" title="독일">독일</a>, <a href="/wiki/%EB%85%B8%EB%A5%B4%EC%9B%A8%EC%9D%B4" title="노르웨이">노르웨이</a>와 <a href="/wiki/%EC%8A%A4%EC%9B%A8%EB%8D%B4" title="스웨덴">스웨덴</a>인들이 미네소타 주에 정착하였다.</p>
<p>1800년대 후반의 뛰어난 이벤트는 풍부한 <a href="/wiki/%EC%B2%A0%EA%B4%91%EC%84%9D" title="철광석">철광석</a> 자원의 개발이었다. <a href="/wiki/1884%EB%85%84" title="1884년">1884년</a> 첫 광석은 버밀리언 광산 지대에서 배로 날라졌다. <a href="/wiki/1890%EB%85%84" title="1890년">1890년</a>에는 근로자들이 <a class="new" href="/w/index.php?title=%EB%A6%AC%EC%96%B4%EB%8B%88%EB%8B%A4%EC%8A%A4_%EB%A9%94%EB%A6%BF&amp;action=edit&amp;redlink=1" title="리어니다스 메릿 (없는 문서)">리어니다스 메릿</a>과 메사비 광산 지대에 있는 철광산 근처에서 광석을 발견한 그의 7명의 형제들에 의하여 고용되었다. 2년 후에 그들의 투기적 사업에 다른 친척들을 관련시킨 후에 메릿 형제들은 메사비 광산 지대에로부터 광석의 첫 체굴량을 배에 실어날랐다.</p>
<p><a href="/wiki/1889%EB%85%84" title="1889년">1889년</a> <a class="new" href="/w/index.php?title=%EC%9C%8C%EB%A6%AC%EC%97%84_%EB%A9%94%EC%9D%B4%EC%9A%94&amp;action=edit&amp;redlink=1" title="윌리엄 메이요 (없는 문서)">윌리엄 메이요</a>와 그의 아들 윌리엄과 찰스는 <a href="/wiki/%EB%A1%9C%EC%B2%B4%EC%8A%A4%ED%84%B0_(%EB%AF%B8%EB%84%A4%EC%86%8C%ED%83%80_%EC%A3%BC)" title="로체스터 (미네소타 주)">로체스터</a>에 메이요 클리닉을 설립하였다. 그 명예는 재빠르게 퍼져갔고, 메이요 가족은 정통적 의료 센터로 바꾸었다. 이 클리닉은 세계에서 지도적인 의료 연구 센터가 되었다.</p>
<p><a href="/wiki/1894%EB%85%84" title="1894년">1894년</a> 숲에 대화재가 발생해서 미네소타 동부의 대략 1,000km2를 휩쓸었다. 그 화재는 힝클리와 샌드스톤의 마을들에 퍼졌다. 400명 이상의 주민들이 사망하고 100만 달러 이상의 재산 피해가 발생했다.</p>
<h3><span class="mw-headline" id="1900.EB.85.84.EB.8C.80_.EC.B4.88.EB.B0.98">1900년대 초반</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%AF%B8%EB%84%A4%EC%86%8C%ED%83%80_%EC%A3%BC&amp;action=edit&amp;section=9" title="부분 편집: 1900년대 초반">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p><a href="/wiki/1911%EB%85%84" title="1911년">1911년</a> 철광석의 첫 수송이 카이어나 광산 지대를 떠났다. <a href="/wiki/1915%EB%85%84" title="1915년">1915년</a> <a href="/wiki/12%EC%9B%94" title="12월">12월</a>에 덜러스에 있는 큰 제철소에서 첫 운영이 시작되었다. <a href="/wiki/1917%EB%85%84" title="1917년">1917년</a> 미국이 <a href="/wiki/%EC%A0%9C1%EC%B0%A8_%EC%84%B8%EA%B3%84_%EB%8C%80%EC%A0%84" title="제1차 세계 대전">제1차 세계 대전</a>에 참전한 후, 미네소타의 제품들을 위한 무거운 요구들이 있었다. 밀과 다른 곡물들의 거대한 수가 군인들을 먹이는 데 재배되었다. <a href="/wiki/1917%EB%85%84" title="1917년">1917년</a>과 <a href="/wiki/1918%EB%85%84" title="1918년">1918년</a> 동안에 철광석의 생산은 총으로 거의 9천만 톤이었다.</p>
<p><a href="/wiki/1918%EB%85%84" title="1918년">1918년</a> 미네소타는 또다른 비참한 화재에 의하여 맞부디쳐졌다. 강한 바람들이 작은 화재의 몇몇을 부채질하여 칼터과 세인트루이스 카운티들의 큰 지역들을 휩쓰는 하나의 큰 화재로 일으켰다. 그로 인하여 400명 이상의 주민들이 사망하였고, 2천 5백만 달러 가치의 재산이 파괴되었다.</p>
<p>1890년대와 1900년대 초반 동안에 많은 미네소타 주의 농부들이 <a href="/wiki/%ED%98%91%EB%8F%99%EC%A1%B0%ED%95%A9" title="협동조합">협동조합</a>에 가입하였다. 그들은 자신들의 소유적 재정과 창고 서비스들을 마련하고 자신들의 생산품들을 위한 교통을 위하여 함께 가입하였다. 농부들은 철도, 은행과 곡물 회사들이 이 서비스들을 위하여 너무나 책임을 지었다고 믿었다. 1920년대 동안에는 새로운 농업-노동당이 농부들을 성원하였다. <a href="/wiki/1931%EB%85%84" title="1931년">1931년</a> <a class="new" href="/w/index.php?title=%ED%94%8C%EB%A1%9C%EC%9D%B4%EB%93%9C_%EC%98%AC%EC%8A%A8&amp;action=edit&amp;redlink=1" title="플로이드 올슨 (없는 문서)">플로이드 올슨</a>이 첫 농업-노동당의 총독이 되었다.</p>
<p>1930년대의 <a href="/wiki/%EB%8C%80%EA%B3%B5%ED%99%A9" title="대공황">대공황</a>은 미네소타 주를 강하게 쳤다. 실업이 도시들에 널리 퍼졌으며, 철광산 근로자들의 대략 70 퍼센트가 자신들의 직업을 잃었다. 1920년대 후반에 떨어지기 시작한 농장 소득이 지속적으로 떨어졌다. 주 정부는 대공황을 싸우는 데 많은 단계들을 취하였고 연방 대리들은 고용과 구제를 마련하기 위하여 세워졌다.</p>
<h3><span class="mw-headline" id="1900.EB.85.84.EB.8C.80_.EC.A4.91.EB.B0.98">1900년대 중반</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%AF%B8%EB%84%A4%EC%86%8C%ED%83%80_%EC%A3%BC&amp;action=edit&amp;section=10" title="부분 편집: 1900년대 중반">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p><a href="/wiki/%EC%A0%9C2%EC%B0%A8_%EC%84%B8%EA%B3%84_%EB%8C%80%EC%A0%84" title="제2차 세계 대전">제2차 세계 대전</a>이 일어나는 동안에 미네소타 주의 경제는 회복되었다. 주의 벌목과 광업들이 군사들을 위한 원료들의 거대한 양을 생산하였다. 그러나 1950년대 후반에 광석을 위한 요구를 하면서 고급의 철광석 공급이 갑자기 몰락하였다. 산업은 쇠퇴하고 몇몇의 광산들이 폐문하였다. 결과로서 주의 철강 산업은 저급의 태코나이트 암을 개발하기 시작하였다.</p>
<p><a href="/wiki/1964%EB%85%84" title="1964년">1964년</a> 투표인들은 철강 산업에 투자를 상승시킨 주 헌법에 수정안을 찬성하였다. 이른바 태코나이트 암 수정안은 25년 동안에 태코나이트 암에 세금들이 다른 생산품들에 세금들보타 더 높은 비율에 상승하지 않는 것을 보증하였다. 이전으로 봐서, 철광 체굴 회사들은 높은 비율에 세금을 내왔고, 생산자들은 태코아니트 광산들을 개발하는 계획을 미루었다. 태코나이트 암 수정안의 경과 후에, 생산자들은 태코나이트 광산들에 1억 달러 이상을 투자하였다.</p>
<p>많은 새로운 제조업들이 1900년대 중반에 미네소타 주에서 운영되기 시작하였다. 이 산업들의 제품들은 항공우주선의 비품, 화학품, <a href="/wiki/%EC%BB%B4%ED%93%A8%ED%84%B0" title="컴퓨터">컴퓨터</a>, <a href="/wiki/%EC%A0%84%EC%9E%90" title="전자">전자</a> 용품과 무거운 기계업을 포함한다. 이 개발들은 주의 졍제적 근거에 다양화를 추가하였다.</p>
<p>다른 주들에서 같이 미네소타 주에는 농장과 농장인들의 수들이 감소하였다. 시골 지역에서 온 가족들의 다수가 도시들로 이주하였다. <a href="/wiki/1950%EB%85%84" title="1950년">1950년</a>으로 봐서, 처음으로 도시 인구가 시골 인구보타 더 많이 늘어났다. <a href="/wiki/1964%EB%85%84" title="1964년">1964년</a> 연방 법원은 그 입법의 구역들이 도시 인구에게 동등한 대표하도록 주에 명령을 내렸다.</p>
<p>농업-노동당은 <a href="/wiki/1944%EB%85%84" title="1944년">1944년</a> <a href="/wiki/%EB%AF%BC%EC%A3%BC%EB%8B%B9_(%EB%AF%B8%EA%B5%AD)" title="민주당 (미국)">민주당</a>에 가입하여 민주-농업-노동당을 형성하였다. <a href="/wiki/1975%EB%85%84" title="1975년">1975년</a> 미네소타의 <a href="/wiki/%EA%B3%B5%ED%99%94%EB%8B%B9_(%EB%AF%B8%EA%B5%AD)" title="공화당 (미국)">공화당</a>은 그 이름을 미네소타 독립 공화당으로 바꾸었다. <a href="/wiki/1995%EB%85%84" title="1995년">1995년</a> 당은 다시 공화당으로 바꾸었다.</p>
<h3><span class="mw-headline" id="1900.EB.85.84.EB.8C.80_.ED.9B.84.EB.B0.98">1900년대 후반</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%AF%B8%EB%84%A4%EC%86%8C%ED%83%80_%EC%A3%BC&amp;action=edit&amp;section=11" title="부분 편집: 1900년대 후반">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>주의 태코나이트 광산들은 공해와 수질 오염의 근거가 되었다. <a href="/wiki/1978%EB%85%84" title="1978년">1978년</a> 미네소타 대법원은 실버 베이의 보호 광업 회사에게 그 광산에서 공해 조절 본위들을 만나도록 명령을 내렸다. <a href="/wiki/1980%EB%85%84" title="1980년">1980년</a> 회사는 대지의 오염 처분 지대를 설립하여 더 이상 폐물이 슈피리어 호 근처로 흘러가지 않도록 하였다. 1980년대와 1990년대에 주 정부는 비지니스들이 자신들의 각자 공해 조절 노력들을 증가시키기 위하여 비지니스들과 일하였다.</p>
<p>1900년대 중반의 미네소타에서 시작된 경제의 다양화가 1990년대에 증가하였다. <a href="/wiki/%EA%B8%88%EC%9C%B5" title="금융">금융</a>, <a href="/wiki/%EB%B3%B4%ED%97%98" title="보험">보험</a>과 <a href="/wiki/%EB%B6%80%EB%8F%99%EC%82%B0" title="부동산">부동산</a> 같은 서비스업들의 증가는 주의 경제적 힘에 공헌하였다. 의료 장치들의 생산은 제조업 부문에서 확장적인 활동을 하였다.</p>
<h3><span class="mw-headline" id="21.EC.84.B8.EA.B8.B0">21세기</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%AF%B8%EB%84%A4%EC%86%8C%ED%83%80_%EC%A3%BC&amp;action=edit&amp;section=12" title="부분 편집: 21세기">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p><a href="/wiki/1998%EB%85%84" title="1998년">1998년</a> 미네소타의 투표인들은 <a href="/wiki/%EC%A0%9C%EC%8B%9C_%EB%B2%A4%ED%88%AC%EB%9D%BC" title="제시 벤투라">제시 벤투라</a>를 주지사로 선출하였다. 벤투라는 미국에서 첫 <a class="mw-disambig" href="/wiki/%EA%B0%9C%ED%98%81%EB%8B%B9" title="개혁당">개혁당</a> 후보로서 주 전체의 사무직을 이기는 데 처음이었다. 그는 <a href="/wiki/2000%EB%85%84" title="2000년">2000년</a> 개혁당을 탈퇴하여 무소속 정당인이 되었다. 그는 재선을 향하지 않았고, 그의 임기는 <a href="/wiki/2003%EB%85%84" title="2003년">2003년</a>에 끝났다.</p>
<p>경궤조 시스템이 <a href="/wiki/2004%EB%85%84" title="2004년">2004년</a> 중순에 미니애폴리스에서 운영되기 시작하였으며, 주에서 그 첫 시스템이었다.</p>
<p><a href="/wiki/2007%EB%85%84" title="2007년">2007년</a> <a href="/wiki/8%EC%9B%94" title="8월">8월</a> 미니애폴리스에서 미시시피 강을 가로지른 인터스테이트 35번 다리의 부분이 떨어져, 13명의 사람들이 사망하고 대략 100명이 부상을 당하였다. 다리의 재편은 <a href="/wiki/2008%EB%85%84" title="2008년">2008년</a> <a href="/wiki/9%EC%9B%94" title="9월">9월</a>에 완공되었다.</p>
<p><a href="/wiki/2008%EB%85%84" title="2008년">2008년</a> 미네소타 주에서의 <a href="/wiki/%EB%AF%B8%EA%B5%AD_%EC%83%81%EC%9B%90" title="미국 상원">미국 상원</a> 경쟁은 <a href="/wiki/11%EC%9B%94_4%EC%9D%BC" title="11월 4일">11월 4일</a> 선거 후에 알려지지 않은 달들로 남는 결과의 가장 가까운 시기였다. 처음의 결과들은 2명의 후보 공화당 상원 <a class="new" href="/w/index.php?title=%EB%86%88_%EC%BD%9C%EB%A8%BC&amp;action=edit&amp;redlink=1" title="놈 콜먼 (없는 문서)">놈 콜먼</a>과 그의 민주당 도전자 <a class="new" href="/w/index.php?title=%EC%95%8C_%ED%94%84%EB%9E%AD%EC%BC%84&amp;action=edit&amp;redlink=1" title="알 프랭켄 (없는 문서)">알 프랭켄</a> 사이의 몇 100개의 투표들 만의 다른 점을 보였다. <a href="/wiki/2009%EB%85%84" title="2009년">2009년</a> <a href="/wiki/1%EC%9B%94" title="1월">1월</a>에 미네소타 법에 의하여 요구된 투표들의 재계표가 나온 후에 선거원들은 프랭켄이 우승자로서 선언하였다. 하지만, 콜먼은 법원에서 재계표의 결과들을 도전하는 소송을 제기하였다. <a href="/wiki/4%EC%9B%94" title="4월">4월</a>에 3명의 재판 배심원들은 선거가 공명정대하지 않게 치루어졌다고 한 콜먼의 주장을 퇴거시켰다. 배심원들은 프랭켄이 선거의 우승자로서 증명되도록 추천하였으나, 콜먼은 주의 대법원에 지배를 호소하였다. <a href="/wiki/6%EC%9B%94" title="6월">6월</a>에 미네소타 대법원은 재계표의 결과들이 유효한 지배를 하고, 프랭켄을 상원 경쟁의 우승자로 선언하였다.</p>

<h2><span class="mw-headline" id=".EA.B2.BD.EC.A0.9C">경제</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%AF%B8%EB%84%A4%EC%86%8C%ED%83%80_%EC%A3%BC&amp;action=edit&amp;section=13" title="부분 편집: 경제">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<h3><span class="mw-headline" id=".EC.84.9C.EB.B9.84.EC.8A.A4.EC.97.85">서비스업</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%AF%B8%EB%84%A4%EC%86%8C%ED%83%80_%EC%A3%BC&amp;action=edit&amp;section=14" title="부분 편집: 서비스업">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<div class="thumb tleft">
<div class="thumbinner" style="width:202px;"><a class="image" href="/wiki/%ED%8C%8C%EC%9D%BC:Best_Buy_Corporate_Campus.jpg"><img alt="" class="thumbimage" data-file-height="463" data-file-width="1051" height="88" src="//upload.wikimedia.org/wikipedia/commons/thumb/8/88/Best_Buy_Corporate_Campus.jpg/200px-Best_Buy_Corporate_Campus.jpg" srcset="//upload.wikimedia.org/wikipedia/commons/thumb/8/88/Best_Buy_Corporate_Campus.jpg/300px-Best_Buy_Corporate_Campus.jpg 1.5x, //upload.wikimedia.org/wikipedia/commons/thumb/8/88/Best_Buy_Corporate_Campus.jpg/400px-Best_Buy_Corporate_Campus.jpg 2x" width="200"/></a>

<a href="/wiki/%EB%B2%A0%EC%8A%A4%ED%8A%B8_%EB%B0%94%EC%9D%B4" title="베스트 바이">베스트 바이</a>의 본사</div>

<p>서비스업은 미네소타의 국내 총생산과 고용의 대락 80 퍼센트를 마련한다. 이 산업들의 대부분은 <a href="/wiki/%EB%AF%B8%EB%8B%88%EC%95%A0%ED%8F%B4%EB%A6%AC%EC%8A%A4" title="미니애폴리스">미니애폴리스</a>-<a href="/wiki/%EC%84%B8%EC%9D%B8%ED%8A%B8%ED%8F%B4_(%EB%AF%B8%EB%84%A4%EC%86%8C%ED%83%80_%EC%A3%BC)" title="세인트폴 (미네소타 주)">세인트폴</a> 지역에 있다.</p>
<p>주도인 세인트폴은 정부 활동의 중심지이다. 미니애폴리스에는 <a href="/wiki/%EB%AF%B8%EA%B5%AD_%EC%9D%98%ED%9A%8C" title="미국 의회">미국 의회</a>에 의하여 설립된 12개의 연방 은행들 중의 하나인 나인스 연방 준비 은행의 본부가 있다. 국가에서 제1의 소매상 회사들의 3개인 <a href="/wiki/%EB%B2%A0%EC%8A%A4%ED%8A%B8_%EB%B0%94%EC%9D%B4" title="베스트 바이">베스트 바이</a>, 타게트 주식 회사와 슈퍼밸류 스토어가 트윈 시티 지역에 기지를 두었다. 트윈 시티 지역에는 많은 <a href="/wiki/%ED%98%B8%ED%85%94" title="호텔">호텔</a>, 식당과 소매 시설들의 본거지이다.</p>
<p>세계에서 가장 큰 의료 센터들 중의 하나인 메이요 클리닉이 <a href="/wiki/%EB%A1%9C%EC%B2%B4%EC%8A%A4%ED%84%B0_(%EB%AF%B8%EB%84%A4%EC%86%8C%ED%83%80_%EC%A3%BC)" title="로체스터 (미네소타 주)">로체스터</a>에 있다. 트윈 시티는 몇몇의 개인적 보건소들과 <a class="mw-redirect" href="/wiki/%EC%9C%A0%EA%B8%B0%EC%B2%B4" title="유기체">유기체</a> 이식의 지도자인 <a href="/wiki/%EB%AF%B8%EB%84%A4%EC%86%8C%ED%83%80_%EB%8C%80%ED%95%99%EA%B5%90" title="미네소타 대학교">미네소타 대학교</a>의 근거지이다.</p>
<h3><span class="mw-headline" id=".EC.A0.9C.EC.A1.B0.EC.97.85">제조업</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%AF%B8%EB%84%A4%EC%86%8C%ED%83%80_%EC%A3%BC&amp;action=edit&amp;section=15" title="부분 편집: 제조업">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p><a href="/wiki/%EC%BB%B4%ED%93%A8%ED%84%B0" title="컴퓨터">컴퓨터</a>와 <a href="/wiki/%EC%A0%84%EC%9E%90" title="전자">전자</a> 제품들은 미네소타의 지도적인 제조품들이다. 이 제품들은 주로 트윈 시티와 로체스터 지역들에서 제조되고 있다. 주의 전자 제품들은 미네소타 대학교와 제휴하는 보건 연구 시설들에 의하여 개발된 의료 장치들을 포함한다.</p>
<p><a class="mw-redirect" href="/wiki/%EC%8B%9D%ED%92%88" title="식품">식품</a>도 또한 중요한 산업이다. 정육업과 <a href="/wiki/%EB%82%99%EB%86%8D" title="낙농">낙농</a> 제조업은 주의 가장중요한 가공업이다. 이 활동들은 미네소타 주의 남부에 중심지를 두고 있다. 미네소타 주는 또한 지도적인 <a href="/wiki/%EB%83%89%EB%8F%99%EC%8B%9D%ED%92%88" title="냉동식품">냉동식품</a>과 <a href="/wiki/%EC%B1%84%EC%86%8C" title="채소">채소</a>와 <a href="/wiki/%EA%B3%BC%EC%9D%BC" title="과일">과일</a>의 통조림의 생산주이다. 냉동식품 생산은 미니애폴리스-세인트폴 지역에 집중되어 있다. 과일과 채소의 통조림은 미네소타 주의 남부에서 생산된다. <a href="/wiki/%EC%82%AC%ED%83%95%EB%AC%B4" title="사탕무">사탕무</a> 정제소들은 레드 강 협곡에서 운영된다.</p>
<p>트윈 시티 지역은 또한 농기구, 건설 기계와 합성 금속 제품들을 생산한다. 합성 금속 제품들은 기계 조립 제품들과 <a href="/wiki/%EB%AC%B4%EA%B8%B0" title="무기">무기</a>와 <a href="/wiki/%ED%83%84%EC%95%BD" title="탄약">탄약</a>을 포함한다. <a href="/wiki/%EC%84%9D%EC%9C%A0" title="석유">석유</a> 정제소들은 세인트폴 지역에 있다. <a href="/wiki/%EC%A2%85%EC%9D%B4" title="종이">종이</a>와 <a href="/wiki/%ED%8E%84%ED%94%84" title="펄프">펄프</a> 제재소들은 주로 주의 북부에 있다. 설상차와 일반 도로 밖에서 쓰이는 차량의 세계에서 지도적인 생산 회사들 중의 하나인 <a class="mw-disambig" href="/wiki/ATV" title="ATV">ATV</a>가 메디나에서 본사를 두고 있으며, 다국적 기업 <a href="/wiki/3M" title="3M">3M</a>의 본사가 메이플우드에 있다.</p>
<h3><span class="mw-headline" id=".EB.86.8D.EC.97.85">농업</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%AF%B8%EB%84%A4%EC%86%8C%ED%83%80_%EC%A3%BC&amp;action=edit&amp;section=16" title="부분 편집: 농업">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<div class="thumb tright">
<div class="thumbinner" style="width:222px;"><a class="image" href="/wiki/%ED%8C%8C%EC%9D%BC:Minnesota-corn-20030826.jpg"><img alt="" class="thumbimage" data-file-height="768" data-file-width="1024" height="165" src="//upload.wikimedia.org/wikipedia/commons/thumb/6/63/Minnesota-corn-20030826.jpg/220px-Minnesota-corn-20030826.jpg" srcset="//upload.wikimedia.org/wikipedia/commons/thumb/6/63/Minnesota-corn-20030826.jpg/330px-Minnesota-corn-20030826.jpg 1.5x, //upload.wikimedia.org/wikipedia/commons/thumb/6/63/Minnesota-corn-20030826.jpg/440px-Minnesota-corn-20030826.jpg 2x" width="220"/></a>

미네소타 주의 <a href="/wiki/%EC%98%A5%EC%88%98%EC%88%98" title="옥수수">옥수수</a>밭</div>

<p>미네소타 주는 해마다의 농장 소득에서 지도적인 주들 중의 랭킹에 들어와있다. 농장 지대는 주의 대략 절반을 덮고 있다.</p>
<p>많은 미네소타의 농부들은 자신들이 높은 가격들을 얻는 데 가공하여 팔고 자신들의 생산품들을 <a class="mw-disambig" href="/wiki/%EB%B0%B0" title="배">배</a>에 실어나르는 데 함께 가입한 <a href="/wiki/%ED%98%91%EB%8F%99%EC%A1%B0%ED%95%A9" title="협동조합">협동조합</a>을 통하여 자신들의 생산물을 판다. 협동조합의 대부분은 낙농업 협동조합이다. 하지만 많은 이들은 곡식과 가축들을 취급한다.</p>
<p>곡물은 미네소타의 농장 소득의 대략 절반을 차지한다. <a href="/wiki/%EC%98%A5%EC%88%98%EC%88%98" title="옥수수">옥수수</a>는 주의 가장 가치적인 곡물이며, <a href="/wiki/%EB%8C%80%EB%91%90" title="대두">대두</a>가 그 다음이다. 미네소타 주는 주의 남부를 통하여 자라는 이 두 곡물의 지도적인 생산지이다. 미네소타 주는 제1의 <a href="/wiki/%EC%82%AC%ED%83%95%EB%AC%B4" title="사탕무">사탕무</a> 생산지이며, 주로 주의 북부와 서부에서 자라고 있다. 레드 강 협곡의 북서부는 <a href="/wiki/%EB%B0%80" title="밀">밀</a>의 생산지이다.</p>
<p>미네소타에서 자라는 다른 곡물들은 <a href="/wiki/%EB%B3%B4%EB%A6%AC" title="보리">보리</a>, <a href="/wiki/%EC%95%84%EB%A7%88" title="아마">아마</a>의 씨와 <a href="/wiki/%EA%B7%80%EB%A6%AC" title="귀리">귀리</a>이다. <a href="/wiki/%EB%8B%B9%EA%B7%BC" title="당근">당근</a>, <a class="mw-redirect" href="/wiki/%EC%99%84%EB%91%90%EC%BD%A9" title="완두콩">완두콩</a>과 사탕 옥수수는 주의 지도적인 채소 생산물이며, <a class="mw-redirect" href="/wiki/%EC%82%AC%EA%B3%BC" title="사과">사과</a>가 제일 많이 나는 과일이다.</p>
<p>가축과 축산물은 주의 해마다 농작물로부터 들어오는 소득의 대략 절반을 마련한다. <a href="/wiki/%EB%8F%BC%EC%A7%80" title="돼지">돼지</a>는 주의 가장 자치적인 축산물이며, 주의 남부에서 가장 많이 사육하고 있다. <a href="/wiki/%EC%A0%96%EC%86%8C" title="젖소">젖소</a>와 <a href="/wiki/%EC%9C%A1%EC%9A%B0" title="육우">육우</a>도 주에서 축산 소득의 중요한 근원이다. 미네소타 주는 제1의 <a href="/wiki/%EC%9A%B0%EC%9C%A0" title="우유">우유</a> 생산주들 중의 하나로 들어와있다. 미네소타의 우유로는 거의 <a href="/wiki/%EC%B9%98%EC%A6%88" title="치즈">치즈</a>와 <a href="/wiki/%EC%95%84%EC%9D%B4%EC%8A%A4%ED%81%AC%EB%A6%BC" title="아이스크림">아이스크림</a>을 만든다. 주는 <a href="/wiki/%EB%8B%AC%EA%B1%80" title="달걀">달걀</a> 생산과 <a href="/wiki/%EC%B9%A0%EB%A9%B4%EC%A1%B0" title="칠면조">칠면조</a> 사육을 하기도 한다.</p>
<h3><span class="mw-headline" id=".EA.B4.91.EC.97.85">광업</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%AF%B8%EB%84%A4%EC%86%8C%ED%83%80_%EC%A3%BC&amp;action=edit&amp;section=17" title="부분 편집: 광업">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p><a href="/wiki/%EC%B2%A0%EA%B4%91%EC%84%9D" title="철광석">철광석</a>은 미네소타의 광업 소득의 대부분을 마련한다. 미네소타 주는 철광석 생산에서 이끌어지는 주이다. 그러나 미네소타 경제에 철광석 체굴의 중요함이 1900년대 후반 동안에 거대하게 쇠퇴하였다. 남아있는 철광석의 대부분은 태코나이트 암이라 불리며 주의 북동부에서 체굴된다.</p>
<p>미네소타의 중부에 있는 채석장들은 보통 <a href="/wiki/%ED%99%94%EA%B0%95%EC%95%94" title="화강암">화강암</a>을 산출한다. <a href="/wiki/%EC%84%9D%ED%9A%8C%EC%95%94" title="석회암">석회암</a>은 주의 남부에서 넓은 매장량으로부터 가져간다. <a href="/wiki/%EB%AA%A8%EB%9E%98" title="모래">모래</a>와 <a href="/wiki/%EC%9E%90%EA%B0%88" title="자갈">자갈</a>은 주의 전역을 통하여 생산된다. 부분적으로 자연 붕괴되어 농업과 상업적 사용들의 다양성을 가진 공장 물질인 <a class="mw-redirect" href="/wiki/%ED%86%A0%ED%83%84" title="토탄">토탄</a>은 주의 북부에서 발견된다.</p>
<h3><span class="mw-headline" id=".EC.88.98.EC.82.B0.EC.97.85">수산업</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%AF%B8%EB%84%A4%EC%86%8C%ED%83%80_%EC%A3%BC&amp;action=edit&amp;section=18" title="부분 편집: 수산업">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>오락적과 상업적인 수산업은 주의 주요 활동이다. 버펄로 물고기, <a href="/wiki/%EC%9E%89%EC%96%B4" title="잉어">잉어</a>, <a href="/wiki/%EB%A9%94%EA%B8%B0" title="메기">메기</a>, 하얀 <a href="/wiki/%EC%86%A1%EC%96%B4" title="송어">송어</a> 등을 포함한 가장 가치적인 물고기들은 <a href="/wiki/%EB%AF%B8%EC%8B%9C%EC%8B%9C%ED%94%BC_%EA%B0%95" title="미시시피 강">미시시피 강</a>에서 잡히며, 호수 송어, 호수 <a href="/wiki/%EC%B2%AD%EC%96%B4" title="청어">청어</a>, 식용 <a href="/wiki/%EB%B9%99%EC%96%B4" title="빙어">빙어</a>, 노란 <a href="/wiki/%EC%B0%BD%EA%BC%AC%EC%B9%98" title="창꼬치">창꼬치</a> 등은 <a href="/wiki/%EC%8A%88%ED%94%BC%EB%A6%AC%EC%96%B4_%ED%98%B8" title="슈피리어 호">슈피리어 호</a>에서 주로 잡히고 있다.</p>
<h2><span class="mw-headline" id=".EC.A3.BC.EB.AF.BC">주민</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%AF%B8%EB%84%A4%EC%86%8C%ED%83%80_%EC%A3%BC&amp;action=edit&amp;section=19" title="부분 편집: 주민">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<div class="thumb tleft">
<div class="thumbinner" style="width:302px;"><a class="image" href="/wiki/%ED%8C%8C%EC%9D%BC:2008-0712-MPLS-panorama.JPG"><img alt="" class="thumbimage" data-file-height="2449" data-file-width="7560" height="97" src="//upload.wikimedia.org/wikipedia/commons/thumb/e/e4/2008-0712-MPLS-panorama.JPG/300px-2008-0712-MPLS-panorama.JPG" srcset="//upload.wikimedia.org/wikipedia/commons/thumb/e/e4/2008-0712-MPLS-panorama.JPG/450px-2008-0712-MPLS-panorama.JPG 1.5x, //upload.wikimedia.org/wikipedia/commons/thumb/e/e4/2008-0712-MPLS-panorama.JPG/600px-2008-0712-MPLS-panorama.JPG 2x" width="300"/></a>
<div class="thumbcaption">
<div class="magnify"><a class="internal" href="/wiki/%ED%8C%8C%EC%9D%BC:2008-0712-MPLS-panorama.JPG" title="실제 크기로"></a></div>
<a href="/wiki/%EB%AF%B8%EB%8B%88%EC%95%A0%ED%8F%B4%EB%A6%AC%EC%8A%A4" title="미니애폴리스">미니애폴리스</a>의 파노라마</div>
</div>
</div>
<p><a href="/wiki/2000%EB%85%84" title="2000년">2000년</a> <a class="mw-redirect" href="/wiki/%EB%AF%B8%EA%B5%AD_%EC%9D%B8%EA%B5%AC_%EC%A1%B0%EC%82%AC%EA%B5%AD" title="미국 인구 조사국">미국 인구 조사국</a>은 미네소타 주에 4,919,479명의 인구가 있다고 보고하였다. 주의 인구는 <a href="/wiki/1990%EB%85%84" title="1990년">1990년</a>으로 본 4,375,099명에서 12.5 퍼센트 증가하였다. <a href="/wiki/2000%EB%85%84" title="2000년">2000년</a> 인구 조사국에 의하면 미네소타는 50개의 주들 사이에 인구의 21위로 들어와있다.</p>
<p>미네소타 주민의 대략 70 퍼센트는 주의 7개 메트로폴리탄 지역들의 하나에 살고 있다. 대락 60 퍼센트는 <a href="/wiki/%EB%AF%B8%EB%8B%88%EC%95%A0%ED%8F%B4%EB%A6%AC%EC%8A%A4" title="미니애폴리스">미니애폴리스</a>-<a href="/wiki/%EC%84%B8%EC%9D%B8%ED%8A%B8%ED%8F%B4_(%EB%AF%B8%EB%84%A4%EC%86%8C%ED%83%80_%EC%A3%BC)" title="세인트폴 (미네소타 주)">세인트폴</a>-<a href="/wiki/%EB%B8%94%EB%A3%A8%EB%B0%8D%ED%84%B4_(%EB%AF%B8%EB%84%A4%EC%86%8C%ED%83%80_%EC%A3%BC)" title="블루밍턴 (미네소타 주)">블루밍턴</a> 메트로폴리탄 지역에 살고 있다.</p>
<p>미니애폴리스는 주에서 가장 큰 도시이며, 주도이자 두번째로 가장 큰 도시 세인트폴과 인접해있다. 이 트윈 시티는 주의 지도적인 문화, 금융과 상업의 중심지이다. 인구의 순서로 봐서 주의 다른 가장 큰 도시들은 덜러스, <a href="/wiki/%EB%A1%9C%EC%B2%B4%EC%8A%A4%ED%84%B0_(%EB%AF%B8%EB%84%A4%EC%86%8C%ED%83%80_%EC%A3%BC)" title="로체스터 (미네소타 주)">로체스터</a>, 블루밍턴, 브루클린 파크와 플리머스이다.</p>
<p><a href="/wiki/%EC%95%84%ED%94%84%EB%A6%AC%EC%B9%B4%EA%B3%84_%EB%AF%B8%EA%B5%AD%EC%9D%B8" title="아프리카계 미국인">아프리카계 미국인</a>은 5 퍼센트, <a href="/wiki/%EC%95%84%EC%8B%9C%EC%95%84" title="아시아">아시아</a>인과 <a href="/wiki/%ED%9E%88%EC%8A%A4%ED%8C%A8%EB%8B%89" title="히스패닉">히스패닉</a>은 4 퍼센트 이상을 차지한다. 주민들의 가장 큰 단체는 1800년대와 1900년대 동안에 미네소타에 정착한 <a href="/wiki/%ED%95%80%EB%9E%80%EB%93%9C" title="핀란드">핀란드</a>, <a href="/wiki/%EB%8F%85%EC%9D%BC" title="독일">독일</a>, <a href="/wiki/%EB%85%B8%EB%A5%B4%EC%9B%A8%EC%9D%B4" title="노르웨이">노르웨이</a>와 <a href="/wiki/%EC%8A%A4%EC%9B%A8%EB%8D%B4" title="스웨덴">스웨덴</a> 이민의 계통들이다. 또한 <a href="/wiki/%EB%8D%B4%EB%A7%88%ED%81%AC" title="덴마크">덴마크</a>, <a href="/wiki/%EC%95%84%EC%9D%BC%EB%9E%9C%EB%93%9C" title="아일랜드">아일랜드</a>, <a href="/wiki/%ED%8F%B4%EB%9E%80%EB%93%9C" title="폴란드">폴란드</a>, <a href="/wiki/%EB%9F%AC%EC%8B%9C%EC%95%84" title="러시아">러시아</a>와 <a href="/wiki/%EC%98%81%EA%B5%AD" title="영국">영국</a> 계통의 주민들도 살고 있다.</p>
