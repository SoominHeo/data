<p><b>미니 메트로</b>(영어: Mini Metro)는 <a class="new" href="/w/index.php?title=Dinosaur_Polo_Club&amp;action=edit&amp;redlink=1" title="Dinosaur Polo Club (없는 문서)">Dinosaur Polo Club</a>이 제작하고 Dinosaur Polo Club, <a class="new" href="/w/index.php?title=AGM_PLAYISM&amp;action=edit&amp;redlink=1" title="AGM PLAYISM (없는 문서)">AGM PLAYISM</a>가 배급하는 <a class="new" href="/w/index.php?title=%EC%9D%B8%EB%94%94&amp;action=edit&amp;redlink=1" title="인디 (없는 문서)">인디</a>, <a class="mw-redirect" href="/wiki/%EC%A0%84%EB%9E%B5_%EC%8B%9C%EB%AE%AC%EB%A0%88%EC%9D%B4%EC%85%98" title="전략 시뮬레이션">전략 시뮬레이션</a> 게임으로 세계 유명 도시의 지하철 노선을 구상해보는 게임이다. PC버전은 2014년 8월 11일에 출시했고, <a class="mw-redirect" href="/wiki/%EC%95%B1%EC%8A%A4%ED%86%A0%EC%96%B4" title="앱스토어">앱스토어</a>(영어: <a class="mw-redirect" href="/wiki/App_Store" title="App Store">App Store</a>)에서는 2015년 12월 25일 업데이트 되었고, <a class="mw-redirect" href="/wiki/%ED%94%8C%EB%A0%88%EC%9D%B4%EC%8A%A4%ED%86%A0%EC%96%B4" title="플레이스토어">플레이스토어</a>에서 2016년 10월 25일 업데이트 되었다. <a class="mw-disambig" href="/wiki/%EC%8A%A4%ED%8C%80" title="스팀">스팀</a>(Steam)에서 제공하는 게임이다. 역과 역 사이에 노선을 그려서 열차 운행을 개시함으로써 게임을 시작한다. 새로운 역이 등장하게되면 노선을 수정해서 노선도를 효율적으로 유지해야한다. 한정된 자원을 어느 곳에 사용할 것인지 결정하여 도시를 얼마나 오래 유지할 수 있는가가 관건이다. 게임 <a class="mw-disambig" href="/wiki/%ED%8A%B8%EB%A0%88%EC%9D%BC%EB%9F%AC" title="트레일러">트레일러</a><sup class="reference" id="cite_ref-3"><a href="#cite_note-3">[3]</a></sup><sup class="reference" id="cite_ref-4"><a href="#cite_note-4">[4]</a></sup>도 창의적으로 잘 만들어져 있다.</p>

<h2><span class="mw-headline" id=".EA.B2.8C.EC.9E.84_.EC.A7.84.ED.96.89_.EB.B0.A9.EC.8B.9D">게임 진행 방식</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%AF%B8%EB%8B%88_%EB%A9%94%ED%8A%B8%EB%A1%9C&amp;action=edit&amp;section=1" title="부분 편집: 게임 진행 방식">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>처음 플레이어의 도시는 3개의 지하철 역과 함께 시작한다. 각각의 역 들을 선으로 연결하여 노선을 구성한다. 역은 다양한 도형들로 구성되어 있는데 게임이 진행되면서 도형이 하나하나 늘어가는 방식이다. 역 옆에는 승객들도 도형으로 표현되는데 이 도형이 승객들이 원하는 역이므로 같은 모양의 역으로 승객을 실어나르면 된다. 기본적으로 <b>동그라미</b>, <b>세모</b>, <b>네모</b>가 있으며, <b>십자가</b>, <b>마름모</b>, <b>별표</b> 등 특이한 도형들도 등장한다. 게임이 진행될 수록 도시는 빠르게 성장하여 연결이 필요한 지하철 역들이 계속해서 생겨난다. 지하철 역이 늘어날 수록 승객들도 늘어나게 되는데 이 승객들이 신속하게 탑승하여 원하는 역으로 갈 수 있도록 노선을 구성해야한다. 역이 과수요 상태에 빠지지 않도록 계속해서 노선을 연장하거나 추가해야하는데 역들 중 단 하나의 역이라도 과수요 상태에 빠지게 되면 그대로 역이 폐쇄 조치에 취해지며 게임이 끝나게 된다. 여담이지만, 게임을 플레이해보면 <a class="new" href="/w/index.php?title=%EC%9A%B0%EB%A6%AC%EB%82%98%EB%9D%BC%EC%9D%98_%EC%A7%80%ED%95%98%EC%B2%A0_%EB%85%B8%EC%84%A0%EB%8F%84&amp;action=edit&amp;redlink=1" title="우리나라의 지하철 노선도 (없는 문서)">우리나라의 지하철 노선도</a>가 얼마나 머리를 쓴 것인지 감탄하게 된다.</p>
<p><a class="image" href="/wiki/%ED%8C%8C%EC%9D%BC:Mini_Metro_screenshot_5.png" title="Mini Metro screenshot 5"><img alt="Mini Metro screenshot 5" data-file-height="1080" data-file-width="1920" height="338" src="//upload.wikimedia.org/wikipedia/commons/thumb/e/e8/Mini_Metro_screenshot_5.png/600px-Mini_Metro_screenshot_5.png" srcset="//upload.wikimedia.org/wikipedia/commons/thumb/e/e8/Mini_Metro_screenshot_5.png/900px-Mini_Metro_screenshot_5.png 1.5x, //upload.wikimedia.org/wikipedia/commons/thumb/e/e8/Mini_Metro_screenshot_5.png/1200px-Mini_Metro_screenshot_5.png 2x" width="600"/></a></p>
<h2><span class="mw-headline" id=".EC.B6.94.EA.B0.80_.EC.9E.90.EC.82.B0">추가 자산</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%AF%B8%EB%8B%88_%EB%A9%94%ED%8A%B8%EB%A1%9C&amp;action=edit&amp;section=2" title="부분 편집: 추가 자산">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>도시가 성장할 수록 추가적인 자원이 주기적으로 주어진다.</p>
<ul>
<li>열차</li>
</ul>
<p>게임 내 시간으로 매주 일요일이 지나게 되면 열차가 주어진다. 열차는 플레이어가 직접 원하는 노선에 끌어다 놓으면 자동으로 운행된다.</p>
<ul>
<li>객차</li>
</ul>
<p>객차는 열차 뒤에 연결할 수 있어서 한번 열차가 지나갈 때 더 많은 승객을 태울 수 있도록 만든다.</p>
<ul>
<li>터널</li>
</ul>
<p>맵에는 굵은 하늘색 선이 지나가게 되는데 이 것은 게임 내에서 강을 의미한다. 이러한 강을 건너는 노선을 만들기 위해서는 터널이 필요하다. 또는, 산악 지형을 통과하기 위해서도 터널 자원이 필요하다.</p>
<ul>
<li>노선</li>
</ul>
<p>새로운 색깔의 노선을 추가할 수 있도록 해준다.</p>
<ul>
<li>특정 보급품</li>
</ul>
<p>오사카 맵에는 <a href="/wiki/%EC%8B%A0%EC%B9%B8%EC%84%BC" title="신칸센">신칸센</a>이 특정 보급품으로 지급되고, 몇몇 맵에서는 교차역이 지급된다.</p>
<h2><span class="mw-headline" id=".EB.A7.B5">맵</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%AF%B8%EB%8B%88_%EB%A9%94%ED%8A%B8%EB%A1%9C&amp;action=edit&amp;section=3" title="부분 편집: 맵">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p><a href="/wiki/%EB%9F%B0%EB%8D%98" title="런던">런던</a>, <a class="mw-disambig" href="/wiki/%ED%8C%8C%EB%A6%AC" title="파리">파리</a>, <a href="/wiki/%EB%89%B4%EC%9A%95" title="뉴욕">뉴욕</a>, <a href="/wiki/%EB%B2%A0%EB%A5%BC%EB%A6%B0" title="베를린">베를린</a>, <a href="/wiki/%EB%A9%9C%EB%B2%84%EB%A5%B8" title="멜버른">멜버른</a>, <a href="/wiki/%ED%99%8D%EC%BD%A9" title="홍콩">홍콩</a>, <a href="/wiki/%EC%98%A4%EC%82%AC%EC%B9%B4" title="오사카">오사카</a>, <a href="/wiki/%EC%83%81%ED%8A%B8%ED%8E%98%ED%85%8C%EB%A5%B4%EB%B6%80%EB%A5%B4%ED%81%AC" title="상트페테르부르크">상트페테르부르크</a>, <a href="/wiki/%EB%AA%AC%ED%8A%B8%EB%A6%AC%EC%98%AC" title="몬트리올">몬트리올</a>, <a href="/wiki/%EC%83%81%ED%8C%8C%EC%9A%B8%EB%A3%A8" title="상파울루">상파울루</a>, <a href="/wiki/%EC%B9%B4%EC%9D%B4%EB%A1%9C" title="카이로">카이로</a>, <a class="mw-disambig" href="/wiki/%EC%98%A4%ED%81%B4%EB%9E%9C%EB%93%9C" title="오클랜드">오클랜드</a>로 구성되어있다. 일반적으로 왼쪽부터 차례대로 난이도 순서이다. 다음 맵을 플레이하기 위해서는 그 전의 맵을 원하는 점수를 충족시켜야만 플레이할 수 있다. 예를 들어 오사카맵을 플레이하기 위해서는 홍콩맵을 플레이하여 몇 점이상을 넘겨야 해금된다. 각 도시마다 맵의 형태가 다르고 난이도가 올라갈 수록 특이한 도형들도 생겨나 플레이어를 당황하게 만든다.</p>
<p><a class="image" href="/wiki/%ED%8C%8C%EC%9D%BC:Mini_Metro_screenshot_1.png" title="Mini Metro screenshot 1"><img alt="Mini Metro screenshot 1" data-file-height="1080" data-file-width="1920" height="281" src="//upload.wikimedia.org/wikipedia/commons/thumb/2/2c/Mini_Metro_screenshot_1.png/500px-Mini_Metro_screenshot_1.png" srcset="//upload.wikimedia.org/wikipedia/commons/thumb/2/2c/Mini_Metro_screenshot_1.png/750px-Mini_Metro_screenshot_1.png 1.5x, //upload.wikimedia.org/wikipedia/commons/thumb/2/2c/Mini_Metro_screenshot_1.png/1000px-Mini_Metro_screenshot_1.png 2x" width="500"/></a></p>
<h2><span class="mw-headline" id=".EB.82.9C.EC.9D.B4.EB.8F.84">난이도</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%AF%B8%EB%8B%88_%EB%A9%94%ED%8A%B8%EB%A1%9C&amp;action=edit&amp;section=4" title="부분 편집: 난이도">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>맵에도 어느 정도 난이도가 부여되어 있지만 설정에서 난이도를 따로 설정할 수도 있다.</p>
<ul>
<li>일반</li>
</ul>
<p>일반난이도는 그냥 아무런 변화가 없는 일반적인 평범한 게임 난이도라고 생각하면 된다. 게임오버가 존재한다.</p>
<ul>
<li>무한</li>
</ul>
<p>무한난이도는 말 그대로 무한으로 일반 모드와 달리 게임오버가 없어서 오직 원활하고 효율적인 노선을 구성하면된다.</p>
<ul>
<li>극한</li>
</ul>
<p>극한난이도는 일반 난이도의 상위 난이도로 노선을 한번 그리면 수정할 수가 없어서 진짜 실제상황과 같이 깊이 생각을 해본 후 노선을 그려야한다.</p>
<h2><span class="mw-headline" id=".ED.8E.B8.EC.9D.98.EC.84.B1">편의성</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%AF%B8%EB%8B%88_%EB%A9%94%ED%8A%B8%EB%A1%9C&amp;action=edit&amp;section=5" title="부분 편집: 편의성">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>미니 메트로에는 <a class="mw-redirect" href="/wiki/%EC%83%89%EC%95%BD" title="색약">색약</a> 모드와 야간 모드가 있다. 아무래도 초기 설정은 인터페이스가 하얀 화면을 배경으로 하기 때문에 야간에 플레이하게 되면 눈이 아플 수 있는데 야간 모드로 설정하게되면 검정 배경으로 바꿀 수 있어서 눈의 피로를 덜어준다. 색약 모드는 점과 색을 구별하기 어려운 플레이어들을 고려하여 만든 설정이다. 개발진이 모든 유저를 배려해 주었다는 점에서 매우 만족스럽다는 평이 많다.</p>
<h2><span class="mw-headline" id=".ED.8F.89.EA.B0.80">평가</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%AF%B8%EB%8B%88_%EB%A9%94%ED%8A%B8%EB%A1%9C&amp;action=edit&amp;section=6" title="부분 편집: 평가">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<ul>
<li>Kill Screen</li>
</ul>
<p>미니 메트로는 우리로 하여금 대중교통을 숭고하게 만든다.(Mini Metro makes mass transportation sublime)<sup class="reference" id="cite_ref-5"><a href="#cite_note-5">[5]</a></sup></p>
<ul>
<li>Boing Boing</li>
</ul>
<p>당신이 안심하는 순간 게임은 끝난다.. 그 즉시 다시 시작해야한다.(It's so satisfying that the moment one game ends ... you'll immediately want to start again)<sup class="reference" id="cite_ref-6"><a href="#cite_note-6">[6]</a></sup><br/></p>
<p><b>유저 평가</b>는 최근 30일 내에 평가도 압도적으로 긍정적이고 (평가 332 개) 전체적으로도 압도적으로 긍정적 (평가 3,481 개)<sup class="reference" id="cite_ref-7"><a href="#cite_note-7">[7]</a></sup></p>
<h2><span class="mw-headline" id=".ED.8C.81">팁</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%AF%B8%EB%8B%88_%EB%A9%94%ED%8A%B8%EB%A1%9C&amp;action=edit&amp;section=7" title="부분 편집: 팁">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>1. 먼저 특이한 도형<sup class="reference" id="cite_ref-8"><a href="#cite_note-8">[8]</a></sup>의 역이 등장하게되면 그 역을 환승역으로 해주는 것이 좋다. 여러 군데에서 특이한 도형의 역을 오려고 하기 때문에 환승 횟수를 줄여준다.<br/></p>
<p>2. <a href="/wiki/%EC%88%9C%ED%99%98%EC%84%A0" title="순환선">순환선</a>을 잘 이용해야한다. 맵의 끝에서 끝까지 가는 노선도 물론 있어야하지만 순환선을 통해 승객을 가까운 역으로 빠르게 보낼 수 있다.<br/></p>
<p>3. 같은 노선에 같은 도형이 너무 겹치지 않도록 해야한다. 만약 역이 세모, 세모, 네모, 세모 이렇게 구성되어 있으면 동그라미를 가고싶은 승객은 내리지 않고 계속 열차 자리를 차지 하기 때문에 도형이 번갈아 가면서 나오도록 노선을 구성해주는 것이 좋다.</p>
<p>4. 승객이 환승할 때 매우 현실적인 판단으로 이루어 지는데, 만약 탑승하는 노선에 원하는 역이 있더라도 환승해서 갈 때 더 빠르다면 내려서 갈아탄다. 이러한 문제 때문에 게임오버 되는 플레이어들이 의외로 많은데 이때는 어쩔 수 없이 근처 플랫폼을 끊거나 다른 플랫폼을 이어줘야 한다.</p>
<p>5. 게임 중 속도 조절을 키보드의 좌.우 로 조절 할 수있다.</p>
