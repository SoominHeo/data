<h1 class="firstHeading" id="firstHeading" lang="ko">미르 아크바르 카이바르</h1>
<div class="mw-body-content" id="bodyContent">

<div id="contentSub"></div>
<div class="mw-jump" id="jump-to-nav">
					이동:					<a href="#mw-head">둘러보기</a>, 					<a href="#p-search">검색</a>
</div>
<div class="mw-content-ltr" dir="ltr" id="mw-content-text" lang="ko"><script>function mfTempOpenSection(id){var block=document.getElementById("mf-section-"+id);block.className+=" open-block";block.previousSibling.className+=" open-block";}</script><p><b>미르 아크바르 카이바르</b>(Mir Akbar Khyber, <a href="/wiki/1925%EB%85%84" title="1925년">1925년</a> ~ <a href="/wiki/1978%EB%85%84" title="1978년">1978년</a> <a href="/wiki/4%EC%9B%94_17%EC%9D%BC" title="4월 17일">4월 17일</a>)은 <a href="/wiki/%EC%95%84%ED%94%84%EA%B0%80%EB%8B%88%EC%8A%A4%ED%83%84" title="아프가니스탄">아프가니스탄</a>의 정치인이자 <a class="new" href="/w/index.php?title=%EC%95%84%ED%94%84%EA%B0%80%EB%8B%88%EC%8A%A4%ED%83%84_%EC%9D%B8%EB%AF%BC_%EB%AF%BC%EC%A3%BC%EB%8B%B9&amp;action=edit&amp;redlink=1" title="아프가니스탄 인민 민주당 (없는 문서)">아프가니스탄 인민 민주당</a>의 지도자이다.</p>
<p><a href="/wiki/1947%EB%85%84" title="1947년">1947년</a> 군사 고등학교를 졸업하였으나 <a href="/wiki/1950%EB%85%84" title="1950년">1950년</a> 혁명 활동 중 수감되었고 교육부에서 고용하다가 <a href="/wiki/1965%EB%85%84" title="1965년">1965년</a> <a href="/wiki/%ED%8C%8D%ED%8B%B0%EC%95%BC_%EC%A3%BC" title="팍티야 주">팍티야 주</a>의 폭동으로 실각했다.</p>
<p><a href="/wiki/%EC%B9%B4%EB%B6%88" title="카불">카불</a>로 돌아온 뒤 신문 편집자가 되었고 <a class="new" href="/w/index.php?title=%EC%95%84%ED%94%84%EA%B0%80%EB%8B%88%EC%8A%A4%ED%83%84_%EC%9D%B8%EB%AF%BC_%EB%AF%BC%EC%A3%BC%EB%8B%B9&amp;action=edit&amp;redlink=1" title="아프가니스탄 인민 민주당 (없는 문서)">아프가니스탄 인민 민주당</a>의 유력 당원과 지도자로도 부상했다. 그러나 <a href="/wiki/1978%EB%85%84" title="1978년">1978년</a> <a href="/wiki/4%EC%9B%94_17%EC%9D%BC" title="4월 17일">4월 17일</a> 자신의 저택에서 <a href="/wiki/%EB%AC%B4%ED%95%A8%EB%A7%88%EB%93%9C_%EB%8B%A4%EC%9A%B0%EB%93%9C" title="무함마드 다우드">무함마드 다우드</a> 정권이 보낸 비밀 요원의 총에 맞아 암살당했다.</p>
<p>사후 <a href="/wiki/4%EC%9B%94_19%EC%9D%BC" title="4월 19일">4월 19일</a> 장례가 치뤄졌고 <a href="/wiki/4%EC%9B%94_28%EC%9D%BC" title="4월 28일">4월 28일</a> 이에 영향을 받은 인민 민주당의 주요 요원들이 쿠데타를 일으켜 다우드 정권을 붕괴되었다.</p>
