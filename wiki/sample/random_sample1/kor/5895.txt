<p><b>미사대교</b> (渼沙大橋)는 <a href="/wiki/%EA%B2%BD%EA%B8%B0%EB%8F%84" title="경기도">경기도</a> <a href="/wiki/%ED%95%98%EB%82%A8%EC%8B%9C" title="하남시">하남시</a> 망월동과 <a href="/wiki/%EB%82%A8%EC%96%91%EC%A3%BC%EC%8B%9C" title="남양주시">남양주시</a> 삼패동을 잇는 길이 1.53km의 <a href="/wiki/%ED%95%9C%EA%B0%95" title="한강">한강</a>의 <a class="mw-redirect" href="/wiki/%EB%8B%A4%EB%A6%AC_(%ED%86%A0%EB%AA%A9)" title="다리 (토목)">다리</a>이다. 공사중에는 남양주대교라는 가칭으로 불렸으며, <a href="/wiki/%ED%95%98%EB%82%A8%EC%8B%9C" title="하남시">하남시</a>에서 제안한 미사대교와 <a href="/wiki/%EB%82%A8%EC%96%91%EC%A3%BC%EC%8B%9C" title="남양주시">남양주시</a>에서 제안한 덕소대교 중 미사대교가 선택되어 정식 명칭이 되었다.<sup class="reference" id="cite_ref-1"><a href="#cite_note-1">[1]</a></sup> <a href="/wiki/%EC%84%9C%EC%9A%B8%EC%96%91%EC%96%91%EA%B3%A0%EC%86%8D%EB%8F%84%EB%A1%9C" title="서울양양고속도로">경춘고속도로</a>를 구성하는 <a href="/wiki/%EA%B3%A0%EC%86%8D%EB%8F%84%EB%A1%9C" title="고속도로">고속도로</a> 교량으로 이 다리가 시작되기 직전 지점에 <a href="/wiki/%EC%84%9C%EC%9A%B8%EC%96%91%EC%96%91%EA%B3%A0%EC%86%8D%EB%8F%84%EB%A1%9C" title="서울양양고속도로">서울양양고속도로</a>의 시점이 위치한다. <a href="/wiki/%EB%AF%B8%EC%82%AC_%EB%82%98%EB%93%A4%EB%AA%A9" title="미사 나들목">미사 나들목</a>과 <a href="/wiki/%EB%8D%95%EC%86%8C%EC%82%BC%ED%8C%A8_%EB%82%98%EB%93%A4%EB%AA%A9" title="덕소삼패 나들목">덕소삼패 나들목</a> 사이에 위치하며, 교량 길이는 약 1.5km이다.</p>

<ul>
<li><a href="/wiki/2004%EB%85%84" title="2004년">2004년</a> <a href="/wiki/3%EC%9B%94_22%EC%9D%BC" title="3월 22일">3월 22일</a> : <a href="/wiki/%EC%84%9C%EC%9A%B8%EC%96%91%EC%96%91%EA%B3%A0%EC%86%8D%EB%8F%84%EB%A1%9C" title="서울양양고속도로">서울양양고속도로</a>의 <a href="/wiki/%EA%B0%95%EC%9D%BC_%EB%82%98%EB%93%A4%EB%AA%A9" title="강일 나들목">강일 나들목</a> ~ <a href="/wiki/%EB%8F%99%ED%99%8D%EC%B2%9C_%EB%82%98%EB%93%A4%EB%AA%A9" title="동홍천 나들목">동홍천 나들목</a> 구간을 착공하면서 동시에 착공하였다.</li>
