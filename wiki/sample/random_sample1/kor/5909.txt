<p><b>미셸 조르주 장 길랭 프뢰돔</b> (<span style="font-size: smaller;"><a href="/wiki/%ED%94%84%EB%9E%91%EC%8A%A4%EC%96%B4" title="프랑스어">프랑스어</a>:</span> <span lang="fr" xml:lang="fr">Michel Georges Jean Ghislain Preud'homme</span>, <a href="/wiki/1959%EB%85%84" title="1959년">1959년</a> <a href="/wiki/1%EC%9B%94_24%EC%9D%BC" title="1월 24일">1월 24일</a> ~ )은 <a href="/wiki/%EB%B2%A8%EA%B8%B0%EC%97%90" title="벨기에">벨기에</a>의 전 축구 선수이자 축구 지도자로, 포지션은 <a href="/wiki/%EA%B3%A8%ED%82%A4%ED%8D%BC" title="골키퍼">골키퍼</a>였다. 현재 <a href="/wiki/%ED%81%B4%EB%A4%BC%ED%94%84_%EB%B8%8C%EB%A4%BC%ED%97%88_KV" title="클뤼프 브뤼허 KV">클뤼프 브뤼허 KV</a>의 감독을 맡고 있다. 선수 시절에는 세계 정상급 골키퍼 가운데 하나였으며, <a href="/wiki/1994%EB%85%84_FIFA_%EC%9B%94%EB%93%9C%EC%BB%B5" title="1994년 FIFA 월드컵">1994년 FIFA 월드컵</a>에서 최고의 골키퍼로서, 처음으로 <a class="mw-redirect" href="/wiki/%EC%95%BC%EC%8B%A0%EC%83%81" title="야신상">야신상</a>을 수상하였다.</p>
<p><a href="/wiki/%EC%8A%A4%ED%83%95%EB%8B%A4%EB%A5%B4_%EB%A6%AC%EC%97%90%EC%A3%BC" title="스탕다르 리에주">스탕다르 리에주</a>, <a href="/wiki/KV_%EB%A9%94%ED%97%AC%EB%A0%8C" title="KV 메헬렌">KV 메헬렌</a>, <a href="/wiki/SL_%EB%B2%A4%ED%94%BC%EC%B9%B4" title="SL 벤피카">SL 벤피카</a>에서 선수 생활을 했으며, 메헬렌에서는 <a href="/wiki/1987%EB%85%84" title="1987년">1987년</a> <a class="new" href="/w/index.php?title=%EB%B2%A8%EA%B8%B0%EC%97%90_%EC%BB%B5&amp;action=edit&amp;redlink=1" title="벨기에 컵 (없는 문서)">벨기에 컵</a> 우승, <a href="/wiki/1988%EB%85%84" title="1988년">1988년</a> <a class="mw-redirect" href="/wiki/UEFA%EC%BB%B5_%EC%9C%84%EB%84%88%EC%8A%A4%EC%BB%B5" title="UEFA컵 위너스컵">컵 위너스컵</a>, <a href="/wiki/UEFA_%EC%8A%88%ED%8D%BC%EC%BB%B5" title="UEFA 슈퍼컵">UEFA 슈퍼컵</a> 우승, <a href="/wiki/1989%EB%85%84" title="1989년">1989년</a> <a class="mw-redirect" href="/wiki/%EB%B2%A8%EA%B8%B0%EC%97%90_%EC%A3%BC%ED%95%84%EB%9F%AC_%EB%A6%AC%EA%B7%B8" title="벨기에 주필러 리그">벨기에 주필러 리그</a> 우승을 경험하였다. <a href="/wiki/1999%EB%85%84" title="1999년">1999년</a> 40세의 나이에 선수에서 은퇴했으며, <a href="/wiki/1979%EB%85%84" title="1979년">1979년</a>부터 <a href="/wiki/1995%EB%85%84" title="1995년">1995년</a>까지 <a href="/wiki/%EB%B2%A8%EA%B8%B0%EC%97%90_%EC%B6%95%EA%B5%AC_%EA%B5%AD%EA%B0%80%EB%8C%80%ED%91%9C%ED%8C%80" title="벨기에 축구 국가대표팀">벨기에 대표</a>로 58경기에 출장하였다. 1994년 FIFA 월드컵 이전 대회인 <a href="/wiki/1990%EB%85%84_FIFA_%EC%9B%94%EB%93%9C%EC%BB%B5" title="1990년 FIFA 월드컵">1990년 FIFA 월드컵</a>에서도 벨기에의 주전 골키퍼로 활약하였다.</p>
<p>은퇴 후에는 지도자의 길을 걸어, 두 차례 (2001년 1월~2002년 6월, 2006년 8월~2008 시즌 끝) <a href="/wiki/%EC%8A%A4%ED%83%95%EB%8B%A4%EB%A5%B4_%EB%A6%AC%EC%97%90%EC%A3%BC" title="스탕다르 리에주">스탕다르 리에주</a>의 감독을 맡았으며, 스탕다르 리에주는 프뢰돔의 지휘하에 25년 만에 벨기에 1부 리그에서 우승하였다. 2008-09 시즌 <a href="/wiki/KAA_%ED%97%A8%ED%8A%B8" title="KAA 헨트">KAA 헨트</a>의 감독이 되었으며, 2009-10 시즌에는 <a class="mw-redirect" href="/wiki/%EB%84%A4%EB%8D%9C%EB%9E%80%EB%93%9C_%EC%97%90%EB%A0%88%EB%94%94%EB%B9%84%EC%8B%9C" title="네덜란드 에레디비시">네덜란드 에레디비시</a>의 <a href="/wiki/FC_%ED%8A%B8%EB%B2%A4%ED%84%B0" title="FC 트벤터">FC 트벤터</a> 감독이 되었다.</p>
