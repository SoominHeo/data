<p><b>미시시피 주</b>(<span lang="en" xml:lang="en">State of Mississippi</span>, <span style="font-size: smaller;"><a href="/wiki/%EB%AC%B8%ED%99%94%EC%96%B4" title="문화어">문화어</a>:</span> 미씨시피 주)는 <a href="/wiki/%EB%AF%B8%EA%B5%AD" title="미국">미국</a> 남부의 <a href="/wiki/%EB%AF%B8%EA%B5%AD%EC%9D%98_%EC%A3%BC" title="미국의 주">주</a>이다. 북쪽으로 <a href="/wiki/%ED%85%8C%EB%84%A4%EC%8B%9C_%EC%A3%BC" title="테네시 주">테네시 주</a>, 동쪽으로 <a href="/wiki/%EC%95%A8%EB%9D%BC%EB%B0%B0%EB%A7%88_%EC%A3%BC" title="앨라배마 주">앨라배마 주</a>, 남쪽으로 <a href="/wiki/%EB%A9%95%EC%8B%9C%EC%BD%94_%EB%A7%8C" title="멕시코 만">멕시코 만</a>과 <a href="/wiki/%EB%A3%A8%EC%9D%B4%EC%A7%80%EC%95%A0%EB%82%98_%EC%A3%BC" title="루이지애나 주">루이지애나 주</a>, 서쪽으로 루이지애나 주와 <a href="/wiki/%EC%95%84%EC%B9%B8%EC%86%8C_%EC%A3%BC" title="아칸소 주">아칸소 주</a>와 접한다. 이 주의 이름인 '미시시피'는 <a href="/wiki/%EC%95%84%EB%A9%94%EB%A6%AC%EC%B9%B4_%EC%9B%90%EC%A3%BC%EB%AF%BC" title="아메리카 원주민">아메리카 원주민</a> 오지브웨이족 언어의 "큰 강"이라는 말에서 유래되었다. 미국에서 비만율이 가장 높은 주이고 흑인 비율도 가장 높다.</p>

<h2><span class="mw-headline" id=".EC.97.AD.EC.82.AC">역사</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%AF%B8%EC%8B%9C%EC%8B%9C%ED%94%BC_%EC%A3%BC&amp;action=edit&amp;section=1" title="부분 편집: 역사">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<h3><span class="mw-headline" id=".EC.B4.88.EA.B8.B0_.EC.97.AD.EC.82.AC">초기 역사</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%AF%B8%EC%8B%9C%EC%8B%9C%ED%94%BC_%EC%A3%BC&amp;action=edit&amp;section=2" title="부분 편집: 초기 역사">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<div class="thumb tleft">
<div class="thumbinner" style="width:222px;"><a class="image" href="/wiki/%ED%8C%8C%EC%9D%BC:Choctaw_Village_by_Francois_Bernard.jpg"><img alt="" class="thumbimage" data-file-height="1026" data-file-width="1460" height="155" src="//upload.wikimedia.org/wikipedia/commons/thumb/7/70/Choctaw_Village_by_Francois_Bernard.jpg/220px-Choctaw_Village_by_Francois_Bernard.jpg" srcset="//upload.wikimedia.org/wikipedia/commons/thumb/7/70/Choctaw_Village_by_Francois_Bernard.jpg/330px-Choctaw_Village_by_Francois_Bernard.jpg 1.5x, //upload.wikimedia.org/wikipedia/commons/thumb/7/70/Choctaw_Village_by_Francois_Bernard.jpg/440px-Choctaw_Village_by_Francois_Bernard.jpg 2x" width="220"/></a>

<div class="magnify"><a class="internal" href="/wiki/%ED%8C%8C%EC%9D%BC:Choctaw_Village_by_Francois_Bernard.jpg" title="실제 크기로"></a></div>
<a href="/wiki/%EC%B4%89%ED%86%A0_%EC%A1%B1" title="촉토 족">촉토 족</a>의 마을 (프랑수아 베르나르 작)</div>
</div>
</div>
<p>25,000명과 30,000명 사이의 <a class="mw-redirect" href="/wiki/%EC%9D%B8%EB%94%94%EC%96%B8" title="인디언">인디언</a>들이 첫 유럽인들이 도착하였을 때에 미시시피 지방에 살았다. 칙소 족은 북부와 동부에 살았고, <a href="/wiki/%EC%B4%89%ED%86%A0_%EC%A1%B1" title="촉토 족">촉토 족</a>은 중부 지역에 살았다. 작은 단체들은 남서부의 <a href="/wiki/%EB%82%98%EC%B2%B4%EC%A6%88_%EC%A1%B1" title="나체즈 족">나체즈 족</a>과 <a href="/wiki/%EC%95%BC%EC%A3%BC_%EA%B0%95" title="야주 강">야주 강</a>을 따라 살던 책치우마 족, 튜니카 족과 야주 족, 그리고 <a class="mw-disambig" href="/wiki/%EA%B1%B8%ED%94%84" title="걸프">걸프</a> 해안의 빌럭시 족과 패스카굴라 족을 포함하였다. 유럽인들이 지방에 온 후에 많은 인디언들은 유럽인들이 소개한 <a href="/wiki/%EC%A7%88%EB%B3%91" title="질병">질병</a>으로 사망하였다.</p>
<h3><span class="mw-headline" id=".ED.83.90.ED.97.98.EA.B3.BC_.EC.B4.88.EA.B8.B0.EC.9D.98_.EC.A0.95.EC.B0.A9">탐험과 초기의 정착</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%AF%B8%EC%8B%9C%EC%8B%9C%ED%94%BC_%EC%A3%BC&amp;action=edit&amp;section=3" title="부분 편집: 탐험과 초기의 정착">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p><a href="/wiki/1540%EB%85%84" title="1540년">1540년</a> <a href="/wiki/%EC%8A%A4%ED%8E%98%EC%9D%B8" title="스페인">스페인</a>의 탐험가 <a href="/wiki/%EC%97%90%EB%A5%B4%EB%82%9C%EB%8F%84_%EB%8D%B0_%EC%86%8C%ED%86%A0" title="에르난도 데 소토">에르난도 데 소토</a>는 미시시피 지방에 첫 유럽인 원정대를 이끌었다. 단체는 <a href="/wiki/%EA%B8%88" title="금">금</a>을 찾는 동안에 <a href="/wiki/1541%EB%85%84" title="1541년">1541년</a> <a href="/wiki/%EB%AF%B8%EC%8B%9C%EC%8B%9C%ED%94%BC_%EA%B0%95" title="미시시피 강">미시시피 강</a>에 도달하였다. 스페인의 탐험가들은 지방에 아무런 보물을 찾지 않았다. 그들은 칙소 족과의 투쟁 후에 떠나고 거기에 아무런 정착지도 설립하지 않았다.</p>
<p><a href="/wiki/1682%EB%85%84" title="1682년">1682년</a> <a href="/wiki/%ED%94%84%EB%9E%91%EC%8A%A4" title="프랑스">프랑스</a>의 탐험가 <a class="mw-redirect" href="/wiki/%EB%A5%B4%EB%84%A4-%EB%A1%9C%EB%B2%A0%EB%A5%B4_%EC%B9%B4%EB%B8%94%EB%A6%AC%EC%97%90_%EB%93%9C_%EB%9D%BC_%EC%82%B4" title="르네-로베르 카블리에 드 라 살">르네-로베르 카블리에 드 라 살</a>은 <a href="/wiki/%EC%98%A4%EB%8C%80%ED%98%B8" title="오대호">오대호</a>로부터 미시시피 강을 통하여 <a href="/wiki/%EB%A9%95%EC%8B%9C%EC%BD%94_%EB%A7%8C" title="멕시코 만">멕시코 만</a>으로 들어왔다. 카블리에는 오늘날의 미시시피 주를 포함한 미시시피 강 유역의 전체를 프랑스를 위한 주장을 하였다. 그는 국왕 <a href="/wiki/%EB%A3%A8%EC%9D%B4_14%EC%84%B8" title="루이 14세">루이 14세</a>을 위하여 지방의 이름을 "루이지애나"라고 지었다.</p>
<p><a href="/wiki/1699%EB%85%84" title="1699년">1699년</a> <a class="new" href="/w/index.php?title=%ED%94%BC%EC%97%90%EB%A5%B4_%EB%A5%B4_%EB%AC%B4%EC%95%88_%EB%94%94%EB%B2%A0%EB%A5%B4%EB%B9%8C&amp;action=edit&amp;redlink=1" title="피에르 르 무안 디베르빌 (없는 문서)">피에르 르 무안 디베르빌</a>은 현재의 오션스프링스에 속하는 올드 블럭시에서 미시시피 지방의 첫 유럽인 정착지를 설립하였다. <a href="/wiki/1716%EB%85%84" title="1716년">1716년</a> 2번째 정착지는 <a class="new" href="/w/index.php?title=%EC%9E%A5_%EB%B0%94%ED%8B%B0%EC%8A%A4%ED%8A%B8_%EB%A5%B4_%EB%AC%B4%EC%95%88_%EB%93%9C_%EB%B9%84%EC%95%B5%EB%B9%8C&amp;action=edit&amp;redlink=1" title="장 바티스트 르 무안 드 비앵빌 (없는 문서)">장 바티스트 르 무안 드 비앵빌</a>에 의하여 현재의 나체즈에 속하는 포트로세일에 설립되었다. 3년 후, <a href="/wiki/1719%EB%85%84" title="1719년">1719년</a>에 <a href="/wiki/%EC%84%9C%EC%95%84%ED%94%84%EB%A6%AC%EC%B9%B4" title="서아프리카">서아프리카</a>로부터 첫 <a href="/wiki/%ED%9D%91%EC%9D%B8" title="흑인">흑인</a> <a class="mw-redirect" href="/wiki/%EB%85%B8%EC%98%88" title="노예">노예</a>들이 미시시피 지방으로 데려와졌다. 그들은 프랑스 식민주의자들의 <a href="/wiki/%EC%8C%80" title="쌀">쌀</a>과 <a href="/wiki/%EB%8B%B4%EB%B0%B0" title="담배">담배</a>의 밭들에서 일하였다.</p>
<p>1600년대 후반과 1700년대 초반 동안에 인디언들과 유럽인들은 서로 교역을 하였다. 그러나 교역품을 위한 경쟁과 유럽인들의 대지를 위한 욕망이 많은 투쟁으로 이끌었다. 인디언 단체와 유럽인 단체들은 자신들 사이와 서로 간에 싸웠다.</p>
<p>1720년대 후반과 1730년대 초반에 포트로세일에서 프랑스인과 나체즈 족 사이의 투쟁들은 프랑스인들이 인디언 종족들을 몰아내면서 끝났다. <a href="/wiki/1736%EB%85%84" title="1736년">1736년</a> 오늘날의 튜폴로에서 일어난 애키아 전투에서 <a href="/wiki/%EC%98%81%EA%B5%AD" title="영국">영국</a>군이 프랑스군과 그들의 촉토 족 동맹군을 꺾는 데 칙소 족을 도왔다. 그 패배는 미시시피 강 유역의 더 나아가서의 통치를 얻는 것으로부터 프랑스인들을 멈추었다.</p>
<p><a href="/wiki/%ED%94%84%EB%A0%8C%EC%B9%98_%EC%9D%B8%EB%94%94%EC%96%B8_%EC%A0%84%EC%9F%81" title="프렌치 인디언 전쟁">프렌치 인디언 전쟁</a>이 일어나는 동안에 영국군과 칙소 족들은 <a href="/wiki/%EC%98%A4%ED%95%98%EC%9D%B4%EC%98%A4_%EA%B0%95" title="오하이오 강">오하이오 강</a> 유역에서 프랑스군에게 가입으로부터 미시시피 강 유역 하뷰에서 프랑스군을 지켰다. <a href="/wiki/%ED%8C%8C%EB%A6%AC_%EC%A1%B0%EC%95%BD_(1763%EB%85%84)" title="파리 조약 (1763년)">파리 조약</a>이 전쟁 후에 조인된 후, 미시시피 강 동부의 모든 대지를 영국에게 넘어갔다. 이와 같이 미시시피 지방은 영국의 통치 아래로 왔다. 북부 지역은 <a class="mw-disambig" href="/wiki/%EC%A1%B0%EC%A7%80%EC%95%84" title="조지아">조지아</a> 식민지의 일부가 되었다. 남부 지역은 "웨스트플로리다"로 불린 영국 식민지의 일부가 되었다.</p>
<h3><span class="mw-headline" id=".EC.98.81.ED.86.A0.EC.A0.81_.EC.84.B8.EC.9B.94">영토적 세월</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%AF%B8%EC%8B%9C%EC%8B%9C%ED%94%BC_%EC%A3%BC&amp;action=edit&amp;section=4" title="부분 편집: 영토적 세월">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p><a href="/wiki/%EB%AF%B8%EA%B5%AD_%EB%8F%85%EB%A6%BD_%EC%A0%84%EC%9F%81" title="미국 독립 전쟁">미국 독립 전쟁</a>이 일어나는 동안에 웨스트플로리다의 대부분의 정착자들은 영국에 충성으로 남아있었다. 그러나 미시시피 지방의 나머지에 있던 인디언, 사냥꾼과 정찰병들은 아메리카 식민지를 성원하였다. <a href="/wiki/1718%EB%85%84" title="1718년">1718년</a> 스페인의 웨스트플로리다의 통치를 얻었다. 2년 후에 영국은 스페인에게 지역을 승인하였다. 미국인들이 영국으로부터 자신들의 독립에 승리를 거둔 후에 대략 32도 선 북부의 미시시피 지방은 미국의 일부로 만들어졌다. 나체즈 32도 선과 31도 선 사이의 지역은 미국과 스페인 양국에 의하여 주장되었다. <a href="/wiki/1795%EB%85%84" title="1795년">1795년</a> 스레인 정부는 핑크니 조약에서 미국의 국경으로서 31도 선을 받아들였다.</p>
<p><a href="/wiki/1798%EB%85%84" title="1798년">1798년</a> <a href="/wiki/%EB%AF%B8%EA%B5%AD_%EC%9D%98%ED%9A%8C" title="미국 의회">미국 의회</a>는 나체즈를 수로서 미시시피 영토를 창조하였다. <a class="new" href="/w/index.php?title=%EC%9C%88%EC%8A%A4%EB%A1%AD_%EC%82%AC%EC%A0%84%ED%8A%B8&amp;action=edit&amp;redlink=1" title="윈스롭 사전트 (없는 문서)">윈스롭 사전트</a>는 새 영토의 첫 총독이 되었다. 그 영토는 31도 선에 의하여 남부, 미시시피 강에 의하여 서부, 야주 강 입구로부터 동부의 선에 의하여 북부와 채터후치 강에 의하여 동부에 경계를 지었다.</p>
<p><a href="/wiki/1803%EB%85%84" title="1803년">1803년</a> <a href="/wiki/%EB%A3%A8%EC%9D%B4%EC%A7%80%EC%95%A0%EB%82%98_%EB%A7%A4%EC%9E%85" title="루이지애나 매입">루이지애나 매입</a>이 미시시피 강을 미국의 일부로 만들었다. 영토의 개발은 강이 미시시피의 상선들을 멕시코 만으로 항해할 수 있도록 허락된 이유로 제휴되었다.</p>
<p><a href="/wiki/1804%EB%85%84" title="1804년">1804년</a> 의회는 미시시피 영토를 <a class="mw-redirect" href="/wiki/%ED%85%8C%EB%84%A4%EC%8B%9C" title="테네시">테네시</a>의 경계를 향하여 북부로 늘였다. 더 많은 대지가 <a href="/wiki/1812%EB%85%84" title="1812년">1812년</a>에 추가되었다. 그해에 펄 강의 동부에 놓인 웨스트플로리다 공화국의 일부가 미시시피 영토로 합병되었다. 공화국은 미국인 정착자들이 스페인으로부터 지방의 통치를 얻은 후에 <a href="/wiki/1810%EB%85%84" title="1810년">1810년</a>에 형성되어 왔다. 이 대지의 합병과 함께 미시시피 영토는 오늘날의 미시시피 주와 <a href="/wiki/%EC%95%A8%EB%9D%BC%EB%B0%B0%EB%A7%88_%EC%A3%BC" title="앨라배마 주">앨라배마 주</a>의 거의에 뻗어졌다.</p>

<p><a class="mw-redirect" href="/wiki/%EB%AF%B8%EC%98%81_%EC%A0%84%EC%9F%81" title="미영 전쟁">미영 전쟁</a>이 일어나는 동안에 푸시마타하 추장 아래의 촉토 족들은 미국인들에게 친근하게 남아있었다. 촉토 족은 <a href="/wiki/%EC%95%A4%EB%93%9C%EB%A3%A8_%EC%9E%AD%EC%8A%A8" title="앤드루 잭슨">앤드루 잭슨</a> 장군이 영국군이 후원한 <a href="/wiki/%ED%81%AC%EB%A6%AC%ED%81%AC_%EC%A1%B1" title="크리크 족">크리크 족</a> 인디언들의 반란을 억제하고 <a href="/wiki/%EB%89%B4%EC%98%AC%EB%A6%AC%EC%96%B8%EC%8A%A4_%EC%A0%84%ED%88%AC" title="뉴올리언스 전투">뉴올리언스 전투</a>에서 영국군을 물리치는 데 도움을 준 미시시피 시민군에 가입하였다.</p>
<h3><span class="mw-headline" id=".EC.A3.BC.EB.A1.9C.EC.84.9C.EC.9D.98_.EC.A7.80.EC.9C.84">주로서의 지위</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%AF%B8%EC%8B%9C%EC%8B%9C%ED%94%BC_%EC%A3%BC&amp;action=edit&amp;section=5" title="부분 편집: 주로서의 지위">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p><a href="/wiki/1817%EB%85%84" title="1817년">1817년</a> 의회는 미시시피 영토를 미시시피 주와 앨라배마 영토로 갈라놓았다. <a href="/wiki/1817%EB%85%84" title="1817년">1817년</a> <a href="/wiki/12%EC%9B%94_10%EC%9D%BC" title="12월 10일">12월 10일</a> 미시시피는 20번째 주로서 합중국에 편입되었다. 초대 미시시피 주지사 <a class="new" href="/w/index.php?title=%EB%8D%B0%EC%9D%B4%EB%B9%84%EB%93%9C_%ED%99%88%EC%A6%88&amp;action=edit&amp;redlink=1" title="데이비드 홈즈 (없는 문서)">데이비드 홈즈</a>는 <a href="/wiki/1809%EB%85%84" title="1809년">1809년</a> 이래 영토의 총독을 지내왔다. <a href="/wiki/1822%EB%85%84" title="1822년">1822년</a> <a href="/wiki/%EC%9E%AD%EC%8A%A8_(%EB%AF%B8%EC%8B%9C%EC%8B%9C%ED%94%BC_%EC%A3%BC)" title="잭슨 (미시시피 주)">잭슨</a>이 주도로 될 때까지 컬럼비아, 나체즈와 워싱턴이 주도로서 지내왔다.</p>
<p>영토적 세월에 인디언 종족들은 미시시피의 거의 3분의 2를 통치하였다. 종족들은 차차 자신들의 영토들을 미국 정부에 포기하였다. <a href="/wiki/1832%EB%85%84" title="1832년">1832년</a> 인디언들의 대부분이 인디언 영토로 이주하였다. 그들이 떠난 대지들은 정착을 위하여 열렸다.</p>
<p>많은 정착자들은 미시시피의 기름진 흙에 농사를 지으러 동부에서 건너왔다. 흙의 거의는 <a class="mw-redirect" href="/wiki/%EB%A9%B4%ED%99%94" title="면화">면화</a> 재배를 위하여 가장 좋았다. 면화 생산은 <a href="/wiki/1793%EB%85%84" title="1793년">1793년</a> <a class="mw-redirect" href="/wiki/%EC%97%98%EB%A6%AC_%ED%9C%98%ED%8A%B8%EB%8B%88" title="엘리 휘트니">엘리 휘트니</a>가 발명한 <a href="/wiki/%EC%A1%B0%EB%A9%B4%EA%B8%B0" title="조면기">조면기</a>의 결과로서 남부를 통하여 증가하였다.</p>
<p><a href="/wiki/1806%EB%85%84" title="1806년">1806년</a> 후에 면화 씨의 향상된 타입은 면화 생산을 증가시키는 데 도움을 주었다. 향상된 다양성은 <a href="/wiki/%EB%A9%95%EC%8B%9C%EC%BD%94" title="멕시코">멕시코</a>로부터 미시시피 주로 가져온 어떤 씨들로부터 개발되었다. 면화 재배가 개발된 클레어본 카운티에 있는 지역을 "페티트 걸프"로 불리었다. 면화 생산자들은 크고 이득적인 면화 재배를 운영하는 데 노예의 노동력을 이용하였다. 1800년대 중반으로 봐서 노예들은 주의 인구의 가장 큰 단체를 이루었다. 미시시피 주는 국가의 많은 큰 재배원들을 가졌다.</p>
<p>1850년대에 미시시피 주의 농부들은 미시시피 강과 야주 강을 통제하는 데 삼각주 지방에서 많은 제방들을 지었다. 소택지의 큰 양은 배수되어 농사를 위하여 어울리지게 만들어졌다.</p>
<h3><span class="mw-headline" id=".EB.82.A8.EB.B6.81_.EC.A0.84.EC.9F.81">남북 전쟁</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%AF%B8%EC%8B%9C%EC%8B%9C%ED%94%BC_%EC%A3%BC&amp;action=edit&amp;section=6" title="부분 편집: 남북 전쟁">편집</a><span class="mw-editsection-bracket">]</span></span></h3>

<p><a href="/wiki/1832%EB%85%84" title="1832년">1832년</a> <a class="mw-redirect" href="/wiki/%EC%82%AC%EC%9A%B0%EC%8A%A4%EC%BA%90%EB%A1%A4%EB%9D%BC%EC%9D%B4%EB%82%98" title="사우스캐롤라이나">사우스캐롤라이나</a>가 탈퇴하는 데 위협에 놓였을 때에 대부분의 미시시피 주민들은 합중국으로부터 탈퇴하는 데 호의를 가지지 않았다. 그러나 그들의 느낌들은 다른 29년 동안에 변하였다. 변화를 위한 이유들은 도망 노예법의 폭력, <a class="mw-redirect" href="/wiki/%EC%BA%94%EC%9E%90%EC%8A%A4" title="캔자스">캔자스</a>에서 노예제에 둘러싼 논쟁, <a href="/wiki/%EA%B3%B5%ED%99%94%EB%8B%B9_(%EB%AF%B8%EA%B5%AD)" title="공화당 (미국)">공화당</a>의 창당과 북부와 남부 사아의 경제적 다른 점들을 포함하였다. 미시시피 주는 주들의 권리와 <a class="mw-redirect" href="/wiki/%EB%85%B8%EC%98%88_%EC%A0%9C%EB%8F%84" title="노예 제도">노예 제도</a>의 강한 방어주가 되었다.</p>
<p><a href="/wiki/1861%EB%85%84" title="1861년">1861년</a> 잭슨에서 올드 캐피톨에 열린 집회에서 분리의 법령을 채택하였다. 사우스캐롤라이나 주가 합중국에서 탈퇴한 후, 미시시피 주는 2번째로 탈퇴한 주가 되었다. 대략 5주 후에 미시시피의 <a href="/wiki/%EC%A0%9C%ED%8D%BC%EC%8A%A8_%EB%8D%B0%EC%9D%B4%EB%B9%84%EC%8A%A4" title="제퍼슨 데이비스">제퍼슨 데이비스</a>가 <a class="mw-redirect" href="/wiki/%EC%95%84%EB%A9%94%EB%A6%AC%EC%B9%B4%EB%A7%B9%EB%B0%A9" title="아메리카맹방">아메리카맹방</a>의 대통령이 되었다. 그는 군인, 경작인과 <a href="/wiki/%EB%AF%B8%EA%B5%AD_%EC%83%81%EC%9B%90" title="미국 상원">미국 상원</a>을 지냈다. 데이비스는 또한 <a href="/wiki/%ED%94%84%EB%9E%AD%ED%81%B4%EB%A6%B0_%ED%94%BC%EC%96%B4%EC%8A%A4" title="프랭클린 피어스">프랭클린 피어스</a> 대통령 아래에서 전쟁 장관을 지내기도 하였다.</p>
<p><a class="mw-disambig" href="/wiki/%EB%82%A8%EB%B6%81_%EC%A0%84%EC%9F%81" title="남북 전쟁">남북 전쟁</a>이 일어나는 동안에 대략 80,000명의 미시시피 군인들이 남군에 복무하였다. 북군과 남군은 주에 많은 장소들과 그 경계들에 부딪쳤다. 중요한 전투들은 코린트, 해리스버그(현재 튜펄로), 홀리스프링스, 이유카, 잭슨, 메리디언과 포트깁슨에서 싸워졌다. <a href="/wiki/1864%EB%85%84" title="1864년">1864년</a> <a href="/wiki/6%EC%9B%94" title="6월">6월</a> <a class="new" href="/w/index.php?title=%EB%84%88%EB%8C%80%EB%8B%88%EC%96%BC_%EB%B2%A0%EB%93%9C%ED%8F%AC%EB%93%9C&amp;action=edit&amp;redlink=1" title="너대니얼 베드포드 (없는 문서)">너대니얼 베드포드</a> 장군 아래의 남군이 브라이스즈 크로스로즈에서 더욱 큰 북군의 기마대를 물리쳤다.</p>
<p><a href="/wiki/%EB%B9%85%EC%8A%A4%EB%B2%84%EA%B7%B8_%EC%A0%84%ED%88%AC" title="빅스버그 전투">빅스버그 전투</a>는 미시시피 주에서 가장 중요한 군사 교전으로 랭킹에 들어왔다. 빅스버그에서 남군의 강한 보존은 47일 간의 포위 공격 이후에 <a href="/wiki/1863%EB%85%84" title="1863년">1863년</a> <a href="/wiki/7%EC%9B%94_4%EC%9D%BC" title="7월 4일">7월 4일</a> <a href="/wiki/%EC%9C%A8%EB%A6%AC%EC%8B%9C%EC%8A%A4_S._%EA%B7%B8%EB%9E%9C%ED%8A%B8" title="율리시스 S. 그랜트">율리시스 S. 그랜트</a> 장군에게 함락되었다. 이 승리는 북군이 미시시피 강의 통치하는 데 주어졌다.</p>
<h3><span class="mw-headline" id=".EC.A0.84.EC.9F.81_.EC.9D.B4.ED.9B.84.EC.9D.98_.EC.84.B8.EC.9B.94">전쟁 이후의 세월</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%AF%B8%EC%8B%9C%EC%8B%9C%ED%94%BC_%EC%A3%BC&amp;action=edit&amp;section=7" title="부분 편집: 전쟁 이후의 세월">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p><a href="/wiki/1867%EB%85%84" title="1867년">1867년</a> 남부 제주의 재통합의 기간 동안에 미국 정부는 미시시피 주를 군사 지배의 아래로 두었다. 미시시피 주는 새 헌법을 채택하고 <a class="mw-redirect" href="/wiki/%EB%AF%B8%EA%B5%AD_%ED%97%8C%EB%B2%95" title="미국 헌법">미국 헌법</a>의 수정 조항 14조와 15조를 비준한 후에 <a href="/wiki/1870%EB%85%84" title="1870년">1870년</a> 합중국에 재편입되었다. 그 일은 주의 전쟁 패배들로부터 원상태로 복구하는 데 많은 세월이 걸렸다.</p>
<p><a href="/wiki/1877%EB%85%84" title="1877년">1877년</a> 재통합이 끝난 후에 다른 남부의 주들에서는 물론 미시시피 주의 백인들은 흑인들과 정치적 권력을 나누는 데 거부하였다. 남부를 통하여 흑인들은 전쟁 이후에 자신들이 얻은 권리들의 대부분을 차차 잃었다. 대부분은 빈곤에 살았다. <a href="/wiki/1890%EB%85%84" title="1890년">1890년</a>의 미시시피 헌법에서 개설된 투표 요구들은 흑인들이 투표하는 데 불가능적으로 만들었다. 1890년대에 미시시피 주와 다른 남부의 주들에서 입법부가 기차와 공공 자동차와 공공 시설에서 흑인들을 위한 갈라진 장소들을 요구한 인종적 분리법들을 통과시키기 시작하였다. 또한 1890년대 동안에 남부에서 수백만명의 흑인들이 폭도들에 의하여 <a href="/wiki/%EB%A6%B0%EC%B9%98" title="린치">린치</a>당하였다.</p>
<h3><span class="mw-headline" id="1900.EB.85.84.EB.8C.80_.EC.B4.88.EB.B0.98">1900년대 초반</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%AF%B8%EC%8B%9C%EC%8B%9C%ED%94%BC_%EC%A3%BC&amp;action=edit&amp;section=8" title="부분 편집: 1900년대 초반">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<div class="thumb tleft">
<div class="thumbinner" style="width:222px;"><a class="image" href="/wiki/%ED%8C%8C%EC%9D%BC:All_these_children.jpg"><img alt="" class="thumbimage" data-file-height="685" data-file-width="993" height="152" src="//upload.wikimedia.org/wikipedia/commons/thumb/9/96/All_these_children.jpg/220px-All_these_children.jpg" srcset="//upload.wikimedia.org/wikipedia/commons/thumb/9/96/All_these_children.jpg/330px-All_these_children.jpg 1.5x, //upload.wikimedia.org/wikipedia/commons/thumb/9/96/All_these_children.jpg/440px-All_these_children.jpg 2x" width="220"/></a>
<div class="thumbcaption">

미시시피 주의 어린이 노동자들</div>

<p>주에서 인종적 관계들은 1900년대의 시작에서 변함 없이 남아졌다. 많은 <a href="/wiki/%EC%95%84%ED%94%84%EB%A6%AC%EC%B9%B4%EA%B3%84_%EB%AF%B8%EA%B5%AD%EC%9D%B8" title="아프리카계 미국인">아프리카계 미국인</a>들이 북부의 도시들에 직업들을 찾으러 주를 떠나기 시작하였다. 그러나 산업과 교육에서 향상들이 일어났다.</p>
<p>철도들의 건설은 미시시피 남동부의 소나무 숲으로 접근들을 마련하여 재목 산업에서 붐을 일으키는 결과를 가져왔다. 이 산업은 <a href="/wiki/1914%EB%85%84" title="1914년">1914년</a> <a href="/wiki/%EC%A0%9C1%EC%B0%A8_%EC%84%B8%EA%B3%84_%EB%8C%80%EC%A0%84" title="제1차 세계 대전">제1차 세계 대전</a>이 일어나기 전에 높은 절정에 도달하였다. 새로운 배수 계획들은 큰 늪의 지역들을 농업으로 열었다. 카운티의 농업 고등학교들이 <a href="/wiki/1908%EB%85%84" title="1908년">1908년</a>에 설립되었다. <a href="/wiki/1916%EB%85%84" title="1916년">1916년</a> 주는 문맹자 성인들을 위하여 특별 교육 프로그램을 시작하는 데 문맹 위원회를 설립하였다. <a href="/wiki/1912%EB%85%84" title="1912년">1912년</a> 미시시피 주의 입법부는 어린이 노동을 규제하는 법률들을 통과시켰다.</p>
<p>1920년대 동안에 입법적 법안들은 교육의 주 위원회(1924년), 주립 도서관 위원회(1926년)와 고속도로 건설 프로그램(1929년)을 포함하였다. 남부에서 첫 연유 제조소가 <a href="/wiki/1926%EB%85%84" title="1926년">1926년</a> 스타크빌에 열렸다.</p>
<p><a href="/wiki/1927%EB%85%84" title="1927년">1927년</a> 미시시피 주는 미시시피 강의 홍수에서 거대하게 고통을 겪었다. 대략 100,000명의 주민들이 삼각주에서 홍수로 잠긴 주택들을 떠났다. 미시시피 주에서 수확물과 재산의 손해는 총 20억 4백만 달러나 들었다. 다음 해에 미국 의회는 <a href="/wiki/%EB%AF%B8%EA%B5%AD_%EC%9C%A1%EA%B5%B0" title="미국 육군">미국 육군</a>의 공병단을 미시시피 강에 홍수 조절을 위하여 책임이 있도록 만들었다.</p>
<p>1930년대의 <a href="/wiki/%EB%8C%80%EA%B3%B5%ED%99%A9" title="대공황">대공황</a>이 일어나는 동안에 미시시피 주는 경제적 개발의 중요한 프로그램들을 발출하였다. 산업 균형 농업법으로 불린 이 프로그램은 <a href="/wiki/1936%EB%85%84" title="1936년">1936년</a> 입법부에 의하여 통과된 특별한 법률들에 의하여 도움을 받았다. 이 법률들은 정확한 세금들을 내는 것으로부터 새로운 산업들을 해방시켰다. 법률은 또한 도시들과 카운티들이 국채를 발행하여 새로운 산업들에 위하여 공장들을 짓는 데 그 돈을 쓰는 데 허락하기도 하였다. 산업 균형 농업법은 주의 경제 개발부에 의하여 관리되었다. 산업 균형 농업법 프로그램의 개발에서 미시시피 주는 새로운 산업들을 흥행하는 데 국가적 광고를 사용한 첫 주들 중의 하나가 되었다. 틴즐리(1939년)와 본(1940년)에서 <a href="/wiki/%EC%84%9D%EC%9C%A0" title="석유">석유</a>의 발견이 주의 경제에 원조하였다.</p>
<h3><span class="mw-headline" id="1900.EB.85.84.EB.8C.80_.EC.A4.91.EB.B0.98">1900년대 중반</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%AF%B8%EC%8B%9C%EC%8B%9C%ED%94%BC_%EC%A3%BC&amp;action=edit&amp;section=9" title="부분 편집: 1900년대 중반">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<div class="thumb tright">
<div class="thumbinner" style="width:222px;"><a class="image" href="/wiki/%ED%8C%8C%EC%9D%BC:Statue_of_Medgar_Evers.jpg"><img alt="" class="thumbimage" data-file-height="1000" data-file-width="442" height="498" src="//upload.wikimedia.org/wikipedia/commons/thumb/4/4f/Statue_of_Medgar_Evers.jpg/220px-Statue_of_Medgar_Evers.jpg" srcset="//upload.wikimedia.org/wikipedia/commons/thumb/4/4f/Statue_of_Medgar_Evers.jpg/330px-Statue_of_Medgar_Evers.jpg 1.5x, //upload.wikimedia.org/wikipedia/commons/thumb/4/4f/Statue_of_Medgar_Evers.jpg/440px-Statue_of_Medgar_Evers.jpg 2x" width="220"/></a>

<p><a href="/wiki/%EC%A0%9C2%EC%B0%A8_%EC%84%B8%EA%B3%84_%EB%8C%80%EC%A0%84" title="제2차 세계 대전">제2차 세계 대전</a>이 일어나는 동안에 미시시피 주에서 많은 전쟁 지대들이 운영되었다. 패스카굴라에 있는 잉골스 조선소들은 중요한 조선업 중심지가 되었다. 전쟁이 끝난 후에 주의 산업 발전은 지속되었다. <a href="/wiki/1954%EB%85%84" title="1954년">1954년</a> 입법부는 노동권의 법률을 통과시켰다. 이 법률은 자신이 일하길 원하지 않는다면 실업자들이 조합에 가입해야 하는 것을 마련하였다. 법률은 <a href="/wiki/1960%EB%85%84" title="1960년">1960년</a> 주의 헌법의 일부가 되었다.</p>
<p>1960년대 동안에 미시시피 주는 새로운 산업들을 끌어들이는 일을 하였다. 농업의 증가하는 기계화가 농업 노동의 과잉을 창조하였고 미시시피 주의 지도자들은 전직 농부들을 고용할 수 있던 새로운 산업들을 원하였다. 입법부는 <a href="/wiki/1960%EB%85%84" title="1960년">1960년</a>에 법률들을 통과시켜 산업의 면세 자세를 넓혔다. <a href="/wiki/1963%EB%85%84" title="1963년">1963년</a> 큰 석유 정제소가 패스카굴라에 지어졌다. <a href="/wiki/1966%EB%85%84" title="1966년">1966년</a>으로 봐서 더욱 많은 미시시피 주민들이 농업보다 제조업에서 일하였다. 그러나 미시시피 주는 아직도 다른 주들과 비교하여 상대적으로 적은 산업을 가졌고, 이 근로자들의 대부분은 적은 소득을 벌었다.</p>
<p><a href="/wiki/1967%EB%85%84" title="1967년">1967년</a> 리턴 산업은 잉골스 조선소를 획득하여 패스카굴라에서 확장된 프로그램을 시작하였다. 다른 해안 지역들에서는 <a class="mw-redirect" href="/wiki/%EA%B4%80%EA%B4%91%EC%97%85" title="관광업">관광업</a>이 붐을 일으켰다.</p>
<p>많은 다른 주들과 같이 1900년대 중반에 미시시피 주에서 인종적 문제들이 주의의 집중을 가져왔다. 주의 헌법은 인종적으로 불리된 학교들을 위하여 마련되었다. 그러나 <a href="/wiki/1954%EB%85%84" title="1954년">1954년</a> <a class="mw-redirect" href="/wiki/%EB%AF%B8%EA%B5%AD_%EB%8C%80%EB%B2%95%EC%9B%90" title="미국 대법원">미국 대법원</a>은 공공 학교들의 의무적 인종 분리는 비헌법적이라고 지배하였다. 인권 운동 단체들에 의한 노력들은 인종 차별의 폐지가 어쩌다 폭력을 만난 것에 대하여 가져왔다.</p>
<p><a href="/wiki/1962%EB%85%84" title="1962년">1962년</a> <a class="new" href="/w/index.php?title=%EC%A0%9C%EC%9E%84%EC%8A%A4_%EB%A9%94%EB%9F%AC%EB%94%94%EC%8A%A4&amp;action=edit&amp;redlink=1" title="제임스 메러디스 (없는 문서)">제임스 메러디스</a>는 <a href="/wiki/%EB%AF%B8%EC%8B%9C%EC%8B%9C%ED%94%BC_%EB%8C%80%ED%95%99%EA%B5%90" title="미시시피 대학교">미시시피 대학교</a>에 입학한 첫 아프리카계 미국인 학생이 되었다. 주 정부는 입학을 반대해왔다. 큰 관중들이 학교에 모였고, 메레디스의 안전을 보증하는 데 연방군이 보내졌다. 일어난 폭동들에서 2명이 사망하였다. <a href="/wiki/1963%EB%85%84" title="1963년">1963년</a> NAACP의 미시시피 지방 연락원 <a class="new" href="/w/index.php?title=%EB%A9%94%EB%93%9C%EA%B1%B0_%EC%97%90%EB%B2%84%EC%8A%A4&amp;action=edit&amp;redlink=1" title="메드거 에버스 (없는 문서)">메드거 에버스</a>가 암살되었다. <a href="/wiki/1964%EB%85%84" title="1964년">1964년</a> "자유의 여름"으로 불린 일이 있는 동안에 인권 운동 기구인 학생 비폭력 협회에 의하여 첫째로 결성된 계획을 위하여 수백명의 대학생들이 미시시피 주로 여행하였다. 그들은 아프리카계 미국인들이 투표하는 데 등록될 수 있도록 도움을 주었다. 미시시피 주 필라델피아 근처에서 3명의 지원자들이 살해되었다.</p>
<p><a href="/wiki/1964%EB%85%84" title="1964년">1964년</a>의 <a href="/wiki/%EA%B0%80%EC%9D%84" title="가을">가을</a>에 미시시피 주에서 첫 공공 학교들이 인종 차별의 폐지를 시작하였다. <a href="/wiki/1969%EB%85%84" title="1969년">1969년</a> 미국 대법원은 인종적으로 분리된 모든 공공 학교들을 끝내는 즉각 명령을 내렸다. 결과로서 <a href="/wiki/%EB%89%B4%EC%98%AC%EB%A6%AC%EC%96%B8%EC%8A%A4" title="뉴올리언스">뉴올리언스</a>의 연방 법원은 <a href="/wiki/1969%EB%85%84" title="1969년">1969년</a> <a href="/wiki/12%EC%9B%94" title="12월">12월</a> 안으로 33개의 미시시피 주 학교 구역들에 인종 차별을 폐지하도록 명령을 내렸다. 많은 백인들은 인종적으로 분리된 사립 학교들을 열어 자신들의 자식들을 입학시켰다.</p>
<p><a href="/wiki/1967%EB%85%84" title="1967년">1967년</a> <a class="new" href="/w/index.php?title=%EB%A1%9C%EB%B2%84%ED%8A%B8_G._%ED%81%B4%EB%9D%BC%ED%81%AC&amp;action=edit&amp;redlink=1" title="로버트 G. 클라크 (없는 문서)">로버트 G. 클라크</a>가 미시시피 주 하원 선거를 이겼다 그는 남부 제주 재통합 이래 하원으로 선출된 첫 아프리카계 미국인 의원이었다. <a href="/wiki/1969%EB%85%84" title="1969년">1969년</a> 메드거의 동생 <a class="new" href="/w/index.php?title=%EC%B0%B0%EC%8A%A4_%EC%97%90%EB%B2%84%EC%8A%A4&amp;action=edit&amp;redlink=1" title="찰스 에버스 (없는 문서)">찰스 에버스</a>가 파예트의 시장으로 선출되었다. 그는 재통합 이래 주의 첫 아프리카계 미국인 시장이 되었다.</p>
<h3><span class="mw-headline" id="1900.EB.85.84.EB.8C.80_.ED.9B.84.EB.B0.98">1900년대 후반</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%AF%B8%EC%8B%9C%EC%8B%9C%ED%94%BC_%EC%A3%BC&amp;action=edit&amp;section=10" title="부분 편집: 1900년대 후반">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>미시시피 주 가족들의 소득들이 1900년대 후반에 솟아 올랐다. 하지만 이 소득들은 국가적 표준에 의하여 낮게 남아졌다. 이 시기에 새로운 산업들이 주로 이동하였다. 미시시피 주 근로자들의 다수가 자라나는 제조업에 고용되었다. 그러나 삼각주의 수천명의 농부들은 농장들에 기계화가 증가된 이유로 무직자들이었다.</p>
<p>이 시기의 동안에 촉토 인디언들의 미시시피 일단이 주의 최고 고용자들 사이에 랭킹에 드는 데 미시시피 주의 가장 가난한 단체들 중의 하나로서부터 떠났다. <a href="/wiki/1969%EB%85%84" title="1969년">1969년</a> 촉토 족은 종족 일원들을 위한 주택들을 짓는 건설 회사를 열었다. 1900년대의 끝으로 봐서 미시시피 주 동중부의 촉토 보호 구역은 또한 산업 공원, 쇼핑 센터, 도박장과 <a href="/wiki/%EA%B3%A8%ED%94%84" title="골프">골프</a> 코스를 포함하였다.</p>
<h3><span class="mw-headline" id="21.EC.84.B8.EA.B8.B0">21세기</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%AF%B8%EC%8B%9C%EC%8B%9C%ED%94%BC_%EC%A3%BC&amp;action=edit&amp;section=11" title="부분 편집: 21세기">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>2000년대의 시작에서 주기는 논쟁의 중심이 되었다. <a href="/wiki/1894%EB%85%84" title="1894년">1894년</a>에 채택된 주기에 다수의 아프리카계 미국인과 다른이들은 새로운 디자인을 요구하였다. 그들은 상징을 노예 제도의 기호로 보았다. 주지사에 의하여 임명된 위원회는 새로운 디자인을 제안하였다. <a href="/wiki/2001%EB%85%84" title="2001년">2001년</a> 미시시피 주의 투표인들은 새로운 디자인을 거절하고 원래 국기를 간직하도록 선택하였다.</p>
<p><a href="/wiki/2002%EB%85%84" title="2002년">2002년</a> 정착지가 장시간에 걸친 인권 운동의 경우에 도달하였다. <a href="/wiki/1975%EB%85%84" title="1975년">1975년</a> <a class="new" href="/w/index.php?title=%EC%A0%9C%EC%9D%B4%ED%81%AC_%EC%97%90%EC%96%B4%EC%8A%A4_%EC%8B%9C%EB%8B%88%EC%96%B4&amp;action=edit&amp;redlink=1" title="제이크 에어스 시니어 (없는 문서)">제이크 에어스 시니어</a>가 잭슨 주립 대학교 학생인 자신의 아들을 대표하여 주를 고발하였다. 그는 주의 전통적 흑인 대학교들이 첫째로 백인들에 의하여 참석된 그 대학교들보다 적은 기금을 받았다고 주장하였다. 주는 알콘 주립 대학교, 잭슨 주립 대학교와 미시시피 밸리 주립 대학교들에게 새로운 프로그램들과 시설들을 위하여 5억 달러를 보증하는 데 동의하였다.</p>
<p><a href="/wiki/2005%EB%85%84" title="2005년">2005년</a> <a href="/wiki/8%EC%9B%94" title="8월">8월</a> <a href="/wiki/%ED%97%88%EB%A6%AC%EC%BC%80%EC%9D%B8_%EC%B9%B4%ED%8A%B8%EB%A6%AC%EB%82%98" title="허리케인 카트리나">허리케인 카트리나</a>가 걸프 해안을 쳐서 핸콕, 해리슨과 잭슨 카운티들에 타격을 주었다. 큰 바람과 큰 파도, 그리고 홍수가 많은 건물, 도로와 다리들을 파괴시켰다. 수천명의 주민들이 노숙자가 되고 말았다. 공무원들은 빌럭스와 걸프포트, 그리고 몇몇의 해안 타운들에서 많은 사망자들을 보고하였다.</p>
<h2><span class="mw-headline" id=".EA.B2.BD.EC.A0.9C">경제</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%AF%B8%EC%8B%9C%EC%8B%9C%ED%94%BC_%EC%A3%BC&amp;action=edit&amp;section=12" title="부분 편집: 경제">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<h3><span class="mw-headline" id=".EC.84.9C.EB.B9.84.EC.8A.A4.EC.97.85">서비스업</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%AF%B8%EC%8B%9C%EC%8B%9C%ED%94%BC_%EC%A3%BC&amp;action=edit&amp;section=13" title="부분 편집: 서비스업">편집</a><span class="mw-editsection-bracket">]</span></span></h3>

<p>서비스업은 미시시피 주의 고용과 국내 총생산 양쪽에서 대략 4분의 3을 차지한다.</p>
<p>주도인 잭슨은 정부 활동들의 중심지이다. 도시는 또한 미시시피 주의 지도적인 금융의 중심지이기도 하다. 주의 주요 <a href="/wiki/%EC%9D%80%ED%96%89" title="은행">은행</a>, <a href="/wiki/%EB%B3%B4%ED%97%98" title="보험">보험</a> 회사와 <a href="/wiki/%EB%B6%80%EB%8F%99%EC%82%B0" title="부동산">부동산</a>들의 대부분은 잭슨 지역에 기지를 두었다. 많은 <a href="/wiki/%ED%98%B8%ED%85%94" title="호텔">호텔</a>, 식당과 소매상들은 <a href="/wiki/%EB%B9%8C%EB%9F%AD%EC%8B%9C" title="빌럭시">빌럭시</a>와 잭슨 지역과 <a href="/wiki/%ED%85%8C%EB%84%A4%EC%8B%9C_%EC%A3%BC" title="테네시 주">테네시 주</a> <a href="/wiki/%EB%A9%A4%ED%94%BC%EC%8A%A4_(%ED%85%8C%EB%84%A4%EC%8B%9C_%EC%A3%BC)" title="멤피스 (테네시 주)">멤피스</a> 남부에 있는 주의 북서부 구석에서 운영된다.</p>
<p>주에서 주요 군사 설립지들은 빌럭시의 키즐러 공군 기지와 핸콕 카운티의 존 C. 스테니스 우주 센터를 포함한다. 스테니스 우주 센터는 <a href="/wiki/%EB%AF%B8%EA%B5%AD_%ED%95%AD%EA%B3%B5%EC%9A%B0%EC%A3%BC%EA%B5%AD" title="미국 항공우주국">미국 항공우주국</a>의 가장 큰 <a class="mw-redirect" href="/wiki/%EB%A1%9C%EC%BC%80%ED%8A%B8" title="로케트">로케트</a> 엔진 시험 지대이다.</p>
<p><a href="/wiki/%EC%B9%B4%EC%A7%80%EB%85%B8" title="카지노">카지노</a> 게임은 주의 도박 소득을 이끄는 미시시피 주에서 가장 큰 산업이다. 많은 카지노들은 빌럭시와 튜니카 카운티에서 운영된다. 남동부의 중심부와 <a href="/wiki/%EB%AF%B8%EC%8B%9C%EC%8B%9C%ED%94%BC_%EA%B0%95" title="미시시피 강">미시시피 강</a>에 놓인 미시시피 주의 위치는 주요 교통의 주로 만든다.</p>
<h3><span class="mw-headline" id=".EC.A0.9C.EC.A1.B0.EC.97.85">제조업</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%AF%B8%EC%8B%9C%EC%8B%9C%ED%94%BC_%EC%A3%BC&amp;action=edit&amp;section=14" title="부분 편집: 제조업">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>미시시피 주는 다른 주들의 가장 많은 제조업 부문들 중의 하나를 가지고 있다. 4개의 지도적인 제조품들 - 화학품, 식품, <a href="/wiki/%EC%84%9D%EC%9C%A0" title="석유">석유</a> 제품과 교통 수단은 소득의 동등한 양의 가깝게 발생시킨다.</p>
<p>화학품들의 지도적인 타입은 <a href="/wiki/%EB%86%8D%EC%95%BD" title="농약">농약</a>, 세척제, 약품과 수지를 포함한다. 패스카굴라는 큰 석유 정제소를 가지고 있다.</p>
<p>정육 제품들은 미시시피 주의 지도적인 가공 식품으로서 랭킹에 들어와 있다. 포러스트, 로럴과 모턴은 큰 가금육 가공 지대들을 가지고 있다. 다른 중요한 식품들은 제과류와 해산물이다.</p>
<p><a href="/wiki/%EC%9E%90%EB%8F%99%EC%B0%A8" title="자동차">자동차</a>, 자동차 부품과 <a class="mw-disambig" href="/wiki/%EB%B0%B0" title="배">배</a>는 지도적인 교통 수단 생산품들이다. 캔턴 근처에 있는 공장 지대는 자동차를 제조한다. 패스카굴라에 있는 조선소는 유조선을 포함한 상선과 구축함을 포함한 군함을 짓는다.</p>
<p>다른 제조품들은 합성 금속 제품, <a href="/wiki/%EA%B0%80%EA%B5%AC" title="가구">가구</a>와 기계류를 포함한다. 합성 금속 제품들은 기계 조립 제품, 금속 캔과 금속 문과 창문을 포함한다. 미시시피 주는 겉천을 깐 가구 생산에서 지도적인 주이다. 가구는 주로 미시시피 주의 북동부 지역에서 만들어진다. <a href="/wiki/%EA%B1%B4%EC%84%A4" title="건설">건설</a>, <a href="/wiki/%EB%86%8D%EC%9E%A5" title="농장">농장</a>과 난방 용품들, 그리고 엔진과 터빈은 기계 산업의 지도적인 타입들이다.</p>
<h3><span class="mw-headline" id=".EB.86.8D.EC.97.85">농업</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%AF%B8%EC%8B%9C%EC%8B%9C%ED%94%BC_%EC%A3%BC&amp;action=edit&amp;section=15" title="부분 편집: 농업">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<div class="thumb tright">
<div class="thumbinner" style="width:222px;"><a class="image" href="/wiki/%ED%8C%8C%EC%9D%BC:PSM_V54_D025_Cotton_field_in_mississippi.jpg"><img alt="" class="thumbimage" data-file-height="1685" data-file-width="2530" height="147" src="//upload.wikimedia.org/wikipedia/commons/thumb/e/e2/PSM_V54_D025_Cotton_field_in_mississippi.jpg/220px-PSM_V54_D025_Cotton_field_in_mississippi.jpg" srcset="//upload.wikimedia.org/wikipedia/commons/thumb/e/e2/PSM_V54_D025_Cotton_field_in_mississippi.jpg/330px-PSM_V54_D025_Cotton_field_in_mississippi.jpg 1.5x, //upload.wikimedia.org/wikipedia/commons/thumb/e/e2/PSM_V54_D025_Cotton_field_in_mississippi.jpg/440px-PSM_V54_D025_Cotton_field_in_mississippi.jpg 2x" width="220"/></a>
<div class="thumbcaption">
<div class="magnify"><a class="internal" href="/wiki/%ED%8C%8C%EC%9D%BC:PSM_V54_D025_Cotton_field_in_mississippi.jpg" title="실제 크기로"></a></div>
<a class="mw-redirect" href="/wiki/%EB%A9%B4%ED%99%94" title="면화">면화</a>는 미시시피 주의 가장 가치적인 농산물이다.</div>
</div>
</div>
<p>농장 지대는 미시시피 주의 대략 3분의 1을 덮고 있다. <a href="/wiki/%EB%8B%AD" title="닭">닭</a>은 미시시피 주에서 지도적인 축산물이며, 주는 지도적인 닭의 사육지로서 랭킹에 들어와있다. 닭의 사육장들의 가장 큰 집중 지대는 해티스버그, 잭슨과 메리디언에 의하여 이루어진 삼각형 지방 안에 놓여 있다. 어떤 농부들은 암탉을 기르고 <a href="/wiki/%EB%8B%AC%EA%B1%80" title="달걀">달걀</a>을 생산한다.</p>
<p><a href="/wiki/%EC%9C%A1%EC%9A%B0" title="육우">육우</a>와 <a href="/wiki/%EC%A0%96%EC%86%8C" title="젖소">젖소</a>도 또한 미시시피 주에서 농장 소득의 중요한 근원들이다. 주의 소를 키우는 농부들은 사료를 위한 장기적 계절들을 위하여 허락된 순한 기후로부터 이익을얻는다. 소의 목장들은 삼각주로 불리는 앨류비얼 평야를 제외한 모든 지방들에서 공통적이다. 어떤 농부들은 <a href="/wiki/%EB%8F%BC%EC%A7%80" title="돼지">돼지</a>나 <a href="/wiki/%EC%96%91" title="양">양</a>을 기른다.</p>
<p><a href="/wiki/%EC%98%A5%EC%88%98%EC%88%98" title="옥수수">옥수수</a>, <a class="mw-redirect" href="/wiki/%EB%A9%B4%ED%99%94" title="면화">면화</a>와 <a href="/wiki/%EB%8C%80%EB%91%90" title="대두">대두</a>는 주의 가장 가치적인 수확물이다. 이 3개의 수확물들은 주로 부의 북서부와 서중부에서 자란다. 미시시피 주는 면화, <a href="/wiki/%EC%8C%80" title="쌀">쌀</a>과 <a href="/wiki/%EA%B3%A0%EA%B5%AC%EB%A7%88" title="고구마">고구마</a> 생산의 지도적인 주들로서 랭킹에 들어와있다.</p>
<h3><span class="mw-headline" id=".EA.B4.91.EC.97.85">광업</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%AF%B8%EC%8B%9C%EC%8B%9C%ED%94%BC_%EC%A3%BC&amp;action=edit&amp;section=16" title="부분 편집: 광업">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p><a href="/wiki/%EC%84%9D%EC%9C%A0" title="석유">석유</a>와 <a class="mw-redirect" href="/wiki/%EC%B2%9C%EC%97%B0%EA%B0%80%EC%8A%A4" title="천연가스">천연가스</a>는 주의 광물들 중의 가장 많은 가격을 차지한다. 석유 생산은 주의 남부에서 가장 대량이다 랭킨 카운티는 천연가스의 주요 생산지이다.</p>
<p>미시시피 주의 다른 광물들은 <a href="/wiki/%EC%A0%90%ED%86%A0" title="점토">점토</a>, 깨진 돌, <a href="/wiki/%EA%B0%88%ED%83%84" title="갈탄">갈탄</a>과 <a href="/wiki/%EB%AA%A8%EB%9E%98" title="모래">모래</a>와 <a href="/wiki/%EC%9E%90%EA%B0%88" title="자갈">자갈</a>이다. 주에서 체굴되는 점토는 벤토나이트와 풀러스 어스이다. 주의 동부 경계에 따라 놓인 <a href="/wiki/%EC%84%9D%ED%9A%8C%EC%95%94" title="석회암">석회암</a> 채석장들은 깨진 돌의 주요 근원이다. 갈탄광은 애커맨 근처에서 운영된다. 모래과 자갈은 주의 전역을 통하여 체굴된다.</p>
<h3><span class="mw-headline" id=".EC.88.98.EC.82.B0.EC.97.85">수산업</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%AF%B8%EC%8B%9C%EC%8B%9C%ED%94%BC_%EC%A3%BC&amp;action=edit&amp;section=17" title="부분 편집: 수산업">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>미시시피 주는 중요한 <a href="/wiki/%EC%83%88%EC%9A%B0" title="새우">새우</a> 잡이의 주이다. 빌럭시는 미시시피 주의 주요 새우 포장업의 항구이다. 패스카굴라는 주의 수산업의 다른 중심지로서 랭킹에 들어와있다. 가장 중요한 잡이들은 <a href="/wiki/%EA%B2%8C" title="게">게</a>, <a href="/wiki/%EC%B2%AD%EC%96%B4" title="청어">청어</a>와 바다 <a href="/wiki/%EC%86%A1%EC%96%B4" title="송어">송어</a>를 포함한다. 미시시피 주는 양식장에 자란 <a href="/wiki/%EB%A9%94%EA%B8%B0" title="메기">메기</a>의 지도적인 생산주이다. 앨러비얼 평야 혹은 삼각주는 주의 번영하는 메기 양식과 가공업의 중심지이다.</p>
<h2><span class="mw-headline" id=".EC.A3.BC.EB.AF.BC">주민</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%AF%B8%EC%8B%9C%EC%8B%9C%ED%94%BC_%EC%A3%BC&amp;action=edit&amp;section=18" title="부분 편집: 주민">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<div class="thumb tleft">
<div class="thumbinner" style="width:222px;"><a class="image" href="/wiki/%ED%8C%8C%EC%9D%BC:JacksonMS_Downtown_Panorama.jpg"><img alt="" class="thumbimage" data-file-height="822" data-file-width="2247" height="80" src="//upload.wikimedia.org/wikipedia/commons/thumb/8/85/JacksonMS_Downtown_Panorama.jpg/220px-JacksonMS_Downtown_Panorama.jpg" srcset="//upload.wikimedia.org/wikipedia/commons/thumb/8/85/JacksonMS_Downtown_Panorama.jpg/330px-JacksonMS_Downtown_Panorama.jpg 1.5x, //upload.wikimedia.org/wikipedia/commons/thumb/8/85/JacksonMS_Downtown_Panorama.jpg/440px-JacksonMS_Downtown_Panorama.jpg 2x" width="220"/></a>
<div class="thumbcaption">
<div class="magnify"><a class="internal" href="/wiki/%ED%8C%8C%EC%9D%BC:JacksonMS_Downtown_Panorama.jpg" title="실제 크기로"></a></div>
다운타운 잭슨의 파노라마</div>
</div>
</div>
<p><a href="/wiki/2000%EB%85%84" title="2000년">2000년</a> <a class="mw-redirect" href="/wiki/%EB%AF%B8%EA%B5%AD_%EC%9D%B8%EA%B5%AC_%EC%A1%B0%EC%82%AC%EA%B5%AD" title="미국 인구 조사국">미국 인구 조사국</a>에 의하면 미시시피 주의 인구가 2,844,658명이라고 보고하였다. 인구는 <a href="/wiki/1990%EB%85%84" title="1990년">1990년</a>으로 본 2,573,216명에서 10.5 퍼센트나 증가하였다. <a href="/wiki/2000%EB%85%84" title="2000년">2000년</a> 조사국에 의하면 미시시피 주가 50개의 주들 중에 인구에서 31위에 들어와있다.</p>
<p>절반에 가까운 미시시피 주민들은 도시 지역들에 산다. 주의 가장 큰 도시는 주도이며 미시시피 주의 비지니스와 금융 활동의 중심지 잭슨이다.</p>
<p>미시시피 주민의 대략 40 퍼센트는 메트로폴리탄 지역에 살고 있다. 이 지역들은 걸포트-빌럭시, 해티스버그, 잭슨과 패스카굴라이다. 멤피스(<a href="/wiki/%ED%85%8C%EB%84%A4%EC%8B%9C_%EC%A3%BC" title="테네시 주">테네시 주</a>) 메트로폴리탄 지역은 미시시피 주에서 디소토, 마샬, 테이트와 튜니카 카운티 지역으로 뻗어 들어간다.</p>
<p>미시시피 주의 6개의 가장 큰 도시들로는 크기의 순서로 봐서 잭슨, 걸포트, 빌럭시, 해티스버그, 그린빌과 메리디언이다. 이 도시들 중에 걸포트는 1990년대에 빠르게 자라났다. <a href="/wiki/1990%EB%85%84" title="1990년">1990년</a>부터 <a href="/wiki/2000%EB%85%84" title="2000년">2000년</a>까지 그 인구는 거의 75 퍼센트나 증가하였다. 주의 몇몇의 대도시들은 <a href="/wiki/%EB%AF%B8%EC%8B%9C%EC%8B%9C%ED%94%BC_%EA%B0%95" title="미시시피 강">미시시피 강</a>에 놓여있다. 이 도시들은 그린빌, 내치즈와 <a href="/wiki/%EB%B9%85%EC%8A%A4%EB%B2%84%EA%B7%B8_(%EB%AF%B8%EC%8B%9C%EC%8B%9C%ED%94%BC_%EC%A3%BC)" title="빅스버그 (미시시피 주)">빅스버그</a>이다.</p>
<p><a href="/wiki/2000%EB%85%84" title="2000년">2000년</a> 조사국은 미시시피 주민의 36 퍼센트가 <a href="/wiki/%EC%95%84%ED%94%84%EB%A6%AC%EC%B9%B4%EA%B3%84_%EB%AF%B8%EA%B5%AD%EC%9D%B8" title="아프리카계 미국인">아프리카계 미국인</a>이라고 보였다. 이것은 여러 다른 주보다 아프리카계 미국인들의 더 큰 비율이다. 주의 다음으로 가장 큰 소수 민족 단체인 <a href="/wiki/%ED%9E%88%EC%8A%A4%ED%8C%A8%EB%8B%89" title="히스패닉">히스패닉</a>은 인구의 1 퍼센트 이상을 차지한다. 미시시피 주의 다른 큰 인구 단체들은 <a href="/wiki/%EC%95%84%EC%9D%BC%EB%9E%9C%EB%93%9C" title="아일랜드">아일랜드</a>, <a href="/wiki/%EC%98%81%EA%B5%AD" title="영국">영국</a>, <a href="/wiki/%EB%8F%85%EC%9D%BC" title="독일">독일</a>과 <a class="mw-redirect" href="/wiki/%EC%95%84%EB%A9%94%EB%A6%AC%EC%B9%B4_%EC%9D%B8%EB%94%94%EC%96%B8" title="아메리카 인디언">아메리카 인디언</a> 계통의 주민들이다.</p>
