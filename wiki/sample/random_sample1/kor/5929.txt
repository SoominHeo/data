<h2><span class="mw-headline" id=".EC.97.AD.EC.82.AC">역사</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%AF%B8%EC%97%90_%ED%98%84&amp;action=edit&amp;section=1" title="부분 편집: 역사">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<div class="thumb tleft">
<div class="thumbinner" style="width:282px;"><a class="image" href="/wiki/%ED%8C%8C%EC%9D%BC:Naiku_01.JPG"><img alt="" class="thumbimage" data-file-height="768" data-file-width="1024" height="210" src="//upload.wikimedia.org/wikipedia/commons/thumb/c/c9/Naiku_01.JPG/280px-Naiku_01.JPG" srcset="//upload.wikimedia.org/wikipedia/commons/thumb/c/c9/Naiku_01.JPG/420px-Naiku_01.JPG 1.5x, //upload.wikimedia.org/wikipedia/commons/thumb/c/c9/Naiku_01.JPG/560px-Naiku_01.JPG 2x" width="280"/></a>
<div class="thumbcaption">
<div class="magnify"><a class="internal" href="/wiki/%ED%8C%8C%EC%9D%BC:Naiku_01.JPG" title="실제 크기로"></a></div>
<a href="/wiki/%EC%9D%B4%EC%84%B8_%EC%8B%A0%EA%B6%81" title="이세 신궁">이세 신궁</a></div>
</div>
</div>
<p>미에 현이 사람이 거주했던 증거는 1만 년 전으로 거슬러 올라간다. <a href="/wiki/%EC%A1%B0%EB%AA%AC_%EC%8B%9C%EB%8C%80" title="조몬 시대">조몬 시대</a>와 <a href="/wiki/%EC%95%BC%EC%9A%94%EC%9D%B4_%EC%8B%9C%EB%8C%80" title="야요이 시대">야요이 시대</a>에 이 지역의 강과 해안을 따라 농업 사회가 형성되기 시작했다. <a href="/wiki/%EC%9D%B4%EC%84%B8_%EC%8B%A0%EA%B6%81" title="이세 신궁">이세 신궁</a>은 야요이 시대에 세워진 것으로 여겨지고 7세기에 현재의 <a href="/wiki/%EB%A9%94%EC%9D%B4%EC%99%80_%EC%A0%95_(%EB%AF%B8%EC%97%90_%ED%98%84)" title="메이와 정 (미에 현)">메이와 정</a>에 재궁(<span lang="ja" xml:lang="ja">斎宮</span>)이 세워졌으며 둘 다 재왕(<span lang="ja" xml:lang="ja">斎王</span>)을 위한 주거와 행정의 중심지 역할을 했다.</p>
<p><a href="/wiki/%EC%97%90%EB%8F%84_%EC%8B%9C%EB%8C%80" title="에도 시대">에도 시대</a>에는 시마 번(志摩藩), 이가 번(伊賀藩), 이세 번(伊勢藩)으로 나뉘어 있었다. <a href="/wiki/%EB%8F%84%EC%B9%B4%EC%9D%B4%EB%8F%84" title="도카이도">도카이도</a>, <a class="new" href="/w/index.php?title=%EC%9D%B4%EC%84%B8_%EA%B0%80%EB%8F%84&amp;action=edit&amp;redlink=1" title="이세 가도 (없는 문서)">이세 가도</a>와 같은 교통망이 건설되었다. 오미나토, 구와나, 아노쓰 같은 항구 마을과 <a href="/wiki/%EC%8A%88%EC%BF%A0%EB%B0%94" title="슈쿠바">역참</a>, <a href="/wiki/%EC%A1%B0%EC%B9%B4%EB%A7%88%EC%B9%98" title="조카마치">성시</a>가 번영하였다.</p>
<p><a href="/wiki/%EB%A9%94%EC%9D%B4%EC%A7%80_%EC%9C%A0%EC%8B%A0" title="메이지 유신">메이지 유신</a> 이후 예전의 <a href="/wiki/%EC%9D%B4%EC%84%B8_%EA%B5%AD" title="이세 국">이세 국</a>, <a href="/wiki/%EC%8B%9C%EB%A7%88_%EA%B5%AD" title="시마 국">시마 국</a>, <a href="/wiki/%EC%9D%B4%EA%B0%80_%EA%B5%AD" title="이가 국">이가 국</a>과 <a href="/wiki/%EA%B8%B0%EC%9D%B4_%EA%B5%AD" title="기이 국">기이 국</a>의 동부는 여러번 재편되었다. <a href="/wiki/1871%EB%85%84" title="1871년">1871년</a>에 기소 3강부터 오늘날의 <a href="/wiki/%EC%93%B0_%EC%8B%9C" title="쓰 시">쓰 시</a> 북쪽까지의 지역은 아노쓰 현이 되었고 남쪽 지역은 와타라이 현이 되었다. <a href="/wiki/1872%EB%85%84" title="1872년">1872년</a>에 아노쓰 현의 현청 소재지는 쓰 시에서 <a href="/wiki/%EC%9A%A7%EC%B9%B4%EC%9D%B4%EC%B9%98_%EC%8B%9C" title="욧카이치 시">욧카이치 시</a>로 이동했고 미에 현으로 개칭되었다. 이듬해에 현청 소재지는 다시 쓰 시로 옮겨졌고 1876년에 남쪽으로 이웃한 와타라이 현과 합병해 현재의 미에 현이 성립되었다.</p>
<p><a href="/wiki/1959%EB%85%84" title="1959년">1959년</a>에 일본 역사상 가장 강력한 태풍인 <a class="mw-redirect" href="/wiki/%EC%9D%B4%EC%84%B8_%EB%A7%8C_%ED%83%9C%ED%92%8D" title="이세 만 태풍">이세 만 태풍</a>이 불어 미에 현의 대부분이 파괴되고 많은 사람들이 사망했다. 경작지는 황폐해지고 방파제는 파괴되었으며 도로와 철도는 심각한 피해를 입었고 많은 사람들이 다치거나 집을 잃었다.</p>
<h2><span class="mw-headline" id=".EC.A7.80.EB.A6.AC">지리</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%AF%B8%EC%97%90_%ED%98%84&amp;action=edit&amp;section=2" title="부분 편집: 지리">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>미에 현은 <a href="/wiki/%EA%B8%B0%EC%9D%B4_%EB%B0%98%EB%8F%84" title="기이 반도">기이 반도</a>(<span lang="ja" xml:lang="ja">紀伊半島</span>)의 동부에 있어 <a href="/wiki/%EC%95%84%EC%9D%B4%EC%B9%98_%ED%98%84" title="아이치 현">아이치 현</a>, <a href="/wiki/%EA%B8%B0%ED%9B%84_%ED%98%84" title="기후 현">기후 현</a>, <a href="/wiki/%EC%8B%9C%EA%B0%80_%ED%98%84" title="시가 현">시가 현</a>, <a href="/wiki/%EA%B5%90%ED%86%A0_%EB%B6%80" title="교토 부">교토 부</a>, <a href="/wiki/%EB%82%98%EB%9D%BC_%ED%98%84" title="나라 현">나라 현</a>, <a href="/wiki/%EC%99%80%EC%B9%B4%EC%95%BC%EB%A7%88_%ED%98%84" title="와카야마 현">와카야마 현</a>과 접하고 있다. 일본의 전통적인 지방 구분으로는 <a href="/wiki/%EA%B8%B4%ED%82%A4_%EC%A7%80%EB%B0%A9" title="긴키 지방">긴키 지방</a>에 속한다. 이 지방 구분은 <a href="/wiki/%EB%A9%94%EC%9D%B4%EC%A7%80_%EC%8B%9C%EB%8C%80" title="메이지 시대">메이지 시대</a>부터 사용되었으며 현재도 학교에서 가르치고 있다.</p>
<p>그러나 미에 현은 방언은 <a href="/wiki/%EA%B5%90%ED%86%A0_%EC%8B%9C" title="교토 시">교토</a>, <a href="/wiki/%EC%98%A4%EC%82%AC%EC%B9%B4_%EC%8B%9C" title="오사카 시">오사카</a> 등과 같은 <a href="/wiki/%EA%B8%B4%ED%82%A4_%EB%B0%A9%EC%96%B8" title="긴키 방언">긴키 방언</a>을 사용하지만 경제적으로는 일부 지역을 제외하고 <a href="/wiki/%EB%82%98%EA%B3%A0%EC%95%BC_%EC%8B%9C" title="나고야 시">나고야 시</a>의 영향력이 강하며, 특히 북부의 구와나 시(<span lang="ja" xml:lang="ja">桑名市</span>) 등은 나고야 시의 <a href="/wiki/%EC%9C%84%EC%84%B1_%EB%8F%84%EC%8B%9C" title="위성 도시">위성 도시</a>가 되어 있다. 따라서 미에 현은 긴키 지방에 속해 있음에도 불구하고 나고야 시를 중심으로 한 <a href="/wiki/%EB%8F%84%EC%B9%B4%EC%9D%B4_%EC%A7%80%EB%B0%A9" title="도카이 지방">도카이 지방</a>에도 포함된다.</p>
<p>미에 현은 남북 길이는 170km, 동서 길이는 80km이며 지리적으로 5개의 지역으로 구분된다. 미에 현의 북반부는 <a href="/wiki/%EC%9D%B4%EC%84%B8_%EB%A7%8C" title="이세 만">이세 만</a> 연안에 평야가 펼쳐지고 있으며 거기에 인구가 집중하고 있다. 북서부에는 스즈카 산맥이 이어지고 있고, 남부는 산과 바다가 접하는 <a href="/wiki/%EB%A6%AC%EC%95%84%EC%8A%A4%EC%8B%9D_%ED%95%B4%EC%95%88" title="리아스식 해안">리아스식 해안</a>이 펼쳐지고 있다. <a href="/wiki/2000%EB%85%84" title="2000년">2000년</a> 조사에서 미에 현의 지형 구분은 삼림이 64.8%, 농지가 11.5%, 주택지가 6.0%, 길이 3.8%, 강이 3.6%가 되어 있다.</p>

<p>미에 현에는 2003년 11월까지 기초자치단체인 시정촌(市町村)이 69개(시 13개, 정 47개, 촌 9개)가 있었으나, 일본 전국에서 진행 중인 시정촌 통합 흐름에 따라 2006년 1월까지 29개 (시 14개, 정 15개)으로 감소되었고, 촌은 없어졌다.</p>
<p>미에 현에서는 옛날의 <a class="mw-redirect" href="/wiki/%EC%9D%BC%EB%B3%B8%EC%9D%98_%EA%B5%AC%EB%8B%88" title="일본의 구니">구니</a>를 기본으로 한 지역 구분을 사용한다. <a href="/wiki/%EC%9D%B4%EC%84%B8_%EA%B5%AD" title="이세 국">이세 국</a>(伊勢國)였던 지역에서는 그 두문자인 "勢"(세이)자에 방위를 붙인 <a class="new" href="/w/index.php?title=%ED%98%B8%EC%BF%A0%EC%84%B8%EC%9D%B4_%EC%A7%80%EC%97%AD&amp;action=edit&amp;redlink=1" title="호쿠세이 지역 (없는 문서)">호쿠세이 지역</a>(北勢), <a class="new" href="/w/index.php?title=%EC%A3%BC%EC%84%B8%EC%9D%B4_%EC%A7%80%EC%97%AD&amp;action=edit&amp;redlink=1" title="주세이 지역 (없는 문서)">주세이 지역</a>(中勢), <a class="new" href="/w/index.php?title=%EB%82%9C%EC%84%B8%EC%9D%B4_%EC%A7%80%EC%97%AD&amp;action=edit&amp;redlink=1" title="난세이 지역 (없는 문서)">난세이 지역</a>(南勢)으로 나누고, <a href="/wiki/%EC%9D%B4%EA%B0%80_%EA%B5%AD" title="이가 국">이가 국</a>(伊賀國)였던 지역은 <a class="mw-redirect" href="/wiki/%EC%9D%B4%EA%B0%80%EB%85%B8%EC%BF%A0%EB%8B%88" title="이가노쿠니">이가 지역</a>, <a href="/wiki/%EA%B8%B0%EC%9D%B4_%EA%B5%AD" title="기이 국">기이 국</a>(紀伊國)였던 지역은 <a class="new" href="/w/index.php?title=%ED%9E%88%EA%B0%80%EC%8B%9C%ED%82%A4%EC%8A%88_%EC%A7%80%EC%97%AD&amp;action=edit&amp;redlink=1" title="히가시키슈 지역 (없는 문서)">히가시키슈 지역</a>(東紀州)이라고 한다. <a href="/wiki/%EC%8B%9C%EB%A7%88_%EA%B5%AD" title="시마 국">시마 국</a>(志摩國)였던 지역은 "난세이 지역"에 포함된다.</p>

<h2><span class="mw-headline" id=".EA.B2.BD.EC.A0.9C">경제</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%AF%B8%EC%97%90_%ED%98%84&amp;action=edit&amp;section=9" title="부분 편집: 경제">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>미에 현은 전통적으로 <a href="/wiki/%EB%8F%84%EC%B9%B4%EC%9D%B4%EB%8F%84" title="도카이도">도카이도</a>와 이세 순례길이 지나는 덕분에 서일본과 동일본의 연결 고리 역할을 해 왔다. 이세 노끈, 요카이치 도기, 스즈카 먹, 이가 도기, 이세 염색과 같은 전통 수공업이 번창하였다. 현의 65%가 숲으로 이루어져 있고 1000km가 넘는 해안선을 가지고 있어 임업과 어업이 발달하였다. 또한 미에 현은 차, 소고기, 진주, 귤 등을 생산한다.</p>
<p>미에 현 북쪽에는 주로 차량과 배와 같은 운송 장비업과 석유 정제업과 같은 중화학 공업의 수많은 제조업 공장들이 입지해 있다. 이와 함께 미에 현의 산업은 반도체, 액정 디스플레이와 같은 첨단 산업으로도 확장되고 있다.</p>
<h2><span class="mw-headline" id=".EC.83.9D.ED.99.9C">생활</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%AF%B8%EC%97%90_%ED%98%84&amp;action=edit&amp;section=10" title="부분 편집: 생활">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>기이 반도 동안에 위치하고, 동쪽으로 이세 만, 남쪽으로는 태평양에 면한다. 해안은 전형적인 리아스식 해안을 이루며, 어업이 발달하였다. 이곳에서는 전갱이·정어리·가자미·방어·가다랭이·다랑어 잡이와 진주·김·조개 등의 양식이 이루어진다.<sup class="reference" id="cite_ref-1"><a href="#cite_note-1">[1]</a></sup></p>
