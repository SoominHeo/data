<p><b>미즈노 다다마사</b>(<span style="font-size: smaller;"><a href="/wiki/%EC%9D%BC%EB%B3%B8%EC%96%B4" title="일본어">일본어</a>:</span> <span lang="ja"><span lang="ja"><ruby><rb>水野 忠政</rb> <rp>(</rp><rt>みずの ただまさ</rt><rp>)</rp></ruby></span></span>, <a href="/wiki/1493%EB%85%84" title="1493년">1493년</a> ~ <a href="/wiki/1543%EB%85%84" title="1543년">1543년</a> <a href="/wiki/8%EC%9B%94_22%EC%9D%BC" title="8월 22일">8월 22일</a>)는 <a href="/wiki/%EC%9D%BC%EB%B3%B8" title="일본">일본</a> <a href="/wiki/%EC%84%BC%EA%B3%A0%EC%BF%A0_%EC%8B%9C%EB%8C%80" title="센고쿠 시대">센고쿠 시대</a>에 살았던 무장으로, <a class="mw-redirect" href="/wiki/%EC%84%BC%EA%B3%A0%EC%BF%A0_%EB%8B%A4%EC%9D%B4%EB%AC%98" title="센고쿠 다이묘">센고쿠 다이묘</a> 중 한 명으로 에도 막부의 초대 쇼군 <a href="/wiki/%EB%8F%84%EC%BF%A0%EA%B0%80%EC%99%80_%EC%9D%B4%EC%97%90%EC%95%BC%EC%8A%A4" title="도쿠가와 이에야스">도쿠가와 이에야스</a>의 외조부이다. 우위문대부 겸 <a href="/wiki/%EC%8B%9C%EB%AA%A8%EC%93%B0%EC%BC%80_%EA%B5%AD" title="시모쓰케 국">시모쓰케 국</a>의 고쿠시 "시모쓰케노카미" 자리를 자칭했다.</p>
<p>원래 <a href="/wiki/%EC%98%A4%EC%99%80%EB%A6%AC_%EA%B5%AD" title="오와리 국">오와리 국</a> <a class="new" href="/w/index.php?title=%EC%98%A4%EA%B0%80%EC%99%80_%EC%84%B1&amp;action=edit&amp;redlink=1" title="오가와 성 (없는 문서)">오가와 성</a> (지금의 아이치 현 지타 군 히가시우라 정에 있었음)을 근거로 <a href="/wiki/%EC%A7%80%ED%83%80_%EB%B0%98%EB%8F%84" title="지타 반도">지타 반도</a> 일대를 지배하고 있었다. <a href="/wiki/1533%EB%85%84" title="1533년">1533년</a>에 <a href="/wiki/%EB%AF%B8%EC%B9%B4%EC%99%80_%EA%B5%AD" title="미카와 국">미카와 국</a> 가리야에 <a href="/wiki/%EA%B0%80%EB%A6%AC%EC%95%BC_%EC%84%B1" title="가리야 성">가리야 성</a>을 새로 지으면서, <a href="/wiki/%EC%98%A4%EB%8B%A4_%EB%85%B8%EB%B6%80%ED%9E%88%EB%8D%B0" title="오다 노부히데">오다 노부히데</a>의 니시미카와 지방 침공에 협력했다. 또한 <a href="/wiki/%EC%98%A4%EC%B9%B4%EC%9E%90%ED%82%A4_%EC%84%B1" title="오카자키 성">오카자키 성</a>을 지배하고 있던 마쓰다이라 히로타다, 가타하라 성을 지배하고 있던 마쓰다이라 이에히로와 사돈 관계를 맺기도 했다.</p>
