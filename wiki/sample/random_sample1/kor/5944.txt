<p><b>미천왕</b>(美川王, ? ~ <a href="/wiki/331%EB%85%84" title="331년">331년</a>, 재위 : <a href="/wiki/300%EB%85%84" title="300년">300년</a> ~ <a href="/wiki/331%EB%85%84" title="331년">331년</a>)은 <a href="/wiki/%EA%B3%A0%EA%B5%AC%EB%A0%A4" title="고구려">고구려</a>의 제15대 <a href="/wiki/%EA%B5%B0%EC%A3%BC" title="군주">군주</a>이다. 이름은 을불(乙弗) 또는 을불리(乙弗利), 우불(憂弗) 등이 전해진다. <a href="/wiki/%EC%84%9C%EC%B2%9C%EC%99%95" title="서천왕">서천왕</a>의 손자이자 고추가(古鄒加) <a href="/wiki/%EB%8F%8C%EA%B3%A0" title="돌고">돌고</a>(咄固)의 아들이다. <b>호양왕</b>(好壤王)이라고도 한다.<sup class="reference" id="cite_ref-.EA.B3.A0.EA.B5.AC.EB.A0.A4.EB.AF.B8.EC.B2.9C.EC.99.95_1-0"><a href="#cite_note-.EA.B3.A0.EA.B5.AC.EB.A0.A4.EB.AF.B8.EC.B2.9C.EC.99.95-1">[1]</a></sup><sup class="reference" id="cite_ref-.EB.AF.B8.EC.B2.9C.EC.99.95_2-0"><a href="#cite_note-.EB.AF.B8.EC.B2.9C.EC.99.95-2">[2]</a></sup></p>

<h2><span class="mw-headline" id=".EC.83.9D.EC.95.A0">생애</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%AF%B8%EC%B2%9C%EC%99%95&amp;action=edit&amp;section=1" title="부분 편집: 생애">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p><a href="/wiki/293%EB%85%84" title="293년">293년</a> 아버지 돌고가 백부인 <a href="/wiki/%EB%B4%89%EC%83%81%EC%99%95" title="봉상왕">봉상왕</a>에게 반역으로 몰려 죽임을 당하자 을불은 도망쳐 신분을 숨기고 고용살이와 소금장수 등을 하면서 숨어 지냈다. 사람들은 봉상왕보다 을불을 더 지지하는 경우가 많았다. <a href="/wiki/300%EB%85%84" title="300년">300년</a> 국상 <a href="/wiki/%EC%B0%BD%EC%A1%B0%EB%A6%AC" title="창조리">창조리</a>(倉助利)가 봉상왕을 폐위할 마음을 품고 을불을 찾아 모셨으며 이후 봉상왕을 폐위한 뒤 왕으로 옹립되었다.</p>
<p>미천왕은 즉위 초부터 <a href="/wiki/%EC%A4%91%EA%B5%AD" title="중국">중국</a> 군현 세력과 치열하게 대립하였다. <a href="/wiki/302%EB%85%84" title="302년">302년</a>에는 <a href="/wiki/%ED%98%84%EB%8F%84%EA%B5%B0" title="현도군">현도군</a>을 공격하여 적 8천여 명을 사로잡았으며 <a href="/wiki/311%EB%85%84" title="311년">311년</a>에는 <a class="new" href="/w/index.php?title=%EC%84%9C%EC%95%88%ED%8F%89&amp;action=edit&amp;redlink=1" title="서안평 (없는 문서)">서안평</a>(西安平)을 점령하여 <a href="/wiki/%EB%82%99%EB%9E%91%EA%B5%B0" title="낙랑군">낙랑군</a>(樂浪郡) 및 <a href="/wiki/%EB%8C%80%EB%B0%A9%EA%B5%B0" title="대방군">대방군</a>(帶方郡)의 보급로를 끊는 데 성공하였다. 이에 따라 <a href="/wiki/313%EB%85%84" title="313년">313년</a>에는 낙랑군의 군벌 장통(張統)이 <a class="mw-redirect" href="/wiki/%EC%9A%94%EB%8F%99" title="요동">요동</a>으로 퇴각하였으며 <a href="/wiki/314%EB%85%84" title="314년">314년</a>에는 대방군을 병합하여 중국 군현 세력을 축출하였다. <a href="/wiki/315%EB%85%84" title="315년">315년</a>에도 다시 <a class="new" href="/w/index.php?title=%ED%98%84%EB%8F%84%EC%84%B1&amp;action=edit&amp;redlink=1" title="현도성 (없는 문서)">현도성</a>을 공격하였다.</p>
<p>중국 군현 세력이 축출된 이후 <a class="new" href="/w/index.php?title=%EC%9A%94%EC%84%9C_%EC%A7%80%EB%B0%A9&amp;action=edit&amp;redlink=1" title="요서 지방 (없는 문서)">요서 지방</a>에서 선비족(鮮卑族)의 일파인 모용부(慕容部)가 세력을 확대함에 따라 대립이 격화되기 시작했다. <a href="/wiki/318%EB%85%84" title="318년">318년</a> 경, <a href="/wiki/%EB%8F%99%EC%A7%84" title="동진">동진</a>(東晉)의 평주 자사(平州刺史)·최비(崔毖)의 작전에 따라 선비족의 다른 일파인 단부(段部)·우문부(宇文部)와 함께 <a href="/wiki/%EB%AA%A8%EC%9A%A9%EC%99%B8" title="모용외">모용외</a>(慕容廆)를 공격하였는데 실패하였으며 이후 모용부가 요동을 장악하면서 본격적인 대립이 시작되었다. <a href="/wiki/319%EB%85%84" title="319년">319년</a>에는 동진의 최비가 요동을 잃고 고구려로 도망쳐 왔으며 하성(河城)을 지키던 여노(如孥)가 모용부의 공격에 포로로 잡히기도 하였다. 이에 미천왕은 군사를 일으켜 요동을 공격하였으며 모용외 역시 두 아들을 보내 침략해 왔다. 이에 미천왕이 휴전을 청하여 잠시 휴전이 성립되었다. <a href="/wiki/320%EB%85%84" title="320년">320년</a> 겨울에도 요동을 침략하였으며 이후 일진일퇴를 거듭하였다. <a href="/wiki/330%EB%85%84" title="330년">330년</a>에는 <a href="/wiki/%ED%9B%84%EC%A1%B0" title="후조">후조</a>(後趙)의 <a href="/wiki/%EC%84%9D%EB%A5%B5" title="석륵">석륵</a>(石勒)에게 사신을 보내 중국 방향에서 <a class="mw-redirect" href="/wiki/%EB%AA%A8%EC%9A%A9%EB%B6%80" title="모용부">모용부</a>를 견제하려 하기도 하였다.</p>
<h2><span class="mw-headline" id=".EC.82.AC.ED.9B.84">사후</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%AF%B8%EC%B2%9C%EC%99%95&amp;action=edit&amp;section=2" title="부분 편집: 사후">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p><a href="/wiki/331%EB%85%84" title="331년">331년</a>에 서거하였으며 미천(美川)의 들에 장사지냈다. <a href="/wiki/342%EB%85%84" title="342년">342년</a>에 고구려로 쳐들어온 <a href="/wiki/%EC%A0%84%EC%97%B0" title="전연">전연</a>(前燕)의 군대가 고구려를 위협하려고 <a class="new" href="/w/index.php?title=%EB%AF%B8%EC%B2%9C%EC%99%95%EB%A6%89&amp;action=edit&amp;redlink=1" title="미천왕릉 (없는 문서)">미천왕릉</a>을 도굴하여 시신을 가져갔으며 <a href="/wiki/%EA%B3%A0%EA%B5%AD%EC%9B%90%EC%99%95" title="고국원왕">고국원왕</a>은 <a href="/wiki/343%EB%85%84" title="343년">343년</a>에 많은 공물을 바치고 시신을 돌려받았다.<sup class="reference" id="cite_ref-.EA.B3.A0.EA.B5.AC.EB.A0.A4.EB.AF.B8.EC.B2.9C.EC.99.95_1-1"><a href="#cite_note-.EA.B3.A0.EA.B5.AC.EB.A0.A4.EB.AF.B8.EC.B2.9C.EC.99.95-1">[1]</a></sup><sup class="reference" id="cite_ref-.EB.AF.B8.EC.B2.9C.EC.99.95_2-1"><a href="#cite_note-.EB.AF.B8.EC.B2.9C.EC.99.95-2">[2]</a></sup></p>
<h2><span class="mw-headline" id=".EC.84.A4.ED.99.94">설화</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%AF%B8%EC%B2%9C%EC%99%95&amp;action=edit&amp;section=3" title="부분 편집: 설화">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p><a href="/wiki/%EA%B3%A0%EA%B5%AC%EB%A0%A4" title="고구려">고구려</a> 미천왕에 관한 전설(傳說)로 《<a href="/wiki/%EC%82%BC%EA%B5%AD%EC%82%AC%EA%B8%B0" title="삼국사기">삼국사기</a>》 〈고구려 본기(本紀)〉 미천왕조에 전한다. 봉상왕의 동생 돌고(咄固)의 아들 을불(乙弗)이 왕의 화를 피해 방랑의 길에 올라 온갖 고생을 하다가 후에 국상 창조리(創造利) 등의 영접을 받아 왕위에 오르게 된다는 설화이다. 방랑중에 있었던 일을 보면 어느 마을 부자집 머슴살이를 하던중 주인이 개구리 울음소리가 듣기 싫다하여 밤새도록 연못에 돌을 던지기도 하였고 그 집을 나와 소금장수를 할 때에는 누명을 쓰고 관가에서 매를 얻어맞기도 하였다.</p>
<div class="rellink boilerplate seealso"><img alt="" data-file-height="128" data-file-width="128" height="16" src="//upload.wikimedia.org/wikipedia/commons/thumb/e/ec/Crystal_Clear_app_xmag.svg/16px-Crystal_Clear_app_xmag.svg.png" srcset="//upload.wikimedia.org/wikipedia/commons/thumb/e/ec/Crystal_Clear_app_xmag.svg/24px-Crystal_Clear_app_xmag.svg.png 1.5x, //upload.wikimedia.org/wikipedia/commons/thumb/e/ec/Crystal_Clear_app_xmag.svg/32px-Crystal_Clear_app_xmag.svg.png 2x" width="16"/> <a href="/wiki/%EA%B3%A0%EA%B5%AC%EB%A0%A4" title="고구려">고구려</a> 미천왕에 대한 <a href="/wiki/%EC%A0%84%EC%84%A4" title="전설">전설</a>이나 <a href="/wiki/%EC%8B%A0%ED%99%94" title="신화">신화</a>에 대해서는 <a class="new" href="/w/index.php?title=%EA%B3%A0%EA%B5%AC%EB%A0%A4%EB%AF%B8%EC%B2%9C%EC%99%95%EC%84%A4%ED%99%94&amp;action=edit&amp;redlink=1" title="고구려미천왕설화 (없는 문서)">고구려미천왕설화</a> 문서를 참고하십시오.</div>
