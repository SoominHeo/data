<p><b>미치 오스틴 모어랜드</b> (Mitchell Austin Moreland ,<a href="/wiki/1985%EB%85%84" title="1985년">1985년</a> <a href="/wiki/9%EC%9B%94_6%EC%9D%BC" title="9월 6일">9월 6일</a> ~ )는 <a href="/wiki/%EB%A9%94%EC%9D%B4%EC%A0%80_%EB%A6%AC%EA%B7%B8_%EB%B2%A0%EC%9D%B4%EC%8A%A4%EB%B3%BC" title="메이저 리그 베이스볼">메이저 리그</a> <a href="/wiki/%EB%B3%B4%EC%8A%A4%ED%84%B4_%EB%A0%88%EB%93%9C%EC%82%AD%EC%8A%A4" title="보스턴 레드삭스">보스턴 레드삭스</a>의 선수이다. <a href="/wiki/2007%EB%85%84" title="2007년">2007년</a> 17라운드 530픽으로 <a href="/wiki/%ED%85%8D%EC%82%AC%EC%8A%A4_%EB%A0%88%EC%9D%B8%EC%A0%80%EC%8A%A4" title="텍사스 레인저스">텍사스 레인저스</a>에 입단 했다. 2010년초 베이스볼 아메리카가 뽑은 텍사스 유망주 순위에서 8위로 선정되었다. 뛰어난 수비력과 파워가 강점이며, 2014년 5월 7일 콜로라도 로키스와의 경기에서 8회말에 투수로 등판해 94마일의 최고 구속을 기록할 만큼 어깨도 강하다.</p>

<h2><span class="mw-headline" id=".EC.83.9D.EC.95.A0_.EB.B0.8F_.EB.8D.B0.EB.B7.94">생애 및 데뷔</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%AF%B8%EC%B9%98_%EB%AA%A8%EC%96%B4%EB%9E%9C%EB%93%9C&amp;action=edit&amp;section=1" title="부분 편집: 생애 및 데뷔">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p><a href="/wiki/%EB%AF%B8%EA%B5%AD" title="미국">미국</a> <a href="/wiki/%EB%AF%B8%EC%8B%9C%EC%8B%9C%ED%94%BC_%EC%A3%BC" title="미시시피 주">미시시피 주</a> <a class="mw-redirect" href="/wiki/%EC%95%84%EB%AA%A8%EB%A6%AC" title="아모리">아모리</a>에서 출생하였다, 미시시피 주립 대학교를 졸업 했으며 <a href="/wiki/2007%EB%85%84" title="2007년">2007년</a> 드래프트 17라운드에서 <a href="/wiki/%ED%85%8D%EC%82%AC%EC%8A%A4_%EB%A0%88%EC%9D%B8%EC%A0%80%EC%8A%A4" title="텍사스 레인저스">텍사스 레인저스</a>에 입단했다. <a href="/wiki/2010%EB%85%84" title="2010년">2010년</a> <a href="/wiki/7%EC%9B%94_29%EC%9D%BC" title="7월 29일">7월 29일</a> 사타구니 부상으로 <a class="mw-redirect" href="/wiki/%EC%9D%B4%EC%95%88_%ED%82%A8%EC%8A%AC%EB%9F%AC" title="이안 킨슬러">이안 킨슬러</a>가 빠진 로스터 한 자리에 모어랜드가 올라오면서 첫 <a href="/wiki/%EB%A9%94%EC%9D%B4%EC%A0%80_%EB%A6%AC%EA%B7%B8_%EB%B2%A0%EC%9D%B4%EC%8A%A4%EB%B3%BC" title="메이저 리그 베이스볼">메이저 리그</a>에 입성을 하였다. <a href="/wiki/2011%EB%85%84" title="2011년">2011년</a> <a href="/wiki/11%EC%9B%94_13%EC%9D%BC" title="11월 13일">11월 13일</a> 15살에 처음 만난 연인 수잔나 히긴스와 2011 시즌 종료 후 <a class="mw-redirect" href="/wiki/%EA%B2%B0%ED%98%BC" title="결혼">결혼</a>에 골인했고, <a href="/wiki/2012%EB%85%84" title="2012년">2012년</a> 첫 아들인 크루를 낳았다. 취미는 <a href="/wiki/%EC%82%AC%EB%83%A5" title="사냥">사냥</a>과 <a href="/wiki/%EB%82%9A%EC%8B%9C" title="낚시">낚시</a>다.</p>
<h2><span class="mw-headline" id=".EC.8B.9C.EC.A6.8C">시즌</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%AF%B8%EC%B9%98_%EB%AA%A8%EC%96%B4%EB%9E%9C%EB%93%9C&amp;action=edit&amp;section=2" title="부분 편집: 시즌">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<h3><span class="mw-headline" id="2010.EB.85.84">2010년</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%AF%B8%EC%B9%98_%EB%AA%A8%EC%96%B4%EB%9E%9C%EB%93%9C&amp;action=edit&amp;section=3" title="부분 편집: 2010년">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p><a href="/wiki/2010%EB%85%84" title="2010년">2010년</a> <a href="/wiki/7%EC%9B%94_29%EC%9D%BC" title="7월 29일">7월 29일</a> <a class="mw-redirect" href="/wiki/%EC%98%A4%ED%81%B4%EB%9E%9C%EB%93%9C_%EC%96%B4%EC%8A%AC%EB%A0%88%ED%8B%B1%EC%8A%A4" title="오클랜드 어슬레틱스">오클랜드 어슬레틱스</a>와의 데뷔전에서 상대투수 빈 마자로로 부터 데뷔 첫 안타를 뽑아내었고 4타수 2안타의 빼어난 활약을 거두면서 <a class="mw-redirect" href="/wiki/%ED%85%8D%EC%82%AC%EC%8A%A4" title="텍사스">텍사스</a>의 7-4 승리를 안겼다. <a href="/wiki/8%EC%9B%94_4%EC%9D%BC" title="8월 4일">8월 4일</a> <a href="/wiki/%EC%8B%9C%EC%95%A0%ED%8B%80_%EB%A7%A4%EB%A6%AC%EB%84%88%EC%8A%A4" title="시애틀 매리너스">시애틀 매리너스</a>전에서는 첫 득점과 타점을 기록했다. 2010시즌 <a href="/wiki/%ED%85%8D%EC%82%AC%EC%8A%A4_%EB%A0%88%EC%9D%B8%EC%A0%80%EC%8A%A4" title="텍사스 레인저스">텍사스 레인저스</a>는 시즌 지구 우승과 함께 <a class="mw-redirect" href="/wiki/%EC%9B%94%EB%93%9C%EC%8B%9C%EB%A6%AC%EC%A6%88" title="월드시리즈">월드시리즈</a>까지 진출하면서 데뷔해에 <a class="mw-redirect" href="/wiki/%EC%9B%94%EB%93%9C%EC%8B%9C%EB%A6%AC%EC%A6%88" title="월드시리즈">월드시리즈</a> 무대까지 밟게 되는 영광을 누리기도 했다. <a href="/wiki/2010%EB%85%84" title="2010년">2010년</a> <a href="/wiki/10%EC%9B%94_30%EC%9D%BC" title="10월 30일">10월 30일</a> <a href="/wiki/%EC%83%8C%ED%94%84%EB%9E%80%EC%8B%9C%EC%8A%A4%EC%BD%94_%EC%9E%90%EC%9D%B4%EC%96%B8%EC%B8%A0" title="샌프란시스코 자이언츠">샌프란시스코 자이언츠</a>와의 <a class="mw-redirect" href="/wiki/%EC%9B%94%EB%93%9C%EC%8B%9C%EB%A6%AC%EC%A6%88" title="월드시리즈">월드시리즈</a> 3차전에서 2회말 조나단 산체스로 부터 3점홈런을 뽑아내 4-2승리를 이끌었다. <a href="/wiki/%ED%85%8D%EC%82%AC%EC%8A%A4_%EB%A0%88%EC%9D%B8%EC%A0%80%EC%8A%A4" title="텍사스 레인저스">텍사스 레인저스</a>는 3차전을 제외한 모든 경기에서 자이언츠에게 패배해 1승 4패로 월드시리즈 타이틀을 <a href="/wiki/%EC%83%8C%ED%94%84%EB%9E%80%EC%8B%9C%EC%8A%A4%EC%BD%94_%EC%9E%90%EC%9D%B4%EC%96%B8%EC%B8%A0" title="샌프란시스코 자이언츠">샌프란시스코 자이언츠</a>에게 내주고 말았다.</p>
<h3><span class="mw-headline" id="2014.EB.85.84">2014년</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%AF%B8%EC%B9%98_%EB%AA%A8%EC%96%B4%EB%9E%9C%EB%93%9C&amp;action=edit&amp;section=4" title="부분 편집: 2014년">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>2013시즌 후 <a href="/wiki/%ED%85%8D%EC%82%AC%EC%8A%A4_%EB%A0%88%EC%9D%B8%EC%A0%80%EC%8A%A4" title="텍사스 레인저스">텍사스 레인저스</a>는 <a class="mw-redirect" href="/wiki/%EB%94%94%ED%8A%B8%EB%A1%9C%EC%9D%B4%ED%8A%B8_%ED%83%80%EC%9D%B4%EA%B1%B0%EC%A6%88" title="디트로이트 타이거즈">디트로이트 타이거즈</a>와의 트레이드를 통해 1루수 자원인 <a href="/wiki/%ED%94%84%EB%A6%B0%EC%8A%A4_%ED%95%84%EB%8D%94" title="프린스 필더">프린스 필더</a>를 영입해 공격력을 강화했다. 그로 인해 1루수로써의 입지가 좁아지게 되면서 2014시즌은 주로 DH로 출전한다. <a href="/wiki/2014%EB%85%84" title="2014년">2014년</a> <a href="/wiki/2%EC%9B%94_14%EC%9D%BC" title="2월 14일">2월 14일</a> 레인저스와 265만 달러의 조건으로 1년 재계약에 합의했다.</p>
