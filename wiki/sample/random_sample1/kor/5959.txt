<h2><span class="mw-headline" id=".ED.81.B4.EB.9F.BD_.EA.B2.BD.EB.A0.A5">클럽 경력</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%AF%B8%EC%BC%80_%ED%95%AD%EC%BC%80&amp;action=edit&amp;section=1" title="부분 편집: 클럽 경력">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>항케는 분데스리가 2001-02 시즌에 <a href="/wiki/FC_%EC%83%AC%EC%BC%80_04" title="FC 샬케 04">FC 샬케 04</a> 선수로 데뷔전을 치루었다. 그는 2005-06 시즌을 앞두고 <a href="/wiki/%ED%86%A0%EB%A7%88%EC%8A%A4_%EC%8A%88%ED%8A%B8%EB%A3%AC%EC%B8%A0" title="토마스 슈트룬츠">토마스 슈트룬츠</a> 감독의 제안 하에 <a href="/wiki/VfL_%EB%B3%BC%ED%94%84%EC%8A%A4%EB%B6%80%EB%A5%B4%ED%81%AC" title="VfL 볼프스부르크">VfL 볼프스부르크</a>로 이적하였다.</p>
<p><a href="/wiki/2007%EB%85%84" title="2007년">2007년</a> <a href="/wiki/5%EC%9B%94" title="5월">5월</a>, 항케는 <a href="/wiki/%ED%95%98%EB%85%B8%EB%B2%84_96" title="하노버 96">하노버 96</a>으로 이적하였고, 11골을 득점하여, <a href="/wiki/%ED%95%98%EB%85%B8%EB%B2%84_96" title="하노버 96">하노버 96</a>이 리그 중위권을 안정적으로 차지하도록 하였다.</p>
<p><a href="/wiki/%ED%95%98%EB%85%B8%EB%B2%84_96" title="하노버 96">하노버 96</a>에서 3시즌 반을 보낸 후, 그는 <a href="/wiki/%EB%B3%B4%EB%A3%A8%EC%8B%9C%EC%95%84_%EB%AC%80%ED%97%A8%EA%B8%80%EB%9D%BC%ED%8A%B8%EB%B0%94%ED%9D%90" title="보루시아 묀헨글라트바흐">보루시아 묀헨글라트바흐</a>로 이적하였다.</p>
<p><a href="/wiki/2013%EB%85%84" title="2013년">2013년</a> <a href="/wiki/7%EC%9B%94_1%EC%9D%BC" title="7월 1일">7월 1일</a>, 항케는 <a href="/wiki/%EB%B3%B4%EB%A3%A8%EC%8B%9C%EC%95%84_%EB%AC%80%ED%97%A8%EA%B8%80%EB%9D%BC%ED%8A%B8%EB%B0%94%ED%9D%90" title="보루시아 묀헨글라트바흐">보루시아 묀헨글라트바흐</a>를 떠나 <a href="/wiki/SC_%ED%94%84%EB%9D%BC%EC%9D%B4%EB%B6%80%EB%A5%B4%ED%81%AC" title="SC 프라이부르크">SC 프라이부르크</a>로 이적하였다.</p>
<h2><span class="mw-headline" id=".EA.B5.AD.EA.B0.80.EB.8C.80.ED.91.9C_.EA.B2.BD.EB.A0.A5">국가대표 경력</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%AF%B8%EC%BC%80_%ED%95%AD%EC%BC%80&amp;action=edit&amp;section=2" title="부분 편집: 국가대표 경력">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>항케는 <a href="/wiki/2005%EB%85%84" title="2005년">2005년</a> <a href="/wiki/6%EC%9B%94_8%EC%9D%BC" title="6월 8일">6월 8일</a>, 2-2로 비긴 <a href="/wiki/%EC%95%84%EB%A5%B4%ED%97%A8%ED%8B%B0%EB%82%98_%EC%B6%95%EA%B5%AC_%EA%B5%AD%EA%B0%80%EB%8C%80%ED%91%9C%ED%8C%80" title="아르헨티나 축구 국가대표팀">아르헨티나</a>와의 <a href="/wiki/2005%EB%85%84_FIFA_%EC%BB%A8%ED%8E%98%EB%8D%94%EB%A0%88%EC%9D%B4%EC%85%98%EC%8A%A4%EC%BB%B5" title="2005년 FIFA 컨페더레이션스컵">2005년 FIFA 컨페더레이션스컵</a> 경기에서 국가대표 데뷔전을 치루었다. 그는 <a href="/wiki/6%EC%9B%94_29%EC%9D%BC" title="6월 29일">6월 29일</a>, <a href="/wiki/%ED%8A%80%EB%8B%88%EC%A7%80_%EC%B6%95%EA%B5%AC_%EA%B5%AD%EA%B0%80%EB%8C%80%ED%91%9C%ED%8C%80" title="튀니지 축구 국가대표팀">튀니지</a>와의 경기에서 첫 골을 기록하였지만, <a href="/wiki/%EB%A9%95%EC%8B%9C%EC%BD%94_%EC%B6%95%EA%B5%AC_%EA%B5%AD%EA%B0%80%EB%8C%80%ED%91%9C%ED%8C%80" title="멕시코 축구 국가대표팀">멕시코 축구 국가대표팀</a>과의 그의 2번째 경기에서 퇴장을 당하였다.</p>
<p>그는 <a href="/wiki/2006%EB%85%84_FIFA_%EC%9B%94%EB%93%9C%EC%BB%B5" title="2006년 FIFA 월드컵">2006년 FIFA 월드컵</a>의 독일 최종 23인 엔트리에 포함되었지만, 컨페더레이션스 컵 마지막 경기에서의 퇴장으로 처음 2경기에 출장할 수 없었다. 그는 <a href="/wiki/%ED%8F%AC%EB%A5%B4%ED%88%AC%EA%B0%88_%EC%B6%95%EA%B5%AC_%EA%B5%AD%EA%B0%80%EB%8C%80%ED%91%9C%ED%8C%80" title="포르투갈 축구 국가대표팀">포르투갈</a>과의 3위 결정전에서 교체 출장하였다.</p>
<h2><span class="mw-headline" id=".EC.82.AC.EC.83.9D.ED.99.9C">사생활</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%AF%B8%EC%BC%80_%ED%95%AD%EC%BC%80&amp;action=edit&amp;section=3" title="부분 편집: 사생활">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>항케와 그의 여자친구 제니 맥마누스는 <a href="/wiki/2007%EB%85%84" title="2007년">2007년</a> <a href="/wiki/3%EC%9B%94_15%EC%9D%BC" title="3월 15일">3월 15일</a>에 태어난 딸 자나타-페이와 <a href="/wiki/2008%EB%85%84" title="2008년">2008년</a> <a href="/wiki/2%EC%9B%94_20%EC%9D%BC" title="2월 20일">2월 20일</a>에 태어난 아들 제이론-케인을 슬하에 두고 있다.</p>
