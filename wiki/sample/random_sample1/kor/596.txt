<h2><span class="mw-headline" id=".EC.88.98.ED.95.99">수학</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=35&amp;action=edit&amp;section=1" title="부분 편집: 수학">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<ul>
<li><a href="/wiki/%ED%95%A9%EC%84%B1%EC%88%98" title="합성수">합성수</a>로, 그 <a href="/wiki/%EC%95%BD%EC%88%98" title="약수">약수</a>는 <a href="/wiki/1" title="1">1</a>, <a href="/wiki/5" title="5">5</a>, <a href="/wiki/7" title="7">7</a>, 35이다. (약수의 합: 48)</li>
<li>5번째 <a href="/wiki/%EC%98%A4%EA%B0%81%EC%88%98" title="오각수">오각수</a>이다.</li>
</ul>
<h2><span class="mw-headline" id=".EA.B3.BC.ED.95.99">과학</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=35&amp;action=edit&amp;section=2" title="부분 편집: 과학">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<ul>
<li><a href="/wiki/%EB%B8%8C%EB%A1%9C%EB%AF%BC" title="브로민">브로민</a>(Br)의 <a class="mw-redirect" href="/wiki/%EC%9B%90%EC%9E%90%EB%B2%88%ED%98%B8" title="원자번호">원자번호</a>.</li>
</ul>
<h2><span class="mw-headline" id=".EB.82.98.EC.9D.B4.28.EB.A7.8C_35.EC.84.B8.29">나이(만 35세)</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=35&amp;action=edit&amp;section=3" title="부분 편집: 나이(만 35세)">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<ul>
<li><a href="/wiki/%EB%AF%B8%EA%B5%AD" title="미국">미국</a> <a href="/wiki/%EB%AF%B8%EA%B5%AD%EC%9D%98_%EB%8C%80%ED%86%B5%EB%A0%B9" title="미국의 대통령">대통령</a>의 나이 하한.</li>
</ul>
<h2><span class="mw-headline" id=".EA.B8.B0.ED.83.80">기타</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=35&amp;action=edit&amp;section=4" title="부분 편집: 기타">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<ul>
<li><a class="mw-redirect" href="/wiki/%ED%95%9C%ED%99%94%EC%9D%B4%EA%B8%80%EC%8A%A4" title="한화이글스">한화이글스</a>의 <a href="/wiki/%EC%9E%A5%EC%A2%85%ED%9B%88" title="장종훈">장종훈</a>의 현역시절번호로 <a href="/wiki/%EC%98%81%EA%B5%AC_%EA%B2%B0%EB%B2%88" title="영구 결번">영구 결번</a> 처리 되었다.</li>
<li><a href="/wiki/KBO_%EB%A6%AC%EA%B7%B8" title="KBO 리그">KBO 리그</a>에서 국민 우익수로 불리고 있는 <a href="/wiki/LG_%ED%8A%B8%EC%9C%88%EC%8A%A4" title="LG 트윈스">LG 트윈스</a> <a href="/wiki/%EC%9D%B4%EC%A7%84%EC%98%81" title="이진영">이진영</a>의 등번호이다.</li>
<li>중학교에서도 단축 수업을 할 때 단위 수업 시간이 35분으로 감축되기도 한다.</li>
<li><a class="mw-redirect" href="/wiki/%EB%AA%A8%EC%B0%A8%EB%A5%B4%ED%8A%B8" title="모차르트">모차르트</a>는 만 35세에 <a class="mw-redirect" href="/wiki/%EC%82%AC%EB%A7%9D" title="사망">사망</a>하였다.</li>
<li><a href="/wiki/%ED%95%98%ED%82%A4" title="하키">하키</a>의 경기 시간은 전, 후반 35분씩이다.</li>
<li><a class="mw-redirect" href="/wiki/%ED%81%AC%EB%A0%88%EC%9A%A9_%EC%8B%A0%EC%A7%B1" title="크레용 신짱">크레용 신짱</a>의 등장인물 <a href="/wiki/%EB%85%B8%ED%95%98%EB%9D%BC_%ED%9E%88%EB%A1%9C%EC%8B%9C" title="노하라 히로시">노하라 히로시</a>(신형만)의 나이가 35세이다.</li>
</ul>
