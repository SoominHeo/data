<p><b>뮈케나이 문명</b> 또는 <b>미케네 문명</b>은 <a href="/wiki/%EA%B7%B8%EB%A6%AC%EC%8A%A4" title="그리스">그리스</a> 남부 <a href="/wiki/%ED%8E%A0%EB%A1%9C%ED%8F%B0%EB%84%A4%EC%86%8C%EC%8A%A4_%EB%B0%98%EB%8F%84" title="펠로폰네소스 반도">펠로폰네소스 반도</a>에 있는 북동부 <a class="mw-redirect" href="/wiki/%EC%95%84%EB%A5%B4%EA%B3%A8%EB%A6%AC%EC%8A%A4" title="아르골리스">아르골리스</a>의 <a href="/wiki/%EB%AF%B8%EC%BC%80%EB%84%A4" title="미케네">미케네</a>의 유적지에서 이름을 따온 <a href="/wiki/%EA%B3%A0%EB%8C%80_%EA%B7%B8%EB%A6%AC%EC%8A%A4" title="고대 그리스">고대 그리스</a>의 문화를 일컫는다. <a class="mw-redirect" href="/wiki/%EC%95%84%ED%85%8C%EB%82%98%EC%9D%B4" title="아테나이">아테나이</a>, <a href="/wiki/%ED%95%84%EB%A1%9C%EC%8A%A4" title="필로스">필로스</a>, <a class="mw-redirect" href="/wiki/%ED%85%8C%EB%B0%94%EC%9D%B4" title="테바이">테바이</a>, <a class="new" href="/w/index.php?title=%ED%8B%B0%EB%A6%B0%EC%8A%A4&amp;action=edit&amp;redlink=1" title="티린스 (없는 문서)">티린스</a>도 미케네 문명의 주요 영역이다. 고대 그리스의 <a href="/wiki/%EC%B2%AD%EB%8F%99%EA%B8%B0_%EC%8B%9C%EB%8C%80" title="청동기 시대">청동기 시대</a> 말기는 <a href="/wiki/%ED%98%B8%EB%A9%94%EB%A1%9C%EC%8A%A4" title="호메로스">호메로스</a>의 <a href="/wiki/%EC%84%9C%EC%82%AC%EC%8B%9C" title="서사시">서사시</a> 등 고대 그리스의 문학과 <a href="/wiki/%EC%8B%A0%ED%99%94" title="신화">신화</a>가 탄생한 시기이다.</p>
<p>무역의 혜택을 입은 <a href="/wiki/%EB%AF%B8%EB%85%B8%EC%8A%A4_%EB%AC%B8%EB%AA%85" title="미노스 문명">미노스 문명</a>과 달리 미케네인들은 정복을 통해 발전했다.</p>

<p>미케네 문명은 <a class="new" href="/w/index.php?title=%ED%97%AC%EB%9D%BC%EB%8F%84%EC%8A%A4_%EB%AC%B8%ED%99%94&amp;action=edit&amp;redlink=1" title="헬라도스 문화 (없는 문서)">헬라도스 문화</a>(Helladic culture)가 <a href="/wiki/%EB%AF%B8%EB%85%B8%EC%8A%A4_%EB%AC%B8%EB%AA%85" title="미노스 문명">미노스 문명</a>의 영향으로 발달하던 기원전 1600년경에서 동부 <a href="/wiki/%EC%A7%80%EC%A4%91%ED%95%B4" title="지중해">지중해</a>에서 <a class="new" href="/w/index.php?title=%EC%B2%AD%EB%8F%99%EA%B8%B0_%EB%AC%B8%EB%AA%85&amp;action=edit&amp;redlink=1" title="청동기 문명 (없는 문서)">청동기 문명</a>이 쇠퇴하던 기원전 1100년경까지 번성했다. 미케네의 멸망은 흔히 <a class="mw-redirect" href="/wiki/%EB%8F%84%EB%A6%AC%EC%95%84%EC%9D%B8" title="도리아인">도리아인</a>의 침략 때문이라고 보는데, 그 밖에 자연 재해나 기후 변화같은 가설도 제기되고 있다.</p>
<p>미케네 문명의 주요 도시는 아르골리스의 미케네와 티린스, 메세니아의 필로스, 아티카의 아테네, 보이오티아의 테베와 <a class="new" href="/w/index.php?title=%EC%98%A4%EB%A5%B4%EC%BD%94%EB%A9%94%EB%85%B8%EC%8A%A4&amp;action=edit&amp;redlink=1" title="오르코메노스 (없는 문서)">오르코메노스</a>, 테살리아의 <a href="/wiki/%EC%9D%B4%EC%98%AC%EC%BD%94%EC%8A%A4" title="이올코스">이올코스</a>이다. <a href="/wiki/%ED%81%AC%EB%A0%88%ED%83%80" title="크레타">크레타</a>에서 미케네인들은 <a href="/wiki/%ED%81%AC%EB%85%B8%EC%86%8C%EC%8A%A4" title="크노소스">크노소스</a>를 정복했다. 또 전형적인 가내 성소의 형태인 <a class="new" href="/w/index.php?title=%EB%A0%88%EB%A5%B4%EB%82%98_%EC%8B%A0%EC%95%99&amp;action=edit&amp;redlink=1" title="레르나 신앙 (없는 문서)">레르나 신앙</a> 같은 의식이 치러진 지역도 있는데, 미케네인들은 일반적인 형태의 신전 건물을 건설하지 않았다. 미케네의 취락은 <a class="mw-redirect mw-disambig" href="/wiki/%EC%97%90%ED%8E%98%EC%9D%B4%EB%A1%9C%EC%8A%A4" title="에페이로스">에페이로스</a><sup class="reference" id="cite_ref-1"><a href="#cite_note-1">[1]</a></sup>, <a class="mw-disambig" href="/wiki/%EB%A7%88%EC%BC%80%EB%8F%84%EB%8B%88%EC%95%84" title="마케도니아">마케도니아</a><sup class="reference" id="cite_ref-2"><a href="#cite_note-2">[2]</a></sup>, <a href="/wiki/%EC%97%90%EA%B2%8C_%ED%95%B4" title="에게 해">에게 해</a>의 섬들, <a class="mw-redirect" href="/wiki/%EC%86%8C%EC%95%84%EC%8B%9C%EC%95%84" title="소아시아">소아시아</a> 해안, <a href="/wiki/%ED%82%A4%ED%94%84%EB%A1%9C%EC%8A%A4" title="키프로스">키프로스</a>에서도 찾아볼 수 있다. <a class="new" href="/w/index.php?title=%EC%84%A0%ED%98%95_B_%EB%AC%B8%EC%9E%90&amp;action=edit&amp;redlink=1" title="선형 B 문자 (없는 문서)">선형 B 문자</a>로 비문이 쓰인 미케네의 유물은 심지어 <a href="/wiki/%EB%8F%85%EC%9D%BC" title="독일">독일</a><sup class="reference" id="cite_ref-3"><a href="#cite_note-3">[3]</a></sup>에서도 발견되며, <a href="/wiki/%EC%A1%B0%EC%A7%80%EC%95%84_(%EA%B5%AD%EA%B0%80)" title="조지아 (국가)">조지아</a>에서 미케네의 검이 발견되기도 했다.<sup class="reference" id="cite_ref-4"><a href="#cite_note-4">[4]</a></sup></p>
<p>미케네 문명은 귀족 전사 계급들이 통치했다. 기원전 1400년경 미케네인들은 <a href="/wiki/%EB%AF%B8%EB%85%B8%EC%8A%A4_%EB%AC%B8%EB%AA%85" title="미노스 문명">미노스 문명</a>의 크레테 섬까지 진출했으며, 미노스 문자(<a class="new" href="/w/index.php?title=%EC%84%A0%ED%98%95_A_%EB%AC%B8%EC%9E%90&amp;action=edit&amp;redlink=1" title="선형 A 문자 (없는 문서)">선형 A 문자</a>)를 받아들여 <a class="new" href="/w/index.php?title=%EC%84%A0%ED%98%95_B_%EB%AC%B8%EC%9E%90&amp;action=edit&amp;redlink=1" title="선형 B 문자 (없는 문서)">선형 B 문자</a>로 차용했다.</p>
<p>미케네인들은 미노스인들을 패배시켰을 뿐 아니라, 그리스 전설에 따르면 미케네의 강력한 적수였던 <a class="mw-redirect" href="/wiki/%EB%8F%84%EC%8B%9C%EA%B5%AD%EA%B0%80" title="도시국가">도시국가</a> <a href="/wiki/%ED%8A%B8%EB%A1%9C%EC%9D%B4%EC%95%84" title="트로이아">트로이아</a>를 두 번이나 패퇴시켰다고 한다. 이 멸망의 증거는 호메로스의 <a href="/wiki/%EC%9D%BC%EB%A6%AC%EC%95%84%EC%8A%A4" title="일리아스">일리아스</a>나 그 밖에 신화에 경도된 사료에서만 찾아 볼 수 있기 때문에, 트로이아와 <a href="/wiki/%ED%8A%B8%EB%A1%9C%EC%9D%B4%EC%95%84_%EC%A0%84%EC%9F%81" title="트로이아 전쟁">트로이아 전쟁</a>의 존재는 확실하지 않다. 1876년 독일 고고학자 <a href="/wiki/%ED%95%98%EC%9D%B8%EB%A6%AC%ED%9E%88_%EC%8A%90%EB%A6%AC%EB%A7%8C" title="하인리히 슐리만">하인리히 슐리만</a>은 소아시아(<a href="/wiki/%ED%84%B0%ED%82%A4" title="터키">터키</a>) 서부 <a class="new" href="/w/index.php?title=%ED%9E%88%EC%82%AC%EB%A6%AC%ED%81%AC&amp;action=edit&amp;redlink=1" title="히사리크 (없는 문서)">히사리크</a>에서 유적을 발굴하고, 이 지역을 트로이아라고 주장했다. 다른 증거에 따르면, 이 유적은 트로이아에 대한 호메로스의 기록과 일치하지 않는다고 주장하지만,<sup class="reference" id="cite_ref-5"><a href="#cite_note-5">[5]</a></sup> 이에 동의하지 않는 사람도 있다.<sup class="reference" id="cite_ref-6"><a href="#cite_note-6">[6]</a></sup></p>
<p>미케네의 귀족들은 아치형으로 된 높은 천장에 크고 둥근 매장실이 있고, 돌로 만든 직선 통로가 있는 <a class="new" href="/w/index.php?title=%EB%B2%8C%EC%A7%91_%EB%AC%B4%EB%8D%A4&amp;action=edit&amp;redlink=1" title="벌집 무덤 (없는 문서)">벌집 무덤</a>(<i>tholoi</i>)에 매장되었다. 미케네인들은 이런 무덤에 망자와 함께 검이나 다른 무구들을 함께 묻었다. 귀족들은 으레 황금 가면, 관, 갑주, 보석 박힌 무기 등도 부장품으로 묻었다. 망자는 앉은 자세로 묻혔으며, 호메로스의 기록에서 <a href="/wiki/%EC%B2%A0%EA%B8%B0_%EC%8B%9C%EB%8C%80" title="철기 시대">철기 시대</a>식으로 <a href="/wiki/%EC%95%84%ED%82%AC%EB%A0%88%EC%9A%B0%EC%8A%A4" title="아킬레우스">아킬레우스</a>나 <a href="/wiki/%ED%8C%8C%ED%8A%B8%EB%A1%9C%ED%81%B4%EB%A1%9C%EC%8A%A4" title="파트로클로스">파트로클로스</a>처럼 화장(火葬)되어 황금 항아리에 담긴 것이 아니라 <a href="/wiki/%EB%AF%B8%EB%9D%BC" title="미라">미라</a>로 매장된 귀족 무덤도 있다.</p>
<p>사제 계급은 아직 나타나지 않았다. 도장, 반지, 이들의 몸짓을 묘사한 봉납한 조상(彫像)을 통해 신과 숭배자가 있었음을 알 수 있는데, 조상을 보면 숭배자들은 팔을 접거나 오른팔을 들거나 혹은 한 손은 이마에 얹는 자세를 취하고 있다. 신은 <a href="/wiki/%EC%98%88%EC%88%98" title="예수">예수</a>가 나타날 때 취하는 자세처럼 두 손을 들거나 팔을 뻗어 무언가를 주거나 받는 모양이다. 미케네의 범신전에는 필로스와 미케네 시대의 크노소스에서 나온 선형 B 문자 비문이 모여있다. 몇몇 신의 이름은 신화 기록의 올림포스의 신에서도 찾아 볼 수 있는데, 알 수 없는 것도 있다. 가령 아레스(Ares) 신은 <a href="/wiki/%EB%B3%84%EB%AA%85" title="별명">별명</a>인 <a class="new" href="/w/index.php?title=%EC%97%90%EB%8B%88%EC%95%8C%EB%A6%AC%EC%98%A4%EC%8A%A4&amp;action=edit&amp;redlink=1" title="에니알리오스 (없는 문서)">에니알리오스</a>(Enyalios)로 나온다. 아폴론은 크노소스에서 "PA-JA-WO"(Paian)이라고 나온다. 더 유명한 것으로는 "A-TA-NA PO-TI-NI-JA"(<i>Athena Potnia</i>, 여신 아테나), "E-RE-U-TI-JA"(<i>Eileithyia</i>, 나중에는 그저 출산 때 숭배되던 신이다), <a class="mw-redirect" href="/wiki/%EB%94%94%EC%98%A4%EB%89%98%EC%86%8C%EC%8A%A4" title="디오뉘소스">디오뉘소스</a>, <a href="/wiki/%ED%8F%AC%EC%84%B8%EC%9D%B4%EB%8F%88" title="포세이돈">포세이돈</a>도 있다. 포세이돈의 경우 이미 "지구를 뒤흔드는 신"으로 알려져 있었으며, 고전기 그리스에는 전해지지 않는 배우자 포세이다(Poseida)와 함께 등장하기도 한다. 또 필로스에서는 분명 <a href="/wiki/%EB%8D%B0%EB%A9%94%ED%85%8C%EB%A5%B4" title="데메테르">데메테르</a>와 <a href="/wiki/%ED%8E%98%EB%A5%B4%EC%84%B8%ED%8F%AC%EB%84%A4" title="페르세포네">페르세포네</a>로 보이는 "두 신"이 함께 등장하기도 한다. 복수의 여신 <a href="/wiki/%EC%97%90%EB%A6%AC%EB%8B%88%EC%97%90%EC%8A%A4" title="에리니에스">에리니에스</a>도 등장한다.</p>
<p>미케네의 궁정 <a href="/wiki/%ED%94%84%EB%A0%88%EC%8A%A4%EC%BD%94" title="프레스코">프레스코</a>화<sup class="reference" id="cite_ref-7"><a href="#cite_note-7">[7]</a></sup>도 발견되었는데, 특히 필로스, 미케네, 오르코메노스, 테바이, 티린스가 유명하며, 궁정이 아닌 개인 소유의 것도 일부 있다. 초기 프레스코 장식은 LH IIA 시기(기원전 1,500년경)의 것이다. 이것들은 <a class="mw-redirect" href="/wiki/%ED%82%A4%ED%81%B4%EB%9D%BC%EB%8D%B0%EC%8A%A4" title="키클라데스">키클라데스</a>의 영향을 받은 것과 미노스에서 직접 유래된 것 모두 미노스 전통을 고수하고 있으며, 경우에 따라 장식을 줄이고 자신들의 지역에 맞는 주제로 꾸민 것도 있다. 가령 알현실 안에 사자나 날개 없는 그리핀을 두거나, 복도에 조상을 줄 지어 놓거나 하는 식이다. 미노스식의 동물 장식과 달리, 미케네는 자연에 대해서도 동물을 묘사할때 오직 인간에 중점을 두거나, 사냥물로서만 나타내고 있다. 황소가 뛰는 프레스코판은 미케네와 티린스에서 발견된다.</p>

<p>기원전 1,100년경 미케네 문명은 멸망한다. 수많은 도시들이 약탈되었고, 이 지역은 역사가들이 <a href="/wiki/%EA%B7%B8%EB%A6%AC%EC%8A%A4_%EC%95%94%ED%9D%91%EA%B8%B0" title="그리스 암흑기">그리스 암흑기</a>로 부르는 시대로 넘어가며, 어떤 미케네인들은 <a href="/wiki/%EC%95%84%EB%82%98%ED%86%A8%EB%A6%AC%EC%95%84" title="아나톨리아">아나톨리아</a>나 다른 그리스 섬들 혹은 키프로스로 도주하기도 했다. 이 시기에 그리스는 인구 감소를 겪었으며, 궁정과 관련된 제한적인 문자도 사라지고 말았다. 역사가들은 전통적으로 이 쇠퇴의 원인을 외부나 다른 <a href="/wiki/%EA%B7%B8%EB%A6%AC%EC%8A%A4%EC%9D%B8" title="그리스인">그리스인</a>(이 지역 주민을 복속게 했다는 도리아인)의 침략 때문이라고 여겼다. 그러나 일련의 지진이나 대규모 가뭄과 같은 자연 재해나 기후 변화와 같은 가설도 제기되고 있다.</p>
