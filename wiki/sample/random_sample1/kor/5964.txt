<p><b>미코얀-구레비치 MiG-25</b>(<span style="font-size: smaller;"><a href="/wiki/%EB%9F%AC%EC%8B%9C%EC%95%84%EC%96%B4" title="러시아어">러시아어</a>:</span> <span lang="ru" xml:lang="ru">Микоян-Гуревич МиГ-25</span>), 나토 코드네임 <b>폭스배트</b>(Foxbat<font color="gray"><small>→<a class="new" href="/w/index.php?title=%ED%81%B0%EB%B0%95%EC%A5%90&amp;action=edit&amp;redlink=1" title="큰박쥐 (없는 문서)">큰박쥐</a></small></font>)는 <a class="mw-redirect" href="/wiki/%EC%86%8C%EB%B9%84%EC%97%90%ED%8A%B8_%EC%97%B0%EB%B0%A9" title="소비에트 연방">소비에트 연방</a> 공군의 <a href="/wiki/%EC%B4%88%EC%9D%8C%EC%86%8D" title="초음속">초음속</a> <a href="/wiki/%EC%9A%94%EA%B2%A9%EA%B8%B0" title="요격기">요격기</a>이자 <a href="/wiki/%EC%A0%95%EC%B0%B0%EA%B8%B0" title="정찰기">정찰기</a>로, 역사상 취역한 군용기, 특히 전투기·요격기 중에서 가장 빠른 기체이다. 구 소련의 미코얀-구레비치 설계국에서 제작한 MiG-25의 원형기는 1964년에 처녀비행을 가졌으며, 1970년에 일선에 취역했다. 최고 속도는 <a href="/wiki/%EB%A7%88%ED%95%98_%EC%88%98" title="마하 수">마하</a> 2.83+에 달하며, 엔진의 파괴를 무릅쓰고 가속하면 마하 3.2까지 비행이 가능하다. 이뿐 아니라 강력한 <a href="/wiki/%EB%A0%88%EC%9D%B4%EB%8D%94" title="레이더">레이더</a>와 4발의 <a class="mw-redirect" href="/wiki/%EA%B3%B5%EB%8C%80%EA%B3%B5_%EB%AF%B8%EC%82%AC%EC%9D%BC" title="공대공 미사일">공대공 미사일</a>을 장착했다.</p>
<p>최초로 정찰 사진에 포착되었을 당시, 커다란 주익 덕분에 고기동성 전투기가 아니냐는 의혹을 샀다. 이때 미국 측 역시 <a href="/wiki/%EB%B2%A0%ED%8A%B8%EB%82%A8_%EC%A0%84%EC%9F%81" title="베트남 전쟁">베트남 전쟁</a>에서 운용할 기동성이 높은 기체를 개발하려고 시도하던 시기였다. 그랬기에 MiG-25의 등장은 <a href="/wiki/%EC%84%9C%EC%96%91" title="서양">서방 세계</a> 각국에 충격을 유발했으며, 1960년대 후반 <a href="/wiki/%EB%A7%A5%EB%8F%84%EB%84%AC%EB%8D%94%EA%B8%80%EB%9F%AC%EC%8A%A4_F-15_%EC%9D%B4%EA%B8%80" title="맥도넬더글러스 F-15 이글">F-15 이글</a> 개발에 박차를 가하게 되는 원인이 된다. 1976년에 <a class="new" href="/w/index.php?title=%EB%B9%85%ED%86%A0%EB%A5%B4_%EB%B2%A8%EB%A0%8C%EC%BD%94&amp;action=edit&amp;redlink=1" title="빅토르 벨렌코 (없는 문서)">빅토르 벨렌코</a>가 <a href="/wiki/%EC%9D%BC%EB%B3%B8" title="일본">일본</a>을 통해 미국으로 <a href="/wiki/%EB%B9%85%ED%86%A0%EB%A5%B4_%EB%B2%A8%EB%A0%8C%EC%BD%94_%EB%A7%9D%EB%AA%85_%EC%82%AC%EA%B1%B4" title="빅토르 벨렌코 망명 사건">망명</a>하면서 MiG-25의 실제 성능이 밝혀졌다. MiG-25는 기동성보다는 고속도, 고고도에서 우위를 가진 기체였으며, 주된 목적은 <a href="/wiki/%ED%8F%AD%EA%B2%A9%EA%B8%B0" title="폭격기">폭격기</a> 요격이었다. MiG-25의 주익이 커다란 이유는 기체 중량이 무거웠기 때문이었다. 즉 서방 세계를 공포에 몰아넣은 MiG-25의 정체는 철저히 방어에 특화된 요격기였던 것이다.</p>
<p>1,190 대의 생산량을 남기고, 1984년 부로 MiG-25 시리즈의 생산은 종료되었다. 위의 성능을 둘러싼 의혹, 그리고 F-15의 개발로 이어지는 해프닝에서 보여지듯 ‘<a href="/wiki/%EB%83%89%EC%A0%84" title="냉전">냉전</a>의 상징적 존재’였던 MiG-25는 현재 <a href="/wiki/%EB%9F%AC%EC%8B%9C%EC%95%84" title="러시아">러시아</a>를 비롯해 과거 <a class="mw-redirect" href="/wiki/%EC%86%8C%EB%B9%84%EC%97%90%ED%8A%B8_%EC%97%B0%EB%B0%A9" title="소비에트 연방">소비에트 연방</a>에 속했던 국가들에서 제한적으로 현역으로 사용되고 있다. 속도와 고고도 비행 능력면에서 미국의 <a href="/wiki/SR-71_%EB%B8%94%EB%9E%99%EB%B2%84%EB%93%9C" title="SR-71 블랙버드">SR-71 블랙버드</a>에게는 뒤지지만 여전히 SR-71의 뒤를 잇는 세계 2위 기체의 자리를 지키고 있다.<sup class="reference" id="cite_ref-Global_Aircraft_--_Top_50_Fastest_Aircraft_2-0"><a href="#cite_note-Global_Aircraft_--_Top_50_Fastest_Aircraft-2">[2]</a></sup></p>
