<p><b>미트리다테스 6세</b>(<span style="font-size: smaller;"><a href="/wiki/%EA%B7%B8%EB%A6%AC%EC%8A%A4%EC%96%B4" title="그리스어">그리스어</a>:</span> <span lang="el" xml:lang="el">Μιθριδάτης</span>, <a href="/wiki/%EA%B8%B0%EC%9B%90%EC%A0%84_132%EB%85%84" title="기원전 132년">기원전 132년</a> - <a href="/wiki/%EA%B8%B0%EC%9B%90%EC%A0%84_63%EB%85%84" title="기원전 63년">기원전 63년</a>)는 <b>미트리다테스 대왕</b>으로 알려져 있는 <a href="/wiki/%EC%95%84%EB%82%98%ED%86%A8%EB%A6%AC%EC%95%84" title="아나톨리아">아나톨리아</a> 북부의 <a class="mw-redirect" href="/wiki/%ED%8F%B0%ED%88%AC%EC%8A%A4" title="폰투스">폰투스 왕국</a>의 왕이다. <a href="/wiki/%EB%A1%9C%EB%A7%88_%EA%B3%B5%ED%99%94%EC%A0%95" title="로마 공화정">로마 공화정</a> 말기 세명의 유명한 로마 장군(<a class="mw-redirect" href="/wiki/%EC%88%A0%EB%9D%BC" title="술라">술라</a>, <a class="mw-redirect" href="/wiki/%EB%A3%A8%EC%BF%A8%EB%A3%A8%EC%8A%A4" title="루쿨루스">루쿨루스</a>, <a class="mw-redirect" href="/wiki/%ED%8F%BC%ED%8E%98%EC%9D%B4%EC%9A%B0%EC%8A%A4" title="폼페이우스">폼페이우스</a>)과 차례로 대적한 것으로 유명하다.</p>

<h2><span class="mw-headline" id=".EC.B4.88.EA.B8.B0">초기</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%AF%B8%ED%8A%B8%EB%A6%AC%EB%8B%A4%ED%85%8C%EC%8A%A4_6%EC%84%B8_%EB%A9%94%EA%B0%80%EC%8A%A4&amp;action=edit&amp;section=1" title="부분 편집: 초기">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>미트리다테스 6세는 <a class="mw-redirect" href="/wiki/%ED%8F%B0%ED%88%AC%EC%8A%A4" title="폰투스">폰투스</a>의 왕, 미트리다테스 5세의 아들로 <a href="/wiki/%EA%B8%B0%EC%9B%90%EC%A0%84_120%EB%85%84" title="기원전 120년">기원전 120년</a>경 아버지가 죽자 어머니 게스파이피리스가 왕국을 통치했다. <a href="/wiki/%EA%B8%B0%EC%9B%90%EC%A0%84_115%EB%85%84" title="기원전 115년">기원전 115년</a> 그는 어머니로부터 왕위를 빼앗고 왕위에 방해되는 수많은 형제들을 죽인후 통치를 확고히 하였다. 당시 <a href="/wiki/%EC%95%84%EB%82%98%ED%86%A8%EB%A6%AC%EC%95%84" title="아나톨리아">아나톨리아</a>의 폰투스 영토는 미트리다테스 5세가 죽은 뒤 크게 줄어들어 있었는데 파플라고니아가 독립했고 <a href="/wiki/%EA%B8%B0%EC%9B%90%EC%A0%84_116%EB%85%84" title="기원전 116년">기원전 116년</a> <a href="/wiki/%ED%94%84%EB%A6%AC%EA%B8%B0%EC%95%84" title="프리기아">프리기아</a>는 <a href="/wiki/%EB%A1%9C%EB%A7%88_%EA%B3%B5%ED%99%94%EC%A0%95" title="로마 공화정">로마 공화정</a>의 아시아 속주에 통합되었다.</p>
<p>야심찬 미트리다테스는 <a href="/wiki/%ED%9D%91%ED%95%B4" title="흑해">흑해</a> 연안에서 폰투스의 영향력을 확대하기로 결심하고 <a class="mw-redirect" href="/wiki/%ED%81%AC%EB%A6%AC%EB%AF%B8%EC%95%84" title="크리미아">크리미아</a>과 <a href="/wiki/%EC%BD%9C%ED%82%A4%EC%8A%A4" title="콜키스">콜키스</a>를 폰투스에 합병했다. 그 지역에 사는 그리스인들에게는 <a class="mw-redirect" href="/wiki/%EC%8A%A4%ED%82%A4%ED%83%80%EC%9D%B4" title="스키타이">스키타이</a>로부터 지켜준다는 명목으로 충성을 약속 받았다. 미트리다테스는 <a href="/wiki/%ED%8C%8C%ED%94%8C%EB%9D%BC%EA%B3%A0%EB%8B%88%EC%95%84" title="파플라고니아">파플라고니아</a>와 <a href="/wiki/%EA%B0%88%EB%9D%BC%ED%8B%B0%EC%95%84" title="갈라티아">갈라티아</a>를 분리하여 <a href="/wiki/%EB%B9%84%ED%8B%B0%EB%8B%88%EC%95%84" title="비티니아">비티니아</a> 왕 <a class="new" href="/w/index.php?title=%EB%8B%88%EC%BD%94%EB%A9%94%EB%8D%B0%EC%8A%A4_3%EC%84%B8&amp;action=edit&amp;redlink=1" title="니코메데스 3세 (없는 문서)">니코메데스 3세</a>와 나누어 가지는 데 성공하고 다시 <a href="/wiki/%EC%B9%B4%ED%8C%8C%EB%8F%84%ED%82%A4%EC%95%84" title="카파도키아">카파도키아</a>를 놓고는 비티니아와 전쟁을 벌였다. <a href="/wiki/%EA%B8%B0%EC%9B%90%EC%A0%84_95%EB%85%84" title="기원전 95년">기원전 95년</a>과 <a href="/wiki/%EA%B8%B0%EC%9B%90%EC%A0%84_92%EB%85%84" title="기원전 92년">기원전 92년</a> 로마는 니코메데스와의 동맹을 구실로 두 차례에 걸쳐 개입하였다.</p>
<h2><span class="mw-headline" id=".EB.AF.B8.ED.8A.B8.EB.A6.AC.EB.8B.A4.ED.85.8C.EC.8A.A4_.EC.A0.84.EC.9F.81">미트리다테스 전쟁</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%AF%B8%ED%8A%B8%EB%A6%AC%EB%8B%A4%ED%85%8C%EC%8A%A4_6%EC%84%B8_%EB%A9%94%EA%B0%80%EC%8A%A4&amp;action=edit&amp;section=2" title="부분 편집: 미트리다테스 전쟁">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>비티니아에서는 로마의 조정을 받는 <a class="mw-redirect" href="/wiki/%EB%8B%88%EC%BD%94%EB%A9%94%EB%8D%B0%EC%8A%A4_4%EC%84%B8" title="니코메데스 4세">니코메데스 4세</a>가 왕이 되자 그는 니코메데스를 몰아낼 계획을 세웠으나 실패했다. 니코메데스는 로마의 사주를 받아 <a href="/wiki/%EA%B8%B0%EC%9B%90%EC%A0%84_88%EB%85%84" title="기원전 88년">기원전 88년</a> 폰투스와 전쟁을 벌였고 미트리다테스는 비티니아로 쳐들어가 영토의 대부분을 점령해 버렸다. 로마는 이에 반발했으나 그는 스스로를 헬레니즘의 보호자로 자처하며 자신의 야심을 로마와 그리스간의 전쟁으로 포장하여 <a href="/wiki/%EC%9D%B4%EC%98%A4%EB%8B%88%EC%95%84" title="이오니아">이오니아</a> 지방의 그리스인들을 부추겼다. <a href="/wiki/%EC%95%84%EB%A5%B4%EB%A9%94%EB%8B%88%EC%95%84_%EC%99%95%EA%B5%AD" title="아르메니아 왕국">아르메니아 왕국</a>의 티그라네스 2세는 미트리다테스와 동맹을 맺고 로마에 대항하기로 하였다.</p>
<h3><span class="mw-headline" id=".EC.A0.9C1.EC.B0.A8_.EB.AF.B8.ED.8A.B8.EB.A6.AC.EB.8B.A4.ED.85.8C.EC.8A.A4_.EC.A0.84.EC.9F.81.28.EA.B8.B0.EC.9B.90.EC.A0.84_88.EB.85.84_-_84.EB.85.84.29">제1차 미트리다테스 전쟁(<a href="/wiki/%EA%B8%B0%EC%9B%90%EC%A0%84_88%EB%85%84" title="기원전 88년">기원전 88년</a> - <a href="/wiki/%EA%B8%B0%EC%9B%90%EC%A0%84_84%EB%85%84" title="기원전 84년">84년</a>)</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%AF%B8%ED%8A%B8%EB%A6%AC%EB%8B%A4%ED%85%8C%EC%8A%A4_6%EC%84%B8_%EB%A9%94%EA%B0%80%EC%8A%A4&amp;action=edit&amp;section=3" title="부분 편집: 제1차 미트리다테스 전쟁(기원전 88년 - 84년)">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>이에 격분한 로마는 <a class="mw-redirect" href="/wiki/%EC%88%A0%EB%9D%BC" title="술라">루키우스 코르넬리우스 술라</a>를 총사령관으로 소아시아에 파견하였고 미트리다테스는 술라를 상대로 4년간의 전쟁을 벌였다. 처음에 그리스인들은 폰투스의 편을 들었으나 점차 술라가 전쟁에서 이기자 로마의 편으로 돌아섰다. 미트리다테스는 아나톨리아의 로마인 8만명을 학살하면서 그리스인에게 결사항전을 주장하였으나 결국 그리스인들의 지지를 돌이키는 데 실패하였다. 한편 로마 본토에서 <a href="/wiki/%EA%B0%80%EC%9D%B4%EC%9A%B0%EC%8A%A4_%EB%A7%88%EB%A6%AC%EC%9A%B0%EC%8A%A4" title="가이우스 마리우스">가이우스 마리우스</a> 문제로 골치 아픈 술라는 서둘러 미트리다테스와 평화협상을 맺고 로마로 돌아갔기 때문에 폰투스는 완전히 괴멸되는 것을 피할 수 있었다.</p>
<h3><span class="mw-headline" id=".EC.A0.9C2.EC.B0.A8_.EB.AF.B8.ED.8A.B8.EB.A6.AC.EB.8B.A4.ED.85.8C.EC.8A.A4_.EC.A0.84.EC.9F.81.28.EA.B8.B0.EC.9B.90.EC.A0.84_83.EB.85.84_-_82.EB.85.84.29">제2차 미트리다테스 전쟁(<a href="/wiki/%EA%B8%B0%EC%9B%90%EC%A0%84_83%EB%85%84" title="기원전 83년">기원전 83년</a> - <a href="/wiki/%EA%B8%B0%EC%9B%90%EC%A0%84_82%EB%85%84" title="기원전 82년">82년</a>)</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%AF%B8%ED%8A%B8%EB%A6%AC%EB%8B%A4%ED%85%8C%EC%8A%A4_6%EC%84%B8_%EB%A9%94%EA%B0%80%EC%8A%A4&amp;action=edit&amp;section=4" title="부분 편집: 제2차 미트리다테스 전쟁(기원전 83년 - 82년)">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>술라는 로마로 돌아가면서 루키우스 리키니우스 무레나를 소아시아에 남겨 놓고 현상을 유지하라고 하였는데 무레나는 <a href="/wiki/%EA%B8%B0%EC%9B%90%EC%A0%84_83%EB%85%84" title="기원전 83년">기원전 83년</a> 미트리다테스가 재무장을 준비한다는 구실로 폰투스를 쳐들어갔다. 이듬해 미트리다테스는 로마를 물리치는 데 성공했다.</p>
<h3><span class="mw-headline" id=".EC.A0.9C3.EC.B0.A8_.EB.AF.B8.ED.8A.B8.EB.A6.AC.EB.8B.A4.ED.85.8C.EC.8A.A4_.EC.A0.84.EC.9F.81.28.EA.B8.B0.EC.9B.90.EC.A0.84_75.EB.85.84_-_65.EB.85.84.29">제3차 미트리다테스 전쟁(<a href="/wiki/%EA%B8%B0%EC%9B%90%EC%A0%84_75%EB%85%84" title="기원전 75년">기원전 75년</a> - <a href="/wiki/%EA%B8%B0%EC%9B%90%EC%A0%84_65%EB%85%84" title="기원전 65년">65년</a>)</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%AF%B8%ED%8A%B8%EB%A6%AC%EB%8B%A4%ED%85%8C%EC%8A%A4_6%EC%84%B8_%EB%A9%94%EA%B0%80%EC%8A%A4&amp;action=edit&amp;section=5" title="부분 편집: 제3차 미트리다테스 전쟁(기원전 75년 - 65년)">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>미트리다테스와 로마는 끈임없이 크고 작은 반목을 계속하다가 <a href="/wiki/%EA%B8%B0%EC%9B%90%EC%A0%84_75%EB%85%84" title="기원전 75년">기원전 75년</a> 전면전으로 확대되었다. 미트리다테스는 칼케돈에서 <a href="/wiki/%EC%A7%91%EC%A0%95%EA%B4%80" title="집정관">집정관</a> 마리우스 아우렐리우스 코타를 패배시켰으나, 뒤이어 투입된 <a class="mw-redirect" href="/wiki/%EB%A3%A8%EC%BF%A8%EB%A3%A8%EC%8A%A4" title="루쿨루스">루쿨루스</a>에 의해 계속해서 패했다. 그는 <a href="/wiki/%EA%B8%B0%EC%9B%90%EC%A0%84_73%EB%85%84" title="기원전 73년">기원전 73년</a>에 키지쿠스 바깥으로 쫓겨났고 루쿠룰스에 의해 추격당해 동맹국인 아르메니아의 티그라네스 2세에게 도망쳤다. 루쿨루스는 티그라노케르타, 아르타크사타에서 큰 승리를 거두고 아르메니아 깊숙이까지 추격하였으나 그 뒤 부대 내에서 일어난 반란으로 궁지에 빠졌다.</p>
<p><a href="/wiki/%EA%B8%B0%EC%9B%90%EC%A0%84_66%EB%85%84" title="기원전 66년">기원전 66년</a> 로마는 루쿨루스를 소환하고 대신 폼페이우스를 투입하였고 폼페이우스는 미트리다테스와 티그라네스 2세의 아르메니아를 모두 완패시켰다. 폼페이우스는 사실상 2년 만에 전투를 완결하고 소아시아를 완전한 로마의 영향력 아래 두는 데 성공했다.</p>
<h2><span class="mw-headline" id=".EC.A3.BD.EC.9D.8C.EA.B3.BC_.EA.B7.B8.EC.9D.B4.ED.9B.84">죽음과 그이후</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%AF%B8%ED%8A%B8%EB%A6%AC%EB%8B%A4%ED%85%8C%EC%8A%A4_6%EC%84%B8_%EB%A9%94%EA%B0%80%EC%8A%A4&amp;action=edit&amp;section=6" title="부분 편집: 죽음과 그이후">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>미트리다테스는 병력과 권력을 모두 잃고 계속 퇴각하였고 <a href="/wiki/%EA%B8%B0%EC%9B%90%EC%A0%84_64%EB%85%84" title="기원전 64년">기원전 64년</a> 킴메리 보스포루스 반도에 있는 판티카파이움에서 병력을 모아 로마에 반격을 시도하려 했으나 아들인 <a href="/wiki/%ED%8C%8C%EB%A5%B4%EB%82%98%EC%BC%80%EC%8A%A4_2%EC%84%B8" title="파르나케스 2세">파르나케스 2세</a>의 반란군에 의해 배반당하고 도망쳤다. 절망에 빠진 미트리다테스는 독약을 먹고 자살하려다 실패하고 부하에게 자신을 죽이도록 명령했다.</p>
<p>그의 시체는 폼페이우스에게 보내졌고, 폼페이우스는 그를 폰투스의 수도 시노페의 왕실묘지에 매장했다.</p>
