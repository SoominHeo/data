<h2><span class="mw-headline" id=".EC.83.9D.EC.95.A0">생애</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%AF%BC%EC%98%81%EC%B0%AC&amp;action=edit&amp;section=1" title="부분 편집: 생애">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>여흥 민씨 세도가의 실세였다가 <a href="/wiki/%EC%9E%84%EC%98%A4%EA%B5%B0%EB%9E%80" title="임오군란">임오군란</a> 때 피살된 <a href="/wiki/%EB%AF%BC%EA%B2%B8%ED%98%B8" title="민겸호">민겸호</a>의 차남으로 <a href="/wiki/%EB%AF%BC%EC%98%81%ED%99%98" title="민영환">민영환</a>이 그의 형이다. <a href="/wiki/%EB%8C%80%ED%95%9C%EC%A0%9C%EA%B5%AD_%EA%B3%A0%EC%A2%85" title="대한제국 고종">고종</a>의 생모이자 <a href="/wiki/%ED%9D%A5%EC%84%A0%EB%8C%80%EC%9B%90%EA%B5%B0" title="흥선대원군">흥선대원군</a>의 처 <a href="/wiki/%EC%97%AC%ED%9D%A5%EB%B6%80%EB%8C%80%EB%B6%80%EC%9D%B8" title="여흥부대부인">여흥부대부인</a> 민씨는 그의 고모이다. 둘째 큰 아버지인 <a href="/wiki/%EB%AF%BC%EC%8A%B9%ED%98%B8" title="민승호">민승호</a>가 <a href="/wiki/%EB%AA%85%EC%84%B1%ED%99%A9%ED%9B%84" title="명성황후">명성황후</a>의 아버지인 <a href="/wiki/%EB%AF%BC%EC%B9%98%EB%A1%9D" title="민치록">민치록</a>에게 입적되었기에 민영환과 민영찬 형제를 명성황후의 조카로 오인되기도 하지만, 입적이란 친부모형제와의 법적 관계가 절연되는 것이기에 <a href="/wiki/%EB%AF%BC%EC%8A%B9%ED%98%B8" title="민승호">민승호</a>가 <a href="/wiki/%EB%AA%85%EC%84%B1%ED%99%A9%ED%9B%84" title="명성황후">명성황후</a>의 양오빠가 되었어도 <a href="/wiki/%EB%AA%85%EC%84%B1%ED%99%A9%ED%9B%84" title="명성황후">명성황후</a>와 이들 형제의 관계는 여전히 13촌 먼 친척에 불과하다.</p>
<p>한 살 차이로 같은 또래인 <a href="/wiki/%EB%8C%80%ED%95%9C%EC%A0%9C%EA%B5%AD_%EC%88%9C%EC%A2%85" title="대한제국 순종">순종</a>과 어릴 때 함께 공부했으며, <a href="/wiki/1889%EB%85%84" title="1889년">1889년</a> <a href="/wiki/%EC%A1%B0%EC%84%A0%EC%9D%98_%EA%B3%BC%EA%B1%B0_%EC%A0%9C%EB%8F%84" title="조선의 과거 제도">과거</a>에 합격해 <a href="/wiki/%ED%99%8D%EB%AC%B8%EA%B4%80" title="홍문관">홍문관</a>에서 관직을 시작해 요직에 근무했다. <a href="/wiki/1897%EB%85%84" title="1897년">1897년</a>에는 <a class="mw-redirect" href="/wiki/%ED%95%9C%EC%84%B1%EC%9D%80%ED%96%89" title="한성은행">한성은행</a> 부장(부은행장)에 임명되기도 했다.</p>
<p><a href="/wiki/1900%EB%85%84" title="1900년">1900년</a> <a class="mw-redirect" href="/wiki/%ED%8C%8C%EB%A6%AC_%EC%8B%9C" title="파리 시">파리</a>에서 열린 <a href="/wiki/%EC%84%B8%EA%B3%84_%EB%B0%95%EB%9E%8C%ED%9A%8C" title="세계 박람회">만국박람회</a>에 특파대사로 파견되었고 <a href="/wiki/1902%EB%85%84" title="1902년">1902년</a> <a href="/wiki/%ED%95%9C%EB%B6%88_%EA%B4%80%EA%B3%84#.EC.A3.BC_.ED.94.84.EB.9E.91.EC.8A.A4_.EB.8C.80.ED.95.9C.EB.AF.BC.EA.B5.AD_.EB.8C.80.EC.82.AC.EA.B4.80" title="한불 관계">주프랑스 공사</a>에 임명되었다. <a href="/wiki/1903%EB%85%84" title="1903년">1903년</a> <a href="/wiki/%EC%8A%A4%EC%9C%84%EC%8A%A4" title="스위스">스위스</a>에서 열린 <a href="/wiki/%EA%B5%AD%EC%A0%9C_%EC%A0%81%EC%8B%AD%EC%9E%90_%EC%9C%84%EC%9B%90%ED%9A%8C" title="국제 적십자 위원회">적십자회의</a>에도 참가하는 등 외교 분야에서 주로 활동했다. <a href="/wiki/1904%EB%85%84" title="1904년">1904년</a> 육군참장에 오르고, <a href="/wiki/1905%EB%85%84" title="1905년">1905년</a> 훈1등에 서훈되었다.</p>
<p><a href="/wiki/1905%EB%85%84" title="1905년">1905년</a> <a href="/wiki/%EC%9D%84%EC%82%AC%EC%A1%B0%EC%95%BD" title="을사조약">을사조약</a>이 강제로 체결되자 프랑스에서 공사로 일하고 있던 그도 고종의 을사조약이 무효라는 서신을 해외에 전달하는 일을 맡았다. 그러나 귀국해서는 새로운 실세로 떠오른 친일파 <a href="/wiki/%EC%86%A1%EB%B3%91%EC%A4%80" title="송병준">송병준</a>에게 민영환 명의의 토지를 강탈당하는 등의 일을 겪었다.<sup class="reference" id="cite_ref-1"><a href="#cite_note-1">[1]</a></sup> 이후 일제에 협력하였고, 중추원의 참의를 지냈다. 1922년 7월경, 아들 민홍식(閔弘植)이 화류계에 출입하고 토지를 함부로 매각하는 등 가산을 탕진하자, 아들을 고소하기도 하였다.<sup class="reference" id="cite_ref-2"><a href="#cite_note-2">[2]</a></sup></p>
<p><a href="/wiki/%ED%83%9C%ED%8F%89%EC%96%91_%EC%A0%84%EC%9F%81" title="태평양 전쟁">태평양 전쟁</a> 종전 후 <a href="/wiki/%EB%B0%98%EB%AF%BC%EC%A1%B1%ED%96%89%EC%9C%84%ED%8A%B9%EB%B3%84%EC%A1%B0%EC%82%AC%EC%9C%84%EC%9B%90%ED%9A%8C" title="반민족행위특별조사위원회">반민족행위특별조사위원회</a>에서 조사를 받은 적이 있으며, <a href="/wiki/2002%EB%85%84" title="2002년">2002년</a> <a href="/wiki/%EB%AF%BC%EC%A1%B1%EC%A0%95%EA%B8%B0%EB%A5%BC_%EC%84%B8%EC%9A%B0%EB%8A%94_%EA%B5%AD%ED%9A%8C%EC%9D%98%EC%9B%90%EB%AA%A8%EC%9E%84" title="민족정기를 세우는 국회의원모임">민족정기를 세우는 국회의원모임</a>이 발표한 <a href="/wiki/%EC%B9%9C%EC%9D%BC%ED%8C%8C_708%EC%9D%B8_%EB%AA%85%EB%8B%A8" title="친일파 708인 명단">친일파 708인 명단</a>과 <a href="/wiki/2008%EB%85%84" title="2008년">2008년</a> <a href="/wiki/%EB%AF%BC%EC%A1%B1%EB%AC%B8%EC%A0%9C%EC%97%B0%EA%B5%AC%EC%86%8C" title="민족문제연구소">민족문제연구소</a>에서 발표한 <a class="mw-redirect" href="/wiki/%EB%AF%BC%EC%A1%B1%EB%AC%B8%EC%A0%9C%EC%97%B0%EA%B5%AC%EC%86%8C%EC%9D%98_%EC%B9%9C%EC%9D%BC%EC%9D%B8%EB%AA%85%EC%82%AC%EC%A0%84_%EC%88%98%EB%A1%9D%EC%98%88%EC%A0%95%EC%9E%90_%EB%AA%85%EB%8B%A8" title="민족문제연구소의 친일인명사전 수록예정자 명단">친일인명사전 수록예정자 명단</a>, <a href="/wiki/2009%EB%85%84" title="2009년">2009년</a> <a href="/wiki/%EB%8C%80%ED%95%9C%EB%AF%BC%EA%B5%AD_%EC%B9%9C%EC%9D%BC%EB%B0%98%EB%AF%BC%EC%A1%B1%ED%96%89%EC%9C%84%EC%A7%84%EC%83%81%EA%B7%9C%EB%AA%85%EC%9C%84%EC%9B%90%ED%9A%8C" title="대한민국 친일반민족행위진상규명위원회">친일반민족행위진상규명위원회</a>가 발표한 <a href="/wiki/%EC%B9%9C%EC%9D%BC%EB%B0%98%EB%AF%BC%EC%A1%B1%ED%96%89%EC%9C%84_705%EC%9D%B8_%EB%AA%85%EB%8B%A8" title="친일반민족행위 705인 명단">친일반민족행위 705인 명단</a>에 모두 포함되었다.</p>
<p><a href="/wiki/1949%EB%85%84" title="1949년">1949년</a> 반민특위 조사 당시, 민영찬은 "민영환 사후 편모만 남아 있었기에 망명이 어려워 회유에 따라 중추원에 참가했으나 회의에서 발언이나 건의를 한 적은 없고, 그나마 가산을 곧 탕진해 <a href="/wiki/%EC%A4%91%EA%B5%AD" title="중국">중국</a>으로 도피하느라 오래 활동하지 못했다"고 답변했다. 이에 반민특위는 부득이한 상황이라 판단하고 불기소 의견서를 냈다.</p>
<p>부인으로는 한국인 강씨(姜氏)와 중국인 호씨(胡氏)가 있었다.<sup class="reference" id="cite_ref-3"><a href="#cite_note-3">[3]</a></sup> 영국인 화가 <a href="/wiki/%EC%97%98%EB%A6%AC%EC%9E%90%EB%B2%A0%EC%8A%A4_%ED%82%A4%EC%8A%A4" title="엘리자베스 키스">엘리자베스 키스</a>가 이름이 전해지지 않는 민영찬의 딸을 그린 것으로 알려진 그림 〈민씨가의 규수〉(1938)가 남아 있다.<sup class="reference" id="cite_ref-4"><a href="#cite_note-4">[4]</a></sup></p>
