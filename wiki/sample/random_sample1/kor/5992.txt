<p><b>민족 대이동</b>(民族大移動, <span style="font-size: smaller;"><a href="/wiki/%EC%98%81%EC%96%B4" title="영어">영어</a>:</span> <span lang="en" xml:lang="en">Migration Period 또는 Barbarian Invasions</span>, <span style="font-size: smaller;"><a href="/wiki/%EB%8F%85%EC%9D%BC%EC%96%B4" title="독일어">독일어</a>:</span> <span lang="de" xml:lang="de">Völkerwanderung</span>)은 <a href="/wiki/4%EC%84%B8%EA%B8%B0" title="4세기">4세기</a> 경부터 <a href="/wiki/6%EC%84%B8%EA%B8%B0" title="6세기">6세기</a> 경에 걸쳐 <a class="mw-redirect" href="/wiki/%EA%B2%8C%EB%A5%B4%EB%A7%8C_%EB%AF%BC%EC%A1%B1" title="게르만 민족">게르만 민족</a> 및 관련 여러 민족이 <a href="/wiki/%EC%84%9C%EC%9C%A0%EB%9F%BD" title="서유럽">서유럽</a>·<a href="/wiki/%EB%82%A8%EC%9C%A0%EB%9F%BD" title="남유럽">남유럽</a> 방면에 이동한 사실을 말하는데, 넓게는 <a href="/wiki/%EB%85%B8%EB%A5%B4%EB%A7%8C%EC%9D%B8" title="노르만인">노르만인</a>의 이동도 포함시켜 <a href="/wiki/11%EC%84%B8%EA%B8%B0" title="11세기">11세기</a> 경까지를 보는 경우도 있다. 이동의 원인으로서는 일반적으로 인구의 증가, 경지의 부족, 타민족의 압박 등을 들고 있으나, 구체적인 대이동의 계기는 <a href="/wiki/%ED%9B%88%EC%A1%B1" title="훈족">훈족</a>의 서진(西進)이며, 게르만 여러 족속은 보다 좋은 기후와 비옥한 땅을 찾아서 당시 방위력이 약화되었던 <a href="/wiki/%EB%A1%9C%EB%A7%88_%EC%A0%9C%EA%B5%AD" title="로마 제국">로마 제국</a>으로 대거 침입했다.</p>
<p>게르만 민족의 일부(<a class="mw-redirect" href="/wiki/%EC%88%98%EC%97%90%EB%B9%84%EC%A1%B1" title="수에비족">수에비족</a> 등)는 일찍이 <a href="/wiki/%EA%B8%B0%EC%9B%90%EC%A0%84_2%EC%84%B8%EA%B8%B0" title="기원전 2세기">기원전 2세기</a> 말 부터 로마 영내에 침입하였는데, 로마 제국 시대에 들어와서는 로마의 장성(長城, <a class="new" href="/w/index.php?title=%EB%A6%AC%EB%A9%94%EC%8A%A4&amp;action=edit&amp;redlink=1" title="리메스 (없는 문서)">리메스</a>)이나 방위군단에 저지되어 <a href="/wiki/%EB%9D%BC%EC%9D%B8_%EA%B0%95" title="라인 강">라인 강</a>의 선에서 제국과 대치하고 있었다. 그러나 <a class="mw-redirect" href="/wiki/%EA%B3%A0%ED%8A%B8%EC%A1%B1" title="고트족">고트족</a> 등은 <a href="/wiki/2%EC%84%B8%EA%B8%B0" title="2세기">2세기</a> 후반에 <a href="/wiki/%ED%9D%91%ED%95%B4" title="흑해">흑해</a>(黑海) 연안 방면에 진출하고 있었으며, 라인 강 방면에서도 게르만인은 용병이나 <a class="new" href="/w/index.php?title=%EC%BD%9C%EB%A1%9C%EB%88%84%EC%8A%A4&amp;action=edit&amp;redlink=1" title="콜로누스 (없는 문서)">콜로누스</a>(토지를 가진 소작인)로서 영내에 조금씩 침입해 들어갔다.</p>
<p>침입한 게르만 민족은 동게르만·서게르만·북게르만 민족 등으로 크게 나눌 수 있을 것이다. 동게르만 민족에는 <a href="/wiki/%EB%B0%98%EB%8B%AC%EC%A1%B1" title="반달족">반달족</a>·<a class="mw-redirect" href="/wiki/%EB%B6%80%EB%A5%B4%EA%B5%B0%ED%8A%B8%EC%A1%B1" title="부르군트족">부르군트족</a>·<a class="mw-redirect" href="/wiki/%EA%B3%A0%ED%8A%B8%EC%A1%B1" title="고트족">고트족</a> 등이 있으며, 훈족 이동의 영향을 받아서 <a href="/wiki/%EC%9D%B4%ED%83%88%EB%A6%AC%EC%95%84" title="이탈리아">이탈리아</a>, <a href="/wiki/%ED%94%84%EB%9E%91%EC%8A%A4" title="프랑스">프랑스</a>, <a class="mw-redirect" href="/wiki/%EC%97%90%EC%8A%A4%ED%8C%8C%EB%83%90" title="에스파냐">에스파냐</a>, <a href="/wiki/%EC%95%84%ED%94%84%EB%A6%AC%EC%B9%B4" title="아프리카">아프리카</a> 등의 여러 지방으로 이동했는데, 대부분은 그곳 민족과 동화하는 경향이 있었다. 이와는 달리 <a class="mw-redirect" href="/wiki/%EC%95%B5%EA%B8%80%EB%A1%9C_%EC%83%89%EC%8A%A8%EC%A1%B1" title="앵글로 색슨족">앵글로 색슨족</a>과 <a class="mw-redirect" href="/wiki/%EB%A1%AC%EB%B0%94%EB%A5%B4%EB%93%9C%EC%A1%B1" title="롬바르드족">롬바르드족</a>·<a class="mw-redirect" href="/wiki/%ED%94%84%EB%9E%91%ED%81%AC%EC%A1%B1" title="프랑크족">프랑크족</a> 등이 서게르만 민족으로서 <a href="/wiki/%EC%98%81%EA%B5%AD" title="영국">영국</a>, <a href="/wiki/%ED%94%84%EB%9E%91%EC%8A%A4" title="프랑스">프랑스</a>, <a href="/wiki/%EC%9D%B4%ED%83%88%EB%A6%AC%EC%95%84" title="이탈리아">이탈리아</a>의 각지에서 건국했다. 북게르만 민족은 소위 노르만인으로서 <a href="/wiki/10%EC%84%B8%EA%B8%B0" title="10세기">10세기</a> 이후에 남하했다. 유럽 각지에 침입한 게르만인의 수는 로마인 등 원주민의 약 3% 이하의 소수였다. 이 때문에 이주한 곳의 원주민에게 문화적으로 동화한다든지 <a href="/wiki/%EB%A1%9C%EB%A7%88_%EA%B0%80%ED%86%A8%EB%A6%AD%EA%B5%90%ED%9A%8C" title="로마 가톨릭교회">가톨릭 교회</a>와 대립한다든지 하여, 단명(短命)으로 끝나는 부족도 적지 않았다. 그러나 민족 이동의 대세로서는 프랑크족에서 볼 수 있는 것과 같이 <a class="mw-redirect" href="/wiki/%EA%B7%B8%EB%A6%AC%EC%8A%A4_%EC%A0%95%EA%B5%90" title="그리스 정교">그리스 정교</a>를 신봉하는 <a class="mw-redirect" href="/wiki/%EB%8F%99%EB%A1%9C%EB%A7%88%EC%A0%9C%EA%B5%AD" title="동로마제국">동로마제국</a>에 대항하여 가톨릭 교회와 제휴하는 <a href="/wiki/%EC%A4%91%EC%84%B8" title="중세">중세</a> 서유럽 세계의 형성을 촉진시켰다.</p>
