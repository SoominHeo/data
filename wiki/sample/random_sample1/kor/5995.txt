<p><b>민주당</b>(民主黨, <span lang="en" xml:lang="en">Democratic Party</span>)은 <a href="/wiki/1955%EB%85%84" title="1955년">1955년</a> <a href="/wiki/%EC%9E%90%EC%9C%A0%EB%8B%B9_(%EB%8C%80%ED%95%9C%EB%AF%BC%EA%B5%AD)" title="자유당 (대한민국)">자유당</a>의 <a class="mw-redirect" href="/wiki/%EC%82%AC%EC%82%AC%EC%98%A4%EC%9E%85%EA%B0%9C%ED%97%8C" title="사사오입개헌">사사오입개헌</a> 사건을 계기로, <a href="/wiki/%EB%AF%BC%EC%A3%BC%EA%B5%AD%EB%AF%BC%EB%8B%B9_(1949%EB%85%84)" title="민주국민당 (1949년)">민주국민당</a>의 보수파와 자유당의 탈당파, 흥사단등의 반이승만 세력이 모여, <a href="/wiki/1955%EB%85%84" title="1955년">1955년</a> <a href="/wiki/9%EC%9B%94_18%EC%9D%BC" title="9월 18일">9월 18일</a>에 창당 되었다.</p>
<p>창당 이후, 구 한국민주당, 민주국민당 출신 구파와 자유당 탈당파, 흥사단계 등 신파의 내부적 갈등이 지속되었다.</p>
<p><a href="/wiki/1956%EB%85%84" title="1956년">1956년</a> 대선 후보 선출 전 <a href="/wiki/%EA%B3%BD%EC%83%81%ED%9B%88" title="곽상훈">곽상훈</a>의 중개로, 대통령 후보에 <a href="/wiki/%EC%8B%A0%EC%9D%B5%ED%9D%AC" title="신익희">신익희</a>, <a href="/wiki/%EB%B6%80%ED%86%B5%EB%A0%B9" title="부통령">부통령</a> 후보에 <a href="/wiki/%EC%9E%A5%EB%A9%B4" title="장면">장면</a>을 내세워 제3대 정부통령 선거에 출마하였으나, <a href="/wiki/%EC%8B%A0%EC%9D%B5%ED%9D%AC" title="신익희">신익희</a>는 선거 직전 사망했고, <a href="/wiki/%EC%9E%A5%EB%A9%B4" title="장면">장면</a>은 <a href="/wiki/%EB%B6%80%ED%86%B5%EB%A0%B9" title="부통령">부통령</a>에 당선되었다.</p>
<p><a href="/wiki/1960%EB%85%84" title="1960년">1960년</a> <a class="mw-redirect" href="/wiki/4.19_%ED%98%81%EB%AA%85" title="4.19 혁명">4.19 혁명</a>에 의해 이승만 정권의 퇴진으로 직후에 실시한 <a href="/wiki/1960%EB%85%84" title="1960년">1960년</a> <a href="/wiki/7%EC%9B%94_29%EC%9D%BC" title="7월 29일">7월 29일</a>의 총선에 크게 승리하여, <a href="/wiki/%EC%9C%A4%EB%B3%B4%EC%84%A0" title="윤보선">윤보선</a> 대통령과 <a href="/wiki/%EC%9E%A5%EB%A9%B4" title="장면">장면</a> 국무총리를 선출하여 <a href="/wiki/%EB%8C%80%ED%95%9C%EB%AF%BC%EA%B5%AD_%EC%A0%9C2%EA%B3%B5%ED%99%94%EA%B5%AD" title="대한민국 제2공화국">제2공화국</a>의 여당이 되었다. 그러나, 구파와 신파간의 분당, 당내 분열 등으로 혼란을 겪다가, <a href="/wiki/1960%EB%85%84" title="1960년">1960년</a> <a href="/wiki/11%EC%9B%94" title="11월">11월</a>에 민주당을 탈당한 구파세력이 신민당을 창당하는 사태가 일어났다. <a href="/wiki/1961%EB%85%84" title="1961년">1961년</a>에 발생한 <a class="mw-redirect" href="/wiki/5%C2%B716_%EA%B5%B0%EC%82%AC%EC%A0%95%EB%B3%80" title="5·16 군사정변">5·16 군사정변</a> 직후에 군부에 의해서 해산되었다.</p>

<h3><span class="mw-headline" id=".EB.AF.BC.EC.A3.BC.EB.8B.B9_.EB.B0.9C.EB.8B.B9.EB.8C.80.ED.9A.8C">민주당 발당대회</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%AF%BC%EC%A3%BC%EB%8B%B9_(%EB%8C%80%ED%95%9C%EB%AF%BC%EA%B5%AD,_1955%EB%85%84)&amp;action=edit&amp;section=9" title="부분 편집: 민주당 발당대회">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p><a href="/wiki/1955%EB%85%84" title="1955년">1955년</a> <a href="/wiki/9%EC%9B%94_19%EC%9D%BC" title="9월 19일">9월 19일</a> 민주당 발당대회는 <a href="/wiki/%EA%B3%BD%EC%83%81%ED%9B%88" title="곽상훈">곽상훈</a> <a href="/wiki/%EB%AF%BC%EC%9D%98%EC%9B%90" title="민의원">민의원</a> 의장을 임시의장으로 선출하고, 정강정책을 채택한 뒤 지도부 선출을 중앙위에 일임했다.</p>
