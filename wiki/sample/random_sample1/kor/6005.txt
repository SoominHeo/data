<p><b>밀워키 브루어스</b>(<span style="font-size: smaller;"><a href="/wiki/%EC%98%81%EC%96%B4" title="영어">영어</a>:</span> <span lang="en" xml:lang="en">Milwaukee Brewers</span>)는 <a href="/wiki/%EB%AF%B8%EA%B5%AD" title="미국">미국</a> <a href="/wiki/%EC%9C%84%EC%8A%A4%EC%BD%98%EC%8B%A0_%EC%A3%BC" title="위스콘신 주">위스콘신 주</a> <a href="/wiki/%EB%B0%80%EC%9B%8C%ED%82%A4" title="밀워키">밀워키</a>를 연고지로 하는 <a href="/wiki/%ED%94%84%EB%A1%9C_%EC%95%BC%EA%B5%AC" title="프로 야구">프로 야구 팀</a>이다. <a href="/wiki/%EB%A9%94%EC%9D%B4%EC%A0%80_%EB%A6%AC%EA%B7%B8_%EB%B2%A0%EC%9D%B4%EC%8A%A4%EB%B3%BC" title="메이저 리그 베이스볼">메이저 리그</a> <a class="mw-redirect" href="/wiki/%EB%82%B4%EC%85%94%EB%84%90_%EB%A6%AC%EA%B7%B8_%EC%A4%91%EB%B6%80_%EC%A7%80%EA%B5%AC" title="내셔널 리그 중부 지구">내셔널 리그 중부 지구</a> 소속이다.</p>
<p>밀워키가 미국 제 1의 <a href="/wiki/%EB%A7%A5%EC%A3%BC" title="맥주">맥주</a> 양조 도시인 이유로 브루어스라는 이름이 붙었다.</p>
<p><a href="/wiki/1969%EB%85%84" title="1969년">1969년</a>에 '시애틀 파일러츠'라는 이름으로 창단하여 1년 만에 밀워키로 이전했는데 <a href="/wiki/1997%EB%85%84" title="1997년">1997년</a>까지 <a href="/wiki/%EC%95%84%EB%A9%94%EB%A6%AC%EC%B9%B8_%EB%A6%AC%EA%B7%B8" title="아메리칸 리그">아메리칸 리그</a> 소속이었으나 <a href="/wiki/1998%EB%85%84" title="1998년">1998년</a> 신생 팀 창단에 따른 리그 지역 조정에 따라 <a href="/wiki/%EB%82%B4%EC%85%94%EB%84%90_%EB%A6%AC%EA%B7%B8_(%EC%95%BC%EA%B5%AC)" title="내셔널 리그 (야구)">내셔널 리그</a> <a href="/wiki/%EB%82%B4%EC%85%94%EB%84%90_%EB%A6%AC%EA%B7%B8_%EC%A4%91%EB%B6%80" title="내셔널 리그 중부">중부 지구</a>로 편입하였다. <a href="/wiki/KBO_%EB%A6%AC%EA%B7%B8" title="KBO 리그">KBO 리그</a> <a href="/wiki/%EB%91%90%EC%82%B0_%EB%B2%A0%EC%96%B4%EC%8A%A4" title="두산 베어스">OB 베어스</a>에서 뛰었던 <a href="/wiki/%ED%88%AC%EC%88%98" title="투수">투수</a> <a href="/wiki/%EB%B0%95%EC%B2%A0%EC%88%9C" title="박철순">박철순</a> 선수가 <a href="/wiki/KBO_%EB%A6%AC%EA%B7%B8" title="KBO 리그">KBO 리그</a> 출범 전에 이 구단 산하의 <a href="/wiki/%EB%A7%88%EC%9D%B4%EB%84%88_%EB%A6%AC%EA%B7%B8_%EB%B2%A0%EC%9D%B4%EC%8A%A4%EB%B3%BC" title="마이너 리그 베이스볼">마이너 리그</a> 팀에 입단하여 진출했으나, <a href="/wiki/%EB%A9%94%EC%9D%B4%EC%A0%80_%EB%A6%AC%EA%B7%B8_%EB%B2%A0%EC%9D%B4%EC%8A%A4%EB%B3%BC" title="메이저 리그 베이스볼">메이저 리그</a>에서 뛰지는 못하고 귀국하여 원년 멤버가 되었다.</p>
<p><a href="/wiki/2008%EB%85%84" title="2008년">2008년</a> 와일드 카드로 플레이오프에 진출했으나 <a href="/wiki/%ED%95%84%EB%9D%BC%EB%8D%B8%ED%94%BC%EC%95%84_%ED%95%84%EB%A6%AC%EC%8A%A4" title="필라델피아 필리스">필라델피아 필리스</a>에게 1승 3패를 당하여 월드시리즈 진출에 실패했다.</p>
