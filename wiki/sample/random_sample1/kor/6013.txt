<h1 class="firstHeading" id="firstHeading" lang="ko">바 (술집)</h1>
<div class="mw-body-content" id="bodyContent">

<a href="/wiki/%EC%8A%A4%ED%8E%98%EC%9D%B8" title="스페인">스페인</a> <a href="/wiki/%EC%82%B4%EB%9D%BC%EB%A7%9D%EC%B9%B4" title="살라망카">살라망카</a>의 장사가 끝난 바의 모습.</div>

<p><b>바</b>(bar)에서는 가게 안에서만 마실 수 있는 음료수(특히 알코올 음료)를 제공한다. 바에는 고객을 위한 길쭉한 카운터와 테이블과 함께 의자가 있다. 어떤 바들은 무대에서 라이브 밴드, 코메디, 고고댄서, 쇼, 스트립쇼 등의 엔터테인먼트를 제공한다. 호텔에 속하는 바들은 가끔 호텔 라운지나 롱바라고도 불린다.</p>
<p>"바"라는 단어는 음료가 서빙되는 카운터에서 나왔고, 그 주변의 모든 것을 포괄하는 비유적 의미를 지닌다. 바 뒷편의 장소엔 병으로 셋팅된 선반(gantry)가 존재한다. 몇몇 바에서 갠트리는 나무장식이나 에칭된 글라스, 거울, 조명등으로 장식되기도 한다. 음식 또한 바에서 주문하고 식사할 수 있도록 되어있다.</p>
<h2><span class="mw-headline" id=".EC.8A.A4.ED.8E.98.EC.9D.B8.EC.9D.98_.EB.B0.94">스페인의 바</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%B0%94_(%EC%88%A0%EC%A7%91)&amp;action=edit&amp;section=1" title="부분 편집: 스페인의 바">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p><a href="/wiki/%EC%8A%A4%ED%8E%98%EC%9D%B8" title="스페인">스페인</a>에서 바는 아주 일반적이고 중요한 일부분이다. <a href="/wiki/%EC%8A%A4%ED%8E%98%EC%9D%B8" title="스페인">스페인</a>에서는, 많은 바들이 심지어 같은 거리에 줄지어 서있기도 한다. 거의 모든 바들은 여름과 같이 날씨가 좋은날, 바 앞의 거리나 플라자 등의 외부에 테이블과 의자를 놓고 장사하기도 한다. <a href="/wiki/%EC%8A%A4%ED%8E%98%EC%9D%B8" title="스페인">스페인</a>의 바에서는 여러 종류의 <a href="/wiki/%EC%83%8C%EB%93%9C%EC%9C%84%EC%B9%98" title="샌드위치">샌드위치</a>와 스낵들(<a href="/wiki/%ED%83%80%ED%8C%8C%EC%8A%A4" title="타파스">타파스</a>나 <a class="new" href="/w/index.php?title=%ED%95%80%EC%B5%B8%EC%8A%A4&amp;action=edit&amp;redlink=1" title="핀쵸스 (없는 문서)">핀쵸스</a>)이 존재한다. 최근 법개정으로, 몇몇 바에서는 금연을 실시하나 아직은 수가 적은 편이다. 큰 바에서는 언제나 <a class="new" href="/w/index.php?title=%ED%9D%A1%EC%97%B0%EA%B5%AC%EC%97%AD&amp;action=edit&amp;redlink=1" title="흡연구역 (없는 문서)">흡연구역</a>이 있는 편이다.</p>
<p><a href="/wiki/%EC%8A%A4%ED%8E%98%EC%9D%B8" title="스페인">스페인</a>은 1000명 당 6개의 바로 인구수에 비해 바의 수가 많은 편이다. 이 숫자는 <a href="/wiki/%EC%98%81%EA%B5%AD" title="영국">영국</a>의 3배, <a href="/wiki/%EB%8F%85%EC%9D%BC" title="독일">독일</a>의 4배에 해당하며, 유럽 15개국의 바의 숫자를 합한 것의 2배에 해당하는 바가 존재한다. <a href="/wiki/%EC%8A%A4%ED%8E%98%EC%9D%B8" title="스페인">스페인</a>에서 바는 특별히 돈을 써야하는 장소가 아닌, 그저 자연스러운 만남의 장소의 의미가 강하다. 이곳에서는 쓰고난 휴지는 바닥에 버리는게 일반화 되어있고, 먹고 난 후 지불하는게 일반적이다.</p>
<p>스페인에서는 바에서 음료를 시키면 일반적으로 음식(<a href="/wiki/%ED%83%80%ED%8C%8C%EC%8A%A4" title="타파스">타파스</a>나 <a class="mw-redirect" href="/wiki/%ED%95%80%EC%B4%88%EC%8A%A4" title="핀초스">핀초스</a>) 같은 게 무료이다. 그래서 사람들은 일반적으로 <a href="/wiki/%EC%95%84%EC%B9%A8" title="아침">아침</a>과 <a href="/wiki/%EC%A0%90%EC%8B%AC" title="점심">점심</a> 사이, <a href="/wiki/%EC%A0%90%EC%8B%AC" title="점심">점심</a>과 <a href="/wiki/%EC%A0%80%EB%85%81" title="저녁">저녁</a>사이에 바에가서 <a href="/wiki/%EC%9D%8C%EB%A3%8C" title="음료">음료</a>와 간단한 <a href="/wiki/%EC%9D%8C%EC%8B%9D" title="음식">음식</a>을 같이 시킨다. 그리고 몇몇 바들은 <a class="mw-redirect" href="/wiki/%EB%A0%88%EC%8A%A4%ED%86%A0%EB%9E%91" title="레스토랑">레스토랑</a>과 같이 있어 식사도 가능하다.</p>
