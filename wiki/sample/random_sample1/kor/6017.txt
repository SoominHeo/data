<p><b>바그라티온 작전</b>(<span style="font-size: smaller;"><a href="/wiki/%EB%9F%AC%EC%8B%9C%EC%95%84%EC%96%B4" title="러시아어">러시아어</a>:</span> <span lang="ru" xml:lang="ru">Oперация Багратион, Operatsiya Bagration</span>)은 <a href="/wiki/1944%EB%85%84" title="1944년">1944년</a> <a class="new" href="/w/index.php?title=%EB%8F%99%EB%B6%80_%ED%8F%B4%EB%9E%80%EB%93%9C&amp;action=edit&amp;redlink=1" title="동부 폴란드 (없는 문서)">동부 폴란드</a>와 <a href="/wiki/%EB%B2%A8%EB%9D%BC%EB%A3%A8%EC%8A%A4" title="벨라루스">벨라루스</a>에 포진해 있던 독일군을 붕괴시키기 위한 소련군의 하계 대공세의 암호명이다.</p>
<p>이 작전이 성공해 <a href="/wiki/%EB%B0%94%EB%A5%B4%EB%B0%94%EB%A1%9C%EC%82%AC_%EC%9E%91%EC%A0%84" title="바르바로사 작전">바르바로사 작전</a>이래 <a href="/wiki/%EB%B2%A8%EB%9D%BC%EB%A3%A8%EC%8A%A4" title="벨라루스">벨라루스</a>를 3년간 점령하고 있던 독일 중앙집단군은 붕괴되었다.</p>
<p>이 작전은 18~19세기의 <a href="/wiki/%EC%A1%B0%EC%A7%80%EC%95%84_(%EA%B5%AD%EA%B0%80)" title="조지아 (국가)">조지아</a> 태생의 <a href="/wiki/%EB%9F%AC%EC%8B%9C%EC%95%84_%EC%A0%9C%EA%B5%AD" title="러시아 제국">러시아 제국</a> 장군이자 러시아를 침략한 나폴레옹군과 싸우다가 <a href="/wiki/%EB%B3%B4%EB%A1%9C%EB%94%94%EB%85%B8_%EC%A0%84%ED%88%AC" title="보로디노 전투">보로디노 전투</a>에서 치명상을 입고 전사한 <a href="/wiki/%ED%91%9C%ED%8A%B8%EB%A5%B4_%EB%B0%94%EA%B7%B8%EB%9D%BC%ED%8B%B0%EC%98%A8" title="표트르 바그라티온">바그라티온</a>의 이름을 따 붙여진 것이다. 작전 개시일은 본래 6월 15일에서 6월 20일었으나 <a href="/wiki/6%EC%9B%94_22%EC%9D%BC" title="6월 22일">6월 22일</a>로 바뀌었다. 이 날은 3년 전 독일의 <a href="/wiki/%EB%B0%94%EB%A5%B4%EB%B0%94%EB%A1%9C%EC%82%AC_%EC%9E%91%EC%A0%84" title="바르바로사 작전">바르바로사 작전</a> 개시일과 같은 날이다.</p>

<h2><span class="mw-headline" id=".EC.9E.91.EC.A0.84_.EC.A0.84.EC.9D.98_.EC.83.81.ED.99.A9">작전 전의 상황</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%B0%94%EA%B7%B8%EB%9D%BC%ED%8B%B0%EC%98%A8_%EC%9E%91%EC%A0%84&amp;action=edit&amp;section=1" title="부분 편집: 작전 전의 상황">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>1941년 <a class="mw-redirect" href="/wiki/%EB%8F%85%EC%86%8C%EC%A0%84%EC%9F%81" title="독소전쟁">독소전쟁</a> 개시이래 스탈린은 계속 <a href="/wiki/%EC%98%81%EA%B5%AD" title="영국">영국</a>과 <a href="/wiki/%EB%AF%B8%EA%B5%AD" title="미국">미국</a>에 유럽의 <a class="new" href="/w/index.php?title=%EC%A0%9C2%EC%A0%84%EC%84%A0&amp;action=edit&amp;redlink=1" title="제2전선 (없는 문서)">제2전선</a>을 개시해달라고 요구하였다. 연합국은 1943년 7월 <a class="mw-redirect" href="/wiki/%ED%97%88%EC%8A%A4%ED%82%A4_%EC%9E%91%EC%A0%84" title="허스키 작전">시칠리아 섬에 상륙</a>했고, 9월에는 <a href="/wiki/%EC%9D%B4%ED%83%88%EB%A6%AC%EC%95%84" title="이탈리아">이탈리아</a> 본토에 상륙했지만 이는 한정된 전선이었기 때문에 스탈린은 <a href="/wiki/%ED%94%84%EB%9E%91%EC%8A%A4" title="프랑스">프랑스</a>에 상륙해 달라고 요구했다. 1943년 11월 개최된 <a href="/wiki/%ED%85%8C%ED%97%A4%EB%9E%80_%ED%9A%8C%EB%8B%B4" title="테헤란 회담">테헤란 회담</a>에서는 영·미군이 북프랑스에 상륙할 것을 공식적으로 결정했다. 그리하여 <a href="/wiki/1944%EB%85%84" title="1944년">1944년</a> <a href="/wiki/6%EC%9B%94_6%EC%9D%BC" title="6월 6일">6월 6일</a> <a class="mw-redirect" href="/wiki/%EB%85%B8%EB%A5%B4%EB%A7%9D%EB%94%94_%EC%83%81%EB%A5%99%EC%9E%91%EC%A0%84" title="노르망디 상륙작전">노르망디 상륙작전</a>에 의해 스탈린이 요구한 제2전선은 만들어졌다.</p>
<p>영미군이 북프랑스에 상륙작전을 계획하고 있다는 것을 인지한 독일군은 주 전장이었던 <a href="/wiki/%EB%8F%99%EB%B6%80_%EC%A0%84%EC%84%A0_(%EC%A0%9C2%EC%B0%A8_%EC%84%B8%EA%B3%84_%EB%8C%80%EC%A0%84)" title="동부 전선 (제2차 세계 대전)">동부 전선</a>에서 <a href="/wiki/%EC%84%9C%EB%B6%80_%EC%A0%84%EC%84%A0_(%EC%A0%9C2%EC%B0%A8_%EC%84%B8%EA%B3%84_%EB%8C%80%EC%A0%84)" title="서부 전선 (제2차 세계 대전)">서부 전선</a>으로 부대를 이동하기 시작했고, 가능한 모든 예비병력을 동원해 서부전선으로 이동하였다. 한편 제공권을 장악한 영미군의 전략폭격이 늘어났기 때문에 동부전선에 전개한 독일 공군도 서부 전선으로 이동하여 동부전선의 하늘의 균형추도 소련군 측으로 기울게 되었다.</p>
<h3><span class="mw-headline" id=".EB.82.A8.EB.B0.A9.EC.A7.91.EB.8B.A8.EA.B5.B0.EC.9D.98_.EA.B4.B4.EB.A9.B8.EA.B3.BC_.EB.B6.81.EB.B0.A9.EC.A7.91.EB.8B.A8.EA.B5.B0.EC.9D.98_.ED.9B.84.ED.87.B4">남방집단군의 괴멸과 북방집단군의 후퇴</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%B0%94%EA%B7%B8%EB%9D%BC%ED%8B%B0%EC%98%A8_%EC%9E%91%EC%A0%84&amp;action=edit&amp;section=2" title="부분 편집: 남방집단군의 괴멸과 북방집단군의 후퇴">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>한편 1943년 8월 <a href="/wiki/%EC%BF%A0%EB%A5%B4%EC%8A%A4%ED%81%AC_%EC%A0%84%ED%88%AC" title="쿠르스크 전투">쿠르스크 전투</a> 이후, 전쟁의 주도권은 소련군에 넘어갔으며 이 방면에서 <a href="/wiki/1943%EB%85%84" title="1943년">1943년</a> 말부터 <a href="/wiki/1944%EB%85%84" title="1944년">1944년</a> 5월까지 소련군은 겨울과 봄에 걸쳐 계속 파상공세를 가하였다. 또한 우크라이나에서는 <a href="/wiki/1944%EB%85%84" title="1944년">1944년</a> <a href="/wiki/1%EC%9B%94_28%EC%9D%BC" title="1월 28일">1월 28일</a>에 <a class="mw-redirect" href="/wiki/%EB%B0%94%ED%88%AC%ED%8B%B4" title="바투틴">바투틴</a> 대장의 제1우크라이나 전선군 과 <a href="/wiki/%EC%9D%B4%EB%B0%98_%EC%BD%94%EB%84%A4%ED%94%84" title="이반 코네프">이반 코네프</a> 원수의 제2우크라이나 전선군이 독일 남방집단군(A,B집단군)에 대한 공세를 시작하였다. 이 공세에서 독일 제11군단과 42군단의 6만 명이 코르순에서 포위되어 섬멸되었다. 또한 3월에는 독일 제1기갑군이 포위되었으나 간신히 탈출하였다. 히틀러는 이 공세에서도 후퇴를 금지했으며, 후퇴를 요구하는 B 집단군의 <a class="mw-redirect" href="/wiki/%EB%A7%8C%EC%8A%88%ED%83%80%EC%9D%B8" title="만슈타인">만슈타인</a>은 해임되었다. 4월에 <a href="/wiki/%EB%B0%9C%ED%84%B0_%EB%AA%A8%EB%8D%B8" title="발터 모델">발터 모델</a>이 부임하여 나머지 패잔병력을 북 우크라이나 집단군으로 재편성하여 이끌었다.</p>
<p>한편 <a class="mw-redirect" href="/wiki/%EC%97%90%EB%B0%9C%ED%8A%B8_%ED%8F%B0_%ED%81%B4%EB%9D%BC%EC%9D%B4%EC%8A%A4%ED%8A%B8" title="에발트 폰 클라이스트">클라이스트</a> 원수가 지휘하는 A집단군은 <a href="/wiki/%EB%A1%9C%EB%94%94%EC%98%A8_%EB%A7%90%EB%A6%AC%EB%86%89%EC%8A%A4%ED%82%A4" title="로디온 말리놉스키">말리노프스키</a> 대장이 지휘하는 제3 우크라이나 전선군에게 밀려서 <a href="/wiki/%EB%B6%80%ED%81%AC_%EA%B0%95" title="부크 강">부크 강</a>을 건너 <a href="/wiki/%EB%AA%B0%EB%8F%84%EB%B0%94" title="몰도바">몰도바</a>까지 후퇴하였다. B집단군과 마찬가지로 이 후퇴는 히틀러의 격노를 사서 클라이스트는 해임되고 잔존병력은 남 우크라이나 집단군으로 개편되어 <a href="/wiki/%ED%8E%98%EB%A5%B4%EB%94%94%EB%82%9C%ED%8A%B8_%EC%87%A0%EB%A5%B4%EB%84%88" title="페르디난트 쇠르너">페르디난트 쇠르너</a> 상급대장이 맡았다. 이 부대는 <a class="mw-redirect" href="/wiki/%EB%93%9C%EB%8B%88%EC%8A%A4%ED%85%8C%EB%A5%B4_%EA%B0%95" title="드니스테르 강">드니스테르 강</a>을 끼고 추축국에 남은 유일한 유전지대인 <a href="/wiki/%EB%A3%A8%EB%A7%88%EB%8B%88%EC%95%84" title="루마니아">루마니아</a>를 방위했다. 한편 소련군에게 <a class="mw-redirect" href="/wiki/%ED%81%AC%EB%A6%AC%EB%AF%B8%EC%95%84_%EB%B0%98%EB%8F%84" title="크리미아 반도">크리미아 반도</a>에 고립된 제17군과 루마니아군 대부분은 해상탈출에 실패하여 7만5천 명의 병력을 잃고 <a href="/wiki/%EC%84%B8%EB%B0%94%EC%8A%A4%ED%86%A0%ED%8F%B4" title="세바스토폴">세바스토폴</a>에서 소련군에게 항복하였다.</p>
<p>이로써 독일 남방집단군 휘하 A집단군, B집단군은 모두 분쇄되어 소련 영토에서 물러났다.</p>
<p>한편 북방에서는 1월 14일에 900여 일 동안 포위되었던 <a href="/wiki/%EB%A0%88%EB%8B%8C%EA%B7%B8%EB%9D%BC%EB%93%9C_%ED%8F%AC%EC%9C%84%EC%A0%84" title="레닌그라드 포위전">레닌그라드</a>의 정면과 <a class="mw-redirect" href="/wiki/%EB%85%B8%EB%B8%8C%EA%B3%A0%EB%A1%9C%EB%93%9C" title="노브고로드">노브고로드</a> 부근에서 소련군이 양방향 공세를 시작하여 <a class="new" href="/w/index.php?title=%ED%80%B4%ED%9D%98%EB%9F%AC&amp;action=edit&amp;redlink=1" title="퀴흘러 (없는 문서)">퀴흘러</a> 원수가 지휘하는 북방집단군은 3월에 <a href="/wiki/%EB%82%98%EB%A5%B4%EB%B0%94" title="나르바">나르바</a>와 <a href="/wiki/%ED%94%84%EC%8A%A4%EC%BD%94%ED%94%84" title="프스코프">프스코프</a>를 잇는 선까지 후퇴하였다. 이리하여 <a class="mw-redirect" href="/wiki/%EB%A0%88%EB%8B%8C%EA%B7%B8%EB%9D%BC%EB%93%9C" title="레닌그라드">레닌그라드</a>의 포위는 900일 만에 뚫렸고, 소련군은 핀란드 방면으로 공세를 개시하여 <a class="mw-redirect" href="/wiki/%EC%B9%B4%EB%A0%90%EB%A6%AC%EC%95%84_%EC%A7%80%ED%98%91" title="카렐리아 지협">카렐리아 지협</a>과 <a href="/wiki/%EB%B9%84%EB%B3%B4%EB%A5%B4%ED%81%AC" title="비보르크">비보르크</a>까지 진출하였다.</p>
<h3><span class="mw-headline" id=".EB.B2.A8.EB.9D.BC.EB.A3.A8.EC.8A.A4.EC.9D.98_.EA.B5.B0.EC.82.AC.EC.A0.81_.EC.83.81.ED.99.A9">벨라루스의 군사적 상황</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%B0%94%EA%B7%B8%EB%9D%BC%ED%8B%B0%EC%98%A8_%EC%9E%91%EC%A0%84&amp;action=edit&amp;section=3" title="부분 편집: 벨라루스의 군사적 상황">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>1944년 5월말까지 수복하지 못한 소련영토는 <a href="/wiki/%EB%B2%A8%EB%9D%BC%EB%A3%A8%EC%8A%A4" title="벨라루스">벨라루스</a>와 <a href="/wiki/%EB%B0%9C%ED%8A%B8_3%EA%B5%AD" title="발트 3국">발트 3국</a>뿐이었다. 벨라루스에는 동부전선 독일군의 주력인 중앙집단군이 <a href="/wiki/%EB%AA%A8%EC%8A%A4%ED%81%AC%EB%B0%94_%EA%B3%B5%EB%B0%A9%EC%A0%84" title="모스크바 공방전">모스크바 공방전</a>에서 후퇴한 이래 같은 곳에서 3년째 버티고 있었다. 중앙집단군은 기갑부대를 많이 보유하고 있었으며 남방, 북방집단군에 비해 전력이 더 강했다.</p>
<p>남방집단군이 붕괴하고 북방집단군이 후퇴했기 때문에, 중앙집단군은 돌출부를 이루게 되었다.</p>
<p>한편 소련군은, 1943년 9월에 모스크바로 향하는 교통의 요지인 <a href="/wiki/%EC%8A%A4%EB%AA%B0%EB%A0%8C%EC%8A%A4%ED%81%AC" title="스몰렌스크">스몰렌스크</a>를 탈환했고, <a href="/wiki/%EB%B0%9C%ED%8A%B8_%ED%95%B4" title="발트 해">발트 해</a>로 흘러드는 <a class="mw-redirect" href="/wiki/%EC%9E%90%ED%8C%8C%EB%93%9C%EB%82%98%EC%95%BC%EB%93%9C%EB%B9%84%EB%82%98_%EA%B0%95" title="자파드나야드비나 강">자파드나야드비나 강</a>에 면한 <a href="/wiki/%EB%B9%84%EC%B3%85%EC%8A%A4%ED%81%AC" title="비쳅스크">비쳅스크</a>와 흑해에 이르는 <a href="/wiki/%EB%93%9C%EB%84%A4%ED%94%84%EB%A5%B4_%EA%B0%95" title="드네프르 강">드네프르 강</a> 유역의 <a href="/wiki/%EC%98%A4%EB%A5%B4%EC%83%A4" title="오르샤">오르샤</a> 사이에 전개하고 있었다. 비쳅스크는 모스크바로 이르는 철도와 간선도로가 있는 교통의 요지였고, 이는 아직 독일군이 점령중이었다. 그래서 중앙집단군을 분쇄하고 독일의 수도 <a href="/wiki/%EB%B2%A0%EB%A5%BC%EB%A6%B0" title="베를린">베를린</a>으로 향하는 최단 경로를 확보하기 위해서는 벨라루스를 탈환하는 것이 전략적으로 매우 중요했다.</p>
<p>그러나 벨라루스는 습지와 숲이 많은 지형이었기 때문에 대규모의 병력 전개가 곤란했다. 특히 기갑부대가 기동할 수 있는 공간은 한정되어 있어서, 독일군은 이런 지역에 강력한 수비진지를 구축해 놓고 있었다. 그러나 소련군 지휘관들은 이전에 <a href="/wiki/%EB%9F%AC%EC%8B%9C%EC%95%84_%EC%A0%9C%EA%B5%AD" title="러시아 제국">러시아 제국</a>의 명장 <a class="mw-redirect" href="/wiki/%EC%88%98%EB%B3%B4%EB%A1%9C%ED%94%84" title="수보로프">수보로프</a>가 말한 "사슴이 갈 수 있는 곳은 러시아 병사도 간다"는 말을 떠올려 작전을 세우기 시작했다.</p>
<h3><span class="mw-headline" id=".EB.8F.85.EC.9D.BC.EA.B5.B0.EC.9D.98_.EB.8C.80.EB.B9.84">독일군의 대비</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%B0%94%EA%B7%B8%EB%9D%BC%ED%8B%B0%EC%98%A8_%EC%9E%91%EC%A0%84&amp;action=edit&amp;section=4" title="부분 편집: 독일군의 대비">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>독일군도 소련군의 여름 공세를 예상하고 있었다. 히틀러는 소련군이 남부의 <a href="/wiki/%EA%B0%88%EB%A6%AC%EC%B9%98%EC%95%84" title="갈리치아">갈리치아</a> 방면에서 출발하여 <a href="/wiki/%EB%B0%9C%ED%8A%B8_%ED%95%B4" title="발트 해">발트 해</a>로 좌회전을 하며 독일군의 돌출부를 잘라내는 대공세를 가해 독일 북방, 중앙집단군의 퇴로를 막아 독일 본토와 연결을 차단할 작전을 펼칠 가능성을 염려했다.</p>
<p>소련군은 거짓된 무선을 흘리고, 독일 정찰기를 속이는 부대이동을 하는 등 기만작전을 펼쳐 이런 판단을 확신하게 했고, 이 결과 독일군의 중앙집단군에 속한 제4기갑군등 주력부대가 북우크라이나 집단군쪽으로 이동하였다. 그래서 민스크-모스크바에 이르는 전선의 전력이 눈에 띠게 약화되었다.</p>
<p>한편 소련 공군은 독일 공군이 서부와 본국으로 이동한 틈을 타서 제공권을 확실히 장악하였으며, 독일군의 보급로를 계속 폭격하였다. 또한 파르티잔도 활발히 활동하여 독일군의 통신선을 절단하고 도로와 철로를 파괴하였다. 이 결과 중앙집단군의 보급은 매우 곤란하게 되었으며, 많은 병력이 <a href="/wiki/%ED%8C%8C%EB%A5%B4%ED%8B%B0%EC%9E%94" title="파르티잔">파르티잔</a>을 막기 위해 후방으로 배치되어 전방의 수비력이 약화되었다.</p>
<h2><span class="mw-headline" id=".EC.9E.91.EC.A0.84_.EC.A0.9C1.EB.8B.A8.EA.B3.84">작전 제1단계</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%B0%94%EA%B7%B8%EB%9D%BC%ED%8B%B0%EC%98%A8_%EC%9E%91%EC%A0%84&amp;action=edit&amp;section=5" title="부분 편집: 작전 제1단계">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>본래 작전 개시일은 6월 15일에서 20일이었으나 보급준비와 기밀유지 문제로 인해 계속 지연되었다. 그래서 스탈린은 아예 <a href="/wiki/1941%EB%85%84" title="1941년">1941년</a> <a href="/wiki/6%EC%9B%94_22%EC%9D%BC" title="6월 22일">6월 22일</a>에 벌어졌던 <a href="/wiki/%EB%B0%94%EB%A5%B4%EB%B0%94%EB%A1%9C%EC%82%AC_%EC%9E%91%EC%A0%84" title="바르바로사 작전">바르바로사 작전</a>을 보복한다는 의미에서 작전 개시일을 6월 22일로 정하였다. 남쪽에서 북쪽 순으로 제1벨라루스, 제2벨라루스, 제3벨라루스, 제1발트 등 4개 전선군이 배치되어 1000km에 걸친 전선에서 총 189개 사단, 200만명의 대병력을 동원하였다. 또한 공군력에 있어서도 소련군은 독일군을 6배로 압도하였다.</p>
<p>6월 22일 새벽 맹렬한 포격을 개시한 후, 소련군 병력은 세 방향에서 독일 중앙집단군을 일제히 공격하기 시작했다. 이는 독일군이 예측한 남쪽 측면의 공세가 아니라, 정면을 강타하는 공격이었다. 공군과 포병의 엄호를 받으면서 소련군 전차와 보병은 독일군 전선을 돌파하여 깊숙히 후방으로 침투했고, 수적으로 열세인 독일군은 각지에서 분단, 포위되었다.</p>
<p>이 작전에서 소련군의 전체적인 지도는 최고지휘관 대리인 <a class="mw-redirect" href="/wiki/%EC%A3%BC%EC%BD%94%ED%94%84" title="주코프">주코프</a> 원수와 총참모장 <a class="mw-redirect" href="/wiki/%EC%95%8C%EB%A0%89%EC%82%B0%EB%93%9C%EB%A5%B4_%EB%B0%94%EC%8B%A4%EB%A0%88%ED%94%84%EC%8A%A4%ED%82%A4" title="알렉산드르 바실레프스키">바실레프스키</a> 원수가 맡았다. 소련군의 지휘관들도 그동안 많은 실전경험을 쌓아왔고, 유사시에 재량하의 임기응변도 허가받을 정도로 실력을 키워 전차, 공군, 포병, 공병의 유기적인 협동공세를 펴는데 전혀 어려움이 없었다. 또한 독일군의 신형 전차인 <a class="mw-redirect" href="/wiki/%ED%8C%90%ED%84%B0" title="판터">판터</a>와 <a class="mw-redirect" href="/wiki/%ED%8B%B0%EA%B1%B0" title="티거">티거</a>에 맞서 <a class="new" href="/w/index.php?title=T-34-85&amp;action=edit&amp;redlink=1" title="T-34-85 (없는 문서)">T-34-85</a>와 <a class="mw-redirect" href="/wiki/IS-2" title="IS-2">IS-2</a> 전차를 대거 투입하였다.</p>
<p>독일 중앙집단군 사령부는 이런 상황에서도 당초의 예측을 고수하였다. 소련군의 공세가 단순한 양동작전이라고 판단하여 당초에 설정된 거점을 지키여 소련군의 공세에 맞서는 방침을 내렸다. 이런 방침때문에 독일군의 각 부대는 포위되어 소련 포병과 공군의 맹공격을 당했고, 급속히 괴멸되었다. 그제서야 소련군의 주공이 정면이라는 것을 안 중앙집단군 사령관 <a href="/wiki/%EC%97%90%EB%A5%B8%EC%8A%A4%ED%8A%B8_%EB%B6%80%EC%8A%88" title="에른스트 부슈">에른스트 부슈</a>원수는 27일에 히틀러에게 철수 허가를 청원했으나, 망연자실한 히틀러는 다음날 부슈를 해임하고, 북우크라이나 집단군의 모델 원수를 후임 사령관으로 임명하였다. 모델 원수는 수비전에서 명성을 떨쳤으나 현재 상황은 그도 어쩔수 없는 것이었다. 7월 1일, 육군 총참모장인 <a class="new" href="/w/index.php?title=%EC%BF%A0%EB%A5%B4%ED%8A%B8_%EC%9E%90%EC%9D%B4%EC%B8%A8%EB%9F%AC&amp;action=edit&amp;redlink=1" title="쿠르트 자이츨러 (없는 문서)">쿠르트 자이츨러</a>는 히틀러가 철퇴안을 허가하지 않자 사임을 했고, 7월 21일에 <a class="mw-redirect" href="/wiki/%EA%B5%AC%EB%8D%B0%EB%A6%AC%EC%95%88" title="구데리안">구데리안</a> 이 새로운 육군 총참모장에 임명되었다.</p>
<h3><span class="mw-headline" id=".EB.B9.84.EC.B3.85.EC.8A.A4.ED.81.AC">비쳅스크</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%B0%94%EA%B7%B8%EB%9D%BC%ED%8B%B0%EC%98%A8_%EC%9E%91%EC%A0%84&amp;action=edit&amp;section=6" title="부분 편집: 비쳅스크">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>독일군의 지휘부가 이렇게 혼란에 빠져있는 동안 중앙집단군은 급속도로 붕괴되었다. 또한 민스크-모스크바 고속도로의 북쪽에 있는 비쳅스크 정면에서는 <a class="mw-redirect" href="/wiki/%ED%99%89%ED%95%98%EB%84%A4%EC%8A%A4_%EB%B0%94%EA%B7%B8%EB%9D%BC%EB%AF%B8%EC%95%88" title="홉하네스 바그라미안">홉하네스 바그라미안</a> 대장의 제1 발트 전선군과 <a href="/wiki/%EC%9D%B4%EB%B0%98_%EC%B2%B4%EB%A5%B4%EB%83%90%ED%99%89%EC%8A%A4%ED%82%A4" title="이반 체르냐홉스키">이반 체르냐홉스키</a> 대장의 제3 벨라루스 전선군이 독일 <a href="/wiki/%EA%B2%8C%EC%98%A4%EB%A5%B4%ED%81%AC%ED%95%9C%EC%8A%A4_%EB%9D%BC%EC%9D%B8%ED%95%98%EB%A5%B4%ED%8A%B8" title="게오르크한스 라인하르트">게오르크한스 라인하르트</a> 상급대장의 제3기갑군을 협격하여 6월 27일에는 3만명의 독일 제53군단을 포위망에서 괴멸시켰다. 공백지인 <a class="new" href="/w/index.php?title=%EB%93%9C%EB%B9%84%EB%82%98_%EA%B0%95&amp;action=edit&amp;redlink=1" title="드비나 강 (없는 문서)">드비나 강</a> 연안에서도 소련 제2친위군이 발트 해 방면으로 급속도로 진격하여 7월 13일에는 <a href="/wiki/%EB%A6%AC%ED%88%AC%EC%95%84%EB%8B%88%EC%95%84" title="리투아니아">리투아니아</a>의 수도 <a class="mw-redirect" href="/wiki/%EB%B9%8C%EB%8B%88%EC%9A%B0%EC%8A%A4" title="빌니우스">빌니우스</a>를 탈환하였다.</p>
<h3><span class="mw-headline" id=".EB.A7.88.ED.9E.90.EB.A3.8C.EC.9A.B0">마힐료우</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%B0%94%EA%B7%B8%EB%9D%BC%ED%8B%B0%EC%98%A8_%EC%9E%91%EC%A0%84&amp;action=edit&amp;section=7" title="부분 편집: 마힐료우">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>중앙집단군의 정면인 <a href="/wiki/%EB%A7%88%ED%9E%90%EB%A3%8C%EC%9A%B0" title="마힐료우">마힐료우</a>에서는 <a class="new" href="/w/index.php?title=%EA%B2%8C%EC%98%A4%EB%A5%B4%EA%B8%B0_%EC%9E%90%ED%95%98%EB%A1%9C%ED%94%84&amp;action=edit&amp;redlink=1" title="게오르기 자하로프 (없는 문서)">게오르기 자하로프</a> 대장의 제2 벨라루스 전선군이 오르샤를 확보한 제3벨라루스 정면군의 좌익에서 협동하여 6월 28일 <a href="/wiki/%EB%A7%88%ED%9E%90%EB%A3%8C%EC%9A%B0" title="마힐료우">마힐료우</a>를 탈환하고, 민스크 공쪽에서 독일 <a class="new" href="/w/index.php?title=%EC%BF%A0%EB%A5%B4%ED%8A%B8_%ED%8F%B0_%ED%8B%B0%ED%8E%A0%EC%8A%A4%ED%82%A4%EB%A5%B4%ED%9E%88&amp;action=edit&amp;redlink=1" title="쿠르트 폰 티펠스키르히 (없는 문서)">쿠르트 폰 티펠스키르히</a> 대장의 제4군을 포위하여(티펠스키르히 대장은 비행기 사고로 독일로 이송되었다.) 후임 사령관인 <a class="new" href="/w/index.php?title=%EB%B9%88%EC%84%BC%ED%8A%B8_%EB%AE%90%EB%9F%AC&amp;action=edit&amp;redlink=1" title="빈센트 뮐러 (없는 문서)">빈센트 뮐러</a> 중장으로부터 7월 11일 항복을 받았다. 이때 전사자는 4만명이고, 포로는 6만명이었다.</p>
<h3><span class="mw-headline" id=".EB.B3.B4.EB.B8.8C.EB.A3.A8.EC.9D.B4.EC.8A.A4.ED.81.AC">보브루이스크</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%B0%94%EA%B7%B8%EB%9D%BC%ED%8B%B0%EC%98%A8_%EC%9E%91%EC%A0%84&amp;action=edit&amp;section=8" title="부분 편집: 보브루이스크">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>또한 남부의 <a class="new" href="/w/index.php?title=%EB%B2%A0%EB%A0%88%EC%A7%80%EB%82%98_%EA%B0%95&amp;action=edit&amp;redlink=1" title="베레지나 강 (없는 문서)">베레지나 강</a> 맞은편에 있는 <a href="/wiki/%EB%B0%94%EB%B8%8C%EB%A3%A8%EC%9D%B4%EC%8A%A4%ED%81%AC" title="바브루이스크">바브루이스크</a>에서도 <a class="mw-redirect" href="/wiki/%EB%A1%9C%EC%BD%94%EC%86%8C%ED%94%84%EC%8A%A4%ED%82%A4" title="로코소프스키">로코소프스키</a>가 지휘하는 제1벨라루스 전선군이 스탈린의 만류에도 불구하고 습지에 통나무를 깔고 그위로 전차부대를 움직이는 대담한 전법으로 양방향으로부터 공세를 펼쳤다. <a class="new" href="/w/index.php?title=%ED%95%9C%EC%8A%A4_%EC%9A%94%EB%A5%B4%EB%8B%A8&amp;action=edit&amp;redlink=1" title="한스 요르단 (없는 문서)">한스 요르단</a>(Hans Jordan) 대장이 지휘하는 독일 제9군은 <a class="new" href="/w/index.php?title=%ED%94%84%EB%A6%AC%ED%94%BC%EC%95%84%ED%8A%B8_%EC%8A%B5%EC%A7%80%EB%8C%80&amp;action=edit&amp;redlink=1" title="프리피아트 습지대 (없는 문서)">프리피아트 습지대</a>로 쫓겨갔다가 7월 5일에 항복했다. 독일군의 피해는 전사 4만명, 포로 2만명이었다.</p>
<h2><span class="mw-headline" id=".EC.9E.91.EC.A0.84_.EC.A0.9C2.EB.8B.A8.EA.B3.84">작전 제2단계</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%B0%94%EA%B7%B8%EB%9D%BC%ED%8B%B0%EC%98%A8_%EC%9E%91%EC%A0%84&amp;action=edit&amp;section=9" title="부분 편집: 작전 제2단계">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<h3><span class="mw-headline" id=".EB.AF.BC.EC.8A.A4.ED.81.AC_.ED.95.B4.EB.B0.A9">민스크 해방</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%B0%94%EA%B7%B8%EB%9D%BC%ED%8B%B0%EC%98%A8_%EC%9E%91%EC%A0%84&amp;action=edit&amp;section=10" title="부분 편집: 민스크 해방">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>이렇게 세 방향에서 각각 공세를 전개한 소련군 제1, 제2, 제3 벨라루스 전선군은 계획대로 <a href="/wiki/%EB%AF%BC%EC%8A%A4%ED%81%AC" title="민스크">민스크</a>에서 합류하였고, 7월 4일에 3년째 독일군에 점령중이던 민스크를 해방시켰다.</p>
<h3><span class="mw-headline" id=".EB.82.A8.EB.B6.81.EC.9C.BC.EB.A1.9C.EB.B6.80.ED.84.B0_.EA.B3.B5.EC.84.B8">남북으로부터 공세</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%B0%94%EA%B7%B8%EB%9D%BC%ED%8B%B0%EC%98%A8_%EC%9E%91%EC%A0%84&amp;action=edit&amp;section=11" title="부분 편집: 남북으로부터 공세">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>주무대인 벨라루스의 남쪽에서는 코네프 원수가 지휘하는 남부의 제1우크라이나 전선군은 <a class="mw-redirect" href="/wiki/%EA%B0%88%EB%A6%AC%EC%8B%9C%EC%95%84" title="갈리시아">갈리시아</a> 방면에 주둔중이던 <a class="new" href="/w/index.php?title=%EA%B3%A0%ED%8A%B8%ED%95%98%EB%A5%B4%ED%8A%B8_%ED%95%98%EC%9D%B8%EB%A6%AC%EC%B9%98&amp;action=edit&amp;redlink=1" title="고트하르트 하인리치 (없는 문서)">고트하르트 하인리치</a>대장의 독일 제1 기갑군 및 <a href="/wiki/%EB%B0%9C%ED%84%B0_%EB%84%A4%EB%A7%81" title="발터 네링">발터 네링</a> 대장의 제4 기갑군이 벨라루스 방면의 지원에 나서지 못하도록 <a class="new" href="/w/index.php?title=%EB%A5%B4%EB%B3%B4%ED%94%84-%EC%82%B0%EB%8F%84%EB%AF%B8%EC%97%90%EB%A5%B4%EC%A6%88_%EA%B3%B5%EC%84%B8&amp;action=edit&amp;redlink=1" title="르보프-산도미에르즈 공세 (없는 문서)">르보프-산도미에르즈 공세</a>를 시작하였다. 바실리 고르도프 중장의 소련군 제3친위군은 독일군 제13군 2만5천 명을 포위했다. 독일군은 중화기를 거의 포기하고 간신히 탈출하였다. 이어 7월 27일에 르보프를 해방시키고 소련-폴란드 국경을 넘어 8월 1일에는 <a href="/wiki/%EB%A3%A8%EB%B8%94%EB%A6%B0" title="루블린">루블린</a>을 점령하였다. 이로써 카르파티아 산맥 동쪽의 독일군은 모두 축출되었다.</p>
<p>한편 벨라루스의 북쪽에서는 바그리미얀 대장의 제1발트 전선군이 <a class="mw-redirect" href="/wiki/%EC%83%A4%EC%9A%B8%EB%A0%88%EC%9D%B4" title="샤울레이">샤울레이</a> 공세를 개시하여 7월 말에는 <a href="/wiki/%EB%9D%BC%ED%8A%B8%EB%B9%84%EC%95%84" title="라트비아">라트비아</a>의 <a href="/wiki/%EB%A6%AC%EA%B0%80" title="리가">리가</a>에 도달하였고, 독일 중앙집단군과 북방집단군을 단절시키는 데 성공했다.</p>
<p>이어 8월에는 <a class="mw-redirect" href="/wiki/%EC%95%88%EB%93%9C%EB%A0%88%EC%9D%B4_%EC%98%88%EB%A0%88%EB%A9%98%EC%BD%94" title="안드레이 예레멘코">안드레이 예레멘코</a> 상급대장의 제2발트 전선군과 <a class="new" href="/w/index.php?title=%EC%9D%B4%EB%B0%98_%EB%A7%90%EB%A0%88%EB%8B%88%EC%BD%94%ED%94%84&amp;action=edit&amp;redlink=1" title="이반 말레니코프 (없는 문서)">이반 말레니코프</a> 대장의 제3 발트전선군, <a href="/wiki/%EB%A0%88%EC%98%A4%EB%8B%88%ED%8A%B8_%EA%B3%A0%EB%B3%B4%EB%A1%9C%ED%94%84" title="레오니트 고보로프">레오니트 고보로프</a> 대장의 레닌그라드 전선군이 <a href="/wiki/%EC%97%90%EC%8A%A4%ED%86%A0%EB%8B%88%EC%95%84" title="에스토니아">에스토니아</a>를 향한 공세를 시작하였다.</p>
<p>페르디난트 쇠르너 대장의 독일 북방집단군은 <a href="/wiki/%EB%AC%B4%EC%9E%A5%EC%B9%9C%EC%9C%84%EB%8C%80" title="무장친위대">무장친위대</a> <a class="new" href="/w/index.php?title=%EA%B7%B8%EB%A1%9C%EC%8A%A4_%EB%8F%84%EC%9D%B4%EC%B9%A0%EB%9E%80%ED%8A%B8_%EC%82%AC%EB%8B%A8&amp;action=edit&amp;redlink=1" title="그로스 도이칠란트 사단 (없는 문서)">그로스 도이칠란트 사단</a>과 중순양함 프린츠 오이겐의 지원을 받아 리투아니아의 <a class="new" href="/w/index.php?title=%EC%A0%A4%EA%B0%80%EB%B0%94&amp;action=edit&amp;redlink=1" title="젤가바 (없는 문서)">젤가바</a>를 탈환하여 다시 중앙집단군과 연결되었다. 그러나 제1발트 전선군에 속한 볼스키 중장의 제5친위전차군에 의해 10월 10일 <a class="mw-redirect" href="/wiki/%EB%A9%94%EB%A9%9C" title="메멜">메멜</a>에서 포위당해 독일 제16, 제18군은 <a href="/wiki/%EB%8F%99%ED%94%84%EB%A1%9C%EC%9D%B4%EC%84%BC" title="동프로이센">동프로이센</a>과 단절되었고 쿠를란트 반도에서 고립되었다. 히틀러는 북부집단군의 잔해를 <a class="new" href="/w/index.php?title=%EC%BF%A0%EB%A5%BC%EB%9E%80%ED%8A%B8_%EC%A7%91%EB%8B%A8%EA%B5%B0&amp;action=edit&amp;redlink=1" title="쿠를란트 집단군 (없는 문서)">쿠를란트 집단군</a>으로 재편하여 <a class="new" href="/w/index.php?title=%EB%A1%9C%ED%83%80%EB%A5%B4_%EB%A0%8C%EB%91%98%EB%A6%AD&amp;action=edit&amp;redlink=1" title="로타르 렌둘릭 (없는 문서)">로타르 렌둘릭</a> 대장을 사령관으로 임명하였다. 이곳은 U보트 기지였기 때문에 독일로서는 쉽게 포기할 수 없는 곳이었다. 쿠를란트 집단군은 포위 중에서도 해상보급으로 연명하면서 종전까지 항복하지 않고 버텼다.</p>
<h2><span class="mw-headline" id=".EC.A0.9C3.EB.8B.A8.EA.B3.84">제3단계</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%B0%94%EA%B7%B8%EB%9D%BC%ED%8B%B0%EC%98%A8_%EC%9E%91%EC%A0%84&amp;action=edit&amp;section=12" title="부분 편집: 제3단계">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>중앙집단군을 격파하고 민스크에서 합류한 제1, 제2 벨라루스 전선군은 중단없이 계속 진격을 하여 7월 13일에는 <a class="mw-redirect" href="/wiki/%EB%B9%8C%EB%8B%88%EC%9A%B0%EC%8A%A4" title="빌니우스">빌니우스</a>-<a href="/wiki/%ED%95%80%EC%8A%A4%ED%81%AC" title="핀스크">핀스크</a>까지 이르렀다. <a href="/wiki/7%EC%9B%94_18%EC%9D%BC" title="7월 18일">7월 18일</a>에는 제1벨라루스 전선군에 속한 제28군이 전쟁 전의 독일-소련의 국경선이었던 <a href="/wiki/%EB%B8%8C%EB%A0%88%EC%8A%A4%ED%8A%B8_(%EB%B2%A8%EB%9D%BC%EB%A3%A8%EC%8A%A4)" title="브레스트 (벨라루스)">브레스트-리토프스크</a> 를 점령하였고, 부그강을 건너 <a href="/wiki/%ED%8F%B4%EB%9E%80%EB%93%9C" title="폴란드">폴란드</a>에 입성하였다. 28일에는 <a href="/wiki/%EC%B9%B4%EC%9A%B0%EB%82%98%EC%8A%A4" title="카우나스">카우나스</a>-<a href="/wiki/%EB%A3%A8%EB%B8%94%EB%A6%B0" title="루블린">루블린</a>의 전선을 형성하고, 교두보를 구축하였다. 8월 초순에는 제1친위 전차군, 제8 친위군(<a href="/wiki/%EC%8A%A4%ED%83%88%EB%A6%B0%EA%B7%B8%EB%9D%BC%EB%93%9C_%EC%A0%84%ED%88%AC" title="스탈린그라드 전투">스탈린그라드 전투</a>의 주역부대인 제62군이 개편된 부대), 제70군으로 이루어진 정예부대가 <a href="/wiki/%EB%B0%94%EB%A5%B4%EC%83%A4%EB%B0%94" title="바르샤바">바르샤바</a>의 맞은편의 <a href="/wiki/%EB%B9%84%EC%8A%A4%EC%99%80_%EA%B0%95" title="비스와 강">비스와 강</a>에 도달하였다. 그러나 교량은 파괴되었고, 급속도의 전진 때문에 보급이 곤란한데다가 강 너머에서는 독일 중앙집단군이 재편되어 강한 저항선을 형성했기 때문에 소련군은 더이상 전진하지 않고 멈추었다.</p>
<p>소련군은 5주간 700km를 전진하였고, 독일군의 기둥뿌리를 빼앗는 전격전을 성공시켰다. 이는 1930년대 완성한 소련군의 <a href="/wiki/%EC%A2%85%EC%8B%AC%EC%9E%91%EC%A0%84%EC%9D%B4%EB%A1%A0" title="종심작전이론">종심타격이론</a>의 확실한 구현이었다. 독일군의 발표에서도 25만명의 전사자와 11만명이 넘는 포로가 나타나는 대타격을 주었고, 동부전선의 승부를 사실상 확정짓었다. 이 작전의 경이적인 성공때문에 <a href="/wiki/%EB%83%89%EC%A0%84" title="냉전">냉전</a>시절 서방국들은 소련군의 전력에 크게 위협을 느꼈다.</p>
<h2><span class="mw-headline" id=".EA.B2.B0.EA.B3.BC">결과</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%B0%94%EA%B7%B8%EB%9D%BC%ED%8B%B0%EC%98%A8_%EC%9E%91%EC%A0%84&amp;action=edit&amp;section=13" title="부분 편집: 결과">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>이 작전의 이전에는 120만명에 달했던 독일 중앙집단군은 전사, 실종, 포로, 부상 등 총 사상자 60만명, 25개 사단을 잃는 괴멸적인 타격을 받고 장성급 지휘관 47명중 31명이 전사, 실종, 부상, 포로가 되었으며 탱크 및 돌격포 1,000대와 중포 및 항공기 1,700대를 잃는 피해를 입었다. 소련군은 이 작전에서 잡은 10만명이 넘는 독일군 포로를 모스크바 시내에서 행군시켜 자국민의 사기를 높이고 영·미군에 과시하였다.</p>
<p>독일군은 애당초 바그라티온 작전 이전에 보유하고 있던 총 병력 숫자가 소련군의 반도 안 되었으며, 처음부터 소련군의 의도를 오판하여 제대로 된 대응을 할 수 없었다. 더욱이 소련군 공세의 와중인 7월 20일에 <a class="mw-redirect" href="/wiki/%ED%9E%88%ED%8B%80%EB%9F%AC_%EC%95%94%EC%82%B4_%EB%AF%B8%EC%88%98_%EC%82%AC%EA%B1%B4" title="히틀러 암살 미수 사건">히틀러 암살 미수 사건</a>이 발생하기도 하여 히틀러와 독일군 지휘부는 정신을 차릴 수가 없었다.</p>
<p>소련군의 피해도 극심하여 총 240만명의 병력 중에 77만명이 넘는 사상자와 탱크 및 돌격포 2,957대, 중포 2,447문, 항공기 822대를 잃었지만 소련은 이런 피해를 금세 보충할 수 있었고, 이 작전이 종료된 8월 이후에는 남부의 <a href="/wiki/%EB%B0%9C%EC%B9%B8_%EB%B0%98%EB%8F%84" title="발칸 반도">발칸 반도</a>와 북부의 <a href="/wiki/%EB%B0%9C%ED%8A%B8_3%EA%B5%AD" title="발트 3국">발트 3국</a>에 대한 연속적인 공세작전이 시작되어 독일의 동맹국인 <a href="/wiki/%EB%A3%A8%EB%A7%88%EB%8B%88%EC%95%84" title="루마니아">루마니아</a>, <a href="/wiki/%EB%B6%88%EA%B0%80%EB%A6%AC%EC%95%84" title="불가리아">불가리아</a>, 그리고 <a href="/wiki/%ED%95%80%EB%9E%80%EB%93%9C" title="핀란드">핀란드</a>가 8-9월에 각각 사실상 항복했다.</p>
<p>또한 소련은 점령지에 괴뢰국가를 세우기 위해 발빠르게 움직였고, 영국에 있던 반공적인 <a class="mw-redirect" href="/wiki/%ED%8F%B4%EB%9E%80%EB%93%9C_%EB%A7%9D%EB%AA%85%EC%A0%95%EB%B6%80" title="폴란드 망명정부">폴란드 망명정부</a>에 대항하여 소련은 자국의 조종을 받는 <a class="new" href="/w/index.php?title=%ED%8F%B4%EB%9E%80%EB%93%9C_%EC%9D%B8%EB%AF%BC%ED%95%B4%EB%B0%A9%EC%9C%84%EC%9B%90%ED%9A%8C&amp;action=edit&amp;redlink=1" title="폴란드 인민해방위원회 (없는 문서)">폴란드 인민해방위원회</a>를 설립하여 폴란드 영토내인 <a href="/wiki/%EB%A3%A8%EB%B8%94%EB%A6%B0" title="루블린">루블린</a>으로 이동시켰다.</p>
<p>8월에 <a href="/wiki/%EB%B0%94%EB%A5%B4%EC%83%A4%EB%B0%94" title="바르샤바">바르샤바</a> 전면에서 멈춘 소련군은 1월까지 꼼짝않고 있으면서 후에 반소적인 폴란드 저항세력이 <a href="/wiki/%EB%B0%94%EB%A5%B4%EC%83%A4%EB%B0%94_%EB%B4%89%EA%B8%B0" title="바르샤바 봉기">독일군에 대해 봉기했을때도</a> 강건너에서 가만히 구경만 하고 있어서 서방 연합국들은 이를 비난하였다. 이는 런던의 폴란드 망명정부를 지지하는 이들 저항세력이 독일군에 진압되도록 놔두었다는 관측도 있다.</p>
<p>그러나 바그라티온 작전 후반기에 다다르면서 전열을 재정비한 독일군의 반격과 바그라티온 작전 초반기부터 계속 누적되어 온 소련군의 피해가 워낙에 극심하였기 때문에 <a href="/wiki/%EB%B0%94%EB%A5%B4%EC%83%A4%EB%B0%94_%EB%B4%89%EA%B8%B0" title="바르샤바 봉기">바르샤바 봉기</a>가 일어난 <a href="/wiki/1944%EB%85%84" title="1944년">1944년</a> 8월에는 실제로 소련군의 공세가 독일군에 의해서 저지되었다. 독일군은 바르샤바를 앞에 둔 폴란드의 <a class="new" href="/w/index.php?title=%EB%B6%80%EA%B7%B8_%EA%B0%95&amp;action=edit&amp;redlink=1" title="부그 강 (없는 문서)">부그 강</a> 일대에서 소련군의 공격을 패퇴시키고 이 일대에서 소련군은 독일군에 의해서 패배를 당해 엄청난 피해를 입었기 때문에 소련군으로서는 바르샤바를 향한 추가적인 공세가 힘들었던 것도 사실이다. 당시 소련군이 진정한 의도는 현재까지 논란의 대상이다.</p>
