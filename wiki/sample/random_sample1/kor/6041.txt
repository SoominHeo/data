<p>《<b>바람의 나라</b>》는 기원 후 <a href="/wiki/18%EB%85%84" title="18년">18년</a>에서 <a href="/wiki/23%EB%85%84" title="23년">23년</a> 사이 동아시아를 배경으로 한 <a class="mw-redirect" href="/wiki/%EA%B9%80%EC%A7%84_(%EB%A7%8C%ED%99%94%EA%B0%80)" title="김진 (만화가)">김진</a>의 만화 《<a href="/wiki/%EB%B0%94%EB%9E%8C%EC%9D%98_%EB%82%98%EB%9D%BC_(%EB%A7%8C%ED%99%94)" title="바람의 나라 (만화)">바람의 나라</a>》를 소재로 하여 <a href="/wiki/%EB%84%A5%EC%8A%A8" title="넥슨">넥슨</a>이 제작한, 현재 서비스 중인 그래픽 도입 <a class="mw-redirect" href="/wiki/MMORPG" title="MMORPG">MMORPG</a> 중에선 가장 오래된 게임이다.<sup class="reference" id="cite_ref-2"><a href="#cite_note-2">[2]</a></sup></p>
<p>바람의 나라는 <a href="/wiki/%EA%B3%A0%EA%B5%AC%EB%A0%A4" title="고구려">고구려</a> 2대 왕인 유리왕의 아들 <a href="/wiki/%EB%8C%80%EB%AC%B4%EC%8B%A0%EC%99%95" title="대무신왕">대무신왕</a> 무휼의 정벌담에, 차비 연, 아들 <a href="/wiki/%ED%98%B8%EB%8F%99" title="호동">호동</a> 왕자과 <a href="/wiki/%EB%82%99%EB%9E%91" title="낙랑">낙랑</a> 공주의 사랑 이야기가 중첩되는 만화가 <a class="mw-redirect" href="/wiki/%EA%B9%80%EC%A7%84_(%EB%A7%8C%ED%99%94%EA%B0%80)" title="김진 (만화가)">김진</a>의 역사 서사극 <a href="/wiki/%EB%B0%94%EB%9E%8C%EC%9D%98_%EB%82%98%EB%9D%BC_(%EB%A7%8C%ED%99%94)" title="바람의 나라 (만화)">바람의 나라</a>를 원작으로 하는 전쟁 이야기를 배경으로 그렸다.</p>
<p><a href="/wiki/1994%EB%85%84" title="1994년">1994년</a> <a href="/wiki/12%EC%9B%94" title="12월">12월</a> 기획되어 <a href="/wiki/1995%EB%85%84" title="1995년">1995년</a> <a href="/wiki/12%EC%9B%94_25%EC%9D%BC" title="12월 25일">12월 25일</a> 베타테스트를 실시하였으며, <a href="/wiki/1996%EB%85%84" title="1996년">1996년</a> <a href="/wiki/4%EC%9B%94_5%EC%9D%BC" title="4월 5일">4월 5일</a> 정식 서비스를 시작하였다. 바람의 나라는 <a href="/wiki/%EB%8C%80%ED%95%9C%EB%AF%BC%EA%B5%AD" title="대한민국">대한민국</a> 내 온라인 게임 중 처음으로 대한민국 외의 국가에서도 서비스를 시작했다. <a href="/wiki/2013%EB%85%84" title="2013년">2013년</a> <a href="/wiki/7%EC%9B%94_10%EC%9D%BC" title="7월 10일">7월 10일</a>에는 넥슨컴퓨터박물관 IT보존연구실장은 소스코드를 복원하는 기술적인 측면뿐만 아니라 오리지널 버전의 감성과 철학도 함께 복원하는 것이라며 이전에 참여한 바람의 나라 제작진과 함께 초기버전을 복원한다고 발표하였고, 2014년 5월 복원이 완료되었다.</p>
