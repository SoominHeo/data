<h1 class="firstHeading" id="firstHeading" lang="ko">바르바로사</h1>
<div class="mw-body-content" id="bodyContent">
<div id="siteSub">위키백과, 우리 모두의 백과사전.</div>
<div id="contentSub"></div>
<div class="mw-jump" id="jump-to-nav">
					이동:					<a href="#mw-head">둘러보기</a>, 					<a href="#p-search">검색</a>
</div>
<div class="mw-content-ltr" dir="ltr" id="mw-content-text" lang="ko"><script>function mfTempOpenSection(id){var block=document.getElementById("mf-section-"+id);block.className+=" open-block";block.previousSibling.className+=" open-block";}</script><p><b>바르바로사</b>(Barbarossa)는 이탈리아어로 "붉은 수염"이라는 뜻이다.</p>
<ul>
<li>"바르바로사"를 별명으로 가지는 유명한 사람은
<ul>
<li><a class="mw-redirect" href="/wiki/%EC%8B%A0%EC%84%B1%EB%A1%9C%EB%A7%88%EC%A0%9C%EA%B5%AD%EC%9D%98_%ED%94%84%EB%A6%AC%EB%93%9C%EB%A6%AC%ED%9E%88_1%EC%84%B8" title="신성로마제국의 프리드리히 1세">프리드리히 1세 바르바로사</a>(1125~1190) : <a class="mw-redirect" href="/wiki/%EC%8B%A0%EC%84%B1%EB%A1%9C%EB%A7%88%EC%A0%9C%EA%B5%AD" title="신성로마제국">신성로마제국</a>의 황제</li>
<li><a class="new" href="/w/index.php?title=%EB%B0%94%EB%B0%94_%EC%9A%B0%EB%A5%B4%EC%A7%80&amp;action=edit&amp;redlink=1" title="바바 우르지 (없는 문서)">바바 우르지 바르바로사</a>(1474-1518) : 바르바리 해적, <a href="/wiki/%EC%98%A4%EC%8A%A4%EB%A7%8C_%EC%A0%9C%EA%B5%AD" title="오스만 제국">오스만 제국</a>의 해군 제독.</li>
<li><a class="mw-redirect" href="/wiki/%EB%B0%9C%EB%B0%94%EB%A1%AF%EC%82%AC_%ED%95%98%EC%9D%B4%EB%A0%88%EB%94%98_%ED%8C%8C%EC%83%A4" title="발바롯사 하이레딘 파샤">하이레딘 바르바로사</a> (1478 – 1546) : <a href="/wiki/%EB%B0%94%EB%A5%B4%EB%B0%94%EB%A6%AC" title="바르바리">바르바리</a> 해적. <a href="/wiki/%EC%98%A4%EC%8A%A4%EB%A7%8C_%EC%A0%9C%EA%B5%AD" title="오스만 제국">오스만 제국</a>의 해군 제독. 바바 우르지의 동생.</li>
</ul>
</li>
</ul>
<ul>
<li><a href="/wiki/%EB%B0%94%EB%A5%B4%EB%B0%94%EB%A1%9C%EC%82%AC_%EC%9E%91%EC%A0%84" title="바르바로사 작전">바르바로사 작전</a>은 <a href="/wiki/%EC%A0%9C2%EC%B0%A8_%EC%84%B8%EA%B3%84_%EB%8C%80%EC%A0%84" title="제2차 세계 대전">제2차 세계 대전</a> 당시 <a class="mw-redirect" href="/wiki/%EB%82%98%EC%B9%98" title="나치">나치</a> 독일이 <a class="mw-redirect" href="/wiki/%EC%86%8C%EB%B9%84%EC%97%90%ED%8A%B8_%EC%97%B0%EB%B0%A9" title="소비에트 연방">소비에트 연방</a>을 침공할 썼던 근시 작전 이름이다. 이 이름을 딴 전쟁 게임이 여럿 있다.</li>
</ul>
