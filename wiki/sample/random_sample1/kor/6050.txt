<p><b>바르샤바 국립경기장</b>(<span style="font-size: smaller;"><a href="/wiki/%ED%8F%B4%EB%9E%80%EB%93%9C%EC%96%B4" title="폴란드어">폴란드어</a>:</span> <span lang="pl" xml:lang="pl">Stadion Narodowy</span>)은 <a href="/wiki/%ED%8F%B4%EB%9E%80%EB%93%9C" title="폴란드">폴란드</a> <a href="/wiki/%EB%B0%94%EB%A5%B4%EC%83%A4%EB%B0%94" title="바르샤바">바르샤바</a>에 위치한 <a href="/wiki/%EC%B6%95%EA%B5%AC" title="축구">축구</a> 경기장이다. 스타디온 지에시엥치올레치아(Stadion Dziesięciolecia, 폴란드어로 "10주년 경기장"이라는 뜻)가 있던 곳에 건설되었고 <a href="/wiki/UEFA_%EC%9C%A0%EB%A1%9C_2012" title="UEFA 유로 2012">UEFA 유로 2012</a> 경기장으로 선정되었다. <a href="/wiki/2012%EB%85%84" title="2012년">2012년</a> <a href="/wiki/1%EC%9B%94_29%EC%9D%BC" title="1월 29일">1월 29일</a>에 준공되었으며 UEFA 유로 2012 개막전과 준결승전 경기가 열렸다. 대회 이후에는 <a href="/wiki/%EB%A0%88%EA%B8%B0%EC%95%84_%EB%B0%94%EB%A5%B4%EC%83%A4%EB%B0%94" title="레기아 바르샤바">레기아 바르샤바</a>와 <a href="/wiki/%ED%8F%B4%EB%A1%9C%EB%8B%88%EC%95%84_%EB%B0%94%EB%A5%B4%EC%83%A4%EB%B0%94" title="폴로니아 바르샤바">폴로니아 바르샤바</a>의 홈 구장으로 사용되고 있다.</p>
<p>2014-15 시즌 <a href="/wiki/UEFA_%EC%9C%A0%EB%A1%9C%ED%8C%8C%EB%A6%AC%EA%B7%B8" title="UEFA 유로파리그">UEFA 유로파리그</a> 결승전 개최지로 선정되었다.</p>
