<p><b>바빌로니아</b>(Babylonia)는 <a href="/wiki/%EB%A9%94%EC%86%8C%ED%8F%AC%ED%83%80%EB%AF%B8%EC%95%84" title="메소포타미아">메소포타미아</a> 남쪽의 고대 왕국으로, 이전 시대의 <a href="/wiki/%EC%88%98%EB%A9%94%EB%A5%B4" title="수메르">수메르</a> 지방과 <a href="/wiki/%EC%95%84%EC%B9%B4%EB%93%9C" title="아카드">아카드</a> 지방을 아우르며, <a class="mw-redirect" href="/wiki/%ED%8B%B0%EA%B7%B8%EB%A6%AC%EC%8A%A4%EA%B0%95" title="티그리스강">티그리스강</a>과 <a class="mw-redirect" href="/wiki/%EC%9C%A0%ED%94%84%EB%9D%BC%ED%85%8C%EC%8A%A4%EA%B0%95" title="유프라테스강">유프라테스강</a> 사이 <a href="/wiki/%EB%A9%94%EC%86%8C%ED%8F%AC%ED%83%80%EB%AF%B8%EC%95%84" title="메소포타미아">메소포타미아</a> 남동쪽의 지명이다.'바빌로니아'라는 이름은 수도였던 바빌론(Babylon)에서 유래하였다. 한글 <a class="mw-redirect" href="/wiki/%EA%B5%AC%EC%95%BD%EC%84%B1%EC%84%9C" title="구약성서">구약성서</a>에서는 바빌론, 바벨론 등으로 음역하고 있다. 바빌론에 대한 최초의 기록은 <a href="/wiki/%EA%B8%B0%EC%9B%90%EC%A0%84_23%EC%84%B8%EA%B8%B0" title="기원전 23세기">기원전 23세기</a>경 <a href="/wiki/%EC%95%84%EC%B9%B4%EB%93%9C" title="아카드">아카드</a>의 사르곤(Sargon) 왕의 지배에 대한 점토판에서 찾을 수 있다.</p>
<p>바빌로니아는 <a href="/wiki/%EC%88%98%EB%A9%94%EB%A5%B4" title="수메르">수메르</a>인과 <a href="/wiki/%EC%95%84%EC%B9%B4%EB%93%9C" title="아카드">아카드</a>인이 차례로 통치하였다. 경작이 용이하고, 상업적, 전략적으로 중요한 지형이어서 이민족의 침입을 많이 받았다.</p>

<p>바빌로니아의 역사는 <i>고대 바빌로니아</i>와 <i>카사이트(Kassite)왕조</i> 그리고 <i>신 바빌로니아 제국</i>(Chaldean Era, 칼데아 기)로 분류된다.</p>
<h3><span class="mw-headline" id=".EA.B3.A0.EB.8C.80_.EB.B0.94.EB.B9.8C.EB.A1.9C.EB.8B.88.EC.95.84">고대 바빌로니아</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%B0%94%EB%B9%8C%EB%A1%9C%EB%8B%88%EC%95%84&amp;action=edit&amp;section=3" title="부분 편집: 고대 바빌로니아">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p><a href="/wiki/%EA%B8%B0%EC%9B%90%EC%A0%84_20%EC%84%B8%EA%B8%B0" title="기원전 20세기">기원전 20세기</a>경, <a class="mw-redirect" href="/wiki/%EC%9A%B0%EB%A5%B4_%EC%A0%9C3_%EC%99%95%EC%A1%B0" title="우르 제3 왕조">우르 제3 왕조</a>에 의한 <a href="/wiki/%EC%88%98%EB%A9%94%EB%A5%B4" title="수메르">수메르</a>의 부흥 이후에, <a href="/wiki/%EC%9C%A0%ED%94%84%EB%9D%BC%ED%85%8C%EC%8A%A4_%EA%B0%95" title="유프라테스 강">유프라테스 강</a> 서쪽으로부터 온 <a class="mw-redirect" href="/wiki/%EC%85%88%EC%96%B4%EC%A1%B1" title="셈어족">셈어족</a>의 <a class="mw-redirect" href="/wiki/%EC%95%84%EB%AA%A8%EB%A6%AC" title="아모리">아모리</a>인이 <a href="/wiki/%EB%A9%94%EC%86%8C%ED%8F%AC%ED%83%80%EB%AF%B8%EC%95%84" title="메소포타미아">메소포타미아</a> 지방을 지배하게 된다.</p>
<p>이후 약 100년간은 <a href="/wiki/%EB%A9%94%EC%86%8C%ED%8F%AC%ED%83%80%EB%AF%B8%EC%95%84" title="메소포타미아">메소포타미아</a> 지방은 통일되지 않은 상태였고, <a href="/wiki/%EC%9D%B4%EC%8B%A0" title="이신">이신</a>(Isin)이 가장 강력한 도시였으나, <a class="mw-redirect" href="/wiki/%EC%95%84%EB%AA%A8%EB%A6%AC" title="아모리">아모리</a>인의 도시인 <a href="/wiki/%EB%B0%94%EB%B9%8C%EB%A1%A0" title="바빌론">바빌론</a>을 중심으로 세력을 확장하기 시작하여 결국 첫 번째 바빌로니아 제국을 이루게 된다. 이후 약 300년간을 <b>고대 바빌로니아</b>로 분류한다.</p>
<p>바빌로니아인들은 <a href="/wiki/%EC%8B%9C%EB%A6%AC%EC%95%84" title="시리아">시리아</a>와 <a href="/wiki/%EA%B0%80%EB%82%98%EC%95%88" title="가나안">가나안</a> 등지의 여러 서부 도시들과 정기적인 무역을 하였고, 바빌로니아에도 무역을 위한 <a class="mw-redirect" href="/wiki/%EC%95%84%EB%AA%A8%EB%A6%AC" title="아모리">아모리</a>인의 이주지가 건설되었다.</p>
<p>도시 <a href="/wiki/%EB%B0%94%EB%B9%8C%EB%A1%A0" title="바빌론">바빌론</a>은 여섯 번째 통치자였던 <a href="/wiki/%ED%95%A8%EB%AC%B4%EB%9D%BC%EB%B9%84" title="함무라비">함무라비</a>왕 때에 <a href="/wiki/%EB%A9%94%EC%86%8C%ED%8F%AC%ED%83%80%EB%AF%B8%EC%95%84" title="메소포타미아">메소포타미아</a> 지역의 패권을 장악하였다. 그는 유능한 지도자로서, 법전을 편찬하여 지역을 안정적으로 통치하였다.</p>
<p>바빌로니아인들은 왕을 신 <a class="mw-redirect" href="/wiki/%EB%A7%88%EB%A5%B4%EB%91%91" title="마르둑">마르둑</a>(Marduk)의 대행자로 믿었고, <a href="/wiki/%EB%B0%94%EB%B9%8C%EB%A1%A0" title="바빌론">바빌론</a>은 "신성한 도시"로서, 모든 왕은 그곳에서 왕권을 인정받아야 한다고 믿었다. 왕권을 강화하기 위해 관료 제도와, 세금 제도, 중앙 정부 체제가 갖추어졌다.</p>
<p>바빌로니아가 일어난 뒤고 대대적인 문예 부흥이 일어났다. 이 첫 번째 바빌론 왕조의 가장 큰 업적은 법전의 편찬이다. <a href="/wiki/%EC%97%98%EB%9E%8C" title="엘람">엘람</a>인을 추방하고 왕국이 정착한뒤 <a href="/wiki/%ED%95%A8%EB%AC%B4%EB%9D%BC%EB%B9%84" title="함무라비">함무라비</a>의 지시로 만들어진 이 법전은 <a href="/wiki/%ED%95%A8%EB%AC%B4%EB%9D%BC%EB%B9%84_%EB%B2%95%EC%A0%84" title="함무라비 법전">함무라비 법전</a>(Code of Hammurabi)으로 불린다. 함무라비 법전의 사본이 <a class="mw-redirect" href="/wiki/%EC%88%98%EC%82%AC_(%EB%A9%94%EC%86%8C%ED%8F%AC%ED%83%80%EB%AF%B8%EC%95%84)" title="수사 (메소포타미아)">수사</a>(Susa)에서 1901년에 발견되었으며 현재는 루브르에 소장되어 있다.</p>
<p>바빌로니아의 잘 훈련된 군대는 <a href="/wiki/%EC%9D%B4%EC%8B%A0" title="이신">이신</a>, <a href="/wiki/%EC%97%98%EB%9E%8C" title="엘람">엘람</a>, <a href="/wiki/%EC%9A%B0%EB%A3%A8%ED%81%AC" title="우루크">우루크</a> 등의 도시국가와 강력한 <a href="/wiki/%EB%A7%88%EB%A6%AC_(%EC%88%98%EB%A9%94%EB%A5%B4)" title="마리 (수메르)">마리</a> 왕국을 무너뜨렸다. 바빌론의 지배는 <a href="/wiki/%EC%A7%80%EC%A4%91%ED%95%B4" title="지중해">지중해</a>까지 미쳤다.</p>
<h3><span class="mw-headline" id=".EC.B9.B4.EC.82.AC.EC.9D.B4.ED.8A.B8_.EC.99.95.EC.A1.B0">카사이트 왕조</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%B0%94%EB%B9%8C%EB%A1%9C%EB%8B%88%EC%95%84&amp;action=edit&amp;section=4" title="부분 편집: 카사이트 왕조">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>고대 바빌로니아의 14번째 왕은 샘수-다이타나(Samsu-Ditana)였다. <a class="mw-redirect" href="/wiki/%EA%B8%B0%EC%9B%90%EC%A0%84_1595%EB%85%84" title="기원전 1595년">기원전 1595년</a>에, <a href="/wiki/%ED%9E%88%ED%83%80%EC%9D%B4%ED%8A%B8" title="히타이트">히타이트</a>왕 무르실리 1세(Mursili I)에 의해 약탈당한 이후에, 그의 치세는 <a class="mw-redirect" href="/wiki/%EC%B9%B4%EC%82%AC%EC%9D%B4%ED%8A%B8" title="카사이트">카사이트</a> 왕조로 넘어가게 된다. <a class="mw-redirect" href="/wiki/%EC%B9%B4%EC%82%AC%EC%9D%B4%ED%8A%B8" title="카사이트">카사이트</a> 왕조는 <a href="/wiki/%EB%B0%94%EB%B9%8C%EB%A1%A0" title="바빌론">바빌론</a>의 이름을 "카르-두니아쉬"(Kar-Duniash)로 바꾸고, 576년간 지배했다.</p>
<p>외세의 지배는 비슷한 시기 <a href="/wiki/%ED%9E%89%EC%86%8C%EC%8A%A4" title="힉소스">힉소스</a>인이 <a href="/wiki/%EC%9D%B4%EC%A7%91%ED%8A%B8" title="이집트">이집트</a>를 지배한 것과 아주 유사하며, 이러한 지배 하에서 바빌로니아는 서부 아시아 지역에 대한 패권을 빼앗기게 된다. 이때 <a class="mw-redirect" href="/wiki/%EC%95%84%EC%8A%88%EB%A5%B4" title="아슈르">아슈르</a>의 고위 성직자들은 스스로를 <a href="/wiki/%EC%95%84%EC%8B%9C%EB%A6%AC%EC%95%84" title="아시리아">아시리아</a>의 왕으로 옹립한다.</p>
<p>이 시기에 대부분의 신성한 왕권을 부여받은 솀족 왕들은 바빌로니아에서 사라지고, 카사이트 군주들에게는 신의 지위가 주어지지 않는다. 그러나 <a href="/wiki/%EB%B0%94%EB%B9%8C%EB%A1%A0" title="바빌론">바빌론</a>은 여전히 제국의 수도이자 서아시아의 신성한 도시로서 기능을 잃지 않았고, 성직자들은 강력한 권력을 유지하였다.</p>
<h3><span class="mw-headline" id=".EC.8B.A0_.EB.B0.94.EB.B9.8C.EB.A1.9C.EB.8B.88.EC.95.84_.EC.A0.9C.EA.B5.AD">신 바빌로니아 제국</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%B0%94%EB%B9%8C%EB%A1%9C%EB%8B%88%EC%95%84&amp;action=edit&amp;section=5" title="부분 편집: 신 바빌로니아 제국">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<div class="rellink relarticle mainarticle"><img alt="" data-file-height="128" data-file-width="128" height="16" src="//upload.wikimedia.org/wikipedia/commons/thumb/e/ec/Crystal_Clear_app_xmag.svg/16px-Crystal_Clear_app_xmag.svg.png" srcset="//upload.wikimedia.org/wikipedia/commons/thumb/e/ec/Crystal_Clear_app_xmag.svg/24px-Crystal_Clear_app_xmag.svg.png 1.5x, //upload.wikimedia.org/wikipedia/commons/thumb/e/ec/Crystal_Clear_app_xmag.svg/32px-Crystal_Clear_app_xmag.svg.png 2x" width="16"/> 이 부분의 본문은 <a class="new" href="/w/index.php?title=%EC%8B%A0_%EB%B0%94%EB%B9%8C%EB%A1%9C%EB%8B%88%EC%95%84&amp;action=edit&amp;redlink=1" title="신 바빌로니아 (없는 문서)">신 바빌로니아</a>입니다.</div>
<p><a href="/wiki/%EC%95%84%EC%8B%9C%EB%A6%AC%EC%95%84" title="아시리아">아시리아</a>가 지배권을 유지하고 있는 동안에도, 바빌로니아는 특권적인 지위를 누렸고, 계속해서 영향력을 확대하여 나갔다. <a href="/wiki/%EC%95%84%EC%8B%9C%EB%A6%AC%EC%95%84" title="아시리아">아시리아</a>는 때로는 더 많은 특권을 부여하고, 때로는 군사적으로 제압하면서 지배력을 유지하였으나, 결국은 <a class="mw-redirect" href="/wiki/%EA%B8%B0%EC%9B%90%EC%A0%84_627%EB%85%84" title="기원전 627년">기원전 627년</a>, <a href="/wiki/%EC%95%84%EC%8B%9C%EB%A6%AC%EC%95%84" title="아시리아">아시리아</a>의 마지막 통치자인 <a href="/wiki/%EC%95%84%EC%8A%88%EB%A5%B4%EB%B0%94%EB%8B%88%ED%8C%94" title="아슈르바니팔">아슈르바니팔</a>(Ashurbanipal)의 죽고 다음 해에 바빌로니아의 <a href="/wiki/%EC%B9%BC%EB%8D%B0%EC%95%84" title="칼데아">칼데아</a>인 나보폴라사르(Nabopolassar)가 반란을 일으킨다.</p>
<p><a class="mw-redirect" href="/wiki/%EA%B8%B0%EC%9B%90%EC%A0%84_612%EB%85%84" title="기원전 612년">기원전 612년</a>에 <a href="/wiki/%EB%A9%94%EB%94%94%EC%95%84" title="메디아">메디아</a>(Medes)인과 함께, <a href="/wiki/%EB%8B%88%EB%84%A4%EB%B2%A0" title="니네베">니네베</a>를 파괴하고 제국의 패권은 바빌로니아로 돌아온다.</p>
<div class="floatright"><a class="image" href="/wiki/%ED%8C%8C%EC%9D%BC:Median_Empire.jpg"><img alt="Median Empire.jpg" data-file-height="735" data-file-width="1306" height="225" src="//upload.wikimedia.org/wikipedia/commons/thumb/c/c6/Median_Empire.jpg/400px-Median_Empire.jpg" srcset="//upload.wikimedia.org/wikipedia/commons/thumb/c/c6/Median_Empire.jpg/600px-Median_Empire.jpg 1.5x, //upload.wikimedia.org/wikipedia/commons/thumb/c/c6/Median_Empire.jpg/800px-Median_Empire.jpg 2x" width="400"/></a></div>
<p>나보폴라사르(Nabopolassar)의 뒤를 이은 <a href="/wiki/%EB%84%A4%EB%B6%80%EC%B9%B4%EB%93%9C%EB%84%A4%EC%9E%90%EB%A5%B4_2%EC%84%B8" title="네부카드네자르 2세">네부카드네자르 2세</a>는 <a class="mw-redirect" href="/wiki/%EA%B8%B0%EC%9B%90%EC%A0%84_585%EB%85%84" title="기원전 585년">기원전 585년</a>에 <a href="/wiki/%ED%8E%98%EB%8B%88%ED%82%A4%EC%95%84" title="페니키아">페니키아</a>를 정복하는 등, 다시 한 번 <a href="/wiki/%EB%B0%94%EB%B9%8C%EB%A1%A0" title="바빌론">바빌론</a>을 문명의 중심으로 만들었다. <sup class="reference" id="cite_ref-1"><a href="#cite_note-1">[1]</a></sup> 그의 연대기는 <a class="mw-redirect" href="/wiki/%EA%B8%B0%EC%9B%90%EC%A0%84_567%EB%85%84" title="기원전 567년">기원전 567년</a> 이집트 공략 등 일부만이 발견되었다.</p>
<p>바빌로니아의 마지막 왕, <a href="/wiki/%EB%82%98%EB%B3%B4%EB%8B%88%EB%91%90%EC%8A%A4" title="나보니두스">나보니두스</a>(<i>Nabu-na'id</i>)와, <a class="mw-redirect" href="/wiki/%ED%82%A4%EB%A1%9C%EC%8A%A4" title="키로스">키로스</a>(Cyrus)에 의한 바빌로니아 정복에 대해서는 많은 사료가 존재한다. 이것은 주로 나보니두스의 연대기를 기록한 석판과, <i>하란</i>의 달의 신전 복원에 관련된 석판, 키루스의 선전포고 등이다.</p>
<p><br/>
기원전 549년, 나보니두스 6년에 아케메네스 왕조의 안샨의 왕이라고 불리었던 엘람의 키루스는 그의 영주이자 만다의 왕이었던 메데스의 아스티아게스에게 반기를 들었다. <a class="new" href="/w/index.php?title=%EC%95%84%EC%8A%A4%ED%8B%B0%EC%95%84%EA%B2%8C%EC%8A%A4&amp;action=edit&amp;redlink=1" title="아스티아게스 (없는 문서)">아스티아게스</a>의 군대는 왕을 배신하였고, 키루스가 엑바타나(Ecbatana)에서 왕을 자처하면서 메데스의 왕국은 종말을 맞이하게 된다.</p>
<p>3년 뒤 키루스는 페르시아의 왕이 되고, <a href="/wiki/%EB%A9%94%EC%86%8C%ED%8F%AC%ED%83%80%EB%AF%B8%EC%95%84" title="메소포타미아">메소포타미아</a> 북부로 원정한다. 이 때, 나보니두스는 그의 왕국의 국경 남쪽 근처의 사막에 진을 치고, 그의 아들 <a class="new" href="/w/index.php?title=%EB%B2%A8%EC%83%A4%EC%9E%90%EB%A5%B4&amp;action=edit&amp;redlink=1" title="벨샤자르 (없는 문서)">벨샤자르</a>(<i>Belshazzar</i> 또는 <i>Belsharutsur</i>)에게 군권을 맡겼다.</p>
<p><a href="/wiki/%EA%B8%B0%EC%9B%90%EC%A0%84_539%EB%85%84" title="기원전 539년">기원전 539년</a> <a href="/wiki/%ED%82%A4%EB%A3%A8%EC%8A%A4_2%EC%84%B8" title="키루스 2세">키루스 2세</a>는 바빌로니아를 공력한다. 전쟁은 오피스(Optis)에서 6월에 시작되었고, 바빌로니아는 곧 패하고, 뒤어서 시파라(Sippara)가 항복한다.</p>
<p>나보니두스는 <a href="/wiki/%EB%B0%94%EB%B9%8C%EB%A1%A0" title="바빌론">바빌론</a>으로 도망쳤으나, 고브리야스(Gobryas)에 계속해서 쫓기어 시파라(Sippara)가 항복한 지 이틀 뒤에, "키루스의 병사들이 싸우지 않고 <a href="/wiki/%EB%B0%94%EB%B9%8C%EB%A1%A0" title="바빌론">바빌론</a>으로 입성"하게 된다. 나보니두스는 은신처에서 나와 왕위를 유지하였다.</p>
<p>키루스는 10월(<i>Marchesvan</i>) 3일까지 도착하지 않았고, 고브리야스(Gobryas)가 그를 대신하다가, <a href="/wiki/%EB%B0%94%EB%B9%8C%EB%A1%A0" title="바빌론">바빌론</a>의 총독이 되었다. 며칠 뒤 나보니두스의 아들이 죽었을 때 6일 동안의 장례식이 치러졌고 키루스의 아들인 <a href="/wiki/%EC%BA%84%EB%B9%84%EC%84%B8%EC%8A%A4_2%EC%84%B8" title="캄비세스 2세">캄비세스 2세</a>(Cambyses II)가 무덤까지 그를 따랐다.</p>
<p>키루스의 바빌로니아 공격은 의심할 여지 없이 내부의 분열과 <a href="/wiki/%EC%9C%A0%EB%8C%80%EC%9D%B8" title="유대인">유대인</a>과 같이 억류되었던 외부 세력에 의해 쉽게 이루어졌다. 키루스가 처음으로 한 일은, 이들이 본국으로 돌아갈 수 있도록 하였다. 이러한 조치는 그의 선언문을 통해 명문화되었는데, 거기서 그는 바빌로니아 왕으로서 권위를 인정받기 원했다.</p>
<p>키루스가 죽기 1년 전인 <a href="/wiki/%EA%B8%B0%EC%9B%90%EC%A0%84_529%EB%85%84" title="기원전 529년">기원전 529년</a>, 그는 그의 아들 <a href="/wiki/%EC%BA%84%EB%B9%84%EC%84%B8%EC%8A%A4_2%EC%84%B8" title="캄비세스 2세">캄비세스 2세</a>(Cambyses)에게 왕위를 넘겼다.</p>
<p>이후 <a href="/wiki/%EB%8B%A4%EB%A6%AC%EC%9A%B0%EC%8A%A4_1%EC%84%B8" title="다리우스 1세">다리우스 1세</a>(Darius Hystaspis)가 페르시아 왕권을 장악하고, 페르시아를 <a class="mw-redirect" href="/wiki/%EC%A1%B0%EB%A1%9C%EC%95%84%EC%8A%A4%ED%84%B0" title="조로아스터">조로아스터</a>의 대리인으로 다스리게 될 때까지, <a href="/wiki/%EB%B0%94%EB%B9%8C%EB%A1%A0" title="바빌론">바빌론</a>은 서아시아의 지배자로서 권위를 인정받게 된다. 다리우스는 정복자로서 바빌론에 입성하였다.</p>
<p>다리우스가 죽은 뒤, 바빌로니아는 니딘타-벨 왕 (<a class="new" href="/w/index.php?title=%EB%84%A4%EB%B6%80%EC%B9%B4%EB%93%9C%EB%84%A4%EC%9E%90%EB%A5%B4_3%EC%84%B8&amp;action=edit&amp;redlink=1" title="네부카드네자르 3세 (없는 문서)">네부카드네자르 3세</a>) 때에 잠시 동안 독립을 회복했다가(<a href="/wiki/%EA%B8%B0%EC%9B%90%EC%A0%84_521%EB%85%84" title="기원전 521년">기원전 521년</a> 10월 ~ <a href="/wiki/%EA%B8%B0%EC%9B%90%EC%A0%84_520%EB%85%84" title="기원전 520년">520년</a> 8월) 페르시아에 공격으로 다시 점령된다.</p>
<p>몇 년 뒤(<a href="/wiki/%EA%B8%B0%EC%9B%90%EC%A0%84_514%EB%85%84" title="기원전 514년">기원전 514년</a>추정) 바빌론의 <a class="new" href="/w/index.php?title=%EC%95%84%EB%9D%BC%EC%B9%B4&amp;action=edit&amp;redlink=1" title="아라카 (없는 문서)">아라카</a>(Arakha)가 다시 봉기하지만, 이때 방벽 일부가 파괴된다. 벨의 거대 사원인 <a class="new" href="/w/index.php?title=%EC%97%90-%EC%82%AC%EA%B8%B8%EB%9D%BC&amp;action=edit&amp;redlink=1" title="에-사길라 (없는 문서)">에-사길라</a>는 계속해서 보수되면서 바빌로니아의 상징으로 유지되지만, 마침내 바빌로니아의 새 수도 <a class="mw-redirect" href="/wiki/%EC%85%80%EB%A0%88%EC%9A%B0%ED%82%A4%EC%95%84" title="셀레우키아">셀레우키아</a>가 건설되면서 많은 인구들이 이동하게 되고, 바빌론 성은 석재로 이용되기 위해 손상되고 말았다.</p>
