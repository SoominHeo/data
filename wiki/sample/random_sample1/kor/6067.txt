<h2><span class="mw-headline" id=".EC.82.B6">삶</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%B0%94%EC%88%98%EB%B0%98%EB%91%90&amp;action=edit&amp;section=2" title="부분 편집: 삶">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p><a href="/wiki/%EA%B0%84%EB%8B%A4%EB%9D%BC" title="간다라">간다라</a> 지방 푸루샤푸라성의 <a class="mw-disambig" href="/wiki/%EB%B8%8C%EB%9D%BC%EB%A7%8C" title="브라만">브라만</a> 가문에서 차남으로 태어났다. 맏형은 그와 같이 사상가로 유명한 <a href="/wiki/%EC%95%84%EC%83%81%EA%B0%80" title="아상가">아상가</a>(무착, 無着)이다. 출가하여 처음에는 <a href="/wiki/%EC%9C%A0%EB%B6%80" title="유부">유부</a>(有部)에 속하였으며, <a href="/wiki/%EC%B9%B4%EC%8A%88%EB%AF%B8%EB%A5%B4" title="카슈미르">카슈미르</a>와 <a href="/wiki/%EA%B0%84%EB%8B%A4%EB%9D%BC" title="간다라">간다라</a>에서 <a href="/wiki/%EC%9C%A0%EB%B6%80" title="유부">유부</a>를 비롯한 여러 <a class="mw-redirect" href="/wiki/%EB%B6%80%ED%8C%8C_%EB%B6%88%EA%B5%90" title="부파 불교">부파 불교</a>의 학설을 수업하고 이들 학설의 요강서(要綱書)인 《<a class="mw-redirect" href="/wiki/%EA%B5%AC%EC%82%AC%EB%A1%A0" title="구사론">구사론</a>(俱舍論)》을 지었다.<sup class="reference" id="cite_ref-.EA.B8.80.EB.A1.9C.EB.B2.8C-.EC.84.B8.EC.B9.9C_2-1"><a href="#cite_note-.EA.B8.80.EB.A1.9C.EB.B2.8C-.EC.84.B8.EC.B9.9C-2">[2]</a></sup></p>
<p>그러나, 후일 형인 아상가(무착)의 감화로 <a href="/wiki/%EB%8C%80%EC%8A%B9%EB%B6%88%EA%B5%90" title="대승불교">대승</a>으로 전향하여 아상가 및 그의 스승 <a class="mw-redirect" href="/wiki/%EB%A7%88%EC%9D%B4%ED%8A%B8%EB%A0%88%EC%95%BC" title="마이트레야">마이트레야</a>의 저서에 주석을 붙여 <a class="mw-redirect" href="/wiki/%EC%9C%A0%EA%B0%80%EC%9C%A0%EC%8B%9D%EC%84%A4" title="유가유식설">유가유식설</a>(瑜伽唯識說)의 완성에 힘썼고, 《<a href="/wiki/%EC%9C%A0%EC%8B%9D%EC%82%BC%EC%8B%AD%EC%86%A1" title="유식삼십송">유식삼십송</a>(唯識三十頌)》을 저술하였다. 더욱이 그 입장에 서서 반대설을 깨뜨리고 《<a class="new" href="/w/index.php?title=%EC%9C%A0%EC%8B%9D%EC%9D%B4%EC%8B%AD%EB%A1%A0&amp;action=edit&amp;redlink=1" title="유식이십론 (없는 문서)">유식이십론</a>(唯識二十論)》을, 또한 <a href="/wiki/%EC%9C%A0%EC%8B%9D%EC%84%A4" title="유식설">유식설</a> 입문서로서 《<a class="new" href="/w/index.php?title=%EB%8C%80%EC%8A%B9%EB%B0%B1%EB%B2%95%EB%AA%85%EB%AC%B8%EB%A1%A0&amp;action=edit&amp;redlink=1" title="대승백법명문론 (없는 문서)">대승백법명문론</a>(大乘百法明門論)》을 지었다.<sup class="reference" id="cite_ref-.EA.B8.80.EB.A1.9C.EB.B2.8C-.EC.84.B8.EC.B9.9C_2-2"><a href="#cite_note-.EA.B8.80.EB.A1.9C.EB.B2.8C-.EC.84.B8.EC.B9.9C-2">[2]</a></sup></p>
<p>《<a href="/wiki/%EC%9C%A0%EC%8B%9D%EC%82%BC%EC%8B%AD%EC%86%A1" title="유식삼십송">유식삼십송</a>》에 대해서는 그의 제자들의 손으로 여러 주석서가 나왔고, 후일 <a href="/wiki/%ED%98%84%EC%9E%A5" title="현장">현장</a>(玄奬)이 <a href="/wiki/%ED%98%B8%EB%B2%95" title="호법">호법</a>(護法)의 주석을 중심으로 <a class="mw-redirect" href="/wiki/10%EB%8C%80%EB%85%BC%EC%82%AC" title="10대논사">10대논사</a>(十大論師)의 여러 주석을 합쳐서 번역한 《<a href="/wiki/%EC%84%B1%EC%9C%A0%EC%8B%9D%EB%A1%A0" title="성유식론">성유식론</a>(成唯識論)》은 <a class="mw-redirect" href="/wiki/%EC%A4%91%EA%B5%AD_%EB%B6%88%EA%B5%90" title="중국 불교">중국 불교</a>의 <a href="/wiki/%EB%B2%95%EC%83%81%EC%A2%85" title="법상종">법상종</a>(法相宗)의 근본경전이 되었다.<sup class="reference" id="cite_ref-.EA.B8.80.EB.A1.9C.EB.B2.8C-.EC.84.B8.EC.B9.9C_2-3"><a href="#cite_note-.EA.B8.80.EB.A1.9C.EB.B2.8C-.EC.84.B8.EC.B9.9C-2">[2]</a></sup></p>
