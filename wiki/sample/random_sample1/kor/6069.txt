<p><b>바스노르망디</b>(<span style="font-size: smaller;"><a href="/wiki/%ED%94%84%EB%9E%91%EC%8A%A4%EC%96%B4" title="프랑스어">프랑스어</a>:</span> <span lang="fr" xml:lang="fr">Basse-Normandie</span>)는 <a href="/wiki/%ED%94%84%EB%9E%91%EC%8A%A4" title="프랑스">프랑스</a> 북부에 위치한 <a href="/wiki/%ED%94%84%EB%9E%91%EC%8A%A4%EC%9D%98_%EB%A0%88%EC%A7%80%EC%98%B9" title="프랑스의 레지옹">레지옹</a>으로 중심 도시는 <a href="/wiki/%EC%BA%89" title="캉">캉</a>이며 면적은 17,589㎢, 인구는 1,453,000명(2007년 기준)이다. <a href="/wiki/%EC%BD%94%ED%83%95%ED%83%B1_%EB%B0%98%EB%8F%84" title="코탕탱 반도">코탕탱 반도</a>에 위치한다.</p>
<p>동쪽으로는 <a href="/wiki/%EC%98%A4%ED%8A%B8%EB%85%B8%EB%A5%B4%EB%A7%9D%EB%94%94" title="오트노르망디">오트노르망디</a>, 남동쪽으로는 <a href="/wiki/%EC%83%81%ED%8A%B8%EB%A5%B4%EB%B0%9C%EB%93%9C%EB%A3%A8%EC%95%84%EB%A5%B4" title="상트르발드루아르">상트르발드루아르</a>, 남쪽으로는 <a href="/wiki/%ED%8E%98%EC%9D%B4%EB%93%9C%EB%9D%BC%EB%A3%A8%EC%95%84%EB%A5%B4" title="페이드라루아르">페이드라루아르</a>, 남서쪽으로는 <a href="/wiki/%EB%B8%8C%EB%A5%B4%ED%83%80%EB%89%B4" title="브르타뉴">브르타뉴</a>, 북쪽과 서쪽으로는 <a href="/wiki/%EC%98%81%EA%B5%AD_%ED%95%B4%ED%98%91" title="영국 해협">영국 해협</a>과 접한다. 3개 주(<a href="/wiki/%EB%A7%9D%EC%8A%88_%EC%A3%BC" title="망슈 주">망슈 주</a>, <a href="/wiki/%EC%98%A4%EB%A5%B8_%EC%A3%BC" title="오른 주">오른 주</a>, <a href="/wiki/%EC%B9%BC%EB%B0%94%EB%8F%84%EC%8A%A4_%EC%A3%BC" title="칼바도스 주">칼바도스 주</a>)를 관할한다.</p>
<p>1956년 <a href="/wiki/%EB%85%B8%EB%A5%B4%EB%A7%9D%EB%94%94" title="노르망디">노르망디</a>를 바스노르망디와 오트노르망디로 나누면서 신설되었다. 2016년 1월 1일을 기해 시행된 레지옹 개편에 따라 바스노르망디와 오트노르망디가 <a href="/wiki/%EB%85%B8%EB%A5%B4%EB%A7%9D%EB%94%94_(%EB%A0%88%EC%A7%80%EC%98%B9)" title="노르망디 (레지옹)">노르망디</a>로 합병되었다.</p>
