<p><b>바실리 이바노비치 알렉세예프</b> (<span style="font-size: smaller;"><a href="/wiki/%EB%9F%AC%EC%8B%9C%EC%95%84%EC%96%B4" title="러시아어">러시아어</a>:</span> <span lang="ru" xml:lang="ru">Василий Иванович Алексеев</span>, <a href="/wiki/1942%EB%85%84" title="1942년">1942년</a> <a href="/wiki/1%EC%9B%94_7%EC%9D%BC" title="1월 7일">1월 7일</a> ~ <a href="/wiki/2011%EB%85%84" title="2011년">2011년</a> <a href="/wiki/11%EC%9B%94_25%EC%9D%BC" title="11월 25일">11월 25일</a>)는 전 <a class="mw-redirect" href="/wiki/%EC%86%8C%EB%B9%84%EC%97%90%ED%8A%B8_%EC%97%B0%EB%B0%A9" title="소비에트 연방">소비에트 연방</a>의 <a href="/wiki/%EC%97%AD%EB%8F%84" title="역도">역도</a> 선수이다. 80개의 세계 기록과 81개의 국내 기록을 세운 그는 <a href="/wiki/1972%EB%85%84" title="1972년">1972년</a>과 <a href="/wiki/1976%EB%85%84" title="1976년">1976년</a> <a href="/wiki/%ED%95%98%EA%B3%84_%EC%98%AC%EB%A6%BC%ED%94%BD" title="하계 올림픽">하계 올림픽</a>에서 110kg급 2연승을 하였다.</p>
<h2><span class="mw-headline" id=".EC.83.9D.EC.95.A0">생애</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%B0%94%EC%8B%A4%EB%A6%AC_%EC%95%8C%EB%A0%89%EC%84%B8%EC%98%88%ED%94%84&amp;action=edit&amp;section=1" title="부분 편집: 생애">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p><a href="/wiki/%EB%9E%B4%EC%9E%94_%EC%A3%BC" title="랴잔 주">랴잔 주</a> 포크로보시슈니코 마을에서 태어난 알렉세예프는 18세 때 역도 연습을 시작하였다.</p>
<p><a href="/wiki/1970%EB%85%84" title="1970년">1970년</a> <a href="/wiki/1%EC%9B%94" title="1월">1월</a> 자신의 첫 세계 기록을 세우고 그해부터 <a href="/wiki/1977%EB%85%84" title="1977년">1977년</a> 사이에 세계 선수권과 유럽 선수권에서 단 한번도 패한 적이 없는 그는 3개의 이벤트에서 600kg이 넘는 첫 선수였다.</p>
<p><a href="/wiki/%EB%AE%8C%ED%97%A8" title="뮌헨">뮌헨</a>과 <a href="/wiki/%EB%AA%AC%ED%8A%B8%EB%A6%AC%EC%98%AC" title="몬트리올">몬트리올</a> 올림픽 2연패 후, <a href="/wiki/1980%EB%85%84_%ED%95%98%EA%B3%84_%EC%98%AC%EB%A6%BC%ED%94%BD" title="1980년 하계 올림픽">1980년 하계 올림픽</a>에서 3연패를 향하였지만 그 경기에서 그의 상연은 실망적이었다. 붙잡을 때 자신의 역기를 너무 높이 고정시켜 들 수가 없게 되자 결과로 0kg 득점이 되고 말았다. 그 후에 정식으로 은퇴하였다.</p>
<p><a href="/wiki/1987%EB%85%84" title="1987년">1987년</a> 인민의 대리 의회를 위한 랴잔 주 대표로 뽑혔다. <a href="/wiki/1990%EB%85%84" title="1990년">1990년</a>부터 <a href="/wiki/1992%EB%85%84" title="1992년">1992년</a> 사이에 코치로 일하였으며, <a href="/wiki/1992%EB%85%84_%ED%95%98%EA%B3%84_%EC%98%AC%EB%A6%BC%ED%94%BD" title="1992년 하계 올림픽">바르셀로나 올림픽</a>에서 그의 지도 아래 <a class="mw-redirect" href="/wiki/%EB%8F%85%EB%A6%BD_%EA%B5%AD%EA%B0%80_%EC%97%B0%ED%95%A9" title="독립 국가 연합">독립 국가 연합</a> 팀이 역도에서 5개의 <a href="/wiki/%EA%B8%88%EB%A9%94%EB%8B%AC" title="금메달">금메달</a>을 포함하여 10개의 메달을 획득하였다.</p>
<h2><span class="mw-headline" id=".EC.88.98.EC.83.81.EA.B3.BC_.EB.AA.85.EC.98.88">수상과 명예</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%B0%94%EC%8B%A4%EB%A6%AC_%EC%95%8C%EB%A0%89%EC%84%B8%EC%98%88%ED%94%84&amp;action=edit&amp;section=2" title="부분 편집: 수상과 명예">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p><a href="/wiki/1999%EB%85%84" title="1999년">1999년</a> <a href="/wiki/%EA%B7%B8%EB%A6%AC%EC%8A%A4" title="그리스">그리스</a>에서 알렉세예프는 20세기의 최조 스포츠맨으로 인정되었다. 레닌 훈장(1972), 인민의 우호 훈장, 명예의 배지 훈장(1970), 노동의 붉은 깃발 훈장(1972)이 수여되었다. <a href="/wiki/1993%EB%85%84" title="1993년">1993년</a>에는 <a href="/wiki/%EA%B5%AD%EC%A0%9C_%EC%97%AD%EB%8F%84_%EC%97%B0%EB%A7%B9" title="국제 역도 연맹">국제 역도 연맹</a> 명예의 전당에 헌액되었다.</p>
<p><a href="/wiki/1975%EB%85%84" title="1975년">1975년</a> <a href="/wiki/4%EC%9B%94_14%EC%9D%BC" title="4월 14일">4월 14일</a> <a href="/wiki/%EC%8A%A4%ED%8F%AC%EC%B8%A0_%EC%9D%BC%EB%9F%AC%EC%8A%A4%ED%8A%B8%EB%A0%88%EC%9D%B4%ED%8B%B0%EB%93%9C" title="스포츠 일러스트레이티드">스포츠 일러스트레이티드</a>의 표지에 "세계에서 가장 힘센 남자" 타이틀 아래 나오기도 하였다.</p>
<h2><span class="mw-headline" id=".EC.82.AC.EB.A7.9D">사망</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%B0%94%EC%8B%A4%EB%A6%AC_%EC%95%8C%EB%A0%89%EC%84%B8%EC%98%88%ED%94%84&amp;action=edit&amp;section=3" title="부분 편집: 사망">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p><a href="/wiki/2011%EB%85%84" title="2011년">2011년</a> <a href="/wiki/11%EC%9B%94_25%EC%9D%BC" title="11월 25일">11월 25일</a> 심한 <a href="/wiki/%EC%8B%AC%EC%9E%A5" title="심장">심장</a> 문제를 겪어오면서 65세의 나이로 <a href="/wiki/%EB%8F%85%EC%9D%BC" title="독일">독일</a>에서 사망하였다. 러시아 역도 연맹은 그의 사망을 보고하면서 그에게 "소비에트 스포츠의 전설"과 "세계에서 가장 힘센 사람들 중의 하나"로 불렀다.</p>
