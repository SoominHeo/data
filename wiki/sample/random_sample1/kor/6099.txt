<p><b>헴마</b>(Hemma), <b>바이에른의 엠므</b>(Emme of Bavaria, Emme von Bayern) 또는 <b>아르톨프의 엠므</b>(Emme of Artlof, <a href="/wiki/803%EB%85%84" title="803년">803년</a> - <a href="/wiki/876%EB%85%84" title="876년">876년</a> <a href="/wiki/1%EC%9B%94_31%EC%9D%BC" title="1월 31일">1월 31일</a>)는 <a href="/wiki/%ED%94%84%EB%9E%91%ED%81%AC_%EC%99%95%EA%B5%AD" title="프랑크 왕국">프랑크 왕국</a>의 귀족, <a class="mw-redirect" href="/wiki/%EB%8F%99%ED%94%84%EB%9E%91%ED%81%AC" title="동프랑크">동프랑크</a>와 <a class="mw-redirect" href="/wiki/%EB%B0%94%EC%9D%B4%EC%97%90%EB%A5%B8" title="바이에른">바이에른</a>의 왕비이며, <a href="/wiki/%EB%A3%A8%ED%8A%B8%EB%B9%84%ED%9E%88_2%EC%84%B8_(%EB%8F%99%ED%94%84%EB%9E%91%ED%81%AC)" title="루트비히 2세 (동프랑크)">루트비히 2세 드 게르만</a>의 정비이다. <a class="new" href="/w/index.php?title=%EC%95%84%EB%A5%B4%ED%86%A8%ED%94%84&amp;action=edit&amp;redlink=1" title="아르톨프 (없는 문서)">아르톨프</a>와 <a class="mw-redirect" href="/wiki/%EB%B0%94%EC%9D%B4%EC%97%90%EB%A5%B8" title="바이에른">바이에른</a>의 공작 <a href="/wiki/%EA%B5%AC%EC%97%98%ED%94%84_1%EC%84%B8" title="구엘프 1세">구엘프 1세</a>의 딸이다.</p>
<p><a class="new" href="/w/index.php?title=%EB%B2%A8%ED%94%84_%EA%B0%80&amp;action=edit&amp;redlink=1" title="벨프 가 (없는 문서)">벨프 가</a> 출신으로, <a href="/wiki/843%EB%85%84" title="843년">843년</a> <a class="mw-redirect" href="/wiki/%EB%B2%A0%EB%A5%B4%EB%8D%A9_%EC%A1%B0%EC%95%BD" title="베르덩 조약">베르덩 조약</a> 후 <a class="mw-redirect" href="/wiki/%EB%8F%99%ED%94%84%EB%9E%91%ED%81%AC" title="동프랑크">동프랑크</a>의 왕비가 되었다. <a class="mw-redirect" href="/wiki/%EC%B9%B4%EB%A5%BC%EB%A1%9C%EB%A7%8C_3%EC%84%B8" title="카를로만 3세">카를로만 3세</a>, <a href="/wiki/%EB%A3%A8%ED%8A%B8%EB%B9%84%ED%9E%88_3%EC%84%B8_(%EB%8F%99%ED%94%84%EB%9E%91%ED%81%AC)" title="루트비히 3세 (동프랑크)">루트비히 3세 청년</a>, <a href="/wiki/%EC%B9%B4%EB%A5%BC_3%EC%84%B8" title="카를 3세">뚱보왕 카를</a>의 모후이며 <a href="/wiki/%EB%A3%A8%ED%8A%B8%EB%B9%84%ED%9E%88_1%EC%84%B8_(%ED%94%84%EB%9E%91%ED%81%AC)" title="루트비히 1세 (프랑크)">경건왕 루트비히</a>의 후처가 된 <a class="mw-disambig" href="/wiki/%EB%B0%94%EC%9D%B4%EC%97%90%EB%A5%B8%EC%9D%98_%EC%9C%A0%EB%94%94%ED%8A%B8" title="바이에른의 유디트">바이에른의 유디트</a>의 친 동생이다.</p>

<h2><span class="mw-headline" id=".EC.83.9D.EC.95.A0">생애</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%B0%94%EC%9D%B4%EC%97%90%EB%A5%B8%EC%9D%98_%EC%97%A0%EB%AF%80&amp;action=edit&amp;section=1" title="부분 편집: 생애">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<h3><span class="mw-headline" id=".EC.B4.88.EA.B8.B0_.EC.82.B6">초기 삶</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%B0%94%EC%9D%B4%EC%97%90%EB%A5%B8%EC%9D%98_%EC%97%A0%EB%AF%80&amp;action=edit&amp;section=2" title="부분 편집: 초기 삶">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>엠므는 <a class="new" href="/w/index.php?title=%EB%B2%A8%ED%94%84_%EA%B0%80%EB%AC%B8&amp;action=edit&amp;redlink=1" title="벨프 가문 (없는 문서)">벨프 가문</a> 출신으로 <a href="/wiki/803%EB%85%84" title="803년">803년</a>경 <a class="new" href="/w/index.php?title=%EC%95%84%EB%A5%B4%ED%86%A8%ED%94%84&amp;action=edit&amp;redlink=1" title="아르톨프 (없는 문서)">아르톨프</a>와 <a class="mw-redirect" href="/wiki/%EB%B0%94%EC%9D%B4%EC%97%90%EB%A5%B8" title="바이에른">바이에른</a>의 공작이자 <a href="/wiki/%EB%A0%88%EA%B2%90%EC%8A%A4%EB%B6%80%EB%A5%B4%ED%81%AC" title="레겐스부르크">레겐스부르크</a>의 영주 <a href="/wiki/%EA%B5%AC%EC%97%98%ED%94%84_1%EC%84%B8" title="구엘프 1세">구엘프 1세</a>와 <a href="/wiki/%ED%9E%88%EC%8A%A4%ED%8C%8C%EB%8B%88%EC%95%84" title="히스파니아">히스파니아</a> 출신 헤드비가의 딸로 태어났다. <a href="/wiki/818%EB%85%84" title="818년">818년</a> <a href="/wiki/%EB%A3%A8%ED%8A%B8%EB%B9%84%ED%9E%88_1%EC%84%B8_(%ED%94%84%EB%9E%91%ED%81%AC)" title="루트비히 1세 (프랑크)">경건왕 루트비히</a>의 후처가 된 <a class="mw-disambig" href="/wiki/%EB%B0%94%EC%9D%B4%EC%97%90%EB%A5%B8%EC%9D%98_%EC%9C%A0%EB%94%94%ED%8A%B8" title="바이에른의 유디트">바이에른의 유디트</a>의 동생으로, <a href="/wiki/820%EB%85%84" title="820년">820년</a> 혹은 <a href="/wiki/827%EB%85%84" title="827년">827년</a> <a href="/wiki/%EB%A3%A8%ED%8A%B8%EB%B9%84%ED%9E%88_1%EC%84%B8_(%ED%94%84%EB%9E%91%ED%81%AC)" title="루트비히 1세 (프랑크)">경건왕 루트비히</a>와 본처 이르멘가르트의 셋째 아들인 <a class="mw-redirect" href="/wiki/%EB%B0%94%EC%9D%B4%EC%97%90%EB%A5%B8" title="바이에른">바이에른</a> <a class="mw-redirect" href="/wiki/%EA%B5%AD%EC%99%95" title="국왕">국왕</a> <a href="/wiki/%EB%A3%A8%ED%8A%B8%EB%B9%84%ED%9E%88_2%EC%84%B8_(%EB%8F%99%ED%94%84%EB%9E%91%ED%81%AC)" title="루트비히 2세 (동프랑크)">루트비히 2세</a>와 <a href="/wiki/%EB%A0%88%EA%B2%90%EC%8A%A4%EB%B6%80%EB%A5%B4%ED%81%AC" title="레겐스부르크">레겐스부르크</a>에서 결혼하였다.</p>
<h3><span class="mw-headline" id=".EC.99.95.EB.B9.84_.EC.8B.9C.EC.A0.88">왕비 시절</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%B0%94%EC%9D%B4%EC%97%90%EB%A5%B8%EC%9D%98_%EC%97%A0%EB%AF%80&amp;action=edit&amp;section=3" title="부분 편집: 왕비 시절">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p><a href="/wiki/%EB%A3%A8%ED%8A%B8%EB%B9%84%ED%9E%88_2%EC%84%B8_(%EB%8F%99%ED%94%84%EB%9E%91%ED%81%AC)" title="루트비히 2세 (동프랑크)">루트비히 2세</a>와의 사이에서 <a class="mw-redirect" href="/wiki/%EC%B9%B4%EB%A5%BC%EB%A1%9C%EB%A7%8C_3%EC%84%B8" title="카를로만 3세">카를로만 3세</a>, <a href="/wiki/%EB%A3%A8%ED%8A%B8%EB%B9%84%ED%9E%88_3%EC%84%B8_(%EB%8F%99%ED%94%84%EB%9E%91%ED%81%AC)" title="루트비히 3세 (동프랑크)">루트비히 3세 청년</a>, <a href="/wiki/%EC%B9%B4%EB%A5%BC_3%EC%84%B8" title="카를 3세">뚱보왕 카를</a>과 딸 신성로마 황후 이르멘가르트(<a class="mw-redirect" href="/wiki/%EC%9D%B4%ED%83%88%EB%A6%AC%EC%95%84%EC%9D%98_%EB%A3%A8%ED%8A%B8%EB%B9%84%ED%9E%88_2%EC%84%B8" title="이탈리아의 루트비히 2세">이탈리아의 루트비히 2세</a>의 황후), 기셀라, 베르타, 힐데가르트, 엠므 등을 두었다. 이르멘가르트는 <a href="/wiki/%EB%A1%9C%ED%83%80%EB%A5%B4_1%EC%84%B8" title="로타르 1세">로타르 1세</a>의 아들인 <a href="/wiki/%EB%A3%A8%ED%8A%B8%EB%B9%84%ED%9E%88_2%EC%84%B8_(%EC%9D%B4%ED%83%88%EB%A6%AC%EC%95%84)" title="루트비히 2세 (이탈리아)">황제 루트비히 2세</a>의 황후가 되었고, 기셀라는 슈바벤의 그라파 베르크톨드와 결혼했는데, 후일 <a href="/wiki/%EC%BD%98%EB%9D%BC%ED%8A%B8_1%EC%84%B8" title="콘라트 1세">콘라트 1세</a>의 왕비가 되는 <a href="/wiki/%EC%8A%88%EB%B0%94%EB%B2%A4%EC%9D%98_%EC%BF%A0%EB%8B%88%EA%B5%B0%EB%8D%B0" title="슈바벤의 쿠니군데">슈바벤의 쿠니군데</a>의 외할머니가 된다.</p>
<p>한 전설에 의하면 그녀는 훌륭한 자질을 가졌다 한다. 루트비히를 축출하려 음모를 꾸미던 <a href="/wiki/%EB%B2%A0%EB%84%A4%EB%B2%A4%ED%86%A0" title="베네벤토">베네벤토</a>의 군주 아델히스(Adelchis)가 루이에 대해 도전을 시도했다가 그녀가 남편 루트비히로 가장하여 그를 찾아가는 바람에 아델히스는 코르시카로 도망쳤다 한다. 한편 아델히스가 그녀에게 농락당한 일을 <a href="/wiki/%EC%9D%B4%ED%83%88%EB%A6%AC%EC%95%84" title="이탈리아">이탈리아</a>의 귀족들은 오래도록 불쾌히 여겼다 한다.</p>
<p>남편 <a href="/wiki/%EB%A3%A8%ED%8A%B8%EB%B9%84%ED%9E%88_2%EC%84%B8_(%EB%8F%99%ED%94%84%EB%9E%91%ED%81%AC)" title="루트비히 2세 (동프랑크)">게르만왕 루트비히 2세</a>는 다른 형제들과 함께 언니이자 시계모이기도 한 <a class="mw-disambig" href="/wiki/%EB%B0%94%EC%9D%B4%EC%97%90%EB%A5%B8%EC%9D%98_%EC%9C%A0%EB%94%94%ED%8A%B8" title="바이에른의 유디트">바이에른의 유디트</a>와 정치적으로 갈등하였으나 왕비인 엠므와의 관계는 원만하였다. <a href="/wiki/843%EB%85%84" title="843년">843년</a> <a class="mw-redirect" href="/wiki/%EB%B2%A0%EB%A5%B4%EB%8D%A9_%EC%A1%B0%EC%95%BD" title="베르덩 조약">베르덩 조약</a>으로 <a href="/wiki/%ED%94%84%EB%9E%91%ED%81%AC_%EC%99%95%EA%B5%AD" title="프랑크 왕국">프랑크 왕국</a>이 분할되고 남편 <a href="/wiki/%EB%A3%A8%ED%8A%B8%EB%B9%84%ED%9E%88_2%EC%84%B8_(%EB%8F%99%ED%94%84%EB%9E%91%ED%81%AC)" title="루트비히 2세 (동프랑크)">게르만왕 루트비히 2세</a>가 <a href="/wiki/%EB%8F%99%ED%94%84%EB%9E%91%ED%81%AC_%EC%99%95%EA%B5%AD" title="동프랑크 왕국">동프랑크 왕국</a>의 국왕이 되면서 왕비로 책봉되었다.</p>
<h3><span class="mw-headline" id=".EC.B5.9C.ED.9B.84">최후</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%B0%94%EC%9D%B4%EC%97%90%EB%A5%B8%EC%9D%98_%EC%97%A0%EB%AF%80&amp;action=edit&amp;section=4" title="부분 편집: 최후">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>876년 <a href="/wiki/1%EC%9B%94_31%EC%9D%BC" title="1월 31일">1월 31일</a>에 사망하였다. 시신은 <a href="/wiki/%EB%A0%88%EA%B2%90%EC%8A%A4%EB%B6%80%EB%A5%B4%ED%81%AC" title="레겐스부르크">레겐스부르크</a>(Regensburg)의 <a class="new" href="/w/index.php?title=%EC%84%B1_%EC%97%90%EB%A9%94%EB%9E%8C_%EC%88%98%EB%8F%84%EC%9B%90&amp;action=edit&amp;redlink=1" title="성 에메람 수도원 (없는 문서)">성 에메람 수도원</a>에 안치되었으며 남편 <a href="/wiki/%EB%A3%A8%ED%8A%B8%EB%B9%84%ED%9E%88_2%EC%84%B8_(%EB%8F%99%ED%94%84%EB%9E%91%ED%81%AC)" title="루트비히 2세 (동프랑크)">게르만왕 루트비히 2세</a>, 아들 <a href="/wiki/%EB%A3%A8%ED%8A%B8%EB%B9%84%ED%9E%88_3%EC%84%B8_(%EB%8F%99%ED%94%84%EB%9E%91%ED%81%AC)" title="루트비히 3세 (동프랑크)">루트비히 3세 청년</a>의 묘 근처에 안치되었다.</p>
