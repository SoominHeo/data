<p><b>바이킹 1호</b>(<span style="font-size: smaller;"><a href="/wiki/%EC%98%81%EC%96%B4" title="영어">영어</a>:</span> <span lang="en" xml:lang="en">Viking 1</span>)는 <a href="/wiki/%ED%99%94%EC%84%B1" title="화성">화성</a>의 생명체 존재 여부를 알아보기 위해 <a href="/wiki/%EB%AF%B8%EA%B5%AD_%ED%95%AD%EA%B3%B5%EC%9A%B0%EC%A3%BC%EA%B5%AD" title="미국 항공우주국">미국 항공우주국</a>(NASA)에서 발사한 <a href="/wiki/%ED%99%94%EC%84%B1" title="화성">화성</a> 탐사선이다. <a href="/wiki/1975%EB%85%84" title="1975년">1975년</a> <a href="/wiki/8%EC%9B%94_20%EC%9D%BC" title="8월 20일">8월 20일</a> 발사되어 <a href="/wiki/1976%EB%85%84" title="1976년">1976년</a> <a href="/wiki/6%EC%9B%94_19%EC%9D%BC" title="6월 19일">6월 19일</a> <a href="/wiki/%ED%99%94%EC%84%B1" title="화성">화성</a> 궤도에 진입하였다. 궤도선은 <a href="/wiki/1980%EB%85%84" title="1980년">1980년</a> <a href="/wiki/8%EC%9B%94_17%EC%9D%BC" title="8월 17일">8월 17일</a>까지, 착륙선은 <a href="/wiki/1982%EB%85%84" title="1982년">1982년</a> <a href="/wiki/11%EC%9B%94_13%EC%9D%BC" title="11월 13일">11월 13일</a>까지 임무를 수행하였다.</p>

<h2><span class="mw-headline" id=".EC.9E.84.EB.AC.B4">임무</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%B0%94%EC%9D%B4%ED%82%B9_1%ED%98%B8&amp;action=edit&amp;section=1" title="부분 편집: 임무">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>바이킹 1호는 <a href="/wiki/1975%EB%85%84" title="1975년">1975년</a> <a href="/wiki/8%EC%9B%94_20%EC%9D%BC" title="8월 20일">8월 20일</a>에 타이탄/센타우르 로켓으로 발사되어 10개월간의 비행 끝에 <a href="/wiki/%ED%99%94%EC%84%B1" title="화성">화성</a>에 도달해, 궤도에 들어가기 5일 정도 전부터 <a href="/wiki/%ED%99%94%EC%84%B1" title="화성">화성</a>의 전체 모습을 보여주기 시작했다. 바이킹 1호의 궤도선은 <a href="/wiki/1976%EB%85%84" title="1976년">1976년</a> <a href="/wiki/6%EC%9B%94_19%EC%9D%BC" title="6월 19일">6월 19일</a>에 <a href="/wiki/%ED%99%94%EC%84%B1" title="화성">화성</a> 궤도에 들어갔고, <a href="/wiki/6%EC%9B%94_21%EC%9D%BC" title="6월 21일">6월 21일</a>에 1513 x 33,000 km 24.66시간 주기의 궤도로 조정되었다. <a href="/wiki/%ED%99%94%EC%84%B1" title="화성">화성</a> 착륙은 <a href="/wiki/1976%EB%85%84" title="1976년">1976년</a> <a href="/wiki/7%EC%9B%94_4%EC%9D%BC" title="7월 4일">7월 4일</a>의 <a href="/wiki/%EB%AF%B8%EA%B5%AD" title="미국">미국</a> 독립 기념일로 예정되어 있었지만, 착륙 후보지의 기복이 큰 것이 촬영한 사진으로부터 발견되어, 안전한 장소가 발견될 때까지 연기되었다. 착륙선은 <a href="/wiki/1976%EB%85%84" title="1976년">1976년</a> <a href="/wiki/7%EC%9B%94_20%EC%9D%BC" title="7월 20일">7월 20일</a> 08:51 세계시에 궤도선으로부터 분리되어, 11:56:06 세계시에 착륙했다.</p>
<h3><span class="mw-headline" id=".EA.B6.A4.EB.8F.84.EC.84.A0">궤도선</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%B0%94%EC%9D%B4%ED%82%B9_1%ED%98%B8&amp;action=edit&amp;section=2" title="부분 편집: 궤도선">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>태양과 합이 시작된 <a href="/wiki/1976%EB%85%84" title="1976년">1976년</a> <a href="/wiki/11%EC%9B%94_5%EC%9D%BC" title="11월 5일">11월 5일</a>에 궤도선의 최초의 임무가 일시 종료되었다가, 합이 끝난 후인 <a href="/wiki/1976%EB%85%84" title="1976년">1976년</a> <a href="/wiki/12%EC%9B%94_14%EC%9D%BC" title="12월 14일">12월 14일</a>부터 재개되었다. <a href="/wiki/1977%EB%85%84" title="1977년">1977년</a> <a href="/wiki/2%EC%9B%94" title="2월">2월</a>에는 <a href="/wiki/%ED%8F%AC%EB%B3%B4%EC%8A%A4_(%EC%9C%84%EC%84%B1)" title="포보스 (위성)">포보스</a>에 접근이 행해졌다. <a href="/wiki/1977%EB%85%84" title="1977년">1977년</a> <a href="/wiki/3%EC%9B%94_11%EC%9D%BC" title="3월 11일">3월 11일</a>에는 근점이 300 km 가 되었다. 임무를 통해서 소규모의 궤도 조정이 행해져 <a href="/wiki/1979%EB%85%84" title="1979년">1979년</a> <a href="/wiki/7%EC%9B%94_20%EC%9D%BC" title="7월 20일">7월 20일</a>에는 근점이 357 km 까지 커졌다. 바이킹 1호의 궤도선의 고도 제어 가스가 적어져, <a href="/wiki/%ED%99%94%EC%84%B1" title="화성">화성</a>에 충돌과 오염이 <a href="/wiki/2019%EB%85%84" title="2019년">2019년</a>까지 일어나지 않도록 <a href="/wiki/1980%EB%85%84" title="1980년">1980년</a> <a href="/wiki/8%EC%9B%94_7%EC%9D%BC" title="8월 7일">8월 7일</a>에 357 x 33943 km 에서 320 x 56000 km 로 궤도를 올렸다. 궤도선은 <a href="/wiki/%ED%99%94%EC%84%B1" title="화성">화성</a>을 1485회 주회한 후, <a href="/wiki/1980%EB%85%84" title="1980년">1980년</a> <a href="/wiki/8%EC%9B%94_17%EC%9D%BC" title="8월 17일">8월 17일</a>에 작동을 종료했다.</p>
<h3><span class="mw-headline" id=".EC.B0.A9.EB.A5.99.EC.84.A0">착륙선</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%B0%94%EC%9D%B4%ED%82%B9_1%ED%98%B8&amp;action=edit&amp;section=3" title="부분 편집: 착륙선">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p><a href="/wiki/1976%EB%85%84" title="1976년">1976년</a> <a href="/wiki/7%EC%9B%94_20%EC%9D%BC" title="7월 20일">7월 20일</a> 08:51 세계시에, 착륙선이 궤도선에서 분리되었다. 분리가 시작되었을 때, 착륙선의 속도는 4 km/s 정도였다. 분리 한 뒤에 로켓을 분사하고, 착륙선의 궤도 이탈이 개시되었다. 수시간 후에 고도 약 300 km 에서 재돌입을 위해서 방향을 전환했다. 대기권의 통과에 수반해 증발성의 내열 보호물에 의한 방호 커버로 감속했고, 이에 병행해 돌입시의 과학 실험도 행해졌다. 고도 6 km , 속도 250 m/s 정도의 시점에서 직경 16 m 의 낙하산이 전개되었다. 7초 후에 방호 커버를 투기, 그로부터 8초 후에 3개의 착륙다리가 전개해, 낙하산에 의해 45초에 60 m/s 까지 감속했다. 고도 1.5 km 로 역추진 로켓이 점화되어 40초 후에 약 2.4 m/s 로 착륙할 때까지 분사되었다. 착륙 로켓에는 18개의 노즐이 있어, <a href="/wiki/%EC%88%98%EC%86%8C" title="수소">수소</a>와 <a href="/wiki/%EC%A7%88%EC%86%8C" title="질소">질소</a>의 배기를 확산시키는 설계가 되어 있었다. 이것은, 표면 온도의 상승을 섭씨 1도 이하로 억제해 표면 물질을 1 mm 이상 날아가지 않게 하기 위해서였다.</p>
<p>바이킹 1호의 착륙선은 크리세 평원(Gold Field)의 서부, 북위 22.697 도, 서경 48.222 도, 적도 반경이 3397.2 km 로 편평률이 0.0105인 기준 타원체로부터 표고 2.69 km 의 지점(행성면 좌표로 북위 22.480 도, 서경 47.967 도)에 11:53:06 세계시(화성 지방시로 16:13)에 착륙했다. 착륙시에는 약 22 kg 의 추진제가 남아 있었다.</p>
<p>착륙 25초 후부터 최초의 표면 영상이 송신되었다. 지진계를 꺼내지 못하고, 표본 채취기의 로크핀이 움직이지 않고 다리를 펼치는 데 5일간을 필요로 했지만, 그 이외에는 큰 문제 없이 명목상 모든 실험이 행해졌다. 바이킹 1호의 착륙선은, 바이킹 영상 팀의 리더를 기념해 <a href="/wiki/1982%EB%85%84" title="1982년">1982년</a> <a href="/wiki/1%EC%9B%94" title="1월">1월</a>에 토머스 매치 기념 기지로 명명되었다.</p>
<p><a href="/wiki/1982%EB%85%84" title="1982년">1982년</a> <a href="/wiki/11%EC%9B%94_13%EC%9D%BC" title="11월 13일">11월 13일</a>에 지상 관제소로부터 잘못된 명령을 보내 통신이 두절될 때까지, 착륙선은 2245 태양일간 운용되었다. 이 명령은, 착륙선의 열화된 배터리 능력을 개선하기 위해서 새로운 배터리 충전 소프트웨어를 업링크하려고 하고 있었지만, 부주의 때문에 안테나 지향 소프트웨어의 데이터 영역에 이중으로 입력되어 버렸다. 이후 4개월간 추정된 안테나 위치에 근거해 착륙선과의 통신이 시도되었지만, 실패로 끝났다.</p>
<p><a href="/wiki/2006%EB%85%84" title="2006년">2006년</a>에, <a href="/wiki/%ED%99%94%EC%84%B1_%EC%A0%95%EC%B0%B0%EC%9C%84%EC%84%B1" title="화성 정찰위성">화성 정찰위성</a>으로부터 <a href="/wiki/%ED%99%94%EC%84%B1" title="화성">화성</a> 표면의 바이킹 1호의 착륙선이 촬영되었다.</p>

<h2><span class="mw-headline" id=".EC.9D.BC.EB.B0.98.EC.83.81.EB.8C.80.EC.84.B1.EC.9D.B4.EB.A1.A0.EC.9D.98_.EC.8B.A4.ED.97.98">일반상대성이론의 실험</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%B0%94%EC%9D%B4%ED%82%B9_1%ED%98%B8&amp;action=edit&amp;section=4" title="부분 편집: 일반상대성이론의 실험">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p><a class="mw-redirect" href="/wiki/%EC%9D%BC%EB%B0%98%EC%83%81%EB%8C%80%EC%84%B1%EC%9D%B4%EB%A1%A0" title="일반상대성이론">일반상대성이론</a>에서는, '중력장에서의 시간의 지연'을 예측했다. 착륙선을 이용해 이 현상을 관측했다. <a href="/wiki/%ED%99%94%EC%84%B1" title="화성">화성</a>의 착륙선에 무선 신호를 보내, 착륙선에 신호를 돌려 보내도록 지시를 내렸다. 그 결과, 신호의 왕복 시간에는 '중력장에서의 시간의 지연'으로서 예상되는 시간의 엇갈림이 있는 것을 알 수 있었다.</p>
