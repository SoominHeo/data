<p><b>바쿠</b>(<span style="font-size: smaller;"><a href="/wiki/%EC%95%84%EC%A0%9C%EB%A5%B4%EB%B0%94%EC%9D%B4%EC%9E%94%EC%96%B4" title="아제르바이잔어">아제르바이잔어</a>:</span> <span lang="az" xml:lang="az">Bakı</span> <small>바크</small>, <span style="font-size: smaller;"><a href="/wiki/%EB%9F%AC%EC%8B%9C%EC%95%84%EC%96%B4" title="러시아어">러시아어</a>:</span> <span lang="ru" xml:lang="ru">Баку</span> <small>바쿠<sup>[<a href="/wiki/%EC%9C%84%ED%82%A4%EB%B0%B1%EA%B3%BC:%EB%9F%AC%EC%8B%9C%EC%95%84%EC%96%B4%EC%9D%98_%ED%95%9C%EA%B8%80_%ED%91%9C%EA%B8%B0" title="위키백과:러시아어의 한글 표기">*</a>]</sup></small>, <span style="font-size: smaller;"><a href="/wiki/%ED%84%B0%ED%82%A4%EC%96%B4" title="터키어">터키어</a>:</span> <span lang="tr" xml:lang="tr">Bakü</span> <small>바퀴<sup>[<a href="/wiki/%EC%9C%84%ED%82%A4%EB%B0%B1%EA%B3%BC:%EC%99%B8%EB%9E%98%EC%96%B4_%ED%91%9C%EA%B8%B0%EB%B2%95/%ED%84%B0%ED%82%A4%EC%96%B4_%EC%8B%9C%EC%95%88" title="위키백과:외래어 표기법/터키어 시안">*</a>]</sup></small>, <span style="font-size: smaller;"><a href="/wiki/%EB%AC%B8%ED%99%94%EC%96%B4" title="문화어">문화어</a>:</span> <b>바꾸</b>)는 <a href="/wiki/%EC%95%84%EC%A0%9C%EB%A5%B4%EB%B0%94%EC%9D%B4%EC%9E%94" title="아제르바이잔">아제르바이잔</a>의 <a href="/wiki/%EC%88%98%EB%8F%84" title="수도">수도</a>이다. <a href="/wiki/%EC%B9%B4%EC%8A%A4%ED%94%BC_%ED%95%B4" title="카스피 해">카스피 해</a> 서쪽에 뻗어 있는 <a class="new" href="/w/index.php?title=%EC%95%95%EC%85%B0%EB%A1%A0_%EB%B0%98%EB%8F%84&amp;action=edit&amp;redlink=1" title="압셰론 반도 (없는 문서)">압셰론 반도</a>에 위치하고, 시가지는 반도 남쪽의 <a class="new" href="/w/index.php?title=%EB%B0%94%EC%BF%A0_%EB%A7%8C&amp;action=edit&amp;redlink=1" title="바쿠 만 (없는 문서)">바쿠 만</a>에 접한 항구 도시이다. 카스피 해의 최대 항구 도시이다. 행정적으로는 11개 구와 48개 마을로 분할되고 있으며 <a href="/wiki/2003%EB%85%84" title="2003년">2003년</a> 시점의 총인구는 182만 9천 명이다.</p>
<p>아제르바이잔에서 가장 큰 도시인 동시에 <a class="mw-redirect" href="/wiki/%EC%9E%90%EC%BA%85%EC%B9%B4%EC%8A%A4" title="자캅카스">자캅카스</a>(남캅카스) 지역에서도 대도시이다. 대규모 <a href="/wiki/%EC%9C%A0%EC%A0%84_(%EC%84%9D%EC%9C%A0)" title="유전 (석유)">유전</a> 지대가 위치해 있고, <a href="/wiki/%EB%9F%AC%EC%8B%9C%EC%95%84_%EC%A0%9C%EA%B5%AD" title="러시아 제국">러시아 제국</a> 시대부터 <a href="/wiki/%EC%84%9D%EC%9C%A0" title="석유">석유</a>의 생산지로 발전해 왔다. 석유 가공업, 석유 기계 제조, 시멘트, 목면, 식료품 등의 공업이 성하다.<sup class="reference" id="cite_ref-1"><a href="#cite_note-1">[1]</a></sup></p>
