<p><b>바투</b>(<a href="/wiki/1207%EB%85%84" title="1207년">1207년</a> ~ <a href="/wiki/1255%EB%85%84" title="1255년">1255년</a>)는 <a href="/wiki/%EB%AA%BD%EA%B3%A8_%EC%A0%9C%EA%B5%AD" title="몽골 제국">몽골 제국</a> <a href="/wiki/%ED%82%B5%EC%B0%A8%ED%81%AC_%EC%B9%B8%EA%B5%AD" title="킵차크 칸국">킵차크 칸국</a>의 칸(재위:<a href="/wiki/1242%EB%85%84" title="1242년">1242년</a>~<a href="/wiki/1255%EB%85%84" title="1255년">1255년</a>)으로 <a href="/wiki/%EC%B9%AD%EA%B8%B0%EC%A6%88_%EC%B9%B8" title="칭기즈 칸">칭기즈 칸</a>의 손자이며 <a class="mw-redirect" href="/wiki/%EC%A3%BC%EC%B9%98_(%EB%AA%BD%EA%B3%A8)" title="주치 (몽골)">주치 칸</a>의 차남이다. 주치 칸의 아들들 중에 훌륭한 칸 이라는 별명의 그는 부친 속령에 대한 통치권을 계승하였다.</p>
<p>바투의 영토는 이르티쉬의 서쪽인 세미팔라틴스크, 악몰린스크, 투르가이 또는 약튜빈스크, 우랄스크, 아다지, 호레즘 본토(히바)를 포함하였고, 또한 킵차크 인들의 땅부터 볼가 강 서쪽의 정복지인 <a href="/wiki/%EC%82%AC%EC%A4%80%EC%82%AC%EA%B5%AC#.EC.82.AC.EA.B5.AC" title="사준사구">제베</a>와 <a href="/wiki/%EC%82%AC%EC%A4%80%EC%82%AC%EA%B5%AC#.EC.82.AC.EA.B5.AC" title="사준사구">수베에테이</a>의 원정으로 얻은 땅을 포함하였다.</p>
<p><a href="/wiki/1227%EB%85%84" title="1227년">1227년</a> <a href="/wiki/8%EC%9B%94_18%EC%9D%BC" title="8월 18일">8월 18일</a> <a class="mw-redirect" href="/wiki/%EC%B9%AD%EA%B8%B0%EC%8A%A4_%EC%B9%B8" title="칭기스 칸">칭기스 칸</a>이 죽은 뒤, 혈통 문제가 불거져 바투와 <a class="mw-redirect" href="/wiki/%EC%B9%AD%EA%B8%B0%EC%8A%A4_%EC%B9%B8" title="칭기스 칸">칭기스 칸</a>의 다른 손자들 간의 알력다툼이 나타났다. <a href="/wiki/%EC%9A%B0%EA%B5%AC%EB%8D%B0%EC%9D%B4_%EC%B9%B8" title="우구데이 칸">우구데이 칸</a>은 주치의 혈통 문제를 거론하며 바투를 모욕한 자신의 아들들을 책망했지만, 사촌들에게 모욕을 당한 바투는 <a href="/wiki/%ED%82%B5%EC%B0%A8%ED%81%AC_%EC%B9%B8%EA%B5%AD" title="킵차크 칸국">킵차크 칸국</a>을 건설하고 몽골 본국과의 직접적인 관계는 끊어버렸다.</p>
<p>바투는 <a href="/wiki/%EB%AA%BD%EA%B3%A8" title="몽골">몽골</a> 전승에는 '훌륭한 임금'이라 하나, 러시아인들에게는 무자비한 정복자로 알려져 있다.<br/>
바투는 <a href="/wiki/1237%EB%85%84" title="1237년">1237년</a>에 <a href="/wiki/%EB%9F%AC%EC%8B%9C%EC%95%84" title="러시아">러시아</a>의 도시 <a href="/wiki/%EC%BF%A0%EB%A5%B4%EC%8A%A4%ED%81%AC" title="쿠르스크">쿠르스크</a>, <a href="/wiki/%EB%B2%A8%EA%B3%A0%EB%A1%9C%EB%93%9C" title="벨고로드">벨고로드</a>, <a href="/wiki/%EB%9E%B4%EC%9E%94" title="랴잔">랴잔</a> 등을 파괴했고 <a href="/wiki/1238%EB%85%84" title="1238년">1238년</a>에는 러시아 <a href="/wiki/%EB%85%B8%EB%B8%8C%EA%B3%A0%EB%A1%9C%EB%93%9C_%EC%A3%BC" title="노브고로드 주">노브고로드 주</a>의 <a href="/wiki/%ED%81%AC%EB%A0%88%EC%8A%A4%ED%85%8C%EC%B8%A0%ED%82%A4_%EA%B5%B0" title="크레스테츠키 군">크레스테츠키 군</a>을 침략했다.<br/>
<a href="/wiki/1241%EB%85%84" title="1241년">1241년</a> 4월 11일 <a class="extiw" href="https://en.wikipedia.org/wiki/Battle_of_Mohi" title="en:Battle of Mohi">셔요 강 다리의 전투</a>에서 <a href="/wiki/%ED%97%9D%EA%B0%80%EB%A6%AC" title="헝가리">헝가리</a>의 벨라 4세를 무찔렀다.</p>
<p>그는 <a href="/wiki/%EC%B9%AD%EA%B8%B0%EC%A6%88_%EC%B9%B8" title="칭기즈 칸">칭기즈 칸</a> 가문의 장로로서 제국의 <a href="/wiki/%EC%B9%B4%EA%B0%84" title="카간">대칸</a>을 둘러싼 권력 분쟁에서 중요한 역할을 감당하여, '대칸 옹립자'도 알려져 있다.</p>
<p><a href="/wiki/1241%EB%85%84" title="1241년">1241년</a>에 <a href="/wiki/%EC%9A%B0%EA%B5%AC%EB%8D%B0%EC%9D%B4_%EC%B9%B8" title="우구데이 칸">우구데이 칸</a>이 죽자 대칸의 자리를 두고 우구데이의 아들인 <a href="/wiki/%EA%B7%80%EC%9C%84%ED%81%AC_%EC%B9%B8" title="귀위크 칸">귀위크</a>와 바투 사이에 분쟁이 발생했다. <a href="/wiki/1246%EB%85%84" title="1246년">1246년</a>, 귀위크는 바투의 반대에도 불구하고 <a href="/wiki/%EC%BF%A0%EB%A6%B4%ED%83%80%EC%9D%B4" title="쿠릴타이">쿠릴타이</a>를 소집하여 대칸 에 올랐다.</p>
<p>귀위크의 즉위 이후 두 사람의 대립은 더욱 격화되어 1248년에는 전쟁 직전까지 치닫았지만 그 해에 귀위크가 사망하여 무산됐다. 그 이후 바투는 <a class="mw-redirect" href="/wiki/%ED%88%B4%EB%A3%A8%EC%9D%B4" title="툴루이">툴루이</a> 의 왕비 <a href="/wiki/%EC%86%8C%EB%A5%B4%EC%B9%B5%ED%83%80%EB%8B%88" title="소르칵타니">소르칵타니</a> 와 손 잡고 <a href="/wiki/%EC%9A%B0%EA%B5%AC%EB%8D%B0%EC%9D%B4_%EC%B9%B8" title="우구데이 칸">우구데이</a>계 와의 분쟁이 다시 발생했다. 바투는 1251년에 <a href="/wiki/%EC%BF%A0%EB%A6%B4%ED%83%80%EC%9D%B4" title="쿠릴타이">쿠릴타이</a>를 통하여 <a class="mw-redirect" href="/wiki/%ED%88%B4%EB%A3%A8%EC%9D%B4" title="툴루이">툴루이</a>의 장남 <a href="/wiki/%EB%AA%BD%EC%BC%80_%EC%B9%B8" title="몽케 칸">몽케</a> 가 <a href="/wiki/%EC%B9%B4%EA%B0%84" title="카간">대칸</a>에 오르도록 도왔다.</p>
<p>바투와 그의 장남 <a href="/wiki/%EC%82%AC%EB%A5%B4%ED%83%80%ED%81%AC" title="사르타크">사르타크</a>가 죽었을 때 바투의 형제 <a href="/wiki/%EB%B2%A0%EB%A5%B4%EC%BC%80_%EC%B9%B8" title="베르케 칸">베르케</a>(1207 이후 ~ 1266)가 킵차크 칸국을 계승하였다. 베르케는 몽골의 그의 사촌들과 통합하려하지 않았으며 <a href="/wiki/%ED%9B%8C%EB%9D%BC%EA%B5%AC" title="훌라구">훌라구</a> 칸과는 전쟁을 하였다. 그러나 베르케는 중국을 그의 이론적인 상부로 공공연히 인식하였다. 사실 베르케는 그때까지 독립적인 군주였다. 다행스럽게도 베르케는 유럽 공략에 관하여 바투와 이해를 공유하지 않았다.</p>
<p>바투는 적어도 4명의 자식을 낳았는데 <a class="extiw" href="https://zh.wikipedia.org/wiki/%E6%89%98%E6%89%98%E7%BD%95" title="zh:托托罕">토코</a> 칸은 차남이다.</p>
