<p><b>박규수</b>(朴珪壽, <a href="/wiki/1807%EB%85%84" title="1807년">1807년</a> <a href="/wiki/10%EC%9B%94_27%EC%9D%BC" title="10월 27일">10월 27일</a> ~ <a href="/wiki/1877%EB%85%84" title="1877년">1877년</a> <a href="/wiki/2%EC%9B%94_9%EC%9D%BC" title="2월 9일">2월 9일</a>)는 <a href="/wiki/%EC%A1%B0%EC%84%A0" title="조선">조선</a> 시대 말기의 문신으로 군인, 외교관, 개화사상가, 철학자이며 화가, 지도 제작자다. <a class="mw-redirect" href="/wiki/%EB%B0%95%EC%A7%80%EC%9B%90_(%EC%8B%A4%ED%95%99%EC%9E%90)" title="박지원 (실학자)">연암 박지원</a>의 손자로 추사 <a href="/wiki/%EA%B9%80%EC%A0%95%ED%9D%AC" title="김정희">김정희</a> 등과 교류가 깊었고, <a class="mw-redirect" href="/wiki/%EC%A0%9C%EB%84%A4%EB%9F%B4%EC%85%94%EB%A8%BC%ED%98%B8_%EC%82%AC%EA%B1%B4" title="제네럴셔먼호 사건">제네럴 셔먼 호</a>를 격퇴하고 <a href="/wiki/%EA%B2%BD%EB%B3%B5%EA%B6%81" title="경복궁">경복궁</a> 재건의 총책임을 맡는 등 <a href="/wiki/%ED%9D%A5%EC%84%A0%EB%8C%80%EC%9B%90%EA%B5%B0" title="흥선대원군">흥선대원군</a>으로부터도 그 능력을 인정받았다. 그러나 할아버지 <a class="mw-redirect" href="/wiki/%EB%B0%95%EC%A7%80%EC%9B%90_(%EC%8B%A4%ED%95%99%EC%9E%90)" title="박지원 (실학자)">연암 박지원</a>의 학문과 사상을 계승해 척화론(斥和論)에 반대하고, <a href="/wiki/%EC%B2%AD%EB%82%98%EB%9D%BC" title="청나라">청나라</a>에 사신으로 갔다온 후에는 <a class="mw-redirect" href="/wiki/%EC%96%91%EB%AC%B4_%EC%9A%B4%EB%8F%99" title="양무 운동">양무 운동</a>을 본받아 서양 기술을 선택적으로 도입하고 외국과 통상할 것을 주장했다. <a href="/wiki/%EA%B9%80%EC%98%A5%EA%B7%A0" title="김옥균">김옥균</a>, <a href="/wiki/%ED%99%8D%EC%98%81%EC%8B%9D" title="홍영식">홍영식</a>, <a href="/wiki/%EB%B0%95%EC%98%81%ED%9A%A8" title="박영효">박영효</a>, <a href="/wiki/%EC%84%9C%EC%9E%AC%ED%95%84" title="서재필">서재필</a>, <a href="/wiki/%EB%B0%95%EC%A0%95%EC%96%91" title="박정양">박정양</a>, <a href="/wiki/%EC%9C%A4%EC%B9%98%ED%98%B8" title="윤치호">윤치호</a> 등 <a class="mw-redirect" href="/wiki/%EA%B0%9C%ED%99%94%ED%8C%8C" title="개화파">개화파</a> 청년들을 길러냈으며 일본과 <a class="mw-redirect" href="/wiki/%EB%B3%91%EC%9E%90_%EC%88%98%ED%98%B8_%EC%A1%B0%EC%95%BD" title="병자 수호 조약">강화도 조약</a>이 체결될 때는 위정척사파의 명분론을 반대하고, 막후에서 조정 대신들을 움직여 조약 체결을 이끌었다.</p>
<p>1848년(헌종 14년) 42세 때 <a href="/wiki/%EC%A1%B0%EC%84%A0%EC%9D%98_%EA%B3%BC%EA%B1%B0_%EC%A0%9C%EB%8F%84" title="조선의 과거 제도">증광시</a>에 합격해 출사했다. 당시 <a href="/wiki/%EC%84%B8%EB%8F%84_%EC%A0%95%EC%B9%98" title="세도 정치">세도 정치</a> 하에서 비주류였던 <a class="mw-redirect" href="/wiki/%EB%B6%81%ED%95%99%ED%8C%8C" title="북학파">북학파</a> 출신으로 요직과 거리가 멀었지만 1862년(철종 13년) <a href="/wiki/%EC%A7%84%EC%A3%BC%EB%AF%BC%EB%9E%80" title="진주민란">진주민란</a>을 수습하고 <a href="/wiki/%EC%A0%9C%EB%84%88%EB%9F%B4%EC%85%94%EB%A8%BC%ED%98%B8_%EC%82%AC%EA%B1%B4" title="제너럴셔먼호 사건">제너럴셔먼호 사건</a>에서 승리하는 등 잇따른 난을 평정한 공으로 크게 승진했다.<sup class="reference" id="cite_ref-lee34_1-0"><a href="#cite_note-lee34-1">[1]</a></sup> <a href="/wiki/%EC%82%AC%ED%97%8C%EB%B6%80" title="사헌부">사헌부</a> <a href="/wiki/%EB%8C%80%EC%82%AC%ED%97%8C" title="대사헌">대사헌</a>과 <a href="/wiki/%ED%99%8D%EB%AC%B8%EA%B4%80" title="홍문관">홍문관</a> <a href="/wiki/%EC%A0%9C%ED%95%99" title="제학">제학</a>을 거쳐 조선 유학의 최고 영예의 하나인 <a href="/wiki/%EB%8C%80%EC%A0%9C%ED%95%99" title="대제학">대제학</a>에 올랐고, 이후 <a href="/wiki/%EC%9D%B4%EC%A1%B0" title="이조">이조</a><a href="/wiki/%EC%B0%B8%ED%8C%90" title="참판">참판</a>, <a class="mw-redirect" href="/wiki/%ED%98%95%EC%A1%B0%ED%8C%90%EC%84%9C" title="형조판서">형조판서</a>를 거쳐 <a href="/wiki/%EC%9A%B0%EC%9D%98%EC%A0%95" title="우의정">우의정</a>까지 지냈다. 1876년 <a href="/wiki/%EA%B0%95%ED%99%94%EB%8F%84_%EC%A1%B0%EC%95%BD" title="강화도 조약">강화도 조약</a> 때까지 줄기차게 개화를 주장하면서 <a href="/wiki/%ED%9D%A5%EC%84%A0%EB%8C%80%EC%9B%90%EA%B5%B0" title="흥선대원군">대원군</a> 및 척사파들과 계속 갈등을 빚고 실각했다. 1877년 <a class="mw-redirect" href="/wiki/%EC%88%98%EC%9B%90%EB%B6%80" title="수원부">수원부</a> <a class="mw-disambig" href="/wiki/%EC%9C%A0%EC%88%98" title="유수">유수</a>로 재직 중 죽었다.</p>
<p>본관은 <a href="/wiki/%EB%B0%98%EB%82%A8_%EB%B0%95%EC%94%A8" title="반남 박씨">반남</a>(潘南)으로 초명은 규학(珪鶴), 자는 환경(桓卿→瓛卿) 또는 정경(鼎卿), 호는 환재(桓齋→瓛齋), 헌재(獻齋), 환재거사(瓛齋居士) 등이다. 이정리, 이정관, 류화, <a href="/wiki/%EA%B9%80%EC%A0%95%ED%9D%AC" title="김정희">김정희</a>의 문인이며 시호는 문익공(文翼公)이다.</p>

<h4><span class="mw-headline" id=".EC.B6.9C.EC.83.9D.EA.B3.BC_.EC.B4.88.EA.B8.B0_.EC.82.B6">출생과 초기 삶</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%B0%95%EA%B7%9C%EC%88%98&amp;action=edit&amp;section=3" title="부분 편집: 출생과 초기 삶">편집</a><span class="mw-editsection-bracket">]</span></span></h4>

<p>1807년<sup class="reference" id="cite_ref-2"><a href="#cite_note-2">[2]</a></sup>(순조 7년) 10월 27일 종로 계동에서 <a class="mw-redirect" href="/wiki/%EB%B0%95%EC%A7%80%EC%9B%90_(%EC%8B%A4%ED%95%99%EC%9E%90)" title="박지원 (실학자)">연암 박지원</a>의 손자이자, <a href="/wiki/%ED%98%84%EA%B0%90" title="현감">현감</a> <a href="/wiki/%EB%B0%95%EC%A2%85%EC%B1%84" title="박종채">박종채</a>(朴宗采)와 정5품 통덕랑 류영(柳詠)의 딸인 전주 류씨의 아들로 태어났다. 아버지 <a href="/wiki/%EB%B0%95%EC%A2%85%EC%B1%84" title="박종채">박종채</a>는 대과에 급제하지 못해 관직은 현감에 그쳤지만, 잇따른 난을 평정한 아들의 공으로 1863년에 <a href="/wiki/%EC%9D%B4%EC%A1%B0" title="이조">이조</a><a href="/wiki/%EC%B0%B8%ED%8C%90" title="참판">참판</a>, 1865년에 <a class="mw-redirect" href="/wiki/%EC%9D%B4%EC%A1%B0%ED%8C%90%EC%84%9C" title="이조판서">이조판서</a>, 그리고 1873년에는 <a href="/wiki/%EC%98%81%EC%9D%98%EC%A0%95" title="영의정">영의정</a>에 거듭 <a href="/wiki/%EC%B6%94%EC%A6%9D" title="추증">추증</a>됐다.</p>
<p>두 부부는 결혼 후 8년이 지나도록 아이를 갖지 못했었다. 어느날 류씨가 혼수품으로 데려온 <a class="mw-redirect" href="/wiki/%ED%95%99" title="학">학</a>이 앞길을 인도하는 태몽을 꿨고, 아버지는 <a class="mw-redirect" href="/wiki/%EB%B0%95%EC%A7%80%EC%9B%90_(%EC%8B%A4%ED%95%99%EC%9E%90)" title="박지원 (실학자)">연암 박지원</a>에게 옥판(玉版)을 선물받는 꿈을 꾸고 임신 사실을 알았다. 정말 어렵게 가진 자식이었다. 태몽 때문에 이름을 규학(珪鶴)이라 했다가 30세에 규수로 개명했다.</p>
<p>대대로 <a href="/wiki/%EB%85%B8%EB%A1%A0" title="노론">노론</a>이었지만 일찍이 고조부 <a href="/wiki/%EB%B0%95%ED%95%84%EA%B7%A0" title="박필균">박필균</a> 때부터 <a class="mw-redirect" href="/wiki/%EB%8B%B9%EC%9F%81" title="당쟁">당쟁</a>을 스스로 거부했고, 할아버지 <a class="mw-redirect" href="/wiki/%EB%B0%95%EC%A7%80%EC%9B%90_(%EC%8B%A4%ED%95%99%EC%9E%90)" title="박지원 (실학자)">연암 박지원</a> 역시 <a href="/wiki/%EB%85%B8%EB%A1%A0" title="노론">노론</a> 내의 외척·탕평당으로서, 당시 주류였던 <a href="/wiki/%EB%B2%BD%ED%8C%8C" title="벽파">벽파</a>와는 다른 <a class="mw-redirect" href="/wiki/%EB%B6%81%ED%95%99%ED%8C%8C" title="북학파">북학파</a>라는 새로운 경향을 창시해 영수(領首)가 됐다. 박규수는 성장 후에도 이런 선대의 사상을 계승해 적극적인 서양문물 도입 및 외국과의 통상강화를 주장했다.</p>
<p>어린 시절부터 풍채가 컸고 <a href="/wiki/%EB%85%BC%EC%96%B4" title="논어">논어</a>를 즐겨읽어 연습장에 반복했다. 특히 '효도하는 사람이 임금을 섬길 수 있다. 군자란 남을 공경할 망정 멸시하지는 않는다. 반면 소인배들은 남을 멸시하고 존중할 줄 모른다 (孝民可以爲臣(효민가이위신) / 君子可敬而不可侮(군자가경이불가모) / 小人可侮以不可敬(소인가모이불가경)'는 구절을 좋아했다고 한다.</p>
<h4><span class="mw-headline" id=".EC.86.8C.EB.85.84_.EC.8B.9C.EC.A0.88">소년 시절</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%B0%95%EA%B7%9C%EC%88%98&amp;action=edit&amp;section=4" title="부분 편집: 소년 시절">편집</a><span class="mw-editsection-bracket">]</span></span></h4>
<p>일곱 살 무렵, 박규수가 외가에 놀러 갔을 때 일화다. 항상 무언가 그리길 좋아했던 그는 땅바닥에 불탑을 그리며 놀았다. 그런 모습을 외종조이자 스승인 <a class="new" href="/w/index.php?title=%EB%A5%98%ED%99%94&amp;action=edit&amp;redlink=1" title="류화 (없는 문서)">류화</a>가 물끄러미 보고 있었다. 류화는 고명한 성리학자였음에도 '선비가 될 놈이 왠 불탑이냐'라는 식으로 면박을 주진 않았다<sup class="reference" id="cite_ref-omious_3-0"><a href="#cite_note-omious-3">[3]</a></sup>. 대신 그에게 시 한 수를 지어줬다.<sup class="reference" id="cite_ref-omious_3-1"><a href="#cite_note-omious-3">[3]</a></sup></p>

<p>단계적으로 학문을 이루라는 충고였다. 단순히 말로만 그친 게 아니라 일곱 살짜리 외조카손자가 혹시라도 가르침을 잊을까 시 한 수<sup class="reference" id="cite_ref-omious_3-3"><a href="#cite_note-omious-3">[3]</a></sup>를 주었다. 학문의 길을 자연스럽게 가르치고자 했던 셈이다. 이렇게 그는 깎아내고 주입식으로 강요하는 교육보다는 물 흐르는 듯한 가르침 속에서 자랐다. 청렴했던 가풍 탓에 어려서는 넉넉치 못해 주로 아버지에게서 글을 배웠다. 가끔 진외종조부<sup class="reference" id="cite_ref-4"><a href="#cite_note-4">[4]</a></sup> <a class="new" href="/w/index.php?title=%EC%9D%B4%EC%A0%95%EB%A6%AC&amp;action=edit&amp;redlink=1" title="이정리 (없는 문서)">이정리</a>(李正履), <a class="new" href="/w/index.php?title=%EC%9D%B4%EC%A0%95%EA%B4%80&amp;action=edit&amp;redlink=1" title="이정관 (없는 문서)">이정관</a>(李正觀)과, 외종조부 류화(柳訸) 등을 찾아가 배우기도 했다.</p>
<p>공부를 잘할 뿐만 아니라 독특하고 폭넓은 견해를 갖게 된 그는 15세의 어린 나이에도 <a class="new" href="/w/index.php?title=%EC%A1%B0%EC%A2%85%EC%98%81&amp;action=edit&amp;redlink=1" title="조종영 (없는 문서)">조종영</a>(趙鍾永) 등 명망높은 성리학자들과 나이를 뛰어넘어 친구가 될 만큼 (망년지교, 忘年之交) 학문적으로 성장했다. 18세 무렵에는 할아버지 <a class="mw-redirect" href="/wiki/%EB%B0%95%EC%A7%80%EC%9B%90_(%EC%8B%A4%ED%95%99%EC%9E%90)" title="박지원 (실학자)">연암 박지원</a>의 문인들을 찾아다니며 이곳저곳 가르침을 청했는데, 개중에는 당대의 명필이자 화가이며 <a class="mw-redirect" href="/wiki/%EA%B8%88%EC%84%9D%ED%95%99%EC%9E%90" title="금석학자">금석학자</a>였던 <a href="/wiki/%EA%B9%80%EC%A0%95%ED%9D%AC" title="김정희">김정희</a>도 있었다. 또한 김정희의 스승은 역시 <a class="mw-redirect" href="/wiki/%EB%B0%95%EC%A7%80%EC%9B%90_(%EC%8B%A4%ED%95%99%EC%9E%90)" title="박지원 (실학자)">연암 박지원</a>의 문인이었던 <a class="mw-redirect" href="/wiki/%EC%8B%A4%ED%95%99%EC%9E%90" title="실학자">실학자</a> <a href="/wiki/%EB%B0%95%EC%A0%9C%EA%B0%80" title="박제가">박제가</a>였다. 할아버지의 <a class="new" href="/w/index.php?title=%ED%95%99%EC%97%B0&amp;action=edit&amp;redlink=1" title="학연 (없는 문서)">학연</a>을 활용한 셈이다.</p>
<p>그는 <a href="/wiki/%EC%A7%80%EA%B5%AC" title="지구">지구</a>가 둥글다는 사실을 배워 알고 있었다. 열여섯 살 때 <a href="/wiki/%EB%8F%84%EB%B4%89%EC%82%B0" title="도봉산">도봉산</a> 정상에서 하늘을 두고 읊은 시 한 수가 남아있다. 10대 시절의 시집인 &lt;금유시집&gt;에 실린 일부다.<sup class="reference" id="cite_ref-omious_3-4"><a href="#cite_note-omious-3">[3]</a></sup></p>

<p>태양(A)·지구(B)·달(C)에 대한 천문학적 통찰을 시로써 정리한 것이다. 요즘 식으로 말하면, '과학탐구 영역'을 공부한 학생이 새로운 지식을 시로 정리하는 것과 유사하다. 시 쓰기를 통해 복습한 셈이다.<sup class="reference" id="cite_ref-omious_3-6"><a href="#cite_note-omious-3">[3]</a></sup> 또 다른 문집인 &lt;장암시집&gt;에는 지구과학적 지식을 정리한 것도 보인다. "아아! 큰 안목으로 볼 때, 지구를 만져보면 호두 속살 같을 거야"라고 했다.<sup class="reference" id="cite_ref-omious_3-7"><a href="#cite_note-omious-3">[3]</a></sup></p>
<p>그를 아는 사람중에는 박규수가 늘 특이한 것에 관심을 둔다며 기인이라 평하기도 했지만, 그의 집안 어른들은 그의 폭넓은 지적 호기심을 나무라기 보다는 오히려 더 북돋아 줬다.<sup class="reference" id="cite_ref-omious_3-8"><a href="#cite_note-omious-3">[3]</a></sup><sup class="reference" id="cite_ref-5"><a href="#cite_note-5">[5]</a></sup></p>
<h4><span class="mw-headline" id=".EC.B2.AD.EB.85.84.EA.B8.B0.EC.99.80_.EC.9D.80.EB.91.94_.EC.83.9D.ED.99.9C">청년기와 은둔 생활</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%B0%95%EA%B7%9C%EC%88%98&amp;action=edit&amp;section=5" title="부분 편집: 청년기와 은둔 생활">편집</a><span class="mw-editsection-bracket">]</span></span></h4>
<p>1828년 약관의 박규수는 <a class="mw-redirect" href="/wiki/%EC%A1%B0%EC%84%A0_%EC%9D%B5%EC%A2%85" title="조선 익종">효명세자</a>(孝明世子)와 친분을 나누며 개화를 논했고, 친구 이상의 관계로 학문과 미래를 토론했다.<sup class="reference" id="cite_ref-hyomms_6-0"><a href="#cite_note-hyomms-6">[6]</a></sup> 주로 《<a href="/wiki/%EC%97%AD%EA%B2%BD" title="역경">주역</a>》과 나랏일이었다.</p>
<p>후일 익종으로 추존되는 <a class="mw-redirect" href="/wiki/%EC%A1%B0%EC%84%A0_%EC%9D%B5%EC%A2%85" title="조선 익종">효명세자</a>는 대리청정 2년째에 아직 벼슬도 없던 20세의 박규수를 불러들여 '박규학의 학문은 누구도 따를 수 없으리만큼 출중하다'며 그를 곁에 뒀다. <a href="/wiki/%EC%97%AD%EA%B2%BD" title="역경">주역</a>을 신하들 앞에서 진강케 하는 한편, 조부 <a href="/wiki/%EB%B0%95%EC%A7%80%EC%9B%90_(1737%EB%85%84)" title="박지원 (1737년)">박지원</a>의 저작을 모두 모으라 명하고, 박규수 자신의 저술도 있으면 같이 올리라 했다. 이때 직접 <a class="new" href="/w/index.php?title=%EC%83%81%EA%B3%A0%EB%8F%84%EC%84%A4&amp;action=edit&amp;redlink=1" title="상고도설 (없는 문서)">상고도설</a>(尙古圖說) 80권을 지어 <a class="mw-redirect" href="/wiki/%EC%A1%B0%EC%84%A0_%EC%9D%B5%EC%A2%85" title="조선 익종">효명세자</a>에게 바쳤다. <a class="mw-redirect" href="/wiki/%EC%A1%B0%EC%84%A0_%EC%9D%B5%EC%A2%85" title="조선 익종">효명세자</a>는 이런 그를 몹시 아꼈다.</p>
<p><a class="mw-redirect" href="/wiki/%EC%A1%B0%EC%84%A0_%EC%9D%B5%EC%A2%85" title="조선 익종">효명세자</a>는 대리청정 중에 <a href="/wiki/%EC%95%88%EB%8F%99_%EA%B9%80%EC%94%A8" title="안동 김씨">안동 김씨</a> <a href="/wiki/%EC%84%B8%EB%8F%84_%EC%A0%95%EC%B9%98" title="세도 정치">세도문벌</a>들을 배제하고 처가인 <a href="/wiki/%ED%92%8D%EC%96%91_%EC%A1%B0%EC%94%A8" title="풍양 조씨">풍양 조씨</a>와 <a href="/wiki/%EB%85%B8%EB%A1%A0" title="노론">노론</a>내 비주류 및 <a href="/wiki/%EB%82%A8%EC%9D%B8" title="남인">남인</a>을 중용하는 한편, <a href="/wiki/%EC%9D%B4%EC%9D%B8%EC%A2%8C%EC%9D%98_%EB%82%9C" title="이인좌의 난">이인좌의 난</a> 이후 축출됐던 <a href="/wiki/%EC%86%8C%EB%A1%A0" title="소론">소론</a>까지 과감히 등용하는 등 개혁군주로서의 싹을 보였다. 박규수는 이런 효명세자의 개혁 가능성에 모든 기대를 걸었다. 그러나 <a class="mw-redirect" href="/wiki/%EC%A1%B0%EC%84%A0_%EC%9D%B5%EC%A2%85" title="조선 익종">효명세자</a>가 1832년 갑자기 훙거하면서 그의 꿈은 꺾였다. 슬픔과 실망이 너무 컸던 나머지 원래 자신의 자와 호의 '환'(桓: <small>굳셀 환</small>)이라는 글자를 '환'(瓛: <small>옥홀<sup class="reference" id="cite_ref-7"><a href="#cite_note-7">[7]</a></sup> 환, 재갈 얼</small>)으로 바꿀 정도였다.</p>
<p>안동 김씨 <a href="/wiki/%EC%84%B8%EB%8F%84_%EC%A0%95%EC%B9%98" title="세도 정치">세도 정치</a>가 계속됐던 데다가 <a class="mw-redirect" href="/wiki/%ED%9A%A8%EB%AA%85%EC%84%B8%EC%9E%90" title="효명세자">효명세자</a>, 어머니 유씨, 아버지 <a href="/wiki/%EB%B0%95%EC%A2%85%EC%B1%84" title="박종채">박종채</a>의 연이은 죽음으로 상심한 그는 20년 칩거에 들어간다.<sup class="reference" id="cite_ref-hyomms_6-1"><a href="#cite_note-hyomms-6">[6]</a></sup> 그는 할아버지 <a href="/wiki/%EB%B0%95%EC%A7%80%EC%9B%90_(1737%EB%85%84)" title="박지원 (1737년)">박지원</a>의 저작들<sup class="reference" id="cite_ref-8"><a href="#cite_note-8">[8]</a></sup>을 거듭 읽어 <a href="/wiki/%EB%B6%81%ED%95%99" title="북학">북학</a> 사상을 정교화하고, 할아버지의 제자들을 찾아다니며 가르침을 청했다. 윤종의(尹宗儀), 남병철(南秉哲), 김영작(金永爵) 등 당대 기라성같은 유학자들과도 깊은 교분을 나눠, 후일 <a href="/wiki/%ED%99%8D%EB%AC%B8%EA%B4%80" title="홍문관">홍문관</a> <a href="/wiki/%EB%8C%80%EC%A0%9C%ED%95%99" title="대제학">대제학</a>에 오를 정도의 주자학적 깊이까지 더하게 된다.</p>
<h3><span class="mw-headline" id=".EA.B3.BC.EA.B1.B0_.EA.B8.89.EC.A0.9C.EC.99.80_.EA.B4.80.EB.A3.8C_.EC.83.9D.ED.99.9C">과거 급제와 관료 생활</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%B0%95%EA%B7%9C%EC%88%98&amp;action=edit&amp;section=6" title="부분 편집: 과거 급제와 관료 생활">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<h4><span class="mw-headline" id=".EA.B4.80.EB.A3.8C_.EC.83.9D.ED.99.9C_.EC.B4.88.EB.B0.98">관료 생활 초반</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%B0%95%EA%B7%9C%EC%88%98&amp;action=edit&amp;section=7" title="부분 편집: 관료 생활 초반">편집</a><span class="mw-editsection-bracket">]</span></span></h4>
<p>1848년(헌종 14년) 42세가 된 그는 <a href="/wiki/%EC%A1%B0%EC%84%A0%EC%9D%98_%EA%B3%BC%EA%B1%B0_%EC%A0%9C%EB%8F%84" title="조선의 과거 제도">문과 증광시</a>(增廣試)에 병과(丙科) 25위로 합격했다. <a href="/wiki/%EC%A1%B0%EC%84%A0_%ED%97%8C%EC%A2%85" title="조선 헌종">헌종</a>은 '일찍이 부왕의 사랑을 받던 너를 내가 너무 늦게 알아보았다. 앞으로 크게 쓸 것이니 진력하라'고 했다. <a href="/wiki/%EC%82%AC%EA%B0%84%EC%9B%90" title="사간원">사간원</a> 정언(正言)이 되고 <a href="/wiki/%EB%B3%91%EC%A1%B0" title="병조">병조</a><a class="new" href="/w/index.php?title=%EC%A0%95%EB%9E%91&amp;action=edit&amp;redlink=1" title="정랑 (없는 문서)">정랑</a>을 지냈으며 <a class="mw-redirect" href="/wiki/%EC%9A%A9%EA%B0%95%EA%B5%B0" title="용강군">용강</a><a href="/wiki/%ED%98%84%EB%A0%B9" title="현령">현령</a>(龍岡縣令)으로 외직에 나간 동안 <a href="/wiki/%EC%A1%B0%EC%84%A0_%ED%97%8C%EC%A2%85" title="조선 헌종">헌종</a>이 사망했다.</p>
<p>1850년(철종 1년) <a href="/wiki/%EC%A0%84%EB%9D%BC%EB%B6%81%EB%8F%84" title="전라북도">전라북도</a> <a href="/wiki/%EB%B6%80%EC%95%88%EA%B5%B0" title="부안군">부안</a><a href="/wiki/%ED%98%84%EA%B0%90" title="현감">현감</a>으로 부임한 그는 <a href="/wiki/%EC%8B%A4%ED%95%99" title="실학">대실학자</a> <a href="/wiki/%EC%9C%A0%ED%98%95%EC%9B%90" title="유형원">반계 유형원</a>의 사적지를 찾아 <a href="/wiki/%EB%B0%98%EA%B3%84%EC%88%98%EB%A1%9D" title="반계수록">반계수록</a> 등 그의 저서들을 입수해 탐독한 후, 세상을 구할 학문이 쓰이지 못했다며 찬탄했다 한다. 그 해 <a href="/wiki/%EC%82%AC%ED%97%8C%EB%B6%80" title="사헌부">사헌부</a> <a class="mw-disambig" href="/wiki/%EC%A7%80%ED%8F%89" title="지평">지평</a>으로 궁에 복귀한 뒤 <a href="/wiki/%ED%99%8D%EB%AC%B8%EA%B4%80" title="홍문관">홍문관</a> <a class="new" href="/w/index.php?title=%EC%88%98%EC%B0%AC&amp;action=edit&amp;redlink=1" title="수찬 (없는 문서)">수찬</a>이 됐다.</p>
<p>1851년(철종 2년) <a href="/wiki/%EC%82%AC%ED%97%8C%EB%B6%80" title="사헌부">사헌부</a> <a class="mw-disambig" href="/wiki/%EC%9E%A5%EB%A0%B9" title="장령">장령</a>에 임명됐다.</p>
<p>1854년(철종 5년) <a href="/wiki/%EC%8A%B9%EC%A0%95%EC%9B%90" title="승정원">승정원</a> <a class="new" href="/w/index.php?title=%EB%8F%99%EB%B6%80%EC%8A%B9%EC%A7%80&amp;action=edit&amp;redlink=1" title="동부승지 (없는 문서)">동부승지</a>를 거쳐 그해 <a href="/wiki/%EA%B2%BD%EC%83%81%EB%8F%84" title="경상도">경상좌도</a> <a href="/wiki/%EC%95%94%ED%96%89%EC%96%B4%EC%82%AC" title="암행어사">암행어사</a>로 민정을 시찰했다.</p>
<p>1855년(철종 6년) 다시 <a href="/wiki/%EA%B2%BD%EC%83%81%EB%8F%84" title="경상도">경상좌도</a> <a href="/wiki/%EC%95%94%ED%96%89%EC%96%B4%EC%82%AC" title="암행어사">암행어사</a>로 파견됐다. 당시 탐관오리들을 봉고파직<sup class="reference" id="cite_ref-9"><a href="#cite_note-9">[9]</a></sup>한 전말을 기록한 <a href="/wiki/%EC%88%98%EA%B3%84" title="수계">수계</a>(繡啓) 등이 남아있다.<sup class="reference" id="cite_ref-10"><a href="#cite_note-10">[10]</a></sup></p>
<p>1858년(철종 9년) <a href="/wiki/%ED%99%A9%ED%95%B4%EB%8F%84" title="황해도">황해도</a> 곡산부사로 나갔다.</p>
<h5><span class="mw-headline" id=".EC.99.B8.EA.B5.90_.ED.99.9C.EB.8F.99.EA.B3.BC_.EC.9E.90.EB.AC.B8_.EC.9E.91.EC.84.B1">외교 활동과 자문 작성</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%B0%95%EA%B7%9C%EC%88%98&amp;action=edit&amp;section=8" title="부분 편집: 외교 활동과 자문 작성">편집</a><span class="mw-editsection-bracket">]</span></span></h5>
<div class="rellink boilerplate seealso"><img alt="" data-file-height="128" data-file-width="128" height="16" src="//upload.wikimedia.org/wikipedia/commons/thumb/e/ec/Crystal_Clear_app_xmag.svg/16px-Crystal_Clear_app_xmag.svg.png" srcset="//upload.wikimedia.org/wikipedia/commons/thumb/e/ec/Crystal_Clear_app_xmag.svg/24px-Crystal_Clear_app_xmag.svg.png 1.5x, //upload.wikimedia.org/wikipedia/commons/thumb/e/ec/Crystal_Clear_app_xmag.svg/32px-Crystal_Clear_app_xmag.svg.png 2x" width="16"/> <a href="/wiki/%ED%95%B4%EB%A6%AC_%ED%8C%8C%ED%81%AC%EC%8A%A4" title="해리 파크스">해리 파크스</a>, <a class="mw-redirect" href="/wiki/%EC%A0%9C2%EC%B0%A8_%EC%95%84%ED%8E%B8%EC%A0%84%EC%9F%81" title="제2차 아편전쟁">제2차 아편전쟁</a> 문서를 참고하십시오.</div>
<p>1860년(철종 11년) 조부 연암 박지원이 그랬듯, <a href="/wiki/%EC%B2%AD%EB%82%98%EB%9D%BC" title="청나라">청나라</a> 사신단에 <a class="mw-redirect" href="/wiki/%EC%97%B4%ED%95%98" title="열하">열하부사</a>(熱河副使)로 임명돼 6개월 간 <a href="/wiki/%EC%97%B0%EA%B2%BD" title="연경">연경</a>에 다녀왔다. 이 때 그는 처음으로 당시 국제정세의 흐름과 <a href="/wiki/%EC%A0%9C%EA%B5%AD%EC%A3%BC%EC%9D%98" title="제국주의">제국주의</a> 침략의 실상을 접했다. 업무 외 시간에는 4년 전 <a href="/wiki/%EC%A0%9C2%EC%B0%A8_%EC%95%84%ED%8E%B8_%EC%A0%84%EC%9F%81" title="제2차 아편 전쟁">애로우 호 사건</a> 때 영·프 군의 <a class="mw-redirect" href="/wiki/%EB%B6%81%EA%B2%BD" title="북경">북경</a>, <a class="mw-redirect" href="/wiki/%ED%86%88%EC%A7%84" title="톈진">톈진</a> 점령 직후 <a href="/wiki/%ED%95%A8%ED%92%8D%EC%A0%9C" title="함풍제">함풍제</a>(咸豊帝)의 대응 방식과 그 후 4년 여의 전쟁에 대해 자세히 조사했다. 박규수가 청국에 당도한 때는 4년 전쟁도 끝이 나고 <a href="/wiki/%EB%B2%A0%EC%9D%B4%EC%A7%95_%EC%A1%B0%EC%95%BD" title="베이징 조약">베이징 조약</a>이 체결될 시점이었다. 거인 중국이 불평등 조약으로 <a href="/wiki/%ED%99%8D%EC%BD%A9" title="홍콩">홍콩</a> <a class="mw-redirect" href="/wiki/%EC%A3%BC%EB%A3%BD_%EB%B0%98%EB%8F%84" title="주룽 반도">주룽 반도</a>와 <a class="mw-redirect" href="/wiki/%EC%97%B0%ED%95%B4%EC%A3%BC" title="연해주">연해주</a>를 생으로 뺏기고 외교적 위신이 깎이는 모습을 현장에서 목격하며 심각한 문제인식을 가지게 됐다. 또한 심병성(沈秉成), 왕증(王拯), 풍지기(馮志沂) 등 백여 명의 현지 문인, 학자들과 교류해 견문을 넓혔다.</p>
<p>1861년(철종 12년) 초 귀국 후 <a href="/wiki/%EC%84%B1%EA%B7%A0%EA%B4%80" title="성균관">성균관</a> <a class="mw-disambig" href="/wiki/%EB%8C%80%EC%82%AC%EC%84%B1" title="대사성">대사성</a>이 됐다. 얼마 후 그는 열하부사로 다시 <a href="/wiki/%EC%B2%AD%EB%82%98%EB%9D%BC" title="청나라">청나라</a>에 갔다. <a href="/wiki/%EC%A0%9C2%EC%B0%A8_%EC%95%84%ED%8E%B8_%EC%A0%84%EC%9F%81" title="제2차 아편 전쟁">제2차 아편 전쟁</a> 직후라 <a href="/wiki/%EC%B2%AD%EB%82%98%EB%9D%BC" title="청나라">청나라</a>를 통해 격변하는 국제 정세를 살피기 위해 일부러 사행(使行)을 다시 지원했다. 그는 이때부터 1871년 <a href="/wiki/%EC%8B%A0%EB%AF%B8%EC%96%91%EC%9A%94" title="신미양요">신미양요</a> 때까지 <a href="/wiki/%EC%8A%B9%EB%AC%B8%EC%9B%90" title="승문원">승문원</a>에서 각종 외교 문서들을 작성했다. <a href="/wiki/%EC%98%81%EC%96%B4" title="영어">영어</a>를 몰랐던 당시 조선 정부였기 때문에 그는 <a href="/wiki/%EC%98%81%EC%96%B4" title="영어">영어</a>를 해석한 중국 문헌들에 기초해서 서양과 외교 문서를 작성했다.</p>
<p>2번에 걸친 양요(洋擾) 때 <a href="/wiki/%EC%B2%AD%EB%82%98%EB%9D%BC" title="청나라">청나라</a>로 보낸 자문(咨文) 및 <a href="/wiki/%EB%AF%B8%EA%B5%AD" title="미국">미국</a>에 대한 힐문장(詰問)과 통상 요구에 대한 거절 문건을 대부분 그가 만들었고, <a href="/wiki/%EA%B0%95%ED%99%94%EB%8F%84_%EC%A1%B0%EC%95%BD" title="강화도 조약">강화도 조약</a> 때도 조정 중론을 모으는 것을 넘어, <a href="/wiki/%EC%9D%BC%EB%B3%B8" title="일본">일본</a>을 상대로 한 외교문서 다수의 자문과 감수, 교열에 참여했다.</p>

<h5><span class="mw-headline" id=".EB.AF.BC.EB.9E.80_.EC.88.98.EC.8A.B5.EA.B3.BC_.EC.B6.9C.EC.84.B8">민란 수습과 출세</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%B0%95%EA%B7%9C%EC%88%98&amp;action=edit&amp;section=9" title="부분 편집: 민란 수습과 출세">편집</a><span class="mw-editsection-bracket">]</span></span></h5>
<p>1862년(철종 13년) 2월 <a href="/wiki/%EC%A7%84%EC%A3%BC%EB%AF%BC%EB%9E%80" title="진주민란">진주민란</a>(晋州民亂)이 <a href="/wiki/%EA%B2%BD%EC%83%81%EB%8F%84" title="경상도">경상우도</a> 일대로 확대되자 안핵사(按覈使)로 나가 사태 수습을 맡았다. 지방관과 지역 향반들의 부패로 인한 백성들의 참상을 보고하고 세금 감면과 구휼을 주청해 성사시켰다. 그는 민란의 원인이 <a href="/wiki/%EC%82%BC%EC%A0%95%EC%9D%98_%EB%AC%B8%EB%9E%80" title="삼정의 문란">삼정의 문란</a>에 있음을 확인하고, 근본적 해결책을 마련하도록 조정의 중론(衆論)을 모을 특별기구를 건의했다. 그의 헌책으로 음력 5월 26일 <a href="/wiki/%EC%82%BC%EC%A0%95%EC%9D%98_%EB%AC%B8%EB%9E%80" title="삼정의 문란">삼정이정청</a>이 설치된다. 동년 10월 <a href="/wiki/%EC%9D%B4%EC%A1%B0" title="이조">이조</a><a href="/wiki/%EC%B0%B8%EC%9D%98" title="참의">참의</a>로 승진했다.</p>
<p>1863년(고종 즉위년) 12월 <a href="/wiki/%EC%8A%B9%EC%A0%95%EC%9B%90" title="승정원">승정원</a> <a href="/wiki/%EB%8F%84%EC%8A%B9%EC%A7%80" title="도승지">도승지</a>에 임명돼 <a class="mw-disambig" href="/wiki/%EA%B3%A0%EC%A2%85" title="고종">고종</a>을 측근에서 모시게 됐다. 막 즉위한 고종은 <a class="mw-redirect" href="/wiki/%ED%9A%A8%EB%AA%85%EC%84%B8%EC%9E%90" title="효명세자">익종(翼宗)</a>의 양자로 입승대통<sup class="reference" id="cite_ref-11"><a href="#cite_note-11">[11]</a></sup>돼 보위에 오를 수 있었는데, <a class="mw-redirect" href="/wiki/%ED%9A%A8%EB%AA%85%EC%84%B8%EC%9E%90" title="효명세자">효명세자</a>(익종)의 정비이자 고종의 양어머니가 된 <a class="mw-redirect" href="/wiki/%EC%A1%B0%EB%8C%80%EB%B9%84" title="조대비">조대비</a>(趙大妃)가 남편의 생전 절친했던 박규수를 <a href="/wiki/%ED%9D%A5%EC%84%A0%EB%8C%80%EC%9B%90%EA%B5%B0" title="흥선대원군">흥선대원군</a>에게 천거했다. '박규수는 벼슬길에 오르지 않았을 때도 <a class="mw-redirect" href="/wiki/%EC%9D%B5%EC%A2%85" title="익종">익종</a>께서 크게 쓰려던 인물이다. 그가 벼슬한 뒤 이제까지 그의 재주를 마음껏 발휘할 자리에 앉아본 적이 없는데, 한 번 써보는 것이 좋겠다'며 간곡히 부탁했다고 한다.</p>
<p>1864년(고종 1년) <a href="/wiki/%EB%B3%91%EC%A1%B0" title="병조">병조</a><a href="/wiki/%EC%B0%B8%ED%8C%90" title="참판">참판</a>을 거쳐 <a href="/wiki/%EC%82%AC%ED%97%8C%EB%B6%80" title="사헌부">사헌부</a><a href="/wiki/%EB%8C%80%EC%82%AC%ED%97%8C" title="대사헌">대사헌</a>, <a href="/wiki/%ED%99%8D%EB%AC%B8%EA%B4%80" title="홍문관">홍문관</a><a href="/wiki/%EC%A0%9C%ED%95%99" title="제학">제학</a>, <a href="/wiki/%EC%9D%B4%EC%A1%B0" title="이조">이조</a><a href="/wiki/%EC%B0%B8%ED%8C%90" title="참판">참판</a>을 두루 거쳤다. (모두 요직이었으나 같은 종2품계)</p>
<p>1865년 <a class="new" href="/w/index.php?title=%EC%98%81%EA%B1%B4%EB%8F%84%EA%B0%90&amp;action=edit&amp;redlink=1" title="영건도감 (없는 문서)">영건도감</a><a class="mw-redirect" href="/wiki/%EC%A0%9C%EC%A1%B0" title="제조">제조</a>(營建都監提調)를 겸해 <a href="/wiki/%ED%9D%A5%EC%84%A0%EB%8C%80%EC%9B%90%EA%B5%B0" title="흥선대원군">흥선대원군</a>이 착수한 <a href="/wiki/%EA%B2%BD%EB%B3%B5%EA%B6%81" title="경복궁">경복궁</a> 중건 작업 실무를 총괄했다.</p>
<p>1865년(고종 2년) <a class="mw-redirect" href="/wiki/%ED%95%9C%EC%84%B1%EB%B6%80_%ED%8C%90%EC%9C%A4" title="한성부 판윤">한성부 판윤</a>(정2품)을 거쳐 <a class="mw-redirect" href="/wiki/%EA%B3%B5%EC%A1%B0%ED%8C%90%EC%84%9C" title="공조판서">공조판서</a> 겸 지경연사(知經筵事)에 전임됐다. <a href="/wiki/%EA%B2%BD%EB%B3%B5%EA%B6%81" title="경복궁">경복궁</a>이 완성될 때까지 영건도감 제조직은 계속 겸임했다. 그 뒤 <a class="mw-redirect" href="/wiki/%EC%98%88%EC%A1%B0%ED%8C%90%EC%84%9C" title="예조판서">예조판서</a>, <a href="/wiki/%EC%82%AC%EA%B0%84%EC%9B%90" title="사간원">사간원</a><a class="mw-redirect" href="/wiki/%EB%8C%80%EC%82%AC%EA%B0%84" title="대사간">대사간</a>을 거쳐 그 해 8월 <a href="/wiki/%EB%8F%88%EB%85%95%EB%B6%80" title="돈녕부">돈녕부</a><a class="mw-disambig" href="/wiki/%EC%A7%80%EC%82%AC" title="지사">지사</a>(敦寧府知事)에 올랐다.</p>
<p>1866년(고종 3년) 고위 관리는 반드시 지방 외직을 순환해야 하는 규정에 따라 그 해 음력 <a href="/wiki/2%EC%9B%94" title="2월">2월</a> <a href="/wiki/%ED%8F%89%EC%95%88%EB%8F%84" title="평안도">평안도</a> <a href="/wiki/%EA%B4%80%EC%B0%B0%EC%82%AC" title="관찰사">관찰사</a>로 부임했다.</p>
<h4><span class="mw-headline" id=".EC.A0.9C.EB.84.88.EB.9F.B4_.EC.85.94.EB.A8.BC_.ED.98.B8_.EC.82.AC.EA.B1.B4.281866.EB.85.84.29"><a class="mw-redirect" href="/wiki/%EC%A0%9C%EB%84%88%EB%9F%B4_%EC%85%94%EB%A8%BC_%ED%98%B8_%EC%82%AC%EA%B1%B4" title="제너럴 셔먼 호 사건">제너럴 셔먼 호 사건</a>(1866년)</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%B0%95%EA%B7%9C%EC%88%98&amp;action=edit&amp;section=10" title="부분 편집: 제너럴 셔먼 호 사건(1866년)">편집</a><span class="mw-editsection-bracket">]</span></span></h4>
<p>1866년(<a class="mw-disambig" href="/wiki/%EA%B3%A0%EC%A2%85" title="고종">고종</a> 3년) <a href="/wiki/%EC%A0%9C%EB%84%88%EB%9F%B4%EC%85%94%EB%A8%BC%ED%98%B8_%EC%82%AC%EA%B1%B4" title="제너럴셔먼호 사건">제너럴셔먼호 사건</a>이 터졌다. 선교사 토마스<small>(한국식 이름 <a class="mw-redirect" href="/wiki/%EC%B5%9C%EB%82%9C%ED%97%8C" title="최난헌">최난헌</a>)</small> 등을 태운 미국 상선 <a class="mw-redirect" href="/wiki/%EC%A0%9C%EB%84%88%EB%9F%B4_%EC%85%94%EB%A8%BC%ED%98%B8" title="제너럴 셔먼호">제너럴 셔먼호</a>(The General Sherman)가 조선 정부의 분명한 통상요구 거절에도 허가없이 <a href="/wiki/%EB%8C%80%EB%8F%99%EA%B0%95" title="대동강">대동강</a>을 거슬러 올라오자 <a class="mw-redirect" href="/wiki/%ED%8F%89%EC%96%91" title="평양">평양</a><a href="/wiki/%EA%B0%90%EC%82%AC" title="감사">감사</a> 박규수가 공격 명령을 내렸다.<sup class="reference" id="cite_ref-12"><a href="#cite_note-12">[12]</a></sup></p>
<p>셔먼호는 상선이었으나 무장을 적재한 상태였고 밀물을 타고 강을 거슬러 올라왔지만 곧 조수가 밀려나가면서 모래톱에 좌초됐다.<sup class="reference" id="cite_ref-lee34_1-1"><a href="#cite_note-lee34-1">[1]</a></sup> 박규수는 체포조를 구성하는데 상금을 걸었다. 이 때 한 교졸<sup class="reference" id="cite_ref-13"><a href="#cite_note-13">[13]</a></sup>이 곧 자원해 어촌에서 징발한 괴피선<sup class="reference" id="cite_ref-14"><a href="#cite_note-14">[14]</a></sup> 여러 척에 기름먹인 섶을 가득 실어 셔먼호 옆에 붙였다. 그리고 궁수들로 일제히 불화살을 당기게 해 화공했다.<sup class="reference" id="cite_ref-lee34_1-2"><a href="#cite_note-lee34-1">[1]</a></sup> 화공이 시작되자 셔먼 호 내부의 인화물질에 옮겨 붙으면서 셔먼 호가 항행 불능에 빠졌다. 미국인들이 다급히 배에서 뛰어내려 도망쳤으나 대개가 사살되고 선장과 선교사 토마스는 평양 부민들에게 맞아죽었다.</p>
<p>조정에 전말이 보고된 후 박규수는 승차<small>(품계가 승진됨)</small>됐고 한낱 지방 아전<small>(중인 계급)</small>에 불과했던 교졸도 정3품 진장<sup class="reference" id="cite_ref-15"><a href="#cite_note-15">[15]</a></sup>(鎭將)에 올랐다.<sup class="reference" id="cite_ref-lee34_1-3"><a href="#cite_note-lee34-1">[1]</a></sup> 외세에 대해 민심이 흉흉했던 차에 박규수는 전국적으로 인기를 얻었고 아울러 <a href="/wiki/%EB%8C%80%EC%9B%90%EA%B5%B0" title="대원군">대원군</a>의 각별한 총애도 얻게 됐다.</p>
<h4><span class="mw-headline" id=".EA.B0.9C.ED.95.AD.2C_.EA.B0.9C.EA.B5.AD.EB.A1.A0.EA.B3.BC_.EA.B0.88.EB.93.B1">개항, 개국론과 갈등</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%B0%95%EA%B7%9C%EC%88%98&amp;action=edit&amp;section=11" title="부분 편집: 개항, 개국론과 갈등">편집</a><span class="mw-editsection-bracket">]</span></span></h4>
<h5><span class="mw-headline" id=".EC.B2.9C.EC.A3.BC.EA.B5.90.EB.8F.84.EC.97.90_.EB.8C.80.ED.95.9C_.EA.B4.80.EB.8C.80.ED.95.9C_.EB.8C.80.EC.9A.B0">천주교도에 대한 관대한 대우</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%B0%95%EA%B7%9C%EC%88%98&amp;action=edit&amp;section=12" title="부분 편집: 천주교도에 대한 관대한 대우">편집</a><span class="mw-editsection-bracket">]</span></span></h5>
<p>1866년(고종 3년) 10월 <a href="/wiki/%EB%B3%91%EC%9D%B8%EB%B0%95%ED%95%B4" title="병인박해">병인박해</a>로 천주교도들이 많이 죽었다. 당시 8천 여 명의 평신도와 <a href="/wiki/%ED%94%84%EB%9E%91%EC%8A%A4" title="프랑스">프랑스</a> <a href="/wiki/%ED%8C%8C%EB%A6%AC_%EC%99%B8%EB%B0%A9%EC%A0%84%EA%B5%90%ED%9A%8C" title="파리 외방전교회">파리 외방전교회</a> 출신의 <a href="/wiki/%EC%84%A0%EA%B5%90%EC%82%AC" title="선교사">선교사</a> 등이 처형된 사건으로, 발단은 <a href="/wiki/%EB%9F%AC%EC%8B%9C%EC%95%84" title="러시아">러시아</a>의 남하 정책에 위기를 느낀 <a href="/wiki/%ED%9D%A5%EC%84%A0%EB%8C%80%EC%9B%90%EA%B5%B0" title="흥선대원군">흥선대원군</a>이 프랑스 선교사들을 통해 한불 동맹을 맺으려다 뜻대로 되지 않자 <a class="mw-redirect" href="/wiki/%EC%B2%9C%EC%A3%BC%EA%B5%90" title="천주교">천주교</a>를 혹세무민에 무군무부의 사상이라며 탄압했기 때문이었다. 박규수는 천주교 박해가 국제 분쟁의 원인이 될 수 있는 데다, 백성이 <a class="mw-redirect" href="/wiki/%EC%B2%9C%EC%A3%BC%EA%B5%90" title="천주교">천주교</a>에 혹하는 것은 결국 가렴주구와 악정 때문이므로 처벌보다는 교화하고 선도하자며 관대한 처분을 상주했다.</p>
<p>전국적으로 혹독한 검거 및 고문 처형이 이어질 때도 그는 구금 정도의 관대한 처벌만 내렸다. <a href="/wiki/%EA%B4%80%EC%B0%B0%EC%82%AC" title="관찰사">관찰사</a> 때 그의 관내에는 처형자가 없었다. 결국 병인박해가 원인이 돼 <a href="/wiki/%EB%B3%91%EC%9D%B8%EC%96%91%EC%9A%94" title="병인양요">병인양요</a>가 터지면서 그의 선견지명이 증명됐다. 그는 민심이반을 막기 위한 방책으로 세금을 감면하고 흉년이 든 농가를 구제하며, 업무 후에도 따로 서실을 열고 선비들을 모아 글을 읽어 평안도 선비들이 과거에 응시해 관직을 얻을 수 있게 애쓰는 등 선정을 베풀었다. 2년 임기 후에도 1869년 4월까지 만 3년 2개월 간 유임됐다.</p>

<p>1871년(고종 8년) <a href="/wiki/%EC%98%88%EB%AC%B8%EA%B4%80" title="예문관">예문관</a> <a href="/wiki/%EC%A0%9C%ED%95%99" title="제학">제학</a>과 <a href="/wiki/%ED%99%8D%EB%AC%B8%EA%B4%80" title="홍문관">홍문관</a> <a href="/wiki/%EC%A0%9C%ED%95%99" title="제학">제학</a>을 겸했고 외교 자문을 직접 <a href="/wiki/%EC%B2%AD%EB%82%98%EB%9D%BC" title="청나라">청나라</a>에 지어보내 글씨좋다는 칭찬을 들었다. 이듬해 <a href="/wiki/%ED%99%8D%EB%AC%B8%EA%B4%80" title="홍문관">홍문관</a> <a href="/wiki/%EB%8C%80%EC%A0%9C%ED%95%99" title="대제학">대제학</a> 겸 <a href="/wiki/%EC%98%88%EB%AC%B8%EA%B4%80" title="예문관">예문관</a> <a href="/wiki/%EB%8C%80%EC%A0%9C%ED%95%99" title="대제학">대제학</a>이 돼 조선 <a class="mw-disambig" href="/wiki/%EC%9C%A0%ED%95%99" title="유학">유학</a>의 종장으로 인정받았다.</p>
<p>1872년(고종 9년) <a class="mw-redirect" href="/wiki/%EC%B2%A0%EC%A2%85_(%EC%A1%B0%EC%84%A0)" title="철종 (조선)">철종</a>의 하나 남은 혈육인 <a href="/wiki/%EC%98%81%ED%98%9C%EC%98%B9%EC%A3%BC" title="영혜옹주">영혜옹주</a>(永惠翁主)의 혼례를 정하게 됐다. 부마자리를 두고 4월 수원부유수 신석희(申錫禧)와 함께 자신의 문하생이자 같은 일족인 <a href="/wiki/%EB%B0%95%EC%98%81%ED%9A%A8" title="박영효">박영효</a>를 부마로 추천했다. 조선 왕실은 그의 추천대로 박영효를 부마로 삼아 금릉위의 봉작을 주는 한 편, 고종의 친척 매제 뻘이 된 10살짜리 박영효에게 삼정승의 품계인 <a href="/wiki/%EC%A1%B0%EC%84%A0%EC%9D%98_%ED%92%88%EA%B3%84" title="조선의 품계">상보국숭록대부</a>도 내리고 왕실종친 반열에 올렸다. 고종의 친형인 <a class="mw-redirect" href="/wiki/%EC%9D%B4%EC%9E%AC%EB%A9%B4" title="이재면">이재면</a>의 품계보다 높았다.</p>
<p>1872년(고종 9년) 청국 황제 <a href="/wiki/%EB%8F%99%EC%B9%98%EC%A0%9C" title="동치제">동치제</a>의 혼례식에 <a class="new" href="/w/index.php?title=%EC%A7%84%ED%95%98%EC%82%AC&amp;action=edit&amp;redlink=1" title="진하사 (없는 문서)">진하사</a>(進賀使) 정사(正使)로 서장관 강문형(姜文馨)과 수역(首譯) <a href="/wiki/%EC%98%A4%EA%B2%BD%EC%84%9D" title="오경석">오경석</a>(吳慶錫) 등의 사신단을 꾸려 건너갔다. 이 때 그는 전년도 <a href="/wiki/%EC%B2%AD%EB%82%98%EB%9D%BC" title="청나라">청나라</a> 사신으로 <a href="/wiki/%ED%94%84%EB%9E%91%EC%8A%A4" title="프랑스">프랑스</a>에 다녀온 삼구통상대신 완안 숭후(完顔崇厚)<sup class="reference" id="cite_ref-16"><a href="#cite_note-16">[16]</a></sup>를 만나 세계 정세를 묻고 아울러 서구의 서적과 무기, 화포, 건축술에 대한 자료도 손에 넣었다. <a href="/wiki/%EC%B2%AD%EB%82%98%EB%9D%BC" title="청나라">청나라</a> 조정이 서구 열강에 대항해 자구적 노력의 일환으로 일으킨 <a href="/wiki/%EC%96%91%EB%AC%B4%EC%9A%B4%EB%8F%99" title="양무운동">양무운동</a>(洋務運動)에 깊은 감명을 받고 귀국 후 개국과 개화의 필요성을 역설했지만, <a href="/wiki/%ED%9D%A5%EC%84%A0%EB%8C%80%EC%9B%90%EA%B5%B0" title="흥선대원군">흥선대원군</a>의 쇄국 정책과 <a href="/wiki/%EC%B5%9C%EC%9D%B5%ED%98%84" title="최익현">최익현</a>, <a class="new" href="/w/index.php?title=%EA%B9%80%ED%8F%89%EB%AC%B5&amp;action=edit&amp;redlink=1" title="김평묵 (없는 문서)">김평묵</a> 등 주자학적 명분론의 공격에 막혀 좌절해야 했다.</p>
<p>1873년(고종 10년) 5월 <a class="mw-redirect" href="/wiki/%ED%98%95%EC%A1%B0%ED%8C%90%EC%84%9C" title="형조판서">형조판서</a>에 임명됐고 음력 12월에 마침내 <a href="/wiki/%EC%9A%B0%EC%9D%98%EC%A0%95" title="우의정">우의정</a>에 올랐다. 그 때까지도 그는 <a href="/wiki/%ED%9D%A5%EC%84%A0%EB%8C%80%EC%9B%90%EA%B5%B0" title="흥선대원군">흥선대원군</a>에게 개국의 필요성을 여러 차례 역설했으나 채택되지 않았다. 박규수는 어디까지나 조정 중론에 따랐으면서도 당시 일본과의 국교문제에 있어서까지 줄기차게 개국을 주장했다.</p>
<h5><span class="mw-headline" id=".EB.8C.80.EC.9B.90.EA.B5.B0.EA.B3.BC.EC.9D.98_.EA.B0.88.EB.93.B1.EA.B3.BC_.EC.8B.A4.EA.B0.81">대원군과의 갈등과 실각</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%B0%95%EA%B7%9C%EC%88%98&amp;action=edit&amp;section=14" title="부분 편집: 대원군과의 갈등과 실각">편집</a><span class="mw-editsection-bracket">]</span></span></h5>
<p>1873년(고종 10년) 12월 <a href="/wiki/%EC%9D%BC%EB%B3%B8" title="일본">일본</a>이 자신들의 <a href="/wiki/%EB%A9%94%EC%9D%B4%EC%A7%80_%EC%9C%A0%EC%8B%A0" title="메이지 유신">왕정복고</a>와 이에 대한 정식수교 요청을 통고해 왔다. <a href="/wiki/%EB%A9%94%EC%9D%B4%EC%A7%80_%EC%9C%A0%EC%8B%A0" title="메이지 유신">메이지 유신</a>을 성공시킨 일본은 종전의 서계 격식을 버리고 <a href="/wiki/%EC%A1%B0%EC%84%A0%EC%9D%98_%EC%97%AD%EB%8C%80_%EA%B5%AD%EC%99%95" title="조선의 역대 국왕">고종</a>에게 '황'(皇), '칙령'(勅令), '대일본'(大日本) 등의 표현을 써보냈다. 이른바 서계문제(書契問題)로서 <a href="/wiki/%EC%A1%B0%EC%84%A0" title="조선">조선</a> 정부로서는 적어도 대등한 위치도 아니고 일본이 상국의 위치에서 써보낸 외교 전문에 대해 수리를 거부했다.</p>
<p>조선 정부에서 충격 속에 격론을 벌이는 가운데 그는 "직함(職銜)을 가서(加書)한 것은 저네들 자신 그 나라의 정령(政令)이 일신되어 그 인군의 우상(優賞)을 입은 것을 과시한 것뿐이다. 소위 <a href="/wiki/%ED%92%88%EA%B3%84" title="품계">관작</a>(官爵)을 승진했다는 것인데, 이것이 우리에게 무슨 상관인가? 종래의 격식과 다르다고 하여 이를 힐책하며 받지 않는데, 이것이 일개 통역관의 견해라면 괴이할 것이 없겠지만, 하필 조정 스스로 이를 교계(較計)하려 하는가? 가히 일소에 붙일 일이다." 라며 그냥 형식적인 것이니 연연하지 말자고 주장했다. <a href="/wiki/%EB%8C%80%EC%9B%90%EA%B5%B0" title="대원군">흥선대원군</a>을 직접 찾아가 일본이 평화적인 뜻으로 수교하려는 한 대국적 견지에서 서계를 받아들이자고 설득하였으나 역시 거부됐다.</p>
<p>1874년(고종 11년) 9월 <a href="/wiki/%EC%98%81%EC%9D%98%EC%A0%95" title="영의정">영의정</a> <a class="mw-disambig" href="/wiki/%EC%9D%B4%EC%9C%A0%EC%9B%90" title="이유원">이유원</a>과도 계속 충돌하고 자신의 뜻이 더이상 통하지 않는 걸 납득하게 된 그는 사퇴했다. 관직에서 물러난 그는 사랑방에 출입하는 젊은 양반자제들에게 <a class="mw-redirect" href="/wiki/%EB%B0%95%EC%A7%80%EC%9B%90_(%EC%8B%A4%ED%95%99%EC%9E%90)" title="박지원 (실학자)">박지원</a>의 사상을 강의하기도 하고 <a href="/wiki/%EC%A4%91%EA%B5%AD" title="중국">중국</a>을 왕래한 사신이나 <a class="new" href="/w/index.php?title=%EC%97%AD%EA%B4%80&amp;action=edit&amp;redlink=1" title="역관 (없는 문서)">역관</a>들이 전하는 새로운 사상을 전하기도 했다. 《연암집 燕巖集》, 《해국도지 海國圖志》 등에 대한 강연과 <a href="/wiki/%EC%9E%90%EB%AA%85%EC%A2%85" title="자명종">자명종</a>, <a href="/wiki/%EC%8B%9C%EA%B3%84" title="시계">시계</a>, <a class="new" href="/w/index.php?title=%ED%83%9C%EC%97%BD&amp;action=edit&amp;redlink=1" title="태엽 (없는 문서)">태엽</a>, <a href="/wiki/%EA%B8%B0%EA%B3%84" title="기계">기계</a>, <a href="/wiki/%EC%84%9C%EC%96%91%ED%99%94" title="서양화">서양화</a> 등을 소개하는 한편, 서양 열강의 무서움 및 중국의 패배와 양무 운동 등 세계의 정세를 전하고 부국강병을 역설했다.</p>
<p>1875년(고종 12년) 5월 조선 정계에서 은퇴한 몸임에도 <a href="/wiki/%EB%8C%80%EC%9B%90%EA%B5%B0" title="대원군">대원군</a>을 찾아가 '만약 저들(일본)이 포성을 한 번 발사하기에 이르면 이후 받으려 해도 이미 때늦어 나라를 욕되게 할 것'이라며 재차 설득했으나 받아들여 지지 않았다.</p>
<h4><span class="mw-headline" id=".EA.B0.95.ED.99.94.EB.8F.84_.EC.A1.B0.EC.95.BD_.EC.9D.B4.ED.9B.84">강화도 조약 이후</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%B0%95%EA%B7%9C%EC%88%98&amp;action=edit&amp;section=15" title="부분 편집: 강화도 조약 이후">편집</a><span class="mw-editsection-bracket">]</span></span></h4>
<p>1875년(고종 12년) 9월 일본은 <a class="mw-redirect" href="/wiki/%EC%9A%B4%EC%9A%94_%ED%98%B8_%EC%82%AC%EA%B1%B4" title="운요 호 사건">운요 호</a>(<a class="mw-redirect" href="/wiki/%EC%9A%B4%EC%96%91%ED%98%B8" title="운양호">운양호</a>)로 강화도를 포격했다. 그와 제자 <a class="mw-disambig" href="/wiki/%EA%B9%80%ED%99%8D%EC%A7%91" title="김홍집">김홍집</a>, 역관 <a href="/wiki/%EC%98%A4%EA%B2%BD%EC%84%9D" title="오경석">오경석</a> 등은 다시 한 번 수교를 강력히 촉구하고 설득해 이듬해 2월 <a href="/wiki/%EA%B0%95%ED%99%94%EB%8F%84_%EC%A1%B0%EC%95%BD" title="강화도 조약">강화도 조약</a>을 맺게 됐다. 이후 다수에 의해 매국노로 규탄받고 모함에 시달린 그는 병석에 누웠다. 그의 문인 중 한 사람인 <a class="mw-disambig" href="/wiki/%EA%B9%80%EC%9C%A4%EC%8B%9D" title="김윤식">운양 김윤식</a>은 "공<small>(박규수를 지칭)</small>은 늘 천장을 쳐다보며 길게 탄식하며, 윤기(倫紀)가 끊어져 나라도 장차 따라서 망하리니 가련한 우리 생민(生民)이 어찌하여 하늘로부터 저버려져야 하는가라고 했다. 드디어 걱정과 분함 때문에 병석에 누웠다."라고 했다.</p>
<p>1875년(고종 12년) 11월 25일 대왕대비의 가상존호옥책문제술관(加上尊號玉冊文製述官)을 겸했다.</p>
<p>1876년(고종 13년) 1월 건강도 좋지 않았을 뿐더러 칠순을 맞은 그는 치사(은거)를 상주했으나 불허됐고 대신 노신들의 모임인 <a href="/wiki/%EA%B8%B0%EB%A1%9C%EC%86%8C" title="기로소">기로소</a>(耆老所)에 소속돼 <a class="new" href="/w/index.php?title=%EA%B6%A4%EC%9E%A5&amp;action=edit&amp;redlink=1" title="궤장 (없는 문서)">궤장</a>(지팡이)과 의자, 안마를 하사받았다.</p>
<p>1876년(고종 13년) 2월 <a class="mw-redirect" href="/wiki/%EC%88%98%EC%9B%90%EB%B6%80" title="수원부">수원부</a> <a class="mw-disambig" href="/wiki/%EC%9C%A0%EC%88%98" title="유수">유수</a>(水原府留守)에 임명됐으나 건강이 악화돼 가지 못했고, 병세가 차도를 보이던 8월 9일 <a class="mw-redirect" href="/wiki/%EC%88%98%EC%9B%90%EB%B6%80" title="수원부">수원부</a> <a class="mw-disambig" href="/wiki/%EC%9C%A0%EC%88%98" title="유수">유수</a>에 재차 임명돼 수원으로 내려갔다.</p>
<p>1877년(고종 14년) 2월 9일<sup class="reference" id="cite_ref-17"><a href="#cite_note-17">[17]</a></sup>에 임지인 <a class="mw-redirect" href="/wiki/%EC%88%98%EC%9B%90%EB%B6%80" title="수원부">수원부</a> 청사에서 71세를 일기로 사망했다. <a class="mw-redirect" href="/wiki/%EC%A1%B0%EC%84%A0_%EA%B3%A0%EC%A2%85" title="조선 고종">고종</a>은 슬퍼하며 '도량과 식견이 고명하고, 문학이 박식해서 내가 의지하고 온 조야(朝野)가 기대하던 사람이다. 근래에 우의정의 벼슬을 벗은 것과 관련하여 특별히 거기에 머물러 살게 한 것은 바로 평시에 정력이 강직하여 잠시 휴식하게 해주면 다시 등용할 날이 있을 것이기 때문이었는데, 어찌 까닭모를 병으로 갑자기 영영 가버릴 줄이야 생각이나 하였겠는가? 내 슬픔과 한탄이야 어찌 그 끝이 있겠는가?' 라며 <a class="new" href="/w/index.php?title=%EC%8A%B9%EC%A7%80&amp;action=edit&amp;redlink=1" title="승지 (없는 문서)">승지</a>를 보내 치제하게 하고, 3년치 녹봉(祿俸)을 부의로 지급했다.</p>
<h4><span class="mw-headline" id=".EC.82.AC.ED.9B.84">사후</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%B0%95%EA%B7%9C%EC%88%98&amp;action=edit&amp;section=16" title="부분 편집: 사후">편집</a><span class="mw-editsection-bracket">]</span></span></h4>

박규수의 문하생들<br/>

<p>1877년(고종 14년) <a href="/wiki/%EA%B2%BD%EA%B8%B0%EB%8F%84" title="경기도">경기도</a> <a href="/wiki/%EC%96%91%EC%A3%BC%EA%B5%B0" title="양주군">양주군</a> 노원면 하계리<small>(현, <a href="/wiki/%EC%84%9C%EC%9A%B8%ED%8A%B9%EB%B3%84%EC%8B%9C" title="서울특별시">서울특별시</a> <a href="/wiki/%EB%85%B8%EC%9B%90%EA%B5%AC" title="노원구">노원구</a> 하계동)</small> 산 20-3번지 현재 <a href="/wiki/%EC%84%9C%EB%9D%BC%EB%B2%8C%EA%B3%A0%EB%93%B1%ED%95%99%EA%B5%90" title="서라벌고등학교">서라벌 고등학교</a> 자리에 안장됐다.<sup class="reference" id="cite_ref-18"><a href="#cite_note-18">[18]</a></sup></p>
<p>1878년(고종 15년) 11월 1일 문익(文翼)의 시호가 내려졌다. 그의 사상과 학문은 <a href="/wiki/%EA%B9%80%EC%98%A5%EA%B7%A0" title="김옥균">김옥균</a>, <a href="/wiki/%EB%B0%95%EC%98%81%ED%9A%A8" title="박영효">박영효</a>, <a href="/wiki/%EC%9C%A0%EA%B8%B8%EC%A4%80" title="유길준">유길준</a>, <a href="/wiki/%EC%9C%A4%EC%9B%85%EB%A0%AC" title="윤웅렬">윤웅렬</a>, <a class="mw-disambig" href="/wiki/%EA%B9%80%ED%99%8D%EC%A7%91" title="김홍집">김홍집</a>, <a href="/wiki/%EC%9C%A4%EC%B9%98%ED%98%B8" title="윤치호">윤치호</a>, <a href="/wiki/%ED%99%8D%EC%98%81%EC%8B%9D" title="홍영식">홍영식</a>, <a href="/wiki/%EC%84%9C%EC%9E%AC%ED%95%84" title="서재필">서재필</a> 등에게 계승됐다. 그가 죽은 후에도 <a class="mw-redirect" href="/wiki/%EC%9C%A0%EB%8C%80%EC%B9%98" title="유대치">유대치</a>, <a href="/wiki/%EC%98%A4%EA%B2%BD%EC%84%9D" title="오경석">오경석</a> 등이 문하생들의 훈육을 맡았다. <a class="mw-redirect" href="/wiki/%EC%96%91%EB%AC%B4_%EC%9A%B4%EB%8F%99" title="양무 운동">양무 운동</a> 모델의 동도서기를 주장했던 박규수와 달리 제자들은 <a href="/wiki/%EB%A9%94%EC%9D%B4%EC%A7%80_%EC%9C%A0%EC%8B%A0" title="메이지 유신">메이지 유신</a> 모델의 급진적인 개화를 향해 나아간다.</p>
<p>1884년(고종 21년) 10월 그의 제자들인 <a href="/wiki/%EA%B9%80%EC%98%A5%EA%B7%A0" title="김옥균">김옥균</a>, <a href="/wiki/%EB%B0%95%EC%98%81%ED%9A%A8" title="박영효">박영효</a>, <a href="/wiki/%EC%84%9C%EC%9E%AC%ED%95%84" title="서재필">서재필</a>, <a href="/wiki/%EC%9C%A0%EA%B8%B8%EC%A4%80" title="유길준">유길준</a> 등이 <a href="/wiki/%EA%B0%91%EC%8B%A0%EC%A0%95%EB%B3%80" title="갑신정변">갑신정변</a>에 실패해 대거 <a href="/wiki/%EC%9D%BC%EB%B3%B8" title="일본">일본</a>과 <a href="/wiki/%EB%AF%B8%EA%B5%AD" title="미국">미국</a>으로 망명갔다. 문하생들이 역도들이 되자 이미 사망한 그 역시 관작을 모두 삭탈당했으며, 집은 헐려 공터가 됐다가 1906년 <a href="/wiki/%EB%B3%B4%EC%84%B1%EC%A4%91%ED%95%99%EA%B5%90_(%EC%84%9C%EC%9A%B8)" title="보성중학교 (서울)">보성중학교</a> 부지가 됐다.</p>
<p>1894년(고종 31년) <a class="mw-redirect" href="/wiki/%EC%B2%AD%EC%9D%BC%EC%A0%84%EC%9F%81" title="청일전쟁">청일전쟁</a>에서 승리한 일본을 등에 업고 개화파들이 조정에 복귀해 <a class="mw-redirect" href="/wiki/%EA%B0%91%EC%98%A4%EA%B2%BD%EC%9E%A5" title="갑오경장">갑오경장</a>을 실시한 뒤에야 사면복권됐다.</p>
<p>1910년 <a href="/wiki/%EB%8C%80%ED%95%9C%EC%A0%9C%EA%B5%AD" title="대한제국">대한제국</a> 멸망 후 그의 개화 사상과 삶을 소개한 문집이 처음 간행됐다. <a href="/wiki/%EC%97%B0%EC%95%94%EC%A7%91" title="연암집">연암집</a> 역시 처음으로 정식 간행됐다.</p>
<p>1921년 3월 31일 <a href="/wiki/%EC%9D%BC%EC%A0%9C_%EA%B0%95%EC%A0%90%EA%B8%B0" title="일제 강점기">일제 강점기</a>였지만 <a href="/wiki/%EC%9D%B4%EC%99%95%EC%A7%81" title="이왕직">이왕직</a>(李王職) 등으로 명맥은 유지하던 <a href="/wiki/%EB%8C%80%ED%95%9C%EC%A0%9C%EA%B5%AD_%EC%88%9C%EC%A2%85" title="대한제국 순종">순종</a>의 명으로 <a href="/wiki/%EC%8B%A0%EC%9D%91%EC%A1%B0" title="신응조">신응조</a>(申應朝), <a class="new" href="/w/index.php?title=%EC%9D%B4%EB%8F%88%EC%9A%B0&amp;action=edit&amp;redlink=1" title="이돈우 (없는 문서)">이돈우</a>(李敦宇), <a href="/wiki/%EB%AF%BC%EC%98%81%ED%99%98" title="민영환">민영환</a>(閔泳煥) 등과 <a href="/wiki/%EB%8C%80%ED%95%9C%EC%A0%9C%EA%B5%AD_%EA%B3%A0%EC%A2%85" title="대한제국 고종">고종</a>의 묘정에 배향됐다.</p>
<p>1950년 <a class="mw-redirect" href="/wiki/6.25_%EC%A0%84%EC%9F%81" title="6.25 전쟁">6.25 전쟁</a> 통에 <a href="/wiki/%EA%B2%BD%EA%B8%B0%EB%8F%84" title="경기도">경기도</a> <a href="/wiki/%EA%B4%91%EC%A3%BC%EA%B5%B0_(%EA%B2%BD%EA%B8%B0%EB%8F%84)" title="광주군 (경기도)">광주군</a>에 보관 중이던 그의 저서와 유물 상당수가 소실됐다. <a class="mw-redirect" href="/wiki/%EA%B0%9C%ED%99%94%ED%8C%8C" title="개화파">개화파</a>들에 대해 해방 이후에도 평가가 좋지 않아 같이 잊혀져 가는 듯했던 그의 사상은 1970년 대가 돼서야 <a href="/wiki/%EA%B5%AD%EC%82%AC%ED%8E%B8%EC%B0%AC%EC%9C%84%EC%9B%90%ED%9A%8C" title="국사편찬위원회">국사편찬위원회</a>를 통해 한글로 번역됐고, 70년대 중반부터 <a href="/wiki/%EC%9D%BC%EB%B3%B8" title="일본">일본</a> <a class="new" href="/w/index.php?title=%EB%8D%B4%EB%A6%AC%EB%8C%80%ED%95%99%EA%B5%90&amp;action=edit&amp;redlink=1" title="덴리대학교 (없는 문서)">덴리대학교</a> <a href="/wiki/%EC%A1%B0%EC%84%A0%ED%95%99%ED%9A%8C" title="조선학회">조선학회</a>에서도 그의 저서와 사상 등에 대한 본격적인 연구가 진행됐다.</p>
<h2><span class="mw-headline" id=".EC.82.AC.EC.83.81.EA.B3.BC_.EC.8B.A0.EB.85.90">사상과 신념</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%B0%95%EA%B7%9C%EC%88%98&amp;action=edit&amp;section=17" title="부분 편집: 사상과 신념">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>그의 개화 사상은 실학 사상의 근대지향적 측면을 내재적으로 계승한 위에 외발적 요인이 작용해 촉발된 것으로, 선대의 <a class="mw-redirect" href="/wiki/%EB%B6%81%ED%95%99%ED%8C%8C" title="북학파">북학파</a> 학자들이 주장한 이용후생(利用厚生)이었다. 두 차례에 걸친 연행을 통하여 세계 정세를 파악하고 서구에 보다 우수한 문명이 있음을 인정, 좋은 것은 과감하게 수용하자는 의견을 개진하게 됐다. 중국의 개화파 관리들과 접촉하면서 개화의 필요성을 절감하고 개항을 역설했다.</p>
<h3><span class="mw-headline" id=".EA.B0.9C.ED.95.AD.2C_.EA.B0.9C.EA.B5.AD.EB.A1.A0">개항, 개국론</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%B0%95%EA%B7%9C%EC%88%98&amp;action=edit&amp;section=18" title="부분 편집: 개항, 개국론">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>그는 서양사정에 밝아 신문물의 수입과 문호개방을 주장했다. 그는 개항을 통해 서구의 선진 문물을 받아들여야 한다고 보았다. 그는 할아버지 <a class="mw-disambig" href="/wiki/%EB%B0%95%EC%A7%80%EC%9B%90" title="박지원">박지원</a>의 사상을 후대의 개화파에게 전달하여 <a class="mw-redirect" href="/wiki/%EB%B6%81%ED%95%99%ED%8C%8C" title="북학파">북학파</a>의 개혁, <a href="/wiki/%EC%8B%A4%EC%9A%A9%EC%A3%BC%EC%9D%98" title="실용주의">실용주의</a> 학문을 가르쳤다. 그는 현실에 유용하게 쓰이지 못하는 학문은 죽은 학문이라 했다.</p>
<p>그의 개국론은 그가 <a class="mw-redirect" href="/wiki/%EC%9A%B4%EC%96%91%ED%98%B8_%EC%82%AC%EA%B1%B4" title="운양호 사건">운양호 사건</a> 직전 '일본이 수호를 운운하면서 병선을 이끌고 온 것은 이해하기 어렵다. 그러나 수호의 사신이라 하니 우리가 먼저 선공해서는 안 된다. 그러나 만일 의외의 일이 있을 것 같으면 무력을 행사하지 않을 수 없다.'고 한 것처럼 유사시 무력 충돌도 불사한다는 자주적 개국으로, 무력적 굴복에 따른 타율적 개국은 아니었던 것이다. 그러나 그의 개국론은 <a href="/wiki/%EC%9D%BC%EB%B3%B8" title="일본">일본</a>에 굴복하는 것처럼 곡해됐다.</p>
<p>그는 척화론을 공리공담과 불필요한 체면으로 규정했다. 할아버지 박지원의 사상을 계승하여 <a href="/wiki/%EC%B5%9C%EC%9D%B5%ED%98%84" title="최익현">최익현</a>, <a class="new" href="/w/index.php?title=%EA%B9%80%ED%8F%89%EB%AC%B5&amp;action=edit&amp;redlink=1" title="김평묵 (없는 문서)">김평묵</a> 등의 주자학적 명분론에 입각한 척화론(斥和論)을 헛된 명분론으로 규정, 반대·비판했다. 그는 적극적인 서양문물의 도입 및 외국과의 통상강화를 주장했고, 북학파의 사상을 개화파에게 전수했다. 정계에서 은퇴한 후 <a class="mw-redirect" href="/wiki/%EA%B0%9C%ED%99%94%ED%8C%8C" title="개화파">개화파</a> 청년들을 지도하여 <a href="/wiki/%EA%B9%80%EC%98%A5%EA%B7%A0" title="김옥균">김옥균</a>, <a href="/wiki/%EB%B0%95%EC%98%81%ED%9A%A8" title="박영효">박영효</a>, <a href="/wiki/%EC%84%9C%EC%9E%AC%ED%95%84" title="서재필">서재필</a>, <a href="/wiki/%EC%9C%A4%EC%B9%98%ED%98%B8" title="윤치호">윤치호</a>, <a href="/wiki/%EB%B0%95%EC%A0%95%EC%96%91" title="박정양">박정양</a>, <a href="/wiki/%EC%9D%B4%EC%83%81%EC%9E%AC" title="이상재">이상재</a> 등에게 영향을 주었다.</p>
<h3><span class="mw-headline" id=".EC.87.84.EA.B5.AD_.EC.A0.95.EC.B1.85.EC.97.90_.EB.8C.80.ED.95.B4">쇄국 정책에 대해</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%B0%95%EA%B7%9C%EC%88%98&amp;action=edit&amp;section=19" title="부분 편집: 쇄국 정책에 대해">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>개항파로 알려진 박규수의 행적에 대해서는 논란의 여지가 있다. <a class="mw-redirect" href="/wiki/%EC%84%B1%EA%B7%A0%EA%B4%80_%EB%8C%80%ED%95%99%EA%B5%90" title="성균관 대학교">성균관 대학교</a> 한국한문학과 김명호 교수에 따르면, 그가 척사론자들과 마찬가지로 대원군의 양이 정책에 동조했으며 다만 교섭 여지를 주어 서양을 <a class="mw-redirect" href="/wiki/%EC%A4%91%ED%99%94%EB%AC%B8%EB%AA%85" title="중화문명">중화문명</a>에 귀의시키려 했을 뿐이란 것이다.<sup class="reference" id="cite_ref-kyunggi_19-0"><a href="#cite_note-kyunggi-19">[19]</a></sup></p>
<p><a class="mw-redirect" href="/wiki/%EC%A0%9C%EB%84%A4%EB%9F%B4%EC%85%94%EB%A8%BC%ED%98%B8_%EC%82%AC%EA%B1%B4" title="제네럴셔먼호 사건">제너럴 셔먼 호</a>를 불태우던 1866년 박규수는 강경한 척화파처럼 보인다. 이어 셰난도어호 내항 때에도 그가 미국·중국 등에 직접 지어 보낸 각종 문서는 어디까지나 정부 측의 강경한 입장 내지 힐문장들이었다.<sup class="reference" id="cite_ref-kyunggi_19-1"><a href="#cite_note-kyunggi-19">[19]</a></sup> 1871년 <a href="/wiki/%EC%8B%A0%EB%AF%B8%EC%96%91%EC%9A%94" title="신미양요">신미양요</a> 때에도 미국과의 교전을 주장하고 이를 관철했다.</p>
<p>김명호 교수에 의하면 “박규수가 양이를 주장하는 <a href="/wiki/%EC%9D%B4%ED%95%AD%EB%A1%9C" title="이항로">이항로</a>의 상소를 칭찬했으며 서양 오랑캐와 화친 불허 등을 담은 대원군의 양이책을 전폭 지지했다”며 박규수를 대표적인 주화론자나 개국을 구상한 인물로 보기는 어렵다는 견해를 밝혔다.<sup class="reference" id="cite_ref-kyunggi_19-2"><a href="#cite_note-kyunggi-19">[19]</a></sup> 그는 “<a href="/wiki/%EC%8B%A0%EB%AF%B8%EC%96%91%EC%9A%94" title="신미양요">신미양요</a> 시기에 박규수가 대미수교를 원했다는 종래의 논의는 단편적인 자료를 통해 확대해석한 결과”라고 주장했다.<sup class="reference" id="cite_ref-kyunggi_19-3"><a href="#cite_note-kyunggi-19">[19]</a></sup></p>
<p>한편 제너럴 셔먼 호에 승선했다가 살해된 <a href="/wiki/%EB%A1%9C%EB%B2%84%ED%8A%B8_%EC%A0%80%EB%A9%94%EC%9D%B8_%ED%86%A0%EB%A8%B8%EC%8A%A4" title="로버트 저메인 토머스">토마스</a> 선교사가 대동강을 거슬러온 것이 박규수의 초청 때문이었다는 설에 대해서도 김명호 교수는 이를 부정했다. <a href="/wiki/%EB%A1%9C%EB%B2%84%ED%8A%B8_%EC%A0%80%EB%A9%94%EC%9D%B8_%ED%86%A0%EB%A8%B8%EC%8A%A4" title="로버트 저메인 토머스">토마스</a> 목사가 <a class="mw-redirect" href="/wiki/%EB%B2%A0%EC%9D%B4%EC%A7%95" title="베이징">베이징</a>에서 박규수를 만나 선교활동의 지지와 후원 약속을 받았다는 설은 전혀 사실 무근으로, 셔먼호 사건을 전후해 박규수는 <a class="mw-redirect" href="/wiki/%EB%B2%A0%EC%9D%B4%EC%A7%95" title="베이징">베이징</a>에 간 사실이 없다는 것이다.<sup class="reference" id="cite_ref-kyunggi_19-4"><a href="#cite_note-kyunggi-19">[19]</a></sup></p>
<h3><span class="mw-headline" id=".EC.9C.84.EC.84.A0.EC.A0.81.EC.9D.B8_.EB.8F.84.EB.8D.95.EA.B4.80.EC.97.90_.EB.8C.80.ED.95.9C_.EB.B9.84.ED.8C.90">위선적인 도덕관에 대한 비판</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%B0%95%EA%B7%9C%EC%88%98&amp;action=edit&amp;section=20" title="부분 편집: 위선적인 도덕관에 대한 비판">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p><a class="mw-redirect" href="/wiki/%EB%B6%81%ED%95%99%ED%8C%8C" title="북학파">북학파</a>들의 문제 인식 속에는 양반 사회의 모순이 늘 자리잡고 있었다. 박규수 역시 사대부들의 도덕과 명분론이 허울이며 위선임을 지적했다. 선대의 선비들이 현실 정치나 벼슬길을 멀리했던 것은 도덕적 문제가 아니라 단지 자기 자신을 위해서라는 것이다. 다음은 벼슬길에 출사한 후 친지에게 보낸 편지 중 일부다.</p>

<p>공부가 완숙해질대로 완숙해진 42세 이후의 출사길이니까 젊은 혈기로 한 말은 아니다. 그를 비롯한 <a class="mw-redirect" href="/wiki/%EB%B6%81%ED%95%99%ED%8C%8C" title="북학파">북학파</a>들의 작품이나 언급에서 나타나는 조선 사회 모순의 핵심은 양반들이었고 그들의 위선과 아집이 역사적 발전을 막았기 때문에 오늘의 문제가 계속된다는 공통적인 문제의식이 있었다.</p>
<h3><span class="mw-headline" id=".ED.95.A9.EB.A6.AC.EC.A0.81.EC.9D.B8_.EC.99.B8.EA.B5.90.EB.A1.A0">합리적인 외교론</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%B0%95%EA%B7%9C%EC%88%98&amp;action=edit&amp;section=21" title="부분 편집: 합리적인 외교론">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>외교관계에 있어서 도의적인 것과 감정 보다는 실리를 찾을 수 있는 것이 외교의 기본이라 주장했다. 이에 따라 <a class="mw-redirect" href="/wiki/%EC%9C%84%EC%A0%95%EC%B2%99%EC%82%AC%ED%8C%8C" title="위정척사파">위정척사파</a>의 맹목적인 폐쇄론에 저항하고 서구의 문물을 받아들이는 한편 외국의 주장이 합당하다면 이를 무조건 배척할 것이 아니라 어느정도 수용해야 됨을 역설했다. 외국의 주장을 수용하는 한편 타협을 통해서 절충안을 찾자고 주장했다.</p>
<p><a href="/wiki/%EC%9D%BC%EB%B3%B8" title="일본">일본</a>과 병자수호조약이 체결될 때는 막후에서 반대파를 설득하여 조약 체결에 결정적으로 공헌했다. 그는 중요한 것은 호칭 문제가 아니라며 실질적으로 조선이 획득할 수 있는 이익을 찾아야 함을 역설했다. 1875년(고종 13년) <a href="/wiki/%EC%9A%B4%EC%9A%94%ED%98%B8_%EC%82%AC%EA%B1%B4" title="운요호 사건">운요호 사건</a>으로 <a href="/wiki/%EC%9D%BC%EB%B3%B8" title="일본">일본</a>이 수교를 요구하자 <a href="/wiki/%EC%B5%9C%EC%9D%B5%ED%98%84" title="최익현">최익현</a>(崔益鉉) 등의 강력한 척화 주장을 물리치고 <a href="/wiki/%EA%B0%95%ED%99%94%EB%8F%84_%EC%A1%B0%EC%95%BD" title="강화도 조약">강화도 조약</a>을 맺게 했다. 그 뒤 그는 척사파로부터 온갖 인신공격에 시달리게 된다.</p>
<h3><span class="mw-headline" id=".EB.8F.99.EB.8F.84.EC.84.9C.EA.B8.B0.EB.A1.A0">동도서기론</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%B0%95%EA%B7%9C%EC%88%98&amp;action=edit&amp;section=22" title="부분 편집: 동도서기론">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>그의 제자들이 <a href="/wiki/%EB%A9%94%EC%9D%B4%EC%A7%80_%EC%9C%A0%EC%8B%A0" title="메이지 유신">메이지 유신</a>식 개화를 주장했기 때문에 그도 같은 입장으로 오해되는 경우가 많으나, 실상 그는 <a href="/wiki/%EC%B2%AD%EB%82%98%EB%9D%BC" title="청나라">청나라</a>의 동도서기론를 주장했다. 어디까지나 양무 운동 모델을 따른 것이다. 그는 서양법(西洋法)에 대한 동양 학문과 도덕성의 우월함을 확신했던 유학자로, <a class="mw-redirect" href="/wiki/%EB%B6%81%ED%95%99%ED%8C%8C" title="북학파">북학파</a> 사상의 연장선상에서의 개국통상론이었다. 그의 제자들은 각자 스승인 박규수의 사상을 시대적 상황에 맞게 실천한 것 뿐이다. 서양의 물질 문명은 역시 우수하나 아편 전쟁이나 포함 외교 등을 미뤄봤을 때 분명 서양의 것을 답습해선 안된다는 인식이 있었다. 동양의 정신을 지키면 서양 역시 배울 점이 있고 개선될 점이 있을 것이라는 견해였다.</p>
<h3><span class="mw-headline" id=".EC.82.AC.EB.8C.80.EB.B6.80.EC.99.80_.EB.B0.B1.EC.84.B1">사대부와 백성</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%B0%95%EA%B7%9C%EC%88%98&amp;action=edit&amp;section=23" title="부분 편집: 사대부와 백성">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>박규수는 백성이 있은 뒤에야 사대부가 있다고 했다. 그리고 사대부가 백성의 윗사람이 아니라 백성들을 위해 일하는 것이 그 직분이라 가르쳤다. 이를 실천에 옮기기도 했다. 다음은 그 일화다.</p>
<p>그의 집안은 고조부 대에까지는 <a href="/wiki/%ED%95%9C%EC%84%B1%EB%B6%80" title="한성부">한성부</a>의 벌열가문이었지만 조부 <a class="mw-disambig" href="/wiki/%EB%B0%95%EC%A7%80%EC%9B%90" title="박지원">박지원</a>의 대부터는 재산이 없었다. 그의 처지를 안타깝게 여긴 문하생인 <a href="/wiki/%EC%83%9D%EC%9B%90" title="생원">생원</a>인 주씨(周氏)가 박규수 모르게 논 80석을 사뒀다. 그런데 신씨(申氏) 성을 가진 시골 노인이 찾아와 '연전에 사기를 당해 대감댁 땅을 모르고 샀으니 이를 어찌하면 좋으냐'고 했다. 박규수는 주 생원을 불러 땅 문서를 신씨 노인에게 주라 했다. 주생원이 후생들을 위해 그러지 말라고 애걸하였으나 박규수는 '백성이 있고 사대부가 있는 법'이라며 끝내 노인에게 주게 했다.</p>
<h2><span class="mw-headline" id=".EA.B8.B0.ED.83.80">기타</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%B0%95%EA%B7%9C%EC%88%98&amp;action=edit&amp;section=24" title="부분 편집: 기타">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<h3><span class="mw-headline" id=".EC.9C.A0.EA.B8.B8.EC.A4.80_.EA.B0.80.EB.AC.B8.EA.B3.BC.EC.9D.98_.EC.95.85.EC.97.B0">유길준 가문과의 악연</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%B0%95%EA%B7%9C%EC%88%98&amp;action=edit&amp;section=25" title="부분 편집: 유길준 가문과의 악연">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p><a href="/wiki/%EB%B0%95%EC%A7%80%EC%9B%90_(1737%EB%85%84)" title="박지원 (1737년)">연암 박지원</a>과 유길준의 5대조 <a href="/wiki/%EC%9C%A0%ED%95%9C%EC%A4%80" title="유한준">유한준</a>은 당대 쌍벽을 이루던 문장가들로 본래 문우(文友)이자 친구였는데, <a href="/wiki/%EB%B0%95%EC%A7%80%EC%9B%90_(1737%EB%85%84)" title="박지원 (1737년)">연암 박지원</a>이 <a href="/wiki/%EC%9C%A0%ED%95%9C%EC%A4%80" title="유한준">유한준</a>의 글을 풍자한 데서 감정싸움이 오가다가 이런저런 일이 있어 둘은 끝내 원수가 됐다. 싸움은 대를 이어 후일 <a href="/wiki/%EB%B0%95%EC%A7%80%EC%9B%90_(1737%EB%85%84)" title="박지원 (1737년)">박지원</a>의 아들 <a href="/wiki/%EB%B0%95%EC%A2%85%EC%B1%84" title="박종채">박종채</a>가 <a class="new" href="/w/index.php?title=%EA%B3%BC%EC%A0%95%EB%A1%9D&amp;action=edit&amp;redlink=1" title="과정록 (없는 문서)">과정록</a>에서 <a href="/wiki/%EC%9C%A0%ED%95%9C%EC%A4%80" title="유한준">유한준</a>을 깎아 내렸다.</p>

<p>저암 유한준과 연암 박지원은 집안끼리 친분이 있었고 연배도 비슷해 젊은 시절부터 절친한 사이였다. 그런데 <a href="/wiki/%EB%B0%95%EC%A7%80%EC%9B%90_(1737%EB%85%84)" title="박지원 (1737년)">연암 박지원</a>이 <a href="/wiki/%EC%9C%A0%ED%95%9C%EC%A4%80" title="유한준">유한준</a>에게 '글이 너무 기교에 치우쳤다'고 여러 차례 혹평했다. 그러자 저암은 연암에게 '오랑캐의 연호를 쓴 글(虜號之稿)을 쓴다'라며 몰아붙였다.<sup class="reference" id="cite_ref-21"><a href="#cite_note-21">[21]</a></sup> 결정적으로 선산 이장 문제가 불거지면서 원수가 되는데, 연암이 조부 <a href="/wiki/%EB%B0%95%ED%95%84%EA%B7%A0" title="박필균">박필균</a>과 친부 <a href="/wiki/%EB%B0%95%EC%82%AC%EC%9C%A0" title="박사유">박사유</a>의 묘를 이장코자 한 곳이 마침 <a href="/wiki/%EA%B8%B0%EA%B3%84_%EC%9C%A0%EC%94%A8" title="기계 유씨">기계 유씨</a> 선산 근처였었다. 유한준은 이를 반대하다가 막을 방법이 없자, 원래 집안의 정자가 있던 곳이라며 어린 나이(15세)에 죽은 자기 손자를 박필균 묘 위에 매장해 법률로 다투게 됐다. 이에 박종채는 유한준의 집안을 일컬어 '백세의 원수'로 규정한 것이었고, 이에 <a href="/wiki/%EC%9C%A0%ED%95%9C%EC%A4%80" title="유한준">유한준</a>의 아들 유만주도 연암을 '매우 잡스러운 인간'이었다라고 받아치는 등 감정의 골은 돌이킬 수 없게 됐었다.</p>
<p>1871년 <a href="/wiki/%ED%99%8D%EB%AC%B8%EA%B4%80" title="홍문관">홍문관</a> 제학 박규수는 <a class="mw-redirect" href="/wiki/%EA%B3%BC%EA%B1%B0_%EC%A0%9C%EB%8F%84" title="과거 제도">향시</a>에서 장원으로 뽑힌 시 한 수를 읽고는 장원급제자를 호출했다. 그가 당시 16세의 <a href="/wiki/%EC%9C%A0%EA%B8%B8%EC%A4%80" title="유길준">유길준</a>으로 바로 <a href="/wiki/%EC%9C%A0%ED%95%9C%EC%A4%80" title="유한준">유한준</a>의 5대손이었다. 그러자 <a href="/wiki/%EC%9C%A0%EA%B8%B8%EC%A4%80" title="유길준">유길준</a>의 아버지 유진수가 '어떻게 원수같은 자를 찾아간다는 말이냐'며 완강히 반대했다. 그러나 홍문관 제학으로 <a href="/wiki/%EC%9C%A0%EA%B8%B8%EC%A4%80" title="유길준">유길준</a>을 만난 박규수가 먼저 손을 내밀어 그를 거듭 칭찬하고는, '너희 집과 우리 집이 지난날 사소한 문제로 불화했으나 이제부터 옛날처럼 다시 화목하게 지낼 수 있다면 어른들이 풀지 못하셨던 감정을 우리가 풀어드리는 셈이 되는게 아니겠냐'며 감개무량해했다. 또한 힘써 공부할 것을 당부하고 구원(舊怨)을 잊고 자주 찾아오라며 은근하게 대했다. 그의 인품에 감복한 <a href="/wiki/%EC%9C%A0%EA%B8%B8%EC%A4%80" title="유길준">유길준</a>은 그 때부터 박규수를 스승으로 예우하고 배웠으며 <a class="mw-redirect" href="/wiki/%EA%B0%9C%ED%99%94%ED%8C%8C" title="개화파">개화파</a>로서 평탄하지만은 않은 삶을 관철했다.</p>
<h3><span class="mw-headline" id=".ED.95.99.EB.AC.B8_.EC.97.B0.EA.B5.AC.EC.99.80_.EA.B5.90.EC.9C.A0">학문 연구와 교유</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%B0%95%EA%B7%9C%EC%88%98&amp;action=edit&amp;section=26" title="부분 편집: 학문 연구와 교유">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>할아버지 <a class="mw-disambig" href="/wiki/%EB%B0%95%EC%A7%80%EC%9B%90" title="박지원">박지원</a>, <a href="/wiki/%EC%9C%A0%ED%98%95%EC%9B%90" title="유형원">유형원</a>, <a href="/wiki/%EB%B0%95%EC%A0%9C%EA%B0%80" title="박제가">박제가</a>, <a href="/wiki/%EC%9D%B4%EC%9D%B5" title="이익">이익</a>, <a href="/wiki/%EC%A0%95%EC%95%BD%EC%9A%A9" title="정약용">정약용</a>, <a href="/wiki/%EC%84%9C%EC%9C%A0%EA%B5%AC" title="서유구">서유구</a>, 김매순(金邁淳), 조종영, 홍석주(洪奭周), 윤정현(尹定鉉) 등을 선배로서 사숙했고, 문우로서 남병철, 김영작, 김상현(金尙鉉), 신응조(申應朝), 윤종의, 신석우(申錫愚) 등과 주로 교유했다.</p>
<p>그의 학풍은 제자의 한 사람인 <a class="mw-disambig" href="/wiki/%EA%B9%80%EC%9C%A4%EC%8B%9D" title="김윤식">김윤식</a>(金允植)의 지적에 의 하면 "크게는 체국경야(體國經野)의 제(制)로부터 작게는 금석(金石)·고고(考古)·의기(儀器)·잡복(雜服) 등의 일까지 연구하여 정확하고 실사구시(實事求是)하지 않는 바가 없고, 규모가 굉대하고 종리(綜理)가 미세 정밀"했다 한다.</p>
<p>박지원의 손자로서 인맥으로도 북학파에 직결되는 그가 사숙한 선배 중에는 <a class="mw-disambig" href="/wiki/%EB%B0%95%EC%A7%80%EC%9B%90" title="박지원">박지원</a>, <a href="/wiki/%EB%B0%95%EC%A0%9C%EA%B0%80" title="박제가">박제가</a> 등 <a href="/wiki/%EB%85%B8%EB%A1%A0" title="노론">노론</a><a class="mw-redirect" href="/wiki/%EB%B6%81%ED%95%99%ED%8C%8C" title="북학파">북학파</a> 외에도 <a href="/wiki/%EB%82%A8%EC%9D%B8" title="남인">남인</a>인 <a href="/wiki/%EC%A0%95%EC%95%BD%EC%9A%A9" title="정약용">정약용</a>(丁若鏞), 서유구(徐有榘), <a href="/wiki/%EB%B6%81%EC%9D%B8" title="북인">북인</a>인 <a href="/wiki/%EC%9C%A0%ED%98%95%EC%9B%90" title="유형원">유형원</a>, <a href="/wiki/%EC%9C%A4%ED%9C%B4" title="윤휴">윤휴</a> 등의 학문도 폭넓게 사숙했다. 다양한 선배 학자들의 학문을 사숙하였던 탓에 어떤 특정한 사상만이 정답은 아니라는 결론을 내리게 된다. 또한 <a href="/wiki/%EC%86%8C%EB%A1%A0" title="소론">소론</a>인 <a href="/wiki/%EC%9C%A0%EC%88%98%EC%9B%90" title="유수원">유수원</a>의 학문에도 관심을 갖기도 했다.</p>
<p>백의정승 <a class="mw-redirect" href="/wiki/%EC%9C%A0%EB%8C%80%EC%B9%98" title="유대치">유대치</a>, 중인 출신 외교관 <a href="/wiki/%EC%98%A4%EA%B2%BD%EC%84%9D" title="오경석">오경석</a> 등과는 신분을 초월하여 친구로 사귀었고, 승려 <a href="/wiki/%EC%9D%B4%EB%8F%99%EC%9D%B8" title="이동인">이동인</a>은 사상을 떠나 친구로 지냈다. 박규수는 사람은 신분이나 지위로 평가할 것이 아니라 그 사람의 됨됨이를 보고 평가해야 된다고 했다. 박규수는 자신의 문하에서 <a href="/wiki/%EA%B9%80%EC%98%A5%EA%B7%A0" title="김옥균">김옥균</a>, <a href="/wiki/%EB%B0%95%EC%98%81%ED%9A%A8" title="박영효">박영효</a>, <a href="/wiki/%EB%B0%95%EC%A0%95%EC%96%91" title="박정양">박정양</a>, <a href="/wiki/%EC%84%9C%EC%9E%AC%ED%95%84" title="서재필">서재필</a>, <a class="mw-disambig" href="/wiki/%EA%B9%80%EC%9C%A4%EC%8B%9D" title="김윤식">김윤식</a>, <a class="mw-disambig" href="/wiki/%EA%B9%80%ED%99%8D%EC%A7%91" title="김홍집">김홍집</a>, <a href="/wiki/%EC%9C%A0%EA%B8%B8%EC%A4%80" title="유길준">유길준</a>, <a href="/wiki/%EC%96%B4%EC%9C%A4%EC%A4%91" title="어윤중">어윤중</a>, <a href="/wiki/%EC%9C%A4%EC%9B%85%EB%A0%AC" title="윤웅렬">윤웅렬</a> 등의 제자, 문인들을 길러냈다.</p>

<p>그림도 좋아해 수백 여 편의 그림과 글씨를 남겼다고 하나 <a class="mw-redirect" href="/wiki/6.25_%EC%A0%84%EC%9F%81" title="6.25 전쟁">6.25 전쟁</a> 때 작품들이 대부분 소실됐다.<sup class="reference" id="cite_ref-omious_3-9"><a href="#cite_note-omious-3">[3]</a></sup> 앞에서 기술했듯 그의 글씨는 청나라 고관대작들의 칭찬을 받기도 했었다.</p>
<p>고전 읽기와 공부 방법을 흥미롭게 엮은 '상고도회문의례' 16권을 지었고, 그가 직접 제작한 <a class="mw-redirect" href="/wiki/%EC%A7%80%EA%B5%AC%EB%B3%B8" title="지구본">지구본</a> 설계도 평혼의(平渾儀)와 천문지도 간평의(簡平儀)의 종이 제작본 등이 현재 전한다.<sup class="reference" id="cite_ref-22"><a href="#cite_note-22">[22]</a></sup></p>
<p>문인화와 수묵화 외에도 또한 <a href="/wiki/%EA%B2%BD%EA%B8%B0%EB%8F%84" title="경기도">경기도</a> 지도인 동진방략(東津方略)을 그렸고, <a href="/wiki/%ED%8F%89%EC%95%88%EB%8F%84" title="평안도">평안도</a> 전도를 그리기도 했다. <a href="/wiki/%EC%B2%AD%EB%82%98%EB%9D%BC" title="청나라">청나라</a>의 세계지도와 <a class="mw-redirect" href="/wiki/%EC%B2%9C%EB%AC%B8" title="천문">천문</a>도 등 여러 문헌을 참고해 <a href="/wiki/%EC%84%B8%EA%B3%84_%EC%A7%80%EB%8F%84" title="세계 지도">세계 지도</a>인 <a class="new" href="/w/index.php?title=%ED%98%BC%ED%8F%89%EC%9D%98&amp;action=edit&amp;redlink=1" title="혼평의 (없는 문서)">혼평의</a>(渾平義)와 <a class="new" href="/w/index.php?title=%EC%B2%9C%EB%AC%B8%EB%8F%84&amp;action=edit&amp;redlink=1" title="천문도 (없는 문서)">천문도</a>, <a class="new" href="/w/index.php?title=%EA%B0%84%ED%8F%89%EC%9D%98&amp;action=edit&amp;redlink=1" title="간평의 (없는 문서)">간평의</a>(簡平義) 등 천문지도를 제작했다.</p>
<p>박규수의 <a href="/wiki/%ED%95%B4%EC%8B%9C%EA%B3%84" title="해시계">해시계</a>이자 천문도인 '<a class="new" href="/w/index.php?title=%EA%B0%84%ED%8F%89%EC%9D%98&amp;action=edit&amp;redlink=1" title="간평의 (없는 문서)">간평의</a>(簡平儀)의 종이 제작본은 2006년 5월 <a href="/wiki/%EC%8B%A4%ED%95%99%EB%B0%95%EB%AC%BC%EA%B4%80" title="실학박물관">실학박물관</a> 기공식 때 공개됐다.<sup class="reference" id="cite_ref-23"><a href="#cite_note-23">[23]</a></sup></p>
