<h2><span class="mw-headline" id=".EC.B6.95.EA.B5.AC.EC.9D.B8_.EA.B2.BD.EB.A0.A5">축구인 경력</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%B0%95%EB%8F%99%ED%98%81_(%EC%B6%95%EA%B5%AC_%EC%84%A0%EC%88%98)&amp;action=edit&amp;section=1" title="부분 편집: 축구인 경력">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<h3><span class="mw-headline" id=".EC.84.A0.EC.88.98_.EA.B2.BD.EB.A0.A5">선수 경력</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%B0%95%EB%8F%99%ED%98%81_(%EC%B6%95%EA%B5%AC_%EC%84%A0%EC%88%98)&amp;action=edit&amp;section=2" title="부분 편집: 선수 경력">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p><a href="/wiki/2002%EB%85%84" title="2002년">2002년</a> <a href="/wiki/%EC%A0%84%EB%B6%81_%ED%98%84%EB%8C%80_%EB%AA%A8%ED%84%B0%EC%8A%A4" title="전북 현대 모터스">전북 현대 모터스</a>에 입단하여 프로 생활을 시작하였다. 이후 주로 로테이션 멤버로 활약하다 <a href="/wiki/2006%EB%85%84" title="2006년">2006년</a> <a href="/wiki/%EC%9A%B8%EC%82%B0_%ED%98%84%EB%8C%80_%EC%B6%95%EA%B5%AC%EB%8B%A8" title="울산 현대 축구단">울산 현대</a>로 이적한 이후 붙박이 주전으로 출전하기 시작하며 <a href="/wiki/%EA%B9%80%EC%A0%95%EB%82%A8_(%EC%B6%95%EA%B5%AC%EC%9D%B8)" title="김정남 (축구인)">김정남</a>이 이끌던 울산의 수비에 크게 공헌하였다. <a href="/wiki/2008%EB%85%84" title="2008년">2008년</a>엔 <a href="/wiki/K%EB%A6%AC%EA%B7%B8_%EB%B2%A0%EC%8A%A4%ED%8A%B8_11" title="K리그 베스트 11">K리그 베스트 11</a> 수비수 부문에 오르기도 하였다.</p>
<p><a href="/wiki/2009%EB%85%84" title="2009년">2009년</a>, 돌연 <a href="/wiki/%EC%9D%BC%EB%B3%B8" title="일본">일본</a> <a href="/wiki/J%EB%A6%AC%EA%B7%B8" title="J리그">J리그</a> <a href="/wiki/%EA%B0%90%EB%B0%94_%EC%98%A4%EC%82%AC%EC%B9%B4" title="감바 오사카">감바 오사카</a>로 이적하였으나 반 시즌만에 하위권에서 맴돌던 <a href="/wiki/%EA%B0%80%EC%8B%9C%EC%99%80_%EB%A0%88%EC%9D%B4%EC%86%94" title="가시와 레이솔">가시와 레이솔</a>로 임대 되었다. 가시와의 강등을 막지는 못하였지만 수비력을 인정받아 시즌이 끝난 후 가시와로 완전 이적 하였다. <a href="/wiki/2010%EB%85%84" title="2010년">2010년</a>, 강등된 가시와가 한 시즌만에 <a href="/wiki/J2%EB%A6%AC%EA%B7%B8" title="J2리그">J2리그</a>에서 우승을 차지하고 승격하는데 일조하였다. 다시 J1리그로 복귀한 후에도 주전 수비수로 활약하며 팀이 <a class="mw-redirect" href="/wiki/J1%EB%A6%AC%EA%B7%B8_2011" title="J1리그 2011">2011 J1리그</a>에서 우승을 차지하는 데 공헌하였다.</p>
<p>가시와를 우승시킨 이후 <a href="/wiki/%EC%A4%91%EA%B5%AD_%EC%8A%88%ED%8D%BC%EB%A6%AC%EA%B7%B8" title="중국 슈퍼리그">중국 슈퍼리그</a>의 <a href="/wiki/%EB%8B%A4%EB%A1%84_%EC%8A%A4%EB%8D%94" title="다롄 스더">다롄 스더</a>로 이적하였다.</p>
<p>2013 시즌을 앞두고 친정팀 <a href="/wiki/%EC%9A%B8%EC%82%B0_%ED%98%84%EB%8C%80_%EC%B6%95%EA%B5%AC%EB%8B%A8" title="울산 현대 축구단">울산 현대</a>로 복귀하였다.</p>
<p>박동혁은 2014년 10월 26일 탄천종합운동장에서 열린 현대오일뱅크 K리그 클래식 2014 33라운드 성남과 경기에서 극적인 결승골을 기록하여 팀의 4-3 역전승을 거두는데 일조를 하였고, 더불어 팀의 상위 스플릿 진출에 큰 기여를 하였다.</p>
<p>2014 시즌이 끝나고, 현역 은퇴를 선언하였으며, 3월 21일 전남과의 홈 경기에서 은퇴식을 가졌다.</p>
<p>은퇴 후, 독일로 떠나 연수를 떠났으며 귀국 후, 울산의 유소년 스카우터로 활동하고 있다.</p>
