<h2><span class="mw-headline" id=".EC.83.9D.EC.95.A0">생애</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%B0%95%EB%B2%94%EC%8B%A0&amp;action=edit&amp;section=1" title="부분 편집: 생애">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p><a href="/wiki/%EC%A0%84%EB%9D%BC%EB%B6%81%EB%8F%84" title="전라북도">전라북도</a> <a href="/wiki/%EC%9D%B5%EC%82%B0%EC%8B%9C" title="익산시">익산군</a> 황화면(현 <a href="/wiki/%EC%B6%A9%EC%B2%AD%EB%82%A8%EB%8F%84" title="충청남도">충청남도</a> <a href="/wiki/%EB%85%BC%EC%82%B0%EC%8B%9C" title="논산시">논산시</a> <a href="/wiki/%EC%97%B0%EB%AC%B4%EC%9D%8D" title="연무읍">연무읍</a>) 봉동리에서 출생하였고, 중학교 때부터 <a href="/wiki/%EB%85%BC%EC%82%B0%EC%8B%9C" title="논산시">충청남도 논산군</a> <a href="/wiki/%EA%B0%95%EA%B2%BD%EC%9D%8D" title="강경읍">강경읍</a>에서 생활하였다. <a href="/wiki/%EC%A0%84%EC%A3%BC%EA%B5%90%EC%9C%A1%EB%8C%80%ED%95%99%EA%B5%90" title="전주교육대학교">전주교육대학</a>을 거쳐 <a href="/wiki/%EC%9B%90%EA%B4%91%EB%8C%80%ED%95%99%EA%B5%90" title="원광대학교">원광대학교</a> 국어국문학과와 <a href="/wiki/%EA%B3%A0%EB%A0%A4%EB%8C%80%ED%95%99%EA%B5%90" title="고려대학교">고려대학교</a> 교육대학원을 졸업하였다. <a href="/wiki/1973%EB%85%84" title="1973년">1973년</a> 《<a href="/wiki/%EC%A4%91%EC%95%99%EC%9D%BC%EB%B3%B4" title="중앙일보">중앙일보</a>》 <a href="/wiki/%EC%8B%A0%EC%B6%98%EB%AC%B8%EC%98%88" title="신춘문예">신춘문예</a>에 단편 〈여름의 잔해(殘骸)〉가 당선되어 등단했다. 이어 〈식구〉,〈말뚝과 굴렁쇠〉,〈못과 망치〉 등의 단편과 《죽음보다 깊은 밤》,《깨소금과 옥떨메》,《풀잎처럼 눕다》,《불의 나라》 등의 장편을 신문이나 잡지에 연재했다. <a href="/wiki/1981%EB%85%84" title="1981년">1981년</a> 장편 《겨울강, 하늬바람》으로 제1회 <a class="new" href="/w/index.php?title=%EB%8C%80%ED%95%9C%EB%AF%BC%EA%B5%AD%EB%AC%B8%ED%95%99%EC%83%81&amp;action=edit&amp;redlink=1" title="대한민국문학상 (없는 문서)">대한민국문학상</a> 신인부문을 수상했다. 창작집 《토끼와 잠수함》이 있고 단편집으로 《아침에 날린 풍선》,《식구》, 중편집 《도시의 이끼》,《그들은 그렇게 잊었다》와 다수의 장편소설을 출간했다.</p>
<p><a href="/wiki/1993%EB%85%84" title="1993년">1993년</a> 한 일간지에 소설을 연재하던 중 절필을 선언하고 <a href="/wiki/1996%EB%85%84" title="1996년">1996년</a> 중반까지 칩거에 들어갔으나, <a href="/wiki/1996%EB%85%84" title="1996년">1996년</a> 《문학동네》 가을호에 중편소설 〈흰소가 끄는 수레〉를 발표하면서 다시 글쓰기를 시작하여, 장편 《침묵의 집》(1999)과 단편 〈향기로운 우물이야기〉(2001년) 등을 발표하였다. <a href="/wiki/1995%EB%85%84" title="1995년">1995년</a>부터 <a href="/wiki/2004%EB%85%84" title="2004년">2004년</a>까지 <a href="/wiki/%EB%AA%85%EC%A7%80%EB%8C%80%ED%95%99%EA%B5%90" title="명지대학교">명지대학교</a> 문예창작학과 교수를 지냈고, <a href="/wiki/2007%EB%85%84" title="2007년">2007년</a> <a href="/wiki/%ED%95%9C%EA%B5%AD%EB%B0%A9%EC%86%A1%EA%B3%B5%EC%82%AC" title="한국방송공사">KBS 한국방송공사</a> 이사장과 서울문화재단 이사장을 맡았으며. <a href="/wiki/%EB%AA%85%EC%A7%80%EB%8C%80%ED%95%99%EA%B5%90" title="명지대학교">명지대학교</a>로 복귀해서 문예창작학과 교수를 지낸 뒤 <a href="/wiki/2011%EB%85%84" title="2011년">2011년</a> 정년퇴임하였고 그 후 <a href="/wiki/%EB%AA%85%EC%A7%80%EB%8C%80%ED%95%99%EA%B5%90" title="명지대학교">명지대학교</a> 문예창작학과 명예교수를 지내고 있으며 <a href="/wiki/2012%EB%85%84" title="2012년">2012년</a>부터는 <a href="/wiki/%EC%83%81%EB%AA%85%EB%8C%80%ED%95%99%EA%B5%90" title="상명대학교">상명대학교</a> 국어국문학과 석좌교수도 지내고 있다.</p>
<p>대한민국문학상(1987)·원광문학상(1998)·김동리문학상(2001)·만해문학상(2003)을 받았다. <sup class="reference" id="cite_ref-1"><a href="#cite_note-1">[1]</a></sup></p>

<h2><span class="mw-headline" id=".EB.85.BC.EB.9E.80">논란</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%B0%95%EB%B2%94%EC%8B%A0&amp;action=edit&amp;section=3" title="부분 편집: 논란">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<ul>
<li>2016년 10월 21일 전직 출판사 <a href="/wiki/%ED%8E%B8%EC%A7%91%EC%9E%90" title="편집자">편집자</a>라고 밝힌 A아무개는 <a href="/wiki/%ED%8A%B8%EC%9C%84%ED%84%B0" title="트위터">트위터</a>를 통해, 박범신 작가가 <a href="/wiki/%EC%B6%9C%ED%8C%90%EC%82%AC" title="출판사">출판사</a> 직원과 방송작가 등 여성 7명과 가진 <a class="mw-redirect" href="/wiki/%EC%88%A0%EC%9E%90%EB%A6%AC" title="술자리">술자리</a>에서 성적인 <a href="/wiki/%EB%86%8D%EB%8B%B4" title="농담">농담</a>을 하고 신체 접촉을 했다고 주장했다. 또한 박범신 작가가 영화 《<a href="/wiki/%EC%9D%80%EA%B5%90_(%EC%98%81%ED%99%94)" title="은교 (영화)">은교</a>》를 제작하던 당시 주연배우 <a href="/wiki/%EA%B9%80%EA%B3%A0%EC%9D%80" title="김고은">김고은</a>과 만나 “<a class="mw-disambig" href="/wiki/%EC%84%B9%EC%8A%A4" title="섹스">섹스</a> 경험이 있나”라고 물었다는 사실도 밝혔다.<sup class="reference" id="cite_ref-2"><a href="#cite_note-2">[2]</a></sup><sup class="reference" id="cite_ref-3"><a href="#cite_note-3">[3]</a></sup> 논란에 대해 박범신 작가는 "나로 인해 기분이 언짢고 불쾌했다면 내 불찰”이며 "상처 받았다면 미안하다"고 사과했다.<sup class="reference" id="cite_ref-4"><a href="#cite_note-4">[4]</a></sup></li>
</ul>
