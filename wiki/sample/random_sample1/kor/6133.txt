<p><b>박봉주</b>(朴奉珠,<sup class="reference" id="cite_ref-1"><a href="#cite_note-1">[1]</a></sup> <a href="/wiki/1939%EB%85%84" title="1939년">1939년</a> <a href="/wiki/4%EC%9B%94" title="4월">4월</a> ~ )는 <a href="/wiki/%EC%A1%B0%EC%84%A0%EB%AF%BC%EC%A3%BC%EC%A3%BC%EC%9D%98%EC%9D%B8%EB%AF%BC%EA%B3%B5%ED%99%94%EA%B5%AD" title="조선민주주의인민공화국">조선민주주의인민공화국</a>의 <a href="/wiki/%EC%A0%95%EC%B9%98%EA%B0%80" title="정치가">정치가</a>이자 <a class="mw-redirect" href="/wiki/%EA%B8%B0%EC%97%85%EC%9D%B8" title="기업인">기업인</a>이다. <a href="/wiki/2003%EB%85%84" title="2003년">2003년</a> <a href="/wiki/9%EC%9B%94_3%EC%9D%BC" title="9월 3일">9월 3일</a>부터 <a href="/wiki/2007%EB%85%84" title="2007년">2007년</a> <a href="/wiki/4%EC%9B%94_11%EC%9D%BC" title="4월 11일">4월 11일</a>까지 내각 총리를 지냈고, 다시 <a href="/wiki/2013%EB%85%84" title="2013년">2013년</a> <a href="/wiki/4%EC%9B%94_1%EC%9D%BC" title="4월 1일">4월 1일</a>부터 현재까지 내각 총리를 역임하고 있다. 그리고 으로, <a href="/wiki/%EC%A1%B0%EC%84%A0%EB%AF%BC%EC%A3%BC%EC%A3%BC%EC%9D%98%EC%9D%B8%EB%AF%BC%EA%B3%B5%ED%99%94%EA%B5%AD_%EA%B5%AD%EB%AC%B4%EC%9C%84%EC%9B%90%ED%9A%8C" title="조선민주주의인민공화국 국무위원회">조선민주주의인민공화국 국무위원회</a> 부위원장과 <a href="/wiki/%EC%A1%B0%EC%84%A0%EB%A1%9C%EB%8F%99%EB%8B%B9_%EC%A4%91%EC%95%99%EC%9C%84%EC%9B%90%ED%9A%8C" title="조선로동당 중앙위원회">조선로동당 중앙위원회</a> 정치국 상무위원을 겸직하고 있다. 기업인으로는 <a href="/wiki/%ED%8F%89%EC%95%88%EB%B6%81%EB%8F%84" title="평안북도">평안북도</a>의 식료품 공장장, 순천비날론연합기업소 지배인을 지냈다.</p>
<p><a href="/wiki/1962%EB%85%84" title="1962년">1962년</a>에 <a href="/wiki/%ED%8F%89%EC%95%88%EB%B6%81%EB%8F%84" title="평안북도">평안북도</a>의 식료품 공장장이 되었고, 그 후 <a href="/wiki/1993%EB%85%84" title="1993년">1993년</a>부터 조선로동당 경공업부 부부장을 맡았고 <a href="/wiki/1998%EB%85%84" title="1998년">1998년</a>부터 화학공업상이 되었다. 당내의 서열은 제6위이다. 2003년 9월 3일 <a href="/wiki/%ED%99%8D%EC%84%B1%EB%82%A8" title="홍성남">홍성남</a> 후임으로 북한내각 국무총리에 임명되었으나 2007년 4월 11일 해임되었다. 해임 사유는‘선군정치’ 시대에 군수경제에 비협조적이라는 이유이며, <a href="/wiki/2007%EB%85%84" title="2007년">2007년</a> <a href="/wiki/4%EC%9B%94" title="4월">4월</a> <a href="/wiki/%ED%8F%89%EC%95%88%EB%82%A8%EB%8F%84" title="평안남도">평안남도</a> <a href="/wiki/%EC%88%9C%EC%B2%9C%EC%8B%9C_(%ED%8F%89%EC%95%88%EB%82%A8%EB%8F%84)" title="순천시 (평안남도)">순천시</a> 순천비날론연합기업소 지배인으로 좌천됐다.<sup class="reference" id="cite_ref-2"><a href="#cite_note-2">[2]</a></sup> 박봉주는 유류 사용 문제를 놓고 군부와 갈등을 빚었다고 한다.<sup class="reference" id="cite_ref-3"><a href="#cite_note-3">[3]</a></sup></p>
<p>하지만 2010년 8월 조선노동당 중앙위원회 경공업부 제1부부장으로 권력의 중심에 복귀하였고, 2012년 4월 경공업부 부장을 거쳐 2013년 4월 내각 총리에 임명되면서 다시 자기의 위치를 되찾았다. 그리고 2016년 5월 9일 권력의 핵심이라 볼 수 있는 조선로동당 정치국 상무위원에 진입하였다. <sup class="reference" id="cite_ref-4"><a href="#cite_note-4">[4]</a></sup></p>
