<h2><span class="mw-headline" id=".ED.99.9C.EB.8F.99">활동</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%B0%95%EC%83%88%EB%B3%84&amp;action=edit&amp;section=2" title="부분 편집: 활동">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>2006년 <a href="/wiki/%EC%97%B0%EC%84%B8%EB%8C%80%ED%95%99%EA%B5%90" title="연세대학교">연세대학교</a> <a class="mw-redirect" href="/wiki/%EC%8B%AC%EB%A6%AC%ED%95%99%EA%B3%BC" title="심리학과">심리학과</a>에 재학하던 중 <a class="mw-redirect" href="/wiki/%EB%A3%A8%EC%8B%9C%EB%93%9C%ED%8F%B4" title="루시드폴">루시드폴</a>의 공연 세션을 시작으로 <a href="/wiki/%EC%97%90%ED%94%BD%ED%95%98%EC%9D%B4" title="에픽하이">에픽하이</a>, <a href="/wiki/%ED%86%A0%EC%9D%B4_(%EB%8C%80%ED%95%9C%EB%AF%BC%EA%B5%AD%EC%9D%98_%EC%9D%8C%EC%95%85_%EA%B7%B8%EB%A3%B9)" title="토이 (대한민국의 음악 그룹)">토이</a> 등의 공연 연주자로 활동하였다. 2008년 10월 15일 첫 미니 앨범 《<a href="/wiki/Diary_(%EB%B0%95%EC%83%88%EB%B3%84%EC%9D%98_%EC%9D%8C%EB%B0%98)" title="Diary (박새별의 음반)">Diary</a>》를 발표하였고, 2010년 3월 정규 1집 앨범이자 본인이 직접 작사/작곡, 편곡, 연주, 프로듀싱을 한《<a href="/wiki/%EC%83%88%EB%B2%BD%EB%B3%84" title="새벽별">새벽별</a>》이 발매되었다.<sup class="reference" id="cite_ref-3"><a href="#cite_note-3">[3]</a></sup></p>
<p>2009년 <a class="mw-redirect" href="/wiki/%EA%B4%91%EC%A3%BC_MBC" title="광주 MBC">광주 MBC</a> 《문화콘서트 난장》을 진행하였으며, <a href="/wiki/%EB%9D%BC%EB%94%94%EC%98%A4" title="라디오">라디오</a> <a href="/wiki/%EB%94%94%EC%8A%A4%ED%81%AC_%EC%9E%90%ED%82%A4" title="디스크 자키">DJ</a>로서 2011년 3월 6일부터 2012년 2월 28일까지 <a href="/wiki/EBS_FM" title="EBS FM">EBS FM</a> 《아름다운 동요세상》을 진행하였다.</p>
<p>2010년 연세대학교 심리학과를 졸업했으며, 2012년 <a href="/wiki/%ED%95%9C%EA%B5%AD%EA%B3%BC%ED%95%99%EA%B8%B0%EC%88%A0%EC%9B%90" title="한국과학기술원">한국과학기술원 문화기술대학원</a>에 입학하였다. 2013년 정규 2집 《<a class="new" href="/w/index.php?title=%ED%95%98%EC%9D%B4%ED%9E%90_(%EC%9D%8C%EB%B0%98)&amp;action=edit&amp;redlink=1" title="하이힐 (음반) (없는 문서)">하이힐</a>》을 발표하였다.</p>
