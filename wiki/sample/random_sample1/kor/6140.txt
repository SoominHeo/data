<h2><span class="mw-headline" id=".EC.B6.95.EA.B5.AC.EC.9D.B8_.EC.83.9D.ED.99.9C">축구인 생활</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%B0%95%EC%84%B1%ED%98%B8_(1982%EB%85%84)&amp;action=edit&amp;section=1" title="부분 편집: 축구인 생활">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<h3><span class="mw-headline" id=".EC.84.A0.EC.88.98_.EC.83.9D.ED.99.9C">선수 생활</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%B0%95%EC%84%B1%ED%98%B8_(1982%EB%85%84)&amp;action=edit&amp;section=2" title="부분 편집: 선수 생활">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p><a href="/wiki/%EB%A7%8C%EC%88%98%EC%A4%91%ED%95%99%EA%B5%90" title="만수중학교">만수중학교</a>, <a href="/wiki/%EB%B6%80%ED%8F%89%EA%B3%A0%EB%93%B1%ED%95%99%EA%B5%90" title="부평고등학교">부평고등학교</a>를 졸업하였다. <a href="/wiki/2001%EB%85%84" title="2001년">2001년</a> <a class="mw-redirect" href="/wiki/%EC%95%88%EC%96%91_LG_%EC%B9%98%ED%83%80%EC%8A%A4" title="안양 LG 치타스">안양 LG 치타스</a> (현 <a href="/wiki/FC_%EC%84%9C%EC%9A%B8" title="FC 서울">FC 서울</a>)에 입단하였고, 3월 28일 열린 <a href="/wiki/%EC%A0%84%EB%82%A8_%EB%93%9C%EB%9E%98%EA%B3%A4%EC%A6%88" title="전남 드래곤즈">전남</a>과의 리그컵 경기에서 데뷔하였다.</p>
<p><a href="/wiki/2004%EB%85%84" title="2004년">2004년</a>에는 군입대를 위하여 <a class="mw-redirect" href="/wiki/%EC%95%88%EC%82%B0_%EA%B2%BD%EC%B0%B0%EC%B2%AD_%EC%B6%95%EA%B5%AC%EB%8B%A8" title="안산 경찰청 축구단">경찰 축구단</a>으로 임대되었다.</p>
<p><a href="/wiki/2006%EB%85%84" title="2006년">2006년</a>, <a href="/wiki/%EB%B6%80%EC%82%B0_%EC%95%84%EC%9D%B4%ED%8C%8C%ED%81%AC" title="부산 아이파크">부산 아이파크</a>에 둥지를 튼 뒤 주전 공격수로 활약하였으나 많은 골을 넣지 못해 비판 받기도 하였다.</p>
<p>이후 <a href="/wiki/2008%EB%85%84" title="2008년">2008년</a>, <a href="/wiki/%EB%8C%80%EC%A0%84_%EC%8B%9C%ED%8B%B0%EC%A6%8C" title="대전 시티즌">대전 시티즌</a>으로 이적한 뒤 <a href="/wiki/K%EB%A6%AC%EA%B7%B8_2008" title="K리그 2008">2008 시즌</a> 24경기에 출전하여 5골을 기록한 것에 이어 <a href="/wiki/K%EB%A6%AC%EA%B7%B8_2009" title="K리그 2009">2009 시즌</a>에는 8골을 기록하며 골을 넣지 못하는 스트라이커의 오명을 벗었다. <a href="/wiki/2010%EB%85%84_7%EC%9B%94" title="2010년 7월">2010년 7월</a>, <a href="/wiki/%EC%9D%BC%EB%B3%B8" title="일본">일본</a> <a href="/wiki/J%EB%A6%AC%EA%B7%B8" title="J리그">J리그</a> <a href="/wiki/%EB%B2%A0%EA%B0%88%ED%83%80_%EC%84%BC%EB%8B%A4%EC%9D%B4" title="베갈타 센다이">베갈타 센다이</a>로 6개월 간 임대되었고 임대 기간동안 센다이에서 9경기에 출전하여 1골 1도움을 기록하였다. 임대 종료 후 원 소속팀인 대전으로 복귀하였고 곧바로 팀의 주장직을 인수받아 <a href="/wiki/K%EB%A6%AC%EA%B7%B8_2011" title="K리그 2011">2011 시즌</a> 28경기에 출전하여 8골을 성공시켰다.</p>
<p><a href="/wiki/2011%EB%85%84_12%EC%9B%94" title="2011년 12월">2011년 12월</a> <a href="/wiki/%ED%8F%AC%ED%95%AD_%EC%8A%A4%ED%8B%B8%EB%9F%AC%EC%8A%A4" title="포항 스틸러스">포항 스틸러스</a>는 <a href="/wiki/%EC%9D%B4%EC%8A%AC%EA%B8%B0_(%EC%B6%95%EA%B5%AC_%EC%84%A0%EC%88%98)" title="이슬기 (축구 선수)">이슬기</a>와 <a href="/wiki/%EA%B9%80%EB%8F%99%ED%9D%AC_(1989%EB%85%84)" title="김동희 (1989년)">김동희</a>을 내주고 현금을 추가 지급하는 조건으로 박성호를 영입하였다고 공식 발표하였다.<sup class="reference" id="cite_ref-1"><a href="#cite_note-1">[1]</a></sup></p>
<p><a href="/wiki/K%EB%A6%AC%EA%B7%B8_2012" title="K리그 2012">2012 시즌</a> 전반기에는 리그에서 공격포인트를 기록하지 못하는 등 부진하여 팬들의 원성을 샀으나, <a href="/wiki/FA%EC%BB%B5_2012" title="FA컵 2012">FA컵 2012</a> 결승에서 연장전 종료 직전 통렬한 헤딩 결승골을 터뜨려 팀의 우승에 공헌하였다. <a href="/wiki/%ED%8F%AC%ED%95%AD_%EC%8A%A4%ED%8B%B8%EB%9F%AC%EC%8A%A4" title="포항 스틸러스">포항 스틸러스</a>에서 보낸 <a href="/wiki/K%EB%A6%AC%EA%B7%B8_2012" title="K리그 2012">2012</a>, <a href="/wiki/K%EB%A6%AC%EA%B7%B8_%ED%81%B4%EB%9E%98%EC%8B%9D_2013" title="K리그 클래식 2013">2013</a> 두 시즌 동안 8월 이후 본격적으로 가동되는 득점력을 보여줘 '가을 남자' 혹은 '가을 전어'라는 별명이 붙기도 했다.<sup class="reference" id="cite_ref-2"><a href="#cite_note-2">[2]</a></sup></p>
<p><a href="/wiki/K%EB%A6%AC%EA%B7%B8_%ED%81%B4%EB%9E%98%EC%8B%9D_2013" title="K리그 클래식 2013">2013 시즌</a> 9월 28일 <a href="/wiki/%EC%9D%B8%EC%B2%9C_%EC%9C%A0%EB%82%98%EC%9D%B4%ED%8B%B0%EB%93%9C_FC" title="인천 유나이티드 FC">인천</a>전과 10월 5일 <a href="/wiki/%EC%88%98%EC%9B%90_%EC%82%BC%EC%84%B1_%EB%B8%94%EB%A3%A8%EC%9C%99%EC%A6%88" title="수원 삼성 블루윙즈">수원</a>전에서는 두 경기 연속 종료 직전의 동점골을 터트리는 활약을 보였고, 12월 1일 <a href="/wiki/%EC%9A%B8%EC%82%B0_%ED%98%84%EB%8C%80_%EC%B6%95%EA%B5%AC%EB%8B%A8" title="울산 현대 축구단">울산</a>과의 리그 마지막 경기에서는 후반 추가시간, <a href="/wiki/%EA%B9%80%EC%9B%90%EC%9D%BC_(%EC%B6%95%EA%B5%AC_%EC%84%A0%EC%88%98)" title="김원일 (축구 선수)">김원일</a>의 극적인 결승골을 어시스트하며 <a href="/wiki/%ED%8F%AC%ED%95%AD_%EC%8A%A4%ED%8B%B8%EB%9F%AC%EC%8A%A4" title="포항 스틸러스">포항 스틸러스</a>의 <a href="/wiki/K%EB%A6%AC%EA%B7%B8_%ED%81%B4%EB%9E%98%EC%8B%9D_2013" title="K리그 클래식 2013">K리그 클래식 2013</a> 우승에 기여하였다.</p>
<p><a href="/wiki/2014%EB%85%84" title="2014년">2014년</a> 1월, <a href="/wiki/J2%EB%A6%AC%EA%B7%B8" title="J2리그">J2리그</a>의 <a href="/wiki/%EC%9A%94%EC%BD%94%ED%95%98%EB%A7%88_FC" title="요코하마 FC">요코하마 FC</a>로 이적하며 1년 계약을 맺었고, 31경기에 나서 3골을 기록하였다.</p>
<p><a href="/wiki/2015%EB%85%84" title="2015년">2015년</a> 1월, <a href="/wiki/%ED%8F%AC%ED%95%AD_%EC%8A%A4%ED%8B%B8%EB%9F%AC%EC%8A%A4" title="포항 스틸러스">포항 스틸러스</a>는 박성호의 재영입을 발표하였고,<sup class="reference" id="cite_ref-3"><a href="#cite_note-3">[3]</a></sup> 2015 시즌 리그 26경기 3골, FA컵 2경기 1골을 기록하였다.</p>
<p><a href="/wiki/2016%EB%85%84" title="2016년">2016년</a> 2월, <a href="/wiki/%EA%B9%80%EC%8B%A0%EC%9A%B1" title="김신욱">김신욱</a>이 <a href="/wiki/%EC%A0%84%EB%B6%81_%ED%98%84%EB%8C%80_%EB%AA%A8%ED%84%B0%EC%8A%A4" title="전북 현대 모터스">전북 현대 모터스</a>로 이적하자 그의 대체자로 <a href="/wiki/%EC%9A%B8%EC%82%B0_%ED%98%84%EB%8C%80_%EC%B6%95%EA%B5%AC%EB%8B%A8" title="울산 현대 축구단">울산 현대</a>로 이적하였다. <sup class="reference" id="cite_ref-4"><a href="#cite_note-4">[4]</a></sup></p>
<h2><span class="mw-headline" id=".ED.94.8C.EB.A0.88.EC.9D.B4_.EC.8A.A4.ED.83.80.EC.9D.BC">플레이 스타일</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%B0%95%EC%84%B1%ED%98%B8_(1982%EB%85%84)&amp;action=edit&amp;section=3" title="부분 편집: 플레이 스타일">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>187cm의 장신이면서도 테크닉과 패스 능력을 갖추었으며, 2선과의 연계플레이 및 측면으로 침투하는 플레이를 즐겨한다.</p>
<h2><span class="mw-headline" id=".EA.B7.B8_.EC.99.B8">그 외</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%B0%95%EC%84%B1%ED%98%B8_(1982%EB%85%84)&amp;action=edit&amp;section=4" title="부분 편집: 그 외">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
