<p><b>박제가</b>(朴齊家, <a href="/wiki/1750%EB%85%84" title="1750년">1750년</a> <a href="/wiki/11%EC%9B%94_5%EC%9D%BC" title="11월 5일">11월 5일</a> ~ <a href="/wiki/1815%EB%85%84" title="1815년">1815년</a><sup class="reference" id="cite_ref-1"><a href="#cite_note-1">[1]</a></sup> <a href="/wiki/4%EC%9B%94_25%EC%9D%BC" title="4월 25일">4월 25일</a>)은 <a class="mw-redirect" href="/wiki/%EC%A1%B0%EC%84%A0_%ED%9B%84%EA%B8%B0" title="조선 후기">조선 후기</a>의 <a href="/wiki/%EC%A0%95%EC%B9%98" title="정치">정치</a>가, 외교관, 통역관, <a href="/wiki/%EC%8B%A4%ED%95%99" title="실학">실학</a>자로 <a class="mw-redirect" href="/wiki/%EB%B6%81%ED%95%99%ED%8C%8C" title="북학파">북학파</a>의 거두이다. 자는 재선(在先), 차수(次修), 수기(修其), 호는 초정(楚亭), 정유(貞否) 또는 정유(貞蕤), 위항도인(葦杭道人)이며, 본관은 <a href="/wiki/%EB%B0%80%EC%96%91_%EB%B0%95%EC%94%A8" title="밀양 박씨">밀양</a>(密陽)이다.<sup class="reference" id="cite_ref-2"><a href="#cite_note-2">[2]</a></sup> <a href="/wiki/%EC%B2%AD%EB%82%98%EB%9D%BC" title="청나라">청나라</a>의 선진 문물 수용과 <a href="/wiki/%EC%A4%91%EC%83%81%EC%A3%BC%EC%9D%98" title="중상주의">중상주의</a> <a href="/wiki/%EA%B2%BD%EC%A0%9C" title="경제">경제</a> 정책을 주장했다.</p>
<p><a href="/wiki/1778%EB%85%84" title="1778년">1778년</a> <a href="/wiki/%EC%B2%AD%EB%82%98%EB%9D%BC" title="청나라">청나라</a>에 <a href="/wiki/%EC%82%AC%EC%9D%80%EC%82%AC" title="사은사">사은사</a>로 파견되는 <a href="/wiki/%EC%B1%84%EC%A0%9C%EA%B3%B5" title="채제공">채제공</a>의 수행원으로 청나라에 다녀왔다. <a href="/wiki/1779%EB%85%84" title="1779년">1779년</a> <a href="/wiki/3%EC%9B%94" title="3월">3월</a> <a class="mw-redirect" href="/wiki/%EC%A0%95%EC%A1%B0" title="정조">정조</a>에 의해 <a href="/wiki/%EA%B7%9C%EC%9E%A5%EA%B0%81" title="규장각">규장각</a><a class="new" href="/w/index.php?title=%EA%B2%80%EC%84%9C%EA%B4%80&amp;action=edit&amp;redlink=1" title="검서관 (없는 문서)">검서관</a>으로 특채되어 이후 <a href="/wiki/%EC%B2%AD%EB%82%98%EB%9D%BC" title="청나라">청나라</a>에 사신이 파견될 때 사신의 수행원으로 다녀왔다. 그 뒤 <a class="new" href="/w/index.php?title=%EC%A0%84%EC%84%A4%EC%84%9C&amp;action=edit&amp;redlink=1" title="전설서 (없는 문서)">전설서</a> 별제와 <a href="/wiki/1791%EB%85%84" title="1791년">1791년</a> <a href="/wiki/%EA%B5%B0%EA%B8%B0%EC%8B%9C" title="군기시">임시 군기시정</a>, <a href="/wiki/1792%EB%85%84" title="1792년">1792년</a> <a href="/wiki/%EA%B5%B0%EA%B8%B0%EC%8B%9C" title="군기시">군기시정</a> 등을 거쳐 <a href="/wiki/1794%EB%85%84" title="1794년">1794년</a>(정조 27년) <a href="/wiki/2%EC%9B%94" title="2월">2월</a> <a class="mw-redirect" href="/wiki/%EA%B3%BC%EA%B1%B0_%EC%A0%9C%EB%8F%84" title="과거 제도">춘당대 무과</a>(春塘臺武科)에 장원으로 급제, <a class="new" href="/w/index.php?title=%EC%98%A4%EC%9C%84%EC%9E%A5&amp;action=edit&amp;redlink=1" title="오위장 (없는 문서)">오위장</a>, <a class="mw-disambig" href="/wiki/%EC%96%91%ED%8F%89" title="양평">양평</a><a href="/wiki/%ED%98%84%EA%B0%90" title="현감">현감</a>, <a class="mw-disambig" href="/wiki/%EC%98%81%ED%8F%89" title="영평">영평</a><a href="/wiki/%ED%98%84%EB%A0%B9" title="현령">현령</a>, <a href="/wiki/%EB%B6%80%EC%97%AC" title="부여">부여</a><a href="/wiki/%ED%98%84%EA%B0%90" title="현감">현감</a> 등을 역임하였다. 상행위와 <a class="mw-redirect" href="/wiki/%EB%AC%B4%EC%97%AD" title="무역">무역</a>을 적극 장려하고 밀무역에 대한 제재를 줄이며, <a class="mw-redirect" href="/wiki/%ED%99%94%ED%8F%90" title="화폐">화폐</a>를 유통할 것, <a class="new" href="/w/index.php?title=%EC%84%9C%EC%96%91%EC%9D%B8&amp;action=edit&amp;redlink=1" title="서양인 (없는 문서)">서양인</a>들을 <a href="/wiki/%EC%A1%B0%EC%84%A0" title="조선">조선</a>으로 초빙하여 화포 제작, 성곽 축조, 선박 건조, 양잠 등의 신기술을 적극 도입, 유치하자고 주장했다. 또한 맹목적인 근검절약은 병폐이며 상품<a class="new" href="/w/index.php?title=%ED%99%94%ED%8F%90_%EA%B2%BD%EC%A0%9C&amp;action=edit&amp;redlink=1" title="화폐 경제 (없는 문서)">화폐 경제</a>의 발전이라는 현실을 인정한 기반을 쌓고 상업·수공업·농업 전반의 생산력 발전을 적극적으로 추진하고 국가경제를 일으킬 것을 역설하였다. 그러나 <a href="/wiki/%EC%83%81%EC%97%85" title="상업">상업</a>과 <a class="mw-redirect" href="/wiki/%EB%AC%B4%EC%97%AD" title="무역">무역</a>을 천시 여기던 당대의 사대부들에 의해 비판받게 된다.</p>
<p>박제가는 네 차례에 걸친 <a href="/wiki/%EC%B2%AD%EB%82%98%EB%9D%BC" title="청나라">청나라</a> 사행(使行)을 통해 100명이 넘는 <a href="/wiki/%EC%A4%91%EA%B5%AD" title="중국">중국</a> 지식인들과 교유하면서 국제적 안목을 갖춘 글로벌 지식인이었다.<sup class="reference" id="cite_ref-cho01_3-0"><a href="#cite_note-cho01-3">[3]</a></sup> <a class="mw-disambig" href="/wiki/%EB%B0%95%EC%A7%80%EC%9B%90" title="박지원">박지원</a>과 <a class="new" href="/w/index.php?title=%EC%9D%B4%EA%B4%80%EC%83%81&amp;action=edit&amp;redlink=1" title="이관상 (없는 문서)">이관상</a>의 문하에서 수학하였고 <a href="/wiki/%EC%A1%B0%EC%84%A0_%EC%A0%95%EC%A1%B0" title="조선 정조">정조</a>와 <a class="new" href="/w/index.php?title=%EC%9C%A4%ED%96%89%EC%9E%84&amp;action=edit&amp;redlink=1" title="윤행임 (없는 문서)">윤행임</a> 등의 배려로 <a href="/wiki/%EC%B2%AD%EB%82%98%EB%9D%BC" title="청나라">청나라</a>에 사신이 파견될 때 수행원으로 여러번 다녀왔다. <a href="/wiki/%EC%A1%B0%EC%84%A0_%EC%A0%95%EC%A1%B0" title="조선 정조">정조</a>의 서얼 중용 정책에 의해 <a href="/wiki/%EA%B7%9C%EC%9E%A5%EA%B0%81" title="규장각">규장각</a>의 검서관으로 등용되었으나 정조 사후 <a href="/wiki/1801%EB%85%84" title="1801년">1801년</a>(순조 1년) 그와 친분이 있던 <a class="new" href="/w/index.php?title=%EC%9C%A4%ED%96%89%EC%9E%84&amp;action=edit&amp;redlink=1" title="윤행임 (없는 문서)">윤행임</a>이 <a href="/wiki/%EB%85%B8%EB%A1%A0" title="노론">노론</a><a href="/wiki/%EB%B2%BD%ED%8C%8C" title="벽파">벽파</a>의 공격을 받고 몰락하면서 유배되었다. 원래 당색은 <a class="mw-redirect" href="/wiki/%EC%86%8C%EB%B6%81" title="소북">소북</a>(小北)이었으나 <a href="/wiki/%EB%85%B8%EB%A1%A0" title="노론">노론</a><a class="mw-redirect" href="/wiki/%EB%B6%81%ED%95%99%ED%8C%8C" title="북학파">북학파</a>로 전향하였다. 연암 <a class="mw-disambig" href="/wiki/%EB%B0%95%EC%A7%80%EC%9B%90" title="박지원">박지원</a>, <a class="new" href="/w/index.php?title=%EC%9D%B4%EA%B4%80%EC%83%81&amp;action=edit&amp;redlink=1" title="이관상 (없는 문서)">이관상</a>의 문인이다.</p>

<h2><span class="mw-headline" id=".EC.83.9D.EC.95.A0">생애</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%B0%95%EC%A0%9C%EA%B0%80&amp;action=edit&amp;section=1" title="부분 편집: 생애">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<h3><span class="mw-headline" id=".EC.83.9D.EC.95.A0_.EC.B4.88.EA.B8.B0">생애 초기</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%B0%95%EC%A0%9C%EA%B0%80&amp;action=edit&amp;section=2" title="부분 편집: 생애 초기">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<h4><span class="mw-headline" id=".EC.B6.9C.EC.83.9D.EA.B3.BC_.EA.B0.80.EA.B3.84">출생과 가계</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%B0%95%EC%A0%9C%EA%B0%80&amp;action=edit&amp;section=3" title="부분 편집: 출생과 가계">편집</a><span class="mw-editsection-bracket">]</span></span></h4>
<p>초정 박제가는 <a href="/wiki/1750%EB%85%84" title="1750년">1750년</a> <a href="/wiki/%EC%8A%B9%EC%A0%95%EC%9B%90" title="승정원">승정원</a><a class="new" href="/w/index.php?title=%EC%9A%B0%EB%B6%80%EC%8A%B9%EC%A7%80&amp;action=edit&amp;redlink=1" title="우부승지 (없는 문서)">우부승지</a>를 지낸 박평(朴坪)의 <a class="mw-redirect" href="/wiki/%EC%84%9C%EC%9E%90" title="서자">서자</a>로 <a href="/wiki/%ED%95%9C%EC%84%B1%EB%B6%80" title="한성부">한성부</a>(지금의 <a class="mw-redirect" href="/wiki/%EC%84%9C%EC%9A%B8" title="서울">서울</a>)에서 태어났다. 그의 선대는 <a href="/wiki/%EA%B3%A0%EB%A0%A4" title="고려">고려</a> <a class="mw-redirect" href="/wiki/%EC%B6%A9%EB%A0%AC%EC%99%95" title="충렬왕">충렬왕</a> 때 내부시승(內府侍丞)을 지낸 충헌공 박척(朴陟)의 후손이었다. 그의 가계는 <a class="mw-redirect" href="/wiki/%EC%86%8C%EB%B6%81" title="소북">소북</a> 계열로 박율(朴栗)의 6대손으로, 5대조 박심(朴尋)은 <a href="/wiki/%EC%82%AC%ED%97%8C%EB%B6%80" title="사헌부">사헌부</a><a class="mw-disambig" href="/wiki/%EC%9E%A5%EB%A0%B9" title="장령">장령</a>을 지냈고, 고조 할아버지 박수문(朴守文) 역시 <a href="/wiki/%EC%82%AC%ED%97%8C%EB%B6%80" title="사헌부">사헌부</a><a class="mw-disambig" href="/wiki/%EC%9E%A5%EB%A0%B9" title="장령">장령</a>을 지냈으며 증조할아버지 박순(朴純)은 <a href="/wiki/%ED%99%A9%ED%95%B4%EB%8F%84" title="황해도">황해도</a><a href="/wiki/%EA%B4%80%EC%B0%B0%EC%82%AC" title="관찰사">관찰사</a>를 지냈고, 할아버지 태동(台東)은 <a class="new" href="/w/index.php?title=%EC%84%B8%EC%9E%90%EC%8B%9C%EA%B0%95%EC%9B%90&amp;action=edit&amp;redlink=1" title="세자시강원 (없는 문서)">세자시강원</a><a class="new" href="/w/index.php?title=%ED%95%84%EC%84%A0&amp;action=edit&amp;redlink=1" title="필선 (없는 문서)">필선</a>(侍講院弼善)을 역임했으며 아버지 <a class="new" href="/w/index.php?title=%EB%B0%95%ED%8F%89&amp;action=edit&amp;redlink=1" title="박평 (없는 문서)">박평</a>은 <a href="/wiki/%EC%8A%B9%EC%A0%95%EC%9B%90" title="승정원">승정원</a><a class="new" href="/w/index.php?title=%EC%9A%B0%EB%B6%80%EC%8A%B9%EC%A7%80&amp;action=edit&amp;redlink=1" title="우부승지 (없는 문서)">우부승지</a>에 이르렀다. 그러나 그의 어머니는 첩이었고, 그는 서자였다.</p>
<p>기억력이 좋고 암기에 능했던 그는 어릴적부터 글을 좋아해 읽은 책은 반드시 세 번씩 베껴 썼고, 입에는 늘 붓을 물고 있었다고 해요. 그리고 변소에가면 그 옆 모래에 그림을 그렸고, 앉아서는 허공에 글쓰기를 연습했다 한다. 훗날 박제가 자신의 회상에 의하면 "내가 글을 처음 배운것은 막 젖을 먹던 때였지"라는 시구를 남기기도 하였다.</p>
<p>서자라는 천대를 받았음에도 아버지 <a class="new" href="/w/index.php?title=%EB%B0%95%ED%8F%89&amp;action=edit&amp;redlink=1" title="박평 (없는 문서)">박평</a>은 만년에 얻은 서자인 그에게 각별하였다. 그러나 <a href="/wiki/%EC%95%84%EB%B2%84%EC%A7%80" title="아버지">아버지</a>가 11세 때 죽고 본댁에서 나오게 되면서 가난해졌으나 <a href="/wiki/%EC%8B%9C_(%EB%AC%B8%ED%95%99)" title="시 (문학)">시</a><a href="/wiki/%EB%AC%B8%ED%95%99" title="문학">문</a>과 <a class="mw-redirect" href="/wiki/%EA%B7%B8%EB%A6%BC" title="그림">그림</a>에 능해 유명해졌다. 그의 가계는 당색으로는 <a class="mw-redirect" href="/wiki/%EC%86%8C%EB%B6%81" title="소북">소북</a>이었으나 박제가는 <a class="mw-disambig" href="/wiki/%EB%B0%95%EC%A7%80%EC%9B%90" title="박지원">박지원</a>의 문인이 되면서 <a href="/wiki/%EB%85%B8%EB%A1%A0" title="노론">노론</a> <a class="mw-redirect" href="/wiki/%EB%B6%81%ED%95%99%ED%8C%8C" title="북학파">북학파</a>로 당적을 옮겼다.</p>
<h4><span class="mw-headline" id=".EC.86.8C.EB.85.84.EA.B8.B0">소년기</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%B0%95%EC%A0%9C%EA%B0%80&amp;action=edit&amp;section=4" title="부분 편집: 소년기">편집</a><span class="mw-editsection-bracket">]</span></span></h4>
<p>일찍부터 시인으로 이름을 날렸고 그의 글재주를 알아본 아버지 <a class="new" href="/w/index.php?title=%EB%B0%95%ED%8F%89&amp;action=edit&amp;redlink=1" title="박평 (없는 문서)">박평</a>은 그가 서자임을 상당히 애석해했다 한다. <a href="/wiki/1760%EB%85%84" title="1760년">1760년</a> 그들 모자를 돌보아주던 아버지 <a class="new" href="/w/index.php?title=%EB%B0%95%ED%8F%89&amp;action=edit&amp;redlink=1" title="박평 (없는 문서)">박평</a>이 사망하자 <a href="/wiki/%ED%95%9C%EC%84%B1%EB%B6%80" title="한성부">한성부</a> 본댁에서 나와 거처를 자주 옮겨다니며 어머니가 생계를 이어갈 정도로 생활이 매우 어려웠다. 그러나 박제가는 밤을 새워가며 품풀이하여 공부시키는 어머니의 지극 정성을 가슴깊이 새겼다. 그의 어머니는 아들의 장래를 위하여 모든것을 바쳤다. 박제가는 당시의 생활을 회상하여 다음과 같이 썼다.</p>

<td style="padding:10px; vertical-align: top;">돌아가신 어머니를 생각해본다. 과부로 가난하게 살면서 10여년동안 성한 옷을 입어보지 못했고 입에 맞는 음식을 자셔보지도 못했으며 밤을 새워가며 삯바느질을 하여 공부하는 이 아들의 뒤바라지를 하였다. 내가 사귀는 사람중에는 이따금 선생과 나이 든 분, 그리고 세상에 알려진 인사들이 많이 있었는데 반드시 그들을 초청케 하여 술과 안주를 극진히 대접한 관계로 그 아들을 대하는 사람으로서는 집안형편이 빈한한것을 모를 정도였다.</td>

<p>이와 같은 사회적환경과 가정정황은 그가 커서 사회적 천대와 멸시, 양반 제도와 계급적 모순에 대해 불만을 갖고 비판하며 빈곤한 농민과 서민을 동정하는 입장에 서게 하는데 대해 큰 영향을 주었다. 한편 우연한 기회에 연암 <a class="mw-disambig" href="/wiki/%EB%B0%95%EC%A7%80%EC%9B%90" title="박지원">박지원</a>(朴趾源)을 만나 그의 문하에 출입하게 된다. 또한 <a href="/wiki/%EC%9D%B4%EC%88%9C%EC%8B%A0" title="이순신">이순신</a>의 5대손인 선비 <a class="new" href="/w/index.php?title=%EC%9D%B4%EA%B4%80%EC%83%81&amp;action=edit&amp;redlink=1" title="이관상 (없는 문서)">이관상</a>(李觀祥)의 문하에도 출입하는데, 그의 학문에 대한 열정과 재주를 아깝게 여긴 이관상은 <a href="/wiki/1766%EB%85%84" title="1766년">1766년</a> 자신의 첩이 낳은 둘째 서녀를 박제가에게 출가시켜 서녀사위로 삼는다.</p>
<h3><span class="mw-headline" id=".ED.95.99.EB.AC.B8_.EC.97.B0.EA.B5.AC.EC.99.80_.EA.B0.9C.ED.98.81.EC.9A.B4.EB.8F.99">학문 연구와 개혁운동</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%B0%95%EC%A0%9C%EA%B0%80&amp;action=edit&amp;section=5" title="부분 편집: 학문 연구와 개혁운동">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<h4><span class="mw-headline" id=".EC.88.98.ED.95.99.EA.B3.BC_.ED.95.99.EB.AC.B8_.EC.97.B0.EA.B5.AC">수학과 학문 연구</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%B0%95%EC%A0%9C%EA%B0%80&amp;action=edit&amp;section=6" title="부분 편집: 수학과 학문 연구">편집</a><span class="mw-editsection-bracket">]</span></span></h4>
<p><a href="/wiki/1766%EB%85%84" title="1766년">1766년</a> 16세에 <a class="new" href="/w/index.php?title=%EC%9D%B4%EA%B4%80%EC%83%81&amp;action=edit&amp;redlink=1" title="이관상 (없는 문서)">이관상</a>의 서녀와 혼인하고 한때 그의 집에서 거주하기도 했다. 비록 서출이고, 서녀의 남편이었지만 <a class="new" href="/w/index.php?title=%EC%9D%B4%EA%B4%80%EC%83%81&amp;action=edit&amp;redlink=1" title="이관상 (없는 문서)">이관상</a>은 그를 자신의 집에서 생활하고 독서를 지원할만큼 그를 아꼈다. 이관상은 그가 서자임을 애석히 여겼다. <a class="new" href="/w/index.php?title=%EC%9D%B4%EA%B4%80%EC%83%81&amp;action=edit&amp;redlink=1" title="이관상 (없는 문서)">이관상</a>은 서녀 내외를 자신의 집에 특별히 거주하게 했고, 그에게도 계속 자신의 서실에 출입하게 하여 <a href="/wiki/%EC%84%B1%EB%A6%AC%ED%95%99" title="성리학">성리학</a>과 글을 가르쳤다.</p>
<p><a href="/wiki/1769%EB%85%84" title="1769년">1769년</a> 19세 때 <a class="mw-redirect" href="/wiki/%EB%B0%95%EC%A7%80%EC%9B%90_(%EC%8B%A4%ED%95%99%EC%9E%90)" title="박지원 (실학자)">박지원</a>을 스승으로 모시면서 많은 <a href="/wiki/%EC%9D%B4%EB%8D%95%EB%AC%B4" title="이덕무">이덕무</a>·<a href="/wiki/%EC%9C%A0%EB%93%9D%EA%B3%B5" title="유득공">유득공</a>·<a class="mw-disambig" href="/wiki/%EC%9D%B4%EC%84%9C%EA%B5%AC" title="이서구">이서구</a> 등 많은 실학자와 만났다. 이중 이덕무와는 절친한 벗이었다.<sup class="reference" id="cite_ref-4"><a href="#cite_note-4">[4]</a></sup></p>

<p>신분과 사람을 차별하지 않는 <a class="mw-disambig" href="/wiki/%EB%B0%95%EC%A7%80%EC%9B%90" title="박지원">박지원</a> 문하에서 많은 동료들을 만났고 <a href="/wiki/%EC%9D%B4%EB%8D%95%EB%AC%B4" title="이덕무">이덕무</a>는 그의 절친한 친구로 그의 재능을 알아주었다. 이때 만나 교류했던 인물들 중 <a href="/wiki/%ED%99%8D%EB%8C%80%EC%9A%A9" title="홍대용">홍대용</a>은 후일 박제가의 문하생이 되는 <a href="/wiki/%EA%B9%80%EC%A0%95%ED%9D%AC" title="김정희">김정희</a>의 장인 홍담용의 사촌간이 된다. 그러나 그는 늘 고민했다. 장인 <a class="new" href="/w/index.php?title=%EC%9D%B4%EA%B4%80%EC%83%81&amp;action=edit&amp;redlink=1" title="이관상 (없는 문서)">이관상</a>과 사람을 가리지 않던 연암 <a class="mw-redirect" href="/wiki/%EB%B0%95%EC%A7%80%EC%9B%90_(%EC%8B%A4%ED%95%99%EC%9E%90)" title="박지원 (실학자)">박지원</a>의 배려로 전통적인 양반 교육을 받았지만, 서자라는 신분적인 제약으로 사회적인 차별대우를 받았기 때문에 봉건적인 신분제도에 반대하는 사상을 전개하였다. <a href="/wiki/%EB%82%A8%EC%9D%B8" title="남인">남인</a>인 <a href="/wiki/%EC%A0%95%EC%95%BD%EC%9A%A9" title="정약용">정약용</a>과도 친교를 맺고 교류하였다.</p>
<p>청년기에 그는 <a class="mw-redirect" href="/wiki/%EB%B0%95%EC%A7%80%EC%9B%90_(%EC%8B%A4%ED%95%99%EC%9E%90)" title="박지원 (실학자)">박지원</a>의 문하에서 가르침을 받은 것 외에도 다양한 분야의 저서를 독자적으로도 탐독하였다. 그는 <a href="/wiki/%EC%9C%A0%ED%98%95%EC%9B%90" title="유형원">유형원</a>이나 <a href="/wiki/%EC%9D%B4%EC%9D%B5" title="이익">이익</a> 등의 토지경제사상과 <a class="new" href="/w/index.php?title=%EC%A4%91%EB%86%8D%EC%82%AC%EC%83%81&amp;action=edit&amp;redlink=1" title="중농사상 (없는 문서)">중농사상</a>을 비판하고, 선진적인 <a href="/wiki/%EC%B2%AD%EB%82%98%EB%9D%BC" title="청나라">청나라</a>의 문물을 받아들일 것과, 상공업을 천시하지 말고 국가가 상공업을 발전시키고 무역을 장려해야 한다고 주장하였다. 또 그는 <a class="new" href="/w/index.php?title=%EC%83%81%EA%B3%B5%EC%97%85&amp;action=edit&amp;redlink=1" title="상공업 (없는 문서)">상공업</a>의 발전을 위하여 국가는 수레(車)를 쓸 수 있도록 길을 내어야 하고 <a class="mw-redirect" href="/wiki/%ED%99%94%ED%8F%90" title="화폐">화폐</a> 사용을 활성화해야 한다고 생각했다.</p>
<p><a href="/wiki/1773%EB%85%84" title="1773년">1773년</a>(영조 50년) <a href="/wiki/3%EC%9B%94" title="3월">3월</a> <a class="mw-redirect" href="/wiki/%EC%9D%8C%EC%84%9C_%EC%A0%9C%EB%8F%84" title="음서 제도">음서</a>로 출사하려 하였으나 서얼이라 하여 관직에 오르지 못했다. <a href="/wiki/1776%EB%85%84" title="1776년">1776년</a>(정조 즉위년) <a href="/wiki/%EC%9D%B4%EB%8D%95%EB%AC%B4" title="이덕무">이덕무</a>·<a href="/wiki/%EC%9C%A0%EB%93%9D%EA%B3%B5" title="유득공">유득공</a>·<a class="mw-disambig" href="/wiki/%EC%9D%B4%EC%84%9C%EA%B5%AC" title="이서구">이서구</a> 등 세 사람과 합작(合作)한 시집 《<a class="new" href="/w/index.php?title=%EA%B1%B4%EC%97%B0%EC%A7%91&amp;action=edit&amp;redlink=1" title="건연집 (없는 문서)">건연집</a>(巾衍集)》이 <a href="/wiki/%EC%B2%AD%EB%82%98%EB%9D%BC" title="청나라">청나라</a>에 소개되어 <a href="/wiki/%EC%A1%B0%EC%84%A0" title="조선">조선</a>의 <a class="new" href="/w/index.php?title=%EC%8B%9C%EB%AC%B8_%EC%82%AC%EB%8C%80%EA%B0%80&amp;action=edit&amp;redlink=1" title="시문 사대가 (없는 문서)">시문 사대가</a>(詩文四大家) 중 한 사람으로 알려졌다.</p>
<h4><span class="mw-headline" id=".EA.B4.80.EB.A3.8C.EC.83.9D.ED.99.9C_.EC.B4.88.EB.B0.98">관료생활 초반</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%B0%95%EC%A0%9C%EA%B0%80&amp;action=edit&amp;section=7" title="부분 편집: 관료생활 초반">편집</a><span class="mw-editsection-bracket">]</span></span></h4>
<p><a href="/wiki/1778%EB%85%84" title="1778년">1778년</a>(정조 2년)인 29세 때는 <a href="/wiki/%EC%B2%AD%EB%82%98%EB%9D%BC" title="청나라">청나라</a>에 파견되는 <a href="/wiki/%EC%82%AC%EC%9D%80%EC%82%AC" title="사은사">사은사</a> <a href="/wiki/%EC%B1%84%EC%A0%9C%EA%B3%B5" title="채제공">채제공</a>의 수행원으로 따라가 청나라의 <a href="/wiki/%EC%9D%B4%EC%A1%B0%EC%9B%90" title="이조원">이조원</a>(李調元), <a class="new" href="/w/index.php?title=%EB%B0%98%EC%A0%95%EA%B7%A0&amp;action=edit&amp;redlink=1" title="반정균 (없는 문서)">반정균</a>(潘庭筠) 등의 학자들과 학문을 교류하고 되돌아왔다. 귀국 직후 그는 도구의 개량과 <a href="/wiki/%EC%82%AC%ED%9A%8C" title="사회">사회</a>, <a href="/wiki/%EC%A0%95%EC%B9%98" title="정치">정치</a> 제도의 개혁에 관한 내용인 《<a href="/wiki/%EB%B6%81%ED%95%99%EC%9D%98" title="북학의">북학의</a>》 내외편을 저술하기 시작했는데, <a href="/wiki/%EB%A7%B9%EC%9E%90" title="맹자">맹자</a>를 일부 인용하였고, 실사구시(實事求是)의 사상을 토대로 내편(內篇)에서는 실생활에서의 기구와 시설의 개선을 다루고, 외편(外篇)에서는 정치·사회제도의 전반적인 모순점을 지적하여 서정(庶政)의 개혁 방안을 서술했다.</p>
<p>그는 <a class="mw-redirect" href="/wiki/%EC%A0%95%EC%A1%B0" title="정조">정조</a>에게 국력의 부강을 위해서는 교역로를 열어야 되고, <a href="/wiki/%EC%B2%AD%EB%82%98%EB%9D%BC" title="청나라">청나라</a>의 문물 제도를 받아들이며, 생산 기술과 도구를 개선하고 상업을 장려하여 대외 무역을 조정에서 장려해야 함을 건의하였다. 그러나 그의 견해는 중신들에 의해 당치않은 소리라는 면박을 당하였다.</p>
<p><a href="/wiki/1779%EB%85%84" title="1779년">1779년</a>(정조 3년) <a href="/wiki/3%EC%9B%94" title="3월">3월</a> <a href="/wiki/%EC%A1%B0%EC%84%A0_%EC%A0%95%EC%A1%B0" title="조선 정조">정조</a>는 규장각에 검서관직(檢書官職)을 설치하고 특명으로 <a href="/wiki/%EA%B7%9C%EC%9E%A5%EA%B0%81" title="규장각">규장각</a><a class="new" href="/w/index.php?title=%EA%B2%80%EC%84%9C%EA%B4%80&amp;action=edit&amp;redlink=1" title="검서관 (없는 문서)">검서관</a>에 임명되었다. 이때 그를 비롯한 <a href="/wiki/%EC%9D%B4%EB%8D%95%EB%AC%B4" title="이덕무">이덕무</a>·<a href="/wiki/%EC%9C%A0%EB%93%9D%EA%B3%B5" title="유득공">유득공</a>·<a class="new" href="/w/index.php?title=%EC%84%9C%EC%9D%B4%EC%88%98&amp;action=edit&amp;redlink=1" title="서이수 (없는 문서)">서이수</a>(徐理修) 등의 서얼 출신 학자들이 등용되었다. 숙직을 자청하여 책을 읽다가 잠든 그를 보고 감동한 정조는 격려의 차원에서 약식을 내리고, 그에게 자신의 담요를 덥혀주기도 했다. 이때부터 규장각 내·외직에 근무하면서 여기에 비장된 서적들을 마음껏 탐독하고, 정조를 비롯한 국내의 저명한 학자들과 깊이 사귀면서 왕명을 받아 많은 책을 교정, 간행하기도 하였다.</p>
<p><a href="/wiki/1779%EB%85%84" title="1779년">1779년</a> 서자였으나 적자와 서자를 가리지 않고 등용하려는 <a href="/wiki/%EC%A1%B0%EC%84%A0_%EC%A0%95%EC%A1%B0" title="조선 정조">정조</a>의 방침에 따라 <a href="/wiki/%EA%B7%9C%EC%9E%A5%EA%B0%81" title="규장각">규장각</a> 검서관 등 여러 벼슬을 거치면서 많은 저서를 썼다. <a href="/wiki/1786%EB%85%84" title="1786년">1786년</a> <a href="/wiki/%EC%9D%8C%EB%A0%A5_1%EC%9B%94" title="음력 1월">음력 1월</a> 박제가는 조선 사회를 뒤흔들 만한 장문의 개혁책을 <a href="/wiki/%EC%A1%B0%EC%84%A0_%EC%A0%95%EC%A1%B0" title="조선 정조">정조</a>에게 올렸다.<sup class="reference" id="cite_ref-6"><a href="#cite_note-6">[6]</a></sup></p>
<p><a href="/wiki/1785%EB%85%84" title="1785년">1785년</a>(정조 9년) <a class="new" href="/w/index.php?title=%EC%A0%84%EC%84%A4%EC%84%9C&amp;action=edit&amp;redlink=1" title="전설서 (없는 문서)">전설서</a>별제(典設署別提)가 되었다. 그는 사직하려 하였으나 <a href="/wiki/%EC%A1%B0%EC%84%A0_%EC%A0%95%EC%A1%B0" title="조선 정조">정조</a>의 간곡한 만류로 취임하였다. 이때 그는 시정 개혁 상소를 올려 서자들 중에도 유능한 인재가 있으나 출사를 막는 것은 부당하다며 <a class="mw-redirect" href="/wiki/%EC%84%9C%EC%9E%90" title="서자">서자</a>들의 허통을 상소하였다. 주변의 완강한 반대가 있었지만 <a href="/wiki/%EC%A1%B0%EC%84%A0_%EC%A0%95%EC%A1%B0" title="조선 정조">정조</a>는 그의 상소를 받아들였다.</p>
<p><a href="/wiki/1786%EB%85%84" title="1786년">1786년</a>(정조 10년) <a class="mw-redirect" href="/wiki/%EC%A0%95%EC%A1%B0" title="정조">정조</a>가 왕명으로 관리들에게 시정의 폐단을 고칠 방안을 구언하는 한편 폐단을 고칠 구폐책(救弊策)을 올리게 했을 때, <a class="new" href="/w/index.php?title=%EC%A0%84%EC%84%A4%EC%84%9C%EB%B3%84%EC%A0%9C&amp;action=edit&amp;redlink=1" title="전설서별제 (없는 문서)">전설서별제</a>의 직에 있으면서 '병오소회 丙午所懷'를 상소로 올렸다. 여기서 그는 상공업 장려, 신분차별 타파, 해외통상, 서양인 <a href="/wiki/%EC%84%A0%EA%B5%90%EC%82%AC" title="선교사">선교사</a>의 초청, 과학기술교육의 진흥 등 국가를 부강하게 하고 국민의 생활을 향상시킬 것을 건의하였다. 그러나 그의 건의는 당시 지배층의 이해와는 상반된 것이었으므로 묵살되었으며, 오히려 <a href="/wiki/%EB%85%B8%EB%A1%A0" title="노론">노론</a> <a href="/wiki/%EB%B2%BD%ED%8C%8C" title="벽파">벽파</a> 세력의 심한 반발과 비판을 받았다. 또한 <a href="/wiki/%EB%85%B8%EB%A1%A0" title="노론">노론</a>은 그가 다른 마음을 먹고 있다고 공격, 당시의 심한 당쟁에 휘말려 비판을 받고 급기야 '문체반정'(文體反正)이라는 사상정화운동의 원인의 하나가 된다.</p>
<h4><span class="mw-headline" id=".EC.99.B8.EA.B5.90_.ED.99.9C.EB.8F.99.EA.B3.BC_.EA.B7.80.EA.B5.AD">외교 활동과 귀국</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%B0%95%EC%A0%9C%EA%B0%80&amp;action=edit&amp;section=8" title="부분 편집: 외교 활동과 귀국">편집</a><span class="mw-editsection-bracket">]</span></span></h4>
<p><a href="/wiki/1790%EB%85%84" title="1790년">1790년</a>(정조 14년) <a href="/wiki/5%EC%9B%94" title="5월">5월</a> <a href="/wiki/%EA%B1%B4%EB%A5%AD%EC%A0%9C" title="건륭제">건륭제</a>의 팔순 잔치를 축하하는 진하사절이 파견될 때 박제가는 진하사(進賀使) <a href="/wiki/%ED%99%A9%EC%9D%B8%EC%A0%90" title="황인점">황인점</a>(黃仁點), 부사 서호수(徐浩修)의 수행원으로 <a href="/wiki/%EC%9C%A0%EB%93%9D%EA%B3%B5" title="유득공">유득공</a> 등과 함께 <a href="/wiki/%EC%B2%AD%EB%82%98%EB%9D%BC" title="청나라">청나라</a>에 갔다.</p>
<p><a href="/wiki/%EA%B1%B4%EB%A5%AD%EC%A0%9C" title="건륭제">건륭제</a>의 팔순을 축하하고 돌아오던 도중 <a href="/wiki/%EC%A1%B0%EC%84%A0_%EC%88%9C%EC%A1%B0" title="조선 순조">원자</a>(元子, 뒤의 <a href="/wiki/%EC%A1%B0%EC%84%A0_%EC%88%9C%EC%A1%B0" title="조선 순조">순조</a>)의 탄생을 축하해준 <a href="/wiki/%EA%B1%B4%EB%A5%AD%EC%A0%9C" title="건륭제">건륭제</a>의 호의에 보답하고자 한 <a href="/wiki/%EC%A1%B0%EC%84%A0_%EC%A0%95%EC%A1%B0" title="조선 정조">정조</a>의 특명으로 정3품 <a href="/wiki/%EA%B5%B0%EA%B8%B0%EC%8B%9C" title="군기시">군기시</a>정(軍器寺正)에 임시로 임명되어 다시 말머리를 돌려 <a href="/wiki/%EC%97%B0%EA%B2%BD" title="연경">연경</a>에 다녀왔다. <a href="/wiki/1791%EB%85%84" title="1791년">1791년</a>(정조 15년) 귀국 후 <a href="/wiki/%EA%B5%B0%EA%B8%B0%EC%8B%9C" title="군기시">군기시정</a>(軍器寺正)이 되었다. 그해 겨울 <a href="/wiki/%EC%B2%AD%EB%82%98%EB%9D%BC" title="청나라">청나라</a>에 <a href="/wiki/%EB%8F%99%EC%A7%80%EC%82%AC" title="동지사">동지사</a>(冬至使)가 파견되자 <a href="/wiki/%EB%8F%99%EC%A7%80%EC%82%AC" title="동지사">동지사</a>를 수행하여 다시 <a href="/wiki/%EC%97%B0%EA%B2%BD" title="연경">연경</a>에 다녀왔다.</p>
<p><a href="/wiki/1792%EB%85%84" title="1792년">1792년</a> <a href="/wiki/%EB%B6%80%EC%97%AC" title="부여">부여</a><a href="/wiki/%ED%98%84%EA%B0%90" title="현감">현감</a>으로 나갔다가 <a href="/wiki/1793%EB%85%84" title="1793년">1793년</a> <a href="/wiki/%EC%8A%B9%EC%A0%95%EC%9B%90" title="승정원">승정원</a>에서 보낸 내각관문(內閣關文)을 받고 '비옥희음송 比屋希音頌'이라는 비속한 문체를 쓰는 데 대한 반성문인 자송문(自訟文)을 왕에게 지어바쳤다. <a href="/wiki/1794%EB%85%84" title="1794년">1794년</a>(정조 27년) <a href="/wiki/2%EC%9B%94" title="2월">2월</a> <a class="mw-redirect" href="/wiki/%EA%B3%BC%EA%B1%B0_%EC%A0%9C%EB%8F%84" title="과거 제도">춘당대무과</a>(春塘臺武科)에 장원으로 급제, <a class="new" href="/w/index.php?title=%EC%98%A4%EC%9C%84%EC%9E%A5&amp;action=edit&amp;redlink=1" title="오위장 (없는 문서)">오위장</a>(五衛將)이 되었다가 <a href="/wiki/1796%EB%85%84" title="1796년">1796년</a> <a class="mw-disambig" href="/wiki/%EC%96%91%ED%8F%89" title="양평">양평</a><a href="/wiki/%ED%98%84%EA%B0%90" title="현감">현감</a>(陽平縣監)이 되고, 그 해 <a class="new" href="/w/index.php?title=%EC%98%81%ED%8F%89%EB%A9%B4&amp;action=edit&amp;redlink=1" title="영평면 (없는 문서)">영평</a> <a href="/wiki/%ED%98%84%EA%B0%90" title="현감">현감</a>(永平縣令)으로 부임하였다. <a href="/wiki/1798%EB%85%84" title="1798년">1798년</a>에는 왕에게 바치기 위해 《북학의(北學議)》 진소본(進疏本)을 작성했다.</p>
<h4><span class="mw-headline" id=".EC.83.81.EC.97.85.2C_.EB.AC.B4.EC.97.AD.EA.B3.BC_.EC.82.AC.ED.9A.8C_.EB.B0.9C.EC.A0.84.EB.A1.A0">상업, 무역과 사회 발전론</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%B0%95%EC%A0%9C%EA%B0%80&amp;action=edit&amp;section=9" title="부분 편집: 상업, 무역과 사회 발전론">편집</a><span class="mw-editsection-bracket">]</span></span></h4>
<p>박제가는 <a href="/wiki/%EC%83%81%EC%97%85" title="상업">상업</a>과 <a class="mw-redirect" href="/wiki/%EB%AC%B4%EC%97%AD" title="무역">무역</a>의 장려와 개선을 주장했다. 그는 <a href="/wiki/%EC%B2%AD%EB%82%98%EB%9D%BC" title="청나라">청나라</a>를 오가며 <a class="mw-disambig" href="/wiki/%EC%95%84%EB%9E%8D" title="아랍">아랍</a>, <a href="/wiki/%EB%B2%A0%ED%8A%B8%EB%82%A8" title="베트남">베트남</a> 등의 무역상들을 통해 신문물을 접하면서 그는 새로운 문물 전파와 문화 교류 방법의 하나는 상업과 무역이라고 봤다. 따라서 그는 농업보다 상업에서 사회발전의 계기를 찾으려 했다. 상업과 무역을 천시하는 것은 잘못이며 이러한 장사와 무역이 국가 경제 개선에 도움을 준다고 봤다. 그는 무조건 근검절약만이 미덕이 아니라고 주장했는데 이는 당대의 지배층들로부터 사치를 권장하느냐는 반박을 받았다. 그는 상업을 발달시키려면 대부분의 <a class="mw-redirect" href="/wiki/%EC%8B%A4%ED%95%99%EC%9E%90" title="실학자">실학자</a>들조차 미덕으로 여겼던 봉건적 절검사상을 배격할 필요가 있다고 봤다. 그는 자신의 저서 &lt;&lt;북학의&gt;&gt; 내편 시정(市井)조에서도 '소비는 단순한 소비에 그치는 것이 아니라 재생산을 자극하는 것'이라 주장했고, '생산과 소비의 유기적 관계'를 해명함으로써 이를 연결하는 장사와 무역의 중요성을 천명했다. 따라서 그의 생각도 상업이 발달하면 농업과 수공업도 아울러 발달한다는 중상적 경제이론에 도달해 있었다.</p>
<p>또한 화폐 유통을 <a class="mw-redirect" href="/wiki/%EC%A0%95%EC%A1%B0" title="정조">정조</a>에게 여러번 건의하였다. 화폐 유통은 장사와 무역을 활발하게 돌아가게 하는 수단이었다. 박제가의 화폐경제 발달론의 본질은 국가의 경제력을 증대시켜 이용후생을 도모하는 데 있었다. 그것은 상업이 주가 되면서 농업·공업이 유기적으로 발전해야만 가능했다. 그래서 그는 상인들과 <a class="mw-redirect" href="/wiki/%EB%AC%B4%EC%97%AD%EC%83%81" title="무역상">무역상</a>들에 대한 지나친 천대와 편견을 자제할 것, <a class="new" href="/w/index.php?title=%EC%88%98%EA%B3%B5%EC%97%85&amp;action=edit&amp;redlink=1" title="수공업 (없는 문서)">수공업</a>자에 대한 국가적 수탈의 금지, <a href="/wiki/%EB%8C%80%EB%9F%89_%EC%83%9D%EC%82%B0" title="대량 생산">대량 생산</a>체제의 구축, 농업기술의 개량을 통한 농업생산성의 증진과 상업적 농업이 실시되어야 한다고 역설하였다. 또한 그는 국가경제를 강화하기 위해 해외통상을 통한 재화의 증식이 필요하다고 했다.</p>
<p>그는 밀무역을 근절할 방안으로도 국가가 상업과 무역에 대한 천대와 경멸, 제재를 줄여야 된다고 건의하였다. 무역과 상행위에 대한 제재가 사라진다면 밀무역 역시 자연스럽게 사라지거나 근절될 것이라고 하였다. 밀무역을 양성화시켜 정상적인 외국무역을 발전시킬 것과 <a class="mw-redirect" href="/wiki/%EA%B0%9C%EC%84%B1" title="개성">개성</a>, <a class="mw-redirect" href="/wiki/%EC%9D%B8%EC%B2%9C" title="인천">인천</a>과 충청도·전라도 일대의 <a class="mw-redirect" href="/wiki/%EC%84%9C%EC%82%B0" title="서산">서산</a>, <a class="mw-redirect" href="/wiki/%ED%83%9C%EC%95%88" title="태안">태안</a>, <a class="mw-disambig" href="/wiki/%EC%9E%A5%EC%A7%84" title="장진">장진</a>, <a class="mw-redirect" href="/wiki/%EC%9D%80%EC%A7%84" title="은진">은진</a>, <a class="mw-disambig" href="/wiki/%EA%B0%95%EA%B2%BD" title="강경">강경</a>, <a class="mw-disambig" href="/wiki/%EC%97%AC%EC%82%B0" title="여산">여산</a> 등지의 강가를 끼고 있는 지역에 무역항을 열고 <a href="/wiki/%EC%A4%91%EA%B5%AD" title="중국">중국</a> 남부 및 <a class="mw-redirect" href="/wiki/%EC%82%B0%EB%91%A5" title="산둥">산둥</a>(山東) 지방과 통상을 확대하고, 상권이 커지고 국력이 자라면 일본·안남·위구르 등 무역대상국을 확대하자고 제안했다. 그는 조선 후기 <a class="new" href="/w/index.php?title=%EC%83%81%ED%92%88%ED%99%94%ED%8F%90%EA%B2%BD%EC%A0%9C&amp;action=edit&amp;redlink=1" title="상품화폐경제 (없는 문서)">상품화폐경제</a>의 발전이라는 현실을 인정한 기반 위에서 상업·수공업·농업 전반의 생산력 발전을 적극적으로 추진하고 국가경제체제를 재조하자고 하였다.</p>
<h4><span class="mw-headline" id=".ED.8E.B8.EA.B2.AC.EA.B3.BC_.EC.B0.A8.EB.B3.84.EB.8C.80.EC.9A.B0.EC.97.90_.EC.A0.80.ED.95.AD">편견과 차별대우에 저항</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%B0%95%EC%A0%9C%EA%B0%80&amp;action=edit&amp;section=10" title="부분 편집: 편견과 차별대우에 저항">편집</a><span class="mw-editsection-bracket">]</span></span></h4>
<p>박제가는 어릴 때부터 통찰력과 판단력, 방대한 학식과 예술적 재능을 타고 났다. 하지만 서얼이라는 신분적 차별과 고분고분하지 않은 성격 때문에 주류 사회에서 따돌림과 무시를 당했다. 그는 허울만 가득한 조선의 양반`학자`선비`지식인 등 편협하고 답답한 집단을 비웃었지만, 기득권 세력의 벽을 부수지는 못했다.<sup class="reference" id="cite_ref-bisos_7-0"><a href="#cite_note-bisos-7">[7]</a></sup></p>
<p>그는 <a class="mw-redirect" href="/wiki/%EC%84%9C%EC%9E%90" title="서자">서자</a>라는 이유로 멸시당했고, 그가 속했던 <a class="mw-redirect" href="/wiki/%EB%B6%81%ED%95%99%ED%8C%8C" title="북학파">북학파</a>는 <a href="/wiki/%EB%85%B8%EB%A1%A0" title="노론">노론</a> 내에서도 비주류로 취급당했다. 그의 화폐유통론과 국가의 무역 장려론은 상거래를 천시 여기돈 <a href="/wiki/%EC%A1%B0%EC%84%A0" title="조선">조선</a>의 <a href="/wiki/%EC%84%B1%EB%A6%AC%ED%95%99" title="성리학">성리학</a>자들에 의해 불순한 사상 내지는 이문을 남기기 위한 협잡 정도로 취급되었다. 또한 그의 집안이 원래 <a class="mw-redirect" href="/wiki/%EC%86%8C%EB%B6%81" title="소북">소북</a>이었다가 박제가의 대에 <a href="/wiki/%EB%85%B8%EB%A1%A0" title="노론">노론</a>으로 전향한 것을 두고도 문제가 되었다.</p>
<p>심한 천대와 멸시 냉대에 실망했고 그는 뜻을 펴볼 기회를 잃었다. 정조에 의해 발탁되었으나 자신의 이상을 펼칠 수 없는 현실적인 한계에 가로 막힌데다 정조의 죽음 이후에는 유배 생활로 생을 마감했다.<sup class="reference" id="cite_ref-bisos_7-1"><a href="#cite_note-bisos-7">[7]</a></sup></p>
<p>그러나 박제가는 사회적 차별에 굴하지 않았다. “고독하고 고매한 사람만을 골라서 남달리 친하게 사귀고, 권세 많고 부유한 사람은 일부러 더 멀리하며”(정유각집 ‘소전’편) 차라리 가난하게 살았다.<sup class="reference" id="cite_ref-isoss_5-1"><a href="#cite_note-isoss-5">[5]</a></sup> 박제가는 그 단단한 습속의 벽과 온몸으로 맞서 싸웠다. 직설과 독설로 맞섰다.<sup class="reference" id="cite_ref-isoss_5-2"><a href="#cite_note-isoss-5">[5]</a></sup></p>
<h3><span class="mw-headline" id=".EC.83.9D.EC.95.A0_.ED.9B.84.EB.B0.98">생애 후반</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%B0%95%EC%A0%9C%EA%B0%80&amp;action=edit&amp;section=11" title="부분 편집: 생애 후반">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<h4><span class="mw-headline" id=".EC.A0.80.EC.88.A0.2C_.EC.9E.91.ED.92.88_.ED.99.9C.EB.8F.99">저술, 작품 활동</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%B0%95%EC%A0%9C%EA%B0%80&amp;action=edit&amp;section=12" title="부분 편집: 저술, 작품 활동">편집</a><span class="mw-editsection-bracket">]</span></span></h4>
<p>시, 그림, 글씨에도 두루 뛰어난 재질을 보였으며 <a href="/wiki/%EC%B2%AD%EB%82%98%EB%9D%BC" title="청나라">청나라</a>의 《사고전서 四庫全書》계열 학자들과 만난 이후 편지 서신등을 통한 꾸준한 교류를 통해 <a href="/wiki/%EC%A1%B0%EC%84%A0" title="조선">조선</a>에 대련 형식(對聯形式)을 수용하고 이를 <a href="/wiki/%EC%A1%B0%EC%84%A0" title="조선">조선</a>에 소개하였다. 글씨는 조선말기의 서풍과 추사체의 형성에 선구적 구 실을 하였으며, 그림은 간결한 필치와 맑고 옅은 채색에 운치와 문기(文氣)가 짙게 풍기는 사의적(寫意的)인 문인화풍의 산수 인물화와 생동감이 넘치는 고기 그림을 잘 그렸다고 한다.<sup class="reference" id="cite_ref-8"><a href="#cite_note-8">[8]</a></sup> 그는 시 재주를 <a href="/wiki/%EC%B2%AD%EB%82%98%EB%9D%BC" title="청나라">청나라</a>의 문인들로부터도 인정받아 생전인 <a href="/wiki/1801%EB%85%84" title="1801년">1801년</a>을 전후해서 그의 시문집인 『정유고략(貞유藁略)』이 중국에서 간행되기도 하였다.</p>
<p>글씨는 <a href="/wiki/%EC%98%88%EC%84%9C" title="예서">예서</a>풍을 띠고 있으며 <a class="mw-disambig" href="/wiki/%ED%95%B4%EC%84%9C" title="해서">해서</a>, <a href="/wiki/%ED%96%89%EC%84%9C" title="행서">행서</a>, <a href="/wiki/%EC%B4%88%EC%84%9C" title="초서">초서</a> 등 다양한 서체를 구사했다. <a class="mw-redirect" href="/wiki/%EC%B6%94%EC%82%AC_%EA%B9%80%EC%A0%95%ED%9D%AC" title="추사 김정희">추사 김정희</a> 역시 그의 문하에서 글과 그림을 배웠는데 조선 말기의 서풍과 <a href="/wiki/%EC%B6%94%EC%82%AC%EC%B2%B4" title="추사체">추사체</a>의 형성에도 영향을 주게 된다. <a href="/wiki/%EA%B5%AC%EC%96%91%EC%88%9C" title="구양순">구양순</a>(歐陽詢)과 <a href="/wiki/%EB%8F%99%EA%B8%B0%EC%B0%BD" title="동기창">동기창</a>(董其昌)체의 <a href="/wiki/%ED%96%89%EC%84%9C" title="행서">행서</a>도 잘 썼으며 필적이 굳세고 활달하면서 강건하였다.</p>
<p>그림에도 능하여 서실을 짓고 학문 외에 화가 제자들도 다수 배출하였다. 그의 그림은 사실적인 묘사를 한 <a href="/wiki/%EB%AC%B8%EC%9D%B8%ED%99%94" title="문인화">문인화</a>풍의 산수, 인물화를 그렸고 중국의 산수화 보다는 <a href="/wiki/%EC%A1%B0%EC%84%A0" title="조선">조선</a> 국내의 풍광과 경치를 그렸다. 또한 생동감이 넘치는 꿩, 새, 고기, 노루 등을 정밀하게 묘사하였다.</p>
<p><a href="/wiki/1798%EB%85%84" title="1798년">1798년</a>(정조 22년) 다시 <a href="/wiki/%EB%B6%80%EC%97%AC" title="부여">부여</a><a href="/wiki/%ED%98%84%EA%B0%90" title="현감">현감</a>이 되었으며, 그해 <a class="mw-redirect" href="/wiki/%EC%98%81%EC%A1%B0" title="영조">영조</a>가 적전(籍田)에 친히 농사에 참여한 지 60주년이 되는 날을 기념하기 위해 <a class="mw-redirect" href="/wiki/%EC%A0%95%EC%A1%B0" title="정조">정조</a>가 널리 농서를 구하자, '북학의'의 내용 일부를 발췌, 골자로 한 '응지농정소 應旨農政疏'와 '소진본북학의 疏進本北學議'를 올렸다.</p>
<h4><span class="mw-headline" id=".EB.AC.B8.EC.B2.B4.EB.B0.98.EC.A0.95">문체반정</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%B0%95%EC%A0%9C%EA%B0%80&amp;action=edit&amp;section=13" title="부분 편집: 문체반정">편집</a><span class="mw-editsection-bracket">]</span></span></h4>
<p>박제가는 고군분투했다. 틀에 박히고 고루하고 진부한 시와 문장을 혐오하며 나만의 글쓰기를 찾아 나섰다. 당시 선비들은 두보의 시를 최고로 여겨 배웠고, 다음은 당나라 시, 그 다음은 송나라·금나라·원나라·명나라 시를 배웠다.<sup class="reference" id="cite_ref-isoss_5-3"><a href="#cite_note-isoss-5">[5]</a></sup> 박제가가 보기에 전범에 매달리는 글쓰기는 남이 한 말의 찌꺼기나 줍는 행태에 불과했다. 자기 시대의 현장을, 자기의 말로 표현하는 것이 진정한 시요 문장이었다. 역설적으로 나만의 글쓰기를 개척하는 것이 진정 고인의 글쓰기에 다가가는 길이었다.<sup class="reference" id="cite_ref-isoss_5-4"><a href="#cite_note-isoss-5">[5]</a></sup></p>
<p>'형암 선생 시집서'(炯菴先生 詩集序)에서 그는 '하늘과 땅 사이에 가득 차 있는 것이 모두 시다. 사계절의 변화와 온갖 만물의 웅성거리는 소리, 그 몸짓과 빛깔, 그리고 음절은 그들 나름대로 존재하고 있다.'며 그는 현실에서 보고 듣고 관찰한 것이 좋은 글, 좋은 작품의 소재가 된다고 보았다.</p>
<p>한편 형식에서 지나치게 벗어났다는 <a href="/wiki/%EB%85%B8%EB%A1%A0" title="노론">노론</a>계 다른 학파의 비난이 빗발치자 <a class="mw-redirect" href="/wiki/%EC%A0%95%EC%A1%B0" title="정조">정조</a>는 그의 스승 <a class="mw-disambig" href="/wiki/%EB%B0%95%EC%A7%80%EC%9B%90" title="박지원">박지원</a>을 비롯한 <a class="mw-redirect" href="/wiki/%EB%B6%81%ED%95%99%ED%8C%8C" title="북학파">북학파</a> 인사들에게 '문체반정'을 하라고 선언하고, 바른 글을 써 내라고 지시한다.</p>

<p>스승 <a href="/wiki/%EB%B0%95%EC%A7%80%EC%9B%90_(1737%EB%85%84)" title="박지원 (1737년)">연암 박지원</a>을 비롯한 동료 문인들은 모두 반성문을 지어 올렸다. 결국 박제가는 '자송문'(自訟文)이라는 반성문을 지어 <a class="mw-redirect" href="/wiki/%EC%A0%95%EC%A1%B0" title="정조">정조</a>에게 바쳤다. 그러나 당대의 문장을 순정한 문체로 되돌리겠다는 정조의 강력한 의지에 부응하여 반성문을 제출하면서도, '자송문'(自訟文)이라는 제목과는 어울리지 않게 반성은 하지 않고 항변하였다.<sup class="reference" id="cite_ref-isoss_5-6"><a href="#cite_note-isoss-5">[5]</a></sup></p>
<h4><span class="mw-headline" id=".EC.9C.A0.EB.B0.B0.EC.99.80_.EC.B5.9C.ED.9B.84">유배와 최후</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%B0%95%EC%A0%9C%EA%B0%80&amp;action=edit&amp;section=14" title="부분 편집: 유배와 최후">편집</a><span class="mw-editsection-bracket">]</span></span></h4>
<p>10여년의 검서관 생활은 그의 시력을 악화시켰다. 밤늦게까지 독서했고 어두운 방에서도 계속 책을 읽고 글을 쓴 것이 원인이었다. 만년의 그는 <a href="/wiki/%EC%95%88%EA%B2%BD" title="안경">안경</a>을 썼는데, 왼쪽 눈이 상하여 <a href="/wiki/%EC%95%88%EA%B2%BD" title="안경">안경</a>을 써도 효과가 없었다. 몇해 뒤에는 오른편 눈의 시력마저 희미해졌다. 또한 시집간 둘째 딸이 그보다 먼저 죽는 일도 겪었다. 그는 관직생활 전후로 서실을 열고 문인들에게 글과 그림을 가르쳤는데, 추사 <a href="/wiki/%EA%B9%80%EC%A0%95%ED%9D%AC" title="김정희">김정희</a>(金正喜) 역시 그의 문하에서 글과 그림을 배웠다.</p>
<p><a href="/wiki/1800%EB%85%84" title="1800년">1800년</a>(정조 24년) <a href="/wiki/%EC%A1%B0%EC%84%A0_%EC%A0%95%EC%A1%B0" title="조선 정조">정조</a> 사후 노론 벽파에게 소론, 남인과 실학파 계열 학자들은 대부분 숙청되었다. 그는 <a class="mw-redirect" href="/wiki/%EC%86%8C%EB%B6%81" title="소북">소북</a>에서 <a href="/wiki/%EB%85%B8%EB%A1%A0" title="노론">노론</a> <a class="mw-redirect" href="/wiki/%EB%B6%81%ED%95%99%ED%8C%8C" title="북학파">북학파</a>로 당적을 옮기고 전향했으나 <a href="/wiki/1801%EB%85%84" title="1801년">1801년</a>에는 청나라에 네 번째 연행을 갔다 돌아왔다가 사돈이었던 윤가기(尹可基)가 주모한 흉서사건에 연루되어 그해 <a href="/wiki/9%EC%9B%94" title="9월">9월</a> <a href="/wiki/%ED%95%A8%EA%B2%BD%EB%B6%81%EB%8F%84" title="함경북도">함경북도</a> <a href="/wiki/%ED%9A%8C%EB%A0%B9%EC%8B%9C" title="회령시">종성</a>(鍾城)으로 유배를 갔다. 사돈인 윤가기의 흉서 사건에 연루된 인물들 중에는 그의 <a href="/wiki/%EC%B2%AD%EB%82%98%EB%9D%BC" title="청나라">청나라</a>행을 후원했던 <a class="new" href="/w/index.php?title=%EC%9C%A4%ED%96%89%EC%9E%84&amp;action=edit&amp;redlink=1" title="윤행임 (없는 문서)">윤행임</a>도 연루됨으로써 그 역시 윤가기, 윤행임의 당여로 몰렸다.</p>
<p>결국 대비 김씨와 노론의 영수 심환지를 비방하는 벽서 사건에 연루되어 유배를 당한다.<sup class="reference" id="cite_ref-isoss_5-7"><a href="#cite_note-isoss-5">[5]</a></sup> <a href="/wiki/1805%EB%85%84" title="1805년">1805년</a> 풀려났으며 <a href="/wiki/1815%EB%85%84" title="1815년">1815년</a> <a href="/wiki/4%EC%9B%94_25%EC%9D%BC" title="4월 25일">4월 25일</a> 병환으로 죽었다. 일설에는 <a href="/wiki/10%EC%9B%94" title="10월">10월</a>에 사망했다고 하며, 사망 연대도 <a href="/wiki/1805%EB%85%84" title="1805년">1805년</a>설, 석방 이듬해인 <a href="/wiki/1806%EB%85%84" title="1806년">1806년</a>설 등이 있어 그의 정확한 사망 년대는 전해지지 않고 있다.</p>
<h3><span class="mw-headline" id=".EC.82.AC.ED.9B.84">사후</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%B0%95%EC%A0%9C%EA%B0%80&amp;action=edit&amp;section=15" title="부분 편집: 사후">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>묘는 <a href="/wiki/%EA%B2%BD%EA%B8%B0%EB%8F%84" title="경기도">경기도</a> <a href="/wiki/%EA%B4%91%EC%A3%BC%EA%B5%B0_(%EA%B2%BD%EA%B8%B0%EB%8F%84)" title="광주군 (경기도)">광주군</a> 암현(庵峴)에 안장되었으나 후에 실전되었다. 그의 집안 족보에 의하면 박제가의 묘는 <a href="/wiki/%EB%82%A8%ED%95%9C%EC%82%B0%EC%84%B1" title="남한산성">남한산성</a> 동문밖 십리에 있었으며, 그의 아버지 묘소와 할아버지 묘소가 근처에 있었다 한다. 그의 사후 <a class="mw-redirect" href="/wiki/%EB%B6%81%ED%95%99%ED%8C%8C" title="북학파">북학파</a>는 <a href="/wiki/%EB%B0%95%EA%B7%9C%EC%88%98" title="박규수">박규수</a>, <a href="/wiki/%EC%98%A4%EA%B2%BD%EC%84%9D" title="오경석">오경석</a>, <a class="mw-redirect" href="/wiki/%EC%9C%A0%EB%8C%80%EC%B9%98" title="유대치">유대치</a> 등으로 이어지며 중상주의와 통상론을 계속 건의하였고 이는 <a href="/wiki/%EA%B9%80%EC%98%A5%EA%B7%A0" title="김옥균">김옥균</a>, <a href="/wiki/%EC%84%9C%EC%9E%AC%ED%95%84" title="서재필">서재필</a>, <a href="/wiki/%EC%9C%A4%EC%B9%98%ED%98%B8" title="윤치호">윤치호</a> 등의 <a href="/wiki/%EB%85%B8%EB%A1%A0" title="노론">노론</a>출신 <a class="mw-redirect" href="/wiki/%EA%B0%9C%ED%99%94%ED%8C%8C" title="개화파">개화파</a>로 이어지게 된다.</p>
<p>그의 학통은 <a href="/wiki/%EA%B9%80%EC%A0%95%ED%9D%AC" title="김정희">김정희</a>를 거쳐서 <a href="/wiki/%ED%9D%A5%EC%84%A0%EB%8C%80%EC%9B%90%EA%B5%B0" title="흥선대원군">흥선대원군</a>으로 이어진다. 한때 추사 <a href="/wiki/%EA%B9%80%EC%A0%95%ED%9D%AC" title="김정희">김정희</a>는 그에게서 글과 그림을 배웠고, 추사 <a href="/wiki/%EA%B9%80%EC%A0%95%ED%9D%AC" title="김정희">김정희</a>의 어머니가 <a href="/wiki/%EC%9D%80%EC%8B%A0%EA%B5%B0" title="은신군">은신군</a> 부인과 자매인 인연으로 <a href="/wiki/%ED%9D%A5%EC%84%A0%EB%8C%80%EC%9B%90%EA%B5%B0" title="흥선대원군">흥선대원군</a>이 김정희의 문하에 들며 글과 그림을 배웠다. 문인 김정희의 부인은 김정희와 교류하던 <a href="/wiki/%ED%99%8D%EB%8C%80%EC%9A%A9" title="홍대용">홍대용</a>의 사촌 홍담용의 딸이 된다.</p>
<p><a href="/wiki/1997%EB%85%84" title="1997년">1997년</a> <a href="/wiki/7%EC%9B%94" title="7월">7월</a> '8월의 문화인물'로 선정되었다.</p>

<h2><span class="mw-headline" id=".EC.82.AC.EC.83.81.EA.B3.BC_.EC.8B.A0.EB.85.90">사상과 신념</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%B0%95%EC%A0%9C%EA%B0%80&amp;action=edit&amp;section=19" title="부분 편집: 사상과 신념">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>박제가는 당시의 공리공담을 일삼던 <a class="mw-redirect" href="/wiki/%EC%A3%BC%EC%9E%90%ED%95%99" title="주자학">주자학</a>적 사상과 <a class="new" href="/w/index.php?title=%ED%92%8D%EC%88%98%EB%8F%84%EC%B0%B8%EC%84%A4&amp;action=edit&amp;redlink=1" title="풍수도참설 (없는 문서)">풍수도참설</a> 등에 비판적이었다. 또한 존명(尊明) 사대주의자들이 제기한 <a href="/wiki/%EB%B6%81%EB%B2%8C%EB%A1%A0" title="북벌론">북벌론</a>의 비현실성을 지적하는 한편 <a href="/wiki/%EC%B2%AD%EB%82%98%EB%9D%BC" title="청나라">청나라</a>에 전래된 서양의 문물을 적극 받아들일 것을 주장하였고, 과거 제도의 폐단을 지적하는 한편 봉건적 신분제를 반대하였다.</p>
<h3><span class="mw-headline" id=".EC.9D.B8.EC.9E.AC_.EB.93.B1.EC.9A.A9.EB.A1.A0">인재 등용론</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%B0%95%EC%A0%9C%EA%B0%80&amp;action=edit&amp;section=20" title="부분 편집: 인재 등용론">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>그는 <a class="mw-redirect" href="/wiki/%EA%B3%BC%EA%B1%B0_%EC%A0%9C%EB%8F%84" title="과거 제도">과거 제도</a>의 무용론을 주장하였다. 과거 제도가 학문적 소양이 되는 인재를 선발하는 것이 목적이었지만 점차 사대부와 고관, 벌열가문이 돈과 재력으로 관직을 세습하기 위한 하나의 수단이 되었다고 비판했다. 박제가는 사회적 생산과 과학기술의 발전을 가로막는 양반과 문벌 및 과거제도의 폐단인 문벌의 의한 관직의 세습에 반대하고, 실제의 쓰임에 맞게 인재를 등용해야 함을 역설했다. 그는 <a href="/wiki/%EC%A1%B0%EA%B4%91%EC%A1%B0" title="조광조">조광조</a>의 현량과를 훌륭한 대안으로 평가하기도 했다. 그는 비천한 집안의 재능 있는 자제를 추천하는 제도 등을 실시하자고 하였다. 그가 반대한 것은 <a class="mw-redirect" href="/wiki/%EA%B3%BC%EA%B1%B0_%EC%A0%9C%EB%8F%84" title="과거 제도">과거 제도</a> 자체가 아니고 과거라는 미명 아래 자행되는 관직의 세습이었다.</p>
<h3><span class="mw-headline" id=".EC.83.81.EC.97.85_.EC.A7.84.ED.9D.A5.EB.A1.A0">상업 진흥론</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%B0%95%EC%A0%9C%EA%B0%80&amp;action=edit&amp;section=21" title="부분 편집: 상업 진흥론">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>그는 국부(國富)를 창출하기 위한 방안으로 상공업 진흥론과 농업 진흥론을 제시하였다. 그는 장사를 천하게 여기지만 실질적으로 돈과 국력, 각종 문물 교역을 이루는 방법은 상거래 밖에 없다고 보았다. <a href="/wiki/%EC%B2%AD%EB%82%98%EB%9D%BC" title="청나라">청나라</a>에 전래된 서구의 선진 문물을 받아들여 상업을 발전시켜야 한다는 상업진흥론을 적극 주장했다. 그리고 교역 활성화를 통해 외부의 문물이 유입될 길을 열자고 하였다.</p>
<p>상공업을 진흥시키기 위한 구체적 방안으로 그는 교통기관을 발달시킬 것(用車論), 화폐의 질을 개선하여 악화주조(惡貨鑄造)를 금지하고 화폐의 질을 높일 것, 밀무역(密貿易)을 양성화하고 무역을 장려할 것, 국내시장을 확보할 것 등을 제시했다. 동시에 만일의 사변을 대비하여 금과 은을 정부에서 축적할 것을 건의하여 미연의 사태를 대비할 것을 <a class="mw-redirect" href="/wiki/%EC%A0%95%EC%A1%B0" title="정조">정조</a>에게 건의하였다. 또한 그는 상업의 유통으로 농산물 판매가 원활해지고 수공업의 발달로 낙후된 농기구를 개량한다면 농업의 합리적 경영이 이루어질 수 있다고 보기도 했다. 한편으로는 <a href="/wiki/%EC%A1%B0%EC%84%A0" title="조선">조선</a> 국내에서 생산되는 금과 은의 해외유출과 중국상품의 유입을 철저히 금지할 것을 건의하였지만 금과 은을 조공으로 바쳐야 했던 현실에서는 받아들여지지 않았다.</p>
<p>그의 상업 진흥책은 상업적 이익을 추구하는 것을 천시하고 위험시하던 사대부들과는 반대되는 견해였다. 이에 상공업 육성에 장애가 되는 양반들을 도태시키기 위해 봉건적인 문물 제도와 <a class="mw-redirect" href="/wiki/%EA%B3%BC%EA%B1%B0_%EC%A0%9C%EB%8F%84" title="과거 제도">과거 제도</a>를 타파할 것도 주장했다.</p>
<h3><span class="mw-headline" id=".ED.8F.90.EC.87.84.EC.84.B1.EA.B3.BC_.EB.B0.B0.ED.83.80.EC.84.B1_.EB.B9.84.ED.8C.90">폐쇄성과 배타성 비판</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%B0%95%EC%A0%9C%EA%B0%80&amp;action=edit&amp;section=22" title="부분 편집: 폐쇄성과 배타성 비판">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>그는 <a href="/wiki/%EC%A1%B0%EC%84%A0" title="조선">조선</a>사회가 상당히 폐쇄적이고 배타적인 사회라고 지적했다. "나와 다르다고 해서 무리지어 비웃고, 또 덩달아 이를 업신여긴다. 좁은 소견으로 헤아릴 수 없는 깊이를 엿보고, 틀에 박힌 안목으로 끝없는 변화를 논하곤 한다.<sup class="reference" id="cite_ref-bisos_7-2"><a href="#cite_note-bisos-7">[7]</a></sup>"며 한탄했다.</p>
<p>폐쇄성과 배타성을 버리지 않는다면 외부의 우수한 문물을 받아들이기는 어렵다고 보았다. 중국의 중화문명만큼 혹은 그보다 더 우수한 문명이 서방에 존재하고 있다고 알렸지만 아무도 듣지 않았다.</p>
<h3><span class="mw-headline" id=".ED.97.88.EB.A1.80.ED.97.88.EC.8B.9D_.EB.B9.84.ED.8C.90">허례허식 비판</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%B0%95%EC%A0%9C%EA%B0%80&amp;action=edit&amp;section=23" title="부분 편집: 허례허식 비판">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>그는 자신의 저서 '<a href="/wiki/%EB%B6%81%ED%95%99%EC%9D%98" title="북학의">북학의</a>'에 대부분의 백성은 아침저녁 먹을거리가 없이 생계를 꾸려가는데, “열 가구가 사는 마을에서 하루 두 끼를 해결하는 자가 몇 집 되지 않는다”고 보고하였다.<sup class="reference" id="cite_ref-uigkae_9-0"><a href="#cite_note-uigkae-9">[9]</a></sup></p>
<p>박제가가 바라는 이상적인 사회는 물질적으로도 풍요롭고, 문화적으로도 향기 나는 사회다.<sup class="reference" id="cite_ref-isoss_5-8"><a href="#cite_note-isoss-5">[5]</a></sup> 재화의 유통이 활발하고, 사치가 가능하며, 문화적 수준도 상당한 사회. 박제가는 <a class="mw-redirect" href="/wiki/%EB%8C%80%EC%A4%91_%EB%AC%B8%ED%99%94" title="대중 문화">문화예술</a>과 사치품에 관해 논할 때 <a class="new" href="/w/index.php?title=%EB%8F%84%EB%8D%95%EC%A3%BC%EC%9D%98&amp;action=edit&amp;redlink=1" title="도덕주의 (없는 문서)">도덕주의</a>적 관념을 개입시키지 않는다.<sup class="reference" id="cite_ref-isoss_5-9"><a href="#cite_note-isoss-5">[5]</a></sup> 그는 돈에는 도덕이 적용되지 않는다며, 부도덕한 사람이라 해서 돈을 벌지 못하지는 않는다고 지적했다. 그는 재물을 탐내지 않는 것을 도덕으로 규정하는 것에 대해서도 위선으로 규정했다.</p>
<p>그는 <a href="/wiki/%EB%8F%84%EB%8D%95" title="도덕">도덕</a>과 <a class="new" href="/w/index.php?title=%EC%9D%B8%EC%9D%98%EC%98%88%EC%A7%80&amp;action=edit&amp;redlink=1" title="인의예지 (없는 문서)">인의예지</a>, <a class="new" href="/w/index.php?title=%EC%9D%B8%EB%A5%9C&amp;action=edit&amp;redlink=1" title="인륜 (없는 문서)">인륜</a>의 명분으로 <a href="/wiki/%EC%A1%B0%EC%A0%95" title="조정">조정</a>에서 <a href="/wiki/%EC%B6%98%ED%99%94" title="춘화">춘화</a>와 <a href="/wiki/%EC%95%BC%EB%8B%B4" title="야담">야담</a><sup class="reference" id="cite_ref-10"><a href="#cite_note-10">[10]</a></sup>을 단속하는 것도 잘못이라 비판했다. 그에 의하면 예술의 아름다움과 사치스러움은 재화와 물품을 마르지 않게 하는 원동력이다.<sup class="reference" id="cite_ref-isoss_5-10"><a href="#cite_note-isoss-5">[5]</a></sup> 그러한 향락을 소비하기 위해서 돈을 쓰는 사람들도 있기 마련이며, 모든 백성이 선비처럼 고결한 삶을 살아야 할 이유는 없다며, 소인은 소인답게 살도록 내버려 두는 것이 자연스럽다고 지적했다. 그는 소박하고 질박한 생활을 표상했던 유학적 가치와 완전히 결별한다.<sup class="reference" id="cite_ref-isoss_5-11"><a href="#cite_note-isoss-5">[5]</a></sup> 당시 위선적인 농본정책에 따른 극빈의 삶으로 온 나라가 가득했다. 그는 그러면서도 "인의(仁義)가 강물처럼 흐르는 요순의 시대"를 재현한다며 허장성세를 보였다고 지적했다.<sup class="reference" id="cite_ref-uigkae_9-1"><a href="#cite_note-uigkae-9">[9]</a></sup></p>
<h3><span class="mw-headline" id=".EB.B6.81.EB.B2.8C.EB.A1.A0_.EB.B9.84.ED.8C.90.EA.B3.BC_.EB.B6.81.ED.95.99.EB.A1.A0">북벌론 비판과 북학론</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%B0%95%EC%A0%9C%EA%B0%80&amp;action=edit&amp;section=24" title="부분 편집: 북벌론 비판과 북학론">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>그는 <a href="/wiki/%EB%B6%81%EB%B2%8C%EB%A1%A0" title="북벌론">북벌론</a>이 당시 <a href="/wiki/%EC%A1%B0%EC%84%A0" title="조선">조선</a>의 국력과 경제적, 군사적 현실을 외면한 공리공담이라고 비판했다. 그는 조선이 북벌(北伐) 대상으로 지목한 청나라가 오랑캐가 아니라 새로운 학문과 서양과학으로 무장한 문명국임을 직시했다. 청나라를 통해 서양문물을 받아들이자는 박제가의 주장은 '우물 안 개구리'였던 조선 지식인들의 낡은 생각을 뒤흔든 혁명적 발상이었다.<sup class="reference" id="cite_ref-cho01_3-1"><a href="#cite_note-cho01-3">[3]</a></sup> 그러나 <a href="/wiki/%EB%B6%81%EB%B2%8C%EB%A1%A0" title="북벌론">북벌론</a>에 대한 비판은 <a href="/wiki/%EB%85%B8%EB%A1%A0" title="노론">노론</a>은 물론 <a href="/wiki/%EB%82%A8%EC%9D%B8" title="남인">남인</a> 강경파에게도 반감을 초래하는 이유가 되었다.</p>
<p>청나라에 다녀온 이후 그는 청나라가 단순한 오랑캐가 아니라고 확신했다. 조선의 빈곤 타파와 갑갑한 습속의 개혁은 청나라의 선진 문물을 배워야만 가능하다고 확신했다. 이 때문에 연행을 다녀온 직후 <a href="/wiki/%EB%B6%81%ED%95%99%EC%9D%98" title="북학의">북학의</a>를 저술한다.</p>
<h3><span class="mw-headline" id=".EC.A4.91.EA.B5.AD.EC.96.B4_.EC.82.AC.EC.9A.A9.EB.A1.A0">중국어 사용론</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%B0%95%EC%A0%9C%EA%B0%80&amp;action=edit&amp;section=25" title="부분 편집: 중국어 사용론">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>조선이 빠르게 청나라에 맞서는 문명국이 되려면 언문이 일치되는 중국어(북경어)를 사용하자고 하였다.<sup class="reference" id="cite_ref-isoss_5-12"><a href="#cite_note-isoss-5">[5]</a></sup> <a href="/wiki/%EC%A4%91%EA%B5%AD" title="중국">중국</a> 문명을 그리워하는 사대부들이 중국어를 제2의 국어로 쓰지 못할 것은 없다고 봤다.</p>
<p>그러나 문명세계를 향한 박제가의 욕망은 중국어공용론으로 거리낌 없이 내달린다.<sup class="reference" id="cite_ref-isoss_5-13"><a href="#cite_note-isoss-5">[5]</a></sup> 이런 정황상 <a href="/wiki/%EB%B6%81%EB%B2%8C%EB%A1%A0" title="북벌론">북벌론</a>을 절대 이념으로 수호했던 당대 선비들이 이 열혈 북학자를 당괴(唐魁) 혹은 당벽(唐癖)이라 비방하게 된다.<sup class="reference" id="cite_ref-isoss_5-14"><a href="#cite_note-isoss-5">[5]</a></sup></p>
<h3><span class="mw-headline" id=".EC.84.9C.EC.96.91_.EA.B8.B0.EC.88.A0_.EB.8F.84.EC.9E.85.EB.A1.A0">서양 기술 도입론</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%B0%95%EC%A0%9C%EA%B0%80&amp;action=edit&amp;section=26" title="부분 편집: 서양 기술 도입론">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>박제가는 조선의 가난을 해결하기 위한 방법으로 중국과의 통상과 신기술 도입을 주장했다. 서양의 신 기술을 도입하기 위해서는 우선 무역과 상권 형성으로 자연스럽게 서양인들이 유입되게 하자고 하였다. 박제가는 <a href="/wiki/1786%EB%85%84" title="1786년">1786년</a> <a href="/wiki/1%EC%9B%94_22%EC%9D%BC" title="1월 22일">1월 22일</a> 조정의 조회(朝會)에 참석해서 통상과 신기술 도입을 건의한다. 그는 중국에 사신을 파견해 '일본과 유구, 안남과 서양 등이 모두 중국의 복건·절강·교주(交州)·광주(廣州)에서 교역하고 있습니다. 바라건대 여러 나라처럼 뱃길을 이용해 통상할 수 있도록 해주십시오'라고 요청하자고 제안했다.<sup class="reference" id="cite_ref-cho01_3-2"><a href="#cite_note-cho01-3">[3]</a></sup></p>
<p>그는 국가 경제와 국가재정에 커다란 영향을 미치는 물자유통과 물가 평준화를 기하기 위해 수레를 보급해야 한다고 주장하였다.</p>
<p>그는 선박의 이용을 늘릴 것을 주장하였으며 병기의 개선, 영농법의 개량, 선진기술의 도입을 주장하였다. 특히 서양인을 초빙해 천문 관측과 농잠(農蠶)·의약·궁궐과 성곽과 다리를 짓는 법, 구리나 옥을 채굴하고 유리를 구워내는 법, <a class="mw-disambig" href="/wiki/%ED%99%94%ED%8F%AC" title="화포">화포</a>를 설치하는 법, 수레를 통행시키고 배를 건조하는 법을 가르치도록 하자는 제안을 하고 있다.<sup class="reference" id="cite_ref-cho01_3-3"><a href="#cite_note-cho01-3">[3]</a></sup> 서양인들을 <a href="/wiki/%EC%A1%B0%EC%84%A0" title="조선">조선</a>에 받아들여 그들로부터 농잠, 의약 제조, 성곽 축조, <a href="/wiki/%EC%84%A0%EB%B0%95" title="선박">선박</a> 축조 등의 기술을 배우게 하자는 것이었다. 경제가 어려워지는 이유로 "나라에 (학문을 공부한답시고) 놀고먹는 자가 갈수록 불어나는 것은 사족(士族)들이 날로 번성하기 때문"이라면서 "물길과 뭍길을 이용해 장사하고 교역하는 모든 일에 사족들이 참여할 수 있도록 허락해야 한다"고 주장했다.<sup class="reference" id="cite_ref-cho01_3-4"><a href="#cite_note-cho01-3">[3]</a></sup> 그밖에 종두법의 연구에도 관심을 쏟았다. 그러나 서양인을 귀신이나 괴물로 여기던 당시의 집권층은 그가 요망한 것에 혹하여 사술을 꾸민다며 반발한다.</p>
<h3><span class="mw-headline" id=".EB.86.8D.EC.A0.95_.EA.B0.9C.ED.98.81.EB.A1.A0">농정 개혁론</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%B0%95%EC%A0%9C%EA%B0%80&amp;action=edit&amp;section=27" title="부분 편집: 농정 개혁론">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>그는 <a href="/wiki/%EC%9D%B4%EC%9D%B5" title="이익">이익</a>이나 <a href="/wiki/%EC%9C%A0%ED%98%95%EC%9B%90" title="유형원">유형원</a>의 중농주의에는 비판, 반대하였으나 농업계획론을 주장했다. 국가의 부력이나 전경제기구 내에서 농업경제의 위치를 규정할 것과, 경제기구 전반의 원활한 운영을 하는 것이 농업경제의 안정이 달려 있다고 보았다. 기구의 개량화와 국가적인 농업, 농지 개량정책으로 농업 경영의 합리적인 개선을 통해 생산성을 높인다면 국가재정이나 경제질서 전반의 안정을 기할 수 있다고 판단했다. 그에 따라 박제가는 다른 산업의 유통이 활발해져야 농기구와 농업시설의 개량이 이루어질 수 있다고 봤다. 농업진흥론은 농업문제 해결을 다른 산업분야와 연결시키는 특징을 지니고 있었다. 즉, 농업문제를 해결하기 위해서는 상업이나 수공업도 아울러 개선해야 함을 역설했다.</p>
<p>농법개량에도 관심을 기울였다. 합리적인 경작, 농경을 위해 전무(田畝) 제도를 개선하고, 시비(施肥)의 효과를 높이기 위해 취분법(取糞法)을 개량하며, 노동력을 절약하기 위해 농기구를 개량해야 된다고 하였다. 또한 농기구를 저렴하고 쉽게 바꿀 수 있어야 함도 강조했다. 시장의 활성화 역시 농기구 개량에 도움이 된다고 보았다. 시장의 활성화로 농산물 판매가 원활해지는 것과, 상거래를 통해 최신 농기구 등의 유입, 수공업의 발달 등으로 낙후된 <a href="/wiki/%EC%A1%B0%EC%84%A0" title="조선">조선</a>의 농기구를 개량한다면 농업의 활성화를 이룩할 수 있다고 보았다.</p>
<p>사대부에게도 노동과 농업을 하게 하도록 하고, 유통경제 부문인 상업활동에도 종사하게 함으로써 일하지 않는 인력을 줄여야 한다는 점을 강조했다. 사대부의 농업, 노동 참여는 토지소유관계의 측면에서보다는, 당시 토지를 점유하던 지주들이 사대부였으므로 사대부를 농업에 참여케 하는 것은 농업을 합리적으로 경영함으로써 농업문제를 해결하려는 것이었다. 농자천하지대본의 기치를 내건 조선에서 유생에게 농사를 하게 하자는 그의 주장에는 반발은 없었지만 <a href="/wiki/%EC%82%AC%EB%8C%80%EB%B6%80" title="사대부">사대부</a>들은 내심 반발하고 그를 꺼렸다. 그러나 토지를 조정에서 매수 또는 몰수해야 된다는 견해에는 반대하였다. 그는 사대부들의 농지 과다 점유를 함부로 탓할 것이 아니라 경영과 기술의 개선과 합리화, 집약화를 통해 수익을 늘리는 것, 소규모의 농지에서 최대의 수익을 올리는 것이 급선무라 하였다. 그는 이를 시험하고자 둔전(屯田)의 설치를 <a class="mw-redirect" href="/wiki/%EC%A0%95%EC%A1%B0" title="정조">정조</a>에게 여러 차례 건의하기도 했다.</p>
<h3><span class="mw-headline" id=".EC.B2.AD.EB.A0.B4_.EB.AC.B4.EC.9A.A9.EB.A1.A0">청렴 무용론</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%B0%95%EC%A0%9C%EA%B0%80&amp;action=edit&amp;section=28" title="부분 편집: 청렴 무용론">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>그는 <a href="/wiki/%EC%A1%B0%EC%84%A0" title="조선">조선</a>의 선비들이 가난한 것은 옳은 것이 아니며, 청렴성을 강조하는 것은 위선이라 하였다. 박제가는 '가난'을 싫어했다. 권력에 아부하기 싫어 차라리 가난하게 산 것이지 가난을 편안하게 여긴 것은 아니었다.<sup class="reference" id="cite_ref-isoss_5-15"><a href="#cite_note-isoss-5">[5]</a></sup> 젊은 시절 장인 이관상이 검소하게 살라고 하자 그는 이를 반박하였다.<sup class="reference" id="cite_ref-isoss_5-16"><a href="#cite_note-isoss-5">[5]</a></sup></p>

<p>그는 가난함은 도덕적으로 고결한 것이 아니라 인간의 정신을 황폐화시키는 것으로 보았다. 그는 '유항산 유항심 무항산 무항심'이라 하여 환경이 사람을 만든다고 하였다. 또한 그는 <a class="mw-redirect" href="/wiki/%ED%92%8D%EC%88%98%EC%A7%80%EB%A6%AC" title="풍수지리">풍수지리</a>설을 비판하여 사람들의 생각과 사회 기풍에 영향을 끼치는 것은 풍수가 아니라 사회 환경이라고 하였다. 그에 따르면 놀기만 좋아하고 일하기 싫어하는 양반 사대부의 나쁜 기풍은 바르지 못한 환경 때문에 조성된 것이라 하였다.</p>

<p>그는 청렴과 청빈함을 강조하는 것은 허위 의식이라고 설파하고 다녔다. 박제가에게 '안빈낙도'는 자신을 속이는 말이었다.<sup class="reference" id="cite_ref-isoss_5-19"><a href="#cite_note-isoss-5">[5]</a></sup> 또한 소비를 샘물에 비유하여 절약보다는 적당한 소비의 중요성을 강조하였다. 무조건적 절약 보다는 화폐를 사용하고 상행위를 해야 경제가 활성화된다는 것이었다.</p>
<h3><span class="mw-headline" id=".EA.B5.AD.EC.88.98.EC.A3.BC.EC.9D.98_.EB.B9.84.ED.8C.90">국수주의 비판</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%B0%95%EC%A0%9C%EA%B0%80&amp;action=edit&amp;section=29" title="부분 편집: 국수주의 비판">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>그는 늘 "언제까지 우리 것만 좋다고 주장할 것인가?<sup class="reference" id="cite_ref-bisos_7-3"><a href="#cite_note-bisos-7">[7]</a></sup>"라며 맹목적인 애국심과 자부심, <a href="/wiki/%EA%B5%AD%EC%88%98%EC%A3%BC%EC%9D%98" title="국수주의">국수주의</a>에 비판을 가하였다.</p>
<p>박제가가 너무나 당연시하고 자랑스러워하는 우리 문화, 우리 것에 대한 습관적인 태도, 맹목적 자부심에 비판을 가하였다.<sup class="reference" id="cite_ref-bisos_7-4"><a href="#cite_note-bisos-7">[7]</a></sup> 그는 형식적인 <a href="/wiki/%EB%B6%81%EB%B2%8C%EB%A1%A0" title="북벌론">북벌론</a>을 말하면서도 북벌에 대한 아무런 준비도 되어 있지 않음을 지적했고, <a href="/wiki/%EC%B2%AD%EB%82%98%EB%9D%BC" title="청나라">청나라</a>가 과거에는 오랑캐였지만 서구의 신문물을 받아들이고 중국 문화에 흡수되었음을 지적했다.</p>
<p>그는 <a href="/wiki/%EC%A1%B0%EC%84%A0" title="조선">조선</a>의 현실이 맹목적으로 애국할 환경인가에 늘 의문을 제기하였다. '사회 상류층도 끼니를 건너뛰기 일쑤고, 선비들은 종이가 없어 책을 쓰지 못하며, 소 한 마리 갖지 못한 농부가 태반인 나라 형편에도 조선은 비효율적인 농본정책과 극단적인 국수주의를 선택했다.<sup class="reference" id="cite_ref-bisos_7-5"><a href="#cite_note-bisos-7">[7]</a></sup>'고 지적했다. '소비와 욕망을 없앤' 비극적인 사회였다. 그러나 그는 세상은커녕 주변사람들조차 설득하지 못했다.<sup class="reference" id="cite_ref-bisos_7-6"><a href="#cite_note-bisos-7">[7]</a></sup> 도리어 그를 역심을 품은 인물로 취급하고 터부시하는 이들도 있었다.</p>
<h3><span class="mw-headline" id=".EC.9D.B8.EC.8A.B5.EC.97.90_.EB.8C.80.ED.95.9C_.EC.A0.80.ED.95.AD">인습에 대한 저항</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%B0%95%EC%A0%9C%EA%B0%80&amp;action=edit&amp;section=30" title="부분 편집: 인습에 대한 저항">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>그는 패기와 자존심으로 똘똘 뭉쳐 시대와 불화하는 길을 선택했다. 그는 당대의 사람들이 지당하게 받아들이는 모든 인습에 저항했다.<sup class="reference" id="cite_ref-isoss_5-20"><a href="#cite_note-isoss-5">[5]</a></sup> 그는 <a href="/wiki/%EC%A1%B0%EC%84%A0" title="조선">조선</a>의 사회를 위선과 가식이 넘치는 사회라고 지적했다.</p>
<p>아교로 붙이고 옻칠을 한 눈꺼풀로 눈을 막고, 귀를 닫고 듣지 않는다고 했다. 그는 늘 '아교로 붙이고 옻칠을 한 눈꺼풀'을 떼어내고 천하를 응시하여 '심지를 열고 이목을 넓히라'고 외쳤다.<sup class="reference" id="cite_ref-isoss_5-21"><a href="#cite_note-isoss-5">[5]</a></sup> 그는 형식적인 것으로 전락해버린 관습이 인간의 본연지성을 무시하고 인간의 자연스러운 생활과 삶과 생각을 틀어막는 장애물이자 짐으로 인식했다.</p>

<p>학문이 실제생활과 국방 등 국가경영에 도움을 주어야 하며, 이를 위해 발달한 외국문물을 받아들여야 한다고 주장해 북학파의 사상을 집대성한 실학자로 꼽힌다. <a href="/wiki/%EC%B2%AD%EB%82%98%EB%9D%BC" title="청나라">청나라</a> 문인 이조원(李調元)은 초정(楚亭) 박제가(朴齊家)의 문집에 쓴 서문을 통해 그의 인물됨됨이를 평했다.<sup class="reference" id="cite_ref-cho01_3-5"><a href="#cite_note-cho01-3">[3]</a></sup> '그 사람은 왜소하지만 굳세고 날카로우며 재치있는 생각이 풍부하다 그의 문장에는 찬란하기가 별빛 같고 조개가 뿜어내는 신기루 같고 용궁의 물과 같은 것이 있다.<sup class="reference" id="cite_ref-cho01_3-6"><a href="#cite_note-cho01-3">[3]</a></sup>'는 것이다. '명분에 매이지 않고 욕망에 솔직했던' 인물이라는 평도 있다.<sup class="reference" id="cite_ref-isoss_5-22"><a href="#cite_note-isoss-5">[5]</a></sup></p>
<p>'박제가는 죽고 죽이는 당쟁이 지배했던 정계 진출보다는 모든 사람들이 잘살 수 있는 나라를 고민하고 꿈꿨다. 비록 자신의 뜻을 제대로 펼치지 못한 채 역사의 뒤안길로 사라졌지만, 누구보다도 날카롭게 현실을 비판하고 통찰했던 선각자다.<sup class="reference" id="cite_ref-bisos_7-7"><a href="#cite_note-bisos-7">[7]</a></sup>'는 평도 있다. 변화의 싹조차 보이지 않았지만, 그럼에도 개혁의 목소리를 높이며 치열하게 살다간 선구자<sup class="reference" id="cite_ref-bisos_7-8"><a href="#cite_note-bisos-7">[7]</a></sup>라는 시각도 있다.</p>
