<h2><span class="mw-headline" id=".EB.82.B4.EC.9A.A9">내용</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%B0%95%EC%A7%84%ED%9D%AC_(%EB%B0%B0%EC%9A%B0)&amp;action=edit&amp;section=1" title="부분 편집: 내용">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p><a href="/wiki/1980%EB%85%84%EB%8C%80" title="1980년대">1980년대</a> 아역 시절에는 <a class="mw-redirect" href="/wiki/MBC" title="MBC">MBC</a> 합창단원으로 활동을 하여 만화주제가나 어린이 드라마 주제곡을 불렀다.</p>
<p>성인이 된후에는 <a href="/wiki/1996%EB%85%84" title="1996년">1996년</a> <a href="/wiki/%ED%95%9C%EA%B5%AD%EB%B0%A9%EC%86%A1%EA%B3%B5%EC%82%AC" title="한국방송공사">한국방송공사</a>의 청소년 드라마 《스타트》로 데뷔하였고, <a href="/wiki/1998%EB%85%84" title="1998년">1998년</a> 영화 《<a href="/wiki/%EC%97%AC%EA%B3%A0%EA%B4%B4%EB%8B%B4" title="여고괴담">여고괴담</a>》에서 그늘진 우등생 소영 역을 맡아 주목을 받았다. 이후 TV 드라마와 영화를 오가며 활동하고 있다. 초기에는 광고 모델로 보여준 섹시한 이미지도 강했으나, 《<a href="/wiki/%EA%B6%81%EB%85%80_(2007%EB%85%84_%EC%98%81%ED%99%94)" title="궁녀 (2007년 영화)">궁녀</a>》, 《<a href="/wiki/%EC%A9%90%EC%9D%98_%EC%A0%84%EC%9F%81" title="쩐의 전쟁">쩐의 전쟁</a>》 등 최근작에서는 정의롭고 바른 이미지의 역할을 많이 맡았다.</p>
<h2><span class="mw-headline" id=".EC.9D.BC.ED.99.94">일화</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%B0%95%EC%A7%84%ED%9D%AC_(%EB%B0%B0%EC%9A%B0)&amp;action=edit&amp;section=2" title="부분 편집: 일화">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<ul>
<li>박진희는 <a href="/wiki/%EC%9D%BC%EC%82%B0%EC%8B%A0%EB%8F%84%EC%8B%9C" title="일산신도시">일산</a> <a href="/wiki/%EC%9E%90%EC%9C%A0%EB%A1%9C" title="자유로">자유로</a>에서 도로 파손으로 인해 자동차 <a href="/wiki/%ED%83%80%EC%9D%B4%EC%96%B4" title="타이어">타이어</a>가 망가지자, 다른 피해자들을 대표하여 <a class="mw-redirect" href="/wiki/%EC%84%9C%EC%9A%B8%EC%8B%9C%EC%B2%AD" title="서울시청">시청</a>으로 부터 보상을 받은 적이 있으며, 동네 하천에 무단 방류되는 <a href="/wiki/%ED%8F%90%EC%88%98" title="폐수">폐수</a>를 신고하였다가 시정되지 않자 인터넷에 민원을 올려서 시정한 일화가 있다.<sup class="reference" id="cite_ref-1"><a href="#cite_note-1">[1]</a></sup></li>
</ul>
<ul>
<li>박진희는 <a href="/wiki/%EC%97%B0%EC%84%B8%EB%8C%80%ED%95%99%EA%B5%90" title="연세대학교">연세대학교</a> 대학원에서 <a class="mw-redirect" href="/wiki/%EC%82%AC%ED%9A%8C%EB%B3%B5%EC%A7%80%ED%95%99" title="사회복지학">사회복지학</a>을 전공하고 있는데, 학교를 통해 맺은 인연으로 <a href="/wiki/2005%EB%85%84" title="2005년">2005년</a> 부터 3년간 매달 <a href="/wiki/%EB%8C%80%ED%95%9C%EB%AF%BC%EA%B5%AD" title="대한민국">대한민국</a> <a class="mw-redirect" href="/wiki/%EC%84%9C%EC%9A%B8" title="서울">서울</a> <a href="/wiki/%EA%B0%80%EC%96%91%EB%8F%99_(%EC%84%9C%EC%9A%B8)" title="가양동 (서울)">가양동</a>에 있는 한 <a class="mw-redirect" href="/wiki/%EB%B3%B5%EC%A7%80%EA%B4%80" title="복지관">복지관</a>에 일정금액을 성금으로 기탁하고, 또 1년에 한번 연말에는 직접 복지관을 찾아서 결손 가정 어린이들을 만났다. 이 일화는 박진희가 진심이 왜곡될까봐 공개를 꺼리었지만, 주민들 사이에서 자연스럽게 소문이 퍼져나가기 시작하면서 알려졌다.<sup class="reference" id="cite_ref-2"><a href="#cite_note-2">[2]</a></sup></li>
</ul>
<ul>
<li><a href="/wiki/2007%EB%85%84" title="2007년">2007년</a> 발생한 <a class="mw-redirect" href="/wiki/2007%EB%85%84_%EC%84%9C%ED%95%B4%EC%95%88_%EC%9B%90%EC%9C%A0_%EC%9C%A0%EC%B6%9C_%EC%82%AC%EA%B3%A0" title="2007년 서해안 원유 유출 사고">2007년 서해안 원유 유출 사고</a> 기름 제거 작업에 박진희도 참가하였는데, 소속사나 언론에는 알리지 않고 참여하였다. 하지만 <a href="/wiki/%EB%88%84%EB%A6%AC%EA%BE%BC" title="누리꾼">누리꾼</a>에 의하여 제거 작업 모습이 포착되어서 알려지게 되었다.<sup class="reference" id="cite_ref-3"><a href="#cite_note-3">[3]</a></sup></li>
</ul>
