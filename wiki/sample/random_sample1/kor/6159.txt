<p><b>박차정</b>(朴次貞, <a href="/wiki/1910%EB%85%84" title="1910년">1910년</a> <a href="/wiki/5%EC%9B%94_7%EC%9D%BC" title="5월 7일">5월 7일</a> ∼ <a href="/wiki/1944%EB%85%84" title="1944년">1944년</a> <a href="/wiki/5%EC%9B%94_27%EC%9D%BC" title="5월 27일">5월 27일</a>)은 <a href="/wiki/%ED%95%9C%EA%B5%AD" title="한국">한국</a>의 독립운동가로, <a href="/wiki/%EA%B9%80%EC%9B%90%EB%B4%89" title="김원봉">김원봉</a>의 두 번째 부인이며<sup class="reference" id="cite_ref-1"><a href="#cite_note-1">[1]</a></sup> 독립운동가이자 북한의 초대 국가원수 겸 초대 최고인민회의 상임위원장 <a href="/wiki/%EA%B9%80%EB%91%90%EB%B4%89" title="김두봉">김두봉</a>의 조카딸이었다. 근우회와 부녀복무단에서 활동했다.</p>
<p><a class="mw-disambig" href="/wiki/%EB%8F%99%EB%9E%98" title="동래">동래</a> <a href="/wiki/%EB%8F%99%EB%9E%98%EC%97%AC%EC%9E%90%EA%B3%A0%EB%93%B1%ED%95%99%EA%B5%90" title="동래여자고등학교">일신여학교</a>에 입학하여 동맹휴학을 주도하고, <a href="/wiki/%EA%B7%BC%EC%9A%B0%ED%9A%8C" title="근우회">근우회</a> 중앙집행위원, 중앙상무위원으로 활동하였다.<sup class="reference" id="cite_ref-new22_2-0"><a href="#cite_note-new22-2">[2]</a></sup> 1930년 근우회사건을 배후에서 지도하다 옥고를 치르기도 하였다. <a href="/wiki/1930%EB%85%84" title="1930년">1930년</a> <a href="/wiki/%EC%A4%91%EA%B5%AD" title="중국">중국</a>으로 망명한 후 <a href="/wiki/%EC%9D%98%EC%97%B4%EB%8B%A8" title="의열단">의열단</a>에 가입, 의열단원으로 활동하다 <a href="/wiki/%EC%9D%98%EC%97%B4%EB%8B%A8" title="의열단">의열단</a>장 <a href="/wiki/%EA%B9%80%EC%9B%90%EB%B4%89" title="김원봉">김원봉</a>과 결혼하였으며 조선혁명군사정치간부학교 여자교관, <a href="/wiki/%EC%A1%B0%EC%84%A0%EB%AF%BC%EC%A1%B1%ED%98%81%EB%AA%85%EB%8B%B9" title="조선민족혁명당">민족혁명당</a> 부녀부 주임, 조선의용대 부녀복무단장 등으로 활동하였다.<sup class="reference" id="cite_ref-new22_2-1"><a href="#cite_note-new22-2">[2]</a></sup> 1939년 2월 장시성 쿤륜산 전투 중에 부상을 입고 그 후유증<sup class="reference" id="cite_ref-new22_2-2"><a href="#cite_note-new22-2">[2]</a></sup>과 성병<sup class="reference" id="cite_ref-3"><a href="#cite_note-3">[3]</a></sup> 등으로 1944년 5월 27일 충칭에서 병사하였다. 독립 운동 중 사용한 가명은 임철애(林哲愛), 임철산(林哲山)이다. <sup class="reference" id="cite_ref-4"><a href="#cite_note-4">[4]</a></sup></p>

<h2><span class="mw-headline" id=".EC.83.9D.EC.95.A0">생애</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%B0%95%EC%B0%A8%EC%A0%95&amp;action=edit&amp;section=1" title="부분 편집: 생애">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<h3><span class="mw-headline" id=".EC.83.9D.EC.95.A0_.EC.B4.88.EB.B0.98">생애 초반</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%B0%95%EC%B0%A8%EC%A0%95&amp;action=edit&amp;section=2" title="부분 편집: 생애 초반">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<div class="thumb tleft">
<div class="thumbinner" style="width:202px;"><a class="image" href="/wiki/%ED%8C%8C%EC%9D%BC:Kwb2.jpg"><img alt="" class="thumbimage" data-file-height="370" data-file-width="500" height="148" src="//upload.wikimedia.org/wikipedia/ko/thumb/c/c9/Kwb2.jpg/200px-Kwb2.jpg" srcset="//upload.wikimedia.org/wikipedia/ko/thumb/c/c9/Kwb2.jpg/300px-Kwb2.jpg 1.5x, //upload.wikimedia.org/wikipedia/ko/thumb/c/c9/Kwb2.jpg/400px-Kwb2.jpg 2x" width="200"/></a>
<div class="thumbcaption">
<div class="magnify"><a class="internal" href="/wiki/%ED%8C%8C%EC%9D%BC:Kwb2.jpg" title="실제 크기로"></a></div>
남편 약산 <a href="/wiki/%EA%B9%80%EC%9B%90%EB%B4%89" title="김원봉">김원봉</a></div>
</div>
</div>
<p>박차정은 <a href="/wiki/1910%EB%85%84" title="1910년">1910년</a> <a href="/wiki/5%EC%9B%94_7%EC%9D%BC" title="5월 7일">5월 7일</a> 당시 <a href="/wiki/%EA%B2%BD%EC%83%81%EB%82%A8%EB%8F%84" title="경상남도">경상남도</a> <a href="/wiki/%EB%8F%99%EB%9E%98%EA%B5%B0" title="동래군">동래군</a>이던 현 <a class="mw-redirect" href="/wiki/%EB%B6%80%EC%82%B0%EC%A7%81%ED%95%A0%EC%8B%9C" title="부산직할시">부산</a>에서 출생했다. 부친은 일제에 항거하여 자결했고, 숙부 <a class="new" href="/w/index.php?title=%EB%B0%95%EC%9D%BC%ED%98%95&amp;action=edit&amp;redlink=1" title="박일형 (없는 문서)">박일형</a>과 외가의 친척들, 오빠들이 항일 운동에 뛰어든 집안에서 자랐다. 일제의 침탈에 항거하여 자결·순국한 아버지와 독립운동가 김두전(金枓全), <a href="/wiki/%EA%B9%80%EB%91%90%EB%B4%89" title="김두봉">김두봉</a>(金枓鳳)과 친척인 어머니, 사회주의계열 독립운동가 숙부 박일형(朴日馨) 등의 영향을 받았다.<sup class="reference" id="cite_ref-new22_2-3"><a href="#cite_note-new22-2">[2]</a></sup> 박차정은 김두봉의 조카딸이었고, 독립운동가 김두봉은 해방 후 북한의 초대 국가원수 겸 초대 최고인민회의 상임위원장이 된 인물이었다.</p>
<p>이러한 집안 분위기 때문에 신간회, 의열단 등에서 활동한 큰오빠 박문희, 둘째오빠 박문호 등과 함께 독립운동에 투신하게 되었다. 그는 <a href="/wiki/%EB%8F%99%EB%9E%98%EC%97%AC%EC%9E%90%EA%B3%A0%EB%93%B1%ED%95%99%EA%B5%90" title="동래여자고등학교">동래여자고등학교</a>의 전신인 일신여학교 시절부터 학생운동에 참가하여 여러차례 체포, 구금 당했다.</p>
<h3><span class="mw-headline" id=".EB.8F.85.EB.A6.BD.EC.9A.B4.EB.8F.99">독립운동</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%B0%95%EC%B0%A8%EC%A0%95&amp;action=edit&amp;section=3" title="부분 편집: 독립운동">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<h4><span class="mw-headline" id=".ED.95.99.EC.B0.BD_.EC.8B.9C.EC.A0.88.EA.B3.BC_.EB.8F.85.EB.A6.BD.EC.9A.B4.EB.8F.99_.EC.B0.B8.EC.97.AC">학창 시절과 독립운동 참여</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%B0%95%EC%B0%A8%EC%A0%95&amp;action=edit&amp;section=4" title="부분 편집: 학창 시절과 독립운동 참여">편집</a><span class="mw-editsection-bracket">]</span></span></h4>
<div class="rellink boilerplate seealso"><img alt="" data-file-height="128" data-file-width="128" height="16" src="//upload.wikimedia.org/wikipedia/commons/thumb/e/ec/Crystal_Clear_app_xmag.svg/16px-Crystal_Clear_app_xmag.svg.png" srcset="//upload.wikimedia.org/wikipedia/commons/thumb/e/ec/Crystal_Clear_app_xmag.svg/24px-Crystal_Clear_app_xmag.svg.png 1.5x, //upload.wikimedia.org/wikipedia/commons/thumb/e/ec/Crystal_Clear_app_xmag.svg/32px-Crystal_Clear_app_xmag.svg.png 2x" width="16"/> <a href="/wiki/%EC%8B%A0%EA%B0%84%ED%9A%8C" title="신간회">신간회</a>, <a href="/wiki/%EA%B7%BC%EC%9A%B0%ED%9A%8C" title="근우회">근우회</a>, <a class="mw-redirect" href="/wiki/6.10_%EB%A7%8C%EC%84%B8_%EC%9A%B4%EB%8F%99" title="6.10 만세 운동">6.10 만세 운동</a> 문서를 참고하십시오.</div>
<p>이곳에서 항일민족의식과 남녀평등사상을 더욱 고취시키며 일신여학교 동맹휴학을 주도하였다. 또한 뛰어난 문학적 기질을 바탕으로 교지 『일신』에 글을 발표하기도 했다.</p>
<p><a href="/wiki/%EA%B2%BD%EC%84%B1%EB%B6%80" title="경성부">경성부</a>로 올라와 <a href="/wiki/%EA%B7%BC%EC%9A%B0%ED%9A%8C" title="근우회">근우회</a>가 결성되자 가입하였다. 근우회 중앙집행위원과 중앙상무위원으로 선임되어 선전과 출판부문에서 활동<sup class="reference" id="cite_ref-new22_2-4"><a href="#cite_note-new22-2">[2]</a></sup>하던 중 1929년 광주학생운동의 연장으로 1930년 1월 서울지역 11개 여학교의 시위투쟁인 이른바 ‘근우회 사건’을 배후에서 지도하였다.<sup class="reference" id="cite_ref-new22_2-5"><a href="#cite_note-new22-2">[2]</a></sup> <a href="/wiki/1930%EB%85%84" title="1930년">1930년</a> <a class="mw-redirect" href="/wiki/%EA%B4%91%EC%A3%BC%ED%95%99%EC%83%9D%EC%9A%B4%EB%8F%99" title="광주학생운동">광주학생운동</a> 후속 시위와 노동자 파업 주동 혐의로 또다시 체포되어 고문을 당하자, 둘째 오빠 <a class="new" href="/w/index.php?title=%EB%B0%95%EB%AC%B8%ED%98%B8&amp;action=edit&amp;redlink=1" title="박문호 (없는 문서)">박문호</a>가 건너가 있던 <a href="/wiki/%EC%A4%91%EA%B5%AD" title="중국">중국</a>으로 망명했다.</p>

<div class="rellink boilerplate seealso"><img alt="" data-file-height="128" data-file-width="128" height="16" src="//upload.wikimedia.org/wikipedia/commons/thumb/e/ec/Crystal_Clear_app_xmag.svg/16px-Crystal_Clear_app_xmag.svg.png" srcset="//upload.wikimedia.org/wikipedia/commons/thumb/e/ec/Crystal_Clear_app_xmag.svg/24px-Crystal_Clear_app_xmag.svg.png 1.5x, //upload.wikimedia.org/wikipedia/commons/thumb/e/ec/Crystal_Clear_app_xmag.svg/32px-Crystal_Clear_app_xmag.svg.png 2x" width="16"/> <a href="/wiki/%EC%9D%98%EC%97%B4%EB%8B%A8" title="의열단">의열단</a>, <a href="/wiki/%EC%A1%B0%EC%84%A0%EA%B3%B5%EC%82%B0%EB%8B%B9" title="조선공산당">조선공산당</a>, <a href="/wiki/%EC%A1%B0%EC%84%A0%EC%9D%98%EC%9A%A9%EA%B5%B0" title="조선의용군">조선의용군</a> 문서를 참고하십시오.</div>
<p><a href="/wiki/1930%EB%85%84" title="1930년">1930년</a> 봄 그는 <a href="/wiki/%EC%A4%91%EA%B5%AD" title="중국">중국</a>에서 <a href="/wiki/%EC%9D%98%EC%97%B4%EB%8B%A8" title="의열단">의열단</a>원으로 활동하는 둘째오빠 <a class="new" href="/w/index.php?title=%EB%B0%95%EB%AC%B8%ED%98%B8&amp;action=edit&amp;redlink=1" title="박문호 (없는 문서)">박문호</a>로부터 중국으로 오라는 연락을 받고 배편으로 중국으로 떠났다. 망명한 그는 국내에서 항일투쟁공로가 오빠와 지인들을 통해 이미 중국까지 알려지면서 공적을 인정받아 <a href="/wiki/%EC%A1%B0%EC%84%A0%EA%B3%B5%EC%82%B0%EB%8B%B9" title="조선공산당">조선공산당</a>재건설동맹 중앙위원과 <a href="/wiki/%EC%9D%98%EC%97%B4%EB%8B%A8" title="의열단">의열단</a> 간부에 선임되었다.</p>

<p><a class="mw-redirect" href="/wiki/%EB%B2%A0%EC%9D%B4%EC%A7%95" title="베이징">베이징</a>에서 <a href="/wiki/%EC%A1%B0%EC%84%A0%EA%B3%B5%EC%82%B0%EB%8B%B9" title="조선공산당">조선공산당</a> 재건 운동을 벌이다가 <a href="/wiki/%EC%95%88%EA%B4%91%EC%B2%9C" title="안광천">안광천</a>의 소개로 <a href="/wiki/%EC%9D%98%EC%97%B4%EB%8B%A8" title="의열단">의열단</a> 단장 김원봉과 <a href="/wiki/1931%EB%85%84" title="1931년">1931년</a> 결혼했고, 김원봉(金元鳳)과 결혼하면서 본격적으로 의열단의 핵심멤버로 활약하였다. <a href="/wiki/%EA%B9%80%EC%9B%90%EB%B4%89" title="김원봉">김원봉</a>은 본래 <a href="/wiki/%ED%8F%89%EC%95%88%EB%8F%84" title="평안도">평안도</a> 출신의 최석진이라는 사람의 딸과 결혼한다.<sup class="reference" id="cite_ref-hyuk244_5-0"><a href="#cite_note-hyuk244-5">[5]</a></sup> 그러나 본처와 헤어지고 나중에 새 장가를 든다. 후일 <a href="/wiki/%EC%A0%95%ED%99%94%EC%95%94" title="정화암">정화암</a>은 그가 <strong class="selflink">박차정</strong>을 만나기 전에 본처가 있었으며 김두봉의 조카딸(박차정)과의 결혼은 새장가라고 증언하였다.<sup class="reference" id="cite_ref-hyuk244_5-1"><a href="#cite_note-hyuk244-5">[5]</a></sup></p>
<h4><span class="mw-headline" id=".EA.B5.B0.EC.82.AC_.ED.99.9C.EB.8F.99">군사 활동</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%B0%95%EC%B0%A8%EC%A0%95&amp;action=edit&amp;section=6" title="부분 편집: 군사 활동">편집</a><span class="mw-editsection-bracket">]</span></span></h4>

<p>이후 난징(南京)으로 거주지를 옮겨 남편이 <a class="mw-redirect" href="/wiki/%EB%82%9C%EC%A7%95" title="난징">난징</a>에 청년투사를 양성하기 위해 설립한 <a class="new" href="/w/index.php?title=%EC%A1%B0%EC%84%A0%ED%98%81%EB%AA%85%EA%B5%B0%EC%82%AC%EC%A0%95%EC%B9%98%EA%B0%84%EB%B6%80%ED%95%99%EA%B5%90&amp;action=edit&amp;redlink=1" title="조선혁명군사정치간부학교 (없는 문서)">조선혁명군사정치간부학교</a> 교관으로 일했다. 당시 그는‘임철애, 임철산’ 등 가명을 사용하며 교관과 지휘관으로 활동하였다.</p>
<p>이후 <a class="mw-redirect" href="/wiki/%EB%AF%BC%EC%A1%B1%ED%98%81%EB%AA%85%EB%8B%B9" title="민족혁명당">민족혁명당</a>의 <a class="new" href="/w/index.php?title=%EB%82%A8%EA%B2%BD%EC%A1%B0%EC%84%A0%EB%B6%80%EB%85%80%ED%9A%8C&amp;action=edit&amp;redlink=1" title="남경조선부녀회 (없는 문서)">남경조선부녀회</a>를 조직했고, <a href="/wiki/%EC%A4%91%EC%9D%BC_%EC%A0%84%EC%9F%81" title="중일 전쟁">중일 전쟁</a>이 시작되자 기고 및 방송 활동으로 일본의 침략 전쟁을 규탄하고 조선 부녀자들의 일치 단결과 투쟁을 촉구하는 등 독립 운동에 매진했다.</p>

<p>만년에 일본군과의 교전에서 얻은 총상의 후유증으로 고생하였다. 또한 지병인 <a href="/wiki/%EA%B4%80%EC%A0%88%EC%97%BC" title="관절염">관절염</a>으로도 오래 고생하였다.</p>
<p><a href="/wiki/%EC%A1%B0%EC%84%A0%EB%AF%BC%EC%A1%B1%EC%A0%84%EC%84%A0%EC%97%B0%EB%A7%B9" title="조선민족전선연맹">조선민족전선연맹</a>이 한중 연합 전선 무장 세력인 <a href="/wiki/%EC%A1%B0%EC%84%A0%EC%9D%98%EC%9A%A9%EB%8C%80" title="조선의용대">조선의용대</a>를 구성하자 여군 격인 부녀복무단을 설치하여 단장을 맡았다. <a href="/wiki/1939%EB%85%84" title="1939년">1939년</a> 2월 <a href="/wiki/%EC%9E%A5%EC%8B%9C_%EC%84%B1" title="장시 성">장시 성</a>(江西省) 쿤륜산(昆崙山) 전투 중에서 일본군과 전투 중 입은 총상의 후유증과 <a href="/wiki/%EA%B4%80%EC%A0%88%EC%97%BC" title="관절염">관절염</a> 등의 후유증을 앓았다. 또한 <a href="/wiki/%EC%84%B1%EB%B3%91" title="성병">성병</a>도 앓았는데, <a href="/wiki/%EC%A0%95%ED%99%94%EC%95%94" title="정화암">정화암</a>에 의하면 박차정은 <a href="/wiki/%EC%84%B1%EB%B3%91" title="성병">성병</a>으로 사망했다 한다.<sup class="reference" id="cite_ref-hyuk244_5-2"><a href="#cite_note-hyuk244-5">[5]</a></sup> <sup class="reference" id="cite_ref-6"><a href="#cite_note-6">[6]</a></sup> <a href="/wiki/1944%EB%85%84" title="1944년">1944년</a> <a href="/wiki/5%EC%9B%94_27%EC%9D%BC" title="5월 27일">5월 27일</a> <a class="mw-redirect" href="/wiki/%EC%B6%A9%EC%B9%AD" title="충칭">충칭</a>에서 병사했다. 당시 그의 나이 향년 34세였다.</p>
<p>유해는 해방 직후 <a href="/wiki/1945%EB%85%84" title="1945년">1945년</a> <a href="/wiki/12%EC%9B%94" title="12월">12월</a> 송환, 김원봉의 고향인 <a href="/wiki/%EB%B0%80%EC%96%91%EC%8B%9C" title="밀양시">밀양</a>의 송산공동묘지에 안장되었다. 작품으로는 시 〈개구리 소래〉와 소설 〈철야〉, <a href="/wiki/%EC%9D%BC%EB%B3%B8%EC%96%B4" title="일본어">일본어</a>로 쓴 수필 〈가을 아침〉 등이 있다.</p>
<h3><span class="mw-headline" id=".EC.82.AC.ED.9B.84">사후</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%B0%95%EC%B0%A8%EC%A0%95&amp;action=edit&amp;section=8" title="부분 편집: 사후">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>사회주의 계열에서 활동했고 남편 김원봉도 월북, 북한에서 장관급 각료를 역임했기 때문에 평가받지 못하였다. 공적에 비해서는 뒤늦게 <a href="/wiki/1995%EB%85%84" title="1995년">1995년</a> <a href="/wiki/%EA%B1%B4%EA%B5%AD%ED%9B%88%EC%9E%A5" title="건국훈장">건국훈장</a> 독립장이 추서되었다. 부산시 동래구현, 동래고등학교 맞은편에 생가가 복원되고<sup class="reference" id="cite_ref-7"><a href="#cite_note-7">[7]</a></sup> <a class="mw-redirect" href="/wiki/%EB%B6%80%EC%82%B0_%EA%B8%88%EC%A0%95%EA%B5%AC" title="부산 금정구">부산 금정구</a>에 동상이 마련되어 있다.<sup class="reference" id="cite_ref-8"><a href="#cite_note-8">[8]</a></sup></p>
