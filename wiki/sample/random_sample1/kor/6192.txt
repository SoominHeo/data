<h2><span class="mw-headline" id=".EC.9A.94.EC.95.BD">요약</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%B0%98%EC%A0%A4%EB%A6%AC%EC%8A%A4&amp;action=edit&amp;section=1" title="부분 편집: 요약">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p><a href="/wiki/1961%EB%85%84" title="1961년">1961년</a>도부터 <a href="/wiki/%ED%98%84%EC%9E%AC" title="현재">현재</a>까지 세계적으로 왕성히 활동하고 있는 <a href="/wiki/%EC%9D%8C%EC%95%85%EA%B0%80" title="음악가">음악가</a>이다.</p>
<p>미국에서는 <a href="/wiki/%EC%98%81%ED%99%94" title="영화">영화</a> '<a href="/wiki/%EB%B6%88%EC%9D%98_%EC%A0%84%EC%B0%A8" title="불의 전차">불의 전차</a>'의 <a class="new" href="/w/index.php?title=%EB%B6%88%EC%9D%98_%EC%A0%84%EC%B0%A8(%EC%95%A8%EB%B2%94)&amp;action=edit&amp;redlink=1" title="불의 전차(앨범) (없는 문서)">음악</a>이 <a href="/wiki/1982%EB%85%84" title="1982년">1982년</a> 3월 3월에 발표된 <a class="new" href="/w/index.php?title=%EC%A0%9C54%ED%9A%8C_%EC%95%84%EC%B9%B4%EB%8D%B0%EB%AF%B8_%EC%83%81&amp;action=edit&amp;redlink=1" title="제54회 아카데미 상 (없는 문서)">제54회 아카데미 상</a><a class="new" href="/w/index.php?title=%EC%95%84%EC%B9%B4%EB%8D%B0%EB%AF%B8_%EC%9E%91%EA%B3%A1%EC%83%81&amp;action=edit&amp;redlink=1" title="아카데미 작곡상 (없는 문서)">정규 작곡상</a>을 수상. 같은 해 빌보드 앨범 및 싱글 차트 1위를 차지. 싱글 5월 8일까지 1위. 또한 1982년 <a href="/wiki/%EB%A6%AC%EB%93%A4%EB%A6%AC_%EC%8A%A4%EC%BD%A7" title="리들리 스콧">리들리 스콧</a> 감독 영화 '<a href="/wiki/%EB%B8%94%EB%A0%88%EC%9D%B4%EB%93%9C_%EB%9F%AC%EB%84%88" title="블레이드 러너">블레이드 러너</a>'의 <a class="new" href="/w/index.php?title=%EB%B8%94%EB%A0%88%EC%9D%B4%EB%93%9C_%EB%9F%AC%EB%84%88(%EC%95%A8%EB%B2%94)&amp;action=edit&amp;redlink=1" title="블레이드 러너(앨범) (없는 문서)">음악</a>을 담당했다.</p>
<p>활동 거점을 유럽으로 잡은 그는 1974년 결성된 <a class="new" href="/w/index.php?title=%EC%98%88%EC%8A%A4(%EB%B0%B4%EB%93%9C)&amp;action=edit&amp;redlink=1" title="예스(밴드) (없는 문서)">예스</a>에의 가입 소동 이전부터 솔로 앨범을 발매했다. 1992년 리들리 스콧 감독 작품 '<a class="new" href="/w/index.php?title=1492_%EC%BD%9C%EB%9F%BC%EB%B2%84%EC%8A%A4&amp;action=edit&amp;redlink=1" title="1492 콜럼버스 (없는 문서)">1492 콜럼버스</a>'의 <a class="new" href="/w/index.php?title=1492_%EC%BD%9C%EB%9F%BC%EB%B2%84%EC%8A%A4(%EC%95%A8%EB%B2%94)&amp;action=edit&amp;redlink=1" title="1492 콜럼버스(앨범) (없는 문서)">음악</a>이 독일에서 앨범이 100만장, 싱글 컷 된 <a class="new" href="/w/index.php?title=1492_%EC%BD%9C%EB%9F%BC%EB%B2%84%EC%8A%A4&amp;action=edit&amp;redlink=1" title="1492 콜럼버스 (없는 문서)">1492 콜럼버스</a>가 150만장 팔린 다른 <a href="/wiki/%EC%9C%A0%EB%9F%BD" title="유럽">유럽</a> 17개국에서 골드 디스크, 플래티넘 디스크를 획득하여 '불의 전차'를 능가하는 히트를 쳤다.</p>
<p>정규 앨범 외에 영화 음악 앨범의 다른 TV 다큐멘터리, 연극 음악, 무용 음악 등도 다수 작곡하고 있다. <a href="/wiki/%EC%B9%BC_%EC%84%B8%EC%9D%B4%EA%B1%B4" title="칼 세이건">칼 세이건</a> 제작의 TV 다큐멘터리 '<a class="new" href="/w/index.php?title=COSMOS&amp;action=edit&amp;redlink=1" title="COSMOS (없는 문서)">COSMOS</a>'에서 그의 음악이 사용되었다(테마곡으로 사용된 앨범 '천국과 지옥'에서 한 구절, 앨범 '<a class="mw-redirect" href="/wiki/%EB%B0%98%EC%82%AC%EC%9C%A8_0.39" title="반사율 0.39">반사율 0.39</a>'에 수록되는 "<a class="mw-disambig" href="/wiki/%EC%95%8C%ED%8C%8C" title="알파">알파</a>"등).</p>
<p>최근에는 순수 음악 활동 보다 국제 이벤트와 관련된 활동을 하는데, <a href="/wiki/2000%EB%85%84" title="2000년">2000년</a> 시드니 올림픽 폐막식에 호주에서 그리스에 올림픽 깃발을 인수할 때 음악과 지휘를 담당, 또한 <a href="/wiki/2004%EB%85%84" title="2004년">2004년</a> 아테네 올림픽에서는 공식 엠블럼에 수반되는 로고 음악도 다루고 있다. 또한 <a href="/wiki/2001%EB%85%84" title="2001년">2001년</a> NASA의 화성 탐사 계획의 하나인 <a href="/wiki/2001_%EB%A7%88%EC%8A%A4_%EC%98%A4%EB%94%94%EC%84%B8%EC%9D%B4" title="2001 마스 오디세이">2001 마스 오디세이</a>의 테마 음악을 담당, 관련하여 아테네의 제우스 신전에서 실시되는 라이브 이벤트 "<a href="/wiki/Mythodea" title="Mythodea">Mythodea</a>"를 다루는 등의 활동을 했다.</p>
<h2><span class="mw-headline" id=".EC.95.BD.EB.A0.A5">약력</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%B0%98%EC%A0%A4%EB%A6%AC%EC%8A%A4&amp;action=edit&amp;section=2" title="부분 편집: 약력">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p><a href="/wiki/%EA%B7%B8%EB%A6%AC%EC%8A%A4" title="그리스">그리스</a> 중부의 <a href="/wiki/%ED%95%AD%EA%B5%AC" title="항구">항구</a> 도시 <a href="/wiki/%EB%B3%BC%EB%A1%9C%EC%8A%A4" title="볼로스">볼로스</a>에서 <a href="/wiki/%ED%99%94%EA%B0%80" title="화가">화가</a>인 <a href="/wiki/%EC%95%84%EB%B2%84%EC%A7%80" title="아버지">아버지</a>와 <a href="/wiki/%EC%9D%8C%EC%95%85%EA%B0%80" title="음악가">음악가</a>인 <a href="/wiki/%EC%96%B4%EB%A8%B8%EB%8B%88" title="어머니">어머니</a> 사이에서 태어났다. 4세부터 독학으로 피아노와 작곡을 시작, 6세에 자작 리사이틀을 열었다고 전해진다. 고교 시절부터 전문적인 음악 활동을 시작, <a href="/wiki/%EC%9E%AC%EC%A6%88" title="재즈">재즈</a>에서 <a class="mw-disambig" href="/wiki/%EB%B0%94%EC%9D%B4%EB%B8%8C" title="바이브">바이브</a>와 <a href="/wiki/%ED%94%BC%EC%95%84%EB%85%B8" title="피아노">피아노</a>, <a href="/wiki/%EC%98%A4%EB%A5%B4%EA%B0%84" title="오르간">오르간</a> 등을 연주했다. 고교 졸업 후, 음악 학교 등에는 진학하지 않고 예술 학교에서 영화와 미술을 배웠다. 또한, 음악 이론은 아리스토텔레스라는 인물에게 개인 교습을 받았다고 한다.</p>
<p><a href="/wiki/1963%EB%85%84" title="1963년">1963년</a> 아테네에서 팝 밴드 "<a class="new" href="/w/index.php?title=%ED%8F%AC%EB%B0%8D%EC%8A%A4&amp;action=edit&amp;redlink=1" title="포밍스 (없는 문서)">포밍스</a>"를 결성, 다수의 싱글을 발표, 그리스 국내에서 히트한다.</p>
<p>다음, <a href="/wiki/%EB%8D%B0%EB%AF%B8%EC%8A%A4_%EB%A3%A8%EC%86%8C%EC%8A%A4" title="데미스 루소스">데미스 루소스</a>(Demis Roussos, 베이스, 보컬), <a class="new" href="/w/index.php?title=%EB%A3%A8%EC%B9%B4%EC%8A%A4_%EC%8B%9C%EB%8D%B0%EB%9D%BC%EC%8A%A4&amp;action=edit&amp;redlink=1" title="루카스 시데라스 (없는 문서)">루카스 시데라스</a>(Loukas Sideras, 드럼), <a class="new" href="/w/index.php?title=%EC%95%84%EA%B8%B0%EB%A6%AC%EC%8A%A4_%EC%BD%94%EB%A1%9C%EB%A6%AC%EC%8A%A4&amp;action=edit&amp;redlink=1" title="아기리스 코로리스 (없는 문서)">아기리스 코로리스</a>(Argyris Koulouris, 기타)등 와 'The Papathanassiou Set'을 결성, 가수활동을 시작한다.</p>
<p><a href="/wiki/1967%EB%85%84" title="1967년">1967년</a>에 '5000 Lies'의 음악으로 처음으로 영화 작품에 관한 작품을 쓴다.</p>
<p><a href="/wiki/1968%EB%85%84" title="1968년">1968년</a> 군사 쿠데타를 계기로 데미스, 루카스와 영국 런던을 중심으로 활동하려 했으나, 시대적 상황으로 프랑스 파리에 남았다. 어쩔 수 없이 3명은 파리에서 <a class="new" href="/w/index.php?title=%EC%95%84%ED%94%84%EB%A1%9C%EB%94%94%ED%85%8C%EC%8A%A4_%EC%B0%A8%EC%9D%BC%EB%93%9C&amp;action=edit&amp;redlink=1" title="아프로디테스 차일드 (없는 문서)">아프로디테스 차일드</a>(Aphrodite's Child)를 결성, 싱글 "Rain and Tears"를 발표했는데 뜻밖에 큰 성공을 거두었다. 그래서 음반사와 정식 계약 후 "I Want to Live", "Spring, Summer, Winter and Fall"등의 히트곡을 낳았다. 그 후 <a href="/wiki/1969%EB%85%84" title="1969년">1969년</a> 산레모 음악제에 참가, 유럽을 중심으로 인기를 모은다. 앨범 '<a class="new" href="/w/index.php?title=End_of_the_World&amp;action=edit&amp;redlink=1" title="End of the World (없는 문서)">End of the World</a>'와 '<a class="new" href="/w/index.php?title=It%27s_Five_O%27Clock&amp;action=edit&amp;redlink=1" title="It's Five O'Clock (없는 문서)">It's Five O'Clock</a>'을 발표 후, 음악 견해 차이로 해산한다. 그러나 <a href="/wiki/1972%EB%85%84" title="1972년">1972년</a> 마지막으로, <a class="mw-redirect" href="/wiki/%EC%8B%A0%EC%95%BD" title="신약">신약</a> 성경의 "요한 계시록"을 소재로 한 2 Disc 앨범 '666'을 제작한다.</p>
<p>이후 영화 음악과 정규작품을 발표하고, <a href="/wiki/1973%EB%85%84" title="1973년">1973년</a>에는 사실상 솔로 앨범 '지구'를 유럽에서 선보인다. <a href="/wiki/1974%EB%85%84" title="1974년">1974년</a> 영국으로 건너가 런던에서 개인 스튜디오 <a class="new" href="/w/index.php?title=NEMO&amp;action=edit&amp;redlink=1" title="NEMO (없는 문서)">NEMO</a>를 개설, 대기업 레코드 회사 <a class="mw-disambig" href="/wiki/RCA" title="RCA">RCA</a> 레코드와 계약을 맺고, 1975년 '천국과 지옥'을 시작으로, 전 세계를 대상으로 작품을 선보인다. 또한, <a href="/wiki/1974%EB%85%84" title="1974년">1974년</a> <a href="/wiki/%EB%A6%AD_%EC%9B%A8%EC%9D%B4%ED%81%AC%EB%A8%BC" title="릭 웨이크먼">릭 웨이크먼</a>(키보드)의 후임으로 <a href="/wiki/%EC%98%88%EC%8A%A4" title="예스">예스</a>(YES)에 가입하려 하였으나 여러가지 난관으로 실패한다. 그러나 이때부터 메인 보컬리스트 <a class="mw-disambig" href="/wiki/%EC%A1%B4_%EC%95%A4%EB%8D%94%EC%8A%A8" title="존 앤더슨">존 앤더슨</a>(Jon Anderson)과의 교제가 시작되어, <a href="/wiki/1979%EB%85%84" title="1979년">1979년</a> 이후 <a class="new" href="/w/index.php?title=%EC%A1%B4_%EC%95%A4_%EB%B0%98%EC%A0%A4%EB%A6%AC%EC%8A%A4&amp;action=edit&amp;redlink=1" title="존 앤 반젤리스 (없는 문서)">존 앤 반젤리스</a>(Jon &amp; Vangelis)라는 이름으로 공​​동 작업을 하였다. <a href="/wiki/1980%EB%85%84" title="1980년">1980년</a>대에는 영화 '불의 전차'(1981년)와 '블레이드 러너'(1982년), 그리고 <a href="/wiki/%EC%9D%BC%EB%B3%B8" title="일본">일본</a>에서는 '<a href="/wiki/%EB%82%A8%EA%B7%B9_%EC%9D%B4%EC%95%BC%EA%B8%B0" title="남극 이야기">남극 이야기</a>'(1983년)등의 사운드 트랙을 담당하여 화제가 된다.</p>
<p><a href="/wiki/1980%EB%85%84" title="1980년">1980년</a>대 중반에 다시 <a class="mw-disambig" href="/wiki/%ED%8C%8C%EB%A6%AC" title="파리">파리</a>에서 활동하였으나, <a href="/wiki/1989%EB%85%84" title="1989년">1989년</a> <a href="/wiki/%EA%B7%B8%EB%A6%AC%EC%8A%A4" title="그리스">그리스</a>에 정착하였고, 이후 <a href="/wiki/%EC%95%84%ED%85%8C%EB%84%A4" title="아테네">아테네</a>에 자리를 잡고 창작 활동을 계속하고 있다.</p>
<h2><span class="mw-headline" id=".EC.9D.8C.EC.95.85.EC.84.B1">음악성</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%B0%98%EC%A0%A4%EB%A6%AC%EC%8A%A4&amp;action=edit&amp;section=3" title="부분 편집: 음악성">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>반젤리스의 음악의 특징은 주로 다음과 같은 점을 들 수 있다.</p>
<h3><span class="mw-headline" id=".EB.A9.9C.EB.A1.9C.EB.94.94">멜로디</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%B0%98%EC%A0%A4%EB%A6%AC%EC%8A%A4&amp;action=edit&amp;section=4" title="부분 편집: 멜로디">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p><strong class="selflink">반젤리스</strong>의 <a class="mw-redirect" href="/wiki/%EB%A9%9C%EB%A1%9C%EB%94%94" title="멜로디">멜로디</a>는 아름답고 강한 인상을 심어준다. 그리스와 지중해 동부 지역에 예부터 전해지는 5음계 선법에 기초한 멜로디를 사용하지만, 5번의 도약(도에서 솔에 : '불의 전차'의 서주 부분 등)을 즐겨 사용하는 것도 특징이다.</p>
<h3><span class="mw-headline" id=".EA.B5.AC.EC.84.B1">구성</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%B0%98%EC%A0%A4%EB%A6%AC%EC%8A%A4&amp;action=edit&amp;section=5" title="부분 편집: 구성">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p><a href="/wiki/%EA%B5%90%ED%96%A5%EA%B3%A1" title="교향곡">교향곡</a>과 같은 여러 악장으로 구성되는 전체 앨범이며, 앨범 전체가 하나의 콘셉트로 이어지고 있는 경우가 많다. 특히 '<a class="mw-redirect" href="/wiki/%EB%8C%80%EC%A7%80%EC%9D%98_%EC%A0%9C%EB%A1%80" title="대지의 제례">대지의 제례</a>'와 '<a class="mw-disambig" href="/wiki/%EB%A7%88%EC%8A%A4%ED%81%AC" title="마스크">마스크</a>', '<a href="/wiki/%EC%97%98_%EA%B7%B8%EB%A0%88%EC%BD%94" title="엘 그레코">엘 그레코</a>' '<a href="/wiki/Mythodea" title="Mythodea">Mythodea</a>'등이 각 파트에 고유한 제목을 붙이지 않고 있다.</p>
<h3><span class="mw-headline" id=".EC.9E.91.EA.B3.A1.EA.B3.BC_.ED.8E.B8.EA.B3.A1">작곡과 편곡</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%B0%98%EC%A0%A4%EB%A6%AC%EC%8A%A4&amp;action=edit&amp;section=6" title="부분 편집: 작곡과 편곡">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p><strong class="selflink">반젤리스</strong>는 <a href="/wiki/%EC%95%85%EB%B3%B4" title="악보">악보</a>를 읽고 쓸 수 없다고 말한 적 있다. 또한 멀티 키보드 형식으로 각종 악기를 주변에 배치하며, 즉흥적으로 연주하는 것이 반젤리스 작곡의 특징이다. 대부분의 악기를 혼자서 연주하고 <a href="/wiki/%EC%8A%A4%ED%8A%9C%EB%94%94%EC%98%A4" title="스튜디오">스튜디오</a>에서 <a class="new" href="/w/index.php?title=%EB%8B%A4%EC%A4%91_%EB%85%B9%EC%9D%8C&amp;action=edit&amp;redlink=1" title="다중 녹음 (없는 문서)">다중 녹음</a>을 한다. 또한 신시사이저를 이용하여 각 악기를 종횡으로 구사한 다채로운 중후한 편곡 양식을 취한다. 특히 "<a href="/wiki/%EC%95%BC%EB%A7%88%ED%95%98" title="야마하">야마하</a> <a class="new" href="/w/index.php?title=CS-80&amp;action=edit&amp;redlink=1" title="CS-80 (없는 문서)">CS-80</a>"과 각 폴리포니 오케스트라를 구성하여 곡을 쓸 때 이용한다. 또한 다양한 민족 악기와 타악기를 사용하여 독특한 색채의 느낌과 향기를 내는 곡도 많다. 반면에, 재즈적인 <a class="new" href="/w/index.php?title=%EC%9E%84%ED%94%84%EB%A1%9C%EB%B9%84%EC%A0%9C%EC%9D%B4%EC%85%98&amp;action=edit&amp;redlink=1" title="임프로비제이션 (없는 문서)">임프로비제이션</a>의 전개와 현대 음악적인 실험성도 보유하고 있으며 1970년대의 라이브 공연도 대부분이 즉흥 연주였다. 이러한 작/편곡 방법을 자신은 "자발적 음악(Sponteus Music)"라고 부르고 있다.</p>
<h3><span class="mw-headline" id=".EC.9E.A5.EB.A5.B4">장르</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%B0%98%EC%A0%A4%EB%A6%AC%EC%8A%A4&amp;action=edit&amp;section=7" title="부분 편집: 장르">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p><strong class="selflink">반젤리스</strong> 작품은 각 장르를 총망라하는 경우가 많다. 앨범 타이틀에 대한 음반사의 분류 역시 소속사와 시기에 따라 다음과 같이 차이가 있다.</p>
<ul>
<li><b>RCA 레코드 : <a class="mw-disambig" href="/wiki/%EB%A1%9D" title="록">록</a>('반사율 0.39' 외)</b></li>
<li><b>Polydor : 신시사이저('차이나'), 록('마스크')</b></li>
<li><b>이스트 웨스트 재팬 : <a class="mw-disambig" href="/wiki/%ED%8C%9D" title="팝">팝</a>('오셔닉')</b></li>
<li><b>유니버설 뮤직 : 팝('Odyssey - The Definitive Collection')</b></li>
</ul>
<h3><span class="mw-headline" id=".EC.9E.AC.EC.A6.88_.EC.97.B0.EC.A3.BC.EC.9E.90.EB.A1.9C.EC.84.9C.EC.9D.98_.EB.B0.98.EC.A0.A4.EB.A6.AC.EC.8A.A4">재즈 연주자로서의 반젤리스</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%B0%98%EC%A0%A4%EB%A6%AC%EC%8A%A4&amp;action=edit&amp;section=8" title="부분 편집: 재즈 연주자로서의 반젤리스">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>신시사이저와 만날 때까지의 반젤리스는 주로 피아노와 오르간, 바이브, <a class="mw-redirect" href="/wiki/%ED%8D%BC%EC%BB%A4%EC%85%98" title="퍼커션">퍼커션</a> 등을 연주했다. 1964~65년경에 그리스의 재즈 피아니스트 George Theodossiadis의 <a class="mw-disambig" href="/wiki/%EC%BD%B0%EB%A5%B4%ED%85%9F" title="콰르텟">콰르텟</a>에서 바이브를 연주했는데 그리스에서 CD화 되었다(Legend / El Capitan 2201556072). <a class="new" href="/w/index.php?title=%EC%8A%A4%ED%83%A0%EB%8D%94%EB%93%9C_%EB%84%98%EB%B2%84&amp;action=edit&amp;redlink=1" title="스탠더드 넘버 (없는 문서)">스탠더드 넘버</a>부터 <a href="/wiki/%EB%B3%B4%EC%82%AC%EB%85%B8%EB%B0%94" title="보사노바">보사노바</a>까지 다양한 장르에서 탁월한 음악을 들려주었다. 반젤리스 음악 사상 가장 오래된 기록으로 보인다.</p>
<h2><span class="mw-headline" id=".EC.A3.BC.EC.9A.94_.EC.9E.91.ED.92.88">주요 작품</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%B0%98%EC%A0%A4%EB%A6%AC%EC%8A%A4&amp;action=edit&amp;section=9" title="부분 편집: 주요 작품">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>반젤리스 작품은 엄청나지만, 그 중에서도 명작으로 불리는 작품 또는 반젤리스에게 전환점이 된 작품이 존재한다(본문에 포함된 차트 순위는 '디렉트' 계정에서 참조할 것!).</p>
<h3><span class="mw-headline" id="RCA_.EC.8B.9C.EB.8C.80_.EC.9D.B4.EC.A0.84">RCA 시대 이전</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%B0%98%EC%A0%A4%EB%A6%AC%EC%8A%A4&amp;action=edit&amp;section=10" title="부분 편집: RCA 시대 이전">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<ul>
<li><b>비와 눈물(Rain and Tears / 싱글)</b></li>
</ul>
<p>1968년에 출시된 아프로디테스 차일드의 싱글이다. 이 싱글은 유럽 전역에서 히트하였다.</p>
<ul>
<li><b>666</b></li>
</ul>
<p>아프로디테스 차일드가 일시적으로 다시 결성하고 1972년 제작한 앨범이다. 밴드의 앨범이지만 작곡은 전곡 반젤리스의 작품이었다.</p>
<ul>
<li><b>지구(Earth)</b></li>
</ul>
<p>1973년에 출시된 반젤리스의 실질적인 첫 정규 솔로 앨범이다.</p>
<h3><span class="mw-headline" id="RCA_.EC.8B.9C.EB.8C.80">RCA 시대</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%B0%98%EC%A0%A4%EB%A6%AC%EC%8A%A4&amp;action=edit&amp;section=11" title="부분 편집: RCA 시대">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<ul>
<li><b>천국과 지옥(Heaven and Hell)</b></li>
<li><b>반사율 0.39(Albedo 0.39)</b></li>
<li><b>나선(Spiral)</b></li>
</ul>
<p>'천국과 지옥'은 1975년에 출시되었다. 반젤리스 최초로 메이저 회사 RCA 에서 발매한 앨범이며, 영국 차트 31위까지 상승했다. (다음 작인 '반사율 0.39'는 영국 차트 18위를 기록했다.) '천국과 지옥'은 1974년 예스 메인 보컬리스트 <a class="mw-disambig" href="/wiki/%EC%A1%B4_%EC%95%A4%EB%8D%94%EC%8A%A8" title="존 앤더슨">존 앤더슨</a>이 보컬로 참여하고 있는 것으로, 제1부 제3악장의 TV시리즈 'COSMOS'의 오프닝에 사용된 것으로 유명하다. '반사율 0.39'와 '나선'도 뉴스 프로그램의 오프닝과 버라이어티 프로그램(타로 감정 결과 발표 BGM 등)에 사용되는 등 반젤리스를 모르는 사람도 귀에 익숙한 수록곡이 많은 작품이다. 또한, '일요일 석간! 여기 데스크'의 테마곡으로 알려진 "펄스타"('반사율 0.39' 제1곡)는 후에 폴 모리아에 의해 커버 연주되었다.</p>
<h3><span class="mw-headline" id="Polydor_.EC.8B.9C.EB.8C.80">Polydor 시대</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%B0%98%EC%A0%A4%EB%A6%AC%EC%8A%A4&amp;action=edit&amp;section=12" title="부분 편집: Polydor 시대">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<ul>
<li><b>쇼트 스토리(Short Stories)</b></li>
</ul>
<p>1979년에 발표된 존 앤더슨과의 조인트 프로젝트 "존 앤 반젤리스"의 첫 앨범. 영국 차트 4위까지 상승했다.</p>
<ul>
<li><b>불의 전차(Chariots of Fire)</b></li>
</ul>
<p>1981년 동명 영화의 음악이다. 앨범 미국 1위(브리티시 5위)가 되고, 아카데미상 정규 작곡 상을 수상했다.</p>
<ul>
<li><b>남극 이야기(Antarctica)</b></li>
</ul>
<p>1983년에 발표된 일본 영화 '<a href="/wiki/%EB%82%A8%EA%B7%B9_%EC%9D%B4%EC%95%BC%EA%B8%B0" title="남극 이야기">남극 이야기</a>'의 사운드 트랙이다. 이 영화가 당시의 일본 영화에서 No.1 히트를 쳤다(또한, 이 기록은 '<a class="mw-redirect" href="/wiki/%EC%9B%90%EB%A0%B9%EA%B3%B5%EC%A3%BC" title="원령공주">원령공주</a>'가 뽑힐 때까지 지속되었다).</p>
<h3><span class="mw-headline" id=".EC.9B.8C.EB.84.88_.EB.AE.A4.EC.A7.81_.EC.8B.9C.EB.8C.80">워너 뮤직 시대</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%B0%98%EC%A0%A4%EB%A6%AC%EC%8A%A4&amp;action=edit&amp;section=13" title="부분 편집: 워너 뮤직 시대">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<ul>
<li><b>1492 콜럼버스(1492 - Conquest of Paradise)</b></li>
</ul>
<p>1992년에 출시된 동명 영화의 사운드 트랙으로, 특히 유럽에서 히트했다(개요 참조).</p>
<ul>
<li><b>블레이드 러너(Blade Runner)</b></li>
</ul>
<p>1994년 출시되었다. 실제로는 사운드 트랙이지만, 부분적으로 사용되지 않는 곡이 수록 되어 있는 음반이다. 한편 메인 테마와 최종 제품의 수록이 불완전한 때문인지 형식상 반젤리스 개인 작품의 모양을 취하고 있다.</p>
<ul>
<li><b>Mythodea</b></li>
</ul>
<p>2001년 출시되었다. NASA의 <a href="/wiki/2001_%EB%A7%88%EC%8A%A4_%EC%98%A4%EB%94%94%EC%84%B8%EC%9D%B4" title="2001 마스 오디세이">2001 마스 오디세이</a>의 테마 음악이다.</p>
<h3><span class="mw-headline" id=".EC.9E.91.ED.92.88.EA.B3.BC_.EC.A7.81.EC.97.85.EC.97.90_.EA.B4.80.ED.95.9C_.EC.A3.BC.EC.9D.98_.EC.82.AC.ED.95.AD">작품과 직업에 관한 주의 사항</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%B0%98%EC%A0%A4%EB%A6%AC%EC%8A%A4&amp;action=edit&amp;section=14" title="부분 편집: 작품과 직업에 관한 주의 사항">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<h4><span class="mw-headline" id=".ED.8F.AC.EB.B0.8D.EC.8A.A4.28Forminx.29"><a class="new" href="/w/index.php?title=%ED%8F%AC%EB%B0%8D%EC%8A%A4&amp;action=edit&amp;redlink=1" title="포밍스 (없는 문서)">포밍스</a>(Forminx)</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%B0%98%EC%A0%A4%EB%A6%AC%EC%8A%A4&amp;action=edit&amp;section=15" title="부분 편집: 포밍스(Forminx)">편집</a><span class="mw-editsection-bracket">]</span></span></h4>
<p>그리스에서 1964년에서 1966까지 반젤리스가 신시사이저 연주자로 재직하고 있던 팝 밴드이다. 그리스의 비틀즈라는 별명을 달았으나, 그리스 국내에서는 9장의 싱글을 발표하여 대번에 대중 음악계를 들썩이게 했다. 해외 진출을 생각하고 있어서인지, 가사는 영어로 된 것이 대부분이다. 총 22곡을 발표했지만, 그 대부분을 반젤리스가 작곡하였다. 또한 리드 보컬을 맡은 TASSOS PAPASTAMATIS가 훗날, 프랑스에서 솔로 가수로 데뷔하기 위해 노래(Days of Love)를 녹음하였다.</p>
<h4><span class="mw-headline" id=".EC.95.84.ED.94.84.EB.A1.9C.EB.94.94.ED.85.8C.EC.8A.A4_.EC.B0.A8.EC.9D.BC.EB.93.9C.28Aphrodite.27s_Child.29">아프로디테스 차일드(Aphrodite's Child)</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%B0%98%EC%A0%A4%EB%A6%AC%EC%8A%A4&amp;action=edit&amp;section=16" title="부분 편집: 아프로디테스 차일드(Aphrodite's Child)">편집</a><span class="mw-editsection-bracket">]</span></span></h4>
<p><a class="new" href="/w/index.php?title=%ED%8F%AC%EB%B0%8D%EC%8A%A4&amp;action=edit&amp;redlink=1" title="포밍스 (없는 문서)">포밍스</a> 해산 후 반젤리스가 결성한 Aphrodite's Child는 'Rain and Tears', 'It's Five O'Clock', '666'의 3개 앨범과 11장의 싱글을 발표했다.</p>
<h4><span class="mw-headline" id=".EC.98.81.ED.99.94_.EC.9D.8C.EC.95.85">영화 음악</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%B0%98%EC%A0%A4%EB%A6%AC%EC%8A%A4&amp;action=edit&amp;section=17" title="부분 편집: 영화 음악">편집</a><span class="mw-editsection-bracket">]</span></span></h4>
<p>반젤리스는 그리스 시대부터 많은 영화 음악을 담당하고 있지만, 1970년대부터 시작된 프레데릭 로시프(Frédéric Rossif, 프랑스 다큐멘터리 영화 감독)와의 공동 작업은 로시프의 죽음(1989년)까지 계속되었다. 앨범으로 정식 발매된 것은'동물의 묵시록', '야생의 제전', '야생'등이 있지만, 그 외에도 그와 많은 음악을 작곡하였다.</p>
<p>특히 '불의 전차'와 '블레이드 러너'가 유명하다. '불의 전차'는 빌보드 싱글, 앨범 차트에서 전미 No.1을 획득하고 있다. 이것은 반젤리스에게 유일한 No.1 싱글 겸 앨범이다. 또한 Instrumental 음악 상 유일한 부동의 No.1 앨범이기도 하다.</p>
<p>'블레이드 러너'는 영화의 크레딧 화면에서 OST 음반의 발매가 시사되었지만, 반젤리스 자신이 "당시 다른 음악으로 제작되었다"는 이유로 발매를 급히 거부했기 때문에 발매되지 않았다. 이후 1992년'블레이드 러너 감독판'의 개봉과 함께, 실로 10년 이상 지난 1994년, 드디어 공식 CD가 출시되었다.</p>
<p>"나는 자신이 영화 음악가라고는 생각하지 않는다.", "예스에 가입하면 나도 좋지만, 그런 사소한 활동에 흥미가 없다." 등의 발언에서 보이는 대로, 반젤리스는 상업적인 성공에 집착을 보이고 있지 않다. 사운드 트랙도 그대로 CD화하는 것이 아니라, 독립적인 음악 작품으로 출반되는 경우가 많다.</p>
<h4><span class="mw-headline" id=".EC.8A.A4.ED.8F.AC.EC.B8.A0_.EC.9D.B4.EB.B2.A4.ED.8A.B8">스포츠 이벤트</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%B0%98%EC%A0%A4%EB%A6%AC%EC%8A%A4&amp;action=edit&amp;section=18" title="부분 편집: 스포츠 이벤트">편집</a><span class="mw-editsection-bracket">]</span></span></h4>
<p><a href="/wiki/%EC%98%AC%EB%A6%BC%ED%94%BD" title="올림픽">올림픽</a>을 시작으로 다양한 스포츠 이벤트가 지속되었으나, 그 중에서도 2002년 3월 싱글로 발표된 2002년 FIFA 월드컵 공식 앤섬은 유명하다. 2002년 6월 텔레비전 방송에서 자주 '온 에어'되어 일본 오리콘 차트에서 싱글 톱 10 진입에 성공했다. 주간 랭킹은 2002년 7월 제1주와 제2주 사이 6위. 신시사이저에 의한 독특한 속도감과 미래 지향적인 세계관으로 장식된 사운드는 당시 한국과 일본 양국의 <a class="mw-disambig" href="/wiki/%EC%9B%94%EB%93%9C%EC%BB%B5" title="월드컵">월드컵</a> 인기를 고조시키는 데 상당한 기여를 했다. 또한 아테네 올림픽 유치 활동을 위해 제작된 애니메이션 사운드 트랙과 <a class="mw-redirect" href="/wiki/%EC%8B%9C%EB%93%9C%EB%8B%88_%EC%98%AC%EB%A6%BC%ED%94%BD" title="시드니 올림픽">시드니 올림픽</a>의 폐막식, 97년 아테네에서 개최되는 세계 육상 대회 개회식 음악 등을 담당하고 있기도 하다.</p>
<h2><span class="mw-headline" id=".EB.94.94.EC.8A.A4.EC.BD.94.EA.B7.B8.EB.9E.98.ED.94.BC">디스코그래피</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%B0%98%EC%A0%A4%EB%A6%AC%EC%8A%A4&amp;action=edit&amp;section=19" title="부분 편집: 디스코그래피">편집</a><span class="mw-editsection-bracket">]</span></span></h2>

<h2><span class="mw-headline" id=".ED.99.94.EA.B0.80">화가</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%B0%98%EC%A0%A4%EB%A6%AC%EC%8A%A4&amp;action=edit&amp;section=20" title="부분 편집: 화가">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>반젤리스는 또한 추상화 화가로 활동하고 있다. 스페인 <a href="/wiki/%EB%B0%9C%EB%A0%8C%EC%8B%9C%EC%95%84" title="발렌시아">발렌시아</a> 협찬, <a href="/wiki/%EC%9D%B4%EB%B2%A0%EB%A6%AC%EC%95%84_%ED%95%AD%EA%B3%B5" title="이베리아 항공">이베리아 항공</a> 후원, <a href="/wiki/%EB%B0%9C%EB%A0%8C%EC%8B%9C%EC%95%84" title="발렌시아">발렌시아</a>에서 성공했지만, 다른 나라에서도 개인전을 개최하였다.</p>
<ul>
<li>'VANGELIS 2003년 세로 23cm * 가로 24cm <a class="internal mw-magiclink-isbn" href="/wiki/%ED%8A%B9%EC%88%98:%EC%B1%85%EC%B0%BE%EA%B8%B0/8448230906">ISBN 84-482-3090-6</a> (서쪽 문 Fernando Castro Frórez, Dora Eliopoulou - Rogan, 영문 번역 Lambe &amp; Nieto, John Davis, 반젤리스 작가의 그림 74점 수록)</li>
</ul>
<h2><span class="mw-headline" id=".EC.82.AC.EC.9A.A9_.EC.95.85.EA.B8.B0">사용 악기</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%B0%98%EC%A0%A4%EB%A6%AC%EC%8A%A4&amp;action=edit&amp;section=21" title="부분 편집: 사용 악기">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>반젤리스는 원칙적으로 <a href="/wiki/%ED%82%A4%EB%B3%B4%EB%93%9C_(%EC%95%85%EA%B8%B0)" title="키보드 (악기)">키보드</a> 연주자, <a class="mw-redirect" href="/wiki/%EC%8B%A0%EC%8B%9C%EC%82%AC%EC%9D%B4%EC%A0%80_%EC%97%B0%EC%A3%BC%EC%9E%90" title="신시사이저 연주자">신시사이저 연주자</a>이지만, 기타와 드럼도 연주하는 <a class="new" href="/w/index.php?title=%EB%A9%80%ED%8B%B0_%ED%94%8C%EB%A0%88%EC%9D%B4%EC%96%B4_(%EC%9D%8C%EC%95%85)&amp;action=edit&amp;redlink=1" title="멀티 플레이어 (음악) (없는 문서)">멀티 플레이어</a>이며, 또한 <a href="/wiki/%EC%8B%A0%EC%8B%9C%EC%82%AC%EC%9D%B4%EC%A0%80" title="신시사이저">신시사이저</a>는 제조 연대, 제조 업체, 사양 변경과 기종을 주기적이므로 업그레이드하므로 사용 악기는 다방면에 걸쳐있다. 다음은 각 앨범의 라이너 등 작성된 한 예이다.</p>
