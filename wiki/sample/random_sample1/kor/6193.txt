<p><b>반즐리</b> (Barnsley)는 <a href="/wiki/%EC%98%81%EA%B5%AD" title="영국">영국</a> <a href="/wiki/%EC%9E%89%EA%B8%80%EB%9E%9C%EB%93%9C" title="잉글랜드">잉글랜드</a> <a href="/wiki/%EC%82%AC%EC%9A%B0%EC%8A%A4%EC%9A%94%ED%81%AC%EC%85%94_%EC%A3%BC" title="사우스요크셔 주">사우스요크셔 주</a>에 있는 도시이다. <a class="new" href="/w/index.php?title=%EB%8D%98_%EA%B0%95&amp;action=edit&amp;redlink=1" title="던 강 (없는 문서)">던 강</a>(River Dearne)을 접하고 있으며, <a href="/wiki/%EC%85%B0%ED%95%84%EB%93%9C" title="셰필드">셰필드</a> 북쪽으로 19 km, <a href="/wiki/%EB%A6%AC%EC%A6%88" title="리즈">리즈</a> 남쪽으로 27 km, <a href="/wiki/%EB%8F%99%EC%BB%A4%EC%8A%A4%ED%84%B0" title="동커스터">동커스터</a> 서쪽으로 23 km 떨어진 곳에 위치해 있다. 반즐리는 <a href="/wiki/%EB%B0%98%EC%A6%90%EB%A6%AC_%EB%8F%84%EC%8B%9C_%EC%9E%90%EC%B9%98%EA%B5%AC" title="반즐리 도시 자치구">반즐리 도시 자치구</a>를 함께 형성하는 소규모의 여러 마을로 둘러싸여 있으며, 그 중 반즐리는 가장 크며 행정 중심이다. 2001년 영국의 인구조사에 따르면 반즐리 도시 자치구의 인구는 218,063 명이었으며, 반즐리 도심은 71,599명이었다.<sup class="reference" id="cite_ref-1"><a href="#cite_note-1">[1]</a></sup></p>
<p>역사적으로 <a class="new" href="/w/index.php?title=%EC%9B%A8%EC%8A%A4%ED%8A%B8%EB%9D%BC%EC%9D%B4%EB%94%A9%EC%98%A4%EB%B8%8C%EC%9A%94%ED%81%AC%EC%85%94_%EC%A3%BC&amp;action=edit&amp;redlink=1" title="웨스트라이딩오브요크셔 주 (없는 문서)">웨스트라이딩오브요크셔 주</a>의 일부였던 반즐리는 <a class="new" href="/w/index.php?title=%ED%83%84%EA%B4%91&amp;action=edit&amp;redlink=1" title="탄광 (없는 문서)">탄광</a>과 <a href="/wiki/%EC%9C%A0%EB%A6%AC" title="유리">유리</a> 생산<sup class="reference" id="cite_ref-BarnsleyLife_2-0"><a href="#cite_note-BarnsleyLife-2">[2]</a></sup> 이 집중되었던 前 산업도시로 유명하지만, 현재 소수의 유리 가공업자와 석탄 생산공장만이 남아있는 상태이다. 비록 20세기에 이 산업 근반이 사양되었을지라도, 반즐리 현지 문화는 <a class="new" href="/w/index.php?title=%EC%82%B0%EC%97%85%EC%9C%A0%EC%82%B0&amp;action=edit&amp;redlink=1" title="산업유산 (없는 문서)">산업유산</a>에 뿌리를 두고 있다. 반즐리는 <a class="new" href="/w/index.php?title=%EB%B8%8C%EB%9D%BC%EC%8A%A4_%EB%B0%B4%EB%93%9C&amp;action=edit&amp;redlink=1" title="브라스 밴드 (없는 문서)">브라스 밴드</a>의 전통이 있으며, 채광 사회의 사교 클럽으로서 초기에 만들어졌다.</p>
<p>반즐리는 반즐리 특유의 방언을 잘 나타냈으며 영국 학교에서 이용중인 영화 《Kes》로 유명하다. <a class="new" href="/w/index.php?title=M1_%EA%B3%A0%EC%86%8D%EB%8F%84%EB%A1%9C&amp;action=edit&amp;redlink=1" title="M1 고속도로 (없는 문서)">M1 고속도로</a> 36번과 38번 교차로 사이에 위치하며, <a class="new" href="/w/index.php?title=%ED%95%A0%EB%9E%8C_%EC%84%A0&amp;action=edit&amp;redlink=1" title="할람 선 (없는 문서)">할람 선</a>과 <a class="new" href="/w/index.php?title=%ED%8E%98%EB%8B%88%EC%8A%A4%ED%86%A4_%EC%84%A0&amp;action=edit&amp;redlink=1" title="페니스톤 선 (없는 문서)">페니스톤 선</a>이 지나가는 <a class="new" href="/w/index.php?title=%EB%B0%98%EC%A6%90%EB%A6%AC_%EC%97%AD&amp;action=edit&amp;redlink=1" title="반즐리 역 (없는 문서)">반즐리 역</a>이 있다. <a href="/wiki/%EB%B0%98%EC%A6%90%EB%A6%AC_FC" title="반즐리 FC">반즐리 FC</a>는 현지 <a href="/wiki/%EC%B6%95%EA%B5%AC" title="축구">축구</a> 클럽이다.</p>

<h2><span class="mw-headline" id=".EC.97.AD.EC.82.AC">역사</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%B0%98%EC%A6%90%EB%A6%AC&amp;action=edit&amp;section=1" title="부분 편집: 역사">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p><a href="/wiki/1086%EB%85%84" title="1086년">1086년</a>에 발행한 <a href="/wiki/%EB%91%A0%EC%8A%A4%EB%8D%B0%EC%9D%B4_%EB%B6%81" title="둠스데이 북">둠스데이 북</a>에서 최초의 역사적 언급이 이루어졌으며, 책속에서 200여명의 인구가 사는 "반즐라이"로 기록되어있다. 반즐리 명칭의 정확한 기원은 여전히 논쟁중이지만, 반즐리 위원회는 헛간이나 창고를 뜻하는 색슨족 단어 "Berne"과 들판을 뜻하는 "Lay"에서 유래했다고 주장한다.</p>
<p>당시 마을은 <a class="new" href="/w/index.php?title=%EC%8B%A4%ED%81%AC%EC%8A%A4%ED%86%A4&amp;action=edit&amp;redlink=1" title="실크스톤 (없는 문서)">실크스톤</a> 사목구 근처에 있으면서 1550년대까지 약간의 발전을 이루었고 그 후 <a class="new" href="/w/index.php?title=%ED%8F%B0%ED%85%8C%ED%94%84%EB%9D%BD%ED%8A%B8_%EC%88%98%EB%8F%84%EC%9B%90&amp;action=edit&amp;redlink=1" title="폰테프락트 수도원 (없는 문서)">폰테프락트 수도원</a>의 관리하에 있게된다. 수도승들은 <a href="/wiki/%EC%85%B0%ED%95%84%EB%93%9C" title="셰필드">셰필드</a>에서 <a href="/wiki/%EC%9B%A8%EC%9D%B4%ED%81%AC%ED%95%84%EB%93%9C" title="웨이크필드">웨이크필드</a>, <a href="/wiki/%EB%A1%9C%EB%8D%94%EB%9F%BC" title="로더럼">로더럼</a>에서 <a class="mw-redirect" href="/wiki/%ED%97%88%EB%8D%94%EC%8A%A4%ED%95%84%EB%93%9C" title="허더스필드">허더스필드</a>, <a class="mw-redirect" href="/wiki/%EC%B2%B4%EC%85%94" title="체셔">체셔</a>에서 <a class="mw-redirect" href="/wiki/%EB%8F%88%EC%BA%90%EC%8A%A4%ED%84%B0" title="돈캐스터">돈캐스터</a>, 총 3 노선이 만나는 곳에 새로운 마을을 건설하기로했다. "올드 반즐리"로서 알려진 마을이 새로운 장소에서 커나갔다.<sup class="reference" id="cite_ref-hey_3-0"><a href="#cite_note-hey-3">[3]</a></sup></p>
<p>수도승들은 <a class="mw-redirect" href="/wiki/%EB%8F%99%EC%A0%95_%EB%A7%88%EB%A6%AC%EC%95%84" title="동정 마리아">동정 마리아</a>에게 바치는 <a class="new" href="/w/index.php?title=%EC%A7%80%EC%84%B1%EB%8B%B9&amp;action=edit&amp;redlink=1" title="지성당 (없는 문서)">지성당</a>을 세웠고, <a class="mw-disambig" href="/wiki/%EC%8B%9C%EC%9E%A5" title="시장">시장</a>이 설립되었다. 마을은 <a class="new" href="/w/index.php?title=%EC%8A%A4%ED%85%8C%EC%9D%B8%ED%81%AC%EB%A1%9C%EC%8A%A4&amp;action=edit&amp;redlink=1" title="스테인크로스 (없는 문서)">스테인크로스</a> <a class="new" href="/w/index.php?title=%EC%99%80%ED%8E%9C%ED%85%8C%EC%9D%B4%ED%81%AC&amp;action=edit&amp;redlink=1" title="와펜테이크 (없는 문서)">와펜테이크</a>의 중심이 되었지만 16세기 중반까지 인구는 600여명 뿐이었다.<sup class="reference" id="cite_ref-hey_3-1"><a href="#cite_note-hey-3">[3]</a></sup></p>
<p>17세기 부터 반즐리는 <a href="/wiki/%EB%A6%AC%EC%A6%88" title="리즈">리즈</a>,<a href="/wiki/%EC%9B%A8%EC%9D%B4%ED%81%AC%ED%95%84%EB%93%9C" title="웨이크필드">웨이크필드</a>,<a href="/wiki/%EC%85%B0%ED%95%84%EB%93%9C" title="셰필드">셰필드</a>,<a href="/wiki/%EB%9F%B0%EB%8D%98" title="런던">런던</a>사이의 노선 상에서 정류 지점으로 발전하였다. 18세기와 19세기 동안 마세포 직단의 중심이었던 반즐리는 제조업 마을로 성장했다. 또한 오랜 유리제조 전통<sup class="reference" id="cite_ref-BarnsleyLife_2-1"><a href="#cite_note-BarnsleyLife-2">[2]</a></sup>으로 유명하지만, 석탄매장지로 가장 유명하다.</p>
