<p><b>반투제어군</b>은 <a class="mw-redirect" href="/wiki/%EB%B0%98%ED%88%AC%EC%96%B4" title="반투어">반투어</a> 또는 <a href="/wiki/%EB%B0%98%ED%88%AC%EC%96%B4%EA%B5%B0" title="반투어군">반투어군</a>과 유사한 일련의 언어 집합으로 여기에 속한 언어의 수는 자그마치 681개에 이른다. 이 숫자는 상위 분류인 <a href="/wiki/%EB%B2%A0%EB%88%84%EC%97%90%EC%BD%A9%EA%B3%A0%EC%96%B4" title="베누에콩고어">베누에콩고어</a>의 언어 수의 3분의 2에 해당하며, <a href="/wiki/%EB%8B%88%EC%A0%9C%EB%A5%B4%EC%BD%A9%EA%B3%A0%EC%96%B4%EC%A1%B1" title="니제르콩고어족">니제르콩고어족</a>에 속한 언어 수의 반에 다소 못 미칠 정도이다. 속한 언어의 수만 놓고 본다면 사실상 하나의 <a href="/wiki/%EC%96%B4%EC%A1%B1" title="어족">어족</a>으로 취급될 수 있을 정도의 규모이다. 이 <a href="/wiki/%EC%96%B4%EA%B5%B0_(%EC%96%B8%EC%96%B4%ED%95%99)" title="어군 (언어학)">어군</a>에 속한 언어들은 <a href="/wiki/%EC%84%9C%EC%95%84%ED%94%84%EB%A6%AC%EC%B9%B4" title="서아프리카">서아프리카</a>와 <a href="/wiki/%EB%82%A8%EC%95%84%ED%94%84%EB%A6%AC%EC%B9%B4" title="남아프리카">남아프리카</a> 일대의 넓게 쓰이고 있다.</p>
