<p><b>발레리 쿠지미치 네폼냐시</b>(<span style="font-size: smaller;"><a href="/wiki/%EB%9F%AC%EC%8B%9C%EC%95%84%EC%96%B4" title="러시아어">러시아어</a>:</span> <span lang="ru" xml:lang="ru">Валерий Кузьмич Непомнящий</span>, <span style="font-size: smaller;"><a href="/wiki/%EC%98%81%EC%96%B4" title="영어">영어</a>:</span> <span lang="en" xml:lang="en">Valeri Kuzmyich Nepomniachi</span>, <a href="/wiki/1943%EB%85%84" title="1943년">1943년</a> <a href="/wiki/8%EC%9B%94_7%EC%9D%BC" title="8월 7일">8월 7일</a> ~ )는 <a class="mw-redirect" href="/wiki/%EC%86%8C%EB%B9%84%EC%97%90%ED%8A%B8_%EC%97%B0%EB%B0%A9" title="소비에트 연방">소련</a> (현재 <a href="/wiki/%EB%9F%AC%EC%8B%9C%EC%95%84" title="러시아">러시아</a>) 출신의 <a href="/wiki/%EC%B6%95%EA%B5%AC" title="축구">축구</a> 감독이다. <a href="/wiki/1995%EB%85%84" title="1995년">1995년</a>부터 <a href="/wiki/1998%EB%85%84" title="1998년">1998년</a>까지 <a href="/wiki/%EC%A0%9C%EC%A3%BC_%EC%9C%A0%EB%82%98%EC%9D%B4%ED%8B%B0%EB%93%9C_FC" title="제주 유나이티드 FC">제주 유나이티드 FC</a>의 <a href="/wiki/%EC%A0%9C%EC%A3%BC_%EC%9C%A0%EB%82%98%EC%9D%B4%ED%8B%B0%EB%93%9C_FC" title="제주 유나이티드 FC">유공 코끼리 / 부천 유공 / 부천 SK</a> 시절 감독을 맡아 강한 인상을 남겼다. 애칭은 니포이며, 그의 축구 스타일을 니포 축구라고 부른다.</p>

<h2><span class="mw-headline" id=".EC.B6.95.EA.B5.AC.EC.9D.B8_.EA.B2.BD.EB.A0.A5">축구인 경력</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%B0%9C%EB%A0%88%EB%A6%AC_%EB%84%A4%ED%8F%BC%EB%83%90%EC%8B%9C&amp;action=edit&amp;section=1" title="부분 편집: 축구인 경력">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<h3><span class="mw-headline" id=".EC.A7.80.EB.8F.84.EC.9E.90_.EA.B2.BD.EB.A0.A5">지도자 경력</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%B0%9C%EB%A0%88%EB%A6%AC_%EB%84%A4%ED%8F%BC%EB%83%90%EC%8B%9C&amp;action=edit&amp;section=2" title="부분 편집: 지도자 경력">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p><a href="/wiki/1990%EB%85%84_FIFA_%EC%9B%94%EB%93%9C%EC%BB%B5" title="1990년 FIFA 월드컵">이탈리아 월드컵</a>에서 <a href="/wiki/%EC%B9%B4%EB%A9%94%EB%A3%AC_%EC%B6%95%EA%B5%AC_%EA%B5%AD%EA%B0%80%EB%8C%80%ED%91%9C%ED%8C%80" title="카메룬 축구 국가대표팀">카메룬</a>의 감독으로 선임되어 <a href="/wiki/%EC%B9%B4%EB%A9%94%EB%A3%AC_%EC%B6%95%EA%B5%AC_%EA%B5%AD%EA%B0%80%EB%8C%80%ED%91%9C%ED%8C%80" title="카메룬 축구 국가대표팀">카메룬</a>의 8강 진출을 이끌었다. 당시 <a href="/wiki/%EC%B9%B4%EB%A9%94%EB%A3%AC_%EC%B6%95%EA%B5%AC_%EA%B5%AD%EA%B0%80%EB%8C%80%ED%91%9C%ED%8C%80" title="카메룬 축구 국가대표팀">카메룬</a>은 개막전에서 디펜딩 챔피언 <a href="/wiki/%EC%95%84%EB%A5%B4%ED%97%A8%ED%8B%B0%EB%82%98_%EC%B6%95%EA%B5%AC_%EA%B5%AD%EA%B0%80%EB%8C%80%ED%91%9C%ED%8C%80" title="아르헨티나 축구 국가대표팀">아르헨티나</a>를 1-0으로 꺾는 등 뛰어난 실력으로 조별 리그를 통과했고, 16강전에서는 <a href="/wiki/%EB%82%A8%EC%95%84%EB%A9%94%EB%A6%AC%EC%B9%B4" title="남아메리카">남아메리카</a>의 강호 <a href="/wiki/%EC%BD%9C%EB%A1%AC%EB%B9%84%EC%95%84_%EC%B6%95%EA%B5%AC_%EA%B5%AD%EA%B0%80%EB%8C%80%ED%91%9C%ED%8C%80" title="콜롬비아 축구 국가대표팀">콜롬비아</a>마저 격침시켰다. 비록 8강에서 <a href="/wiki/%EC%9E%89%EA%B8%80%EB%9E%9C%EB%93%9C_%EC%B6%95%EA%B5%AC_%EA%B5%AD%EA%B0%80%EB%8C%80%ED%91%9C%ED%8C%80" title="잉글랜드 축구 국가대표팀">잉글랜드</a>에게 2-3으로 패했으나, 네폼냐시 감독은 변방에 머물던 <a href="/wiki/%EC%95%84%ED%94%84%EB%A6%AC%EC%B9%B4" title="아프리카">아프리카</a> 축구를 세계 축구의 중심으로 이끄는 계기를 마련했다.</p>
<p><a href="/wiki/1990%EB%85%84_FIFA_%EC%9B%94%EB%93%9C%EC%BB%B5" title="1990년 FIFA 월드컵">이탈리아 월드컵</a> 이후 <a href="/wiki/%ED%84%B0%ED%82%A4" title="터키">터키</a>의 프로축구 팀을 맡았던 네폼냐시는 <a href="/wiki/1994%EB%85%84" title="1994년">1994년</a>말 <a href="/wiki/%EC%A0%9C%EC%A3%BC_%EC%9C%A0%EB%82%98%EC%9D%B4%ED%8B%B0%EB%93%9C_FC" title="제주 유나이티드 FC">유공 코끼리</a>의 감독으로 선임되어 <a href="/wiki/K%EB%A6%AC%EA%B7%B8_1995" title="K리그 1995">1995</a>시즌 <a href="/wiki/%EC%A0%9C%EC%A3%BC_%EC%9C%A0%EB%82%98%EC%9D%B4%ED%8B%B0%EB%93%9C_FC" title="제주 유나이티드 FC">유공 코끼리</a>의 감독으로 공식 취임했다. 이후 <a href="/wiki/1998%EB%85%84" title="1998년">1998년</a>까지 4시즌 동안 감독을 맡으면서 '니포 축구'라는 신조어까지 낳으며 1990년대 <a href="/wiki/%EB%8C%80%ED%95%9C%EB%AF%BC%EA%B5%AD" title="대한민국">대한민국</a> 축구에 큰 영향을 끼쳤다.</p>
<p>당시 거칠고 투박한 플레이만 난무하였던 <a href="/wiki/%EB%8C%80%ED%95%9C%EB%AF%BC%EA%B5%AD" title="대한민국">대한민국</a> 축구계에서, 미드필드에서 세밀하고 정교한 패스를 강조하는 우아하고 화려한 축구 스타일을 <a href="/wiki/%EC%A0%9C%EC%A3%BC_%EC%9C%A0%EB%82%98%EC%9D%B4%ED%8B%B0%EB%93%9C_FC" title="제주 유나이티드 FC">부천 SK</a>에서 꽃피웠다. 당시 전술의 핵심 미드필더 <a href="/wiki/%EC%9C%A4%EC%A0%95%ED%99%98" title="윤정환">윤정환</a>을 중심으로 탄탄한 패스웍을 바탕으로 미드필드를 지배하는 <a href="/wiki/%EB%B6%80%EC%B2%9C%EC%8B%9C" title="부천시">부천</a>의 축구 스타일은 많은 사람들을 매혹시켰다. 또한 전술적인 측면뿐만 아니라 선수단 관리 및 팀 운영에서도 기존의 권위적이고 일방적인 관계 대신 자율성을 강조하면서도 체계적으로 팀을 관리해 <a href="/wiki/K%EB%A6%AC%EA%B7%B8_%ED%81%B4%EB%9E%98%EC%8B%9D" title="K리그 클래식">K리그</a>에 선진적인 문화를 들여왔다.</p>
<p><a href="/wiki/%EC%A0%9C%EC%A3%BC_%EC%9C%A0%EB%82%98%EC%9D%B4%ED%8B%B0%EB%93%9C_FC" title="제주 유나이티드 FC">유공 코끼리 / 부천 유공 / 부천 SK</a> 감독 시절의 가장 큰 업적은 <a href="/wiki/1996%EB%85%84" title="1996년">1996년</a> <a href="/wiki/%EC%A0%9C%EC%A3%BC_%EC%9C%A0%EB%82%98%EC%9D%B4%ED%8B%B0%EB%93%9C_FC" title="제주 유나이티드 FC">부천 유공</a>을 <a href="/wiki/%EB%A6%AC%EA%B7%B8%EC%BB%B5_(%EB%8C%80%ED%95%9C%EB%AF%BC%EA%B5%AD)" title="리그컵 (대한민국)">리그컵</a> 우승으로 이끌었던 것이다.<sup class="reference" id="cite_ref-1"><a href="#cite_note-1">[1]</a></sup></p>
<p>또한 네폼냐시 감독을 보좌하던 <a href="/wiki/%EC%A1%B0%EC%9C%A4%ED%99%98" title="조윤환">조윤환</a>, <a href="/wiki/%EC%B5%9C%EC%9C%A4%EA%B2%B8" title="최윤겸">최윤겸</a>, <a class="mw-disambig" href="/wiki/%ED%95%98%EC%9E%AC%ED%9B%88" title="하재훈">하재훈</a> 등 코칭 스태프들에게 큰 영향을 미쳐, 이들이 <a href="/wiki/K%EB%A6%AC%EA%B7%B8_%ED%81%B4%EB%9E%98%EC%8B%9D" title="K리그 클래식">K리그</a>에서 감독으로 활약할 수 있도록 돕기도 했다. 네폼냐시 감독에게 전술의 핵이었고, 그를 통해 테크니션으로 발돋움했던 <a href="/wiki/%EB%AF%B8%EB%93%9C%ED%95%84%EB%8D%94" title="미드필더">미드필더</a> <a href="/wiki/%EC%9C%A4%EC%A0%95%ED%99%98" title="윤정환">윤정환</a>은 현재도 네폼냐시 감독이 자신의 롤 모델이라고 언급했다.<sup class="reference" id="cite_ref-2"><a href="#cite_note-2">[2]</a></sup></p>
<p>이후 <a href="/wiki/%EC%9D%BC%EB%B3%B8" title="일본">일본</a> <a href="/wiki/J%EB%A6%AC%EA%B7%B8" title="J리그">J리그</a>의 <a href="/wiki/%EC%82%B0%ED%94%84%EB%A0%88%EC%B2%B4_%ED%9E%88%EB%A1%9C%EC%8B%9C%EB%A7%88" title="산프레체 히로시마">산프레체 히로시마</a>와 <a href="/wiki/%EC%A4%91%EA%B5%AD_%EC%8A%88%ED%8D%BC%EB%A6%AC%EA%B7%B8" title="중국 슈퍼리그">중국 슈퍼리그</a>의 <a class="mw-redirect" href="/wiki/%EC%82%B0%EB%91%A5_%EB%A3%A8%EB%84%9D" title="산둥 루넝">산둥 루넝</a>, <a class="mw-redirect" href="/wiki/%EC%83%81%ED%95%98%EC%9D%B4_%EC%84%A0%ED%99%94" title="상하이 선화">상하이 선화</a> 등에서 감독을 맡기도 하는 등, 극동 3국에서 모두 프로 축구 팀 감독을 맡는 이색 경력을 가지고 있기도 하다.</p>
<p><a href="/wiki/2011%EB%85%84" title="2011년">2011년</a> 톰스크의 감독직에서 물러난 뒤 <a href="/wiki/PFC_CSKA_%EB%AA%A8%EC%8A%A4%ED%81%AC%EB%B0%94" title="PFC CSKA 모스크바">PFC CSKA 모스크바</a>의 기술고문으로 일하고 있다.<sup class="reference" id="cite_ref-3"><a href="#cite_note-3">[3]</a></sup></p>
