<h1 class="firstHeading" id="firstHeading" lang="ko">발레리안 쿠이비셰프</h1>

<p><b>발레리안 블라디미로비치 쿠이비셰프</b> (Валериан Владимирович Куйбышев, 1888년 6월 6일 ~ 1935년 1월 25일) 는 <a href="/wiki/%EB%9F%AC%EC%8B%9C%EC%95%84" title="러시아">러시아</a>의 <a class="mw-disambig" href="/wiki/%ED%98%81%EB%AA%85%EA%B0%80" title="혁명가">혁명가</a>이다. <a href="/wiki/%EB%9F%AC%EC%8B%9C%EC%95%84_%EB%82%B4%EC%A0%84" title="러시아 내전">러시아 내전</a> 동안 <a href="/wiki/%EB%B6%89%EC%9D%80_%EA%B5%B0%EB%8C%80" title="붉은 군대">붉은 군대</a>의 고위 <a href="/wiki/%EC%A0%95%EC%B9%98%EC%A7%80%EB%8F%84%EC%9B%90" title="정치지도원">정치지도원</a>이었으며, <a class="mw-redirect" href="/wiki/%EC%86%8C%EB%B9%84%EC%97%90%ED%8A%B8_%EC%97%B0%EB%B0%A9" title="소비에트 연방">소비에트 연방</a>의 저명한 정치인이었다.</p>
<h2><span class="mw-headline" id=".EC.83.9D.EC.95.A0.EC.99.80_.EA.B2.BD.EB.A0.A5">생애와 경력</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%B0%9C%EB%A0%88%EB%A6%AC%EC%95%88_%EC%BF%A0%EC%9D%B4%EB%B9%84%EC%85%B0%ED%94%84&amp;action=edit&amp;section=1" title="부분 편집: 생애와 경력">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>쿠이비셰프는 1888년 <a href="/wiki/%EC%98%B4%EC%8A%A4%ED%81%AC" title="옴스크">옴스크</a>의 군인 가정에서 태어나, 옴스크 육군 유년 학교에서 공부하였다. 1904년 <a href="/wiki/%EB%9F%AC%EC%8B%9C%EC%95%84_%EC%82%AC%ED%9A%8C%EB%AF%BC%EC%A3%BC%EB%85%B8%EB%8F%99%EB%8B%B9" title="러시아 사회민주노동당">러시아 사회민주노동당</a>에 참여했으며, 1905년 군사 의학 아카데미에 입학했으나, 1906년 퇴학당했다. 1906년 ~ 1914년 쿠이비셰프는 <a href="/wiki/%EB%9F%AC%EC%8B%9C%EC%95%84_%EC%A0%9C%EA%B5%AD" title="러시아 제국">러시아 제국</a> 각지에서 <a href="/wiki/%EB%B3%BC%EC%85%B0%EB%B9%84%ED%82%A4" title="볼셰비키">볼셰비키</a> 활동을 벌이다가, 시베리아의 나림(Нарым)으로 추방되었다.</p>
<p>1917년 <a href="/wiki/%EB%9F%AC%EC%8B%9C%EC%95%84_2%EC%9B%94_%ED%98%81%EB%AA%85" title="러시아 2월 혁명">러시아 2월 혁명</a> 이후, <a href="/wiki/%EC%82%AC%EB%A7%88%EB%9D%BC_(%EB%9F%AC%EC%8B%9C%EC%95%84)" title="사마라 (러시아)">사마라</a>로 이동해 지방 소비에트의 의장이 되었다. <a href="/wiki/%EB%9F%AC%EC%8B%9C%EC%95%84_%EB%82%B4%EC%A0%84" title="러시아 내전">러시아 내전</a> 동안 사마라 행정부의 우두머리와 제1군, 제4군의 정치지도원을 역임하였다.</p>
<p>1928년 ~ 1930년 소비에트 국가경제최고위원회 (Высший Совет Народного Хозяйства) 의장, 1930년 ~ 1932년 <a class="mw-redirect" href="/wiki/%EA%B3%A0%EC%8A%A4%ED%94%8C%EB%9E%80" title="고스플란">고스플란</a> (Госпла́н, 소비에트 국가계획위원회) 의장, 1934년 ~ 1935년 인민통제위원회 의장, <a class="new" href="/w/index.php?title=%EC%86%8C%EB%B9%84%EC%97%90%ED%8A%B8_%EC%97%B0%EB%B0%A9_%EC%9D%B8%EB%AF%BC%EC%9C%84%EC%9B%90%ED%9A%8C%EC%9D%98&amp;action=edit&amp;redlink=1" title="소비에트 연방 인민위원회의 (없는 문서)">소비에트 연방 인민위원회의</a> 부의장을 지냈으며, <a href="/wiki/%EC%84%B8%EB%A5%B4%EA%B2%8C%EC%9D%B4_%ED%82%A4%EB%A1%9C%ED%94%84" title="세르게이 키로프">세르게이 키로프</a> 암살 사건을 조사하던 중, <a href="/wiki/%EB%AA%A8%EC%8A%A4%ED%81%AC%EB%B0%94" title="모스크바">모스크바</a>에서 <a class="mw-disambig" href="/wiki/%EC%8B%AC%EC%9E%A5%EB%A7%88%EB%B9%84" title="심장마비">심장마비</a>로 사망하였다. 유해는 크렘린 벽에 매장되었다.</p>
<p>1935년부터 1991년까지 <a href="/wiki/%EC%82%AC%EB%A7%88%EB%9D%BC_(%EB%9F%AC%EC%8B%9C%EC%95%84)" title="사마라 (러시아)">사마라</a>(<a href="/wiki/%EB%9F%AC%EC%8B%9C%EC%95%84" title="러시아">러시아</a> <a href="/wiki/%EC%82%AC%EB%A7%88%EB%9D%BC_%EC%A3%BC" title="사마라 주">사마라 주</a>의 주도), <a href="/wiki/%EB%B3%BC%EA%B0%80%EB%A5%B4" title="볼가르">볼가르</a>(<a href="/wiki/%EB%9F%AC%EC%8B%9C%EC%95%84" title="러시아">러시아</a> <a class="mw-redirect" href="/wiki/%ED%83%80%ED%83%80%EB%A5%B4%EC%8A%A4%ED%83%84" title="타타르스탄">타타르스탄</a>), <a class="new" href="/w/index.php?title=%ED%95%98%EA%B0%80%EB%A5%B4%EC%B9%9C&amp;action=edit&amp;redlink=1" title="하가르친 (없는 문서)">하가르친</a> (<a href="/wiki/%EC%95%84%EB%A5%B4%EB%A9%94%EB%8B%88%EC%95%84" title="아르메니아">아르메니아</a>)은 그의 이름을 딴 쿠이비셰프라는 이름으로 있었으며, <a href="/wiki/%EB%9F%AC%EC%8B%9C%EC%95%84" title="러시아">러시아</a> <a href="/wiki/%EB%85%B8%EB%B3%B4%EC%8B%9C%EB%B9%84%EB%A5%B4%EC%8A%A4%ED%81%AC_%EC%A3%BC" title="노보시비르스크 주">노보시비르스크 주</a>의 쿠이비셰프와 아르메니아의 쿠이비셰프 이 두 도시는 아직도 쿠이비셰프라는 이름을 사용하고 있다.</p>
