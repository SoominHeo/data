<p><b>발렌티니아누스 1세</b> (Flavius Valentinianus, <a href="/wiki/321%EB%85%84" title="321년">321년</a> - <a href="/wiki/375%EB%85%84" title="375년">375년</a> <a href="/wiki/11%EC%9B%94_17%EC%9D%BC" title="11월 17일">11월 17일</a>)는 <a href="/wiki/364%EB%85%84" title="364년">364년</a>부터 <a href="/wiki/375%EB%85%84" title="375년">375년</a> 죽을 때까지 <a href="/wiki/%EB%A1%9C%EB%A7%88_%EC%A0%9C%EA%B5%AD" title="로마 제국">로마 제국</a>의 황제였다.</p>
<h2><span class="mw-headline" id=".EC.83.9D.EC.95.A0">생애</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%B0%9C%EB%A0%8C%ED%8B%B0%EB%8B%88%EC%95%84%EB%88%84%EC%8A%A4_1%EC%84%B8&amp;action=edit&amp;section=1" title="부분 편집: 생애">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>발렌티니아누스는 <a class="mw-redirect" href="/wiki/%ED%8C%90%EB%85%B8%EB%8B%88%EC%95%84" title="판노니아">판노니아</a>의 밧줄만드는 집안에서 태어나 어려서 군대에 들어가 아버지와 함께 아프리카에서 복무했다. 발렌티니아누스는 일개병사에서 시작해 혼자 힘으로 상당한 계급까지 승진한 입지전적인 인물이었다. 일설에 의하면 율리아누스 휘하에서 부관으로 있을 때 기독교 신앙을 고집해 율리아누스의 눈밖에 났다고 한다. 그는 <a href="/wiki/363%EB%85%84" title="363년">363년</a> 율리아누스의 페르시아 원정에 참여했으며, 율리아누스의 전사 이후 황제가 된 <a href="/wiki/%EC%9A%94%EB%B9%84%EC%95%84%EB%88%84%EC%8A%A4" title="요비아누스">요비아누스</a>는 발렌티니아누스를 진급시킨 직후 세상을 떠났다.</p>
<p>요비아누스가 죽은 지 10일 만에 그는 <a class="mw-redirect" href="/wiki/%EB%8B%88%EC%BC%80%EC%95%84" title="니케아">니케아</a>(지금의 <a href="/wiki/%ED%84%B0%ED%82%A4" title="터키">터키</a> 이즈니크)에서 황제로 선포되었고 <a href="/wiki/%EC%BD%98%EC%8A%A4%ED%83%84%ED%8B%B0%EB%85%B8%ED%8F%B4%EB%A6%AC%EC%8A%A4" title="콘스탄티노폴리스">콘스탄티노폴리스</a>로 돌아와서 <a href="/wiki/3%EC%9B%94_28%EC%9D%BC" title="3월 28일">3월 28일</a>에 그는 동생 <a href="/wiki/%EB%B0%9C%EB%A0%8C%EC%8A%A4" title="발렌스">발렌스</a>를 공동 황제로 임명하고 제국의 동부를 맡게하고 자신은 서부를 다스렸다. 황제가 된 이후 발렌티니아누스는 계속된 야만족의 침입과 맞서 싸워야 했다. <a href="/wiki/365%EB%85%84" title="365년">365년</a> <a href="/wiki/%EA%B0%88%EB%A6%AC%EC%95%84" title="갈리아">갈리아</a>에서 <a class="mw-redirect" href="/wiki/%EC%95%8C%EB%A0%88%EB%A7%88%EB%8B%88%EC%A1%B1" title="알레마니족">알레마니족</a>의 침입으로 로마군이 패하자 그는 <a class="mw-redirect" href="/wiki/%ED%8C%8C%EB%A6%AC_%EC%8B%9C" title="파리 시">파리</a>에 근거지를 마련하여 계속 그 곳에서 머물면서 야만족과 싸웠다. 동생 발렌스가 <a href="/wiki/%ED%94%84%EB%A1%9C%EC%BD%94%ED%94%BC%EC%9A%B0%EC%8A%A4" title="프로코피우스">프로코피우스</a>의 반란으로 도움을 요청했을 때 그는 알레만니족과의 싸움으로 병력을 빼 수 없다고 그 도움을 거절하면서 이렇게 말했다고 한다. <i>"프로코피우스는 우리 형제만의 적이지만, 알레마니족은 로마 세계 전체의 적이다."</i><sup class="reference" id="cite_ref-1"><a href="#cite_note-1">[1]</a></sup></p>
<p>발렌티니아누스의 장군인 요비누스는 알레마니족을 상대로 3차례 승리하고 그 후 여러 해 동안 갈리아 지방을 안전하게 했으나 발렌티니아누스는 <a href="/wiki/367%EB%85%84" title="367년">367년</a> <a href="/wiki/%EB%B8%8C%EB%A6%AC%ED%83%80%EB%8B%88%EC%95%84" title="브리타니아">브리타니아</a>에서 <a class="new" href="/w/index.php?title=%ED%94%BC%ED%8A%B8%EC%A1%B1&amp;action=edit&amp;redlink=1" title="피트족 (없는 문서)">피트족</a>과 <a class="mw-redirect" href="/wiki/%EC%8A%A4%EC%BD%94%ED%8A%B8%EC%A1%B1" title="스코트족">스코트족</a>의 침입을 받았다. 자신은 갈리아에 있어야 했으므로 <a class="new" href="/w/index.php?title=%ED%94%8C%EB%9D%BC%EB%B9%84%EC%9A%B0%EC%8A%A4_%ED%85%8C%EC%98%A4%EB%8F%84%EC%8B%9C%EC%9A%B0%EC%8A%A4&amp;action=edit&amp;redlink=1" title="플라비우스 테오도시우스 (없는 문서)">플라비우스 테오도시우스</a>(황제 <a href="/wiki/%ED%85%8C%EC%98%A4%EB%8F%84%EC%8B%9C%EC%9A%B0%EC%8A%A4_1%EC%84%B8" title="테오도시우스 1세">테오도시우스</a>의 아버지)에게 브리타니아 원정을 맡겼는데 테오도시우스는 성공적으로 원정을 마쳤다. 이후 발렌티니아누스는 아들의 왕위계승권을 강화하기 위해 당시 9세인 아들 <a href="/wiki/%EA%B7%B8%EB%9D%BC%ED%8B%B0%EC%95%84%EB%88%84%EC%8A%A4" title="그라티아누스">그라티아누스</a>를 공동 황제로 임명했다. 그는 <a href="/wiki/%ED%8A%B8%EB%A6%AC%EC%96%B4" title="트리어">트리어</a>에서 7년 동안 머물면서, 제국의 국경을 안정화시키고 <a href="/wiki/373%EB%85%84" title="373년">373년</a> 갈리아에서 돌아왔다.</p>
<p><a href="/wiki/374%EB%85%84" title="374년">374년</a> 판노니아의 <a href="/wiki/%EB%8F%84%EB%82%98%EC%9A%B0_%EA%B0%95" title="도나우 강">도나우 강</a> 건너편에 살던 <a class="new" href="/w/index.php?title=%EC%BD%B0%EB%94%94%EC%A1%B1&amp;action=edit&amp;redlink=1" title="콰디족 (없는 문서)">콰디족</a>이 로마가 요새를 세운데 불만을 품고 제국의 국경을 침입하였다. 이듬해 발렌티니아누스는 콰디족의 사절을 만난 자리에서 갑자기 뇌졸증으로 쓰러졌고 분노한채로 죽고 말았다.</p>
