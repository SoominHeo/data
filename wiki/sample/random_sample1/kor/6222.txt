<p><b>발렌틴 파니아과 코라사오</b>(<span style="font-size: smaller;"><a href="/wiki/%EC%8A%A4%ED%8E%98%EC%9D%B8%EC%96%B4" title="스페인어">스페인어</a>:</span> <span lang="es" xml:lang="es">Valentín Paniagua Corazao</span>, <span style="font-size: smaller;"><a href="/wiki/%EB%AC%B8%ED%99%94%EC%96%B4" title="문화어">문화어</a>:</span> 발렌띤 빠니아과 꼬라싸오, <a href="/wiki/1936%EB%85%84" title="1936년">1936년</a> <a href="/wiki/9%EC%9B%94_23%EC%9D%BC" title="9월 23일">9월 23일</a> ~ <a href="/wiki/2006%EB%85%84" title="2006년">2006년</a> <a href="/wiki/10%EC%9B%94_16%EC%9D%BC" title="10월 16일">10월 16일</a>)는 <a href="/wiki/%ED%8E%98%EB%A3%A8" title="페루">페루</a>의 제91대 대통령이었다.</p>
<h2><span class="mw-headline" id=".EC.9D.BC.EC.83.9D">일생</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%B0%9C%EB%A0%8C%ED%8B%B4_%ED%8C%8C%EB%8B%88%EC%95%84%EA%B3%BC&amp;action=edit&amp;section=1" title="부분 편집: 일생">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>파니아과의 아버지는 <a href="/wiki/%EB%B3%BC%EB%A6%AC%EB%B9%84%EC%95%84" title="볼리비아">볼리비아</a> 태생이지만 일생의 대부분을 <a href="/wiki/%ED%8E%98%EB%A3%A8" title="페루">페루</a>에서 살았다. 파니아과는 <a href="/wiki/%EC%BF%A0%EC%8A%A4%EC%BD%94" title="쿠스코">쿠스코</a>에서 태어났으며 그곳에 있는 <a class="new" href="/w/index.php?title=%EC%82%B4%EB%A0%88%EC%8A%A4%ED%8B%B0%EC%95%88_%EA%B3%A0%EB%93%B1%ED%95%99%EA%B5%90&amp;action=edit&amp;redlink=1" title="살레스티안 고등학교 (없는 문서)">살레스티안 고등학교</a>를 나왔다. 그리고 나서, 그는 <a href="/wiki/%EC%BF%A0%EC%8A%A4%EC%BD%94" title="쿠스코">쿠스코</a>에 있는 <a class="new" href="/w/index.php?title=%EC%82%B0%EC%95%88%ED%86%A0%EB%8B%88%EC%98%A4_%EC%95%84%EB%B0%94%EB%93%9C_%EB%8C%80%ED%95%99&amp;action=edit&amp;redlink=1" title="산안토니오 아바드 대학 (없는 문서)">산안토니오 아바드 대학</a>에서 법학을 전공, 후에 <a href="/wiki/%EB%A6%AC%EB%A7%88" title="리마">리마</a>에 있는 <a class="new" href="/w/index.php?title=%EC%82%B0%EB%A7%88%EB%A5%B4%EC%BD%94%EC%8A%A4_%EC%8B%9C%EC%9E%A5_%EB%8C%80%ED%95%99&amp;action=edit&amp;redlink=1" title="산마르코스 시장 대학 (없는 문서)">산마르코스 시장 대학</a>을 졸업하였다. 후에 그는 정계에 들어갔다.</p>
<p><a href="/wiki/2000%EB%85%84" title="2000년">2000년</a> <a href="/wiki/%EC%95%8C%EB%B2%A0%EB%A5%B4%ED%86%A0_%ED%9B%84%EC%A7%80%EB%AA%A8%EB%A6%AC" title="알베르토 후지모리">알베르토 후지모리</a>가 갑작스런 사임을 하게 되자, 대통령이 되었다. 하지만 그것은 임시에 지나지 않았다. <a href="/wiki/2001%EB%85%84" title="2001년">2001년</a> 그는 <a href="/wiki/%EC%95%8C%EB%A0%88%ED%95%9C%EB%93%9C%EB%A1%9C_%ED%86%A8%EB%A0%88%EB%8F%84" title="알레한드로 톨레도">알레한드로 톨레도</a>에게 대통령직을 넘겨 주었다.</p>
<p><a href="/wiki/2006%EB%85%84" title="2006년">2006년</a>부터 그에게 이상증세가 찾아오기 시작하였고, 병원을 자주 방문하게 된다. 그러다 결국 사망하였다<sup class="reference" id="cite_ref-1"><a href="#cite_note-1">[1]</a></sup><sup class="reference" id="cite_ref-2"><a href="#cite_note-2">[2]</a></sup>.</p>
