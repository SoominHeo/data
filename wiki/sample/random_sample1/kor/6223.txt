<p><b>발렌틴 세르게예비치 파블로프</b> (Валентин Сергеевич Павлов, 1937년 9월 26일 ~ 2003년 3월 30일) 은 <a class="mw-redirect" href="/wiki/%EC%86%8C%EB%B9%84%EC%97%90%ED%8A%B8_%EC%97%B0%EB%B0%A9" title="소비에트 연방">소비에트 연방</a>의 정치인이다. 1991년 1월 ~ 1991년 8월 소비에트 연방의 총리를 지냈으며, <a href="/wiki/%EB%AF%B8%ED%95%98%EC%9D%BC_%EA%B3%A0%EB%A5%B4%EB%B0%94%EC%B4%88%ED%94%84" title="미하일 고르바초프">미하일 고르바초프</a>를 상대로 일으킨 <a href="/wiki/8%EC%9B%94_%EC%BF%A0%EB%8D%B0%ED%83%80" title="8월 쿠데타">8월 쿠데타</a>의 주모자 가운데 한 명이다.</p>
<p>파블로프는 <a href="/wiki/%EB%AA%A8%EC%8A%A4%ED%81%AC%EB%B0%94" title="모스크바">모스크바</a>에서 태어나, 모스크바 재정연구소 (Moscow Finance Institute) 를 졸업하고, 정부의 경제 학자 및 <a class="mw-redirect" href="/wiki/%EA%B3%A0%EC%8A%A4%ED%94%8C%EB%9E%80" title="고스플란">고스플란</a> (Госпла́н, 소비에트 연방 국가계획위원회) 재정 부문의 수장으로 있었으며, 1959년 이래 재무부에서 일하고, 1989년 소비에트 연방의 경제장관이 되었다. 1990년 <a class="new" href="/w/index.php?title=%EC%86%8C%EB%B9%84%EC%97%90%ED%8A%B8_%EC%97%B0%EB%B0%A9_%EC%A0%95%EC%B9%98%EA%B5%AD&amp;action=edit&amp;redlink=1" title="소비에트 연방 정치국 (없는 문서)">소비에트 연방 정치국</a>에 참가했으며, <a class="new" href="/w/index.php?title=%EB%8B%88%EC%BD%9C%EB%9D%BC%EC%9D%B4_%EB%A6%AC%EC%A6%88%EC%BD%94%ED%94%84&amp;action=edit&amp;redlink=1" title="니콜라이 리즈코프 (없는 문서)">니콜라이 리즈코프</a>의 후임으로 소비에트 연방의 총리가 되었다.</p>
<p>당시 소비에트 연방 서기장 <a href="/wiki/%EB%AF%B8%ED%95%98%EC%9D%BC_%EA%B3%A0%EB%A5%B4%EB%B0%94%EC%B4%88%ED%94%84" title="미하일 고르바초프">미하일 고르바초프</a>는 <a href="/wiki/%ED%8E%98%EB%A0%88%EC%8A%A4%ED%8A%B8%EB%A1%9C%EC%9D%B4%EC%B9%B4" title="페레스트로이카">페레스트로이카</a>에 대한 강경 보수파의 저항에 직면해 있었으며, 파블로프는 소비에트 연방에서 공화국의 권력 분산을 추진하는 고르바초프의 새로운 연방 조약에 반대하여, 부통령 <a href="/wiki/%EA%B2%90%EB%82%98%EB%94%94_%EC%95%BC%EB%82%98%EC%98%88%ED%94%84" title="겐나디 야나예프">겐나디 야나예프</a>, 내무장관 <a class="new" href="/w/index.php?title=%EB%B3%B4%EB%A6%AC%EC%8A%A4_%ED%91%B8%EA%B3%A0&amp;action=edit&amp;redlink=1" title="보리스 푸고 (없는 문서)">보리스 푸고</a>, 국방장관 <a href="/wiki/%EB%93%9C%EB%AF%B8%ED%8A%B8%EB%A6%AC_%EC%95%BC%EC%A1%B0%ED%94%84" title="드미트리 야조프">드미트리 야조프</a>를 비롯한 보수 강경파가 주도하는 국가비상사태위원회 (Государственный Комитет по Чрезвычайному Положению) 에 참여하였다. 위원회는 1991년 8월, 고르바초프의 퇴진을 요구하는 쿠데타를 일으켰으나, 쿠데타는 실패로 끝났고, 파블로프는 총리에서 파면된 후 체포되었다. 몇 개월 후 감옥에서 석방되었으며, 1994년 사면 받았다.</p>
<p>파블로프는 석방 후, <a class="mw-redirect" href="/wiki/%EC%83%81%EC%97%85_%EC%9D%80%ED%96%89" title="상업 은행">상업 은행</a> 은행장과 여러 아카데미의 소장을 맡은 것 외에, 러시아 자유 경제 사회 및 국제 관리 아카데미의 부사장을 지내기도 하였다. 2001년에는 <a href="/wiki/%EB%B8%94%EB%9D%BC%EB%94%94%EB%AF%B8%EB%A5%B4_%ED%91%B8%ED%8B%B4" title="블라디미르 푸틴">블라디미르 푸틴</a>에 대한 지지 의사를 표명했으며, 2003년 사망하였다.</p>
