<h1 class="firstHeading" id="firstHeading" lang="ko">발출론</h1>
<div class="mw-body-content" id="bodyContent">
<div id="siteSub">위키백과, 우리 모두의 백과사전.</div>
<div id="contentSub"></div>
<div class="mw-jump" id="jump-to-nav">
					이동:					<a href="#mw-head">둘러보기</a>, 					<a href="#p-search">검색</a>
</div>
<div class="mw-content-ltr" dir="ltr" id="mw-content-text" lang="ko"><script>function mfTempOpenSection(id){var block=document.getElementById("mf-section-"+id);block.className+=" open-block";block.previousSibling.className+=" open-block";}</script><p><b>발출론</b>(發出論, Emanationism) 또는 <b>유출설</b>(流出說)은 샘에서 물이 흘러나오듯이 <a href="/wiki/%ED%95%98%EB%8A%90%EB%8B%98" title="하느님">세계의 본원</a>이 세계와 만물을 구성하는 <a href="/wiki/%EC%97%90%EB%84%88%EC%A7%80" title="에너지">에너지</a>와 <a class="new" href="/w/index.php?title=%EC%A7%88%EB%A3%8C&amp;action=edit&amp;redlink=1" title="질료 (없는 문서)">질료</a>를 <a href="/wiki/%EC%98%81%EC%84%B1" title="영성">정묘한 상태</a>에서 <a href="/wiki/%EB%AC%BC%EC%A7%88" title="물질">거친 상태</a>로 계층적 또는 단계적으로 발출 또는 유출시켜 <a href="/wiki/%EB%AC%BC%EC%A7%88_%EC%9A%B0%EC%A3%BC" title="물질 우주">물질 우주</a>의 세계와 만물을 현현시키고 성립시키며, 또한 이렇게 성립된 <a href="/wiki/%EC%98%81%EC%84%B1" title="영성">정묘한 상태</a>로부터 <a href="/wiki/%EB%AC%BC%EC%A7%88" title="물질">거친 상태</a>로의 계층적 <b>·</b> 단계적 구조와 질서를 통해 <a href="/wiki/%EC%9A%B0%EC%A3%BC" title="우주">우주</a> 전체를 운행한다는 이론 또는 교의이다. <a class="mw-disambig" href="/wiki/%EC%98%81" title="영">영</a>(<a class="mw-disambig" href="/wiki/%EC%8A%A4%ED%94%BC%EB%A6%BF" title="스피릿">스피릿</a>) 또는 <a href="/wiki/%EC%A0%95%EC%8B%A0" title="정신">정신</a>을 본원으로 하는 <a href="/wiki/%EC%9C%A0%EC%8B%AC%EB%A1%A0" title="유심론">유심론</a>(<a href="/wiki/%EA%B4%80%EB%85%90%EB%A1%A0" title="관념론">관념론</a>)의 <a href="/wiki/%EC%84%B8%EA%B3%84%EA%B4%80" title="세계관">세계관</a>으로 물질 세계의 현실의 <a href="/wiki/%EC%A1%B4%EC%9E%AC" title="존재">존재</a>를 설명하는 이론 또는 주장이다.</p>
<p>발출론의 <a href="/wiki/%EC%84%B8%EA%B3%84%EA%B4%80" title="세계관">세계관</a>, <a href="/wiki/%EC%9A%B0%EC%A3%BC%EB%A1%A0" title="우주론">우주론</a> 혹은 <a class="new" href="/w/index.php?title=%EC%9A%B0%EC%A3%BC%EB%B0%9C%EC%83%9D%EB%A1%A0&amp;action=edit&amp;redlink=1" title="우주발생론 (없는 문서)">우주발생론</a>을 가진 대표적인 경우로는 <a href="/wiki/%ED%9E%8C%EB%91%90_%EC%B2%A0%ED%95%99" title="힌두 철학">힌두 철학</a>의 <a href="/wiki/%EC%82%BC%ED%82%A4%EC%95%84_%ED%95%99%ED%8C%8C" title="삼키아 학파">삼키아 학파</a> <b>·</b> <a href="/wiki/%ED%94%BC%ED%83%80%EA%B3%A0%EB%9D%BC%EC%8A%A4_%ED%95%99%ED%8C%8C" title="피타고라스 학파">피타고라스 학파</a> <b>·</b> <a href="/wiki/%EC%8B%A0%ED%94%BC%ED%83%80%EA%B3%A0%EB%9D%BC%EC%8A%A4_%ED%95%99%ED%8C%8C" title="신피타고라스 학파">신피타고라스 학파</a> <b>·</b> <a class="mw-redirect" href="/wiki/%EB%82%98%EC%8A%A4%ED%8B%B0%EC%8B%9C%EC%A6%98" title="나스티시즘">나스티시즘</a> <b>·</b> <a href="/wiki/%EC%8B%A0%ED%94%8C%EB%9D%BC%ED%86%A4%EC%A3%BC%EC%9D%98" title="신플라톤주의">신플라톤주의</a> <b>·</b> <a href="/wiki/%ED%97%A4%EB%A5%B4%EB%A9%94%EC%8A%A4%EC%A3%BC%EC%9D%98" title="헤르메스주의">헤르메스주의</a> <b>·</b> <a href="/wiki/%EC%B9%B4%EB%B0%9C%EB%9D%BC" title="카발라">카발라</a> <b>·</b> <a class="mw-redirect" href="/wiki/%EC%88%98%ED%94%BC%EC%A6%98" title="수피즘">수피즘</a> <b>·</b> <a href="/wiki/%EC%9E%A5%EB%AF%B8%EC%8B%AD%EC%9E%90%ED%9A%8C" title="장미십자회">장미십자회</a>(<a class="new" href="/w/index.php?title=%EB%B0%80%EA%B5%90%EC%A0%81_%EA%B8%B0%EB%8F%85%EA%B5%90&amp;action=edit&amp;redlink=1" title="밀교적 기독교 (없는 문서)">밀교적 기독교</a>) <b>·</b> <a href="/wiki/%EC%8B%A0%EC%A7%80%ED%95%99" title="신지학">신지학</a> 등이 있다.</p>
<h2><span class="mw-headline" id=".EC.9A.A9.EC.96.B4">용어</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%B0%9C%EC%B6%9C%EB%A1%A0&amp;action=edit&amp;section=1" title="부분 편집: 용어">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p><b>유출설</b>(流出說)이라는 낱말에는 <a href="/wiki/%EC%9D%98%EC%A7%80" title="의지">의지</a>적으로 유출시킨다는 의미도 들어있지만, <a href="/wiki/%ED%95%98%EB%8A%90%EB%8B%98" title="하느님">세계의 본원</a>이 가진 <a href="/wiki/%EC%97%90%EB%84%88%EC%A7%80" title="에너지">에너지</a>와 <a class="new" href="/w/index.php?title=%EC%A7%88%EB%A3%8C&amp;action=edit&amp;redlink=1" title="질료 (없는 문서)">질료</a>가 흘러넘쳐서 만물이 성립되었다고 해석할 여지가 있다. 이에 비해 <b>발출론</b>(發出論)이라는 낱말은 <a href="/wiki/%ED%95%98%EB%8A%90%EB%8B%98" title="하느님">세계의 본원</a>이 <a href="/wiki/%EC%9D%98%EC%A7%80" title="의지">의지</a>적으로 <a href="/wiki/%EC%97%90%EB%84%88%EC%A7%80" title="에너지">에너지</a>와 <a class="new" href="/w/index.php?title=%EC%A7%88%EB%A3%8C&amp;action=edit&amp;redlink=1" title="질료 (없는 문서)">질료</a>를 내보내어 <a href="/wiki/%EC%84%B8%EA%B3%84" title="세계">세계</a>와 만물을 성립시킨다는 의미가 유출설이라는 낱말보다는 더 뚜렷하다.</p>
