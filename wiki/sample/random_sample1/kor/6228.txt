<h1 class="firstHeading" id="firstHeading" lang="ko">발칸 동맹</h1>
<div class="mw-body-content" id="bodyContent">
<div id="siteSub">위키백과, 우리 모두의 백과사전.</div>
<div id="contentSub"></div>
<div class="mw-jump" id="jump-to-nav">
					이동:					<a href="#mw-head">둘러보기</a>, 					<a href="#p-search">검색</a>
</div>
<div class="mw-content-ltr" dir="ltr" id="mw-content-text" lang="ko"><script>function mfTempOpenSection(id){var block=document.getElementById("mf-section-"+id);block.className+=" open-block";block.previousSibling.className+=" open-block";}</script><p><b>발칸 동맹</b>은 <a href="/wiki/1912%EB%85%84" title="1912년">1912년</a> <a href="/wiki/%EB%B0%9C%EC%B9%B8_%EB%B0%98%EB%8F%84" title="발칸 반도">발칸 반도</a>에 위치한 <a href="/wiki/%EA%B7%B8%EB%A6%AC%EC%8A%A4_%EC%99%95%EA%B5%AD" title="그리스 왕국">그리스 왕국</a>, <a href="/wiki/%EB%B6%88%EA%B0%80%EB%A6%AC%EC%95%84_%EC%99%95%EA%B5%AD" title="불가리아 왕국">불가리아 왕국</a>, <a href="/wiki/%EC%84%B8%EB%A5%B4%EB%B9%84%EC%95%84_%EC%99%95%EA%B5%AD" title="세르비아 왕국">세르비아 왕국</a>, <a href="/wiki/%EB%AA%AC%ED%85%8C%EB%84%A4%EA%B7%B8%EB%A1%9C_%EC%99%95%EA%B5%AD" title="몬테네그로 왕국">몬테네그로 왕국</a>이 <a href="/wiki/%EC%98%A4%EC%8A%A4%EB%A7%8C_%EC%A0%9C%EA%B5%AD" title="오스만 제국">오스만 제국</a>에 대항하기 위해 결성한 방위 동맹이다.</p>
<p><a href="/wiki/1911%EB%85%84" title="1911년">1911년</a>에 발발한 <a href="/wiki/%EC%9D%B4%ED%83%88%EB%A6%AC%EC%95%84-%ED%8A%80%EB%A5%B4%ED%81%AC_%EC%A0%84%EC%9F%81" title="이탈리아-튀르크 전쟁">이탈리아-튀르크 전쟁</a>은 오스만 제국의 쇠퇴, 발칸 반도 국가의 영향력 확대를 가져다 주었다. 발칸 반도에서 세력을 확장하고 있던 <a href="/wiki/%EC%98%A4%EC%8A%A4%ED%8A%B8%EB%A6%AC%EC%95%84-%ED%97%9D%EA%B0%80%EB%A6%AC_%EC%A0%9C%EA%B5%AD" title="오스트리아-헝가리 제국">오스트리아-헝가리 제국</a>을 견제하기 위한 차원에서 <a href="/wiki/%EB%9F%AC%EC%8B%9C%EC%95%84_%EC%A0%9C%EA%B5%AD" title="러시아 제국">러시아 제국</a>의 주도로 결성되었다.</p>
<p>이 동맹은 <a href="/wiki/%EB%B0%9C%EC%B9%B8_%EB%B0%98%EB%8F%84" title="발칸 반도">발칸 반도</a>에서의 <a href="/wiki/%EC%8A%AC%EB%9D%BC%EB%B8%8C%EC%A1%B1" title="슬라브족">슬라브족</a> 세력의 결집, <a href="/wiki/%EB%B2%94%EC%8A%AC%EB%9D%BC%EB%B8%8C%EC%A3%BC%EC%9D%98" title="범슬라브주의">범슬라브주의</a>의 구체적인 출현으로 이어졌지만 제1차 발칸 전쟁에서 승리한 발칸 동맹 국가들 간의 전후 처리 문제에 대한 대립으로 인해 <a href="/wiki/%EC%A0%9C2%EC%B0%A8_%EB%B0%9C%EC%B9%B8_%EC%A0%84%EC%9F%81" title="제2차 발칸 전쟁">제2차 발칸 전쟁</a>이 발발하면서 붕괴되고 만다.</p>
