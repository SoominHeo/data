<p><b>발칸컵</b>(Balkans Cup)은 <a href="/wiki/%EB%8F%99%EB%82%A8%EC%9C%A0%EB%9F%BD" title="동남유럽">동남유럽</a>에 위치한 국가인 <a href="/wiki/%EC%95%8C%EB%B0%94%EB%8B%88%EC%95%84" title="알바니아">알바니아</a>, <a href="/wiki/%EB%B6%88%EA%B0%80%EB%A6%AC%EC%95%84" title="불가리아">불가리아</a>, <a href="/wiki/%EA%B7%B8%EB%A6%AC%EC%8A%A4" title="그리스">그리스</a>, <a href="/wiki/%EB%A3%A8%EB%A7%88%EB%8B%88%EC%95%84" title="루마니아">루마니아</a>, <a href="/wiki/%ED%84%B0%ED%82%A4" title="터키">터키</a>, <a href="/wiki/%EC%9C%A0%EA%B3%A0%EC%8A%AC%EB%9D%BC%EB%B9%84%EC%95%84" title="유고슬라비아">유고슬라비아</a>의 축구 클럽이 참가했던 국제 클럽 축구 대회이다. <a href="/wiki/1961%EB%85%84" title="1961년">1961년</a>부터 <a href="/wiki/1994%EB%85%84" title="1994년">1994년</a>까지 개최되었다.</p>
