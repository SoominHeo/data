<div class="mw-content-ltr" dir="ltr" id="mw-content-text" lang="ko"><script>function mfTempOpenSection(id){var block=document.getElementById("mf-section-"+id);block.className+=" open-block";block.previousSibling.className+=" open-block";}</script><p><b>발터 벤야민</b>(Walter Bendix Schönflies Benjamin, <a href="/wiki/1892%EB%85%84" title="1892년">1892년</a> <a href="/wiki/7%EC%9B%94_15%EC%9D%BC" title="7월 15일">7월 15일</a> ~ <a href="/wiki/1940%EB%85%84" title="1940년">1940년</a> <a href="/wiki/9%EC%9B%94_27%EC%9D%BC" title="9월 27일">9월 27일</a>)은 유대계 <a href="/wiki/%EB%8F%85%EC%9D%BC" title="독일">독일</a>인으로 <a class="mw-redirect" href="/wiki/%EB%A7%88%EB%A5%B4%ED%81%AC%EC%8A%A4%EC%A3%BC%EC%9D%98%EC%9E%90" title="마르크스주의자">마르크스주의자</a>이자 <a class="mw-redirect" href="/wiki/%ED%8F%89%EB%A1%A0%EA%B0%80" title="평론가">문학평론가</a>이며 <a href="/wiki/%EC%B2%A0%ED%95%99%EC%9E%90" title="철학자">철학자</a>이다. 그는 <a href="/wiki/%EA%B2%8C%EB%A5%B4%EC%88%8C_%EC%88%84%EB%A0%98" title="게르숌 숄렘">게르숌 숄렘</a>의 <a href="/wiki/%EC%B9%B4%EB%B0%9C%EB%9D%BC" title="카발라">유대교 신비주의</a>와 <a href="/wiki/%EB%B2%A0%EB%A5%B4%ED%86%A8%ED%8A%B8_%EB%B8%8C%EB%A0%88%ED%9E%88%ED%8A%B8" title="베르톨트 브레히트">베르톨트 브레히트</a>로부터 마르크시즘의 영향을 크게 받았으며 또한 <a class="new" href="/w/index.php?title=%EB%B9%84%ED%8C%90%EC%9D%B4%EB%A1%A0&amp;action=edit&amp;redlink=1" title="비판이론 (없는 문서)">비판이론</a>의 <a href="/wiki/%ED%94%84%EB%9E%91%ED%81%AC%ED%91%B8%EB%A5%B4%ED%8A%B8_%ED%95%99%ED%8C%8C" title="프랑크푸르트 학파">프랑크푸르트 학파</a>와도 관련이 있다.</p>
