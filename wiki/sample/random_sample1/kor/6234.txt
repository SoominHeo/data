<p><b>발트 함대</b>( - 艦隊, <span style="font-size: smaller;"><a href="/wiki/%EB%9F%AC%EC%8B%9C%EC%95%84%EC%96%B4" title="러시아어">러시아어</a>:</span> <span lang="ru" xml:lang="ru">Балтийский флот, <b>БФ</b></span>, <span style="font-size: smaller;"><a href="/wiki/%EC%98%81%EC%96%B4" title="영어">영어</a>:</span> <span lang="en" xml:lang="en">Baltic Fleet</span>)는 <a href="/wiki/%EB%9F%AC%EC%8B%9C%EC%95%84" title="러시아">러시아</a>와 <a href="/wiki/%EC%86%8C%EB%A0%A8" title="소련">소련</a>의 함대이다. 소련 시대 이중으로 <a class="new" href="/w/index.php?title=%EC%A0%81%EA%B8%B0%ED%9B%88%EC%9E%A5&amp;action=edit&amp;redlink=1" title="적기훈장 (없는 문서)">적기훈장</a>을 수여받았다. <a href="/wiki/1703%EB%85%84" title="1703년">1703년</a> <a href="/wiki/5%EC%9B%94_18%EC%9D%BC" title="5월 18일">5월 18일</a> <a class="mw-redirect" href="/wiki/%EB%8C%80%EB%B6%81%EB%B0%A9%EC%A0%84%EC%9F%81" title="대북방전쟁">대북방전쟁</a> 와중에 <a class="mw-redirect" href="/wiki/%ED%91%9C%ED%8A%B8%EB%A5%B4_%EB%8C%80%EC%A0%9C" title="표트르 대제">표트르 대제</a>에 의해 창립되어, 러시아 해군 함대 중 가장 오랜 역사를 가지고 있다.<sup class="reference" id="cite_ref-1"><a href="#cite_note-1">[1]</a></sup> 발트함대는 <a class="mw-redirect" href="/wiki/%EB%9F%AC%EC%9D%BC%EC%A0%84%EC%9F%81" title="러일전쟁">러일전쟁</a> 당시 <a href="/wiki/%EB%9F%AC%EC%8B%9C%EC%95%84_%ED%83%9C%ED%8F%89%EC%96%91_%ED%95%A8%EB%8C%80" title="러시아 태평양 함대">태평양함대</a>가 패배한 이후, <a href="/wiki/%EC%93%B0%EC%8B%9C%EB%A7%88_%ED%95%B4%EC%A0%84" title="쓰시마 해전">쓰시마 해전</a>에서 패하기까지 중요한 역할을 수행하였으며, <a href="/wiki/%EC%A7%80%EB%85%B8%EB%B9%84_%EB%A1%9C%EC%A0%9C%EC%8A%A4%ED%8A%B8%EB%B2%A4%EC%8A%A4%ED%82%A4" title="지노비 로제스트벤스키">지노비 로제스트벤스키</a> 제독의 지휘하에 동아시아까지의 먼 항해를 감행하였다.</p>
<p>발트함대의 구성원들은 <a class="mw-redirect" href="/wiki/%EB%B3%BC%EC%85%B0%EB%B9%84%ED%82%A4_%ED%98%81%EB%AA%85" title="볼셰비키 혁명">볼셰비키 혁명</a>에서 중요한 역할을 수행했다. 그러나 점차 볼셰비키 신정부와의 관계가 악화되어 <a href="/wiki/1921%EB%85%84" title="1921년">1921년</a> <a href="/wiki/%ED%81%AC%EB%A1%A0%EC%8B%9C%ED%83%80%ED%8A%B8_%EB%B0%98%EB%9E%80" title="크론시타트 반란">크론시타트 반란</a>의 원인이 되기도 했다. 반란이 진압된 이후 한동안 정상적인 함대로서의 기능이 정지되었다. <a class="mw-redirect" href="/wiki/%EA%B2%A8%EC%9A%B8%EC%A0%84%EC%9F%81" title="겨울전쟁">겨울전쟁</a> 때는 제한적인 역할만을 수행하였고, 1941년 8월 <a href="/wiki/%EC%86%8C%EB%A0%A8%EC%9D%98_%ED%83%88%EB%A6%B0_%EC%B2%A0%EC%88%98" title="소련의 탈린 철수">소련의 탈린 철수</a>를 수행하였다.</p>
