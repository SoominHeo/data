<p><b>발트의 길</b>(<span style="font-size: smaller;"><a href="/wiki/%EC%97%90%EC%8A%A4%ED%86%A0%EB%8B%88%EC%95%84%EC%96%B4" title="에스토니아어">에스토니아어</a>:</span> <span lang="et" xml:lang="et">Balti kett</span>, <span style="font-size: smaller;"><a href="/wiki/%EB%9D%BC%ED%8A%B8%EB%B9%84%EC%95%84%EC%96%B4" title="라트비아어">라트비아어</a>:</span> <span lang="lv" xml:lang="lv">Baltijas ceļš</span>, <span style="font-size: smaller;"><a href="/wiki/%EB%A6%AC%ED%88%AC%EC%95%84%EB%8B%88%EC%95%84%EC%96%B4" title="리투아니아어">리투아니아어</a>:</span> <span lang="lt" xml:lang="lt">Baltijos kelias</span>)은 <a href="/wiki/1989%EB%85%84" title="1989년">1989년</a> <a href="/wiki/8%EC%9B%94_23%EC%9D%BC" title="8월 23일">8월 23일</a> <a href="/wiki/%EB%B0%9C%ED%8A%B8_3%EA%B5%AD" title="발트 3국">발트 3국</a>(<a href="/wiki/%EC%97%90%EC%8A%A4%ED%86%A0%EB%8B%88%EC%95%84" title="에스토니아">에스토니아</a>, <a href="/wiki/%EB%9D%BC%ED%8A%B8%EB%B9%84%EC%95%84" title="라트비아">라트비아</a>, <a href="/wiki/%EB%A6%AC%ED%88%AC%EC%95%84%EB%8B%88%EC%95%84" title="리투아니아">리투아니아</a>) 국민 200만 명이 만든 약 600km의 인간 사슬이다. <a href="/wiki/1939%EB%85%84" title="1939년">1939년</a> <a href="/wiki/8%EC%9B%94_23%EC%9D%BC" title="8월 23일">8월 23일</a>에 체결된 <a href="/wiki/%EB%8F%85%EC%9D%BC-%EC%86%8C%EB%A0%A8_%EB%B6%88%EA%B0%80%EC%B9%A8_%EC%A1%B0%EC%95%BD" title="독일-소련 불가침 조약">독일-소련 불가침 조약</a>에 따라 발트 3국이 <a href="/wiki/%EC%86%8C%EB%A0%A8" title="소련">소련</a>에 편입된 지 50주년이 되는 날을 맞아 열린 시위였으며 인간 사슬은 <a href="/wiki/%ED%83%88%EB%A6%B0" title="탈린">탈린</a>, <a href="/wiki/%EB%A6%AC%EA%B0%80" title="리가">리가</a>, <a href="/wiki/%EB%B9%8C%EB%89%B4%EC%8A%A4" title="빌뉴스">빌뉴스</a>까지 이어졌다.</p>
