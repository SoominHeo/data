<p><b>선왕</b>(宣王, ? ~ <a href="/wiki/830%EB%85%84" title="830년">830년</a>, 재위 : <a href="/wiki/818%EB%85%84" title="818년">818년</a> ~ <a href="/wiki/830%EB%85%84" title="830년">830년</a>) 은 <a href="/wiki/%EB%B0%9C%ED%95%B4" title="발해">발해</a>의 제10대 <a href="/wiki/%EA%B5%B0%EC%A3%BC" title="군주">국왕</a>이며 <a href="/wiki/%ED%94%BC%ED%9C%98" title="피휘">휘</a>는 <b>인수</b>(仁秀)이다.</p>
<h2><span class="mw-headline" id=".EC.B9.98.EC.84.B8">치세</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%B0%9C%ED%95%B4_%EC%84%A0%EC%99%95&amp;action=edit&amp;section=1" title="부분 편집: 치세">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>선왕은 <a class="mw-redirect" href="/wiki/%EB%8C%80%EC%A1%B0%EC%98%81" title="대조영">대조영</a>(大祚榮)의 아우인 <a href="/wiki/%EB%8C%80%EC%95%BC%EB%B0%9C" title="대야발">대야발</a>(大野勃)의 현손(玄孫)이며, <a href="/wiki/%EC%97%B0%ED%98%B8" title="연호">연호</a>는 <b><a href="/wiki/%EA%B1%B4%ED%9D%A5_(%EB%B0%9C%ED%95%B4)" title="건흥 (발해)">건흥</a></b>(建興)이라 하였다.</p>
<p>선왕은 발해의 영토를 크게 확장시켰는데, 이때 발해의 영토는 남으로 <a href="/wiki/%EC%8B%A0%EB%9D%BC" title="신라">신라</a>와 접하고, 서로는 <a href="/wiki/%EB%9E%B4%EC%98%A4%ED%97%88" title="랴오허">요하강</a> 유역을 차지하였으며, <a href="/wiki/%EA%B3%A0%EA%B5%AC%EB%A0%A4" title="고구려">고구려</a>와 <a href="/wiki/%EB%B6%80%EC%97%AC" title="부여">부여</a>의 옛 영토를 대부분 회복하였다. 또한 북쪽의 <a href="/wiki/%ED%9D%91%EC%88%98%EB%A7%90%EA%B0%88" title="흑수말갈">흑수말갈</a> 등 발해에 대항하던 모든 <a href="/wiki/%EB%A7%90%EA%B0%88" title="말갈">말갈</a>의 부족들도 복속시켰다.</p>
<p>선왕은 발해 중흥의 대업을 이룬 중흥군주(中興君主)로서 <a href="/wiki/%EB%8B%B9%EB%82%98%EB%9D%BC" title="당나라">당나라</a>로부터 <b><a class="mw-redirect" href="/wiki/%ED%95%B4%EB%8F%99%EC%84%B1%EA%B5%AD" title="해동성국">해동성국</a></b>(海東盛國)이라는 칭호를 얻었다. 당과 밀접한 외교를 하여 문화를 발달시켰으며, 당나라 문물을 받아들여 행정구역을 5경(京) 15부(府) 62주(州)로 개편하고 학술을 진흥시키는 등 발해의 최전성기를 이루었다.</p>
