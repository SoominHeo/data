<p>《<b>밤비노</b>》(<span style="font-size: smaller;"><a href="/wiki/%EC%9D%BC%EB%B3%B8%EC%96%B4" title="일본어">일본어</a>:</span> <span lang="ja" xml:lang="ja">バンビ〜ノ!</span>, <span style="font-size: smaller;"><a href="/wiki/%EC%98%81%EC%96%B4" title="영어">영어</a>:</span> <span lang="en" xml:lang="en">Bambino!</span>)는 <a class="new" href="/w/index.php?title=%EC%84%B8%ED%82%A4%EC%95%BC_%ED%85%8C%EC%B8%A0%EC%A7%80&amp;action=edit&amp;redlink=1" title="세키야 테츠지 (없는 문서)">세키야 테츠지</a>에 의한 <a href="/wiki/%EC%9D%BC%EB%B3%B8" title="일본">일본</a>의 요리 만화로 이를 원작으로 한 드라마이다. &lt;빅 코믹 스피리츠&gt; (<a href="/wiki/%EC%87%BC%EA%B0%80%EC%BF%A0%EC%B9%B8" title="쇼가쿠칸">쇼가쿠칸</a>)에서 <a href="/wiki/2005%EB%85%84" title="2005년">2005년</a> 2호부터 연재를 시작하며 제1부로서 <a href="/wiki/2009%EB%85%84" title="2009년">2009년</a> 13호까지 연재되었다. 그 후, 동년 19호에서 <a href="/wiki/2013%EB%85%84" title="2013년">2013년</a> 3호 제2부 "밤비노 SECONDO"가 연재되었다. 단행본은 <a href="/wiki/%EC%87%BC%EA%B0%80%EC%BF%A0%EC%B9%B8" title="쇼가쿠칸">쇼가쿠칸</a> 빅 코믹스에서 제1부가 전체 15권, 제2부이 전체 13권으로 간행되고 있다. 제목의 '밤비노'는 <a href="/wiki/%EC%9D%B4%ED%83%88%EB%A6%AC%EC%95%84%EC%96%B4" title="이탈리아어">이탈리아어</a>로 <a href="/wiki/%EC%86%8C%EB%85%84" title="소년">소년</a>의 의미한다.</p>
<p><a href="/wiki/2007%EB%85%84" title="2007년">2007년</a> 제53회 <a href="/wiki/%EC%87%BC%EA%B0%80%EC%BF%A0%EC%B9%B8" title="쇼가쿠칸">쇼가쿠칸</a> 만화상 일반 부문을 수상했다.</p>

<h2><span class="mw-headline" id=".EC.8A.A4.ED.86.A0.EB.A6.AC">스토리</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%B0%A4%EB%B9%84%EB%85%B8!&amp;action=edit&amp;section=1" title="부분 편집: 스토리">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>반 쇼고는 <a href="/wiki/%ED%9B%84%EC%BF%A0%EC%98%A4%EC%B9%B4_%EC%8B%9C" title="후쿠오카 시">후쿠오카</a>에서 대학에 다니며 <a href="/wiki/%EC%9D%B4%ED%83%88%EB%A6%AC%EC%95%84_%EC%9A%94%EB%A6%AC" title="이탈리아 요리">이탈리안 레스토랑</a>에서 아르바이트를 하고 있다. 어느 날 주인의 권유로 봄방학 동안만 <a href="/wiki/%EB%8F%84%EC%BF%84_%EB%8F%84" title="도쿄 도">도쿄</a> <a href="/wiki/%EB%A1%AF%ED%8F%B0%EA%B8%B0" title="롯폰기">롯폰기</a>의 유명한 <a href="/wiki/%EC%9D%B4%ED%83%88%EB%A6%AC%EC%95%84_%EC%9A%94%EB%A6%AC" title="이탈리아 요리">이탈리안 레스토랑</a> <b>바카날레</b>에서 조수로 일하기로 한다. 나름대로 요리를 잘한다고 자신만만했던 반은 자기가 일하던 레스토랑과는 비교도 안 되는 규모와 인원, 그리고 요리 속도에 놀라 쉽게 적응하지 못한다.</p>
<p>잇단 실수로 도움이 되기는커녕 오히려 주방에서 거치적거리는 존재가 된 반은 자신이 얼마나 우물 안 개구리였는지를 깨닫는다. 그리고 반은 처음부터 다시 시작하는 마음으로 그곳에서 버텨나가기로 결심한다.</p>

<p>《<b>밤비노!</b>》는 <a href="/wiki/%EB%8B%9B%ED%8F%B0_TV_%EB%B0%A9%EC%86%A1%EB%A7%9D" title="닛폰 TV 방송망">NTV</a>에서 <a href="/wiki/2007%EB%85%84" title="2007년">2007년</a> <a href="/wiki/4%EC%9B%94_18%EC%9D%BC" title="4월 18일">4월 18일</a>부터 <a href="/wiki/2007%EB%85%84" title="2007년">2007년</a> <a href="/wiki/6%EC%9B%94_27%EC%9D%BC" title="6월 27일">6월 27일</a>까지 매주 <a href="/wiki/%EC%88%98%EC%9A%94%EC%9D%BC" title="수요일">수요일</a> 밤 10시에 방송된 <a href="/wiki/%EC%9D%BC%EB%B3%B8" title="일본">일본</a>의 <a href="/wiki/%EB%93%9C%EB%9D%BC%EB%A7%88" title="드라마">드라마</a>이다.</p>
<h3><span class="mw-headline" id=".EA.B0.9C.EC.9A.94">개요</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%B0%A4%EB%B9%84%EB%85%B8!&amp;action=edit&amp;section=3" title="부분 편집: 개요">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<ul>
<li><a href="/wiki/%EB%8B%9B%ED%8F%B0_TV_%EB%B0%A9%EC%86%A1%EB%A7%9D_%EC%88%98%EC%9A%94%EC%9D%BC_%EB%B0%A4_10%EC%8B%9C_%EB%93%9C%EB%9D%BC%EB%A7%88" title="닛폰 TV 방송망 수요일 밤 10시 드라마">NTV 수요 드라마</a>에서 남성의 배우가가 주역을 맡는 것은 <a href="/wiki/2004%EB%85%84" title="2004년">2004년</a> 10월 ~ 12월에 방영 된 《가장 소중한 사람은 누구입니까?》의 <a class="new" href="/w/index.php?title=%ED%82%A4%EC%8B%9C%ED%83%80%EB%8B%88_%EA%B3%A0%EB%A1%9C&amp;action=edit&amp;redlink=1" title="키시타니 고로 (없는 문서)">키시타니 고로</a> 이후 2년 반만의 일이다 . 또한 주연의 <a href="/wiki/%EB%A7%88%EC%B8%A0%EB%AA%A8%ED%86%A0_%EC%A4%80" title="마츠모토 준">마츠모토 준</a>에게 있어서 《소년 탐정 김전일》 이후 6년만의 연속 드라마 단독 주연 및 《[[고쿠센 ]》 이래 5년만의 수요 드라마 범위의 주연이다.</li>
<li><a href="/wiki/%EC%BA%90%EC%B9%98%ED%94%84%EB%A0%88%EC%9D%B4%EC%A6%88" title="캐치프레이즈">캐치프레이즈</a>는 〈<b>"처음에는 누구나 애송이 (밤비)이다 !!!"</b>〉</li>
</ul>
<h3><span class="mw-headline" id=".EB.93.B1.EC.9E.A5.EC.9D.B8.EB.AC.BC">등장인물</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%B0%A4%EB%B9%84%EB%85%B8!&amp;action=edit&amp;section=4" title="부분 편집: 등장인물">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<dl>
<dt>트랏토리아 '밧카나레'(Trattoria Baccanale)</dt>
</dl>
<ul>
<li>반 쇼고 (21세) - <a class="mw-redirect" href="/wiki/%EB%A7%88%EC%93%B0%EB%AA%A8%ED%86%A0_%EC%A4%80" title="마쓰모토 준">마츠모토 준</a></li>
</ul>
<dl>
<dd>본작의 주인공이자, 자기 싫어하는 성격과 순수하며 곧은 마음의 소유자이다. 하카타에 있는 작은 레스토랑에서 도쿄 롯폰기의 일류 트랏토리아 &lt;밧카날레&gt;에 보조로 온 열정 있는 대학교 4학년생으로 예쁜 여자친구도 있다. 하카타에서는 아르바이트생 곧 비정규직 노동자이면서도 가게를 맡는 등 실력에는 자신이 있다. 대학교의 봄방학이 되자 엔도의 추천으로 도쿄 록폰기에 있는 &lt;바카날레&gt;에 보조로 오게된다. 요리만큼은 자신이 있다고 생각했지만 프로만 모여있는 "바카날레"에 오고 나서는 보조가 아닌 별다르지 않은 아르바이트 생으로 전략하고 만다. 하지만 그런 <b>바카날레</b>에서 매력을 느끼게 되면서 좌절하지 않고 자신의 꿈을 향해 전진한다.</dd>
</dl>
<ul>
<li>시시도 텟칸 (55세) - <a href="/wiki/%EC%9D%B4%EC%B9%98%EB%AC%B4%EB%9D%BC_%EB%A7%88%EC%82%AC%EC%B9%98%EC%B9%B4" title="이치무라 마사치카">이치무라 마사치카</a></li>
</ul>
<dl>
<dd>트랏토리아 &lt;밧카날레&gt;의 오너 쉐프. 옛날부터 아는 사이엔 엔도의 부탁을 흔쾌히 받아들여 주방 보조를 반을 받아들인다. 쉐프로서의 솜씨를 도쿄 전체에 알려진 유명 쉐프로 성공한 사람으로서의 자신감이 베어 나온다. 이 당당한 풍격과 눈에 띄지 않는 배려로 &lt;밧카날레의 얼굴&gt;로서 매일매일 손님들을 접대한다. 너무나도 아끼는 &lt;밧카날레&gt;의 모든 것을 컨트롤하며 지탱하고 있다.</dd>
</dl>
<ul>
<li>시시도 마유키 (32세) - <a href="/wiki/%EC%9A%B0%EC%B9%98%EB%8B%A4_%EC%9C%A0%ED%82%A4" title="우치다 유키">우치다 유키</a></li>
</ul>
<dl>
<dd>트랏토리아 &lt;밧카날레&gt;의 디레또레(direttore), 지배인. 오너 쉐프인 시시도 텟칸의 딸. 정장이 잘 어울리는 세련된 여성으로 아름다우면서도 당당한 분위기로 &lt;밧카날레&gt;의 경영을 맡고 있는 수완가이다. 요나미네, 쿠와바라 두 사람과 과거 연인관계였던 듯하다.</dd>
</dl>
<dl>
<dt>Cucina (주방)</dt>
</dl>
<ul>
<li>히비노 아스카 (23세) - <a class="mw-redirect" href="/wiki/%EA%B0%80%EB%A6%AC%EB%82%98" title="가리나">카리나</a></li>
</ul>
<dl>
<dd>트랏토리아 &lt;밧카날레&gt;의 쉐프. 안티파스토 &lt;전채&gt; 담당. 주방 안에서도 홍일점이지만 기가 세고 남자들에게 물러서지 않는다. 지기 싫어하는 성격으로 보이지 않는 곳에서 남들 이상으로 연습과 연구를 거듭하는 노력가이다. 평상시에는 아주 평범한 여성스러움을 지녔으며 일이 좀처럼 진척되지 않아 주춤거리는 반에게 여러 조언을 하는 등 자상한 일면도 있다.</dd>
</dl>
<ul>
<li>카토리 노조미 (28세) - <a href="/wiki/%EC%82%AC%ED%86%A0_%EB%A5%98%ED%83%80" title="사토 류타">사토 류타</a></li>
</ul>
<dl>
<dd>트랏토리아 &lt;밧카날레&gt;의 쉐프. <b>프리모 피아토</b>(Primo Piatto) &lt;파스타, 리조트&gt; 담당. 고교를 중퇴하고 이 업계에 뛰어들었으며, 밑바닥부터 올라온 쉐프로 일에 대한 자부심을 갖고 있으며 다른 사람이든 자신에게든 엄하다. 그 때문에 일이 있어서는 조금도 타협을 하지 않으며, 지도를 맡게 된 반에게도 엄하게 대한다. 반이 제일 부딪히게 되는 벽이다.</dd>
</dl>
<ul>
<li>쿠와바라 아츠시 (35세) - <a class="mw-redirect" href="/wiki/%EC%82%AC%EC%82%AC%ED%82%A4_%EA%B5%AC%EB%9D%BC%EB%85%B8%EC%8A%A4%EC%BC%80" title="사사키 구라노스케">사사키 쿠라노스케</a></li>
</ul>
<dl>
<dd>트랏토리아 &lt;밧카날레&gt;의 부요리장. 세콘도 삐야또(Secondo Piatto) &lt;육류, 어류 요리&gt; 담당으로 야심가에 견실하면서도 착실한 성격으로 완벽주의자이다. 신경질적인 면이 있어 감정적으로 행동하는 것과 분쟁을 싫어하며, 날카로운 관찰력으로 전장과 같은 주방을 통솔한다. 이른바 &lt;밧카날레&gt; 주방의 꽃이다.</dd>
</dl>
<ul>
<li>오다 토시오 (34세) - <a class="new" href="/w/index.php?title=%ED%99%8B%EC%83%A8&amp;action=edit&amp;redlink=1" title="홋샨 (없는 문서)">홋샨</a></li>
</ul>
<dl>
<dd>트랏토리아 &lt;밧카날레&gt;의 파스티츠첼레 &lt;돌체 담당&gt;이다. 거의 입을 열지 않는, 수수께끼의 인물이지만 실력은 확실해서 모든 사람들을 매료시키는 섬세하면서도 독창적인 디저트를 잇달아 만들어낸다.</dd>
</dl>
<ul>
<li>세노오 마사시 (20세) - <a href="/wiki/%EB%AC%B4%EC%B9%B4%EC%9D%B4_%EC%98%A4%EC%82%AC%EB%AC%B4" title="무카이 오사무">무카이 오사무</a> </li>
</ul>
<dl>
<dd>주방 업 렌 디스타 / 견습 [제1화 ~ 제9화] → 안티파스토 / 전채 담당 [제10화, 최종화])</dd>
</dl>
<dl>
<dt>Sala (홀)</dt>
</dl>
<ul>
<li>요나미네 츠카사 (35세) - <a href="/wiki/%ED%82%A4%ED%83%80%EB%AC%B4%EB%9D%BC_%EC%B9%B4%EC%A6%88%ED%82%A4" title="키타무라 카즈키">키타무라 카즈키</a></li>
</ul>
<dl>
<dd>트랏토리아 &lt;<b>밧카날레</b>&gt;의 급사장. 항상 미소가 끊이질 않고 우아한 자태로 완벽하게 손님을 대한다. 그가 일단 홀에 나오면 밝고 화사한 가운데서도 금세 활기가 넘치고, 그를 만나기 위해 오는 단골손님도 많다. &lt;밧카날레&gt; 홀의 진정한 꽃이다.</dd>
</dl>
<ul>
<li>마나가와 코즈에 (20세) - <a class="mw-redirect" href="/wiki/%EC%BD%94%EB%A7%88%EC%B8%A0_%EC%95%84%EC%95%BC%EC%B9%B4" title="코마츠 아야카">코마츠 아야카</a></li>
</ul>
<dl>
<dd>트랏토리아 &lt;밧카날레&gt;의 까메이에라 (웨이트리스). 홀의 아이돌 같은 존재로 귀여운 미소가 손님뿐만 아니라 홀의 직원들의 마음을 달래준다. 사실은 쉐프를 지망하고 있었는데 그 때문에 갑자기 나타난 반에게 질투심을 갖는다.</dd>
</dl>

<h3><span class="mw-headline" id=".EC.8A.A4.ED.95.80_.EC.98.A4.ED.94.84_.EB.93.9C.EB.9D.BC.EB.A7.88">스핀 오프 드라마</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%B0%A4%EB%B9%84%EB%85%B8!&amp;action=edit&amp;section=8" title="부분 편집: 스핀 오프 드라마">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>매회 방송 종료 후에 공식 사이트 및 모바일 사이트에서 스핀 오프 드라마가 전달되었다. 《밤비노 스핀 오프》의 각본은 본편과 마찬가지로 오카다 요시카즈가 담당하고, 그 밖에도 연출을 무라세 켄이 담당하는 이외는 같은 제작진으로 구성되어 있다. 스토리는 "밧카나레" 영업 종료 후 홀을 중심으로 전개하고 있다. 5월 13일까지 사이에 50만의 조회수를 기록했다. <a href="/wiki/2007%EB%85%84_12%EC%9B%94" title="2007년 12월">2007년 12월</a>에 배달 완료.</p>
