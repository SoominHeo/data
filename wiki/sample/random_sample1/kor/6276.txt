<p><br/>
<b>배철수의 음악캠프</b>는 <a href="/wiki/%EB%8C%80%ED%95%9C%EB%AF%BC%EA%B5%AD" title="대한민국">대한민국</a>의 FM 음악전문 라디오 방송채널인 <a href="/wiki/MBC_FM4U" title="MBC FM4U">MBC FM4U</a>의 프로그램으로, <a href="/wiki/%EB%B0%B0%EC%B2%A0%EC%88%98" title="배철수">배철수</a>가 진행하며 매일 오후 6시부터 오후 8시까지 전국에서 방송된다. 줄여서 <b>배ㆍ캠</b>이라고도 한다. <a href="/wiki/1990%EB%85%84" title="1990년">1990년</a> <a href="/wiki/3%EC%9B%94_19%EC%9D%BC" title="3월 19일">3월 19일</a>부터 방송을 시작했으며, <a class="mw-redirect" href="/wiki/%EC%98%81%EB%AF%B8%EA%B6%8C" title="영미권">영미권</a>의 <a href="/wiki/%EB%8C%80%EC%A4%91%EC%9D%8C%EC%95%85" title="대중음악">팝</a> 음악을 전문적으로 다룬다.</p>
<p>화요일<sup class="reference" id="cite_ref-2"><a href="#cite_note-2">[2]</a></sup>과 주말<sup class="reference" id="cite_ref-3"><a href="#cite_note-3">[3]</a></sup>을 제외한 요일은 생방송으로 진행한다.</p>

<h2><span class="mw-headline" id=".EC.97.AD.EC.82.AC">역사</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%B0%B0%EC%B2%A0%EC%88%98%EC%9D%98_%EC%9D%8C%EC%95%85%EC%BA%A0%ED%94%84&amp;action=edit&amp;section=1" title="부분 편집: 역사">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>배철수의 음악캠프의 전신은 <a href="/wiki/1983%EB%85%84" title="1983년">1983년</a>부터 <b>젊음의 음악캠프</b>라는 이름으로 시작된 MBC 라디오 방송이다. <a href="/wiki/%EA%B0%95%EC%84%9D" title="강석">강석</a>과 <a href="/wiki/%EC%86%90%EC%84%9D%ED%9D%AC" title="손석희">손석희</a>, <a href="/wiki/%EC%9D%B4%EC%88%98%EB%A7%8C" title="이수만">이수만</a> 등이 차례로 진행하였다. <a href="/wiki/%EB%B0%B0%EC%B2%A0%EC%88%98" title="배철수">배철수</a>는 <a href="/wiki/1990%EB%85%84" title="1990년">1990년</a> <a href="/wiki/3%EC%9B%94_19%EC%9D%BC" title="3월 19일">3월 19일</a>부터 첫 방송을 시작하였다. 그 당시에는 지금과 다른 저녁 8시부터 10시까지 방송을 했었다. 그러다가 1992년 MBC의 가을 개편으로 저녁 7시부터 9시까지 방송하다가 1995년 10월 9일 가을 개편때부터 지금과 같은 시간대로 변경되어 오늘에 이르게 되었다. <a href="/wiki/2009%EB%85%84" title="2009년">2009년</a>에는 방송횟수 7000회를 기념하는 음반이 발매되었으며, <a href="/wiki/2010%EB%85%84" title="2010년">2010년</a> <a href="/wiki/10%EC%9B%94_29%EC%9D%BC" title="10월 29일">10월 29일</a>에는 이 프로그램의 DJ인 배철수가 진행 20주년을 맞이하여 MBC 골든마우스상을 수상했다.</p>
<p><a href="/wiki/2017%EB%85%84" title="2017년">2017년</a> 중에는 방송 10000회를 맞게 될 예정이다.</p>

<h2><span class="mw-headline" id=".EA.B0.81.EC.A3.BC">각주</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%B0%B0%EC%B2%A0%EC%88%98%EC%9D%98_%EC%9D%8C%EC%95%85%EC%BA%A0%ED%94%84&amp;action=edit&amp;section=20" title="부분 편집: 각주">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<ol class="references">
<li id="cite_note-1"><span class="mw-cite-backlink"><a href="#cite_ref-1">↑</a></span> <span class="reference-text">단, 〈사람과 음악〉 코너가 방송되는 금요일에는 분야인 팝 음악과 상관없이 모든 장르를 통틀어서 음악을 선곡하는 편이다.</span></li>
<li id="cite_note-2"><span class="mw-cite-backlink"><a href="#cite_ref-2">↑</a></span> <span class="reference-text"><a href="/wiki/%EC%BD%98%EC%84%9C%ED%8A%B8_7080" title="콘서트 7080">콘서트 7080</a> 녹화 일정과 겹치므로 녹음방송으로 진행. 그러나 휴가로 인해 다른 DJ가 대타로 진행하거나 방송사의 편성 변경으로 인해 녹화일정이 취소될 경우 생방송.</span></li>
<li id="cite_note-3"><span class="mw-cite-backlink"><a href="#cite_ref-3">↑</a></span> <span class="reference-text"><a href="/wiki/2009%EB%85%84" title="2009년">2009년</a> <a href="/wiki/5%EC%9B%94_24%EC%9D%BC" title="5월 24일">5월 24일</a> 일요일 방송분에서 <a href="/wiki/%EB%85%B8%EB%AC%B4%ED%98%84" title="노무현">노무현</a> 전 대통령의 서거로 분위기를 가라앉히기 위해 1부만 생방송을 했었다. DJ 배철수는 "사실 어제는 녹음방송이었습니다. 솔직히 말씀드리면 생방하러 나올 수도 있었습니다 어제. 나오려고 하다가 '나가서 무슨 얘기를 할 것인가 방송에서...' 그래서 안 나왔습니다. 나왔어야 되었는 게 맞나요?" 라고 당일 방송에서 멘트를 하였다. 또한 <a href="/wiki/2014%EB%85%84" title="2014년">2014년</a> <a href="/wiki/9%EC%9B%94_14%EC%9D%BC" title="9월 14일">9월 14일</a>에도 생방송으로 진행하였다.</span></li>
</ol>
