<h1 class="firstHeading" id="firstHeading" lang="ko">배터리 (소설)</h1>
<div class="mw-body-content" id="bodyContent">
<div id="siteSub">위키백과, 우리 모두의 백과사전.</div>
<div id="contentSub"></div>

<div class="mw-content-ltr" dir="ltr" id="mw-content-text" lang="ko"><script>function mfTempOpenSection(id){var block=document.getElementById("mf-section-"+id);block.className+=" open-block";block.previousSibling.className+=" open-block";}</script><p>《<b>배터리</b>》 (<span lang="ja" xml:lang="ja">バッテリー</span>)는 <a class="new" href="/w/index.php?title=%EC%95%84%EC%82%AC%EB%85%B8_%EC%95%84%EC%93%B0%EC%BD%94&amp;action=edit&amp;redlink=1" title="아사노 아쓰코 (없는 문서)">아사노 아쓰코</a>의 <a href="/wiki/%EC%95%84%EB%8F%99_%EB%AC%B8%ED%95%99" title="아동 문학">아동 문학</a>이다.</p>
<p><a href="/wiki/2000%EB%85%84" title="2000년">2000년</a>에는 <a href="/wiki/NHK-FM%EB%B0%A9%EC%86%A1" title="NHK-FM방송">NHK-FM방송</a>에서 <a href="/wiki/%EB%9D%BC%EB%94%94%EC%98%A4_%EB%93%9C%EB%9D%BC%EB%A7%88" title="라디오 드라마">라디오 드라마</a>(총10화)로 만들어졌으며, <a href="/wiki/2007%EB%85%84" title="2007년">2007년</a>에는 <a class="mw-redirect" href="/wiki/%ED%95%98%EC%95%BC%EC%8B%9C_%EA%B2%90%ED%86%A0" title="하야시 겐토">하야시 겐토</a>룰 주연으로 <a href="/wiki/%EC%98%81%ED%99%94" title="영화">영화</a>, <a href="/wiki/2008%EB%85%84" title="2008년">2008년</a> 4월에는 <a href="/wiki/%EB%82%98%EC%B9%B4%EC%95%BC%EB%A7%88_%EC%9C%A0%EB%A7%88" title="나카야마 유마">나카야마 유마</a>룰 주연으로 NHK에서 <a href="/wiki/%ED%85%94%EB%A0%88%EB%B9%84%EC%A0%84_%EB%93%9C%EB%9D%BC%EB%A7%88" title="텔레비전 드라마">텔레비전 드라마</a>로 만들어졌다.</p>

<h2><span class="mw-headline" id=".EA.B0.9C.EC.9A.94">개요</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%B0%B0%ED%84%B0%EB%A6%AC_(%EC%86%8C%EC%84%A4)&amp;action=edit&amp;section=1" title="부분 편집: 개요">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>뛰어난 재능과 거만할 정도의 자신감을 가진 <a href="/wiki/%ED%88%AC%EC%88%98" title="투수">투수</a> 하라다 다쿠미와 <a href="/wiki/%EB%B0%B0%ED%84%B0%EB%A6%AC_(%EC%95%BC%EA%B5%AC)" title="배터리 (야구)">배터리</a>를 이루는 <a href="/wiki/%ED%8F%AC%EC%88%98" title="포수">포수</a> 나가쿠라 고의 1년간의 이야기이다. 소년들의 진지한 자세와 아름다운 배경 묘사가 매력이다.</p>
