<p>《<b>배틀스타 갈락티카</b>》(Battlestar Galactica, 배틀스타 갤럭티카)는 <a href="/wiki/1978%EB%85%84" title="1978년">1978년</a>과 <a href="/wiki/2003%EB%85%84" title="2003년">2003년</a>부터 <a href="/wiki/2009%EB%85%84" title="2009년">2009년</a>사이에 <a href="/wiki/%EB%AF%B8%EA%B5%AD" title="미국">미국</a>에서 방영된 <a class="mw-redirect" href="/wiki/%EA%B3%BC%ED%95%99_%EC%86%8C%EC%84%A4" title="과학 소설">SF</a> 시리즈이다. 오리지널 시리즈는 <a href="/wiki/1978%EB%85%84" title="1978년">1978년</a>에 영화와 TV 시리즈로 방영되었고, 리메이크 시리즈는 <a href="/wiki/2003%EB%85%84" title="2003년">2003년</a> 12월 8일에 <a class="new" href="/w/index.php?title=Syfy_%EC%B1%84%EB%84%90&amp;action=edit&amp;redlink=1" title="Syfy 채널 (없는 문서)">Syfy</a>를 통해 미국에서 총 3시간 분량으로 미니시리즈가 방송되었고, <a href="/wiki/2004%EB%85%84" title="2004년">2004년</a> 10월 <a href="/wiki/%EC%98%81%EA%B5%AD" title="영국">영국</a>에서 TV 시리즈의 방송을 시작했으며, 미국에서는 <a href="/wiki/2005%EB%85%84" title="2005년">2005년</a> 1월부터 방영되어 <a href="/wiki/2009%EB%85%84" title="2009년">2009년</a> <a href="/wiki/3%EC%9B%94" title="3월">3월</a>에 종영했다.</p>
<p>또한 배틀스타 갈락티카는 TV 시리즈 내에서 <a class="mw-disambig" href="/wiki/%EC%82%AC%EC%9D%BC%EB%9F%B0" title="사일런">사일런</a>과의 전쟁에 등장하는 카프리카의 우주전함이다.</p>
