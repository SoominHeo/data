<div class="mw-content-ltr" dir="ltr" id="mw-content-text" lang="ko"><script>function mfTempOpenSection(id){var block=document.getElementById("mf-section-"+id);block.className+=" open-block";block.previousSibling.className+=" open-block";}</script><p>《<a href="/wiki/%EB%B0%B0%ED%8B%80%EC%8A%A4%ED%83%80_%EA%B0%88%EB%9D%BD%ED%8B%B0%EC%B9%B4_(2004%EB%85%84_%EB%93%9C%EB%9D%BC%EB%A7%88)" title="배틀스타 갈락티카 (2004년 드라마)">배틀스타 갈락티카</a>》는 <a href="/wiki/%EB%AF%B8%EA%B5%AD" title="미국">미국</a> <a class="new" href="/w/index.php?title=Sci-Fi_%EC%B1%84%EB%84%90&amp;action=edit&amp;redlink=1" title="Sci-Fi 채널 (없는 문서)">Sci-Fi 채널</a>에서 방송된 <a class="new" href="/w/index.php?title=%EA%B5%B0%EC%82%AC_%EA%B3%B5%EC%83%81_%EA%B3%BC%ED%95%99&amp;action=edit&amp;redlink=1" title="군사 공상 과학 (없는 문서)">공상 과학</a> 장르의 <a class="mw-redirect" href="/wiki/TV_%EB%93%9C%EB%9D%BC%EB%A7%88" title="TV 드라마">TV 드라마</a>이다. <a class="new" href="/w/index.php?title=%EB%A1%9C%EB%82%A0%EB%93%9C_D._%EB%AC%B4%EC%96%B4&amp;action=edit&amp;redlink=1" title="로날드 D. 무어 (없는 문서)">로날드 D. 무어</a>와 <a class="new" href="/w/index.php?title=%EB%B0%B0%ED%8B%80%EC%8A%A4%ED%83%80_%EA%B0%88%EB%9D%BD%ED%8B%B0%EC%B9%B4_(1978_TV_%EC%8B%9C%EB%A6%AC%EC%A6%88&amp;action=edit&amp;redlink=1" title="배틀스타 갈락티카 (1978 TV 시리즈 (없는 문서)">원작 배틀스타 갈락티카</a>의 기획자인 <a class="new" href="/w/index.php?title=%EA%B8%80%EB%A0%8C_A._%EB%9D%BC%EC%8A%A8&amp;action=edit&amp;redlink=1" title="글렌 A. 라슨 (없는 문서)">글렌 A. 라슨</a>이 제작을 맡았다. 2003년 12월 세 시간 분량의 <a class="mw-redirect" href="/wiki/%EB%B0%B0%ED%8B%80%EC%8A%A4%ED%83%80_%EA%B0%88%EB%9D%BD%ED%8B%B0%EC%B9%B4_(TV_%EB%AF%B8%EB%8B%88%EC%8B%9C%EB%A6%AC%EC%A6%88)" title="배틀스타 갈락티카 (TV 미니시리즈)">미니시리즈</a>가 방송되었으며 이후 2004년부터 2009년까지 총 4시즌 75회분이 방송되었다. 첫 방송은 <a href="/wiki/2004%EB%85%84" title="2004년">2004년</a> <a href="/wiki/10%EC%9B%94_18%EC%9D%BC" title="10월 18일">10월 18일</a> 영국 <a href="/wiki/Sky1" title="Sky1">Sky1</a>에서 시작했으며 미국에서는 <a href="/wiki/2005%EB%85%84" title="2005년">2005년</a> <a href="/wiki/1%EC%9B%94_14%EC%9D%BC" title="1월 14일">1월 14일</a> Sci-Fi 채널을 통해 첫 방송이 이루어졌다.</p>
<p>배틀스타 갈락티카의 기본 줄거리는 먼 태양계에 <a class="new" href="/w/index.php?title=12_%EC%8B%9D%EB%AF%BC%EC%A7%80&amp;action=edit&amp;redlink=1" title="12 식민지 (없는 문서)">12 식민지</a>를 배경으로 한다. 각 부족의 이름을 딴 12개의 행성에서 살고있는 문명화된 인류는 그들이 만들어낸 기계 종족인 <a class="mw-disambig" href="/wiki/%EC%82%AC%EC%9D%BC%EB%9F%B0" title="사일런">사일런</a>과의 전쟁을 치른 후 휴전 상태에 있었다. <a class="mw-redirect" href="/wiki/6%ED%98%B8" title="6호">사일런 6호</a>의 꾀임에 빠진 <a class="new" href="/w/index.php?title=%EA%B0%80%EC%9D%B4%EC%9A%B0%EC%8A%A4_%EB%B0%9C%ED%83%80&amp;action=edit&amp;redlink=1" title="가이우스 발타 (없는 문서)">가이우스 발타 박사</a> 때문에 모든 식민지는 사일런의 갑작스런 핵 공격을 받아 멸망했고, 단 5만 여 명의 인류만이 살아남게 되었다. 퇴역 직전의 전함 <a class="new" href="/w/index.php?title=%EB%B0%B0%ED%8B%80%EC%8A%A4%ED%83%80_%EA%B0%88%EB%9D%BD%ED%8B%B0%EC%B9%B4_(%EC%A0%84%ED%95%A8)&amp;action=edit&amp;redlink=1" title="배틀스타 갈락티카 (전함) (없는 문서)">배틀스타 갈락티카</a>호를 이끄는 사령관 <a class="new" href="/w/index.php?title=%EC%9C%8C%EB%A6%AC%EC%97%84_%EC%95%84%EB%8B%A4%EB%A7%88&amp;action=edit&amp;redlink=1" title="윌리엄 아다마 (없는 문서)">윌리엄 아다마</a>는 내각 위원들의 죽음으로 대통령직을 승계하게된 <a class="new" href="/w/index.php?title=%EB%A1%9C%EB%9D%BC_%EB%A1%9C%EC%8A%AC%EB%A6%B0&amp;action=edit&amp;redlink=1" title="로라 로슬린 (없는 문서)">로라 로슬린</a>과 함께 살아남은 인류를 이끌고 13번째 행성으로 알려진 "지구"를 찾아나선다.</p>
