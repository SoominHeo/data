<h2><span class="mw-headline" id=".EC.A4.91.EA.B5.AD">중국</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%B0%B1_(%EC%84%B1%EC%94%A8)&amp;action=edit&amp;section=1" title="부분 편집: 중국">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>백씨(白氏)는 고대 황제 헌원(軒轅)의 후예로 중국 풍익(馮翊)에서 계출(系出)되었다고 한다.</p>
<p><b>백</b>(白)씨는 <a class="new" href="/w/index.php?title=%EC%9A%B0%EB%82%98%EB%9D%BC&amp;action=edit&amp;redlink=1" title="우나라 (없는 문서)">우(虞)나라</a>, <a class="mw-disambig" href="/wiki/%EC%B4%88%EB%82%98%EB%9D%BC" title="초나라">초나라</a> 백씨와 <a class="mw-redirect" href="/wiki/%EB%B0%B1%EC%A1%B1" title="백족">백족</a>(白族) 백씨가 있다.</p>
<ul>
<li>백씨는 <a href="/wiki/%ED%95%98%EB%82%98%EB%9D%BC" title="하나라">하나라</a> 이전에 <a class="mw-redirect" href="/wiki/%EC%97%BC%EC%A0%9C" title="염제">염제</a> 부락에 백부(白阜)라는 사람이 있었다.</li>
<li><a href="/wiki/%EC%A3%BC%EB%82%98%EB%9D%BC" title="주나라">주나라</a> 개국 이후에는 춘추시대 우(虞)나라에 집성촌이 있었다. <a href="/wiki/%EC%A7%84%EB%82%98%EB%9D%BC" title="진나라">진나라</a>를 구성하는 민족 가운데도 백족(白族)이 있었다. 《진본기》(秦本記)에 따르면, <a href="/wiki/%EA%B8%B0%EC%9B%90%EC%A0%84_6%EC%84%B8%EA%B8%B0" title="기원전 6세기">기원전 6세기</a>에 우(虞)나라 출신 <a href="/wiki/%EB%B0%B1%EB%A6%AC%ED%95%B4" title="백리해">백리해</a>(百里奚)의 자손들이 <a href="/wiki/%EC%A7%84%EB%82%98%EB%9D%BC" title="진나라">진(秦)나라</a>에 들어가 등용되었다.</li>
<li>《초한세가》(楚漢世家)에 따르면, <a class="mw-disambig" href="/wiki/%EC%B4%88%EB%82%98%EB%9D%BC" title="초나라">초나라</a> 백씨는 <a href="/wiki/%EA%B8%B0%EC%9B%90%EC%A0%84_4%EC%84%B8%EA%B8%B0" title="기원전 4세기">기원전 4세기</a>에 <a class="mw-redirect" href="/wiki/%EC%B4%88_(%EC%B6%98%EC%B6%94%EC%A0%84%EA%B5%AD)" title="초 (춘추전국)">초나라</a> 30대 혜왕(惠王)의 손자 <b>승</b>(勝)이 백공(白公)에 봉(封)해짐으로 백씨가 되었다고 한다.</li>
<li><a class="new" href="/w/index.php?title=%EC%A4%91%EA%B5%AD_%EC%84%9C%EB%B6%80&amp;action=edit&amp;redlink=1" title="중국 서부 (없는 문서)">중국 서부</a> <a class="mw-redirect" href="/wiki/%EC%9A%B4%EB%82%A8%EC%84%B1" title="운남성">운남성</a> 일대에는 <a href="/wiki/%ED%8B%B0%EB%B2%A0%ED%8A%B8" title="티베트">티베트</a>, <a href="/wiki/%EB%B2%A0%ED%8A%B8%EB%82%A8" title="베트남">베트남</a> 계통의 <a class="mw-redirect" href="/wiki/%EC%A4%91%EA%B5%AD%EC%9D%98_%EC%86%8C%EC%88%98%EB%AF%BC%EC%A1%B1" title="중국의 소수민족">소수민족</a> <a class="mw-redirect" href="/wiki/%EB%B0%B1%EC%A1%B1" title="백족">백족</a>이 있었다. <a href="/wiki/%EB%8B%B9%EB%82%98%EB%9D%BC" title="당나라">당나라</a>는 <a href="/wiki/%EB%B2%A0%ED%8A%B8%EB%82%A8" title="베트남">베트남</a>과 국경을 맞대고 있는 중국 남부 <a class="mw-redirect" href="/wiki/%EC%9A%B4%EB%82%A8%EC%84%B1" title="운남성">운남성</a>, <a class="mw-redirect" href="/wiki/%EA%B4%91%EC%84%9C%EC%84%B1" title="광서성">광서성</a>(廣西省)에 백주(白州)를 설치하였다. 백족(白族)들은 한족과 교류하게 되면서 백(白)이라는 성을 사용한 것으로 전한다. 백족도 종류가 많고, 현재는 200만 명이지만, 고대로부터 한쪽 성씨를 다양하게 사용하여 실제로는 더 많다고 한다.</li>
<li>백규(白圭)라는 인물이 <a class="mw-redirect" href="/wiki/%EB%B0%B1%EC%A0%81" title="백적">백적</a> <a class="mw-redirect" href="/wiki/%EC%84%A0%EC%9A%B0%EB%B6%80" title="선우부">선우부</a>가 세운 중산국에 있었다. 출신은 정확하지 않다.</li>
<li>《제왕전승성원도》를 보면, 백씨(白氏)의 시조는 <a href="/wiki/%ED%99%A9%EC%A0%9C_(%EC%A0%84%EC%84%A4)" title="황제 (전설)">황제</a>의 후손인 백을병(白乙丙)이라고 기재하고 있다.<sup class="reference" id="cite_ref-1"><a href="#cite_note-1">[1]</a></sup></li>
</ul>
<h2><span class="mw-headline" id=".ED.95.9C.EA.B5.AD">한국</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%B0%B1_(%EC%84%B1%EC%94%A8)&amp;action=edit&amp;section=2" title="부분 편집: 한국">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>백씨는 그 선조가 중국에서 동래하였다는데, 그 이전부터 백씨 성을 가진, 고조선 유민 계통의 선주 토착민들이 이미 신라나 백제에 살고 있었다고도 한다. 한반도의 모든 백씨들은 같은 선조의 후손들이라 여겨지고 있다.</p>
<h3><span class="mw-headline" id=".ED.95.9C.EA.B5.AD_.EB.B0.B1.EC.94.A8.EC.9D.98_.EC.97.B0.EC.9B.90">한국 백씨의 연원</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%B0%B1_(%EC%84%B1%EC%94%A8)&amp;action=edit&amp;section=3" title="부분 편집: 한국 백씨의 연원">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<ul>
<li><b>&lt;<a class="new" href="/w/index.php?title=%EB%B0%B1%EC%94%A8_(%EC%8B%A0%EB%9D%BC)&amp;action=edit&amp;redlink=1" title="백씨 (신라) (없는 문서)">신라계</a>설&gt;</b> <a href="/wiki/%EC%8B%A0%EB%9D%BC" title="신라">신라</a> 초기부터 <a class="new" href="/w/index.php?title=%EB%B0%B1%EC%94%A8_(%EC%8B%A0%EB%9D%BC)&amp;action=edit&amp;redlink=1" title="백씨 (신라) (없는 문서)">백씨</a>가 있었다고 한다. <a href="/wiki/%EA%B2%BD%EC%A3%BC_%EC%84%A4%EC%94%A8" title="경주 설씨">계림 설씨</a>(鷄林, 慶州薛氏) 시조인 설거백(薛居伯)의 현손으로 신라 7대 일성왕(逸聖王, 재위 <a href="/wiki/134%EB%85%84" title="134년">134년</a> ∼ <a href="/wiki/154%EB%85%84" title="154년">154년</a>) 때 태장령(台藏令) 설희천(薛喜天)의 부인이 <a class="new" href="/w/index.php?title=%EC%9B%94%EC%82%B0_%EB%B0%B1%EC%94%A8&amp;action=edit&amp;redlink=1" title="월산 백씨 (없는 문서)">월산 백씨</a>(月山白氏), 희천의 증손으로 12대 첨해왕(沾解王, <a href="/wiki/247%EB%85%84" title="247년">247년</a> ~ <a href="/wiki/261%EB%85%84" title="261년">261년</a>) 때 대사도(大司徒) 설명상(薛明尙)의 부인이 <a class="new" href="/w/index.php?title=%ED%99%94%EC%82%B0_%EB%B0%B1%EC%94%A8&amp;action=edit&amp;redlink=1" title="화산 백씨 (없는 문서)">화산 백씨</a>(華山白氏)라는 등 설씨 상계의 배위(配位)로 여러 백씨부인(白氏夫人)들이 기록되어 있다.<sup class="reference" id="cite_ref-.EC.84.A4.EC.94.A8.EC.A1.B1.EB.B3.B4_2-0"><a href="#cite_note-.EC.84.A4.EC.94.A8.EC.A1.B1.EB.B3.B4-2">[2]</a></sup>
<ul>
<li><a class="new" href="/w/index.php?title=%EB%B0%B1%EB%A3%A1_(%EC%8B%A0%EB%9D%BC)&amp;action=edit&amp;redlink=1" title="백룡 (신라) (없는 문서)">백룡</a>(白龍) 629년(진평왕 51) 신라의 파진찬으로 있었으나, 백씨인지는 확실치 않다.</li>

<li><a href="/wiki/%EB%B0%B1%EB%A7%A4%EC%88%9C" title="백매순">백매순</a>(白昧淳)은 백제의 노반박사(鑪盤博士)로 <a href="/wiki/588%EB%85%84" title="588년">588년</a>(<a href="/wiki/%EB%B0%B1%EC%A0%9C" title="백제">백제</a> <a href="/wiki/%EC%9C%84%EB%8D%95%EC%99%95" title="위덕왕">위덕왕</a> 35) 사원 건축기술자로 <a href="/wiki/%EC%99%9C" title="왜">왜</a>에 파견되었다.</li>

<h3><span class="mw-headline" id=".EB.B0.B1.EC.94.A8_.EB.B3.B8.EA.B4.80_.EC.97.AD.EC.82.AC_.EA.B8.B0.EB.A1.9D">백씨 본관 역사 기록</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%B0%B1_(%EC%84%B1%EC%94%A8)&amp;action=edit&amp;section=4" title="부분 편집: 백씨 본관 역사 기록">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p><a class="mw-redirect" href="/wiki/%EB%B0%B1%EC%94%A8" title="백씨">백씨</a>는 <a href="/wiki/%EC%A6%9D%EB%B3%B4%EB%AC%B8%ED%97%8C%EB%B9%84%EA%B3%A0" title="증보문헌비고">증보문헌비고</a>에 157개 본관이 전하나, 고려 초부터 말까지 사료로 확인된 본관들과 그 전거는 다음과 같다.</p>

<h2><span class="mw-headline" id=".EC.88.98.EC.9B.90_.EB.B3.B8.EA.B4.80_.ED.86.B5.ED.95.A9">수원 본관 통합</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%B0%B1_(%EC%84%B1%EC%94%A8)&amp;action=edit&amp;section=6" title="부분 편집: 수원 본관 통합">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p><a class="mw-redirect" href="/wiki/%EB%B0%B1%EC%94%A8" title="백씨">백씨</a>는 <a href="/wiki/%EC%88%98%EC%9B%90_%EB%B0%B1%EC%94%A8" title="수원 백씨">수원</a> 단본으로 알려져 있다. <a href="/wiki/%EC%88%98%EC%9B%90_%EB%B0%B1%EC%94%A8" title="수원 백씨">수원 백씨</a> 측에서는 여타의 관향이 모두 <a href="/wiki/%EC%88%98%EC%9B%90_%EB%B0%B1%EC%94%A8" title="수원 백씨">수원</a>에서 분적하였다고 주장하지만 역사적 전거가 없어 다른 본관에서는 이를 인정하지 않는다.<span class="fix" title="이 부분은 출처가 필요합니다.">1997년 <a href="/wiki/%EC%88%98%EC%9B%90_%EB%B0%B1%EC%94%A8" title="수원 백씨">수원 백씨</a> 측에서 백씨의 본관을 수원으로 통합하자고 제의하자, 나머지 33개 본관은 동의하지 않았다.</span><sup class="정리_필요 noprint">[<a href="/wiki/%EC%9C%84%ED%82%A4%EB%B0%B1%EA%B3%BC:%EC%B6%9C%EC%B2%98_%EB%B0%9D%ED%9E%88%EA%B8%B0" title="위키백과:출처 밝히기">출처 필요</a>]</sup> 역사 기록상 <a class="mw-redirect" href="/wiki/%EC%88%98%EC%9B%90" title="수원">수원</a>이라는 본관을 처음으로 사용한 백씨는 <a class="new" href="/w/index.php?title=%EB%B0%B1%EC%B2%9C%EC%9E%A5&amp;action=edit&amp;redlink=1" title="백천장 (없는 문서)">백천장</a>(白天藏)의 후손으로 조선초 보성현감(甫城縣監)을 지낸 백회(白薈,(繪))의 아들인 <b><a class="new" href="/w/index.php?title=%EB%B0%B1%ED%9A%A8%EC%82%BC&amp;action=edit&amp;redlink=1" title="백효삼 (없는 문서)">백효삼</a></b> (白效參)이다.<sup class="reference" id="cite_ref-35"><a href="#cite_note-35">[35]</a></sup> <a href="/wiki/%EC%A1%B0%EC%84%A0" title="조선">조선</a>조에 <a href="/wiki/%EC%88%98%EC%9B%90_%EB%B0%B1%EC%94%A8" title="수원 백씨">수원 백씨</a>가 번성하여 1850년대에 <a href="/wiki/%EC%88%98%EC%9B%90_%EB%B0%B1%EC%94%A8" title="수원 백씨">수원</a>으로 본관을 통일하려는 논의가 있기 전까지는 회(薈)의 아들인 효삼 4형제(효옹(效雍), 효연(效淵), 효약(效若)) 자손들을 <a href="/wiki/%EC%88%98%EC%9B%90_%EB%B0%B1%EC%94%A8" title="수원 백씨">수원 백씨</a>라 하였다.</p>
<ul>
<li>동조(同祖) 근계(近系)의 통합 동의
<ul>
<li><a class="new" href="/w/index.php?title=%EB%B0%B1%EC%B2%9C%EC%9E%A5&amp;action=edit&amp;redlink=1" title="백천장 (없는 문서)">백천장</a>(白天藏)의 증손 백회(白薈)의 형인 <a class="mw-disambig" href="/wiki/%EB%B0%B1%EC%9E%A5" title="백장">백장</a>(白莊)의 후손들이 장흥(長興), 장수(長水), 해미(海美), 태인(泰仁) 등을 본관으로 하다가 <a href="/wiki/%EC%88%98%EC%9B%90_%EB%B0%B1%EC%94%A8" title="수원 백씨">수원</a>으로 통합하였다.</li>
<li><a class="new" href="/w/index.php?title=%EB%B0%B1%EC%B2%9C%EC%9E%A5&amp;action=edit&amp;redlink=1" title="백천장 (없는 문서)">천장</a>의 다른 아들(전(琠), 환(環),<sup class="reference" id="cite_ref-36"><a href="#cite_note-36">[36]</a></sup> 산(珊), 황(璜), 거(琚) 등) 자손들도 덕천(德川), 평양(平壤), 김천(金泉), 정평(定平), 송화(松禾), 부녕(富寧) 등지에 산거하다가 백련(白璉) 계통의 <a href="/wiki/%EC%88%98%EC%9B%90_%EB%B0%B1%EC%94%A8" title="수원 백씨">수원</a>으로 통합하였다.</li>
<li><a class="new" href="/w/index.php?title=%EB%B0%B1%EC%B2%9C%EC%9E%A5&amp;action=edit&amp;redlink=1" title="백천장 (없는 문서)">천장</a>의 동생으로 임천(林川) 본관인 <a class="new" href="/w/index.php?title=%EB%B0%B1%EC%A7%80%EC%9E%A5&amp;action=edit&amp;redlink=1" title="백지장 (없는 문서)">백지장</a>(白地藏)의 자손들도 <a class="mw-redirect" href="/wiki/%EC%88%98%EC%9B%90" title="수원">수원 백씨</a> 동림공파로 들어갔다.</li>
</ul>
</li>
</ul>
<p>이렇게 수원 관조(貫祖) <a class="new" href="/w/index.php?title=%EB%B0%B1%EC%B2%9C%EC%9E%A5&amp;action=edit&amp;redlink=1" title="백천장 (없는 문서)">천장</a>의 선계인 <a href="/wiki/%EB%B0%B1%EC%98%81%EC%A0%95" title="백영정">백원정</a>(白元貞)의 후손들은 모두 <a class="mw-redirect" href="/wiki/%EC%88%98%EC%9B%90" title="수원">수원</a>으로 본관을 통합하였다.</p>
<ul>
<li>백씨보 상으로 장파(長派)인 대구(大邱)와 선산(善山) 본관 및 차파(次派)인 정주(定州) 본관의 백씨들도 수원관 통합에 동의하였다.</li>
</ul>
<p><br/></p>
<dl>
<dt>수원 본관 통합에 대한 이의와 반론</dt>
</dl>
<p><a href="/wiki/%EC%88%98%EC%9B%90_%EB%B0%B1%EC%94%A8" title="수원 백씨">수원 백씨</a>의 동조(同祖)인 <a href="/wiki/%EB%B0%B1%EC%98%81%EC%A0%95" title="백영정">백원정</a>(白元貞, 永貞),<sup class="reference" id="cite_ref-37"><a href="#cite_note-37">[37]</a></sup>은 1260년에 장군을 지냈는데, 백씨대동보에 따르면, 그의 선계가 고려 전기의 <a class="new" href="/w/index.php?title=%EB%B0%B1%EA%B2%BD%EC%8B%A0&amp;action=edit&amp;redlink=1" title="백경신 (없는 문서)">백경신</a>(白景臣, 1060년경 출생) 대에 <a class="mw-redirect" href="/wiki/%EC%A7%81%EC%82%B0" title="직산">직산</a>(稷山)에서 <a class="mw-redirect" href="/wiki/%EC%9E%84%EC%B2%9C" title="임천">임천</a>(林川)으로 이거하였다<sup class="reference" id="cite_ref-38"><a href="#cite_note-38">[38]</a></sup>고 하니 <a href="/wiki/%EC%88%98%EC%9B%90_%EB%B0%B1%EC%94%A8" title="수원 백씨">수원 백씨</a>는 조선 초에 <a href="/wiki/%EC%9E%84%EC%B2%9C_%EB%B0%B1%EC%94%A8" title="임천 백씨">임천</a>에서 분적한 것으로 보인다.<sup class="reference" id="cite_ref-39"><a href="#cite_note-39">[39]</a></sup> 백씨 문중에서 원정-천장 계와 가까운 여러 일족들이 수원 본관으로 통합하였지만, 고려시대 본관인 <a href="/wiki/%EC%A7%81%EC%82%B0_%EB%B0%B1%EC%94%A8" title="직산 백씨">직산</a>, <a href="/wiki/%EB%82%A8%ED%8F%AC_%EB%B0%B1%EC%94%A8" title="남포 백씨">남포</a>, <a href="/wiki/%EB%8C%80%ED%9D%A5_%EB%B0%B1%EC%94%A8" title="대흥 백씨">대흥</a>, <a href="/wiki/%EB%B6%80%EC%97%AC_%EB%B0%B1%EC%94%A8" title="부여 백씨">부여</a>, <a href="/wiki/%EC%B2%AD%EB%8F%84_%EB%B0%B1%EC%94%A8" title="청도 백씨">청도</a> 등의 본관에서는 역사적 전거가 없는 '백휘수원봉군설'(白揮水原封君說)과 '백천장수성백설'(白天藏水城伯說)을 인정하지 않아 수원 관(貫)에 동의하지 않고 있다.<sup class="reference" id="cite_ref-40"><a href="#cite_note-40">[40]</a></sup><sup class="reference" id="cite_ref-41"><a href="#cite_note-41">[41]</a></sup></p>
<p>이런 이유로 1982년에 간행된 임술대동보는 '수원'을 빼고 『백씨대동보』라 하였고, 1997년에 간행된 정축대동보는 '수원'을 넣어 『수원백씨대동보』라 하였다. 백씨 전체 인구의 90% 이상이 <a href="/wiki/%EC%88%98%EC%9B%90_%EB%B0%B1%EC%94%A8" title="수원 백씨">수원</a>을 본관으로 하고 있다.</p>
