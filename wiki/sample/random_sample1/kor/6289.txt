<p><b>백년 전쟁</b>(百年戰爭)은 <a href="/wiki/%EC%9E%89%EA%B8%80%EB%9E%9C%EB%93%9C_%EC%99%95%EA%B5%AD" title="잉글랜드 왕국">잉글랜드 왕국</a>과 <a href="/wiki/%ED%94%84%EB%9E%91%EC%8A%A4_%EC%99%95%EA%B5%AD" title="프랑스 왕국">프랑스 왕국</a> 간에 벌어진 전쟁으로서, 프랑스를 전장으로 하여 여러 차례 휴전과 전쟁을 되풀이하면서, <a href="/wiki/1337%EB%85%84" title="1337년">1337년</a>부터 <a href="/wiki/1453%EB%85%84" title="1453년">1453년</a>까지 116년 동안 계속되었다. 명분은 프랑스 왕위 계승 문제였고, 실제 원인은 영토 문제였다. 백년 전쟁은 <a href="/wiki/1360%EB%85%84" title="1360년">1360년</a>의 <a class="new" href="/w/index.php?title=%EB%B8%8C%EB%A0%88%ED%8B%B0%EB%8B%88-%EC%B9%BC%EB%A0%88_%EC%A1%B0%EC%95%BD&amp;action=edit&amp;redlink=1" title="브레티니-칼레 조약 (없는 문서)">브레티니-칼레 조약</a>의 체결까지를 제1기, <a href="/wiki/1415%EB%85%84" title="1415년">1415년</a>의 <a href="/wiki/%EC%95%84%EC%9F%81%EC%BF%A0%EB%A5%B4_%EC%A0%84%ED%88%AC" title="아쟁쿠르 전투">아쟁쿠르 전투</a> 또는 <a href="/wiki/1420%EB%85%84" title="1420년">1420년</a>의 <a class="new" href="/w/index.php?title=%ED%8A%B8%EB%A3%A8%EC%95%84_%EC%A1%B0%EC%95%BD&amp;action=edit&amp;redlink=1" title="트루아 조약 (없는 문서)">트루아 조약</a>의 전과 후를 제2기·제3기로 나눈다. 세계에서 2번째로 긴 전쟁이다.</p>

<h2><span class="mw-headline" id=".EC.9B.90.EC.9D.B8">원인</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%B0%B1%EB%85%84_%EC%A0%84%EC%9F%81&amp;action=edit&amp;section=1" title="부분 편집: 원인">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>잉글랜드는 <a class="mw-redirect" href="/wiki/%EB%85%B8%EB%A5%B4%EB%A7%8C_%EC%99%95%EC%A1%B0" title="노르만 왕조">노르만 왕조</a>의 성립 이후 프랑스의 일부 영토를 소유하였는데 이 때문에 양국 사이에는 분쟁이 계속되었다. 13세기 말에 이르러서는 잉글랜드 국왕의 프랑스 내 영토가 프랑스 국왕보다 더 많은 지경에 이르렀다. 그러나 중세 <a class="mw-redirect" href="/wiki/%EB%B4%89%EA%B1%B4%EC%A0%9C%EB%8F%84" title="봉건제도">봉건제도</a> 하에서 잉글랜드 국왕은 잉글랜드 왕국의 군주이면서 동시에 프랑스 국왕의 신하라는 이중 지위를 갖고 있었다.</p>
<p>상황이 이렇게 된 것은 중세 봉건 제도의 특징상, 결혼을 하게 되면 여자가 남자에게 자신의 봉토를 결혼 지참금으로 넘겼기 때문이었다. 노르만 왕조 성립 이후 잉글랜드 국왕은 역시 애초 프랑스 국왕의 봉신이었던 <a href="/wiki/%EB%85%B8%EB%A5%B4%EB%A7%9D%EB%94%94" title="노르망디">노르망디</a> 공국의 영주였고, 노르만 왕조의 뒤를 이은 <a class="mw-redirect" href="/wiki/%ED%94%8C%EB%9E%9C%ED%83%9C%EC%A0%80%EB%84%B7_%EC%99%95%EA%B0%80" title="플랜태저넷 왕가">플랜태저넷 왕가</a>(<a href="/wiki/1154%EB%85%84" title="1154년">1154년</a> ~ <a href="/wiki/1399%EB%85%84" title="1399년">1399년</a>) 역시 본래 프랑스의 앙주 백이었다. 플랜태저넷 왕조는 잉글랜드 국왕으로서 노르망디도 당연히 계승하게 되었고, 이렇게 되자 프랑스 내에서 잉글랜드 왕의 입김은 프랑스 국왕보다 더욱 셌지만, 법률상으로는 잉글랜드 국왕은 프랑스 국왕의 신하였다. 이 무렵 프랑스의 <a href="/wiki/%EC%B9%B4%ED%8E%98_%EC%99%95%EC%A1%B0" title="카페 왕조">카페 왕조</a>의 영향력은 현재의 <a class="mw-redirect" href="/wiki/%ED%8C%8C%EB%A6%AC_%EC%8B%9C" title="파리 시">파리 시</a>를 중심으로 한 <a class="mw-redirect" href="/wiki/%EC%9D%BC-%EB%93%9C-%ED%94%84%EB%9E%91%EC%8A%A4" title="일-드-프랑스">일-드-프랑스</a> 지역에 국한되어 있었다. 잉글랜드 국왕의 입장에서는 당연히 불만이 없을 수가 없었다.</p>
<p><a href="/wiki/1328%EB%85%84" title="1328년">1328년</a> 프랑스 카페 왕조의 <a href="/wiki/%EC%83%A4%EB%A5%BC_4%EC%84%B8" title="샤를 4세">샤를 4세</a>가 남자 후계자 없이 사망하자, 그의 4촌 형제인 <a class="mw-redirect" href="/wiki/%EB%B0%9C%EB%A3%A8%EC%95%84_%EC%99%95%EA%B0%80" title="발루아 왕가">발루아 왕가</a>의 <a href="/wiki/%ED%95%84%EB%A6%AC%ED%94%84_6%EC%84%B8" title="필리프 6세">필리프 6세</a>(재위:<a href="/wiki/1328%EB%85%84" title="1328년">1328년</a> ~ <a href="/wiki/1350%EB%85%84" title="1350년">1350년</a>)가 왕위에 올랐다. 그러나 여자가 직접 왕위 계승이 불가능하다 하더라도 만일 그녀의 아들에게 계승시킬 수 있다면 잉글랜드 국왕 <a class="mw-redirect" href="/wiki/%EC%9E%89%EA%B8%80%EB%9E%9C%EB%93%9C%EC%9D%98_%EC%97%90%EB%93%9C%EC%9B%8C%EB%93%9C_2%EC%84%B8" title="잉글랜드의 에드워드 2세">에드워드 2세</a>의 왕비 이사벨라(마지막 카페 왕조의 국왕이었던 샤를 4세의 누이)의 아들인 <a href="/wiki/%EC%97%90%EB%93%9C%EC%9B%8C%EB%93%9C_3%EC%84%B8" title="에드워드 3세">에드워드 3세</a>(재위:<a href="/wiki/1327%EB%85%84" title="1327년">1327년</a>~<a href="/wiki/1377%EB%85%84" title="1377년">1377년</a>)가 왕위 계승자가 된다는 주장도 성립되었다. 이것을 핑계로 삼아 잉글랜드 국왕 에드워드 3세는 프랑스 왕위를 자신이 계승해야 한다고 주장하여, 양국간에 심각한 대립을 빚게 되었다. 이 참에 잉글랜드와 프랑스를 하나의 거대한 왕국으로 통합할 욕심을 갖게 된 것이다.</p>
<p>필리프 6세의 입장에서도 자신의 신하 중에 자신을 능가하는 세력이 있다는 것은 결코 좋은 일이 아니었다. 게다가 그는 다른 나라의 국왕이기도 했다. 잉글랜드 국왕이 프랑스 국왕의 신하로서 가지고 있는 봉토를 몰수하여 프랑스 국왕의 위신을 높여야 할 필요성과 욕심을 가지고 있었다. 잉글랜드 국왕의 왕위계승권 주장도 참을 수 없는 일이었다. 영토 문제와 왕위계승권 문제로 인한 두 왕가의 갈등은 대화로 풀 수 있는 상황이 아니었다.</p>
<h2><span class="mw-headline" id=".EA.B2.BD.EA.B3.BC">경과</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%B0%B1%EB%85%84_%EC%A0%84%EC%9F%81&amp;action=edit&amp;section=2" title="부분 편집: 경과">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<h3><span class="mw-headline" id=".EC.8B.9C.EC.9E.91">시작</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%B0%B1%EB%85%84_%EC%A0%84%EC%9F%81&amp;action=edit&amp;section=3" title="부분 편집: 시작">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>잉글랜드를 견제할 세력이 필요했던 필리프 6세는 잉글랜드에 쫓겨 산악지대로 들어갔던 <a href="/wiki/%EC%8A%A4%EC%BD%94%ED%8B%80%EB%9E%9C%EB%93%9C_%EC%99%95%EA%B5%AD" title="스코틀랜드 왕국">스코틀랜드 왕국</a>에 주목했다. 당시 잉글랜드는 스코틀랜드를 정복하기 위해 많은 노력을 기울이는 중이었다. 또, 필리프 6세는 에드워드 3세의 신종서약에 트집을 잡고 잉글랜드 국왕의 봉토였던 아키텐령을 무력으로 점령하고 <a href="/wiki/1337%EB%85%84" title="1337년">1337년</a>에는 아키텐령의 몰수를 선언하였다. 또 노르망디 해안에는 함대를 보내 잉글랜드를 위협했다.</p>
<p>상황이 이렇게 되자 <a href="/wiki/1336%EB%85%84" title="1336년">1336년</a>, 에드워드 3세는 필리프 6세에게 공식적인 도전장을 띄우고, 전쟁 준비에 들어갔다. 도전장은 제목부터 ＜발루아 가문의 필리프, 자칭 프랑스 왕＞이라는 도발적인 것이었다. 전쟁 준비는 자체 군사력을 준비하는 것 말고도 외교전도 치열하게 펼쳤다. 국가간 외교라기보다는 왕가간 외교였는데, <a href="/wiki/%EC%8B%A0%EC%84%B1_%EB%A1%9C%EB%A7%88_%EC%A0%9C%EA%B5%AD" title="신성 로마 제국">신성 로마 제국</a> 황제와 <a href="/wiki/%EB%9D%BC%EC%9D%B8_%EA%B0%95" title="라인 강">라인 강</a> 및 <a href="/wiki/%EB%84%A4%EB%8D%9C%EB%9E%80%EB%93%9C" title="네덜란드">네덜란드</a> 제후와 절충을 거듭하여 ＜황제대관＞의 지위와 공수동맹의 약속을 성립시켰다. 그의 외교의 핵심은 <a href="/wiki/%ED%94%8C%EB%9E%91%EB%93%9C%EB%A5%B4" title="플랑드르">플랑드르</a> 지방이었다.</p>
<p>원래 <a href="/wiki/%ED%94%8C%EB%9E%91%EB%93%9C%EB%A5%B4" title="플랑드르">플랑드르</a>는 프랑스 왕의 종주권 아래에 있었지만, 중세를 통하여 유럽 최대의 <a class="new" href="/w/index.php?title=%EB%AA%A8%EC%A7%81%EB%AC%BC&amp;action=edit&amp;redlink=1" title="모직물 (없는 문서)">모직물</a> 공업지대로서 번창하여, 원료인 <a href="/wiki/%EC%96%91%EB%AA%A8" title="양모">양모</a>의 최대 공급국인 잉글랜드가 이 지방을 경제적으로 지배하고 있었다. 이 때문에 전쟁 중 플랑드르는 잉글랜드 편에 섰다. 잉글랜드는 플랑드르 지방 도시들에 대해 새로운 종주권을 주장했으며, 그들과 동맹을 맺었다. 에드워드 7세가 구축한 이른바 북방의 환상동맹에 대해 필리프 6세는 <a href="/wiki/%EC%A0%9C%EB%84%A4%EB%B0%94" title="제네바">제네바</a>와 <a href="/wiki/%EC%B9%B4%EC%8A%A4%ED%8B%B0%EC%95%BC" title="카스티야">카스티야</a>의 지원을 받았다.</p>
<p>아키텐령에 속한 <a href="/wiki/%EA%B0%80%EC%8A%A4%EC%BD%94%EB%89%B4" title="가스코뉴">가스코뉴</a> 역시 유럽 최대의 <a href="/wiki/%ED%8F%AC%EB%8F%84%EC%A3%BC" title="포도주">포도주</a> 생산지였으므로, 프랑스 왕들은 항상 이 두 지방의 탈환을 바라고 있었다. 따라서 전쟁은 이 두 지방의 쟁탈을 목표로 하게 되었다. 왕위 계승 문제는 늘 그렇듯이 명분에 불과했다. 이것의 백년전쟁의 첫 시작이었다.</p>
<h3><span class="mw-headline" id=".EC.A0.9C1.EA.B8.B0">제1기</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%B0%B1%EB%85%84_%EC%A0%84%EC%9F%81&amp;action=edit&amp;section=4" title="부분 편집: 제1기">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>백년 전쟁은 1340년 6월에 잉글랜드-플랑드르 연합함대가 라인 강의 하구에 있는 브뤼주(지금의 <a href="/wiki/%EB%B8%8C%EB%A4%BC%ED%97%88" title="브뤼허">브뤼허</a>) 슬뢰이스항에 대한 프랑스 함대의 봉쇄를 저지하기 위해 공격을 시작하면서 시작되었다. 이 전투에서 잉글랜드군이 승리하였고, 이후 <a href="/wiki/%EB%8F%84%EB%B2%84_%ED%95%B4%ED%98%91" title="도버 해협">도버 해협</a>의 제해권은 잉글랜드가 차지하게 되었다. 116년이라는 기간 동안 프랑스만 전쟁터가 된 것도 이 때문이다.</p>
<p><a href="/wiki/1345%EB%85%84" title="1345년">1345년</a> <a href="/wiki/%EC%97%90%EB%93%9C%EC%9B%8C%EB%93%9C_3%EC%84%B8" title="에드워드 3세">에드워드 3세</a>는 그의 맏아들인 <a href="/wiki/%ED%9D%91%ED%83%9C%EC%9E%90_%EC%97%90%EB%93%9C%EC%9B%8C%EB%93%9C" title="흑태자 에드워드">흑태자 에드워드</a>와 함께 <a href="/wiki/%EB%85%B8%EB%A5%B4%EB%A7%9D%EB%94%94" title="노르망디">노르망디</a>에 상륙하였다. 이듬해인 <a href="/wiki/1346%EB%85%84" title="1346년">1346년</a>에 유명한 <a href="/wiki/%ED%81%AC%EB%A0%88%EC%8B%9C_%EC%A0%84%ED%88%AC" title="크레시 전투">크레시 전투</a>가 있었다. 이 전투에서 잉글랜드군은 농민들과 사냥꾼들로 구성된 보병 <a href="/wiki/%EC%9E%A5%EA%B6%81" title="장궁">장궁</a>대를 내세워 전력이 훨씬 우세한 프랑스 기사군을 격파하였다. 보병 장궁대의 기사군 격퇴는 <a class="new" href="/w/index.php?title=%ED%99%94%EC%95%BD_%EB%AC%B4%EA%B8%B0&amp;action=edit&amp;redlink=1" title="화약 무기 (없는 문서)">화약 무기</a>의 등장 이전에 중장기병의 몰락을 예고한 것으로 평가되고 있다. 이 전투에서 프랑스 기사군은 익숙한 전법으로 일제히 돌격을 시도하였으나 진흙탕을 거치면서 기병의 특유한 충격력을 상실한 상태에서 장궁대의 공격을 받은 것이다.</p>
<p>이때 잉글랜드군이 동원한 장궁은 그 이전에 <a href="/wiki/%EC%84%9D%EA%B6%81" title="석궁">석궁</a>에 비해서 훨씬 사거리 및 위력 뿐만이 아니라 연사력이 더 뛰어난 것이었다. 이 장궁은 본래 <a href="/wiki/%EC%9B%A8%EC%9D%BC%EC%8A%A4" title="웨일스">웨일스</a> 지방의 사냥꾼들이 사용하던 것이었으나 영국 왕실은 그 위력을 알아보고 전쟁에 동원한 것이다. 반면 프랑스는 석궁병들을 배치했으나, 많은 전투에서 석궁병들은 영국 장궁병에게 패배하였다.</p>
<p>잉글랜드군은 크레시 전투에 승리한 그 여세를 몰아 <a href="/wiki/%EC%B9%BC%EB%A0%88" title="칼레">칼레</a> 시(市)로 진격하여 이 성을 완전히 포위하였다. 칼레 시민들은 완강하게 계속 저항하였으나, <a href="/wiki/1347%EB%85%84" title="1347년">1347년</a> 마침내 잉글랜드군에게 항복하였다(<a class="new" href="/w/index.php?title=%EC%B9%BC%EB%A0%88_%ED%8F%AC%EC%9C%84%EC%A0%84&amp;action=edit&amp;redlink=1" title="칼레 포위전 (없는 문서)">칼레 포위전</a>). 칼레 지구가 잉글랜드령으로 확정되면서 에드워드의 전략은 앙주-플랜태저넷 왕가의 옛 대륙 영토 전 지역의 회복을 겨냥한 것이었다.</p>
<p>그 뒤 양국에 페스트가 유행한 데다 재정사정도 악화되어 한때 전쟁이 중단되기도 하였으나, <a href="/wiki/1355%EB%85%84" title="1355년">1355년</a> 흑태자는 다시 남프랑스를 침입하였다. <a href="/wiki/1350%EB%85%84" title="1350년">1350년</a> 프랑스에서는 필리프 6세가 죽고 그 뒤를 이어 장 2세가 즉위하였으나, <a href="/wiki/1356%EB%85%84" title="1356년">1356년</a> 흑태자는 <a class="mw-disambig" href="/wiki/%EC%9E%A5_2%EC%84%B8" title="장 2세">장 2세</a>가 인솔한 프랑스군을 <a href="/wiki/%ED%91%B8%EC%95%84%ED%8B%B0%EC%97%90_%EC%A0%84%ED%88%AC" title="푸아티에 전투">푸아티에 전투</a>에서 격파하고 장 2세를 포로로 잡았다. 이처럼 전쟁 초기에 거둔 영국군의 일방적 승리는, 독립적 자영 농민을 주력으로 한 보병의 장궁대 전법이 프랑스의 봉건 기사군의 전법에 비해 우수하였기 때문이었다.</p>
<p><a href="/wiki/1359%EB%85%84" title="1359년">1359년</a> 다시 공격을 시작한 에드워드에게 프랑스는 화의를 청하였다. <a href="/wiki/1360%EB%85%84" title="1360년">1360년</a> <a href="/wiki/%EC%83%A4%EB%A5%B4%ED%8A%B8%EB%A5%B4" title="샤르트르">샤르트르</a> 근교 <a class="new" href="/w/index.php?title=%EB%B8%8C%EB%A0%88%ED%8B%B0%EB%8B%88&amp;action=edit&amp;redlink=1" title="브레티니 (없는 문서)">브레티니</a>에서 프랑스 왕가의 대표단과 협의하고 뒤에 칼레에서 푸아티에 전투에서 포로가 되었던 프랑스왕 장 2세(재위 <a href="/wiki/1350%EB%85%84" title="1350년">1350년</a> ~ <a href="/wiki/1364%EB%85%84" title="1364년">1364년</a>)의 서명을 얻어 발효한 휴전 조약이 <a class="new" href="/w/index.php?title=%EB%B8%8C%EB%A0%88%ED%8B%B0%EB%8B%88-%EC%B9%BC%EB%A0%88_%EC%A1%B0%EC%95%BD&amp;action=edit&amp;redlink=1" title="브레티니-칼레 조약 (없는 문서)">브레티니-칼레 조약</a>이다. 브레티니-칼레 조약은 푸아티에를 비롯하여 옛 앙주왕령의 잉글랜드 왕실로 귀속을 규정한 다음 제12조에서 프랑스 왕은 그 여러 영토에 대한 종주권을 행사하지 않을 것, 잉글랜드 왕은 특히 《프랑스 왕관과 왕국의 이름 및 권리에 대한 청구권》을 포기할 것을 규정하였다. 이것은 애초부터 잉글랜드 왕은 프랑스 왕관보다는 영토를 더 중요시했다는 반증이기도 하다.</p>
<p>프랑스는 프랑스대로 전쟁의 참화와 잉글랜드군의 약탈로 농민의 피폐가 격심하여, <a href="/wiki/1358%EB%85%84" title="1358년">1358년</a> 농민 반란인 <a href="/wiki/%EC%9E%90%ED%81%AC%EB%A6%AC%EC%9D%98_%EB%82%9C" title="자크리의 난">자크리의 난</a>이 일어났다. 이 반란은 즉시 진압되었지만, 프랑스 왕가를 매우 심한 궁지에 빠뜨렸다. 브레티니-칼레 조약을 맺을 수밖에 없었던 뒷배경이기도 하다. 아무튼 프랑스는 <a class="mw-disambig" href="/wiki/%EC%9E%A5_2%EC%84%B8" title="장 2세">장 2세</a>의 석방보상금으로 300만 크라운을 지불하고, 아키텐 지방 전부와 칼레 시(市) 등의 영토를 잉글랜드에게 할양하였다. 이 당시에는 적군이라 하더라도 귀족들은 몸값을 받고 석방하는 게 보통이었다. 현대에서는 전혀 상상할 수 없지만 그 당시에는 보편화된 방식이었고, 지위와 명성이 높을수록 몸값은 비싸게 매겨졌다.</p>
<p>이렇게 해서 백년전쟁의 1기는 막을 내린다.</p>
<h3><span class="mw-headline" id=".EC.A0.9C2.EA.B8.B0">제2기</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%B0%B1%EB%85%84_%EC%A0%84%EC%9F%81&amp;action=edit&amp;section=5" title="부분 편집: 제2기">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p><a href="/wiki/1364%EB%85%84" title="1364년">1364년</a> 프랑스에서는 장 2세가 죽고 샤를 5세가 즉위하였다. 그는 내정의 정비와 재정(財政)의 재건에 착수하였으며, 아키텐의 귀족들을 선동하여 잉글랜드의 지배에 반항하게 하였으므로, 양국 사이가 다시 악화되어 마침내 전쟁이 재개되었다. <a href="/wiki/1369%EB%85%84" title="1369년">1369년</a> 흑태자의 동생인 고트의 존이 이끄는 잉글랜드군이 프랑스로 침입하였으나 프랑스군의 분전으로 패배하고, 잉글랜드 해군 역시 카스티야 해군과 동맹을 맺은 프랑스 해군에게 잇달아 패전하였다. 따라서 브레티니-칼레 조약에서 잉글랜드에 할양한 영토의 대부분을 탈환하고, <a href="/wiki/1375%EB%85%84" title="1375년">1375년</a> 부르지에서 휴전 협정을 체결하였다. 이처럼 2기 초기에는 프랑스에 유리하게 진행되었다.</p>
<p>그 뒤 <a href="/wiki/1377%EB%85%84" title="1377년">1377년</a> 잉글랜드에서는 <a href="/wiki/%EC%97%90%EB%93%9C%EC%9B%8C%EB%93%9C_3%EC%84%B8" title="에드워드 3세">에드워드 3세</a>가 죽고 <a href="/wiki/%EB%A6%AC%EC%B2%98%EB%93%9C_2%EC%84%B8" title="리처드 2세">리처드 2세</a>가 왕위에 올랐으며, 프랑스에서도 <a href="/wiki/1380%EB%85%84" title="1380년">1380년</a> <a href="/wiki/%EC%83%A4%EB%A5%BC_5%EC%84%B8" title="샤를 5세">샤를 5세</a>가 죽고 그의 아들 <a href="/wiki/%EC%83%A4%EB%A5%BC_6%EC%84%B8" title="샤를 6세">샤를 6세</a>가 그 뒤를 이었다. 그러나 두 왕이 모두 미성년(未成年)이었고, 특히 잉글랜드에서는 <a href="/wiki/1381%EB%85%84" title="1381년">1381년</a> <a href="/wiki/%EC%99%80%ED%8A%B8_%ED%83%80%EC%9D%BC%EB%9F%AC%EC%9D%98_%EB%82%9C" title="와트 타일러의 난">와트 타일러의 난</a>이 일어난 데다가, 더욱이 리처드 왕의 지배에 대한 귀족의 반항까지 겹쳤기 때문에, 두 나라 사이의 전쟁은 오랫동안 중단되었다.</p>
<p>덕분에 앙주는 발루아 가계의 앙주 공령이 되어 브레티니-칼레 조약 체결을 주도했던 프랑스왕의 대리인인 샤를(뒤에 샤를 5세)에 이어 다시 1364년 이후에는 샤를 5세의 동생 루이의 소유령이 되었다. <a href="/wiki/%EB%B8%8C%EB%A5%B4%ED%83%80%EB%89%B4" title="브르타뉴">브르타뉴</a>에는 친프랑스 공가의 가계가 확립되었다. 1380년 샤를 5세가 사망한 후 샤를 6세(재위 <a href="/wiki/1380%EB%85%84" title="1380년">1380년</a> ~ <a href="/wiki/1422%EB%85%84" title="1422년">1422년</a>) 시대에 프랑스는 왕족 제후에 의한 왕정 후견 시대로 들어서게 되었다. <a href="/wiki/1404%EB%85%84" title="1404년">1404년</a> 국왕고문회의의 대표 부르고뉴공 필리프의 사후, 그 아들 장과 왕의 동생 오를레앙공 루이의 불화가 표면화하여 이른바 부르고뉴파 대 아르마냐크파 대립의 국면을 맞이하였으나 샤를 5세가 확립한 왕정 방식은 그 기본구조를 무너뜨리는 일이 없이 유지되었다.</p>
<p>국제관계에서 주목해야 할 점은 부르고뉴 공가의 입장이었다. 플랑드르를 가령으로 한 공가는 잉글랜드에 대하여 화친정책을 취하였다. 잉글랜드 왕가측에서는 플랜태저넷 왕조 최후의 왕 <a href="/wiki/%EB%A6%AC%EC%B2%98%EB%93%9C_2%EC%84%B8" title="리처드 2세">리처드 2세</a>(재위 <a href="/wiki/1377%EB%85%84" title="1377년">1377년</a> ~ <a href="/wiki/1399%EB%85%84" title="1399년">1399년</a>)가 프랑스에 대한 화친 방향을 모색하였다. 프랑스 왕의 딸 이사벨을 아내로 맞은 것도 그 일환이며 이를 주선한 사람이 바로 부르고뉴 공 필리프였다. 결국 프랑스에 대한 융화정책이 반대당파의 결성을 촉진하여 <a href="/wiki/1399%EB%85%84" title="1399년">1399년</a> 리처드 2세는 폐위되었다. 리처드 2세가 폐위된 후 랭커스터 가의 <a href="/wiki/%ED%97%A8%EB%A6%AC_4%EC%84%B8" title="헨리 4세">헨리 4세</a>가 왕위에 올라 프랑스에 대한 전쟁을 재개하였다. 프랑스에서는 종종 정신착란을 일으키는 샤를 6세를 대신하여 국정의 실권을 장악하려고 한 귀족들이 부르고뉴파와 아르마냐크파로 나뉘어 내란 상태에 빠졌다. 부르고뉴파는 왕실을 둘러싼 아르마냐크파를 견제하기 위해 잉글랜드에 접근하게 된다. 전쟁 후반기에 <a href="/wiki/%EC%9E%94_%EB%8B%A4%EB%A5%B4%ED%81%AC" title="잔 다르크">잔 다르크</a>를 잉글랜드에 팔아넘긴 것도 부르고뉴파였다.</p>
<h3><span class="mw-headline" id=".EC.A0.9C3.EA.B8.B0">제3기</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%B0%B1%EB%85%84_%EC%A0%84%EC%9F%81&amp;action=edit&amp;section=6" title="부분 편집: 제3기">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>백년전쟁의 마지막 시기로 분류되는 3기는 <a href="/wiki/1415%EB%85%84" title="1415년">1415년</a>에 시작되었다. 이 시기는 <a href="/wiki/%EC%98%A4%EB%A5%BC%EB%A0%88%EC%95%99" title="오를레앙">오를레앙</a>과 <a href="/wiki/%EC%9E%94_%EB%8B%A4%EB%A5%B4%ED%81%AC" title="잔 다르크">잔 다르크</a>로 요약될 수 있을 것이다.</p>
<p><a href="/wiki/1413%EB%85%84" title="1413년">1413년</a> 헨리 4세의 뒤를 이어 잉글랜드 국왕으로 즉위한 <a href="/wiki/%ED%97%A8%EB%A6%AC_5%EC%84%B8" title="헨리 5세">헨리 5세</a>는 프랑스의 내분을 이용하여 부르고뉴파와 결탁하고, <a href="/wiki/1415%EB%85%84" title="1415년">1415년</a> 맹렬한 기세로 노르망디를 진공, <a href="/wiki/%EC%95%84%EC%9F%81%EC%BF%A0%EB%A5%B4_%EC%A0%84%ED%88%AC" title="아쟁쿠르 전투">아쟁쿠르 전투</a>에서 압도적으로 우세한 프랑스군을 대패시켜 북프랑스의 여러 도시를 탈취하였다. 헨리 5세는 이를 배경으로 <a href="/wiki/1420%EB%85%84" title="1420년">1420년</a> <a class="new" href="/w/index.php?title=%ED%8A%B8%EB%A3%A8%EC%95%84_%EC%A1%B0%EC%95%BD&amp;action=edit&amp;redlink=1" title="트루아 조약 (없는 문서)">트루아 조약</a>을 맺고, 스스로 샤를 6세의 딸 카트린과 결혼함으로써 자신의 프랑스 왕위 계승권을 승인시켰다. 그러나 샤를 6세의 아들인 왕세자 샤를(샤를 7세)과 이를 지지하는 아르마냐크파는, 이와 같은 잉글랜드측의 강요를 인정하지 않은 채 중남부 프랑스에 거점을 두고 전쟁을 계속하였다. <a href="/wiki/1422%EB%85%84" title="1422년">1422년</a> 잉글랜드의 헨리 5세와 프랑스의 샤를 6세가 잇달아 죽자 나이 어린 <a href="/wiki/%ED%97%A8%EB%A6%AC_6%EC%84%B8" title="헨리 6세">헨리 6세</a>가 잉글랜드와 프랑스 두 나라의 국왕을 자칭하고 나섰고, 프랑스에서는 왕세자가 샤를 7세로서 프랑스의 왕위에 올랐음을 선언하였다.</p>
<p>이에 잉글랜드는 <a href="/wiki/1428%EB%85%84" title="1428년">1428년</a> 샤를 7세의 거점인 <a href="/wiki/%EC%98%A4%EB%A5%BC%EB%A0%88%EC%95%99" title="오를레앙">오를레앙</a>을 포위하였다. <a href="/wiki/%EC%83%A4%EB%A5%BC_7%EC%84%B8" title="샤를 7세">샤를 7세</a>는 궁지에 몰리게 되었다. 아직 그는 정식 국왕도 아니었다. <a href="/wiki/%EB%9E%AD%EC%8A%A4_%EB%8C%80%EC%84%B1%EB%8B%B9" title="랭스 대성당">랭스 대성당</a>에서 대관식을 거쳐야 했으나 잉글랜드군 때문에 그러지 못한 것이다. 오를레앙은 1년여를 버티고 있었지만, 희망은 별로 없어 보였다. 샤를 7세는 오를레앙을 구원할 힘도 갖고 있지 않았다. 게다가 프랑스는 전황의 불리함 때문에 병력 동원도 쉽지 않았다. 용병이 주였던 이 당시에 프랑스의 불리한 전황은 용병을 모으기 어렵게 하는 조건이었다.</p>
<p>이와 같이 불리한 전황을 승전으로 전환하게 한 것은 <a href="/wiki/%EC%9E%94_%EB%8B%A4%EB%A5%B4%ED%81%AC" title="잔 다르크">잔 다르크</a>의 출현이었다. 그녀는 적은 수의 프랑스 병사로 오를레앙의 잉글랜드군을 격파하였으며, 그 뒤 항상 선두에 서서 잉글랜드군을 격파하였다. 그러나 <a href="/wiki/1430%EB%85%84" title="1430년">1430년</a>에 <a href="/wiki/%EC%9E%94_%EB%8B%A4%EB%A5%B4%ED%81%AC" title="잔 다르크">잔 다르크</a>는 부르고뉴파에게 체포되어 잉글랜드측으로 인도된 끝에, <a href="/wiki/1431%EB%85%84" title="1431년">1431년</a> 루앙에서 <a class="mw-redirect" href="/wiki/%EB%A7%88%EB%85%80_%EC%9E%AC%ED%8C%90" title="마녀 재판">마녀 재판</a>을 받고 마녀로 판결되어 화형에 처해졌다. <a href="/wiki/%EB%A1%9C%EB%A7%88_%EA%B5%90%ED%99%A9%EC%B2%AD" title="로마 교황청">로마 교황청</a>은 잔 다르크에 대한 새로운 조사 위원회를 설치해서 재조사한 결과, 1456년 7월 잔 다르크에게 화형 판결을 내린 재판을 폐기하고 무효화하는 선언을 발표함으로써 그녀의 명예 역시 회복되었다. 그리고 잔 다르크는 1909년 4월 18일에야 비로소 시복되었고, 1920년 5월 16일 성인으로 시성되었다. 그리하여 잔 다르크는 이단자이자 마녀라는 혐의를 벗고 가톨릭교회의 성인이 되었다.</p>
<p><a href="/wiki/1429%EB%85%84" title="1429년">1429년</a>에 샤를 7세는 정식으로 대관식을 거행하고 적법한 프랑스 국왕의 지위를 확립하였다. 따라서 샤를 6세가 영국과 맺은 트루아 조약은 효력을 상실하였다. 그러나 이 무렵 전황은 이미 프랑스 쪽으로 기울어졌으며, 수십 년간 항쟁을 계속한 부르고뉴파와 아르마냐크파도 <a href="/wiki/%EC%95%84%EB%9D%BC%EC%8A%A4" title="아라스">아라스</a>에서 화의를 맺음으로써 프랑스의 내란은 끝을 맺었다. 또 잉글랜드와 프랑스 사이에 산발적인 전투가 계속되었으나, <a href="/wiki/1444%EB%85%84" title="1444년">1444년</a> 투르에서 휴전이 성립되었다. 그러나 잉글랜드는 프랑스 내에 멘, 보르도, 칼레, 노르망디 등의 영토를 계속 보유하고 있었다.</p>
<h2><span class="mw-headline" id=".EC.9D.98.EC.9D.98.EC.99.80_.EC.98.81.ED.96.A5">의의와 영향</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%B0%B1%EB%85%84_%EC%A0%84%EC%9F%81&amp;action=edit&amp;section=7" title="부분 편집: 의의와 영향">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<h3><span class="mw-headline" id=".EB.B4.89.EA.B1.B4_.EC.A0.9C.EB.8F.84.EC.9D.98_.EB.AA.B0.EB.9D.BD.EA.B3.BC_.EC.A4.91.EC.95.99.EC.A7.91.EA.B6.8C.ED.99.94.EC.9D.98_.EC.8B.9C.EC.9E.91">봉건 제도의 몰락과 중앙집권화의 시작</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%B0%B1%EB%85%84_%EC%A0%84%EC%9F%81&amp;action=edit&amp;section=8" title="부분 편집: 봉건 제도의 몰락과 중앙집권화의 시작">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>프랑스는 장기간에 걸친 백년전쟁과 내란에 의해서 <a class="new" href="/w/index.php?title=%EB%B4%89%EA%B1%B4_%EA%B7%80%EC%A1%B1&amp;action=edit&amp;redlink=1" title="봉건 귀족 (없는 문서)">봉건 귀족</a> 세력이 극도로 약화된 반면, 국왕의 권력이 크게 신장되었다. 샤를 7세는 <a class="new" href="/w/index.php?title=%EC%9E%90%ED%81%AC_%EC%BF%A0%EB%A5%B4&amp;action=edit&amp;redlink=1" title="자크 쿠르 (없는 문서)">자크 쿠르</a>를 등용하여 왕실의 재정을 정비하고, 국왕의 <a href="/wiki/%EC%83%81%EB%B9%84%EA%B5%B0" title="상비군">상비군</a>을 강화하며 귀족세력을 누르고 <a class="new" href="/w/index.php?title=%EC%A4%91%EC%95%99_%EC%A7%91%EA%B6%8C%EC%A0%9C&amp;action=edit&amp;redlink=1" title="중앙 집권제 (없는 문서)">중앙 집권제</a>를 추진해 나갔다.</p>
<p>잉글랜드에서는 전쟁이 끝난 뒤 이제는 왕위계승문제를 둘러싸고 30년간에 걸친 랭카스터 가문과 요크 가문간에 <a href="/wiki/%EC%9E%A5%EB%AF%B8_%EC%A0%84%EC%9F%81" title="장미 전쟁">장미 전쟁</a>(<a href="/wiki/1455%EB%85%84" title="1455년">1455년</a> ~ <a href="/wiki/1485%EB%85%84" title="1485년">1485년</a>)이 일어났으니, 이것은 프랑스에서 돌아온 귀족들의 생존 경쟁이었다. 그러나 30년에 걸친 전쟁은 양 가문 및 그들에게 합류한 귀족들 자신이 함께 쓰러지는 결과를 초래하여, 그만큼 귀족 세력을 약화시키고 정략결혼을 통해 장미전쟁을 수습한 <a href="/wiki/%ED%97%A8%EB%A6%AC_7%EC%84%B8" title="헨리 7세">헨리 7세</a>에 의한 <a class="mw-redirect" href="/wiki/%EC%A4%91%EC%95%99_%EC%A7%91%EA%B6%8C" title="중앙 집권">중앙 집권</a>화의 기초를 제공하였다. 이와 같이 백년전쟁은 잉글랜드와 프랑스 두 나라에서 공히 왕정이 강화되고 중앙집권 체제로 발전하는 계기가 되었다는 점에서 의의를 찾을 수 있다. 특히 프랑스는 서양 봉건제도가 시작된 발원지이자 가장 완성된 형태였다는 점에서 그 의의가 더욱 크다 하겠다. 잉글랜드와 프랑스 모두 봉건 기사의 세력이 무너지고(이 부분은 <a href="/wiki/%ED%99%94%EC%95%BD" title="화약">화약</a> 무기의 등장도 일조를 하였다), 농노 해방의 진전, <a class="mw-redirect" href="/wiki/%EB%B6%80%EB%A5%B4%EC%A3%BC%EC%95%84" title="부르주아">부르주아</a> 계급의 대두, 왕권의 확대 등을 초래한 것이다.</p>
<p>그러나 전쟁 이후 두 나라의 중앙집권화의 양상은 각기 사뭇 달랐다. 잉글랜드는 <a href="/wiki/1215%EB%85%84" title="1215년">1215년</a> <a href="/wiki/%EB%A7%88%EA%B7%B8%EB%82%98_%EC%B9%B4%EB%A5%B4%ED%83%80" title="마그나 카르타">마그나 카르타</a> 이후 존재한 의회제도 덕분에 귀족 영주와 국왕의 대립 구도에서 <a href="/wiki/%EC%9D%98%ED%9A%8C" title="의회">의회</a>와 국왕의 대립 구도로 바뀌고 권력이 분산되어 19세기에 최초의 <a class="mw-redirect" href="/wiki/%EC%9E%85%ED%97%8C%EA%B5%B0%EC%A3%BC%EA%B5%AD" title="입헌군주국">입헌군주국</a>으로 발전한 반면, 프랑스는 전쟁 와중에 강력해진 상비군 덕분에 국왕에게 모든 권력이 집중되었던 것이다. 이러한 프랑스 상황의 상징은 <a href="/wiki/%EC%82%BC%EB%B6%80%ED%9A%8C" title="삼부회">삼부회</a>다. 삼부회도 잉글랜드의 의회처럼 전쟁 비용 징수에 대해 범계층의 동의를 통해 징수할 의도로 설립되었으나, 상비군 체제와 왕권이 강화되면서 이름만 있고 더 이상 개최되지 않은 것이다. 마지막으로 삼부회가 열린 것은 17세기였고, <a href="/wiki/%EB%A3%A8%EC%9D%B4_14%EC%84%B8" title="루이 14세">루이 14세</a> 이후 한번도 개최되지 않았던 것이다. 다시 열린 것은 <a href="/wiki/1789%EB%85%84" title="1789년">1789년</a>이었고, <a class="mw-redirect" href="/wiki/%ED%94%84%EB%9E%91%EC%8A%A4_%EB%8C%80%ED%98%81%EB%AA%85" title="프랑스 대혁명">프랑스 대혁명</a>으로 이어졌다.</p>
<h3><span class="mw-headline" id=".ED.94.84.EB.9E.91.EC.8A.A4.EC.9D.B8.EA.B3.BC_.EC.9E.89.EA.B8.80.EB.9E.9C.EB.93.9C.EC.9D.B8.EC.9D.98_.ED.98.95.EC.84.B1">프랑스인과 잉글랜드인의 형성</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%B0%B1%EB%85%84_%EC%A0%84%EC%9F%81&amp;action=edit&amp;section=9" title="부분 편집: 프랑스인과 잉글랜드인의 형성">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>한편, 잉글랜드는 프랑스 내의 영토를 상실함으로써 그 이후 유럽 대륙에서 벌어지는 복잡한 문제들에 휩쓸리지 않게 되어, 독자적인 <a class="mw-redirect" href="/wiki/%EA%B5%AD%EB%AF%BC%EA%B5%AD%EA%B0%80" title="국민국가">국민국가</a>를 형성할 수 있게 되었다. 뿐만 아니라 전쟁의 공포를 피하여 기술이민들이 플랑드르로부터 영국으로 이주함으로써, 플랑드르를 능가하는 모직물 공업이 발전하게 되어 경제적 발전의 기초가 다듬어졌다.</p>
<p>프랑스 역시 <a href="/wiki/%EC%9E%94_%EB%8B%A4%EB%A5%B4%ED%81%AC" title="잔 다르크">잔 다르크</a>로 상징되는 프랑스 인이 이 시기에 형성되었다. 사실상 그 이전에는 "잉글랜드인"이나 "프랑스 인"이라는 국민의식 같은 것은 큰 의미가 없었다고 봐야 할 것이다. 프랑스인이라는 국민감정은 이때 잉글랜드인에 대한 증오의식에서 자리를 잡아갔다.</p>
<p>이후에 두 나라가 손을 잡게 되는 것은 <a href="/wiki/1854%EB%85%84" title="1854년">1854년</a> <a href="/wiki/%ED%81%AC%EB%A6%BC_%EC%A0%84%EC%9F%81" title="크림 전쟁">크림 전쟁</a>이 사실상 처음이었으며, <a href="/wiki/%EB%8F%85%EC%9D%BC_%EC%A0%9C%EA%B5%AD" title="독일 제국">독일 제국</a>이 성립된 이후 두 나라는 동맹국 사이가 되었으나 민족 감정은 남아 있었다.</p>
<p>백년전쟁 종전 이후 잉글랜드, 프랑스, 플랑드르 지방 간의 이러한 관계는 <a href="/wiki/%EC%A0%9C1%EC%B0%A8_%EC%84%B8%EA%B3%84_%EB%8C%80%EC%A0%84" title="제1차 세계 대전">제1차 세계 대전</a> 당시까지도 독일이 <a href="/wiki/%EB%B2%A8%EA%B8%B0%EC%97%90" title="벨기에">벨기에</a>를 침공했다는 이유로 잉글랜드가 선전포고를 하기 이르렀고, 벨기에가 <a href="/wiki/%EB%84%A4%EB%8D%9C%EB%9E%80%EB%93%9C%EC%96%B4" title="네덜란드어">네덜란드어</a> 사용권 및 <a href="/wiki/%ED%94%84%EB%9E%91%EC%8A%A4%EC%96%B4" title="프랑스어">프랑스어</a> 사용권으로 나뉘게 되었다.</p>
