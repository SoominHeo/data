<p><b>백마고지역</b>(白馬高地驛)은 <a href="/wiki/%EA%B0%95%EC%9B%90%EB%8F%84_(%EB%82%A8)" title="강원도 (남)">강원도</a> <a href="/wiki/%EC%B2%A0%EC%9B%90%EA%B5%B0_(%EB%82%A8)" title="철원군 (남)">철원군</a> <a class="mw-disambig" href="/wiki/%EC%B2%A0%EC%9B%90%EC%9D%8D" title="철원읍">철원읍</a> 대마리에 위치한 <a href="/wiki/%EA%B2%BD%EC%9B%90%EC%84%A0" title="경원선">경원선</a>의 <a href="/wiki/%EC%B2%A0%EB%8F%84%EC%97%AD" title="철도역">철도역</a>이다.</p>
<p>현재 경원선 <a href="/wiki/%ED%86%B5%EA%B7%BC%EC%97%B4%EC%B0%A8_(%EB%8C%80%ED%95%9C%EB%AF%BC%EA%B5%AD)" title="통근열차 (대한민국)">통근열차</a>의 시종착역이며, <a href="/wiki/%EB%8F%99%ED%95%B4%EB%B6%81%EB%B6%80%EC%84%A0" title="동해북부선">동해북부선</a> <a href="/wiki/%EC%A0%9C%EC%A7%84%EC%97%AD" title="제진역">제진역</a>을 제외하면 대한민국의 최북단역이나, <a href="/wiki/2017%EB%85%84" title="2017년">2017년</a> <a href="/wiki/11%EC%9B%94" title="11월">11월</a>에 <a href="/wiki/%EC%9B%94%EC%A0%95%EB%A6%AC%EC%97%AD" title="월정리역">월정리역</a>이 개통되면 월정리역이 대한민국의 여객취급역 중 최북단역이 될 것이나, <a href="/wiki/%EB%AF%BC%EA%B0%84%EC%9D%B8%EC%B6%9C%EC%9E%85%ED%86%B5%EC%A0%9C%EA%B5%AC%EC%97%AD" title="민간인출입통제구역">민간인출입통제구역</a> 내에 있어 민간인들만 따진다면 이 역이 최북단역이된다.<sup class="reference" id="cite_ref-1"><a href="#cite_note-1">[1]</a></sup></p>

<p>경원선 복원 시 기존 <a href="/wiki/%EC%B2%A0%EC%9B%90%EC%97%AD" title="철원역">철원역</a>이 <a class="mw-redirect" href="/wiki/%EB%AF%BC%ED%86%B5%EC%84%A0" title="민통선">민통선</a> 내부에 있어 민통선 밖의 철원읍 대마리에 이 역이 대체·신설되기에 이르렀다. 하지만 <a href="/wiki/2015%EB%85%84" title="2015년">2015년</a> <a href="/wiki/8%EC%9B%94_5%EC%9D%BC" title="8월 5일">8월 5일</a>에 이 역에서 <a href="/wiki/%EC%9B%94%EC%A0%95%EB%A6%AC%EC%97%AD" title="월정리역">월정리역</a>까지 가는 연장선이 착공하면서 다시 민통선 안으로 들어갈 수 있을 전망이다. 철원의 일부 문학회 회원들과 인근 오대미마을(소설가 <a href="/wiki/%EC%9D%B4%ED%83%9C%EC%A4%80" title="이태준">이태준</a>의 생가 터가 있는 곳이다.) 주민들은 이 역의 이름을 <b>이태준역</b>으로 하기를 희망하기도 하였으나,<sup class="reference" id="cite_ref-2"><a href="#cite_note-2">[2]</a></sup> 인근에서 벌어진 <a href="/wiki/%EB%B0%B1%EB%A7%88%EA%B3%A0%EC%A7%80_%EC%A0%84%ED%88%AC" title="백마고지 전투">백마고지 전투</a>에서 이름을 따와 현재의 이름으로 정해졌다. 처음에는 경원선의 기존 노반을 그대로 활용할 예정이었으나, 관할 군(軍)부대의 반대로 인하여 터널과 고가 구간을 거쳐 넘어오는 현재의 경로로 변경되었다.<sup class="reference" id="cite_ref-3"><a href="#cite_note-3">[3]</a></sup></p>
<ul>
<li><a href="/wiki/2007%EB%85%84" title="2007년">2007년</a> <a href="/wiki/12%EC%9B%94" title="12월">12월</a> : <a href="/wiki/%EC%8B%A0%ED%83%84%EB%A6%AC%EC%97%AD" title="신탄리역">신탄리</a>-<strong class="selflink">백마고지</strong> 착공</li>
<li><a href="/wiki/2012%EB%85%84" title="2012년">2012년</a> <a href="/wiki/11%EC%9B%94_20%EC%9D%BC" title="11월 20일">11월 20일</a> : 영업 개시및 7~8회 운행</li>
<li><a href="/wiki/2014%EB%85%84" title="2014년">2014년</a> <a href="/wiki/4%EC%9B%94_14%EC%9D%BC" title="4월 14일">4월 14일</a> : 경원선 통근열차 11회 운행</li>
<li><a href="/wiki/2014%EB%85%84" title="2014년">2014년</a> <a href="/wiki/8%EC%9B%94_1%EC%9D%BC" title="8월 1일">8월 1일</a> : <a href="/wiki/%ED%8F%89%ED%99%94%EC%97%B4%EC%B0%A8" title="평화열차">평화열차(DMZ train)</a> 운행 개시</li>
<li><a href="/wiki/2015%EB%85%84" title="2015년">2015년</a> <a href="/wiki/8%EC%9B%94_5%EC%9D%BC" title="8월 5일">8월 5일</a> : 경원선 <strong class="selflink">백마고지</strong> - <a href="/wiki/%EC%9B%94%EC%A0%95%EB%A6%AC%EC%97%AD" title="월정리역">월정리</a> 단선 비전철 복원 구간 기공식이 백마고지역에서 개최</li>
</ul>
<h2><span class="mw-headline" id=".EC.97.AD_.EA.B5.AC.EC.A1.B0">역 구조</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%B0%B1%EB%A7%88%EA%B3%A0%EC%A7%80%EC%97%AD&amp;action=edit&amp;section=2" title="부분 편집: 역 구조">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<ul>
<li><a href="/wiki/%EC%8B%A0%ED%83%84%EB%A6%AC%EC%97%AD" title="신탄리역">신탄리역</a> 북단으로부터 이 역의 북단인 <a href="/wiki/%EC%9A%A9%EC%82%B0%EC%97%AD" title="용산역">용산역</a> 기점 94.6km까지는 단선 구조이며, 그 구간 내에는 회차선이 없다. 이 때문에 기관차 견인 열차는 정상적으로 운행할 수 없고, 동차형 차량만 운행할 수 있다.</li>
<li>승강장의 길이는 <a href="/wiki/%ED%86%B5%EA%B7%BC%EC%97%B4%EC%B0%A8_(%EB%8C%80%ED%95%9C%EB%AF%BC%EA%B5%AD)" title="통근열차 (대한민국)">통근열차</a> 4량을 수용할 수 있으며 북쪽으로 약간 더 있다. 다른 <a href="/wiki/%EA%B2%BD%EC%9B%90%EC%84%A0" title="경원선">경원선</a>의 승강장 길이에 비하여 짧다.</li>
<li>역 건물은 화장실이 구비되어 있는 현대식이나, 역무원은 없다.</li>
<li>역 구내에는 철원군 농특산물 판매장이 병설되어 있다(역과 무관한 별도의 인원 상주).</li>
<li>열차는 2시간에 1회 운행되며, 이 역에 종착 후 약 5분 뒤 동두천행으로 반복운행한다<sup class="reference" id="cite_ref-4"><a href="#cite_note-4">[4]</a></sup>.</li>
<li><a href="/wiki/%ED%8F%89%ED%99%94%EC%97%B4%EC%B0%A8" title="평화열차">평화열차</a>는 이 역에 종착 후 약 8분 뒤 동두천 방향으로 출발하여 <a href="/wiki/%EC%8B%A0%ED%83%84%EB%A6%AC%EC%97%AD" title="신탄리역">신탄리역</a>에서 주박을 한 후 다시 이 역으로 돌아와 <a href="/wiki/%EC%84%9C%EC%9A%B8%EC%97%AD" title="서울역">서울</a> 방향으로 출발한다.</li>
</ul>

<h3><span class="mw-headline" id=".EC.95.88.EB.B3.B4.EA.B4.80.EA.B4.91.EC.85.94.ED.8B.80.EB.B2.84.EC.8A.A4">안보관광셔틀버스</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%B0%B1%EB%A7%88%EA%B3%A0%EC%A7%80%EC%97%AD&amp;action=edit&amp;section=4" title="부분 편집: 안보관광셔틀버스">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p><a href="/wiki/%ED%8F%89%ED%99%94%EC%97%B4%EC%B0%A8" title="평화열차">평화열차</a> 시간에 맞추어 백마고지역에서 오전 10시 40분 1일 1회 출발하며 매주 <a href="/wiki/%ED%99%94%EC%9A%94%EC%9D%BC" title="화요일">화요일</a>은 운행하지 않는다. 백마고지역에서 출발하는 코스는 안보관광코스 제6코스로 <a class="mw-redirect" href="/wiki/%EC%A0%9C2%EB%95%85%EA%B5%B4" title="제2땅굴">제2땅굴</a>~<a class="new" href="/w/index.php?title=%ED%8F%89%ED%99%94%EC%A0%84%EB%A7%9D%EB%8C%80&amp;action=edit&amp;redlink=1" title="평화전망대 (없는 문서)">평화전망대</a>~<a href="/wiki/%EC%9B%94%EC%A0%95%EB%A6%AC%EC%97%AD" title="월정리역">월정리역</a> 및 두루미관~백마고지역의 순으로 돌아보게 된다. 탑승시에는 <a href="/wiki/%EB%AF%BC%EA%B0%84%EC%9D%B8%EC%B6%9C%EC%9E%85%ED%86%B5%EC%A0%9C%EA%B5%AC%EC%97%AD" title="민간인출입통제구역">민간인출입통제구역</a> 출입 수속을 거치기 위하여 신분증과 함께 평화열차 왕복승차권을 제시하여야 한다.</p>
