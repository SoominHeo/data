<p><b>버넌 마이클 웰스 3세</b>(Vernon Michael Wells III, <a href="/wiki/1978%EB%85%84" title="1978년">1978년</a> <a href="/wiki/12%EC%9B%94_8%EC%9D%BC" title="12월 8일">12월 8일</a> ~ )는 <a href="/wiki/%EB%AF%B8%EA%B5%AD" title="미국">미국</a>출신의 <a class="mw-disambig" href="/wiki/%EB%A9%94%EC%9D%B4%EC%A0%80_%EB%A6%AC%EA%B7%B8" title="메이저 리그">메이저 리그</a>선수이다. 소속팀은 <a href="/wiki/%EB%89%B4%EC%9A%95_%EC%96%91%ED%82%A4%EC%8A%A4" title="뉴욕 양키스">뉴욕 양키스</a>이며, 포지션은 <a href="/wiki/%EC%99%B8%EC%95%BC%EC%88%98" title="외야수">외야수</a>이다.</p>
<p>과거 <a href="/wiki/%ED%86%A0%EB%A1%A0%ED%86%A0_%EB%B8%94%EB%A3%A8%EC%A0%9C%EC%9D%B4%EC%8A%A4" title="토론토 블루제이스">토론토 블루제이스</a>에 지명되어 활약을 펼치며 프랜차이즈 스타가 되었지만, <a href="/wiki/2011%EB%85%84" title="2011년">2011년</a> <a href="/wiki/1%EC%9B%94" title="1월">1월</a>에 트레이드 되어 <a href="/wiki/%EB%A1%9C%EC%8A%A4%EC%95%A4%EC%A0%A4%EB%A0%88%EC%8A%A4_%EC%97%90%EC%9D%B8%EC%A0%88%EC%8A%A4_%EC%98%A4%EB%B8%8C_%EC%95%A0%EB%84%88%ED%95%98%EC%9E%84" title="로스앤젤레스 에인절스 오브 애너하임">로스앤젤레스 에인절스 오브 애너하임</a>로 이적하게 된다.<sup class="reference" id="cite_ref-1"><a href="#cite_note-1">[1]</a></sup> 이보다 좀더 이른 시기에는 <a href="/wiki/%ED%86%A0%EB%A1%A0%ED%86%A0_%EB%B8%94%EB%A3%A8%EC%A0%9C%EC%9D%B4%EC%8A%A4" title="토론토 블루제이스">토론토</a>가 프랜차이즈 스타인 <a href="/wiki/%EB%A1%9C%EC%9D%B4_%ED%95%A0%EB%9F%AC%EB%8D%B0%EC%9D%B4" title="로이 할러데이">로이 할러데이</a>를 <a href="/wiki/%ED%95%84%EB%9D%BC%EB%8D%B8%ED%94%BC%EC%95%84_%ED%95%84%EB%A6%AC%EC%8A%A4" title="필라델피아 필리스">필라델피아</a>로 트레이드 시키기도 했다.<sup class="reference" id="cite_ref-2"><a href="#cite_note-2">[2]</a></sup></p>
