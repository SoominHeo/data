<p><b>베갈타 센다이</b>(<span style="font-size: smaller;"><a href="/wiki/%EC%9D%BC%EB%B3%B8%EC%96%B4" title="일본어">일본어</a>:</span> <span lang="ja" xml:lang="ja">ベガルタ仙台</span> <small>베가루타 센다이<sup>[<a href="/wiki/%EC%9C%84%ED%82%A4%EB%B0%B1%EA%B3%BC:%EC%9D%BC%EB%B3%B8%EC%96%B4%EC%9D%98_%ED%95%9C%EA%B8%80_%ED%91%9C%EA%B8%B0" title="위키백과:일본어의 한글 표기">*</a>]</sup></small>, Vegalta Sendai)는 <a href="/wiki/%EC%9D%BC%EB%B3%B8" title="일본">일본</a> <a href="/wiki/%EB%AF%B8%EC%95%BC%EA%B8%B0_%ED%98%84" title="미야기 현">미야기 현</a> <a href="/wiki/%EC%84%BC%EB%8B%A4%EC%9D%B4_%EC%8B%9C" title="센다이 시">센다이 시</a>를 연고로 하는 <a href="/wiki/J%EB%A6%AC%EA%B7%B8" title="J리그">J리그</a> 축구팀이다. 명칭은 '<a href="/wiki/%EB%B2%A0%EA%B0%80" title="베가">직녀성</a>'를 뜻하는 Vega와 '<a href="/wiki/%EC%95%8C%ED%83%80%EC%9D%B4%EB%A5%B4" title="알타이르">견우성</a>'를 뜻하는 Altair의 <a href="/wiki/%ED%98%BC%EC%84%B1%EC%96%B4" title="혼성어">합성어</a>다. 이는 '센다이 <a href="/wiki/%EC%B9%A0%EC%84%9D" title="칠석">칠석제</a>' 중 <a href="/wiki/%EA%B2%AC%EC%9A%B0%EC%A7%81%EB%85%80" title="견우직녀">견우와 직녀</a>의 전설을 상징한다. 현재 <a href="/wiki/%EB%9F%89%EC%9A%A9%EA%B8%B0" title="량용기">량용기</a>가 뛰고 있는 소속팀이다.</p>
<h2><span class="mw-headline" id=".EC.97.AD.EC.82.AC">역사</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%B2%A0%EA%B0%88%ED%83%80_%EC%84%BC%EB%8B%A4%EC%9D%B4&amp;action=edit&amp;section=1" title="부분 편집: 역사">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>1988년 '도호쿠 전기회사 축구부'가 전신이다. 1995년 JFL에 가맹하였고 1999년 J2리그로 승격한다. 2001년 J2리그 1위를 기록하여 <a href="/wiki/%EB%8F%84%ED%98%B8%EC%BF%A0_%EC%A7%80%EB%B0%A9" title="도호쿠 지방">도호쿠 지방</a> 팀으로는 최초로 J1으로 승격하였으나 2003년 전·후기 통합 15위를 기록, J2리그로 다시 강등되었다. 2008년 에 J2리그 3위를 기록하여 <a class="mw-redirect" href="/wiki/J%EB%A6%AC%EA%B7%B8_%EA%B5%90%EC%B2%B4%EC%A0%84" title="J리그 교체전">리그 교체전</a>에 진출, <a href="/wiki/%EC%A3%BC%EB%B9%8C%EB%A1%9C_%EC%9D%B4%EC%99%80%ED%83%80" title="주빌로 이와타">주빌로 이와타</a>에게 패하여 잔류하였으나 이듬해 1위를 기록하여 7년 만에 J1으로 승격했고 2011년 <a class="mw-redirect" href="/wiki/%EB%8F%99%EC%9D%BC%EB%B3%B8_%EB%8C%80%EC%A7%80%EC%A7%84" title="동일본 대지진">동일본 대지진</a>의 충격을 극복하고 선전하여, <a href="/wiki/%EC%9A%94%EC%BD%94%ED%95%98%EB%A7%88_F._%EB%A7%88%EB%A6%AC%EB%85%B8%EC%8A%A4" title="요코하마 F. 마리노스">요코하마 F. 마리노스</a>를 골득실차로 꺾고 리그 4위를 기록(리그 최소실점-25)하였고 이듬해 <a href="/wiki/%EC%82%B0%ED%94%84%EB%A0%88%EC%B2%B4_%ED%9E%88%EB%A1%9C%EC%8B%9C%EB%A7%88" title="산프레체 히로시마">산프레체 히로시마</a>에 이어 2위에 올라 팀 역사상 최초로 <a href="/wiki/AFC_%EC%B1%94%ED%94%BC%EC%96%B8%EC%8A%A4%EB%A6%AC%EA%B7%B8" title="AFC 챔피언스리그">AFC 챔피언스리그</a>에 진출하게 되었다.</p>
