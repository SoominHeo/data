<p><b>베네수엘라의 정치</b> 문서에서는 <a class="new" href="/w/index.php?title=%EB%B2%A0%EB%84%A4%EC%88%98%EC%97%98%EB%9D%BC%EC%9D%98_%EC%A0%95%EB%B6%80&amp;action=edit&amp;redlink=1" title="베네수엘라의 정부 (없는 문서)">베네수엘라의 정부</a>에서 벌어지는 일들을 서술한다.</p>
<p><a href="/wiki/%EB%B2%A0%EB%84%A4%EC%88%98%EC%97%98%EB%9D%BC" title="베네수엘라">베네수엘라</a>는 <a href="/wiki/%EB%8B%A4%EB%8B%B9%EC%A0%9C" title="다당제">다당제</a> 국가이며 동시에 <a class="mw-redirect" href="/wiki/%EC%9D%BC%EB%8B%B9%EC%9A%B0%EC%9C%84%EC%A0%9C" title="일당우위제">일당우위제</a> 국가이다. 현재 집권당인 <a href="/wiki/%EB%B2%A0%EB%84%A4%EC%88%98%EC%97%98%EB%9D%BC_%EC%97%B0%ED%95%A9%EC%82%AC%ED%9A%8C%EB%8B%B9" title="베네수엘라 연합사회당">베네수엘라 연합사회당</a>은 2007년 <a class="mw-redirect" href="/wiki/%ED%9C%B4%EA%B3%A0_%EC%B0%A8%EB%B2%A0%EC%8A%A4" title="휴고 차베스">휴고 차베스</a>가 이끌던 <a class="new" href="/w/index.php?title=5%EA%B3%B5%ED%99%94%EA%B5%AD_%EC%9A%B4%EB%8F%99&amp;action=edit&amp;redlink=1" title="5공화국 운동 (없는 문서)">5공화국 운동</a>과 기타 지지세력의 통합으로 만들어진 당이다. 이들은 1998년부터 2014년 1월까지 베네수엘라 행정부의 수반인 대통령을 배출했으며, <a class="new" href="/w/index.php?title=%EB%B2%A0%EB%84%A4%EC%88%98%EC%97%98%EB%9D%BC_%EC%9D%98%ED%9A%8C&amp;action=edit&amp;redlink=1" title="베네수엘라 의회 (없는 문서)">베네수엘라 의회</a>의 다수 의석을 점하고 있다.</p>
<p>연합사회당에 맞서는 야권 정당연합은 <a href="/wiki/%EB%AF%BC%EC%A3%BC%EC%97%B0%ED%95%A9_%EC%9B%90%ED%83%81%ED%9A%8C%EC%9D%98" title="민주연합 원탁회의">민주연합 원탁회의</a>다. 2008년 결성된 원탁회의는 좌우를 막론한 다양한 야당의 야권연대 기구다. 원탁회의에 참여하지 않고 제3지대에서 활동하는 야당도 있다.</p>
<p>최근까지 베네수엘라 정치에서 가장 중요한 인물은 1998년부터 2013년까지 대통령을 지낸 <a class="mw-redirect" href="/wiki/%ED%9C%B4%EA%B3%A0_%EC%B0%A8%EB%B2%A0%EC%8A%A4" title="휴고 차베스">휴고 차베스</a>였다. 2013년 초 차베스는 사망했지만 그 후계자인 <a href="/wiki/%EB%8B%88%EC%BD%9C%EB%9D%BC%EC%8A%A4_%EB%A7%88%EB%91%90%EB%A1%9C" title="니콜라스 마두로">니콜라스 마두로</a>가 보궐선거에서 당선됐다.(<a class="new" href="/w/index.php?title=2013%EB%85%84_%EB%B2%A0%EB%84%A4%EC%88%98%EC%97%98%EB%9D%BC_%EB%8C%80%EC%84%A0&amp;action=edit&amp;redlink=1" title="2013년 베네수엘라 대선 (없는 문서)">2013년 베네수엘라 대선</a> 참고)</p>

<h2><span class="mw-headline" id=".EC.97.AD.EC.82.AC">역사</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%B2%A0%EB%84%A4%EC%88%98%EC%97%98%EB%9D%BC%EC%9D%98_%EC%A0%95%EC%B9%98&amp;action=edit&amp;section=1" title="부분 편집: 역사">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>이 문단에서는 10년간의 군사독재가 종식된 1958년의 역사를 서술한다.</p>
<h3><span class="mw-headline" id="1958.7E1999">1958~1999</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%B2%A0%EB%84%A4%EC%88%98%EC%97%98%EB%9D%BC%EC%9D%98_%EC%A0%95%EC%B9%98&amp;action=edit&amp;section=2" title="부분 편집: 1958~1999">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>베네수엘라 민주주의의 역사를 설명하려면 1941년 창설된 <a class="mw-disambig" href="/wiki/%EB%AF%BC%EC%A3%BC%ED%96%89%EB%8F%99%EB%8B%B9" title="민주행동당">민주행동당</a>을 빼놓을 수 없다. 민주행동당의 초기 지도자들은 베네수엘라 민주주의의 기반을 닦은 것으로 평가된다. 민주행동당은 1945년부터 1948년까지 정권을 잡았으나, 이후 10년간 지속된 군사독재 기간동안에는 권력에서 멀어져 있었다. 1958년 군사독재가 종식된 베네수엘라 국민들은 직접선거를 통해 대통령을 선출했다. 1958년부터 1999년까지의 기간을 흔히 '제4공화국'이라고 부른다. 제4공화국의 성격을 결정한 중요한 사건은 1958년의 <a class="new" href="/w/index.php?title=%ED%91%BC%ED%86%A0_%ED%94%BC%EC%A1%B0_%ED%98%91%EC%A0%95&amp;action=edit&amp;redlink=1" title="푼토 피조 협정 (없는 문서)">푼토 피조 협정</a>이었다. 이 협정에는 민주행동당, <a class="mw-redirect" href="/wiki/COPEI" title="COPEI">COPEI</a>, 그리고 당시로선 주요 정당이었던 <a class="new" href="/w/index.php?title=%EB%AF%BC%EC%A3%BC%EA%B3%B5%ED%99%94%EC%97%B0%ED%95%A9&amp;action=edit&amp;redlink=1" title="민주공화연합 (없는 문서)">민주공화연합</a>이 참여했다. 이로 인해 실질적인 양당제 체제가 성립됐고, <a href="/wiki/%EB%B2%A0%EB%84%A4%EC%88%98%EC%97%98%EB%9D%BC_%EA%B3%B5%EC%82%B0%EB%8B%B9" title="베네수엘라 공산당">베네수엘라 공산당</a>이 추방됐다.</p>
<p>하지만 1990년대 말 들어 양당제의 신뢰성은 바닥을 기었다. 베네수엘라 사람들은 부패와 가난에 시달렸다. 1970년대 베네수엘라는 석유로 인한 막대한 수입을 탕진해 버렸고, 1980년대에는 국가 부채 위기에 시달렸다. 민주행동당의 마지막 대통령인 <a class="new" href="/w/index.php?title=%EC%B9%B4%EB%A5%BC%EB%A1%9C%EC%8A%A4_%EC%95%88%EB%93%9C%EB%A0%88%EC%8A%A4_%ED%8E%98%EB%A0%88%EC%A6%88&amp;action=edit&amp;redlink=1" title="카를로스 안드레스 페레즈 (없는 문서)">카를로스 안드레스 페레즈</a>는 1993년 부패 혐의로 탄핵당한 뒤 수년을 감옥에서 지냈다.</p>
<p>민주행동당과 더불어 양당제를 이끌었던 당은 2차례 대통령을 배출한 <a class="mw-redirect" href="/wiki/COPEI" title="COPEI">COPEI</a>였다. 하지만 양당의 신뢰가 무너지면서 새로운 세력이 등장하게 됐다. <a class="new" href="/w/index.php?title=1993%EB%85%84_%EB%B2%A0%EB%84%A4%EC%88%98%EC%97%98%EB%9D%BC_%EB%8C%80%EC%84%A0&amp;action=edit&amp;redlink=1" title="1993년 베네수엘라 대선 (없는 문서)">1993년 베네수엘라 대선</a>의 승리자는 30%를 득표한 <a href="/wiki/%EB%9D%BC%ED%8C%8C%EC%97%98_%EC%B9%BC%EB%8D%B0%EB%9D%BC" title="라파엘 칼데라">라파엘 칼데라</a>였다. 그는 새로운 선거연합인 <a class="new" href="/w/index.php?title=%EA%B5%AD%EB%AF%BC_%EB%8B%A8%EA%B2%B0&amp;action=edit&amp;redlink=1" title="국민 단결 (없는 문서)">국민 단결</a> 소속이었다. 이후에도 기존 양당의 지지도는 꾸준히 하락세를 겪었다. 결국 <a class="new" href="/w/index.php?title=1998%EB%85%84_%EB%B2%A0%EB%84%A4%EC%88%98%EC%97%98%EB%9D%BC_%EB%8C%80%EC%84%A0&amp;action=edit&amp;redlink=1" title="1998년 베네수엘라 대선 (없는 문서)">1998년 베네수엘라 대선</a>에서는 그동안 기성 정치권의 외부에 있었던 <a class="mw-redirect" href="/wiki/%ED%9C%B4%EA%B3%A0_%EC%B0%A8%EB%B2%A0%EC%8A%A4" title="휴고 차베스">휴고 차베스</a>가 승리하게 된다.</p>
<h3><span class="mw-headline" id="1999.EB.85.84.7E.ED.98.84.EC.9E.AC">1999년~현재</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%B2%A0%EB%84%A4%EC%88%98%EC%97%98%EB%9D%BC%EC%9D%98_%EC%A0%95%EC%B9%98&amp;action=edit&amp;section=3" title="부분 편집: 1999년~현재">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>차베스는 집권 이후 대규모 사회 운동인 <a href="/wiki/%EB%B3%BC%EB%A6%AC%EB%B0%94%EB%A5%B4_%ED%98%81%EB%AA%85" title="볼리바르 혁명">볼리바르 혁명</a>을 시작했다. 그는 공약에 따라 <a class="new" href="/w/index.php?title=1999%EB%85%84_%EB%B2%A0%EB%84%A4%EC%88%98%EC%97%98%EB%9D%BC_%EC%A0%9C%ED%97%8C_%EC%9D%98%ED%9A%8C&amp;action=edit&amp;redlink=1" title="1999년 베네수엘라 제헌 의회 (없는 문서)">1999년 베네수엘라 제헌 의회</a>를 소집했고, 제헌의회는 새로운 <a class="new" href="/w/index.php?title=%EB%B2%A0%EB%84%A4%EC%88%98%EC%97%98%EB%9D%BC_%ED%97%8C%EB%B2%95&amp;action=edit&amp;redlink=1" title="베네수엘라 헌법 (없는 문서)">베네수엘라 헌법</a>을 제정했다.</p>
<p>차베스의 반대파들은 <a class="new" href="/w/index.php?title=2002%EB%85%84_%EB%B2%A0%EB%84%A4%EC%88%98%EC%97%98%EB%9D%BC_%EC%BF%A0%EB%8D%B0%ED%83%80_%EC%8B%9C%EB%8F%84&amp;action=edit&amp;redlink=1" title="2002년 베네수엘라 쿠데타 시도 (없는 문서)">2002년 쿠데타</a>를 통해 차베스를 제거하려 했으나 무위로 돌아갔다. 이후에도 반대파들은 총파업을 벌였고, 2004년에는 대통령 국민소환을 시도했지만 모두 실패했다. 이어 야권은 2005년 총선을 보이콧했지만 2006년 차베스는 3선에 성공한다. 하지만 차베스도 2007년 개헌 국민투표에서 패배한다. 당시 개헌안에는 선거 연령 하향 조정, 주간 노동시간 단축 등이 들어 있었지만, 대통령의 연임 제한을 철폐한 것이 논쟁이 됐다.</p>
<p>2007년에는 새로운 여당인 <a href="/wiki/%EB%B2%A0%EB%84%A4%EC%88%98%EC%97%98%EB%9D%BC_%EC%97%B0%ED%95%A9%EC%82%AC%ED%9A%8C%EB%8B%B9" title="베네수엘라 연합사회당">베네수엘라 연합사회당</a>이 출범한다. 이는 차베스와 <a href="/wiki/%EB%B3%BC%EB%A6%AC%EB%B0%94%EB%A5%B4_%ED%98%81%EB%AA%85" title="볼리바르 혁명">볼리바르 혁명</a>을 지지하는 세력들이 모인 정당이다. 2008년 야권은 <a href="/wiki/%EB%AF%BC%EC%A3%BC%EC%97%B0%ED%95%A9_%EC%9B%90%ED%83%81%ED%9A%8C%EC%9D%98" title="민주연합 원탁회의">민주연합 원탁회의</a>를 창설했다. 2012년 대선에서 야권은 후보 단일화에 성공했지만 차베스의 4선을 막지 못했다.</p>
<p>2013년 3월, 차베스가 지병으로 숨을 거뒀다. 4월 치러진 대통령 보궐선거에서 차베스의 후계자인 <a href="/wiki/%EB%8B%88%EC%BD%9C%EB%9D%BC%EC%8A%A4_%EB%A7%88%EB%91%90%EB%A1%9C" title="니콜라스 마두로">니콜라스 마두로</a>는 50.6%를 득표해 49.1%를 얻은 야권 단일후보 <a href="/wiki/%EC%97%94%EB%A6%AC%EC%BC%80_%EC%B9%B4%ED%94%84%EB%A6%B4%EB%A0%88%EC%8A%A4_%EB%9D%BC%EB%8F%88%EC%8A%A4%ED%82%A4" title="엔리케 카프릴레스 라돈스키">엔리케 카프릴레스 라돈스키</a>를 제치고 당선됐다.</p>
<h2><span class="mw-headline" id=".EC.84.A0.EA.B1.B0">선거</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%B2%A0%EB%84%A4%EC%88%98%EC%97%98%EB%9D%BC%EC%9D%98_%EC%A0%95%EC%B9%98&amp;action=edit&amp;section=4" title="부분 편집: 선거">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<div class="rellink boilerplate seealso"><img alt="" data-file-height="128" data-file-width="128" height="16" src="//upload.wikimedia.org/wikipedia/commons/thumb/e/ec/Crystal_Clear_app_xmag.svg/16px-Crystal_Clear_app_xmag.svg.png" srcset="//upload.wikimedia.org/wikipedia/commons/thumb/e/ec/Crystal_Clear_app_xmag.svg/24px-Crystal_Clear_app_xmag.svg.png 1.5x, //upload.wikimedia.org/wikipedia/commons/thumb/e/ec/Crystal_Clear_app_xmag.svg/32px-Crystal_Clear_app_xmag.svg.png 2x" width="16"/> <a class="new" href="/w/index.php?title=%EB%B2%A0%EB%84%A4%EC%88%98%EC%97%98%EB%9D%BC%EC%9D%98_%EC%84%A0%EA%B1%B0&amp;action=edit&amp;redlink=1" title="베네수엘라의 선거 (없는 문서)">베네수엘라의 선거</a> 문서를 참고하십시오.</div>
<p>베네수엘라의 국민들은 직접선거를 통해 <a href="/wiki/%EB%B2%A0%EB%84%A4%EC%88%98%EC%97%98%EB%9D%BC%EC%9D%98_%EB%8C%80%ED%86%B5%EB%A0%B9" title="베네수엘라의 대통령">베네수엘라의 대통령</a>을 선출한다. 베네수엘라의 대통령은 <a class="mw-redirect" href="/wiki/%EA%B5%AD%EA%B0%80_%EC%9B%90%EC%88%98" title="국가 원수">국가 원수</a>이면서 동시에 <a href="/wiki/%ED%96%89%EC%A0%95%EB%B6%80_%EC%88%98%EB%B0%98" title="행정부 수반">행정부 수반</a>이다. 또한 베네수엘라 국민들은 <a class="new" href="/w/index.php?title=%EB%B2%A0%EB%84%A4%EC%88%98%EC%97%98%EB%9D%BC_%EC%9D%98%ED%9A%8C&amp;action=edit&amp;redlink=1" title="베네수엘라 의회 (없는 문서)">베네수엘라 의회</a>의 의원들을 선출하는데, 베네수엘라 의회는 <a href="/wiki/%EB%8B%A8%EC%9B%90%EC%A0%9C" title="단원제">단원제</a>를 취하고 있다.</p>
<p>2009년 개헌 이후 베네수엘라의 대통령은 재선이 가능하며 임기는 6년이다. 베네수엘라 의회 의원의 숫자는 165명이며, 임기는 5년이다.</p>
