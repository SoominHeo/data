<p><b>베다</b>(<span style="font-size: smaller;"><a href="/wiki/%EA%B3%A0%EB%8C%80_%EC%98%81%EC%96%B4" title="고대 영어">고대 영어</a>:</span> <span lang="ang" xml:lang="ang">Bēda/Bǣda</span>, <a href="/wiki/672%EB%85%84" title="672년">672년</a> 또는 <a href="/wiki/673%EB%85%84" title="673년">673년</a>경~<a href="/wiki/735%EB%85%84" title="735년">735년</a> <a href="/wiki/5%EC%9B%94_27%EC%9D%BC" title="5월 27일">5월 27일</a>)는 <b>성 비드</b>(<span style="font-size: smaller;"><a href="/wiki/%EB%9D%BC%ED%8B%B4%EC%96%B4" title="라틴어">라틴어</a>:</span> <span lang="la" xml:lang="la">Beda Venerabilis</span>, <span lang="en" xml:lang="en">the Venerable Bede</span>)로 널리 알려진 <a class="mw-redirect" href="/wiki/%EB%85%B8%EC%84%AC%EB%B8%8C%EB%A6%AC%EC%95%84" title="노섬브리아">노섬브리아</a> 출신의 <a href="/wiki/%EA%B8%B0%EB%8F%85%EA%B5%90" title="기독교">기독교</a> 수도사로, <a class="mw-redirect" href="/wiki/%EC%8B%A0%ED%95%99%EC%9E%90" title="신학자">신학자</a>, <a href="/wiki/%EC%97%AD%EC%82%AC%EA%B0%80" title="역사가">역사가</a>, 연대기 학자이다.</p>
<h2><span class="mw-headline" id=".EA.B2.BD.EB.A0.A5">경력</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%B2%A0%EB%8B%A4_%EB%B2%A0%EB%84%A4%EB%9D%BC%EB%B9%8C%EB%A6%AC%EC%8A%A4&amp;action=edit&amp;section=1" title="부분 편집: 경력">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<h3><span class="mw-headline" id=".EC.88.98.EB.8F.84.EC.82.AC">수도사</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%B2%A0%EB%8B%A4_%EB%B2%A0%EB%84%A4%EB%9D%BC%EB%B9%8C%EB%A6%AC%EC%8A%A4&amp;action=edit&amp;section=2" title="부분 편집: 수도사">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>오늘날의 <a href="/wiki/%EC%84%A0%EB%8D%9C%EB%9E%9C%EB%93%9C" title="선덜랜드">선덜랜드</a> 지방의 일부인 위어머스(Wearmouth)의 성 베드로 수도원과 오늘날의 <a class="new" href="/w/index.php?title=%EC%9E%AC%EB%A1%9C&amp;action=edit&amp;redlink=1" title="재로 (없는 문서)">재로</a>의 성 바오로 수도원에서 수도하였다. 《잉글랜드인들의 교회사(Historia ecclesiastica gentis Anglorum, 약칭 영국교회사)》를 집필한 학자로 유명하여 '잉글랜드 역사의 아버지'라 불린다. 음악과 운율학, 자연 과학, 성서 주석 등 다양한 분야에 걸친 저술 활동을 하였다. 그의 저작은 그가 당대의 지식을 두루 섭렵했다는 것을 보여 준다.</p>

<h3><span class="mw-headline" id=".EC.84.B1.EC.84.9C_.EB.B2.88.EC.97.AD">성서 번역</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%B2%A0%EB%8B%A4_%EB%B2%A0%EB%84%A4%EB%9D%BC%EB%B9%8C%EB%A6%AC%EC%8A%A4&amp;action=edit&amp;section=3" title="부분 편집: 성서 번역">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p><a class="mw-redirect" href="/wiki/%EC%84%B1%EC%84%9C" title="성서">성서</a>의 <a href="/wiki/%EB%9D%BC%ED%8B%B4%EC%96%B4" title="라틴어">라틴어</a> <a href="/wiki/%EB%B6%88%EA%B0%80%ED%83%80" title="불가타">불가타</a>(Vulgata) 번역을 한 권으로 묶어 재편집한 베다의 역본은 <a href="/wiki/%EC%A2%85%EA%B5%90_%EA%B0%9C%ED%98%81" title="종교 개혁">종교 개혁</a> 때까지 서방 교회 세계를 통틀어서 쓰인 공식 역본이었고 <a class="mw-redirect" href="/wiki/%EB%A1%9C%EB%A7%88_%EA%B0%80%ED%86%A8%EB%A6%AD_%EA%B5%90%ED%9A%8C" title="로마 가톨릭 교회">로마 가톨릭 교회</a>에서는 <a href="/wiki/1966%EB%85%84" title="1966년">1966년</a>까지 쓰였다. 성서의 일부를 <a href="/wiki/%EA%B3%A0%EB%8C%80_%EC%98%81%EC%96%B4" title="고대 영어">고대 영어</a>(앵글로색슨어)로 번역하였지만 전해지지 않는다. 임종의 자리에서 끝마친 마지막 저작은 <a class="mw-redirect" href="/wiki/%EC%9A%94%ED%95%9C%EB%B3%B5%EC%9D%8C%EC%84%9C" title="요한복음서">요한복음서</a>의 <a href="/wiki/%EA%B3%A0%EB%8C%80_%EC%98%81%EC%96%B4" title="고대 영어">고대 영어</a> 번역본과 <a href="/wiki/%EA%B3%A0%EB%8C%80_%EC%98%81%EC%96%B4" title="고대 영어">고대 영어</a>로 쓴 <a href="/wiki/%EC%B5%9C%ED%9B%84%EC%9D%98_%EC%8B%AC%ED%8C%90" title="최후의 심판">최후의 심판</a>에 관한 <a href="/wiki/%EC%8B%9C_(%EB%AC%B8%ED%95%99)" title="시 (문학)">시</a>라고 한다. 오늘날의 학자들에게 〈베다의 임종시〉로 알려진 이 5행시는 베다의 제자인 <a class="new" href="/w/index.php?title=%EB%A6%B0%EB%94%94%EC%8A%A4%ED%8C%90%EC%9D%98_%EC%BB%A4%EC%8A%A4%EB%B2%84%ED%8A%B8&amp;action=edit&amp;redlink=1" title="린디스판의 커스버트 (없는 문서)">린디스판의 커스버트</a>(Cuthbert of Lindisfarne)가 쓴 편지에 전해지는데 그 내용만으로는 베다의 자작시인지 확실하지 않지만 라틴어로 쓴 편지에 고대 영어로 시를 인용한 점과 베다가 같은 주제로 <a href="/wiki/%EB%9D%BC%ED%8B%B4%EC%96%B4" title="라틴어">라틴어</a> 시를 지었던 점, 또 베다가 “우리의 시에 대한 학식을 갖추었다”라는 커스버트의 주장에 비추어 베다의 자작시일 가능성이 충분하다.</p>
<p><a class="mw-redirect" href="/wiki/%EB%8B%A8%ED%85%8C" title="단테">단테</a>의 《<a href="/wiki/%EC%8B%A0%EA%B3%A1" title="신곡">신곡</a>》 〈천국〉편에 나오는 유일한 잉글랜드인이며 잉글랜드인으로는 유일한 <a href="/wiki/%EA%B5%90%ED%9A%8C%ED%95%99%EC%9E%90" title="교회학자">교회학자</a>(Doctor Ecclesiae)이다. 참고로 <a href="/wiki/%EB%8C%80%ED%95%9C%EC%84%B1%EA%B3%B5%ED%9A%8C" title="대한성공회">대한성공회</a> 대학로교회의 수호<a class="mw-disambig" href="/wiki/%EC%84%B1%EC%9D%B8" title="성인">성인</a>은 성 베다이다.</p>
