<p><b>베다드 이비셰비치</b>(Vedad Ibišević, <a href="/wiki/1984%EB%85%84" title="1984년">1984년</a> <a href="/wiki/8%EC%9B%94_6%EC%9D%BC" title="8월 6일">8월 6일</a> ~ )는 <a href="/wiki/%EB%B3%B4%EC%8A%A4%EB%8B%88%EC%95%84_%ED%97%A4%EB%A5%B4%EC%B2%B4%EA%B3%A0%EB%B9%84%EB%82%98" title="보스니아 헤르체고비나">보스니아 헤르체고비나</a>의 축구 선수로, 현재 <a href="/wiki/%ED%91%B8%EC%8A%A4%EB%B0%9C-%EB%B6%84%EB%8D%B0%EC%8A%A4%EB%A6%AC%EA%B0%80" title="푸스발-분데스리가">독일 분데스리가</a>의 <a href="/wiki/%ED%97%A4%EB%A5%B4%ED%83%80_BSC" title="헤르타 BSC">헤르타 BSC</a> 소속이다.</p>
<p>미국의 <a class="new" href="/w/index.php?title=%EC%84%B8%EC%9D%B8%ED%8A%B8%EB%A3%A8%EC%9D%B4%EC%8A%A4_%EB%8C%80%ED%95%99%EA%B5%90&amp;action=edit&amp;redlink=1" title="세인트루이스 대학교 (없는 문서)">세인트루이스 대학교</a>를 나온 이비셰비치는 <a href="/wiki/2004%EB%85%84" title="2004년">2004년</a>, <a class="mw-redirect" href="/wiki/%EB%A5%B4%EC%83%B9%ED%94%BC%EC%98%A4%EB%82%98" title="르샹피오나">르샹피오나</a> 팀들 중 하나인 <a href="/wiki/%ED%8C%8C%EB%A6%AC_%EC%83%9D%EC%A0%9C%EB%A5%B4%EB%A7%B9_FC" title="파리 생제르맹 FC">파리 생제르맹</a>에 입단하였다. 그러나, 데뷔 시즌에는 4경기 무득점에 그쳐, 2004-05 시즌에 프랑스의 또 다른 팀인 디종 FCO로 이적하게 된다.</p>
<p>디종에서 2004-05시즌 12경기 4득점, 2005-06시즌 21경기 6득점을 기록, <a class="mw-redirect" href="/wiki/%EB%8F%85%EC%9D%BC_%EB%B6%84%EB%8D%B0%EC%8A%A4%EB%A6%AC%EA%B0%80" title="독일 분데스리가">독일 분데스리가</a>의 알레마니아 아헨을 거쳐, 당시 <a class="mw-redirect" href="/wiki/2_%EB%B6%84%EB%8D%B0%EC%8A%A4%EB%A6%AC%EA%B0%80" title="2 분데스리가">2 분데스리가</a> 소속이었던 <a href="/wiki/TSG_1899_%ED%98%B8%ED%8E%9C%ED%95%98%EC%9E%84" title="TSG 1899 호펜하임">TSG 1899 호펜하임</a>으로 이적하였다. 이적 초기에는 그리 주목을 받지 못하던 공격수였으나, 승격 시즌인 2008-2009 시즌에는 팀의 주축 공격수로 활약하며 전반기에만 18골을 몰아쳐 호펜하임의 리그 1위라는 돌풍을 이끌었으나, 윈터브레이크 시기에 치명적인 무릎 부상으로 아웃되며 시즌을 마감했고, 그의 부재에 덩달아 리그 1위를 달리던 호펜하임은, 극도의 부진을 보이며 <a class="mw-redirect" href="/wiki/%EC%9C%A0%EB%A1%9C%ED%8C%8C%EB%A6%AC%EA%B7%B8" title="유로파리그">유로파리그</a> 진출에도 실패하고 말았으며, 그의 성적은 17경기 18골로 분데스리가 득점 5위에 올랐다.</p>
