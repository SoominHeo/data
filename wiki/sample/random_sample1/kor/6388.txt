<h2><span class="mw-headline" id=".EC.A0.80.EC.9E.90">저자</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%B2%A0%EB%93%9C%EB%A1%9C%EC%9D%98_%EC%B2%AB%EC%A7%B8_%ED%8E%B8%EC%A7%80&amp;action=edit&amp;section=1" title="부분 편집: 저자">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>첫 문장에서 저자는 스스로 "예수의 사도 베드로"라고 밝혔으며 <a class="new" href="/w/index.php?title=%EC%9D%B4%EB%A0%88%EB%82%98%EC%9A%B0%EC%8A%A4&amp;action=edit&amp;redlink=1" title="이레나우스 (없는 문서)">이레나우스</a>(140~203), <a class="mw-redirect" href="/wiki/%ED%84%B0%ED%88%B4%EB%A6%AC%EC%95%88" title="터툴리안">터툴리안</a>(150~222), 알렉산드리아의 <a class="new" href="/w/index.php?title=%ED%81%B4%EB%A0%88%EB%A9%98%ED%8A%B8&amp;action=edit&amp;redlink=1" title="클레멘트 (없는 문서)">클레멘트</a>(155-215) 그리고 알렉산드리아의 <a class="mw-redirect" href="/wiki/%EC%98%A4%EB%A6%AC%EA%B2%90" title="오리겐">오리겐</a>(185-253) 등 초기의 교부들도 그대로 받아들였다. 그러나 <a href="/wiki/19%EC%84%B8%EA%B8%B0" title="19세기">19세기</a> 이래로 <a class="mw-redirect" href="/wiki/%EA%B0%88%EB%A6%B4%EB%A6%AC_%ED%98%B8%EC%88%98" title="갈릴리 호수">갈릴래아 호수</a>의 어부인 <a class="mw-redirect" href="/wiki/%ED%8E%98%ED%8A%B8%EB%A3%A8%EC%8A%A4" title="페트루스">사도 베드로</a>를 저자가 아니라는 주장이 대두되어 왔다.</p>
<ul>
<li>베드로 전서에는 <a class="mw-redirect" href="/wiki/%EA%B0%88%EB%A6%B4%EB%A6%AC" title="갈릴리">갈릴리</a> 태생인 <a href="/wiki/%EB%B2%A0%EB%93%9C%EB%A1%9C" title="베드로">베드로</a>가 접할 수 없는 세련된 도시 <a href="/wiki/%EA%B7%B8%EB%A6%AC%EC%8A%A4%EC%96%B4" title="그리스어">그리스어</a>를 사용하고 있다.</li>
<li>저자가 역사적인 나사렛 예수(<a href="/wiki/%EC%97%AD%EC%82%AC%EC%A0%81_%EC%98%88%EC%88%98" title="역사적 예수">역사적 예수</a>)를 만났다는 것을 의미하는 단서가 없다.</li>
</ul>
<p>저작연대는 주후 60년에서 68년경으로 추정된다.</p>
<h2><span class="mw-headline" id=".EB.8C.80.EC.83.81">대상</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%B2%A0%EB%93%9C%EB%A1%9C%EC%9D%98_%EC%B2%AB%EC%A7%B8_%ED%8E%B8%EC%A7%80&amp;action=edit&amp;section=2" title="부분 편집: 대상">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>책의 첫머리에 <a class="new" href="/w/index.php?title=%EB%B3%B8%EB%8F%84&amp;action=edit&amp;redlink=1" title="본도 (없는 문서)">본도</a>, <a href="/wiki/%EA%B0%88%EB%9D%BC%ED%8B%B0%EC%95%84" title="갈라티아">갈라티아</a>, <a href="/wiki/%EC%B9%B4%ED%8C%8C%EB%8F%84%ED%82%A4%EC%95%84" title="카파도키아">카파도키아</a>, <a href="/wiki/%EC%95%84%EC%8B%9C%EC%95%84" title="아시아">아시아</a>와 <a class="new" href="/w/index.php?title=%EB%B9%84%EB%91%90%EB%8B%88%EC%95%84&amp;action=edit&amp;redlink=1" title="비두니아 (없는 문서)">비두니아</a>에 흩어진 나그네에게 편지를 쓰다며, <a class="mw-redirect" href="/wiki/%EC%86%8C%EC%95%84%EC%8B%9C%EC%95%84" title="소아시아">소아시아</a>의 지방 중 다섯 지방이 명시되어 있다. 이 <a class="mw-redirect" href="/wiki/%EC%84%9C%ED%95%9C" title="서한">서한</a>은 "널리 흩어진 이방인"들에게 씌어졌으나, 내용을 볼 때 <a href="/wiki/%EB%94%94%EC%95%84%EC%8A%A4%ED%8F%AC%EB%9D%BC" title="디아스포라">디아스포라</a> <a href="/wiki/%EC%9C%A0%EB%8C%80%EC%9D%B8" title="유대인">유대인</a>보다는 이방인들을 위하여 쓰여진 것으로 판단 된다. 사도행전 16:6-7과 18:23에 따르면 이 지역은 바오로의하여 복음이 전해진 곳이다.</p>
<h2><span class="mw-headline" id=".EB.82.B4.EC.9A.A9">내용</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%B2%A0%EB%93%9C%EB%A1%9C%EC%9D%98_%EC%B2%AB%EC%A7%B8_%ED%8E%B8%EC%A7%80&amp;action=edit&amp;section=3" title="부분 편집: 내용">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<ul>
<li>1장 : 인사, 희망에 대한 감사, 희망에 합당한 거룩한 생활, 신자 생활 (2장으로 이어짐.)</li>
<li>2장 : 신자 생활, 교회의 기초와 사명, 신자와 이교인, 선량한 국민, 충실한 하인</li>
<li>3장 : 아내와 남편, 참다운 신앙 공동체, 박해에 맞선 확신, 그리스도의 승리</li>
<li>4장 : 참그리스도인, 종말과 공동체 생활, 그리스도인이 받는 고난</li>
<li>5장 : 지도자들의 의무, 겸손과 깨어 있음, 끝 인사</li>
</ul>
<ol>
<li>박해에도 불구하며 확고한 신앙과 인내를 지킬 것 (1-2:10)</li>
<li>그리스도인의 성스러운 삶의 실천적 의무 (2:11-3:13)</li>
<li>그리스도의 모범과 타 이유들을 인내와 성스러움에서 그 근원을 찾아내며 (3:14-4:19)</li>
<li>황제 <a href="/wiki/%EB%84%A4%EB%A1%9C" title="네로">네로</a>의 <a href="/wiki/%EA%B8%B0%EB%8F%85%EA%B5%90" title="기독교">기독교</a>박해로 고통받는 교우들에 대한 충고 (5장)으로 끝낸다.</li>
</ol>
<p>이 서한은 <a class="mw-redirect" href="/wiki/%EB%B0%94%EC%9A%B8%EB%A1%9C" title="바울로">바울로</a>의 교리를 확인하는 것을 목표로 여러가지 방법으로 통해 자신을 사도 바울로와 같은 진영에 위치시킨다.</p>
<h2><span class="mw-headline" id=".ED.8E.B8.EC.A7.80.EC.9D.98_.EC.A7.84.EC.9C.84">편지의 진위</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%B2%A0%EB%93%9C%EB%A1%9C%EC%9D%98_%EC%B2%AB%EC%A7%B8_%ED%8E%B8%EC%A7%80&amp;action=edit&amp;section=4" title="부분 편집: 편지의 진위">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>초기부터 <a href="/wiki/%EB%B2%A0%EB%93%9C%EB%A1%9C%EC%9D%98_%EB%91%98%EC%A7%B8_%ED%8E%B8%EC%A7%80" title="베드로의 둘째 편지">베드로의 둘째 편지</a>는 의심되어 왔지만 첫째 편지는 신약성경의 정경(canon)으로 받아들여져 왔다.</p>
<h2><span class="mw-headline" id=".EC.98.88.EC.88.98.EC.9D.98_.EC.A7.80.EC.98.A5_.ED.95.98.EA.B0.95">예수의 지옥 하강</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%B2%A0%EB%93%9C%EB%A1%9C%EC%9D%98_%EC%B2%AB%EC%A7%B8_%ED%8E%B8%EC%A7%80&amp;action=edit&amp;section=5" title="부분 편집: 예수의 지옥 하강">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>베드로전서 3:19-20과 베드로전서 4:6에는 예수의 지옥 하강이 기록되어 있는데 이것은 유럽의 중세 문학에 많은 영향을 끼쳤다. 루터교의 신앙고백인 <a class="new" href="/w/index.php?title=%ED%98%91%ED%99%94%EC%8B%A0%EC%A1%B0&amp;action=edit&amp;redlink=1" title="협화신조 (없는 문서)">협화신조</a>에는 "예수가 장사 후에 지옥으로 내려가서 마귀를 정복하고 지옥의 권세를 깨어뜨리고 마귀로부터 모든 그의 능력을 빼앗았다."라는 귀절이 포함되어 있고 성공회의 사도신경에도 '죽음의 세계에 내려가시어'라는 지옥 하강에 대한 언급이 있다.</p>
