<h1 class="firstHeading" id="firstHeading" lang="ko">베르나르 7세 다르마냐크</h1>

<p><b>베르나르 7세 다르마냐크</b> (Bernard VII d'Armagnac, 1360년 – 1418년 6월 12일)는 <a href="/wiki/%EC%95%84%EB%A5%B4%EB%A7%88%EB%83%90%ED%81%AC_%EB%B0%B1%EC%9E%91" title="아르마냐크 백작">아르마냐크 백작</a>이자 <a class="new" href="/w/index.php?title=%EC%BD%94%EB%84%A4%ED%83%80%EB%B8%94_%EB%93%9C_%ED%94%84%EB%9E%91%EC%8A%A4&amp;action=edit&amp;redlink=1" title="코네타블 드 프랑스 (없는 문서)">코네타블 드 프랑스</a>이다. 그는 <a class="new" href="/w/index.php?title=%EC%9E%A5_2%EC%84%B8_%EB%8B%A4%EB%A5%B4%EB%A7%88%EB%83%90%ED%81%AC&amp;action=edit&amp;redlink=1" title="장 2세 다르마냐크 (없는 문서)">장 2세 다르마냐크</a>와 잔 드 페리고르(Jeanne de Périgord) 사이에서 태어났다. 그는 1391년에 그의 형제인 <a href="/wiki/%EC%9E%A5_3%EC%84%B8_%EB%8B%A4%EB%A5%B4%EB%A7%88%EB%83%90%ED%81%AC" title="장 3세 다르마냐크">장 3세</a>가 사망하자, 백작 직위를 계승하였다. 1412년에 오랜 싸움 끝에 <a class="new" href="/w/index.php?title=%EC%BD%94%EB%A7%9D%EC%A3%BC_%EB%B0%B1%EC%9E%91&amp;action=edit&amp;redlink=1" title="코망주 백작 (없는 문서)">코망주 백작</a>이 되었다.</p>
<p>1389년 말에 그의 형제가 <a class="new" href="/w/index.php?title=%EB%A7%88%EC%9A%94%EB%A5%B4%EC%B9%B4_%EC%99%95%EA%B5%AD&amp;action=edit&amp;redlink=1" title="마요르카 왕국 (없는 문서)">마요르카 왕국</a>을 요구하며, <a class="mw-redirect" href="/wiki/%EC%B9%B4%ED%83%88%EB%A3%A8%EB%83%90" title="카탈루냐">카탈루냐</a> 북부를 침공했을 때, 왕국의 대륙 영토(<a class="new" href="/w/index.php?title=%EB%A3%A8%EC%8B%9C%EC%9A%A9_%EB%B0%B1%EA%B5%AD&amp;action=edit&amp;redlink=1" title="루시용 백국 (없는 문서)">루시용 백국</a>) 포위를 하면서, 베르나르는 그의 군대 일부를 보내주었다.</p>
<p>베르나르의 아내는 <a href="/wiki/%EB%B2%A0%EB%A6%AC_%EA%B3%B5%EC%9E%91_%EC%9E%A5" title="베리 공작 장">베리 공작 장</a>과 사보이아 백작 <a href="/wiki/%EC%95%84%EB%A9%94%EB%8D%B0%EC%98%A4_8%EC%84%B8_%EB%94%94_%EC%82%AC%EB%B3%B4%EC%9D%B4%EC%95%84" title="아메데오 8세 디 사보이아">아메데오 8세</a>의 미망인 사이에 태어난 <a class="new" href="/w/index.php?title=%EB%B3%B8%EB%8A%90_%EB%93%9C_%EB%B2%A0%EB%A6%AC&amp;action=edit&amp;redlink=1" title="본느 드 베리 (없는 문서)">본느 드 베리</a>다.<sup class="reference" id="cite_ref-1"><a href="#cite_note-1">[1]</a></sup> 그는 <a href="/wiki/%EC%98%A4%EB%A5%BC%EB%A0%88%EC%95%99_%EA%B3%B5%EC%9E%91_%EB%A3%A8%EC%9D%B4_1%EC%84%B8_%EB%93%9C_%EB%B0%9C%EB%A3%A8%EC%95%84" title="오를레앙 공작 루이 1세 드 발루아">오를레앙 공작 루이 1세 드 발루아</a>가 <a class="mw-redirect" href="/wiki/%EB%B0%80%EB%9D%BC%EB%85%B8_%EA%B3%B5%EC%9E%91" title="밀라노 공작">밀라노 공작</a> <a href="/wiki/%EC%9E%94_%EA%B0%88%EB%A0%88%EC%95%84%EC%B4%88_%EB%B9%84%EC%8A%A4%EC%BD%98%ED%8B%B0" title="잔 갈레아초 비스콘티">잔 갈레아초 비스콘티</a>의 딸 <a class="new" href="/w/index.php?title=%EB%B0%9C%EB%A0%8C%ED%8B%B0%EB%82%98_%EB%B9%84%EC%8A%A4%EC%BD%98%ED%8B%B0&amp;action=edit&amp;redlink=1" title="발렌티나 비스콘티 (없는 문서)">발렌티나 비스콘티</a>와 혼인하면서 프랑스 왕실에 처음으로 영향력을 가졌다. 베르나르의 자매인 베아트리스(Beatrice)는 발렌티나의 동생 카를로와 혼인하였다.</p>
<p>1407년에 <a href="/wiki/%EC%98%A4%EB%A5%BC%EB%A0%88%EC%95%99_%EA%B3%B5%EC%9E%91_%EB%A3%A8%EC%9D%B4_1%EC%84%B8_%EB%93%9C_%EB%B0%9C%EB%A3%A8%EC%95%84" title="오를레앙 공작 루이 1세 드 발루아">루이 1세</a>가 암살된 후, 아르마냐크는 오를레앙에 중요성을 두고, 그는 1410년에 <a href="/wiki/%EC%98%A4%EB%A5%BC%EB%A0%88%EC%95%99_%EA%B3%B5%EC%9E%91_%EC%83%A4%EB%A5%BC" title="오를레앙 공작 샤를">오를레앙 공작 샤를</a>에게 그의 딸 <a class="new" href="/w/index.php?title=%EB%B3%B8%EB%8A%90_%EB%8B%A4%EB%A5%B4%EB%A7%88%EB%83%90%ED%81%AC&amp;action=edit&amp;redlink=1" title="본느 다르마냐크 (없는 문서)">본느</a>를 시집보내었다.<sup class="reference" id="cite_ref-2"><a href="#cite_note-2">[2]</a></sup> 베르나르 다르마냐크는 <a href="/wiki/%EB%B6%80%EB%A5%B4%EA%B3%A0%EB%89%B4_%EA%B3%B5%EA%B5%AD" title="부르고뉴 공국">부르고뉴 공국</a>의 <a href="/wiki/%EC%9A%A9%EB%A7%B9%EA%B3%B5_%EC%9E%A5" title="용맹공 장">용맹공 장</a>의 사실상의 반대 세력의 지도자가 되었으며, 그의 세력은 그 결과로 <i>아르마냐크파</i>(Armagnacs)로 불리게 되었다B. (<a href="/wiki/%EC%95%84%EB%A5%B4%EB%A7%88%EB%83%90%ED%81%AC%ED%8C%8C" title="아르마냐크파">아르마냐크파</a> 보기)</p>
<p>그는 1415년에 <a class="new" href="/w/index.php?title=%EC%BD%94%EB%84%A4%ED%83%80%EB%B8%94_%EB%93%9C_%ED%94%84%EB%9E%91%EC%8A%A4&amp;action=edit&amp;redlink=1" title="코네타블 드 프랑스 (없는 문서)">코네타블 드 프랑스</a>(프랑스군 총사령관)이 되었고 1418년까지 <a href="/wiki/%EB%8F%84%ED%8C%BD" title="도팽">도팽</a> 세력의 수장이였다. 그는 1418년 6월 12일 부르고뉴군들을 파리로 이송하던 도중 반란으로 사망하였다.<sup class="reference" id="cite_ref-3"><a href="#cite_note-3">[3]</a></sup></p>

<h2><span class="mw-headline" id=".EA.B0.81.EC.A3.BC">각주</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%B2%A0%EB%A5%B4%EB%82%98%EB%A5%B4_7%EC%84%B8_%EB%8B%A4%EB%A5%B4%EB%A7%88%EB%83%90%ED%81%AC&amp;action=edit&amp;section=2" title="부분 편집: 각주">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
