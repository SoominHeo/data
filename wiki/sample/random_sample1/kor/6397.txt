<p><b>베르나르도 세풀베다 아모르</b>(<span style="font-size: smaller;"><a href="/wiki/%EC%8A%A4%ED%8E%98%EC%9D%B8%EC%96%B4" title="스페인어">스페인어</a>:</span> <span lang="es" xml:lang="es">Bernardo Sepúlveda Amor</span>, <a href="/wiki/1941%EB%85%84" title="1941년">1941년</a> <a href="/wiki/12%EC%9B%94_14%EC%9D%BC" title="12월 14일">12월 14일</a> ~ )는 <a href="/wiki/%EB%A9%95%EC%8B%9C%EC%BD%94" title="멕시코">멕시코</a>의 법학자, 정치가, 외교관이자 <a class="mw-redirect" href="/wiki/%EA%B5%AD%EC%A0%9C_%EC%82%AC%EB%B2%95_%EC%9E%AC%ED%8C%90%EC%86%8C" title="국제 사법 재판소">국제 사법 재판소</a>의 재판관이다. 2006년에 국제 사법 재판소의 재판관으로 선임되었으며, 2012년부터 임기 3년의 제21대 국제 사법 재판소의 부소장으로 근무하고 있다.<sup class="reference" id="cite_ref-1"><a href="#cite_note-1">[1]</a></sup></p>
<h2><span class="mw-headline" id=".EC.83.9D.EC.95.A0">생애</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%B2%A0%EB%A5%B4%EB%82%98%EB%A5%B4%EB%8F%84_%EC%84%B8%ED%92%80%EB%B2%A0%EB%8B%A4_%EC%95%84%EB%AA%A8%EB%A5%B4&amp;action=edit&amp;section=1" title="부분 편집: 생애">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p><a href="/wiki/1941%EB%85%84" title="1941년">1941년</a> <a href="/wiki/12%EC%9B%94_14%EC%9D%BC" title="12월 14일">12월 14일</a>에 <a href="/wiki/%EB%A9%95%EC%8B%9C%EC%BD%94" title="멕시코">멕시코</a> <a class="mw-redirect" href="/wiki/%EB%A9%95%EC%8B%9C%EC%BD%94_%EC%8B%9C%ED%8B%B0" title="멕시코 시티">멕시코 시티</a>에서 태어났다. 1964년에 <a href="/wiki/%EB%A9%95%EC%8B%9C%EC%BD%94_%EA%B5%AD%EB%A6%BD_%EC%9E%90%EC%B9%98_%EB%8C%80%ED%95%99%EA%B5%90" title="멕시코 국립 자치 대학교">멕시코 국립 자치 대학교</a>에서 법학사를 취득하였고, 1966년에 <a href="/wiki/%EC%98%81%EA%B5%AD" title="영국">영국</a> <a href="/wiki/%EC%BC%80%EC%9E%84%EB%B8%8C%EB%A6%AC%EC%A7%80" title="케임브리지">케임브리지</a> <a class="new" href="/w/index.php?title=%ED%80%B8%EC%A6%88_%EC%B9%BC%EB%A6%AC%EC%A7%80&amp;action=edit&amp;redlink=1" title="퀸즈 칼리지 (없는 문서)">퀸즈 칼리지</a>에서 국제법 디플로마와 법학석사(LL.M.) 학위를 취득하였다. <a class="new" href="/w/index.php?title=%EB%A9%95%EC%8B%9C%EC%BD%94_%EB%8C%80%ED%95%99%EA%B5%90&amp;action=edit&amp;redlink=1" title="멕시코 대학교 (없는 문서)">멕시코 대학교</a>(El Colegio de México)에서 국제법과 국제 기구 담당 교수로 재직하였고, 마티아스 로메로 연구소(Matías Romero Institute), 경제연구교육센터(Centro de Investigación y Docencia Económicas)와 멕시코 국립 자치 대학교에서도 강의를 하였다.</p>
<p>1982년 3월 16일부터 11월 30일까지 주미 대사로 근무하였고, 1982년부터 1988년까지 <a href="/wiki/%EB%AF%B8%EA%B2%94_%EB%8D%B0_%EB%9D%BC_%EB%A7%88%EB%93%9C%EB%A6%AC%EB%93%9C" title="미겔 데 라 마드리드">미겔 데 라 마드리드</a> 대통령의 내각에서 외무장관을 지냈다. 외무장관 재직시에 <a href="/wiki/%EC%A4%91%EC%95%99%EC%95%84%EB%A9%94%EB%A6%AC%EC%B9%B4" title="중앙아메리카">중앙아메리카</a>의 평화 구축을 위한 <a class="new" href="/w/index.php?title=%EC%BD%98%ED%83%80%EB%8F%84%EB%9D%BC_%EA%B7%B8%EB%A3%B9&amp;action=edit&amp;redlink=1" title="콘타도라 그룹 (없는 문서)">콘타도라 그룹</a>(Contadora Group)을 설립하는데 공헌하였고, 중남미 국가의 정책협의체 <a class="new" href="/w/index.php?title=%EB%A6%AC%EC%9A%B0_%EA%B7%B8%EB%A3%B9&amp;action=edit&amp;redlink=1" title="리우 그룹 (없는 문서)">리우 그룹</a>(Rio Group)의 전신 오초 그룹(Grupo de Ocho)의 설립에도 노력하였다. 이와 같은 공헌에 대하여 1984년에 프린시페 데 아스투리아스 재단이 시상하는 <a href="/wiki/%ED%94%84%EB%A6%B0%EC%8B%9C%ED%8E%98_%EB%8D%B0_%EC%95%84%EC%8A%A4%ED%88%AC%EB%A6%AC%EC%95%84%EC%8A%A4_%EC%83%81" title="프린시페 데 아스투리아스 상">프린시페 데 아스투리아스 상</a>(Premios Príncipe de Asturias)을 수여받았고, 1985년에는 <a class="mw-redirect" href="/wiki/%EA%B5%AD%EC%A0%9C_%EC%97%B0%ED%95%A9_%EA%B5%90%EC%9C%A1_%EA%B3%BC%ED%95%99_%EB%AC%B8%ED%99%94_%EA%B8%B0%EA%B5%AC" title="국제 연합 교육 과학 문화 기구">국제 연합 교육 과학 문화 기구</a>에서 <a class="new" href="/w/index.php?title=%EA%B5%AD%EC%A0%9C_%EC%8B%9C%EB%AA%AC_%EB%B3%BC%EB%A6%AC%EB%B0%94%EB%A5%B4_%EC%83%81&amp;action=edit&amp;redlink=1" title="국제 시몬 볼리바르 상 (없는 문서)">국제 시몬 볼리바르 상</a>(International Simón Bolivar Prize)을 수여받았다. 1989년부터 1993년까지 주영대사로 근무하였고, 1996년에는 <a class="mw-redirect" href="/wiki/%EA%B5%AD%EC%A0%9C_%EC%97%B0%ED%95%A9" title="국제 연합">국제 연합</a> <a href="/wiki/%EA%B5%AD%EC%A0%9C%EB%B2%95_%EC%9C%84%EC%9B%90%ED%9A%8C" title="국제법 위원회">국제법 위원회</a>의 위원으로 선출되었으며, 2001년에 동 위원회의 위원으로 재선되었다. 2005년 11월 7일에 임기 9년의 국제 사법 재판소의 재판관으로 선출되어 2006년 2월 6일부터 재판관으로 재직하고 있으며, 2012년 2월 6일부터 임기 3년의 국제 사법 재판소의 부소장으로 근무하고 있다.</p>
