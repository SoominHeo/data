<p><b>베리 고디 주니어</b>(Berry Gordy Jr., <a href="/wiki/1929%EB%85%84" title="1929년">1929년</a> <a href="/wiki/11%EC%9B%94_28%EC%9D%BC" title="11월 28일">11월 28일</a> ~ )는 성공적인 <a href="/wiki/%EB%AF%B8%EA%B5%AD" title="미국">미국</a>의 <a class="mw-disambig" href="/wiki/%EB%A0%88%EC%BD%94%EB%93%9C" title="레코드">레코드</a>, <a href="/wiki/%EC%98%81%ED%99%94" title="영화">영화</a> <a class="mw-disambig" href="/wiki/%ED%94%84%EB%A1%9C%EB%93%80%EC%84%9C" title="프로듀서">프로듀서</a> 관리자이며, <a href="/wiki/%EB%94%94%ED%8A%B8%EB%A1%9C%EC%9D%B4%ED%8A%B8" title="디트로이트">디트로이트</a>에서 <a class="mw-redirect" href="/wiki/%EB%AA%A8%ED%83%80%EC%9A%B4_%EB%A0%88%EC%BD%94%EB%93%9C" title="모타운 레코드">모타운</a> 레코드사를 창립하였다. 모타운 레코드사는 많은 톱 <a href="/wiki/%EC%95%84%ED%94%84%EB%A6%AC%EC%B9%B4%EA%B3%84_%EB%AF%B8%EA%B5%AD%EC%9D%B8" title="아프리카계 미국인">아프리카계 미국인</a> 가수들, 특히 <a class="mw-disambig" href="/wiki/%EC%86%8C%EC%9A%B8" title="소울">소울</a>와 <a href="/wiki/%EB%A6%AC%EB%93%AC_%EC%95%A4_%EB%B8%94%EB%A3%A8%EC%8A%A4" title="리듬 앤 블루스">R&amp;B</a> 장르를 녹음하였다. 부드럽고 곡조가 외우기 쉬운 모타운 사운드의 인기는 흑인들의 음악을 미국 <a href="/wiki/%ED%8C%9D_%EC%9D%8C%EC%95%85" title="팝 음악">팝 음악</a>의 주류에 가져왔다.</p>
<p>디트로이트에서 출생하여 <a class="mw-redirect" href="/wiki/%ED%8F%AC%EB%93%9C_%EC%9E%90%EB%8F%99%EC%B0%A8_%ED%9A%8C%EC%82%AC" title="포드 자동차 회사">포드 자동차 회사</a> 공장에서 일하였고 <a href="/wiki/1959%EB%85%84" title="1959년">1959년</a> 탬러 레코드사를 설립하기 전에 가수 <a href="/wiki/%EC%9E%AC%ED%82%A4_%EC%9C%8C%EC%8A%A8" title="재키 윌슨">재키 윌슨</a>의 곡들을 작사하였다. 탬러 레코드사는 <a href="/wiki/1960%EB%85%84" title="1960년">1960년</a> 모타운 레코드사로 합병되었다. 그해에 모타운 레코드사는 그 첫 국내 히트곡 "Shop Around"가 미라클스에 의하여 녹음되었고, 고디와 미라클즈의 리드 싱어 <a href="/wiki/%EC%8A%A4%EB%AA%A8%ED%82%A4_%EB%A1%9C%EB%B9%88%EC%8A%A8" title="스모키 로빈슨">스모키 로빈슨</a>과 함께 공동 작사되었다. 다른 모타운의 음악가들은 <a href="/wiki/%EC%8A%A4%ED%8B%B0%EB%B9%84_%EC%9B%90%EB%8D%94" title="스티비 원더">스티비 원더</a>, <a href="/wiki/%ED%8F%AC_%ED%83%91%EC%8A%A4" title="포 탑스">포탑스</a>, <a class="mw-redirect" href="/wiki/%EC%88%98%ED%94%84%EB%A6%BC%EC%8A%A4" title="수프림스">수프림스</a>, <a href="/wiki/%ED%85%9C%ED%85%8C%EC%9D%B4%EC%85%98%EC%8A%A4" title="템테이션스">템테이션스</a>, <a href="/wiki/%EB%A7%88%EB%B9%88_%EA%B2%8C%EC%9D%B4" title="마빈 게이">마빈 게이</a>, <a href="/wiki/%EB%8B%A4%EC%9D%B4%EC%95%A0%EB%82%98_%EB%A1%9C%EC%8A%A4" title="다이애나 로스">다이애나 로스</a>, <a class="mw-redirect" href="/wiki/%EC%9E%AD%EC%8A%A8_%ED%8C%8C%EC%9D%B4%EB%B8%8C" title="잭슨 파이브">잭슨 파이브</a> 등을 포함하였다.</p>
<p><a href="/wiki/1972%EB%85%84" title="1972년">1972년</a> 고디는 모타운의 운영을 <a href="/wiki/%EB%A1%9C%EC%8A%A4%EC%95%A4%EC%A0%A4%EB%A0%88%EC%8A%A4" title="로스앤젤레스">로스앤젤레스</a>로 이전시켰다. <a href="/wiki/1975%EB%85%84" title="1975년">1975년</a> 다이애나 로스가 출연한 영화 《마호가니》를 감독하였고, 《블루스를 부르는 여인들》(1972), 《올스타의 긴 여행》(1976)와 《라스트 드래곤즈》(1988) 등을 제작하였다. <a href="/wiki/1988%EB%85%84" title="1988년">1988년</a> 모타운 레코드사를 팔았다.</p>
