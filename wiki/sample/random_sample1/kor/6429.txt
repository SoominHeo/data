<p><b>베스트도이체 알게마이네 차이퉁</b>(<span style="font-size: smaller;"><a href="/wiki/%EB%8F%85%EC%9D%BC%EC%96%B4" title="독일어">독일어</a>:</span> <span lang="de" xml:lang="de">Westdeutsche Allgemeine Zeitung</span>, 줄여서 <b>WAZ</b>)은 <a href="/wiki/%EB%8F%85%EC%9D%BC" title="독일">독일</a> <a href="/wiki/%EC%97%90%EC%84%BC" title="에센">에센</a>에서 발행되는 일간 <a href="/wiki/%EC%8B%A0%EB%AC%B8" title="신문">신문</a>이다. WAZ미디어그룹에 속하며, <a href="/wiki/1948%EB%85%84" title="1948년">1948년</a> 4월 3일 창간되었다. 베스트도이체 알게마이네 차이퉁은 독일의 가장 큰 지역신문이다. <a href="/wiki/%EC%97%90%EC%84%BC" title="에센">에센</a>과 <a href="/wiki/%EB%B3%B4%ED%9B%94" title="보훔">보훔</a>, <a href="/wiki/%EA%B2%94%EC%A0%A0%ED%82%A4%EB%A5%B4%ED%97%A8" title="겔젠키르헨">겔젠키르헨</a>, 두이스부르크, <a href="/wiki/%EC%98%A4%EB%B2%84%ED%95%98%EC%9A%B0%EC%A0%A0" title="오버하우젠">오버하우젠</a>, <a class="mw-redirect" href="/wiki/%EB%AE%90%ED%95%98%EC%9E%84" title="뮐하임">뮐하임</a>에서는 일간신문으로서 가장 많은 독자를 보유하고 있다.</p>

<p>베스트도이체 알게마이네 차이퉁의 본사는 에센과 부분적으로 도로트문트에도 있다. WAZ는 루어지방 전역에 걸쳐 28개의 지방판이 발행된다. 발행출판사는 루어지방 신문출판사 GmbH &amp; Co. 에센 KG이다. 총 320명의 <a href="/wiki/%ED%8E%B8%EC%A7%91%EC%9E%90" title="편집자">편집자</a>와 <a class="mw-redirect" href="/wiki/%EC%82%AC%EC%A7%84%EC%9E%91%EA%B0%80" title="사진작가">사진작가</a>가 근무하고 있으며, 고용형태는 상이하다. 에센에 위치한 본사 외에는 23개의 지역본부가있다. 2009년 7월 기준 397 145부가 <a href="/wiki/%ED%8C%90%EB%A7%A4" title="판매">판매</a>되었다. <sup class="reference" id="cite_ref-1"><a href="#cite_note-1">[1]</a></sup></p>
<h2><span class="mw-headline" id=".EC.97.AD.EC.82.AC">역사</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%B2%A0%EC%8A%A4%ED%8A%B8%EB%8F%84%EC%9D%B4%EC%B2%B4_%EC%95%8C%EA%B2%8C%EB%A7%88%EC%9D%B4%EB%84%A4_%EC%B0%A8%EC%9D%B4%ED%89%81&amp;action=edit&amp;section=2" title="부분 편집: 역사">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>베스트도이체 알게마이네 차이퉁은 1948년 4월 3일 창간되었으며, 핵심인물은 에리히 브로스트와 야콤 풍케였다. 베스트도이체 알게마이네 차이퉁이 자리를 잡으면서 WAZ 미디어 그룹이 탄생하였다. WAZ 미디어 그룹은 해외로도 사업을 확장하였다. <a class="mw-redirect" href="/wiki/%EB%8C%80%ED%91%9C%EC%9D%B4%EC%82%AC" title="대표이사">대표이사</a>는 보도 홈바흐와 크리스티안 닌하우스다. <sup class="reference" id="cite_ref-2"><a href="#cite_note-2">[2]</a></sup></p>
<p>베스트도이체 알게마이네 차이퉁은 <a class="mw-disambig" href="/wiki/%EB%A0%88%EC%9D%B4%EC%95%84%EC%9B%83" title="레이아웃">레이아웃</a>과 <a href="/wiki/%ED%98%95%EC%8B%9D" title="형식">형식</a>에서 매우 크게 변하였다. 특히 여러번에 걸친 <a href="/wiki/%ED%91%9C%EC%A7%80" title="표지">표지</a>의 변화가 두드러진다. 예를들어 표지와 두 번째 페이지에 연예인에 관련된 부분이 추가되었다. 또한 <a href="/wiki/%EC%82%AC%ED%9A%8C" title="사회">사회</a> 분야에 관련된 <a class="mw-disambig" href="/wiki/%EB%B3%B4%EB%8F%84" title="보도">보도</a>가 증가하는 반면, <a href="/wiki/%EC%A0%95%EC%B9%98" title="정치">정치</a> <a href="/wiki/%EA%B2%BD%EC%A0%9C" title="경제">경제</a> 분야에 대한 보도는 감소하였다.</p>
<p>2005년부터 울리히 라이츠가 편집장을 맡고 있다. 라이츠는 이전 라이니쉐 포스트(Rheinische Post)에서 편집장을 지내왔었다.</p>
<h2><span class="mw-headline" id=".EB.85.BC.EB.9E.80">논란</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%B2%A0%EC%8A%A4%ED%8A%B8%EB%8F%84%EC%9D%B4%EC%B2%B4_%EC%95%8C%EA%B2%8C%EB%A7%88%EC%9D%B4%EB%84%A4_%EC%B0%A8%EC%9D%B4%ED%89%81&amp;action=edit&amp;section=3" title="부분 편집: 논란">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>2006년에 베스트도이체 알게마이네 차이퉁은 북루어지방에 위치한 지역본부를 운영중지하기 결정하였다. 그리고 바우어 출판사를 포함한 다른 <a class="mw-redirect" href="/wiki/%EC%8B%A0%EB%AC%B8%EC%82%AC" title="신문사">신문사</a>들이 베스트도이체 알게마이네 차이퉁이 지역본부를 보유한 지역분부를 운영중지하였다. 이에 따라 출판사들이 발행지역을 <a href="/wiki/%EB%8B%B4%ED%95%A9" title="담합">담합</a>하여 서로 나누어가진다는 논란이 일었다. 이는 독자의 <a href="/wiki/%EA%B6%8C%EB%A6%AC" title="권리">권리</a>를 침해한다는 비판이 가해졌다. 그러나 베스트도이체 알게마이네 차이퉁은 지역담합을 부인하였다. <sup class="reference" id="cite_ref-3"><a href="#cite_note-3">[3]</a></sup> 베스트도이체 알게마이네 차이퉁과 연루된 다른 <a href="/wiki/%EC%B6%9C%ED%8C%90%EC%82%AC" title="출판사">출판사</a>의 <a href="/wiki/%EA%B8%B0%EC%9E%90_(%EC%A7%81%EC%97%85)" title="기자 (직업)">기자</a>들은 항의시위를 하였지만 베스트도이체 알게마이네 차이퉁에는 해당 <a class="mw-disambig" href="/wiki/%EC%8B%9C" title="시">시</a>위가 보도되지 않았다.</p>
