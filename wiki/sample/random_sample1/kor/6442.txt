<p><b>베일리 제임스 카길</b> (Baily James Cargill , <a href="/wiki/1995%EB%85%84" title="1995년">1995년</a> <a href="/wiki/10%EC%9B%94_13%EC%9D%BC" title="10월 13일">10월 13일</a> ~ )는 <a href="/wiki/%EC%9E%89%EA%B8%80%EB%9E%9C%EB%93%9C" title="잉글랜드">잉글랜드</a>의 <a href="/wiki/%EC%B6%95%EA%B5%AC" title="축구">축구</a> 선수으로, 현재 <a href="/wiki/%EC%A7%88%EB%A7%81%EC%97%84_FC" title="질링엄 FC">질링엄</a>의 선수이며, 포지션은 센터백 이다. <a href="/wiki/AFC_%EB%B3%B8%EB%A8%B8%EC%8A%A4" title="AFC 본머스">AFC 본머스</a>에서 <a href="/wiki/%EC%B9%BC%EB%9F%BC_%EC%9C%8C%EC%8A%A8" title="칼럼 윌슨">칼럼 윌슨</a>, <a href="/wiki/%EB%9D%BC%EC%9D%B4%EC%96%B8_%ED%94%84%EB%A0%88%EC%9D%B4%EC%A0%80" title="라이언 프레이저">라이언 프레이저</a>과 더불어 발굴해낸 최고의 유스 자원이 될것으로 평가 받는다.</p>
<h2><span class="mw-headline" id=".EA.B2.BD.EB.A0.A5">경력</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%B2%A0%EC%9D%BC%EB%A6%AC_%EC%B9%B4%EA%B8%B8&amp;action=edit&amp;section=1" title="부분 편집: 경력">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p><a href="/wiki/2003%EB%85%84" title="2003년">2003년</a> 지금 현재의 잉글랜드 <a href="/wiki/%ED%94%84%EB%A6%AC%EB%AF%B8%EC%96%B4%EB%A6%AC%EA%B7%B8" title="프리미어리그">프리미어리그</a>의 <a class="mw-redirect" href="/wiki/%EB%A0%88%EC%8A%A4%ED%84%B0_%EC%8B%9C%ED%8B%B0" title="레스터 시티">레스터 시티</a>의 유소년 팀에 입단하였다. 그후 3년 뒤인 <a href="/wiki/2006%EB%85%84" title="2006년">2006년</a> 영국내에서도 최고의 유스 시스템을 갖추기로 소문이 자자한 <a href="/wiki/%EC%82%AC%EC%9A%B0%EC%83%98%ED%94%84%ED%84%B4" title="사우샘프턴">사우샘프턴</a>으로 이적하였다. 그곳의 좋은 유스 시설의 영향으로 좋은 센터백자원이 되는 기초를 닦았다. 그후 <a href="/wiki/2008%EB%85%84" title="2008년">2008년</a> 현재 그의 지금의 소속팀인 <a href="/wiki/AFC_%EB%B3%B8%EB%A8%B8%EC%8A%A4" title="AFC 본머스">본머스</a>과의 유소년 계약을 맺었다. 그후 성인계약을 바로 맺었으나 12-13 시즌은 <a class="new" href="/w/index.php?title=AFC_%ED%86%A0%ED%86%A4&amp;action=edit&amp;redlink=1" title="AFC 토톤 (없는 문서)">AFC 토톤</a>, 2013-2014,에는 <a href="/wiki/%EC%9B%B0%EB%A7%81_%EC%9C%A0%EB%82%98%EC%9D%B4%ED%8B%B0%EB%93%9C_FC" title="웰링 유나이티드 FC">웰링 유나이티드 FC</a>에서 <a href="/wiki/%ED%86%A0%ED%82%A4_%EC%9C%A0%EB%82%98%EC%9D%B4%ED%8B%B0%EB%93%9C_FC" title="토키 유나이티드 FC">토키 유나이티드 FC</a>로 재임대 되는 어처구니 없는일까지 있었다. 주로 하부리그에서 임대경험을 쌓다가 14-15시즌이 시작되기 직전에 드디어 자신의 소속팀인 <a href="/wiki/AFC_%EB%B3%B8%EB%A8%B8%EC%8A%A4" title="AFC 본머스">본머스</a>의 라인업에 들게 되었다. 하지만 리그에서의 출전시간은 없고 주로 컵대회에서 경기를 나왔는데 <a href="/wiki/%EB%A6%AC%EB%B2%84%ED%92%80_FC" title="리버풀 FC">리버풀 FC</a>와 같은 강팀과의 컵대회에서 경기에 나오는것을 보면 그만큼 <a href="/wiki/AFC_%EB%B3%B8%EB%A8%B8%EC%8A%A4" title="AFC 본머스">본머스</a>가 카길에 대한 기대가 크다는 것을 알수가 있다.</p>
<h2><span class="mw-headline" id=".EA.B5.AD.EA.B0.80.EB.8C.80.ED.91.9C_.EA.B2.BD.EB.A0.A5">국가대표 경력</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%B2%A0%EC%9D%BC%EB%A6%AC_%EC%B9%B4%EA%B8%B8&amp;action=edit&amp;section=2" title="부분 편집: 국가대표 경력">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
