<h2><span class="mw-headline" id=".EC.84.B1.EC.A7.88">성질</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%B2%A0%EC%A3%BC_%EC%A0%95%EC%97%AD&amp;action=edit&amp;section=2" title="부분 편집: 성질">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>베주 정역은 <a class="new" href="/w/index.php?title=%EC%B5%9C%EB%8C%80%EA%B3%B5%EC%95%BD%EC%88%98_%EC%A0%95%EC%97%AD&amp;action=edit&amp;redlink=1" title="최대공약수 정역 (없는 문서)">최대공약수 정역</a>(<span style="font-size: smaller;"><a href="/wiki/%EC%98%81%EC%96%B4" title="영어">영어</a>:</span> <span lang="en" xml:lang="en">GCD domain</span>)이자 <a class="mw-redirect" href="/wiki/%ED%94%84%EB%A4%BC%ED%8D%BC_%EC%A0%95%EC%97%AD" title="프뤼퍼 정역">프뤼퍼 정역</a>(<span style="font-size: smaller;"><a href="/wiki/%EC%98%81%EC%96%B4" title="영어">영어</a>:</span> <span lang="en" xml:lang="en">Prüfer domain</span>)이다.</p>
<p>가환환에 대하여, 다음 조건들이 서로 <a href="/wiki/%EB%8F%99%EC%B9%98" title="동치">동치</a>이다.</p>
<ul>
<li><a href="/wiki/%EC%A3%BC_%EC%95%84%EC%9D%B4%EB%94%94%EC%96%BC_%EC%A0%95%EC%97%AD" title="주 아이디얼 정역">주 아이디얼 정역</a>이다.</li>
<li>베주 정역이며 <a href="/wiki/%EC%9C%A0%EC%9D%BC_%EC%9D%B8%EC%88%98_%EB%B6%84%ED%95%B4_%EC%A0%95%EC%97%AD" title="유일 인수 분해 정역">유일 인수 분해 정역</a>이다.</li>
<li>베주 정역이며 <a href="/wiki/%EB%87%8C%ED%84%B0_%ED%99%98" title="뇌터 환">뇌터 환</a>이다.</li>
</ul>
<p>가환환에 대하여, 다음 두 조건이 서로 <a href="/wiki/%EB%8F%99%EC%B9%98" title="동치">동치</a>이다.</p>
<ul>
<li>베주 정역이자 <a href="/wiki/%EA%B5%AD%EC%86%8C%ED%99%98" title="국소환">국소환</a>이다.</li>
<li><a href="/wiki/%EA%B0%92%EB%A7%A4%EA%B9%80%ED%99%98" title="값매김환">값매김환</a>이다.</li>
</ul>
