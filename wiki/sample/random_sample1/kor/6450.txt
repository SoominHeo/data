<h2><span class="mw-headline" id=".EA.B8.B0.EC.9B.90">기원</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%B2%A0%ED%83%80_%EC%9D%B4%EC%8A%A4%EB%9D%BC%EC%97%98&amp;action=edit&amp;section=1" title="부분 편집: 기원">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p><a href="/wiki/%EA%B5%AC%EC%95%BD%EC%84%B1%EA%B2%BD" title="구약성경">구약성경</a> <a href="/wiki/%EC%97%B4%EC%99%95%EA%B8%B0" title="열왕기">열왕기</a> 상에는 <a class="mw-redirect" href="/wiki/%EC%86%94%EB%A1%9C%EB%AA%AC_%EC%99%95" title="솔로몬 왕">솔로몬 왕</a>이 <a class="mw-redirect" href="/wiki/%EC%8B%9C%EB%B0%94%EC%9D%98_%EC%97%AC%EC%99%95" title="시바의 여왕">시바의 여왕</a>과 사랑을 나눈 사실이 기록되어있다. 솔로몬 왕이 시바의 여왕과 <a href="/wiki/%EC%84%B1%EA%B5%90" title="성교">육체적인 사랑</a>을 나누었는지 정신적인 사랑을 나누었는지는 기록되어있지않지만 육체적인 사랑을 나누었을 가능성이 높다고 암시하고있다. 에티오피아의 구전에 따르면 이들 사이에서 태어난 <a class="new" href="/w/index.php?title=%EB%A9%94%EB%84%AC%EB%A6%AC%ED%81%AC_1%EC%84%B8&amp;action=edit&amp;redlink=1" title="메넬리크 1세 (없는 문서)">메넬리크 1세</a>가 베타 이스라엘의 기원이라고한다.</p>
<p>유전자 검사 결과 초기 연구들에서는 베타 이스라엘과 유대인 사이에 연관성이 없는 것으로 나왔지만, 2012년에 나온 연구결과에서 베타 이스라엘은 에티오피아 원주민과 닮았지만 약간의 유대인 혈통을 가진것으로 확인됐다. 오래전에 에티오피아로 건너간 유대인들이 유대교를 전도하는데 성공해 원주민을 유대교로 개종시키고 서로 혼혈되고 동화되어 베타 이스라엘이 형성됐다는 것을 보여준다. 이 결과는 베타 이스라엘들이 (이스라엘에 돌아와 현대 유대교를 받아들이기 전까지) <a href="/wiki/%ED%95%98%EB%88%84%EC%B9%B4" title="하누카">하누카</a>를 쇠지 않았던 이유를 설명한다. <sup class="reference" id="cite_ref-1"><a href="#cite_note-1">[1]</a></sup></p>
<h2><span class="mw-headline" id=".ED.98.84.EB.8C.80">현대</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%B2%A0%ED%83%80_%EC%9D%B4%EC%8A%A4%EB%9D%BC%EC%97%98&amp;action=edit&amp;section=2" title="부분 편집: 현대">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>19세기에 개신교 선교사들이 베타 이스라엘을 기독교로 개종시키기 위해 선교활동을 했다. 개신교로 개종한 베타 이스라엘을 <a href="/wiki/%ED%8C%94%EB%9D%BC%EC%8B%9C_%EB%AC%B4%EB%9D%BC" title="팔라시 무라">팔라시 무라</a>라고한다. <a href="/wiki/%EC%9D%B4%EC%8A%A4%EB%9D%BC%EC%97%98" title="이스라엘">이스라엘</a>이 건국된 후 처음에 이스라엘 정부는 베타 이스라엘을 유대인으로 인정하지 않았다. 몇몇 베타 이스라엘들은 관광비자로 이스라엘에 입국한 뒤에 <a class="mw-redirect" href="/wiki/%EB%B6%88%EB%B2%95%EC%B2%B4%EB%A5%98%EC%9E%90" title="불법체류자">불법체류자</a>로 이스라엘에 정착했다.</p>
<p>1975년에 <a href="/wiki/%EC%9D%B4%EC%B8%A0%ED%95%98%ED%81%AC_%EB%9D%BC%EB%B9%88" title="이츠하크 라빈">이츠하크 라빈</a> 정부는 베타 이스라엘은 <a href="/wiki/%EC%9C%A0%EB%8C%80%EC%9D%B8" title="유대인">유대인</a>이므로 <a href="/wiki/%EA%B7%80%ED%99%98%EB%B2%95" title="귀환법">귀환법</a>의 적용대상이라고 선언했다. <a href="/wiki/%EB%A9%94%EB%82%98%ED%97%B4_%EB%B2%A0%EA%B8%B4" title="메나헴 베긴">메나헴 베긴</a> 총리는 베타 이스라엘이 <a class="new" href="/w/index.php?title=%EC%82%AC%EB%9D%BC%EC%A7%84_10%EC%A7%80%ED%8C%8C&amp;action=edit&amp;redlink=1" title="사라진 10지파 (없는 문서)">사라진 10지파</a>의 후손이라고 선포했다.</p>
<p><a href="/wiki/1974%EB%85%84" title="1974년">1974년</a> <a href="/wiki/9%EC%9B%94_12%EC%9D%BC" title="9월 12일">9월 12일</a> 마르크스주의를 표방하는 <a class="new" href="/w/index.php?title=%EB%8D%B0%EB%A5%B4%EA%B7%B8&amp;action=edit&amp;redlink=1" title="데르그 (없는 문서)">데르그</a>가 황제 <a href="/wiki/%ED%95%98%EC%9D%BC%EB%A0%88_%EC%85%80%EB%9D%BC%EC%8B%9C%EC%97%90_1%EC%84%B8" title="하일레 셀라시에 1세">하일레 셀라시에 1세</a>에 대항한 <a href="/wiki/%EC%97%90%ED%8B%B0%EC%98%A4%ED%94%BC%EC%95%84_%EB%82%B4%EC%A0%84" title="에티오피아 내전">에티오피아 내전</a>이 일어나고 데르그가 에티오피아를 장악하게되자 베타 이스라엘들이 대거 이스라엘로 입국하는 <a href="/wiki/%EC%95%8C%EB%A6%AC%EC%95%BC" title="알리야">알리야</a>가 있었다.</p>
<p>1979년부터 1984년까지 <a href="/wiki/%EB%AA%A8%EC%82%AC%EB%93%9C" title="모사드">모사드</a>는 베타 이스라엘을 수단으로 입국시킨 뒤에 유럽을 거쳐 이스라엘로 수송했다. 1983년에 에티오피아 곤다르 주지사가 국외 이주 규제를 폐지하자 많은 베타 이스라엘들이 <a href="/wiki/%EC%88%98%EB%8B%A8" title="수단">수단</a>을 향해 떠났다. 1985년 1월에 시행된 <a class="new" href="/w/index.php?title=%EB%AA%A8%EC%84%B8_%EC%9E%91%EC%A0%84&amp;action=edit&amp;redlink=1" title="모세 작전 (없는 문서)">모세 작전</a>과 몇주 뒤에 시행된 <a class="new" href="/w/index.php?title=%EC%97%AC%ED%98%B8%EC%88%98%EC%95%84_%EC%9E%91%EC%A0%84&amp;action=edit&amp;redlink=1" title="여호수아 작전 (없는 문서)">여호수아 작전</a>을 통해 수단에 머무르던 베타 이스라엘들이 이스라엘로 입국하였다. 공산주의가 붕괴된 1990년부터 베타 이스라엘들이 에티오피아에서 직접 <a href="/wiki/%EC%9D%B4%EC%8A%A4%EB%9D%BC%EC%97%98" title="이스라엘">이스라엘</a>로 갈 수 있게 되었다. 1991년에 에티오피아의 정정 불안이 심해지자 이스라엘은 <a href="/wiki/%EC%86%94%EB%A1%9C%EB%AA%AC_%EC%9E%91%EC%A0%84" title="솔로몬 작전">솔로몬 작전</a>을 실시하여 베타 이스라엘들을 이스라엘로 구출했다.</p>
<h2><span class="mw-headline" id=".ED.8C.94.EB.9D.BC.EC.8B.9C_.EB.AC.B4.EB.9D.BC_.EB.85.BC.EC.9F.81">팔라시 무라 논쟁</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%B2%A0%ED%83%80_%EC%9D%B4%EC%8A%A4%EB%9D%BC%EC%97%98&amp;action=edit&amp;section=3" title="부분 편집: 팔라시 무라 논쟁">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>기독교로 개종한 베타 이스라엘인 <a href="/wiki/%ED%8C%94%EB%9D%BC%EC%8B%9C_%EB%AC%B4%EB%9D%BC" title="팔라시 무라">팔라시 무라</a>들이 귀환법의 적용대상인지 논쟁이 있다.</p>
<h2><span class="mw-headline" id=".EC.9D.B4.EC.8A.A4.EB.9D.BC.EC.97.98.EC.97.90.EC.84.9C.EC.9D.98_.EC.B0.A8.EB.B3.84">이스라엘에서의 차별</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%B2%A0%ED%83%80_%EC%9D%B4%EC%8A%A4%EB%9D%BC%EC%97%98&amp;action=edit&amp;section=4" title="부분 편집: 이스라엘에서의 차별">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>1996년에는 이스라엘 혈액 당국에서 에티오피아계 유대인들이 <a href="/wiki/%ED%97%8C%ED%98%88" title="헌혈">헌혈</a>한 혈액을 폐기했다는 사실이 드러나 차별 논란이 일었다.<sup class="reference" id="cite_ref-2"><a href="#cite_note-2">[2]</a></sup> 2013년에는 적수정에서 에티오피아에서 태어난 뒤 세살때부터 이스라엘에 거주한 에티오피아계 유대인 여성 국회의원의 헌혈을 거부하여 파장이 일었다.<sup class="reference" id="cite_ref-3"><a href="#cite_note-3">[3]</a></sup></p>
