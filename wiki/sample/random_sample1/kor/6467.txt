<p><b>벤저민 솔로몬 "벤" 카슨</b>(Benjamin Solomon "Ben" Carson, Sr. <a href="/wiki/1951%EB%85%84" title="1951년">1951년</a> <a href="/wiki/9%EC%9B%94_18%EC%9D%BC" title="9월 18일">9월 18일</a> ~ )은 <a href="/wiki/%EB%AF%B8%EA%B5%AD" title="미국">미국</a>의 <a href="/wiki/%EC%8B%A0%EA%B2%BD%EC%99%B8%EA%B3%BC" title="신경외과">신경외과</a> 의사이자 2016년 <a href="/wiki/%EB%AF%B8%EA%B5%AD_%EB%8C%80%ED%86%B5%EB%A0%B9_%EC%84%A0%EA%B1%B0" title="미국 대통령 선거">미국 대통령 선거</a>에서의 <a href="/wiki/%EA%B3%B5%ED%99%94%EB%8B%B9_(%EB%AF%B8%EA%B5%AD)" title="공화당 (미국)">공화당</a> 출신 후보이다.</p>
<p>1984년부터 2013년까지 <a href="/wiki/%EB%A9%94%EB%A6%B4%EB%9E%9C%EB%93%9C_%EC%A3%BC" title="메릴랜드 주">메릴랜드 주</a>의 <a class="new" href="/w/index.php?title=%EC%A1%B4%EC%8A%A4_%ED%99%89%ED%82%A8%EC%8A%A4_%EB%B3%91%EC%9B%90&amp;action=edit&amp;redlink=1" title="존스 홉킨스 병원 (없는 문서)">존스 홉킨스 병원</a>에서 근무했으며, 이후 <a class="new" href="/w/index.php?title=%EB%85%BC%EC%84%A4%EA%B0%80&amp;action=edit&amp;redlink=1" title="논설가 (없는 문서)">논설가</a>, 정치 <a class="mw-redirect" href="/wiki/%ED%8F%89%EB%A1%A0%EA%B0%80" title="평론가">평론가</a> 등으로 활동하며 명성을 얻었다. 또한 신경외과 의사로 활동할 당시 <a href="/wiki/%EA%B2%B0%ED%95%A9_%EC%8C%8D%EB%91%A5%EC%9D%B4" title="결합 쌍둥이">결합 쌍둥이</a>를 성공적으로 분리시킨 것과 <a class="new" href="/w/index.php?title=%EB%87%8C_%EB%B0%9C%EC%9E%91&amp;action=edit&amp;redlink=1" title="뇌 발작 (없는 문서)">뇌 발작</a>을 억제시키는 기술을 연구한 것을 인정받아 2008년 <a href="/wiki/%EB%8C%80%ED%86%B5%EB%A0%B9_%EC%9E%90%EC%9C%A0_%ED%9B%88%EC%9E%A5" title="대통령 자유 훈장">대통령 자유 훈장</a>을 수여받은 바 있다.</p>
<p>2013년 <a class="new" href="/w/index.php?title=%EA%B5%AD%EA%B0%80%EC%A1%B0%EC%B0%AC%EA%B8%B0%EB%8F%84%ED%9A%8C&amp;action=edit&amp;redlink=1" title="국가조찬기도회 (없는 문서)">국가조찬기도회</a>에서의 연설이 언론의 대대적인 관심을 받아 <a href="/wiki/%EB%B3%B4%EC%88%98%EC%A3%BC%EC%9D%98" title="보수주의">보수주의</a> 진영의 주요 인사로 자리잡았으며,<sup class="reference" id="cite_ref-1"><a href="#cite_note-1">[1]</a></sup> 2015년 5월 4일 자신의 고향인 <a href="/wiki/%EB%94%94%ED%8A%B8%EB%A1%9C%EC%9D%B4%ED%8A%B8" title="디트로이트">디트로이트</a>에서 <a href="/wiki/2016%EB%85%84_%EB%AF%B8%EA%B5%AD_%EB%8C%80%ED%86%B5%EB%A0%B9_%EC%84%A0%EA%B1%B0_%EA%B3%B5%ED%99%94%EB%8B%B9_%EC%98%88%EB%B9%84%EA%B2%BD%EC%84%A0" title="2016년 미국 대통령 선거 공화당 예비경선">2016년 미국 대통령 선거 공화당 예비경선</a>에 출마할 것을 선언하였다.<sup class="reference" id="cite_ref-2"><a href="#cite_note-2">[2]</a></sup></p>
