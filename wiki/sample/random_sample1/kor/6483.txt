<p><b>벨기에의 조제핀샤를로트</b> (<span style="font-size: smaller;"><a href="/wiki/%ED%94%84%EB%9E%91%EC%8A%A4%EC%96%B4" title="프랑스어">프랑스어</a>:</span> <span lang="fr" xml:lang="fr">Joséphine-Charlotte of Belgium</span>, <a href="/wiki/1927%EB%85%84" title="1927년">1927년</a> <a href="/wiki/10%EC%9B%94_11%EC%9D%BC" title="10월 11일">10월 11일</a>~<a href="/wiki/2005%EB%85%84" title="2005년">2005년</a> <a href="/wiki/1%EC%9B%94_10%EC%9D%BC" title="1월 10일">1월 10일</a>)는 <a href="/wiki/%EB%A3%A9%EC%85%88%EB%B6%80%EB%A5%B4%ED%81%AC" title="룩셈부르크">룩셈부르크</a>의 대공비이다.</p>
<h2><span class="mw-headline" id=".EC.83.9D.EC.95.A0">생애</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%B2%A8%EA%B8%B0%EC%97%90%EC%9D%98_%EC%A1%B0%EC%A0%9C%ED%95%80%EC%83%A4%EB%A5%BC%EB%A1%9C%ED%8A%B8&amp;action=edit&amp;section=1" title="부분 편집: 생애">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p><a class="mw-redirect" href="/wiki/%EB%B2%A8%EA%B8%B0%EC%97%90%EC%9D%98_%EB%A0%88%EC%98%A4%ED%8F%B4_3%EC%84%B8" title="벨기에의 레오폴 3세">벨기에의 레오폴 3세</a>와 <a href="/wiki/%EC%8A%A4%EC%9B%A8%EB%8D%B4" title="스웨덴">스웨덴</a>의 아스트리드 사이에서 장녀로 태어났다. <a href="/wiki/%EB%B2%A8%EA%B8%B0%EC%97%90%EC%9D%98_%EA%B5%B0%EC%A3%BC" title="벨기에의 군주">벨기에 국왕</a>이었던 <a class="mw-redirect" href="/wiki/%EB%B2%A8%EA%B8%B0%EC%97%90%EC%9D%98_%EB%B3%B4%EB%91%90%EC%95%B5_1%EC%84%B8" title="벨기에의 보두앵 1세">보두앵 1세</a>와 <a class="mw-redirect" href="/wiki/%EB%B2%A8%EA%B8%B0%EC%97%90%EC%9D%98_%EC%95%8C%EB%B2%A0%EB%A5%B4_2%EC%84%B8" title="벨기에의 알베르 2세">알베르 2세</a>는 그녀의 남동생들이다. <a href="/wiki/1953%EB%85%84" title="1953년">1953년</a> <a href="/wiki/4%EC%9B%94_9%EC%9D%BC" title="4월 9일">4월 9일</a> 룩셈부르크의 대공세자 <a class="mw-redirect" href="/wiki/%EB%A3%A9%EC%85%88%EB%B6%80%EB%A5%B4%ED%81%AC%EC%9D%98_%EC%9E%A5" title="룩셈부르크의 장">장</a>과 결혼하였으며, 시어머니 <a class="mw-redirect mw-disambig" href="/wiki/%EC%83%A4%EB%A5%BC%EB%A1%9C%ED%8A%B8" title="샤를로트">샤를로트</a>가 장에게 양위하자, 그녀 역시 남편과 함께 즉위하여 대공비가 되었다. <a href="/wiki/%ED%8F%90%EC%95%94" title="폐암">폐암</a>으로 <a href="/wiki/2005%EB%85%84" title="2005년">2005년</a> <a href="/wiki/1%EC%9B%94_10%EC%9D%BC" title="1월 10일">1월 10일</a>에 77세의 나이로 사망하였다.</p>
