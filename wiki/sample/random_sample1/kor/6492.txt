<p><b>벨레스 시</b>(<span style="font-size: smaller;"><a href="/wiki/%EB%A7%88%EC%BC%80%EB%8F%84%EB%8B%88%EC%95%84%EC%96%B4" title="마케도니아어">마케도니아어</a>:</span> <span lang="mk" xml:lang="mk">Општина Велес</span>)는 <a href="/wiki/%EB%A7%88%EC%BC%80%EB%8F%84%EB%8B%88%EC%95%84_%EA%B3%B5%ED%99%94%EA%B5%AD" title="마케도니아 공화국">마케도니아 공화국</a> 중부에 위치한 지방 자치체로 행정 중심지는 <a href="/wiki/%EB%B2%A8%EB%A0%88%EC%8A%A4_(%EB%8F%84%EC%8B%9C)" title="벨레스 (도시)">벨레스</a>이며 면적은 427.45㎢, 인구는 55,108명(2002년 기준)이다. 서쪽으로는 <a href="/wiki/%EC%B0%A8%EC%8A%88%EC%B9%B4_%EC%8B%9C" title="차슈카 시">차슈카 시</a>와 <a href="/wiki/%EC%A0%A4%EB%A0%88%EB%8B%88%EC%BD%94%EB%B3%B4_%EC%8B%9C" title="젤레니코보 시">젤레니코보 시</a>, 북쪽으로는 <a href="/wiki/%ED%8E%98%ED%8A%B8%EB%A1%9C%EB%B2%A0%EC%B8%A0_%EC%8B%9C" title="페트로베츠 시">페트로베츠 시</a>, 동쪽으로는 <a href="/wiki/%EA%B7%B8%EB%9D%BC%EB%93%9C%EC%8A%A4%EC%BD%94_%EC%8B%9C" title="그라드스코 시">그라드스코 시</a>, <a href="/wiki/%EB%A1%9C%EC%A1%B0%EB%B3%B4_%EC%8B%9C" title="로조보 시">로조보 시</a>, <a href="/wiki/%EC%8A%A4%EB%B2%A0%ED%8B%B0%EB%8B%88%EC%BD%9C%EB%A0%88_%EC%8B%9C" title="스베티니콜레 시">스베티니콜레 시</a>와 접한다.</p>
