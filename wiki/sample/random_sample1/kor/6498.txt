<h2><span class="mw-headline" id=".EC.B4.88.EA.B8.B0_.EC.83.9D.EC.95.A0">초기 생애</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%B2%A8%EB%A6%AC%EC%82%AC%EB%A6%AC%EC%9A%B0%EC%8A%A4&amp;action=edit&amp;section=1" title="부분 편집: 초기 생애">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p><a href="/wiki/%EC%9D%BC%EB%A6%AC%EB%A6%AC%EC%95%84" title="일리리아">일리리아</a>(현재 <a href="/wiki/%EC%9C%A0%EA%B3%A0%EC%8A%AC%EB%9D%BC%EB%B9%84%EC%95%84" title="유고슬라비아">유고슬라비아</a>)에서 태어나 젊어서부터 군에 입대했다. <a href="/wiki/530%EB%85%84" title="530년">530년</a> <a class="new" href="/w/index.php?title=%EB%8B%A4%EB%9D%BC%EC%9D%98_%EC%A0%84%ED%88%AC&amp;action=edit&amp;redlink=1" title="다라의 전투 (없는 문서)">다라의 전투</a>에서 <a href="/wiki/%EC%82%AC%EC%82%B0_%EC%A0%9C%EA%B5%AD" title="사산 제국">사산 제국</a>의 군대와 싸워 승리했으나 <a href="/wiki/531%EB%85%84" title="531년">531년</a> <a href="/wiki/%EC%9C%A0%ED%94%84%EB%9D%BC%ED%85%8C%EC%8A%A4_%EA%B0%95" title="유프라테스 강">유프라테스 강</a>가의 <a class="new" href="/w/index.php?title=%EC%B9%BC%EB%A6%AC%EB%8B%88%EC%BF%B0%EC%9D%98_%EC%A0%84%ED%88%AC&amp;action=edit&amp;redlink=1" title="칼리니쿰의 전투 (없는 문서)">칼리니쿰의 전투</a>에서 완패, 페르시아와 강화를 맺었다. <a href="/wiki/532%EB%85%84" title="532년">532년</a> <a href="/wiki/%EC%BD%98%EC%8A%A4%ED%83%84%ED%8B%B0%EB%85%B8%ED%8F%B4%EB%A6%AC%EC%8A%A4" title="콘스탄티노폴리스">콘스탄티노폴리스</a>에서 장교가 되었다.</p>

<h3><span class="mw-headline" id=".EB.B0.98.EB.8B.AC.EC.A1.B1.EA.B3.BC.EC.9D.98_.EC.A0.84.EC.9F.81">반달족과의 전쟁</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%B2%A8%EB%A6%AC%EC%82%AC%EB%A6%AC%EC%9A%B0%EC%8A%A4&amp;action=edit&amp;section=3" title="부분 편집: 반달족과의 전쟁">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p><a href="/wiki/533%EB%85%84" title="533년">533년</a> <a href="/wiki/%EC%95%84%ED%94%84%EB%A6%AC%EC%B9%B4" title="아프리카">아프리카</a>로 항해해 <a class="new" href="/w/index.php?title=%EB%A0%88%ED%94%84%ED%82%A4%EC%8A%A4_%EB%A7%88%EA%B7%B8%EB%82%98&amp;action=edit&amp;redlink=1" title="레프키스 마그나 (없는 문서)">레프키스 마그나</a> 근처에 상륙, 당시 <a href="/wiki/%EB%B0%98%EB%8B%AC%EC%A1%B1" title="반달족">반달족</a>이 수도로 하고 있던 <a href="/wiki/%EC%B9%B4%EB%A5%B4%ED%83%80%EA%B3%A0" title="카르타고">카르타고</a>로 향했다. 반달왕국의 왕 <a class="new" href="/w/index.php?title=%EA%B2%94%EB%A6%AC%EB%A9%94%EB%A5%B4&amp;action=edit&amp;redlink=1" title="겔리메르 (없는 문서)">겔리메르</a>를 맞은 벨리사리우스의 비잔틴군은 카르타고에서 10마일 떨어진 곳에서 <a class="new" href="/w/index.php?title=%EC%95%84%EB%93%9C_%EB%8D%B0%ED%82%A4%EB%AF%B8%EC%9B%80%EC%9D%98_%EC%A0%84%ED%88%AC&amp;action=edit&amp;redlink=1" title="아드 데키미움의 전투 (없는 문서)">아드 데키미움의 전투</a>를 벌여 완패했다. 그러나 겔리메르의 조카가 전투에서 죽어 전쟁에 집중하지 못하고 있는 사이 전열을 가다듬은 벨리사리우스는 <a class="new" href="/w/index.php?title=%ED%8B%B0%EC%B9%B4%EB%A9%94%EB%A1%A0%EC%9D%98_%EC%A0%84%ED%88%AC&amp;action=edit&amp;redlink=1" title="티카메론의 전투 (없는 문서)">티카메론의 전투</a>에서 반달군을 격파, <a href="/wiki/534%EB%85%84" title="534년">534년</a> 항복하게 만들었다.</p>
<h3><span class="mw-headline" id=".EB.8F.99.EA.B3.A0.ED.8A.B8.EC.A1.B1.EA.B3.BC.EC.9D.98_.EC.A0.84.EC.9F.81">동고트족과의 전쟁</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%B2%A8%EB%A6%AC%EC%82%AC%EB%A6%AC%EC%9A%B0%EC%8A%A4&amp;action=edit&amp;section=4" title="부분 편집: 동고트족과의 전쟁">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p><a href="/wiki/535%EB%85%84" title="535년">535년</a>, 유스티니아누스 황제는 <a href="/wiki/%EB%82%98%EB%A5%B4%EC%84%B8%EC%8A%A4" title="나르세스">나르세스</a>를 <a href="/wiki/%EB%8F%99%EA%B3%A0%ED%8A%B8_%EC%99%95%EA%B5%AD" title="동고트 왕국">동고트 왕국</a>전선에서 해임하고 벨리사리우스를 투입했다. 벨리사리우스는 빠르게 <a href="/wiki/%EC%8B%9C%EC%B9%A0%EB%A6%AC%EC%95%84" title="시칠리아">시칠리아</a>를 점령하고 <a href="/wiki/%EC%9D%B4%ED%83%88%EB%A6%AC%EC%95%84" title="이탈리아">이탈리아</a> 본토로 진격했다. <a href="/wiki/536%EB%85%84" title="536년">536년</a> <a href="/wiki/%EB%82%98%ED%8F%B4%EB%A6%AC" title="나폴리">나폴리</a>와 <a href="/wiki/%EB%A1%9C%EB%A7%88" title="로마">로마</a>를 수복하고 <a href="/wiki/540%EB%85%84" title="540년">540년</a> 동고트 수도인 <a href="/wiki/%EB%9D%BC%EB%B2%A4%EB%82%98" title="라벤나">라벤나</a>를 점령했다. 동고트족을 비잔틴 황제는 믿지 않았지만 벨리사리우스는 괜찮다고 여겨 그에게 왕국 절반을 주었고, 벨리사리우스는 이에 왕관을 받았는데 이것이 황제의 분노를 샀다.</p>
<p>수도로 돌아온 벨리사리우스는 바로 <a class="mw-redirect" href="/wiki/%EC%82%AC%EC%82%B0%EC%A1%B0_%ED%8E%98%EB%A5%B4%EC%8B%9C%EC%95%84" title="사산조 페르시아">사산조 페르시아</a> 전선에 재투입되었으며, 빼앗긴 <a href="/wiki/%EC%8B%9C%EB%A6%AC%EC%95%84" title="시리아">시리아</a>를 놓고 페르시아와 한동안 교전을 벌이다가 <a href="/wiki/545%EB%85%84" title="545년">545년</a> 금 5천 파운드를 전쟁 보상금으로 페르시아에 넘겨주는 대신 페르시아군이 로마령을 5년간 침공하지 않는 것으로 강화협정을 맺었다.</p>
<p><a href="/wiki/541%EB%85%84" title="541년">541년</a> 벨리사리우스가 이탈리아로 돌아오자 상황은 달라져 있었다. 그가 옹립한 동고트 왕 <a href="/wiki/%EC%9D%BC%EB%94%94%EB%B0%94%EB%93%9C" title="일디바드">일디바드</a>는 암살되었고 <a href="/wiki/%ED%86%A0%ED%8B%B8%EB%9D%BC" title="토틸라">토틸라</a>가 왕이 되어 있었으며, 토틸라의 동고트군은 비잔틴군을 무섭게 몰아내고 있었다. 벨리사리우스는 토틸라에 대해 공세로 들어갔으나, 머지않아 유스티니아누스 황제에 의해 <a href="/wiki/%EB%82%98%EB%A5%B4%EC%84%B8%EC%8A%A4" title="나르세스">나르세스</a>로 교체되었다.</p>
<h2><span class="mw-headline" id=".EB.A7.90.EB.85.84">말년</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%B2%A8%EB%A6%AC%EC%82%AC%EB%A6%AC%EC%9A%B0%EC%8A%A4&amp;action=edit&amp;section=5" title="부분 편집: 말년">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p><a href="/wiki/559%EB%85%84" title="559년">559년</a> 벨리사리우스는 <a href="/wiki/%EB%8F%84%EB%82%98%EC%9A%B0_%EA%B0%95" title="도나우 강">도나우 강</a>을 건넌 <a href="/wiki/%EB%B6%88%EA%B0%80%EB%A5%B4%EC%A1%B1" title="불가르족">불가르족</a> 전선에 투입되어 그의 장수로서 경력의 마지막 승리를 거두었다.</p>
<p><a href="/wiki/562%EB%85%84" title="562년">562년</a> 벨리사리우스는 콘스탄티노폴리스에서 횡령 혐의로 고소되어 징역이 선고되었으나, 현대 연구자들에 따르면 <a href="/wiki/%ED%94%84%EB%A1%9C%EC%BD%94%ED%94%BC%EC%9A%B0%EC%8A%A4" title="프로코피우스">프로코피우스</a>가 꾸며낸 모함이었다. 황제는 곧 벨리사리우스를 풀어주었다.</p>
<p><a href="/wiki/565%EB%85%84" title="565년">565년</a> 벨리사리우스와 유스티니아누스 황제가 모두 세상을 떠났다.</p>
