<p><b>변장</b>(邊章, ? ~ <a href="/wiki/186%EB%85%84" title="186년">186년</a>)은 <a href="/wiki/%EC%A4%91%EA%B5%AD" title="중국">중국</a> <a href="/wiki/%ED%9B%84%ED%95%9C" title="후한">후한</a> 말 <a href="/wiki/%EC%96%91%EC%A3%BC_(%E6%B6%BC%E5%B7%9E)" title="양주 (涼州)">양주</a>(涼州) 금성군(金城郡) <a href="/wiki/%EC%82%AC%EB%9E%8C" title="사람">사람</a>이다. <a href="/wiki/%EC%96%91%EC%A3%BC%EC%9D%98_%EB%82%9C" title="양주의 난">양주의 난</a>에 휘말려 타의로 반군에 가담하였고 <a href="/wiki/%EC%A0%95%EB%B6%80" title="정부">정부</a>의 현상을 피하고자 개명하였다. 본명은 <b>변윤</b>(邊允)이었다.</p>
<h2><span class="mw-headline" id=".EC.83.9D.EC.95.A0">생애</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%B3%80%EC%9E%A5_(%EC%82%BC%EA%B5%AD%EC%A7%80)&amp;action=edit&amp;section=1" title="부분 편집: 생애">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>신안<a href="/wiki/%ED%98%84%EB%A0%B9" title="현령">현령</a>(新安―)과<sup class="reference" id="cite_ref-.EB.B3.80.EC.9E.A5.EC.9D.B4_.EB.B3.80.EC.9E.A5_1-0"><a href="#cite_note-.EB.B3.80.EC.9E.A5.EC.9D.B4_.EB.B3.80.EC.9E.A5-1">[1]</a></sup> 독군종사(督軍從事)를 지냈고 양주에서 <a href="/wiki/%EC%9D%B4%EB%A6%84" title="이름">이름</a>이 있었다.<sup class="reference" id="cite_ref-2"><a href="#cite_note-2">[2]</a></sup> <a href="/wiki/184%EB%85%84" title="184년">184년</a>(<a href="/wiki/%EC%A4%91%ED%8F%89" title="중평">중평</a> 원년) <a href="/wiki/%EA%B2%A8%EC%9A%B8" title="겨울">겨울</a> 북지군(北地郡), 농서군(隴西郡) 부한현(枹罕縣)<sup class="reference" id="cite_ref-3"><a href="#cite_note-3">[3]</a></sup>·하관현(河關縣)의 <a class="mw-redirect" href="/wiki/%EC%84%A0%EB%A0%B9%EA%B0%95" title="선령강">선령강</a>(先零羌)과 <a class="mw-redirect" href="/wiki/%EC%9D%98%EC%A2%85%ED%98%B8" title="의종호">의종호</a>(義從胡) 등 여러 세력이 북궁백옥(北宮伯玉)과 이문후(李文侯)를 장군으로 추대하고 <a href="/wiki/%ED%95%9C%EB%82%98%EB%9D%BC" title="한나라">한나라</a>에 <a href="/wiki/%EC%96%91%EC%A3%BC%EC%9D%98_%EB%82%9C" title="양주의 난">대항하</a>였다. 그 과정에서 호강교위(護羌校尉) 영징(泠徵)과 금성<a href="/wiki/%ED%83%9C%EC%88%98" title="태수">태수</a> 진의(陳懿)는 <a class="mw-redirect" href="/wiki/%EB%AA%A9%EC%88%A8" title="목숨">목숨</a>을 잃었고 변장과 <a href="/wiki/%ED%95%9C%EC%88%98_(%ED%9B%84%ED%95%9C)" title="한수 (후한)">한수</a>는 위협을 받아 그 군정을 맡았다. 변장과 한수는 반군이라고 널리 알려져서 <a href="/wiki/%EC%8B%9D%EC%9D%8D" title="식읍">식읍</a> 1,000호의 <a href="/wiki/%EC%A0%9C%ED%9B%84" title="제후">제후</a> 자리가 현상(懸賞)으로 걸렸다. 둘은 이를 피하고자 각각 변장과 한수로 <a href="/wiki/%EA%B0%9C%EB%AA%85" title="개명">개명</a>하였다.<sup class="reference" id="cite_ref-.EB.B3.80.EC.9E.A5.EC.9D.B4_.EB.B3.80.EC.9E.A5_1-1"><a href="#cite_note-.EB.B3.80.EC.9E.A5.EC.9D.B4_.EB.B3.80.EC.9E.A5-1">[1]</a></sup></p>
<p><a href="/wiki/185%EB%85%84" title="185년">185년</a> <a href="/wiki/%EB%B4%84" title="봄">봄</a> <a href="/wiki/%ED%99%98%EA%B4%80" title="환관">환관</a>을 주살한다는 명분으로 원릉이 있는 <a class="mw-disambig" href="/wiki/%EC%82%BC%EB%B3%B4" title="삼보">삼보</a>(三輔)까지 침범하였다. <a class="mw-redirect" href="/wiki/%EC%A2%8C%EA%B1%B0%EA%B8%B0%EC%9E%A5%EA%B5%B0" title="좌거기장군">좌거기장군</a> <a href="/wiki/%ED%99%A9%EB%B3%B4%EC%88%AD" title="황보숭">황보숭</a>이 진압하러 왔다가 <a href="/wiki/%EC%9E%A5%EC%96%91_(%ED%9B%84%ED%95%9C)" title="장양 (후한)">장양</a>에게 <a href="/wiki/%EB%87%8C%EB%AC%BC" title="뇌물">뇌물</a>을 주지 않아서<sup class="reference" id="cite_ref-.EB.B3.B4.EC.88.AD.EC.9D.B4_4-0"><a href="#cite_note-.EB.B3.B4.EC.88.AD.EC.9D.B4-4">[4]</a></sup> 7월(<a class="mw-redirect" href="/wiki/%EC%9D%8C%EB%A0%A5" title="음력">음력</a>)에<sup class="reference" id="cite_ref-5"><a href="#cite_note-5">[5]</a></sup> 면직되고 <a href="/wiki/%EC%9E%A5%EC%98%A8_(%ED%9B%84%ED%95%9C)" title="장온 (후한)">장온</a>이 대신 부임하였다. 미양(美陽)에서 한군과 맞붙어서 잘 싸웠으나 11월 <a class="mw-redirect" href="/wiki/%EB%B3%84%EB%98%A5%EB%B3%84" title="별똥별">별똥별</a>이 진영을 비추면서 병사들이 불안해하였다. 파로장군(破虜將軍) <a href="/wiki/%EB%8F%99%ED%83%81" title="동탁">동탁</a>과 <a href="/wiki/%EC%9A%B0%EB%B6%80%ED%92%8D" title="우부풍">우부풍</a> 포홍(鮑鴻)이 그 틈을 놓치지 않고 내습해와서 변장과 한수는 수천 명이 죽는 등 크게 패하고 유중(楡中)으로 퇴각하였다. 탕구장군(盪寇―) 주신(周愼)이 3만 명으로 쫓아왔다. 변장과 한수가 따로 군사를 내어 그 보급로를 끊자 주신은 치중을 버리고 도주하였다. <a href="/wiki/186%EB%85%84" title="186년">186년</a> 겨울 한수에게 <a class="mw-redirect" href="/wiki/%EC%82%B4%ED%95%B4" title="살해">살해</a>당했다.<sup class="reference" id="cite_ref-6"><a href="#cite_note-6">[6]</a></sup></p>
