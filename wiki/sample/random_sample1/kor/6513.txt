<p><b>변재섭</b>(<a href="/wiki/1975%EB%85%84" title="1975년">1975년</a> <a href="/wiki/9%EC%9B%94_17%EC%9D%BC" title="9월 17일">9월 17일</a> ~ )은 <a href="/wiki/%EB%8C%80%ED%95%9C%EB%AF%BC%EA%B5%AD" title="대한민국">대한민국</a>의 전 <a href="/wiki/%EC%B6%95%EA%B5%AC" title="축구">축구</a> 선수 및 지도자이다. 전주대학교를 졸업하였으며 <a class="mw-redirect" href="/wiki/%EC%A0%84%EB%B6%81%ED%98%84%EB%8C%80" title="전북현대">전북현대</a>의1997년 에 대뷔하여 전성기를 맞이 했으며 <a href="/wiki/K%EB%A6%AC%EA%B7%B8_1999" title="K리그 1999">1999</a> 시즌 <a href="/wiki/K%EB%A6%AC%EA%B7%B8_%EB%8F%84%EC%9B%80%EC%83%81" title="K리그 도움상">K리그 도움상</a>을 수상하였다. <a class="mw-redirect" href="/wiki/%EC%A0%84%EB%B6%81%ED%98%84%EB%8C%80" title="전북현대">전북현대</a>과 재계약 실패후 <a class="new" href="/w/index.php?title=%EB%B6%80%EC%B2%9Csk&amp;action=edit&amp;redlink=1" title="부천sk (없는 문서)">부천sk</a>-<a class="mw-redirect" href="/wiki/%EC%A0%9C%EC%A3%BC%EC%9C%A0%EB%82%98%EC%9D%B4%ED%8B%B0%EB%93%9C" title="제주유나이티드">제주유나이티드</a>에 이적하였다 하지만 2007년에 친정팀 <a class="mw-redirect" href="/wiki/%EC%A0%84%EB%B6%81%ED%98%84%EB%8C%80" title="전북현대">전북현대</a>의 다시 입단하였으며 2007년에 은퇴하였다 그리고 아쉽게 20골-20도움에 가입하지 못했으며 지도자로 순서대로 <a href="/wiki/%EC%A0%84%EC%A3%BC%EB%8C%80%ED%95%99%EA%B5%90" title="전주대학교">전주대학교</a>-광양제철고등학교-<a class="mw-redirect" href="/wiki/%EC%A0%9C%EC%A3%BC%EC%9C%A0%EB%82%98%EC%9D%B4%ED%8B%B0%EB%93%9C" title="제주유나이티드">제주유나이티드</a>에 지도자였고 <a class="mw-redirect" href="/wiki/%EC%9A%B8%EC%82%B0%ED%98%84%EB%8C%80" title="울산현대">울산현대</a>에 코치다</p>
