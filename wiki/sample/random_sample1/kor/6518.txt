<p>《<b>별의 커비</b>》(<span style="font-size: smaller;"><a href="/wiki/%EC%9D%BC%EB%B3%B8%EC%96%B4" title="일본어">일본어</a>:</span> <span lang="ja" xml:lang="ja">星のカービィ</span> <small>호시노 카비<sup>[<a href="/wiki/%EC%9C%84%ED%82%A4%EB%B0%B1%EA%B3%BC:%EC%9D%BC%EB%B3%B8%EC%96%B4%EC%9D%98_%ED%95%9C%EA%B8%80_%ED%91%9C%EA%B8%B0" title="위키백과:일본어의 한글 표기">*</a>]</sup></small>, <span lang="en" xml:lang="en"><i>Kirby's Dream Land</i></span>)는 <a href="/wiki/1992%EB%85%84" title="1992년">1992년</a> <a href="/wiki/4%EC%9B%94_27%EC%9D%BC" title="4월 27일">4월 27일</a>에 <a href="/wiki/%EB%8B%8C%ED%85%90%EB%8F%84" title="닌텐도">닌텐도</a>에서 발매된 <a href="/wiki/%EA%B2%8C%EC%9E%84%EB%B3%B4%EC%9D%B4" title="게임보이">게임보이</a>용 액션 게임이다. 개발사는 <a class="mw-redirect" href="/wiki/HAL_%EC%97%B0%EA%B5%AC%EC%86%8C" title="HAL 연구소">HAL 연구소</a>. 기획자는 제작 당시 22살이었던 '사쿠라이 마사히로'이다. 별의 커비 시리즈 제1편의 제목이다.<a class="mw-redirect" href="/wiki/%EB%B3%84%EC%9D%98_%EC%BB%A4%EB%B9%84_%EC%8A%88%ED%8D%BC_%EB%94%94%EB%9F%AD%EC%8A%A4" title="별의 커비 슈퍼 디럭스">별의 커비 슈퍼 디럭스</a>와 <a href="/wiki/%EB%B3%84%EC%9D%98_%EC%BB%A4%EB%B9%84_%EC%9A%B8%ED%8A%B8%EB%9D%BC_%EC%8A%88%ED%8D%BC_%EB%94%94%EB%9F%AD%EC%8A%A4" title="별의 커비 울트라 슈퍼 디럭스">별의 커비 울트라 슈퍼 디럭스</a>에서 '봄바람을 타고'로 리메이크되었다.</p>
<h2><span class="mw-headline" id=".EC.8A.A4.ED.86.A0.EB.A6.AC">스토리</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%B3%84%EC%9D%98_%EC%BB%A4%EB%B9%84&amp;action=edit&amp;section=1" title="부분 편집: 스토리">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>현재의 '봄바람을 타고'와 매우 유사하다. <a href="/wiki/%EB%94%94%EB%94%94%EB%94%94_%EB%8C%80%EC%99%95" title="디디디 대왕">디디디 대왕</a>이 푸푸푸 랜드의 모든 음식과 음식을 구하는 데 쓰인 스파클링 스타를 훔쳐가서, 정의의 용사 커비가 디디디 대왕을 혼내주고, 모든 음식을 되찾아온다는 이야기이다.</p>
<h2><span class="mw-headline" id=".ED.8A.B9.EC.A7.95">특징</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%B3%84%EC%9D%98_%EC%BB%A4%EB%B9%84&amp;action=edit&amp;section=2" title="부분 편집: 특징">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>지금 우리가 흔하게 알고 있는 다른 타이틀과 별반 차이가 없다. 십자 키를 이용한 조작이나 호버링, 흡입 등 지금과 꽤나 비슷하나 카피 능력이 없다. 게임을 어떻게 할지 걱정할지도 모르지만 흡입 기능은 처음부터 있던 요소여서 지금처럼 별 발사도 가능했다. 어떻게 보면 지금보다 참신한 게, 맵에는 가끔 아이템이 배치되어 있어 이 아이템을 흡입해서 이용하는 방식이다. 물론 체력 회복 아이템도 있다. 지금의 '마이크' 능력의 모티브로 데스크톱 마이크가 있고, '봄'과 '크래쉬' 능력의 모티브인 폭탄도 존재했다. 게임을 하다 보면 가끔 나오는 '엄청 매운 카레'나 '녹찻잎'는 지금의 무적사탕의 개념과 비슷한데 전자의 경우는 카피 능력 '파이어'로, 녹찻잎의 경우는 호버링 뒤 나오는 공기탄보다 센 정도로 생각하면 된다. 커비의 에어라이드에서는 이 때 나오는 배경음이 스테이지 맵의 우라곡으로 쓰이기도 했다. 대시 기능도 처음부터 존재했던 게 아니므로 느리다. <a class="mw-redirect" href="/wiki/%EB%B3%84%EC%9D%98_%EC%BB%A4%EB%B9%84_%EC%8A%88%ED%8D%BC_%EB%94%94%EB%9F%AD%EC%8A%A4" title="별의 커비 슈퍼 디럭스">별의 커비 슈퍼 디럭스</a>와 <a href="/wiki/%EB%B3%84%EC%9D%98_%EC%BB%A4%EB%B9%84_%EC%9A%B8%ED%8A%B8%EB%9D%BC_%EC%8A%88%ED%8D%BC_%EB%94%94%EB%9F%AD%EC%8A%A4" title="별의 커비 울트라 슈퍼 디럭스">별의 커비 울트라 슈퍼 디럭스</a>의 '봄바람을 타고'에서 유일하게 리메이크가 안된 'Castle LoLoLo'는 보스와 테마가 대거 이사했다. Castle LoLoLo의 보스인 로로&amp;라라는 Float Islands의 보스로 이사했고 Float Islands의 원래 보스인 공중포대 카부라는 <a href="/wiki/%EB%B3%84%EC%9D%98_%EC%BB%A4%EB%B9%84_%EC%9A%B8%ED%8A%B8%EB%9D%BC_%EC%8A%88%ED%8D%BC_%EB%94%94%EB%9F%AD%EC%8A%A4" title="별의 커비 울트라 슈퍼 디럭스">별의 커비 울트라 슈퍼 디럭스</a>의 '디디디 리벤지'의 The Sky Over Mt.DeDeDe의 보스에 정착했다. Castle LoLoLo의 테마는 <a class="mw-redirect" href="/wiki/%EB%B3%84%EC%9D%98_%EC%BB%A4%EB%B9%84_%EC%8A%88%ED%8D%BC_%EB%94%94%EB%9F%AD%EC%8A%A4" title="별의 커비 슈퍼 디럭스">별의 커비 슈퍼 디럭스</a>의 보스 (마인 왐밤록과 왐밤 주얼 등)에게 넘어갔다.</p>
