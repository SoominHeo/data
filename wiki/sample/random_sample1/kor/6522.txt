<p><b>병산서원</b>(屛山書院)은 서애 <a href="/wiki/%EB%A5%98%EC%84%B1%EB%A3%A1" title="류성룡">류성룡</a> 선생을 기리기 위해 만든 <a href="/wiki/%EA%B2%BD%EC%83%81%EB%B6%81%EB%8F%84" title="경상북도">경상북도</a> <a href="/wiki/%EC%95%88%EB%8F%99%EC%8B%9C" title="안동시">안동시</a> <a href="/wiki/%ED%92%8D%EC%B2%9C%EB%A9%B4" title="풍천면">풍천면</a> 병산리에 있는 서원이다. <a href="/wiki/1978%EB%85%84" title="1978년">1978년</a> <a href="/wiki/3%EC%9B%94_31%EC%9D%BC" title="3월 31일">3월 31일</a> <a href="/wiki/%EB%8C%80%ED%95%9C%EB%AF%BC%EA%B5%AD%EC%9D%98_%EC%82%AC%EC%A0%81" title="대한민국의 사적">대한민국의 사적</a> 제260호로 지정되었다. 고려 시대부터 사림의 교육기관이었던, 풍산현에 있던 풍악서당(豊岳書堂)을 1572년 선조5년 서애 류성룡이 이곳으로 옮겨왔다.</p>

<h2><span class="mw-headline" id=".EA.B0.9C.EC.9A.94">개요</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%B3%91%EC%82%B0%EC%84%9C%EC%9B%90&amp;action=edit&amp;section=1" title="부분 편집: 개요">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>고려 시대부터 사림의 교육기관이었던 풍악서당(豊岳書堂)을 풍산현에서 <a href="/wiki/1572%EB%85%84" title="1572년">1572년</a> 선조 5년 서애 류성룡이 안동으로 옮겨오면서부터 병산 서원의 역사가 시작되었다. 1607년 서애가 타계하자 정경세(鄭經世) 등 지방 유림의 공의로 선생의 학문과 덕행을 추모하기 위하여 <a href="/wiki/1613%EB%85%84" title="1613년">1613년</a> 광해군 5년에 존덕사(尊德祠)를 창건하고 위패를 봉안하였다. <a href="/wiki/1614%EB%85%84" title="1614년">1614년</a> 병산서원으로 개칭하였다.</p>
<p><a href="/wiki/1620%EB%85%84" title="1620년">1620년</a> <a class="mw-redirect" href="/wiki/%EA%B4%91%ED%95%B4%EA%B5%B0" title="광해군">광해군</a> 12년에 유림의 공론에 따라 퇴계 선생을 모시는 여강서원(廬江書院)으로 위패를 옮기게 되었다. 그 뒤 <a href="/wiki/1629%EB%85%84" title="1629년">1629년</a> 인조 9년에 별도의 위패를 마련하여 존덕사에 모셨으며, 그의 셋째 아들 수암(修巖) 류진(柳袗)을 추가로 배향하였다. <a href="/wiki/1863%EB%85%84" title="1863년">1863년</a> <a href="/wiki/%EC%A1%B0%EC%84%A0_%EC%B2%A0%EC%A2%85" title="조선 철종">철종</a> 14년에 사액(賜額)되어 서원으로 승격하였다.</p>
<p>유림 선현을 모시고, 지방교육의 일익을 담당하여 많은 학자를 배출하였으며, <a href="/wiki/1868%EB%85%84" title="1868년">1868년</a> 고종 5년 <a href="/wiki/%ED%9D%A5%EC%84%A0%EB%8C%80%EC%9B%90%EA%B5%B0" title="흥선대원군">흥선대원군</a>의 서원철폐령이 내렸을 때에도 훼철(毁撤)되지 않고 보호되었다. 일제강점기에 대대적인 보수가 행해졌으며 강당은 1921년에, 사당은 1937년 각각 다시 지어졌다. 매년 3월 중정과 9월 중정에 향사례를 지내고 있다.</p>
<p>사적 제 260호로 지정되어 있으며 서애 선생의 문집을 비롯하여 각종 문헌 1,000여 종 3,000여 책이 소장되어 있다.<sup class="reference" id="cite_ref-1"><a href="#cite_note-1">[1]</a></sup></p>
