<h1 class="firstHeading" id="firstHeading" lang="ko">병자성사</h1>
<div class="mw-body-content" id="bodyContent">
<div id="siteSub">위키백과, 우리 모두의 백과사전.</div>
<div id="contentSub"></div>
<div class="mw-jump" id="jump-to-nav">
					이동:					<a href="#mw-head">둘러보기</a>, 					<a href="#p-search">검색</a>
</div>
<div class="mw-content-ltr" dir="ltr" id="mw-content-text" lang="ko"><script>function mfTempOpenSection(id){var block=document.getElementById("mf-section-"+id);block.className+=" open-block";block.previousSibling.className+=" open-block";}</script><p><b>병자 성사</b>(病者聖事)는 <a href="/wiki/%EA%B8%B0%EB%8F%85%EA%B5%90" title="기독교">기독교</a>에서 몸이 불편한 교우를 <a href="/wiki/%EC%82%AC%EC%A0%9C" title="사제">사제</a>가 <a href="/wiki/%EC%84%B1%EC%9C%A0" title="성유">성유</a>를 바르면서 회복을 기원하는 <a href="/wiki/%EC%84%B1%EC%82%AC" title="성사">성사</a>를 말한다. <a href="/wiki/%EC%84%B1%EA%B3%B5%ED%9A%8C" title="성공회">성공회</a>에서는 <b>조병예식</b>으로 준성사로 취급하고, <a class="mw-redirect" href="/wiki/%EC%A0%95%EA%B5%90%ED%9A%8C" title="정교회">정교회</a>에서는 <b>성유성사</b>라고 한다.</p>
<p>몸이 아프면 교회 원로(<a href="/wiki/%EC%82%AC%EC%A0%9C" title="사제">사제</a>)들을 불러서 기름을 바르며 회복을 요청하는 기도를 부탁하라는 <a href="/wiki/%EC%95%BC%EA%B3%A0%EB%B3%B4%EC%9D%98_%ED%8E%B8%EC%A7%80" title="야고보의 편지">야고보의 편지</a> 내용에 근거한다. 교파별로 성사인지 아닌지 보는 관점이 다르며 <a href="/wiki/%EB%A1%9C%EB%A7%88_%EA%B0%80%ED%86%A8%EB%A6%AD%EA%B5%90%ED%9A%8C" title="로마 가톨릭교회">로마 가톨릭교회</a>와 <a class="mw-redirect" href="/wiki/%EC%A0%95%EA%B5%90%ED%9A%8C" title="정교회">정교회</a>에서는 성사로 유지하고 있고, <a href="/wiki/%EB%8C%80%ED%95%9C%EC%84%B1%EA%B3%B5%ED%9A%8C" title="대한성공회">대한성공회</a>에서는 조병성사로 불러왔으나, <a href="/wiki/2004%EB%85%84" title="2004년">2004년</a> 기도서 개정 이후 조병예식으로 명칭을 변경하여 성사가 아닌 성사적 예식으로 본다. 그나마 성공회 이외의 <a href="/wiki/%EA%B0%9C%EC%8B%A0%EA%B5%90" title="개신교">개신교</a>에서는 전혀 관찰할 수 없다.</p>
