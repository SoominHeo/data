<p><b>병점기지선</b>(餠店基地線)은 <a href="/wiki/%EA%B2%BD%EA%B8%B0%EB%8F%84" title="경기도">경기도</a> <a href="/wiki/%ED%99%94%EC%84%B1%EC%8B%9C" title="화성시">화성시</a>의 <a href="/wiki/%EB%B3%91%EC%A0%90%EC%97%AD" title="병점역">병점역</a>과 <a href="/wiki/%EC%98%A4%EC%82%B0%EC%8B%9C" title="오산시">오산시</a>의 <a href="/wiki/%EC%84%9C%EB%8F%99%ED%83%84%EC%97%AD" title="서동탄역">서동탄역</a>을 잇는 <a href="/wiki/%ED%95%9C%EA%B5%AD%EC%B2%A0%EB%8F%84%EA%B3%B5%EC%82%AC" title="한국철도공사">한국철도공사</a>의 <a href="/wiki/%EA%B2%BD%EB%B6%80%EC%84%A0" title="경부선">경부선</a> <a class="mw-redirect" href="/wiki/%EC%A7%80%EC%84%A0_%EC%B2%A0%EB%8F%84" title="지선 철도">지선 철도</a> 노선이다.</p>
<p>과거 이 노선은 <a href="/wiki/%EA%B2%BD%EB%B6%80%EC%84%A0" title="경부선">경부선</a> 본선의 일부였으나, <a href="/wiki/%EA%B2%BD%EB%B6%80%EC%84%A0" title="경부선">경부선</a>의 복복선화 및 직선화 과정 중 <a href="/wiki/%EB%B3%91%EC%A0%90%EC%97%AD" title="병점역">병점역</a>에서 지금의 <a href="/wiki/%EC%98%A4%EC%82%B0%EB%8C%80%EC%97%AD" title="오산대역">오산대역</a>에 이르는 구간이 이설되고 기존선 루트 상에 <a href="/wiki/%EB%B3%91%EC%A0%90%EC%B0%A8%EB%9F%89%EC%82%AC%EC%97%85%EC%86%8C" title="병점차량사업소">병점차량사업소</a>를 건설함에 따라 병점역과 병점차량사업소를 잇는 구간만을 남겨 병점기지선이라고 부르게 되었다.</p>
<p>이후 <a href="/wiki/%ED%99%94%EC%84%B1%EC%8B%9C" title="화성시">화성시</a>의 <a href="/wiki/%EB%8F%99%ED%83%84%EC%8B%A0%EB%8F%84%EC%8B%9C" title="동탄신도시">동탄신도시</a>가 본격적으로 조성되면서 병점역에 비해 병점차량사업소가 더욱 인접한 점을 토대로 <a href="/wiki/%ED%99%94%EC%84%B1%EC%8B%9C" title="화성시">화성시</a>가 건설비 전액을 투자하여 <a href="/wiki/%EC%98%A4%EC%82%B0%EC%8B%9C" title="오산시">오산시</a> 외삼미동에 <a href="/wiki/%EC%84%9C%EB%8F%99%ED%83%84%EC%97%AD" title="서동탄역">서동탄역</a>을 신설하였다.</p>
