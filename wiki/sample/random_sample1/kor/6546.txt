<p><b>보름달</b>(<span style="font-size: smaller;"><a href="/wiki/%EB%AC%B8%ED%99%94%EC%96%B4" title="문화어">문화어</a>:</span> 옹근달,<span style="font-size: smaller;"><a href="/wiki/%EC%98%81%EC%96%B4" title="영어">영어</a>:</span> <span lang="en" xml:lang="en">full moon</span>)은 <a href="/wiki/%EB%8B%AC" title="달">달</a>이 <a href="/wiki/%ED%83%9C%EC%96%91" title="태양">태양</a>과 <a href="/wiki/%EC%A7%80%EA%B5%AC" title="지구">지구</a>의 서로 반대쪽에 위치하여 지구에서 달의 전체가 밝게 보이는 것을 말한다. <b>만월</b>(滿月) 또는 <b>망월</b>(望月)이라고도 한다. 보름달의 밝기는 <a href="/wiki/%EA%B8%88%EC%84%B1" title="금성">금성</a>이 제일 밝을 때의 1500배에 달한다. 보름달 밝기는 태양 다음으로 밝은 -12등급이라고 한다.</p>
<p>한국에서는 예로부터 <a href="/wiki/%EC%9D%8C%EB%A0%A5_1%EC%9B%94_15%EC%9D%BC" title="음력 1월 15일">음력 1월 15일</a>의 보름달을 <a class="mw-redirect" href="/wiki/%EC%A0%95%EC%9B%94%EB%8C%80%EB%B3%B4%EB%A6%84" title="정월대보름">정월대보름</a>달이라고 하고, <a href="/wiki/%EC%9D%8C%EB%A0%A5_8%EC%9B%94_15%EC%9D%BC" title="음력 8월 15일">음력 8월 15일</a> 때에는 <b><a class="mw-redirect" href="/wiki/%ED%95%9C%EA%B0%80%EC%9C%84" title="한가위">한가위</a></b>(<a href="/wiki/%EC%B6%94%EC%84%9D" title="추석">추석</a>)이라고 부른다. 그리고 때로는 태양과 지구의 일직선 상에 놓이면 <a class="mw-redirect" href="/wiki/%EA%B0%9C%EA%B8%B0_%EC%9B%94%EC%8B%9D" title="개기 월식">개기 월식</a> 또는 <a class="new" href="/w/index.php?title=%EB%B6%80%EB%B6%84_%EC%9B%94%EC%8B%9D&amp;action=edit&amp;redlink=1" title="부분 월식 (없는 문서)">부분 월식</a>이 관측되기도 한다. 보름달은 저녁무렵에 뜨기 시작하여, 남쪽하늘에 다다랐을 무렵에는 한밤중이므로 벌써 하루가 지나고 만다. 그리하여 새벽무렵에는 서쪽하늘에 보였다가 이른아침이 되면 서쪽으로 넘어간다.</p>

<h2><span class="mw-headline" id=".EA.B8.B0.ED.83.80">기타</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%B3%B4%EB%A6%84%EB%8B%AC&amp;action=edit&amp;section=1" title="부분 편집: 기타">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>음력 15일에 "항상" 보름달이 뜨지는 않는다. <a href="/wiki/%EB%8B%AC" title="달">달</a>은 <a href="/wiki/%EC%A7%80%EA%B5%AC" title="지구">지구</a> 주위를 <a href="/wiki/%EC%9B%90_(%EA%B8%B0%ED%95%98%ED%95%99)" title="원 (기하학)">원</a>에 가까운 <a class="mw-redirect" href="/wiki/%ED%83%80%EC%9B%90%ED%98%95" title="타원형">타원형</a>으로 <a href="/wiki/%EA%B3%B5%EC%A0%84" title="공전">공전</a>하는데 (달의 공전 궤도 <a href="/wiki/%EC%9D%B4%EC%8B%AC%EB%A5%A0" title="이심률">이심률</a>은 0.05488이다.<sup class="reference" id="cite_ref-1"><a href="#cite_note-1">[1]</a></sup>), 달이 <a class="mw-redirect" href="/wiki/%EC%9B%90%EC%A7%80%EC%A0%90" title="원지점">원지점</a><sup class="reference" id="cite_ref-2"><a href="#cite_note-2">[2]</a></sup>을 통과하는 경우에는 음력 16일, 17일에 보름달이 뜬다.<sup class="reference" id="cite_ref-3"><a href="#cite_note-3">[3]</a></sup></p>
<p>그리고 '태양 - 지구 - 달'의 위치로 배열될 때 보름달이 뜨게 되는데, 이는 <a href="/wiki/%EC%9B%94%EC%8B%9D" title="월식">월식</a>이 일어나는 경우와 비슷하다. 하지만, 보름달이 뜰 때마다 월식이 일어나지 않는데, 그 이유는 지구의 공전 궤도와 달의 공전 궤도가 약 5도 정도 기울어져 있기 때문이다. '태양 - 지구 - 달'이 정확히 일직선이 됐을 때에 월식이 일어난다. 이는 <a href="/wiki/%EC%8B%A0%EC%9B%94" title="신월">신월</a>과 <a href="/wiki/%EC%9D%BC%EC%8B%9D" title="일식">일식</a>의 경우에도 마찬가지이다.</p>
<p>또한, 보름달이 뜰 때에는 <a class="mw-redirect" href="/wiki/%EB%B0%80%EB%AC%BC" title="밀물">밀물</a>과 <a class="mw-redirect" href="/wiki/%EC%8D%B0%EB%AC%BC" title="썰물">썰물</a> 사이의 높이 차이, 즉 조수간만의 차, <a class="mw-disambig" href="/wiki/%EC%A1%B0%EC%B0%A8" title="조차">조차</a>가 가장 크다. 이는 <a href="/wiki/%EA%B7%B8%EB%AF%90%EB%8B%AC" title="그믐달">그믐달</a><sup class="reference" id="cite_ref-4"><a href="#cite_note-4">[4]</a></sup>일 때에도 마찬가지이다.<sup class="reference" id="cite_ref-5"><a href="#cite_note-5">[5]</a></sup></p>

<li id="cite_note-2"><span class="mw-cite-backlink"><a href="#cite_ref-2">↑</a></span> <span class="reference-text">달의 공전 궤도 중에서 지구와의 거리가 가장 먼 지점</span></li>
