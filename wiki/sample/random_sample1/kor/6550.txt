<td class="mbox-text"><span class="mbox-text-span">이 문서의 내용은 <b><a href="/wiki/%EC%9C%84%ED%82%A4%EB%B0%B1%EA%B3%BC:%EC%B6%9C%EC%B2%98_%EB%B0%9D%ED%9E%88%EA%B8%B0" title="위키백과:출처 밝히기">출처</a>가 분명하지 않습니다.</b><span class="hide-when-compact"><br/>
지금 바로 이 <span class="plainlinks"><a class="external text" href="//ko.wikipedia.org/w/index.php?title=%EB%B3%B4%EB%A6%AC%EC%8A%A4_3%EC%84%B8&amp;action=edit">문서를 편집</a></span>하여, 참고하신 문헌이나 <a href="/wiki/%EC%9C%84%ED%82%A4%EB%B0%B1%EA%B3%BC:%EC%8B%A0%EB%A2%B0%ED%95%A0_%EC%88%98_%EC%9E%88%EB%8A%94_%EC%B6%9C%EC%B2%98" title="위키백과:신뢰할 수 있는 출처">신뢰할 수 있는 출처</a>를 각주 등으로 표기해 주세요. <a href="/wiki/%EC%9C%84%ED%82%A4%EB%B0%B1%EA%B3%BC:%ED%99%95%EC%9D%B8_%EA%B0%80%EB%8A%A5" title="위키백과:확인 가능">검증</a>되지 않은 내용은 삭제될 수도 있습니다. 내용에 대한 의견은 <a class="new" href="/w/index.php?title=%ED%86%A0%EB%A1%A0:%EB%B3%B4%EB%A6%AC%EC%8A%A4_3%EC%84%B8&amp;action=edit&amp;redlink=1" title="토론:보리스 3세 (없는 문서)">토론 문서</a>에서 나누어 주세요.</span> <small>(2013년 6월)</small></span></td>

<p><b>보리스 3세</b>(<span style="font-size: smaller;"><a href="/wiki/%EB%B6%88%EA%B0%80%EB%A6%AC%EC%95%84%EC%96%B4" title="불가리아어">불가리아어</a>:</span> <span lang="bg" xml:lang="bg">Борис III</span>, <a href="/wiki/1894%EB%85%84" title="1894년">1894년</a> <a href="/wiki/1%EC%9B%94_30%EC%9D%BC" title="1월 30일">1월 30일</a> - <a href="/wiki/1943%EB%85%84" title="1943년">1943년</a> <a href="/wiki/8%EC%9B%94_28%EC%9D%BC" title="8월 28일">8월 28일</a>)는 <a href="/wiki/%EB%B6%88%EA%B0%80%EB%A6%AC%EC%95%84_%EC%99%95%EA%B5%AD" title="불가리아 왕국">불가리아</a>의 2대 <a class="mw-redirect" href="/wiki/%EA%B5%AD%EC%99%95" title="국왕">차르</a>이다.</p>
<p><a href="/wiki/%EC%A0%9C1%EC%B0%A8_%EC%84%B8%EA%B3%84_%EB%8C%80%EC%A0%84" title="제1차 세계 대전">제1차 세계 대전</a>에 참전, 선왕 <a class="mw-redirect" href="/wiki/%EB%B6%88%EA%B0%80%EB%A6%AC%EC%95%84%EC%9D%98_%ED%8E%98%EB%A5%B4%EB%94%94%EB%82%9C%EB%93%9C_1%EC%84%B8" title="불가리아의 페르디난드 1세">페르디난드 1세</a>가 패전을 책임지고 사임하자, 곧바로 즉위했다. <a href="/wiki/%EC%A0%9C2%EC%B0%A8_%EC%84%B8%EA%B3%84_%EB%8C%80%EC%A0%84" title="제2차 세계 대전">제2차 세계 대전</a> 때 추축국에 협력, <a href="/wiki/1943%EB%85%84" title="1943년">1943년</a>에 서거하였다. 왕위는 6살이었던 <a href="/wiki/%EC%8B%9C%EB%A9%94%EC%98%A8_2%EC%84%B8" title="시메온 2세">시메온 2세</a>에게 계승되었으나 <a href="/wiki/1946%EB%85%84" title="1946년">1946년</a>에 퇴위, 공산화되었다.</p>
<p><a href="/wiki/%EC%9D%B4%ED%83%88%EB%A6%AC%EC%95%84_%EC%99%95%EA%B5%AD_(1861%EB%85%84_~_1946%EB%85%84)" title="이탈리아 왕국 (1861년 ~ 1946년)">이탈리아</a>의 <a href="/wiki/%EB%B9%84%ED%86%A0%EB%A6%AC%EC%98%A4_%EC%97%90%EB%A7%88%EB%88%84%EC%97%98%EB%A0%88_3%EC%84%B8" title="비토리오 에마누엘레 3세">비토리오 에마누엘레 3세</a>의 딸인 조반나 공주와 결혼해 1남 1녀를 두었다.</p>
