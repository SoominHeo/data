<h1 class="firstHeading" id="firstHeading" lang="ko">보스턴 차 사건</h1>
<div class="mw-body-content" id="bodyContent">

<p><b>보스턴 차 사건</b>(<span lang="en" xml:lang="en">Boston Tea Party</span>)은 <a href="/wiki/%EA%B7%B8%EB%A0%88%EC%9D%B4%ED%8A%B8%EB%B8%8C%EB%A6%AC%ED%8A%BC_%EC%99%95%EA%B5%AD" title="그레이트브리튼 왕국">그레이트브리튼 왕국</a>의 지나친 세금 징수에 반발한 <a href="/wiki/%EB%B6%81%EC%95%84%EB%A9%94%EB%A6%AC%EC%B9%B4" title="북아메리카">북아메리카</a>의 식민지 주민들이 <a class="mw-redirect" href="/wiki/%EC%95%84%EB%A9%94%EB%A6%AC%EC%B9%B4_%ED%86%A0%EC%B0%A9%EB%AF%BC" title="아메리카 토착민">아메리카 토착민</a>으로 위장해 <a href="/wiki/1773%EB%85%84" title="1773년">1773년</a> <a href="/wiki/12%EC%9B%94_16%EC%9D%BC" title="12월 16일">12월 16일</a> <a href="/wiki/%EB%B3%B4%EC%8A%A4%ED%84%B4" title="보스턴">보스턴</a> 항에 정박한 <a class="mw-redirect" href="/wiki/%EB%B0%B0_(%ED%83%88%EA%B2%83)" title="배 (탈것)">배</a>에 실려 있던 <a href="/wiki/%ED%99%8D%EC%B0%A8" title="홍차">홍차</a> 상자들을 바다에 버린 사건이다. 이 사건은 <a href="/wiki/%EB%AF%B8%EA%B5%AD_%EB%8F%85%EB%A6%BD_%EC%A0%84%EC%9F%81" title="미국 독립 전쟁">미국 독립 전쟁</a>의 불씨를 일으키는 데 일조한 것으로 여겨져 왔다.</p>
<h2><span class="mw-headline" id=".EC.82.AC.EA.B1.B4.EC.9D.98_.EB.B0.B0.EA.B2.BD">사건의 배경</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%B3%B4%EC%8A%A4%ED%84%B4_%EC%B0%A8_%EC%82%AC%EA%B1%B4&amp;action=edit&amp;section=1" title="부분 편집: 사건의 배경">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p><a href="/wiki/1765%EB%85%84" title="1765년">1765년</a>의 〈<a href="/wiki/%EC%9D%B8%EC%A7%80%EC%84%B8%EB%B2%95" title="인지세법">인지세법</a>〉과 <a href="/wiki/1767%EB%85%84" title="1767년">1767년</a>의 〈<a class="mw-redirect" href="/wiki/%ED%83%80%EC%9A%B4%EC%A0%A0%EB%93%9C%EB%B2%95" title="타운젠드법">타운젠드법</a>〉은 의회 대표가 없는 <a href="/wiki/%EC%8B%9D%EB%AF%BC%EC%A7%80" title="식민지">식민지</a>에도 과세하기로 한 <a href="/wiki/%EC%98%81%EA%B5%AD" title="영국">영국</a>의 결정과 관련하여 식민지 주민들의 반발을 불러 일으켰다.</p>
<p>그레이트브리튼 왕국에 저항하던 대표적인 인물인 <a href="/wiki/%EC%A1%B4_%ED%95%B8%EC%BD%95" title="존 핸콕">존 핸콕</a>은 <a href="/wiki/%EC%98%81%EA%B5%AD_%EB%8F%99%EC%9D%B8%EB%8F%84_%ED%9A%8C%EC%82%AC" title="영국 동인도 회사">영국 동인도 회사</a>의 <a href="/wiki/%EC%A4%91%EA%B5%AD" title="중국">중국</a>산 차에 대한 <a class="mw-redirect" href="/wiki/%EB%B6%88%EB%A7%A4_%EC%9A%B4%EB%8F%99" title="불매 운동">불매 운동</a>을 조직하였고, 곧 판매량은 320,000 파운드(145,000 kg)에서 520 파운드(240 kg)로 급감하였다. <a href="/wiki/1773%EB%85%84" title="1773년">1773년</a>에 이르러 동인도 회사의 적자는 크게 불어났고, 창고에는 언제 판매될지도 모르는 찻잎들이 쌓여갔는데, 이는 <a href="/wiki/%EC%A1%B4_%ED%95%B8%EC%BD%95" title="존 핸콕">존 핸콕</a>을 비롯한 밀수업자들이 관세를 물지 않고 차를 수입해 판매했기 때문이었다. 영국 정부는 〈<a href="/wiki/%EC%B0%A8%EB%B2%95" title="차법">차법</a>〉을 통과시켜 <a href="/wiki/%EC%98%81%EA%B5%AD_%EB%8F%99%EC%9D%B8%EB%8F%84_%ED%9A%8C%EC%82%AC" title="영국 동인도 회사">동인도 회사</a>가 식민지에 직접 차를 판매할 수 있도록 하였다. 그리하여 동인도 회사가 대영제국 업자들과 밀수업자들보다 싼 가격에 차를 판매할 수 있게 되어, 영국 상인들과 밀수업자들은 파산을 면치 못할 정도로 큰 피해를 보았다.</p>
<p>북아메리카의 대부분의 항구에서는 <a href="/wiki/%EC%98%81%EA%B5%AD_%EB%8F%99%EC%9D%B8%EB%8F%84_%ED%9A%8C%EC%82%AC" title="영국 동인도 회사">동인도 회사</a>의 차를 실은 배의 하역을 거부하였으나, 보스턴에서는 영국 정부가 임명한 총독인 <a class="new" href="/w/index.php?title=%ED%86%A0%EB%A8%B8%EC%8A%A4_%ED%97%88%EC%B9%9C%EC%8A%A8&amp;action=edit&amp;redlink=1" title="토머스 허친슨 (없는 문서)">토머스 허친슨</a>의 도움을 받을 수 있었다. 영국 군함들의 호위 아래 차들을 하역하기 위한 계획이 세워졌다.</p>
