<p><b>5번가</b>(Fifth Avenue)는 <a href="/wiki/%EB%AF%B8%EA%B5%AD" title="미국">미국</a> <a href="/wiki/%EB%89%B4%EC%9A%95" title="뉴욕">뉴욕</a> <a href="/wiki/%EB%A7%A8%ED%95%B4%ED%8A%BC" title="맨해튼">맨해튼</a>을 남북으로 종단하는 거리이다. <a href="/wiki/%EC%84%BC%ED%8A%B8%EB%9F%B4_%ED%8C%8C%ED%81%AC" title="센트럴 파크">센트럴 파크</a>를 조망할 수있는 고급 아파트와 역사적인 저택이 들어서 있다. 남쪽에서 시작하여 <a class="mw-redirect" href="/wiki/%EA%B7%B8%EB%A6%AC%EB%8B%88%EC%B9%98_%EB%B9%8C%EB%A6%AC%EC%A7%80" title="그리니치 빌리지">그리니치 빌리지</a>, <a href="/wiki/%EC%B2%BC%EC%8B%9C_(%EB%A7%A8%ED%95%B4%ED%8A%BC)" title="첼시 (맨해튼)">첼시</a>, <a href="/wiki/%EB%AF%B8%EB%93%9C%ED%83%80%EC%9A%B4_%EB%A7%A8%ED%95%B4%ED%8A%BC" title="미드타운 맨해튼">미드타운 맨해튼</a>, <a class="mw-redirect" href="/wiki/%EC%96%B4%ED%8D%BC_%EC%9D%B4%EC%8A%A4%ED%8A%B8_%EC%82%AC%EC%9D%B4%EB%93%9C" title="어퍼 이스트 사이드">어퍼 이스트 사이드</a>, <a class="mw-redirect" href="/wiki/%EC%8A%A4%ED%8C%A8%EB%8B%88%EC%89%AC_%ED%95%A0%EB%A0%98" title="스패니쉬 할렘">스패니쉬 할렘</a>, <a href="/wiki/%ED%95%A0%EB%A0%98" title="할렘">할렘</a> 지역을 통과하고 분위기나 거리는 곳곳에서 다양하다.</p>
