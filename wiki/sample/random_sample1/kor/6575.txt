<h1 class="firstHeading" id="firstHeading" lang="ko">보안 (아일랜드 신화)</h1>

<p><b>보안</b>(<i>Boann</i> 또는 <i>Boand</i>)은 <a href="/wiki/%EC%95%84%EC%9D%BC%EB%9E%9C%EB%93%9C_%EC%8B%A0%ED%99%94" title="아일랜드 신화">아일랜드 신화</a>에 등장하는 <a class="new" href="/w/index.php?title=%EB%B3%B4%EC%9D%B8_%EA%B0%95&amp;action=edit&amp;redlink=1" title="보인 강 (없는 문서)">보인 강</a>의 여신이다. 보인 강은 <a href="/wiki/%EC%95%84%EC%9D%BC%EB%9E%9C%EB%93%9C" title="아일랜드">아일랜드</a>의 <a href="/wiki/%EB%A0%8C%EC%8A%A4%ED%84%B0" title="렌스터">렌스터</a>에 있다. 《<a href="/wiki/%EC%97%90%EB%A6%B0_%EC%B9%A8%EB%9E%B5%EC%9D%98_%EC%84%9C" title="에린 침략의 서">에린 침략의 서</a>》에 따르면, 보안은 <a href="/wiki/%ED%88%AC%EC%95%84%ED%95%98_%EB%8D%B0_%EB%8B%A4%EB%82%9C" title="투아하 데 다난">투아하 데 다난</a>의 엘라하의 아들 델베흐의 딸이었다.<sup class="reference" id="cite_ref-1"><a href="#cite_note-1">[1]</a></sup> 보안의 남편은 <a class="new" href="/w/index.php?title=%EB%84%A4%ED%81%AC%ED%83%84&amp;action=edit&amp;redlink=1" title="네크탄 (없는 문서)">네크탄</a> 또는 <a class="new" href="/w/index.php?title=%EC%97%98%ED%81%AC%EC%99%80%EB%A5%B4&amp;action=edit&amp;redlink=1" title="엘크와르 (없는 문서)">엘크와르</a> 혹은 <a class="mw-redirect" href="/wiki/%EB%88%84%EC%95%84%EB%8B%A4" title="누아다">누아다</a>라고 하지만 <a href="/wiki/%EB%8B%A4%EA%B7%B8%EB%8B%A4_%EB%AA%A8%EB%A5%B4" title="다그다 모르">다그다</a>와 관계하여 <a href="/wiki/%EC%95%99%EA%B5%AC%EC%8A%A4" title="앙구스">앙구스</a>를 낳았다. 자신의 부정을 감추기 위해 다그다는 <a href="/wiki/%ED%83%9C%EC%96%91" title="태양">태양</a>이 한 자리에 9달 동안 멈춰 있도록 마법을 걸었고, 앙구스는 수태된 지 하루만에 태어난 것이 되었다.<sup class="reference" id="cite_ref-2"><a href="#cite_note-2">[2]</a></sup></p>
<p>또한 <i><a class="new" href="/w/index.php?title=Metrical_Dindshenchas&amp;action=edit&amp;redlink=1" title="Metrical Dindshenchas (없는 문서)">metrical Dindshenchas</a></i>에는 보안이 <a class="new" href="/w/index.php?title=%EB%B3%B4%EC%9D%B8_%EA%B0%95&amp;action=edit&amp;redlink=1" title="보인 강 (없는 문서)">보인 강</a>을 만든 이야기가 나온다.<sup class="reference" id="cite_ref-3"><a href="#cite_note-3">[3]</a></sup> 남편이 금지했음에도 불구하고, 보안은 <a class="new" href="/w/index.php?title=%EC%A7%80%ED%98%9C%EC%9D%98_%EC%9A%B0%EB%AC%BC&amp;action=edit&amp;redlink=1" title="지혜의 우물 (없는 문서)">지혜의 우물</a>에 접근했다. 이때 우물 주위에 우거진 <a href="/wiki/%EA%B0%9C%EC%95%94%EB%82%98%EB%AC%B4" title="개암나무">개암나무</a>에서 개암 열매가 우물로 떨어져 거기 살고 있던 <a href="/wiki/%EC%97%B0%EC%96%B4" title="연어">연어</a>가 그것을 삼켰다(아일랜드 신화에서 개암 열매와 연어는 <a href="/wiki/%EC%A7%80%ED%98%9C" title="지혜">지혜</a>의 상징이다). 보안은 우물 주위를 시계 반대 방향으로 돌면서 우물의 힘에 도전했는데, 이 때문에 우물물이 폭발적으로 끓어오르더니 우물 밖으로 쏟아져 나와 바다를 향해 흘러갔다. 이 물길이 지금의 보인 강이 되었다는 것이다. 이 재앙의 와중에 보안은 홍수에 휩쓸려 팔을 잃고, 그 다음에 다리와 눈을 잃고는 최후에 목숨까지 잃고 말았다. 시인은 <a href="/wiki/%EC%84%B8%EB%B2%88_%EA%B0%95" title="세번 강">세번 강</a>, <a class="mw-redirect" href="/wiki/%ED%8B%B0%EB%B2%A0%EB%A5%B4_%EA%B0%95" title="티베르 강">티베르 강</a>, <a href="/wiki/%EC%9A%94%EB%A5%B4%EB%8B%A8_%EA%B0%95" title="요르단 강">요르단 강</a> 등 다른 나라의 유명한 강들과 보인 강을 동등하게 취급하고 있다.</p>
<p>현대의 주석가들과 네오페이건들은 보안을 <a href="/wiki/%EB%B8%8C%EB%A6%AC%EC%9D%B4%EB%93%9C" title="브리이드">브리이드</a>와 동일시하거나 브리이드의 어머니로 취급하기도 한다.<sup class="reference" id="cite_ref-4"><a href="#cite_note-4">[4]</a></sup> 그러나 켈트 신화 상에 그런 설을 입증할 증거는 존재하지 않는다.</p>
