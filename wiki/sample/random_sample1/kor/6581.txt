<h1 class="firstHeading" id="firstHeading" lang="ko">보육원</h1>

<p><b>보육원</b>(保育院)은 보호자가 없는 고아(孤兒)·기아(棄兒)·미아(迷兒) 등을 수용하여 일정한 연령까지 양육하는 시설이다. 보육원에 수용된 아동은 보육원에서 일정한 나이까지 성장하기도 하고 일정한 자격을 갖춘 <a href="/wiki/%EA%B0%80%EC%A0%95" title="가정">가정</a>으로 <a href="/wiki/%EC%96%91%EC%9E%90_(%EB%8C%80%ED%95%9C%EB%AF%BC%EA%B5%AD_%EB%AF%BC%EB%B2%95)" title="양자 (대한민국 민법)">양자</a>로 들어가기도 한다.<sup class="reference" id="cite_ref-1"><a href="#cite_note-1">[1]</a></sup> <sup class="reference" id="cite_ref-2"><a href="#cite_note-2">[2]</a></sup> <sup class="reference" id="cite_ref-3"><a href="#cite_note-3">[3]</a></sup> <sup class="reference" id="cite_ref-4"><a href="#cite_note-4">[4]</a></sup> <sup class="reference" id="cite_ref-5"><a href="#cite_note-5">[5]</a></sup></p>
<p>보육원은 그리스·로마시대부터 있었고 중세에는 교회·사원·길드·자선단체 등에서 설치하여 운영하였다. 현대적 의미의 최초의 보육원은 <a href="/wiki/1698%EB%85%84" title="1698년">1698년</a> A.H.프랑케가 <a href="/wiki/%ED%95%A0%EB%A0%88" title="할레">할레</a>에 설립한 것으로 독일 각지에 보육원의 개설을 촉진시켰다. 18세기 <a href="/wiki/%EC%9A%94%ED%95%9C_%ED%95%98%EC%9D%B8%EB%A6%AC%ED%9E%88_%ED%8E%98%EC%8A%A4%ED%83%88%EB%A1%9C%EC%B9%98" title="요한 하인리히 페스탈로치">요한 하인리히 페스탈로치</a>가 실시한 고아교육사업은 전 유럽의 주목을 끌었다.</p>
<p><a href="/wiki/%EB%8C%80%ED%95%9C%EB%AF%BC%EA%B5%AD" title="대한민국">대한민국</a>의 <a href="/wiki/%EC%95%84%EB%8F%99%EB%B3%B5%EC%A7%80%EB%B2%95" title="아동복지법">아동복지법</a>에 의하면 0세부터 5세 미만의 아이를 수용하는 시설은 '영아원'이고, 5세부터 18세까지의 아이를 수용하는 시설은 '육아원'이다. 개정된 법에 의해 <a href="/wiki/2014%EB%85%84" title="2014년">2014년</a> 현재 모든 보육원이 0세부터 18세까지의 아동을 보호한다. <a href="/wiki/%EA%B3%A0%EB%A0%A4" title="고려">고려</a>시대에 고아를 사원에 집단적으로 수용하여 보호하던 예가 있고, <a href="/wiki/%EC%A1%B0%EC%84%A0" title="조선">조선</a>시대의 현종·숙종·영조·정조 때에는 국가적으로 고아사업을 실시하였다. 현대적 의미의 보육원은 <a href="/wiki/1888%EB%85%84" title="1888년">1888년</a> <a href="/wiki/%ED%94%84%EB%9E%91%EC%8A%A4" title="프랑스">프랑스</a> 교회가 <a href="/wiki/%EB%AA%85%EB%8F%99" title="명동">명동</a>에 설치·운영한 천주교 보육원을 들 수 있다.</p>
