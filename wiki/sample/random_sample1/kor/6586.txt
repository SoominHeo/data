<p><b>보잉 787 드림라이너</b>(<span style="font-size: smaller;"><a href="/wiki/%EC%98%81%EC%96%B4" title="영어">영어</a>:</span> <span lang="en" xml:lang="en">Boeing 787 Dreamliner</span>)는 <a class="mw-redirect" href="/wiki/%EB%B3%B4%EC%9E%89_%EC%82%AC" title="보잉 사">보잉 사</a>의 중형 쌍발 <a href="/wiki/%EA%B4%91%ED%8F%AD%EB%8F%99%EC%B2%B4_%ED%95%AD%EA%B3%B5%EA%B8%B0" title="광폭동체 항공기">광동체</a> 여객기다. <a href="/wiki/%EB%B3%B4%EC%9E%89_757" title="보잉 757">보잉 757</a> 및 <a href="/wiki/%EB%B3%B4%EC%9E%89_767" title="보잉 767">보잉 767</a>을 대체하는 모델이다.</p>
<p><a href="/wiki/%EB%B3%B4%EC%9E%89" title="보잉">보잉</a>의 항공기 중 처음으로 기체 대부분에 <a class="new" href="/w/index.php?title=%ED%83%84%EC%86%8C%EB%B3%B5%ED%95%A9&amp;action=edit&amp;redlink=1" title="탄소복합 (없는 문서)">탄소복합</a> 재료를 사용한 비행기이다. 개발 코드는 본래 7E7(효율성 Efficient, 경제성 Economic 친환경 Environment)이었으나, <a href="/wiki/2005%EB%85%84" title="2005년">2005년</a> <a href="/wiki/1%EC%9B%94_28%EC%9D%BC" title="1월 28일">1월 28일</a> 787로 변경하였다. 역사상 가장 짧은 기간 동안 가장 많이 판매된 광폭 항공기이기도 하다.<sup class="reference" id="cite_ref-1"><a href="#cite_note-1">[1]</a></sup></p>
<p>2012년 <a href="/wiki/%EB%A6%AC%ED%8A%AC_%EC%9D%B4%EC%98%A8_%EC%A0%84%EC%A7%80" title="리튬 이온 전지">리튬 이온 전지</a>로 인한 기체 결함으로 사고가 잇따르자, <a class="mw-redirect" href="/wiki/%EB%AF%B8%EA%B5%AD_%EC%97%B0%EB%B0%A9%ED%95%AD%EA%B3%B5%EC%B2%AD" title="미국 연방항공청">FAA</a>에서는 보잉 787에 대해 운항 중지 명령<sup class="reference" id="cite_ref-2"><a href="#cite_note-2">[2]</a></sup>을 내렸으나 현재는 해제된 상태다.</p>
<p><a href="/wiki/%EB%8C%80%ED%95%9C%EB%AF%BC%EA%B5%AD" title="대한민국">대한민국</a> 국적사들 중에서는 <a href="/wiki/%EB%8C%80%ED%95%9C%ED%95%AD%EA%B3%B5" title="대한항공">대한항공</a>만 787-9를 주문하였다.</p>

<p><a href="/wiki/1990%EB%85%84%EB%8C%80" title="1990년대">1990년대</a> 후반 <a href="/wiki/%EB%B3%B4%EC%9E%89" title="보잉">보잉</a>은 <a href="/wiki/%EC%97%90%EC%96%B4%EB%B2%84%EC%8A%A4" title="에어버스">에어버스</a> <a href="/wiki/%EC%97%90%EC%96%B4%EB%B2%84%EC%8A%A4_A330" title="에어버스 A330">A330-200</a>의 판매로 인해 <a href="/wiki/%EB%B3%B4%EC%9E%89_767" title="보잉 767">보잉 767</a>의 판매가 감소하자, <a href="/wiki/%EB%B3%B4%EC%9E%89_767" title="보잉 767">보잉 767</a>의 대체 기종의 생산을 고려하였다. 또한 <a href="/wiki/%EB%B3%B4%EC%9E%89_747" title="보잉 747">보잉 747-400</a>의 판매도 감소하자 <a href="/wiki/%EB%B3%B4%EC%9E%89" title="보잉">보잉</a>은 개발이 완료된다는 조건 아래 <a href="/wiki/%EC%97%90%EC%96%B4%EB%B2%84%EC%8A%A4_A380" title="에어버스 A380">에어버스 A380</a>에 대응해 <a href="/wiki/%EB%B3%B4%EC%9E%89_747" title="보잉 747">보잉 747-400</a>을 늘이고 복합재료 <a class="new" href="/w/index.php?title=%EC%B4%88%EC%9E%84%EA%B3%84%EC%9D%B5&amp;action=edit&amp;redlink=1" title="초임계익 (없는 문서)">초임계익</a>을 사용해 효율을 높인 <a href="/wiki/%EB%B3%B4%EC%9E%89_747" title="보잉 747">보잉 747X</a> (<a href="/wiki/%EB%B3%B4%EC%9E%89_747-8" title="보잉 747-8">보잉 747-8I</a>)와 <a class="mw-redirect" href="/wiki/%EB%A7%88%ED%95%98" title="마하">마하</a> 0.98에 가까운 속도를 내면서 연료소모는 <a href="/wiki/%EB%B3%B4%EC%9E%89_767" title="보잉 767">보잉 767</a>과 비슷한 <a href="/wiki/%EB%B3%B4%EC%9E%89_%EC%86%8C%EB%8B%89_%ED%81%AC%EB%A3%A8%EC%A0%80" title="보잉 소닉 크루저">보잉 소닉 크루저</a>(<span style="font-size: smaller;"><a href="/wiki/%EC%98%81%EC%96%B4" title="영어">영어</a>:</span> <span lang="en" xml:lang="en">Sonic Cruiser</span>), 두 개의 신형 항공기를 <a class="mw-redirect" href="/wiki/%EC%BD%98%ED%8B%B0%EB%84%A8%ED%83%88_%ED%95%AD%EA%B3%B5" title="콘티넨탈 항공">콘티넨탈 항공</a>을 비롯한 <a href="/wiki/%EB%AF%B8%EA%B5%AD" title="미국">미국</a>의 주요 항공사들에 제안하였다.</p>
<p>제안 당시 항공사들은 비록 운항 비용에 대해 걱정하면서 비행 시간 단축이 고객 만족과 비행기 운영의 효울성을 증대시킬 수 있기에 소닉 크루저에 관심을 보였으나 <a href="/wiki/2001%EB%85%84" title="2001년">2001년</a> <a href="/wiki/9%C2%B711_%ED%85%8C%EB%9F%AC" title="9·11 테러">9·11 테러</a>로 인한 항공 수요 감소와 유가 상승으로 인해 항공사들은 속도보다는 연료효율에 더 관심을 가지게 되면서 기존에 개발 중이였던 소닉크루저를 <a href="/wiki/2002%EB%85%84" title="2002년">2002년</a> <a href="/wiki/12%EC%9B%94_20%EC%9D%BC" title="12월 20일">12월 20일</a> 자로 취소하고, 대체기종인 7E7을 제안하게 되었는데 이것이 보잉 787이었다.</p>

<p>일찍이 공개된 구상은 급진적이고(radical design) 매우 유연(highly curved)하였다. 그러나 <a href="/wiki/%EB%B3%B4%EC%9E%89" title="보잉">보잉</a>은 프로그램이 시작된 지 1년 뒤인 <a href="/wiki/2005%EB%85%84" title="2005년">2005년</a> <a href="/wiki/4%EC%9B%94_26%EC%9D%BC" title="4월 26일">4월 26일</a>, 좀 더 고전적인 디자인으로 확정되었다.</p>
<p>원래 <a href="/wiki/2007%EB%85%84" title="2007년">2007년</a> <a href="/wiki/9%EC%9B%94" title="9월">9월</a>에 최초 비행을 거쳐 <a href="/wiki/2008%EB%85%84" title="2008년">2008년</a> <a href="/wiki/5%EC%9B%94" title="5월">5월</a>에 운항을 시작할 예정이었으나, 최초 비행은 <a href="/wiki/2009%EB%85%84" title="2009년">2009년</a> <a href="/wiki/12%EC%9B%94_15%EC%9D%BC" title="12월 15일">12월 15일</a> <a href="/wiki/%EC%8B%9C%EC%95%A0%ED%8B%80" title="시애틀">시애틀</a> 근교 <a href="/wiki/%EB%B3%B4%EC%9E%89" title="보잉">보잉</a> 공장과 인접한 <a class="new" href="/w/index.php?title=%ED%8E%98%EC%9D%B8%ED%95%84%EB%93%9C_%EA%B3%B5%ED%95%AD&amp;action=edit&amp;redlink=1" title="페인필드 공항 (없는 문서)">페인필드 공항</a>에서 시행하였으며 <a href="/wiki/2011%EB%85%84" title="2011년">2011년</a> 초, <a href="/wiki/%EC%A0%84%EC%9D%BC%EB%B3%B8%EA%B3%B5%EC%88%98" title="전일본공수">ANA</a>에 인도되었다.</p>
<h2><span class="mw-headline" id=".EA.B8.B0.EC.A1.B4_.ED.95.AD.EA.B3.B5.EA.B8.B0.EC.99.80.EC.9D.98_.EC.B0.A8.EC.9D.B4.EC.A0.90">기존 항공기와의 차이점</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%B3%B4%EC%9E%89_787_%EB%93%9C%EB%A6%BC%EB%9D%BC%EC%9D%B4%EB%84%88&amp;action=edit&amp;section=2" title="부분 편집: 기존 항공기와의 차이점">원본 편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<ul>
<li>경제성: 동체가 대부분 탄소섬유로 제작되기 때문에 내구성 향상은 물론 제작과정의 단순화가 가능해져 종래 항공기에서 보기 드문 경제성과 또한 승객 편의성을 자랑한다.</li>
<li>생산과정 : 탄소섬유로 만들어지기 때문에 787의 동체는 거대한 터널 모양의 <a class="new" href="/w/index.php?title=%EC%98%A4%ED%86%A0_%ED%81%B4%EB%A0%88%EC%9D%B4%EB%B8%8C&amp;action=edit&amp;redlink=1" title="오토 클레이브 (없는 문서)">오토 클레이브</a>라는 온도를 전반적으로 일정히 유지시킬 수 있는 특수 오븐에서 구워진다.</li>
<li>승객 편의: 보잉 787은 기내 압력이 6,000피트 수준으로 유지되고 또한 습도가 종전 항공기에 비해 40~60%에서 유지됨으로써 종전의 항공기에서 보기 힘든 승객 편리를 제공한다.</li>
<li>추진력: <a href="/wiki/%EC%A0%9C%EB%84%88%EB%9F%B4_%EC%9D%BC%EB%A0%89%ED%8A%B8%EB%A6%AD" title="제너럴 일렉트릭">GE</a> 사제 GEnX,<a href="/wiki/%EB%A1%A4%EC%8A%A4%EB%A1%9C%EC%9D%B4%EC%8A%A4_plc" title="롤스로이스 plc">롤스로이스</a> 사제 Trent 1000을 사용, 증가된 추력에도 불구, 소음은 더 적고 뛰어난 연비를 제공할 예정이다.</li>
<li>소음: 엔진 나셀 후방부의 모양을 물결모양으로 만들어 다른 여객기에 비해 소음을 낮춰 승객들을 더 편안하게 한다</li>
<li>외장 : 공기역학적으로 효율적인 상태에서 나타나는 <a href="/wiki/%EC%B8%B5%EB%A5%98" title="층류">층류</a>가 엔진 파일론에 덧칠해진 페인트로 인해 손상되는 것을 방지하기 위해 엔진 파일론이 회색의 특수 페인트로 칠해졌다</li>
</ul>

<h2><span class="mw-headline" id=".ED.8C.8C.EC.83.9D.ED.98.95">파생형</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%B3%B4%EC%9E%89_787_%EB%93%9C%EB%A6%BC%EB%9D%BC%EC%9D%B4%EB%84%88&amp;action=edit&amp;section=4" title="부분 편집: 파생형">원본 편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>보잉 787은 파생형에 따라 210명에서 290명까지 탑승할 수 있다.</p>

<p>787-3은 항속거리 4600~5600 km로 주로 수요가 많은 중거리 노선에 적합하도록 개발되었다. 현재의 787 시리즈와는 달리 윙렛이 장착된 디자인으로 시작되었으며 기존의 <a href="/wiki/%EB%B3%B4%EC%9E%89_757" title="보잉 757">보잉 757-200/-300</a>과 <a href="/wiki/%EB%B3%B4%EC%9E%89_767" title="보잉 767">보잉 767-200ER</a>기를 대체하기 위하여 개발 되었다. 최초로 주문한 회사는 <a href="/wiki/%EC%A0%84%EC%9D%BC%EB%B3%B8%EA%B3%B5%EC%88%98" title="전일본공수">전일본공수</a>이나 주문 부족과 787-8의 개발 지연으로 개발이 취소되었다.</p>

<a href="/wiki/%EC%97%90%EC%96%B4_%EB%89%B4%EC%A7%88%EB%9E%9C%EB%93%9C" title="에어 뉴질랜드">에어 뉴질랜드</a>의 <a href="/wiki/%EB%B3%B4%EC%9E%89" title="보잉">보잉</a> 787-9</div>

<p>최대 항속거리 15,700km다. 787-8의 동체 연장형으로, 초장거리 노선의 운영에 적합한 모델이다. <a href="/wiki/%EC%97%90%EC%96%B4_%EB%89%B4%EC%A7%88%EB%9E%9C%EB%93%9C" title="에어 뉴질랜드">에어 뉴질랜드</a>에서 <a href="/wiki/2006%EB%85%84" title="2006년">2006년</a>에 최초로 주문하였으며, <a href="/wiki/2014%EB%85%84" title="2014년">2014년</a>에 인도받았다. 최초로 인도받은 회사는 <a href="/wiki/%EC%A0%84%EC%9D%BC%EB%B3%B8%EA%B3%B5%EC%88%98" title="전일본공수">전일본공수</a>이며, <a href="/wiki/2014%EB%85%84" title="2014년">2014년</a> 8월부터 운항에 들어갔다. <a href="/wiki/%EB%8C%80%ED%95%9C%ED%95%AD%EA%B3%B5" title="대한항공">대한항공</a>이 <a href="/wiki/2011%EB%85%84" title="2011년">2011년</a>에 787-8에서 이 모델로 주문 기종을 변경하였다. <a href="/wiki/KLM" title="KLM">KLM</a>도 787-9를 주문하였다.</p>
<h3><span class="mw-headline" id="787-10">787-10</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%B3%B4%EC%9E%89_787_%EB%93%9C%EB%A6%BC%EB%9D%BC%EC%9D%B4%EB%84%88&amp;action=edit&amp;section=8" title="부분 편집: 787-10">원본 편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<div class="thumb tright">
<div class="thumbinner" style="width:222px;"><a class="image" href="/wiki/%ED%8C%8C%EC%9D%BC:Dreamliner_render_787-10.jpg"><img alt="" class="thumbimage" data-file-height="390" data-file-width="750" height="114" src="//upload.wikimedia.org/wikipedia/commons/thumb/c/c4/Dreamliner_render_787-10.jpg/220px-Dreamliner_render_787-10.jpg" srcset="//upload.wikimedia.org/wikipedia/commons/thumb/c/c4/Dreamliner_render_787-10.jpg/330px-Dreamliner_render_787-10.jpg 1.5x, //upload.wikimedia.org/wikipedia/commons/thumb/c/c4/Dreamliner_render_787-10.jpg/440px-Dreamliner_render_787-10.jpg 2x" width="220"/></a>
<div class="thumbcaption">
<div class="magnify"><a class="internal" href="/wiki/%ED%8C%8C%EC%9D%BC:Dreamliner_render_787-10.jpg" title="실제 크기로"></a></div>
<a href="/wiki/%EB%B3%B4%EC%9E%89" title="보잉">보잉</a> 787-10</div>
</div>
</div>
<p>787-9의 동체 연장형으로 최대 항속거리 14,800 km으로 쌍발기 항속거리의 대부분을 커버할 수 있도록 개발되었다. 기존의 <a href="/wiki/%EB%B3%B4%EC%9E%89_777" title="보잉 777">보잉 777-200</a>, <a href="/wiki/%EC%97%90%EC%96%B4%EB%B2%84%EC%8A%A4_A330" title="에어버스 A330">에어버스 A330-300</a>, <a href="/wiki/%EC%97%90%EC%96%B4%EB%B2%84%EC%8A%A4_A340" title="에어버스 A340">에어버스 A340-200/-300</a>을 대체하기 위해 개발되었다. 본 모델은 아직 개발중에 있으며 <a href="/wiki/2018%EB%85%84" title="2018년">2018년</a>부터 생산이 시작될 예정이다.</p>

<h2><span class="mw-headline" id=".EB.85.BC.EB.9E.80">논란</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%B3%B4%EC%9E%89_787_%EB%93%9C%EB%A6%BC%EB%9D%BC%EC%9D%B4%EB%84%88&amp;action=edit&amp;section=16" title="부분 편집: 논란">원본 편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<div class="rellink relarticle mainarticle"><img alt="" data-file-height="128" data-file-width="128" height="16" src="//upload.wikimedia.org/wikipedia/commons/thumb/e/ec/Crystal_Clear_app_xmag.svg/16px-Crystal_Clear_app_xmag.svg.png" srcset="//upload.wikimedia.org/wikipedia/commons/thumb/e/ec/Crystal_Clear_app_xmag.svg/24px-Crystal_Clear_app_xmag.svg.png 1.5x, //upload.wikimedia.org/wikipedia/commons/thumb/e/ec/Crystal_Clear_app_xmag.svg/32px-Crystal_Clear_app_xmag.svg.png 2x" width="16"/> 이 부분의 본문은 <a href="/wiki/%EB%B3%B4%EC%9E%89_787_%EB%93%9C%EB%A6%BC%EB%9D%BC%EC%9D%B4%EB%84%88%EC%9D%98_%EB%B0%B0%ED%84%B0%EB%A6%AC_%EB%AC%B8%EC%A0%9C" title="보잉 787 드림라이너의 배터리 문제">보잉 787 드림라이너의 배터리 문제</a>입니다.</div>
<p>보잉 787 드림라이너에는 각종 기술적인 결함이 많이 있으나, 엔진폭발 사고 및 화재까지 일으킨 <a href="/wiki/%EC%9D%BC%EB%B3%B8%ED%95%AD%EA%B3%B5" title="일본항공">일본항공</a>의 경우 일본제 엔진에 생긴 문제나 하자가 발생되는 등 <a href="/wiki/%EB%B3%B4%EC%9E%89" title="보잉">보잉</a> 787 여객기의 <a class="mw-redirect" href="/wiki/%EC%95%88%EC%A0%84%EC%84%B1" title="안전성">안전성</a> 문제가 크게 거론되기도 하여 운항계획을 유보 및 연기되는 일이 있다. 그래서 Nightmareliner(악몽라이너)라는 별명까지 붙여지기도 하였다.</p>
<h2><span class="mw-headline" id=".EA.B8.B0.ED.83.80">기타</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%B3%B4%EC%9E%89_787_%EB%93%9C%EB%A6%BC%EB%9D%BC%EC%9D%B4%EB%84%88&amp;action=edit&amp;section=17" title="부분 편집: 기타">원본 편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<table class="notice section" style="text-align: left; margin:0 0 0.5em 0.5em; clear:none; font-size:smaller;">
<tr>
<td width="auto"><img alt="" data-file-height="204" data-file-width="262" height="19" src="//upload.wikimedia.org/wikipedia/commons/thumb/6/64/Question_book-4.svg/25px-Question_book-4.svg.png" srcset="//upload.wikimedia.org/wikipedia/commons/thumb/6/64/Question_book-4.svg/38px-Question_book-4.svg.png 1.5x, //upload.wikimedia.org/wikipedia/commons/thumb/6/64/Question_book-4.svg/50px-Question_book-4.svg.png 2x" width="25"/> 이 문단의 <b>내용은</b> <b><a href="/wiki/%EC%9C%84%ED%82%A4%EB%B0%B1%EA%B3%BC:%EC%B6%9C%EC%B2%98_%EB%B0%9D%ED%9E%88%EA%B8%B0" title="위키백과:출처 밝히기">출처</a>가 분명하지 않습니다.</b> 지금 바로 이 <span class="plainlinks"><a class="external text" href="//ko.wikipedia.org/w/index.php?title=%EB%B3%B4%EC%9E%89_787_%EB%93%9C%EB%A6%BC%EB%9D%BC%EC%9D%B4%EB%84%88&amp;action=edit">문단을 편집</a></span>하여, 참고하신 문헌이나 <a href="/wiki/%EC%9C%84%ED%82%A4%EB%B0%B1%EA%B3%BC:%EC%8B%A0%EB%A2%B0%ED%95%A0_%EC%88%98_%EC%9E%88%EB%8A%94_%EC%B6%9C%EC%B2%98" title="위키백과:신뢰할 수 있는 출처">신뢰할 수 있는 출처</a>를 각주 등으로 표기해 주세요. <a href="/wiki/%EC%9C%84%ED%82%A4%EB%B0%B1%EA%B3%BC:%ED%99%95%EC%9D%B8_%EA%B0%80%EB%8A%A5" title="위키백과:확인 가능">검증</a>되지 않은 내용은 삭제될 수도 있습니다. 내용에 대한 의견이 있으시다면 <a href="/wiki/%ED%86%A0%EB%A1%A0:%EB%B3%B4%EC%9E%89_787_%EB%93%9C%EB%A6%BC%EB%9D%BC%EC%9D%B4%EB%84%88#.23" title="토론:보잉 787 드림라이너">토론 문서</a>에서 나누어 주세요. (2010년 2월 12일에 문단의 출처가 요청되었습니다.)</td>
</tr>
</table>
<ul>
<li>보잉 787의 날개 말단의 레이키드 윙팁을 대한항공에서 담당하고 있다.<sup class="reference" id="cite_ref-10"><a href="#cite_note-10">[10]</a></sup>. 대한항공은 777-300ER과 777-200LR의 유사 부품을 제작하고 있다. <a class="mw-redirect" href="/wiki/KAI" title="KAI">KAI</a>에서도 보잉 787의 부품을 제작하고 있다.</li>
<li>대한민국의 중소기업 하이즈 항공은 보잉 787기의 중앙 날개 및 좌우 주날개의 조향장치 조립을 맡고 있다.<sup class="reference" id="cite_ref-11"><a href="#cite_note-11">[11]</a></sup></li>
<li>보잉 787은 민간여객기로서는 최초로 탄소섬유강화 플라스틱(CFRP:Carbon Fibre Reinforced Plastic)을 동체 구조물에 사용하였으나 제작 공정중 작은 기포가 구조물에 생길 경우 그 반복되는 비행을 통해 기포의 크기가 증가하면서 기체의 피로도 증가에 기여할 수 있고, 증가한 피로도로 동체가 파손될 경우 치명적인 사고로 이어질 수 있다.</li>
<li>FAA가 소재의 화재시 내구성 등이 비교적 덜 알려짐 점을 이유로 화재 부분에 관해서 좀 더 정밀한 테스트를 요구하였다.</li>
<li>이 비행기 기종의 이름을 따서 2010년에 치러진 온게임넷의 <a href="/wiki/%EB%8C%80%ED%95%9C%ED%95%AD%EA%B3%B5_%EC%8A%A4%ED%83%80%EB%A6%AC%EA%B7%B8_2010_Season_2" title="대한항공 스타리그 2010 Season 2">대한항공 스타리그 2010 Season 2</a>의 공식맵 중 하나는 비상 드림라이너로 명명되었다. 실제로 맵의 모양도 비행기의 모양을 따서 만들어졌다.</li>
</ul>
