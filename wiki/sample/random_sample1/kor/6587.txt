<p><b>보잉 X-32</b>은 미국을 주도로 F시리즈 전투기들을 운용하고 있는 영국 및 다른 유럽 국가들이 공동으로 참여하고 있는 프로젝트인 차세대 전투기 사업 <b><a class="new" href="/w/index.php?title=%ED%86%B5%ED%95%A9%EA%B3%B5%EA%B2%A9%EA%B8%B0&amp;action=edit&amp;redlink=1" title="통합공격기 (없는 문서)">통합공격기</a></b>(JSF: Joint Strike Fighter) 개발 프로젝트에서 만든 실험기체 중 하나이다. 미국 <a href="/wiki/%EB%B3%B4%EC%9E%89" title="보잉">보잉</a>사가 개발했으며, 설계안대로 실제 실험기체 형태로 제작 운영한 기체기종을 개발해서 X 명칭을 붙였다. 5세대 <a href="/wiki/%EC%8A%A4%ED%85%94%EC%8A%A4%EA%B8%B0" title="스텔스기">스텔스 전투기</a>이며, <a class="mw-disambig" href="/wiki/%ED%95%B4%EB%A6%AC%EC%96%B4" title="해리어">해리어</a>처럼 <a class="mw-redirect" href="/wiki/%EC%88%98%EC%A7%81%EC%9D%B4%EC%B0%A9%EB%A5%99%EA%B8%B0" title="수직이착륙기">수직이착륙기</a>이다. 또한 미국 <a class="new" href="/w/index.php?title=%EB%A1%9D%ED%9E%88%EB%93%9C%EB%A7%88%ED%8B%B4%EC%82%AC&amp;action=edit&amp;redlink=1" title="록히드마틴사 (없는 문서)">록히드마틴사</a>가 개발한 <b><a href="/wiki/F-35_%EB%9D%BC%EC%9D%B4%ED%8A%B8%EB%8B%9D_II" title="F-35 라이트닝 II">X-35</a></b>(현개발중인 <a href="/wiki/F-35_%EB%9D%BC%EC%9D%B4%ED%8A%B8%EB%8B%9D_II" title="F-35 라이트닝 II">F-35</a>)의 경쟁상대였다. 2000년 9월 미국 에드우드 공군기지에서 X-32A의 첫 시험 비행을 했으며, 단거리이륙과 수직착륙의 실험비행은 2001년 3월에 X-32B을 통해 실행되었다.</p>
<p>통합공격기 개발 산업에 처음 4개의 업체가 참여했지만 최종적으로 보잉의 X-32와 록히드 마틴의 <a href="/wiki/F-35_%EB%9D%BC%EC%9D%B4%ED%8A%B8%EB%8B%9D_II" title="F-35 라이트닝 II">X-35</a> (현재 개발중인 <a href="/wiki/F-35_%EB%9D%BC%EC%9D%B4%ED%8A%B8%EB%8B%9D_II" title="F-35 라이트닝 II">F-35</a>)만 남아 나머지 업체는 경쟁에서 탈락했다. 록히드마틴사의 X-35는 최대한 기존의 기술을 적용하고 핵심적인 부분에만 신기술을 적용한 것과 비교해보면, 보잉사의 X-32는 최대한 미래지향적인 디자인과 신기술을 접목했다는 평가다. 또한 스텔스 성능면에서도 스텔스 기능을 강화하기 위해 날개의 수를 줄여 X-35보다 뛰어나고, 유형별로 분리된 X-35와 달리 간단한 부품 교체를 통해서 수직이착륙기과 초음속 기동이 가능하도록 설계되었다. 경쟁했던 X-35가 성능을 위주로 개발로 기체가격이 높으것에 비해 상대적 X-32는 저렴한 기체 가격이 장점으로 볼 수 있다.</p>
<p>하지만 기존 전투기와 다른 이질적인 디자인, 기존 방식과 다른 설계, 날개와 일체된 기체 제작의 어려움, 개발 당시 잦은 설계 변경과 X-32 실험기체와 향후 실전 기체와 설계 변경 차이가 큰 점, 가격 상승률이 개발 과정에서 상대적으로 높아질수 있다는 점, 수직이착륙의 안정성의 문제 ( 엔진 노즐에서 발생되는 기류가 다시 기체의 엔진 안으로 들어가 작동 엔진에 문제가 생기는 문제로 기존 <a href="/wiki/%ED%95%B4%EB%A6%AC%EC%96%B4%EA%B8%B0" title="해리어기">해리어기</a>가 가지고 있던 결함과 유사함)보였다.</p>
<p>특히 록히드 마틴사의 <a class="mw-redirect" href="/wiki/X-35" title="X-35">X-35</a>는 리프트팬을 따로 장착해서 엔진 노즐에서 발생한 기류가 다시 엔진으로 들어가는 것 방지함으로 수직이착륙시에 안정성이 상대적으로 더 높았다. 또한 X-32는 실험 기동에서 수직이착륙에서 기체 무게를 가볍게 하기 위해 부품을 떼어내고 초음속 기동시에 부품을 다시 장착했던 실험 기동 과정에 비해서 [[X-35]은 별도의 부품 탈부착 과정이 없이 단거리 이륙과 수직 착륙과 함께 동시에 초음속 기동 실험 기동에서 차이를 보여 주었다. 결국 차세대전투기 개발 프로젝트 경쟁에서 <a href="/wiki/F-35_%EB%9D%BC%EC%9D%B4%ED%8A%B8%EB%8B%9D_II" title="F-35 라이트닝 II">X-35</a>에 밀려났다.</p>
