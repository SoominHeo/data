<h2><span class="mw-headline" id=".EC.82.AC.EA.B1.B4">사건</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=5%EC%9B%94_27%EC%9D%BC&amp;action=edit&amp;section=1" title="부분 편집: 사건">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<ul>
<li><a href="/wiki/1829%EB%85%84" title="1829년">1829년</a> - 프로이센, 헤센, 바이에른, 뷔르템베르크 사이에 통상조약 체결. 훗날 <a class="mw-redirect" href="/wiki/%EB%8F%85%EC%9D%BC_%EA%B4%80%EC%84%B8_%EB%8F%99%EB%A7%B9" title="독일 관세 동맹">독일 관세 동맹</a>으로 발전.</li>
<li><a href="/wiki/1862%EB%85%84" title="1862년">1862년</a> - <a class="mw-redirect" href="/wiki/%EB%AF%B8%EA%B5%AD_%EB%82%A8%EB%B6%81%EC%A0%84%EC%9F%81" title="미국 남북전쟁">미국 남북전쟁</a>, 포터 장군의 1사단이 하노버 코트 하우스의 남군을 공격. 북군 승리</li>
<li><a href="/wiki/1905%EB%85%84" title="1905년">1905년</a> - <a href="/wiki/%EB%9F%AC%EC%9D%BC_%EC%A0%84%EC%9F%81" title="러일 전쟁">러일 전쟁</a>: <a class="mw-redirect" href="/wiki/%EC%93%B0%EC%8B%9C%EB%A7%88_%EC%A0%84%ED%88%AC" title="쓰시마 전투">쓰시마 전투</a>에서 <a href="/wiki/%EB%9F%AC%EC%8B%9C%EC%95%84" title="러시아">러시아</a>가 패하다.</li>
<li><a href="/wiki/1952%EB%85%84" title="1952년">1952년</a> - <a class="mw-redirect" href="/wiki/EU" title="EU">EU</a> <a class="new" href="/w/index.php?title=%EC%9C%A0%EB%9F%BD%EB%B0%A9%EC%9C%84%EA%B8%B0%EA%B5%AC&amp;action=edit&amp;redlink=1" title="유럽방위기구 (없는 문서)">EDC</a> 창설 조약을 체결했다.</li>
<li><a href="/wiki/1960%EB%85%84" title="1960년">1960년</a> - <a class="mw-redirect" href="/wiki/%EC%B9%B4%EB%A7%90_%EA%B7%80%EB%A5%B4%EC%85%80" title="카말 귀르셀">카말 귀르셀</a>이 <a href="/wiki/%ED%84%B0%ED%82%A4" title="터키">터키</a>의 대통령에 취임했다.</li>
<li><a href="/wiki/1980%EB%85%84" title="1980년">1980년</a> - <a class="mw-redirect" href="/wiki/%EA%B4%91%EC%A3%BC_%EB%AF%BC%EC%A3%BC%ED%99%94_%EC%9A%B4%EB%8F%99" title="광주 민주화 운동">광주 민주화 운동</a>이 전두환 군부정권에 의해 완전히 진압되다.</li>
<li><a href="/wiki/1997%EB%85%84" title="1997년">1997년</a> - 26일 오후 8시부터 <a href="/wiki/%EC%A0%84%EB%82%A8%EB%8C%80%ED%95%99%EA%B5%90" title="전남대학교">전남대학교</a> 동아리회관 지하실과 2층에서 전남대 및 조선대 <a class="new" href="/w/index.php?title=%EB%82%A8%EC%B4%9D%EB%A0%A8&amp;action=edit&amp;redlink=1" title="남총련 (없는 문서)">남총련</a> 간부 15명에게 구타당한 <a href="/wiki/%EC%9D%B4%EC%A2%85%EA%B6%8C_%EA%B3%A0%EB%AC%B8_%EC%B9%98%EC%82%AC_%EC%82%AC%EA%B1%B4" title="이종권 고문 치사 사건">이종권</a>이 새벽 3시경에 사망, 당일 오전 5시경에 학생회관 건물 근처에서 변사체로 발견되었다.</li>
<li><a href="/wiki/2005%EB%85%84" title="2005년">2005년</a> -
<ul>
<li>야치 쇼타로 <a href="/wiki/%EC%9D%BC%EB%B3%B8" title="일본">일본</a> 외무성 사무<a href="/wiki/%EC%B0%A8%EA%B4%80" title="차관">차관</a>이 자신의 발언이 오해를 일으켜 유감스럽게 생각한다고 밝혔다.</li>
<li><a href="/wiki/%EC%A0%95%EB%B6%80" title="정부">정부</a>와 각 시도지사가 <a href="/wiki/%EA%B3%B5%EA%B3%B5%EA%B8%B0%EA%B4%80" title="공공기관">공공기관</a> 지방이전 기본협약을 체결했다. 이에 따르면 <a href="/wiki/%ED%95%9C%EA%B5%AD%EC%A0%84%EB%A0%A5%EA%B3%B5%EC%82%AC" title="한국전력공사">한국전력공사</a> 본사를 유치하게 되는 지역은 한전 본사외에 관련회사 2개사만을 유치할 수 있게 된다.</li>
<li><a href="/wiki/%EB%85%B8%EB%AC%B4%ED%98%84" title="노무현">노무현</a> <a href="/wiki/%EB%8C%80%ED%86%B5%EB%A0%B9" title="대통령">대통령</a>은 <a href="/wiki/%ED%96%89%EB%8B%B4%EB%8F%84" title="행담도">행담도</a> 개발사업에 관련해 적절치 못한 직무행위를 한 문정인 동북아시대위원장과 <a href="/wiki/%EC%A0%95%ED%83%9C%EC%9D%B8" title="정태인">정태인</a> 국민경제자문회의 사무차장의 사표를 수리하기로 하다.</li>
<li>사측의 노조탄압혐의와 노조의 폭력시위로 문제가 된 울산건설플랜트 노사가 조합활동을 허용하고 불법 다단계 <a href="/wiki/%ED%95%98%EB%8F%84%EA%B8%89" title="하도급">하도급</a>을 없애는 데 대해 합의하다.</li>
</ul>
</li>
<li><a href="/wiki/2007%EB%85%84" title="2007년">2007년</a> - 베네수엘라의 대표적인 민영방송인 RCTV가 정부의 명령으로 방송을 중단하여야만 했다. 이 방송은 7월 16일 다시 케이블과 위성방송을 통해 방송을 재개하였다.</li>
<li><a href="/wiki/2008%EB%85%84" title="2008년">2008년</a> - <a href="/wiki/%EB%8C%80%ED%95%9C%EB%AF%BC%EA%B5%AD" title="대한민국">대한민국</a>과 <a href="/wiki/%EC%A4%91%ED%99%94%EC%9D%B8%EB%AF%BC%EA%B3%B5%ED%99%94%EA%B5%AD" title="중화인민공화국">중화인민공화국</a>의 관계가 ‘전략적 협력 동반자 관계’로 격상되었다.</li>
<li><a href="/wiki/2012%EB%85%84" title="2012년">2012년</a> - <a class="mw-redirect" href="/wiki/%EA%B5%AD%EC%A0%9C_%EC%97%B0%ED%95%A9_%EC%95%88%EC%A0%84_%EB%B3%B4%EC%9E%A5_%EC%9D%B4%EC%82%AC%ED%9A%8C" title="국제 연합 안전 보장 이사회">유엔 안전보장이사회</a>가 최소 116명이 사망한 <a class="mw-redirect" href="/wiki/2011~2012%EB%85%84_%EC%8B%9C%EB%A6%AC%EC%95%84_%EB%B4%89%EA%B8%B0" title="2011~2012년 시리아 봉기">시리아</a>의 <a class="new" href="/w/index.php?title=%ED%9B%8C%EB%9D%BC_%ED%95%99%EC%82%B4&amp;action=edit&amp;redlink=1" title="훌라 학살 (없는 문서)">훌라 학살</a>을 강력하게 규탄하다.</li>

<li><a href="/wiki/%EB%9D%BC%EC%98%A4%EC%8A%A4" title="라오스">라오스</a>에서 <a href="/wiki/%EB%B6%81%ED%95%9C%EC%9D%B4%ED%83%88%EC%A3%BC%EB%AF%BC" title="북한이탈주민">탈북 청소년</a> 9명이 <a href="/wiki/%EC%A1%B0%EC%84%A0%EB%AF%BC%EC%A3%BC%EC%A3%BC%EC%9D%98%EC%9D%B8%EB%AF%BC%EA%B3%B5%ED%99%94%EA%B5%AD" title="조선민주주의인민공화국">조선민주주의인민공화국</a>으로 <a href="/wiki/2013%EB%85%84_%EB%9D%BC%EC%98%A4%EC%8A%A4_%ED%83%88%EB%B6%81%EC%B2%AD%EC%86%8C%EB%85%84_%EB%B6%81%EC%86%A1_%EC%82%AC%EA%B1%B4" title="2013년 라오스 탈북청소년 북송 사건">강제 북송</a>된 것으로 알려지다.</li>
