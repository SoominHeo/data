<p><b>보퍼트 해</b>(<span lang="en" xml:lang="en">Beaufort Sea</span>, <span style="font-size: smaller;"><a href="/wiki/%ED%94%84%EB%9E%91%EC%8A%A4%EC%96%B4" title="프랑스어">프랑스어</a>:</span> <span lang="fr" xml:lang="fr">mer de Beaufort</span>)는 <a href="/wiki/%EB%85%B8%EC%8A%A4%EC%9B%A8%EC%8A%A4%ED%8A%B8_%EC%A4%80%EC%A3%BC" title="노스웨스트 준주">노스웨스트 준주</a>, <a href="/wiki/%EC%9C%A0%EC%BD%98_%EC%A4%80%EC%A3%BC" title="유콘 준주">유콘 준주</a>, <a href="/wiki/%EC%95%8C%EB%9E%98%EC%8A%A4%EC%B9%B4_%EC%A3%BC" title="알래스카 주">알래스카 주</a>의 북쪽, <a href="/wiki/%EB%B6%81%EA%B7%B9_%EC%A0%9C%EB%8F%84" title="북극 제도">북극 제도</a>의 서쪽에 펼쳐진 <a href="/wiki/%EB%B6%81%EA%B7%B9%ED%95%B4" title="북극해">북극해</a>의 일부를 말한다. 보퍼트 해의 이름은 <a href="/wiki/%EC%88%98%EB%A1%9C" title="수로">수로</a> 측량을 했던 <a class="mw-redirect" href="/wiki/%EC%98%81%EA%B5%AD_%ED%95%B4%EA%B5%B0" title="영국 해군">영국 해군</a>의 <a href="/wiki/%ED%94%84%EB%9E%9C%EC%8B%9C%EC%8A%A4_%EB%B3%B4%ED%8D%BC%ED%8A%B8" title="프랜시스 보퍼트">프랜시스 보퍼트</a>에서 왔다.</p>
<p><a class="new" href="/w/index.php?title=%EB%A9%94%EC%BC%84%EC%A7%80_%EA%B0%95&amp;action=edit&amp;redlink=1" title="메켄지 강 (없는 문서)">메켄지 강</a>을 비롯한 여러 강이 흘러든다. <a href="/wiki/%EA%B3%A0%EB%9E%98" title="고래">고래</a>를 비롯한 여러 바닷새들의 주요 서식지이다.</p>
<p>보퍼트 해는 일 년의 대부분이 <a href="/wiki/%EC%96%BC%EC%9D%8C" title="얼음">얼음</a>으로 덮여 있다.</p>
