<h1 class="firstHeading" id="firstHeading" lang="ko">보편 논쟁</h1>
<div class="mw-body-content" id="bodyContent">
<div id="siteSub">위키백과, 우리 모두의 백과사전.</div>
<div id="contentSub"></div>
<div class="mw-jump" id="jump-to-nav">
					이동:					<a href="#mw-head">둘러보기</a>, 					<a href="#p-search">검색</a>
</div>
<div class="mw-content-ltr" dir="ltr" id="mw-content-text" lang="ko"><script>function mfTempOpenSection(id){var block=document.getElementById("mf-section-"+id);block.className+=" open-block";block.previousSibling.className+=" open-block";}</script><p><b>보편 논쟁</b>(普遍論爭)에서 보편의 문제는 중세철학 전체를 일관하는 가장 중요한 문제로서 <a class="mw-redirect" href="/wiki/%EC%8A%A4%EC%BD%9C%EB%9D%BC_%EC%B2%A0%ED%95%99" title="스콜라 철학">스콜라 철학</a>은 이 문제로부터 시작해서 이 문제로 끝났다고까지 말할 수 있다.</p>
<h2><span class="mw-headline" id=".EA.B0.9C.EC.9A.94">개요</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%B3%B4%ED%8E%B8_%EB%85%BC%EC%9F%81&amp;action=edit&amp;section=1" title="부분 편집: 개요">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p><a class="mw-redirect" href="/wiki/%EC%95%84%EB%B2%A8%EB%9D%BC%EB%A5%B4%EB%91%90%EC%8A%A4" title="아벨라르두스">아벨라르두스</a>가 그의 스승 <a href="/wiki/%EC%83%B9%ED%8F%AC%EC%9D%98_%EA%B8%B0%EC%9A%A4" title="샹포의 기욤">샹포의 기욤</a>으로 하여금 자기의 학설을 변경시키게 한 문제였고, <a class="new" href="/w/index.php?title=%EC%86%94%EC%A6%88%EB%B2%A0%EB%A6%AC%EC%9D%98_%EC%9A%94%ED%95%98%EB%84%A4%EC%8A%A4&amp;action=edit&amp;redlink=1" title="솔즈베리의 요하네스 (없는 문서)">솔즈베리의 요하네스</a>에 의하면 인간이 이 문제를 해결한다는 것은 불가능하다. 논쟁의 중심점은 보편개념(類와 種)의 타당가치 문제 및 개념의 실제성·객관성의 문제이다. 초기 스콜라 철학에서는 신학상의 삼위일체론이나 교회의 보편성과 결합되어 문제가 됐고, 12세기의 사상계를 들끓게 하였다.</p>
<p>발단은 <a class="mw-redirect" href="/wiki/%EB%B3%B4%EC%97%90%ED%8B%B0%EC%9A%B0%EC%8A%A4" title="보에티우스">보에티우스</a>가 <a href="/wiki/%EB%9D%BC%ED%8B%B4%EC%96%B4" title="라틴어">라틴어</a>로 번역한 <a href="/wiki/%ED%8F%AC%EB%A5%B4%ED%94%BC%EB%A6%AC%EC%98%A4%EC%8A%A4" title="포르피리오스">포르피리오스</a>의 &lt;범주론 서설(範疇論序說)&gt;에 있다.</p>
<ol>
<li>유와 종은 실재인가, 또는 사고상의 존재인가,</li>
<li>실재한다면 물체인가, 또는 비(非)물체인가,</li>
<li>감각적 대상으로부터 분리되어 존재하는가, 또는 감각적 대상 자체 안에 존재하는가</li>
</ol>
<p>이상의 세 가지 문제를 그는 제기하였으나 해결을 보여주지는 못했던 것이다. 이 설문을 보에티우스는 &lt;범주론 주석&gt;에서 제기하고 실재하는 것은 사물인가, 또는 단순한 음성인가, 하는 형태로 변경시켰다. 사물이라고 하는 <a href="/wiki/%EC%8B%A4%EC%9E%AC%EB%A1%A0" title="실재론">실재론</a>(實在論)과 음성이라고 하는 <a href="/wiki/%EC%9C%A0%EB%AA%85%EB%A1%A0" title="유명론">유명론</a>(唯名論)에 분류되는 논쟁을 야기시켰다. <a href="/wiki/%EC%BD%A9%ED%94%BC%EC%97%90%EB%89%B4" title="콩피에뉴">콩피에뉴</a>의 <a href="/wiki/%EB%A1%9C%EC%8A%A4%EC%BC%88%EB%A6%AC%EB%88%84%EC%8A%A4" title="로스켈리누스">로스켈리누스</a>와 <a href="/wiki/%EC%83%B9%ED%8F%AC%EC%9D%98_%EA%B8%B0%EC%9A%A4" title="샹포의 기욤">샹포의 기욤</a>이 출현하자 양진영의 대립은 첨예화되었다. <a class="mw-redirect" href="/wiki/%EC%8A%A4%EC%BD%94%ED%88%AC%EC%8A%A4_%EC%97%90%EB%A6%AC%EC%9A%B0%EA%B2%8C%EB%82%98" title="스코투스 에리우게나">스코투스 에리우게나</a>로부터 <a class="mw-redirect" href="/wiki/%EC%BA%94%ED%84%B0%EB%B2%A0%EB%A6%AC%EC%9D%98_%EC%95%88%EC%85%80%EB%AC%B4%EC%8A%A4" title="캔터베리의 안셀무스">캔터베리의 안셀무스</a>에 이르는 전통적 견해는 신학적 이유도 있어서, 실재론에 기울어져 있고, 명확하게 유명론을 제창한 것은 로스켈리누스이다. 극단적인 형태로 실재론을 주장한 것은 윌리엄이었다. 여러가지 가설 중에서 어느 것을 선택하는가 하는 것은 하나의 철학을 완성하지 않고서는 불가능하다. <a href="/wiki/%EC%95%84%EB%A6%AC%EC%8A%A4%ED%86%A0%ED%85%94%EB%A0%88%EC%8A%A4" title="아리스토텔레스">아리스토텔레스</a>와 <a href="/wiki/%ED%94%8C%EB%9D%BC%ED%86%A4" title="플라톤">플라톤</a> 중에서 누구를 선택하는가, 그렇지 않으면 이들을 결합시키는가 하는 문제가 되면 <a class="new" href="/w/index.php?title=%EC%9E%90%EC%97%B0%ED%95%99&amp;action=edit&amp;redlink=1" title="자연학 (없는 문서)">자연학</a>·<a href="/wiki/%ED%98%95%EC%9D%B4%EC%83%81%ED%95%99" title="형이상학">형이상학</a>의 문제까지 얽힌다.</p>
<p><a class="mw-redirect" href="/wiki/%EC%95%84%EB%B2%A8%EB%9D%BC%EB%A5%B4%EB%91%90%EC%8A%A4" title="아벨라르두스">아벨라르두스</a>에 의해 보편의 문제는 더욱 촉진되었다. 그는 유명론적 색채를 가지면서 조정적 입장을 취해 <a href="/wiki/%ED%86%A0%EB%A7%88%EC%8A%A4_%EC%95%84%ED%80%B4%EB%82%98%EC%8A%A4" title="토마스 아퀴나스">토마스 아퀴나스</a>의 온건한 실재론에 이르는 길을 열었다. 그에 의하면 보편은 추상에 의해 획득된 보편개념을 표시하기 위한 선언인 것이다. 사물에 있어서 어떤 객관적으로 있는 것은 문제가 되지 않는다. 사물은 개별적인 것으로 보편은 아니기 때문이다.</p>
<p>토마스 아퀴나스는 새로이 알려진 아리스토텔레스와 <a href="/wiki/%EC%9D%B4%EB%B8%90_%EC%8B%9C%EB%82%98" title="이븐 시나">이븐 시나</a>의 저술의 영향을 받아 온건한 실재론을 완성하였다. 보편개념에 있어서 보편의 내용과 형식을 구별하고 사유 밖의 실재와 구체적 개체에 내재(內在)하는 본질을 보편개념의 내용에 대응시킨다. 개념상 보편의 형식은 어떤 주관적인 것으로서 사유의 산물이지만, 사유형식의 기초는 지성의 추상활동(抽象活動)에 있다고 한다.</p>
<p>14, 15세기의 스콜라 철학자인 <a class="new" href="/w/index.php?title=%EC%98%A4%EB%A6%AC%EC%98%AC%EC%9D%98_%ED%94%BC%EC%97%90%EB%A5%B4&amp;action=edit&amp;redlink=1" title="오리올의 피에르 (없는 문서)">오리올의 피에르</a>, <a href="/wiki/%EC%98%A4%EC%BB%B4%EC%9D%98_%EC%9C%8C%EB%A6%AC%EC%97%84" title="오컴의 윌리엄">오컴의 윌리엄</a>, <a class="new" href="/w/index.php?title=%EC%95%84%EC%9D%B4%EC%9D%B4%EC%9D%98_%ED%94%BC%EC%97%90%EB%A5%B4&amp;action=edit&amp;redlink=1" title="아이이의 피에르 (없는 문서)">아이이의 피에르</a> 등은 온건한 실재론을 고려하지 않고 다시금 유명론의 주관적 주장을 폈다. 따라서 <a href="/wiki/%EC%8B%A0%ED%95%99" title="신학">신학</a>과 <a href="/wiki/%EC%B2%A0%ED%95%99" title="철학">철학</a>을 분리시키고 스콜라적 종합을 붕괴시켰다.</p>
