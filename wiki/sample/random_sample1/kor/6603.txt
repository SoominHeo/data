<p><b>복양 전투</b>(濮陽戰鬪)는 <a href="/wiki/194%EB%85%84" title="194년">194년</a>~<a href="/wiki/195%EB%85%84" title="195년">195년</a>에 두 차례에 걸쳐 <a class="mw-disambig" href="/wiki/%EC%97%B0%EC%A3%BC" title="연주">연주</a>와 <a class="mw-disambig" href="/wiki/%EB%B3%B5%EC%96%91" title="복양">복양</a>을 두고 <a href="/wiki/%EC%A1%B0%EC%A1%B0" title="조조">조조</a>와 <a href="/wiki/%EC%97%AC%ED%8F%AC" title="여포">여포</a>가 싸운 전투로 <a href="/wiki/%EC%A1%B0%EC%A1%B0" title="조조">조조</a>가 이 전투에서 승리해 다시 <a class="mw-disambig" href="/wiki/%EC%97%B0%EC%A3%BC" title="연주">연주</a>를 되찾고 <a href="/wiki/%EC%97%AC%ED%8F%AC" title="여포">여포</a>는 <a href="/wiki/%EC%89%AC%EC%A0%80%EC%9A%B0_%EC%8B%9C" title="쉬저우 시">서주</a>의 <a href="/wiki/%EC%9C%A0%EB%B9%84" title="유비">유비</a>에게로 가 의탁한다.</p>
<h2><span class="mw-headline" id=".EB.B0.B0.EA.B2.BD">배경</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%B3%B5%EC%96%91_%EC%A0%84%ED%88%AC&amp;action=edit&amp;section=1" title="부분 편집: 배경">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>당시 <a href="/wiki/%EC%A1%B0%EC%A1%B0" title="조조">조조</a>는 서주 태수 <a class="mw-disambig" href="/wiki/%EB%8F%84%EA%B2%B8" title="도겸">도겸</a>을 정벌하러 <a href="/wiki/%EC%84%9C%EC%A3%BC%EC%84%B1_%EC%A0%84%ED%88%AC" title="서주성 전투">서주성 전투</a>를 치르고 있어 조조의 근거지인 <a class="mw-disambig" href="/wiki/%EC%97%B0%EC%A3%BC" title="연주">연주</a>는 비어 있었다.</p>
<p>이 틈을 노려 진류 태수 <a href="/wiki/%EC%9E%A5%EB%A7%89_(%ED%9B%84%ED%95%9C)" title="장막 (후한)">장막</a>과 동생 광릉 태수 <a href="/wiki/%EC%9E%A5%EC%B4%88" title="장초">장초</a>에게 의탁하고 있던 <a href="/wiki/%EC%97%AC%ED%8F%AC" title="여포">여포</a>는 동군종사 <a href="/wiki/%EC%A7%84%EA%B6%81" title="진궁">진궁</a>의 의견으로 <a href="/wiki/%EC%9E%A5%EB%A7%89_(%ED%9B%84%ED%95%9C)" title="장막 (후한)">장막</a>과 함께 연주를 공격해 연주성을 함락시켰고 조조의 모사 <a class="mw-redirect" href="/wiki/%EC%88%9C%EC%9A%B1_(%ED%9B%84%ED%95%9C)" title="순욱 (후한)">순욱</a>과 <a href="/wiki/%EC%A0%95%EC%9A%B1_(%EC%A1%B0%EC%9C%84)" title="정욱 (조위)">정욱</a>이 3만 군사로 겨우 <a class="mw-redirect" href="/wiki/%EA%B2%AC%EC%84%B1" title="견성">견성</a>, <a href="/wiki/%EB%8F%99%EC%95%84" title="동아">동아</a>, <a class="new" href="/w/index.php?title=%EB%B2%94%ED%98%95&amp;action=edit&amp;redlink=1" title="범형 (없는 문서)">범형</a> 세 고을만 지켜낸다.</p>
<h2><span class="mw-headline" id=".EC.A0.9C1.EC.B0.A8_.EC.A0.84.ED.88.AC">제1차 전투</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%B3%B5%EC%96%91_%EC%A0%84%ED%88%AC&amp;action=edit&amp;section=2" title="부분 편집: 제1차 전투">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>이 소식은 <a href="/wiki/%EC%A1%B0%EC%A1%B0" title="조조">조조</a>에게로 전해져 <a href="/wiki/%EC%84%9C%EC%A3%BC%EC%84%B1_%EC%A0%84%ED%88%AC" title="서주성 전투">서주성 전투</a>를 중단하고 <a class="mw-disambig" href="/wiki/%EC%97%B0%EC%A3%BC" title="연주">연주</a>로 가 <a href="/wiki/%EC%A1%B0%EC%9D%B8" title="조인">조인</a> 등과 합류한다. 연주성에 있던 <a href="/wiki/%EC%97%AC%ED%8F%AC" title="여포">여포</a>는 <a class="mw-disambig" href="/wiki/%EB%B3%B5%EC%96%91" title="복양">복양</a>성으로 가 군사를 주둔하기로 하고 조조는 <a class="mw-disambig" href="/wiki/%EB%B3%B5%EC%96%91" title="복양">복양</a> 근처에 도착해 영채를 세우고 맞섰으나 여포의 부하 <a href="/wiki/%EC%9E%A5%EB%A3%8C" title="장료">장료</a>와 <a href="/wiki/%EC%9E%A5%ED%8C%A8" title="장패">장패</a>, <a href="/wiki/%ED%95%99%EB%A7%B9" title="학맹">학맹</a>, <a href="/wiki/%EC%A1%B0%EC%84%B1_(%ED%9B%84%ED%95%9C)" title="조성 (후한)">조성</a>, <a href="/wiki/%EC%84%B1%EB%A0%B4" title="성렴">성렴</a>, <a href="/wiki/%EC%86%A1%ED%97%8C" title="송헌">송헌</a>, <a href="/wiki/%EC%9C%84%EC%86%8D" title="위속">위속</a>, <a href="/wiki/%ED%9B%84%EC%84%B1" title="후성">후성</a> 등이 나서면서 첫 싸움에서 패한다.</p>
<p>이에 그날 밤 조조는 여포의 영채를 기습 공격을 하지만 <a href="/wiki/%EC%A7%84%EA%B6%81" title="진궁">진궁</a>의 계책으로 적들이 다시 반격을 해 다시 물러나고 만다. 그리고 조조는 복양성 안에서 전(田)씨 성을 쓰는 부호가 자신들을 도와준다는 소식을 듣지만 이것도 조조를 성 안으로 끌어들려 고립시켜 무찌르려는 <a href="/wiki/%EC%A7%84%EA%B6%81" title="진궁">진궁</a>의 계책이었다.</p>
<p>갑작스런 공격을 받은 조조는 겨우 탈출하고 여포는 조조를 추격하지만 조조의 반격으로 크게 패해 복양성으로 다시 들어가 나오지 않았다. 조조도 군사를 거두어 견성으로 돌아갔다.</p>
<h2><span class="mw-headline" id=".EC.A0.9C2.EC.B0.A8_.EC.A0.84.ED.88.AC">제2차 전투</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%B3%B5%EC%96%91_%EC%A0%84%ED%88%AC&amp;action=edit&amp;section=3" title="부분 편집: 제2차 전투">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p><a href="/wiki/%EC%A1%B0%EC%A1%B0" title="조조">조조</a>는 연주를 되찾기 위해 힘을 기를 준비를 한다. 먼저 동족에 <a href="/wiki/%ED%99%A9%EA%B1%B4%EC%A0%81" title="황건적">황건적</a> 잔당들을 공격해 무찌르고 힘세고 뛰어난 장수 <a class="mw-redirect" href="/wiki/%ED%97%88%EC%A0%80_(%EC%A1%B0%EC%9C%84)" title="허저 (조위)">허저</a>를 얻는다. 그리고 <a class="mw-redirect" href="/wiki/%EA%B2%AC%EC%84%B1" title="견성">견성</a>에 있던 조조는 여포가 연주성을 비우고 <a href="/wiki/%EC%97%AC%ED%8F%AC" title="여포">여포</a>의 부하 <a href="/wiki/%EC%84%A4%EB%9E%80" title="설란">설란</a>과 <a href="/wiki/%EC%9D%B4%EB%B4%89_(%ED%9B%84%ED%95%9C)" title="이봉 (후한)">이봉</a>만 지키고 있자 연주성을 되찾기로 결정한다.</p>
<p>조조는 <a class="mw-redirect" href="/wiki/%ED%97%88%EC%A0%80_(%EC%A1%B0%EC%9C%84)" title="허저 (조위)">허저</a>를 시켜 <a href="/wiki/%EC%9D%B4%EB%B4%89_(%ED%9B%84%ED%95%9C)" title="이봉 (후한)">이봉</a>을 베고 <a href="/wiki/%EC%97%AC%EA%B1%B4_(%EC%A1%B0%EC%9C%84)" title="여건 (조위)">여건</a>을 시켜 활을 쏴 <a href="/wiki/%EC%84%A4%EB%9E%80" title="설란">설란</a>을 죽여 연주성을 쉽게 되찾는다. 여포는 복양성에서 나와 조조에 맞서고 조조는 부하 <a class="mw-redirect" href="/wiki/%ED%97%88%EC%A0%80_(%EC%A1%B0%EC%9C%84)" title="허저 (조위)">허저</a>, <a href="/wiki/%EC%A0%84%EC%9C%84_(%ED%9B%84%ED%95%9C)" title="전위 (후한)">전위</a>, <a href="/wiki/%ED%95%98%ED%9B%84%EB%8F%88" title="하후돈">하후돈</a>, <a href="/wiki/%ED%95%98%ED%9B%84%EC%97%B0" title="하후연">하후연</a>, <a href="/wiki/%EC%95%85%EC%A7%84" title="악진">악진</a>, <a href="/wiki/%EC%9D%B4%EC%A0%84_(%ED%9B%84%ED%95%9C)" title="이전 (후한)">이전</a> 등 6명의 장수들을 시켜 여포를 무찌르고 여포는 복양성으로 가지만 전씨가 배반하여 연주성의 성문을 닫자 진퇴양난에 빠진 여포는 장막과 장초, 진궁 등을 이끌고 정도성으로 후퇴한다.</p>
<p>이에 조조가 남은 군사들까지 모두 모아 정도성을 총공격하자 장초는 불에 뛰어들어 자살하고 장막도 겨우 탈출해 원술에게로 가려하지만 부하들에게 살해당했고 여포 역시 군사의 2/3를 잃었고 부하 <a href="/wiki/%EC%84%B1%EB%A0%B4" title="성렴">성렴</a>도 조조의 부하 <a href="/wiki/%EC%95%85%EC%A7%84" title="악진">악진</a>의 활에 맞아 전사한다.</p>
<p>여포는 <a href="/wiki/%EA%B8%B0%EC%A3%BC" title="기주">기주</a>의 <a href="/wiki/%EC%9B%90%EC%86%8C_(%ED%9B%84%ED%95%9C)" title="원소 (후한)">원소</a>에게로 가려하지만 <a href="/wiki/%EC%9B%90%EC%86%8C_(%ED%9B%84%ED%95%9C)" title="원소 (후한)">원소</a>까지 조조의 편을 들어 부하 <a href="/wiki/%EC%95%88%EB%9F%89" title="안량">안량</a>에게 5만 군사를 주어 여포를 공격한다.</p>
<p>결국 여포는 서주의 <a href="/wiki/%EC%9C%A0%EB%B9%84" title="유비">유비</a>에게로 가 의탁하기로 하고 조조는 연주를 되찾아 다시 세력을 넓힌다.</p>
