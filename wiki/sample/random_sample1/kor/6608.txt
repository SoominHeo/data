<h1 class="firstHeading" id="firstHeading" lang="ko">복합 프로그래머블 논리 소자</h1>

<p><b>복합 프로그래머블 논리 소자</b> (<span lang="en" xml:lang="en">Complex Programmable Logic Device</span>)는 <a class="mw-redirect" href="/wiki/%ED%98%84%EC%9E%A5_%ED%94%84%EB%A1%9C%EA%B7%B8%EB%9E%98%EB%A8%B8%EB%B8%94_%EA%B2%8C%EC%9D%B4%ED%8A%B8_%EC%96%B4%EB%A0%88%EC%9D%B4" title="현장 프로그래머블 게이트 어레이">현장 프로그래머블 게이트 어레이</a> (FPGA)와 <a href="/wiki/%ED%94%84%EB%A1%9C%EA%B7%B8%EB%9E%98%EB%A8%B8%EB%B8%94_%EC%96%B4%EB%A0%88%EC%9D%B4_%EB%85%BC%EB%A6%AC" title="프로그래머블 어레이 논리">프로그래머블 어레이 논리</a> (PAL)의 복합성과 구조적 특성을 지니고 있는 <a class="mw-redirect" href="/wiki/%ED%94%84%EB%A1%9C%EA%B7%B8%EB%9E%98%EB%A8%B8%EB%B8%94_%EB%85%BC%EB%A6%AC_%EC%86%8C%EC%9E%90" title="프로그래머블 논리 소자">프로그래머블 논리 소자</a> (PLD)이다. 제조된 복합 프로그래머블 논리 소자의 블록은 <a class="new" href="/w/index.php?title=%EB%85%BC%EB%A6%AC%EC%A0%81_%ED%91%9C%EC%A4%80%ED%98%95&amp;action=edit&amp;redlink=1" title="논리적 표준형 (없는 문서)">논리적 표준형</a> 식과 더 특별한 논리 연산을 실행하는 논리가 포함된 <b>매크로셀</b> (<span lang="en" xml:lang="en">macrocell</span>)이다.</p>
<p><a href="/wiki/%ED%94%84%EB%A1%9C%EA%B7%B8%EB%9E%98%EB%A8%B8%EB%B8%94_%EC%96%B4%EB%A0%88%EC%9D%B4_%EB%85%BC%EB%A6%AC" title="프로그래머블 어레이 논리">프로그래머블 어레이 논리</a>과 동일한 특징:</p>
<ul>
<li>비휘발 설정 메모리. 대다수 현장 프로그래머블 게이트 어레이와 다르게 외부 설정 <a class="mw-disambig" href="/wiki/ROM" title="ROM">ROM</a>이 불필요하고, 복합 프로그래머블 논리 소자는 시스템 시작하자마자 동작할 수 있다.</li>
<li>특히 큰 소자는 라우팅 대부분이 (내부 상태저장이나 깊은 레이어 논리의 작은 이점으로) 외부핀과 연결된 입력과 출력신호를 가지는 논리블록을 수반한다.</li>
</ul>
<p><a class="mw-redirect" href="/wiki/%ED%98%84%EC%9E%A5_%ED%94%84%EB%A1%9C%EA%B7%B8%EB%9E%98%EB%A8%B8%EB%B8%94_%EA%B2%8C%EC%9D%B4%ED%8A%B8_%EC%96%B4%EB%A0%88%EC%9D%B4" title="현장 프로그래머블 게이트 어레이">현장 프로그래머블 게이트 어레이</a>와 동일한 특징:</p>
<ul>
<li>큰수의 게이트가 가능함. 복합 프로그래머블 논리 소자는 일반적으로 수천에서 수십만의 <a class="mw-redirect" href="/wiki/%EB%85%BC%EB%A6%AC_%EA%B2%8C%EC%9D%B4%ED%8A%B8" title="논리 게이트">논리 게이트</a>와 동등하며, 적당히 복잡한 데이터 처리 장치의 수행이 가능하다. 프로그래머블 어레이 논리는 일반적으로 수백 게이트와 동등하며, 현장 프로그래머블 게이트 어레이는 일반적으로 수백만에서 수백만의 범위이다.</li>
<li>어떤 제품은 논리가 <a class="new" href="/w/index.php?title=%EB%85%BC%EB%A6%AC%ED%95%A9_%ED%91%9C%EC%A4%80%EC%8B%9D&amp;action=edit&amp;redlink=1" title="논리합 표준식 (없는 문서)">결과의 합</a>식보다 유연하도록 매크로셀 사이에 복합한 피드백 라인과 (내부 연산회로 같은) 일반적으로 사용되는 다양한 기능을 실행하는 특별한 논리를 포함한다.</li>
</ul>
<p>큰 복합 프로그래머블 논리 소자와 작은 현장 프로그래머블 게이트 어레이의 가장큰 다른점은 복합 프로그래머블 논리 소자에는 칩내부에 비활성 메모리가 존재한다. 구조적 접근에서 다른점은 중간체 영역으로부터 분명히 더 멀다.</p>
<p>비휘발의 특징은 복합 프로그래머블 논리 소자가 이런 능력을 가지지 않는 다른 소자를 제어 이전에 처리하는 <a class="mw-redirect" href="/wiki/%EB%B6%80%ED%8A%B8_%EB%A1%9C%EB%8D%94" title="부트 로더">부트 로더</a> 기능을 수행하는 현대 디지털 설계에 종종 사용되는 것을 의미한다. 좋은 예로 복합 프로그래머블 논리 소자가 비활성 메모리에서 현장 프로그래머블 게이트 어레이에 환경설정 데이터를 올리는 데 사용된다.</p>
<p>(처음으로 <a class="new" href="/w/index.php?title=%EC%8B%9C%EA%B7%B8%EB%84%A4%ED%8B%B1%EC%8A%A4&amp;action=edit&amp;redlink=1" title="시그네틱스 (없는 문서)">시그네틱스</a>에 의하여 판매된) <a class="new" href="/w/index.php?title=%ED%94%84%EB%A1%9C%EA%B7%B8%EB%9E%98%EB%A8%B8%EB%B8%94_%EB%85%BC%EB%A6%AC_%EC%96%B4%EB%A0%88%EC%9D%B4&amp;action=edit&amp;redlink=1" title="프로그래머블 논리 어레이 (없는 문서)">프로그래머블 논리 어레이</a> (PLA)와 <a href="/wiki/%ED%94%84%EB%A1%9C%EA%B7%B8%EB%9E%98%EB%A8%B8%EB%B8%94_%EC%96%B4%EB%A0%88%EC%9D%B4_%EB%85%BC%EB%A6%AC" title="프로그래머블 어레이 논리">프로그래머블 어레이 논리</a>보다 우월하면서 심지어 더 작은 소자로 진보적인 단계가 복합 프로그래머블 논리 소자이다.</p>
<p>두 소자 사이에서 향상된 현장 프로그래머블 게이트 어레이는 두가지 종류가 근본적으로 구별할 수 없을때까지 의미해지는 경향이 계속되었다.</p>
