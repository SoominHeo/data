<p>《<b>본내퍼 드래곤의 전설</b>》(<span style="font-size: smaller;"><a href="/wiki/%EC%98%81%EC%96%B4" title="영어">영어</a>:</span> <span lang="en" xml:lang="en">Legend of the Boneknapper Dragon</span>)은 <a href="/wiki/%EB%AF%B8%EA%B5%AD" title="미국">미국</a>의 단편 <a href="/wiki/%EC%95%A0%EB%8B%88%EB%A9%94%EC%9D%B4%EC%85%98_%EC%98%81%ED%99%94" title="애니메이션 영화">애니메이션 영화</a>이다.</p>
<p>2010년 제작된 애니메이션 영화 <a href="/wiki/%EB%93%9C%EB%9E%98%EA%B3%A4_%EA%B8%B8%EB%93%A4%EC%9D%B4%EA%B8%B0_(%EC%98%81%ED%99%94)" title="드래곤 길들이기 (영화)">드래곤 길들이기</a>의 외전으로 <a href="/wiki/%EB%93%9C%EB%A6%BC%EC%9B%8D%EC%8A%A4_%EC%95%A0%EB%8B%88%EB%A9%94%EC%9D%B4%EC%85%98" title="드림웍스 애니메이션">드림웍스 애니메이션</a>이 제작하였으며, <a href="/wiki/2010%EB%85%84" title="2010년">2010년</a> <a href="/wiki/10%EC%9B%94_14%EC%9D%BC" title="10월 14일">10월 14일</a> <a href="/wiki/%EB%AF%B8%EA%B5%AD" title="미국">미국</a>에서 <a href="/wiki/%ED%85%94%EB%A0%88%EB%B9%84%EC%A0%84_%EC%98%81%ED%99%94" title="텔레비전 영화">텔레비전 영화</a>로 공개되었다. <a href="/wiki/%EB%93%9C%EB%9E%98%EA%B3%A4_%EA%B8%B8%EB%93%A4%EC%9D%B4%EA%B8%B0_(%EC%98%81%ED%99%94)" title="드래곤 길들이기 (영화)">드래곤 길들이기</a> <a href="/wiki/DVD" title="DVD">DVD</a> 출시시 부가 영상으로 수록되었다.</p>
