<p><b>본시라속</b> 또는 <b>살라노이아속</b>(<i>Salanoia</i>)은 <a href="/wiki/%EB%A7%88%EB%8B%A4%EA%B0%80%EC%8A%A4%EC%B9%B4%EB%A5%B4%EC%8B%9D%EC%9C%A1%EA%B3%BC" title="마다가스카르식육과">마다가스카르식육과</a>의 <a href="/wiki/%EB%A7%88%EB%8B%A4%EA%B0%80%EC%8A%A4%EC%B9%B4%EB%A5%B4%EB%AA%BD%EA%B5%AC%EC%8A%A4%EC%95%84%EA%B3%BC" title="마다가스카르몽구스아과">마다가스카르몽구스아과</a>에 속하는 하위 속이다. 2종의 <a href="/wiki/%EB%A7%88%EB%8B%A4%EA%B0%80%EC%8A%A4%EC%B9%B4%EB%A5%B4" title="마다가스카르">마다가스카르</a> 섬 육식동물 토착종으로 이루어져 있다.<sup class="reference" id="cite_ref-durbin_1-0"><a href="#cite_note-durbin-1">[1]</a></sup> 겉모습이 <a href="/wiki/%EB%AA%BD%EA%B5%AC%EC%8A%A4" title="몽구스">몽구스</a>를 닮았기때문에 하위 종 <a class="mw-redirect" href="/wiki/%EA%B0%88%EC%83%89%EA%BC%AC%EB%A6%AC%EB%AA%BD%EA%B5%AC%EC%8A%A4" title="갈색꼬리몽구스">갈색꼬리몽구스</a>처럼 이름에 몽구스라는 단어가 들어 가 있지만 최근에는 "갈색꼬리본시라"라고도 부른다.<sup class="reference" id="cite_ref-isbn1-84162-245-1_2-0"><a href="#cite_note-isbn1-84162-245-1-2">[2]</a></sup></p>
<p>"살라노이아"(<i>Salanoia</i>)라는 학명은 <a href="/wiki/%EA%B0%88%EC%83%89%EA%BC%AC%EB%A6%AC%EB%B3%B8%EC%8B%9C%EB%9D%BC" title="갈색꼬리본시라">갈색꼬리본시라</a>(<i>Salanoia concolor</i>)의 다른 이름인 <a class="mw-redirect" href="/wiki/%EC%82%B4%EB%9D%BC%EB%85%B8" title="살라노">살라노</a>(<i>Salano</i>)에서 유래했다.<sup class="reference" id="cite_ref-isbn1-84162-245-1_2-1"><a href="#cite_note-isbn1-84162-245-1-2">[2]</a></sup></p>
<p>2010년에 두번째 종 <a href="/wiki/%EB%93%80%EB%A0%90%EB%B3%B8%EC%8B%9C%EB%9D%BC" title="듀렐본시라">듀렐본시라</a>가 발견된어 명명되었다.<sup class="reference" id="cite_ref-3"><a href="#cite_note-3">[3]</a></sup><sup class="reference" id="cite_ref-4"><a href="#cite_note-4">[4]</a></sup> "본시라"(<i>Vontsira</i>)라는 이름은 <a href="/wiki/%EB%A7%88%EB%8B%A4%EA%B0%80%EC%8A%A4%EC%B9%B4%EB%A5%B4" title="마다가스카르">마다가스카르</a> 섬 현지에서 <a href="/wiki/%EB%AA%BD%EA%B5%AC%EC%8A%A4" title="몽구스">몽구스</a>를 닮은 살라노이아속과 <a href="/wiki/%EC%A4%84%EB%AC%B4%EB%8A%AC%EA%BC%AC%EB%A6%AC%EB%AA%BD%EA%B5%AC%EC%8A%A4" title="줄무늬꼬리몽구스">줄무늬꼬리몽구스속</a> 그리고 <a href="/wiki/%EB%9D%A0%EB%AA%BD%EA%B5%AC%EC%8A%A4%EC%86%8D" title="띠몽구스속">띠몽구스속</a>의 육식동물을 가리키는 명칭이다.<sup class="reference" id="cite_ref-isbn1-84162-245-1_2-2"><a href="#cite_note-isbn1-84162-245-1-2">[2]</a></sup></p>
<h2><span class="mw-headline" id=".ED.95.98.EC.9C.84_.EC.A2.85">하위 종</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%B3%B8%EC%8B%9C%EB%9D%BC%EC%86%8D&amp;action=edit&amp;section=1" title="부분 편집: 하위 종">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>2종을 포함하고 있다.<sup class="reference" id="cite_ref-durbin_1-1"><a href="#cite_note-durbin-1">[1]</a></sup></p>
<ul>
<li><a href="/wiki/%EB%93%80%EB%A0%90%EB%B3%B8%EC%8B%9C%EB%9D%BC" title="듀렐본시라">듀렐본시라</a> 또는 <a class="mw-redirect" href="/wiki/%EB%93%80%EB%A0%90%EB%AA%BD%EA%B5%AC%EC%8A%A4" title="듀렐몽구스">듀렐몽구스</a> (<i>Salanoia durrelli</i>)</li>
<li><a href="/wiki/%EA%B0%88%EC%83%89%EA%BC%AC%EB%A6%AC%EB%B3%B8%EC%8B%9C%EB%9D%BC" title="갈색꼬리본시라">갈색꼬리본시라</a> 또는 <a class="mw-redirect" href="/wiki/%EA%B0%88%EC%83%89%EA%BC%AC%EB%A6%AC%EB%AA%BD%EA%B5%AC%EC%8A%A4" title="갈색꼬리몽구스">갈색꼬리몽구스</a>, <a class="mw-redirect" href="/wiki/%EC%82%B4%EB%9D%BC%EB%85%B8" title="살라노">살라노</a> (<i>Salanoia concolor</i>)</li>
</ul>
