<p><b>볼프강 쾰러</b>(Wolfgang Köhler, <a href="/wiki/1887%EB%85%84" title="1887년">1887년</a> <a href="/wiki/1%EC%9B%94_22%EC%9D%BC" title="1월 22일">1월 22일</a> ~ <a href="/wiki/1967%EB%85%84" title="1967년">1967년</a> <a href="/wiki/6%EC%9B%94_11%EC%9D%BC" title="6월 11일">6월 11일</a>)는 <a href="/wiki/%EB%8F%85%EC%9D%BC" title="독일">독일</a>의 심리학자 현상학자로 <a href="/wiki/%EB%A7%89%EC%8A%A4_%EB%B2%A0%EB%A5%B4%ED%8A%B8%ED%95%98%EC%9D%B4%EB%A8%B8" title="막스 베르트하이머">막스 베르트하이머</a>, <a href="/wiki/%EC%BF%A0%EB%A5%B4%ED%8A%B8_%EC%BD%94%ED%94%84%EC%B9%B4" title="쿠르트 코프카">쿠르트 코프카</a>와 함께 <a href="/wiki/%EA%B2%8C%EC%8A%88%ED%83%88%ED%8A%B8_%EC%8B%AC%EB%A6%AC%ED%95%99" title="게슈탈트 심리학">게슈탈트 심리학</a>을 처음 형성한 것으로 유명하다. <a class="mw-redirect" href="/wiki/%EB%82%98%EC%B9%98" title="나치">나치</a> 정권 하에서 <a class="mw-redirect" href="/wiki/%EC%9C%A0%ED%83%9C%EC%9D%B8" title="유태인">유태인</a> 교수 해고에 항의하며 <a href="/wiki/1935%EB%85%84" title="1935년">1935년</a> <a href="/wiki/%EB%AF%B8%EA%B5%AD" title="미국">미국</a>으로 이민을 갔고 <a href="/wiki/%EC%8A%A4%EC%99%80%EC%8A%A4%EB%AA%A8%EC%96%B4_%EB%8C%80%ED%95%99%EA%B5%90" title="스와스모어 대학교">스와스모어 대학교</a>에서 교수 업무를 하며 연구를 했다.</p>
