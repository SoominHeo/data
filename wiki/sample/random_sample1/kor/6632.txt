<p><b>볼호프</b>(<span style="font-size: smaller;"><a href="/wiki/%EB%9F%AC%EC%8B%9C%EC%95%84%EC%96%B4" title="러시아어">러시아어</a>:</span> <span lang="ru" xml:lang="ru">Болхов</span>)는 <a href="/wiki/%EB%9F%AC%EC%8B%9C%EC%95%84" title="러시아">러시아</a> <a href="/wiki/%EC%98%A4%EB%A3%94_%EC%A3%BC" title="오룔 주">오룔 주</a> 북부에 있는 <a href="/wiki/%EB%8F%84%EC%8B%9C" title="도시">도시</a>로, 누그르 강과 접한다. 인구는 12,148명(2002년 기준)이며 <a href="/wiki/%EC%98%A4%EB%A3%94" title="오룔">오룔</a>(오룔 주의 주도)에서 동쪽으로 56km 정도 떨어진 곳에 위치한다.</p>
<p><a href="/wiki/13%EC%84%B8%EA%B8%B0" title="13세기">13세기</a> <a href="/wiki/%EB%AA%BD%EA%B3%A8_%EC%A0%9C%EA%B5%AD" title="몽골 제국">몽골 제국</a>의 러시아 침공 이후부터 작은 공국의 수도가 되었고 <a href="/wiki/16%EC%84%B8%EA%B8%B0" title="16세기">16세기</a> <a class="mw-redirect" href="/wiki/%EB%AA%A8%EC%8A%A4%ED%81%AC%EB%B0%94_%EA%B3%B5%EA%B5%AD" title="모스크바 공국">모스크바 공국</a>의 요새가 건설되면서 남쪽에 있는 <a href="/wiki/%ED%83%80%ED%83%80%EB%A5%B4%EC%A1%B1" title="타타르족">타타르족</a>으로부터 모스크바를 지키는 역할을 담당했다. <a href="/wiki/1608%EB%85%84" title="1608년">1608년</a> <a class="new" href="/w/index.php?title=%EB%B0%94%EC%8B%A4%EB%A6%AC_4%EC%84%B8&amp;action=edit&amp;redlink=1" title="바실리 4세 (없는 문서)">바실리 4세</a>가 이끄는 군대가 <a href="/wiki/%EA%B0%80%EC%A7%9C_%EB%93%9C%EB%AF%B8%ED%8A%B8%EB%A6%AC_2%EC%84%B8" title="가짜 드미트리 2세">가짜 드미트리 2세</a>를 물리쳤다.</p>
<p><a href="/wiki/18%EC%84%B8%EA%B8%B0" title="18세기">18세기</a>에 건설된 교회 4개가 남아 있으며 5개의 돔을 가진 삼위일체 대성당(<a href="/wiki/1688%EB%85%84" title="1688년">1688년</a>부터 <a href="/wiki/1706%EB%85%84" title="1706년">1706년</a>까지 건설됨)과 인근에 있는 가장 큰 성당인 구세주 현영 대성당(<a href="/wiki/1841%EB%85%84" title="1841년">1841년</a>부터 <a href="/wiki/1851%EB%85%84" title="1851년">1851년</a>까지 건설됨)이 있다.</p>
<p><span style="font-size: small;white-space:nowrap;"><span id="coordinates"><a href="/wiki/%EC%A7%80%EB%A6%AC_%EC%A2%8C%ED%91%9C%EA%B3%84" title="지리 좌표계">좌표</a>: <span class="plainlinks nourlexpansion"><a class="external text" href="//tools.wmflabs.org/geohack/geohack.php?language=ko&amp;pagename=%EB%B3%BC%ED%98%B8%ED%94%84_(%EC%98%A4%EB%A3%94_%EC%A3%BC)&amp;params=52_58_N_37_03_E_"><span class="geo-default"><span class="geo-dms" title="이 위치에 대한 지도와 항공사진 등의 기타 정보를 보여줍니다"><span class="latitude">북위 52° 58′</span> <span class="longitude">동경 37° 03′</span></span></span> <span class="geo-multi-punct">﻿ / ﻿</span><span class="geo-nondefault"><span class="geo-dec" title="이 위치에 대한 지도와 항공사진 등의 기타 정보를 보여줍니다">북위 52.967° 동경 37.050°</span><span style="display:none">﻿ / <span class="geo">52.967; 37.050</span></span></span></a></span></span></span></p>
