<p><b>부대찌개</b>(部隊 - )는 문자 그대로 <a href="/wiki/%EA%B5%B0%EB%8C%80" title="군대">군대</a>의 <a href="/wiki/%EC%B0%8C%EA%B0%9C" title="찌개">찌개</a>란 뜻으로, 서구의 <a href="/wiki/%EC%8A%A4%ED%8A%9C" title="스튜">스튜</a>처럼 진한 한국의 국물 요리이다. 가장 일반적으로 쓰이는 재료에는 <a href="/wiki/%EB%8F%BC%EC%A7%80%EA%B3%A0%EA%B8%B0" title="돼지고기">돼지고기</a>, <a href="/wiki/%ED%96%84" title="햄">햄</a>, <a class="mw-redirect" href="/wiki/%EC%86%8C%EC%84%B8%EC%A7%80" title="소세지">소세지</a>, 부대찌개 <a href="/wiki/%EC%BD%A9" title="콩">콩</a>, <a href="/wiki/%EA%B9%80%EC%B9%98" title="김치">김치</a>, <a href="/wiki/%EB%9D%BC%EB%A9%B4" title="라면">라면</a>, <a href="/wiki/%EB%96%A1" title="떡">떡</a>, <a href="/wiki/%EB%91%90%EB%B6%80" title="두부">두부</a> 등이 있다.</p>

<h2><span class="mw-headline" id=".EC.9C.A0.EB.9E.98">유래</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%B6%80%EB%8C%80%EC%B0%8C%EA%B0%9C&amp;action=edit&amp;section=1" title="부분 편집: 유래">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p><a href="/wiki/%ED%95%9C%EA%B5%AD_%EC%A0%84%EC%9F%81" title="한국 전쟁">한국 전쟁</a>(625사변)이 일어난 후 한국은 먹거리가 부족하게 되었다. 이에 미군과 관련된 일을 하던 한국사람들이 <a href="/wiki/%EA%B2%BD%EA%B8%B0%EB%8F%84" title="경기도">경기도</a> <a href="/wiki/%EB%8F%99%EB%91%90%EC%B2%9C%EC%8B%9C" title="동두천시">동두천시</a>와 <a href="/wiki/%EC%96%91%EC%A3%BC%EC%8B%9C" title="양주시">양주시</a>, <a href="/wiki/%EC%9D%98%EC%A0%95%EB%B6%80%EC%8B%9C" title="의정부시">의정부시</a>, <a href="/wiki/%EA%B2%BD%EA%B8%B0%EB%8F%84" title="경기도">경기도</a> <a href="/wiki/%EC%86%A1%ED%83%84%EC%8B%9C" title="송탄시">송탄시</a>에 주둔하던 미군 부대의 <a href="/wiki/%ED%95%AB%EB%8F%84%EA%B7%B8" title="핫도그">핫도그</a>나 깡통에 든 햄(예를 들어 스팸)을 이용하여 <a href="/wiki/%EA%B3%A0%EC%B6%94%EC%9E%A5" title="고추장">고추장</a>과 함께 <a href="/wiki/%EC%B0%8C%EA%B0%9C" title="찌개">찌개</a>를 만들었다. 초기에는 미군이 먹다 남거나 미군의 보급품을 몰래 빼낸 음식을 재료로 사용하였기에 부대찌개라는 명칭으로 불렸다. 이 부대찌개는 가난을 벗어난 한국에서 여전히 인기가 있고 현대의 즉석 음식인 <a href="/wiki/%EB%9D%BC%EB%A9%B4" title="라면">라면</a>을 첨가하기도 한다. 휴전선 인접 지역인 <a href="/wiki/%EC%9D%98%EC%A0%95%EB%B6%80%EC%8B%9C" title="의정부시">의정부</a>는 미군 육군 부대 <a href="/wiki/%EA%B8%B0%EC%A7%80" title="기지">기지</a>가 많이 들어서 있는 곳이며 부대찌개로 유명하다. <a href="/wiki/%EC%86%A1%ED%83%84%EC%8B%9C" title="송탄시">송탄</a>도 미군 공군 부대 <a href="/wiki/%EA%B8%B0%EC%A7%80" title="기지">기지</a>가 많이 들어서 있는 곳이며 부대찌개로 유명하다.</p>

<h2><span class="mw-headline" id=".ED.98.95.EC.8B.9D">형식</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%B6%80%EB%8C%80%EC%B0%8C%EA%B0%9C&amp;action=edit&amp;section=2" title="부분 편집: 형식">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>크게 의정부식과 송탄식으로 나뉜다.<sup class="reference" id="cite_ref-1"><a href="#cite_note-1">[1]</a></sup></p>
<h3><span class="mw-headline" id=".EC.9D.98.EC.A0.95.EB.B6.80.EC.8B.9D">의정부식</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%B6%80%EB%8C%80%EC%B0%8C%EA%B0%9C&amp;action=edit&amp;section=3" title="부분 편집: 의정부식">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>맑은 육수를 사용하고 소시지와 햄을 적당히 넣어 김치 맛과 잘 어우러져 개운한 맛을 낸다.</p>
<h3><span class="mw-headline" id=".EC.86.A1.ED.83.84.EC.8B.9D">송탄식</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%B6%80%EB%8C%80%EC%B0%8C%EA%B0%9C&amp;action=edit&amp;section=4" title="부분 편집: 송탄식">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>의정부식에 비해 소시지와 햄을 훨씬 많이 넣고 치즈와 <a href="/wiki/%EA%B0%95%EB%82%AD%EC%BD%A9" title="강낭콩">강낭콩</a> 통조림을 첨가하여 맛이 진하다.</p>
<h2><span class="mw-headline" id=".EC.A1.B4.EC.8A.A8.ED.83.95">존슨탕</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%B6%80%EB%8C%80%EC%B0%8C%EA%B0%9C&amp;action=edit&amp;section=5" title="부분 편집: 존슨탕">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p><a href="/wiki/%EC%84%9C%EC%9A%B8%ED%8A%B9%EB%B3%84%EC%8B%9C" title="서울특별시">서울</a> <a href="/wiki/%EC%9A%A9%EC%82%B0%EA%B5%AC" title="용산구">용산구</a>에 주둔한 미군 부대를 중심으로 발달한 부대찌개의 한 형태를 <b>존슨탕</b>이라고 부른다.<sup class="reference" id="cite_ref-2"><a href="#cite_note-2">[2]</a></sup></p>
<p>‘존슨탕’이라는 명칭은 1966년 <a href="/wiki/%EB%A6%B0%EB%93%A0_B._%EC%A1%B4%EC%8A%A8" title="린든 B. 존슨">린든 B. 존슨</a> 미국 대통령의 방한에서 유래되었다고 한다. 존슨탕은 <a class="mw-redirect" href="/wiki/%EC%84%9C%EC%9A%B8" title="서울">서울</a>에서만 볼 수 있고, <a href="/wiki/%EC%82%AC%EA%B3%A8" title="사골">사골</a>로 국물을 내고 라면과 김치가 들어가지 않는 등 부대찌개와 구별된다고도 한다.<sup class="reference" id="cite_ref-tbs_3-0"><a href="#cite_note-tbs-3">[3]</a></sup></p>
