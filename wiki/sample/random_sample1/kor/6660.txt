<p><b>부르고뉴 전쟁</b>은 <a href="/wiki/%EB%B6%80%EB%A5%B4%EA%B3%A0%EB%89%B4_%EA%B3%B5%EA%B5%AD" title="부르고뉴 공국">부르고뉴 공작</a>(Dukes of Burgundy)과 <a class="mw-redirect" href="/wiki/%EB%B0%9C%EB%A3%A8%EC%95%84_%EC%99%95%EA%B0%80" title="발루아 왕가">프랑스 왕</a> 사이에서 벌어진 일련의 전쟁으로 이후 <a class="mw-redirect" href="/wiki/%EC%8A%A4%EC%9C%84%EC%8A%A4_%EC%97%B0%EB%B0%A9" title="스위스 연방">스위스 연방</a>도 참여하여 이 전쟁에서 결정적인 역할을 해내었다.</p>
<p><a href="/wiki/1474%EB%85%84" title="1474년">1474년</a> 개전해 몇 년의 시간이 흐른 뒤 <a href="/wiki/%EB%B6%80%EB%A5%B4%EA%B3%A0%EB%89%B4_%EA%B3%B5%EC%9E%91" title="부르고뉴 공작">부르고뉴 공작</a> <a href="/wiki/%EC%9A%A9%EB%8B%B4%EA%B3%B5_%EC%83%A4%EB%A5%BC" title="용담공 샤를">용담공 샤를</a>(Charles the Bold)은 1477년 <a href="/wiki/%EB%82%AD%EC%8B%9C_%EC%A0%84%ED%88%AC" title="낭시 전투">낭시 전투</a>(Battle of Nancy)에서 크게 패하고 전장에서 전사했다. <a href="/wiki/%EB%B6%80%EB%A5%B4%EA%B3%A0%EB%89%B4_%EA%B3%B5%EA%B5%AD" title="부르고뉴 공국">부르고뉴 공국</a>과 기타 부르고뉴 영지는 <a href="/wiki/%ED%94%84%EB%9E%91%EC%8A%A4" title="프랑스">프랑스</a>에 병합되었으나, <a class="new" href="/w/index.php?title=%EB%B6%80%EB%A5%B4%EA%B3%A0%EB%89%B4%EB%A0%B9_%EB%84%A4%EB%8D%9C%EB%9E%80%EB%93%9C&amp;action=edit&amp;redlink=1" title="부르고뉴령 네덜란드 (없는 문서)">부르고뉴령 네덜란드</a>(Burgundian Netherlands)와 <a href="/wiki/%ED%94%84%EB%9E%91%EC%8A%88%EC%BD%A9%ED%85%8C" title="프랑슈콩테">프랑슈콩테</a>( Franche Comté)는 샤를의 딸 <a class="mw-redirect" href="/wiki/%EB%B6%80%EB%A5%B4%EA%B3%A0%EB%89%B4%EC%9D%98_%EB%A7%88%EB%A6%AC" title="부르고뉴의 마리">부르고뉴의 마리</a>(Mary of Burgundy)가 상속받고, 그녀가 <a class="mw-redirect" href="/wiki/%EC%8B%A0%EC%84%B1%EB%A1%9C%EB%A7%88%EC%A0%9C%EA%B5%AD%EC%9D%98_%EB%A7%89%EC%8B%9C%EB%B0%80%EB%A6%AC%EC%95%88_1%EC%84%B8" title="신성로마제국의 막시밀리안 1세">신성로마제국의 막시밀리안 1세</a>(Maximilian I, Holy Roman Emperor)와 결혼한 뒤 사망했기 때문에 최종적으로 <a href="/wiki/%ED%95%A9%EC%8A%A4%EB%B6%80%EB%A5%B4%ED%81%AC_%EA%B0%80" title="합스부르크 가">합스부르크 가</a>(House of Habsburg)의 영지가 되었다.</p>

<p><a class="mw-redirect" href="/wiki/%EB%B0%9C%EB%A3%A8%EC%95%84_%EC%99%95%EA%B0%80#.EB.B6.80.EB.A5.B4.EA.B3.A0.EB.89.B4.EC.9D.98_.EA.B3.B5.EC.9E.91" title="발루아 왕가">부르고뉴 공작</a>들은 <a class="mw-redirect" href="/wiki/%EC%8B%A0%EC%84%B1%EB%A1%9C%EB%A7%88%EC%A0%9C%EA%B5%AD" title="신성로마제국">신성로마제국</a>과 <a href="/wiki/%ED%94%84%EB%9E%91%EC%8A%A4_%EC%99%95%EA%B5%AD" title="프랑스 왕국">프랑스 왕국</a>이 지배를 확립할 때까지 약 100년 이상에 걸쳐 영화를 지속했다. 공국은 본거지인 <a href="/wiki/%ED%94%84%EB%9E%91%EC%8A%88%EC%BD%A9%ED%85%8C" title="프랑슈콩테">프랑슈콩테</a>와 <a href="/wiki/%EB%B6%80%EB%A5%B4%EA%B3%A0%EB%89%B4_%EA%B3%B5%EA%B5%AD" title="부르고뉴 공국">부르고뉴 공국</a>의 기타 <a href="/wiki/%ED%94%8C%EB%9E%91%EB%93%9C%EB%A5%B4" title="플랑드르">플랑드르</a>(Flanders) 및 <a class="mw-disambig" href="/wiki/%EB%B8%8C%EB%9D%BC%EB%B0%98%ED%8A%B8" title="브라반트">브라반트</a>(Brabant), <a href="/wiki/%EB%A3%A9%EC%85%88%EB%B6%80%EB%A5%B4%ED%81%AC" title="룩셈부르크">룩셈부르크</a>(Luxembourg)란 경제력이 풍부한 지역을 소유하고 있었다.</p>
<p>일반적으로 부르고뉴 공작들은 확장정책을 채택해 특히 <a href="/wiki/%EC%95%8C%EC%9E%90%EC%8A%A4" title="알자스">알자스</a>(Alsace)와 <a href="/wiki/%EB%A1%9C%EB%A0%8C" title="로렌">로렌</a>(Lorraine)의 획득에 적극적이어서 지리적으로 남북으로 떨어진 영지를 하나로 합치는 것이 목표였는데 <a class="mw-redirect" href="/wiki/%EB%B0%B1%EB%85%84%EC%A0%84%EC%9F%81" title="백년전쟁">백년전쟁</a>에서 부르고뉴 공작은 <a href="/wiki/%EC%9E%89%EA%B8%80%EB%9E%9C%EB%93%9C" title="잉글랜드">잉글랜드</a>측에 참가해 프랑스 왕과는 이미 대립하였다.</p>
<p>샤를의 라인 강으로 진출은 합스부르크 가문 특히 신성로마제국 <a href="/wiki/%ED%94%84%EB%A6%AC%EB%93%9C%EB%A6%AC%ED%9E%88_3%EC%84%B8_(%EC%8B%A0%EC%84%B1_%EB%A1%9C%EB%A7%88_%EC%A0%9C%EA%B5%AD)" title="프리드리히 3세 (신성 로마 제국)">프리드리히 3세</a>와의 다툼의 원인이 되었다.</p>
<h2><span class="mw-headline" id=".EB.8C.80.EB.A6.BD">대립</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%B6%80%EB%A5%B4%EA%B3%A0%EB%89%B4_%EC%A0%84%EC%9F%81&amp;action=edit&amp;section=2" title="부분 편집: 대립">편집</a><span class="mw-editsection-bracket">]</span></span></h2>

<p><a href="/wiki/1469%EB%85%84" title="1469년">1469년</a> 처음엔 <a class="new" href="/w/index.php?title=%EC%98%A4%EC%8A%A4%ED%8A%B8%EB%A6%AC%EC%95%84_%EB%8C%80%EA%B3%B5_%EC%A7%80%EA%B7%B8%EB%AC%B8%ED%8A%B8&amp;action=edit&amp;redlink=1" title="오스트리아 대공 지그문트 (없는 문서)">합스부르크의 지그문트</a>(Sigismund of Habsburg) 공작은 <a href="/wiki/%EB%B6%80%EB%A5%B4%EA%B3%A0%EB%89%B4_%EA%B3%B5%EC%9E%91" title="부르고뉴 공작">부르고뉴 공작</a> <a href="/wiki/%EC%9A%A9%EB%8B%B4%EA%B3%B5_%EC%83%A4%EB%A5%BC" title="용담공 샤를">용담공 샤를</a>과 함께 <a class="new" href="/w/index.php?title=Eidgenossen&amp;action=edit&amp;redlink=1" title="Eidgenossen (없는 문서)">Eidgenossen</a>(혹은 <a class="mw-redirect" href="/wiki/%EC%8A%A4%EC%9C%84%EC%8A%A4_%EC%97%B0%EB%B0%A9" title="스위스 연방">스위스 연방</a>)의 확대를 막기 위해 <a href="/wiki/%EC%95%8C%EC%9E%90%EC%8A%A4" title="알자스">알자스</a>에 있던 자신의 영지를 담보로 부르고뉴 공작에게 양도했다.</p>
<p>그러나 샤를의 <a href="/wiki/%EB%9D%BC%EC%9D%B8_%EA%B0%95" title="라인 강">라인 강</a> 서쪽과의 관계는 지그문트가 바라던 스위스를 공격하는 동기가 되지 못했다. 샤를은 <a href="/wiki/%EB%B0%94%EC%A0%A4" title="바젤">바젤</a>(Basel) 및 <a href="/wiki/%EC%8A%A4%ED%8A%B8%EB%9D%BC%EC%8A%A4%EB%B6%80%EB%A5%B4" title="스트라스부르">스트라스부르</a>(Strasbourg), <a href="/wiki/%EB%AE%90%EB%A3%A8%EC%A6%88" title="뮐루즈">뮐루즈</a>(Mulhouse) 등의 도시에 대항해 통상 금지령(embargo)를 지방 행정관(reeve) <a class="new" href="/w/index.php?title=%ED%8E%98%ED%84%B0_%ED%8F%B0_%ED%97%A4%EA%B2%90%EB%B0%94%ED%9D%90&amp;action=edit&amp;redlink=1" title="페터 폰 헤겐바흐 (없는 문서)">페터 폰 헤겐바흐</a>(Peter von Hagenbach)에게 실행하게 했다.</p>
<p>헤겐바흐는 압정을 펼쳤기 때문에 부르고뉴에 대해 도시 주민의 반감은 강해지게 되었고, 도시는 <a href="/wiki/%EB%B2%A0%EB%A5%B8" title="베른">베른</a>(Berne)에게 구원을 요청했다. 샤를의 확장 전략은 1473년~1474년에 걸쳐 이루어진 <a class="new" href="/w/index.php?title=%EC%BE%B0%EB%A5%B8%EC%9D%98_%EB%8C%80%EC%A3%BC%EA%B5%90&amp;action=edit&amp;redlink=1" title="쾰른의 대주교 (없는 문서)">쾰른의 대주교</a>(Archbishopric of Cologne)에 대한 공격, <a class="new" href="/w/index.php?title=%EB%85%B8%EC%9D%B4%EC%8A%A4_%EA%B3%B5%EC%84%B1%EC%A0%84&amp;action=edit&amp;redlink=1" title="노이스 공성전 (없는 문서)">노이스 공성전</a>(Siege of Neuss)의 실패로 처음으로 좌절되었다.</p>
<p>1474년 지기문트는 스위스와 화평을 획책하고, <a href="/wiki/%EC%BD%98%EC%8A%A4%ED%83%84%EC%B8%A0" title="콘스탄츠">콘스탄츠</a>(Konstanz)에서 후에 "영구협정"(Ewige Richtung)이라고 부르는 협정을 맺었다. 또 한편 지그문트는 샤를에게 양도한 영지를 도로 구입하려 했으나 이것은 거절당했다. 같은 해 4월 30일 알자스에서 반 부르고뉴파에게 사로잡힌 헤겐바흐가 참수형에 처해졌다.</p>
<p>알자스 및 스위스의 도시와 지그문트는 결속하여 "대부르고뉴 동맹"을 맺고, 같은해 <a href="/wiki/11%EC%9B%94_13%EC%9D%BC" title="11월 13일">11월 13일</a> <a class="new" href="/w/index.php?title=%EC%97%90%EB%A6%AC%EC%BF%A0%EB%A5%B4_%EC%A0%84%ED%88%AC&amp;action=edit&amp;redlink=1" title="에리쿠르 전투 (없는 문서)">에리쿠르 전투</a>( Battle of Héricourt)에서 승리해 <a href="/wiki/%ED%94%84%EB%9E%91%EC%8A%88%EC%BD%A9%ED%85%8C" title="프랑슈콩테">프랑슈콩테</a>의 쥐라(Jura)를 정복했다. 다음 해 <a href="/wiki/1475%EB%85%84" title="1475년">1475년</a> 베른군이 샤를과 동맹관계에 있던 <a class="mw-redirect" href="/wiki/%EC%82%AC%EB%B3%B4%EC%9D%B4%EC%95%84_%EA%B3%B5%EA%B5%AD" title="사보이아 공국">사보이아 공국</a>(Duchy of Savoy)의 <a href="/wiki/%EB%B3%B4_%EC%A3%BC" title="보 주">보</a>(Vaud) 지방을 정복해 파괴했다.</p>
<p><a href="/wiki/%EB%B0%9C%EB%A0%88_%EC%A3%BC" title="발레 주">발레</a>(Valais)에서는 <a href="/wiki/1475%EB%85%84" title="1475년">1475년</a> 11월 <a class="new" href="/w/index.php?title=%ED%94%8C%EB%9E%80%ED%83%80_%EC%A0%84%ED%88%AC&amp;action=edit&amp;redlink=1" title="플란타 전투 (없는 문서)">플란타 전투</a>(Battle on the Planta)에서 독립공화국 <i>지벤 젠덴</i>(Sieben Zenden)이 베른군 및 기타 동맹국의 도움에 의해 사보이아인을 저지 발레(lower Valais)에서 몰아냈다.</p>
<p><a href="/wiki/1476%EB%85%84" title="1476년">1476년</a> 3월 샤를은 보복을 위해 사보이아의 <a class="new" href="/w/index.php?title=%ED%94%BC%EC%97%90%EB%A5%B4_%EB%93%9C_%EB%A1%9C%EB%AA%BD&amp;action=edit&amp;redlink=1" title="피에르 드 로몽 (없는 문서)">피에르 드 로몽</a>(Pierre de Romont)의 영지 <a class="new" href="/w/index.php?title=%EA%B7%B8%EB%9E%91%EC%86%A1&amp;action=edit&amp;redlink=1" title="그랑송 (없는 문서)">그랑송</a>(Grandson)으로 진군했다. 이곳은 같은해 1월 부르고뉴 군이 공략할 때 항복한 스위스 병사를 본보기를 위해 교수형에 처하고, 호수에 익사시켰던 인연이 있는 땅이었다.</p>
<p>스위스 동맹군이 수일 후 이곳에 도착했고, <a href="/wiki/%EA%B7%B8%EB%9E%91%EC%86%A1_%EC%A0%84%ED%88%AC" title="그랑송 전투">그랑송 전투</a>(Battle of Grandson)에서 샤를은 뼈아픈 패배를 겪었다. 그는 전장에서 물러날 여유를 얻지 못해, 대포 및 많은 식량, 보물을 남기고 철수하였다. 새로운 군대를 편성하였으나 샤를은 다시 <a class="new" href="/w/index.php?title=%EB%AA%A8%EB%9D%BC_%EC%A0%84%ED%88%AC&amp;action=edit&amp;redlink=1" title="모라 전투 (없는 문서)">모라 전투</a>(Battle of Morat)에서 스위스 군에게 패배했다.</p>
<p><a href="/wiki/1477%EB%85%84" title="1477년">1477년</a> 샤를은 <a href="/wiki/%EB%82%AD%EC%8B%9C_%EC%A0%84%ED%88%AC" title="낭시 전투">낭시 전투</a>에서 <a class="new" href="/w/index.php?title=%EB%A1%9C%EB%A0%8C%EC%9D%98_%EA%B5%B0%EC%A3%BC&amp;action=edit&amp;redlink=1" title="로렌의 군주 (없는 문서)">로렌 공작</a> <a class="new" href="/w/index.php?title=%EB%A1%9C%EB%A0%8C_%EA%B3%B5%EC%9E%91_%EB%A5%B4%EB%84%A4_2%EC%84%B8&amp;action=edit&amp;redlink=1" title="로렌 공작 르네 2세 (없는 문서)">르네 2세</a>(René II, Duke of Lorraine)가 이끄는 로렌 군과 그를 따르는 스위스 병사와 교전하다 전사했다.</p>
<h2><span class="mw-headline" id=".EC.A0.84.EC.9F.81.EC.9D.98_.EC.98.81.ED.96.A5">전쟁의 영향</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%B6%80%EB%A5%B4%EA%B3%A0%EB%89%B4_%EC%A0%84%EC%9F%81&amp;action=edit&amp;section=3" title="부분 편집: 전쟁의 영향">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>샤를의 죽음으로 부르고뉴 공작 가문(발루아-부르고뉴 가)의 남자는 맥이 끊겼다. 부르고뉴 공국의 영토인 <a href="/wiki/%ED%94%8C%EB%9E%91%EB%93%9C%EB%A5%B4" title="플랑드르">플랑드르</a>는 <a href="/wiki/%EB%A7%89%EC%8B%9C%EB%B0%80%EB%A6%AC%EC%95%88_1%EC%84%B8_(%EC%8B%A0%EC%84%B1_%EB%A1%9C%EB%A7%88_%EC%A0%9C%EA%B5%AD)" title="막시밀리안 1세 (신성 로마 제국)">오스트리아의 대공 막시밀리안</a>이 훗날 <a class="mw-redirect" href="/wiki/%EC%8B%A0%EC%84%B1_%EB%A1%9C%EB%A7%88_%EC%A0%9C%EA%B5%AD_%ED%99%A9%EC%A0%9C" title="신성 로마 제국 황제">신성 로마 제국 황제</a>가 되고, 샤를의 딸 <a class="mw-redirect" href="/wiki/%EB%B6%80%EB%A5%B4%EA%B3%A0%EB%89%B4%EC%9D%98_%EB%A7%88%EB%A6%AC" title="부르고뉴의 마리">마리</a>와 결혼함으로써 마리의 사후 합스부르크 가문 영지가 되었다.</p>
<p>부르고뉴 공작령은 <a href="/wiki/%EB%A3%A8%EC%9D%B4_11%EC%84%B8" title="루이 11세">루이 11세</a>(Louis XI)가 다스리는 <a href="/wiki/%ED%94%84%EB%9E%91%EC%8A%A4" title="프랑스">프랑스</a>에 귀속되었고 프랑슈콩테도 프랑스에 병합되었으나 <a href="/wiki/1493%EB%85%84" title="1493년">1493년</a> <a class="new" href="/w/index.php?title=%EC%83%9D%EB%A6%AC%EC%8A%A4_%EC%A1%B0%EC%95%BD&amp;action=edit&amp;redlink=1" title="생리스 조약 (없는 문서)">생리스 조약</a>(treaty of Senlis)에 의해 프랑스 왕 <a href="/wiki/%EC%83%A4%EB%A5%BC_8%EC%84%B8" title="샤를 8세">샤를 8세</a>(Charles VIII)로부터 막시밀리안 1세의 아들 <a class="mw-disambig" href="/wiki/%ED%8E%A0%EB%A6%AC%ED%8E%98_1%EC%84%B8" title="펠리페 1세">단려공 필리프</a>에게 양도되었다. 이것은 샤를 8세의 <a href="/wiki/%EC%9D%B4%ED%83%88%EB%A6%AC%EC%95%84" title="이탈리아">이탈리아</a> 침략 때 막시밀리안 1세에게 중립의 입장을 지키게 하기 위한 매수공작이었다.</p>
<p>또한 당시 유럽 최강이라고 널리 알려졌던 부르고뉴 군에게 승리한 것에서 스위스는 거의 무적이라는 명성을 얻었다. 부르고뉴 전쟁은 유럽의 전장에서 <a class="mw-redirect" href="/wiki/%EC%8A%A4%EC%9C%84%EC%8A%A4_%EC%9A%A9%EB%B3%91" title="스위스 용병">스위스 용병</a>의 평가를 특출 나게 만들었다.</p>
