<p><b>부르봉 가</b>(<span style="font-size: smaller;"><a href="/wiki/%ED%94%84%EB%9E%91%EC%8A%A4%EC%96%B4" title="프랑스어">프랑스어</a>:</span> <span lang="fr" xml:lang="fr">Maison de Bourbon</span>)는 <a href="/wiki/%EC%B9%B4%ED%8E%98_%EC%99%95%EC%A1%B0" title="카페 왕조">카페 왕조</a>의 한 계열로 <a class="mw-redirect" href="/wiki/%ED%94%84%EB%9E%91%EC%8A%A4%EC%9D%98_%EB%A3%A8%EC%9D%B4_9%EC%84%B8" title="프랑스의 루이 9세">루이 9세</a>의 마지막 아들 클레르몽의 백작 <a href="/wiki/%EB%A1%9C%EB%B2%A0%EB%A5%B4_%EB%93%9C_%ED%94%84%EB%9E%91%EC%8A%A4" title="로베르 드 프랑스">로베르 드 프랑스</a>(Robert de France)로부터 시작되었다. 부르봉(Bourbon)의 어원은 켈트어로 ‘진흙(Borvo, 혹은 Borbo)’을 뜻한다.</p>

<h2><span class="mw-headline" id=".EB.B6.80.EB.A5.B4.EB.B4.89_.EC.99.95.EA.B0.80.EC.9D.98_.ED.83.84.EC.83.9D.EA.B3.BC_.EB.B0.9C.EC.A0.84">부르봉 왕가의 탄생과 발전</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%B6%80%EB%A5%B4%EB%B4%89%EA%B0%80&amp;action=edit&amp;section=1" title="부분 편집: 부르봉 왕가의 탄생과 발전">원본 편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>부르봉 왕가는 <a class="mw-redirect" href="/wiki/%ED%94%84%EB%9E%91%EC%8A%A4%EC%9D%98_%EB%A3%A8%EC%9D%B4_9%EC%84%B8" title="프랑스의 루이 9세">루이 9세</a>의 6번째 이자 막내아들인 <a href="/wiki/%EB%A1%9C%EB%B2%A0%EB%A5%B4_%EB%93%9C_%ED%94%84%EB%9E%91%EC%8A%A4" title="로베르 드 프랑스">로베르 드 프랑스</a>(Robert de France)로부터 시작된다. 그는 샤로래(<span style="font-size: smaller;"><a href="/wiki/%ED%94%84%EB%9E%91%EC%8A%A4%EC%96%B4" title="프랑스어">프랑스어</a>:</span> <span lang="fr" xml:lang="fr">Charolais</span>)의 영주 장 드 부르고뉴(<span style="font-size: smaller;"><a href="/wiki/%ED%94%84%EB%9E%91%EC%8A%A4%EC%96%B4" title="프랑스어">프랑스어</a>:</span> <span lang="fr" xml:lang="fr">Jean de Bourgogne</span>)와 아니에스 드 부르봉(<span style="font-size: smaller;"><a href="/wiki/%ED%94%84%EB%9E%91%EC%8A%A4%EC%96%B4" title="프랑스어">프랑스어</a>:</span> <span lang="fr" xml:lang="fr">Agnès de Bourbon</span>)의 외동딸 베아트리스(<span style="font-size: smaller;"><a href="/wiki/%ED%94%84%EB%9E%91%EC%8A%A4%EC%96%B4" title="프랑스어">프랑스어</a>:</span> <span lang="fr" xml:lang="fr">Béatrice</span>)와 결혼을 함으로써 부르봉(<span style="font-size: smaller;"><a href="/wiki/%ED%94%84%EB%9E%91%EC%8A%A4%EC%96%B4" title="프랑스어">프랑스어</a>:</span> <span lang="fr" xml:lang="fr">Bourbon</span>)의 영주가 되었다.</p>
<p>부르봉 영지는 <a href="/wiki/1327%EB%85%84" title="1327년">1327년</a> 부르봉가의 루이 1세(<span style="font-size: smaller;"><a href="/wiki/%ED%94%84%EB%9E%91%EC%8A%A4%EC%96%B4" title="프랑스어">프랑스어</a>:</span> <span lang="fr" xml:lang="fr">Louis Ier de Bourbon</span>)를 위하여 공작령으로 승격되었다. 부르봉가의 루이 2세(<span style="font-size: smaller;"><a href="/wiki/%ED%94%84%EB%9E%91%EC%8A%A4%EC%96%B4" title="프랑스어">프랑스어</a>:</span> <span lang="fr" xml:lang="fr">Louis II de Bourbon</span>) (1337-1410)는 <a href="/wiki/1371%EB%85%84" title="1371년">1371년</a> 안 도베르뉴(<span style="font-size: smaller;"><a href="/wiki/%ED%94%84%EB%9E%91%EC%8A%A4%EC%96%B4" title="프랑스어">프랑스어</a>:</span> <span lang="fr" xml:lang="fr">Anne d'Auvergne</span>)와 결혼을 하였다. 이 결혼으로 영지를 확장시켰다.</p>
<p>광신도적인 수도사 <a class="new" href="/w/index.php?title=%EC%9E%90%ED%81%AC_%ED%81%AC%EB%A0%88%EB%A7%9D&amp;action=edit&amp;redlink=1" title="자크 크레망 (없는 문서)">자크 크레망</a>(<span style="font-size: smaller;"><a href="/wiki/%ED%94%84%EB%9E%91%EC%8A%A4%EC%96%B4" title="프랑스어">프랑스어</a>:</span> <span lang="fr" xml:lang="fr">Jacques Clément</span>)이 <a class="mw-redirect" href="/wiki/%ED%94%84%EB%9E%91%EC%8A%A4%EC%9D%98_%EC%95%99%EB%A6%AC_3%EC%84%B8" title="프랑스의 앙리 3세">앙리3세</a>를 암살하자 <a class="mw-redirect" href="/wiki/%EB%B0%9C%EB%A3%A8%EC%95%84_%EC%99%95%EA%B0%80" title="발루아 왕가">발루아 왕가</a>에는 왕위를 이을 사람이 아무도 없었다. <a class="mw-redirect" href="/wiki/%EC%82%B4%EB%A6%AC%EC%B9%B4_%EB%B2%95" title="살리카 법">살리카 법</a>이라고 불리는 당시 통용되었던 계승법으로 사망한 <a class="mw-redirect" href="/wiki/%ED%94%84%EB%9E%91%EC%8A%A4%EC%9D%98_%EC%95%99%EB%A6%AC_3%EC%84%B8" title="프랑스의 앙리 3세">앙리3세</a>의 22촌인 <a class="mw-redirect" href="/wiki/%ED%94%84%EB%9E%91%EC%8A%A4%EC%9D%98_%EC%95%99%EB%A6%AC_4%EC%84%B8" title="프랑스의 앙리 4세">앙리4세</a>가 왕위를 계승했다.</p>
<h2><span class="mw-headline" id=".EC.97.AD.EC.82.AC">역사</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%B6%80%EB%A5%B4%EB%B4%89%EA%B0%80&amp;action=edit&amp;section=2" title="부분 편집: 역사">원본 편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>부르봉 왕조는 <a href="/wiki/%ED%94%84%EB%9E%91%EC%8A%A4_%ED%98%81%EB%AA%85" title="프랑스 혁명">프랑스 혁명</a>이 일어나기 전인 <a href="/wiki/1789%EB%85%84" title="1789년">1789년</a>까지, <a href="/wiki/%EB%82%98%ED%8F%B4%EB%A0%88%EC%98%B9_%EB%B3%B4%EB%82%98%ED%8C%8C%EB%A5%B4%ED%8A%B8" title="나폴레옹 보나파르트">나폴레옹 보나파르트</a>가 물러난 <a href="/wiki/1815%EB%85%84" title="1815년">1815년</a>에서 <a href="/wiki/7%EC%9B%94_%ED%98%81%EB%AA%85" title="7월 혁명">7월 혁명</a>이 일어난 <a href="/wiki/1830%EB%85%84" title="1830년">1830년</a>까지 지속되었다. <a class="mw-redirect" href="/wiki/%EC%97%90%EC%8A%A4%ED%8C%8C%EB%83%90" title="에스파냐">에스파냐</a>에서는 1701년부터 통치하고 있는데, <a href="/wiki/1874%EB%85%84" title="1874년">1874년</a>부터 <a href="/wiki/1875%EB%85%84" title="1875년">1875년</a>까지, <a href="/wiki/1931%EB%85%84" title="1931년">1931년</a>부터 <a href="/wiki/1975%EB%85%84" title="1975년">1975년</a>까지는 공화정으로 인하여 다른 나라로 도피하였다. 그 뒤 <a href="/wiki/%ED%9B%84%EC%95%88_%EC%B9%B4%EB%A5%BC%EB%A1%9C%EC%8A%A4_1%EC%84%B8" title="후안 카를로스 1세">후안 카를로스 1세</a>가 <a class="mw-redirect" href="/wiki/%EC%9E%85%ED%97%8C_%EA%B5%B0%EC%A3%BC%EC%A0%9C" title="입헌 군주제">입헌 군주제</a> <a class="mw-redirect" href="/wiki/%EC%97%90%EC%8A%A4%ED%8C%8C%EB%83%90" title="에스파냐">에스파냐</a>의 <a href="/wiki/%EC%8A%A4%ED%8E%98%EC%9D%B8%EC%9D%98_%EA%B5%B0%EC%A3%BC" title="스페인의 군주">왕</a>으로 즉위하였다. 이외에도 부르봉 왕가는 나바르, 프랑스, 스페인, 양시칠리아, 룩셈부르크, 안도라, 루카, 파르마와 같이 유럽의 여러나라를 다스렸고, 여러 분파로 나뉘었다.</p>

<h3><span class="mw-headline" id=".ED.8C.8C.EB.A5.B4.EB.A7.88_.EA.B3.B5.EC.9E.91">파르마 공작</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%B6%80%EB%A5%B4%EB%B4%89%EA%B0%80&amp;action=edit&amp;section=11" title="부분 편집: 파르마 공작">원본 편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<ul>
<li><a class="mw-redirect" href="/wiki/%EC%8A%A4%ED%8E%98%EC%9D%B8%EC%9D%98_%EC%B9%B4%EB%A5%BC%EB%A1%9C%EC%8A%A4_3%EC%84%B8" title="스페인의 카를로스 3세">카를로 1세</a> (1731년 ~ 1735년)</li>
<li><a class="mw-redirect" href="/wiki/%ED%8C%8C%EB%A5%B4%EB%A7%88%EC%9D%98_%ED%95%84%EB%A6%AC%ED%8F%AC_1%EC%84%B8" title="파르마의 필리포 1세">필리포 1세</a> (1748년 ~ 1765년)</li>
<li><a class="mw-redirect" href="/wiki/%ED%8C%8C%EB%A5%B4%EB%A7%88%EC%9D%98_%ED%8E%98%EB%A5%B4%EB%94%94%EB%82%9C%EB%8F%84_1%EC%84%B8" title="파르마의 페르디난도 1세">페르디난도 1세</a> (1765년 ~ 1802년)</li>
<li><a href="/wiki/%ED%8C%8C%EB%A5%B4%EB%A7%88%EC%9D%98_%EC%B9%B4%EB%A5%BC%EB%A1%9C_2%EC%84%B8" title="파르마의 카를로 2세">카를로 2세</a> (1847년 ~ 1848년)</li>
<li><a class="new" href="/w/index.php?title=%ED%8C%8C%EB%A5%B4%EB%A7%88%EC%9D%98_%EC%B9%B4%EB%A5%BC%EB%A1%9C_3%EC%84%B8&amp;action=edit&amp;redlink=1" title="파르마의 카를로 3세 (없는 문서)">카를로 3세</a> (1848년 ~ 1854년)</li>

<p>파르마 공국은 <a href="/wiki/%EC%82%AC%EB%A5%B4%EB%8D%B0%EB%83%90_%EC%99%95%EA%B5%AD" title="사르데냐 왕국">사르데냐 왕국</a>이 주도한 <a class="mw-redirect" href="/wiki/%EB%A6%AC%EC%86%8C%EB%A5%B4%EC%A7%80%EB%A9%98%ED%86%A0" title="리소르지멘토">리소르지멘토</a>로 사르데냐 왕국에 합병된다.</p>
<h2><span class="mw-headline" id=".EB.B6.80.EB.A5.B4.EB.B4.89_.EB.82.98.ED.8F.B4.EB.A6.AC_.EC.99.95.EA.B0.80">부르봉 나폴리 왕가</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%B6%80%EB%A5%B4%EB%B4%89%EA%B0%80&amp;action=edit&amp;section=12" title="부분 편집: 부르봉 나폴리 왕가">원본 편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<h3><span class="mw-headline" id=".EB.82.98.ED.8F.B4.EB.A6.AC.EC.99.80_.EC.8B.9C.EC.B9.A0.EB.A6.AC.EC.95.84.EC.9D.98_.EC.99.95">나폴리와 시칠리아의 왕</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%B6%80%EB%A5%B4%EB%B4%89%EA%B0%80&amp;action=edit&amp;section=13" title="부분 편집: 나폴리와 시칠리아의 왕">원본 편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p><a href="/wiki/%EB%82%98%ED%8F%B4%EB%A6%AC_%EC%99%95%EA%B5%AD" title="나폴리 왕국">나폴리 왕국</a>과 <a href="/wiki/%EC%8B%9C%EC%B9%A0%EB%A6%AC%EC%95%84_%EC%99%95%EA%B5%AD" title="시칠리아 왕국">시칠리아 왕국</a>은 오래전부터 동군연합을 이루어왔다. 페르디난도 1세는 1806년부터 1815년까지는 나폴리의 왕을 차지하지 못했다. 그 이유는 <a class="mw-redirect" href="/wiki/%EB%B3%B4%EB%82%98%ED%8C%8C%EB%A5%B4%ED%8A%B8_%EC%99%95%EA%B0%80" title="보나파르트 왕가">보나파르트 왕가</a>가 나폴리 왕위를 차지하였기 때문이다.</p>
<ul>
<li><a class="mw-redirect" href="/wiki/%EC%8A%A4%ED%8E%98%EC%9D%B8%EC%9D%98_%EC%B9%B4%EB%A5%BC%EB%A1%9C%EC%8A%A4_3%EC%84%B8" title="스페인의 카를로스 3세">카를로 7세</a> (1734년 ~ 1759년)</li>
<li><a class="mw-redirect" href="/wiki/%EC%96%91%EC%8B%9C%EC%B9%A0%EB%A6%AC%EC%95%84%EC%9D%98_%ED%8E%98%EB%A5%B4%EB%94%94%EB%82%9C%EB%8F%84_1%EC%84%B8" title="양시칠리아의 페르디난도 1세">페르디난도 1세</a> (1759년 ~ 1816년)</li>
</ul>
<h2><span class="mw-headline" id=".EB.B6.80.EB.A5.B4.EB.B4.89_.EC.96.91.EC.8B.9C.EC.B9.A0.EB.A6.AC.EC.95.84_.EC.99.95.EA.B0.80">부르봉 양시칠리아 왕가</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%B6%80%EB%A5%B4%EB%B4%89%EA%B0%80&amp;action=edit&amp;section=14" title="부분 편집: 부르봉 양시칠리아 왕가">원본 편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>1816년에는 나폴리 왕국이 시칠리아 왕국을 합병시키고 양시칠리아의 왕이 되었다. 따라서 왕가의 이름도 부르봉 양시칠리아로 변하였다.</p>
<h3><span class="mw-headline" id=".EC.96.91.EC.8B.9C.EC.B9.A0.EB.A6.AC.EC.95.84.EC.9D.98_.EC.99.95">양시칠리아의 왕</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%B6%80%EB%A5%B4%EB%B4%89%EA%B0%80&amp;action=edit&amp;section=15" title="부분 편집: 양시칠리아의 왕">원본 편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<ul>

<li><a class="mw-redirect" href="/wiki/%EC%96%91%EC%8B%9C%EC%B9%A0%EB%A6%AC%EC%95%84%EC%9D%98_%ED%94%84%EB%9E%80%EC%B2%B4%EC%8A%A4%EC%BD%94_1%EC%84%B8" title="양시칠리아의 프란체스코 1세">프란체스코 1세</a> (1825년 ~ 1830년)</li>
<li><a class="new" href="/w/index.php?title=%EC%96%91%EC%8B%9C%EC%B9%A0%EB%A6%AC%EC%95%84%EC%9D%98_%ED%8E%98%EB%A5%B4%EB%94%94%EB%82%9C%EB%8F%84_2%EC%84%B8&amp;action=edit&amp;redlink=1" title="양시칠리아의 페르디난도 2세 (없는 문서)">페르디난도 2세</a> (1830년 ~ 1859년)</li>

</ul>
<p>양시칠리아 왕국은 <a href="/wiki/%EC%82%AC%EB%A5%B4%EB%8D%B0%EB%83%90_%EC%99%95%EA%B5%AD" title="사르데냐 왕국">사르데냐 왕국</a>이 주도한 <a class="mw-redirect" href="/wiki/%EB%A6%AC%EC%86%8C%EB%A5%B4%EC%A7%80%EB%A9%98%ED%86%A0" title="리소르지멘토">리소르지멘토</a>로 사르데냐 왕국에 합병된다.</p>
<h3><span class="mw-headline" id=".EB.B6.80.EB.A5.B4.EB.B4.89_.EC.96.91.EC.8B.9C.EC.B9.A0.EB.A6.AC.EC.95.84_.EC.99.95.EA.B0.80.EC.9D.98_.EC.88.98.EC.9E.A5">부르봉 양시칠리아 왕가의 수장</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%B6%80%EB%A5%B4%EB%B4%89%EA%B0%80&amp;action=edit&amp;section=16" title="부분 편집: 부르봉 양시칠리아 왕가의 수장">원본 편집</a><span class="mw-editsection-bracket">]</span></span></h3>
