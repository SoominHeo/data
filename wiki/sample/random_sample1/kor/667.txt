<p><b>666</b>(육백육십육)은 <a href="/wiki/665" title="665">665</a>보다 크고 <a href="/wiki/667" title="667">667</a>보다 작은 <a href="/wiki/%EC%9E%90%EC%97%B0%EC%88%98" title="자연수">자연수</a>이다.</p>
<h2><span class="mw-headline" id=".EC.88.98.ED.95.99">수학</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=666&amp;action=edit&amp;section=1" title="부분 편집: 수학">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<ul>
<li>544번째 <a href="/wiki/%ED%95%A9%EC%84%B1%EC%88%98" title="합성수">합성수</a>로, 그 <a href="/wiki/%EC%95%BD%EC%88%98" title="약수">약수</a>는 <a href="/wiki/1" title="1">1</a>, <a href="/wiki/2" title="2">2</a>, <a href="/wiki/3" title="3">3</a>, <a href="/wiki/6" title="6">6</a>, <a href="/wiki/9" title="9">9</a>, <a href="/wiki/18" title="18">18</a>, <a href="/wiki/37" title="37">37</a>, <a href="/wiki/74" title="74">74</a>, <a href="/wiki/111" title="111">111</a>, 222, 333, 666이다.</li>
<li>666번째 <a href="/wiki/%EC%86%8C%EC%88%98_(%EC%88%98%EB%A1%A0)" title="소수 (수론)">소수</a>는 4973이다. 666번째 합성수는 <a href="/wiki/2" title="2">2</a>×<a href="/wiki/13" title="13">13</a>×<a href="/wiki/31" title="31">31</a> = 816이다.</li>
<li>36번째 <a href="/wiki/%EC%82%BC%EA%B0%81%EC%88%98" title="삼각수">삼각수</a>이다. 즉 1에서 36까지를 모두 더하면 666이 된다. (1 + 2 + 3 + 4 + ... + 35 + 36 = 666)</li>
<li><a href="/wiki/%EC%98%A4%EC%8B%9C%EB%A6%AC%EC%8A%A4" title="오시리스">오시리스</a> 숫자. 6X6X6 = 216이 된다. 태양이 별자리 하나를 이동하는 시간은 2,160년. 성경의 666은 지축의 이동으로 야기되는 재해를 경고함.</li>
</ul>
<h2><span class="mw-headline" id=".EA.B3.BC.ED.95.99">과학</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=666&amp;action=edit&amp;section=2" title="부분 편집: 과학">편집</a><span class="mw-editsection-bracket">]</span></span></h2>

<h2><span class="mw-headline" id=".EA.B8.B0.ED.83.80">기타</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=666&amp;action=edit&amp;section=3" title="부분 편집: 기타">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<ul>
<li><a href="/wiki/%EA%B8%B0%EB%8F%85%EA%B5%90" title="기독교">기독교</a>에서 <a href="/wiki/%EC%A7%90%EC%8A%B9%EC%9D%98_%EC%88%AB%EC%9E%90" title="짐승의 숫자">짐승의 숫자</a>로 부른다. (<a class="mw-redirect" href="/wiki/%EC%9A%94%ED%95%9C_%EB%AC%B5%EC%8B%9C%EB%A1%9D" title="요한 묵시록">요한 묵시록</a> 13:18)</li>
<li><a href="/wiki/%EB%A1%9C%EB%A7%88_%EC%88%AB%EC%9E%90" title="로마 숫자">로마 숫자</a>로 썼을 때, 666 (DCLXVI)은 1에서 500을 나타내는 여섯 글자가 모두 한번씩 크기의 역순으로 나타난다. (D = 500, C = 100, L = 50, X = 10, V = 5, I = 1).</li>
<li>666은 <a class="mw-disambig" href="/wiki/%EB%91%A0" title="둠">둠</a> 게임을 네트워크로 할 때 쓰는 TCP 포트 번호이다.</li>
<li><a href="/wiki/%EB%A0%88%EB%B0%8D%EC%A6%88" title="레밍즈">레밍즈</a>의 스테이지 중 <a class="new" href="/w/index.php?title=All_the_6%27s_........&amp;action=edit&amp;redlink=1" title="All the 6's ........ (없는 문서)">All the 6's ........</a>는 맵 구조가 666 모양으로 되어 있는 스테이지이다.</li>
<li>666은 넥스트의 6번째 정규 앨범으로 3부작으로 제작중이다(Part1 12/9 발매)</li>
</ul>
