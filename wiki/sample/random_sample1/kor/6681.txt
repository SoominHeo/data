<p><b>부산은행</b>(釜山銀行, Busan Bank)은 <a href="/wiki/%EB%8C%80%ED%95%9C%EB%AF%BC%EA%B5%AD" title="대한민국">대한민국</a>의 <a href="/wiki/%EC%A7%80%EB%B0%A9%EC%9D%80%ED%96%89" title="지방은행">지방은행</a>이자 <a href="/wiki/BNK%EA%B8%88%EC%9C%B5%EA%B7%B8%EB%A3%B9" title="BNK금융그룹">BNK금융그룹</a>의 자회사이다. 본점은 <a href="/wiki/%EB%B6%80%EC%82%B0%EA%B4%91%EC%97%AD%EC%8B%9C" title="부산광역시">부산광역시</a> <a href="/wiki/%EB%82%A8%EA%B5%AC_(%EB%B6%80%EC%82%B0%EA%B4%91%EC%97%AD%EC%8B%9C)" title="남구 (부산광역시)">남구</a> <a href="/wiki/%EB%AC%B8%ED%98%84%EB%8F%99" title="문현동">문현동</a> <a href="/wiki/%EB%AC%B8%ED%98%84%EA%B8%88%EC%9C%B5%EB%8B%A8%EC%A7%80" title="문현금융단지">문현금융단지</a>에 있다. <a href="/wiki/%EB%B6%80%EC%82%B0%EA%B4%91%EC%97%AD%EC%8B%9C" title="부산광역시">부산광역시</a>를 중심으로 <a href="/wiki/%EA%B2%BD%EC%83%81%EB%82%A8%EB%8F%84" title="경상남도">경상남도</a>, <a href="/wiki/%EB%8C%80%EA%B5%AC%EA%B4%91%EC%97%AD%EC%8B%9C" title="대구광역시">대구광역시</a>, <a href="/wiki/%EC%84%9C%EC%9A%B8%ED%8A%B9%EB%B3%84%EC%8B%9C" title="서울특별시">서울특별시</a>, <a href="/wiki/%EC%9D%B8%EC%B2%9C%EA%B4%91%EC%97%AD%EC%8B%9C" title="인천광역시">인천광역시</a>, <a href="/wiki/%EA%B4%91%EC%A3%BC%EA%B4%91%EC%97%AD%EC%8B%9C" title="광주광역시">광주광역시</a>, <a href="/wiki/%EC%9A%B8%EC%82%B0%EA%B4%91%EC%97%AD%EC%8B%9C" title="울산광역시">울산광역시</a>, <a href="/wiki/%EB%8C%80%EC%A0%84%EA%B4%91%EC%97%AD%EC%8B%9C" title="대전광역시">대전광역시</a>, <a href="/wiki/%EA%B2%BD%EC%83%81%EB%B6%81%EB%8F%84" title="경상북도">경상북도</a>에 있다. <a href="/wiki/%EB%8C%80%ED%95%9C%EB%AF%BC%EA%B5%AD" title="대한민국">대한민국</a>의 <a href="/wiki/%EC%A7%80%EB%B0%A9%EC%9D%80%ED%96%89" title="지방은행">지방은행</a> 중에서 유일하게 모든 광역시에 영업망을 갖추고 있는 <a href="/wiki/%EC%A7%80%EB%B0%A9%EC%9D%80%ED%96%89" title="지방은행">지방은행</a>이며, <a href="/wiki/%EB%8C%80%ED%95%9C%EB%AF%BC%EA%B5%AD" title="대한민국">대한민국</a>의 <a href="/wiki/%EC%A7%80%EB%B0%A9%EC%9D%80%ED%96%89" title="지방은행">지방은행</a> 중 가장 규모가 크다.</p>

<h2><span class="mw-headline" id=".EC.B9.B4.EB.93.9C_.EC.97.85.EB.AC.B4">카드 업무</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%B6%80%EC%82%B0%EC%9D%80%ED%96%89&amp;action=edit&amp;section=5" title="부분 편집: 카드 업무">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>부산은행은 <a href="/wiki/%EB%B9%84%EC%94%A8%EC%B9%B4%EB%93%9C" title="비씨카드">비씨카드</a> 회원사다. 신용카드 연회비의 경우, 상품 구분없이 연 240만원 이상 사용 실적이 있을시 다음년도 연회비를 면제받는다. 체크카드는 발급 수수료 1,000원이 부과되고, 갱신이나 훼손에 따른 재발급시 수수료는 부과되지 않으나 <a href="/wiki/%EB%8C%80%EA%B5%AC%EC%9D%80%ED%96%89" title="대구은행">대구은행</a>처럼 발급 수수료가 캐쉬백되는 제도는 없다. 다만, 신 상품이 나오면 프로모션 차원에서 발급비를 면제하는 행사를 열기도 한다. 국내외 사용이 가능한 하이체크 글로벌 카드에 대해서만 연회비 1,000원을 받지만, 해외에서는 직불카드로만 이용할 수 있다. 체크카드는 <a href="/wiki/%EC%A4%91%EA%B5%AD%EC%9D%80%EB%A0%A8" title="중국은련">중국은련</a>과 비씨 글로벌만 취급하며, 비씨 글로벌은 해외신용판매가 안 되고 <a href="/wiki/%ED%98%84%EA%B8%88_%EC%9E%90%EB%8F%99_%EC%9E%85%EC%B6%9C%EA%B8%88%EA%B8%B0" title="현금 자동 입출금기">ATM</a> 이용만 가능하다. <a href="/wiki/%EC%B2%B4%ED%81%AC%EC%B9%B4%EB%93%9C" title="체크카드">체크카드</a>는 <a class="mw-redirect" href="/wiki/%EB%A7%88%EC%8A%A4%ED%83%80%EC%B9%B4%EB%93%9C" title="마스타카드">마스타카드</a>가 달린 상품이 없고, 하이체크 글로벌 카드에는 <a href="/wiki/%EB%A7%88%EC%8A%A4%ED%84%B0%EC%B9%B4%EB%93%9C" title="마스터카드">마스터카드</a>의 직불카드망인 <a href="/wiki/%EB%A7%88%EC%97%90%EC%8A%A4%ED%8A%B8%EB%A1%9C_(%EC%A7%81%EB%B6%88%EC%B9%B4%EB%93%9C)" title="마에스트로 (직불카드)">마에스트로</a>가 달려 있다. 그 외에는 <a href="/wiki/%EB%A1%AF%EB%8D%B0%EC%B9%B4%EB%93%9C" title="롯데카드">롯데카드</a> 등지에서 부산은행을 결제 계좌로 연결할 수 있다.</p>
<p><a href="/wiki/2016%EB%85%84" title="2016년">2016년</a>에는 <a href="/wiki/%EB%A1%AF%EB%8D%B0%EC%B9%B4%EB%93%9C" title="롯데카드">롯데카드</a>와 제휴를 추진하고 있으며,<sup class="reference" id="cite_ref-2"><a href="#cite_note-2">[2]</a></sup> 썸뱅크의 런칭 이후에는 썸뱅크 제휴 <a href="/wiki/%EB%A1%AF%EB%8D%B0%EC%B9%B4%EB%93%9C" title="롯데카드">롯데카드</a>(신용/체크)가 출시됐다.</p>

<h2><span class="mw-headline" id=".ED.8A.B9.EC.A7.95">특징</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%B6%80%EC%82%B0%EC%9D%80%ED%96%89&amp;action=edit&amp;section=8" title="부분 편집: 특징">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p><a href="/wiki/%EB%A1%AF%EB%8D%B0%EA%B7%B8%EB%A3%B9" title="롯데그룹">롯데그룹</a>이 BS금융지주의 지분 13.6%를 보유하고 있으며, 이에 따라 롯데피에스넷과 제휴하여 전국 <a href="/wiki/%EC%84%B8%EB%B8%90%EC%9D%BC%EB%A0%88%EB%B8%90" title="세븐일레븐">세븐일레븐</a>, <a href="/wiki/%EB%A1%AF%EB%8D%B0%EB%A7%88%ED%8A%B8" title="롯데마트">롯데마트</a>, <a href="/wiki/%EB%A1%AF%EB%8D%B0%EC%87%BC%ED%95%91" title="롯데쇼핑">롯데백화점</a>에 설치되어 있는 롯데ATM에서 부산은행의 수수료만 내고 현금인출 등의 거래에 사용할 수 있다. 따라서 부산은행의 <a href="/wiki/%EC%A7%80%EB%B0%A9%EC%9D%80%ED%96%89_%EA%B3%B5%EB%8F%99%EC%A0%95%EB%B3%B4%EB%A7%9D" title="지방은행 공동정보망">뱅크라인통장</a>이 없어도 롯데ATM을 통해 전국의 <a href="/wiki/%EC%84%B8%EB%B8%90%EC%9D%BC%EB%A0%88%EB%B8%90" title="세븐일레븐">세븐일레븐</a> 등지에서 편리하게 이용할 수 있다. <a href="/wiki/%EB%B6%80%EC%82%B0%EA%B4%91%EC%97%AD%EC%8B%9C" title="부산광역시">부산광역시</a> 관내에 설치되어 있는 롯데ATM에서는 영업시간 내 부산은행 계좌끼리의 이체만 수수료가 없고, 현금인출을 포함한 나머지 거래는 거액의 수수료가 부과되었으나 2015년 11월 19일부터 타 지역 롯데ATM처럼 <a href="/wiki/%EB%B6%80%EC%82%B0%EA%B4%91%EC%97%AD%EC%8B%9C" title="부산광역시">부산광역시</a> 관내에 설치된 롯데ATM도 부산은행의 수수료만 내고 이용할 수 있도록 개선하였다.</p>
<p>2014년 10월에 <a href="/wiki/%EA%B2%BD%EB%82%A8%EC%9D%80%ED%96%89" title="경남은행">경남은행</a>을 인수한 이후 2016년 1월 18일부터 <a href="/wiki/%EA%B2%BD%EB%82%A8%EC%9D%80%ED%96%89" title="경남은행">경남은행</a>도 롯데ATM에서 이용이 가능하나, 이는 <a href="/wiki/%EA%B2%BD%EB%82%A8%EC%9D%80%ED%96%89" title="경남은행">경남은행</a>의 주 영업 구역인 <a href="/wiki/%EC%9A%B8%EC%82%B0%EA%B4%91%EC%97%AD%EC%8B%9C" title="울산광역시">울산광역시</a>와 <a href="/wiki/%EA%B2%BD%EC%83%81%EB%82%A8%EB%8F%84" title="경상남도">경상남도</a> 소재의 롯데ATM 한정이다.</p>
<p>2016년 3월 28일에는 <a href="/wiki/%EB%A1%AF%EB%8D%B0%EA%B7%B8%EB%A3%B9" title="롯데그룹">롯데그룹</a>과 합작한 핀테크 모바일 비대면식 금융서비스인 썸뱅크를 런칭하였다.<sup class="reference" id="cite_ref-3"><a href="#cite_note-3">[3]</a></sup></p>
