<h2><span class="mw-headline" id=".EC.97.AD.EC.82.AC">역사</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%B6%80%EC%84%B8%EB%A1%9C_%EC%97%AD&amp;action=edit&amp;section=1" title="부분 편집: 역사">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>부세로 역은 <a href="/wiki/%EB%B0%80%EB%9D%BC%EB%85%B8" title="밀라노">밀라노</a>-<a class="new" href="/w/index.php?title=%EA%B3%A0%EB%A5%B4%EA%B3%A4%EC%B4%90%EB%9D%BC_(%EB%B0%80%EB%9D%BC%EB%85%B8_%ED%98%84)&amp;action=edit&amp;redlink=1" title="고르곤촐라 (밀라노 현) (없는 문서)">고르곤촐라</a> 트램고속선의 정거장으로 <a href="/wiki/1968%EB%85%84" title="1968년">1968년</a> <a href="/wiki/5%EC%9B%94_5%EC%9D%BC" title="5월 5일">5월 5일</a>에 문을 열었다.<sup class="reference" id="cite_ref-1"><a href="#cite_note-1">[1]</a></sup> 트램고속선 원안에 포함되어 있지 않았던 네 역 중 하나 (나머지는 <a href="/wiki/%EC%B9%B4%EC%8B%9C%EB%82%98_%EB%B6%80%EB%A1%9C%EB%82%98_%EC%97%AD" title="카시나 부로나 역">카시나 부로나 역</a>, <a href="/wiki/%EB%B9%8C%EB%9D%BC_%ED%94%BC%EC%98%A4%EB%A6%AC%ED%83%80_%EC%97%AD" title="빌라 피오리타 역">빌라 피오리타 역</a>, <a href="/wiki/%EB%B9%8C%EB%9D%BC_%ED%8F%BC%ED%8E%98%EC%95%84_%EC%97%AD" title="빌라 폼페아 역">빌라 폼페아 역</a>)였다.<sup class="reference" id="cite_ref-2"><a href="#cite_note-2">[2]</a></sup> <a href="/wiki/1972%EB%85%84" title="1972년">1972년</a> <a href="/wiki/12%EC%9B%94_4%EC%9D%BC" title="12월 4일">12월 4일</a>부터 역이 소재한 <a href="/wiki/%EC%B9%B4%EC%8B%9C%EB%82%98_%EA%B3%A0%EB%B0%94_%EC%97%AD" title="카시나 고바 역">카시나 고바</a> - <a href="/wiki/%EA%B3%A0%EB%A5%B4%EA%B3%A4%EC%B4%90%EB%9D%BC_%EC%97%AD" title="고르곤촐라 역">고르곤촐라</a> 구간이 <a href="/wiki/%EB%B0%80%EB%9D%BC%EB%85%B8_%EC%A7%80%ED%95%98%EC%B2%A0_2%ED%98%B8%EC%84%A0" title="밀라노 지하철 2호선">밀라노 지하철 2호선</a>과 연결되었고 그 이후로 지금까지 지하철 2호선의 일부로 운영하고 있다.<sup class="reference" id="cite_ref-3"><a href="#cite_note-3">[3]</a></sup></p>
<h2><span class="mw-headline" id=".EC.97.AD_.EA.B5.AC.EC.A1.B0">역 구조</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%B6%80%EC%84%B8%EB%A1%9C_%EC%97%AD&amp;action=edit&amp;section=2" title="부분 편집: 역 구조">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>빌라 피오리타 역은 반투명 지붕이 달린 두 콘크리트 승강장과 두 선로로 되어 있다.<sup class="reference" id="cite_ref-4"><a href="#cite_note-4">[4]</a></sup> 지하철 역사는 승강장 북쪽 끝에 있다.<sup class="reference" id="cite_ref-5"><a href="#cite_note-5">[5]</a></sup> 역 전체를 조립재로 지었으며 그 구조는 같은 시기에 지어진 <a href="/wiki/%EC%B9%B4%EC%8B%9C%EB%82%98_%EB%B6%80%EB%A1%9C%EB%82%98_%EC%97%AD" title="카시나 부로나 역">카시나 부로나 역</a>, <a href="/wiki/%EB%B9%8C%EB%9D%BC_%ED%94%BC%EC%98%A4%EB%A6%AC%ED%83%80_%EC%97%AD" title="빌라 피오리타 역">빌라 피오리타 역</a>, <a href="/wiki/%EB%B9%8C%EB%9D%BC_%ED%8F%BC%ED%8E%98%EC%95%84_%EC%97%AD" title="빌라 폼페아 역">빌라 폼페아 역</a>과 동일하다.</p>
<h2><span class="mw-headline" id=".EC.8B.9C.EC.84.A4">시설</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%B6%80%EC%84%B8%EB%A1%9C_%EC%97%AD&amp;action=edit&amp;section=3" title="부분 편집: 시설">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
