<p><b>부여융</b>(扶餘隆, <a href="/wiki/615%EB%85%84" title="615년">615년</a> ~ <a href="/wiki/682%EB%85%84" title="682년">682년</a>)은 <a href="/wiki/%EB%B0%B1%EC%A0%9C" title="백제">백제</a> <a href="/wiki/%EC%9D%98%EC%9E%90%EC%99%95" title="의자왕">의자왕</a>의 <a href="/wiki/%EC%95%84%EB%93%A4" title="아들">아들</a>로 <a href="/wiki/%EC%82%BC%EA%B5%AD_%EC%8B%9C%EB%8C%80" title="삼국 시대">삼국</a> 말기의 대표적인 경계인이다. <a href="/wiki/%EC%9E%90_(%EC%9D%B4%EB%A6%84)" title="자 (이름)">자</a>는 <a href="/wiki/%EC%9D%B4%EB%A6%84" title="이름">이름</a>과 같은 융(隆)이다. <b>숭</b>(崇)이라고도 한다.<sup class="reference" id="cite_ref-1"><a href="#cite_note-1">[1]</a></sup> 백제 멸망 후 <a href="/wiki/%EB%8B%B9%EB%82%98%EB%9D%BC" title="당나라">당나라</a>에 <a class="mw-redirect" href="/wiki/%EA%B7%80%EC%88%9C" title="귀순">귀순</a>하여 <a href="/wiki/%EB%B0%B1%EC%A0%9C_%EB%B6%80%ED%9D%A5%EC%9A%B4%EB%8F%99" title="백제 부흥운동">백제 부흥운동</a> 토벌에 협조하고 백제군공 웅진도독으로 부임하여 백제 유민들을 위무하였다. 광록대부 태상원외경 겸 웅진도독 대방군왕까지 임명됐으나 <a href="/wiki/%EC%9B%85%EC%A7%84%EB%8F%84%EB%8F%85%EB%B6%80" title="웅진도독부">웅진도독부</a>는 <a href="/wiki/%EC%8B%A0%EB%9D%BC" title="신라">신라</a>에게 완전히 밀려나 <a class="mw-redirect" href="/wiki/%EC%9A%94%EB%8F%99" title="요동">요동</a>으로 옮길 수밖에 없었다. 마지막 <a class="mw-redirect" href="/wiki/%ED%83%9C%EC%9E%90" title="태자">태자</a>인지에 관해서는 학설 대립이 있다.</p>

<h3><span class="mw-headline" id=".EC.99.95.EC.9E.90_.EC.8B.9C.EC.A0.88">왕자 시절</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%B6%80%EC%97%AC%EC%9C%B5&amp;action=edit&amp;section=2" title="부분 편집: 왕자 시절">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>의자왕의 아들로 태어나 어려서부터 남다른 면이 있어 효심이 깊고 신중했으며 <a href="/wiki/%EC%84%A0_(%EC%B2%A0%ED%95%99)" title="선 (철학)">선</a>(善)과 <a href="/wiki/%EC%9D%98" title="의">의</a>(義)를 따랐다.<sup class="reference" id="cite_ref-.EC.B9.B4.EB.A5.BC_.EC.9C.B5_2-0"><a href="#cite_note-.EC.B9.B4.EB.A5.BC_.EC.9C.B5-2">[2]</a></sup> <a href="/wiki/637%EB%85%84" title="637년">637년</a>(<a href="/wiki/%EB%AC%B4%EC%99%95_(%EB%B0%B1%EC%A0%9C)" title="무왕 (백제)">무왕</a> 38년) <a href="/wiki/%EB%8B%B9_%ED%83%9C%EC%A2%85" title="당 태종">당 태종</a>에게 <a href="/wiki/%EC%A1%B0%EA%B3%B5" title="조공">조공</a> 사신으로 다녀왔다.<sup class="reference" id="cite_ref-3"><a href="#cite_note-3">[3]</a></sup> <a href="/wiki/644%EB%85%84" title="644년">644년</a>(의자왕 4년) 태자에 <a href="/wiki/%EC%B1%85%EB%B4%89" title="책봉">책봉</a>되었으나 의자왕 말년에 <a href="/wiki/%EB%B6%80%EC%97%AC%ED%9A%A8" title="부여효">부여효</a>로 교체되었다. <a href="/wiki/660%EB%85%84" title="660년">660년</a> 7월(<a <h3><span class="mw-headline" id=".EB.B0.B1.EC.A0.9C_.EB.B6.80.ED.9D.A5.EC.9A.B4.EB.8F.99_.ED.86.A0.EB.B2.8C">백제 부흥운동 토벌</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%B6%80%EC%97%AC%EC%9C%B5&amp;action=edit&amp;section=3" title="부분 편집: 백제 부흥운동 토벌">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>백제는 멸망했으나 <a href="/wiki/%EB%B0%B1%EC%A0%9C_%EB%B6%80%ED%9D%A5%EC%9A%B4%EB%8F%99" title="백제 부흥운동">백제 부흥운동</a>의 강성함으로 인해 <a href="/wiki/%EC%9B%85%EC%A7%84%EB%8F%84%EB%8F%85%EB%B6%80" title="웅진도독부">웅진도독부</a>가 고립되면서 <a href="/wiki/662%EB%85%84" title="662년">662년</a>(<a href="/wiki/%EC%9A%A9%EC%82%AD" title="용삭">용삭</a> 2년) 당 조정에서는 웅진<a href="/wiki/%EB%8F%84%EB%8F%85" title="도독">도독</a> <a class="new" href="/w/index.php?title=%EC%9C%A0%EC%9D%B8%EC%9B%90_(%EB%8B%B9)&amp;action=edit&amp;redlink=1" title="유인원 (당) (없는 문서)">유인원</a>(劉仁願)에게 철수 명령을 내렸다. 그러나 <a class="new" href="/w/index.php?title=%EC%9C%A0%EC%9D%B8%EA%B6%A4&amp;action=edit&amp;redlink=1" title="유인궤 (없는 문서)">유인궤</a>(劉仁軌)는 이곳을 지켜야 <a href="/wiki/%EA%B3%A0%EA%B5%AC%EB%A0%A4" title="고구려">고구려</a>도 무너트릴 수 있다며 강력히 반대했고 오히려 증원군을 요청했다.<sup class="reference" id="cite_ref-9"><a href="#cite_note-9">[9]</a></sup> 이에 <a href="/wiki/663%EB%85%84" title="663년">663년</a> 손인사(孫仁師)와 부여융이 7,000명을 이끌고 파견되었다.<sup class="reference" id="cite_ref-10"><a href="#cite_note-10">[10]</a></sup> 6월(음력)<sup class="reference" id="cite_ref-11"><a href="#cite_note-11">[11]</a></sup> 명장 <a class="mw-redirect" href="/wiki/%EB%B3%B5%EC%8B%A0" title="복신">복신</a>이 <a class="mw-redirect" href="/wiki/%EB%B6%80%EC%97%AC%ED%92%8D" title="부여풍">부여풍</a>에게 처형당하는 등 부흥군의 내부 분열을 포착한 나당연합군은 그들의 본거지인 <a href="/wiki/%EC%A3%BC%EB%A5%98%EC%84%B1" title="주류성">주류성</a>을 직공하기로 하였다. 유인원, 손인사, 문무왕은 <a href="/wiki/%EC%9C%A1%EA%B5%B0" title="육군">육군</a>을 통솔하고 유인궤, 두상(杜爽), 부여융은 <a href="/wiki/%EC%88%98%EA%B5%B0" title="수군">수군</a>과 <a class="mw-redirect" href="/wiki/%EA%B5%B0%EB%9F%89" title="군량">군량</a>선을 인솔하였다. 8월(음력) <a href="/wiki/%EB%B0%B1%EA%B0%95_%EC%A0%84%ED%88%AC" title="백강 전투">백강 전투</a>에서 <a href="/wiki/%EC%99%9C" title="왜">왜</a><a href="/wiki/%EC%84%A0%EB%B0%95" title="선박">선</a> 400척을 불사르는 대승을 거두고 주류성도 함락시킴으로써 백제부흥군을 완전히 토벌하였다.<sup class="reference" id="cite_ref-12"><a href="#cite_note-12">[12]</a></sup></p>
<h3><span class="mw-headline" id=".EB.B0.B1.EC.A0.9C_.EC.9C.A0.EB.AF.BC_.EC.9C.84.EB.AC.B4">백제 유민 위무</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%B6%80%EC%97%AC%EC%9C%B5&amp;action=edit&amp;section=4" title="부분 편집: 백제 유민 위무">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p><a href="/wiki/664%EB%85%84" title="664년">664년</a>(<a href="/wiki/%EC%9D%B8%EB%8D%95_(%EB%8B%B9)" title="인덕 (당)">인덕</a> 원년) 칙사 유인원의 주재로 신라의 <a href="/wiki/%EA%B9%80%EC%9D%B8%EB%AC%B8" title="김인문">김인문</a>·<a class="mw-redirect" href="/wiki/%EA%B9%80%EC%B2%9C%EC%A1%B4" title="김천존">김천존</a>과 화친을 맹약했다.<sup class="reference" id="cite_ref-13"><a href="#cite_note-13">[13]</a></sup> 10월(음력) 대방주<a href="/wiki/%EC%9E%90%EC%82%AC_(%EA%B4%80%EC%A7%81)" title="자사 (관직)">자사</a>(帶方州―) 겸 검교웅진도독 유인궤가 표를 올려 비로소 웅진도독이 되었고<sup class="reference" id="cite_ref-14"><a href="#cite_note-14">[14]</a></sup> 이내 백제군공에 봉해지고 웅진도총관 겸 마한도안무대사(馬韓道安撫大使)에 임명되어 백제 유민들을 위무하였다.<sup class="reference" id="cite_ref-.EC.B9.B4.EB.A5.BC_.EC.9C.B5_2-1"><a href="#cite_note-.EC.B9.B4.EB.A5.BC_.EC.9C.B5-2">[2]</a></sup> <a href="/wiki/665%EB%85%84" title="665년">665년</a> 유인원이 다시 칙사로 와서 신라 <a href="/wiki/%EB%AC%B8%EB%AC%B4%EC%99%95" title="문무왕">문무왕</a>과의 취리산(就利山) 회맹을 주선하니 드디어 백제의 옛 땅 일부에 대한 지배를 용인받고<sup class="reference" id="cite_ref-15"><a href="#cite_note-15">[15]</a></sup> 1도독부 7<a href="/wiki/%EC%A3%BC_(%ED%96%89%EC%A0%95_%EA%B5%AC%EC%97%AD)" title="주 (행정 구역)">주</a> 51<a href="/wiki/%ED%98%84_(%ED%96%89%EC%A0%95_%EA%B5%AC%EC%97%AD)" title="현 (행정 구역)">현</a>을 획정하였다.<sup class="reference" id="cite_ref-16"><a href="#cite_note-16">[16]</a></sup> 직후에 유인궤와 함께 당나라로 건너가서 <a href="/wiki/666%EB%85%84" title="666년">666년</a>(<a href="/wiki/%EA%B1%B4%EB%B4%89" title="건봉">건봉</a> 원년) 고종의 봉선(封禪)에도 참여하였다.</p>
<p>악화되던 나당관계가 고구려 멸망 후 표면화되면서 웅진도독부와 신라는 다시 격렬하게 <a href="/wiki/%ED%98%88%EC%95%A1" title="혈액">피</a>를 흘렸고 결국 <a href="/wiki/672%EB%85%84" title="672년">672년</a>(<a href="/wiki/%ED%95%A8%ED%98%95" title="함형">함형</a> 3년)에 축출되고 말았다.<sup class="reference" id="cite_ref-17"><a href="#cite_note-17">[17]</a></sup> 웅진도독부는 <a href="/wiki/676%EB%85%84" title="676년">676년</a>(<a href="/wiki/%EC%83%81%EC%9B%90_(%EB%8B%B9_%EA%B3%A0%EC%A2%85)" title="상원 (당 고종)">상원</a> 3년) 고구려의 성이었던 <a class="new" href="/w/index.php?title=%EA%B1%B4%EC%95%88%EC%84%B1&amp;action=edit&amp;redlink=1" title="건안성 (없는 문서)">건안성</a>(建安城)에 재설치되어 서주(徐州)와 연주(兗州) 등에 있던 백제 유민들도 옮겨갔다.<sup class="reference" id="cite_ref-18"><a href="#cite_note-18">[18]</a></sup> <a href="/wiki/677%EB%85%84" title="677년">677년</a>(<a href="/wiki/%EC%9D%98%EB%B4%89" title="의봉">의봉</a> 2년) <a class="mw-redirect" href="/wiki/%EB%82%98%EB%8B%B9%EC%A0%84%EC%9F%81" title="나당전쟁">나당전쟁</a>에서 당나라가 패배하여 부여융은 백제를 영영 잃고 다시는 옛 땅으로 돌아가지 못하였다. 대신 광록대부(光綠大夫) 태상원외경(太常員外卿) 사지절(使持節) 웅진도독 대방군왕을 임명받아 <a class="mw-redirect" href="/wiki/%EC%9A%94%EB%8F%99" title="요동">요동</a>에서 백제 유민들을 관리했다.<sup class="reference" id="cite_ref-19"><a href="#cite_note-19">[19]</a></sup> <a href="/wiki/682%EB%85%84" title="682년">682년</a>(<a href="/wiki/%EC%98%81%EC%88%9C_(%EB%8B%B9)" title="영순 (당)">영순</a> 원년) 68세의 <a href="/wiki/%EB%82%98%EC%9D%B4" title="나이">나이</a>에 뤄양에서 병사해 <a class="new" href="/w/index.php?title=%EB%B6%81%EB%A7%9D%EC%82%B0&amp;action=edit&amp;redlink=1" title="북망산 (없는 문서)">북망산</a>에 묻혔고 보국대장군(輔國大將軍)을 <a href="/wiki/%EC%B6%94%EC%A6%9D" title="추증">추증</a>받았다.<sup class="reference" id="cite_ref-.EC.B9.B4.EB.A5.BC_.EC.9C.B5_2-2"><a href="#cite_note-.EC.B9.B4.EB.A5.BC_.EC.9C.B5-2">[2]</a></sup></p>
<h3><span class="mw-headline" id=".EC.82.AC.ED.9B.84">사후</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%B6%80%EC%97%AC%EC%9C%B5&amp;action=edit&amp;section=5" title="부분 편집: 사후">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>지위는 손자 <a href="/wiki/%EB%B6%80%EC%97%AC%EA%B2%BD" title="부여경">부여경</a>이 물려받았다. <a href="/wiki/1920%EB%85%84" title="1920년">1920년</a> 뤄양에서 〈부여융묘지명〉을 발견하여 현재 중국 하남성 박물관에서 소장하고 있다. 이 인연으로 뤄양시와 <a href="/wiki/%EB%B6%80%EC%97%AC%EA%B5%B0" title="부여군">부여군</a>은 자매결연을 맺고 의자왕의 <a class="mw-redirect" href="/wiki/%EB%AC%98" title="묘">묘</a> 수색 작업을 벌였다. 그러나 끝내 찾지 못하고 그 <a href="/wiki/%ED%9D%99" title="흙">흙</a>만 가져와서 <a href="/wiki/%EB%B6%80%EC%97%AC_%EB%8A%A5%EC%82%B0%EB%A6%AC_%EA%B3%A0%EB%B6%84%EA%B5%B0" title="부여 능산리 고분군">부여 능산리 고분군</a>에 <a href="/wiki/%EC%9D%98%EC%9E%90%EC%99%95" title="의자왕">아버지</a>와 나란히 가묘(假墓)를 조성하였다.</p>
<h2><span class="mw-headline" id=".EB.AF.B8.EA.B6.81.EC.9D.98_.EB.A7.88.EC.A7.80.EB.A7.89_.ED.83.9C.EC.9E.90">미궁의 마지막 태자</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%B6%80%EC%97%AC%EC%9C%B5&amp;action=edit&amp;section=6" title="부분 편집: 미궁의 마지막 태자">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p><a href="/wiki/%EC%A4%91%EA%B5%AD" title="중국">중국</a>의 기록은 일관되게 부여융이 <a class="mw-redirect" href="/wiki/%ED%83%9C%EC%9E%90" title="태자">태자</a>이며 맏아들이라고 했으나 《<a href="/wiki/%EC%82%BC%EA%B5%AD%EC%82%AC%EA%B8%B0" title="삼국사기">삼국사기</a>》에서는 <a href="/wiki/%EB%B6%80%EC%97%AC%ED%9A%A8" title="부여효">부여효</a>가 태자이며 맏아들이라고 하여 서로 상반된다. 그래서 크게 3가지 설이 대립하고 있다.</p>
<h3><span class="mw-headline" id=".EC.9C.B5.EC.9D.B4_.ED.83.9C.EC.9E.90.EC.9D.B4.EB.A9.B0_.EB.A7.8F.EC.95.84.EB.93.A4.EC.84.A4">융이 태자이며 맏아들설</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%B6%80%EC%97%AC%EC%9C%B5&amp;action=edit&amp;section=7" title="부분 편집: 융이 태자이며 맏아들설">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>중국의 각종 기록과 《<a href="/wiki/%EC%82%BC%EA%B5%AD%EC%9C%A0%EC%82%AC" title="삼국유사">삼국유사</a>》<sup class="reference" id="cite_ref-20"><a href="#cite_note-20">[20]</a></sup>를 우선한 설이다. 무엇보다 〈<a class="mw-redirect" href="/wiki/%EB%8C%80%EB%8B%B9%ED%8F%89%EB%B0%B1%EC%A0%9C%EA%B5%AD%EB%B9%84" title="대당평백제국비">대당평백제국비</a>〉와 〈<a class="mw-redirect" href="/wiki/%EB%8B%B9%EC%9C%A0%EC%9D%B8%EC%9B%90%EA%B8%B0%EA%B3%B5%EB%B9%84" title="당유인원기공비">당유인원기공비</a>〉가 당대의 기록이기 때문에 신뢰성이 높다. 물론 승자의 기록이기는 하지만 《삼국사기》 역시 승자의 기록을 기반으로 했을 뿐더러 시기는 오히려 <a href="/wiki/12%EC%84%B8%EA%B8%B0" title="12세기">12세기</a>로 늦다. 그러므로 《삼국사기》가 오류라고 한다. 부여융이 615년생이라는 점도 강점이다. 정리하면 융이 644년 태자에 책봉된 이래 그 자리는 바뀌지 않았다고 한다.<sup class="reference" id="cite_ref-21"><a href="#cite_note-21">[21]</a></sup><sup class="reference" id="cite_ref-.EC.96.91.EA.B0.80_.EC.95.84.EC.A0.80.EC.94.A8_22-0"><a href="#cite_note-.EC.96.91.EA.B0.80_.EC.95.84.EC.A0.80.EC.94.A8-22">[22]</a></sup></p>
<h3><span class="mw-headline" id=".ED.9A.A8.EA.B0.80_.ED.83.9C.EC.9E.90.EC.9D.B4.EB.A9.B0_.EB.A7.8F.EC.95.84.EB.93.A4.EC.84.A4">효가 태자이며 맏아들설</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%B6%80%EC%97%AC%EC%9C%B5&amp;action=edit&amp;section=8" title="부분 편집: 효가 태자이며 맏아들설">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>《삼국사기》를 우선한 설이다. 《삼국사기》가 <a href="/wiki/%EC%8B%A0%EB%9D%BC" title="신라">신라</a>사를 제외하고는 상당수 중국의 기록을 원전으로 했음에도 굳이 태자를 융에서 효로 고치고 <a href="/wiki/%EB%B6%80%EC%97%AC%EB%AC%B8%EC%82%AC" title="부여문사">부여문사</a>가 사비성에서 얘기하는 상대도 부여융으로 설정했다. 융의 항복 시기도 의자왕보다 앞서며 〈김인문전〉에서도 태자를 효라 하는<sup class="reference" id="cite_ref-23"><a href="#cite_note-23">[23]</a></sup> 등의 꼼꼼함이 보인다. 중국의 기록은 태자가 누구인지 잘 몰랐거나 효를 태자로 인정하지 않았기 때문이라고 한다. 융의 태자 책봉 기사에 대한 설명은 다시 두 가지 설로 나뉜다.</p>
<ol>
<li>융의 태자 책봉 기사는 효의 오기로 처음부터 끝까지 효가 태자였다.<sup class="reference" id="cite_ref-24"><a href="#cite_note-24">[24]</a></sup></li>
<li>644년 융이 태자로 책봉되기는 하였으나 의자왕 말년에 정국 변동이 일어나 효로 교체되었다.<sup class="reference" id="cite_ref-25"><a href="#cite_note-25">[25]</a></sup></li>
</ol>
<h3><span class="mw-headline" id=".ED.9A.A8.EA.B0.80_.ED.83.9C.EC.9E.90.EC.9D.B4.EB.A9.B0_.EC.82.BC.EB.82.A8_.EC.9D.B4.ED.95.98.EC.84.A4">효가 태자이며 삼남 이하설</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%B6%80%EC%97%AC%EC%9C%B5&amp;action=edit&amp;section=9" title="부분 편집: 효가 태자이며 삼남 이하설">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>효로의 태자 교체설을 기본으로 하면서도 융 태자설과 일부 부합하는 면이 있다. 즉 융이 첫째였기에 644년 태자로 책봉됐지만 의자왕 말년에 효로 교체되었다고 한다.<sup class="reference" id="cite_ref-26"><a href="#cite_note-26">[26]</a></sup></p>
<h2><span class="mw-headline" id=".ED.8F.89.EA.B0.80">평가</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%B6%80%EC%97%AC%EC%9C%B5&amp;action=edit&amp;section=10" title="부분 편집: 평가">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>부여융의 행위를 <a href="/wiki/%EB%B0%B1%EC%A0%9C_%EB%B6%80%ED%9D%A5%EC%9A%B4%EB%8F%99" title="백제 부흥운동">백제 부흥운동</a>의 한 일환으로 보기도 한다. 냉철한 현실 인식 하에서 <a href="/wiki/%EB%8B%B9%EB%82%98%EB%9D%BC" title="당나라">당나라</a>를 등에 업고 <a href="/wiki/%EB%B0%B1%EC%A0%9C" title="백제">백제</a>를 다시 일으켜 세우려 했다는 것이다. 부흥운동에 참여한 자들은 부여융과의 <a href="/wiki/%EC%A0%95%EC%B9%98" title="정치">정치</a> 세력이 달랐거나<sup class="reference" id="cite_ref-.EC.96.91.EA.B0.80_.EC.95.84.EC.A0.80.EC.94.A8_22-1"><a href="#cite_note-.EC.96.91.EA.B0.80_.EC.95.84.EC.A0.80.EC.94.A8-22">[22]</a></sup> 웅진도독부를 당의 지배기구로 인식했던 것으로 보인다.<sup class="reference" id="cite_ref-27"><a href="#cite_note-27">[27]</a></sup> 부여융은 그저 당나라의 지배정책을 실행한 것이라는 시각도 있다.<sup class="reference" id="cite_ref-28"><a href="#cite_note-28">[28]</a></sup> 이러한 일련의 평가는 <a href="/wiki/%EC%9B%85%EC%A7%84%EB%8F%84%EB%8F%85%EB%B6%80" title="웅진도독부">웅진도독부</a>의 성격 규명과도 관련이 있다.</p>
