<h1 class="firstHeading" id="firstHeading" lang="ko">부천 필하모닉 오케스트라</h1>
<div class="mw-body-content" id="bodyContent">
<div id="siteSub">위키백과, 우리 모두의 백과사전.</div>
<div id="contentSub"></div>
<div class="mw-jump" id="jump-to-nav">
					이동:					<a href="#mw-head">둘러보기</a>, 					<a href="#p-search">검색</a>
</div>
<div class="mw-content-ltr" dir="ltr" id="mw-content-text" lang="ko"><script>function mfTempOpenSection(id){var block=document.getElementById("mf-section-"+id);block.className+=" open-block";block.previousSibling.className+=" open-block";}</script><p><b>부천 필하모닉 오케스트라</b>(Bucheon Philharmonic Orchestra)는 <a href="/wiki/%EB%8C%80%ED%95%9C%EB%AF%BC%EA%B5%AD" title="대한민국">대한민국</a>의 대표적인 <a href="/wiki/%EA%B4%80%ED%98%84%EC%95%85%EB%8B%A8" title="관현악단">관현악단</a> 중 하나로, <a href="/wiki/%EA%B2%BD%EA%B8%B0%EB%8F%84" title="경기도">경기도</a> <a href="/wiki/%EB%B6%80%EC%B2%9C%EC%8B%9C" title="부천시">부천시</a>에서 운영하고 있다. <a href="/wiki/1988%EB%85%84" title="1988년">1988년</a>에 <a class="new" href="/w/index.php?title=%EB%B6%80%EC%B2%9C%EC%8B%9C%EB%A6%BD%EC%98%88%EC%88%A0%EB%8B%A8&amp;action=edit&amp;redlink=1" title="부천시립예술단 (없는 문서)">부천시립예술단</a>의 직속 단체로 창단되었으며, 이듬해 <a href="/wiki/%EC%9E%84%ED%97%8C%EC%A0%95" title="임헌정">임헌정</a>이 초대 상임지휘자에 취임했다.</p>
<p>창단 후 <a href="/wiki/%EC%95%84%EB%A5%B4%EB%86%80%ED%8A%B8_%EC%87%A4%EB%B2%A0%EB%A5%B4%ED%81%AC" title="아르놀트 쇤베르크">쇤베르크</a>와 <a href="/wiki/%EC%95%8C%EB%B0%98_%EB%B2%A0%EB%A5%B4%ED%81%AC" title="알반 베르크">베르크</a>, <a href="/wiki/%EB%B2%84%EB%A5%B4%ED%86%A0%ED%81%AC_%EB%B2%A8%EB%9F%AC" title="버르토크 벨러">버르토크</a> 등의 20세기 관현악 작품 연주회, <a href="/wiki/%EB%A3%A8%ED%8A%B8%EB%B9%84%ED%9E%88_%ED%8C%90_%EB%B2%A0%ED%86%A0%EB%B2%A4" title="루트비히 판 베토벤">베토벤</a>과 <a href="/wiki/%EC%9A%94%ED%95%98%EB%84%A4%EC%8A%A4_%EB%B8%8C%EB%9E%8C%EC%8A%A4" title="요하네스 브람스">브람스</a>의 교향곡 전곡 연주 시리즈 등으로 화제가 되었다.</p>
<p><a href="/wiki/1999%EB%85%84" title="1999년">1999년</a>에는 국내 관현악단으로는 최초로 <a href="/wiki/%EA%B5%AC%EC%8A%A4%ED%83%80%ED%94%84_%EB%A7%90%EB%9F%AC" title="구스타프 말러">말러</a>의 교향곡 전곡 연주를 시작하여 2003년 완료해, 한국에 '말러 붐'을 일으켰다는 평가를 받기도 했다.</p>
<p>2002년 9월에는 <a class="new" href="/w/index.php?title=%EC%95%84%EC%8B%9C%EC%95%84_%EC%98%A4%EC%BC%80%EC%8A%A4%ED%8A%B8%EB%9D%BC_%EC%9C%84%ED%81%AC&amp;action=edit&amp;redlink=1" title="아시아 오케스트라 위크 (없는 문서)">아시아 오케스트라 위크</a>(Asia Orchestra Week)에 한국 대표 관현악단으로 참가했으며, 2007년부터 <a href="/wiki/%EC%95%88%ED%86%A4_%EB%B8%8C%EB%A3%A8%ED%81%AC%EB%84%88" title="안톤 브루크너">브루크너</a> 교향곡 전곡 연주회를 3~4개월 간격으로 개최하고 있다. 또한 <a href="/wiki/%EC%98%88%EC%88%A0%EC%9D%98_%EC%A0%84%EB%8B%B9" title="예술의 전당">예술의 전당</a>에서 주최하는 <a href="/wiki/%EA%B5%90%ED%96%A5%EC%95%85%EC%B6%95%EC%A0%9C" title="교향악축제">교향악축제</a>에도 해마다 참가하고 있다.</p>
<p>2009년에는 <a href="/wiki/%EC%9C%A0%EB%8B%88%EB%B2%84%EC%84%A4_%EB%AE%A4%EC%A7%81_%EA%B7%B8%EB%A3%B9" title="유니버설 뮤직 그룹">유니버설 뮤직 그룹</a> 한국지사를 통해 브람스의 <a href="/wiki/%EA%B5%90%ED%96%A5%EA%B3%A1_1%EB%B2%88_(%EB%B8%8C%EB%9E%8C%EC%8A%A4)" title="교향곡 1번 (브람스)">교향곡 1번</a>을 CD로 발매했다.</p>
<p>상임지휘자 외에도 <a href="/wiki/%EC%A0%95%EC%B9%98%EC%9A%A9" title="정치용">정치용</a>, <a href="/wiki/%EA%B8%88%EB%85%B8%EC%83%81" title="금노상">금노상</a>, <a class="mw-disambig" href="/wiki/%EA%B9%80%EC%A2%85%EB%8D%95" title="김종덕">김종덕</a>, <a class="new" href="/w/index.php?title=%EA%B9%80%EB%8D%95%EA%B8%B0_(%EC%A7%80%ED%9C%98%EC%9E%90)&amp;action=edit&amp;redlink=1" title="김덕기 (지휘자) (없는 문서)">김덕기</a> 등 한국 지휘자들 뿐 아니라 필립 베르놀드(Philippe Bernold), 클라우스 아르프(Klaus Arp), 테오 월터스(Theo Wolters) 등 외국 지휘자들과도 함께한 바 있다.</p>
<p>주공연장은 부천 시민회관 대공연장이나, 예술의 전당 콘서트홀 등 <a href="/wiki/%EC%84%9C%EC%9A%B8%ED%8A%B9%EB%B3%84%EC%8B%9C" title="서울특별시">서울</a> 소재 공연장에서의 연주회도 많이 열린다.</p>
<p>2005년에는 부천시립예술단 조직 개편과 함께 <a class="new" href="/w/index.php?title=%EB%B6%80%EC%B2%9C%EC%8B%9C%EB%A6%BD%ED%95%A9%EC%B0%BD%EB%8B%A8&amp;action=edit&amp;redlink=1" title="부천시립합창단 (없는 문서)">부천시립합창단</a>이 '부천 필하모닉 오케스트라 코러스'로 개명하여 오케스트라와 통합 편성되었으나, 2013년 7월에 다시 부천시립합창단으로 이름을 바꾸고 분리하였다.</p>
