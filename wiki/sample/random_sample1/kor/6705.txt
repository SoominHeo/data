<h1 class="firstHeading" id="firstHeading" lang="ko">부쿠레슈티 조약 (1918년)</h1>
<div class="mw-body-content" id="bodyContent">
<div id="siteSub">위키백과, 우리 모두의 백과사전.</div>
<div id="contentSub"></div>

<a href="/wiki/%ED%94%84%EB%9E%91%EC%8A%A4" title="프랑스">프랑스</a> 측이 그린 부쿠레슈티 강화 조약 포스터이다. 그림에서는 독일황제 <a href="/wiki/%EB%B9%8C%ED%97%AC%EB%A6%84_2%EC%84%B8" title="빌헬름 2세">빌헬름 2세</a>가 루마니아라고 하는 여성을 칼로 들이대고 있고, 러시아라고 하는 남자는 그녀가 평화 조약을 맺는 동안 빌헬름 2세에게 흉부가 짓밟혀져 있는 것으로 묘사되어 있다. 즉, 독일이 루마니아에게 조약을 맺으라고 강요한 것이다.</div>
</div>
</div>
<p><b>부쿠레슈티 강화 조약</b>은 <a href="/wiki/1918%EB%85%84" title="1918년">1918년</a> <a href="/wiki/5%EC%9B%94_7%EC%9D%BC" title="5월 7일">5월 7일</a>, <a href="/wiki/%EB%A3%A8%EB%A7%88%EB%8B%88%EC%95%84" title="루마니아">루마니아</a>의 <a href="/wiki/%EB%B6%80%ED%94%84%ED%85%8C%EC%95%84" title="부프테아">부프테아</a>에서 맺어진 <a href="/wiki/%EB%A3%A8%EB%A7%88%EB%8B%88%EC%95%84_%EC%99%95%EA%B5%AD" title="루마니아 왕국">루마니아</a>와 <a href="/wiki/%EB%8F%85%EC%9D%BC_%EC%A0%9C%EA%B5%AD" title="독일 제국">독일</a>, <a href="/wiki/%EC%98%A4%EC%8A%A4%ED%8A%B8%EB%A6%AC%EC%95%84-%ED%97%9D%EA%B0%80%EB%A6%AC_%EC%A0%9C%EA%B5%AD" title="오스트리아-헝가리 제국">오스트리아-헝가리 제국</a>, <a href="/wiki/%EB%B6%88%EA%B0%80%EB%A6%AC%EC%95%84_%EC%99%95%EA%B5%AD" title="불가리아 왕국">불가리아</a>, <a href="/wiki/%EC%98%A4%EC%8A%A4%EB%A7%8C_%EC%A0%9C%EA%B5%AD" title="오스만 제국">오스만 제국</a> 간에 맺어진 조약이다. 이 조약으로 루마니아는 동맹국에 유일하게 항복한 나라로 기록되었다. 정작 당시 루마니아의 왕이었던 <a href="/wiki/%ED%8E%98%EB%A5%B4%EB%94%94%EB%82%9C%EB%93%9C_1%EC%84%B8_(%EB%A3%A8%EB%A7%88%EB%8B%88%EC%95%84)" title="페르디난드 1세 (루마니아)">페르디난드 1세</a>는 조약에 참가하지도 않았고, 서명하지도 않았다고 한다(이는 <a href="/wiki/%EB%8C%80%ED%95%9C%EC%A0%9C%EA%B5%AD" title="대한제국">대한제국</a>의 <a class="mw-redirect" href="/wiki/%EA%B3%A0%EC%A2%85%ED%99%A9%EC%A0%9C" title="고종황제">고종황제</a>가 <a class="mw-redirect" href="/wiki/%EC%9D%84%EC%82%AC%EB%8A%91%EC%95%BD" title="을사늑약">을사늑약</a> 서명을 거부한 것과도 같다). 이 조약은 <a href="/wiki/%EB%B2%A0%EB%A5%B4%EC%82%AC%EC%9C%A0_%EC%A1%B0%EC%95%BD" title="베르사유 조약">베르사유 조약</a>으로 인해 파기되었으며 동맹국에 빼앗겼던 영토도 수복했다.</p>
<h2><span class="mw-headline" id=".EC.A3.BC.EC.9A.94_.EB.82.B4.EC.9A.A9">주요 내용</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%B6%80%EC%BF%A0%EB%A0%88%EC%8A%88%ED%8B%B0_%EC%A1%B0%EC%95%BD_(1918%EB%85%84)&amp;action=edit&amp;section=1" title="부분 편집: 주요 내용">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>주요 4개 조항은 다음과 같다.<br/></p>
<ul>
<li>루마니아는 남부 도브루자(카드릴라테르)를 불가리아에 할양하고, 북도브루자를 불가리아에 할양하며, 다른 지역은 동맹국의 관리 아래에 둔다.</li>
<li>카르파티아 산맥의 통로를 오스트리아-헝가리 제국의 관리 하에 둔다.</li>
<li>루마니아는 90년간 독일에게 유전을 빌려준다.</li>
<li>동맹국은 루마니아의 <a href="/wiki/%EB%B2%A0%EC%82%AC%EB%9D%BC%EB%B9%84%EC%95%84" title="베사라비아">베사라비아</a> 합병을 인정한다.</li>
</ul>
