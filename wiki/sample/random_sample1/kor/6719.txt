<p><b>북버지니아군</b> (Army of Northern Virginia)은 <a class="mw-disambig" href="/wiki/%EB%82%A8%EB%B6%81_%EC%A0%84%EC%9F%81" title="남북 전쟁">남북 전쟁</a> 당시 주로 <a href="/wiki/%EB%8F%99%EB%B6%80_%EC%A0%84%EC%84%A0_(%EB%82%A8%EB%B6%81_%EC%A0%84%EC%9F%81)" title="동부 전선 (남북 전쟁)">동부 전선</a>에서 주로 활약한 <a class="mw-redirect" href="/wiki/%EB%82%A8%EB%B6%80%EB%8F%99%EB%A7%B9" title="남부동맹">남부동맹</a>의 군부대다. <a href="/wiki/%EB%B6%81%EA%B5%B0" title="북군">북군</a>의 <a class="mw-redirect" href="/wiki/%ED%8F%AC%ED%86%A0%EB%A7%A5%EA%B5%B0" title="포토맥군">포토맥군</a>과 자주 전투를 치렀다.</p>

<h2><span class="mw-headline" id=".EA.B8.B0.EC.9B.90">기원</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%B6%81%EB%B2%84%EC%A7%80%EB%8B%88%EC%95%84_%EA%B5%B0&amp;action=edit&amp;section=1" title="부분 편집: 기원">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p><b>북버지니아군</b>이란 이름은 다른 <a class="new" href="/w/index.php?title=%EB%82%A8%EB%B6%80%EB%8F%99%EB%A7%B9%EA%B5%B0&amp;action=edit&amp;redlink=1" title="남부동맹군 (없는 문서)">남부동맹군</a> 부대들이 그랬던 것처럼 주 작전 지역을 나타내는 것이다. 북버지니아군은 북버지니아의 모든 전투 가능한 부대를 통합하여 <a href="/wiki/1861%EB%85%84" title="1861년">1861년</a> <a href="/wiki/6%EC%9B%94_20%EC%9D%BC" title="6월 20일">6월 20일</a>에 창설된 <a class="new" href="/w/index.php?title=%ED%8F%AC%ED%86%A0%EB%A7%A5%EA%B5%B0_(%EB%82%A8%EA%B5%B0)&amp;action=edit&amp;redlink=1" title="포토맥군 (남군) (없는 문서)">포토맥군 (남군)</a>이 기원이다. <a href="/wiki/7%EC%9B%94_20%EC%9D%BC" title="7월 20일">7월 20일</a>과 <a href="/wiki/7%EC%9B%94_21%EC%9D%BC" title="7월 21일">7월 21일</a>, <a class="new" href="/w/index.php?title=%EC%85%B0%EB%84%8C%EB%8F%84%EC%96%B4_%EA%B3%84%EA%B3%A1%EA%B5%B0&amp;action=edit&amp;redlink=1" title="셰넌도어 계곡군 (없는 문서)">셰넌도어 계곡군</a>과 하퍼스페리 군관구에서 온 부대가 추가되었다. <a class="new" href="/w/index.php?title=%EB%B6%81%EC%84%9C%EA%B5%B0&amp;action=edit&amp;redlink=1" title="북서군 (없는 문서)">북서군</a>이 <a href="/wiki/1862%EB%85%84" title="1862년">1862년</a> <a href="/wiki/3%EC%9B%94_14%EC%9D%BC" title="3월 14일">3월 14일</a>에 합쳐졌고, 그날 북버지니아군으로 명칭을 변경했다. 같은 해 <a href="/wiki/4%EC%9B%94_12%EC%9D%BC" title="4월 12일">4월 12일</a>에 <a class="new" href="/w/index.php?title=%ED%8E%98%EB%8B%8C%EC%84%A4%EB%9D%BC_%EA%B5%B0&amp;action=edit&amp;redlink=1" title="페닌설라 군 (없는 문서)">페닌설라 군</a>(Army of the Peninsula)이 통합되었다<sup class="reference" id="cite_ref-e889_1-0"><a href="#cite_note-e889-1">[1]</a></sup></p>
<p>버지니아인들 외에 남부동맹을 통틀어서 멀리서 온 <a href="/wiki/%ED%85%8D%EC%82%AC%EC%8A%A4_%EC%A3%BC" title="텍사스 주">텍사스 주</a>나 <a href="/wiki/%EC%95%84%EC%B9%B8%EC%86%8C_%EC%A3%BC" title="아칸소 주">아칸소 주</a>같은 먼 곳에서 온 부대도 북버지니아군 소속으로 싸웠으며, 심지어 <a href="/wiki/%EB%89%B4%EB%A9%95%EC%8B%9C%EC%BD%94_%EC%A4%80%EC%A3%BC" title="뉴멕시코 준주">뉴멕시코 준주</a>나 <a href="/wiki/%EC%95%A0%EB%A6%AC%EC%A1%B0%EB%82%98_%EC%A4%80%EC%A3%BC" title="애리조나 준주">애리조나 준주</a>에서 온 <a class="mw-redirect" href="/wiki/%EB%AF%BC%EB%B3%91%EB%8C%80" title="민병대">민병대</a>도 북버지나이군에 속해 있었다. 북버지니아군은 남군의 실질적인 주력이었다.</p>
<h2><span class="mw-headline" id=".EC.A7.80.ED.9C.98_.EC.B2.B4.EA.B3.84">지휘 체계</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%B6%81%EB%B2%84%EC%A7%80%EB%8B%88%EC%95%84_%EA%B5%B0&amp;action=edit&amp;section=2" title="부분 편집: 지휘 체계">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>북버지니아군의 초대 지휘관은 <a href="/wiki/%ED%94%BC%EC%97%90%EB%A5%B4_%EB%B3%B4%EC%9A%B0%EB%A6%AC%EA%B0%80%EB%93%9C" title="피에르 보우리가드">피에르 보우리가드</a>였다. 그가 지휘했을 당시 부대 이름은 포토맥군이었다. 이후 지휘권은 <a class="new" href="/w/index.php?title=%EC%A1%B0%EC%A7%80%ED%94%84_%EC%A1%B4%EC%8A%A4%ED%84%B4&amp;action=edit&amp;redlink=1" title="조지프 존스턴 (없는 문서)">조지프 존스턴</a>으로 넘어갔고, 짧은 기간 동안 <a class="new" href="/w/index.php?title=%EA%B5%AC%EC%8A%A4%ED%83%80%EB%B6%80%EC%8A%A4_%EC%8A%A4%EB%AF%B8%EC%8A%A4&amp;action=edit&amp;redlink=1" title="구스타부스 스미스 (없는 문서)">구스타부스 스미스</a>가 지휘했다. 그러나 <a href="/wiki/1862%EB%85%84" title="1862년">1862년</a> <a href="/wiki/6%EC%9B%94_11%EC%9D%BC" title="6월 11일">6월 11일</a>에 가장 유명하고 마지막 지휘괸인 <a href="/wiki/%EB%A1%9C%EB%B2%84%ED%8A%B8_E._%EB%A6%AC" title="로버트 E. 리">로버트 E. 리</a> 장군이 부상당한 존스턴의 뒤를 이어 북버지니아군의 지휘권을 인수했다. 스미스는 <a class="new" href="/w/index.php?title=%EC%84%B8%EB%B8%90_%ED%8C%8C%EC%9D%B8%EC%8A%A4_%EC%A0%84%ED%88%AC&amp;action=edit&amp;redlink=1" title="세븐 파인스 전투 (없는 문서)">세븐 파인스 전투</a>에서 신경쇠약으로 걸린 것 같은 고통을 겪었다.</p>
<p>지휘를 맡은 첫해에 리는 2명의 주요 차급 지휘관을 거느렸다. 우익은 <a href="/wiki/%EC%A0%9C%EC%9E%84%EC%8A%A4_%EB%A1%B1%EC%8A%A4%ED%8A%B8%EB%A6%AC%ED%8A%B8" title="제임스 롱스트리트">제임스 롱스트리트</a> 중장이었고, 좌익은 <a href="/wiki/%EC%8A%A4%ED%86%A4%EC%9B%94_%EC%9E%AD%EC%8A%A8" title="스톤월 잭슨">스톤월 잭슨</a> 중장의 지휘를 받고 있었다. 이 날개들은 <a href="/wiki/1862%EB%85%84" title="1862년">1862년</a> <a href="/wiki/11%EC%9B%94_6%EC%9D%BC" title="11월 6일">11월 6일</a>, 1군단(롱스트리트)과 2군단 (잭슨)으로 재편성되었다. <a class="mw-redirect" href="/wiki/%EC%B1%88%EC%8A%AC%EB%9F%AC%EC%8A%A4%EB%B9%8C_%EC%A0%84%ED%88%AC" title="챈슬러스빌 전투">챈슬러스빌 전투</a> 후 잭슨이 사망하자 리는 <a href="/wiki/1863%EB%85%84" title="1863년">1863년</a> <a href="/wiki/5%EC%9B%94_30%EC%9D%BC" title="5월 30일">5월 30일</a>에 북버지니아군을 3개 군단으로 재편성했다. 롱스트리트는 여전히 1군단을 지휘했고, 잭슨의 2군단이 2개 군단으로 나뉘어 각기 <a href="/wiki/%EB%A6%AC%EC%B2%98%EB%93%9C_S._%EC%9D%B4%EC%9B%94" title="리처드 S. 이월">리처드 S. 이월</a> 중장과 <a class="new" href="/w/index.php?title=A.P_%ED%9E%90&amp;action=edit&amp;redlink=1" title="A.P 힐 (없는 문서)">A.P 힐</a> 중장이 지휘하게 되었다. <a class="new" href="/w/index.php?title=%EB%A6%AC%EC%B2%98%EB%93%9C_H._%EC%95%A4%EB%8D%94%EC%8A%A8&amp;action=edit&amp;redlink=1" title="리처드 H. 앤더슨 (없는 문서)">리처드 H. 앤더슨</a>의 4군단이 <a href="/wiki/1864%EB%85%84" title="1864년">1864년</a> <a href="/wiki/10%EC%9B%94_19%EC%9D%BC" title="10월 19일">10월 19일</a> 편성되었고, <a href="/wiki/1865%EB%85%84" title="1865년">1865년</a> <a href="/wiki/4%EC%9B%94_8%EC%9D%BC" title="4월 8일">4월 8일</a> 4군단은 2군단에 흡수/통합되었다. 3개 군단의 지휘관들은 1864년과 1865년에 자주 바뀌었다. 기병군단은 <a href="/wiki/%EC%A0%AD_%EC%8A%A4%ED%8A%9C%EC%96%B4%ED%8A%B8" title="젭 스튜어트">젭 스튜어트</a>가 지휘했다. 기병군단은 <a href="/wiki/1862%EB%85%84" title="1862년">1862년</a> <a href="/wiki/8%EC%9B%94_17%EC%9D%BC" title="8월 17일">8월 17일</a> 창설되었고, <a href="/wiki/1864%EB%85%84" title="1864년">1864년</a> <a href="/wiki/5%EC%9B%94_11%EC%9D%BC" title="5월 11일">5월 11일</a>, 스튜어트가 치명적인 부상을 입은 날 해체되었다. 예비 포병대는 <a class="new" href="/w/index.php?title=%EC%9C%8C%EB%A6%AC%EC%97%84_%ED%8E%9C%EB%93%A4%ED%84%B4&amp;action=edit&amp;redlink=1" title="윌리엄 펜들턴 (없는 문서)">윌리엄 펜들턴</a> 준장이 지휘했다.<sup class="reference" id="cite_ref-e889_1-1"><a href="#cite_note-e889-1">[1]</a></sup></p>

<p><a href="/wiki/1865%EB%85%84" title="1865년">1865년</a> <a href="/wiki/4%EC%9B%94_9%EC%9D%BC" title="4월 9일">4월 9일</a> 북버지니아군은 <a class="mw-redirect" href="/wiki/%EC%95%A0%ED%8F%AC%EB%A7%A4%ED%86%A1%EC%8A%A4_%EC%BD%94%ED%8A%B8_%ED%95%98%EC%9A%B0%EC%8A%A4" title="애포매톡스 코트 하우스">애포매톡스 코트 하우스</a>에서 <a class="mw-redirect" href="/wiki/%ED%8F%AC%ED%86%A0%EB%A7%A5%EA%B5%B0" title="포토맥군">포토맥군</a>에 항복했다. 리 장군이 <a class="mw-redirect" href="/wiki/%EC%9C%A8%EB%A6%AC%EC%8B%9C%EC%8A%A4_%EA%B7%B8%EB%9E%9C%ED%8A%B8" title="율리시스 그랜트">율리시스 그랜트</a> 장군에게 항복 문서에 서명함으로써 실질적으로 전쟁은 이 날 끝났다.</p>
