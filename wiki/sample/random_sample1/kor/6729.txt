<p><b>북위</b>(北魏, <a href="/wiki/386%EB%85%84" title="386년">386년</a>~<a href="/wiki/534%EB%85%84" title="534년">534년</a>)는 <a href="/wiki/%EC%A4%91%EA%B5%AD" title="중국">중국</a> <a href="/wiki/%EC%98%A4%ED%98%B8_%EC%8B%AD%EC%9C%A1%EA%B5%AD_%EC%8B%9C%EB%8C%80" title="오호 십육국 시대">오호 십육국 시대</a>에 <a class="mw-redirect" href="/wiki/%EC%84%A0%EB%B9%84%EC%A1%B1" title="선비족">선비족</a> <a class="mw-redirect" href="/wiki/%ED%83%81%EB%B0%9C%EB%B6%80" title="탁발부">탁발부</a>에 의해 <a class="mw-redirect" href="/wiki/%ED%99%94%EB%B6%81" title="화북">화북</a>에 건국되어 <a href="/wiki/%EB%82%A8%EB%B6%81%EC%A1%B0_%EC%8B%9C%EB%8C%80" title="남북조 시대">남북조 시대</a>까지 이어진 왕조이다. 국호가 위(魏)였기 때문에 <a href="/wiki/%EC%A0%84%EA%B5%AD_%EC%8B%9C%EB%8C%80" title="전국 시대">전국 시대</a> <a href="/wiki/%EC%9C%84_(%EC%A0%84%EA%B5%AD)" title="위 (전국)">위나라</a>와 <a href="/wiki/%EC%82%BC%EA%B5%AD_%EC%8B%9C%EB%8C%80" title="삼국 시대">삼국 시대</a> <a href="/wiki/%EC%A1%B0%EC%A1%B0" title="조조">조조</a>의 <a href="/wiki/%EC%9C%84_(%EC%82%BC%EA%B5%AD)" title="위 (삼국)">위나라</a>와 구별하기 위해 북위라고 불리고 있다. 또는 후위(後魏), 탁발씨(拓跋氏)가 지배했다고 하여 탁발위(拓跋魏), 원씨(元氏)가 지배했다고 하여 원위(元魏)라고도 한다.</p>

<h2><span class="mw-headline" id=".EC.97.AD.EC.82.AC">역사</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%B6%81%EC%9C%84_(%EB%B6%81%EC%A1%B0)&amp;action=edit&amp;section=1" title="부분 편집: 역사">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<h3><span class="mw-headline" id=".ED.83.81.EB.B0.9C.EB.B6.80.EC.99.80_.EB.8C.80.EB.82.98.EB.9D.BC">탁발부와 대나라</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%B6%81%EC%9C%84_(%EB%B6%81%EC%A1%B0)&amp;action=edit&amp;section=2" title="부분 편집: 탁발부와 대나라">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p><a href="/wiki/261%EB%85%84" title="261년">261년</a> 탁발부의 <a href="/wiki/%ED%83%81%EB%B0%9C%EC%97%AD%EB%AF%B8" title="탁발역미">탁발역미</a>는 조조의 <a href="/wiki/%EC%9C%84_(%EC%82%BC%EA%B5%AD)" title="위 (삼국)">위나라</a>에 조공을 바쳤는데, 이것이 훗날 국호를 위나라로 정하는 이유가 된다.</p>
<p>3세기 중엽 탁발부는 내몽골의 파언탑랍(巴彦塔拉) 지방에서 세력을 넓혔다. 삼국을 통일한 <a href="/wiki/%EC%84%9C%EC%A7%84" title="서진">서진</a>은 탁발부의 세력을 이용해 북방을 안정시키기 위해 지금의 <a href="/wiki/%EC%82%B0%EC%8B%9C_%EC%84%B1_(%EC%82%B0%EC%84%9C%EC%84%B1)" title="산시 성 (산서성)">산시 성</a> 북부의 땅을 내어주어 살게 했다. 이에 따라 탁발부는 그곳에서 세력을 신장하였다.</p>
<p><a href="/wiki/315%EB%85%84" title="315년">315년</a>에 족장 <a href="/wiki/%ED%83%81%EB%B0%9C%EC%9D%98%EB%A1%9C" title="탁발의로">탁발의로</a>는 서진의 관작을 받고 <a class="mw-redirect" href="/wiki/%EB%8C%80%EB%82%98%EB%9D%BC" title="대나라">대나라</a>(代)의 왕으로 봉해졌다.</p>
<p>대나라는 <a href="/wiki/%ED%83%81%EB%B0%9C%EC%8B%AD%EC%9D%B5%EA%B1%B4" title="탁발십익건">탁발십익건</a> 때 <a href="/wiki/%EC%A0%84%EC%A7%84_(%EC%98%A4%ED%98%B8_%EC%8B%AD%EC%9C%A1%EA%B5%AD)" title="전진 (오호 십육국)">전진</a>의 <a href="/wiki/%EB%B6%80%EA%B2%AC" title="부견">부견</a>과의 싸움에 패하여 <a href="/wiki/376%EB%85%84" title="376년">376년</a>에 전진에게 멸망당했다.</p>
<h3><span class="mw-headline" id=".EB.B6.81.EC.9C.84.EC.9D.98_.EC.84.B1.EB.A6.BD">북위의 성립</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%B6%81%EC%9C%84_(%EB%B6%81%EC%A1%B0)&amp;action=edit&amp;section=3" title="부분 편집: 북위의 성립">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>대나라가 멸망했으나, <a href="/wiki/383%EB%85%84" title="383년">383년</a>에 전진이 <a class="mw-redirect" href="/wiki/%EB%B9%84%EC%88%98_%EC%A0%84%ED%88%AC" title="비수 전투">비수 전투</a>에서 패한 기회를 이용하여 탁발십익건의 손자 탁발규는 <a href="/wiki/386%EB%85%84" title="386년">386년</a>에 <a class="mw-redirect" href="/wiki/%EB%8C%80%EB%82%98%EB%9D%BC" title="대나라">대나라</a>를 재건하고 칭왕하여 자립했다. 이후에 황제를 자칭하고 국호를 <b>위</b>(魏)라고 하니, 이 탁발규가 바로 초대 황제인 <a href="/wiki/%EB%B6%81%EC%9C%84_%EB%8F%84%EB%AC%B4%EC%A0%9C" title="북위 도무제">도무제</a>이다.</p>
<p>도무제는 지금의 <a class="mw-redirect" href="/wiki/%EB%82%B4%EB%AA%BD%EA%B3%A8" title="내몽골">내몽골</a> 일대로 북상해 고비사막계, 정령족, <a href="/wiki/%ED%9D%89%EB%85%B8" title="흉노">흉노</a>족 등 여러 부족을 토벌하여 평정하고 오르도스 일대를 지배하에 두었다. 이어서 남쪽으로 진출해 지금의 <a href="/wiki/%ED%97%88%EB%B2%A0%EC%9D%B4_%EC%84%B1" title="허베이 성">허베이 성</a> 일대를 영유하고 있던 후연을 공격하였고, <a href="/wiki/397%EB%85%84" title="397년">397년</a>에 수도 중산(中山)을 함락하고 후연을 멸망시켰다. <a href="/wiki/398%EB%85%84" title="398년">398년</a>에 수도를 평성(平城), 즉 지금의 <a href="/wiki/%EC%82%B0%EC%8B%9C_%EC%84%B1_(%EC%82%B0%EC%84%9C%EC%84%B1)" title="산시 성 (산서성)">산시 성</a> <a class="mw-redirect" href="/wiki/%EB%8B%A4%ED%89%81" title="다퉁">다퉁</a>으로 옮겼고, 그 부근 태행산맥 동쪽의 6주의 백성 46만명을 소개시켰다.</p>
<p>도무제는 중국 전통의 국가체제를 채용하여 탁발부 밑에 있던 여러 부족을 해산시키고, 족장이하 부족민은 모두 중국의 호적에 편입시켜 한족과 혼합시켰다. 더불어 한족 출신의 명족(名族) 인재를 등용하여 국정에 참여하게 하였다. 이로써 북조 귀족제의 기초를 형성하게 된다.</p>
<h3><span class="mw-headline" id=".EB.B6.81.EC.9C.84.EC.9D.98_.EB.B0.9C.EC.A0.84">북위의 발전</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%B6%81%EC%9C%84_(%EB%B6%81%EC%A1%B0)&amp;action=edit&amp;section=4" title="부분 편집: 북위의 발전">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>도무제의 뒤를 이은 <a href="/wiki/%EB%B6%81%EC%9C%84_%EB%AA%85%EC%9B%90%EC%A0%9C" title="북위 명원제">명원제</a>는 <a href="/wiki/%EC%86%A1_(%EB%82%A8%EC%A1%B0)" title="송 (남조)">유송</a>을 공격해 하남을 빼앗았고, 명원제를 이은 <a href="/wiki/%EB%B6%81%EC%9C%84_%ED%83%9C%EB%AC%B4%EC%A0%9C" title="북위 태무제">태무제</a>는 <a href="/wiki/%ED%95%98_(%EC%98%A4%ED%98%B8_%EC%8B%AD%EC%9C%A1%EA%B5%AD)" title="하 (오호 십육국)">호하</a>, <a href="/wiki/%EB%B6%81%EC%97%B0" title="북연">북연</a>, <a href="/wiki/%EB%B6%81%EB%9F%89" title="북량">북량</a>을 차례로 멸망시킴으로써 <a href="/wiki/439%EB%85%84" title="439년">439년</a>에 화북을 통일하였다. 서역에서 조공을 바쳐오는 나라만 해도 20여개국에 이르니 북위의 국세는 크게 울려퍼졌다. 이때부터 중국은 <a href="/wiki/%EB%82%A8%EB%B6%81%EC%A1%B0_%EC%8B%9C%EB%8C%80" title="남북조 시대">남북조 시대</a>로 들어서게 되었다.</p>
<p>태무제는 내정을 정비하면서 또한 유송에 대한 공격을 재개해 회남과 강북을 빼앗았다. 그런데 이때 도사 <a href="/wiki/%EA%B5%AC%EA%B2%B8%EC%A7%80" title="구겸지">구겸지</a>가 도교 교단을 확립하였고 이어서 한족 출신 관료 <a class="mw-disambig" href="/wiki/%EC%B5%9C%ED%98%B8" title="최호">최호</a>와 손을 잡고 태무제에게 진언하여 폐불을 단행하게 하였다.(<a class="mw-redirect" href="/wiki/%EC%82%BC%EB%AC%B4%EC%9D%BC%EC%A2%85%EC%9D%98_%ED%8F%90%EB%B6%88" title="삼무일종의 폐불">삼무일종의 폐불</a> 첫 번째) 이 시기에 선비족의 한인 동화와 도시 귀족화가 진행되면서 서서히 북위 내부에 균열이 생기기 시작했다.</p>
<p><a href="/wiki/%EB%B6%81%EC%9C%84_%ED%9A%A8%EB%AC%B8%EC%A0%9C" title="북위 효문제">효문제</a> 때 풍태후가 섭정하면서 유교적인 예법을 받아들이고 <a href="/wiki/%EA%B7%A0%EC%A0%84%EC%A0%9C" title="균전제">균전제</a>를 시행하였으며 또한 삼장제를 확립했다. 풍태후 사후에 친정을 시작한 효문제는 <a href="/wiki/493%EB%85%84" title="493년">493년</a>에 수도를 평성에서 <a class="mw-redirect" href="/wiki/%EB%82%99%EC%96%91" title="낙양">낙양</a>으로 천도하였다. 그는 선비족의 복장과 언어를 금지하고, 성씨도 중국풍으로 바꿔 <a href="/wiki/%ED%83%81%EB%B0%9C%EC%94%A8" title="탁발씨">탁발씨</a>를 원씨(元氏)로 고치고, 선비족과 한족과의 혼인을 장려하면서 한족 문화와 동화하는 정책을 추진하여 북위의 문화는 크게 발전하였으나 반면 지금까지 내려온 선비족 고유의 소박한 상무정신은 사치와 문약으로 인해 바뀌게 되었다.</p>
<h3><span class="mw-headline" id=".EB.B6.81.EC.9C.84.EC.9D.98_.EC.87.A0.ED.87.B4_.EB.B0.8F_.EB.A9.B8.EB.A7.9D">북위의 쇠퇴 및 멸망</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%B6%81%EC%9C%84_(%EB%B6%81%EC%A1%B0)&amp;action=edit&amp;section=5" title="부분 편집: 북위의 쇠퇴 및 멸망">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>효문제 사후 어린 <a href="/wiki/%EB%B6%81%EC%9C%84_%ED%9A%A8%EB%AA%85%EC%A0%9C" title="북위 효명제">효명제</a>를 대신해 섭정한 <a href="/wiki/%EC%98%81%ED%83%9C%ED%9B%84" title="영태후">영태후</a>는 정치를 하면서 불교에 심취해 전국에 사탑을 건립하여 재정을 어지럽혔다. 그로 인해 국내에 도적들이 봉기하고, 그동안 누적된 한화정책에 대한 선비족의 불만은 <a href="/wiki/523%EB%85%84" title="523년">523년</a>에 일어난 <a class="mw-redirect" href="/wiki/6%EC%A7%84%EC%9D%98_%EB%82%9C" title="6진의 난">6진의 난</a>을 초래하여 북위 멸망의 원인이 되었다.</p>
<p>6진은 원래 북위의 수도 평성을 지키던 6곳의 군사기지로 이곳엔 왕실 계통의 유력자들이 머물면서 국경을 지켰기에 대우가 좋았다. 하지만 급진적인 한화정책과 수도의 이전으로 인해 대우가 소홀해지면서 그 불만이 누적되어 <a class="mw-redirect" href="/wiki/6%EC%A7%84%EC%9D%98_%EB%82%9C" title="6진의 난">6진의 난</a>을 초래하게 된 것이다.</p>
<p>영태후는 효명제를 독살하고 황제를 두 번이나 바꿔 세우지만 6진의 난의 진압과정 중 대군벌으로 성장한 지금의 <a href="/wiki/%EC%82%B0%EC%8B%9C_%EC%84%B1_(%EC%82%B0%EC%84%9C%EC%84%B1)" title="산시 성 (산서성)">산시 성</a> 삭현 지방의 흉노족 출신 <a href="/wiki/%EC%9D%B4%EC%A3%BC%EC%98%81_(%EB%B6%81%EC%9C%84)" title="이주영 (북위)">이주영</a>이 군사를 일으켜 영태후에 맞서 <a href="/wiki/%EB%B6%81%EC%9C%84_%ED%9A%A8%EC%9E%A5%EC%A0%9C" title="북위 효장제">효장제</a>를 옹립했다. 이주영은 <a href="/wiki/528%EB%85%84" title="528년">528년</a>에 낙양을 함락한 뒤 영태후와 영태후가 세운 황제를 황하에 빠뜨려 죽이고, 승상 이하 대신 천여 명을 처형하니 역사에서는 이를 <a href="/wiki/%ED%95%98%EC%9D%8C%EC%9D%98_%EB%B3%80" title="하음의 변">하음의 변</a>이라고 한다.</p>
<p>이주영은 6진의 난을 평정하고 진양(晉陽)에서 조정과 효장제를 마음대로 휘둘렀으나 이주씨 휘하의 장군 <a href="/wiki/%EA%B3%A0%ED%99%98" title="고환">고환</a>이 그 폭정을 비판하여 이주씨와 대립하고 <a href="/wiki/532%EB%85%84" title="532년">532년</a> 낙양에 들어와 이주씨에 의해 세워진 <a class="mw-redirect" href="/wiki/%EC%A0%88%EB%AF%BC%EC%A0%9C" title="절민제">절민제</a>를 폐위하고 <a class="mw-redirect" href="/wiki/%ED%9A%A8%EB%AC%B4%EC%A0%9C_(%EB%B6%81%EC%9C%84)" title="효무제 (북위)">효무제</a>를 세웠다. 고환은 이주씨 일당을 각지에서 격파하여 정권은 그에게로 넘어갔다.</p>
<p>그러나 효무제는 이번엔 고환의 전횡을 싫어해 낙양을 탈출해 <a href="/wiki/%EC%9E%A5%EC%95%88" title="장안">장안</a>으로 도망쳐 <a href="/wiki/%EC%9A%B0%EB%AC%B8%ED%83%9C" title="우문태">우문태</a>에게 의지하자, 고환은 <a class="mw-redirect" href="/wiki/%ED%9A%A8%EC%A0%95%EC%A0%9C" title="효정제">효정제</a>를 옹립하고 <a href="/wiki/534%EB%85%84" title="534년">534년</a> 수도를 <a href="/wiki/%EC%97%85_(%EC%A7%80%EB%AA%85)" title="업 (지명)">업</a>으로 옮겼다. 같은 해 우문태는 효무제를 살해하고 문제를 즉위시키니 이로써 북위는 동서로 분열하게 되었다. 역사상 이것을 <a href="/wiki/%EB%8F%99%EC%9C%84_(%EB%B6%81%EC%A1%B0)" title="동위 (북조)">동위</a>, <a href="/wiki/%EC%84%9C%EC%9C%84_(%EB%B6%81%EC%A1%B0)" title="서위 (북조)">서위</a>라 부른다.</p>
<p>동위의 정권은 모두 고씨의 손에 있어, 고환의 아들 <a class="mw-redirect" href="/wiki/%EA%B3%A0%EC%96%91" title="고양">고양</a>이 재상이 되어 <a href="/wiki/550%EB%85%84" title="550년">550년</a>에 효정제로부터 제위를 빼앗아 <a href="/wiki/%EB%B6%81%EC%A0%9C" title="북제">북제</a>를 건국했다. 서위에서도 정권은 우문씨가 차지하여 우문태의 아들 <a href="/wiki/%EC%9A%B0%EB%AC%B8%EA%B0%81" title="우문각">우문각</a>이 <a href="/wiki/556%EB%85%84" title="556년">556년</a>에 공제를 폐위하고 다음 해 <a href="/wiki/%EB%B6%81%EC%A3%BC" title="북주">북주</a>를 건국하니 이로써 동서 양위는 멸망했다.</p>
<h3><span class="mw-headline" id=".EB.AA.A8.EC.82.AC.EC.9E.90.EA.B7.80.EC.A0.9C.EB.8F.84">모사자귀제도</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%B6%81%EC%9C%84_(%EB%B6%81%EC%A1%B0)&amp;action=edit&amp;section=6" title="부분 편집: 모사자귀제도">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>북위에선 외척의 내정간섭을 막기 위해 실시한 모사자귀제도라는 잔혹한 제도를 실시했다. 이것은 후궁에게서 아이를 얻으면 생모를 죽이고, 아이는 따로 키우는 상황이 발생했다.</p>
