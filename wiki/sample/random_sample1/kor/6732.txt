<div class="mw-content-ltr" dir="ltr" id="mw-content-text" lang="ko"><script>function mfTempOpenSection(id){var block=document.getElementById("mf-section-"+id);block.className+=" open-block";block.previousSibling.className+=" open-block";}</script><p><b>북위 도무제</b>(北魏 道武帝, <a href="/wiki/371%EB%85%84" title="371년">371년</a> ~ <a href="/wiki/409%EB%85%84" title="409년">409년</a>)는 <a href="/wiki/%EC%A4%91%EA%B5%AD" title="중국">중국</a> <a href="/wiki/%EC%98%A4%ED%98%B8_%EC%8B%AD%EC%9C%A1%EA%B5%AD_%EC%8B%9C%EB%8C%80" title="오호 십육국 시대">오호 십육국 시대</a> <a href="/wiki/%EB%B6%81%EC%9C%84_(%EB%B6%81%EC%A1%B0)" title="북위 (북조)">북위</a>의 제1대 황제(재위 : <a href="/wiki/386%EB%85%84" title="386년">386년</a> ~ <a href="/wiki/409%EB%85%84" title="409년">409년</a>)이다. 성명은 <b>탁발규</b>(拓跋珪), <a href="/wiki/%EB%AC%98%ED%98%B8" title="묘호">묘호</a>는 <b>태조</b>(太祖), <a href="/wiki/%EC%8B%9C%ED%98%B8" title="시호">시호</a>는 <b>도무황제</b>(道武皇帝)이다. 원래 묘호는 <b>열조</b>(烈祖), 원래 시호는 <b>선무황제</b>(宣武皇帝)였다. 할아버지는 <a href="/wiki/%ED%83%81%EB%B0%9C%EC%8B%AD%EC%9D%B5%EA%B1%B4" title="탁발십익건">탁발십익건</a>(拓跋什翼犍)이며, 아버지는 <a href="/wiki/%ED%83%81%EB%B0%9C%EC%8B%9D" title="탁발식">탁발식</a>(拓跋寔)이다.</p>

<h2><span class="mw-headline" id=".EC.83.9D.EC.95.A0">생애</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%B6%81%EC%9C%84_%EB%8F%84%EB%AC%B4%EC%A0%9C&amp;action=edit&amp;section=1" title="부분 편집: 생애">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<h3><span class="mw-headline" id=".EC.83.9D.EC.95.A0_.EC.B4.88.EA.B8.B0">생애 초기</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%B6%81%EC%9C%84_%EB%8F%84%EB%AC%B4%EC%A0%9C&amp;action=edit&amp;section=2" title="부분 편집: 생애 초기">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>탁발규는 <a href="/wiki/371%EB%85%84" title="371년">371년</a>에 태어났다. 그의 아버지는 대나라 왕 <a href="/wiki/%ED%83%81%EB%B0%9C%EC%8B%AD%EC%9D%B5%EA%B1%B4" title="탁발십익건">탁발십익건</a>의 아들, <a href="/wiki/%ED%83%81%EB%B0%9C%EC%8B%9D" title="탁발식">탁발식</a>이며, 어머니는 하란부 출신인 하씨 부인이다. 탁발규가 태어나기 몇 달 전 대나라의 반란에서 아버지 <a href="/wiki/%ED%83%81%EB%B0%9C%EC%8B%9D" title="탁발식">탁발식</a>이 전사했다. 몇달 후, 하씨 부인은 탁발규를 낳았다. 이때 <a href="/wiki/%ED%83%81%EB%B0%9C%EC%8B%AD%EC%9D%B5%EA%B1%B4" title="탁발십익건">탁발십익건</a>은 하란부 세력과의 관계가 약화 되는 것을 꺼려 임신 중인 며느리를 자신의 아내로 삼았다. 그 후 십익건과 하씨 부인은 아들 셋을 더 낳았는데, 그들이 <a class="new" href="/w/index.php?title=%ED%83%81%EB%B0%9C%EC%9D%98&amp;action=edit&amp;redlink=1" title="탁발의 (없는 문서)">탁발의</a>(拓跋儀), <a class="new" href="/w/index.php?title=%ED%83%81%EB%B0%9C%EB%A0%AC&amp;action=edit&amp;redlink=1" title="탁발렬 (없는 문서)">탁발렬</a>(拓跋烈), <a class="new" href="/w/index.php?title=%ED%83%81%EB%B0%9C%EA%B3%A0&amp;action=edit&amp;redlink=1" title="탁발고 (없는 문서)">탁발고</a>(拓跋觚)이다.</p>
<p>376년 10월, 전진의 철기부대가 대나라를 쳐들어와 <a href="/wiki/%ED%83%81%EB%B0%9C%EC%8B%AD%EC%9D%B5%EA%B1%B4" title="탁발십익건">탁발십익건</a>과 하씨부인, 탁발규 등의 왕족은 터전을 떠났다. 그 와중에 십익건의 아들 <a class="new" href="/w/index.php?title=%ED%83%81%EB%B0%9C%EC%8B%9D%EA%B5%B0&amp;action=edit&amp;redlink=1" title="탁발식군 (없는 문서)">탁발식군</a>(拓跋寔君)이 반란을 일으켜 형제들을 죽였다. 그래서 하씨는 십익건을 사로잡고, 탁발규가 아버지를 포박하여 항복을 한다는 조건으로 전진의 군대에 투항했다.</p>
<p>전진의 군대는 빠르게 식군의 난을 평정하고, 십익건과 귀족들을 장안으로 보냈다. 당시 전진의 왕은 <a href="/wiki/%EB%B6%80%EA%B2%AC" title="부견">부견</a>(苻堅)은 도량이 커 적국의 귀족을 극진히 대접하였는데, 그때문에 <a href="/wiki/%EB%B6%80%EA%B2%AC" title="부견">부견</a>은 후일 <a class="mw-redirect" href="/wiki/%EB%AA%A8%EC%9A%A9%EB%B6%80" title="모용부">모용부</a>(慕容部)에게 배신을 당한다. 부견은 십익건을 장안의 태학에 보내 교육을 시켰으나, 그의 명은 길지 못했다. 이에 <a href="/wiki/%EB%B6%80%EA%B2%AC" title="부견">부견</a>은 크게 아쉬워 했다. 하지만 탁발식군과 탁발규는 달랐다. 탁발식군은 형제를 죽이고 난을 일으킨 죄로 거열형에 처했다. 그리고 탁발규와 하씨 부인에게는 아버지를 묶고 투항한 죄를 물어 촉땅으로 유배를 보냈다.</p>
<h3><span class="mw-headline" id=".EB.B9.84.EC.88.98_.EB.8C.80.EC.A0.84">비수 대전</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%B6%81%EC%9C%84_%EB%8F%84%EB%AC%B4%EC%A0%9C&amp;action=edit&amp;section=3" title="부분 편집: 비수 대전">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p><a href="/wiki/378%EB%85%84" title="378년">378년</a> 4월, 부견은 <a class="mw-disambig" href="/wiki/%EB%B6%80%EB%B9%84" title="부비">부비</a>(符丕)에게 17만을 주고 동진의 양양을 공격케했다. 그리고 <a href="/wiki/379%EB%85%84" title="379년">379년</a> 2월, 양양자사 <a href="/wiki/%EC%A3%BC%EC%84%9C" title="주서">주서</a>가 항복해왔다. 부견은 크게 기뻐했고, 전진의 군사는 이어 동진의 광릉을 압박했으나 패배했다. 부견은 노하여 대대적인 병력으로 동진을 치자고 했으나, 모든 관료들이 반대했고, 그가 믿는 동생 <a href="/wiki/%EB%B6%80%EC%9C%B5" title="부융">부융</a>도 반대했으나, 오직 <a class="mw-redirect" href="/wiki/%EB%AA%A8%EC%9A%A9%EB%B6%80" title="모용부">모용부</a>의 <a href="/wiki/%EB%AA%A8%EC%9A%A9%EC%88%98" title="모용수">모용수</a>만이 찬성했다. 부견은 이에 힘을 입어 그의 뜻대로 남정을 감행했다.</p>
<p>하지만 부견은 적 <a href="/wiki/%EB%8F%99%EC%A7%84" title="동진">동진</a>을 너무 얕잡아 보았고, <a href="/wiki/383%EB%85%84" title="383년">383년</a> 비수대전에서 <a class="mw-disambig" href="/wiki/%EC%A0%84%EC%A7%84" title="전진">전진</a>의 군대는 <a href="/wiki/%EC%82%AC%EC%95%88" title="사안">사안</a>(謝安)이 이끄는 군대에게 패했다.</p>
<p>하씨와 탁발규 모자는 이 틈을 타 <a href="/wiki/%EC%9E%A5%EC%95%88" title="장안">장안</a>에 기거하는 <a href="/wiki/%EC%A0%84%EC%97%B0" title="전연">전연</a>의 귀족 <a href="/wiki/%EB%AA%A8%EC%9A%A9%EC%88%98" title="모용수">모용수</a>와 함께 북방의 <a class="mw-redirect" href="/wiki/%EC%A4%91%EC%82%B0" title="중산">중산</a>으로 가 고향 땅을 밟을 준비를 했다.모자는 독고부에 도착했지만, 얼마 후 내란이 일어나 <a class="mw-disambig" href="/wiki/%EC%9C%A0%ED%98%84" title="유현">유현</a>이 우두머리가 되었다. <a class="mw-disambig" href="/wiki/%EC%9C%A0%ED%98%84" title="유현">유현</a>은 그 지방의 맹주가 될 야망을 가지고 있었으므로 탁발규 형제는 그에게 정치적으로 위협이 되었다. 그래서 유현은 이들을 죽이려 하였으나, 탁발규의 고모는 유현의 제수였다. 그녀는 이 소식을 하씨에게 알렸고, 하씨는 유현에게 술을 먹여 취하게 하고, 형제를 하란부로 도주시켰다. 그 사실을 안 유현은 하씨를 죽이려 했으나, 그의 동생 집안 식구가 말려 하씨는 죽음을 면하고, 후일 하란부로 도망했다.</p>
<h3><span class="mw-headline" id=".EA.B1.B4.EA.B5.AD">건국</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%B6%81%EC%9C%84_%EB%8F%84%EB%AC%B4%EC%A0%9C&amp;action=edit&amp;section=4" title="부분 편집: 건국">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>탁발규가 하란부에 도착하자 과거의 부족 수령들이 투항해오니, <a href="/wiki/386%EB%85%84" title="386년">386년</a> 정월에 선비 연맹이 재수립되고, 탁발규를 대왕으로 추대했다. 하여 탁발규는 국호를 <a class="mw-disambig" href="/wiki/%EB%B6%81%EC%9C%84" title="북위">위</a>(북위)로 바꾸고 연호를 등국이라 하고, 도읍을 성락에 정했다. 유현은 이에 분개하고 탁발규의 숙부인 굴돌을 이용했다. 이에 탁발규의 즉위는 합법성을 의심받았다. 하씨의 도움으로 탁발규는 독고부 군대의 공격을 피해 백성을 데리고 하란부로 이주하고 <a href="/wiki/%ED%9B%84%EC%97%B0" title="후연">후연</a>에 지원을 요청했다. <a href="/wiki/%ED%9B%84%EC%97%B0" title="후연">후연</a>은 <a href="/wiki/%EB%AA%A8%EC%9A%A9%EC%88%98" title="모용수">모용수</a>가 세운 나라였다. 굴돌이 의지하는 세력은 <a href="/wiki/%EB%AA%A8%EC%9A%A9%EC%88%98" title="모용수">모용수</a>에 적대적인 <a class="mw-disambig" href="/wiki/%EC%84%9C%EC%97%B0" title="서연">서연</a>이었다. 하여 <a href="/wiki/%EB%AA%A8%EC%9A%A9%EC%88%98" title="모용수">모용수</a>는 망설임 없이 탁발규를 도와 굴돌을 무찔렀다. 그로인해 자연스럽게 유현의 세력도 약해졌다. 독고부에서 내란이 일어나자, 이 틈을 타서 탁발규와 모용수는 연합해 독고부를 쳤고, 대승을 거두었다.</p>
<h3><span class="mw-headline" id=".EC.B0.B8.ED.95.A9.ED.8C.8C.EC.9D.98_.EA.B2.B0.EC.A0.84.EA.B3.BC_.ED.9B.84.EC.97.B0.EC.9D.98_.EB.A9.B8.EB.A7.9D">참합파의 결전과 후연의 멸망</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%B6%81%EC%9C%84_%EB%8F%84%EB%AC%B4%EC%A0%9C&amp;action=edit&amp;section=5" title="부분 편집: 참합파의 결전과 후연의 멸망">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>탁발규의 위나라가 강성해지는 것엔 후연의 도움이 큰 역할을 했다. 하지만 시간이 흐르면서 두 나라의 관계가 틀어졌다. 등국 6년, <a href="/wiki/%EB%AA%A8%EC%9A%A9%EC%88%98" title="모용수">모용수</a>는 <a class="mw-redirect" href="/wiki/%EC%A4%91%EC%82%B0" title="중산">중산</a>에 사신으로 간 탁발규의 아우 <a class="new" href="/w/index.php?title=%ED%83%81%EB%B0%9C%EA%B3%A0&amp;action=edit&amp;redlink=1" title="탁발고 (없는 문서)">탁발고</a>를 억류하고, 전쟁에 쓸 말을 내놓으라 협박했다. 하지만 탁발규는 이미 형이 죽으면 동생이 그 자리를 물려받는 유목민족의 제도를 폐지하고, 부자세습 제도를 도입할 작정이었으므로, 이를 거절했다. 이때부터 탁발고는 <a href="/wiki/%ED%9B%84%EC%97%B0" title="후연">후연</a>에서 살해당했고, <a href="/wiki/%ED%9B%84%EC%97%B0" title="후연">후연</a>과 <a class="mw-disambig" href="/wiki/%EB%B6%81%EC%9C%84" title="북위">북위</a>도 적대 관계로 변했다. 탁발규는 모용수와 적대관계가 되면서 <a class="mw-disambig" href="/wiki/%EC%84%9C%EC%97%B0" title="서연">서연</a>의 <a href="/wiki/%EB%AA%A8%EC%9A%A9%EC%98%81" title="모용영">모용영</a>과 우호관계를 맺었다.</p>
<p>등국 10년인 <a href="/wiki/395%EB%85%84" title="395년">395년</a>, <a href="/wiki/%ED%9B%84%EC%97%B0" title="후연">후연</a>과 <a class="mw-disambig" href="/wiki/%EB%B6%81%EC%9C%84" title="북위">북위</a>는 전쟁에 돌입했다. <a href="/wiki/%EB%AA%A8%EC%9A%A9%EC%88%98" title="모용수">모용수</a>는 태자 <a href="/wiki/%EB%AA%A8%EC%9A%A9%EB%B3%B4" title="모용보">모용보</a>를 총사령관으로 보내 북위를 멸망시키려 들었다. 그러나 모용보는 전쟁에 소질이 없는 위인이었다. 탁발규는 강변에 많은 방어 진지를 구축하고, 병사들에게 방비를 당부하면서, 대군을 파견해 <a href="/wiki/%EB%AA%A8%EC%9A%A9%EB%B3%B4" title="모용보">모용보</a>와 <a class="mw-redirect" href="/wiki/%EC%A4%91%EC%82%B0" title="중산">중산</a>의 연락을 두절시켰다.</p>
<p>탁발규는 서신을 위조해 70세의 <a href="/wiki/%EB%AA%A8%EC%9A%A9%EC%88%98" title="모용수">모용수</a>가 죽었다고 보냈다. <a href="/wiki/%EB%AA%A8%EC%9A%A9%EB%B3%B4" title="모용보">모용보</a>는 이를 믿었고, 황제의 위를 뺏길 것을 염려한 <a href="/wiki/%EB%AA%A8%EC%9A%A9%EB%B3%B4" title="모용보">모용보</a>는 서들러 회군했다. 탁발규는 이를 놓치지 않고 정예군 2만을 지휘해 신속하게 강을 건넜다. 밤낮을 가리지 않고 길을 재촉하자, 오늘날 산시 다퉁 부근의 <a class="new" href="/w/index.php?title=%EC%B0%B8%ED%95%A9%ED%8C%8C&amp;action=edit&amp;redlink=1" title="참합파 (없는 문서)">참합파</a>까지 추격했다. 후연의 군대는 이곳에서 쉬고 있었는데, 탁발규가 기습하자 대혼란이 일었다. 그 통에 수만명이 죽었고, 수만명이 투항했으며 태자 <a href="/wiki/%EB%AA%A8%EC%9A%A9%EB%B3%B4" title="모용보">모용보</a>는 몇천의 군대와 함께 달아났다. 이때 탁발규는 돌이킬 수 없는 실수를 저질렀는데, 그가 투항한 후연군들을 한명도 남김없이 죽여버린 것이다.</p>
<p><a href="/wiki/%EB%AA%A8%EC%9A%A9%EB%B3%B4" title="모용보">모용보</a>가 <a class="mw-redirect" href="/wiki/%EC%A4%91%EC%82%B0" title="중산">중산</a>에 도착해보니 세상을 떠났다던 아버지가 멀쩡하게 살아있었다. 그는 그제서야 속은 것을 알고 후회를 했지만 늦었다. 그는 <a href="/wiki/%EB%AA%A8%EC%9A%A9%EC%88%98" title="모용수">모용수</a>에게 실수를 만회하겠다 하였으나, 아버지는 아들의 실수를 용납하지 않았다. 그리하여 자신이 직접 군대를 정비해 이듬해인 <a href="/wiki/396%EB%85%84" title="396년">396년</a> 3월 출정했다. 후연군들은 몰래 행군하여 순식간에 평성에 도착했다. 탁발규는 이때 순시 중이었으므로 후연군들은 방비가 허술한 평성을 쉽게 얻었다. 승리를 들은 <a href="/wiki/%EB%AA%A8%EC%9A%A9%EC%88%98" title="모용수">모용수</a>는 직접 평성으로 향했다. 하지만 평성으로 가던 도중에 참합파를 지나게 되었는데, 아군 병사들이 죽어 시체가 산을 이룬 참혹한 광경을 목격해야만 했다. 병사들은 울부짖었고, 대성통곡을 하니 그 소리가 천지를 진동했다. 이미 70세의 나이가 있고 중병에 걸린 <a href="/wiki/%EB%AA%A8%EC%9A%A9%EC%88%98" title="모용수">모용수</a>는 충격을 이기지 못하고 피를 토하며 쓰러졌다. 그의 병이 위중해지자 병사들은 평성으로 향하지 못하고 중산으로 말머리를 돌렸다. 행렬리 상곡에 이르렀을 때, <a href="/wiki/%EB%AA%A8%EC%9A%A9%EC%88%98" title="모용수">모용수</a>는 그만 세상을 떠나고, 태자 <a href="/wiki/%EB%AA%A8%EC%9A%A9%EB%B3%B4" title="모용보">모용보</a>가 제위를 물려받았으니, 그가 <a href="/wiki/%EB%AA%A8%EC%9A%A9%EB%B3%B4" title="모용보">혜민제</a>이다.</p>
<p>모용수가 죽었다는 소식을 들은 탁발규는 두려워 않고 40만 대군을 풀어 전진했다. <a class="mw-disambig" href="/wiki/%EB%B6%81%EC%9C%84" title="북위">위나라</a>의 군대는 순조롭게 진격하여 상곡, 병주, 상산등을 함락하고, 황하 이북의 후연 영토 중 중산, 업, 신 이 세 도성만 남겨둔 채로 모두 위나라의 영토가 되었다. 탁발규는 군대를 셋으로 나누어 도성들을 공격했다. 하지만 예상 밖으로 위나라의 군사들의 피해만 늘어났다. 이 까닭은 탁발규가 항복한 포로들을 모조리 죽였다는 것을 알기 때문이었다. 이를 알기 때문에 후연 사람들은 남녀노소를 가리지 않고 기를 쓰고 성을 사수했다. 이에 탁발규는 크게 후회했다. 하여 탁발규는 포로를 모조리 죽이자 한 장수를 파면하고 투항한 사람들을 우대했다. 이러한 노력은 결실을 맺어 <a href="/wiki/397%EB%85%84" title="397년">397년</a> 정월, 탁발규는 신 땅을 공략해 승리를 거두었다.</p>
<h3><span class="mw-headline" id=".ED.99.A9.EC.A0.9C_.EB.93.B1.EA.B7.B9">황제 등극</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%B6%81%EC%9C%84_%EB%8F%84%EB%AC%B4%EC%A0%9C&amp;action=edit&amp;section=6" title="부분 편집: 황제 등극">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p><a href="/wiki/398%EB%85%84" title="398년">398년</a>, 탁발규는 황제에 즉위하고 수도를 <a class="mw-redirect" href="/wiki/%ED%8F%89%EC%84%B1" title="평성">평성</a>으로 정함으로써 <a class="mw-disambig" href="/wiki/%EB%B6%81%EC%9C%84" title="북위">북위</a>를 연 개국황제가 되었다. 그는 지식인을 널리 받아들여 재주를 발휘할 기회를 주었다. 제도와 예의범절을 정비하고, 궁전과 종묘를 짓고 통일된 도량형을 규정하고, 관직의 품급과 각종 의식에 쓰이는 음악을 제정하고 법령과 각종 금령을 명시했다. 행정 구역과 군사 제도도 확실히 하여 나라를 질서 정연하게 정비해나갔다. 탁발규는 각 지역 상황에 따라 대단히 상세한 조세와 부역 제도를 제정하여 과거의 씨족 구성원들을 안정된 생활을 하는 농민으로 변모시켰다. 북위의 경제도 이를 계기로 크게 발전하여 국가의 재정 상태도 풍족해졌다.</p>
<h3><span class="mw-headline" id=".ED.95.99.EC.A0.95.EA.B3.BC_.EB.A9.B8.EB.A7.9D">학정과 멸망</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%B6%81%EC%9C%84_%EB%8F%84%EB%AC%B4%EC%A0%9C&amp;action=edit&amp;section=7" title="부분 편집: 학정과 멸망">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p><a href="/wiki/402%EB%85%84" title="402년">402년</a> 5월, 탁발규는 시벽에서 <a class="mw-disambig" href="/wiki/%ED%9B%84%EC%A7%84" title="후진">후진</a>의 수만 대군을 물리쳐 시국을 안정시켰다. 탁발규는 중원의 황제처럼 자신의 아들에게 제위를 상속시키려 했다. 탁발규의 동생 <a class="new" href="/w/index.php?title=%ED%83%81%EB%B0%9C%EC%9D%98&amp;action=edit&amp;redlink=1" title="탁발의 (없는 문서)">탁발의</a>는 용감하여 많은 사람의 존경을 받았기 때문에 탁발규는 항상 그가 마음에 걸렸다. 손자 <a class="mw-redirect" href="/wiki/%ED%83%81%EB%B0%9C%EB%8F%84" title="탁발도">탁발도</a>가 태어났을 때, 탁발규는 <a class="new" href="/w/index.php?title=%ED%83%81%EB%B0%9C%EC%9D%98&amp;action=edit&amp;redlink=1" title="탁발의 (없는 문서)">탁발의</a>를 불러 그를 시험했고, 탁발의는 제위를 양보했다. 탁발규는 이외에도 항상 불안했다. 그래서 의심이 가는 사람들을 죽여 자손들이 순조롭게 즉위하는 데의 걸림돌을 제거하려고 했다. 그는 예전에 굴돌과 내왕했던 고읍공 막제를 죽였고, 다른 대신들도 의심이 가면 죽였다. 주위에 도사린 죽음의 위협을 느낀 조정 관리와 백성들은 불똥이 언제 튈지 몰라 공포의 날을 보냈고, 관리는 일을 하기는커녕 제 몸 사리기에 급급하니, 법 기강이 문란해졌다. 탁발규는 그의 아들 <a class="new" href="/w/index.php?title=%ED%83%81%EB%B0%9C%EC%86%8C&amp;action=edit&amp;redlink=1" title="탁발소 (없는 문서)">탁발소</a>(拓跋昭)의 손에 살해당했다. 북위의 2대 황제 자리는 그의 장남인 <a class="mw-disambig" href="/wiki/%ED%83%81%EB%B0%9C%EC%82%AC" title="탁발사">탁발사</a>(拓跋嗣)에게 돌아갔는데, 그가 바로 <a class="mw-disambig" href="/wiki/%ED%83%81%EB%B0%9C%EC%82%AC" title="탁발사">명원제</a>이다.</p>
