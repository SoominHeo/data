<p><b>불라트 샬보비치 오쿠자바</b> (<span lang="en" xml:lang="en">Bulat Shalvovich Okudzhava</span>, <span style="font-size: smaller;"><a href="/wiki/%EB%9F%AC%EC%8B%9C%EC%95%84%EC%96%B4" title="러시아어">러시아어</a>:</span> <span lang="ru" xml:lang="ru">Булат Шалвович Окуджава</span>, <a href="/wiki/1924%EB%85%84" title="1924년">1924년</a> <a href="/wiki/5%EC%9B%94_9%EC%9D%BC" title="5월 9일">5월 9일</a> ~ <a href="/wiki/1997%EB%85%84" title="1997년">1997년</a> <a href="/wiki/6%EC%9B%94_12%EC%9D%BC" title="6월 12일">6월 12일</a>)는 <a href="/wiki/%EB%9F%AC%EC%8B%9C%EC%95%84" title="러시아">러시아</a>의 시인이며 작곡자이자 가수이다.</p>
<p>오쿠자바는 <a href="/wiki/%EB%AA%A8%EC%8A%A4%ED%81%AC%EB%B0%94" title="모스크바">모스크바</a>에서 태어났다. 그는 노래를 직접 쓰고 부르는 러시아 <a href="/wiki/%EC%9D%8C%EC%9C%A0%EC%8B%9C%EA%B0%80" title="음유시가">음유시가</a>의 개척자로 평생 200여 곡을 작곡하고 노래함으로써 <a href="/wiki/%EB%B8%94%EB%9D%BC%EB%94%94%EB%AF%B8%EB%A5%B4_%EB%B9%84%EC%86%8C%EC%B8%A0%ED%82%A4" title="블라디미르 비소츠키">블라디미르 비소츠키</a>와 더불어 <a href="/wiki/%EC%9E%94%EB%82%98_%EB%B9%84%EC%B3%85%EC%8A%A4%EC%B9%B4%EC%95%BC" title="잔나 비쳅스카야">잔나 비쳅스카야</a>와 같은 <a class="mw-redirect" href="/wiki/%EC%9D%8C%EC%9C%A0%EC%8B%9C%EC%9D%B8" title="음유시인">음유시인</a>들에게 지대한 영향을 끼쳤다.</p>
<p>오쿠자바가 어렸을 때 <a href="/wiki/%EC%A1%B0%EC%A7%80%EC%95%84%EC%9D%B8" title="조지아인">조지아인</a> 아버지는 <a href="/wiki/%EC%86%8C%EB%A0%A8_%EA%B3%B5%EC%82%B0%EB%8B%B9" title="소련 공산당">소련 공산당</a>에 의하여 사형 당하였으며 <a href="/wiki/%EC%95%84%EB%A5%B4%EB%A9%94%EB%8B%88%EC%95%84%EC%9D%B8" title="아르메니아인">아르메니아인</a> 어머니도 감옥에서 18년간이나 복역하는 아픔을 겪어 일찍부터 독재에 대한 저항의식을 가지게 되었다. <a class="mw-redirect" href="/wiki/%EC%A0%9C2%EC%B0%A8_%EC%84%B8%EA%B3%84%EB%8C%80%EC%A0%84" title="제2차 세계대전">제2차 세계대전</a>에 참전한 후 트빌리시 주립대학교에 진학하여 <a href="/wiki/1950%EB%85%84" title="1950년">1950년</a>에 졸업하였다. 졸업 후 교사로 재직하다가 <a href="/wiki/1956%EB%85%84" title="1956년">1956년</a> <a href="/wiki/%EB%AA%A8%EC%8A%A4%ED%81%AC%EB%B0%94" title="모스크바">모스크바</a>로 이주하여 몰도다야 그바르디야 출판사에 근무하면서 작곡을 하고 노래 부르기 시작했다. 서정적인 멜로디를 지적인 음성으로 부른 그의 노래는 1970년대 말까지 공식적으로 출판되지 못하였으나 그의 노래를 사랑하는 사람들에 의해서 비공식적으로 녹음되어 여러 나라에 널리 퍼지게 되었다.</p>
<p>오쿠자바의 노래와 시는 1980년도에 들어 와서와 공식적으로 출판되었으며 <a href="/wiki/1991%EB%85%84" title="1991년">1991년</a> [USSR State Prize]상을 수상하였다. 그는 <a href="/wiki/1997%EB%85%84" title="1997년">1997년</a> <a href="/wiki/6%EC%9B%94_12%EC%9D%BC" title="6월 12일">6월 12일</a> <a class="mw-redirect" href="/wiki/%ED%8C%8C%EB%A6%AC_%EC%8B%9C" title="파리 시">파리</a>에서 사망하였으며 <a href="/wiki/%EB%AA%A8%EC%8A%A4%ED%81%AC%EB%B0%94" title="모스크바">모스크바</a>의 <a class="new" href="/w/index.php?title=%EB%B0%94%EA%B0%84%EC%BD%94%EB%B3%B4_%EB%AC%98%EC%A7%80&amp;action=edit&amp;redlink=1" title="바간코보 묘지 (없는 문서)">바간코보 묘지</a>에 묻혔다. <a href="/wiki/%EB%AA%A8%EC%8A%A4%ED%81%AC%EB%B0%94" title="모스크바">모스크바</a>의 <a class="new" href="/w/index.php?title=%EC%95%84%EB%A5%B4%EB%B0%94%ED%8A%B8&amp;action=edit&amp;redlink=1" title="아르바트 (없는 문서)">아르바트</a> 거리에 그를 추모하는 동상이 세워져 있다.</p>
