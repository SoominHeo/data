<h2><span class="mw-headline" id=".ED.94.84.EB.9E.91.EC.8A.A4_.EB.AC.B8.ED.95.99">프랑스 문학</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%B6%88%EB%AC%B8%ED%95%99&amp;action=edit&amp;section=1" title="부분 편집: 프랑스 문학">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<ul>
<li><a href="/wiki/%ED%94%84%EB%9E%91%EC%8A%A4_%EB%AC%B8%ED%95%99" title="프랑스 문학">프랑스 문학</a></li>
</ul>
<p>8세기부터 불어 문헌이 발견되며 11세기부터 <a href="/wiki/%EB%AC%B4%ED%9B%88%EC%8B%9C" title="무훈시">무훈시</a>와 <a href="/wiki/%EC%84%9C%EC%A0%95%EC%8B%9C" title="서정시">서정시</a>를 통해 본격적인 문학활동이 시작된 프랑스 문학은 사실상 불문학의 절대적인 주류를 이루고 있다.</p>
<h2><span class="mw-headline" id=".EC.9C.A0.EB.9F.BD.EC.9D.98_.EB.B6.88.EB.AC.B8.ED.95.99">유럽의 불문학</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%B6%88%EB%AC%B8%ED%95%99&amp;action=edit&amp;section=2" title="부분 편집: 유럽의 불문학">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<h3><span class="mw-headline" id=".EB.B2.A8.EA.B8.B0.EC.97.90_.EB.B6.88.EB.AC.B8.ED.95.99">벨기에 불문학</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%B6%88%EB%AC%B8%ED%95%99&amp;action=edit&amp;section=3" title="부분 편집: 벨기에 불문학">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<ul>
<li><a class="new" href="/w/index.php?title=%EB%B2%A8%EA%B8%B0%EC%97%90_%EB%AC%B8%ED%95%99&amp;action=edit&amp;redlink=1" title="벨기에 문학 (없는 문서)">벨기에 문학</a></li>
</ul>
<p>벨기에와 스위스의 <a class="mw-redirect" href="/wiki/%EB%B6%88%EC%96%B4%EA%B6%8C" title="불어권">불어권</a> 지방에서 창작된 문학도 불문학사에서 중요한 위치를 차지하고 있다. 현재의 벨기에에 해당하는 지역에는 <a href="/wiki/16%EC%84%B8%EA%B8%B0" title="16세기">16세기</a>부터 불어가 상류층에서 사용되기 시작하나, 대다수 서민들의 언어는 <a href="/wiki/%EC%99%88%EB%A1%B1%EC%96%B4" title="왈롱어">왈롱어</a>나 <a href="/wiki/%ED%94%8C%EB%9D%BC%EB%A7%9D%EC%96%B4" title="플라망어">플라망어</a> 등의 방언이었다. 현재의 벨기에는 <a href="/wiki/1830%EB%85%84" title="1830년">1830년</a>에 건국된 것으로, <a href="/wiki/%EC%98%A4%EB%9D%BC%EB%85%9C_%EA%B3%B5" title="오라녜 공">오라녜 공</a> 치하의 네덜란드 지배에 반발해서 독립한 불어권 국가로서, 모든 계층의 국민들을 불어화하려는 노력이 이루어졌다. 곧이어 <a href="/wiki/19%EC%84%B8%EA%B8%B0" title="19세기">19세기</a> 말부터 벨기에 불문학은 개화기를 맞아, <a href="/wiki/%EB%AA%A8%EB%A6%AC%EC%8A%A4_%EB%A7%88%ED%85%8C%EB%A5%BC%EB%A7%81%ED%81%AC" title="모리스 마테를링크">모리스 마테를링크</a>, <a class="new" href="/w/index.php?title=%EC%A1%B0%EB%A5%B4%EC%A3%BC_%EB%A1%9C%EB%8D%B4%EB%B0%94%ED%9D%90&amp;action=edit&amp;redlink=1" title="조르주 로덴바흐 (없는 문서)">조르주 로덴바흐</a> 등의 작가가 나오게 되었다. 현재 벨기에는 <a href="/wiki/%EB%84%A4%EB%8D%9C%EB%9E%80%EB%93%9C%EC%96%B4" title="네덜란드어">네덜란드어</a>도 공히 쓰이는 2중국어 국가가 되었으나, 불어권 벨기에의 문학 생명력은 아직도 활발하다.</p>
<h3><span class="mw-headline" id=".EC.8A.A4.EC.9C.84.EC.8A.A4_.EB.B6.88.EB.AC.B8.ED.95.99">스위스 불문학</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%B6%88%EB%AC%B8%ED%95%99&amp;action=edit&amp;section=4" title="부분 편집: 스위스 불문학">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<ul>
<li><a class="new" href="/w/index.php?title=%EC%8A%A4%EC%9C%84%EC%8A%A4_%EB%AC%B8%ED%95%99&amp;action=edit&amp;redlink=1" title="스위스 문학 (없는 문서)">스위스 문학</a></li>
</ul>
<p><a href="/wiki/%EC%8A%A4%EC%9C%84%EC%8A%A4" title="스위스">스위스</a>는 크게 <a href="/wiki/%EB%8F%85%EC%9D%BC%EC%96%B4" title="독일어">독일어</a>, <a href="/wiki/%ED%94%84%EB%9E%91%EC%8A%A4%EC%96%B4" title="프랑스어">불어</a>, <a href="/wiki/%EC%9D%B4%ED%83%88%EB%A6%AC%EC%95%84%EC%96%B4" title="이탈리아어">이탈리아어</a>의 3국어권으로 나뉜 국가이다. 현대에 인구 60만 정도인 불어권 스위스는 비록 작으나 수많은 작가를 배출해왔으며, <a href="/wiki/%EB%B3%BC%ED%85%8C%EB%A5%B4" title="볼테르">볼테르</a>나 <a href="/wiki/%EC%9E%A5%EC%9E%90%ED%81%AC_%EB%A3%A8%EC%86%8C" title="장자크 루소">장자크 루소</a> 등의 프랑스 작가들이 스위스에 머물며 작품활동을 하기도 하였다. 다만 불어권 <a href="/wiki/%EC%8A%A4%EC%9C%84%EC%8A%A4%EC%9D%98_%EC%A3%BC" title="스위스의 주">칸톤</a>들은 일체적 정체성이 부족하고 이들이 19세기 초에 스위스의 일부가 되기 전까지 스위스는 사실상 독어권 국가였으므로, 프랑스 문학과 별개로 볼 수 있는 스위스 불문학 전통이 존재하는가는 논란의 대상이 되고있다.</p>
<h2><span class="mw-headline" id=".EB.B9.84.EC.9C.A0.EB.9F.BD_.EB.B6.88.EB.AC.B8.ED.95.99">비유럽 불문학</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%B6%88%EB%AC%B8%ED%95%99&amp;action=edit&amp;section=5" title="부분 편집: 비유럽 불문학">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>프랑스가 <a href="/wiki/%EB%B6%81%EC%95%84%EB%A9%94%EB%A6%AC%EC%B9%B4" title="북아메리카">북미</a>대륙과 <a href="/wiki/%EC%95%84%ED%94%84%EB%A6%AC%EC%B9%B4" title="아프리카">아프리카</a>대륙 등지에 <a href="/wiki/%EC%8B%9D%EB%AF%BC%EC%A7%80" title="식민지">식민지</a>를 건설하면서 유럽대륙 외에서도 불어로 문학작품이 쓰이게 되었다. 유럽권의 불어와는 달리 보통 식민지의 불어는 상당히 다른 양태를 띠게 되므로, 그러한 차이에서 빚어지는 창작적 갈등은 특히 퀘벡 문학에 잘 드러나 있으며, 아프리카 제국 등의 작가들은 자신의 모국어와 불어를 혼합하여 창작하려는 시도를 하기도 하였다.</p>

<li><a class="new" href="/w/index.php?title=%EC%95%84%EC%B9%B4%EB%94%94_%EB%AC%B8%ED%95%99&amp;action=edit&amp;redlink=1" title="아카디 문학 (없는 문서)">아카디 문학</a></li>
</ul>
<p>캐나다는 초기에 프랑스의 식민지였다가 영국에게 넘어간 곳으로, 현재도 퀘벡 주 및 동부의 기타 지역에서는 불어가 쓰이고 있다. 초기 <a class="mw-redirect" href="/wiki/%EB%88%84%EB%B2%A8_%ED%94%84%EB%9E%91%EC%8A%A4" title="누벨 프랑스">누벨 프랑스</a>에는 식민지 개척자들의 탐험문학이 주를 이루었고, 대영제국 치하 초기에는 불어 출판물의 인쇄소조차 없었기 때문에 문학전통이 미약했으나, 1830년대에 첫 캐나다 불어 소설이 출판되고 문학활동이 활발해졌으며, 1960년대 <a class="new" href="/w/index.php?title=%EC%A1%B0%EC%9A%A9%ED%95%9C_%ED%98%81%EB%AA%85&amp;action=edit&amp;redlink=1" title="조용한 혁명 (없는 문서)">조용한 혁명</a>이라는 사회변혁기를 맞이하면서 퀘벡의 문학전통을 기록, 연구하고자 하는 의지도 강해졌고, 또 주류 프랑스어의 문체만이 아니라 <a href="/wiki/%ED%80%98%EB%B2%A1_%ED%94%84%EB%9E%91%EC%8A%A4%EC%96%B4" title="퀘벡 프랑스어">퀘벡 프랑스어</a> 및 노동자계층 방언인 <a class="new" href="/w/index.php?title=%EC%A3%BC%EC%95%8C&amp;action=edit&amp;redlink=1" title="주알 (없는 문서)">주알</a>(Joual)로 창작하려는 시도도 이루어졌다. 일반적으로 캐나다의 불문학은 퀘벡 문학을 일컬으나 또다른 프랑스 식민지였던 <a href="/wiki/%EC%95%84%EC%B9%B4%EB%94%94" title="아카디">아카디</a>에서도 <a class="new" href="/w/index.php?title=%EC%95%84%EC%B9%B4%EB%94%94_%EB%AC%B8%ED%95%99&amp;action=edit&amp;redlink=1" title="아카디 문학 (없는 문서)">아카디 문학</a> 전통이 있으며 <a class="new" href="/w/index.php?title=%EC%95%99%ED%86%A0%EB%8B%8C_%EB%A7%88%EC%9D%B4%EC%98%88&amp;action=edit&amp;redlink=1" title="앙토닌 마이예 (없는 문서)">앙토닌 마이예</a> 같은 작가가 대표적이다.</p>
<h3><span class="mw-headline" id=".EB.A7.88.EA.B7.B8.EB.A0.88.EB.B8.8C_.EB.B6.88.EB.AC.B8.ED.95.99">마그레브 불문학</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%B6%88%EB%AC%B8%ED%95%99&amp;action=edit&amp;section=7" title="부분 편집: 마그레브 불문학">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<ul>
<li><a class="new" href="/w/index.php?title=%EB%AA%A8%EB%A1%9C%EC%BD%94_%EB%AC%B8%ED%95%99&amp;action=edit&amp;redlink=1" title="모로코 문학 (없는 문서)">모로코 문학</a></li>
<li><a class="new" href="/w/index.php?title=%EC%95%8C%EC%A0%9C%EB%A6%AC_%EB%AC%B8%ED%95%99&amp;action=edit&amp;redlink=1" title="알제리 문학 (없는 문서)">알제리 문학</a></li>
<li><a class="new" href="/w/index.php?title=%ED%8A%80%EB%8B%88%EC%A7%80_%EB%AC%B8%ED%95%99&amp;action=edit&amp;redlink=1" title="튀니지 문학 (없는 문서)">튀니지 문학</a></li>
</ul>
<p>프랑스의 식민지였던 <a href="/wiki/%EB%A7%88%EA%B7%B8%EB%A0%88%EB%B8%8C" title="마그레브">마그레브</a>의 세 국가에서는 20세기 초부터 프랑스어로 문학작품이 쓰여지기 시작했으며 이 불문학의 전통은 현재까지도 <a href="/wiki/%EC%95%84%EB%9E%8D%EC%96%B4" title="아랍어">아랍어</a> 문학과 더불어 계속되고 있다.</p>
<h3><span class="mw-headline" id=".ED.9D.91.EC.9D.B8_.EB.B6.88.EB.AC.B8.ED.95.99">흑인 불문학</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%B6%88%EB%AC%B8%ED%95%99&amp;action=edit&amp;section=8" title="부분 편집: 흑인 불문학">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<ul>
<li><a href="/wiki/%EB%84%A4%EA%B7%B8%EB%A6%AC%ED%8A%80%EB%93%9C" title="네그리튀드">네그리튀드</a></li>
</ul>
<p><a href="/wiki/%EC%84%B8%EB%84%A4%EA%B0%88" title="세네갈">세네갈</a>이나 <a class="mw-disambig" href="/wiki/%EC%BD%A9%EA%B3%A0" title="콩고">콩고</a> 등 사하라 이남의 프랑스, 벨기에 식민지에서 2차대전 이후로 흑인 불문학 작가들 사이에 반식민지주의, 흑인민족주의 등을 주제로 한 문학활동이 일어났으며 이를 네그리튀드라 한다. <a class="new" href="/w/index.php?title=%EC%97%90%EB%A9%94_%EC%84%B8%EC%9E%90%EB%A5%B4&amp;action=edit&amp;redlink=1" title="에메 세자르 (없는 문서)">에메 세자르</a>나 <a href="/wiki/%EB%A0%88%EC%98%A4%ED%8F%B4_%EC%84%B8%EB%8B%A4%EB%A5%B4_%EC%83%81%EA%B3%A0%EB%A5%B4" title="레오폴 세다르 상고르">레오폴 세다르 상고르</a> 등이 대표적이다.</p>
<h3><span class="mw-headline" id=".EA.B7.B8_.EC.99.B8">그 외</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%B6%88%EB%AC%B8%ED%95%99&amp;action=edit&amp;section=9" title="부분 편집: 그 외">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p><a href="/wiki/%EC%95%84%EC%9D%B4%ED%8B%B0" title="아이티">아이티</a>나 <a href="/wiki/%EB%A7%88%EB%A5%B4%ED%8B%B0%EB%8B%88%ED%81%AC" title="마르티니크">마르티니크</a> 등 프랑스의 <a class="new" href="/w/index.php?title=%EC%95%A4%ED%8B%B8%EB%9F%AC%EC%8A%A4&amp;action=edit&amp;redlink=1" title="앤틸러스 (없는 문서)">앤틸러스</a> 구식민지나 해외영토인 <a href="/wiki/%EB%A0%88%EC%9C%84%EB%8B%88%EC%98%B9" title="레위니옹">레위니옹</a> 등에서도 프랑스 문학과 구분되는 문학활동이 존재하고 있다.</p>
