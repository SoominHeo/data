<h1 class="firstHeading" id="firstHeading" lang="ko">불충실한 선거인단</h1>
<div class="mw-body-content" id="bodyContent">
<div id="siteSub">위키백과, 우리 모두의 백과사전.</div>
<div id="contentSub"></div>

<span style="margin:0px; padding-bottom:1px; font-size:90%; display:block;"><span style="border: 1px solid; border: black; background-color:red; color:white">    </span> 불성실한 선거인단을 처벌하는 법이 유효한 주</span></div>
</div>

<p><b>불충실한 선거인단</b>(<span style="font-size: smaller;"><a href="/wiki/%EC%98%81%EC%96%B4" title="영어">영어</a>:</span> <span lang="en" xml:lang="en">Faithless elector</span>)은 미국의 대통령 선거에서 <a href="/wiki/%EC%84%A0%EA%B1%B0%EC%9D%B8%EB%8B%A8" title="선거인단">선거인단</a>이 어떠한 이유로 자신이 투표하여야 할 대통령 후보자나 부통령 후보자에게 투표하지 않는 것을 일컫는 용어이다. 이 경우, 다른 후보자에게 투표하거나 투표를 하지 않는 것 모두 포함한다.</p>
<p>선거인단은 보통 정당이나 정당의 대통령 후보자가 선택하거나 지명하는데, 이들은 일반적으로 선거 후보자나 정당에서 높은 신임을 받는 당원이다. 불충실한 선거인단은 당의 질책을 받거나 정치적 보복을 받을 위험이 있다. 몇몇 주에서는 이러한 불충실한 선거인단을 잠재적인 범죄라고 규정하기도 한다. 선거인단 후보자는 선거일보다 몇 달 앞서서 주 정당에서 선출한다. 몇몇 주에서는 예비선거에서 선거인단을 선출하여, 같은 방식으로 다른 선거인단을 지명하는 방식을 사용하기도 한다. <a href="/wiki/%EC%98%A4%ED%81%B4%EB%9D%BC%ED%98%B8%EB%A7%88_%EC%A3%BC" title="오클라호마 주">오클라호마 주</a>, <a href="/wiki/%EB%B2%84%EC%A7%80%EB%8B%88%EC%95%84_%EC%A3%BC" title="버지니아 주">버지니아 주</a>, <a class="mw-redirect" href="/wiki/%EB%85%B8%EC%8A%A4_%EC%BA%90%EB%A1%A4%EB%9D%BC%EC%9D%B4%EB%82%98_%EC%A3%BC" title="노스 캐롤라이나 주">노스 캐롤라이나 주</a>와 같은 몇몇 주에서는 정당대회에서 선거인단을 선출한다. <a href="/wiki/%ED%8E%9C%EC%8B%A4%EB%B2%A0%EC%9D%B4%EB%8B%88%EC%95%84_%EC%A3%BC" title="펜실베이니아 주">펜실베이니아 주</a>에서는 선거위원회에서 선거인단을 위한 후보자를 지명한다.<sup class="reference" id="cite_ref-ElectoralVote_1-0"><a href="#cite_note-ElectoralVote-1">[1]</a></sup></p>
<p>역사상 157번의 불충실한 선거인이 발생하였으나, 대통령 선거의 결과를 뒤집은 적은 아직까지 없다. 1836년에는 <a href="/wiki/%EB%B2%84%EC%A7%80%EB%8B%88%EC%95%84_%EC%A3%BC" title="버지니아 주">버지니아 주</a>의 선거인 23명이 인종간 결혼이 부도덕하다는 <a class="mw-redirect" href="/wiki/%EC%9D%B8%EC%A2%85%EC%B0%A8%EB%B3%84" title="인종차별">인종차별</a>적인 생각에 따라 자신이 투표하여야 할 부통령 후보 <a href="/wiki/%EB%A6%AC%EC%B2%98%EB%93%9C_%EB%A9%98%ED%84%B0_%EC%A1%B4%EC%8A%A8" title="리처드 멘터 존슨">리처드 멘터 존슨</a>에게 투표하지 않는 사건이 발생하였다. 그러나 어느 부통령 후보도 선거인단 선거에서 과반 득표를 하지 못했기 때문에 상원에서 존슨을 부통령으로 선출하면서 결과적으로 불충실한 선거인이 선거 결과를 뒤집지 못하였다.</p>
<h2><span class="mw-headline" id=".EB.B2.95.EC.A0.81.EC.9D.B8_.EC.9E.85.EC.9E.A5">법적인 입장</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%B6%88%EC%B6%A9%EC%8B%A4%ED%95%9C_%EC%84%A0%EA%B1%B0%EC%9D%B8%EB%8B%A8&amp;action=edit&amp;section=1" title="부분 편집: 법적인 입장">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>21개의 주는 선거인단에게 지정된 후보자에게 투표하도록 강제하는 법안이 없다.<sup class="reference" id="cite_ref-2"><a href="#cite_note-2">[2]</a></sup> 29개의 주(컬럼비아 구를 포함)는 불충실한 선거인단을 처벌하는 법안이 있다.<sup class="reference" id="cite_ref-3"><a href="#cite_note-3">[3]</a></sup> 이 때에 투표는 강제성이 없어 처벌은 받되 투표는 유효하다.<sup class="reference" id="cite_ref-faithless_4-0"><a href="#cite_note-faithless-4">[4]</a></sup> 그러나 <a href="/wiki/%EB%AF%B8%EC%8B%9C%EA%B0%84_%EC%A3%BC" title="미시간 주">미시간 주</a>나 <a href="/wiki/%EB%AF%B8%EB%84%A4%EC%86%8C%ED%83%80_%EC%A3%BC" title="미네소타 주">미네소타 주</a>와 같은 몇몇 주에서는 불성실한 선거인단의 투표를 무효로 하고 있다.<sup class="reference" id="cite_ref-5"><a href="#cite_note-5">[5]</a></sup></p>
<p>2008년까지 미네소타 주 선거인단은 비밀투표로 진행되었는데, 이러한 이유로 누가 불충실한 선거인인지 알 수가 없었다. 2004년에 선거인단 중 누군가가 불충실한 선거를 하였는데, 미네소타 주 법은 대중에게 선거인단의 투표를 제공하고, 불충실한 선거인의 투표를 무효로 하도록 개정하였다.<sup class="reference" id="cite_ref-6"><a href="#cite_note-6">[6]</a></sup></p>
<p>이러한 법률들의 합헌성은 1952년 <a class="new" href="/w/index.php?title=%EB%A0%88%EC%9D%B4_%EB%8C%80_%EB%B8%94%EB%A0%88%EC%96%B4_%EC%82%AC%EA%B1%B4&amp;action=edit&amp;redlink=1" title="레이 대 블레어 사건 (없는 문서)">레이 대 블레어 사건</a>에 대한 미국 대법원의 판결에서 나타난다. 법원은 선거인단에게 지정된 후보자에게 투표하도록 하는 주의 권리에 호의적이었으며, 다른 후보자에게 투표한, 즉 불충실한 선거인단의 표를 무효하는 것에도 호의적으로 판결하였다. 한 번 선거인단이 투표를 하면, 그들의 투표를 주가 변경할 수 있는데, <a href="/wiki/%EB%AF%B8%EC%8B%9C%EA%B0%84_%EC%A3%BC" title="미시간 주">미시간 주</a>나 <a href="/wiki/%EB%AF%B8%EB%84%A4%EC%86%8C%ED%83%80_%EC%A3%BC" title="미네소타 주">미네소타 주</a>처럼 불충실한 선거인단의 표를 무효로 하는 주들이 대표적인 경우이다. 24개 주에서 불충실한 선거인은 선거인이 투표를 한 이후에 처벌받을 수 있도록 하는데, 미국 대법원은 주 선거를 통해 선발된 선거인단은 연방 정부가 아니라 주의 의견을 반영하여 행동하여야 한다고 규정하고 있다. 그러므로 각각의 주는 선거인단을 규제할 권리를 가지며, 불충실한 선거인단을 처벌하기 위한 주 법률의 합헌성은 미국 대법원이 판결하거나 결정할 수 있는 부분이 아니다.</p>
<h2><span class="mw-headline" id=".EC.82.AC.EB.A1.80">사례</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%B6%88%EC%B6%A9%EC%8B%A4%ED%95%9C_%EC%84%A0%EA%B1%B0%EC%9D%B8%EB%8B%A8&amp;action=edit&amp;section=2" title="부분 편집: 사례">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<ul>
<li><b><a href="/wiki/1796%EB%85%84_%EB%AF%B8%EA%B5%AD_%EB%8C%80%ED%86%B5%EB%A0%B9_%EC%84%A0%EA%B1%B0" title="1796년 미국 대통령 선거">1796년 미국 대통령 선거</a></b> - <a href="/wiki/%ED%8E%9C%EC%8B%A4%EB%B2%A0%EC%9D%B4%EB%8B%88%EC%95%84_%EC%A3%BC" title="펜실베이니아 주">펜실베이니아 주</a>의 선거인 새뮤얼 마일스는 선거에 따라 대통령은 <a href="/wiki/%EC%97%B0%EB%B0%A9%EB%8B%B9" title="연방당">연방당</a>의 <a href="/wiki/%EC%A1%B4_%EC%95%A0%EB%8D%A4%EC%8A%A4" title="존 애덤스">존 애덤스</a>에 투표해야 했으나 <a href="/wiki/%EB%AF%BC%EC%A3%BC%EA%B3%B5%ED%99%94%EB%8B%B9_(%EB%AF%B8%EA%B5%AD)" title="민주공화당 (미국)">민주공화당</a>의 <a href="/wiki/%ED%86%A0%EB%A8%B8%EC%8A%A4_%EC%A0%9C%ED%8D%BC%EC%8A%A8" title="토머스 제퍼슨">토머스 제퍼슨</a>에게 투표하였다. 그는 다른 표에서도 <a href="/wiki/%EC%B0%B0%EC%8A%A4_%ED%95%91%ED%81%AC%EB%8B%88" title="찰스 핑크니">찰스 핑크니</a>를 택하였다. <a href="/wiki/%EB%AF%B8%EA%B5%AD_%EC%88%98%EC%A0%95_%ED%97%8C%EB%B2%95_%EC%A0%9C12%EC%A1%B0" title="미국 수정 헌법 제12조">수정 헌법 12조</a>의 발효 전이라 선거인단은 2명에 투표할 수 있었다.</li>
<li><b><a href="/wiki/1808%EB%85%84_%EB%AF%B8%EA%B5%AD_%EB%8C%80%ED%86%B5%EB%A0%B9_%EC%84%A0%EA%B1%B0" title="1808년 미국 대통령 선거">1808년 미국 대통령 선거</a></b> - <a href="/wiki/%EB%89%B4%EC%9A%95_%EC%A3%BC" title="뉴욕 주">뉴욕 주</a>의 선거인 6명은 선거에 따라 대통령은 <a href="/wiki/%EB%AF%BC%EC%A3%BC%EA%B3%B5%ED%99%94%EB%8B%B9_(%EB%AF%B8%EA%B5%AD)" title="민주공화당 (미국)">민주공화당</a>의 <a href="/wiki/%EC%A0%9C%EC%9E%84%EC%8A%A4_%EB%A7%A4%EB%94%94%EC%8A%A8" title="제임스 매디슨">제임스 매디슨</a>에게, 부통령은 <a href="/wiki/%EC%A1%B0%EC%A7%80_%ED%81%B4%EB%A6%B0%ED%84%B4" title="조지 클린턴">조지 클린턴</a>에게 투표해야 했으나 3명은 대통령에 클린턴, 부통령은 매디슨에 투표했으며, 다른 3명은 대통령에 클린턴, 부통령은 <a href="/wiki/%EC%A0%9C%EC%9E%84%EC%8A%A4_%EB%A8%BC%EB%A1%9C" title="제임스 먼로">제임스 먼로</a>에 투표하였다.</li>
<li><b><a href="/wiki/1972%EB%85%84_%EB%AF%B8%EA%B5%AD_%EB%8C%80%ED%86%B5%EB%A0%B9_%EC%84%A0%EA%B1%B0" title="1972년 미국 대통령 선거">1972년 미국 대통령 선거</a></b> - <a href="/wiki/%EB%B2%84%EC%A7%80%EB%8B%88%EC%95%84_%EC%A3%BC" title="버지니아 주">버지니아 주</a>의 선거인 <a class="new" href="/w/index.php?title=%EB%A1%9C%EC%A0%80_%EB%A7%A5%EB%B8%8C%EB%9D%BC%EC%9D%B4%EB%93%9C&amp;action=edit&amp;redlink=1" title="로저 맥브라이드 (없는 문서)">로저 맥브라이드</a>는 선거에 따라 대통령은 <a href="/wiki/%EA%B3%B5%ED%99%94%EB%8B%B9_(%EB%AF%B8%EA%B5%AD)" title="공화당 (미국)">공화당</a>의 <a href="/wiki/%EB%A6%AC%EC%B2%98%EB%93%9C_%EB%8B%89%EC%8A%A8" title="리처드 닉슨">리처드 닉슨</a>에게 투표해야 했으나 군소 정당인 <a href="/wiki/%EC%9E%90%EC%9C%A0%EB%8B%B9_(%EB%AF%B8%EA%B5%AD)" title="자유당 (미국)">자유당</a>의 <a class="new" href="/w/index.php?title=%EC%A1%B4_%ED%98%B8%EC%8A%A4%ED%8D%BC%EC%8A%A4&amp;action=edit&amp;redlink=1" title="존 호스퍼스 (없는 문서)">존 호스퍼스</a>에게 투표하였다. 맥브라이드의 불충실표에 따라 같은 당 부통령 후보였던 <a class="new" href="/w/index.php?title=%EC%8B%9C%EC%96%B4%EB%8F%84%EB%9D%BC_%EB%82%98%EC%82%B0&amp;action=edit&amp;redlink=1" title="시어도라 나산 (없는 문서)">시어도라 나산</a>은 미국 최초로 선거인단을 확보한 여성 후보가 되었다. 맥브라이드는 4년 뒤에 자유당 소속으로 대선에 출마하였다.</li>
<li><b><a href="/wiki/1976%EB%85%84_%EB%AF%B8%EA%B5%AD_%EB%8C%80%ED%86%B5%EB%A0%B9_%EC%84%A0%EA%B1%B0" title="1976년 미국 대통령 선거">1976년 미국 대통령 선거</a></b> - <a href="/wiki/%EC%9B%8C%EC%8B%B1%ED%84%B4_%EC%A3%BC" title="워싱턴 주">워싱턴 주</a>의 선거인 마이크 패든은 선거에 따라 대통령은 <a href="/wiki/%EA%B3%B5%ED%99%94%EB%8B%B9_(%EB%AF%B8%EA%B5%AD)" title="공화당 (미국)">공화당</a>의 <a href="/wiki/%EC%A0%9C%EB%9F%B4%EB%93%9C_%ED%8F%AC%EB%93%9C" title="제럴드 포드">제럴드 포드</a>에게 투표해야 했으나 공화당 예비경선에서의 경쟁자였던 <a href="/wiki/%EB%A1%9C%EB%84%90%EB%93%9C_%EB%A0%88%EC%9D%B4%EA%B1%B4" title="로널드 레이건">로널드 레이건</a>에게 투표하였다.</li>
<li><b><a href="/wiki/1988%EB%85%84_%EB%AF%B8%EA%B5%AD_%EB%8C%80%ED%86%B5%EB%A0%B9_%EC%84%A0%EA%B1%B0" title="1988년 미국 대통령 선거">1988년 미국 대통령 선거</a></b> - <a href="/wiki/%EC%9B%A8%EC%8A%A4%ED%8A%B8%EB%B2%84%EC%A7%80%EB%8B%88%EC%95%84_%EC%A3%BC" title="웨스트버지니아 주">웨스트버지니아 주</a>의 선거인 마거릿 리치는 선거에 따라 대통령은 <a href="/wiki/%EB%AF%BC%EC%A3%BC%EB%8B%B9_(%EB%AF%B8%EA%B5%AD)" title="민주당 (미국)">민주당</a>의 <a href="/wiki/%EB%A7%88%EC%9D%B4%ED%81%B4_%EB%93%80%EC%B9%B4%ED%82%A4%EC%8A%A4" title="마이클 듀카키스">마이클 듀카키스</a>에게, 부통령은 <a class="new" href="/w/index.php?title=%EB%A1%9C%EC%9D%B4%EB%93%A0_%EB%B2%A4%EC%8A%A8&amp;action=edit&amp;redlink=1" title="로이든 벤슨 (없는 문서)">로이든 벤슨</a>에게 투표해야 했으나 대통령은 벤슨에게, 부통령은 듀카키스에게 역투표하였다.</li>
<li><b><a href="/wiki/2000%EB%85%84_%EB%AF%B8%EA%B5%AD_%EB%8C%80%ED%86%B5%EB%A0%B9_%EC%84%A0%EA%B1%B0" title="2000년 미국 대통령 선거">2000년 미국 대통령 선거</a></b> - <a href="/wiki/%EC%9B%8C%EC%8B%B1%ED%84%B4_D.C." title="워싱턴 D.C.">워싱턴 D.C.</a>의 선거인 <a class="new" href="/w/index.php?title=%EB%B0%94%EB%B2%84%EB%9D%BC_%EB%A0%88%ED%8A%B8%EC%8B%9C%EB%AA%AC%EC%A6%88&amp;action=edit&amp;redlink=1" title="바버라 레트시몬즈 (없는 문서)">바버라 레트시몬즈</a>는 선거에 따라 대통령은 <a href="/wiki/%EB%AF%BC%EC%A3%BC%EB%8B%B9_(%EB%AF%B8%EA%B5%AD)" title="민주당 (미국)">민주당</a>의 <a href="/wiki/%EC%95%A8_%EA%B3%A0%EC%96%B4" title="앨 고어">앨 고어</a>에게, 부통령은 <a href="/wiki/%EC%A1%B0_%EB%A6%AC%EB%B2%84%EB%A8%BC" title="조 리버먼">조 리버먼</a>에게 투표해야 했으나 백지표를 제출하였다. 그녀는 제출 사유로 워싱턴 D.C.가 미국 본토에 해당됨에도 입법 대표를 선출할 수 없는 상황을 비판하기 위해서라고 답했다.</li>
<li><b><a href="/wiki/2004%EB%85%84_%EB%AF%B8%EA%B5%AD_%EB%8C%80%ED%86%B5%EB%A0%B9_%EC%84%A0%EA%B1%B0" title="2004년 미국 대통령 선거">2004년 미국 대통령 선거</a></b> - <a href="/wiki/%EB%AF%B8%EB%84%A4%EC%86%8C%ED%83%80_%EC%A3%BC" title="미네소타 주">미네소타 주</a>의 한 선거인은 선거에 따라 대통령은 <a href="/wiki/%EB%AF%BC%EC%A3%BC%EB%8B%B9_(%EB%AF%B8%EA%B5%AD)" title="민주당 (미국)">민주당</a>의 <a href="/wiki/%EC%A1%B4_%EC%BC%80%EB%A6%AC" title="존 케리">존 케리</a>에게, 부통령은 <a href="/wiki/%EC%A1%B4_%EC%97%90%EB%93%9C%EC%9B%8C%EC%A6%88" title="존 에드워즈">존 에드워즈</a>에게 투표해야 했으나 대통령과 부통령 모두 에드워즈에게 투표하였다. 이 투표는 선거인의 착각으로 발생하였다 여겨지는데, 이는 미네소타 주는 선거인단이 비밀 투표를 하기 때문이다.<sup class="reference" id="cite_ref-7"><a href="#cite_note-7">[7]</a></sup> 이 문제는 지역 내에서 한동안 논란을 일으켰으며 어떤 선거인이 불충실표를 던졌는지 밝혀내기 위한 시도는 모두 실패했다. 미네소타 주의회는 이 선거 이후 선거인단의 투표를 공개 투표로 변경하였다.<sup class="reference" id="cite_ref-8"><a href="#cite_note-8">[8]</a></sup></li>
<li><b><a href="/wiki/2016%EB%85%84_%EB%AF%B8%EA%B5%AD_%EB%8C%80%ED%86%B5%EB%A0%B9_%EC%84%A0%EA%B1%B0" title="2016년 미국 대통령 선거">2016년 미국 대통령 선거</a></b> - 7명의 선거인이 서약한 후보와 다른 후보에게 투표했다. <a href="/wiki/%EC%9B%8C%EC%8B%B1%ED%84%B4_%EC%A3%BC" title="워싱턴 주">워싱턴 주</a>의 선거인 세 명은 선출하기로 한 <a href="/wiki/%ED%9E%90%EB%9F%AC%EB%A6%AC_%ED%81%B4%EB%A6%B0%ED%84%B4" title="힐러리 클린턴">힐러리 클린턴</a> 대신 <a href="/wiki/%EC%BD%9C%EB%A6%B0_%ED%8C%8C%EC%9B%94" title="콜린 파월">콜린 파월</a> 전 장관을 대통령으로 투표하였다. 이들은 각각 <a class="new" href="/w/index.php?title=%EC%88%98%EC%A0%84_%EC%BD%9C%EB%A6%B0%EC%8A%A4&amp;action=edit&amp;redlink=1" title="수전 콜린스 (없는 문서)">수전 콜린스</a>(Susan Collins), <a class="new" href="/w/index.php?title=%EB%A7%88%EB%A6%AC%EC%95%84_%EC%BA%94%ED%8A%B8%EC%9B%B0&amp;action=edit&amp;redlink=1" title="마리아 캔트웰 (없는 문서)">마리아 캔트웰</a>(Maria Cantwell), <a href="/wiki/%EC%97%98%EB%A6%AC%EC%9E%90%EB%B2%A0%EC%8A%A4_%EC%9B%8C%EB%9F%B0" title="엘리자베스 워런">엘리자베스 워런</a>을 부통령으로 투표하였다. <a href="/wiki/%ED%95%98%EC%99%80%EC%9D%B4_%EC%A3%BC" title="하와이 주">하와이 주</a>의 한 선거인은 선출하기로 한 클린턴 대신 <a href="/wiki/%EB%B2%84%EB%8B%88_%EC%83%8C%EB%8D%94%EC%8A%A4" title="버니 샌더스">버니 샌더스</a>를 대통령으로 워런을 부통령으로 투표하였다. 워싱턴 주의 또다른 한 명은 선출하기로 한 클린턴 대신 미국 원주민 환경운동가 <a href="/wiki/%ED%8E%98%EC%9D%B4%EC%8A%A4_%EC%8A%A4%ED%8C%8C%ED%8B%B0%EB%93%9C_%EC%9D%B4%EA%B8%80" title="페이스 스파티드 이글">페이스 스파티드 이글</a>(Faith Spotted Eagle)을 대통령으로, 원주민과 백인의 혼혈인 환경운동가 <a href="/wiki/%EC%9C%84%EB%85%B8%EC%95%84_%EB%9D%BC%EB%91%90%ED%81%AC" title="위노아 라두크">위노아 라두크</a>를 부통령으로 투표하였다. <a href="/wiki/%ED%85%8D%EC%82%AC%EC%8A%A4_%EC%A3%BC" title="텍사스 주">텍사스 주</a>의 한 선거인은 <a href="/wiki/%EC%A1%B4_%EC%BC%80%EC%9D%B4%EC%8B%9D" title="존 케이식">존 케이식</a>을, 한 선거인은 <a href="/wiki/%EB%A1%A0_%ED%8F%B4" title="론 폴">론 폴</a>을 대통령으로 투표하였고 한 명은 <a href="/wiki/%EC%B9%BC%EB%A6%AC_%ED%94%BC%EC%98%A4%EB%A6%AC%EB%82%98" title="칼리 피오리나">칼리 피오리나</a>를 부통령으로 투표하였다. 부통령으로 피오리나를 찍은 텍사스 주의 선거인이 대통령으로 누구를 찍었는지는 알려지지 않았다. 텍사스 주의 선거인은 대통령과 부통령을 각각 다른 투표용지에 적어 비밀투표를 하기 때문이다.</li>
</ul>
