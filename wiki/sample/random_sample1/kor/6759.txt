<p><b>붉은 요새</b>은 '적성(赤城)' 즉 <b>붉은 성</b>이라는 별칭을 가진 성으로서 <a href="/wiki/%EC%9D%B8%EB%8F%84" title="인도">인도</a>의 <a href="/wiki/%EB%8D%B8%EB%A6%AC" title="델리">델리</a>에 있는 성이다. 현재 <a class="mw-redirect" href="/wiki/%EC%9C%A0%EB%84%A4%EC%8A%A4%EC%BD%94" title="유네스코">유네스코</a> 문화유산으로 등재되어 있다.</p>
<h2><span class="mw-headline" id=".EC.97.AD.EC.82.AC">역사</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%B6%89%EC%9D%80_%EC%9A%94%EC%83%88&amp;action=edit&amp;section=1" title="부분 편집: 역사">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<div class="thumb tright">
<div class="thumbinner" style="width:222px;"><a class="image" href="/wiki/%ED%8C%8C%EC%9D%BC:Redfort.JPG"><img alt="" class="thumbimage" data-file-height="2000" data-file-width="3008" height="146" src="//upload.wikimedia.org/wikipedia/commons/thumb/e/ee/Redfort.JPG/220px-Redfort.JPG" srcset="//upload.wikimedia.org/wikipedia/commons/thumb/e/ee/Redfort.JPG/330px-Redfort.JPG 1.5x, //upload.wikimedia.org/wikipedia/commons/thumb/e/ee/Redfort.JPG/440px-Redfort.JPG 2x" width="220"/></a>
<div class="thumbcaption">
<div class="magnify"><a class="internal" href="/wiki/%ED%8C%8C%EC%9D%BC:Redfort.JPG" title="실제 크기로"></a></div>
붉은 요새 사진</div>
</div>
</div>
<p>붉은 성과 성이 위치한 샤자하나바드는 <a href="/wiki/%EC%83%A4_%EC%9E%90%ED%95%9C" title="샤 자한">샤 자한</a> 황제가 조성한 것이다. 성의 기본 도안은 기존의 성 양식을 통합하여 보강하도록 설계되었다. 성곽형태의 궁전은 중요한 도시적 면모를 띠며 인도의 중세 시기를 잘 드러내고 있다. 전체적인 면모와 당시 상황을 고려하면 <a href="/wiki/%EB%AC%B4%EA%B5%B4_%EC%A0%9C%EA%B5%AD" title="무굴 제국">무굴 제국</a>의 활동과 창조성에 대해서도 찾아볼 수 있다.</p>
<p>건물의 일대 변화는 영국 식민 지배하에서의 1차 독립 전쟁 때였다. 독립 이후 구조 변형과 보수에 따라 상당 부분이 교체되었다. 영국 식민 시기 때에는 군의 병영지로 쓰이고 독립 후 <a href="/wiki/2003%EB%85%84" title="2003년">2003년</a>까지 인도 군의 속영 하에 있었다.</p>
<p>붉은 성을 짓도록 한 무굴 황제 <a href="/wiki/%EC%83%A4_%EC%9E%90%ED%95%9C" title="샤 자한">샤 자한</a>은 <a href="/wiki/%EC%95%84%EA%B7%B8%EB%9D%BC_%EC%84%B1" title="아그라 성">아그라 성</a>에서 천도하여 당대의 번영을 뽐내길 원하였다. 그는 천도를 하면서 방대한 건물 계획과 능력을 자축하였다. 시의 동쪽 외곽에 있으며 붉은 성이라는 이름은 성의 거대한 벽이 붉은 빛을 띠고 있어서이다. 전체 벽은 2.5km에 달하며 높이는 차이가 나지만 대개 16~33m이다.</p>
<p>야무나 강변에 서있는 성은 벽의 대부분이 연못으로 둘려 싸여있어 해자 형태를 띤다. 건축은 <a href="/wiki/1638%EB%85%84" title="1638년">1638년</a> 시작되었으며 <a href="/wiki/1648%EB%85%84" title="1648년">1648년</a> 완공되었다.</p>
<p><a href="/wiki/1783%EB%85%84" title="1783년">1783년</a> <a href="/wiki/5%EC%9B%94_11%EC%9D%BC" title="5월 11일">5월 11일</a>, <a href="/wiki/%EC%8B%9C%ED%81%AC%EA%B5%90" title="시크교">시크 교도</a>가 성으로 들어오면서 다와니암이 성을 점유하게 되었다. 붉은 성은 전체적으로 계획의 통일성에 충실하여 지어져 있었다. 하지만 18세기 이후 여러 상해를 입게 되었다. 또한 <a href="/wiki/1857%EB%85%84" title="1857년">1857년</a> 세포이 항쟁 이후 성이 영국군의 본부로 쓰이면서 5분의 4에 해당하는 전각을 해체하였고 정원도 망가뜨려버렸다. 복구 계획이 <a href="/wiki/1903%EB%85%84" title="1903년">1903년</a> 시작되었다.</p>
<h2><span class="mw-headline" id=".EA.B1.B4.EC.B6.95">건축</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%B6%89%EC%9D%80_%EC%9A%94%EC%83%88&amp;action=edit&amp;section=2" title="부분 편집: 건축">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>붉은 성의 건축은 장식과 미술적 양식을 고도로 유지하고 있다는 점에서 주목받는다. 미술적으로는 <a href="/wiki/%EC%9D%B4%EB%9E%80" title="이란">페르시아</a>, <a href="/wiki/%EC%9C%A0%EB%9F%BD" title="유럽">유럽</a>, 인도 미술을 고루 배치하였고 결과적으로는 샤자한 왕 대의 미술 양식을 고도로 표현하는 방식을 낳았다. 색채와 표현법, 형체에 이르기까지 모든 균형을 두루 갖추었다.</p>
<p>때문에 붉은 성은 모든 인도의 건축을 대표할만큼 중요한 건물이다. 시간과 공간을 거슬러 현대에 과거의 시간을 전하는 역할을 하고 있기도 하다. 인도 전체의 번영과 예술적 능력을 상징하기에 <a href="/wiki/1913%EB%85%84" title="1913년">1913년</a>에는 성을 보수할 적에 전국민의 시선이 집중되기도 했었다.</p>
<h2><span class="mw-headline" id=".EA.B7.BC.EB.8C.80">근대</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%B6%89%EC%9D%80_%EC%9A%94%EC%83%88&amp;action=edit&amp;section=3" title="부분 편집: 근대">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>붉은 성은 델리의 가장 대중적인 관광지이기도 하며 매년 수백만의 관광객이 찾는다. 또한 인도의 초대 총리가 영국으로부터의 독립을 선언한 곳이 이곳이다.</p>
<p>원래는 3,000명 정도의 주민이 요새와 인근 단지에 거주하였으나 <a href="/wiki/1857%EB%85%84" title="1857년">1857년</a> 영국의 소유처럼 궁전이 사용되면서 주거 기능이 사라졌다. <a href="/wiki/1947%EB%85%84" title="1947년">1947년</a> 인도가 독립하자 인도군이 성의 소유권을 가져갔다. <a href="/wiki/2003%EB%85%84" title="2003년">2003년</a> 12월 군이 소유권을 인도 관광청에 넘겼다.</p>
