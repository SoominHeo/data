<p><b>엘리자베트 폰 뷔르템베르크</b>(Elisabeth Wilhelmine Louise von Württemberg, <a href="/wiki/1767%EB%85%84" title="1767년">1767년</a> <a href="/wiki/4%EC%9B%94_21%EC%9D%BC" title="4월 21일">4월 21일</a> ~ <a href="/wiki/1790%EB%85%84" title="1790년">1790년</a> <a href="/wiki/2%EC%9B%94_18%EC%9D%BC" title="2월 18일">2월 18일</a>)는 <a href="/wiki/%EC%8B%A0%EC%84%B1_%EB%A1%9C%EB%A7%88_%EC%A0%9C%EA%B5%AD" title="신성 로마 제국">신성 로마 제국</a>의 황제 <a href="/wiki/%ED%94%84%EB%9E%80%EC%B8%A0_2%EC%84%B8" title="프란츠 2세">프란츠 2세</a>의 첫 번째 아내이다.</p>
<h2><span class="mw-headline" id=".EC.83.9D.EC.95.A0">생애</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%B7%94%EB%A5%B4%ED%85%9C%EB%B2%A0%EB%A5%B4%ED%81%AC_%EA%B3%B5%EB%85%80_%EC%97%98%EB%A6%AC%EC%9E%90%EB%B2%A0%ED%8A%B8&amp;action=edit&amp;section=1" title="부분 편집: 생애">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>뷔르텐베르크 공 프리드리히 2세 오이겐의 셋째딸로, 형제들로는 <a class="new" href="/w/index.php?title=%EB%B7%94%EB%A5%B4%ED%85%9C%EB%B2%A0%EB%A5%B4%ED%81%AC%EC%9D%98_%ED%94%84%EB%A6%AC%EB%93%9C%EB%A6%AC%ED%9E%88_1%EC%84%B8&amp;action=edit&amp;redlink=1" title="뷔르템베르크의 프리드리히 1세 (없는 문서)">뷔르템베르크의 프리드리히 1세</a>, <a class="mw-redirect" href="/wiki/%EB%9F%AC%EC%8B%9C%EC%95%84%EC%9D%98_%ED%99%A9%ED%9B%84" title="러시아의 황후">러시아의 황후</a> <a href="/wiki/%EB%A7%88%EB%A6%AC%EC%95%84_%ED%91%9C%EB%8F%84%EB%A1%9C%EB%B8%8C%EB%82%98_(1759%EB%85%84)" title="마리아 표도로브나 (1759년)">마리아 표도로브나</a> 등이 있다. <a href="/wiki/%EC%9A%94%EC%A0%9C%ED%94%84_2%EC%84%B8" title="요제프 2세">요제프 2세</a>는 엘리자베트가 열 다섯 살이 되던 해, 그녀를 <a href="/wiki/%EB%B9%88" title="빈">빈</a>으로 불러들여 <a href="/wiki/%EA%B0%80%ED%86%A8%EB%A6%AD" title="가톨릭">가톨릭</a>으로 개종시키고 수준높은 교육을 시켰다. <a href="/wiki/1788%EB%85%84" title="1788년">1788년</a> <a href="/wiki/1%EC%9B%94_6%EC%9D%BC" title="1월 6일">1월 6일</a> 엘리자베트는 요제프 2세의 조카 프란츠 대공과 결혼했고 그녀의 젊음과 활기는 늙고 쇠약한 요제프 2세의 만년을 즐겁게 했다. 그러나 이듬해인 <a href="/wiki/1789%EB%85%84" title="1789년">1789년</a> 엘리자베트는 딸 루도비카 엘리자베트를 낳고 산욕으로 죽었다. 루도비카 엘리자베트 또한 태어난 지 16개월만에 요절했다.</p>
