<h2><span class="mw-headline" id=".EA.B2.BD.EB.A0.A5">경력</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%B8%8C%EB%9D%BC%EC%9D%B4%EC%96%B8_%EA%B3%A0%EB%93%A0&amp;action=edit&amp;section=1" title="부분 편집: 경력">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>브라이언 고든은 <a href="/wiki/1997%EB%85%84" title="1997년">1997년</a> <a href="/wiki/%EC%95%A0%EB%A6%AC%EC%A1%B0%EB%82%98_%EB%8B%A4%EC%9D%B4%EC%95%84%EB%AA%AC%EB%93%9C%EB%B0%B1%EC%8A%A4" title="애리조나 다이아몬드백스">애리조나 다이아몬드백스</a> 산하 <a href="/wiki/%EB%A7%88%EC%9D%B4%EB%84%88_%EB%A6%AC%EA%B7%B8_%EB%B2%A0%EC%9D%B4%EC%8A%A4%EB%B3%BC" title="마이너 리그 베이스볼">마이너 리그</a> 팀인 모빌 베이베어스에 지명을 받아 <a href="/wiki/%EC%99%B8%EC%95%BC%EC%88%98" title="외야수">외야수</a>로 입단했다. 그는 <a href="/wiki/2006%EB%85%84" title="2006년">2006년</a>까지 <a href="/wiki/%EC%99%B8%EC%95%BC%EC%88%98" title="외야수">외야수</a>로 활동하다가 <a href="/wiki/%EB%A9%94%EC%9D%B4%EC%A0%80_%EB%A6%AC%EA%B7%B8_%EB%B2%A0%EC%9D%B4%EC%8A%A4%EB%B3%BC" title="메이저 리그 베이스볼">메이저 리그</a> <a href="/wiki/%ED%88%AC%EC%88%98" title="투수">투수</a> 출신인 <a href="/wiki/%EB%86%80%EB%9F%B0_%EB%9D%BC%EC%9D%B4%EC%96%B8" title="놀런 라이언">놀런 라이언</a> 선수의 조언으로 <a href="/wiki/2007%EB%85%84" title="2007년">2007년</a> 시즌부터 <a href="/wiki/%ED%88%AC%EC%88%98" title="투수">투수</a>로 전향했다. 이후 <a href="/wiki/2011%EB%85%84" title="2011년">2011년</a>에 리하이 밸리 아이언 피즈(<a href="/wiki/%ED%95%84%EB%9D%BC%EB%8D%B8%ED%94%BC%EC%95%84_%ED%95%84%EB%A6%AC%EC%8A%A4" title="필라델피아 필리스">필라델피아 필리스</a> 산하 <a class="mw-disambig" href="/wiki/%ED%8A%B8%EB%A6%AC%ED%94%8C_A" title="트리플 A">트리플 A</a> 팀), <a href="/wiki/%EB%89%B4%EC%9A%95_%EC%96%91%ED%82%A4%EC%8A%A4" title="뉴욕 양키스">뉴욕 양키스</a>에서 활동하다가 <a href="/wiki/2011%EB%85%84" title="2011년">2011년</a> <a href="/wiki/7%EC%9B%94_12%EC%9D%BC" title="7월 12일">7월 12일</a> <a href="/wiki/%EC%A7%90_%EB%A7%A4%EA%B7%B8%EB%A0%88%EC%9D%B8" title="짐 매그레인">짐 매그레인</a>의 대체 선수로 <a href="/wiki/SK_%EC%99%80%EC%9D%B4%EB%B2%88%EC%8A%A4" title="SK 와이번스">SK 와이번스</a>에 합류하였다. <a href="/wiki/2011%EB%85%84" title="2011년">2011년</a> <a href="/wiki/7%EC%9B%94_29%EC%9D%BC" title="7월 29일">7월 29일</a> <a href="/wiki/%ED%95%9C%ED%99%94_%EC%9D%B4%EA%B8%80%EC%8A%A4" title="한화 이글스">한화 이글스</a>와의 원정 경기에서 6⅓ 이닝 4피안타 2볼넷 9탈삼진 3실점의 성적을 올리고 첫 승리를 거두었다. 하지만 선발로는 이닝 소화 능력이 떨어진다는 단점을 갖고 있었고, 시즌 후 <a href="/wiki/SK_%EC%99%80%EC%9D%B4%EB%B2%88%EC%8A%A4" title="SK 와이번스">SK 와이번스</a>와의 협상이 결렬되어 자유계약 공시되었다. 때마침 <a href="/wiki/%EC%A0%80%EC%8A%A4%ED%8B%B4_%EC%A0%80%EB%A7%88%EB%85%B8" title="저스틴 저마노">저마노</a>를 <a href="/wiki/%EC%82%BC%EC%84%B1_%EB%9D%BC%EC%9D%B4%EC%98%A8%EC%A6%88" title="삼성 라이온즈">삼성 라이온즈</a>에서 놓치게 되자, <a href="/wiki/2011%EB%85%84_%ED%95%9C%EA%B5%AD%EC%8B%9C%EB%A6%AC%EC%A6%88" title="2011년 한국시리즈">2011년 한국시리즈</a>에서 만났던 삼성에서 그를 <a href="/wiki/2012%EB%85%84" title="2012년">2012년</a> <a href="/wiki/1%EC%9B%94_13%EC%9D%BC" title="1월 13일">1월 13일</a>에 영입하였다. 유니폼을 바꿔 입은 후 <a href="/wiki/2012%EB%85%84_%ED%95%9C%EA%B5%AD%EC%8B%9C%EB%A6%AC%EC%A6%88" title="2012년 한국시리즈">2012년 한국시리즈</a>에서 <a href="/wiki/SK_%EC%99%80%EC%9D%B4%EB%B2%88%EC%8A%A4" title="SK 와이번스">SK 와이번스</a>와 다시 만났다. 삼성 이적 이후 11승 3패를 기록했지만 SK때부터 지적되어 왔던 이닝 소화 능력의 결여와 안정감이 떨어진다는 단점이 드러나 시즌 후 재계약에 실패했다. <a href="/wiki/2012%EB%85%84" title="2012년">2012년</a> <a href="/wiki/12%EC%9B%94_23%EC%9D%BC" title="12월 23일">12월 23일</a>(한국시간) 미국 <a href="/wiki/%EB%A9%94%EC%9D%B4%EC%A0%80_%EB%A6%AC%EA%B7%B8_%EB%B2%A0%EC%9D%B4%EC%8A%A4%EB%B3%BC" title="메이저 리그 베이스볼">메이저 리그</a> <a href="/wiki/%EC%98%A4%ED%81%B4%EB%9E%9C%EB%93%9C_%EC%95%A0%EC%8A%AC%EB%A0%88%ED%8B%B1%EC%8A%A4" title="오클랜드 애슬레틱스">오클랜드 애슬레틱스</a>와 마이너리그 계약을 체결했다.<sup class="reference" id="cite_ref-1"><a
