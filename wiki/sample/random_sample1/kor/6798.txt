<p><b>브란코 오블라크</b>(<span style="font-size: smaller;"><a href="/wiki/%EC%8A%AC%EB%A1%9C%EB%B2%A0%EB%8B%88%EC%95%84%EC%96%B4" title="슬로베니아어">슬로베니아어</a>:</span> <span lang="sl" xml:lang="sl">Branko Oblak</span>, <a href="/wiki/1947%EB%85%84" title="1947년">1947년</a> <a href="/wiki/5%EC%9B%94_27%EC%9D%BC" title="5월 27일">5월 27일</a> ~ )는 은퇴한 <a href="/wiki/%EC%9C%A0%EA%B3%A0%EC%8A%AC%EB%9D%BC%EB%B9%84%EC%95%84_%EC%82%AC%ED%9A%8C%EC%A3%BC%EC%9D%98_%EC%97%B0%EB%B0%A9%EA%B3%B5%ED%99%94%EA%B5%AD" title="유고슬라비아 사회주의 연방공화국">유고슬라비아</a>와 <a href="/wiki/%EC%8A%AC%EB%A1%9C%EB%B2%A0%EB%8B%88%EC%95%84" title="슬로베니아">슬로베니아</a>의 축구 선수이다.</p>
<h2><span class="mw-headline" id=".EC.84.A0.EC.88.98_.EA.B2.BD.EB.A0.A5">선수 경력</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%B8%8C%EB%9E%80%EC%BD%94_%EC%98%A4%EB%B8%94%EB%9D%BC%ED%81%AC&amp;action=edit&amp;section=1" title="부분 편집: 선수 경력">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>오블라크는 <a class="new" href="/w/index.php?title=%EC%8A%A4%EB%B3%B4%EB%B3%B4%EB%8B%A4_%EB%A5%98%EB%B8%94%EB%9E%B4%EB%82%98&amp;action=edit&amp;redlink=1" title="스보보다 류블랴나 (없는 문서)">스보보다 류블랴나</a> 유소년 팀에서 축구를 시작하였고, <a href="/wiki/1965%EB%85%84" title="1965년">1965년</a> <a class="new" href="/w/index.php?title=NK_%EC%98%AC%EB%A6%BC%ED%94%BC%EC%95%BC_%EB%A5%98%EB%B8%94%EB%9E%B4%EB%82%98_(1911)&amp;action=edit&amp;redlink=1" title="NK 올림피야 류블랴나 (1911) (없는 문서)">NK 올림피야 류블랴나</a>로 팀을 옮긴 후 <a href="/wiki/1966%EB%85%84" title="1966년">1966년</a> <a href="/wiki/5%EC%9B%94_20%EC%9D%BC" title="5월 20일">5월 20일</a> <a href="/wiki/FK_%ED%8C%8C%EB%A5%B4%ED%8B%B0%EC%9E%94" title="FK 파르티잔">FK 파르티잔</a> 전에서 데뷔와 동시에 골을 넣었다. <a href="/wiki/1973%EB%85%84" title="1973년">1973년</a>까지 팀에서 뛰며 181경기에 나와 33골을 넣었다.</p>
<p><a href="/wiki/1973%EB%85%84" title="1973년">1973년</a> <a href="/wiki/HNK_%ED%95%98%EC%9D%B4%EB%91%90%ED%81%AC_%EC%8A%A4%ED%94%8C%EB%A6%AC%ED%8A%B8" title="HNK 하이두크 스플리트">HNK 하이두크 스플리트</a>로 팀을 옮긴 후 두 시즌동안 뛰었다. 그가 뛰는 동안 <a href="/wiki/HNK_%ED%95%98%EC%9D%B4%EB%91%90%ED%81%AC_%EC%8A%A4%ED%94%8C%EB%A6%AC%ED%8A%B8" title="HNK 하이두크 스플리트">하이두크 스플리트</a>는 2번의 리그 우승을 달성하였다. <a href="/wiki/1975%EB%85%84" title="1975년">1975년</a> <a href="/wiki/%EB%8F%85%EC%9D%BC" title="독일">독일</a>로 건너가 <a href="/wiki/FC_%EC%83%AC%EC%BC%80_04" title="FC 샬케 04">FC 샬케 04</a>에 입단했고, <a href="/wiki/1977%EB%85%84" title="1977년">1977년</a> <a href="/wiki/FC_%EB%B0%94%EC%9D%B4%EC%97%90%EB%A5%B8_%EB%AE%8C%ED%97%A8" title="FC 바이에른 뮌헨">FC 바이에른 뮌헨</a>으로 이적하였다. <a href="/wiki/FC_%EB%B0%94%EC%9D%B4%EC%97%90%EB%A5%B8_%EB%AE%8C%ED%97%A8" title="FC 바이에른 뮌헨">바이에른 뮌헨</a>에 두시즌 동안 머무르면서 1979-80시즌 우승에 일조하였다. 그 후 <a href="/wiki/%EC%98%A4%EC%8A%A4%ED%8A%B8%EB%A6%AC%EC%95%84" title="오스트리아">오스트리아</a>의 아마추어 클럽인 <a class="new" href="/w/index.php?title=SV_%EC%A6%88%ED%94%BC%ED%83%88/%EB%93%9C%EB%9D%BC%EC%9A%B0&amp;action=edit&amp;redlink=1" title="SV 즈피탈/드라우 (없는 문서)">SV 즈피탈/드라우</a>에서 뛴 후 은퇴하였다.</p>
<p>오블라크는 또한 <a href="/wiki/%EC%9C%A0%EA%B3%A0%EC%8A%AC%EB%9D%BC%EB%B9%84%EC%95%84_%EC%B6%95%EA%B5%AC_%EA%B5%AD%EA%B0%80%EB%8C%80%ED%91%9C%ED%8C%80" title="유고슬라비아 축구 국가대표팀">유고슬라비아 축구 국가대표팀</a>의 일원으로서 활약하기도 했는데, 50경기에 나와 8골을 넣었다. <a href="/wiki/1971%EB%85%84" title="1971년">1971년</a> <a href="/wiki/%EB%A3%A8%EB%A7%88%EB%8B%88%EC%95%84_%EC%B6%95%EA%B5%AC_%EA%B5%AD%EA%B0%80%EB%8C%80%ED%91%9C%ED%8C%80" title="루마니아 축구 국가대표팀">루마니아</a>를 상대로 국가대표 데뷔전을 치루었으며, <a href="/wiki/%ED%8E%A0%EB%A0%88" title="펠레">펠레</a>의 은퇴 경기에도 뛰었다. <a href="/wiki/1974%EB%85%84_FIFA_%EC%9B%94%EB%93%9C%EC%BB%B5" title="1974년 FIFA 월드컵">1974년 FIFA 월드컵</a>에서는 팀 동료 <a class="new" href="/w/index.php?title=%EB%8B%A4%EB%8B%90%EB%A1%9C_%ED%8F%AC%ED%94%BC%EB%B3%B4%EB%8B%A4&amp;action=edit&amp;redlink=1" title="다닐로 포피보다 (없는 문서)">다닐로 포피보다</a>와 같이 <a href="/wiki/FIFA_%EC%9B%94%EB%93%9C%EC%BB%B5" title="FIFA 월드컵">FIFA 월드컵</a> 무대를 밟은 첫 번째 슬로베니아인이 되었다. <a href="/wiki/1976%EB%85%84" title="1976년">1976년</a>에는 <a href="/wiki/%EC%9C%A0%EA%B3%A0%EC%8A%AC%EB%9D%BC%EB%B9%84%EC%95%84" title="유고슬라비아">유고슬라비아</a>에서 열린 <a href="/wiki/UEFA_%EC%9C%A0%EB%A1%9C_1976" title="UEFA 유로 1976">UEFA 유로 1976</a>에 참가하였고, 대회가 끝난 후 국가대표팀에서 은퇴하였다.</p>
<h2><span class="mw-headline" id=".EA.B0.90.EB.8F.85_.EA.B2.BD.EB.A0.A5">감독 경력</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%B8%8C%EB%9E%80%EC%BD%94_%EC%98%A4%EB%B8%94%EB%9D%BC%ED%81%AC&amp;action=edit&amp;section=2" title="부분 편집: 감독 경력">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>선수 은퇴 후 그는 감독이 되어 <a class="new" href="/w/index.php?title=NK_%EC%98%AC%EB%A6%BC%ED%94%BC%EC%95%BC_%EB%A5%98%EB%B8%94%EB%9E%B4%EB%82%98_(1911)&amp;action=edit&amp;redlink=1" title="NK 올림피야 류블랴나 (1911) (없는 문서)">NK 올림피야 류블랴나</a>와 <a href="/wiki/FC_%EC%BD%94%ED%8E%98%EB%A5%B4" title="FC 코페르">FC 코페르</a>의 감독으로 있었다. <a href="/wiki/2004%EB%85%84_2%EC%9B%94" title="2004년 2월">2004년 2월</a> <a href="/wiki/%EC%8A%AC%EB%A1%9C%EB%B2%A0%EB%8B%88%EC%95%84_%EC%B6%95%EA%B5%AC_%EA%B5%AD%EA%B0%80%EB%8C%80%ED%91%9C%ED%8C%80" title="슬로베니아 축구 국가대표팀">슬로베니아 U-21</a> 감독이 되었고, 그 해 <a href="/wiki/5%EC%9B%94_24%EC%9D%BC" title="5월 24일">5월 24일</a> <a href="/wiki/%EC%8A%AC%EB%A1%9C%EB%B2%A0%EB%8B%88%EC%95%84_%EC%B6%95%EA%B5%AC_%EA%B5%AD%EA%B0%80%EB%8C%80%ED%91%9C%ED%8C%80" title="슬로베니아 축구 국가대표팀">슬로베니아 축구 국가대표팀</a>의 감독직을 수락하였다. <a href="/wiki/2006%EB%85%84_FIFA_%EC%9B%94%EB%93%9C%EC%BB%B5_%EC%9C%A0%EB%9F%BD_%EC%A7%80%EC%97%AD_%EC%98%88%EC%84%A0" title="2006년 FIFA 월드컵 유럽 지역 예선">2006년 FIFA 월드컵 유럽 지역 예선</a>에서 <a href="/wiki/%EC%9D%B4%ED%83%88%EB%A6%AC%EC%95%84_%EC%B6%95%EA%B5%AC_%EA%B5%AD%EA%B0%80%EB%8C%80%ED%91%9C%ED%8C%80" title="이탈리아 축구 국가대표팀">이탈리아</a>를 1-0으로 이기는 등 이변을 연출했으나 졸전을 거듭해 본선 진출에는 실패하고, <a href="/wiki/2006%EB%85%84" title="2006년">2006년</a> 말에 해임당했다. <a href="/wiki/2009%EB%85%84" title="2009년">2009년</a> <a class="new" href="/w/index.php?title=NK_%EC%98%AC%EB%A6%BC%ED%94%BC%EC%95%BC_%EB%A5%98%EB%B8%94%EB%9E%B4%EB%82%98_(2005)&amp;action=edit&amp;redlink=1" title="NK 올림피야 류블랴나 (2005) (없는 문서)">NK 올림피야 류블랴나</a>로 다시 돌아갔으나 4경기만에 감독자리에서 물러났다.</p>
<p><a href="/wiki/2003%EB%85%84" title="2003년">2003년</a> <a href="/wiki/UEFA_%EC%B0%BD%EB%A6%BD_50%EC%A3%BC%EB%85%84_%EA%B8%B0%EB%85%90%EC%83%81" title="UEFA 창립 50주년 기념상">UEFA 창립 50주년 기념상</a>을 수상하였다.</p>
