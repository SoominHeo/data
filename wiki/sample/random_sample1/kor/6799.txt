<h1 class="firstHeading" id="firstHeading" lang="ko">브래넌 매닝</h1>
<div class="mw-body-content" id="bodyContent">

<div class="mw-content-ltr" dir="ltr" id="mw-content-text" lang="ko"><script>function mfTempOpenSection(id){var block=document.getElementById("mf-section-"+id);block.className+=" open-block";block.previousSibling.className+=" open-block";}</script><p><b>브래넌 매닝</b>(Brennan Manning, <a href="/wiki/1934%EB%85%84" title="1934년">1934년</a> <a href="/wiki/4%EC%9B%94_27%EC%9D%BC" title="4월 27일">4월 27일</a>~<a href="/wiki/2013%EB%85%84" title="2013년">2013년</a> <a href="/wiki/4%EC%9B%94_12%EC%9D%BC" title="4월 12일">4월 12일</a>)은 <a href="/wiki/%EB%AF%B8%EA%B5%AD" title="미국">미국</a> <a class="mw-redirect" href="/wiki/%EB%A1%9C%EB%A7%88_%EA%B0%80%ED%86%A8%EB%A6%AD_%EA%B5%90%ED%9A%8C" title="로마 가톨릭 교회">로마 가톨릭 교회</a> <a class="mw-redirect" href="/wiki/%EC%8B%A0%EB%B6%80_(%EA%B8%B0%EB%8F%85%EA%B5%90)" title="신부 (기독교)">신부</a>이자 영성가이다.</p>
<h2><span class="mw-headline" id=".EC.83.9D.EC.95.A0">생애</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%B8%8C%EB%9E%98%EB%84%8C_%EB%A7%A4%EB%8B%9D&amp;action=edit&amp;section=1" title="부분 편집: 생애">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>로마 가톨릭 교회 가정에서 태어났으며, <a href="/wiki/%EA%B3%A0%EB%93%B1%ED%95%99%EA%B5%90" title="고등학교">고등학교</a> 졸업 후 <a href="/wiki/%EB%AF%B8%EA%B5%AD_%ED%95%B4%EB%B3%91%EB%8C%80" title="미국 해병대">미국 해병대원</a>으로 <a class="mw-redirect" href="/wiki/%ED%95%9C%EA%B5%AD%EC%A0%84%EC%9F%81" title="한국전쟁">한국전쟁</a>에 참여했다. <a class="new" href="/w/index.php?title=%EC%83%8C%ED%94%84%EB%9E%80%EC%8B%9C%EC%8A%A4%EC%BD%94_%EB%8C%80%ED%95%99%EA%B5%90&amp;action=edit&amp;redlink=1" title="샌프란시스코 대학교 (없는 문서)">샌프란시스코 대학교</a>졸업후 신학교(Semitary)에 진학했다. <a href="/wiki/1963%EB%85%84" title="1963년">1963년</a> 신학교를 졸업한 브래넌은 <a class="mw-redirect" href="/wiki/%ED%94%84%EB%9E%80%EC%B2%B4%EC%8A%A4%EC%BD%94_%EC%88%98%EB%8F%84%ED%9A%8C" title="프란체스코 수도회">프란체스코 수도회</a>에 입회하였다. <a href="/wiki/%EC%88%98%EB%8F%84%EC%9B%90" title="수도원">수도원</a>에서 <a class="mw-redirect" href="/wiki/%EC%84%9C%ED%92%88%EC%84%B1%EC%82%AC" title="서품성사">사제서품</a>을 받은 그는 수년후 프란체스코 수도회에 소속되어 있는 작은 예수형제회(Little Brothers of Jesus)에서 노동에 종사하면서 수도생활을 했다.이때 브래넌은 <a href="/wiki/%ED%95%9C%EA%B5%AD" title="한국">한국</a> 등의 여러 나라에서 온 형제들과 사귀면서 용서에 대해서 배우는 영적인 성숙을 한다. 현재 그는 <a class="mw-redirect" href="/wiki/%EB%A1%9C%EB%A7%88_%EA%B0%80%ED%86%A8%EB%A6%AD_%EA%B5%90%ED%9A%8C" title="로마 가톨릭 교회">로마 가톨릭 교회</a>와 <a class="mw-redirect" href="/wiki/%EA%B0%9C%EC%8B%A0%EA%B5%90%ED%9A%8C" title="개신교회">개신교회</a>의 구분없이 강연을 하고 있으며, 저술활동을 통해 <a href="/wiki/%EC%98%88%EC%88%98" title="예수">예수</a>의 삶을 따를 것을 촉구한다.</p>
