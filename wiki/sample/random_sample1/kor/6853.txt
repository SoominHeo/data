<p><b>브리앤 시도니 디소니어스</b>(<span style="font-size: smaller;"><a href="/wiki/%EC%98%81%EC%96%B4" title="영어">영어</a>:</span> <span lang="en" xml:lang="en">Brianne Sidonie Desaulniers</span>, <a href="/wiki/1989%EB%85%84" title="1989년">1989년</a> <a href="/wiki/10%EC%9B%94_1%EC%9D%BC" title="10월 1일">10월 1일</a> ~ )는 <b>브리 라슨</b>(<span style="font-size: smaller;"><a href="/wiki/%EC%98%81%EC%96%B4" title="영어">영어</a>:</span> <span lang="en" xml:lang="en">Brie Larson</span>)으로 잘 알려진 <a href="/wiki/%EB%AF%B8%EA%B5%AD" title="미국">미국</a>의 <a href="/wiki/%EB%B0%B0%EC%9A%B0" title="배우">배우</a>이자 <a href="/wiki/%EA%B0%80%EC%88%98" title="가수">가수</a>이다.</p>
<p>2015년에는 드라마 영화 《<a href="/wiki/%EB%A3%B8_(2015%EB%85%84_%EC%98%81%ED%99%94)" title="룸 (2015년 영화)">룸</a>》에서 조이 "마" 뉴섬 역으로 호평을 받았으며, 코미디 영화 《<a href="/wiki/%EB%82%98%EB%A5%BC_%EB%AF%B8%EC%B9%98%EA%B2%8C_%ED%95%98%EB%8A%94_%EC%97%AC%EC%9E%90" title="나를 미치게 하는 여자">나를 미치게 하는 여자</a>》에도 출연하였다. 《룸》에서의 연기로 <a class="mw-redirect" href="/wiki/%EC%98%81%EA%B5%AD_%EC%95%84%EC%B9%B4%EB%8D%B0%EB%AF%B8%EC%83%81" title="영국 아카데미상">영국 아카데미상</a>, <a href="/wiki/%ED%81%AC%EB%A6%AC%ED%8B%B1%EC%8A%A4_%EC%B4%88%EC%9D%B4%EC%8A%A4_%EC%98%81%ED%99%94%EC%83%81" title="크리틱스 초이스 영화상">크리틱스 초이스 영화상</a>, <a href="/wiki/%EA%B3%A8%EB%93%A0_%EA%B8%80%EB%A1%9C%EB%B8%8C%EC%83%81" title="골든 글로브상">골든 글로브상</a>, <a href="/wiki/%EC%9D%B8%EB%94%94%ED%8E%9C%EB%8D%98%ED%8A%B8_%EC%8A%A4%ED%94%BC%EB%A6%BF_%EC%96%B4%EC%9B%8C%EB%93%9C" title="인디펜던트 스피릿 어워드">인디펜던트 스피릿 어워드</a>, <a href="/wiki/%EB%AF%B8%EA%B5%AD_%EB%B0%B0%EC%9A%B0_%EC%A1%B0%ED%95%A9%EC%83%81" title="미국 배우 조합상">미국 배우 조합상</a>, <a href="/wiki/%EC%A0%9C88%ED%9A%8C_%EC%95%84%EC%B9%B4%EB%8D%B0%EB%AF%B8%EC%83%81" title="제88회 아카데미상">제88회 아카데미상</a> <a href="/wiki/%EC%95%84%EC%B9%B4%EB%8D%B0%EB%AF%B8_%EC%97%AC%EC%9A%B0%EC%A3%BC%EC%97%B0%EC%83%81" title="아카데미 여우주연상">여우주연상</a>을 수상하였다.</p>
