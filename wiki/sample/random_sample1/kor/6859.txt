<p><b>브리즈번 로어 FC</b> (Brisbane Roar FC)는 <a href="/wiki/A%EB%A6%AC%EA%B7%B8" title="A리그">A리그</a>에 소속된 <a href="/wiki/%EC%98%A4%EC%8A%A4%ED%8A%B8%EB%A0%88%EC%9D%BC%EB%A6%AC%EC%95%84" title="오스트레일리아">오스트레일리아</a>의 프로 축구단으로, <a href="/wiki/%ED%80%B8%EC%A6%90%EB%9E%9C%EB%93%9C_%EC%A3%BC" title="퀸즐랜드 주">퀸즐랜드 주</a> <a href="/wiki/%EB%B8%8C%EB%A6%AC%EC%A6%88%EB%B2%88" title="브리즈번">브리즈번</a>를 연고지로 한다. A리그 초기 퀸즐랜드 로어 FC라는 이름으로 참여하였다가 2009년 브리즈번 로어 FC가 되었다.</p>
<h2><span class="mw-headline" id=".EC.97.AD.EC.82.AC">역사</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%B8%8C%EB%A6%AC%EC%A6%88%EB%B2%88_%EB%A1%9C%EC%96%B4_FC&amp;action=edit&amp;section=1" title="부분 편집: 역사">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>1957년 네덜란드 계 이주민들이 바탕이 된 홀란디아 이날라 사커 클럽을 모태로 하고 있다. 홀란디아 이날라는 1970년대에 이름을 브리즈번 라이언즈로 변경하였고, 1990년대 <a href="/wiki/%EB%9F%AD%EB%B9%84" title="럭비">럭비</a> 팀과 <a href="/wiki/%EC%B6%95%EA%B5%AC" title="축구">축구</a> 팀이 분리되면서 축구 팀의 이름이 <a class="new" href="/w/index.php?title=%ED%80%B8%EC%A6%90%EB%9E%9C%EB%93%9C_%EB%9D%BC%EC%9D%B4%EC%96%B8%EC%A6%88&amp;action=edit&amp;redlink=1" title="퀸즐랜드 라이언즈 (없는 문서)">퀸즐랜드 라이언즈</a>로 변경되었다. 2004년 <a href="/wiki/A%EB%A6%AC%EA%B7%B8" title="A리그">A리그</a> 발족을 앞두고 <a href="/wiki/%ED%80%B8%EC%A6%90%EB%9E%9C%EB%93%9C_%EC%A3%BC" title="퀸즐랜드 주">퀸즐랜드 주</a>를 기반으로 클럽 라이선스를 확보하며 일부가 분리, 퀸즐랜드 로어 FC라는 이름으로 창단하였다.</p>
<p>2009년 <a href="/wiki/A%EB%A6%AC%EA%B7%B8" title="A리그">A리그</a> 클럽 라이선스의 확대로 각각 <a href="/wiki/%ED%80%B8%EC%A6%90%EB%9E%9C%EB%93%9C_%EC%A3%BC" title="퀸즐랜드 주">퀸즐랜드 주</a> <a class="mw-disambig" href="/wiki/%EA%B3%A8%EB%93%9C%EC%BD%94%EC%8A%A4%ED%8A%B8" title="골드코스트">골드코스트</a>와 <a class="mw-redirect" href="/wiki/%EC%BC%80%EC%96%B8%EC%A6%88" title="케언즈">케언즈</a>를 연고로 하는 <a class="mw-redirect" href="/wiki/%EA%B3%A8%EB%93%9C%EC%BD%94%EC%8A%A4%ED%8A%B8_%EC%9C%A0%EB%82%98%EC%9D%B4%ED%8B%B0%EB%93%9C" title="골드코스트 유나이티드">골드코스트 유나이티드</a>와 <a class="mw-redirect" href="/wiki/%EB%85%B8%EC%8A%A4%ED%80%B8%EC%A6%90%EB%9E%9C%EB%93%9C_%ED%93%A8%EC%96%B4%EB%A6%AC" title="노스퀸즐랜드 퓨어리">노스퀸즐랜드 퓨어리</a>가 창단되면서 로어 측과 <a class="new" href="/w/index.php?title=%EC%98%A4%EC%8A%A4%ED%8A%B8%EB%A0%88%EC%9D%BC%EB%A6%AC%EC%95%84_%EC%B6%95%EA%B5%AC_%EC%97%B0%EB%A7%B9&amp;action=edit&amp;redlink=1" title="오스트레일리아 축구 연맹 (없는 문서)">오스트레일리아 축구 연맹</a> 측에서는 보다 명확한 연고를 규정할 필요가 생겼고, 이에 기존 <a href="/wiki/%ED%80%B8%EC%A6%90%EB%9E%9C%EB%93%9C_%EC%A3%BC" title="퀸즐랜드 주">퀸즐랜드 주</a>에서 <a href="/wiki/%EB%B8%8C%EB%A6%AC%EC%A6%88%EB%B2%88" title="브리즈번">브리즈번</a>으로 연고를 축소하며 이름을 브리즈번 로어 FC로 변경하였다.</p>
<p>2010-11 시즌엔 A리그에 참가한 이후 처음으로 정규 리그에서 우승을 차지하는 이변을 일으켰으며 그랜드 파이널에서도 승리하며 최종 우승팀이 되었다. 이어진 2011-12 시즌에선 정규 리그는 2위로 마쳤으나 플레이오프를 거쳐 참가한 그랜드 파이널에서 승리하여 <a href="/wiki/A%EB%A6%AC%EA%B7%B8" title="A리그">A리그</a> 최초로 2연패를 달성하였다. 또한 2010년 9월 18일에서 2011년 12월 4일까지 치러진 36회의 리그 경기에서 단 한 번도 패하지 않으며 <a href="/wiki/%EC%98%A4%EC%8A%A4%ED%8A%B8%EB%A0%88%EC%9D%BC%EB%A6%AC%EC%95%84" title="오스트레일리아">오스트레일리아</a> 프로 스포츠 사상 최장 무패 기록을 달성하였다. 하지만 <a href="/wiki/AFC_%EC%B1%94%ED%94%BC%EC%96%B8%EC%8A%A4%EB%A6%AC%EA%B7%B8_2012" title="AFC 챔피언스리그 2012">2012 ACL</a>에서는 <a href="/wiki/FC_%EB%8F%84%EC%BF%84" title="FC 도쿄">FC 도쿄</a>와 <a href="/wiki/%EC%9A%B8%EC%82%B0_%ED%98%84%EB%8C%80_%EC%B6%95%EA%B5%AC%EB%8B%A8" title="울산 현대 축구단">울산 현대</a>에 밀려 조별 예선에서 탈락하였다.</p>
<p>2012-13 시즌에는 파이널 시리즈에서 <a class="mw-redirect" href="/wiki/%EC%9B%A8%EC%8A%A4%ED%84%B4_%EC%8B%9C%EB%93%9C%EB%8B%88_%EC%9B%90%EB%8D%94%EB%9F%AC%EC%8A%A4" title="웨스턴 시드니 원더러스">웨스턴 시드니 원더러스</a>에 패하여 리그 3연패에는 실패하였지만 2013-14 시즌 그랜드 파이널에서 다시 웨스턴 시드니를 만나 2-1로 승리하며 우승컵을 되찾아왔다.</p>
