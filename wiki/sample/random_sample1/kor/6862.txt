<p><b>브리타니아</b>(<span style="font-size: smaller;"><a href="/wiki/%EB%9D%BC%ED%8B%B4%EC%96%B4" title="라틴어">라틴어</a>:</span> <span lang="la" xml:lang="la">Britannia</span>)는 <a href="/wiki/%EA%B3%A0%EB%8C%80_%EB%A1%9C%EB%A7%88" title="고대 로마">고대 로마</a> 시대에 오늘날 <a href="/wiki/%EC%98%81%EA%B5%AD" title="영국">영국</a>의 <a href="/wiki/%EA%B7%B8%EB%A0%88%EC%9D%B4%ED%8A%B8%EB%B8%8C%EB%A6%AC%ED%8A%BC_%EC%84%AC" title="그레이트브리튼 섬">그레이트브리튼 섬</a>을 이르던 말이다.</p>
<p>브리타니아라는 이름은 원래 이 지역의 원주민이던 <a href="/wiki/%EC%BC%88%ED%8A%B8%EC%A1%B1" title="켈트족">켈트족</a>의 일부인 <a class="mw-redirect" href="/wiki/%EB%B8%8C%EB%A6%AC%ED%8A%BC%EC%A1%B1" title="브리튼족">브리튼족</a>에서 따왔다고 한다. 로마군의 침입 이전인 철기 시대에도 유럽 대륙과 접촉하였지만 본격적인 문명화는 로마의 침공 이후에 일어났다.</p>
<p><a href="/wiki/%EA%B8%B0%EC%9B%90%EC%A0%84_55%EB%85%84" title="기원전 55년">기원전 55년</a>, <a href="/wiki/%EA%B8%B0%EC%9B%90%EC%A0%84_54%EB%85%84" title="기원전 54년">기원전 54년</a> 두 차례에 걸쳐 로마 장군 <a class="mw-redirect" href="/wiki/%EA%B0%80%EC%9D%B4%EC%9A%B0%EC%8A%A4_%EC%9C%A8%EB%A6%AC%EC%9A%B0%EC%8A%A4_%EC%B9%B4%EC%9D%B4%EC%82%AC%EB%A5%B4" title="가이우스 율리우스 카이사르">가이우스 율리우스 카이사르</a>의 공략을 받았으나, 당시에는 <a href="/wiki/%EA%B0%88%EB%A6%AC%EC%95%84" title="갈리아">갈리아</a>의 지배를 위한 방어적 성격의 일시적인 침공으로 그쳤다.</p>
<p>그 후 <a href="/wiki/1%EC%84%B8%EA%B8%B0" title="1세기">1세기</a> 중엽 <a href="/wiki/%ED%81%B4%EB%9D%BC%EC%9A%B0%EB%94%94%EC%9A%B0%EC%8A%A4" title="클라우디우스">클라우디우스</a> 황제 때 중남부 지역이 다시 로마군의 침공을 받아 로마의 속주가 되고, 1세기 말 <a href="/wiki/%EB%8F%84%EB%AF%B8%ED%8B%B0%EC%95%84%EB%88%84%EC%8A%A4" title="도미티아누스">도미티아누스</a> 황제 통치 시에는 <a href="/wiki/%EC%8A%A4%EC%BD%94%ED%8B%80%EB%9E%9C%EB%93%9C" title="스코틀랜드">스코틀랜드</a>와 서부 지역이 로마에 정복되어 이 지역에 대한 로마의 지배권이 강력해졌다.</p>
<p><a href="/wiki/%ED%95%98%EB%93%9C%EB%A6%AC%EC%95%84%EB%88%84%EC%8A%A4" title="하드리아누스">하드리아누스</a> 황제는 <a class="mw-redirect" href="/wiki/%EC%8A%A4%EC%BD%94%ED%8A%B8%EC%A1%B1" title="스코트족">스코트족</a>의 침입을 막기 위해 <a href="/wiki/%ED%95%98%EB%93%9C%EB%A6%AC%EC%95%84%EB%88%84%EC%8A%A4_%EB%B0%A9%EB%B2%BD" title="하드리아누스 방벽">하드리아누스 방벽</a>을 구축하였고 그 이후로는 더 이상 북쪽에 대한 정복을 시도하지는 않았다. <a href="/wiki/410%EB%85%84" title="410년">410년</a> 로마군이 철수함으로써 이 지역은 <a class="mw-redirect" href="/wiki/%EC%95%B5%EA%B8%80%EB%A1%9C%EC%A1%B1" title="앵글로족">앵글로족</a>과 <a class="mw-redirect" href="/wiki/%EC%83%89%EC%8A%A8%EC%A1%B1" title="색슨족">색슨족</a>의 무대가 되었다.</p>
<p>또한, 로마의 침공 당시 로마인들은 <a href="/wiki/43%EB%85%84" title="43년">43년</a>에 지금의 <a class="mw-redirect" href="/wiki/%ED%85%9C%EC%A6%88_%EA%B0%95" title="템즈 강">템즈 강</a> 유역에 요새를 건설했는데, 그 요새의 이름이 론디니움(Londinium)이었다. 훗날 론디니움은 지금 영국의 수도인 <a href="/wiki/%EB%9F%B0%EB%8D%98" title="런던">런던</a>의 어원이 된다.</p>
