<p><b>블라디미르 알렉산드로비치 보예보츠키</b>(<span style="font-size: smaller;"><a href="/wiki/%EB%9F%AC%EC%8B%9C%EC%95%84%EC%96%B4" title="러시아어">러시아어</a>:</span> <span lang="ru" xml:lang="ru">Влади́мир Александро́вич Воево́дский</span>, <a href="/wiki/1966%EB%85%84" title="1966년">1966년</a> <a href="/wiki/6%EC%9B%94_4%EC%9D%BC" title="6월 4일">6월 4일</a> ~ )는 <a href="/wiki/%EB%9F%AC%EC%8B%9C%EC%95%84" title="러시아">러시아</a>의 <a href="/wiki/%EC%88%98%ED%95%99%EC%9E%90" title="수학자">수학자</a>이다. 현재는 <a href="/wiki/%EB%AF%B8%EA%B5%AD" title="미국">미국</a> <a href="/wiki/%EB%89%B4%EC%A0%80%EC%A7%80_%EC%A3%BC" title="뉴저지 주">뉴저지 주</a> <a href="/wiki/%ED%94%84%EB%A6%B0%EC%8A%A4%ED%84%B4_%EA%B3%A0%EB%93%B1%EC%97%B0%EA%B5%AC%EC%86%8C" title="프린스턴 고등연구소">프린스턴 고등연구소</a>의 수학 <a href="/wiki/%EA%B5%90%EC%88%98" title="교수">교수</a>로 재직 중이다.</p>
<p>보예보츠키는 <a href="/wiki/%EB%8C%80%EC%88%98%EA%B8%B0%ED%95%98%ED%95%99" title="대수기하학">대수기하학</a>의 주 연구 대상인 <a href="/wiki/%EB%8C%80%EC%88%98%EB%8B%A4%EC%96%91%EC%B2%B4" title="대수다양체">대수다양체</a>의 연구에 <a href="/wiki/%EB%8C%80%EC%88%98%EC%A0%81_%EC%9C%84%EC%83%81%EC%88%98%ED%95%99" title="대수적 위상수학">대수적 위상수학</a>에서 주로 쓰이는 <a class="mw-redirect" href="/wiki/%ED%98%B8%EB%AA%A8%ED%86%A0%ED%94%BC_%EC%9D%B4%EB%A1%A0" title="호모토피 이론">호모토피 이론</a>을 접목하여, <a href="/wiki/%EB%8C%80%EC%88%98%EB%8B%A4%EC%96%91%EC%B2%B4" title="대수다양체">대수다양체</a>의 <span class="mwe-math-element"><span class="mwe-math-mathml-inline mwe-math-mathml-a11y" style="display: none;"><math xmlns="http://www.w3.org/1998/Math/MathML">

<p>2009년에는 <a href="/wiki/%EB%AA%A8%ED%8B%B0%EB%B8%8C_%EC%BD%94%ED%98%B8%EB%AA%B0%EB%A1%9C%EC%A7%80" title="모티브 코호몰로지">모티브 코호몰로지</a>를 사용하여 일반적인 <a class="mw-redirect" href="/wiki/%EB%B8%94%EB%A1%9D-%EA%B0%80%ED%86%A0_%EC%B6%94%EC%B8%A1" title="블록-가토 추측">블록-가토 추측</a>의 증명을 발표하였다.</p>
<h2><span class="mw-headline" id=".EC.9D.B4.EB.A0.A5">이력</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%B8%94%EB%9D%BC%EB%94%94%EB%AF%B8%EB%A5%B4_%EB%B3%B4%EC%98%88%EB%B3%B4%EC%B8%A0%ED%82%A4&amp;action=edit&amp;section=1" title="부분 편집: 이력">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>보예보츠키는 <a class="mw-redirect" href="/wiki/%EC%86%8C%EB%B9%84%EC%97%90%ED%8A%B8_%EC%97%B0%EB%B0%A9" title="소비에트 연방">소비에트 연방</a> <a href="/wiki/%EB%AA%A8%EC%8A%A4%ED%81%AC%EB%B0%94" title="모스크바">모스크바</a>에서 태어나 <a class="mw-redirect" href="/wiki/%EB%AA%A8%EC%8A%A4%ED%81%AC%EB%B0%94_%EB%8C%80%ED%95%99%EA%B5%90" title="모스크바 대학교">모스크바 대학교</a>에서 학부를 마친 후, <a href="/wiki/%EB%AF%B8%EA%B5%AD" title="미국">미국</a>으로 건너가서 <a href="/wiki/1992%EB%85%84" title="1992년">1992년</a> <a href="/wiki/%ED%95%98%EB%B2%84%EB%93%9C_%EB%8C%80%ED%95%99%EA%B5%90" title="하버드 대학교">하버드 대학교</a>에서 <a class="new" href="/w/index.php?title=%EB%8D%B0%EC%9D%B4%EB%B9%84%EB%93%9C_%EC%B9%B4%EC%A6%88%EB%8B%A8&amp;action=edit&amp;redlink=1" title="데이비드 카즈단 (없는 문서)">데이비드 카즈단</a>(David Kazhdan) 교수의 지도로 박사학위를 받았다. 그 후, 미국 <a href="/wiki/%EC%9D%BC%EB%A6%AC%EB%85%B8%EC%9D%B4_%EC%A3%BC" title="일리노이 주">일리노이 주</a> <a class="mw-redirect mw-disambig" href="/wiki/%EC%97%90%EB%B0%98%EC%8A%A4%ED%84%B4" title="에반스턴">에반스턴</a>에 위치한 <a href="/wiki/%EB%85%B8%EC%8A%A4%EC%9B%A8%EC%8A%A4%ED%84%B4_%EB%8C%80%ED%95%99%EA%B5%90" title="노스웨스턴 대학교">노스웨스턴 대학교</a>에서 근무하다가 <a href="/wiki/%ED%94%84%EB%A6%B0%EC%8A%A4%ED%84%B4_%EA%B3%A0%EB%93%B1%EC%97%B0%EA%B5%AC%EC%86%8C" title="프린스턴 고등연구소">프린스턴 고등연구소</a>로 옮겨갔다.</p>
<p>가장 유명한 저서로, <a href="/wiki/%EB%85%B8%EC%8A%A4%EC%9B%A8%EC%8A%A4%ED%84%B4_%EB%8C%80%ED%95%99%EA%B5%90" title="노스웨스턴 대학교">노스웨스턴 대학교</a>의 <a class="new" href="/w/index.php?title=%EC%95%88%EB%93%9C%EB%A0%88%EC%9D%B4_%EC%88%98%EC%8A%AC%EB%A6%B0&amp;action=edit&amp;redlink=1" title="안드레이 수슬린 (없는 문서)">안드레이 수슬린</a>(Andrei Suslin)과 <a class="mw-redirect" href="/wiki/%EC%84%9C%EB%8D%98_%EC%BA%98%EB%A6%AC%ED%8F%AC%EB%8B%88%EC%95%84_%EB%8C%80%ED%95%99%EA%B5%90" title="서던 캘리포니아 대학교">서던 캘리포니아 대학교</a>의 <a class="new" href="/w/index.php?title=%EC%97%90%EB%A6%AD_%ED%94%84%EB%A6%AC%EB%93%9C%EB%9E%9C%EB%8D%94&amp;action=edit&amp;redlink=1" title="에릭 프리드랜더 (없는 문서)">에릭 프리드랜더</a>(Eric M. Friedlander)와 함께 <i>Cycles, Transfers and Motivic Homology Theories</i>라는 책을 썼다.<sup class="reference" id="cite_ref-1"><a href="#cite_note-1">[1]</a></sup></p>
