<p>《<b>블랙 호크 다운</b>》(<span lang="en" xml:lang="en">Black Hawk Down</span>)은 <a href="/wiki/2001%EB%85%84" title="2001년">2001년</a>에 제작된 전쟁 영화이다. <a href="/wiki/%EB%A6%AC%EB%93%A4%EB%A6%AC_%EC%8A%A4%EC%BD%A7" title="리들리 스콧">리들리 스콧</a>이 감독이며, <a href="/wiki/1993%EB%85%84" title="1993년">1993년</a> 일어난 <a href="/wiki/%EB%AA%A8%EA%B0%80%EB%94%94%EC%8A%88_%EC%A0%84%ED%88%AC_(1993%EB%85%84)" title="모가디슈 전투 (1993년)">모가디슈 전투</a>를 그리고 있다. <a href="/wiki/1999%EB%85%84" title="1999년">1999년</a> 발행된 <a class="new" href="/w/index.php?title=%EB%A7%88%ED%81%AC_%EB%B3%B4%EB%93%A0&amp;action=edit&amp;redlink=1" title="마크 보든 (없는 문서)">마크 보든</a>(Mark Bowden)의 넌픽션 <a href="/wiki/%EB%B8%94%EB%9E%99_%ED%98%B8%ED%81%AC_%EB%8B%A4%EC%9A%B4_(%EC%B1%85)" title="블랙 호크 다운 (책)">책</a> 《블랙 호크 다운》(Black Hawk Down ; A Story of Modern War)이 원작이다.</p>

<h2><span class="mw-headline" id=".EB.93.B1.EC.9E.A5.ED.95.98.EB.8A.94_.EB.AC.B4.EA.B8.B0.EB.93.A4">등장하는 무기들</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%B8%94%EB%9E%99_%ED%98%B8%ED%81%AC_%EB%8B%A4%EC%9A%B4_(%EC%98%81%ED%99%94)&amp;action=edit&amp;section=3" title="부분 편집: 등장하는 무기들">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<ul>
<li>75레인저 연대 대원들의 경우 <a class="mw-redirect" href="/wiki/M16A2_%EC%86%8C%EC%B4%9D" title="M16A2 소총">M16A2 소총</a>, <a class="mw-redirect" href="/wiki/M249_%EB%B6%84%EB%8C%80%EC%A7%80%EC%9B%90%ED%99%94%EA%B8%B0" title="M249 분대지원화기">M249 분대지원화기</a>, <a href="/wiki/M60_%EA%B8%B0%EA%B4%80%EC%B4%9D" title="M60 기관총">M60 기관총</a>을 사용한다.</li>
<li>험비에 장착된 화기는 <a class="mw-disambig" href="/wiki/M2" title="M2">M2</a> 브라운 중기관총을 사용한다.</li>
<li><a href="/wiki/%EB%8D%B8%ED%83%80_%ED%8F%AC%EC%8A%A4" title="델타 포스">델타 포스</a> 대원의 경우 <a class="new" href="/w/index.php?title=%EC%BD%9C%ED%8A%B8_%EC%BD%94%EB%A7%8C%EB%8F%84&amp;action=edit&amp;redlink=1" title="콜트 코만도 (없는 문서)">콜트 M733 코만도</a> <a class="new" href="/w/index.php?title=M1911_A1&amp;action=edit&amp;redlink=1" title="M1911 A1 (없는 문서)">M1911 A1</a> <a href="/wiki/%EA%B6%8C%EC%B4%9D" title="권총">권총</a>을 사용한다. 슈퍼 64 파일럿을 구출하러 간 <a class="new" href="/w/index.php?title=%EB%9E%9C%EB%94%94_%EC%85%94%EA%B7%B8%ED%95%98%ED%8A%B8&amp;action=edit&amp;redlink=1" title="랜디 셔그하트 (없는 문서)">랜디 셔그하트</a>의 경우 <a href="/wiki/M14_%EC%86%8C%EC%B4%9D" title="M14 소총">M14 소총</a> M1911 A1 권총을, <a href="/wiki/%EA%B2%8C%EB%A6%AC_%EA%B3%A0%EB%93%A0" title="게리 고든">게리 고든</a>은 <a class="new" href="/w/index.php?title=CAR-15&amp;action=edit&amp;redlink=1" title="CAR-15 (없는 문서)">CAR-15</a> M1911 A1 권총을 사용한다.</li>
<li><a class="mw-redirect" href="/wiki/UH-60_%EB%B8%94%EB%9E%99%ED%98%B8%ED%81%AC" title="UH-60 블랙호크">UH-60 블랙호크</a> <a href="/wiki/%ED%97%AC%EB%A6%AC%EC%BD%A5%ED%84%B0" title="헬리콥터">헬리콥터</a> 파일럿 마이클 듀란트는 <a class="mw-redirect" href="/wiki/MP5A3" title="MP5A3">MP5A3</a> 기관단총을 사용한다.</li>
<li>UH-60 블랙호크에 <a class="new" href="/w/index.php?title=M314&amp;action=edit&amp;redlink=1" title="M314 (없는 문서)">M314</a> <a class="mw-disambig" href="/wiki/%EB%B2%8C%EC%BB%A8" title="벌컨">벌컨</a>을 장착해 사용한다.</li>
<li><a href="/wiki/%EC%86%8C%EB%A7%90%EB%A6%AC%EC%95%84" title="소말리아">소말리아</a> 민병대는 <a href="/wiki/%EB%8F%8C%EA%B2%A9%EC%86%8C%EC%B4%9D" title="돌격소총">돌격소총</a> <a href="/wiki/AK-74" title="AK-74">AK-74</a>과 <a class="new" href="/w/index.php?title=%EB%8C%80%EC%A0%84%EC%B0%A8_%EB%A1%9C%EC%BC%93&amp;action=edit&amp;redlink=1" title="대전차 로켓 (없는 문서)">대전차 로켓</a> <a href="/wiki/RPG-7" title="RPG-7">RPG-7</a>을 사용한다.</li>
</ul>
