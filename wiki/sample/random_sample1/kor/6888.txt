<p><b>블랙스크린</b>(<span style="font-size: smaller;"><a href="/wiki/%EC%98%81%EC%96%B4" title="영어">영어</a>:</span> <span lang="en" xml:lang="en">black screen of death, <b>BkSoD</b></span> )은 <a class="mw-redirect" href="/wiki/%EB%A7%88%EC%9D%B4%ED%81%AC%EB%A1%9C%EC%86%8C%ED%94%84%ED%8A%B8_%EC%9C%88%EB%8F%84" title="마이크로소프트 윈도">윈도</a>의 <a href="/wiki/%EB%B6%88%EB%B2%95_%EB%B3%B5%EC%A0%9C" title="불법 복제">불법 복제</a>임이 밝혀지거나 <a class="mw-redirect" href="/wiki/%EC%A0%95%ED%92%88_%EC%9D%B8%EC%A6%9D" title="정품 인증">정품 인증</a>이 실패할 경우, <a href="/wiki/%EC%BB%B4%ED%93%A8%ED%84%B0" title="컴퓨터">컴퓨터</a>의 <a class="mw-redirect" href="/wiki/%EB%B0%94%ED%83%95%ED%99%94%EB%A9%B4" title="바탕화면">바탕화면</a>이 <a class="mw-redirect" href="/wiki/%EA%B2%80%EC%9D%80%EC%83%89" title="검은색">검은색</a>으로 변하는 것이다. 또는 <a href="/wiki/%EB%A7%A4%ED%82%A8%ED%86%A0%EC%8B%9C" title="매킨토시">매킨토시</a> 컴퓨터의 치명적 오류를 알리는 것이기도 한다.</p>
<h2><span class="mw-headline" id=".EB.A7.88.EC.9D.B4.ED.81.AC.EB.A1.9C.EC.86.8C.ED.94.84.ED.8A.B8_.EC.9C.88.EB.8F.84">마이크로소프트 윈도</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%B8%94%EB%9E%99%EC%8A%A4%ED%81%AC%EB%A6%B0&amp;action=edit&amp;section=1" title="부분 편집: 마이크로소프트 윈도">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>윈도의 정품 인증이 실패했을 경우, 컴퓨터의 바탕화면 색으로 검정색이 나오고, 불법 복제임을 알리는 팝업이 뜬다. 또한 부트 파일 오류시, <a href="/wiki/%EC%BB%A4%EB%84%90_%ED%8C%A8%EB%8B%89" title="커널 패닉">커널 패닉</a> 발생시에도 검은색이 나오기도 한다. <a href="/wiki/MS-DOS" title="MS-DOS">MS-DOS</a>, <a href="/wiki/%EC%9C%88%EB%8F%84%EC%9A%B0_95" title="윈도우 95">윈도우 95</a>, <a href="/wiki/%EC%9C%88%EB%8F%84%EC%9A%B0_98" title="윈도우 98">윈도우 98</a>, <a href="/wiki/%EC%9C%88%EB%8F%84%EC%9A%B0_2000" title="윈도우 2000">윈도우 2000</a>, <a href="/wiki/%EC%9C%88%EB%8F%84%EC%9A%B0_%EB%AF%B8" title="윈도우 미">윈도우 미</a>, <a href="/wiki/%EC%9C%88%EB%8F%84%EC%9A%B0_XP" title="윈도우 XP">윈도우 XP</a>, <a href="/wiki/%EC%9C%88%EB%8F%84%EC%9A%B0_%EB%B9%84%EC%8A%A4%ED%83%80" title="윈도우 비스타">윈도우 비스타</a>, <a href="/wiki/%EC%9C%88%EB%8F%84%EC%9A%B0_7" title="윈도우 7">윈도우 7</a>, <a href="/wiki/%EC%9C%88%EB%8F%84%EC%9A%B0_8" title="윈도우 8">윈도우 8</a>에서도 마찬가지로, 블랙스크린이 표시되는데, 보통 <a href="/wiki/%EC%8B%9C%EC%8A%A4%ED%85%9C" title="시스템">시스템</a> <a href="/wiki/%EB%B6%80%ED%8C%85" title="부팅">부팅</a>이 불가능한 경우에 표시된다.</p>
<h2><span class="mw-headline" id=".EB.A7.A5_OS">맥 OS</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%B8%94%EB%9E%99%EC%8A%A4%ED%81%AC%EB%A6%B0&amp;action=edit&amp;section=2" title="부분 편집: 맥 OS">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p><a href="/wiki/%EB%A7%A4%ED%82%A8%ED%86%A0%EC%8B%9C" title="매킨토시">매킨토시 컴퓨터</a> 의 치명적 오류시, <a href="/wiki/%EB%B8%94%EB%A3%A8%EC%8A%A4%ED%81%AC%EB%A6%B0" title="블루스크린">블루스크린</a>을 대신하여, 블랙스크린이 뜨기도 한다.</p>
