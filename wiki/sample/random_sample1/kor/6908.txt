<p><b>블루스크린</b>(<span style="font-size: smaller;"><a href="/wiki/%EC%98%81%EC%96%B4" title="영어">영어</a>:</span> <span lang="en" xml:lang="en">blue screen of death</span> <small>블루 스크린 오브 데스<sup>[<a href="/wiki/%EC%9C%84%ED%82%A4%EB%B0%B1%EA%B3%BC:%EC%98%81%EC%96%B4%EC%9D%98_%ED%95%9C%EA%B8%80_%ED%91%9C%EA%B8%B0" title="위키백과:영어의 한글 표기">*</a>]</sup></small>)은 <a href="/wiki/%EB%A7%88%EC%9D%B4%ED%81%AC%EB%A1%9C%EC%86%8C%ED%94%84%ED%8A%B8_%EC%9C%88%EB%8F%84%EC%9A%B0" title="마이크로소프트 윈도우">마이크로소프트 윈도우</a> 운영체제에서 시스템 오류를 복구하지 못할 때 나타나는 화면을 뜻한다. 이 화면은 <a class="mw-redirect" href="/wiki/%EC%9C%88%EB%8F%84_1.0" title="윈도 1.0">윈도 1.0</a>부터 존재했고, 특히 9x 계열(95, 98, me)에 비해 <a class="mw-redirect" href="/wiki/%EC%9C%88%EB%8F%84_NT" title="윈도 NT">윈도 NT</a> 계열이 비교적 자세한 오류 메시지를 출력한다.</p>
<p><a href="/wiki/%EC%9C%88%EB%8F%84%EC%9A%B0_%EB%B9%84%EC%8A%A4%ED%83%80" title="윈도우 비스타">윈도우 비스타</a>(당시 코드명 롱혼)의 초기 버전(빌드 4096 이하)은 부트 로더 오류가 났을 때 빨간 화면이 나타났었다. <a href="/wiki/%EC%9C%88%EB%8F%84%EC%9A%B0_98" title="윈도우 98">윈도우 98</a>에서는 실행 창에 C:\CON\CON을 입력하여 볼 수 있다.<sup class="reference" id="cite_ref-1"><a href="#cite_note-1">[1]</a></sup>윈도 NT 계열의 윈도우에서는 <a href="/wiki/%EC%84%B8%EC%85%98_%EA%B4%80%EB%A6%AC%EC%9E%90_%ED%95%98%EC%9C%84_%EC%8B%9C%EC%8A%A4%ED%85%9C" title="세션 관리자 하위 시스템">smss.exe</a>나 <a href="/wiki/Svchost.exe" title="Svchost.exe">svchost.exe</a>를 종료시키면 볼 수 있다.<sup class="reference" id="cite_ref-2"><a href="#cite_note-2">[2]</a></sup></p>
<p>블루스크린은 기본적으로 파란 배경 화면(EGA : 0x01; HTML : #0000AA)에 하얀색 글자(EGA : 0x0F; HTML : #FFFFFF)로 오류 내용을 표시한다. 윈도 95, 98 그리고 Me 는 80x25 텍스트 모드에 <a href="/wiki/%ED%94%BD%EC%8A%A4%ED%8A%B8%EC%8B%9C%EC%8A%A4" title="픽스트시스">픽스트시스</a> 폰트를 사용한다. NT는 80x50 텍스트 모드에 화면 해상도는 720x400이다. XP는 <a class="new" href="/w/index.php?title=%EB%A3%A8%EC%8B%9C%EB%8B%A4_%EC%BD%98%EC%86%94&amp;action=edit&amp;redlink=1" title="루시다 콘솔 (없는 문서)">루시다 콘솔</a> 폰트를 사용한다.</p>

<h2><span class="mw-headline" id=".EC.83.9D.EA.B9.80.EC.83.88">생김새</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%B8%94%EB%A3%A8%EC%8A%A4%ED%81%AC%EB%A6%B0&amp;action=edit&amp;section=1" title="부분 편집: 생김새">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>블루스크린은 윈도우 1.0부터 있었다. 윈도우 1.0의 블루스크린은 알 수 없는 문자로 되어 있다. 이후 윈도우 3.x버전부터 무엇이 문제인지 알려주고, 윈도우 XP부터 자세한 정보를 표시하게 되었다.</p>
<h2><span class="mw-headline" id=".EC.A3.BD.EC.9D.8C.EC.9D.98_.EB.B8.94.EB.A3.A8.EC.8A.A4.ED.81.AC.EB.A6.B0_.EC.9D.B4.ED.95.B4">죽음의 블루스크린 이해</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%B8%94%EB%A3%A8%EC%8A%A4%ED%81%AC%EB%A6%B0&amp;action=edit&amp;section=2" title="부분 편집: 죽음의 블루스크린 이해">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>블루스크린은 0x00000022와 같은 숫자와 더불어 FILE_SYSTEM과 같은 메시지를 보여 준다. 이 메시지에 따라 오류 원인을 파악할 수 있다. 블루 스크린에 나타나는 변수는 보통 다음과 같다:</p>
<blockquote>
<p><i><b>오류 번호 (변수, 변수, 변수, 변수) 오류 이름</b></i></p>
</blockquote>
<p>위의 정보는 블루 스크린의 원인을 이해하고 판단하는 데 중요하다.<sup class="reference" id="cite_ref-3"><a href="#cite_note-3">[3]</a></sup></p>
<p>“오류 번호” 값에 따라, 변수의 일부가 임의의 장소에서 잘못 연결된 데이터를 포함할 수 있다.</p>
<h2><span class="mw-headline" id=".EB.8C.80.ED.91.9C.EC.A0.81.EC.9C.BC.EB.A1.9C_.EB.B0.9C.EC.83.9D.ED.95.98.EB.8A.94_.EC.9D.BC">대표적으로 발생하는 일</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%B8%94%EB%A3%A8%EC%8A%A4%ED%81%AC%EB%A6%B0&amp;action=edit&amp;section=3" title="부분 편집: 대표적으로 발생하는 일">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<ul>
<li>프로그램이 응답을 하지 않을 경우에는 <b>시스템이 사용 중입니다</b>라고 제목으로 블루스크린이 뜨는데 이는 사용자가 종료를 시도할 때에 나오는 일이다.</li>
<li>하지만, 시스템을 종료하지 않은 상황인데 응답하지 않아 화면이 캄캄하거나 작동을 하지 않은 상황에서 Ctrl+alt+Del버튼을 누르면 <b>주의!</b>라고 뜬다. <b>시스템이 사용 중입니다</b>라고 블루스크린이 뜰 때에 아무키를 눌러도 <b>주의!</b>라고 스크린이 바뀔 때가 있는데 이는 사용자가 종료하지 않은 상황 즉 시스템이 계속 사용 중일 때 흔히 나오는 스크린이다. 하지만, 시스템이 불안정할 때에도 <b>주의!</b>가 뜨기도 하는데 레드스크린으로 나온다.</li>
</ul>
<h2><span class="mw-headline" id=".EA.B8.B0.ED.83.80">기타</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%B8%94%EB%A3%A8%EC%8A%A4%ED%81%AC%EB%A6%B0&amp;action=edit&amp;section=4" title="부분 편집: 기타">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
