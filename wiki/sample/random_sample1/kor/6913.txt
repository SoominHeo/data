<p><b>블리니</b>(blini) 또는 <b>블린</b>(blin)은 <a class="mw-redirect" href="/wiki/%EB%A9%94%EB%B0%80%EA%B0%80%EB%A3%A8" title="메밀가루">메밀가루</a>와 <a href="/wiki/%EB%B0%80%EA%B0%80%EB%A3%A8" title="밀가루">밀가루</a>를 섞거나 단독으로 사용하여 얇고 둥글게 부친 <a href="/wiki/%EB%9F%AC%EC%8B%9C%EC%95%84" title="러시아">러시아</a>식 <a href="/wiki/%ED%8C%AC%EC%BC%80%EC%9D%B4%ED%81%AC" title="팬케이크">팬케이크</a>를 뜻하며, 러시아 고어 <b>믈린</b>(mlin), 제분한 것 이라는 단어에서 유래됐다고 알려져있다.<sup class="reference" id="cite_ref-1"><a href="#cite_note-1">[1]</a></sup> <a href="/wiki/%EB%B2%84%ED%84%B0" title="버터">버터</a>, <a class="new" href="/w/index.php?title=%ED%9B%88%EC%A0%9C_%EC%97%B0%EC%96%B4&amp;action=edit&amp;redlink=1" title="훈제 연어 (없는 문서)">훈제 연어</a>, <a class="mw-redirect" href="/wiki/%EC%BA%90%EB%B9%84%EC%96%B4" title="캐비어">캐비어</a>, <a class="mw-redirect" href="/wiki/%EC%8A%A4%EB%A9%94%ED%83%80%EB%82%98" title="스메타나">스메타나</a>, <a class="mw-redirect" href="/wiki/%EA%BF%80" title="꿀">꿀</a> 등 여러 식재료와 곁들여 먹는다.<sup class="reference" id="cite_ref-2"><a href="#cite_note-2">[2]</a></sup></p>
<h2><span class="mw-headline" id=".EB.B8.94.EB.A6.AC.EB.8B.88.EC.9D.98_.EC.97.AD.EC.82.AC">블리니의 역사</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%B8%94%EB%A6%AC%EB%8B%88&amp;action=edit&amp;section=1" title="부분 편집: 블리니의 역사">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<div class="thumb tright">
<div class="thumbinner" style="width:222px;"><a class="image" href="/wiki/%ED%8C%8C%EC%9D%BC:Blini_with_salmon_roe.jpg"><img alt="" class="thumbimage" data-file-height="1389" data-file-width="1920" height="159" src="//upload.wikimedia.org/wikipedia/commons/thumb/d/d7/Blini_with_salmon_roe.jpg/220px-Blini_with_salmon_roe.jpg" srcset="//upload.wikimedia.org/wikipedia/commons/thumb/d/d7/Blini_with_salmon_roe.jpg/330px-Blini_with_salmon_roe.jpg 1.5x, //upload.wikimedia.org/wikipedia/commons/thumb/d/d7/Blini_with_salmon_roe.jpg/440px-Blini_with_salmon_roe.jpg 2x" width="220"/></a>
<div class="thumbcaption">
<div class="magnify"><a class="internal" href="/wiki/%ED%8C%8C%EC%9D%BC:Blini_with_salmon_roe.jpg" title="실제 크기로"></a></div>
연어알 블리니</div>
</div>
</div>
<p>정확한 시기는 알려지지 않았지만 <a class="mw-redirect" href="/wiki/%ED%82%A4%EC%98%88%ED%94%84_%EA%B3%B5%EA%B5%AD" title="키예프 공국">키예프 공국</a>이 10세기 말 <a href="/wiki/%EB%9F%AC%EC%8B%9C%EC%95%84_%EC%A0%95%EA%B5%90%ED%9A%8C" title="러시아 정교회">러시아 정교회</a>를 국교로 수용하기 이전부터 블리니는 존재했다고 여겨진다. 초기의 블리니는 다양한 곡식을 섞어 만들어 식감이 거칠었다.</p>
<p>거친 블리니는 <a class="mw-redirect" href="/wiki/%EC%8A%AC%EB%9D%BC%EB%B8%8C%EC%9D%B8" title="슬라브인">슬라브인</a>들이 긴 겨울이 끝나가는 시기에 농경, 목축의 신 <a class="mw-disambig" href="/wiki/%EB%B2%A8%EB%A0%88%EC%8A%A4" title="벨레스">벨레스</a>를 숭배하는 풍습을 러시아 정교에서 변형하여 수용함으로써 역사가 지속되었다고 본다.</p>
<p>블리니는 <a href="/wiki/%EC%B6%95%EC%A0%9C" title="축제">축제</a>의 성격을 띠고 있는 음식이기 때문에 러시아에선 아이를 낳은 산모가 먹기도하고 장례식에 참석한 이들이 먹는 음식이다. 하지만 일상적인 요리의 성격 또한 갖고 있다.</p>
<h2><span class="mw-headline" id=".EB.B8.94.EB.A6.AC.EB.8B.88.EC.9D.98_.EC.A1.B0.EB.A6.AC.EB.B2.95">블리니의 조리법</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%B8%94%EB%A6%AC%EB%8B%88&amp;action=edit&amp;section=2" title="부분 편집: 블리니의 조리법">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>따뜻한 물에 <a href="/wiki/%EC%84%A4%ED%83%95" title="설탕">설탕</a>과 <a href="/wiki/%ED%9A%A8%EB%AA%A8" title="효모">효모</a>를 넣고 부풀어오르면 <a href="/wiki/%EB%B2%84%ED%84%B0" title="버터">버터</a>, <a href="/wiki/%EC%9A%B0%EC%9C%A0" title="우유">우유</a>, <a class="new" href="/w/index.php?title=%EA%B3%A1%EB%AC%BC%EA%B0%80%EB%A3%A8&amp;action=edit&amp;redlink=1" title="곡물가루 (없는 문서)">곡물가루</a>를 넣어 섞어 발효 시킨다.</p>
<p>발효시킨 반죽이 2배 정도 부풀어오르면 우유, 달걀의 노른자, <a class="mw-redirect" href="/wiki/%EC%82%AC%EC%9B%8C%ED%81%AC%EB%A6%BC" title="사워크림">사워크림</a>, 곡물가루, 소금을 넣고 발효시키고 달걀 흰자를 거품 내어 발효 반죽에 섞는다.</p>
<p>반죽을 원형으로 얇게 펴서 굽는다. 블리니는 간단히 버터에 발라먹기도 하고 보통은 전채음식으로 준비할 때는 스메타나, 훈제 연어, 캐비어등으로 짭짤하게 먹는경우가 많고 후식식으로 준비할 때는 코티지 치즈, 과일, 과일잼, 꿀 등과 곁들여 먹는다.</p>
