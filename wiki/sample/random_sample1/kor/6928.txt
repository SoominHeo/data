<p><b>비동맹 운동</b>(非同盟運動, <span style="font-size: smaller;"><a href="/wiki/%EC%98%81%EC%96%B4" title="영어">영어</a>:</span> <span lang="en">Non-Aligned Movement, <b>NAM</b></span>)은 주요 강대국 블록에 공식적으로 속하지 않거나 이에 대항하려는 국가들로 이룬 국제 조직이다. 이 운동은 <a href="/wiki/%EC%9C%A0%EA%B3%A0%EC%8A%AC%EB%9D%BC%EB%B9%84%EC%95%84" title="유고슬라비아">유고슬라비아</a>의 <a href="/wiki/%EC%9A%94%EC%8B%9C%ED%94%84_%EB%B8%8C%EB%A1%9C%EC%A6%88_%ED%8B%B0%ED%86%A0" title="요시프 브로즈 티토">요시프 브로즈 티토</a> 대통령, <a href="/wiki/%EC%9D%B8%EB%8F%84" title="인도">인도</a>의 초대 수상 <a href="/wiki/%EC%9E%90%EC%99%80%ED%95%A0%EB%9E%84_%EB%84%A4%EB%A3%A8" title="자와할랄 네루">자와할랄 네루</a>, <a href="/wiki/%EC%9D%B8%EB%8F%84%EB%84%A4%EC%8B%9C%EC%95%84" title="인도네시아">인도네시아</a>의 초대 대통령 <a href="/wiki/%EC%88%98%EC%B9%B4%EB%A5%B4%EB%85%B8" title="수카르노">수카르노</a>, <a href="/wiki/%EC%9D%B4%EC%A7%91%ED%8A%B8" title="이집트">이집트</a>의 대통령 <a href="/wiki/%EA%B0%80%EB%A7%90_%EC%95%95%EB%8D%B8_%EB%82%98%EC%84%B8%EB%A5%B4" title="가말 압델 나세르">가말 압델 나세르</a>의 창안에서 나왔다. 비동맹 운동은 1961년 세르비아의 수도 <a href="/wiki/%EB%B2%A0%EC%98%A4%EA%B7%B8%EB%9D%BC%EB%93%9C" title="베오그라드">베오그라드</a>에서 창설되어, 2011년 기준으로 120개 회원국과 18개 참관국이 있다. 비동맹 운동의 목표는 <a href="/wiki/1979%EB%85%84" title="1979년">1979년</a> 아바나 선언에서 천명한 "<a href="/wiki/%EA%B0%95%EB%8C%80%EA%B5%AD" title="강대국">강대국</a>이나 블록에 대항할 뿐 아니라, <a href="/wiki/%EC%A0%9C%EA%B5%AD%EC%A3%BC%EC%9D%98" title="제국주의">제국주의</a>, <a href="/wiki/%EC%8B%9D%EB%AF%BC%EC%A3%BC%EC%9D%98" title="식민주의">식민주의</a>, <a href="/wiki/%EC%8B%A0%EC%8B%9D%EB%AF%BC%EC%A3%BC%EC%9D%98" title="신식민주의">신식민주의</a>, <a class="mw-redirect" href="/wiki/%EC%9D%B8%EC%A2%85%EC%A3%BC%EC%9D%98" title="인종주의">인종주의</a>, 모든 형태의 외국 침략, <a href="/wiki/%EC%A0%90%EB%A0%B9" title="점령">점령</a>, 지배, 간섭, 패권과 투쟁하여" "비동맹 국가들의 독립, 주권, 영토 통일, 안보"를 보장하는 것이다.<sup class="reference" id="cite_ref-1"><a href="#cite_note-1">[1]</a></sup> 이들은 <a class="mw-redirect" href="/wiki/%EA%B5%AD%EC%A0%9C_%EC%97%B0%ED%95%A9" title="국제 연합">국제 연합</a> 회원국 중 2/3를 차지하며, 특히 대부분 <a href="/wiki/%EC%A0%9C3%EC%84%B8%EA%B3%84" title="제3세계">제3세계</a>의 <a href="/wiki/%EA%B0%9C%EB%B0%9C%EB%8F%84%EC%83%81%EA%B5%AD" title="개발도상국">개발도상국</a>에 속하는 세계 인구의 55%를 점하고 있다.<sup class="reference" id="cite_ref-autogenerated1_2-0"><a href="#cite_note-autogenerated1-2">[2]</a></sup></p>
<p>회원국으로는 <a href="/wiki/%EC%9C%A0%EA%B3%A0%EC%8A%AC%EB%9D%BC%EB%B9%84%EC%95%84" title="유고슬라비아">유고슬라비아</a>, <a href="/wiki/%EC%9D%B8%EB%8F%84" title="인도">인도</a>, <a href="/wiki/%EA%B0%80%EB%82%98" title="가나">가나</a>, <a href="/wiki/%ED%8C%8C%ED%82%A4%EC%8A%A4%ED%83%84" title="파키스탄">파키스탄</a>, <a href="/wiki/%EC%95%8C%EC%A0%9C%EB%A6%AC" title="알제리">알제리</a>, <a href="/wiki/%EB%A6%AC%EB%B9%84%EC%95%84" title="리비아">리비아</a>, <a href="/wiki/%EC%8A%A4%EB%A6%AC%EB%9E%91%EC%B9%B4" title="스리랑카">스리랑카</a>, <a href="/wiki/%EC%9D%B4%EC%A7%91%ED%8A%B8" title="이집트">이집트</a>, <a href="/wiki/%EC%9D%B8%EB%8F%84%EB%84%A4%EC%8B%9C%EC%95%84" title="인도네시아">인도네시아</a>, <a href="/wiki/%EC%BF%A0%EB%B0%94" title="쿠바">쿠바</a>, <a href="/wiki/%EC%BD%9C%EB%A1%AC%EB%B9%84%EC%95%84" title="콜롬비아">콜롬비아</a>, <a href="/wiki/%EB%B2%A0%EB%84%A4%EC%88%98%EC%97%98%EB%9D%BC" title="베네수엘라">베네수엘라</a>, 1994년 이후에는 <a href="/wiki/%EB%82%A8%EC%95%84%ED%94%84%EB%A6%AC%EC%B9%B4_%EA%B3%B5%ED%99%94%EA%B5%AD" title="남아프리카 공화국">남아프리카 공화국</a>, <a href="/wiki/%EC%9D%B4%EB%9E%80" title="이란">이란</a>, <a href="/wiki/%EB%A7%90%EB%A0%88%EC%9D%B4%EC%8B%9C%EC%95%84" title="말레이시아">말레이시아</a>, 그리고 때론 <a href="/wiki/%EC%A4%91%ED%99%94%EC%9D%B8%EB%AF%BC%EA%B3%B5%ED%99%94%EA%B5%AD" title="중화인민공화국">중화인민공화국</a> 등이 있었다. <a href="/wiki/%EB%B8%8C%EB%9D%BC%EC%A7%88" title="브라질">브라질</a>은 비동맹 운동의 공식 회원국은 아니었으나, 이 운동의 목표에 동조했으며 비동맹 회의 정상 회담에 참관단을 자주 보냈다. 비동맹 운동은 <a class="mw-redirect" href="/wiki/%EB%B6%81%EB%8C%80%EC%84%9C%EC%96%91_%EC%A1%B0%EC%95%BD%EA%B8%B0%EA%B5%AC" title="북대서양 조약기구">북대서양 조약기구</a>나 <a class="mw-redirect" href="/wiki/%EB%B0%94%EB%A5%B4%EC%83%A4%EB%B0%94_%EC%A1%B0%EC%95%BD%EA%B8%B0%EA%B5%AC" title="바르샤바 조약기구">바르샤바 조약기구</a>처럼 긴밀하게 단결하고자 했으나, 제대로 응집하지 못했으며 회원국 상당수가 <a href="/wiki/%EA%B0%95%EB%8C%80%EA%B5%AD" title="강대국">강대국</a>과 사실상 동맹 관계를 맺었다. 게다가 회원국 사이에 심각한 분쟁이 벌어지기도 했다. (가령 인도와 파키스탄, 이란과 이라크) 비동맹 운동은 1979년 <a class="mw-redirect" href="/wiki/%EC%86%8C%EB%B9%84%EC%97%90%ED%8A%B8_%EC%97%B0%EB%B0%A9%EC%9D%98_%EC%95%84%ED%94%84%EA%B0%80%EB%8B%88%EC%8A%A4%ED%83%84_%EC%B9%A8%EA%B3%B5" title="소비에트 연방의 아프가니스탄 침공">소비에트 연방이 아프가니스탄을 침공</a>하자 내부적으로 분열되었다. 소비에트 연방과 동맹을 맺은 회원국들은 이 침공을 지지했지만 회원국(특히 이슬람 국가들)들은 그렇지 않았다.</p>
<p>비동맹 운동은 <a href="/wiki/%EB%83%89%EC%A0%84" title="냉전">냉전</a> 구도를 깨기 위해 창설되었으므로<sup class="reference" id="cite_ref-suvedi_3-0"><a href="#cite_note-suvedi-3">[3]</a></sup>, 냉전이 끝나자 그 의미가 퇴색된다. 창립 회원국인 <a href="/wiki/%EC%9C%A0%EA%B3%A0%EC%8A%AC%EB%9D%BC%EB%B9%84%EC%95%84_%ED%95%B4%EC%B2%B4" title="유고슬라비아 해체">유고슬라비아가 해체</a>된 뒤 유고슬라비아의 후계 국가들은 일부가 참관국 자격이긴 했지만 비동맹 운동에 별 관심을 나타내지 않았다. 2004년 <a href="/wiki/%EB%AA%B0%ED%83%80" title="몰타">몰타</a>와 <a href="/wiki/%ED%82%A4%ED%94%84%EB%A1%9C%EC%8A%A4" title="키프로스">키프로스</a>가 비동맹 운동에서 탈퇴하고 <a href="/wiki/%EC%9C%A0%EB%9F%BD_%EC%97%B0%ED%95%A9" title="유럽 연합">유럽 연합</a>에 가입했다.</p>
