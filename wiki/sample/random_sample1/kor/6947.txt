<p><b>비비언 수</b>(<small><a href="/wiki/%EC%A4%91%EA%B5%AD%EC%96%B4" title="중국어">중국어</a></small><small>:</small> <span lang="zh" xml:lang="zh">徐若瑄</span>, <small><a href="/wiki/%ED%95%9C%EC%96%B4_%EB%B3%91%EC%9D%8C" title="한어 병음">병음</a>:</small> <font face="Lucida Sans Unicode">Xú Ruòxuān</font>,  <small>쉬뤄쉬안<sup>[<a href="/wiki/%EC%9C%84%ED%82%A4%EB%B0%B1%EA%B3%BC:%EC%A4%91%EA%B5%AD%EC%96%B4%EC%9D%98_%ED%95%9C%EA%B8%80_%ED%91%9C%EA%B8%B0" title="위키백과:중국어의 한글 표기">*</a>]</sup></small>, <small><a href="/wiki/%ED%95%9C%EA%B5%AD_%ED%95%9C%EC%9E%90%EC%9D%8C" title="한국 한자음">한자음</a>:</small> 서약선, <span style="font-size: smaller;"><a href="/wiki/%EC%98%81%EC%96%B4" title="영어">영어</a>:</span> <span lang="en" xml:lang="en">Vivian Hsu</span>, <span style="font-size: smaller;"><a href="/wiki/%EC%9D%BC%EB%B3%B8%EC%96%B4" title="일본어">일본어</a>:</span> <span lang="ja" xml:lang="ja">ビビアン・スー</span>, <a href="/wiki/1975%EB%85%84" title="1975년">1975년</a> <a href="/wiki/3%EC%9B%94_19%EC%9D%BC" title="3월 19일">3월 19일</a> ~ )는 <a href="/wiki/%EC%A4%91%ED%99%94%EB%AF%BC%EA%B5%AD" title="중화민국">중화민국</a> <a href="/wiki/%ED%83%80%EC%9D%B4%EC%A4%91_%ED%98%84" title="타이중 현">타이중 현</a> 출신의 <a class="new" href="/w/index.php?title=%EC%97%AC%EC%84%B1_%ED%83%A4%EB%9F%B0%ED%8A%B8&amp;action=edit&amp;redlink=1" title="여성 탤런트 (없는 문서)">여성 탤런트</a>로, <a href="/wiki/%ED%95%9C%EC%A1%B1" title="한족">한족</a>과 <a href="/wiki/%ED%83%80%EC%9D%B4%EC%99%84_%EC%9B%90%EC%A3%BC%EB%AF%BC" title="타이완 원주민">타이완 원주민</a>인 <a class="new" href="/w/index.php?title=%EC%95%84%ED%83%80%EC%96%84%EC%A1%B1&amp;action=edit&amp;redlink=1" title="아타얄족 (없는 문서)">아타얄족</a>의 혼혈이다. 주로 <a href="/wiki/%EC%A4%91%ED%99%94%EB%AF%BC%EA%B5%AD" title="중화민국">타이완</a>과 <a href="/wiki/%EC%9D%BC%EB%B3%B8" title="일본">일본</a>에서 활동한다.</p>
