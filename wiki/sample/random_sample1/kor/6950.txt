<div style="font-size: 9pt">비셰그라드의 위치</div>
</div>
<p><b>비셰그라드</b>(<span style="font-size: smaller;"><a href="/wiki/%ED%97%9D%EA%B0%80%EB%A6%AC%EC%96%B4" title="헝가리어">헝가리어</a>:</span> <span lang="hu" xml:lang="hu">Visegrád</span>, <span style="font-size: smaller;"><a href="/wiki/%EC%8A%AC%EB%A1%9C%EB%B0%94%ED%82%A4%EC%95%84%EC%96%B4" title="슬로바키아어">슬로바키아어</a>:</span> <span lang="sk" xml:lang="sk">Vyšehrad</span> <small>비셰흐라트</small>, <span style="font-size: smaller;"><a href="/wiki/%EB%8F%85%EC%9D%BC%EC%96%B4" title="독일어">독일어</a>:</span> <span lang="de" xml:lang="de">Plintenburg</span> <small>핀텐부르크<sup>[<a href="/wiki/%EC%9C%84%ED%82%A4%EB%B0%B1%EA%B3%BC:%EB%8F%85%EC%9D%BC%EC%96%B4%EC%9D%98_%ED%95%9C%EA%B8%80_%ED%91%9C%EA%B8%B0" title="위키백과:독일어의 한글 표기">*</a>]</sup></small>)는 <a href="/wiki/%ED%97%9D%EA%B0%80%EB%A6%AC" title="헝가리">헝가리</a> <a href="/wiki/%ED%8E%98%EC%8A%88%ED%8A%B8_%EC%A3%BC" title="페슈트 주">페슈트 주</a>에 위치한 도시로 면적은 33.27㎢, 인구는 1,711명(2004년 기준), 인구 밀도는 51.42명/㎢이다. <a class="mw-redirect" href="/wiki/%EB%8B%A4%EB%89%B4%EB%B8%8C_%EA%B0%95" title="다뉴브 강">다뉴브 강</a> 우안과 접하고 있고 <a href="/wiki/%EB%B6%80%EB%8B%A4%ED%8E%98%EC%8A%A4%ED%8A%B8" title="부다페스트">부다페스트</a> 북쪽에 위치한다.</p>
<p>초기 <a href="/wiki/%EB%A5%B4%EB%84%A4%EC%83%81%EC%8A%A4" title="르네상스">르네상스</a> 시대의 유적이 남아 있으며 헝가리의 국왕 <a href="/wiki/%EB%A7%88%EC%B0%A8%EC%8B%9C_1%EC%84%B8" title="마차시 1세">마차시 1세</a>의 여름 궁전과 중세 시대 요새로 유명한 도시이다. 도시 이름은 <a href="/wiki/%EC%8A%AC%EB%9D%BC%EB%B8%8C%EC%96%B4%ED%8C%8C" title="슬라브어파">슬라브어파</a>에 속하는 언어로 "위쪽에 있는 요새" 또는 "위쪽에 있는 마을"을 뜻하는 단어에서 유래된 이름이다.</p>
<h2><span class="mw-headline" id=".EC.97.AD.EC.82.AC">역사</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%B9%84%EC%85%B0%EA%B7%B8%EB%9D%BC%EB%93%9C_(%ED%97%9D%EA%B0%80%EB%A6%AC)&amp;action=edit&amp;section=1" title="부분 편집: 역사">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<div class="thumb tright">
<div class="thumbinner" style="width:252px;"><a class="image" href="/wiki/%ED%8C%8C%EC%9D%BC:Visegr%C3%A1d_Mountain_Fastness_01.jpg"><img alt="" class="thumbimage" data-file-height="734" data-file-width="1039" height="177" src="//upload.wikimedia.org/wikipedia/commons/thumb/1/1f/Visegr%C3%A1d_Mountain_Fastness_01.jpg/250px-Visegr%C3%A1d_Mountain_Fastness_01.jpg" srcset="//upload.wikimedia.org/wikipedia/commons/thumb/1/1f/Visegr%C3%A1d_Mountain_Fastness_01.jpg/375px-Visegr%C3%A1d_Mountain_Fastness_01.jpg 1.5x, //upload.wikimedia.org/wikipedia/commons/thumb/1/1f/Visegr%C3%A1d_Mountain_Fastness_01.jpg/500px-Visegr%C3%A1d_Mountain_Fastness_01.jpg 2x" width="250"/></a>
<div class="thumbcaption">
<div class="magnify"><a class="internal" href="/wiki/%ED%8C%8C%EC%9D%BC:Visegr%C3%A1d_Mountain_Fastness_01.jpg" title="실제 크기로"></a></div>
비셰그라드 성</div>
</div>

<p><a href="/wiki/1009%EB%85%84" title="1009년">1009년</a> 문헌에 처음 등장했다. <a href="/wiki/1242%EB%85%84" title="1242년">1242년</a> <a href="/wiki/%EB%AA%BD%EA%B3%A8_%EC%A0%9C%EA%B5%AD" title="몽골 제국">몽골 제국</a>의 유럽 침공 때 남쪽으로 약간 옮겨진 채로 재건되었다. <a href="/wiki/1325%EB%85%84" title="1325년">1325년</a> 헝가리의 국왕 <a href="/wiki/%EC%B9%B4%EB%A1%9C%EC%9D%B4_1%EC%84%B8" title="카로이 1세">카로이 1세</a>는 자신의 거주지였던 이 곳을 왕실의 거처로 선포하게 된다.</p>
<p><a href="/wiki/1335%EB%85%84" title="1335년">1335년</a> 헝가리의 국왕 카로이 1세는 <a href="/wiki/%EB%B3%B4%ED%97%A4%EB%AF%B8%EC%95%84" title="보헤미아">보헤미아</a>의 국왕 <a href="/wiki/%EC%96%80_%EB%A3%A8%EC%B3%84%EB%B6%80%EB%A5%B4%EC%8A%A4%ED%82%A4" title="얀 루쳄부르스키">얀 루쳄부르스키</a>, <a href="/wiki/%ED%8F%B4%EB%9E%80%EB%93%9C" title="폴란드">폴란드</a>의 국왕 <a href="/wiki/%EC%B9%B4%EC%A7%80%EB%AF%B8%EC%97%90%EC%8B%9C_3%EC%84%B8" title="카지미에시 3세">카지미에시 3세</a>와 함께 2개월에 걸친 회의를 개최했다. 이 회의는 헝가리와 보헤미아, 폴란드 세 왕국 간의 평화를 이끄는 한편 헝가리와 폴란드가 <a href="/wiki/%EC%98%A4%EC%8A%A4%ED%8A%B8%EB%A6%AC%EC%95%84" title="오스트리아">오스트리아</a>의 <a class="mw-redirect" href="/wiki/%ED%95%A9%EC%8A%A4%EB%B6%80%EB%A5%B4%ED%81%AC_%EC%99%95%EA%B0%80" title="합스부르크 왕가">합스부르크 왕가</a>에 대항하기 위해 동맹을 결성하는 계기가 된다. <a href="/wiki/1338%EB%85%84" title="1338년">1338년</a> 또 다른 회의가 개최되었다.</p>
<p><a href="/wiki/1405%EB%85%84" title="1405년">1405년</a>부터 <a href="/wiki/1408%EB%85%84" title="1408년">1408년</a> 사이에 <a href="/wiki/%EC%8B%A0%EC%84%B1_%EB%A1%9C%EB%A7%88_%EC%A0%9C%EA%B5%AD" title="신성 로마 제국">신성 로마 제국</a>의 황제 겸 헝가리의 국왕이었던 <a href="/wiki/%EC%A7%80%EA%B8%B0%EC%8A%A4%EB%AC%B8%ED%8A%B8_(%EC%8B%A0%EC%84%B1_%EB%A1%9C%EB%A7%88_%EC%A0%9C%EA%B5%AD)" title="지기스문트 (신성 로마 제국)">지기스문트</a>가 왕실의 거처를 <a href="/wiki/%EB%B6%80%EB%8D%94" title="부더">부더</a>로 옮겼다. <a href="/wiki/%EB%A7%88%EC%B0%A8%EC%8B%9C_1%EC%84%B8" title="마차시 1세">마차시 1세</a> 시대에는 한때 왕궁 소재지로 여겨지기도 했지만 <a href="/wiki/1526%EB%85%84" title="1526년">1526년</a> 헝가리 왕국이 <a href="/wiki/%EB%AA%A8%ED%95%98%EC%B9%98_%EC%A0%84%ED%88%AC" title="모하치 전투">모하치 전투</a>를 계기로 분할되면서 중요성을 상실하게 된다.</p>
<p><a href="/wiki/1991%EB%85%84" title="1991년">1991년</a> <a href="/wiki/%ED%97%9D%EA%B0%80%EB%A6%AC" title="헝가리">헝가리</a>와 <a href="/wiki/%EC%B2%B4%EC%BD%94%EC%8A%AC%EB%A1%9C%EB%B0%94%ED%82%A4%EC%95%84" title="체코슬로바키아">체코슬로바키아</a>, <a href="/wiki/%ED%8F%B4%EB%9E%80%EB%93%9C" title="폴란드">폴란드</a>의 정치 지도자들은 1335년 이 곳에서 열렸던 회의를 상기시키는 한편 이들 국가 간의 지역 협력체인 <a href="/wiki/%EB%B9%84%EC%85%B0%EA%B7%B8%EB%9D%BC%EB%93%9C_%EA%B7%B8%EB%A3%B9" title="비셰그라드 그룹">비셰그라드 그룹</a>을 결성했다. <a href="/wiki/2000%EB%85%84" title="2000년">2000년</a> 시로 승격되었다.</p>
