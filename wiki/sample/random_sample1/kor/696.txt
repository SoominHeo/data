<h2><span class="mw-headline" id=".EC.88.98.ED.95.99">수학</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=81&amp;action=edit&amp;section=1" title="부분 편집: 수학">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<ul>
<li><a href="/wiki/%ED%95%A9%EC%84%B1%EC%88%98" title="합성수">합성수</a>로, 그 <a href="/wiki/%EC%95%BD%EC%88%98" title="약수">약수</a>는 <a href="/wiki/1" title="1">1</a>, <a href="/wiki/3" title="3">3</a>, <a href="/wiki/9" title="9">9</a>, <a href="/wiki/27" title="27">27</a>, 81이다. (세제곱루트81은 3의세제곱루트3이다.)</li>
<li>9번째 <a class="mw-redirect" href="/wiki/%EC%A0%9C%EA%B3%B1%EC%88%98" title="제곱수">제곱수</a>이다.</li>
<li>3번째 <a href="/wiki/%EB%84%A4%EC%A0%9C%EA%B3%B1%EC%88%98" title="네제곱수">네제곱수</a>이다.</li>
</ul>
<h2><span class="mw-headline" id=".EA.B3.BC.ED.95.99">과학</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=81&amp;action=edit&amp;section=2" title="부분 편집: 과학">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<ul>
<li><a href="/wiki/%ED%83%88%EB%A5%A8" title="탈륨">탈륨</a>(Tl)의 <a class="mw-redirect" href="/wiki/%EC%9B%90%EC%9E%90%EB%B2%88%ED%98%B8" title="원자번호">원자번호</a>.</li>
</ul>
<h2><span class="mw-headline" id=".EA.B8.B0.ED.83.80">기타</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=81&amp;action=edit&amp;section=3" title="부분 편집: 기타">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<ul>
<li>전화번호: <a href="/wiki/%EC%9D%BC%EB%B3%B8" title="일본">일본</a>의 <a href="/wiki/%EA%B5%AD%EC%A0%9C%EC%A0%84%ED%99%94" title="국제전화">국제전화</a> 국가번호이다.</li>
<li>일반적인 <a href="/wiki/%EC%8A%A4%EB%8F%84%EC%BF%A0" title="스도쿠">스도쿠</a>에는 총 81칸이 있다.</li>
<li>드라마 <a href="/wiki/%EC%A3%BC%EB%AA%BD_(%EB%93%9C%EB%9D%BC%EB%A7%88)" title="주몽 (드라마)">주몽</a>과 <a href="/wiki/%ED%86%A0%EC%9A%94%EB%AF%B8%EC%8A%A4%ED%85%8C%EB%A6%AC_%EA%B7%B9%EC%9E%A5" title="토요미스테리 극장">토요미스테리 극장</a>은 총 81회 방영되었다.</li>
</ul>
