<p><b>비야네 스트롭스트룹</b>(<span style="font-size: smaller;"><a href="/wiki/%EB%8D%B4%EB%A7%88%ED%81%AC%EC%96%B4" title="덴마크어">덴마크어</a>:</span> <span lang="da" xml:lang="da">Bjarne Stroustrup</span>, <a href="/wiki/1950%EB%85%84" title="1950년">1950년</a> <a href="/wiki/12%EC%9B%94_30%EC%9D%BC" title="12월 30일">12월 30일</a> ~ )은 <a href="/wiki/C%2B%2B" title="C++">C++ 프로그래밍 언어</a>를 개발한 것으로 유명한 <a href="/wiki/%EB%8D%B4%EB%A7%88%ED%81%AC" title="덴마크">덴마크</a>의 컴퓨터 과학자이며, 2014년 현재 <a href="/wiki/%EB%AA%A8%EA%B1%B4_%EC%8A%A4%ED%83%A0%EB%A6%AC" title="모건 스탠리">모건 스탠리</a>에서 기술부서 전무(Managing Director)로 재직중이며, <a href="/wiki/%EC%BB%AC%EB%9F%BC%EB%B9%84%EC%95%84_%EB%8C%80%ED%95%99%EA%B5%90" title="컬럼비아 대학교">컬럼비아 대학교</a> 컴퓨터공학과의 객원교수이자 <a href="/wiki/%ED%85%8D%EC%82%AC%EC%8A%A4_A%26M_%EB%8C%80%ED%95%99%EA%B5%90" title="텍사스 A&amp;M 대학교">텍사스 A&amp;M 대학교</a> 컴퓨터공학과의 연구특임교수이다.<sup class="reference" id="cite_ref-1"><a href="#cite_note-1">[1]</a></sup></p>
<p>그의 이름을 어떻게 발음해야 하는지에 대해서는 <a class="external text" href="http://www.stroustrup.com/bs_faq.html#pronounce" rel="nofollow">비야네의 홈페이지</a>를 참조하기 바란다. Stroustrup의 첫 번째 u는 v의 음가를 갖는다고 설명하고 있다.</p>

<h2><span class="mw-headline" id=".EA.B5.90.EC.9C.A1">교육</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%B9%84%EC%95%BC%EB%84%A4_%EC%8A%A4%ED%8A%B8%EB%A1%AD%EC%8A%A4%ED%8A%B8%EB%A3%B9&amp;action=edit&amp;section=1" title="부분 편집: 교육">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>스트롭스트룹은 <a href="/wiki/1975%EB%85%84" title="1975년">1975년</a> <a href="/wiki/%EB%8D%B4%EB%A7%88%ED%81%AC" title="덴마크">덴마크</a>의 Aarhus 대학에서 수학과 컴퓨터 과학 석사를 받았고, <a href="/wiki/1979%EB%85%84" title="1979년">1979년</a> <a href="/wiki/%EC%98%81%EA%B5%AD" title="영국">영국</a> <a href="/wiki/%EC%BC%80%EC%9E%84%EB%B8%8C%EB%A6%AC%EC%A7%80_%EB%8C%80%ED%95%99%EA%B5%90" title="케임브리지 대학교">케임브리지 대학교</a>에서 컴퓨터 과학 분야의 박사학위를 받았다. 캠브릿지에서 그의 논문 지도교수는 David Wheeler이다</p>
<h2><span class="mw-headline" id=".EA.B2.BD.EB.A0.A5">경력</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%B9%84%EC%95%BC%EB%84%A4_%EC%8A%A4%ED%8A%B8%EB%A1%AD%EC%8A%A4%ED%8A%B8%EB%A3%B9&amp;action=edit&amp;section=2" title="부분 편집: 경력">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>스트롭스트룹은 1978년부터 C++을 개발하기 시작하였으며 (당시엔 "클래스를 가진 C"(C with Classes)라고 불렸다), 그의 말에 따르면 "C++을 발명하였고, 초기 정의 부분을 썼으며, 최초의 실행물을 만들었고, C++의 디자인 규격을 선택하고 생성하였으며, 모든 주요기능을 디자인하였고, C++ 표준위원에서 확장제안을 처리하는 책임자였다"고 한다. 또한 스트롭스트룹은 많은 사람들이 C++언어의 표준 참고서라고 생각하는 책인 <a class="mw-redirect" href="/wiki/C%2B%2B%ED%94%84%EB%A1%9C%EA%B7%B8%EB%9E%98%EB%B0%8D_%EC%96%B8%EC%96%B4" title="C++프로그래밍 언어">C++프로그래밍 언어</a>를 저술하였다.</p>
<p>스트롭스트룹은 <a href="/wiki/2002%EB%85%84" title="2002년">2002년</a>까지는 AT&amp;T 벨 연구소에서 대형 프로그래밍 연구부(Large-scale Programming Research department)의 부서장을 역임하였다. 또한 미국 공학학회 회원으로 2004년에 선출된 적이 있다. 스트롭스트룹은 또한 ACM(1994)의 회원(Fellow)이며 IEEE회원(Fellow)이다. 그는 현재 저명한 교수로써 <a href="/wiki/%EB%AF%B8%EA%B5%AD" title="미국">미국</a><a class="mw-redirect" href="/wiki/%ED%85%8D%EC%82%AC%EC%8A%A4" title="텍사스">텍사스</a>A&amp;M대학교의 공과대학에서 컴퓨터 과학 석좌교수로 재직중이다.</p>
<h2><span class="mw-headline" id=".EC.A0.80.EC.84.9C">저서</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%B9%84%EC%95%BC%EB%84%A4_%EC%8A%A4%ED%8A%B8%EB%A1%AD%EC%8A%A4%ED%8A%B8%EB%A3%B9&amp;action=edit&amp;section=3" title="부분 편집: 저서">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>스트롭스트룹은 다음과 같은 여러 책의 저자 또는 공저자이다.</p>
