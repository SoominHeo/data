<p><b>비야리카 산</b>(Villarrica)은 <a href="/wiki/%EC%B9%A0%EB%A0%88" title="칠레">칠레</a> 남부에 있는 큰 <a href="/wiki/%EC%84%B1%EC%B8%B5_%ED%99%94%EC%82%B0" title="성층 화산">성층 화산</a>으로, 높이는 2,860m이다. 여전히 활동하는 <a href="/wiki/%ED%99%9C%ED%99%94%EC%82%B0" title="활화산">활화산</a>이다. 이곳은 <a href="/wiki/%EC%B9%A0%EB%A0%88" title="칠레">칠레</a>의 <a href="/wiki/%ED%99%94%EC%82%B0" title="화산">화산</a>들 중에서도 활동이 가장 활발한 화산 중 하나로 꼽히며, 때때로 산정 <a href="/wiki/%ED%99%94%EA%B5%AC" title="화구">화구</a>에서 <a href="/wiki/%EC%9A%A9%EC%95%94" title="용암">용암</a>을 동반한 분화를 일으키기도 한다. <a href="/wiki/%ED%99%94%EA%B5%AC" title="화구">화구</a> 안에는 <a href="/wiki/%EC%9A%A9%EC%95%94%ED%98%B8" title="용암호">용암호</a>가 나타나기도 하며, 분석구가 형성되는 경우도 있다.</p>
<p>많은 관광객들은 등산을 하러 이 곳을 찾으며, <a href="/wiki/2016%EB%85%84" title="2016년">2016년</a>에 마지막으로 <a href="/wiki/%EB%B6%84%ED%99%94_(%ED%99%94%EC%82%B0%ED%95%99)" title="분화 (화산학)">분화</a>하였다. 비야리카라는 이름은 <a href="/wiki/%ED%99%94%EC%82%B0" title="화산">화산</a> 옆에 <a href="/wiki/%EB%B9%84%EC%95%BC%EB%A6%AC%EC%B9%B4_(%EC%B9%A0%EB%A0%88)" title="비야리카 (칠레)">비야리카</a>라는 이름의 도시가 있어 붙은 이름이다.</p>
