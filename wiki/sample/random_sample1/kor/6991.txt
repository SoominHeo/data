<p><b>비치케이 베르털런</b>(<span style="font-size: smaller;"><a href="/wiki/%ED%97%9D%EA%B0%80%EB%A6%AC%EC%96%B4" title="헝가리어">헝가리어</a>:</span> <span lang="hu" xml:lang="hu">Bicskei Bertalan</span>, <a href="/wiki/1944%EB%85%84" title="1944년">1944년</a> <a href="/wiki/9%EC%9B%94_17%EC%9D%BC" title="9월 17일">9월 17일</a> ~ <a href="/wiki/2011%EB%85%84" title="2011년">2011년</a> <a href="/wiki/7%EC%9B%94_17%EC%9D%BC" title="7월 17일">7월 17일</a>)은 <a href="/wiki/%ED%97%9D%EA%B0%80%EB%A6%AC" title="헝가리">헝가리</a> 출신의 전 <a href="/wiki/%EC%B6%95%EA%B5%AC" title="축구">축구</a> 선수 및 지도자이다.</p>
<p><a href="/wiki/%ED%97%9D%EA%B0%80%EB%A6%AC_%EC%B6%95%EA%B5%AC_%EA%B5%AD%EA%B0%80%EB%8C%80%ED%91%9C%ED%8C%80" title="헝가리 축구 국가대표팀">헝가리 축구 국가대표팀</a> 감독을 역임하였으며 <a href="/wiki/1990%EB%85%84" title="1990년">1990년</a> 11월 <a class="mw-redirect" href="/wiki/%EB%B6%80%EC%82%B0_%EB%8C%80%EC%9A%B0_%EB%A1%9C%EC%96%84%EC%A6%88" title="부산 대우 로얄즈">대우 로얄즈</a>의 감독으로 선임되었으며 <a href="/wiki/K%EB%A6%AC%EA%B7%B8_1991" title="K리그 1991">1991</a>시즌을 우승으로 이끌었다. 원래는 <a href="/wiki/1990%EB%85%84_FIFA_%EC%9B%94%EB%93%9C%EC%BB%B5" title="1990년 FIFA 월드컵">1990년 FIFA 월드컵</a>에서 <a href="/wiki/%EC%B2%B4%EC%BD%94%EC%8A%AC%EB%A1%9C%EB%B0%94%ED%82%A4%EC%95%84_%EC%B6%95%EA%B5%AC_%EA%B5%AD%EA%B0%80%EB%8C%80%ED%91%9C%ED%8C%80" title="체코슬로바키아 축구 국가대표팀">체코슬로바키아 축구 국가대표팀</a>을 8강에 올려놓은 <a class="extiw" href="https://en.wikipedia.org/wiki/Jozef_Venglo%C5%A1" title="en:Jozef Vengloš">벵글로스</a> 감독을 선임하려고 하였으나 높아진 몸값으로 데려오지 못하였다.<sup class="reference" id="cite_ref-1"><a href="#cite_note-1">[1]</a></sup> <a href="/wiki/K%EB%A6%AC%EA%B7%B8_1991" title="K리그 1991">K리그 1991 시즌</a> 대우의 감독으로 재임하며 21경기 무패 신기록 (1991/5/8~1991/8/31)을 세우며 클럽 통산 3번째 우승으로 이끌며 첫 번째 K리그를 제패하고 <a href="/wiki/K%EB%A6%AC%EA%B7%B8_%EA%B0%90%EB%8F%85%EC%83%81" title="K리그 감독상">K리그 감독상</a>을 수상한 외국인 감독이 되었으나 재계약에 실패하여 <a href="/wiki/K%EB%A6%AC%EA%B7%B8" title="K리그">K리그</a>에서 우승하고도 감독직을 연장하지 못한 첫 번째 감독이 되었다. <a href="/wiki/2011%EB%85%84" title="2011년">2011년</a>에 타계하였으며 비츠케이의 아들이 과거 대우 로얄즈 시절 비츠케이 감독의 영상을 부탁하기도 하였다.<sup class="reference" id="cite_ref-2"><a href="#cite_note-2">[2]</a></sup></p>
