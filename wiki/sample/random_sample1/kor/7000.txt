<p><b>비탈리 블라디미로비치 쿠투조프</b> (<span style="font-size: smaller;"><a href="/wiki/%EB%B2%A8%EB%9D%BC%EB%A3%A8%EC%8A%A4%EC%96%B4" title="벨라루스어">벨라루스어</a>:</span> <span lang="be" xml:lang="be">Віталь Кутузаў</span>, <a href="/wiki/1980%EB%85%84" title="1980년">1980년</a> <a href="/wiki/3%EC%9B%94_20%EC%9D%BC" title="3월 20일">3월 20일</a>, <a href="/wiki/%ED%95%80%EC%8A%A4%ED%81%AC" title="핀스크">핀스크</a> ~) 는 <a href="/wiki/%EB%B2%A8%EB%9D%BC%EB%A3%A8%EC%8A%A4" title="벨라루스">벨라루스</a>의 축구 선수이다.</p>
<p><a href="/wiki/%EB%B2%A8%EB%9D%BC%EB%A3%A8%EC%8A%A4_%ED%94%84%EB%A6%AC%EB%AF%B8%EC%96%B4%EB%A6%AC%EA%B7%B8" title="벨라루스 프리미어리그">벨라루스 프리미어리그</a>의 <a class="mw-redirect" href="/wiki/FC_BATE" title="FC BATE">FC BATE</a>에서 선수 생활을 시작해, <a href="/wiki/2001%EB%85%84" title="2001년">2001년</a> <a class="mw-redirect" href="/wiki/%EC%9D%B4%ED%83%88%EB%A6%AC%EC%95%84_%EC%84%B8%EB%A6%AC%EC%97%90_A" title="이탈리아 세리에 A">이탈리아 세리에 A</a>의 <a href="/wiki/AC_%EB%B0%80%EB%9E%80" title="AC 밀란">AC 밀란</a>으로 이적하였다. 쿠투조프는 AC 밀란 1군이 된 후 시즌 반 동안 겨우 두 경기에 출장해 팀 내 입지 확보에 실패하였다.</p>
<p>2002-03 시즌 <a class="mw-redirect" href="/wiki/%ED%8F%AC%EB%A5%B4%ED%88%AC%EA%B0%88_%EB%A6%AC%EA%B0%80" title="포르투갈 리가">포르투갈 리가</a>의 <a href="/wiki/%EC%8A%A4%ED%8F%AC%EB%A5%B4%ED%8C%85_CP" title="스포르팅 CP">스포르팅 CP</a>로 임대되었으며, <a class="mw-redirect" href="/wiki/UEFA%EC%BB%B5" title="UEFA컵">UEFA컵</a>과 여러 컵 대회를 포함해 30경기에서 7골을 기록하였다. 2003-04 시즌 다시 임대돼 <a href="/wiki/%EC%84%B8%EB%A6%AC%EC%97%90_B" title="세리에 B">세리에 B</a>의 <a class="mw-redirect" href="/wiki/AS_%EC%95%84%EB%B2%A8%EB%A6%AC%EB%85%B8" title="AS 아벨리노">US 아벨리노</a>로 자리를 옮겼으며, 43경기에서 15골을 넣으며 중요한 선수 중 하나로서 팬들의 주목을 받았으나, 그의 활약에도 불구하고 팀은 <a class="mw-redirect" href="/wiki/%EC%84%B8%EB%A6%AC%EC%97%90_C1" title="세리에 C1">세리에 C1</a>으로 강등되었다.</p>
<p><a href="/wiki/%EC%84%B8%EB%A6%AC%EC%97%90_B" title="세리에 B">세리에 B</a>에서 성공적인 시즌을 보낸 후, <a href="/wiki/%EC%84%B8%EB%A6%AC%EC%97%90_A" title="세리에 A">세리에 A</a>의 <a href="/wiki/UC_%EC%82%BC%ED%94%84%EB%8F%84%EB%A6%AC%EC%95%84" title="UC 삼프도리아">UC 삼프도리아</a>로 이적했으며, 두 시즌 간 (2004-05, 2005-06) 62경기에서 7골을 넣었다.</p>
<p><a href="/wiki/2006%EB%85%84_6%EC%9B%94" title="2006년 6월">2006년 6월</a> <a class="mw-redirect" href="/wiki/%ED%8C%8C%EB%A5%B4%EB%A7%88_FC" title="파르마 FC">파르마</a>와 3년 계약을 체결했으며, <a href="/wiki/2007%EB%85%84_7%EC%9B%94" title="2007년 7월">2007년 7월</a>에는 새롭게 <a href="/wiki/%EC%84%B8%EB%A6%AC%EC%97%90_B" title="세리에 B">세리에 B</a>로 승격된 <a href="/wiki/AC_%ED%94%BC%EC%82%AC_1909" title="AC 피사 1909">AS 피사</a>로 임대되었다. <a href="/wiki/2008%EB%85%84" title="2008년">2008년</a> 여름 파르마에 복귀하였다.</p>
<p><a href="/wiki/2009%EB%85%84_1%EC%9B%94" title="2009년 1월">2009년 1월</a>, <a class="mw-redirect" href="/wiki/AS_%EB%B0%94%EB%A6%AC" title="AS 바리">AS 바리</a>로 자리를 옮겼다.</p>
