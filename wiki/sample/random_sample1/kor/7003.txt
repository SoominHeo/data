<td class="mbox-text"><span class="mbox-text-span">이 문서의 내용은 <b><a href="/wiki/%EC%9C%84%ED%82%A4%EB%B0%B1%EA%B3%BC:%EC%B6%9C%EC%B2%98_%EB%B0%9D%ED%9E%88%EA%B8%B0" title="위키백과:출처 밝히기">출처</a>가 분명하지 않습니다.</b><span class="hide-when-compact"><br/>
지금 바로 이 <span class="plainlinks"><a class="external text" href="//ko.wikipedia.org/w/index.php?title=%EB%B9%84%ED%86%A0%EB%A6%AC%EC%98%A4_%EC%97%90%EB%A7%88%EB%88%84%EC%97%98%EB%A0%88_3%EC%84%B8&amp;action=edit">문서를 편집</a></span>하여, 참고하신 문헌이나 <a href="/wiki/%EC%9C%84%ED%82%A4%EB%B0%B1%EA%B3%BC:%EC%8B%A0%EB%A2%B0%ED%95%A0_%EC%88%98_%EC%9E%88%EB%8A%94_%EC%B6%9C%EC%B2%98" title="위키백과:신뢰할 수 있는 출처">신뢰할 수 있는 출처</a>를 각주 등으로 표기해 주세요. <a href="/wiki/%EC%9C%84%ED%82%A4%EB%B0%B1%EA%B3%BC:%ED%99%95%EC%9D%B8_%EA%B0%80%EB%8A%A5" title="위키백과:확인 가능">검증</a>되지 않은 내용은 삭제될 수도 있습니다. 내용에 대한 의견은 <a href="/wiki/%ED%86%A0%EB%A1%A0:%EB%B9%84%ED%86%A0%EB%A6%AC%EC%98%A4_%EC%97%90%EB%A7%88%EB%88%84%EC%97%98%EB%A0%88_3%EC%84%B8" title="토론:비토리오 에마누엘레 3세">토론 문서</a>에서 나누어 주세요.</span> <small>(2013년 6월)</small></span></td>

<p><b>비토리오 에마누엘레 3세</b> (<span style="font-size: smaller;"><a href="/wiki/%EC%9D%B4%ED%83%88%EB%A6%AC%EC%95%84%EC%96%B4" title="이탈리아어">이탈리아어</a>:</span> <span lang="it" xml:lang="it">Vittorio Emanuele III</span>, <a href="/wiki/1869%EB%85%84" title="1869년">1869년</a> <a href="/wiki/11%EC%9B%94_11%EC%9D%BC" title="11월 11일">11월 11일</a> ~ <a href="/wiki/1947%EB%85%84" title="1947년">1947년</a> <a href="/wiki/12%EC%9B%94_28%EC%9D%BC" title="12월 28일">12월 28일</a>)는 <a href="/wiki/%EC%9D%B4%ED%83%88%EB%A6%AC%EC%95%84%EC%9D%98_%EA%B5%B0%EC%A3%BC" title="이탈리아의 군주">이탈리아의 국왕</a>(재위 <a href="/wiki/1900%EB%85%84" title="1900년">1900년</a> <a href="/wiki/7%EC%9B%94_29%EC%9D%BC" title="7월 29일">7월 29일</a>~<a href="/wiki/1946%EB%85%84" title="1946년">1946년</a> <a href="/wiki/5%EC%9B%94_9%EC%9D%BC" title="5월 9일">5월 9일</a>)이다. 재위 중 <a href="/wiki/%EC%97%90%ED%8B%B0%EC%98%A4%ED%94%BC%EC%95%84%EC%9D%98_%EA%B5%B0%EC%A3%BC" title="에티오피아의 군주">에티오피아 황제</a>(1936년~41년)와 <a class="new" href="/w/index.php?title=%EC%95%8C%EB%B0%94%EB%8B%88%EC%95%84%EC%9D%98_%EA%B5%B0%EC%A3%BC&amp;action=edit&amp;redlink=1" title="알바니아의 군주 (없는 문서)">알바니아 국왕</a>(1939년~1943년)을 겸하였으나, 열강(列強)들은 이를 인정하지 않았다. 그의 재위기간 중에는 <a href="/wiki/%ED%8C%8C%EC%8B%9C%EC%A6%98" title="파시즘">파시즘</a>의 탄생과 대두, 그리고 몰락이 있었다.</p>
<h2><span class="mw-headline" id=".EC.83.9D.EC.95.A0">생애</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%B9%84%ED%86%A0%EB%A6%AC%EC%98%A4_%EC%97%90%EB%A7%88%EB%88%84%EC%97%98%EB%A0%88_3%EC%84%B8&amp;action=edit&amp;section=1" title="부분 편집: 생애">원본 편집</a><span class="mw-editsection-bracket">]</span></span></h2>

<p><a class="new" href="/w/index.php?title=%EC%9D%B4%ED%83%88%EB%A6%AC%EC%95%84%EC%9D%98_%EA%B5%AD%EC%99%95&amp;action=edit&amp;redlink=1" title="이탈리아의 국왕 (없는 문서)">이탈리아의 국왕</a> <a href="/wiki/%EC%9B%80%EB%B2%A0%EB%A5%B4%ED%86%A0_1%EC%84%B8" title="움베르토 1세">움베르토 1세</a>와 그 왕비 마르게리타 공주 사이에서 태어났다.</p>
<p>신장 198cm였던 오촌 조카 3대 아오스타공(公) 아메데오와는 대조적으로, 키 153cm의 단신(短身)이었던 그는 태어날 때부터 이탈리아의 왕태자에게 주어지는 칭호인 <b>나폴리 공</b>(公)이었다. <a href="/wiki/1896%EB%85%84" title="1896년">1896년</a> <a href="/wiki/10%EC%9B%94_24%EC%9D%BC" title="10월 24일">10월 24일</a>, <a href="/wiki/%EB%AA%AC%ED%85%8C%EB%84%A4%EA%B7%B8%EB%A1%9C" title="몬테네그로">몬테네그로</a>의 <a href="/wiki/%EB%AA%AC%ED%85%8C%EB%84%A4%EA%B7%B8%EB%A1%9C%EC%9D%98_%EC%97%98%EB%A0%88%EB%82%98" title="몬테네그로의 엘레나">엘레나</a> 공녀와 결혼하였으며, <a href="/wiki/1900%EB%85%84" title="1900년">1900년</a> 아버지 <a href="/wiki/%EC%9B%80%EB%B2%A0%EB%A5%B4%ED%86%A0_1%EC%84%B8" title="움베르토 1세">움베르토 1세</a>가 무정부주의자에게 암살된 직후 즉위했다.</p>
<p>당시의 이탈리아는 <a class="mw-redirect" href="/wiki/%EC%9E%85%ED%97%8C%EA%B5%B0%EC%A3%BC%EA%B5%AD" title="입헌군주국">입헌군주국</a>이었지만, <a class="new" href="/w/index.php?title=%EC%B9%B4%EB%A5%BC%EB%A1%9C_%EC%95%8C%EB%B2%A0%EB%A5%B4%ED%86%A0&amp;action=edit&amp;redlink=1" title="카를로 알베르토 (없는 문서)">카를로 알베르토</a> 국왕 때의 헌법에 따라, 국왕은 총리 임명권을 포함한 권한을 가지고 있었다. 수줍어하고 소극적이었던 성격의 왕은 1900~1922년 사이에 열 번도 넘게 정치적 위기가 찾아왔던 이탈리아 정치에 스트레스를 받고 있었다.</p>
<p>이탈리아는 <a href="/wiki/%EC%A0%9C1%EC%B0%A8_%EC%84%B8%EA%B3%84_%EB%8C%80%EC%A0%84" title="제1차 세계 대전">제1차 세계 대전</a> 당시 연합국으로 참전하여 승전국이 되었으나, 종전 후 사회 양극화로 몸살을 앓았다. 이 와중인 <a href="/wiki/1922%EB%85%84" title="1922년">1922년</a>에 <a href="/wiki/%EB%B2%A0%EB%8B%88%ED%86%A0_%EB%AC%B4%EC%86%94%EB%A6%AC%EB%8B%88" title="베니토 무솔리니">베니토 무솔리니</a>가 '검은 셔츠단'을 이끌고 쿠데타를 일으켰으며, 그는 무솔리니를 총리로 임명하였다. 무솔리니는 21년간 총리로 재임한다.</p>
<p>이탈리아는 <a href="/wiki/%EC%A0%9C2%EC%B0%A8_%EC%84%B8%EA%B3%84_%EB%8C%80%EC%A0%84" title="제2차 세계 대전">제2차 세계 대전</a> 때 추축국으로 참전하였다. 그러나 <a href="/wiki/1943%EB%85%84" title="1943년">1943년</a>, 연합국 군대가 이탈리아에 상륙하자 이탈리아는 곧 항복하였다. 이 때 비토리오 에마누엘레 3세는 <a class="new" href="/w/index.php?title=%ED%8C%8C%EC%8B%9C%EC%8A%A4%ED%8A%B8_%ED%8F%89%EC%9D%98%ED%9A%8C&amp;action=edit&amp;redlink=1" title="파시스트 평의회 (없는 문서)">파시스트 평의회</a>에서 <a href="/wiki/%EB%B2%A0%EB%8B%88%ED%86%A0_%EB%AC%B4%EC%86%94%EB%A6%AC%EB%8B%88" title="베니토 무솔리니">베니토 무솔리니</a>의 실각을 결의한 후, 그를 찾아온 무솔리니에게 평의회의 결정을 따를 것임을 통고하고 즉시 경찰을 불러 그를 감금시켰다. 결국 무솔리니는 총리에서 물러나고, <a href="/wiki/%ED%94%BC%EC%97%90%ED%8A%B8%EB%A1%9C_%EB%B0%94%EB%8F%8C%EB%A6%AC%EC%98%A4" title="피에트로 바돌리오">피에트로 바돌리오</a> 내각이 들어섰다.</p>
<p>그러나 무솔리니는 <a class="new" href="/w/index.php?title=%EB%82%98%EC%B9%98_%EB%8F%85%EC%9D%BC%EA%B5%B0&amp;action=edit&amp;redlink=1" title="나치 독일군 (없는 문서)">나치 독일군</a>에 의해 구출되었고, 북부 이탈리아에 <a class="mw-redirect" href="/wiki/%EC%9D%B4%ED%83%88%EB%A6%AC%EC%95%84_%EC%82%AC%ED%9A%8C_%EA%B3%B5%ED%99%94%EA%B5%AD" title="이탈리아 사회 공화국">이탈리아 사회 공화국</a>을 세운다. 비토리오 에마누엘레 역시 무솔리니를 내각으로 임명했다는 비판에서 벗어날 수 없었으며, 그 자신 또한 그것을 느끼고 있었다. 결국 그는 <a href="/wiki/1944%EB%85%84" title="1944년">1944년</a> <a href="/wiki/4%EC%9B%94" title="4월">4월</a>이 되면 대부분의 권한을 <a href="/wiki/%EC%9B%80%EB%B2%A0%EB%A5%B4%ED%86%A0_2%EC%84%B8" title="움베르토 2세">움베르토 왕세자</a>에게 넘겨주었다.</p>
<p><a href="/wiki/1945%EB%85%84" title="1945년">1945년</a> 패색이 짙자 무솔리니는 도주하였고 이때 무솔리니를 완전히 정계에서 축출할 수 있었다. 무솔리니와 파시스트당의 당원들은 1945년 4월 공산당 파르티잔 부대원에 의해 이탈리아 국경에서 사로잡혀 처형되었다. 여론은 군주제 유지냐, 공화국 수립이냐를 둘러싼 <a href="/wiki/1946%EB%85%84_%EC%9D%B4%ED%83%88%EB%A6%AC%EC%95%84_%EA%B0%9C%ED%97%8C_%EA%B5%AD%EB%AF%BC%ED%88%AC%ED%91%9C" title="1946년 이탈리아 개헌 국민투표">국민투표</a>를 요구했고, 군주제 유지에 도움이 되고자 <a href="/wiki/1946%EB%85%84" title="1946년">1946년</a> <a href="/wiki/5%EC%9B%94_9%EC%9D%BC" title="5월 9일">5월 9일</a> 움베르토에게 양위하였다. 사실, 현대의 역사가들은 비토리오 에마누엘레가 1943년, 연합군이 <a href="/wiki/%EC%8B%9C%EC%B9%A0%EB%A6%AC%EC%95%84_%EC%84%AC" title="시칠리아 섬">시칠리아 섬</a>에 상륙한 직후에 퇴위했다면 움베르토의 인기가 이탈리아의 군주제를 유지시킬 수 있으리라고 보고 있다. 그러나 너무 늦었다. 최종적으로, 6월 2일의 투표 결과에서 군주제는 46%의 지지밖에 얻지 못했다. 투표가 조작되었다는 의혹에도 불구하고 <a class="mw-redirect" href="/wiki/%EC%82%AC%EB%B3%B4%EC%9D%B4_%EC%99%95%EA%B0%80" title="사보이 왕가">사보이 왕가</a> 사람들은 이탈리아를 떠나야 했으며, 그는 <a href="/wiki/%EC%9D%B4%EC%A7%91%ED%8A%B8" title="이집트">이집트</a>로 망명하여 <a href="/wiki/1947%EB%85%84" title="1947년">1947년</a> 사망했다.</p>
