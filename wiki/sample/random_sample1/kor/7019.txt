<p><b>빅 텐 콘퍼런스</b>(<span lang="en" xml:lang="en">Big Ten Conference</span>, Big 10)는 미국 중서부 지역의 <a class="mw-redirect" href="/wiki/%EC%95%84%EC%9D%B4%EC%98%A4%EC%99%80" title="아이오와">아이오와</a>(Iowa, Minnesota), <a class="mw-redirect" href="/wiki/%EB%84%A4%EB%B8%8C%EB%9E%98%EC%8A%A4%EC%B9%B4" title="네브래스카">네브래스카</a>(Nebraska) 지역으로부터 동부의 <a class="mw-redirect" href="/wiki/%ED%8E%9C%EC%8B%A4%EB%B2%A0%EC%9D%B4%EB%8B%88%EC%95%84" title="펜실베이니아">펜실베이니아</a>(Penssylvania) 지역에 이르기까지 스포츠 팀들의 명성과 더불어 학문적으로도 특화되어 명성을 날리는 최상의 수준으로 꼽히는 10개의 대학을 가리켰지만, 1990년 <a class="mw-redirect" href="/wiki/%ED%8E%9C%EC%8B%A4%EB%B2%A0%EC%9D%B4%EB%8B%88%EC%95%84_%EC%A3%BC%EB%A6%BD%EB%8C%80" title="펜실베이니아 주립대">펜실베이니아 주립대</a>, 2011년 <a href="/wiki/%EB%84%A4%EB%B8%8C%EB%9E%98%EC%8A%A4%EC%B9%B4_%EB%8C%80%ED%95%99%EA%B5%90" title="네브래스카 대학교">네브래스카 대학교</a>, 2014년 메리랜드 대학, 러커스대학이 합류하면서 현재는 총 14곳이다. <a class="mw-redirect" href="/wiki/%EB%AF%B8%EA%B5%AD_%EB%8C%80%ED%95%99_%EC%B2%B4%EC%9C%A1_%ED%98%91%ED%9A%8C" title="미국 대학 체육 협회">미국 대학 체육 협회</a>(NCAA) 와 <a href="/wiki/%EB%AF%B8%EC%8B%9D%EC%B6%95%EA%B5%AC" title="미식축구">미식축구</a>(football) 등의 스포츠로 아주 유명하다.</p>

<h3><span class="mw-headline" id="1896.EB.85.84">1896년</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%B9%85_%ED%85%90_%EC%BD%98%ED%8D%BC%EB%9F%B0%EC%8A%A4&amp;action=edit&amp;section=3" title="부분 편집: 1896년">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p><a href="/wiki/%EC%8B%9C%EC%B9%B4%EA%B3%A0_%EB%8C%80%ED%95%99%EA%B5%90" title="시카고 대학교">시카고</a>· <a class="mw-disambig" href="/wiki/%EC%9D%BC%EB%A6%AC%EB%85%B8%EC%9D%B4_%EB%8C%80%ED%95%99%EA%B5%90" title="일리노이 대학교">일리노이</a>· <a href="/wiki/%EB%AF%B8%EC%8B%9C%EA%B0%84_%EB%8C%80%ED%95%99%EA%B5%90" title="미시간 대학교">미시간</a>· <a href="/wiki/%EB%AF%B8%EB%84%A4%EC%86%8C%ED%83%80_%EB%8C%80%ED%95%99%EA%B5%90" title="미네소타 대학교">미네소타</a>· <a href="/wiki/%EB%85%B8%EC%8A%A4%EC%9B%A8%EC%8A%A4%ED%84%B4_%EB%8C%80%ED%95%99%EA%B5%90" title="노스웨스턴 대학교">노스웨스턴</a>· <a href="/wiki/%ED%8D%BC%EB%93%80_%EB%8C%80%ED%95%99%EA%B5%90" title="퍼듀 대학교">퍼듀</a>· <a href="/wiki/%EC%9C%84%EC%8A%A4%EC%BD%98%EC%8B%A0_%EB%8C%80%ED%95%99%EA%B5%90" title="위스콘신 대학교">위스콘신 대학교</a> 등으로 창시되었으며 1899년<a href="/wiki/%EC%9D%B8%EB%94%94%EC%95%A0%EB%82%98_%EB%8C%80%ED%95%99%EA%B5%90" title="인디애나 대학교">인디애나 대학교</a>와 <a href="/wiki/%EC%95%84%EC%9D%B4%EC%98%A4%EC%99%80_%EB%8C%80%ED%95%99%EA%B5%90" title="아이오와 대학교">아이오와 대학교</a>, 그리고 1912년 <a href="/wiki/%EC%98%A4%ED%95%98%EC%9D%B4%EC%98%A4_%EC%A3%BC%EB%A6%BD_%EB%8C%80%ED%95%99%EA%B5%90" title="오하이오 주립 대학교">오하이오 주립 대학교</a>가 가입하게 된다.</p>
<h3><span class="mw-headline" id="1939.EB.85.84">1939년</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%B9%85_%ED%85%90_%EC%BD%98%ED%8D%BC%EB%9F%B0%EC%8A%A4&amp;action=edit&amp;section=4" title="부분 편집: 1939년">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>축구 팀을 해체한 <a href="/wiki/%EC%8B%9C%EC%B9%B4%EA%B3%A0_%EB%8C%80%ED%95%99%EA%B5%90" title="시카고 대학교">시카고 대학교</a>는 연맹에서 공식적으로 탈퇴를 선언하며 한 때 <a class="new" href="/w/index.php?title=%EB%B9%85_%EB%82%98%EC%9D%B8&amp;action=edit&amp;redlink=1" title="빅 나인 (없는 문서)">빅 나인</a>(Big Nine)으로 유명해지기 시작한다.</p>
<h3><span class="mw-headline" id="1949.EB.85.84">1949년</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%B9%85_%ED%85%90_%EC%BD%98%ED%8D%BC%EB%9F%B0%EC%8A%A4&amp;action=edit&amp;section=5" title="부분 편집: 1949년">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>1949년 <a href="/wiki/%EB%AF%B8%EC%8B%9C%EA%B0%84_%EC%A3%BC%EB%A6%BD_%EB%8C%80%ED%95%99%EA%B5%90" title="미시간 주립 대학교">미시간 주립 대학교</a>가 가입되면서 다시 10개 대학을 회원으로 갖게 되며 빅 텐 경기연맹은 전통적으로 미국에서 가장 강한 미식축구 경기연맹의 하나로, 1947년부터는 대개 그해의 우승 팀을 역사가 가장 오래된 시즌 후 초청대회인 <a href="/wiki/%EB%A1%9C%EC%A6%88_%EB%B3%BC" title="로즈 볼">로즈 볼</a>에 대표로 파견했다. 이 경기연맹에 가입한 대학들간의 경기는 나중에 다른 종목으로도 확대되었다.</p>
<h2><span class="mw-headline" id=".EB.AF.B8.EC.8B.9D_.EC.B6.95.EA.B5.AC">미식 축구</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%B9%85_%ED%85%90_%EC%BD%98%ED%8D%BC%EB%9F%B0%EC%8A%A4&amp;action=edit&amp;section=6" title="부분 편집: 미식 축구">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>미국대학 미식축구는 엔에프엘 풋볼(<a class="mw-redirect" href="/wiki/NFL" title="NFL">NFL</a>)보다는 다소 수준이 떨어지지만 프로팀과는 다른 <a class="new" href="/w/index.php?title=%EB%8C%80%ED%95%99%EB%AF%B8%EC%8B%9D%EC%B6%95%EA%B5%AC&amp;action=edit&amp;redlink=1" title="대학미식축구 (없는 문서)">대학미식축구</a> 만의 재미가 있어서 굉장히 많은 팬들을 가지고 있다. 물론 학교에 따라서 팀 실력은 차이를 보이지만 비슷한 실력을 가진 학교끼리의 경기가 있는 날이면 대학도시들은 축제 분위기가 된다. 보통 대학팀별 등급은 <a class="mw-redirect" href="/wiki/NCAA" title="NCAA">NCAA</a> 디비전 1-A, 1-AA, 11, NAIA 로 구분된다. 이 가운데 <a class="new" href="/w/index.php?title=%EB%B9%85_%ED%85%90_%EB%A6%AC%EA%B7%B8&amp;action=edit&amp;redlink=1" title="빅 텐 리그 (없는 문서)">빅 텐 리그</a>는 디비전 1-A리그에 포함되며 여기서의 <a class="mw-disambig" href="/wiki/%EC%BD%98%ED%8D%BC%EB%9F%B0%EC%8A%A4" title="콘퍼런스">콘퍼런스</a>는 같은 수준의 대학이 지역별로 조직되어 있다. 특히 디비전 1-A 는 규모가 가장 큰 112개 대학이 속한다.</p>
<h3><span class="mw-headline" id="BCS_.EB.82.B4.EC.85.94.EB.84.90_.EC.B1.94.ED.94.BC.EC.96.B8.EC.8B.AD_.EA.B2.8C.EC.9E.84">BCS 내셔널 챔피언십 게임</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%B9%85_%ED%85%90_%EC%BD%98%ED%8D%BC%EB%9F%B0%EC%8A%A4&amp;action=edit&amp;section=7" title="부분 편집: BCS 내셔널 챔피언십 게임">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>전국 챔프전 의 공식 명칭은 <a class="new" href="/w/index.php?title=BCS_%EB%82%B4%EC%85%94%EB%84%90_%EC%B1%94%ED%94%BC%EC%96%B8%EC%8B%AD_%EA%B2%8C%EC%9E%84&amp;action=edit&amp;redlink=1" title="BCS 내셔널 챔피언십 게임 (없는 문서)">BCS 내셔널 챔피언십 게임</a>이다. <a href="/wiki/%EB%A1%9C%EC%A6%88_%EB%B3%BC" title="로즈 볼">볼</a>(bowl)이라는 말은 <a class="mw-redirect" href="/wiki/%EB%8C%80%ED%95%99_%ED%92%8B%EB%B3%BC" title="대학 풋볼">대학 풋볼</a>에서만 사용되던 이름이지만 1967년 처음으로<a class="mw-redirect" href="/wiki/NFL" title="NFL">NFL</a>에서 당시 <a class="mw-redirect" href="/wiki/%EB%AF%B8%EA%B5%AD_%ED%92%8B%EB%B3%BC_%EB%A6%AC%EA%B7%B8" title="미국 풋볼 리그">AFL</a>과 양대리고 챔피언 결정전을 시작하면서 <a href="/wiki/%EC%8A%88%ED%8D%BC%EB%B3%BC" title="슈퍼볼">슈퍼볼</a>이라는 이름을 사용하게 되면서 <a class="new" href="/w/index.php?title=%EB%8C%80%ED%95%99%ED%92%8B%EB%B3%BC%EB%A6%AC%EA%B7%B8&amp;action=edit&amp;redlink=1" title="대학풋볼리그 (없는 문서)">대학풋볼리그</a>에서는 <a class="new" href="/w/index.php?title=BSC_%EB%82%B4%EC%85%94%EB%84%90_%EC%B1%94%ED%94%BC%EC%96%B8_%EA%B2%8C%EC%9E%84&amp;action=edit&amp;redlink=1" title="BSC 내셔널 챔피언 게임 (없는 문서)">BSC 내셔널 챔피언 게임</a>이라는 이름을 사용하고 있다.</p>
<h3><span class="mw-headline" id=".EC.B1.94.ED.94.BC.EC.96.B8_.EA.B2.B0.EC.A0.95.EC.A0.84">챔피언 결정전</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%B9%85_%ED%85%90_%EC%BD%98%ED%8D%BC%EB%9F%B0%EC%8A%A4&amp;action=edit&amp;section=8" title="부분 편집: 챔피언 결정전">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>전국 챔피언 결정전은 <a class="new" href="/w/index.php?title=BCS&amp;action=edit&amp;redlink=1" title="BCS (없는 문서)">BCS</a> 1, 2위 두 팀이 맞 붙게되고, <a href="/wiki/%EB%A1%9C%EC%A6%88_%EB%B3%BC" title="로즈 볼">로즈 볼</a>은 <a class="new" href="/w/index.php?title=%EB%B9%85_%ED%85%90_%EC%B1%94%ED%94%84&amp;action=edit&amp;redlink=1" title="빅 텐 챔프 (없는 문서)">빅 텐 챔프</a>와 <a class="new" href="/w/index.php?title=%ED%8C%A9%ED%85%90_%EC%B1%94%ED%94%84&amp;action=edit&amp;redlink=1" title="팩텐 챔프 (없는 문서)">팩텐 챔프</a>가 맞 붙게 된다. 나머지 3개의 보울 게임은 3개 콘퍼런스 <a class="new" href="/w/index.php?title=SEC_BIG_12_ACC&amp;action=edit&amp;redlink=1" title="SEC BIG 12 ACC (없는 문서)">SEC BIG 12 ACC</a> 우승팀들이 출전권을 자동으로 가지게 되며 보통 <a class="new" href="/w/index.php?title=%EB%82%B4%EC%85%94%EB%84%90_%EC%B1%94%ED%94%BC%EC%96%B8%EC%8B%AD&amp;action=edit&amp;redlink=1" title="내셔널 챔피언십 (없는 문서)">내셔널 챔피언십</a>에 나가게 되는 팀들은 소속된 <a class="mw-disambig" href="/wiki/%EC%BD%98%ED%8D%BC%EB%9F%B0%EC%8A%A4" title="콘퍼런스">콘퍼런스</a> 챔피언일 경우가 많아서 이 때에는 나머지 4개 <a class="new" href="/w/index.php?title=BCS&amp;action=edit&amp;redlink=1" title="BCS (없는 문서)">BCS</a> 게임의 <a class="new" href="/w/index.php?title=%EC%BD%98%ED%8D%BC%EB%9F%B0%EC%8A%A4_%EC%B1%94%ED%94%BC%EC%96%B8&amp;action=edit&amp;redlink=1" title="콘퍼런스 챔피언 (없는 문서)">콘퍼런스 챔피언</a> 자리는 그 <a class="mw-disambig" href="/wiki/%EC%BD%98%ED%8D%BC%EB%9F%B0%EC%8A%A4" title="콘퍼런스">콘퍼런스</a> 2위 팀이 진출하게 된다.</p>
<h2><span class="mw-headline" id=".EB.82.A8.EC.9E.90_.EB.86.8D.EA.B5.AC">남자 농구</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%B9%85_%ED%85%90_%EC%BD%98%ED%8D%BC%EB%9F%B0%EC%8A%A4&amp;action=edit&amp;section=9" title="부분 편집: 남자 농구">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>빅 텐 회원 대학은 1904년 이후부터 농구대회에 참여하기 시작했고 1978년 이후 매 시즌마다 <a class="new" href="/w/index.php?title=%EC%B1%94%ED%94%BC%EC%96%B8_%EA%B2%B0%EC%A0%95%EC%A0%84&amp;action=edit&amp;redlink=1" title="챔피언 결정전 (없는 문서)">챔피언 결정전</a>에 올랐다. 특히 남자 농구는 전미 최고의 실력을 자랑하며, 수많은 우승 컵을 차지했다.</p>
<p>한 해 평균 4팀 이상의 <a class="mw-redirect" href="/wiki/NCAA" title="NCAA">NCAA</a> 남자 농구 토너먼트에 진출시켜왔으며, 특히 <a href="/wiki/%EC%9D%B8%EB%94%94%EC%95%A0%EB%82%98_%EB%8C%80%ED%95%99%EA%B5%90" title="인디애나 대학교">인디애나 대학교</a> 5번, <a href="/wiki/%EB%AF%B8%EC%8B%9C%EA%B0%84_%EB%8C%80%ED%95%99%EA%B5%90" title="미시간 대학교">미시간 대학교</a> 2번, <a href="/wiki/%EC%9C%84%EC%8A%A4%EC%BD%98%EC%8B%A0_%EB%8C%80%ED%95%99%EA%B5%90" title="위스콘신 대학교">위스콘신 대학교</a>, <a href="/wiki/%EB%AF%B8%EC%8B%9C%EA%B0%84_%EC%A3%BC%EB%A6%BD_%EB%8C%80%ED%95%99%EA%B5%90" title="미시간 주립 대학교">미시간 주립 대학교</a>, <a href="/wiki/%EC%98%A4%ED%95%98%EC%9D%B4%EC%98%A4_%EC%A3%BC%EB%A6%BD_%EB%8C%80%ED%95%99%EA%B5%90" title="오하이오 주립 대학교">오하이오 주립 대학교</a> 등은 한 번씩 우승을 차지해 왔다.</p>
<p>특히 <a href="/wiki/%EC%98%A4%ED%95%98%EC%9D%B4%EC%98%A4_%EC%A3%BC%EB%A6%BD_%EB%8C%80%ED%95%99%EA%B5%90" title="오하이오 주립 대학교">오하이오 주립 대학교</a>의 경우, 1939년 최초로 <a class="new" href="/w/index.php?title=NCAA_%ED%86%A0%EB%84%88%EB%A8%BC%ED%8A%B8_%EC%B1%94%ED%94%BC%EC%98%A8_%EA%B2%B0%EC%A0%95%EC%A0%84&amp;action=edit&amp;redlink=1" title="NCAA 토너먼트 챔피온 결정전 (없는 문서)">NCAA 토너먼트 챔피온 결정전</a>에서 <a href="/wiki/%EC%98%A4%EB%A6%AC%EA%B1%B4_%EB%8C%80%ED%95%99%EA%B5%90" title="오리건 대학교">오리건 대학교</a>에 패했지만, <a class="new" href="/w/index.php?title=%EC%A7%80%EB%AF%B8_%ED%9B%8C&amp;action=edit&amp;redlink=1" title="지미 훌 (없는 문서)">지미 훌</a> 선수는 첫 번째 <a class="new" href="/w/index.php?title=NCAA_%ED%86%A0%EB%84%88%EB%A8%BC%ED%8A%B8&amp;action=edit&amp;redlink=1" title="NCAA 토너먼트 (없는 문서)">NCAA 토너먼트</a> MVP에 선정되기도 하였고, 처음 3번째 토너먼트 MVP 는 모두 빅 텐에서 배출되었다.</p>
<p>빅 텐 회원 대학 팀들은 <a class="new" href="/w/index.php?title=%ED%8F%AC%EC%8A%A4%ED%8A%B8_%EC%8B%9C%EC%A6%8C_NIT&amp;action=edit&amp;redlink=1" title="포스트 시즌 NIT (없는 문서)">포스트 시즌 NIT</a>에서 역시 뛰어난 성적을 나타내었고, 1974년부터 빅 텐 대학의 팀들이 13번이나 챔피온 결정전에 올라 8번의 챔피언을 차지했다.</p>
