<p><b>빅토르 안</b>(<span style="font-size: smaller;"><a href="/wiki/%EB%9F%AC%EC%8B%9C%EC%95%84%EC%96%B4" title="러시아어">러시아어</a>:</span> <span lang="ru" xml:lang="ru">Виктор Ан</span>, <a href="/wiki/1985%EB%85%84" title="1985년">1985년</a> <a href="/wiki/11%EC%9B%94_23%EC%9D%BC" title="11월 23일">11월 23일</a> ~ ) 또는 <b>안현수</b>(安賢洙)는 <a href="/wiki/%EB%8C%80%ED%95%9C%EB%AF%BC%EA%B5%AD" title="대한민국">대한민국</a> 출신의 <a href="/wiki/%EB%9F%AC%EC%8B%9C%EC%95%84" title="러시아">러시아</a> <a href="/wiki/%EC%87%BC%ED%8A%B8%ED%8A%B8%EB%9E%99" title="쇼트트랙">쇼트트랙</a> 선수이다. 대한민국 국적일 당시 <a href="/wiki/2006%EB%85%84_%EB%8F%99%EA%B3%84_%EC%98%AC%EB%A6%BC%ED%94%BD" title="2006년 동계 올림픽">2006년 동계 올림픽</a>에서 대한민국 쇼트트랙 국가대표 선수로 출전하여 <a href="/wiki/%EA%B8%88%EB%A9%94%EB%8B%AC" title="금메달">금메달</a> 3개, <a href="/wiki/%EB%8F%99%EB%A9%94%EB%8B%AC" title="동메달">동메달</a> 1개를 획득하고, 2003년부터 2007년까지 <a href="/wiki/%EC%84%B8%EA%B3%84_%EC%87%BC%ED%8A%B8%ED%8A%B8%EB%9E%99_%EC%84%A0%EC%88%98%EA%B6%8C_%EB%8C%80%ED%9A%8C" title="세계 쇼트트랙 선수권 대회">세계 쇼트트랙 선수권 대회</a>를 5연속 제패하였으며, <a href="/wiki/2003%EB%85%84" title="2003년">2003년</a> 쇼트트랙 남자 1500m와 3000m에서 세계신기록을 수립하였다.(이 기록들은 모두 2011년 대한민국의 쇼트트랙 선수 <a href="/wiki/%EB%85%B8%EC%A7%84%EA%B7%9C" title="노진규">노진규</a>로부터 깨지게 된다.) 2011년 대한민국 국적을 포기하고 <a href="/wiki/%EB%9F%AC%EC%8B%9C%EC%95%84" title="러시아">러시아</a>로 귀화하여<sup class="reference" id="cite_ref-1"><a href="#cite_note-1">[1]</a></sup><sup class="reference" id="cite_ref-2"><a href="#cite_note-2">[2]</a></sup><sup class="reference" id="cite_ref-3"><a href="#cite_note-3">[3]</a></sup> 러시아 국가대표로 활동 중이다. <a href="/wiki/2014%EB%85%84_%EB%8F%99%EA%B3%84_%EC%98%AC%EB%A6%BC%ED%94%BD" title="2014년 동계 올림픽">2014년 동계 올림픽</a>에서 러시아 국가대표 선수로 출전하여 금메달 3개, 동메달 1개를 획득하여 토리노 대회 이후 다시 한 번 3관왕에 오른데 이어 2014년 세계 쇼트트랙 선수권 대회에서 우승을 하게 되며 총 여섯 번의 우승을 차지했다. 따라서 안현수는 현재 올림픽에서만 8개의 메달을 따 쇼트트랙에서는 미국의 <a class="mw-redirect" href="/wiki/%EC%95%84%ED%8F%B4%EB%A1%9C_%EC%95%88%ED%86%A4_%EC%98%A4%EB%85%B8" title="아폴로 안톤 오노">아폴로 안톤 오노</a>와 메달 수에서 타이 기록을 세웠다.</p>

<h2><span class="mw-headline" id=".EC.83.9D.EC.95.A0">생애</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%B9%85%ED%86%A0%EB%A5%B4_%EC%95%88&amp;action=edit&amp;section=2" title="부분 편집: 생애">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<h3><span class="mw-headline" id=".EC.B4.88.EA.B8.B0">초기</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%B9%85%ED%86%A0%EB%A5%B4_%EC%95%88&amp;action=edit&amp;section=3" title="부분 편집: 초기">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p><a href="/wiki/1985%EB%85%84" title="1985년">1985년</a> <a href="/wiki/11%EC%9B%94_23%EC%9D%BC" title="11월 23일">11월 23일</a> <a href="/wiki/%EB%8C%80%ED%95%9C%EB%AF%BC%EA%B5%AD" title="대한민국">대한민국</a> <a href="/wiki/%EC%84%9C%EC%9A%B8%ED%8A%B9%EB%B3%84%EC%8B%9C" title="서울특별시">서울</a>에서 태어났다. 명지초등학교 2학년 때 스케이트를 시작해 <a href="/wiki/1996%EB%85%84" title="1996년">1996년</a> 전국 남녀 학생 종별 대회에서 종합 우승하였고, 명지중학교 시절 <a class="mw-redirect" href="/wiki/%EC%A0%84%EA%B5%AD%EB%8F%99%EA%B3%84%EC%B2%B4%EC%9C%A1%EB%8C%80%ED%9A%8C" title="전국동계체육대회">동계체전</a> 3연패를 하며 주목을 받기 시작했다.</p>
<p><a href="/wiki/%EC%8B%A0%EB%AA%A9%EA%B3%A0%EB%93%B1%ED%95%99%EA%B5%90" title="신목고등학교">신목고등학교</a> 진학 후 그는 전국 고등부 1위를 차지했다. 2002년 1월 세계 주니어 선수권 대회에서 1000m와 1500m에서 1위를 기록하는 등 종합 우승을 하였으며, <a class="mw-redirect" href="/wiki/%EB%8C%80%ED%95%9C%EB%AF%BC%EA%B5%AD_%EC%87%BC%ED%8A%B8%ED%8A%B8%EB%9E%99_%EC%8A%A4%ED%94%BC%EB%93%9C_%EC%8A%A4%EC%BC%80%EC%9D%B4%ED%8C%85_%EA%B5%AD%EA%B0%80%EB%8C%80%ED%91%9C%ED%8C%80" title="대한민국 쇼트트랙 스피드 스케이팅 국가대표팀">쇼트트랙 국가대표</a>로 전격적으로 선발되어 <a href="/wiki/2002%EB%85%84_%EB%8F%99%EA%B3%84_%EC%98%AC%EB%A6%BC%ED%94%BD" title="2002년 동계 올림픽">2002년 동계 올림픽</a>에 참가했다. 당시 만 16세에 불과했던 그는 처음 출전한 세계 규모의 국제 대회에서 1000m에 결승까지 진출하는 돌풍을 일으켰으며, 결승 마지막 바퀴까지 <a href="/wiki/%EC%95%84%ED%8F%B4%EB%A1%9C_%EC%95%A4%ED%84%B4_%EC%98%A4%EB%85%B8" title="아폴로 앤턴 오노">아폴로 앤턴 오노</a>, <a href="/wiki/%EB%A6%AC%EC%9E%90%EC%A5%94" title="리자쥔">리자쥔</a>, 마티외 튀르코트 등의 유명 선수들과 선두를 다투었으나, 이들과 엉겨 넘어지면서 4위로 결승선을 통과하여 메달을 놓쳤다. 곧이어 벌어진 세계 선수권 대회에서 <a href="/wiki/%EA%B9%80%EB%8F%99%EC%84%B1_(1980%EB%85%84)" title="김동성 (1980년)">김동성</a>의 뒤를 잇는 선수로 주목받았으며, 5000m 계주에서 금메달을 땄다.</p>
<h3><span class="mw-headline" id=".EC.A0.84.EC.84.B1.EA.B8.B0">전성기</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%B9%85%ED%86%A0%EB%A5%B4_%EC%95%88&amp;action=edit&amp;section=4" title="부분 편집: 전성기">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>2002년 동계 올림픽 대회 다음 시즌인 2002년 ~ 2003년, 김동성이 부상으로 대회에 빠지면서 그는 대한민국과 세계의 각종 대회를 석권하였다. 2003년 <a href="/wiki/%ED%8F%B4%EB%9E%80%EB%93%9C" title="폴란드">폴란드</a> <a href="/wiki/%EB%B0%94%EB%A5%B4%EC%83%A4%EB%B0%94" title="바르샤바">바르샤바</a>에서 열린 세계 선수권 대회에서 종합 우승하였으며, 이후 2007년까지 남자 선수로는 처음으로 5년 연속으로 세계 선수권을 제패하였다. 2006년 <a href="/wiki/%EC%9D%B4%ED%83%88%EB%A6%AC%EC%95%84" title="이탈리아">이탈리아</a> <a href="/wiki/%ED%94%BC%EC%97%90%EB%AA%AC%ED%85%8C_%EC%A3%BC" title="피에몬테 주">피에몬테 주</a> <a href="/wiki/%ED%86%A0%EB%A6%AC%EB%85%B8" title="토리노">토리노</a>에서 열린 <a href="/wiki/2006%EB%85%84_%EB%8F%99%EA%B3%84_%EC%98%AC%EB%A6%BC%ED%94%BD" title="2006년 동계 올림픽">동계 올림픽</a> 대회에서 1000m·1500m 개인과 5000m 계주에서 금메달, 500m에서 동메달을 획득, 이 대회 쇼트트랙 남자부 전종목에서 메달을 땄다. 또한 <a href="/wiki/%EB%8C%80%ED%95%9C%EB%AF%BC%EA%B5%AD" title="대한민국">대한민국</a>의 스포츠 선수로는 올림픽 대회에서 한 번에 가장 많은 메달을 딴 선수로 기록되었다.</p>
<p>한편, <a href="/wiki/%ED%95%9C%EA%B5%AD%EC%B2%B4%EC%9C%A1%EB%8C%80%ED%95%99%EA%B5%90" title="한국체육대학교">한국체육대학교</a> 소속이었던 안현수는 한체대와 비한체대 소속 간의 파벌 논란에 연루되어 곤란을 겪었다.<sup class="reference" id="cite_ref-4"><a href="#cite_note-4">[4]</a></sup> 이러한 심정을 현재 빙상연맹 이사로 있는 이진성의 미니홈피에 토로하기도 했다. 그 후 2007년 <a href="/wiki/%EC%84%B8%EA%B3%84_%EC%87%BC%ED%8A%B8%ED%8A%B8%EB%9E%99_%EC%84%A0%EC%88%98%EA%B6%8C_%EB%8C%80%ED%9A%8C" title="세계 쇼트트랙 선수권 대회">세계 쇼트트랙 선수권 대회</a>을 제패하였다.</p>
<h3><span class="mw-headline" id=".EB.B6.80.EC.83.81.EA.B3.BC_.EA.B5.AD.EA.B0.80.EB.8C.80.ED.91.9C_.ED.83.88.EB.9D.BD">부상과 국가대표 탈락</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%B9%85%ED%86%A0%EB%A5%B4_%EC%95%88&amp;action=edit&amp;section=5" title="부분 편집: 부상과 국가대표 탈락">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>2007년부터 2008년 시즌 초반에도 안현수는 월드컵 대회에서 여러 차례 우승하는 등 계속하여 좋은 성적을 냈으며, 2007년 말, 졸업을 앞두고 쇼트트랙 선수로는 역대 최고 계약금을 받아 <a href="/wiki/%EC%84%B1%EB%82%A8%EC%8B%9C" title="성남시">성남시청</a>에 입단했다.<sup class="reference" id="cite_ref-5"><a href="#cite_note-5">[5]</a></sup> 그러나 2008년 1월 16일, 훈련 도중 심한 부상을 당해<sup class="reference" id="cite_ref-6"><a href="#cite_note-6">[6]</a></sup> 남은 시즌 대회에 출전하지 못했으며, 치료와 재활에 전념했다. 부상에서 어느 정도 회복되어 2008년 말부터 훈련을 재개하였다. <a href="/wiki/%EB%8C%80%ED%95%9C%EB%B9%99%EC%83%81%EA%B2%BD%EA%B8%B0%EC%97%B0%EB%A7%B9" title="대한빙상경기연맹">대한빙상경기연맹</a>은 2009년 4월부터 2010년 시즌 국가대표 선발전을 치렀으나, 그는 종합 순위 9위로 대표팀에 선발되지 못하여 <a href="/wiki/2010%EB%85%84_%EB%8F%99%EA%B3%84_%EC%98%AC%EB%A6%BC%ED%94%BD" title="2010년 동계 올림픽">2010년 밴쿠버 동계 올림픽</a>에 참가할 수 없게 되었다.<sup class="reference" id="cite_ref-7"><a href="#cite_note-7">[7]</a></sup> 빙상연맹이 2009년 ~ 2010년 시즌 개막을 많이 남긴 시점에서 대표선발전을 치른 것에 대해 빙상연맹과의 관계가 좋지 못하고 파벌 논란의 중심에 서 있는 그를 견제하기 위한 것이었다는 논란이 있었다. 동계올림픽에 참가하지 못한 그는, 올림픽 기간 중에 열린 2010년 전국동계체육대회에 출전, 1500m, 3000m 개인과 3000m 계주 부문에서 우승, 3관왕을 차지하였다.</p>
<h3><span class="mw-headline" id=".EB.8C.80.ED.95.9C.EB.B9.99.EC.83.81.EA.B2.BD.EA.B8.B0.EC.97.B0.EB.A7.B9.EA.B3.BC.EC.9D.98_.EA.B0.88.EB.93.B1">대한빙상경기연맹과의 갈등</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%B9%85%ED%86%A0%EB%A5%B4_%EC%95%88&amp;action=edit&amp;section=6" title="부분 편집: 대한빙상경기연맹과의 갈등">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p><a href="/wiki/2010%EB%85%84_%EB%8F%99%EA%B3%84_%EC%98%AC%EB%A6%BC%ED%94%BD" title="2010년 동계 올림픽">2010년 동계 올림픽</a> 후 <a href="/wiki/%EB%8C%80%ED%95%9C%EB%B9%99%EC%83%81%EA%B2%BD%EA%B8%B0%EC%97%B0%EB%A7%B9" title="대한빙상경기연맹">대한빙상경기연맹</a>의 파벌과 관련된 잡음으로 논란이 이어졌고, 부친은 <a href="/wiki/%EB%9F%AC%EC%8B%9C%EC%95%84" title="러시아">러시아</a>행을 권유하였다.<sup class="reference" id="cite_ref-8"><a href="#cite_note-8">[8]</a></sup> 일단 안현수는 <a href="/wiki/2010%EB%85%84" title="2010년">2010년</a> 4월에 치러질 국가대표 선발전을 준비하고 있었으나, 대한빙상경기연맹은 갑자기 국가대표 선발전 연기를 발표하여 훈련 계획에 차질을 빚었다.<sup class="reference" id="cite_ref-9"><a href="#cite_note-9">[9]</a></sup><sup class="reference" id="cite_ref-10"><a href="#cite_note-10">[10]</a></sup> 안현수는 <a href="/wiki/2010%EB%85%84" title="2010년">2010년</a> 5월 기초 군사훈련을 이수하기로 되어 있었기 때문에 선발전 연기 결정이 안현수를 견제하기 위한 것이었다는 의혹이 제기되었다. 또한 국가대표 선발전을 쇼트트랙에서는 드문 타임레이스 방식으로 치르기로 하여 경기방식도 그에게 불리하게 바꾼 것이라는 의혹도 제기되었다. 결국, 2010년 9월 타임레이스 방식으로 치러진 국가대표 선발전에서 탈락하여 <a href="/wiki/2010%EB%85%84" title="2010년">2010년</a>부터 <a href="/wiki/2011%EB%85%84" title="2011년">2011년</a>까지 시즌 국가대표로 선발되지 못했다. 이후 대한빙상경기연맹은 <a href="/wiki/2011%EB%85%84" title="2011년">2011년</a>부터 <a href="/wiki/2012%EB%85%84" title="2012년">2012년</a>까지 시즌 국가대표 선발전은 다시 4월에 치를 것이라고 하여, 그는 이에 대한 대비를 하였다. 그러던 중 소속팀인 <a href="/wiki/%EC%84%B1%EB%82%A8%EC%8B%9C" title="성남시">성남시</a>는 <a href="/wiki/2010%EB%85%84" title="2010년">2010년</a> <a href="/wiki/7%EC%9B%94_12%EC%9D%BC" title="7월 12일">7월 12일</a> 판교특별회계에서 악화된 지방 재정으로 인해 <a class="mw-redirect" href="/wiki/%EC%A7%80%EA%B8%89_%EC%9C%A0%EC%98%88" title="지급 유예">모라토리엄</a>을 선언하였고, 그것의 여파로 <a href="/wiki/2010%EB%85%84" title="2010년">2010년</a> <a href="/wiki/12%EC%9B%94_30%EC%9D%BC" title="12월 30일">12월 30일</a> 부로 소속 15개 체육팀 중 하키, 펜싱, 육상 3종목을 제외한 모든 팀을 해체하면서 빙상팀을 해체하기로 결정하였다. 안현수는 새로 옮길 소속팀을 찾지 못한 채 소속팀 없이 훈련하는 어려운 상황을 맞게 되었다.<sup class="reference" id="cite_ref-11"><a href="#cite_note-11">[11]</a></sup> 결국 <a href="/wiki/2011%EB%85%84" title="2011년">2011년</a> 4월에 열린 국가대표 선발전에서는 5위를 하여, 4위까지 주어진 국가대표 자격을 얻지 못했다.<sup class="reference" id="cite_ref-12"><a href="#cite_note-12">[12]</a></sup> 그리고 안현수는 <a href="/wiki/%EB%9F%AC%EC%8B%9C%EC%95%84" title="러시아">러시아</a>행을 결심한다.<sup class="reference" id="cite_ref-13"><a href="#cite_note-13">[13]</a></sup></p>
<h3><span class="mw-headline" id=".EB.9F.AC.EC.8B.9C.EC.95.84_.EA.B7.80.ED.99.94">러시아 귀화</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%B9%85%ED%86%A0%EB%A5%B4_%EC%95%88&amp;action=edit&amp;section=7" title="부분 편집: 러시아 귀화">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p><a href="/wiki/%EB%8C%80%ED%95%9C%EB%B9%99%EC%83%81%EA%B2%BD%EA%B8%B0%EC%97%B0%EB%A7%B9" title="대한빙상경기연맹">대한빙상경기연맹</a>과의 갈등 및 파벌 논란, 소속 팀 해체, 국가대표 선발전 탈락 등으로 어려운 처지에 빠진 안현수는 새로운 여건에서 운동할 수 있는 <a href="/wiki/%EB%9F%AC%EC%8B%9C%EC%95%84" title="러시아">러시아</a>로 귀화하기로 결정하였다.<sup class="reference" id="cite_ref-14"><a href="#cite_note-14">[14]</a></sup> <a href="/wiki/2011%EB%85%84" title="2011년">2011년</a> <a href="/wiki/6%EC%9B%94" title="6월">6월</a> <a href="/wiki/%EB%9F%AC%EC%8B%9C%EC%95%84" title="러시아">러시아</a>로 이주하였고,<sup class="reference" id="cite_ref-15"><a href="#cite_note-15">[15]</a></sup> <a href="/wiki/%EB%9F%AC%EC%8B%9C%EC%95%84" title="러시아">러시아</a> 국적까지 취득하여 <a href="/wiki/%EB%9F%AC%EC%8B%9C%EC%95%84" title="러시아">러시아</a> 국가대표로 국제대회에 출전할 계획도 세웠다.<sup class="reference" id="cite_ref-16"><a href="#cite_note-16">[16]</a></sup> 귀화에 앞서 2011년 7월, 국민체육진흥공단에서 자신의 남은 연금을 일시불로 수령하였다.<sup class="reference" id="cite_ref-17"><a href="#cite_note-17">[17]</a></sup> <a href="/wiki/2011%EB%85%84" title="2011년">2011년</a> <a href="/wiki/12%EC%9B%94_28%EC%9D%BC" title="12월 28일">12월 28일</a> <a href="/wiki/%EB%9F%AC%EC%8B%9C%EC%95%84" title="러시아">러시아</a> <a href="/wiki/%EB%93%9C%EB%AF%B8%ED%8A%B8%EB%A6%AC_%EB%A9%94%EB%93%9C%EB%B2%A0%EB%8D%B0%ED%94%84" title="드미트리 메드베데프">드미트리 메드베데프</a> 대통령은 안현수의 <a href="/wiki/%EB%9F%AC%EC%8B%9C%EC%95%84" title="러시아">러시아</a> 국적 취득을 인정하는 대통령령에 서명하여, 그는 정식으로 <a href="/wiki/%EB%9F%AC%EC%8B%9C%EC%95%84" title="러시아">러시아</a> 국적을 취득했다.<sup class="reference" id="cite_ref-18"><a href="#cite_note-18">[18]</a></sup> 귀화와 동시에 <a href="/wiki/%EB%9F%AC%EC%8B%9C%EC%95%84" title="러시아">러시아</a>에서 명성을 떨친 <a href="/wiki/%EC%9A%B0%ED%81%AC%EB%9D%BC%EC%9D%B4%EB%82%98" title="우크라이나">우크라이나</a>-고려인 혼혈 록 가수 <a href="/wiki/%EB%B9%85%ED%86%A0%EB%A5%B4_%EC%B4%88%EC%9D%B4" title="빅토르 초이">빅토르 초이</a>처럼 유명해졌으면 좋겠다는 생각에 빅토르 안(<span lang="ru" xml:lang="ru">Виктор Ан</span>)으로 개명하였다고 한다.<sup class="reference" id="cite_ref-19"><a href="#cite_note-19">[19]</a></sup><sup class="reference" id="cite_ref-20"><a href="#cite_note-20">[20]</a></sup> <a href="/wiki/%EB%9F%AC%EC%8B%9C%EC%95%84" title="러시아">러시아</a> 국적 취득으로 <a href="/wiki/%EB%8C%80%ED%95%9C%EB%AF%BC%EA%B5%AD_%EA%B5%AD%EC%A0%81%EB%B2%95" title="대한민국 국적법">대한민국 국적법</a>에 의해 그의 <a href="/wiki/%EB%8C%80%ED%95%9C%EB%AF%BC%EA%B5%AD" title="대한민국">대한민국</a> 국적은 자동 소멸되었다.</p>

<div class="thumbinner" style="width:202px;"><a class="image" href="/wiki/%ED%8C%8C%EC%9D%BC:Vladimir_Putin_and_Viktor_Ahn_24_February_2014.jpeg"><img alt="" class="thumbimage" data-file-height="434" data-file-width="650" height="134" src="//upload.wikimedia.org/wikipedia/commons/thumb/8/8e/Vladimir_Putin_and_Viktor_Ahn_24_February_2014.jpeg/200px-Vladimir_Putin_and_Viktor_Ahn_24_February_2014.jpeg" srcset="//upload.wikimedia.org/wikipedia/commons/thumb/8/8e/Vladimir_Putin_and_Viktor_Ahn_24_February_2014.jpeg/300px-Vladimir_Putin_and_Viktor_Ahn_24_February_2014.jpeg 1.5x, //upload.wikimedia.org/wikipedia/commons/thumb/8/8e/Vladimir_Putin_and_Viktor_Ahn_24_February_2014.jpeg/400px-Vladimir_Putin_and_Viktor_Ahn_24_February_2014.jpeg 2x" width="200"/></a>

<a href="/wiki/2014%EB%85%84" title="2014년">2014년</a> <a href="/wiki/2%EC%9B%94_24%EC%9D%BC" title="2월 24일">2월 24일</a> <a href="/wiki/%EB%B8%94%EB%9D%BC%EB%94%94%EB%AF%B8%EB%A5%B4_%ED%91%B8%ED%8B%B4" title="블라디미르 푸틴">블라디미르 푸틴</a> 대통령으로부터 훈장을 받은 빅토르 안.</div>

<p><a href="/wiki/%EB%9F%AC%EC%8B%9C%EC%95%84" title="러시아">러시아</a>에 귀화한 빅토르 안은 <a href="/wiki/%EB%9F%AC%EC%8B%9C%EC%95%84" title="러시아">러시아</a> 국가대표로 선발되었으며, <a href="/wiki/2012%EB%85%84_2%EC%9B%94" title="2012년 2월">2012년 2월</a>, <a href="/wiki/%EB%AA%A8%EC%8A%A4%ED%81%AC%EB%B0%94" title="모스크바">모스크바</a>에서 열린 월드컵 대회 계주팀의 일원으로 참가하여 약 4년 만에 국제 대회에 복귀하였다. <a href="/wiki/2013%EB%85%84_%EC%84%B8%EA%B3%84_%EC%87%BC%ED%8A%B8%ED%8A%B8%EB%9E%99_%EC%84%A0%EC%88%98%EA%B6%8C_%EB%8C%80%ED%9A%8C" title="2013년 세계 쇼트트랙 선수권 대회">2013년 세계 쇼트트랙 선수권 대회</a>에 참가하여 500m와 5000m 계주에서 은메달을 획득하였다. 러시아 쇼트트랙 선수로는 처음으로 세계 선수권에서 메달을 획득한 것이다. <a href="/wiki/%EB%9F%AC%EC%8B%9C%EC%95%84" title="러시아">러시아</a> <a href="/wiki/%EC%86%8C%EC%B9%98" title="소치">소치</a>에서 열린 <a href="/wiki/2014%EB%85%84_%EB%8F%99%EA%B3%84_%EC%98%AC%EB%A6%BC%ED%94%BD" title="2014년 동계 올림픽">2014년 동계 올림픽</a> 개인 1500m에서 동메달을 획득하여 8년 만에 올림픽 메달을 땄고, 이어 1000m에서 금메달을 획득하였다.<sup class="reference" id="cite_ref-21"><a href="#cite_note-21">[21]</a></sup><sup class="reference" id="cite_ref-22"><a href="#cite_note-22">[22]</a></sup> 그리고 22일 500m 경기, 5,000m 계주 경기에서도 금메달을 따면서 러시아에 금메달 3개, 동메달 1개를 안기며 <a href="/wiki/2006%EB%85%84_%EB%8F%99%EA%B3%84_%EC%98%AC%EB%A6%BC%ED%94%BD" title="2006년 동계 올림픽">2006년 동계 올림픽</a> 이후로 다시 한번 <a href="/wiki/%EC%87%BC%ED%8A%B8%ED%8A%B8%EB%9E%99" title="쇼트트랙">쇼트트랙</a> 3관왕의 기록을 세우고 <a href="/wiki/2014%EB%85%84_%EB%8F%99%EA%B3%84_%EC%98%AC%EB%A6%BC%ED%94%BD" title="2014년 동계 올림픽">2014년 동계 올림픽</a>에서 <a href="/wiki/%EC%87%BC%ED%8A%B8%ED%8A%B8%EB%9E%99" title="쇼트트랙">쇼트트랙</a> 전 종목 메달을 석권하였다. <a href="/wiki/2014%EB%85%84" title="2014년">2014년</a> <a href="/wiki/2%EC%9B%94_24%EC%9D%BC" title="2월 24일">2월 24일</a> <a href="/wiki/%EB%B8%94%EB%9D%BC%EB%94%94%EB%AF%B8%EB%A5%B4_%ED%91%B8%ED%8B%B4" title="블라디미르 푸틴">블라디미르 푸틴</a> <a href="/wiki/%EB%9F%AC%EC%8B%9C%EC%95%84" title="러시아">러시아</a> <a href="/wiki/%EB%8C%80%ED%86%B5%EB%A0%B9" title="대통령">대통령</a>으로부터 "제4급 조국공헌 훈장"을 받았고, <a href="/wiki/%EB%AA%A8%EC%8A%A4%ED%81%AC%EB%B0%94" title="모스크바">모스크바</a>의 고급 <a href="/wiki/%EC%95%84%ED%8C%8C%ED%8A%B8" title="아파트">아파트</a>를 포상으로 받았다.<sup class="reference" id="cite_ref-23"><a href="#cite_note-23">[23]</a></sup> 이어 2014년 <a href="/wiki/%EC%84%B8%EA%B3%84_%EC%87%BC%ED%8A%B8%ED%8A%B8%EB%9E%99_%EC%84%A0%EC%88%98%EA%B6%8C_%EB%8C%80%ED%9A%8C" title="세계 쇼트트랙 선수권 대회">세계 쇼트트랙 선수권 대회</a>에서 1000m 1위, 3000m 슈퍼 파이널 3위를 기록하고 500m, 1500m, 5000m 계주에서 4위를 기록해 6번째 종합우승을 거머쥐었다. 빅토르 안이 소치에서 세운 기록은 최초의 두번째 3관왕, 올림픽 쇼트트랙 전종목 석권(<a href="/wiki/%ED%86%A0%EB%A6%AC%EB%85%B8" title="토리노">토리노</a> : 1500, 1000, 계주 금메달 / <a href="/wiki/%EC%86%8C%EC%B9%98" title="소치">소치</a> : 500, 1000, 계주 금메달) 등이 있다. 이후 <a href="/wiki/%EC%BA%90%EB%82%98%EB%8B%A4" title="캐나다">캐나다</a> <a href="/wiki/%EB%AA%AC%ED%8A%B8%EB%A6%AC%EC%98%AC" title="몬트리올">몬트리올</a>에서 열린 2014년 <a href="/wiki/%EC%84%B8%EA%B3%84_%EC%87%BC%ED%8A%B8%ED%8A%B8%EB%9E%99_%EC%84%A0%EC%88%98%EA%B6%8C_%EB%8C%80%ED%9A%8C" title="세계 쇼트트랙 선수권 대회">세계 쇼트트랙 선수권 대회</a>에서 7년 만에 종합우승을 차지하여 전이경, 양양(A)와 함께 세계선수권에서 6번 우승한 선수가 되었다.</p>
