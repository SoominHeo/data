<p><b>빅토르 스테파노비치 체르노미르딘</b> (<span style="font-size: smaller;"><a href="/wiki/%EB%9F%AC%EC%8B%9C%EC%95%84%EC%96%B4" title="러시아어">러시아어</a>:</span> <span lang="ru" xml:lang="ru">Ви́ктор Степа́нович Черномы́рдин</span>, <a href="/wiki/1938%EB%85%84" title="1938년">1938년</a> <a href="/wiki/4%EC%9B%94_9%EC%9D%BC" title="4월 9일">4월 9일</a> ~ <a href="/wiki/2010%EB%85%84" title="2010년">2010년</a> <a href="/wiki/11%EC%9B%94_3%EC%9D%BC" title="11월 3일">11월 3일</a><sup class="reference" id="cite_ref-1"><a href="#cite_note-1">[1]</a></sup>)은 <a href="/wiki/%EB%9F%AC%EC%8B%9C%EC%95%84" title="러시아">러시아</a>의 정치인이다. 체르노미르딘은 1992년부터 1998년까지 <a href="/wiki/%EB%9F%AC%EC%8B%9C%EC%95%84%EC%9D%98_%EC%B4%9D%EB%A6%AC" title="러시아의 총리">러시아의 총리</a>였다. 2001년부터 2009년까지, 그는 <a href="/wiki/%EC%9A%B0%ED%81%AC%EB%9D%BC%EC%9D%B4%EB%82%98" title="우크라이나">우크라이나</a>의 대사에 재임하였다. 그 후, 그는 대통령 보좌관으로 지명되었다.<sup class="reference" id="cite_ref-kremlin110609_2-0"><a href="#cite_note-kremlin110609-2">[2]</a></sup></p>

<h2><span class="mw-headline" id=".EC.B2.AD.EB.85.84.EC.8B.9C.EC.A0.88.EA.B3.BC_.EA.B5.90.EC.9C.A1">청년시절과 교육</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%B9%85%ED%86%A0%EB%A5%B4_%EC%B2%B4%EB%A5%B4%EB%85%B8%EB%AF%B8%EB%A5%B4%EB%94%98&amp;action=edit&amp;section=1" title="부분 편집: 청년시절과 교육">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>체르노미르딘의 아버지는 인부였다. 빅토르는 그의 다섯 아이 중 한 명이었다. 체르노미르딘은 <a href="/wiki/1957%EB%85%84" title="1957년">1957년</a>에 학교 교육을 마치고 <a href="/wiki/%EC%98%A4%EB%A5%B4%EC%8A%A4%ED%81%AC" title="오르스크">오르스크</a>에 있는 <a class="new" href="/w/index.php?title=%EC%84%9D%EC%9C%A0_%EC%A0%95%EC%A0%9C&amp;action=edit&amp;redlink=1" title="석유 정제 (없는 문서)">석유 정제소</a>에 수리공으로 취직했다. 그는 <a href="/wiki/1957%EB%85%84" title="1957년">1957년</a>부터 <a href="/wiki/1960%EB%85%84" title="1960년">1960년</a>까지 <a href="/wiki/%EC%A7%95%EB%B3%91%EC%A0%9C" title="징병제">군 복무</a> 2년을 제외하고, <a href="/wiki/1962%EB%85%84" title="1962년">1962년</a>까지 그곳에서 일했다.</p>
<p>그는 <a href="/wiki/1961%EB%85%84" title="1961년">1961년</a>에 <a class="mw-redirect" href="/wiki/%EC%86%8C%EB%B9%84%EC%97%90%ED%8A%B8_%EC%97%B0%EB%B0%A9_%EA%B3%B5%EC%82%B0%EB%8B%B9" title="소비에트 연방 공산당">소비에트 연방 공산당</a>의 당원이 되었다. 1962년에, 그는 (나중에 <a href="/wiki/%EC%82%AC%EB%A7%88%EB%9D%BC_(%EB%9F%AC%EC%8B%9C%EC%95%84)" title="사마라 (러시아)">사마라</a> 폴리테크닉 대학교로 개명한) 큐비셰프 산업 대학교에 입학할 자격을 얻었다. 그는 입학 시험을 형편없이 치렀었다. 그는 수학을 낙방하여 입학 시험을 다시 치렀다. 그는 러시아어만 B 하나를 얻었으며, 다른 분야에서는 C를 받았다. 그는 매우 낮은 경쟁률 때문에 입학할 수 있었다. <a href="/wiki/1966%EB%85%84" title="1966년">1966년</a>에 그는 대학을 졸업하였다.</p>

<p><a href="/wiki/1967%EB%85%84" title="1967년">1967년</a>과 <a href="/wiki/1973%EB%85%84" title="1973년">1973년</a> 사이에 오르스크에서 공산당을 위해 일하면서 정치적 경력을 발현시키기 시작하였다. 1973년부터 <a href="/wiki/1978%EB%85%84" title="1978년">1978년</a>까지 <a href="/wiki/%EC%98%A4%EB%A0%8C%EB%B6%80%EB%A5%B4%ED%81%AC" title="오렌부르크">오렌부르크</a>에서 <a class="mw-redirect" href="/wiki/%EC%B2%9C%EC%97%B0%EA%B0%80%EC%8A%A4" title="천연가스">천연가스</a> 정련소의 사장으로 임명되었다. 1978년까지 <a href="/wiki/1982%EB%85%84" title="1982년">1982년</a>까지 소련 공산당 중앙 위원회의 중공업 지부로 일하였다.</p>
<p>1982년부터 소련의 천연 가스 산업의 부장관으로 임명되었가가, <a href="/wiki/1985%EB%85%84" title="1985년">1985년</a>부터 <a href="/wiki/1989%EB%85%84" title="1989년">1989년</a>까지 정식 장관을 지냈다.</p>
<h3><span class="mw-headline" id=".EA.B0.80.EC.A6.88.ED.94.84.EB.A1.AC.EC.9D.98_.EC.B0.BD.EC.8B.9C.EC.9E.90">가즈프롬의 창시자</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%B9%85%ED%86%A0%EB%A5%B4_%EC%B2%B4%EB%A5%B4%EB%85%B8%EB%AF%B8%EB%A5%B4%EB%94%98&amp;action=edit&amp;section=3" title="부분 편집: 가즈프롬의 창시자">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p><a href="/wiki/1989%EB%85%84" title="1989년">1989년</a> <a href="/wiki/8%EC%9B%94" title="8월">8월</a> 체르노미르딘의 지도 아래 가스 산업부는 러시아 최초의 국가적 조직 기업 <a class="mw-redirect" href="/wiki/%EA%B0%80%EC%A6%88%ED%94%84%EB%A1%AC" title="가즈프롬">가즈프롬</a>으로 변경되었다. 체르노미르딘은 그 첫 사장이 되었다.</p>
<p><a href="/wiki/1991%EB%85%84" title="1991년">1991년</a> 소련이 붕괴되면서, 전 소련을 이루는 국가들이 독자적인 가스회사를 창설하기 시작하였다. 하지만, 가즈프롬은 러시아 영토 안에서 계속 유지되면서 2000년대에는 러시아에서 가장 큰 회사와 세계에서 가장 큰 천연가스 추출 장치가 되었다.</p>
<h2><span class="mw-headline" id=".EB.9F.AC.EC.8B.9C.EC.95.84.EC.9D.98_.EC.B4.88.EB.8C.80_.EC.B4.9D.EB.A6.AC">러시아의 초대 총리</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%B9%85%ED%86%A0%EB%A5%B4_%EC%B2%B4%EB%A5%B4%EB%85%B8%EB%AF%B8%EB%A5%B4%EB%94%98&amp;action=edit&amp;section=4" title="부분 편집: 러시아의 초대 총리">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p><a href="/wiki/1992%EB%85%84" title="1992년">1992년</a> <a href="/wiki/%EB%B3%B4%EB%A6%AC%EC%8A%A4_%EC%98%90%EC%B9%9C" title="보리스 옐친">보리스 옐친</a> 대통령에 의하여 러시아 연방의 초대 총리로 임명되었다.</p>
<p><a href="/wiki/1995%EB%85%84" title="1995년">1995년</a> <a href="/wiki/4%EC%9B%94" title="4월">4월</a>에는 러시아 <a href="/wiki/%EC%9A%B0%EB%A6%AC_%EC%A7%91_%EB%9F%AC%EC%8B%9C%EC%95%84" title="우리 집 러시아">우리 집 러시아</a>당을 창당하여 의회에서 중앙적 기력이 되는 데 목표를 세웠으나 투표의 10%밖에 얻지 못하였다.</p>
<p><a href="/wiki/6%EC%9B%94_18%EC%9D%BC" title="6월 18일">6월 18일</a>에는 <a class="new" href="/w/index.php?title=%EC%83%A4%EB%B0%80_%EB%B0%94%EC%83%A4%EC%98%88%ED%94%84&amp;action=edit&amp;redlink=1" title="샤밀 바샤예프 (없는 문서)">샤밀 바샤예프</a>가 이끄는 <a class="mw-redirect" href="/wiki/%ED%85%8C%EB%9F%AC%EB%A6%AC%EC%8A%A4%ED%8A%B8" title="테러리스트">테러리스트</a>들이 <a class="new" href="/w/index.php?title=%EB%B6%80%EB%94%94%EC%98%A8%EB%85%B8%ED%91%B8%EC%8A%A4%ED%81%AC&amp;action=edit&amp;redlink=1" title="부디온노푸스크 (없는 문서)">부디온노푸스크</a>에서 1,500명의 사람들을 인질로 데리고 있을 때, 체르노미르딘과 바사예프 간의 협상들은 1차 <a class="mw-redirect" href="/wiki/%EC%B2%B4%EC%B2%B8" title="체첸">체첸</a> <a href="/wiki/%EC%A0%84%EC%9F%81" title="전쟁">전쟁</a>의 전환이 되는 타협을 이끌었다. 인질들 교환에서 러시아 정부는 체첸에서 군사 교전을 멈추는 데 동의하였고, 여러가지 협상들을 시작하였다.</p>
<p><a href="/wiki/1996%EB%85%84" title="1996년">1996년</a> <a href="/wiki/11%EC%9B%94_6%EC%9D%BC" title="11월 6일">11월 6일</a> 옐친이 심장 수술을 하자, 23시간 동안 대통령 직무 대행으로 있었다.</p>
<p><a href="/wiki/1998%EB%85%84" title="1998년">1998년</a> <a href="/wiki/3%EC%9B%94_23%EC%9D%BC" title="3월 23일">3월 23일</a>에 총리직을 면직당하였으나, 같은 해 8월 러시아의 <a href="/wiki/%EA%B8%88%EC%9C%B5" title="금융">금융</a> 공황이 일어나자 옐친 대통령이 체르노미르딘을 다시 총리에 임명하였다. 러시아 국회는 2번이나 거부하였고, 그의 후계자로 <a href="/wiki/%EC%98%88%ED%94%84%EA%B2%8C%EB%8B%88_%ED%94%84%EB%A6%AC%EB%A7%88%EC%BD%94%ED%94%84" title="예프게니 프리마코프">예프게니 프리마코프</a>가 총리에 취임하였다.</p>

<p><a href="/wiki/1999%EB%85%84" title="1999년">1999년</a> <a href="/wiki/%EC%9C%A0%EA%B3%A0%EC%8A%AC%EB%9D%BC%EB%B9%84%EC%95%84" title="유고슬라비아">유고슬라비아</a>에 <a class="mw-redirect" href="/wiki/%EB%82%98%ED%86%A0" title="나토">나토</a>의 열풍이 몰아쳤을 때 유고슬라비아에서 러시아 특별 대표로 있었다. <a href="/wiki/%EC%8A%AC%EB%A1%9C%EB%B3%B4%EB%8B%A8_%EB%B0%80%EB%A1%9C%EC%85%B0%EB%B9%84%EC%B9%98" title="슬로보단 밀로셰비치">슬로보단 밀로셰비치</a>에게 휴전에 동의하고, <a href="/wiki/%EC%BD%94%EC%86%8C%EB%B3%B4" title="코소보">코소보</a>를 <a href="/wiki/%EC%9C%A0%EC%97%94" title="유엔">유엔</a> 통제 아래로 두자고 설득시켰다. <a href="/wiki/12%EC%9B%94" title="12월">12월</a>에는 국회 의원으로 선출되었다.</p>
<p><a href="/wiki/2001%EB%85%84" title="2001년">2001년</a> <a href="/wiki/5%EC%9B%94" title="5월">5월</a> <a href="/wiki/%EB%B8%94%EB%9D%BC%EB%94%94%EB%AF%B8%EB%A5%B4_%ED%91%B8%ED%8B%B4" title="블라디미르 푸틴">블라디미르 푸틴</a> 대통령은 체르노미르딘을 <a href="/wiki/%EC%9A%B0%ED%81%AC%EB%9D%BC%EC%9D%B4%EB%82%98" title="우크라이나">우크라이나</a> 주재 대사로 임명하였다. <a href="/wiki/2009%EB%85%84" title="2009년">2009년</a> 기자회견에서 "우크라이나의 지도력과 함께한 여러가지에 협정이 오는데 불가능하다. 만약 색다른 사람들이 온다면, 두고 보리라!"고 말할 때 러시아와 우크라이나 간의 관계를 다시 긴장시켰다.</p>
<p>2009년 <a href="/wiki/6%EC%9B%94_11%EC%9D%BC" title="6월 11일">6월 11일</a> <a href="/wiki/%EB%93%9C%EB%AF%B8%ED%8A%B8%EB%A6%AC_%EB%A9%94%EB%93%9C%EB%B2%A0%EB%8D%B0%ED%94%84" title="드미트리 메드베데프">드미트리 메드베데프</a> 대통령은 체르노미르딘을 우크라이나 주재 대사로서 면제하고 대통령 보좌관 겸 <a class="mw-redirect" href="/wiki/%EB%8F%85%EB%A6%BD_%EA%B5%AD%EA%B0%80_%EC%97%B0%ED%95%A9" title="독립 국가 연합">독립 국가 연합</a>의 회원국들과 경제적 협력을 위한 특별 대통령 대리인으로 임명하였다.</p>
