<h2><span class="mw-headline" id=".EC.97.AD.EC.82.AC">역사</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%B9%88&amp;action=edit&amp;section=1" title="부분 편집: 역사">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<div class="thumb tleft">
<div class="thumbinner" style="width:302px;"><a class="image" href="/wiki/%ED%8C%8C%EC%9D%BC:Wien-1683(1686)-Allen.jpg"><img alt="" class="thumbimage" data-file-height="2883" data-file-width="5943" height="146" src="//upload.wikimedia.org/wikipedia/commons/thumb/d/d4/Wien-1683%281686%29-Allen.jpg/300px-Wien-1683%281686%29-Allen.jpg" srcset="//upload.wikimedia.org/wikipedia/commons/thumb/d/d4/Wien-1683%281686%29-Allen.jpg/450px-Wien-1683%281686%29-Allen.jpg 1.5x, //upload.wikimedia.org/wikipedia/commons/thumb/d/d4/Wien-1683%281686%29-Allen.jpg/600px-Wien-1683%281686%29-Allen.jpg 2x" width="300"/></a>
<div class="thumbcaption">
<div class="magnify"><a class="internal" href="/wiki/%ED%8C%8C%EC%9D%BC:Wien-1683(1686)-Allen.jpg" title="실제 크기로"></a></div>
1683년 경의 빈</div>
</div>
</div>
<p>원시 시대부터 현재의 빈에 사람이 살았던 것으로 알려졌고, <a href="/wiki/%EA%B8%B0%EC%9B%90%EC%A0%84_15%EB%85%84" title="기원전 15년">기원전 15년</a>에 로마군이 경계선을 세워 “빈드보나”라고 이름을 지었다. <a href="/wiki/%EA%B2%8C%EB%A5%B4%EB%A7%8C%EC%A1%B1" title="게르만족">게르만족</a>이 거기를 차지할 때까지 로마의 지배를 받았고, <a href="/wiki/800%EB%85%84%EB%8C%80" title="800년대">800년대</a> 후반에는 <a class="mw-redirect" href="/wiki/%EB%A7%88%EC%9E%90%EB%A5%B4%EC%A1%B1" title="마자르족">마자르족</a>이 빈을 통치하였다. <a href="/wiki/%EB%8F%85%EC%9D%BC" title="독일">독일</a> 황제군이 <a href="/wiki/900%EB%85%84%EB%8C%80" title="900년대">900년대</a> 후반에 빈을 정복하였고, <a href="/wiki/1150%EB%85%84" title="1150년">1150년</a>에는 <a class="mw-redirect" href="/wiki/%EB%B0%94%EB%B2%A4%EB%B2%A0%EB%A5%B4%ED%81%AC_%EA%B0%80%EB%AC%B8" title="바벤베르크 가문">바벤베르크 가문</a>의 주요 관저가 되었다.</p>
<p><a href="/wiki/1273%EB%85%84" title="1273년">1273년</a>에는 <a class="mw-redirect" href="/wiki/%ED%95%A9%EC%8A%A4%EB%B6%80%EB%A5%B4%ED%81%AC_%EC%99%95%EA%B0%80" title="합스부르크 왕가">합스부르크 왕가</a>의 <a href="/wiki/%EB%A3%A8%EB%8F%8C%ED%94%84_1%EC%84%B8" title="루돌프 1세">루돌프 1세</a>가 <a href="/wiki/%EC%8B%A0%EC%84%B1_%EB%A1%9C%EB%A7%88_%EC%A0%9C%EA%B5%AD" title="신성 로마 제국">신성 로마 제국</a>의 황제가 되었다. 빈은 합스부르크 왕가의 자리로서 번영하였다.</p>
<p><a href="/wiki/%EA%B5%90%ED%86%B5" title="교통">교통</a>의 요지여서 <a href="/wiki/11%EC%84%B8%EA%B8%B0" title="11세기">11세기</a>에는 <a href="/wiki/%EC%8B%AD%EC%9E%90%EA%B5%B0" title="십자군">십자군</a>의 통로로서 융성하였고, <a href="/wiki/15%EC%84%B8%EA%B8%B0" title="15세기">15세기</a> 이후는 신성 로마 제국의 수도로서 발전하였다.</p>
<p>전성기 시절엔 인구 200만을 넘나드는 대도시였다.</p>
<p>근세에는 오스만 제국군이 두 차례(<a href="/wiki/%EB%B9%88_%EA%B3%B5%EB%B0%A9%EC%A0%84" title="빈 공방전">1529년</a>, <a href="/wiki/%EB%B9%88_%EC%A0%84%ED%88%AC" title="빈 전투">1683년</a>) 빈 문턱 앞까지 왔으나 모두 저지되었다.</p>
<p><a href="/wiki/%EB%82%98%ED%8F%B4%EB%A0%88%EC%98%B9_%EC%A0%84%EC%9F%81" title="나폴레옹 전쟁">나폴레옹 전쟁</a> 시기에는 <a href="/wiki/%ED%94%84%EB%9E%91%EC%8A%A4" title="프랑스">프랑스</a>군이 두 번이나 빈을 공략하였다. <a href="/wiki/%EC%A7%80%EA%B7%B8%EB%AC%B8%ED%8A%B8_%ED%94%84%EB%A1%9C%EC%9D%B4%ED%8A%B8" title="지그문트 프로이트">지그문트 프로이트</a>가 빈에서 《<a href="/wiki/%EA%BF%88%EC%9D%98_%ED%95%B4%EC%84%9D" title="꿈의 해석">꿈의 해석</a>》을 출간한 1900년경, 빈은 다소 고색창연하기는 하지만 우아한 대도시로 비쳐졌다. 도시 한가운데에는 성 슈테판 대성당이 우뚝 서 있었다. 대성당의 고딕식 첨탑은 바로크 스타일의 여러 교회를 보듬고 있었다. 법원 건물은 화려와 우울의 기묘한 결합이었다. <sup class="reference" id="cite_ref-2"><a href="#cite_note-2">[2]</a></sup> 황제는 여전히 접시 오른쪽에 은제 식기를 놓고 스페인 식으로 식사를 했다. <sup class="reference" id="cite_ref-3"><a href="#cite_note-3">[3]</a></sup></p>
<p><a href="/wiki/%EC%A0%9C1%EC%B0%A8_%EC%84%B8%EA%B3%84_%EB%8C%80%EC%A0%84" title="제1차 세계 대전">제1차 세계 대전</a> 후에는 오스트리아의 수도가 되었다.</p>
<p><a href="/wiki/1938%EB%85%84" title="1938년">1938년</a> 독일이 오스트리아를 병합하자, <a href="/wiki/%EC%A0%9C2%EC%B0%A8_%EC%84%B8%EA%B3%84_%EB%8C%80%EC%A0%84" title="제2차 세계 대전">제2차 세계 대전</a>까지 독일군이 빈에 주둔하고 있었다. 전쟁 말기 <a href="/wiki/%EC%86%8C%EB%A0%A8" title="소련">소련</a>군의 공세로 벌어진 <a class="new" href="/w/index.php?title=%EB%B9%88_%EA%B3%B5%EC%84%B8&amp;action=edit&amp;redlink=1" title="빈 공세 (없는 문서)">빈 공세</a>는 <a href="/wiki/%EB%B2%A0%EB%A5%BC%EB%A6%B0_%EA%B3%B5%EB%B0%A9%EC%A0%84" title="베를린 공방전">베를린 공방전</a> 다음으로 치열하였다. <a href="/wiki/1955%EB%85%84" title="1955년">1955년</a>까지 연합국(<a href="/wiki/%EB%AF%B8%EA%B5%AD" title="미국">미국</a>, <a href="/wiki/%EC%98%81%EA%B5%AD" title="영국">영국</a>, <a href="/wiki/%ED%94%84%EB%9E%91%EC%8A%A4" title="프랑스">프랑스</a>, <a href="/wiki/%EC%86%8C%EB%A0%A8" title="소련">소련</a>)의 점령 아래 있었다.</p>
<h2><span class="mw-headline" id=".EC.A3.BC.EB.AF.BC">주민</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%B9%88&amp;action=edit&amp;section=2" title="부분 편집: 주민">편집</a><span class="mw-editsection-bracket">]</span></span></h2>

<p>시내에는 <a class="mw-redirect" href="/wiki/%EC%84%B1_%EC%8A%88%ED%85%8C%ED%8C%90_%EB%8C%80%EC%84%B1%EB%8B%B9" title="성 슈테판 대성당">성 슈테판 대성당</a> 등 각 시대의 건축 양식을 보이는 교회와 <a href="/wiki/%EC%87%A4%EB%B8%8C%EB%A3%AC_%EA%B6%81%EC%A0%84" title="쇤브룬 궁전">쇤브룬 궁전</a>·<a href="/wiki/%EB%B9%88_%EA%B5%AD%EB%A6%BD_%EC%98%A4%ED%8E%98%EB%9D%BC_%EA%B7%B9%EC%9E%A5" title="빈 국립 오페라 극장">빈 국립 오페라 극장</a>·<a href="/wiki/%EB%B2%A8%EB%B2%A0%EB%8D%B0%EB%A0%88_%EA%B6%81%EC%A0%84" title="벨베데레 궁전">벨베데레 궁전</a>·<a href="/wiki/%ED%98%B8%ED%94%84%EB%B6%80%EB%A5%B4%ED%81%AC_%EC%99%95%EA%B6%81" title="호프부르크 왕궁">호프 부르크 궁전</a>·<a href="/wiki/%EB%AF%B8%EC%88%A0%EA%B4%80" title="미술관">미술관</a> 등 문화 시설이 많다. 이처럼 문화와 관광의 중심지인 동시에, 도나우 강변에는 오스트리아 유수의 공업지대가 펼쳐져 있다.</p>
<p>또한 음악의 도시로도 알려졌으며, <a class="mw-redirect" href="/wiki/%EB%B2%A0%ED%86%A0%EB%B2%A4" title="베토벤">베토벤</a>을 비롯한 <a class="mw-redirect" href="/wiki/%EB%AA%A8%EC%B0%A8%EB%A5%B4%ED%8A%B8" title="모차르트">모차르트</a>, <a class="mw-redirect" href="/wiki/%EC%8A%88%EB%B2%A0%EB%A5%B4%ED%8A%B8" title="슈베르트">슈베르트</a>, <a class="mw-redirect" href="/wiki/%EB%B8%8C%EB%9E%8C%EC%8A%A4" title="브람스">브람스</a>, <a class="mw-redirect" href="/wiki/%ED%95%98%EC%9D%B4%EB%93%A0" title="하이든">하이든</a> 그리고 <a class="mw-redirect" href="/wiki/%EB%A7%90%EB%9F%AC" title="말러">말러</a> 등의 음악가들이 여기서 활약을 하였다. <a class="mw-redirect" href="/wiki/%EB%B9%88_%ED%95%84%ED%95%98%EB%AA%A8%EB%8B%89_%EC%98%A4%EC%BC%80%EC%8A%A4%ED%8A%B8%EB%9D%BC" title="빈 필하모닉 오케스트라">빈 필하모닉 오케스트라</a>단과 <a href="/wiki/%EB%B9%88_%EC%86%8C%EB%85%84_%ED%95%A9%EC%B0%BD%EB%8B%A8" title="빈 소년 합창단">빈 소년 합창단</a>이 유명하다.</p>
<h2><span class="mw-headline" id=".EA.B4.80.EA.B4.91.EC.A7.80">관광지</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%B9%88&amp;action=edit&amp;section=5" title="부분 편집: 관광지">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
