<p><b>빈 대학교</b> (<span style="font-size: smaller;"><a href="/wiki/%EB%8F%85%EC%9D%BC%EC%96%B4" title="독일어">독일어</a>:</span> <span lang="de" xml:lang="de">Universität Wien</span>)는 <a href="/wiki/1365%EB%85%84" title="1365년">1365년</a>에 세워진 <a href="/wiki/%EC%98%A4%EC%8A%A4%ED%8A%B8%EB%A6%AC%EC%95%84" title="오스트리아">오스트리아</a>의 <a class="mw-redirect" href="/wiki/%EB%8C%80%ED%95%99%EA%B5%90" title="대학교">대학교</a>이다. <a href="/wiki/%EB%8F%85%EC%9D%BC%EC%96%B4" title="독일어">독일어</a>권 지역에서 가장 오래되고 가장 큰 대학교이다. 1365년 <a class="mw-redirect" href="/wiki/%ED%95%A9%EC%8A%A4%EB%B6%80%EB%A5%B4%ED%81%AC_%EC%99%95%EA%B0%80" title="합스부르크 왕가">합스부르크 왕가</a>의 <a class="new" href="/w/index.php?title=%EB%A3%A8%EB%8F%8C%ED%94%84_4%EC%84%B8&amp;action=edit&amp;redlink=1" title="루돌프 4세 (없는 문서)">루돌프 4세</a>에 의하여 설립되었다. 설립 당시에는 <a href="/wiki/%EB%B2%95%ED%95%99" title="법학">법학</a>·<a href="/wiki/%EC%9D%98%ED%95%99" title="의학">의학</a>·<a href="/wiki/%EB%AC%B8%ED%95%99" title="문학">문학</a>의 세 학부로 출발하였으나 <a href="/wiki/1384%EB%85%84" title="1384년">1384년</a> <a href="/wiki/%EC%8B%A0%ED%95%99" title="신학">신학부</a>가 부설되면서 파리에서 초청한 신학자 앙리 드 강이 <a class="mw-redirect" href="/wiki/%ED%8C%8C%EB%A6%AC_%EB%8C%80%ED%95%99" title="파리 대학">파리 대학</a>을 모델로 대학을 재조직하였다. <a href="/wiki/1520%EB%85%84" title="1520년">1520년</a>에는 독일어권 대학 중에서 가장 많은 학생을 수용하기에 이르렀으나 <a class="mw-redirect" href="/wiki/%EC%A2%85%EA%B5%90%EA%B0%9C%ED%98%81" title="종교개혁">종교개혁</a>의 영향으로 급격히 쇠퇴하여 <a href="/wiki/1623%EB%85%84" title="1623년">1623년</a>에 <a href="/wiki/%EC%98%88%EC%88%98%ED%9A%8C" title="예수회">예수회</a>에서 인수하였다.</p>
<p>18세기 <a class="mw-redirect" href="/wiki/%EC%8B%A0%EC%84%B1%EB%A1%9C%EB%A7%88%EC%A0%9C%EA%B5%AD" title="신성로마제국">신성로마제국</a>의 여왕 <a href="/wiki/%EB%A7%88%EB%A6%AC%EC%95%84_%ED%85%8C%EB%A0%88%EC%A7%80%EC%95%84" title="마리아 테레지아">마리아 테레지아</a>의 교육개혁에 의해 부활되고, <a href="/wiki/%EC%9D%98%ED%95%99" title="의학">의학</a>·<a href="/wiki/%EB%B2%95%ED%95%99" title="법학">법학</a>·<a href="/wiki/%EC%97%AD%EC%82%AC%ED%95%99" title="역사학">역사학</a> 등의 분야에서 <a class="new" href="/w/index.php?title=%EB%B9%88_%ED%95%99%ED%8C%8C&amp;action=edit&amp;redlink=1" title="빈 학파 (없는 문서)">빈 학파</a>를 형성하여 그 이름을 떨쳤으며, <a href="/wiki/1848%EB%85%84" title="1848년">1848년</a>∼<a href="/wiki/1850%EB%85%84" title="1850년">1850년</a> 문교부장관 L.튤의 적극적인 노력에 힘입어 많은 개혁을 단행하였다. 가톨릭 신학, 프로테스탄트 신학, 법학·정치학, 사회과학·경제학, 의학, 철학, 자연과학, 인문학 등의 학부가 있다.</p>
