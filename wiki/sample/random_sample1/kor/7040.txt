<p><b>빈 소년 합창단</b>(Wien 少年合唱團, <span style="font-size: smaller;"><a href="/wiki/%EB%8F%85%EC%9D%BC%EC%96%B4" title="독일어">독일어</a>:</span> <span lang="de" xml:lang="de">Wiener Sängerknaben</span>, <span lang="en" xml:lang="en">Vienna Boys' Choir</span>)은 <a href="/wiki/%EC%98%A4%EC%8A%A4%ED%8A%B8%EB%A6%AC%EC%95%84" title="오스트리아">오스트리아</a> <a href="/wiki/%EB%B9%88" title="빈">빈</a>에 기반을 둔 <a href="/wiki/%EB%B3%B4%EC%9D%B4_%EC%86%8C%ED%94%84%EB%9D%BC%EB%85%B8" title="보이 소프라노">보이 소프라노</a>와 <a class="new" href="/w/index.php?title=%EB%B3%B4%EC%9D%B4_%EC%95%8C%ED%86%A0&amp;action=edit&amp;redlink=1" title="보이 알토 (없는 문서)">보이 알토</a> (<a href="/wiki/%EC%95%8C%ED%86%A0" title="알토">알토</a>)의 <a class="mw-redirect" href="/wiki/%ED%95%A9%EC%B0%BD%EB%8B%A8" title="합창단">합창단</a>이다.</p>

<h2><span class="mw-headline" id=".EC.97.AD.EC.82.AC">역사</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%B9%88_%EC%86%8C%EB%85%84_%ED%95%A9%EC%B0%BD%EB%8B%A8&amp;action=edit&amp;section=1" title="부분 편집: 역사">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>빈 소년합창단은 1498년 합스부르크 왕가의 황제 <a href="/wiki/%EB%A7%89%EC%8B%9C%EB%B0%80%EB%A6%AC%EC%95%88_1%EC%84%B8_(%EC%8B%A0%EC%84%B1_%EB%A1%9C%EB%A7%88_%EC%A0%9C%EA%B5%AD)" title="막시밀리안 1세 (신성 로마 제국)">막시밀리안 1세</a>의 칙령에 의해 설립된 궁정소년합창단이며, 최초에는 궁정예배당에 봉사하는 합창단이었으나, 차차 일반의 콘서트에도 출연하게 되었다.<sup class="reference" id="cite_ref-.EA.B8.80.EB.A1.9C.EB.B2.8C_1_1-0"><a href="#cite_note-.EA.B8.80.EB.A1.9C.EB.B2.8C_1-1">[1]</a></sup> 20세기 초 합스부르크 왕가의 몰락 이후 민영체제로 전환되었다.</p>
<h2><span class="mw-headline" id=".EA.B5.AC.EC.84.B1">구성</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%B9%88_%EC%86%8C%EB%85%84_%ED%95%A9%EC%B0%BD%EB%8B%A8&amp;action=edit&amp;section=2" title="부분 편집: 구성">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>단원은 대체로 7세부터 13세까지의 변성기 전의 100명 남짓한 소년으로 이루어졌으며 엄격한 기숙사 생활을 하면서 학문과 음악을 공부한다. 30명 정도로 편성된 4개의 그룹이 있는데, 그 중 3개는 해외에서 활동하며 나머지 하나가 Wiener Hofmusikkapelle의 예배나 콘서트 등에 출연한다.<sup class="reference" id="cite_ref-.EA.B8.80.EB.A1.9C.EB.B2.8C_1_1-1"><a href="#cite_note-.EA.B8.80.EB.A1.9C.EB.B2.8C_1-1">[1]</a></sup></p>
<p>소년들은 개인 오디션을 통해서 뽑는다. 보통은 <a href="/wiki/%EC%98%A4%EC%8A%A4%ED%8A%B8%EB%A6%AC%EC%95%84" title="오스트리아">오스트리아</a>에서 뽑지만, <a href="/wiki/%EC%8A%A4%EC%9C%84%EC%8A%A4" title="스위스">스위스</a>, <a href="/wiki/%EB%9F%AC%EC%8B%9C%EC%95%84" title="러시아">러시아</a>,, <a href="/wiki/%ED%8E%98%EB%A3%A8" title="페루">페루</a>, <a href="/wiki/%EC%8A%AC%EB%A1%9C%EB%B0%94%ED%82%A4%EC%95%84" title="슬로바키아">슬로바키아</a>, <a href="/wiki/%EB%8F%85%EC%9D%BC" title="독일">독일</a>, <a href="/wiki/%ED%97%9D%EA%B0%80%EB%A6%AC" title="헝가리">헝가리</a>, <a href="/wiki/%EC%BA%90%EB%82%98%EB%8B%A4" title="캐나다">캐나다</a>, <a href="/wiki/%EB%AF%B8%EA%B5%AD" title="미국">미국</a>, <a href="/wiki/%EC%9D%BC%EB%B3%B8" title="일본">일본</a> 등 다른 나라에서 뽑는 경우도 있다. 2010년 6월에는 <a class="new" href="/w/index.php?title=%EC%A1%B0%EC%9C%A4%EC%83%81&amp;action=edit&amp;redlink=1" title="조윤상 (없는 문서)">조윤상</a>이 <a href="/wiki/%EB%8C%80%ED%95%9C%EB%AF%BC%EA%B5%AD" title="대한민국">대한민국</a> 국적으로는 최초로 빈 소년 합창단에 뽑혔고,<sup class="reference" id="cite_ref-.EC.84.9C.EC.9A.B8_1_2-0"><a href="#cite_note-.EC.84.9C.EC.9A.B8_1-2">[2]</a></sup> 2012년 9월부터 <a class="mw-disambig" href="/wiki/%EA%B9%80%EB%B3%B4%EB%AF%B8" title="김보미">김보미</a>가 최초의 여성 지휘자로 활동하고 있다.<sup class="reference" id="cite_ref-.EA.B0.9D.EC.84.9D_3-0"><a href="#cite_note-.EA.B0.9D.EC.84.9D-3">[3]</a></sup></p>
<p>1498년부터 지금까지 유니폼을 입는데 현재 유니폼은 <a href="/wiki/%EC%84%B8%EC%9D%BC%EB%9F%AC%EB%B3%B5" title="세일러복">세일러복</a> 형태이다. <a href="/wiki/1948%EB%85%84" title="1948년">1948년</a>부터 팔레 아우가르텐(Palais Augarten)이 주 연습장이다.</p>
<h2><span class="mw-headline" id=".EA.B3.A1.EB.AA.A9">곡목</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%B9%88_%EC%86%8C%EB%85%84_%ED%95%A9%EC%B0%BD%EB%8B%A8&amp;action=edit&amp;section=3" title="부분 편집: 곡목">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>레퍼토리는 광범하여 종교음악, 독일가곡이나 민요, 빈 왈츠뿐만 아니라 가벼운 오페라 등을 노래하기도 하는데,<sup class="reference" id="cite_ref-.EA.B8.80.EB.A1.9C.EB.B2.8C_1_1-2"><a href="#cite_note-.EA.B8.80.EB.A1.9C.EB.B2.8C_1-1">[1]</a></sup> 성가곡, 성탄절 캐롤 등을 즐겨 부른다.</p>
<h2><span class="mw-headline" id=".ED.8F.89.EA.B0.80">평가</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%B9%88_%EC%86%8C%EB%85%84_%ED%95%A9%EC%B0%BD%EB%8B%A8&amp;action=edit&amp;section=4" title="부분 편집: 평가">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>세계 최고 수준의 실력으로 평가되는 빈 소년 합창단은 <a href="/wiki/%ED%8C%8C%EB%A6%AC_%EB%82%98%EB%AC%B4%EC%8B%AD%EC%9E%90%EA%B0%80_%EC%86%8C%EB%85%84%ED%95%A9%EC%B0%BD%EB%8B%A8" title="파리 나무십자가 소년합창단">파리 나무십자가 소년합창단</a>과 함께 항상 '세계 3대 소년합창단' 안에 꼽힌다.</p>
<p>500년 가까운 역사를 가진 합창단이므로 이 합창단 출신의 대음악가도 많다. <a class="mw-redirect" href="/wiki/%ED%95%98%EC%9D%B4%EB%93%A0" title="하이든">하이든</a>, <a class="mw-redirect" href="/wiki/%EC%8A%88%EB%B2%A0%EB%A5%B4%ED%8A%B8" title="슈베르트">슈베르트</a>, <a class="mw-disambig" href="/wiki/%ED%81%B4%EB%A0%88%EB%A9%98%EC%8A%A4" title="클레멘스">클레멘스</a>, <a class="new" href="/w/index.php?title=%ED%81%AC%EB%9D%BC%EC%9A%B0%EC%8A%A4&amp;action=edit&amp;redlink=1" title="크라우스 (없는 문서)">크라우스</a> 등도 그러하다.<sup class="reference" id="cite_ref-.EA.B8.80.EB.A1.9C.EB.B2.8C_1_1-3"><a href="#cite_note-.EA.B8.80.EB.A1.9C.EB.B2.8C_1-1">[1]</a></sup></p>
<p>이 합창단의 연주를 들은 <a href="/wiki/%EA%B5%90%ED%99%A9_%EB%B9%84%EC%98%A4_11%EC%84%B8" title="교황 비오 11세">비오 11세</a>는 "마치 천사의 노래를 듣는 것 같다"고 격찬한 바 있어 그 이후로 이 합창단을 '천사의 소리'라고도 부르게 되었다. 비브라토(목소리를 떠는 창법)가 없는 맑고 깨끗한 소년 특유의 목소리 때문에 전세계 많은 팬들에게 사랑받고 있다.<sup class="reference" id="cite_ref-.EA.B8.80.EB.A1.9C.EB.B2.8C_1_1-4"><a href="#cite_note-.EA.B8.80.EB.A1.9C.EB.B2.8C_1-1">[1]</a></sup></p>
