<p><b>빈 회의</b>(<span style="font-size: smaller;"><a href="/wiki/%EB%8F%85%EC%9D%BC%EC%96%B4" title="독일어">독일어</a>:</span> <span lang="de" xml:lang="de">der Wiener Kongress</span>)는 <a href="/wiki/%EB%82%98%ED%8F%B4%EB%A0%88%EC%98%B9_%EC%A0%84%EC%9F%81" title="나폴레옹 전쟁">나폴레옹 전쟁</a>의 결과를 수습하기 위해 <a href="/wiki/%EC%98%A4%EC%8A%A4%ED%8A%B8%EB%A6%AC%EC%95%84_%EC%A0%9C%EA%B5%AD" title="오스트리아 제국">오스트리아</a>의 재상 <a href="/wiki/%ED%81%B4%EB%A0%88%EB%A9%98%EC%8A%A4_%ED%8F%B0_%EB%A9%94%ED%85%8C%EB%A5%B4%EB%8B%88%ED%9E%88" title="클레멘스 폰 메테르니히">클레멘스 폰 메테르니히</a>의 주도하에 <a href="/wiki/%EC%98%81%EA%B5%AD" title="영국">영국</a>, <a href="/wiki/%ED%94%84%EB%A1%9C%EC%9D%B4%EC%84%BC_%EC%99%95%EA%B5%AD" title="프로이센 왕국">프로이센</a>, <a href="/wiki/%EC%98%A4%EC%8A%A4%ED%8A%B8%EB%A6%AC%EC%95%84_%EC%A0%9C%EA%B5%AD" title="오스트리아 제국">오스트리아</a>, <a href="/wiki/%EB%9F%AC%EC%8B%9C%EC%95%84_%EC%A0%9C%EA%B5%AD" title="러시아 제국">러시아</a> 등이 모여 한 회의이다. 오스트리아 수도 <a href="/wiki/%EB%B9%88" title="빈">빈</a>에서 열렸기에 빈 회의라고 한다.</p>

<h2><span class="mw-headline" id=".EA.B0.9C.EC.9A.94">개요</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%B9%88_%ED%9A%8C%EC%9D%98&amp;action=edit&amp;section=1" title="부분 편집: 개요">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p><a href="/wiki/1814%EB%85%84" title="1814년">1814년</a> <a href="/wiki/9%EC%9B%94_1%EC%9D%BC" title="9월 1일">9월 1일</a>에서 <a href="/wiki/1815%EB%85%84" title="1815년">1815년</a> <a href="/wiki/6%EC%9B%94_9%EC%9D%BC" title="6월 9일">6월 9일</a>까지 열린 이 회의의 목적은 <a href="/wiki/%EB%82%98%ED%8F%B4%EB%A0%88%EC%98%B9_%EC%A0%84%EC%9F%81" title="나폴레옹 전쟁">나폴레옹 전쟁</a>의 혼란을 수습하고, <a href="/wiki/%EC%9C%A0%EB%9F%BD" title="유럽">유럽</a>의 상태를 전쟁 전으로 돌리는 것이 목표였다. 즉 <a href="/wiki/%ED%94%84%EB%9E%91%EC%8A%A4_%ED%98%81%EB%AA%85" title="프랑스 혁명">프랑스 혁명</a> 이전의 유럽의 왕정 체제를 다시 보수하고 유지하는 것이 목표였다. 따라서 나폴레옹 전쟁과 같은 유럽의 기존 체제를 위협할 일을 예방하기 위해 <a href="/wiki/%ED%94%84%EB%9E%91%EC%8A%A4" title="프랑스">프랑스</a>가 다시 강국이 되지 못하도록 견제하는 것도 빈 회의의 중요한 목표였다.(그러나 승전 4개국의 분열과 프랑스 외무상 탈레랑의 활약으로 이러한 목표는 잘 달성되지 못하였다) 이러한 <a href="/wiki/%EB%B0%98%EB%8F%99%EC%A3%BC%EC%9D%98" title="반동주의">반동</a>·<a href="/wiki/%EB%B3%B4%EC%88%98%EC%A3%BC%EC%9D%98" title="보수주의">보수</a>적 성격을 가진 회의의 결과 만들어진 것이 <a href="/wiki/%EB%B9%88_%EC%B2%B4%EC%A0%9C" title="빈 체제">빈 체제</a> 혹은 유럽 협조 체제(Concert of Europe)였다. 결론적으로 빈 체제는 빈 회의의 결과 만들어진 유럽의 반동적 체제를 의미한다. 여기서 반동이란 역사의 진보를 역행하는 과거로의 회귀를 의미한다.</p>
<p>빈 회의에서는 프랑스 혁명으로 몰락한 <a class="mw-redirect" href="/wiki/%EB%B6%80%EB%A5%B4%EB%B4%89_%EC%99%95%EA%B0%80" title="부르봉 왕가">부르봉 왕가</a>를 프랑스와 에스파냐에 복귀시키는 등 왕정이 몰락한 곳에 다시 <a class="mw-redirect" href="/wiki/%EC%99%95%EC%A0%95" title="왕정">왕정</a>을 부활시키고, <a href="/wiki/%EC%9E%90%EC%9C%A0%EC%A3%BC%EC%9D%98" title="자유주의">자유주의</a>, <a href="/wiki/%EB%AF%BC%EC%A1%B1%EC%A3%BC%EC%9D%98" title="민족주의">민족주의</a> 운동에 대한 반대 입장을 분명히 한다. 그리고 이 체제를 유지하기 위해 유럽 열강들은 소위 <a class="mw-redirect" href="/wiki/%EC%84%B8%EB%A0%A5_%EA%B7%A0%ED%98%95_%EC%B2%B4%EC%A0%9C" title="세력 균형 체제">세력 균형 체제</a>를 형성하게 된다. 그것이 앞에서 말한 빈 체제의 또다른 성격이다. 빈 회의의 결과로 형성된 빈 체제는 19세기 중엽까지 유럽 정치의 큰 틀이 된다.</p>
<h2><span class="mw-headline" id=".EB.8C.80.ED.91.9C.EC.9E.90">대표자</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%B9%88_%ED%9A%8C%EC%9D%98&amp;action=edit&amp;section=2" title="부분 편집: 대표자">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>상술된 바와 같이, 회의는 5대 강국에 의해 주도되었다.</p>
<ul>
<li><a href="/wiki/%EC%98%81%EA%B5%AD" title="영국">영국</a>에서는 외무상인 <a class="mw-redirect" href="/wiki/%EC%BA%90%EC%8A%AC%EB%A0%88%EC%9D%B4" title="캐슬레이">캐슬레이</a> 자작과 <a href="/wiki/%EC%9B%B0%EB%A7%81%ED%84%B4_%EA%B3%B5%EC%9E%91" title="웰링턴 공작">웰링턴 공작</a>이 대표로 참석하였다.</li>
<li><a href="/wiki/%ED%94%84%EB%A1%9C%EC%9D%B4%EC%84%BC_%EC%99%95%EA%B5%AD" title="프로이센 왕국">프로이센</a>에서는 <a href="/wiki/%ED%94%84%EB%A6%AC%EB%93%9C%EB%A6%AC%ED%9E%88_%EB%B9%8C%ED%97%AC%EB%A6%84_3%EC%84%B8" title="프리드리히 빌헬름 3세">프리드리히</a> 국왕이 직접 참석하였으나, 사절단을 대표하고 있는 것은 <a href="/wiki/%EC%9E%AC%EC%83%81" title="재상">재상</a>인 <a class="mw-redirect" href="/wiki/%ED%95%98%EB%A5%B4%EB%8D%B4%EB%B2%A0%EB%A5%B4%ED%81%AC" title="하르덴베르크">하르덴베르크</a> 공작이었다. <a href="/wiki/%EB%B9%8C%ED%97%AC%EB%A6%84_%ED%8F%B0_%ED%9B%94%EB%B3%BC%ED%8A%B8" title="빌헬름 폰 훔볼트">훔볼트</a>경이 직업외교관이자 학자로서 <a class="mw-redirect" href="/wiki/%ED%95%98%EB%A5%B4%EB%8D%B4%EB%B2%A0%EB%A5%B4%ED%81%AC" title="하르덴베르크">하르덴베르크</a> 공작을 보좌하고 있었다.</li>
<li><a href="/wiki/%EC%98%A4%EC%8A%A4%ED%8A%B8%EB%A6%AC%EC%95%84_%EC%A0%9C%EA%B5%AD" title="오스트리아 제국">오스트리아</a>는 회의의 주재국으로서 외무상인 <a class="mw-redirect" href="/wiki/%EB%A9%94%ED%85%8C%EB%A5%B4%EB%8B%88%ED%9E%88" title="메테르니히">메테르니히</a>가 대표하고 있었다. <a class="mw-redirect" href="/wiki/%EB%A9%94%ED%85%8C%EB%A5%B4%EB%8B%88%ED%9E%88" title="메테르니히">메테르니히</a>는 회의의 주재자이기도 하였으므로, 실질적으로 <a href="/wiki/%EC%98%A4%EC%8A%A4%ED%8A%B8%EB%A6%AC%EC%95%84_%EC%A0%9C%EA%B5%AD" title="오스트리아 제국">오스트리아</a>의 입장을 강경하게 대변하였던 것은 차석대표인 <a class="mw-redirect" href="/wiki/%EB%B2%A0%EC%84%BC%EB%B2%A0%EB%A5%B4%ED%81%AC" title="베센베르크">베센베르크</a> 남작이었다. 그 외에 <a href="/wiki/%EC%98%A4%EC%8A%A4%ED%8A%B8%EB%A6%AC%EC%95%84_%EC%A0%9C%EA%B5%AD" title="오스트리아 제국">오스트리아</a> 사절단의 군사고문으로 <a class="mw-redirect" href="/wiki/%EB%9D%BC%EB%8D%B0%EC%B8%A0%ED%82%A4" title="라데츠키">라데츠키</a> 등의 보좌관이 있었다.</li>
<li><a href="/wiki/%EB%9F%AC%EC%8B%9C%EC%95%84_%EC%A0%9C%EA%B5%AD" title="러시아 제국">러시아</a>는 황제인 <a href="/wiki/%EC%95%8C%EB%A0%89%EC%82%B0%EB%93%9C%EB%A5%B4_1%EC%84%B8" title="알렉산드르 1세">알렉산드르 1세</a>가 직접 참석하였으며, 형식상으로는 외무상인 <a class="mw-redirect" href="/wiki/%EB%84%A4%EC%85%80%EB%A1%9C%EB%8D%B0" title="네셀로데">네셀로데</a> 백작이 사절단을 대표하였으나, 실제로는 황제 본인이 각종 결정에 직접 참가하고 의견을 정하였다.</li>
<li><a href="/wiki/%ED%94%84%EB%9E%91%EC%8A%A4" title="프랑스">프랑스</a>는 패전국으로서 발언권이 높을 수 없었으나, 사절단을 대표한 외무상 <a class="mw-redirect" href="/wiki/%ED%83%88%EB%A0%88%EB%9E%91" title="탈레랑">탈레랑</a>의 유능함과 승전 4국의 분열을 교묘하게 이용하는 책략으로 점차 발언력을 높여 결국 회의의 합의사항 전반에 걸쳐 의견을 개진하고 반영시킬 수 있었다.</li>
</ul>
<p>상기한 5대 강국 외에도 유럽 각국이 사절단을 파견하여 회의에 자국의 입장을 반영하고 조그마한 이익이라도 얻기 위해 노력하였다. <a class="mw-redirect" href="/wiki/%EC%97%90%EC%8A%A4%ED%8C%8C%EB%83%90" title="에스파냐">에스파냐</a>는 <a class="mw-redirect" href="/wiki/%EC%8A%A4%EC%9B%A8%EB%8D%B4-%EB%85%B8%EB%A5%B4%EC%9B%A8%EC%9D%B4_%EC%97%B0%ED%95%A9%EC%99%95%EA%B5%AD" title="스웨덴-노르웨이 연합왕국">스웨덴-노르웨이 연합왕국</a>은 <a class="mw-redirect" href="/wiki/%EB%A1%9C%EC%9B%AC%ED%9E%88%EC%97%A0" title="로웬히엠">로웬히엠</a> 백작을 대표로 하는 사절단을 파견하였다. <a href="/wiki/%EB%82%98%ED%8F%B4%EB%A6%AC_%EC%99%95%EA%B5%AD" title="나폴리 왕국">나폴리 왕국</a>은 <a href="/wiki/%EC%A1%B0%EC%95%84%EC%83%9D_%EB%AE%88%EB%9D%BC" title="조아생 뮈라">뮈라</a>의 입장을 대변하는 사절단과 <a class="mw-redirect" href="/wiki/%EB%B6%80%EB%A5%B4%EB%B4%89_%EC%99%95%EA%B0%80" title="부르봉 왕가">부르봉 왕가</a>의 입장을 대변하는 사절단이 각각 파견되어 남부 이탈리아와 <a href="/wiki/%EC%8B%9C%EC%B9%A0%EB%A6%AC%EC%95%84" title="시칠리아">시칠리아</a>의 장래를 둘러싼 합의에 결사적으로 영향을 미치려 노력하였다. <a href="/wiki/%EA%B5%90%ED%99%A9%EB%A0%B9" title="교황령">교황령</a>에서는 <a class="mw-redirect" href="/wiki/%EC%BD%98%EC%82%B4%EB%B9%84_%EC%B6%94%EA%B8%B0%EA%B2%BD" title="콘살비 추기경">콘살비 추기경</a>, <a href="/wiki/%EC%98%A4%EC%8A%A4%EB%A7%8C_%EC%A0%9C%EA%B5%AD" title="오스만 제국">오스만 제국</a>에서는 마브로제니 파샤를 파견하였으며, <a href="/wiki/%EB%84%A4%EB%8D%9C%EB%9E%80%EB%93%9C" title="네덜란드">네덜란드</a>에서는 자국 주재 영국대사인 <a class="mw-redirect" href="/wiki/%ED%81%B4%EB%A0%8C%EC%B9%B4%ED%8B%B0_%EB%B0%B1%EC%9E%91" title="클렌카티 백작">클렌카티 백작</a>을 대표로 참석시켰다. 그 외에도 <a class="mw-redirect" <h2><span class="mw-headline" id=".ED.95.A9.EC.9D.98.EC.82.AC.ED.95.AD">합의사항</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%B9%88_%ED%9A%8C%EC%9D%98&amp;action=edit&amp;section=3" title="부분 편집: 합의사항">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>빈 회의의 결과, 합의에 도달한 빈 최종의정서의 주요 내용은 다음과 같다.</p>
<ul>
<li><a href="/wiki/%EB%9F%AC%EC%8B%9C%EC%95%84_%EC%A0%9C%EA%B5%AD" title="러시아 제국">러시아</a>는 <a class="mw-redirect" href="/wiki/%EB%B0%94%EB%A5%B4%EC%83%A4%EB%B0%94_%EB%8C%80%EA%B3%B5%EA%B5%AD" title="바르샤바 대공국">바르샤바 대공국</a>의 대부분을 차지하며, <a href="/wiki/%ED%95%80%EB%9E%80%EB%93%9C" title="핀란드">핀란드</a>를 계속 보유하는 것을 승인한다(1809년에 <a href="/wiki/%EC%8A%A4%EC%9B%A8%EB%8D%B4" title="스웨덴">스웨덴</a>으로부터 빼앗은 바 있다). 또한, 오스만 제국의 <a href="/wiki/%EB%B2%A0%EC%82%AC%EB%9D%BC%EB%B9%84%EC%95%84" title="베사라비아">베사라비아</a>의 영유권도 인정한다.</li>
<li><a href="/wiki/%ED%94%84%EB%A1%9C%EC%9D%B4%EC%84%BC_%EC%99%95%EA%B5%AD" title="프로이센 왕국">프로이센</a>은 <a href="/wiki/%EC%9E%91%EC%84%BC_%EC%99%95%EA%B5%AD" title="작센 왕국">작센</a>의 영토 58%를 얻으며, <a class="mw-redirect" href="/wiki/%EB%B0%94%EB%A5%B4%EC%83%A4%EB%B0%94_%EB%8C%80%EA%B3%B5%EA%B5%AD" title="바르샤바 대공국">바르샤바 대공국</a>의 일부인 <a class="new" href="/w/index.php?title=%ED%8F%AC%EC%A0%A0_%EA%B3%B5%EA%B5%AD&amp;action=edit&amp;redlink=1" title="포젠 공국 (없는 문서)">포젠 공국</a>과 <a class="mw-redirect" href="/wiki/%EB%8B%A8%EC%B9%98%ED%9E%88" title="단치히">단치히</a>, <a href="/wiki/%EB%9D%BC%EC%9D%B8%EB%9E%80%ED%8A%B8" title="라인란트">라인란트</a>, <a href="/wiki/%EB%B2%A0%EC%8A%A4%ED%8A%B8%ED%8C%94%EB%A0%8C_%EC%99%95%EA%B5%AD" title="베스트팔렌 왕국">베스트팔렌</a>을 차지한다.</li>
<li>옛 <a class="mw-redirect" href="/wiki/%EC%8B%A0%EC%84%B1%EB%A1%9C%EB%A7%88%EC%A0%9C%EA%B5%AD" title="신성로마제국">신성로마제국</a>의 제후국들을 정리하여 38개의 <a href="/wiki/%EC%98%81%EB%B0%A9%EA%B5%AD%EA%B0%80" title="영방국가">영방국가</a>를 모아 <a href="/wiki/%EB%8F%85%EC%9D%BC_%EC%97%B0%EB%B0%A9" title="독일 연방">독일 연방</a>을 창설한다. 연방의 의장은 <a href="/wiki/%EC%98%A4%EC%8A%A4%ED%8A%B8%EB%A6%AC%EC%95%84_%EC%A0%9C%EA%B5%AD" title="오스트리아 제국">오스트리아</a> 황제가 맡는다.</li>
<li><a href="/wiki/%EB%84%A4%EB%8D%9C%EB%9E%80%EB%93%9C" title="네덜란드">네덜란드</a>는 <a class="mw-redirect" href="/wiki/%EB%82%98%EC%82%AC%EC%9A%B0_%EB%B0%B1%EC%9E%91%EB%A0%B9" title="나사우 백작령">나사우</a>를 <a href="/wiki/%ED%94%84%EB%A1%9C%EC%9D%B4%EC%84%BC_%EC%99%95%EA%B5%AD" title="프로이센 왕국">프로이센</a>에 양여하는 대신, <a href="/wiki/%EC%98%A4%EC%8A%A4%ED%8A%B8%EB%A6%AC%EC%95%84%EB%A0%B9_%EB%84%A4%EB%8D%9C%EB%9E%80%EB%93%9C" title="오스트리아령 네덜란드">오스트리아령 네덜란드</a>와 <a class="new" href="/w/index.php?title=%EB%A6%AC%EC%97%90%EC%A3%BC_%EC%A3%BC%EA%B5%90%EB%A0%B9&amp;action=edit&amp;redlink=1" title="리에주 주교령 (없는 문서)">리에주 주교령</a>을 흡수하여 <a href="/wiki/%EB%84%A4%EB%8D%9C%EB%9E%80%EB%93%9C_%EC%97%B0%ED%95%A9%EC%99%95%EA%B5%AD" title="네덜란드 연합왕국">네덜란드 연합왕국</a>을 구성한다. 또한 <a href="/wiki/%EB%8F%85%EC%9D%BC_%EC%97%B0%EB%B0%A9" title="독일 연방">독일 연방</a>의 가맹국이 되는 <a href="/wiki/%EB%A3%A9%EC%85%88%EB%B6%80%EB%A5%B4%ED%81%AC" title="룩셈부르크">룩셈부르크 대공국</a>은 <a href="/wiki/%EB%84%A4%EB%8D%9C%EB%9E%80%EB%93%9C_%EC%97%B0%ED%95%A9%EC%99%95%EA%B5%AD" title="네덜란드 연합왕국">네덜란드 연합왕국</a>과 <a href="/wiki/%EB%8F%99%EA%B5%B0%EC%97%B0%ED%95%A9" title="동군연합">동군연합</a>을 이룬다.</li>
<li><a href="/wiki/%EC%8A%A4%EC%9B%A8%EB%8D%B4" title="스웨덴">스웨덴</a>은 <a href="/wiki/%ED%8F%AC%EB%A9%94%EB%9D%BC%EB%8B%88%EC%95%84" title="포메라니아">포메라니아</a>를 <a href="/wiki/%ED%94%84%EB%A1%9C%EC%9D%B4%EC%84%BC_%EC%99%95%EA%B5%AD" title="프로이센 왕국">프로이센</a>에 양여한다.</li>
<li><a href="/wiki/%EC%8A%A4%EC%9C%84%EC%8A%A4" title="스위스">스위스</a>의 <a href="/wiki/%EC%A4%91%EB%A6%BD" title="중립">중립</a>은 8국위원회(<a href="/wiki/%EC%98%81%EA%B5%AD" title="영국">영국</a>, <a href="/wiki/%ED%94%84%EB%A1%9C%EC%9D%B4%EC%84%BC_%EC%99%95%EA%B5%AD" title="프로이센 왕국">프로이센</a>, <a href="/wiki/%EC%98%A4%EC%8A%A4%ED%8A%B8%EB%A6%AC%EC%95%84_%EC%A0%9C%EA%B5%AD" title="오스트리아 제국">오스트리아</a>, <a href="/wiki/%EB%9F%AC%EC%8B%9C%EC%95%84_%EC%A0%9C%EA%B5%AD" title="러시아 제국">러시아</a>, <a href="/wiki/%ED%94%84%EB%9E%91%EC%8A%A4" title="프랑스">프랑스</a>, <a href="/wiki/%EC%8A%A4%EC%9B%A8%EB%8D%B4" title="스웨덴">스웨덴</a>, <a href="/wiki/%ED%8F%AC%EB%A5%B4%ED%88%AC%EA%B0%88" title="포르투갈">포르투갈</a>, <a class="mw-redirect" href="/wiki/%EC%97%90%EC%8A%A4%ED%8C%8C%EB%83%90" title="에스파냐">에스파냐</a>)에 의하여 보장된다.</li>
<li><a href="/wiki/%ED%95%98%EB%85%B8%EB%B2%84_%EC%99%95%EA%B5%AD" title="하노버 왕국">하노버</a>로부터 <a class="new" href="/w/index.php?title=%EC%9E%91%EC%84%BC%EB%9D%BC%EC%9A%B0%EC%97%94%EB%B6%80%EB%A5%B4%ED%81%AC_%EA%B3%B5%EA%B5%AD&amp;action=edit&amp;redlink=1" title="작센라우엔부르크 공국 (없는 문서)">작센라우엔부르크 공국</a>과의 <a href="/wiki/%EB%8F%99%EA%B5%B0%EC%97%B0%ED%95%A9" title="동군연합">동군연합</a>을 해체한다. 이를 <a href="/wiki/%EB%AE%8C%EC%8A%A4%ED%84%B0_%EC%A3%BC%EA%B5%90%EB%A0%B9" title="뮌스터 주교령">뮌스터 주교령</a>, 프로이센령 동프리지아와 합쳐 왕국으로 만들어 <a href="/wiki/%EB%8D%B4%EB%A7%88%ED%81%AC" title="덴마크">덴마크</a>와 <a href="/wiki/%EB%8F%99%EA%B5%B0%EC%97%B0%ED%95%A9" title="동군연합">동군연합</a>을 이루도록 한다.</li>
<li><a class="mw-redirect" href="/wiki/%EB%B0%94%EC%9D%B4%EC%97%90%EB%A5%B8" title="바이에른">바이에른</a>은 <a class="new" href="/w/index.php?title=%EB%9D%BC%EC%9D%B8_%EA%B6%81%EC%A4%91%EB%B0%B1&amp;action=edit&amp;redlink=1" title="라인 궁중백 (없는 문서)">라인 궁중백</a>의 영지를 흡수하고, <a class="mw-redirect" href="/wiki/%ED%94%84%EB%9E%91%ED%81%AC%ED%91%B8%EB%A5%B4%ED%8A%B8" title="프랑크푸르트">프랑크푸르트</a> 대공국을 얻는다. <a class="mw-redirect" href="/wiki/%ED%97%A4%EC%84%BC-%EB%8B%A4%EB%A6%84%EC%8A%88%ED%83%80%ED%8A%B8" title="헤센-다름슈타트">헤센-다름슈타트</a>는 <a href="/wiki/%EB%B2%A0%EC%8A%A4%ED%8A%B8%ED%8C%94%EB%A0%8C_%EC%99%95%EA%B5%AD" title="베스트팔렌 왕국">베스트팔렌</a>을 <a href="/wiki/%ED%94%84%EB%A1%9C%EC%9D%B4%EC%84%BC" title="프로이센">프로이센</a>에 양여하는 대가로, <a class="new" href="/w/index.php?title=%EB%A7%88%EC%9D%B8%EC%B8%A0_%EC%A3%BC%EA%B5%90%EB%A0%B9&amp;action=edit&amp;redlink=1" title="마인츠 주교령 (없는 문서)">마인츠 주교령</a>을 얻는다.</li>
<li><a href="/wiki/%EC%98%A4%EC%8A%A4%ED%8A%B8%EB%A6%AC%EC%95%84_%EC%A0%9C%EA%B5%AD" title="오스트리아 제국">오스트리아</a>는 남독일과 남부 네덜란드를 포기하는 대가로, 이탈리아 지역에서 보상을 얻는다. <a href="/wiki/%ED%8B%B0%EB%A1%A4" title="티롤">티롤</a>과 <a href="/wiki/%EC%9E%98%EC%B8%A0%EB%B6%80%EB%A5%B4%ED%81%AC" title="잘츠부르크">잘츠부르크</a>를 회복하고, <a class="mw-redirect" href="/wiki/%EB%A1%AC%EB%B0%94%EB%A5%B4%EB%94%94%EC%95%84" title="롬바르디아">롬바르디아</a>, <a href="/wiki/%EB%B2%A0%EB%84%A4%EC%B9%98%EC%95%84" title="베네치아">베네치아</a>를 새로이 얻으며, <a href="/wiki/%EB%8B%AC%EB%A7%88%ED%8B%B0%EC%95%84" title="달마티아">달마티아</a> 지방의 <a class="mw-disambig" href="/wiki/%EB%9D%BC%EA%B5%AC%EC%82%AC" title="라구사">라구사</a>도 차지한다. 또한 <a class="mw-redirect" href="/wiki/%ED%95%A9%EC%8A%A4%EB%B6%80%EB%A5%B4%ED%81%AC%EB%A1%9C%ED%8A%B8%EB%A7%81%EA%B2%90_%EC%99%95%EA%B0%80" title="합스부르크로트링겐 왕가">합스부르크로트링겐 왕가</a>의 분가가 다스리던 <a href="/wiki/%ED%86%A0%EC%8A%A4%EC%B9%B4%EB%82%98_%EB%8C%80%EA%B3%B5%EA%B5%AD" title="토스카나 대공국">토스카나 대공국</a>과 <a class="mw-redirect" href="/wiki/%EB%AA%A8%EB%8D%B0%EB%82%98_%EA%B3%B5%EA%B5%AD" title="모데나 공국">모데나 공국</a>에 군주들이 다시 복귀한다.</li>
<li><a href="/wiki/%EA%B5%90%ED%99%A9%EB%A0%B9" title="교황령">교황령</a>은 <a href="/wiki/%EB%82%98%ED%8F%B4%EB%A0%88%EC%98%B9_%EC%A0%84%EC%9F%81" title="나폴레옹 전쟁">나폴레옹 전쟁</a> 이전으로 복귀하여 옛 영지를 모두 회복하여 복구한다. 단, <a href="/wiki/%EC%95%84%EB%B9%84%EB%87%BD" title="아비뇽">아비뇽</a>만은 <a href="/wiki/%ED%94%84%EB%9E%91%EC%8A%A4" title="프랑스">프랑스</a>의 영역으로 남는다.</li>
<li><a href="/wiki/%EC%98%81%EA%B5%AD" title="영국">영국</a>은 유럽 대륙 안에서 <a href="/wiki/%EB%AA%B0%ED%83%80_%EC%84%AC" title="몰타 섬">몰타 섬</a>과 <a href="/wiki/%EC%9D%B4%EC%98%A4%EB%8B%88%EC%95%84_%EC%A0%9C%EB%8F%84" title="이오니아 제도">이오니아 제도</a>, <a href="/wiki/%EC%BC%80%EC%9D%B4%ED%94%84_%EC%8B%9D%EB%AF%BC%EC%A7%80" title="케이프 식민지">케이프 식민지</a> 그리고 <a href="/wiki/%EC%8A%A4%EB%A6%AC%EB%9E%91%EC%B9%B4" title="스리랑카">실론</a> 등의 영유권을 인정받는다.</li>
<li><a href="/wiki/%EC%82%AC%EB%A5%B4%EB%8D%B0%EB%83%90_%EC%99%95%EA%B5%AD" title="사르데냐 왕국">사르데냐 왕국</a>은 <a class="mw-redirect" href="/wiki/%ED%94%BC%EC%97%90%EB%AA%AC%ED%85%8C" title="피에몬테">피에몬테</a>와 <a href="/wiki/%EB%8B%88%EC%8A%A4_(%ED%94%84%EB%9E%91%EC%8A%A4)" title="니스 (프랑스)">니스</a>, <a href="/wiki/%EC%82%AC%EB%B3%B4%EC%9D%B4_%EA%B3%B5%EA%B5%AD" title="사보이 공국">사부아</a>의 영지를 회복하며, 추가로 <a href="/wiki/%EC%A0%9C%EB%85%B8%EB%B0%94_%EA%B3%B5%ED%99%94%EA%B5%AD" title="제노바 공화국">제노바 공화국</a>을 왕국의 영토로 흡수한다.</li>
<li><a href="/wiki/%ED%8C%8C%EB%A5%B4%EB%A7%88_%EA%B3%B5%EA%B5%AD" title="파르마 공국">파르마피아첸차 공국</a>은 <a class="mw-redirect" href="/wiki/%EB%82%98%ED%8F%B4%EB%A0%88%EC%98%B9" title="나폴레옹">나폴레옹</a>의 아내, <a class="mw-redirect" href="/wiki/%EB%A7%88%EB%A6%AC%EC%95%84_%EB%A3%A8%EC%9D%B4%EC%82%AC" title="마리아 루이사">오스트리아의 마리 루이즈</a>에게 주어진다.</li>
<li><a href="/wiki/%ED%8C%8C%EB%A5%B4%EB%A7%88_%EA%B3%B5%EA%B5%AD" title="파르마 공국">파르마피아첸차 공국</a>을 잃은 <a class="mw-redirect" href="/wiki/%EB%B6%80%EB%A5%B4%EB%B4%89%ED%8C%8C%EB%A5%B4%EB%A7%88_%EA%B0%80%EB%AC%B8" title="부르봉파르마 가문">부르봉파르마 가문</a>을 위해 <a class="new" href="/w/index.php?title=%EB%A3%A8%EC%B9%B4_%EA%B3%B5%EA%B5%AD&amp;action=edit&amp;redlink=1" title="루카 공국 (없는 문서)">루카 공국</a>을 신설한다. <a class="mw-redirect" href="/wiki/%EB%A7%88%EB%A6%AC%EC%95%84_%EB%A3%A8%EC%9D%B4%EC%82%AC" title="마리아 루이사">오스트리아의 마리 루이즈</a> 사후에는 파르마피아첸차 공국도 회복한다.</li>
<li><a href="/wiki/%EB%82%98%ED%8F%B4%EB%A6%AC_%EC%99%95%EA%B5%AD" title="나폴리 왕국">나폴리 왕국</a>은 <a class="new" href="/w/index.php?title=%EB%B6%80%EB%A5%B4%EB%B4%89%EC%96%91%EC%8B%9C%EC%B9%A0%EB%A6%AC%EC%95%84_%EC%99%95%EA%B0%80&amp;action=edit&amp;redlink=1" title="부르봉양시칠리아 왕가 (없는 문서)">부르봉나폴리 왕가</a>의 <a href="/wiki/%ED%8E%98%EB%A5%B4%EB%94%94%EB%82%9C%EB%8F%84_1%EC%84%B8_(%EC%96%91%EC%8B%9C%EC%B9%A0%EB%A6%AC%EC%95%84)" title="페르디난도 1세 (양시칠리아)">페르디난도 4세</a>가 복위하며, <a href="/wiki/%EB%8F%99%EA%B5%B0%EC%97%B0%ED%95%A9" title="동군연합">동군연합</a>을 이루던 <a href="/wiki/%EC%8B%9C%EC%B9%A0%EB%A6%AC%EC%95%84_%EC%99%95%EA%B5%AD" title="시칠리아 왕국">시칠리아 왕국</a>을 합병한다.</li>
<li><a class="mw-redirect" href="/wiki/%EB%85%B8%EC%98%88%EB%AC%B4%EC%97%AD" title="노예무역">노예무역</a>이 인도주의에 반함을 선언한다. 이는 회의 기간 내내 노예무역 금지의 선언을 강력히 주장한 <a href="/wiki/%EC%98%81%EA%B5%AD" title="영국">영국</a>의 입장이 어느 정도 반영된 것이었다.</li>
<li><a href="/wiki/%EB%9D%BC%EC%9D%B8_%EA%B0%95" title="라인 강">라인 강</a>을 자유항행하천으로 정한다.</li>
</ul>
