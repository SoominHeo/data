<p><b>윌리엄 펠턴 "빌" 러셀</b>(William Felton "Bill" Russell, <a href="/wiki/1934%EB%85%84" title="1934년">1934년</a> <a href="/wiki/2%EC%9B%94_12%EC%9D%BC" title="2월 12일">2월 12일</a> ~ )은 전 미국의 농구 선수이다. 미국 프로 농구 <a href="/wiki/%EB%B3%B4%EC%8A%A4%ED%84%B4_%EC%85%80%ED%8B%B1%EC%8A%A4" title="보스턴 셀틱스">보스턴 셀틱스</a>의 선수로 활약하면서 통산 11번의 NBA 우승을 이뤘다. 네이스미스 농구 명예의 전당과 미국 대학 농구 명예의 전당에 헌액되기도 하였다.</p>
<h2><span class="mw-headline" id=".EC.84.A0.EC.88.98_.EA.B2.BD.EB.A0.A5">선수 경력</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%B9%8C_%EB%9F%AC%EC%85%80&amp;action=edit&amp;section=1" title="부분 편집: 선수 경력">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p><a href="/wiki/%EB%A3%A8%EC%9D%B4%EC%A7%80%EC%95%A0%EB%82%98_%EC%A3%BC" title="루이지애나 주">루이지애나 주</a> 먼로에서 태어나 <a class="new" href="/w/index.php?title=%EC%83%8C%ED%94%84%EB%9E%80%EC%8B%9C%EC%8A%A4%EC%BD%94_%EB%8C%80%ED%95%99%EA%B5%90&amp;action=edit&amp;redlink=1" title="샌프란시스코 대학교 (없는 문서)">샌프란시스코 대학교</a>에 진학하였고, 대학 시절 자신의 팀이 1954~55 시즌과 1955~56 시즌 동안에 58개의 경기들 중에 57개를 승리하는 데 기여했다. 신장 6 피트 10 인치(2.08m)로 <a class="mw-redirect" href="/wiki/1956%EB%85%84_%EB%A9%9C%EB%B2%84%EB%A5%B8_%EC%98%AC%EB%A6%BC%ED%94%BD" title="1956년 멜버른 올림픽">1956년 멜버른 올림픽</a>에 국가 대표로 나가 <a href="/wiki/%EA%B8%88%EB%A9%94%EB%8B%AC" title="금메달">금메달</a>을 획득한 후, <a href="/wiki/%EB%B3%B4%EC%8A%A4%ED%84%B4_%EC%85%80%ED%8B%B1%EC%8A%A4" title="보스턴 셀틱스">셀틱스</a>에 입단하여 13년 동안 활약하면서 팀이 11개의 <a class="mw-redirect" href="/wiki/NBA" title="NBA">NBA</a> 선수권을 우승하는 데 도움이 되었다. 러셀은 셀틱스의 센터로서 블로킹 슛과 리바운드의 마스터가 되었다. <a href="/wiki/%EC%9C%8C%ED%8A%B8_%EC%B2%B4%EC%9E%84%EB%B2%8C%EB%A6%B0" title="윌트 체임벌린">윌트 체임벌린</a>에 이은 <a class="mw-redirect" href="/wiki/NBA" title="NBA">NBA</a> 통산 리바운드 2위의 기록을 가지고 있다.</p>
<h2><span class="mw-headline" id=".EA.B0.90.EB.8F.85_.EA.B2.BD.EB.A0.A5">감독 경력</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%B9%8C_%EB%9F%AC%EC%85%80&amp;action=edit&amp;section=2" title="부분 편집: 감독 경력">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p><a href="/wiki/1966%EB%85%84" title="1966년">1966년</a>부터 <a href="/wiki/1969%EB%85%84" title="1969년">1969년</a>에 은퇴할 때까지 감독 겸 선수로 활약한 러셀은 프로 스포츠 역사상 최초의 <a href="/wiki/%EC%95%84%ED%94%84%EB%A6%AC%EC%B9%B4%EA%B3%84_%EB%AF%B8%EA%B5%AD%EC%9D%B8" title="아프리카계 미국인">아프리카계 미국인</a> 총감독이 되었다. <a href="/wiki/1973%EB%85%84" title="1973년">1973년</a>부터 <a href="/wiki/1977%EB%85%84" title="1977년">1977년</a>까지 <a class="mw-redirect" href="/wiki/%EC%98%A4%ED%81%B4%EB%9D%BC%ED%98%B8%EB%A7%88%EC%8B%9C%ED%8B%B0_%EC%8D%AC%EB%8D%94" title="오클라호마시티 썬더">시애틀 수퍼소닉스</a>의 총지배인 겸 감독을 맡았으며, <a href="/wiki/1987%EB%85%84" title="1987년">1987년</a>부터 <a href="/wiki/1988%EB%85%84" title="1988년">1988년</a>까지 <a href="/wiki/%EC%83%88%ED%81%AC%EB%9D%BC%EB%A9%98%ED%86%A0_%ED%82%B9%EC%8A%A4" title="새크라멘토 킹스">새크라멘토 킹스</a>의 감독을 지내다가, <a href="/wiki/1989%EB%85%84" title="1989년">1989년</a>까지 그 팀의 부회장으로 있었다. 이후에 감독 양도들 사이의 <a class="mw-redirect" href="/wiki/TV" title="TV">TV</a> 스포츠의 시사해설자가 되었다. 저서로 자신의 인생과 전망을 논의한 《영광을 지원하며》(1970)와 《두 번째의 바람》(1979)이 있다.</p>
