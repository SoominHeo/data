<h1 class="firstHeading" id="firstHeading" lang="ko">8주기 원소</h1>
<div class="mw-body-content" id="bodyContent">
<div id="siteSub">위키백과, 우리 모두의 백과사전.</div>
<div id="contentSub"></div>
<div class="mw-jump" id="jump-to-nav">
					이동:					<a href="#mw-head">둘러보기</a>, 					<a href="#p-search">검색</a>
</div>
<div class="mw-content-ltr" dir="ltr" id="mw-content-text" lang="ko"><script>function mfTempOpenSection(id){var block=document.getElementById("mf-section-"+id);block.className+=" open-block";block.previousSibling.className+=" open-block";}</script><p><b>8주기 원소(<span style="font-size: smaller;"><a href="/wiki/%EC%98%81%EC%96%B4" title="영어">영어</a>:</span> <span lang="en" xml:lang="en">Period 8 Element</span>)</b>는 주기율표에서 8주기에 있는 원소들을 말한다. 모두 미발견된 원소들이며, <a href="/wiki/%EC%9A%B0%EB%88%84%EB%84%A8%EB%8A%84" title="우누넨늄">우누넨늄</a>과 <a href="/wiki/%EC%9A%B4%EB%B9%84%EB%8B%90%EB%A5%A8" title="운비닐륨">운비닐륨</a>, 그리고 <a href="/wiki/%EC%9A%B4%EB%B9%84%EB%B8%80" title="운비븀">운비븀</a>, <a href="/wiki/%EC%9A%B4%EB%B9%84%ED%97%A5%EC%8A%98" title="운비헥슘">운비헥슘</a>은 발견될 가능성이 충분히 있는 원소들이다. <a href="/wiki/%EC%8C%93%EC%9D%8C_%EC%9B%90%EB%A6%AC" title="쌓음 원리">쌓음 원리</a> 가 무너져 164번에서 끝난다.</p>
<h2><span class="mw-headline" id=".EB.B6.84.EB.A5.98">분류</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=8%EC%A3%BC%EA%B8%B0_%EC%9B%90%EC%86%8C&amp;action=edit&amp;section=1" title="부분 편집: 분류">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>8주기 원소는 119번과 120번 원소, <a href="/wiki/%EC%8A%88%ED%8D%BC%EC%95%85%ED%8B%B0%EB%8A%84%EC%A1%B1" title="슈퍼악티늄족">슈퍼악티늄족</a>, 7d <a class="mw-redirect" href="/wiki/%EC%A0%84%EC%9D%B4%EA%B8%88%EC%86%8D" title="전이금속">전이금속</a>으로 나눌 수 있다. <a href="/wiki/%EC%8C%93%EC%9D%8C_%EC%9B%90%EB%A6%AC" title="쌓음 원리">쌓음 원리</a> 가 무너져 8p<sub>1/2</sub> 오비탈의 전자 2개가 <a href="/wiki/%EC%8A%88%ED%8D%BC%EC%95%85%ED%8B%B0%EB%8A%84%EC%A1%B1" title="슈퍼악티늄족">슈퍼악티늄족</a>에서 먼저 차버리게 된다. 나머지 8p오비탈인 8p<sub>3/2</sub> 오비탈보다 9s 오비탈이 먼저 차기 때문에 164번에서 8주기 원소가 끝나고 165번 이후는 <a href="/wiki/9%EC%A3%BC%EA%B8%B0_%EC%9B%90%EC%86%8C" title="9주기 원소">9주기 원소</a>로 분류한다. 따라서 8주기 원소에는 <a href="/wiki/%EC%A0%84%EC%9D%B4%ED%9B%84_%EA%B8%88%EC%86%8D" title="전이후 금속">전이후 금속</a>이나 <a href="/wiki/%EC%A4%80%EA%B8%88%EC%86%8D" title="준금속">준금속</a>, <a href="/wiki/%EB%B9%84%EA%B8%88%EC%86%8D" title="비금속">비금속</a>이 존재하지 않는다.</p>
