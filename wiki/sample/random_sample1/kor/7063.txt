<p><b>빗추 다카마쓰 성 전투</b>(<span style="font-size: smaller;"><a href="/wiki/%EC%9D%BC%EB%B3%B8%EC%96%B4" title="일본어">일본어</a>:</span> <span lang="ja"><span lang="ja"><ruby><rb>備中高松城の戦い</rb> <rp>(</rp><rt>びっちゅうたかまつじょうのたたかい</rt><rp>)</rp></ruby></span></span>)는 <a href="/wiki/%EC%9D%BC%EB%B3%B8" title="일본">일본</a> <a class="mw-redirect" href="/wiki/%EC%95%84%EC%A6%88%EC%B9%98%EB%AA%A8%EB%AA%A8%EC%95%BC%EB%A7%88_%EC%8B%9C%EB%8C%80" title="아즈치모모야마 시대">아즈치모모야마 시대</a>의 전투로, <a class="mw-disambig" href="/wiki/%EB%8D%B4%EC%87%BC" title="덴쇼">덴쇼</a> 10년(<a href="/wiki/1582%EB%85%84" title="1582년">1582년</a>)에 <a href="/wiki/%EC%98%A4%EB%8B%A4_%EB%85%B8%EB%B6%80%EB%82%98%EA%B0%80" title="오다 노부나가">오다 노부나가</a>의 명령을 받은 <a class="mw-redirect" href="/wiki/%ED%95%98%EC%8B%9C%EB%B0%94_%ED%9E%88%EB%8D%B0%EC%9A%94%EC%8B%9C" title="하시바 히데요시">하시바 히데요시</a>가 <a class="new" href="/w/index.php?title=%EB%AA%A8%EB%A6%AC_%EA%B0%80%EB%AC%B8&amp;action=edit&amp;redlink=1" title="모리 가문 (없는 문서)">모리 가문</a>의 가신 <a href="/wiki/%EC%8B%9C%EB%AF%B8%EC%A6%88_%EB%AC%B4%EB%84%A4%ED%95%98%EB%A3%A8" title="시미즈 무네하루">시미즈 무네하루</a>(清水宗治)가 지키는 <a href="/wiki/%EB%B9%97%EC%B6%94_%EA%B5%AD" title="빗추 국">빗추</a> <a href="/wiki/%EB%8B%A4%EC%B9%B4%EB%A7%88%EC%93%B0_%EC%84%B1_(%EC%98%A4%EC%B9%B4%EC%95%BC%EB%A7%88_%ED%98%84)" title="다카마쓰 성 (오카야마 현)">다카마쓰 성</a>(高松城)을 공격한 전투이다.</p>
<p>덴쇼 10년 4월 히데요시는 다카마쓰 성을 공격하지만, 성 주변의 습지로 교착상태에 빠졌다. 5월 8일 히데요시는 군사 <a href="/wiki/%EA%B5%AC%EB%A1%9C%EB%8B%A4_%EC%9A%94%EC%8B%9C%ED%83%80%EC%B9%B4" title="구로다 요시타카">구로다 요시타카</a>의 책략으로 저지대에 위치한 다카마쓰 성을 수몰시키기 위하여 성 주위에 제방을 쌓기 시작한다. 제방이 완성되자 장마로 인해 성은 수몰되었다. 이를 <b>다카마쓰 성 수공</b>(高松城水攻め)이라고 한다.</p>
<p>그러나, 히데요시에게 유리한 상황으로 전투가 진행되고 있던 중에, <a href="/wiki/%EA%B5%90%ED%86%A0" title="교토">교토</a>의 <a href="/wiki/%ED%98%BC%EB%85%B8%EC%A7%80" title="혼노지">혼노지</a>에서 주군 오다 노부나가가 <a href="/wiki/%EC%95%84%EC%BC%80%EC%B9%98_%EB%AF%B8%EC%93%B0%ED%9E%88%EB%8D%B0" title="아케치 미쓰히데">아케치 미쓰히데</a>의 모반으로 목숨을 잃는 사건이 발생하였다(<a href="/wiki/%ED%98%BC%EB%85%B8%EC%A7%80%EC%9D%98_%EB%B3%80" title="혼노지의 변">혼노지의 변</a>). 이 급보를 들은 히데요시는 성주 시미즈 무네하루만 할복하면 성 수비병의 목숨을 모두 구해주겠다는 조건으로 즉시 모리 측에 화의를 제의하였다. 히데요시는 성주 시미즈 무네하루의 할복을 지켜본 뒤, 아케치 미쓰히데를 토벌하기 위하여 군을 이끌고 <a href="/wiki/%EC%95%BC%EB%A7%88%EC%8B%9C%EB%A1%9C_%EA%B5%AD" title="야마시로 국">야마시로 국</a>을 향하여 급히 회군하였다.</p>
<h2><span class="mw-headline" id=".EC.A0.84.ED.88.AC_.ED.9B.84.EC.9D.98_.EC.A0.95.EC.84.B8">전투 후의 정세</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%B9%97%EC%B6%94_%EB%8B%A4%EC%B9%B4%EB%A7%88%EC%93%B0_%EC%84%B1_%EC%A0%84%ED%88%AC&amp;action=edit&amp;section=1" title="부분 편집: 전투 후의 정세">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>히데요시는 그해 <a href="/wiki/%EC%9D%8C%EB%A0%A5_6%EC%9B%94_13%EC%9D%BC" title="음력 6월 13일">6월 13일</a>부터 시작된 <a href="/wiki/%EC%95%BC%EB%A7%88%EC%9E%90%ED%82%A4_%EC%A0%84%ED%88%AC" title="야마자키 전투">야마자키 전투</a>에서 미쓰히데를 격파하고 오다 노부나가의 실질적인 후계자의 길을 걷게 되었다. <a class="new" href="/w/index.php?title=%EC%A3%BC%EA%B3%A0%EC%BF%A0_%EC%A0%95%EB%B2%8C&amp;action=edit&amp;redlink=1" title="주고쿠 정벌 (없는 문서)">주고쿠 정벌</a>로 히데요시의 실력을 익히 알고 있는 모리 가문은 히데요시와의 화의를 유지하며, 그의 패업을 지원하였다. 이에 대한 보답으로 모리 가문은 후에 <a href="/wiki/%EB%8F%84%EC%9A%94%ED%86%A0%EB%AF%B8_%EC%A0%95%EA%B6%8C" title="도요토미 정권">도요토미 정권</a>에서 중용되어, <a href="/wiki/%EB%AA%A8%EB%A6%AC_%EB%8D%B0%EB%A3%A8%EB%AA%A8%ED%86%A0" title="모리 데루모토">모리 데루모토</a>와 <a href="/wiki/%EA%B3%A0%EB%B0%94%EC%95%BC%EC%B9%B4%EC%99%80_%EB%8B%A4%EC%B9%B4%EC%B9%B4%EA%B2%8C" title="고바야카와 다카카게">고바야카와 다카카게</a>는 <a class="mw-redirect" href="/wiki/%EA%B3%A0%EB%8B%A4%EC%9D%B4%EB%A1%9C" title="고다이로">고다이로</a>가 되었다.</p>
<p>이 전투 뒤에 빗추 다카마쓰 성에는 우키타 가문(宇喜多氏)의 가로 <a class="new" href="/w/index.php?title=%ED%95%98%EB%82%98%EB%B6%80%EC%82%AC_%EA%B0%80%EB%AC%B8&amp;action=edit&amp;redlink=1" title="하나부사 가문 (없는 문서)">하나부사 가문</a>(花房氏)가 입성하였다. 후에 하나부사 가문은 <a href="/wiki/%EC%84%B8%ED%82%A4%EA%B0%80%ED%95%98%EB%9D%BC_%EC%A0%84%ED%88%AC" title="세키가하라 전투">세키가하라 전투</a>에서 서군의 부대장이 된 주군 <a href="/wiki/%EC%9A%B0%ED%82%A4%ED%83%80_%ED%9E%88%EB%8D%B0%EC%9D%B4%EC%97%90" title="우키타 히데이에">우키타 히데이에</a>를 배신하고 동군에 붙었기 때문에 <a href="/wiki/%EC%97%90%EB%8F%84_%EC%8B%9C%EB%8C%80" title="에도 시대">에도 시대</a>에는 <a href="/wiki/%ED%95%98%ED%83%80%EB%AA%A8%ED%86%A0" title="하타모토">하타모토</a>까지 출세하였다. 몇 년 후 하나부사 가문이 빗추 <a href="/wiki/%EC%86%8C%EC%9E%90_%EC%8B%9C" title="소자 시">아소</a>로 거소를 옮기자, 다카마쓰 성은 폐성되었다.</p>
