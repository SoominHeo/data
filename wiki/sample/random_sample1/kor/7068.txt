<p><b>빙엔의 힐데가르트</b>(<span style="font-size: smaller;"><a href="/wiki/%EB%8F%85%EC%9D%BC%EC%96%B4" title="독일어">독일어</a>:</span> <span lang="de" xml:lang="de">Hildegard von Bingen</span> 힐데가르트 폰 빙엔; <span style="font-size: smaller;"><a href="/wiki/%EB%9D%BC%ED%8B%B4%EC%96%B4" title="라틴어">라틴어</a>:</span> <span lang="la" xml:lang="la">Hildegardis Bingensis</span>, <a href="/wiki/1098%EB%85%84" title="1098년">1098년</a> – <a href="/wiki/1179%EB%85%84" title="1179년">1179년</a> <a href="/wiki/9%EC%9B%94_17%EC%9D%BC" title="9월 17일">9월 17일</a>)은 <b>축복받은 힐데가르트</b>(Blessed Hildegard) 또는 <b>성 힐데가르트</b>(Saint Hildegard)로도 알려져 있다. 그녀는 <a href="/wiki/%EB%8F%85%EC%9D%BC" title="독일">독일</a>의 <a href="/wiki/%EC%88%98%EB%85%80" title="수녀">수녀</a>, <a href="/wiki/%EC%98%88%EC%88%A0%EA%B0%80" title="예술가">예술가</a>, <a href="/wiki/%EC%9E%91%EA%B0%80" title="작가">작가</a>, <a class="mw-redirect mw-disambig" href="/wiki/%EC%B9%B4%EC%9A%B4%EC%85%80%EB%9F%AC" title="카운셀러">카운셀러</a>, <a class="mw-redirect" href="/wiki/%EC%96%B8%EC%96%B4%ED%95%99%EC%9E%90" title="언어학자">언어학자</a>, <a class="new" href="/w/index.php?title=%EC%9E%90%EC%97%B0%ED%95%99%EC%9E%90&amp;action=edit&amp;redlink=1" title="자연학자 (없는 문서)">자연학자</a>, <a href="/wiki/%EA%B3%BC%ED%95%99%EC%9E%90" title="과학자">과학자</a>, <a href="/wiki/%EC%B2%A0%ED%95%99%EC%9E%90" title="철학자">철학자</a>, <a href="/wiki/%EC%9D%98%EC%82%AC" title="의사">의사</a>, <a class="new" href="/w/index.php?title=%EC%95%BD%EC%B4%88%ED%95%99%EC%9E%90&amp;action=edit&amp;redlink=1" title="약초학자 (없는 문서)">약초학자</a>, <a href="/wiki/%EC%8B%9C%EC%9D%B8" title="시인">시인</a>, <a class="mw-redirect" href="/wiki/%EC%9A%B4%EB%8F%99%EA%B0%80" title="운동가">운동가</a>, <a href="/wiki/%EC%98%88%EC%96%B8%EC%9E%90" title="예언자">예언자</a>, <a href="/wiki/%EC%9E%91%EA%B3%A1%EA%B0%80" title="작곡가">작곡가</a>였다. <a href="/wiki/1136%EB%85%84" title="1136년">1136년</a> <a class="extiw" href="https://en.wiktionary.org/wiki/magistra" title="wiktionary:magistra">마기스트라</a>로 뽑힌 후, 그녀는 1150년에 <a class="new" href="/w/index.php?title=%EB%A3%A8%ED%8E%98%EB%A5%B4%EC%B8%A0%EB%B2%A0%EB%A5%B4%ED%81%AC&amp;action=edit&amp;redlink=1" title="루페르츠베르크 (없는 문서)">루페르츠베르크</a> 수도원과, <a href="/wiki/1165%EB%85%84" title="1165년">1165년</a>에 <a class="new" href="/w/index.php?title=%EC%95%84%EC%9D%B4%EB%B9%99%EA%B2%90&amp;action=edit&amp;redlink=1" title="아이빙겐 (없는 문서)">아이빙겐</a> 수도원을 세운다.</p>
<p>그녀는 일관되게 작곡한 최초의 작곡가였다. 그의 작품 "오르도 비르투툼(Ordo Virtutum)는 첫 번째 형식(the first form)이라고 불리며, <a href="/wiki/%EC%98%A4%ED%8E%98%EB%9D%BC" title="오페라">오페라</a>의 기원일 것이다.<sup class="reference" id="cite_ref-1"><a href="#cite_note-1">[1]</a></sup><sup class="reference" id="cite_ref-2"><a href="#cite_note-2">[2]</a></sup></p>
<p>그녀는 <a href="/wiki/%EC%8B%A0%ED%95%99" title="신학">신학</a>, <a href="/wiki/%EC%8B%9D%EB%AC%BC%ED%95%99" title="식물학">식물학</a>, <a href="/wiki/%EC%9D%98%ED%95%99" title="의학">의학</a> 서적을 썼으며, 편지, 서정시, 음악, 그리고 현재 남아 있는 가장 오래된 도덕극을 썼다.</p>
<p>적어도 하나의 현대 전기작가는 빙엔의 힐데가르트를 <a class="new" href="/w/index.php?title=%EB%A7%8C%EB%AC%BC%EB%B0%95%EC%82%AC&amp;action=edit&amp;redlink=1" title="만물박사 (없는 문서)">만물박사</a>로 그린다.<sup class="reference" id="cite_ref-3"><a href="#cite_note-3">[3]</a></sup></p>
<h2><span class="mw-headline" id=".EC.98.81.ED.99.94">영화</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%B9%99%EC%97%94%EC%9D%98_%ED%9E%90%EB%8D%B0%EA%B0%80%EB%A5%B4%ED%8A%B8&amp;action=edit&amp;section=1" title="부분 편집: 영화">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p><a href="/wiki/%EB%A7%88%EA%B0%80%EB%A0%88%ED%85%8C_%ED%8F%B0_%ED%8A%B8%EB%A1%9C%ED%83%80" title="마가레테 폰 트로타">마가레테 폰 트로타</a> 감독의 영화 '위대한 계시'(2009)는 힐데가르트 폰 빙겐의 생애를 담고 있다.</p>
