<p><b>빠따니 주</b>(<span style="font-size: smaller;"><a href="/wiki/%ED%83%9C%EA%B5%AD%EC%96%B4" title="태국어">태국어</a>:</span> <span lang="th" xml:lang="th">ปัตตานี</span>)는 <a href="/wiki/%ED%83%80%EC%9D%B4" title="타이">타이</a> 남부의 주(짱왓)의 하나이다. 이웃한 주는 남쪽부터 시계 방향으로 <a href="/wiki/%EB%82%98%EB%9D%BC%ED%8B%B0%EC%99%93_%EC%A3%BC" title="나라티왓 주">나라티왓 주</a>, <a href="/wiki/%EC%96%84%EB%9D%BC_%EC%A3%BC" title="얄라 주">얄라 주</a>, <a href="/wiki/%EC%86%A1%ED%81%B4%EB%9D%BC_%EC%A3%BC" title="송클라 주">송클라 주</a>이다.</p>

<h2><span class="mw-headline" id=".EC.A7.80.EB.A6.AC">지리</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%B9%A0%EB%94%B0%EB%8B%88_%EC%A3%BC&amp;action=edit&amp;section=1" title="부분 편집: 지리">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>빠따니는 <a href="/wiki/%EB%A7%90%EB%A0%88%EC%9D%B4_%EB%B0%98%EB%8F%84" title="말레이 반도">말레이 반도</a>에 위치하고 북쪽은 <a href="/wiki/%ED%83%80%EC%9D%B4_%EB%A7%8C" title="타이 만">타이 만</a>의 해안이다. 남쪽은 산깔라키리 산맥의 산지이고, 부도-수-응아이 빠디 국립공원이 얄라 주, 나라티왓 주와의 경계에 위치해, 희귀한 식물들을 포함한 숲이 보호되고 있다. 송클라 주, 얄라 주와의 경계에 있는 남똑사이카오는 삼림 공원으로, 사이카오 폭포로 유명하다.</p>
<h2><span class="mw-headline" id=".EC.97.AD.EC.82.AC">역사</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%B9%A0%EB%94%B0%EB%8B%88_%EC%A3%BC&amp;action=edit&amp;section=2" title="부분 편집: 역사">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>빠따니라는 이름은 <a href="/wiki/%EB%B9%A0%EB%94%B0%EB%8B%88_%EB%A7%90%EB%A0%88%EC%9D%B4%EC%96%B4" title="빠따니 말레이어">빠따니 말레이어</a>로 '이 해변'을 의미하는 파타니(ڤتنا)에서 유래한 것이다.</p>
<p>역사적으로 빠따니 주는 반독립적인 <a class="mw-redirect" href="/wiki/%EB%A7%90%EB%A0%88%EC%9D%B4%EC%A1%B1" title="말레이족">말레이족</a>의 <a class="mw-redirect" href="/wiki/%EB%B9%A0%EB%94%B0%EB%8B%88_%EC%99%95%EA%B5%AD" title="빠따니 왕국">빠따니 왕국</a>의 중심지로서, 이들은 타이의 <a class="mw-redirect" href="/wiki/%EC%88%98%EC%BD%94%ED%83%80%EC%9D%B4_%EC%99%95%EA%B5%AD" title="수코타이 왕국">수코타이 왕국</a>과 <a class="mw-redirect" href="/wiki/%EC%95%84%EC%9C%A0%ED%83%80%EC%95%BC_%EC%99%95%EA%B5%AD" title="아유타야 왕국">아유타야 왕국</a>에 공물을 바쳤다. 1767년에 아유타야가 버마에 멸망한 후에 빠따니의 왕국은 완전한 독립을 얻었으나, <a href="/wiki/%EB%9D%BC%EB%A7%88_1%EC%84%B8" title="라마 1세">라마 1세</a>는 이들을 다시 시암의 지배하에 두었다.</p>
<p>1909년에 시암과 영국간의 방콕 조약으로 시암의 일부로 병합되었다. 얄라와 나라티왓은 원래 빠따니의 일부였으나 독립된 주가 되었다. 사뚠과 송클라는 거의 반독립적인 국가였다.</p>
<p>2004년 이후부터 빠따니 분리주의 운동이 일어나고 있다.</p>
<h2><span class="mw-headline" id=".EC.9D.B8.EA.B5.AC">인구</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%B9%A0%EB%94%B0%EB%8B%88_%EC%A3%BC&amp;action=edit&amp;section=3" title="부분 편집: 인구">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>빠따니는 <a class="mw-redirect" href="/wiki/%EB%A7%90%EB%A0%88%EC%9D%B4%EC%A1%B1" title="말레이족">말레이족</a> 무슬림이 인구의 다수인 타이의 4주의 하나로, 88%가 무슬림이다. 이들은 <a href="/wiki/%EB%B9%A0%EB%94%B0%EB%8B%88_%EB%A7%90%EB%A0%88%EC%9D%B4%EC%96%B4" title="빠따니 말레이어">빠따니 말레이어</a>를 사용한다. 빠따니 말레이족은 민족적, 문화적으로 <a href="/wiki/%EB%A7%90%EB%A0%88%EC%9D%B4%EC%8B%9C%EC%95%84" title="말레이시아">말레이시아</a> <a class="mw-redirect" href="/wiki/%EC%BC%88%EB%9E%80%ED%83%84_%EC%A3%BC" title="켈란탄 주">켈란탄 주</a>의 말레이족과 매우 유사하다.</p>
<h2><span class="mw-headline" id=".ED.96.89.EC.A0.95_.EA.B5.AC.EC.97.AD">행정 구역</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EB%B9%A0%EB%94%B0%EB%8B%88_%EC%A3%BC&amp;action=edit&amp;section=4" title="부분 편집: 행정 구역">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>주에는 12개의 <a href="/wiki/%EC%95%94%ED%94%84" title="암프">암프</a>(군)와 더 세부 행정구역인 115개의 <a href="/wiki/%EB%95%80%EB%B3%B8" title="땀본">땀본</a> 그리고 629개의 <a class="mw-redirect" href="/wiki/%EB%AC%B4%EB%B0%98" title="무반">무반</a>이 있다.</p>
