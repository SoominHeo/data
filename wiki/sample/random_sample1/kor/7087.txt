<p><b>삐야퐁 피우온</b>(<span style="font-size: smaller;"><a href="/wiki/%ED%83%9C%EA%B5%AD%EC%96%B4" title="태국어">태국어</a>:</span> <span lang="th" xml:lang="th">ปิยะพงษ์ ผิวอ่อน</span> <small>Piyapong Pue-on<sup>[<a class="mw-redirect" href="/wiki/%EC%9C%84%ED%82%A4%EB%B0%B1%EA%B3%BC:%ED%83%80%EC%9D%B4%EC%96%B4%EC%9D%98_%ED%95%9C%EA%B8%80_%ED%91%9C%EA%B8%B0" title="위키백과:타이어의 한글 표기">*</a>]</sup></small>, <a href="/wiki/1959%EB%85%84" title="1959년">1959년</a> <a href="/wiki/11%EC%9B%94_14%EC%9D%BC" title="11월 14일">11월 14일</a> ~ )은 <a href="/wiki/%ED%83%80%EC%9D%B4" title="타이">타이</a>의 전 <a href="/wiki/%EC%B6%95%EA%B5%AC" title="축구">축구</a> 선수이며 현 지도자이다. 그는 타이에서 가장 성공한 축구 선수로서 타이의 축구 영웅이다. 그는 또한 <a href="/wiki/%EC%95%84%EB%B0%94%EC%8A%A4_%EC%98%A4%EB%B2%A0%EC%9D%B4%EB%93%9C_%EC%9E%90%EC%8B%AC" title="아바스 오베이드 자심">아바스 오베이드 자심</a>, <a href="/wiki/%EC%84%B8%EB%A5%B4%EB%B2%A0%EB%A5%B4_%EC%A0%9C%ED%8C%8C%EB%A1%9C%ED%94%84" title="세르베르 제파로프">세르베르 제파로프</a>와 더불어 <a href="/wiki/K%EB%A6%AC%EA%B7%B8" title="K리그">K리그</a>에서 가장 성공한 <a href="/wiki/%EC%95%84%EC%8B%9C%EC%95%84" title="아시아">아시아</a> 출신 선수이기도 하다. <a href="/wiki/K%EB%A6%AC%EA%B7%B8" title="K리그">K리그</a>에 등록된 공식 한글 이름은 <b>피아퐁</b>이다.</p>

<h2><span class="mw-headline" id=".ED.81.B4.EB.9F.BD_.EA.B2.BD.EB.A0.A5">클럽 경력</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%82%90%EC%95%BC%ED%90%81_%ED%94%BC%EC%9A%B0%EC%98%A8&amp;action=edit&amp;section=1" title="부분 편집: 클럽 경력">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p><a href="/wiki/FC_%EC%84%9C%EC%9A%B8" title="FC 서울">럭키금성 황소</a> (현 <a href="/wiki/FC_%EC%84%9C%EC%9A%B8" title="FC 서울">FC 서울</a>)의 창단 멤버로 1983년 10월부터 영입이 추진되어<sup class="reference" id="cite_ref-1"><a href="#cite_note-1">[1]</a></sup> <a href="/wiki/1984%EB%85%84" title="1984년">1984년</a> 5월 31일 입단에 합의하였고<sup class="reference" id="cite_ref-2"><a href="#cite_note-2">[2]</a></sup>당시 공군상사였던 피아퐁의 군문제 등 태국에서 신변 정리 후 1984년 8월 29일 럭키금성에 합류하였다.<sup class="reference" id="cite_ref-3"><a href="#cite_note-3">[3]</a></sup></p>
<p><a href="/wiki/K%EB%A6%AC%EA%B7%B8_1984" title="K리그 1984">1984시즌</a> 9월 8일 <a href="/wiki/%ED%8F%AC%ED%95%AD_%EC%8A%A4%ED%8B%B8%EB%9F%AC%EC%8A%A4" title="포항 스틸러스">포항제철 돌핀스</a>전에 출장하여 데뷔골을 잡아내며 K리그 데뷔전을 화력하게 장식하였으며 이 경기 포함 그 후 4경기 연속골을 잡아내며 일대 센세이션을 일으켰다<sup class="reference" id="cite_ref-4"><a href="#cite_note-4">[4]</a></sup><sup class="reference" id="cite_ref-5"><a href="#cite_note-5">[5]</a></sup></p>
<p><a href="/wiki/K%EB%A6%AC%EA%B7%B8_1985" title="K리그 1985">1985시즌</a> <a href="/wiki/FC_%EC%84%9C%EC%9A%B8" title="FC 서울">럭키금성 황소</a>의 첫 번째 <a href="/wiki/K%EB%A6%AC%EA%B7%B8" title="K리그">K리그</a> 우승에 큰 공헌을 하였으며 그 해 <a href="/wiki/K%EB%A6%AC%EA%B7%B8_%EB%93%9D%EC%A0%90%EC%83%81" title="K리그 득점상">득점상</a>과 <a href="/wiki/K%EB%A6%AC%EA%B7%B8_%EB%8F%84%EC%9B%80%EC%83%81" title="K리그 도움상">도움상</a>을 최초로 동시 수상한 선수가 되며 최고 전성기를 맞았으며 <a href="/wiki/%ED%94%84%EB%A1%9C%EC%B6%95%EA%B5%AC%EC%84%A0%EC%88%98%EA%B6%8C%EB%8C%80%ED%9A%8C" title="프로축구선수권대회">1986년 프로축구선수권대회</a> 7월 2일 <a href="/wiki/%EC%9A%B8%EC%82%B0_%ED%98%84%EB%8C%80_%EC%B6%95%EA%B5%AC%EB%8B%A8" title="울산 현대 축구단">현대 호랑이</a>전을 마지막으로 2년간의 계약 기간을 마치고 팀을 떠났다.</p>
<p>1987년 5월 <a href="/wiki/FC_%EC%84%9C%EC%9A%B8" title="FC 서울">럭키금성 황소</a>로 재입단이 추진되기도 하였다.<sup class="reference" id="cite_ref-6"><a href="#cite_note-6">[6]</a></sup></p>
<p>덧붙여 피아퐁을 스카우트했던 <a href="/wiki/FC_%EC%84%9C%EC%9A%B8" title="FC 서울">럭키금성 황소</a>의 모기업인 <a class="mw-redirect" href="/wiki/%EB%9F%AD%ED%82%A4%EA%B8%88%EC%84%B1%EA%B7%B8%EB%A3%B9" title="럭키금성그룹">럭키금성그룹</a> 또한 태국에서 피아퐁의 인기 후광에 힘입어 방콕 최대 백화점인 소보 백화점의 엘리베이터, 에스컬레이터 수주를 따내기도 하였다.<sup class="reference" id="cite_ref-7"><a href="#cite_note-7">[7]</a></sup></p>
<h2><span class="mw-headline" id=".EA.B5.AD.EA.B0.80.EB.8C.80.ED.91.9C_.EA.B2.BD.EB.A0.A5">국가대표 경력</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%82%90%EC%95%BC%ED%90%81_%ED%94%BC%EC%9A%B0%EC%98%A8&amp;action=edit&amp;section=2" title="부분 편집: 국가대표 경력">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p><a href="/wiki/%ED%83%80%EC%9D%B4_%EC%B6%95%EA%B5%AC_%EA%B5%AD%EA%B0%80%EB%8C%80%ED%91%9C%ED%8C%80" title="타이 축구 국가대표팀">타이 축구 국가대표팀</a>에서 129경기를 뛰었으며 특히 <a href="/wiki/1997%EB%85%84" title="1997년">1997년</a> <a href="/wiki/3%EC%9B%94_2%EC%9D%BC" title="3월 2일">3월 2일</a> <a class="mw-redirect" href="/wiki/1998%EB%85%84_%ED%94%84%EB%9E%91%EC%8A%A4_%EC%9B%94%EB%93%9C%EC%BB%B5" title="1998년 프랑스 월드컵">1998년 프랑스 월드컵</a> 아시아 지역 예선에서 <a href="/wiki/%EB%8C%80%ED%95%9C%EB%AF%BC%EA%B5%AD_%EC%B6%95%EA%B5%AC_%EA%B5%AD%EA%B0%80%EB%8C%80%ED%91%9C%ED%8C%80" title="대한민국 축구 국가대표팀">대한민국</a>과의 경기에 출전하여 골을 터트리며 노익장을 과시했다.</p>
<h2><span class="mw-headline" id=".EC.A7.80.EB.8F.84.EC.9E.90_.EA.B2.BD.EB.A0.A5">지도자 경력</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%82%90%EC%95%BC%ED%90%81_%ED%94%BC%EC%9A%B0%EC%98%A8&amp;action=edit&amp;section=3" title="부분 편집: 지도자 경력">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>타이 공군 <a href="/wiki/%EB%8C%80%EC%9C%84" title="대위">대위</a>의 신분으로 <a class="mw-redirect" href="/wiki/%EC%97%90%EC%96%B4_%ED%8F%AC%EC%8A%A4_%EC%9C%A0%EB%82%98%EC%9D%B4%ED%8B%B0%EB%93%9C_FC" title="에어 포스 유나이티드 FC">타이 왕실 공군 FC</a>의 감독과 <a href="/wiki/%ED%83%80%EC%9D%B4_%EC%B6%95%EA%B5%AC_%EA%B5%AD%EA%B0%80%EB%8C%80%ED%91%9C%ED%8C%80" title="타이 축구 국가대표팀">타이 U-18 국가대표팀</a> 코치를 겸했다.</p>
<h2><span class="mw-headline" id=".EA.B7.B8_.EC.99.B8">그 외</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%82%90%EC%95%BC%ED%90%81_%ED%94%BC%EC%9A%B0%EC%98%A8&amp;action=edit&amp;section=4" title="부분 편집: 그 외">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>그는 <a href="/wiki/2004%EB%85%84" title="2004년">2004년</a>에 태국 영화 <a class="new" href="/w/index.php?title=%EB%B3%B8_%ED%88%AC_%ED%8C%8C%EC%9D%B4%ED%8A%B8&amp;action=edit&amp;redlink=1" title="본 투 파이트 (없는 문서)">본 투 파이트</a>에 출연했다.</p>
<p>그는 <a href="/wiki/2007%EB%85%84" title="2007년">2007년</a> <a href="/wiki/K%EB%A6%AC%EA%B7%B8" title="K리그">K리그</a> 득점왕의 모임인 <a class="new" href="/w/index.php?title=%ED%99%A9%EA%B8%88%EB%B0%9C_%ED%81%B4%EB%9F%BD&amp;action=edit&amp;redlink=1" title="황금발 클럽 (없는 문서)">황금발 클럽</a>의 초청으로 21년 만에 은퇴한 외국인 선수로는 처음으로 올스타전에 초청되어 대한민국을 방문하였으며<sup class="reference" id="cite_ref-8"><a href="#cite_note-8">[8]</a></sup> 친정팀인 <a href="/wiki/FC_%EC%84%9C%EC%9A%B8" title="FC 서울">FC 서울</a>의 8월 1일 FA컵 16강전 홈경기 행사에도 참석하였다.<sup class="reference" id="cite_ref-9"><a href="#cite_note-9">[9]</a></sup></p>
<p>그는 <a href="/wiki/K%EB%A6%AC%EA%B7%B8" title="K리그">K리그</a>에서 활약할 당시 현재 축구 감독으로 활동하고 있는 <a href="/wiki/%EB%B0%95%ED%95%AD%EC%84%9C" title="박항서">박항서</a>와 절친한 사이였으며 <a href="/wiki/2008%EB%85%84" title="2008년">2008년</a> <a href="/wiki/%EC%A0%84%EB%82%A8_%EB%93%9C%EB%9E%98%EA%B3%A4%EC%A6%88" title="전남 드래곤즈">전남 드래곤즈</a>가 <a href="/wiki/AFC_%EC%B1%94%ED%94%BC%EC%96%B8%EC%8A%A4%EB%A6%AC%EA%B7%B8" title="AFC 챔피언스리그">AFC 챔피언스리그</a>에 출전할 당시 <a href="/wiki/%EB%B0%95%ED%95%AD%EC%84%9C" title="박항서">박항서</a>가 <a href="/wiki/%ED%83%80%EC%9D%B4" title="타이">타이</a> 축구팀인 <a href="/wiki/%EC%B4%8C%EB%B6%80%EB%A6%AC_FC" title="촌부리 FC">촌부리 FC</a>의 전력을 문의하기도 하였다.<sup class="reference" id="cite_ref-10"><a href="#cite_note-10">[10]</a></sup><sup class="reference" id="cite_ref-11"><a href="#cite_note-11">[11]</a></sup></p>
<p>그의 아들인 <a class="new" href="/w/index.php?title=%EB%BD%95%EC%82%90%EC%88%AB_%EC%81%98%EC%98%A8&amp;action=edit&amp;redlink=1" title="뽕삐숫 쁘온 (없는 문서)">뽕삐숫 쁘온</a>(Pongpisut Pue-on) 또한 축구 선수로 활동하였으며 <a href="/wiki/2004%EB%85%84" title="2004년">2004년</a> 국내 에이전트와 계약을 맺고 <a href="/wiki/K%EB%A6%AC%EA%B7%B8" title="K리그">K리그</a> 진출을 모색하기도 하였다.<sup class="reference" id="cite_ref-12"><a href="#cite_note-12">[12]</a></sup> 그러나 뽕삐숫은 이후 부상으로 축구 선수의 꿈을 접고 태국의 유명 리포터로 활약하고 있다.</p>
<p>2010년 10월 13일 아들 뽕삐숫 쁘온과 함께 친정팀 <a href="/wiki/FC_%EC%84%9C%EC%9A%B8" title="FC 서울">FC 서울</a>의 훈련장인 <a href="/wiki/GS%EC%B1%94%ED%94%BC%EC%96%B8%EC%8A%A4%ED%8C%8C%ED%81%AC" title="GS챔피언스파크">GS챔피언스파크</a>를 방문하여 당시 <a class="mw-redirect" href="/wiki/%EB%84%AC%EB%A1%9C_%EB%B9%99%EA%B0%80%EB%8B%A4" title="넬로 빙가다">넬로 빙가다</a> 감독 및 선수들과 상견례를 갖고 훈련 시설을 둘러보며 친정팀에서 즐거운 시간을 가졌다.<sup class="reference" id="cite_ref-13"><a href="#cite_note-13">[13]</a></sup></p>
