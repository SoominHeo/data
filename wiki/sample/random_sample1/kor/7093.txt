<p><b>사나다 노부유키</b>(真田信之, <a href="/wiki/1566%EB%85%84" title="1566년">1566년</a> ~ <a href="/wiki/1658%EB%85%84" title="1658년">1658년</a>)는 <a href="/wiki/%EC%9D%BC%EB%B3%B8" title="일본">일본</a> <a href="/wiki/%EC%84%BC%EA%B3%A0%EC%BF%A0_%EC%8B%9C%EB%8C%80" title="센고쿠 시대">센고쿠 시대</a> 말기에서 <a href="/wiki/%EC%97%90%EB%8F%84_%EC%8B%9C%EB%8C%80" title="에도 시대">에도 시대</a> 초기에 활동한 <a href="/wiki/%EB%AC%B4%EC%9E%A5" title="무장">무장</a>이다. <a href="/wiki/%EC%82%AC%EB%82%98%EB%8B%A4_%EB%A7%88%EC%82%AC%EC%9C%A0%ED%82%A4" title="사나다 마사유키">사나다 마사유키</a>(真田昌幸)의 아들로, 사나다 가문을 대표하여 <a href="/wiki/%EB%8F%84%EC%BF%A0%EA%B0%80%EC%99%80_%EC%9D%B4%EC%97%90%EC%95%BC%EC%8A%A4" title="도쿠가와 이에야스">도쿠가와 이에야스</a>(德川家康)에 협력하였다. 끝까지 이에야스와 대결하다 <a href="/wiki/%EC%98%A4%EC%82%AC%EC%B9%B4_%EC%A0%84%ED%88%AC" title="오사카 전투">오사카 여름 전투</a> 에서 전사한 동생 <a href="/wiki/%EC%82%AC%EB%82%98%EB%8B%A4_%EB%85%B8%EB%B6%80%EC%8B%9C%EA%B2%8C" title="사나다 노부시게">사나다 유키무라</a>(真田幸村; 사나다 노부시게(真田信繁)를 말함)와는 대조적인 모습을 보여준다.</p>

<h2><span class="mw-headline" id=".EC.83.9D.EC.95.A0">생애</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%82%AC%EB%82%98%EB%8B%A4_%EB%85%B8%EB%B6%80%EC%9C%A0%ED%82%A4&amp;action=edit&amp;section=1" title="부분 편집: 생애">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>아버지 마사유키가 <a href="/wiki/%EA%B0%80%EC%9D%B4_%EA%B5%AD" title="가이 국">가이</a>의 <a href="/wiki/%EB%8B%A4%EC%BC%80%EB%8B%A4_%EC%94%A8" title="다케다 씨">다케다 가문</a>의 신하였기 때문에, 노부유키는 다케다 가문에 보내져 인질 생활을 보내게 된다. <a href="/wiki/1582%EB%85%84" title="1582년">1582년</a> 다케다 가문이 <a href="/wiki/%EC%98%A4%EB%8B%A4_%EB%85%B8%EB%B6%80%EB%82%98%EA%B0%80" title="오다 노부나가">오다 노부나가</a>(織田信長)와 <a href="/wiki/%EB%8F%84%EC%BF%A0%EA%B0%80%EC%99%80_%EC%9D%B4%EC%97%90%EC%95%BC%EC%8A%A4" title="도쿠가와 이에야스">도쿠가와 이에야스</a> 연합군에 의해 멸망하자 아버지와 함께 피신했다. <a href="/wiki/1585%EB%85%84" title="1585년">1585년</a> 도쿠가와 이에야스는 마사유키에게 본거지인 <a class="new" href="/w/index.php?title=%EC%9A%B0%EC%97%90%EB%8B%A4_%EC%84%B1&amp;action=edit&amp;redlink=1" title="우에다 성 (없는 문서)">우에다 성</a>을 내줄 것을 요구했으나 마사유키는 이를 거절했고, 이에 격분한 도쿠가와 군이 쳐들어오자 노부유키는 아버지와 함께 도쿠가와 군과 싸워 이를 격퇴했다(<a class="new" href="/w/index.php?title=%EC%A0%9C1%EC%B0%A8_%EC%9A%B0%EC%97%90%EB%8B%A4_%EC%84%B1_%EC%A0%84%ED%88%AC&amp;action=edit&amp;redlink=1" title="제1차 우에다 성 전투 (없는 문서)">제1차 우에다 성 전투</a>). 그 후 마사유키는 <a href="/wiki/%EB%8F%84%EC%9A%94%ED%86%A0%EB%AF%B8_%ED%9E%88%EB%8D%B0%EC%9A%94%EC%8B%9C" title="도요토미 히데요시">도요토미 히데요시</a>(豐臣秀吉)의 신하가 되었다. 히데요시의 명령으로 <a href="/wiki/1589%EB%85%84" title="1589년">1589년</a> 이에야스와 화친을 맺고 이에야스 휘하의 가신이 되었다. 이에야스는 노부유키의 재능을 높이 평가하여, 도쿠가와 가문의 중신인 <a href="/wiki/%ED%98%BC%EB%8B%A4_%EB%8B%A4%EB%8B%A4%EC%B9%B4%EC%93%B0" title="혼다 다다카쓰">혼다 다다카쓰</a>(本多忠勝)의 딸인 <a class="new" href="/w/index.php?title=%EC%9D%B4%EB%82%98%ED%9E%88%EB%A9%94&amp;action=edit&amp;redlink=1" title="이나히메 (없는 문서)">이나히메</a>(稲姫) 자신의 장녀로 삼은 뒤 노부유키에게 시집보냈다.</p>
<p><a href="/wiki/1590%EB%85%84" title="1590년">1590년</a> 도요토미 히데요시는 좀처럼 자신에게 복종하려 하지 않는 <a class="mw-redirect" href="/wiki/%EA%B3%A0%ED%98%B8%EC%A1%B0_%EC%94%A8" title="고호조 씨">호조 가문</a>을 토벌하기 위해 대군을 일으켰고, 노부유키는 <a class="new" href="/w/index.php?title=%EC%98%A4%EB%8B%A4%EC%99%80%EB%9D%BC_%EC%A0%95%EB%B2%8C&amp;action=edit&amp;redlink=1" title="오다와라 정벌 (없는 문서)">오다와라 정벌</a>에서 공을 세워 <a class="new" href="/w/index.php?title=%EB%88%84%EB%A7%88%ED%83%80_%EC%84%B1&amp;action=edit&amp;redlink=1" title="누마타 성 (없는 문서)">누마타 성</a>의 성주가 된다. <a href="/wiki/1594%EB%85%84" title="1594년">1594년</a> <a href="/wiki/%EC%9D%B4%EC%A6%88_%EA%B5%AD" title="이즈 국">이즈</a>노카미라는 관직을 받는다. 히데요시가 조선을 침략하자(<a href="/wiki/%EC%9E%84%EC%A7%84%EC%99%9C%EB%9E%80" title="임진왜란">임진왜란</a>, <a href="/wiki/%EC%A0%95%EC%9C%A0%EC%9E%AC%EB%9E%80" title="정유재란">정유재란</a>), 노부유키는 당시 침략군의 본영이 있던 규슈의 <a class="mw-redirect" href="/wiki/%EB%82%98%EA%B3%A0%EC%95%BC_%EC%84%B1" title="나고야 성">나고야 성</a>에 갔다. 히데요시가 죽자 도요토미 정권 내부의 권력 다툼으로 <a href="/wiki/%EC%98%A4%EB%8C%80%EB%A1%9C" title="오대로">오대로</a>의 필두였던 도쿠가와 이에야스와 <a href="/wiki/%EC%98%A4%EB%B4%89%ED%96%89" title="오봉행">오봉행</a>의 필두 <a href="/wiki/%EC%9D%B4%EC%8B%9C%EB%8B%A4_%EB%AF%B8%EC%93%B0%EB%82%98%EB%A6%AC" title="이시다 미쓰나리">이시다 미쓰나리</a>(石田三成) 사이의 대립이 격화되었고, 미쓰나리가 거병하여 이에야스 군을 토벌하자는 격문을 보내자 사나다 가문도 동군과 서군 사이에서 선택의 기로에 놓이게 되었다. 아버지 마사유키와 동생 노부시게는 미쓰나리의 서군에 가세했지만, 노부유키는 이에야스의 동군에 참가하기로 하여, <a href="/wiki/%EB%8F%84%EC%BF%A0%EA%B0%80%EC%99%80_%ED%9E%88%EB%8D%B0%ED%83%80%EB%8B%A4" title="도쿠가와 히데타다">도쿠가와 히데타다</a>(德川秀忠)의 군에 소속되어 <a class="new" href="/w/index.php?title=%EC%A0%9C2%EC%B0%A8_%EC%9A%B0%EC%97%90%EB%8B%A4_%EC%84%B1_%EC%A0%84%ED%88%AC&amp;action=edit&amp;redlink=1" title="제2차 우에다 성 전투 (없는 문서)">제2차 우에다 성 전투</a>에 참가하여 아버지와 동생과 적이 되어 싸웠다. 마사유키가 이끄는 2천의 사나다 군을 상대로 3만 8천의 히데타다 군은 지나치게 시간을 소비하여, 정작 <a href="/wiki/%EC%84%B8%ED%82%A4%EA%B0%80%ED%95%98%EB%9D%BC_%EC%A0%84%ED%88%AC" title="세키가하라 전투">세키가하라 전투</a>에 기일을 맞추지 못하여 결전이 끝난 후에야 도착했다. 세키가하라 전투 후에 노부유키는 마사유키의 영지였던 우에다와 누마타의 9만 5천 석을 계승하였다. 노부유키는 서군에 속해 있었던 마사유키와 노부시게의 구명을 탄원하여, 마사유키와 노부시게는 사형 대신 <a href="/wiki/%EA%B8%B0%EC%9D%B4_%EA%B5%AD" title="기이 국">기이</a> <a class="new" href="/w/index.php?title=%EA%B5%AC%EB%8F%84_%EC%82%B0&amp;action=edit&amp;redlink=1" title="구도 산 (없는 문서)">구도 산</a>에 유배되는 형벌로 그쳤다. 그 대신 노부유키(信幸)는 아버지 마사유키(昌幸)와의 결별을 선언하기 위해 노부유키(信之)로 이름을 바꾸었다. <a href="/wiki/%EC%98%A4%EC%82%AC%EC%B9%B4_%EC%A0%84%ED%88%AC" title="오사카 전투">오사카 전투</a>에서도 활약하여, <a href="/wiki/1622%EB%85%84" title="1622년">1622년</a> 13만 석의 마쓰시로로 영지를 옮겼다.</p>
<p>장남인 <a class="new" href="/w/index.php?title=%EC%82%AC%EB%82%98%EB%8B%A4_%EB%85%B8%EB%B6%80%EC%9A%94%EC%8B%9C&amp;action=edit&amp;redlink=1" title="사나다 노부요시 (없는 문서)">노부요시</a>는 이미 죽었기 때문에, 1656년에 차남 <a class="new" href="/w/index.php?title=%EC%82%AC%EB%82%98%EB%8B%A4_%EB%85%B8%EB%B6%80%EB%A7%88%EC%82%AC&amp;action=edit&amp;redlink=1" title="사나다 노부마사 (없는 문서)">노부마사</a>에게 가독(가문의 우두머리로서의 권리)을 넘겨주었지만, 1658년에 노부마사가 죽고, 노부마사의 아들인 <a class="new" href="/w/index.php?title=%EC%82%AC%EB%82%98%EB%8B%A4_%EC%9C%A0%ED%82%A4%EB%AF%B8%EC%B9%98&amp;action=edit&amp;redlink=1" title="사나다 유키미치 (없는 문서)">유키미치</a>가 가독을 계승한다. 노부유키도 이 해에 죽는다. 향년 93세.</p>
<h2><span class="mw-headline" id=".EC.82.AC.ED.9B.84.EC.9D.98_.EC.82.AC.EB.82.98.EB.8B.A4_.EA.B0.80.EB.AC.B8">사후의 사나다 가문</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%82%AC%EB%82%98%EB%8B%A4_%EB%85%B8%EB%B6%80%EC%9C%A0%ED%82%A4&amp;action=edit&amp;section=2" title="부분 편집: 사후의 사나다 가문">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>마쓰시로 번 제6대 번주 <a class="new" href="/w/index.php?title=%EC%82%AC%EB%82%98%EB%8B%A4_%EC%9C%A0%ED%82%A4%ED%9E%88%EB%A1%9C&amp;action=edit&amp;redlink=1" title="사나다 유키히로 (없는 문서)">유키히로</a>(幸弘)대에 <a class="new" href="/w/index.php?title=%EC%82%AC%EB%82%98%EB%8B%A4_%EC%9C%A0%ED%82%A4%ED%83%80%EC%B9%B4_(1770%EB%85%84)&amp;action=edit&amp;redlink=1" title="사나다 유키타카 (1770년) (없는 문서)">사나다 유키타카</a>(真田幸専, 친부 <a href="/wiki/%EC%97%90%EB%8F%84_%EB%A7%89%EB%B6%80" title="에도 막부">에도 막부</a> <a class="mw-redirect" href="/wiki/%EB%8B%A4%EC%9D%B4%EB%A1%9C" title="다이로">다이로</a> <a href="/wiki/%EC%9D%B4%EC%9D%B4_%EB%82%98%EC%98%A4%ED%9E%88%EB%8D%B0" title="이이 나오히데">이이 나오히데</a>井伊直幸의 9남)가 사나다 가문의 양자로서 가독을 이어 노부유키의 혈통이 끊기기는 했지만, 사나다 가문은 <a href="/wiki/%EB%A7%89%EB%A7%90" title="막말">막말</a>까지 지속되었어 <a class="new" href="/w/index.php?title=%EC%82%AC%EB%82%98%EB%8B%A4_%EC%9C%A0%ED%82%A4%EB%AA%A8%ED%86%A0&amp;action=edit&amp;redlink=1" title="사나다 유키모토 (없는 문서)">사나다 유키모토</a>(真田幸民, 친부 <a href="/wiki/%EC%9D%B4%EC%9A%94_%EA%B5%AD" title="이요 국">이요</a> <a class="new" href="/w/index.php?title=%EC%9A%B0%EC%99%80%EC%A7%80%EB%A7%88_%EB%B2%88&amp;action=edit&amp;redlink=1" title="우와지마 번 (없는 문서)">우와지마 번</a> 제8대 번주 <a href="/wiki/%EB%8B%A4%ED%85%8C_%EB%AC%B4%EB%84%A4%EB%82%98%EB%A6%AC" title="다테 무네나리">다테 무네나리</a>伊達宗城의 장남)대에는 <a href="/wiki/%EB%A9%94%EC%9D%B4%EC%A7%80_%EC%8B%9C%EB%8C%80" title="메이지 시대">메이지 시대</a>에도 <a href="/wiki/%EC%9E%90%EC%9E%91" title="자작">자작</a>(子爵) 가문으로 존속했을 뿐 아니라 후에 <a href="/wiki/%EB%B0%B1%EC%9E%91" title="백작">백작</a>(伯爵) 가문으로 격상된다.</p>
<h2><span class="mw-headline" id=".EC.9D.B8.EB.AC.BC">인물</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%82%AC%EB%82%98%EB%8B%A4_%EB%85%B8%EB%B6%80%EC%9C%A0%ED%82%A4&amp;action=edit&amp;section=3" title="부분 편집: 인물">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>동생인 <a href="/wiki/%EC%82%AC%EB%82%98%EB%8B%A4_%EB%85%B8%EB%B6%80%EC%8B%9C%EA%B2%8C" title="사나다 노부시게">사나다 노부시게</a>가 전설로까지 추앙된 것에 비하면 노부유키의 존재감은 크지 않은 것처럼 느껴진다. 더욱이 노부유키와 노부시게의 관계를 '어리석은 형과 현명한 동생'의 구도로 여기는 사람도 있다.</p>
<p>그러나 혼란의 <a href="/wiki/%EC%84%BC%EA%B3%A0%EC%BF%A0_%EC%8B%9C%EB%8C%80" title="센고쿠 시대">센고쿠 시대</a> 말기를 주군인 <a href="/wiki/%EB%8F%84%EC%BF%A0%EA%B0%80%EC%99%80_%EC%9D%B4%EC%97%90%EC%95%BC%EC%8A%A4" title="도쿠가와 이에야스">이에야스</a>와 아버지 <a href="/wiki/%EC%82%AC%EB%82%98%EB%8B%A4_%EB%A7%88%EC%82%AC%EC%9C%A0%ED%82%A4" title="사나다 마사유키">마사유키</a> 사이에서 가문을 혼란을 잠재우고, 그 탁월한 정치력과 생명력으로 '사나다'라는 이름을 지켜낸 것은 노부유키의 역량에 의한 것이며, 그 공적은 동생 노부시게의 화려한 무용담에도 뒤지지 않는 것이다. '정략(政略)으로 가문을 지킨다'는 점에서 보면, 노부유키는 아버지 마사유키의 피를 가장 진하게 이어받은 인물이라고도 할 수 있다. 그러나 노부시게의 존재 때문에 곧잘 과소평가되곤 하는 인물이다.</p>
