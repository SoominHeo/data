<h1 class="firstHeading" id="firstHeading" lang="ko">사냥개자리 알파²형 변광성</h1>
<div class="mw-body-content" id="bodyContent">
<div id="siteSub">위키백과, 우리 모두의 백과사전.</div>
<div id="contentSub"></div>
<div class="mw-jump" id="jump-to-nav">
					이동:					<a href="#mw-head">둘러보기</a>, 					<a href="#p-search">검색</a>
</div>
<div class="mw-content-ltr" dir="ltr" id="mw-content-text" lang="ko"><script>function mfTempOpenSection(id){var block=document.getElementById("mf-section-"+id);block.className+=" open-block";block.previousSibling.className+=" open-block";}</script><p><b>사냥개자리 알파²형 변광성</b>(<span lang="en" xml:lang="en">Alpha² Canum Venaticorum variable</span>)은 <a href="/wiki/%EB%B3%80%EA%B4%91%EC%84%B1" title="변광성">변광성</a>의 일종이다. 화학적으로 <a href="/wiki/%ED%8A%B9%EC%9D%B4%EB%B3%84" title="특이별">특이</a>한 <a class="mw-redirect" href="/wiki/%EC%A3%BC%EA%B3%84%EC%97%B4%EC%84%B1" title="주계열성">주계열성</a>으로 <a class="mw-redirect" href="/wiki/%EB%B6%84%EA%B4%91%ED%98%95" title="분광형">분광형</a>은 B8p애서 A7p 사이이다. <a href="/wiki/%EC%9E%90%EA%B8%B0%EC%9E%A5" title="자기장">자기장</a>이 강하며 <a href="/wiki/%EA%B7%9C%EC%86%8C" title="규소">규소</a>, <a href="/wiki/%EC%8A%A4%ED%8A%B8%EB%A1%A0%ED%8A%AC" title="스트론튬">스트론튬</a>, <a class="mw-redirect" href="/wiki/%ED%81%AC%EB%A1%AC" title="크롬">크롬</a>의 분광선이 강하게 나타난다. 밝기는 대개 0.01에서 0.1 수준으로 변동하며 변광주기는 0.5일에서 160일 정도이다.<sup class="reference" id="cite_ref-1"><a href="#cite_note-1">[1]</a></sup></p>
<p>사냥개자리 알파²형 변광성의 밝기가 변할 때 분광선의 세기와 종류, 그리고 자기장 세기도 함께 변한다. 이 물리량들의 변동 주기는 모두 동일하며, 이는 항성의 <a href="/wiki/%EC%9E%90%EC%A0%84" title="자전">자전</a> 주기와 동일할 것으로 추측된다. 변광의 이유는 항성의 <a href="/wiki/%EB%8C%80%EA%B8%B0" title="대기">대기</a> 중에 금속이 불균일하게 분포하여 각 면마다 표면밝기가 달라져서일 것이라고 추측된다.<sup class="reference" id="cite_ref-2"><a href="#cite_note-2">[2]</a></sup></p>
<p>사냥개자리 알파²형 변광성이라는 용어는 <a href="/wiki/%EC%82%AC%EB%83%A5%EA%B0%9C%EC%9E%90%EB%A6%AC" title="사냥개자리">사냥개자리</a> 북쪽 <a href="/wiki/%EC%BD%94%EB%A5%B4_%EC%B9%B4%EB%A1%A4%EB%A6%AC" title="코르 카롤리">코르 카롤리</a> 쌍성계를 이루는 동반성인 <a class="new" href="/w/index.php?title=%EC%82%AC%EB%83%A5%EA%B0%9C%EC%9E%90%EB%A6%AC_%EC%95%8C%ED%8C%8C%C2%B2&amp;action=edit&amp;redlink=1" title="사냥개자리 알파² (없는 문서)">사냥개자리 알파²</a>로부터 유래했다. 사냥개자리 알파²의 밝기는 5.47일 주기로 0.14 등급씩 변한다.<sup class="reference" id="cite_ref-3"><a href="#cite_note-3">[3]</a></sup></p>
