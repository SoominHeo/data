<p>《<b>사랑따윈 필요없어</b>》(Love me not)는 <a href="/wiki/2006%EB%85%84" title="2006년">2006년</a>에 개봉한 대한민국의 멜러 장르 영화이다. <a href="/wiki/%EC%9D%B4%EC%B2%A0%ED%95%98" title="이철하">이철하</a> 감독의 작품으로 <a href="/wiki/%EA%B9%80%EC%A3%BC%ED%98%81_(%EB%B0%B0%EC%9A%B0)" title="김주혁 (배우)">김주혁</a>, <a href="/wiki/%EB%AC%B8%EA%B7%BC%EC%98%81" title="문근영">문근영</a> 주연이다. 2002년 일본 드라마 <a href="/wiki/%EC%82%AC%EB%9E%91%EB%94%B0%EC%9C%88_%ED%95%84%EC%9A%94%EC%97%86%EC%96%B4,_%EC%97%AC%EB%A6%84" title="사랑따윈 필요없어, 여름">사랑따윈 필요없어, 여름</a>을 리메이크한 작품이다. <a href="/wiki/2006%EB%85%84" title="2006년">2006년</a> <a href="/wiki/11%EC%9B%94_9%EC%9D%BC" title="11월 9일">11월 9일</a> 개봉하였다.</p>
<p>같은 날짜에 개봉한 《<a href="/wiki/%EC%97%B4%ED%98%88%EB%82%A8%EC%95%84_(2006%EB%85%84_%EC%98%81%ED%99%94)" title="열혈남아 (2006년 영화)">열혈남아</a>》가 같은 제작사에서 제작된 영화라 논란이 일기도 하였다.<sup class="reference" id="cite_ref-1"><a href="#cite_note-1">[1]</a></sup><sup class="reference" id="cite_ref-2"><a href="#cite_note-2">[2]</a></sup></p>

<ul>
<li>류민 역의 <a href="/wiki/%EB%AC%B8%EA%B7%BC%EC%98%81" title="문근영">문근영</a>이 MBC 일일극 <a href="/wiki/%EC%82%AC%EB%9E%91%EC%9D%80_%EC%95%84%EB%AC%B4%EB%8F%84_%EB%AA%BB%EB%A7%90%EB%A0%A4" title="사랑은 아무도 못말려">사랑은 아무도 못말려</a> 주연 제의를 거절하고 선택한 영화인데 <a href="/wiki/%EB%AC%B8%EA%B7%BC%EC%98%81" title="문근영">문근영</a>이 주연으로 나온 &lt;사랑따윈 필요없어&gt;는 약 55만 명의 관객을 동원하는 데 그쳐 흥행에 실패했다.</li>
<li>결국 <a href="/wiki/%EC%82%AC%EB%9E%91%EC%9D%80_%EC%95%84%EB%AC%B4%EB%8F%84_%EB%AA%BB%EB%A7%90%EB%A0%A4" title="사랑은 아무도 못말려">사랑은 아무도 못말려</a>는 <a href="/wiki/%EB%AC%B8%EA%B7%BC%EC%98%81" title="문근영">문근영</a> 대신 당시 조연급에 그친 <a href="/wiki/%EC%9D%B4%EC%98%81%EC%95%84_(%EB%B0%B0%EC%9A%B0)" title="이영아 (배우)">이영아</a>가 대타로 들어갔는데 방영 내내 KBS 1TV 일일극 <a href="/wiki/%EB%B3%84%EB%82%9C%EC%97%AC%EC%9E%90_%EB%B3%84%EB%82%9C%EB%82%A8%EC%9E%90" title="별난여자 별난남자">별난여자 별난남자</a>(이하 '별녀별남') 때문에 시청률 면에서 고전했다.</li>
<li>아울러, &lt;별녀별남&gt; 종영 후 시청률이 2배 가까이 상승했으나 이 작품의 후속작인 <a href="/wiki/%EC%97%B4%EC%95%84%ED%99%89_%EC%88%9C%EC%A0%95_(%EB%93%9C%EB%9D%BC%EB%A7%88)" title="열아홉 순정 (드라마)">열아홉 순정</a>에도 줄곧 시청률이 뒤지는 수모를 당했지만 아무런 화제도 남기지 못한 데 이어 &lt;별녀별남&gt; 탓인지 한자릿수 시청률에 머무른 채 130부작에서 63회로 조기종영된 전작 <a href="/wiki/%EB%A7%A8%EB%B0%9C%EC%9D%98_%EC%B2%AD%EC%B6%98_(2005%EB%85%84_%EB%93%9C%EB%9D%BC%EB%A7%88)" title="맨발의 청춘 (2005년 드라마)">맨발의 청춘</a>과는 달리 처음 주연을 맡았던 <a href="/wiki/%EC%9D%B4%EC%98%81%EC%95%84_(%EB%B0%B0%EC%9A%B0)" title="이영아 (배우)">이영아</a>를 스타덤에 올려놨으며, <a href="/wiki/%EC%9D%B4%EC%98%81%EC%95%84_(%EB%B0%B0%EC%9A%B0)" title="이영아 (배우)">이영아</a>의 상대역이자 가수에서 연기자로 변신을 시도한 <a href="/wiki/%ED%99%8D%EA%B2%BD%EB%AF%BC" title="홍경민">홍경민</a>의 가능성을 입증한 바 있었는데 마지막회에서 자체 최고 시청률인 16.9%를 거뒀다.</li>
</ul>
