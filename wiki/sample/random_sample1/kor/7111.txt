<p>《<b>사랑을 위한 여행</b>》(<span style="font-size: smaller;"><a href="/wiki/%EC%98%81%EC%96%B4" title="영어">영어</a>:</span> <span lang="en" xml:lang="en">The Yellow Handkerchief</span>)은 <a href="/wiki/2008%EB%85%84" title="2008년">2008년</a>에 공개된 <a class="mw-redirect" href="/wiki/%EB%AF%B8%EA%B5%AD_%EC%98%81%ED%99%94" title="미국 영화">미국</a>의 <a href="/wiki/%EB%8F%85%EB%A6%BD_%EC%98%81%ED%99%94" title="독립 영화">독립</a> <a href="/wiki/%EB%93%9C%EB%9D%BC%EB%A7%88_%EC%98%81%ED%99%94" title="드라마 영화">드라마 영화</a>이다. 이 영화는 <a href="/wiki/1977%EB%85%84" title="1977년">1977년</a>에 개봉 된 <a href="/wiki/%EC%95%BC%EB%A7%88%EB%8B%A4_%EC%9A%94%EC%A7%80" title="야마다 요지">야마다 요지</a> 감독의 동명의 영화(<span style="font-size: smaller;"><a href="/wiki/%EC%9D%BC%EB%B3%B8%EC%96%B4" title="일본어">일본어</a>:</span> <span lang="ja" xml:lang="ja">幸福の黄色いハンカチ</span>)를 리메이크했다.<sup class="reference" id="cite_ref-1"><a href="#cite_note-1">[1]</a></sup> 현대 시대의 <a href="/wiki/%EB%AF%B8%EA%B5%AD_%EB%82%A8%EB%B6%80" title="미국 남부">미국 남부</a>를 배경으로, 복역 후, 떠나야만 했던 자신의 연인을 찾아 소년, 소녀와 여행을 함께하는 '브렛' 역할에는 <a class="mw-redirect" href="/wiki/%EC%9C%8C%EB%A6%AC%EC%95%94_%ED%97%88%ED%8A%B8" title="윌리암 허트">윌리암 허트</a>, 그의 연인 '메이' 역할에는 <a href="/wiki/%EB%A7%88%EB%A6%AC%EC%95%84_%EB%B2%A8%EB%A1%9C" title="마리아 벨로">마리아 벨로</a>, 소녀 '마틴' 역할에는 <a href="/wiki/%ED%81%AC%EB%A6%AC%EC%8A%A4%ED%8B%B4_%EC%8A%A4%ED%8A%9C%EC%96%B4%ED%8A%B8" title="크리스틴 스튜어트">크리스틴 스튜어트</a>, 소녀의 제안에 얼떨결에 여행을 떠나게 되는 소년 '고디' 역할에는 <a href="/wiki/%EC%97%90%EB%94%94_%EB%A0%88%EB%93%9C%EB%A9%94%EC%9D%B8" title="에디 레드메인">에디 레드메인</a>이 출연했다.<sup class="reference" id="cite_ref-FA01_2-0"><a href="#cite_note-FA01-2">[2]</a></sup></p>
<p>감독은 <a class="new" href="/w/index.php?title=%EC%9A%B0%EB%8B%A4%EC%96%80_%ED%94%84%EB%9D%BC%EC%82%AC%EB%93%9C&amp;action=edit&amp;redlink=1" title="우다얀 프라사드 (없는 문서)">우다얀 프라사드</a>, 제작에는 아서 콘이 참여해 2008년 선댄스 영화제에서 첫 상영 되었고, <a href="/wiki/2010%EB%85%84" title="2010년">2010년</a> <a href="/wiki/2%EC%9B%94_26%EC%9D%BC" title="2월 26일">2월 26일</a> 새뮤얼 골드윈 필름스 배급으로 미국 전역에 개봉 되었다.</p>
