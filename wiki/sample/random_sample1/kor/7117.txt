<p><b>사례금</b>(謝禮金) 또는 <b>팁</b>(<span lang="en" xml:lang="en">tip</span>)은 <a href="/wiki/%EC%A0%9C3%EC%B0%A8_%EC%82%B0%EC%97%85" title="제3차 산업">제3차 산업</a>에서 일하는 사람들에게 정해진 가격보다 더 올려서 주는 웃돈을 말한다. <a href="/wiki/%EC%9D%8C%EC%8B%9D%EC%A0%90" title="음식점">음식점</a>, <a href="/wiki/%ED%98%B8%ED%85%94" title="호텔">호텔</a> 등에서 일하는 <a class="mw-redirect" href="/wiki/%EC%A2%85%EC%97%85%EC%9B%90" title="종업원">종업원</a> 등에게 손님이 요금 이외로 주는 돈, <a class="new" href="/w/index.php?title=%EB%B4%89%EC%82%AC%EB%A3%8C&amp;action=edit&amp;redlink=1" title="봉사료 (없는 문서)">봉사료</a> 의 뜻이다.</p>

<h2><span class="mw-headline" id=".EC.82.AC.EB.A1.80.EA.B8.88.EC.9D.98_.EC.97.AD.EC.82.AC_.EB.B0.8F_.EA.B8.B0.EC.9B.90">사례금의 역사 및 기원</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%82%AC%EB%A1%80%EA%B8%88&amp;action=edit&amp;section=1" title="부분 편집: 사례금의 역사 및 기원">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>사례금, 곧 팁(tip)이라는 것은 "받은 서비스에 대한 감사의 마음을 표현하는 것"이라는 의미로서 18세기 <a href="/wiki/%EC%98%81%EA%B5%AD" title="영국">영국</a>의 한 술집에서부터 <a class="new" href="/w/index.php?title=%EC%9C%A0%EB%9E%98&amp;action=edit&amp;redlink=1" title="유래 (없는 문서)">유래</a>되었다. 좋은 <a href="/wiki/%EC%84%9C%EB%B9%84%EC%8A%A4" title="서비스">서비스</a>와 신속하며 빠른 서비스를 원한다면 돈을 더 지불하라는 "<i>To Insure Promptness</i>"라는 말이 술집벽에 적혀 있었고 이것의 첫 머리를 따서 사람들이 팁(Tip)이라는 말을 만들어졌다는 <a href="/wiki/%EC%9D%BC%ED%99%94" title="일화">일화</a>가 있다.</p>
<p>우리가 알고 있듯이, 현재 세계에서 가장 보편적인 사례금 <a href="/wiki/%EB%AC%B8%ED%99%94" title="문화">문화</a>가 이루어지는 나라는 <a href="/wiki/%EB%AF%B8%EA%B5%AD" title="미국">미국</a>이며, <a href="/wiki/%EB%AF%B8%EA%B5%AD" title="미국">미국</a>의 사례금 문화는 <a href="/wiki/%EC%8B%9D%EB%AF%BC%EC%A7%80" title="식민지">식민지</a> 시절 <a href="/wiki/%EC%98%81%EA%B5%AD" title="영국">영국</a>으로부터 사례금 문화를 전수 받았다고 알려져 있으며, 현재 서비스의 대가를 지불하는 의미로 쓰이고 있지만, 전 세계적으로 서비스에 대한 감사의 표시를 넘어서 통상적인 관례가 되어가고 있다.</p>
<h2><span class="mw-headline" id=".ED.95.84.EC.9A.94.EC.84.B1">필요성</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%82%AC%EB%A1%80%EA%B8%88&amp;action=edit&amp;section=2" title="부분 편집: 필요성">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>사례금에 대한 관념의 바탕은 "감사"이므로 개인적으로 무엇을 부탁했을 경우나 <a class="new" href="/w/index.php?title=%EC%8B%A0%EC%84%B8&amp;action=edit&amp;redlink=1" title="신세 (없는 문서)">신세</a>를 지면 지불해야 하는 대가이다. 비록 사례금은 강제적인 것은 아니지만 <a href="/wiki/%EB%AF%B8%EA%B5%AD" title="미국">미국</a>의 서비스 업계종사자들에게는 생계 수단이며, 주 수입원이 될 수 있으므로 사례금을 주는 것은 아주 중요한 부분이다.</p>
<h3><span class="mw-headline" id=".EB.8C.80.ED.95.9C.EB.AF.BC.EA.B5.AD"><a href="/wiki/%EB%8C%80%ED%95%9C%EB%AF%BC%EA%B5%AD" title="대한민국">대한민국</a></span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%82%AC%EB%A1%80%EA%B8%88&amp;action=edit&amp;section=3" title="부분 편집: 대한민국">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p><a href="/wiki/%EB%8C%80%ED%95%9C%EB%AF%BC%EA%B5%AD" title="대한민국">대한민국</a>에서는 <a class="new" href="/w/index.php?title=%EB%AC%BC%ED%92%88&amp;action=edit&amp;redlink=1" title="물품 (없는 문서)">물품</a> 및 <a href="/wiki/%EC%84%9C%EB%B9%84%EC%8A%A4" title="서비스">서비스</a> 가격에 사례금이 포함되어 있기 때문에, 따로 주지 않는다.</p>
<h3><span class="mw-headline" id=".EC.A4.91.EA.B5.AD"><a href="/wiki/%EC%A4%91%EA%B5%AD" title="중국">중국</a></span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%82%AC%EB%A1%80%EA%B8%88&amp;action=edit&amp;section=4" title="부분 편집: 중국">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p><a href="/wiki/%EC%A4%91%EA%B5%AD" title="중국">중국</a>에서도 이러한 사례금이 사실상 없으나, <a href="/wiki/%ED%98%B8%ED%85%94" title="호텔">호텔</a>의 해외 관광객들에게는 사례금을 허용한다.</p>
<h3><span class="mw-headline" id=".EC.9D.BC.EB.B3.B8"><a href="/wiki/%EC%9D%BC%EB%B3%B8" title="일본">일본</a></span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%82%AC%EB%A1%80%EA%B8%88&amp;action=edit&amp;section=5" title="부분 편집: 일본">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p><a href="/wiki/%EC%9D%BC%EB%B3%B8" title="일본">일본</a>에서도 특별한 <a href="/wiki/%EC%84%9C%EB%B9%84%EC%8A%A4" title="서비스">서비스</a>를 제외하고는 사례금 문화가 일상화되어 있지 않다.</p>
<h3><span class="mw-headline" id=".EC.9C.A0.EB.9F.BD"><a href="/wiki/%EC%9C%A0%EB%9F%BD" title="유럽">유럽</a></span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%82%AC%EB%A1%80%EA%B8%88&amp;action=edit&amp;section=6" title="부분 편집: 유럽">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p><a href="/wiki/%EC%9C%A0%EB%9F%BD" title="유럽">유럽</a>에서 역시 <a href="/wiki/%EB%AF%B8%EA%B5%AD" title="미국">미국</a>처럼 사례금을 주는 문화가 관례화되어 있다. 특히 <a href="/wiki/%ED%98%B8%ED%85%94" title="호텔">호텔</a>에서는 1<a href="/wiki/%EC%9C%A0%EB%A1%9C" title="유로">유로</a> 정도를 베개나 탁자 등에 두고 가는 것이 기본 <a class="mw-redirect" href="/wiki/%EC%97%90%ED%8B%B0%EC%BC%93" title="에티켓">에티켓</a>이며, 식사 후 <a href="/wiki/%EC%8B%9D%EB%8B%B9" title="식당">레스토랑</a>에서는 식비의 15 퍼센트 정도를 주는 것이 통상적이다.</p>
<h2><span class="mw-headline" id=".EC.9E.A5.EC.86.8C.EC.97.90_.EB.94.B0.EB.A5.B8_.EC.82.AC.EB.A1.80.EA.B8.88_.EC.A7.80.EB.B6.88_.EC.A0.95.EB.8F.84">장소에 따른 사례금 지불 정도</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%82%AC%EB%A1%80%EA%B8%88&amp;action=edit&amp;section=7" title="부분 편집: 장소에 따른 사례금 지불 정도">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<h3><span class="mw-headline" id=".EC.8B.9D.EB.8B.B9">식당</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%82%AC%EB%A1%80%EA%B8%88&amp;action=edit&amp;section=8" title="부분 편집: 식당">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<ul>
<li><a href="/wiki/%EC%8B%9D%EB%8B%B9" title="식당">레스토랑</a>의 <a href="/wiki/%EC%9B%A8%EC%9D%B4%ED%84%B0" title="웨이터">웨이트리스</a>와 <a href="/wiki/%EC%9B%A8%EC%9D%B4%ED%84%B0" title="웨이터">웨이터</a>는 대부분 사례금으로 생활하며 가게 주인으로부터 급료를 받지 않는다. 사례금은 보통 지불해야 하는 <i>음식값의 10-20%</i>이다. 사례금의 액수는 <a href="/wiki/%EC%84%9C%EB%B9%84%EC%8A%A4" title="서비스">서비스</a>의 좋고 나쁨에 따라서 결정한다. <a href="/wiki/%EC%84%9C%EB%B9%84%EC%8A%A4" title="서비스">서비스</a>가 아주 나쁜 경우는 전혀 안 주기도 하며 <a href="/wiki/%EC%84%9C%EB%B9%84%EC%8A%A4" title="서비스">서비스</a>가 매우 좋을 경우는 30%를 지불하기도 한다.</li>
<li><a href="/wiki/%ED%8F%AC%EB%8F%84%EC%A3%BC" title="포도주">와인</a> <a href="/wiki/%EC%9B%A8%EC%9D%B4%ED%84%B0" title="웨이터">웨이터</a>가 있을 경우 와인 값의 10-15%를 사례금으로 직접 건네준다. 또 테이블까지 안내한 <a href="/wiki/%EC%9B%A8%EC%9D%B4%ED%84%B0" title="웨이터">웨이터</a>에게는 대개 사례금을 주지 않지만 사람의 수가 많아 특별히 테이블을 마련해 주거나 <a href="/wiki/%EC%B0%A8%EB%A6%BC%ED%91%9C" title="차림표">메뉴</a>에 없는 요리를 특별히 만들도록 배려해 준 경우는 2-3 달러를 사례금으로 건넨다.</li>
<li><a href="/wiki/%EB%82%98%EC%9D%B4%ED%8A%B8%ED%81%B4%EB%9F%BD" title="나이트클럽">나이트클럽</a>, <a class="mw-redirect" href="/wiki/%EC%8A%A4%ED%83%A0%EB%93%9C%EB%B0%94" title="스탠드바">바</a>(bar) 등은 보통 15%의 사례금을 준다.</li>
<li><a href="/wiki/%EC%BB%A4%ED%94%BC%EC%A7%91" title="커피집">커피집</a>이나 <a href="/wiki/%EC%8A%A4%EB%82%B5%EB%B0%94" title="스낵바">스낵바</a>에서 <a href="/wiki/%EC%BB%A4%ED%94%BC" title="커피">커피</a>를 한 잔 마셨을 경우 사례금을 지불하지 않는 사람도 있으나, 지불할 경우 50센트 정도 하면 된다.</li>
</ul>
<h3><span class="mw-headline" id=".EC.84.9C.EB.B9.84.EC.8A.A4_.EC.97.85">서비스 업</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%82%AC%EB%A1%80%EA%B8%88&amp;action=edit&amp;section=9" title="부분 편집: 서비스 업">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p><a href="/wiki/%EC%9D%B4%EB%B0%9C%EC%86%8C" title="이발소">이발소</a>, <a href="/wiki/%EB%AF%B8%EC%9A%A9%EC%8B%A4" title="미용실">미용실</a> : 보통 <a href="/wiki/%EC%9D%B4%EB%B0%9C%EC%86%8C" title="이발소">이발소</a>에서 한 사람에게 머리를 자르고, <a href="/wiki/%EB%A9%B4%EB%8F%84" title="면도">면도</a>, <a href="/wiki/%EC%83%B4%ED%91%B8" title="샴푸">샴푸</a>를 하는 경우 그 사람에게 15-20%의 사례금을 준다. 그러나 일류 <a href="/wiki/%ED%98%B8%ED%85%94" title="호텔">호텔</a> 내의 <a class="new" href="/w/index.php?title=%EA%B3%A0%EA%B8%89_%EC%9D%B4%EB%B0%9C%EC%86%8C&amp;action=edit&amp;redlink=1" title="고급 이발소 (없는 문서)">고급 이발소</a>, <a href="/wiki/%EB%AF%B8%EC%9A%A9%EC%8B%A4" title="미용실">미용실</a>에서는 나누어서 일을 하므로 각각의 사람에게 따로따로 10-15%의 사례금을 주어야 한다.</p>
<h3><span class="mw-headline" id=".EC.88.99.EC.86.8C">숙소</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%82%AC%EB%A1%80%EA%B8%88&amp;action=edit&amp;section=10" title="부분 편집: 숙소">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p><a href="/wiki/%ED%98%B8%ED%85%94" title="호텔">호텔</a>: 입구에 서있는 <a class="mw-disambig" href="/wiki/%EC%88%98%EC%9C%84" title="수위">수위</a>에게는 짐이 있든지 없든지 1달러 정도를 주면 좋다. 다음에 짐을 나르는 <a class="new" href="/w/index.php?title=%EB%B2%A8_%EB%B3%B4%EC%9D%B4&amp;action=edit&amp;redlink=1" title="벨 보이 (없는 문서)">벨 보이</a>에게 보통 짐 한 개당 1달러 이상을 준다. 사례금을 <a class="mw-disambig" href="/wiki/%EC%88%98%EC%9C%84" title="수위">수위</a>, <a class="new" href="/w/index.php?title=%ED%94%84%EB%A1%A0%ED%8A%B8&amp;action=edit&amp;redlink=1" title="프론트 (없는 문서)">프론트</a>, <a class="new" href="/w/index.php?title=%EB%B2%A8_%EB%B3%B4%EC%9D%B4&amp;action=edit&amp;redlink=1" title="벨 보이 (없는 문서)">벨 보이</a>에게 주면 된다.</p>
<h3><span class="mw-headline" id=".EA.B8.B0.ED.83.80">기타</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%82%AC%EB%A1%80%EA%B8%88&amp;action=edit&amp;section=11" title="부분 편집: 기타">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>나중에 <a href="/wiki/%EA%B7%B9%EC%9E%A5" title="극장">극장</a>, <a href="/wiki/%EC%8B%9D%EB%8B%B9" title="식당">레스토랑</a>, <a href="/wiki/%EA%B4%80%EA%B4%91%EB%B2%84%EC%8A%A4" title="관광버스">관광버스</a> 예약이나 <a class="mw-redirect" href="/wiki/%EC%9A%B0%ED%8E%B8%EB%AC%BC" title="우편물">우편물</a> 보내는 일, <a href="/wiki/%ED%83%9D%EC%8B%9C" title="택시">택시</a> 부르는 일 등 여러 가지 서비스를 받을 수 있다. 방 청소를 하는 <a class="new" href="/w/index.php?title=%EB%A3%B8_%EB%A9%94%EC%9D%B4%EB%93%9C&amp;action=edit&amp;redlink=1" title="룸 메이드 (없는 문서)">룸 메이드</a>에게는 하루 밤 잘 때마다 매일 아침 <a href="/wiki/%EB%B2%A0%EA%B0%9C" title="베개">베개</a> 밑이나 <a class="new" href="/w/index.php?title=%EB%82%98%EC%9D%B4%ED%8A%B8_%EB%9E%A8%ED%94%84&amp;action=edit&amp;redlink=1" title="나이트 램프 (없는 문서)">나이트 램프</a> 밑에 놓아 둔다. 음식을 자기 방에서 들고 싶을 때에는 전화를 하면 배달해 주는데 그때 <a href="/wiki/%EC%9B%A8%EC%9D%B4%ED%84%B0" title="웨이터">웨이터</a>에게 주는 <a href="/wiki/%EB%A3%B8_%EC%84%9C%EB%B9%84%EC%8A%A4" title="룸 서비스">룸 서비스</a>를 위한 사례금은 10-20% 정도이다.</p>
<p>휴대품 보관소 (Cloak) : <a href="/wiki/%EB%AF%B8%EC%88%A0%EA%B4%80" title="미술관">미술관</a>, <a href="/wiki/%EC%98%81%ED%99%94%EA%B4%80" title="영화관">영화관</a>, <a href="/wiki/%EA%B7%B9%EC%9E%A5" title="극장">극장</a>, <a href="/wiki/%EC%8B%9D%EB%8B%B9" title="식당">레스토랑</a>, <a href="/wiki/%EB%82%98%EC%9D%B4%ED%8A%B8%ED%81%B4%EB%9F%BD" title="나이트클럽">나이트클럽</a> 등의 <a class="new" href="/w/index.php?title=%ED%9C%B4%EB%8C%80%ED%92%88_%EB%B3%B4%EA%B4%80%EC%86%8C&amp;action=edit&amp;redlink=1" title="휴대품 보관소 (없는 문서)">휴대품 보관소</a>에서는 <a class="mw-disambig" href="/wiki/%EC%BD%94%ED%8A%B8" title="코트">코트</a> 한 벌 당 50센트에서 1달러를 주며 <a class="mw-redirect" href="/wiki/%EC%84%B8%EB%A9%B4%EB%8C%80" title="세면대">세면대</a>가 있는 화장실에는 <a class="mw-redirect" href="/wiki/%ED%83%80%EC%9B%94" title="타월">타월</a> 한 장 당 25센트를 준다.</p>
<p>항공회사, 역 : 항공회사의 사원이나 역원, <a href="/wiki/%EB%B9%84%ED%96%89%EA%B8%B0" title="비행기">비행기</a> 안에서는 사례금을 주지 않는다. 그러나 <a href="/wiki/%EC%84%A0%EB%B0%95" title="선박">배</a>를 이용한 여행에서는 <a href="/wiki/%ED%98%B8%ED%85%94" title="호텔">호텔</a>에 준한 사례금을 주어야 한다.</p>
<h2><span class="mw-headline" id=".EC.82.AC.EB.A1.80.EA.B8.88.EC.9D.84_.EC.A3.BC.EC.A7.80_.EC.95.8A.EC.95.84.EB.8F.84_.EB.90.A0_.EA.B2.BD.EC.9A.B0">사례금을 주지 않아도 될 경우</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%82%AC%EB%A1%80%EA%B8%88&amp;action=edit&amp;section=12" title="부분 편집: 사례금을 주지 않아도 될 경우">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p><a href="/wiki/%EB%B2%84%EC%8A%A4" title="버스">버스</a> 운전기사, <a class="mw-redirect mw-disambig" href="/wiki/%EB%A7%88%EC%BC%93" title="마켓">마켓</a>의 점원, <a href="/wiki/%EB%B0%B1%ED%99%94%EC%A0%90" title="백화점">백화점</a>의 점원, <a class="mw-redirect" href="/wiki/%ED%8C%A8%EC%8A%A4%ED%8A%B8%ED%91%B8%EB%93%9C" title="패스트푸드">패스트푸드</a>의 점원, <a href="/wiki/%EC%8A%88%ED%8D%BC%EB%A7%88%EC%BC%93" title="슈퍼마켓">슈퍼마켓</a> 안에 있는 <a class="mw-disambig" href="/wiki/%EC%B9%B4%EC%9A%B4%ED%84%B0" title="카운터">카운터</a>에서 <a href="/wiki/%EC%83%8C%EB%93%9C%EC%9C%84%EC%B9%98" title="샌드위치">샌드위치</a>, <a class="new" href="/w/index.php?title=%EB%93%9C%EB%A7%81%ED%81%AC&amp;action=edit&amp;redlink=1" title="드링크 (없는 문서)">드링크</a> 류를 파는 사람, <a href="/wiki/%EC%8B%9D%EB%8B%B9" title="식당">레스토랑</a>에서 식기 등을 치우는 사람, 사무실의 청소부, 항공회사 직원, <a href="/wiki/%EC%A3%BC%EC%9C%A0%EC%86%8C" title="주유소">주유소</a>의 종업원, 극장, 영화관의 안내원, 세탁소 주인 등에게는 특별히 사례금을 주지 않아도 된다.</p>
