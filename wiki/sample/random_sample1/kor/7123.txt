<p><b>사리원시</b>(沙里院市)는 <a href="/wiki/%EC%A1%B0%EC%84%A0%EB%AF%BC%EC%A3%BC%EC%A3%BC%EC%9D%98%EC%9D%B8%EB%AF%BC%EA%B3%B5%ED%99%94%EA%B5%AD" title="조선민주주의인민공화국">조선민주주의인민공화국</a> <a href="/wiki/%ED%99%A9%ED%95%B4%EB%B6%81%EB%8F%84" title="황해북도">황해북도</a> 서부에 있는 시이자, 도 소재지이다. 시의 서부에는 <a href="/wiki/%EC%9E%AC%EB%A0%B9%EA%B0%95" title="재령강">재령강</a>이 북쪽으로 흐르며, <a href="/wiki/%EC%9E%AC%EB%A0%B9%ED%8F%89%EC%95%BC" title="재령평야">재령평야</a>의 동부에 위치한다. <a href="/wiki/%EA%B2%BD%EC%9D%98%EC%84%A0" title="경의선">경의선</a>과 <a href="/wiki/%ED%99%A9%ED%95%B4%EC%B2%AD%EB%85%84%EC%84%A0" title="황해청년선">황해청년선</a>이 교차하는 철도 교통의 요지로 도시가 성장하였다. <a href="/wiki/%EB%AC%B4%EC%97%B0%ED%83%84" title="무연탄">무연탄</a> 등의 자원이 풍부한 공업도시이다. 황해북도 인민위원회, 조선로동당 황해북도당위원회 등이 소재한다. 당 책임비서는 양명철이다.</p>

<p>황해북도의 서부에 위치한다. 원래는 <a href="/wiki/%EB%B4%89%EC%82%B0%EA%B5%B0" title="봉산군">봉산군</a>의 일부로, <a href="/wiki/%EA%B2%BD%EC%9D%98%EC%84%A0" title="경의선">경의선</a>을 따라서 발달한 도시이다. 북쪽과 동쪽은 봉산군에 둘러싸여있고 남쪽은 <a href="/wiki/%EC%9D%80%ED%8C%8C%EA%B5%B0" title="은파군">은파군</a>과 접한다. 서쪽은 <a href="/wiki/%ED%99%A9%ED%95%B4%EB%82%A8%EB%8F%84" title="황해남도">황해남도</a>이다. <a href="/wiki/%EC%9E%AC%EB%A0%B9%EA%B0%95" title="재령강">재령강</a> 연안에 위치해 있고 북쪽에서 남쪽으로 갈수록 <a href="/wiki/%ED%8F%89%EC%95%BC" title="평야">평야</a>에 가까워진다. 여러 산들이 솟아 있으며, 최고봉은 <a class="new" href="/w/index.php?title=%EA%B0%80%EB%A7%88%EB%B4%89&amp;action=edit&amp;redlink=1" title="가마봉 (없는 문서)">가마봉</a>이다. 사리원시의 산들은 400m미만의 산들이다. 하천들은 소규모로 흐른다. 시 면적의 21.9%를 삼림이 차지한다. <a href="/wiki/%EC%9A%B4%ED%95%98" title="운하">운하</a>와 <a href="/wiki/%ED%98%B8%EC%88%98" title="호수">호수</a>가 위치해 있다.</p>
<h2><span class="mw-headline" id=".EC.97.AD.EC.82.AC">역사</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%82%AC%EB%A6%AC%EC%9B%90%EC%8B%9C&amp;action=edit&amp;section=2" title="부분 편집: 역사">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>원래는 '사리'라는 지명이었지만, <a href="/wiki/%EC%A1%B0%EC%84%A0" title="조선">조선</a> 시대에 공인된 역참이 소재하는 토지를 원이라고 칭하게 되었기 때문에 사리원으로 불리게 되었다.</p>
<ul>
<li><a href="/wiki/1910%EB%85%84" title="1910년">1910년</a> 12월 - 봉산군의 군청이 놓여졌다.</li>
<li><a href="/wiki/1921%EB%85%84" title="1921년">1921년</a> 12월 - 봉산군의 면리를 합병해 사리원면이 생겼다.</li>
<li><a href="/wiki/1931%EB%85%84" title="1931년">1931년</a> <a href="/wiki/4%EC%9B%94_1%EC%9D%BC" title="4월 1일">4월 1일</a> 봉산군 사리원면이 사리원읍으로 승격하였다.<sup class="reference" id="cite_ref-1"><a href="#cite_note-1">[1]</a></sup></li>
<li><a href="/wiki/1954%EB%85%84" title="1954년">1954년</a> <a href="/wiki/10%EC%9B%94" title="10월">10월</a> 황해북도가 신설되면서 도 소재지로 승격됨.</li>
</ul>
<h2><span class="mw-headline" id=".ED.96.89.EC.A0.95_.EA.B5.AC.EC.97.AD">행정 구역</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%82%AC%EB%A6%AC%EC%9B%90%EC%8B%9C&amp;action=edit&amp;section=3" title="부분 편집: 행정 구역">편집</a><span class="mw-editsection-bracket">]</span></span></h2>

<h2><span class="mw-headline" id=".EA.B5.90.ED.86.B5">교통</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%82%AC%EB%A6%AC%EC%9B%90%EC%8B%9C&amp;action=edit&amp;section=4" title="부분 편집: 교통">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>사리원시는 <a href="/wiki/%ED%99%A9%ED%95%B4%EB%B6%81%EB%8F%84" title="황해북도">황해북도</a>의 교통, 산업, 사회문화의 중심지이다. 주요 철도 노선에는 <a href="/wiki/%ED%8F%89%EC%9D%98%EC%84%A0" title="평의선">평의선</a>과 <a href="/wiki/%ED%99%A9%ED%95%B4%EC%B2%AD%EB%85%84%EC%84%A0" title="황해청년선">황해청년선</a>이 있으며, <a href="/wiki/1992%EB%85%84" title="1992년">1992년</a>에 <a href="/wiki/%ED%8F%89%EC%96%91%EA%B0%9C%EC%84%B1%EA%B3%A0%EC%86%8D%EB%8F%84%EB%A1%9C" title="평양개성고속도로">평양개성고속도로</a>가 개통되어 교통 인프라가 잘 구축되어 있다.</p>
<h2><span class="mw-headline" id=".EA.B4.80.EA.B4.91.EA.B3.BC_.EC.82.B0.EC.97.85.EC.8B.9C.EC.84.A4">관광과 산업시설</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%82%AC%EB%A6%AC%EC%9B%90%EC%8B%9C&amp;action=edit&amp;section=5" title="부분 편집: 관광과 산업시설">편집</a><span class="mw-editsection-bracket">]</span></span></h2>

<p><b>사리원시</b>는 많은 <a class="mw-redirect" href="/wiki/%EC%84%9D%ED%9A%8C%EC%84%9D" title="석회석">석회석</a>이 매장되어 <a href="/wiki/%EC%8B%9C%EB%A9%98%ED%8A%B8" title="시멘트">시멘트</a>를 이용한 중공업이 잘 발달되어 있다. <a class="new" href="/w/index.php?title=2.8%EC%8B%9C%EB%A9%98%ED%8A%B8%EA%B3%B5%EC%9E%A5&amp;action=edit&amp;redlink=1" title="2.8시멘트공장 (없는 문서)">2.8시멘트공장</a> 이 가장 유명하며, 이곳의 시멘트는 매우 질이 좋아 해외로 수출한다.</p>
<p>한편, 관광업이 발달하였으며, 가장 유명한 곳은 <a class="new" href="/w/index.php?title=%EA%B2%BD%EC%95%94%EC%82%B0%EC%84%B1&amp;action=edit&amp;redlink=1" title="경암산성 (없는 문서)">경암산성</a>과 <a href="/wiki/%EC%A0%95%EB%B0%A9%EC%82%B0%EC%84%B1" title="정방산성">정방산성</a> 등이 있다. 경암산성은 사리원시에 남서쪽에 위치하였으며, 공원이 있다. 경암산성은 많은 외국인과 해외동포들이 찾는다.</p>
<p>최근 <a class="mw-disambig" href="/wiki/%EB%AF%B8%EA%B3%A1%EB%A6%AC" title="미곡리">미곡리</a>에는 주민들의 노력으로 문화회관 및, 다양한 모양의 주택이 만들어졌다. 김정은 조선로동당 위원장의 지시로 건설한 방송에 소개된 시립수영장이 2014년에 개관해, 오락시설들이 점점 늘어가고 있다.</p>
