<p><b>사마휘</b> (司馬徽, ? ~ <a href="/wiki/208%EB%85%84" title="208년">208년</a>)는 <a href="/wiki/%EC%A4%91%EA%B5%AD" title="중국">중국</a> <a href="/wiki/%ED%9B%84%ED%95%9C" title="후한">후한</a> 말의 인물로, <a href="/wiki/%EC%9E%90_(%EC%9D%B4%EB%A6%84)" title="자 (이름)">자</a>는 <b>덕조</b>(德操)이다.</p>
<h2><span class="mw-headline" id=".EC.83.9D.EC.95.A0">생애</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%82%AC%EB%A7%88%ED%9C%98&amp;action=edit&amp;section=1" title="부분 편집: 생애">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>그의 감정능력 및 인재발굴 능력에 특출했다고 전해진다. 또한, <a href="/wiki/%EC%9C%A0%EB%B9%84" title="유비">유비</a>가 <a href="/wiki/%EC%84%9C%EC%84%9C" title="서서">서서(徐庶)</a>가 <a href="/wiki/%EC%A1%B0%EC%A1%B0" title="조조">조조(曹操)</a>의 가짜 편지에 속아 <a href="/wiki/%EC%A1%B0%EC%A1%B0" title="조조">조조</a>에게 떠나며 <a href="/wiki/%EC%A0%9C%EA%B0%88%EB%9F%89" title="제갈량">제갈량</a>을 알려 주었다는 것을 이야기해 주었을 때 <a href="/wiki/%EC%84%9C%EC%84%9C" title="서서">서서</a>가 가지 않았으면 <a href="/wiki/%EC%84%9C%EC%84%9C" title="서서">서서</a>의 노모가 무사했을 거라며 한탄하기도 하였다.</p>
<p>실제로 그 후에 서서의 노모는 아들에 대한 실망감과 조조에 대한 분노로 자살했다고 전해진다. 일명 <b>수경선생</b>으로 알려져있다. 형주 자사 <a class="mw-redirect" href="/wiki/%EC%9C%A0%ED%91%9C_(%ED%9B%84%ED%95%9C)" title="유표 (후한)">유표</a>가 죽은 후 조조가 형주를 점령했을 때, 병으로 죽었다.</p>
<p>그의 제자로는 <a href="/wiki/%EC%84%9C%EC%84%9C" title="서서">서서</a>, <a href="/wiki/%EC%A0%9C%EA%B0%88%EB%9F%89" title="제갈량">제갈량</a>, <a href="/wiki/%EB%B0%A9%ED%86%B5" title="방통">방통</a>이 대표적이며, <a class="new" href="/w/index.php?title=%EC%B5%9C%EC%A3%BC%ED%8F%89&amp;action=edit&amp;redlink=1" title="최주평 (없는 문서)">최주평</a>, <a class="new" href="/w/index.php?title=%EC%84%9D%EB%8F%84_(%ED%9B%84%ED%95%9C)&amp;action=edit&amp;redlink=1" title="석도 (후한) (없는 문서)">석도</a>, <a class="new" href="/w/index.php?title=%EB%A7%B9%EA%B1%B4&amp;action=edit&amp;redlink=1" title="맹건 (없는 문서)">맹건</a> 등도 제자로 알려져있다. 그러나 이것은 모두 《<a href="/wiki/%EC%82%BC%EA%B5%AD%EC%A7%80%EC%97%B0%EC%9D%98" title="삼국지연의">삼국지연의</a>》상의 허구이며, 그들은 제자가 아닌 그저 그에게 인정받은 인재들이라고 한다. <a class="new" href="/w/index.php?title=%EC%9D%B4%EC%84%A0_(%EC%B4%89)&amp;action=edit&amp;redlink=1" title="이선 (촉) (없는 문서)">이선</a>과 <a href="/wiki/%EC%9C%A4%EB%AC%B5" title="윤묵">윤묵</a>은 형주 유학 시절에 사마휘, <a class="mw-redirect" href="/wiki/%EC%86%A1%EC%B6%A9_(%ED%9B%84%ED%95%9C)" title="송충 (후한)">송충</a> 등에게서 수학하였다.</p>
<h2><span class="mw-headline" id=".E3.80.8A.EC.82.BC.EA.B5.AD.EC.A7.80.EC.97.B0.EC.9D.98.E3.80.8B.EC.83.81.EC.9D.98_.EC.82.AC.EB.A7.88.ED.9C.98">《<a href="/wiki/%EC%82%BC%EA%B5%AD%EC%A7%80%EC%97%B0%EC%9D%98" title="삼국지연의">삼국지연의</a>》상의 사마휘</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%82%AC%EB%A7%88%ED%9C%98&amp;action=edit&amp;section=2" title="부분 편집: 《삼국지연의》상의 사마휘">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>삼국지연의 상에서 그는 누구의 말에도 항상 ‘좋다, 좋아!’라고 대답을 해서, ‘호호선생’이라는 별명을 얻는다.<sup class="reference" id="cite_ref-1"><a href="#cite_note-1">[1]</a></sup></p>
