<p><b>사모스 현</b>은 <a href="/wiki/%EA%B7%B8%EB%A6%AC%EC%8A%A4" title="그리스">그리스</a>의 <a href="/wiki/%EC%82%AC%EB%AA%A8%EC%8A%A4_%EC%84%AC" title="사모스 섬">사모스 섬</a>, <a class="new" href="/w/index.php?title=%EC%9D%B4%EC%B9%B4%EB%A6%AC%EC%95%84_%EC%84%AC&amp;action=edit&amp;redlink=1" title="이카리아 섬 (없는 문서)">이카리아 섬</a>과 <a class="new" href="/w/index.php?title=%ED%91%B8%EB%A5%B4%EB%8B%88_%EC%BD%94%EB%A5%B4%EC%84%B8%EC%98%A8&amp;action=edit&amp;redlink=1" title="푸르니 코르세온 (없는 문서)">푸르니 코르세온</a>의 여러 작은 섬으로 이루어진 <a href="/wiki/%EA%B7%B8%EB%A6%AC%EC%8A%A4%EC%9D%98_%ED%98%84" title="그리스의 현">현</a>이다. 중심 도시이자 주요 승선 항구는 <a href="/wiki/%EB%B0%94%ED%8B%B0" title="바티">바티</a>(혹은 '사모스'로도 불린다)이며, 사모스 섬의 다른 항구로는 섬에서 두 번째로 인구가 많은 <a class="new" href="/w/index.php?title=%EC%B9%B4%EB%A5%BC%EB%A1%9C%EB%B0%94%EC%8B%9C&amp;action=edit&amp;redlink=1" title="카를로바시 (없는 문서)">카를로바시</a>와 더불어, 고대부터 사모스 섬의 도읍이 있던 <a class="new" href="/w/index.php?title=%ED%94%BC%ED%83%80%EA%B3%A0%EB%A6%AC%EC%98%A8&amp;action=edit&amp;redlink=1" title="피타고리온 (없는 문서)">피타고리온</a>(20년대 중반까지는 '티가니'로 불렸으며 <a class="mw-redirect" href="/wiki/%EC%9C%A0%EB%84%A4%EC%8A%A4%EC%BD%94" title="유네스코">유네스코</a> <a class="mw-redirect" href="/wiki/%EC%84%B8%EA%B3%84%EB%AC%B8%ED%99%94%EC%9C%A0%EC%82%B0" title="세계문화유산">세계문화유산</a>으로 지정되었다)이 있다.</p>
