<p><b>사뮈엘 윔티티</b>(<span style="font-size: smaller;"><a href="/wiki/%ED%94%84%EB%9E%91%EC%8A%A4%EC%96%B4" title="프랑스어">프랑스어</a>:</span> <span lang="fr" xml:lang="fr">Samuel Umtiti</span>, 1993년 11월 14일, <a href="/wiki/%EC%95%BC%EC%9A%B4%EB%8D%B0" title="야운데">야운데</a> ~)는 <a href="/wiki/%ED%94%84%EB%9E%91%EC%8A%A4" title="프랑스">프랑스</a>의 <a href="/wiki/%EC%B6%95%EA%B5%AC" title="축구">축구</a> 선수로, 현재 스페인 <a href="/wiki/%EB%9D%BC%EB%A6%AC%EA%B0%80" title="라리가">라 리가</a>의 <a href="/wiki/FC_%EB%B0%94%EB%A5%B4%EC%85%80%EB%A1%9C%EB%82%98" title="FC 바르셀로나">바르셀로나</a> 소속이다. 그는 <a href="/wiki/%EC%88%98%EB%B9%84%EC%88%98" title="수비수">중앙</a>이나 <a href="/wiki/%EC%88%98%EB%B9%84%EC%88%98" title="수비수">좌측 수비수</a>로 기용 가능하다.<sup class="reference" id="cite_ref-2"><a href="#cite_note-2">[2]</a></sup> 움티티는 <a class="new" href="/w/index.php?title=%ED%94%84%EB%9E%91%EC%8A%A4_U-17_%EC%B6%95%EA%B5%AC_%EA%B5%AD%EA%B0%80%EB%8C%80%ED%91%9C%ED%8C%80&amp;action=edit&amp;redlink=1" title="프랑스 U-17 축구 국가대표팀 (없는 문서)">U-17</a>, <a class="new" href="/w/index.php?title=%ED%94%84%EB%9E%91%EC%8A%A4_U-18_%EC%B6%95%EA%B5%AC_%EA%B5%AD%EA%B0%80%EB%8C%80%ED%91%9C%ED%8C%80&amp;action=edit&amp;redlink=1" title="프랑스 U-18 축구 국가대표팀 (없는 문서)">U-18</a>, <a class="new" href="/w/index.php?title=%ED%94%84%EB%9E%91%EC%8A%A4_U-19_%EC%B6%95%EA%B5%AC_%EA%B5%AD%EA%B0%80%EB%8C%80%ED%91%9C%ED%8C%80&amp;action=edit&amp;redlink=1" title="프랑스 U-19 축구 국가대표팀 (없는 문서)">U-19</a> 연령대의 <a class="new" href="/w/index.php?title=%ED%94%84%EB%9E%91%EC%8A%A4_%EC%B2%AD%EC%86%8C%EB%85%84_%EC%B6%95%EA%B5%AC_%EA%B5%AD%EA%B0%80%EB%8C%80%ED%91%9C%ED%8C%80&amp;action=edit&amp;redlink=1" title="프랑스 청소년 축구 국가대표팀 (없는 문서)">청소년 국가대표팀</a>으로 차출된 적이 있다. 그는 <a class="new" href="/w/index.php?title=2010%EB%85%84_UEFA_U-17_%EC%B6%95%EA%B5%AC_%EC%84%A0%EC%88%98%EA%B6%8C_%EB%8C%80%ED%9A%8C&amp;action=edit&amp;redlink=1" title="2010년 UEFA U-17 축구 선수권 대회 (없는 문서)">2010년 UEFA U-17 축구 선수권 대회</a>에 참가한 U-17 대표팀의 일원이었다. 움티티는 2012년 1월 8일, 3-1로 이긴 <a class="new" href="/w/index.php?title=AS_%EB%A6%AC%EC%98%B9_%EB%92%A4%EC%85%B0%EB%A5%B4&amp;action=edit&amp;redlink=1" title="AS 리옹 뒤셰르 (없는 문서)">리옹 뒤셰르</a>와의 경기에서 프로 데뷔전을 치렀고, 선발로 나서서 끝가지 경기에 임하였다.<sup class="reference" id="cite_ref-3"><a href="#cite_note-3">[3]</a></sup> <a href="/wiki/%EC%A0%9C%EB%A0%88%EB%AF%B8_%EB%A7%88%ED%8B%B0%EC%99%B8" title="제레미 마티외">제레미 마티외</a>가 부상으로 낙마함에 따라 움티티는 <a href="/wiki/UEFA_%EC%9C%A0%EB%A1%9C_2016" title="UEFA 유로 2016">UEFA 유로 2016</a>에 참가할 데샹호에 승선하였다. <sup class="reference" id="cite_ref-Euro_4-0"><a href="#cite_note-Euro-4">[4]</a></sup></p>
