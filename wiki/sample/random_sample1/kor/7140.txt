<p><b>사반왕</b>(沙伴王, 생몰년 미상, <small>재위:</small> <a href="/wiki/234%EB%85%84" title="234년">234년</a> ~ <a href="/wiki/234%EB%85%84" title="234년">234년</a>)은 <a href="/wiki/%EB%B0%B1%EC%A0%9C" title="백제">백제</a>의 제7대 <a href="/wiki/%EA%B5%B0%EC%A3%BC" title="군주">국왕</a>이다. <b>사비왕</b>(沙沸王) 또는 <b>사이왕</b>(沙伊王)으로도 부른다. 선왕(先王) <a class="mw-redirect" href="/wiki/%EB%B0%B1%EC%A0%9C_%EA%B5%AC%EC%88%98%EC%99%95" title="백제 구수왕">구수왕</a>의 장자이다.</p>
<p><a href="/wiki/234%EB%85%84" title="234년">234년</a>에 잠시 <a href="/wiki/%EC%96%B4%EB%9D%BC%ED%95%98" title="어라하">백제 어라하</a>로 재위하는 동안 <a href="/wiki/%EC%99%95%EC%A1%B1" title="왕족">왕족 종실</a>인 <a class="mw-redirect" href="/wiki/%EB%B0%B1%EC%A0%9C_%EA%B3%A0%EC%9D%B4%EC%99%95" title="백제 고이왕">부여고이</a>(훗날 <a class="mw-redirect" href="/wiki/%EB%B0%B1%EC%A0%9C_%EA%B3%A0%EC%9D%B4%EC%99%95" title="백제 고이왕">백제 고이왕</a>)가 <a href="/wiki/%EC%84%AD%EC%A0%95" title="섭정">섭정</a>하였고, 재위한 지 1년도 채 안 돼서 그에게 보위 찬탈되었다. 왕으로서 기록된 것은 《삼국유사》이고, 정사인 《삼국사기》에서는 아예 왕 취급도 하지 않고 그냥 이름인 "사반"으로 지칭된다. 그가 죽고 70년만에 즉위한 <a href="/wiki/%EB%B9%84%EB%A5%98%EC%99%95" title="비류왕">비류왕</a>과의 관계에 대해서는 이견이 있다.</p>

<h2><span class="mw-headline" id=".EC.83.9D.EC.95.A0">생애</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%82%AC%EB%B0%98%EC%99%95&amp;action=edit&amp;section=1" title="부분 편집: 생애">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p><a href="/wiki/%EA%B5%AC%EC%88%98%EC%99%95" title="구수왕">구수왕</a>이 <a href="/wiki/234%EB%85%84" title="234년">234년</a>에 사망하자, 장남인 사반왕이 제7대 <a href="/wiki/%EB%B0%B1%EC%A0%9C" title="백제">백제</a> 임금이 되었다. 그러나 5대 <a href="/wiki/%EC%B4%88%EA%B3%A0%EC%99%95" title="초고왕">초고왕</a>의 동생 또는 동생의 후손이라는 설이 있는 <a href="/wiki/%EA%B3%A0%EC%9D%B4%EC%99%95" title="고이왕">고이왕</a>에게 나이가 어리다는 이유로 왕위를 빼앗겼다. 그 후의 행적에 대한 기록은 남아 있지 않다. 다만, 왕으로 즉위하고 얼마 안되어 고이왕에게 왕위를 빼앗긴 것으로 보아, 폐위 직후 살해당했을 것으로 추정된다.</p>
<p><a href="/wiki/%EC%82%BC%EA%B5%AD%EC%82%AC%EA%B8%B0" title="삼국사기">삼국사기</a>에 의하면 <a href="/wiki/234%EB%85%84" title="234년">234년</a> <a href="/wiki/%EA%B5%AC%EC%88%98%EC%99%95" title="구수왕">구수왕</a>이 죽자 사반이 맏아들로서 왕위에 올랐으나 나이가 어려 정사를 잘 처리하지 못하므로 제5대 <a href="/wiki/%EC%B4%88%EA%B3%A0%EC%99%95" title="초고왕">초고왕</a>의 동생 <a href="/wiki/%EA%B3%A0%EC%9D%B4%EC%99%95" title="고이왕">고이왕</a>이 즉위했다고 한다. 또 <a href="/wiki/%EC%82%BC%EA%B5%AD%EC%9C%A0%EC%82%AC" title="삼국유사">삼국유사</a>에는 사비왕(沙沸王) 또는 사이왕(沙伊王)으로 나오며 폐위되었다고 전하기도 한다. 사반왕이 폐위되고 <a href="/wiki/%EA%B3%A0%EC%9D%B4%EC%99%95" title="고이왕">고이왕</a>이 즉위한 것을 왕실 방계(傍系)의 성장에 의한 왕위교체로 파악하고 <a href="/wiki/%EA%B3%A0%EC%9D%B4%EC%99%95" title="고이왕">고이왕</a>을 지지한 진씨(眞氏) 세력이 사반왕을 지지한 해씨(解氏) 세력을 압도한 결과로 보는 견해도 있다.</p>
<h3><span class="mw-headline" id=".EB.B9.84.EB.A5.98.EC.99.95.EA.B3.BC.EC.9D.98_.EA.B4.80.EA.B3.84_.EB.85.BC.EB.9E.80">비류왕과의 관계 논란</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%82%AC%EB%B0%98%EC%99%95&amp;action=edit&amp;section=2" title="부분 편집: 비류왕과의 관계 논란">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p><a href="/wiki/%EC%82%BC%EA%B5%AD%EC%82%AC%EA%B8%B0" title="삼국사기">삼국사기</a>에 의하면 사반왕은 <a href="/wiki/%EA%B5%AC%EC%88%98%EC%99%95" title="구수왕">구수왕</a>의 아들이자 <a href="/wiki/%EB%B9%84%EB%A5%98%EC%99%95" title="비류왕">비류왕</a>의 형이다. 또, <a href="/wiki/%EB%B9%84%EB%A5%98%EC%99%95" title="비류왕">비류왕</a>은 평민으로 살면서 명성을 얻었다고도 한다. <a href="/wiki/%EB%B9%84%EB%A5%98%EC%99%95" title="비류왕">비류왕</a>과 <a href="/wiki/%EB%B9%84%EB%A5%98%EC%99%95" title="비류왕">비류왕</a>의 형 사반왕 사이에는 70년 이상의 연대 차이가 나므로 형제 여부에 관하여 이견이 있다. 《<a href="/wiki/%EC%82%BC%EA%B5%AD%EC%82%AC%EA%B8%B0" title="삼국사기">삼국사기</a>》에서 <a href="/wiki/%EB%B9%84%EB%A5%98%EC%99%95" title="비류왕">비류왕</a>의 아버지로 지목하는 <a class="mw-redirect" href="/wiki/%EB%B0%B1%EC%A0%9C_%EA%B5%AC%EC%88%98%EC%99%95" title="백제 구수왕">구수왕</a>과 형으로 지목한 사반왕이 물러난 뒤 70년만에 즉위하여 40년을 통치한다. 또한 《<a href="/wiki/%EC%82%BC%EA%B5%AD%EC%82%AC%EA%B8%B0" title="삼국사기">삼국사기</a>》에서는 <a href="/wiki/%EB%B9%84%EB%A5%98%EC%99%95" title="비류왕">비류왕</a>을 <a href="/wiki/%EA%B5%AC%EC%88%98%EC%99%95" title="구수왕">구수왕</a>의 아들이자 사반왕의 동생이라 기술하고도 같은 조의 다른 줄에는 평민으로 살면서 명성을 얻었다는 구절이 나타난다.</p>
