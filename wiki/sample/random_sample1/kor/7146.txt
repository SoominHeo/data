<p><b>사비나 미키나</b> (Səbinə Mikina, Sabina Mikina, <a href="/wiki/1987%EB%85%84" title="1987년">1987년</a> <a href="/wiki/10%EC%9B%94_24%EC%9D%BC" title="10월 24일">10월 24일</a>, <a href="/wiki/%EB%B0%94%EC%BF%A0" title="바쿠">바쿠</a> ~) 는 <a href="/wiki/%EC%95%84%EC%A0%9C%EB%A5%B4%EB%B0%94%EC%9D%B4%EC%9E%94" title="아제르바이잔">아제르바이잔</a>의 <a href="/wiki/%EC%82%AC%EB%B8%8C%EB%A5%B4_(%ED%8E%9C%EC%8B%B1)" title="사브르 (펜싱)">사브르</a> <a href="/wiki/%ED%8E%9C%EC%8B%B1" title="펜싱">펜싱</a> 선수이다.<sup class="reference" id="cite_ref-london-2012_1-0"><a href="#cite_note-london-2012-1">[1]</a></sup><sup class="reference" id="cite_ref-2"><a href="#cite_note-2">[2]</a></sup> 미키나는 <a href="/wiki/%EB%9F%B0%EB%8D%98" title="런던">런던</a>에서 열린 <a href="/wiki/2012%EB%85%84_%ED%95%98%EA%B3%84_%EC%98%AC%EB%A6%BC%ED%94%BD" title="2012년 하계 올림픽">2012년 하계 올림픽</a>의 <a class="mw-redirect" href="/wiki/2012%EB%85%84_%ED%95%98%EA%B3%84_%EC%98%AC%EB%A6%BC%ED%94%BD_%ED%8E%9C%EC%8B%B1_%EC%97%AC%EC%9E%90_%EC%82%AC%EB%B8%8C%EB%A5%B4_%EA%B0%9C%EC%9D%B8%EC%A0%84" title="2012년 하계 올림픽 펜싱 여자 사브르 개인전">여자 사브르 개인전</a>에 아제르바이잔을 대표로 출전하였다. 32강전에서 그녀는 헝가리 태생의 독일인 선수 <a href="/wiki/%EC%96%BC%EB%A0%89%EC%84%A0%EB%93%9C%EB%9F%AC_%EB%B6%80%EC%9D%B4%EB%8F%84%EC%86%8C" title="얼렉선드러 부이도소">얼렉선드러 부이도소</a>를 15-13으로 꺾었으나, 이어지는 16강전에서 우크라이나의 <a class="mw-redirect" href="/wiki/%EC%98%AC%EA%B0%80_%EC%B9%B4%EB%A5%BC%EB%9E%80" title="올가 카를란">올가 카를란</a>에게 10-15로 패하며 탈락하였다.<sup class="reference" id="cite_ref-3"><a href="#cite_note-3">[3]</a></sup></p>
