<p><b>사세보 해군 기지</b>는(<span lang="ja"><ruby><rb>佐世保基地</rb> <rp>(</rp><rt>させぼきち</rt><rp>)</rp></ruby></span> <small>사세보 키치<sup>[<a href="/wiki/%EC%9C%84%ED%82%A4%EB%B0%B1%EA%B3%BC:%EC%9D%BC%EB%B3%B8%EC%96%B4%EC%9D%98_%ED%95%9C%EA%B8%80_%ED%91%9C%EA%B8%B0" title="위키백과:일본어의 한글 표기">*</a>]</sup></small>, <span style="font-size: smaller;"><a href="/wiki/%EC%98%81%EC%96%B4" title="영어">영어</a>:</span> <span lang="en">United States Fleet Activities Sasebo</span>)는 <a href="/wiki/%EC%9D%BC%EB%B3%B8" title="일본">일본</a> <a href="/wiki/%EB%82%98%EA%B0%80%EC%82%AC%ED%82%A4_%ED%98%84" title="나가사키 현">나가사키 현</a> <a href="/wiki/%EC%82%AC%EC%84%B8%EB%B3%B4_%EC%8B%9C" title="사세보 시">사세보 시</a>에 있는 <a href="/wiki/%EB%AF%B8%EA%B5%AD_%ED%95%B4%EA%B5%B0" title="미국 해군">미국 해군</a>의 해외 <a class="new" href="/w/index.php?title=%ED%95%B4%EA%B5%B0%EA%B8%B0%EC%A7%80&amp;action=edit&amp;redlink=1" title="해군기지 (없는 문서)">시설</a>이다.</p>
<p>요코스카 해군 시설과 마찬가지로 해상자위대는 일부 시설을 같이 쓰고 있고, 양쪽 기지의 이름이 같아서 앞에 조직명("미해군/해상자위대")을 붙여서 구별한다. 규슈방위국에서는 "사세보 해군 시설"(<span lang="ja">佐世保海軍施設</span>로 명명하고 있다.<sup class="reference" id="cite_ref-1"><a href="#cite_note-1">[1]</a></sup></p>

<h2><span class="mw-headline" id=".EC.97.AD.EC.82.AC">역사</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%82%AC%EC%84%B8%EB%B3%B4_%ED%95%B4%EA%B5%B0_%EA%B8%B0%EC%A7%80&amp;action=edit&amp;section=1" title="부분 편집: 역사">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<div class="rellink boilerplate seealso"><img alt="" data-file-height="128" data-file-width="128" height="16" src="//upload.wikimedia.org/wikipedia/commons/thumb/e/ec/Crystal_Clear_app_xmag.svg/16px-Crystal_Clear_app_xmag.svg.png" srcset="//upload.wikimedia.org/wikipedia/commons/thumb/e/ec/Crystal_Clear_app_xmag.svg/24px-Crystal_Clear_app_xmag.svg.png 1.5x, //upload.wikimedia.org/wikipedia/commons/thumb/e/ec/Crystal_Clear_app_xmag.svg/32px-Crystal_Clear_app_xmag.svg.png 2x" width="16"/> <a href="/wiki/%EC%82%AC%EC%84%B8%EB%B3%B4_%EC%A7%84%EC%88%98%EB%B6%80" title="사세보 진수부">사세보 진수부</a>, <a href="/wiki/%EC%82%AC%EC%84%B8%EB%B3%B4_%ED%95%B4%EA%B5%B0%EA%B3%B5%EC%B0%BD" title="사세보 해군공창">사세보 해군공창</a> 문서를 참고하십시오.</div>
<h3><span class="mw-headline" id=".EC.B4.88.EA.B8.B0">초기</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%82%AC%EC%84%B8%EB%B3%B4_%ED%95%B4%EA%B5%B0_%EA%B8%B0%EC%A7%80&amp;action=edit&amp;section=2" title="부분 편집: 초기">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>1945년 9월 22일, <a class="new" href="/w/index.php?title=%EC%A0%9C5%ED%95%B4%EB%B3%91%EC%82%AC%EB%8B%A8&amp;action=edit&amp;redlink=1" title="제5해병사단 (없는 문서)">제5해병사단</a>이 사세보에 상륙하여 진수부를 접수하고, 이듬해 1946년 6월 22일에 사세보 해군 기지가 설립되었다.</p>
<h3><span class="mw-headline" id=".EB.83.89.EC.A0.84">냉전</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%82%AC%EC%84%B8%EB%B3%B4_%ED%95%B4%EA%B5%B0_%EA%B8%B0%EC%A7%80&amp;action=edit&amp;section=3" title="부분 편집: 냉전">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p><a href="/wiki/%ED%95%9C%EA%B5%AD_%EC%A0%84%EC%9F%81" title="한국 전쟁">한국 전쟁</a>의 휴전 이후, 수백톤의 탄약, 연료, 전차, 화물차와 보급품이 사세보를 거쳐 주한 유엔군으로 흘러 들어갔다. 기지는 잠시동안 2만명까지 수용했었다. 일본 정부는 <a href="/wiki/%EC%9E%90%EC%9C%84%EB%8C%80" title="자위대">자위대</a>를 창설하여 <a class="new" href="/w/index.php?title=%EC%82%AC%EC%84%B8%EB%B3%B4_%EA%B8%B0%EC%A7%80_(%ED%95%B4%EC%83%81%EC%9E%90%EC%9C%84%EB%8C%80)&amp;action=edit&amp;redlink=1" title="사세보 기지 (해상자위대) (없는 문서)">사세보</a>를 <a href="/wiki/%ED%95%B4%EC%83%81%EC%9E%90%EC%9C%84%EB%8C%80" title="해상자위대">해상자위대</a>의 모항 중 하나로 삼고 그들의 함정을 배치(<a class="new" href="/w/index.php?title=%EC%82%AC%EC%84%B8%EB%B3%B4_%EC%A7%80%EB%B0%A9%EB%8C%80&amp;action=edit&amp;redlink=1" title="사세보 지방대 (없는 문서)">사세보 지방대</a>)하였다.</p>
<p>1977년 3월 15일, 기지의 일부 토지가 일본 정부에 반환되었다.<sup class="reference" id="cite_ref-2"><a href="#cite_note-2">[2]</a></sup></p>
<p>1980년 7월 4일, 사세보 해군 기지에 <a href="/wiki/%EC%A0%9C7%ED%95%A8%EB%8C%80_(%EB%AF%B8%EA%B5%AD)" title="제7함대 (미국)">제7함대</a>의 함정이 전진 배치되었다. <a href="/wiki/USS_%EB%B3%B8%ED%97%98_%EB%A6%AC%EC%B2%98%EB%93%9C_(LHD-6)" title="USS 본험 리처드 (LHD-6)">USS <i>본험 리처드</i> (LHD-6)</a>, <a class="new" href="/w/index.php?title=USS_%EC%95%A0%EC%89%AC%EB%9E%9C%EB%93%9C_(LSD-48)&amp;action=edit&amp;redlink=1" title="USS 애쉬랜드 (LSD-48) (없는 문서)">USS <i>애쉬랜드</i> (LSD-48)</a>, <a class="new" href="/w/index.php?title=USS_%EA%B2%8C%EB%A5%B4%EB%A7%8C_%ED%83%80%EC%9A%B4_(LSD-42)&amp;action=edit&amp;redlink=1" title="USS 게르만 타운 (LSD-42) (없는 문서)">USS <i>게르만 타운</i> (LSD-42)</a>, <a class="new" href="/w/index.php?title=USS_%EA%B7%B8%EB%A6%B0_%EB%B2%A0%EC%9D%B4_(LPD-20)&amp;action=edit&amp;redlink=1" title="USS 그린 베이 (LPD-20) (없는 문서)">USS <i>그린 베이</i> (LPD-20)</a>, <a class="new" href="/w/index.php?title=USS_%EC%9B%8C%EB%A6%AC%EC%96%B4_(MCM-10)&amp;action=edit&amp;redlink=1" title="USS 워리어 (MCM-10) (없는 문서)">USS <i>워리어</i> (MCM-10)</a>, <a class="new" href="/w/index.php?title=USS_%ED%8C%A8%ED%8A%B8%EB%A6%AC%EC%96%B4%ED%8A%B8_(MCM-7)&amp;action=edit&amp;redlink=1" title="USS 패트리어트 (MCM-7) (없는 문서)">USS <i>패트리어트</i> (MCM-7)</a>, <a class="new" href="/w/index.php?title=USS_%ED%8C%8C%EC%9D%B4%EC%98%A4%EB%8B%88%EC%96%B4_(MCM-9)&amp;action=edit&amp;redlink=1" title="USS 파이오니어 (MCM-9) (없는 문서)">USS <i>파이오니어</i> (MCM-9)</a>, <a class="new" href="/w/index.php?title=USS_%EC%B9%98%ED%94%84_(MCM-14)&amp;action=edit&amp;redlink=1" title="USS 치프 (MCM-14) (없는 문서)">USS <i>치프</i> (MCM-14)</a>, 5,900명의 군인과 그들의 가족들도 기지에 입항하였다.</p>
<p>사막 <a class="mw-redirect" href="/wiki/%EC%82%AC%EB%A7%89_%EB%B0%A9%ED%8C%A8_%EC%9E%91%EC%A0%84" title="사막 방패 작전">방패</a>/<a class="mw-redirect" href="/wiki/%EC%82%AC%EB%A7%89_%ED%8F%AD%ED%92%8D_%EC%9E%91%EC%A0%84" title="사막 폭풍 작전">폭풍</a> 작전이 수행된 1990년~1991년 동안 해병대의 페르시아 걸프 전구 작전, 병참과 함정 급유 등을 위한 보급거점 역할을 맡았다.</p>
<h3><span class="mw-headline" id="21.EC.84.B8.EA.B8.B0">21세기</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%82%AC%EC%84%B8%EB%B3%B4_%ED%95%B4%EA%B5%B0_%EA%B8%B0%EC%A7%80&amp;action=edit&amp;section=4" title="부분 편집: 21세기">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>2015년 2월 19일, <a class="new" href="/w/index.php?title=USS_%EA%B7%B8%EB%A6%B0%EB%B2%A0%EC%9D%B4_(LPD-20)&amp;action=edit&amp;redlink=1" title="USS 그린베이 (LPD-20) (없는 문서)">USS <i>그린베이</i> (LPD-20)</a>가 사세보 기지에 입항하였다.<sup class="reference" id="cite_ref-3"><a href="#cite_note-3">[3]</a></sup></p>
<p>2016년 2월 12일, 미국 해군은 2017년 회계연도에서 사세보 해군 기지에 <a class="new" href="/w/index.php?title=%EC%A4%8C%EC%99%88%ED%8A%B8%EA%B8%89_%EA%B5%AC%EC%B6%95%ED%95%A8&amp;action=edit&amp;redlink=1" title="줌왈트급 구축함 (없는 문서)">줌왈트급 구축함</a>을 수용할 수 있도록 설비를 보수하는 비용 1,642만 달러와 2019년에 배치할 새로운 강습상륙함의 예산안을 미리 편성하였다고 밝혔다. 설비 보수에는 2017년 5월부터 2018년 10월까지 약 1년 5개월이 걸릴 예정이다.<sup class="reference" id="cite_ref-4"><a href="#cite_note-4">[4]</a></sup></p>
