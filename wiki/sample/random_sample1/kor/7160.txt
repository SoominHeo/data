<p><b>사에코</b>(<span lang="ja" xml:lang="ja">紗栄子</span>, <a href="/wiki/1986%EB%85%84" title="1986년">1986년</a> <a href="/wiki/11%EC%9B%94_16%EC%9D%BC" title="11월 16일">11월 16일</a> ~ )는 <a href="/wiki/%EC%9D%BC%EB%B3%B8" title="일본">일본</a>의 배우이며, 탤런트이다. 본명은 <b>도큐 사에코</b>(<span lang="ja" xml:lang="ja">道休冴子</span>)이다.</p>
<p><a href="/wiki/2006%EB%85%84" title="2006년">2006년</a> 4월에 <a href="/wiki/%EC%99%80%EC%84%B8%EB%8B%A4_%EB%8C%80%ED%95%99" title="와세다 대학">와세다 대학</a> 인간정보과학과에 입학했다. <a href="/wiki/2007%EB%85%84" title="2007년">2007년</a>에 <a href="/wiki/%EC%9D%BC%EB%B3%B8_%ED%94%84%EB%A1%9C_%EC%95%BC%EA%B5%AC" title="일본 프로 야구">일본 프로 야구</a> <a href="/wiki/%ED%8D%BC%EC%8B%9C%ED%94%BD_%EB%A6%AC%EA%B7%B8" title="퍼시픽 리그">퍼시픽 리그</a> 팀 <a href="/wiki/%ED%99%8B%EC%B9%B4%EC%9D%B4%EB%8F%84_%EB%8B%9B%ED%8F%B0%ED%96%84_%ED%8C%8C%EC%9D%B4%ED%84%B0%EC%8A%A4" title="홋카이도 닛폰햄 파이터스">홋카이도 닛폰햄 파이터스</a>의 <a href="/wiki/%ED%88%AC%EC%88%98" title="투수">투수</a> <a href="/wiki/%EB%8B%A4%EB%A5%B4%EB%B9%97%EC%8A%88_%EC%9C%A0" title="다르빗슈 유">다르빗슈 유</a>와 교제 6개월 만인 2007년 10월 임신사실과 함께 결혼 발표를 했고, <a href="/wiki/2008%EB%85%84" title="2008년">2008년</a>에 장남, <a href="/wiki/2010%EB%85%84" title="2010년">2010년</a>에 차남이 태어났으나 <a href="/wiki/2011%EB%85%84" title="2011년">2011년</a>부터 별거에 들어갔고 <a href="/wiki/2012%EB%85%84" title="2012년">2012년</a> 1월 <a href="/wiki/%EB%8B%A4%EB%A5%B4%EB%B9%97%EC%8A%88_%EC%9C%A0" title="다르빗슈 유">다르빗슈</a>가 <a href="/wiki/%ED%85%8D%EC%82%AC%EC%8A%A4_%EB%A0%88%EC%9D%B8%EC%A0%80%EC%8A%A4" title="텍사스 레인저스">텍사스 레인저스</a>에 입단한 날 합의 이혼하였다.(4년 2개월 결혼생활에 결국 마침표를 찍고, 양측은 두 아이 양육권을 사에코가 가지며 위자료는 없는 것으로 합의 이혼했다.)</p>
