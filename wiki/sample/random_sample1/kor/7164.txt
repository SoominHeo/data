<p><b>사오양</b>(소양, <small><a href="/wiki/%EC%A4%91%EA%B5%AD%EC%96%B4" title="중국어">중국어</a></small><small>:</small> <span lang="zh" xml:lang="zh">邵阳</span>, <small><a href="/wiki/%ED%95%9C%EC%96%B4_%EB%B3%91%EC%9D%8C" title="한어 병음">병음</a>:</small> <font face="Lucida Sans Unicode">Shàoyáng</font>)은 <a href="/wiki/%EC%A4%91%ED%99%94%EC%9D%B8%EB%AF%BC%EA%B3%B5%ED%99%94%EA%B5%AD" title="중화인민공화국">중화인민공화국</a> <a href="/wiki/%ED%9B%84%EB%82%9C_%EC%84%B1" title="후난 성">후난 성</a>의 지급시이다.</p>
<p>사오양은 2500년의 역사를 가지고 있고 후난 성의 중요한 상업과 교통의 중심지이다. 하나의 도시권과 교외의 8개의 현이 있고 인구는 700만이 넘는다.</p>
<p>사오양은 후난 성의 주요 삼림 보존 지역 중 하나이다. 42.7%가 숲으로 덮여있어 후난 성에서 가장 넓다. 난산 목장은 면적이 9.3㎢로 중국 남부에서 가장 큰 매일 목장이자 후난 성의 육류 생산지이다.</p>
<p><a href="/wiki/%EC%83%B9%EC%96%B4" title="샹어">샹어</a>의 일종인 사오양 화가 이곳에서 사용된다.</p>

<h2><span class="mw-headline" id=".EC.A7.80.EB.A6.AC">지리</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%82%AC%EC%98%A4%EC%96%91_%EC%8B%9C&amp;action=edit&amp;section=1" title="부분 편집: 지리">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>후난 성의 중서부에 위치하고 <a href="/wiki/%ED%99%94%EC%9D%B4%ED%99%94_%EC%8B%9C" title="화이화 시">화이화 시</a>, <a href="/wiki/%EB%A4%84%EB%94%94_%EC%8B%9C" title="뤄디 시">뤄디 시</a>, <a href="/wiki/%ED%97%9D%EC%96%91_%EC%8B%9C" title="헝양 시">헝양 시</a>, <a href="/wiki/%EC%9C%B5%EC%A0%80%EC%9A%B0_%EC%8B%9C" title="융저우 시">융저우 시</a>, <a href="/wiki/%EA%B4%91%EC%8B%9C_%EC%A2%A1%EC%A1%B1_%EC%9E%90%EC%B9%98%EA%B5%AC" title="광시 좡족 자치구">광시 좡족 자치구</a>와 접한다. 시내 주요 하천으로 <a class="new" href="/w/index.php?title=%EC%AF%94_%EA%B0%95&amp;action=edit&amp;redlink=1" title="쯔 강 (없는 문서)">쯔 강</a>이 있다.</p>
<h2><span class="mw-headline" id=".EC.97.AD.EC.82.AC">역사</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%82%AC%EC%98%A4%EC%96%91_%EC%8B%9C&amp;action=edit&amp;section=2" title="부분 편집: 역사">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>사오양에 처음으로 행정구역이 설정된 것은 <a href="/wiki/%ED%95%9C%EB%82%98%EB%9D%BC" title="한나라">한나라</a> 초기의 소릉현(昭陵县)이다. 266년, <a class="mw-disambig" href="/wiki/%EC%98%A4%EB%82%98%EB%9D%BC" title="오나라">오나라</a>는 영릉군 북부를 새롭게 소릉군(昭陵郡)으로 개편했고, 280년, <a href="/wiki/%EC%84%9C%EC%A7%84" title="서진">서진</a>에 의해 소릉군(邵陵郡)으로 개칭되었다.</p>
<p><a href="/wiki/%EB%8B%B9%EB%82%98%EB%9D%BC" title="당나라">당나라</a> 때 군을 대신해 새롭게 주제가 시행되면서 소주(邵州)가 설치되었고 송대에는 연호인 <a href="/wiki/%EB%B3%B4%EA%B2%BD" title="보경">보경</a>을 채택해 1225년(보경 원년)에 보경부(宝慶府)로 개칭되어 이후 민국까지 그 명칭이 계속되었다.</p>
<p>1926년, 바오칭 현(宝慶县)이 사오양 현으로 개칭되었고 <a href="/wiki/%EC%A4%91%ED%99%94%EC%9D%B8%EB%AF%BC%EA%B3%B5%ED%99%94%EA%B5%AD" title="중화인민공화국">중화인민공화국</a> 성립 후인 1950년에 사오양 행정전공서(邵陽行政専公署) 관할의 현급시로서 사오양 시로 승격, 1977년에 성할시가 되었으며, 1980년에 지급시에 개편되어 현재에 이르고 있다.</p>
