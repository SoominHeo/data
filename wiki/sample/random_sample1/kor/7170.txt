<p><b>사우 해</b>(<span style="font-size: smaller;"><a href="/wiki/%EC%9D%B8%EB%8F%84%EB%84%A4%EC%8B%9C%EC%95%84%EC%96%B4" title="인도네시아어">인도네시아어</a>:</span> <span lang="id" xml:lang="id">Laut Sawu</span>)는 <a href="/wiki/%EC%9D%B8%EB%8F%84%EB%84%A4%EC%8B%9C%EC%95%84" title="인도네시아">인도네시아</a>의 바다이다. 남쪽으로 <a class="new" href="/w/index.php?title=%EC%82%AC%EC%9A%B0_%EC%84%AC&amp;action=edit&amp;redlink=1" title="사우 섬 (없는 문서)">사우 섬</a>과 <a class="new" href="/w/index.php?title=%EB%9D%BC%EC%9D%B4%EC%A3%BC%EC%95%84_%EC%84%AC&amp;action=edit&amp;redlink=1" title="라이주아 섬 (없는 문서)">라이주아 섬</a>, 동쪽으로 <a class="new" href="/w/index.php?title=%EB%A1%9C%ED%8A%B8_%EC%84%AC&amp;action=edit&amp;redlink=1" title="로트 섬 (없는 문서)">로트 섬</a>과 <a href="/wiki/%ED%8B%B0%EB%AA%A8%EB%A5%B4_%EC%84%AC" title="티모르 섬">티모르 섬</a>, 북쪽과 북서쪽으로 <a href="/wiki/%ED%94%8C%EB%A1%9C%EB%A0%88%EC%8A%A4_%EC%84%AC" title="플로레스 섬">플로레스 섬</a>과 <a class="new" href="/w/index.php?title=%EC%95%8C%EB%A1%9C%EB%A5%B4_%EC%A0%9C%EB%8F%84&amp;action=edit&amp;redlink=1" title="알로르 제도 (없는 문서)">알로르 제도</a>, 북서쪽과 서쪽으로 <a href="/wiki/%EC%88%A8%EB%B0%94_%EC%84%AC" title="숨바 섬">숨바 섬</a>으로 둘러쌓여 있다. 남쪽과 서쪽은 <a href="/wiki/%EC%9D%B8%EB%8F%84%EC%96%91" title="인도양">인도양</a>, 북쪽은 <a href="/wiki/%ED%94%8C%EB%A1%9C%EB%A0%88%EC%8A%A4_%ED%95%B4" title="플로레스 해">플로레스 해</a>, 북동쪽은 <a href="/wiki/%EB%B0%98%EB%8B%A4_%ED%95%B4" title="반다 해">반다 해</a>와 연결된다.</p>
