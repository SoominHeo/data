<p><b>사이토 마코토</b>(<span style="font-size: smaller;"><a href="/wiki/%EC%9D%BC%EB%B3%B8%EC%96%B4" title="일본어">일본어</a>:</span> <span lang="ja"><span lang="ja"><ruby><rb>齋藤 實</rb> <rp>(</rp><rt>さいとう まこと</rt><rp>)</rp></ruby></span></span>, <a href="/wiki/1858%EB%85%84" title="1858년">1858년</a> <a href="/wiki/12%EC%9B%94_2%EC%9D%BC" title="12월 2일">12월 2일</a> ~ <a href="/wiki/1936%EB%85%84" title="1936년">1936년</a> <a href="/wiki/2%EC%9B%94_26%EC%9D%BC" title="2월 26일">2월 26일</a>)는 <a href="/wiki/%EC%9D%BC%EB%B3%B8" title="일본">일본</a>의 해군 군인이자 관료, 정치가이다.</p>
<p><a class="mw-redirect mw-disambig" href="/wiki/%ED%95%B4%EA%B5%B0%EB%B3%91%ED%95%99%EA%B5%90" title="해군병학교">해군병학교</a>를 졸업한 후 미국에서 유학한 자수성가형 인물로, <a href="/wiki/%ED%95%B4%EA%B5%B0" title="해군">해군</a> <a class="mw-redirect" href="/wiki/%EB%8C%80%EC%9E%A5_(%EA%B5%B0%EC%9D%B8_%EA%B3%84%EA%B8%89)" title="대장 (군인 계급)">대장</a>과 해군 대신을 지냈다. <a href="/wiki/1919%EB%85%84" title="1919년">1919년</a>부터 <a href="/wiki/1927%EB%85%84" title="1927년">1927년</a>까지 제3대, <a href="/wiki/1929%EB%85%84" title="1929년">1929년</a>부터 <a href="/wiki/1931%EB%85%84" title="1931년">1931년</a>까지 제5대 <a href="/wiki/%EC%A1%B0%EC%84%A0%EC%B4%9D%EB%8F%85%EB%B6%80" title="조선총독부">조선총독부</a> 총독으로 근무했다.</p>
<p><a href="/wiki/1919%EB%85%84" title="1919년">1919년</a> <a href="/wiki/9%EC%9B%94" title="9월">9월</a> 조선 총독으로 부임하러 <a href="/wiki/%EC%A1%B0%EC%84%A0" title="조선">조선</a>의 <a href="/wiki/%EC%84%9C%EC%9A%B8%EC%97%AD" title="서울역">남대문역</a>에서 내리다가 <a href="/wiki/%EA%B0%95%EC%9A%B0%EA%B7%9C" title="강우규">강우규</a>(姜宇奎) 등의 폭탄 습격을 받았으나 구사일생으로 죽음을 면하였다. <a class="mw-redirect" href="/wiki/3%C2%B71%EC%9A%B4%EB%8F%99" title="3·1운동">3·1운동</a> 이후 조선 총독으로서 종전의 '무단정치'에서 '문화정치'로 통치 방법을 전환시켜 식민지 반발을 무마하려 노력했다.<sup class="reference" id="cite_ref-1"><a href="#cite_note-1">[1]</a></sup></p>
<p>1925년에 <a href="/wiki/%EC%9E%90%EC%9E%91" title="자작">자작</a>(子爵)이 되고, <a href="/wiki/1932%EB%85%84" title="1932년">1932년</a>부터 <a href="/wiki/1934%EB%85%84" title="1934년">1934년</a>까지 <a class="mw-disambig" href="/wiki/%EC%B4%9D%EB%A6%AC%EB%8C%80%EC%8B%A0" title="총리대신">총리대신</a>, <a href="/wiki/1932%EB%85%84" title="1932년">1932년</a>부터 <a href="/wiki/1933%EB%85%84" title="1933년">1933년</a>까지 겸임 외무대신을 지냈다. <a href="/wiki/1933%EB%85%84" title="1933년">1933년</a> 겸임 <a class="new" href="/w/index.php?title=%EB%AC%B8%EB%B6%80%EB%8C%80%EC%8B%A0&amp;action=edit&amp;redlink=1" title="문부대신 (없는 문서)">문부대신</a>을 지내다가 <a href="/wiki/1934%EB%85%84" title="1934년">1934년</a> 사퇴하였다. 이후 내대신으로 재직 중 <a href="/wiki/1936%EB%85%84" title="1936년">1936년</a> <a href="/wiki/2%EC%9B%94_26%EC%9D%BC" title="2월 26일">2월 26일</a>에 발발한 <a href="/wiki/2%C2%B726_%EC%82%AC%EA%B1%B4" title="2·26 사건">2·26 사건</a>에 가담한 청년 장교들에 의해 암살되었다.</p>

<h2><span class="mw-headline" id=".EC.83.9D.EC.95.A0.EC.99.80_.EA.B2.BD.EB.A0.A5">생애와 경력</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%82%AC%EC%9D%B4%ED%86%A0_%EB%A7%88%EC%BD%94%ED%86%A0&amp;action=edit&amp;section=1" title="부분 편집: 생애와 경력">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<h3><span class="mw-headline" id=".EC.88.98.ED.95.99.EA.B3.BC_.ED.95.B4.EA.B5.B0_.EB.B3.B5.EB.AC.B4">수학과 해군 복무</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%82%AC%EC%9D%B4%ED%86%A0_%EB%A7%88%EC%BD%94%ED%86%A0&amp;action=edit&amp;section=2" title="부분 편집: 수학과 해군 복무">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<div class="thumb tleft">

<p><a href="/wiki/%EC%9D%B4%EC%99%80%ED%85%8C_%ED%98%84" title="이와테 현">이와테 현</a>(岩手縣, 현 오쓰 시(奥州市) 마쓰사와 구(水沢区)에서 몰락한 사족(士族) 집안에서 태어났다. 처음 이름은 토미고로(富五郎)였다가 나중에 마코토(實)로 개명하였다. 열심히 공부하여 <a class="mw-redirect mw-disambig" href="/wiki/%ED%95%B4%EA%B5%B0%EB%B3%91%ED%95%99%EA%B5%90" title="해군병학교">해군병학교</a>를 졸업, 이후 <a href="/wiki/1884%EB%85%84" title="1884년">1884년</a> <a href="/wiki/%EB%AF%B8%EA%B5%AD" title="미국">미국</a>으로 유학하였으며 유학 중 겸임 주미<a href="/wiki/%EC%9D%BC%EB%B3%B8" title="일본">일본</a>공사관 주재무관의 자격이 부여되어 <a href="/wiki/1888%EB%85%84" title="1888년">1888년</a>까지 그곳의 일본 최초의 주재무관으로 머물렀다. 이때 국제적인 안목을 기르고 유창한 영어를 습득했다.</p>
<p>후에 일본으로 돌아와 아키즈노즈(秋津洲) 호와 쿠마(嚴島)호의 함장을 지냈다. 이후 자수성가하여 야전과 참모, 궁정직을 돌아가면서 섭렵했고 <a href="/wiki/1897%EB%85%84" title="1897년">1897년</a>에는 한 달 만에 2계급을 뛰어넘는 고속승진을 하여 대좌에 올랐다. 1900년 5월 해군 소장이 되고 1904년 6월 해군 중장으로 승진했으며 러일 전쟁에 참전하였다.</p>
<h3><span class="mw-headline" id=".EC.9E.A5.EA.B5.B0_.EC.A7.84.EA.B8.89.EA.B3.BC_.EA.B0.81.EB.A3.8C_.EC.B7.A8.EC.9E.84">장군 진급과 각료 취임</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%82%AC%EC%9D%B4%ED%86%A0_%EB%A7%88%EC%BD%94%ED%86%A0&amp;action=edit&amp;section=3" title="부분 편집: 장군 진급과 각료 취임">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<div class="rellink boilerplate seealso"><img alt="" data-file-height="128" data-file-width="128" height="16" src="//upload.wikimedia.org/wikipedia/commons/thumb/e/ec/Crystal_Clear_app_xmag.svg/16px-Crystal_Clear_app_xmag.svg.png" srcset="//upload.wikimedia.org/wikipedia/commons/thumb/e/ec/Crystal_Clear_app_xmag.svg/24px-Crystal_Clear_app_xmag.svg.png 1.5x, //upload.wikimedia.org/wikipedia/commons/thumb/e/ec/Crystal_Clear_app_xmag.svg/32px-Crystal_Clear_app_xmag.svg.png 2x" width="16"/> <a href="/wiki/%EB%9F%AC%EC%9D%BC_%EC%A0%84%EC%9F%81" title="러일 전쟁">러일 전쟁</a> 문서를 참고하십시오.</div>
<p>1905년 1월에는 선박 건조를 담당하는 부서인 해군 감정본부장(艦政本部長)이 되고 <a href="/wiki/%EB%9F%AC%EC%9D%BC_%EC%A0%84%EC%9F%81" title="러일 전쟁">러일 전쟁</a> 때는 해군차관에 임명되었다. 이후 7년간 해군차관으로 복무하며 <a href="/wiki/%EC%A4%91%EC%9E%A5" title="중장">중장</a>까지 승진, 대해군 건설에 힘쏟아 <a href="/wiki/%EB%9F%AC%EC%9D%BC_%EC%A0%84%EC%9F%81" title="러일 전쟁">러일 전쟁</a>에서 일본 해군이 러시아 해군을 격파하는 데 큰 공을 세웠다. 해군 대장에 이르렀다.</p>
<p><a href="/wiki/1907%EB%85%84" title="1907년">1907년</a> <a href="/wiki/%EB%82%A8%EC%9E%91" title="남작">남작</a>의 작위를 받았다.</p>
<h3><span class="mw-headline" id=".EA.B4.80.EB.A3.8C_.EC.83.9D.ED.99.9C">관료 생활</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%82%AC%EC%9D%B4%ED%86%A0_%EB%A7%88%EC%BD%94%ED%86%A0&amp;action=edit&amp;section=4" title="부분 편집: 관료 생활">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<div class="rellink boilerplate seealso"><img alt="" data-file-height="128" data-file-width="128" height="16" src="//upload.wikimedia.org/wikipedia/commons/thumb/e/ec/Crystal_Clear_app_xmag.svg/16px-Crystal_Clear_app_xmag.svg.png" srcset="//upload.wikimedia.org/wikipedia/commons/thumb/e/ec/Crystal_Clear_app_xmag.svg/24px-Crystal_Clear_app_xmag.svg.png 1.5x, //upload.wikimedia.org/wikipedia/commons/thumb/e/ec/Crystal_Clear_app_xmag.svg/32px-Crystal_Clear_app_xmag.svg.png 2x" width="16"/> <a class="new" href="/w/index.php?title=%EC%A7%80%EB%A9%98%EC%8A%A4_%EC%82%AC%EA%B1%B4&amp;action=edit&amp;redlink=1" title="지멘스 사건 (없는 문서)">지멘스 사건</a>, <a class="mw-redirect" href="/wiki/%EC%82%AC%EC%9D%B4%ED%86%A0_%EC%B4%9D%EB%8F%85_%EC%A0%80%EA%B2%A9_%EC%82%AC%EA%B1%B4" title="사이토 총독 저격 사건">사이토 총독 저격 사건</a> 문서를 참고하십시오.</div>
<p><a href="/wiki/1906%EB%85%84" title="1906년">1906년</a> <a href="/wiki/%EC%82%AC%EC%9D%B4%EC%98%A8%EC%A7%80_%EA%B8%B4%EB%AA%A8%EC%B9%98" title="사이온지 긴모치">사이온지</a> 내각에서 해군대신이 되었고, 이후의 내각에서도 다섯 차례 연속으로 해군대신을 지냈다. <a href="/wiki/1912%EB%85%84" title="1912년">1912년</a>에는 <a class="mw-redirect" href="/wiki/%EB%8C%80%EC%9E%A5_(%EA%B5%B0%EC%9D%B8_%EA%B3%84%EA%B8%89)" title="대장 (군인 계급)">대장</a>에 올랐으나, <a href="/wiki/1914%EB%85%84" title="1914년">1914년</a> 해군의 독직(瀆職) 사건<sup class="reference" id="cite_ref-2"><a href="#cite_note-2">[2]</a></sup>(지멘스 스캔들)으로 사임하였다.</p>
<p><a href="/wiki/1919%EB%85%84" title="1919년">1919년</a> <a href="/wiki/3%C2%B71_%EC%9A%B4%EB%8F%99" title="3·1 운동">3·1 운동</a> 직후 조선 총독으로 임명되었다. 1919년 <a href="/wiki/9%EC%9B%94_2%EC%9D%BC" title="9월 2일">9월 2일</a> 조선 총독으로 부임하러 <a href="/wiki/%EC%84%9C%EC%9A%B8%EC%97%AD" title="서울역">남대문역</a>에서 내리다가 <a href="/wiki/%EA%B0%95%EC%9A%B0%EA%B7%9C" title="강우규">강우규</a>의 폭탄 습격을 받았으나 살아남았다. 부임 후 조선 총독으로서 강압적 식민통치를 완화하여 조선인의 저항을 무력화하려는 <a href="/wiki/%EB%AC%B8%ED%99%94_%ED%86%B5%EC%B9%98" title="문화 통치">문화 통치</a> 정책을 추진하였다. 3·1운동 이후의 후폭풍을 우려하여 종전의 '무단정치'에서 '문화정치'로 통치방법을 전환시켜 식민지 반발을 무마하려 노력했다.</p>
<p><a href="/wiki/1925%EB%85%84" title="1925년">1925년</a>에 <a href="/wiki/%EC%9E%90%EC%9E%91" title="자작">자작</a>이 되고, <a href="/wiki/1927%EB%85%84" title="1927년">1927년</a> 제네바 군축협상에 전권대표로 참석하기 위해 조선총독직에서 물러났다. <a href="/wiki/1929%EB%85%84" title="1929년">1929년</a>에 <a class="mw-redirect" href="/wiki/%EC%A1%B0%EC%84%A0%EC%B4%9D%EB%8F%85" title="조선총독">조선총독</a>에 재임명되었고, <a href="/wiki/1931%EB%85%84" title="1931년">1931년</a> 조선총독직을 다시 사임하였고, 이듬해 <a class="mw-redirect" href="/wiki/%EC%9D%BC%EB%B3%B8%EC%9D%98_%EC%B4%9D%EB%A6%AC" title="일본의 총리">총리</a>가 되었다.</p>
<h3><span class="mw-headline" id=".EC.B4.9D.EB.A6.AC_.EC.82.AC.ED.87.B4.EC.99.80_.EC.B5.9C.ED.9B.84">총리 사퇴와 최후</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%82%AC%EC%9D%B4%ED%86%A0_%EB%A7%88%EC%BD%94%ED%86%A0&amp;action=edit&amp;section=5" title="부분 편집: 총리 사퇴와 최후">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<div class="thumb tright">

<p>1932년 거국일치 내각을 조직하여 수상에 취임하였다. 만주국 승인·국제연맹 탈퇴·농촌 구제사업을 추진하여 민심과 내분을 수습하려 했으나 실패했다. <a href="/wiki/1932%EB%85%84" title="1932년">1932년</a> <a class="mw-disambig" href="/wiki/%EC%99%B8%EB%AC%B4%EB%8C%80%EC%8B%A0" title="외무대신">외무대신</a>을 겸하였고, 1933년 외무대신직에서 물러났으나 겸임 학무대신이 되었다. <a href="/wiki/1934%EB%85%84" title="1934년">1934년</a> 내각의 비리로 총리직을 사임했고 이후 <a href="/wiki/%EC%9D%BC%EB%B3%B8_%EC%B2%9C%ED%99%A9" title="일본 천황">일본 천황</a>의 보좌관격인 내대신(內大臣)이 되었다.</p>
<p><a class="mw-redirect" href="/wiki/%EB%A7%8C%EC%A3%BC_%EC%82%AC%EB%B3%80" title="만주 사변">만주 사변</a> 이후 국가의 총체적 군국화를 주장하며 발호하는 혁신 우익(<a href="/wiki/%ED%99%A9%EB%8F%84%ED%8C%8C" title="황도파">황도파</a>)에 맞서 현상 유지를 노력했기 때문에 황도파의 젊은 장교들의 주적이 되었다. <a href="/wiki/1936%EB%85%84" title="1936년">1936년</a> 친영파(親英派)와 친미파(親美派) 정치인의 한 사람으로 지목되어 그해 <a href="/wiki/2%EC%9B%94" title="2월">2월</a> <a href="/wiki/2%C2%B726_%EC%82%AC%EA%B1%B4" title="2·26 사건">2·26 사건</a> 당시 반란을 일으킨 <a href="/wiki/%ED%99%A9%EB%8F%84%ED%8C%8C" title="황도파">황도파</a> 청년<a href="/wiki/%EC%9E%A5%EA%B5%90" title="장교">장교</a>들에게 <a href="/wiki/%EB%8F%84%EC%BF%84" title="도쿄">도쿄</a> 자택에서 78세의 나이로 살해당했다.</p>
