<p><b>사진 렌즈</b>(寫眞 lens, <span lang="en" xml:lang="en">photographic lens</span>)는 <a class="mw-redirect" href="/wiki/%EC%B9%B4%EB%A9%94%EB%9D%BC" title="카메라">카메라</a>에 연결하여 <a href="/wiki/%EC%82%AC%EC%A7%84_%ED%95%84%EB%A6%84" title="사진 필름">사진 필름</a>이나 <a class="mw-redirect" href="/wiki/%EA%B3%A0%EC%B2%B4_%EC%B4%AC%EC%83%81_%EC%86%8C%EC%9E%90" title="고체 촬상 소자">고체 촬상 소자</a>에 적절한 빛을 전달하는 역할을 하는 <a href="/wiki/%EA%B4%91%ED%95%99" title="광학">광학</a> <a href="/wiki/%EB%A0%8C%EC%A6%88" title="렌즈">렌즈</a>, 또는 렌즈의 구성체이다. 보통 여러 장의 렌즈가 적절하게 겹쳐있다.</p>
<h2><span class="mw-headline" id=".EB.B6.84.EB.A5.98">분류</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%82%AC%EC%A7%84_%EB%A0%8C%EC%A6%88&amp;action=edit&amp;section=1" title="부분 편집: 분류">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>사진 렌즈는 <a class="mw-redirect" href="/wiki/%EC%B4%88%EC%A0%90_%EA%B1%B0%EB%A6%AC" title="초점 거리">초점 거리</a>와 초점 거리의 변동 가능성 혹은 집광 방식에 따라서 여러 가지로 분류할 수 있다. 일반적으로 사용하는 <a class="new" href="/w/index.php?title=%EB%A7%9D%EC%9B%90_%EB%A0%8C%EC%A6%88&amp;action=edit&amp;redlink=1" title="망원 렌즈 (없는 문서)">망원 렌즈</a>, <a href="/wiki/%ED%91%9C%EC%A4%80_%EB%A0%8C%EC%A6%88" title="표준 렌즈">표준 렌즈</a>, <a href="/wiki/%EA%B4%91%EA%B0%81_%EB%A0%8C%EC%A6%88" title="광각 렌즈">광각 렌즈</a> 등은 초점 거리에 따라서 분류한 것이고, <a class="mw-redirect" href="/wiki/%EC%A4%8C_%EB%A0%8C%EC%A6%88" title="줌 렌즈">줌 렌즈</a>와 <a class="new" href="/w/index.php?title=%EB%8B%A8%EC%B4%88%EC%A0%90_%EB%A0%8C%EC%A6%88&amp;action=edit&amp;redlink=1" title="단초점 렌즈 (없는 문서)">단초점 렌즈</a>등은 초점 거리의 변동유무를 통해서 나눈 것이다. 한편 <a class="new" href="/w/index.php?title=%EB%B0%98%EC%82%AC_%EB%A0%8C%EC%A6%88&amp;action=edit&amp;redlink=1" title="반사 렌즈 (없는 문서)">반사 렌즈</a>와 같이 유리로 구성된 렌즈 대신 <a href="/wiki/%EA%B1%B0%EC%9A%B8" title="거울">거울</a>을 사용하는 렌즈 역시 존재한다. 한편, 사진의 왜곡 조정이나 해상력, 최소 초점 거리에 따라서 <a class="new" href="/w/index.php?title=%EB%A7%A4%ED%81%AC%EB%A1%9C_%EB%A0%8C%EC%A6%88&amp;action=edit&amp;redlink=1" title="매크로 렌즈 (없는 문서)">매크로 렌즈</a>나 소프트 렌즈, 어안 렌즈, TS(Tilt-Shift) 렌즈등의 다양한 세부 분류 역시 존재한다.</p>
