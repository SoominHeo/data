<p><b>사천 전투</b>(泗川 戰鬪)는 <a href="/wiki/1598%EB%85%84" title="1598년">1598년</a> 10월, <a href="/wiki/%EC%A1%B0%EC%84%A0" title="조선">조선</a> <a href="/wiki/%EA%B2%BD%EC%83%81%EB%8F%84" title="경상도">경상도</a> <a href="/wiki/%EC%82%AC%EC%B2%9C%EC%8B%9C" title="사천시">사천</a> <a class="mw-redirect" href="/wiki/%EC%84%A0%EC%A7%84%EB%A6%AC%EC%84%B1" title="선진리성">선진리성</a>에서 <a href="/wiki/%EC%8B%9C%EB%A7%88%EC%A6%88_%EC%9A%94%EC%8B%9C%ED%9E%88%EB%A1%9C" title="시마즈 요시히로">시마즈 요시히로</a>(島津義弘)가 인솔하는 시마즈군 7천 명이 약 4만 명의 조·명 연합군과 싸워 승리한 전투이다. <sup class="reference" id="cite_ref-1"><a href="#cite_note-1">[1]</a></sup></p>

<h2><span class="mw-headline" id=".EB.B0.B0.EA.B2.BD">배경</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%82%AC%EC%B2%9C%EC%84%B1_%EC%A0%84%ED%88%AC&amp;action=edit&amp;section=1" title="부분 편집: 배경">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p><a href="/wiki/1598%EB%85%84" title="1598년">1598년</a> 9월 말부터 10월 초에 걸쳐 조·명 연합군은 서로군·중로군·동로군을 편성하여 서쪽에서 <a href="/wiki/%EC%88%9C%EC%B2%9C%EC%99%9C%EC%84%B1" title="순천왜성">순천왜성</a>(<a href="/wiki/%EA%B3%A0%EB%8B%88%EC%8B%9C_%EC%9C%A0%ED%82%A4%EB%82%98%EA%B0%80" title="고니시 유키나가">고니시</a>군), <a href="/wiki/%EC%82%AC%EC%B2%9C_%EC%84%A0%EC%A7%84%EB%A6%AC%EC%99%9C%EC%84%B1" title="사천 선진리왜성">사천 선진리왜성</a>(<a href="/wiki/%EC%8B%9C%EB%A7%88%EC%A6%88_%EC%9A%94%EC%8B%9C%ED%9E%88%EB%A1%9C" title="시마즈 요시히로">시마즈</a>군), <a href="/wiki/%EC%9A%B8%EC%82%B0%EC%99%9C%EC%84%B1" title="울산왜성">울산왜성</a>(<a href="/wiki/%EA%B0%80%ED%86%A0_%EA%B8%B0%EC%9A%94%EB%A7%88%EC%82%AC" title="가토 기요마사">가토</a>군)에 대해서 동시에 남진하였고, 육상의 삼로군과 수로군을 동시에 병진하게 하여 일본군을 공격한다는 '사로병진작전(四路竝進作戰)'에 돌입했다.</p>
<p>이 삼로군 중 <a href="/wiki/%EB%AA%85%EB%82%98%EB%9D%BC" title="명나라">명나라</a> 장수 <a href="/wiki/%EB%8F%99%EC%9D%BC%EC%9B%90" title="동일원">동일원</a>이 인솔하는 중로군 4만 명의 조·명 연합군이 시마즈군이 주둔하는 <a href="/wiki/%EC%82%AC%EC%B2%9C%EC%8B%9C" title="사천시">사천</a>으로 남진해 들어갔다. 사천은 일본군의 근원지인 <a class="mw-redirect" href="/wiki/%EB%B6%80%EC%82%B0" title="부산">부산</a>과 일본군 왼쪽 끝의 <a href="/wiki/%EC%88%9C%EC%B2%9C%EC%99%9C%EC%84%B1" title="순천왜성">순천왜성</a>과 <a class="new" href="/w/index.php?title=%EB%82%A8%ED%95%B4%EC%99%9C%EC%84%B1&amp;action=edit&amp;redlink=1" title="남해왜성 (없는 문서)">남해왜성</a>의 중간에 있기 때문에, 이곳을 함락시키면 서쪽으로 있는 일본군과의 연락이 분단될 위험이 있었다.</p>
<p>당시 사천일대의 일본군은 <a href="/wiki/%EC%8B%9C%EB%A7%88%EC%A6%88_%EC%9A%94%EC%8B%9C%ED%9E%88%EB%A1%9C" title="시마즈 요시히로">시마즈 요시히로</a>(島津義弘)를 주축으로 한 시마즈 가문의 5개 부대가 다소 느슨한 지휘체계로 결합해 있었다. <a href="/wiki/1597%EB%85%84" title="1597년">1597년</a> 시마즈 요시히로는 <a href="/wiki/%EB%AA%A8%EB%A6%AC_%EA%B0%80%EC%93%B0%EB%85%B8%EB%B6%80" title="모리 가쓰노부">모리 가쓰노부</a>가 축성한 <a href="/wiki/%EC%82%AC%EC%B2%9C_%EC%84%A0%EC%A7%84%EB%A6%AC%EC%99%9C%EC%84%B1" title="사천 선진리왜성">사천 선진리왜성</a>에 주둔하며 사천에서 진주에 이르는 요해(要害)에 진영을 설치하는 한편, <a href="/wiki/%EC%A7%84%EC%A3%BC%EC%84%B1" title="진주성">진주성</a>에 미하라 시게다네(三原重鍾)를 주둔시키고 <a class="new" href="/w/index.php?title=%EB%A7%9D%EC%A7%84%EC%99%9C%EC%84%B1&amp;action=edit&amp;redlink=1" title="망진왜성 (없는 문서)">망진왜성</a>에는 중로군이 남강을 건너 남진하는 것을 막기 위해 <a class="new" href="/w/index.php?title=%EB%8D%B0%EB%9D%BC%EC%95%BC%EB%A7%88_%ED%9E%88%EC%82%AC%EA%B0%80%EB%84%A4&amp;action=edit&amp;redlink=1" title="데라야마 히사가네 (없는 문서)">데라야마 히사가네</a>(寺山久兼)로 하여금 수비를 담당시켰다. 또한, 영춘과 사천에 성을 축조·보수하여 <a class="new" href="/w/index.php?title=%EC%98%81%EC%B6%98%EC%99%9C%EC%84%B1&amp;action=edit&amp;redlink=1" title="영춘왜성 (없는 문서)">영춘왜성</a>과 <a class="mw-redirect" href="/wiki/%EC%82%AC%EC%B2%9C%EA%B5%AC%EC%84%B1" title="사천구성">사천구성</a><sup class="reference" id="cite_ref-2"><a href="#cite_note-2">[2]</a></sup>에는 <a class="new" href="/w/index.php?title=%EA%B0%80%EC%99%80%EC%B9%B4%EB%AF%B8_%ED%9E%88%EC%82%AC%ED%86%A0%EB%AA%A8&amp;action=edit&amp;redlink=1" title="가와카미 히사토모 (없는 문서)">가와카미 히사토모</a>(川上久智)와 <a class="new" href="/w/index.php?title=%EA%B0%80%EC%99%80%EC%B9%B4%EB%AF%B8_%EB%8B%A4%EB%8B%A4%EB%85%B8%EB%A6%AC&amp;action=edit&amp;redlink=1" title="가와카미 다다노리 (없는 문서)">가와카미 다다자네</a>(川上忠実)로 하여금 각각 수비하게 하여 <a href="/wiki/%EC%A7%84%EC%A3%BC%EC%8B%9C" title="진주시">진주</a> <a class="mw-redirect" href="/wiki/%EB%82%A8%EA%B0%95_(%EA%B2%BD%EC%83%81%EB%8F%84)" title="남강 (경상도)">남강</a>을 최전 방위선으로 삼고 또 망진왜성·영춘왜성·사천구성의 3성을 사천 선진리왜성의 전진 방어진지로 삼고 있었다. 시마즈군의 병력은 시마즈 요시히로와 요시히로의 아들 <a href="/wiki/%EC%8B%9C%EB%A7%88%EC%A6%88_%EB%8B%A4%EB%8B%A4%EC%93%B0%EB%84%A4" title="시마즈 다다쓰네">시마즈 다다쓰네</a>(島津忠恒)가 인솔하는 7천 명뿐이었다.</p>
<p><a href="/wiki/%EC%86%8C_%EC%9A%94%EC%8B%9C%ED%86%A0%EC%8B%9C" title="소 요시토시">소</a>(宗)군이나 <a href="/wiki/%EB%8B%A4%EC%B9%98%EB%B0%94%EB%82%98_%EB%AC%B4%EB%84%A4%EC%8B%9C%EA%B2%8C" title="다치바나 무네시게">타치바나</a>(立花)군이 원군을 신청하였지만, 요시히로는 이 제의를 거절해,<sup class="reference" id="cite_ref-3"><a href="#cite_note-3">[3]</a></sup> 시마즈 가의 군세만으로 조·명의 대군을 맞아 싸우게 되었다.</p>
<h2><span class="mw-headline" id=".EA.B2.BD.EA.B3.BC">경과</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%82%AC%EC%B2%9C%EC%84%B1_%EC%A0%84%ED%88%AC&amp;action=edit&amp;section=2" title="부분 편집: 경과">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>조·명 연합군의 중로군 본대는 <a href="/wiki/1598%EB%85%84" title="1598년">1598년</a> <a href="/wiki/9%EC%9B%94_20%EC%9D%BC" title="9월 20일">9월 20일</a> 시마즈군이 주둔하고 있는 진주의 <a class="new" href="/w/index.php?title=%EB%A7%9D%EC%A7%84%EC%99%9C%EC%84%B1&amp;action=edit&amp;redlink=1" title="망진왜성 (없는 문서)">망진왜성</a>과 <a class="new" href="/w/index.php?title=%EC%98%81%EC%B6%98%EC%99%9C%EC%84%B1&amp;action=edit&amp;redlink=1" title="영춘왜성 (없는 문서)">영춘왜성</a>을 점령하고, 사천고성과 <a href="/wiki/%EC%82%AC%EC%B2%9C_%EC%84%A0%EC%A7%84%EB%A6%AC%EC%99%9C%EC%84%B1" title="사천 선진리왜성">사천 선진리왜성</a>으로 육박하여 공세를 취하였다. 이 상황을 보고받은 <a href="/wiki/%EC%8B%9C%EB%A7%88%EC%A6%88_%EC%9A%94%EC%8B%9C%ED%9E%88%EB%A1%9C" title="시마즈 요시히로">시마즈 요시히로</a>는 망진왜성, 영춘왜성, 곤양왜성에 주둔하던 병력을 철수시켜 사천고성과 <a href="/wiki/%EC%82%AC%EC%B2%9C_%EC%84%A0%EC%A7%84%EB%A6%AC%EC%99%9C%EC%84%B1" title="사천 선진리왜성">사천 선진리왜성</a>에 투입하였다. <a href="/wiki/%EC%8B%9C%EB%A7%88%EC%A6%88_%EC%9A%94%EC%8B%9C%ED%9E%88%EB%A1%9C" title="시마즈 요시히로">시마즈 요시히로</a>는 <a class="new" href="/w/index.php?title=%EA%B0%80%EC%99%80%EC%B9%B4%EB%AF%B8_%EB%8B%A4%EB%8B%A4%EB%85%B8%EB%A6%AC&amp;action=edit&amp;redlink=1" title="가와카미 다다노리 (없는 문서)">가와카미 다다노리</a>(川上忠実)에게 수백의 군사를 주어 사천고성을 지키게 해 대략 1 만석의 식량을 두었다. 이 시마즈군은 소수이면서도 완강하게 저항, 연합군을 도발했다. 그 후 동정군(東征軍)이 사천고성과 사천 선진리왜성에 압박을 가하자 시마즈는 사천고성을 포기하더라도 병력을 한곳에 집중시켜 조명연합군의 공세를 막기 위해 병력을 사천 선진리왜성으로 이동시키려 했다.</p>
<p>사천고성에서의 철수 준비를 하고 있던 <a href="/wiki/9%EC%9B%94_28%EC%9D%BC" title="9월 28일">9월 28일</a> 밤, 경상우병사 <a href="/wiki/%EC%A0%95%EA%B8%B0%EB%A3%A1" title="정기룡">정기룡</a>의 조선군이 사천고성을 포위하고 야습을 감행했다. 일본군은 성문을 열고 포위망을 돌파하려다가 많은 병력손실을 입고 사천 선진리왜성으로 철수했다. 조·명 연합군은 사천고성을 점령하고 <a href="/wiki/10%EC%9B%94_1%EC%9D%BC" title="10월 1일">10월 1일</a> 선진리왜성 공격을 시작했다.</p>
<p>전투 도중, <a class="new" href="/w/index.php?title=%EA%B0%80%EC%99%80%EC%B9%B4%EB%AF%B8_%EB%8B%A4%EB%8B%A4%EB%85%B8%EB%A6%AC&amp;action=edit&amp;redlink=1" title="가와카미 다다노리 (없는 문서)">가와카미 다다노리</a>는 <a class="new" href="/w/index.php?title=%EC%84%B8%ED%86%A0%EA%B5%AC%EC%B9%98_%EC%8B%9C%EA%B2%8C%ED%95%98%EB%A3%A8&amp;action=edit&amp;redlink=1" title="세토구치 시게하루 (없는 문서)">세토구치 시게하루</a>(瀬戸口重治)에게 명하여 적의 식량창고를 불태웠다. 연합군은 식량이 부족한 상황에서 식량창고가 불타버리자 한층 더 궁지에 빠지고, 단기 결전을 피할 수 없게 되었다. 시마즈 요시히로는 사천 선진리왜성의 뒤에 굳세고 튼튼한 진을 치고 복병을 배치했다. 요시히로는 연합군의 공격에 대량의 총을 사용하고 지뢰를 묻는 방법으로 연합군의 공격을 잘 방어했다. 또한 쇳조각이나 철정을 포탄 대신에 장전 한 대포도 사용했다. 연합군이 공격하다 지치자 요시히로는 복병을 출동시켜 적의 대열을 끊어 교란, 요시히로 본대도 공세로 전환했다. 연합군은 양식문제로 고민하고있었고, 요시히로군의 공세에 지쳐 철수하고 만다.</p>
