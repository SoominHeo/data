<p><b>사카이 시</b>(<span style="font-size: smaller;"><a href="/wiki/%EC%9D%BC%EB%B3%B8%EC%96%B4" title="일본어">일본어</a>:</span> <span lang="ja" xml:lang="ja">堺市</span>, 계시)는 <a href="/wiki/%EC%9D%BC%EB%B3%B8" title="일본">일본</a> <a href="/wiki/%EC%98%A4%EC%82%AC%EC%B9%B4_%EB%B6%80" title="오사카 부">오사카 부</a> 중남부에 있는 <a href="/wiki/%EC%8B%9C_(%ED%96%89%EC%A0%95_%EA%B5%AC%EC%97%AD)" title="시 (행정 구역)">시</a>이다. 중세 시대 이래로 일본의 가장 크고 중요한 항구 중 하나였다.</p>
<p><a href="/wiki/2005%EB%85%84" title="2005년">2005년</a> <a href="/wiki/2%EC%9B%94_1%EC%9D%BC" title="2월 1일">2월 1일</a>, 인접했던 미나미카와치 군(<span lang="ja" xml:lang="ja">南河內郡</span>) 미하라 정(<span lang="ja" xml:lang="ja">美原町</span>)을 편입함으로써 인구가 <a href="/wiki/%EC%A0%95%EB%A0%B9%EC%A7%80%EC%A0%95%EB%8F%84%EC%8B%9C" title="정령지정도시">정령지정도시</a> 승격의 요건인 80만명을 넘었으며, <a href="/wiki/2006%EB%85%84" title="2006년">2006년</a> <a href="/wiki/4%EC%9B%94_1%EC%9D%BC" title="4월 1일">4월 1일</a>에 15번째의 정령지정도시로 지정되어 7개의 구가 설치되었다. 한편, 인접한 <a href="/wiki/%EB%8B%A4%EC%B9%B4%EC%9D%B4%EC%8B%9C_%EC%8B%9C" title="다카이시 시">다카이시 시</a>(<span lang="ja" xml:lang="ja">高石市</span>), <a href="/wiki/%EC%98%A4%EC%82%AC%EC%B9%B4%EC%82%AC%EC%95%BC%EB%A7%88_%EC%8B%9C" title="오사카사야마 시">오사카사야마 시</a>(<span lang="ja" xml:lang="ja">大阪狭山市</span>)와의 통합이 거론되었으나, 두 시의 반대로 무산되었다.</p>
<p>사카이 시는 5세기 때 만들어진 열쇠 구멍 모양의 <a class="new" href="/w/index.php?title=%EA%B3%A0%EB%B6%84&amp;action=edit&amp;redlink=1" title="고분 (없는 문서)">고분</a>으로 유명하다. 이들 중 가장 유명한 것은 다이센 고분으로 <a href="/wiki/%EB%8B%8C%ED%86%A0%EC%BF%A0_%EC%B2%9C%ED%99%A9" title="닌토쿠 천황">닌토쿠 천황</a>의 무덤으로 여겨지고 면적 기준으로 세계에서 가장 큰 무덤이다. 한 때 <a class="mw-redirect" href="/wiki/%EA%B0%80%ED%83%80%EB%82%98" title="가타나">가타나</a>(사무라이 검)으로 알려졌고 현재 양질의 식칼로 유명하며 이것의 생산이 도시의 주요 산업이다.</p>

<h2><span class="mw-headline" id=".EC.A7.80.EB.A6.AC">지리</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%82%AC%EC%B9%B4%EC%9D%B4_%EC%8B%9C_(%EC%98%A4%EC%82%AC%EC%B9%B4_%EB%B6%80)&amp;action=edit&amp;section=1" title="부분 편집: 지리">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p><a href="/wiki/%EC%98%A4%EC%82%AC%EC%B9%B4_%EB%B6%80" title="오사카 부">오사카 부</a> 중앙부를 흐르는 <a class="new" href="/w/index.php?title=%EC%95%BC%EB%A7%88%ED%86%A0_%EA%B0%95&amp;action=edit&amp;redlink=1" title="야마토 강 (없는 문서)">야마토 강</a>의 하구에 위치해 있다. 야마토 강을 사이에 두고 <a href="/wiki/%EC%98%A4%EC%82%AC%EC%B9%B4_%EC%8B%9C" title="오사카 시">오사카 시</a>의 남쪽에 위치하며, 북동쪽은 <a href="/wiki/%EB%A7%88%EC%93%B0%EB%B0%94%EB%9D%BC_%EC%8B%9C" title="마쓰바라 시">마쓰바라 시</a>, 동쪽은 <a href="/wiki/%ED%95%98%EB%B9%84%ED%82%A4%EB%85%B8_%EC%8B%9C" title="하비키노 시">하비키노 시</a>, <a href="/wiki/%EB%8F%88%EB%8B%A4%EB%B0%94%EC%95%BC%EC%8B%9C_%EC%8B%9C" title="돈다바야시 시">돈다바야시 시</a>, 남동쪽은 <a href="/wiki/%EC%98%A4%EC%82%AC%EC%B9%B4%EC%82%AC%EC%95%BC%EB%A7%88_%EC%8B%9C" title="오사카사야마 시">오사카사야마 시</a>, <a href="/wiki/%EA%B0%80%EC%99%80%EC%B9%98%EB%82%98%EA%B0%80%EB%85%B8_%EC%8B%9C" title="가와치나가노 시">가와치나가노 시</a>, 남서쪽은 <a href="/wiki/%EC%9D%B4%EC%A6%88%EB%AF%B8_%EC%8B%9C_(%EC%98%A4%EC%82%AC%EC%B9%B4_%EB%B6%80)" title="이즈미 시 (오사카 부)">이즈미 시</a>, 서쪽은 <a href="/wiki/%EB%8B%A4%EC%B9%B4%EC%9D%B4%EC%8B%9C_%EC%8B%9C" title="다카이시 시">다카이시 시</a>에 접하고 있다.</p>
<h2><span class="mw-headline" id=".EC.97.AD.EC.82.AC">역사</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%82%AC%EC%B9%B4%EC%9D%B4_%EC%8B%9C_(%EC%98%A4%EC%82%AC%EC%B9%B4_%EB%B6%80)&amp;action=edit&amp;section=2" title="부분 편집: 역사">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p><a href="/wiki/%EB%AC%B4%EB%A1%9C%EB%A7%88%EC%B9%98_%EC%8B%9C%EB%8C%80" title="무로마치 시대">무로마치 시대</a> 이후 발달하였으며, <a href="/wiki/%EC%95%BC%EB%A7%88%ED%86%A0_%EA%B5%AD" title="야마토 국">야마토 국</a>과 해안 지역을 연결하는 역할을 해왔다. 상인들에 의해 자치적으로 운영되었으며, 승려 <a class="new" href="/w/index.php?title=%EC%9E%87%ED%81%90_%EC%86%8C%EC%A4%80&amp;action=edit&amp;redlink=1" title="잇큐 소준 (없는 문서)">잇큐 소준</a>, 선교사 <a class="mw-redirect" href="/wiki/%ED%94%84%EB%9E%80%EC%8B%9C%EC%8A%A4_%ED%95%98%EB%B9%84%EC%97%90%EB%A5%B4" title="프란시스 하비에르">프란시스 하비에르</a> 등 종교인들이 거주하거나 방문하기도 하였다. <a href="/wiki/%EC%84%BC%EA%B3%A0%EC%BF%A0_%EC%8B%9C%EB%8C%80" title="센고쿠 시대">센고쿠 시대</a> 이후 유럽인과의 소통이 잦아지면서, 사카이는 화기의 제조 기지가 되었으며, 화기를 얻기 위해 사카이를 자주 방문하던 <a href="/wiki/%EC%98%A4%EB%8B%A4_%EB%85%B8%EB%B6%80%EB%82%98%EA%B0%80" title="오다 노부나가">오다 노부나가</a>는 사카이로부터 자치권을 빼앗으려 하였다. 사카이의 시민들은 그의 명령을 거절하고 노부나가의 군대에 필사적인 저항을 하였으나, 끝내 사카이를 점령한 노부나가는 사카이를 불태웠다. 하지만, 사카이는 노부나가의 뒤를 이은 <a href="/wiki/%EB%8F%84%EC%9A%94%ED%86%A0%EB%AF%B8_%ED%9E%88%EB%8D%B0%EC%9A%94%EC%8B%9C" title="도요토미 히데요시">도요토미 히데요시</a>의 통치 하에 다시 번영하는 도시가 되었다.</p>
<p><a class="new" href="/w/index.php?title=%EC%9D%BC%EB%B3%B8_%EB%8B%A4%EB%8F%84&amp;action=edit&amp;redlink=1" title="일본 다도 (없는 문서)">일본 다도</a>를 정립한 것으로 유명한 <a href="/wiki/%EC%84%BC%EB%85%B8_%EB%A6%AC%ED%81%90" title="센노 리큐">센노 리큐</a>도 원래 사카이의 상인이었다. 다도와 선종 불교의 밀접한 관계와 시민들의 부유함 덕분에 사카이는 일본 다도의 중심지 중 하나가 되었다. <a href="/wiki/%EC%97%90%EB%8F%84_%EC%8B%9C%EB%8C%80" title="에도 시대">에도 시대</a>의 사카이는 여전히 중요한 항구였으나, <a class="mw-redirect" href="/wiki/%EB%8F%84%EC%BF%A0%EA%B0%80%EC%99%80_%EB%A7%89%EB%B6%80" title="도쿠가와 막부">도쿠가와 막부</a>의 쇄국 정책 때문에 오직 국내의 항구로만 한정되었다. 막부 말기에 서양인들은 다시 사카이에 상륙했으나 일본인들과 외국인들 사이의 상호간의 무지 때문에 비극적인 사건이 발생했다. 뒤플렉스 호의 프랑스 선원들과 사카이 시민들이 충돌했다. 프랑스인 몇 명이 사망했고 일본인 책임자들은 할복할 것을 선고받았다. 이 사건을 <a class="new" href="/w/index.php?title=%EC%82%AC%EC%B9%B4%EC%9D%B4_%EC%82%AC%EA%B1%B4&amp;action=edit&amp;redlink=1" title="사카이 사건 (없는 문서)">사카이 사건</a>(堺事件)으로 부른다.</p>
<p>근대 이후에는 <a href="/wiki/%EC%98%A4%EC%82%AC%EC%B9%B4_%EC%8B%9C" title="오사카 시">오사카 시</a> 근교의 주택도시, 산업도시로서 발전해 왔으며, 특히 1960년대 이후에는 <a href="/wiki/%EC%98%A4%EC%82%AC%EC%B9%B4_%EB%A7%8C" title="오사카 만">오사카 만</a>(<span lang="ja" xml:lang="ja">大阪湾</span>) 연안에 <a class="mw-redirect" href="/wiki/%EC%8B%A0%EC%9D%BC%EB%B3%B8%EC%A0%9C%EC%B2%A0" title="신일본제철">신일본제철</a>(新日本製鐵) 사카이 제철소가 입지했고, 또 교외에는 <a href="/wiki/%EC%9D%B4%EC%A6%88%EB%AF%B8_%EA%B5%AD" title="이즈미 국">센보쿠</a>(泉北) 뉴타운이 개발됨으로써 오사카 부에서 두 번째로 인구가 많은 도시가 되었다.</p>
