<p><b>사카테카스 주</b>(<span style="font-size: smaller;"><a href="/wiki/%EC%8A%A4%ED%8E%98%EC%9D%B8%EC%96%B4" title="스페인어">스페인어</a>:</span> <span lang="es" xml:lang="es">Zacatecas</span>)는 <a href="/wiki/%EB%A9%95%EC%8B%9C%EC%BD%94" title="멕시코">멕시코</a> 중북부에 위치한 주로 주도는 <a href="/wiki/%EC%82%AC%EC%B9%B4%ED%85%8C%EC%B9%B4%EC%8A%A4" title="사카테카스">사카테카스</a>이며 면적은 75,284㎢, 인구는 1,514,618명(2012년 기준), 인구 밀도는 20명/㎢이다. 1823년 12월 23일에 신설되었으며 58개 지방 자치체를 관할한다. 북서쪽으로는 <a href="/wiki/%EB%91%90%EB%9E%91%EA%B3%A0_%EC%A3%BC" title="두랑고 주">두랑고 주</a>, 북쪽으로는 <a href="/wiki/%EC%BD%94%EC%95%84%EC%9A%B0%EC%9D%BC%EB%9D%BC_%EC%A3%BC" title="코아우일라 주">코아우일라 주</a>, 서쪽으로는 <a href="/wiki/%EB%82%98%EC%95%BC%EB%A6%AC%ED%8A%B8_%EC%A3%BC" title="나야리트 주">나야리트 주</a>, 동쪽으로는 <a href="/wiki/%EC%82%B0%EB%A3%A8%EC%9D%B4%EC%8A%A4%ED%8F%AC%ED%86%A0%EC%8B%9C_%EC%A3%BC" title="산루이스포토시 주">산루이스포토시 주</a>와 <a href="/wiki/%EB%88%84%EC%97%90%EB%B3%B4%EB%A0%88%EC%98%A8_%EC%A3%BC" title="누에보레온 주">누에보레온 주</a>, 남쪽으로는 <a href="/wiki/%ED%95%A0%EB%A6%AC%EC%8A%A4%EC%BD%94_%EC%A3%BC" title="할리스코 주">할리스코 주</a>, <a href="/wiki/%EA%B3%BC%EB%82%98%ED%9B%84%EC%95%84%ED%86%A0_%EC%A3%BC" title="과나후아토 주">과나후아토 주</a>, <a href="/wiki/%EC%95%84%EA%B3%BC%EC%8A%A4%EC%B9%BC%EB%A6%AC%EC%97%94%ED%85%8C%EC%8A%A4_%EC%A3%BC" title="아과스칼리엔테스 주">아과스칼리엔테스 주</a>와 접한다.</p>
