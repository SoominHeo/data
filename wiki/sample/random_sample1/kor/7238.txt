<p><b>사토 나오타케</b> (<span style="font-size: smaller;"><a href="/wiki/%EC%9D%BC%EB%B3%B8%EC%96%B4" title="일본어">일본어</a>:</span> <span lang="ja"><span lang="ja"><ruby><rb>佐藤 尚武</rb> <rp>(</rp><rt>さとう なおたけ</rt><rp>)</rp></ruby></span></span>, <a href="/wiki/1882%EB%85%84" title="1882년">1882년</a> <a href="/wiki/10%EC%9B%94_30%EC%9D%BC" title="10월 30일">10월 30일</a> ~ <a href="/wiki/1971%EB%85%84" title="1971년">1971년</a> <a href="/wiki/12%EC%9B%94_18%EC%9D%BC" title="12월 18일">12월 18일</a>) 는 <a href="/wiki/%EC%9D%BC%EB%B3%B8" title="일본">일본</a>의 <a href="/wiki/%EC%99%B8%EA%B5%90%EA%B4%80" title="외교관">외교관</a>이자 <a class="mw-redirect" href="/wiki/%EC%A0%95%EC%B9%98%EC%9D%B8" title="정치인">정치인</a>으로, <a href="/wiki/%ED%95%98%EC%95%BC%EC%8B%9C_%EC%84%BC%EC%A3%BC%EB%A1%9C" title="하야시 센주로">하야시 센주로</a> 내각에서 외무대신을 지냈다. 전후에는 <a href="/wiki/%EC%B0%B8%EC%9D%98%EC%9B%90_(%EC%9D%BC%EB%B3%B8)" title="참의원 (일본)">참의원</a> 의장 등을 지냈으며, <a href="/wiki/%EC%A0%9C2%EC%B0%A8_%EC%84%B8%EA%B3%84_%EB%8C%80%EC%A0%84" title="제2차 세계 대전">제2차 세계 대전</a> 말기 <a class="mw-redirect" href="/wiki/%EC%86%8C%EB%B9%84%EC%97%90%ED%8A%B8_%EC%97%B0%EB%B0%A9" title="소비에트 연방">소비에트 연방</a>의 대일 선전포고 당시, 주 <a class="mw-redirect" href="/wiki/%EC%86%8C%EB%B9%84%EC%97%90%ED%8A%B8_%EC%97%B0%EB%B0%A9" title="소비에트 연방">소비에트 연방</a> 대사이기도 했다.</p>
<h2><span class="mw-headline" id=".EC.9D.B8.EB.AC.BC">인물</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%82%AC%ED%86%A0_%EB%82%98%EC%98%A4%ED%83%80%EC%BC%80&amp;action=edit&amp;section=1" title="부분 편집: 인물">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>사토는 1882년, <a href="/wiki/%EC%98%A4%EC%82%AC%EC%B9%B4_%EB%B6%80" title="오사카 부">오사카 부</a>에서 <a class="mw-redirect" href="/wiki/%EC%93%B0%EA%B0%80%EB%A3%A8_%EB%B2%88" title="쓰가루 번">쓰가루 번</a> 무사의 아들로 태어났다. 1904년 도쿄 상업학교 (지금의 <a href="/wiki/%ED%9E%88%ED%86%A0%EC%93%B0%EB%B0%94%EC%8B%9C_%EB%8C%80%ED%95%99" title="히토쓰바시 대학">히토쓰바시 대학</a>) 전과를 졸업하고, 1905년 외교관 및 영사관 시험에 합격해 <a href="/wiki/%EC%9D%BC%EB%B3%B8_%EC%99%B8%EB%AC%B4%EC%84%B1" title="일본 외무성">외무성</a>에 들어갔다.</p>
<p><a class="mw-redirect" href="/wiki/%ED%95%98%EC%96%BC%EB%B9%88" title="하얼빈">하얼빈</a> 총영사, <a href="/wiki/%EA%B5%AD%EC%A0%9C_%EC%97%B0%EB%A7%B9" title="국제 연맹">국제 연맹</a> 사무국장을 거쳐 1929년 <a class="mw-redirect" href="/wiki/%EB%9F%B0%EB%8D%98_%ED%95%B4%EA%B5%B0_%EA%B5%B0%EC%B6%95_%EC%A1%B0%EC%95%BD" title="런던 해군 군축 조약">런던 해군 군축 조약</a> 사무총장, 1930년 주 <a href="/wiki/%EB%B2%A8%EA%B8%B0%EC%97%90" title="벨기에">벨기에</a> 특명 전권대사, 1933년 주 <a href="/wiki/%ED%94%84%EB%9E%91%EC%8A%A4" title="프랑스">프랑스</a> 특명 전권대사, 1937년 일본의 외무대신, 1938년 일본 외무성 외교 고문, 1940년 주 <a href="/wiki/%EC%9D%B4%ED%83%88%EB%A6%AC%EC%95%84_%EC%99%95%EA%B5%AD_(1861%EB%85%84_~_1946%EB%85%84)" title="이탈리아 왕국 (1861년 ~ 1946년)">이탈리아</a> 특명 전권대사를 지내고, 1942년 <a href="/wiki/%EB%8F%84%EA%B3%A0_%EC%8B%9C%EA%B2%8C%EB%85%B8%EB%A6%AC" title="도고 시게노리">도고 시게노리</a> 당시 일본 외무대신의 요청으로 주 <a class="mw-redirect" href="/wiki/%EC%86%8C%EB%B9%84%EC%97%90%ED%8A%B8_%EC%97%B0%EB%B0%A9" title="소비에트 연방">소비에트 연방</a> 특명 전권대사가 되었다.</p>
<p>외무대신 시절에는 제국 의회에서 전쟁 회피를 주장하였으며, 그의 주 소비에트 연방 대사 임명은, 일본의 전세 악화 가운데 <a class="mw-redirect" href="/wiki/%EC%9D%BC%EC%86%8C_%EC%A4%91%EB%A6%BD_%EC%A1%B0%EC%95%BD" title="일소 중립 조약">일소 중립 조약</a>을 체결했던 <a class="mw-redirect" href="/wiki/%EC%86%8C%EB%B9%84%EC%97%90%ED%8A%B8_%EC%97%B0%EB%B0%A9" title="소비에트 연방">소비에트 연방</a>의 중개를 통해 연합국과의 평화 모색을 위한 임무 중 하나로 간주되었다. 하지만, 사토는 이미 전쟁의 대세가 정해진 이상 소비에트 연방이 중개역에 설 가능성은 적다고 판단하였고, 조기 종전을 재촉하는 기밀 전보를 도쿄로 돌려보냈다. 결국 1945년 8월 8일, <a class="mw-redirect" href="/wiki/%ED%81%AC%EB%A0%98%EB%A6%B0" title="크렘린">크렘린</a>에서 소비에트 연방의 외무장관 <a href="/wiki/%EB%B1%8C%EC%B2%B4%EC%8A%AC%EB%9D%BC%ED%94%84_%EB%AA%B0%EB%A1%9C%ED%86%A0%ED%94%84" title="뱌체슬라프 몰로토프">뱌체슬라프 몰로토프</a>에 의해 대일 선전 포고가 선언되었으며, 일제가 그 해 8월 15일 항복하여 전쟁은 막을 내렸다.</p>
<p>전후에는 <a href="/wiki/%EC%B0%B8%EC%9D%98%EC%9B%90_(%EC%9D%BC%EB%B3%B8)" title="참의원 (일본)">참의원</a> 의원에 당선되었으며, 1949년부터 1953년까지 참의원 의장을 지냈다.</p>
