<p><b>사토 데쓰타로</b> (<span style="font-size: smaller;"><a href="/wiki/%EC%9D%BC%EB%B3%B8%EC%96%B4" title="일본어">일본어</a>:</span> <span lang="ja"><span lang="ja"><ruby><rb>佐藤 鐡太郎</rb> <rp>(</rp><rt>さとう てつたろう</rt><rp>)</rp></ruby></span></span>, <a href="/wiki/1866%EB%85%84" title="1866년">1866년</a> <a href="/wiki/8%EC%9B%94_22%EC%9D%BC" title="8월 22일">8월 22일</a> ~ <a href="/wiki/1942%EB%85%84" title="1942년">1942년</a> <a href="/wiki/3%EC%9B%94_4%EC%9D%BC" title="3월 4일">3월 4일</a>)는 <a href="/wiki/%EC%9D%BC%EB%B3%B8" title="일본">일본</a>의 <a href="/wiki/%EC%9D%BC%EB%B3%B8_%EC%A0%9C%EA%B5%AD_%ED%95%B4%EA%B5%B0" title="일본 제국 해군">해군</a> <a href="/wiki/%EA%B5%B0%EC%9D%B8" title="군인">군인</a>이다. <a class="mw-redirect" href="/wiki/%EC%9D%BC%EB%B3%B8_%EA%B7%80%EC%A1%B1%EC%9B%90" title="일본 귀족원">일본 귀족원</a> 의원, <a href="/wiki/%EA%B0%80%EC%BF%A0%EC%8A%88%EC%9D%B8" title="가쿠슈인">가쿠슈인</a> 교수를 지내기도 했다.</p>
<p><a href="/wiki/%EC%87%BC%EB%82%98%EC%9D%B4_%EB%B2%88" title="쇼나이 번">쇼나이</a> <a href="/wiki/%EB%B2%88%EC%82%AC" title="번사">번사</a> 히라무키 유지로의 아들이지만 자라면서 사토 야스유키의 양자로 들어갔다. <a class="mw-redirect mw-disambig" href="/wiki/%ED%95%B4%EA%B5%B0%EB%B3%91%ED%95%99%EA%B5%90" title="해군병학교">해군병학교</a>를 14기로 졸업하고 제국 해군에 들어갔다. <a href="/wiki/%EC%B2%AD%EC%9D%BC_%EC%A0%84%EC%9F%81" title="청일 전쟁">청일 전쟁</a> 당시 포함 아카기 항해장으로 참전했는데, 이 때 함장 <a class="new" href="/w/index.php?title=%EC%82%AC%EC%B9%B4%EB%AA%A8%ED%86%A0_%ED%95%98%EC%B9%98%EB%A1%9C%ED%83%80&amp;action=edit&amp;redlink=1" title="사카모토 하치로타 (없는 문서)">사카모토 하치로타</a>가 전사하자 대신 아카기를 지휘했다. 그 후 <a class="new" href="/w/index.php?title=%ED%95%B4%EA%B5%B0_%EB%8C%80%ED%95%99%EA%B5%90&amp;action=edit&amp;redlink=1" title="해군 대학교 (없는 문서)">해군 대학교</a> 교관 등을 거쳐 <a href="/wiki/%EB%9F%AC%EC%9D%BC_%EC%A0%84%EC%9F%81" title="러일 전쟁">러일 전쟁</a> 때는 <a class="new" href="/w/index.php?title=%EC%9A%B0%EC%97%90%EB%AC%B4%EB%9D%BC_%ED%9E%88%EC%BD%94%EB%85%B8%EC%A1%B0&amp;action=edit&amp;redlink=1" title="우에무라 히코노조 (없는 문서)">우에무라 히코노조</a> (上村彦之丞)가 이끄는 제2함대의 선임 참모로 참전했다. 전쟁 후 해군대학교 교관을 다시 맡았으며, 군령부 차장, 해군대학교 교장, <a href="/wiki/%EB%A7%88%EC%9D%B4%EC%A6%88%EB%A3%A8_%EC%8B%9C" title="마이즈루 시">마이즈루</a> 진수부 사령장관 등을 지냈지만, 당시 총리였던 <a href="/wiki/%EA%B0%80%ED%86%A0_%EB%8F%84%EB%AA%A8%EC%82%AC%EB%B6%80%EB%A1%9C" title="가토 도모사부로">가토 도모사부로</a>와 군비 감축을 두고 다투다 결국 <a href="/wiki/1923%EB%85%84" title="1923년">1923년</a> 퇴역했다. 퇴역 후 가쿠슈인 교수를 맡아 <a href="/wiki/%EC%87%BC%EC%99%80_%EC%B2%9C%ED%99%A9" title="쇼와 천황">쇼와 천황</a>을 가르쳤으며, <a href="/wiki/1934%EB%85%84" title="1934년">1934년</a> 일본 귀족원 의원이 되었다. 만년에는 <a href="/wiki/%EB%8F%84%EC%A1%B0_%ED%9E%88%EB%8D%B0%ED%82%A4" title="도조 히데키">도조 히데키</a>의 아버지인 <a href="/wiki/%EB%8F%84%EC%A1%B0_%ED%9E%88%EB%8D%B0%EB%85%B8%EB%A6%AC" title="도조 히데노리">도조 히데노리</a> 등처럼 전쟁사를 연구했다.</p>
