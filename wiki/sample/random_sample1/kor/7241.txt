<p><b>사투마레 주</b>(<span style="font-size: smaller;"><a href="/wiki/%EB%A3%A8%EB%A7%88%EB%8B%88%EC%95%84%EC%96%B4" title="루마니아어">루마니아어</a>:</span> <span lang="ro" xml:lang="ro">Judeţul Satu Mare</span>, <span style="font-size: smaller;"><a href="/wiki/%ED%97%9D%EA%B0%80%EB%A6%AC%EC%96%B4" title="헝가리어">헝가리어</a>:</span> <span lang="hu" xml:lang="hu">Szatmár megye</span>)는 <a href="/wiki/%EB%A3%A8%EB%A7%88%EB%8B%88%EC%95%84" title="루마니아">루마니아</a> 북서부 <a href="/wiki/%ED%8A%B8%EB%9E%80%EC%8B%A4%EB%B0%94%EB%8B%88%EC%95%84" title="트란실바니아">트란실바니아</a> 지방에 위치한 주로, 주도는 <a href="/wiki/%EC%82%AC%ED%88%AC%EB%A7%88%EB%A0%88" title="사투마레">사투마레</a>이며 면적은 4,418㎢, 인구는 366,270명(2002년 기준), 인구 밀도는 83명/㎢, <a href="/wiki/ISO_3166-2" title="ISO 3166-2">ISO 3166-2</a> 코드는 RO-SM이다. 동쪽으로는 <a href="/wiki/%EB%A7%88%EB%9D%BC%EB%AC%B4%EB%A0%88%EC%8A%88_%EC%A3%BC" title="마라무레슈 주">마라무레슈 주</a>, 서쪽으로는 <a href="/wiki/%ED%97%9D%EA%B0%80%EB%A6%AC" title="헝가리">헝가리</a> <a href="/wiki/%EC%84%9C%EB%B3%BC%EC%B9%98%EC%84%9C%ED%8A%B8%EB%A7%88%EB%A5%B4%EB%B2%A0%EB%A0%88%EA%B7%B8_%EC%A3%BC" title="서볼치서트마르베레그 주">서볼치서트마르베레그 주</a>, 북쪽으로는 <a href="/wiki/%EC%9A%B0%ED%81%AC%EB%9D%BC%EC%9D%B4%EB%82%98" title="우크라이나">우크라이나</a> <a href="/wiki/%EC%9E%90%EC%B9%B4%EB%A5%B4%ED%8C%8C%ED%83%B8_%EC%A3%BC" title="자카르파탸 주">자카르파탸 주</a>, 남쪽으로는 <a href="/wiki/%EB%B9%84%ED%98%B8%EB%A5%B4_%EC%A3%BC" title="비호르 주">비호르 주</a>, <a href="/wiki/%EC%84%A4%EB%9D%BC%EC%A7%80_%EC%A3%BC" title="설라지 주">설라지 주</a>와 접하며 2개 시와 4개 마을, 59개 지방 자치체를 관할한다.</p>
<p>주 전체 인구 가운데 59%는 <a href="/wiki/%EB%A3%A8%EB%A7%88%EB%8B%88%EC%95%84%EC%9D%B8" title="루마니아인">루마니아인</a>, 35%는 <a href="/wiki/%ED%97%9D%EA%B0%80%EB%A6%AC%EC%9D%B8" title="헝가리인">헝가리인</a>, 3.5%는 <a href="/wiki/%EB%A1%AC_(%EB%AF%BC%EC%A1%B1)" title="롬 (민족)">로마인</a>, 1.7%는 <a href="/wiki/%EB%8F%85%EC%9D%BC%EC%9D%B8" title="독일인">독일인</a>이며 <a href="/wiki/%EB%A3%A8%EC%8B%A0%EC%9D%B8" title="루신인">루신인</a>과 <a href="/wiki/%EC%9A%B0%ED%81%AC%EB%9D%BC%EC%9D%B4%EB%82%98%EC%9D%B8" title="우크라이나인">우크라이나인</a>, <a href="/wiki/%EC%8A%AC%EB%A1%9C%EB%B0%94%ED%82%A4%EC%95%84%EC%9D%B8" title="슬로바키아인">슬로바키아인</a> 등 그 외의 소수 민족도 거주한다. 주의 주요 산업은 직물 산업, 기계 부품과 자동차 부품 산업, 식품 산업, 임업과 가구 제조업이다.</p>
