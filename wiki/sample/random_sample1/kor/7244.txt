<p><b>사파르무라트 아타예비치 니야조프</b>(<span style="font-size: smaller;"><a href="/wiki/%ED%88%AC%EB%A5%B4%ED%81%AC%EB%A9%98%EC%96%B4" title="투르크멘어">투르크멘어</a>:</span> <span lang="tk" xml:lang="tk">Saparmyrat Ataýewiç Nyýazow</span> <small>사파르미라트 아타예비치 니야조프</small>, <span style="font-size: smaller;"><a href="/wiki/%EB%9F%AC%EC%8B%9C%EC%95%84%EC%96%B4" title="러시아어">러시아어</a>:</span> <span lang="ru" xml:lang="ru">Сапармурат Атаевич Ниязов</span> <small>사파르무라트 아타예비치 니야조프<sup>[<a href="/wiki/%EC%9C%84%ED%82%A4%EB%B0%B1%EA%B3%BC:%EB%9F%AC%EC%8B%9C%EC%95%84%EC%96%B4%EC%9D%98_%ED%95%9C%EA%B8%80_%ED%91%9C%EA%B8%B0" title="위키백과:러시아어의 한글 표기">*</a>]</sup></small>, <span style="font-size: smaller;"><a href="/wiki/%EB%AC%B8%ED%99%94%EC%96%B4" title="문화어">문화어</a>:</span> 싸빠르무라뜨 니야조브, <a href="/wiki/1940%EB%85%84" title="1940년">1940년</a> <a href="/wiki/2%EC%9B%94_19%EC%9D%BC" title="2월 19일">2월 19일</a> ~ <a href="/wiki/2006%EB%85%84" title="2006년">2006년</a> <a href="/wiki/12%EC%9B%94_21%EC%9D%BC" title="12월 21일">12월 21일</a>)는 <a href="/wiki/%ED%88%AC%EB%A5%B4%ED%81%AC%EB%A9%94%EB%8B%88%EC%8A%A4%ED%83%84" title="투르크메니스탄">투르크메니스탄</a>의 <a href="/wiki/%ED%88%AC%EB%A5%B4%ED%81%AC%EB%A9%94%EB%8B%88%EC%8A%A4%ED%83%84%EC%9D%98_%EB%8C%80%ED%86%B5%EB%A0%B9" title="투르크메니스탄의 대통령">대통령</a>이다. <a href="/wiki/1940%EB%85%84" title="1940년">1940년</a> 지금의 <a href="/wiki/%ED%88%AC%EB%A5%B4%ED%81%AC%EB%A9%94%EB%8B%88%EC%8A%A4%ED%83%84" title="투르크메니스탄">투르크메니스탄</a>의 수도 <a href="/wiki/%EC%95%84%EC%8B%9C%EA%B0%80%EB%B0%94%ED%8A%B8" title="아시가바트">아시가바트</a>에서 10km 정도 떨어진 작은 <a href="/wiki/%EB%A7%88%EC%9D%84" title="마을">마을</a>인 기프야크에서 태어났으며 어릴 적 부모를 모두 여의고 고아 생활을 하였다. 투르크메니스탄 정부의 공식 입장에 따르면 니야조프의 아버지는 <a href="/wiki/%EC%A0%9C2%EC%B0%A8_%EC%84%B8%EA%B3%84_%EB%8C%80%EC%A0%84" title="제2차 세계 대전">제2차 세계 대전</a> 때 <a href="/wiki/%EB%82%98%EC%B9%98_%EB%8F%85%EC%9D%BC" title="나치 독일">나치 독일</a>에 저항하다 죽었고 나머지 가족들은 <a href="/wiki/1948%EB%85%84" title="1948년">1948년</a> <a href="/wiki/%EC%95%84%EC%8B%9C%EA%B0%80%EB%B0%94%ED%8A%B8" title="아시가바트">아시가바트</a> <a href="/wiki/%EC%A7%80%EC%A7%84" title="지진">지진</a>(<span style="font-size: smaller;"><a href="/wiki/%EC%98%81%EC%96%B4" title="영어">영어</a>:</span> <span lang="en" xml:lang="en"><a class="extiw" href="https://en.wikipedia.org/wiki/1948_Ashgabat_earthquake" title="en:1948 Ashgabat earthquake">en:1948 Ashgabat earthquake</a></span>)으로 죽었다고 언급되어 있다. 그 후 대학 생활을 마친 뒤 1970년 <a href="/wiki/%EC%86%8C%EB%A0%A8_%EA%B3%B5%EC%82%B0%EB%8B%B9" title="소련 공산당">공산당</a>에 가입하여 1985년부터 1990년까지 <a href="/wiki/%ED%88%AC%EB%A5%B4%ED%81%AC%EB%A9%98_%EC%86%8C%EB%B9%84%EC%97%90%ED%8A%B8_%EC%82%AC%ED%9A%8C%EC%A3%BC%EC%9D%98_%EA%B3%B5%ED%99%94%EA%B5%AD" title="투르크멘 소비에트 사회주의 공화국">투르크멘 소비에트 사회주의 공화국</a>의 당 서기장을 지낸 바 있다.</p>
<p><a href="/wiki/1990%EB%85%84" title="1990년">1990년</a> 투르크멘 의회의 결의안으로 그는 단독 후보로 대통령 선거에 출마, 당선된다. 1991년 투르크메니스탄 공화국의 독립이 있은 직후 1992년 이 공화국의 대통령으로 다시 출마 당선된다. 그 뒤 1999년 종신 대통령으로 다시 당선되었다. 2006년 12월 21일 새벽에 심장마비로 세상을 떠났다.</p>
<h2><span class="mw-headline" id=".EA.B8.B0.ED.96.89">기행</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%82%AC%ED%8C%8C%EB%A5%B4%EB%AC%B4%EB%9D%BC%ED%8A%B8_%EB%8B%88%EC%95%BC%EC%A1%B0%ED%94%84&amp;action=edit&amp;section=1" title="부분 편집: 기행">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>니야조프는 <a href="/wiki/%ED%88%AC%EB%A5%B4%ED%81%AC%EB%A9%94%EB%8B%88%EC%8A%A4%ED%83%84" title="투르크메니스탄">투르크메니스탄</a>의 기후를 무시한 채 1,000명이 들어갈 수 있는 크기의 얼음 궁전을 지으라고 지시하였으며 실제로 수도 <a href="/wiki/%EC%95%84%EC%8B%9C%EA%B0%80%EB%B0%94%ED%8A%B8" title="아시가바트">아시가바트</a> 외곽의 산악 지대에 얼음 궁전을 짓기 위한 조사가 실시되었다.</p>
<p><a href="/wiki/2001%EB%85%84" title="2001년">2001년</a>에는 국민 정서에 맞지 않는다는 이유로 <a href="/wiki/%EB%B0%9C%EB%A0%88" title="발레">발레</a>와 <a href="/wiki/%EC%98%A4%ED%8E%98%EB%9D%BC" title="오페라">오페라</a>를 금지시켰으며 <a href="/wiki/2005%EB%85%84" title="2005년">2005년</a>에는 <a href="/wiki/%EB%A6%BD%EC%8B%B1%ED%81%AC" title="립싱크">립싱크</a>를 금지시키기도 하였다.</p>
<p>또한 <a href="/wiki/2003%EB%85%84" title="2003년">2003년</a> <a href="/wiki/6%EC%9B%94" title="6월">6월</a>에는 해외 학위와 국가 발전은 양립할 수 없다는 이유로 <a href="/wiki/1993%EB%85%84" title="1993년">1993년</a> 이후 외국에서 학위를 취득한 교사, 의사 등을 해고, 추방하도록 명령하였다. 이로 인해 전문인력이 부족해져 학교나 병원 등이 큰 피해를 입었다.</p>
<p>또한 고등교육 과정을 2년으로 단축시키고 학교 수도 절반으로 줄였다. 이로 인해 교육계는 부정 부패가 만연하였다. 그러나 일반 교과목의 쇄신은 일어나지 않았으며 오히려 자신을 신격화하여 국민들이 숭배하도록 유도하기도 했는데, 그는 자신이 <b>튀르크멘바시</b>(Türkmenbaşy, <a href="/wiki/%ED%88%AC%EB%A5%B4%ED%81%AC%EB%A9%98%EC%9D%B8" title="투르크멘인">투르크멘인</a>의 아버지)임을 자칭했다. (원래 자신을 아타튀르크멘(Atatürkmen)으로 칭했으나 <a href="/wiki/%EB%AC%B4%EC%8A%A4%ED%83%80%ED%8C%8C_%EC%BC%80%EB%A7%90_%EC%95%84%ED%83%80%ED%8A%80%EB%A5%B4%ED%81%AC" title="무스타파 케말 아타튀르크">무스타파 케말 아타튀르크</a>를 연상시킨다는 이유로 <a href="/wiki/%ED%84%B0%ED%82%A4" title="터키">터키</a> 정부가 항의해 튀르크멘바시로 바꿨다는 일화도 있다) 또한 <b>루흐나마</b>(영혼의 책)라는 경전을 발간한 뒤에는 이를 학교 교육 과정에 넣어 의무적으로 배우게 하였다.</p>
<p>또한 이로 인해 공석이 생긴 의료 인력을 군인이 대체하는 상황에 이르렀는데, 그는 오히려 이를 의료비 절감을 가져온 혁신적 정치라고 주장하였다.</p>
<p>같은 해 <a href="/wiki/2%EC%9B%94" title="2월">2월</a>에는 턱수염과 장발을 금지하였으며 <a href="/wiki/4%EC%9B%94" title="4월">4월</a>에는 금니도 금지시켰다. <a href="/wiki/1997%EB%85%84" title="1997년">1997년</a>에는 자신이 심장 수술 이후에 <a href="/wiki/%EB%8B%B4%EB%B0%B0" title="담배">담배</a>를 끊은 이후 공직자의 흡연과 공공장소에서 흡연을 금지시켰으며 건강이 악화되자 국민들이 내게 기도를 너무 많이 해 힘들다며 국민들에게 기도를 줄이라고 권고하기도 했다.</p>
<p>지방 사람들은 책을 읽지 않는다면서 수도와 대학을 제외한 지역의 <a href="/wiki/%EB%8F%84%EC%84%9C%EA%B4%80" title="도서관">도서관</a>을 폐쇄하기도 하였다. <a href="/wiki/%EB%AA%A8%EC%8A%A4%ED%81%AC" title="모스크">모스크</a>에는 루흐나마가 <a class="mw-redirect" href="/wiki/%EC%BF%A0%EB%9E%80" title="쿠란">쿠란</a>과 함께 놓이기도 하였으며 운전면허 시험에서도 이를 출제하였다. 이 때문에 운전면허 시험 시간도 16시간에 달하였다.</p>
<p><a href="/wiki/2002%EB%85%84" title="2002년">2002년</a>에는 자신과 어머니 이름을 따서 각 달의 이름을 새로 붙였으며 1년도 12달이 아니라 8달이었다. 인생을 12년 주기로 나눈 <b>니야조프 분류법</b>에 따르면 노년층은 85세에서 시작하며 73세는 <b>지혜의 나이</b>, 61세는 <b>영감의 나이</b>이다. 또 니야조프는 최근 쓴 시에서 자신을 "<b>투르크멘의 정신이며 황금시대를 가져온 구세주</b>"로 묘사했다. 같은 해 <a href="/wiki/2%EC%9B%94_18%EC%9D%BC" title="2월 18일">2월 18일</a> 자신의 62번째 생일 기념행사 때 62세에 사망한 무함마드에 경의를 표한다는 명목으로 모든 국민은 62세가 되는 날 정부 수당과 함께 3일간의 휴가를 갖게 될 것이다고 선언하였다.</p>
<p><a href="/wiki/1997%EB%85%84" title="1997년">1997년</a>에는 새로운 <a href="/wiki/%ED%88%AC%EB%A5%B4%ED%81%AC%EB%A9%94%EB%8B%88%EC%8A%A4%ED%83%84%EC%9D%98_%EA%B5%AD%EA%B0%80" title="투르크메니스탄의 국가">투르크메니스탄의 국가</a>의 가사를 직접 작사했으며 이 국가는 첫 소절이 "튀르크멘바시가 만든 위대한 건조물"(Türkmenbaşyň guran beýik binasy)로 시작되었다. 이 국가는 이후 <a href="/wiki/2008%EB%85%84" title="2008년">2008년</a>에 해당 부분이 개사되었다.</p>
<p><a href="/wiki/2005%EB%85%84" title="2005년">2005년</a>에는 <a href="/wiki/%EC%A7%80%EA%B5%AC_%EC%98%A8%EB%82%9C%ED%99%94" title="지구 온난화">지구 온난화</a>로 갈 곳을 잃은 <a href="/wiki/%ED%8E%AD%EA%B7%84" title="펭귄">펭귄</a>을 보호하기 위해 <a href="/wiki/%EC%B9%B4%EB%9D%BC%EC%BF%B0_%EC%82%AC%EB%A7%89" title="카라쿰 사막">카라쿰 사막</a>에 <a href="/wiki/%EB%8F%99%EB%AC%BC%EC%9B%90" title="동물원">동물원</a>을 건설하기로 결정하였으며 같은 해 <a href="/wiki/12%EC%9B%94_16%EC%9D%BC" title="12월 16일">12월 16일</a>에는 신임 석유·가스 장관을 임명하며 향후 6개월 안에 영어에 능통하지 못하면 다시 해임하겠다고 위협하기도 하였다.</p>
<p><a href="/wiki/2006%EB%85%84" title="2006년">2006년</a> <a class="new" href="/w/index.php?title=%ED%84%B0%ED%82%A4%EB%A0%A5&amp;action=edit&amp;redlink=1" title="터키력 (없는 문서)">터키력</a> 신년을 축하하는 행사에서 "누구나 이 책(<a href="/wiki/%EB%A3%A8%ED%9D%90%EB%82%98%EB%A7%88" title="루흐나마">루흐나마</a>)을 한 번만 읽으면 지혜로워지며 새벽과 일몰 때 등 하루 세 번 읽으면 신비한 존재를 인식하게 돼 바로 천국에 갈 수 있다. 이는 내가 <a href="/wiki/%EC%95%8C%EB%9D%BC" title="알라">알라</a>께 여쭤 본 일이다" 라고 말하였다. 그의 발언은 전국에 텔레비전으로 생중계됐다.</p>
<p>이 외에도 <a href="/wiki/%EA%B3%BC%EC%9D%BC" title="과일">과일</a>인 <a href="/wiki/%EB%A9%9C%EB%A1%A0" title="멜론">멜론</a>을 매우 좋아하여 매년 8월 둘째 주 일요일을 멜론의 날이라는 기념일까지 지정하기도 했다.<sup class="reference" id="cite_ref-1"><a href="#cite_note-1">[1]</a></sup></p>
