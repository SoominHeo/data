<p><b>사하라줄무늬족제비</b>(Saharan striped polecat 또는 Saharan striped weasel, <i>Ictonyx libycus</i>)는 <a href="/wiki/%EC%A1%B1%EC%A0%9C%EB%B9%84%EA%B3%BC" title="족제비과">족제비과</a>에 속하는 <a href="/wiki/%ED%8F%AC%EC%9C%A0%EB%A5%98" title="포유류">포유류</a>의 일종이다. <b>리비아줄무늬족제비</b>(Libyan striped weasel) 또는 <b>북아프리카줄무늬족제비</b>(North African striped weasel)로도 알려져 있다.<sup class="reference" id="cite_ref-Newman.2C_Buesching.2C_and_Wolff_2005_632_2-0"><a href="#cite_note-Newman.2C_Buesching.2C_and_Wolff_2005_632-2">[2]</a></sup> <a href="/wiki/%EC%95%84%ED%94%84%EB%A6%AC%EC%B9%B4%EC%A4%84%EB%AC%B4%EB%8A%AC%EC%A1%B1%EC%A0%9C%EB%B9%84" title="아프리카줄무늬족제비">아프리카줄무늬족제비속</a>의 일부 종으로 간주하기도 하며, <a href="/wiki/%EC%A4%84%EB%AC%B4%EB%8A%AC%EC%A1%B1%EC%A0%9C%EB%B9%84" title="줄무늬족제비">줄무늬족제비</a>의 채색을 닮았다.<sup class="reference" id="cite_ref-3"><a href="#cite_note-3">[3]</a></sup> <a href="/wiki/%EC%95%8C%EC%A0%9C%EB%A6%AC" title="알제리">알제리</a>와 <a href="/wiki/%EC%B0%A8%EB%93%9C" title="차드">차드</a>, <a href="/wiki/%EC%9D%B4%EC%A7%91%ED%8A%B8" title="이집트">이집트</a>, <a href="/wiki/%EC%97%90%ED%8B%B0%EC%98%A4%ED%94%BC%EC%95%84" title="에티오피아">에티오피아</a>, <a href="/wiki/%EB%A6%AC%EB%B9%84%EC%95%84" title="리비아">리비아</a>, <a href="/wiki/%EB%A7%90%EB%A6%AC" title="말리">말리</a>, <a class="mw-redirect" href="/wiki/%EB%AA%A8%EB%A6%AC%ED%83%80%EB%8B%88%EC%95%84" title="모리타니아">모리타니아</a>, <a href="/wiki/%EB%AA%A8%EB%A1%9C%EC%BD%94" title="모로코">모로코</a>, <a href="/wiki/%EB%8B%88%EC%A0%9C%EB%A5%B4" title="니제르">니제르</a>, <a href="/wiki/%EB%82%98%EC%9D%B4%EC%A7%80%EB%A6%AC%EC%95%84" title="나이지리아">나이지리아</a>, <a href="/wiki/%EC%84%B8%EB%84%A4%EA%B0%88" title="세네갈">세네갈</a>, <a href="/wiki/%EC%88%98%EB%8B%A8" title="수단">수단</a>, <a href="/wiki/%ED%8A%80%EB%8B%88%EC%A7%80" title="튀니지">튀니지</a>를 포함한 다양한 북아프리카 국가에서 발견된다. 건조한 지역에서 주로 발견되며, 솔 모양의 아주 작은 꼬리가 특징적이다.<sup class="reference" id="cite_ref-the_International_Union_for_Conservation_of_Nature_and_Natural_Resources_4-0"><a href="#cite_note-the_International_Union_for_Conservation_of_Nature_and_Natural_Resources-4">[4]</a></sup></p>

<li><i>Ictonyx libyca multivittata</i></li>
