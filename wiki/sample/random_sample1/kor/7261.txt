<p><b>사회철학</b>(社會哲學, <span lang="en" xml:lang="en">social philosophy</span>)은 (일반적으로 인간의) 사회적 행동 양상에 관한 <a href="/wiki/%EC%B2%A0%ED%95%99" title="철학">철학</a>적 연구이다. 사회철학은 다양한 주제들을 다루는데 이는 개인적 의미론에서부터 법의 정당성까지, <a href="/wiki/%EC%82%AC%ED%9A%8C_%EA%B3%84%EC%95%BD" title="사회 계약">사회 계약</a>에서 <a href="/wiki/%ED%98%81%EB%AA%85" title="혁명">혁명</a>의 전제까지, 일상 활동에서부터 <a href="/wiki/%EA%B3%BC%ED%95%99" title="과학">과학</a>이 <a href="/wiki/%EB%AC%B8%ED%99%94" title="문화">문화</a>에 끼치는 영향까지, 인류 <a class="new" href="/w/index.php?title=%EC%9D%B8%EA%B5%AC_%EA%B5%AC%EC%84%B1&amp;action=edit&amp;redlink=1" title="인구 구성 (없는 문서)">인구 구성</a> 변화부터 <a href="/wiki/%EA%BF%80%EB%B2%8C" title="꿀벌">꿀벌</a>의 사회질서까지 등을 포괄한다.</p>
<h2><span class="mw-headline" id=".ED.95.98.EC.9C.84_.EB.B2.94.EC.A3.BC">하위 범주</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%82%AC%ED%9A%8C%EC%B2%A0%ED%95%99&amp;action=edit&amp;section=1" title="부분 편집: 하위 범주">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>사회철학은 <a href="/wiki/%EC%82%AC%ED%9A%8C" title="사회">사회</a>의 경향과 복합 의미, 변화와 지향성들을 이해하려고 한다. 이것은 수많은 하위 학문을 파생시킨다.</p>
<p>사회철학은 <a class="mw-redirect" href="/wiki/%EC%9C%A4%EB%A6%AC" title="윤리">윤리</a>와 <a href="/wiki/%EA%B0%80%EC%B9%98%EB%A1%A0" title="가치론">가치론</a>과 중첩되는 분야가 많다. 사회철학의 다른 형태들은 <a href="/wiki/%EA%B5%AD%EA%B0%80" title="국가">국가</a>와 <a href="/wiki/%EC%A0%95%EB%B6%80" title="정부">정부</a>를 사고하는 <a href="/wiki/%EC%A0%95%EC%B9%98%EC%B2%A0%ED%95%99" title="정치철학">정치철학</a>과 <a href="/wiki/%EB%B2%95%EC%B2%A0%ED%95%99" title="법철학">법철학</a>등이 있다. 이들 학문은 타 <a href="/wiki/%EC%82%AC%ED%9A%8C%EA%B3%BC%ED%95%99" title="사회과학">사회과학</a>과 많은 부분을 공유한다. 동시에 사회과학은 <a class="new" href="/w/index.php?title=%EC%82%AC%ED%9A%8C%EA%B3%BC%ED%95%99%EC%B2%A0%ED%95%99&amp;action=edit&amp;redlink=1" title="사회과학철학 (없는 문서)">사회과학철학</a>의 탐구 대상이 된다.</p>
<p>이 밖에 <a href="/wiki/%EC%96%B8%EC%96%B4%EC%B2%A0%ED%95%99" title="언어철학">언어철학</a>과 <a class="new" href="/w/index.php?title=%EC%82%AC%ED%9A%8C%EC%9D%B8%EC%8B%9D%EB%A1%A0&amp;action=edit&amp;redlink=1" title="사회인식론 (없는 문서)">사회인식론</a>등이 있다.</p>
