<p><b>산 조르지오급 강습상륙함</b>은 <a href="/wiki/%EC%9D%B4%ED%83%88%EB%A6%AC%EC%95%84_%ED%95%B4%EA%B5%B0" title="이탈리아 해군">이탈리아 해군</a>의 만재배수량 8천톤급 강습상륙함이다. 2만톤인 독도함 보다 2배 작은 헬리콥터 항공모함이다. 프랑스 해군의 BIP-8, 일본의 <a class="mw-redirect" href="/wiki/%EC%98%A4%EC%98%A4%EC%8A%A4%EB%AF%B8%ED%95%A8" title="오오스미함">오오스미함</a>이 이와 비슷하다.</p>

<h2><span class="mw-headline" id=".EC.88.98.EC.86.A1.EB.8A.A5.EB.A0.A5">수송능력</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%82%B0_%EC%A1%B0%EB%A5%B4%EC%A7%80%EC%98%A4%EA%B8%89_%EA%B0%95%EC%8A%B5%EC%83%81%EB%A5%99%ED%95%A8&amp;action=edit&amp;section=1" title="부분 편집: 수송능력">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>350명의 보병과 30대의 중형 전차, 또는 36대의 장갑차를 수송할 수 있다.</p>
<p>3대의 <a class="mw-redirect" href="/wiki/%EC%8B%9C_%ED%82%B9" title="시 킹">시 킹</a> 헬리콥터, 또는 5대의 <a href="/wiki/%EB%B2%A8_212" title="벨 212">벨 212</a> 헬리콥터를 탑재한다.</p>
<p>인도적 군사작전을 위한 병원 시설을 갖추고 있다.</p>

<h2><span class="mw-headline" id=".ED.95.A8.EB.B2.88">함번</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%82%B0_%EC%A1%B0%EB%A5%B4%EC%A7%80%EC%98%A4%EA%B8%89_%EA%B0%95%EC%8A%B5%EC%83%81%EB%A5%99%ED%95%A8&amp;action=edit&amp;section=3" title="부분 편집: 함번">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<ul>
<li>산 조르지오함, L9892, 취역 1987년</li>
<li>산 마르코함, L9893, 취역 1988년</li>
<li>산 쥬스토, L9894, 취역 1994년</li>
</ul>
<h2><span class="mw-headline" id=".EA.B0.9C.EB.9F.89">개량</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%82%B0_%EC%A1%B0%EB%A5%B4%EC%A7%80%EC%98%A4%EA%B8%89_%EA%B0%95%EC%8A%B5%EC%83%81%EB%A5%99%ED%95%A8&amp;action=edit&amp;section=4" title="부분 편집: 개량">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>길이 133 미터인 산 조르지오급은 원래 3대의 헬리콥터를 동시에 이착륙할 수 있는 비행갑판을 갖고 있으나, 개량을 하여, 산 조르지오함, 산 마르코함은 4대의 헬기 이착륙이 가능하다. 비행갑판의 길이를 약간 늘렸다.</p>
<h2><span class="mw-headline" id=".EA.B5.90.EC.B2.B4">교체</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%82%B0_%EC%A1%B0%EB%A5%B4%EC%A7%80%EC%98%A4%EA%B8%89_%EA%B0%95%EC%8A%B5%EC%83%81%EB%A5%99%ED%95%A8&amp;action=edit&amp;section=5" title="부분 편집: 교체">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p><a class="new" href="/w/index.php?title=%EC%82%B0_%EC%A1%B0%EB%A5%B4%EC%A7%80%EC%98%A4%ED%95%A8&amp;action=edit&amp;redlink=1" title="산 조르지오함 (없는 문서)">산 조르지오함</a>, <a class="new" href="/w/index.php?title=%EC%82%B0_%EB%A7%88%EB%A5%B4%EC%BD%94%ED%95%A8&amp;action=edit&amp;redlink=1" title="산 마르코함 (없는 문서)">산 마르코함</a>은 오래되어 후속함으로 교체될 계획이다. 이탈리아 해군은 만재배수량 2만톤, 길이 190 미터인 <a class="mw-redirect" href="/wiki/%EB%8F%85%EB%8F%84%ED%95%A8" title="독도함">독도함</a> 크기의 강습상륙함 2척을 계획하고 있다.</p>
<h2><span class="mw-headline" id=".EC.8B.A4.EC.A0.84.EA.B2.BD.EB.A0.A5">실전경력</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%82%B0_%EC%A1%B0%EB%A5%B4%EC%A7%80%EC%98%A4%EA%B8%89_%EA%B0%95%EC%8A%B5%EC%83%81%EB%A5%99%ED%95%A8&amp;action=edit&amp;section=6" title="부분 편집: 실전경력">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<ul>
<li>1999년 10월 26일 - 3번함 <a class="new" href="/w/index.php?title=%EC%82%B0_%EC%A5%AC%EC%8A%A4%ED%86%A0%ED%95%A8&amp;action=edit&amp;redlink=1" title="산 쥬스토함 (없는 문서)">산 쥬스토함</a>이 <a href="/wiki/%EB%8F%99%ED%8B%B0%EB%AA%A8%EB%A5%B4" title="동티모르">동티모르</a>에 파병되었다. 호주군이 이끄는 <a class="mw-redirect" href="/wiki/INTERFET" title="INTERFET">INTERFET</a> <a href="/wiki/%EC%9C%A0%EC%97%94_%ED%8F%89%ED%99%94%EC%9C%A0%EC%A7%80%EA%B5%B0" title="유엔 평화유지군">유엔 평화유지군</a>에 배속되었다. 2000년 2월 15일 복귀했다. <a class="mw-redirect" href="/wiki/INTERFET" title="INTERFET">INTERFET</a>에는 한국의 <a href="/wiki/%EC%83%81%EB%A1%9D%EC%88%98_%EB%B6%80%EB%8C%80" title="상록수 부대">상록수 부대</a>도 파병되었다.</li>
<li>2005년 - 3번함 <a class="new" href="/w/index.php?title=%EC%82%B0_%EC%A5%AC%EC%8A%A4%ED%86%A0%ED%95%A8&amp;action=edit&amp;redlink=1" title="산 쥬스토함 (없는 문서)">산 쥬스토함</a>이 "소르베 로열 훈련"의 함대 기함이 되었다. <a href="/wiki/%EC%9A%B0%ED%81%AC%EB%9D%BC%EC%9D%B4%EB%82%98" title="우크라이나">우크라이나</a>, <a href="/wiki/%EB%9F%AC%EC%8B%9C%EC%95%84" title="러시아">러시아</a>가 참여한 <a class="mw-redirect" href="/wiki/%EB%82%98%ED%86%A0" title="나토">나토</a> 연합 해군 훈련으로, 침몰된 <a href="/wiki/%EC%9E%A0%EC%88%98%ED%95%A8" title="잠수함">잠수함</a>을 구조하는 훈련이다.</li>
</ul>
