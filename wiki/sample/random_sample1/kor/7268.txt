<h1 class="firstHeading" id="firstHeading" lang="ko">산 크리소고노 성당</h1>
<div class="mw-body-content" id="bodyContent">
<div id="siteSub">위키백과, 우리 모두의 백과사전.</div>

<p><b>산 크리소고노 성당</b>(<span style="font-size: smaller;"><a href="/wiki/%EC%9D%B4%ED%83%88%EB%A6%AC%EC%95%84%EC%96%B4" title="이탈리아어">이탈리아어</a>:</span> <span lang="it" xml:lang="it">Basilica di San Crisogono</span>)은 <a href="/wiki/%EC%9D%B4%ED%83%88%EB%A6%AC%EC%95%84" title="이탈리아">이탈리아</a> <a href="/wiki/%EB%A1%9C%EB%A7%88" title="로마">로마</a> <a class="new" href="/w/index.php?title=%ED%8A%B8%EB%9D%BC%EC%8A%A4%ED%85%8C%EB%B2%A0%EB%A0%88&amp;action=edit&amp;redlink=1" title="트라스테베레 (없는 문서)">트라스테베레</a> 거리에 있는 <a href="/wiki/%EB%A1%9C%EB%A7%88_%EA%B0%80%ED%86%A8%EB%A6%AD%EA%B5%90%ED%9A%8C" title="로마 가톨릭교회">로마 가톨릭교회</a>의 <a href="/wiki/%EC%84%B1%EB%8B%B9" title="성당">성당</a>이다. <a class="mw-redirect" href="/wiki/%EC%88%98%ED%98%B8%EC%84%B1%EC%9D%B8" title="수호성인">수호성인</a>은 <a href="/wiki/%ED%81%AC%EB%A6%AC%EC%86%8C%EA%B3%A0%EB%85%B8" title="크리소고노">성 크리소고노</a>이다.</p>
<h2><span class="mw-headline" id=".EC.97.AD.EC.82.AC">역사</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%82%B0_%ED%81%AC%EB%A6%AC%EC%86%8C%EA%B3%A0%EB%85%B8_%EC%84%B1%EB%8B%B9&amp;action=edit&amp;section=1" title="부분 편집: 역사">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>산 크리소고노 성당은 로마의 <a class="new" href="/w/index.php?title=%EB%AA%85%EC%9D%98%EB%B3%B8%EB%8B%B9&amp;action=edit&amp;redlink=1" title="명의본당 (없는 문서)">명의본당</a> 가운데 하나이며, 로마의 첫 <a href="/wiki/%EC%82%AC%EB%AA%A9%EA%B5%AC_%EC%84%B1%EB%8B%B9" title="사목구 성당">본당</a>이다. 4세기에 <a href="/wiki/%EA%B5%90%ED%99%A9_%EC%8B%A4%EB%B2%A0%EC%8A%A4%ED%85%8C%EB%A5%B4_1%EC%84%B8" title="교황 실베스테르 1세">교황 실베스테르 1세</a>(314-335)의 지시로 건립된 것으로 추정되며, 12세기에 <a class="new" href="/w/index.php?title=%ED%81%AC%EB%A0%88%EB%A7%88%EC%9D%98_%EC%9A%94%ED%95%9C&amp;action=edit&amp;redlink=1" title="크레마의 요한 (없는 문서)">크레마의 요한</a>과 <a class="new" href="/w/index.php?title=%EC%A1%B0%EB%B0%98%EB%8B%88_%EB%B0%94%ED%8B%B0%EC%8A%A4%ED%83%80_%EC%86%8C%EB%A6%AC%EC%95%84&amp;action=edit&amp;redlink=1" title="조반니 바티스타 소리아 (없는 문서)">조반니 바티스타 소리아</a>에 의해 재건축되었으며, 17세기 초에 <a class="new" href="/w/index.php?title=%EC%8B%9C%ED%94%BC%EC%98%A4%EB%84%A4_%EB%B3%B4%EB%A5%B4%EA%B2%8C%EC%84%B8&amp;action=edit&amp;redlink=1" title="시피오네 보르게세 (없는 문서)">시피오네 보르게세</a>에 의해 오늘날의 형태로 완성되었다.</p>
<p>1907년 L. 만프레디니 신부와 C. 피콜로미니 신부가 성당의 제의실 밑을 파본 결과, 초창기 성당의 유적들을 발견하였다. 이후 성당 지하에 대한 본격적인 발굴 조사가 시작되었다.</p>
<p>오늘날 산 크리소고노 성당의 관리는 <a class="new" href="/w/index.php?title=%EC%82%BC%EC%9C%84%EC%9D%BC%EC%B2%B4%ED%9A%8C&amp;action=edit&amp;redlink=1" title="삼위일체회 (없는 문서)">삼위일체회</a>에서 담당하고 있다. 산 크리소고노 성당의 명의사제 추기경 가운데 유명한 인물로는 훗날 <a href="/wiki/%EA%B5%90%ED%99%A9_%EB%A0%88%EC%98%A4_13%EC%84%B8" title="교황 레오 13세">교황 레오 13세</a>로 선출된 빈첸초 조아키노 라파엘레 루이지 페치 추기경과 <a href="/wiki/%EB%8C%80%ED%95%9C%EB%AF%BC%EA%B5%AD" title="대한민국">대한민국</a>의 <a href="/wiki/%EC%97%BC%EC%88%98%EC%A0%95" title="염수정">염수정</a>(안드레아) 추기경이 있다.</p>
<h2><span class="mw-headline" id=".EB.AF.B8.EC.88.A0.EA.B3.BC_.EA.B1.B4.EC.B6.95">미술과 건축</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%82%B0_%ED%81%AC%EB%A6%AC%EC%86%8C%EA%B3%A0%EB%85%B8_%EC%84%B1%EB%8B%B9&amp;action=edit&amp;section=2" title="부분 편집: 미술과 건축">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<div class="thumb tleft">
<div class="thumbinner" style="width:222px;"><a class="image" href="/wiki/%ED%8C%8C%EC%9D%BC:Trastevere_-_san_Crisogono_1000360_interno.JPG"><img alt="" class="thumbimage" data-file-height="2112" data-file-width="2816" height="165" src="//upload.wikimedia.org/wikipedia/commons/thumb/5/5d/Trastevere_-_san_Crisogono_1000360_interno.JPG/220px-Trastevere_-_san_Crisogono_1000360_interno.JPG" srcset="//upload.wikimedia.org/wikipedia/commons/thumb/5/5d/Trastevere_-_san_Crisogono_1000360_interno.JPG/330px-Trastevere_-_san_Crisogono_1000360_interno.JPG 1.5x, //upload.wikimedia.org/wikipedia/commons/thumb/5/5d/Trastevere_-_san_Crisogono_1000360_interno.JPG/440px-Trastevere_-_san_Crisogono_1000360_interno.JPG 2x" width="220"/></a>
<div class="thumbcaption">
<div class="magnify"><a class="internal" href="/wiki/%ED%8C%8C%EC%9D%BC:Trastevere_-_san_Crisogono_1000360_interno.JPG" title="실제 크기로"></a></div>
성당 내부</div>

<p>성당의 종탑은 12세기에 재건축한 것이며, 성당 내부는 12세기에 재건된 성당 자리에 1620년대에 재건축한 것이다. 성당 내부에서 천장을 떠받치는 22개의 화강암 기둥은 고대 시절의 기둥을 손본 후에 재활용한 것이다. 바닥 부분은 <a class="new" href="/w/index.php?title=%EC%BD%94%EC%8A%A4%EB%A7%88%ED%85%8C%EC%8A%A4%ED%81%AC&amp;action=edit&amp;redlink=1" title="코스마테스크 (없는 문서)">코스마테스크</a> 장식이 되어 있지만, 대부분 긴 회중석에 의해 가려져 있다. <a class="new" href="/w/index.php?title=%EC%A0%9C%EB%8B%A8_(%EA%B8%B0%EB%8F%85%EA%B5%90)&amp;action=edit&amp;redlink=1" title="제단 (기독교) (없는 문서)">제단</a> 밑 <a href="/wiki/%ED%81%AC%EB%A6%BD%ED%8A%B8" title="크립트">지하실</a>은 8세기에 지어진 것이다. 중앙 제대는 1127년에, 제대 위를 덮은 천개(발다키노)는 1627년 또는 1641년에 <a href="/wiki/%EC%9E%94_%EB%A1%9C%EB%A0%8C%EC%B4%88_%EB%B2%A0%EB%A5%B4%EB%8B%88%EB%8B%88" title="잔 로렌초 베르니니">잔 로렌초 베르니니</a>가 제작한 것이다. 제대 위 <a class="mw-redirect" href="/wiki/%EB%B0%94%EB%A1%9C%ED%81%AC_%EC%96%91%EC%8B%9D" title="바로크 양식">바로크 양식</a>의 천장 중앙 부분에는 <a class="new" href="/w/index.php?title=%EA%B5%AC%EC%97%90%EB%A5%B4%EC%B9%98%EB%85%B8&amp;action=edit&amp;redlink=1" title="구에르치노 (없는 문서)">구에르치노</a>가 그린 《성 크리소고노의 영광》이라는 제목의 천장화가 그려져 있다.</p>
<p>성당 내부 왼쪽 끝에는 <a class="new" href="/w/index.php?title=%EC%95%88%EB%82%98_%EB%A7%88%EB%A6%AC%EC%95%84_%ED%83%80%EC%9D%B4%EC%A7%80&amp;action=edit&amp;redlink=1" title="안나 마리아 타이지 (없는 문서)">복녀 안나 마리아 타이지</a>의 유해를 모신 부속 제대가 자리잡고 있으며, 입구 왼쪽에는 조반노 자코포 밀로 추기경에게 헌정된 기념비가 자리잡고 있다. 이 기념비는 <a class="new" href="/w/index.php?title=%EC%B9%B4%EB%A5%BC%EB%A1%9C_%EB%A7%88%EB%A5%B4%ED%82%A4%EC%98%A4%EB%8B%88&amp;action=edit&amp;redlink=1" title="카를로 마르키오니 (없는 문서)">카를로 마르키오니</a>와 <a class="new" href="/w/index.php?title=%ED%94%BC%EC%97%90%ED%8A%B8%EB%A1%9C_%EB%B8%8C%EB%9D%BC%EC%B9%98&amp;action=edit&amp;redlink=1" title="피에트로 브라치 (없는 문서)">피에트로 브라치</a>가 제작한 것이다. 성당 오른쪽에는 《성녀 프란체스카 로마나의 십자가형》을 포함한 프레스코화가 그려져 있다.</p>
