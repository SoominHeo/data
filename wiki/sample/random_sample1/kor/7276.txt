<p>《<b>산림경제</b>》(山林經濟)는 <a href="/wiki/%EC%A1%B0%EC%84%A0" title="조선">조선</a> 후기의 <a class="mw-redirect" href="/wiki/%EC%8B%A4%ED%95%99%EC%9E%90" title="실학자">실학자</a> 유암 <a href="/wiki/%ED%99%8D%EB%A7%8C%EC%84%A0" title="홍만선">홍만선</a>(流巖 洪萬選：<a href="/wiki/1643%EB%85%84" title="1643년">1643년</a>~<a href="/wiki/1715%EB%85%84" title="1715년">1715년</a>)이 지은 <a href="/wiki/%EB%86%8D%EC%97%85" title="농업">농업</a> 책이자 <a href="/wiki/%EA%B0%80%EC%A0%95" title="가정">가정생활</a> 책이다. 이 책은 <a href="/wiki/%EC%A1%B0%EC%84%A0_%EC%88%99%EC%A2%85" title="조선 숙종">숙종</a> 때, 지은이가 만년인 <a href="/wiki/17%EC%84%B8%EA%B8%B0" title="17세기">17세기</a> 말에서 <a href="/wiki/18%EC%84%B8%EA%B8%B0" title="18세기">18세기</a> 초에 걸쳐 편찬되었으며, <a href="/wiki/%ED%95%84%EC%82%AC%EB%B3%B8" title="필사본">필사본</a>의 4권 4책으로 구성된다. 이 책은 <a href="/wiki/%EA%B3%A0%EC%B6%94" title="고추">고추</a>에 관해 언급한 최초의 문헌이라는 것에 큰 중요성을 가지며, 현재 <a href="/wiki/%EA%B5%AD%EB%A6%BD%EC%A4%91%EC%95%99%EB%8F%84%EC%84%9C%EA%B4%80" title="국립중앙도서관">국립중앙도서관</a>과 <a class="new" href="/w/index.php?title=%EA%B7%9C%EC%9E%A5%EA%B0%81%EB%8F%84%EC%84%9C%EA%B4%80&amp;action=edit&amp;redlink=1" title="규장각도서관 (없는 문서)">규장각도서관</a>, <a class="new" href="/w/index.php?title=%EC%9E%A5%EC%84%9C%EA%B0%81%EB%8F%84%EC%84%9C&amp;action=edit&amp;redlink=1" title="장서각도서 (없는 문서)">장서각도서</a>에서 소장하고 있다.</p>

<h2><span class="mw-headline" id=".EB.82.B4.EC.9A.A9">내용</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%82%B0%EB%A6%BC%EA%B2%BD%EC%A0%9C&amp;action=edit&amp;section=1" title="부분 편집: 내용">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>이 책은 복거(卜居)·섭생·치농(治農)·치포(治圃)·구급·종수(種樹)·치선(治膳)·구황(救荒)·벽한(酸寒)·벽온(酸瘟)·벽충(酸筮)·양화(養花)·양잠·목양(牧養)·잡방(雜方) 등 15지(志)로 분류되어 있다. 박물지, 곧 <a href="/wiki/%EB%B0%B1%EA%B3%BC%EC%82%AC%EC%A0%84" title="백과사전">백과사전</a>의 성격을 띤 이 책에는 전원 생활에 중요한 다방면의 보건위생과 경제면 및 정서 생활을 위한 고도의 지식이 수록되어 있다.</p>
<h2><span class="mw-headline" id=".EA.B5.AC.EC.84.B1">구성</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%82%B0%EB%A6%BC%EA%B2%BD%EC%A0%9C&amp;action=edit&amp;section=2" title="부분 편집: 구성">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>산림경제는 전4권 4책에 총 16지(志)로 구성되어 있다.</p>
<ul>
<li>산림경제 제1권
<ul>
<li>서(序)</li>
<li>복거(卜居) - <a class="mw-redirect" href="/wiki/%EC%A3%BC%ED%83%9D" title="주택">주택</a>의 선정과 <a href="/wiki/%EA%B1%B4%EC%B6%95" title="건축">건축</a></li>
<li>섭생(攝生) - 건강</li>
<li>치농(治農) - <a class="mw-redirect" href="/wiki/%EA%B3%A1%EC%8B%9D" title="곡식">곡식</a>과 <a href="/wiki/%EB%AA%A9%ED%99%94" title="목화">목화</a> 및 기타 <a class="new" href="/w/index.php?title=%ED%8A%B9%EC%9A%A9%EC%9E%91%EB%AC%BC&amp;action=edit&amp;redlink=1" title="특용작물 (없는 문서)">특용작물</a>의 재배법</li>
<li>치포(治圃) - 채소류, 화초류, <a href="/wiki/%EB%8B%B4%EB%B0%B0" title="담배">담배</a>, 약초류 재배법</li>
</ul>
</li>
</ul>
<ul>
<li>산림경제 제2권
<ul>
<li>종수(種樹) - <a class="mw-redirect" href="/wiki/%EA%B3%BC%EC%88%98" title="과수">과수</a>와 <a class="new" href="/w/index.php?title=%EC%9E%84%EB%AA%A9&amp;action=edit&amp;redlink=1" title="임목 (없는 문서)">임목</a>의 육성</li>
<li>양화(養花) - <a href="/wiki/%ED%99%94%EC%B4%88" title="화초">화초</a>, 화목, 정원수의 가꿈</li>
<li>양잠(養蠶) - <a class="new" href="/w/index.php?title=%EB%B2%8C_(%EC%83%9D%EB%AC%BC)&amp;action=edit&amp;redlink=1" title="벌 (생물) (없는 문서)">벌</a> <a class="mw-disambig" href="/wiki/%EC%96%91%EC%8B%9D" title="양식">양식</a></li>
<li>목양(牧養) - <a href="/wiki/%EA%B0%80%EC%B6%95" title="가축">가축</a>, <a href="/wiki/%EA%B0%80%EA%B8%88" title="가금">가금</a>, 벌, <a href="/wiki/%EB%AC%BC%EA%B3%A0%EA%B8%B0" title="물고기">물고기</a> 양식</li>
<li>치선(治膳) - <a class="new" href="/w/index.php?title=%EC%8B%9D%ED%92%88%EC%A0%80%EC%9E%A5%EB%B2%95&amp;action=edit&amp;redlink=1" title="식품저장법 (없는 문서)">식품저장법</a>, <a href="/wiki/%EC%A1%B0%EB%A6%AC%EB%B2%95" title="조리법">조리법</a>, 가공법</li>
</ul>
</li>
</ul>
<ul>
<li>산림경제 제3권
<ul>
<li>구급(救急) - 150가지 <a class="new" href="/w/index.php?title=%EC%9D%91%EA%B8%89%EC%B2%98%EB%A6%AC&amp;action=edit&amp;redlink=1" title="응급처리 (없는 문서)">응급처리</a>법</li>
<li>구황(救荒) - <a class="new" href="/w/index.php?title=%ED%9D%89%EB%85%84&amp;action=edit&amp;redlink=1" title="흉년 (없는 문서)">흉년</a>에 대비하는 비상조치법</li>
<li>벽온(辟瘟) - <a class="mw-redirect" href="/wiki/%EC%A0%84%EC%97%BC%EB%B3%91" title="전염병">전염병</a>을 물리치는 법</li>
<li>벽충(辟蟲) - <a href="/wiki/%ED%95%B4%EC%B6%A9" title="해충">해충</a>, <a href="/wiki/%EC%A5%90" title="쥐">쥐</a>, <a href="/wiki/%EB%B1%80" title="뱀">뱀</a> 등을 물리치는 처방</li>
</ul>
</li>
</ul>
<ul>
<li>산림경제 제4권
<ul>
<li>치약(治藥) - <a href="/wiki/%EC%95%BD" title="약">약</a>의 조제, 금기하는 약 등을 논함</li>
<li>선택(選擇) - <a class="new" href="/w/index.php?title=%EA%B8%B8%ED%9D%89%EC%9D%BC&amp;action=edit&amp;redlink=1" title="길흉일 (없는 문서)">길흉일</a>과 <a href="/wiki/%EB%B0%A9%ED%96%A5" title="방향">방향</a>의 선택</li>
<li>잡방(雜方) - 그림, 글씨, 촉대, 도자기, 악기, 장검 등을 손질하는 방법</li>
</ul>
</li>
</ul>
