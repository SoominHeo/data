<p><b>리라</b>(<span style="font-size: smaller;"><a href="/wiki/%EC%9D%B4%ED%83%88%EB%A6%AC%EC%95%84%EC%96%B4" title="이탈리아어">이탈리아어</a>:</span> <span lang="it" xml:lang="it">lira</span>, 복수형: <span lang="it" xml:lang="it">lire</span>)는 <a href="/wiki/1860%EB%85%84%EB%8C%80" title="1860년대">1860년대</a>부터 <a href="/wiki/%EC%9C%A0%EB%A1%9C" title="유로">유로</a>가 도입된 <a href="/wiki/2002%EB%85%84" title="2002년">2002년</a>까지 통용된 <a href="/wiki/%EC%82%B0%EB%A7%88%EB%A6%AC%EB%85%B8" title="산마리노">산마리노</a>의 통화로, <a href="/wiki/%EC%9D%B4%ED%83%88%EB%A6%AC%EC%95%84_%EB%A6%AC%EB%9D%BC" title="이탈리아 리라">이탈리아 리라</a>와 동등한 수준의 비율을 갖고 있었다. 이탈리아의 동전과 지폐는 <a href="/wiki/%EC%82%B0%EB%A7%88%EB%A6%AC%EB%85%B8" title="산마리노">산마리노</a>와 <a href="/wiki/%EB%B0%94%ED%8B%B0%EC%B9%B8_%EC%8B%9C%EA%B5%AD" title="바티칸 시국">바티칸</a>에서 법화로 인정되었으며, <a href="/wiki/%EC%9D%B4%ED%83%88%EB%A6%AC%EC%95%84" title="이탈리아">이탈리아</a> <a href="/wiki/%EB%A1%9C%EB%A7%88" title="로마">로마</a>에서 주조된 산마리노의 동전 또한 이탈리아와 바티칸에서 법화로 인정되었다.</p>

<h2><span class="mw-headline" id=".EB.8F.99.EC.A0.84">동전</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%82%B0%EB%A7%88%EB%A6%AC%EB%85%B8_%EB%A6%AC%EB%9D%BC&amp;action=edit&amp;section=1" title="부분 편집: 동전">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>산마리노는 <a href="/wiki/1864%EB%85%84" title="1864년">1864년</a>에 최초의 동전을 발행했는데, 그 동전은 구리로 된 5 첸테시미 동전이었다. 이후 <a href="/wiki/1875%EB%85%84" title="1875년">1875년</a>에 구리로 된 10 첸테시미 동전이 발행되었으나 구리로 된 동전은 <a href="/wiki/1894%EB%85%84" title="1894년">1894년</a>에 발행이 중단했다. <a href="/wiki/1898%EB%85%84" title="1898년">1898년</a>에는 은으로 된 50 첸테시미 동전과 1, 2, 5 리라 동전을 발행했으며, <a href="/wiki/1906%EB%85%84" title="1906년">1906년</a>에 1, 2 리라 동전을 추가로 발행했다.</p>
<p>산마리노는 <a href="/wiki/1931%EB%85%84" title="1931년">1931년</a>에 동전 발행을 재개했으며, <a href="/wiki/1935%EB%85%84" title="1935년">1935년</a>에 은으로 된 5, 10, 20 리라 동전과 청동으로 된 5, 10 첸테시미 동전이 추가로 발행되었다. 이들 동전은 <a href="/wiki/1938%EB%85%84" title="1938년">1938년</a>까지 통용되었다.</p>
<p>산마리노는 <a href="/wiki/1972%EB%85%84" title="1972년">1972년</a>에 동전 발행을 재개했는데, 이 때 발행된 1, 2, 5, 10, 20, 50, 100, 500 리라 동전 모두 당시 통용되던 이탈리아의 동전과 같은 모양으로 주조되었다. 200 리라 동전은 <a href="/wiki/1978%EB%85%84" title="1978년">1978년</a>에, 바이메탈로 된 500, 1000 리라 동전은 각각 <a href="/wiki/1982%EB%85%84" title="1982년">1982년</a>과 <a href="/wiki/1997%EB%85%84" title="1997년">1997년</a>에 발행되었다. 이러한 동전 디자인은 매년 변경되어 왔다.</p>
