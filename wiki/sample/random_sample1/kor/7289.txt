<div class="mw-body-content" id="bodyContent">

<div class="mw-content-ltr" dir="ltr" id="mw-content-text" lang="ko"><script>function mfTempOpenSection(id){var block=document.getElementById("mf-section-"+id);block.className+=" open-block";block.previousSibling.className+=" open-block";}</script><p><a href="/wiki/%EC%88%98%ED%95%99" title="수학">수학</a>에서, <b>산술 기하 평균</b>(算術幾何平均, <span style="font-size: smaller;"><a href="/wiki/%EC%98%81%EC%96%B4" title="영어">영어</a>:</span> <span lang="en" xml:lang="en">arithmetic–geometric mean</span>)은 <a href="/wiki/%EC%82%B0%EC%88%A0_%ED%8F%89%EA%B7%A0" title="산술 평균">산술 평균</a>과 <a href="/wiki/%EA%B8%B0%ED%95%98_%ED%8F%89%EA%B7%A0" title="기하 평균">기하 평균</a> 연산에 의한 <a class="mw-redirect" href="/wiki/%EC%A0%90%ED%99%94_%EC%88%98%EC%97%B4" title="점화 수열">점화 수열</a>에 <a href="/wiki/%EC%88%98%EC%97%B4%EC%9D%98_%EA%B7%B9%ED%95%9C" title="수열의 극한">극한</a>을 취하여 얻어진 <a class="mw-redirect" href="/wiki/%ED%8F%89%EA%B7%A0%EA%B0%92" title="평균값">평균값</a>이다. 구체적으로, 두 <a href="/wiki/%EC%8B%A4%EC%88%98" title="실수">실수</a> <span class="texhtml" style="font-family: 'Nimbus Roman No9 L', 'Times New Roman', Times, serif; white-space: nowrap; line-height: 1; font-size: 118%;"><i>x</i>, <i>y</i></span>의 산술 기하 평균 <span class="texhtml" style="font-family: 'Nimbus Roman No9 L', 'Times New Roman', Times, serif; white-space: nowrap; line-height: 1; font-size: 118%;"><i>M</i>(<i>x</i>, <i>y</i>)</span>는 다음과 같이 정의된다.</p>
<p>우선 두 수 <span class="texhtml" style="font-family: 'Nimbus Roman No9 L', 'Times New Roman', Times, serif; white-space: nowrap; line-height: 1; font-size: 118%;"><i>x</i>, <i>y</i></span>의 <a href="/wiki/%EC%82%B0%EC%88%A0_%ED%8F%89%EA%B7%A0" title="산술 평균">산술 평균</a>을 <span class="texhtml" style="font-family: 'Nimbus Roman No9 L', 'Times New Roman', Times, serif; white-space: nowrap; line-height: 1; font-size: 118%;"><i>a</i><sub>1</sub></span>, <a href="/wiki/%EA%B8%B0%ED%95%98_%ED%8F%89%EA%B7%A0" title="기하 평균">기하 평균</a>을 <span class="texhtml" style="font-family: 'Nimbus Roman No9 L', 'Times New Roman', Times, serif; white-space: nowrap; line-height: 1; font-size: 118%;"><i>g</i><sub>1</sub></span>라고 하자.</p>

<p><a href="/wiki/%EA%B8%B0%ED%95%98_%EC%A1%B0%ED%99%94_%ED%8F%89%EA%B7%A0" title="기하 조화 평균">기하 조화 평균</a>은 이와 비슷하게 기하 평균과 <a href="/wiki/%EC%A1%B0%ED%99%94_%ED%8F%89%EA%B7%A0" title="조화 평균">조화 평균</a>을 사용해 정의한 수열의 극한값이다. 산술 조화 평균 또한 비슷한 방법으로 얻어지나, 이는 곧 <a href="/wiki/%EA%B8%B0%ED%95%98_%ED%8F%89%EA%B7%A0" title="기하 평균">기하 평균</a>과 같다.</p>
<p>산술 기하 평균은 <a href="/wiki/%EB%A1%9C%EA%B7%B8" title="로그">로그</a>와 <a class="new" href="/w/index.php?title=%ED%83%80%EC%9B%90_%EC%A0%81%EB%B6%84&amp;action=edit&amp;redlink=1" title="타원 적분 (없는 문서)">제1종 완전 타원 적분</a>을 계산하는 데에 사용된다. 산술 기하 평균의 변형을 이용하여 <a class="new" href="/w/index.php?title=%ED%83%80%EC%9B%90_%EC%A0%81%EB%B6%84&amp;action=edit&amp;redlink=1" title="타원 적분 (없는 문서)">제2종 완전 타원 적분</a>을 효율적으로 계산할 수 있다.<sup class="reference" id="cite_ref-4"><a href="#cite_note-4">[4]</a></sup></p>
<h2><span class="mw-headline" id="M.EC.9D.98_.EC.A1.B4.EC.9E.AC.EC.84.B1_.EC.A6.9D.EB.AA.85"><span class="texhtml mvar" style="font-family: 'Nimbus Roman No9 L', 'Times New Roman', Times, serif; font-size: 118%; white-space: nowrap; font-style: italic;">M</span>의 존재성 증명</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%82%B0%EC%88%A0_%EA%B8%B0%ED%95%98_%ED%8F%89%EA%B7%A0&amp;action=edit&amp;section=5" title="부분 편집: M의 존재성 증명">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
