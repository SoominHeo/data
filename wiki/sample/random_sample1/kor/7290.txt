<p><b>산시 성</b>(섬서성, <small><a href="/wiki/%EC%A4%91%EA%B5%AD%EC%96%B4" title="중국어">중국어</a></small> <small><a href="/wiki/%EA%B0%84%EC%B2%B4%EC%9E%90" title="간체자">간체</a>:</small> <span lang="zh-hans" xml:lang="zh-hans">陕西省</span>, <small><a href="/wiki/%EC%A0%95%EC%B2%B4%EC%9E%90" title="정체자">정체</a>:</small> <span lang="zh-hant" xml:lang="zh-hant">陝西省</span>, <small><a href="/wiki/%ED%95%9C%EC%96%B4_%EB%B3%91%EC%9D%8C" title="한어 병음">병음</a>:</small> <font face="Lucida Sans Unicode">Shǎnxī Shěng</font>)은 <a href="/wiki/%EC%A4%91%ED%99%94%EC%9D%B8%EB%AF%BC%EA%B3%B5%ED%99%94%EA%B5%AD" title="중화인민공화국">중화인민공화국</a> 중부에 위치한 성이다.</p>

<h2><span class="mw-headline" id=".EC.9D.B4.EB.A6.84">이름</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%82%B0%EC%8B%9C_%EC%84%B1_(%EC%84%AC%EC%84%9C%EC%84%B1)&amp;action=edit&amp;section=1" title="부분 편집: 이름">원본 편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>한국어 한자 발음은 <b>섬서성</b>이 맞으나, <b>협서성</b>이라고 잘못 읽는 사람도 있고,<sup class="reference" id="cite_ref-1"><a href="#cite_note-1">[1]</a></sup> <b>峽西省</b>이라고 한자로 잘못 쓰기도 한다. 영어 등의 외국어로 표기할 때에는 <a class="mw-redirect" href="/wiki/%ED%95%9C%EC%96%B4%EB%B3%91%EC%9D%8C" title="한어병음">한어병음</a> 방안을 그대로 따라가면서 성조 표기를 생략할 경우 Shanxi로 표기하는 <a href="/wiki/%EC%82%B0%EC%8B%9C_%EC%84%B1_(%EC%82%B0%EC%84%9C%EC%84%B1)" title="산시 성 (산서성)">산시 성 (산서성)</a>과의 혼동이 생길 수가 있어서 이를 피하기 위해 <b>Shaanxi</b>라고 표기한다.</p>
<h2><span class="mw-headline" id=".EC.A7.80.EB.A6.AC">지리</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%82%B0%EC%8B%9C_%EC%84%B1_(%EC%84%AC%EC%84%9C%EC%84%B1)&amp;action=edit&amp;section=2" title="부분 편집: 지리">원본 편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>성 내로 <a href="/wiki/%ED%99%A9%ED%95%98" title="황하">황하</a>가 흐르며 아래에는 <a href="/wiki/%EC%B9%9C%EB%A7%81_%EC%82%B0%EB%A7%A5" title="친링 산맥">친링 산맥</a>이 있다. 중국 대륙의 거의 중앙에 위치하고 있고 북부는 <a href="/wiki/%EB%82%B4%EB%AA%BD%EA%B3%A8_%EC%9E%90%EC%B9%98%EA%B5%AC" title="내몽골 자치구">내몽골 자치구</a>와 접하고 있으며 동부는 <a href="/wiki/%EC%82%B0%EC%8B%9C_%EC%84%B1_(%EC%82%B0%EC%84%9C%EC%84%B1)" title="산시 성 (산서성)">산시 성</a>(山西), <a href="/wiki/%ED%97%88%EB%82%9C_%EC%84%B1" title="허난 성">허난 성</a>, 서부는 <a href="/wiki/%EB%8B%9D%EC%83%A4_%ED%9B%84%EC%9D%B4%EC%A1%B1_%EC%9E%90%EC%B9%98%EA%B5%AC" title="닝샤 후이족 자치구">닝샤 후이족 자치구</a>, <a href="/wiki/%EA%B0%84%EC%91%A4_%EC%84%B1" title="간쑤 성">간쑤 성</a>, 남부는 <a href="/wiki/%EC%93%B0%EC%B4%A8_%EC%84%B1" title="쓰촨 성">쓰촨 성</a>, <a href="/wiki/%EC%B6%A9%EC%B9%AD_%EC%8B%9C" title="충칭 시">충칭 시</a>, 동남부는 <a href="/wiki/%ED%9B%84%EB%B2%A0%EC%9D%B4_%EC%84%B1" title="후베이 성">후베이 성</a>과 접하고 있다.</p>
<h2><span class="mw-headline" id=".EC.97.AD.EC.82.AC">역사</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%82%B0%EC%8B%9C_%EC%84%B1_(%EC%84%AC%EC%84%9C%EC%84%B1)&amp;action=edit&amp;section=3" title="부분 편집: 역사">원본 편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>산시 성과 성도 <a href="/wiki/%EC%8B%9C%EC%95%88_%EC%8B%9C" title="시안 시">시안</a>(장안)은 고대부터 중국 문명의 요람 중의 하나였다. <a href="/wiki/%EC%A3%BC%EB%82%98%EB%9D%BC" title="주나라">주</a>부터 <a href="/wiki/%EB%8B%B9%EB%82%98%EB%9D%BC" title="당나라">당</a>에 이르기까지 1100 년이 넘는 기간 동안 13개의 왕조들이 이 지역에서 발흥하여 수도를 이 곳에 두었다. 산시 성은 <a href="/wiki/%EC%9C%A0%EB%9F%BD" title="유럽">유럽</a>, <a class="mw-redirect" href="/wiki/%EC%95%84%EB%9D%BC%EB%B9%84%EC%95%84" title="아라비아">아라비아</a>, <a href="/wiki/%EC%95%84%ED%94%84%EB%A6%AC%EC%B9%B4" title="아프리카">아프리카</a>로 이어졌던 비단길의 시점이기도 했다.</p>
<p>신석기 시대에 이 지역은 채색토기가 특징적인 <a href="/wiki/%EC%96%91%EC%82%AC%EC%98%A4_%EB%AC%B8%ED%99%94" title="양사오 문화">양사오 문화</a>가 발달한 지역이다. <a class="mw-redirect" href="/wiki/%EC%9D%80%EB%82%98%EB%9D%BC" title="은나라">은</a>(殷) 말기에 주족(周族)의 우두머리였던 문왕이 풍(豊, 시안 부근)에 도읍을 정하고 나라를 강하게 했으며, 그의 아들 <a class="mw-redirect" href="/wiki/%EB%AC%B4%EC%99%95" title="무왕">무왕</a>은 호(鎬, 시안 두문현 일대)에 도읍을 정하고 은(殷)을 공격하여 멸망시키고 <a href="/wiki/%EC%A3%BC%EB%82%98%EB%9D%BC" title="주나라">주나라</a>를 세웠다(<a class="mw-redirect" href="/wiki/%EA%B8%B0%EC%9B%90%EC%A0%84_1122%EB%85%84" title="기원전 1122년">기원전 1122년</a> 또는 <a class="mw-redirect" href="/wiki/%EA%B8%B0%EC%9B%90%EC%A0%84_1027%EB%85%84" title="기원전 1027년">기원전 1027년</a>). 전설에 따르면, 주나라 첫해에 무왕의 형제 <a class="mw-disambig" href="/wiki/%EC%A3%BC%EA%B3%B5" title="주공">주공</a>과 <a class="new" href="/w/index.php?title=%EC%86%8C%EA%B3%B5&amp;action=edit&amp;redlink=1" title="소공 (없는 문서)">소공</a> 사이의 알력으로 하남성 서부의 섬현을 경계로 하여 동서를 주공과 소공이 나누어 가지면서 섬서라는 이름이 유래하였다고 한다.</p>
<p>기원전 10세기에 주 효왕이 목축으로 유명했던 비자(非子)에게 진 지역(현재의 <a href="/wiki/%EA%B0%84%EC%91%A4_%EC%84%B1" title="간쑤 성">간쑤 성</a> <a class="new" href="/w/index.php?title=%ED%85%90%EC%88%98%EC%9D%B4&amp;action=edit&amp;redlink=1" title="텐수이 (없는 문서)">텐수이</a> 서남부)을 하사하면서 <a href="/wiki/%EC%A7%84%EB%82%98%EB%9D%BC" title="진나라">진나라</a>가 성립된다. 기원전 771년 <a class="mw-redirect" href="/wiki/%EA%B2%AC%EC%9C%B5" title="견융">견융</a>의 침입으로 호(鎬)가 함락당하고, 유왕이 살해당하는 일이 있었는데 이때 진의 양공이 견융을 치고 유왕의 아들 평왕을 낙양까지 호송한 공으로 산시 성 서부를 하사받게 된다. 기원전 659년 진 목공은 동쪽의 진(晉)나라를 쳐서 관중 전역을 지배하게 된다. 이러한 이유로 산시 성 지역은 진(秦)으로도 불린다. 진은 기원전 221년 중국을 통일한다.</p>
<p>단명한 진의 뒤를 이은 <a href="/wiki/%ED%95%9C%EB%82%98%EB%9D%BC" title="한나라">한나라</a>는 <a href="/wiki/%EC%9E%A5%EC%95%88" title="장안">장안</a>(長安)에 도읍을 정하고 장안 인근을 군(郡)에 해당하는 경조윤, 좌풍익, 우부풍 세 지역으로 나누어 통치한다. <a href="/wiki/%EC%99%95%EB%A7%9D" title="왕망">왕망</a>에 의해 왕조가 잠시 끊어진 후 한나라는 수도를 낙양으로 옮긴다. 한말에 <a href="/wiki/%EB%8F%99%ED%83%81" title="동탁">동탁</a>이 낙양을 불태우고 장안으로 천도하나 곧 죽고, 혼란이 계속되다가 결국 <a href="/wiki/%EC%9C%84_(%EC%82%BC%EA%B5%AD)" title="위 (삼국)">위</a>에 점령당한다. 이후 중원으로 진출하려는 <a class="mw-redirect" href="/wiki/%EC%B4%89%EB%82%98%EB%9D%BC" title="촉나라">촉</a>과 이를 막으려는 위의 격전이 치산을 중심으로하여 계속되었다. <a class="mw-redirect" href="/wiki/%EC%98%A4%ED%98%B8%EC%8B%AD%EC%9C%A1%EA%B5%AD" title="오호십육국">오호십육국</a>의 혼란기 동안 유목민족의 여러 나라들이 이 지역을 점령한다.</p>
<p><a href="/wiki/%EB%8B%B9%EB%82%98%EB%9D%BC" title="당나라">당나라</a> 초기에 섬북은 관내도, 섬남은 산남도에서 다스렸다가 <a class="mw-redirect" href="/wiki/%EC%95%88%EC%82%AC%EC%9D%98_%EB%82%9C" title="안사의 난">안사의 난</a> 이후 섬서 절도사가 설치되었다. <a class="mw-redirect" href="/wiki/%EC%98%A4%EB%8C%80_%EC%8B%AD%EA%B5%AD" title="오대 십국">오대 십국</a> 시대에는 잠시 기(岐)나라가 세워진다. <a href="/wiki/%EC%86%A1%EB%82%98%EB%9D%BC" title="송나라">송</a>대에는 섬서로가 설치되었으나 그 영역은 지금의 섬서성과 달랐다. 13세기 <a href="/wiki/%EB%AA%BD%EA%B3%A8" title="몽골">몽골</a>의 지배기간에 산시 성은 섬서행성이 설치되었다. <a href="/wiki/%EB%AA%85%EB%82%98%EB%9D%BC" title="명나라">명</a>대 동안에는 서쪽의 <a href="/wiki/%EA%B0%84%EC%91%A4_%EC%84%B1" title="간쑤 성">간쑤 성</a>과 합병되었으나 <a href="/wiki/%EC%B2%AD%EB%82%98%EB%9D%BC" title="청나라">청</a>대에 들어와서 다시 분리되었다.</p>
<p><a class="mw-redirect" href="/wiki/%EC%8B%A4%ED%81%AC%EB%A1%9C%EB%93%9C" title="실크로드">실크로드</a>의 동쪽 종점일 만킁 중요한 교통의 요지였던 장안은 당나라 때에 이미 인구 100만이 넘는 거대 도시로 발전하였는데, 당시 장안에는 <a href="/wiki/%EB%8F%99%EC%95%84%EC%8B%9C%EC%95%84" title="동아시아">동아시아</a>뿐만 아니라 <a href="/wiki/%EC%A4%91%EC%95%99%EC%95%84%EC%8B%9C%EC%95%84" title="중앙아시아">중앙아시아</a>, <a href="/wiki/%ED%8E%98%EB%A5%B4%EC%8B%9C%EC%95%84_%EC%A0%9C%EA%B5%AD" title="페르시아 제국">페르시아 제국</a>, <a class="mw-redirect" href="/wiki/%EC%95%84%EB%9D%BC%EB%B9%84%EC%95%84" title="아라비아">아라비아</a> 출신 무슬림들도 대거 거주하고 있었다. <a href="/wiki/%EB%B6%81%EC%86%A1" title="북송">북송</a>이 건국되고부터 <a href="/wiki/%EB%82%A8%EC%86%A1" title="남송">남송</a>이 망할 때까지(960~1279)의 기간은 <a href="/wiki/%EC%9A%94%EB%82%98%EB%9D%BC" title="요나라">요나라</a>, <a href="/wiki/%EA%B8%88%EB%82%98%EB%9D%BC" title="금나라">금나라</a>, <a href="/wiki/%EC%84%9C%ED%95%98" title="서하">서하</a>, 몽골 등과 끊임없는 전란이 있었던 시기이다. 그동안 섬서 지역에서는 한족 인구 대신에 <a href="/wiki/%EB%AC%B4%EC%8A%AC%EB%A6%BC" title="무슬림">무슬림</a>들이 많이 들어와 살게 되었다.</p>
<p>1556년 1월 23일 산시 성 남동부의 <a href="/wiki/%ED%99%94%EC%82%B0" title="화산">화산</a>에서 파멸적인 <a href="/wiki/%EC%A7%80%EC%A7%84" title="지진">지진</a>(<a href="/wiki/%EC%82%B0%EC%8B%9C_%EB%8C%80%EC%A7%80%EC%A7%84" title="산시 대지진">산시 대지진</a>)이 일어나 83만 명이 죽는 참사가 있었다.</p>
<p>1876년부터 1928년 사이의 기간 동안 수 차례의 기근이 이 지역을 휩쓸고 지나갔으며, 그동안 800만 명이 사망한 것으로 추정된다. 가난에 지쳐있던 이 지역의 농민들은 <a href="/wiki/%EB%A7%88%EC%98%A4%EC%A9%8C%EB%91%A5" title="마오쩌둥">마오쩌둥</a>이 이끈 공산혁명기간 동안 강한 지지를 보냈다.</p>
<p>1935년 10월에 <a class="mw-redirect" href="/wiki/%EA%B0%95%EC%84%9C_%EC%86%8C%EB%B9%84%EC%97%90%ED%8A%B8" title="강서 소비에트">강서 소비에트</a>에서 탈출한 마오쩌둥을 비롯한 중국 공산주의자들이 <a href="/wiki/%EC%9E%A5%EC%A0%95" title="장정">장정</a> 끝에 산시 성 북부의 <a class="mw-redirect" href="/wiki/%EC%98%8C%EC%95%88" title="옌안">옌안</a>에 도착하였고 여기서 중국 공산화의 기틀을 마련하였다. 이듬해 말 <a href="/wiki/%EC%9E%A5%EC%A0%9C%EC%8A%A4" title="장제스">장제스</a>는 산서에서 세력을 키우고 있는 홍군을 박멸하기 위해 <a href="/wiki/%EC%8B%9C%EC%95%88_%EC%8B%9C" title="시안 시">시안</a>에 왔다. 이때 만주군벌 <a href="/wiki/%EC%9E%A5%EC%AD%A4%EB%A6%B0" title="장쭤린">장쭤린</a>의 아들 <a href="/wiki/%EC%9E%A5%EC%89%90%EB%9F%89" title="장쉐량">장쉐량</a>이 장제스를 감금한 뒤, 내전을 멈추고 항일전선에 나설것을 요구한다. 장제스는 이를 수용하여 <a class="mw-redirect" href="/wiki/%EC%A0%9C2%EC%B0%A8_%EA%B5%AD%EA%B3%B5%ED%95%A9%EC%9E%91" title="제2차 국공합작">제2차 국공합작</a>이 성립되는데, 이 사건을 <a class="mw-redirect" href="/wiki/%EC%8B%9C%EC%95%88%EC%82%AC%EB%B3%80" title="시안사변">시안사변</a>이라고 한다.</p>
<h2><span class="mw-headline" id=".ED.96.89.EC.A0.95.EA.B5.AC.EC.97.AD">행정구역</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%82%B0%EC%8B%9C_%EC%84%B1_(%EC%84%AC%EC%84%9C%EC%84%B1)&amp;action=edit&amp;section=4" title="부분 편집: 행정구역">원본 편집</a><span class="mw-editsection-bracket">]</span></span></h2>

<h2><span class="mw-headline" id=".EA.B2.BD.EC.A0.9C">경제</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%82%B0%EC%8B%9C_%EC%84%B1_(%EC%84%AC%EC%84%9C%EC%84%B1)&amp;action=edit&amp;section=5" title="부분 편집: 경제">원본 편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>산시 성의 2008년 명목 GDP는 6851억 위안이었고 1인당 GDP는 18246위안(2671달러)로 중국 전체에서 22위를 차지했다.</p>
<h2><span class="mw-headline" id=".EC.9D.B8.EA.B5.AC">인구</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%82%B0%EC%8B%9C_%EC%84%B1_(%EC%84%AC%EC%84%9C%EC%84%B1)&amp;action=edit&amp;section=6" title="부분 편집: 인구">원본 편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>산시 성의 인구 거의 대부분이 <a href="/wiki/%ED%95%9C%EC%A1%B1" title="한족">한족</a>이고 북서쪽으로 <a class="mw-redirect" href="/wiki/%EB%8B%9D%EC%83%A4" title="닝샤">닝샤</a>와 인접한 곳에 약간의 <a href="/wiki/%ED%9B%84%EC%9D%B4%EC%A1%B1" title="후이족">후이족</a>이 거주한다. 산시 성 남쪽 성도 시안이 위치한 관중으로 알려진 곳은 북쪽보다 인구가 밀집되어있다.</p>
