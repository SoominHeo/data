<p><b>산인 중앙 TV 방송</b>은 <a href="/wiki/1970%EB%85%84" title="1970년">1970년</a> <a href="/wiki/4%EC%9B%94_1%EC%9D%BC" title="4월 1일">4월 1일</a> 시마네현의 민영방송국으로 개국했다가 <a href="/wiki/1972%EB%85%84" title="1972년">1972년</a> 돗토리현에도 방송을 시작한 방송국이다.</p>
<p>약칭은 예전 영어회사명 <b>Television Shimane K.K.</b>에서 따온 TSK, 콜사인은 JOMI-DTV이다.</p>

<li><a href="/wiki/1970%EB%85%84" title="1970년">1970년</a> <a href="/wiki/4%EC%9B%94_1%EC%9D%BC" title="4월 1일">4월 1일</a> <a href="/wiki/%EC%8B%9C%EB%A7%88%EB%84%A4_%ED%98%84" title="시마네 현">시마네 현</a>에서 방송 개시.</li>

<li><a href="/wiki/2006%EB%85%84" title="2006년">2006년</a> <a href="/wiki/10%EC%9B%94_1%EC%9D%BC" title="10월 1일">10월 1일</a> 지상파 디지털 텔레비전 방송 개시.</li>

<h2><span class="mw-headline" id=".EB.84.A4.ED.8A.B8.EC.9B.8C.ED.81.AC_.EB.B3.80.EC.B2.9C.EC.82.AC">네트워크 변천사</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%82%B0%EC%9D%B8_%EC%A4%91%EC%95%99_TV_%EB%B0%A9%EC%86%A1&amp;action=edit&amp;section=2" title="부분 편집: 네트워크 변천사">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<ul>
<li><a href="/wiki/1970%EB%85%84" title="1970년">1970년</a> <a href="/wiki/4%EC%9B%94_1%EC%9D%BC" title="4월 1일">4월 1일</a> <a class="mw-redirect" href="/wiki/%EC%8B%9C%EB%A7%88%EB%84%A4%ED%98%84" title="시마네현">시마네현</a>에서 텔레비전 방송 개시(시마네현 2번째 민영방송국).그와 동시에 산인지역 첫 번째 완전가맹국으로 개국했다.개국 당시부터 <a href="/wiki/%ED%9B%84%EC%A7%80_TV" title="후지 TV">후지 TV</a>계열국이다.(개국과 동시에<a href="/wiki/%ED%9B%84%EC%A7%80_%EB%89%B4%EC%8A%A4_%EB%84%A4%ED%8A%B8%EC%9B%8C%ED%81%AC" title="후지 뉴스 네트워크">FNN</a>·<a href="/wiki/%ED%9B%84%EC%A7%80_%EB%84%A4%ED%8A%B8%EC%9B%8C%ED%81%AC" title="후지 네트워크">FNS</a>에 가맹.)</li>
<li><a href="/wiki/1972%EB%85%84" title="1972년">1972년</a> <a href="/wiki/9%EC%9B%94_22%EC%9D%BC" title="9월 22일">9월 22일</a> 시마네·톳토리 방송구역 광역화에 의해 <a class="mw-redirect" href="/wiki/%EB%8F%97%ED%86%A0%EB%A6%AC%ED%98%84" title="돗토리현">돗토리현</a>에서 방송 개시(돗토리현 3번째 민영방송국).이와 동시에 <a href="/wiki/%EB%8B%88%ED%98%BC%EC%B9%B4%EC%9D%B4_TV" title="니혼카이 TV">니혼카이 TV</a>(돗토리 1번째 민영방송국·시마네 3번째 민영방송국)가 담당하고 있던 <a href="/wiki/%EB%8F%97%ED%86%A0%EB%A6%AC_%ED%98%84" title="돗토리 현">돗토리 현</a>에서의 후지텔레비전 계열 업무를 계승했다.</li>
</ul>
<h3><span class="mw-headline" id=".ED.9B.84.EC.A7.80.ED.85.94.EB.A0.88.EB.B9.84.EC.A0.84_.EA.B3.84.EC.97.B4.EA.B5.AD.EC.9D.B4_.EB.90.9C_.EA.B2.BD.EC.9C.84">후지텔레비전 계열국이 된 경위</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%82%B0%EC%9D%B8_%EC%A4%91%EC%95%99_TV_%EB%B0%A9%EC%86%A1&amp;action=edit&amp;section=3" title="부분 편집: 후지텔레비전 계열국이 된 경위">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<ul>
<li>산인중앙텔레비전은 <a href="/wiki/%EC%82%B0%EC%9D%B8_%EC%A4%91%EC%95%99_%EC%8B%A0%EB%B3%B4" title="산인 중앙 신보">산인 중앙 신보</a>등에 의해서 설립된 방송국이다.원래 이 신문은 <a href="/wiki/%EC%9A%94%EB%AF%B8%EC%9A%B0%EB%A6%AC_%EC%8B%A0%EB%AC%B8" title="요미우리 신문">요미우리 신문</a>과 친밀했기 때문에 텔레비전 방송국을 개국하면 <a class="mw-redirect" href="/wiki/%EB%8B%88%ED%98%BCTV" title="니혼TV">니혼TV</a>계열국이 되기를 희망하고 있었다.그렇지만 훗날 <a class="mw-redirect" href="/wiki/%EB%8F%97%ED%86%A0%EB%A6%AC%ED%98%84" title="돗토리현">돗토리현</a>과의 방송구역 광역화를 실시하는 것을 고려하는 동시에(당시 니혼TV 계열 프로그램의 상당수를 NKT가 방송했다.) 당시<a href="/wiki/%ED%9B%84%EC%A7%80_TV" title="후지 TV">후지 TV</a>가 UHF 방송국 대량 개국에 따라 네트워크 가맹국을 늘리려 했기 때문에 후지TV 계열(FNN, FNS)방송국으로 개국했다.</li>
<li>이와 반대로 니혼카이TV는 원래<a href="/wiki/%EC%82%B0%EC%BC%80%EC%9D%B4_%EC%8B%A0%EB%AC%B8" title="산케이 신문">산케이 신문</a>이 주체가 되어 개국한 텔레비전 방송국이지만 니혼TV가 후지TV보다 스포츠 중계 등 여러 가지 면에서 유리해 라이벌 방송국이라 여겨졌던 <a class="mw-redirect" href="/wiki/%EB%8F%84%EC%BF%84%EB%B0%A9%EC%86%A1" title="도쿄방송">라디오도쿄(현재의 TBS)</a>보다 영업 성적이 좋았다.또한 뉴스 네트워크도 JNN에서 탈퇴한 이후 자연히 니혼TV 뉴스를 방송하기 시작했다.이런 이유로 TSK 개국시에 NKT는 사실상 니혼TV 계열국으로 변해가고 있었던 것이다.</li>
<li>이렇게 해서 방송구역 광역화 후에 두 방송국의 신문자본 관계가 이상하게 꼬였지만 후에 요미우리·후지산케이 진영에서 자기들이 가지고 있던 각 방송국의 자본을 교환하면서 TSK는 후지산케이 그룹과 친밀한 관계를 맺게 되었다.</li>
</ul>
