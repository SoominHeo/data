<p><b>산타크루스데테네리페 주</b>(<a class="mw-redirect" href="/wiki/%EC%97%90%EC%8A%A4%ED%8C%8C%EB%83%90%EC%96%B4" title="에스파냐어">에스파냐어</a>: Santa Cruz de Tenerife)는 <a class="mw-redirect" href="/wiki/%EC%97%90%EC%8A%A4%ED%8C%8C%EB%83%90" title="에스파냐">에스파냐</a>의 <a class="mw-redirect" href="/wiki/%EC%97%90%EC%8A%A4%ED%8C%8C%EB%83%90%EC%9D%98_%EC%A3%BC" title="에스파냐의 주">주</a>로서 <a href="/wiki/%EC%B9%B4%EB%82%98%EB%A6%AC%EC%95%84_%EC%A0%9C%EB%8F%84" title="카나리아 제도">카나리아 제도</a> <a class="mw-redirect" href="/wiki/%EC%97%90%EC%8A%A4%ED%8C%8C%EB%83%90%EC%9D%98_%EC%9E%90%EC%B9%98_%EC%A7%80%EB%B0%A9" title="에스파냐의 자치 지방">자치 지방</a>의 서쪽 부분을 이룬다. 산타 크루스 데 테네리페 주는 <a href="/wiki/%EB%8C%80%EC%84%9C%EC%96%91" title="대서양">대서양</a> 군도의 절반 가량을 차지하며 <a href="/wiki/%ED%85%8C%EB%84%A4%EB%A6%AC%ED%8E%98_%EC%84%AC" title="테네리페 섬">테네리페 섬</a>, <a href="/wiki/%EB%9D%BC%EA%B3%A0%EB%A9%94%EB%9D%BC_%EC%84%AC" title="라고메라 섬">라고메라 섬</a>, <a href="/wiki/%EC%97%98%EC%9D%B4%EC%97%90%EB%A1%9C_%EC%84%AC" title="엘이에로 섬">엘이에로 섬</a>, <a href="/wiki/%EB%9D%BC%ED%8C%94%EB%A7%88_%EC%84%AC" title="라팔마 섬">라팔마 섬</a> 등의 섬이 있다.</p>
<p>주도는 <a href="/wiki/%ED%85%8C%EB%84%A4%EB%A6%AC%ED%8E%98_%EC%84%AC" title="테네리페 섬">테네리페 섬</a>의 <a href="/wiki/%EC%82%B0%ED%83%80%ED%81%AC%EB%A3%A8%EC%8A%A4%EB%8D%B0%ED%85%8C%EB%84%A4%EB%A6%AC%ED%8E%98" title="산타크루스데테네리페">산타크루스데테네리페</a>이다. 주 인구 971,647명(2006년 기준) 가운데 24%가 주도에 살고 있다. 이 주에는 53개 <a class="mw-redirect" href="/wiki/%EC%A7%80%EC%9E%90%EC%B2%B4" title="지자체">지자체</a>가 있다. 테네리페는 이 주에서 가장 사람이 많이 사는 섬으로, 에스파냐 전체로 봐도 섬 가운데 사람이 가장 많이 사는 곳이다. (이 군도의 다른 반쪽을 차지하는 주는 <a href="/wiki/%EB%9D%BC%EC%8A%A4%ED%8C%94%EB%A7%88%EC%8A%A4_%EC%A3%BC" title="라스팔마스 주">라스팔마스 주</a>이다.)</p>
<p>산타 크루스 데 테네리페 주에는 <a class="new" href="/w/index.php?title=%EC%97%90%EC%8A%A4%ED%8C%8C%EB%83%90%EC%9D%98_%EA%B5%AD%EB%A6%BD%EA%B3%B5%EC%9B%90&amp;action=edit&amp;redlink=1" title="에스파냐의 국립공원 (없는 문서)">에스파냐의 국립공원</a> 세 곳이 있어, 다른 주보다 많다. 라팔마 섬의 <a class="new" href="/w/index.php?title=%EB%9D%BC_%EC%B9%BC%EB%8D%B0%EB%9D%BC_%EB%8D%B0_%ED%83%80%EB%B6%80%EB%A6%AC%EC%97%94%ED%85%8C_%EA%B5%AD%EB%A6%BD%EA%B3%B5%EC%9B%90&amp;action=edit&amp;redlink=1" title="라 칼데라 데 타부리엔테 국립공원 (없는 문서)">라 칼데라 데 타부리엔테 국립공원</a>, 라 고메라의 <a class="new" href="/w/index.php?title=%EA%B0%80%EB%9D%BC%ED%98%B8%EB%82%98%EC%9D%B4_%EA%B5%AD%EB%A6%BD%EA%B3%B5%EC%9B%90&amp;action=edit&amp;redlink=1" title="가라호나이 국립공원 (없는 문서)">가라호나이 국립공원</a>, 그리고 테네리페에 있으며 에스파냐에서 가장 높은 산이자 휴화산인 <a class="mw-redirect" href="/wiki/%ED%85%8C%EC%9D%B4%EB%8D%B0_%EC%82%B0" title="테이데 산">테이데 산</a>이 있는 <a class="new" href="/w/index.php?title=%ED%85%8C%EC%9D%B4%EB%8D%B0_%EA%B5%AD%EB%A6%BD%EA%B3%B5%EC%9B%90&amp;action=edit&amp;redlink=1" title="테이데 국립공원 (없는 문서)">테이데 국립공원</a>이 바로 그러하다. <a href="/wiki/%EC%82%B0%ED%81%AC%EB%A6%AC%EC%8A%A4%ED%86%A0%EB%B0%9C%EB%8D%B0%EB%9D%BC%EB%9D%BC%EA%B5%AC%EB%82%98" title="산크리스토발데라라구나">산크리스토발데라라구나</a>(<a class="mw-redirect" href="/wiki/%EC%84%B8%EA%B3%84%EB%AC%B8%ED%99%94%EC%9C%A0%EC%82%B0" title="세계문화유산">세계문화유산</a>)는 테네리페 섬 제2의 도시이며, 군도에서 3번째로 큰 도시이다. 엘 테이도 역시 세계 문화 유산이다.</p>
