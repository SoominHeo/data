<p><b>산헤드린</b>(<span style="font-size: smaller;"><a href="/wiki/%ED%9E%88%EB%B8%8C%EB%A6%AC%EC%96%B4" title="히브리어">히브리어</a>:</span> <span lang="he" xml:lang="he">סנהדרין</span>, <span style="font-size: smaller;"><a href="/wiki/%EA%B7%B8%EB%A6%AC%EC%8A%A4%EC%96%B4" title="그리스어">그리스어</a>:</span> <span lang="el" xml:lang="el">συνέδριον</span><sup class="reference" id="cite_ref-1"><a href="#cite_note-1">[1]</a></sup>, synedrion, <span lang="en" xml:lang="en">sanhderin</span> 모여 앉는다)은 모든 유대의 도시에 <a class="new" href="/w/index.php?title=%EC%9C%A0%EB%8C%80%EA%B5%90_%EB%B2%95&amp;action=edit&amp;redlink=1" title="유대교 법 (없는 문서)">유대교 법</a>에 의해 세워진 23명의 판관들의 모임이다.<sup class="reference" id="cite_ref-2"><a href="#cite_note-2">[2]</a></sup> <a class="mw-redirect" href="/wiki/%EB%A1%9C%EB%A7%88%EC%A0%9C%EA%B5%AD" title="로마제국">로마제국</a>에서는 반(反)로마투쟁을 하지 않는다면, <a href="/wiki/%EC%8B%9D%EB%AF%BC%EC%A7%80" title="식민지">식민지</a>의 <a href="/wiki/%EC%A2%85%EA%B5%90" title="종교">종교</a>와 자치활동을 인정했으므로 고대 <a href="/wiki/%EC%9D%B4%EC%8A%A4%EB%9D%BC%EC%97%98" title="이스라엘">이스라엘</a> 사회에서 산헤드린 의회는 영향력이 있는 집단이었다.</p>
<p><b>대 산헤드린</b>(<span lang="en" xml:lang="en">Great Sanhedrin</span>)은 <a href="/wiki/%EC%9C%A0%EB%8C%80%EA%B5%90" title="유대교">유대교</a>의 판관들의 모임으로 최고 법원 역할을 하는 고대 <a href="/wiki/%EC%9D%B4%EC%8A%A4%EB%9D%BC%EC%97%98" title="이스라엘">이스라엘</a>의 재판 기구였다. 대 산헤드린은 지도자인 <a class="mw-redirect" href="/wiki/%EB%82%98%EC%89%AC" title="나쉬">나쉬</a>, <a href="/wiki/%EC%BD%94%ED%97%A8_%EA%B0%80%EB%8F%8C" title="코헨 가돌">코헨 가돌</a>(대제사장/대사제), 부 판관인 <a class="new" href="/w/index.php?title=%EC%95%84%EB%B8%8C_%EB%B2%A0%EC%9E%87_%EB%94%98&amp;action=edit&amp;redlink=1" title="아브 베잇 딘 (없는 문서)">아브 베잇 딘</a>, 그리고 69명의 평회원으로 이루어졌다. <a class="new" href="/w/index.php?title=%EA%B0%9C%EC%97%AD_%ED%95%9C%EA%B8%80%ED%8C%90&amp;action=edit&amp;redlink=1" title="개역 한글판 (없는 문서)">개역 한글판</a> 성서에서는 <b>공회</b><sup class="reference" id="cite_ref-3"><a href="#cite_note-3">[3]</a></sup>로, <a href="/wiki/%EA%B3%B5%EB%8F%99%EB%B2%88%EC%97%AD%EC%84%B1%EC%84%9C" title="공동번역성서">공동번역성서</a>에서는 <b>의회</b>로 번역한 산헤드린은 대 산헤드린을 이른다. <a href="/wiki/70%EB%85%84" title="70년">70년</a> 멸망하기 전의 <a href="/wiki/%EC%98%88%EB%A3%A8%EC%82%B4%EB%A0%98" title="예루살렘">예루살렘</a>의 <a class="new" href="/w/index.php?title=%EC%A0%9C_2%EC%84%B1%EC%A0%84&amp;action=edit&amp;redlink=1" title="제 2성전 (없는 문서)">제 2성전</a>시기에, 대 산헤드린은 성전 안에 있는 <a class="new" href="/w/index.php?title=%EB%8B%A4%EB%93%AC%EC%9D%80_%EB%8F%8C_%EB%9C%B0&amp;action=edit&amp;redlink=1" title="다듬은 돌 뜰 (없는 문서)">다듬은 돌 뜰</a>에 낮 동안 열렸으며, <a href="/wiki/%EC%95%88%EC%8B%9D%EC%9D%BC" title="안식일">안식일</a>과 <a class="new" href="/w/index.php?title=%EC%9C%A0%EB%8C%80%EA%B5%90_%EC%B6%95%EC%A0%9C%EC%9D%BC&amp;action=edit&amp;redlink=1" title="유대교 축제일 (없는 문서)">유대교 축제일</a>에는 열리지 않았다.</p>
<p><a href="/wiki/%EC%9C%A0%EB%8C%80" title="유대">유대</a>에서의 주요한 역할 이외에 산헤드린은 <a href="/wiki/%EA%B8%B0%EB%8F%85%EA%B5%90" title="기독교">기독교</a>를 세우는 중요한 회의가 되었으며, <a class="new" href="/w/index.php?title=%EC%98%88%EC%88%98%EC%9D%98_%EC%82%B0%ED%97%A4%EB%93%9C%EB%A6%B0_%EC%9E%AC%ED%8C%90&amp;action=edit&amp;redlink=1" title="예수의 산헤드린 재판 (없는 문서)">예수의 산헤드린 재판</a><sup class="reference" id="cite_ref-4"><a href="#cite_note-4">[4]</a></sup>과 관련하여 <a href="/wiki/%EB%B3%B5%EC%9D%8C%EC%84%9C" title="복음서">복음서</a>에 기술되어 있다. 하지만 성서학자들중에는 예수의 산헤드린 의회에서의 재판에 대해 역사적 사실성을 의심한다. 일개 <a href="/wiki/%EB%AF%BC%EC%A4%91" title="민중">민중</a>에 불과한 <a href="/wiki/%EC%98%88%EC%88%98" title="예수">예수</a>를 산헤드린 의회에는 엘리트 집단에서 <a href="/wiki/%EC%9E%AC%ED%8C%90" title="재판">재판</a>을 했을지, 유대교 율법에서는 밤에는 재판을 할 수 없는데 <a class="mw-redirect" href="/wiki/%EB%A7%88%ED%83%9C%EC%98%A4_%EB%B3%B5%EC%9D%8C%EC%84%9C" title="마태오 복음서">마태오 복음서</a>와 마르코 복음서에서는 밤에 재판이 열렸다고 기술하기 때문이다.<sup class="reference" id="cite_ref-5"><a href="#cite_note-5">[5]</a></sup> <sup class="reference" id="cite_ref-6"><a href="#cite_note-6">[6]</a></sup><a class="mw-redirect" href="/wiki/%EC%82%AC%EB%8F%84%ED%96%89%EC%A0%84" title="사도행전">사도행전</a>에도 성 <a class="mw-redirect" href="/wiki/%EB%B0%94%EC%9A%B8" title="바울">바울</a>과 성 <a class="mw-redirect" href="/wiki/%EC%8A%A4%EB%8D%B0%ED%8C%8C%EB%85%B8" title="스데파노">스데파노</a>가 산헤드린 의회에서 재판을 받은 이야기가 나온다.<sup class="reference" id="cite_ref-7"><a href="#cite_note-7">[7]</a></sup><sup class="reference" id="cite_ref-8"><a href="#cite_note-8">[8]</a></sup></p>
