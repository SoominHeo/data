<p><b>살바도르 카바냐스 오르테가</b>(<span style="font-size: smaller;"><a href="/wiki/%EC%8A%A4%ED%8E%98%EC%9D%B8%EC%96%B4" title="스페인어">스페인어</a>:</span> <span lang="es" xml:lang="es">Salvador Cabañas Ortega</span>, <a href="/wiki/1980%EB%85%84" title="1980년">1980년</a> <a href="/wiki/8%EC%9B%94_5%EC%9D%BC" title="8월 5일">8월 5일</a> <a href="/wiki/%EC%95%84%EC%88%9C%EC%8B%9C%EC%98%A8" title="아순시온">아순시온</a> ~ )는 <a href="/wiki/%ED%8C%8C%EB%9D%BC%EA%B3%BC%EC%9D%B4" title="파라과이">파라과이</a>의 <a href="/wiki/%EC%B6%95%EA%B5%AC" title="축구">축구</a> 선수로, 포지션은 <a href="/wiki/%EA%B3%B5%EA%B2%A9%EC%88%98" title="공격수">공격수</a>이다.</p>
<p><a href="/wiki/1998%EB%85%84" title="1998년">1998년</a> <a href="/wiki/%EB%8F%84%EC%84%B8_%EB%8D%B0_%EC%98%A5%ED%88%AC%EB%B8%8C%EB%A0%88_FC" title="도세 데 옥투브레 FC">도세 데 옥투브레</a>에서 데뷔했으며 <a href="/wiki/2001%EB%85%84" title="2001년">2001년</a> <a href="/wiki/%EC%B9%A0%EB%A0%88" title="칠레">칠레</a>의 <a href="/wiki/%EC%95%84%EC%9A%B0%EB%8B%A5%EC%8A%A4_%EC%9D%B4%ED%83%88%EB%A6%AC%EC%95%84%EB%85%B8" title="아우닥스 이탈리아노">아우닥스 이탈리아노</a>로 이적했고 <a href="/wiki/2003%EB%85%84" title="2003년">2003년</a>에는 <a href="/wiki/%EB%A9%95%EC%8B%9C%EC%BD%94" title="멕시코">멕시코</a>의 <a class="new" href="/w/index.php?title=%ED%95%98%EA%B3%BC%EB%A0%88%EC%8A%A4_%EB%8D%B0_%EC%B9%98%EC%95%84%ED%8C%8C%EC%8A%A4&amp;action=edit&amp;redlink=1" title="하과레스 데 치아파스 (없는 문서)">하과레스 데 치아파스</a>로 이적했다. 2003년 아페르투라 시즌에서 18경기에 출전하면서 5득점을 기록했으며 <a href="/wiki/2004%EB%85%84" title="2004년">2004년</a> 클라우수라 시즌에서 20경기에 출전하면서 15득점을 기록했다.</p>
<p><a href="/wiki/2006%EB%85%84" title="2006년">2006년</a> 클라우수라 시즌에서 14득점을 기록하면서 득점왕에 올랐고 같은 해에 <a href="/wiki/%ED%81%B4%EB%A3%A8%EB%B8%8C_%EC%95%84%EB%A9%94%EB%A6%AC%EC%B9%B4" title="클루브 아메리카">클루브 아메리카</a>로 이적했다. <a href="/wiki/2007%EB%85%84" title="2007년">2007년</a> <a href="/wiki/%EC%BD%94%ED%8C%8C_%EB%A6%AC%EB%B2%A0%EB%A5%B4%ED%83%80%EB%8F%84%EB%A0%88%EC%8A%A4" title="코파 리베르타도레스">코파 리베르타도레스</a>에서 10득점을 기록하면서 대회 득점왕에 올랐고 같은 해에 파라과이 올해의 축구 선수와 <a href="/wiki/%EB%82%A8%EC%95%84%EB%A9%94%EB%A6%AC%EC%B9%B4_%EC%98%AC%ED%95%B4%EC%9D%98_%EC%B6%95%EA%B5%AC_%EC%84%A0%EC%88%98" title="남아메리카 올해의 축구 선수">남아메리카 올해의 축구 선수</a>로 선정되었다. <a href="/wiki/2008%EB%85%84" title="2008년">2008년</a> 코파 리베르타도레스에서 8득점을 기록하면서 2회 연속으로 대회 득점왕에 올랐다.</p>
<p><a href="/wiki/2010%EB%85%84" title="2010년">2010년</a> <a href="/wiki/1%EC%9B%94_25%EC%9D%BC" title="1월 25일">1월 25일</a> 부인과 함께 <a class="mw-redirect" href="/wiki/%EB%A9%95%EC%8B%9C%EC%BD%94_%EC%8B%9C" title="멕시코 시">멕시코 시티</a>에 있는 한 나이트클럽에 갔다가 화장실에서 괴한들이 쏜 총에 머리를 맞아 중태에 빠졌고<sup class="reference" id="cite_ref-1"><a href="#cite_note-1">[1]</a></sup> 인근 병원으로 후송되어 머리에 맞았던 22mm 크기의 총알을 제거하는 수술을 받았다.<sup class="reference" id="cite_ref-2"><a href="#cite_note-2">[2]</a></sup> 얼마 뒤 현역으로 복귀해 자국 아마추어격 리그에서 선수생활을 이어가게 되었다.</p>
