<p><b>삼별초의 난</b>(三別抄의 亂, <a href="/wiki/1270%EB%85%84" title="1270년">1270년</a> ~ <a href="/wiki/1273%EB%85%84" title="1273년">1273년</a>)은 <a href="/wiki/%EA%B3%A0%EB%A0%A4-%EB%AA%BD%EA%B3%A8_%EC%A0%84%EC%9F%81" title="고려-몽골 전쟁">고려-몽골 전쟁</a>이 끝난 뒤 <a href="/wiki/%EC%82%BC%EB%B3%84%EC%B4%88" title="삼별초">삼별초</a>가 <a href="/wiki/%EB%AA%BD%EA%B3%A8_%EC%A0%9C%EA%B5%AD" title="몽골 제국">몽골</a>(<a href="/wiki/%EC%9B%90%EB%82%98%EB%9D%BC" title="원나라">원나라</a>) 및 <a href="/wiki/%EA%B3%A0%EB%A0%A4" title="고려">고려</a> 왕조에 대항하여 일으킨 항쟁이다. <b>삼별초의 대몽항쟁</b>(三別抄의 對蒙抗爭) 또는 <b>삼별초의 항쟁</b>(三別抄의 抗爭)으로 불리기도 한다.</p>

<h2><span class="mw-headline" id=".EB.B0.B0.EA.B2.BD">배경</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%82%BC%EB%B3%84%EC%B4%88%EC%9D%98_%EB%82%9C&amp;action=edit&amp;section=1" title="부분 편집: 배경">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>다른 반란군들과는 다르게 <a href="/wiki/%EC%82%BC%EB%B3%84%EC%B4%88" title="삼별초">삼별초</a>는 난을 일으키기 전에는 고려 정부의 지휘하에 있었다.</p>
<p>삼별초는 <b><a class="mw-redirect" href="/wiki/%EC%B5%9C%EC%94%A8_%EB%AC%B4%EC%8B%A0%EC%A0%95%EA%B6%8C" title="최씨 무신정권">최씨 무신정권</a>의 사병 집단</b>이었고, <b>군부 독재를 유지</b>하는 역할을 했다. <a href="/wiki/%EB%AA%BD%EA%B3%A8" title="몽골">몽골</a>이 고려를 침입해 오자, <a class="mw-redirect" href="/wiki/%EC%B5%9C%EC%94%A8_%EB%AC%B4%EC%8B%A0%EC%A0%95%EA%B6%8C" title="최씨 무신정권">최씨 무신정권</a>은 <a href="/wiki/1232%EB%85%84" title="1232년">1232년</a>에 <a href="/wiki/%EA%B0%95%ED%99%94%EB%8F%84" title="강화도">강화도</a>로 수도를 옮긴 후 <a href="/wiki/%EB%AA%BD%EA%B3%A8" title="몽골">몽골</a>의 침략에 맞섰다. 하지만 허약한 기반으로 말미암아 고려는 잦은 반란에 직면하게 된다. 반란 중 일부는 평정되었지만, 북부 지역의 반란 무리는 몽골로 이탈하고, 그 영토는 몽골 제국에 병합되었다. 또한 <a class="mw-redirect" href="/wiki/%EC%B5%9C%EC%94%A8_%EB%AC%B4%EC%8B%A0%EC%A0%95%EA%B6%8C" title="최씨 무신정권">최씨 무신정권</a>이 무너지고 고려는 몽골 제국에 항복하였다.</p>
<p>몽골의 지원으로 군부 정권으로부터 권력을 되찾게 된 <a href="/wiki/%EA%B3%A0%EB%A0%A4_%EC%9B%90%EC%A2%85" title="고려 원종">원종</a>은 <a href="/wiki/1270%EB%85%84" title="1270년">1270년</a>(원종 11년) 삼별초를 해산하기로 결정하였다. 그러나 <a href="/wiki/%EB%B0%B0%EC%A4%91%EC%86%90" title="배중손">배중손</a>이 이끄는 강화도의 삼별초는 원종의 고려 정부에 반기를 들고 봉기하였다.<sup class="reference" id="cite_ref-.EC.A1.B0.EC.84.A0.EB.8B.B7.EC.BB.B4_1-0"><a href="#cite_note-.EC.A1.B0.EC.84.A0.EB.8B.B7.EC.BB.B4-1">[1]</a></sup><sup class="reference" id="cite_ref-.EC.98.A4.EB.A7.88.EC.9D.B4_2-0"><a href="#cite_note-.EC.98.A4.EB.A7.88.EC.9D.B4-2">[2]</a></sup></p>
<h2><span class="mw-headline" id=".EA.B2.BD.EA.B3.BC">경과</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%82%BC%EB%B3%84%EC%B4%88%EC%9D%98_%EB%82%9C&amp;action=edit&amp;section=2" title="부분 편집: 경과">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>애초에 무신정권의 사병 집단이면서 대몽 항쟁에서 선봉에 섰던 삼별초는 그러한 강화(講和)와 무신정권의 화해를 못마땅하게 여겼으며, 개경 환도가 발표되자 즉각 반기를 들었다.</p>
<p>삼별초 지유(指諭) <a href="/wiki/%EB%B0%B0%EC%A4%91%EC%86%90" title="배중손">배중손</a>과 야별초(夜別抄) 지유 <a class="new" href="/w/index.php?title=%EB%85%B8%EC%98%81%ED%9D%AC&amp;action=edit&amp;redlink=1" title="노영희 (없는 문서)">노영희</a>(盧永禧) 등은 항쟁을 결의하였다. 배중손을 지도자로 추대하고 강화도와 육지와의 교통을 끊었으며, 왕족 승화후 온(承化侯溫)을 왕으로 추대하여 관부(官府)를 설치하고 관리를 임명하여 반몽정권(反蒙政權)을 수립했다.<sup class="reference" id="cite_ref-.EC.A1.B0.EC.84.A0.EB.8B.B7.EC.BB.B4_1-1"><a href="#cite_note-.EC.A1.B0.EC.84.A0.EB.8B.B7.EC.BB.B4-1">[1]</a></sup></p>
<p>그러나 이탈자가 속출하여 경계가 어렵게 되자 1천여 함선을 징발하여, 고려 정부의 재화와 백성을 모두 싣고 강화도를 떠나 서해안 요지를 공략하며 남행하여 <a class="mw-redirect" href="/wiki/%EC%A7%84%EB%8F%84_(%EC%A7%84%EB%8F%84%EA%B5%B0)" title="진도 (진도군)">진도</a>에 이르렀다. 그곳에 항구적인 근거지를 두고 용장사를 행궁으로 삼았다. 그 뒤 용장사 주변에 <a class="mw-redirect" href="/wiki/%EC%82%B0%EC%84%B1_(%EA%B1%B4%EC%A1%B0%EB%AC%BC)" title="산성 (건조물)">산성</a>을 쌓고 관아도 세웠으며 제법 도읍지의 면모를 갖추고 그들은 역동적으로 움직였으며, 고려의 유일한 정통 정부임을 주장했으며<sup class="reference" id="cite_ref-.EC.98.A4.EB.A7.88.EC.9D.B4_2-1"><a href="#cite_note-.EC.98.A4.EB.A7.88.EC.9D.B4-2">[2]</a></sup> 일본과 연계하고자 외교도 펼쳤다. 당시 진도와 그 인근 지역에는 과거 <a class="mw-redirect" href="/wiki/%EC%B5%9C%EC%94%A8_%EB%AC%B4%EC%8B%A0%EC%A0%95%EA%B6%8C" title="최씨 무신정권">최씨 무신정권</a>이 소유한 농장이 그때까지도 대규모로 존재했다. 동시에 <a href="/wiki/%EA%B2%BD%EC%83%81%EB%8F%84" title="경상도">경상도</a>와 <a href="/wiki/%EC%A0%84%EB%9D%BC%EB%8F%84" title="전라도">전라도</a> 지방의 세곡이 서울로 운송되는 길목에 진도가 있었으며, 이는 <a class="mw-redirect" href="/wiki/%EC%A1%B0%EC%84%B8" title="조세">세곡</a>으로 운반되는 식량과 자금을 빼앗아 자체 군량으로 쓰는 동시에 개경 정부를 압박할 수 있는 요충지였다.<sup class="reference" id="cite_ref-.EC.A1.B0.EC.84.A0.EB.8B.B7.EC.BB.B4_1-2"><a href="#cite_note-.EC.A1.B0.EC.84.A0.EB.8B.B7.EC.BB.B4-1">[1]</a></sup></p>
<p>또한 남해 연안과 각 도서·<a href="/wiki/%EB%82%98%EC%A3%BC%EC%8B%9C" title="나주시">나주</a>·<a class="mw-redirect" href="/wiki/%EC%9E%A5%ED%9D%A5" title="장흥">장흥</a>에서 동으로는 <a href="/wiki/%EB%A7%88%EC%82%B0%EC%8B%9C" title="마산시">마산</a>·<a class="mw-redirect" href="/wiki/%EA%B9%80%ED%95%B4" title="김해">김해</a>·<a class="mw-redirect" href="/wiki/%EB%B6%80%EC%82%B0" title="부산">부산</a> 등까지, 북으로는 <a href="/wiki/%EC%A0%84%EC%A3%BC%EC%8B%9C" title="전주시">전주</a>에까지 출병하여 관군을 격파하고 위세를 떨쳤으며, 그해 음력 11월에 이르러 삼별초군은 <a href="/wiki/%EC%A0%9C%EC%A3%BC%EB%8F%84" title="제주도">제주도</a>까지 점령하였는 등<sup class="reference" id="cite_ref-.EC.A1.B0.EC.84.A0.EB.8B.B7.EC.BB.B4_1-3"><a href="#cite_note-.EC.A1.B0.EC.84.A0.EB.8B.B7.EC.BB.B4-1">[1]</a></sup> <a href="/wiki/1271%EB%85%84" title="1271년">1271년</a> 초까지 여러 차례 여몽 연합군과의 싸움에서 승리하며 개경 정부를 위협하였다.</p>
<p>그러나 <a href="/wiki/1271%EB%85%84" title="1271년">1271년</a>(김진관 12년) 음력 5월 상장군 <a href="/wiki/%EA%B9%80%EB%B0%A9%EA%B2%BD" title="김방경">김방경</a>과 <a class="mw-redirect" href="/wiki/%ED%9D%94%EB%8F%84" title="흔도">흔도</a>(炘都)가 지휘하는 여몽 연합군이 세 방향에서 <a class="mw-redirect" href="/wiki/%EC%A7%84%EB%8F%84_(%EC%A7%84%EB%8F%84%EA%B5%B0)" title="진도 (진도군)">진도</a>를 향해 공격한다. 삼별초는 진도의 관문인 벽파진에서 중군을 막는 데 주력했지만, 삼별초가 중군을 방어하는 동안 좌군과 우군이 배후와 측면에서 기습 공격을 하였다.<sup class="reference" id="cite_ref-.EC.A1.B0.EC.84.A0.EB.8B.B7.EC.BB.B4_1-4"><a href="#cite_note-.EC.A1.B0.EC.84.A0.EB.8B.B7.EC.BB.B4-1">[1]</a></sup> 진도 정권이 수립된 지 9개월 만에 진도는 함락당하였다. 혼란에 빠진 삼별초는 순식간에 무너졌고, 배중손은 <a class="mw-redirect" href="/wiki/%EB%82%A8%EB%8F%84%EC%84%9D%EC%84%B1" title="남도석성">남도석성</a>에서 전사하였다.</p>
<p>1271년 말, <a href="/wiki/%EA%B9%80%ED%86%B5%EC%A0%95" title="김통정">김통정</a>이 이끄는 잔존 세력은 탐라(<a href="/wiki/%EC%A0%9C%EC%A3%BC%EB%8F%84" title="제주도">제주도</a>)로 거점을 옮겨 항쟁을 계속하였다. 탐라에서의 삼별초는 처음 1년 동안 자체적인 조직 정비 및 방어 시설의 구축에 주력하였고, 이후 약 반년간 전라도 연해안에 대한 군사 활동을 전개하여 그 세력이 <a href="/wiki/%EC%B6%A9%EC%B2%AD%EB%8F%84" title="충청도">충청도</a>와 <a href="/wiki/%EA%B2%BD%EA%B8%B0%EB%8F%84" title="경기도">경기도</a> 연해안까지 확대되었으며, 개경까지 위협하였다. 그러나 여몽연합군의 조직적 공략으로 1273년(원종 14년) 음력 4월 제주 삼별초 역시 무너지고 말았다.</p>
<h2><span class="mw-headline" id=".EA.B2.B0.EA.B3.BC">결과</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%82%BC%EB%B3%84%EC%B4%88%EC%9D%98_%EB%82%9C&amp;action=edit&amp;section=3" title="부분 편집: 결과">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<h3><span class="mw-headline" id=".EA.B3.A0.EB.A0.A4">고려</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%82%BC%EB%B3%84%EC%B4%88%EC%9D%98_%EB%82%9C&amp;action=edit&amp;section=4" title="부분 편집: 고려">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>삼별초의 항전은 고려 대몽항전의 최후를 장식한 것으로서 이후 고려는 14세기 중반까지 원의 정치적 관리하에 놓이게 된다. 그러나 당시 고려 민중 중에서 삼별초를 지지하는 사람들이 많이 생겼고, 남부 지방의 조세가 개경으로 운송되지 않기도 하였다.</p>
<h3><span class="mw-headline" id=".EC.9B.90.EB.82.98.EB.9D.BC">원나라</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%82%BC%EB%B3%84%EC%B4%88%EC%9D%98_%EB%82%9C&amp;action=edit&amp;section=5" title="부분 편집: 원나라">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>몽골은 <a href="/wiki/%ED%83%90%EB%9D%BC%EC%B4%9D%EA%B4%80%EB%B6%80" title="탐라총관부">탐라총관부</a>를 설치하여 <a href="/wiki/1294%EB%85%84" title="1294년">1294년</a>까지 탐라를 직접 통치하였다. 한편 고려의 반란 진압에 도움을 줌으로써 <a class="mw-redirect" href="/wiki/%EB%AA%BD%EA%B3%A8%EC%9D%98_%EC%9D%BC%EB%B3%B8_%EC%9B%90%EC%A0%95" title="몽골의 일본 원정">일본 원정</a>에서 고려 정부의 지원을 더 많이 얻어낼 수 있게 되었다.</p>
<h2><span class="mw-headline" id=".ED.8F.89.EA.B0.80">평가</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%82%BC%EB%B3%84%EC%B4%88%EC%9D%98_%EB%82%9C&amp;action=edit&amp;section=6" title="부분 편집: 평가">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>오늘날 삼별초의 난은 외세 침략에 대해 완강한 저항을 한 국가적 영웅으로 추앙받고 있기도 하지만, 반대로 무신 사병 집단의 저항 정도로만 보는 시각 또한 존재한다. <a href="/wiki/1978%EB%85%84" title="1978년">1978년</a> 삼별초의 난을 기리기 위한 기념물이 제주도에 세워졌다. 진도에 있는 <a class="mw-redirect" href="/wiki/%EC%9A%A9%EC%9E%A5%EC%82%B0%EC%84%B1" title="용장산성">용장산성</a>과 같은 유적지는 관광지로 유지되고 있다.</p>

<h2><span class="mw-headline" id=".EC.B0.B8.EA.B3.A0_.EC.9E.90.EB.A3.8C">참고 자료</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%82%BC%EB%B3%84%EC%B4%88%EC%9D%98_%EB%82%9C&amp;action=edit&amp;section=9" title="부분 편집: 참고 자료">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
