<p><b>삼산동</b>(三山洞)은 <a href="/wiki/%EC%9A%B8%EC%82%B0%EA%B4%91%EC%97%AD%EC%8B%9C" title="울산광역시">울산광역시</a> <a href="/wiki/%EB%82%A8%EA%B5%AC_(%EC%9A%B8%EC%82%B0%EA%B4%91%EC%97%AD%EC%8B%9C)" title="남구 (울산광역시)">남구</a>의 <a href="/wiki/%EB%B2%95%EC%A0%95%EB%8F%99" title="법정동">법정</a><a href="/wiki/%EB%8F%99_(%ED%96%89%EC%A0%95_%EA%B5%AC%EC%97%AD)" title="동 (행정 구역)">동</a>, 행정동이다. 삼산동은 <a href="/wiki/%ED%83%9C%ED%99%94%EA%B0%95%EC%97%AD" title="태화강역">태화강역</a>, <a href="/wiki/%EC%9A%B8%EC%82%B0%EA%B4%91%EC%97%AD%EC%8B%9C%EC%9D%98_%EC%8B%9C%EC%99%B8%EB%B2%84%EC%8A%A4" title="울산광역시의 시외버스">시외ㆍ고속버스터미널</a>과 시내버스 공용터미널이 있는 교통의 요충지이고, KBS방송국, UBC 울산방송국이 소재하며 울산광역시문화예술회관, 남구문화원 등 문화시설과 금융기관이 밀집해 있으며 <a href="/wiki/%ED%98%84%EB%8C%80%EB%B0%B1%ED%99%94%EC%A0%90" title="현대백화점">현대백화점</a>, <a href="/wiki/%EB%A1%AF%EB%8D%B0%EB%B0%B1%ED%99%94%EC%A0%90" title="롯데백화점">롯데백화점</a>, 농수산물도매시장, 기계공구 상가가 있는 유통산업의 중심지로 부상하는 신개발지역으로 다양하고 복잡한 행정수요가 나날이 증가하고 있는 지역이다.<sup class="reference" id="cite_ref-1"><a href="#cite_note-1">[1]</a></sup></p>

<h2><span class="mw-headline" id=".EC.A7.80.EB.AA.85">지명</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%82%BC%EC%82%B0%EB%8F%99_(%EC%9A%B8%EC%82%B0)&amp;action=edit&amp;section=1" title="부분 편집: 지명">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p><a href="/wiki/%EC%A1%B0%EC%84%A0_%EC%88%99%EC%A2%85" title="조선 숙종">조선 숙종</a>, <a href="/wiki/%EC%A1%B0%EC%84%A0_%EC%98%81%EC%A1%B0" title="조선 영조">영조</a> 때는 신리란 단일 마을이었다. <a href="/wiki/%EC%A1%B0%EC%84%A0_%EC%A0%95%EC%A1%B0" title="조선 정조">정조</a> 때는 신리와 삼산, <a href="/wiki/%EC%A1%B0%EC%84%A0_%EC%88%9C%EC%A1%B0" title="조선 순조">순조</a> 때는 신리, 신복, 삼산, 신기, 중리, <a class="mw-redirect" href="/wiki/%EC%A1%B0%EC%84%A0_%EA%B3%A0%EC%A2%85" title="조선 고종">고종</a> 때는 신리, 신복, 삼산, 중리로 나뉘었고, <a href="/wiki/1911%EB%85%84" title="1911년">1911년</a>에는 신리, 삼산, 중리로 갈라져 있다가, <a class="mw-redirect" href="/wiki/%EC%9D%BC%EC%A0%9C%EC%8B%9C%EB%8C%80" title="일제시대">일제시대</a>인 <a href="/wiki/1914%EB%85%84" title="1914년">1914년</a> 행정구역 개편 때부터 이를 합하면서 산봉우리가 3개가 솟아 있었다 하여 ‘삼산’이라 하였다.<sup class="reference" id="cite_ref-2"><a href="#cite_note-2">[2]</a></sup></p>
