<h2><span class="mw-headline" id=".EC.99.B8.ED.98.95_.EB.B0.8F_.EB.B0.A9.EC.88.98.2F.EB.B0.A9.EC.A7.84">외형 및 방수/방진</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%82%BC%EC%84%B1_%EA%B0%A4%EB%9F%AD%EC%8B%9C_S5&amp;action=edit&amp;section=2" title="부분 편집: 외형 및 방수/방진">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>스피커와 마이크, 이어폰단자를 제외한 부분은 별도의 마개로 처리하였으며, 뒷면은 구멍이 뚫려있는 펀칭 패턴이 적용 되었다.</p>
<p>방수/방진 수준에 대해서는 <a class="extiw" href="https://en.wikipedia.org/wiki/IP_Code" title="en:IP Code">IP67</a> 등급을 인증받았다 (먼지로부터 완벽하게 보호되며, 1m 이내의 수심에서 30분을 견딜 수 있다.).</p>
<h2><span class="mw-headline" id="CPU">CPU</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%82%BC%EC%84%B1_%EA%B0%A4%EB%9F%AD%EC%8B%9C_S5&amp;action=edit&amp;section=3" title="부분 편집: CPU">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>일반 LTE 모델 (SM-G900F를 포함한 국내 모델 및 미국 통신사 모델) 은 <a href="/wiki/%ED%80%84%EC%BB%B4" title="퀄컴">퀄컴</a>의 <a href="/wiki/%EC%8A%A4%EB%83%85%EB%93%9C%EB%9E%98%EA%B3%A4" title="스냅드래곤">스냅드래곤 801</a> 프로세서를 탑재했으며, 광대역 LTE-A 모델 (SM-G906S/K/L)의 경우 <a href="/wiki/%ED%80%84%EC%BB%B4" title="퀄컴">퀄컴</a> <a href="/wiki/%EC%8A%A4%EB%83%85%EB%93%9C%EB%9E%98%EA%B3%A4" title="스냅드래곤">스냅드래곤 805</a> 프로세서를 탑재하였다.</p>
<p>국내에 출시되지 않은 3G와 GSM용 G900H 모델 혹은 그를 기반으로 하는 변종모델의 경우 <a href="/wiki/Big.LITTLE" title="Big.LITTLE">big.LITTLE</a> 기술이 적용된 <a href="/wiki/%EC%82%BC%EC%84%B1%EC%A0%84%EC%9E%90" title="삼성전자">삼성전자</a> <a class="mw-redirect" href="/wiki/%EC%97%91%EC%8B%9C%EB%85%B8%EC%8A%A4" title="엑시노스">엑시노스</a> 5 옥타 (5422)가 사용되었다.</p>
<p>전작과는 달리 클러스터간 이동(Cluster Migration) 외에도 인커널 스위처 (In Kernel Switcher)와 다른 기종간 다중 처리(Heterogeneous Multi Processing)를 지원하여 상황에 따라 CPU를 1~8개까지 작동시킬수 있게 되어, <a href="/wiki/Big.LITTLE" title="Big.LITTLE">big.LITTLE</a>을 완벽하게 지원한다.</p>
<h2><span class="mw-headline" id=".ED.99.94.EB.A9.B4">화면</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%82%BC%EC%84%B1_%EA%B0%A4%EB%9F%AD%EC%8B%9C_S5&amp;action=edit&amp;section=4" title="부분 편집: 화면">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<div class="rellink relarticle mainarticle"><img alt="" data-file-height="128" data-file-width="128" height="16" src="//upload.wikimedia.org/wikipedia/commons/thumb/e/ec/Crystal_Clear_app_xmag.svg/16px-Crystal_Clear_app_xmag.svg.png" srcset="//upload.wikimedia.org/wikipedia/commons/thumb/e/ec/Crystal_Clear_app_xmag.svg/24px-Crystal_Clear_app_xmag.svg.png 1.5x, //upload.wikimedia.org/wikipedia/commons/thumb/e/ec/Crystal_Clear_app_xmag.svg/32px-Crystal_Clear_app_xmag.svg.png 2x" width="16"/> 이 부분의 본문은 <a href="/wiki/%EC%8A%88%ED%8D%BC_AMOLED#.ED.92.80_HD_.EC.8A.88.ED.8D.BC_AMOLED" title="슈퍼 AMOLED">슈퍼 AMOLED#풀 HD 슈퍼 AMOLED</a>입니다.</div>
<p>1920x1080의 <a href="/wiki/1080p" title="1080p">1080p</a> 해상도의 <a class="mw-redirect" href="/wiki/%ED%92%80_HD_%EC%8A%88%ED%8D%BC_AMOLED" title="풀 HD 슈퍼 AMOLED">풀 HD 슈퍼 AMOLED</a> 디스플레이를 적용하였다. 전작인 <a class="mw-redirect" href="/wiki/%EA%B0%A4%EB%9F%AD%EC%8B%9C_S4" title="갤럭시 S4">갤럭시 S4</a> 대비 발광소자의 사용수명과 색감, 선명도 등에서 개선이 이루어졌다.</p>
<p>SM-G901F, SM-G906S/K/L의 갤럭시 S5 광대역 LTE-A 기종의 경우 2560x1440의 <a href="/wiki/WQHD" title="WQHD">QHD</a> 해상도의 디스플레이를 적용하였다.</p>
<h2><span class="mw-headline" id=".EC.B9.B4.EB.A9.94.EB.9D.BC">카메라</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%82%BC%EC%84%B1_%EA%B0%A4%EB%9F%AD%EC%8B%9C_S5&amp;action=edit&amp;section=5" title="부분 편집: 카메라">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>이미지 센서는 기존의 BSI <a href="/wiki/CMOS_%EC%9D%B4%EB%AF%B8%EC%A7%80_%EC%84%BC%EC%84%9C" title="CMOS 이미지 센서">CMOS 이미지 센서</a>에서 아이소셀 CMOS 이미지 센서로 변경하였으며, 크기는 1/2.6"로 전작보다 커져서 전작들에 비해 빛을 더 많이 받아들일 수 있으며, 색감을 더 뚜렷하게 구분한다.</p>
<p>또한 자동 초점의 속도는 0.3초로 전작들보다 향상되었다. 일부 모델의 경우, 소니 IMX240 BSI CMOS 이미지탑재되었다.</p>
<h2><span class="mw-headline" id=".EC.A7.80.EB.AC.B8_.EC.9D.B8.EC.8B.9D">지문 인식</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%82%BC%EC%84%B1_%EA%B0%A4%EB%9F%AD%EC%8B%9C_S5&amp;action=edit&amp;section=6" title="부분 편집: 지문 인식">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>홈버튼에 스와이프 방식의 지문 인식 센서가 내장되어 있으며, 이를 잠금화면 해제, 갤러리등의 사적인 애플리케이션의 보호, 결제등에 활용 할 수 있다.</p>
<h2><span class="mw-headline" id=".EC.8B.AC.EC.9E.A5_.EB.B0.95.EB.8F.99_.EC.9D.B8.EC.8B.9D">심장 박동 인식</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%82%BC%EC%84%B1_%EA%B0%A4%EB%9F%AD%EC%8B%9C_S5&amp;action=edit&amp;section=7" title="부분 편집: 심장 박동 인식">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>뒷면의 플래시 옆에는 심박 센서가 장착되어있어, 심장 박동수를 측정 할 수 있다.</p>

<li>셀렉티브 포커스 : 초점이 다른 2가지 사진을 촬영해 합성하는 방식으로, DSLR 카메라에서 볼 수 있는 촬영기법인 아웃포커싱을 소프트웨어적으로 지원하며, 피사체와 배경 중 어디에 초점을 맞출지 촬영 후에 선택해 원하는 사진을 꾸밀 수 있다.</li>
</ul>
<ul>
<li>그룹 플레이 : 그룹에 참여한 사용자들이 각자 촬영한 동영상을 그룹 생성자가 편집할 수 있는 ‘그룹 캠코더’ 기능이 추가되었다.</li>
</ul>
<ul>
<li>울트라 파워 세이빙 모드: 휴대전화 사용에 필요한 최소한의 서비스를 제외한 모든 서비스를 종료시키며, 흑백화면 및 최소밝기 조정을 통해 배터리 사용량을 획기적으로 줄여준다. (공식발표 자료 기준 10%에서 24시간 동안 대기가능)</li>
</ul>
<ul>
<li>심박수 측정 : 기기 뒷면에 탑재된 심박수 측정 센서로 심박수를 확인하거나 기록할 수 있다.</li>
</ul>
<ul>
<li>긴급 모드 : 위급한 상황에 사용할 수 있도록 손전등, 위급 상황 알람 등의 비상 상황 발생시 필요한 기능을 홈 화면에 배치하고 배터리 수명을 최적화함으로써 긴급 상황에 대비할 수 있도록 만들어준다.</li>
</ul>
<ul>
<li>도움 요청 메시지 : 긴급 상황에서 전원버튼을 3번 누르면 미리 설정한 연락처에 긴급 메시지가 전송되면 사용자의 위치와 함께 현장의 소리가 5초간 녹음되어 전송된다.</li>
</ul>
<ul>
<li>Geo 뉴스 : 사용자의 위치 정보를 기반으로 황사, 건조, 대설과 같은 기상특보 및 지진, 낙뢰 등의 재해정보를 제공한다.</li>
</ul>
<ul>
<li>운동 코칭 : <a href="/wiki/%EC%82%BC%EC%84%B1_%EA%B8%B0%EC%96%B4_2" title="삼성 기어 2">삼성 기어 2</a>, <a class="mw-redirect" href="/wiki/%EC%82%BC%EC%84%B1_%EA%B8%B0%EC%96%B4_2_%EB%84%A4%EC%98%A4" title="삼성 기어 2 네오">삼성 기어 2 네오</a>, <a href="/wiki/%EC%82%BC%EC%84%B1_%EA%B8%B0%EC%96%B4_%ED%95%8F" title="삼성 기어 핏">삼성 기어 핏</a>을 연동하여 운동목표 및 운동계획에 맞춰 운동을 도와주는 개인 트레이너 기능이 탑재되었다.</li>
</ul>
<h3><span class="mw-headline" id=".EC.95.88.EB.93.9C.EB.A1.9C.EC.9D.B4.EB.93.9C_5.0_.EB.A1.A4.EB.A6.AC.ED.8C.9D">안드로이드 5.0 롤리팝</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%82%BC%EC%84%B1_%EA%B0%A4%EB%9F%AD%EC%8B%9C_S5&amp;action=edit&amp;section=12" title="부분 편집: 안드로이드 5.0 롤리팝">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>2014년 12월 5일 갤럭시S5(SM-G900S/K/L) 모델에 대한 안드로이드 5.0 롤리팝 업데이트가 시행되었으며, 후속기종인 갤럭시S5 광대역 LTE-A(SM-G906S/K/L) 기종은 2015년 3월 9일 5.0.1로 업데이트 되었다.</p>
<h3><span class="mw-headline" id=".EC.95.88.EB.93.9C.EB.A1.9C.EC.9D.B4.EB.93.9C_6.0.1_.EB.A7.88.EC.8B.9C.EB.A9.9C.EB.A1.9C">안드로이드 6.0.1 마시멜로</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%82%BC%EC%84%B1_%EA%B0%A4%EB%9F%AD%EC%8B%9C_S5&amp;action=edit&amp;section=13" title="부분 편집: 안드로이드 6.0.1 마시멜로">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>2016년 3월 21일 갤럭시S5(SM-G900S/K/L)에 대한 안드로이드 6.0.1 마시멜로가 업데이트 되었으며 2016년 5월 17일 갤럭시S5 광대역 LTE-A(SM-G906S/K/L에 대한 안드로이드 6.0.1 마시멜로가 업데이트 되었다.</p>
<h2><span class="mw-headline" id=".ED.8A.B9.EC.88.98_.EC.95.A1.EC.84.B8.EC.84.9C.EB.A6.AC">특수 액세서리</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%82%BC%EC%84%B1_%EA%B0%A4%EB%9F%AD%EC%8B%9C_S5&amp;action=edit&amp;section=14" title="부분 편집: 특수 액세서리">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<h3><span class="mw-headline" id="S_.EB.B7.B0_.EC.BB.A4.EB.B2.84">S 뷰 커버</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%82%BC%EC%84%B1_%EA%B0%A4%EB%9F%AD%EC%8B%9C_S5&amp;action=edit&amp;section=15" title="부분 편집: S 뷰 커버">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>전작인 <a class="mw-redirect" href="/wiki/%EA%B0%A4%EB%9F%AD%EC%8B%9C_S4" title="갤럭시 S4">갤럭시 S4</a>의 S 뷰 커버에 비해 커버를 닫았을때 표시되는 화면 영역을 넓혔으며, 전체적인 외형은 <a class="mw-redirect" href="/wiki/%EA%B0%A4%EB%9F%AD%EC%8B%9C_%EB%85%B8%ED%8A%B8_3" title="갤럭시 노트 3">갤럭시 노트 3</a>의 S 뷰 커버와 비슷하지만, 생활 방수 기능을 제공하며, 모서리 코팅을 통해 측면을 보호하도록 만들어졌다.</p>
<p>커버를 닫은 상태로 커버 전면에 위치한 스크린을 통해 수신 거절 메시지를 전송하거나, 알람, 이벤트, MMS/SMS를 미리볼 수 있으며, 카메라를 빨리 실행할 수 있는 퀵 카메라 기능을 제공한다.</p>
<p>색상은 블랙, 화이트, 로즈골드. 핑크, 그린의 5가지다.</p>
<h3><span class="mw-headline" id=".ED.94.8C.EB.A6.BD_.EC.9B.94.EB.A0.9B">플립 월렛</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%82%BC%EC%84%B1_%EA%B0%A4%EB%9F%AD%EC%8B%9C_S5&amp;action=edit&amp;section=16" title="부분 편집: 플립 월렛">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>S 뷰 커버에서 전면의 스크린을 제거하여 만들어진 제품으로, S 뷰 기능 작동을 위한 스크린 대신 대신 카드를 수납할 수 있는 공간 만들어 출시한 제품이다.</p>
<p>S 뷰 커버와 동일하게 방수를 지원하여 모서리 코팅이 되어있으며. 제공 색상 역시 S 뷰 커버와 동일하다.</p>
<ul>
<li>패션형 플립 월렛 : 모스키노와 디자이너 니콜라스 커크우드가 공동 제작한 플립 월렛으로 기본 플립 월렛과 달리 무늬가 있으며 색상은 2가지로 제공된다.</li>
</ul>
