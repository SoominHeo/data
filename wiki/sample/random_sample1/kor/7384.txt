삼투압에 의한 현상의 예. 용액의 종류에 따라 적혈구의 모양이 달라진다.</div>

<p><b>삼투</b>(滲透, <span style="font-size: smaller;"><a href="/wiki/%EB%AC%B8%ED%99%94%EC%96%B4" title="문화어">문화어</a>:</span> 스밈압력, <a href="/wiki/%ED%94%84%EB%9E%91%EC%8A%A4%EC%96%B4" title="프랑스어">프랑스어</a>, <a href="/wiki/%EB%8F%85%EC%9D%BC%EC%96%B4" title="독일어">독일어</a>: Osmose, <a href="/wiki/%EC%98%81%EC%96%B4" title="영어">영어</a>, <a href="/wiki/%EB%9D%BC%ED%8B%B4%EC%96%B4" title="라틴어">라틴어</a>, <a href="/wiki/%EC%8A%A4%ED%8E%98%EC%9D%B8%EC%96%B4" title="스페인어">스페인어</a>: Osmosis) 또는 <b>삼투현상</b>이란, 묽은 <a href="/wiki/%EC%9A%A9%EC%95%A1" title="용액">용액</a>과 진한 용액이 <a href="/wiki/%EB%B0%98%ED%88%AC%EA%B3%BC%EC%84%B1%EB%A7%89" title="반투과성막">반투과성막</a>을 사이에 두고 있을 때, 농도가 더 진한 쪽으로 <a href="/wiki/%EC%9A%A9%EB%A7%A4" title="용매">용매</a>(일반적 <a href="/wiki/%EB%AC%BC" title="물">물</a>)가 이동하는 현상이다. 이때 발생하는 <a href="/wiki/%EC%95%95%EB%A0%A5" title="압력">압력</a>의 크기를 삼투압이라고 한다. Passive transport.</p>
<h2><span class="mw-headline" id=".EC.82.BC.ED.88.AC.EC.95.95.EC.97.90_.EC.9D.98.ED.95.9C_.ED.98.84.EC.83.81.EC.9D.98_.EC.98.88">삼투압에 의한 현상의 예</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%82%BC%ED%88%AC&amp;action=edit&amp;section=1" title="부분 편집: 삼투압에 의한 현상의 예">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<ul>
<li><a href="/wiki/%EB%B0%B0%EC%B6%94" title="배추">배추</a>를 <a href="/wiki/%EC%86%8C%EA%B8%88" title="소금">소금</a>에 절여두면 배추의 수분이 밖으로 빠져나간다.</li>
<li><a href="/wiki/%EB%91%90%EB%B6%80" title="두부">두부</a>를 틀에 넣고 무거운 돌 등으로 눌러두면 두부의 수분이 빠져나가면서 두부가 단단해진다.</li>
<li><a href="/wiki/%EC%98%A4%EC%9D%B4" title="오이">오이</a>를 <a href="/wiki/%EC%8B%9D%EC%B4%88" title="식초">식초</a>에 넣으면 오이에서 수분이 빠져나가면서 <a href="/wiki/%ED%94%BC%ED%81%B4" title="피클">피클</a>이 된다.</li>
<li><a href="/wiki/%EC%A0%81%ED%98%88%EA%B5%AC" title="적혈구">적혈구</a>를 고장액(Hypertonic)에 넣으면 적혈구에서 물이 빠져나가 쪼그라든다.</li>
<li><a href="/wiki/%EC%A0%81%ED%98%88%EA%B5%AC" title="적혈구">적혈구</a>를 저장액(Hypotonic)에 넣으면 바깥에서 적혈구로 물이 들어오면서 세포가 파괴된다. (용혈, hemolysis)</li>
</ul>
