<p><b>삼히타</b>(<span style="font-size: smaller;"><a href="/wiki/%EC%82%B0%EC%8A%A4%ED%81%AC%EB%A6%AC%ED%8A%B8%EC%96%B4" title="산스크리트어">산스크리트어</a>:</span> <span lang="sa" xml:lang="sa">saṃhita</span>, <span style="font-size: smaller;"><a href="/wiki/%EC%98%81%EC%96%B4" title="영어">영어</a>:</span> <span lang="en" xml:lang="en">Samhita</span>) 또는 <b>상히타</b>의 문자 그대로의 뜻은 "결합된", "수집된", 또는 "집성된"이다. 삼히타는 다음 중 하나를 의미한다. 특히, 그냥 삼히타라고 할 경우 흔히 이들 중 <a href="/wiki/%ED%88%AC%EB%A6%AC%EC%95%BC" title="투리야">투리야</a>를 의미한다.</p>
<ul>
<li><a href="/wiki/%EB%B2%A0%EB%8B%A4_%EC%8B%9C%EB%8C%80" title="베다 시대">베다 시대</a>에 성립된 <a class="mw-redirect" href="/wiki/%EB%B2%A0%EB%8B%A4_%EB%AC%B8%ED%97%8C" title="베다 문헌">베다 문헌</a> 전체의 근간이 되는 <a class="mw-redirect" href="/wiki/%EB%B2%A0%EB%8B%A4%EC%84%9C" title="베다서">베다서</a> 각각을 삼히타라고 한다.
<ul>
<li><b><a href="/wiki/%ED%88%AC%EB%A6%AC%EC%95%BC" title="투리야">투리야</a></b>(<span lang="sa" xml:lang="sa">तुरीय</span> <small>Turīya</small>): 4종의 삼히타라는 뜻으로, 힌두교의 정전(正典)을 이루는 네 가지 <a class="mw-redirect" href="/wiki/%EB%B2%A0%EB%8B%A4%EC%84%9C" title="베다서">베다서</a>인 《<a href="/wiki/%EB%A6%AC%EA%B7%B8%EB%B2%A0%EB%8B%A4" title="리그베다">리그베다</a>》<b>·</b>《<a class="new" href="/w/index.php?title=%EC%95%BC%EC%A3%BC%EB%A5%B4%EB%B2%A0%EB%8B%A4&amp;action=edit&amp;redlink=1" title="야주르베다 (없는 문서)">야주르베다</a>》<b>·</b>《<a class="new" href="/w/index.php?title=%EC%82%AC%EB%A7%88%EB%B2%A0%EB%8B%A4&amp;action=edit&amp;redlink=1" title="사마베다 (없는 문서)">사마베다</a>》<b>·</b>《<a class="new" href="/w/index.php?title=%EC%95%84%ED%83%80%EB%A5%B4%EB%B0%94%EB%B2%A0%EB%8B%A4&amp;action=edit&amp;redlink=1" title="아타르바베다 (없는 문서)">아타르바베다</a>》를 가리킨다. 4종의 삼히타는 전체 <a class="mw-redirect" href="/wiki/%EB%B2%A0%EB%8B%A4_%EB%AC%B8%ED%97%8C" title="베다 문헌">베다 문헌</a>의 근간을 이루고 있다.</li>
<li><b>트라야</b>(Traya): 3종의 삼히타라는 뜻으로, <a href="/wiki/%ED%88%AC%EB%A6%AC%EC%95%BC" title="투리야">투리야</a> 중 《<a class="new" href="/w/index.php?title=%EC%95%84%ED%83%80%EB%A5%B4%EB%B0%94%EB%B2%A0%EB%8B%A4&amp;action=edit&amp;redlink=1" title="아타르바베다 (없는 문서)">아타르바베다</a>》를 제외한 3종의 <a class="mw-redirect" href="/wiki/%EB%B2%A0%EB%8B%A4%EC%84%9C" title="베다서">베다서</a>를 가리킨다.</li>
</ul>
