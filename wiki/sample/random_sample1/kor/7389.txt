<p><b>삿사 나리마사</b>(<span style="font-size: smaller;"><a href="/wiki/%EC%9D%BC%EB%B3%B8%EC%96%B4" title="일본어">일본어</a>:</span> <span lang="ja" xml:lang="ja">佐々成政</span>, <a href="/wiki/1536%EB%85%84" title="1536년">1536년</a> <a href="/wiki/2%EC%9B%94_6%EC%9D%BC" title="2월 6일">2월 6일</a> ~ <a href="/wiki/1588%EB%85%84" title="1588년">1588년</a> <a href="/wiki/7%EC%9B%94_7%EC%9D%BC" title="7월 7일">7월 7일</a> <a href="/wiki/%EC%9D%8C%EB%A0%A5_5%EC%9B%94_14%EC%9D%BC" title="음력 5월 14일">음력 5월 14일</a>)는 <a href="/wiki/%EC%84%BC%EA%B3%A0%EC%BF%A0_%EC%8B%9C%EB%8C%80" title="센고쿠 시대">센고쿠 시대</a> 부터 <a class="mw-redirect" href="/wiki/%EC%95%84%EC%A6%88%EC%B9%98%EB%AA%A8%EB%AA%A8%EC%95%BC%EB%A7%88_%EC%8B%9C%EB%8C%80" title="아즈치모모야마 시대">아즈치모모야마 시대</a>까지의 <a href="/wiki/%EB%AC%B4%EC%9E%A5" title="무장">무장</a>이며, <a href="/wiki/%ED%9E%88%EA%B3%A0_%EA%B5%AD" title="히고 국">히고 국</a>의 <a href="/wiki/%EB%8B%A4%EC%9D%B4%EB%AC%98" title="다이묘">다이묘</a>이다.</p>
<h2><span class="mw-headline" id=".EC.83.9D.EC.95.A0">생애</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%82%BF%EC%82%AC_%EB%82%98%EB%A6%AC%EB%A7%88%EC%82%AC&amp;action=edit&amp;section=1" title="부분 편집: 생애">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>삿사 나리마사는 <a class="new" href="/w/index.php?title=%EC%82%BF%EC%82%AC_%EB%82%98%EB%A6%AC%EB%AC%B4%EB%84%A4&amp;action=edit&amp;redlink=1" title="삿사 나리무네 (없는 문서)">삿사 나리무네</a>(佐々成宗)의 3남으로 태어났다. 삿사 나리마사는 형이 2명이 있지만 형들이 연달아 싸움에서 <a href="/wiki/%EC%9E%91%EC%A0%84_%EC%A4%91_%EC%82%AC%EB%A7%9D" title="작전 중 사망">전사</a>하였기 때문에, <a href="/wiki/%EC%97%90%EC%9D%B4%EB%A1%9C%EC%BF%A0" title="에이로쿠">에이로쿠</a>(永禄) 3년 <a href="/wiki/1560%EB%85%84" title="1560년">1560년</a>에 <a href="/wiki/%EA%B0%80%EB%8F%85" title="가독">가독</a>의 후계자로 히라성 성주가 되었다.</p>
<p><a href="/wiki/%EC%98%A4%EB%8B%A4_%EB%85%B8%EB%B6%80%EB%82%98%EA%B0%80" title="오다 노부나가">오다 노부나가</a>의 시종으로 들어간 삿사 나리마사는 <a class="new" href="/w/index.php?title=%EC%9A%B0%EB%A7%88%EB%A7%88%EC%99%80%EB%A6%AC&amp;action=edit&amp;redlink=1" title="우마마와리 (없는 문서)">우마마와리</a>(馬廻)로 활약하여 전공을 세우고, <a class="mw-redirect" href="/wiki/%EC%98%A4%EB%8B%A4_%EA%B0%80%EB%AC%B8" title="오다 가문">오다 가문</a>의 무장으로 여러 전투에서 활약하여 오다군의 흑모의 무리의 장수가 되었다.</p>
<p><a class="mw-disambig" href="/wiki/%EB%8D%B4%EC%87%BC" title="덴쇼">덴쇼</a>(天正) 3년 <a href="/wiki/1575%EB%85%84" title="1575년">1575년</a> <a href="/wiki/5%EC%9B%94" title="5월">5월</a>의 <a href="/wiki/%EB%82%98%EA%B0%80%EC%8B%9C%EB%85%B8_%EC%A0%84%ED%88%AC" title="나가시노 전투">나가시노 전투</a>에서 <a href="/wiki/%EB%A7%88%EC%97%90%EB%8B%A4_%EB%8F%84%EC%8B%9C%EC%9D%B4%EC%97%90" title="마에다 도시이에">마에다 도시이에</a>(前田利家), <a href="/wiki/%EB%85%B8%EB%85%B8%EB%AC%B4%EB%9D%BC_%EB%A7%88%EC%82%AC%EC%8B%9C%EA%B2%8C" title="노노무라 마사시게">노노무라 마사시게</a>(野々村正成), <a class="new" href="/w/index.php?title=%ED%9B%84%EC%BF%A0%EB%8F%84%EB%AF%B8_%ED%9E%88%EB%8D%B0%EC%B9%B4%EC%93%B0&amp;action=edit&amp;redlink=1" title="후쿠도미 히데카쓰 (없는 문서)">후쿠도미 히데카쓰</a>(福富秀勝), <a class="new" href="/w/index.php?title=%EB%B0%98_%EB%82%98%EC%98%A4%EB%A7%88%EC%82%AC&amp;action=edit&amp;redlink=1" title="반 나오마사 (없는 문서)">반 나오마사</a>(塙直政)와 함께 철포대를 이끌었다. <a href="/wiki/%EC%98%A4%EB%B4%89%ED%96%89" title="오봉행">철포봉행</a>(砲奉行)의 한 사람이기도 하다.</p>
<p>오다 노부나가의 에치젠 정벌 이후 시바타 가쓰이에가 <a href="/wiki/%EC%97%90%EC%B9%98%EC%A0%A0_%EA%B5%AD" title="에치젠 국">에치젠</a> 영지를 받자 그곳의 한 군을 받아, 시바타 휘하에 소속되어 <a href="/wiki/%EA%B0%80%EA%B0%80_%EA%B5%AD" title="가가 국">가가</a>의 <a href="/wiki/%EC%9D%BC%ED%96%A5%EC%A2%85" title="일향종">일향종</a>과 싸움을 거듭하였다.</p>
<p>이 때 삿사 나리마사는 <a href="/wiki/%EB%A7%88%EC%97%90%EB%8B%A4_%EB%8F%84%EC%8B%9C%EC%9D%B4%EC%97%90" title="마에다 도시이에">마에다 도시이에</a>, <a class="new" href="/w/index.php?title=%ED%9B%84%EC%99%80_%EB%AF%B8%EC%93%B0%ED%95%98%EB%A3%A8&amp;action=edit&amp;redlink=1" title="후와 미쓰하루 (없는 문서)">후와 미쓰하루</a>(不破光治)와 함께 에치젠 후추 10만 석을 같이 받았지만, 함께 받는 사람이 3명이였기에 3만 3000석으로 나누어 다스렸다.</p>
<p><a href="/wiki/%EC%9A%B0%EC%97%90%EC%8A%A4%EA%B8%B0_%EA%B2%90%EC%8B%A0" title="우에스기 겐신">우에스기 겐신</a>의 사망 이후 오다군이 엣츄를 점령하면서, 엣추 국을 다스리는 권한을 얻게 되었다.</p>
<p>덴쇼 10년(1582년) <a href="/wiki/%ED%98%BC%EB%85%B8%EC%A7%80%EC%9D%98_%EB%B3%80" title="혼노지의 변">혼노지의 변</a>이 일어났을 때, 호쿠리쿠 방면군은 우에스기 군의 마지막 거점인 엣추 우오즈 성을 떨어뜨렸으나, 혼노지의 변이 각 장수한테 전해지자 장수들은 각자의 영지로 돌아갔기 때문에 우에스기군의 저지를 받게 되었고, 결국 <a href="/wiki/%EC%8B%9C%EB%B0%94%ED%83%80_%EA%B0%80%EC%93%B0%EC%9D%B4%EC%97%90" title="시바타 가쓰이에">시바타 가쓰이에</a>의 몰락 원인이 된다.</p>
<p>삿사 나리마사는 <a href="/wiki/%EC%9A%B0%EC%97%90%EC%8A%A4%EA%B8%B0_%EA%B0%80%EA%B2%8C%EC%B9%B4%EC%93%B0" title="우에스기 가게카쓰">우에스기 가게카쓰</a>에 대한 대비를 하기 위해, <a href="/wiki/%EC%8B%9C%EC%A6%88%EA%B0%80%ED%83%80%EC%BC%80_%EC%A0%84%ED%88%AC" title="시즈가타케 전투">시즈가타케 전투</a>에는 참여하지 않고 숙부인 <a class="new" href="/w/index.php?title=%EC%82%BF%EC%82%AC_%ED%97%A4%EC%9D%B4%EC%9E%90%EC%97%90%EB%AA%AC&amp;action=edit&amp;redlink=1" title="삿사 헤이자에몬 (없는 문서)">삿사 헤이자에몬</a>(佐々平左衛門)를 지원군으로 보냈었다. 시바타 가쓰이에가 패배한 이후, 마에다 도시이에의 배반과 우에스기 가게카쓰의 압박으로 딸을 인질로 보내고 머리를 삭발하는 것으로 히데요시에게 항복하였다.</p>
<p>덴쇼 12년 (<a href="/wiki/1584%EB%85%84" title="1584년">1584년</a>), <a href="/wiki/%EA%B3%A0%EB%A7%88%ED%82%A4%C2%B7%EB%82%98%EA%B0%80%EC%BF%A0%ED%85%8C_%EC%A0%84%ED%88%AC" title="고마키·나가쿠테 전투">고마키·나가쿠테 전투</a>가 일어나자 삿사 나리마사는 <a href="/wiki/%EB%8F%84%EC%9A%94%ED%86%A0%EB%AF%B8_%ED%9E%88%EB%8D%B0%EC%9A%94%EC%8B%9C" title="도요토미 히데요시">도요토미 히데요시</a> 측으로 가담하지 않고, 여름 무렵이 되자 <a href="/wiki/%EB%8F%84%EC%BF%A0%EA%B0%80%EC%99%80_%EC%9D%B4%EC%97%90%EC%95%BC%EC%8A%A4" title="도쿠가와 이에야스">도쿠가와 이에야스</a>, <a class="mw-redirect" href="/wiki/%EC%98%A4%EB%8B%A4_%EB%85%B8%EB%B6%80%EC%B9%B4%EC%93%B0_(1558%EB%85%84)" title="오다 노부카쓰 (1558년)">오다 노부카쓰</a> 측에 가담하였다. 그렇게 되어, 히데요시 측에 붙었던 마에다 도시이에와의 <a class="new" href="/w/index.php?title=%EC%8A%A4%EC%97%90%EB%AA%A8%EB%A6%AC%EC%84%B1_%EC%A0%84%ED%88%AC&amp;action=edit&amp;redlink=1" title="스에모리성 전투 (없는 문서)">스에모리성 전투</a>가 일어났고, 이 때 <a href="/wiki/%EC%97%90%EC%B9%98%EA%B3%A0_%EA%B5%AD" title="에치고 국">에치고</a>의 우에스기 가게카쓰와도 관계가 나빴기 때문에 양쪽으로 공격을 당해 고전하였다. 도요토미 히데요시와 도쿠가와 이에야스와의 화의가 성립되자, 삿사 나리마사는 이에야스의 재기 독촉을 위해 히다 산맥을 넘어 도쿠가와 이에야스를 만났지만, 성과를 얻지 못하였다.</p>
<p><a href="/wiki/1583%EB%85%84" title="1583년">1583년</a>, 히데요시가 10만 대군으로 토야마 성을 포위하였다. 삿사 나리마사는 오다 노부카쓰의 중재로, 도요토미 히데요시에게 항복하였다. 히데요시의 조정으로 목숨은 건질 수 있었으나, 자신의 영토인 <a class="mw-redirect" href="/wiki/%EC%97%A3%EC%B8%84_%EA%B5%AD" title="엣츄 국">엣츄 국</a> 동부의 니시카와 군을 제외한 모든 영토를 몰수당했다.</p>
<p>이후, <a href="/wiki/%EA%B7%9C%EC%8A%88_%EC%A0%95%EB%B2%8C" title="규슈 정벌">규슈 정벌</a>에 참가하여 <a href="/wiki/%ED%9E%88%EA%B3%A0_%EA%B5%AD" title="히고 국">히고 국</a>을 받았으나, 히고의 호족들이 일제히 삿사의 정치에 들고 일어나는 바람에 할복 명령을 받게 되었다.</p>
<h2><span class="mw-headline" id=".EC.82.AC.ED.9B.84">사후</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%82%BF%EC%82%AC_%EB%82%98%EB%A6%AC%EB%A7%88%EC%82%AC&amp;action=edit&amp;section=2" title="부분 편집: 사후">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>나리마사의 외손녀 <a href="/wiki/%EB%8B%A4%EC%B9%B4%EC%93%B0%EC%B9%B4%EC%82%AC_%EB%8B%A4%EC%B9%B4%EC%BD%94" title="다카쓰카사 다카코">다카쓰카사 다카코</a>(<a class="new" href="/w/index.php?title=%EC%82%BF%EC%82%AC_%EB%8D%B0%EB%A3%A8%EC%BD%94&amp;action=edit&amp;redlink=1" title="삿사 데루코 (없는 문서)">데루코</a>의 딸)는 <a href="/wiki/%EC%97%90%EB%8F%84_%EB%A7%89%EB%B6%80" title="에도 막부">에도 막부</a> 제3대 쇼군 <a href="/wiki/%EB%8F%84%EC%BF%A0%EA%B0%80%EC%99%80_%EC%9D%B4%EC%97%90%EB%AF%B8%EC%93%B0" title="도쿠가와 이에미쓰">도쿠가와 이에미쓰</a>와 혼인, <a href="/wiki/%EB%AF%B8%EB%8B%A4%EC%9D%B4%EB%8F%84%EC%BD%94%EB%A1%9C" title="미다이도코로">미다이도코로</a>가 된다.</p>
<h2><span class="mw-headline" id=".EC.9D.BC.ED.99.94">일화</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%82%BF%EC%82%AC_%EB%82%98%EB%A6%AC%EB%A7%88%EC%82%AC&amp;action=edit&amp;section=3" title="부분 편집: 일화">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
