<p><b>상륙준비단</b>(Amphibious ready group, ARG)은 미국 해군의 상륙전 부대 단위이다. 상륙함들로 구성된 <a href="/wiki/%EB%AF%B8%EA%B5%AD_%ED%95%B4%EA%B5%B0" title="미국 해군">미국 해군</a>의 상륙 태스크포스와 <a href="/wiki/%EB%AF%B8%EA%B5%AD_%ED%95%B4%EB%B3%91%EB%8C%80" title="미국 해병대">미국 해병대</a>나 <a href="/wiki/%EB%AF%B8%EA%B5%AD_%EC%9C%A1%EA%B5%B0" title="미국 육군">미국 육군</a>으로 구성된 <a href="/wiki/%EC%83%81%EB%A5%99%EC%9E%91%EC%A0%84" title="상륙작전">상륙군대</a>로 구성되며, 모두 5,000명 규모이다. <a class="new" href="/w/index.php?title=%ED%95%AD%EB%AA%A8%EC%A0%84%ED%88%AC%EB%8B%A8&amp;action=edit&amp;redlink=1" title="항모전투단 (없는 문서)">항모전투단</a>(carrier battle group, CVBG)의 보호를 받는다.</p>
<h2><span class="mw-headline" id=".EA.B5.AC.EC.84.B1.EC.9A.94.EC.86.8C">구성요소</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%83%81%EB%A5%99%EC%A4%80%EB%B9%84%EB%8B%A8&amp;action=edit&amp;section=1" title="부분 편집: 구성요소">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<ul>
<li>한척의 <a href="/wiki/%EA%B0%95%EC%8A%B5%EC%83%81%EB%A5%99%ED%95%A8" title="강습상륙함">강습상륙함</a>(<a href="/wiki/LHA" title="LHA">LHA</a> 또는 <a class="mw-disambig" href="/wiki/LHD" title="LHD">LHD</a>) - 소형 항공모함이다. 첫번째 임무는 헬기 상륙작전이며, 두번째 임무는 해리어 전투기 운용이다. <a href="/wiki/LCAC" title="LCAC">LCAC</a> 호버크래프트도 탑재한다. 배수량 4만톤이며, 11척을 유지하고 있다.</li>
<li>한척의 <a class="new" href="/w/index.php?title=LPD&amp;action=edit&amp;redlink=1" title="LPD (없는 문서)">LPD</a> - <a href="/wiki/LCAC" title="LCAC">LCAC</a> 호버크래프트의 모함이며, 비행갑판이 있어서 헬기 상륙작전도 가능하다. 배수량 25,000톤 <a class="new" href="/w/index.php?title=%EC%83%8C_%EC%95%88%ED%86%A0%EB%8B%88%EC%98%A4%EA%B8%89&amp;action=edit&amp;redlink=1" title="샌 안토니오급 (없는 문서)">샌 안토니오급</a>은 <a href="/wiki/V-22_%EC%98%A4%EC%8A%A4%ED%94%84%EB%A6%AC" title="V-22 오스프리">MV-22 오스프리</a> 2대가 동시에 이착륙할 수 있고, 최대 4대를 비행갑판 위에 탑재할 수 있다. 해병대 700명이 탑승한다. 12척을 계획중이다.</li>
<li>한척의 <a class="mw-redirect" href="/wiki/LSD" title="LSD">LSD</a> - <a href="/wiki/LCAC" title="LCAC">LCAC</a> 호버크래프트의 모함이다.</li>
<li><a class="mw-redirect" href="/wiki/MEU" title="MEU">MEU</a> - 해병 원정대(MEU)의 부대장은 대령이며 2,300명으로 된 연대급 규모이다. 7개 해병 원정대가 창설되어 있다. 과거에는 상륙준비단 소속이었으나 1990년대 이후에는 <a href="/wiki/%EC%9B%90%EC%A0%95_%ED%83%80%EA%B2%A9%EB%8B%A8" title="원정 타격단">원정 타격단</a> 소속이다.</li>
<li><a class="mw-disambig" href="/wiki/%ED%95%B4%EB%A6%AC%EC%96%B4" title="해리어">해리어</a> 전투기</li>
<li><a href="/wiki/%EC%8B%9C%EC%BD%94%EB%A5%B4%EC%8A%A4%ED%82%A4_CH-53E_%EC%8A%88%ED%8D%BC_%EC%8A%A4%ED%83%A4%EB%A6%AC%EC%98%A8" title="시코르스키 CH-53E 슈퍼 스탤리온">CH-53E 슈퍼 스탤리온</a> 대형 수송헬기 - 무게가 33톤이나 되는 대형 장거리 수송 헬기이다. 완전무장한 해병 50여명을 태우고 1,800 km나 비행한다.</li>
<li><a class="new" href="/w/index.php?title=%EC%8A%88%ED%8D%BC_%EC%BD%94%EB%B8%8C%EB%9D%BC&amp;action=edit&amp;redlink=1" title="슈퍼 코브라 (없는 문서)">슈퍼 코브라</a> 또는 <a class="mw-disambig" href="/wiki/%EB%B0%94%EC%9D%B4%ED%8D%BC" title="바이퍼">바이퍼</a> 공격헬기 - 60년대 육군의 4.3톤 단발엔진 <a href="/wiki/AH-1_%EC%BD%94%EB%B8%8C%EB%9D%BC" title="AH-1 코브라">AH-1 코브라</a> 공격헬기를 8.4톤 쌍발엔진으로 업그레이드한 것으로, 상륙부대의 화력지원을 한다. 2010년 부터 실전배치되었다.</li>
<li><a class="new" href="/w/index.php?title=%EC%8A%88%ED%8D%BC_%ED%9C%B4%EC%9D%B4&amp;action=edit&amp;redlink=1" title="슈퍼 휴이 (없는 문서)">슈퍼 휴이</a> 헬기 - 월남전 때 유명한 60년대 육군의 4.3톤 단발엔진 <a href="/wiki/UH-1_%EC%9D%B4%EB%A1%9C%EC%BF%BC%EC%9D%B4" title="UH-1 이로쿼이">휴이</a> 수송헬기를 8.4톤 쌍발엔진으로 업그레이드했다. 2008년 부터 실전배치되었다.</li>
<li><a href="/wiki/V-22_%EC%98%A4%EC%8A%A4%ED%94%84%EB%A6%AC" title="V-22 오스프리">MV-22 오스프리</a> 수직이착륙 수송기 - 27톤 MV-22 오스프리는 2007년 부터 실전배치되었다. 60년대 11톤 <a href="/wiki/CH-46_%EC%8B%9C%EB%82%98%EC%9D%B4%ED%8A%B8" title="CH-46 시나이트">CH-46 시나이트</a> 수송헬기를 대체했다. 둘 다 20여명의 보병을 수송하지만, 시나이트 전투행동반경이 296 km인데 비해, 오스프리는 722 km나 된다. 2012년 퇴역한 시나이트를 한국에 기부했다.</li>
</ul>
