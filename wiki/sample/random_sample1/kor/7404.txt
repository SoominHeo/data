<h1 class="firstHeading" id="firstHeading" lang="ko">상원군</h1>
<div class="mw-body-content" id="bodyContent">
<div id="siteSub">위키백과, 우리 모두의 백과사전.</div>
<div id="contentSub"></div>
<div class="mw-jump" id="jump-to-nav">
					이동:					<a href="#mw-head">둘러보기</a>, 					<a href="#p-search">검색</a>
</div>
<div class="mw-content-ltr" dir="ltr" id="mw-content-text" lang="ko"><script>function mfTempOpenSection(id){var block=document.getElementById("mf-section-"+id);block.className+=" open-block";block.previousSibling.className+=" open-block";}</script><div class="dablink"><a href="/wiki/%EC%9C%84%ED%82%A4%EB%B0%B1%EA%B3%BC:%EB%8F%99%EC%9D%8C%EC%9D%B4%EC%9D%98%EC%96%B4_%EB%AC%B8%EC%84%9C" title="위키백과:동음이의어 문서"><img alt="" data-file-height="200" data-file-width="260" height="18" src="//upload.wikimedia.org/wikipedia/commons/thumb/4/4a/Disambig_grey.svg/23px-Disambig_grey.svg.png" srcset="//upload.wikimedia.org/wikipedia/commons/thumb/4/4a/Disambig_grey.svg/35px-Disambig_grey.svg.png 1.5x, //upload.wikimedia.org/wikipedia/commons/thumb/4/4a/Disambig_grey.svg/46px-Disambig_grey.svg.png 2x" width="23"/></a> 이 문서는 조선민주주의인민공화국의 행정 구역에 관한 것입니다. 조선의 왕족에 대해서는 <a href="/wiki/%EC%83%81%EC%9B%90%EA%B5%B0_(%EC%99%95%EC%A1%B1)" title="상원군 (왕족)">상원군 (왕족)</a> 문서를 참조하십시오.</div>
<p><b>상원군</b>(祥原郡)은 <a href="/wiki/%EC%A1%B0%EC%84%A0%EB%AF%BC%EC%A3%BC%EC%A3%BC%EC%9D%98%EC%9D%B8%EB%AF%BC%EA%B3%B5%ED%99%94%EA%B5%AD" title="조선민주주의인민공화국">조선민주주의인민공화국</a> <a href="/wiki/%ED%99%A9%ED%95%B4%EB%B6%81%EB%8F%84" title="황해북도">황해북도</a>에 위치한다. 면적은 약 490㎢ 이고 인구는 <a href="/wiki/2008%EB%85%84" title="2008년">2008년</a> 기준으로 92,831명이다.<sup class="reference" id="cite_ref-1"><a href="#cite_note-1">[1]</a></sup> 남쪽으로 <a href="/wiki/%ED%99%A9%EC%A3%BC%EA%B5%B0" title="황주군">황주군</a>, <a href="/wiki/%EC%97%B0%ED%83%84%EA%B5%B0" title="연탄군">연탄군</a>, <a href="/wiki/%EC%88%98%EC%95%88%EA%B5%B0" title="수안군">수안군</a>과 접하고 북쪽으로 남강을 사이에 두고 <a href="/wiki/%ED%8F%89%EC%96%91%EC%A7%81%ED%95%A0%EC%8B%9C" title="평양직할시">평양직할시</a> <a href="/wiki/%EA%B0%95%EB%8F%99%EA%B5%B0" title="강동군">강동군</a>, 동쪽으로 황해북도 <a href="/wiki/%EC%97%B0%EC%82%B0%EA%B5%B0_(%ED%99%A9%ED%95%B4%EB%B6%81%EB%8F%84)" title="연산군 (황해북도)">연산군</a>, 서쪽으로 <a href="/wiki/%EC%A4%91%ED%99%94%EA%B5%B0" title="중화군">중화군</a>과 접한다. 군내에는 협동농장과 소형 공업단지가 존재한다. 이 곳은 <a href="/wiki/%EA%B5%AC%EC%84%9D%EA%B8%B0_%EC%8B%9C%EB%8C%80" title="구석기 시대">구석기</a>사람들이 남긴 <a href="/wiki/%EA%B2%80%EC%9D%80%EB%AA%A8%EB%A3%A8%EB%8F%99%EA%B5%B4%EC%9C%A0%EC%A0%81" title="검은모루동굴유적">검은모루 동굴</a>이 있다. 원래 평양직할시에 위치하였지만 <a href="/wiki/2010%EB%85%84" title="2010년">2010년</a> 황해북도로 편입되었다. 한국의 지질에서 <a class="new" href="/w/index.php?title=%EC%83%81%EC%9B%90%EA%B3%84&amp;action=edit&amp;redlink=1" title="상원계 (없는 문서)">상원계</a>는 이 지역에서 유래했다.</p>

<h2><span class="mw-headline" id=".EC.97.AD.EC.82.AC">역사</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%83%81%EC%9B%90%EA%B5%B0&amp;action=edit&amp;section=1" title="부분 편집: 역사">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<ul>
<li><a href="/wiki/1914%EB%85%84" title="1914년">1914년</a> 이전에는 <a href="/wiki/%ED%8F%89%EC%95%88%EB%82%A8%EB%8F%84" title="평안남도">평안남도</a> 상원군 이었다.</li>
<li><a href="/wiki/1914%EB%85%84" title="1914년">1914년</a> <a href="/wiki/%EC%A4%91%ED%99%94%EA%B5%B0" title="중화군">중화군</a>에 합쳐져 폐지되었다.</li>
<li><a href="/wiki/1952%EB%85%84" title="1952년">1952년</a> <a href="/wiki/12%EC%9B%94" title="12월">12월</a> - 평안남도 중화군 상원면, 수산면, 천곡면과 강동군 풍동면 및 강동면의 일부, 황해도 황주군의 천주면의 일부를 합병해 상원군을 신설했다.</li>
<li><a href="/wiki/1963%EB%85%84" title="1963년">1963년</a> <a href="/wiki/5%EC%9B%94" title="5월">5월</a> - 평안남도에서 분리되어 평양직할시에 편입되었다.</li>
<li><a href="/wiki/2010%EB%85%84" title="2010년">2010년</a> - 평양직할시에 분리되어 황해북도로 편입되었다.</li>
</ul>
