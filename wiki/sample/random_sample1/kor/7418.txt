<p><b>상하이국제금융센터</b>(<span style="font-size: smaller;"><a href="/wiki/%EC%A4%91%EA%B5%AD%EC%96%B4" title="중국어">중국어</a>:</span> <span lang="zh" xml:lang="zh">上海国际金融中心</span> <small>상해국제금융중심<sup>[<a href="/wiki/%EC%9C%84%ED%82%A4%EB%B0%B1%EA%B3%BC:%EC%A4%91%EA%B5%AD%EC%96%B4%EC%9D%98_%ED%95%9C%EA%B8%80_%ED%91%9C%EA%B8%B0" title="위키백과:중국어의 한글 표기">*</a>]</sup></small>, <span style="font-size: smaller;"><a href="/wiki/%EC%98%81%EC%96%B4" title="영어">영어</a>:</span> <span lang="en" xml:lang="en">Shanghai International Finance Centre</span>, Shanghai IFC)는 <a href="/wiki/%EC%A4%91%EA%B5%AD" title="중국">중국</a> <a class="mw-redirect" href="/wiki/%EC%83%81%ED%95%98%EC%9D%B4" title="상하이">상하이</a>의 <a class="mw-redirect" href="/wiki/%ED%91%B8%EB%91%A5_%EC%A7%80%EA%B5%AC" title="푸둥 지구">푸둥 지구</a> <a href="/wiki/%EB%A3%A8%EC%9E%90%EC%AD%88%EC%9D%B4" title="루자쭈이">루지아즈이</a> 금융지구에 위치한 <a class="mw-disambig" href="/wiki/%EA%B5%AD%EC%A0%9C%EA%B8%88%EC%9C%B5%EC%84%BC%ED%84%B0" title="국제금융센터">국제금융센터</a> 겸 <a href="/wiki/%EB%A7%88%EC%B2%9C%EB%A3%A8" title="마천루">마천루</a> 단지이다. 중국의 기업 선 흥 카이 부동산 그룹과 <a href="/wiki/%ED%99%8D%EC%BD%A9%EC%83%81%ED%95%98%EC%9D%B4%EC%9D%80%ED%96%89" title="홍콩상하이은행">홍콩 상하이 은행</a>이 1998년 개장한 <a class="mw-redirect" href="/wiki/%ED%99%8D%EC%BD%A9%EA%B5%AD%EC%A0%9C%EA%B8%88%EC%9C%B5%EC%84%BC%ED%84%B0" title="홍콩국제금융센터">홍콩국제금융센터</a>를 <a href="/wiki/%EB%B2%A4%EC%B9%98%EB%A7%88%ED%82%B9" title="벤치마킹">벤치마킹</a>하여 건설한 <a class="mw-disambig" href="/wiki/%EA%B5%AD%EC%A0%9C%EA%B8%88%EC%9C%B5%EC%84%BC%ED%84%B0" title="국제금융센터">국제금융센터</a>이며, 남부 타워는 <a href="/wiki/2009%EB%85%84" title="2009년">2009년</a>에, 북부 타워는 <a href="/wiki/2010%EB%85%84" title="2010년">2010년</a>에 착공하여 <a href="/wiki/2012%EB%85%84" title="2012년">2012년</a> 개장하였다. 약칭인 <b>상하이 IFC</b>로 더 알려져 있다.<br/>
101층짜리 <a href="/wiki/%EC%83%81%ED%95%98%EC%9D%B4_%EC%84%B8%EA%B3%84%EA%B8%88%EC%9C%B5%EC%84%BC%ED%84%B0" title="상하이 세계금융센터">상하이 세계금융센터</a>와 혼동하는 경우가 있으나 전혀 다른 건물이며, 세계금융센터와는 약 한 블럭 정도 떨어져 있다.</p>
<p><a href="/wiki/2008%EB%85%84" title="2008년">2008년</a>에 개장한 <a href="/wiki/%EC%83%81%ED%95%98%EC%9D%B4_%EC%84%B8%EA%B3%84%EA%B8%88%EC%9C%B5%EC%84%BC%ED%84%B0" title="상하이 세계금융센터">상하이 세계금융센터</a>(101층, 492m)의 기능을 보조하며 상하이 시의 금융업무의 활성화를 위하여 건설하였다. 북쪽의 A타워가 <b>지상 60층, 260m</b>이며, 남쪽의 B타워가 <b>지상 57층, 250m</b>의 높이로 건설되었다. 남부 타워는 흔히 HSBC 빌딩이라는 별칭이 있다.</p>

<h2><span class="mw-headline" id=".EA.B5.AC.EC.84.B1">구성</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%83%81%ED%95%98%EC%9D%B4%EA%B5%AD%EC%A0%9C%EA%B8%88%EC%9C%B5%EC%84%BC%ED%84%B0&amp;action=edit&amp;section=2" title="부분 편집: 구성">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>쌍둥이 빌딩 두 채와 호텔 1동, 지하의 IFC 몰로 구성되어 있는데, 쌍둥이 빌딩은 A동이 지상 60층, 260m, B동이 지상 57층, 250m정도로 건축되었으며, 그 외에 22층, 88m의 호텔과 지하의 IFC몰 등이 입점해 있다.</p>
