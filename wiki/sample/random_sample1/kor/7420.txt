<p><b>새만금 사업</b>(새萬金事業)은 <a href="/wiki/%EB%8C%80%ED%95%9C%EB%AF%BC%EA%B5%AD" title="대한민국">대한민국</a> <a href="/wiki/%EC%A0%84%EB%9D%BC%EB%B6%81%EB%8F%84" title="전라북도">전라북도</a>의 <a href="/wiki/%EA%B5%B0%EC%82%B0%EC%8B%9C" title="군산시">군산시</a> <a class="mw-redirect" href="/wiki/%EB%B9%84%EC%9D%91%EB%8F%84%EB%8F%99" title="비응도동">비응도동</a>부터 <a href="/wiki/%EA%B3%A0%EA%B5%B0%EC%82%B0%EA%B5%B0%EB%8F%84" title="고군산군도">고군산군도</a>의 <a href="/wiki/%EC%8B%A0%EC%8B%9C%EB%8F%84" title="신시도">신시도</a>를 거쳐 <a href="/wiki/%EB%B6%80%EC%95%88%EA%B5%B0" title="부안군">부안군</a> <a href="/wiki/%EB%B3%80%EC%82%B0%EB%A9%B4" title="변산면">변산면</a> 대항리까지 총 33.9km에 이르는 <a class="mw-redirect" href="/wiki/%EC%83%88%EB%A7%8C%EA%B8%88_%EB%B0%A9%EC%A1%B0%EC%A0%9C" title="새만금 방조제">새만금 방조제</a>를 건설해 <a class="mw-redirect" href="/wiki/%EC%84%9C%ED%95%B4" title="서해">서해안</a>의 <a class="mw-redirect" href="/wiki/%EA%B0%AF%EB%B2%8C" title="갯벌">갯벌</a>과 <a href="/wiki/%EB%B0%94%EB%8B%A4" title="바다">바다</a>를 육지로 바꾸는 <a href="/wiki/%EA%B0%84%EC%B2%99" title="간척">간척</a> 사업이다.</p>
<p><a href="/wiki/1991%EB%85%84" title="1991년">1991년</a> <a href="/wiki/11%EC%9B%94_16%EC%9D%BC" title="11월 16일">11월 16일</a>에 기공하여 <a href="/wiki/2006%EB%85%84" title="2006년">2006년</a> <a href="/wiki/4%EC%9B%94_21%EC%9D%BC" title="4월 21일">4월 21일</a>에 물막이 공사가 완료되었고, 보강 및 성토작업 등을 거쳐 <a href="/wiki/2010%EB%85%84" title="2010년">2010년</a> <a href="/wiki/4%EC%9B%94_27%EC%9D%BC" title="4월 27일">4월 27일</a> 준공되었다. <a href="/wiki/2007%EB%85%84" title="2007년">2007년</a> <a href="/wiki/11%EC%9B%94_22%EC%9D%BC" title="11월 22일">11월 22일</a>, 이 사업의 개발, 이용 촉진을 지원하기 위해 <a href="/wiki/%EB%8C%80%ED%95%9C%EB%AF%BC%EA%B5%AD_%EA%B5%AD%ED%9A%8C" title="대한민국 국회">국회</a>에서 '새만금 사업 촉진을 위한 특별법'이 통과되었다.</p>
<p><a class="mw-redirect" href="/wiki/%EC%83%88%EB%A7%8C%EA%B8%88_%EB%B0%A9%EC%A1%B0%EC%A0%9C" title="새만금 방조제">새만금 방조제</a>는 기존에 세계에서 가장 긴 <a href="/wiki/%EB%B0%A9%EC%A1%B0%EC%A0%9C" title="방조제">방조제</a>로 알려졌던 <a href="/wiki/%EB%84%A4%EB%8D%9C%EB%9E%80%EB%93%9C" title="네덜란드">네덜란드</a>의 <a class="mw-redirect" href="/wiki/%EC%9E%90%EC%9C%84%EB%8D%94%EB%A5%B4_%EB%B0%A9%EC%A1%B0%EC%A0%9C" title="자위더르 방조제">자위더르 방조제</a>(32.5 km)보다 1.4 km 더 길며, 세계에서 가장 긴 <a href="/wiki/%EB%B0%A9%EC%A1%B0%EC%A0%9C" title="방조제">방조제</a>로 <a class="mw-redirect" href="/wiki/%EA%B8%B0%EB%84%A4%EC%8A%A4%EB%B6%81" title="기네스북">기네스북</a>에 등재되었다.<sup class="reference" id="cite_ref-1"><a href="#cite_note-1">[1]</a></sup></p>

<h2><span class="mw-headline" id=".EA.B0.9C.EC.9A.94">개요</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%83%88%EB%A7%8C%EA%B8%88_%EA%B0%84%EC%B2%99_%EC%82%AC%EC%97%85&amp;action=edit&amp;section=1" title="부분 편집: 개요">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p><b>새만금 개발 사업</b>은 <a href="/wiki/%EC%A0%84%EB%9D%BC%EB%B6%81%EB%8F%84" title="전라북도">전라북도</a> <a href="/wiki/%EA%B5%B0%EC%82%B0%EC%8B%9C" title="군산시">군산시</a>, <a href="/wiki/%EA%B9%80%EC%A0%9C%EC%8B%9C" title="김제시">김제시</a>, <a href="/wiki/%EB%B6%80%EC%95%88%EA%B5%B0" title="부안군">부안군</a> 앞바다를 연결하는 방조제 33.9 km를 쌓아 그 안에 간척토지 291 km², <a href="/wiki/%ED%98%B8%EC%88%98" title="호수">호수</a>(새만금호) 118 km²를 만드는 계획이다.</p>
<p>이 사업은 전북 <a href="/wiki/%EC%98%A5%EA%B5%AC%EA%B5%B0" title="옥구군">옥구군</a>(현 <a href="/wiki/%EA%B5%B0%EC%82%B0%EC%8B%9C" title="군산시">군산시</a>) <a href="/wiki/%EC%98%A5%EC%84%9C%EB%A9%B4" title="옥서면">옥서면</a>을 중심으로 <a href="/wiki/%EA%B8%88%EA%B0%95" title="금강">금강</a>, <a href="/wiki/%EB%A7%8C%EA%B2%BD%EA%B0%95" title="만경강">만경강</a>, <a href="/wiki/%EB%8F%99%EC%A7%84%EA%B0%95" title="동진강">동진강</a> 하구를 둘러싼 갯벌을 간척하려던 <b>옥서지구 농업개발계획</b>에서 출발하였다. <a href="/wiki/1971%EB%85%84" title="1971년">1971년</a> 계획된 옥서지구 농업개발계획은 1단계로 <a class="mw-redirect" href="/wiki/%EA%B8%88%EA%B0%95%ED%95%98%EA%B5%AC%EB%91%91" title="금강하구둑">금강하구둑</a>을 건설하여 <a href="/wiki/%EA%B8%88%EA%B0%95" title="금강">금강</a> 주변 농지의 배수시설을 개선하고, 2단계로 김제지구에 9.6 km의 방조제(<a href="/wiki/%EA%B5%B0%EC%82%B0%EC%8B%9C" title="군산시">군산시</a> <a href="/wiki/%EC%98%A5%EC%84%9C%EB%A9%B4" title="옥서면">옥서면</a> 선연리 ~ <a href="/wiki/%EB%B6%80%EC%95%88%EA%B5%B0" title="부안군">부안군</a> <a href="/wiki/%EA%B3%84%ED%99%94%EB%A9%B4" title="계화면">계화면</a> 계화리)를 쌓아 120 km²의 갯벌을 간척하고 <a href="/wiki/%EB%A7%8C%EA%B2%BD%EA%B0%95" title="만경강">만경강</a> 주변 농경지의 관개배수시설을 개선하는 것을 내용으로 하고 있었다. 새만금 개발 사업은 옥서지구 농업개발계획 2단계사업을 4배 가까이 확대한 사업으로, <a href="/wiki/%EB%8C%80%ED%95%9C%EB%AF%BC%EA%B5%AD_%EC%A0%9C13%EB%8C%80_%EB%8C%80%ED%86%B5%EB%A0%B9_%EC%84%A0%EA%B1%B0" title="대한민국 제13대 대통령 선거">1987년 대통령 선거</a>에 출마한 <a href="/wiki/%EB%85%B8%ED%83%9C%EC%9A%B0" title="노태우">노태우</a>가 같은 해 <a href="/wiki/12%EC%9B%94_10%EC%9D%BC" title="12월 10일">12월 10일</a> <a class="mw-redirect" href="/wiki/%EC%A0%84%EC%A3%BC" title="전주">전주</a> 유세에서 이를 대선공약으로 발표했다.<sup class="reference" id="cite_ref-2"><a href="#cite_note-2">[2]</a></sup></p>
<p>이 사업은 시작부터 정치적 목적으로 저개발 상태인 전북 지역에 대한 개발 공약으로 제시되었기 때문에, 농지 공급 부족 대책으로 사업시행을 허가한다는 애초의 국가적 명분과는 달리 시간이 갈수록 공업, 레저 등 다른 산업용지의 비중이 대폭 늘어나고 있다.</p>
<h3><span class="mw-headline" id=".EC.83.88.EB.A7.8C.EA.B8.88.EC.9D.98_.EB.AA.85.EC.B9.AD.EA.B3.BC_.EC.9D.98.EB.AF.B8">새만금의 명칭과 의미</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%83%88%EB%A7%8C%EA%B8%88_%EA%B0%84%EC%B2%99_%EC%82%AC%EC%97%85&amp;action=edit&amp;section=2" title="부분 편집: 새만금의 명칭과 의미">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>'새만금'이란 전국 최대의 곡창지대인 <a class="new" href="/w/index.php?title=%EB%A7%8C%EA%B2%BD%ED%8F%89%EC%95%BC&amp;action=edit&amp;redlink=1" title="만경평야 (없는 문서)">만경평야</a>와 <a class="new" href="/w/index.php?title=%EA%B9%80%EC%A0%9C%ED%8F%89%EC%95%BC&amp;action=edit&amp;redlink=1" title="김제평야 (없는 문서)">김제평야</a>를 합친 만큼의 새로운 땅이 생긴다는 뜻의 말로, 만경평야의 '만'(萬)자와 김제평야의 '금'(金)자를 따서 새만금이라 하였다. <a href="/wiki/%EC%A0%84%EB%9D%BC%EB%B6%81%EB%8F%84" title="전라북도">전라북도</a> <a href="/wiki/%EA%B9%80%EC%A0%9C%EC%8B%9C" title="김제시">김제시</a>의 김제·만경평야는 예부터 '금만평야'로 불렸는데, 새만금은 이 '금만'이라는 말을 '만금'으로 바꾸고, 새롭다는 뜻의 '새'를 덧붙여 만든 말이다. 오래전부터 옥토로 유명한 만경·김제평야와 같은 옥토를 새로이 일구어 내겠다는 의미가 담겨 있다.</p>
<p>새만금의 명칭이 공식적으로 사용된 때는 <a href="/wiki/1987%EB%85%84" title="1987년">1987년</a> 11월 2일이다. 당시 정인용 부총리 주재로 열린 관계장관회의에서 황인성 농림수산부 장관이 처음으로 <a class="mw-redirect" href="/wiki/%EA%B5%B0%EC%82%B0" title="군산">군산</a>과 <a class="mw-redirect" href="/wiki/%EB%B6%80%EC%95%88" title="부안">부안</a> 사이의 서해안 간척사업을 가리켜 '새만금 간척사업'이라는 이름을 공식 사용하였다.</p>

<h3><span class="mw-headline" id=".EC.83.88.EB.A7.8C.EA.B8.88_.EC.8B.A0.EB.8F.84.EC.8B.9C">새만금 신도시</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%83%88%EB%A7%8C%EA%B8%88_%EA%B0%84%EC%B2%99_%EC%82%AC%EC%97%85&amp;action=edit&amp;section=8" title="부분 편집: 새만금 신도시">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>2010년 1월 29일에 <a href="/wiki/%EC%9D%B4%EB%AA%85%EB%B0%95_%EC%A0%95%EB%B6%80" title="이명박 정부">이명박 정부</a>는 새만금 신도시 계획에 따라 새만금에 지을 신도시 이름을 <b>아리울</b>로 정하였다. 이에 대해 아리울은 물을 뜻하는 ‘아리’와 울타리, 터전을 뜻하는 '울'을 합성해 만든 <a href="/wiki/%ED%95%9C%EA%B5%AD%EC%96%B4%EC%9D%98_%EA%B3%A0%EC%9C%A0%EC%96%B4" title="한국어의 고유어">토박이말</a> 이름이라고 설명하였다.<sup class="reference" id="cite_ref-5"><a href="#cite_note-5">[5]</a></sup></p>
<p>'아리'는 <a href="/wiki/%EC%9D%B4%EB%AA%85%EB%B0%95" title="이명박">이명박</a>이 <a class="mw-redirect" href="/wiki/%EC%84%9C%EC%9A%B8%EC%8B%9C%EC%9E%A5" title="서울시장">서울시장</a>이던 <a href="/wiki/2004%EB%85%84" title="2004년">2004년</a>에 서울시 수돗물에 이름을 붙인 <a href="/wiki/%EC%95%84%EB%A6%AC%EC%88%98" title="아리수">아리수</a>에서 따온 말로, 아리수는 <a href="/wiki/%EA%B4%91%EA%B0%9C%ED%86%A0%EC%99%95%EB%A6%89%EB%B9%84" title="광개토왕릉비">광개토왕릉비</a> 등에서 <a href="/wiki/%EA%B3%A0%EA%B5%AC%EB%A0%A4" title="고구려">고구려</a>가 <a href="/wiki/%ED%95%9C%EA%B0%95" title="한강">한강</a>(漢江)을 지칭한 '아리수(阿利水)'에서 본 딴 것으로, 아리라는 말 자체는 <a href="/wiki/%EB%AC%BC" title="물">물</a>을 뜻하지 않는다. 다만, 한강의 '한'을 '크다'라는 뜻의 순우리말에서 온 것으로 새긴 후 아리를 그와 같은 뜻으로 보아 '<b>크다'</b>라는 뜻으로 보는 견해가 있을 뿐이다. 또, '울'은 '울타리' 또는 '우리', 즉 <a href="/wiki/%EC%B6%95%EC%82%AC" title="축사">축사</a>(畜舍)라는 뜻이다. 이 말은 <a class="mw-redirect" href="/wiki/%ED%99%A9%ED%95%B4%EB%8F%84_%EB%B0%A9%EC%96%B8" title="황해도 방언">황해도 방언</a>에서만 <a class="mw-redirect" href="/wiki/%EB%9C%B0" title="뜰">뜰</a>, 즉 마당이라는 뜻으로도 쓰인다.</p>
<h2><span class="mw-headline" id=".EB.B0.A9.EC.A1.B0.EC.A0.9C.EC.99.80_.EA.B0.91.EB.AC.B8">방조제와 갑문</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%83%88%EB%A7%8C%EA%B8%88_%EA%B0%84%EC%B2%99_%EC%82%AC%EC%97%85&amp;action=edit&amp;section=9" title="부분 편집: 방조제와 갑문">편집</a><span class="mw-editsection-bracket">]</span></span></h2>

<h2><span class="mw-headline" id=".ED.96.89.EC.A0.95.EA.B5.AC.EC.97.AD">행정구역</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%83%88%EB%A7%8C%EA%B8%88_%EA%B0%84%EC%B2%99_%EC%82%AC%EC%97%85&amp;action=edit&amp;section=10" title="부분 편집: 행정구역">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>새만금 간척지의 <a href="/wiki/%ED%96%89%EC%A0%95_%EA%B5%AC%EC%97%AD" title="행정 구역">행정 구역</a>은 아직 확정되지 않은 상태로, <a href="/wiki/%EA%B5%B0%EC%82%B0%EC%8B%9C" title="군산시">군산시</a>, <a href="/wiki/%EA%B9%80%EC%A0%9C%EC%8B%9C" title="김제시">김제시</a>, <a href="/wiki/%EB%B6%80%EC%95%88%EA%B5%B0" title="부안군">부안군</a> 사이에 <a href="/wiki/%ED%96%89%EC%A0%95_%EA%B5%AC%EC%97%AD" title="행정 구역">행정 구역</a>의 획정에 관한 분쟁이 있다. <a href="/wiki/%EA%B5%B0%EC%82%B0%EC%8B%9C" title="군산시">군산시</a>는 기존의 해상경계선을 기준으로, <a href="/wiki/%EA%B9%80%EC%A0%9C%EC%8B%9C" title="김제시">김제시</a>는 매립으로 길어지는 <a href="/wiki/%EB%A7%8C%EA%B2%BD%EA%B0%95" title="만경강">만경강</a>과 <a href="/wiki/%EB%8F%99%EC%A7%84%EA%B0%95" title="동진강">동진강</a>을 기준으로, <a href="/wiki/%EB%B6%80%EC%95%88%EA%B5%B0" title="부안군">부안군</a>은 기존의 만경강, 동진강과 바다 수심이 가장 깊은 지점을 이은 선을 기준으로 행정구역을 획정할 것을 각각 주장하며 대립해 왔는데, 이는 결국 <a href="/wiki/%EC%8B%A0%EC%8B%9C%EB%8F%84" title="신시도">신시도</a>와 <a href="/wiki/%EA%B0%80%EB%A0%A5%EB%8F%84" title="가력도">가력도</a> 사이를 잇는 2호 방조제의 관할권을 3개 시군 중 어느 <a class="mw-redirect" href="/wiki/%EC%A7%80%EB%B0%A9%EC%9E%90%EC%B9%98%EB%8B%A8%EC%B2%B4" title="지방자치단체">지방자치단체</a>가 갖느냐의 다툼이었다.<sup class="reference" id="cite_ref-.ED.96.89.EC.A0.95.EA.B5.AC.EC.97.AD_6-0"><a href="#cite_note-.ED.96.89.EC.A0.95.EA.B5.AC.EC.97.AD-6">[6]</a></sup></p>
<p><a href="/wiki/2013%EB%85%84" title="2013년">2013년</a> <a href="/wiki/11%EC%9B%94_14%EC%9D%BC" title="11월 14일">11월 14일</a> 대법원은 <a href="/wiki/%EA%B5%B0%EC%82%B0%EC%8B%9C" title="군산시">군산시</a> 육지와 <a href="/wiki/%EC%8B%A0%EC%8B%9C%EB%8F%84" title="신시도">신시도</a>를 잇는 3호·4호 방조제를 <a href="/wiki/%EA%B5%B0%EC%82%B0%EC%8B%9C" title="군산시">군산시</a>가 관할하도록 판결하였으며,<sup class="reference" id="cite_ref-.ED.96.89.EC.A0.95.EA.B5.AC.EC.97.AD_6-1"><a href="#cite_note-.ED.96.89.EC.A0.95.EA.B5.AC.EC.97.AD-6">[6]</a></sup> <a href="/wiki/2015%EB%85%84" title="2015년">2015년</a> <a href="/wiki/10%EC%9B%94_26%EC%9D%BC" title="10월 26일">10월 26일</a> <a class="mw-redirect" href="/wiki/%ED%96%89%EC%A0%95%EC%9E%90%EC%B9%98%EB%B6%80" title="행정자치부">행정자치부</a> 중앙분쟁조정위원회는 <a href="/wiki/%EB%B6%80%EC%95%88%EA%B5%B0" title="부안군">부안군</a> 육지와 <a href="/wiki/%EA%B0%80%EB%A0%A5%EB%8F%84" title="가력도">가력도</a>를 잇는 1호 방조제를 <a href="/wiki/%EB%B6%80%EC%95%88%EA%B5%B0" title="부안군">부안군</a>이 관할하고, 2호 방조제를 <a href="/wiki/%EA%B9%80%EC%A0%9C%EC%8B%9C" title="김제시">김제시</a>가 관할하도록 결정하였다.<sup class="reference" id="cite_ref-7"><a href="#cite_note-7">[7]</a></sup><sup class="reference" id="cite_ref-8"><a href="#cite_note-8">[8]</a></sup></p>
<h2><span class="mw-headline" id=".ED.99.98.EA.B2.BD_.EA.B4.80.EB.A0.A8_.EB.85.BC.EB.9E.80">환경 관련 논란</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%83%88%EB%A7%8C%EA%B8%88_%EA%B0%84%EC%B2%99_%EC%82%AC%EC%97%85&amp;action=edit&amp;section=11" title="부분 편집: 환경 관련 논란">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>사업 시행 당시의 쌀 공급 부족 추세와는 달리, 현재 쌀 공급이 수요를 웃돌고 쌀시장의 완전 개방이 가시화된 시점에서 농업용지의 가치가 줄어들자 매립용지를 공업, 상업, 도시용도로 사용하기 위한 계획이 추진중이다. 이에 대해 <a href="/wiki/%EB%8C%80%ED%95%9C%EB%AF%BC%EA%B5%AD" title="대한민국">대한민국</a>과 여러 나라의 <a class="new" href="/w/index.php?title=%ED%99%98%EA%B2%BD%EB%8B%A8%EC%B2%B4&amp;action=edit&amp;redlink=1" title="환경단체 (없는 문서)">환경단체</a>는 개발로 인해 방대한 영역의 갯벌과 <a href="/wiki/%EB%B0%94%EB%8B%A4" title="바다">해양</a> <a href="/wiki/%EC%83%9D%ED%83%9C%EA%B3%84" title="생태계">생태계</a>가 파괴될 것을 우려해 <a href="/wiki/%EC%A0%95%EB%B6%80" title="정부">정부</a>를 상대로 소송을 벌이는 등 개발 반대의 목소리를 높이고 있다. 또한 최근에는 새만금 갯벌의 생태적 가치가 월등히 높다는 <a href="/wiki/2004%EB%85%84" title="2004년">2004년</a> 환경부의 보고서가 공개되어 한때 논란이 지속되었다.</p>
<h3><span class="mw-headline" id=".EB.B2.95.EC.A0.81_.EC.86.8C.EC.86.A1">법적 소송</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%83%88%EB%A7%8C%EA%B8%88_%EA%B0%84%EC%B2%99_%EC%82%AC%EC%97%85&amp;action=edit&amp;section=12" title="부분 편집: 법적 소송">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p><a href="/wiki/2006%EB%85%84" title="2006년">2006년</a> <a href="/wiki/3%EC%9B%94_16%EC%9D%BC" title="3월 16일">3월 16일</a>, <a href="/wiki/%EB%8C%80%EB%B2%95%EC%9B%90" title="대법원">대법원</a> 전원합의체(13명)는 환경단체와 전북지역 주민 등이 <a href="/wiki/%EB%8C%80%ED%95%9C%EB%AF%BC%EA%B5%AD_%EB%86%8D%EB%A6%BC%EC%88%98%EC%82%B0%EC%8B%9D%ED%92%88%EB%B6%80" title="대한민국 농림수산식품부">농림부</a>(현 농림수산식품부) 등을 상대로 낸 '새만금 사업' 계획 취소 청구소송에서 원고패소 판결한 원심을 확정하였다. 다수 의견(11명)으로 새만금 사업의 합법성을 인정했지만, 환경 보전 가치를 개발 가치보다 우선해야 한다며 새만금 사업 취소를 명시한 소수 의견(2명)과 환경 친화적인 개발 방안을 찾아야 한다는 보충 의견(4명)도 있었다.</p>
<p><a class="external text" href="http://www.lawtimes.co.kr/images/대법원%202006두330%20새만금사건판결문-중부조치계획최소등.pdf" rel="nofollow">대법원 판결문 전문</a></p>
<h3><span class="mw-headline" id=".EC.B0.AC.EC.84.B1_.EC.9D.98.EA.B2.AC">찬성 의견</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%83%88%EB%A7%8C%EA%B8%88_%EA%B0%84%EC%B2%99_%EC%82%AC%EC%97%85&amp;action=edit&amp;section=13" title="부분 편집: 찬성 의견">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>새만금사업을 통해 국토확장, 대규모 우량농지 조성, 연간 10억 톤의 수자원 확보, 수해상습지 해소, 육운개선 및 종합관광권 형성 등의 효과를 기대할 수 있다.</p>
<ul>
<li>국토확장</li>
</ul>
<dl>
<dd>새만금사업이 완료되면 한국은 <a href="/wiki/%EC%97%AC%EC%9D%98%EB%8F%84" title="여의도">여의도</a> 면적의 140배에 달하는 새로운 땅을 얻게 된다.</dd>
</dl>
<ul>
<li>대규모 우량 농지 조성</li>
</ul>
<dl>
<dd>새만금사업을 통해 생겨난 비옥한 토지에는 식량작물 이외에도 필요에 따라 각종 <a href="/wiki/%EC%9B%90%EC%98%88" title="원예">원예</a> 및 사료 작물 등을 다양하게 재배할 수 있다.</dd>
</dl>
<ul>
<li>수자원 확보</li>
</ul>
<dl>
<dd>새만금사업이 완료되면 중규모 저수지 200개의 수량에 해당하는 10억 톤의 수자원을 확보할 수 있어 미래의 물 부족사태에 대비할 수 있다.</dd>
</dl>
<ul>
<li>수해상습지 해소</li>
</ul>
<dl>
<dd>새만금지구의 <a href="/wiki/%EB%A7%8C%EA%B2%BD%EA%B0%95" title="만경강">만경강</a>과 유역은 매년 12000ha에 달하는 상습적인 침수피해를 겪고 있다. 새만금방조제는 이러한 상습침수피해를 예방하는 기능을 할 것이며, 매년 4000억 원 이상의 예산을 절감하는 효과를 가져올 것이다.</dd>
</dl>
<ul>
<li>육운개선 및 종합관광권 형성</li>
</ul>
<dl>
<dd>방조제가 완공되면 군산~부안 간 교통거리가 단축되고, 육지와 섬지역이 연결되어 교통환경이 개선되고, 또한 새만금사업으로 생겨나는 드넓은 토지와 호수는 변산국립공원 등 천혜의 관광자원과 어우러져 세계적인 관광권을 형성, 지역경제 활성화에 기여할 것이다.</dd>
</dl>
<h3><span class="mw-headline" id=".EB.B0.98.EB.8C.80_.EC.9D.98.EA.B2.AC">반대 의견</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%83%88%EB%A7%8C%EA%B8%88_%EA%B0%84%EC%B2%99_%EC%82%AC%EC%97%85&amp;action=edit&amp;section=14" title="부분 편집: 반대 의견">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<ul>
<li>국민의 혈세 낭비</li>
</ul>
<dl>
<dd>새만금의 <a href="/wiki/%EB%B0%A9%EC%A1%B0%EC%A0%9C" title="방조제">방조제</a> 60%를 만드는 데에 들어간 돈만도 1조 250억 원이다. 또한 앞으로 방조제 완공과 내부개발, 새만금호 유입수의 수질오염방지와 관리를 위해 돈이 얼마나 더 들지 예측하기조차 어렵다.</dd>
</dl>
<ul>
<li>주민들의 삶의 터전 파괴</li>
</ul>
<dl>
<dd>새만금 사업으로 바다 물길이 막히면서 <a class="mw-redirect" href="/wiki/%EA%B0%AF%EB%B2%8C" title="갯벌">갯벌</a>이 썩기 시작했고, 고기가 잡히지 않아 어민의 생존이 위협당하고 있으며, 지역 경제는 바닥을 기고 있다. 이로 인해 주민들이 점점 바다 일을 포기하고 마을을 떠나 공사판을 떠돌고 있다.</dd>
</dl>
<ul>
<li>생물종 보존 문제</li>
</ul>
<dl>
<dd>새만금 갯벌은 한반도 전체 갯벌의 10%를 차지하고 있다. 이 갯벌에는 눈에 보이지 않는 미세한 <a href="/wiki/%ED%94%8C%EB%9E%91%ED%81%AC%ED%86%A4" title="플랑크톤">플랑크톤</a>에서부터 각종 조개류, 게 등의 <a href="/wiki/%EC%A0%80%EC%84%9C%EC%83%9D%EB%AC%BC" title="저서생물">저서생물</a>들이 살아가고 있다. 이 갯벌이 없어지면 다양한 생물종이 살아가는 터전이 없어지며, 결국 <a href="/wiki/%EC%83%9D%ED%83%9C%EA%B3%84" title="생태계">생태계</a>의 <a class="mw-redirect" href="/wiki/%EB%A8%B9%EC%9D%B4%EC%82%AC%EC%8A%AC" title="먹이사슬">먹이사슬</a>이 끊어지게 되고, 어족 자원은 줄어들어 그 영향은 서해안 전체에 미친다. 따라서 서해의 황금어장이 궤멸될 위험이 있다. 벌써 방조제 밖의 어획량이 현저히 줄어들었다고 한다.</dd>
<dd>또한 새만금 지역은 한반도 최대의 철새도래지이다. 새만금 갯벌이 사라지면 <a class="mw-redirect" href="/wiki/%EB%8F%84%EC%9A%94%EC%83%88" title="도요새">도요새</a>와 같은 새들이 멸종될 수도 있다고 한다.</dd>
<dd><a href="/wiki/2011%EB%85%84" title="2011년">2011년</a> 3월에는 멸종위기종인 <a href="/wiki/%EC%83%81%EA%B4%AD%EC%9D%B4" title="상괭이">상괭이</a> 230여 마리가 집단 질식사된 채로 발견되어 환경 오염과 생태계 악화가 급속히 진행되고 있음을 알 수 있다.<sup class="reference" id="cite_ref-9"><a href="#cite_note-9">[9]</a></sup></dd>
</dl>
<ul>
<li>수질 관리 문제</li>
</ul>
<dl>
<dd>새만금호 수질은, 해제될 예정인 전주권 개발제한구역(<a class="mw-redirect" href="/wiki/%EA%B7%B8%EB%A6%B0%EB%B2%A8%ED%8A%B8" title="그린벨트">그린벨트</a>)을 녹지로 묶고 <a class="new" href="/w/index.php?title=%EC%98%A4%EC%97%BC%EC%B4%9D%EB%9F%89%EA%B4%80%EB%A6%AC%EC%A0%9C%EB%8F%84&amp;action=edit&amp;redlink=1" title="오염총량관리제도 (없는 문서)">오염총량관리제도</a>를 도입하는 등 대책을 세워 더 이상 상류지역의 개발을 억제하지 않는다면, 농업용수 환경기준인 4급수를 달성하지 못하는 것으로 밝혀졌다.</dd>
</dl>
<ul>
<li>환경 파괴 사업</li>
</ul>
<dl>
<dd>새만금 갯벌 간척은 갯벌뿐만 아니라 <a href="/wiki/%EA%B3%A0%EA%B5%B0%EC%82%B0%EA%B5%B0%EB%8F%84" title="고군산군도">고군산군도</a>의 섬들과 주변 해창석산 등 산까지 파괴하고 있는 종합적인 환경파괴사업이다. 특히 옆의 <a class="new" href="/w/index.php?title=%EB%B3%80%EC%82%B0%ED%95%B4%EC%88%98%EC%9A%95%EC%9E%A5&amp;action=edit&amp;redlink=1" title="변산해수욕장 (없는 문서)">변산해수욕장</a>은 새만금이 주는 변화로 수질 오염이 심각하다.<sup class="reference" id="cite_ref-10"><a href="#cite_note-10">[10]</a></sup></dd>
</dl>
<h3><span class="mw-headline" id=".ED.99.98.EA.B2.BD_.EB.AC.B8.EC.A0.9C">환경 문제</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%83%88%EB%A7%8C%EA%B8%88_%EA%B0%84%EC%B2%99_%EC%82%AC%EC%97%85&amp;action=edit&amp;section=15" title="부분 편집: 환경 문제">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p><a class="mw-redirect" href="/wiki/%EC%83%88%EB%A7%8C%EA%B8%88_%EB%B0%A9%EC%A1%B0%EC%A0%9C" title="새만금 방조제">새만금 방조제</a>의 물막이 공사가 진행된 이후, 이로 인한 생태계 파괴의 우려는 현재까지 계속 제기되고 있다.<sup class="reference" id="cite_ref-11"><a href="#cite_note-11">[11]</a></sup> 새만금 방조제 물막이 공사 후 이로 인해 형성된 인공호수인 <a class="mw-redirect" href="/wiki/%EC%83%88%EB%A7%8C%EA%B8%88%ED%98%B8" title="새만금호">새만금호</a>의 연간 종다양성지수는 1.11로, 물막이 공사 중의 2.47에서 낮아져 새만금호의 어류 서식처의 변화가 종조성(種造成)의 변화에 상당한 영향을 미쳤음이 확인되었다. 물막이 공사 후 새만금호의 종다양성지수가 낮아진 것은 내만성 어류인 <a href="/wiki/%EC%A0%84%EC%96%B4" title="전어">전어</a>의 양이 크게 증가한 이유가 크고,<sup class="reference" id="cite_ref-12"><a href="#cite_note-12">[12]</a></sup> 그 밖에 일시적으로 새만금호에 몰려온 멸치나 삼치와 같은 특정 부어류들이 조사 때 기선저인망(otter trawl)에 대량 채집되어 소수종의 우점도가 높아진 것으로 분석되고 있지만, 물막이공사가 종다양성을 해쳤음을 부인하기는 어렵다.<sup class="reference" id="cite_ref-13"><a href="#cite_note-13">[13]</a></sup></p>
<p>일반적으로 예견된 바 대로, 물막이 공사 이후 염도 변화 등으로 인해 해산어(바다물고기)의 비율이 줄고, 점차 소하어 및 강하어를 포함한 주연성 어류가 증가하였고, 담수어(민물고기)도 꽤 늘어 새만금호가 해수 생태계에서 담수 생태계로 변화 중임이 확인되었다. '금강 및 새만금지역 조류변화상 연구'에 따르면 채집된 어류 총 14과35종 중 1차 담수어가 10종, 소하어 및 강하어를 포함하는 주연성 어류가 23종인데 반해, 해산어는 2종에 불과했으며, <a class="mw-redirect" href="/wiki/%EC%83%88%EB%A7%8C%EA%B8%88_%EB%B0%A9%EC%A1%B0%EC%A0%9C" title="새만금 방조제">새만금 방조제</a> 건설 전과 상대적인 어종의 구성을 비교해 볼 때 해산어의 비율이 크게 줄고 주연성 어류의 비중이 대폭 늘어났다. 해당 보고서는 향후 담수화가 진행될수록 서식어종의 분포는 해산어류에서 주연성 어류, 주연성 어류에서 순수 담수어류로 수중 생태환경이 변화되어 갈 것으로 예측했다.<sup class="reference" id="cite_ref-14"><a href="#cite_note-14">[14]</a></sup></p>
<h2><span class="mw-headline" id=".EC.83.88.EB.A7.8C.EA.B8.88_.EC.8B.A0.EA.B3.B5.ED.95.AD_.EB.85.BC.EB.9E.80">새만금 신공항 논란</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%83%88%EB%A7%8C%EA%B8%88_%EA%B0%84%EC%B2%99_%EC%82%AC%EC%97%85&amp;action=edit&amp;section=16" title="부분 편집: 새만금 신공항 논란">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>전북권을 대표하는 <a href="/wiki/%EA%B5%B0%EC%82%B0%EA%B3%B5%ED%95%AD" title="군산공항">군산공항</a>은 주한 <a href="/wiki/%EB%AF%B8%EA%B5%AD_%EA%B3%B5%EA%B5%B0" title="미국 공군">미국 공군</a>의 소유여서 국제선 취항이 금지되어 있어 신공항을 건설해야 한다는 주장이 있다. 옛 <a href="/wiki/%EA%B9%80%EC%A0%9C%EA%B3%B5%ED%95%AD" title="김제공항">김제공항</a> 부지, 새만금사업의 일부로 조성 중인 <a href="/wiki/%EB%A7%8C%EA%B2%BD%EC%9D%8D" title="만경읍">만경읍</a>-<a href="/wiki/%EC%A7%84%EB%B4%89%EB%A9%B4" title="진봉면">진봉면</a>부근의 매립지(만경화포지구), 새만금지구 세 곳을 대상으로 타당성을 조사하고 있다.</p>
<h2><span class="mw-headline" id=".EC.B2.AD.EC.99.80.EB.8C.80_.EB.B9.84.EC.84.9C.EA.B4.80_.EB.B0.8F_.ED.96.89.EC.A0.95.EA.B4.80_.EC.83.88.EB.A7.8C.EA.B8.88_.EC.86.8C.EB.B0.A9_.ED.97.AC.EA.B8.B0_.EC.8B.9C.EC.B0.B0_.EB.85.BC.EB.9E.80">청와대 비서관 및 행정관 새만금 소방 헬기 시찰 논란</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%83%88%EB%A7%8C%EA%B8%88_%EA%B0%84%EC%B2%99_%EC%82%AC%EC%97%85&amp;action=edit&amp;section=17" title="부분 편집: 청와대 비서관 및 행정관 새만금 소방 헬기 시찰 논란">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>2003년 6월 6일 <a class="new" href="/w/index.php?title=%EC%A0%95%EB%AA%85%EC%B1%84&amp;action=edit&amp;redlink=1" title="정명채 (없는 문서)">정명채</a> 농어촌TF팀장(1급), <a href="/wiki/%EB%B0%95%ED%83%9C%EC%A3%BC" title="박태주">박태주</a> 노동개혁TF팀장(2급), <a class="new" href="/w/index.php?title=%EC%A1%B0%EC%9E%AC%ED%9D%AC&amp;action=edit&amp;redlink=1" title="조재희 (없는 문서)">조재희</a> 정책관리비서관(1급)을 비롯해 행정관 5명, 농어촌 대책 TF 소속 직원 2명 등 11명이 새만금 시찰에 참가해 <a href="/wiki/%EC%A0%84%EB%9D%BC%EB%B6%81%EB%8F%84_%EC%86%8C%EB%B0%A9%EB%B3%B8%EB%B6%80" title="전라북도 소방본부">전라북도 소방본부</a> 소속 소방헬기를 동원해 헬기에 부인과 자녀 등이 헬기에 동반 탑승해 논란이 일었다.<sup class="reference" id="cite_ref-15"><a href="#cite_note-15">[15]</a></sup> 이에 6월 25일 <a href="/wiki/%EB%85%B8%EB%AC%B4%ED%98%84" title="노무현">노무현</a> 대통령은 가족들을 동반해 소방 헬기를 이용해 물의를 빚은 <a class="new" href="/w/index.php?title=%EC%A0%95%EB%AA%85%EC%B1%84&amp;action=edit&amp;redlink=1" title="정명채 (없는 문서)">정명채</a>, <a href="/wiki/%EB%B0%95%ED%83%9C%EC%A3%BC" title="박태주">박태주</a>, <a class="new" href="/w/index.php?title=%EC%A1%B0%EC%9E%AC%ED%9D%AC&amp;action=edit&amp;redlink=1" title="조재희 (없는 문서)">조재희</a> 등 3인의 사표를 수리했다.<sup class="reference" id="cite_ref-16"><a href="#cite_note-16">[16]</a></sup></p>
<h2><span class="mw-headline" id=".EC.83.88.EB.A7.8C.EA.B8.88.EC.82.AC.EC.97.85_.EC.B4.89.EC.A7.84.EC.9D.84_.EC.9C.84.ED.95.9C_.ED.8A.B9.EB.B3.84.EB.B2.95">새만금사업 촉진을 위한 특별법</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%83%88%EB%A7%8C%EA%B8%88_%EA%B0%84%EC%B2%99_%EC%82%AC%EC%97%85&amp;action=edit&amp;section=18" title="부분 편집: 새만금사업 촉진을 위한 특별법">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>'제1조(목적) 이 법은 새만금간척사업으로 조성되는 새만금사업지역을 농업을 기조로 하는 환경친화적인 개발, 이용 및 보전에 관한 사항을 규정하여 지속가능한 개발을 추진함으로써 지역균형발전과 국가경쟁력 강화에 이바지하는 것을 목적으로 한다.'라는 조항으로 시작되는 '새만금사업 촉진을 위한 특별법(법률 제8795호)'이 2007년 12월 27일 제정되었다. <a class="external text" href="http://likms.assembly.go.kr/law/jsp/Law.jsp?WORK_TYPE=LAW_BON&amp;LAW_ID=A2119&amp;PROM_NO=08795&amp;PROM_DT=20071227&amp;HanChk=Y" rel="nofollow">특별법 전문</a></p>
<h3><span class="mw-headline" id=".EC.83.88.EB.A7.8C.EA.B8.88_.ED.8A.B9.EB.B3.84.EB.B2.95.EC.9D.98_.ED.9A.A8.EA.B3.BC">새만금 특별법의 효과</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%83%88%EB%A7%8C%EA%B8%88_%EA%B0%84%EC%B2%99_%EC%82%AC%EC%97%85&amp;action=edit&amp;section=19" title="부분 편집: 새만금 특별법의 효과">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<ol>
<li>정부에 대한 신뢰가 회복되기 시작하였다. 개발 초기에는 개발 기간을 14년으로 잡았는데, 10년 이상이 지연되었다. 그러나 새만금 특별법으로 새만금 사업을 조기에 진할 수 있는 원동력이 되었다.</li>
<li>'새만금 내부토지 개발 기본구상'의 실천을 위한 제도적 장치가 되었다. 도시, 농업, 산업, 관광용지 등 개발 토지의 용도에 따라서 개발 주체별로 개발된다.</li>
<li>과다한 시간소요로 인한 추가적인 사업 지연이 방지된다. 개별법에 따른 규제와 절차의 특례와 의제처리로 인해서 시간이 단축되고, 인허가의 의제처리와 같은 법적 절차의 복잡하고 까다로운 과정들이 간소화되어 시간적, 경제적인 비용을 감축시킬 수 있다.</li>
</ol>

<h2><span class="mw-headline" id=".EC.83.88.EB.A7.8C.EA.B8.88.ED.99.8D.EB.B3.B4.EA.B4.80">새만금홍보관</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%83%88%EB%A7%8C%EA%B8%88_%EA%B0%84%EC%B2%99_%EC%82%AC%EC%97%85&amp;action=edit&amp;section=32" title="부분 편집: 새만금홍보관">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>새만금홍보관은 (구)전시관을 철거한 후 2010년에 착공하여 <a href="/wiki/2012%EB%85%84" title="2012년">2012년</a> <a href="/wiki/7%EC%9B%94_19%EC%9D%BC" title="7월 19일">7월 19일</a>에 개관하였다. 전북 부안군 변산면 새만금로 6에 소재하고 있으며, 13,784㎡의 부지에 지하 1층, 지상 3층, 연면적 3,610㎡의 규모이다.</p>
<p>종합홍보관 전시실에는 한국 간척기술의 발전사, 국토이용상의 현안문제, 상류유역수질 개선대책, 호소내 수질 및 환경친화적 개발, 우리나라 주요 철새 도래지, 새만금 지구, 모형 설명, 배수갑문 모형, 새만금 위성사진, 간척이후 형성된 새로운 갯벌, 새만금지구 시대별 간척지, 방조제 표준단면, 종합관광권 형성, 사업효과 등을 볼 수 있도록 전시되어 있다.<sup class="reference" id="cite_ref-19"><a href="#cite_note-19">[19]</a></sup> 관람시간은 오전 9시 ~ 오후 5시이고, 1월 1일과 매주 월요일은 휴관한다.</p>
