<p><b>생물</b>(生物)은 <a href="/wiki/%EC%83%9D%EB%AA%85" title="생명">생명</a>이 있는 것을 말하며, 보통 <a href="/wiki/%EB%8F%99%EB%AC%BC" title="동물">동물</a>과 <a href="/wiki/%EC%8B%9D%EB%AC%BC" title="식물">식물</a> 또는 <a href="/wiki/%EC%82%AC%EB%9E%8C" title="사람">사람</a> 등의 존재를 두루 일컫는다. <a href="/wiki/%EC%A7%80%EA%B5%AC" title="지구">지구</a> 위에 사는 모든 생물에게는 공통의 조상이 있으며, 그 자손들이 번식함으로써 <a href="/wiki/%EC%9C%A0%EC%A0%84%EC%9E%90" title="유전자">유전자</a>에 여러 변이가 생김으로써 <a href="/wiki/%EC%A7%84%ED%99%94" title="진화">진화</a>가 일어났다고 여겨진다. <a class="mw-redirect" href="/wiki/%EB%B0%95%ED%85%8C%EB%A6%AC%EC%95%84" title="박테리아">박테리아</a>부터 <a href="/wiki/%EC%82%AC%EB%9E%8C" title="사람">사람</a>에 이르기까지 다양한 생물이 존재하며, 복잡한 관계의 생명권을 이룬다. <b>생물체</b>, <b>생명체</b>, <b>유기체</b>라고도 한다.</p>

<h2><span class="mw-headline" id=".EC.83.9D.EB.AC.BC.EC.9D.98_.ED.8A.B9.EC.84.B1">생물의 특성</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%83%9D%EB%AC%BC&amp;action=edit&amp;section=1" title="부분 편집: 생물의 특성">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>생물이 <a class="new" href="/w/index.php?title=%EB%AC%B4%EC%83%9D%EB%AC%BC&amp;action=edit&amp;redlink=1" title="무생물 (없는 문서)">무생물</a>과 구별되는 일반적인 특징으로서, 생물은 자기증식능력, 에너지변환능력, 항상성 유지능력이라고 하는 3가지의 능력을 가지고 있다.</p>
<p>1. <a href="/wiki/%EB%AC%BC%EC%A7%88%EB%8C%80%EC%82%AC" title="물질대사">물질대사</a>를 한다.</p>
<ul>
<li>체내에 필요한 물질이 합성, 분해되는 화학반응</li>
<li><a href="/wiki/%EC%97%90%EB%84%88%EC%A7%80" title="에너지">에너지</a>의 출입이 반드시 따름</li>
<li><a href="/wiki/%ED%9A%A8%EC%86%8C" title="효소">효소</a>가 반응을 매개함.</li>
</ul>
<p>2. <a class="new" href="/w/index.php?title=%EC%9E%90%EA%B7%B9%EA%B3%BC_%EB%B0%98%EC%9D%91&amp;action=edit&amp;redlink=1" title="자극과 반응 (없는 문서)">자극과 반응</a></p>
<ul>
<li>생존을 위해 필수적인 것으로 환경의 변화(<a class="new" href="/w/index.php?title=%EC%9E%90%EA%B7%B9&amp;action=edit&amp;redlink=1" title="자극 (없는 문서)">자극</a>)를 감지하고 반응함</li>
</ul>
<p>3. 항상성 유지</p>
<ul>
<li>외부 환경이 변하더라도 체내 환경을 일정하게 유지하려는 성질을 가진다.</li>
</ul>
<dl>
<dt><a href="/wiki/%EC%B2%B4%EC%98%A8" title="체온">체온</a>, <a class="mw-redirect" href="/wiki/%ED%98%88%EB%8B%B9%EB%9F%89" title="혈당량">혈당량</a>, <a href="/wiki/%EC%B2%B4%EC%95%A1" title="체액">체액</a> 농도 등을 유지</dt>
</dl>
<p>4. <a href="/wiki/%EC%83%9D%EC%8B%9D" title="생식">생식</a>과 <a href="/wiki/%EC%9C%A0%EC%A0%84" title="유전">유전</a></p>
<ul>
<li>종족 유지위해 자신과 닮은 자손을 남기는 것.</li>
</ul>
<p>5. <a class="mw-disambig" href="/wiki/%EB%B0%9C%EC%83%9D" title="발생">발생</a>과 <a href="/wiki/%EC%83%9D%EC%9E%A5" title="생장">생장</a></p>
<ul>
<li>세포가 커지거나, <a href="/wiki/%EC%84%B8%ED%8F%AC_%EB%B6%84%EC%97%B4" title="세포 분열">세포 분열</a>에 의해 수가 늘어나며 자람.</li>
</ul>
<p>6. 적응과 <a href="/wiki/%EC%A7%84%ED%99%94" title="진화">진화</a></p>
<ul>
<li>현재의 환경에 적응되고, 환경이 변함에 따라 변화할 수 있음.</li>
</ul>
<p>진화론에서는 변화된 형질이 환경에 유리하여 살아남게 되면, 새로운 종으로 된다고 주장.</p>
<p>7. 정교하고 복잡한 체제</p>
<ul>
<li>모든 생물은 <a href="/wiki/%EC%84%B8%ED%8F%AC" title="세포">세포</a>로 구성</li>
</ul>
<h2><span class="mw-headline" id=".EC.83.9D.EB.AC.BC_.EB.B6.84.EB.A5.98">생물 분류</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%83%9D%EB%AC%BC&amp;action=edit&amp;section=2" title="부분 편집: 생물 분류">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>유기체의 <a href="/wiki/%EC%83%9D%EB%AC%BC_%EB%B6%84%EB%A5%98" title="생물 분류">생물 분류</a>는 다음과 같다.</p>
<ul>
<li><a href="/wiki/%EC%84%B8%EA%B7%A0" title="세균">세균</a> (Bacteria)</li>
<li><a href="/wiki/%EA%B3%A0%EC%84%B8%EA%B7%A0" title="고세균">고세균</a> (Archaea)</li>
<li><a href="/wiki/%EC%A7%84%ED%95%B5%EC%83%9D%EB%AC%BC" title="진핵생물">진핵생물</a> (Eukarya)
<ul>
<li><a href="/wiki/%EC%9B%90%EC%83%9D%EC%83%9D%EB%AC%BC" title="원생생물">원생생물계</a> (Protista)</li>
<li><a class="mw-redirect" href="/wiki/%EA%B7%A0%EB%A5%98" title="균류">균계</a> (Fungus)</li>

</ul>
</li>

<h2><span class="mw-headline" id=".EA.B5.AC.EC.84.B1_.EB.AC.BC.EC.A7.88">구성 물질</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%83%9D%EB%AC%BC&amp;action=edit&amp;section=3" title="부분 편집: 구성 물질">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p><a href="/wiki/%EB%AC%BC" title="물">물</a>, <a href="/wiki/%EB%8B%A8%EB%B0%B1%EC%A7%88" title="단백질">단백질</a>, <a href="/wiki/%EC%A7%80%EB%B0%A9" title="지방">지방질</a>, <a href="/wiki/%ED%83%84%EC%88%98%ED%99%94%EB%AC%BC" title="탄수화물">탄수화물</a>, <a href="/wiki/%ED%95%B5%EC%82%B0" title="핵산">핵산</a>은 생물을 이루는 주성분이다. 그 가운데 화학 반응이 일어나는 장소를 제공하는 것이 물이며, 지극히 중요한 성분이다.</p>
<p>생물을 복잡하게 만드는 물질 가운데 하나가 <a href="/wiki/%EB%8B%A8%EB%B0%B1%EC%A7%88" title="단백질">단백질</a>이다. 단백질은 20가지 <a href="/wiki/%EC%95%84%EB%AF%B8%EB%85%B8%EC%82%B0" title="아미노산">아미노산</a>이 수십~수백 개 결합한 것이지만, 이러한 <a href="/wiki/%EC%88%9C%EC%97%B4" title="순열">순열</a> 편성에 따라 그 종류는 몇 천만까지 이른다. 어느 단백질은 화학 반응을 <a href="/wiki/%EC%B4%89%EB%A7%A4" title="촉매">촉진</a>하는 <a href="/wiki/%ED%9A%A8%EC%86%8C" title="효소">효소</a>로서 어느 생물의 구조를 지지하는 골격으로 작용하며 여러 기능을 한다.</p>
<p><a href="/wiki/%EB%A1%9C%EB%B2%84%ED%8A%B8_%ED%9B%85" title="로버트 훅">로버트 훅</a>이 처음 <a href="/wiki/%EC%84%B8%ED%8F%AC" title="세포">세포</a>를 발견했을 때 이를 작은 방(셀, cell)으로 이름을 붙인 것처럼, 세포는 경계가 진 공간이며, 바깥 세계로부터 격리시킴으로써 생물을 성립한다. 이를 나누는 것을 <a href="/wiki/%EC%84%B8%ED%8F%AC%EB%A7%89" title="세포막">세포막</a>이라고 하며, <a href="/wiki/%EC%A7%80%EB%B0%A9" title="지방">지방질</a>이 그에 대한 주성분이다. 지방질은 <a href="/wiki/%EC%97%90%EB%84%88%EC%A7%80" title="에너지">에너지</a>를 저장하기에 좋은 물질이다.</p>
<p>생물은 경계가 있는 격리된 공간이지만, 완전하게 바깥 세계로부터 차단된 것은 아니다. 바깥으로부터 <a href="/wiki/%EC%97%90%EB%84%88%EC%A7%80" title="에너지">에너지</a>를 끌어와서 내부에서 소비하기도 한다. 생물 사이의 에너지 유통에 있어 <a href="/wiki/%ED%83%84%EC%88%98%ED%99%94%EB%AC%BC" title="탄수화물">탄수화물</a>은 중요하고, 주로 <a href="/wiki/%EC%8B%9D%EB%AC%BC" title="식물">식물</a>이 <a href="/wiki/%EA%B4%91%ED%95%A9%EC%84%B1" title="광합성">광합성</a>을 함으로써 이를 생산한다.</p>
<p><a href="/wiki/2010%EB%85%84" title="2010년">2010년</a> <a href="/wiki/12%EC%9B%94_2%EC%9D%BC" title="12월 2일">12월 2일</a> <a href="/wiki/%EC%9D%B8" title="인">인</a> 대신 <a href="/wiki/%EB%B9%84%EC%86%8C" title="비소">비소</a>를 사용하는 생물인 <a href="/wiki/GFAJ-1" title="GFAJ-1">GFAJ-1</a>의 존재를 <a class="mw-redirect" href="/wiki/NASA" title="NASA">NASA</a>에서 공식 발표하였다.</p>

<div class="rellink relarticle mainarticle"><img alt="" data-file-height="128" data-file-width="128" height="16" src="//upload.wikimedia.org/wikipedia/commons/thumb/e/ec/Crystal_Clear_app_xmag.svg/16px-Crystal_Clear_app_xmag.svg.png" srcset="//upload.wikimedia.org/wikipedia/commons/thumb/e/ec/Crystal_Clear_app_xmag.svg/24px-Crystal_Clear_app_xmag.svg.png 1.5x, //upload.wikimedia.org/wikipedia/commons/thumb/e/ec/Crystal_Clear_app_xmag.svg/32px-Crystal_Clear_app_xmag.svg.png 2x" width="16"/> 이 부분의 본문은 <a href="/wiki/%EC%A7%80%EC%A7%88_%EC%8B%9C%EB%8C%80" title="지질 시대">지질 시대</a>입니다.</div>

<li>6억 년 전 - <a class="mw-redirect" href="/wiki/%EC%BA%84%EB%B8%8C%EB%A6%AC%EC%95%84%EA%B8%B0%EC%9D%98_%EB%8C%80%ED%8F%AD%EB%B0%9C" title="캄브리아기의 대폭발">캄브리아</a>가 폭발한다. (→<a class="new" href="/w/index.php?title=%EB%B0%94%EC%A0%9C%EC%8A%A4_%EB%8F%99%EB%AC%BC%EA%B5%B0&amp;action=edit&amp;redlink=1" title="바제스 동물군 (없는 문서)">바제스 동물군</a>)</li>
<li>5억 년 전 - <a class="mw-redirect" href="/wiki/%EC%96%B4%EB%A5%98" title="어류">어류</a>가 출현한다. 식물과 절족 동물이 육지로 올라온다.</li>
<li>4억 년 전 - <a href="/wiki/%EC%96%91%EC%84%9C%EB%A5%98" title="양서류">양서류</a>가 육지로 올라온다.</li>
<li>3억 년 전 - <a href="/wiki/%EA%B3%B5%EB%A3%A1" title="공룡">공룡</a> 시대가 시작한다.</li>
<li>2억 년 전 - <a href="/wiki/%ED%8F%AC%EC%9C%A0%EB%A5%98" title="포유류">포유류</a>, <a href="/wiki/%EC%83%88" title="새">조류</a>가 출현한다.</li>
<li>6500만 년 전 - <a href="/wiki/%EA%B3%B5%EB%A3%A1" title="공룡">공룡</a>이 멸종한다.</li>
<li>400만 년 전 - <a href="/wiki/%EC%98%A4%EC%8A%A4%ED%8A%B8%EB%9E%84%EB%A1%9C%ED%94%BC%ED%85%8C%EC%BF%A0%EC%8A%A4" title="오스트랄로피테쿠스">오스트랄로피테쿠스</a>가 출현한다.</li>
<li>160만 년 전 - <a href="/wiki/%ED%98%B8%EB%AA%A8_%EC%97%90%EB%A0%89%ED%88%AC%EC%8A%A4" title="호모 에렉투스">호모 에렉투스</a>가 출현한다.</li>
<li>30만 년 전 - <a href="/wiki/%EC%82%AC%EB%9E%8C" title="사람">호모 사피엔스</a>가 출현한다.</li>
</ul>
