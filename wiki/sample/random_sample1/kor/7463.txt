<p><b>생식 기관</b>(生殖 器官)은 <a href="/wiki/%EC%83%9D%EC%8B%9D" title="생식">생식</a>을 목적으로 하는 <a href="/wiki/%EC%83%9D%EB%AC%BC" title="생물">생물</a>의 <a class="mw-redirect" href="/wiki/%EA%B8%B0%EA%B4%80_(%EC%83%9D%EB%AC%BC)" title="기관 (생물)">기관</a>이다. 생식에는 생식기관 외에도 <a href="/wiki/%EC%B2%B4%EC%95%A1" title="체액">체액</a>, <a href="/wiki/%ED%98%B8%EB%A5%B4%EB%AA%AC" title="호르몬">호르몬</a>, <a href="/wiki/%ED%8E%98%EB%A1%9C%EB%AA%AC" title="페로몬">페로몬</a>과 같은 물질들이 관여한다.<sup class="reference" id="cite_ref-1"><a href="#cite_note-1">[1]</a></sup> 생식 기관은 대다수의 다른 기관과 달리 <a href="/wiki/%EC%84%B1%EB%B3%84" title="성별">성별</a>에 따라 구조가 다르다.<sup class="reference" id="cite_ref-body_guide_2-0"><a href="#cite_note-body_guide-2">[2]</a></sup></p>
<p><a class="mw-redirect" href="/wiki/%EC%9D%B8%EA%B0%84" title="인간">인간</a>의 생식 기관은 <a href="/wiki/%EC%9D%8C%EA%B2%BD" title="음경">음경</a>과 <a href="/wiki/%EC%A7%88" title="질">질</a> 같은 <a class="mw-redirect" href="/wiki/%EC%83%9D%EC%8B%9D%EA%B8%B0" title="생식기">생식기</a>와 <a class="mw-disambig" href="/wiki/%EC%A0%95%EC%9E%90" title="정자">정자</a> <a href="/wiki/%EB%82%9C%EC%9E%90" title="난자">난자</a>와 같은 <a class="mw-redirect" href="/wiki/%EC%83%9D%EC%8B%9D_%EC%84%B8%ED%8F%AC" title="생식 세포">생식 세포</a>를 만들어 내는 <a href="/wiki/%EA%B3%A0%ED%99%98" title="고환">고환</a> <a href="/wiki/%EB%82%9C%EC%86%8C" title="난소">난소</a> 등의 내부 생식 기관인 <a class="mw-redirect" href="/wiki/%EC%83%9D%EC%8B%9D%EC%86%8C" title="생식소">생식소</a>로 이루어져 있다. 생식기관은 각종 <a href="/wiki/%EA%B0%90%EC%97%BC%EB%B3%91" title="감염병">감염병</a>과 <a href="/wiki/%EC%84%B1%EB%B3%91" title="성병">성병</a>에 의해 병에 걸릴 수 있다.<sup class="reference" id="cite_ref-3"><a href="#cite_note-3">[3]</a></sup></p>
<p>대부분의 척추 동물은 정소, 생식기 등을 포함하는 생식기관을 갖고 있으며 <a href="/wiki/%EC%A7%84%ED%99%94" title="진화">진화</a>에 따른 적응의 결과 매우 다양한 구조를 보인다.</p>

<h2><span class="mw-headline" id=".EC.9D.B8.EA.B0.84.EC.9D.98_.EC.83.9D.EC.8B.9D">인간의 생식</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%83%9D%EC%8B%9D%EA%B3%84%ED%86%B5&amp;action=edit&amp;section=1" title="부분 편집: 인간의 생식">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p><a class="mw-redirect" href="/wiki/%EC%9D%B8%EA%B0%84" title="인간">인간</a>의 <a href="/wiki/%EC%83%9D%EC%8B%9D" title="생식">생식</a>은 <a href="/wiki/%EC%84%B1%EA%B5%90" title="성교">성교</a>에 의한 체내 수정으로 이루어진다. 이 과정에서 <a href="/wiki/%EB%82%A8%EC%84%B1" title="남성">남성</a>의 <a href="/wiki/%EB%B0%9C%EA%B8%B0" title="발기">발기</a>된 <a href="/wiki/%EC%9D%8C%EA%B2%BD" title="음경">음경</a>이 <a href="/wiki/%EC%97%AC%EC%84%B1" title="여성">여성</a>의 <a href="/wiki/%EC%A7%88" title="질">질</a> 안으로 삽입되어 <a class="mw-disambig" href="/wiki/%EC%A0%95%EC%9E%90" title="정자">정자</a>가 포함된 <a href="/wiki/%EC%A0%95%EC%95%A1" title="정액">정액</a>을 <a href="/wiki/%EC%82%AC%EC%A0%95_(%EC%83%9D%EB%A6%AC%ED%95%99)" title="사정 (생리학)">사정</a>한다. 정액은 <a href="/wiki/%EC%A7%88" title="질">질</a>과 <a href="/wiki/%EC%9E%90%EA%B6%81%EA%B2%BD%EB%B6%80" title="자궁경부">자궁경부</a>를 거쳐 <a href="/wiki/%EC%9E%90%EA%B6%81" title="자궁">자궁</a>이나 <a href="/wiki/%EB%82%98%ED%8C%94%EA%B4%80" title="나팔관">나팔관</a>에 도달하며 <a href="/wiki/%EB%82%9C%EC%9E%90" title="난자">난자</a>와 만나 <a href="/wiki/%EC%88%98%EC%A0%95_(%EC%83%9D%EB%AC%BC%ED%95%99)" title="수정 (생물학)">수정</a>이 이루어진다. 수정과 <a href="/wiki/%EC%B0%A9%EC%83%81" title="착상">착상</a>에 성공한 <a class="mw-disambig" href="/wiki/%EB%B0%B0%EC%95%84" title="배아">배아</a>는 <a href="/wiki/%ED%83%9C%EC%95%84" title="태아">태아</a>가 되어 자궁에서 자라며 이를 <a href="/wiki/%EC%9E%84%EC%8B%A0" title="임신">임신</a>이라 한다. 인간의 임신 기간은 평균적으로 40주 정도이다. 임신 기간이 지나면 <a class="mw-redirect" href="/wiki/%EC%B6%9C%EC%82%B0" title="출산">출산</a>을 거쳐 <a href="/wiki/%EC%95%84%EA%B8%B0" title="아기">아기</a>가 태어난다. 인간의 아이는 매우 연약하여 수 년간의 육아가 필요하다. 특히 갓 태어난 영아는 생후 일정 기간 동안 <a class="mw-disambig" href="/wiki/%EC%88%98%EC%9C%A0" title="수유">수유</a> 받아야 한다.<sup class="reference" id="cite_ref-4"><a href="#cite_note-4">[4]</a></sup></p>
<p>인간의 생식 기관은 성별에 따라 뚜렷이 구분되며 <a class="new" href="/w/index.php?title=2%EC%B0%A8_%EC%84%B1%EC%A7%95&amp;action=edit&amp;redlink=1" title="2차 성징 (없는 문서)">2차 성징</a>이 나타난 후에는 신체의 모든 부분에서 남녀를 분명히 구분할 수 있다.</p>
<h3><span class="mw-headline" id=".EB.82.A8.EC.84.B1.EC.9D.98_.EC.83.9D.EC.8B.9D_.EA.B8.B0.EA.B4.80">남성의 생식 기관</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%83%9D%EC%8B%9D%EA%B3%84%ED%86%B5&amp;action=edit&amp;section=2" title="부분 편집: 남성의 생식 기관">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>남성의 생식 기관은 신체 하부에 몸 밖으로 돌출해 있다. 남성의 생식 기관은 크게 세 부분으로 구분할 수 있다. 첫 번째 부위는 <a href="/wiki/%EC%9D%8C%EB%82%AD" title="음낭">음낭</a>으로 남성의 <a class="mw-redirect" href="/wiki/%EC%A0%95%EC%86%8C" title="정소">정소</a>로 <a href="/wiki/%EC%83%9D%EC%8B%9D%EC%84%B8%ED%8F%AC" title="생식세포">생식세포</a>인 <a class="mw-disambig" href="/wiki/%EC%A0%95%EC%9E%90" title="정자">정자</a>를 만드는 <a href="/wiki/%EA%B3%A0%ED%99%98" title="고환">고환</a>과 정자를 보관하고 이동시키는 <a href="/wiki/%EB%B6%80%EA%B3%A0%ED%99%98" title="부고환">부고환</a>이 <a href="/wiki/%EC%9D%8C%EB%82%AD" title="음낭">음낭</a>에 둘러싸여 있다. 두 번째 부위는 몸 안에 자리잡고 있는 <a href="/wiki/%EC%A0%95%EB%82%AD" title="정낭">정낭</a>, <a href="/wiki/%EC%88%98%EC%A0%95%EA%B4%80" title="수정관">수정관</a>, <a class="mw-redirect" href="/wiki/%EC%A0%84%EB%A6%BD%EC%84%A0" title="전립선">전립선</a>과 같은 기관들로 <a href="/wiki/%EC%A0%95%EC%95%A1" title="정액">정액</a>의 생성, 보관, 운반에 관계한다. 세 번째 부위는 <a href="/wiki/%EC%84%B1%EA%B5%90" title="성교">성교</a>와 <a href="/wiki/%EC%82%AC%EC%A0%95_(%EC%83%9D%EB%A6%AC%ED%95%99)" title="사정 (생리학)">사정</a>에 관계된 기관들로 <a href="/wiki/%EC%9D%8C%EA%B2%BD" title="음경">음경</a>, <a href="/wiki/%EC%9A%94%EB%8F%84" title="요도">요도</a>, <a href="/wiki/%EC%88%98%EC%A0%95%EA%B4%80" title="수정관">수정관</a>, <a class="mw-redirect" href="/wiki/%EC%BF%A0%ED%8D%BC_%EC%83%98" title="쿠퍼 샘">쿠퍼 샘</a> 등이 있다.<sup class="reference" id="cite_ref-body_guide_2-1"><a href="#cite_note-body_guide-2">[2]</a></sup></p>
<p>남성은 2차 성징이 일어나면 <a href="/wiki/%EA%B7%BC%EC%9C%A1" title="근육">근육</a>이 발달하고 <a class="mw-redirect" href="/wiki/%EB%AA%A9%EC%86%8C%EB%A6%AC" title="목소리">목소리</a>가 굵어지며, 얼굴에 <a href="/wiki/%EC%88%98%EC%97%BC" title="수염">수염</a>이 나고 몸에는 <a href="/wiki/%ED%84%B8" title="털">털</a>이 굵어진다. 또한 <a href="/wiki/%EC%96%B4%EA%B9%A8" title="어깨">어깨</a>가 벌어지고 목에는 <a href="/wiki/%ED%9B%84%EB%91%90%EC%9C%B5%EA%B8%B0" title="후두융기">후두융기</a>가 생긴다. 중요한 남성 <a href="/wiki/%ED%98%B8%EB%A5%B4%EB%AA%AC" title="호르몬">호르몬</a>으로는 <a href="/wiki/%EC%95%88%EB%93%9C%EB%A1%9C%EA%B2%90" title="안드로겐">안드로겐</a>과 <a href="/wiki/%ED%85%8C%EC%8A%A4%ED%86%A0%EC%8A%A4%ED%85%8C%EB%A1%A0" title="테스토스테론">테스토스테론</a>이 있다.<sup class="reference" id="cite_ref-Kimball_5-0"><a href="#cite_note-Kimball-5">[5]</a></sup></p>
<h3><span class="mw-headline" id=".EC.97.AC.EC.84.B1.EC.9D.98_.EC.83.9D.EC.8B.9D_.EA.B8.B0.EA.B4.80">여성의 생식 기관</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%83%9D%EC%8B%9D%EA%B3%84%ED%86%B5&amp;action=edit&amp;section=3" title="부분 편집: 여성의 생식 기관">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>여성의 대부분의 생식기관은 사타구니의 질구를 중심으로 몸 안으로 밀려 들어가 있으며, 외부로는 <a href="/wiki/%EC%A7%88" title="질">질 개부</a>만이 드러나 있다. 여성의 생식 기관 역시 크게 세 부분으로 구분할 수 있는데 <a href="/wiki/%EC%A7%88" title="질">질</a>, <a href="/wiki/%EC%9E%90%EA%B6%81" title="자궁">자궁</a>, <a href="/wiki/%EB%82%9C%EC%86%8C" title="난소">난소</a>가 각 부위의 중심 기관이다. <a class="mw-redirect" href="/wiki/%EC%9C%A0%EB%B0%A9_(%EB%8F%99%EB%AC%BC)" title="유방 (동물)">유방</a>은 생식에는 직접적인 연관이 없으나 <a class="mw-redirect" href="/wiki/%EC%B6%9C%EC%82%B0" title="출산">출산</a> 이후 <a class="mw-disambig" href="/wiki/%EC%88%98%EC%9C%A0" title="수유">수유</a> 기관으로서 육아에 관여한다.</p>
<p><a href="/wiki/%EC%99%B8%EC%9D%8C%EB%B6%80" title="외음부">외음부</a>는 <a href="/wiki/%EC%A7%88" title="질">질</a>로 연결되며 <a href="/wiki/%EC%9D%8C%ED%95%B5" title="음핵">음핵</a>, <a href="/wiki/%EB%8C%80%EC%9D%8C%EC%88%9C" title="대음순">대음순</a>, <a href="/wiki/%EC%86%8C%EC%9D%8C%EC%88%9C" title="소음순">소음순</a>, <a href="/wiki/%EC%9A%94%EB%8F%84" title="요도">요도</a> 등으로 이루어져 있다. 또한 <a class="mw-redirect" href="/wiki/%EB%B0%94%EB%A5%B4%ED%86%A8%EB%A6%B0_%EC%84%A0" title="바르톨린 선">바르톨린 선</a>이 있어 성교시 윤활액(질액: 애액은 틀린 표현임)이 분비된다. <a href="/wiki/%EC%A7%88" title="질">질</a>은 <a class="mw-redirect" href="/wiki/%EC%9E%90%EA%B6%81_%EA%B2%BD%EB%B6%80" title="자궁 경부">자궁 경부</a>를 통해 <a href="/wiki/%EC%9E%90%EA%B6%81" title="자궁">자궁</a>과 연결되고 <a href="/wiki/%EC%9E%90%EA%B6%81" title="자궁">자궁</a>과 <a href="/wiki/%EB%82%9C%EC%86%8C" title="난소">난소</a>는 <a href="/wiki/%EB%82%98%ED%8C%94%EA%B4%80" title="나팔관">나팔관</a>으로 연결된다. <a href="/wiki/%EB%82%9C%EC%86%8C" title="난소">난소</a>에서는 보통 28 <a class="mw-redirect" href="/wiki/%EC%9D%BC_(%EC%8B%9C%EA%B0%84)" title="일 (시간)">일</a> 주기로 <a href="/wiki/%EB%82%9C%EC%9E%90" title="난자">난자</a>가 배출된다. 배출된 난자는 나팔관을 통해 자궁 내막에 도달하며 <a href="/wiki/%EC%88%98%EC%A0%95_(%EC%83%9D%EB%AC%BC%ED%95%99)" title="수정 (생물학)">수정</a>을 이루지 않은 난자는 자궁내막과 함께 질로 배출되어 <a href="/wiki/%EC%9B%94%EA%B2%BD" title="월경">월경</a>이 이루어진다.<sup class="reference" id="cite_ref-body_guide_2-2"><a href="#cite_note-body_guide-2">[2]</a></sup></p>
<p>여성은 2차 성징이 일어나면 피하 지방이 발달하고 <a href="/wiki/%EA%B3%A8%EB%B0%98" title="골반">골반</a>이 넓어지며 <a class="mw-redirect" href="/wiki/%EC%9C%A0%EB%B0%A9_(%EB%8F%99%EB%AC%BC)" title="유방 (동물)">유방</a>이 발달한다. 중요한 여성 호르몬으로는 <a href="/wiki/%EC%97%90%EC%8A%A4%ED%8A%B8%EB%A1%9C%EA%B2%90" title="에스트로겐">에스트로겐</a>과 <a class="mw-redirect" href="/wiki/%ED%94%84%EB%A1%9C%EA%B2%8C%EC%8A%A4%ED%85%8C%EB%A1%A0" title="프로게스테론">프로게스테론</a>이 있다.<sup class="reference" id="cite_ref-Kimball_5-1"><a href="#cite_note-Kimball-5">[5]</a></sup></p>
