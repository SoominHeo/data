<p><b>생폴드방스</b> (<span style="font-size: smaller;"><a href="/wiki/%ED%94%84%EB%9E%91%EC%8A%A4%EC%96%B4" title="프랑스어">프랑스어</a>:</span> <span lang="fr" xml:lang="fr">Saint-Paul-de-Vence</span>) 또는 <b>생폴</b> (<span style="font-size: smaller;"><a href="/wiki/%ED%94%84%EB%9E%91%EC%8A%A4%EC%96%B4" title="프랑스어">프랑스어</a>:</span> <span lang="fr" xml:lang="fr">Saint-Paul</span>, <span style="font-size: smaller;"><a href="/wiki/%EC%98%A4%ED%81%AC%EC%96%B4" title="오크어">오크어</a>:</span> <span lang="oc" xml:lang="oc">Sant Pau</span>)은 <a href="/wiki/%ED%94%84%EB%9E%91%EC%8A%A4" title="프랑스">프랑스</a> 남부 <a href="/wiki/%EC%95%8C%ED%94%84%EB%A7%88%EB%A6%AC%ED%8C%80_%EC%A3%BC" title="알프마리팀 주">알프마리팀 주</a>에 자리한 <a href="/wiki/%ED%94%84%EB%9E%91%EC%8A%A4%EC%9D%98_%EC%BD%94%EB%AE%8C" title="프랑스의 코뮌">코뮌</a>이다. <a href="/wiki/%EC%BD%94%ED%8A%B8%EB%8B%A4%EC%A5%90%EB%A5%B4" title="코트다쥐르">코트다쥐르</a>의 유서 깊은 중세 마을 중 하나로, <a href="/wiki/%EA%B7%BC%EB%8C%80_%EB%AF%B8%EC%88%A0" title="근대 미술">근대</a>와 <a href="/wiki/%ED%98%84%EB%8C%80_%EB%AF%B8%EC%88%A0" title="현대 미술">현대 미술</a>을 전시하는 박물관 및 미술관으로 유명하다. 대표적인 미술관으로는 바로 근처에 위치한 <a class="new" href="/w/index.php?title=%EB%A7%A4%EA%B7%B8_%EB%AF%B8%EC%88%A0%EA%B4%80&amp;action=edit&amp;redlink=1" title="매그 미술관 (없는 문서)">매그 미술관</a>이 있다.<sup class="reference" id="cite_ref-PlanetFilou2012_1-0"><a href="#cite_note-PlanetFilou2012-1">[1]</a></sup></p>

<h2><span class="mw-headline" id=".EC.9C.A0.EB.AA.85.EC.9D.B8">유명인</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%83%9D%ED%8F%B4%EB%93%9C%EB%B0%A9%EC%8A%A4&amp;action=edit&amp;section=2" title="부분 편집: 유명인">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>생폴드방스는 오랜 기간 동안 그 유명세를 떨쳐 왔다. 1960년대에는 배우인 <a href="/wiki/%EC%9D%B4%EB%B8%8C_%EB%AA%BD%ED%83%95" title="이브 몽탕">이브 몽탕</a>, <a href="/wiki/%EC%8B%9C%EB%AA%AC_%EC%8B%9C%EB%87%A8%EB%A0%88" title="시몬 시뇨레">시몬 시뇨레</a>, <a href="/wiki/%EB%A6%AC%EB%85%B8_%EB%B2%A4%ED%88%AC%EB%9D%BC" title="리노 벤투라">리노 벤투라</a>, 그리고 시인 <a href="/wiki/%EC%9E%90%ED%81%AC_%ED%94%84%EB%A0%88%EB%B2%A0%EB%A5%B4" title="자크 프레베르">자크 프레베르</a>가 자주 찾았던 것으로 알려져 있다.</p>
<p>생폴드방스는 <a class="new" href="/w/index.php?title=%EC%9E%90%ED%81%AC_%EB%9D%BC%EB%B2%A0%EB%9D%BC&amp;action=edit&amp;redlink=1" title="자크 라베라 (없는 문서)">자크 라베라</a>, <a class="new" href="/w/index.php?title=%EA%B7%B8%EC%9B%AC_%EB%9D%BC%EB%B2%A0%EB%9D%BC&amp;action=edit&amp;redlink=1" title="그웬 라베라 (없는 문서)">그웬 라베라</a>, <a href="/wiki/%EB%A7%88%EB%A5%B4%ED%81%AC_%EC%83%A4%EA%B0%88" title="마르크 샤갈">마르크 샤갈</a>, 그리고 좀더 최근에 와선 <a class="new" href="/w/index.php?title=%EB%B2%A0%EB%A5%B4%EB%82%98%EB%A5%B4%EC%95%99%EB%A6%AC_%EB%A0%88%EB%B9%84&amp;action=edit&amp;redlink=1" title="베르나르앙리 레비 (없는 문서)">베르나르앙리 레비</a>와 <a href="/wiki/%EC%95%84%EB%A6%AC%EC%97%98_%EB%8F%99%EB%B0%94%EC%8A%AC" title="아리엘 동바슬">아리엘 동바슬</a> 등의 유명 예술가들이 거주했던 마을로도 유명하다<sup class="reference" id="cite_ref-2"><a href="#cite_note-2">[2]</a></sup> <a class="mw-redirect" href="/wiki/%EB%A1%A4%EB%A7%81_%EC%8A%A4%ED%86%A4%EC%A6%88" title="롤링 스톤즈">롤링 스톤즈</a>의 전 베이시스트였던 <a class="new" href="/w/index.php?title=%EB%B9%8C_%EC%99%80%EC%9D%B4%EB%93%9C&amp;action=edit&amp;redlink=1" title="빌 와이드 (없는 문서)">빌 와이드</a>도 이곳에 살고 있다. 미국의 작가인 <a class="new" href="/w/index.php?title=%EC%A0%9C%EC%9E%84%EC%8A%A4_%EB%B3%BC%EB%93%9C%EC%9C%88_(%EC%9E%91%EA%B0%80)&amp;action=edit&amp;redlink=1" title="제임스 볼드윈 (작가) (없는 문서)">제임스 볼드윈</a>은 1987년 이곳에서 사망하기도 했으며<sup class="reference" id="cite_ref-3"><a href="#cite_note-3">[3]</a></sup> 영국 배우 <a class="new" href="/w/index.php?title=%EB%8F%84%EB%84%90%EB%93%9C_%ED%94%8C%EB%A6%AC%EC%A0%84%EC%8A%A4&amp;action=edit&amp;redlink=1" title="도널드 플리전스 (없는 문서)">도널드 플리전스</a>도 1995년 이곳에서 생을 마쳤다.</p>
