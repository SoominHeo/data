<p><b>스테판 샤르보니에</b>(<span lang="fr" xml:lang="fr">Stéphane Charbonnier</span>, <a href="/wiki/1967%EB%85%84" title="1967년">1967년</a> <a href="/wiki/8%EC%9B%94_21%EC%9D%BC" title="8월 21일">8월 21일</a> ~ <a href="/wiki/2015%EB%85%84" title="2015년">2015년</a> <a href="/wiki/1%EC%9B%94_7%EC%9D%BC" title="1월 7일">1월 7일</a>)는 <b>샤르브</b>(<span lang="fr" xml:lang="fr">Charb</span>)라는 필명으로 잘 알려진 <a href="/wiki/%ED%94%84%EB%9E%91%EC%8A%A4" title="프랑스">프랑스</a>의 <a href="/wiki/%EB%A7%8C%ED%99%94%EA%B0%80" title="만화가">만화가</a>이다. 샤르보니에는 중산층 가정에서 태어나 자랐고, 만화에 관심이 많아 <a href="/wiki/1990%EB%85%84%EB%8C%80" title="1990년대">1990년대</a>부터 만화 활동을 시작하였다. 샤르보니에는 <a href="/wiki/2000%EB%85%84%EB%8C%80" title="2000년대">2000년대</a>에 <a href="/wiki/%EC%83%A4%EB%A5%BC%EB%A6%AC_%EC%97%90%EB%B8%8C%EB%8F%84" title="샤를리 에브도">샤를리 에브도</a>에 들어갔고, 풍자에 관심을 가지기 시작했다. <a href="/wiki/2009%EB%85%84" title="2009년">2009년</a> <a href="/wiki/%EC%83%A4%EB%A5%BC%EB%A6%AC_%EC%97%90%EB%B8%8C%EB%8F%84" title="샤를리 에브도">샤를리 에브도</a>의 편집장이 사임하자, 샤르보니에는 샤르브라는 필명으로 편집장이 되었다. 평소 자기 성격을 언론에 드러내는 것을 좋아하는 성격의 샤르브는 언론의 주목을 받았고, 앞 사진이 찍히고 3일 뒤(<a href="/wiki/2011%EB%85%84" title="2011년">2011년</a> <a href="/wiki/11%EC%9B%94_5%EC%9D%BC" title="11월 5일">11월 5일</a>)에도 인터뷰를 받았다. 그러나 최근 이슬람 테러조직이 샤르브에 만평과 풍자에 불만을 품고 여러 차레 테러를 저질러서 호위병들이 샤르브를 보호하던 상황이었다. 샤르브는 〈<a href="/wiki/%EC%83%A4%EB%A5%BC%EB%A6%AC_%EC%97%90%EB%B8%8C%EB%8F%84" title="샤를리 에브도">샤를리 에브도</a>〉에서 2015년 1월 발생한 <a class="mw-redirect" href="/wiki/%EC%83%A4%EB%A5%BC%EB%A6%AC_%EC%97%90%EB%B8%8C%EB%8F%84_%EC%B4%9D%EA%B2%A9" title="샤를리 에브도 총격">총격 사건</a>으로 사망하였다. 샤르브는 〈샤를리 에브도〉의 편집장이었다.</p>
