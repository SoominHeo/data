<p><b>샤를 6세</b>(<span style="font-size: smaller;"><a href="/wiki/%ED%94%84%EB%9E%91%EC%8A%A4%EC%96%B4" title="프랑스어">프랑스어</a>:</span> <span lang="fr" xml:lang="fr">Charles VI</span>, <a href="/wiki/1368%EB%85%84" title="1368년">1368년</a> <a href="/wiki/12%EC%9B%94_3%EC%9D%BC" title="12월 3일">12월 3일</a> ~ <a href="/wiki/1422%EB%85%84" title="1422년">1422년</a> <a href="/wiki/10%EC%9B%94_21%EC%9D%BC" title="10월 21일">10월 21일</a>)는 <a href="/wiki/%ED%94%84%EB%9E%91%EC%8A%A4" title="프랑스">프랑스</a>의 국왕이다. <b>친애왕</b>(<span style="font-size: smaller;"><a href="/wiki/%ED%94%84%EB%9E%91%EC%8A%A4%EC%96%B4" title="프랑스어">프랑스어</a>:</span> <span lang="fr" xml:lang="fr">le Bienaimé</span>), <b>광인왕</b>(<span style="font-size: smaller;"><a href="/wiki/%ED%94%84%EB%9E%91%EC%8A%A4%EC%96%B4" title="프랑스어">프랑스어</a>:</span> <span lang="fr" xml:lang="fr">le Fol 또는 le Fou</span>)으로 불리기도 한다.</p>

<p>샤를 6세는 <a href="/wiki/1368%EB%85%84" title="1368년">1368년</a> <a href="/wiki/12%EC%9B%94_3%EC%9D%BC" title="12월 3일">12월 3일</a> <a href="/wiki/%ED%94%84%EB%9E%91%EC%8A%A4" title="프랑스">프랑스</a> 국왕 <a class="mw-redirect" href="/wiki/%ED%94%84%EB%9E%91%EC%8A%A4%EC%9D%98_%EC%83%A4%EB%A5%BC_5%EC%84%B8" title="프랑스의 샤를 5세">샤를 5세</a>와 <a href="/wiki/%EB%B6%80%EB%A5%B4%EB%B4%89%EC%9D%98_%EC%9E%94" title="부르봉의 잔">부르봉의 잔</a> 사이의 아들로 <a class="mw-redirect" href="/wiki/%ED%8C%8C%EB%A6%AC_%EC%8B%9C" title="파리 시">파리</a>에서 태어났다. 태어나자마자 그는 <a href="/wiki/%EB%8F%84%ED%8C%BD" title="도팽">도팽</a>의 작위를 얻었고, 최초로 유아기에 도팽의 작위를 얻은 프랑스의 왕자가 되었다. 그는 메지에르의 필리프에게 교육을 받았고, <a href="/wiki/1380%EB%85%84" title="1380년">1380년</a> 12살의 나이로 프랑스 국왕이 되었으나 왕으로서의 권한은 숙부인 <a class="mw-redirect" href="/wiki/%EC%95%99%EC%A3%BC_%EA%B3%B5%EC%9E%91_%EB%A3%A8%EC%9D%B4" title="앙주 공작 루이">앙주 공작 루이</a>, <a href="/wiki/%EB%B2%A0%EB%A6%AC_%EA%B3%B5%EC%9E%91_%EC%9E%A5" title="베리 공작 장">베리 공작 장</a>, <a class="mw-redirect" href="/wiki/%EB%B6%80%EB%A5%B4%EA%B3%A0%EB%89%B4_%EA%B3%B5%EC%9E%91_%ED%95%84%EB%A6%AC%ED%94%84_2%EC%84%B8" title="부르고뉴 공작 필리프 2세">부르고뉴 공작 필리프 2세</a>와 외삼촌 <a class="new" href="/w/index.php?title=%EB%B6%80%EB%A5%B4%EB%B4%89_%EA%B3%B5%EC%9E%91_%EB%A3%A8%EC%9D%B4_2%EC%84%B8&amp;action=edit&amp;redlink=1" title="부르봉 공작 루이 2세 (없는 문서)">부르봉 공작 루이 2세</a> 등에게 할당되었다. 사망한 샤를 5세의 칙령으로 인하여 숙부 <a class="mw-redirect" href="/wiki/%EC%95%99%EC%A3%BC_%EA%B3%B5%EC%9E%91_%EB%A3%A8%EC%9D%B4" title="앙주 공작 루이">앙주 공작 루이</a>가 샤를 6세의 섭정을 맡게 되었고 샤를 6세의 보호와 <a class="mw-redirect" href="/wiki/%ED%8C%8C%EB%A6%AC_%EC%8B%9C" title="파리 시">파리</a>와 <a href="/wiki/%EB%85%B8%EB%A5%B4%EB%A7%9D%EB%94%94" title="노르망디">노르망디</a>의 관리는 부르고뉴 공작과 부르봉 공작이 맡았다. 베리 공작은 핵심 직책을 맡지 못한 대신 <a href="/wiki/%EB%9E%91%EA%B7%B8%EB%8F%84%ED%81%AC" title="랑그도크">랑그도크</a>와 <a class="new" href="/w/index.php?title=%EA%B8%B0%EC%98%8C&amp;action=edit&amp;redlink=1" title="기옌 (없는 문서)">기옌</a>을 차지하게 되었다. 앙주 공작의 섭정은 오랫동안 지속되지 못하였고, 그는 상당한 양의 세금을 내면서 샤를 5세의 영지를 가지는 것에서 만족해야했다. 앙주 공작은 <a href="/wiki/1382%EB%85%84" title="1382년">1382년</a> <a href="/wiki/%EC%9D%B4%ED%83%88%EB%A6%AC%EC%95%84" title="이탈리아">이탈리아</a>의 <a href="/wiki/%EB%82%98%ED%8F%B4%EB%A6%AC" title="나폴리">나폴리</a>를 공격하는 데 일조하기 위해서 떠났다.</p>
<p><a class="mw-redirect" href="/wiki/%ED%94%84%EB%9E%91%EC%8A%A4%EC%9D%98_%EC%83%A4%EB%A5%BC_5%EC%84%B8" title="프랑스의 샤를 5세">샤를 5세</a>가 사망할 당시 남부 <a href="/wiki/%ED%94%84%EB%9E%91%EC%8A%A4" title="프랑스">프랑스</a> 지역에는 불만을 가진 세력들이 있었는데, 앙주 공작이 샤를 5세가 면제해 주었던 세금을 다시 거두려고 하자 퓌와 <a class="new" href="/w/index.php?title=%EB%AA%BD%ED%95%84%EB%A6%AC%EC%97%90&amp;action=edit&amp;redlink=1" title="몽필리에 (없는 문서)">몽필리에</a>에서 반란이 일어났고, <a class="mw-redirect" href="/wiki/%ED%8C%8C%EB%A6%AC_%EC%8B%9C" title="파리 시">파리</a>, <a href="/wiki/%EB%A3%A8%EC%95%99" title="루앙">루앙</a>, <a href="/wiki/%ED%94%8C%EB%9E%91%EB%93%9C%EB%A5%B4" title="플랑드르">플랑드르</a>, <a href="/wiki/%EC%95%84%EB%AF%B8%EC%95%B5" title="아미앵">아미앵</a>, <a href="/wiki/%EC%98%A4%EB%A5%BC%EB%A0%88%EC%95%99" title="오를레앙">오를레앙</a>, <a href="/wiki/%EB%9E%AD%EC%8A%A4" title="랭스">랭스</a> 등의 수많은 지역에서 반란이 발생하였다. 샤를 6세는 플랑드르 백작을 돕기 위해서 출정하였고, <a href="/wiki/1382%EB%85%84" title="1382년">1382년</a> <a href="/wiki/11%EC%9B%94_27%EC%9D%BC" title="11월 27일">11월 27일</a> <a class="new" href="/w/index.php?title=%EB%A3%A8%EC%8A%A4%EB%B9%84%ED%81%AC_%EC%A0%84%ED%88%AC&amp;action=edit&amp;redlink=1" title="루스비크 전투 (없는 문서)">루스비크 전투</a>에서 반란군 지도자 <a class="new" href="/w/index.php?title=%ED%95%84%EB%A6%BD_%EB%B0%98_%EC%95%84%EB%A5%B4%ED%85%8C%EB%B2%A8%EB%8D%B0&amp;action=edit&amp;redlink=1" title="필립 반 아르테벨데 (없는 문서)">필립 반 아르테벨데</a>를 격파하였고 다음해 1월 파리로 돌아왔다. <a href="/wiki/%EC%9E%89%EA%B8%80%EB%9E%9C%EB%93%9C" title="잉글랜드">잉글랜드</a>가 플랑드르의 도시들에게 보낸 원조로 인해서 샤를 6세는 잉글랜드 원정을 결정하였다. 그러나 <a href="/wiki/1385%EB%85%84" title="1385년">1385년</a> <a class="new" href="/w/index.php?title=%EC%9E%A5_%EB%93%9C_%EB%B9%84%EC%97%94&amp;action=edit&amp;redlink=1" title="장 드 비엔 (없는 문서)">장 드 비엔</a>은 <a href="/wiki/%EC%8A%A4%EC%BD%94%ED%8B%80%EB%9E%9C%EB%93%9C" title="스코틀랜드">스코틀랜드</a>에 원활하게 상륙하지 못하였고, 원정을 위한 <a class="new" href="/w/index.php?title=%EC%8A%AC%EB%A3%A8%EC%9D%B4%EC%8A%A4&amp;action=edit&amp;redlink=1" title="슬루이스 (없는 문서)">슬루이스</a>의 함대 역시 잉글랜드 군에 의해서 파괴되어 출정이 지연되었다.</p>
<h2><span class="mw-headline" id=".EC.A0.95.EC.8B.A0.EB.B3.91">정신병</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%83%A4%EB%A5%BC_6%EC%84%B8&amp;action=edit&amp;section=2" title="부분 편집: 정신병">편집</a><span class="mw-editsection-bracket">]</span></span></h2>

<p><a href="/wiki/1385%EB%85%84" title="1385년">1385년</a> 샤를 6세는 <a class="new" href="/w/index.php?title=%EC%8A%88%ED%85%8C%ED%8C%90_3%EC%84%B8&amp;action=edit&amp;redlink=1" title="슈테판 3세 (없는 문서)">슈테판 3세</a>의 딸인 <a href="/wiki/%EB%B0%94%EC%9D%B4%EC%97%90%EB%A5%B8%EC%9D%98_%EC%9D%B4%EC%9E%90%EB%B3%B4" title="바이에른의 이자보">바이에른의 이자보</a>와 결혼하였다. 프랑스에서 그녀는 이자보라는 이름으로 불리었다. <a href="/wiki/1388%EB%85%84" title="1388년">1388년</a> <a href="/wiki/%EC%98%A4%EB%A5%BC%EB%A0%88%EC%95%99_%EA%B3%B5%EC%9E%91_%EB%A3%A8%EC%9D%B4_1%EC%84%B8_%EB%93%9C_%EB%B0%9C%EB%A3%A8%EC%95%84" title="오를레앙 공작 루이 1세 드 발루아">투렌 공작 루이 도를레앙</a>의 도움으로 샤를 6세는 삼촌들로부터 벗어나 친정에 나설 수 있었다. 그는 궁정을 <a class="mw-redirect" href="/wiki/%ED%94%84%EB%9E%91%EC%8A%A4%EC%9D%98_%EC%83%A4%EB%A5%BC_5%EC%84%B8" title="프랑스의 샤를 5세">샤를 5세</a>의 신하들로 채워나갔다. <a href="/wiki/1390%EB%85%84" title="1390년">1390년</a> 그는 베리 공작으로부터 <a href="/wiki/%EB%9E%91%EA%B7%B8%EB%8F%84%ED%81%AC" title="랑그도크">랑그도크</a> 지역을 몰수하였다. 몇 년간 그는 국가를 잘 이끌어 나갔으나, <a href="/wiki/1392%EB%85%84" title="1392년">1392년</a> <a href="/wiki/%EB%B8%8C%EB%A5%B4%ED%83%80%EB%89%B4" title="브르타뉴">브르타뉴</a>로 가는 중 <a href="/wiki/%EB%A5%B4%EB%A7%9D" title="르망">르망</a>에서 샤를 6세에게 정신병이 발생하였다. 이로 인해서 샤를 6세는 더 이상 국정을 직접 이끌어가기가 힘들게 되었다. 샤를 6세의 정신병으로 인해서 샤를 6세가 임명하였던 샤를 5세의 신하들은 다시 밀려났으며, 왕족들이 다시 권력을 차지하게 되었다. 이들은 왕을 대신하여 국가를 다스리기 시작한 <a class="mw-redirect" href="/wiki/%EB%B6%80%EB%A5%B4%EA%B3%A0%EB%89%B4_%EA%B3%B5%EC%9E%91_%ED%95%84%EB%A6%AC%ED%94%84_2%EC%84%B8" title="부르고뉴 공작 필리프 2세">부르고뉴 공작 필리프 2세</a>와 이에 반대하는 <a href="/wiki/%EC%98%A4%EB%A5%BC%EB%A0%88%EC%95%99_%EA%B3%B5%EC%9E%91_%EB%A3%A8%EC%9D%B4_1%EC%84%B8_%EB%93%9C_%EB%B0%9C%EB%A3%A8%EC%95%84" title="오를레앙 공작 루이 1세 드 발루아">오를레앙 공작 루이 1세 드 발루아</a>를 중심으로 두 파로 나뉘었다.</p>
<p><a href="/wiki/1396%EB%85%84" title="1396년">1396년</a> <a href="/wiki/%EC%9E%89%EA%B8%80%EB%9E%9C%EB%93%9C" title="잉글랜드">잉글랜드</a>의 국왕 <a class="mw-redirect" href="/wiki/%EC%9E%89%EA%B8%80%EB%9E%9C%EB%93%9C%EC%9D%98_%EB%A6%AC%EC%B2%98%EB%93%9C_2%EC%84%B8" title="잉글랜드의 리처드 2세">리처드 2세</a>와 샤를 6세의 딸인 <a href="/wiki/%EB%B0%9C%EB%A3%A8%EC%95%84%EC%9D%98_%EC%9D%B4%EC%9E%90%EB%B2%A8" title="발루아의 이자벨">발루아의 이자벨</a>의 결혼으로 인하여 잉글랜드와 <a href="/wiki/%ED%94%84%EB%9E%91%EC%8A%A4" title="프랑스">프랑스</a> 간에는 평화가 찾아올 것처럼 보였다. 그러나 <a href="/wiki/1399%EB%85%84" title="1399년">1399년</a> 리처드 2세가 <a class="mw-redirect" href="/wiki/%EC%9E%89%EA%B8%80%EB%9E%9C%EB%93%9C%EC%9D%98_%ED%97%A8%EB%A6%AC_4%EC%84%B8" title="잉글랜드의 헨리 4세">헨리 볼링브룩</a>에게 왕위를 찬탈 당하자 이러한 연합 역시도 효력을 잃게 되었다.</p>
<p><a href="/wiki/1398%EB%85%84" title="1398년">1398년</a> <a class="mw-redirect" href="/wiki/%ED%8C%8C%EB%A6%AC_%EC%8B%9C" title="파리 시">파리</a>에서 열린 성직자 회의에서 프랑스 교회는 <a class="new" href="/w/index.php?title=%EB%8C%80%EB%A6%BD_%EA%B5%90%ED%99%A9_%ED%81%B4%EB%A0%88%EB%A9%98%EC%8A%A4_7%EC%84%B8&amp;action=edit&amp;redlink=1" title="대립 교황 클레멘스 7세 (없는 문서)">대립 교황 클레멘스 7세</a>를 계승한 <a class="new" href="/w/index.php?title=%EB%8C%80%EB%A6%BD_%EA%B5%90%ED%99%A9_%EB%B2%A0%EB%84%A4%EB%94%95%ED%86%A0_13%EC%84%B8&amp;action=edit&amp;redlink=1" title="대립 교황 베네딕토 13세 (없는 문서)">베네딕토 13세</a>를 인정하기를 거부하였다. 베네딕트 13세는 오를레앙 공작의 지지를 받고 있었지만, 부르고뉴 공작과 <a href="/wiki/%ED%8C%8C%EB%A6%AC_%EB%8C%80%ED%95%99%EA%B5%90" title="파리 대학교">파리 대학교</a>는 베네딕트 13세를 인정하기를 거부하였다. <a href="/wiki/1403%EB%85%84" title="1403년">1403년</a> 베네딕토 13세는 권한을 되찾을 수 있었지만 <a href="/wiki/1408%EB%85%84" title="1408년">1408년</a> 샤를 6세가 스스로를 프랑스 교회의 보호자로 선포하자, 베네딕토 13세는 프랑스 교회에 대한 권한을 잃게 되었다.</p>
<h2><span class="mw-headline" id=".EA.B6.8C.EB.A0.A5_.EB.8B.A4.ED.88.BC">권력 다툼</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%83%A4%EB%A5%BC_6%EC%84%B8&amp;action=edit&amp;section=3" title="부분 편집: 권력 다툼">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>샤를 6세의 병세는 시간이 지날수록 심해져갔고, <a href="/wiki/1404%EB%85%84" title="1404년">1404년</a> 궁중 내에서 상당한 세력을 가지고 있던 <a class="mw-redirect" href="/wiki/%EB%B6%80%EB%A5%B4%EA%B3%A0%EB%89%B4_%EA%B3%B5%EC%9E%91_%ED%95%84%EB%A6%AC%ED%94%84_2%EC%84%B8" title="부르고뉴 공작 필리프 2세">부르고뉴 공작 필리프 2세</a>가 사망하자 권력을 두고 다툼이 일어나기 시작하였다. 새롭게 공작이 된 <a class="mw-redirect" href="/wiki/%EC%9E%A5_1%EC%84%B8_(%EB%B6%80%EB%A5%B4%EA%B3%A0%EB%89%B4_%EA%B3%B5)" title="장 1세 (부르고뉴 공)">장 1세</a>는 오를레앙 공작의 세력 확장으로 그의 아버지의 권력을 그대로 계승받지 못하였다. 왕비 <a href="/wiki/%EB%B0%94%EC%9D%B4%EC%97%90%EB%A5%B8%EC%9D%98_%EC%9D%B4%EC%9E%90%EB%B3%B4" title="바이에른의 이자보">이자보</a>는 원래 부르고뉴 공작파를 지지하였으나, 샤를 6세의 병이 깊어지자 사실상 왕과는 단절되어있는 상태였다. 이자보의 위치는 부르고뉴파의 <a class="new" href="/w/index.php?title=%EC%98%A4%EB%8D%B0%ED%8A%B8_%EB%93%9C_%EC%83%B4%EB%94%94%EB%B8%8C%EB%A5%B4&amp;action=edit&amp;redlink=1" title="오데트 드 샴디브르 (없는 문서)">오데트 드 샴디브르</a>가 대신하게 되었다. 이자보는 오를레앙 공작과 음모를 꾸몄다고 하여 고발당했다.</p>
<p>부르고뉴 공작과 오를레앙 공작의 다툼은 점점 커졌고, 결국 <a href="/wiki/1407%EB%85%84" title="1407년">1407년</a> <a href="/wiki/11%EC%9B%94_23%EC%9D%BC" title="11월 23일">11월 23일</a> 오를레앙 공작이 <a class="mw-redirect" href="/wiki/%ED%8C%8C%EB%A6%AC_%EC%8B%9C" title="파리 시">파리</a>의 거리에서 살해당했다. 새롭게 오를레앙 공작이 된 <a class="mw-redirect" href="/wiki/%EC%83%A4%EB%A5%BC_%EB%8F%84%EB%A5%BC%EB%A0%88%EC%95%99" title="샤를 도를레앙">샤를</a>은 <a class="new" href="/w/index.php?title=%EC%95%84%EB%A5%B4%EB%A7%88%EB%83%90%ED%81%AC_%EB%B0%B1%EC%9E%91_%EB%B2%A0%EB%A5%B4%EB%82%98%EB%A5%B4_7%EC%84%B8&amp;action=edit&amp;redlink=1" title="아르마냐크 백작 베르나르 7세 (없는 문서)">아르마냐크 백작 베르나르 7세</a>의 딸과 결혼하였고, 이를 이용하여 베리, 부르봉, 브르고뉴 공작 등과 함께 부르고뉴 공작에 대항하였다. 이들은 아르마냐크 파로 불리었다. <a href="/wiki/1411%EB%85%84" title="1411년">1411년</a> 부르고뉴 공작은 <a href="/wiki/%EC%9E%89%EA%B8%80%EB%9E%9C%EB%93%9C" title="잉글랜드">잉글랜드</a>의 국왕 <a class="mw-redirect" href="/wiki/%EC%9E%89%EA%B8%80%EB%9E%9C%EB%93%9C%EC%9D%98_%ED%97%A8%EB%A6%AC_4%EC%84%B8" title="잉글랜드의 헨리 4세">헨리 4세</a>와 동맹을 채결하였다. 그러나 <a href="/wiki/1412%EB%85%84" title="1412년">1412년</a> 아르마냐크 파 역시 <a href="/wiki/%EC%95%84%ED%82%A4%ED%85%90" title="아키텐">아키텐</a>을 잉글랜드에 넘겨줄 것을 약속하고 잉글랜드의 지원을 받을 수 있었다.</p>
<p><a href="/wiki/1413%EB%85%84" title="1413년">1413년</a> 4월 부르고뉴파인 <a class="new" href="/w/index.php?title=%EC%8B%9C%EB%AA%BD_%EC%B9%B4%EB%B3%B4%EC%8A%88&amp;action=edit&amp;redlink=1" title="시몽 카보슈 (없는 문서)">시몽 카보슈</a>가 군중들을 이끌고 <a href="/wiki/%EB%8F%84%ED%8C%BD" title="도팽">도팽</a>의 궁전을 공격하는 일이 발생하자 <a href="/wiki/1413%EB%85%84" title="1413년">1413년</a> <a href="/wiki/5%EC%9B%94_27%EC%9D%BC" title="5월 27일">5월 27일</a> 샤를 6세는 행정 개혁을 위한 칙령을 발표하였고 그 결과 왕의 고문 중 일부가 권한을 잃었다. 그러나 이 소요가 계속되자 오를레앙 공작은 사태를 진압할 명분을 가지게 되었고 결국 샤를 6세는 칙령을 다시 거두어들이고 가담자들을 추방하였다.</p>
<p><a href="/wiki/1414%EB%85%84" title="1414년">1414년</a> <a href="/wiki/%EC%9E%89%EA%B8%80%EB%9E%9C%EB%93%9C" title="잉글랜드">잉글랜드</a> 국왕 <a class="mw-redirect" href="/wiki/%EC%9E%89%EA%B8%80%EB%9E%9C%EB%93%9C%EC%9D%98_%ED%97%A8%EB%A6%AC_5%EC%84%B8" title="잉글랜드의 헨리 5세">헨리 5세</a>는 부르고뉴파와 연합을 계속할 것을 결정하고 다시 프랑스 왕위를 주장하기 시작하였다. 헨리 5세는 <a href="/wiki/1415%EB%85%84" title="1415년">1415년</a> <a href="/wiki/%EC%95%84%EC%9F%81%EC%BF%A0%EB%A5%B4_%EC%A0%84%ED%88%AC" title="아쟁쿠르 전투">아쟁쿠르 전투</a>에서 승리를 거두었게 되었다. 같은 해 샤를 6세의 아들인 <a class="new" href="/w/index.php?title=%EA%B8%B0%EC%98%8C_%EA%B3%B5%EC%9E%91_%EB%A3%A8%EC%9D%B4&amp;action=edit&amp;redlink=1" title="기옌 공작 루이 (없는 문서)">기옌 공작 루이</a>가 사망하였고 <a href="/wiki/1417%EB%85%84" title="1417년">1417년</a> 다른 아들인 <a class="new" href="/w/index.php?title=%ED%88%AC%EB%A0%8C_%EA%B3%B5%EC%9E%91_%EC%9E%A5&amp;action=edit&amp;redlink=1" title="투렌 공작 장 (없는 문서)">투렌 공작 장</a>이 사망함에 따라 <a class="mw-redirect" href="/wiki/%ED%94%84%EB%9E%91%EC%8A%A4%EC%9D%98_%EC%83%A4%EB%A5%BC_7%EC%84%B8" title="프랑스의 샤를 7세">퐁티외 공작 샤를</a>이 그의 상속자가 되었다. <a class="mw-redirect" href="/wiki/%ED%8C%8C%EB%A6%AC_%EC%8B%9C" title="파리 시">파리</a>는 아르마냐크 백작에 의해서 통치받고 있었고 왕비 <a href="/wiki/%EB%B0%94%EC%9D%B4%EC%97%90%EB%A5%B8%EC%9D%98_%EC%9D%B4%EC%9E%90%EB%B3%B4" title="바이에른의 이자보">이자보</a>는 <a href="/wiki/%ED%88%AC%EB%A5%B4" title="투르">투르</a>에 갇혀 있다가 부르고뉴로 탈출하였다. <a href="/wiki/1418%EB%85%84" title="1418년">1418년</a> <a href="/wiki/5%EC%9B%94_29%EC%9D%BC" title="5월 29일">5월 29일</a> 부르고뉴파에 의해서 파리가 함락당하였고 <a class="new" href="/w/index.php?title=%EB%B6%80%EB%A5%B4%EA%B3%A0%EB%89%B4_%EA%B3%B5%EC%9E%91_%EC%9E%A5_2%EC%84%B8&amp;action=edit&amp;redlink=1" title="부르고뉴 공작 장 2세 (없는 문서)">장 2세</a>와 왕비 이자보가 파리에 입성하였다. 그동안 <a class="mw-redirect" href="/wiki/%EC%9E%89%EA%B8%80%EB%9E%9C%EB%93%9C%EC%9D%98_%ED%97%A8%EB%A6%AC_5%EC%84%B8" title="잉글랜드의 헨리 5세">헨리 5세</a>는 <a href="/wiki/%EB%85%B8%EB%A5%B4%EB%A7%9D%EB%94%94" title="노르망디">노르망디</a>의 정벌을 완료하였다.</p>
<p><a href="/wiki/1419%EB%85%84" title="1419년">1419년</a> <a class="mw-redirect" href="/wiki/%EC%9E%A5_1%EC%84%B8_(%EB%B6%80%EB%A5%B4%EA%B3%A0%EB%89%B4_%EA%B3%B5)" title="장 1세 (부르고뉴 공)">장 1세</a>가 <a href="/wiki/%EB%8F%84%ED%8C%BD" title="도팽">도팽</a> <a class="mw-redirect" href="/wiki/%ED%94%84%EB%9E%91%EC%8A%A4%EC%9D%98_%EC%83%A4%EB%A5%BC_7%EC%84%B8" title="프랑스의 샤를 7세">샤를</a>의 눈앞에서 아르마냐크 파에 의해서 살해당하였다. 샤를은 뒤를 이어 부르고뉴 공작이 된 <a class="new" href="/w/index.php?title=%EB%B6%80%EB%A5%B4%EA%B3%A0%EB%89%B4_%EA%B3%B5%EC%9E%91_%ED%95%84%EB%A6%AC%ED%94%84_3%EC%84%B8&amp;action=edit&amp;redlink=1" title="부르고뉴 공작 필리프 3세 (없는 문서)">필리프 3세</a>, 왕비 이사보, 그리고 <a href="/wiki/%EC%9E%89%EA%B8%80%EB%9E%9C%EB%93%9C" title="잉글랜드">잉글랜드</a> 국왕 <a class="mw-redirect" href="/wiki/%EC%9E%89%EA%B8%80%EB%9E%9C%EB%93%9C%EC%9D%98_%ED%97%A8%EB%A6%AC_5%EC%84%B8" title="잉글랜드의 헨리 5세">헨리 5세</a>와 함께 <a class="new" href="/w/index.php?title=%ED%8A%B8%EB%A3%A8%EC%95%84_%EC%A1%B0%EC%95%BD&amp;action=edit&amp;redlink=1" title="트루아 조약 (없는 문서)">트루아 조약</a>에 서명하였다. 그 결과 헨리 5세는 샤를 대신 프랑스의 왕위계승자가 되었고, 북부 프랑스가 잉글랜드에게 넘어가게 되었다. 샤를 6세는 <a href="/wiki/%EC%83%81%EB%A6%AC%EC%8A%A4" title="상리스">상리스</a>에서 머물하다 <a href="/wiki/1422%EB%85%84" title="1422년">1422년</a> 9월 <a class="mw-redirect" href="/wiki/%ED%8C%8C%EB%A6%AC_%EC%8B%9C" title="파리 시">파리</a>로 돌아왔고 같은 해 <a href="/wiki/10%EC%9B%94_21%EC%9D%BC" title="10월 21일">10월 21일</a>에 사망하였다.</p>
