<p><b>샤모 쿠아예</b>(Shamo Quaye, <a href="/wiki/1971%EB%85%84" title="1971년">1971년</a> <a href="/wiki/10%EC%9B%94_22%EC%9D%BC" title="10월 22일">10월 22일</a> ~ <a href="/wiki/1997%EB%85%84" title="1997년">1997년</a> <a href="/wiki/11%EC%9B%94_30%EC%9D%BC" title="11월 30일">11월 30일</a>)는 <a href="/wiki/%EA%B0%80%EB%82%98" title="가나">가나</a>의 전 <a href="/wiki/%EC%B6%95%EA%B5%AC" title="축구">축구</a> <a href="/wiki/%EB%AF%B8%EB%93%9C%ED%95%84%EB%8D%94" title="미드필더">미드필더</a>였다.</p>
<p><a href="/wiki/1989%EB%85%84" title="1989년">1989년</a> 가나의 <a href="/wiki/%EC%95%84%ED%81%AC%EB%9D%BC_%ED%95%98%EC%B8%A0_%EC%98%A4%EB%B8%8C_%EC%98%A4%ED%81%AC_SC" title="아크라 하츠 오브 오크 SC">아크라 하츠 오브 오크 SC</a>에서 데뷔했으며, <a href="/wiki/1993%EB%85%84" title="1993년">1993년</a> <a class="mw-redirect" href="/wiki/%EC%82%AC%EC%9A%B0%EB%94%94%EC%95%84%EB%9D%BC%EB%B9%84%EC%95%84_%ED%94%84%EB%A6%AC%EB%AF%B8%EC%96%B4%EB%A6%AC%EA%B7%B8" title="사우디아라비아 프리미어리그">사우디아라비아 프리미어리그</a>의 <a class="new" href="/w/index.php?title=%EC%95%8C_%EC%B9%B4%EB%94%94%EC%8B%9C%EC%95%BC_%EC%95%8C_%EC%BF%A0%EB%B0%94%EB%A5%B4&amp;action=edit&amp;redlink=1" title="알 카디시야 알 쿠바르 (없는 문서)">알 카디시야 알 쿠바르</a>로 이적해 활약하였다. 이후 <a href="/wiki/1995%EB%85%84" title="1995년">1995년</a> 다시 아크라 하츠 오브 오크 SC로 복귀했으며, <a href="/wiki/1996%EB%85%84" title="1996년">1996년</a> <a href="/wiki/%EC%8A%A4%EC%9B%A8%EB%8D%B4" title="스웨덴">스웨덴</a> <a href="/wiki/%EC%95%8C%EC%8A%A4%EB%B2%A4%EC%8A%A4%EC%B9%B8" title="알스벤스칸">알스벤스칸</a>의 <a class="new" href="/w/index.php?title=%EC%9A%B0%EB%A9%94%EC%98%A4_FC&amp;action=edit&amp;redlink=1" title="우메오 FC (없는 문서)">우메오 FC</a>로 팀을 옮겼다.</p>
<p>또한 <a href="/wiki/1988%EB%85%84" title="1988년">1988년</a> <a href="/wiki/%EA%B0%80%EB%82%98_%EC%B6%95%EA%B5%AC_%EA%B5%AD%EA%B0%80%EB%8C%80%ED%91%9C%ED%8C%80" title="가나 축구 국가대표팀">가나 축구 국가대표팀</a>에 정식으로 데뷔한 뒤, 총 41경기에 출전했으며, 올림픽 국가대표팀으로도 활동하면서 팀이 <a href="/wiki/1992%EB%85%84_%ED%95%98%EA%B3%84_%EC%98%AC%EB%A6%BC%ED%94%BD" title="1992년 하계 올림픽">1992년 하계 올림픽</a>에서 동메달을 따는 데 기여했다.</p>
<p><a href="/wiki/1997%EB%85%84" title="1997년">1997년</a> <a href="/wiki/11%EC%9B%94_30%EC%9D%BC" title="11월 30일">11월 30일</a> 자국에서 열린 친선경기에 참가해 경기를 하던 도중 얼굴에 <a href="/wiki/%EC%B6%95%EA%B5%AC%EA%B3%B5" title="축구공">축구공</a>을 맞아 의식을 잃었으며, 병원에 도착하였을 때는 <a href="/wiki/%EB%8D%B0%EB%93%9C_%EC%98%A8_%EC%96%B4%EB%9D%BC%EC%9D%B4%EB%B2%8C" title="데드 온 어라이벌">이미 사망한 상태</a>였다.<sup class="reference" id="cite_ref-1"><a href="#cite_note-1">[1]</a></sup></p>
