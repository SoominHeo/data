<p><b>샤움부르크리페</b>(<span style="font-size: smaller;"><a href="/wiki/%EB%8F%85%EC%9D%BC%EC%96%B4" title="독일어">독일어</a>:</span> <span lang="de" xml:lang="de">Schaumburg-Lippe</span>)는 <a href="/wiki/1946%EB%85%84" title="1946년">1946년</a>까지 <a href="/wiki/%EB%8F%85%EC%9D%BC" title="독일">독일</a> 서부에 있던 연방이다. 영토 북동부에 슈타인후더(Steinhuder) 호를 끼고 있었다. 수도는 <a href="/wiki/%EB%B7%94%EC%BC%80%EB%B6%80%EB%A5%B4%ED%81%AC" title="뷔케부르크">뷔케부르크</a>에 두고 있었고, 면적은 340㎢, 인구는 <a href="/wiki/1925%EB%85%84" title="1925년">1925년</a> 당시 4만 8000명.</p>

<h2><span class="mw-headline" id=".EC.97.AD.EC.82.AC">역사</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%83%A4%EC%9B%80%EB%B6%80%EB%A5%B4%ED%81%AC%EB%A6%AC%ED%8E%98&amp;action=edit&amp;section=1" title="부분 편집: 역사">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p><a href="/wiki/1290%EB%85%84" title="1290년">1290년</a> 샤움부르크 백작령이 되었다가, <a href="/wiki/1647%EB%85%84" title="1647년">1647년</a> <a href="/wiki/%EB%A6%AC%ED%8E%98" title="리페">리페</a>와 분리되어, 샤움부르크리페 백작령이 되었다. <a href="/wiki/1807%EB%85%84" title="1807년">1807년</a> 후작국이 되어 <a href="/wiki/1918%EB%85%84" title="1918년">1918년</a>까지 지속하였다. <a href="/wiki/1815%EB%85%84" title="1815년">1815년</a> <a href="/wiki/%EB%B6%81%EB%8F%85%EC%9D%BC_%EC%97%B0%EB%B0%A9" title="북독일 연방">북독일 연방</a>에 가맹하였고, <a href="/wiki/1871%EB%85%84" title="1871년">1871년</a> <a href="/wiki/%EB%8F%85%EC%9D%BC_%EC%A0%9C%EA%B5%AD" title="독일 제국">독일 제국</a>의 일부가 되었다. <a href="/wiki/1918%EB%85%84" title="1918년">1918년</a> 자유주가 된 이후에 <a href="/wiki/%EB%B0%94%EC%9D%B4%EB%A7%88%EB%A5%B4_%EA%B3%B5%ED%99%94%EA%B5%AD" title="바이마르 공화국">바이마르 공화국</a>과 <a href="/wiki/%EB%82%98%EC%B9%98_%EB%8F%85%EC%9D%BC" title="나치 독일">나치 독일</a> 치하에서도 독립된 주로 유지하였으나, <a href="/wiki/%EC%A0%9C2%EC%B0%A8_%EC%84%B8%EA%B3%84_%EB%8C%80%EC%A0%84" title="제2차 세계 대전">제2차 세계 대전</a> 이후에 <a href="/wiki/%EB%8B%88%EB%8D%94%EC%9E%91%EC%84%BC_%EC%A3%BC" title="니더작센 주">니더작센 주</a>에 흡수되었다. 샤움부르크리페는 <a href="/wiki/%EB%8F%85%EC%9D%BC" title="독일">독일</a> 내에서 가장 작은 영방이었다.</p>
