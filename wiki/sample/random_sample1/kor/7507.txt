<h2><span class="mw-headline" id=".EA.B0.9C.EC.9A.94">개요</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%83%A4%ED%94%84%ED%95%98%EC%9A%B0%EC%A0%A0_%EC%A3%BC&amp;action=edit&amp;section=1" title="부분 편집: 개요">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>스위스 중북부에 위치하며, <a href="/wiki/%EB%8F%85%EC%9D%BC" title="독일">독일</a>과 국경을 접한다. 스위스 최북단의 주이며, 삼면이 독일에 둘러싸여 있다. 국경으로 <a href="/wiki/%EB%9D%BC%EC%9D%B8_%EA%B0%95" title="라인 강">라인 강</a>이 흐르며, 주변은 완만한 구릉지역이다. <a href="/wiki/%EC%9C%A0%EB%9F%BD" title="유럽">유럽</a> 최대의 <a href="/wiki/%ED%8F%AD%ED%8F%AC" title="폭포">폭포</a>인 <a href="/wiki/%EB%9D%BC%EC%9D%B8_%ED%8F%AD%ED%8F%AC" title="라인 폭포">라인 폭포</a>가 이 곳에 있다. 동쪽 일부는 <a href="/wiki/%EB%B3%B4%EB%8D%B4_%ED%98%B8" title="보덴 호">보덴 호</a>에 접한다. <a href="/wiki/11%EC%84%B8%EA%B8%B0" title="11세기">11세기</a>에 <a class="mw-redirect" href="/wiki/%EB%B2%A0%EB%84%A4%EB%94%95%ED%86%A0%ED%9A%8C" title="베네딕토회">베네딕토회</a>의 <a href="/wiki/%EC%88%98%EB%8F%84%EC%9B%90" title="수도원">수도원</a>이 생겨 수도원을 중심으로 발전하기 시작했다. <a href="/wiki/1415%EB%85%84" title="1415년">1415년</a> 이 부근에서 열린 <a href="/wiki/%EC%BD%98%EC%8A%A4%ED%83%84%EC%B8%A0_%EA%B3%B5%EC%9D%98%ED%9A%8C" title="콘스탄츠 공의회">콘스탄츠 공의회</a> 이후 <a class="mw-redirect" href="/wiki/%ED%95%A9%EC%8A%A4%EB%B6%80%EB%A5%B4%ED%81%AC_%EC%99%95%EA%B0%80" title="합스부르크 왕가">합스부르크 왕가</a>로부터 독립했다. <a href="/wiki/1451%EB%85%84" title="1451년">1451년</a> 스위스 동맹과 제휴했고, <a href="/wiki/1501%EB%85%84" title="1501년">1501년</a> 동맹에 가입했다. <a href="/wiki/%EC%A2%85%EA%B5%90_%EA%B0%9C%ED%98%81" title="종교 개혁">종교 개혁</a> 후 <a href="/wiki/%EA%B0%9C%EC%8B%A0%EA%B5%90" title="개신교">개신교</a> 지역으로 바뀌었고, <a href="/wiki/1876%EB%85%84" title="1876년">1876년</a> 주 헌법이 제정되었다. 처음 <a href="/wiki/%EC%83%A4%ED%94%84%ED%95%98%EC%9A%B0%EC%A0%A0" title="샤프하우젠">샤프하우젠</a>에서 영역을 넓힌 곳이 불규칙하여 주는 독일에 둘러싸여 있고, 일부 지역은 <a href="/wiki/%EC%9B%94%EA%B2%BD%EC%A7%80" title="월경지">월경지</a>로 분리되어 있다.</p>
<p><a href="/wiki/%ED%8F%AC%EB%8F%84" title="포도">포도</a>가 재배되며, <a href="/wiki/%EB%B0%B1%ED%8F%AC%EB%8F%84%EC%A3%BC" title="백포도주">백포도주</a> 제조로 유명하다. <a href="/wiki/%EC%8B%9C%EA%B3%84" title="시계">시계</a>·<a href="/wiki/%EA%B8%88%EC%86%8D" title="금속">금속</a>·<a href="/wiki/%EA%B8%B0%EA%B3%84" title="기계">기계</a> 공업이 이루어진다. <a href="/wiki/%EB%8F%85%EC%9D%BC%EC%96%B4" title="독일어">독일어</a>가 널리 쓰이며, <a href="/wiki/%EA%B0%9C%EC%8B%A0%EA%B5%90" title="개신교">개신교</a> 신자가 많다.</p>
