<div class="thumbinner" style="width:182px;"><a class="new" href="/wiki/위키백과:파일_올리기?wpDestFile=Tridevi.png" title="파일:Tridevi.png">파일:Tridevi.png</a>
<div class="thumbcaption">삭티파는 <a href="/wiki/%ED%9E%8C%EB%91%90%EA%B5%90" title="힌두교">힌두교</a>의 신성한 어머니인 <a href="/wiki/%EC%83%A5%ED%8B%B0" title="샥티">샥티</a>를 숭배한다 - 이 그림에서 <a class="new" href="/w/index.php?title=%EC%8B%A0%EC%84%B1%ED%95%9C_%EC%96%B4%EB%A8%B8%EB%8B%88&amp;action=edit&amp;redlink=1" title="신성한 어머니 (없는 문서)">신성한 어머니</a>는 오른쪽의 <a class="mw-redirect" href="/wiki/%EC%82%AC%EB%9D%BC%EC%8A%A4%EC%99%80%ED%8B%B0" title="사라스와티">사라스와티</a> <b>·</b> 왼쪽의 <a href="/wiki/%EB%9D%BD%EC%8A%88%EB%AF%B8" title="락슈미">락슈미</a> <b>·</b> 가운데의 <a href="/wiki/%ED%8C%8C%EB%A5%B4%EB%B0%94%ED%8B%B0" title="파르바티">파르바티</a>의 세 여신이 결합된 존재인 "<a href="/wiki/%ED%8A%B8%EB%A6%AC%EB%8D%B0%EB%B9%84" title="트리데비">트리데비</a>(<a class="extiw" href="https://en.wikipedia.org/wiki/Tridevi" title="en:Tridevi"><span style="color: black">Tridevi</span></a>)"로 표현되어 있다</div>
</div>
</div>
<p><b>샥티파</b>(<span style="font-size: smaller;"><a href="/wiki/%EC%82%B0%EC%8A%A4%ED%81%AC%EB%A6%AC%ED%8A%B8%EC%96%B4" title="산스크리트어">산스크리트어</a>:</span> <span lang="sa" xml:lang="sa">शाक्तं</span> Śāktaṃ, <span style="font-size: smaller;"><a href="/wiki/%EC%98%81%EC%96%B4" title="영어">영어</a>:</span> <span lang="en" xml:lang="en"><a class="extiw" href="https://en.wikipedia.org/wiki/Shaktism" title="en:Shaktism"><span style="color: black">Shaktism</span></a></span>)는 <a href="/wiki/%ED%9E%8C%EB%91%90%EA%B5%90" title="힌두교">힌두교</a>의 <a class="new" href="/w/index.php?title=%EC%8B%A0%EC%84%B1%ED%95%9C_%EC%96%B4%EB%A8%B8%EB%8B%88&amp;action=edit&amp;redlink=1" title="신성한 어머니 (없는 문서)">신성한 어머니</a>(<a class="extiw" href="https://en.wikipedia.org/wiki/Divine_Mother" title="en:Divine Mother"><span style="color: black">Divine Mother</span></a>)인 <a class="new" href="/w/index.php?title=%EC%95%84%EB%94%94_%ED%8C%8C%EB%9D%BC%EC%83%A5%ED%8B%B0&amp;action=edit&amp;redlink=1" title="아디 파라샥티 (없는 문서)">아디 파라샥티</a>(<a class="extiw" href="https://en.wikipedia.org/wiki/Adi_parashakti" title="en:Adi parashakti"><span style="color: black">Adi parashakti</span></a>) 또는 <a href="/wiki/%EC%83%A5%ED%8B%B0" title="샥티">샥티</a>(<a class="extiw" href="https://en.wikipedia.org/wiki/Shakti" title="en:Shakti"><span style="color: black">Shakti</span></a>) 또는 <a href="/wiki/%EB%8D%B0%EB%B9%84" title="데비">데비</a>(<a class="extiw" href="https://en.wikipedia.org/wiki/Devi" title="en:Devi"><span style="color: black">Devi</span></a>)를 절대적 또는 궁극적 신성으로 숭배하는 <a href="/wiki/%ED%9E%8C%EB%91%90%EA%B5%90_%EC%A2%85%ED%8C%8C" title="힌두교 종파">종파</a>이다. 샥티파의 <a href="/wiki/%EC%82%B0%EC%8A%A4%ED%81%AC%EB%A6%AC%ED%8A%B8%EC%96%B4" title="산스크리트어">산스크리트어</a> 명칭인 <b>샥탐</b>(Śāktaṃ)의 문자 그대로의 뜻은 "(신의) 권능에 대한 가르침" 또는 <span class="nowrap">"여신에</span> 대한 가르침"이다. 샥티파는 <a href="/wiki/%EB%B9%84%EC%8A%88%EB%88%84%ED%8C%8C" title="비슈누파">비슈누파</a>(<a class="extiw" href="https://en.wikipedia.org/wiki/Vaishnavism" title="en:Vaishnavism"><span style="color: black">Vaishnavism</span></a>)와 <a href="/wiki/%EC%8B%9C%EB%B0%94%ED%8C%8C" title="시바파">시바파</a>(<a class="extiw" href="https://en.wikipedia.org/wiki/Shaivism" title="en:Shaivism"><span style="color: black">Shaivism</span></a>)와 더불어 <a href="/wiki/%ED%9E%8C%EB%91%90%EA%B5%90" title="힌두교">힌두교</a>의 주요 세 종파들 중의 하나이다. 여기서 종파는 신상을 예배하는 등의 종교적인 성격을 띠는 힌두교 분파를 뜻하는 것으로 <a class="mw-redirect" href="/wiki/%ED%9E%8C%EB%91%90%EA%B5%90_%EC%B2%A0%ED%95%99_%ED%95%99%ED%8C%8C" title="힌두교 철학 학파">힌두교 철학 학파</a>와는 구별할 필요가 있다.</p>
<p>샥티파의 신자들 또는 수행자들을 <b>샥티교인</b> 또는 <b>샥티교도</b>라고 하는데, <a href="/wiki/%EC%82%B0%EC%8A%A4%ED%81%AC%EB%A6%AC%ED%8A%B8%EC%96%B4" title="산스크리트어">산스크리트어</a>로는 <b>샥타</b>(<span style="font-size: smaller;"><a href="/wiki/%EC%82%B0%EC%8A%A4%ED%81%AC%EB%A6%AC%ED%8A%B8%EC%96%B4" title="산스크리트어">산스크리트어</a>:</span> <span lang="sa" xml:lang="sa">शाक्त</span>, Śākta, <span style="font-size: smaller;"><a href="/wiki/%EC%98%81%EC%96%B4" title="영어">영어</a>:</span> <span lang="en" xml:lang="en">Shakta</span>)라고 한다.</p>
<p>샥티교에서 <a class="new" href="/w/index.php?title=%EC%8B%A0%EC%84%B1%ED%95%9C_%EC%96%B4%EB%A8%B8%EB%8B%88&amp;action=edit&amp;redlink=1" title="신성한 어머니 (없는 문서)">신성한 어머니</a>인 <a href="/wiki/%EC%83%A5%ED%8B%B0" title="샥티">샥티</a>는 <a class="mw-redirect" href="/wiki/%EC%82%AC%EB%9D%BC%EC%8A%A4%EC%99%80%ED%8B%B0" title="사라스와티">사라스와티</a>(<a class="extiw" href="https://en.wikipedia.org/wiki/Saraswati" title="en:Saraswati"><span style="color: black">Saraswati</span></a>) <b>·</b> <a href="/wiki/%EB%9D%BD%EC%8A%88%EB%AF%B8" title="락슈미">락슈미</a>(<a class="extiw" href="https://en.wikipedia.org/wiki/Lakshmi" title="en:Lakshmi"><span style="color: black">Lakshmi</span></a>) <b>·</b> <a href="/wiki/%ED%8C%8C%EB%A5%B4%EB%B0%94%ED%8B%B0" title="파르바티">파르바티</a>(<a class="extiw" href="https://en.wikipedia.org/wiki/Parvati" title="en:Parvati"><span style="color: black">Parvati</span></a>)의 세 여신이 결합된 존재인 "<a href="/wiki/%ED%8A%B8%EB%A6%AC%EB%8D%B0%EB%B9%84" title="트리데비">트리데비</a>(<a class="extiw" href="https://en.wikipedia.org/wiki/Tridevi" title="en:Tridevi"><span style="color: black">Tridevi</span></a>)"로 표현되기도 한다.</p>
