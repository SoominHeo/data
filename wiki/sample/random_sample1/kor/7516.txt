더 좋은 문서가 되도록 <span class="plainlinks"><a class="external text" href="//ko.wikipedia.org/w/index.php?title=%EC%83%B4%EB%AC%B4%EB%9D%BC%EB%A7%88%ED%8A%B8&amp;action=edit">문서 수정을 도와주세요</a></span>. 내용에 대한 의견이 있으시다면 <a href="/wiki/%ED%86%A0%EB%A1%A0:%EC%83%B4%EB%AC%B4%EB%9D%BC%EB%A7%88%ED%8A%B8" title="토론:샴무라마트">토론 문서</a>에서 나누어 주세요.</span> <small>(2008년 2월 22일)</small></span></td>

<p><b>샤무-라마트</b>(Shammu-ramat, 집정관, 재위 <a class="mw-redirect" href="/wiki/%EA%B8%B0%EC%9B%90%EC%A0%84_811%EB%85%84" title="기원전 811년">기원전 811년</a> ~ <a class="mw-redirect" href="/wiki/%EA%B8%B0%EC%9B%90%EC%A0%84_808%EB%85%84" title="기원전 808년">808년</a>)는 <a class="mw-redirect" href="/wiki/%EC%95%84%EC%8B%9C%EB%A6%AC%EC%95%84_%EC%A0%9C%EA%B5%AD" title="아시리아 제국">아시리아 제국</a>의 전설상의 여왕이었다. 그녀의 모델은 기원전 9세기 아시리아 제국의 왕 <a class="new" href="/w/index.php?title=%EC%83%B4%EC%8B%9C-%EC%95%84%EB%8B%A4%EB%93%9C_5%EC%84%B8&amp;action=edit&amp;redlink=1" title="샴시-아다드 5세 (없는 문서)">샴시-아다드 5세</a>의 왕비로 그녀의 왕자 <a href="/wiki/%EC%95%84%EB%8B%A4%EB%93%9C-%EB%8B%88%EB%9D%BC%EB%A6%AC_3%EC%84%B8" title="아다드-니라리 3세">아다드-니라리 3세</a>의 섭정을 하였던 <b>샤무라마트</b>로, <a href="/wiki/%ED%82%AC%EB%A6%AC%ED%82%A4%EC%95%84" title="킬리키아">킬리키아</a>로 건너가 <a href="/wiki/%EC%84%B8%EB%AF%B8%EB%9D%BC%EB%AF%B8%EC%8A%A4" title="세미라미스">세미라미스</a>가 되었다.</p>
<p>전설에 따르면, 그녀는 재색을 겸비하였고, 사치와 허영을 좋아하여 호색하며 잔학무도하였다고도 전해진다. 그녀는 <a class="mw-redirect" href="/wiki/%EC%84%B8%EA%B3%84%EC%9D%98_7%EB%8C%80_%EB%B6%88%EA%B0%80%EC%82%AC%EC%9D%98" title="세계의 7대 불가사의">세계의 7대 불가사의</a> 중 하나인 <a class="mw-redirect" href="/wiki/%EB%B0%94%EB%B9%8C%EB%A1%A0%EC%9D%98_%EA%B3%B5%EC%A4%91%EC%A0%95%EC%9B%90" title="바빌론의 공중정원">바빌론의 공중정원</a>을 만들었다고 한다.</p>
<p>시리아의 여신 <a class="new" href="/w/index.php?title=%EB%8D%B0%EB%A5%B4%EC%BC%80%ED%8A%B8&amp;action=edit&amp;redlink=1" title="데르케트 (없는 문서)">데르케트</a>와 어느 시리아인과의 사이에서 태어났지만, 어려서부터 버림을 받아, 비둘기에게 교육을 받았다. 성장한 후, <a href="/wiki/%EB%8B%88%EB%84%A4%EB%B2%A0" title="니네베">니네베</a>의 수도를 건축했던 아시리아 왕 <a href="/wiki/%EB%8B%88%EB%88%84%EC%8A%A4" title="니누스">니누스</a>의 총애를 받아 왕자 니뉴아스를 낳았다.</p>
<p>그러나 최후에는 니뉴아스에게 배반당하게 되었는데, 신탁을 보다 니뉴아스의 모반을 알고 그에게 권력을 이양하고 비둘기가 되어 승천하였다고 전해 내려온다. 향년 62세, 섭정으로서재위는 42년에 달하였다.</p>
<p>또, “니누스 왕은 세미라미스가 독살하였다”라는 전설이 남아 있으며, 이것은 전설에 남아 있는 가장 오래된 독살 사건 가운데 하나이다.</p>
<p>그리하여 세미라미스 전설은 유럽의 연극과 가극의 소재로 즐겨 사용되었다.</p>
