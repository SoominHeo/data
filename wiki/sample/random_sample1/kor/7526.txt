<p><b>서강대역</b>(西江大驛)은 <a href="/wiki/%EC%84%9C%EC%9A%B8%ED%8A%B9%EB%B3%84%EC%8B%9C" title="서울특별시">서울특별시</a> <a href="/wiki/%EB%A7%88%ED%8F%AC%EA%B5%AC" title="마포구">마포구</a> <a href="/wiki/%EC%84%9C%EA%B5%90%EB%8F%99_(%EC%84%9C%EC%9A%B8)" title="서교동 (서울)">노고산동</a>에 위치한 <a href="/wiki/%EC%9A%A9%EC%82%B0%EC%84%A0" title="용산선">용산선</a>의 <a href="/wiki/%EC%A0%84%EC%B2%A0%EC%97%AD" title="전철역">전철역</a>이다. 처음 역명은 <b>서강역</b>이었으며 <a href="/wiki/1929%EB%85%84" title="1929년">1929년</a>에 개업하여 지상에 있던 역이었으나, <a href="/wiki/2012%EB%85%84" title="2012년">2012년</a> <a href="/wiki/12%EC%9B%94_15%EC%9D%BC" title="12월 15일">12월 15일</a>에 <a href="/wiki/%EC%88%98%EB%8F%84%EA%B6%8C_%EC%A0%84%EC%B2%A0_%EA%B2%BD%EC%9D%98%EC%84%A0" title="수도권 전철 경의선">수도권 전철 경의중앙선</a> 복선전철 개통과 함께 지하역으로 바뀌었다.</p>

<h2><span class="mw-headline" id=".ED.8A.B9.EC.A7.95">특징</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%84%9C%EA%B0%95%EB%8C%80%EC%97%AD&amp;action=edit&amp;section=1" title="부분 편집: 특징">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p><a href="/wiki/1929%EB%85%84" title="1929년">1929년</a>에 영업을 시작하였으나 <a href="/wiki/1975%EB%85%84" title="1975년">1975년</a> 이후에는 철도 화물만 취급하던 화물 전용 역으로 존재하였다. <a href="/wiki/2005%EB%85%84" title="2005년">2005년</a>에 <a href="/wiki/%EA%B2%BD%EC%9D%98%EC%84%A0" title="경의선">경의중앙선</a> <a href="/wiki/%EB%B3%B5%EC%84%A0_(%EC%B2%A0%EB%8F%84)" title="복선 (철도)">복선</a> 전철화와 <a href="/wiki/%EC%9D%B8%EC%B2%9C%EA%B5%AD%EC%A0%9C%EA%B3%B5%ED%95%AD%EC%B2%A0%EB%8F%84" title="인천국제공항철도">인천국제공항철도</a> 공사를 위하여 <a href="/wiki/%EC%9A%A9%EC%82%B0%EC%84%A0" title="용산선">용산선</a>이 철거되면서 영업이 중지되었다. <a href="/wiki/2012%EB%85%84" title="2012년">2012년</a> <a href="/wiki/12%EC%9B%94_15%EC%9D%BC" title="12월 15일">12월 15일</a> <a class="mw-redirect" href="/wiki/%EC%88%98%EB%8F%84%EA%B6%8C_%EC%A0%84%EC%B2%A0_%EA%B2%BD%EC%9D%98%EC%A4%91%EC%95%99%EC%84%A0" title="수도권 전철 경의중앙선">수도권 전철 경의중앙선</a>의 <a href="/wiki/%EC%9A%A9%EC%82%B0%EC%84%A0" title="용산선">용산선</a> 구간이 <a href="/wiki/%EA%B3%B5%EB%8D%95%EC%97%AD" title="공덕역">공덕역</a>까지 1단계로 개통되면서, 지하로 이설하여 영업을 재개하였다. 동시에 본래 지상역이 있었던 <a href="/wiki/%EC%8B%A0%EC%88%98%EB%8F%99_(%EC%84%9C%EC%9A%B8)" title="신수동 (서울)">신수동</a>에서 <a href="/wiki/%EC%84%9C%EA%B5%90%EB%8F%99_(%EC%84%9C%EC%9A%B8)" title="서교동 (서울)">노고산동</a> 지하로 관할 구역이 변경되었다. <a href="/wiki/%EC%8B%A0%EC%B4%8C%EC%97%AD_(%EC%A7%80%ED%95%98)" title="신촌역 (지하)">2호선 신촌역</a>과는 환승되지 않지만, <a href="/wiki/%EC%8B%A0%EC%B4%8C%EC%97%AD_(%EC%A7%80%ED%95%98)" title="신촌역 (지하)">2호선 신촌역</a> 근처의 신촌임시공영주차장(구 신촌다주상가)에서 서강역이 바로 보인다. <a href="/wiki/2014%EB%85%84" title="2014년">2014년</a> <a href="/wiki/3%EC%9B%94_17%EC%9D%BC" title="3월 17일">3월 17일</a>부로 <b>서강대역</b>으로 역명을 변경하였다.</p>

<ul>
