<p><b>서경 18도</b>(西經18度)는 <a href="/wiki/%EB%B3%B8%EC%B4%88_%EC%9E%90%EC%98%A4%EC%84%A0" title="본초 자오선">본초 자오선</a>에서 서쪽으로 18도 떨어진 <a href="/wiki/%EA%B2%BD%EC%84%A0" title="경선">경선</a>이다. <a href="/wiki/%EB%B6%81%EA%B7%B9%EC%A0%90" title="북극점">북극점</a>에서 시작하여 <a href="/wiki/%EB%B6%81%EA%B7%B9%ED%95%B4" title="북극해">북극해</a>, <a href="/wiki/%EA%B7%B8%EB%A6%B0%EB%9E%80%EB%93%9C" title="그린란드">그린란드</a>, <a href="/wiki/%EC%95%84%EC%9D%B4%EC%8A%AC%EB%9E%80%EB%93%9C" title="아이슬란드">아이슬란드</a>, <a href="/wiki/%EC%B9%B4%EB%82%98%EB%A6%AC%EC%95%84_%EC%A0%9C%EB%8F%84" title="카나리아 제도">카나리아 제도</a>, <a href="/wiki/%EB%8C%80%EC%84%9C%EC%96%91" title="대서양">대서양</a>, <a href="/wiki/%EB%82%A8%EA%B7%B9%ED%95%B4" title="남극해">남극해</a>, <a href="/wiki/%EB%82%A8%EA%B7%B9" title="남극">남극</a>을 거쳐 <a href="/wiki/%EB%82%A8%EA%B7%B9%EC%A0%90" title="남극점">남극점</a>에 이른다.</p>
<p>서경 18도는 <a class="new" href="/w/index.php?title=%EB%8F%99%EA%B2%BD_162%EB%8F%84&amp;action=edit&amp;redlink=1" title="동경 162도 (없는 문서)">동경 162도</a>와 이어져 지구의 둘레를 이룬다.</p>
<p><a href="/wiki/%EB%B6%81%EA%B7%B9%EC%A0%90" title="북극점">북극점</a>에서 <a href="/wiki/%EB%82%A8%EA%B7%B9%EC%A0%90" title="남극점">남극점</a>까지 서경 18도선은 다음 지역을 지나간다.</p>
