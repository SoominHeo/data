<h1 class="firstHeading" id="firstHeading" lang="ko">서동시집 관현악단</h1>
<div class="mw-body-content" id="bodyContent">
<div id="siteSub">위키백과, 우리 모두의 백과사전.</div>
<div id="contentSub"></div>
<div class="mw-jump" id="jump-to-nav">
					이동:					<a href="#mw-head">둘러보기</a>, 					<a href="#p-search">검색</a>
</div>
<div class="mw-content-ltr" dir="ltr" id="mw-content-text" lang="ko"><script>function mfTempOpenSection(id){var block=document.getElementById("mf-section-"+id);block.className+=" open-block";block.previousSibling.className+=" open-block";}</script><p><b>서동시집 관현악단</b>(West-Eastern Divan Orchestra)은 <a href="/wiki/%EC%8A%A4%ED%8E%98%EC%9D%B8" title="스페인">스페인</a>의 <a href="/wiki/%EC%84%B8%EB%B9%84%EC%95%BC" title="세비야">세비야</a>를 거점으로 하고 있는 청소년 <a href="/wiki/%EA%B4%80%ED%98%84%EC%95%85%EB%8B%A8" title="관현악단">관현악단</a>으로, <a href="/wiki/%EB%A0%88%EB%B0%94%EB%85%BC" title="레바논">레바논</a>과 <a href="/wiki/%EC%8B%9C%EB%A6%AC%EC%95%84" title="시리아">시리아</a>, <a href="/wiki/%EC%9A%94%EB%A5%B4%EB%8B%A8" title="요르단">요르단</a>, <a href="/wiki/%EC%9D%B4%EB%9E%80" title="이란">이란</a>, <a href="/wiki/%EC%9D%B4%EC%8A%A4%EB%9D%BC%EC%97%98" title="이스라엘">이스라엘</a>, <a href="/wiki/%EC%9D%B4%EC%A7%91%ED%8A%B8" title="이집트">이집트</a>, <a href="/wiki/%EC%BF%A0%EC%9B%A8%EC%9D%B4%ED%8A%B8" title="쿠웨이트">쿠웨이트</a>, <a href="/wiki/%ED%8C%94%EB%A0%88%EC%8A%A4%ED%83%80%EC%9D%B8_%EC%9E%90%EC%B9%98_%EC%A0%95%EB%B6%80" title="팔레스타인 자치 정부">팔레스타인</a> 등 중동 지역과 스페인의 청소년 연주자들이 단원으로 활동하고 있다. <a href="/wiki/1999%EB%85%84" title="1999년">1999년</a>에 <a href="/wiki/%EC%95%84%EB%A5%B4%ED%97%A8%ED%8B%B0%EB%82%98" title="아르헨티나">아르헨티나</a> 태생의 유태인 피아니스트 겸 지휘자 <a href="/wiki/%EB%8B%A4%EB%8B%88%EC%97%98_%EB%B0%94%EB%A0%8C%EB%B3%B4%EC%9E%84" title="다니엘 바렌보임">다니엘 바렌보임</a>과 팔레스타인 태생의 미국 영문학자 <a href="/wiki/%EC%97%90%EB%93%9C%EC%9B%8C%EB%93%9C_%EC%82%AC%EC%9D%B4%EB%93%9C" title="에드워드 사이드">에드워드 사이드</a>가 공동으로 창단했으며, 같은 해 <a href="/wiki/%EB%8F%85%EC%9D%BC" title="독일">독일</a>의 <a href="/wiki/%EB%B0%94%EC%9D%B4%EB%A7%88%EB%A5%B4" title="바이마르">바이마르</a>에서 첫 리허설과 오디션, 워크숍을 개최했다.</p>
<p>악단 명칭은 <a href="/wiki/%EC%9A%94%ED%95%9C_%EB%B3%BC%ED%94%84%EA%B0%95_%ED%8F%B0_%EA%B4%B4%ED%85%8C" title="요한 볼프강 폰 괴테">괴테</a>가 <a class="mw-disambig" href="/wiki/%ED%8E%98%EB%A5%B4%EC%8B%9C%EC%95%84" title="페르시아">페르시아</a> 시인 <a href="/wiki/%ED%95%98%ED%94%BC%EC%A6%88" title="하피즈">하피즈</a>의 번역시를 읽고 감명받아 집필했다는 '서동시집(Westöstlicher Divan)' 에서 따왔으며, 중동 지역의 젊고 유망한 연주자들을 양성함과 동시에 심각한 문제로 남아 있는 아랍-이스라엘 분쟁의 평화적인 해결을 모색한다는 것이 창단 의도로 발표되었다. 이에 따라 연주회는 이스라엘과 아랍 국가들을 위시해 전 세계를 돌며 진행하고 있으며, 수석 주자나 독주자의 경우에도 아랍인과 유태인 연주자를 골고루 배분해 기용·초빙하고 있다.</p>
<p><a href="/wiki/2002%EB%85%84" title="2002년">2002년</a>에는 스페인의 <a href="/wiki/%EC%95%88%EB%8B%AC%EB%A3%A8%EC%8B%9C%EC%95%84_%EC%A7%80%EB%B0%A9" title="안달루시아 지방">안달루시아 지방</a> 정부와 악단 운영을 지원하는 사설 재단들이 지원금을 모아 세비야에 활동 거점을 마련했으며, <a href="/wiki/2009%EB%85%84" title="2009년">2009년</a> 현재까지 상주하고 있다. 2년 뒤인 <a href="/wiki/2004%EB%85%84" title="2004년">2004년</a>에는 역시 세비야를 거점으로 창단자들의 이름을 딴 바렌보임-사이드 재단이 설립되었고, 악단 운영과 관현악 활동 강습회, 팔레스타인 청소년들을 위한 음악 교육 운동, 세비야 유소년 음악교육 계획 등의 사업을 후원하고 있다.</p>
<p>CD와 DVD는 전속사인 <a class="new" href="/w/index.php?title=%ED%85%94%EB%8D%B1&amp;action=edit&amp;redlink=1" title="텔덱 (없는 문서)">텔덱</a>을 비롯해 <a class="new" href="/w/index.php?title=%EC%9C%A0%EB%A1%9C%EC%95%84%EC%B8%A0&amp;action=edit&amp;redlink=1" title="유로아츠 (없는 문서)">유로아츠</a> 등에서 제작되고 있으며, 악단 활동상을 담은 다큐멘터리도 DVD와 함께 발매되고 있다. 창단 이래 200명 이상의 청소년 음악인들이 악단을 거쳐갔으며, 퇴단 후에도 모국의 음악원 등 음악교육 기관의 강사나 관현악단의 단원으로 활동하고 있다.</p>
