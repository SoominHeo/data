<p><a href="/wiki/%EC%84%9C%EC%9A%B8%ED%8A%B9%EB%B3%84%EC%8B%9C" title="서울특별시">서울특별시</a> 출생으로 <a class="mw-disambig" href="/wiki/%EC%98%81%EB%8F%84%EC%A4%91%ED%95%99%EA%B5%90" title="영도중학교">영도중학교</a>, <a href="/wiki/%EC%88%98%EB%8F%84%EC%A0%84%EA%B8%B0%EA%B3%B5%EC%97%85%EA%B3%A0%EB%93%B1%ED%95%99%EA%B5%90" title="수도전기공업고등학교">수도전기공업고등학교</a>, <a href="/wiki/%EC%97%B0%EC%84%B8%EB%8C%80%ED%95%99%EA%B5%90" title="연세대학교">연세대학교</a>를 졸업하였다. 날이 선 강력한 슈팅으로 골을 만들어 '캐넌 슈터'라는 별명을 얻었다.</p>
<h2><span class="mw-headline" id=".EC.B6.95.EA.B5.AC.EC.9D.B8_.EC.83.9D.ED.99.9C">축구인 생활</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%84%9C%EB%8F%99%EC%9B%90_(1975%EB%85%84)&amp;action=edit&amp;section=2" title="부분 편집: 축구인 생활">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<h3><span class="mw-headline" id=".EC.84.A0.EC.88.98_.EC.83.9D.ED.99.9C">선수 생활</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%84%9C%EB%8F%99%EC%9B%90_(1975%EB%85%84)&amp;action=edit&amp;section=3" title="부분 편집: 선수 생활">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p><a href="/wiki/1998%EB%85%84" title="1998년">1998년</a> <a href="/wiki/%EB%8C%80%EC%A0%84_%EC%8B%9C%ED%8B%B0%EC%A6%8C" title="대전 시티즌">대전 시티즌</a>에서 데뷔하였고, <a href="/wiki/2001%EB%85%84" title="2001년">2001년</a> <a href="/wiki/%EC%88%98%EC%9B%90_%EC%82%BC%EC%84%B1_%EB%B8%94%EB%A3%A8%EC%9C%99%EC%A6%88" title="수원 삼성 블루윙즈">수원 삼성 블루윙즈</a>로 옮겼지만,<sup class="reference" id="cite_ref-2"><a href="#cite_note-2">[2]</a></sup> 이렇다 할 활약을 보이지 못하며 그 해 <a href="/wiki/%EC%A0%84%EB%B6%81_%ED%98%84%EB%8C%80_%EB%AA%A8%ED%84%B0%EC%8A%A4" title="전북 현대 모터스">전북 현대 모터스</a>로 다시 이적하였다.<sup class="reference" id="cite_ref-3"><a href="#cite_note-3">[3]</a></sup> <a href="/wiki/2003%EB%85%84" title="2003년">2003년</a> <a class="mw-redirect" href="/wiki/%EA%B4%91%EC%A3%BC_%EC%83%81%EB%AC%B4_%EB%B6%88%EC%82%AC%EC%A1%B0" title="광주 상무 불사조">광주 상무 불사조</a>에 입대해 <a href="/wiki/2005%EB%85%84" title="2005년">2005년</a> 군 복무를 마치고 <a href="/wiki/%EC%A0%84%EB%B6%81_%ED%98%84%EB%8C%80_%EB%AA%A8%ED%84%B0%EC%8A%A4" title="전북 현대 모터스">전북 현대 모터스</a>에 복귀하였고, <a href="/wiki/%EC%9D%B8%EC%B2%9C_%EC%9C%A0%EB%82%98%EC%9D%B4%ED%8B%B0%EB%93%9C_FC" title="인천 유나이티드 FC">인천 유나이티드</a>로 이적하여 그 해 <a class="mw-redirect" href="/wiki/K-%EB%A6%AC%EA%B7%B8" title="K-리그">K-리그</a> 준우승에 공헌하였다. <a href="/wiki/2006%EB%85%84" title="2006년">2006년</a> 7월 <a class="mw-redirect" href="/wiki/%EC%84%B1%EB%82%A8_%EC%9D%BC%ED%99%94_%EC%B2%9C%EB%A7%88" title="성남 일화 천마">성남 일화 천마</a>로 이적하여,<sup class="reference" id="cite_ref-4"><a href="#cite_note-4">[4]</a></sup> 그 해 <a class="mw-redirect" href="/wiki/K-%EB%A6%AC%EA%B7%B8" title="K-리그">K-리그</a> 우승 등에 공헌하였다. <a href="/wiki/2008%EB%85%84" title="2008년">2008년</a> 6월 <a href="/wiki/%EB%B6%80%EC%82%B0_%EC%95%84%EC%9D%B4%ED%8C%8C%ED%81%AC" title="부산 아이파크">부산 아이파크</a>로 이적하여,<sup class="reference" id="cite_ref-5"><a href="#cite_note-5">[5]</a></sup> <a href="/wiki/2010%EB%85%84" title="2010년">2010년</a>까지 활동했다.</p>
<h3><span class="mw-headline" id=".EA.B5.AD.EA.B0.80_.EB.8C.80.ED.91.9C_.EC.83.9D.ED.99.9C">국가 대표 생활</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%84%9C%EB%8F%99%EC%9B%90_(1975%EB%85%84)&amp;action=edit&amp;section=4" title="부분 편집: 국가 대표 생활">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p><a href="/wiki/1998%EB%85%84" title="1998년">1998년</a> <a href="/wiki/1%EC%9B%94_25%EC%9D%BC" title="1월 25일">1월 25일</a>, <a href="/wiki/%ED%83%80%EC%9D%B4" title="타이">타이</a> <a class="mw-disambig" href="/wiki/%ED%82%B9%EC%8A%A4%EC%BB%B5" title="킹스컵">킹스컵</a>에서 <a href="/wiki/%EB%8D%B4%EB%A7%88%ED%81%AC_%EC%B6%95%EA%B5%AC_%EA%B5%AD%EA%B0%80%EB%8C%80%ED%91%9C%ED%8C%80" title="덴마크 축구 국가대표팀">덴마크</a>와의 경기로 <a href="/wiki/A%EB%A7%A4%EC%B9%98" title="A매치">A매치</a> 데뷔전을 치렀다. <a href="/wiki/A%EB%A7%A4%EC%B9%98" title="A매치">A매치</a> 8경기에서 1골을 넣었다.</p>
<h2><span class="mw-headline" id=".ED.94.8C.EB.A0.88.EC.9D.B4_.EC.8A.A4.ED.83.80.EC.9D.BC">플레이 스타일</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%84%9C%EB%8F%99%EC%9B%90_(1975%EB%85%84)&amp;action=edit&amp;section=5" title="부분 편집: 플레이 스타일">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>강한 체력과 지구력을 바탕으로 미드필드 싸움을 주도하며 '살림꾼' 역할을 충실히 수행한다. 상황에 따라서는 스위퍼에서 최전방 공격수에 이르기까지 폭넓게 활용되는 '올라운드 플레이어' 역할을 맡기도 한다.</p>
<p>강하고 정교하여 문전을 향하는 날카로운 프리킥과 강력한 대포알 슈팅, 적재 적소에 볼을 배급하는 공격 지원 능력까지 갖추고 있어 상대에게 공격수보다 더 무서운 존재가 되기도 한다.</p>
