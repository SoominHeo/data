<h2><span class="mw-headline" id=".EC.97.B0.ED.98.81">연혁</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%84%9C%EB%93%A0%EC%96%B4%ED%83%9D&amp;action=edit&amp;section=1" title="부분 편집: 연혁">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<ul>
<li>클로즈 베타 테스트 : <a href="/wiki/2005%EB%85%84" title="2005년">2005년</a> <a href="/wiki/2%EC%9B%94_11%EC%9D%BC" title="2월 11일">2월 11일</a> ~ <a href="/wiki/3%EC%9B%94_17%EC%9D%BC" title="3월 17일">3월 17일</a> (오후 3시 ~ 11시)</li>
<li>정식 서비스 (<a href="/wiki/%EB%84%B7%EB%A7%88%EB%B8%94" title="넷마블">넷마블</a>) : <a href="/wiki/2005%EB%85%84" title="2005년">2005년</a> <a href="/wiki/8%EC%9B%94_23%EC%9D%BC" title="8월 23일">8월 23일</a> 오후 3시 ~ <a href="/wiki/2013%EB%85%84" title="2013년">2013년</a> <a href="/wiki/7%EC%9B%94_10%EC%9D%BC" title="7월 10일">7월 10일</a> 오후 11시 59분</li>
<li>정식 서비스 (<a href="/wiki/%EB%84%A5%EC%8A%A8" title="넥슨">넥슨</a>, <a href="/wiki/%EB%84%B7%EB%A7%88%EB%B8%94" title="넷마블">넷마블</a> 공동 퍼블리싱) : <a href="/wiki/2011%EB%85%84" title="2011년">2011년</a> <a href="/wiki/7%EC%9B%94_11%EC%9D%BC" title="7월 11일">7월 11일</a> ~ <a href="/wiki/2013%EB%85%84" title="2013년">2013년</a> <a href="/wiki/7%EC%9B%94_10%EC%9D%BC" title="7월 10일">7월 10일</a> 오후 1시 50분. <a href="/wiki/%EB%84%A5%EC%8A%A8" title="넥슨">넥슨</a> 포털에서의 서든어택 단독 서비스를 추진하려 하였으나, <a href="/wiki/CJ_E%26M" title="CJ E&amp;M">CJ E&amp;M</a>이 이용자 DB를 넘겨줄 수 없다는 입장을 밝혀 양 사의 갈등이 깊어지던 중, 극적으로 공동 퍼블리싱 계약이 성사되어 넥슨과 CJ E&amp;M이 2년 간 넥슨 포털과 넷마블에서의 공동 서비스를 시작했다.<sup class="reference" id="cite_ref-2"><a href="#cite_note-2">[2]</a></sup></li>
<li>정식 서비스 (<a href="/wiki/%EB%84%A5%EC%8A%A8" title="넥슨">넥슨</a>) : <a href="/wiki/2013%EB%85%84" title="2013년">2013년</a> <a href="/wiki/7%EC%9B%94_11%EC%9D%BC" title="7월 11일">7월 11일</a> ~. 넥슨 포털에서의 단독 서비스 개시.</li>
</ul>
<h3><span class="mw-headline" id=".EC.9D.BC.EB.B3.B8">일본</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%84%9C%EB%93%A0%EC%96%B4%ED%83%9D&amp;action=edit&amp;section=2" title="부분 편집: 일본">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<ul>
<li><a href="/wiki/2007%EB%85%84" title="2007년">2007년</a> <a href="/wiki/6%EC%9B%94_14%EC%9D%BC" title="6월 14일">6월 14일</a>, <a href="/wiki/%EB%84%A5%EC%8A%A8%EC%A7%80%ED%8B%B0" title="넥슨지티">넥슨지티</a>는 <a href="/wiki/%EC%9D%BC%EB%B3%B8" title="일본">일본</a>에서의 서비스를 개시하기 위해 일본 지부를 설립하고 발표회를 열었다.</li>
<li><a href="/wiki/6%EC%9B%94_15%EC%9D%BC" title="6월 15일">6월 15일</a>부터 클로즈 베타 테스터를 모집하여 <a href="/wiki/6%EC%9B%94_28%EC%9D%BC" title="6월 28일">6월 28일</a>부터 <a href="/wiki/7%EC%9B%94_17%EC%9D%BC" title="7월 17일">7월 17일</a>까지 클로즈 베타 테스트를 거친 뒤<sup class="reference" id="cite_ref-jp_announce_3-0"><a href="#cite_note-jp_announce-3">[3]</a></sup>, 퍼블리싱 업데이트와 함께 <a href="/wiki/7%EC%9B%94_26%EC%9D%BC" title="7월 26일">7월 26일</a>에 오픈 베타 서비스를 개시하였다<sup class="reference" id="cite_ref-jp_release_1-1"><a href="#cite_note-jp_release-1">[1]</a></sup>.</li>
</ul>
<h2><span class="mw-headline" id=".EB.85.BC.EB.9E.80">논란</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%84%9C%EB%93%A0%EC%96%B4%ED%83%9D&amp;action=edit&amp;section=3" title="부분 편집: 논란">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<ul>
<li>서든어택이 2012년 9월 20일에 출시한 신규맵 '알파존', '베타존', '감마존'이 미국 <a href="/wiki/%EB%B0%B8%EB%B8%8C_%EC%BD%94%ED%8D%BC%EB%A0%88%EC%9D%B4%EC%85%98" title="밸브 코퍼레이션">밸브 코퍼레이션</a> 사의 비디오 퍼즐 게임 《<a href="/wiki/%ED%8F%AC%ED%83%88_(%EA%B2%8C%EC%9E%84)" title="포탈 (게임)">포탈</a>》을 표절했다는 논란에 휩싸였다. 넥슨측은 '타 게임과 유사성 논란이 있어' 맵의 서비스를 중지하겠다고 밝혔다.<sup class="reference" id="cite_ref-4"><a href="#cite_note-4">[4]</a></sup></li>
</ul>
