<h2><span class="mw-headline" id=".ED.8A.B9.EC.A7.95">특징</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%84%9C%EC%95%84%EC%8B%9C%EC%95%84&amp;action=edit&amp;section=4" title="부분 편집: 특징">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>지형적으로 북부와 남부에 고원을 이룬 곳이 많으며, 그 사이에 <a class="new" href="/w/index.php?title=%EB%A9%94%EC%86%8C%ED%8C%8C%ED%83%80%EB%AF%B8%EC%95%84&amp;action=edit&amp;redlink=1" title="메소파타미아 (없는 문서)">메소파타미아</a>의 평야, 혹은 해안의 저지가 있다. 기온은 건조하며, 이라크의 7월 평균 온도는 약 36도나 되며, <a class="mw-redirect" href="/wiki/%EC%95%84%EB%9D%BC%EB%B9%84%EC%95%84" title="아라비아">아라비아</a>의 사막 지역은 그보다 더 덥다. 하지만, 겨울엔 온도가 비교적 낮아 고산지역 같은 곳에선 적설을 볼 수 있다. <a class="mw-redirect" href="/wiki/%EA%B0%95%EC%88%98%EB%9F%89" title="강수량">강수량</a>은 이라크로 따지면 연 140mm, 비가 많이 내리지 않아 사람이 사는 지역은 보통 강, 혹은 오아시스가 있는 곳이다. 그래서 옛날에는 <a href="/wiki/%ED%8B%B0%EA%B7%B8%EB%A6%AC%EC%8A%A4_%EA%B0%95" title="티그리스 강">티그리스 강</a>과 <a href="/wiki/%EC%9C%A0%ED%94%84%EB%9D%BC%ED%85%8C%EC%8A%A4_%EA%B0%95" title="유프라테스 강">유프라테스 강</a>이 매우 중요했다. 서아시아 지역 대부분은 건조지역이라 오랫동안 <a href="/wiki/%EC%9C%A0%EB%AA%A9" title="유목">유목</a>이 발달했으나, 지금은 국경이 생겨 나라를 옮겨다니기 힘들어 유목민의 수가 줄어들고 있다. 오아시스나 하천이 있는 데는 농업이 발달하기도 했으며, 지금은 건조 지역에 인공 수로로 물을 올려와 농사를 짓는다. 아라비아 사막은 석유가 많이 나 1980년대에는 이스라엘을 고립시키기 위해 <a class="mw-redirect" href="/wiki/%EC%84%9D%EC%9C%A0_%ED%8C%8C%EB%8F%99" title="석유 파동">석유 파동</a>을 일으키기도 했다. 사우디아라비아는 세계에서 석유가 가장 많이 나며, 그 주변 국가도 석유 수출국 10위 내에 든다.</p>
<p>서아시아는 지리적으로 동양과 서양의 사이에 있어 과거에 침략의 대상이었다. 그래서 <a class="mw-disambig" href="/wiki/%ED%8E%98%EB%A5%B4%EC%8B%9C%EC%95%84" title="페르시아">페르시아</a>와 <a class="mw-redirect" href="/wiki/%EC%95%8C%EB%A0%89%EC%82%B0%EB%93%9C%EB%A1%9C%EC%8A%A4_%EB%8C%80%EC%99%95" title="알렉산드로스 대왕">알렉산드로스 대왕</a>의 <a class="mw-redirect" href="/wiki/%EB%A7%88%EC%BC%80%EB%8F%84%EB%8B%88%EC%95%84_%EC%A0%9C%EA%B5%AD" title="마케도니아 제국">마케도니아 제국</a>, <a href="/wiki/%EB%B0%94%EB%B9%8C%EB%A1%A0" title="바빌론">바빌론</a> 같은 거대한 제국은 서아시아에 있었다. 이러한 이유로 서아시아는 역사가 매우 복잡하다. 서아시아는 이스라엘을 제외하고는 모두 <a href="/wiki/%EC%9D%B4%EC%8A%AC%EB%9E%8C%EA%B5%90" title="이슬람교">이슬람교</a>를 믿는 아랍 국가다. 따라서 이스라엘은 제외하곤 공용어가 <a href="/wiki/%EC%95%84%EB%9E%8D%EC%96%B4" title="아랍어">아랍어</a>다. (이스라엘은 <a href="/wiki/%ED%9E%88%EB%B8%8C%EB%A6%AC%EC%96%B4" title="히브리어">히브리어</a>와 아랍어 둘 다 사용) 하지만, 이스라엘의 <a href="/wiki/%EC%98%88%EB%A3%A8%EC%82%B4%EB%A0%98" title="예루살렘">예루살렘</a>은 <a href="/wiki/%EC%9C%A0%EB%8C%80%EA%B5%90" title="유대교">유대교</a>, <a href="/wiki/%EA%B8%B0%EB%8F%85%EA%B5%90" title="기독교">기독교</a>, <a href="/wiki/%EC%9D%B4%EC%8A%AC%EB%9E%8C%EA%B5%90" title="이슬람교">이슬람교</a>의 공동성지다. 유대교는 <a href="/wiki/%ED%86%B5%EA%B3%A1%EC%9D%98_%EB%B2%BD" title="통곡의 벽">통곡의 벽</a>이, 기독교는 <a class="mw-redirect" href="/wiki/%EC%84%B1%EB%AC%98_%EA%B5%90%ED%9A%8C" title="성묘 교회">성묘 교회</a>, 이슬람교는 <a class="new" href="/w/index.php?title=%EB%B0%94%EC%9C%84_%EB%8F%94_%EC%82%AC%EC%9B%90&amp;action=edit&amp;redlink=1" title="바위 돔 사원 (없는 문서)">바위 돔 사원</a>(<a class="new" href="/w/index.php?title=%EC%9A%B0%EB%A7%88%EB%A5%B4_%EC%82%AC%EC%9B%90&amp;action=edit&amp;redlink=1" title="우마르 사원 (없는 문서)">우마르 사원</a>)이 있기 때문이다. <a href="/wiki/%EC%82%AC%EC%9A%B0%EB%94%94%EC%95%84%EB%9D%BC%EB%B9%84%EC%95%84" title="사우디아라비아">사우디아라비아</a> <a href="/wiki/%EB%A9%94%EC%B9%B4" title="메카">메카</a>와 <a href="/wiki/%EB%A9%94%EB%94%94%EB%82%98" title="메디나">메디나</a>는 이슬람 3대 성지라 예루살렘 바위 돔 사원과 같이 매년 수많은 아랍인이 순례를 다녀온다.</p>
<h2><span class="mw-headline" id=".EC.9E.90.EC.97.B0.ED.99.98.EA.B2.BD">자연환경</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%84%9C%EC%95%84%EC%8B%9C%EC%95%84&amp;action=edit&amp;section=5" title="부분 편집: 자연환경">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>지형은 대부분 사막으로 이루어져 있으며 대표적인 사막은 <a href="/wiki/%EB%A3%B9%EC%95%8C%ED%95%A0%EB%A6%AC_%EC%82%AC%EB%A7%89" title="룹알할리 사막">룹알할리 사막</a>이 있다. 강으로는 <a href="/wiki/%EC%9C%A0%ED%94%84%EB%9D%BC%ED%85%8C%EC%8A%A4_%EA%B0%95" title="유프라테스 강">유프라테스 강</a>, <a href="/wiki/%ED%8B%B0%EA%B7%B8%EB%A6%AC%EC%8A%A4_%EA%B0%95" title="티그리스 강">티그리스 강</a> 등의 강이 있고 산맥은 <a class="new" href="/w/index.php?title=%EC%88%A0%EB%9D%BC%EC%9D%B4%EB%A7%8C_%EC%82%B0%EB%A7%A5&amp;action=edit&amp;redlink=1" title="술라이만 산맥 (없는 문서)">술라이만 산맥</a>, <a class="new" href="/w/index.php?title=%EC%BD%94%ED%8E%98%ED%8A%B8_%EC%82%B0%EB%A7%A5&amp;action=edit&amp;redlink=1" title="코페트 산맥 (없는 문서)">코페트 산맥</a> 등이 있다. 기후는 주로 <a class="mw-redirect" href="/wiki/%EA%B1%B4%EC%A1%B0%EA%B8%B0%ED%9B%84" title="건조기후">건조기후</a>이며 <a class="mw-redirect" href="/wiki/%EC%A7%80%EC%A4%91%ED%95%B4_%EC%A7%80%EC%97%AD" title="지중해 지역">지중해 지역</a>은 <a href="/wiki/%EC%A7%80%EC%A4%91%ED%95%B4%EC%84%B1_%EA%B8%B0%ED%9B%84" title="지중해성 기후">지중해성 기후</a>를 띤다. 대부분의 지역의 연평균 강수량은 50mm이하이며 <a href="/wiki/%EB%A9%94%EC%86%8C%ED%8F%AC%ED%83%80%EB%AF%B8%EC%95%84" title="메소포타미아">메소포타미아</a>와 유프라테스 강과 티그리스 강 근처에는 약 250mm~700mm가 이 지역의 연평균 강수량이다.</p>
