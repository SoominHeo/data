<h1 class="firstHeading" id="firstHeading" lang="ko">서양의 밀교</h1>
<div class="mw-body-content" id="bodyContent">
<div id="siteSub">위키백과, 우리 모두의 백과사전.</div>
<div id="contentSub"></div>
<div class="mw-jump" id="jump-to-nav">
					이동:					<a href="#mw-head">둘러보기</a>, 					<a href="#p-search">검색</a>
</div>
<div class="mw-content-ltr" dir="ltr" id="mw-content-text" lang="ko"><script>function mfTempOpenSection(id){var block=document.getElementById("mf-section-"+id);block.className+=" open-block";block.previousSibling.className+=" open-block";}</script><p><b>서양의 밀교</b>(西洋의 密敎 <b>·</b> Western esotericism) 또는 <b>서양의 비교</b>(西洋의 秘敎)라는 낱말의 문자 그대로의 뜻은 "<a class="mw-redirect" href="/wiki/%EC%84%9C%EC%96%91_%EC%84%B8%EA%B3%84" title="서양 세계">서양 세계</a>의 <a href="/wiki/%EB%B0%80%EA%B5%90" title="밀교">비밀한 또는 내적인 가르침</a>(Secret or esoteric teachings of the Western world)"이다. 서양의 밀교는 <a class="mw-redirect" href="/wiki/%EC%84%9C%EC%96%91_%EC%84%B8%EA%B3%84" title="서양 세계">서양 세계</a>의 <a class="mw-redirect" href="/wiki/%EC%8B%A0%EB%B9%84_%EC%A2%85%EA%B5%90" title="신비 종교">신비적</a> <b>·</b> <a href="/wiki/%EC%8B%A0%EB%B9%84%EC%A3%BC%EC%9D%98" title="신비주의">신비주의적</a> <b>·</b> <a href="/wiki/%EB%B0%80%EA%B5%90" title="밀교">밀교적</a> 지식의 총체를 일컫는 낱말이며, 또한, <a href="/wiki/%EC%84%9C%EC%96%91" title="서양">서양</a> 사회에서 발견되는, 이러한 지식을 중요시하는 여러 <a href="/wiki/%EC%98%81%EC%84%B1" title="영성">영적</a>인 전통들을 총괄하여 일컫는 낱말이다.</p>
<p>서양의 밀교에 포함되는 것들로는 <a class="mw-redirect" href="/wiki/%EA%B3%A0%EB%8C%80" title="고대">고대</a> <a href="/wiki/%EA%B3%A0%EB%8C%80_%EC%9D%B4%EC%A7%91%ED%8A%B8" title="고대 이집트">이집트</a> <b>·</b> <a href="/wiki/%EA%B3%A0%EB%8C%80_%EA%B7%B8%EB%A6%AC%EC%8A%A4" title="고대 그리스">그리스</a> <b>·</b> <a href="/wiki/%EA%B3%A0%EB%8C%80_%EB%A1%9C%EB%A7%88" title="고대 로마">로마</a>의 <a class="mw-redirect" href="/wiki/%EC%8B%A0%EB%B9%84_%EC%A2%85%EA%B5%90" title="신비 종교">신비 종교들</a>(<a class="extiw" href="https://en.wikipedia.org/wiki/Mystery_religions" title="en:Mystery religions">Mystery Religions</a>), <a class="mw-redirect" href="/wiki/%ED%94%BC%ED%83%80%EA%B3%A0%EB%9D%BC%EC%8A%A4%EC%A3%BC%EC%9D%98" title="피타고라스주의">피타고라스주의</a>(<a class="extiw" href="https://en.wikipedia.org/wiki/Pythagoreanism" title="en:Pythagoreanism">Pythagoreanism</a>), <a class="mw-redirect" href="/wiki/%EB%82%98%EC%8A%A4%ED%8B%B0%EC%8B%9C%EC%A6%98" title="나스티시즘">나스티시즘</a>(<a href="/wiki/%EC%98%81%EC%A7%80%EC%A3%BC%EC%9D%98" title="영지주의">영지주의</a> <b>·</b> <a class="extiw" href="https://en.wikipedia.org/wiki/Gnosticism" title="en:Gnosticism">Gnosticism</a>), <a href="/wiki/%EC%97%B0%EA%B8%88%EC%88%A0" title="연금술">연금술</a>(<a class="extiw" href="https://en.wikipedia.org/wiki/Alchemy" title="en:Alchemy">Alchemy</a>), <a href="/wiki/%EC%A0%90%EC%84%B1%EC%88%A0" title="점성술">점성술</a>(<a class="extiw" href="https://en.wikipedia.org/wiki/Astrology" title="en:Astrology">Astrology</a>), <a href="/wiki/%EB%A7%88%EB%B2%95" title="마법">마법</a>(<a class="extiw" href="https://en.wikipedia.org/wiki/Magic" title="en:Magic">Magic</a>), <a class="new" href="/w/index.php?title=%EC%8B%A0%EC%84%B1_%EB%A7%88%EB%B2%95&amp;action=edit&amp;redlink=1" title="신성 마법 (없는 문서)">신성 마법</a>(<a class="extiw" href="https://en.wikipedia.org/wiki/Theurgy" title="en:Theurgy">Theurgy</a>), <a href="/wiki/%EC%8B%A0%EC%A7%80%ED%95%99" title="신지학">신지학</a>(<a class="extiw" href="https://en.wikipedia.org/wiki/Theosophy" title="en:Theosophy">Theosophy</a>), <a href="/wiki/%ED%97%A4%EB%A5%B4%EB%A9%94%EC%8A%A4%EC%A3%BC%EC%9D%98" title="헤르메스주의">헤르메스주의</a>(<a class="extiw" href="https://en.wikipedia.org/wiki/Hermeticism" title="en:Hermeticism">Hermeticism</a>), <a href="/wiki/%EC%B9%B4%EB%B0%9C%EB%9D%BC" title="카발라">카발라</a>(<a class="extiw" href="https://en.wikipedia.org/wiki/Kabbalah" title="en:Kabbalah">Kabbalah</a>), <a class="new" href="/w/index.php?title=%ED%97%88%EB%B8%8C%ED%95%99&amp;action=edit&amp;redlink=1" title="허브학 (없는 문서)">허브학</a>(<a class="mw-redirect" href="/wiki/%EC%95%BD%EC%B4%88%ED%95%99" title="약초학">약초학</a> <b>·</b> <a class="extiw" href="https://en.wikipedia.org/wiki/Herbalism" title="en:Herbalism">Herbalism</a>), <a href="/wiki/%EC%9E%A5%EB%AF%B8%EC%8B%AD%EC%9E%90%ED%9A%8C" title="장미십자회">장미십자회</a>(<a class="extiw" href="https://en.wikipedia.org/wiki/Rosicrucianism" title="en:Rosicrucianism">Rosicrucianism</a>), <a href="/wiki/%ED%94%84%EB%A6%AC%EB%A9%94%EC%9D%B4%EC%8A%A8" title="프리메이슨">프리메이슨</a>(<a class="extiw" href="https://en.wikipedia.org/wiki/Freemasonry" title="en:Freemasonry">Freemasonry</a>), <a class="mw-redirect" href="/wiki/%EC%8B%A0%EC%9D%B4%EA%B5%90%EC%A3%BC%EC%9D%98" title="신이교주의">신이교주의</a>(<a class="extiw" href="https://en.wikipedia.org/wiki/Neopaganism" title="en:Neopaganism">Neopaganism</a>) 등이 있으며, 이 보다 더 많은 가르침, 인물, 단체 또는 전통들이 있다. 서양의 밀교에는 유대교 신비주의인 <a href="/wiki/%EC%B9%B4%EB%B0%9C%EB%9D%BC" title="카발라">카발라</a>와 <a class="new" href="/w/index.php?title=%EA%B8%B0%EB%8F%85%EA%B5%90_%EC%8B%A0%EB%B9%84%EC%A3%BC%EC%9D%98&amp;action=edit&amp;redlink=1" title="기독교 신비주의 (없는 문서)">기독교 신비주의</a>(예를 들어, <a href="/wiki/%EC%95%BC%EC%BD%A5_%EB%B5%88%EB%A9%94" title="야콥 뵈메">야콥 뵈메</a>)와 같이 그 기원은 동양 세계에 있지만 서양 세계에서 받아들여 사용하게 된 것들도 함께 포함된다.</p>
<p>서양의 밀교 전통은, "내적 지식(inner knowledge)" 또는 "앎"(<a class="mw-redirect" href="/wiki/%EA%B7%B8%EB%85%B8%EC%8B%9C%EC%8A%A4" title="그노시스">그노시스</a>)에 강조를 준다는 점 외에는, 원천이 되는 특정한 하나의 가르침이 있거나 통일된 문헌이 있지 않으며 또한 언제나 내세우는 특정한 교의가 있지도 않다. 현재 서양에서는 자신들이 위의 밀교적 전통들의 부활 또는 연속이라고 생각하며 이들 전통들의 가르침을 행하는 많은 다양한 소규모의 단체들이 있다.</p>
<p><a href="/wiki/%ED%97%A4%EB%A5%B4%EB%A9%94%EC%8A%A4%EC%A3%BC%EC%9D%98" title="헤르메스주의">헤르메스주의</a>(<a class="extiw" href="https://en.wikipedia.org/wiki/Hermeticism" title="en:Hermeticism">Hermeticism</a>), 서양의 헤르메스 전통(Western Hermetic Tradition), 서양 신비주의(Western mysticism), 서양의 내적 전통(Western Inner Tradition), 서양 오컬트 전통(Western occult tradition), 서양의 신비 전통(Western mystery tradition) 등과 같은 낱말은 문맥에 따라 <b>서양의 밀교</b> 전체를 뜻하기도 한다.</p>
