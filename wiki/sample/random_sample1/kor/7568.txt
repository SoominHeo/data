홈구장으로 사용되었던 서울올림픽주경기장</div>
</div>

<p><b>서울 유나이티드</b>(<span lang="en" xml:lang="en">Seoul United Football Club</span>)는 <a href="/wiki/K3%EB%A6%AC%EA%B7%B8" title="K3리그">K3리그</a>에 속해있는 <a href="/wiki/%EB%8C%80%ED%95%9C%EB%AF%BC%EA%B5%AD" title="대한민국">대한민국</a>의 시민 <a href="/wiki/%EC%B6%95%EA%B5%AC" title="축구">축구</a>단이며 줄여서 <b>서유</b>라고도 부른다. <a href="/wiki/%EC%84%9C%EC%9A%B8%ED%8A%B9%EB%B3%84%EC%8B%9C" title="서울특별시">서울특별시</a>를 연고지로 하고 있고 <a href="/wiki/%EC%86%A1%ED%8C%8C%EA%B5%AC" title="송파구">송파구</a>의 <a href="/wiki/%EC%84%9C%EC%9A%B8%EC%98%AC%EB%A6%BC%ED%94%BD%EC%A3%BC%EA%B2%BD%EA%B8%B0%EC%9E%A5" title="서울올림픽주경기장">서울올림픽주경기장</a>을 홈 구장으로 사용했으며 <a href="/wiki/2011%EB%85%84" title="2011년">2011년</a>부터 <a href="/wiki/%EB%85%B8%EC%9B%90%EA%B5%AC" title="노원구">노원구</a>의 <a href="/wiki/%EB%A7%88%EB%93%A4%EC%8A%A4%ED%83%80%EB%94%94%EC%9B%80" title="마들스타디움">마들스타디움</a>을 홈 구장으로 사용하고 있다. <a href="/wiki/2001%EB%85%84" title="2001년">2001년</a>부터 창단 작업에 들어가 <a href="/wiki/2004%EB%85%84" title="2004년">2004년</a>에 법인을 설립하였고, <a href="/wiki/2007%EB%85%84" title="2007년">2007년</a> <a href="/wiki/3%EC%9B%94_31%EC%9D%BC" title="3월 31일">3월 31일</a>에 창단식을 거행하고 그해 시작된 <a class="mw-redirect" href="/wiki/%EC%B1%8C%EB%A6%B0%EC%A0%80%EC%8A%A4%EB%A6%AC%EA%B7%B8" title="챌린저스리그">K3리그</a>에 원년멤버로 참가하였다. 이 팀은 시민이 자발해서 만든 시민 구단이다. 팬 제일주의를 뜻하는 'Fans Come First'를 슬로건으로 사용한다. 팬들의 투자로 설립된 주식회사 형태의 구단으로 팬들이 구단의 지분을 직접 소유하고 있다. 기업이나 지자체가 아닌 팬들만의 출자로 정규 리그에 참가하는 구단이 설립된 것은 대한민국 축구 역사상 서울 유나이티드가 최초이다.</p>
<p><a href="/wiki/2008%EB%85%84" title="2008년">2008년</a> <a href="/wiki/K3%EB%A6%AC%EA%B7%B8" title="K3리그">K3리그</a> 창립 멤버이며, <a href="/wiki/2007%EB%85%84" title="2007년">2007년</a> <a href="/wiki/K3%EB%A6%AC%EA%B7%B8" title="K3리그">K3리그</a> 시범리그 우승 팀이다. 서울은 <a href="/wiki/K3%EB%A6%AC%EA%B7%B8" title="K3리그">K3리그</a> 팀으로서는 유일하게 2년 연속 <a href="/wiki/FA%EC%BB%B5_(%EB%8C%80%ED%95%9C%EB%AF%BC%EA%B5%AD)" title="FA컵 (대한민국)">FA컵</a> 예선 3라운드에 진출한 팀이고, 최초로 공식 경기에서 대학교 축구부를 격파한 전적이 있는 <a href="/wiki/K3%EB%A6%AC%EA%B7%B8" title="K3리그">K3리그</a> 팀이기도 하다. 서울은 <a href="/wiki/K3%EB%A6%AC%EA%B7%B8" title="K3리그">K3리그</a>에서 가장 많은 수의 기업들에게 후원을 받고 있는 팀이며 궁극적으로 <a href="/wiki/K%EB%A6%AC%EA%B7%B8" title="K리그">K리그</a> 승격을 목표로 하고 있다. <a href="/wiki/J3%EB%A6%AC%EA%B7%B8" title="J3리그">J3리그</a>의 <a href="/wiki/FC_%EB%A5%98%ED%81%90" title="FC 류큐">FC 류큐</a>와 자매결연 관계이기도 하다.</p>
<p><a href="/wiki/K%EB%A6%AC%EA%B7%B8_%EB%93%9D%EC%A0%90%EC%83%81" title="K리그 득점상">K리그 득점상</a> 수상자 출신인 <a href="/wiki/%EC%9E%84%EA%B7%BC%EC%9E%AC" title="임근재">임근재</a> 감독이 <a href="/wiki/2007%EB%85%84" title="2007년">2007년</a>부터 2시즌 간 초대감독을 맡았으며, 현재 감독은 10대 감독 <a href="/wiki/%EC%A1%B0%EB%8F%99%ED%98%84_(%EC%B6%95%EA%B5%AC%EC%9D%B8)" title="조동현 (축구인)">조동현</a> 감독이다.</p>
<p>팀의 상징은 <a href="/wiki/%EB%B0%B1%ED%98%B8" title="백호">백호</a>이며 홈 유니폼 색깔은 흑색과 백색의 스트라이프이다. 흑백의 스트라이프는 백호의 줄무늬를 의미하기도 한다.</p>
<p>성인팀 외에 구단 자체 인터넷 방송인 SUTV를 운영중이며, <a href="/wiki/%EC%A0%84%EA%B5%AD_%EC%B4%88%EC%A4%91%EA%B3%A0_%EC%B6%95%EA%B5%AC%EB%A6%AC%EA%B7%B8" title="전국 초중고 축구리그">전국 초중고 축구리그</a>에도 U-18 고등부 팀을 운영중이다.</p>
<p>스카 밴드인 <a href="/wiki/%ED%82%B9%EC%8A%A4%ED%84%B4_%EB%A3%A8%EB%94%94%EC%8A%A4%EC%B9%B4" title="킹스턴 루디스카">킹스턴 루디스카</a>와 풋볼 프리스타일 세계챔피온 <a href="/wiki/%EC%9A%B0%ED%9D%AC%EC%9A%A9" title="우희용">우희용</a> 등이 클럽 홍보대사로 활동하고 있고, 2007 시즌을 소재로 <a href="/wiki/%EC%84%9C%EC%9A%B8_%EC%9C%A0%EB%82%98%EC%9D%B4%ED%8B%B0%EB%93%9C,_%EC%9D%B4%EC%A0%9C_%EC%8B%9C%EC%9E%91%EC%9D%B4%EB%8B%A4" title="서울 유나이티드, 이제 시작이다">서울 유나이티드, 이제 시작이다</a>라는 다큐멘터리 영화가 제작되기도 했다.</p>
<p>2013년 들어 협동조합 형태로 구단을 재편하였으며, 프로축구 2부리그인 <a href="/wiki/K%EB%A6%AC%EA%B7%B8_%EC%B1%8C%EB%A6%B0%EC%A7%80" title="K리그 챌린지">K리그 챌린지</a> 참가를 시도하기도 했다.</p>

<h2><span class="mw-headline" id=".EC.97.AD.EC.82.AC">역사</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%84%9C%EC%9A%B8_%EC%9C%A0%EB%82%98%EC%9D%B4%ED%8B%B0%EB%93%9C_FC&amp;action=edit&amp;section=2" title="부분 편집: 역사">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<h3><span class="mw-headline" id=".EC.B0.BD.EB.8B.A8_.EC.9D.B4.EC.A0.84">창단 이전</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%84%9C%EC%9A%B8_%EC%9C%A0%EB%82%98%EC%9D%B4%ED%8B%B0%EB%93%9C_FC&amp;action=edit&amp;section=3" title="부분 편집: 창단 이전">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p><a href="/wiki/%ED%95%9C%EA%B5%AD%ED%94%84%EB%A1%9C%EC%B6%95%EA%B5%AC%EC%97%B0%EB%A7%B9" title="한국프로축구연맹">한국프로축구연맹</a>은 수도 연고 구단의 필요성을 느끼고 <a href="/wiki/1999%EB%85%84" title="1999년">1999년</a>부터 <a href="/wiki/K%EB%A6%AC%EA%B7%B8" title="K리그">K리그</a>에 다시 서울 연고 프로축구단을 만들기로 결정한다. 그리고 이듬해 <a href="/wiki/2000%EB%85%84" title="2000년">2000년</a> <a href="/wiki/5%EC%9B%94_12%EC%9D%BC" title="5월 12일">5월 12일</a> 이사회에서 신생구단 창단을 우선하고 불가시 연고지 이전을 허용한다는 원칙<sup class="reference" id="cite_ref-1"><a href="#cite_note-1">[1]</a></sup> 을 세우고 서울연고팀을 만들기 위한 준비작업을 실시한다. 그러나 서울 연고 신생팀을 추진했던 <a href="/wiki/FC_%EC%84%9C%EC%9A%B8_(%EC%98%A5%ED%83%80%EA%B3%A4_%EC%BB%A8%EC%86%8C%EC%8B%9C%EC%97%84)" title="FC 서울 (옥타곤 컨소시엄)">옥타곤 컨소시엄</a>의 창단이 무산되면서 서울 연고팀 문제는 다시 수면 아래로 가라앉게 되었다.</p>
<p>한편 기업과는 별도로 <a href="/wiki/2001%EB%85%84" title="2001년">2001년</a>부터 서울시민들에 의해 서울 연고 프로축구단 창단 운동이 일어났다. 서울의 축구 팬들은 <a href="/wiki/K%EB%A6%AC%EA%B7%B8_%EC%98%AC%EC%8A%A4%ED%83%80%EC%A0%84" title="K리그 올스타전">K리그 올스타전</a>에서 '서울 FC'라는 통천을 올리고, 서명운동과 국제심포지엄을 개최하는 등의 활발한 활동을 벌였다. 이러한 팬들의 노력 끝에 K리그에 서울팀을 창단한다는 결정이 내려지게 된다. 그러나 이러한 창단 운동은 <a href="/wiki/2003%EB%85%84" title="2003년">2003년</a> <a class="mw-redirect" href="/wiki/%EC%95%88%EC%96%91_LG_%EC%B9%98%ED%83%80%EC%8A%A4" title="안양 LG 치타스">안양 LG 치타스</a>의 서울로의 연고 이전으로 퇴보를 겪었다. K리그에서 손꼽히는 서포터들이었던 안양 팬들과 국가대표 서포터즈 클럽인 <a href="/wiki/%EB%B6%89%EC%9D%80%EC%95%85%EB%A7%88" title="붉은악마">붉은악마</a>, K리그 서포터즈 연합 등 대한민국 축구팬들의 격렬한 반대에도 불구하고, <a class="mw-redirect" href="/wiki/%EC%95%88%EC%96%91_LG_%EC%B9%98%ED%83%80%EC%8A%A4" title="안양 LG 치타스">안양 LG 치타스</a>를 보유하고 있던 <a href="/wiki/LG%EC%8A%A4%ED%8F%AC%EC%B8%A0" title="LG스포츠">LG스포츠</a>는 일방적으로 서울로의 연고이전을 발표하였다. 팬들은 LG스포츠의 연고 이전 발표 후 공동 명의의 성명서를 발표하고 <a href="/wiki/%EB%8C%80%ED%95%9C%EC%B6%95%EA%B5%AC%ED%98%91%ED%9A%8C" title="대한축구협회">대한축구협회</a> 사옥 앞에서 대규모 항의 집회를 여는 등 강력한 반대 운동을 전개하였으나 이러한 반대에도 불구하고 LG스포츠는 연고 이전을 강행하였다. 이후 <a class="mw-disambig" href="/wiki/LG" title="LG">LG</a>가 그룹 분리를 하면서 LG스포츠도 <a href="/wiki/GS%EA%B7%B8%EB%A3%B9" title="GS그룹">GS그룹</a>에 분리되어 넘어가게 되었고 그 과정에서 <a href="/wiki/FC_%EC%84%9C%EC%9A%B8" title="FC 서울">FC 서울</a>로 구단명을 변경하고 <a href="/wiki/%EC%84%9C%EC%9A%B8%EC%9B%94%EB%93%9C%EC%BB%B5%EA%B2%BD%EA%B8%B0%EC%9E%A5" title="서울월드컵경기장">서울월드컵경기장</a>을 홈 구장으로 쓰기 시작했다.</p>
<p><a class="mw-redirect" href="/wiki/%EC%95%88%EC%96%91" title="안양">안양</a> 축구 팬들 뿐만 아니라 연고 이전을 반대한 서울 축구 팬들은 계속해서 창단 운동을 지속해 나갔으나 기업 구단이 서울에 들어온 상황에서 서울 시민 구단이 기업들의 지원을 받기는 쉽지 않아, 창단 운동은 벽에 부딪히게 된다. 그러던 중 <a href="/wiki/2007%EB%85%84" title="2007년">2007년</a> <a class="mw-redirect" href="/wiki/%EC%B1%8C%EB%A6%B0%EC%A0%80%EC%8A%A4%EB%A6%AC%EA%B7%B8" title="챌린저스리그">K3리그</a>가 출범하면서 시민구단 창단을 추진해 온 팬들은 K3리그 구단 설립과 참가를 결정한다. 서울 연고의 아마추어 팀인 <i>굿 프렌즈</i>와 <i>진서울 FC</i>가 합병하여 팀을 만들고 <a href="/wiki/%EC%84%9C%EC%9A%B8%EC%98%AC%EB%A6%BC%ED%94%BD%EC%A3%BC%EA%B2%BD%EA%B8%B0%EC%9E%A5" title="서울올림픽주경기장">서울올림픽주경기장</a>을 홈 구장으로 선택했다. 주주공모를 통해 50명의 시민주주가 각각 100만원 이상의 자본금을 출연하여 서울 유나이티드 주식회사가 출범하였고, <a class="mw-redirect" href="/wiki/%EB%82%98%EC%9D%B4%ED%82%A4_(%EA%B8%B0%EC%97%85)" title="나이키 (기업)">나이키</a>와 <a href="/wiki/KT" title="KT">SHOW</a>, <a class="new" href="/w/index.php?title=Goal.com&amp;action=edit&amp;redlink=1" title="Goal.com (없는 문서)">Goal.com</a>등 대형 스폰서의 지원을 얻어 내었다. 7년간의 팬들의 노력으로 만들어진 서울 유나이티드는 <a href="/wiki/2007%EB%85%84" title="2007년">2007년</a> <a href="/wiki/3%EC%9B%94_31%EC%9D%BC" title="3월 31일">3월 31일</a> 해오름식(출범식)을 갖고, <a href="/wiki/4%EC%9B%94_21%EC%9D%BC" title="4월 21일">4월 21일</a> <a class="mw-redirect" href="/wiki/%EC%B0%BD%EC%9B%90_%EB%91%90%EB%8C%80_FC" title="창원 두대 FC">창원 두대 FC</a>와의 K3리그 개막전으로 역사적인 첫 경기를 가졌다.</p>
<h3><span class="mw-headline" id=".EC.B0.BD.EB.8B.A8_.EC.B4.88.EC.B0.BD.EA.B8.B0_.282007.7E2009.EB.85.84.29">창단 초창기 (2007~2009년)</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%84%9C%EC%9A%B8_%EC%9C%A0%EB%82%98%EC%9D%B4%ED%8B%B0%EB%93%9C_FC&amp;action=edit&amp;section=4" title="부분 편집: 창단 초창기 (2007~2009년)">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p><b>서울 유나이티드 2007 시즌</b></p>
<p>창단 첫 해인 <a href="/wiki/K3%EB%A6%AC%EA%B7%B8_2007" title="K3리그 2007">2007년 K3리그</a>에서 전기리그 3위, 후기리그 2위, 통합 1위의 성적으로 플레이오프에 진출하였고, <a class="mw-redirect" href="/wiki/%EC%B2%9C%EC%95%88_FC" title="천안 FC">천안 FC</a>를 플레이오프에서 제압하고 <a class="mw-redirect" href="/wiki/%ED%99%94%EC%84%B1_%EC%8B%A0%EC%9A%B0%EC%A0%84%EC%9E%90" title="화성 신우전자">화성 신우전자</a>를 챔피온결정전에서 꺾으며 K3리그 첫 우승 팀이 되었다. 팀의 스트라이커인 7번 <a href="/wiki/%EC%A0%9C%EC%9A%A9%EC%82%BC" title="제용삼">제용삼</a>은 K3리그 득점왕, K3리그 도움왕, K3리그 MVP를 싹쓸이하는 맹활약을 펼쳤다. 시즌이 끝난 후 <a href="/wiki/%EB%8C%80%ED%95%9C%EC%B6%95%EA%B5%AC%ED%98%91%ED%9A%8C" title="대한축구협회">대한축구협회</a> 선정 2007 일반부 최우수단체상을 수상하였고, <a href="/wiki/%EC%9E%84%EA%B7%BC%EC%9E%AC" title="임근재">임근재</a> 감독은 일반부 최우수 지도자상을 수상하였다. 미드필더 <a class="mw-disambig" href="/wiki/%EC%9D%B4%EC%9E%AC%EB%AA%85" title="이재명">이재명</a>은 팀에서 주최한 팬 선정 올해의 선수에 선정되었다.</p>
<p><b>서울 유나이티드 2008 시즌</b></p>
<p><a href="/wiki/2007%EB%85%84" title="2007년">2007년</a> K3리그 우승 팀 자격으로 <a href="/wiki/FA%EC%BB%B5_2008" title="FA컵 2008">2008년 FA컵</a> 예선 2라운드에 진출하여 <a class="mw-redirect" href="/wiki/%EC%A0%84%EC%A3%BC_EM_FC" title="전주 EM FC">전주 EM</a>을 제압하고 예선 3라운드에 진출하였으나 <a href="/wiki/%EA%B3%A0%EB%A0%A4%EB%8C%80%ED%95%99%EA%B5%90_%EC%B6%95%EA%B5%AC%EB%B6%80" title="고려대학교 축구부">고려대학교</a>에 승부차기 끝에 패배하여 32강 본선 플레이오프 진출에는 실패하였다.</p>
<p>서울 유나이티드는 팀의 두 번째 정규 리그 시즌인 <a href="/wiki/K3%EB%A6%AC%EA%B7%B8_2008" title="K3리그 2008">2008년 K3리그</a>에서 15개의 참가팀 중 전·후기리그 통합 5위의 성적을 거두었다. 2008 시즌에는 <a class="mw-redirect" href="/wiki/%EC%B1%8C%EB%A6%B0%EC%A0%80%EC%8A%A4%EB%A6%AC%EA%B7%B8" title="챌린저스리그">챌린저스리그</a> 페어플레이팀상을 수상하였다.</p>
<p><b><a class="mw-redirect" href="/wiki/%EC%84%9C%EC%9A%B8_%EC%9C%A0%EB%82%98%EC%9D%B4%ED%8B%B0%EB%93%9C_2009_%EC%8B%9C%EC%A6%8C" title="서울 유나이티드 2009 시즌">서울 유나이티드 2009 시즌</a></b></p>
<p>2009 시즌 개막을 앞두고 새로운 스폰서로 <a class="mw-redirect" href="/wiki/%EC%8A%A4%ED%8F%AC%EC%B8%A0%ED%86%A0%ED%86%A0" title="스포츠토토">스포츠토토</a>가 합류하였고, 홈 구장이 위치한 <a href="/wiki/%EC%86%A1%ED%8C%8C%EA%B5%AC" title="송파구">송파구</a>와 연고 협약을 체결하여 협력관계를 구축하였다. 또한 <a href="/wiki/%EB%A7%A5%EB%8F%84%EB%82%A0%EB%93%9C" title="맥도날드">맥도날드</a>와 어린이 축구교실을 공동으로 진행하기로 합의하였다.</p>
<p>서울 유나이티드는 2008시즌 K3리그 5위팀까지 주어지는 <a class="mw-disambig" href="/wiki/FA%EC%BB%B5" title="FA컵">FA컵</a> 예선 진출권을 얻어 <a href="/wiki/FA%EC%BB%B5_2009" title="FA컵 2009">2009년 FA컵</a> 예선에 참가하였다. 예선 1라운드에서 2008 시즌 K3리그 통합순위 1위팀인 <a href="/wiki/%EC%9A%A9%EC%9D%B8_%EC%8B%9C%EB%AF%BC%EC%B6%95%EA%B5%AC%EB%8B%A8" title="용인 시민축구단">용인 시민축구단</a>을 맞아 1:0의 승리를 거둔 후, 예선 2라운드에서 2008년 <a class="new" href="/w/index.php?title=%EC%A0%84%EA%B5%AD%EC%B6%98%EA%B3%84%EB%8C%80%ED%95%99%EC%B6%95%EA%B5%AC%EC%97%B0%EB%A7%B9%EC%A0%84&amp;action=edit&amp;redlink=1" title="전국춘계대학축구연맹전 (없는 문서)">전국춘계대학축구연맹전</a> 준우승 팀인 <a href="/wiki/%EC%A0%84%EC%A3%BC%EB%8C%80%ED%95%99%EA%B5%90" title="전주대학교">전주대학교</a>와 격돌하여 또다시 1:0으로 승리하였다. 서울의 이 승리는 K3리그 팀이 대학 팀과 맞붙어 역사상 처음으로 거둔 승리이다. 서울은 예선 3라운드에서 2008 <a href="/wiki/U%EB%A6%AC%EA%B7%B8" title="U리그">U리그</a> 우승 팀인 <a href="/wiki/%EA%B2%BD%ED%9D%AC%EB%8C%80%ED%95%99%EA%B5%90" title="경희대학교">경희대학교</a>와 격돌하였으나 1:1로 비긴 후 승부차기에서 패하여 FA컵 본선 진출이 좌절되었다.</p>
<p><a href="/wiki/2009%EB%85%84" title="2009년">2009년</a> <a href="/wiki/%EC%A0%84%EA%B5%AD%EC%B2%B4%EC%9C%A1%EB%8C%80%ED%9A%8C" title="전국체육대회">전국체육대회</a> 서울특별시 예선 결승전에 진출하여 <a href="/wiki/7%EC%9B%94_16%EC%9D%BC" title="7월 16일">7월 16일</a> 팀 역사상 최초로 <a href="/wiki/%EB%82%B4%EC%85%94%EB%84%90%EB%A6%AC%EA%B7%B8_(%EB%8C%80%ED%95%9C%EB%AF%BC%EA%B5%AD)" title="내셔널리그 (대한민국)">내셔널리그</a> 팀인 <a class="mw-redirect" href="/wiki/%EB%85%B8%EC%9B%90_%ED%97%98%EB%A9%9C" title="노원 험멜">노원 험멜</a>과 맞붙었으나 연장 승부 끝에 2-3으로 패배하여 본선 진출 자격 획득에 실패하였다.</p>
<p>한편 2009 시즌 중반부터 2010 시즌까지 서울 유나이티드의 홈 경기는 <a href="/wiki/%ED%9A%A8%EC%B0%BD%EC%9A%B4%EB%8F%99%EC%9E%A5" title="효창운동장">효창운동장</a>에서 열렸다. 이것은 비용 절감을 위한 것이기도 하지만, 리그 일정이 늦게 확정되면서 서울이 2009 시즌의 몇몇 홈 경기 날짜에 홈구장인 <a href="/wiki/%EC%84%9C%EC%9A%B8%EC%98%AC%EB%A6%BC%ED%94%BD%EC%A3%BC%EA%B2%BD%EA%B8%B0%EC%9E%A5" title="서울올림픽주경기장">서울올림픽주경기장</a>을 예약하지 못하였기 때문이다.</p>
<p>서울 유나이티드는 팀의 세 번째 시즌인 <a href="/wiki/K3%EB%A6%AC%EA%B7%B8_2009" title="K3리그 2009">2009년 K3리그</a>에서 17개 팀 중 13위를 차지하였다. 서울은 시즌 막바지에 4연승을 거두는 등 선전했으나, 전반기의 얇은 선수층으로 인한 시즌 중반까지의 부진을 극복하지 못하고 13위에 머물렀다. 2009 시즌에는 <a class="mw-disambig" href="/wiki/%EC%A0%95%EB%AA%85%ED%98%B8" title="정명호">정명호</a>가 팬 선정 올해의 선수로 선정되었다.</p>
<h3><span class="mw-headline" id=".EC.9E.A0.EC.8B.A4.EC.9D.84_.EB.96.A0.EB.82.98.EB.8B.A4_.282010.EB.85.84.7E2012.EB.85.84.29">잠실을 떠나다 (2010년~2012년)</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%84%9C%EC%9A%B8_%EC%9C%A0%EB%82%98%EC%9D%B4%ED%8B%B0%EB%93%9C_FC&amp;action=edit&amp;section=5" title="부분 편집: 잠실을 떠나다 (2010년~2012년)">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p><b>서울 유나이티드 2010 시즌</b></p>
<p><a href="/wiki/2010%EB%85%84_1%EC%9B%94" title="2010년 1월">2010년 1월</a>, 서울은 <a href="/wiki/%EA%B9%80%EA%B0%95%EB%82%A8" title="김강남">김강남</a>을 신임 감독으로 선임하였다. 2010시즌엔 주로 <a href="/wiki/%ED%9A%A8%EC%B0%BD%EC%9A%B4%EB%8F%99%EC%9E%A5" title="효창운동장">효창운동장</a>에서 홈경기를 치뤘다. <a href="/wiki/7%EC%9B%94" title="7월">7월</a>부터 <a class="mw-redirect" href="/wiki/%EC%84%B1%EB%82%A8_%EC%9D%BC%ED%99%94_%EC%B2%9C%EB%A7%88" title="성남 일화 천마">성남 일화 천마</a>의 축구 선수인 <a href="/wiki/%EC%9E%A5%ED%95%99%EC%98%81" title="장학영">장학영</a>이 <a class="mw-redirect" href="/wiki/%EA%B3%B5%EC%9D%B5%EA%B7%BC%EB%AC%B4%EC%9A%94%EC%9B%90" title="공익근무요원">공익근무요원</a>의 신분으로 2년간 서울 유나이티드에서 활약하였다. 양대 리그제로 치러진 2010 시즌에 서울은 B조 5위, K3리그 통합 10위의 성적을 기록하였다.</p>
<p><b><a class="mw-redirect" href="/wiki/%EC%84%9C%EC%9A%B8_%EC%9C%A0%EB%82%98%EC%9D%B4%ED%8B%B0%EB%93%9C_2011_%EC%8B%9C%EC%A6%8C" title="서울 유나이티드 2011 시즌">서울 유나이티드 2011 시즌</a></b></p>
<p>2011 시즌부터 서울유나이티드는 기존의 <a href="/wiki/%EC%86%A1%ED%8C%8C%EA%B5%AC" title="송파구">송파구</a>에 이어 <a href="/wiki/%EB%85%B8%EC%9B%90%EA%B5%AC" title="노원구">노원구</a>와 추가로 협약을 맺고 <a href="/wiki/%EB%A7%88%EB%93%A4%EC%8A%A4%ED%83%80%EB%94%94%EC%9B%80" title="마들스타디움">마들스타디움</a>을 홈 경기장으로 사용하고 있다. <a href="/wiki/%EC%B1%8C%EB%A6%B0%EC%A0%80%EC%8A%A4%EB%A6%AC%EA%B7%B8_2011" title="챌린저스리그 2011">챌린저스리그 2011</a>은 2010 시즌에 이어 양대리그제를 채택하였으며, 서울은 추첨을 통해 A조에 배정되어 조 4위, 통합 10위의 성적을 거두었다. 한편, <a href="/wiki/FA%EC%BB%B5_2011" title="FA컵 2011">2011년 FA컵</a> 1라운드 진출권은 리그 통합 성적 2위부터 9위까지 부여되었으나, 2010 시즌 2위팀이었던 <a class="mw-redirect" href="/wiki/%EC%82%BC%EC%B2%99_%EC%8B%A0%EC%9A%B0%EC%A0%84%EC%9E%90" title="삼척 신우전자">삼척 신우전자</a>가 2011 시즌 리그 불참을 선언하면서 서울이 2011년 FA컵 1라운드에 출전하게 되었다. 서울은 1라운드에서 <a href="/wiki/%EA%B4%91%EC%9A%B4%EB%8C%80%ED%95%99%EA%B5%90" title="광운대학교">광운대학교</a>와 맞붙었으나 수비 불안으로 3-4로 패하며 2라운드 진출에 실패하였다. 서울은 2011 시즌 서울특별시장기 축구대회에 출전하여 1라운드에서 <a href="/wiki/%EC%84%9C%EC%9A%B8_FC_%EB%A7%88%EB%A5%B4%ED%8B%B0%EC%8A%A4" title="서울 FC 마르티스">서울 FC 마르티스</a>를 제압하고 결승에 진출하여 내셔널리그 팀인 <a class="mw-redirect" href="/wiki/%EC%B6%A9%EC%A3%BC_%ED%97%98%EB%A9%9C" title="충주 험멜">충주 험멜</a>과 격돌하여 연장전까지 0-0으로 승부를 가리지 못한 끝에 승부차기에서 승리하며 우승을 차지하였다. 서울특별시장기 축구대회는 전국체전 서울특별시 대표 선발전을 겸하기 때문에 서울은 2011 전국체전에 서울특별시 대표로 출전하였고, 본선 1라운드에서 <a class="mw-disambig" href="/wiki/%EA%B2%BD%EC%B0%B0_%EC%B6%95%EA%B5%AC%EB%8B%A8" title="경찰 축구단">경찰 축구단</a>과 대결하였으나 0-4로 패했다.</p>
<p><b>서울 유나이티드 2012 시즌</b></p>
<p>2012 시즌부터 <a class="new" href="/w/index.php?title=%EB%B0%B0%ED%98%95%EB%A0%AC&amp;action=edit&amp;redlink=1" title="배형렬 (없는 문서)">배형렬</a> 수석코치가 감독으로 선임되었다. <a href="/wiki/%EC%B1%8C%EB%A6%B0%EC%A0%80%EC%8A%A4%EB%A6%AC%EA%B7%B8_2012" title="챌린저스리그 2012">챌린저스리그 2012</a>는 지난 두 시즌과 마찬가지로 양대리그제를 채택하였고, 서울은 추첨 결과 B조에 배정되었다. 서울은 2011년과 마찬가지로 2012년에도 FA컵 진출권을 확보하지 못했으나 전년도 리그 9위팀인 전주 EM이 출전을 포기하면서 FA컵 1라운드 출전권을 얻게 되었다. FA컵 1라운드에서 2011시즌 U리그 중부리그 무패우승 팀인 <a href="/wiki/%ED%95%9C%EB%82%A8%EB%8C%80%ED%95%99%EA%B5%90" title="한남대학교">한남대학교</a>와 대결한 끝에 2-4로 패하며 2라운드 진출이 좌절되었다. 서울은 2011년도 서울특별시장기 우승 팀으로서 시드 배정을 받아 서울특별시장기 축구대회 결승으로 직행하였으며, 예선을 거쳐 올라온 <a href="/wiki/%EC%A4%91%EB%9E%91_%EC%BD%94%EB%9F%AC%EC%8A%A4_%EB%AC%B4%EC%8A%A4%ED%83%95" title="중랑 코러스 무스탕">중랑 코러스 무스탕</a>과 대전해 6-3으로 승리하여 2년 연속 서울특별시장기 우승을 차지하게 되었다. 한편, 2012시즌 중반부터 서울 유나이티드와 서울특별시간 협력관계가 강화됨에 따라 서울특별시의 공식 슬로건인 '희망서울'이 중앙에 부착된 유니폼을 사용하고 있다. 2012 시즌의 일부 경기는 다시 <a href="/wiki/%EC%84%9C%EC%9A%B8%EC%98%AC%EB%A6%BC%ED%94%BD%EC%A3%BC%EA%B2%BD%EA%B8%B0%EC%9E%A5" title="서울올림픽주경기장">서울올림픽주경기장</a>에서 개최되었다.</p>
<h3><span class="mw-headline" id=".EB.B3.B8.EA.B2.A9.EC.A0.81.EC.9D.B8_.ED.94.84.EB.A1.9C.ED.99.94_.EC.9B.80.EC.A7.81.EC.9E.84_.282013.EB.85.84.7E.ED.98.84.EC.9E.AC.29">본격적인 프로화 움직임 (2013년~현재)</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%84%9C%EC%9A%B8_%EC%9C%A0%EB%82%98%EC%9D%B4%ED%8B%B0%EB%93%9C_FC&amp;action=edit&amp;section=6" title="부분 편집: 본격적인 프로화 움직임 (2013년~현재)">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p><b>서울 유나이티드 2013 시즌</b></p>
<p>2013 시즌에는 <a href="/wiki/%EA%B9%80%EC%B0%BD%EA%B2%B8" title="김창겸">김창겸</a> 감독이 7대 감독으로 선임되었다. 2013년 서울특별시장기 축구대회 결승에서 우승해 3년 연속 <a class="mw-redirect" href="/wiki/%EC%A0%84%EA%B5%AD%EC%B2%B4%EC%A0%84" title="전국체전">전국체전</a> 서울시 대표로 출전했다. <a href="/wiki/%EB%B0%95%EC%9B%90%EC%88%9C" title="박원순">박원순</a> 서울시장은 유럽 순방중 바르셀로나 방문에서 서울유나이티드의 <a href="/wiki/%ED%98%91%EB%8F%99%EC%A1%B0%ED%95%A9" title="협동조합">협동조합</a>화를 언급하였고<sup class="reference" id="cite_ref-2"><a href="#cite_note-2">[2]</a></sup> 서울유나이티드는 자체적으로 협동조합을 통한 프로화를 추진 중이라는 사실을 밝혔다.<sup class="reference" id="cite_ref-3"><a href="#cite_note-3">[3]</a></sup> 이에 팬들은 <a href="/wiki/%EC%84%9C%EC%9A%B8%EC%98%AC%EB%A6%BC%ED%94%BD%EC%A3%BC%EA%B2%BD%EA%B8%B0%EC%9E%A5" title="서울올림픽주경기장">서울올림픽주경기장</a>에서 열린 <a href="/wiki/2013%EB%85%84_EAFF_%EB%8F%99%EC%95%84%EC%8B%9C%EC%95%88%EC%BB%B5" title="2013년 EAFF 동아시안컵">2013년 EAFF 동아시안컵</a> 3차전에서 구단의 프로화를 지지하는 걸개를 걸었으며<sup class="reference" id="cite_ref-4"><a href="#cite_note-4">[4]</a></sup> 구단은 2013년 8월 7일부로 서울시로부터 협동조합 신고필증을 받았다.<sup class="reference" id="cite_ref-5"><a href="#cite_note-5">[5]</a></sup> 2013년 11월, 공식 홈페이지에 K리그 챌린지 관련 구단 상황에 대한 글이 올라왔다. 재정, 조직, 선수단 구성 미흡을 들어 서울시 측이 연고협상을 거부하고 있는 것이다.<sup class="reference" id="cite_ref-6"><a href="#cite_note-6">[6]</a></sup> 이에 서포터즈 클럽 서유불패는 2013년 11월 19일 구단의 프로화를 지지하고 서울시 연고협약 체결을 촉구하는 성명서를 발표하였다.<sup class="reference" id="cite_ref-7"><a href="#cite_note-7">[7]</a></sup> 하지만 K리그 챌린지 진출에 실패했다는 구단 공지가 올라오며, 최종적으로 2014 K리그 챌린지 진출은 무산되었다.</p>
<p><b>서울 유나이티드 2014 시즌</b></p>
<p>2014 시즌에는 <a href="/wiki/%EC%9C%A0%EA%B8%B0%ED%9D%A5" title="유기흥">유기흥</a> 감독이 8대 감독으로 선임됐다. FA컵 1라운드에서 <a href="/wiki/%EC%83%81%EC%A7%80%EB%8C%80%ED%95%99%EA%B5%90" title="상지대학교">상지대학교</a>에게 패하며, 또 한번 1라운드에 머물러야 했다. 또한 시즌 초반부터 연패를 당하며 하위권에 머무르는 불안한 행보를 보였다. 2014년 서울특별시장기 축구대회 결승에선 패배하여 준우승에 머물렀고, 2014 전국체전에도 나갈 수 없게 되었다. 결국 리그 최약체 <a href="/wiki/%EC%84%9C%EC%9A%B8_FC_%EB%A7%88%EB%A5%B4%ED%8B%B0%EC%8A%A4" title="서울 FC 마르티스">서울 FC 마르티스</a>에게까지 패하며 14위로 시즌을 마감했으며, 또한 구단 역사상 최소 득점을 기록한 시즌이 되었다.</p>
<p><b>서울 유나이티드 2015 시즌</b></p>
<p><a href="/wiki/%EC%B5%9C%EC%83%81%EA%B5%AD" title="최상국">최상국</a> 감독을 9대 감독으로 선임하고, 비시즌 동안 시즌을 빠르게 대비하는 움직임을 보였다. <a href="/wiki/J3%EB%A6%AC%EA%B7%B8" title="J3리그">J3리그</a> 소속 <a href="/wiki/FC_%EB%A5%98%ED%81%90" title="FC 류큐">FC 류큐</a>와 자매 결연을 맺어 정기 교류전을 가지기로 합의했으며, 3월 1일 첫 교류전을 치뤘다. 3월 14일, <a class="mw-redirect" href="/wiki/%EA%B3%A0%EC%96%91%EC%8B%9C%EB%AF%BC%EC%B6%95%EA%B5%AC%EB%8B%A8" title="고양시민축구단">고양시민축구단</a>과의 <a href="/wiki/K3%EB%A6%AC%EA%B7%B8" title="K3리그">K3리그</a> 경기에서 시즌 첫 승을 거뒀다. FA컵은 2라운드에서 상지대에게 일격을 당해 탈락했다. 시즌 초반 2개월 동안이나 승리를 거두지 못하며 하위권으로 떨어지는 부진을 겪었으나 중반부터 득점력이 살아나며 중위권에 안착했다. 그러나 후반기 리그 개막 직전 <a href="/wiki/%EC%98%A4%EC%84%B1%EC%A7%84" title="오성진">오성진</a>, 최예찬, 임동현 등 핵심 선수들의 잇따른 이탈로 조직력이 약화되었고, 후기 조별리그 전패를 기록하였다. 최종 성적은 조 9위, 통합 17위로 구단 역사상 가장 낮은 순위로 시즌을 마감하게 되었다. 다만, 득점 부분에선 공격적인 패스축구를 구사하며 작년에 비해 공격력이 개선되었다는 평을 받았다.</p>
<p><b><a href="/wiki/%EC%84%9C%EC%9A%B8_%EC%9C%A0%EB%82%98%EC%9D%B4%ED%8B%B0%EB%93%9C_FC_2016_%EC%8B%9C%EC%A6%8C" title="서울 유나이티드 FC 2016 시즌">서울 유나이티드 2016 시즌</a></b></p>
<p>프리시즌 동안 <a href="/wiki/%EC%B5%9C%EC%83%81%EA%B5%AD" title="최상국">최상국</a> 감독과의 계약을 1년 더 연장하고, <a href="/wiki/%EC%98%A4%EC%84%B1%EC%A7%84" title="오성진">오성진</a>, 최예찬 등의 핵심 선수를 복귀시킨데 이어 <a class="mw-redirect" href="/wiki/%EC%B2%AD%EC%B6%98FC" title="청춘FC">청춘FC</a>에서 활약한 길정현, <a href="/wiki/%EC%9E%84%EA%B7%BC%EC%98%81" title="임근영">임근영</a>, 명승호, 최원태 선수를 영입하며 팬들의 기대감을 높였다. FA컵은 1라운드에서 SMC엔지니어링에 패해 탈락했다. 6월 10일, <a href="/wiki/%EC%A1%B0%EB%8F%99%ED%98%84_(%EC%B6%95%EA%B5%AC%EC%9D%B8)" title="조동현 (축구인)">조동현</a> 감독을 10대 감독으로 선임하였다. 같은 달 27일, 곽선우 전 <a href="/wiki/%EC%84%B1%EB%82%A8_FC" title="성남 FC">성남 FC</a> 대표이사를 신임 사장으로 임명했다. 리그 최종 순위 17위를 기록하여 2017시즌 K3 베이직리그로의 강등이 확정되었다.</p>

<h2><span class="mw-headline" id=".ED.81.B4.EB.9F.BD_.EB.AC.B8.ED.99.94">클럽 문화</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%84%9C%EC%9A%B8_%EC%9C%A0%EB%82%98%EC%9D%B4%ED%8B%B0%EB%93%9C_FC&amp;action=edit&amp;section=12" title="부분 편집: 클럽 문화">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<h3><span class="mw-headline" id=".EC.97.A0.EB.B8.94.EB.9F.BC.EA.B3.BC_.EC.9C.A0.EB.8B.88.ED.8F.BC">엠블럼과 유니폼</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%84%9C%EC%9A%B8_%EC%9C%A0%EB%82%98%EC%9D%B4%ED%8B%B0%EB%93%9C_FC&amp;action=edit&amp;section=13" title="부분 편집: 엠블럼과 유니폼">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>엠블럼은 공개 모집을 통해 회원들에 의해 정해진 안을 갖고 다시 수정을 거쳐 2007시즌을 앞두고 확정되었다. 축구 전문 디자이너 장부다 씨가 디자인한 것으로 알려져있다. 아리수, 광화문 등 <a href="/wiki/%EC%84%9C%EC%9A%B8%ED%8A%B9%EB%B3%84%EC%8B%9C" title="서울특별시">서울특별시</a>를 상징하는 문양들을 새겨넣었으며, 백호를 배치하여 향후 구단 마스코트로 만들 가능성도 남겨놓았다. <a href="/wiki/%EB%89%B4%EC%BA%90%EC%8A%AC_%EC%9C%A0%EB%82%98%EC%9D%B4%ED%8B%B0%EB%93%9C_FC" title="뉴캐슬 유나이티드 FC">뉴캐슬 유나이티드 FC</a>의 엠블럼과 흡사하다는 비판도 있다.<sup class="reference" id="cite_ref-8"><a href="#cite_note-8">[8]</a></sup> 유니폼은 제왕의 상징인 흑색과 순결함의 상징인 백색의 줄무늬로 구성된 홈 유니폼을 창단부터 고수해왔으며, 원정 유니폼은 창단 때는 파란색 단색을 택했으나 잠시 백색, 형광색으로 시즌을 보내기도 했다. 현재는 창단 초기의 유니폼 색과 같은 흑백 줄무늬의 홈 유니폼과 청백 줄무늬의 원정 유니폼을 사용 중이다.</p>
<h3><span class="mw-headline" id=".ED.8C.AC">팬</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%84%9C%EC%9A%B8_%EC%9C%A0%EB%82%98%EC%9D%B4%ED%8B%B0%EB%93%9C_FC&amp;action=edit&amp;section=14" title="부분 편집: 팬">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>창단 초창기 <a href="/wiki/%EC%84%9C%EC%9A%B8%EC%98%AC%EB%A6%BC%ED%94%BD%EC%A3%BC%EA%B2%BD%EA%B8%B0%EC%9E%A5" title="서울올림픽주경기장">서울올림픽주경기장</a>을 사용했을 당시의 팬들보다는 상당히 숫자가 줄었으나 여전히 서포터즈 클럽 &lt;서유불패&gt;를 주축으로 한 팬들이 상당수 존재한다. <a class="mw-redirect" href="/wiki/%EC%B1%8C%EB%A6%B0%EC%A0%80%EC%8A%A4%EB%A6%AC%EA%B7%B8" title="챌린저스리그">챌린저스리그</a>에서 가장 많은 팬을 보유하고 있으며, 2013년엔 <a href="/wiki/%EC%84%9C%EC%9A%B8%EC%98%AC%EB%A6%BC%ED%94%BD%EC%A3%BC%EA%B2%BD%EA%B8%B0%EC%9E%A5" title="서울올림픽주경기장">서울올림픽주경기장</a>에서 열린 <a href="/wiki/2013%EB%85%84_EAFF_%EB%8F%99%EC%95%84%EC%8B%9C%EC%95%88%EC%BB%B5" title="2013년 EAFF 동아시안컵">2013년 EAFF 동아시안컵</a> 3차전에서 구단의 프로화를 지지하는 걸개를 걸기도 하였다. 과거 월등히 많았던 팬 수 때문에 경기장 폭력사태가 일어나 무관중 징계를 받기도 했다.</p>
<h3><span class="mw-headline" id=".EB.AF.B8.EB.94.94.EC.96.B4">미디어</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%84%9C%EC%9A%B8_%EC%9C%A0%EB%82%98%EC%9D%B4%ED%8B%B0%EB%93%9C_FC&amp;action=edit&amp;section=15" title="부분 편집: 미디어">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>창단부터 함께 해왔던 자체 중계 방송인 SUTV<sup class="reference" id="cite_ref-9"><a href="#cite_note-9">[9]</a></sup> 가 하이라이트 업로드와 경기 생중계를 맡고 있다. 중계는 <a href="/wiki/%EB%8B%A4%EC%9D%8C_%ED%8C%9F%ED%94%8C%EB%A0%88%EC%9D%B4%EC%96%B4" title="다음 팟플레이어">다음 팟플레이어</a>로 진행되며, 방송인 교육기관 나비스피치와의 협약을 통해 좀 더 전문화를 시도 중이다. 또한 2013년부터 명예기자 시스템을 새롭게 운영하며 매 경기 구단 경기 리뷰 기사와 인터뷰 등을 진행중이며 기자들에게 SUTV 해설위원을 맡기기도 하는 모습을 보이고 있다. 각종 기사는 <a href="/wiki/%EB%8B%A4%EC%9D%8C" title="다음">다음</a> 스포츠 뉴스에 업로드되어 접근성이 높은 편이다. 초기부터 지적되어왔던 미디어의 부족함을 잘 메꾸고, 홍보 효과까지 얻었다는 점에서 미디어 부분은 팬들에게 상당한 호평을 받고 있다.</p>
<h3><span class="mw-headline" id=".EB.9D.BC.EC.9D.B4.EB.B2.8C">라이벌</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%84%9C%EC%9A%B8_%EC%9C%A0%EB%82%98%EC%9D%B4%ED%8B%B0%EB%93%9C_FC&amp;action=edit&amp;section=16" title="부분 편집: 라이벌">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>같은 <a class="mw-redirect" href="/wiki/%EC%84%9C%EC%9A%B8%EC%8B%9C" title="서울시">서울시</a>를 연고로 하는 <a href="/wiki/%EC%84%9C%EC%9A%B8_%EC%9D%B4%EB%9E%9C%EB%93%9C_FC" title="서울 이랜드 FC">서울 이랜드 FC</a>는 과거 서유의 홈구장이었던 <a class="mw-redirect" href="/wiki/%EC%9E%A0%EC%8B%A4%EC%98%AC%EB%A6%BC%ED%94%BD%EC%A3%BC%EA%B2%BD%EA%B8%B0%EC%9E%A5" title="잠실올림픽주경기장">잠실올림픽주경기장</a>을 사용하게 되며 서유의 프로화시 라이벌 구도가 형성될 듯 하다. 또한 다른 서울 팀인 <a href="/wiki/FC_%EC%84%9C%EC%9A%B8" title="FC 서울">FC 서울</a>의 2군 팀과 서유가 연습경기를 가진 적도 있었다. <a href="/wiki/%EB%B6%80%EC%B2%9C_FC_1995" title="부천 FC 1995">부천 FC 1995</a>가 <a href="/wiki/K3%EB%A6%AC%EA%B7%B8" title="K3리그">K3리그</a>에 참여했던 프로화 이전의 시기에는 비교적 많았던 두 팀의 팬들로 인해 두 팀간의 경쟁심이 생겨났고, 언론들도 이를 다루며 라이벌이 형성되었다. 두 팀의 경기는 종종 선수간의 폭력도 일어나며 거친 경기가 많았고, 서포터 간의 충돌도 가끔 일어난 것으로 알려져있다. 2011년부터 서유가 <a href="/wiki/%EC%A4%91%EA%B3%84%EC%97%AD" title="중계역">중계역</a> 인근 <a href="/wiki/%EB%A7%88%EB%93%A4%EC%8A%A4%ED%83%80%EB%94%94%EC%9B%80" title="마들스타디움">마들스타디움</a>으로 경기장을 옮기며 7호선 더비라는 이름을 얻기도 했다. <a class="mw-redirect" href="/wiki/%EC%B6%A9%EC%A3%BC_%ED%97%98%EB%A9%9C" title="충주 험멜">충주 험멜</a>과는 전국체전에서의 악연이 존재한다. 노원구에서 충주로 연고를 이전했음에도 등록은 서울시 소속이라, 2010년 충주가 서유를 꺾고 서울시 대표로 2010년 전국체전에 나서는 해프닝이 벌어졌으며, 이 과정에서 오심으로 인해 두 팀의 사이가 악화되었다.</p>
<h3><span class="mw-headline" id=".EC.9C.A0.EC.86.8C.EB.85.84_.EC.9C.A1.EC.84.B1">유소년 육성</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%84%9C%EC%9A%B8_%EC%9C%A0%EB%82%98%EC%9D%B4%ED%8B%B0%EB%93%9C_FC&amp;action=edit&amp;section=17" title="부분 편집: 유소년 육성">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>서울시 내 자치구들과 협력하여 <a href="/wiki/%EC%84%B1%EB%8F%99%EA%B5%AC" title="성동구">성동구</a>, <a href="/wiki/%EC%84%9C%EB%8C%80%EB%AC%B8%EA%B5%AC" title="서대문구">서대문구</a>, <a href="/wiki/%EB%85%B8%EC%9B%90%EA%B5%AC" title="노원구">노원구</a>, <a href="/wiki/%EC%84%9C%EC%B4%88%EA%B5%AC" title="서초구">서초구</a>에서 유소년 축구교실들을 운영중이다. 선수들을 코치로 보내어 지역 내 축구 꿈나무 육성 및 클럽 홍보의 두마리 토끼를 잡으려는 시도를 하고 있다. 또한 프로화를 대비하여 <a href="/wiki/%EC%A0%84%EA%B5%AD_%EC%B4%88%EC%A4%91%EA%B3%A0_%EC%B6%95%EA%B5%AC%EB%A6%AC%EA%B7%B8" title="전국 초중고 축구리그">전국 초중고 축구리그</a>에 <a href="/wiki/%EA%B9%80%EA%B2%BD%EB%B2%94" title="김경범">김경범</a> 감독이 이끄는 U-18 고등부 팀을 운영중이다. 2014시즌을 앞두고 고등부에서 수비수 김한주 선수가 성인팀으로 합류하여 개막전에 출장하며 구단 유스 출신 1호 선수가 되었다.</p>
