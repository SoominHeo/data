<p><b>서울 이랜드 FC</b>(<span lang="en" xml:lang="en">Seoul E-Land FC</span>)는 <a href="/wiki/2015%EB%85%84" title="2015년">2015년</a>부터 <a href="/wiki/%EB%8C%80%ED%95%9C%EB%AF%BC%EA%B5%AD" title="대한민국">대한민국</a>의 <a href="/wiki/%EC%84%9C%EC%9A%B8%ED%8A%B9%EB%B3%84%EC%8B%9C" title="서울특별시">서울특별시</a>를 연고지로 하는 대한민국 프로<a href="/wiki/%EC%B6%95%EA%B5%AC" title="축구">축구</a> 2부 리그인 <a href="/wiki/K%EB%A6%AC%EA%B7%B8_%EC%B1%8C%EB%A6%B0%EC%A7%80" title="K리그 챌린지">K리그 챌린지</a>에 참가하는 축구단이다. <a href="/wiki/2014%EB%85%84" title="2014년">2014년</a> <a href="/wiki/4%EC%9B%94_9%EC%9D%BC" title="4월 9일">4월 9일</a>에 <a href="/wiki/%EC%9D%B4%EB%9E%9C%EB%93%9C%EA%B7%B8%EB%A3%B9" title="이랜드그룹">이랜드그룹</a>에서 창단 관련 보도자료를 배포하였으며<sup class="reference" id="cite_ref-1"><a href="#cite_note-1">[1]</a></sup>, <a href="/wiki/4%EC%9B%94_14%EC%9D%BC" title="4월 14일">4월 14일</a>에 공식 창단 발표 기자회견과 함께 창단의향서를 <a href="/wiki/%ED%95%9C%EA%B5%AD%ED%94%84%EB%A1%9C%EC%B6%95%EA%B5%AC%EC%97%B0%EB%A7%B9" title="한국프로축구연맹">한국프로축구연맹</a>에 제출하였다.<sup class="reference" id="cite_ref-2"><a href="#cite_note-2">[2]</a></sup> 홈구장은 서울 <a class="mw-disambig" href="/wiki/%EA%B0%95%EB%82%A8" title="강남">강남</a> 지역인 <a href="/wiki/%EC%86%A1%ED%8C%8C%EA%B5%AC" title="송파구">송파구</a> 올림픽로에 위치한 <a href="/wiki/%EC%84%9C%EC%9A%B8%EC%98%AC%EB%A6%BC%ED%94%BD%EC%A3%BC%EA%B2%BD%EA%B8%B0%EC%9E%A5" title="서울올림픽주경기장">서울올림픽주경기장(잠실종합운동장)</a>을 사용한다.</p>

<h2><span class="mw-headline" id=".EC.97.AD.EC.82.AC">역사</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%84%9C%EC%9A%B8_%EC%9D%B4%EB%9E%9C%EB%93%9C_FC&amp;action=edit&amp;section=1" title="부분 편집: 역사">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p><a href="/wiki/2014%EB%85%84" title="2014년">2014년</a> <a href="/wiki/4%EC%9B%94_9%EC%9D%BC" title="4월 9일">4월 9일</a>, <a href="/wiki/%EC%9D%B4%EB%9E%9C%EB%93%9C%EA%B7%B8%EB%A3%B9" title="이랜드그룹">이랜드그룹</a>은 보도자료를 통해 서울 연고 프로 축구단을 창단한다고 밝혔다. <a href="/wiki/4%EC%9B%94_14%EC%9D%BC" title="4월 14일">4월 14일</a>에는 공식 창단 발표 기자회견과 함께 창단의향서를 <a href="/wiki/%ED%95%9C%EA%B5%AD%ED%94%84%EB%A1%9C%EC%B6%95%EA%B5%AC%EC%97%B0%EB%A7%B9" title="한국프로축구연맹">한국프로축구연맹</a>에 제출하였다. <a href="/wiki/5%EC%9B%94_1%EC%9D%BC" title="5월 1일">5월 1일</a> <a href="/wiki/%EC%84%9C%EC%9A%B8%ED%8A%B9%EB%B3%84%EC%8B%9C" title="서울특별시">서울특별시</a>와 연고지 협약을 체결하고 본격적인 창단 준비에 들어갔고<sup class="reference" id="cite_ref-3"><a href="#cite_note-3">[3]</a></sup> 16일에는 팬들을 대상으로 팀명과 팀의 상징, 팀 컬러에 대한 공모를 시작하였다.<sup class="reference" id="cite_ref-4"><a href="#cite_note-4">[4]</a></sup> <a href="/wiki/7%EC%9B%94_17%EC%9D%BC" title="7월 17일">7월 17일</a> <a href="/wiki/%EB%A9%94%EC%9D%B4%EC%A0%80_%EB%A6%AC%EA%B7%B8_%EC%82%AC%EC%BB%A4" title="메이저 리그 사커">메이저 리그 사커</a> <a href="/wiki/%EB%B0%B4%EC%BF%A0%EB%B2%84_%ED%99%94%EC%9D%B4%ED%8A%B8%EC%BA%A1%EC%8A%A4_FC" title="밴쿠버 화이트캡스 FC">밴쿠버 화이트캡스</a>에서 감독을 맡았던 <a href="/wiki/%EB%A7%88%ED%8B%B4_%EB%A0%88%EB%8B%88" title="마틴 레니">마틴 레니</a>를 초대 감독으로 선임하였다.<sup class="reference" id="cite_ref-5"><a href="#cite_note-5">[5]</a></sup> <a href="/wiki/8%EC%9B%94_22%EC%9D%BC" title="8월 22일">8월 22일</a> <a href="/wiki/%ED%95%9C%EA%B5%AD%ED%94%84%EB%A1%9C%EC%B6%95%EA%B5%AC%EC%97%B0%EB%A7%B9" title="한국프로축구연맹">한국프로축구연맹</a>의 창단 승인이 완료되었으며<sup class="reference" id="cite_ref-6"><a href="#cite_note-6">[6]</a></sup> <a href="/wiki/8%EC%9B%94_26%EC%9D%BC" title="8월 26일">8월 26일</a> 구단명칭을 '서울 이랜드 FC'로 확정하였다.<sup class="reference" id="cite_ref-7"><a href="#cite_note-7">[7]</a></sup> <a href="/wiki/2015%EB%85%84" title="2015년">2015년</a> <a href="/wiki/2%EC%9B%94_7%EC%9D%BC" title="2월 7일">2월 7일</a>, 서울 강남구 대치동 SETEC 컨벤션센터 1층에서 이루어진 팬포럼에서 구단 엠블럼을 공개하였다.<sup class="reference" id="cite_ref-8"><a href="#cite_note-8">[8]</a></sup></p>
<p>2015년 3월 29일 홈 경기장인 <a href="/wiki/%EC%84%9C%EC%9A%B8%EC%98%AC%EB%A6%BC%ED%94%BD%EC%A3%BC%EA%B2%BD%EA%B8%B0%EC%9E%A5" title="서울올림픽주경기장">서울올림픽주경기장</a>에서 <a href="/wiki/FC_%EC%95%88%EC%96%91" title="FC 안양">FC 안양</a>을 상대로 창단 후 첫 공식 경기를 치렀으며 4월 11일 열린 <a href="/wiki/FA%EC%BB%B5_2015" title="FA컵 2015">2015 FA컵</a> 3라운드에서 <a href="/wiki/%EC%84%A0%EB%AC%B8%EB%8C%80%ED%95%99%EA%B5%90" title="선문대학교">선문대학교</a>를 상대로 공식 경기 첫 승을 거두었고 5월 2일 <a href="/wiki/%EC%88%98%EC%9B%90_FC" title="수원 FC">수원 FC</a>와의 원정 경기에선 리그 첫 승을 챙겼다. 이후 <a href="/wiki/%EC%A3%BC%EB%AF%BC%EA%B7%9C" title="주민규">주민규</a>, <a href="/wiki/%EC%9D%B4%EC%A7%80%EC%A2%85_%EB%A3%A8%EC%9D%B4%EC%8A%A4_%EB%91%90%EC%8A%A4_%EC%82%B0%ED%88%AC%EC%8A%A4" title="이지종 루이스 두스 산투스">타라바이</a> 등의 활약으로 리그 상위권을 유지하였으나 하반기에 주춤하여 정규 리그를 4위로 마감하였다. 정규 리그 종료 후 진행된 승격 플레이오프에서 정규시즌 3위인 <a href="/wiki/%EC%88%98%EC%9B%90_FC" title="수원 FC">수원 FC</a>를 만났으나 3-3 무승부로 경기를 마쳐 무승부시 홈팀 승격 규정에 의해 플레이오프에서 탈락하며 첫 시즌을 마감하였다.</p>
<p>이듬해에는 6위로 처지며 승격권에서 멀어지자, 2016년 6월 15일 마틴 레니 감독을 전격 해임하였고, 2대 감독으로 축구 국가대표팀 코치를 역임했던 <a href="/wiki/%EB%B0%95%EA%B1%B4%ED%95%98" title="박건하">박건하</a>를 서울 이랜드의 새 감독으로 선임하였고, 이후 상승세를 타는 듯 했지만 리그 6위를 기록하며 끝내 플레이오프 진입에 실패하였다.</p>
<p>2017시즌을 앞두고 영입을 발표하였던 외국인 <a href="/wiki/%EB%B9%85%ED%86%A0%EB%A5%B4_%EC%95%84%ED%82%A4%EB%85%B8" title="빅토르 아키노">빅토르 아키노</a>하고 <a href="/wiki/%ED%8C%8C%EB%B8%94%EB%A1%9C_%EB%A7%88%EB%A5%B4%ED%8B%B4_%EB%A3%A8%EC%9D%B4%EC%8A%A4" title="파블로 마르틴 루이스">파블로 루이스</a> 모두의 영입이 번복이 되면서 논란이 일었다.</p>
<p><a href="/wiki/1%EC%9B%94_9%EC%9D%BC" title="1월 9일">1월 9일</a> 박건하 감독이 사임하고 <a href="/wiki/%EC%98%81%EB%82%A8%EB%8C%80%ED%95%99%EA%B5%90" title="영남대학교">영남대학교</a> 감독 출신 <a href="/wiki/%EA%B9%80%EB%B3%91%EC%88%98_(%EC%B6%95%EA%B5%AC%EC%9D%B8)" title="김병수 (축구인)">김병수</a>가 감독으로 내정되었다.</p>
