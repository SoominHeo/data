<h1 class="firstHeading" id="firstHeading" lang="ko">서울외국인묘지공원</h1>
<div class="mw-body-content" id="bodyContent">

<div id="contentSub"></div>
<div class="mw-jump" id="jump-to-nav">
					이동:					<a href="#mw-head">둘러보기</a>, 					<a href="#p-search">검색</a>
</div>
<div class="mw-content-ltr" dir="ltr" id="mw-content-text" lang="ko"><script>function mfTempOpenSection(id){var block=document.getElementById("mf-section-"+id);block.className+=" open-block";block.previousSibling.className+=" open-block";}</script><p><b>서울외국인묘지공원</b>(<small><a class="mw-redirect" href="/wiki/%ED%95%9C%EA%B5%AD_%ED%95%9C%EC%9E%90" title="한국 한자">한국 한자</a>: </small>-外國人墓地公園)은 <a href="/wiki/%EB%8C%80%ED%95%9C%EB%AF%BC%EA%B5%AD" title="대한민국">대한민국</a> <a class="mw-redirect" href="/wiki/%EC%84%9C%EC%9A%B8_%EB%A7%88%ED%8F%AC%EA%B5%AC" title="서울 마포구">서울 마포구</a> 합정동에 있는 <a href="/wiki/%EA%B3%B5%EC%9B%90" title="공원">공원</a>이다. 다른 이름은 <b>양화진외국인선교사묘원</b>(<small><a class="mw-redirect" href="/wiki/%ED%95%9C%EA%B5%AD_%ED%95%9C%EC%9E%90" title="한국 한자">한국 한자</a>: </small>楊花津外國人宣敎師墓園)로서 <a href="/wiki/%EC%84%9C%EC%9A%B8_%EC%96%91%ED%99%94%EB%82%98%EB%A3%A8%EC%99%80_%EC%9E%A0%EB%91%90%EB%B4%89_%EC%9C%A0%EC%A0%81" title="서울 양화나루와 잠두봉 유적">서울 양화나루와 잠두봉 유적</a>의 일부이다.</p>
<p><a class="mw-redirect" href="/wiki/%EC%A1%B0%EC%84%A0_%EC%8B%9C%EB%8C%80" title="조선 시대">조선 시대</a> <a class="mw-redirect" href="/wiki/%EC%96%91%ED%99%94%EC%A7%84" title="양화진">양화진</a>(楊花津) 나루터를 수비하던 양화진영이 있던 곳으로 1860년에 외국인 묘지로 조성되었다. 이 곳에는 조선 말 <a class="mw-redirect" href="/wiki/%EC%A1%B0%EC%84%A0_%EA%B3%A0%EC%A2%85" title="조선 고종">고종</a> 때부터 <a href="/wiki/%ED%95%9C%EA%B5%AD" title="한국">한국</a>을 위해 공헌한 언론계, 교육계, 종교계 외국인 인사들 500여 명이 묻혀 있다. 한국의 암흑기였던 1900년 전후에 언론창달의 기수 역할을 했던 <a href="/wiki/%EB%8C%80%ED%95%9C%EB%A7%A4%EC%9D%BC%EC%8B%A0%EB%B3%B4" title="대한매일신보">대한매일신보</a>의 <a href="/wiki/%EC%96%B4%EB%8B%88%EC%8A%A4%ED%8A%B8_%EB%B2%A0%EB%8D%B8" title="어니스트 베델">어니스트 베델</a>, 장로교 선교사 <a href="/wiki/%ED%98%B8%EB%9F%AC%EC%8A%A4_%EA%B7%B8%EB%9E%9C%ED%8A%B8_%EC%96%B8%EB%8D%94%EC%9A%B0%EB%93%9C" title="호러스 그랜트 언더우드">호러스 그랜트 언더우드</a>, <a href="/wiki/%EC%9D%B4%ED%99%94%EC%97%AC%EC%9E%90%EB%8C%80%ED%95%99%EA%B5%90" title="이화여자대학교">이화여대</a> 설립에 공이 큰 <a href="/wiki/%EA%B0%90%EB%A6%AC%EA%B5%90" title="감리교">감리교</a> <a href="/wiki/%EC%84%A0%EA%B5%90%EC%82%AC" title="선교사">선교사</a> <a href="/wiki/%ED%97%A8%EB%A6%AC_%EC%95%84%ED%8E%9C%EC%A0%A4%EB%9F%AC" title="헨리 아펜젤러">헨리 아펜젤러</a>와 그의 가족들, 세브란스 의대를 세운 <a class="new" href="/w/index.php?title=%EB%8D%94%EA%B8%80%EB%9F%AC%EC%8A%A4_%EC%97%90%EB%B9%84%EC%8A%A8&amp;action=edit&amp;redlink=1" title="더글러스 에비슨 (없는 문서)">더글러스 B. 에비슨</a>, 한국의 은인으로 추앙 받는 <a href="/wiki/%ED%98%B8%EB%A8%B8_%ED%97%90%EB%B2%84%ED%8A%B8" title="호머 헐버트">호머 헐버트</a> 박사, <a href="/wiki/%EB%8C%80%ED%95%9C%EC%A0%9C%EA%B5%AD" title="대한제국">대한제국</a> 국가를 작곡한 <a href="/wiki/%ED%94%84%EB%9E%80%EC%B8%A0_%EC%97%90%EC%BC%80%EB%A5%B4%ED%8A%B8" title="프란츠 에케르트">프란츠 에케르트</a>의 묘도 이곳에 있다.</p>
<p>현재 이 곳 부지를 한국 기독교 100주년 기념 사업 협의회와 <a class="new" href="/w/index.php?title=100%EC%A3%BC%EB%85%84_%EA%B8%B0%EB%85%90%EA%B5%90%ED%9A%8C&amp;action=edit&amp;redlink=1" title="100주년 기념교회 (없는 문서)">100주년 기념교회</a>가 관할하고 있으나 현재 묘지 공원의 권리를 주장하는 경성구미인묘지회, 유니온교회, 양화진 선교회, <a href="/wiki/%EB%8C%80%ED%95%9C%EC%98%88%EC%88%98%EA%B5%90%EC%9E%A5%EB%A1%9C%ED%9A%8C(%ED%86%B5%ED%95%A9)" title="대한예수교장로회(통합)">예장통합</a>과의 이권 논쟁이 지속되고 있다.<sup class="reference" id="cite_ref-1"><a href="#cite_note-1">[1]</a></sup></p>
<p>그러나 2013년 2월 28일 대법원에서 ‘양화진의 법적 소유주는 100주년기념재단이고, 유니온교회는 쫓겨난 것이 아니다’는 확정 판결을 받았다..<sup class="reference" id="cite_ref-2"><a href="#cite_note-2">[2]</a></sup></p>
