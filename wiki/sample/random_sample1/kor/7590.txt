<p><b>서하 문자</b>(西夏文字)는 <a class="mw-redirect" href="/wiki/%EC%84%9C%ED%95%98_%EC%99%95%EA%B5%AD" title="서하 왕국">서하 왕국</a>의 초대 왕 이원호(李元昊) 때 제정된 문자이다. <a href="/wiki/1036%EB%85%84" title="1036년">1036년</a>경에 반포되었다. 문자수는 6,000을 넘고, 구조는 비슷하게 등장한 <a href="/wiki/%EA%B1%B0%EB%9E%80_%EB%AC%B8%EC%9E%90" title="거란 문자">거란 문자</a>, <a href="/wiki/%EC%97%AC%EC%A7%84_%EB%AC%B8%EC%9E%90" title="여진 문자">여진 문자</a>와 견주었을 때 <a href="/wiki/%ED%95%9C%EC%9E%90" title="한자">한자</a>와 거리가 더 멀다. 즉 거란 문자, 여진 문자는 한자의 편, 방 등의 구조를 그대로 차용한 부분이 보이지만, 서하 문자는 글자꼴 등에서 한자의 기본 짜임새를 직접 따오지 않았다. <a class="mw-redirect" href="/wiki/%ED%83%95%EA%B5%AC%ED%8A%B8_%EC%A1%B1" title="탕구트 족">서하인</a>은 현재 <a class="mw-redirect" href="/wiki/%ED%8B%B0%EB%B2%A0%ED%8A%B8%EC%9D%B8" title="티베트인">티베트인</a>에 가까운 종족이었기 때문에 <a href="/wiki/%EC%95%8C%ED%83%80%EC%9D%B4%EC%96%B4%EC%A1%B1" title="알타이어족">알타이어족</a> 계통으로 생각되는 교착어인 <a href="/wiki/%EA%B1%B0%EB%9E%80%EC%96%B4" title="거란어">거란어</a>(거란 문자), <a href="/wiki/%EC%97%AC%EC%A7%84%EC%96%B4" title="여진어">여진어</a>(여진 문자)와는 서하 왕국의 <a class="new" href="/w/index.php?title=%ED%83%95%EA%B5%AC%ED%8A%B8%EC%96%B4&amp;action=edit&amp;redlink=1" title="탕구트어 (없는 문서)">탕구트어</a>는 달리 단음절, 성조어로서 이러한 언어적 특징도 문자에 반영되었다. 현재 <a href="/wiki/%EC%9C%A0%EB%8B%88%EC%BD%94%EB%93%9C" title="유니코드">유니코드</a>에 서하 문자를 추가하는 것이 제안되었다.</p>

<h2><span class="mw-headline" id=".ED.95.B4.EB.8F.85">해독</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%84%9C%ED%95%98_%EB%AC%B8%EC%9E%90&amp;action=edit&amp;section=1" title="부분 편집: 해독">원본 편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>오랫동안 뜻을 알아내지 못했지만, 최근에 학자들이 대부분의 뜻을 파악하고 있다. 서하인은 중국 고전을 활발하게 옮기는 작업을 해서 다른 두 문자에 비해 현존자료가 넉넉한 편으로, 이 때문에 해독작업도 더 앞서있는 편이다.</p>
<h2><span class="mw-headline" id=".ED.8A.B9.EC.A7.95">특징</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%84%9C%ED%95%98_%EB%AC%B8%EC%9E%90&amp;action=edit&amp;section=2" title="부분 편집: 특징">원본 편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>서하 문자의 일부에는 중국에 대한 대항의식과 자존감이 보이는 부분이 있는데 그 예로 중국이란 뜻의 서하 문자는 "작은"이란 뜻의 의미요소와 "벌레"라는 뜻의 의미요소가 더해져서 이루어진 부분이 그것이다. 또한 <a href="/wiki/%EA%B1%B0%EB%9E%80_%EB%AC%B8%EC%9E%90" title="거란 문자">거란 문자</a>의 일부 글자와 비슷한 글자도 존재하지만, 서로 관련이 있는지는 확실하지 않다.</p>
