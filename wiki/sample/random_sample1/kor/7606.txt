<p><b>선부동</b>(仙府洞)은 <a href="/wiki/%EA%B2%BD%EA%B8%B0%EB%8F%84" title="경기도">경기도</a> <a href="/wiki/%EC%95%88%EC%82%B0%EC%8B%9C" title="안산시">안산시</a> <a href="/wiki/%EB%8B%A8%EC%9B%90%EA%B5%AC" title="단원구">단원구</a>의 법정동이다. 행정동으로 선부1동, 선부2동, 선부3동으로 구성되어 있다.</p>

<h2><span class="mw-headline" id=".EA.B0.9C.EC.9A.94">개요</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%84%A0%EB%B6%80%EB%8F%99&amp;action=edit&amp;section=1" title="부분 편집: 개요">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>선부동 이름의 유래로는 옛날에 선녀가 내려와 목욕을 하던 우물이 있었다고 하여 仙(신선 선), 府(마을 부)를 사용하여 선부동이라고 하였다.( 그 우물의 이름은 `석수골 우물` 이라고 부름). <sup class="reference" id="cite_ref-1"><a href="#cite_note-1">[1]</a></sup>선부동은 <a class="mw-redirect" href="/wiki/%EC%A1%B0%EC%84%A0%EC%8B%9C%EB%8C%80" title="조선시대">조선시대</a>에 <a href="/wiki/%EC%95%88%EC%82%B0%EA%B5%B0" title="안산군">안산군</a> 대월면 선부리와 달산리로 불리었다가, <a href="/wiki/1914%EB%85%84" title="1914년">1914년</a> <a href="/wiki/3%EC%9B%94_1%EC%9D%BC" title="3월 1일">3월 1일</a> <a href="/wiki/%EC%8B%9C%ED%9D%A5%EA%B5%B0" title="시흥군">시흥군</a> 군자면 선부리로 개칭되었고<sup class="reference" id="cite_ref-2"><a href="#cite_note-2">[2]</a></sup>, 이어 <a href="/wiki/1986%EB%85%84" title="1986년">1986년</a> <a href="/wiki/1%EC%9B%94_1%EC%9D%BC" title="1월 1일">1월 1일</a> 안산시 선부동으로 되었다.<sup class="reference" id="cite_ref-3"><a href="#cite_note-3">[3]</a></sup> 그 후 <a href="/wiki/1991%EB%85%84" title="1991년">1991년</a> <a href="/wiki/11%EC%9B%94_18%EC%9D%BC" title="11월 18일">11월 18일</a> 인구증가에 따라 선부동이 선부1, 2동으로 분리 개칭되어 42개통 200개반으로 현재에 이르고 있다.<sup class="reference" id="cite_ref-4"><a href="#cite_note-4">[4]</a></sup> 폐동되기 전 자연취락 마을로 관모산이 있었는데 이는 현재 관산초등학교 뒷산이 관리가 쓴 제모처럼 생겨 관모산으로 불리다가 이산 서쪽 기슭에 마을이 형성되자 마을 이름을 ‘관모산’이라 칭하였으며 1983년 폐동 전까지 24호가 거주하였다.<sup class="reference" id="cite_ref-5"><a href="#cite_note-5">[5]</a></sup> <a href="/wiki/1993%EB%85%84" title="1993년">1993년</a> <a href="/wiki/12%EC%9B%94_6%EC%9D%BC" title="12월 6일">12월 6일</a> 선부2동에서 선부3동이 분리되었다.<sup class="reference" id="cite_ref-seonbu3_6-0"><a href="#cite_note-seonbu3-6">[6]</a></sup> <a href="/wiki/1995%EB%85%84" title="1995년">1995년</a> <a href="/wiki/4%EC%9B%94_25%EC%9D%BC" title="4월 25일">4월 25일</a> 시흥시 화정동 일부를 선부3동에 편입하였다.<sup class="reference" id="cite_ref-seonbu3_6-1"><a href="#cite_note-seonbu3-6">[6]</a></sup></p>
<h2><span class="mw-headline" id=".EB.B2.95.EC.A0.95.EB.8F.99">법정동</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%84%A0%EB%B6%80%EB%8F%99&amp;action=edit&amp;section=2" title="부분 편집: 법정동">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<ul>
<li>화정동 (3동)</li>
