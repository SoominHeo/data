<p><b>선유도</b>(仙遊島)는 <a href="/wiki/%EC%A0%84%EB%9D%BC%EB%B6%81%EB%8F%84" title="전라북도">전라북도</a> <a href="/wiki/%EA%B5%B0%EC%82%B0%EC%8B%9C" title="군산시">군산시</a> <a href="/wiki/%EC%98%A5%EB%8F%84%EB%A9%B4" title="옥도면">옥도면</a> 선유도리의 섬으로, <a href="/wiki/%EA%B3%A0%EA%B5%B0%EC%82%B0%EA%B5%B0%EB%8F%84" title="고군산군도">고군산군도</a>의 중심지이자 관광지로 유명하다. 면적은 2.13 ㎢, 해안선 길이는 12.8 ㎞이다.</p>

<h2><span class="mw-headline" id=".EC.97.AD.EC.82.AC">역사</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%84%A0%EC%9C%A0%EB%8F%84_(%EA%B5%B0%EC%82%B0%EC%8B%9C)&amp;action=edit&amp;section=1" title="부분 편집: 역사">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>선유도라는 이름은 섬의 경치가 매우 아름다워 신선이 놀았다 하여 부르게 된 것이라고 전하는데, 이 섬의 본래 이름은 군산도(群山島)였다.<sup class="reference" id="cite_ref-1"><a href="#cite_note-1">[1]</a></sup></p>
<p><a class="mw-redirect" href="/wiki/%EA%B3%A0%EB%A0%A4%EC%8B%9C%EB%8C%80" title="고려시대">고려시대</a>에는 고려-<a href="/wiki/%EC%86%A1%EB%82%98%EB%9D%BC" title="송나라">송나라</a> 무역로의 기항지였을 뿐만 아니라, <a href="/wiki/%EC%B5%9C%EB%AC%B4%EC%84%A0" title="최무선">최무선</a>이 왜구와의 전투에서 승리한 진포해전(鎭浦海戰)의 기지였다. 고려 때에 수군(水軍) 기지를 두고 군산진(群山鎭)이라 불렀는데, <a href="/wiki/%EC%A1%B0%EC%84%A0" title="조선">조선</a> <a class="mw-redirect" href="/wiki/%EC%84%B8%EC%A2%85%EB%8C%80%EC%99%95" title="세종대왕">세종대왕</a> 때에 수군 기지가 육지로 이전하면서 '군산'이라는 원래의 이름은 기지가 옮긴 육지로 가고, 이 지역에는 이름 앞에 '고'(古)자가 붙었다.</p>
<p>특히, <a href="/wiki/%EC%9D%B4%EC%88%9C%EC%8B%A0" title="이순신">이순신</a> 장군이 <a class="mw-redirect" href="/wiki/%EB%AA%85%EB%9F%89%ED%95%B4%EC%A0%84" title="명량해전">명량해전</a> 승리 후 이곳에서 열하루 동안 머물며 전열을 재정비하는 등 임진왜란 때는 함선의 정박기지로 기능을 수행했던 해상요지였다.<sup class="reference" id="cite_ref-2"><a href="#cite_note-2">[2]</a></sup> 선유도의 중심 마을인 진말(鎭里)은 군산진(群山鎭)이 있었던 곳으로, 마을 이름에 수군기지가 있었던 흔적을 고스란히 가지고 있다.</p>
<h2><span class="mw-headline" id=".EC.A7.80.EB.A6.AC">지리</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%84%A0%EC%9C%A0%EB%8F%84_(%EA%B5%B0%EC%82%B0%EC%8B%9C)&amp;action=edit&amp;section=2" title="부분 편집: 지리">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>섬은 남섬과 북섬으로 나눌 수 있는데, 남섬이 0.9 ㎢, 북섬이 1.2 ㎢이며, 그 사이에 긴 <a href="/wiki/%EC%82%AC%EC%A3%BC_(%EC%A7%80%EC%A7%88%ED%95%99)" title="사주 (지질학)">사주</a>(砂洲)가 발달되어 하나의 섬으로 연결하고 있다. 사주의 동쪽에는 <a class="mw-redirect" href="/wiki/%EA%B0%AF%EB%B2%8C" title="갯벌">갯벌</a>이 넓게 펼쳐져 있다. 북섬의 남쪽에는 망주봉(104.5 m), 북쪽에는 남악산(155.6 m), 남섬의 서쪽에는 선유봉(111 m)이 있다.</p>
<p>취락은 남섬에 집중되어 있는데, 남섬 북쪽에는 진말이, 남쪽에는 통계 마을이 있다. 북섬에는 새터, 밭너머, 남악 등 3개 소규모 마을이 있다. 주민들은 대부분 어업과 관광 관련 산업에 종사한다. 연근해에서는 멸치·조기·바지락·꼴뚜기·주꾸미 등이 많이 잡히며, 김과 굴 양식이 이루어진다. 농산물로 쌀·보리·고구마·마늘·고추 등이 소량 생산된다.</p>
<p>섬의 중심인 진말에 초·중학교, 보건지소, 우체국이 있다.</p>
<h2><span class="mw-headline" id=".EC.84.A0.EC.9C.A08.EA.B2.BD">선유8경</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%84%A0%EC%9C%A0%EB%8F%84_(%EA%B5%B0%EC%82%B0%EC%8B%9C)&amp;action=edit&amp;section=3" title="부분 편집: 선유8경">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>남섬과 북섬을 잇는 명사십리(明沙十里)의 선유도 <a href="/wiki/%ED%95%B4%EC%88%98%EC%9A%95%EC%9E%A5" title="해수욕장">해수욕장</a>은 길이 1.3 ㎞, 폭 50m 규모의 백사장이 펼쳐져 있는데, 고운 모래와 깨끗한 바닷물, 기암절벽과 낙조(<a href="/wiki/%EB%85%B8%EC%9D%84" title="노을">노을</a>)가 아름답기로 유명하다.</p>
<ul>
<li>귀양간 <a href="/wiki/%EC%84%A0%EB%B9%84" title="선비">선비</a>가 <a href="/wiki/%EC%99%95" title="왕">임금</a>을 그리며 흘리는 <a href="/wiki/%EB%88%88%EB%AC%BC" title="눈물">눈물</a>같다는 <b>망주폭포</b>(望主瀑布)</li>
<li>10리 길이의 <a href="/wiki/%EB%AA%A8%EB%9E%98" title="모래">모래</a>사장으로 유명한 선유도 <b>명사십리</b>(明沙十里)</li>
<li>모래사장에서 자라난 <a href="/wiki/%ED%8C%BD%EB%82%98%EB%AC%B4" title="팽나무">팽나무</a>가 마치 <a href="/wiki/%EA%B8%B0%EB%9F%AC%EA%B8%B0" title="기러기">기러기</a> 내려앉은 모습과 같다는 <b>평사낙안</b>(平沙落雁)</li>
<li>선유도 앞에 있는 3개의 섬의 모습이 마치 돛단배가 섬으로 들어오는 것처럼 보인다는 <b>삼도귀범</b>(三島歸帆)</li>
<li><a href="/wiki/%EC%9E%A5%EC%9E%90%EB%8F%84" title="장자도">장자도</a> 앞바다에서 <a href="/wiki/%EB%B0%A4_(%EC%8B%9C%EA%B0%84)" title="밤 (시간)">밤</a>에 조업하는 어선들의 불빛을 뜻하는 <b>장자어화</b>(壯子漁火)</li>

<li><a href="/wiki/%EC%8B%A0%EC%8B%9C%EB%8F%84" title="신시도">신시도</a>의 고운 <a href="/wiki/%EB%8B%A8%ED%92%8D" title="단풍">단풍</a>이 <a href="/wiki/%EB%8B%AC" title="달">달</a>빛 그림자와 함께 바다에 비친다는 <b>월영단풍</b>(月影丹楓)</li>

</ul>
<h2><span class="mw-headline" id=".EA.B5.90.ED.86.B5">교통</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%84%A0%EC%9C%A0%EB%8F%84_(%EA%B5%B0%EC%82%B0%EC%8B%9C)&amp;action=edit&amp;section=4" title="부분 편집: 교통">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>군산항 연안여객터미널·선유도간 정기 여객선이 운항된다.</p>
<p><a href="/wiki/1986%EB%85%84" title="1986년">1986년</a> 12월에 완공된 선유교와 장자교를 통해 각각 섬 동쪽의 <a href="/wiki/%EB%AC%B4%EB%85%80%EB%8F%84_(%EA%B5%B0%EC%82%B0%EC%8B%9C)" title="무녀도 (군산시)">무녀도</a>, 섬 서쪽의 <a href="/wiki/%EC%9E%A5%EC%9E%90%EB%8F%84" title="장자도">장자도</a>와 연결되는데, 두 다리는 길이 268 m, 폭 3 m, 높이 30 m, 설계하중 0.5톤(500kg)까지 똑같은 쌍둥이 다리이다. 좁은 폭과 설계하중 때문에 <a href="/wiki/%EC%9E%90%EC%A0%84%EA%B1%B0" title="자전거">자전거</a>나 <a class="mw-redirect" href="/wiki/%EC%98%A4%ED%86%A0%EB%B0%94%EC%9D%B4" title="오토바이">오토바이</a> 정도만 이용할 수 있는데, 장자교는 <a href="/wiki/2011%EB%85%84" title="2011년">2011년</a> <a href="/wiki/8%EC%9B%94_8%EC%9D%BC" title="8월 8일">8월 8일</a> 해상 크레인이 다리와 충돌해 다리 상판이 절단, 붕괴되는 사고까지 있어서 <a class="mw-redirect" href="/wiki/%EC%98%A4%ED%86%A0%EB%B0%94%EC%9D%B4" title="오토바이">오토바이</a>도 다닐 수 없다.<sup class="reference" id="cite_ref-3"><a href="#cite_note-3">[3]</a></sup></p>
<p><a href="/wiki/2007%EB%85%84" title="2007년">2007년</a> <a href="/wiki/%EC%A7%81%EB%8F%84" title="직도">직도</a>사격장 허가의 보상차원에서 <a class="mw-redirect" href="/wiki/%EC%83%88%EB%A7%8C%EA%B8%88%EB%B0%A9%EC%A1%B0%EC%A0%9C" title="새만금방조제">새만금방조제</a>~<a href="/wiki/%EC%8B%A0%EC%8B%9C%EB%8F%84" title="신시도">신시도</a>~<a href="/wiki/%EB%AC%B4%EB%85%80%EB%8F%84_(%EA%B5%B0%EC%82%B0%EC%8B%9C)" title="무녀도 (군산시)">무녀도</a>~선유도~<a href="/wiki/%EC%9E%A5%EC%9E%90%EB%8F%84" title="장자도">장자도</a>를 연결하는 폭 15.5 m의 왕복 2차선 도로(<a href="/wiki/%EA%B5%AD%EB%8F%84_%EC%A0%9C4%ED%98%B8%EC%84%A0" title="국도 제4호선">국도 제4호선</a>)가 건설 중인데, 국비 지원의 차질 등으로 공사가 지연되어 당초 계획보다 4년 늦은 <a href="/wiki/2017%EB%85%84" title="2017년">2017년</a> 말에 완전 개통될 예정이다.<sup class="reference" id="cite_ref-4"><a href="#cite_note-4">[4]</a></sup><sup class="reference" id="cite_ref-5"><a href="#cite_note-5">[5]</a></sup><sup class="reference" id="cite_ref-6"><a href="#cite_note-6">[6]</a></sup></p>
