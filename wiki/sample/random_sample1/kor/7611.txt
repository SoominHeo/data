<p><b>선종</b>(腺腫, adenoma)은 <a class="new" href="/w/index.php?title=%EC%84%A0%EC%83%81%ED%94%BC&amp;action=edit&amp;redlink=1" title="선상피 (없는 문서)">선상피</a>에서 발생하는 <a class="extiw" href="https://en.wikipedia.org/wiki/Benign_tumor" title="en:Benign tumor">양성 종양</a>(良性腫瘍)이다. 선종은 <a href="/wiki/%EA%B2%B0%EC%9E%A5" title="결장">결장</a>, <a href="/wiki/%EB%B6%80%EC%8B%A0" title="부신">부신</a>, <a href="/wiki/%EB%87%8C%ED%95%98%EC%88%98%EC%B2%B4" title="뇌하수체">뇌하수체</a>, <a class="mw-redirect" href="/wiki/%EA%B0%91%EC%83%81%EC%84%A0" title="갑상선">갑상선</a> 등의 여러 장기에서 발생할 수 있다. 선종은 <a class="extiw" href="https://en.wikipedia.org/wiki/Benign_tumor" title="en:Benign tumor">양성 종양</a>으로 시작하였더라도 <a class="extiw" href="https://en.wikipedia.org/wiki/Adenocarcinoma" title="en:Adenocarcinoma">선암</a>(腺癌)이라고 불리는 <a class="extiw" href="https://en.wikipedia.org/wiki/Malignancy" title="en:Malignancy">악성</a>으로 진행될 수 있다. 선종은 양성 상태일 동안 다른 장기를 압박하여 심각한 <a href="/wiki/%ED%95%A9%EB%B3%91%EC%A6%9D" title="합병증">합병증</a>을 유발시킬 수 있으며, <a href="/wiki/%ED%98%B8%EB%A5%B4%EB%AA%AC" title="호르몬">호르몬</a>의 생산을 제어하지 못하게 하여 많은 양의 호르몬을 만들어 내기도 한다.</p>
<h2><span class="mw-headline" id=".EC.A1.B0.EC.A7.81.EB.B3.91.EB.A6.AC.ED.95.99">조직병리학</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%84%A0%EC%A2%85_(%EC%A2%85%EC%96%91)&amp;action=edit&amp;section=1" title="부분 편집: 조직병리학">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>선종은 <a class="mw-disambig" href="/wiki/%EC%9C%84" title="위">위</a>의 <a href="/wiki/%EC%A0%90%EB%A7%89" title="점막">점막</a>, <a class="mw-disambig" href="/wiki/%EC%86%8C%EC%9E%A5" title="소장">소장</a>, <a href="/wiki/%EA%B2%B0%EC%9E%A5" title="결장">결장</a>, <a class="mw-redirect" href="/wiki/%EB%B6%84%EB%B9%84%EC%84%A0" title="분비선">분비선</a> 체계의 <a href="/wiki/%EC%83%81%ED%94%BC_%EC%A1%B0%EC%A7%81" title="상피 조직">상피 조직</a>에서 발생하는 상피성의 양성 종양이다. 속이 빈 장기 (<a class="new" href="/w/index.php?title=%EC%86%8C%ED%99%94%EA%B4%80&amp;action=edit&amp;redlink=1" title="소화관 (없는 문서)">소화관</a>)에서, 선종은 빈 공간에서 자라 <a class="extiw" href="https://en.wikipedia.org/wiki/Polyp_(medicine)" title="en:Polyp (medicine)">폴립</a> (또는 폴립 형태)가 된다. 첨가물의 종류에 따라, 선종은 <a href="/wiki/%EC%86%8C%EC%97%BD" title="소엽">소엽</a>의 머리와 길고 가느다란 줄기를 가진 육경이나 다양한 형태의 무경이 될 수 있다.</p>
<p>선종의 확산은 세포 <a class="new" href="/w/index.php?title=%EA%B3%BC%EC%97%BC%EC%83%89%EC%84%B1&amp;action=edit&amp;redlink=1" title="과염색성 (없는 문서)">과염색성</a>의 핵, (가짜) 층을 이룬 핵, <a href="/wiki/%ED%95%B5%EC%86%8C%EC%B2%B4" title="핵소체">핵소체</a>, <a href="/wiki/%EC%B2%B4%EC%84%B8%ED%8F%AC_%EB%B6%84%EC%97%B4" title="체세포 분열">체세포 분열</a> 등의 비정상적인 세포 (<a href="/wiki/%EC%9D%B4%ED%98%95%EC%84%B1" title="이형성">이형성</a> 또는 정상적인 상피의 감소)의 형성 이상의 차이 정도에 따라 특징지어진다. 선종은 관 모양이거나, <a href="/wiki/%EC%9C%B5%ED%84%B8" title="융털">융털</a>이 있거나, 관 모양의 융털이 있는 형태이다. 세포막과 점막의 하층부는 손상되지 않는다.</p>
