<p><b>선형대수학</b>(線型代數學, <span style="font-size: smaller;"><a href="/wiki/%EC%98%81%EC%96%B4" title="영어">영어</a>:</span> <span lang="en" xml:lang="en">linear algebra</span>)은 <a href="/wiki/%EB%B2%A1%ED%84%B0_%EA%B3%B5%EA%B0%84" title="벡터 공간">벡터 공간</a>, <a class="mw-redirect" href="/wiki/%EB%B2%A1%ED%84%B0_(%EC%84%A0%ED%98%95%EB%8C%80%EC%88%98%ED%95%99)" title="벡터 (선형대수학)">벡터</a>, <a href="/wiki/%EC%84%A0%ED%98%95_%EB%B3%80%ED%99%98" title="선형 변환">선형 변환</a>, <a href="/wiki/%ED%96%89%EB%A0%AC" title="행렬">행렬</a>, <a class="mw-redirect" href="/wiki/%EC%97%B0%EB%A6%BD_%EC%84%A0%ED%98%95_%EB%B0%A9%EC%A0%95%EC%8B%9D" title="연립 선형 방정식">연립 선형 방정식</a> 등을 연구하는 <a href="/wiki/%EB%8C%80%EC%88%98%ED%95%99" title="대수학">대수학</a>의 한 분야이다. 현대 선형대수학은 그중에서도 벡터 공간이 주 연구 대상이다. <a href="/wiki/%EC%B6%94%EC%83%81%EB%8C%80%EC%88%98%ED%95%99" title="추상대수학">추상대수학</a>, <a href="/wiki/%ED%95%A8%EC%88%98%ED%95%B4%EC%84%9D%ED%95%99" title="함수해석학">함수해석학</a>에 널리 쓰이고 있다.</p>
<p>선형대수학은 <a href="/wiki/%EC%9E%90%EC%97%B0%EA%B3%BC%ED%95%99" title="자연과학">자연과학</a>과 <a href="/wiki/%EA%B3%B5%ED%95%99" title="공학">공학</a>에도 널리 활용된다. 선형 연립방정식을 푸는 좋은 방법으로는 <a href="/wiki/%EC%86%8C%EA%B1%B0%EB%B2%95" title="소거법">소거법</a>과 <a href="/wiki/%ED%96%89%EB%A0%AC%EC%8B%9D" title="행렬식">행렬식</a>이 있다.</p>

<h2><span class="mw-headline" id=".EA.B8.B0.EC.B4.88">기초</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%84%A0%ED%98%95%EB%8C%80%EC%88%98%ED%95%99&amp;action=edit&amp;section=1" title="부분 편집: 기초">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>선형대수학은 2차원 혹은 3차원의 <a href="/wiki/%EC%A7%81%EA%B5%90_%EC%A2%8C%ED%91%9C%EA%B3%84" title="직교 좌표계">직교 좌표계</a>에 대한 연구로 부터 시작되었다.</p>
<p>선형대수학에서 기본적인 정의는 다음과 같다.</p>
<ul>
<li><a class="mw-redirect" href="/wiki/%EB%B2%A1%ED%84%B0_(%EC%84%A0%ED%98%95%EB%8C%80%EC%88%98%ED%95%99)" title="벡터 (선형대수학)">벡터</a>: 크기와 방향성을 갖는 성분. 물리학에서 주로 쓰인다.</li>
<li><b>벡터 연산</b>: 두 벡터끼리의 합, 혹은 벡터와 스칼라(크기만 있고 방향성은 없는 성분)사이의 곱이 벡터의 기본 연산이다.</li>
<li><b><a href="/wiki/%EB%B2%A1%ED%84%B0_%EA%B3%B5%EA%B0%84" title="벡터 공간">벡터 공간</a></b>: 벡터의 기본 연산을 만족하는 모든 벡터의 모음을 뜻한다.</li>
<li><b><a href="/wiki/%EC%B0%A8%EC%9B%90" title="차원">차원</a></b>: 흔히 평면을 2차원, 공간을 3차원이라고 부른다. 이때 차원을 구성하는 각각의 요소(3차원의 경우 x,y,z)는 서로 독립적인데 이에 대한 개념을 확장한 것이 바로 선형대수학의 차원이다.</li>
<li><b><a href="/wiki/%ED%96%89%EB%A0%AC" title="행렬">행렬</a></b>: 여러개의 숫자들을 직사각형의 모양으로 한데 묶어 나타낸 성분. 벡터를 하나의 행 혹은 하나의 열로 구성된 행렬로 볼 수도 있다.</li>
</ul>
<p>보통 3차원까지의 벡터는 그림 등으로 시각적 표현이 가능하지만 그 이상의 벡터는 벡터의 각 구성요소를 괄호 안에 나열함으로써 표기한다.</p>
<p>여러 가지 문제를 수학으로 해결하는 데 있어 선형대수학의 개념은 매우 중요한데, 선형화 혹은 선형 근사를 통해, 복잡한 비선형 방정식 문제를 간단한 선형 방정식 문제로 변환해 문제를 해결할 수 있기 때문이다.</p>

<h2><span class="mw-headline" id=".ED.95.99.EB.B6.80.EA.B3.BC.EC.A0.95">학부과정</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%84%A0%ED%98%95%EB%8C%80%EC%88%98%ED%95%99&amp;action=edit&amp;section=3" title="부분 편집: 학부과정">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>학부과정에서 가르치는 선형대수학의 내용들은 다음과 같다. 다만 이 내용은 일반적으로 이와 같이 가르치는 내용이며 각 학교마다 비중있게 다루는 부분이 다를 수 있고 내용을 추가하거나 뛰어넘을 수 있다.</p>
<ul>
<li><a class="mw-redirect" href="/wiki/%EB%B2%A1%ED%84%B0_(%EC%84%A0%ED%98%95%EB%8C%80%EC%88%98%ED%95%99)" title="벡터 (선형대수학)">벡터</a>와 <a href="/wiki/%ED%96%89%EB%A0%AC" title="행렬">행렬</a> : 벡터의 개념과 행렬의 개념에 대해 강의한다. 이에 대한 내용은 앞의 '기초'와 <a href="/wiki/%ED%96%89%EB%A0%AC" title="행렬">행렬</a>문서를 참고하라.</li>
</ul>
<ul>
<li><a class="mw-redirect" href="/wiki/%EA%B0%80%EC%9A%B0%EC%8A%A4-%EC%9A%94%EB%A5%B4%EB%8B%A8_%EC%86%8C%EA%B1%B0%EB%B2%95" title="가우스-요르단 소거법">가우스-요르단 소거법</a>: 가우스-요르단 소거법은 행렬의 행 간의 연산이다. 이 연산은 행렬로 구성된 방정식의 해를 구하는 방법을 제시한다. 또한 이 계산과정을 뒷받침하는 이론에 대해서도 공부하며, 소거법의 결과로 구해진 해를 해석하는 방법도 공부한다. 가우스-요르단 소거법은 방정식의 해를 보존할 수 있는 연산들로 이루어져 있으며, 세 가지가 존재한다.
<ul>
<li>1. 행렬의 행을 그 행의 상수배만큼으로 대체하여도 그 행렬 방정식의 해는 보존된다.</li>
<li>2. 행렬의 한 행의 상수배를 다른 행에 더하더라도 그 행렬 방정식의 해는 보존된다.</li>
<li>3. 행렬의 한 행과 다른 행을 교환하더라도 그 행렬 방정식의 해는 보존된다.</li>
</ul>
</li>
</ul>
<ul>
<li>행렬 <a href="/wiki/%EC%97%B0%EC%82%B0%EC%9E%90" title="연산자">연산자</a>와 특정 형태의 행렬: 행렬에 관계된 연산자들과 특정한 형태의 행렬에 대해 배운다. <a class="mw-redirect" href="/wiki/%EC%A0%84%EC%B9%98" title="전치">전치</a>, <a class="mw-redirect" href="/wiki/%ED%8A%B8%EB%A0%88%EC%9D%B4%EC%8A%A4_(%EC%88%98%ED%95%99)" title="트레이스 (수학)">트레이스</a>, <a class="mw-redirect" href="/wiki/%EC%97%AD%ED%96%89%EB%A0%AC" title="역행렬">역행렬</a>등이 중요한 행렬연산자이다. 특정한 형태의 행렬로는 <a href="/wiki/%EB%8B%A8%EC%9C%84%ED%96%89%EB%A0%AC" title="단위행렬">단위행렬</a>, <a class="new" href="/w/index.php?title=%EC%83%81%EB%B6%80%EC%82%BC%EA%B0%81%ED%96%89%EB%A0%AC&amp;action=edit&amp;redlink=1" title="상부삼각행렬 (없는 문서)">상부삼각행렬</a>, <a class="new" href="/w/index.php?title=%ED%95%98%EB%B6%80%EC%82%BC%EA%B0%81%ED%96%89%EB%A0%AC&amp;action=edit&amp;redlink=1" title="하부삼각행렬 (없는 문서)">하부삼각행렬</a>, <a href="/wiki/%EB%8C%80%EC%B9%AD%ED%96%89%EB%A0%AC" title="대칭행렬">대칭행렬</a> 등에 대해 배운다. 상부삼각행렬과 하부삼각행렬을 이용해 행렬을 표현하는 <a class="new" href="/w/index.php?title=LU%EB%B6%84%ED%95%B4%EB%B2%95&amp;action=edit&amp;redlink=1" title="LU분해법 (없는 문서)">LU분해법</a>도 배운다.</li>
</ul>
<ul>
<li><a class="mw-redirect" href="/wiki/%EC%84%A0%ED%98%95%EB%8F%85%EB%A6%BD" title="선형독립">선형독립</a>: 벡터들의 일차독립에 대해 가르친다. 행렬을 통한 일차독립 판별에 대해 공부한다.</li>
<li><a href="/wiki/%ED%96%89%EB%A0%AC%EC%8B%9D" title="행렬식">행렬식</a>(판별식): 행렬식의 정의와 행렬식을 구하는 방법을 공부한다. 또한 대수적으로 행렬식을 표현하고 행렬식에 관계된 정리들을 배운다.</li>
<li><a class="mw-redirect" href="/wiki/%EA%B3%A0%EC%9C%A0%EB%B2%A1%ED%84%B0" title="고유벡터">고유벡터</a>와 <a href="/wiki/%EA%B3%A0%EC%9C%B3%EA%B0%92" title="고윳값">고윳값</a>: 행렬의 고윳값과 고유벡터에 대해 공부한다. 행렬식을 통해 고윳값을 찾고, 고윳값과 가우스 소거법을 통해 고유벡터를 찾는 과정을 익힌다. 그 외에도 고윳값과 고유벡터에 관계된 정리들에 대해 공부한다.</li>
<li><a class="mw-redirect" href="/wiki/%EC%84%A0%ED%98%95%EC%97%B0%EC%82%B0%EC%9E%90" title="선형연산자">선형연산자</a>: 이 문서의 '선형'을 참조하라. 특히, 이 부분에서는 선형연산과 행렬 간의 상호성에 대해 주의 깊게 다룬다.</li>
<li><a href="/wiki/%EC%A7%81%EA%B5%90%ED%96%89%EB%A0%AC" title="직교행렬">직교행렬</a>: 직교화된 연산과 행렬에 대해 공부한다. 직교행렬이란 그것의 전치행렬과 그것의 역행렬이 같은 경우를 말한다. 직교화된 연산이란 연산대상 벡터의 크기가 보존되고 벡터들의 내적이 보존되는 경우를 말한다.</li>
</ul>
<ul>
<li><a href="/wiki/%EB%B2%A1%ED%84%B0_%EA%B3%B5%EA%B0%84" title="벡터 공간">벡터 공간</a>: 벡터 공간을 행렬을 통해 해석하는 방법을 익힌다. 선형연산과 행렬 간의 상호성과 마찬가지로 벡터 공간과 행렬 사이에는 깊은 상호성이 있다. 중요한 개념들로는 다음과 같은 것들이 있다.
<ul>
<li><a href="/wiki/%EA%B8%B0%EC%A0%80_(%EC%84%A0%ED%98%95%EB%8C%80%EC%88%98%ED%95%99)" title="기저 (선형대수학)">기저</a>, <a href="/wiki/%EC%B0%A8%EC%9B%90" title="차원">차원</a>: 기저란, 어떤 벡터 공간을 이루는 벡터들을 말한다. 이 벡터들은 일차독립이여야 하며, 이 벡터들의 선형조합으로 그 벡터 공간의 모든 벡터를 표현할 수 있어야 한다. 직관적인 예를 들면 x축, y축, z축은 3차원공간의 기저이다. 차원이란 기저를 구성하는 벡터들의 숫자를 말한다.</li>
<li><a class="new" href="/w/index.php?title=%EA%B8%B0%EB%B3%B8%EA%B3%B5%EA%B0%84&amp;action=edit&amp;redlink=1" title="기본공간 (없는 문서)">기본공간</a>, <a href="/wiki/%EC%B0%A8%EC%9B%90_%EC%A0%95%EB%A6%AC" title="차원 정리">차원 정리</a>, <a class="new" href="/w/index.php?title=%EA%B3%84%EC%88%98%EC%A0%95%EB%A6%AC&amp;action=edit&amp;redlink=1" title="계수정리 (없는 문서)">계수정리</a>, <a class="new" href="/w/index.php?title=%ED%94%BC%EB%B4%87%EC%A0%95%EB%A6%AC&amp;action=edit&amp;redlink=1" title="피봇정리 (없는 문서)">피봇정리</a>: 기본공간은 행렬과 벡터 공간 사이의 다리와 같은 역할을 한다. 기본공간에는 영공간, 행공간, 열공간 등이 있다. 차원정리, 계수정리, 피봇정리는 이 기본공간들의 차원과 기저에 대해 유용한 알고리즘을 제공한다.</li>
<li>벡터의 <a class="new" href="/w/index.php?title=%EC%A7%81%EA%B5%90%ED%99%94&amp;action=edit&amp;redlink=1" title="직교화 (없는 문서)">직교화</a>: 직교화된 벡터들에 대해 공부한다. 직교화된 벡터들이란 다른 벡터와의 내적값이 0인 벡터들을 의미한다.</li>
</ul>
</li>
</ul>
<ul>
<li><a class="mw-redirect" href="/wiki/%EA%B7%B8%EB%9E%8C-%EC%8A%88%EB%AF%B8%ED%8A%B8_%EC%A7%81%EA%B5%90%EC%A0%95%EA%B7%9C%ED%99%94" title="그람-슈미트 직교정규화">그람-슈미트 직교정규화</a>: 그람-슈미트 직교정규화를 통해 주어진 벡터의 집합을 직교화된 벡터의 집합으로 변환하는 법을 다룬다. 벡터의 직교화에서 배운 개념을 바탕으로 전개해 나간다.</li>
</ul>
<ul>
<li><a class="new" href="/w/index.php?title=%EC%83%81%EC%82%AC%EC%84%B1_(%EC%84%A0%ED%98%95%EB%8C%80%EC%88%98%ED%95%99)&amp;action=edit&amp;redlink=1" title="상사성 (선형대수학) (없는 문서)">상사성</a>과 <a class="new" href="/w/index.php?title=%EB%8C%80%EA%B0%81%ED%99%94&amp;action=edit&amp;redlink=1" title="대각화 (없는 문서)">대각화</a>: 상사성이란 두 행렬이 동일한 연산을 의미한다는 뜻이다. 즉, 두 행렬이 서로 다른 두 벡터 공간에서 동일한 연산을 처리하고 있다는 의미이다. 그러므로 상사성을 가진 두 행렬은 적당한 기저를 선택해서 서로를 표현할 수 있다. 대각화란 이 상사성을 계산 측면에서 응용한 것으로 특정 행렬을 대각행렬로 표현하는 과정이다.</li>
</ul>
<ul>
<li>이 외에도 복소수 <a href="/wiki/%EA%B3%A0%EC%9C%B3%EA%B0%92" title="고윳값">고윳값</a>, 벡터 공간의 공리 등에 대해 다루기도 한다.</li>
</ul>

<h2><span class="mw-headline" id=".EC.9C.A0.EC.9A.A9.ED.95.9C_.EC.A0.95.EB.A6.AC.EB.93.A4">유용한 정리들</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%84%A0%ED%98%95%EB%8C%80%EC%88%98%ED%95%99&amp;action=edit&amp;section=4" title="부분 편집: 유용한 정리들">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<ul>
<li>모든 벡터 공간은 <a href="/wiki/%EA%B8%B0%EC%A0%80_(%EC%84%A0%ED%98%95%EB%8C%80%EC%88%98%ED%95%99)" title="기저 (선형대수학)">기저</a>가 존재한다.</li>
<li>행렬의 역행렬이 존재할 <a href="/wiki/%ED%95%84%EC%9A%94%EC%B6%A9%EB%B6%84%EC%A1%B0%EA%B1%B4" title="필요충분조건">필요충분조건</a>은 <a href="/wiki/%ED%96%89%EB%A0%AC%EC%8B%9D" title="행렬식">행렬식</a>의 값이 0이 아니어야 한다.</li>
<li>행렬의 역행렬이 존재할 필요충분조건은 행렬로 표현할 수 있는 <a href="/wiki/%EC%84%A0%ED%98%95_%EB%B3%80%ED%99%98" title="선형 변환">선형 변환</a>이 <a href="/wiki/%EB%8F%99%ED%98%95_%EC%82%AC%EC%83%81" title="동형 사상">동형 사상</a>이어야 한다.</li>
<li>행렬의 <a href="/wiki/%EA%B3%A0%EC%9C%B3%EA%B0%92" title="고윳값">고윳값</a>들의 곱은 행렬식의 값과 같으며, 합은 행렬의 <a href="/wiki/%EB%8C%80%EA%B0%81%ED%95%A9" title="대각합">대각합</a>과 같다.</li>
</ul>
