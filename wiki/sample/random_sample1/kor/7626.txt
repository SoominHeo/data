<p><b>섬나라</b> 또는 <b>도국</b>(島國)은 모든 <a href="/wiki/%EC%98%81%ED%86%A0" title="영토">영토</a>가 <a href="/wiki/%EC%84%AC" title="섬">섬</a>으로만 구성된 국가이다. 국토의 사면(모든 면)이 <a href="/wiki/%EB%B0%94%EB%8B%A4" title="바다">바다</a>로 둘러싸여 있다.</p>
<p>섬나라와 반대되는 개념은 <a href="/wiki/%EB%82%B4%EB%A5%99%EA%B5%AD" title="내륙국">내륙국</a>이다. <a href="/wiki/%EC%98%A4%EC%8A%A4%ED%8A%B8%EB%A0%88%EC%9D%BC%EB%A6%AC%EC%95%84" title="오스트레일리아">오스트레일리아</a>는 사면이 <a href="/wiki/%EB%B0%94%EB%8B%A4" title="바다">바다</a>로 둘러싸여 있으나, <a href="/wiki/%EC%98%A4%EC%8A%A4%ED%8A%B8%EB%A0%88%EC%9D%BC%EB%A6%AC%EC%95%84_%EB%8C%80%EB%A5%99" title="오스트레일리아 대륙">오스트레일리아 대륙</a>은 <a href="/wiki/%EC%84%AC" title="섬">섬</a>으로 분류하지 않는다. 한국의 경우에는 역사상 <a href="/wiki/%EC%9A%B0%EC%82%B0%EA%B5%AD" title="우산국">우산국</a>과 <a class="mw-redirect" href="/wiki/%ED%83%90%EB%9D%BC%EA%B5%AD" title="탐라국">탐라국</a>이 섬나라였다. <b>도서 국가</b>(島嶼國家)라고도 한다.</p>
