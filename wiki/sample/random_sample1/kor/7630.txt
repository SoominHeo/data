성 게오르기우스 리본의 패턴은 불과 화약을 나타낸다. 또는 원래 <a class="mw-redirect" href="/wiki/%EB%9F%AC%EC%8B%9C%EC%95%84_%EC%A0%9C%EA%B5%AD%EC%9D%98_%EA%B5%AD%EC%9E%A5" title="러시아 제국의 국장">러시아 제국의 국장</a>에서 유래되었다고도 본다 (검은 독수리와 금색 배경).</div>

<p><b>성 게오르기우스 리본</b>(聖 ~ , <span style="font-size: smaller;"><a href="/wiki/%EB%9F%AC%EC%8B%9C%EC%95%84%EC%96%B4" title="러시아어">러시아어</a>:</span> <span lang="ru" xml:lang="ru">Георгиевская ленточка</span>, <i>Georgiyevskaya lentochka</i>)은 <a class="mw-disambig" href="/wiki/%ED%98%84%EB%8C%80" title="현대">현대</a> <a href="/wiki/%EB%9F%AC%EC%8B%9C%EC%95%84" title="러시아">러시아 연방</a>에서 가장 잘 알려지고 존중받는 군(軍)의 상징 중 하나이다. 도안은 <a class="mw-redirect" href="/wiki/%EA%B2%80%EC%A0%95%EC%83%89" title="검정색">검정색</a> 세 줄과 <a class="mw-redirect" href="/wiki/%EC%98%A4%EB%A0%8C%EC%A7%80%EC%83%89" title="오렌지색">오렌지색</a> 두 줄의 패턴으로 이루어져 있다. 성 게오르기우스 리본은 옛 <a href="/wiki/%EB%9F%AC%EC%8B%9C%EC%95%84_%EC%A0%9C%EA%B5%AD" title="러시아 제국">러시아 제국</a>, <a href="/wiki/%EC%86%8C%EB%A0%A8" title="소련">소련</a>과 현대 러시아에 이르기까지 많은 고급 <a href="/wiki/%ED%9B%88%EC%9E%A5" title="훈장">훈장</a>의 도안에 나타나고 있다. 러시아 연방에서 부활된 <a class="new" href="/w/index.php?title=%EC%84%B1_%EA%B2%8C%EC%98%A4%EB%A5%B4%EA%B8%B0%EC%9A%B0%EC%8A%A4_%ED%9B%88%EC%9E%A5&amp;action=edit&amp;redlink=1" title="성 게오르기우스 훈장 (없는 문서)">성 게오르기우스 훈장</a>과 <a class="new" href="/w/index.php?title=%EC%84%B1_%EA%B2%8C%EC%98%A4%EB%A5%B4%EA%B8%B0%EC%9A%B0%EC%8A%A4_%EC%8B%AD%EC%9E%90_%EB%A9%94%EB%8B%AC&amp;action=edit&amp;redlink=1" title="성 게오르기우스 십자 메달 (없는 문서)">성 게오르기우스 십자 메달</a>을 비롯, 소련의 <a class="new" href="/w/index.php?title=%EC%98%81%EC%98%88%ED%9B%88%EC%9E%A5&amp;action=edit&amp;redlink=1" title="영예훈장 (없는 문서)">영예훈장</a>에도 나타난다.</p>
<p>또한 러시아 애국주의의 상징으로도 사용되며, <a href="/wiki/%EC%A0%9C2%EC%B0%A8_%EC%84%B8%EA%B3%84_%EB%8C%80%EC%A0%84" title="제2차 세계 대전">제2차 세계 대전</a> 관련 기념식에도 자주 등장한다. <a href="/wiki/2014%EB%85%84" title="2014년">2014년</a> 들어 <a href="/wiki/%EC%9A%B0%ED%81%AC%EB%9D%BC%EC%9D%B4%EB%82%98" title="우크라이나">우크라이나</a> 특히 남동부 지역에서는 <a class="new" href="/w/index.php?title=%EC%95%88%ED%8B%B0%EB%A7%88%EC%9D%B4%EB%8B%A8&amp;action=edit&amp;redlink=1" title="안티마이단 (없는 문서)">안티마이단</a> 관련 운동가들과 <a class="mw-redirect" href="/wiki/%EC%B9%9C%EB%9F%AC%EC%8B%9C%EC%95%84" title="친러시아">러시아 민족주의</a>의 상징으로 사용되었다.<sup class="reference" id="cite_ref-1"><a href="#cite_note-1">[1]</a></sup><sup class="reference" id="cite_ref-colUWP_2-0"><a href="#cite_note-colUWP-2">[2]</a></sup> 또한 이 리본은 준군사 조직인 <a class="mw-redirect" href="/wiki/%EB%8F%88%EB%B0%94%EC%8A%A4_%EC%9D%B8%EB%AF%BC%EA%B5%B0" title="돈바스 인민군">돈바스 인민군</a>의 구성원들이 패용(佩用)한다. 반면 <a href="/wiki/%EC%9C%A0%EB%A1%9C%EB%A7%88%EC%9D%B4%EB%8B%A8" title="유로마이단">유로마이단</a> 활동가들은 성 게오르기우스 리본을 "콜로라도 리본"이라고 비하하기도 하는데, <a href="/wiki/%EC%BD%9C%EB%A1%9C%EB%9D%BC%EB%8F%84%EA%B0%90%EC%9E%90%EC%9E%8E%EB%B2%8C%EB%A0%88" title="콜로라도감자잎벌레">콜로라도감자잎벌레</a>의 색깔과 비슷하기 때문이다.<sup class="reference" id="cite_ref-colUWP_2-1"><a href="#cite_note-colUWP-2">[2]</a></sup><sup class="reference" id="cite_ref-3"><a href="#cite_note-3">[3]</a></sup></p>
