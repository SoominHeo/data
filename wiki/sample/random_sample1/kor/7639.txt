<p><b>성공회</b>(聖公會, <span style="font-size: smaller;"><a href="/wiki/%EC%98%81%EC%96%B4" title="영어">영어</a>:</span> <span lang="en" xml:lang="en">Anglicanism</span>)는 전 세계 165여 개국, 38개의 독립적이고 자치적인 성공회(<a href="/wiki/%EA%B4%80%EA%B5%AC" title="관구">관구</a>)로 이루어진 <a href="/wiki/%EA%B8%B0%EB%8F%85%EA%B5%90" title="기독교">기독교</a> 교파이며, 신자는 약 8천 5백만명이다.</p>

<h2><span class="mw-headline" id=".EA.B0.9C.EC.9A.94">개요</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%84%B1%EA%B3%B5%ED%9A%8C&amp;action=edit&amp;section=1" title="부분 편집: 개요">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>16세기 잉글랜드 종교개혁으로 "잉글랜드 교회"(English Church)에서 공식 명칭을 "<a href="/wiki/%EC%9E%89%EA%B8%80%EB%9E%9C%EB%93%9C_%EC%84%B1%EA%B3%B5%ED%9A%8C" title="잉글랜드 성공회">잉글랜드 성공회</a>"(Church of England)로 불러왔으며, 19세기 이후 다국가에 형성된 성공회를 제도적으로 정비하고 현재의 "세계성공회공동체"(Anglican Communion)로 칭하였다. 흔히 성공회하면 영국 국교회를 떠올리지만, "영국국교회"라는 용어는 성공회 전체를 지칭하는 것이 아니라, <a href="/wiki/%EC%9E%89%EA%B8%80%EB%9E%9C%EB%93%9C_%EC%84%B1%EA%B3%B5%ED%9A%8C" title="잉글랜드 성공회">잉글랜드 성공회</a>만을 가리킨다. 교파로는 초기 개혁 교회에서 파생된만큼 개혁 교회의 취지인 교황의 권위를 인정하지 않는 개신교적 전통이 존재하나, 사도들의 성사를 인정하고 있다는 점에서 대륙개신교와 비교가 된다. 따라서 개혁 교회 교파 안에 포함이 되어있다고 하더라도 대륙개신교(감리회, 장로회, 신루터파, 회중회 등)와 따로 분리해서 보기도 한다. 전 세계 성공회 중에서 성공회를 국교로 삼은 국가는 <a href="/wiki/%EC%9E%89%EA%B8%80%EB%9E%9C%EB%93%9C" title="잉글랜드">잉글랜드</a>가 유일하며, <a href="/wiki/%ED%95%9C%EA%B5%AD" title="한국">한국</a> 성공회의 공식 명칭은 <a href="/wiki/%EB%8C%80%ED%95%9C%EC%84%B1%EA%B3%B5%ED%9A%8C" title="대한성공회">대한성공회</a>(大韓聖公會)이다. <a href="/wiki/%EC%BA%94%ED%84%B0%EB%B2%A0%EB%A6%AC_%EB%8C%80%EC%A3%BC%EA%B5%90" title="캔터베리 대주교">캔터베리 대주교</a>는 <a href="/wiki/%EC%9E%89%EA%B8%80%EB%9E%9C%EB%93%9C_%EC%84%B1%EA%B3%B5%ED%9A%8C" title="잉글랜드 성공회">잉글랜드 성공회</a>의 최고위 성직자이자, 세계 성공회 공동체의 명예상 대표로서 활동한다.</p>
<h2><span class="mw-headline" id=".EA.B5.90.EB.8B.A8_.EB.AA.85.EC.B9.AD">교단 명칭</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%84%B1%EA%B3%B5%ED%9A%8C&amp;action=edit&amp;section=2" title="부분 편집: 교단 명칭">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<ul>
<li>성공회(聖公會)는 <a href="/wiki/%ED%95%9C%EC%9E%90" title="한자">한자</a>문화권인 <a href="/wiki/%ED%95%9C%EA%B5%AD" title="한국">한국</a>, <a href="/wiki/%EC%A4%91%EA%B5%AD" title="중국">중국</a>, <a href="/wiki/%EC%9D%BC%EB%B3%B8" title="일본">일본</a> 등에서 사용하는 교회이름으로, <a href="/wiki/%EC%82%AC%EB%8F%84%EC%8B%A0%EA%B2%BD" title="사도신경">사도신경</a>의 '거룩한 <a class="mw-redirect" href="/wiki/%EB%B3%B4%ED%8E%B8%EA%B5%90%ED%9A%8C" title="보편교회">보편교회</a>'(The Holy Catholic Church)를 한자로 옮긴 이름이다.</li>
</ul>
<h2><span class="mw-headline" id=".EC.A0.95.EC.B2.B4.EC.84.B1.EA.B3.BC_.EB.93.B1.EC.9E.A5">정체성과 등장</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%84%B1%EA%B3%B5%ED%9A%8C&amp;action=edit&amp;section=3" title="부분 편집: 정체성과 등장">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<h3><span class="mw-headline" id=".EC.A0.95.EC.B2.B4.EC.84.B1">정체성</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%84%B1%EA%B3%B5%ED%9A%8C&amp;action=edit&amp;section=4" title="부분 편집: 정체성">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>성공회는 자신의 정체성을 <b>개혁하는 <a class="mw-redirect" href="/wiki/%EB%B3%B4%ED%8E%B8%EA%B5%90%ED%9A%8C" title="보편교회">보편교회</a></b>(Reforming Catholic Church)<sup class="reference" id="cite_ref-1"><a href="#cite_note-1">[1]</a></sup>로 표현한다.</p>
<h3><span class="mw-headline" id=".EB.93.B1.EC.9E.A5_.EB.B0.B0.EA.B2.BD">등장 배경</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%84%B1%EA%B3%B5%ED%9A%8C&amp;action=edit&amp;section=5" title="부분 편집: 등장 배경">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<h4><span class="mw-headline" id=".EC.9E.89.EA.B8.80.EB.9E.9C.EB.93.9C_.EC.A2.85.EA.B5.90.EA.B0.9C.ED.98.81">잉글랜드 종교개혁</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%84%B1%EA%B3%B5%ED%9A%8C&amp;action=edit&amp;section=6" title="부분 편집: 잉글랜드 종교개혁">편집</a><span class="mw-editsection-bracket">]</span></span></h4>
<p>흔히 성공회하면 <a href="/wiki/%ED%97%A8%EB%A6%AC_8%EC%84%B8" title="헨리 8세">헨리 8세</a>가 <a href="/wiki/%EC%9D%B4%ED%98%BC" title="이혼">이혼</a>하려고 만든 종교라고 잘못 이해하는 사람들이 많이 있다. <a class="mw-disambig" href="/wiki/%EC%9D%B4%EC%9B%90%EB%B3%B5" title="이원복">이원복</a>작가가 그린 먼나라 이웃나라의 영향으로 생각되는데, 이는 종교개혁운동을 신학적으로 보지 못하고, 정치사회적으로 이해한 것이다. <a class="mw-redirect" href="/wiki/%EC%84%B8%EA%B3%84_%EC%84%B1%EA%B3%B5%ED%9A%8C_%EA%B3%B5%EB%8F%99%EC%B2%B4" title="세계 성공회 공동체">세계 성공회 공동체</a>(Anglican Communion)의 기원이 되는 <a href="/wiki/%EC%9E%89%EA%B8%80%EB%9E%9C%EB%93%9C_%EC%84%B1%EA%B3%B5%ED%9A%8C" title="잉글랜드 성공회">잉글랜드 성공회</a>(the Church of England)는 <a href="/wiki/%EB%8F%85%EC%9D%BC" title="독일">독일</a>,<a href="/wiki/%EB%84%A4%EB%8D%9C%EB%9E%80%EB%93%9C" title="네덜란드">네덜란드</a>, <a href="/wiki/%EC%8A%A4%EC%9C%84%EC%8A%A4" title="스위스">스위스</a>, <a href="/wiki/%ED%94%84%EB%9E%91%EC%8A%A4" title="프랑스">프랑스</a>, <a href="/wiki/%EC%9E%89%EA%B8%80%EB%9E%9C%EB%93%9C" title="잉글랜드">잉글랜드</a> 등에서 진행된 <a href="/wiki/16%EC%84%B8%EA%B8%B0" title="16세기">16세기</a> 유럽<a class="mw-redirect" href="/wiki/%EC%A2%85%EA%B5%90%EA%B0%9C%ED%98%81" title="종교개혁">종교개혁</a>의 일부이다. 독일의 개신교 예전학자인 빌리암 나아겔이 쓴 《그리스도교의 예배의 역사》(박근원 옮김,대한기독교서회)에 따르면 종교개혁자 <a href="/wiki/%EB%A7%88%EB%A5%B4%ED%8B%B4_%EB%A3%A8%ED%84%B0" title="마르틴 루터">마르틴 루터</a> 등의 사상이 1520년대 잉글랜드에 영향을 주어 잉글랜드 교회에서의 교회개혁운동이 움트고 있었다. 잉글랜드 종교개혁이전에도 <a class="mw-redirect" href="/wiki/%EC%9C%84%ED%81%B4%EB%A6%AC%ED%94%84" title="위클리프">위클리프</a>(Wycliff)가 실천한 영어성서번역같은 교회개혁운동이 실천되고 있었다. 특히 당시 영국을 지배하던 로마 가톨릭 교회에 대한 반대와 반성직자 운동이 이미 곳곳에서 일어나고 있었다.</p>
<h4><span class="mw-headline" id=".EC.98.81.EC.96.B4.EA.B8.B0.EB.8F.84.EC.84.9C">영어기도서</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%84%B1%EA%B3%B5%ED%9A%8C&amp;action=edit&amp;section=7" title="부분 편집: 영어기도서">편집</a><span class="mw-editsection-bracket">]</span></span></h4>
<p>잉글랜드 종교개혁의 상징은 <a href="/wiki/%ED%86%A0%EB%A8%B8%EC%8A%A4_%ED%81%AC%EB%9E%9C%EB%A8%B8" title="토머스 크랜머">토머스 크랜머</a>전 캔터베리 <a href="/wiki/%EB%8C%80%EC%A3%BC%EA%B5%90" title="대주교">대주교</a>가 이끌었고, 그 결과물이 현재 성공회의 큰 자산인 잉글랜드 성공회 <a class="mw-disambig" href="/wiki/%EC%98%88%EC%A0%84" title="예전">예전</a>(Liturgy)을 담은 책인 영어<a class="mw-redirect" href="/wiki/%EC%84%B1%EA%B3%B5%ED%9A%8C%EA%B8%B0%EB%8F%84%EC%84%9C" title="성공회기도서">성공회기도서</a>(The Book of Common Prayer)이다. <a href="/wiki/1549%EB%85%84" title="1549년">1549년</a> <a href="/wiki/%EC%98%81%EC%96%B4" title="영어">영어</a>로 쓴 성공회 기도서의 등장으로 <a href="/wiki/%EC%98%81%EA%B5%AD" title="영국">영국</a>의 <a href="/wiki/%EA%B8%B0%EB%8F%85%EA%B5%90%EC%9D%B8" title="기독교인">기독교인</a>들은 공부를 많이 한 학자가 아니면 무슨 뜻인지 읽을 수 없는 <a class="new" href="/w/index.php?title=%EB%9D%BC%ED%8B%B4%EB%A7%90&amp;action=edit&amp;redlink=1" title="라틴말 (없는 문서)">라틴어</a>가 아닌, <a href="/wiki/%EB%AC%B8%ED%95%B4" title="문해">문해</a>자가 아니라면 누구나 이해할 수 있는 영어로써 <a href="/wiki/%EC%98%88%EB%B0%B0" title="예배">예배</a>하였다. <a class="mw-redirect" href="/wiki/%EC%84%B1%EA%B3%B5%ED%9A%8C%EA%B8%B0%EB%8F%84%EC%84%9C" title="성공회기도서">성공회기도서</a>는 현재에도 성공회의 신앙생활, 예배생활, <a href="/wiki/%EC%8B%A0%ED%95%99" title="신학">신학</a>과 영성의 중요한 바탕이 된다.</p>
<h2><span class="mw-headline" id=".EB.B0.9C.EC.A0.84">발전</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%84%B1%EA%B3%B5%ED%9A%8C&amp;action=edit&amp;section=8" title="부분 편집: 발전">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<h3><span class="mw-headline" id=".EC.84.B8.EA.B3.84_.EC.84.B1.EA.B3.B5.ED.9A.8C_.EA.B3.B5.EB.8F.99.EC.B2.B4">세계 성공회 공동체</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%84%B1%EA%B3%B5%ED%9A%8C&amp;action=edit&amp;section=9" title="부분 편집: 세계 성공회 공동체">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<div class="thumb tright">
<div class="thumbinner" style="width:222px;"><a class="image" href="/wiki/%ED%8C%8C%EC%9D%BC:Compassrose_Flag.svg"><img alt="" class="thumbimage" data-file-height="1151" data-file-width="1726" height="147" src="//upload.wikimedia.org/wikipedia/commons/thumb/6/6c/Compassrose_Flag.svg/220px-Compassrose_Flag.svg.png" srcset="//upload.wikimedia.org/wikipedia/commons/thumb/6/6c/Compassrose_Flag.svg/330px-Compassrose_Flag.svg.png 1.5x, //upload.wikimedia.org/wikipedia/commons/thumb/6/6c/Compassrose_Flag.svg/440px-Compassrose_Flag.svg.png 2x" width="220"/></a>
<div class="thumbcaption">
<div class="magnify"><a class="internal" href="/wiki/%ED%8C%8C%EC%9D%BC:Compassrose_Flag.svg" title="실제 크기로"></a></div>
세계성공회공동체 기</div>

<p>근대 성공회는 <a class="mw-redirect" href="/wiki/%EB%8C%80%EC%98%81%EC%A0%9C%EA%B5%AD" title="대영제국">대영제국</a>의 <a href="/wiki/%EC%8B%9D%EB%AF%BC%EC%A7%80" title="식민지">식민지</a> 건설과 <a href="/wiki/%EC%84%A0%EA%B5%90%EC%82%AC" title="선교사">선교사</a>들의 선교활동으로 <a href="/wiki/%EC%9E%89%EA%B8%80%EB%9E%9C%EB%93%9C_%EC%84%B1%EA%B3%B5%ED%9A%8C" title="잉글랜드 성공회">잉글랜드 성공회</a>(Church of England)에서 세계성공회(the Anglican Communion)로 발전하는 두 단계의 과정을 겪는다.</p>
<p>첫 단계는 <a href="/wiki/17%EC%84%B8%EA%B8%B0" title="17세기">17세기</a>에 <a href="/wiki/%EC%98%81%EA%B5%AD" title="영국">영국</a>의 <a href="/wiki/%EC%8B%9D%EB%AF%BC%EC%A7%80" title="식민지">식민지</a> 개척정책에 의해 <strong class="selflink">성공회</strong>가 <a href="/wiki/%EC%98%A4%EC%8A%A4%ED%8A%B8%EB%A0%88%EC%9D%BC%EB%A6%AC%EC%95%84" title="오스트레일리아">오스트레일리아</a>와 <a href="/wiki/%EC%BA%90%EB%82%98%EB%8B%A4" title="캐나다">캐나다</a>, <a href="/wiki/%EB%89%B4%EC%A7%88%EB%9E%9C%EB%93%9C" title="뉴질랜드">뉴질랜드</a>, <a href="/wiki/%EB%82%A8%EC%95%84%ED%94%84%EB%A6%AC%EC%B9%B4" title="남아프리카">남아프리카</a> 등으로 퍼졌다.</p>
<p>두 번째 단계는 <a href="/wiki/18%EC%84%B8%EA%B8%B0" title="18세기">18세기</a>에 펼쳐지는데, 이때 성공회는 전 세계로 확대되었다. 이는 <a href="/wiki/%EC%9E%89%EA%B8%80%EB%9E%9C%EB%93%9C" title="잉글랜드">잉글랜드</a>, <a href="/wiki/%EC%95%84%EC%9D%BC%EB%9E%9C%EB%93%9C" title="아일랜드">아일랜드</a>, <a href="/wiki/%EC%9B%A8%EC%9D%BC%EC%8A%A4" title="웨일스">웨일스</a> 성공회의 선교 노력이 낳은 결과였다.</p>
<p>이를 바탕으로 <a href="/wiki/19%EC%84%B8%EA%B8%B0" title="19세기">19세기</a>에 들어서 독립되고 자치적인 지역 성공회 교회들을 곧 관구들이 현재 세계성공회 공동체의 기틀을 마련하였다.</p>
<h3><span class="mw-headline" id=".EB.9E.8C.EB.B2.A0.EC.8A.A4_.ED.9A.8C.EC.9D.98">람베스 회의</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%84%B1%EA%B3%B5%ED%9A%8C&amp;action=edit&amp;section=10" title="부분 편집: 람베스 회의">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p><a class="mw-redirect" href="/wiki/%EC%84%B8%EA%B3%84%EC%84%B1%EA%B3%B5%ED%9A%8C" title="세계성공회">세계성공회</a>(the Anglican Communion)에 속한 각 지역 성공회 교회들은 교회는 교회가 위치한 국가나 지역에 따라 자치적이고 독립적인 교회로 성장했다.</p>
<p><a href="/wiki/19%EC%84%B8%EA%B8%B0" title="19세기">19세기</a> 말부터는 그 역사적 기원과 신학적 전통을 공유하는 하나된 교회의 인식을 강화하기 시작했다. 세계 성공회 공동체의 당면한 문제를 협의하기 위하여 전 세계 성공회 주교회의인 <a class="new" href="/w/index.php?title=%EB%9E%8C%EB%B2%A0%EC%8A%A4_%ED%9A%8C%EC%9D%98&amp;action=edit&amp;redlink=1" title="람베스 회의 (없는 문서)">람베스 회의</a>를 약 10년마다 열어 모인다. 람베스 회의는 치리 권한을 가진 법적 기구가 아니라 협의 기구이다. <a href="/wiki/%EC%BA%94%ED%84%B0%EB%B2%A0%EB%A6%AC_%EB%8C%80%EC%A3%BC%EA%B5%90" title="캔터베리 대주교">캔터베리 대주교</a>는 세계 성공회 공동체의 일치를 드러내는 상징이지만, 치리 권한을 갖지 않으며 각 나라의 성공회는 서로 동등한 자치적인 권한을 갖는다.</p>
<p><a href="/wiki/1888%EB%85%84" title="1888년">1888년</a> 람베스 회의에서는 세계성공회뿐만 아니라 전 세계 <a href="/wiki/%EA%B8%B0%EB%8F%85%EA%B5%90" title="기독교">기독교</a>의 일치를 위한 신앙적 기준을 마련했는데 이는 기독교의 기본 신앙 선언을 함축하고 있다.</p>
<p>람베스-시카고 4개 조항(the Lambeth-Chicago Quadrilateral, <a href="/wiki/1888%EB%85%84" title="1888년">1888년</a>)이라 불리는 이 선언의 내용은 다음과 같다.</p>
<ol>
<li><a class="mw-redirect" href="/wiki/%EA%B5%AC%EC%95%BD%EC%84%B1%EC%84%9C" title="구약성서">구약성서</a>과 <a class="mw-redirect" href="/wiki/%EC%8B%A0%EC%95%BD%EC%84%B1%EC%84%9C" title="신약성서">신약성서</a> 66권은 “<a href="/wiki/%EA%B5%AC%EC%9B%90" title="구원">구원</a>에 필요한 모든 것을 담고 있는” 하느님의 계시된 말씀이다.</li>
<li><a class="mw-redirect" href="/wiki/%EC%B4%88%EB%8C%80%EA%B5%90%ED%9A%8C" title="초대교회">초대교회</a>의 신앙고백인 <a href="/wiki/%EC%82%AC%EB%8F%84%EC%8B%A0%EA%B2%BD" title="사도신경">사도신경</a>과 <a href="/wiki/%EB%8B%88%EC%BC%80%EC%95%84-%EC%BD%98%EC%8A%A4%ED%83%84%ED%8B%B0%EB%85%B8%ED%8F%B4%EB%A6%AC%EC%8A%A4_%EC%8B%A0%EA%B2%BD" title="니케아-콘스탄티노폴리스 신경">니케아 신경</a>은 <a href="/wiki/%EA%B8%B0%EB%8F%85%EA%B5%90" title="기독교">기독교</a> 신앙을 드러내기에 충분한 선언이다.</li>
<li><a class="mw-redirect" href="/wiki/%EC%84%B8%EB%A1%80%EC%84%B1%EC%82%AC" title="세례성사">세례성사</a>와 <a class="mw-redirect" href="/wiki/%EC%84%B1%EC%B0%AC%EB%A1%80" title="성찬례">성찬례</a>는 <a href="/wiki/%EA%B7%B8%EB%A6%AC%EC%8A%A4%EB%8F%84" title="그리스도">그리스도</a>께서 친히 제정하신 두 가지 <a href="/wiki/%EC%84%B1%EC%82%AC" title="성사">성사</a>(Sacrament,성례전)이다.</li>
<li>역사적 <a href="/wiki/%EC%A3%BC%EA%B5%90" title="주교">주교</a>직(the historical episcopate)은 <a href="/wiki/%EA%B5%90%ED%9A%8C" title="교회">교회</a>의 일치를 위한 적절한 치리 방법이며, 그 형태는 다양할 수 있다.</li>
</ol>
<p>람베스 회의의 발전 속에서 거듭된 관심사는 지역 관구 교회의 독립성에 대한 인정과 교회의 일치에 관한 것이었다.</p>
<p>모순되는 듯한 이 해결책을 람베스 회의와 세계성공회는 “가시적 친교”라는 소명 속에서 “교제”(fellowship)와 “상호 친교”(intercommunion), 그리고 “상호 책임”(mutual responsibility)이라는 개념으로 발전시켰고, 이를 위해 구체적인 협의 기구를 형성하여 일치를 유지하려고 애썼다.</p>

<p>성공회는 <a href="/wiki/1867%EB%85%84" title="1867년">1867년</a>부터 <a href="/wiki/2008%EB%85%84" title="2008년">2008년</a>까지 모두 14차례에 걸친 회의를 통하여 세계 성공회의 일치와 <a class="mw-redirect" href="/wiki/%EA%B0%9C%EC%8B%A0%EA%B5%90%ED%9A%8C" title="개신교회">개신교회</a>, <a href="/wiki/%EB%A1%9C%EB%A7%88_%EA%B0%80%ED%86%A8%EB%A6%AD%EA%B5%90%ED%9A%8C" title="로마 가톨릭교회">로마 가톨릭교회</a>, <a class="mw-redirect" href="/wiki/%EC%A0%95%EA%B5%90%ED%9A%8C" title="정교회">정교회</a> 등 다른 전통 교단들과 나누는 교제와 일치 등에 대한 이해를 발전시켰다.</p>
<p>그 특징은 포용적인 태도, <a href="/wiki/%EB%AF%BC%EC%A1%B1" title="민족">민족</a>적이고 자치적인 태도, 그리고 일치를 지향하는 태도라고 할 수 있다. 세계 성공회 공동체는 이러한 신학적 태도와 교회 이해에 기초하여 <a href="/wiki/20%EC%84%B8%EA%B8%B0" title="20세기">20세기</a> 중반부터 활성화되기 시작한 <a class="mw-redirect" href="/wiki/%EA%B5%90%ED%9A%8C%EC%9D%BC%EC%B9%98%EC%9A%B4%EB%8F%99" title="교회일치운동">교회일치운동</a>에 <a href="/wiki/%EC%8B%A0%ED%95%99" title="신학">신학</a>적 근거를 제공하는 한편, 주도적인 역할을 담당해 왔다.<sup class="정리_필요 noprint">[<a href="/wiki/%EC%9C%84%ED%82%A4%EB%B0%B1%EA%B3%BC:%EC%B6%9C%EC%B2%98_%EB%B0%9D%ED%9E%88%EA%B8%B0" title="위키백과:출처 밝히기">출처 필요</a>]</sup></p>
<p>한편, 21세기에 들어서 <a href="/wiki/%EB%AF%B8%EA%B5%AD_%EC%84%B1%EA%B3%B5%ED%9A%8C" title="미국 성공회">미국 성공회</a>, <a href="/wiki/%EC%BA%90%EB%82%98%EB%8B%A4_%EC%84%B1%EA%B3%B5%ED%9A%8C" title="캐나다 성공회">캐나다 성공회</a>의 <a href="/wiki/%EB%8F%99%EC%84%B1%EC%95%A0" title="동성애">동성애</a>자에 대한 시민 결합 축복, 그리고 성직 서품 허용으로 성공회 내 신학 논쟁이 격화되었고 여성<a href="/wiki/%EC%A3%BC%EA%B5%90" title="주교">주교</a>서품 등을 둘러싸고 <a class="mw-redirect" href="/wiki/%EC%84%B8%EA%B3%84_%EC%84%B1%EA%B3%B5%ED%9A%8C_%EA%B3%B5%EB%8F%99%EC%B2%B4" title="세계 성공회 공동체">세계 성공회 공동체</a> 내부에서 논쟁이 계속되고 있다.</p>
<p>성공회는 획일성과 강요가 아닌 토론과 관용을 통해 <a href="/wiki/%EA%B5%90%ED%9A%8C" title="교회">교회</a>가 가진 문제를 해결해가야 한다는 입장(성공회 용어로 '모호성')에 따라, <a href="/wiki/%EA%B5%90%ED%9A%8C" title="교회">교회</a>의 교리 내용과 범위를 명시적으로 정하지 않는다.</p>
<p>이런 점에서 <a href="/wiki/%EB%8F%99%EC%84%B1%EC%95%A0" title="동성애">동성애</a>, 여성 주교 서품 등에 관한 각 지역 성공회 관구의 입장은 해당 관구의 사회 문화 정치적 상황에 크게 의존하는 경향을 보인다. 따라서 성적 소수자와 여성의 <a href="/wiki/%EC%9D%B8%EA%B6%8C" title="인권">인권</a>을 존중하는 미국과 캐나다 등과 다른 국가들간의 입장 차이가 크다.</p>
<p>성공회의 최고 영적 지도자는 교회 전통에 따라 <a class="mw-disambig" href="/wiki/%EC%98%81%EA%B5%AD_%EC%84%B1%EA%B3%B5%ED%9A%8C" title="영국 성공회">영국 성공회</a>의 <a href="/wiki/%EC%BA%94%ED%84%B0%EB%B2%A0%EB%A6%AC_%EB%8C%80%EC%A3%BC%EA%B5%90" title="캔터베리 대주교">캔터베리 대주교</a>이다. 그러나 개별 국가나 지역별 성공회는 독립적이고 자주적인 <a href="/wiki/%EA%B4%80%EA%B5%AC" title="관구">관구</a>교회들이기 때문에 <a href="/wiki/%EC%BA%94%ED%84%B0%EB%B2%A0%EB%A6%AC_%EB%8C%80%EC%A3%BC%EA%B5%90" title="캔터베리 대주교">캔터베리 대주교</a>는 다른 관구에 대한 치리 권한이 전혀 없다. 지역 관구 교회 안에서 일어나는 교리 해석상의 변경이나 내부 조정을 위한 권위는 그 개별 관구의 의회(관구 의회)를 통해서 결정된다.</p>
<p>현재 세계 성공회 내의 논란과 갈등은 주로 <a href="/wiki/%EC%95%84%ED%94%84%EB%A6%AC%EC%B9%B4" title="아프리카">아프리카</a> 지역에 분포한 성공회 관구들을 중심으로 한 소위 <a class="new" href="/w/index.php?title=%EA%B8%80%EB%A1%9C%EB%B2%8C_%EC%82%AC%EC%9A%B0%EC%8A%A4_%EA%B7%B8%EB%A3%B9&amp;action=edit&amp;redlink=1" title="글로벌 사우스 그룹 (없는 문서)">글로벌 사우스 그룹</a>과 미국, 캐나다 성공회 등 진보적인 성공회 관구 사이에서 발생하고 있다.</p>
<h2><span class="mw-headline" id=".EC.84.B1.EA.B3.B5.ED.9A.8C_.EC.8B.A0.ED.95.99.EC.9D.98_.EC.84.B8_.ED.9D.90.EB.A6.84">성공회 신학의 세 흐름</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%84%B1%EA%B3%B5%ED%9A%8C&amp;action=edit&amp;section=12" title="부분 편집: 성공회 신학의 세 흐름">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<div class="thumb tright">
<div class="thumbinner" style="width:222px;"><a class="image" href="/wiki/%ED%8C%8C%EC%9D%BC:Book_of_common_prayer_1596.jpg"><img alt="" class="thumbimage" data-file-height="195" data-file-width="300" height="143" src="//upload.wikimedia.org/wikipedia/commons/thumb/c/c7/Book_of_common_prayer_1596.jpg/220px-Book_of_common_prayer_1596.jpg" srcset="//upload.wikimedia.org/wikipedia/commons/c/c7/Book_of_common_prayer_1596.jpg 1.5x" width="220"/></a>

1596년판 성공회 기도서</div>
</div>
</div>
<p>성공회는 역사적으로 다양한 신학적 입장들이 서로 견제하며 공존하고 있는 전통을 유지하며 발전했다. 아래에 서술되는 신학적 경향은 이해의 편의를 위한 것일 뿐, 분명하게 나뉘는 것은 아니다.</p>
<h3><span class="mw-headline" id=".EA.B3.A0.EA.B5.90.ED.9A.8C.ED.8C.8C">고교회파</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%84%B1%EA%B3%B5%ED%9A%8C&amp;action=edit&amp;section=13" title="부분 편집: 고교회파">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p><a href="/wiki/%EA%B3%A0%EA%B5%90%ED%9A%8C%ED%8C%8C" title="고교회파">고교회파</a>(High Church)는 <a href="/wiki/%EA%B5%90%ED%9A%8C" title="교회">교회</a>의 성사성과 그 권위를 '높이' 평가한다는 의미에서 나온 말이다. 이 경향은 16세기 이래로 지속되었다. 특히 <a href="/wiki/19%EC%84%B8%EA%B8%B0" title="19세기">19세기</a> 중엽에 영국에서 일어난 <a href="/wiki/%EC%98%A5%EC%8A%A4%ED%8D%BC%EB%93%9C_%EC%9A%B4%EB%8F%99" title="옥스퍼드 운동">옥스퍼드 운동</a>은 <a class="mw-disambig" href="/wiki/%EC%98%81%EA%B5%AD_%EC%84%B1%EA%B3%B5%ED%9A%8C" title="영국 성공회">영국 성공회</a>가 <a class="mw-redirect mw-disambig" href="/wiki/%EC%98%81%EA%B5%AD_%EA%B5%AD%EA%B5%90%ED%9A%8C" title="영국 국교회">영국 국교회</a>로 안주하는 것에 반대하였으며, <a class="mw-redirect" href="/wiki/%EC%82%AC%EB%8F%84%EC%A0%81_%EC%A0%84%EC%8A%B9" title="사도적 전승">사도적 전승</a>에 따른 <a href="/wiki/%EA%B5%90%ED%9A%8C" title="교회">교회</a>의 역사적 연속성과 <a href="/wiki/%EC%84%B1%EC%82%AC" title="성사">성사</a>적 신앙을 강조했다. 이 흐름에 따르는 이들은 세속 사회의 산업화와 급진주의, 비종교적 사고의 확산에 비판적인 시각을 발전시켰고, 성스러운 질서와 권위를 강조하면서 교회의 권위와 교회의 가시적 일치, <a href="/wiki/%EC%A0%84%EB%A1%80" title="전례">전례</a>와 <a href="/wiki/%EC%84%B1%EC%82%AC" title="성사">성사</a>를 중요시한다.</p>
<h3><span class="mw-headline" id=".EC.A0.80.EA.B5.90.ED.9A.8C.ED.8C.8C">저교회파</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%84%B1%EA%B3%B5%ED%9A%8C&amp;action=edit&amp;section=14" title="부분 편집: 저교회파">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p><a href="/wiki/%EC%A0%80%EA%B5%90%ED%9A%8C%ED%8C%8C" title="저교회파">저교회파</a>(Low Church)는 위의 고교회에 대한 상대적 표현이다. 즉 교회의 성사성과 그 권위을 대체로 '낮게' 평가한다는 의미에서 나온 말이다. 이 경향도 16세기 이래로 지속되었다. 대체로 영국 성공회 교회 내 <a href="/wiki/%EC%B2%AD%EA%B5%90%EB%8F%84" title="청교도">청교도</a>의 전통이라고 할 수 있다. 특히 <a href="/wiki/18%EC%84%B8%EA%B8%B0" title="18세기">18세기</a> 영국 성공회의 <a href="/wiki/%EB%B3%B5%EC%9D%8C%EC%A3%BC%EC%9D%98" title="복음주의">복음주의</a> 운동은 개신교 <a class="mw-redirect" href="/wiki/%EC%A2%85%EA%B5%90%EA%B0%9C%ED%98%81" title="종교개혁">종교개혁</a> 전통에 따라 <a href="/wiki/%EC%84%B1%EB%A0%B9" title="성령">성령</a>에 의한 개인의 회심, <a class="mw-redirect" href="/wiki/%EC%84%B1%EC%84%9C" title="성서">성서</a>의 최우위성, <a href="/wiki/%EB%B3%B5%EC%9D%8C" title="복음">복음</a>의 설교에 강조를 두었으며, 신앙에 의한 의인 교리를 강조하였다. 이 시기 지금의 세계성공회공동체 모체가 되는 잉글랜드 성공회 사제 <a href="/wiki/%EC%A1%B4_%EC%9B%A8%EC%8A%AC%EB%A6%AC" title="존 웨슬리">존 웨슬리</a> 신부의 신앙 쇄신 운동은 원래 고교회 전통에서 기원했지만, 그의 운동은 점차로 <a href="/wiki/%EB%B3%B5%EC%9D%8C%EC%A3%BC%EC%9D%98" title="복음주의">복음주의</a> 운동으로 발전했다. 하지만 당시 성공회의 감리회 활동 금지명령과 웨슬리 사목 활동 정지 등의 조치<sup class="reference" id="cite_ref-2"><a href="#cite_note-2">[2]</a></sup>로 웨슬리의 의도와는 달리, 이 운동은 <a href="/wiki/%EA%B0%90%EB%A6%AC%EA%B5%90" title="감리교">감리교</a>라는 새로운 교파로 발전하였다. 존 웨슬리는 감리교회의 감독이면서도 성공회 사제이기도 했다. 근대 <a class="mw-disambig" href="/wiki/%EC%98%81%EA%B5%AD_%EC%84%B1%EA%B3%B5%ED%9A%8C" title="영국 성공회">영국 성공회</a>의 복음주의자들은 <a class="mw-redirect" href="/wiki/%EB%85%B8%EC%98%88" title="노예">노예</a>제도 반대운동,<a href="/wiki/%EC%A3%BC%EC%9D%BC%ED%95%99%EA%B5%90" title="주일학교">주일학교</a> 운동등으로 영국 사회를 바꾸었다.</p>
<h3><span class="mw-headline" id=".EA.B4.91.EA.B5.90.ED.9A.8C.ED.8C.8C">광교회파</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%84%B1%EA%B3%B5%ED%9A%8C&amp;action=edit&amp;section=15" title="부분 편집: 광교회파">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>위의 두 경향과는 별도로 모든 분파를 "널리" 아우르는 통합적인 국민 교회의 성격을 강조하는 주된 흐름이 있었다. 이런 태도에서 <a href="/wiki/%EA%B4%91%EA%B5%90%ED%9A%8C%ED%8C%8C" title="광교회파">광교회파</a>(Broad Church)라는 이름이 나왔다. 특히 <a href="/wiki/19%EC%84%B8%EA%B8%B0" title="19세기">19세기</a>에 이르러서 이 경향에 있는 신자와 성직자들은 <a href="/wiki/%EC%84%B1%EC%84%9C%EB%B9%84%ED%8F%89%ED%95%99" title="성서비평학">성서비평학</a>과 신학적 <a href="/wiki/%EC%9E%90%EC%9C%A0%EC%A3%BC%EC%9D%98" title="자유주의">자유주의</a>를 적극 받아들였다. 또한 신앙의 본질과 권위를 협소하게 이해하는 태도들을 비판하고, <a href="/wiki/%EB%B3%B5%EC%9D%8C" title="복음">복음</a>의 사회와 문화에 대한 적응성을 강조했다.</p>
<h3><span class="mw-headline" id=".EC.8B.A0.ED.95.99.EC.A0.81_.EB.8B.A4.EC.96.91.EC.84.B1">신학적 다양성</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%84%B1%EA%B3%B5%ED%9A%8C&amp;action=edit&amp;section=16" title="부분 편집: 신학적 다양성">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>성공회 내의 이러한 다양한 전통은 서로 신학적 반성의 기회를 제공하면서 지금도 여러 모습으로 여전히 이어오고 있다. 그러나 이러한 신학적 태도의 다양성은 성공회 전통을 분열시키지 않고 나름대로 성공회의 신학과 정신의 풍요로움에 기여하고 있다고 평가된다.<sup class="정리_필요 noprint">[<a href="/wiki/%EC%9C%84%ED%82%A4%EB%B0%B1%EA%B3%BC:%EC%B6%9C%EC%B2%98_%EB%B0%9D%ED%9E%88%EA%B8%B0" title="위키백과:출처 밝히기">출처 필요</a>]</sup></p>
<h2><span class="mw-headline" id=".EC.8B.A0.EC.95.99.EC.9D.98_.EC.84.B8_.EA.B0.80.EC.A7.80_.EA.B8.B0.EC.A4.80">신앙의 세 가지 기준</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%84%B1%EA%B3%B5%ED%9A%8C&amp;action=edit&amp;section=17" title="부분 편집: 신앙의 세 가지 기준">편집</a><span class="mw-editsection-bracket">]</span></span></h2>

<a class="mw-disambig" href="/wiki/%EC%98%81%EA%B5%AD_%EC%84%B1%EA%B3%B5%ED%9A%8C" title="영국 성공회">영국 성공회</a>에서 번역한 <a class="mw-redirect" href="/wiki/%ED%9D%A0%EC%A0%95%EC%97%AD_%EC%84%B1%EA%B2%BD" title="흠정역 성경">흠정역 성경</a>의 초판 표지</div>

<p>성공회는 <a href="/wiki/%EA%B8%B0%EB%8F%85%EA%B5%90" title="기독교">기독교</a> 신앙을 바르게 지탱하는 권위에 대해서 독특한 관점을 발전시켰다. 즉 <a href="/wiki/%EA%B8%B0%EB%8F%85%EA%B5%90" title="기독교">기독교</a> 신앙을 판단하는 권위를 세 가지 기준(code)을 통해 세웠다. 성서와 이성과 전통이 그것이다. 이러한 세 가지 기준은 성공회의 역사적 발전과 관련이 깊다.</p>
<h3><span class="mw-headline" id=".EC.84.B1.EC.84.9C">성서</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%84%B1%EA%B3%B5%ED%9A%8C&amp;action=edit&amp;section=18" title="부분 편집: 성서">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p><a class="mw-redirect" href="/wiki/%EC%84%B1%EC%84%9C" title="성서">성서</a>는 <a class="mw-redirect" href="/wiki/%EC%A2%85%EA%B5%90%EA%B0%9C%ED%98%81" title="종교개혁">종교개혁</a>의 출발점이었다. 즉 <a href="/wiki/%EC%A0%84%ED%86%B5" title="전통">전통</a>을 통해 굳어진 실체를 진리로 잘못 알고 있는 것에 대한 비판과 진리의 회복으로서 <a class="mw-redirect" href="/wiki/%EC%A2%85%EA%B5%90%EA%B0%9C%ED%98%81" title="종교개혁">종교개혁</a>이 존재하는데, 그 비판과 회복을 위한 최우선의 권위가 바로 성서이다.</p>
<p>사실 이러한 <a class="mw-redirect" href="/wiki/%EC%84%B1%EC%84%9C" title="성서">성서</a>의 <a href="/wiki/%EA%B6%8C%EC%9C%84" title="권위">권위</a>는 <a href="/wiki/%EC%A0%84%ED%86%B5" title="전통">전통</a>과 <a href="/wiki/%EA%B5%90%EB%A6%AC" title="교리">교리</a>에 빗대어 <a href="/wiki/%EC%8B%A0%ED%95%99" title="신학">신학</a>적 궁극성이나 절대적 원리를 내세우는 모든 주장을 상대화하려는 <a href="/wiki/%EB%B9%84%ED%8C%90" title="비판">비판</a>적 원리라고 이해할 수 있다. 따라서 <a href="/wiki/%EA%B5%90%ED%9A%8C" title="교회">교회</a>의 내적 성찰을 위하여 <a class="mw-redirect" href="/wiki/%EC%A2%85%EA%B5%90%EA%B0%9C%ED%98%81" title="종교개혁">종교개혁</a> 전통에서는 성서를 강조한다.</p>
<h3><span class="mw-headline" id=".EC.9D.B4.EC.84.B1">이성</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%84%B1%EA%B3%B5%ED%9A%8C&amp;action=edit&amp;section=19" title="부분 편집: 이성">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>여기서 이러한 환상을 피하기 위한 인간 이성의 개입이 불가피하다. 그러나 이때 “이성”은 해석자의 자의를 말하는 것이 아니다.</p>
<p>이성은 <a href="/wiki/%ED%95%98%EB%8A%90%EB%8B%98" title="하느님">하느님</a>께서 인간에게 주신 선물이며 <a href="/wiki/%EA%B5%90%ED%9A%8C" title="교회">교회</a> 공동체가 함께 공유하고 판단하는 이성(cosmic and corporate Reason)이다. 또한 이성은 전통보다 앞선다. 전통은 이성을 통해서 형성되기 때문이다. 이성은 인간의 경험을 사물의 본질과 조화할 수 있도록 하는 것이다.</p>
<h3><span class="mw-headline" id=".EC.A0.84.ED.86.B5">전통</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%84%B1%EA%B3%B5%ED%9A%8C&amp;action=edit&amp;section=20" title="부분 편집: 전통">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>한편 <a href="/wiki/%EC%A0%84%ED%86%B5" title="전통">전통</a>은 인간의 경험과 실천, 그리고 합의가 최종으로 만들어 낸 교회의 공동체적인 산물이다. 이렇게 만들어진 전통은 교회의 중요한 권위이다. <a class="mw-redirect" href="/wiki/%EC%84%B1%EC%84%9C" title="성서">성서</a>에 대한 이성적 작업으로 축적된 신앙의 결과물이 바로 전통이다. 그러므로 전통은 성서에 대한 이성적 해석에 종속되어야 한다.</p>
<p>전통은 성서에 속한 초자연적인 최고 진리를 가져다 줄 수 없지만, 자의적 판단의 위험성을 피하는 데 중요한 역할을 한다. 그러나 전통은 성서와 이성에 근거한 인간의 “실천, 경험, 동의”에 의해서 필요에 따라서 변경되고 폐지될 수 있다.</p>
<p>성공회는 이렇게 성서와 이성과 전통의 긴장 관계를 통해서 교회사에 나타나는 극단적인 주장과 오류를 피하는 “중용”의 정신을 구현했다고 평가된다.</p>
<h2><span class="mw-headline" id=".EC.A0.84.EB.A1.80">전례</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%84%B1%EA%B3%B5%ED%9A%8C&amp;action=edit&amp;section=21" title="부분 편집: 전례">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>성공회는 일정한 <a href="/wiki/%EC%98%88%EB%B0%B0" title="예배">예배</a> 양식을 따르는 <a href="/wiki/%EC%A0%84%EB%A1%80" title="전례">전례</a>적 예배를 드린다. 성공회의 전례에서 가장 중요한 것은 <b>감사성찬례</b>(Eucharist)(줄여서 "<a class="mw-redirect" href="/wiki/%EC%84%B1%EC%B0%AC%EB%A1%80" title="성찬례">성찬례</a>")이다. 성공회 감사성찬례의 구조는 입당예식, 말씀의 전례, 성찬의 전례, 파송예식으로 되어 있다. 말씀의 전례에서는 <a href="/wiki/%EC%84%B1%EC%84%9C%EC%A0%95%EA%B3%BC" title="성서정과">성서정과</a>(<a href="/wiki/%EA%B5%90%ED%9A%8C%EB%A0%A5" title="교회력">교회력</a>에 맞추어 배열한 성서읽기표)에 나오는 성서말씀을 제1독서(<a class="mw-redirect" href="/wiki/%EA%B5%AC%EC%95%BD%EC%84%B1%EC%84%9C" title="구약성서">구약성서</a>), 제2독서(서신서), <a href="/wiki/%EC%8B%9C%ED%8E%B8" title="시편">시편</a>(성시), <a href="/wiki/%EB%B3%B5%EC%9D%8C%EC%84%9C" title="복음서">복음서</a>로 나누어 읽는다.</p>
<p>성공회의 감사성찬례는 <a href="/wiki/%EC%82%AC%EC%A0%9C" title="사제">사제</a>와 <a href="/wiki/%EC%A3%BC%EA%B5%90" title="주교">주교</a>가 집전하고 설교나 강론을 하며, <a href="/wiki/%EA%B5%90%EA%B5%AC_(%EA%B8%B0%EB%8F%85%EA%B5%90)" title="교구 (기독교)">교구</a>로부터 설교 면허를 받은 일반 신자도 <a href="/wiki/%EC%84%A4%EA%B5%90" title="설교">설교</a>를 할 수 있다. <a href="/wiki/%EC%A3%BC%EA%B5%90" title="주교">주교</a>의 인가를 받은 신자는 영성체시에 잔을 나누어 주는 "보혈 조력"을 할 수 있고, 신자는 전례 시에 성서 독서를 한다.</p>
<h2><span class="mw-headline" id=".ED.95.9C.EA.B5.AD.EC.9D.98_.EC.84.B1.EA.B3.B5.ED.9A.8C">한국의 성공회</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%84%B1%EA%B3%B5%ED%9A%8C&amp;action=edit&amp;section=22" title="부분 편집: 한국의 성공회">편집</a><span class="mw-editsection-bracket">]</span></span></h2>

<p><a href="/wiki/1890%EB%85%84" title="1890년">1890년</a> <a href="/wiki/9%EC%9B%94_29%EC%9D%BC" title="9월 29일">9월 29일</a> 영국 성공회 선교사가 인천항에 도착하여 <a class="mw-redirect" href="/wiki/%EC%84%9C%EC%9A%B8" title="서울">서울</a>과 <a href="/wiki/%EA%B2%BD%EA%B8%B0%EB%8F%84" title="경기도">경기도</a> 그리고 <a href="/wiki/%EC%B6%A9%EC%B2%AD%EB%8F%84" title="충청도">충청도</a> 지방에서 주로 성공회 선교를 시작하였다.</p>
<p><a href="/wiki/%EB%8C%80%ED%95%9C%EC%A0%9C%EA%B5%AD" title="대한제국">대한제국</a> 개화기에 신교육을 보급하기 위하여 각지에 <a class="new" href="/w/index.php?title=%EC%8B%A0%EB%AA%85%ED%95%99%EA%B5%90&amp;action=edit&amp;redlink=1" title="신명학교 (없는 문서)">신명학교</a>를 설립하고, 인천, 여주, 진천 등지에 병원을 설립하였으며, 수원과 안중에는 보육원을 개설하였다.</p>
<p>비슷한 시기에 전래된 <a href="/wiki/%EC%9E%A5%EB%A1%9C%EA%B5%90" title="장로교">장로교</a>, <a href="/wiki/%EA%B0%90%EB%A6%AC%EA%B5%90" title="감리교">감리교</a> 등과는 달리 적극적인 전도보다는 <a href="/wiki/%EC%82%AC%ED%9A%8C%EC%84%A0%EA%B5%90" title="사회선교">사회선교</a>와 <a href="/wiki/%EA%B8%B0%EB%8F%85%EA%B5%90" title="기독교">기독교</a> 신앙을 실천하는 것에 무게를 두었다. 대한 성공회는 1980년대 후반 이후에 도시 빈민 선교 기관인 "나눔의 집"을 중심으로 사회 선교 운동을 전개했고, 지금도 <a class="mw-redirect" href="/wiki/%EB%82%98%EB%88%94%EC%9D%98_%EC%A7%91_%ED%98%91%EC%9D%98%ED%9A%8C" title="나눔의 집 협의회">나눔의 집 협의회</a> 등을 설립해 다양한 사회선교를 활발히 펼치고 있다.</p>
<p><a href="/wiki/%EB%8C%80%ED%95%9C%EC%84%B1%EA%B3%B5%ED%9A%8C" title="대한성공회">대한성공회</a>는 선교 초기부터 한국 문화를 존중하고 그 토양 깊이 뿌리를 내린 교회를 추구하여 기독교 토착화에 힘썼다. 그 사례로 한국 건축 양식(한옥)으로 지은 성공회 성당들이 현재까지 강화 (<a class="mw-redirect" href="/wiki/%EC%84%B1%EA%B3%B5%ED%9A%8C_%EA%B0%95%ED%99%94%EC%84%B1%EB%8B%B9" title="성공회 강화성당">강화성당</a>), 진천 (<a class="mw-redirect" href="/wiki/%EC%84%B1%EA%B3%B5%ED%9A%8C_%EC%A7%84%EC%B2%9C%EC%84%B1%EB%8B%B9" title="성공회 진천성당">진천성당</a>), 청주 (<a href="/wiki/%EC%84%B1%EA%B3%B5%ED%9A%8C_%EB%82%B4%EB%8F%99%EC%84%B1%EB%8B%B9" title="성공회 내동성당">내동성당</a>) 등에 남아 있다.</p>
<p><a href="/wiki/1965%EB%85%84" title="1965년">1965년</a>에는 대한 성공회 역사 상 처음으로 한국인 주교 <a href="/wiki/%EC%9D%B4%EC%B2%9C%ED%99%98" title="이천환">이천환</a> 주교가 성품되어 서울교구장이 되었다.</p>
<p><a href="/wiki/1993%EB%85%84" title="1993년">1993년</a>에 대한 성공회는 캔터베리 대교구 관할에서 벗어나, 세계 성공회 독립 관구가 되었다. 초대 관구장은 <a class="mw-disambig" href="/wiki/%EA%B9%80%EC%84%B1%EC%88%98" title="김성수">김성수</a> 주교였다.</p>
<p><a href="/wiki/2001%EB%85%84" title="2001년">2001년</a>에 대한 성공회 부산교구의 <a href="/wiki/%EB%AF%BC%EB%B3%91%EC%98%A5" title="민병옥">민병옥</a>이 사제로 서품되어 첫 여성 사제를 배출했다.</p>
<p>현재 <a href="/wiki/%EB%8C%80%ED%95%9C%EC%84%B1%EA%B3%B5%ED%9A%8C" title="대한성공회">대한성공회</a>는 <a href="/wiki/%EB%8C%80%ED%95%9C%EC%84%B1%EA%B3%B5%ED%9A%8C_%EC%84%9C%EC%9A%B8%EA%B5%90%EA%B5%AC" title="대한성공회 서울교구">서울</a>, <a href="/wiki/%EB%8C%80%ED%95%9C%EC%84%B1%EA%B3%B5%ED%9A%8C_%EB%B6%80%EC%82%B0%EA%B5%90%EA%B5%AC" title="대한성공회 부산교구">부산</a>, <a href="/wiki/%EB%8C%80%ED%95%9C%EC%84%B1%EA%B3%B5%ED%9A%8C_%EB%8C%80%EC%A0%84%EA%B5%90%EA%B5%AC" title="대한성공회 대전교구">대전</a>에 교구를 두고 있고, 교육기관으로는 <a href="/wiki/%EC%84%B1%EA%B3%B5%ED%9A%8C%EB%8C%80%ED%95%99%EA%B5%90" title="성공회대학교">성공회대학교</a> 등이 있다. 수도 단체로는 <a class="new" href="/w/index.php?title=%EC%84%B1%EA%B0%80%EC%88%98%EB%85%80%ED%9A%8C&amp;action=edit&amp;redlink=1" title="성가수녀회 (없는 문서)">성가수녀회</a>, <a class="new" href="/w/index.php?title=%EC%84%B1_%EB%B6%84%EB%8F%84_%EC%88%98%EB%85%80%ED%9A%8C&amp;action=edit&amp;redlink=1" title="성 분도 수녀회 (없는 문서)">성 분도 수녀회</a>, <a class="new" href="/w/index.php?title=%EC%84%B1_%ED%94%84%EB%9E%80%EC%8B%9C%EC%8A%A4_%EC%88%98%EB%8F%84%ED%9A%8C&amp;action=edit&amp;redlink=1" title="성 프란시스 수도회 (없는 문서)">성 프란시스 수도회</a>와 미국 성공회 선교사 출신인 <a href="/wiki/%EB%8C%80%EC%B2%9C%EB%8D%95" title="대천덕">대천덕</a> 신부가 세운 <a href="/wiki/%EC%98%88%EC%88%98%EC%9B%90" title="예수원">예수원</a>이 있다.</p>
<p><a class="mw-redirect" href="/wiki/%EB%8C%80%ED%95%9C_%EC%84%B1%EA%B3%B5%ED%9A%8C" title="대한 성공회">대한 성공회</a>는 <a class="mw-redirect" href="/wiki/%EC%84%B8%EA%B3%84%EA%B5%90%ED%9A%8C%ED%98%91%EC%9D%98%ED%9A%8C" title="세계교회협의회">세계교회협의회</a>의 정식 회원이며, <a href="/wiki/%ED%95%9C%EA%B5%AD%EA%B8%B0%EB%8F%85%EA%B5%90%EA%B5%90%ED%9A%8C%ED%98%91%EC%9D%98%ED%9A%8C" title="한국기독교교회협의회">한국기독교교회협의회</a>, <a class="mw-redirect" href="/wiki/%EA%B8%B0%EB%8F%85%EA%B5%90%EB%B0%A9%EC%86%A1" title="기독교방송">기독교방송</a> 등의 기관들과도 협력하고 있다.</p>
