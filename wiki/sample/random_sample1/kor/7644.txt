					이동:					<a href="#mw-head">둘러보기</a>, 					<a href="#p-search">검색</a>

<div class="mw-content-ltr" dir="ltr" id="mw-content-text" lang="ko"><script>function mfTempOpenSection(id){var block=document.getElementById("mf-section-"+id);block.className+=" open-block";block.previousSibling.className+=" open-block";}</script><p><b>성민</b>(<a href="/wiki/1982%EB%85%84" title="1982년">1982년</a> <a href="/wiki/10%EC%9B%94_15%EC%9D%BC" title="10월 15일">10월 15일</a> ~ )은 <a href="/wiki/%EB%8C%80%ED%95%9C%EB%AF%BC%EA%B5%AD" title="대한민국">대한민국</a>의 <a href="/wiki/%EC%88%98%EC%98%81" title="수영">수영</a> 선수이다. 종교는 <a href="/wiki/%EA%B0%9C%EC%8B%A0%EA%B5%90" title="개신교">개신교</a>이다.<sup class="reference" id="cite_ref-1"><a href="#cite_note-1">[1]</a></sup></p>
<p>현재 <a href="/wiki/%ED%95%9C%EA%B5%AD%EC%B2%B4%EC%9C%A1%EB%8C%80%ED%95%99%EA%B5%90" title="한국체육대학교">한국체육대학교</a>를 졸업하고 <a class="new" href="/w/index.php?title=%EA%B2%BD%EC%83%81%EB%B6%81%EB%8F%84%EC%B2%B4%EC%9C%A1%ED%9A%8C&amp;action=edit&amp;redlink=1" title="경상북도체육회 (없는 문서)">경상북도체육회</a>소속이다.</p>
<p><a href="/wiki/2000%EB%85%84" title="2000년">2000년</a> <a class="mw-redirect" href="/wiki/%EC%8B%9C%EB%93%9C%EB%8B%88_%EC%98%AC%EB%A6%BC%ED%94%BD" title="시드니 올림픽">시드니 올림픽</a>과 <a href="/wiki/2004%EB%85%84" title="2004년">2004년</a> <a href="/wiki/2004%EB%85%84_%ED%95%98%EA%B3%84_%EC%98%AC%EB%A6%BC%ED%94%BD" title="2004년 하계 올림픽">아테네 올림픽</a>에 국가 대표로 출전했다. <a href="/wiki/2000%EB%85%84" title="2000년">2000년</a> 아시아선수권 대회 100미터 배영에서 3위, <a href="/wiki/2002%EB%85%84" title="2002년">2002년</a> 월드컵쇼트코스 수영 배영 200미터에서 1위를 했다.</p>
