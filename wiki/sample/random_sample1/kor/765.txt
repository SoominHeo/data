<p><b>AS 리보르노 칼초</b>(Associazione Sportiva Livorno Calcio SrL)는 <a href="/wiki/%EB%A6%AC%EB%B3%B4%EB%A5%B4%EB%85%B8" title="리보르노">리보르노</a>에 위치한 <a class="new" href="/w/index.php?title=%EC%8A%A4%ED%83%80%EB%94%94%EC%98%A4_%EC%95%84%EB%A5%B4%EB%A7%8C%EB%8F%84_%ED%94%BC%ED%82%A4&amp;action=edit&amp;redlink=1" title="스타디오 아르만도 피키 (없는 문서)">스타디오 아르만도 피키</a>를 근거로 하는 <a href="/wiki/%EC%9D%B4%ED%83%88%EB%A6%AC%EC%95%84" title="이탈리아">이탈리아</a>의 <a href="/wiki/%EC%B6%95%EA%B5%AC" title="축구">축구</a> 클럽이다. 현재 감독은 <a class="new" href="/w/index.php?title=%EB%8B%A4%EB%B9%84%EB%8D%B0_%EB%8B%88%EC%BD%9C%EB%9D%BC&amp;action=edit&amp;redlink=1" title="다비데 니콜라 (없는 문서)">다비데 니콜라</a>이다. 이 축구단은 <a href="/wiki/1915%EB%85%84" title="1915년">1915년</a> 창단되었고 최근에는 <a href="/wiki/%EC%84%B8%EB%A6%AC%EC%97%90_A" title="세리에 A">세리에 A</a>에 참가중이다. 팀의 색깔은 암적색과 고동색(팀의 별명인 Amaranto도 여기서 유래함)이다. 세리에 A에서의 최고 성적은 선두였던 <a href="/wiki/%ED%86%A0%EB%A6%AC%EB%85%B8_FC" title="토리노 FC">토리노</a>외 경쟁하여 2위를 기록했던 1942-43 시즌이다.</p>

<h2><span class="mw-headline" id=".EC.97.AD.EC.82.AC">역사</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=AS_%EB%A6%AC%EB%B3%B4%EB%A5%B4%EB%85%B8_%EC%B9%BC%EC%B4%88&amp;action=edit&amp;section=1" title="부분 편집: 역사">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p><b>US 리보르노</b>로 1915년에 창단을 했던 클럽은 결승전에서 <a href="/wiki/FC_%EC%9D%B8%ED%85%8C%EB%A5%B4%EB%82%98%EC%B9%98%EC%98%A4%EB%82%A0%EB%A0%88_%EB%B0%80%EB%9D%BC%EB%85%B8" title="FC 인테르나치오날레 밀라노">인테르나치오날레</a>에게 결승전에 패하며 <a class="mw-redirect" href="/wiki/%EC%9D%B4%ED%83%88%EB%A6%AC%EC%95%88_%ED%92%8B%EB%B3%BC_%EC%B1%94%ED%94%BC%EC%96%B8%EC%89%BD" title="이탈리안 풋볼 챔피언쉽">이탈리안 풋볼 챔피언쉽</a> 1919-20 시즌에 2위를 차지했다. 1년 뒤, 그들은 준결승전에서 최대의 라이벌 <a class="mw-redirect" href="/wiki/%ED%94%BC%EC%82%AC_%EC%B9%BC%EC%B4%88" title="피사 칼초">피사</a>에게 패했다. 1933년에 클럽은 나중에 파시스트 독재자 <a href="/wiki/%EB%B2%A0%EB%8B%88%ED%86%A0_%EB%AC%B4%EC%86%94%EB%A6%AC%EB%8B%88" title="베니토 무솔리니">베니토 무솔리니</a>의 딸, <a class="new" href="/w/index.php?title=%EC%97%90%EB%8B%A4_%EC%B0%A8%EB%85%B8_%EB%AC%B4%EC%86%94%EB%A6%AC%EB%8B%88&amp;action=edit&amp;redlink=1" title="에다 차노 무솔리니 (없는 문서)">에다 차노 무솔리니</a>의 이름이 붙여진 현재의 경기장으로 옮겼다. 리보르노는 1942-1943 시즌에 준우승을 차지하며 성공적인 시즌을 보냈다. 리보르노는 1949년에 세리에 A를 떠나 연이어 7 시즌을 보냈다. 그들은 곧 <a class="mw-redirect" href="/wiki/%EC%84%B8%EB%A6%AC%EC%97%90_C" title="세리에 C">세리에 C</a>로 강등을 당했고, 1955년에 승격을 하여 한 시즌을 보냈고 다시 1964년부터 1972년을 보냈다. 클럽은 <a class="new" href="/w/index.php?title=%EC%97%90%EC%B2%BC%EB%A0%8C%EC%B0%A8&amp;action=edit&amp;redlink=1" title="에첼렌차 (없는 문서)">에첼렌차</a>에서 시작을 하기로 했으나 1991년에 취소됐고 연속하여 두 번 승격을 하여 <a class="new" href="/w/index.php?title=%EC%84%B8%EB%A6%AC%EC%97%90_C2&amp;action=edit&amp;redlink=1" title="세리에 C2 (없는 문서)">세리에 C2</a>로 돌아왔다. 1997년에 <a class="mw-redirect" href="/wiki/%EC%84%B8%EB%A6%AC%EC%97%90_C1" title="세리에 C1">세리에 C1</a>으로 승격했고 알도 스피넬리가 구단주가 부임했다. 새로운 자본하에, 리보르노는 2001년에 세리에 B로 승격했다.</p>
<p>리보르노는 6개의 구단이 승격하던때인 <a class="new" href="/w/index.php?title=%EC%84%B8%EB%A6%AC%EC%97%90_B_2003-04&amp;action=edit&amp;redlink=1" title="세리에 B 2003-04 (없는 문서)">세리에 B 2003-04</a> 시즌에 3위로 마감하여 세리에 A로 승격했다. 리보르노는 마지막으로 1부리그에 머물었던지 55년 만에 승격을 했고 이 결과적으로 가장 바로 세리에 B로 돌아갈 클럽으로 예측되었다.</p>
