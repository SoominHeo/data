<p><b>세르게이 게오르기예비치 고르시코프</b>(<span lang="ru" xml:lang="ru">Сергей Георгиевич Горшков</span>,<a href="/wiki/1910%EB%85%84" title="1910년">1910년</a> <a href="/wiki/2%EC%9B%94_26%EC%9D%BC" title="2월 26일">2월 26일</a> ~ <a href="/wiki/1988%EB%85%84" title="1988년">1988년</a> <a href="/wiki/5%EC%9B%94_12%EC%9D%BC" title="5월 12일">5월 12일</a>)는 <a class="mw-redirect" href="/wiki/%EC%86%8C%EB%B9%84%EC%97%90%ED%8A%B8_%EC%97%B0%EB%B0%A9" title="소비에트 연방">소비에트 연방</a>의 군인으로 1956년부터 1985년동안 소련 해군 총사령관 지위에 있었으며, <a href="/wiki/%EB%83%89%EC%A0%84" title="냉전">냉전</a> 당시 <a class="mw-redirect" href="/wiki/%EC%86%8C%EB%B9%84%EC%97%90%ED%8A%B8_%EC%97%B0%EB%B0%A9_%ED%95%B4%EA%B5%B0" title="소비에트 연방 해군">소비에트 연방 해군</a>과 현재 <a href="/wiki/%EB%9F%AC%EC%8B%9C%EC%95%84_%ED%95%B4%EA%B5%B0" title="러시아 해군">러시아 해군</a>에 막대한 기여를 했다. <a href="/wiki/%EB%8B%88%EC%BD%9C%EB%9D%BC%EC%9D%B4_%EC%BF%A0%EC%A6%88%EB%84%A4%EC%B4%88%ED%94%84" title="니콜라이 쿠즈네초프">니콜라이 쿠즈네초프</a>를 소련 해군의 아버지로 부르는 것에 빗대어 고르시코프는 소련 해군의 어머니로 불리고 있다.</p>

<h2><span class="mw-headline" id=".EC.84.B1.EC.9E.A5">성장</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%84%B8%EB%A5%B4%EA%B2%8C%EC%9D%B4_%EA%B3%A0%EB%A5%B4%EC%8B%9C%EC%BD%94%ED%94%84&amp;action=edit&amp;section=1" title="부분 편집: 성장">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>세르게이 고르시코프는 <a href="/wiki/1910%EB%85%84" title="1910년">1910년</a> <a href="/wiki/%EB%9F%AC%EC%8B%9C%EC%95%84_%EC%A0%9C%EA%B5%AD" title="러시아 제국">러시아 제국</a> <a class="new" href="/w/index.php?title=%EC%B9%B4%EB%AF%B8%EC%95%BC%EB%84%A4%EC%B9%98%ED%8F%AC%EB%8F%8C%EC%8B%9C%ED%82%A4&amp;action=edit&amp;redlink=1" title="카미야네치포돌시키 (없는 문서)">카미야네치포돌시키</a>에서 태어나 <a href="/wiki/%EB%AA%A8%EC%8A%A4%ED%81%AC%EB%B0%94_%EC%A3%BC" title="모스크바 주">모스크바 주</a>의 콜롬나에서 자랐다. <a href="/wiki/1926%EB%85%84" title="1926년">1926년</a> <a class="mw-redirect" href="/wiki/%EB%A0%88%EB%8B%8C%EA%B7%B8%EB%9D%BC%EB%93%9C_%EB%8C%80%ED%95%99%EA%B5%90" title="레닌그라드 대학교">레닌그라드 대학교</a> 물리 수학과에 입학했으나, <a href="/wiki/1927%EB%85%84" title="1927년">1927년</a> <a class="mw-redirect" href="/wiki/%ED%94%84%EB%A3%AC%EC%A0%9C_%EA%B5%B0%EC%82%AC_%EC%95%84%EC%B9%B4%EB%8D%B0%EB%AF%B8" title="프룬제 군사 아카데미">프룬제 군사 아카데미</a>로 편입하였고 소련 해군에 입대했으며, <a href="/wiki/1931%EB%85%84" title="1931년">1931년</a> 프룬제 군사 아카데미를 졸업한다. 졸업 후 흑해 함대에 배속되어 흑해함대 구축함 프룬제의 항해사로 근무하다가 1932년 다시 태평양 함대 기뢰전 함선 톰스크로 재배치되었고 이후 대 여단장까지 역임한 후 1937년과 1941년간 해군 지휘관 코스를 졸업해 31살에 최연소 해군 제독이 된다.</p>
<h2><span class="mw-headline" id=".EC.A0.9C2.EC.B0.A8_.EC.84.B8.EA.B3.84.EB.8C.80.EC.A0.84.EA.B3.BC_.EC.A0.84.ED.9B.84"><a class="mw-redirect" href="/wiki/%EC%A0%9C2%EC%B0%A8_%EC%84%B8%EA%B3%84%EB%8C%80%EC%A0%84" title="제2차 세계대전">제2차 세계대전</a>과 전후</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%84%B8%EB%A5%B4%EA%B2%8C%EC%9D%B4_%EA%B3%A0%EB%A5%B4%EC%8B%9C%EC%BD%94%ED%94%84&amp;action=edit&amp;section=2" title="부분 편집: 제2차 세계대전과 전후">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>독소전쟁이 발발한 해 10월에 아조프 소함대 사령관에 임명되고, <a class="new" href="/w/index.php?title=%EC%BC%80%EB%A5%B4%EC%B9%98_%EB%B0%98%EB%8F%84&amp;action=edit&amp;redlink=1" title="케르치 반도 (없는 문서)">케르치 반도</a> 상륙에서 두각을 나타내었다. 그리고 1년 뒤 1942년 9월에 제47군을 임시 지휘하여 <a href="/wiki/%EB%85%B8%EB%B3%B4%EB%A1%9C%EC%8B%9C%EC%8A%A4%ED%81%AC" title="노보로시스크">노보로시스크</a>를 방어, 1943년 아조프 소함대에 복귀한 뒤 그해 말에 다뉴브 소함대 사령관에 임명되었다가 44년 10월 흑해함대 전대 사령관이 된다. 2차 세계대전이 끝난 후 1948년~1951년 함대 참모장, 1951년~1955년 흑해함대 사령관, 그리고 1955년~1956년 해군 부사령관을 역임하였고, <a href="/wiki/1956%EB%85%84" title="1956년">1956년</a> <a href="/wiki/%EB%8B%88%EC%BD%9C%EB%9D%BC%EC%9D%B4_%EC%BF%A0%EC%A6%88%EB%84%A4%EC%B4%88%ED%94%84" title="니콜라이 쿠즈네초프">쿠즈네초프</a>의 뒤를 이어서 소련 해군 총사령관과 국방 차관에 임명되었다.</p>
<h2><span class="mw-headline" id=".EC.9E.AC.EC.9E.84_.EC.8B.9C.EC.A0.88">재임 시절</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%84%B8%EB%A5%B4%EA%B2%8C%EC%9D%B4_%EA%B3%A0%EB%A5%B4%EC%8B%9C%EC%BD%94%ED%94%84&amp;action=edit&amp;section=3" title="부분 편집: 재임 시절">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>소련 해군 총사령관으로 있던 동안 고르시코프는 소련 해군을 세계 2위의 해군으로 만들어 놓았다. <a class="mw-redirect" href="/wiki/%EB%B8%8C%EB%A0%88%EC%A6%88%EB%84%A4%ED%94%84" title="브레즈네프">브레즈네프</a>, <a class="mw-redirect" href="/wiki/%EC%95%88%EB%93%9C%EB%A1%9C%ED%8F%AC%ED%94%84" title="안드로포프">안드로포프</a>, <a class="mw-redirect" href="/wiki/%EC%B2%B4%EB%A5%B4%EB%84%A8%EC%BD%94" title="체르넨코">체르넨코</a> 시절을 통해 균형 함대 발전론을 주창하면서 1970년 크반트 연구소를 방문해 소비에트 해군용 카라트 컴퓨터의 개발을 의뢰하는등 소련 해군의 현대화와 해군력 강화에 노력해 소련 함대의 전술,전력 강화,건함및 설계 사상등 모든 해군 분야에 일대 혁신을 불러왔다. 그는 소련 해군의 교리에도 막대한 영향을 미쳤는데 <a href="/wiki/%EB%83%89%EC%A0%84" title="냉전">냉전</a> 당시 <a class="mw-redirect" href="/wiki/%EC%86%8C%EB%B9%84%EC%97%90%ED%8A%B8_%EC%97%B0%EB%B0%A9_%ED%95%B4%EA%B5%B0" title="소비에트 연방 해군">소련 해군</a>과 현 <a href="/wiki/%EB%9F%AC%EC%8B%9C%EC%95%84_%ED%95%B4%EA%B5%B0" title="러시아 해군">러시아 해군</a>은 사실상 그가 만들었다고 봐도 과언이 아닐 정도다. 그가 주장한 소련해군의 전략은 필요한 때에 필요한 해역을 장악한다는 전략이다. 이런 전략을 주장하게 된 데에는 대함미사일과 <a href="/wiki/%EC%9E%A0%EC%88%98%ED%95%A8" title="잠수함">잠수함</a>의 발달이 있다. 고르시코프는 미사일과 잠수함이 발달한 이상 해상 어디에서라도 기습의 위험을 피할 수 없게 되었고 생각하였고 실제로 이 전략은 <a href="/wiki/%EB%83%89%EC%A0%84" title="냉전">냉전</a> 시기 내내 소련 해군의 정식 교리였다.</p>
