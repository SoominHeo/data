<p><b>세르게이 미하일로비치 브린</b>(<span style="font-size: smaller;"><a href="/wiki/%EB%9F%AC%EC%8B%9C%EC%95%84%EC%96%B4" title="러시아어">러시아어</a>:</span> <span lang="ru" xml:lang="ru">Серге́й Миха́йлович Брин</span>, <span style="font-size: smaller;"><a href="/wiki/%EC%98%81%EC%96%B4" title="영어">영어</a>:</span> <span lang="en" xml:lang="en">Sergey Brin</span> <small>서게이 브린<sup>[<a href="/wiki/%EC%9C%84%ED%82%A4%EB%B0%B1%EA%B3%BC:%EC%98%81%EC%96%B4%EC%9D%98_%ED%95%9C%EA%B8%80_%ED%91%9C%EA%B8%B0" title="위키백과:영어의 한글 표기">*</a>]</sup></small>, <a href="/wiki/1973%EB%85%84" title="1973년">1973년</a> <a href="/wiki/8%EC%9B%94_21%EC%9D%BC" title="8월 21일">8월 21일</a> ~)은 <a href="/wiki/%EB%9F%AC%EC%8B%9C%EC%95%84" title="러시아">러시아</a> 출신의 <a href="/wiki/%EB%AF%B8%EA%B5%AD" title="미국">미국</a> 시민권자 기업인으로서 <a href="/wiki/%EB%9E%98%EB%A6%AC_%ED%8E%98%EC%9D%B4%EC%A7%80" title="래리 페이지">래리 페이지</a>와 함께 <a href="/wiki/%EA%B5%AC%EA%B8%80" title="구글">구글</a>을 창립했다.</p>
<p>브린은 <a href="/wiki/%EB%9F%AC%EC%8B%9C%EC%95%84" title="러시아">러시아</a>의 <a href="/wiki/%EB%AA%A8%EC%8A%A4%ED%81%AC%EB%B0%94" title="모스크바">모스크바</a>에서 <a class="mw-redirect" href="/wiki/%EB%AA%A8%EC%8A%A4%ED%81%AC%EB%B0%94_%EB%8C%80%ED%95%99" title="모스크바 대학">모스크바 대학</a>을 졸업한 유대인 부부의 아들로 태어났다. 6살 때 미국에 온 그는 후에 <a href="/wiki/%EB%A9%94%EB%A6%B4%EB%9E%9C%EB%93%9C_%EB%8C%80%ED%95%99%EA%B5%90" title="메릴랜드 대학교">메릴랜드 대학</a>에서 수학과 교수로 재직하게 된 아버지 덕분에 수학에 많은 흥미를 보였으며, 이를 계기로 그는 고등학교 졸업 후 메릴랜드 대학에 입학하여 수학과 컴퓨터 과학 전공으로 학부를 마쳤다. 이후 그는 <a href="/wiki/%EC%8A%A4%ED%83%A0%ED%8D%BC%EB%93%9C_%EB%8C%80%ED%95%99%EA%B5%90" title="스탠퍼드 대학교">스탠포드 대학교</a> 대학원으로 진학하여 컴퓨터 과학을 전공하였다. 컴퓨터 과학 석사학위를 취득한 그는 박사과정을 밟았으나, 도중에 래리 페이지를 만나 박사과정을 그만두고 <a href="/wiki/1998%EB%85%84" title="1998년">1998년</a> 친구의 차고에서 구글을 창립하였다.</p>
<p>브린은 현재 구글의 기술 부문 사장(President of Technology)이며 <a href="/wiki/2008%EB%85%84" title="2008년">2008년</a> <a href="/wiki/3%EC%9B%94_5%EC%9D%BC" title="3월 5일">3월 5일</a> 현재 약 187억 달러의 순자산(net worth)을 가지고 있으며 세계 32번째 부자이다<sup class="reference" id="cite_ref-1"><a href="#cite_note-1">[1]</a></sup>.</p>
