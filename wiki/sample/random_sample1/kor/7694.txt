<td class="mbox-text"><span class="mbox-text-span">이 문서의 내용은 <b><a href="/wiki/%EC%9C%84%ED%82%A4%EB%B0%B1%EA%B3%BC:%EC%B6%9C%EC%B2%98_%EB%B0%9D%ED%9E%88%EA%B8%B0" title="위키백과:출처 밝히기">출처</a>가 분명하지 않습니다.</b><span class="hide-when-compact"><br/>
지금 바로 이 <span class="plainlinks"><a class="external text" href="//ko.wikipedia.org/w/index.php?title=%EC%84%B8%EB%A5%B4%EA%B2%8C%EC%9D%B4_%ED%82%A4%EB%A1%9C%ED%94%84&amp;action=edit">문서를 편집</a></span>하여, 참고하신 문헌이나 <a href="/wiki/%EC%9C%84%ED%82%A4%EB%B0%B1%EA%B3%BC:%EC%8B%A0%EB%A2%B0%ED%95%A0_%EC%88%98_%EC%9E%88%EB%8A%94_%EC%B6%9C%EC%B2%98" title="위키백과:신뢰할 수 있는 출처">신뢰할 수 있는 출처</a>를 각주 등으로 표기해 주세요. <a href="/wiki/%EC%9C%84%ED%82%A4%EB%B0%B1%EA%B3%BC:%ED%99%95%EC%9D%B8_%EA%B0%80%EB%8A%A5" title="위키백과:확인 가능">검증</a>되지 않은 내용은 삭제될 수도 있습니다. 내용에 대한 의견은 <a href="/wiki/%ED%86%A0%EB%A1%A0:%EC%84%B8%EB%A5%B4%EA%B2%8C%EC%9D%B4_%ED%82%A4%EB%A1%9C%ED%94%84" title="토론:세르게이 키로프">토론 문서</a>에서 나누어 주세요.</span> <small>(2013년 8월)</small></span></td>

<p><br/>
<b>세르게이 키로프</b>(<span style="font-size: smaller;"><a href="/wiki/%EB%9F%AC%EC%8B%9C%EC%95%84%EC%96%B4" title="러시아어">러시아어</a>:</span> <span lang="ru" xml:lang="ru">Серге́й Миро́нович Ки́ров</span>, <a href="/wiki/1886%EB%85%84" title="1886년">1886년</a> <a href="/wiki/3%EC%9B%94_27%EC%9D%BC" title="3월 27일">3월 27일</a>(<a class="mw-redirect" href="/wiki/%EA%B5%AC%EB%A0%A5" title="구력">구력</a> 3월 15일) ~ <a href="/wiki/1934%EB%85%84" title="1934년">1934년</a> <a href="/wiki/12%EC%9B%94_1%EC%9D%BC" title="12월 1일">12월 1일</a>)는 <a href="/wiki/%EC%86%8C%EB%A0%A8" title="소련">소련</a>의 정치가이자 <a href="/wiki/%EA%B3%B5%EC%82%B0%EB%8B%B9" title="공산당">공산당</a> 지도자로 <a href="/wiki/1934%EB%85%84" title="1934년">1934년</a>에 암살되었다. 그의 죽음은 <a href="/wiki/%EC%9D%B4%EC%98%A4%EC%8B%9C%ED%94%84_%EC%8A%A4%ED%83%88%EB%A6%B0" title="이오시프 스탈린">스탈린</a>의 대숙청 시대를 여는 계기가 되었다.</p>
<p>키로프는 <a href="/wiki/1886%EB%85%84" title="1886년">1886년</a> <a href="/wiki/%EB%9F%AC%EC%8B%9C%EC%95%84_%EC%A0%9C%EA%B5%AD" title="러시아 제국">러시아 제국</a> <a class="mw-redirect" href="/wiki/%EB%B1%8C%ED%8A%B8%EC%B9%B4" title="뱌트카">뱌트카</a> <a class="new" href="/w/index.php?title=%EC%9A%B0%EB%A5%B4%EC%A6%88%EC%9B%80&amp;action=edit&amp;redlink=1" title="우르즈움 (없는 문서)">우르즈움</a>에서 태어났다. <a href="/wiki/1917%EB%85%84" title="1917년">1917년</a> <a href="/wiki/10%EC%9B%94_%ED%98%81%EB%AA%85" title="10월 혁명">10월 혁명</a> 이래에 혁명 활동으로 수차례 투옥되었으며, <a href="/wiki/1921%EB%85%84" title="1921년">1921년</a>에 <a href="/wiki/%EC%95%84%EC%A0%9C%EB%A5%B4%EB%B0%94%EC%9D%B4%EC%9E%94" title="아제르바이잔">아제르바이잔</a> 공산당 제1서기로 임명되었다.</p>
<p><a href="/wiki/1922%EB%85%84" title="1922년">1922년</a> 3월에 <a class="new" href="/w/index.php?title=%EC%9E%90%EC%B9%B4%ED%94%84%EC%B9%B4%EC%A7%80%EC%97%90&amp;action=edit&amp;redlink=1" title="자카프카지에 (없는 문서)">자카프카지에</a> 지방에 소비에트 연방 사회주의 공화국을 건설하는 데에 성공했으며, <a href="/wiki/1926%EB%85%84" title="1926년">1926년</a>에 <a href="/wiki/%EC%9D%B4%EC%98%A4%EC%8B%9C%ED%94%84_%EC%8A%A4%ED%83%88%EB%A6%B0" title="이오시프 스탈린">스탈린</a>에 의해 <a class="mw-redirect" href="/wiki/%EB%A0%88%EB%8B%8C%EA%B7%B8%EB%9D%BC%EB%93%9C" title="레닌그라드">레닌그라드</a> 지부 서기장으로 임명되었다. 1930년대 초에는 공산당 레닌그라드 지부에서 점차 독립성을 확대하여 스탈린과 맞먹을 정도의 권력을 가지기에 이르렀다.</p>
<p>그러나 <a href="/wiki/1934%EB%85%84" title="1934년">1934년</a> <a href="/wiki/12%EC%9B%94_1%EC%9D%BC" title="12월 1일">12월 1일</a>에 레닌그라드 공산당 본부에서 남몰래 스탈린의 정적 <a class="mw-redirect" href="/wiki/%ED%8A%B8%EB%A1%9C%EC%B8%A0%ED%82%A4" title="트로츠키">트로츠키</a>를 따르던 학생 <a href="/wiki/%EB%A0%88%EC%98%A4%EB%8B%88%ED%8A%B8_%EB%8B%88%EC%BD%9C%EB%9D%BC%EC%98%88%ED%94%84" title="레오니트 니콜라예프">레오니트 니콜라예프</a>에게 암살당했다. 이로써 <a href="/wiki/1936%EB%85%84" title="1936년">1936년</a> 8월 키로프 암살사건에 관련된 니콜라예프와 13명의 공범들이 총살되면서 스탈린의 대숙청 시대가 열리게 되었다.</p>
<p>하지만 후에 여러 학자들과 소련의 비밀문건으로 인해 스탈린이 대숙청 구실을 만들기 위한 음모가 아닌가 하는 의혹을 받고 있다.</p>
<p><a href="/wiki/%EB%9F%AC%EC%8B%9C%EC%95%84" title="러시아">러시아</a>에는 현재 그의 이름을 딴 <a class="mw-redirect" href="/wiki/%ED%82%A4%EB%A1%9C%ED%94%84%EA%B7%B8%EB%9D%BC%EB%93%9C" title="키로프그라드">키로프그라드</a>, <a href="/wiki/%ED%82%A4%EB%A1%9C%ED%94%84_%EC%A3%BC" title="키로프 주">키로프 주</a>, <a class="mw-redirect" href="/wiki/%ED%82%A4%EB%A1%9C%ED%94%84%EC%8A%A4%ED%81%AC_(%EB%AC%B4%EB%A5%B4%EB%A7%8C%EC%8A%A4%ED%81%AC_%EC%A3%BC)" title="키로프스크 (무르만스크 주)">키로프스크</a> 등의 도시가 있다.</p>
