<p><b>세르히 나자로비치 부브카</b>(<span style="font-size: smaller;"><a href="/wiki/%EC%9A%B0%ED%81%AC%EB%9D%BC%EC%9D%B4%EB%82%98%EC%96%B4" title="우크라이나어">우크라이나어</a>:</span> <span lang="uk" xml:lang="uk">Сергій Назарович Бубка</span>, <span style="font-size: smaller;"><a href="/wiki/%EB%9F%AC%EC%8B%9C%EC%95%84%EC%96%B4" title="러시아어">러시아어</a>:</span> <span lang="ru" xml:lang="ru">Сергей Назарович Бубка</span> <small>세르게이 나자로비치 붑카</small>; <a href="/wiki/1963%EB%85%84" title="1963년">1963년</a> <a href="/wiki/12%EC%9B%94_4%EC%9D%BC" title="12월 4일">12월 4일</a> -)는 <a href="/wiki/%EC%9A%B0%ED%81%AC%EB%9D%BC%EC%9D%B4%EB%82%98" title="우크라이나">우크라이나</a>의 <a href="/wiki/%EC%9E%A5%EB%8C%80%EB%86%92%EC%9D%B4%EB%9B%B0%EA%B8%B0" title="장대높이뛰기">장대높이뛰기</a> 선수이다. 현역 초기에는 옛 <a class="mw-redirect" href="/wiki/%EC%86%8C%EB%B9%84%EC%97%90%ED%8A%B8_%EC%97%B0%EB%B0%A9" title="소비에트 연방">소비에트 연방</a>을 대표해 국제경기에 참가했기 때문에 러시아어 이름인 <b>세르게이 붑카</b>로도 알려져 있다. 6.00미터와 6.10미터를 넘은 최초의 장대높이뛰기 선수이다. <a href="/wiki/1994%EB%85%84" title="1994년">1994년</a> <a href="/wiki/7%EC%9B%94_31%EC%9D%BC" title="7월 31일">7월 31일</a> <a href="/wiki/%EC%9D%B4%ED%83%88%EB%A6%AC%EC%95%84" title="이탈리아">이탈리아</a>의 세스트리에레에서 현 세계 기록인 6.14미터를 넘었다.</p>
<p>소비에트 연방의 보로실로브그라드(현 우크라이나의 <a href="/wiki/%EB%A3%A8%ED%95%9C%EC%8A%A4%ED%81%AC" title="루한스크">루한스크</a>)에서 태어나 아홉 살 때부터 장대높이뛰기를 했다. <a href="/wiki/1978%EB%85%84" title="1978년">1978년</a> 15세의 나이로 <a href="/wiki/%EB%8F%84%EB%84%A4%EC%B8%A0%ED%81%AC" title="도네츠크">도네츠크</a>로 옮겼다. <a href="/wiki/1983%EB%85%84" title="1983년">1983년</a> 국제 육상계에 데뷔해 <a href="/wiki/%ED%97%AC%EC%8B%B1%ED%82%A4" title="헬싱키">헬싱키</a>에서 5.70미터를 기록하며 우승한 후 장대높이뛰기의 역사를 새로 쓰기 시작했다. <a href="/wiki/1985%EB%85%84" title="1985년">1985년</a> <a href="/wiki/7%EC%9B%94_13%EC%9D%BC" title="7월 13일">7월 13일</a> <a class="mw-redirect" href="/wiki/%ED%8C%8C%EB%A6%AC_%EC%8B%9C" title="파리 시">파리</a>에서 오랫동안 불가능하다고 여겼던 6.00미터를 넘었다. 그 후 조금씩 세계 기록을 높여가며 <a href="/wiki/1988%EB%85%84" title="1988년">1988년</a> <a href="/wiki/%ED%94%84%EB%9E%91%EC%8A%A4" title="프랑스">프랑스</a> <a href="/wiki/%EB%8B%88%EC%8A%A4_(%ED%94%84%EB%9E%91%EC%8A%A4)" title="니스 (프랑스)">니스</a>에서 6.06미터를 넘어, <a class="mw-redirect" href="/wiki/%EC%84%9C%EC%9A%B8" title="서울">서울</a>에서 열리는 <a href="/wiki/1988%EB%85%84_%ED%95%98%EA%B3%84_%EC%98%AC%EB%A6%BC%ED%94%BD" title="1988년 하계 올림픽">1988년 하계 올림픽</a>에서 6.10미터를 넘을 수 있을지 큰 관심을 끌었다. 서울에서 그는 5.90미터를 기록, 기대에는 못미쳤지만 금메달을 목에 걸었다.</p>
<p>부브카는 1984년에서 1988년까지 4년 동안 세계 기록을 21센티미터나 향상시켰는데 그 이전 12년간 다른 장대높이뛰기 선수들이 세계 기록을 향상시켰던 높이보다 더 높은 수치이다. <a href="/wiki/1991%EB%85%84" title="1991년">1991년</a> <a class="mw-redirect" href="/wiki/%EC%97%90%EC%8A%A4%ED%8C%8C%EB%83%90" title="에스파냐">에스파냐</a> <a href="/wiki/%EC%82%B0%EC%84%B8%EB%B0%94%EC%8A%A4%ED%8B%B0%EC%95%88" title="산세바스티안">산세바스티안</a>에서는 최초로 6.10미터를 넘었다. 그러나 <a href="/wiki/%EB%B0%94%EB%A5%B4%EC%85%80%EB%A1%9C%EB%82%98" title="바르셀로나">바르셀로나</a>에서 열린 <a href="/wiki/1992%EB%85%84_%ED%95%98%EA%B3%84_%EC%98%AC%EB%A6%BC%ED%94%BD" title="1992년 하계 올림픽">1992년 하계 올림픽</a>에서는 예상 외로 부진해 메달 획득에 실패했다. 그래서 전성기가 지났다는 말이 나왔지만, <a href="/wiki/1994%EB%85%84" title="1994년">1994년</a>에는 현 세계 기록인 6.14미터를 넘었다. <a class="mw-redirect" href="/wiki/%EC%95%A0%ED%8B%80%EB%9E%80%ED%83%80" title="애틀란타">애틀란타</a>에서 열린 <a href="/wiki/1996%EB%85%84_%ED%95%98%EA%B3%84_%EC%98%AC%EB%A6%BC%ED%94%BD" title="1996년 하계 올림픽">1996년 하계 올림픽</a>에도 진출했지만 부상으로 참가하지 못했다.</p>
<p>1984년에서 1986년까지 3년 연속으로 소비에트 연방 최우수 스포츠맨에 뽑혔다. <a href="/wiki/2002%EB%85%84" title="2002년">2002년</a>부터는 우크라이나 국회의원으로 청소년 정책과 체육, 스포츠, 관광을 다루는 위원회에 속해 있다.</p>
<p>아들인 세르히 부브카 2세는 테니스 선수이다.</p>
