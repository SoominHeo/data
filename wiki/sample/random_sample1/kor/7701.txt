<p><b>세르히오 다니엘 바티스타</b>(<span style="font-size: smaller;"><a href="/wiki/%EC%8A%A4%ED%8E%98%EC%9D%B8%EC%96%B4" title="스페인어">스페인어</a>:</span> <span lang="es" xml:lang="es">Sergio Daniel Batista</span>, <a href="/wiki/1962%EB%85%84" title="1962년">1962년</a> <a href="/wiki/11%EC%9B%94_9%EC%9D%BC" title="11월 9일">11월 9일</a> ~ )는 전직 <a href="/wiki/%EC%95%84%EB%A5%B4%ED%97%A8%ED%8B%B0%EB%82%98" title="아르헨티나">아르헨티나</a> 출신의 <a href="/wiki/%EC%B6%95%EA%B5%AC" title="축구">축구</a> 선수이자 현직 아르헨티나 출신의 축구 감독으로, 현재 <a href="/wiki/%EB%B0%94%EB%A0%88%EC%9D%B8_%EC%B6%95%EA%B5%AC_%EA%B5%AD%EA%B0%80%EB%8C%80%ED%91%9C%ED%8C%80" title="바레인 축구 국가대표팀">바레인</a>의 감독이다. 현역 시절 <b>엘 체초</b>(El Checho)라는 별명으로 불렸다.</p>
<p>그는 현역 시절 수비수로서 <a href="/wiki/1986%EB%85%84_FIFA_%EC%9B%94%EB%93%9C%EC%BB%B5" title="1986년 FIFA 월드컵">1986년 FIFA 월드컵</a>과 <a href="/wiki/1990%EB%85%84_FIFA_%EC%9B%94%EB%93%9C%EC%BB%B5" title="1990년 FIFA 월드컵">1990년 FIFA 월드컵</a>의 아르헨티나 국가대표 엔트리에 올라 월드컵 우승과 준우승을 모두 경험했으며, 감독으로서 <a href="/wiki/2008%EB%85%84_%ED%95%98%EA%B3%84_%EC%98%AC%EB%A6%BC%ED%94%BD" title="2008년 하계 올림픽">2008년 하계 올림픽</a>에 감독으로 출전하여 올림픽 금메달 리스트의 경험도 갖고 있다. 그는 <a href="/wiki/2010%EB%85%84_FIFA_%EC%9B%94%EB%93%9C%EC%BB%B5" title="2010년 FIFA 월드컵">2010년 FIFA 월드컵</a>에서 <a href="/wiki/%EB%94%94%EC%97%90%EA%B3%A0_%EB%A7%88%EB%9D%BC%EB%8F%84%EB%82%98" title="디에고 마라도나">디에고 마라도나</a>가 이끌었던 아르헨티나가 부진한 성적을 거둔 뒤 마라도나의 후임 감독이 되었다. 2011년 코파아메리카 대회 직후 아르헨티나 축구협회가 아르헨티나 대표팀이 8강에서 탈락한 책임을 물어 바티스타를 경질했다.</p>
<p>바티스타는 <a href="/wiki/%EB%A6%AC%EC%98%A4%EB%84%AC_%EB%A9%94%EC%8B%9C" title="리오넬 메시">리오넬 메시</a>가 존경하는 스승으로도 유명하며<sup class="reference" id="cite_ref-1"><a href="#cite_note-1">[1]</a></sup>, 아르헨티나를 대표하는 축구 선수 중 하나인 <a href="/wiki/%EC%95%99%ED%97%AC_%EB%94%94_%EB%A7%88%EB%A6%AC%EC%95%84" title="앙헬 디 마리아">앙헬 디 마리아</a>를 발굴해 낸 장본인이기도 하다.</p>

<h2><span class="mw-headline" id=".EC.84.A0.EC.88.98_.EA.B2.BD.EB.A0.A5">선수 경력</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%84%B8%EB%A5%B4%ED%9E%88%EC%98%A4_%EB%B0%94%ED%8B%B0%EC%8A%A4%ED%83%80&amp;action=edit&amp;section=1" title="부분 편집: 선수 경력">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>바티스타는 <a href="/wiki/%EC%95%84%EB%A5%B4%ED%97%A8%ED%8B%B0%EB%85%B8%EC%8A%A4_%EC%A3%BC%EB%8B%88%EC%96%B4%EC%8A%A4" title="아르헨티노스 주니어스">아르헨티노스 주니어스</a> 유소년팀에서 축구 경력을 시작했다. 이후 프로 무대에 데뷔하여 <a href="/wiki/1985%EB%85%84" title="1985년">1985년</a> <a class="new" href="/w/index.php?title=%EB%A9%94%ED%8A%B8%EB%A1%9C%ED%8F%B4%EB%A6%AC%ED%83%80%EB%85%B8&amp;action=edit&amp;redlink=1" title="메트로폴리타노 (없는 문서)">메트로폴리타노</a>와 <a href="/wiki/%EC%BD%94%ED%8C%8C_%EB%A6%AC%EB%B2%A0%EB%A5%B4%ED%83%80%EB%8F%84%EB%A0%88%EC%8A%A4" title="코파 리베르타도레스">코파 리베르타도레스</a>에서 우승을 차지하였다. <a href="/wiki/1985%EB%85%84" title="1985년">1985년</a> 그는 <a href="/wiki/%EB%A6%AC%EB%B2%84_%ED%94%8C%EB%A0%88%EC%9D%B4%ED%8A%B8" title="리버 플레이트">리버 플레이트</a>로 이적하였고, 1989-90 시즌 팀이 리그 우승을 차지하는 데 크게 공헌하였다. 이후 그는 은퇴하기 전까지 아르헨티나의 <a href="/wiki/CA_%EB%88%84%EC%97%90%EB%B0%94_%EC%8B%9C%EC%B9%B4%EA%B3%A0" title="CA 누에바 시카고">CA 누에바 시카고</a>와 <a class="new" href="/w/index.php?title=%EC%98%AC_%EB%B3%B4%EC%9D%B4%EC%8A%A4&amp;action=edit&amp;redlink=1" title="올 보이스 (없는 문서)">올 보이스</a>, <a href="/wiki/%EC%9D%BC%EB%B3%B8" title="일본">일본</a>의 <a href="/wiki/%EC%82%AC%EA%B0%84_%EB%8F%84%EC%8A%A4" title="사간 도스">도스 퓨처스</a>에서 활약하였다.</p>
<p>그는 <a href="/wiki/%EC%95%84%EB%A5%B4%ED%97%A8%ED%8B%B0%EB%82%98_%EC%B6%95%EA%B5%AC_%EA%B5%AD%EA%B0%80%EB%8C%80%ED%91%9C%ED%8C%80" title="아르헨티나 축구 국가대표팀">아르헨티나 축구 국가대표팀</a>의 일원으로 <a href="/wiki/1986%EB%85%84_FIFA_%EC%9B%94%EB%93%9C%EC%BB%B5" title="1986년 FIFA 월드컵">1986년 FIFA 월드컵</a>에서 우승을 차지하였으며, <a href="/wiki/1990%EB%85%84_FIFA_%EC%9B%94%EB%93%9C%EC%BB%B5" title="1990년 FIFA 월드컵">1990년 FIFA 월드컵</a>에서는 결승전에 진출하였다.</p>
<h2><span class="mw-headline" id=".EC.A7.80.EB.8F.84.EC.9E.90_.EA.B2.BD.EB.A0.A5">지도자 경력</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%84%B8%EB%A5%B4%ED%9E%88%EC%98%A4_%EB%B0%94%ED%8B%B0%EC%8A%A4%ED%83%80&amp;action=edit&amp;section=2" title="부분 편집: 지도자 경력">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>바티스타는 <a href="/wiki/2000%EB%85%84" title="2000년">2000년</a> <a href="/wiki/%EC%9A%B0%EB%A3%A8%EA%B3%BC%EC%9D%B4" title="우루과이">우루과이</a>의 축구팀인 <a href="/wiki/CA_%EB%B2%A0%EC%95%BC_%EB%B9%84%EC%8A%A4%ED%83%80" title="CA 베야 비스타">CA 베야 비스타</a>에서 감독 경력을 시작하였다. <a href="/wiki/2001%EB%85%84" title="2001년">2001년</a> <a href="/wiki/%EC%95%84%EB%A5%B4%ED%97%A8%ED%8B%B0%EB%85%B8%EC%8A%A4_%EC%A3%BC%EB%8B%88%EC%96%B4%EC%8A%A4" title="아르헨티노스 주니어스">아르헨티노스 주니어스</a>에서 2년을 보낸 뒤 <a class="mw-redirect" href="/wiki/%ED%83%80%EC%98%88%EB%A0%88%EC%8A%A4_%EB%8D%B0_%EC%BD%94%EB%A5%B4%EB%8F%84%EB%B0%94" title="타예레스 데 코르도바">타예레스 데 코르도바</a>에서 1년 동안 활약하였다. <a href="/wiki/2004%EB%85%84" title="2004년">2004년</a> 잠시 <a href="/wiki/%EC%95%84%EB%A5%B4%ED%97%A8%ED%8B%B0%EB%85%B8%EC%8A%A4_%EC%A3%BC%EB%8B%88%EC%96%B4%EC%8A%A4" title="아르헨티노스 주니어스">아르헨티노스 주니어스</a>로 복귀한 뒤, <a href="/wiki/CA_%EB%88%84%EC%97%90%EB%B0%94_%EC%8B%9C%EC%B9%B4%EA%B3%A0" title="CA 누에바 시카고">CA 누에바 시카고</a>로 옮겨 1년 동안 팀의 감독직을 맡았다. <a href="/wiki/2005%EB%85%84" title="2005년">2005년</a>부터 <a href="/wiki/2006%EB%85%84" title="2006년">2006년</a>까지는 <a href="/wiki/CA_%EC%82%B0%EB%A1%9C%EB%A0%8C%EC%86%8C" title="CA 산로렌소">CA 산로렌소</a>의 <a href="/wiki/%EC%98%A4%EC%8A%A4%EC%B9%B4%EB%A5%B4_%EB%A3%A8%ED%97%A4%EB%A6%AC" title="오스카르 루헤리">오스카르 루헤리</a> 밑에서 코치로 활동하였다.</p>
<p><a href="/wiki/2008%EB%85%84_%ED%95%98%EA%B3%84_%EC%98%AC%EB%A6%BC%ED%94%BD" title="2008년 하계 올림픽">2008년 하계 올림픽</a> 당시 <a href="/wiki/%EC%95%84%EB%A5%B4%ED%97%A8%ED%8B%B0%EB%82%98_%EC%B6%95%EA%B5%AC_%EA%B5%AD%EA%B0%80%EB%8C%80%ED%91%9C%ED%8C%80" title="아르헨티나 축구 국가대표팀">아르헨티나 올림픽 국가대표팀</a> 감독으로 선임된 바티스타는 <a href="/wiki/%EB%A6%AC%EC%98%A4%EB%84%AC_%EB%A9%94%EC%8B%9C" title="리오넬 메시">리오넬 메시</a>, <a href="/wiki/%EC%95%99%ED%97%AC_%EB%94%94_%EB%A7%88%EB%A6%AC%EC%95%84" title="앙헬 디 마리아">앙헬 디 마리아</a>, <a href="/wiki/%ED%95%98%EB%B9%84%EC%97%90%EB%A5%B4_%EB%A7%88%EC%8A%A4%EC%B2%B4%EB%9D%BC%EB%85%B8" title="하비에르 마스체라노">하비에르 마스체라노</a> 등의 호화로운 멤버들을 이끌고 올림픽에 출전했다. 바티스타가 지휘하는 아르헨티나는 <a href="/wiki/%EC%BD%94%ED%8A%B8%EB%94%94%EB%B6%80%EC%95%84%EB%A5%B4_%EC%B6%95%EA%B5%AC_%EA%B5%AD%EA%B0%80%EB%8C%80%ED%91%9C%ED%8C%80" title="코트디부아르 축구 국가대표팀">코트디부아르</a>, <a href="/wiki/%EC%98%A4%EC%8A%A4%ED%8A%B8%EB%A0%88%EC%9D%BC%EB%A6%AC%EC%95%84_%EC%B6%95%EA%B5%AC_%EA%B5%AD%EA%B0%80%EB%8C%80%ED%91%9C%ED%8C%80" title="오스트레일리아 축구 국가대표팀">오스트레일리아</a>, <a href="/wiki/%EC%84%B8%EB%A5%B4%EB%B9%84%EC%95%84_%EC%B6%95%EA%B5%AC_%EA%B5%AD%EA%B0%80%EB%8C%80%ED%91%9C%ED%8C%80" title="세르비아 축구 국가대표팀">세르비아</a>를 모두 이기고 8강에 진출했다. 조별리그에서 아르헨티나는 1실점에 지나지 않았다. 8강 상대는 <a href="/wiki/%EB%84%A4%EB%8D%9C%EB%9E%80%EB%93%9C_%EC%B6%95%EA%B5%AC_%EA%B5%AD%EA%B0%80%EB%8C%80%ED%91%9C%ED%8C%80" title="네덜란드 축구 국가대표팀">네덜란드</a>였는데 네덜란드와는 연장 혈투 끝에 디 마리아의 결승골로 준결승에 진출했다. 아르헨티나는 준결승에서 <a href="/wiki/%EB%B8%8C%EB%9D%BC%EC%A7%88_%EC%B6%95%EA%B5%AC_%EA%B5%AD%EA%B0%80%EB%8C%80%ED%91%9C%ED%8C%80" title="브라질 축구 국가대표팀">브라질</a>을 3-0으로, 결승에서 <a href="/wiki/%EB%82%98%EC%9D%B4%EC%A7%80%EB%A6%AC%EC%95%84_%EC%B6%95%EA%B5%AC_%EA%B5%AD%EA%B0%80%EB%8C%80%ED%91%9C%ED%8C%80" title="나이지리아 축구 국가대표팀">나이지리아</a>를 1-0으로 각각 연파하고 우승을 차지했다. 본선에서 역시 단 1실점으로 브라질과 나이지리아를 연파한 아르헨티나는 승부차기 없이 총합 11득점 2실점의 모든 경기 전승으로 우승을 차지했다. 바티스타가 이끌던 당시의 올림픽 대표팀의 전력은 그 멤버 그대로 월드컵에 출전해도 우승한다는 극찬을 받았다.</p>
<p><a href="/wiki/2010%EB%85%84_FIFA_%EC%9B%94%EB%93%9C%EC%BB%B5" title="2010년 FIFA 월드컵">2010년 FIFA 월드컵</a>에서 <a href="/wiki/%EB%94%94%EC%97%90%EA%B3%A0_%EB%A7%88%EB%9D%BC%EB%8F%84%EB%82%98" title="디에고 마라도나">디에고 마라도나</a>가 감독으로 있던 아르헨티나가 독일에게 0-4로 충격적인 완패를 당한 뒤 8강에서 탈락하자 마라도나의 후임으로 감독에 취임하였으며, <a href="/wiki/%EC%95%84%EC%9D%BC%EB%9E%9C%EB%93%9C_%EC%B6%95%EA%B5%AC_%EA%B5%AD%EA%B0%80%EB%8C%80%ED%91%9C%ED%8C%80" title="아일랜드 축구 국가대표팀">아일랜드</a>와의 평가전에서 신승을 거둔 후 아르헨티나 독립 200주년 기념행사의 일환으로 <a href="/wiki/%EC%8A%A4%ED%8E%98%EC%9D%B8_%EC%B6%95%EA%B5%AC_%EA%B5%AD%EA%B0%80%EB%8C%80%ED%91%9C%ED%8C%80" title="스페인 축구 국가대표팀">스페인</a>과 친선경기에서 4-1 완승을 거뒀다. 이로 인하여 바티스타의 신뢰는 높아져가고 있으며, 메시를 비롯한 아르헨티나의 선수들은 바티스타가 <a href="/wiki/2014%EB%85%84_FIFA_%EC%9B%94%EB%93%9C%EC%BB%B5" title="2014년 FIFA 월드컵">2014년 FIFA 월드컵</a>이후까지 국가대표 감독을 담당하길 원하고 있다.</p>
<p>그러나 2010년 10월 8일 <a href="/wiki/%EC%9D%BC%EB%B3%B8_%EC%B6%95%EA%B5%AC_%EA%B5%AD%EA%B0%80%EB%8C%80%ED%91%9C%ED%8C%80" title="일본 축구 국가대표팀">일본</a>과의 평가전에서 일격을 당하기도 했다. 그러나 이는 상당수의 선수들이 부상으로 출전이 어려운 상태였으며 <a href="/wiki/%EB%A6%AC%EC%98%A4%EB%84%AC_%EB%A9%94%EC%8B%9C" title="리오넬 메시">리오넬 메시</a>는 부상을 당함에도 불구하고 억지로 출전하여 제 기량을 발휘하기 힘든 상황이였기 때문에 가능한 것이다.<sup class="reference" id="cite_ref-2"><a href="#cite_note-2">[2]</a></sup></p>
<p>이후 2010년 11월 18일에는 아르헨티나의 라이벌 <a href="/wiki/%EB%B8%8C%EB%9D%BC%EC%A7%88_%EC%B6%95%EA%B5%AC_%EA%B5%AD%EA%B0%80%EB%8C%80%ED%91%9C%ED%8C%80" title="브라질 축구 국가대표팀">브라질</a>을 상대로 신승을 거두었으며 2011년 2월 10일 <a href="/wiki/%ED%81%AC%EB%A6%AC%EC%8A%A4%ED%8B%B0%EC%95%84%EB%88%84_%ED%98%B8%EB%82%A0%EB%91%90" title="크리스티아누 호날두">크리스티아누 호날두</a>의 <a href="/wiki/%ED%8F%AC%EB%A5%B4%ED%88%AC%EA%B0%88_%EC%B6%95%EA%B5%AC_%EA%B5%AD%EA%B0%80%EB%8C%80%ED%91%9C%ED%8C%80" title="포르투갈 축구 국가대표팀">포르투갈</a>과의 평가전에서 명승부를 연출한 끝에 신승을 거두는 등 좋은 성적을 거두면서 아르헨티나 축구 국가대표팀의 정식 감독으로 임명되었다. 정식 감독으로 임명된 이후에는 국가대표 예비팀을 만들어서 원래 있던 국가대표 본팀과 국내파 위주로 편성한 국가대표 예비팀을 같이 운용하고 있다.</p>
<p>그러나 아르헨티나에서 개최된 <a href="/wiki/2011%EB%85%84_%EC%BD%94%ED%8C%8C_%EC%95%84%EB%A9%94%EB%A6%AC%EC%B9%B4" title="2011년 코파 아메리카">2011년 코파 아메리카</a>에서 우루과이를 상대로 승부차기까지 가는 접전을 벌였으나 양팀에서 나온 10명의 키커들 중에 <a href="/wiki/%EC%B9%B4%EB%A5%BC%EB%A1%9C%EC%8A%A4_%ED%85%8C%EB%B2%A0%EC%8A%A4" title="카를로스 테베스">카를로스 테베스</a>혼자서만 실축하고 나머지 9명의 키커가 모두 성공하는 바람에 우루과이에 밀려 준결승 진출이 좌절되자 이에 대한 책임을 물어서 <a href="/wiki/%EC%95%84%EB%A5%B4%ED%97%A8%ED%8B%B0%EB%82%98_%EC%B6%95%EA%B5%AC_%EA%B5%AD%EA%B0%80%EB%8C%80%ED%91%9C%ED%8C%80" title="아르헨티나 축구 국가대표팀">아르헨티나 축구 국가대표팀</a>의 감독에서 전격 경질되었다. 경기내용 자체는 아르헨티나가 우루과이보다 훨씬 우세했지만 우루과이의 골키퍼인 <a href="/wiki/%ED%8E%98%EB%A5%B4%EB%82%9C%EB%8F%84_%EB%AC%B4%EC%8A%AC%EB%A0%88%EB%9D%BC" title="페르난도 무슬레라">페르난도 무슬레라</a>가 20회 이상 유효슈팅을 선방하는 데에 성공한 탓에 승부차기까지 갔다. 그 후 <a href="/wiki/%EC%95%84%EB%A5%B4%ED%97%A8%ED%8B%B0%EB%82%98_%EC%B6%95%EA%B5%AC_%EA%B5%AD%EA%B0%80%EB%8C%80%ED%91%9C%ED%8C%80" title="아르헨티나 축구 국가대표팀">아르헨티나</a>에서는 그의 후임으로 <a href="/wiki/%EC%95%8C%EB%A0%88%ED%95%9C%EB%93%9C%EB%A1%9C_%EC%82%AC%EB%B2%A8%EB%9D%BC" title="알레한드로 사벨라">알레한드로 사벨라</a>를 선임하였다.</p>
<p><a href="/wiki/2012%EB%85%84" title="2012년">2012년</a> <a href="/wiki/5%EC%9B%94_30%EC%9D%BC" title="5월 30일">5월 30일</a> <a class="mw-redirect" href="/wiki/%EC%83%81%ED%95%98%EC%9D%B4_%EC%84%A0%ED%99%94" title="상하이 선화">상하이 선화</a>의 감독으로 부임하였다.</p>

<h2><span class="mw-headline" id=".EA.B8.B0.ED.83.80">기타</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%84%B8%EB%A5%B4%ED%9E%88%EC%98%A4_%EB%B0%94%ED%8B%B0%EC%8A%A4%ED%83%80&amp;action=edit&amp;section=4" title="부분 편집: 기타">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<ul>
<li>선수 시절에는 수염을 길렀으나 감독이 된 이후에는 선수시절 길렀던 수염을 말끔하게 면도했다.</li>
<li>현재 바티스타는 <a href="/wiki/%EC%95%84%EB%A5%B4%ED%97%A8%ED%8B%B0%EB%82%98_%EC%B6%95%EA%B5%AC_%EA%B5%AD%EA%B0%80%EB%8C%80%ED%91%9C%ED%8C%80" title="아르헨티나 축구 국가대표팀">아르헨티나</a>를 이끌고 있는데 아르헨티나 축구 국가대표 본팀<sup class="reference" id="cite_ref-3"><a href="#cite_note-3">[3]</a></sup> 과 아르헨티나 축구 국가대표 예비팀<sup class="reference" id="cite_ref-4"><a href="#cite_note-4">[4]</a></sup> 으로 나눠서 두 팀을 같이 운용한다. 바티스타는 이 두 팀을 따로 사용하기도 하지만 적절히 섞어서 사용하기도 한다. <a href="/wiki/%EC%8A%A4%ED%8E%98%EC%9D%B8_%EC%B6%95%EA%B5%AC_%EA%B5%AD%EA%B0%80%EB%8C%80%ED%91%9C%ED%8C%80" title="스페인 축구 국가대표팀">스페인</a>, <a href="/wiki/%EB%B8%8C%EB%9D%BC%EC%A7%88_%EC%B6%95%EA%B5%AC_%EA%B5%AD%EA%B0%80%EB%8C%80%ED%91%9C%ED%8C%80" title="브라질 축구 국가대표팀">브라질</a>, <a href="/wiki/%ED%8F%AC%EB%A5%B4%ED%88%AC%EA%B0%88_%EC%B6%95%EA%B5%AC_%EA%B5%AD%EA%B0%80%EB%8C%80%ED%91%9C%ED%8C%80" title="포르투갈 축구 국가대표팀">포르투갈</a>처럼 아르헨티나 축구 국가대표팀과 박빙의 역량을 낼 수 있는 팀은 본팀만으로 평가전을 치르고 <a href="/wiki/%EB%AF%B8%EA%B5%AD_%EC%B6%95%EA%B5%AC_%EA%B5%AD%EA%B0%80%EB%8C%80%ED%91%9C%ED%8C%80" title="미국 축구 국가대표팀">미국</a>처럼 아르헨티나 축구 국가대표팀보다 한 수 아래의 팀은 본팀과 예비팀을 적절히 섞어서 평가전을 치르며 <a href="/wiki/%ED%8F%B4%EB%9E%80%EB%93%9C_%EC%B6%95%EA%B5%AC_%EA%B5%AD%EA%B0%80%EB%8C%80%ED%91%9C%ED%8C%80" title="폴란드 축구 국가대표팀">폴란드</a>처럼 아르헨티나의 입장에서 아예 상대가 안되는 약체는 예비팀으로만 평가전을 치른다. 이런 지도 방식은 <a href="/wiki/2002%EB%85%84_FIFA_%EC%9B%94%EB%93%9C%EC%BB%B5" title="2002년 FIFA 월드컵">2002년 FIFA 월드컵</a>당시 <a href="/wiki/%EB%8C%80%ED%95%9C%EB%AF%BC%EA%B5%AD_%EC%B6%95%EA%B5%AC_%EA%B5%AD%EA%B0%80%EB%8C%80%ED%91%9C%ED%8C%80" title="대한민국 축구 국가대표팀">대한민국</a>을 이끌었을 당시의 <a href="/wiki/%EA%B1%B0%EC%8A%A4_%ED%9E%88%EB%94%A9%ED%81%AC" title="거스 히딩크">거스 히딩크</a>보다 한 단계 발전된 지도방식이다.</li>
</ul>
