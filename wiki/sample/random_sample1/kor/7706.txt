<p>1989년 <a href="/wiki/%EC%BD%94%ED%8C%8C_%EC%95%84%EB%A9%94%EB%A6%AC%EC%B9%B4" title="코파 아메리카">코파 아메리카</a>에서 브라질의 우승을 이끌었으며 같은 해에 남아메리카 올해의 축구 감독으로 선정되었다. <a href="/wiki/1990%EB%85%84_FIFA_%EC%9B%94%EB%93%9C%EC%BB%B5" title="1990년 FIFA 월드컵">1990년 FIFA 월드컵</a>에서 <a href="/wiki/%EB%B8%8C%EB%9D%BC%EC%A7%88_%EC%B6%95%EA%B5%AC_%EA%B5%AD%EA%B0%80%EB%8C%80%ED%91%9C%ED%8C%80" title="브라질 축구 국가대표팀">브라질 대표팀</a> 감독을 역임하면서 3-5-2 시스템을 도입했지만 브라질은 <a href="/wiki/%EC%95%84%EB%A5%B4%ED%97%A8%ED%8B%B0%EB%82%98_%EC%B6%95%EA%B5%AC_%EA%B5%AD%EA%B0%80%EB%8C%80%ED%91%9C%ED%8C%80" title="아르헨티나 축구 국가대표팀">아르헨티나</a>와의 16강전에서 패하면서 탈락하고 만다. 브라질 대표팀 감독을 역임하는 동안 35경기를 치렀으며 21승 7무 7패를 기록했다.</p>
<p>브라질의 <a href="/wiki/CR_%ED%94%8C%EB%9D%BC%EB%A9%A9%EA%B5%AC" title="CR 플라멩구">플라멩구</a>와 <a href="/wiki/CR_%EB%B0%94%EC%8A%A4%EC%BF%A0_%EB%8B%A4_%EA%B0%80%EB%A7%88" title="CR 바스쿠 다 가마">바스쿠 다 가마</a>, <a href="/wiki/%EA%B7%B8%EB%A0%88%EB%AF%B8%EC%9A%B0" title="그레미우">그레미우</a>, <a href="/wiki/%EB%B3%B4%ED%83%80%ED%8F%AC%EA%B5%AC_FR" title="보타포구 FR">보타포구</a>, 카타르의 <a href="/wiki/%EC%95%8C%EC%95%84%ED%9D%98%EB%A6%AC_(%EB%8F%84%ED%95%98)" title="알아흘리 (도하)">알아흘리 도하</a>, 사우디아라비아의 <a href="/wiki/%EC%95%8C%ED%9E%90%EB%9E%84_(%EB%A6%AC%EC%95%BC%EB%93%9C)" title="알힐랄 (리야드)">알힐랄</a>, 이탈리아의 <a href="/wiki/ACF_%ED%94%BC%EC%98%A4%EB%A0%8C%ED%8B%B0%EB%82%98" title="ACF 피오렌티나">피오렌티나</a>, 터키의 <a href="/wiki/%ED%8E%98%EB%84%A4%EB%A5%B4%EB%B0%94%ED%9D%90%EC%B2%B4_SK" title="페네르바흐체 SK">페네르바흐체</a>, 중국의 <a class="mw-redirect" href="/wiki/%EC%83%81%ED%95%98%EC%9D%B4_%EC%84%A0%ED%99%94" title="상하이 선화">상하이 선화</a>, 일본의 <a href="/wiki/%EC%9A%94%EC%BD%94%ED%95%98%EB%A7%88_F._%EB%A7%88%EB%A6%AC%EB%85%B8%EC%8A%A4" title="요코하마 F. 마리노스">요코하마 F. 마리노스</a> 등 몇몇 축구 클럽에서 감독을 역임했다.</p>
