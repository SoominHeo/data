<p><b>세바스티아누스</b> 또는 <b>세바스티아노</b>(<span style="font-size: smaller;"><a href="/wiki/%EC%9D%B4%ED%83%88%EB%A6%AC%EC%95%84%EC%96%B4" title="이탈리아어">이탈리아어</a>:</span> <span lang="it" xml:lang="it">San Sebastiano</span>, ? - <a href="/wiki/287%EB%85%84" title="287년">287년</a> <a href="/wiki/1%EC%9B%94_20%EC%9D%BC" title="1월 20일">1월 20일</a>)는 초기 <a href="/wiki/%EA%B8%B0%EB%8F%85%EA%B5%90" title="기독교">기독교</a>의 순교자 가운데 한 사람으로 <a href="/wiki/3%EC%84%B8%EA%B8%B0" title="3세기">3세기</a> <a class="mw-redirect" href="/wiki/%EB%A1%9C%EB%A7%88_%ED%99%A9%EC%A0%9C" title="로마 황제">로마 황제</a>의 친위대원이었으나 기독교로 개종한 죄목으로 파직되어 순교하였다.</p>
<p>기독교의 <a href="/wiki/%EC%84%B1%EC%9D%B8_(%EC%A2%85%EA%B5%90)" title="성인 (종교)">성인</a>. <a class="mw-redirect" href="/wiki/%EB%A1%9C%EB%A7%88_%EA%B0%80%ED%86%A8%EB%A6%AD" title="로마 가톨릭">로마 가톨릭</a>에서의 축일은 <a href="/wiki/1%EC%9B%94_20%EC%9D%BC" title="1월 20일">1월 20일</a>이며 <a class="mw-redirect" href="/wiki/%EB%8F%99%EB%B0%A9%EC%A0%95%EA%B5%90%ED%9A%8C" title="동방정교회">동방정교회</a>에서의 축일은 <a href="/wiki/12%EC%9B%94_18%EC%9D%BC" title="12월 18일">12월 18일</a>이다. 그의 이름은 <a href="/wiki/%EA%B7%B8%EB%A6%AC%EC%8A%A4%EC%96%B4" title="그리스어">그리스어</a>로 ‘존경스러운’이란 뜻을 갖고 있다. <a href="/wiki/7%EC%84%B8%EA%B8%B0" title="7세기">7</a>-<a href="/wiki/8%EC%84%B8%EA%B8%B0" title="8세기">8세기</a>에는 <a href="/wiki/%EC%8B%AD%EC%9E%90%EA%B0%80" title="십자가">십자가</a>와 종려나무 가지를 들고 있거나 면류관을 쓴 젊은이로 그려졌으나, <a href="/wiki/%EC%A4%91%EC%84%B8" title="중세">중세</a>에는 활과 화살을 가진 기사로 그려졌다.</p>
<p>그러나 <a href="/wiki/%EB%A5%B4%EB%84%A4%EC%83%81%EC%8A%A4" title="르네상스">르네상스</a> 이후에는 원기둥에 밧줄로 묶인 채 화살을 맞은 젊은이의 모습을 선호하게 되었다. 대중적인 전설에서 그의 상처는 <a href="/wiki/%EC%98%88%EC%88%98" title="예수">예수</a>의 성흔에 비유되었다. 그러한 이유로 그의 몸을 꿰뚫은 화살은 총 5개로 그려진다. 운동선수와 사수·교통순경·태피스트리 제작자·전염병 희생자의 수호 성인이다.</p>
<h2><span class="mw-headline" id=".ED.96.89.EC.A0.81">행적</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%84%B8%EB%B0%94%EC%8A%A4%ED%8B%B0%EC%95%84%EB%88%84%EC%8A%A4&amp;action=edit&amp;section=1" title="부분 편집: 행적">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<div class="thumb tleft">
<div class="thumbinner" style="width:152px;"><a class="image" href="/wiki/%ED%8C%8C%EC%9D%BC:Paolo_Veronese_028.jpg"><img alt="" class="thumbimage" data-file-height="1367" data-file-width="800" height="256" src="//upload.wikimedia.org/wikipedia/commons/thumb/9/91/Paolo_Veronese_028.jpg/150px-Paolo_Veronese_028.jpg" srcset="//upload.wikimedia.org/wikipedia/commons/thumb/9/91/Paolo_Veronese_028.jpg/225px-Paolo_Veronese_028.jpg 1.5x, //upload.wikimedia.org/wikipedia/commons/thumb/9/91/Paolo_Veronese_028.jpg/300px-Paolo_Veronese_028.jpg 2x" width="150"/></a>

<p>세바스티아누스는 <a href="/wiki/%EA%B0%88%EB%A6%AC%EC%95%84" title="갈리아">갈리아</a> 출신의 <a href="/wiki/%EB%A1%9C%EB%A7%88_%EC%A0%9C%EA%B5%AD" title="로마 제국">로마 제국</a> 군인으로 나르본에서 태어났다고 한다. 그는 <a href="/wiki/%EB%B0%80%EB%9D%BC%EB%85%B8" title="밀라노">밀라노</a>에서 <a class="mw-redirect" href="/wiki/%EB%A1%9C%EB%A7%88_%ED%99%A9%EC%A0%9C" title="로마 황제">로마 황제</a> <a href="/wiki/%EB%94%94%EC%98%A4%ED%81%B4%EB%A0%88%ED%8B%B0%EC%95%84%EB%88%84%EC%8A%A4" title="디오클레티아누스">디오클레티아누스</a>의 군인이 되었다.</p>
<p><a href="/wiki/%EA%B8%B0%EB%8F%85%EA%B5%90" title="기독교">기독교</a>로 개종한 후 그는 자신의 지위를 이용하여 감옥에 갇힌 다른 <a class="mw-redirect" href="/wiki/%EA%B7%B8%EB%A6%AC%EC%8A%A4%EB%8F%84%EC%9D%B8" title="그리스도인">그리스도인</a>들을 도왔다. 그러나 그가 그리스도인이라는 것이 밝혀지자 황제는 그에게 사형을 명했다. 세바스티아누스는 기둥에 묶여 화살을 맞고 죽도록 버려졌다. 그러나 이레네라는 미망인이 그를 발견하여 치료해주었다.</p>
<p>세바스티아누스는 다시 한 번 디오클레티아누스 황제 앞에 서서 기독교를 박해하는 황제를 호되게 꾸짖었다. 황제는 그를 붙잡아 몽둥이로 쳐서 죽이라고 명령을 내렸으며 그의 시신은 로마의 하수구에 던져졌다.</p>
<p>한 그리스도인 여인의 꿈에 세바스티아누스가 나타나 자신의 시신이 있는 곳을 알려주었으며 여인은 세바스티아누스의 시신을 찾아 매장했다.</p>
