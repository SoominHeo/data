<p><b>세실 존 로즈</b>(Cecil John Rhodes, <a href="/wiki/1853%EB%85%84" title="1853년">1853년</a> <a href="/wiki/7%EC%9B%94_5%EC%9D%BC" title="7월 5일">7월 5일</a> ~ <a href="/wiki/1902%EB%85%84" title="1902년">1902년</a> <a href="/wiki/3%EC%9B%94_26%EC%9D%BC" title="3월 26일">3월 26일</a>)는 <a href="/wiki/%EC%98%81%EA%B5%AD" title="영국">영국</a>의 <a class="mw-redirect" href="/wiki/%EC%A0%95%EC%B9%98%EC%9D%B8" title="정치인">정치인</a>이다. <a href="/wiki/%EC%BC%80%EC%9D%B4%ED%94%84_%EC%8B%9D%EB%AF%BC%EC%A7%80" title="케이프 식민지">케이프 식민지</a>의 총리로 <a href="/wiki/%EC%98%81%EA%B5%AD" title="영국">영국</a> 정부가 추진한 <a href="/wiki/%EC%95%84%ED%94%84%EB%A6%AC%EC%B9%B4_%EC%A2%85%EB%8B%A8_%EC%A0%95%EC%B1%85" title="아프리카 종단 정책">아프리카 종단 정책</a>에 가담한 것으로 유명한 <a href="/wiki/%EC%A0%9C%EA%B5%AD%EC%A3%BC%EC%9D%98" title="제국주의">제국주의자</a>이다.</p>
<p><a href="/wiki/1870%EB%85%84" title="1870년">1870년</a> <a href="/wiki/%EB%82%A8%EC%95%84%ED%94%84%EB%A6%AC%EC%B9%B4" title="남아프리카">남아프리카</a>에 이주하여 <a class="mw-disambig" href="/wiki/%ED%82%B4%EB%B2%8C%EB%A6%AC" title="킴벌리">킴벌리</a>에서 <a href="/wiki/%EB%8B%A4%EC%9D%B4%EC%95%84%EB%AA%AC%EB%93%9C" title="다이아몬드">다이아몬드</a> 채굴에 종사하였고, <a href="/wiki/%EB%A1%9C%EB%94%94%EC%A7%80%EC%95%84" title="로디지아">로디지아</a>의 개발을 위해 <a class="new" href="/w/index.php?title=%EC%98%81%EA%B5%AD_%EB%82%A8%EC%95%84%ED%94%84%EB%A6%AC%EC%B9%B4_%ED%9A%8C%EC%82%AC&amp;action=edit&amp;redlink=1" title="영국 남아프리카 회사 (없는 문서)">영국 남아프리카 회사</a> 설립의 특허를 받았다. 그 사이 다이아몬드 광산을 통합 독점하고 거부(巨富)가 되었다. <a href="/wiki/%EC%BC%80%EC%9D%B4%ED%94%84_%EC%8B%9D%EB%AF%BC%EC%A7%80" title="케이프 식민지">케이프 식민지</a> 정계에 투신, 1890년에 총리가 되었으며, 행정 개혁과 영국 식민지의 확대를 위해 노력하였으나, R. 제임슨의 침입을 원조한 이유로 사직하였다. 그 후로도 로디지아 개발에 노력하였고 <a href="/wiki/%EB%B3%B4%EC%96%B4_%EC%A0%84%EC%9F%81" title="보어 전쟁">보어 전쟁</a>이 일어나자 킴벌리에서 농성하였으나, 건강이 나빠져서 전쟁이 끝나기 전에 <a class="mw-redirect" href="/wiki/%EC%BC%80%EC%9D%B4%ED%94%84_%ED%83%80%EC%9A%B4" title="케이프 타운">케이프 타운</a> 근처에서 죽었다. <a href="/wiki/%EC%B9%B4%EC%9D%B4%EB%A1%9C" title="카이로">카이로</a>와 <a class="mw-redirect" href="/wiki/%EC%BC%80%EC%9D%B4%ED%94%84_%ED%83%80%EC%9A%B4" title="케이프 타운">케이프 타운</a>을 잇는 전신선의 완성을 풍자한 만화도 있다.</p>
