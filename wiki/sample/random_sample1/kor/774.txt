<p><b>ATS-27 평택함</b>은 <a href="/wiki/%EB%8C%80%ED%95%9C%EB%AF%BC%EA%B5%AD_%ED%95%B4%EA%B5%B0" title="대한민국 해군">대한민국 해군</a>의 <a class="new" href="/w/index.php?title=%EA%B5%AC%EB%82%9C%ED%95%A8&amp;action=edit&amp;redlink=1" title="구난함 (없는 문서)">구난함</a>이다. <a class="mw-redirect" href="/wiki/%EB%AF%B8_%ED%95%B4%EA%B5%B0" title="미 해군">미 해군</a>이 쓰던 중고 구난함을 도입한 것이다. <a href="/wiki/1968%EB%85%84" title="1968년">1968년</a>에 건조되어 <a href="/wiki/1972%EB%85%84" title="1972년">1972년</a> 미 해군에 취역, 1996년 퇴역과 동시에 대한민국 해군에 재취역했다가 <a href="/wiki/2016%EB%85%84" title="2016년">2016년</a>에 최종적으로 퇴역했다.</p>
<h2><span class="mw-headline" id=".ED.95.A8.EC.82.AC">함사</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=ATS-27_%ED%8F%89%ED%83%9D&amp;action=edit&amp;section=1" title="부분 편집: 함사">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>평택함은 <a href="/wiki/1968%EB%85%84" title="1968년">1968년</a> <a href="/wiki/2%EC%9B%94_19%EC%9D%BC" title="2월 19일">2월 19일</a> <a href="/wiki/%EC%98%81%EA%B5%AD" title="영국">영국</a>의 <a class="new" href="/w/index.php?title=%EB%B8%8C%EB%A3%A8%ED%81%AC_%EB%A7%88%EB%A6%B0&amp;action=edit&amp;redlink=1" title="브루크 마린 (없는 문서)">브루크 마린</a> 사에서 건조되어 <a href="/wiki/1972%EB%85%84" title="1972년">1972년</a>부터 <a href="/wiki/1996%EB%85%84" title="1996년">1996년</a>까지 24년간 <a href="/wiki/%EB%AF%B8%EA%B5%AD_%ED%95%B4%EA%B5%B0" title="미국 해군">미국 해군</a>에서 임무를 수행하고 퇴역했다.<sup class="reference" id="cite_ref-.ED.87.B4.EC.97.AD_1-1"><a href="#cite_note-.ED.87.B4.EC.97.AD-1">[1]</a></sup> 당시 <a href="/wiki/%EB%8C%80%ED%95%9C%EB%AF%BC%EA%B5%AD_%ED%95%B4%EA%B5%B0" title="대한민국 해군">대한민국 해군</a>은 광활한 작전해역에서 보다 우수한 능력을 보유한 구조함이 필요하다고 판단하여<sup class="reference" id="cite_ref-.ED.87.B4.EC.97.AD_1-2"><a href="#cite_note-.ED.87.B4.EC.97.AD-1">[1]</a></sup> 미국의 <a class="new" href="/w/index.php?title=%EC%9D%B4%EB%93%A0%ED%84%B4%EA%B8%89_%EA%B5%AC%EB%82%9C%ED%95%A8&amp;action=edit&amp;redlink=1" title="이든턴급 구난함 (없는 문서)">이든턴급 구난함</a>인 뷰포트함과 브룬스윅함 2척을 중고로 구입했다. 뷰포트함은 <a class="mw-redirect" href="/wiki/%ED%8F%89%ED%83%9D%ED%95%A8" title="평택함">평택함</a>으로, 브룬스윅함은 <a class="mw-redirect" href="/wiki/%EA%B4%91%EC%96%91%ED%95%A8" title="광양함">광양함</a>으로 명명했다.</p>
<p>평택함은 <a href="/wiki/1997%EB%85%84" title="1997년">1997년</a> <a href="/wiki/4%EC%9B%94_1%EC%9D%BC" title="4월 1일">4월 1일</a> 취역과 함께 5성분전단 55구조·군수지원전대에 예속돼 <a href="/wiki/2016%EB%85%84" title="2016년">2016년</a>까지 20년 동안 동·서·남해 모든 해역에서 침몰 및 좌초된 선박의 구조, 함정 예인, 항만 및 수로 장애물 제거 지원 등 다양한 임무를 수행했다.<sup class="reference" id="cite_ref-.ED.87.B4.EC.97.AD_1-3"><a href="#cite_note-.ED.87.B4.EC.97.AD-1">[1]</a></sup></p>
<p>평택함은 <a class="new" href="/w/index.php?title=2007%EB%85%84_%ED%83%9C%EC%95%88_%EA%B8%B0%EB%A6%84%EC%9C%A0%EC%B6%9C_%EC%82%AC%EA%B3%A0&amp;action=edit&amp;redlink=1" title="2007년 태안 기름유출 사고 (없는 문서)">2007년 태안 기름유출 사고</a> 방재작전, <a class="new" href="/w/index.php?title=2010%EB%85%84_%EC%B2%9C%EC%95%88%ED%95%A8_%EC%82%AC%EA%B1%B4&amp;action=edit&amp;redlink=1" title="2010년 천안함 사건 (없는 문서)">2010년 천안함</a> 구조 및 인양작전, 추락 링스 헬기 탐색작전, 참수리 295호정 인양작전 등을 수행했으며 지난 <a class="mw-redirect" href="/wiki/2014%EB%85%84_%EC%84%B8%EC%9B%94%ED%98%B8_%EC%B0%B8%EC%82%AC" title="2014년 세월호 참사">2014년 세월호 참사</a> 당시에는 실종자 구조와 탐색작전에도 활약했다.<sup class="reference" id="cite_ref-.ED.87.B4.EC.97.AD_1-4"><a href="#cite_note-.ED.87.B4.EC.97.AD-1">[1]</a></sup></p>
<p>평택함에는 함상감압실이 있었다. 기압을 천천히 높이는 방으로, <a href="/wiki/%EC%9E%A0%EC%88%98%EB%B3%91" title="잠수병">잠수병</a>을 1차적으로 치료해 준다. 그 후 군병원으로 이송되는 방식으로 운영됐다. 이를 통해 제주 <a href="/wiki/%ED%95%B4%EB%85%80" title="해녀">해녀</a>의 <a class="mw-redirect" href="/wiki/%EA%B0%90%EC%95%95%EB%B3%91" title="감압병">감압병</a>에 치료 지원하기도 했다. 또한 연평도 폐<a href="/wiki/%EC%96%B4%EB%A7%9D" title="어망">어망</a> 수거 작전에 지속적으로 투입됐는데, 퇴역 직전인 2016년 8월에는 연평도 폐그물 수거작전에서 144톤에 달하는 역대 최대 폐그물 수거 실적을 달성했다.<sup class="reference" id="cite_ref-.ED.87.B4.EC.97.AD_1-5"><a href="#cite_note-.ED.87.B4.EC.97.AD-1">[1]</a></sup></p>
<p>평택함은 <a href="/wiki/2016%EB%85%84" title="2016년">2016년</a> <a href="/wiki/12%EC%9B%94_28%EC%9D%BC" title="12월 28일">12월 28일</a> 오후 2시, <a class="new" href="/w/index.php?title=%EC%A7%84%ED%95%B4%EA%B5%B0%ED%95%AD&amp;action=edit&amp;redlink=1" title="진해군항 (없는 문서)">진해군항</a> 서해대에서 <a class="mw-disambig" href="/wiki/%EA%B9%80%EC%A2%85%EC%9D%BC" title="김종일">김종일</a> <a class="mw-disambig" href="/wiki/%ED%95%B4%EA%B5%B0%EC%9E%91%EC%A0%84%EC%82%AC%EB%A0%B9%EB%B6%80" title="해군작전사령부">해군작전사령부</a> 부사령관(소장) 주관으로 평택함 퇴역식을 거쳐 최종 퇴역했다. 이날 퇴역식에는 역대 평택함장과 승조원, <a class="new" href="/w/index.php?title=%EC%A3%BC%ED%95%9C%EB%AF%B8%ED%95%B4%EA%B5%B0%EC%82%AC%EB%A0%B9%EB%B6%80&amp;action=edit&amp;redlink=1" title="주한미해군사령부 (없는 문서)">주한미해군사령부</a>(CNFK) 참모장 <a class="new" href="/w/index.php?title=%ED%97%A8%EB%A6%AC_%ED%82%B4&amp;action=edit&amp;redlink=1" title="헨리 킴 (없는 문서)">헨리 킴</a>(Henry Kim) 대령, <a href="/wiki/%EA%B3%B5%EC%9E%AC%EA%B4%91" title="공재광">공재광</a> <a href="/wiki/%ED%8F%89%ED%83%9D%EC%8B%9C" title="평택시">평택시</a>장, 주한 영국 무관 등 200여명이 참석했다. 평택함은 미 해군에서 도입해 마지막까지 운영한 함정이었기 때문에 앞으로 대한민국 해군은 자국 건조된 함정들로만 운영하게 되었다. 평택함은 내부 개조 과정을 거쳐 <a href="/wiki/2018%EB%85%84" title="2018년">2018년</a> 전반기 즈음에 자매결연 도시인 <a href="/wiki/%ED%8F%89%ED%83%9D%EC%8B%9C" title="평택시">평택시</a>에 양도돼 대국민 안보 체험장으로 남게 될 예정이다.<sup class="reference" id="cite_ref-.ED.87.B4.EC.97.AD_1-6"><a href="#cite_note-.ED.87.B4.EC.97.AD-1">[1]</a></sup></p>
