<h2><span class="mw-headline" id=".EC.9D.B8.EB.AC.BC">인물</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%84%B8%EC%A2%85_(%EB%8F%99%EC%9D%8C%EC%9D%B4%EC%9D%98)&amp;action=edit&amp;section=1" title="부분 편집: 인물">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<h3><span class="mw-headline" id=".ED.95.9C.EA.B5.AD">한국</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%84%B8%EC%A2%85_(%EB%8F%99%EC%9D%8C%EC%9D%B4%EC%9D%98)&amp;action=edit&amp;section=2" title="부분 편집: 한국">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<ul>
<li><b><a href="/wiki/%EB%85%B8%EB%A6%AC%EB%B6%80" title="노리부">세종</a></b>(世宗, ? ~ ?)은 신라 시대의 인물이다.</li>
<li><b><a href="/wiki/%EC%A1%B0%EC%84%A0_%EC%84%B8%EC%A2%85" title="조선 세종">세종</a></b>(世宗, 1397년 ~ 1450년)은 <a href="/wiki/%EC%A1%B0%EC%84%A0" title="조선">조선</a>의 제4대 왕으로 흔히 <b>세종대왕</b>으로 불린다. 보통 <b>세종</b>은 이 인물을 뜻한다.</li>
</ul>
<h3><span class="mw-headline" id=".EC.A4.91.EA.B5.AD">중국</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%84%B8%EC%A2%85_(%EB%8F%99%EC%9D%8C%EC%9D%B4%EC%9D%98)&amp;action=edit&amp;section=3" title="부분 편집: 중국">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<ul>
<li><b><a href="/wiki/%EC%A1%B0%EA%B0%91" title="조갑">세종</a></b>(世宗)은 <a href="/wiki/%EC%83%81%EB%82%98%EB%9D%BC" title="상나라">상나라</a>의 24대왕 정왕의 묘호이다.</li>
<li><b><a href="/wiki/%EC%A0%84%ED%95%9C_%EB%AC%B4%EC%A0%9C" title="전한 무제">세종</a></b>(世宗, 기원전 156년 ~ 기원전 87년, 재위: 기원전 141년 ~ 기원전 87년)은 <a href="/wiki/%EC%A0%84%ED%95%9C" title="전한">전한</a>의 제7대 황제 무제의 묘호이다.</li>
<li><b><a href="/wiki/%EC%82%AC%EB%A7%88%EC%82%AC" title="사마사">세종</a></b>(世宗, 208년 ~ 255년)은 <a href="/wiki/%EC%84%9C%EC%A7%84" title="서진">서진</a> <a href="/wiki/%EC%82%AC%EB%A7%88%EC%9D%98" title="사마의">선제</a>의 장남인 경제의 묘호이다.</li>
<li><b><a href="/wiki/%EC%9E%A5%EC%A4%91%ED%99%94" title="장중화">세종</a></b>(世宗, 330년 ~ 353년, 재위: 346년 ~ 353년)은 <a href="/wiki/%EC%A0%84%EB%9F%89" title="전량">전량</a>의 제5대 왕인 명왕의 묘호이다.</li>
<li><b><a class="mw-redirect" href="/wiki/%EC%9B%90%EA%B0%81" title="원각">세종</a></b>(世宗, 483년 ~ 515년, 재위: 499년 ~ 515년)는 <a href="/wiki/%EB%B6%81%EC%9C%84_(%EB%B6%81%EC%A1%B0)" title="북위 (북조)">북위</a>의 제8대 황제 선무제의 묘호이다.</li>
<li><b><a href="/wiki/%EA%B3%A0%EC%A7%95" title="고징">세종</a></b>(世宗, 521년 ~ 549년)은 <a href="/wiki/%EB%B6%81%EC%A0%9C" title="북제">북제</a>의 초대황제인 <a class="mw-redirect" href="/wiki/%EA%B3%A0%EC%96%91_(%EB%B6%81%EC%A0%9C)" title="고양 (북제)">문선제</a>의 형이자 추존 황제인 문양제의 묘호이다.</li>
<li><b><a href="/wiki/%EC%9A%B0%EB%AC%B8%EC%9C%A1" title="우문육">세종</a></b>(世宗, 534년 ~ 560년, 재위: 557년 ~ 560년)은 <a href="/wiki/%EB%B6%81%EC%A3%BC" title="북주">북주</a>의 제2대 황제 명제의 묘호이다.</li>
<li><b><a href="/wiki/%EB%9D%BC%EC%84%B1%EC%97%BC" title="라성염">세종</a></b>(世宗, 634년 ~ 712년, 재위: 674년 ~ 712년)은 <a href="/wiki/%EB%82%A8%EC%A1%B0_(%EC%99%95%EA%B5%AD)" title="남조 (왕국)">남조 (몽)</a>의 제2대 황제 흥종왕의 묘호이다.</li>
<li><b><a href="/wiki/%EC%A0%84%EC%9B%90%EA%B4%80" title="전원관">세종</a></b>(世宗, 887년 ~ 941년, 재위: 932년 ~ 941년)은 <a href="/wiki/%EC%98%A4%EC%9B%94" title="오월">오월</a>의 제2대 왕 문목왕의 묘호이다.</li>
<li><b><a class="mw-redirect" href="/wiki/%ED%9B%84%EC%A3%BC_%EC%84%B8%EC%A2%85" title="후주 세종">세종</a></b>(世宗, 921년 ~ 959년, 재위: 954년 ~ 959년)은 <a href="/wiki/%ED%9B%84%EC%A3%BC" title="후주">후주</a>의 2대 황제이다.</li>
<li><b><a href="/wiki/%EC%9A%94_%EC%84%B8%EC%A2%85" title="요 세종">세종</a></b>(世宗, 918년 ~ 951년)은 <a href="/wiki/%EC%9A%94%EB%82%98%EB%9D%BC" title="요나라">요나라</a>의 제4대 황제다.</li>
<li><b><a href="/wiki/%EA%B0%80%EC%A0%95%EC%A0%9C" title="가정제">세종</a></b>(世宗, 1507년 ~ 1567년, 재위: 1521년 ~ 1567년)은 <a href="/wiki/%EB%AA%85%EB%82%98%EB%9D%BC" title="명나라">명나라</a>의 '가정제'의 묘호이다.</li>
<li><b><a href="/wiki/%EC%98%B9%EC%A0%95%EC%A0%9C" title="옹정제">세종</a></b>(世宗, 1678년 ~ 1735년, 재위: 1722년 ~ 1735년)은 <a href="/wiki/%EC%B2%AD%EB%82%98%EB%9D%BC" title="청나라">청나라</a>의 제5대 황제 '옹정제'의 묘호이다.</li>
</ul>
<h3><span class="mw-headline" id=".EB.B2.A0.ED.8A.B8.EB.82.A8">베트남</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%84%B8%EC%A2%85_(%EB%8F%99%EC%9D%8C%EC%9D%B4%EC%9D%98)&amp;action=edit&amp;section=4" title="부분 편집: 베트남">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<ul>
<li>세종(世宗, 1567년 ~ 1599년, 재위 : 1573년 ~ 1599년)은 <b><a href="/wiki/%ED%9B%84_%EB%A0%88_%EC%99%95%EC%A1%B0" title="후 레 왕조">후 레 왕조</a></b>의 제 16대 황제 의제의 묘호이다.</li>
<li>세종(世宗, 1714년 ~ 1765년, 재위 : 1738년 ~ 1765년)은 광남국 <b><a href="/wiki/%EC%9D%91%EC%9A%B0%EC%98%8C_%EC%99%95%EC%A1%B0" title="응우옌 왕조">응우옌 왕조</a></b>의 제8대 황제 효무제의 묘호이다.</li>
</ul>
<h2><span class="mw-headline" id=".EC.A7.80.EC.97.AD">지역</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%84%B8%EC%A2%85_(%EB%8F%99%EC%9D%8C%EC%9D%B4%EC%9D%98)&amp;action=edit&amp;section=5" title="부분 편집: 지역">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<ul>
<li><b><a href="/wiki/%EC%84%B8%EC%A2%85%ED%8A%B9%EB%B3%84%EC%9E%90%EC%B9%98%EC%8B%9C" title="세종특별자치시">세종특별자치시</a></b>(世宗特別自治市)는 <a href="/wiki/%EB%8C%80%ED%95%9C%EB%AF%BC%EA%B5%AD" title="대한민국">대한민국</a>의 <a href="/wiki/%ED%8A%B9%EB%B3%84%EC%9E%90%EC%B9%98%EC%8B%9C" title="특별자치시">특별자치시</a>이다.</li>
<li><b>세종로</b>(世宗路)는 <a href="/wiki/%EC%84%9C%EC%9A%B8%ED%8A%B9%EB%B3%84%EC%8B%9C" title="서울특별시">서울특별시</a> <a href="/wiki/%EC%A2%85%EB%A1%9C%EA%B5%AC" title="종로구">종로구</a>의 <a href="/wiki/%EB%B2%95%EC%A0%95%EB%8F%99" title="법정동">법정동</a>이다.</li>

<li><b><a href="/wiki/%EC%84%B8%EC%A2%85%EA%B3%A0%EB%93%B1%ED%95%99%EA%B5%90_(%EA%B2%BD%EB%82%A8)" title="세종고등학교 (경남)">세종고등학교</a></b>는 <a href="/wiki/%EA%B2%BD%EC%83%81%EB%82%A8%EB%8F%84" title="경상남도">경상남도</a> <a href="/wiki/%EB%B0%80%EC%96%91%EC%8B%9C" title="밀양시">밀양시</a> 가곡동에 있는 사립 고등학교이다.</li>
<li><b><a href="/wiki/%EC%84%B8%EC%A2%85%EA%B3%A0%EB%93%B1%ED%95%99%EA%B5%90_(%EC%84%B8%EC%A2%85)" title="세종고등학교 (세종)">세종고등학교</a></b>는 <a href="/wiki/%EC%84%B8%EC%A2%85%ED%8A%B9%EB%B3%84%EC%9E%90%EC%B9%98%EC%8B%9C" title="세종특별자치시">세종특별자치시</a>에 있는 공립 고등학교이다.</li>
<li><b><a href="/wiki/%EC%84%B8%EC%A2%85%EA%B3%A0%EB%93%B1%ED%95%99%EA%B5%90_(%EA%B2%BD%EA%B8%B0)" title="세종고등학교 (경기)">세종고등학교</a></b>는 <a href="/wiki/%EA%B2%BD%EA%B8%B0%EB%8F%84" title="경기도">경기도</a> <a href="/wiki/%EC%97%AC%EC%A3%BC%EC%8B%9C" title="여주시">여주시</a>에 위치한 공립 고등학교이다.</li>

<li><b><a href="/wiki/%EC%84%B8%EC%A2%85%EA%B5%AD%EC%A0%9C%EA%B3%A0%EB%93%B1%ED%95%99%EA%B5%90" title="세종국제고등학교">세종국제고등학교</a></b>는 세종특별자치시 아름동 궁동에 있는 공립 국제고등학교이다.</li>
<li><b><a href="/wiki/%EC%84%9C%EC%9A%B8%EC%84%B8%EC%A2%85%EA%B3%A0%EB%93%B1%ED%95%99%EA%B5%90" title="서울세종고등학교">서울세종고등학교</a></b>는 <a href="/wiki/%EC%84%9C%EC%9A%B8%ED%8A%B9%EB%B3%84%EC%8B%9C" title="서울특별시">서울특별시</a> <a href="/wiki/%EA%B0%95%EB%82%A8%EA%B5%AC" title="강남구">강남구</a> 수서동에 있는 사립 고등학교이다.</li>
</ul>
