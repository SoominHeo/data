<h1 class="firstHeading" id="firstHeading" lang="ko">세척</h1>
<div class="mw-body-content" id="bodyContent">
<div id="siteSub">위키백과, 우리 모두의 백과사전.</div>
<div id="contentSub"></div>
<div class="mw-jump" id="jump-to-nav">
					이동:					<a href="#mw-head">둘러보기</a>, 					<a href="#p-search">검색</a>
</div>
<div class="mw-content-ltr" dir="ltr" id="mw-content-text" lang="ko"><script>function mfTempOpenSection(id){var block=document.getElementById("mf-section-"+id);block.className+=" open-block";block.previousSibling.className+=" open-block";}</script><p><b>세척</b> (洗滌)은 <a href="/wiki/%EB%AC%BC" title="물">물</a>이나 <a href="/wiki/%EC%84%B8%EC%A0%9C" title="세제">세제</a> 등의 <a class="new" href="/w/index.php?title=%EC%84%B8%EC%B2%99%EC%95%A1&amp;action=edit&amp;redlink=1" title="세척액 (없는 문서)">세척액</a>을 이용해 더러움을 없애는 행위.</p>
<p><a href="/wiki/%EC%8B%A4%ED%97%98" title="실험">실험</a>에 이용한 <a class="new" href="/w/index.php?title=%EC%9C%A0%EB%A6%AC_%EA%B8%B0%EA%B5%AC&amp;action=edit&amp;redlink=1" title="유리 기구 (없는 문서)">유리 기구</a>는 보통 <a class="new" href="/w/index.php?title=%ED%81%B4%EB%A6%B0%EC%A0%80&amp;action=edit&amp;redlink=1" title="클린저 (없는 문서)">클린저</a>를 붙이고 브러쉬로 비비어 세척한다. 더러움이 심한 경우에는 드라이아이스 세척기나 <a class="new" href="/w/index.php?title=%EC%B4%88%EC%9D%8C%ED%8C%8C_%EC%84%B8%EC%B2%99%EA%B8%B0&amp;action=edit&amp;redlink=1" title="초음파 세척기 (없는 문서)">초음파 세척기</a>, <a href="/wiki/%EC%88%98%EC%82%B0%ED%99%94_%EB%82%98%ED%8A%B8%EB%A5%A8" title="수산화 나트륨">수산화 나트륨</a>이나 <a href="/wiki/%EC%88%98%EC%82%B0%ED%99%94_%EC%B9%BC%EB%A5%A8" title="수산화 칼륨">수산화 칼륨</a>을 <a href="/wiki/%EC%95%84%EC%9D%B4%EC%86%8C%ED%94%84%EB%A1%9C%ED%95%84_%EC%95%8C%EC%BD%94%EC%98%AC" title="아이소프로필 알코올">아이소프로필 알코올</a>에 녹인 알칼리 버스를 이용한다. 옛날에는 <a class="new" href="/w/index.php?title=%ED%81%AC%EB%A1%AC%EC%82%B0%EC%82%B0%ED%99%94&amp;action=edit&amp;redlink=1" title="크롬산산화 (없는 문서)">크롬산혼액</a>을 사용하기도 했지만, 사용하는 <a class="new" href="/w/index.php?title=2_%ED%81%AC%EB%A1%AC%EC%82%B0%EC%B9%BC%EB%A5%A8&amp;action=edit&amp;redlink=1" title="2 크롬산칼륨 (없는 문서)">2 크롬산칼륨</a>이나 <a class="new" href="/w/index.php?title=2_%ED%81%AC%EB%A1%AC%EC%82%B0%EB%82%98%ED%8A%B8%EB%A5%A8&amp;action=edit&amp;redlink=1" title="2 크롬산나트륨 (없는 문서)">2 크롬산나트륨</a>이 육가 크롬이며, 환경에의 부하가 높기 때문에 폐수 처리 등의 문제로 현재는 거의 이용되지 않는다. 더러움이 완전하게 떨어진 일은 유리 표면이 물을 튕기지 않게 되는 일로 확인할 수 있다.</p>
<p><a href="/wiki/%EB%B0%98%EB%8F%84%EC%B2%B4" title="반도체">반도체</a> 부품을 생산하는 데에 <a class="mw-redirect" href="/wiki/%EC%8B%A4%EB%A6%AC%EC%BD%98_%EC%9B%A8%EC%9D%B4%ED%8D%BC" title="실리콘 웨이퍼">실리콘 웨이퍼</a>의 표면을 세척할 때에 이전에는 <a class="mw-redirect" href="/wiki/%ED%94%84%EB%A0%88%EC%98%A8" title="프레온">프레온</a>이 세척액으로서 사용되고 있었지만, 프레온은 <a href="/wiki/%EC%98%A4%EC%A1%B4%EC%B8%B5" title="오존층">오존층</a>을 파괴하는 원인으로 여겨졌기 때문에 이래 <a class="new" href="/w/index.php?title=%EC%88%9C%EC%88%98%ED%95%9C_%EB%AC%BC&amp;action=edit&amp;redlink=1" title="순수한 물 (없는 문서)">순수한 물</a>이 사용되게 되었다.</p>
<p>공업계의 세척에서는 드라이아이스 펠릿을 사용한 <a class="new" href="/w/index.php?title=%EB%93%9C%EB%9D%BC%EC%9D%B4%EC%95%84%EC%9D%B4%EC%8A%A4_%EC%84%B8%EC%B2%99&amp;action=edit&amp;redlink=1" title="드라이아이스 세척 (없는 문서)">드라이아이스 세척</a>이 폐수를 내지 않는 세척 방법으로서 환경적으로도 주목되어 오고 있다. 특히 <a class="new" href="/w/index.php?title=%EC%9E%90%EB%8F%99%EC%B0%A8_%EC%82%B0%EC%97%85&amp;action=edit&amp;redlink=1" title="자동차 산업 (없는 문서)">자동차 산업</a>을 중심으로 한 기름의 세척이나, 도료의 세척, 플라스틱 성형 금형·고무 성형 금형 등의 세척에도 많이 사용되고 있다.</p>
<p><a href="/wiki/%EC%8B%9D%EA%B8%B0" title="식기">식기</a>나 <a class="new" href="/w/index.php?title=%EC%9D%BC%EC%9A%A9%ED%92%88&amp;action=edit&amp;redlink=1" title="일용품 (없는 문서)">일용품</a>의 세척에는 <a href="/wiki/%EC%8A%A4%ED%8E%80%EC%A7%80" title="스펀지">스펀지</a>나 <a class="mw-redirect" href="/wiki/%EB%B8%8C%EB%9F%AC%EC%89%AC" title="브러쉬">브러쉬</a>, <a class="mw-disambig" href="/wiki/%EC%88%98%EC%84%B8%EB%AF%B8" title="수세미">수세미</a>가 편리하다.</p>
