<p><b>세포질</b>(細胞質, <span style="font-size: smaller;"><a href="/wiki/%EC%98%81%EC%96%B4" title="영어">영어</a>:</span> <span lang="en" xml:lang="en">cytoplasm</span>)은 <a href="/wiki/%EC%84%B8%ED%8F%AC" title="세포">세포</a> 내부를 채우고 있는 균일하고, 일반적으로는 투명한 점액 형태의 물질이다. 세포질은 <a class="mw-redirect" href="/wiki/%EC%84%B8%ED%8F%AC%ED%95%B5" title="세포핵">세포핵</a>을 제외한 <a class="new" href="/w/index.php?title=%EC%84%B8%ED%8F%AC%EC%95%A1&amp;action=edit&amp;redlink=1" title="세포액 (없는 문서)">세포액</a>과 <a href="/wiki/%EC%84%B8%ED%8F%AC%EC%86%8C%EA%B8%B0%EA%B4%80" title="세포소기관">세포소기관</a>으로 이루어진다. 세포액은 수분, 염분, 세포소기관을 이루는 분자, 그리고 반응 촉매로 작용하는 <a href="/wiki/%ED%9A%A8%EC%86%8C" title="효소">효소</a>로 구성된다. 세포질은 세포에서 중요한 역할을 하는데, <a href="/wiki/%EC%84%B8%ED%8F%AC%EB%A7%89" title="세포막">세포막</a>으로 둘러 싸여 있는 이른바 "분자 수프"를 구성하며, 그 속에서 세포핵 및 세포소기관이 서로 뭉쳐있을 수 있게 해준다.</p>
<h2><span class="mw-headline" id=".EC.84.B8.ED.8F.AC.EC.A7.88.EC.9D.98_.EA.B5.AC.EC.84.B1">세포질의 구성</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%84%B8%ED%8F%AC%EC%A7%88&amp;action=edit&amp;section=1" title="부분 편집: 세포질의 구성">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>세포질의 최대 80%까지 차지하는 액체 성분은 <a href="/wiki/%EC%9D%B4%EC%98%A8" title="이온">이온</a> 및 용해되어 있는 <a href="/wiki/%ED%9A%A8%EC%86%8C" title="효소">효소</a>, <a href="/wiki/%ED%83%84%EC%88%98%ED%99%94%EB%AC%BC" title="탄수화물">탄수화물</a>, <a href="/wiki/%EC%97%BC" title="염">염</a>, <a href="/wiki/%EB%8B%A8%EB%B0%B1%EC%A7%88" title="단백질">단백질</a>, <a href="/wiki/RNA" title="RNA">RNA</a>와 같은 <a class="new" href="/w/index.php?title=%EA%B1%B0%EB%8C%80_%EB%B6%84%EC%9E%90&amp;action=edit&amp;redlink=1" title="거대 분자 (없는 문서)">거대 분자</a>로 구성되어 있다. 세포질의 액체 성분은 <a class="new" href="/w/index.php?title=%ED%88%AC%EB%AA%85%EC%A7%88&amp;action=edit&amp;redlink=1" title="투명질 (없는 문서)">투명질</a>이라고도 불린다.</p>
<p>액체 성분은 주변 상태나 <a href="/wiki/%EC%84%B8%ED%8F%AC" title="세포">세포</a>의 활동 상태에 따라 <a href="/wiki/%EC%A0%A4" title="젤">젤</a> 형태나 액체 형태일 수 있다. 젤 상태일 경우, <a class="new" href="/w/index.php?title=%EC%84%B8%ED%8F%AC_%EC%A0%A4&amp;action=edit&amp;redlink=1" title="세포 젤 (없는 문서)">세포 젤</a>이라고 불리며, 점성의 단단한 덩어리이다. 액체 상태일 경우는 <a class="new" href="/w/index.php?title=%EC%84%B8%ED%8F%AC%EC%95%A1&amp;action=edit&amp;redlink=1" title="세포액 (없는 문서)">세포액</a>이라고 불리며, 유동성 있는 액체이다. 일반적으로 세포의 가장자리는 젤과 비슷한 상태이며, 세포 내부는 액체 상태이다.</p>
<p>세포질 내부의 비용해성 구성 요소로는 <a href="/wiki/%EB%AF%B8%ED%86%A0%EC%BD%98%EB%93%9C%EB%A6%AC%EC%95%84" title="미토콘드리아">미토콘드리아</a>, <a href="/wiki/%EC%97%BD%EB%A1%9D%EC%B2%B4" title="엽록체">엽록체</a>, <a href="/wiki/%EB%A6%AC%EB%B3%B4%EC%86%9C" title="리보솜">리보솜</a>, <a class="new" href="/w/index.php?title=%EA%B3%BC%EC%82%B0%ED%99%94%EC%86%8C%EC%B2%B4&amp;action=edit&amp;redlink=1" title="과산화소체 (없는 문서)">과산화소체</a>, <a href="/wiki/%EB%A6%AC%EB%B3%B4%EC%86%9C" title="리보솜">리보솜</a>과 같은 <a href="/wiki/%EC%84%B8%ED%8F%AC%EC%86%8C%EA%B8%B0%EA%B4%80" title="세포소기관">세포소기관</a> 및 일부 <a href="/wiki/%EC%95%A1%ED%8F%AC" title="액포">액포</a>, <a href="/wiki/%EC%84%B8%ED%8F%AC%EA%B3%A8%EA%B2%A9" title="세포골격">세포골격</a> 등이 있으며, <a href="/wiki/%EC%86%8C%ED%8F%AC%EC%B2%B4" title="소포체">소포체</a>나 <a href="/wiki/%EA%B3%A8%EC%A7%80%EC%B2%B4" title="골지체">골지체</a>와 같은 복잡한 <a href="/wiki/%EC%84%B8%ED%8F%AC%EB%A7%89" title="세포막">세포막</a> 구조 역시 구성 요소의 하나이다.</p>
<h2><span class="mw-headline" id=".EC.84.B8.ED.8F.AC.EC.97.90_.EB.94.B0.EB.A5.B8_.EC.84.B8.ED.8F.AC.EC.A7.88.EC.9D.98_.EC.B0.A8.EC.9D.B4">세포에 따른 세포질의 차이</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%84%B8%ED%8F%AC%EC%A7%88&amp;action=edit&amp;section=2" title="부분 편집: 세포에 따른 세포질의 차이">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>모든 <a href="/wiki/%EC%84%B8%ED%8F%AC" title="세포">세포</a>는 세포질을 가지고 있지만, <a href="/wiki/%EC%83%9D%EB%AC%BC_%EB%B6%84%EB%A5%98" title="생물 분류">생물 분류</a>에 따라 그 특성은 매우 다르다. <a href="/wiki/%EB%8F%99%EB%AC%BC" title="동물">동물</a> 세포에서는 세포질이 거의 세포 부피의 절반을 차지하지만, <a href="/wiki/%EC%8B%9D%EB%AC%BC" title="식물">식물</a> 세포에서는 <a href="/wiki/%EC%95%A1%ED%8F%AC" title="액포">액포</a>의 존재로 인해 세포질이 훨씬 적은 부피를 차지한다.</p>
<h2><span class="mw-headline" id=".EA.B8.B0.EB.8A.A5">기능</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%84%B8%ED%8F%AC%EC%A7%88&amp;action=edit&amp;section=3" title="부분 편집: 기능">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>세포질은 세포의 모양 및 항상성을 유지하며, 세포소기관을 지탱해주는 역할을 한다. 또한 대개 <a href="/wiki/%EB%AC%BC%EC%A7%88%EB%8C%80%EC%82%AC" title="물질대사">물질대사</a>에 사용되면서 생명 유지에 반드시 필요한 화학 물질을 저장하는 장소이기도 하다.</p>
