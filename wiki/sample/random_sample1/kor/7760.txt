<p><b>알파 센타우리</b> 또는 <b>알파 켄타우리</b> 또는 <b>센타우루스자리 알파</b>(α Centauri, 또는 α Cen)는 <a href="/wiki/%EC%84%BC%ED%83%80%EC%9A%B0%EB%A3%A8%EC%8A%A4%EC%9E%90%EB%A6%AC" title="센타우루스자리">센타우루스자리</a>에서 가장 밝은 별로, <a class="mw-redirect" href="/wiki/%EC%A3%BC%EA%B3%84%EC%97%B4%EC%84%B1" title="주계열성">주계열성</a>이다. <a class="mw-redirect" href="/wiki/%EC%8B%A4%EC%8B%9C%EB%93%B1%EA%B8%89" title="실시등급">실시등급</a>(實視等級)은 -0.01등성으로, <a href="/wiki/%EC%B2%9C%EA%B5%AC" title="천구">천구</a>에서 <a class="mw-redirect" href="/wiki/%EA%B0%80%EC%9E%A5_%EB%B0%9D%EC%9D%80_%EB%B3%84_%EB%AA%A9%EB%A1%9D" title="가장 밝은 별 목록">네 번째로 밝은 별</a>이다.</p>
<p>육안(肉眼)으로는 하나로 보이지만 사실 이 별은 <a class="mw-redirect" href="/wiki/%EC%8C%8D%EC%84%B1%EA%B3%84" title="쌍성계">쌍성계</a>(雙星系)로, <a href="/wiki/%EC%9C%A0%EC%82%AC_%ED%83%9C%EC%96%91" title="유사 태양">태양과 매우 비슷한</a> 센타우루스자리 알파 <b>A</b>, <a class="mw-redirect" href="/wiki/%EC%98%A4%EB%A0%8C%EC%A7%80%EC%83%89_%EC%99%9C%EC%84%B1" title="오렌지색 왜성">오렌지색 왜성</a>으로 태양보다 좀 더 가볍고 차가운 센타우루스자리 알파 <b>B</b>로 이루어져 있다. 밤하늘에서 이들과 조금 떨어진 곳에 <a class="mw-redirect" href="/wiki/%EC%A0%81%EC%83%89_%EC%99%9C%EC%84%B1" title="적색 왜성">적색 왜성</a> <b><a href="/wiki/%EC%84%BC%ED%83%80%EC%9A%B0%EB%A3%A8%EC%8A%A4%EC%9E%90%EB%A6%AC_%ED%94%84%EB%A1%9D%EC%8B%9C%EB%A7%88" title="센타우루스자리 프록시마">센타우루스자리 프록시마</a></b>가 있는데, 센타우루스자리 알파 항성계와 중력으로 묶여 있는지 그렇지 않은지에 대해 논란이 있다.</p>
<p>센타우루스자리 알파 A,B는 태양에서 가장 가까운 <a href="/wiki/%ED%95%AD%EC%84%B1%EA%B3%84" title="항성계">항성계</a>로, 4.37광년(光年) 떨어져 있다. 센타우루스자리 프록시마는 태양에서 가장 가까운 별로, 4.22광년(光年) 떨어져 있다.</p>
<p>상대적으로 가까운 거리 때문에 센타우루스자리 알파는 <a class="new" href="/w/index.php?title=%EC%84%B1%EA%B0%84%EC%97%AC%ED%96%89&amp;action=edit&amp;redlink=1" title="성간여행 (없는 문서)">성간여행</a>(星間旅行)을 소재로 한 <a class="mw-redirect" href="/wiki/%EA%B3%BC%ED%95%99%EC%86%8C%EC%84%A4" title="과학소설">과학소설</a> 및 <a href="/wiki/%EB%B9%84%EB%94%94%EC%98%A4_%EA%B2%8C%EC%9E%84" title="비디오 게임">비디오 게임</a>들의 소재로 쓰여 왔다. 센타우루스자리 알파는 인류가 성간여행을 현실화할 경우 가장 먼저 방문할 후보들 중 하나이다.</p>

<h2><span class="mw-headline" id=".EB.AA.85.EC.B9.AD">명칭</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%84%BC%ED%83%80%EC%9A%B0%EB%A3%A8%EC%8A%A4%EC%9E%90%EB%A6%AC_%EC%95%8C%ED%8C%8C&amp;action=edit&amp;section=1" title="부분 편집: 명칭">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<h3><span class="mw-headline" id=".ED.95.99.EB.AA.85">학명</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%84%BC%ED%83%80%EC%9A%B0%EB%A3%A8%EC%8A%A4%EC%9E%90%EB%A6%AC_%EC%95%8C%ED%8C%8C&amp;action=edit&amp;section=2" title="부분 편집: 학명">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>"센타우루스자리 알파"("리길 켄타우루스")는 맨눈으로 볼 때 센타우루스자리에서 하나의 발광체(發光體)처럼 보이는 존재에 붙은 이름이다. <a href="/wiki/%EB%A7%9D%EC%9B%90%EA%B2%BD" title="망원경">망원경</a>의 도움을 받으면 알파는 가까이 붙어 돌고 있는 <a href="/wiki/%EC%8C%8D%EC%84%B1" title="쌍성">쌍성</a>으로 분리되어 보이게 되는데, 쌍성이라는 의미에서 붙은 이름은 "센타우루스자리 알파 AB 항성계" 또는 "센타우루스자리 α"이며, 줄여서 "α Cen AB"로 표기한다.</p>
<p>"센타우루스자리 알파 A(α Cen A)"와 "센타우루스자리 알파 B(α Cen B)"는 쌍성계 내 독립적인 별들로, 하나가 아닌 두 개체임을 표시하기 위해 A와 B로 구별하여 표시한다. 지구에서 볼 때 밤하늘에서 A와 B로부터 2.18도 떨어진 곳에 동반 천체 <a href="/wiki/%EC%84%BC%ED%83%80%EC%9A%B0%EB%A3%A8%EC%8A%A4%EC%9E%90%EB%A6%AC_%ED%94%84%EB%A1%9D%EC%8B%9C%EB%A7%88" title="센타우루스자리 프록시마">센타우루스자리 프록시마</a>가 있다(줄여서 "프록시마" 또는 "센타우루스자리 알파 C"로 쓴다). 만약 프록시마가 맨눈으로 보일 정도로 밝은 별이었다면 센타우루스자리 프록시마는 밤하늘에서 AB와 분리된 하나의 별처럼 보였을 것이다. 센타우루스자리 알파 AB와 프록시마는 <a href="/wiki/%EC%9D%B4%EC%A4%91%EC%84%B1" title="이중성">안시 이중성</a>을 구성하며, 실제로 중력으로 서로 묶여 있을 것으로 추측된다. 그러나 프록시마가 보편적인 <a href="/wiki/%EC%8C%8D%EC%84%B1" title="쌍성">쌍성</a>들이 보여주는 타원 궤도를 그리면서 AB를 돌고 있는지의 직접적 증거는 아직 발견되지 않았다.<sup class="reference" id="cite_ref-wds2006_1-0"><a href="#cite_note-wds2006-1">[1]</a></sup></p>
<p>세 별이 함께 <a href="/wiki/%ED%95%AD%EC%84%B1%EA%B3%84" title="항성계">삼중성계</a>를 구성한다고 해석할 경우 "센타우루스자리 알파 AB-C"로 표기할 수 있다.</p>
<h3><span class="mw-headline" id=".EC.95.A0.EC.B9.AD">애칭</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%84%BC%ED%83%80%EC%9A%B0%EB%A3%A8%EC%8A%A4%EC%9E%90%EB%A6%AC_%EC%95%8C%ED%8C%8C&amp;action=edit&amp;section=3" title="부분 편집: 애칭">편집</a><span class="mw-editsection-bracket">]</span></span></h3>

<p>센타우루스자리 알파는 여러 가지 고유 명칭을 갖고 있다. 대표적인 것들은 <b>리길 켄타우루스</b><sup class="reference" id="cite_ref-2"><a href="#cite_note-2">[2]</a></sup>(종종 줄여서 <b>리길 켄트</b><sup class="reference" id="cite_ref-K.26S_3-0"><a href="#cite_note-K.26S-3">[3]</a></sup> 로도 표기한다), <b>리글 켄타우루스</b><sup class="reference" id="cite_ref-4"><a href="#cite_note-4">[4]</a></sup><sup class="reference" id="cite_ref-5"><a href="#cite_note-5">[5]</a></sup>, <a href="/wiki/%ED%8F%AC%EB%A5%B4%ED%88%AC%EA%B0%88%EC%96%B4" title="포르투갈어">포르투갈어</a>로 <b>리구엘 켄타우루스</b><sup class="reference" id="cite_ref-6"><a href="#cite_note-6">[6]</a></sup> 라고 부른다. 이들의 어원은 <a href="/wiki/%EC%95%84%EB%9E%8D%EC%96%B4" title="아랍어">아랍어</a> 문구 <i>Rijl Qan<u>t</u>ūris</i><sup class="reference" id="cite_ref-K.26S_3-1"><a href="#cite_note-K.26S-3">[3]</a></sup>(또는 '<a href="/wiki/%EC%BC%84%ED%83%80%EC%9A%B0%EB%A1%9C%EC%8A%A4" title="켄타우로스">켄타우로스</a>의 다리'를 뜻하는 <i>Rijl al-Qan<u>t</u>ūris</i>,<sup class="reference" id="cite_ref-7"><a href="#cite_note-7">[7]</a></sup>)에서 왔다. 그러나 가장 자주 사용되는 명칭은 <a href="/wiki/%EB%B0%94%EC%9D%B4%EC%96%B4_%EB%AA%85%EB%AA%85%EB%B2%95" title="바이어 명명법">바이어 명명법</a>의 발음인 <b>알파 센타우리</b>이다. 또 다른 이름으로 <b>톨리만</b>이 있는데 이 단어의 어원은 <a href="/wiki/%ED%83%80%EC%A1%B0" title="타조">타조</a>를 뜻하는 <a href="/wiki/%EC%95%84%EB%9E%8D%EC%96%B4" title="아랍어">아랍어</a> <i>al-<u>Z</u>ulmān</i><sup class="reference" id="cite_ref-K.26S_3-2"><a href="#cite_note-K.26S-3">[3]</a></sup> 에서 왔다. 가끔씩 불리는 다른 이름으로는 <b>분굴라</b><sup class="reference" id="cite_ref-8"><a href="#cite_note-8">[8]</a></sup> 가 있는데, 이 말은 <a href="/wiki/%EB%9D%BC%ED%8B%B4%EC%96%B4" title="라틴어">라틴어</a>로 발굽을 뜻하는 운굴라에 "β"를 합쳐서 생긴 단어로 보인다.<sup class="reference" id="cite_ref-K.26S_3-3"><a href="#cite_note-K.26S-3">[3]</a></sup> 그러나 분굴라는 현대에 들어와서는 거의 사용되지 않는다.</p>
<p><a href="/wiki/%EC%A4%91%EA%B5%AD%EC%96%B4" title="중국어">중국어</a>로는 센타우루스자리 알파를 난먼얼(南門二)이라 하며, 이는 '남쪽 문의 두 번째 별'이라는 뜻이다. 이름의 뜻대로, 센타우루스자리 알파와 <a class="mw-redirect" href="/wiki/%EC%84%BC%ED%83%80%EC%9A%B0%EB%A3%A8%EC%8A%A4%EC%9E%90%EB%A6%AC_%EB%B2%A0%ED%83%80" title="센타우루스자리 베타">베타</a>는 함께 <a href="/wiki/%EB%82%A8%EC%8B%AD%EC%9E%90%EC%9E%90%EB%A6%AC" title="남십자자리">남십자자리</a>를 가리키는 '남쪽 지시자' 역할을 한다.</p>
<h2><span class="mw-headline" id=".ED.95.AD.EC.84.B1.EA.B3.84">항성계</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%84%BC%ED%83%80%EC%9A%B0%EB%A3%A8%EC%8A%A4%EC%9E%90%EB%A6%AC_%EC%95%8C%ED%8C%8C&amp;action=edit&amp;section=4" title="부분 편집: 항성계">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<h3><span class="mw-headline" id=".EC.84.BC.ED.83.80.EC.9A.B0.EB.A3.A8.EC.8A.A4.EC.9E.90.EB.A6.AC_.EC.95.8C.ED.8C.8C_A.2C_B">센타우루스자리 알파 A, B</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%84%BC%ED%83%80%EC%9A%B0%EB%A3%A8%EC%8A%A4%EC%9E%90%EB%A6%AC_%EC%95%8C%ED%8C%8C&amp;action=edit&amp;section=5" title="부분 편집: 센타우루스자리 알파 A, B">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<div class="thumb tleft">

<p>센타우루스자리 알파 A, B 둘은 하나의 발광체로서 -0.27의 밝기로 빛나는데 이는 <a href="/wiki/%EC%8B%9C%EB%A6%AC%EC%9A%B0%EC%8A%A4" title="시리우스">시리우스</a>나 <a href="/wiki/%EC%B9%B4%EB%85%B8%ED%91%B8%EC%8A%A4" title="카노푸스">카노푸스</a>보다 어두운 값이다.<sup class="reference" id="cite_ref-9"><a href="#cite_note-9">[9]</a></sup> 밤하늘에서 센타우루스자리 알파 다음으로 밝은 별은 <a class="mw-redirect" href="/wiki/%EC%95%84%ED%81%AC%ED%88%AC%EB%A5%B4%EC%8A%A4" title="아크투르스">아크투르스</a>이다. 구성원 둘을 분리할 경우, A의 밝기는 + 0.01로 밤하늘에서 네 번째, B는 + 1.33으로 밤하늘에서 21번째로 밝다.<sup class="reference" id="cite_ref-RECONS_10-0"><a href="#cite_note-RECONS-10">[10]</a></sup></p>
<p><a class="mw-redirect" href="/wiki/%EC%A3%BC%EC%84%B1" title="주성">주성</a>(主星)인 센타우루스자리 알파 A는 표면온도 5800<a href="/wiki/%EC%BC%88%EB%B9%88" title="켈빈">K</a>에 <a class="mw-redirect" href="/wiki/%ED%95%AD%EC%84%B1_%EB%B6%84%EB%A5%98#.EB.B6.84.EA.B4.91.ED.98.95" title="항성 분류">분광형</a>은 <a href="/wiki/%ED%83%9C%EC%96%91" title="태양">태양</a>과 같은 <a class="mw-redirect" href="/wiki/G2V" title="G2V">G2V</a>인 주계열성으로, <a href="/wiki/%EC%A7%80%EB%A6%84" title="지름">지름</a>과 <a href="/wiki/%EA%B4%91%EB%8F%84" title="광도">광도</a> 모두 태양보다 조금씩 크다.<sup class="reference" id="cite_ref-RECONS_10-1"><a href="#cite_note-RECONS-10">[10]</a></sup> 값이 밝혀진 상호 궤도 요소로부터 A의 질량이 태양보다 10퍼센트 더 크며 반지름은 23퍼센트 큼을 알 수 있다.<sup class="reference" id="cite_ref-eso_11-0"><a href="#cite_note-eso-11">[11]</a></sup> A의 1 자전 주기가 41일이므로, 자전 속도는 1.1±0.8 km·s<sup>−1</sup>가 나온다.<sup class="reference" id="cite_ref-aaa470_12-0"><a href="#cite_note-aaa470-12">[12]</a></sup> (예전에 알려졌던 A의 1 자전 주기는 36.8일이었다)<sup class="reference" id="cite_ref-iau6259_13-0"><a href="#cite_note-iau6259-13">[13]</a></sup></p>
<p>제1<a class="mw-redirect" href="/wiki/%EB%B0%98%EC%84%B1" title="반성">반성</a>(伴星)인 센타우루스자리 알파 B는 표면온도 5300K에 분광형이 <a class="mw-redirect" href="/wiki/K0V" title="K0V">K0V</a>인 주계열성으로, 지름은 태양의 80%, 광도는 태양의 50% 정도이다. B는 주성에 비해 어둡지만, 스펙트럼상 A보다 더 많은 <a href="/wiki/%EC%97%91%EC%8A%A4%EC%84%A0" title="엑스선">엑스선</a>을 뿜는다. B의 광도곡선은 단기적으로 불규칙하며 지금까지 최소 한 번의 플레어 현상이 관측되었다.<sup class="reference" id="cite_ref-14"><a href="#cite_note-14">[14]</a></sup></p>
<p>A와 B의 최단거리는 11<a class="mw-redirect" href="/wiki/%EC%B2%9C%EB%AC%B8_%EB%8B%A8%EC%9C%84" title="천문 단위">AU</a>(또는 16억 7천만 킬로미터)로 토성과 태양 사이의 거리 정도에 해당되며, 가장 멀리 떨어질 경우 36<a class="mw-redirect" href="/wiki/%EC%B2%9C%EB%AC%B8_%EB%8B%A8%EC%9C%84" title="천문 단위">AU</a>로, 이는 태양과 명왕성 사이의 거리와 비슷하다.<sup class="reference" id="cite_ref-SixthCatOrbVisBin_15-0"><a href="#cite_note-SixthCatOrbVisBin-15">[15]</a></sup><sup class="reference" id="cite_ref-16"><a href="#cite_note-16">[16]</a></sup> 둘은 서로의 <a href="/wiki/%EC%A7%88%EB%9F%89_%EC%A4%91%EC%8B%AC" title="질량 중심">질량 중심</a>을 기준으로 79.91년을 1주기로<sup class="reference" id="cite_ref-SixthCatOrbVisBin_15-1"><a href="#cite_note-SixthCatOrbVisBin-15">[15]</a></sup> <a href="/wiki/%EA%B3%B5%EC%A0%84" title="공전">공전</a>한다. 궤도 요소를 통해 A와 B의 질량을 합치면 태양의 2.0 배인 것을 알 수 있다.(<span><span class="mwe-math-mathml-inline mwe-math-mathml-a11y mw-math-element" style="display: none;"><math xmlns="http://www.w3.org/1998/Math/MathML">

<h3><span class="mw-headline" id=".EC.84.BC.ED.83.80.EC.9A.B0.EB.A3.A8.EC.8A.A4.EC.9E.90.EB.A6.AC_.ED.94.84.EB.A1.9D.EC.8B.9C.EB.A7.88">센타우루스자리 프록시마</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%84%BC%ED%83%80%EC%9A%B0%EB%A3%A8%EC%8A%A4%EC%9E%90%EB%A6%AC_%EC%95%8C%ED%8C%8C&amp;action=edit&amp;section=6" title="부분 편집: 센타우루스자리 프록시마">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<div class="rellink relarticle mainarticle"><img alt="" data-file-height="128" data-file-width="128" height="16" src="//upload.wikimedia.org/wikipedia/commons/thumb/e/ec/Crystal_Clear_app_xmag.svg/16px-Crystal_Clear_app_xmag.svg.png" srcset="//upload.wikimedia.org/wikipedia/commons/thumb/e/ec/Crystal_Clear_app_xmag.svg/24px-Crystal_Clear_app_xmag.svg.png 1.5x, //upload.wikimedia.org/wikipedia/commons/thumb/e/ec/Crystal_Clear_app_xmag.svg/32px-Crystal_Clear_app_xmag.svg.png 2x" width="16"/> 이 부분의 본문은 <a href="/wiki/%EC%84%BC%ED%83%80%EC%9A%B0%EB%A3%A8%EC%8A%A4%EC%9E%90%EB%A6%AC_%ED%94%84%EB%A1%9D%EC%8B%9C%EB%A7%88" title="센타우루스자리 프록시마">센타우루스자리 프록시마</a>입니다.</div>
<p>센타우루스자리 알파 A의 제2반성(伴星)인 센타우루스자리 프록시마는 <a class="mw-redirect" href="/wiki/%EC%A0%81%EC%83%89_%EC%99%9C%EC%84%B1" title="적색 왜성">적색 왜성</a>으로, 표면온도 2700K에 분광형이 <a class="mw-redirect" href="/wiki/M5.5Ve" title="M5.5Ve">M5.5Ve</a>, <a href="/wiki/%EC%A0%88%EB%8C%80%EB%93%B1%EA%B8%89" title="절대등급">절대등급</a>은 15.53으로 <a href="/wiki/%ED%83%9C%EC%96%91" title="태양">태양</a>과 비교하면 훨씬 더 어둡다. 질량은 약 0.123±0.06 <i>M</i><sub>☉</sub>(반올림하여 0.12 <i>M</i><sub>☉</sub>) 또는 태양의 약 8분의 1이다.<sup class="reference" id="cite_ref-aaa397_23-0"><a href="#cite_note-aaa397-23">[23]</a></sup></p>
<p>센타우루스자리 프록시마와 센타우루스자리 알파 A/B와의 거리는 약 13,000AU(약 0.2광년, 1조 9400억 킬로미터)이며<sup class="reference" id="cite_ref-adsabs.harvard.edu_19-1"><a href="#cite_note-adsabs.harvard.edu-19">[19]</a></sup><sup class="reference" id="cite_ref-Matt93_22-1"><a href="#cite_note-Matt93-22">[22]</a></sup><sup class="reference" id="cite_ref-DoubleStarsHeintz_24-0"><a href="#cite_note-DoubleStarsHeintz-24">[24]</a></sup>, <a href="/wiki/%EC%B2%9C%EA%B5%AC" title="천구">천구</a>(天球) 상에서는 약 2도 정도 떨어져 있다. 이 거리는 태양과 센타우루스자리 알파 사이 거리의 약 5퍼센트에 해당한다. 이 별은 태양에서 4.22광년 떨어져 있어, <a href="/wiki/%ED%83%9C%EC%96%91%EA%B3%84" title="태양계">태양계</a>에서 가장 가까운데, 프록시마라는 뜻은 「가장 가깝다」는 뜻이다.</p>
<p>프록시마는 센타우루스자리 알파 A, B와 중력적으로 묶여 있을 가능성이 있으며, 만약 그럴 경우 공전 주기는 약 10만 년에서 50만 년 사이가 된다.<sup class="reference" id="cite_ref-Matt93_22-2"><a href="#cite_note-Matt93-22">[22]</a></sup> 그러나 프록시마는 중력적으로 상관이 없는 독립된 천체이며, 단지 알파 항성계 근처를 <a href="/wiki/%ED%8F%AC%EB%AC%BC%EC%84%A0" title="포물선">포물선</a> 궤도를 그리면서<sup class="reference" id="cite_ref-25"><a href="#cite_note-25">[25]</a></sup> 지나가는 중일 가능성도 있다.<sup class="reference" id="cite_ref-DoubleStarsHeintz_24-1"><a href="#cite_note-DoubleStarsHeintz-24">[24]</a></sup> 프록시마가 중력으로 묶여 있다는 중요한 증거로, 이들은 <a class="mw-redirect" href="/wiki/%EA%B3%A0%EC%9C%A0_%EC%9A%B4%EB%8F%99" title="고유 운동">고유 운동</a>을 공유하고 있다는 사실이 있다. 이는 우연히 발생할 가능성이 별로 없다.<sup class="reference" id="cite_ref-Matt93_22-3"><a href="#cite_note-Matt93-22">[22]</a></sup> 이론적으로 프록시마는 알파 항성계를 수백만 년 후 탈출할 가능성이 있다.<sup class="reference" id="cite_ref-Matthews_26-0"><a href="#cite_note-Matthews-26">[26]</a></sup> 아직까지 프록시마와 알파 항성계가 서로 중력적으로 묶여 있는지 검증되지는 않았다.<sup class="reference" id="cite_ref-27"><a href="#cite_note-27">[27]</a></sup></p>

<h3><span class="mw-headline" id=".EC.84.BC.ED.83.80.EC.9A.B0.EB.A3.A8.EC.8A.A4.EC.9E.90.EB.A6.AC_.EC.95.8C.ED.8C.8C_Bb">센타우루스자리 알파 Bb</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%84%BC%ED%83%80%EC%9A%B0%EB%A3%A8%EC%8A%A4%EC%9E%90%EB%A6%AC_%EC%95%8C%ED%8C%8C&amp;action=edit&amp;section=8" title="부분 편집: 센타우루스자리 알파 Bb">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<div class="rellink relarticle mainarticle"><img alt="" data-file-height="128" data-file-width="128" height="16" src="//upload.wikimedia.org/wikipedia/commons/thumb/e/ec/Crystal_Clear_app_xmag.svg/16px-Crystal_Clear_app_xmag.svg.png" srcset="//upload.wikimedia.org/wikipedia/commons/thumb/e/ec/Crystal_Clear_app_xmag.svg/24px-Crystal_Clear_app_xmag.svg.png 1.5x, //upload.wikimedia.org/wikipedia/commons/thumb/e/ec/Crystal_Clear_app_xmag.svg/32px-Crystal_Clear_app_xmag.svg.png 2x" width="16"/> 이 부분의 본문은 <a href="/wiki/%EC%84%BC%ED%83%80%EC%9A%B0%EB%A3%A8%EC%8A%A4%EC%9E%90%EB%A6%AC_%EC%95%8C%ED%8C%8C_Bb" title="센타우루스자리 알파 Bb">센타우루스자리 알파 Bb</a>입니다.</div>
<p>2012년 센타우루스자리 알파 B를 도는 외계 행성 하나가 발견되었다. 그러나 2015년 새로운 분석 결과 Bb는 존재하지 않을 확률이 높으며, 자료분석 중 나타난 인위적 결과를 행성이 존재하는 것으로 착각한 것으로 드러났다.<sup class="reference" id="cite_ref-28"><a href="#cite_note-28">[28]</a></sup><sup class="reference" id="cite_ref-29"><a href="#cite_note-29">[29]</a></sup><sup class="reference" id="cite_ref-30"><a href="#cite_note-30">[30]</a></sup></p>
<h3><span class="mw-headline" id=".EC.B6.94.ED.9B.84_.ED.96.89.EC.84.B1.EC.9D.B4_.EB.B0.9C.EA.B2.AC.EB.90.A0_.EA.B0.80.EB.8A.A5.EC.84.B1">추후 행성이 발견될 가능성</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%84%BC%ED%83%80%EC%9A%B0%EB%A3%A8%EC%8A%A4%EC%9E%90%EB%A6%AC_%EC%95%8C%ED%8C%8C&amp;action=edit&amp;section=9" title="부분 편집: 추후 행성이 발견될 가능성">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p><a href="/wiki/%EC%84%B8%ED%8E%98%EC%9A%B0%EC%8A%A4%EC%9E%90%EB%A6%AC_%EA%B0%90%EB%A7%88" title="세페우스자리 감마">세페우스자리 감마</a>처럼 <a class="mw-redirect" href="/wiki/%EC%8C%8D%EC%84%B1%EA%B3%84" title="쌍성계">쌍성계</a>에 있는 외계 행성들이 발견되면서 센타우루스자리 알파에 행성이 있을 가능성을 높게 보았다. 2012년 발견되었던 센타우루스자리 알파 Bb는 3년 후 정밀연구 결과 존재하지 않는 것으로 결론이 났다. 그러나 천문학자들은 아직 발견되지 않은 행성이 있으리라고 믿고 있다. 가상 행성은 A와 B 별을 가까이서 돌거나, 또는 A와 B를 하나의 별처럼 삼아 멀리서 공전할 수도 있다. A와 B 모두 우리 <a href="/wiki/%ED%83%9C%EC%96%91" title="태양">태양</a>과 매우 비슷하다는 점 때문에 천문학자들은 특별히 센타우루스자리 알파 항성계 내의 행성을 면밀히 탐색해 왔다. 여러 연구진들이 시선속도법이나 트랜싯법을 사용하여 두 별에 있음직한 행성체를 찾아 왔다.<sup class="reference" id="cite_ref-universetoday.com_31-0"><a href="#cite_note-universetoday.com-31">[31]</a></sup> 지금까지 관측된 모든 자료에 따르면 <a class="mw-redirect" href="/wiki/%EA%B0%88%EC%83%89_%EC%99%9C%EC%84%B1" title="갈색 왜성">갈색 왜성</a>이나 <a class="mw-redirect" href="/wiki/%EA%B0%80%EC%8A%A4_%ED%96%89%EC%84%B1" title="가스 행성">가스 행성</a>은 이 항성계에 없었다.<sup class="reference" id="cite_ref-universetoday.com_31-1"><a href="#cite_note-universetoday.com-31">[31]</a></sup><sup class="reference" id="cite_ref-32"><a href="#cite_note-32">[32]</a></sup></p>
<p>2009년 컴퓨터 시뮬레이션에서 센타우루스자리 B의 생명체 거주가능 영역(0.5~0.9 AU) 안쪽 경계에서 행성이 태어날 수 있다는 결과가 나왔다. 알파별 A와 B가 처음에는 멀리서 생겨났다가 이후 가까워졌다는 가정을 하면 행성이 어머니 별에서 보다 먼 곳에서 생겨날 가능성도 충분했을 것이다.<sup class="reference" id="cite_ref-Thebault_2009_33-0"><a href="#cite_note-Thebault_2009-33">[33]</a></sup> A 주위를 도는 천체는 보다 강한 중력 때문에 B 주위 천체보다 조금 더 먼 곳을 돌 것이다. 또한 A와 B 근처에 갈색 왜성이 없기 때문에 지구형 행성이 존재할 가능성은 크다.<sup class="reference" id="cite_ref-lackofany_34-0"><a href="#cite_note-lackofany-34">[34]</a></sup> 시선속도법을 이용하여 행성을 찾아내는 것을 이론적으로 연구했을 때 구경 1미터 망원경으로 3년 이내에 B 주위를 도는 지구질량의 1.8배 천체까지 잡아낼 수 있음이 밝혀졌다.<sup class="reference" id="cite_ref-guedesetal2008_35-0"><a href="#cite_note-guedesetal2008-35">[35]</a></sup></p>
<p>센타우루스자리 알파 B를 HARPS로 관측한 결과 생물권 내(공전주기 200일)에 지구질량 4배 이상 천체는 없는 것으로 드러났다.</p>
<h3><span class="mw-headline" id=".ED.83.90.EC.82.AC_.EB.8C.80.EC.83.81.EC.9C.BC.EB.A1.9C.EC.84.9C.EC.9D.98_.EA.B0.80.EB.8A.A5.EC.84.B1">탐사 대상으로서의 가능성</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%84%BC%ED%83%80%EC%9A%B0%EB%A3%A8%EC%8A%A4%EC%9E%90%EB%A6%AC_%EC%95%8C%ED%8C%8C&amp;action=edit&amp;section=10" title="부분 편집: 탐사 대상으로서의 가능성">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>지구와 비슷한 행성을 찾는다는 목표 아래 NASA는 SIM 계획에 "티어 1"(<i>Tier 1</i>)이라는 탐사 목표 항성들의 목록을 작성했고 센타우루스자리 알파 AB 역시 이 목록에 포함되었다. 이 계획을 통해 어머니 항성에서 2AU 안쪽을 도는 지구질량 3배 이하의 행성들을 찾는 계획이 수립되었다.<sup class="reference" id="cite_ref-numbers_36-0"><a href="#cite_note-numbers-36">[36]</a></sup> 그러나 SIM 계획은 2010년 예산문제로 취소되었다.<sup class="reference" id="cite_ref-37"><a href="#cite_note-37">[37]</a></sup></p>
<p>2012년 10월 B 주위를 도는 행성이 발견되면서 센타우루스자리 알파는 다시금 성간 탐사의 유력한 목표지로 떠올랐다.<sup class="reference" id="cite_ref-ScienceNews_38-0"><a href="#cite_note-ScienceNews-38">[38]</a></sup><sup class="reference" id="cite_ref-DiscoveryProbe_39-0"><a href="#cite_note-DiscoveryProbe-39">[39]</a></sup> 세로 톨로로 범미주 천문대 소속 데브라 피셔는 "우리가 우주선 또는 탐사선을 어디론가 보내려 한다면 알파 항성계가 첫 목적지일 것입니다."라고 말했다. 이론적으로 태양 돛이나 반물질 로켓, 핵 펄스 추진 로켓 기술을 쓰면 수백 년에서 수십 년 정도로 시간을 단축할 수 있을 것이다.<sup class="reference" id="cite_ref-ScienceNews_38-1"><a href="#cite_note-ScienceNews-38">[38]</a></sup><sup class="reference" id="cite_ref-40"><a href="#cite_note-40">[40]</a></sup> 다만 행성과학자 그렉 래플린은 앞의 기술들은 현 시점에서는 아직 비현실적이며 도박성이 강하고, 현재 기술력으로 알파 항성계에 탐사선을 보내려면 4만 년은 걸린다고 말했다. 그러나 알파 항성계 내에 생명체가 살 수 있는 행성이 있음이 확인된다면 사람의 수명보다 짧은 시간 내 알파 항성계에 갈 수 있는 기술 개발에 박차가 가해질 것이라고 했다. 이는 과거 <a href="/wiki/%EB%8B%AC" title="달">달</a>에 대한 호기심이 월면 탐사로 이어진 것과 유사한 과정이다.<sup class="reference" id="cite_ref-Time_41-0"><a href="#cite_note-Time-41">[41]</a></sup></p>
<h3><span class="mw-headline" id=".EA.B0.80.EC.83.81_.ED.96.89.EC.84.B1">가상 행성</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%84%BC%ED%83%80%EC%9A%B0%EB%A3%A8%EC%8A%A4%EC%9E%90%EB%A6%AC_%EC%95%8C%ED%8C%8C&amp;action=edit&amp;section=11" title="부분 편집: 가상 행성">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>과거 컴퓨터 가상 모형으로는 센타우루스자리 알파 A와 B 주변 모두 지구형 행성이 탄생할 수 있다는 결과가 나왔으나<sup class="reference" id="cite_ref-42"><a href="#cite_note-42">[42]</a></sup> 이후 연구를 통해 짝별의 중력 때문에 행성이 뭉쳐져 태어나는 작용이 방해를 받음이 드러났다.<sup class="reference" id="cite_ref-43"><a href="#cite_note-43">[43]</a></sup> 이러한 변수들에도 불구하고 태양과 분광형, 나이, 안정된 공전궤도 등의 이유로 알파 항성계는 외계 생명을 품는 행성을 거느릴 확률이 높다고 인정되어 왔고<sup class="reference" id="cite_ref-Wiegert_44-0"><a href="#cite_note-Wiegert-44">[44]</a></sup><sup class="reference" id="cite_ref-45"><a href="#cite_note-45">[45]</a></sup><sup class="reference" id="cite_ref-46"><a href="#cite_note-46">[46]</a></sup><sup class="reference" id="cite_ref-47"><a href="#cite_note-47">[47]</a></sup> 2012년 10월 센타우루스자리 알파 Bb의 발견은 그러한 예측이 맞았음을 입증했다.</p>
<p>일부 천문학자들은 알파 항성계에 행성이 있다고 하더라도 표면은 바싹 말랐거나 대기가 날아가고 없으리라고 추측했다. 태양계에서 목성과 토성은 <a href="/wiki/%ED%98%9C%EC%84%B1" title="혜성">혜성</a>들의 궤도를 내행성 쪽으로 트는 데 중요한 역할을 했다. 이 혜성들은 내행성에 물 또는 여러 형태의 얼음 물질을 공급했다.<sup class="reference" id="cite_ref-Croswell_48-0"><a href="#cite_note-Croswell-48">[48]</a></sup> 그러나 <a href="/wiki/%EC%84%BC%ED%83%80%EC%9A%B0%EB%A3%A8%EC%8A%A4%EC%9E%90%EB%A6%AC_%ED%94%84%EB%A1%9D%EC%8B%9C%EB%A7%88" title="센타우루스자리 프록시마">센타우루스자리 프록시마</a>가 우리 태양계의 목성·토성 역할을 하여 A와 B 근처에 휘발성 물질을 공급했을 가능성이 있다.<sup class="reference" id="cite_ref-49"><a href="#cite_note-49">[49]</a></sup> 또는 B와 A가 자신의 중력으로 혜성의 궤도를 틀어 상대 짝별 내행성 궤도에 혜성을 접근하도록 작용했을 수도 있다. 혜성들은 처음에는 항성계 멀리 있는 거대한 <a href="/wiki/%EC%98%A4%EB%A5%B4%ED%8A%B8_%EA%B5%AC%EB%A6%84" title="오르트 구름">오르트 구름</a> 내에 자리잡고 있다가 가스 행성이나 주변을 지나가는 다른 별의 중력 때문에 어머니 별을 향해 궤도를 틀게 되며, 이들이 내행성으로 접근하게 된다.<sup class="reference" id="cite_ref-Matthews_26-1"><a href="#cite_note-Matthews-26">[26]</a></sup> 그럼에도 센타우루스자리 알파 쌍성계 주변에 이런 오르트 구름이 있다는 증거는 없으며, 이론적으로 오르트 구름은 항성계 생성 초기에 완전히 사라졌을 것이다.</p>
<p>만약 센타우루스자리 알파 A 주변 <a href="/wiki/%EC%83%9D%EB%AA%85%EC%B2%B4_%EA%B1%B0%EC%A3%BC%EA%B0%80%EB%8A%A5_%EC%98%81%EC%97%AD" title="생명체 거주가능 영역">생명체 거주가능 영역</a>에 지구와 비슷한 환경이 조성되려면 대략 1.25 AU 떨어져 있어야 할 것이다(이는 지구-화성 중간 거리 정도이다). 이 거리 정도에서 물은 액체 상태로 존재할 수 있다. A보다 좀 더 작고 차가운 B의 경우 그 거리는 0.7 AU 정도로 줄어들며 이는 태양~<a href="/wiki/%EA%B8%88%EC%84%B1" title="금성">금성</a> 거리 정도 된다.<sup class="reference" id="cite_ref-Croswell_48-1"><a href="#cite_note-Croswell-48">[48]</a></sup><sup class="reference" id="cite_ref-50"><a href="#cite_note-50">[50]</a></sup></p>

<p>센타우루스자리 알파 A,B의 주변을 <a href="/wiki/%EA%B3%B5%EC%A0%84" title="공전">공전</a>하는 <a class="new" href="/w/index.php?title=%EA%B0%80%EC%83%81%ED%96%89%EC%84%B1&amp;action=edit&amp;redlink=1" title="가상행성 (없는 문서)">가상행성</a>(假想行星)에서 바라본 밤하늘은 지구에서 바라보는 밤하늘과 크게 다르지 않을 것이다. 그러나 센타우루스자리의 가장 밝은 별은 보이지 않을 것이다. 태양은 <a href="/wiki/%EC%B9%B4%EC%8B%9C%EC%98%A4%ED%8E%98%EC%9D%B4%EC%95%84%EC%9E%90%EB%A6%AC" title="카시오페이아자리">카시오페이아자리</a>에서 0.5등성의 가장 밝은 별로 보일 것이다. 태양은 <a class="mw-redirect" href="/wiki/%EC%B9%B4%EC%8B%9C%EC%98%A4%ED%8E%98%EC%9D%B4%EC%95%84%EC%9E%90%EB%A6%AC_%EC%97%A1%EC%8B%A4%EB%A1%A0" title="카시오페이아자리 엡실론">카시오페이아자리 엡실론</a>의 옆에 위치할 것이며, 지구에서 바라볼 때 \/\/모양의 카시오페이아자리는 /\/\/로 바뀔 것이다.<sup class="reference" id="cite_ref-Croswell_48-2"><a href="#cite_note-Croswell-48">[48]</a></sup></p>
<p>지구에서 상대적으로 가까운 별들인 <a href="/wiki/%EC%8B%9C%EB%A6%AC%EC%9A%B0%EC%8A%A4" title="시리우스">시리우스</a>, <a href="/wiki/%ED%94%84%EB%A1%9C%ED%82%A4%EC%98%A8" title="프로키온">프로키온</a>, <a href="/wiki/%EC%95%8C%ED%83%80%EC%9D%B4%EB%A5%B4" title="알타이르">알타이르</a> 등은 위치가 크게 변할 것이다. 그보다 좀 더 먼 별들인 <a href="/wiki/%ED%8F%AC%EB%A7%90%ED%95%98%EC%9A%B0%ED%8A%B8" title="포말하우트">포말하우트</a>나 <a class="mw-redirect" href="/wiki/%EC%A7%81%EB%85%80%EC%84%B1" title="직녀성">직녀성</a>의 경우 위치가 약간 변할 것이다. 센타우루스자리 프록시마는 0.2광년밖에 떨어져 있지 않음에도, 낮은 광도 때문에 4.5등성의 어두운 별로 보일 것이다.<sup class="reference" id="cite_ref-Croswell_48-3"><a href="#cite_note-Croswell-48">[48]</a></sup></p>
<p>가상의 행성에서 바라본 동반성(同伴星)은 '어두운 태양'으로 보일 것이다. 예를 들면, A의 주위를 1.25AU 떨어져서 도는 가상행성은 A로부터 태양과 비슷한 빛을 받는데, 여기서 관찰한 B는 -21~-18.2등급의 밝기를 보일 것이다. 이는 주성의 밝기에 비해 190~2700배 어둡지만, <a href="/wiki/%EB%B3%B4%EB%A6%84%EB%8B%AC" title="보름달">보름달</a>보다 170~2300배 밝다. 반대로 B의 주위를 0.71AU 떨어져서 도는 가상행성은 B로부터 역시 태양과 비슷한 빛을 받는데, 여기서 관찰한 A는 -22.1~-19.4등급의 밝기를 보일 것이다. 이는 주성의 밝기에 비하여 70~840배 어둡지만, 보름달에 비하여 520~6300배 밝다. 두 경우 모두 반대쪽 동반성은 가상행성의 하늘을 횡단하는 것처럼 보일 것이다.<sup class="reference" id="cite_ref-Croswell_48-4"><a href="#cite_note-Croswell-48">[48]</a></sup></p>
<p>만약 가상행성의 궤도경사각이 A와 B의 공전 궤도와 크게 차이가 나지 않는다고 가정하면, 동반성이 가상행성의 어머니 항성과 합의 위치에 올 경우 두 항성은 행성의 하늘에서 서로 반대 위치에 자리잡게 된다. 이 경우 가상행성이 만약 지구와 비슷한 대기를 지니고 있다고 가정한다면, 이 행성의 거주민이 바라보는 밤하늘은 1년 중 절반에 걸쳐 반성 때문에 암흑이 아니라 어두운 청색으로 보일 것이다. 이는 지구에서 <a href="/wiki/%EC%9D%BC%EC%8B%9D" title="일식">일식</a>이 일어날 때의 밝기와 비슷한 수준이다. 이 경우 가상행성의 거주자는 인공조명 없이도 책을 읽을 수 있다. A와 B는 가장 가까이 접근해도 11AU의 거리를 보이므로, 동반성으로부터 받는 빛과 열은 가상행성에서 살아가는 <a href="/wiki/%EC%8B%9D%EB%AC%BC" title="식물">식물</a>의 <a href="/wiki/%EA%B4%91%ED%95%A9%EC%84%B1" title="광합성">광합성</a>이나 행성의 <a href="/wiki/%EA%B8%B0%ED%9B%84" title="기후">기후</a>에 부정적인 효과를 가져올 정도는 아니다.<sup class="reference" id="cite_ref-Croswell_48-5"><a href="#cite_note-Croswell-48">[48]</a></sup></p>
<h2><span class="mw-headline" id=".EA.B2.89.EB.B3.B4.EA.B8.B0.EC.9A.B4.EB.8F.99.EA.B3.BC_.EA.B3.A0.EC.9C.A0.EC.9A.B4.EB.8F.99">겉보기운동과 고유운동</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%84%BC%ED%83%80%EC%9A%B0%EB%A3%A8%EC%8A%A4%EC%9E%90%EB%A6%AC_%EC%95%8C%ED%8C%8C&amp;action=edit&amp;section=13" title="부분 편집: 겉보기운동과 고유운동">편집</a><span class="mw-editsection-bracket">]</span></span></h2>

<p>약 4000년 후, 밤하늘의 센타우루스자리 알파는 <a class="mw-redirect" href="/wiki/%EC%84%BC%ED%83%80%EC%9A%B0%EB%A3%A8%EC%8A%A4%EC%9E%90%EB%A6%AC_%EB%B2%A0%ED%83%80" title="센타우루스자리 베타">센타우루스자리 베타</a> 근처로 접근하여, 마치 <a class="mw-redirect" href="/wiki/%EC%95%88%EC%8B%9C%EC%9D%B4%EC%A4%91%EC%84%B1" title="안시이중성">안시이중성</a>처럼 보일 것이다. 그러나 이것은 지구에서 바라보았을 때 바로 옆에 있는 것처럼 보이는 것일 뿐, 실제로 센타우루스자리 베타는 알파보다 지구에서 훨씬 더 멀리 떨어져 있으며, 알파보다 수천 배 밝다.</p>
<div class="thumb tleft">
<div class="thumbinner" style="width:252px;"><a class="image" href="/wiki/%ED%8C%8C%EC%9D%BC:Near-stars-past-future-de.svg"><img alt="" class="thumbimage" data-file-height="800" data-file-width="999" height="200" src="//upload.wikimedia.org/wikipedia/commons/thumb/8/8e/Near-stars-past-future-de.svg/250px-Near-stars-past-future-de.svg.png" srcset="//upload.wikimedia.org/wikipedia/commons/thumb/8/8e/Near-stars-past-future-de.svg/375px-Near-stars-past-future-de.svg.png 1.5x, //upload.wikimedia.org/wikipedia/commons/thumb/8/8e/Near-stars-past-future-de.svg/500px-Near-stars-past-future-de.svg.png 2x" width="250"/></a>
<div class="thumbcaption">
<div class="magnify"><a class="internal" href="/wiki/%ED%8C%8C%EC%9D%BC:Near-stars-past-future-de.svg" title="실제 크기로"></a></div>
태양계 근처의 항성들과 지구 사이의 거리 변화를 나타낸 그래프. 가로축은 시간이며 세로축은 거리(광년)이다.</div>
</div>
</div>
<p>센타우루스자리 알파는 지금도 초속 22킬로미터의 속도로 태양에 가까워지고 있다.(그러나 센타우루스자리 프록시마는 상대적으로 접근 속도가 느려서 초속 16킬로미터이다). 이 추세라면 알파는 약 2만 8천 년 후 태양계에 3광년까지 접근할 것이다. 알파는 지금의 <a href="/wiki/%EB%8F%9B%EC%9E%90%EB%A6%AC" title="돛자리">돛자리</a>와 <a href="/wiki/%EB%B0%94%EB%8B%A4%EB%B1%80%EC%9E%90%EB%A6%AC" title="바다뱀자리">바다뱀자리</a> 사이 위치로 자리를 옮길 것이며, <a class="mw-redirect" href="/wiki/%EA%B2%89%EB%B3%B4%EA%B8%B0_%EB%B0%9D%EA%B8%B0" title="겉보기 밝기">겉보기 밝기</a>는 -1.28까지 상승한다. 이 시점에서 알파보다 밝은 별은 <a href="/wiki/%EC%8B%9C%EB%A6%AC%EC%9A%B0%EC%8A%A4" title="시리우스">시리우스</a> 하나뿐이다.</p>
