<p><b>레알 클루브 셀타 데 비고</b>(<span lang="es" xml:lang="es">Real Club Celta de Vigo</span>), 약칭 <b>셀타 비고</b>(<span lang="en" xml:lang="en">Celta Vigo</span>)는 <a href="/wiki/%EC%8A%A4%ED%8E%98%EC%9D%B8" title="스페인">스페인</a> <a href="/wiki/%EA%B0%88%EB%A6%AC%EC%8B%9C%EC%95%84_%EC%A7%80%EB%B0%A9" title="갈리시아 지방">갈리시아 지방</a>의 도시인 <a href="/wiki/%EB%B9%84%EA%B3%A0" title="비고">비고</a>를 연고로 하는 <a href="/wiki/%EC%B6%95%EA%B5%AC" title="축구">축구</a> 클럽 팀으로, 현재 <a class="mw-redirect" href="/wiki/%ED%94%84%EB%A6%AC%EB%A9%94%EB%9D%BC%EB%A6%AC%EA%B0%80" title="프리메라리가">프리메라리가</a> 소속이다. 셀타 비고와 <a href="/wiki/%EB%8D%B0%ED%8F%AC%EB%A5%B4%ED%8B%B0%EB%B3%B4_%EB%9D%BC%EC%BD%94%EB%A3%A8%EB%83%90" title="데포르티보 라코루냐">데포르티보 라코루냐</a>와의 경기는 이른바 <a class="mw-redirect" href="/wiki/%EA%B0%88%EB%A6%AC%EC%8B%9C%EC%95%84" title="갈리시아">갈리시아</a> 더비로 유명하다.</p>

<h2><span class="mw-headline" id=".EC.97.AD.EC.82.AC">역사</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%85%80%ED%83%80_%EB%B9%84%EA%B3%A0&amp;action=edit&amp;section=1" title="부분 편집: 역사">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<h3><span class="mw-headline" id=".EB.AA.85.EC.B9.AD.2C_.EC.A7.80.EC.97.AD">명칭, 지역</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%85%80%ED%83%80_%EB%B9%84%EA%B3%A0&amp;action=edit&amp;section=2" title="부분 편집: 명칭, 지역">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p><a class="mw-redirect" href="/wiki/%EB%A0%88%EC%95%8C_%ED%81%B4%EB%9F%BD_%EC%85%80%ED%83%80_%EB%8D%B0_%EB%B9%84%EA%B3%A0" title="레알 클럽 셀타 데 비고">레알 클럽 셀타 데 비고</a>라고 한다. <a href="/wiki/%EC%8A%A4%ED%8E%98%EC%9D%B8" title="스페인">스페인</a><a href="/wiki/%EB%A6%AC%EA%B7%B8_(%EC%8A%A4%ED%8F%AC%EC%B8%A0)" title="리그 (스포츠)">리그</a> <a class="mw-redirect" href="/wiki/%ED%94%84%EB%A6%AC%EB%A9%94%EB%9D%BC%EB%A6%AC%EA%B0%80" title="프리메라리가">프리메라리가</a>에 소속된 프로축구클럽으로, <a href="/wiki/1923%EB%85%84" title="1923년">1923년</a> 창단하였다. 레알 비고 스포르팅과 레알 클럽 포르투나 데 비고가 합병됨으로써 창설됐다.</p>
<p>연고지는 폰테베드라주 <a href="/wiki/%EB%B9%84%EA%B3%A0" title="비고">비고</a>이다. 스페인 갈리시아 지방에서 가장 큰 도시로, 인구 30만정도의 중소도시이다. 공식 클럽명은 레알 클럽 셀타 데 비고이며, 팀 <a class="mw-redirect" href="/wiki/%EC%95%A0%EC%B9%AD" title="애칭">애칭</a>은 셀티냐, 포투게세스 또는 <a href="/wiki/%ED%95%98%EB%8A%98%EC%83%89" title="하늘색">하늘색</a>이란 <a class="mw-redirect" href="/wiki/%EB%9C%BB" title="뜻">뜻</a>의 셀레스테스이다. 갈리시아는 본래 켈트족의 땅이었기 때문에 그 피를 계승한다는 의미로 '셀타(Celta)'라는 이름이 붙여졌다. 셀타라는 이름은 스페인어로 읽으면 "셀타"라고 읽긴 하지만 이 것을 영어로 읽으면 "켈타" 즉 켈트족이 살던곳이라는 뜻이 되어버린다.</p>
<p>오랜 <a href="/wiki/%EC%97%AD%EC%82%AC" title="역사">역사</a>를 지닌 팀으로, <a class="mw-redirect" href="/wiki/%ED%94%84%EB%A6%AC%EB%A9%94%EB%9D%BC%EB%A6%AC%EA%B0%80" title="프리메라리가">프리메라리가</a>의 <a href="/wiki/10" title="10">10</a>개 창설<a class="mw-redirect" href="/wiki/%ED%81%B4%EB%9F%BD" title="클럽">클럽</a>에 속하며 <a href="/wiki/%EA%B2%BD%EC%9F%81" title="경쟁">경쟁</a>팀인 <a href="/wiki/%EB%8D%B0%ED%8F%AC%EB%A5%B4%ED%8B%B0%EB%B3%B4_%EB%9D%BC%EC%BD%94%EB%A3%A8%EB%83%90" title="데포르티보 라코루냐">데포르티보 라코루냐</a>와 함께 <a class="mw-redirect" href="/wiki/%EC%97%90%EC%8A%A4%ED%8C%8C%EB%83%90" title="에스파냐">에스파냐</a> <a href="/wiki/%EB%B6%81%EC%84%9C%EC%AA%BD" title="북서쪽">북서쪽</a> <a href="/wiki/%EA%B0%88%EB%A6%AC%EC%8B%9C%EC%95%84_%EC%A7%80%EB%B0%A9" title="갈리시아 지방">갈리시아 지방</a>에 있는 <a href="/wiki/2" title="2">2</a>대 <a class="mw-redirect" href="/wiki/%ED%81%B4%EB%9F%BD" title="클럽">클럽</a>의 <a class="mw-disambig" href="/wiki/%ED%95%98%EB%82%98" title="하나">하나</a>이다. <a href="/wiki/%EB%8D%B0%ED%8F%AC%EB%A5%B4%ED%8B%B0%EB%B3%B4_%EB%9D%BC%EC%BD%94%EB%A3%A8%EB%83%90" title="데포르티보 라코루냐">데포르티보 라코루냐</a>와는 <a class="new" href="/w/index.php?title=%EA%B0%88%EB%A6%AC%EC%8B%9C%EC%95%84_%EB%8D%94%EB%B9%84&amp;action=edit&amp;redlink=1" title="갈리시아 더비 (없는 문서)">갈리시아 더비</a> 라이벌로 스페인에서 가장 치열한 더비 중 하나로 꼽히고 있다. (<a class="mw-redirect" href="/wiki/%EB%A0%88%EC%95%8C_%EB%A7%88%EB%93%9C%EB%A6%AC%EB%93%9C" title="레알 마드리드">레알 마드리드</a>와 <a href="/wiki/FC_%EB%B0%94%EB%A5%B4%EC%85%80%EB%A1%9C%EB%82%98" title="FC 바르셀로나">FC 바르셀로나</a>와의 경기가 펼쳐지더라도 인구가 그다지 많지 않아서 관중석이 꽉차는 경우가 별로없지만, <a class="new" href="/w/index.php?title=%EA%B0%88%EB%A6%AC%EC%8B%9C%EC%95%84_%EB%8D%94%EB%B9%84&amp;action=edit&amp;redlink=1" title="갈리시아 더비 (없는 문서)">갈리시아 더비</a>가 펼쳐지게 되면 놀랍게도 관중이 만원을 이루는 광경이 펼쳐진다.)</p>
<p>셀타 비고는 <a class="mw-redirect" href="/wiki/%EB%8D%B0%ED%8F%AC%EB%A5%B4%ED%8B%B0%EB%B3%B4_%EB%9D%BC_%EC%BD%94%EB%A3%A8%EB%83%90" title="데포르티보 라 코루냐">데포르티보 라 코루냐</a>와 함께 <a class="mw-redirect" href="/wiki/%EA%B0%88%EB%A6%AC%EC%8B%9C%EC%95%84" title="갈리시아">갈리시아</a>를 대표하는 팀으로 명성을 얻어왔다. 그러나 이 지역의 클럽들이 스페인 프로무대에서 크게 영향력을 발휘하기 시작한 것은 그리 오래된 일이 아니다. <a class="mw-redirect" href="/wiki/%EB%A0%88%EC%95%8C_%EB%A7%88%EB%93%9C%EB%A6%AC%EB%93%9C" title="레알 마드리드">레알 마드리드</a>와 <a href="/wiki/%EC%95%84%ED%8B%80%EB%A0%88%ED%8B%B0%EC%BD%94_%EB%A7%88%EB%93%9C%EB%A6%AC%EB%93%9C" title="아틀레티코 마드리드">아틀레티코 마드리드</a>가 버티고 있는 <a class="new" href="/w/index.php?title=%EA%B9%8C%EC%8A%A4%ED%8B%B0%EC%95%BC&amp;action=edit&amp;redlink=1" title="까스티야 (없는 문서)">까스티야</a> 지방, <a href="/wiki/FC_%EB%B0%94%EB%A5%B4%EC%85%80%EB%A1%9C%EB%82%98" title="FC 바르셀로나">FC 바르셀로나</a>로 대변되는 <a class="mw-redirect" href="/wiki/%EA%B9%8C%EB%94%B8%EB%A3%A8%EB%83%90" title="까딸루냐">까딸루냐</a> 지방 전통의 명문 <a href="/wiki/%EC%84%B8%EB%B9%84%EC%95%BC" title="세비야">세비야</a>와 <a href="/wiki/%EB%A0%88%EC%95%8C_%EB%B2%A0%ED%8B%B0%EC%8A%A4" title="레알 베티스">레알 베티스</a>의 <a href="/wiki/%EC%95%88%EB%8B%AC%EB%A3%A8%EC%8B%9C%EC%95%84_%EC%A7%80%EB%B0%A9" title="안달루시아 지방">안달루시아 지방</a>, <a href="/wiki/%EB%B9%8C%EB%B0%94%EC%98%A4" title="빌바오">빌바오</a>와 <a class="new" href="/w/index.php?title=%EC%86%8C%EC%8B%9C%EC%97%90%EB%8B%A4%EB%93%9C&amp;action=edit&amp;redlink=1" title="소시에다드 (없는 문서)">소시에다드</a>가 주축이 되는 <a href="/wiki/%EB%B0%94%EC%8A%A4%ED%81%AC" title="바스크">바스크</a> 지방, 마지막으로 '박쥐군단' <a href="/wiki/%EB%B0%9C%EB%A0%8C%EC%8B%9C%EC%95%84" title="발렌시아">발렌시아</a>의 발렌시아 연안을 대표하는 팀들이 <a class="new" href="/w/index.php?title=%ED%94%84%EB%A6%AC%EB%A9%94%EB%9D%BC&amp;action=edit&amp;redlink=1" title="프리메라 (없는 문서)">프리메라</a> 리가의 왕좌 자리를 번갈아가며 차지했던 역사를 돌이켜 본다면, <a class="mw-redirect" href="/wiki/%EA%B0%88%EB%A6%AC%EC%8B%9C%EC%95%84" title="갈리시아">갈리시아</a> 팀들은 그야말로 패배의 역사를 반복 해왔다 해도 지난친 말은 아니다.</p>
<p>그러나 90년대로 접어들면서 이야기는 달라졌다. <a class="new" href="/w/index.php?title=%EB%A0%8C%EB%8F%84%EC%9D%B4%EB%A1%9C&amp;action=edit&amp;redlink=1" title="렌도이로 (없는 문서)">렌도이로</a> 구단주의 취임과 함께 신흥강호로 급부상한 셀타의 '더비 라이벌' <a class="mw-redirect" href="/wiki/%EB%8D%B0%ED%8F%AC%EB%A5%B4%ED%8B%B0%EB%B3%B4_%EB%9D%BC_%EC%BD%94%EB%A3%A8%EB%83%90" title="데포르티보 라 코루냐">데포르티보 라 코루냐</a>는 99/00 시즌에 <a class="mw-redirect" href="/wiki/%EA%B0%88%EB%A6%AC%EC%8B%9C%EC%95%84" title="갈리시아">갈리시아</a> 지방에 첫 우승컵을 안겨주는 쾌거를 이룩했고, 셀타 비고 또한 90년대 후반부터 <a href="/wiki/%EC%8A%A4%ED%8E%98%EC%9D%B8" title="스페인">스페인</a>의 새로운 강팀으로 자리를 잡는데 성공했다.</p>
<p>홈 구장은 31,800명 수용규모의 <a class="new" href="/w/index.php?title=%EB%B0%9C%EB%9D%BC%EC%9D%B4%EB%8F%84%EC%8A%A4&amp;action=edit&amp;redlink=1" title="발라이도스 (없는 문서)">발라이도스</a>이다.</p>

<h3><span class="mw-headline" id=".EB.A8.BC_.EC.8B.9C.EC.A6.8C_.EC.84.B1.EC.A0.81">먼 시즌 성적</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%85%80%ED%83%80_%EB%B9%84%EA%B3%A0&amp;action=edit&amp;section=3" title="부분 편집: 먼 시즌 성적">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>공식적으로 1923년에 창단되었지만 이들의 역사는 거의 100년전으로 거슬러 올라간다. 원래 셀타 비고라는팀은 비고의 한 조그마한 클럽이었다. 갈리시안컵 우승을 했던 1908년부터 1923년 까지 무려 8번이나 우승했고, 비고시에서 자체적으로 벌어진 축구대회에서도 거의 싹쓸이를 하다시피 우승을 차지하였다. 공식적으로 라리가에 대뷔해서 1923년을 창단연도로 두고 있지만 그 뿌리는 꽤나 깊은 장수한 클럽임을 알수 있다.</p>
<p>초기의 <strong class="selflink">셀타 비고</strong>는 결정적인 경기에서 승리하면서 놀라운 성장을 한 클럽이다. 1923년 9월 23일 그들은 그들의 클럽 역사의 첫 경기을 승리하면서 강한 인상으로 클럽 데뷔를 했었고 1924-1928년 사이에 4번의 <a class="new" href="/w/index.php?title=%EA%B0%88%EB%A6%AC%EC%8B%9C%EC%95%88&amp;action=edit&amp;redlink=1" title="갈리시안 (없는 문서)">갈리시안</a> 챔피언을 획득했다.그리고 1928년 스페인 리그가 생기면서 그들은 지역의 챔피언쉽을 버리면서 <a class="new" href="/w/index.php?title=%ED%94%84%EB%A6%AC%EB%A9%94%EB%9D%BC_%EB%94%94%EB%B9%84%EC%A0%BC&amp;action=edit&amp;redlink=1" title="프리메라 디비젼 (없는 문서)">프리메라 디비젼</a>에 참가했고 그들은 <a class="new" href="/w/index.php?title=%ED%94%84%EB%A6%AC%EB%A9%94%EB%9D%BC_%EB%94%94%EB%B9%84%EC%A0%BC&amp;action=edit&amp;redlink=1" title="프리메라 디비젼 (없는 문서)">프리메라 디비젼</a>의 창립 클럽이 되었다.1936년 그들이 1부 리그에 올라오면서 18년 동안 한번을 제외하고는 1부 리그를 유지하면서 괜찮은 활약을 하였고 1947-48년 최고의 활약을 한다. 전설이 된 골키퍼 <a href="/wiki/%EB%A6%AC%EC%B9%B4%EB%A5%B4%EB%8F%84_%EC%82%AC%EB%AA%A8%EB%9D%BC" title="리카르도 사모라">자모라</a>는 감독으로써 셀타 비고를 바르셀로나에 이어 준우승시킬 정도였으니 전성기였다고 봐도 무관하다.그러나 그 이후 셀타 비고는 1954년 2부 리그로 강등되었다.</p>
<p>1954년부터 1969년까지 셀타 비고는 2부 리그에서 프리메라로 올라오기 위해 엄청난 투쟁을 했는데 물론 59-60, 60-61, 65-66 시즌에 그들이 <a class="mw-redirect" href="/wiki/%ED%94%84%EB%A6%AC%EB%A9%94%EB%9D%BC_%EB%A6%AC%EA%B0%80" title="프리메라 리가">프리메라 리가</a>에서 뛸 수 있는 능력이 있음을 증명했지만 아쉽게 실패하고 만다. 그러나 그들은 <a class="new" href="/w/index.php?title=Uefa_%EC%BB%B5&amp;action=edit&amp;redlink=1" title="Uefa 컵 (없는 문서)">Uefa 컵</a> 출전 자격을 얻음으로서 그들의 역사상 처음으로 유럽 대항전에 나갈 기회를 잡는다.그리고 1979-80 시즌에 셀타 비고는 기어이 3부 리그까지 강등되고 만다. 그들의 역사상 가장 치욕적인 기록이 아닐 수 없지만 이로써 그들은 바닥을 경험했고 그들은 92년 드디어 다시 복귀하면서 그들의 실력이 프리메라 리가의 정상은 아니지만 프리메라 리가에서 쫓겨날 정도도 역시 아니라는 것을 증명하였다. 그리고 1994년 드디어 그들은 <a class="new" href="/w/index.php?title=%EC%8A%A4%ED%8E%98%EC%9D%B8_%EC%BB%B5&amp;action=edit&amp;redlink=1" title="스페인 컵 (없는 문서)">스페인 컵</a>의 결승까지 오르지만 우세한 경기 내용에도 불구하고 집중력 부족으로 <a href="/wiki/%EB%A0%88%EC%95%8C_%EC%82%AC%EB%9D%BC%EA%B3%A0%EC%82%AC" title="레알 사라고사">레알 사라고사</a>에게 패널티 킥으로 지고 만다.</p>

<h3><span class="mw-headline" id=".EC.A3.BC.EC.9A.94_.EC.8B.9C.EC.A6.8C_.EC.84.B1.EC.A0.81">주요 시즌 성적</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%85%80%ED%83%80_%EB%B9%84%EA%B3%A0&amp;action=edit&amp;section=4" title="부분 편집: 주요 시즌 성적">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>1969/70 시즌, <a class="mw-redirect" href="/wiki/%ED%94%84%EB%A6%AC%EB%A9%94%EB%9D%BC_%EB%A6%AC%EA%B0%80" title="프리메라 리가">프리메라 리가</a>에 승격한 셀타 비고는 70/71 시즌 홈 무패와 함께 6위를 차지하며 '강팀 킬러'로서의 명성을 떨쳤다.</p>
<p><br/>

title="프리메라리가">프리메라리가</a>로 올라온 셀타 비고는 1996년 <a class="new" href="/w/index.php?title=%EC%98%A4%EB%9D%BC%EC%8B%9C%EC%98%A4_%EA%B3%A0%EB%A9%94%EC%8A%A4&amp;action=edit&amp;redlink=1" title="오라시오 고메스 (없는 문서)">오라시오 고메스</a>가 총재로 발탁된 이후 급격한 발전을 시작하게 된다. 1996년에 고메스는 자신이 구단주에 당선되자마자 전력강화의 일환으로 <a href="/wiki/1994%EB%85%84_FIFA_%EC%9B%94%EB%93%9C%EC%BB%B5" title="1994년 FIFA 월드컵">1994년 FIFA 월드컵</a> 우승주역인 <a href="/wiki/%EC%84%9C%EC%9A%B8_%EC%98%AC%EB%A6%BC%ED%94%BD" title="서울 올림픽">서울 올림픽</a> 금메달 리스트인 <a href="/wiki/%EC%95%8C%EB%A0%89%EC%82%B0%EB%93%9C%EB%A5%B4_%EB%AA%A8%EC%8A%A4%ED%86%A0%EB%B3%B4%EC%9D%B4" title="알렉산드르 모스토보이">알렉산드르 모스토보이</a>, 프랑스 출신의 골리인 <a class="new" href="/w/index.php?title=%EB%92%A4%ED%8A%B8%EB%A6%AC%EC%97%98&amp;action=edit&amp;redlink=1" title="뒤트리엘 (없는 문서)">뒤트리엘</a>을 영입했으며, 그 다음해인 1997년에는 러시아의 <a href="/w/index.php?title=%EC%A1%B0%EB%A1%9C%EB%B9%84%EC%B9%98&amp;action=edit&amp;redlink=1" title="조로비치 (없는 문서)">조로비치</a>, 노르웨이의 <a class="new" href="/w/index.php?title=%EC%97%90%EA%B2%90&amp;action=edit&amp;redlink=1" title="에겐 (없는 문서)">에겐</a>, <a class="mw-redirect" href="/wiki/%EB%A0%88%EC%95%8C_%EB%A7%88%EB%93%9C%EB%A6%AC%EB%93%9C" 마케렐레,셀라데스,페네프와 같은 우수한 선수들을 영입한다.</p>

1997/98 시즌부터 2002/03 시즌까지는 셀타의 황금기 였다. 당시 셀타 비고는 <a class="new" href="/w/index.php?title=%EB%B9%85%ED%86%A0%EB%A5%B4_%ED%8E%98%EB%A5%B4%EB%82%9C%EB%8D%B0%EC%8A%A4&amp;action=edit&amp;redlink=1" title="빅토르 페르난데스 (없는 문서)">빅토르 페르난데스</a> 감독의 지휘 아래 '짜르(러시아 황제)' <a href="/wiki/%EC%95%8C%EB%A0%89%EC%82%B0%EB%93%9C%EB%A5%B4_%EB%AA%A8%EC%8A%A4%ED%86%A0%EB%B3%B4%EC%9D%B4" title="알렉산드르 모스토보이">알렉산드르 모스토보이</a>를 중심으로 <a href="/wiki/%EB%B0%9C%EB%A0%88%EB%A6%AC_%EC%B9%B4%EB%A5%B4%ED%95%80" title="발레리 카르핀">발레리 카르핀</a>, <a class="new" href="/w/index.php?title=%EA%B5%AC%EC%8A%A4%ED%83%80%EB%B3%B4_%EB%A1%9C%ED%8E%98%EC%8A%A4&amp;action=edit&amp;redlink=1" title="구스타보 로페스 (없는 문서)">구스타보 로페스</a>, <a class="new" href="/w/index.php?title=%ED%81%B4%EB%A1%9C%EB%93%9C_%EB%A7%88%EC%BC%80%EB%A0%90%EB%A0%88&amp;action=edit&amp;redlink=1" title="클로드 마케렐레 (없는 문서)">클로드 마케렐레</a>, <a class="new" href="/w/index.php?title=%EB%AF%B8%EC%85%B8_%EC%82%B4%EA%B0%80%EB%8F%84&amp;action=edit&amp;redlink=1" title="미셸 살가도 (없는 문서)">미셸 살가도</a>, <a href="/wiki/%ED%8C%8C%EB%B8%94%EB%A1%9C_%EC%B9%B4%EB%B0%94%EC%98%88%EB%A1%9C" title="파블로 카바예로">파블로 카바예로</a>, <a class="new" href="/w/index.php?title=%ED%95%98%EC%9E%84_%EB%A5%B4%ED%94%BC%ED%8F%AC&amp;action=edit&amp;redlink=1" title="하임 르피포 (없는 문서)">하임 르피포</a>, 그리고 <a class="mw-redirect" href="/wiki/%EB%B2%A0%EB%8B%88_%EB%A7%A5%EC%B9%B4%EC%8B%9C" title="베니 맥카시">베니 맥카시</a> 등이 활약했다. 당시의 셀타 비고는 5위에서 7위권을 오가며 <a class="mw-redirect" href="/wiki/UEFA%EC%BB%B5" title="UEFA컵">UEFA컵</a>에 단골로 진출했고, 이로 인해 유로셀타 (EuroCelta, 에우로 셀타)라는 애칭을 얻기에 이르렀다.</p>
<p>98/99 시즌에는 레알 마드리드를 <a class="new" href="/w/index.php?title=%EB%B0%9C%EB%9D%BC%EC%9D%B4%EB%8F%84%EC%8A%A4&amp;action=edit&amp;redlink=1" title="발라이도스 (없는 문서)">발라이도스</a>에서 5:1로 대파했고 원정경기인 <a class="new" href="/w/index.php?title=%EB%B2%A0%EB%A5%B4%EB%82%98%EB%B2%A0%EC%9A%B0&amp;action=edit&amp;redlink=1" title="베르나베우 (없는 문서)">베르나베우</a>에서는 2:1로 승리했다. <a class="mw-redirect" href="/wiki/UEFA%EC%BB%B5" title="UEFA컵">UEFA컵</a> 에서는 아스톤빌라를 무려 7:0으로 이기는 파란을 연출하기도 했다.</p>
<p>99/00 시즌에는 초반에 선두로 올라서며 <a class="mw-disambig" href="/wiki/%EB%8D%B0%ED%8F%AC%EB%A5%B4%ED%8B%B0%EB%B3%B4" title="데포르티보">데포르티보</a>,<a href="/wiki/%EB%9D%BC%EC%9A%94_%EB%B0%94%EC%98%88%EC%B9%B4%EB%85%B8" title="라요 바예카노">라요 바예카노</a>,<a href="/wiki/%EB%B0%94%EB%A5%B4%EC%85%80%EB%A1%9C%EB%82%98" title="바르셀로나">바르셀로나</a>,<a href="/wiki/%EB%B0%9C%EB%A0%8C%EC%8B%9C%EC%95%84" title="발렌시아">발렌시아</a>,<a href="/wiki/%EC%82%AC%EB%9D%BC%EA%B3%A0%EC%82%AC" title="사라고사">사라고사</a>등과 함께 치열한 선두 다툼을 벌이기도 하는등 셀타 비고는 이제 그냥 그저 그런팀이 아닌 라 리가의 확실한 강호로 인정 받았다. 특히나 99/00 시즌 <a class="mw-redirect" href="/wiki/UEFA%EC%BB%B5" title="UEFA컵">UEFA컵</a>에서 셀타 비고는 다비즈, 델피에로, 인자기, 지단과 같은 특급 스타들이 즐비한 유벤투스를 무려 4:0으로 물리치는 이변을 연출했다. 이에 유벤투스 팬들은 "발라이도스에서 지옥을 보았다." 라고 까지 표현하면서 치를 떨었다고 한다.</p>
<p><br/>
1997/98 시즌 리그 6위에 오르며 <a class="mw-redirect" href="/wiki/UEFA%EC%BB%B5" title="UEFA컵">UEFA컵</a> 진출권을 획득한 셀타는 이듬 해 <a class="mw-redirect" href="/wiki/UEFA%EC%BB%B5" title="UEFA컵">UEFA컵</a> 8강에 올랐다. 98년, <a class="new" href="/w/index.php?title=%EC%85%80%ED%83%80&amp;action=edit&amp;redlink=1" title="셀타 (없는 문서)">셀타</a>는 <a href="/wiki/%EB%A6%AC%EB%B2%84%ED%92%80" title="리버풀">리버풀</a>과 <a class="mw-redirect" href="/wiki/%EC%95%84%EC%8A%A4%ED%86%A4_%EB%B9%8C%EB%9D%BC" title="아스톤 빌라">아스톤 빌라</a>를 차례로 무너뜨리며 <a class="mw-redirect" href="/wiki/UEFA%EC%BB%B5" title="UEFA컵">UEFA컵</a> 8강에 진출하는 기염을 토했다. 당시 셀타는 <a class="mw-redirect" href="/wiki/UEFA%EC%BB%B5" title="UEFA컵">UEFA컵</a>에서의 활약은 물론 <a class="new" href="/w/index.php?title=%ED%94%84%EB%A6%AC%EB%A9%94%EB%9D%BC_%EB%A6%AC%EA%B7%B8&amp;action=edit&amp;redlink=1" title="프리메라 리그 (없는 문서)">프리메라 리그</a>에서도 발군의 활약을 보였는데, 그들은 에스타디오 발라이도스(셀타의 홈 구장)에서 1년이 넘게 단 한번도 패하지 않을 정도로 막강한 팀이었다. 셀타는 당시 빼어난 활약을 보인 팀의 간판 <a class="new" href="/w/index.php?title=%EB%AA%A8%EC%8A%A4%ED%86%A0%EB%B3%B4%EC%9D%B4&amp;action=edit&amp;redlink=1" title="모스토보이 (없는 문서)">모스토보이</a>, <a class="mw-redirect" href="/wiki/%EB%A7%88%EC%BC%88%EB%A0%90%EB%A0%88" title="마켈렐레">마켈렐레</a>, <a class="new" href="/w/index.php?title=%EC%82%B4%EA%B0%80%EB%8F%84&amp;action=edit&amp;redlink=1" title="살가도 (없는 문서)">살가도</a>를 비롯해 <a class="mw-redirect" href="/wiki/1994_%EB%AF%B8%EA%B5%AD_%EC%9B%94%EB%93%9C%EC%BB%B5" title="1994 미국 월드컵">1994 미국 월드컵</a> 우승을 경험한 <a href="/wiki/%EB%B8%8C%EB%9D%BC%EC%A7%88" title="브라질">브라질</a> 출신 <a class="new" href="/w/index.php?title=%EB%A7%88%EC%A7%95%EC%9A%94&amp;action=edit&amp;redlink=1" title="마징요 (없는 문서)">마징요</a>까지 보유하고 있을 정도로 강력한 선수단을 보유했었다. 98년 시즌이 진행되는 동한 4위권 밖을 거의 벗어나지 않을 정도로 안정된 팀이었지만, 시즌 마지막 경기에서 통한의 패배를 당하며 <a href="/wiki/UEFA_%EC%B1%94%ED%94%BC%EC%96%B8%EC%8A%A4%EB%A6%AC%EA%B7%B8" title="UEFA 챔피언스리그">UEFA 챔피언스리그</a> 티켓을 놓치고 만다. 그 이후 <a class="new" href="/w/index.php?title=%EC%85%80%ED%83%80&amp;action=edit&amp;redlink=1" title="셀타 (없는 문서)">셀타</a>는 몇몇의 주전급 선수들이 은퇴, 혹은 빅클럽으로 이적하며 전력 손실이 예상 됐지만 3,000만 유로가 넘는 당시로서는 어마어마한 액수를 투자하며 <a class="new" href="/w/index.php?title=%EB%A3%A8%EC%9D%B4%EC%8A%A4_%EC%97%90%EB%91%90&amp;action=edit&amp;redlink=1" title="루이스 에두 (없는 문서)">루이스 에두</a>, <a class="new" href="/w/index.php?title=%ED%97%A4%EC%88%A0%EB%A6%AC&amp;action=edit&amp;redlink=1" title="헤술리 (없는 문서)">헤술리</a>, <a class="new" href="/w/index.php?title=%EC%B9%B4%ED%83%80%EB%83%90&amp;action=edit&amp;redlink=1" title="카타냐 (없는 문서)">카타냐</a>, <a class="mw-redirect" href="/wiki/%EC%B9%B4%EB%A5%B4%ED%95%80" title="카르핀">카르핀</a>, <a class="mw-redirect" href="/wiki/%EB%B2%A0%EB%8B%88_%EB%A7%A5%EC%B9%B4%EC%8B%9C" title="베니 맥카시">베니 맥카시</a> 등을 영입하며 선수단 보강을 성공적으로 마친다.</p>
<p><br/>
2002/03 시즌에는 리그 4위로 <a class="mw-disambig" href="/wiki/%EC%B1%94%ED%94%BC%EC%96%B8%EC%8A%A4%EB%A6%AC%EA%B7%B8" title="챔피언스리그">챔피언스리그</a>에 진출하는 기염을 토했다. 2003-04 시즌 <a class="mw-redirect mw-disambig" href="/wiki/%EC%B1%94%EC%8A%A4" title="챔스">챔스</a>에서도 셀타는 <a class="mw-disambig" href="/wiki/%EC%B1%94%ED%94%BC%EC%96%B8%EC%8A%A4%EB%A6%AC%EA%B7%B8" title="챔피언스리그">챔피언스리그</a> 진출에만 만족하지 않고 32강전마저 통과하며 16강에서 <a href="/wiki/%ED%94%84%EB%A6%AC%EB%AF%B8%EC%96%B4%EB%A6%AC%EA%B7%B8" title="프리미어리그">프리미어리그</a>의 강자 <a class="mw-redirect" href="/wiki/%EC%95%84%EC%8A%A4%EB%82%A0" title="아스날">아스날</a>을 만난다. 그들은 비록 <a class="mw-redirect" href="/wiki/%EC%95%84%EC%8A%A4%EB%82%A0" title="아스날">아스날</a>에게 패하며 8강 진출에는 실패하지만 <a class="mw-disambig" href="/wiki/%EC%B1%94%ED%94%BC%EC%96%B8%EC%8A%A4%EB%A6%AC%EA%B7%B8" title="챔피언스리그">챔피언스리그</a>에서의 선전으로 1,400만 유로의 수입을 올렸다. 그러나 당시 <a class="new" href="/w/index.php?title=%EC%85%80%ED%83%80&amp;action=edit&amp;redlink=1" title="셀타 (없는 문서)">셀타</a> 구단과 선수들은 큰 무대 경험이 전무했다. 결국 그들은 <a class="mw-disambig" href="/wiki/%EC%B1%94%ED%94%BC%EC%96%B8%EC%8A%A4%EB%A6%AC%EA%B7%B8" title="챔피언스리그">챔피언스리그</a>에서의 활약과는 달리 <a class="new" href="/w/index.php?title=%ED%94%84%EB%A6%AC%EB%A9%94%EB%9D%BC%EB%A6%AC%EA%B0%80%EC%99%80&amp;action=edit&amp;redlink=1" title="프리메라리가와 (없는 문서)">프리메라리가와</a> <a class="mw-disambig" href="/wiki/%EC%B1%94%ED%94%BC%EC%96%B8%EC%8A%A4%EB%A6%AC%EA%B7%B8" title="챔피언스리그">챔피언스리그</a>를 병행하는 혹독한 일정을 이겨내지 못하고 체력문제를 노출하며 같은 시즌에 <a class="mw-redirect" href="/wiki/%ED%94%84%EB%A6%AC%EB%A9%94%EB%9D%BC%EB%A6%AC%EA%B0%80" title="프리메라리가">프리메라리가</a>를 19위로 마감하며 충격의 강등을 당하게 된다. 아무런 준비도 없이 출전한 <a class="mw-disambig" href="/wiki/%EC%B1%94%ED%94%BC%EC%96%B8%EC%8A%A4%EB%A6%AC%EA%B7%B8" title="챔피언스리그">챔피언스리그</a>에 총력을 기울인 <a class="new" href="/w/index.php?title=%EC%85%80%ED%83%80&amp;action=edit&amp;redlink=1" title="셀타 (없는 문서)">셀타</a>는 정작 리그에서 처참한 결과와 함께 2부리그로 강등당한 것이다.</p>
<p><br/>
2005년 <a class="new" href="/w/index.php?title=%EC%85%80%ED%83%80&amp;action=edit&amp;redlink=1" title="셀타 (없는 문서)">셀타</a>는 <a class="mw-redirect" href="/wiki/%EC%84%B8%EA%B5%B0%EB%8B%A4_%EB%94%94%EB%B9%84%EC%A0%84" title="세군다 디비전">세군다 디비전</a> 준우승을 차지하며 <a class="mw-redirect" href="/wiki/%ED%94%84%EB%A6%AC%EB%A9%94%EB%9D%BC%EB%A6%AC%EA%B0%80" title="프리메라리가">프리메라리가</a>로 승격한다. 이후 구단은 재정난을 겪고 있는 현실에도 불구하고 다시 한번 유럽무대 재진출을 목표로 하며 팀의 주축 멤버이자 우루과이 출신 플레이메이커 바이아노를 잔류 시키기 위해 200만 유로를 투자했고, 이외에도 과거 선수단에 버금가는 팀을 구축하기 위해 위험한 투자를 아끼지 않았다. 결국 셀타는 2005-2006시즌을 6위로 마감하며 <a class="mw-redirect" href="/wiki/UEFA%EC%BB%B5" title="UEFA컵">UEFA컵</a>에 진출하는 기쁨을 맛보지만. 또 다시 유럽 대항전과 라 리가를 병행하다 강등 당하는 모습을 보였다. 당시 셀타 구단은 5,000만 유로 이상의 부채에 시달리며 재정난에 허덕였고, <a class="mw-disambig" href="/wiki/%EC%B1%94%ED%94%BC%EC%96%B8%EC%8A%A4%EB%A6%AC%EA%B7%B8" title="챔피언스리그">챔피언스리그</a> 진출을 위해 무리한 투자를 했던 98년과 같은 어리석은 실수를 반복한 2006년에 또다시 유럽무대와 프리메라리가 병행에 철저히 실패하며 2년 만에 두 번째 강등을 당하게 된다. 셀타는 <a class="mw-redirect" href="/wiki/UEFA%EC%BB%B5" title="UEFA컵">UEFA컵</a>에서도 16강에 진출했지만 <a class="mw-redirect" href="/wiki/%EB%B6%84%EB%8D%B0%EC%8A%A4%EB%A6%AC%EA%B0%80" title="분데스리가">분데스리가</a> 명문 <a class="mw-redirect" href="/wiki/%EB%B2%A0%EB%A5%B4%EB%8D%94_%EB%B8%8C%EB%A0%88%EB%A9%98" title="베르더 브레멘">베르더 브레멘</a>에게 패했으며, <a class="new" href="/w/index.php?title=%ED%94%84%EB%A6%AC%EB%A9%94%EB%9D%BC%EB%A6%AC%EA%B7%B8&amp;action=edit&amp;redlink=1" title="프리메라리그 (없는 문서)">프리메라리그</a>에서 역시 지역 라이벌 <a class="mw-redirect" href="/wiki/%EB%8D%B0%ED%8F%AC%EB%A5%B4%ED%8B%B0%EB%B3%B4_%EB%9D%BC_%EC%BD%94%EB%A3%A8%EB%83%90" title="데포르티보 라 코루냐">데포르티보 라 코루냐</a>를 상대로 거둔 원정 승리를 제외하고는 별다른 성과를 올리지 못했다.</p>
<p><br/>
<a class="new" href="/w/index.php?title=%EC%85%80%ED%83%80&amp;action=edit&amp;redlink=1" title="셀타 (없는 문서)">셀타</a>의 전성기를 주도한 인물로는 <a class="new" href="/w/index.php?title=%EB%B9%85%ED%86%A0%EB%A5%B4_%ED%8E%98%EB%A5%B4%EB%82%9C%EB%8D%B0%EC%8A%A4&amp;action=edit&amp;redlink=1" title="빅토르 페르난데스 (없는 문서)">빅토르 페르난데스</a> 감독을 비롯, '짜르' <a class="new" href="/w/index.php?title=%EB%AA%A8%EC%8A%A4%ED%86%A0%EB%B3%B4%EC%9D%B4&amp;action=edit&amp;redlink=1" title="모스토보이 (없는 문서)">모스토보이</a>와 <a class="new" href="/w/index.php?title=%EA%B5%AC%EC%8A%A4%ED%83%80%EB%B3%B4_%EB%A1%9C%ED%8E%98%EC%8A%A4&amp;action=edit&amp;redlink=1" title="구스타보 로페스 (없는 문서)">구스타보 로페스</a>, <a class="new" href="/w/index.php?title=%ED%8E%98%ED%85%8C_%EB%A4%BC%EC%8C%A9&amp;action=edit&amp;redlink=1" title="페테 뤼쌩 (없는 문서)">페테 뤼쌩</a> 등이 손꼽힌다. <a class="mw-redirect" href="/wiki/%EB%A7%88%EC%BC%88%EB%A0%90%EB%A0%88" title="마켈렐레">마켈렐레</a>와 <a href="/wiki/%EB%8B%A4%EB%B9%84%EB%93%9C_%EC%8B%A4%EB%B0%94" title="다비드 실바">다비드 실바</a> 등 우리에게 익숙한 수많은 스타 플레이어들이 <strong class="selflink">셀타 비고</strong>를 거쳐갔다. 불과 10년 전까지만 하더라도 <a class="mw-redirect" href="/wiki/%EB%9D%BC_%EB%A6%AC%EA%B0%80" title="라 리가">라 리가</a>를 대표하는 상위권 팀으로 유명했다.</p>
<h3><span class="mw-headline" id=".EC.A0.84.EC.84.B1.EA.B8.B0_.EC.A3.BC.EC.9A.94_.EC.84.B1.EC.A0.81_.EC.9A.94.EC.95.BD">전성기 주요 성적 요약</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%85%80%ED%83%80_%EB%B9%84%EA%B3%A0&amp;action=edit&amp;section=5" title="부분 편집: 전성기 주요 성적 요약">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<ul>
<li>1992/93 시즌 1부리그 승격</li>
<li>2000년 UEFA 인터 토토컵 우승</li>
<li>2001/02 시즌 라리가 5위</li>
<li>2002/03 시즌 라리가 4위</li>
<li>2002/03 시즌 UEFA챔피언스리그 16강</li>
<li>UEFA컵 3년 연속 8강</li>
</ul>
<p><br/></p>
<h3><span class="mw-headline" id="2012.2F13_.EB.9D.BC_.EB.A6.AC.EA.B0.80.EB.A1.9C.EC.9D.98_.EB.B3.B5.EA.B7.80">2012/13 라 리가로의 복귀</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%85%80%ED%83%80_%EB%B9%84%EA%B3%A0&amp;action=edit&amp;section=6" title="부분 편집: 2012/13 라 리가로의 복귀">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>셀타는 천문학적인 부채에 따른 선수 영입의 실패, 주축선수들의 노쇠화로 인해 2006/07 시즌 불운의 강등을 맞이했다. 이후 5시즌을 <a class="new" href="/w/index.php?title=%EC%84%B8%EA%B5%B0%EB%8B%A4_%EB%A6%AC%EA%B0%80&amp;action=edit&amp;redlink=1" title="세군다 리가 (없는 문서)">세군다 리가</a>에서 보낸 <strong class="selflink">셀타 비고</strong>는 2010년 <a href="/wiki/%EB%9D%BC%ED%8C%8C%EC%97%98_%EB%B2%A0%EB%8B%88%ED%85%8C%EC%8A%A4" title="라파엘 베니테스">라파엘 베니테스</a> 아래에서 <a href="/wiki/%EB%A6%AC%EB%B2%84%ED%92%80" title="리버풀">리버풀</a> 수석 코치 역할도 수행했던 <a class="new" href="/w/index.php?title=%ED%8C%8C%EC%BD%94_%EC%97%90%EB%A0%88%EB%9D%BC&amp;action=edit&amp;redlink=1" title="파코 에레라 (없는 문서)">파코 에레라</a> 감독 부임 후 차근차근 전력을 다져나갔고, 유스팀에 상당히 많은 신경을 쏟았다. 승격 역시 지난 5년동안의 노력이 결실을 맺었다고 할 수있다. 1군의 80%가 셀타의 연고지 지방인 갈리시아 출신이였고 현재 베스트 11의 절반이 유스팀 출신이거나 갈리시아 지방 출신이다. 셀타의 유스힘은 서서히 빛을 발하고 있는데, 2011/12 시즌 라요에서 좋은 활약을 보이며 12/13시즌에 <a class="mw-redirect" href="/wiki/%EC%8A%A4%EC%99%84%EC%A7%80_%EC%8B%9C%ED%8B%B0" title="스완지 시티">스완지 시티</a>로 이적한 <a class="mw-disambig" href="/wiki/%EB%AF%B8%EC%B6%94" title="미추">미추</a> 역시 셀타에서 프로 생활을 시작했다.</p>
<p><br/></p>
<ul>
<li>2007/08 시즌 : 세군다 리가 16위</li>
<li>2008/09 시즌 : 세군다 리가 17위</li>
<li>2009/10 시즌 : 세군다 리가 12위</li>
<li>2010/11 시즌 : 세군다 리가 6위</li>
<li>2011/12 시즌 : 세군다 리가 2위 (1위는 라이벌팀인 <a class="mw-redirect" href="/wiki/%EB%8D%B0%ED%8F%AC%EB%A5%B4%ED%8B%B0%EB%B3%B4_%EB%9D%BC_%EC%BD%94%EB%A3%A8%EB%83%90" title="데포르티보 라 코루냐">데포르티보 라 코루냐</a>)</li>
</ul>
<p><br/>
리그 순위에서 보이다 시피 차근차근 발전 해 왔다. 세군다 리가에서 한 시즌 돌풍을 일으키며 전력이 약한 팀들과는 다르다는 것을 알 수있다. 유스팀을 중심으로 5년이라는 적지 않은 시간 동안 팀을 리빌딩 해 왔기 때문이다. 11/12 시즌 세군다 리가에서 승격한 팀들(데포르티보, 셀타, 바야돌리드)은 세군다 리가 역사상 가장 치열한 승격 싸움을 벌이고 올라왔으며 라리가에서도 현재 좋은 경기력을 선보이고 있다.</p>
<p>셀타의 경우에는 이번 시즌보다 2년~3년후가 더 기대되는 팀이다. 선수단의 대부분이 라리가 경험이 없기 때문에 12/13 시즌은 선수들이 라리가의 적응 및 경험을 쌓는 것이 목적이다. 또한 베스트11의 절반이 20대 초중반으로 성장이 가능한 연령대라는것이 고무적이다. 그렇기에 12/13 시즌 잔류에 성공한다면 기존 선수들의 기량 향상 및 질높은 영입등으로 다음 시즌은 한층 더 강해진 모습을 보여줄 수 있는 팀이다.</p>

<h3><span class="mw-headline" id=".EC.9C.A0.EB.AA.85_.EC.84.A0.EC.88.98">유명 선수</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%85%80%ED%83%80_%EB%B9%84%EA%B3%A0&amp;action=edit&amp;section=8" title="부분 편집: 유명 선수">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>러시아가 낳은 두 축구 천재인 <a href="/wiki/%EC%95%8C%EB%A0%89%EC%82%B0%EB%93%9C%EB%A5%B4_%EB%AA%A8%EC%8A%A4%ED%86%A0%EB%B3%B4%EC%9D%B4" title="알렉산드르 모스토보이">알렉산드르 모스토보이</a>와 <a href="/wiki/%EB%B0%9C%EB%A0%88%EB%A6%AC_%EC%B9%B4%EB%A5%B4%ED%95%80" title="발레리 카르핀">발레리 카르핀</a>을 비롯해 많은 축구 팬들에게 익숙한 선수들로는 수비형 미드필더중 최고로 손꼽히는 <a class="new" href="/w/index.php?title=%ED%81%B4%EB%A1%9C%EB%93%9C_%EB%A7%88%EC%BC%88%EC%BC%88%EB%A0%88&amp;action=edit&amp;redlink=1" title="클로드 마켈켈레 (없는 문서)">클로드 마켈켈레</a>, 블랙번 소속으로 좋은 활약을 펼쳤던 <a class="new" href="/w/index.php?title=%EB%B2%A0%EB%8B%88_%EB%A9%95%EC%B9%B4%EC%8B%9C&amp;action=edit&amp;redlink=1" title="베니 멕카시 (없는 문서)">베니 멕카시</a>, 이영표와 토튼햄에서 한솥밥을 먹었던 <a href="/wiki/%EB%AF%B8%EB%8F%84" title="미도">미도</a> 그리고 현재 바르셀로나에서 뛰고 있는 <a class="mw-redirect" href="/wiki/%ED%95%80%ED%88%AC" title="핀투">핀투</a> 역시 셀타를 거쳤다. 이외에도 아르헨티나 국가대표를 지냈던 <a href="/wiki/%ED%8C%8C%EB%B8%94%EB%A1%9C_%EC%B9%B4%EB%B0%94%EC%98%88%EB%A1%9C" title="파블로 카바예로">파블로 카바예로</a>, <a class="new" href="/w/index.php?title=%EA%B5%AC%EC%8A%A4%ED%83%80%EB%B3%B4_%EB%A1%9C%ED%8E%98%EC%8A%A4&amp;action=edit&amp;redlink=1" title="구스타보 로페스 (없는 문서)">구스타보 로페스</a>, 바르샤의 두 유망주 <a class="mw-redirect" href="/wiki/%EB%9D%BC%ED%8C%8C_%EC%95%8C%EC%B9%B8%ED%83%80%EB%9D%BC" title="라파 알칸타라">라파 알칸타라</a>와 <a href="/wiki/%ED%8B%B0%EC%95%84%EA%B3%A0_%EC%95%8C%EC%B9%B8%ED%83%80%EB%9D%BC" title="티아고 알칸타라">티아고 알칸타라</a>의 아버지로 잘 알려진 <a class="new" href="/w/index.php?title=%EB%A7%88%EC%A7%95%EC%9A%94&amp;action=edit&amp;redlink=1" title="마징요 (없는 문서)">마징요</a>까지도 역시 셀타에 몸 담았고 이스라엘이 낳은 테크니션 레비보 또한 그랬다. <a
