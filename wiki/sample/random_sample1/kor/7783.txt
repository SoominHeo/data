<p><b>셰이크 하시나</b>(<span style="font-size: smaller;"><a href="/wiki/%EB%B2%B5%EA%B3%A8%EC%96%B4" title="벵골어">벵골어</a>:</span> <span lang="bn" xml:lang="bn">শেখ হাসিনা ওয়াজেদ</span>, <span style="font-size: smaller;"><a href="/wiki/%EC%98%81%EC%96%B4" title="영어">영어</a>:</span> <span lang="en" xml:lang="en">Sheikh Hasina</span>, <a href="/wiki/1947%EB%85%84" title="1947년">1947년</a> <a href="/wiki/9%EC%9B%94_28%EC%9D%BC" title="9월 28일">9월 28일</a>~)는 <a href="/wiki/2009%EB%85%84" title="2009년">2009년</a> <a href="/wiki/1%EC%9B%94_6%EC%9D%BC" title="1월 6일">1월 6일</a> 이후 <a href="/wiki/%EB%B0%A9%EA%B8%80%EB%9D%BC%EB%8D%B0%EC%8B%9C" title="방글라데시">방글라데시</a>의 수상을 역임하고 있다. 또한 <a href="/wiki/1996%EB%85%84" title="1996년">1996년</a>에서 <a href="/wiki/2001%EB%85%84" title="2001년">2001년</a>까지 수상을 역임하기도 했다. 하시나는 <a href="/wiki/1981%EB%85%84" title="1981년">1981년</a>부터 <a class="mw-redirect" href="/wiki/%EB%B0%A9%EA%B8%80%EB%9D%BC%EB%8D%B0%EC%8B%9C_%EC%95%84%EC%99%80%EB%AF%B8_%EC%97%B0%EB%A7%B9" title="방글라데시 아와미 연맹">방글라데시 아와미 연맹</a>을 이끌어 왔다. 방글라데시 독립 운동의 아버지이며, 건국의 아버지 그리고 초대 대통령인 <a href="/wiki/%EC%85%B0%EC%9D%B4%ED%81%AC_%EB%AC%B4%EC%A7%80%EB%B6%80%EB%A5%B4_%EB%9D%BC%ED%9D%90%EB%A7%8C" title="셰이크 무지부르 라흐만">셰이크 무지부르 라흐만</a>의 장녀이기도 한다. 방글라데시 핵무기 과학자인 M. A. 와제드 마아흐와 결혼하였으며, 현재는 미망인이다. 하시나가 이끄는 정당은 2008 방글라데시 총선에서 BNP 주도 4당 연합을 물리치고 수상을 자리를 차지했다.</p>
<p>그녀는 40년 이상의 정치적 경력을 쌓은 야당의 지도자이자, 이미 수상을 역임하기도 했다. 야당지도자로서 <a href="/wiki/2004%EB%85%84" title="2004년">2004년</a> 암살의 표적이 되기도 했다. 2006년-2008년 방글라데시 정치 위기 중, 2007년에는 군부의 <a class="mw-redirect" href="/wiki/%EA%B5%AD%EA%B0%80%EB%B9%84%EC%83%81%EC%82%AC%ED%83%9C" title="국가비상사태">국가비상사태</a> 선포로 군부를 등에 업은 카레타커 정부로부터 부패혐의와 살인교사 혐으로 체포되기도 했다.</p>
<p>그녀는 2008년 총선에서 아와미 연맹 주도의 대연정으로 총선에서 의석의 2/3을 차지하는 압승을 거두며, 다시 복귀를 하였다.</p>
<h2><span class="mw-headline" id=".EC.83.9D.EC.95.A0">생애</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%85%B0%EC%9D%B4%ED%81%AC_%ED%95%98%EC%8B%9C%EB%82%98&amp;action=edit&amp;section=1" title="부분 편집: 생애">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p><a href="/wiki/%ED%8C%8C%ED%82%A4%EC%8A%A4%ED%83%84" title="파키스탄">파키스탄</a>에서 방글라데시의 독립을 완수한 독립의 아버지이며, 건국의 아버지인 <a class="mw-redirect" href="/wiki/%EB%AC%B4%EC%A7%80%EB%B6%80%EB%A5%B4_%EB%9D%BC%ED%9D%90%EB%A7%8C" title="무지부르 라흐만">무지부르 라흐만</a>의 장녀로 <a href="/wiki/1975%EB%85%84" title="1975년">1975년</a>에 쿠데타로 아버지가 가족과 함께 암살되었을 때 <a href="/wiki/%EC%98%81%EA%B5%AD" title="영국">영국</a>에 있어 살아 남았다. 이후 <a href="/wiki/%EC%9D%B8%EB%8F%84" title="인도">인도</a>에서 망명 생활을 했지만, <a href="/wiki/1981%EB%85%84" title="1981년">1981년</a>에 귀국하였으며, <a href="/wiki/%EC%95%84%EC%99%80%EB%AF%B8_%EC%97%B0%EB%A7%B9" title="아와미 연맹">아와미 연맹</a>의 당수가 되었다. 그런 다음 <a href="/wiki/1983%EB%85%84" title="1983년">1983년</a>에 <a href="/wiki/%ED%9B%84%EC%84%B8%EC%9D%B8_%EB%AC%B4%ED%95%98%EB%A9%94%EB%93%9C_%EC%97%90%EB%A5%B4%EC%83%A4%EB%93%9C" title="후세인 무하메드 에르샤드">후세인 무하메드 에르샤드</a>가 쿠데타로 정권을 잡고 대통령이되면서 아와미 연맹을 이끌고 이 정권에 대한 반대 운동을 전개하였으며, 따라서 오랜 기간동안 가택연금 생활을 하였다.</p>
<p>에르샤드 대통령이 이끄는 군사 독재 정권이 무너진 후, <a href="/wiki/1991%EB%85%84" title="1991년">1991년</a> 선거에서 경쟁하던, <a href="/wiki/%EC%B9%BC%EB%A0%88%EB%8B%A4_%EC%A7%80%EC%95%84" title="칼레다 지아">칼레다 지아</a>가 이끄는 <a href="/wiki/%EB%B0%A9%EA%B8%80%EB%9D%BC%EB%8D%B0%EC%8B%9C_%EB%AF%BC%EC%A1%B1%EC%A3%BC%EC%9D%98%EB%8B%B9" title="방글라데시 민족주의당">방글라데시 민족주의당</a>에 패했지만, 그 다음 <a href="/wiki/1996%EB%85%84" title="1996년">1996년</a> 의회 선거에서 승리해 총리에 취임했다. 그러나 2001년 의회 선거에서 다시 <a href="/wiki/%EC%B9%BC%EB%A0%88%EB%8B%A4_%EC%A7%80%EC%95%84" title="칼레다 지아">칼레다 지아</a>의 민족주의당에 패배해 물러나게 되었다. <a href="/wiki/2007%EB%85%84" title="2007년">2007년</a>에 예정된 다음 선거는 <a class="mw-redirect" href="/wiki/%EA%B5%AD%EA%B0%80%EB%B9%84%EC%83%81%EC%82%AC%ED%83%9C" title="국가비상사태">국가비상사태</a> 선언으로 연기되었고, 하시나도 공갈 혐의로 수사를 받는 이례적인 사태가 일어났지만, 결국 선거는 <a href="/wiki/2008%EB%85%84" title="2008년">2008년</a> 12월에 실시되었고, 아와미 연맹을 중심으로 에르샤드 등을 결집한 정당 연합 ‘대연합’이 압승을 거두고, <a href="/wiki/2009%EB%85%84" title="2009년">2009년</a> 1월에 하시나가 다시 총리에 취임했다.</p>
