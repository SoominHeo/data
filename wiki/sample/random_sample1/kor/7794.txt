<div class="mw-body-content" id="bodyContent">

<p><b>소그디아나</b> 또는 <b>소그디아</b>(Суғд , 수구다, سغد, Σογδιανῆ)는 <a href="/wiki/%EC%9D%B4%EB%9E%80" title="이란">이란</a>계 민족(<a href="/wiki/%EC%86%8C%EA%B7%B8%EB%93%9C%EC%9D%B8" title="소그드인">소그드인</a>)이 일으킨 고대 문명권이다. 고대 한자문헌은 속특(粟特)이라고 표기했다.</p>
<p><a href="/wiki/%EB%8B%A4%EB%A6%AC%EC%9A%B0%EC%8A%A4_1%EC%84%B8" title="다리우스 1세">다리우스 1세</a>의 <a href="/wiki/%EB%B2%A0%ED%9E%88%EC%8A%A4%ED%88%B0_%EB%B9%84%EB%AC%B8" title="베히스툰 비문">베히스툰 비문</a>에 기록된 바와 같이 <a class="mw-redirect" href="/wiki/%EC%95%84%EC%BC%80%EB%A9%94%EB%84%A4%EC%8A%A4_%EC%99%95%EC%A1%B0" title="아케메네스 왕조">아케메네스 조 페르시아 제국</a>의 속주였으며, <a href="/wiki/%EC%A1%B0%EB%A1%9C%EC%95%84%EC%8A%A4%ED%84%B0%EA%B5%90" title="조로아스터교">조로아스터교</a> <a href="/wiki/%EB%B2%A4%EB%94%94%EB%8B%A4%EB%93%9C" title="벤디다드">벤디다드</a> 기록에서 <a href="/wiki/%EC%95%84%ED%9B%84%EB%9D%BC_%EB%A7%88%EC%A6%88%EB%8B%A4" title="아후라 마즈다">아후라 마즈다</a>가 창조한 열여섯 지역 중 두 번째(Gava) 지역도 이곳으로 비정(比定)되고 있다. 소그디아나는 여러 시기를 걸쳐 <a class="mw-redirect" href="/wiki/%EC%82%AC%EB%A7%88%EB%A5%B4%EC%B9%B8%EB%93%9C" title="사마르칸드">사마르칸드</a>, <a href="/wiki/%EB%B6%80%ED%95%98%EB%9D%BC" title="부하라">부하라</a>, 케슈Кеш(현재의 샤흐리사브즈Shahrisabz), <a href="/wiki/%EC%88%98%EA%B7%B8%EB%93%9C_%EC%A3%BC" title="수그드 주">수그드 주</a>의 <a href="/wiki/%ED%9B%84%EC%9E%94%ED%8A%B8" title="후잔트">후잔트</a>, <a href="/wiki/%ED%8C%90%EC%9E%90%EC%BC%84%ED%8A%B8" title="판자켄트">판자켄트</a>를 포함하는 오늘날의 <a href="/wiki/%EC%9A%B0%EC%A6%88%EB%B2%A0%ED%82%A4%EC%8A%A4%ED%83%84" title="우즈베키스탄">우즈베키스탄</a>과 <a href="/wiki/%ED%83%80%EC%A7%80%ED%82%A4%EC%8A%A4%ED%83%84" title="타지키스탄">타지키스탄</a> 지역을 지칭했다.</p>
<p>소그드 국가들은 끝내 정치적으로 통합하지는 못하였지만, 주요도시였던 마라칸다(<a class="mw-redirect" href="/wiki/%EC%82%AC%EB%A7%88%EB%A5%B4%EC%B9%B8%EB%93%9C" title="사마르칸드">사마르칸드</a>)를 중심으로 결집하고 있었다. 또한 <a href="/wiki/%EC%A4%91%EC%95%99%EC%95%84%EC%8B%9C%EC%95%84" title="중앙아시아">중앙아시아</a> <a class="mw-redirect" href="/wiki/%EC%8B%A4%ED%81%AC%EB%A1%9C%EB%93%9C" title="실크로드">실크로드</a>의 요충지이기 때문에 주변 세력의 침입이 현저하게 더 많아 <a href="/wiki/%EC%9B%94%EC%A7%80" title="월지">월지</a>, <a href="/wiki/%EA%B0%95%EA%B1%B0" title="강거">강거</a>, <a class="mw-redirect" href="/wiki/%EC%BF%A0%EC%83%A8_%EC%99%95%EC%A1%B0" title="쿠샨 왕조">쿠샨 왕조</a>, <a href="/wiki/%EC%97%90%ED%94%84%ED%83%88" title="에프탈">에프탈</a>, <a href="/wiki/%EB%8F%8C%EA%B6%90" title="돌궐">돌궐</a> 등의 유목 국가나 <a class="mw-disambig" href="/wiki/%ED%8E%98%EB%A5%B4%EC%8B%9C%EC%95%84" title="페르시아">페르시아</a>의 <a class="mw-redirect" href="/wiki/%EC%82%AC%EC%82%B0_%EC%99%95%EC%A1%B0" title="사산 왕조">사산 왕조</a>의 지배를 받았다.</p>
<p>소그디아나는 <a href="/wiki/%EB%B0%95%ED%8A%B8%EB%A6%AC%EC%95%84" title="박트리아">박트리아</a> 북부에서 <a class="mw-redirect" href="/wiki/%ED%98%B8%EB%9D%BC%EC%A6%98" title="호라즘">호라즘</a>의 동부, <a href="/wiki/%EA%B0%95%EA%B1%B0" title="강거">강거</a>의 남동부에 이르렀으며, 비옥한 자라프샨(고대 희랍지명 폴리티메투스, 고대 중국지명 나밀수那密水) 계곡을 아우른 <a class="mw-redirect" href="/wiki/%EC%98%A5%EC%88%98%EC%8A%A4_%EA%B0%95" title="옥수스 강">옥수스 강</a>(<a href="/wiki/%EC%95%84%EB%AC%B4%EB%8B%A4%EB%A6%AC%EC%95%BC_%EA%B0%95" title="아무다리야 강">아무다리야 강</a>)과 야크사르 강(<a href="/wiki/%EC%8B%9C%EB%A5%B4%EB%8B%A4%EB%A6%AC%EC%95%BC_%EA%B0%95" title="시르다리야 강">시르다리야 강</a>) 사이의 지역에 해당하였다. 이는 현 <a href="/wiki/%EC%9A%B0%EC%A6%88%EB%B2%A0%ED%82%A4%EC%8A%A4%ED%83%84" title="우즈베키스탄">우즈베키스탄</a> 행정구역의 <a class="mw-redirect" href="/wiki/%EC%82%AC%EB%A7%88%EB%A5%B4%EC%B9%B8%EB%93%9C_%EC%A3%BC" title="사마르칸드 주">사마르칸드 주</a>와 <a href="/wiki/%EB%B6%80%ED%95%98%EB%9D%BC_%EC%A3%BC" title="부하라 주">부하라 주</a>, 그리고 <a href="/wiki/%ED%83%80%EC%A7%80%ED%82%A4%EC%8A%A4%ED%83%84" title="타지키스탄">타지키스탄</a> 행정구역의 <a href="/wiki/%EC%88%98%EA%B7%B8%EB%93%9C_%EC%A3%BC" title="수그드 주">수그드 주</a>와 대체로 일치한다.</p>
<p>이후 <a href="/wiki/8%EC%84%B8%EA%B8%B0" title="8세기">8세기</a>경에 <a href="/wiki/%EC%95%84%EB%9E%8D%EC%9D%B8" title="아랍인">아랍인</a>에 의해 정복되어 <a class="mw-redirect" href="/wiki/%EC%9D%B4%EC%8A%AC%EB%9E%8C" title="이슬람">이슬람</a>을 수용했다. 후에는 스텝 지역에 있었던 <a class="mw-redirect" href="/wiki/%ED%88%AC%EB%A5%B4%ED%81%AC%EC%A1%B1" title="투르크족">투르크족</a>의 서진에 따라 투르크화가 진행되었다.</p>
