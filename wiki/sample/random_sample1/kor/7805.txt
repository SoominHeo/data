<h1 class="firstHeading" id="firstHeading" lang="ko">소닉 2017 프로젝트</h1>
<div class="mw-body-content" id="bodyContent">
<div id="siteSub">위키백과, 우리 모두의 백과사전.</div>
<div id="contentSub"></div>
<div class="mw-jump" id="jump-to-nav">
					이동:					<a href="#mw-head">둘러보기</a>, 					<a href="#p-search">검색</a>
</div>
<div class="mw-content-ltr" dir="ltr" id="mw-content-text" lang="ko"><script>function mfTempOpenSection(id){var block=document.getElementById("mf-section-"+id);block.className+=" open-block";block.previousSibling.className+=" open-block";}</script><p><b>소닉 2017 프로젝트</b>는 <a href="/wiki/2017%EB%85%84" title="2017년">2017년</a>에 발매될 예정인 <a class="mw-redirect" href="/wiki/%EC%86%8C%EB%8B%89_%EB%8D%94_%ED%97%A4%EC%A7%80%ED%98%B9_%EC%8B%9C%EB%A6%AC%EC%A6%88" title="소닉 더 헤지혹 시리즈">소닉 더 헤지혹 시리즈</a>의 신작이다. <a class="new" href="/w/index.php?title=%EC%9D%B4%EC%9D%B4%EC%A6%88%EC%B9%B4_%ED%83%80%EC%B9%B4%EC%8B%9C&amp;action=edit&amp;redlink=1" title="이이즈카 타카시 (없는 문서)">이이즈카 타카시</a>에 의해 시리즈 25주년 예고가 알려지면서 서서히 존재가 알려졌다. 개발 중인 <a href="/wiki/%EB%B9%84%EB%94%94%EC%98%A4_%EA%B2%8C%EC%9E%84" title="비디오 게임">게임</a>의 명칭은 아직 미공개이며, 본 문서의 임시 제목인 '소닉 2017 프로젝트'는 6월에 일본에서 개최된 '소닉 25주년 애니버서리 파티'에서 공개된 이름이다. 영미권에서는 '프로젝트 소닉 2017(Project Sonic 2017)'이라 부른다. 이후 1개월 뒤인 <a href="/wiki/7%EC%9B%94_23%EC%9D%BC" title="7월 23일">7월 23일</a>에 <a href="/wiki/%EC%83%8C%EB%94%94%EC%97%90%EC%9D%B4%EA%B3%A0" title="샌디에이고">샌디에이고</a>에서 열린 파티에서 첫 트레일러 영상이 공개되었다. 게임 제작은 <a href="/wiki/%EC%86%8C%EB%8B%89_%ED%8C%80" title="소닉 팀">소닉 팀</a>이 담당한다.</p>
<p>현재 <a href="/wiki/%EB%8B%8C%ED%85%90%EB%8F%84_%EC%8A%A4%EC%9C%84%EC%B9%98" title="닌텐도 스위치">닌텐도 스위치</a>, <a href="/wiki/%ED%94%8C%EB%A0%88%EC%9D%B4%EC%8A%A4%ED%85%8C%EC%9D%B4%EC%85%98_4" title="플레이스테이션 4">플레이스테이션 4</a>, <a class="mw-redirect" href="/wiki/XBOX_ONE" title="XBOX ONE">XBOX ONE</a>, <a class="mw-disambig" href="/wiki/PC" title="PC">PC</a>로 발매될 예정이다.</p>
<h2><span class="mw-headline" id=".EC.8A.A4.ED.86.A0.EB.A6.AC">스토리</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%86%8C%EB%8B%89_2017_%ED%94%84%EB%A1%9C%EC%A0%9D%ED%8A%B8&amp;action=edit&amp;section=1" title="부분 편집: 스토리">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>현재 소닉 더 헤지호그 공식 <a class="mw-disambig" href="/wiki/%ED%8A%B8%EB%A0%88%EC%9D%BC%EB%9F%AC" title="트레일러">트레일러</a>를 보면 <a class="mw-redirect" href="/wiki/%EC%86%8C%EB%8B%89_%EB%8D%94_%ED%97%A4%EC%A7%80%ED%98%B8%EA%B7%B8:_%EB%84%A5%EC%8A%A4%ED%8A%B8_%EC%A0%9C%EB%84%88%EB%A0%88%EC%9D%B4%EC%85%98" title="소닉 더 헤지호그: 넥스트 제너레이션">소닉 더 헤지호그: 넥스트 제너레이션즈</a>의 크라이시스 시티와 같이 파괴된 도시와 광부용 모자를 쓴 로봇이 등장하는데, 이 때 화면에 나타나는 첫 문구가 'IN THE DARKEST OF TIMES'인 것을 보아 게임의 배경은 <a class="mw-redirect" href="/wiki/%EC%86%8C%EB%8B%89_%EB%8D%94_%ED%97%A4%EC%A7%80%ED%98%B9_%EC%8B%9C%EB%A6%AC%EC%A6%88" title="소닉 더 헤지혹 시리즈">소닉 더 헤지혹 시리즈</a> 역사상 가장 어둡다고 추측할 수 있다는 의견이 있다. 또한 이 의견에서는 <a href="/wiki/%EC%86%8C%EB%8B%89_%EC%96%B8%EB%A6%AC%EC%89%AC%EB%93%9C" title="소닉 언리쉬드">소닉 언리쉬드</a>의 루프탑 런을 사용한 것 같다는 의견을 내고 있다. 이 트레일러에서는 소닉 제너레이션즈의 클래식 시리즈의 소닉이 나오며, 모던 소닉의 경우 <a href="/wiki/%EC%86%8C%EB%8B%89_%EB%A1%9C%EC%8A%A4%ED%8A%B8_%EC%9B%94%EB%93%9C" title="소닉 로스트 월드">소닉 로스트 월드</a>와 같이 스핀 대시를 사용하고 클래식 소닉과 같은 부스트 <a class="mw-redirect" href="/wiki/%EC%9D%8C%ED%96%A5" title="음향">음향</a>을 사용한다. 또한 영상 마지막에 JOIN THE RESISTANCE(저항군에 동참하라)라는 문구가 나온다.</p>
<p>이이즈카 타카시의 말에 따르면, 이 게임은 기존 게임의 후속작이 아닌 완전한 <a class="mw-disambig" href="/wiki/%EC%8B%A0%EC%9E%91" title="신작">신작</a>이라고 한다. 일단 다른 스타일의 소닉들을 조작한다는 기획 자체는 소닉 제너레이션즈와 동일하므로, 스토리적으로 완전 신작이라는 의미로 해석이 가능하다는 의견이 있다.</p>
<h2><span class="mw-headline" id=".EC.BB.A8.ED.8A.B8.EB.A1.A4">컨트롤</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%86%8C%EB%8B%89_2017_%ED%94%84%EB%A1%9C%EC%A0%9D%ED%8A%B8&amp;action=edit&amp;section=2" title="부분 편집: 컨트롤">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>현재 소닉의 기능 중에는 잠시 대기했다 달리는 '스핀 대시', '슬라이드' 등이 포함된 것으로 보이며, <a class="mw-disambig" href="/wiki/%EB%B6%80%EC%8A%A4%ED%8A%B8" title="부스트">부스트</a>기능을 포함하고 있다.</p>
