<h1 class="firstHeading" id="firstHeading" lang="ko">소닉 어드밴스</h1>
<div class="mw-body-content" id="bodyContent">
<div id="siteSub">위키백과, 우리 모두의 백과사전.</div>
<div id="contentSub"></div>
<div class="mw-jump" id="jump-to-nav">
					이동:					<a href="#mw-head">둘러보기</a>, 					<a href="#p-search">검색</a>
</div>
<div class="mw-content-ltr" dir="ltr" id="mw-content-text" lang="ko"><script>function mfTempOpenSection(id){var block=document.getElementById("mf-section-"+id);block.className+=" open-block";block.previousSibling.className+=" open-block";}</script><p><b>소닉 어드밴스</b>는 <a href="/wiki/2001%EB%85%84" title="2001년">2001년</a> <a href="/wiki/%EC%86%8C%EB%8B%89_%ED%8C%80" title="소닉 팀">소닉 팀</a>과 <a href="/wiki/%EB%94%A4%ED%94%84%EC%8A%A4" title="딤프스">딤프스</a>가 발행한 <a href="/wiki/%EA%B2%8C%EC%9E%84%EB%B3%B4%EC%9D%B4_%EC%96%B4%EB%93%9C%EB%B0%B4%EC%8A%A4" title="게임보이 어드밴스">게임보이 어드밴스</a>용 <a class="mw-redirect" href="/wiki/%EC%86%8C%EB%8B%89_%EB%8D%94_%ED%97%A4%EC%A7%80%ED%98%B8%EA%B7%B8_%EC%8B%9C%EB%A6%AC%EC%A6%88" title="소닉 더 헤지호그 시리즈">소닉 더 헤지호그 시리즈</a>의 게임이다. 이후 <a href="/wiki/THQ" title="THQ">THQ</a>에 의해 <a class="mw-redirect" href="/wiki/%EB%B6%81%EB%AF%B8" title="북미">북미</a>에 출시되었고 <a class="new" href="/w/index.php?title=%EC%9D%B8%ED%8F%AC%EA%B2%8C%EC%9E%84%EC%8A%A4&amp;action=edit&amp;redlink=1" title="인포게임스 (없는 문서)">인포게임스</a>에 의해 <a class="mw-redirect" href="/wiki/%ED%98%B8%EC%A3%BC" title="호주">호주</a> 지역에 출시되었다.</p>
<p>소닉 어드밴스는 이후 <a href="/wiki/2003%EB%85%84" title="2003년">2003년</a> <a href="/wiki/10%EC%9B%94_7%EC%9D%BC" title="10월 7일">10월 7일</a>에 <a href="/wiki/%EB%85%B8%ED%82%A4%EC%95%84" title="노키아">노키아</a> <a class="new" href="/w/index.php?title=%EC%97%94%EA%B2%8C%EC%9D%B4%EC%A7%80&amp;action=edit&amp;redlink=1" title="엔게이지 (없는 문서)">엔게이지</a>에 등록되어 <b>소닉 N</b>이라는 이름으로 발매하였고, <a class="mw-disambig" href="/wiki/%EC%95%88%EB%93%9C%EB%A1%9C%EC%9D%B4%EB%93%9C" title="안드로이드">안드로이드</a>에는 <a href="/wiki/2011%EB%85%84" title="2011년">2011년</a> <a href="/wiki/11%EC%9B%94_25%EC%9D%BC" title="11월 25일">11월 25일</a> 출시되었다. 또한 <a href="/wiki/Wii_U" title="Wii U">Wii U</a> <a href="/wiki/%EB%B2%84%EC%B6%94%EC%96%BC_%EC%BD%98%EC%86%94" title="버추얼 콘솔">버추얼 콘솔</a>에 출시되기도 하였다.</p>
