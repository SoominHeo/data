<p><b>소닉 어드벤처 1</b>(<span style="font-size: smaller;"><a href="/wiki/%EC%9D%BC%EB%B3%B8%EC%96%B4" title="일본어">일본어</a>:</span> <span lang="ja" xml:lang="ja">ソニックアドベンチャー</span> <span style="font-size: smaller;"><a href="/wiki/%EC%98%81%EC%96%B4" title="영어">영어</a>:</span> <span lang="en" xml:lang="en">Sonic Adventure 1</span>)은 <a class="mw-redirect" href="/wiki/%EC%84%B8%EA%B0%80" title="세가">세가</a>가 발매하고 소닉 팀이 만든 소닉 시리즈의 첫 <a href="/wiki/%EB%93%9C%EB%A6%BC%EC%BA%90%EC%8A%A4%ED%8A%B8" title="드림캐스트">드림캐스트</a> 게임이다. 후에 PC 및 <a href="/wiki/%EB%8B%8C%ED%85%90%EB%8F%84_%EA%B2%8C%EC%9E%84%ED%81%90%EB%B8%8C" title="닌텐도 게임큐브">게임 큐브</a> 용으로 소닉 어드벤처 DX가 발매되었다. 소닉 어드벤처 DX는 옛날 <a href="/wiki/%EA%B2%8C%EC%9E%84_%EA%B8%B0%EC%96%B4" title="게임 기어">게임기어</a>로 나온 소닉게임을 열 수 있도록 했다.</p>
<p>2010년에는 PSN, XBLA에 HD 버전의 소닉 어드벤처를 다운받을 수 있게 되었다. DX버전으로 업그레이드 할 수 있는 DLC가 나왔으며, 드림캐스트 콜렉션에도 포함되었다.</p>

<h2><span class="mw-headline" id=".EA.B2.8C.EC.9E.84.ED.94.8C.EB.A0.88.EC.9D.B4">게임플레이</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%86%8C%EB%8B%89_%EC%96%B4%EB%93%9C%EB%B2%A4%EC%B2%98&amp;action=edit&amp;section=1" title="부분 편집: 게임플레이">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>메가드라이브에서 드림캐스트로 넘어오면서 만들어진 소닉 어드벤처엔 많은 변화가 있었다. 일단 기존의 메가드라이브의 소닉 더 헤지호그 시리즈는 2D 횡스크롤 게임플레이 인데 비해, 소닉 어드벤처는 3D 게임플레이이다. 특히 소닉은 가까운 적이나 스프링을 향해 날아가는 호밍어택 기능이 추가 되었다. 또한 소닉, 테일즈, 너클즈, 에이미의 눈 색깔과 키가 바뀌는 등 캐릭터 디자인도 변했다.(일러스트 포함)</p>
<h2><span class="mw-headline" id=".EC.8A.A4.ED.86.A0.EB.A6.AC">스토리</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%86%8C%EB%8B%89_%EC%96%B4%EB%93%9C%EB%B2%A4%EC%B2%98&amp;action=edit&amp;section=2" title="부분 편집: 스토리">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<table class="notice section" style="text-align: left; margin:0 0 0.5em 0.5em; clear:none; font-size:smaller;">
<tr>
<td width="auto"><img alt="" data-file-height="400" data-file-width="400" height="25" src="//upload.wikimedia.org/wikipedia/commons/thumb/2/2c/Broom_icon.svg/25px-Broom_icon.svg.png" srcset="//upload.wikimedia.org/wikipedia/commons/thumb/2/2c/Broom_icon.svg/38px-Broom_icon.svg.png 1.5x, //upload.wikimedia.org/wikipedia/commons/thumb/2/2c/Broom_icon.svg/50px-Broom_icon.svg.png 2x" width="25"/> 이 문단은 <b><a href="/wiki/%EC%9C%84%ED%82%A4%EB%B0%B1%EA%B3%BC:%ED%8E%B8%EC%A7%91_%EC%A7%80%EC%B9%A8" title="위키백과:편집 지침">위키백과의 편집 지침</a></b>에 맞춰 다듬어야 합니다. 더 좋은 문단이 되도록 <span class="plainlinks"><a class="external text" href="//ko.wikipedia.org/w/index.php?title=%EC%86%8C%EB%8B%89_%EC%96%B4%EB%93%9C%EB%B2%A4%EC%B2%98&amp;action=edit">문단 수정을 도와주세요</a></span>. 내용에 대한 의견이 있으시다면 <a href="/wiki/%ED%86%A0%EB%A1%A0:%EC%86%8C%EB%8B%89_%EC%96%B4%EB%93%9C%EB%B2%A4%EC%B2%98#.23" title="토론:소닉 어드벤처">토론 문서</a>에서 나누어 주세요. (2014년 8월 18일에 문단의 정리가 요청되었습니다.)</td>

<p><a href="/wiki/%EC%86%8C%EB%8B%89_%EB%8D%94_%ED%97%A4%EC%A7%80%ED%98%B8%EA%B7%B8_(%EC%BA%90%EB%A6%AD%ED%84%B0)" title="소닉 더 헤지호그 (캐릭터)">소닉</a>은 도시를 거닐던 도중 경찰들이 달려드는 것을 보고 따라가보니 카오스를 만나게 되었다. 그렇다. <a href="/wiki/%EB%8B%A5%ED%84%B0_%EC%97%90%EA%B7%B8%EB%A7%A8" title="닥터 에그맨">에그맨</a>은 도시를 파괴하고 에그맨랜드를 건설할 목적으로 카오스를 불러들였다. 그리고 에그맨은 자신의 부하로 <a class="new" href="/w/index.php?title=E-102_%EA%B0%90%EB%A7%88&amp;action=edit&amp;redlink=1" title="E-102 감마 (없는 문서)">감마</a>와 다른 로봇들을 만들었다. 그리고 중심 기지인 파이널 에그에서 자신의 비행선인 에그 캐리어를 쏘았다. 그러나 카오스 에메랄드를 갖고 있는 개구리를 찾는 명령을 내린 에그맨이 감마를 제외한 나머지 로봇이 다른 개구리를 찾자, 이에 분노하며, 다른 로봇들을 분해하였다. 분해하는 모습을 본 감마는 다른 로봇들을 에그맨으로부터 풀어주기 위해 곳곳을 돌아다니며 로봇들을 파괴하고 자신도 죽게 된다.<sup class="reference" id="cite_ref-2"><a href="#cite_note-2">[2]</a></sup> 한편, 소닉은 에그 캐리어의 공격에 떨어져 다시 도시에 도착하게 된다. 그런데 도시를 거닐던 도중, 쇼핑 중이었던 <a href="/wiki/%EC%97%90%EC%9D%B4%EB%AF%B8_%EB%A1%9C%EC%A6%88" title="에이미 로즈">에이미</a>와 우연히 만나게 되었다. 그러나 에이미가 트윙클 파크에서 추적하고 있던 에그맨의 로봇에 의해 납치당하게 되었다. 결국 다시 에이미를 찾으러 에그 캐리어를 습격하고, 파이널 에그를 파괴하였다. 그러나, 에그맨이 카오스의 컨트롤에 실패하면서 카오스가 폭주하기 시작한다. 에그맨은 카오스의 폭주를 알려주고 소닉은 슈퍼 소닉으로 변신하여 카오스를 봉인한다.</p>
<h2><span class="mw-headline" id=".ED.94.8C.EB.A0.88.EC.9D.B4_.EA.B0.80.EB.8A.A5_.EC.BA.90.EB.A6.AD.ED.84.B0">플레이 가능 캐릭터</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%86%8C%EB%8B%89_%EC%96%B4%EB%93%9C%EB%B2%A4%EC%B2%98&amp;action=edit&amp;section=3" title="부분 편집: 플레이 가능 캐릭터">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<ul>
<li><a class="mw-disambig" href="/wiki/%EC%86%8C%EB%8B%89_%EB%8D%94_%ED%97%A4%EC%A7%80%ED%98%B8%EA%B7%B8" title="소닉 더 헤지호그">소닉 더 헤지호그</a></li>
</ul>
<p>기존의 소닉 게임과 마찬가지로 빠르게 골에 도착하는 것이 목표.2D였던 소닉 시리즈를 3D로 옮기면서 많은 요소들이 추가되면서 지금까지와는 다른 감각의 스피디한 느낌을 느낄 수 있게 되었다. <a href="/wiki/%EC%86%8C%EB%8B%89_%EC%96%B4%EB%93%9C%EB%B2%A4%EC%B2%98_2" title="소닉 어드벤처 2">소닉 어드벤처 2</a>에서 소닉과 더불어 <a class="mw-redirect" href="/wiki/%EC%84%80%EB%8F%84%EC%9A%B0_%EB%8D%94_%ED%97%A4%EC%A7%80%ED%98%B9" title="섀도우 더 헤지혹">섀도우 더 헤지혹</a>의 게임플레이로 계승된다.</p>
<ul>
<li><a class="mw-redirect" href="/wiki/%EB%A7%88%EC%9D%BC%EC%A6%88_%22%ED%85%8C%EC%9D%BC%EC%A6%88%22_%ED%94%84%EB%9D%BC%EC%9A%B0%EC%96%B4" title='마일즈 "테일즈" 프라우어'>마일즈 "테일즈" 프라우어</a></li>
</ul>
<p>골에 도착하는 것은 소닉과 마찬가지이나 소닉(일부 스테이지에는 에그맨)과 경주를 하여 먼저 도착하는 것이 목표.</p>
<ul>
<li><a href="/wiki/%EB%84%88%ED%81%B4%EC%A6%88_%EB%94%94_%EC%97%90%ED%82%A4%EB%93%9C%EB%82%98" title="너클즈 디 에키드나">너클즈 디 에키드나</a></li>
</ul>
<p>필드를 돌아다니면서 마스터 에메랄드 조각 3개를 모으는 것이 목표. 상당히 플레이가 길어지기 때문에 좋은 평가를 받지 못했다. <a href="/wiki/%EC%86%8C%EB%8B%89_%EC%96%B4%EB%93%9C%EB%B2%A4%EC%B2%98_2" title="소닉 어드벤처 2">소닉 어드벤처 2</a>에서 너클즈와 더불어 <a href="/wiki/%EB%A3%A8%EC%A6%88_%EB%8D%94_%EB%B1%83" title="루즈 더 뱃">루즈 더 뱃</a>의 게임플레이로 계승된다.</p>
<ul>
<li><a href="/wiki/%EC%97%90%EC%9D%B4%EB%AF%B8_%EB%A1%9C%EC%A6%88" title="에이미 로즈">에이미 로즈</a></li>
</ul>
<p>골에 도착하는 것은 소닉과 마찬가지이나 속도가 소닉보다 느리며 액션 버튼으로 해머 공격을 할 수 있다.</p>
<ul>
<li><a class="new" href="/w/index.php?title=%EB%B9%85_%EB%8D%94_%EC%BA%A3&amp;action=edit&amp;redlink=1" title="빅 더 캣 (없는 문서)">빅 더 캣</a></li>
</ul>
<p>다른 캐릭터들과 달리 <a href="/wiki/%EB%82%9A%EC%8B%9C" title="낚시">낚시</a>로 대상을 낚는 것이 목표.</p>
<ul>
<li><a class="new" href="/w/index.php?title=E-102_%EA%B0%90%EB%A7%88&amp;action=edit&amp;redlink=1" title="E-102 감마 (없는 문서)">E-102 감마</a></li>
</ul>
<p>타겟을 록온하면서 골에 도착하는 것이 목표. 다른 캐릭터들과는 달리 미션에 시간 제한이 정해져 있다. 타겟을 맞추면 제한시간이 늘어난다. <a href="/wiki/%EC%86%8C%EB%8B%89_%EC%96%B4%EB%93%9C%EB%B2%A4%EC%B2%98_2" title="소닉 어드벤처 2">소닉 어드벤처 2</a>에서 테일즈와 에그맨의 게임플레이로 계승된다.</p>

<h2><span class="mw-headline" id=".EB.B0.98.EC.9D.91">반응</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%86%8C%EB%8B%89_%EC%96%B4%EB%93%9C%EB%B2%A4%EC%B2%98&amp;action=edit&amp;section=9" title="부분 편집: 반응">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>반응은 아주 뜨거웠다. 총 250만장이 드림캐스트로 팔렸고, 드림캐스트에서 제일 많이 팔린 게임이 되었다. 속편 <a href="/wiki/%EC%86%8C%EB%8B%89_%EC%96%B4%EB%93%9C%EB%B2%A4%EC%B2%98_2" title="소닉 어드벤처 2">소닉 어드벤처 2</a>가 3년 뒤에 나왔고, 2011년 시리즈의 20주년 게임 <a class="mw-redirect" href="/wiki/%EC%86%8C%EB%8B%89_%EC%A0%9C%EB%84%A4%EB%A0%88%EC%9D%B4%EC%85%98%EC%A6%88" title="소닉 제네레이션즈">소닉 제네레이션즈</a>에 소닉 어드벤처의 2 스테이지, 스피드 하이웨이와 에메랄드 코스트가 각각 콘솔 버전과 3DS버전에 나왔다.</p>
