<h1 class="firstHeading" id="firstHeading" lang="ko">소당류</h1>
<div class="mw-body-content" id="bodyContent">
<div id="siteSub">위키백과, 우리 모두의 백과사전.</div>
<div id="contentSub"></div>
<div class="mw-jump" id="jump-to-nav">
					이동:					<a href="#mw-head">둘러보기</a>, 					<a href="#p-search">검색</a>
</div>
<div class="mw-content-ltr" dir="ltr" id="mw-content-text" lang="ko"><script>function mfTempOpenSection(id){var block=document.getElementById("mf-section-"+id);block.className+=" open-block";block.previousSibling.className+=" open-block";}</script><p><b>소당류</b>(少糖類), <b>과당류</b>(寡糖類) 또는 <b>올리고당류</b>(oligosaccharide)는 <a href="/wiki/%EB%8B%A8%EB%8B%B9%EB%A5%98" title="단당류">단당류</a>가 2~10개 결합되어 있는 <a href="/wiki/%ED%83%84%EC%88%98%ED%99%94%EB%AC%BC" title="탄수화물">탄수화물</a>을 의미한다. <a href="/wiki/%EB%8B%B9%EB%8B%A8%EB%B0%B1%EC%A7%88" title="당단백질">당단백질</a>이나 <a class="new" href="/w/index.php?title=%EB%8B%B9%EC%A7%80%EC%A7%88&amp;action=edit&amp;redlink=1" title="당지질 (없는 문서)">당지질</a>의 구성성분으로서 <a href="/wiki/%EC%84%B8%ED%8F%AC" title="세포">세포</a>내에서는 주로 생체막에 부착되어 있고, <a href="/wiki/%EC%86%8C%ED%8F%AC%EC%B2%B4" title="소포체">소포체</a>와 <a href="/wiki/%EA%B3%A8%EC%A7%80%EC%B2%B4" title="골지체">골지체</a> 등의 분비형 단백질과 결합되어 있다. 일부 기능성 올리고당의 경우 소화효소에 의해 분해되지 않고 <a class="mw-redirect" href="/wiki/%EB%8C%80%EC%9E%A5_(%EC%9E%A5%EA%B8%B0)" title="대장 (장기)">대장</a>으로 내려가 장내 유익균의 영양원이 되어 대장 환경을 개선하는 데 도움을 준다고 알려져 있다.<sup class="reference" id="cite_ref-1"><a href="#cite_note-1">[1]</a></sup></p>
<p>당은 우리 몸에서 에너지원으로 사용되지만 그 당이 연결된 당쇄는 그 외에도 생명현상을 유지하기 위해 중요한 역할을 담당하고 있다. 세포간의 정보 전달, 세포 분화, <a href="/wiki/%EB%B0%94%EC%9D%B4%EB%9F%AC%EC%8A%A4" title="바이러스">바이러스</a>나 <a href="/wiki/%EC%84%B8%EA%B7%A0" title="세균">세균</a>의 <a href="/wiki/%EA%B0%90%EC%97%BC" title="감염">감염</a>, <a href="/wiki/%EC%95%94%EC%84%B8%ED%8F%AC" title="암세포">암세포</a>의 <a class="mw-disambig" href="/wiki/%EC%A0%84%EC%9D%B4" title="전이">전이</a> 등을 예로 들 수 있다. <a href="/wiki/DNA" title="DNA">DNA</a>의 경우 <a href="/wiki/%EB%89%B4%ED%81%B4%EB%A0%88%EC%98%A4%ED%83%80%EC%9D%B4%EB%93%9C" title="뉴클레오타이드">뉴클레오타이드</a>가, <a href="/wiki/%EB%8B%A8%EB%B0%B1%EC%A7%88" title="단백질">단백질</a>의 경우 <a href="/wiki/%EC%95%84%EB%AF%B8%EB%85%B8%EC%82%B0" title="아미노산">아미노산</a>이 1차원적으로 연결된 <a class="new" href="/w/index.php?title=%EC%83%9D%EC%B2%B4%EA%B3%A0%EB%B6%84%EC%9E%90&amp;action=edit&amp;redlink=1" title="생체고분자 (없는 문서)">생체고분자</a>로서 존재하지만 당쇄의 경우 가지친 구조를 지닐 수 있기에 더욱 다양한 <a href="/wiki/%EB%B6%84%EC%9E%90" title="분자">분자</a> 구조를 지닐 수 있다.</p>
