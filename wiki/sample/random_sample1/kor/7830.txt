<h2><span class="mw-headline" id=".ED.8B.B8.EC.A7.80.ED.8A.B8_.EC.8B.9C.EB.8C.80">틸지트 시대</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%86%8C%EB%B2%A0%EC%B8%A0%ED%81%AC_(%EC%B9%BC%EB%A6%AC%EB%8B%8C%EA%B7%B8%EB%9D%BC%EB%93%9C_%EC%A3%BC)&amp;action=edit&amp;section=1" title="부분 편집: 틸지트 시대">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p><b>틸지트</b>는 <a href="/wiki/1288%EB%85%84" title="1288년">1288년</a>에 <a href="/wiki/%ED%8A%9C%ED%84%B4_%EA%B8%B0%EC%82%AC%EB%8B%A8" title="튜턴 기사단">튜턴 기사단</a>(독일기사단)에 의해서 지어진 샬라우너 하우스(Schalauner Haus)라는 성(城)의 주변에 발달한 도시이다. <a href="/wiki/1552%EB%85%84" title="1552년">1552년</a>에 도시로 성장하였으며, <a href="/wiki/1807%EB%85%84" title="1807년">1807년</a> 7월에 이 곳에서 <a href="/wiki/%ED%94%84%EB%9E%91%EC%8A%A4" title="프랑스">프랑스</a>의 <a class="mw-redirect" href="/wiki/%EB%82%98%ED%8F%B4%EB%A0%88%EC%98%B9_1%EC%84%B8" title="나폴레옹 1세">나폴레옹 1세</a>와 <a href="/wiki/%ED%94%84%EB%A1%9C%EC%9D%B4%EC%84%BC" title="프로이센">프로이센</a>의 <a href="/wiki/%ED%94%84%EB%A6%AC%EB%93%9C%EB%A6%AC%ED%9E%88_%EB%B9%8C%ED%97%AC%EB%A6%84_3%EC%84%B8" title="프리드리히 빌헬름 3세">프리드리히 빌헬름 3세</a>, <a href="/wiki/%EB%9F%AC%EC%8B%9C%EC%95%84" title="러시아">러시아</a>의 <a href="/wiki/%EC%95%8C%EB%A0%89%EC%82%B0%EB%93%9C%EB%A5%B4_1%EC%84%B8" title="알렉산드르 1세">알렉산드르 1세</a> 사이에 <a href="/wiki/%ED%8B%B8%EC%A7%80%ED%8A%B8_%EC%A1%B0%EC%95%BD" title="틸지트 조약">평화조약</a>이 맺어졌다. 프로이센의 영토는 이 때 크게 줄어들었으나, 프로이센의 왕비 <a href="/wiki/%EB%A9%94%ED%81%B4%EB%A0%8C%EB%B6%80%EB%A5%B4%ED%81%AC%EC%8A%88%ED%8A%B8%EB%A0%90%EB%A6%AC%EC%B8%A0%EC%9D%98_%EB%A3%A8%EC%9D%B4%EC%A0%9C" title="메클렌부르크슈트렐리츠의 루이제">루이제</a>의 의연한 교섭은 망국의 기로에 있던 프로이센인들에게 큰 지지를 얻었다. <a href="/wiki/1945%EB%85%84" title="1945년">1945년</a> 이 도시가 독일로부터 이탈할 때까지, 루이제 왕비와 프리드리히 빌헬름 3세 부부가 머물렀던 집에는 대리석의 명판이 내걸렸다. 또한 시내에는 이 도시 출신의 시인 <a class="new" href="/w/index.php?title=%EB%A7%89%EC%8A%A4_%ED%8F%B0_%EC%86%85%EC%BC%84%EB%8F%84%EB%A5%B4%ED%94%84&amp;action=edit&amp;redlink=1" title="막스 폰 솅켄도르프 (없는 문서)">막스 폰 솅켄도르프</a>(1783~1817)를 기념하는 솅켄도르프 광장이 있었다. <a href="/wiki/1900%EB%85%84" title="1900년">1900년</a>경에는 34,500명의 인구를 가진 도시가 되어 <a href="/wiki/%EB%85%B8%EB%A9%B4%EC%A0%84%EC%B0%A8" title="노면전차">노면전차</a>가 생겼으며 <a href="/wiki/%EC%BE%A8%EB%8B%88%ED%9E%88%EC%8A%A4%EB%B2%A0%EB%A5%B4%ED%81%AC" title="쾨니히스베르크">쾨니히스베르크</a>와 <a href="/wiki/%ED%8F%B4%EB%A0%88%EC%8A%A4%ED%81%AC" title="폴레스크">라비아우</a>로 이어지는 철도가 운행되었다.</p>
<p><a class="mw-redirect" href="/wiki/%EC%A0%9C_1%EC%B0%A8_%EC%84%B8%EA%B3%84%EB%8C%80%EC%A0%84" title="제 1차 세계대전">제 1차 세계대전</a> 이후 <a class="new" href="/w/index.php?title=%EB%A9%94%EB%A9%9C%EB%9E%80%ED%8A%B8&amp;action=edit&amp;redlink=1" title="메멜란트 (없는 문서)">메멜란트</a>가 <a href="/wiki/%EB%A6%AC%ED%88%AC%EC%95%84%EB%8B%88%EC%95%84" title="리투아니아">리투아니아</a> 영토가 됨에 따라, 틸지트는 네만 강을 사이에 두고 리투아니아와 <a href="/wiki/%EA%B5%AD%EA%B2%BD" title="국경">국경</a>을 접하게 되었다. <a class="mw-redirect" href="/wiki/%EC%A0%9C_2%EC%B0%A8_%EC%84%B8%EA%B3%84%EB%8C%80%EC%A0%84" title="제 2차 세계대전">제 2차 세계대전</a> 이전에는 독일 <a href="/wiki/%EB%8F%99%ED%94%84%EB%A1%9C%EC%9D%B4%EC%84%BC" title="동프로이센">동프로이센</a>의 도시였지만, <a href="/wiki/1945%EB%85%84" title="1945년">1945년</a> <a href="/wiki/1%EC%9B%94_20%EC%9D%BC" title="1월 20일">1월 20일</a> <a href="/wiki/%EC%86%8C%EB%A0%A8%EA%B5%B0" title="소련군">소련군</a>이 입성하였고, <a href="/wiki/%EB%8F%85%EC%9D%BC%EC%9D%B8" title="독일인">독일인</a> 주민들은 추방되었다.</p>
