<p><b>소비칠레</b>(<span style="font-size: smaller;"><a href="/wiki/%EC%9D%B4%ED%83%88%EB%A6%AC%EC%95%84%EC%96%B4" title="이탈리아어">이탈리아어</a>:</span> <span lang="it" xml:lang="it">Sovicille</span>)는 <a href="/wiki/%EC%9D%B4%ED%83%88%EB%A6%AC%EC%95%84" title="이탈리아">이탈리아</a> <a href="/wiki/%ED%86%A0%EC%8A%A4%EC%B9%B4%EB%82%98_%EC%A3%BC" title="토스카나 주">토스카나 주</a> <a href="/wiki/%EC%8B%9C%EC%97%90%EB%82%98_%ED%98%84" title="시에나 현">시에나 현</a>에 있는 코무네다. <a href="/wiki/%ED%94%BC%EB%A0%8C%EC%B2%B4" title="피렌체">피렌체</a>에서 남쪽 60km, <a href="/wiki/%EC%8B%9C%EC%97%90%EB%82%98" title="시에나">시에나</a>에서 남서쪽 10km 거리에 있다.</p>
<h2><span class="mw-headline" id=".EC.97.AD.EC.82.AC">역사</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%86%8C%EB%B9%84%EC%B9%A0%EB%A0%88&amp;action=edit&amp;section=1" title="부분 편집: 역사">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>소비칠레의 명칭은 1004년에 처음으로 역사에서 언급됐으며, 이곳의 기원은 더욱 오래될 것으로 추정된다(2002년에 로마 시대의 모자이크가 교회에서 발견되었고, 현재 지역에는 에트루리아 시대의 유적지는 없다).</p>
<p>중세 시대 기간에 <a href="/wiki/%EC%8B%9C%EC%97%90%EB%82%98" title="시에나">시에나</a>와 <a href="/wiki/%EB%B3%BC%ED%85%8C%EB%9D%BC" title="볼테라">볼테라</a> 국경 사이에 볼테라의 지배를 받던 세레나 수도원이 세워졌는데, 수도원의 영지에 소비칠레가 사유 재산으로서 속해있었다.</p>
<p>본래 시에나 출신인 <a href="/wiki/%EA%B5%90%ED%99%A9_%EC%95%8C%EB%A0%89%EC%82%B0%EB%8D%B0%EB%A5%B4_3%EC%84%B8" title="교황 알렉산데르 3세">교황 알렉산데르 3세</a>가 남긴 1123년 4월 23일 문서 기록에 따르면, 이곳을 수피칠레(Sufficille)라고 하였다. 그 명칭은 <a href="/wiki/%EB%9D%BC%ED%8B%B4%EC%96%B4" title="라틴어">라틴어</a> 단어인 "수브"(sub, 아래의)와 "피키눌라이/피쿠스"(ficinulae/ficus, 작은 무화과 나무)에서 유래하였다.</p>
<p>자치 도시로서의 기록은 13세기로 거슬러 올라간다. 13세기에 <a href="/wiki/%EC%8B%9C%EC%97%90%EB%82%98_%EA%B3%B5%ED%99%94%EA%B5%AD" title="시에나 공화국">시에나 공화국</a>은 일부 지방 자치제를 허용했었는데, 그중에 소비칠레가 포함되었고, 1238년에 자치 도시 지위를 얻었다.</p>
<p>1260년 소비칠레는 <a href="/wiki/%ED%94%BC%EB%A0%8C%EC%B2%B4_%EA%B3%B5%ED%99%94%EA%B5%AD" title="피렌체 공화국">피렌체</a>에게 점령당하나, 그 해에 피렌체군이 <a href="/wiki/%EB%AA%AC%ED%83%80%ED%8E%98%EB%A5%B4%ED%8B%B0_%EC%A0%84%ED%88%AC" title="몬타페르티 전투">몬타페르티 전투</a>에서 패배함에 따라, 소비칠레는 시에나로 돌아왔다.</p>
<p>1333년 소비칠레는 <a href="/wiki/%ED%94%BC%EC%82%AC_%EA%B3%B5%ED%99%94%EA%B5%AD" title="피사 공화국">피사</a>에 의해 불태워졌다.</p>
<p>메디치 가문은 1557년에 2백만 두캇을 지불하고 <a href="/wiki/%EC%B9%B4%EB%A5%BC_5%EC%84%B8" title="카를 5세">신성 로마 황제</a>에게서 시에나의 영지를 사들였고, 이로 인해 민주 제도는 폐지되었고 봉건정으로 복귀하게 되었다. 소비칠레는 메디치 가문의 영지가 되었고 새롭게 확장된 영토의 접근점을 보호할 목적으로 요새로 전환되었다.</p>
<p>메디치 가문의 마지막 통치자가 사망하고, 그 모든 권한은 <a href="/wiki/%EB%82%98%ED%8F%B4%EB%A0%88%EC%98%B9_%EB%B3%B4%EB%82%98%ED%8C%8C%EB%A5%B4%ED%8A%B8" title="나폴레옹 보나파르트">나폴레옹 보나파르트</a>가 침략하기 이전인 18세기까지 <a class="new" href="/w/index.php?title=%EB%A1%9C%EB%A0%8C_%EA%B3%B5%EC%9E%91&amp;action=edit&amp;redlink=1" title="로렌 공작 (없는 문서)">로렌 공작</a>에게 넘어가게 되었다.</p>
<p>1814년에 <a href="/wiki/%EB%B9%88_%ED%9A%8C%EC%9D%98" title="빈 회의">빈 회의</a>가 열리게 되었고, 소비칠레는 <a href="/wiki/%ED%86%A0%EC%8A%A4%EC%B9%B4%EB%82%98_%EB%8C%80%EA%B3%B5%EA%B5%AD" title="토스카나 대공국">토스카나 대공국</a>에 합병되었다. 1859년 단명했던 국가인 <a class="new" href="/w/index.php?title=%EC%A4%91%EB%B6%80_%EC%9D%B4%ED%83%88%EB%A6%AC%EC%95%84_%ED%95%A9%EC%A4%91%EA%B5%AD&amp;action=edit&amp;redlink=1" title="중부 이탈리아 합중국 (없는 문서)">중부 이탈리아 합중국</a>의 지배를 받다가 <a href="/wiki/%EC%9D%B4%ED%83%88%EB%A6%AC%EC%95%84_%EC%99%95%EA%B5%AD_(1861%EB%85%84_~_1946%EB%85%84)" title="이탈리아 왕국 (1861년 ~ 1946년)">이탈리아 왕국</a>에 흡수되었다.</p>
