<p><b>소앤틸리스 제도</b>(Lesser Antilles)는 <a href="/wiki/%EC%B9%B4%EB%A6%AC%EB%B8%8C_%EC%A0%9C%EB%8F%84" title="카리브 제도">카리브 제도</a>(<a href="/wiki/%EC%84%9C%EC%9D%B8%EB%8F%84_%EC%A0%9C%EB%8F%84" title="서인도 href="/wiki/%EC%95%A4%ED%8B%B8%EB%A6%AC%EC%8A%A4_%EC%A0%9C%EB%8F%84" title="앤틸리스 제도">앤틸리스 제도</a>를 이룬다. 소앤틸리스 제도에는 <a class="mw-redirect" href="/wiki/%EB%B2%84%EC%A7%84%EC%95%84%EC%9D%BC%EB%9E%9C%EB%93%9C" title="버진아일랜드">버진아일랜드</a>, <a href="/wiki/%EC%95%B5%EA%B7%88%EB%9D%BC" title="앵귈라">앵귈라</a>, <a title="앤티가 바부다">앤티가 바부다</a>, <a href="/wiki/%EC%98%81%EA%B5%AD" title="영국">영국</a>령 <a href="/wiki/%EB%AA%AC%ED%8A%B8%EC%84%B8%EB%9E%AB" title="몬트세랫">몬트세랫</a>, <a href="/wiki/%ED%94%84%EB%9E%91%EC%8A%A4" title="프랑스">프랑스</a>령 <a href="/wiki/%EA%B3%BC%EB%93%A4%EB%A3%A8%ED%94%84" title="과들루프">과들루프</a>, <a href="/wiki/%EB%8F%84%EB%AF%B8%EB%8B%88%EC%B9%B4_%EC%97%B0%EB%B0%A9" title="도미니카 연방">도미니카 연방</a>, <a href="/wiki/%ED%94%84%EB%9E%91%EC%8A%A4" title="프랑스">프랑스</a>령 <a href="/wiki/%EB%A7%88%EB%A5%B4%ED%8B%B0%EB%8B%88%ED%81%AC" title="마르티니크">마르티니크</a>, <a href="/wiki/%EC%84%B8%EC%9D%B8%ED%8A%B8%EB%A3%A8%EC%8B%9C%EC%95%84" title="세인트루시아">세인트루시아</a>, <a href="/wiki/%EC%84%B8%EC%9D%B8%ED%8A%B8%EB%B9%88%EC%84%BC%ED%8A%B8_%EA%B7%B8%EB%A0%88%EB%82%98%EB%94%98" title="세인트빈센트 그레나딘">세인트빈센트 그레나딘</a>, <a href="/wiki/%EB%B0%94%EB%B2%A0%EC%9D%B4%EB%8F%84%EC%8A%A4" title="바베이도스">바베이도스</a>, <a href="/wiki/%EA%B7%B8%EB%A0%88%EB%82%98%EB%8B%A4" title="그레나다">그레나다</a>, <a href="/wiki/%ED%8A%B8%EB%A6%AC%EB%8B%88%EB%8B%A4%EB%93%9C_%ED%86%A0%EB%B0%94%EA%B3%A0" title="트리니다드 토바고">트리니다드 토바고</a>, <a href="/wiki/%EB%84%A4%EB%8D%9C%EB%9E%80%EB%93%9C" title="네덜란드">네덜란드</a>령 앤틸리스 제도가 속해있다. 대부분 <a href="/wiki/%EC%8A%A4%ED%8E%98%EC%9D%B8" title="스페인">스페인</a>의 식민지였기 때문에 스페인의 영향을 많이 받았다.</p>
