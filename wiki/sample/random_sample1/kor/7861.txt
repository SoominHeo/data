<p><b>소혜왕후 한씨</b>(昭惠王后 韓氏, <a href="/wiki/1437%EB%85%84" title="1437년">1437년</a> <a href="/wiki/10%EC%9B%94_7%EC%9D%BC" title="10월 7일">10월 7일</a>(<a href="/wiki/%EC%9D%8C%EB%A0%A5_9%EC%9B%94_8%EC%9D%BC" title="음력 9월 8일">음력 9월 8일</a>) ~ <a href="/wiki/1504%EB%85%84" title="1504년">1504년</a> <a href="/wiki/5%EC%9B%94_11%EC%9D%BC" title="5월 11일">5월 11일</a>(<a href="/wiki/%EC%9D%8C%EB%A0%A5_4%EC%9B%94_27%EC%9D%BC" title="음력 4월 27일">음력 4월 27일</a>)는 <a href="/wiki/%EC%A1%B0%EC%84%A0" title="조선">조선</a> 초기의 <a class="mw-redirect" href="/wiki/%EC%84%B8%EC%9E%90%EB%B9%88" title="세자빈">세자빈</a>이자 <a href="/wiki/%EC%A1%B0%EC%84%A0_%EB%8D%95%EC%A2%85" title="조선 덕종">덕종</a>(德宗, 추존왕)의 왕비이며 <a href="/wiki/%EC%8B%9C%ED%98%B8" title="시호">시호</a>는 <b>인수자숙휘숙명의소혜왕후</b>(仁粹慈淑徽肅明懿昭惠王后)이다. <a href="/wiki/1450%EB%85%84" title="1450년">1450년</a>(문종 즉위년)에 <a href="/wiki/%EC%A1%B0%EC%84%A0_%EC%84%B8%EC%A1%B0" title="조선 세조">수양대군</a>의 큰아들인 <a href="/wiki/%EC%A1%B0%EC%84%A0_%EB%8D%95%EC%A2%85" title="조선 덕종">도원군</a>(의경세자, 덕종)과 혼인하여 <a href="/wiki/%EA%B5%B0%EB%B6%80%EC%9D%B8" title="군부인">군부인</a>에 봉작되었으며, <a href="/wiki/1455%EB%85%84" title="1455년">1455년</a>(세조 즉위년)에 시아버지 수양대군이 왕위로 즉위하여 자신은 맏며느리로서 <a class="mw-redirect" href="/wiki/%EC%84%B8%EC%9E%90%EB%B9%88" title="세자빈">세자빈</a>이 되어 궁궐에 들어갔으나, <a href="/wiki/1457%EB%85%84" title="1457년">1457년</a>(세조 3년)에 남편 의경세자가 20세의 나이로 갑자기 죽어 사가로 물러났다. 그러나 <a href="/wiki/1469%EB%85%84" title="1469년">1469년</a>에 자신의 둘째 아들인 <a href="/wiki/%EC%A1%B0%EC%84%A0_%EC%84%B1%EC%A2%85" title="조선 성종">자을산군</a>(성종)이 왕위에 등극하고, 자신도 궁궐에 다시 들어가 곧 <a href="/wiki/%EC%A1%B0%EC%84%A0%EC%9D%98_%EC%97%AD%EB%8C%80_%EC%99%95%EB%B9%84" title="조선의 역대 왕비">왕비</a>로 진봉되었다가 <a href="/wiki/1475%EB%85%84" title="1475년">1475년</a>(성종 6년) <a href="/wiki/%EC%A1%B0%EC%84%A0%EC%9D%98_%EC%97%AD%EB%8C%80_%EC%99%95%EB%8C%80%EB%B9%84" title="조선의 역대 왕대비">왕대비</a>에 올라 <b>인수대비</b>(仁粹大妃)가 되었다.</p>
<p><a href="/wiki/%EC%9D%98%EC%A0%95%EB%B6%80" title="의정부">의정부</a><a href="/wiki/%EC%A2%8C%EC%9D%98%EC%A0%95" title="좌의정">좌의정</a>을 지낸 서원부원군 양절공 <a href="/wiki/%ED%95%9C%ED%99%95" title="한확">한확</a>(西城府院君 襄節公 韓確)과 남양부부인 홍씨(南陽府夫人 洪氏) 여섯째 딸로, 본관은 <a href="/wiki/%EC%B2%AD%EC%A3%BC_%ED%95%9C%EC%94%A8" title="청주 한씨">청주</a>(淸州)이다. 당시 여성으로서는 드물게 한문과 유교 경전에 능통하여 《열녀》,《여교》,《명감》,《소학》등에서 발췌하여 엮어 《<a href="/wiki/%EB%82%B4%ED%9B%88" title="내훈">내훈</a>》(內訓)을 편찬하였다. 내훈은 조선시대 사대부 여인들의 수신서이자 당시 여성교육의 기본서가 되었다. 또한 그녀는 <a href="/wiki/%EB%B6%88%EA%B5%90" title="불교">불교</a> 옹호론자로 <a class="mw-redirect" href="/wiki/%EC%88%AD%EC%9C%A0%EC%96%B5%EB%B6%88" title="숭유억불">불교 억압 정책</a>에 강력히 반발하였으며, 이 때문에 당시 조정의 신하들과 4차례의 격한 논쟁을 벌였는데 대표적인 것이 금승법(禁僧法)<sup class="reference" id="cite_ref-1"><a href="#cite_note-1">[1]</a></sup> 과 그녀가 추진한 <a href="/wiki/%EB%B4%89%EC%84%A0%EC%82%AC" title="봉선사">봉선사</a> 금자경 간행 작업이다.</p>
<p>그녀는 <a class="mw-redirect" href="/wiki/%EC%97%B0%EC%82%B0%EA%B5%B0" title="연산군">연산군</a>의 할머니로도 유명한데, 며느리이자 연산군의 생모가 되는 <a href="/wiki/%ED%8F%90%EB%B9%84_%EC%9C%A4%EC%94%A8" title="폐비 윤씨">윤씨</a>가 왕비 시절 성종의 얼굴을 할퀴는 사건으로 내쫓기고 사사되는 데에는 거의 전적으로 그녀의 의지로 단행되었기 때문이다.<sup class="reference" id="cite_ref-2"><a href="#cite_note-2">[2]</a></sup> <a href="/wiki/1504%EB%85%84" title="1504년">1504년</a>(연산군 10년) 봄에 연산군은 생모인 폐비 윤씨를 제헌왕후(齊獻王后)로 추숭했다. 이 과정에서 윤씨를 폐비하고 사사하는 데 개입한 수많은 사람들을 죽이고 추방했는데, 인수대비는 당사자가 되므로 손자인 연산군과 갈등을 빚었다. 능은 <a href="/wiki/%EA%B2%BD%EA%B8%B0%EB%8F%84" title="경기도">경기도</a> <a href="/wiki/%EA%B3%A0%EC%96%91%EC%8B%9C" title="고양시">고양시</a> 덕양구 용두동에 위치한 <a href="/wiki/%EA%B2%BD%EB%A6%89_(%EC%A1%B0%EC%84%A0_%EB%8D%95%EC%A2%85)" title="경릉 (조선 덕종)">경릉</a>(敬陵)이다.</p>

소혜왕후의 작은 고모부<br/>
<a href="/wiki/%EC%84%A0%EB%8D%95%EC%A0%9C" title="선덕제">명나라 제5대 황제 선종</a></div>

<p>한씨의 아버지 <a href="/wiki/%ED%95%9C%ED%99%95" title="한확">한확</a>은 당시 중추원사(中樞院使)를 지내면서 <a href="/wiki/%EB%AA%85%EB%82%98%EB%9D%BC" title="명나라">명나라</a>로부터 광록시소경(光祿寺少卿)작위를 하사받은 조선 최고의 세도가였다.</p>
<p>또한 한씨의 큰 고모는 <a href="/wiki/%EB%AA%85%EB%82%98%EB%9D%BC" title="명나라">명나라</a> 제 3대 황제 <a href="/wiki/%EC%98%81%EB%9D%BD%EC%A0%9C" title="영락제">영락제 성조</a>의 후궁이었던 강혜장숙여비(康惠莊淑麗妃)였으며, 작은 고모 역시 <a href="/wiki/%EB%AA%85%EB%82%98%EB%9D%BC" title="명나라">명나라</a> 제 5대 황제 <a href="/wiki/%EC%84%A0%EB%8D%95%EC%A0%9C" title="선덕제">선덕제 선종</a>의 후궁인 공신태비(恭愼太妃)였다. 큰고모가 영락제 사후 순사되었는데, 그의 절개를 높이 평가한다는 명목으로 명나라 선덕제는 그의 둘째 고모를 후궁으로 맞이하였다.</p>
<p>그리고 한씨의 둘째언니는 <a href="/wiki/%EC%A1%B0%EC%84%A0_%EC%84%B8%EC%A2%85" title="조선 세종">세종</a>의 둘째 서자인 <a class="mw-redirect" href="/wiki/%EA%B3%84%EC%96%91%EA%B5%B0_(%EC%99%95%EC%A1%B1)" title="계양군 (왕족)">계양군</a>의 부인으로 한씨의 가문은 양국의 왕실과 인척관계를 맺었고 한확은 명나라와의 외교를 전담하던 당시의 외교관으로서 명나라의 총애를 바탕으로 출세가도를 달렸다.</p>
<h3><span class="mw-headline" id=".EA.B5.B0.EB.B6.80.EC.9D.B8_.EC.8B.9C.EC.A0.88">군부인 시절</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%86%8C%ED%98%9C%EC%99%95%ED%9B%84&amp;action=edit&amp;section=3" title="부분 편집: 군부인 시절">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<div class="rellink relarticle mainarticle"><img alt="" data-file-height="128" data-file-width="128" height="16" src="//upload.wikimedia.org/wikipedia/commons/thumb/e/ec/Crystal_Clear_app_xmag.svg/16px-Crystal_Clear_app_xmag.svg.png" srcset="//upload.wikimedia.org/wikipedia/commons/thumb/e/ec/Crystal_Clear_app_xmag.svg/24px-Crystal_Clear_app_xmag.svg.png 1.5x, //upload.wikimedia.org/wikipedia/commons/thumb/e/ec/Crystal_Clear_app_xmag.svg/32px-Crystal_Clear_app_xmag.svg.png 2x" width="16"/> 이 부분의 본문은 <a class="mw-redirect" href="/wiki/%EC%88%98%EC%96%91%EB%8C%80%EA%B5%B0" title="수양대군">수양대군</a>, <a class="mw-redirect" href="/wiki/%EC%9D%98%EA%B2%BD%EC%84%B8%EC%9E%90" title="의경세자">의경세자</a>입니다.</div>
<p><a href="/wiki/1450%EB%85%84" title="1450년">1450년</a>, <a href="/wiki/%EC%A1%B0%EC%84%A0_%EB%AC%B8%EC%A2%85" title="조선 문종">문종</a>이 병약하고 <a href="/wiki/%EC%A1%B0%EC%84%A0_%EB%8B%A8%EC%A2%85" title="조선 단종">어린 세자</a>사이에서 <a class="mw-disambig" href="/wiki/%EA%B9%80%EC%A2%85%EC%84%9C" title="김종서">김종서</a>와 대립하고 있었던 <a href="/wiki/%EC%A1%B0%EC%84%A0_%EC%84%B8%EC%A1%B0" title="조선 세조">수양대군</a>은 자신의 정치적 입지를 넓히기 위해서 큰아들인 <a href="/wiki/%EC%A1%B0%EC%84%A0_%EB%8D%95%EC%A2%85" title="조선 덕종">도원군</a>과 세도가 <a href="/wiki/%ED%95%9C%ED%99%95" title="한확">한확</a>의 막내 여식인 한씨의 정략결혼을 추진했다. 소혜왕후는 수양대군의 이복동생인 계양군의 처제였고, 시어머니가 될 정희왕후의 형부인 홍원용의 생질녀(누이의 딸)였고, 외종언니인 강녕부부인(홍이용의 딸)은 <a href="/wiki/%ED%8F%89%EC%9B%90%EB%8C%80%EA%B5%B0" title="평원대군">평원대군</a>(세종의 일곱번째 왕자)의 정실이었으므로 여러모로 왕실과 인연이 깊었다.</p>
<p>한씨의 나이 13살에 수양대군의 큰아들 도원군과 혼인하여 도원군부인(桃源君夫人)가 되었다. 남편 도원군이 당시 대군(大君)의 적장자로 아직 승습(承襲)하지 않았고, 종친부에는 정의대부(正義大夫)로 기록된 점으로 보아 한씨의 품계는 정2품(正二品) 군부인(郡夫人)임을 알 수 있으나 작호는 <a href="/wiki/%EC%A1%B0%EC%84%A0%EC%99%95%EC%A1%B0%EC%8B%A4%EB%A1%9D" title="조선왕조실록">조선왕조실록</a>에 따로 기록된 바가 없다.<sup class="reference" id="cite_ref-3"><a href="#cite_note-3">[3]</a></sup></p>
<p><a href="/wiki/1454%EB%85%84" title="1454년">1454년</a> <a class="mw-redirect" href="/wiki/%EA%B0%91%EC%88%A0%EB%85%84" title="갑술년">갑술년</a>에는 잠저에서 첫 번째 자녀인 <a href="/wiki/%EC%9B%94%EC%82%B0%EB%8C%80%EA%B5%B0" title="월산대군">월산군</a>을 낳았다.</p>
<h3><span class="mw-headline" id=".EC.99.95.EC.84.B8.EC.9E.90.EB.B9.88_.EC.8B.9C.EC.A0.88">왕세자빈 시절</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%86%8C%ED%98%9C%EC%99%95%ED%9B%84&amp;action=edit&amp;section=4" title="부분 편집: 왕세자빈 시절">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p><a href="/wiki/1455%EB%85%84" title="1455년">1455년</a>, 시아버지 수양대군이 단종에게 양위를 받아 즉위하고 아들인 도원군을 <a href="/wiki/%EC%99%95%EC%84%B8%EC%9E%90" title="왕세자">왕세자</a>로 책봉하자 자신도 <a href="/wiki/%EC%99%95%EC%84%B8%EC%9E%90%EB%B9%88" title="왕세자빈">왕세자빈</a>이 되었다.<sup class="reference" id="cite_ref-4"><a href="#cite_note-4">[4]</a></sup> 그해 동궁(東宮)에서 두 번째 자녀인 <a href="/wiki/%EB%AA%85%EC%88%99%EA%B3%B5%EC%A3%BC" title="명숙공주">태안군주</a>를 낳고, 익익년(翌翌年) <a href="/wiki/1457%EB%85%84" title="1457년">1457년</a>에는 세 번째 자녀인 <a href="/wiki/%EC%A1%B0%EC%84%A0_%EC%84%B1%EC%A2%85" title="조선 성종">자산군</a>(훗날 성종)을 낳았다.</p>

소혜왕후의 시아버지<br/>
<a href="/wiki/%EC%A1%B0%EC%84%A0_%EC%84%B8%EC%A1%B0" title="조선 세조">수양대군(세조)</a></div>

<p>왕세자빈 한씨는 젊어서부터 빈틈이 없고, 시부모(세조와 <a href="/wiki/%EC%A0%95%ED%9D%AC%EC%99%95%ED%9B%84" title="정희왕후">정희왕후</a>) 섬기는 일엔 극직해서 세조로부터 효부(孝婦)라는 칭찬을 늘 들었지만 두아들 월산군과 자산군에겐 엄한 어머니였다. 사소한 과실만 있어도 추호도 감싸는 법 없이 정색을 하고 꾸짖었다는 것이다. 그래서 세조(수양대군), 중전 윤씨(정희왕후) 내외가 농담 삼아 부르게 된 별명이 폭빈(暴嬪)이었다.</p>
<h4><span class="mw-headline" id=".EB.82.A8.ED.8E.B8.EC.9D.98_.EC.A3.BD.EC.9D.8C">남편의 죽음</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%86%8C%ED%98%9C%EC%99%95%ED%9B%84&amp;action=edit&amp;section=5" title="부분 편집: 남편의 죽음">편집</a><span class="mw-editsection-bracket">]</span></span></h4>
<p><a href="/wiki/1457%EB%85%84" title="1457년">1457년</a>, 친정아버지 한확의 객사와 함께 남편인 의경세자(懿敬世子)가 20살의 나이로 요절하였다. 그녀의 연이은 불행을 안쓰럽게 여긴 세조는 그를 특별히 총애하여 궁궐에서 살도록 허용하였으나 스스로 사양하였다.</p>
<p>시동생이 되는 <a href="/wiki/%EC%A1%B0%EC%84%A0_%EC%98%88%EC%A2%85" title="조선 예종">해양대군(예종)</a>이 남편의 뒤를 이어 왕세자에 책봉되면서 시아버지 세조로부터 정빈(貞嬪)의 작호를 받아 정빈 한씨(貞嬪 韓氏)가 되었으나, <a class="mw-redirect" href="/wiki/%EC%9D%84%EC%9C%A0%EB%85%84" title="을유년">을유년</a>인 <a href="/wiki/1465%EB%85%84" title="1465년">1465년</a>에 세조가 교지를 통해 <a href="/wiki/%EC%9B%90%EA%B2%BD%EC%99%95%ED%9B%84" title="원경왕후">원경왕후</a>가 세자빈이 되었을 때 정빈으로 책봉하였으므로 한씨의 작호를 수빈(粹嬪)으로 고쳤다.<sup class="reference" id="cite_ref-5"><a href="#cite_note-5">[5]</a></sup></p>
<h3><span class="mw-headline" id=".EC.88.98.EB.B9.88.28.E7.B2.B9.E5.AC.AA.29_.EC.8B.9C.EC.A0.88">수빈(粹嬪) 시절</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%86%8C%ED%98%9C%EC%99%95%ED%9B%84&amp;action=edit&amp;section=6" title="부분 편집: 수빈(粹嬪) 시절">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<div class="rellink boilerplate seealso"><img alt="" data-file-height="128" data-file-width="128" height="16" src="//upload.wikimedia.org/wikipedia/commons/thumb/e/ec/Crystal_Clear_app_xmag.svg/16px-Crystal_Clear_app_xmag.svg.png" srcset="//upload.wikimedia.org/wikipedia/commons/thumb/e/ec/Crystal_Clear_app_xmag.svg/24px-Crystal_Clear_app_xmag.svg.png 1.5x, //upload.wikimedia.org/wikipedia/commons/thumb/e/ec/Crystal_Clear_app_xmag.svg/32px-Crystal_Clear_app_xmag.svg.png 2x" width="16"/> <a href="/wiki/%EB%8D%95%EC%88%98%EA%B6%81" title="덕수궁">덕수궁</a>, <a href="/wiki/%ED%95%9C%EB%AA%85%ED%9A%8C" title="한명회">한명회</a> 문서를 참고하십시오.</div>
<p>한씨는 <a href="/wiki/%EC%99%95%EC%84%B8%EC%9E%90%EB%B9%88" title="왕세자빈">왕세자빈</a> 자격으로 궁궐에서 살았으나 남편 의경세자의 죽음으로 그 지위를 잃고 두 아들(월산군, 자산군)과 함께 출궁해야만 했다. 이를 안타깝게 여기던 시아버지 세조가 그녀에게 특별히 궁궐에서 살아도 좋다고 하였으나 한씨가 이를 사양하였다.</p>
<p>세조가 맏며느리 한씨에게 특별히 궁궐에서 살아도 좋다고 하였음에도 이를 사양하고 두 아들과 궁궐을 떠나려하자 이를 가엽게 여긴 세조는 그녀를 위해 특별히 집을 지어주었는데 그곳이 바로 지금의 <a href="/wiki/%EB%8D%95%EC%88%98%EA%B6%81" title="덕수궁">덕수궁</a>이다. 한씨의 죽은 남편 의경세자 사당 옆에 건설되었으며, 일반적으로 고위 사대부나 왕족이 살던 사저보다는 그 규모가 크고 웅장하여 훗날 <a href="/wiki/%EC%A1%B0%EC%84%A0_%EC%84%A0%EC%A1%B0" title="조선 선조">선조</a>가 <a href="/wiki/%EC%9E%84%EC%A7%84%EC%99%9C%EB%9E%80" title="임진왜란">임진왜란</a>중에 임시 거처로 사용하기에 이른다. 후에 한씨의 작은 아들 자산군이 보위에 올라 그녀도 모후(임금의 어머니)로서 다시 궁궐에 들어가자 그녀의 큰아들 월산대군이 사저를 물려받았다.</p>

소혜왕후가 수빈 시절 거처했던<br/>
<a href="/wiki/%EB%8D%95%EC%88%98%EA%B6%81" title="덕수궁">덕수궁 중화전</a></div>

<p><a href="/wiki/1467%EB%85%84" title="1467년">1467년</a>에는 수빈 한씨가 병으로 자신의 둘째 언니인 계양군부인 집에서 요양하고 있었는데 시아버지이자 임금이었던 세조가 궁궐에서 몸소 나와 계양군부인 집에 문병차 거둥하였다.<sup class="reference" id="cite_ref-6"><a href="#cite_note-6">[6]</a></sup> 그리고 병조(兵曹)에 명하여, 군사 10명을 보내어 수빈이 요양하고 있는 계양군부인 집을 지키고 호위하게 하였다. 그리고 한 달 뒤에는 정희왕후와 동행하였다. 이듬해에는 엄청난 양의 곡물과 땅을 하사하였다.<sup class="reference" id="cite_ref-7"><a href="#cite_note-7">[7]</a></sup><sup class="reference" id="cite_ref-8"><a href="#cite_note-8">[8]</a></sup><sup class="reference" id="cite_ref-9"><a href="#cite_note-9">[9]</a></sup></p>
<p>남편 의경세자의 죽음으로 중전의 꿈을 접어야만 했지만 정치적 야심이 대단했던 그녀는 당대의 권신 <a href="/wiki/%ED%95%9C%EB%AA%85%ED%9A%8C" title="한명회">한명회</a>의 넷째 딸 <a href="/wiki/%EA%B3%B5%ED%98%9C%EC%99%95%ED%9B%84" title="공혜왕후">한씨</a>(훗날 공혜왕후)와 자신의 둘째 아들 자산군을 혼인시켜 사돈관계를 맺고 자신의 아들에게 왕위를 계승시키기 위한 노력을 계속하였다. 궁궐을 나왔지만 세조와 정희왕후를 정성껏 봉양했고, 특히 왕위 결정권자인 정희왕후와의 관계를 돈독히 했다.</p>
<p>또한 <a href="/wiki/%EC%8B%A0%EC%88%99%EC%A3%BC" title="신숙주">신숙주</a> 등과도 긴밀하게 교류하였는데, 이러한 노력의 결과로 세조의 뒤를 이어 즉위한 예종이 승하한 후에 한명회의 강력한 추천과 시어머니인 정희왕후의 지지에 힘입어 당시 원자였던 예종과 <a href="/wiki/%EC%95%88%EC%88%9C%EC%99%95%ED%9B%84" title="안순왕후">안순왕후</a>의 아들 <a href="/wiki/%EC%A0%9C%EC%95%88%EB%8C%80%EA%B5%B0" title="제안대군">제안대군</a> 대신 자산군이 즉위할 수 있었다.</p>
<h4><span class="mw-headline" id=".EC.8B.9C.EB.8F.99.EC.83.9D.28.EC.98.88.EC.A2.85.29_.EC.A6.89.EC.9C.84">시동생(예종) 즉위</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%86%8C%ED%98%9C%EC%99%95%ED%9B%84&amp;action=edit&amp;section=7" title="부분 편집: 시동생(예종) 즉위">편집</a><span class="mw-editsection-bracket">]</span></span></h4>
<p>수빈 한씨의 시아버지 세조가 한씨에게는 시동생이 되는 해양대군(예종)에게 양위하고 곧 승하하였다. 왕의 맏며느리에서, 왕의 형수로 처지가 바뀌었지만 영향력과 위세는 여전히 대단했다.</p>
<p><a href="/wiki/%EC%A1%B0%EC%84%A0_%EC%98%88%EC%A2%85" title="조선 예종">예종</a>은 병조에“수빈(粹嬪)에게 대군(大君)의 예(例)에 의하여 반인(수행원)을 지급하라."고 명하였고<sup class="reference" id="cite_ref-10"><a href="#cite_note-10">[10]</a></sup> 엽관운동(獵官運動), 즉 분경(奔競)이 심했었던 <a href="/wiki/1469%EB%85%84" title="1469년">1469년</a>에는 교지를 통해 "대비(정희왕후)와 중궁(안순왕후), 수빈 이외에 벼슬을 자청(自請)하는 자는 그 직첩(職牒)을 거두고 영원히 서용(敍用)하지 말라."고 했을정도로 수빈의 위세와 영향력이 대단하였다.<sup class="reference" id="cite_ref-11"><a href="#cite_note-11">[11]</a></sup></p>
<p>또한 수빈의 동생 <a href="/wiki/%ED%95%9C%EC%B9%98%EC%9D%98" title="한치의">한치의</a>(韓致義)는 그녀의 아우라는 이유만으로 <a href="/wiki/%EB%8B%B9%EC%83%81%EA%B4%80" title="당상관">당상관</a>(堂上官)이 되어 공신의 호를 받고 청성군(淸城君)에 올랐으며, 현재의 <a href="/wiki/%EB%8C%80%ED%95%9C%EB%AF%BC%EA%B5%AD%EC%9D%98_%EA%B5%AD%EB%B0%A9%EB%B6%80_%EC%9E%A5%EA%B4%80" title="대한민국의 국방부 장관">국방부장관</a>에 해당하는 <a class="mw-redirect" href="/wiki/%EB%B3%91%EC%A1%B0%ED%8C%90%EC%84%9C" title="병조판서">병조판서</a>에까지 오른다.<sup class="reference" id="cite_ref-12"><a href="#cite_note-12">[12]</a></sup></p>
<p>이 시기에 그녀가 정치적인 행보를 이어나간 것은 시어머니가 되는 대비 윤씨와 함께 숭은전(崇恩殿)에 나아갔을 때였다.<br/>
능성군(綾城君) 구치관(具致寬), 영순군(永順君) 이부(李溥), 밀성군(密城君) 이침(李琛), 은천군(銀川君) 이찬(李穳), 사산군(蛇山君) 이호(李灝), 운산군(雲山君) 이계(李誡), 강양군(江陽君) 이융(李瀜), 곡성군(鵠城君) 이금손(李金孫), 모양 도정(牟陽都正) 이직(李稙), 청송군(靑松君) 심회(沈澮), 하성군(河城君) 정현조(鄭顯祖), 중추부 지사(中樞府知事) 윤사흔(尹士昕), 우찬성(右贊成) 노사신(盧思愼), 호조 참판(戶曹參判) 한치인(韓致仁), 병조 참판(兵曹參判) 한의(韓嶬) 등의 수많은 공신들과 종친, 외척들이 그녀와 대비를 호위하였다.<sup class="reference" id="cite_ref-13"><a href="#cite_note-13">[13]</a></sup> 이것은 당시 그녀의 위세와 영향력을 보여주는 또 하나의 예이다.</p>
<p>시동생 예종이 재위기간 14개월만에 갑작스럽게 죽자 조정에서 왕위 계승을 둘러싸고 논의가 있었다. 마땅히 예종의 맏아들이 되는 <a href="/wiki/%EC%A0%9C%EC%95%88%EB%8C%80%EA%B5%B0" title="제안대군">원자</a>(제안대군)이 보위에 올랐어야 했으나 원자의 나이가 겨우 3살밖에 되질 않았을뿐더러, 수빈 한씨의 둘째 아들인 자산군의 장인이 바로 권신 한명회이었던 점이 강력하게 작용하여 조정은 이미 자을산군을 새 임금으로 추대하는 과정으로 접어들고 있었다. 왕실의 어른 대비 윤씨(정희왕후)와 원상 신숙주, 한명회 등의 결정으로 의경세자와 수빈 한씨의 둘째 아들 자산군이 보위에 오르게 된다.<sup class="reference" id="cite_ref-14"><a href="#cite_note-14">[14]</a></sup></p>
<h3><span class="mw-headline" id=".EC.84.B1.EC.A2.85_.EC.A6.89.EC.9C.84_.EC.B4.88">성종 즉위 초</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%86%8C%ED%98%9C%EC%99%95%ED%9B%84&amp;action=edit&amp;section=8" title="부분 편집: 성종 즉위 초">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>수빈 한씨의 아들 자산군이 성종으로 왕위에 올랐으나 그녀는 아들의 인사를 받지 못하였다. 임금의 어머니(모후)가 되므로 마땅히 인사를 받아야 했지만, 성종은 생부인 의경세자가 아닌 작은아버지가 되는 예종의 아들로 입적하여 왕위(王位)의 대통(大統)을 이었기 때문에 법적으로는 수빈 한씨가 성종의 어머니로 인정받지 못한 것이다.</p>
<p>오히려 예종의 계비인 <a href="/wiki/%EC%95%88%EC%88%9C%EC%99%95%ED%9B%84" title="안순왕후">안순왕후 한씨</a>가 인혜왕대비(仁惠王大妃)로 성종의 법적 어머니가 되었다. 따라서 수빈 한씨는 국왕의 모후(母后)가 되는 왕대비의 자격이 아니라 그저 왕세자의 부인으로서 지위가 세자빈에 불과했기 때문에 군신관계(君臣關係)상 아들의 인사를 받지 못한 것이다.</p>
<p>다만 국왕의 생모가 되므로, 궁궐에 들어와 살았는데 그녀의 호(呼)를 붙여 수빈궁(粹嬪宮)이라 했다.<sup class="reference" id="cite_ref-15"><a href="#cite_note-15">[15]</a></sup> 이전에 거주하고 있었던 사저(<a href="/wiki/%EB%8D%95%EC%88%98%EA%B6%81" title="덕수궁">덕수궁</a>)는 그녀의 장남인 월산군이 물려받았다.</p>
<p>수빈 한씨가 성종의 어머니임에도 인사를 받지 못하는 불상사가 벌어지자 조정에서 이 일이 논의 중에 있었다. 문제는 그녀를 왕비로 추숭하느냐, 왕대비로 추숭하느냐였다.</p>
<p>이 문제는 그녀의 남편인 의경세자를 왕(王)으로 추숭하느냐, 왕(王)과 더불어 종(宗)으로 추숭하느냐의 문제와 연관되어 있었다. 의경세자를 단순히 왕(王)으로 추숭하면 한씨는 왕비가 되는 것이었고, 의경세자를 왕(王)과 더불어 종(宗)으로 추숭하면 한씨는 왕대비가 되는 것이었다.<sup class="reference" id="cite_ref-insu100_16-0"><a href="#cite_note-insu100-16">[16]</a></sup></p>
<p>한씨를 왕비(王妃)로 추숭해야 한다고 주장한 대신들은 하동부원군(河東府院君) <a href="/wiki/%EC%A0%95%EC%9D%B8%EC%A7%80" title="정인지">정인지</a>, 봉원부원군(蓬原府院君) <a href="/wiki/%EC%A0%95%EC%B0%BD%EC%86%90" title="정창손">정창손</a>, 고령부원군(高靈府院君) 신숙주, 상당부원군(上黨府院君) 한명회, 능성부원군(綾城府院君) <a href="/wiki/%EA%B5%AC%EC%B9%98%EA%B4%80" title="구치관">구치관</a>(具致寬), 인산부원군(仁山府院君) <a href="/wiki/%ED%99%8D%EC%9C%A4%EC%84%B1" title="홍윤성">홍윤성</a> 등으로 전부 <a href="/wiki/%EC%98%81%EC%9D%98%EC%A0%95" title="영의정">영의정</a>과 <a href="/wiki/%EB%B6%80%EC%9B%90%EA%B5%B0" title="부원군">부원군</a>의 작위를 갖고 있었던 영향력 있는 공신들이었고<sup class="reference" id="cite_ref-insu100_16-1"><a href="#cite_note-insu100-16">[16]</a></sup></p>
<p>한씨를 왕대비로 추숭해야 한다고 주장한 대신들은 남원군(南原君) <a href="/wiki/%EC%96%91%EC%84%B1%EC%A7%80" title="양성지">양성지</a>(梁誠之), 양성군(陽城君) <a class="new" href="/w/index.php?title=%EC%9D%B4%EC%8A%B9%EC%86%8C&amp;action=edit&amp;redlink=1" title="이승소 (없는 문서)">이승소</a>(李承召) 등이었다.<sup class="reference" id="cite_ref-insu100_16-2"><a href="#cite_note-insu100-16">[16]</a></sup></p>
<p>힘있는 권신들의 생각이 모두 "수빈 한씨를 왕비로 추숭해야 한다."로 쏠렸으므로 <a href="/wiki/%EC%8B%A0%EC%88%99%EC%A3%BC" title="신숙주">신숙주</a>가 "의경세자를 추숭하여 의경왕(懿敬王)을 삼으나 종(宗)이라 일컫지 말며, 수빈은 다만 인수왕비(仁粹王妃)라고만 일컬어 예종비 인혜왕대비(안순왕후)와 구별하는 것이 편리하겠습니다." 하여 그대로 되었다.<sup class="reference" id="cite_ref-insu100_16-3"><a href="#cite_note-insu100-16">[16]</a></sup></p>

<h4><span class="mw-headline" id=".EC.9D.B8.EC.88.98.EC.99.95.EB.B9.84.28.E4.BB.81.E7.B2.B9.E7.8E.8B.E5.A6.83.29.EA.B0.80_.EB.90.98.EB.8B.A4">인수왕비(仁粹王妃)가 되다</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%86%8C%ED%98%9C%EC%99%95%ED%9B%84&amp;action=edit&amp;section=9" title="부분 편집: 인수왕비(仁粹王妃)가 되다">편집</a><span class="mw-editsection-bracket">]</span></span></h4>
<p>결국 조정의 논의 끝에 수빈 한씨가 인수왕비(仁粹王妃)가 되어 아들 성종의 인사를 받을 수 있었으나 그녀에게는 달가운 일이 아니었다. 왕비라 함은 왕(王)의 부인을 일컫는 것인데, 한씨는 왕(王)의 어머니였기 때문에 그녀의 입장에서는 왕대비가 되어야 마땅했다. 바로 인혜왕대비(仁惠王大妃)와의 서열문제가 화두가 되었다. 왕실에는 성종의 할머니가 되는 자성대왕대비 윤씨(정희왕후), 법적 어머니이자 숙모가 되는 인혜왕대비 한씨(안순왕후), 그리고 친어머니이지만 법적으로는 큰어머니가 되는 인수왕비 한씨 이렇게 3명의 윗전이 있었다. 왕실 서열 1위는 할머니가 되는 자성대왕대비 윤씨(慈聖大王大妃 尹氏)로 문제가 없었으나 왕실 서열 2위가 인혜왕대비냐 인수왕비냐를 놓고 조정에서 논의가 있었다.</p>
<p>당시 사관(史官)이 성종에게 아뢰길 "예종비였던 인혜왕대비는 당시 <a href="/wiki/%EC%A1%B0%EC%84%A0%EC%9D%98_%EC%97%AD%EB%8C%80_%EC%99%95%EB%B9%84" title="조선의 역대 왕비">왕비</a>였고, 인수왕비는 수빈(粹嬪)으로 군신관계에 있었기 때문에 마땅히 인혜왕대비를 윗전으로 해야합니다." 라고 하였으나 원상(院相) 신숙주가 아뢰길 "인수왕비는 이미 존호(尊號)를 높이어 명위(名位)가 이미 정하여졌으니, 마땅히 형제의 서열로 차서를 하여야 합니다." 하였고<sup class="reference" id="cite_ref-17"><a href="#cite_note-17">[17]</a></sup> 왕실의 최고 어른이 되는 대왕대비 또한 <a href="/wiki/%EC%98%88%EC%A1%B0" title="예조">예조</a>에 전지(傳旨)하기를 "인혜왕대비의 서열이 일찍이 인수왕비의 위에 있었으나, 세조대왕께서 항시 인수왕비에게 명하여 예종을 보호하게 하고 시양(侍養)이라고 일컬었으며, 또 장유(長幼)의 차서(次序)가 있으니, 인수왕비의 서열은 마땅히 인혜왕대비의 위에 두어야 한다."고 하였다. 따라서 인수왕비는 세자빈 출신임에도 불구하고 왕실 서열 2위가 되었고, 인혜왕대비는 왕비 출신임에도 왕실 서열 3위로 밀려났다.<sup class="reference" id="cite_ref-18"><a href="#cite_note-18">[18]</a></sup></p>

<p>인수왕비 한씨가 왕실 서열 2위로 올라섰지만, 그녀에게는 <a href="/wiki/%EC%99%95%EB%8C%80%EB%B9%84" title="왕대비">왕대비</a>에 오르는 것이 여전히 숙제로 남아있었다. <a href="/wiki/1474%EB%85%84" title="1474년">1474년</a>, 마침내 성종의 의지로 생부인 의경왕(懿敬王)을 추봉(追封)하였고, 그 결과 의경왕이 덕종(德宗)의 묘호를 받았다. 왕의 생모가 되는 인수왕비 또한 왕대비로 진봉되어 인수왕대비(仁粹王大妃)가 되었다.<sup class="reference" id="cite_ref-19"><a href="#cite_note-19">[19]</a></sup></p>
<p>왕실에 세 윗전 자성대왕대비, 인수왕대비, 인혜왕대비를 모시기 위해 <a href="/wiki/%EC%B0%BD%EA%B2%BD%EA%B6%81" title="창경궁">창경궁</a>을 건축하였다. 명정전(明政殿), 편전인 문정전(文政殿), 침전인 수령전(壽寧殿), 그리고 환경전(歡慶殿), 경춘전(景春殿), 인양전(仁陽殿), 통명전(通明殿), 양화당(養和堂), 여휘당(麗暉堂), 사성각(思誠閣)등이 건립되었으며 인수왕대비는 이 중 경춘전(景春殿)에서 거처하였고 그곳에서 또한 죽음을 맞이한다.</p>
<h3><span class="mw-headline" id=".EB.8C.80.EC.99.95.EB.8C.80.EB.B9.84.28.E5.A4.A7.E7.8E.8B.E5.A4.A7.E5.A6.83.29_.EC.8B.9C.EC.A0.88">대왕대비(大王大妃) 시절</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%86%8C%ED%98%9C%EC%99%95%ED%9B%84&amp;action=edit&amp;section=11" title="부분 편집: 대왕대비(大王大妃) 시절">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p><a href="/wiki/1494%EB%85%84" title="1494년">1494년</a>, 성종이 승하하고 왕세자 융이 즉위하였는데 그가 바로 조선의 제10대 임금 <a class="mw-redirect" href="/wiki/%EC%97%B0%EC%82%B0%EA%B5%B0" title="연산군">연산군</a>이다. 인수대비 한씨는 연산군의 할머니가 되므로 <a href="/wiki/%EB%8C%80%EC%99%95%EB%8C%80%EB%B9%84" title="대왕대비">대왕대비</a>로 진봉되었으며, 휘호가 자숙(慈淑)으로 새로이 올려졌다. 다만 실록에는 자숙대왕대비(慈淑大王大妃)라는 명칭이 휘호를 올린 당시에만 등장할 뿐, 인수대왕대비(仁粹大王大妃)로 계속 일컬어 진다.<sup class="reference" id="cite_ref-20"><a href="#cite_note-20">[20]</a></sup></p>
<p>왕세자 시절 자신이 <a href="/wiki/%EC%A0%95%ED%98%84%EC%99%95%ED%9B%84" title="정현왕후">정현왕후</a>의 아들인 줄로만 알고 있었던 연산군은 즉위 이후 자신이 폐비 윤씨의 아들임을 알게 되고 급기야는 폐비 윤씨를 모함하여 사사(賜死)시킨 <a href="/wiki/%EA%B7%80%EC%9D%B8_%EC%A0%95%EC%94%A8_(%EC%84%B1%EC%A2%85)" title="귀인 정씨 (성종)">귀인 정씨</a>와 <a href="/wiki/%EA%B7%80%EC%9D%B8_%EC%97%84%EC%94%A8" title="귀인 엄씨">귀인 엄씨</a>를 철퇴로 내리쳐 죽인 뒤, 그 시체를 찢어 젓갈로 만들어 야산에 뿌리도록 했고 그 뒤에도 귀인 정씨의 아들인 <a href="/wiki/%EC%95%88%EC%96%91%EA%B5%B0" title="안양군">안양군</a>과 <a href="/wiki/%EB%B4%89%EC%95%88%EA%B5%B0" title="봉안군">봉안군</a>을 강제로 대왕대비전으로 끌고가 인수대비에게 술잔을 강제로 따르게 하였으며 그 뒤에도 '왜 나의 어머니를 죽였느냐'고 인수대비에게 고함을 지르며 행패를 부렸다.<sup class="reference" id="cite_ref-21"><a href="#cite_note-21">[21]</a></sup></p>
<p>이후 연산군이 생모인 <a href="/wiki/%ED%8F%90%EB%B9%84_%EC%9C%A4%EC%94%A8" title="폐비 윤씨">폐비 윤씨</a>를 제헌왕후(齊獻王后)로 추존하려 하자 병상에 있던 인수대왕대비가 이를 꾸짖었으나 화가 치밀어오른 연산군은 인수대왕대비의 가슴을 밀쳤다. 이에 인수대왕대비는 얼마 후 그 충격으로 <a class="mw-redirect" href="/wiki/%EA%B0%91%EC%9E%90%EB%85%84" title="갑자년">갑자년</a>인 1504년 4월 27일, 68세의 나이로 승하하였다.<sup class="reference" id="cite_ref-insu101_22-0"><a href="#cite_note-insu101-22">[22]</a></sup></p>
<p>인수대왕대비는 이미 죽기 서너 달 전인 1504년 1월경에 노환으로 목숨이 매우 위태로운 지경에 이르렀는데, 이 때문에 당시 연산군은 의정부의 삼정승(三政丞)과 육조의 판서들을 불러 미리 상제(喪制)를 의논하고 있던 시기였다.<sup class="reference" id="cite_ref-23"><a href="#cite_note-23">[23]</a></sup><sup class="reference" id="cite_ref-24"><a href="#cite_note-24">[24]</a></sup> 그런 상황에서 연산군은 병석에 누워있던 할머니에게 고함을 하면서 박치기를 하는 행패를 부려 죽음에 이르게 한 것이다. 조선은 <a href="/wiki/%EC%9C%A0%EA%B5%90" title="유교">유교</a> 사상을 근본으로 하는 나라였고, 그 중 임금에게 적용되는 가장 큰 덕목은 효(孝)이므로, 이와 같은 연산군의 행동은 패륜으로 훗날 <a href="/wiki/%EB%B0%95%EC%9B%90%EC%A2%85" title="박원종">박원종</a>이 중심이 되는 <a href="/wiki/%EC%A4%91%EC%A2%85%EB%B0%98%EC%A0%95" title="중종반정">중종반정</a>의 명분을 제공하였다.</p>

소혜왕후가 승하한 <a href="/wiki/%EC%B0%BD%EA%B2%BD%EA%B6%81" title="창경궁">경춘전</a>(景春殿)<br/>
경춘전은 소혜왕후(인수대왕대비)를 위해 건축되었고, 훗날 대비를 위한 거처로 사용되어 <a href="/wiki/%EC%9D%B8%ED%98%84%EC%99%95%ED%9B%84" title="인현왕후">인현왕후</a>,<a class="mw-redirect" href="/wiki/%ED%98%9C%EA%B2%BD%EA%B6%81_%ED%99%8D%EC%94%A8" title="혜경궁 홍씨">혜경궁 홍씨</a>등도 경춘전에서 사망하였다.</div>

<h3><span class="mw-headline" id=".EC.82.AC.ED.9B.84">사후</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%86%8C%ED%98%9C%EC%99%95%ED%9B%84&amp;action=edit&amp;section=12" title="부분 편집: 사후">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<h4><span class="mw-headline" id=".EC.9E.A5.EB.A1.80_.EB.AC.B8.EC.A0.9C">장례 문제</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%86%8C%ED%98%9C%EC%99%95%ED%9B%84&amp;action=edit&amp;section=13" title="부분 편집: 장례 문제">편집</a><span class="mw-editsection-bracket">]</span></span></h4>
<p>인수대왕대비가 죽어서도 할머니에 대한 연산군의 격하는 그치지 않았다. 화두에 오른 것이 장례 문제였는데, 연산군은 할머니의 제사를 왕이나 왕비의 격식이 아닌 그보다 한 단계 아래인 왕세자 왕세자빈의 격식에 합당하지 않느냐며 조정 대신들에게 하문하였는데, 연산군은 그 이유로 덕종(의경세자 : 인수대왕대비의 남편이자 연산군의 할아버지)의 장례가 왕세자의 장례로 치루어 졌기 때문이라고 하였으나 영의정 <a class="mw-disambig" href="/wiki/%EC%9C%A0%EC%88%9C" title="유순">유순</a>이 아뢰길 '덕종대왕의 상례로 치루면 주상 전하(연산군)의 본명(本命)을 범하게 됩니다.'를 이유로 들어 반대하였고 <a href="/wiki/%EB%8C%80%EC%A0%9C%ED%95%99" title="대제학">대제학</a> 김감(金勘)도 이에 반대하여 무산되었다.<sup class="reference" id="cite_ref-insu101_22-1"><a href="#cite_note-insu101-22">[22]</a></sup></p>
<p>그러자 연산군은 타협안으로 덕종(의경세자)보다는 조금 높게 하고, 예종비 안순왕후보다는 조금 낮게 하라고 명하였으나 이것 또한 영의정 유순, 대제학 김감등의 대신들이 예문에 근거가 없기에 의논하기에 어렵다고 주장하여 마찬가지로 무산되었다.<sup class="reference" id="cite_ref-insu101_22-2"><a href="#cite_note-insu101-22">[22]</a></sup> 결국 인수대왕대비의 장례는 안순왕후와 같은 상제에 의한 왕후(王后)의 격식으로 치러졌고 연산군의 계획은 이루어지지 않았으나 상제를 단축하여 시행하라는 어명을 내렸다.<sup class="reference" id="cite_ref-insu101_22-3"><a href="#cite_note-insu101-22">[22]</a></sup></p>
<p>이에 자순대비 윤씨(정현왕후)가 연산군에게 "예문에 근거가 없으니 나는 감히 따르지 못하겠다.' 하였으나 연산군이 화를 몹시 내며 "여자는 남편을 잃으면 아들의 뜻에 따라야 한다(삼종의 예)."라고 하니 자순대비가 "내가 인수대왕대비에게 죄를 얻게 될 것이 분명하다." 라고 한탄하였다.<sup class="reference" id="cite_ref-25"><a href="#cite_note-25">[25]</a></sup> 그러나 중종실록에 기록된 이 내용은 연산군 폐위 이후에 작성된 것이라 거짓일 가능성이 높고, 실제 성종때 승하한 세조의 비 정희왕후의 상제역시 한달로 단축시킨 예가 있는 만큼, 조선시대 왕실의 상제는 연산군 시대가 아니더라도 수없이 단축해왔다. 이것은 상제기간 동안 왕이 국사를 돌볼 수가 없으므로 조정의 국사를 위해서 단축시켜 온 것이다. 그러므로 승하한 왕이 친아버지인 경우가 아니라면 대부분의 왕실의 장례는 단축해왔던 예가 수도 없다.</p>
<p>꼭 연산군이 인수대비를 증오하여 상제를 단축했다는 것은 모함일 가능성이 높다. 인수대비의 상례는 앞서 승하한 안순왕후의 상례에 따른 것이었다. 안순왕후는 13일만에 상복을 벗었으나 당시 그 누구도 의의를 제기하지 않았다. 또한 연산군은 안순왕후보다 서열이 낮은 인수대비의 상례에서 13일만에 상복을 벗어야 했음에도 27일까지 상복을 입었으니 일반적인 상례보다 더 성의를 보인 것이다.</p>
<p>인수대비는 죽기전 남긴 유언으로 "자신이 죽은 후 3일 이내에 왕이 고기를 들도록 하라"고 했다. 죽어가면서까지 인수대비는 연산군을 걱정했는데 연산군이 이런 인수대비를 머리를 받쳐 죽게 만들었다는 것은 낭설에 가깝다.</p>
<h2><span class="mw-headline" id=".EC.82.AC.EC.83.81.EA.B3.BC_.EC.8B.A0.EB.85.90">사상과 신념</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%86%8C%ED%98%9C%EC%99%95%ED%9B%84&amp;action=edit&amp;section=14" title="부분 편집: 사상과 신념">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<h3><span class="mw-headline" id=".EB.B6.80.EB.8D.95.EB.A1.A0">부덕론</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%86%8C%ED%98%9C%EC%99%95%ED%9B%84&amp;action=edit&amp;section=15" title="부분 편집: 부덕론">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>소혜왕후는 생전에 독실한 <a href="/wiki/%EC%9C%A0%EA%B5%90" title="유교">유교</a> 인텔리임과 동시에 <a href="/wiki/%EB%B6%88%EA%B5%90" title="불교">불교</a>신자였으며 불경에 조예가 깊어 범어(梵語,<a href="/wiki/%EC%82%B0%EC%8A%A4%ED%81%AC%EB%A6%AC%ED%8A%B8%EC%96%B4" title="산스크리트어">산스크리트어</a>), 한어(漢語), 국어(國語) 3자체(三字體)로 서술한 불경과 부녀자의 예의범절을 가르치기 위하여 편찬한 《<a href="/wiki/%EB%82%B4%ED%9B%88" title="내훈">내훈</a>》(內訓)을 남겨 후세에 귀중한 연구자료가 되었다.</p>
<h3><span class="mw-headline" id=".EB.B6.88.EA.B5.90_.EC.98.B9.ED.98.B8">불교 옹호</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%86%8C%ED%98%9C%EC%99%95%ED%9B%84&amp;action=edit&amp;section=16" title="부분 편집: 불교 옹호">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>성종은 즉위 직후 <a href="/wiki/%EC%84%B1%EB%A6%AC%ED%95%99" title="성리학">성리학</a>적 왕도정치에 입각하여 <a href="/wiki/%EA%B8%B8%EC%9E%AC" title="길재">길재</a>, <a href="/wiki/%EA%B9%80%EC%88%99%EC%9E%90_(1389%EB%85%84)" title="김숙자 (1389년)">김숙자</a>, <a href="/wiki/%EA%B9%80%EC%A2%85%EC%A7%81" title="김종직">김종직</a>의 제자와 길재학파의 사류를 적극 등용했다. 이들은 왕도정치의 한 방법으로 <a href="/wiki/%EB%B6%88%EA%B5%90" title="불교">불교</a>, <a href="/wiki/%EB%8F%84%EA%B5%90" title="도교">도교</a>, 무속 등의 종교행사를 미신으로 간주, 미신타파를 부르짖었다. 한편 소혜왕후는 불교신자이자 불교 옹호론자로 사림의 미신타파 주장과 <a class="mw-redirect" href="/wiki/%EC%88%AD%EC%9C%A0%EC%96%B5%EB%B6%88" title="숭유억불">불교 억압 정책</a>에 강력 반발하였다. 또한 <a href="/wiki/%EC%A1%B0%EC%84%A0_%EC%84%B8%EC%A2%85" title="조선 세종">세종</a>과 세조가 불당에 출입한 고사를 들어 궐내 법당을 철폐하려는 사림의 주장에 맞서곤 했다. 궁궐에 설치한 불당은 <a href="/wiki/%EC%A1%B0%EC%84%A0_%EC%84%A0%EC%A1%B0" title="조선 선조">선조</a>대에 가서야 철폐되었다.</p>

<h2><span class="mw-headline" id=".EA.B8.B0.ED.83.80">기타</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%86%8C%ED%98%9C%EC%99%95%ED%9B%84&amp;action=edit&amp;section=22" title="부분 편집: 기타">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>인수대비의 성격은 강직했던 것으로 유명하였으며 성종의 정비(正妃)인 <a href="/wiki/%EA%B3%B5%ED%98%9C%EC%99%95%ED%9B%84" title="공혜왕후">공혜왕후</a>의 승하 이후 계비로 책봉된 <a href="/wiki/%ED%8F%90%EB%B9%84_%EC%9C%A4%EC%94%A8" title="폐비 윤씨">중전 윤씨</a>가 자신의 기대에 미치지 못하고, 급기야는 성종의 용안에 손톱 자국을 내는 지경에 이르자 진노한 인수대비는 중전 윤씨를 폐서인하여 사가로 내쫓았으며 폐비 윤씨를 투기하였던 성종의 다른 후궁들과 함께 성종으로 하여금 폐비 윤씨에게 사약을 내리게 하였다. 그러한 까닭으로 인수대비는 당시 원자이자 폐비 윤씨의 아들인 <a href="/wiki/%EC%A1%B0%EC%84%A0_%EC%97%B0%EC%82%B0%EA%B5%B0" title="조선 연산군">연산군</a>의 왕위 계승에도 부정적이었으며 성종의 계비인 <a href="/wiki/%EC%A0%95%ED%98%84%EC%99%95%ED%9B%84" title="정현왕후">정현왕후</a>의 소생 <a href="/wiki/%EC%A1%B0%EC%84%A0_%EC%A4%91%EC%A2%85" title="조선 중종">진성대군</a>을 편애하였다.</p>
<h3><span class="mw-headline" id=".EC.8B.A0.EB.B6.84.EC.83.81.EC.9C.BC.EB.A1.9C_.EC.A1.B0.EC.84.B1.EB.90.9C_.EA.B2.BD.EB.A6.89.28.EA.B2.BD.EB.A6.89.29">신분상으로 조성된 경릉(경릉)</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%86%8C%ED%98%9C%EC%99%95%ED%9B%84&amp;action=edit&amp;section=23" title="부분 편집: 신분상으로 조성된 경릉(경릉)">편집</a><span class="mw-editsection-bracket">]</span></span></h3>

남편 <a href="/wiki/%EC%A1%B0%EC%84%A0_%EB%8D%95%EC%A2%85" title="조선 덕종">의경세자(덕종)</a>의 봉분</div>

<p>소혜왕후의 능은 <a href="/wiki/%EA%B2%BD%EA%B8%B0%EB%8F%84" title="경기도">경기도</a> <a href="/wiki/%EA%B3%A0%EC%96%91%EC%8B%9C" title="고양시">고양시</a> <a href="/wiki/%EC%84%9C%EC%98%A4%EB%A6%89" title="서오릉">서오릉</a> 내에 위치한 <a href="/wiki/%EA%B2%BD%EB%A6%89_(%EC%A1%B0%EC%84%A0_%EB%8D%95%EC%A2%85)" title="경릉 (조선 덕종)">경릉</a>(敬陵)으로 남편인 덕종과 같은 묘역에 묻혀있다. 본디 왕릉을 쓸 때는 우측에 왕을, 좌측에 왕비를 안장한다. 살아있을 때는 좌측(앉아있는 사람 기준)이 상석이니 죽으면 그 반대라고 여겼던 것인데<sup class="정리_필요 noprint">[<a href="/wiki/%EC%9C%84%ED%82%A4%EB%B0%B1%EA%B3%BC:%EC%B6%9C%EC%B2%98_%EB%B0%9D%ED%9E%88%EA%B8%B0" title="위키백과:출처 밝히기">출처 필요</a>]</sup>, 경릉은 부인인 소혜왕후가 상좌인 우측에 안장되어 있는 것이다. 이것은 승하할 당시의 신분 차이 때문인데, 덕종의 경우는 승하할 때 세자의 신분이었지만, 소혜왕후의 경우는 왕실의 최고 어른인 대왕대비로서 승하했기 때문에 군신 관계에 따라 위가 되었던 것이다. 이에 따라 남편 덕종의 무덤에는 석물도 문인석만 서 있다. 무인석은 왕권의 상징이라 왕이 되지 못한 세자, 세자빈 이하의 무덤에는 세울 수 없다. 소혜왕후 능상의 석물은 무인석까지 갖추고 있다. 신분상으로 조성된 능으로는 서오릉의 경릉과 동구릉의 문조와 신정왕후의 수릉이 있다.</p>
