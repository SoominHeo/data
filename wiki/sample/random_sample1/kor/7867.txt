<p><b>손더스 매클레인</b>(<span style="font-size: smaller;"><a href="/wiki/%EC%98%81%EC%96%B4" title="영어">영어</a>:</span> <span lang="en" xml:lang="en">Saunders Mac Lane</span> <small><a href="/wiki/%EC%9C%84%ED%82%A4%EB%B0%B1%EA%B3%BC:IPA" title="위키백과:IPA">IPA</a>: </small><span class="IPA" title="IPA 발음">[ˈsɔːndə(ɹ)s məˈkleɪn]</span>, 1909–2005)은 <a href="/wiki/%EB%AF%B8%EA%B5%AD" title="미국">미국</a>의 <a href="/wiki/%EC%88%98%ED%95%99%EC%9E%90" title="수학자">수학자</a>다. <a href="/wiki/%EC%82%AC%EB%AC%B4%EC%97%98_%EC%97%90%EC%9D%BC%EB%A0%8C%EB%B2%A0%EB%A5%B4%ED%81%AC" title="사무엘 에일렌베르크">사무엘 에일렌베르크</a>와 함께 <a href="/wiki/%EB%B2%94%EC%A3%BC%EB%A1%A0" title="범주론">범주론</a>을 창안하였다.</p>

<h2><span class="mw-headline" id=".EC.83.9D.EC.95.A0">생애</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%86%90%EB%8D%94%EC%8A%A4_%EB%A7%A4%ED%81%B4%EB%A0%88%EC%9D%B8&amp;action=edit&amp;section=1" title="부분 편집: 생애">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>1909년 8월 4일 미국 <a href="/wiki/%EC%BD%94%EB%84%A4%ED%8B%B0%EC%BB%B7_%EC%A3%BC" title="코네티컷 주">코네티컷 주</a> 태프트빌(Taftville)에서 아버지 도널드 매클레인(<span style="font-size: smaller;"><a href="/wiki/%EC%98%81%EC%96%B4" title="영어">영어</a>:</span> <span lang="en" xml:lang="en">Donald MacLane</span>)과 어머니 위니프레드 손더스(<span style="font-size: smaller;"><a href="/wiki/%EC%98%81%EC%96%B4" title="영어">영어</a>:</span> <span lang="en" xml:lang="en">Winifred Saunders</span>)에게 태어났다. 매클레인 가문은 <a href="/wiki/%EC%8A%A4%EC%BD%94%ED%8B%80%EB%9E%9C%EB%93%9C" title="스코틀랜드">스코틀랜드</a> 출신이었으며, 성의 철자는 원래 <span style="font-size: smaller;"><a href="/wiki/%EC%98%81%EC%96%B4" title="영어">영어</a>:</span> <span lang="en" xml:lang="en">McLean</span>이었지만, 미국에 이민하면서 더 발음하기 쉽게 <span style="font-size: smaller;"><a href="/wiki/%EC%98%81%EC%96%B4" title="영어">영어</a>:</span> <span lang="en" xml:lang="en">MacLane</span>으로 바뀌었다. 매클레인의 본명은 레슬리 손더스 매클레인(<span lang="en" xml:lang="en">Leslie Saunders MacLane</span>)이었으나, 어릴 때부터 "레슬리"는 생략하게 되었다.<sup class="reference" id="cite_ref-1"><a href="#cite_note-1">[1]</a></sup></p>
<p>1926년에 <a href="/wiki/%EC%98%88%EC%9D%BC_%EB%8C%80%ED%95%99%EA%B5%90" title="예일 대학교">예일 대학교</a>에 입학하여 원래는 <a href="/wiki/%ED%99%94%ED%95%99" title="화학">화학</a>을 전공하였으나 곧 <a href="/wiki/%EC%88%98%ED%95%99" title="수학">수학</a>으로 전공을 바꾸었다. 1930년에 <a href="/wiki/%EC%98%88%EC%9D%BC_%EB%8C%80%ED%95%99%EA%B5%90" title="예일 대학교">예일 대학교</a>에서 학사(Bachelor of Arts) 학위를 받았고, 1931년에 <a href="/wiki/%EC%8B%9C%EC%B9%B4%EA%B3%A0_%EB%8C%80%ED%95%99%EA%B5%90" title="시카고 대학교">시카고 대학교</a>에서 석사(Master of Arts) 학위를 받았다. 석사 기간 동안에 첫 논문을 발표하였는데, <a href="/wiki/%EB%AC%BC%EB%A6%AC%ED%95%99" title="물리학">물리학</a>에 대한 것이었다. 그 뒤 <a href="/wiki/%EA%B4%B4%ED%8C%85%EA%B2%90_%EB%8C%80%ED%95%99%EA%B5%90" title="괴팅겐 대학교">괴팅겐 대학교</a>에서 <a href="/wiki/%EC%97%90%EB%AF%B8_%EB%87%8C%ED%84%B0" title="에미 뇌터">에미 뇌터</a>, <a href="/wiki/%ED%97%A4%EB%A5%B4%EB%A7%8C_%EB%B0%94%EC%9D%BC" title="헤르만 바일">헤르만 바일</a>, <a href="/wiki/%ED%8C%8C%EC%9A%B8_%EB%B2%A0%EB%A5%B4%EB%82%98%EC%9D%B4%EC%8A%A4" title="파울 베르나이스">파울 베르나이스</a> 등 아래서 박사 학위를 밟았고, 1934년에 졸업하였다. 1933년에 도로시 존스(<span style="font-size: smaller;"><a href="/wiki/%EC%98%81%EC%96%B4" title="영어">영어</a>:</span> <span lang="en" xml:lang="en">Dorothy Jones</span>)와 결혼하였다. 아내 도로시의 요구에 의하여 성 "매클레인"을 띄어쓰기하게 되었다 (<span lang="en" xml:lang="en">MacLane</span> → <span lang="en" xml:lang="en">Mac Lane</span>).</p>
<p>1934년~1938년에 <a href="/wiki/%ED%95%98%EB%B2%84%EB%93%9C_%EB%8C%80%ED%95%99%EA%B5%90" title="하버드 대학교">하버드 대학교</a>, <a href="/wiki/%EC%BD%94%EB%84%AC_%EB%8C%80%ED%95%99%EA%B5%90" title="코넬 대학교">코넬 대학교</a>, <a href="/wiki/%EC%8B%9C%EC%B9%B4%EA%B3%A0_%EB%8C%80%ED%95%99%EA%B5%90" title="시카고 대학교">시카고 대학교</a>에 강사로 있었다. 1938년~1947년에는 <a href="/wiki/%ED%95%98%EB%B2%84%EB%93%9C_%EB%8C%80%ED%95%99%EA%B5%90" title="하버드 대학교">하버드 대학교</a>에 교수로 있었고, 1947년부터 <a href="/wiki/%EC%8B%9C%EC%B9%B4%EA%B3%A0_%EB%8C%80%ED%95%99%EA%B5%90" title="시카고 대학교">시카고 대학교</a>에서 교수로 있었다.</p>
<p>1985년 2월 3일에 부인 도로시가 <a href="/wiki/%EB%87%8C%EC%97%BC" title="뇌염">뇌염</a>으로 인해 사망하였다. 1986년에 예술가 오사 스코팅(<span style="font-size: smaller;"><a href="/wiki/%EC%98%81%EC%96%B4" title="영어">영어</a>:</span> <span lang="en" xml:lang="en">Osa Skotting</span>)과 재혼하였다. (스코팅은 수학자 <a class="new" href="/w/index.php?title=%EC%96%B4%EB%B9%99_%EC%8B%9C%EA%B1%B8&amp;action=edit&amp;redlink=1" title="어빙 시걸 (없는 문서)">어빙 시걸</a>(<span style="font-size: smaller;"><a href="/wiki/%EC%98%81%EC%96%B4" title="영어">영어</a>:</span> <span lang="en" xml:lang="en">Irving Segal</span>)의 아내였으나, 1977년에 이혼하였다.)</p>
<p>2005년 4월 14일에 사망하였다.</p>
