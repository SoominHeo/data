<p><b>손석희</b>(孫石熙, <a href="/wiki/1956%EB%85%84" title="1956년">1956년</a> <a href="/wiki/7%EC%9B%94_27%EC%9D%BC" title="7월 27일">7월 27일</a>(<a href="/wiki/%EC%9D%8C%EB%A0%A5_6%EC%9B%94_20%EC%9D%BC" title="음력 6월 20일">음력 6월 20일</a>) ~ )는 <a href="/wiki/%EB%8C%80%ED%95%9C%EB%AF%BC%EA%B5%AD" title="대한민국">대한민국</a>의 <a class="mw-redirect" href="/wiki/%EC%96%B8%EB%A1%A0%EC%9D%B8" title="언론인">언론인</a>이다. 1984년부터 2013년까지 <a href="/wiki/%EB%AC%B8%ED%99%94%EB%B0%A9%EC%86%A1" title="문화방송">MBC</a>에서 아나운서, 기자를 역임했고, <a href="/wiki/%EC%84%B1%EC%8B%A0%EC%97%AC%EC%9E%90%EB%8C%80%ED%95%99%EA%B5%90" title="성신여자대학교">성신여자대학교</a> 등에서 대학 교수직을 맡기도 했다. 2013년 5월 <a href="/wiki/JTBC" title="JTBC">JTBC</a> 보도 담당 사장으로 채용되어 일하는 한편 앵커로도 계속 활동을 했다. <a href="/wiki/%EC%8B%9C%EC%82%AC%EC%A0%80%EB%84%90" title="시사저널">시사저널</a> '가장 영향력 있는 언론인' 조사에서 <a href="/wiki/2005%EB%85%84" title="2005년">2005년</a> 이후 12년 연속 1위를 차지하였다.<sup class="reference" id="cite_ref-1"><a href="#cite_note-1">[1]</a></sup> 2016년 12월 <a href="/wiki/%EB%AF%BC%EC%A3%BC%EC%96%B8%EB%A1%A0%EC%8B%9C%EB%AF%BC%EC%97%B0%ED%95%A9" title="민주언론시민연합">민주언론시민연합</a>에서 시상하는 민주시민언론상 '본상'을 수상했다.<sup class="reference" id="cite_ref-2"><a href="#cite_note-2">[2]</a></sup> <sup class="reference" id="cite_ref-3"><a href="#cite_note-3">[3]</a></sup></p>

<p><a href="/wiki/%ED%9C%98%EB%AC%B8%EA%B3%A0%EB%93%B1%ED%95%99%EA%B5%90" title="휘문고등학교">휘문고등학교</a> 재학 시절 <a href="/wiki/%EB%B0%A9%EC%86%A1" title="방송">방송</a>반에서 활동하였다. <a href="/wiki/1984%EB%85%84" title="1984년">1984년</a> <a class="mw-redirect" href="/wiki/MBC" title="MBC">MBC</a>에 입사했다.<sup class="reference" id="cite_ref-4"><a href="#cite_note-4">[4]</a></sup> 한 때 한국 최초의 아나운서와 기자를 겸한 방송인으로 알려졌으나 이는 사실이 아니며 <a href="/wiki/1987%EB%85%84" title="1987년">1987년</a>부터 <a href="/wiki/1989%EB%85%84" title="1989년">1989년</a>까지 보도국 사회부 기자로 재직하다가 <a href="/wiki/1989%EB%85%84" title="1989년">1989년</a>부터 다시 아나운서국에서 활동하게 되었다.<sup class="reference" id="cite_ref-5"><a href="#cite_note-5">[5]</a></sup> 뉴스 프로그램의 앵커로 얼굴과 이름이 알려진 상태에서 <a href="/wiki/1992%EB%85%84" title="1992년">1992년</a> MBC 노조 파업 때 주동자로 몰려 구속되는 과정을 거치면서 바른 말을 잘 하는 곧은 이미지를 쌓았다.<sup class="reference" id="cite_ref-6"><a href="#cite_note-6">[6]</a></sup> <sup class="reference" id="cite_ref-7"><a href="#cite_note-7">[7]</a></sup> <sup class="reference" id="cite_ref-8"><a href="#cite_note-8">[8]</a></sup> <a href="/wiki/1997%EB%85%84" title="1997년">1997년</a>부터 <a href="/wiki/1999%EB%85%84" title="1999년">1999년</a>까지 <a href="/wiki/%EB%AF%B8%EA%B5%AD" title="미국">미국</a> <a href="/wiki/%EB%AF%B8%EB%84%A4%EC%86%8C%ED%83%80_%EB%8C%80%ED%95%99%EA%B5%90" title="미네소타 대학교">미네소타 대학교</a>에서 유학한 뒤, MBC 아침뉴스 2000으로 앵커에 복귀하였다. 시사 프로그램의 진행을 많이 맡아 대중적인 영향력을 갖춘 방송인으로 꼽혀 왔다.</p>
<p><a href="/wiki/%EC%84%B1%EA%B7%A0%EA%B4%80%EB%8C%80%ED%95%99%EA%B5%90" title="성균관대학교">성균관대학교</a> 신문방송학과 겸임교수, <a href="/wiki/%EC%97%B0%EC%84%B8%EB%8C%80%ED%95%99%EA%B5%90" title="연세대학교">연세대학교</a> 신문방송학과 겸임 교수를 거쳐서 <a href="/wiki/2006%EB%85%84" title="2006년">2006년</a> <a href="/wiki/2%EC%9B%94" title="2월">2월</a> <a class="mw-redirect" href="/wiki/MBC" title="MBC">MBC</a>에서 물러난 후 <a href="/wiki/%EC%84%B1%EC%8B%A0%EC%97%AC%EC%9E%90%EB%8C%80%ED%95%99%EA%B5%90" title="성신여자대학교">성신여자대학교</a> 미디어커뮤니케이션학과 교수를 맡았다. <a href="/wiki/2013%EB%85%84" title="2013년">2013년</a> <a href="/wiki/5%EC%9B%94" title="5월">5월</a> MBC <a href="/wiki/%EB%9D%BC%EB%94%94%EC%98%A4" title="라디오">라디오</a> 《<a class="mw-redirect" href="/wiki/%EC%86%90%EC%84%9D%ED%9D%AC%EC%9D%98_%EC%8B%9C%EC%84%A0%EC%A7%91%EC%A4%91" title="손석희의 시선집중">손석희의 시선집중</a>》 진행자와 성신여자대학교 교수직에서 사임하고 JTBC 보도부문 총괄 사장을 맡게 되었다.<sup class="reference" id="cite_ref-9"><a href="#cite_note-9">[9]</a></sup> 이후, <a href="/wiki/2013%EB%85%84" title="2013년">2013년</a> <a href="/wiki/9%EC%9B%94_16%EC%9D%BC" title="9월 16일">9월 16일</a>부터 <a href="/wiki/2014%EB%85%84" title="2014년">2014년</a> <a href="/wiki/9%EC%9B%94" title="9월">9월</a>까지 <a class="mw-redirect" href="/wiki/JTBC_%EB%89%B4%EC%8A%A4_9" title="JTBC 뉴스 9">JTBC 뉴스 9</a> 주중 진행을 담당하였다.<sup class="reference" id="cite_ref-10"><a href="#cite_note-10">[10]</a></sup> <a href="/wiki/%EC%8B%9C%EC%82%AC%EC%A0%80%EB%84%90" title="시사저널">시사저널</a> '가장 영향력 있는 언론인' 조사에서 <a href="/wiki/2005%EB%85%84" title="2005년">2005년</a> 이후 12년 연속 1위를 차지하였다.<sup class="reference" id="cite_ref-11"><a href="#cite_note-11">[11]</a></sup> <sup class="reference" id="cite_ref-12"><a href="#cite_note-12">[12]</a></sup> <a href="/wiki/2014%EB%85%84" title="2014년">2014년</a> <a href="/wiki/9%EC%9B%94_22%EC%9D%BC" title="9월 22일">9월 22일</a>부터는 1시간 40분 동안 진행되는 JTBC의 메인뉴스 <a
