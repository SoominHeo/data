<h1 class="firstHeading" id="firstHeading" lang="ko">손화</h1>
<div class="mw-body-content" id="bodyContent">
<div id="siteSub">위키백과, 우리 모두의 백과사전.</div>

</div>
<div class="mw-content-ltr" dir="ltr" id="mw-content-text" lang="ko"><script>function mfTempOpenSection(id){var block=document.getElementById("mf-section-"+id);block.className+=" open-block";block.previousSibling.className+=" open-block";}</script><p><b>오 문황제 손화</b>(吳 文皇帝 孫和, <a href="/wiki/224%EB%85%84" title="224년">224년</a> ~ <a href="/wiki/253%EB%85%84" title="253년">253년</a>)는 <a class="mw-redirect" href="/wiki/%EB%8F%99%EC%98%A4" title="동오">오</a>(吳)나라의 <a class="mw-redirect" href="/wiki/%EC%B6%94%EC%A1%B4" title="추존">추존</a><a href="/wiki/%ED%99%A9%EC%A0%9C" title="황제">황제</a>로, 자는 <b>자효</b>(子孝)이다.</p>
<h2><span class="mw-headline" id=".ED.96.89.EC.A0.81">행적</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%86%90%ED%99%94&amp;action=edit&amp;section=1" title="부분 편집: 행적">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p><a href="/wiki/%EC%86%90%EA%B6%8C" title="손권">대제</a>(大帝)의 삼남으로, 어머니는 낭야 사람 대의황후(大懿皇后) 왕씨(王氏)이다.</p>
<p>그의 형인 태자 <a href="/wiki/%EC%86%90%EB%93%B1" title="손등">손등</a>(孫登)이 <a href="/wiki/241%EB%85%84" title="241년">241년</a>에 요절하자, 작은 형 <a href="/wiki/%EC%86%90%EB%A0%A4" title="손려">손려</a>(孫慮)는 이미 죽었으므로, <a href="/wiki/%ED%99%A9%ED%83%9C%EC%9E%90" title="황태자">황태자</a>에 책봉되었다. 그러나 <a href="/wiki/%EC%86%90%EA%B6%8C" title="손권">손권</a>이 손화와 <a href="/wiki/%EC%86%90%ED%8C%A8" title="손패">손패</a>의 황태자와 <a class="mw-redirect" href="/wiki/%EB%B2%88%EC%99%95" title="번왕">번왕</a>의 구별을 없앴으므로, 오나라의 신료들이 손화파와 손패파로 갈라서게 되었다. 많은 이들이 <a href="/wiki/%EC%86%90%EA%B6%8C" title="손권">손권</a>에게 손화의 황태자 지위를 공고히 할 것을 간언하였으나, <a href="/wiki/%EC%86%90%EA%B6%8C" title="손권">손권</a>은 이를 모두 물리쳤다. 후에 <a href="/wiki/%EC%86%90%EA%B6%8C" title="손권">손권</a>은 <a href="/wiki/%EC%86%90%ED%8C%A8" title="손패">손패</a>를 죽이고 손화를 황태자에서 쫓아내고, 남양왕(南陽王)으로 임명하여, <a class="mw-redirect" href="/wiki/%EC%B0%BD%EC%82%AC" title="창사">장사</a>로 귀양보냈다. 이후 <a href="/wiki/%EC%86%90%EB%9F%89" title="손량">손량</a>(孫亮) 시대에, 정권을 잡은 <a href="/wiki/%EC%A0%9C%EA%B0%88%EA%B0%81" title="제갈각">제갈각</a>(諸葛恪)이 손화의 정비 장씨의 외숙이었고, 민간에는 <a href="/wiki/%EC%A0%9C%EA%B0%88%EA%B0%81" title="제갈각">제갈각</a>이 손화를 맞아들이려 한다는 말이 있었다. <a href="/wiki/%EC%86%90%EC%A4%80" title="손준">손준</a>(孫峻)이 <a href="/wiki/%EC%A0%9C%EA%B0%88%EA%B0%81" title="제갈각">제갈각</a>을 암살한 후, 이 때문에 <a href="/wiki/%EC%86%90%EC%A4%80" title="손준">손준</a>은 손화의 인수를 빼앗고, 신도로 보내어, 사사했다. 향년 30세였다.</p>
<p>그의 아들 <a href="/wiki/%EC%86%90%ED%98%B8" title="손호">손호</a>(孫皓)가 <a href="/wiki/%EC%86%90%ED%9C%B4" title="손휴">손휴</a>(孫休) 대에 <a class="new" href="/w/index.php?title=%EC%98%A4%EC%A0%95%ED%9B%84&amp;action=edit&amp;redlink=1" title="오정후 (없는 문서)">오정후</a>에 봉해지고, 손휴 사후 황제에 오른다. 손호는 황제 즉위 후 손화를 소헌제(昭獻帝)로 추존하였고, 얼마 후 문황제(文皇帝)로 바꾸었다.</p>
<h2><span class="mw-headline" id=".EA.B0.80.EC.A1.B1_.EA.B4.80.EA.B3.84">가족 관계</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%86%90%ED%99%94&amp;action=edit&amp;section=2" title="부분 편집: 가족 관계">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<ul>
<li>장씨(張氏) (적비): <a href="/wiki/%EC%9E%A5%EC%8A%B9_(%EB%8F%99%EC%98%A4)" title="장승 (동오)">장승</a>(張承)의 딸이며 <a class="mw-redirect" href="/wiki/%EC%9E%A5%EC%86%8C_(%EB%8F%99%EC%98%A4)" title="장소 (동오)">장소</a>(張昭)의 손녀이자 <a href="/wiki/%EC%A0%9C%EA%B0%88%EA%B7%BC" title="제갈근">제갈근</a>(諸葛瑾)의 외손이다. 손화가 사사하자, 자살했다.
<ul>
<li>손준(孫俊) (아들): 《오력》에 <a href="/wiki/%EC%9E%A5%EC%8A%B9_(%EB%8F%99%EC%98%A4)" title="장승 (동오)">장승</a>의 외손이라 하므로 장씨의 아들이다. 총명하여 널리 칭찬을 받았으며, <a href="/wiki/%EC%86%90%ED%98%B8" title="손호">손호</a>에게 죽임을 당했다.</li>
</ul>
</li>
<li>소헌황후(昭獻皇后) 하씨(何氏) 혹 하희(何姬) (첩): 기병 하수(何遂)의 딸로, <a href="/wiki/%EC%86%90%ED%98%B8" title="손호">손호</a>의 생모며, 적비 장씨가 죽은 후 <a href="/wiki/%EC%86%90%ED%98%B8" title="손호">손호</a>와 그의 아우 셋을 길렀다. <a href="/wiki/%EC%86%90%ED%98%B8" title="손호">손호</a>가 즉위한 후 손화를 소헌황제로 추존하면서 소헌황후가 되었으며, 황태후로 올랐다.
<ul>
