<p>《오력》에 따르면, <a class="mw-disambig" href="/wiki/%EC%86%90%EB%A6%B0" title="손린">손린</a>에게는 작위를 이은 손묘 외에도 아들이 더 있어, 손술은 무창독 · 평형주사가 되었고, 손진은 무난독이 되었고, 손해는 성문교위가 되었고, 손흠은 악향독이 되었다.</p>
<p><a class="mw-disambig" href="/wiki/%EC%B2%9C%EA%B8%B0" title="천기">천기</a> 4년(<a href="/wiki/280%EB%85%84" title="280년">280년</a>), <a href="/wiki/%EC%84%9C%EC%A7%84" title="서진">서진</a>에서 대대적으로 군사를 출동시켜 오나라를 정벌하였다. <a href="/wiki/%EB%91%90%EC%98%88" title="두예">두예</a>(杜預)가 아문장 관정(管定), <a href="/wiki/%EC%A3%BC%EC%A7%80" title="주지">주지</a>(周旨), 오소(伍巢) 등에게 8백 명을 주어 악향을 습격하고, 기치를 많이 벌이고 파산에서 불을 일으키자, 손흠은 두려워하고 놀라 강릉독 <a href="/wiki/%EC%98%A4%EC%97%B0" title="오연">오연</a>(伍延)에게 <b>북쪽에서 내려온 여러 군사들이 마치 장강을 날아서 넘어온 것 같습니다.</b> 라는 내용의 편지를 보냈다. 이어 손흠은 <a href="/wiki/%EC%99%95%EC%A4%80_(%EC%84%9C%EC%A7%84)" title="왕준 (서진)">왕준</a>(王濬)의 군대를 요격했다가 패퇴하고 성내로 돌아왔는데, 퇴각 도중에 매복하고 있던 주지, 오소 등의 군대가 끼어들어갔으나 이를 몰라 진중에서 주지, 오소에게 결국 사로잡혔다.</p>
<p><a href="/wiki/%EC%99%95%EC%A4%80_(%EC%84%9C%EC%A7%84)" title="왕준 (서진)">왕준</a>이 미리 손흠의 머리를 서울로 올려보냈는데, <a href="/wiki/%EB%91%90%EC%98%88" title="두예">두예</a>가 나중에 손흠을 산 채로 보냈으므로 웃음거리가 되었다.</p>
<h2><span class="mw-headline" id=".E3.80.8A.EC.82.BC.EA.B5.AD.EC.A7.80.EC.97.B0.EC.9D.98.E3.80.8B.EC.9D.98_.EB.AC.98.EC.82.AC">《<a href="/wiki/%EC%82%BC%EA%B5%AD%EC%A7%80%EC%97%B0%EC%9D%98" title="삼국지연의">삼국지연의</a>》의 묘사</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%86%90%ED%9D%A0&amp;action=edit&amp;section=2" title="부분 편집: 《삼국지연의》의 묘사">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>천기 4년, 표기장군으로써 장제의 상주에 따라 <a href="/wiki/%EB%91%90%EC%98%88" title="두예">두예</a>의 군대를 거기장군 오연, <a href="/wiki/%EC%9C%A1%EA%B2%BD_(%EB%8F%99%EC%98%A4)" title="육경 (동오)">육경</a>(陸景)과 함께 요격했다. 그러나 <a href="/wiki/%EB%91%90%EC%98%88" title="두예">두예</a>와 싸워 패배하고 성으로 돌아가는데, <a href="/wiki/%EC%A3%BC%EC%A7%80" title="주지">주지</a>가 거느린 군사 8백 명이 성을 취하자 크게 놀라 “북쪽에서 온 군대는 강을 날아서 건넜는가!”라고 말하며 물러서려고 했으나 뒤에서 나타난 <a href="/wiki/%EC%A3%BC%EC%A7%80" title="주지">주지</a>에게 죽임을 당하였다.</p>
