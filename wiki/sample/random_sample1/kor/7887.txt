<p><b>솔붓꽃</b>(<i>Iris ruthenica</i>)은 <a href="/wiki/%EB%B0%B1%ED%95%A9%EB%AA%A9" title="백합목">백합목</a> 붓꽃과에 속하는 <a href="/wiki/%EC%99%B8%EB%96%A1%EC%9E%8E%EC%8B%9D%EB%AC%BC" title="외떡잎식물">외떡잎식물</a> 여러해살이풀로써 <a href="/wiki/%EB%8C%80%ED%95%9C%EB%AF%BC%EA%B5%AD" title="대한민국">대한민국</a> 멸종위기 식물이다.<sup class="reference" id="cite_ref-_1-0"><a href="#cite_note--1">[1]</a></sup></p>

<h2><span class="mw-headline" id=".ED.8A.B9.EC.A7.95">특징</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%86%94%EB%B6%93%EA%BD%83&amp;action=edit&amp;section=1" title="부분 편집: 특징">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>솔붓꽃의 <a href="/wiki/%EA%BD%83" title="꽃">꽃</a>은 4~5월에 피며 색상은 보라색이며 꽃자루 끝에 두개씩 달린다. 높이는 8~13cm로 자라며, <a href="/wiki/%EC%9E%8E" title="잎">잎</a>은 비함몰상 타원형으로 길이는 15cm 정도 나비 4mm정도 나며, <a href="/wiki/%EC%97%B4%EB%A7%A4" title="열매">열매</a>는 둥근 모양이다. 솔붓꽃은 환경부 멸종위기 야생생물 II급로 지정 되어있다.<sup class="reference" id="cite_ref-doopedia_2-0"><a href="#cite_note-doopedia-2">[2]</a></sup><sup class="reference" id="cite_ref-academy_3-0"><a href="#cite_note-academy-3">[3]</a></sup><sup class="reference" id="cite_ref-forest_4-0"><a href="#cite_note-forest-4">[4]</a></sup></p>
<h2><span class="mw-headline" id=".EB.B6.84.ED.8F.AC_.EC.A7.80.EC.97.AD">분포 지역</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%86%94%EB%B6%93%EA%BD%83&amp;action=edit&amp;section=2" title="부분 편집: 분포 지역">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>솔붓꽃은 대한민국 지역인 <a class="mw-redirect" href="/wiki/%EB%8C%80%EA%B5%AC" title="대구">대구</a>, <a class="mw-redirect" href="/wiki/%EC%B6%A9%EB%82%A8" title="충남">충남</a>, <a class="mw-disambig" href="/wiki/%EA%B2%BD%EA%B8%B0" title="경기">경기</a>의 일부 지역과 <a href="/wiki/%EC%A0%84%EB%9D%BC%EB%82%A8%EB%8F%84" title="전라남도">전라남도</a> 해안과 <a href="/wiki/%EC%A4%91%EA%B5%AD" title="중국">중국</a>에서 자란다.<sup class="reference" id="cite_ref-doopedia_2-1"><a href="#cite_note-doopedia-2">[2]</a></sup><sup class="reference" id="cite_ref-forest_4-1"><a href="#cite_note-forest-4">[4]</a></sup><sup class="reference" id="cite_ref-academy_3-1"><a href="#cite_note-academy-3">[3]</a></sup><sup class="reference" id="cite_ref-yeonhap_5-0"><a href="#cite_note-yeonhap-5">[5]</a></sup></p>
<h2><span class="mw-headline" id=".EC.93.B0.EC.9E.84.EC.83.88">쓰임새</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%86%94%EB%B6%93%EA%BD%83&amp;action=edit&amp;section=3" title="부분 편집: 쓰임새">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
