<p><b>A리그 2015-16</b>은 <a href="/wiki/A%EB%A6%AC%EA%B7%B8" title="A리그">A리그</a>의 열한 번째 시즌이다.<br/>
<br/>
정규 시즌은 <a href="/wiki/2015%EB%85%84" title="2015년">2015년</a> <a href="/wiki/10%EC%9B%94_8%EC%9D%BC" title="10월 8일">10월 8일</a> 개막하여 <a href="/wiki/2016%EB%85%84" title="2016년">2016년</a> <a href="/wiki/4%EC%9B%94_10%EC%9D%BC" title="4월 10일">4월 10일</a> 막을 내렸다. <a href="/wiki/%EC%95%A0%EB%93%A4%EB%A0%88%EC%9D%B4%EB%93%9C_%EC%9C%A0%EB%82%98%EC%9D%B4%ED%8B%B0%EB%93%9C_FC" title="애들레이드 유나이티드 FC">애들레이드 유나이티드</a>가 우승을 차지하였고, 준우승을 차지한 <a href="/wiki/%EC%9B%A8%EC%8A%A4%ED%84%B4_%EC%8B%9C%EB%93%9C%EB%8B%88_%EC%9B%90%EB%8D%94%EB%9F%AC%EC%8A%A4_FC" title="웨스턴 시드니 원더러스 FC">웨스턴 시드니 원더러스</a>가 <a href="/wiki/AFC_%EC%B1%94%ED%94%BC%EC%96%B8%EC%8A%A4%EB%A6%AC%EA%B7%B8_2017" title="AFC 챔피언스리그 2017">AFC 챔피언스리그 2017</a> 출전권을 획득하였다. 3위를 차지한<a href="/wiki/%EB%B8%8C%EB%A6%AC%EC%A6%88%EB%B2%88_%EB%A1%9C%EC%96%B4_FC" title="브리즈번 로어 FC">브리즈번 로어</a>는 <a href="/wiki/AFC_%EC%B1%94%ED%94%BC%EC%96%B8%EC%8A%A4%EB%A6%AC%EA%B7%B8_2017" title="AFC 챔피언스리그 2017">AFC 챔피언스리그 2017</a> 2차 예선부터 참가하게 되었다.<br/>
<br/>
<a class="new" href="/w/index.php?title=A%EB%A6%AC%EA%B7%B8_%EA%B7%B8%EB%9E%9C%EB%93%9C_%ED%8C%8C%EC%9D%B4%EB%84%90_2016&amp;action=edit&amp;redlink=1" title="A리그 그랜드 파이널 2016 (없는 문서)">A리그 그랜드 파이널 2016</a>은 <a href="/wiki/2016%EB%85%84" title="2016년">2016년</a> <a href="/wiki/5%EC%9B%94_1%EC%9D%BC" title="5월 1일">5월 1일</a> 개막하였고, <a href="/wiki/%EC%95%A0%EB%93%A4%EB%A0%88%EC%9D%B4%EB%93%9C_%EC%9C%A0%EB%82%98%EC%9D%B4%ED%8B%B0%EB%93%9C_FC" title="애들레이드 유나이티드 FC">애들레이드 유나이티드</a>가 <a href="/wiki/%EC%9B%A8%EC%8A%A4%ED%84%B4_%EC%8B%9C%EB%93%9C%EB%8B%88_%EC%9B%90%EB%8D%94%EB%9F%AC%EC%8A%A4_FC" title="웨스턴 시드니 원더러스 FC">웨스턴 시드니 원더러스</a>를 상대로 3-1 승리를 거두면서 우승을 차지했다.</p>
