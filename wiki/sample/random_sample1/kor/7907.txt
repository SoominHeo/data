<p><b>송익필</b>(宋翼弼, <a href="/wiki/1534%EB%85%84" title="1534년">1534년</a> <a href="/wiki/2%EC%9B%94_10%EC%9D%BC" title="2월 10일">2월 10일</a>∼<a href="/wiki/1599%EB%85%84" title="1599년">1599년</a> <a href="/wiki/8%EC%9B%94_8%EC%9D%BC" title="8월 8일">8월 8일</a>)은 <a href="/wiki/%EC%A1%B0%EC%84%A0" title="조선">조선</a> 중기의 서얼 출신 유학자, 정치인이다. 자(字)는 운장, 호는 구봉(龜峰) 또는 현승(玄繩), 본관은 <a href="/wiki/%EC%97%AC%EC%82%B0_%EC%86%A1%EC%94%A8" title="여산 송씨">여산</a>이다. 시호는 문경(文敬)이다.</p>
<p><a href="/wiki/%EC%95%88%EB%8B%B9" title="안당">안당</a>의 진외종손으로, 아버지는 서얼 출신 문신 <a href="/wiki/%EC%86%A1%EC%82%AC%EB%A0%A8" title="송사련">송사련</a>이며, 진외증조모는 안당 가문의 노비였다. 그는 <a href="/wiki/%EC%84%9C%EC%9D%B8" title="서인">서인</a> 예학의 태두인 <a href="/wiki/%EA%B9%80%EC%9E%A5%EC%83%9D" title="김장생">김장생</a>과 <a href="/wiki/%EA%B9%80%EC%A7%91_(1574%EB%85%84)" title="김집 (1574년)">김집</a>, <a href="/wiki/%EA%B9%80%EB%B0%98" title="김반">김반</a> 부자 및 <a class="mw-redirect" href="/wiki/%EC%9D%B8%EC%A1%B0_%EB%B0%98%EC%A0%95" title="인조 반정">인조 반정</a>의 공신 <a class="mw-redirect" href="/wiki/%EA%B9%80%EC%9C%A0_(1571%EB%85%84)" title="김유 (1571년)">김유</a> 등을 문하에서 길러냈다.</p>
<p>서얼 출신으로, 출생문제에 대한 시비 및 아버지가 안당 일족과 사림 인사들을 역모로 몬 것에 대해 세간의 비난을 받았다. 관직을 단념하고 고향에서 학문 연구와 후학 교육에 일생을 바쳤다. <a class="mw-redirect" href="/wiki/%EC%9C%A8%EA%B3%A1_%EC%9D%B4%EC%9D%B4" title="율곡 이이">율곡 이이</a>와 <a class="mw-redirect" href="/wiki/%EC%9A%B0%EA%B3%84_%EC%84%B1%ED%98%BC" title="우계 성혼">우계 성혼</a>, <a class="mw-redirect" href="/wiki/%EC%86%A1%EA%B0%95_%EC%A0%95%EC%B2%A0" title="송강 정철">송강 정철</a> 등의 절친한 벗으로, 서인의 이론가이자 <a class="new" href="/w/index.php?title=%EC%98%88%ED%95%99&amp;action=edit&amp;redlink=1" title="예학 (없는 문서)">예학</a>, <a href="/wiki/%EC%84%B1%EB%A6%AC%ED%95%99" title="성리학">성리학</a>, <a href="/wiki/%EA%B2%BD%ED%95%99" title="경학">경학</a>에 능하였다.</p>
<p>후일 안당의 증손부가 소송을 제기하여 환천의 위기에 처했으나, 제자 김장생의 숙부인 <a class="new" href="/w/index.php?title=%EA%B9%80%EC%9D%80%ED%9C%98&amp;action=edit&amp;redlink=1" title="김은휘 (없는 문서)">김은휘</a>가 그의 일족을 배려하여 10년간 먹여 살렸다고 한다. <a href="/wiki/1591%EB%85%84" title="1591년">1591년</a>(선조 24년) <a href="/wiki/%ED%8F%89%EC%95%88%EB%8F%84" title="평안도">평안도</a> <a class="mw-redirect" href="/wiki/%ED%9D%AC%EC%B2%9C" title="희천">희천</a>(熙川)으로 유배되었다가 <a href="/wiki/1593%EB%85%84" title="1593년">1593년</a> <a href="/wiki/9%EC%9B%94" title="9월">9월</a> 석방되었다. 사후 <a href="/wiki/%EC%82%AC%ED%97%8C%EB%B6%80" title="사헌부">사헌부</a><a class="mw-disambig" href="/wiki/%EC%A7%80%ED%8F%89" title="지평">지평</a>에 <a href="/wiki/%EC%B6%94%EC%A6%9D" title="추증">추증</a>되었다가 <a href="/wiki/1910%EB%85%84" title="1910년">1910년</a>(융희 4년) 때 다시 <a href="/wiki/%ED%99%8D%EB%AC%B8%EA%B4%80" title="홍문관">홍문관</a><a href="/wiki/%EC%A0%9C%ED%95%99" title="제학">제학</a>에 추증되었다.</p>

<h2><span class="mw-headline" id=".EC.83.9D.EC.95.A0">생애</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%86%A1%EC%9D%B5%ED%95%84&amp;action=edit&amp;section=1" title="부분 편집: 생애">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<h3><span class="mw-headline" id=".EC.83.9D.EC.95.A0_.EC.B4.88.EB.B0.98">생애 초반</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%86%A1%EC%9D%B5%ED%95%84&amp;action=edit&amp;section=2" title="부분 편집: 생애 초반">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<h4><span class="mw-headline" id=".EC.B6.9C.EC.83.9D.EA.B3.BC_.EA.B0.80.EA.B3.84">출생과 가계</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%86%A1%EC%9D%B5%ED%95%84&amp;action=edit&amp;section=3" title="부분 편집: 출생과 가계">편집</a><span class="mw-editsection-bracket">]</span></span></h4>
<p>송익필은 <a href="/wiki/1534%EB%85%84" title="1534년">1534년</a> <a href="/wiki/2%EC%9B%94_10%EC%9D%BC" title="2월 10일">2월 10일</a> <a href="/wiki/%EC%86%A1%EC%82%AC%EB%A0%A8" title="송사련">송사련</a>과 <a href="/wiki/%EC%97%B0%EC%9D%BC_%EC%A0%95%EC%94%A8" title="연일 정씨">연일 정씨</a>(延日鄭氏)의 4남 1녀 중 3남으로 <a href="/wiki/%ED%95%9C%EC%84%B1%EB%B6%80" title="한성부">한성부</a><sup class="reference" id="cite_ref-dangjs_1-0"><a href="#cite_note-dangjs-1">[1]</a></sup>에서 태어났다. 장남이 인필(仁弼), 차남이 부필(富弼)이고 셋째가 송익필이며 넷째는 운곡거사(雲谷居士) <a href="/wiki/%EC%86%A1%ED%95%9C%ED%95%84" title="송한필">송한필</a>이다.</p>
<p>그의 선조는 <a href="/wiki/%EA%B3%A0%EB%A0%A4" title="고려">고려</a> 때 상장군을 지낸 정렬공(貞烈公) <a href="/wiki/%EC%86%A1%EC%86%A1%EB%A1%80" title="송송례">송송례</a>(宋松禮)이다. <a href="/wiki/%EA%B3%A0%EB%A0%A4_%EC%9B%90%EC%A2%85" title="고려 원종">고려 원종</a>조에 <a href="/wiki/%EC%83%81%EC%9E%A5%EA%B5%B0" title="상장군">상장군</a>을 지낸 인물로 <a href="/wiki/%EB%AC%B4%EC%8B%A0%EC%A0%95%EA%B6%8C" title="무신정권">무신정권</a> 말에 국왕 원종의 편에 서서 무신정권 인사들을 타도하고 국권을 왕에게 환원시키는데 노력하였다. 그는 송송례의 후손이었으나 그 뒤 그의 가문은 점차 몰락하여 평민 신분으로 떨어졌다. 증조부는 송자근쇠<sup class="reference" id="cite_ref-2"><a href="#cite_note-2">[2]</a></sup>이고 할아버지는 갑사(甲士) 출신으로 <a href="/wiki/%EC%A7%81%EC%9E%A5" title="직장">직장</a>(直長)을 지낸 송린(宋麟, 또는 宋璘)이다.</p>
<p>아버지 송사련은 정승 <a href="/wiki/%EC%95%88%EB%8B%B9" title="안당">안당</a>의 아버지 <a class="new" href="/w/index.php?title=%EC%95%88%EB%8F%88%ED%9B%84&amp;action=edit&amp;redlink=1" title="안돈후 (없는 문서)">안돈후</a>가 비첩(婢妾) 중금(重今)에게서 얻은 얼녀 감정(甘丁)과 평민 출신 갑사 송린의 아들이었다. 진외증조모 중금의 신분이 본래 노비였기에 신분제에 따라 얼녀 감정은 안당 가문의 노비라 할 수 있다. 이후 감정이 평민 송린과 혼인하여 송사련을 낳았는데 송사련의 신분 또한 어머니와 마찬가지로 안당 가문의 재산에 속하나 송사련은 안당의 도움으로 천한 신분을 면하고 관직에 올라 잡직인 <a href="/wiki/%EA%B4%80%EC%83%81%EA%B0%90" title="관상감">관상감</a><a class="mw-disambig" href="/wiki/%ED%8C%90%EA%B4%80" title="판관">판관</a>(觀象監判官)을 지냈다.</p>
<p>이후, <a href="/wiki/%EA%B8%B0%EB%AC%98%EC%82%AC%ED%99%94" title="기묘사화">기묘사화</a>가 일어나 <a href="/wiki/%EC%82%AC%EB%A6%BC%ED%8C%8C" title="사림파">사림파</a>가 숙청되었으며 <a href="/wiki/%EC%A1%B0%EA%B4%91%EC%A1%B0" title="조광조">조광조</a>를 따르던 안당 일족도 위기에 처하자, 안당의 아들인 <a class="new" href="/w/index.php?title=%EC%95%88%EC%B2%98%EA%B2%B8&amp;action=edit&amp;redlink=1" title="안처겸 (없는 문서)">안처겸</a>·<a class="new" href="/w/index.php?title=%EC%95%88%EC%B2%98%EA%B7%BC&amp;action=edit&amp;redlink=1" title="안처근 (없는 문서)">안처근</a> 형제는 기묘사화를 일으킨 <a href="/wiki/%EB%82%A8%EA%B3%A4" title="남곤">남곤</a>, <a href="/wiki/%EC%8B%AC%EC%A0%95_(1471%EB%85%84)" title="심정 (1471년)">심정</a> 등이 자신들마저도 가만두지 않을 것이라는 생각을 가지고 주위에 불평을 하고 다니며<sup class="reference" id="cite_ref-dangjs_1-1"><a href="#cite_note-dangjs-1">[1]</a></sup> 이들을 제거할 계획을 세웠다. 송사련은 이들의 대화 내용을 엿듣고 역모죄로 고발하였으며 안당의 부인의 부음을 듣고 찾아온 문상객과 장례를 도운 일꾼들을 거사 참여 세력으로 꾸며 고변하였다. 그의 고변으로 안당과 세 아들은 사형당하고 안당의 집안은 멸문되고 재산과 노비는 송사련이 차지하게 되었으며, 그 공으로 벼슬도 당상관 첨지까지 오르게 되었다.</p>
<h4><span class="mw-headline" id=".EC.86.8C.EB.85.84.EA.B8.B0.EC.99.80_.EC.B2.AD.EB.85.84.EA.B8.B0">소년기와 청년기</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%86%A1%EC%9D%B5%ED%95%84&amp;action=edit&amp;section=4" title="부분 편집: 소년기와 청년기">편집</a><span class="mw-editsection-bracket">]</span></span></h4>
<p>신분과 도리를 중시하는 유생들은 송사련에게 등을 돌리며 심히 비난하였다. 송사련은 80세로 죽을 때까지 양반으로 부귀를 누렸지만 선비들은 이를 무시하였고, 사림파가 <a href="/wiki/%EC%A1%B0%EC%84%A0_%EB%AA%85%EC%A2%85" title="조선 명종">명종</a>말엽에 집권하게 되면서 그 화(禍)는 송익필 형제들에게 대물림된다.</p>
<p>아버지 사련이 안처겸·안처근 형제가 남곤, 심정, <a href="/wiki/%EA%B9%80%EC%A0%84" title="김전">김전</a> 등을 제거하려는 모의를 폭로, 고변한 공로로 공신에 책봉되고 양반이 되었기에 그의 형제들은 유복한 환경에서 교육받았다. 어려서부터 기억력이 좋았던 그는 나이 7～8세에 붓을 잡되 그 조어(造語)가 사람을 놀라게 했다. 아버지 송사련은 생모가 노비 출신이고, 자신은 서얼 신분이었으므로 아들들의 장래에 누가 될까봐 걱정하였다. 스승이 없이 독학으로 <a href="/wiki/%EC%84%B1%EB%A6%AC%ED%95%99" title="성리학">성리학</a>을 수학하였으며, 차차 자라면서 문장력이 늘고 학문 연구를 게을리하지 않았다.</p>
<p>아버지 송사련의 후광으로 다양한 이들과 교유하기 시작했다.</p>
<h3><span class="mw-headline" id=".EC.B2.AD.EB.85.84.EA.B8.B0">청년기</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%86%A1%EC%9D%B5%ED%95%84&amp;action=edit&amp;section=5" title="부분 편집: 청년기">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<h4><span class="mw-headline" id=".EC.97.B0.EC.A2.8C.EC.A0.9C.EC.99.80_.EA.B4.80.EC.A7.81_.EB.8B.A8.EB.85.90">연좌제와 관직 단념</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%86%A1%EC%9D%B5%ED%95%84&amp;action=edit&amp;section=6" title="부분 편집: 연좌제와 관직 단념">편집</a><span class="mw-editsection-bracket">]</span></span></h4>
<p>20대에 이미 문장력을 인정받아 문장가로 지목되기도 하였다. <a href="/wiki/1558%EB%85%84" title="1558년">1558년</a>(명종 13년) 25세 때 아우 송학필과 같이 초시(初試)에 합격하였다. 이후 그의 답안지가 과거 시험 대과에 뽑혔으나 사관(史官) <a class="mw-disambig" href="/wiki/%EC%9D%B4%ED%95%B4%EC%88%98" title="이해수">이해수</a>(李海壽)등은 상소를 올려 "사련은 예의를 저버린 죄인이니 그 상직(賞職)을 없애야 한다"며 "그 자식들은 역시 얼손(孼孫)들이니 법(法)을 어기고 과거에 나아감은 부당하다."고 규탄, 과거의 고시관이던 동료들과 의논하여 그의 과거 응시를 정지시키고 벼슬길을 막아버렸다. 아버지의 무고행위에 대한 비난과 출생신분의 문제 때문에 세간의 비난을 받았고, 결국 이후 벼슬에 진출하기를 단념하고 <a href="/wiki/%EA%B2%BD%EA%B8%B0%EB%8F%84" title="경기도">경기도</a> <a href="/wiki/%EA%B5%90%ED%95%98%EA%B5%B0" title="교하군">교하군</a> 심학산으로 낙향하여 학문 연구와 제자 교육에 전념하였다.</p>
<p>명종말엽 사림파가 집권하면서 <a href="/wiki/1566%EB%85%84" title="1566년">1566년</a>(명종 21) 안당이 아무 잘못이 없음이 밝혀져 신원(伸寃), 복권되고 그 직첩이 환급되었으며, 안처근 형제의 거사를 폭로하고 무고까지 씌운 송사련에 대한 비난여론이 조성되었다. 사관 이해수는 서손의 과거 응시는 부당하다며 다시 들고 일어났다. 이후 그는 <a href="/wiki/%EA%B2%BD%EA%B8%B0%EB%8F%84" title="경기도">경기도</a> <a href="/wiki/%EA%B3%A0%EC%96%91%EA%B5%B0" title="고양군">고양군</a>으로 물러나 고양 구봉산(龜峰山) 아래 은거하여 학문에 몰두하면서 후진양성에 힘쓴다. 그의 호인 구봉은 이 산의 이름을 딴 것이다.</p>
<h4><span class="mw-headline" id=".ED.95.99.EB.AC.B8_.EC.97.B0.EA.B5.AC">학문 연구</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%86%A1%EC%9D%B5%ED%95%84&amp;action=edit&amp;section=7" title="부분 편집: 학문 연구">편집</a><span class="mw-editsection-bracket">]</span></span></h4>
<p>27세를 전후한 시기부터 교육자의 길을 걷게 된다. 그가 후학을 양성하던 곳은 경기도 교하에 있는 심학산의 구봉 아래였다.<sup class="reference" id="cite_ref-dangjs_1-2"><a href="#cite_note-dangjs-1">[1]</a></sup> 교하 심학산에 서당을 짓고 이후 유생들과 문인들에게 글과 학문을 가르쳤다.</p>
<p>이이·성혼 등과 사귀어 성리학을 논하여 통하였고 예학에도 뛰어났으며 특히 문장에 능해 <a href="/wiki/%EC%9D%B4%EC%82%B0%ED%95%B4" title="이산해">이산해</a>, <a href="/wiki/%EC%B5%9C%EA%B2%BD%EC%B0%BD" title="최경창">최경창</a>, <a href="/wiki/%EB%B0%B1%EA%B4%91%ED%9B%88" title="백광훈">백광훈</a> 등과 함께 8문장가<sup class="reference" id="cite_ref-3"><a href="#cite_note-3">[3]</a></sup>의 한 사람으로 시와 글씨에도 탁월하였다. <a href="/wiki/%EA%B9%80%EC%9E%A5%EC%83%9D" title="김장생">김장생</a>·정엽 등 많은 제자를 배출하였으며, 특히 김장생은 스승의 예학을 계승하여 조선 예학의 대가가 되었다.<sup class="reference" id="cite_ref-4"><a href="#cite_note-4">[4]</a></sup> 특히 예학에 밝아 김장생에게 큰 영향을 주었으며, 이는 후대의 <a href="/wiki/%EC%86%A1%EC%8B%9C%EC%97%B4" title="송시열">송시열</a>, <a href="/wiki/%EC%9C%A4%EC%84%A0%EA%B1%B0" title="윤선거">윤선거</a>, <a href="/wiki/%EC%86%A1%EC%A4%80%EA%B8%B8" title="송준길">송준길</a> 등에게로 학문이 계승되었다. 또한 정치적 감각이 뛰어나 서인 세력의 막후실력자가 되기도 하였다.</p>
<p>송익필에게 배운 제자 중에서 유명한 이가 김장생으로, 친구인 이이의 수제자이기도 했다.<sup class="reference" id="cite_ref-5"><a href="#cite_note-5">[5]</a></sup> 송익필은 미리 자세히 설명하지 않고 제자가 스스로 노력해서 터득하게 지도했기 때문에 김장생은 먼저 스스로 수없이 배우고, 읽고 생각하여 깨달은 바가 있은 뒤에야 의문 나는 바를 물어 보았다고 한다. 그가 교하에서 서당을 열고 후학을 양성하는 동안 인근에 살던 이이와 성혼과의 교유가 깊어 이들과 만나서 토론하거나 서신을 주고받으며 철학의 이치와 세상사는 도리를 서로 탐구하였는데, 대개 이이와 성혼의 질문에 송익필이 답하는 식이었다. 이들의 우정은 서로가 죽을 때까지 계속되었다.</p>

<p><a href="/wiki/1575%EB%85%84" title="1575년">1575년</a>(선조 8) 그의 나이 42세 때 아버지 송사련이 사망하였고 그 해에 <a class="new" href="/w/index.php?title=%EC%8B%A0%EC%82%AC%EB%AC%B4%EC%98%A5&amp;action=edit&amp;redlink=1" title="신사무옥 (없는 문서)">신사무옥</a>이 송사련이 안처겸·안처근 형제의 대화를 엿듣고 폭로한 것이 밝혀졌다. <a href="/wiki/1586%EB%85%84" title="1586년">1586년</a>(선조 19) 사화 때 살아남아 그때까지 숨어살던 안당의 증손자인 안로(安璐)의 처 윤씨(尹氏)가 신원 상소를 올렸다. 죽은 안당과 세 아들, 손자들이 모두 복권되었고 안당에게는 '정민'(貞愍)이란 시호가 내려졌다. 동시에 자신과 아버지를 관직으로 추천한 안당 일가를 죽음으로 몰고 간 송사련에 대한 비난 여론이 조성되었다. 송씨 집안도 맞상소하여 싸웠으나 결국 패하여 관작이 삭탈되었다. 이이와 성혼은 아버지의 잘못을 그에게까지 적용하는 것은 무리한 것이라고 변호하였으나 동인에서는 계속 이를 문제삼았다.</p>
<p><a href="/wiki/1584%EB%85%84" title="1584년">1584년</a>(선조 17) 그의 절친한 친구이자 그의 후원자인 율곡 이이가 격무와 과로로 병이 심해져 향년 49세의 나이로 세상을 뜨게 된다. 안당의 후손들은 계속 그를 공격하였는데, 그를 변호해주던 율곡 이이의 죽음은 그에게 큰 타격을 주었다. 동인과 서인이 한창 대립각을 세우고 있었는데, 동서 양당의 분쟁을 조절하려 노력했으나 서인이었던 율곡 이이에 대한 동인의 공격은 그의 사후에도 수그러들지 않았다.</p>
<p>이때 우계 성혼과 조헌, 이이의 제자이자 자신의 제자이기도 한 묵재 <a href="/wiki/%EC%9D%B4%EA%B7%80" title="이귀">이귀</a>(李貴) 등과 함께 이에 맞서 율곡을 변호하는 상소를 올렸는데, 송익필은 적극 나서서 자신의 상소 외에 이귀의 상소문 초안까지 작성해 주었다. 이때 송익필은 동인으로부터 '서인의 모주'(謀主)로 지목되면서 동인들은 그를 필사적으로 제거하려 하였다. 이후 율곡에 대한 비난이 그에게로 방향이 바뀐다. 동인들은 송익필의 제거를 위해 송사련에게 원한이 사무친 안당의 증손자며느리 윤씨를 비롯한 안당의 후손들을 지지하고 도와준다.</p>
<p>성혼과 이이를 구수(仇讎)로 여기던 인사들은 노여움을 그에게로 돌려 계속 공격했다. 조헌은 글을 올리되 백의 차임으로 도끼를 들고 궐문에 엎드려 '송익필을 굳이 벌 주려거든 나도 죽여 달라'고 청하였다. 그가 이를 듣고필기하되'조여식(趙汝式, 조선의 자)과 만난지가 10년이 넘었는데 소인들이 글 가운데에 나를 못난 사람이라 함으로써 함께 이같이 억울함을 받노라'하며 한탄하였다. <a href="/wiki/1586%EB%85%84" title="1586년">1586년</a> 그의 형제들이 모두 자취를 감추고 구수를 피하였다. 이때 조헌이 글을 올려 그의 억울함을 풀어 주도록 극언(極言)하고 또한 그의 현명(賢明)함을 들어 벼슬을 줄 것을 상소하였으나 선조는 들어주지 않았다.</p>
<h4><span class="mw-headline" id=".EB.8F.99.EC.9D.B8.EA.B3.BC.EC.9D.98_.EA.B0.88.EB.93.B1">동인과의 갈등</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%86%A1%EC%9D%B5%ED%95%84&amp;action=edit&amp;section=10" title="부분 편집: 동인과의 갈등">편집</a><span class="mw-editsection-bracket">]</span></span></h4>
<p>동인 중 강경파였던 <a class="mw-disambig" href="/wiki/%EC%9D%B4%EB%B0%9C" title="이발">이발</a>(李潑), <a class="new" href="/w/index.php?title=%EB%B0%B1%EC%9C%A0%EC%96%91&amp;action=edit&amp;redlink=1" title="백유양 (없는 문서)">백유양</a>(白惟讓) 등은 안당의 후손들의 소송과 공격을 적극 후원하였다. 일설에는 송익필의 조모가 안돈후의 서녀가 아니라 노비인 중금의 전남편 소생 딸이라는 설도 있었는데, 이발, 백유양 등은 안당의 후손들을 사주하여 이 부분을 물고 늘어져 "구봉의 진외증조모(할머니의 어머니) 중금은 속양(贖良)되지 않은 천인(賤人)이며, 할머니 감정은 안감정이 아니며, 안씨의 핏줄이 아닌 노비인 중금의 전남편 소생"이라면서 "감정은 안감정이 아니고 노비가 맞다"며 그 자손들은 당연히 안씨의 노비라고 주장했다.</p>
<p>일설에는 안당 집안의 문적 중 안당이 송사련과 그의 어머니 감정, 조모 중금을 속량시킨 문서로 4대에 걸쳐 내려온 송씨 집안의 양적(良籍, 양인임을 나타내는 문서)이 있었다는 설도 있는데, 이때 속량 문서까지 모두 없애 버리고, 송사련이 은혜를 저버리고 안처겸 형제의 거사를 폭로한 것을 물고늘어져, 2대 이상 양역(良役)하면 노비를 면할 수 있다는 법 규정을 적용받지 못하게 하여 <a href="/wiki/1586%EB%85%84" title="1586년">1586년</a>(선조 19)에 송씨 집안을 완전히 환천(還賤)시켜 버렸다.</p>

<p>안당의 증손자며느리 윤씨는 소송을 걸었고, 노비로 환천됨과 동시에 부친 송사련은 사후 관작이 추탈되고 안씨 집안 사람들에 의해 무덤이 파헤쳐지고 무덤과 시신이 크게 훼손당하는 변을 당한다. 2대 이상 양역을 하면 면천하게 하는 규정까지 적용이 되지 않고 속량문서까지 사라지면서 안씨네 노비가 되어 버린 송익필의 4형제와 일가 70여 명은, 피맺힌 복수심에 온갖 핍박을 가하려는 그들을 피해 살길을 찾아 각지로 피신, 흩어졌다. 이때 송익필은 동인인 <a href="/wiki/%EC%9D%B4%EC%82%B0%ED%95%B4" title="이산해">이산해</a>로부터 율곡 이이를 비난해 보라는 권유를 받자 이를 완강히 거절하고, 오히려 그가 시세에 결탁하며 음모를 꾸민다며 그를 풍자하는 시를 지어줌으로써 이산해의 분노를 사게 된다.</p>
<p>조헌이 이산해를 비열하다고 비난하는 상소를 올렸는데 이때 조헌의 이산해 비난 상소의 배후 조종 인물 혐의를 받아 왕명으로 동생 송한필과 함께 구속되었다. <a href="/wiki/1589%EB%85%84" title="1589년">1589년</a> <a href="/wiki/%EC%A0%95%EC%97%AC%EB%A6%BD" title="정여립">정여립</a>의 역모 고변으로 <a href="/wiki/%EA%B8%B0%EC%B6%95%EC%98%A5%EC%82%AC" title="기축옥사">기축옥사</a>가 벌어지면서 풀려나게 된다. 이 때문에 그는 기축옥사의 배후로도 지목하는 주장이 있어 그 사실여부를 두고 지금까지 논쟁이 되고 있다.</p>
<h4><span class="mw-headline" id=".EB.8F.99.EC.9D.B8.EC.9D.98_.EA.B3.B5.EC.84.B8">동인의 공세</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%86%A1%EC%9D%B5%ED%95%84&amp;action=edit&amp;section=11" title="부분 편집: 동인의 공세">편집</a><span class="mw-editsection-bracket">]</span></span></h4>
<p>송익필은 이이, 성혼 등과 너나 하며 평교하던 사이로, 지혜가 뛰어나 '<a href="/wiki/%EC%84%9C%EC%9D%B8" title="서인">서인</a>의 <a class="mw-redirect" href="/wiki/%EC%A0%9C%EA%B0%88%EA%B3%B5%EB%AA%85" title="제갈공명">제갈공명</a>'이란 말을 들었는데, 그 신분에 서얼이라는 결정적인 약점을 갖고 있었다. 그런데 송익필의 이러한 약점은 '곽사원의 제방 송사'에서 드디어 폭발하였다.<sup class="reference" id="cite_ref-kimjaejo111_6-0"><a href="#cite_note-kimjaejo111-6">[6]</a></sup> 1579년의 곽사원의 제방 송사가가 <a href="/wiki/%EC%A0%95%EC%96%B8%EC%A7%80" title="정언지">정언지</a> 등에 의해 <a href="/wiki/1589%EB%85%84" title="1589년">1589년</a> 터지게 된다. 이 10년 된 사건은 결국 주관 부처인 공조의 <a href="/wiki/%EC%B0%B8%ED%8C%90" title="참판">참판</a> 정언지의 상소로 선조의 결심을 얻어 조사하게 되었다. <sup class="reference" id="cite_ref-kimjaejo111_6-1"><a href="#cite_note-kimjaejo111-6">[6]</a></sup>그 과정에서 이이 등이 의혹이 있는 송익필 일가의 뒤를 이이가 봐주었다는 것이다.</p>
<p>토목 공사를 둘러싼 부정 사건에 송익필의 조카 사위(송익필의 동생 송한필의 사위)가 관련되었고, 그 동안 이이가 이 사건을 극력으로 돌보아주었다<sup class="reference" id="cite_ref-kimjaejo111_6-2"><a href="#cite_note-kimjaejo111-6">[6]</a></sup>는 의혹이 있다. 결국 선조가 교서를 내려 송익필을 축출하였다.</p>

<p>결국 그가 음모를 꾸며서 이런 송사 사건을 은폐했다는 의혹이 제기되었다. 얼마 후 이발, 이길, 백유양 등 동인당 편에서 송익필에게 공격의 화살을 집중시켰다.<sup class="reference" id="cite_ref-kimjaejo112_7-0"><a href="#cite_note-kimjaejo112-7">[7]</a></sup></p>

<p><a href="/wiki/1589%EB%85%84" title="1589년">1589년</a>(선조 22년) <a class="mw-redirect" href="/wiki/%EC%A0%95%EC%97%AC%EB%A6%BD%EC%9D%98_%EC%98%A5%EC%82%AC" title="정여립의 옥사">정여립의 옥사</a>가 발생한다. 선조는 <a href="/wiki/%EC%A2%8C%EC%9D%98%EC%A0%95" title="좌의정">좌의정</a> 이산해, <a href="/wiki/%EC%9A%B0%EC%9D%98%EC%A0%95" title="우의정">우의정</a> <a href="/wiki/%EC%A0%95%EC%96%B8%EC%8B%A0" title="정언신">정언신</a> 등에게 위관(委官)이 되어 죄인들을 심문하게 했다.<sup class="reference" id="cite_ref-lee161_8-0"><a href="#cite_note-lee161-8">[8]</a></sup> 그러나 정철이 차자를 올려 정언신이 정여립의 일가이니 재판관으로는 적당하지 않으므로 교체해야 한다고 주장했다. 결국 선조는 정언신 대신 정철을 우의정으로 제수하고 위관으로 삼았다.<sup class="reference" id="cite_ref-lee161_8-1"><a href="#cite_note-lee161-8">[8]</a></sup> 기축옥사 결과 2천 명의 동인 당원이 희생되었다.</p>
<p><a href="/wiki/1591%EB%85%84" title="1591년">1591년</a> 여름 세자건저 문제로 조헌과 정철이 귀양을 가자 시(詩)로써 위로를 표하였다.</p>
<p>그러나 이산해 등은 기회를 노려 <a href="/wiki/%EA%B1%B4%EC%A0%80_%EB%AC%B8%EC%A0%9C" title="건저 문제">건저 문제</a>로 서인들을 실각시킨다. 동인은 그가 기축옥사를 날조했다고 의심했고, 송익필은 김장생, <a class="new" href="/w/index.php?title=%EA%B9%80%EC%9D%80%ED%9C%98&amp;action=edit&amp;redlink=1" title="김은휘 (없는 문서)">김은휘</a>, 조헌, 성혼, 정철, 윤근수, 윤두수 등이 그를 도와 <a href="/wiki/%EC%B6%A9%EC%B2%AD%EB%8F%84" title="충청도">충청도</a>와 <a href="/wiki/%EC%A0%84%EB%9D%BC%EB%8F%84" title="전라도">전라도</a> <a class="mw-redirect" href="/wiki/%EA%B4%91%EC%A3%BC" title="광주">광주</a> 등지로 피신시켰다.</p>
<h3><span class="mw-headline" id=".EC.83.9D.EC.95.A0_.ED.9B.84.EB.B0.98">생애 후반</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%86%A1%EC%9D%B5%ED%95%84&amp;action=edit&amp;section=13" title="부분 편집: 생애 후반">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<h4><span class="mw-headline" id=".EB.85.B8.EB.B9.84_.ED.99.98.EC.B2.9C.EA.B3.BC_.EC.9C.A0.EB.B0.B0.2C_.EC.84.9D.EB.B0.A9">노비 환천과 유배, 석방</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%86%A1%EC%9D%B5%ED%95%84&amp;action=edit&amp;section=14" title="부분 편집: 노비 환천과 유배, 석방">편집</a><span class="mw-editsection-bracket">]</span></span></h4>
<div class="thumb tleft">

<div class="rellink boilerplate seealso"><img alt="" data-file-height="128" data-file-width="128" height="16" src="//upload.wikimedia.org/wikipedia/commons/thumb/e/ec/Crystal_Clear_app_xmag.svg/16px-Crystal_Clear_app_xmag.svg.png" srcset="//upload.wikimedia.org/wikipedia/commons/thumb/e/ec/Crystal_Clear_app_xmag.svg/24px-Crystal_Clear_app_xmag.svg.png 1.5x, //upload.wikimedia.org/wikipedia/commons/thumb/e/ec/Crystal_Clear_app_xmag.svg/32px-Crystal_Clear_app_xmag.svg.png 2x" width="16"/> <a href="/wiki/%EC%9E%84%EC%A7%84%EC%99%9C%EB%9E%80" title="임진왜란">임진왜란</a> 문서를 참고하십시오.</div>
<p>안씨 집안의 노비로 환천을 피하여 은신하였다. 그러나 기축옥사로 2천여 명 이상의 동인과 그 일가족이 숙청되자 동인들은 옥사의 배후로 그를 지목하고 추적하였다. 그는 성혼, 김계휘 등이 피난처를 주선해주어 옮겨다녔다. 김계휘의 동생이자 제자 김장생의 숙부인 김은휘는 자신의 집에 송익필과 송한필 가족을 숨겨주기도 했다.</p>
<p>그러나 곧 조정에 자수하여 <a href="/wiki/1591%EB%85%84" title="1591년">1591년</a>(선조 24) 탄핵을 받고 <a href="/wiki/%ED%8F%89%EC%95%88%EB%8F%84" title="평안도">평안도</a> <a class="mw-redirect" href="/wiki/%ED%9D%AC%EC%B2%9C" title="희천">희천</a>(熙川)으로 유배되었다가 <a href="/wiki/%EC%9E%84%EC%A7%84%EC%99%9C%EB%9E%80" title="임진왜란">임진왜란</a>이 터지고 <a href="/wiki/1593%EB%85%84" title="1593년">1593년</a> <a href="/wiki/9%EC%9B%94" title="9월">9월</a>에 석방되었다. 유배에서 풀려나왔지만 그에 대한 공격이 계속되었으므로 그는 은신처를 피해 숨어다녀야 했다.</p>
<p>이후 그는 임진왜란 기간 중 <a href="/wiki/%EC%A0%84%EB%9D%BC%EB%8F%84" title="전라도">전라도</a> <a href="/wiki/%EC%A7%84%EC%95%88%EA%B5%B0" title="진안군">진안군</a>의 <a href="/wiki/%EC%9A%B4%EC%9E%A5%EC%82%B0" title="운장산">운장산</a>에 은거하며 난리를 피했다.<sup class="reference" id="cite_ref-woonj_9-0"><a href="#cite_note-woonj-9">[9]</a></sup></p>
<h3><span class="mw-headline" id=".EC.B5.9C.ED.9B.84">최후</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%86%A1%EC%9D%B5%ED%95%84&amp;action=edit&amp;section=15" title="부분 편집: 최후">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>만년에는 정해진 주거지를 얻지 못하고 전국 여기저기를 떠돌다 마침내 만년의 은신처인 당진군의 마양촌(현재 송산면 매곡리에 있는 숨은골(숨어골))에 오게 된다.<sup class="reference" id="cite_ref-dangjs_1-3"><a href="#cite_note-dangjs-1">[1]</a></sup> 이 때의 나이가 63세이다.</p>
<p>이후 <a href="/wiki/%EC%9A%B4%EC%9E%A5%EC%82%B0" title="운장산">운장산</a>에서 내려와 <a href="/wiki/1596%EB%85%84" title="1596년">1596년</a>(선조 29) <a class="new" href="/w/index.php?title=%EA%B9%80%EC%A7%84%EB%A0%A4&amp;action=edit&amp;redlink=1" title="김진려 (없는 문서)">김진려</a>의 도움으로 <a href="/wiki/%EC%B6%A9%EC%B2%AD%EB%8F%84" title="충청도">충청도</a> <a class="mw-redirect" href="/wiki/%EB%8B%B9%EC%A7%84%EA%B5%B0" title="당진군">당진군</a> 마양촌(馬羊村)에 집을 마련하게 되면서 안정된 만년을 보냈다. 그러나 그의 소식을 들은 문인들과 젊은 학도들이 속속들이 찾아왔고, 이미 노비로 환천된 데다가 석방되었지만 세상의 비난과 공격을 피해 도망다니는 처지인데도 불구하고, 그의 가르침을 받으려는 발길은 끊이지 않았다. 그는 당진 마양촌에서 은거하며 <a class="mw-disambig" href="/wiki/%EC%A3%BC%EC%9E%90" title="주자">주자</a>의 《가례》(家禮)를 보충 설명하고 주석을 달은 《가례주설》(家禮註說)을 짓고, 이이, 성혼, 정철, 윤두수 등과 주고받은 서신들을 묶어 현승편(玄繩編)을 엮었다.</p>
<p>평생 험한 일을 겪으며 세상을 살았음에도 그는 누구를 탓하지 않았다.<sup class="reference" id="cite_ref-dangjs_1-4"><a href="#cite_note-dangjs-1">[1]</a></sup> 저서로는 구봉집, 현승집(玄繩集), 이이, 성혼, 정철, 윤두수 등과의 서신과 학문에 대한 견해를 주고받은 모음집 현승편(玄繩編) 등이 있다.<sup class="reference" id="cite_ref-10"><a href="#cite_note-10">[10]</a></sup></p>
<p>1599년(선조 32) 8월 8일 <a class="mw-redirect" href="/wiki/%EB%8B%B9%EC%A7%84" title="당진">당진</a> 원당동의 우거지에서 사망하였다. 당진 북면(北面) 원당동(元堂洞) 숨은골 뒤편 야산에 안장되었다.</p>
<h3><span class="mw-headline" id=".EC.82.AC.ED.9B.84">사후</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%86%A1%EC%9D%B5%ED%95%84&amp;action=edit&amp;section=16" title="부분 편집: 사후">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>제자와 후학들은 스승의 삶을 안타까워하며 누대에 걸쳐 그를 위한 신원운동을 벌인다. 사후 여러번의 신원 상소가 올려졌으나 모두 묵살당했고, <a href="/wiki/%EC%9D%B8%EC%A1%B0%EB%B0%98%EC%A0%95" title="인조반정">인조반정</a> 후 <a href="/wiki/1625%EB%85%84" title="1625년">1625년</a>(인조 3) 김장생 등의 제자들이 상소하여 스승의 양민 환원을 요청했으나 이뤄지지 않았다. 이후 김장생이 송익필의 복권 상소를 계속 올려 복권시켰다.</p>
<p><a href="/wiki/1752%EB%85%84" title="1752년">1752년</a>(영조 28) <a href="/wiki/%EC%B6%A9%EC%B2%AD%EB%8F%84" title="충청도">충청도</a><a href="/wiki/%EA%B4%80%EC%B0%B0%EC%82%AC" title="관찰사">관찰사</a> <a href="/wiki/%ED%99%8D%EA%B3%84%ED%9D%AC" title="홍계희">홍계희</a>의 상소로 <a href="/wiki/%EC%82%AC%ED%97%8C%EB%B6%80" title="사헌부">사헌부</a><a class="mw-disambig" href="/wiki/%EC%A7%80%ED%8F%89" title="지평">지평</a>(司憲府持平)에 <a class="mw-redirect" href="/wiki/%EC%A6%9D%EC%A7%81" title="증직">증직</a>되었다. 후에 충청도 당진군 원당에 그의 사당인 입한재가 세워졌다.</p>
<p><a href="/wiki/1762%EB%85%84" title="1762년">1762년</a>(영조 38)에는 제자인 김장생의 6대손에 의해 그의 저서 《구봉집》이 발간되었고, 족보에서 누락되어 빠져있던 그의 이름도 사후 3백여 년이 지난 <a href="/wiki/1905%EB%85%84" title="1905년">1905년</a>에 족보에 등재되었다. <a href="/wiki/1910%EB%85%84" title="1910년">1910년</a>(순종 4) <a href="/wiki/%ED%99%8D%EB%AC%B8%EA%B4%80" title="홍문관">홍문관</a><a href="/wiki/%EC%A0%9C%ED%95%99" title="제학">제학</a>(弘文館提學)에 추증되고 문경(文敬)의 시호가 내려졌다.</p>
<p><a href="/wiki/2007%EB%85%84" title="2007년">2007년</a> <a href="/wiki/8%EC%9B%94" title="8월">8월</a>, <a href="/wiki/%EA%B2%BD%EA%B8%B0%EB%8F%84" title="경기도">경기도</a>에서 '북부 8월의 문화인물'로 선정되었다.<sup class="reference" id="cite_ref-11"><a href="#cite_note-11">[11]</a></sup></p>

<h2><span class="mw-headline" id=".ED.95.99.EB.AC.B8.EC.A0.81_.EC.B9.98.EC.A0.81">학문적 치적</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%86%A1%EC%9D%B5%ED%95%84&amp;action=edit&amp;section=19" title="부분 편집: 학문적 치적">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>그는 무너진 사회 질서를 회복시키는 방안으로 예의와 예절을 강조하였다. 제자들 중 사계 김장생은 단연 돋보이는 인물로, 그는 송익필에게서 예의와 예절에 대한 예론을 전수받아 이를 아들 <a href="/wiki/%EA%B9%80%EC%A7%91_(1574%EB%85%84)" title="김집 (1574년)">김집</a>과 함께 조선 예학의 대가가 되었다. 그리고 이들 부자의 문하에서 우암 송시열, 동춘당 송준길, 미촌 <a href="/wiki/%EC%9C%A4%EC%84%A0%EA%B1%B0" title="윤선거">윤선거</a>, 명재 <a href="/wiki/%EC%9C%A4%EC%A6%9D" title="윤증">윤증</a> 등이 배출되어 예학을 더욱 크게 계승 발전시킨다.</p>
<p>그러나 그의 미천한 신분과 함께 조소의 대상이 되고<sup class="reference" id="cite_ref-12"><a href="#cite_note-12">[12]</a></sup>, 인정받지 못했다.</p>
<h2><span class="mw-headline" id=".ED.8F.89.EA.B0.80.EC.99.80_.EB.B9.84.ED.8C.90">평가와 비판</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%86%A1%EC%9D%B5%ED%95%84&amp;action=edit&amp;section=20" title="부분 편집: 평가와 비판">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<h3><span class="mw-headline" id=".EA.B8.8D.EC.A0.95.EC.A0.81_.ED.8F.89.EA.B0.80">긍정적 평가</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%86%A1%EC%9D%B5%ED%95%84&amp;action=edit&amp;section=21" title="부분 편집: 긍정적 평가">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>송익필은 스승이 없이 독학으로 공부하였으나 성리학을 깊이 연구하여 그 이치에 밝았다.</p>
<p>자신의 학문과 재능에 대한 자부심이 강하여 스스로 고고하게 처신하였고, 아무리 고관·귀족이라도 한번 친구로 사귀면 ‘자(字)’로 부르고 관으로 부르지 않았다. 이러한 태도가 그의 미천한 신분과 맞물려 조소의 대상이 되기도 하였다. 율곡 이이는 그를 평하기를 "성리학을 논할 만한 자는 오직 송익필 형제 뿐"이라고 할 정도였다.</p>
<p>그러나 인조반정의 1등 공신 9명을 직·간접 제자로 둔 서인 세력의 정신적 구심점<sup class="reference" id="cite_ref-jo01_13-0"><a href="#cite_note-jo01-13">[13]</a></sup>이었으며, 정여립 사건을 비롯한 역사의 고비마다 논리와 심리를 파고드는 예리한 상소를 통해 동인의 몰락<sup class="reference" id="cite_ref-jo01_13-1"><a href="#cite_note-jo01-13">[13]</a></sup>을 불러오게 했다는 평가도 있다.</p>
<p>조헌은 말하기를, “늙도록 글 읽기를 쉬지 않아, 학문이 깊고 경(經)에 밝았으며, 행실이 방정하고 말이 정직하여 아버지의 허물을 덮기에 충분하였다. 그러므로 성혼·이이 두 선생이 다 스승 같은 벗으로 대하였다. 또한 가르치는 방법에 있어서도 상대방을 잘 일깨워주었으니, 스스로 분발하여 뜻을 세우도록 유도하였다.”라 하였고, <a class="mw-redirect" href="/wiki/%ED%86%A0%EC%A0%95_%EC%9D%B4%EC%A7%80%ED%95%A8" title="토정 이지함">토정 이지함</a>은 말하기를, “천지를 가슴속에 간직하였으니, <a href="/wiki/%EA%B3%B5%EC%9E%90" title="공자">공자</a>·<a href="/wiki/%EB%A7%B9%EC%9E%90" title="맹자">맹자</a>의 도(道)도 진실로 멀지 않았다.”라 하였고, 상촌 신흠은 말하기를, “천품(天稟)이 매우 높고 문장(文章) 신묘하였다.”라 하였고, 택당 <a class="mw-disambig" href="/wiki/%EC%9D%B4%EC%8B%9D" title="이식">이식</a>은 말하기를, “타고난 자질이 투철하고 지혜로워서, 이치를 분석함이 정밀하였다.”라고 평하였다.</p>
<p>실록에는 '송익필은 비록 송사련의 아들이나, 노년에도 독서에 힘써 학문이 깊고 경서에 밝았으며 언행이 바르고 곧아 아비의 허물을 덮기에 충분하였다. 이리하여 이이, 성혼도 모두 존경하는 친구로 여겨... (이하 생략)... 이이가 <a href="/wiki/%EC%84%9C%EC%96%BC" title="서얼">서얼</a>의 임용을 허가하자고 주장한 의도는 다만 훌륭한 인물을 구하여 임금을 보필하자는 것일 뿐, 일개 송익필에게 사심을 둔 것은 아니었는데도 사람들은 대부분 이이의 과실로 돌린다.'는 평도 있다.<sup class="reference" id="cite_ref-14"><a href="#cite_note-14">[14]</a></sup></p>
<h3><span class="mw-headline" id=".EB.B6.80.EC.A0.95.EC.A0.81_.ED.8F.89.EA.B0.80">부정적 평가</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%86%A1%EC%9D%B5%ED%95%84&amp;action=edit&amp;section=22" title="부분 편집: 부정적 평가">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>아버지 송사련의 어머니와 외할머니가 안당 집안의 노비로서 안돈후의 첩이었기 때문에, 안처겸 형제의 대화를 엿듣고 고변한 송사련은 주종관계와 친척관계를 파탄시킨 인간이라는 지탄을 계속 받게 되었다. 이는 신분제도와 위계질서, 계급제, 가족제가 중요시되던 조선사회에서는 용납되기 힘든 일이었다. 이 일로 인해 송익필은 생전과 사후에도 아버지가 행한 밀고의 멍에를 평생 지고, 사후에도 노비 출신이 참람하게 예를 논한다며 인신공격을 당하였다.</p>
<p>그러나 당쟁의 기획자<sup class="reference" id="cite_ref-jo01_13-2"><a href="#cite_note-jo01-13">[13]</a></sup>, 선조 이후 펼쳐진 서인·노론 집권 플랜을 설계한 책략가라는 평이 있다.<sup class="reference" id="cite_ref-jo01_13-3"><a href="#cite_note-jo01-13">[13]</a></sup></p>
<p>아버지의 무고사건 때문에 노비로 전락해 평생 관직에 나가지 못한 불운한 인물<sup class="reference" id="cite_ref-jo01_13-4"><a href="#cite_note-jo01-13">[13]</a></sup>이라는 시각도 있다.</p>
<h2><span class="mw-headline" id=".EC.82.AC.EC.83.81.EA.B3.BC_.EC.8B.A0.EB.85.90">사상과 신념</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%86%A1%EC%9D%B5%ED%95%84&amp;action=edit&amp;section=23" title="부분 편집: 사상과 신념">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<h3><span class="mw-headline" id=".EA.B5.90.EC.9C.A1.EA.B4.80">교육관</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%86%A1%EC%9D%B5%ED%95%84&amp;action=edit&amp;section=24" title="부분 편집: 교육관">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>그는 유생들과 문인들에게 처음부터 정답을 가르쳐주지 않고 스스로 생각하고 깨닭을 것을 강조하였다.</p>
<p>송익필은 그에게 배우고자 찾아온 유생들에게 자상하게 설명해 주는 법이 없었다. 스스로 정신을 집중하여 다시 읽고 생각해 볼 수밖에 없게 만드는 것이다. 지식을 습득하는데 있어 사색을 중요시 하되 연구자 스스로 노력하는 것을 강조한 때문이었다.<sup class="reference" id="cite_ref-dangjs_1-5"><a href="#cite_note-dangjs-1">[1]</a></sup> 송익필은 선생에게만 의지하는 주입식 교육이 자칫 학생들의 창의성을 저해할 수 있는 폐단을 생각했던 것이다.<sup class="reference" id="cite_ref-dangjs_1-6"><a href="#cite_note-dangjs-1">[1]</a></sup></p>
<p>사물을 연구할 때도 무엇이 옳고 잘못되었나, 왜 옳고 옳지 않은가를 스스로 생각하게 했다. <a href="/wiki/%EA%B3%B5%EC%9E%90" title="공자">공자</a>와 <a href="/wiki/%EB%A7%B9%EC%9E%90" title="맹자">맹자</a>, <a class="new" href="/w/index.php?title=%EC%82%AC%EC%84%9C%EC%9C%A1%EA%B2%BD&amp;action=edit&amp;redlink=1" title="사서육경 (없는 문서)">사서육경</a>, <a class="mw-disambig" href="/wiki/%EC%A3%BC%EC%9E%90" title="주자">주자</a>, <a class="new" href="/w/index.php?title=%ED%83%9C%EA%B7%B9%EC%84%A4&amp;action=edit&amp;redlink=1" title="태극설 (없는 문서)">태극설</a> 등을 강의함에 있어서도 무조건 수용옳다고 하기 전에 왜 옳은가를 유생, 문인들 스스로 생각하게 했다.</p>
<h3><span class="mw-headline" id=".EC.A7.81_.EC.B2.A0.ED.95.99">직 철학</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%86%A1%EC%9D%B5%ED%95%84&amp;action=edit&amp;section=25" title="부분 편집: 직 철학">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>인간본성인 선(善)을 회복하는 실천 방법으로 예절 외에 거짓됨이 없는 바른 행동, 직(直)에 대한 공부를 강조하였다. 그는 평생 직(直)을 생활철학으로 삼아 직심(直心), 직언(直言), 직행(直行)의 3직 실천에 힘썼다. 그리고 제자들에게도 학문도 중요하나 정직함 역시 중요하다고 역설하였다. 높은 학문의 경지와 뛰어난 재주 못지않게 직(直)의 실천으로 행실도 바르고 곧았기에, 자신의 출세를 위해 주종관계와 친척관계의 파탄을 저지른 부친과 신분적 한계를 가지고도 당대 사림(士林)의 존경을 받을 수 있었다.</p>
<h2><span class="mw-headline" id=".EB.85.BC.EB.9E.80.EA.B3.BC_.EC.9D.98.ED.98.B9">논란과 의혹</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%86%A1%EC%9D%B5%ED%95%84&amp;action=edit&amp;section=26" title="부분 편집: 논란과 의혹">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<h3><span class="mw-headline" id=".EA.B8.B0.EC.B6.95.EC.98.A5.EC.82.AC_.EB.B0.B0.ED.9B.84_.EC.9D.98.ED.98.B9">기축옥사 배후 의혹</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%86%A1%EC%9D%B5%ED%95%84&amp;action=edit&amp;section=27" title="부분 편집: 기축옥사 배후 의혹">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>그는 정철과 함께 기축옥사에 개입했는데, 정철의 배후에서 실질적으로 기축옥사를 조작한 이로 지목되기도 한다. 서자의 후손인 송익필은 문장력이 뛰어나 서인의 참모 격으로 활약했는데, 자신과 그의 가족 70여 인을 환천(還賤)시키고자 한 동인의 이발, 백유양 등에게 복수하기 위해 이 사건을 조작했다는 것이다.</p>
<h3><span class="mw-headline" id=".EA.B9.80.EC.9D.80.ED.9C.98.EC.9D.98_.EB.B0.B0.EB.A0.A4">김은휘의 배려</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%86%A1%EC%9D%B5%ED%95%84&amp;action=edit&amp;section=28" title="부분 편집: 김은휘의 배려">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>그러나 안당의 증손부의 상소 외에도 그의 아버지인 송사련이 역모를 날조하였으므로, 이때문에 친척들이 환천되는 등 세론의 증오를 받아 멸문의 궁지에 몰리자 10년 동안이나 먹여 살렸다. 김은휘는 그의 제자인 김장생의 숙부이자, 다른 제자인 김집, 김반의 종조부였다.</p>

<p><a href="/wiki/%EC%A0%84%EB%9D%BC%EB%B6%81%EB%8F%84" title="전라북도">전라북도</a> <a href="/wiki/%EC%A7%84%EC%95%88%EA%B5%B0" title="진안군">진안군</a>의 <a href="/wiki/%EC%9A%B4%EC%9E%A5%EC%82%B0" title="운장산">운장산</a>은 원래 운장산(雲藏山)이었으나 그가 임진왜란 전후에 은거했다 하여 그의 호를 따서 운장산(雲長山)이라 부르게 되었다.<sup class="reference" id="cite_ref-woonj_9-1"><a href="#cite_note-woonj-9">[9]</a></sup></p>
<p>동인이던 <a href="/wiki/%ED%99%8D%EA%B0%80%EC%8B%A0" title="홍가신">홍가신</a>은 개인적으로 송익필과 친분이 있어 늘 따르고 존경하였다. 홍가신의 동생 홍경신은 형이 미거한 신분과 친교를 맺는걸 못마땅하게 여기고 있었다. 하루는 홍경신이 "형님께서는 어째서 송익필과 친하게 지내십니까? 반드시 송가 녀석을 만나면 모욕을 주겠습니다" 라고 하자, 홍가신은 껄껄 웃으며 '과연 너가 그렇게 할 수 있겠느냐. 결코 못할 것이다.'라고 답하였다. 마침내 어느 날 송익필이 홍가신의 집을 방문할때 잔뜩 벼르고있던 홍경신은 송익필을 마주하자 그의 안광이 벼락치는듯하고 선풍도골의 풍채가 위압적인지라 자기도 모르게 마당에 내려가 주저앉고 맞으며 말하기를, “내가 절한 것이 아니라 무릎이 스스로 굽혀진 것입니다.”라고 둘러대었다.</p>
<p><a class="mw-redirect" href="/wiki/%EC%9C%A8%EA%B3%A1" title="율곡">율곡</a>(栗谷)의 소개로 가르침을 구하려 송익필을 찾은 <a href="/wiki/%EC%9D%B4%EC%88%9C%EC%8B%A0" title="이순신">이순신</a>(李舜臣)은 병풍 속 앉은 학이 자신이 상상하던 거북선과 비슷하여 몇 개 구멍을 뚫었다. 돌아온 송익필이 몇 개 구멍을 뚫었느냐고 묻자 네 개라 했다. 이순신이 묻기를, "원래 완전 하려면 구멍이 몇 개입니까"냐고 묻자 송익필은 48구멍이 전부라고 답했다. 용병(用兵)과 진법(陣法)등에 관해서 또한 둘이 논하니 밤이 새는줄 몰랐다. 그후 송익필 선생의 제자로 입문하여 훗날의 국난에 대비하는 많은 수련을 쌓았는데 충무공에게 내린 시가 전해져 온다.</p>
<p>月黑雁飛高 單于夜遁逃 달 밝은 밤 기러기 높이 나니 선우는 밤에 도망치리라 (밤 깊을 때에 도주하는 왜적을 섬멸할것을 암시)</p>
<p>毒龍潛處水偏淸 伐木丁丁山更幽 독룡 숨은 곳의 물은 편벽되게 맑고 산에서 나무하는 소리는 정정 울리니 산이 다시 그윽하도다 (수로에 수중철색을 설치하여 적을 붙잡아두고 가로막아 쳐부술것이며 현대의 횡렬진에 해당하는 일자진(一字陣)을 해안과 산에서 협공할것이다)</p>
