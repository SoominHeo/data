<p><b>송정역</b>(松亭驛)은 <a href="/wiki/%EB%B6%80%EC%82%B0%EA%B4%91%EC%97%AD%EC%8B%9C" title="부산광역시">부산광역시</a> <a href="/wiki/%ED%95%B4%EC%9A%B4%EB%8C%80%EA%B5%AC" title="해운대구">해운대구</a> <a href="/wiki/%EC%86%A1%EC%A0%95%EB%8F%99_(%ED%95%B4%EC%9A%B4%EB%8C%80%EA%B5%AC)" title="송정동 (해운대구)">송정동</a>에 있는 <a href="/wiki/%EB%8F%99%ED%95%B4%EC%84%A0" title="동해선">동해선</a>의 <a href="/wiki/%EC%A0%84%EC%B2%A0%EC%97%AD" title="전철역">전철역</a>이다. <a href="/wiki/2013%EB%85%84" title="2013년">2013년</a> <a href="/wiki/12%EC%9B%94_2%EC%9D%BC" title="12월 2일">12월 2일</a> 현 역사로 이전하였으며, 이 역에 정차한 <a href="/wiki/%EB%AC%B4%EA%B6%81%ED%99%94%ED%98%B8" title="무궁화호">무궁화호</a> 열차는 <a href="/wiki/%EA%B4%91%EC%97%AD%EC%B2%A0%EB%8F%84" title="광역철도">광역철도</a> 고상홈 옆에 설치된 임시 저상홈에 정차하였었다. <a href="/wiki/2016%EB%85%84" title="2016년">2016년</a> <a href="/wiki/12%EC%9B%94_9%EC%9D%BC" title="12월 9일">12월 9일</a> 시간표 개정과 함께 모든 <a href="/wiki/%EB%AC%B4%EA%B6%81%ED%99%94%ED%98%B8" title="무궁화호">무궁화호</a> 무정차 통과하였다가, <a href="/wiki/12%EC%9B%94_30%EC%9D%BC" title="12월 30일">12월 30일</a>부터 <a class="mw-redirect" href="/wiki/%EB%8F%99%ED%95%B4%EC%84%A0_(%EA%B4%91%EC%97%AD%EC%B2%A0%EB%8F%84)" title="동해선 (광역철도)">동해선 광역철도</a> 개통과 함께 영업을 재개하여 <a href="/wiki/%EA%B4%91%EC%97%AD%EC%B2%A0%EB%8F%84" title="광역철도">광역철도</a>가 운행된다.</p>

<h2><span class="mw-headline" id=".EA.B5.AC_.EC.97.AD.EC.82.AC">구 역사</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%86%A1%EC%A0%95%EC%97%AD_(%EB%B6%80%EC%82%B0)&amp;action=edit&amp;section=6" title="부분 편집: 구 역사">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>구 역사는 건축 당시 원형을 유지하고 있고, <a href="/wiki/1940%EB%85%84%EB%8C%80" title="1940년대">1940년대</a>의 전형적인 역사건축의 형태를 보이고 있어 건축사적 가치를 인정받았다. 역 전체가 <a href="/wiki/%EB%8C%80%ED%95%9C%EB%AF%BC%EA%B5%AD%EC%9D%98_%EB%93%B1%EB%A1%9D%EB%AC%B8%ED%99%94%EC%9E%AC" title="대한민국의 등록문화재">등록문화재</a> 제302호로 지정되어 <a class="mw-redirect" href="/wiki/%EB%AC%B8%ED%99%94%EC%9E%AC%EC%B2%AD" title="문화재청">문화재청</a>이 관리하고 있다.</p>
