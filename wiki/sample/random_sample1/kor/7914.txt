<div style="font-size: 9pt">쇠데르텔리에의 위치</div>
</div>
<p><b>쇠데르텔리에</b>(<span style="font-size: smaller;"><a href="/wiki/%EC%8A%A4%EC%9B%A8%EB%8D%B4%EC%96%B4" title="스웨덴어">스웨덴어</a>:</span> <span lang="sv" xml:lang="sv">Södertälje</span>)는 <a href="/wiki/%EC%8A%A4%EC%9B%A8%EB%8D%B4" title="스웨덴">스웨덴</a> <a href="/wiki/%EC%8A%A4%ED%86%A1%ED%99%80%EB%A6%84_%EC%A3%BC" title="스톡홀름 주">스톡홀름 주</a>에 위치한 도시로, 면적은 25.84㎢, 인구는 64,619명(2010년 기준), 인구 밀도는 2,500명/㎢이며 <a href="/wiki/%EC%8A%A4%ED%86%A1%ED%99%80%EB%A6%84" title="스톡홀름">스톡홀름</a>에서 남쪽으로 약 30km 정도 떨어진 곳에 위치한 공업 도시이다.</p>
<p>전체 인구 가운데 약 39%는 외국인 이민자이며 매년 1.5% 정도 증가하는 추세이다. 이 가운데 <a href="/wiki/%EC%95%84%EC%8B%9C%EB%A6%AC%EC%95%84%EC%9D%B8" title="아시리아인">아시리아인</a> 인구가 가장 많으며 아시리아인의 교회, 학교, 상점 등이 들어서 있다. <a href="/wiki/%EC%8A%A4%EC%9B%A8%EB%8D%B4%EC%96%B4" title="스웨덴어">스웨덴어</a> 외에 <a href="/wiki/%EC%95%84%EB%9E%8C%EC%96%B4" title="아람어">아람어</a>와 <a href="/wiki/%EC%95%84%EB%9E%8D%EC%96%B4" title="아랍어">아랍어</a>가 널리 쓰이며 <a href="/wiki/%ED%95%80%EB%9E%80%EB%93%9C%EC%96%B4" title="핀란드어">핀란드어</a>와 <a href="/wiki/%EC%84%B8%EB%A5%B4%EB%B9%84%EC%95%84%EC%96%B4" title="세르비아어">세르비아어</a>를 제2언어로 쓰는 주민들도 거주한다. <a href="/wiki/%EB%B0%9C%ED%8A%B8_3%EA%B5%AD" title="발트 3국">발트 3국</a>과 <a href="/wiki/%EC%BA%85%EC%B9%B4%EC%8A%A4" title="캅카스">캅카스</a> 국가를 비롯해서 <a href="/wiki/%EB%AA%A8%EB%A1%9C%EC%BD%94" title="모로코">모로코</a> 등 <a href="/wiki/%EB%B6%81%EC%95%84%ED%94%84%EB%A6%AC%EC%B9%B4" title="북아프리카">북아프리카</a> 국가, <a href="/wiki/%EC%9D%B8%EB%8F%84" title="인도">인도</a>와 <a href="/wiki/%ED%8C%8C%ED%82%A4%EC%8A%A4%ED%83%84" title="파키스탄">파키스탄</a> 등 남아시아 국가, <a href="/wiki/%ED%95%84%EB%A6%AC%ED%95%80" title="필리핀">필리핀</a>과 <a href="/wiki/%EB%8C%80%ED%95%9C%EB%AF%BC%EA%B5%AD" title="대한민국">대한민국</a> 등 동아시아 국가, <a href="/wiki/%EB%82%A8%EC%95%84%EB%A9%94%EB%A6%AC%EC%B9%B4" title="남아메리카">남아메리카</a> 국가 출신 이민자들도 거주한다.</p>
