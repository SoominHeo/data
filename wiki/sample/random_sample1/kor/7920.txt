<p><b>쇼리키 마쓰타로</b>(<span style="font-size: smaller;"><a href="/wiki/%EC%9D%BC%EB%B3%B8%EC%96%B4" title="일본어">일본어</a>:</span> <span lang="ja"><span lang="ja"><ruby><rb>正力松太郎</rb> <rp>(</rp><rt>しょうりき まつたろう</rt><rp>)</rp></ruby></span></span>, <a href="/wiki/1885%EB%85%84" title="1885년">1885년</a> <a href="/wiki/4%EC%9B%94_11%EC%9D%BC" title="4월 11일">4월 11일</a> ~ <a href="/wiki/1969%EB%85%84" title="1969년">1969년</a> <a href="/wiki/10%EC%9B%94_9%EC%9D%BC" title="10월 9일">10월 9일</a>)는 <a href="/wiki/%EC%9D%BC%EB%B3%B8" title="일본">일본</a>의 <a href="/wiki/%EA%B2%BD%EC%B0%B0" title="경찰">경찰</a> 관료, <a href="/wiki/%EC%82%AC%EC%97%85%EA%B0%80" title="사업가">사업가</a>, <a class="mw-redirect" href="/wiki/%EC%A0%95%EC%B9%98%EC%9D%B8" title="정치인">정치인</a>이다. 원래 <a class="new" href="/w/index.php?title=%EC%9A%94%EB%AF%B8%EC%9A%B0%EB%A6%AC_%EC%8B%A0%EB%AC%B8%EC%82%AC&amp;action=edit&amp;redlink=1" title="요미우리 신문사 (없는 문서)">요미우리 신문사</a>의 사주이며, 위계는 지배 2위이다. <a href="/wiki/%EB%8F%84%EC%95%BC%EB%A7%88_%ED%98%84" title="도야마 현">도야마 현</a> <a href="/wiki/%EB%8B%A4%EC%B9%B4%EC%98%A4%EC%B9%B4_%EC%8B%9C" title="다카오카 시">다카오카 시</a> 명예 시민이며, 게이세이 전철 OB이다.</p>
