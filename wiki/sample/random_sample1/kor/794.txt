<p><b>BABYMETAL</b>(베비메탈)은, <a href="/wiki/%EC%9D%BC%EB%B3%B8" title="일본">일본</a>의 <a href="/wiki/%EC%97%AC%EC%84%B1" title="여성">여성</a> <a class="mw-disambig" href="/wiki/%EC%95%84%EC%9D%B4%EB%8F%8C" title="아이돌">아이돌</a> 그룹이다. 소속사는 <a href="/wiki/%EC%95%84%EB%AE%A4%EC%A6%88" title="아뮤즈">아뮤즈</a>이고, 레이블은 <a href="/wiki/%ED%86%A0%EC%9D%B4%EC%8A%A4_%ED%8C%A9%ED%86%A0%EB%A6%AC" title="토이스 팩토리">토이스 팩토리</a>이다.</p>

<table class="notice section" style="text-align: left; margin:0 0 0.5em 0.5em; clear:none; font-size:smaller;">
<tr>
<td width="auto"><img alt="" data-file-height="400" data-file-width="400" height="25" src="//upload.wikimedia.org/wikipedia/commons/thumb/2/2c/Broom_icon.svg/25px-Broom_icon.svg.png" srcset="//upload.wikimedia.org/wikipedia/commons/thumb/2/2c/Broom_icon.svg/38px-Broom_icon.svg.png 1.5x, //upload.wikimedia.org/wikipedia/commons/thumb/2/2c/Broom_icon.svg/50px-Broom_icon.svg.png 2x" width="25"/> 이 문단은 <b><a href="/wiki/%EC%9C%84%ED%82%A4%EB%B0%B1%EA%B3%BC:%ED%8E%B8%EC%A7%91_%EC%A7%80%EC%B9%A8" title="위키백과:편집 지침">위키백과의 편집 지침</a></b>에 맞춰 다듬어야 합니다. 더 좋은 문단이 되도록 <span class="plainlinks"><a class="external text" href="//ko.wikipedia.org/w/index.php?title=BABYMETAL&amp;action=edit">문단 수정을 도와주세요</a></span>. 내용에 대한 의견이 있으시다면 <a href="/wiki/%ED%86%A0%EB%A1%A0:BABYMETAL#.23" title="토론:BABYMETAL">토론 문서</a>에서 나누어 주세요. <p><a href="/wiki/2010%EB%85%84" title="2010년">2010년</a> <a href="/wiki/11%EC%9B%94" title="11월">11월</a>, '<a class="mw-disambig" href="/wiki/%EC%95%84%EC%9D%B4%EB%8F%8C" title="아이돌">아이돌</a>과 <a href="/wiki/%ED%97%A4%EB%B9%84_%EB%A9%94%ED%83%88" title="헤비 메탈">헤비 메탈</a>의 융합'을 테마로 결성하였다. 여성 아이돌 그룹 <a href="/wiki/%EC%82%AC%EC%BF%A0%EB%9D%BC_%ED%95%99%EC%9B%90" title="사쿠라 학원">사쿠라 학원</a>에서 파생한 클럽 활동 유닛 중 하나인 중음부로서 시작하였다<sup class="reference" id="cite_ref-shikakenin_1-0"><a href="#cite_note-shikakenin-1">[1]</a></sup>. 목표는 세계 정복으로 하고 있다<sup class="reference" id="cite_ref-2"><a href="#cite_note-2">[2]</a></sup>.</p>
<p><a href="/wiki/2011%EB%85%84" title="2011년">2011년</a> <a href="/wiki/10%EC%9B%94" title="10월">10월</a>, 인디 데뷔곡인 〈두·근·두·근☆모닝〉이 <a href="/wiki/%EC%9C%A0%ED%8A%9C%EB%B8%8C" title="유튜브">유튜브</a>를 통해서 공개되었으며 일본 외에 해외에서도 주목을 모았다<sup class="reference" id="cite_ref-3"><a href="#cite_note-3">[3]</a></sup>.</p>
<p><a href="/wiki/2012%EB%85%84" title="2012년">2012년</a> <a href="/wiki/8%EC%9B%94" title="8월">8월</a>, 《<a href="/wiki/%EC%84%9C%EB%A8%B8_%EC%86%8C%EB%8B%89" title="서머 소닉">서머 소닉</a> 2012》 2일째 도쿄 회장에 서머 소닉 사상 최연소로 출연하였다<sup class="reference" id="cite_ref-4"><a href="#cite_note-4">[4]</a></sup>. 또한 11월에는 첫 해외 공연인 《<a class="new" href="/w/index.php?title=Anime_Festival_Asia&amp;action=edit&amp;redlink=1" title="Anime Festival Asia (없는 문서)">Anime Festival Asia</a> 2012》에 출연하였다<sup class="reference" id="cite_ref-5"><a href="#cite_note-5">[5]</a></sup>.</p>
<p><a href="/wiki/2013%EB%85%84" title="2013년">2013년</a> <a href="/wiki/1%EC%9B%94" title="1월">1월</a>, 〈괴롭힘, 안 돼, 절대〉로 메이저 무대에 데뷔했다<sup class="reference" id="cite_ref-6"><a href="#cite_note-6">[6]</a></sup>.</p>
<p>2013년 <a href="/wiki/2%EC%9B%94" title="2월">2월</a>, 라이브 공연 중에 <a class="new" href="/w/index.php?title=%EB%82%98%EC%B9%B4%EB%AA%A8%ED%86%A0_%EC%8A%A4%EC%A6%88%EC%B9%B4&amp;action=edit&amp;redlink=1" title="나카모토 스즈카 (없는 문서)">나카모토 스즈카</a>(SU-METAL)가 사쿠라 학원을 졸업한 후에도 BABYMETAL은 해산하지 않고 기한부로 활동을 계속함을 발표하였다<sup class="reference" id="cite_ref-7"><a href="#cite_note-7">[7]</a></sup>. 거기에 4월부터 독립하여 미즈노 유이(YUIMETAL)와 키쿠치 모아(MOAMETAL)는 사쿠라 학원과 병행해 과외 활동으로서 참가하게 되었다<sup class="reference" id="cite_ref-8"><a href="#cite_note-8">[8]</a></sup>.</p>
<p>2013년 10월, 메탈 페스티벌 《<a class="new" href="/w/index.php?title=Loud_Park_Festival&amp;action=edit&amp;redlink=1" title="Loud Park Festival (없는 문서)">Loud Park Festival</a> 13》에 참가하였다.<sup class="reference" id="cite_ref-loudpark_9-0"><a href="#cite_note-loudpark-9">[9]</a></sup>. 이 공연은 메탈 팬으로부터 찬반양론이 일어났다<sup class="reference" id="cite_ref-loudpark_9-1"><a href="#cite_note-loudpark-9">[9]</a></sup>.</p>
<p>2013년 <a href="/wiki/12%EC%9B%94" title="12월">12월</a>, <a href="/wiki/%EC%8B%B1%EA%B0%80%ED%8F%AC%EB%A5%B4" title="싱가포르">싱가포르</a>에서 공연을 해 해외에서 처음으로 단독 공연을 실시했다<sup class="reference" id="cite_ref-10"><a href="#cite_note-10">[10]</a></sup>.</p>
<p><a href="/wiki/2014%EB%85%84" title="2014년">2014년</a> 2월, 1st 앨범 《BABYMETAL》을 발매<sup class="reference" id="cite_ref-11"><a href="#cite_note-11">[11]</a></sup>. <a href="/wiki/%EC%95%84%EC%9D%B4%ED%8A%A0%EC%8A%A4_%EC%8A%A4%ED%86%A0%EC%96%B4" title="아이튠스 스토어">iTunes Store</a>에서는 7개국의 록 앨범 차트에서 베스트 10에 들어갔다<sup class="reference" id="cite_ref-billboard_12-0"><a href="#cite_note-billboard-12">[12]</a></sup>. 그리고 미국의 <a href="/wiki/%EB%B9%8C%EB%B3%B4%EB%93%9C" title="빌보드">빌보드</a> 차트에 들어간 가장 어린 일본인이 되었다 <sup class="reference" id="cite_ref-billboard_12-1"><a href="#cite_note-billboard-12">[12]</a></sup>.</p>
<p>2014년 <a href="/wiki/3%EC%9B%94" title="3월">3월</a>, 2일간에 걸쳐서 <a href="/wiki/%EC%9D%BC%EB%B3%B8%EB%AC%B4%EB%8F%84%EA%B4%80" title="일본무도관">일본무도관</a>에서 단독 공연을 실시했다<sup class="reference" id="cite_ref-budoukan_13-0"><a href="#cite_note-budoukan-13">[13]</a></sup>. 이 때 멤버들의 평균 연령이 14.7세 였기에 지금까지의 여성 최연소 기록을 갱신했다<sup class="reference" id="cite_ref-budoukan_13-1"><a href="#cite_note-budoukan-13">[13]</a></sup>. 또한 이 공연을 끝으로 일본에서의 활동이 일정 기간 중단 상태가 되었다<sup class="reference" id="cite_ref-14"><a href="#cite_note-14">[14]</a></sup>.</p>
<h2><span class="mw-headline" id=".EC.9D.B4.EB.A6.84.EC.9D.98_.EC.9C.A0.EB.9E.98">이름의 유래</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=BABYMETAL&amp;action=edit&amp;section=2" title="부분 편집: 이름의 유래">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>그룹명은 프로듀서·KOBAMETAL이 “헤비 메탈”에서 따와서 지은 것이다. <b>BABY</b>에서는 귀여움을, <b>METAL</b>에서는 격렬함을 표현할 수 있는 것을 뜻한다. 또한 〈새로운 메탈이 탄생한다〉라는 소망의 의미도 담겨 있다고 말했다<sup class="reference" id="cite_ref-shikakenin_1-1"><a href="#cite_note-shikakenin-1">[1]</a></sup>.</p>
