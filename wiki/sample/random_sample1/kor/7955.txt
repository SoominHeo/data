<p><b>수메르어</b>(<span style="font-size: smaller;"><strong class="selflink">수메르어</strong>:</span> <span lang="sux" xml:lang="sux">𒅴𒂠</span>)는 대략 기원전 4000년 경부터 고대 <a href="/wiki/%EC%88%98%EB%A9%94%EB%A5%B4" title="수메르">수메르</a>에서 사용된 <a href="/wiki/%EC%96%B8%EC%96%B4" title="언어">언어</a>이다. 수메르는 오늘날 이라크가 있는 메소포타미아 남쪽에 있었다. 점토판에 새겨진 <a class="mw-redirect" href="/wiki/%EC%90%90%EA%B8%B0%EB%AC%B8%EC%9E%90" title="쐐기문자">쐐기문자</a>(설형문자)를 해독함으로써 널리 알려지게 되었다. 수메르어는 <a href="/wiki/%EC%9D%B4%EC%A7%91%ED%8A%B8%EC%96%B4" title="이집트어">이집트어</a>와 함께 인류 역사상 문자로 기록된 가장 오래된 언어에 속한다.</p>
<p>기원전 3000년대에 수메르어는 <a href="/wiki/%EC%85%88%EC%96%B4%ED%8C%8C" title="셈어파">셈어파</a>의 <a href="/wiki/%EC%95%84%EC%B9%B4%EB%93%9C%EC%96%B4" title="아카드어">아카드어</a>와 혼용되었으며 이로 인해 아카드어는 수메르어에서 많은 영향을 받았다.<sup class="reference" id="cite_ref-Deutscher_1-0"><a href="#cite_note-Deutscher-1">[1]</a></sup> 기원전 2000년대 말기에 <a href="/wiki/%EC%85%88%EC%A1%B1" title="셈족">셈족</a>인 <a class="mw-redirect" href="/wiki/%EC%95%84%EC%B9%B4%EB%93%9C%EC%9D%B8" title="아카드인">아카드인</a>들이 <a href="/wiki/%EC%88%98%EB%A9%94%EB%A5%B4" title="수메르">수메르</a>를 정복하였으나, 아카드인들은 오히려 수메르의 문자 체계를 배우고 기술과 문학을 받아들였다. 이로 인해 수메르어는 수메르가 멸망한 뒤에도 오랫동안 문학, 의식, 기술에서 사용되었다.<sup class="reference" id="cite_ref-woods_2-0"><a href="#cite_note-woods-2">[2]</a></sup></p>
<p>수메르어는 한국어, 일본어, 몽골어, 터키어 등과 같은 <a href="/wiki/%EA%B5%90%EC%B0%A9%EC%96%B4" title="교착어">교착어</a>에 속한다. 수메르어는 <a class="mw-redirect" href="/wiki/%EC%9D%B8%EB%8F%84%EC%9C%A0%EB%9F%BD%EC%96%B4" title="인도유럽어">인도유럽어</a>나 <a class="mw-redirect" href="/wiki/%EC%85%88%EC%96%B4" title="셈어">셈어</a>와 같이 어형 변화는 없고, <a href="/wiki/%EC%95%8C%ED%83%80%EC%9D%B4%EC%96%B4" title="알타이어">알타이어</a>와는 닮은 단어가 있으며 그 특징인 <a class="mw-redirect" href="/wiki/%EB%AA%A8%EC%9D%8C%EC%A1%B0%ED%99%94" title="모음조화">모음조화</a>의 경향도 있다.<sup class="reference" id="cite_ref-3"><a href="#cite_note-3">[3]</a></sup> 수메르어는 다른 <a href="/wiki/%EC%96%B4%EC%A1%B1" title="어족">어족</a>과 연관성을 찾을 수 없는 <a href="/wiki/%EA%B3%A0%EB%A6%BD%EC%96%B4_(%EB%B9%84%EA%B5%90%EC%96%B8%EC%96%B4%ED%95%99)" title="고립어 (비교언어학)">고립어</a>이다.<sup class="reference" id="cite_ref-britanica_4-0"><a href="#cite_note-britanica-4">[4]</a></sup><sup class="reference" id="cite_ref-5"><a href="#cite_note-5">[5]</a></sup></p>

<h2><span class="mw-headline" id=".ED.8A.B9.EC.A7.95">특징</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%88%98%EB%A9%94%EB%A5%B4%EC%96%B4&amp;action=edit&amp;section=1" title="부분 편집: 특징">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>수메르어는 <a href="/wiki/%EA%B5%90%EC%B0%A9%EC%96%B4" title="교착어">교착어</a>로서 <a href="/wiki/%ED%98%95%ED%83%9C%EC%86%8C" title="형태소">형태소</a>에 <a href="/wiki/%EC%A0%91%EC%82%AC" title="접사">접사</a>가 연결되는 구조를 갖는다. 수메르어는 <a href="/wiki/%EB%8A%A5%EA%B2%A9#.EB.B6.84.EC.97.B4.EB.8A.A5.EA.B2.A9" title="능격">분열능격</a>을 갖는 언어이자 1인칭이나 2인칭 대명사를 사용할 경우 주어와 술어의 <a class="new" href="/w/index.php?title=%EC%83%81_(%EB%AC%B8%EB%B2%95)&amp;action=edit&amp;redlink=1" title="상 (문법) (없는 문서)">상</a>을 일치시키는 <a class="new" href="/w/index.php?title=%EC%A3%BC%EA%B2%A9-%EB%8C%80%EA%B2%A9_%EC%96%B8%EC%96%B4&amp;action=edit&amp;redlink=1" title="주격-대격 언어 (없는 문서)">주격-대격 언어</a>이다. 그러나 직설화법을 사용할 경우에는 <a href="/wiki/%EB%8A%A5%EA%B2%A9-%EC%A0%88%EB%8C%80%EA%B2%A9_%EC%96%B8%EC%96%B4" title="능격-절대격 언어">능격-절대격 언어</a>의 특징을 나타내기도 한다. 이러한 언어학적 특징은 분열 능격을 갖는 대부분의 언어에서도 나타난다. 수메르어의 경우 능격은 “-e”를 접미사로 갖는 경우와 접미사 없이 절대격으로 나타나는 경우가 있다.<sup class="reference" id="cite_ref-6"><a href="#cite_note-6">[주해 1]</a></sup> 수메르어에 나타난 능격의 예로는 <i>lugal-e e<sub>2</sub> mu-un-du<sub>3</sub></i>(왕이 집을 지었다)와 같은 예문을 들 수 있다.</p>
<p>수메르어는 생물과 관련한 낱말에 대해 <a href="/wiki/%EC%84%B1_(%EB%AC%B8%EB%B2%95)" title="성 (문법)">남성과 여성</a>을 구분하여 표기하였으나 발음할 때는 구분하지 않았다. 수메르어에는 과거 시제와 현재-미래 시제의 두 가지 시제가 있으며, 완료형과 미완료형을 표시하기 위해 <a class="new" href="/w/index.php?title=%EC%83%81_(%EB%AC%B8%EB%B2%95)&amp;action=edit&amp;redlink=1" title="상 (문법) (없는 문서)">상</a>을 구별한다. 또한 수메르어는 주격, 장소격, 비교격, 동격 등에 따른 <a class="new" href="/w/index.php?title=%EA%B2%A9%EB%B3%80%ED%99%94&amp;action=edit&amp;redlink=1" title="격변화 (없는 문서)">격변화</a>가 있다.</p>
<p>수메르어의 또 다른 특징은 수 많은 <a href="/wiki/%EB%8F%99%EC%9D%8C%EC%9D%B4%EC%9E%90" title="동음이자">동음이자</a>가 존재한다는 것이다. 예를 들어 du는 글자에 따라 du = "가다"를 뜻하거나, du<sub>3</sub> = dù = "(집을) 짓다"를 뜻할 수 있다. 이 때문에 수메르어를 연구하는 학계에서는 발음으로 수메르어를 표기할 때에는 번호를 부여하여 이를 구분하는 관례가 수립되었다.<sup class="reference" id="cite_ref-7"><a href="#cite_note-7">[6]</a></sup></p>

<h3><span class="mw-headline" id=".EB.B0.A9.EC.96.B8">방언</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%88%98%EB%A9%94%EB%A5%B4%EC%96%B4&amp;action=edit&amp;section=4" title="부분 편집: 방언">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>수메르어에는 에메-응이르(eme-ĝir)와 에메-살(eme-sal)이라 불린 방언이 있었다. 에메-응이르는 표준어를 뜻하고 에메-살은 여자들이 쓰는 말이란 뜻이다.<sup class="reference" id="cite_ref-9"><a href="#cite_note-9">[8]</a></sup></p>
<h2><span class="mw-headline" id=".EB.AC.B8.EC.9E.90">문자</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%88%98%EB%A9%94%EB%A5%B4%EC%96%B4&amp;action=edit&amp;section=5" title="부분 편집: 문자">편집</a><span class="mw-editsection-bracket">]</span></span></h2>

<p><a class="mw-redirect" href="/wiki/%EC%88%98%EB%A9%94%EB%A5%B4%EC%9D%B8" title="수메르인">수메르인</a>들은 기원전 3000년대 말 경에 점토에 <a href="/wiki/%EB%AC%B8%EC%9E%90" title="문자">문자</a>를 새기기 시작하였다. 이 때에는 <a href="/wiki/%EC%83%81%ED%98%95%EB%AC%B8%EC%9E%90" title="상형문자">상형문자</a>에 가까웠으며 단순한 행정기록만을 남길 수 있었다. 이후 수 세기 동안 문자체계를 수정하고 형성하여 아주 양식화된 순수한 <a href="/wiki/%ED%91%9C%EC%9D%8C%EB%AC%B8%EC%9E%90" title="표음문자">표음문자</a> 체계로 발전하여 기원전 2500~2000년 사이에는 가장 복잡한 역사적, 문학적 작품도 어렵지 않게 표현할 수 있는 충분한 감각과 유연성을 갖춘 <a class="mw-redirect" href="/wiki/%EC%90%90%EA%B8%B0%EB%AC%B8%EC%9E%90" title="쐐기문자">쐐기문자</a>(설형(楔形)문자) 체계가 되었다. 기원전 2000년대가 끝나기 전에 수메르 인들이 점토판, 기둥, 원주 등에 당시까지 구전되어 내려오던 문학적 창작물들을 새겼다.<sup class="reference" id="cite_ref-10"><a href="#cite_note-10">[9]</a></sup></p>
<p>기원전 약 3000년 무렵까지 초기 수메르어 문자는 사물을 간략히 그려 넣은 그림 기호에서 출발하였다. 이후 이 그림 문자들이 보다 간략히 정리되면서 점토에 쐐기 모양의 철필을 이용하여 새긴 <a class="mw-redirect" href="/wiki/%EC%84%A4%ED%98%95%EB%AC%B8%EC%9E%90" title="설형문자">설형문자</a>로 발전하였다. 초기 설형문자는 여전히 하나의 문자가 하나의 의미를 뜻하는 <a href="/wiki/%EC%83%81%ED%98%95%EB%AC%B8%EC%9E%90" title="상형문자">상형문자</a>의 성격을 갖고 있었으나, 시간이 지남에 따라 음절의 음가만을 표기하는 음절문자로 발전하게 되었다.<sup class="reference" id="cite_ref-11"><a href="#cite_note-11">[10]</a></sup></p>
<p><a href="/wiki/%EC%9A%B0%EB%A5%B4_%EC%A0%9C3%EC%99%95%EC%A1%B0" title="우르 제3왕조">우르 제3왕조</a> 시기에 <a href="/wiki/%EC%95%84%EC%B9%B4%EB%93%9C%EC%96%B4" title="아카드어">아카드어</a>-<b>수메르어</b> 사전이 만들어졌다. 오늘날 수메르어 문자의 해석은 이를 바탕으로 한 것이다.</p>
<h2><span class="mw-headline" id=".ED.95.B4.EB.8F.85">해독</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%88%98%EB%A9%94%EB%A5%B4%EC%96%B4&amp;action=edit&amp;section=6" title="부분 편집: 해독">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<h3><span class="mw-headline" id=".EC.84.A4.ED.98.95.EB.AC.B8.EC.9E.90.EC.9D.98_.ED.95.B4.EB.8F.85">설형문자의 해독</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%88%98%EB%A9%94%EB%A5%B4%EC%96%B4&amp;action=edit&amp;section=7" title="부분 편집: 설형문자의 해독">편집</a><span class="mw-editsection-bracket">]</span></span></h3>

<p><a class="mw-redirect" href="/wiki/%EC%84%A4%ED%98%95%EB%AC%B8%EC%9E%90" title="설형문자">설형문자</a>를 해독하는데 결정적인 단서를 제공한 것은 <a href="/wiki/%EB%B2%A0%ED%9E%88%EC%8A%A4%ED%88%B0_%EB%B9%84%EB%AC%B8" title="베히스툰 비문">베히스툰 비문</a>이다. 베히스툰 비문은 <a href="/wiki/%EA%B3%A0%EB%8C%80_%ED%8E%98%EB%A5%B4%EC%8B%9C%EC%95%84%EC%96%B4" title="고대 페르시아어">고대 페르시아어</a>, <a href="/wiki/%EC%97%98%EB%9E%8C%EC%96%B4" title="엘람어">엘람어</a>, <a href="/wiki/%EC%95%84%EC%B9%B4%EB%93%9C%EC%96%B4" title="아카드어">아카드어</a>의 세 언어로 같은 내용을 새겨놓았다.</p>
<p>1802년 독일의 <a href="/wiki/%EA%B8%88%EC%84%9D%ED%95%99" title="금석학">금석학자</a>인 <a class="new" href="/w/index.php?title=%EA%B2%8C%EC%98%A4%EB%A5%B4%EA%B7%B8_%ED%94%84%EB%A6%AC%EB%93%9C%EB%A6%AC%ED%9E%88_%EA%B7%B8%EB%A1%9C%ED%85%8C%ED%8E%9C%ED%8A%B8&amp;action=edit&amp;redlink=1" title="게오르그 프리드리히 그로테펜트 (없는 문서)">게오르그 프리드리히 그로테펜트</a>가 설형문자의 해독을 시작하였다. 이후 1838년 영국의 <a class="new" href="/w/index.php?title=%ED%95%B8%EB%A6%AC_%EB%A1%9C%EC%9A%B0%EB%A6%B0%EC%8A%A8&amp;action=edit&amp;redlink=1" title="핸리 로우린슨 (없는 문서)">핸리 로우린슨</a>이 그로테펜트의 작업 내용과 현대 <a href="/wiki/%ED%8E%98%EB%A5%B4%EC%8B%9C%EC%95%84%EC%96%B4" title="페르시아어">페르시아어</a>에 대한 지식을 바탕으로 베히스툰 비문의 페르시아어 부분을 번역하기 시작하였다. 1843년 로우린슨은 비문의 페르시아어 부분에 대한 해독작업을 마쳤다. 이 과정에서 로우린슨과 다른 학자들은 고대 페르시아어를 통해 확인할 수 있는 아카드어 설형문자 37개를 확보하였다. 이후 <a href="/wiki/%EA%B3%A0%EA%B3%A0%ED%95%99" title="고고학">고고학</a> 발굴이 활기차게 진행됨에 따라 더 많은 설형문자 텍스트가 발견되었고, 이 설형문자들이 대부분 아카드어로 된 것임을 알게 되었다. 비문을 통해 해독의 단서를 찾은 금석학자들은 아카드어를 해독하였고 아카드어가 <a href="/wiki/%EC%85%88%EC%96%B4%ED%8C%8C" title="셈어파">셈어파</a>에 속한다는 것을 확인하였다. 한편, 1850년 아일랜드의 <a class="new" href="/w/index.php?title=%EC%97%90%EB%93%9C%EC%9B%8C%EB%93%9C_%ED%9E%9D%ED%81%AC%EC%8A%A4&amp;action=edit&amp;redlink=1" title="에드워드 힝크스 (없는 문서)">에드워드 힝크스</a>는 셈어파인 아카드어의 특징으로는 설명할 수 없는 어근을 사용하는 특이한 설형문자를 발견하였다. 그는 나중에 이 설형문자가 수메르어로 된 것이라는 것을 알게 되었다.<sup class="reference" id="cite_ref-12"><a href="#cite_note-12">[11]</a></sup><sup class="reference" id="cite_ref-13"><a href="#cite_note-13">[12]</a></sup></p>
<h3><span class="mw-headline" id=".EC.88.98.EB.A9.94.EB.A5.B4.EC.96.B4.EC.9D.98_.ED.95.B4.EB.8F.85">수메르어의 해독</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%88%98%EB%A9%94%EB%A5%B4%EC%96%B4&amp;action=edit&amp;section=8" title="부분 편집: 수메르어의 해독">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>1885년 로우린슨은 <a href="/wiki/%EB%B0%94%EB%B9%8C%EB%A1%9C%EB%8B%88%EC%95%84" title="바빌로니아">바빌로니아</a> 남부 지역인 <a href="/wiki/%EB%8B%88%ED%91%B8%EB%A5%B4" title="니푸르">니푸르</a>, <a href="/wiki/%EB%9D%BC%EB%A5%B4%EC%82%AC" title="라르사">라르사</a>, <a class="mw-redirect" href="/wiki/%EC%9A%B0%EB%A5%B4%ED%81%AC" title="우르크">우르크</a> 등지에서 발견된 셈어파가 아닌 설형문자에 대해 학계에 보고하였다. 이 설형문자에 대해 <a class="new" href="/w/index.php?title=%EC%A4%84%EB%A6%AC%EC%9A%B0%EC%8A%A4_%EC%98%A4%ED%8E%98%EB%A5%B4%ED%8A%B8&amp;action=edit&amp;redlink=1" title="줄리우스 오페르트 (없는 문서)">줄리우스 오페르트</a>와 같은 학자는 <a class="new" href="/w/index.php?title=%ED%88%AC%EB%A5%B4%EC%95%88&amp;action=edit&amp;redlink=1" title="투르안 (없는 문서)">투르안</a>의 것이라 추측하기도 하였다. 1856년 힝크스는 자신이 발견한 설형문자가 <a href="/wiki/%EA%B5%90%EC%B0%A9%EC%96%B4" title="교착어">교착어</a>의 성질을 보인다는 점을 들어 오페르트의 견해를 비판하였다. 1869년 오페르는 셈어파가 아닌 이 언어를 수메르어라고 하자고 제안하였는데, 이는 발견된 설형문자에 나오는 "아카드와 수메르의 왕"이라는 표현에서 아카드 왕국에 아카드어가 아닌 언어를 사용하는 <a href="/wiki/%EC%88%98%EB%A9%94%EB%A5%B4" title="수메르">수메르</a>라는 지역이 있었을 것이라는 추측에서 비롯된 것이었다. 수메르와 아카드어를 비교하는 최초의 과학적 연구 성과는 1879년 발간된 <a class="new" href="/w/index.php?title=%ED%8C%8C%EC%9A%B8_%ED%95%98%EC%9A%B0%ED%94%84%ED%8A%B8&amp;action=edit&amp;redlink=1" title="파울 하우프트 (없는 문서)">파울 하우프트</a>의 《수메르 가족법》(<span style="font-size: smaller;"><a href="/wiki/%EB%8F%85%EC%9D%BC%EC%96%B4" title="독일어">독일어</a>:</span> <span lang="de" xml:lang="de">Die sumerischen Familiengesetze</span>)이었다.<sup class="reference" id="cite_ref-14"><a href="#cite_note-14">[13]</a></sup></p>
<p>1877년 프랑스의 <a class="new" href="/w/index.php?title=%EC%97%90%EB%A5%B4%EB%84%A4%EC%8A%A4%ED%8A%B8_%EB%8D%B0_%EC%82%AC%EB%A5%B4%EC%A0%9C&amp;action=edit&amp;redlink=1" title="에르네스트 데 사르제 (없는 문서)">에르네스트 데 사르제</a>는 <a href="/wiki/%EB%9D%BC%EA%B0%80%EC%8B%9C" title="라가시">라가시</a>의 수도였던 <a href="/wiki/%EA%B8%B0%EB%A5%B4%EC%88%98" title="기르수">기르수</a>가 있었던 텔로에서 수메르 유적을 발굴하였다. 1884년 사르제는 자신이 발견한 수메르어 점토판을 번역하여 《칼데아에서의 발견》(<span style="font-size: smaller;"><a href="/wiki/%ED%94%84%EB%9E%91%EC%8A%A4%EC%96%B4" title="프랑스어">프랑스어</a>:</span> <span lang="fr" xml:lang="fr">Découvertes en Chaldée</span>)을 출간하였다.<sup class="reference" id="cite_ref-15"><a href="#cite_note-15">[14]</a></sup> 1888년에는 <a href="/wiki/%ED%8E%9C%EC%8B%A4%EB%B2%A0%EC%9D%B4%EB%8B%88%EC%95%84_%EB%8C%80%ED%95%99%EA%B5%90" title="펜실베이니아 대학교">펜실베이니아 대학교</a>에서 <a href="/wiki/%EB%8B%88%ED%91%B8%EB%A5%B4" title="니푸르">니푸르</a>를 발굴하였다.<sup class="reference" id="cite_ref-16"><a href="#cite_note-16">[15]</a></sup></p>
<p>1874년 프랑스에서 활동하던 터키 출신의 <a href="/wiki/%EB%8F%99%EC%96%91%ED%95%99" title="동양학">동양학자</a> <a class="new" href="/w/index.php?title=%EC%A1%B0%EC%84%B8%ED%94%84_%ED%95%98%EB%A0%88%EB%B9%84&amp;action=edit&amp;redlink=1" title="조세프 하레비 (없는 문서)">조세프 하레비</a>는 다양한 음성을 구분하기 위한 복잡한 숫자들과 판독이 어려운 기호들이 나열되어 있는 수메르어를 해독하기 위해서는 자연 언어를 다루는 방법으로 접근하는 것보다는 차라리 <a href="/wiki/%EC%95%94%ED%98%B8%ED%95%99" title="암호학">암호학</a>의 방법이 효과적일 것이라고 주장하였다. 이 주장을 놓고 <a href="/wiki/%EC%95%84%EC%8B%9C%EB%A6%AC%EC%95%84%ED%95%99" title="아시리아학">아시리아학</a>을 연구하는 학자들은 격론을 벌였으나, 1885년에 이르러서는 아시리아학의 권위자였던 <a class="new" href="/w/index.php?title=%ED%94%84%EB%A6%AC%EB%93%9C%EB%A6%AC%ED%9E%88_%EB%8D%B0%EB%A6%AC%EC%B7%A8&amp;action=edit&amp;redlink=1" title="프리드리히 데리취 (없는 문서)">프리드리히 데리취</a>마저 하레비의 의견을 받아들이게 되었다.<sup class="reference" id="cite_ref-17"><a href="#cite_note-17">[16]</a></sup></p>
<p>1908년 아시리아학자였던 <a class="new" href="/w/index.php?title=%EC%8A%A4%ED%85%8C%ED%8E%80_%EB%9E%AD%EB%8D%98&amp;action=edit&amp;redlink=1" title="스테펀 랭던 (없는 문서)">스테펀 랭던</a>은 <a class="new" href="/w/index.php?title=%EC%83%A4%EB%A5%BC_%EB%B9%84%EB%A1%9C%EB%A2%B0%EB%93%9C&amp;action=edit&amp;redlink=1" title="샤를 비로뢰드 (없는 문서)">샤를 비로뢰드</a>의 <a href="/wiki/%EC%9A%B0%EA%B0%80%EB%A6%AC%ED%8A%B8" title="우가리트">우가리트</a> 발굴 결과를 바탕으로 《<a class="external text" href="http://quod.lib.umich.edu/cgi/t/text/pageviewer-idx?c=genpub;cc=genpub;sid=925969e1cdb54224dccf86fdb4bdef76;rgn=full%20text;idno=ACG1616.0002.001;view=image;seq=00000217" rel="nofollow">수메르어-아시리아어 어휘</a>》를 출간하였다. 랭던의 작업은 한동안 신뢰를 받지 못하였으나 부르노 마이스너는 랭던의 약자 표기를 높게 평가하였다.</p>
<p>1944년 수메르학자 <a class="new" href="/w/index.php?title=%EC%82%AC%EB%AC%B4%EC%97%98_%EB%85%B8%EC%95%84_%ED%81%AC%EB%9D%BC%EB%A9%94%EB%A5%B4&amp;action=edit&amp;redlink=1" title="사무엘 노아 크라메르 (없는 문서)">사무엘 노아 크라메르</a>는 자신의 해독 결과를 정리하여 《<a class="external text" href="http://www.sacred-texts.com/ane/sum/sum05.htm" rel="nofollow">수메르 신화</a>》를 출간하였다.</p>
<h2><span class="mw-headline" id=".EB.85.BC.EB.9E.80">논란</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%88%98%EB%A9%94%EB%A5%B4%EC%96%B4&amp;action=edit&amp;section=9" title="부분 편집: 논란">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>수메르어는 다른 언어와 연관성을 찾을 수 없는 <a class="mw-disambig" href="/wiki/%EA%B3%A0%EB%A6%BD%EC%96%B4" title="고립어">고립어</a>에 속한다고 알려져 왔다. 그러나 이스라엘 히브리대학의 조철수 교수는 수메르어와 한국어는 같은 뿌리에서 나온 언어라는 주장을 제시했다. 조철수 교수는 한국언어학회 논문집인 《언어학》에 〈수메르어·국어고어 문법범주 대조분석〉이라는 학술논문을 싣고<sup class="reference" id="cite_ref-18"><a href="#cite_note-18">[17]</a></sup>, 수메르어와 한국어는 같은 <a href="/wiki/%EA%B5%90%EC%B0%A9%EC%96%B4" title="교착어">교착어</a>로서 비슷한 어휘가 상당히 많다는 점을 지적했다. 예를 들어 수메르어의 복수형 어미 '네'는 한국어 '남정네'의 '네'와 같다는 것이다.<sup class="reference" id="cite_ref-.EA.B2.BD.ED.96.A5.EC.8B.A0.EB.AC.B8_19-0"><a href="#cite_note-.EA.B2.BD.ED.96.A5.EC.8B.A0.EB.AC.B8-19">[18]</a></sup> 서울고전고대문헌연구소 박기용 소장은 "우랄-알타이어족의 상위 개념으로서 수메르어와 한국어를 포괄하는 어족을 설정할 수 있다"는 주장을 제시하고, 두 언어의 유사성에 대해 더욱 깊은 연구가 이뤄져야 한다고 말했다.<sup class="reference" id="cite_ref-.EA.B2.BD.ED.96.A5.EC.8B.A0.EB.AC.B8_19-1"><a href="#cite_note-.EA.B2.BD.ED.96.A5.EC.8B.A0.EB.AC.B8-19">[18]</a></sup></p>
