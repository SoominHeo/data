<p><b>수산화 나트륨</b>(水酸化─ , <a href="/wiki/%ED%99%94%ED%95%99%EC%8B%9D" title="화학식">화학식</a> : NaOH)은 <a href="/wiki/%EB%AC%BC" title="물">물</a>에 녹아 <a href="/wiki/%EA%B0%95%EC%97%BC%EA%B8%B0" title="강염기">강염기</a>성 <a href="/wiki/%EC%88%98%EC%9A%A9%EC%95%A1" title="수용액">수용액</a>을 만든다. <a class="new" href="/w/index.php?title=%EB%B0%A9%EC%A7%81&amp;action=edit&amp;redlink=1" title="방직 (없는 문서)">방직</a>, <a class="mw-redirect" href="/wiki/%EC%8B%9D%EC%9D%8C%EB%A3%8C" title="식음료">식음료</a>, <a href="/wiki/%EB%B9%84%EB%88%84" title="비누">비누</a> 등의 산업에서 널리 사용된다. 1998년에 세계적으로 4500만<a href="/wiki/%ED%86%A4" title="톤">톤</a>이 생산되었다. 화학 실험에서 가장 널리 사용되는 <a href="/wiki/%EC%97%BC%EA%B8%B0" title="염기">염기</a>이다.</p>
<p>순수한 수산화 나트륨은 <a class="mw-redirect" href="/wiki/%ED%9D%B0%EC%83%89" title="흰색">흰색</a> <a href="/wiki/%EA%B3%A0%EC%B2%B4" title="고체">고체</a>이다. 대기 중에서 수증기와 <a href="/wiki/%EC%9D%B4%EC%82%B0%ED%99%94_%ED%83%84%EC%86%8C" title="이산화 탄소">이산화 탄소</a>를 잘 흡수한다. 온도에 무관하게 물에 잘 녹으며 이 때 많은 열이 발생한다. <a href="/wiki/%EC%88%98%EC%82%B0%ED%99%94_%EC%B9%BC%EB%A5%A8" title="수산화 칼륨">수산화 칼륨</a>은 잘 녹지 않는 <a href="/wiki/%EC%97%90%ED%83%84%EC%98%AC" title="에탄올">에탄올</a>과 <a href="/wiki/%EB%A9%94%ED%83%84%EC%98%AC" title="메탄올">메탄올</a>에도 녹는다. <a class="mw-redirect" href="/wiki/%EB%8B%A4%EC%9D%B4%EC%97%90%ED%8B%B8_%EC%97%90%ED%85%8C%EB%A5%B4" title="다이에틸 에테르">에테르</a>나 다른 <a class="mw-redirect" href="/wiki/%EB%AC%B4%EA%B7%B9%EC%84%B1" title="무극성">무극성</a> <a href="/wiki/%EC%9A%A9%EB%A7%A4" title="용매">용매</a>에는 녹일 수 없다. 수산화 나트륨 수용액은 <a href="/wiki/%EC%84%AC%EC%9C%A0" title="섬유">섬유</a>와 <a href="/wiki/%EC%A2%85%EC%9D%B4" title="종이">종이</a>에 <a class="mw-redirect" href="/wiki/%EB%85%B8%EB%9E%80%EC%83%89" title="노란색">노란색</a> <a class="new" href="/w/index.php?title=%EC%96%BC%EB%A3%A9&amp;action=edit&amp;redlink=1" title="얼룩 (없는 문서)">얼룩</a>을 남긴다.</p>

<h2><span class="mw-headline" id=".EC.A0.9C.EC.A1.B0">제조</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%88%98%EC%82%B0%ED%99%94_%EB%82%98%ED%8A%B8%EB%A5%A8&amp;action=edit&amp;section=1" title="부분 편집: 제조">편집</a><span class="mw-editsection-bracket">]</span></span></h2>

<h2><span class="mw-headline" id=".EC.9A.A9.EB.8F.84">용도</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%88%98%EC%82%B0%ED%99%94_%EB%82%98%ED%8A%B8%EB%A5%A8&amp;action=edit&amp;section=2" title="부분 편집: 용도">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<h3><span class="mw-headline" id=".EB.B0.94.EC.9D.B4.EC.98.A4.EB.94.94.EC.A0.A4">바이오디젤</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%88%98%EC%82%B0%ED%99%94_%EB%82%98%ED%8A%B8%EB%A5%A8&amp;action=edit&amp;section=3" title="부분 편집: 바이오디젤">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p><a href="/wiki/%EB%B0%94%EC%9D%B4%EC%98%A4%EB%94%94%EC%A0%A4" title="바이오디젤">바이오디젤</a>을 생산할 때 수산화 나트륨은 교환반응의 <a href="/wiki/%EC%B4%89%EB%A7%A4" title="촉매">촉매</a>로 사용된다. 물과 지방은 <a href="/wiki/%EB%B9%84%EB%88%84" title="비누">비누</a>화반응을 일으키기 때문에 <a class="new" href="/w/index.php?title=%EB%AC%B4%EC%88%98&amp;action=edit&amp;redlink=1" title="무수 (없는 문서)">무수</a>수산화나트륨만 이용할 수 있다. 더 싸고 적은 양이 필요하기 때문에 <a href="/wiki/%EC%88%98%EC%82%B0%ED%99%94_%EC%B9%BC%EB%A5%A8" title="수산화 칼륨">수산화 칼륨</a>에 비해 자주 쓰인다.</p>
<h3><span class="mw-headline" id=".EB.A7.88.EC.95.BD_.EC.A0.9C.EC.A1.B0">마약 제조</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%88%98%EC%82%B0%ED%99%94_%EB%82%98%ED%8A%B8%EB%A5%A8&amp;action=edit&amp;section=4" title="부분 편집: 마약 제조">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>수산화 나트륨은 <a href="/wiki/%EB%A9%94%EC%8A%A4%EC%95%94%ED%8E%98%ED%83%80%EB%AF%BC" title="메스암페타민">메스암페타민</a>이나 다른 <a href="/wiki/%EB%A7%88%EC%95%BD" title="마약">마약</a>을 제조하는 데 필수적인 물질이다. 언론에 널리 알려진 것과 다르게 이것은 제조 원료로서 들어가는 것은 아니며 단지 화학 반응상 pH를 조절하기 위한 강염기로 사용되는 것이다.</p>
<h2><span class="mw-headline" id=".EC.95.88.EC.A0.84.EC.84.B1">안전성</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%88%98%EC%82%B0%ED%99%94_%EB%82%98%ED%8A%B8%EB%A5%A8&amp;action=edit&amp;section=5" title="부분 편집: 안전성">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>고체 수산화 나트륨이나 고농도의 수산화 나트륨 수용액은 <a class="new" href="/w/index.php?title=%ED%99%94%ED%95%99%EC%A0%81_%ED%99%94%EC%83%81&amp;action=edit&amp;redlink=1" title="화학적 화상 (없는 문서)">화학적 화상</a>을 유발할 수 있으며 영구적인 상처나 <a href="/wiki/%EC%8B%A4%EB%AA%85" title="실명">실명</a>을 유발할 수 있다.</p>
<p>알루미늄과 수산화 나트륨은 많은 수소 기체를 발생시킨다.</p>
<dl>
<dd>2<a href="/wiki/%EC%95%8C%EB%A3%A8%EB%AF%B8%EB%8A%84" title="알루미늄">Al</a>(<a href="/wiki/%EA%B3%A0%EC%B2%B4" title="고체">s</a>) + 6NaOH(aq) → 3<a href="/wiki/%EC%88%98%EC%86%8C" title="수소">H<sub>2</sub></a>(<a href="/wiki/%EA%B8%B0%EC%B2%B4" title="기체">g</a>) + 2Na<sub>3</sub>AlO<sub>3</sub>(aq).<br/></dd>
</dl>
<p>밀폐된 공간에서 수소 기체를 대량으로 발생시킬 경우 위험하다.</p>
