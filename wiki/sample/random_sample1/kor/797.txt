<h1 class="firstHeading" id="firstHeading" lang="ko">BBC 노스웨스트</h1>

<div class="mw-content-ltr" dir="ltr" id="mw-content-text" lang="ko"><script>function mfTempOpenSection(id){var block=document.getElementById("mf-section-"+id);block.className+=" open-block";block.previousSibling.className+=" open-block";}</script><p><b>BBC 노스웨스트</b> (BBC North West)는 <a href="/wiki/%EC%B2%B4%EC%85%94_%EC%A3%BC" title="체셔 주">체셔 주</a>, <a class="mw-redirect" href="/wiki/%EA%B7%B8%EB%A0%88%EC%9D%B4%ED%84%B0%EB%A7%A8%EC%B2%B4%EC%8A%A4%ED%84%B0" title="그레이터맨체스터">그레이터맨체스터</a>, <a href="/wiki/%EB%9E%AD%EC%BB%A4%EC%85%94_%EC%A3%BC" title="랭커셔 주">랭커셔 주</a>, <a href="/wiki/%EB%A8%B8%EC%A7%80%EC%82%AC%EC%9D%B4%EB%93%9C_%EC%A3%BC" title="머지사이드 주">머지사이드 주</a>, <a href="/wiki/%EB%85%B8%EC%8A%A4%EC%9A%94%ED%81%AC%EC%85%94_%EC%A3%BC" title="노스요크셔 주">노스요크셔 주</a> (<a href="/wiki/%ED%81%AC%EB%A0%88%EC%9D%B4%EB%B8%90" title="크레이븐">크레이븐</a> 서부) <a href="/wiki/%EC%9B%A8%EC%8A%A4%ED%8A%B8%EC%9A%94%ED%81%AC%EC%85%94_%EC%A3%BC" title="웨스트요크셔 주">웨스트요크셔 주</a> (월스덴), <a href="/wiki/%EB%8D%94%EB%B9%84%EC%85%94_%EC%A3%BC" title="더비셔 주">더비셔 주</a> (<a class="new" href="/w/index.php?title=%ED%95%98%EC%9D%B4%ED%94%BC%ED%81%AC_(%EB%8D%94%EB%B9%84%EC%85%94_%EC%A3%BC)&amp;action=edit&amp;redlink=1" title="하이피크 (더비셔 주) (없는 문서)">하이피크</a> 서부), <a href="/wiki/%EC%BB%B4%EB%B8%8C%EB%A6%AC%EC%95%84_%EC%A3%BC" title="컴브리아 주">컴브리아 주</a> (<a href="/wiki/%EB%B0%B0%EB%A1%9C%EC%9D%B8%ED%8D%BC%EB%8B%88%EC%8A%A4_%EA%B5%AC" title="배로인퍼니스 구">배로인퍼니스 구</a>와 <a href="/wiki/%EC%82%AC%EC%9A%B0%EC%8A%A4%EB%A0%88%EC%9D%B4%ED%81%B4%EB%9E%9C%EB%93%9C" title="사우스레이클랜드">사우스레이클랜드</a>), <a href="/wiki/%EB%A7%A8_%EC%84%AC" title="맨 섬">맨 섬</a>을 대상으로 한 <a href="/wiki/BBC_%EC%9E%89%EA%B8%80%EB%A6%AC%EC%8B%9C_%EB%A6%AC%EC%A0%84%EC%8A%A4" title="BBC 잉글리시 리전스">BBC 잉글랜드 지방</a> 방송이다.</p>
<p>노스웨스트는 1980년대 후반까지 나머지 컴브리아 지역까지도 포괄하고 자체 지역 텔레비전 뉴스 방송까지 진행했지만, 지역내 높은 시청 수요로 인해 <a class="new" href="/w/index.php?title=BBC_%EB%85%B8%EC%8A%A4%EC%9D%B4%EC%8A%A4%ED%8A%B8&amp;action=edit&amp;redlink=1" title="BBC 노스이스트 (없는 문서)">BBC 노스이스트</a> 소속으로 이관됐다. 현재 해당 컴브리아 지역은 <a class="new" href="/w/index.php?title=BBC_%EB%85%B8%EC%8A%A4&amp;action=edit&amp;redlink=1" title="BBC 노스 (없는 문서)">BBC 노스 지국</a> 소속이며 <a class="new" href="/w/index.php?title=%EC%83%90%ED%8D%BC%EB%93%9C%EC%BF%BC%EC%9D%B4%EC%8A%A4&amp;action=edit&amp;redlink=1" title="샐퍼드쿼이스 (없는 문서)">샐퍼드쿼이스</a>의 <a class="new" href="/w/index.php?title=%EB%AF%B8%EB%94%94%EC%96%B4%EC%8B%9C%ED%8B%B0UK&amp;action=edit&amp;redlink=1" title="미디어시티UK (없는 문서)">미디어시티UK</a>에서 보내는 방송을 수신받고 있다.</p>

<h2><span class="mw-headline" id=".EB.B0.A9.EC.86.A1">방송</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=BBC_%EB%85%B8%EC%8A%A4%EC%9B%A8%EC%8A%A4%ED%8A%B8&amp;action=edit&amp;section=1" title="부분 편집: 방송">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<h3><span class="mw-headline" id=".ED.85.94.EB.A0.88.EB.B9.84.EC.A0.84">텔레비전</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=BBC_%EB%85%B8%EC%8A%A4%EC%9B%A8%EC%8A%A4%ED%8A%B8&amp;action=edit&amp;section=2" title="부분 편집: 텔레비전">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>BBC 노스웨스트는 지역 텔레비전 뉴스, 시사 프로와 스포츠 프로를 방송하며 야간 뉴스 프로그램인 &lt;<a class="new" href="/w/index.php?title=%EB%85%B8%EC%8A%A4%EC%9B%A8%EC%8A%A4%ED%8A%B8_%ED%88%AC%EB%82%98%EC%9E%87&amp;action=edit&amp;redlink=1" title="노스웨스트 투나잇 (없는 문서)">노스웨스트 투나잇</a>&gt;을 필두로, 낮시간대 뉴스 단신 프로그램 &lt;노스웨스트 투데이&gt;와, &lt;<a href="/wiki/BBC_%EB%B8%8C%EB%A0%89%ED%8D%BC%EC%8A%A4%ED%8A%B8" title="BBC 브렉퍼스트">BBC 브렉퍼스트</a>&gt; 시간에 내보내는 자체 뉴스방송으로 이뤄져 있다.<sup class="reference" id="cite_ref-1"><a href="#cite_note-1">[1]</a></sup></p>
<p>뉴스가 아닌 프로그램으로는 화제 보도 프로그램인 &lt;<a class="new" href="/w/index.php?title=%EC%9D%B8%EC%82%AC%EC%9D%B4%EB%93%9C_%EC%95%84%EC%9B%83_(BBC)&amp;action=edit&amp;redlink=1" title="인사이드 아웃 (BBC) (없는 문서)">인사이드 아웃</a>&gt;과, &lt;<a class="new" href="/w/index.php?title=%EC%84%A0%EB%8D%B0%EC%9D%B4_%ED%8F%B4%EB%A6%AC%ED%8B%B1%EC%8A%A4&amp;action=edit&amp;redlink=1" title="선데이 폴리틱스 (없는 문서)">선데이 폴리틱스</a>&gt; 및 늦은밤 축구소식 프로그램 &lt;<a class="new" href="/w/index.php?title=%EB%A0%88%EC%9D%B4%ED%8A%B8_%ED%82%A5_%EC%98%A4%ED%94%84&amp;action=edit&amp;redlink=1" title="레이트 킥 오프 (없는 문서)">레이트 킥 오프</a>&gt; 시간에 내보내는 20분짜리 자체방송이 있다. <a class="new" href="/w/index.php?title=BBC_%EC%9A%94%ED%81%AC%EC%85%94&amp;action=edit&amp;redlink=1" title="BBC 요크셔 (없는 문서)">BBC 요크셔</a> 리즈 스튜디오에서 PDI 미디어가 제작하는 럭비 리그경기 하이라이트 프로그램인 &lt;<a class="new" href="/w/index.php?title=%EC%8A%88%ED%8D%BC_%EB%A6%AC%EA%B7%B8_%EC%87%BC&amp;action=edit&amp;redlink=1" title="슈퍼 리그 쇼 (없는 문서)">슈퍼 리그 쇼</a>&gt;도 노스웨스트 지역 시청자들에게 방송된다.</p>
<h3><span class="mw-headline" id=".EB.9D.BC.EB.94.94.EC.98.A4">라디오</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=BBC_%EB%85%B8%EC%8A%A4%EC%9B%A8%EC%8A%A4%ED%8A%B8&amp;action=edit&amp;section=3" title="부분 편집: 라디오">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>BBC 노스웨스트는 <a class="new" href="/w/index.php?title=BBC_%EB%9D%BC%EB%94%94%EC%98%A4_%EB%9E%AD%EC%BB%A4%EC%85%94&amp;action=edit&amp;redlink=1" title="BBC 라디오 랭커셔 (없는 문서)">BBC 라디오 랭커셔</a>, <a class="new" href="/w/index.php?title=BBC_%EB%9D%BC%EB%94%94%EC%98%A4_%EB%A7%A8%EC%B2%B4%EC%8A%A4%ED%84%B0&amp;action=edit&amp;redlink=1" title="BBC 라디오 맨체스터 (없는 문서)">BBC 라디오 맨체스터</a>, <a class="new" href="/w/index.php?title=BBC_%EB%9D%BC%EB%94%94%EC%98%A4_%EB%A8%B8%EC%A7%80%EC%82%AC%EC%9D%B4%EB%93%9C&amp;action=edit&amp;redlink=1" title="BBC 라디오 머지사이드 (없는 문서)">BBC 라디오 머지사이드</a>를 총괄한다.</p>
<p>라디오 맨체스터는 하루 24시간 방송되며 라디오 머지사이드와 라디오 랭커셔는 평일 오전 5시~오전 1시, 주말에는 랭커셔가 오전 6시~자정 12시, 머지사이드가 오전 6시~새벽 1시에 방송된다. 세 방송국은 매 저녁마다 &lt;BBC 뉴스 인 더 노스웨스트&gt;만 방송하고, 여기서 라디오 맨체스터와 라디오 랭커셔는 평일 늦은 오후 쇼프로와 일요일 저녁의 아시아 소수민족 대상 프로그램인 &lt;인더스&gt;를 추가로 내보낸다.</p>
<h3><span class="mw-headline" id=".EC.98.A8.EB.9D.BC.EC.9D.B8.EA.B3.BC_.EC.83.81.ED.98.B8_.EB.B0.A9.EC.86.A1">온라인과 상호 방송</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=BBC_%EB%85%B8%EC%8A%A4%EC%9B%A8%EC%8A%A4%ED%8A%B8&amp;action=edit&amp;section=4" title="부분 편집: 온라인과 상호 방송">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>BBC 노스이스트 컴브리아는 <a class="new" href="/w/index.php?title=BBC_%EB%A0%88%EB%93%9C_%EB%B2%84%ED%8A%BC&amp;action=edit&amp;redlink=1" title="BBC 레드 버튼 (없는 문서)">BBC 레드 버튼</a>에 지방 뉴스와 지역 라디오 페이지, 그리고 각 주와 맨 섬 전용 'BBC 로컬 뉴스' 웹사이트도 제공하고 있다.</p>
<h2><span class="mw-headline" id=".EC.97.AD.EC.82.AC">역사</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=BBC_%EB%85%B8%EC%8A%A4%EC%9B%A8%EC%8A%A4%ED%8A%B8&amp;action=edit&amp;section=5" title="부분 편집: 역사">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<h3><span class="mw-headline" id=".EB.9D.BC.EB.94.94.EC.98.A4_2">라디오</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=BBC_%EB%85%B8%EC%8A%A4%EC%9B%A8%EC%8A%A4%ED%8A%B8&amp;action=edit&amp;section=6" title="부분 편집: 라디오">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>노스웨스트 지역방송은 BBC가 리버풀 소재 라디오 기지국인 2LY를 개국한 뒤 2년이 지난 1922년 11월, 맨체스터의 라디오 기지국 <a class="new" href="/w/index.php?title=2ZY&amp;action=edit&amp;redlink=1" title="2ZY (없는 문서)">2ZY</a>을 개국하면서 시작됐다. 당시 맨체스터 지국은 리즈 (2LS), 헐 (6KH), 노팅엄 (5NG), 스토크온트렌트 (6ST)에 송출국을 두고 2ZY 방송을 내보내며 노스오브잉글랜드 지역과 미들랜즈 북부 지역을 전부 담당했다. 초창기에는 올드트래퍼드에 있는 <a class="new" href="/w/index.php?title=%EB%A9%94%ED%8A%B8%EB%A1%9C%ED%8F%B4%EB%A6%AC%ED%83%84-%EB%B9%84%EC%BB%A4%EC%8A%A4&amp;action=edit&amp;redlink=1" title="메트로폴리탄-비커스 (없는 문서)">메트로폴리탄-비커스</a> 전파공작소 (Metropolitan-Vickers electricity works)에서 이뤄졌다.</p>
<p>런던에서 <a class="new" href="/w/index.php?title=BBC_%EB%82%B4%EC%85%94%EB%84%90_%ED%94%84%EB%A1%9C%EA%B7%B8%EB%9E%A8&amp;action=edit&amp;redlink=1" title="BBC 내셔널 프로그램 (없는 문서)">BBC 내셔널 프로그램</a> 방송을 개시하자 맨체스터 지국 방송은 <a class="new" href="/w/index.php?title=BBC_%EB%A6%AC%EC%A0%80%EB%84%90_%ED%94%84%EB%A1%9C%EA%B7%B8%EB%9E%A8&amp;action=edit&amp;redlink=1" title="BBC 리저널 프로그램 (없는 문서)">BBC 리저널 프로그램</a>으로 옮겨갔고, 1929년에는 피카딜리 가든스 분수 앞에 위치한 전용 스튜디오 복합단지인 브로드캐스팅 하우스에서 방송 운영을 시작했다. 1936년 3월에는 맨체스터에 본부를 둔 노던 라디오 서비스가 <a class="new" href="/w/index.php?title=%EB%A6%AC%EC%8A%A4%EB%84%88%EA%B1%B0%EB%B2%A0%EC%9D%B4_%EC%86%A1%EC%8B%A0%EC%86%8C&amp;action=edit&amp;redlink=1" title="리스너거베이 송신소 (없는 문서)">리스너거베이</a>에 송신소를 신축하면서 북아일랜드를 대상으로 방송을 개시했다 (단, 추가 자체 지역방송도 진행). 1년 후에는 스태그쇼 송신소도 완공되면서 노스이스트 컴브리아 지역 청취자들에게 기본 전국방송에서 자체방송으로 전환해 내보낼 수 있게 되었다.</p>
<p>제2차 세계 대전이 끝나고 리저널 라디오 서비스가 재개되자 <a class="new" href="/w/index.php?title=BBC_%ED%99%88_%EC%84%9C%EB%B9%84%EC%8A%A4&amp;action=edit&amp;redlink=1" title="BBC 홈 서비스 (없는 문서)">BBC 홈 서비스</a> 북부방송에 맨체스터발 프로그램을 내보내기 시작했고, 1963년 1월까지 북아일랜드를 대상으로 방송됐다. 1967년 11월 22일에는 리버풀에 <a class="new" href="/w/index.php?title=BBC_%EB%9D%BC%EB%94%94%EC%98%A4_%EB%A8%B8%EC%A7%80%EC%82%AC%EC%9D%B4%EB%93%9C&amp;action=edit&amp;redlink=1" title="BBC 라디오 머지사이드 (없는 문서)">BBC 라디오 머지사이드</a> 설립과 동시에 노스웨스트 지역에 <a class="new" href="/w/index.php?title=BBC_%EB%A1%9C%EC%BB%AC_%EB%9D%BC%EB%94%94%EC%98%A4&amp;action=edit&amp;redlink=1" title="BBC 로컬 라디오 (없는 문서)">BBC 로컬 라디오</a> 방송이 시작됐고, 그 뒤를 이어 1970년 9월 10일에는 <a class="new" href="/w/index.php?title=BBC_%EB%9D%BC%EB%94%94%EC%98%A4_%EB%A7%A8%EC%B2%B4%EC%8A%A4%ED%84%B0&amp;action=edit&amp;redlink=1" title="BBC 라디오 맨체스터 (없는 문서)">BBC 라디오 맨체스터</a>, 1971년 1월 6일에는 <a class="new" href="/w/index.php?title=BBC_%EB%9D%BC%EB%94%94%EC%98%A4_%EB%B8%94%EB%9E%99%EB%B2%88&amp;action=edit&amp;redlink=1" title="BBC 라디오 블랙번 (없는 문서)">BBC 라디오 블랙번</a> (이후 BBC 라디오 랭커셔로 재출범)이 개국했다.</p>
<p>현지 라디오 방송 확대로 현 <a href="/wiki/BBC_%EB%9D%BC%EB%94%94%EC%98%A4_4" title="BBC 라디오 4">BBC 라디오 4</a>의 지역 라디오 프로그램 (홈모스 송신소에서 보내는 노스웨스트와 요크셔 지역 대상 방송)도 점차 줄어들었다. 라디오 4의 마지막 지역 프로그램은 이른 저녁 뉴스 프로그램으로 1980년 9월 12월 금요일에 방송됐다. 노스웨스트 대상 지역 프로그램은 나중에 BBC 로컬 라디오 방송에서 저녁과 늦은밤 시간대에 모습을 보이게 되었다.</p>
<p>한편 <a href="/wiki/%EB%A7%A8_%EC%84%AC" title="맨 섬">맨 섬</a> 지역에는 BBC 전국 라디오 방송은 있으나 BBC가 지역을 대상으로 한 라디오 방송은 진행한 적이 없다. 맨 섬의 현지 라디오 방송은 <a class="new" href="/w/index.php?title=%EB%A7%B9%ED%81%AC%EC%8A%A4_%EB%9D%BC%EB%94%94%EC%98%A4&amp;action=edit&amp;redlink=1" title="맹크스 라디오 (없는 문서)">맹크스 라디오</a>와 <a class="new" href="/w/index.php?title=%EC%97%90%EB%84%88%EC%A7%80_FM_(%EB%A7%A8_%EC%84%AC)&amp;action=edit&amp;redlink=1" title="에너지 FM (맨 섬) (없는 문서)">에너지 FM</a> 등 맨 섬 내에 자리한 방송국에서 맡고 있다.</p>
<h3><span class="mw-headline" id=".ED.85.94.EB.A0.88.EB.B9.84.EC.A0.84_2">텔레비전</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=BBC_%EB%85%B8%EC%8A%A4%EC%9B%A8%EC%8A%A4%ED%8A%B8&amp;action=edit&amp;section=7" title="부분 편집: 텔레비전">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>1957년 9월 BBC가 일일 지역 텔레비전 뉴스 방송을 개시했을 때, 초창기 북부 잉글랜드 전역에서는 맨체스터에서 내보내는 방송 하나로만 만족해야 했다. 같은해 9월 30일에는 짧은 뉴스 단신 프로인 &lt;뉴스 프롬 더 노스&gt;가 현재의 노스웨스트, <a class="new" href="/w/index.php?title=BBC_%EC%9A%94%ED%81%AC%EC%85%94&amp;action=edit&amp;redlink=1" title="BBC 요크셔 (없는 문서)">요크셔-노스미들랜즈</a>, <a class="new" href="/w/index.php?title=BBC_%EC%9A%94%ED%81%AC%EC%85%94_%EB%A7%81%EC%BB%A8%EC%85%94&amp;action=edit&amp;redlink=1" title="BBC 요크셔 링컨셔 (없는 문서)">이스트요스셔-링컨셔</a>, <a href="/wiki/BBC_%EB%85%B8%EC%8A%A4%EC%9D%B4%EC%8A%A4%ED%8A%B8_%EC%BB%B4%EB%B8%8C%EB%A6%AC%EC%95%84" title="BBC 노스이스트 컴브리아">노스이스트-컴브리아</a> 지역에 방송되기 시작했다.</p>
<p>2년 후 노스이스트 컴브리아 지역에 자체 지역 뉴스 단신이 방송되면서, 노스웨스트와 요크셔 링컨셔 지역에도 지역 대상 자체방송에 다시 초점이 모아졌다. 1962년부터는 뉴스 단신 프로그램이 20분짜리 보도 프로그램으로 확대됐고 이름도 &lt;노스 앳 식스&gt;로 바뀌었다. 지금의 노스웨스트 방송권역은 1968년 3월 25일 요크셔, 링컨셔, 노스미들랜즈 지역을 대상으로 한 <a href="/wiki/%EB%A6%AC%EC%A6%88" title="리즈">리즈</a>발 별도 &lt;<a class="new" href="/w/index.php?title=%EB%A3%A9_%EB%85%B8%EC%8A%A4_(%EC%9A%94%ED%81%AC%EC%85%94_%EB%85%B8%EC%8A%A4%EB%AF%B8%EB%93%A4%EB%9E%9C%EC%A6%88&amp;action=edit&amp;redlink=1" title="룩 노스 (요크셔 노스미들랜즈 (없는 문서)">룩 노스</a>&gt; 프로그램이 시작되면서 첫모습을 갖추게 되었다.</p>
<p>1976년 4월 5일부터 1980년 9월 12일까지 BBC 노스웨스트는 BBC1의 평일 저녁 프로그램 시간에 지역별 막간 프로와 아나운서들을 이용해 지역방송을 했다. 또 1980년에는 맨체스터발 &lt;룩 노스&gt; 프로그램이 바로 이웃한 요크셔 지역 시청자들에게 혼란을 주지 않도록 &lt;룩 노스웨스트&gt;로 이름를 바꿨다. 이 지역 텔레비전 뉴스는 맨체스터 피카딜리에 있는 구 BBC 브로드캐스팅 하우스에서 진행하다가 1981년 5월 18일 옥스퍼드 로에 위치한 신 브로드캐스팅 하우스로 이전했다. 1984년 9월 3일에는 현재의 제목인 &lt;<a class="new" href="/w/index.php?title=%EB%85%B8%EC%8A%A4%EC%9B%A8%EC%8A%A4%ED%8A%B8_%ED%88%AC%EB%82%98%EC%9E%87&amp;action=edit&amp;redlink=1" title="노스웨스트 투나잇 (없는 문서)">노스웨스트 투나잇</a>&gt;으로 이름을 바꿨다. 1986년부터 1989년까지 BBC 노스웨스트는 예전에 뉴캐슬발 &lt;룩 노스&gt;가 방송되던 컴브리아 지역을 맡게 되었고, 지역을 대상으로 자체 뉴스를 점심 시간대에 컬드백 송신소를 통해 내보냈다. 그러나 시청자들의 항의로 인해 컴브리아 지역 뉴스 방송은 뉴캐슬발 &lt;룩 노스&gt;로 되돌아가게 되었다.</p>
<p>30여년간 뉴 브로드캐스팅 하우스에서 방송을 진행한 끝에, BBC 노스웨스트 텔레비전은 2011년 11월 28일 샐퍼드쿼이스에 있는 미디어시티UK로 본부를 이전하고 방송을 개시했고 지금까지 이어지고 있다.</p>
<h2><span class="mw-headline" id=".EC.8A.A4.ED.8A.9C.EB.94.94.EC.98.A4">스튜디오</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=BBC_%EB%85%B8%EC%8A%A4%EC%9B%A8%EC%8A%A4%ED%8A%B8&amp;action=edit&amp;section=8" title="부분 편집: 스튜디오">편집</a><span class="mw-editsection-bracket">]</span></span></h2>

<p>1929년부터 1981년까지 BBC의 맨체스터 거점 방송 운영은 대부분 맨체스터 도심 내 피카딜리 가든스 지구에 자리한 브로드캐스팅 하우스에서 이뤄졌다. 브로스캐스팅 하우스는 1층에 있던 은행으로부터 BBC가 임대받아 사용해 왔으며, 맨체스터발 지역 라디오 방송과 네트워크 라디오 방송 진행을 맡은 중앙 총괄국이 되었다.</p>
<p>1954년 <a class="new" href="/w/index.php?title=%EB%9F%AC%EC%85%A4&amp;action=edit&amp;redlink=1" title="러셤 (없는 문서)">러셤</a> 디큰슨 로에 있던 옛 감리교회 건물이 BBC의 첫 맨체스터 소재 텔레비전 스튜디오가 되었다. 원래는 <a class="new" href="/w/index.php?title=%EB%A7%A8%EC%BB%A4%EB%8B%88%EC%96%B8_%ED%95%84%EB%A6%84&amp;action=edit&amp;redlink=1" title="맨커니언 필름 (없는 문서)">맨커니언 필름</a>사에서 매입해 용도를 바꿨다가 다시 BBC에게 매각한 것이었다. 디큰슨 로의 방송국은 멘체스터발 네트워크 방송 제작소의 중심지가 되었는데, &lt;<a href="/wiki/%ED%83%91_%EC%98%A4%EB%B8%8C_%EB%8D%94_%ED%8C%9D%EC%8A%A4" title="탑 오브 더 팝스">탑 오브 더 팝스</a>&gt;나 &lt;<a class="new" href="/w/index.php?title=%EC%96%B4_%ED%80%98%EC%8A%A4%EC%B2%9C_%EC%98%A4%EB%B8%8C_%EC%8A%A4%ED%8F%AC%EC%B8%A0&amp;action=edit&amp;redlink=1" title="어 퀘스천 오브 스포츠 (없는 문서)">어 퀘스천 오브 스포츠</a>&gt; 같은 장수 시리즈 프로그램들이 초창기에 이곳에서 제작되기도 했다. BBC의 북부 외곽 지역 대상 방송기지국은 <a class="new" href="/w/index.php?title=%EB%A1%B1%EC%82%AC%EC%9D%B4%ED%8A%B8&amp;action=edit&amp;redlink=1" title="롱사이트 (없는 문서)">롱사이트</a>에 있는 또다른 옛 교회 건물로 대신했다. 그러는 한편 지역 텔레비전 뉴스와 보도가 1957년부터 시작했는데, 처음엔 디큰슨 로의 지하실 건물에서 진행하다 곧이어 맨체스터 피카딜리 가든스에 있는 브로드캐스팅 하우스로 자리를 옮겼다.</p>
<p>이후 네트워크급과 지역급을 모두 아우르는 라디오•텔레비전 방송 스튜디오와, BBC 노던 심포니 오케스트라 (<a class="new" href="/w/index.php?title=BBC_%ED%95%84%ED%95%98%EB%AA%A8%EB%8B%89&amp;action=edit&amp;redlink=1" title="BBC 필하모닉 (없는 문서)">BBC 필하모닉</a>의 전신) 전용 리허설 공간 (스튜디오 7)이 들어설 새로운 방송국 사옥의 건설이 1967년부터 계획되기 시작했다. 이렇게 해서 옥스퍼드 로에 들어선 <a class="new" href="/w/index.php?title=%EB%89%B4_%EB%B8%8C%EB%A1%9C%EB%93%9C%EC%BA%90%EC%8A%A4%ED%8C%85_%ED%95%98%EC%9A%B0%EC%8A%A4_(%EB%A7%A8%EC%B2%B4%EC%8A%A4%ED%84%B0)&amp;action=edit&amp;redlink=1" title="뉴 브로드캐스팅 하우스 (맨체스터) (없는 문서)">뉴 브로드캐스팅 하우스</a>는 1975년 7월부터 방송을 개시했고, 그와 동시에 딕슨 로에서 이뤄지던 네트워크 텔레비전 방송 제작도 더욱 넓어진 스튜디오 A 건물로 자리를 옮기게 되었다. 스튜디오 B가 그 뒤를 이어 1981년 5월 완공되어 지역 텔레비전 뉴스 (&lt;룩 노스웨스트&gt;와 &lt;노스웨스트 투나잇&gt;) 스튜디오로 사용되기 시작, BBC는 52년간 이어온 피카딜리 스튜디오 시대를 마감하게 되었다.</p>
<p>2000년부터는 스튜디오 A에서 정기 프로그램 제작을 하지 않게 되었다. 한편 BBC는 <a class="new" href="/w/index.php?title=ITV_%EA%B7%B8%EB%9D%BC%EB%82%98%EB%8B%A4&amp;action=edit&amp;redlink=1" title="ITV 그라나다 (없는 문서)">ITV 그라나다</a>와 함께 공동 방송시설 기업인 <a class="new" href="/w/index.php?title=%EC%8A%A4%EB%A6%AC%EC%8B%9D%EC%8A%A4%ED%8B%B0%EB%AF%B8%EB%94%94%EC%96%B4&amp;action=edit&amp;redlink=1" title="스리식스티미디어 (없는 문서)">스리식스티미디어</a>를 출범시켜, 양쪽 방송 인력이 BBC의 옥스퍼드 로드 스튜디오와 <a class="new" href="/w/index.php?title=%EB%A7%A8%EC%B2%B4%EC%8A%A4%ED%84%B0_%EC%8A%A4%ED%8A%9C%EB%94%94%EC%98%A4&amp;action=edit&amp;redlink=1" title="맨체스터 스튜디오 (없는 문서)">그라나다의 쿼이 스트리트 복합단지</a>의 스튜디오 장비들을 공유해 사용할 수 있도록 했다. 그 결과 스튜디오 A는 2005년 임대 모집 형식으로 다시 문을 열었고 초창기에는 &lt;<a href="/wiki/%EB%9D%BC%EC%9D%B4%ED%94%84_%EC%98%A8_%EB%A7%88%EC%8A%A4" title="라이프 온 마스">라이프 온 마스</a>&gt;를 비롯한 드라마 제작용으로 쓰이게 되었다.</p>
<p>2011년 가을 BBC 노스웨스트는 <a class="new" href="/w/index.php?title=%EC%83%90%ED%8D%BC%EB%93%9C%EC%BF%BC%EC%9D%B4%EC%8A%A4&amp;action=edit&amp;redlink=1" title="샐퍼드쿼이스 (없는 문서)">샐퍼드쿼이스</a>에 자리한 새로운 스튜디오로 자리를 옮겼다. 새 조성단지에는 <a href="/wiki/CBBC" title="CBBC">BBC 칠드런스</a>, <a href="/wiki/BBC_%EC%8A%A4%ED%8F%AC%EC%B8%A0" title="BBC 스포츠">BBC 스포츠</a>, <a href="/wiki/BBC_%EB%9D%BC%EB%94%94%EC%98%A4_5_%EB%9D%BC%EC%9D%B4%EB%B8%8C" title="BBC 라디오 5 라이브">BBC 라디오 5 라이브</a>, <a href="/wiki/BBC_%EB%B8%8C%EB%A0%89%ED%8D%BC%EC%8A%A4%ED%8A%B8" title="BBC 브렉퍼스트">BBC 브렉퍼스트</a>, <a class="new" href="/w/index.php?title=BBC_%EB%9F%AC%EB%8B%9D&amp;action=edit&amp;redlink=1" title="BBC 러닝 (없는 문서)">BBC 러닝</a>, BBC 퓨처미디어를 비롯해 기존에 뉴 브로드캐스팅에 있던 모든 부서들이 입주하게 되었다. 이전 과정에서 기존에 쓰던 뉴 브로드캐스팅 하우스는 2011년 4월 리얼리티 에스테이트 사에 1000만 파운드의 거래금으로 매각됐다.</p>
<p>BBC 노스웨스트는 총괄 본부 외에도 다수의 <a class="new" href="/w/index.php?title=BBC_%EB%A1%9C%EC%BB%AC_%EB%9D%BC%EB%94%94%EC%98%A4&amp;action=edit&amp;redlink=1" title="BBC 로컬 라디오 (없는 문서)">로컬 라디오</a> 기지국, <a href="/wiki/%EB%A6%AC%EB%B2%84%ED%92%80" title="리버풀">리버풀</a>과 <a href="/wiki/%EB%B8%94%EB%9E%99%EB%B2%88" title="블랙번">블랙번</a>에 보도국 사무소, <a href="/wiki/%EC%B2%B4%EC%8A%A4%ED%84%B0" title="체스터">체스터</a>에 지역국을 두고 있다.</p>
