<h1 class="firstHeading" id="firstHeading" lang="ko">수증기</h1>
<div class="mw-body-content" id="bodyContent">
<div id="siteSub">위키백과, 우리 모두의 백과사전.</div>
<div id="contentSub"></div>
<div class="mw-jump" id="jump-to-nav">
					이동:					<a href="#mw-head">둘러보기</a>, 					<a href="#p-search">검색</a>
</div>
<div class="mw-content-ltr" dir="ltr" id="mw-content-text" lang="ko"><script>function mfTempOpenSection(id){var block=document.getElementById("mf-section-"+id);block.className+=" open-block";block.previousSibling.className+=" open-block";}</script><p><b>수증기</b>(水蒸氣)는 <a href="/wiki/%EB%AC%BC" title="물">물</a>의 <a href="/wiki/%EA%B8%B0%EC%B2%B4" title="기체">기체</a> <a class="mw-redirect" href="/wiki/%EC%83%81_(%EB%AC%BC%EB%A6%AC)" title="상 (물리)">상</a>이다. <a href="/wiki/%EC%A7%80%EA%B5%AC" title="지구">지구</a>에서 수증기는 <a href="/wiki/%EC%88%98%EA%B6%8C" title="수권">수권</a> 내부의 <a href="/wiki/%EB%AC%BC%EC%9D%98_%EC%88%9C%ED%99%98" title="물의 순환">물의 순환</a>의 한 가지 상태이다. 수증기는 <a href="/wiki/%EC%95%A1%EC%B2%B4" title="액체">액체</a> 상태의 물이 <a href="/wiki/%EC%A6%9D%EB%B0%9C" title="증발">증발</a>하거나, <a href="/wiki/%EA%B3%A0%EC%B2%B4" title="고체">고체</a> 상태인 <a href="/wiki/%EC%96%BC%EC%9D%8C" title="얼음">얼음</a>이 <a href="/wiki/%EC%8A%B9%ED%99%94_(%ED%99%94%ED%95%99)" title="승화 (화학)">승화</a>하는 것을 통해 만들어진다. 일반적인 대기 상태에서, 수증기는 끊임없이 <a href="/wiki/%EA%B8%B0%ED%99%94" title="기화">기화</a>와 <a class="mw-redirect" href="/wiki/%EC%95%A1%ED%99%94" title="액화">액화</a>를 반복한다. 일반적으로 수증기는 <a class="new" href="/w/index.php?title=%EB%A7%A8_%EB%88%88&amp;action=edit&amp;redlink=1" title="맨 눈 (없는 문서)">맨 눈</a>으로는 관찰할 수 없다.</p>
<p>수증기는 또한 지구 <a href="/wiki/%EC%98%A8%EC%8B%A4_%ED%9A%A8%EA%B3%BC" title="온실 효과">온실 효과</a>의 36~70%를 담당하는 <a href="/wiki/%EC%98%A8%EC%8B%A4_%EA%B8%B0%EC%B2%B4" title="온실 기체">온실 기체</a>이다.</p>

<h2><span class="mw-headline" id=".EC.88.98.EC.A6.9D.EA.B8.B0.EC.9D.98_.EC.9D.BC.EB.B0.98.EC.A0.81.EC.9D.B8_.ED.8A.B9.EC.A7.95">수증기의 일반적인 특징</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%88%98%EC%A6%9D%EA%B8%B0&amp;action=edit&amp;section=1" title="부분 편집: 수증기의 일반적인 특징">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<h3><span class="mw-headline" id=".EA.B8.B0.ED.99.94.2F.EC.8A.B9.ED.99.94">기화/승화</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%88%98%EC%A6%9D%EA%B8%B0&amp;action=edit&amp;section=2" title="부분 편집: 기화/승화">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<div class="thumb tright">
<div class="thumbinner" style="width:252px;"><a class="image" href="/wiki/%ED%8C%8C%EC%9D%BC:Watervapor_cup.jpg"><img alt="" class="thumbimage" data-file-height="300" data-file-width="300" height="250" src="//upload.wikimedia.org/wikipedia/commons/thumb/3/37/Watervapor_cup.jpg/250px-Watervapor_cup.jpg" srcset="//upload.wikimedia.org/wikipedia/commons/3/37/Watervapor_cup.jpg 1.5x" width="250"/></a>
<div class="thumbcaption">
<div class="magnify"><a class="internal" href="/wiki/%ED%8C%8C%EC%9D%BC:Watervapor_cup.jpg" title="실제 크기로"></a></div>
뜨거운 찻잔 위에서 액화하는 수증기</div>
</div>
</div>
<p>물 분자가 표면을 떠나 수증기가 되는 것은 기화라고 한다. 물 분자가 수증기로 될 때에, <a href="/wiki/%EC%97%B4" title="열">열</a>을 가지고 떠나게 되는데, 이러한 열 작용을 <a class="new" href="/w/index.php?title=%EA%B8%B0%ED%99%94_%EB%83%89%EA%B0%81&amp;action=edit&amp;redlink=1" title="기화 냉각 (없는 문서)">기화 냉각</a>이라고 한다. 이렇게 수증기가 된 물 분자가 다시 표면으로 돌아오는 비율은 공기중의 수증기의 양에 따라 결정된다. 만약, 순수하게 기화만 일어난다면, 액체는 수분을 잃어감에 따라 냉각되게 된다.</p>
<p>기화 냉각은 대기의 상태에 따라 제약을 받는다. 공기 중의 수증기의 양은 <a href="/wiki/%EC%8A%B5%EB%8F%84" title="습도">습도</a>라고 한다. 습도의 측정은 <a href="/wiki/%EC%8A%B5%EB%8F%84%EA%B3%84" title="습도계">습도계</a>에 의해 이루어지게 된다. 습도의 측정은 크게 두 가지 기준이 존재하며, <a class="new" href="/w/index.php?title=%EC%A0%88%EB%8C%80%EC%8A%B5%EB%8F%84&amp;action=edit&amp;redlink=1" title="절대습도 (없는 문서)">절대습도</a>와 <a class="mw-redirect" href="/wiki/%EC%83%81%EB%8C%80%EC%8A%B5%EB%8F%84" title="상대습도">상대습도</a>이다. 대기와 물 표면 온도는 포화수증기압을 결정하며, 상대습도로 100%는 공기중의 수증기압이 포화수증기압과 같을 때를 나타낸다.</p>
<p>액체에서 기체로 되는 과정 외에도 승화라는 다른 기화 형태가 존재한다. 승화는 고체에서 액체를 거치지 않고 바로 기체로 되는 현상이다. 동일한 원리로, 얼음이 주변에 비해 높은 온도를 가질때, 승화가 자주 발생한다. 눈이나 얼음이 녹기에는 추운 겨울에도, 서서히 사라져 가는 것은 바로 이러한 승화에 의해서이다.</p>
<h3><span class="mw-headline" id=".EC.95.A1.ED.99.94">액화</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%88%98%EC%A6%9D%EA%B8%B0&amp;action=edit&amp;section=3" title="부분 편집: 액화">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<div class="thumb tright">
<div class="thumbinner" style="width:222px;"><a class="image" href="/wiki/%ED%8C%8C%EC%9D%BC:%EC%88%98%EC%A6%9D%EA%B8%B0_(2).jpg"><img alt="" class="thumbimage" data-file-height="2902" data-file-width="2340" height="273" src="//upload.wikimedia.org/wikipedia/commons/thumb/9/95/%EC%88%98%EC%A6%9D%EA%B8%B0_%282%29.jpg/220px-%EC%88%98%EC%A6%9D%EA%B8%B0_%282%29.jpg" srcset="//upload.wikimedia.org/wikipedia/commons/thumb/9/95/%EC%88%98%EC%A6%9D%EA%B8%B0_%282%29.jpg/330px-%EC%88%98%EC%A6%9D%EA%B8%B0_%282%29.jpg 1.5x, //upload.wikimedia.org/wikipedia/commons/thumb/9/95/%EC%88%98%EC%A6%9D%EA%B8%B0_%282%29.jpg/440px-%EC%88%98%EC%A6%9D%EA%B8%B0_%282%29.jpg 2x" width="220"/></a>
<div class="thumbcaption">
<div class="magnify"><a class="internal" href="/wiki/%ED%8C%8C%EC%9D%BC:%EC%88%98%EC%A6%9D%EA%B8%B0_(2).jpg" title="실제 크기로"></a></div>
수증기</div>
</div>
</div>
<p>수증기는 자신의 온도보다 차가운 액체의 표면 위에 닿았거나, 혹은 포화 수증기압이 초과되었을 경우에 액화된다. 수증기 분자는 열을 지니고 있으며, 수증기가 액체 표면에 액화할 경우, 표면은 가열된다. 반면, 공기 중의 온도는 약간 감소하게 된다. <a href="/wiki/%EB%8C%80%EA%B8%B0" title="대기">대기</a>에서, 액화는 <a href="/wiki/%EA%B5%AC%EB%A6%84" title="구름">구름</a>, <a href="/wiki/%EC%95%88%EA%B0%9C" title="안개">안개</a>,등을 만들어낸다. <a href="/wiki/%EC%9D%B4%EC%8A%AC%EC%A0%90" title="이슬점">이슬점</a>이란 수증기가 응결하여 이슬이 생길 정도의 차가운 온도를 나타낸다.</p>
<p>또한, 표면의 온도가 대기의 이슬점 이하일 경우, 순수한 액화가 발생한다. 기체가 바로 고체가 되는 승화 역시 발생한다. <a href="/wiki/%EC%84%9C%EB%A6%AC" title="서리">서리</a>나 <a href="/wiki/%EB%88%88_(%EB%82%A0%EC%94%A8)" title="눈 (날씨)">눈</a>은 승화의 예이다.</p>
<h3><span class="mw-headline" id=".EA.B3.A0.EC.B0.B0">고찰</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%88%98%EC%A6%9D%EA%B8%B0&amp;action=edit&amp;section=4" title="부분 편집: 고찰">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>대기 중의 수증기의 양은 <a href="/wiki/%EB%8C%80%EA%B8%B0%EC%95%95" title="대기압">대기압</a> 및 <a href="/wiki/%EA%B8%B0%EC%98%A8" title="기온">기온</a>에 의해 영향을 받는다. <a href="/wiki/%EC%9D%B4%EC%8A%AC%EC%A0%90" title="이슬점">이슬점</a> 온도 및 <a class="mw-redirect" href="/wiki/%EC%83%81%EB%8C%80%EC%8A%B5%EB%8F%84" title="상대습도">상대습도</a>는 <a href="/wiki/%EB%AC%BC%EC%9D%98_%EC%88%9C%ED%99%98" title="물의 순환">물의 순환</a>에서 수증기 과정을 나타내는 지표이다. <a class="mw-redirect" href="/wiki/%ED%83%9C%EC%96%91%EA%B4%91" title="태양광">태양광</a>과 같은 에너지는 <a href="/wiki/%ED%95%B4%EC%88%98%EB%A9%B4" title="해수면">해수면</a>에서의 보다 많은 <a href="/wiki/%EA%B8%B0%ED%99%94" title="기화">기화</a>를 유발시키며, 이와 함께 산 정상 <a href="/wiki/%EC%96%BC%EC%9D%8C" title="얼음">얼음</a>의 보다 많은 <a href="/wiki/%EC%8A%B9%ED%99%94_(%ED%99%94%ED%95%99)" title="승화 (화학)">승화</a>를 유발한다. <a class="mw-redirect" href="/wiki/%EC%95%A1%ED%99%94" title="액화">액화</a>와 기화의 균형은 <a class="new" href="/w/index.php?title=%EC%88%98%EC%A6%9D%EA%B8%B0%EC%95%95&amp;action=edit&amp;redlink=1" title="수증기압 (없는 문서)">수증기압</a>이라는 수치로 표현된다.</p>
<p>공기 중의 포화수증기압은 기온 및 수증기 혼합 정도에 따라 차이를 보인다. 포화수증기압을 나타내는 여러 실험적인 공식이 있지만, 그 중에서 가장 자주 사용되는 것은 <a class="extiw" href="https://en.wikipedia.org/wiki/Goff-Gratch_equation" title="en:Goff-Gratch equation">Goff-Gratch 식</a>이다.</p>

<p>위 식은 대략 -50 ~ 102 °C 에서 유효하다. 하지만, 극도로 차가운 물의 수증기압을 측정하는 방법은 그다지 존재하지 않는다. 다른 공식 및 공식의 비교는 <a class="external text" href="http://cires.colorado.edu/~voemel/vp.html" rel="nofollow">여기</a>에서 찾을 수 있다.</p>
<p>반면에, 물이 끓는점에 이르면, 일반 대기 상태에서라면 상대습도에 관계 없이 항상 순수 기화가 발생하게 된다. 사람이 내뿜는 숨은 체온에서 거의 포화상태이며, 만약 추운 겨울이라면, 이러한 숨은 곧 응결해서, 작은 물방으로 이루어진 안개나 김과 같이 보이게 된다.</p>
<h2><span class="mw-headline" id=".EC.A7.80.EA.B5.AC_.EB.8C.80.EA.B8.B0.EA.B6.8C.EC.9D.98_.EC.88.98.EC.A6.9D.EA.B8.B0">지구 대기권의 수증기</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%88%98%EC%A6%9D%EA%B8%B0&amp;action=edit&amp;section=5" title="부분 편집: 지구 대기권의 수증기">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p><a href="/wiki/%EC%A7%80%EA%B5%AC_%EB%8C%80%EA%B8%B0%EA%B6%8C" title="지구 대기권">지구 대기권</a>에는 수증기가 조금 존재하긴 하지만, 환경에서는 매우 커다란 역할을 수행한다. 대부분의 수증기는 <a href="/wiki/%EB%8C%80%EB%A5%98%EA%B6%8C" title="대류권">대류권</a>에 존재한다. <a href="/wiki/%EC%A7%80%EA%B5%AC" title="지구">지구</a> 자체의 <a href="/wiki/%EC%98%A8%EC%8B%A4_%ED%9A%A8%EA%B3%BC" title="온실 효과">온실 효과</a>에 많은 영향을 주는 것 이외에도, 수증기는 <a href="/wiki/%EA%B5%AC%EB%A6%84" title="구름">구름</a>을 형성하며, 구름은 경우에 따라 지표면을 따뜻하게 하기도, 차갑게 하기도 한다. 일반적으로, 대기 중의 수증기는 <a href="/wiki/%EB%82%A0%EC%94%A8" title="날씨">날씨</a>에 의해 크게 영향을 받고, 또한 반대로 날씨에도 크게 영향을 주는데, 이러한 날씨는 <a href="/wiki/%EA%B8%B0%ED%9B%84" title="기후">기후</a>에 크게 영향을 받는다.</p>
<p><a href="/wiki/%EC%95%88%EA%B0%9C" title="안개">안개</a>나 <a href="/wiki/%EA%B5%AC%EB%A6%84" title="구름">구름</a>은 응결핵 주의로 응결이 일어나서 형성된다. 응결핵이 없다면, 훨씬 더 낮은 온도가 되어야만 응결이 발생한다. 지속적으로 응결이 일어날 경우, 구름내에 물방울이나 눈결정이 형성되며, 어느 정도의 질량에 이르면, <a href="/wiki/%EA%B0%95%EC%88%98" title="강수">강수</a> 현상이 일어난다.</p>
<p><a href="/wiki/%EB%8C%80%EB%A5%98%EA%B6%8C" title="대류권">대류권</a>에서 물 분자가 평균적으로 체류하는 시간은 일주일 가량이다. 강수 현상에 의해 물은 대기로부터 사라지며, <a href="/wiki/%EB%B0%94%EB%8B%A4" title="바다">바다</a>, <a href="/wiki/%ED%98%B8%EC%88%98" title="호수">호수</a>, <a href="/wiki/%EA%B0%95" title="강">강</a>, 식물의 증산 작용 등에서 일어나는 증발로 인해 대기로 보충된다.</p>
<p>수증기 농도는 <a class="new" href="/w/index.php?title=%EC%A0%88%EB%8C%80%EC%8A%B5%EB%8F%84&amp;action=edit&amp;redlink=1" title="절대습도 (없는 문서)">절대습도</a>와 퍼센트 <a class="mw-redirect" href="/wiki/%EC%83%81%EB%8C%80%EC%8A%B5%EB%8F%84" title="상대습도">상대습도</a>로 표현된다. 지구 전체의연간 평균 수증기 농도는 만약 순간적으로 응결할 경우 지구 전체 지표면을 25 mm의 두께로 덮을 정도이다. 하지만 지구 전체의 연간 강수량은 약 1 미터이며, 이는 물의 순환이 빠른 속도로 이루어진다는 것을 의미한다.</p>
<h3><span class="mw-headline" id=".EB.A0.88.EC.9D.B4.EB.8D.94_.EB.B0.8F_.EC.9C.84.EC.84.B1_.EC.98.81.EC.83.81">레이더 및 위성 영상</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%88%98%EC%A6%9D%EA%B8%B0&amp;action=edit&amp;section=6" title="부분 편집: 레이더 및 위성 영상">편집</a><span class="mw-editsection-bracket">]</span></span></h3>

<p>물 분자는 <a href="/wiki/%EA%B7%B9%EC%B4%88%EB%8B%A8%ED%8C%8C" title="극초단파">극초단파</a>(microwave) 및 <a class="mw-redirect" href="/wiki/%EB%9D%BC%EB%94%94%EC%98%A4%ED%8C%8C" title="라디오파">라디오파</a>를 흡수하는 특성이 있으므로, 대기중의 수분은 <a href="/wiki/%EB%A0%88%EC%9D%B4%EB%8D%94" title="레이더">레이더</a> 신호를 감소시킨다. 게다가 대기 수분은 전파를 반사시키거나 굴절시키며 그 정도는 수분이 액체냐 고체 상태냐에 따라 크게 달라진다.</p>
<p>일반적으로 레이더 신호는 <a class="mw-redirect" href="/wiki/%EB%8C%80%EA%B8%B0%EA%B6%8C" title="대기권">대기권</a>에서 나아감[[[{{{{{{〈〈††〉〉}}}}}}]]]에 따라 서서히 약해진다. 다른 주파수의 신호는 서로 다른 비율로 감소하며, 그러므로 어떠한 파장에 대해 투명한 공기 구성은 다른 파장에 대해서는 불투명하기도 하다. 방송 및 통신에 사용되는 라디오파 역시 이러한 특성을 지니고 있다.</p>
<p>수증기는 물의 다른 두 형태에 비해서 레이더 신호를 보다 덜 반사시킨다. 물방울이나 얼음 결정 형태의 수분은 기체 형태와는 달리 <a href="/wiki/%ED%94%84%EB%A6%AC%EC%A6%98" title="프리즘">프리즘</a> 역할을 수행하여 많이 반사시킨다.</p>
<p>위성 영상 비교는 대기중의 수증기양을 보여준다. 수증기는 지구 전역에 존재하지만, 균일하지 않다.</p>
<h3><span class="mw-headline" id=".EB.B2.88.EA.B0.9C_.ED.98.95.EC.84.B1">번개 형성</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%88%98%EC%A6%9D%EA%B8%B0&amp;action=edit&amp;section=7" title="부분 편집: 번개 형성">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>수증기는 대기 중의 <a href="/wiki/%EB%B2%88%EA%B0%9C" title="번개">번개</a> 생성에 매우 중요하다. 일반적으로 <a href="/wiki/%EA%B5%AC%EB%A6%84" title="구름">구름</a>은 <a href="/wiki/%EC%A7%80%EA%B5%AC_%EB%8C%80%EA%B8%B0%EA%B6%8C" title="지구 대기권">지구 대기권</a>에 존재하는 <a class="new" href="/w/index.php?title=%EC%A0%95%EC%A0%84%ED%95%98&amp;action=edit&amp;redlink=1" title="정전하 (없는 문서)">정전하</a>를 생성한다. 하지만, 구름이 얼마나 많은 전하를 지니고 있을 수 있는지를 나타내는 <a class="mw-redirect" href="/wiki/%EC%A0%84%EA%B8%B0_%EC%9A%A9%EB%9F%89" title="전기 용량">전기 용량</a>은 수증기의 양에 정비례한다.</p>
<p>수증기의 양은 공기 중의 <a href="/wiki/%EC%9C%A0%EC%A0%84%EC%9C%A8" title="유전율">유전율</a>과 직접적으로 연관되어 있다. 습도가 낮을 때에는 방전이 쉽고 빠르게 일어난다. 하지만 습도가 높을 때에는 방전이 덜 일어난다. 전기 용량 및 유전율은 엄청난 양의 전력을 지니는 번개를 형성하는 이유이다.</p>
<p>구름이 전하 생성을 시작하면, 대기 중의 수증기는 <a href="/wiki/%EC%A0%88%EC%97%B0%EC%B2%B4" title="절연체">절연체</a> 역할을 하며, 구름이 전하를 방전시키는 것을 막는다. 일정 시간이 지난 뒤, 구름이 보다 많은 전하를 생성한다면, 대기 중의 수증기에 의해 형성된 절연체가 파괴되며, 방전이 일어난다. 방전은 국지적으로 발생하며, 번개가 형성된다. 번개의 세기는 대기중의 유전율, 전기 용량, 구름의 전하 생성 능력에 직접적으로 비례한다.</p>
<p><a href="/wiki/%EB%B0%B4%EB%8D%94%EA%B7%B8%EB%9E%98%ED%94%84_%EB%B0%9C%EC%A0%84%EA%B8%B0" title="밴더그래프 발전기">밴더그래프 발전기</a>도 참조하기 바란다.</p>
<h2><span class="mw-headline" id=".EC.A7.80.EA.B5.AC_.EB.B0.96.EC.9D.98_.EC.88.98.EC.A6.9D.EA.B8.B0">지구 밖의 수증기</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%88%98%EC%A6%9D%EA%B8%B0&amp;action=edit&amp;section=8" title="부분 편집: 지구 밖의 수증기">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p><a href="/wiki/%ED%98%9C%EC%84%B1" title="혜성">혜성</a> 꼬리가 빛나는 이유는 대개 수증기 때문이다. 혜성이 <a href="/wiki/%ED%83%9C%EC%96%91" title="태양">태양</a>에 접근함에 따라, 혜성이 지니고 있는 얼음이 수증기로 <a href="/wiki/%EC%8A%B9%ED%99%94_(%ED%99%94%ED%95%99)" title="승화 (화학)">승화</a>하며, 이는 태양으로부터의 빛을 반사한다. 혜성과 태양의 거리를 알고 있다면, 혜성 꼬리가 빛나는 정도로부터 혜성이 지닌 수분의 양을 유추할 수 있다. 차갑고 먼 위치의 혜성이 빛나는 꼬리를 지녔다면, <a class="mw-redirect" href="/wiki/%EC%9D%BC%EC%82%B0%ED%99%94%ED%83%84%EC%86%8C" title="일산화탄소">일산화탄소</a> 승화란 것을 의미한다.</p>
<p><a href="/wiki/%ED%99%94%EC%84%B1" title="화성">화성</a>을 연구하는 과학자들은 물이 화성 주변을 움직이고 있다면 수증기 형태로 움직일 것이라는 가설을 세웠다. 화성의 대부분의 물은 북극지방에 얼음의 형태로 존재한다. 화성의 여름동안에, 이 얼음이 승화하면서 대규모의 계절성 폭풍을 유발하며, 이러한 현상은 엄청난 양의 물을 적도지방으로 수송한다는 설명이다.</p>
