<h1 class="firstHeading" id="firstHeading" lang="ko">수치해석학</h1>
<div class="mw-body-content" id="bodyContent">
<div id="siteSub">위키백과, 우리 모두의 백과사전.</div>

<p><b>수치해석학</b>(數値解析學, numerical analytics)은 <a href="/wiki/%ED%95%B4%EC%84%9D%ED%95%99_(%EC%88%98%ED%95%99)" title="해석학 (수학)">해석학</a> 문제에서 수치적인 근사값을 구하는 <a href="/wiki/%EC%95%8C%EA%B3%A0%EB%A6%AC%EC%A6%98" title="알고리즘">알고리즘</a>을 연구하는 학문이다.</p>
<p>가장 오래된 수치해석에 대한 수학적 기술은 <a href="/wiki/%EB%B0%94%EB%B9%8C%EB%A1%9C%EB%8B%88%EC%95%84" title="바빌로니아">바빌로니아</a> 사람들이 점토판에 <a href="/wiki/%EC%9C%A1%EC%8B%AD%EC%A7%84%EB%B2%95" title="육십진법">육십진법</a>으로 단위길이 사각형의 대각선의 길이인 <span><span class="mwe-math-mathml-inline mwe-math-mathml-a11y" style="display: none;"><math xmlns="http://www.w3.org/1998/Math/MathML">
<semantics>
<mrow class="MJX-TeXAtom-ORD">
<mstyle displaystyle="true" scriptlevel="0">
<mrow class="MJX-TeXAtom-ORD">
<msqrt>
<mn>2</mn>
</msqrt>
</mrow>
</mstyle>
</mrow>
<annotation encoding="application/x-tex">{\displaystyle {\sqrt {2}}}</annotation>
</semantics>
</math></span><img alt="{\sqrt {2}}" aria-hidden="true" class="mwe-math-fallback-image-inline" src="https://wikimedia.org/api/rest_v1/media/math/render/svg/b4afc1e27d418021bf10898eb44a7f5f315735ff" style="vertical-align: -0.671ex; width:3.119ex; height:3.009ex;"/></span>의 수치적 근사값을 구해놓은 것이다.<sup class="reference" id="cite_ref-1"><a href="#cite_note-1">[1]</a></sup> 삼각형의 한 변의 길이를 구하는 문제(<a href="/wiki/%EC%A0%9C%EA%B3%B1%EA%B7%BC" title="제곱근">제곱근</a>의 값을 구하는 문제)는 <a class="mw-redirect" href="/wiki/%ED%86%A0%EB%AA%A9" title="토목">토목</a>과 <a href="/wiki/%EA%B1%B4%EC%B6%95" title="건축">건축</a>등 여러 분야에서 매우 중요한 의미를 갖는다.<sup class="reference" id="cite_ref-2"><a href="#cite_note-2">[2]</a></sup></p>
<p>수치해석은 실생활에서 널리 사용된다. 바빌로니아 사람들이 루트 2의 근사값을 구한 예에서 볼 수 있듯이, 현대의 수치해석 역시 정확한 해를 구하지는 않는다. 왜냐하면 정확한 해를 구하는 것이 실제로는 불가능한 경우가 많기 때문이다. 그대신 대다수의 경우, 수치해석에서는 합리적인 수준의 오차를 갖는 근사값을 구하는 것에 집중한다.</p>
<p>수치해석의 응용분야는 일반적으로 공학과 물리학이다. 하지만 21세기에 들어서면서 그 응용분야가 확대되어, 생명공학과 과학적인 계산을 적용한 예술분야에서도 사용된다. <a class="mw-redirect" href="/wiki/%EC%83%81%EB%AF%B8%EB%B6%84_%EB%B0%A9%EC%A0%95%EC%8B%9D" title="상미분 방정식">상미분 방정식</a>은 행성들의 움직임과, 포트폴리오 관리의 <a href="/wiki/%EC%B5%9C%EC%A0%81%ED%99%94_%EB%AC%B8%EC%A0%9C" title="최적화 문제">최적화</a> 등에 이용되며, 선형대수학은 데이터 분석에 중요하게 쓰인다. <a class="new" href="/w/index.php?title=%ED%99%95%EB%A5%A0%EB%AF%B8%EB%B6%84%EB%B0%A9%EC%A0%95%EC%8B%9D&amp;action=edit&amp;redlink=1" title="확률미분방정식 (없는 문서)">확률미분방정식</a>과 <a href="/wiki/%EB%A7%88%EB%A5%B4%EC%BD%94%ED%94%84_%EC%97%B0%EC%87%84" title="마르코프 연쇄">마르코프 연쇄</a> 또한 의약과 생명분야에서 살아있는 세포에 대한 시뮬레이션을 하기위한 필수항목이다.</p>
<p>컴퓨터의 발달전 수치해석은 크게 인쇄된 <a href="/wiki/%EB%B3%B4%EA%B0%84%EB%B2%95" title="보간법">보간법</a> 표와 손으로 하는 반복계산이 주를 이루었다. 이는 20세기 중반 컴퓨터의 발달로 대체 되었지만 그럼에도 불구하고 보간법 알고리즘들은 <a class="mw-redirect" href="/wiki/%EB%AF%B8%EB%B6%84_%EB%B0%A9%EC%A0%95%EC%8B%9D" title="미분 방정식">미분 방정식</a>의 해를 구하는 소프트웨어의 일부분으로써 여전히 사용되고 있다.</p>

<h2><span class="mw-headline" id=".EC.98.88.EC.A0.9C">예제</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%88%98%EC%B9%98%ED%95%B4%EC%84%9D%ED%95%99&amp;action=edit&amp;section=1" title="부분 편집: 예제">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<h3><span class="mw-headline" id=".EB.AF.B8.EB.B6.84_.EB.B0.A9.EC.A0.95.EC.8B.9D">미분 방정식</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%88%98%EC%B9%98%ED%95%B4%EC%84%9D%ED%95%99&amp;action=edit&amp;section=2" title="부분 편집: 미분 방정식">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>수치적으로 미분 방정식을 푼다는 것은 주어진 미분 방정식의 근사해를 찾는다는 것을 의미한다. 이 때는 다음과 같은 전제 조건이 필요하다.</p>
<p>수치 해석적인 방법으로 미분 방정식을 푸는 것에는 다음과 같은 것들이 있다.</p>
<ul>
<li><a href="/wiki/%EC%98%A4%EC%9D%BC%EB%9F%AC_%EB%B0%A9%EB%B2%95" title="오일러 방법">오일러 방법</a>(Euler's Methods)</li>
<li><a href="/wiki/%EB%A3%BD%EA%B2%8C-%EC%BF%A0%ED%83%80_%EB%B0%A9%EB%B2%95" title="룽게-쿠타 방법">룽게-쿠타 방법</a>(Runge-Kutta Methods)</li>
</ul>
<h3><span class="mw-headline" id=".ED.8E.B8.EB.AF.B8.EB.B6.84_.EB.B0.A9.EC.A0.95.EC.8B.9D">편미분 방정식</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%88%98%EC%B9%98%ED%95%B4%EC%84%9D%ED%95%99&amp;action=edit&amp;section=3" title="부분 편집: 편미분 방정식">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>상위의 미분 방정식과 비슷한 원리이지만, 차원이 더 커지므로 다른 방식의 수치적 방법을 이용해야 풀수 있다. 아래에는 두개의 수치적 방법이 있다.</p>
<ol>
<li>유한차분법 (FDM : Finite Difference Method)</li>
<li><a href="/wiki/%EC%9C%A0%ED%95%9C%EC%9A%94%EC%86%8C%EB%B2%95" title="유한요소법">유한요소법</a> (FEM : Finite Element Method)</li>
</ol>
