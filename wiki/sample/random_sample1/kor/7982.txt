<p><b>수쿰비오스 주</b>(<span style="font-size: smaller;"><a href="/wiki/%EC%8A%A4%ED%8E%98%EC%9D%B8%EC%96%B4" title="스페인어">스페인어</a>:</span> <span lang="es" xml:lang="es">Provincia de Sucumbíos</span>)는 <a href="/wiki/%EC%97%90%EC%BD%B0%EB%8F%84%EB%A5%B4" title="에콰도르">에콰도르</a> 북동부에 위치한 주로 주도는 <a href="/wiki/%EB%88%84%EC%97%90%EB%B0%94%EB%A1%9C%ED%95%98" title="누에바로하">누에바로하</a>이며 면적은 18,084.42㎢, 인구는 176,472명(2010년 기준), 인구 밀도는 9.8명/㎢이다. 1989년 2월 11일에 신설되었으며 7개 지방 자치체를 관할한다. 북쪽으로는 <a href="/wiki/%EC%BD%9C%EB%A1%AC%EB%B9%84%EC%95%84" title="콜롬비아">콜롬비아</a>, 동쪽으로는 <a href="/wiki/%ED%8E%98%EB%A3%A8" title="페루">페루</a>와 국경을 접하며 남쪽으로는 <a href="/wiki/%EC%98%A4%EB%A0%88%EC%95%BC%EB%82%98_%EC%A3%BC" title="오레야나 주">오레야나 주</a>와 <a href="/wiki/%EB%82%98%ED%8F%AC_%EC%A3%BC" title="나포 주">나포 주</a>, 서쪽으로는 <a href="/wiki/%ED%94%BC%EC%B9%9C%EC%B0%A8_%EC%A3%BC" title="피친차 주">피친차 주</a>, <a href="/wiki/%EC%9E%84%EB%B0%94%EB%B6%80%EB%9D%BC_%EC%A3%BC" title="임바부라 주">임바부라 주</a>, <a href="/wiki/%EC%B9%B4%EB%A5%B4%EC%B9%98_%EC%A3%BC" title="카르치 주">카르치 주</a>와 접한다.</p>
