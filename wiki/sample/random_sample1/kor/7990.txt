<h1 class="firstHeading" id="firstHeading" lang="ko">수학</h1>
<div class="mw-body-content" id="bodyContent">
<div id="siteSub">위키백과, 우리 모두의 백과사전.</div>
<div id="contentSub"></div>
<div class="mw-jump" id="jump-to-nav">
					이동:					<a href="#mw-head">둘러보기</a>, 					<a href="#p-search">검색</a>
</div>
<div class="mw-content-ltr" dir="ltr" id="mw-content-text" lang="ko"><script>function mfTempOpenSection(id){var block=document.getElementById("mf-section-"+id);block.className+=" open-block";block.previousSibling.className+=" open-block";}</script><p><b>수학</b>(數學)은 <a href="/wiki/%EC%96%91" title="양">양</a>, <a href="/wiki/%EA%B5%AC%EC%A1%B0" title="구조">구조</a>, <a href="/wiki/%EA%B3%B5%EA%B0%84" title="공간">공간</a>, <a href="/wiki/%EB%B3%80%ED%99%94" title="변화">변화</a> 등 <a href="/wiki/%EA%B0%9C%EB%85%90" title="개념">개념</a>을 다루는 <a href="/wiki/%ED%95%99%EB%AC%B8" title="학문">학문</a>이다. 현대 수학은 형식 논리를 이용해서 <a href="/wiki/%EA%B3%B5%EB%A6%AC" title="공리">공리</a>로 구성된 추상적 구조를 연구하는 학문으로 여겨지기도 한다. 수학은 그 구조와 발전 과정에서는 <a href="/wiki/%EC%9E%90%EC%97%B0%EA%B3%BC%ED%95%99" title="자연과학">자연과학</a>에 속하는 <a href="/wiki/%EB%AC%BC%EB%A6%AC%ED%95%99" title="물리학">물리학</a>을 비롯한 다른 학문들과 깊은 연관을 맺고 있으나, 여느 과학의 분야들과는 달리, 자연계에서 관측되지 않는 개념들에 대해서까지 <a href="/wiki/%EC%9D%B4%EB%A1%A0" title="이론">이론</a>을 일반화 및 <a class="mw-disambig" href="/wiki/%EC%B6%94%EC%83%81%ED%99%94" title="추상화">추상화</a>시킬 수 있다는 차이가 있다. <a href="/wiki/%EC%88%98%ED%95%99%EC%9E%90" title="수학자">수학자</a>들은 그러한 개념들에 대해서 <a href="/wiki/%EC%B6%94%EC%B8%A1" title="추측">추측</a>을 하고, 적절하게 선택된 <a href="/wiki/%EC%A0%95%EC%9D%98" title="정의">정의</a>와 공리로부터의 엄밀한 <a href="/wiki/%EC%97%B0%EC%97%AD" title="연역">연역</a>을 통해서 추측들의 진위를 파악한다.<sup class="reference" id="cite_ref-1"><a href="#cite_note-1">[1]</a></sup></p>
<p>수학은 숫자 세기, <a href="/wiki/%EA%B3%84%EC%82%B0" title="계산">계산</a>, <a href="/wiki/%EC%B8%A1%EC%A0%95" title="측정">측정</a> 및 물리적 대상의 모양과 움직임을 추상화하고, 이에 <a class="mw-disambig" href="/wiki/%EB%85%BC%EB%A6%AC" title="논리">논리</a>적 <a href="/wiki/%EC%B6%94%EB%A1%A0" title="추론">추론</a>을 적용하여 나타났다. 이런 기본 개념들은 <a href="/wiki/%EA%B3%A0%EB%8C%80_%EC%9D%B4%EC%A7%91%ED%8A%B8" title="고대 이집트">고대 이집트</a>, <a href="/wiki/%EB%A9%94%EC%86%8C%ED%8F%AC%ED%83%80%EB%AF%B8%EC%95%84" title="메소포타미아">메소포타미아</a>, <a href="/wiki/%EA%B3%A0%EB%8C%80_%EC%9D%B8%EB%8F%84" title="고대 인도">고대 인도</a>, 고대 중국 및 <a href="/wiki/%EA%B3%A0%EB%8C%80_%EA%B7%B8%EB%A6%AC%EC%8A%A4" title="고대 그리스">고대 그리스</a>의 수학책에서 찾아볼 수 있으며, <a class="mw-redirect" href="/wiki/%EC%9C%A0%ED%81%B4%EB%A6%AC%EB%93%9C%EC%9D%98_%EC%9B%90%EB%A1%A0" title="유클리드의 원론">유클리드의 원론</a>에서는 엄밀한 논증이 발견된다. 이런 발전은 그 뒤로도 계속되어, <a href="/wiki/16%EC%84%B8%EA%B8%B0" title="16세기">16세기</a>의 <a href="/wiki/%EB%A5%B4%EB%84%A4%EC%83%81%EC%8A%A4" title="르네상스">르네상스</a>에 이르러서는 수학적 발전과 <a href="/wiki/%EA%B3%BC%ED%95%99%EC%A0%81_%EB%B0%A9%EB%B2%95" title="과학적 방법">과학적 방법</a>들의 <a class="mw-redirect" href="/wiki/%EC%83%81%ED%98%B8_%EC%9E%91%EC%9A%A9" title="상호 작용">상호 작용</a>이 일어나, 혁명적인 연구들이 진행되며 인류 문명에 큰 영향을 미치게 되었고, 이는 현재까지도 계속되고 있다.<sup class="reference" id="cite_ref-2"><a href="#cite_note-2">[2]</a></sup></p>
<p>오늘날 수학은 <a href="/wiki/%EC%9E%90%EC%97%B0%EA%B3%BC%ED%95%99" title="자연과학">자연과학</a>, <a href="/wiki/%EA%B3%B5%ED%95%99" title="공학">공학</a>, <a href="/wiki/%EC%9D%98%ED%95%99" title="의학">의학</a>뿐만 아니라, <a href="/wiki/%EA%B2%BD%EC%A0%9C%ED%95%99" title="경제학">경제학</a> 등의 <a href="/wiki/%EC%82%AC%ED%9A%8C%EA%B3%BC%ED%95%99" title="사회과학">사회과학</a>에서도 중요한 도구로서 사용된다. 수학을 이런 분야들에 적용한 <a href="/wiki/%EC%9D%91%EC%9A%A9%EC%88%98%ED%95%99" title="응용수학">응용수학</a>은 그 결과로써 수학 자체의 발전을 이끌고 새로운 분야들을 낳았다. 응용이 아닌 수학 자체의 아름다움과 재미를 추구하며 연구하는 것을 <a href="/wiki/%EC%88%9C%EC%88%98%EC%88%98%ED%95%99" title="순수수학">순수수학</a>이라 하는데, 긴 시간이 지난 뒤에 순수수학적 연구를 다른 분야에 응용할 방법이 발견된 경우도 많았다고 한다.<sup class="reference" id="cite_ref-3"><a href="#cite_note-3">[3]</a></sup></p>
<p>대부분 자료를 보면, "mathematics"는 "수리적인"이라는 뜻을 가진 <a href="/wiki/%EB%9D%BC%ED%8B%B4%EC%96%B4" title="라틴어">라틴어</a> mathmaticus와 <a href="/wiki/%EA%B7%B8%EB%A6%AC%EC%8A%A4%EC%96%B4" title="그리스어">그리스어</a> mathematikos에서 유래되었거나, "학식있는"을 뜻하는 mathema와 "배우다"를 뜻하는 manthanein에서 유래되었다고 한다.</p>

<h2><span class="mw-headline" id=".EC.97.AD.EC.82.AC">역사</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%88%98%ED%95%99&amp;action=edit&amp;section=1" title="부분 편집: 역사">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<div class="rellink relarticle mainarticle"><img alt="" data-file-height="128" data-file-width="128" height="16" src="//upload.wikimedia.org/wikipedia/commons/thumb/e/ec/Crystal_Clear_app_xmag.svg/16px-Crystal_Clear_app_xmag.svg.png" srcset="//upload.wikimedia.org/wikipedia/commons/thumb/e/ec/Crystal_Clear_app_xmag.svg/24px-Crystal_Clear_app_xmag.svg.png 1.5x, //upload.wikimedia.org/wikipedia/commons/thumb/e/ec/Crystal_Clear_app_xmag.svg/32px-Crystal_Clear_app_xmag.svg.png 2x" width="16"/> 이 부분의 본문은 <a class="mw-redirect" href="/wiki/%EC%88%98%ED%95%99%EC%9D%98_%EC%97%AD%EC%82%AC" title="수학의 역사">수학의 역사</a>입니다.</div>
<p>수학은 기원전 600년 경에 살았던 <a href="/wiki/%ED%83%88%EB%A0%88%EC%8A%A4" title="탈레스">탈레스</a>로부터 시작됐다. 하지만 탈레스가 태어나기 전에도 수학을 연구한 사람이 있을 수도 있기 때문에, 인류의 역사와 더불어 시작되었다고 할 수 있다. 교역•분배•과세 등의 인류의 사회 생활에 필요한 모든 계산을 수학이 담당해 왔고, 농경 생활에 필수적인 천문 관측과 달력의 제정, 토지의 측량 또한 수학이 직접적으로 관여한 분야이다. 고대 수학을 크게 발전시킨 나라로는 <a href="/wiki/%EC%9D%B4%EC%A7%91%ED%8A%B8" title="이집트">이집트</a>, <a href="/wiki/%EC%9D%B8%EB%8F%84" title="인도">인도</a>, <a href="/wiki/%EA%B7%B8%EB%A6%AC%EC%8A%A4" title="그리스">그리스</a>, <a href="/wiki/%EC%A4%91%EA%B5%AD" title="중국">중국</a> 등이 있다. 그 중에서도 <a href="/wiki/%EA%B7%B8%EB%A6%AC%EC%8A%A4" title="그리스">그리스</a>는 처음으로 수학의 <a href="/wiki/%EB%B0%A9%EC%A0%95%EC%8B%9D" title="방정식">방정식</a>에서 <a href="/wiki/%EB%B3%80%EC%88%98" title="변수">변수</a>를 <a href="/wiki/%EB%AC%B8%EC%9E%90" title="문자">문자</a>로 쓴 나라이다.</p>
<p><a href="/wiki/%ED%95%9C%EA%B5%AD" title="한국">한국</a>의 수학은 약 1,500년 전부터 기록으로 보이기 시작한다. <a href="/wiki/%EC%8B%A0%EB%9D%BC" title="신라">신라</a> 시대에 수학을 가르쳤으며, 탈레스가 최초로 발견한 <a href="/wiki/%EC%9D%BC%EC%8B%9D" title="일식">일식</a>과 <a href="/wiki/%EC%9B%94%EC%8B%9D" title="월식">월식</a>을 예측할 정도로 발달했다. <a href="/wiki/%EC%A1%B0%EC%84%A0" title="조선">조선</a> 시대에 <a href="/wiki/%ED%9B%88%EB%AF%BC%EC%A0%95%EC%9D%8C" title="훈민정음">훈민정음</a>을 창제한 <a class="mw-redirect" href="/wiki/%EC%84%B8%EC%A2%85_%EB%8C%80%EC%99%95" title="세종 대왕">세종 대왕</a>은 <a href="/wiki/%EC%A7%91%ED%98%84%EC%A0%84" title="집현전">집현전</a> 학자들에게 수학 연구를 명하는 등, 조선의 수학 수준을 향상시키기 위해서 많은 노력을 기울였다. 하지만 <a href="/wiki/%EC%9E%84%EC%A7%84%EC%99%9C%EB%9E%80" title="임진왜란">임진왜란</a>으로 많은 서적들이 불타고, <a href="/wiki/%EC%B2%9C%EB%AC%B8%ED%95%99" title="천문학">천문학</a> 분야에서 큰 손실을 입었다. 조선 후기의 한국의 수학은 <a class="mw-redirect" href="/wiki/%EC%8B%A4%ED%95%99%EC%9E%90" title="실학자">실학자</a>들을 중심으로 다시 발전하였고, 새로운 결과도 성취되었다. 그렇게 해서 한국은 수학이 현재까지 발전하게 되었다.</p>
<h2><span class="mw-headline" id=".EC.84.B8.EB.B6.80_.EB.B6.84.EC.95.BC">세부 분야</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%88%98%ED%95%99&amp;action=edit&amp;section=2" title="부분 편집: 세부 분야">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<div class="thumb tright">
<div class="thumbinner" style="width:222px;"><a class="image" href="/wiki/%ED%8C%8C%EC%9D%BC:Abacus_6.png"><img alt="" class="thumbimage" data-file-height="145" data-file-width="247" height="129" src="//upload.wikimedia.org/wikipedia/commons/thumb/a/af/Abacus_6.png/220px-Abacus_6.png" srcset="//upload.wikimedia.org/wikipedia/commons/a/af/Abacus_6.png 1.5x" width="220"/></a>
<div class="thumbcaption">
<div class="magnify"><a class="internal" href="/wiki/%ED%8C%8C%EC%9D%BC:Abacus_6.png" title="실제 크기로"></a></div>
<a href="/wiki/%EC%A3%BC%ED%8C%90" title="주판">주판</a>은 고대로부터 계산 도구로 사용되어왔다.</div>
</div>
</div>
<p>수학의 각 분야들은 상업에 필요한 계산을 하기 위해, 숫자들의 관계를 이해하기 위해, 토지를 측량하기 위해, 그리고 <a href="/wiki/%EC%B2%9C%EB%AC%B8%ED%95%99" title="천문학">천문학</a>적 사건들을 예견하기 위해 발전되어왔다. 이 네 가지 목적은 대략적으로 수학이 다루는 대상인 양, 구조, 공간 및 변화에 대응되며, 이들을 다루는 수학의 분야를 각각 <a href="/wiki/%EC%82%B0%EC%88%A0" title="산술">산술</a>, <a href="/wiki/%EB%8C%80%EC%88%98%ED%95%99" title="대수학">대수학</a>, <a href="/wiki/%EA%B8%B0%ED%95%98%ED%95%99" title="기하학">기하학</a>, <a href="/wiki/%ED%95%B4%EC%84%9D%ED%95%99_(%EC%88%98%ED%95%99)" title="해석학 (수학)">해석학</a>이라 한다. 또한 이 밖에도 근대 이후에 나타난 <a href="/wiki/%EC%88%98%ED%95%99%EA%B8%B0%EC%B4%88%EB%A1%A0" title="수학기초론">수학기초론</a>과 <a href="/wiki/%EC%9D%B4%EC%82%B0%EC%88%98%ED%95%99" title="이산수학">이산수학</a> 및 <a href="/wiki/%EC%9D%91%EC%9A%A9%EC%88%98%ED%95%99" title="응용수학">응용수학</a> 등이 있다.</p>
<h3><span class="mw-headline" id=".EC.82.B0.EC.88.A0">산술</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%88%98%ED%95%99&amp;action=edit&amp;section=3" title="부분 편집: 산술">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<div class="rellink relarticle mainarticle"><img alt="" data-file-height="128" data-file-width="128" height="16" src="//upload.wikimedia.org/wikipedia/commons/thumb/e/ec/Crystal_Clear_app_xmag.svg/16px-Crystal_Clear_app_xmag.svg.png" srcset="//upload.wikimedia.org/wikipedia/commons/thumb/e/ec/Crystal_Clear_app_xmag.svg/24px-Crystal_Clear_app_xmag.svg.png 1.5x, //upload.wikimedia.org/wikipedia/commons/thumb/e/ec/Crystal_Clear_app_xmag.svg/32px-Crystal_Clear_app_xmag.svg.png 2x" width="16"/> 이 부분의 본문은 <a href="/wiki/%EC%82%B0%EC%88%A0" title="산술">산술</a>입니다.</div>
<p>산술은 <a href="/wiki/%EC%9E%90%EC%97%B0%EC%88%98" title="자연수">자연수</a>와 <a href="/wiki/%EC%A0%95%EC%88%98" title="정수">정수</a> 및 이에 대한 <a class="mw-redirect" href="/wiki/%EC%82%AC%EC%B9%99%EC%97%B0%EC%82%B0" title="사칙연산">사칙연산</a>에 대한 연구로서 시작했다. <a class="mw-redirect" href="/wiki/%EC%88%98%EB%A1%A0" title="수론">수론</a>은 이런 주제들을 보다 깊게 다루는 학문으로, 그 결과로는 <a href="/wiki/%ED%8E%98%EB%A5%B4%EB%A7%88%EC%9D%98_%EB%A7%88%EC%A7%80%EB%A7%89_%EC%A0%95%EB%A6%AC" title="페르마의 마지막 정리">페르마의 마지막 정리</a> 등이 유명하다. 또한 <a href="/wiki/%EC%8C%8D%EB%91%A5%EC%9D%B4_%EC%86%8C%EC%88%98_%EC%B6%94%EC%B8%A1" title="쌍둥이 소수 추측">쌍둥이 소수 추측</a>과 <a class="mw-redirect" href="/wiki/%EA%B3%A8%ED%8A%B8%EB%B0%94%ED%9D%90_%EC%B6%94%EC%B8%A1" title="골트바흐 추측">골트바흐 추측</a> 등을 비롯해 오랜 세월 동안 해결되지 않고 남아있는 문제들도 여럿 있다.</p>
<p>수의 체계가 보다 발전하면서, 정수의 집합을 <a href="/wiki/%EC%9C%A0%EB%A6%AC%EC%88%98" title="유리수">유리수</a>의 집합의 <a href="/wiki/%EB%B6%80%EB%B6%84%EC%A7%91%ED%95%A9" title="부분집합">부분집합</a>으로 여기게 되었다. 또한 유리수의 집합은 <a href="/wiki/%EC%8B%A4%EC%88%98" title="실수">실수</a>의 집합의 부분집합이며, 이는 또다시 <a href="/wiki/%EB%B3%B5%EC%86%8C%EC%88%98" title="복소수">복소수</a> 집합의 일부분으로 볼 수 있다. 여기에서 더 나아가면 <a href="/wiki/%EC%82%AC%EC%9B%90%EC%88%98" title="사원수">사원수</a>와 <a href="/wiki/%ED%8C%94%EC%9B%90%EC%88%98" title="팔원수">팔원수</a> 등의 개념을 생각할 수도 있다. 이와는 약간 다른 방향으로, 자연수를 무한대까지 세어나간다는 개념을 형식화하여 <a href="/wiki/%EC%88%9C%EC%84%9C%EC%88%98" title="순서수">순서수</a>의 개념을 얻으며, 집합의 크기 비교를 이용하여 무한대를 다루기 위한 또다른 방법으로는 <a href="/wiki/%EA%B8%B0%EC%88%98_(%EC%88%98%ED%95%99)" title="기수 (수학)">기수</a>의 개념도 있다.</p>

<h3><span class="mw-headline" id=".EB.8C.80.EC.88.98.ED.95.99">대수학</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%88%98%ED%95%99&amp;action=edit&amp;section=4" title="부분 편집: 대수학">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<div class="rellink relarticle mainarticle"><img alt="" data-file-height="128" data-file-width="128" height="16" src="//upload.wikimedia.org/wikipedia/commons/thumb/e/ec/Crystal_Clear_app_xmag.svg/16px-Crystal_Clear_app_xmag.svg.png" srcset="//upload.wikimedia.org/wikipedia/commons/thumb/e/ec/Crystal_Clear_app_xmag.svg/24px-Crystal_Clear_app_xmag.svg.png 1.5x, //upload.wikimedia.org/wikipedia/commons/thumb/e/ec/Crystal_Clear_app_xmag.svg/32px-Crystal_Clear_app_xmag.svg.png 2x" width="16"/> 이 부분의 본문은 <a href="/wiki/%EB%8C%80%EC%88%98%ED%95%99" title="대수학">대수학</a>입니다.</div>
<p>수 대신 문자를 써서 문제해결을 쉽게 하는 것과, 마찬가지로 수학적 법칙을 일반적이고 간명하게 나타내는 것을 포함한다. 고전대수학은 대수방정식 및 연립방정식의 해법에서 시작하여 군, 환, 체 등의 추상대수학을 거쳐 현대에 와서는 대수계의 구조를 보는 것을 중심으로 하는 선형대수학으로 전개되었다. 수의 집합이나 함수와 같은 많은 수학적 대상들은 내재적인 구조를 보인다. 이러한 대상들의 구조적 특성들이 <a href="/wiki/%EA%B5%B0%EB%A1%A0" title="군론">군론</a>, <a href="/wiki/%ED%99%98%EB%A1%A0" title="환론">환론</a>, <a class="mw-redirect" href="/wiki/%EC%B2%B4%EB%A1%A0" title="체론">체론</a> 그리고 그 외의 수많은 대수적 구조들을 연구하면서 다루어지며, 그것들 하나하나가 내재적 구조를 지닌 수학적 대상이다. 이 분야에서 중요한 개념은 벡터, <a href="/wiki/%EB%B2%A1%ED%84%B0_%EA%B3%B5%EA%B0%84" title="벡터 공간">벡터 공간</a>으로의 일반화, 그리고 <a href="/wiki/%EC%84%A0%ED%98%95%EB%8C%80%EC%88%98%ED%95%99" title="선형대수학">선형대수학</a>에서의 지식들이다. 벡터의 연구에는 <a href="/wiki/%EC%82%B0%EC%88%A0" title="산술">산술</a>, <a href="/wiki/%EB%8C%80%EC%88%98%ED%95%99" title="대수학">대수</a>, <a class="mw-redirect" href="/wiki/%EA%B8%B0%ED%95%98" title="기하">기하</a>라는 수학의 중요한 세개의 분야가 조합되어 있다. <a href="/wiki/%EB%B2%A1%ED%84%B0_%EB%AF%B8%EC%A0%81%EB%B6%84%ED%95%99" title="벡터 미적분학">벡터 미적분학</a>은 여기에 해석학의 영역이 추가된다. 텐서 미적분학은 대칭성과 회전축의 영향 아래에서 벡터의 움직임을 연구한다. 눈금없는 자와 컴퍼스와 관련된 많은 고대의 미해결 문제들이 <a href="/wiki/%EA%B0%88%EB%A3%A8%EC%95%84_%EC%9D%B4%EB%A1%A0" title="갈루아 이론">갈루아 이론</a>을 사용하여 비로소 해결되었다.</p>

<h3><span class="mw-headline" id=".EA.B8.B0.ED.95.98.ED.95.99">기하학</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%88%98%ED%95%99&amp;action=edit&amp;section=5" title="부분 편집: 기하학">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<div class="rellink relarticle mainarticle"><img alt="" data-file-height="128" data-file-width="128" height="16" src="//upload.wikimedia.org/wikipedia/commons/thumb/e/ec/Crystal_Clear_app_xmag.svg/16px-Crystal_Clear_app_xmag.svg.png" srcset="//upload.wikimedia.org/wikipedia/commons/thumb/e/ec/Crystal_Clear_app_xmag.svg/24px-Crystal_Clear_app_xmag.svg.png 1.5x, //upload.wikimedia.org/wikipedia/commons/thumb/e/ec/Crystal_Clear_app_xmag.svg/32px-Crystal_Clear_app_xmag.svg.png 2x" width="16"/> 이 부분의 본문은 <a href="/wiki/%EA%B8%B0%ED%95%98%ED%95%99" title="기하학">기하학</a>입니다.</div>
<p>공간에 대한 연구는 <a href="/wiki/%EA%B8%B0%ED%95%98%ED%95%99" title="기하학">기하학</a>에서 시작되었고, 특히 <a href="/wiki/%EC%9C%A0%ED%81%B4%EB%A6%AC%EB%93%9C_%EA%B8%B0%ED%95%98%ED%95%99" title="유클리드 기하학">유클리드 기하학</a>에서 비롯되었다. <a href="/wiki/%EC%82%BC%EA%B0%81%EB%B2%95" title="삼각법">삼각법</a>은 공간과 수들을 결합하였고, 잘 알려진 <a href="/wiki/%ED%94%BC%ED%83%80%EA%B3%A0%EB%9D%BC%EC%8A%A4%EC%9D%98_%EC%A0%95%EB%A6%AC" title="피타고라스의 정리">피타고라스의 정리</a>를 포함한다. 현대에 와서 공간에 대한 연구는, 이러한 개념들은 더 높은 차원의 기하학을 다루기 위해 <a href="/wiki/%EB%B9%84%EC%9C%A0%ED%81%B4%EB%A6%AC%EB%93%9C_%EA%B8%B0%ED%95%98%ED%95%99" title="비유클리드 기하학">비유클리드 기하학</a>(상대성이론에서 핵심적인 역할을 함)과 <a href="/wiki/%EC%9C%84%EC%83%81%EC%88%98%ED%95%99" title="위상수학">위상수학</a>으로 일반화되었다. 수론과 공간에 대한 이해는 모두 해석 기하학, <a href="/wiki/%EB%AF%B8%EB%B6%84%EA%B8%B0%ED%95%98%ED%95%99" title="미분기하학">미분기하학</a>, <a href="/wiki/%EB%8C%80%EC%88%98%EA%B8%B0%ED%95%98%ED%95%99" title="대수기하학">대수기하학</a>에 중요한 역할을 한다. <a href="/wiki/%EB%A6%AC_%EA%B5%B0" title="리 군">리 군</a>도 공간과 구조, 변화를 다루는데 사용된다. 위상수학은 20세기 수학의 다양한 지류속에서 괄목할만한 성장을 한 분야이며, <a href="/wiki/%ED%91%B8%EC%95%B5%EC%B9%B4%EB%A0%88_%EC%B6%94%EC%B8%A1" title="푸앵카레 추측">푸앵카레 추측</a>과 인간에 의해서 증명되지 못하고 오직 컴퓨터로만 증명된 <a href="/wiki/4%EC%83%89%EC%A0%95%EB%A6%AC" title="4색정리">4색정리</a>를 포함한다.</p>

<h3><span class="mw-headline" id=".ED.95.B4.EC.84.9D.ED.95.99">해석학</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%88%98%ED%95%99&amp;action=edit&amp;section=6" title="부분 편집: 해석학">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<div class="rellink relarticle mainarticle"><img alt="" data-file-height="128" data-file-width="128" height="16" src="//upload.wikimedia.org/wikipedia/commons/thumb/e/ec/Crystal_Clear_app_xmag.svg/16px-Crystal_Clear_app_xmag.svg.png" srcset="//upload.wikimedia.org/wikipedia/commons/thumb/e/ec/Crystal_Clear_app_xmag.svg/24px-Crystal_Clear_app_xmag.svg.png 1.5x, //upload.wikimedia.org/wikipedia/commons/thumb/e/ec/Crystal_Clear_app_xmag.svg/32px-Crystal_Clear_app_xmag.svg.png 2x" width="16"/> 이 부분의 본문은 <a href="/wiki/%ED%95%B4%EC%84%9D%ED%95%99_(%EC%88%98%ED%95%99)" title="해석학 (수학)">해석학 (수학)</a>입니다.</div>
<p>변화에 대한 이해와 묘사는 <a href="/wiki/%EC%9E%90%EC%97%B0%EA%B3%BC%ED%95%99" title="자연과학">자연과학</a>에 있어서 일반적인 주제이며, <a href="/wiki/%EB%AF%B8%EC%A0%81%EB%B6%84%ED%95%99" title="미적분학">미적분학</a>은 변화를 탐구하는 강력한 도구로서 발전되었다. <a href="/wiki/%ED%95%A8%EC%88%98" title="함수">함수</a>는 변화하는 양을 묘사함에 있어서 중추적인 개념으로써 떠오르게 된다. 실수와 실변수로 구성된 함수의 엄밀한 탐구가 <a href="/wiki/%EC%8B%A4%ED%95%B4%EC%84%9D%ED%95%99" title="실해석학">실해석학</a>이라는 분야로 알려지게 되었고, <a href="/wiki/%EB%B3%B5%EC%86%8C%EC%88%98" title="복소수">복소수</a>에 대한 이와 같은 탐구 분야는 <a href="/wiki/%EB%B3%B5%EC%86%8C%ED%95%B4%EC%84%9D%ED%95%99" title="복소해석학">복소해석학</a>이라고 한다. <a href="/wiki/%ED%95%A8%EC%88%98%ED%95%B4%EC%84%9D%ED%95%99" title="함수해석학">함수해석학</a>은 함수의 공간(특히 무한차원)의 탐구에 주목한다. 함수해석학의 많은 응용분야 중 하나가 <a href="/wiki/%EC%96%91%EC%9E%90%EC%97%AD%ED%95%99" title="양자역학">양자역학</a>이다. 많은 문제들이 자연스럽게 양과 그 양의 변화율의 관계로 귀착되고, 이러한 문제들이 <a href="/wiki/%EB%AF%B8%EB%B6%84%EB%B0%A9%EC%A0%95%EC%8B%9D" title="미분방정식">미분방정식</a>으로 다루어진다. 자연의 많은 현상들이 동역학계로 기술될 수 있다. <a href="/wiki/%ED%98%BC%EB%8F%88_%EC%9D%B4%EB%A1%A0" title="혼돈 이론">혼돈 이론</a>은 이러한 예측 불가능한 현상을 탐구하는 데 상당한 기여를 한다.</p>

<h3><span class="mw-headline" id=".EC.88.98.ED.95.99.EA.B8.B0.EC.B4.88.EB.A1.A0_.EA.B4.80.EB.A0.A8_.EB.B6.84.EC.95.BC">수학기초론 관련 분야</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%88%98%ED%95%99&amp;action=edit&amp;section=7" title="부분 편집: 수학기초론 관련 분야">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<div class="rellink boilerplate seealso"><img alt="" data-file-height="128" data-file-width="128" height="16" src="//upload.wikimedia.org/wikipedia/commons/thumb/e/ec/Crystal_Clear_app_xmag.svg/16px-Crystal_Clear_app_xmag.svg.png" srcset="//upload.wikimedia.org/wikipedia/commons/thumb/e/ec/Crystal_Clear_app_xmag.svg/24px-Crystal_Clear_app_xmag.svg.png 1.5x, //upload.wikimedia.org/wikipedia/commons/thumb/e/ec/Crystal_Clear_app_xmag.svg/32px-Crystal_Clear_app_xmag.svg.png 2x" width="16"/> <a href="/wiki/%EC%88%98%ED%95%99%EA%B8%B0%EC%B4%88%EB%A1%A0" title="수학기초론">수학기초론</a> 문서를 참고하십시오.</div>
<p>수학의 기초를 확실히 세우기 위해, 수리논리학과 집합론이 발전하였고, 이와 더불어 범주론이 최근에도 발전되고 있다. “근본 위기”라는 말은 대략 1900년에서 1930년 사이에 일어난, 수학의 엄밀한 기초에 대한 탐구를 상징적으로 보여주는 말이다. 수학의 엄밀한 기초에 대한 몇 가지 의견 불일치는 오늘날에도 계속되고 있다. 수학의 기초에 대한 위기는 그 당시 수많은 논쟁에 의해 촉발되었으며, 그 논쟁에는 칸토어의 집합론과 브라우어-힐베르트 논쟁이 포함되었다.</p>
