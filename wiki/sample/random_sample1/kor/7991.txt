<h1 class="firstHeading" id="firstHeading" lang="ko">수호 성인</h1>
<div class="mw-body-content" id="bodyContent">
<div id="siteSub">위키백과, 우리 모두의 백과사전.</div>
<div id="contentSub"></div>
<div class="mw-jump" id="jump-to-nav">
					이동:					<a href="#mw-head">둘러보기</a>, 					<a href="#p-search">검색</a>
</div>
<div class="mw-content-ltr" dir="ltr" id="mw-content-text" lang="ko"><script>function mfTempOpenSection(id){var block=document.getElementById("mf-section-"+id);block.className+=" open-block";block.previousSibling.className+=" open-block";}</script><p><b>수호 성인</b>(守護聖人) 또는 <b>주보 성인</b>(主保聖人)이란 <a href="/wiki/%EA%B8%B0%EB%8F%85%EA%B5%90" title="기독교">기독교</a>에서 특별히 보호자로 설정하여 받드는 <a href="/wiki/%EC%84%B1%EC%9D%B8_(%EC%A2%85%EA%B5%90)" title="성인 (종교)">성인</a>을 의미한다.</p>
<p><a href="/wiki/%EB%A1%9C%EB%A7%88_%EA%B0%80%ED%86%A8%EB%A6%AD%EA%B5%90%ED%9A%8C" title="로마 가톨릭교회">로마 가톨릭교회</a>와 <a href="/wiki/%EB%8F%99%EB%B0%A9_%EC%A0%95%EA%B5%90%ED%9A%8C" title="동방 정교회">동방 정교회</a> 등에서는 어떤 직업, 장소, 국가, 개인이 특정한 성인을 보호자로 삼아 존경하며, 그 성인을 통하여 <a href="/wiki/%ED%95%98%EB%8A%90%EB%8B%98" title="하느님">하느님</a>께 청원하며, 하느님의 보호를 받을수 있다고 보는데, 이러한 성인을 수호성인, 혹은 주보성인이라 한다. 이러한 관습은 두 개의 교리에 그 기초를 두고 있다. 하나는 <a href="/wiki/%EC%82%AC%EB%8F%84%EC%8B%A0%EA%B2%BD" title="사도신경">사도신경</a>의 모든 성인의 통공(1고린10,16: 2고린 13,13)이고, 다른 하나는 <a href="/wiki/%ED%95%98%EB%8A%90%EB%8B%98%EC%9D%98_%EB%82%98%EB%9D%BC" title="하느님의 나라">하느님의 나라</a>의 구성원들은 각자가 특수한 기능을 수행하고 있다(1고린 1,9: 12,8. 13)는 바울로의 가르침이다.</p>
<p>수호성인을 세우는 관습은 순교자의 묘지 위에 <a href="/wiki/%EC%84%B1%EB%8B%B9" title="성당">성당</a>을 건립하고 그 순교자를 수호성인을 모시는 일이 많았던 사실에서 비롯한다. 그리하여 3세기경까지는 순교자만이 성당의 수호성인이 될 수 있었으나 <a href="/wiki/%EA%B8%B0%EB%8F%85%EA%B5%90" title="기독교">기독교</a>가 <a href="/wiki/%EB%A1%9C%EB%A7%88" title="로마">로마</a>의 국교로 공인된 이후에는 증거자, <a href="/wiki/%EC%A3%BC%EA%B5%90" title="주교">주교</a>, 선교사, 성당의 창설자, 신비(예를 들면 <a href="/wiki/%EC%82%BC%EC%9C%84%EC%9D%BC%EC%B2%B4" title="삼위일체">삼위일체</a>, <a href="/wiki/%EC%8B%AD%EC%9E%90%EA%B0%80" title="십자가">십자가</a>, <a href="/wiki/%EB%A9%94%EC%8B%9C%EC%95%84" title="메시아">구세주</a>) 등도 성당의 수호성인이 되었다. 이처럼 수호성인의 관습은 성인 공경의 한 형태로 생겨난 것이었다.</p>
<p>한 개인이 수호성인을 모시는 관습은 이보다 늦게 생겨났다. 그러나 4세기 초에는 기독교적인 이름이나 성서적인 이름을 <a href="/wiki/%EC%84%B8%EB%A1%80%EB%AA%85" title="세례명">세례명</a>(christian name)으로 선택하는 일이 상당히 널리 퍼져 있었다. 이것은 세례명으로 선택한 성인을 따라 살겠다는 의지임과 동시에 이름을 바꿈에 따라 그 사람도 변화한다는 성서의 내용에 의거한 것이다. 즉 아브람이 <a href="/wiki/%EC%95%84%EB%B8%8C%EB%9D%BC%ED%95%A8" title="아브라함">아브라함</a>으로, 시몬이 <a href="/wiki/%EB%B2%A0%EB%93%9C%EB%A1%9C" title="베드로">베드로</a>로, 사울이 <a class="mw-redirect" href="/wiki/%EB%B0%94%EC%98%A4%EB%A1%9C" title="바오로">바오로</a>로 개명한 사례가 그것이다. 그러나 <a href="/wiki/%EA%B7%B8%EB%A6%AC%EC%8A%A4" title="그리스">그리스</a>나 <a href="/wiki/%EC%8A%A4%ED%8E%98%EC%9D%B8" title="스페인">스페인</a> 외에 일반적으로 <a href="/wiki/%EC%98%88%EC%88%98" title="예수">예수</a>나 <a href="/wiki/%EA%B7%B8%EB%A6%AC%EC%8A%A4%EB%8F%84" title="그리스도">그리스도</a>는 세례명으로 선택하지는 않는다.</p>
<p>한편 직업이나 단체에 대한 수호성인도 있다. 이것은 <a href="/wiki/%EA%B5%90%ED%99%A9" title="교황">교황</a>에 의해 결정되는데, 예컨대 <a href="/wiki/%EC%9A%94%EC%85%89" title="요셉">요셉</a>은 교회, <a href="/wiki/%EC%95%8C%EB%A1%9C%EC%9D%B4%EC%8B%9C%EC%98%A4_%EA%B3%A4%EC%9E%90%EA%B0%80" title="알로이시오 곤자가">알로이시오 곤자가</a>는 청년과 학생, <a class="mw-redirect" href="/wiki/%EB%B9%88%EC%B2%B8%EC%8B%9C%EC%98%A4_%EC%95%84_%EB%B0%94%EC%98%A4%EB%A1%9C" title="빈첸시오 아 바오로">빈첸시오 아 바오로</a>는 자선단체, <a class="mw-redirect" href="/wiki/%ED%94%84%EB%9E%80%EC%B9%98%EC%8A%A4%EC%BD%94_%EC%82%AC%EB%B2%A0%EB%A6%AC%EC%98%A4" title="프란치스코 사베리오">프란치스코 사베리오</a>는 출판단체, <a class="new" href="/w/index.php?title=%EC%95%84%EA%B8%B0_%EC%98%88%EC%88%98%EC%9D%98_%EC%84%B1%EB%85%80_%EB%8D%B0%EB%A0%88%EC%82%AC&amp;action=edit&amp;redlink=1" title="아기 예수의 성녀 데레사 (없는 문서)">아기 예수의 성녀 데레사</a>는 세계포교의 수호성인이다.<sup class="reference" id="cite_ref-1"><a href="#cite_note-1">[1]</a></sup></p>
