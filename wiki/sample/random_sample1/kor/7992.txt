<h2><span class="mw-headline" id=".EA.B0.9C.EB.B0.9C">개발</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%88%98%ED%98%B8%EC%9D%B4_Su-37&amp;action=edit&amp;section=1" title="부분 편집: 개발">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>소련은 세계최강이라고 불리는 미국의 <a class="mw-redirect" href="/wiki/F-15" title="F-15">F-15</a>에 대응하기 위해 <a class="mw-redirect" href="/wiki/Su-27" title="Su-27">Su-27</a>(나토명 플랭커)을 개발했다. 그러나 <a class="mw-redirect" href="/wiki/Su-27" title="Su-27">Su-27</a>은 전자조종장치가 아니라 기계식조종장치인데, 이것을 F-15와 같은 전자조종장치로 개조한 것이 <a href="/wiki/%EC%88%98%ED%98%B8%EC%9D%B4_Su-35" title="수호이 Su-35">Su-35</a>이다. <a href="/wiki/%EC%88%98%ED%98%B8%EC%9D%B4_Su-35" title="수호이 Su-35">Su-35</a>에 <a href="/wiki/%EC%B6%94%EB%A0%A5%ED%8E%B8%ED%96%A5" title="추력편향">추력편향</a> 엔진을 장착한 것이 Su-37(나토명 플랭커-F)이다. 전투기는 공중전에서 매우 빠른 360도 회전이 필요하다. 추력편향 엔진을 장착하면, 360도 회전시간이 획기적으로 줄어든다.</p>
<p>Su-35는 처음에 Su-27M 이라고 불렸다.<sup class="reference" id="cite_ref-Eden_2-0"><a href="#cite_note-Eden-2">[2]</a></sup> Su-27M 시제기 (T-10S-70)는 1988년에 최초로 비행했다. Su-27에서 바뀐 점은 카나드 날개 장착, 엔진 업그레이드, 새로운 레이더, 디지털 <a href="/wiki/%EB%B9%84%ED%96%89_%EC%A1%B0%EC%A2%85_%EC%8B%9C%EC%8A%A4%ED%85%9C" title="비행 조종 시스템">FCS</a> 채택이다.<sup class="reference" id="cite_ref-Frawley_mil_3-0"><a href="#cite_note-Frawley_mil-3">[3]</a></sup> Sukhoi changed the Su-27M's designation to Su-35 in 1993. Later Su-35 prototypes added a glass cockpit and modified tailfins.<sup class="reference" id="cite_ref-Eden_2-1"><a href="#cite_note-Eden-2">[2]</a></sup></p>
<p>Su-35에 2차원 <a href="/wiki/%EC%B6%94%EB%A0%A5%ED%8E%B8%ED%96%A5" title="추력편향">추력편향</a> 노즐을 장착하여 Su-37 가 되었다.<sup class="reference" id="cite_ref-Frawley_mil_3-1"><a href="#cite_note-Frawley_mil-3">[3]</a></sup> Su-37 시제기는 1996년 4월에 초도비행했다.<sup class="reference" id="cite_ref-FI_Su-37_thrust_4-0"><a href="#cite_note-FI_Su-37_thrust-4">[4]</a></sup> Su-37는 1996 Farnborough 에어쇼에 모습을 드러냈다.<sup class="reference" id="cite_ref-Eden_2-2"><a href="#cite_note-Eden-2">[2]</a></sup> Yevgeny Frolov가 조종했다. Su-37 2호기는 1998년에 모습을 드러내었다.<sup class="reference" id="cite_ref-Gordon_p34-37_5-0"><a href="#cite_note-Gordon_p34-37-5">[5]</a></sup></p>
<p>수호이사는 Su-27 시리즈들, 예를들어 <a class="new" href="/w/index.php?title=Su-30MKI&amp;action=edit&amp;redlink=1" title="Su-30MKI (없는 문서)">Su-30MKI</a>, <a class="new" href="/w/index.php?title=Su-35BM&amp;action=edit&amp;redlink=1" title="Su-35BM (없는 문서)">Su-35BM</a>등 에도 이러한 개조, 즉 디지털화, 레이더 개량, 추력편향 엔진을 추가 등을 할 것이라고 했다.</p>
<h2><span class="mw-headline" id=".EC.84.A4.EA.B3.84">설계</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%88%98%ED%98%B8%EC%9D%B4_Su-37&amp;action=edit&amp;section=2" title="부분 편집: 설계">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>Su-37은 1인승, 전천후 다목적 전투기이다. <a class="mw-redirect" href="/wiki/Su-27" title="Su-27">Su-27</a>에서 여러가지 개조를 했다. <a class="new" href="/w/index.php?title=MESA&amp;action=edit&amp;redlink=1" title="MESA (없는 문서)">MESA</a> 레이더를 장착했고, 지형맵핑, 후미에 접근경보레이더도 장착했다. 메탈소재인 <a class="mw-redirect" href="/wiki/Su-27" title="Su-27">Su-27</a>와 달리, 복합소재를 사용했다. 2차원 <a href="/wiki/%EC%B6%94%EB%A0%A5%ED%8E%B8%ED%96%A5" title="추력편향">추력편향</a>이 가능한 AL-37FU 엔진을 사용하여 <a href="/wiki/%EC%BD%94%EB%B8%8C%EB%9D%BC_%EA%B8%B0%EB%8F%99" title="코브라 기동">코브라 기동</a>이 가능하다.</p>
<h3><span class="mw-headline" id=".EC.A1.B0.EC.A2.85.EC.84.9D">조종석</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%88%98%ED%98%B8%EC%9D%B4_Su-37&amp;action=edit&amp;section=3" title="부분 편집: 조종석">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>러시아 전투기 최초로 조종석에 <a class="new" href="/w/index.php?title=HOTAS&amp;action=edit&amp;redlink=1" title="HOTAS (없는 문서)">HOTAS</a>를 채택했다. <a href="/wiki/%ED%94%8C%EB%9D%BC%EC%9D%B4_%EB%B0%94%EC%9D%B4_%EC%99%80%EC%9D%B4%EC%96%B4" title="플라이 바이 와이어">플라이 바이 와이어</a> 기능이 있다.</p>
<h2><span class="mw-headline" id=".EB.8C.80.ED.95.9C.EB.AF.BC.EA.B5.AD">대한민국</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%88%98%ED%98%B8%EC%9D%B4_Su-37&amp;action=edit&amp;section=4" title="부분 편집: 대한민국">편집</a><span class="mw-editsection-bracket">]</span></span></h2>

<h2><span class="mw-headline" id=".EC.A0.9C.EC.9B.90_.28Su-37.29">제원 (Su-37)</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%88%98%ED%98%B8%EC%9D%B4_Su-37&amp;action=edit&amp;section=6" title="부분 편집: 제원 (Su-37)">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p style="font-size: 90%; margin:0;">정보의 출처: Gordon and Davison<sup class="reference" id="cite_ref-Gordon_p92_6-0"><a href="#cite_note-Gordon_p92-6">[6]</a></sup></p>
<p><big style="padding-top: .2em;">일반 특성</big></p>
<ul>
<li><b>승무원:</b> 1, pilot</li>
<li><b>길이:</b> 22.183 m (72 ft 9 in)</li>
<li><b>날개폭:</b> 14.7 m (48 ft 3 in)</li>
<li><b>높이:</b> 6.43 m (21 ft 1 in)</li>
<li><b>날개면적:</b> 62.0 m² (667 ft²)</li>
<li><b>공허중량:</b> 18,500 kg (40,790 lb)</li>
<li><b>최대이륙중량:</b> 35,000 kg (77,160 lb)</li>
<li><b>엔진:</b> 2× <a class="new" href="/w/index.php?title=Saturn_AL-31&amp;action=edit&amp;redlink=1" title="Saturn AL-31 (없는 문서)">Lyulka AL-37FU</a> <a class="new" href="/w/index.php?title=Turbofan&amp;action=edit&amp;redlink=1" title="Turbofan (없는 문서)">turbofans</a>
<ul>
<li><b>최대추력:</b> 7,600 kgf (74.5 kN, 16,750 lbf) 각각</li>
<li><b>재연소시추력:</b> 145 kN (32,000 lbf) 각각</li>
</ul>
</li>
</ul>
<p><big>성능</big></p>
<ul>
<li><b>최대속도:</b> Mach 2.30 (1,550 mph<sup class="reference" id="cite_ref-7"><a href="#cite_note-7">[7]</a></sup>)</li>
<li><b>항속거리:</b> 3,700 km (2,230 mi)</li>
<li><b>상승한도:</b> 18,000 m (59,100 ft)</li>
</ul>
<ul>
<li><b>Maximum G loading:</b> 9/-3 g</li>
</ul>
<p><big>무장</big></p>
<ul>
<li>1 × 30 mm <a class="new" href="/w/index.php?title=Gryazev-Shipunov_GSh-30-1&amp;action=edit&amp;redlink=1" title="Gryazev-Shipunov GSh-30-1 (없는 문서)">GSh-30</a> cannon with 150 rounds</li>
<li>12 × wing and fuselage stations for up to 8,200 kg (18,080 lb) of ordnance</li>
</ul>
