<p>《<b>순자</b>(筍子)》는 <a href="/wiki/%EC%A4%91%EA%B5%AD" title="중국">중국</a> <a class="mw-redirect" href="/wiki/%EC%A0%84%EA%B5%AD%EC%8B%9C%EB%8C%80" title="전국시대">전국시대</a>(戰國時代: <a href="/wiki/%EA%B8%B0%EC%9B%90%EC%A0%84_403%EB%85%84" title="기원전 403년">BC 403</a>~<a href="/wiki/%EA%B8%B0%EC%9B%90%EC%A0%84_221%EB%85%84" title="기원전 221년">221</a>) 말기의 <a href="/wiki/%EC%9C%A0%EA%B0%80" title="유가">유가</a> 사상가 <a href="/wiki/%EC%88%9C%EC%9E%90" title="순자">순자</a>(荀子: <a href="/wiki/%EA%B8%B0%EC%9B%90%EC%A0%84_298%EB%85%84" title="기원전 298년">BC 298</a>?~<a href="/wiki/%EA%B8%B0%EC%9B%90%EC%A0%84_238%EB%85%84" title="기원전 238년">238</a>?)의 말과 글을 모은 것이다.<sup class="reference" id="cite_ref-.EA.B8.80.EB.A1.9C.EB.B2.8C-.EC.88.9C.EC.9E.90_1-0"><a href="#cite_note-.EA.B8.80.EB.A1.9C.EB.B2.8C-.EC.88.9C.EC.9E.90-1">[1]</a></sup> 《<b>순경신서</b>(荀卿新書)》<b>·</b>《<b>손경자</b>(孫卿子)》라고도 불린다.<sup class="reference" id="cite_ref-.EA.B8.80.EB.A1.9C.EB.B2.8C-.EC.88.9C.EC.9E.90_1-1"><a href="#cite_note-.EA.B8.80.EB.A1.9C.EB.B2.8C-.EC.88.9C.EC.9E.90-1">[1]</a></sup></p>
<p>이 책의 〈성악(性惡)〉 편에서는 <a href="/wiki/%EB%A7%B9%EC%9E%90" title="맹자">맹자</a>의 <a href="/wiki/%EC%84%B1%EC%84%A0%EC%84%A4" title="성선설">성선설</a>(性善說)에 반대하여 <a href="/wiki/%EC%84%B1%EC%95%85%EC%84%A4" title="성악설">성악설</a>(性惡說)이 주장되어 있다.<sup class="reference" id="cite_ref-.EA.B8.80.EB.A1.9C.EB.B2.8C-.EC.88.9C.EC.9E.90_1-2"><a href="#cite_note-.EA.B8.80.EB.A1.9C.EB.B2.8C-.EC.88.9C.EC.9E.90-1">[1]</a></sup> 이 책의 저자인 <a href="/wiki/%EC%88%9C%EC%9E%90" title="순자">순자</a>는 <a href="/wiki/%ED%95%9C%EB%82%98%EB%9D%BC" title="한나라">한나라</a>에서는 <a href="/wiki/%EC%9C%A0%EA%B0%80" title="유가">유가</a>의 정통적 인물로 보았으나, <a href="/wiki/%EC%86%A1%EB%82%98%EB%9D%BC" title="송나라">송나라</a> 시대의 <a href="/wiki/%EC%84%B1%EB%A6%AC%ED%95%99" title="성리학">성리학</a>의 학풍 속에서 <a href="/wiki/%EC%84%B1%EC%95%85%EC%84%A4" title="성악설">성악설</a>이 비난받으면서부터 <a href="/wiki/%EC%9C%A0%EA%B0%80" title="유가">유가</a>의 이단자로 간주되어 왔다.<sup class="reference" id="cite_ref-.EA.B8.80.EB.A1.9C.EB.B2.8C-.EC.88.9C.EC.9E.90-.EC.9D.B8.EB.AC.BC_2-0"><a href="#cite_note-.EA.B8.80.EB.A1.9C.EB.B2.8C-.EC.88.9C.EC.9E.90-.EC.9D.B8.EB.AC.BC-2">[2]</a></sup> 그에 대한 재평가는 <a href="/wiki/%EC%B2%AD%EB%82%98%EB%9D%BC" title="청나라">청나라</a> 말기에서야 다시 이루어졌다.<sup class="reference" id="cite_ref-.EA.B8.80.EB.A1.9C.EB.B2.8C-.EC.88.9C.EC.9E.90-.EC.9D.B8.EB.AC.BC_2-1"><a href="#cite_note-.EA.B8.80.EB.A1.9C.EB.B2.8C-.EC.88.9C.EC.9E.90-.EC.9D.B8.EB.AC.BC-2">[2]</a></sup></p>
<p>이 책은 <a href="/wiki/%EC%88%9C%EC%9E%90" title="순자">순자</a>의 사상을 알 수 있을 뿐만 아니라 <a class="mw-redirect" href="/wiki/%EC%84%A0%EC%A7%84_%EC%8B%9C%EB%8C%80" title="선진 시대">선진 시대</a>(先秦時代)의 <a href="/wiki/%EC%9C%A0%EA%B0%80" title="유가">유가</a>(儒家) 사상의 총괄서로서 큰 의의를 가진다.<sup class="reference" id="cite_ref-.EA.B8.80.EB.A1.9C.EB.B2.8C-.EC.88.9C.EC.9E.90_1-3"><a href="#cite_note-.EA.B8.80.EB.A1.9C.EB.B2.8C-.EC.88.9C.EC.9E.90-1">[1]</a></sup></p>

<h2><span class="mw-headline" id=".EA.B0.9C.EC.9A.94">개요</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%88%9C%EC%9E%90_(%EC%B1%85)&amp;action=edit&amp;section=1" title="부분 편집: 개요">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>전부 32편으로 구성되어 있다. 본래는 322편이었던 것을 <a href="/wiki/%EC%A0%84%ED%95%9C" title="전한">전한</a>(前漢: <a href="/wiki/%EA%B8%B0%EC%9B%90%EC%A0%84_206%EB%85%84" title="기원전 206년">BC 206</a>~<a href="/wiki/%EA%B8%B0%EC%9B%90%EC%A0%84_8%EB%85%84" title="기원전 8년">8</a>) 시대에 <a href="/wiki/%EC%9C%A0%ED%96%A5_(%EC%A0%84%ED%95%9C)" title="유향 (전한)">유향</a>(劉向)이 중복되는 290편을 제거하여 32편으로 정착시켰다. 《순자(荀子)》에 처음으로 주석을 한 것은 <a href="/wiki/%EB%8B%B9%EB%82%98%EB%9D%BC" title="당나라">당나라</a>(唐)의 <a class="new" href="/w/index.php?title=%EC%96%91%EB%9F%89&amp;action=edit&amp;redlink=1" title="양량 (없는 문서)">양량</a>(楊凉)인데 32편을 20권으로 나누었고 순서도 조금 변경하였다.</p>
<p>내용은 <a href="/wiki/%EC%B2%A0%ED%95%99" title="철학">철학</a> <b>·</b> <a class="mw-disambig" href="/wiki/%EC%8B%AC%EB%A6%AC" title="심리">심리</a> <b>·</b> <a href="/wiki/%EB%8F%84%EB%8D%95" title="도덕">도덕</a> <b>·</b> <a class="mw-disambig" href="/wiki/%EB%85%BC%EB%A6%AC" title="논리">논리</a> <b>·</b> <a href="/wiki/%EA%B5%90%EC%9C%A1" title="교육">교육</a> <b>·</b> <a href="/wiki/%EC%A0%95%EC%B9%98" title="정치">정치</a> <b>·</b> <a href="/wiki/%EA%B2%BD%EC%A0%9C" title="경제">경제</a> <b>·</b> <a href="/wiki/%EC%A0%84%EB%9E%B5" title="전략">전략</a> <b>·</b> <a class="mw-redirect" href="/wiki/%EC%82%AC%EC%83%81%EA%B0%80" title="사상가">사상가</a> 비판 등 광범위한 주제를 다루고 있다. 또한, <a href="/wiki/%EC%9A%B4%EB%AC%B8" title="운문">운문</a>(韻文) <b>·</b> <a href="/wiki/%EC%84%A4%ED%99%94" title="설화">설화</a>에 대한 내용도 들어있다. 다루는 주제가 다양하고 광범위할 뿐만 아니라 문체도 각 부분에 걸쳐 상당한 차이가 있어 동일인의 저술이라고 볼 수는 없다.</p>
<p>특히 〈성악(性惡)〉 편에서는 <a href="/wiki/%EB%A7%B9%EC%9E%90" title="맹자">맹자</a>의 <a href="/wiki/%EC%84%B1%EC%84%A0%EC%84%A4" title="성선설">성선설</a>(性善說)에 반대하여 <a class="mw-redirect" href="/wiki/%EC%9D%B8%EA%B0%84" title="인간">인간</a>은 <a href="/wiki/%EC%95%85" title="악">악</a>을 좋아하고 <a href="/wiki/%EC%9D%B4%EC%9D%B5" title="이익">이익</a>(利)를 좇기 쉬우므로 <a href="/wiki/%EC%9C%A0%EA%B0%80" title="유가">유가</a>의 경전을 배우고 <a class="mw-disambig" href="/wiki/%EC%98%88" title="예">예</a>(禮)를 지켜서 이러한 성품을 변화시키고 극복하여야 "인격이 완성"(僞 = 人爲)된다는 <a href="/wiki/%EC%84%B1%EC%95%85%EC%84%A4" title="성악설">성악설</a>(性惡說)을 주장하였다. <a class="new" href="/w/index.php?title=%EC%96%91%EB%9F%89&amp;action=edit&amp;redlink=1" title="양량 (없는 문서)">양량</a>(楊凉) 이후 <a class="mw-redirect" href="/wiki/%EC%86%A1%ED%95%99" title="송학">송학</a>(宋學)의 발흥과 함께 <a class="mw-redirect mw-disambig" href="/wiki/4%EC%84%9C" title="4서">4서</a>의 도통이 중시되어 《순자》는 <a href="/wiki/%EC%9C%A0%EA%B5%90" title="유교">유교</a>에서 이단의 서적으로서 오랫동안 배척되었다.</p>
<h2><span class="mw-headline" id=".EB.82.B4.EC.9A.A9">내용</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%88%9C%EC%9E%90_(%EC%B1%85)&amp;action=edit&amp;section=2" title="부분 편집: 내용">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>최초의 '권학(勸學)', '수신(修身)' 편은 개인의 수양을 논하였는데, 좋은 환경 속에서 양사(良師)를 따라서 유가의 경전을 배우고 예를 지키는 일에 의하여 인격이 완성된다고 한다.</p>
<p>'성악' 편에서는 <a href="/wiki/%EB%A7%B9%EC%9E%90" title="맹자">맹자</a>의 성선설에 반대하여 인간은 악을 좋아하고 이(利)를 좇기 쉬우므로 예를 배워서 '위(僞) → 인위(人爲)'로 변경해야 한다고 주장하였다. 그 구체적인 방법에 관해서는 앞서 말한 '권학' 편에 기술하고 있다.</p>
<p>'천론(天論)', '비상(非相)', '정론(正論)'편은 전통적 사고, 미신, 숙명론의 타파를 기도해 본 것이다. 특히 천론은 천(天)에 대한 종교적 이해에서 벗어나 천(天)을 인간생활과는 무관계한 자연 존재라고 인정하였다. 여기에서 처음으로 인간성의 참된 이해가 가능토록 한 점에서 유가사상사에 있어서 특필해야 할 의의를 갖는다.</p>
<p>'정명(正名)' 편은 논리와 인식의 문제에 대하여, '해폐(解蔽)'편은 방법론의 문제에 대하여, '악론(樂論)'편은 음악 이론에 대하여, '왕제(王制)' '부국(富國)'편은 사회에 있어서의 정치적 원칙을, '의병'편은 군사 이론에 대하여 각각 기술하고 있다.</p>
<p>'비12자(非十二子)' 편은 <a href="/wiki/%EA%B3%B5%EC%9E%90" title="공자">공자</a>, <a href="/wiki/%EC%9E%90%EA%B3%B5" title="자공">자공</a>에 대해 존경하는 태도를 보인 외에 <a href="/wiki/%EC%9E%90%EC%82%AC" title="자사">자사</a>, <a href="/wiki/%EB%A7%B9%EC%9E%90" title="맹자">맹자</a> 등을 포함한 12명의 사상가를 비판한 것인데 선진 제자(先秦諸子)의 사상을 알게 하는 점에서 귀중한 기록이다.</p>
<p>'성상(成相)'편은 민가의 체재를 취하여 순자의 정치사상을 서술한 것이며, '부(賦)'는 그의 문학작품이다.</p>
<p>그러나 양량(楊凉)이 19권과 20권에 수록한 각편('大略'에서 '堯問'까지의 6편)은 순자의 후학의 기록으로 여겨진다.</p>
