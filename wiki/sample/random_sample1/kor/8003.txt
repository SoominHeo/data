<p><b>순천만국가정원</b>(順天灣國家庭園)은 <a href="/wiki/%EB%8C%80%ED%95%9C%EB%AF%BC%EA%B5%AD" title="대한민국">대한민국</a> <a href="/wiki/%EC%A0%84%EB%9D%BC%EB%82%A8%EB%8F%84" title="전라남도">전라남도</a> <a href="/wiki/%EC%88%9C%EC%B2%9C%EC%8B%9C_(%EC%A0%84%EB%9D%BC%EB%82%A8%EB%8F%84)" title="순천시 (전라남도)">순천시</a>의 정원이다. <b>2013순천만국제정원박람회</b>가 폐막한 후 그 회장을 개조하여 조성했다.</p>

<h2><span class="mw-headline" id="2013.EC.88.9C.EC.B2.9C.EB.A7.8C.EA.B5.AD.EC.A0.9C.EC.A0.95.EC.9B.90.EB.B0.95.EB.9E.8C.ED.9A.8C">2013순천만국제정원박람회</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%88%9C%EC%B2%9C%EB%A7%8C%EA%B5%AD%EA%B0%80%EC%A0%95%EC%9B%90&amp;action=edit&amp;section=1" title="부분 편집: 2013순천만국제정원박람회">편집</a><span class="mw-editsection-bracket">]</span></span></h2>

<td class="mbox-text"><span class="mbox-text-span">이 문서의 주된 기여자가 <b>주제와 밀접하게 관련된 <a href="/wiki/%EC%9C%84%ED%82%A4%EB%B0%B1%EA%B3%BC:%EC%9D%B4%ED%95%B4%EA%B4%80%EA%B3%84%EC%9D%98_%EC%B6%A9%EB%8F%8C" title="위키백과:이해관계의 충돌">이해당사자</a>일 수 있습니다.</b><span class="hide-when-compact"><br/>
<a class="mw-redirect" href="/wiki/%EC%9C%84%ED%82%A4%EB%B0%B1%EA%B3%BC:%EC%95%84%EB%8B%98#.EB.8C.80.ED.91.9C" title="위키백과:아님">위키백과는 특정 개인이나 단체의 이익을 대변하는 곳이 아니며</a>, 모든 콘텐츠는 위키백과의 정책 중 하나인 <a href="/wiki/%EC%9C%84%ED%82%A4%EB%B0%B1%EA%B3%BC:%EC%A4%91%EB%A6%BD%EC%A0%81_%EC%8B%9C%EA%B0%81" title="위키백과:중립적 시각">중립적 시각</a>을 따라야 합니다. 이해당사자가 편집한 내용이 중립적으로 보일지라도 제3자에 의한 확인이 필요할 수 있습니다.</span> <small>(2016년 6월 16일)</small></span></td>

<p>2013순천만국제정원박람회(2013 順天灣國際庭園博覽會, <span lang="en">International Garden Exposition Suncheon Bay Korea 2013</span>)는 <a href="/wiki/%EB%8C%80%ED%95%9C%EB%AF%BC%EA%B5%AD" title="대한민국">대한민국</a> <a href="/wiki/%EC%A0%84%EB%9D%BC%EB%82%A8%EB%8F%84" title="전라남도">전라남도</a> <a href="/wiki/%EC%88%9C%EC%B2%9C%EC%8B%9C_(%EC%A0%84%EB%9D%BC%EB%82%A8%EB%8F%84)" title="순천시 (전라남도)">순천시</a>에서 개최되었다. <a href="/wiki/%EB%8C%80%ED%95%9C%EB%AF%BC%EA%B5%AD" title="대한민국">대한민국</a>에서는 처음으로 개최되는 국제정원박람회로서 <a class="new" href="/w/index.php?title=AIPH&amp;action=edit&amp;redlink=1" title="AIPH (없는 문서)">AIPH</a>(국제원예생산자협회)에서 <a href="/wiki/2009%EB%85%84" title="2009년">2009년</a> <a href="/wiki/9%EC%9B%94_16%EC%9D%BC" title="9월 16일">9월 16일</a> 유치가 확정되었다.</p>
<ul>
<li>행사명: 순천만 국제정원 박람회<br/></li>
<li>행사 기간: 2013년 4월 20일 ~ 10월 20일(6개월간)<br/></li>
<li>장소 : <a href="/wiki/%EC%A0%84%EB%9D%BC%EB%82%A8%EB%8F%84" title="전라남도">전라남도</a> <a href="/wiki/%EC%88%9C%EC%B2%9C%EC%8B%9C_(%EC%A0%84%EB%9D%BC%EB%82%A8%EB%8F%84)" title="순천시 (전라남도)">순천시</a> <a href="/wiki/%ED%92%8D%EB%8D%95%EB%8F%99" title="풍덕동">풍덕동</a> · <a class="mw-disambig" href="/wiki/%EC%98%A4%EC%B2%9C%EB%8F%99" title="오천동">오천동</a> 일원, <a href="/wiki/%EC%88%9C%EC%B2%9C%EB%A7%8C" title="순천만">순천만</a>(1,112천㎡)<br/></li>
<li>참가국 규모: 23개국, 83개 정원(세계정원 11개, 참여정원 61개, 테마정원 11개)</li>
</ul>

<li>작가, 시민, 기업체 등이 공모를 통해서 참여하게 되며, 형식에 얽매이지 않은 30여개의 다양한 정원이 조성되어 박람회장의 풍성함을 더하고 박람회 기간 이후에도 참가자들이 정원에 관심을 가지고 꾸며나갈 수 있다.</li>
</ul>
</li>
<li class="mw-empty-elt"></li>
<li><b>물의정원, 숲의정원</b>
<ul>
<li>정원과 호수, 연못, 계곡 습지 등 물이 가지는 다양한 경관을 즐길 수 있는 10개의 물의정원과 상수리나무숲, 메타세콰이어 숲, 소나무숲, 편백나무 숲 등 숲의정원이 박람회장내 곳곳에 배치되었다.</li>
</ul>
</li>
</ul>

<div class="thumbinner" style="width:222px;"><a class="image" href="/wiki/%ED%8C%8C%EC%9D%BC:A_hill_of_International_Garden_Exposition_Suncheon_Bay_Korea_2013.jpg"><img alt="" class="thumbimage" data-file-height="642" data-file-width="856" height="165" src="//upload.wikimedia.org/wikipedia/commons/thumb/9/98/A_hill_of_International_Garden_Exposition_Suncheon_Bay_Korea_2013.jpg/220px-A_hill_of_International_Garden_Exposition_Suncheon_Bay_Korea_2013.jpg" srcset="//upload.wikimedia.org/wikipedia/commons/thumb/9/98/A_hill_of_International_Garden_Exposition_Suncheon_Bay_Korea_2013.jpg/330px-A_hill_of_International_Garden_Exposition_Suncheon_Bay_Korea_2013.jpg 1.5x, //upload.wikimedia.org/wikipedia/commons/thumb/9/98/A_hill_of_International_Garden_Exposition_Suncheon_Bay_Korea_2013.jpg/440px-A_hill_of_International_Garden_Exposition_Suncheon_Bay_Korea_2013.jpg 2x" width="220"/></a>

호수 정원가의 언덕</div>
</div>
</div>
<ul>
<li><b>한방약초원</b>
<ul>
<li>‘건강과 웰빙’실현의 장으로서 정원박람회장에는 약초가 가지고 있는 특징과 효능을 테마로한 감상과 체험 공간인 ‘한방약초원’으로 구성된다.</li>
<li>산, 들, 물가 등 대한민국에서 야생하는 모든 약초를 모아 15만㎡의 재배공원과 체험관을 조성했다 체질 진맥, 아토피와 당뇨 등 현대사회에 유행하는 병을 치료하는 프로그램을 운영하여, 휴식하고 체험하는 약초의 또 다른 세계를 경험하도록 할 것입니다.</li>
</ul>
</li>
</ul>
<ul>
<li><b>수목원</b>
<ul>
<li>세계 각국에서 정원수로 활용되는 다양한 수종을 한곳에 모아둔 세계정원수원이 있다. 조선시대 정원으로 조성된 한국정원이 있다.</li>
<li>우리나라의 기후변화에 따른 난대수종의 식생 변화 모습을 관찰 할 수 있는 난대수목원과 순천시의 대표 수종인 철쭉을 대규모로 조성한 철쭉원으로 구성되는 남도 숲 탐방 존으로 구성되어 있다.</li>
</ul>
</li>
</ul>
<ul>
<li><b>국제습지센터</b>
<ul>
<li>정원박람회의 주제관이다.</li>
<li>다양한 수생식물들이 군락을 이루어 생태환경교육을 위한 최적의 습지환경을 제공하게 될 것이며 비오톱 습지는 수생 동식물 및 조류를 관찰하는 생태 학습장으로 활용됨과 더불어 습지의 정화기능을 수행하게 된다.</li>
</ul>
</li>
</ul>
<ul>
<li><b>저류지</b>
<ul>
<li>홍수예방 기능을 겸비하여 조성되는 저류지 공원은 평상시 축구장, 파크골프장, 잔디광장 등의 다양한 용도로 활용되어 휴식공간을 제공된다.</li>
</ul>
</li>
</ul>
