<p><b>숲주머니날개박쥐</b>(<i>Saccopteryx canescens</i>)는 <a href="/wiki/%EB%8C%80%EA%BC%AC%EB%A6%AC%EB%B0%95%EC%A5%90%EA%B3%BC" title="대꼬리박쥐과">대꼬리박쥐과</a>에 속하는 <a href="/wiki/%EB%82%A8%EC%95%84%EB%A9%94%EB%A6%AC%EC%B9%B4" title="남아메리카">남아메리카</a> <a href="/wiki/%EB%B0%95%EC%A5%90" title="박쥐">박쥐</a>의 일종이다. <a href="/wiki/%EB%B8%8C%EB%9D%BC%EC%A7%88" title="브라질">브라질</a> 북부와 <a href="/wiki/%EC%BD%9C%EB%A1%AC%EB%B9%84%EC%95%84" title="콜롬비아">콜롬비아</a>, <a href="/wiki/%EC%97%90%EC%BD%B0%EB%8F%84%EB%A5%B4" title="에콰도르">에콰도르</a>, <a href="/wiki/%ED%94%84%EB%9E%91%EC%8A%A4%EB%A0%B9_%EA%B8%B0%EC%95%84%EB%82%98" title="프랑스령 기아나">프랑스령 기아나</a>, <a href="/wiki/%EA%B0%80%EC%9D%B4%EC%95%84%EB%82%98" title="가이아나">가이아나</a>, <a href="/wiki/%ED%8E%98%EB%A3%A8" title="페루">페루</a>, <a href="/wiki/%EC%88%98%EB%A6%AC%EB%82%A8" title="수리남">수리남</a>, <a href="/wiki/%EB%B2%A0%EB%84%A4%EC%88%98%EC%97%98%EB%9D%BC" title="베네수엘라">베네수엘라</a>에서 발견되며, <a href="/wiki/%EB%B3%BC%EB%A6%AC%EB%B9%84%EC%95%84" title="볼리비아">볼리비아</a>에서도 서식하는 것으로 추정하고 있다.<sup class="reference" id="cite_ref-iucn_1-1"><a href="#cite_note-iucn-1">[1]</a></sup><sup class="reference" id="cite_ref-Simmons_2-0"><a href="#cite_note-Simmons-2">[2]</a></sup></p>
