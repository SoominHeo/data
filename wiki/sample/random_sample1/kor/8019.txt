<p><b>숴저우</b>(삭주, <small><a href="/wiki/%EC%A4%91%EA%B5%AD%EC%96%B4" title="중국어">중국어</a></small><small>:</small> <span lang="zh" xml:lang="zh">朔州</span>, <small><a href="/wiki/%ED%95%9C%EC%96%B4_%EB%B3%91%EC%9D%8C" title="한어 병음">병음</a>:</small> <font face="Lucida Sans Unicode">Shuòzhōu</font>)은 <a href="/wiki/%EC%A4%91%EA%B5%AD" title="중국">중국</a> <a href="/wiki/%EC%82%B0%EC%8B%9C_%EC%84%B1_(%EC%82%B0%EC%84%9C%EC%84%B1)" title="산시 성 (산서성)">산시 성</a>의 서북부에 위치 있다. 산시 성 직하에 <a href="/wiki/%EC%A7%80%EA%B8%89%EC%8B%9C" title="지급시">지급시</a>중 하나이고 면적은 10662평방킬로미터, 총인구는 154만명이다. <a href="/wiki/%ED%8E%80%ED%97%88" title="펀허">펀허</a>의 상류에 위치하고 시 중심은 <a href="/wiki/%EC%88%B4%EC%B2%AD_%EA%B5%AC" title="숴청 구">숴청 구</a>이다.</p>

<h2><span class="mw-headline" id=".EC.A7.80.EB.A6.AC">지리</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%88%B4%EC%A0%80%EC%9A%B0_%EC%8B%9C&amp;action=edit&amp;section=1" title="부분 편집: 지리">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>산시 성 서북부에 위치하고 있으며 <a href="/wiki/%EB%8B%A4%ED%89%81_%EC%8B%9C" title="다퉁 시">다퉁 시</a>, <a href="/wiki/%EC%8B%A0%EC%A0%80%EC%9A%B0_%EC%8B%9C" title="신저우 시">신저우 시</a>, <a href="/wiki/%EB%82%B4%EB%AA%BD%EA%B3%A8_%EC%9E%90%EC%B9%98%EA%B5%AC" title="내몽골 자치구">내몽골 자치구</a>에 접한다. 위안쯔허와 후이허가 합류해 <a class="mw-redirect" href="/wiki/%EC%8C%8D%EC%B2%B8%ED%97%88" title="쌍첸허">쌍첸허</a>가 되어 허베이 방면으로 흐르고있다.</p>
<p>연강수량은 428mm이고 연평균 기온은 6.4℃이다.</p>
<h2><span class="mw-headline" id=".EC.97.AD.EC.82.AC">역사</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%88%B4%EC%A0%80%EC%9A%B0_%EC%8B%9C&amp;action=edit&amp;section=2" title="부분 편집: 역사">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p><a class="mw-redirect" href="/wiki/%EC%B6%98%EC%B6%94%EC%8B%9C%EB%8C%80" title="춘추시대">춘추시대</a> 이전에 삭주는 <a href="/wiki/%EB%B6%81%EC%A0%81" title="북적">북적</a>의 활동 영역이었고 <a class="mw-redirect" href="/wiki/%EC%A0%84%EA%B5%AD%EC%8B%9C%EB%8C%80" title="전국시대">전국시대</a>에 <a class="mw-disambig" href="/wiki/%EC%A1%B0%EB%82%98%EB%9D%BC" title="조나라">조나라</a>의 판도에 들어갔다. <a class="mw-redirect" href="/wiki/%EC%A7%84_(%EC%B6%98%EC%B6%94%EC%A0%84%EA%B5%AD)" title="진 (춘추전국)">진나라</a>에 의해 안문군이 설치되었고 <a href="/wiki/%ED%95%9C%EB%82%98%EB%9D%BC" title="한나라">한나라</a> 초기에 마읍현, 중릉현, 극양현, 영관현, 왕도현, 날현, 누번현, 선무현의 8현이 설치되어 안문군(雁門郡)에 속했다. <a href="/wiki/%ED%9B%84%ED%95%9C" title="후한">후한</a> 말의 전란기에 삭주에서 농민의 도망자가 잇따라 이러한 현들은 인구가 희박해져 명목적인 존재가 되었다.</p>
<p><a href="/wiki/%EC%84%9C%EC%A7%84" title="서진">진대</a>되면서 황폐한 삭주 부흥을 목적으로 욕령(현재의 옌먼 관) 이북으로 이민을 장려했다. 이것 때문에 인구가 증가했고 <a href="/wiki/%EB%B6%81%EC%9C%84_(%EB%B6%81%EC%A1%B0)" title="북위 (북조)">북위</a>가 산시 성에 도읍을 정하면서 삭주에는 상건군(<a href="/wiki/%EC%82%B0%EC%9D%B8_%ED%98%84" title="산인 현">산인 현</a> 동부), 번치군(<a href="/wiki/%EC%9E%89_%ED%98%84" title="잉 현">잉 현</a> 동부), 마읍군의 3군이 설치되었다.</p>
<p><a href="/wiki/621%EB%85%84" title="621년">621년</a>, <a href="/wiki/%EB%8B%B9%EB%82%98%EB%9D%BC" title="당나라">당나라</a>는 마읍군(馬邑郡)을 삭주로 개칭하여 현재의 지명이 사서에 등장하게 되었다. 그러나 <a href="/wiki/742%EB%85%84" title="742년">742년</a> 다시 마읍군으로 개명되었다.</p>
<p><a class="mw-disambig" href="/wiki/%ED%9B%84%EC%A7%84" title="후진">후진</a> 때인 <a href="/wiki/936%EB%85%84" title="936년">936년</a>, 삭주는 <a href="/wiki/%EC%9A%94%EB%82%98%EB%9D%BC" title="요나라">요나라</a>에 할양되어 순의군절도가 되어 부경(副京)의 하나인 서경도에 속했다. 오대, 요, 금, 원대에 걸쳐 선양현(鄯陽縣)으로 불렸고 송은 삭녕부(朔寧府)를, 금과 원은 삭주를 설치하였다.</p>
<p>명청대에 삭주는 대동부로 이관되어 <a href="/wiki/%EC%A4%91%ED%99%94%EB%AF%BC%EA%B5%AD" title="중화민국">중화민국</a>의 성립 때까지 이어졌다. <a href="/wiki/1912%EB%85%84" title="1912년">1912년</a>, 주가 폐지되면서 숴저우는 숴 현(朔県)으로 개칭되어 옌먼 도(雁門道)에 속했지만 도의 폐지에 따라 이듬해 산시 성으로 이관되었다.</p>
<p><a href="/wiki/%EC%A4%91%ED%99%94%EC%9D%B8%EB%AF%BC%EA%B3%B5%ED%99%94%EA%B5%AD" title="중화인민공화국">중화인민공화국</a>이 성립하면서 처음에는 <a href="/wiki/%EC%B0%A8%ED%95%98%EC%96%BC_%EC%84%B1" title="차하얼 성">차하얼 성</a>의 관할이었지만 <a href="/wiki/1952%EB%85%84" title="1952년">1952년</a>에 산시 성으로 이관되었다. <a href="/wiki/1989%EB%85%84" title="1989년">1989년</a> 1월 숴 현, 핑루 현, 산인 현을 관할하는 지급시로서 숴저우 시가 성립되었고, <a href="/wiki/1993%EB%85%84" title="1993년">1993년</a> 7월 옌베이 지구의 폐지에 따라 그 관할이었던 옌 현, 유위 현, 화이런 현을 편입하여 현재에 이른다.</p>
<h2><span class="mw-headline" id=".EA.B4.80.EA.B4.91">관광</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%88%B4%EC%A0%80%EC%9A%B0_%EC%8B%9C&amp;action=edit&amp;section=3" title="부분 편집: 관광">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<div class="thumb tright">

<p>1056년 <a href="/wiki/%EC%9A%94%EB%82%98%EB%9D%BC" title="요나라">요나라</a> 때 지어진 잉셴 탑, 서예 박물관, 구석기 유적지가 있다.</p>
<h2><span class="mw-headline" id=".EA.B2.BD.EC.A0.9C">경제</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%88%B4%EC%A0%80%EC%9A%B0_%EC%8B%9C&amp;action=edit&amp;section=4" title="부분 편집: 경제">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>시의 주요 산업은 석탄 채굴업이고 다른 광물로 철, 보크사이트, 운모, 망간과 흑연이 있다. 다른 경제 부문들로 농업, 화학, 도자기 제조와 어업이 있다.</p>
