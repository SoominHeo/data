<h1 class="firstHeading" id="firstHeading" lang="ko">쉐보레 실버라도</h1>
<div class="mw-body-content" id="bodyContent">
<div id="siteSub">위키백과, 우리 모두의 백과사전.</div>
<div id="contentSub"></div>
<div class="mw-jump" id="jump-to-nav">
					이동:					<a href="#mw-head">둘러보기</a>, 					<a href="#p-search">검색</a>
</div>
<div class="mw-content-ltr" dir="ltr" id="mw-content-text" lang="ko"><script>function mfTempOpenSection(id){var block=document.getElementById("mf-section-"+id);block.className+=" open-block";block.previousSibling.className+=" open-block";}</script><p><b>쉐보레 실버라도</b>(<span lang="en" xml:lang="en">The Chevrolet Silverado</span>)는 <a href="/wiki/%EC%89%90%EB%B3%B4%EB%A0%88" title="쉐보레">쉐보레</a> 브랜드로 출시한 <a href="/wiki/%EC%A0%9C%EB%84%88%EB%9F%B4_%EB%AA%A8%ED%84%B0%EC%8A%A4" title="제너럴 모터스">제너럴 모터스</a>의 마지막 대형 픽업 트럭 모델이다.</p>
<p><a href="/wiki/1998%EB%85%84" title="1998년">1998년</a>에 <a href="/wiki/%EB%AF%B8%EA%B5%AD" title="미국">미국</a>에서 <a class="new" href="/w/index.php?title=%EC%89%90%EB%B3%B4%EB%A0%88_C/K&amp;action=edit&amp;redlink=1" title="쉐보레 C/K (없는 문서)">C/K 시리즈</a>의 후속 차종이자, <a class="mw-redirect" href="/wiki/%ED%8F%AC%EB%93%9C_%EC%9E%90%EB%8F%99%EC%B0%A8_%ED%9A%8C%EC%82%AC" title="포드 자동차 회사">포드</a> 에프 시리즈 (F-Series) 픽업 트럭 다음으로 큰 크기의 픽업 트럭으로 출시됐다. 실버라도는 <a href="/wiki/GMC_(%EC%9E%90%EB%8F%99%EC%B0%A8)" title="GMC (자동차)">GMC</a> 시에라 (Sierra)와 플랫폼을 공유하고 있다. 실버라도는 <a href="/wiki/2008%EB%85%84" title="2008년">2008년</a> 상반기 미국에서 231,320대가 팔렸다.<sup class="reference" id="cite_ref-1"><a href="#cite_note-1">[1]</a></sup> 실버라도와 경쟁하는 <a class="mw-redirect" href="/wiki/%ED%8F%AC%EB%93%9C_%EC%9E%90%EB%8F%99%EC%B0%A8_%ED%9A%8C%EC%82%AC" title="포드 자동차 회사">포드</a> 에프 시리즈는 같은 시기에 274,713대가 팔렸고<sup class="reference" id="cite_ref-2"><a href="#cite_note-2">[2]</a></sup>, <a href="/wiki/GMC_(%EC%9E%90%EB%8F%99%EC%B0%A8)" title="GMC (자동차)">GMC</a> 시에라는 83,174대가 팔렸다. <a href="/wiki/%EC%A0%9C%EB%84%88%EB%9F%B4_%EB%AA%A8%ED%84%B0%EC%8A%A4" title="제너럴 모터스">제너럴 모터스</a>의 브랜드인 <a href="/wiki/%EC%89%90%EB%B3%B4%EB%A0%88" title="쉐보레">쉐보레</a>, <a href="/wiki/GMC_(%EC%9E%90%EB%8F%99%EC%B0%A8)" title="GMC (자동차)">GMC</a>의 픽업트럭 판매량을 모두 합치면 <a class="mw-redirect" href="/wiki/%ED%8F%AC%EB%93%9C_%EC%9E%90%EB%8F%99%EC%B0%A8_%ED%9A%8C%EC%82%AC" title="포드 자동차 회사">포드</a> 에프 시리즈보다 14.5% 더 많이 팔렸으며 전년 상반기 대비 15% 증가한 수치이다. <a href="/wiki/%EC%89%90%EB%B3%B4%EB%A0%88_%EC%84%9C%EB%B2%84%EB%B2%88" title="쉐보레 서버번">쉐보레 서버번</a>과 <a href="/wiki/%EC%89%90%EB%B3%B4%EB%A0%88_%ED%83%80%ED%98%B8" title="쉐보레 타호">쉐보레 타호</a>의 차체를 공유한다. <a href="/wiki/%EB%8C%80%ED%95%9C%EB%AF%BC%EA%B5%AD" title="대한민국">대한민국</a>에서는 미국에서 직수입하여 <a href="/wiki/%ED%95%9C%EA%B5%ADGM" title="한국GM">한국GM</a>을 통해 판매를 하고 있으며, 중장기적으로는 <a href="/wiki/%EB%8C%80%ED%95%9C%EB%AF%BC%EA%B5%AD" title="대한민국">대한민국</a>의 <a href="/wiki/%ED%95%9C%EA%B5%ADGM" title="한국GM">한국GM</a> 현지 생산 공장을 통해 양산할 계획에 있다.</p>
