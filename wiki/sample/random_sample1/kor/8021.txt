<p><b>쉐보레 카프리스</b>(<span lang="en" xml:lang="en">Chevrolet Caprice</span>)는 <a href="/wiki/%EC%A0%9C%EB%84%88%EB%9F%B4_%EB%AA%A8%ED%84%B0%EC%8A%A4" title="제너럴 모터스">제너럴 모터스</a> <a href="/wiki/%EC%89%90%EB%B3%B4%EB%A0%88" title="쉐보레">쉐보레</a> 사업부에서 출시한 대형 승용차이다. <a href="/wiki/1965%EB%85%84" title="1965년">1965년</a> <a href="/wiki/%EC%89%90%EB%B3%B4%EB%A0%88" title="쉐보레">쉐보레</a>는 <a href="/wiki/%EC%89%90%EB%B3%B4%EB%A0%88_%EC%9E%84%ED%8C%94%EB%9D%BC" title="쉐보레 임팔라">임팔라</a>보다 비싼 카프리스를 출시했다. 1세대부터 4세대까지는 <a href="/wiki/%EC%89%90%EB%B3%B4%EB%A0%88_%EC%9E%84%ED%8C%94%EB%9D%BC" title="쉐보레 임팔라">쉐보레 임팔라</a>와 동일한 차종으로 판매되었으며 5세대부터는 <a class="mw-redirect" href="/wiki/%ED%98%B8%EC%A3%BC" title="호주">호주</a> <a href="/wiki/%ED%99%80%EB%8D%B4" title="홀덴">홀덴</a>의 <a href="/wiki/GM%EB%8C%80%EC%9A%B0_%EC%8A%A4%ED%85%8C%EC%9D%B4%EC%B8%A0%EB%A7%A8" title="GM대우 스테이츠맨">스테이츠맨</a>과 동일한 차종으로 판매되았다. 이중 5세대의 경우 <a href="/wiki/%EB%8C%80%ED%95%9C%EB%AF%BC%EA%B5%AD" title="대한민국">대한민국</a>에서는 <a href="/wiki/%ED%95%9C%EA%B5%ADGM" title="한국GM">한국GM</a>에서 <a href="/wiki/GM%EB%8C%80%EC%9A%B0_%EC%8A%A4%ED%85%8C%EC%9D%B4%EC%B8%A0%EB%A7%A8" title="GM대우 스테이츠맨">스테이츠맨</a>으로 판매되었고 6세대는 <a href="/wiki/GM%EB%8C%80%EC%9A%B0_%EB%B2%A0%EB%A6%AC%ED%83%80%EC%8A%A4" title="GM대우 베리타스">베리타스</a>로 판매되었다.</p>

<h2><span class="mw-headline" id="1.EC.84.B8.EB.8C.80">1세대</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%89%90%EB%B3%B4%EB%A0%88_%EC%B9%B4%ED%94%84%EB%A6%AC%EC%8A%A4&amp;action=edit&amp;section=1" title="부분 편집: 1세대">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p><a href="/wiki/1965%EB%85%84" title="1965년">1965년</a>에 출시되었다. GM의 B-바디를 기반으로 했고, 2도어 <a href="/wiki/%EC%BF%A0%ED%8E%98" title="쿠페">쿠페</a>, 4도어 하드탑 <a href="/wiki/%EC%84%B8%EB%8B%A8" title="세단">세단</a>, 4도어 <a href="/wiki/%EC%8A%A4%ED%85%8C%EC%9D%B4%EC%85%98_%EC%99%9C%EA%B1%B4" title="스테이션 왜건">스테이션 왜건</a>이 있었다.</p>
<div style="clear: both;"></div>
<h2><span class="mw-headline" id="2.EC.84.B8.EB.8C.80">2세대</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%89%90%EB%B3%B4%EB%A0%88_%EC%B9%B4%ED%94%84%EB%A6%AC%EC%8A%A4&amp;action=edit&amp;section=2" title="부분 편집: 2세대">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p><a href="/wiki/1971%EB%85%84" title="1971년">1971년</a>에 출시되었다. 역대 모델 중 차체 형식이 가장 많았는데, 2도어 <a href="/wiki/%EC%BF%A0%ED%8E%98" title="쿠페">쿠페</a>와 하드탑, 2도어 <a href="/wiki/%EC%BB%A8%EB%B2%84%ED%84%B0%EB%B8%94" title="컨버터블">컨버터블</a>, 4도어 <a href="/wiki/%EC%84%B8%EB%8B%A8" title="세단">세단</a> 및 하드탑, 4도어 <a href="/wiki/%EC%8A%A4%ED%85%8C%EC%9D%B4%EC%85%98_%EC%99%9C%EA%B1%B4" title="스테이션 왜건">스테이션 왜건</a> 등 6가지였다.</p>

<p><a href="/wiki/1990%EB%85%84" title="1990년">1990년</a> <a href="/wiki/4%EC%9B%94_12%EC%9D%BC" title="4월 12일">4월 12일</a>에 출시되었다. <a href="/wiki/%EC%89%90%EB%B3%B4%EB%A0%88" title="쉐보레">쉐보레</a> 차종 중 GM B-바디를 사용한 마지막 차종이며, 역대 카프리스 중 최후의 독자 모델이다. <a class="new" href="/w/index.php?title=%EB%B7%B0%EC%9D%B5_%EB%A1%9C%EB%93%9C%EB%A7%88%EC%8A%A4%ED%84%B0&amp;action=edit&amp;redlink=1" title="뷰익 로드마스터 (없는 문서)">뷰익 로드마스터</a>와 <a class="new" href="/w/index.php?title=%EC%98%AC%EC%A6%88%EB%AA%A8%EB%B9%8C_%EC%BB%A4%EC%8A%A4%ED%85%80_%ED%81%AC%EB%A3%A8%EC%A0%B8&amp;action=edit&amp;redlink=1" title="올즈모빌 커스텀 크루져 (없는 문서)">올즈모빌 커스텀 크루져</a> 등과 플랫폼을 같이 사용했다. 디자인은 딕 러진(Dick Ruzzin)이 지휘했으며, 그의 지휘 하에 있었던 벤 살바도르(Ben Salvador)의 제안을 기반으로 했다. 인테리어 디자인의 경우는 쉐보레 측의 인테리어 수석디자이너 <a class="new" href="/w/index.php?title=%EB%B8%94%EB%A0%88%EC%9D%B8_%EC%A0%A0%ED%82%A8%EC%8A%A4&amp;action=edit&amp;redlink=1" title="블레인 젠킨스 (없는 문서)">블레인 젠킨스</a>(Blaine Jenkins)가 담당해, 실내를 부드럽게 감싸는 간결한 대시보드가 장착되었다. 기술적으로는 기존의 B-바디를 그대로 쓰되 플랫폼 크기를 10% 줄이는 것을 목표로 삼아 개량이 이루어졌다.</p>
<p>왜건 버전은 뷰익 로드마스터 에스테이트 왜건/올즈모빌 커스텀 크루져 등과 완전히 똑같은 차체를 사용해 그릴과 엠블럼, 휠 정도만 다른 수준이었고, 경우에 따라선 모조 우드그레인이나 투톤 컬러를 선택할 수 있었다. 원래 개발 과정에서는 공기저항을 줄이기 위해 뒷창문을 지붕을 타고 넘어가는 디자인도 고려했지만 경첩 설계 때문에 뒷부분을 완전히 재설계해야 했기 때문(생산 및 개발비의 절감도 목표에 있었다)에 취소된 적이 있었다고 한다. 기존의 클래식(Classic)과 브로엄(Brougham) 트림은 기본형과 클래식으로 바뀌었고, <a href="/wiki/%EC%A0%9C%EB%84%88%EB%9F%B4_%EB%AA%A8%ED%84%B0%EC%8A%A4" title="제너럴 모터스">GM</a>에서는 실내외 디자인을 확 바꾼 이번 카프리스를 통해 북미에서의 실적을 끌어 올리려고 했다. 엔진으로는 200마력 4.3리터 V8엔진을 사용했으며, <a href="/wiki/1994%EB%85%84" title="1994년">1994년</a>년형부터는 <a href="/wiki/%EC%89%90%EB%B3%B4%EB%A0%88_%EC%BD%94%EB%A5%B4%EB%B2%B3" title="쉐보레 코르벳">코르벳</a>의 260마력 LT1 5.7리터 V8엔진을 디튠해 옵션으로 제공했다.</p>
<p>이전 세대처럼 <a class="new" href="/w/index.php?title=%EB%AA%A8%ED%84%B0_%ED%8A%B8%EB%A0%8C%EB%93%9C&amp;action=edit&amp;redlink=1" title="모터 트렌드 (없는 문서)">모터 트렌드</a>의 1991년 "올해의 차" 상을 수상했다. 4세대 역시 경찰차와 택시로 많이 쓰였는데 <a href="/wiki/1990%EB%85%84%EB%8C%80" title="1990년대">1990년대</a> 초반부터 <a href="/wiki/2000%EB%85%84%EB%8C%80" title="2000년대">2000년대</a> 초반까지 <a href="/wiki/%EB%AF%B8%EA%B5%AD" title="미국">미국</a> 전역에서 다녔다.</p>
<p>일반 세단과 9C1 모두 <a href="/wiki/1993%EB%85%84" title="1993년">1993년</a>에 후측면 펜더를 살짝 바꾸는 페이스 리프트를 거쳤고, 이 측면 디자인은 카프리스의 고성능형으로 잠시 부활한 <a href="/wiki/%EC%89%90%EB%B3%B4%EB%A0%88_%EC%9E%84%ED%8C%94%EB%9D%BC" title="쉐보레 임팔라">임팔라</a> 7세대에도 적용되었다. <a href="/wiki/1994%EB%85%84" title="1994년">1994년</a>에는 <a href="/wiki/%EC%89%90%EB%B3%B4%EB%A0%88_%EC%B9%B4%EB%A7%88%EB%A1%9C" title="쉐보레 카마로">카마로</a>의 스티어링 휠과 디지털 계기판 등을 이식해 인테리어를 바꾸었으며, <a href="/wiki/1995%EB%85%84" title="1995년">1995년</a>에는 C필러의 뒷창문 형상을 <a href="/wiki/BMW" title="BMW">BMW</a>의 호프마이스터 킥과 비슷한 형태로 끝부분을 잘라내 바꾸고 왜건 버전의 백미러도 세단과 같은 것을 공유했다. 중동 지역에서는 <a href="/wiki/1996%EB%85%84" title="1996년">1996년</a>에 <a href="/wiki/%EC%89%90%EB%B3%B4%EB%A0%88_%EC%9E%84%ED%8C%94%EB%9D%BC" title="쉐보레 임팔라">임팔라 SS</a>를 카프리스 SS로 수출한 바가 있다. <a href="/wiki/1996%EB%85%84" title="1996년">1996년</a> <a href="/wiki/12%EC%9B%94_13%EC%9D%BC" title="12월 13일">12월 13일</a>에 <a class="new" href="/w/index.php?title=%EB%B7%B0%EC%9D%B5_%EB%A1%9C%EB%93%9C%EB%A7%88%EC%8A%A4%ED%84%B0&amp;action=edit&amp;redlink=1" title="뷰익 로드마스터 (없는 문서)">뷰익 로드마스터</a>와 함께 후속 없이 단종되면서, <a href="/wiki/1926%EB%85%84" title="1926년">1926년</a>부터 70년간 이어졌던 B-바디 플랫폼도 완전히 퇴역했다.</p>

<p><a href="/wiki/1999%EB%85%84" title="1999년">1999년</a>에 출시되었다. 4세대까지는 독자적으로 개발했지만, 5세대부터는 <a href="/wiki/%ED%99%80%EB%8D%B4" title="홀덴">홀덴</a>을 OEM 방식으로 받는데 <a class="mw-redirect" href="/wiki/%ED%99%80%EB%8D%B4_%EC%8A%A4%ED%85%8C%EC%9D%B4%EC%B8%A0%EB%A7%A8" title="홀덴 스테이츠맨">스테이츠맨</a>을 <a href="/wiki/%EC%89%90%EB%B3%B4%EB%A0%88" title="쉐보레">쉐보레</a> 브랜드로 리뱃징되었다. 따라서 이전에 생산하던 카프리스와는 관련 없는 차종이 되었다. <a href="/wiki/%EB%B6%81%EC%95%84%EB%A9%94%EB%A6%AC%EC%B9%B4" title="북아메리카">북미</a>에는 판매되지 않았고, 주로 <a href="/wiki/%EC%A4%91%EB%8F%99" title="중동">중동</a> 시장에 판매되었다.</p>
