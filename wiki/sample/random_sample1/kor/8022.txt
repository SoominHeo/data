<p><a href="/wiki/%EB%8C%80%ED%95%9C%EB%AF%BC%EA%B5%AD" title="대한민국">대한민국</a>에서는 <a href="/wiki/GM%EB%8C%80%EC%9A%B0_%EB%9D%BC%EC%84%B8%ED%8B%B0#2.EC.84.B8.EB.8C.80.28J300.29" title="GM대우 라세티">라세티 프리미어</a>라는 차명으로 <a href="/wiki/2008%EB%85%84_11%EC%9B%94" title="2008년 11월">2008년 11월</a>에 출시되었으며, 세계적으로는 <a href="/wiki/%EC%89%90%EB%B3%B4%EB%A0%88" title="쉐보레">쉐보레</a> 브랜드 또는 <a href="/wiki/%ED%99%80%EB%8D%B4" title="홀덴">홀덴</a> 브랜드를 통해 크루즈라는 차명으로 판매된다. <a href="/wiki/%EB%AF%B8%EA%B5%AD" title="미국">미국</a>에서는 <a href="/wiki/%EC%89%90%EB%B3%B4%EB%A0%88_%EC%BD%94%EB%B0%9C%ED%8A%B8" title="쉐보레 코발트">코발트</a>의 후속 차종으로 선보였다. <a href="/wiki/%EB%8C%80%ED%95%9C%EB%AF%BC%EA%B5%AD" title="대한민국">대한민국</a>에서는 <a href="/wiki/%EC%A0%84%EB%9D%BC%EB%B6%81%EB%8F%84" title="전라북도">전라북도</a> <a href="/wiki/%EA%B5%B0%EC%82%B0%EC%8B%9C" title="군산시">군산시</a> <a href="/wiki/%EC%86%8C%EB%A3%A1%EB%8F%99" title="소룡동">소룡동</a>에 있는 <a href="/wiki/%ED%95%9C%EA%B5%ADGM" title="한국GM">한국GM</a> 공장에서 생산된다. <a href="/wiki/2011%EB%85%84_3%EC%9B%94" title="2011년 3월">2011년 3월</a>부터 <a href="/wiki/%ED%95%9C%EA%B5%ADGM" title="한국GM">GM대우</a>의 회사명이 <a href="/wiki/%ED%95%9C%EA%B5%ADGM" title="한국GM">한국GM</a>으로 변경되고, <a href="/wiki/%EC%89%90%EB%B3%B4%EB%A0%88" title="쉐보레">쉐보레</a> 브랜드를 도입하였다. 이에 따라 이미 <a href="/wiki/%EB%8C%80%ED%95%9C%EB%AF%BC%EA%B5%AD" title="대한민국">대한민국</a> 이외의 국가에서 크루즈라는 차명으로 판매되는 기존의 <a class="mw-redirect" href="/wiki/%EB%8C%80%EC%9A%B0_%EB%9D%BC%EC%84%B8%ED%8B%B0#2.EC.84.B8.EB.8C.80.28J300.29" title="대우 라세티">라세티 프리미어</a>의 차명이 <a href="/wiki/%EB%8C%80%ED%95%9C%EB%AF%BC%EA%B5%AD" title="대한민국">대한민국</a>에서도 크루즈로 변경되었다. <a href="/wiki/2011%EB%85%84" title="2011년">2011년</a> <a href="/wiki/6%EC%9B%94_1%EC%9D%BC" title="6월 1일">6월 1일</a>에는 5도어 <a href="/wiki/%ED%95%B4%EC%B9%98%EB%B0%B1" title="해치백">해치백</a> 차종인 크루즈 5가 시판에 들어갔다. <a href="/wiki/2012%EB%85%84" title="2012년">2012년</a> <a href="/wiki/2%EC%9B%94_1%EC%9D%BC" title="2월 1일">2월 1일</a>에는 17인치 블랙 알루미늄 휠과 블랙 가죽 시트, 리어 스포일러가 적용된 더 퍼팩트 블랙이 선보였다.<sup class="reference" id="cite_ref-1"><a href="#cite_note-1">[1]</a></sup> 그해 <a href="/wiki/3%EC%9B%94" title="3월">3월</a>에 개최된 <a class="new" href="/w/index.php?title=%EC%A0%9C%EB%84%A4%EB%B0%94_%EB%AA%A8%ED%84%B0%EC%87%BC&amp;action=edit&amp;redlink=1" title="제네바 모터쇼 (없는 문서)">제네바 모터쇼</a>에서 <a href="/wiki/%EC%8A%A4%ED%85%8C%EC%9D%B4%EC%85%98_%EC%99%9C%EA%B1%B4" title="스테이션 왜건">스테이션 왜건</a>이 선보였으나, <a href="/wiki/%EC%8A%A4%ED%85%8C%EC%9D%B4%EC%85%98_%EC%99%9C%EA%B1%B4" title="스테이션 왜건">스테이션 왜건</a>의 인기가 상대적으로 낮은 <a href="/wiki/%EB%8C%80%ED%95%9C%EB%AF%BC%EA%B5%AD" title="대한민국">대한민국</a>에서는 판매되지 않는 대신 해당 자리는 같은 플랫폼을 쓰는 미니밴인 <a href="/wiki/%EC%89%90%EB%B3%B4%EB%A0%88_%EC%98%AC%EB%9E%80%EB%8F%84" title="쉐보레 올란도">올란도</a>가 대신하고 있다. 그해 <a href="/wiki/6%EC%9B%94_4%EC%9D%BC" title="6월 4일">6월 4일</a>에는 범퍼와 라디에이터 그릴의 디자인의 변경, 1.6ℓ 가솔린 엔진의 삭제, 마이 링크 적용 등의 변화가 이뤄진 더 퍼펙트 크루즈가 출시되었다. <a href="/wiki/2013%EB%85%84" title="2013년">2013년</a> <a href="/wiki/4%EC%9B%94_17%EC%9D%BC" title="4월 17일">4월 17일</a>에는 변속감이 향상된 차세대 GEN Ⅱ 6단 자동변속기와 방향 지시등 내장 아웃 사이드 미러 등이 적용된 <a href="/wiki/2014%EB%85%84" title="2014년">2014년</a>형을 선보였다. 같은 해 <a href="/wiki/8%EC%9B%94" title="8월">8월</a>에는 <a href="/wiki/%EB%8C%80%ED%95%9C%EB%AF%BC%EA%B5%AD" title="대한민국">대한민국</a>에서의 판매량이 적은 6단 <a href="/wiki/%EC%88%98%EB%8F%99%EB%B3%80%EC%86%8D%EA%B8%B0" title="수동변속기">수동변속기</a>가 단종되었고, <a href="/wiki/10%EC%9B%94_18%EC%9D%BC" title="10월 18일">10월 18일</a>에는 <a href="/wiki/%EC%89%90%EB%B3%B4%EB%A0%88_%ED%8A%B8%EB%9E%99%EC%8A%A4" title="쉐보레 트랙스">트랙스</a>에 장착되는 1.4ℓ 가솔린 터보 엔진이 장착된 크루즈 터보가 선보였다.<sup class="reference" id="cite_ref-2"><a href="#cite_note-2">[2]</a></sup> 크루즈 터보의 최고 출력은 <a href="/wiki/%EC%89%90%EB%B3%B4%EB%A0%88_%ED%8A%B8%EB%9E%99%EC%8A%A4" title="쉐보레 트랙스">트랙스</a>와 같은 140마력이 아닌 130마력으로 디튠되었다. <a href="/wiki/2014%EB%85%84" title="2014년">2014년</a> <a href="/wiki/4%EC%9B%94_14%EC%9D%BC" title="4월 14일">4월 14일</a>에는 크롬 마감이 추가된 범퍼, 스포티한 사이드 스커트, 17인치 메탈 그레이 알루미늄 휠이 적용된 스타일 패키지가 추가되었다. 같은 해 <a href="/wiki/10%EC%9B%94_30%EC%9D%BC" title="10월 30일">10월 30일</a>에 <a href="/wiki/%ED%95%9C%EA%B5%ADGM" title="한국GM">한국GM</a>은 1.8ℓ 가솔린 엔진의 공인 연비가 과장된 점을 <a href="/wiki/%EB%8C%80%ED%95%9C%EB%AF%BC%EA%B5%AD_%EA%B5%AD%ED%86%A0%EA%B5%90%ED%86%B5%EB%B6%80" title="대한민국 국토교통부">국토교통부</a>에 자진 신고했고,<sup class="reference" id="cite_ref-3"><a href="#cite_note-3">[3]</a></sup> 공인 연비도 정정했다.</p>

<h3><span class="mw-headline" id=".EC.96.B4.EB.A9.94.EC.9D.B4.EC.A7.95_.EB.89.B4_.ED.81.AC.EB.A3.A8.EC.A6.88.282015.EB.85.84_1.EC.9B.94.7E2017.EB.85.84_1.EC.9B.94.29">어메이징 뉴 크루즈(2015년 1월~2017년 1월)</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%89%90%EB%B3%B4%EB%A0%88_%ED%81%AC%EB%A3%A8%EC%A6%88&amp;action=edit&amp;section=3" title="부분 편집: 어메이징 뉴 크루즈(2015년 1월~2017년 1월)">편집</a><span class="mw-editsection-bracket">]</span></span></h3>

<p><a href="/wiki/2015%EB%85%84" title="2015년">2015년</a> <a href="/wiki/1%EC%9B%94_22%EC%9D%BC" title="1월 22일">1월 22일</a>에 출시되었다. 페이스 리프트를 거쳐 이중으로 나눠진 듀얼 포트 크롬 라디에이터 그릴과 LED 포지셔닝 램프가 새롭게 적용되었다. 뒷 번호판 장착 위치가 트렁크 리드에서 범퍼 하단으로 옮겨졌고, 리어 램프는 듀얼 스퀘어 모양으로 바뀌었다. 인테리어 색상으로 브라운 톤의 새들 업 색상이 추가되었다. 같은 해 <a href="/wiki/7%EC%9B%94_16%EC%9D%BC" title="7월 16일">7월 16일</a>에는 프로젝션 헤드 램프와 신규 익스테리어 색상이 적용된 <a href="/wiki/2016%EB%85%84" title="2016년">2016년</a>형이 출시되었다. 2.0ℓ 디젤 엔진은 <a href="/wiki/2016%EB%85%84" title="2016년">2016년</a>형에서 제외되었으며, <a href="/wiki/10%EC%9B%94_12%EC%9D%BC" title="10월 12일">10월 12일</a>에는 블랙 바디 컬러와 스페셜 데칼, 17인치 블랙 알루미늄 휠 등이 적용된 퍼펙트 블랙 에디션이 선보였다. <a href="/wiki/2016%EB%85%84" title="2016년">2016년</a> <a href="/wiki/2%EC%9B%94_17%EC%9D%BC" title="2월 17일">2월 17일</a>에 유로 6 규제를 충족시키는 1.6ℓ 디젤 엔진이 추가되었으며, 출시에 앞서 사전 계약이 실시되었다.</p>

<h3><span class="mw-headline" id=".EC.98.AC_.EB.89.B4_.ED.81.AC.EB.A3.A8.EC.A6.88.282017.EB.85.84_1.EC.9B.94.7E.ED.98.84.EC.9E.AC.29">올 뉴 크루즈(2017년 1월~현재)</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%89%90%EB%B3%B4%EB%A0%88_%ED%81%AC%EB%A3%A8%EC%A6%88&amp;action=edit&amp;section=5" title="부분 편집: 올 뉴 크루즈(2017년 1월~현재)">편집</a><span class="mw-editsection-bracket">]</span></span></h3>

<p><a href="/wiki/2017%EB%85%84" title="2017년">2017년</a> <a href="/wiki/1%EC%9B%94_17%EC%9D%BC" title="1월 17일">1월 17일</a>에 출시되었다. 동급 최대의 차체 크기를 지녔고, 낮은 공기 저항 계수(0.28Cd)를 실현하였다. 차체 크기가 커졌음에도 이전보다 공차 중량을 최대 110kg까지 줄였다. <a href="/wiki/%EC%89%90%EB%B3%B4%EB%A0%88" title="쉐보레">쉐보레</a>의 새로운 패밀리 룩으로 자리잡은 듀얼 포트 라디에이터 그릴과 볼륨감 있는 입체적인 캐릭터 라인이 적용되었다. 듀얼 콕핏 센터페시아를 중심으로 하는 인테리어는 인체 공학적인 설계가 적용되었으며, 실내 공간 확대와 소재 품질 향상에 주력하였다. 차체의 74.6%에 초고장력 및 고장력 강판을 적용되었고, 성형에도 유리한 첨단 소부경화강 적용 비율을 21%까지 높여 높은 수준의 안전성을 확보하였다.</p>
