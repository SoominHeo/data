<p><b>쉼표</b> 또는 <b>콤마</b>(comma)는 <a href="/wiki/%EB%AC%B8%EC%9E%A5_%EB%B6%80%ED%98%B8" title="문장 부호">문장 부호</a>의 하나이다. 반점(,), 모점(、), 가운뎃점(·), 쌍점(:), 빗금(/)이 있으며 대개 반점을 가리킨다. 대개의 글꼴에서 <a href="/wiki/%EC%95%84%ED%8F%AC%EC%8A%A4%ED%8A%B8%EB%A1%9C%ED%94%BC" title="아포스트로피">아포스트로피</a>나 <a href="/wiki/%EB%94%B0%EC%98%B4%ED%91%9C" title="따옴표">작은따옴표</a>와 모양이 같지만 글에 놓이는 위치는 다르다. 문장 가운데에 놓여 개념을 나열하고 기술한다.</p>
<h2><span class="mw-headline" id=".EC.96.B8.EC.96.B4.EB.B3.84_.EC.82.AC.EC.9A.A9">언어별 사용</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%89%BC%ED%91%9C_(%EB%AC%B8%EC%9E%A5_%EB%B6%80%ED%98%B8)&amp;action=edit&amp;section=1" title="부분 편집: 언어별 사용">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<ul>
<li><a href="/wiki/%EB%8C%80%ED%95%9C%EB%AF%BC%EA%B5%AD" title="대한민국">대한민국</a>에서의 사용</li>
</ul>
<p>대한민국에서의 반점과 모점의 사용법은 가로쓰기에서는 반점(,) 세로쓰기에서는 모점(、)을 사용하였으나 세로쓰기가 줄어듬을 이유로 한글 맞춤법 부록: 문장 부호에서 고리점(。)과 함께 삭제되었다.</p>
<ul>
<li><a href="/wiki/%EC%9D%BC%EB%B3%B8" title="일본">일본</a>에서의 사용</li>
</ul>
<p>일본에서는 숫자 단위에서는 반점(,)을 쓰며, 평소에는 모점(、)을 사용한다.</p>
<ul>
<li><a href="/wiki/%EC%A4%91%EA%B5%AD" title="중국">중국</a>에서의 사용</li>
</ul>
<p>중국에서는 문장을 나눌 때에는 반점(,)을 사용하고, 명사의 나열은 모점(、)을 사용한다.</p>
<ul>
<li>영어권 국가에서의 사용</li>
</ul>
<p>영어권 국가에서는 모점(、)이 존재하지 않으며 반점(,)만이 사용된다.</p>
<ul>
<li><a href="/wiki/%EC%A1%B0%EC%84%A0%EB%AF%BC%EC%A3%BC%EC%A3%BC%EC%9D%98%EC%9D%B8%EB%AF%BC%EA%B3%B5%ED%99%94%EA%B5%AD" title="조선민주주의인민공화국">조선민주주의인민공화국</a>에서의 사용</li>
</ul>
<p>모점(、)을 사용하지 않으며, 반점(,)만을 사용한다.</p>
<h2><span class="mw-headline" id=".EC.BB.B4.ED.93.A8.ED.84.B0">컴퓨터</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%89%BC%ED%91%9C_(%EB%AC%B8%EC%9E%A5_%EB%B6%80%ED%98%B8)&amp;action=edit&amp;section=2" title="부분 편집: 컴퓨터">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>컴퓨터 환경의 <a href="/wiki/%EC%9C%A0%EB%8B%88%EC%BD%94%EB%93%9C" title="유니코드">유니코드</a>와 <a class="mw-disambig" href="/wiki/%EC%95%84%EC%8A%A4%ED%82%A4" title="아스키">아스키</a> 체계에서 44번째 문자 (<a href="/wiki/%EC%8B%AD%EC%9C%A1%EC%A7%84%EB%B2%95" title="십육진법">0x002C</a>)와 쉼표 기호가 일치한다.</p>
<p><a href="/wiki/CSV_(%ED%8C%8C%EC%9D%BC_%ED%98%95%EC%8B%9D)" title="CSV (파일 형식)">CSV</a>(comma-separated values) 형식은 쉼표를 기준으로 항목을 구분하여 저장한 데이터를 가리킨다.</p>
