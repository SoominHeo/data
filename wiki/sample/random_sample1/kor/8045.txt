<p><b>슈타이너 분견군</b>(<span style="font-size: smaller;"><a href="/wiki/%EC%98%81%EC%96%B4" title="영어">영어</a>:</span> <span lang="en" xml:lang="en">Armeeabteilung Steiner</span>)은 <a href="/wiki/%EC%95%BC%EC%A0%84%EA%B5%B0" title="야전군">야전군</a>보다 작고 <a href="/wiki/%EA%B5%B0%EB%8B%A8" title="군단">군단</a>보다 규모가 큰 임시 편제로 1945년 4월 21일 <a href="/wiki/%EB%B2%A0%EB%A5%BC%EB%A6%B0_%EA%B3%B5%EB%B0%A9%EC%A0%84" title="베를린 공방전">베를린 공방전</a> 당시 총통 <a href="/wiki/%EC%95%84%EB%8F%8C%ED%94%84_%ED%9E%88%ED%8B%80%EB%9F%AC" title="아돌프 히틀러">아돌프 히틀러</a>에 의해 서류상으로만 편성된 부대였다. 이 분견군의 지휘관은 <a href="/wiki/%ED%8E%A0%EB%A6%AD%EC%8A%A4_%EC%8A%88%ED%83%80%EC%9D%B4%EB%84%88" title="펠릭스 슈타이너">펠릭스 슈타이너</a> <a class="mw-redirect" href="/wiki/SS%EB%8C%80%EC%9E%A5" title="SS대장">SS대장</a>이었다. 히틀러는 슈타이너 SS대장이 동 분견군을 이끌고 베를린을 목표로 하고 있던 소련군의 북쪽 주공점에 효과적인 역공을 가하길 바랐었다. 4월 22일 작전 회의 당시 히틀러는 실제로 그 역공이 불가능하다는 걸 깨닫자 눈물까지 흘리며 분노를 표출했다. 그는 전쟁에 패배했다고 선언하고 장군들을 비난했다. 그리고 함락될 때까지 베를린에 머물겠으며, 그후에는 자살하겠다고 밝혔다.</p>
<p>베를린 공방전 둘째날인 4월 17일, <a class="new" href="/w/index.php?title=%EB%B0%94%EC%9D%B5%EC%85%80_%EC%A7%91%EB%8B%A8%EA%B5%B0&amp;action=edit&amp;redlink=1" title="바익셀 집단군 (없는 문서)">바익셀 집단군</a> 사령관 <a class="new" href="/w/index.php?title=%EA%B3%A0%ED%8A%B8%ED%95%98%EB%A5%B4%ED%8A%B8_%ED%95%98%EC%9D%B8%EB%A6%AC%EC%B9%98&amp;action=edit&amp;redlink=1" title="고트하르트 하인리치 (없는 문서)">고트하르트 하인리치</a> 상급대장은 집단군 예비 부대인 슈타이너 SS대장의 <a class="new" href="/w/index.php?title=3_SS_%EA%B8%B0%EA%B0%91_%EA%B5%B0%EB%8B%A8&amp;action=edit&amp;redlink=1" title="3 SS 기갑 군단 (없는 문서)">3 SS 기갑 군단</a>에서 가장 강력한 전력을 자랑했던 <a class="mw-redirect" href="/wiki/11_SS_%EC%9D%98%EC%9A%A9_%EA%B8%B0%EA%B0%91%EC%B2%99%ED%83%84%EB%B3%91_%EC%82%AC%EB%8B%A8_%EB%85%B8%EB%A5%B4%ED%8A%B8%EB%9E%80%ED%8A%B8" title="11 SS 의용 기갑척탄병 사단 노르트란트">11 SS 의용 기갑척탄병 사단 노르트란트</a>와 <a class="mw-redirect" href="/wiki/23_SS_%EC%9D%98%EC%9A%A9_%EA%B8%B0%EA%B0%91%EC%B2%99%ED%83%84%EB%B3%91_%EC%82%AC%EB%8B%A8_%EB%84%A4%EB%8D%B0%EB%A5%B4%EB%9E%80%ED%8A%B8" title="23 SS 의용 기갑척탄병 사단 네데르란트">23 SS 의용 기갑척탄병 사단 네데르란트</a> 두 사단을 차출했다. 하인리치 상급대장은 <a class="new" href="/w/index.php?title=%ED%85%8C%EC%98%A4%EB%8F%84%EC%96%B4_%EB%B6%80%EC%84%B8&amp;action=edit&amp;redlink=1" title="테오도어 부세 (없는 문서)">테오도어 부세</a> 보병대장의 <a class="mw-redirect" href="/wiki/9_%EA%B5%B0_(%EB%8F%85%EC%9D%BC_%EA%B5%AD%EB%B0%A9%EA%B5%B0)" title="9 군 (독일 국방군)">9 군</a>에 두 사단과 3 SS 기갑 군단에 속해 있던 여타 부대 대부분을 배속시켰다. 노르트란트 사단은 <a class="new" href="/w/index.php?title=9_%EA%B3%B5%EC%88%98_%EC%82%AC%EB%8B%A8_(%EB%8F%85%EC%9D%BC)&amp;action=edit&amp;redlink=1" title="9 공수 사단 (독일) (없는 문서)">9 공수 사단</a>이 수비하고 있던 <a class="new" href="/w/index.php?title=%EC%A0%A4%EB%A1%9C_%EA%B3%A0%EC%A7%80&amp;action=edit&amp;redlink=1" title="젤로 고지 (없는 문서)">젤로 고지</a>의 방어를 강화하기 위해 <a href="/wiki/%ED%97%AC%EB%AC%B4%ED%8A%B8_%EB%B0%94%EC%9D%B4%ED%8B%80%EB%A7%81" title="헬무트 바이틀링">헬무트 바이틀링</a> 포병대장의 <a class="new" href="/w/index.php?title=56_%EA%B8%B0%EA%B0%91_%EA%B5%B0%EB%8B%A8&amp;action=edit&amp;redlink=1" title="56 기갑 군단 (없는 문서)">56 기갑 군단</a>에 배속되었다. 네데르란트 사단은 <a class="mw-redirect" href="/wiki/%ED%94%84%EB%9E%91%ED%81%AC%ED%91%B8%EB%A5%B4%ED%8A%B8_%EC%95%88_%EB%8D%B0%EC%96%B4_%EC%98%A4%EB%8D%B0%EB%A5%B4" title="프랑크푸르트 안 데어 오데르">프랑크푸르트 안 데어 오데르</a> 남서쪽으로 보내져 <a class="new" href="/w/index.php?title=5_SS_%EC%82%B0%EC%95%85_%EA%B5%B0%EB%8B%A8&amp;action=edit&amp;redlink=1" title="5 SS 산악 군단 (없는 문서)">5 SS 산악 군단</a>에 배속되었으며, <a class="new" href="/w/index.php?title=%ED%95%A0%EB%B2%A0_%EC%A0%84%ED%88%AC&amp;action=edit&amp;redlink=1" title="할베 전투 (없는 문서)">할베 전투</a>에서 괴멸하는 운명을 맞이하게 되었다.<sup class="reference" id="cite_ref-1"><a href="#cite_note-1">[1]</a></sup></p>
<p>하인리치 상급대장은 <a class="new" href="/w/index.php?title=%EC%A0%A4%EB%A1%9C_%EA%B3%A0%EC%A7%80&amp;action=edit&amp;redlink=1" title="젤로 고지 (없는 문서)">젤로 고지</a>를 돌파한 후 베를린을 포위 중에 있던 <a href="/wiki/%EA%B2%8C%EC%98%A4%EB%A5%B4%EA%B8%B0_%EC%A3%BC%EC%BD%94%ED%94%84" title="게오르기 주코프">주코프</a> 원수의 소련 제1 벨로루시 전선군으로부터 <a class="new" href="/w/index.php?title=3_%EA%B8%B0%EA%B0%91%EA%B5%B0_(%EB%8F%85%EC%9D%BC)&amp;action=edit&amp;redlink=1" title="3 기갑군 (독일) (없는 문서)">3 기갑군</a>의 남익을 방어하기 위해 3개 대대와 한줌의 전차로 줄어버린 제3 SS 기갑 군단에 어떻게든 병력을 그러모아 핀노 운하를 따라 방어선을 구축하라고 명령했다.<sup class="reference" id="cite_ref-Beevor-267-268_2-0"><a href="#cite_note-Beevor-267-268-2">[2]</a></sup><sup class="reference" id="cite_ref-Ziemke-87-88_3-0"><a href="#cite_note-Ziemke-87-88-3">[3]</a></sup></p>
<p><a href="/wiki/4%EC%9B%94_21%EC%9D%BC" title="4월 21일">4월 21일</a>, 현실을 직시하지 못하고 있던 아돌프 히틀러는 전력이 피폐해진 부대들을 슈타이너 SS대장 휘하에 모아 슈타이너 분견군을 조직했다. 그리고 슈타이너 SS대장에게 제1 벨로루시 전선군의 돌파에 의해 생긴 거대 돌출부의 북익을 공격하라고 명령했다. 그와 동시에 제9군이 돌출부 남쪽을 공격할 예정이었다. 이 공격을 위해 슈타이너 SS대장은—핀로 운하 북쪽에 배치되어 있던<sup class="reference" id="cite_ref-4"><a href="#cite_note-4">[4]</a></sup>—제9군 소속의 <a class="mw-redirect" href="/wiki/4_SS%EA%B2%BD%EC%B0%B0%EA%B8%B0%EA%B0%91%EC%B2%99%ED%83%84%EB%B3%91%EC%82%AC%EB%8B%A8" title="4 SS경찰기갑척탄병사단">4 SS경찰기갑척탄병사단</a>, <a class="new" href="/w/index.php?title=5_%EA%B2%BD%EB%B3%B4%EB%B3%91_%EC%82%AC%EB%8B%A8_(%EB%8F%85%EC%9D%BC)&amp;action=edit&amp;redlink=1" title="5 경보병 사단 (독일) (없는 문서)">5 경보병 사단</a>, <a class="new" href="/w/index.php?title=25_%EA%B8%B0%EA%B0%91%EC%B2%99%ED%83%84%EB%B3%91_%EC%82%AC%EB%8B%A8_(%EB%8F%85%EC%9D%BC)&amp;action=edit&amp;redlink=1" title="25 기갑척탄병 사단 (독일) (없는 문서)">25 기갑척탄병 사단</a> 3개 사단을 배속받았다. 바이틀링 포병대장의 56 기갑 군단은 <a class="new" href="/w/index.php?title=%EB%B2%A0%EB%A5%B4%EB%85%B8%EC%9D%B4%ED%97%A8&amp;action=edit&amp;redlink=1" title="베르노이헨 (없는 문서)">베르노이헨</a> 바로 아래 북익을 둔 채 베를린 동쪽에 위치해 있었다.<sup class="reference" id="cite_ref-Beevor-267-268_2-1"><a href="#cite_note-Beevor-267-268-2">[2]</a></sup><sup class="reference" id="cite_ref-Ziemke-87-88_3-1"><a href="#cite_note-Ziemke-87-88-3">[3]</a></sup></p>
<p>북익에 있던 앞서 언급한 3개 사단은 <a class="new" href="/w/index.php?title=%EC%97%90%EB%B2%84%EC%8A%A4%EB%B0%9C%EB%8D%B0&amp;action=edit&amp;redlink=1" title="에버스발데 (없는 문서)">에버스발데</a>(베를린 동쪽 24킬로미터)에서 남쪽에 위치한 제56 기갑 군단 방향으로 공격해 나아가 <a class="new" href="/w/index.php?title=1_%EB%B2%A8%EB%A1%9C%EB%A3%A8%EC%8B%9C_%EC%A0%84%EC%84%A0%EA%B5%B0&amp;action=edit&amp;redlink=1" title="1 벨로루시 전선군 (없는 문서)">1 벨로루시 전선군</a>의 돌출부를 분단할 예정이었다. 슈타이너 SS대장은 하인리치 상급대장에게 전화해 제5 경보병 사단과 제25 기갑척탄병 사단을 방어 임무에 투입해야 하고, <a class="mw-redirect" href="/wiki/2_%ED%95%B4%EA%B5%B0_%EB%B3%B4%EB%B3%91_%EC%82%AC%EB%8B%A8_(%EB%8F%85%EC%9D%BC)" title="2 해군 보병 사단 (독일)">2 해군 사단</a>이 내륙에 도착해 교대하기 전까지는 두 사단을 빼낼 수 없다며 계획의 실행이 불가능함을 알렸다. 이를 제외하고 작전에 동원 가능한 건 무장조차 제대로 갖추지 못한 4 SS 경찰 기갑척탄병 사단의 2개 대대뿐이었다. 하인리치 상급대장은 육군 참모총장인 <a class="mw-disambig" href="/wiki/%ED%95%9C%EC%8A%A4_%ED%81%AC%EB%A0%99%EC%8A%A4" title="한스 크렙스">한스 크렙스</a> 보병대장에게 전화해 계획의 불가함을 말하며 히틀러에게 계획의 재고를 요청했지만, 총통이 바쁘다는 소리만을 들을 수 있을 뿐이었다.<sup class="reference" id="cite_ref-Beevor-267-268_2-2"><a href="#cite_note-Beevor-267-268-2">[2]</a></sup><sup class="reference" id="cite_ref-Ziemke-87-88_3-2"><a href="#cite_note-Ziemke-87-88-3">[3]</a></sup></p>
<p>4월 22일 오후 작전 회의에서 히틀러는 슈타이너 SS대장이 공격을 가하지 않을 걸 확신하고 눈물까지 흘릴 정도로 격분하고 말았다. 그는 전쟁에 패배했다고 선언하고 장군을 비난했다. 그리고 종말이 올 때까지 베를린에 머물겠으며, 그후에는 자살하겠다고 밝혔다.<sup class="reference" id="cite_ref-Beevor-275_5-0"><a href="#cite_note-Beevor-275-5">[5]</a></sup><sup class="reference" id="cite_ref-Ziemke-89_6-0"><a href="#cite_note-Ziemke-89-6">[6]</a></sup> 4월 22일 이후 슈타이너 분견군은 <a class="new" href="/w/index.php?title=%EC%B4%9D%ED%86%B5_%EB%B2%99%EC%BB%A4&amp;action=edit&amp;redlink=1" title="총통 벙커 (없는 문서)">총통 벙커</a>에서 더 이상 언급되지 않았다.<sup class="reference" id="cite_ref-Beevor-298_7-0"><a href="#cite_note-Beevor-298-7">[7]</a></sup></p>
<h2><span class="mw-headline" id=".EB.8D.94_.EB.B3.B4.EA.B8.B0">더 보기</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%8A%88%ED%83%80%EC%9D%B4%EB%84%88_%EB%B6%84%EA%B2%AC%EA%B5%B0&amp;action=edit&amp;section=1" title="부분 편집: 더 보기">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p><a class="mw-redirect" href="/wiki/11_SS_%EA%B8%B0%EA%B0%91%EA%B5%B0" title="11 SS 기갑군">11 SS 기갑군</a>은 <a class="new" href="/w/index.php?title=%EB%B0%94%EC%9D%B5%EC%85%80_%EC%A7%91%EB%8B%A8%EA%B5%B0&amp;action=edit&amp;redlink=1" title="바익셀 집단군 (없는 문서)">바익셀 집단군</a> 소속으로 슈타이너 SS대장 지휘하에 1945년 2월 당시 <a href="/wiki/%EC%98%A4%EB%8D%B0%EB%A5%B4_%EA%B0%95" title="오데르 강">오데르 강</a> 동안에서 전투를 벌였다. 1945년 3월 동군은 11 군으로 개칭 및 재편되어 서방 연합군을 상대하기 위해 <a class="new" href="/w/index.php?title=%EB%8F%85%EC%9D%BC_%EC%84%9C%EB%B6%80_%EC%A0%84%EA%B5%AC_%EC%B4%9D%EC%82%AC%EB%A0%B9%EB%B6%80&amp;action=edit&amp;redlink=1" title="독일 서부 전구 총사령부 (없는 문서)">서부 전구 총사령부</a>에 배속되었다. 슈타이너 SS대장은 <a class="new" href="/w/index.php?title=3_SS_%EA%B8%B0%EA%B0%91_%EA%B5%B0%EB%8B%A8%EC%9E%A5&amp;action=edit&amp;redlink=1" title="3 SS 기갑 군단장 (없는 문서)">3 SS 기갑 군단장</a>으로 복귀했고, 동 군단은 11 군에서 바익셀 집단군 소속의 <a class="new" href="/w/index.php?title=3_%EA%B8%B0%EA%B0%91%EA%B5%B0&amp;action=edit&amp;redlink=1" title="3 기갑군 (없는 문서)">3 기갑군</a>으로 전속되었다. 1945년 당시 슈타이너 SS대장이 11 군을 지휘했기 때문인지 종종 슈타이너 분견군과 혼동되고 있다.</p>
