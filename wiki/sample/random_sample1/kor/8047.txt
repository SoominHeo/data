<p><b>슈테판 츠바이크</b>(<span style="font-size: smaller;"><a href="/wiki/%EB%8F%85%EC%9D%BC%EC%96%B4" title="독일어">독일어</a>:</span> <span lang="de" xml:lang="de">Stefan Zweig</span>, <a href="/wiki/1881%EB%85%84" title="1881년">1881년</a> <a href="/wiki/11%EC%9B%94_28%EC%9D%BC" title="11월 28일">11월 28일</a> ~ <a href="/wiki/1942%EB%85%84" title="1942년">1942년</a> <a href="/wiki/2%EC%9B%94_22%EC%9D%BC" title="2월 22일">2월 22일</a>)는 <a href="/wiki/%EC%98%A4%EC%8A%A4%ED%8A%B8%EB%A6%AC%EC%95%84" title="오스트리아">오스트리아</a>의 <a href="/wiki/%EC%86%8C%EC%84%A4%EA%B0%80" title="소설가">소설가</a>·<a href="/wiki/%EC%A0%80%EB%84%90%EB%A6%AC%EC%8A%A4%ED%8A%B8" title="저널리스트">저널리스트</a>·<a href="/wiki/%EA%B7%B9%EC%9E%91%EA%B0%80" title="극작가">극작가</a>·전기작가이다. <a href="/wiki/%EB%B9%88" title="빈">빈</a>에서 태어났으며<sup class="reference" id="cite_ref-Lohrmann2003_1-0"><a href="#cite_note-Lohrmann2003-1">[1]</a></sup>, <a class="mw-redirect" href="/wiki/%EB%AF%BC%EC%A1%B1%EC%82%AC%ED%9A%8C%EC%A3%BC%EC%9D%98_%EB%8F%85%EC%9D%BC_%EB%85%B8%EB%8F%99%EC%9E%90%EB%8B%B9" title="민족사회주의 독일 노동자당">나치</a>가 정권을 잡자 <a href="/wiki/%EB%B8%8C%EB%9D%BC%EC%A7%88" title="브라질">브라질</a>로 망명하였다가, 마지막 작품인 《발자크》를 미처 완성하지 못한 채 <a href="/wiki/%ED%8E%98%ED%8A%B8%EB%A1%9C%ED%8F%B4%EB%A6%AC%EC%8A%A4" title="페트로폴리스">페트로폴리스</a>에서 젊은 아내와 함께 <a href="/wiki/%EC%9E%90%EC%82%B4" title="자살">자살</a>하였다. 형식적 완성미가 풍부하고, <a href="/wiki/%EC%A7%80%EA%B7%B8%EB%AC%B8%ED%8A%B8_%ED%94%84%EB%A1%9C%EC%9D%B4%ED%8A%B8" title="지그문트 프로이트">프로이트</a>의 <a href="/wiki/%EC%8B%AC%EB%A6%AC%ED%95%99" title="심리학">심리학</a>을 응용하여 쓴 우수한 단편 소설들이 많다. <a href="/wiki/%EC%9C%A0%EB%9F%BD" title="유럽">유럽</a> 문화의 전통을 유지하려고 노력하였고, 대표작으로 《감정의 혼란》(1926)이 있다.</p>

<h2><span class="mw-headline" id=".EC.83.9D.EC.95.A0">생애</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%8A%88%ED%85%8C%ED%8C%90_%EC%B8%A0%EB%B0%94%EC%9D%B4%ED%81%AC&amp;action=edit&amp;section=1" title="부분 편집: 생애">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>슈테판 츠바이크는 <a href="/wiki/%EC%98%A4%EC%8A%A4%ED%8A%B8%EB%A6%AC%EC%95%84" title="오스트리아">오스트리아</a>의 부유한 <a href="/wiki/%EC%A7%81%EB%AC%BC" title="직물">직물</a> 공장 대표인 아버지 모리츠 츠바이크와 <a href="/wiki/%EC%9C%A0%EB%8C%80%EC%9D%B8" title="유대인">유대계</a> 은행가 가문 출신인 어머니 이다 브레타우어 사이에서 태어났다. 20세에 시집 《은(銀)의 현(絃)》(1901)을 통하여 등단하였고, <a href="/wiki/%EB%B9%88_%EB%8C%80%ED%95%99%EA%B5%90" title="빈 대학교">빈 대학교</a>에서 <a href="/wiki/%EC%B2%A0%ED%95%99" title="철학">철학</a>을 공부하여 <a href="/wiki/1904%EB%85%84" title="1904년">1904년</a> 23세 되던 해 「이폴리트 테느의 철학」이라는 <a href="/wiki/%EB%85%BC%EB%AC%B8" title="논문">논문</a>으로 <a href="/wiki/%EB%B0%95%EC%82%AC" title="박사">박사</a>학위를 취득하였다. 그의 학업에 있어서 종교적 요소는 그다지 큰 영향을 미치지 않았지만, 그의 소설 《책벌레 멘델》에서 <a href="/wiki/%EC%9C%A0%EB%8C%80%EC%9D%B8" title="유대인">유대인</a>과 <a href="/wiki/%EC%9C%A0%EB%8C%80%EA%B5%90" title="유대교">유대교</a>에 대한 내용을 포함하는 등 자신의 <a href="/wiki/%EC%9C%A0%EB%8C%80%EA%B5%90" title="유대교">유대교</a>적 신념을 포기하지는 않았다. 비록 그의 <a href="/wiki/%EC%88%98%ED%95%84" title="수필">수필</a> 작품들이 <a class="mw-redirect" href="/wiki/%EC%8B%9C%EC%98%A4%EB%8B%88%EC%A6%98" title="시오니즘">시오니즘</a>을 선도하는 <a href="/wiki/%ED%85%8C%EC%98%A4%EB%8F%84%EC%96%B4_%ED%97%A4%EB%A5%B4%EC%B8%A8" title="테오도어 헤르츨">테오도어 헤르츨</a>이 운영하는 《신 자유신문》(<i>Neue Freie Presse</i>)에 실리기는 하였지만, 츠바이크는 헤르츨의 유대적 민족주의(<a class="mw-redirect" href="/wiki/%EC%8B%9C%EC%98%A4%EB%8B%88%EC%A6%98" title="시오니즘">시오니즘</a>)에 동조하지는 않았다.</p>
<p>또한 슈테판 츠바이크는 <a href="/wiki/%EC%B2%B4%EC%BD%94" title="체코">체코</a>의 작가 에곤 호스토프스키와 친했다. 호스토프스키는 츠바이크와의 관계를 ‘먼 친척되는 사이’라고 묘사하곤 했는데<sup class="reference" id="cite_ref-2"><a href="#cite_note-2">[2]</a></sup>, 몇몇 매체는 그들의 사이를 사촌이라고 표현하였다.</p>
<p><a class="mw-redirect" href="/wiki/%EC%A0%9C1%EC%B0%A8_%EC%84%B8%EA%B3%84%EB%8C%80%EC%A0%84" title="제1차 세계대전">제1차 세계대전</a>이 시작되고, 애국주의적 정서가 확대되었는데, 이때 츠바이크뿐만 아니라 <a href="/wiki/%EB%A7%88%EB%A5%B4%ED%8B%B4_%EB%B6%80%EB%B2%84" title="마르틴 부버">마르틴 부버</a>, 헤르만 코엔과 같은 독일인과 오스트리아계 유대인들도 지지의 의사를 내비쳤고, 이러한 정서는 더욱 널리 퍼져나갔다.<sup class="reference" id="cite_ref-3"><a href="#cite_note-3">[3]</a></sup>. 츠바이크는 비록 이에 동조하기는 하였지만, 전쟁에 직접 참여하는 것은 거부하였고, 대신 국방부 기록 보관소에서 근무하였다. 하지만 곧 그는 <a href="/wiki/1915%EB%85%84" title="1915년">1915년</a> <a href="/wiki/%EB%85%B8%EB%B2%A8_%EB%AC%B8%ED%95%99%EC%83%81" title="노벨 문학상">노벨 문학상</a> 수상자인 그의 친구 <a href="/wiki/%EB%A1%9C%EB%A7%B9_%EB%A1%A4%EB%9E%91" title="로맹 롤랑">로맹 롤랑</a>과 함께 <a href="/wiki/%ED%8F%89%ED%99%94%EC%A3%BC%EC%9D%98" title="평화주의">평화주의</a>를 접하게 되었고, 그 이후로 츠바이크는 인생 전반에 걸쳐서 <a href="/wiki/%ED%8F%89%ED%99%94%EC%A3%BC%EC%9D%98" title="평화주의">평화주의</a>를 주장하며 <a href="/wiki/%EC%9C%A0%EB%9F%BD" title="유럽">유럽</a>의 통합을 지지했다. 이후 롤랑처럼 츠바이크 또한 수많은 전기문을 썼는데, 《에라스무스 평전》이 대표작이다.</p>
<p><a href="/wiki/1934%EB%85%84" title="1934년">1934년</a> 츠바이크는 <a href="/wiki/%EC%95%84%EB%8F%8C%ED%94%84_%ED%9E%88%ED%8B%80%EB%9F%AC" title="아돌프 히틀러">아돌프 히틀러</a>의 <a href="/wiki/%EB%8F%85%EC%9D%BC" title="독일">독일</a>이 힘을 떨치자 이를 피해 아내 프리데리케와 함께 <a href="/wiki/%EC%98%A4%EC%8A%A4%ED%8A%B8%EB%A6%AC%EC%95%84" title="오스트리아">오스트리아</a>를 떠나 <a href="/wiki/%EB%9F%B0%EB%8D%98" title="런던">런던</a>으로 피신하였다. <a href="/wiki/1940%EB%85%84" title="1940년">1940년</a> <a class="mw-redirect" href="/wiki/%EB%82%98%EC%B9%98" title="나치">나치</a> <a href="/wiki/%EA%B5%B0%EB%8C%80" title="군대">군대</a>가 <a href="/wiki/%ED%94%84%EB%9E%91%EC%8A%A4" title="프랑스">프랑스</a>를 거쳐 <a href="/wiki/%EC%84%9C%EC%9C%A0%EB%9F%BD" title="서유럽">서유럽</a>으로 빠르게 진군하자 츠바이크 부부는 <a href="/wiki/%EB%8C%80%EC%84%9C%EC%96%91" title="대서양">대서양</a>을 건너 <a href="/wiki/%EB%AF%B8%EA%B5%AD" title="미국">미국</a> <a href="/wiki/%EB%89%B4%EC%9A%95" title="뉴욕">뉴욕</a>으로 옮겼고, 같은 해 <a href="/wiki/8%EC%9B%94_20%EC%9D%BC" title="8월 20일">8월 20일</a> 다시 <a href="/wiki/%EB%B8%8C%EB%9D%BC%EC%A7%88" title="브라질">브라질</a> <a href="/wiki/%EB%A6%AC%EC%9A%B0%EB%8D%B0%EC%9E%90%EB%84%A4%EC%9D%B4%EB%A3%A8" title="리우데자네이루">리우데자네이루</a>의 위성 도시인 <a href="/wiki/%ED%8E%98%ED%8A%B8%EB%A1%9C%ED%8F%B4%EB%A6%AC%EC%8A%A4" title="페트로폴리스">페트로폴리스</a>로 옮겼다<sup class="reference" id="cite_ref-DW_30.04.2009_4-0"><a href="#cite_note-DW_30.04.2009-4">[4]</a></sup>. 편협한 사고와 <a href="/wiki/%EA%B6%8C%EC%9C%84%EC%A3%BC%EC%9D%98" title="권위주의">권위주의</a> 그리고 <a href="/wiki/%EB%82%98%EC%B9%98%EC%A6%98" title="나치즘">나치즘</a>에 의해 그의 <a href="/wiki/%EC%9A%B0%EC%9A%B8%EC%A6%9D" title="우울증">우울증</a>은 깊어져만 갔고, 인류와 미래에 대한 희망이 사라져감을 느낀 츠바이크는 그의 자포자기적인 심정을 노트에 적었다. 결국 츠바이크 부부는 <a href="/wiki/1942%EB%85%84" title="1942년">1942년</a> <a href="/wiki/2%EC%9B%94_23%EC%9D%BC" title="2월 23일">2월 23일</a> 수면제 과다 복용으로 그들의 집에서 손을 잡고 죽은 채 발견되었다<sup class="reference" id="cite_ref-Time1942_5-0"><a href="#cite_note-Time1942-5">[5]</a></sup><sup class="reference" id="cite_ref-6"><a href="#cite_note-6">[6]</a></sup>.</p>
<h2><span class="mw-headline" id=".EC.9E.91.ED.92.88_.EA.B2.BD.ED.96.A5">작품 경향</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%8A%88%ED%85%8C%ED%8C%90_%EC%B8%A0%EB%B0%94%EC%9D%B4%ED%81%AC&amp;action=edit&amp;section=2" title="부분 편집: 작품 경향">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<div class="thumb tleft">
<div class="thumbinner" style="width:122px;"><a class="image" href="/wiki/%ED%8C%8C%EC%9D%BC:Stefan_Zweig,_Die_Welt_von_gestern_1942.jpg"><img alt="" class="thumbimage" data-file-height="420" data-file-width="246" height="205" src="//upload.wikimedia.org/wikipedia/commons/thumb/3/3e/Stefan_Zweig%2C_Die_Welt_von_gestern_1942.jpg/120px-Stefan_Zweig%2C_Die_Welt_von_gestern_1942.jpg" srcset="//upload.wikimedia.org/wikipedia/commons/thumb/3/3e/Stefan_Zweig%2C_Die_Welt_von_gestern_1942.jpg/180px-Stefan_Zweig%2C_Die_Welt_von_gestern_1942.jpg 1.5x, //upload.wikimedia.org/wikipedia/commons/thumb/3/3e/Stefan_Zweig%2C_Die_Welt_von_gestern_1942.jpg/240px-Stefan_Zweig%2C_Die_Welt_von_gestern_1942.jpg 2x" width="120"/></a>
<div class="thumbcaption">

자서전 《어제의 세계》(1942)</div>

</div>
<p>슈테판 츠바이크는 <a href="/wiki/1920%EB%85%84%EB%8C%80" title="1920년대">1920년대</a>와 <a href="/wiki/1930%EB%85%84%EB%8C%80" title="1930년대">1930년대</a> 최고 유명 작가였다. 또한 <a href="/wiki/%EC%95%84%EB%A5%B4%ED%88%AC%EC%96%B4_%EC%8A%88%EB%8B%88%EC%B8%A8%EB%9F%AC" title="아르투어 슈니츨러">아르투어 슈니츨러</a>, <a href="/wiki/%EC%A7%80%EA%B7%B8%EB%AC%B8%ED%8A%B8_%ED%94%84%EB%A1%9C%EC%9D%B4%ED%8A%B8" title="지그문트 프로이트">지그문트 프로이트</a>의 친구이기도 했다<sup class="reference" id="cite_ref-7"><a href="#cite_note-7">[7]</a></sup>. 그는 특히 <a href="/wiki/%EB%AF%B8%EA%B5%AD" title="미국">미국</a>, <a class="mw-redirect" href="/wiki/%EB%82%A8%EB%AF%B8" title="남미">남미</a> 그리고 <a href="/wiki/%EC%9C%A0%EB%9F%BD" title="유럽">유럽</a> 대륙에서 유명했다. 하지만 <a href="/wiki/%EC%98%81%EA%B5%AD" title="영국">영국</a> 출판계에서는 무시당했는데<sup class="reference" id="cite_ref-Hofmann2010_8-0"><a href="#cite_note-Hofmann2010-8">[8]</a></sup>, 이는 곧 <a class="mw-redirect" href="/wiki/%EB%AF%B8%EC%A3%BC" title="미주">미주</a>에서의 명성이 하락하는 결과로 나타나기도 하였다. 그러나 <a href="/wiki/1990%EB%85%84%EB%8C%80" title="1990년대">1990년대</a> 이후부터 <a href="/wiki/%EB%AF%B8%EA%B5%AD" title="미국">미국</a>의 몇몇 유명 출판사에서 그의 작품들을 <a href="/wiki/%EC%98%81%EC%96%B4" title="영어">영어</a>로 번역하여 출판하기 시작했다<sup class="reference" id="cite_ref-Lezard2009_9-0"><a href="#cite_note-Lezard2009-9">[9]</a></sup>.</p>
<p>그의 작품에 대한 비평은 극과 극으로 나뉘는데, 혹자는 그의 문체가 가볍고, 피상적이라면서 부정적 평가를 내리기도 하며, 혹자는 그의 <a href="/wiki/%EC%9D%B8%EB%AC%B8%EC%A3%BC%EC%9D%98" title="인문주의">휴머니즘</a>과 간결하지만 설득력 있는 문체가 <a href="/wiki/%EC%9C%A0%EB%9F%BD" title="유럽">유럽</a>의 전통에 더욱 더 매료되게 한다며 긍정적 평가를 내리기도 한다.</p>
<p>츠바이크의 유명 작품으로 중편 소설인 《체스》(1922), 《아모크》(1922), 《모르는 여인으로부터의 편지》(1922)와 소설인 《감정의 혼란》(1926), 《연민》(1939) 그리고 전기문인 《에라무스 평전》(1934), 《메리 스튜어트》(1935), 《바다의 정복자: 마젤란 이야기》(1938)가 있다. 한 때, 츠바이크는 반(反) <a href="/wiki/%EB%8F%85%EC%9D%BC" title="독일">독일</a> 감정이 확대되자 자신의 이름인 슈테판 츠바이크를 <a href="/wiki/%EC%98%81%EC%96%B4" title="영어">영어</a>로 그대로 번역한 스테픈 브랜치(<i>Stephen Branch</i>)라는 가명으로 활동하기도 하였다. 그의 작품 《<a href="/wiki/%EB%A7%88%EB%A6%AC_%EC%95%99%ED%88%AC%EC%95%84%EB%84%A4%ED%8A%B8_%EB%B2%A0%EB%A5%B4%EC%82%AC%EC%9C%A0%EC%9D%98_%EC%9E%A5%EB%AF%B8" title="마리 앙투아네트 베르사유의 장미">마리 앙투아네트 베르사유의 장미</a>》(1932)는 <a href="/wiki/1938%EB%85%84" title="1938년">1938년</a> <a href="/wiki/%EB%AF%B8%EA%B5%AD%EC%9D%98_%EC%98%81%ED%99%94" title="미국의 영화">헐리우드 영화</a>로 제작되었다.</p>
<p>츠바이크는 <a href="/wiki/%EC%9D%8C%EC%95%85%EA%B0%80" title="음악가">음악가</a> <a href="/wiki/%EB%A6%AC%ED%95%98%EB%A5%B4%ED%8A%B8_%EC%8A%88%ED%8A%B8%EB%9D%BC%EC%9A%B0%EC%8A%A4" title="리하르트 슈트라우스">리하르트 슈트라우스</a>와 함께 있는 것을 좋아했는데, 슈트라우스에게 <a href="/wiki/%EC%98%A4%ED%8E%98%EB%9D%BC" title="오페라">오페라</a> 《말없는 여자》(<i>Die schweigsame Frau</i>)의 대본을 제공하기도 했다. 슈트라우스는 <a href="/wiki/1935%EB%85%84" title="1935년">1935년</a> <a href="/wiki/6%EC%9B%94_24%EC%9D%BC" title="6월 24일">6월 24일</a> <a href="/wiki/%EB%93%9C%EB%A0%88%EC%8A%A4%EB%8D%B4" title="드레스덴">드레스덴</a>에서 자신의 첫 작품을 선보일 때, 프로그램에서 츠바이크의 이름을 삭제하라는 <a class="mw-redirect" href="/wiki/%EB%AF%BC%EC%A1%B1%EC%82%AC%ED%9A%8C%EC%A3%BC%EC%9D%98_%EB%8F%85%EC%9D%BC_%EB%85%B8%EB%8F%99%EC%9E%90%EB%8B%B9" title="민족사회주의 독일 노동자당">나치</a> 정권의 요구에 반항한 것으로 유명한데<sup class="reference" id="cite_ref-StraussZweigLetters_10-0"><a href="#cite_note-StraussZweigLetters-10">[10]</a></sup>, 그 결과 원래 오페라에 참석하기로 했던 <a href="/wiki/%EC%9A%94%EC%A0%9C%ED%94%84_%EA%B4%B4%EB%B2%A8%EC%8A%A4" title="요제프 괴벨스">요제프 괴벨스</a>가 오지 않았고 이후 공연 또한 상영 금지 처분을 받아야했다. <a href="/wiki/1937%EB%85%84" title="1937년">1937년</a> 츠바이크는 요제프 그레고르와 오페라 《대프니》(<i>Daphne</i>)의 대본을 공동 제작하여 슈트라우스에게 제공하기도 하였다.</p>
<h2><span class="mw-headline" id=".EA.B8.B0.ED.83.80">기타</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%8A%88%ED%85%8C%ED%8C%90_%EC%B8%A0%EB%B0%94%EC%9D%B4%ED%81%AC&amp;action=edit&amp;section=3" title="부분 편집: 기타">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
