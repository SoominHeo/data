<p><b>슈퍼 마리오 3D월드</b>(SUPER MARIO 3D WORLD, 약칭 <b>SM3DW</b>)는 <a href="/wiki/%EC%9D%BC%EB%B3%B8" title="일본">일본</a>에서 먼저 <a href="/wiki/2013%EB%85%84" title="2013년">2013년</a> <a href="/wiki/11%EC%9B%94_21%EC%9D%BC" title="11월 21일">11월 21일</a>에 발매한 <a href="/wiki/Wii_U" title="Wii U">Wii U</a> 게임으로 <a href="/wiki/%EB%8B%8C%ED%85%90%EB%8F%84_3DS" title="닌텐도 3DS">닌텐도 3DS</a>의 <a class="mw-redirect" href="/wiki/%EC%8A%88%ED%8D%BC_%EB%A7%88%EB%A6%AC%EC%98%A4_3D_%EB%9E%9C%EB%93%9C" title="슈퍼 마리오 3D 랜드">슈퍼 마리오 3D 랜드</a>(SUPER MARIO 3D LAND)의 후속작이다.</p>

<h2><span class="mw-headline" id=".EC.A4.84.EA.B1.B0.EB.A6.AC">줄거리</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%8A%88%ED%8D%BC_%EB%A7%88%EB%A6%AC%EC%98%A4_3D%EC%9B%94%EB%93%9C&amp;action=edit&amp;section=2" title="부분 편집: 줄거리">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p><a href="/wiki/%EB%A7%88%EB%A6%AC%EC%98%A4" title="마리오">마리오</a>, <a href="/wiki/%EB%A3%A8%EC%9D%B4%EC%A7%80" title="루이지">루이지</a>, <a href="/wiki/%ED%94%BC%EC%B9%98_%EA%B3%B5%EC%A3%BC" title="피치 공주">피치 공주</a>, <a href="/wiki/%ED%82%A4%EB%85%B8%ED%94%BC%EC%98%A4" title="키노피오">키노피오</a>(파랑색) 4명이 같이 길을 걷고 있었는데 길에 있는 투명 토관(한국 정식 명칭 미정)안에서 갑자기 아이템 등이 나오고 초록색 요정 공주 하나가 나와 다른 색깔의 요정 공주들이 <a href="/wiki/%EC%BF%A0%ED%8C%8C" title="쿠파">쿠파</a>한테 납치되었다고 <a href="/wiki/%EB%A7%88%EB%A6%AC%EC%98%A4" title="마리오">마리오</a>, <a href="/wiki/%EB%A3%A8%EC%9D%B4%EC%A7%80" title="루이지">루이지</a>, <a href="/wiki/%ED%94%BC%EC%B9%98_%EA%B3%B5%EC%A3%BC" title="피치 공주">피치 공주</a>, <a href="/wiki/%ED%82%A4%EB%85%B8%ED%94%BC%EC%98%A4" title="키노피오">키노피오</a> 4명에게 말을 하였다. 그 순간 투명 토관에서 <a href="/wiki/%EC%BF%A0%ED%8C%8C" title="쿠파">쿠파</a>가 나와 말을 하고 있던 요정 공주 마저 납치되고 <a href="/wiki/%EC%BF%A0%ED%8C%8C" title="쿠파">쿠파</a>를 쫓아 투명 토관으로 들어가 모험을 하게 된다.</p>
