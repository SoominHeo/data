<p><b>BK 헤켄</b>(BK Häcken)은 <a href="/wiki/%EC%8A%A4%EC%9B%A8%EB%8D%B4" title="스웨덴">스웨덴</a>의 <a href="/wiki/%EC%B6%95%EA%B5%AC" title="축구">축구</a> 클럽으로 <a href="/wiki/%EC%98%88%ED%85%8C%EB%B3%B4%EB%A6%AC" title="예테보리">예테보리</a>에 연고를 두고 있는 클럽이다. 하켄은 <a href="/wiki/1940%EB%85%84" title="1940년">1940년</a> <a href="/wiki/8%EC%9B%94_2%EC%9D%BC" title="8월 2일">8월 2일</a>에 창단되었으며, <a href="/wiki/2006%EB%85%84" title="2006년">2006년</a>의 시즌 때에 스웨덴 리그의 2부로 강등된 이후로 현재까지 <a href="/wiki/%EC%88%98%ED%8E%98%EB%A0%88%ED%83%84" title="수페레탄">수페레탄</a>에서 플레이하고 있다.</p>
<p>팀 최고 성적은 <a href="/wiki/%EC%95%8C%EC%8A%A4%EB%B2%A4%EC%8A%A4%EC%B9%B8" title="알스벤스칸">알스벤스칸</a> 6위가 최고 성적이며, 컵대회의 경우 <a class="new" href="/w/index.php?title=%EC%8A%A4%EC%9B%A8%EB%8D%B4_%EC%BB%B5&amp;action=edit&amp;redlink=1" title="스웨덴 컵 (없는 문서)">스웨덴 컵</a> 준우승 1회 기록이 전부이다.</p>
<p>하켄은 2007-08시즌의 <a class="mw-redirect" href="/wiki/UEFA%EC%BB%B5" title="UEFA컵">UEFA컵</a>에 UEFA 페어 플레이 수상으로 최초로 유럽 대회에 진출하였다. 2차예선부터 참가하여 스코틀랜드의 던펄라인 애슬래틱을 합계 2-1, 1승 1무로 물리치고 1회전에 올라왔으나 <a href="/wiki/%EB%9F%AC%EC%8B%9C%EC%95%84" title="러시아">러시아</a>의 <a href="/wiki/FC_%EC%8A%A4%ED%8C%8C%EB%A5%B4%ED%83%80%ED%81%AC_%EB%AA%A8%EC%8A%A4%ED%81%AC%EB%B0%94" title="FC 스파르타크 모스크바">FC 스파르타크 모스크바</a>에 1-8 2패로 패하여 탈락하였다.</p>
