<p><b>슈피겔</b>(<span style="font-size: smaller;"><a href="/wiki/%EB%8F%85%EC%9D%BC%EC%96%B4" title="독일어">독일어</a>:</span> <span lang="de" xml:lang="de">Der Spiegel</span> <small>데어 슈피겔<sup>[<a href="/wiki/%EC%9C%84%ED%82%A4%EB%B0%B1%EA%B3%BC:%EB%8F%85%EC%9D%BC%EC%96%B4%EC%9D%98_%ED%95%9C%EA%B8%80_%ED%91%9C%EA%B8%B0" title="위키백과:독일어의 한글 표기">*</a>]</sup></small>, <small><a href="/wiki/%EC%9C%84%ED%82%A4%EB%B0%B1%EA%B3%BC:IPA" title="위키백과:IPA">IPA</a>: </small><span class="IPA" title="IPA 발음">[deːɐ ˈʃpiːɡəl]</span>)은 <a class="new" href="/w/index.php?title=%EB%A3%A8%EB%8F%8C%ED%94%84_%EC%95%84%EC%9A%B0%ED%81%AC%EC%8A%88%ED%83%80%EC%9D%B8&amp;action=edit&amp;redlink=1" title="루돌프 아우크슈타인 (없는 문서)">루돌프 아우크슈타인</a>이 창립한 <a href="/wiki/%EB%8F%85%EC%9D%BC" title="독일">독일</a>의 대표적 주간 시사<a href="/wiki/%EC%9E%A1%EC%A7%80" title="잡지">잡지</a>이다. 정치논조는 진보성향이 짙다. 본사는 <a href="/wiki/%ED%95%A8%EB%B6%80%EB%A5%B4%ED%81%AC" title="함부르크">함부르크</a>에 있다. 주당 1백만 부 이상이 발행돼, <a href="/wiki/%EC%9C%A0%EB%9F%BD" title="유럽">유럽</a>과 <a href="/wiki/%EB%8F%85%EC%9D%BC" title="독일">독일</a>에서 가장 많이 읽히는 잡지들 중 하나이기도 하다.<sup class="reference" id="cite_ref-1"><a href="#cite_note-1">[1]</a></sup> 슈피겔은 독일어로 "거울"을 뜻한다.</p>
<h2><span class="mw-headline" id=".EA.B0.9C.EA.B4.80">개관</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%8A%88%ED%94%BC%EA%B2%94&amp;action=edit&amp;section=1" title="부분 편집: 개관">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>슈피겔지는 <a href="/wiki/1946%EB%85%84" title="1946년">1946년</a> 창간되었다. 초판은 <a href="/wiki/1947%EB%85%84" title="1947년">1947년</a> <a href="/wiki/1%EC%9B%94_4%EC%9D%BC" title="1월 4일">1월 4일</a> 토요일, <a href="/wiki/%ED%95%98%EB%85%B8%EB%B2%84" title="하노버">하노버</a>에서 발행되었다. 초판 발행은 <a href="/wiki/%EC%98%81%EA%B5%AD" title="영국">영국</a>의 군정(British occupational administration)의 후원 하에서 이루어진 것으로, 1946년 11월 발행된 디제 보헤(Diese Woche, "이번 주")라는 잡지를 잇는 것이었다. 영국과 의견 일치를 보지 못했던 디제 보헤는 <a class="new" href="/w/index.php?title=%EB%A3%A8%EB%8F%8C%ED%94%84_%EC%95%84%EC%9A%B0%ED%81%AC%EC%8A%88%ED%83%80%EC%9D%B8&amp;action=edit&amp;redlink=1" title="루돌프 아우크슈타인 (없는 문서)">루돌프 아우크슈타인</a> 편집장에게 넘어갔으며, 데어 슈피겔(Der Speigel)로 명칭이 바뀌었다. 루돌프 아우크슈타인은 1947년 1월 초판부터 그가 사망한 <a href="/wiki/2002%EB%85%84" title="2002년">2002년</a> <a href="/wiki/11%EC%9B%94_7%EC%9D%BC" title="11월 7일">11월 7일</a>까지 편집장의 자리를 지켰다.</p>
<p><a href="/wiki/1950%EB%85%84" title="1950년">1950년</a> 이후, 슈피겔은 아우크슈타인과 <a class="new" href="/w/index.php?title=%EC%A1%B4_%EC%95%BC&amp;action=edit&amp;redlink=1" title="존 야 (없는 문서)">존 야</a>(John Jahr)가 소유하고 있었는데, 야의 지분은 <a href="/wiki/1965%EB%85%84" title="1965년">1965년</a> <a class="new" href="/w/index.php?title=%EB%A6%AC%ED%95%98%EB%A5%B4%ED%8A%B8_%EA%B7%B8%EB%A3%A8%EB%84%88&amp;action=edit&amp;redlink=1" title="리하르트 그루너 (없는 문서)">리하르트 그루너</a>(Richard Gruner)의 합명회사인 출판업체 그루너 + 야로 넘어갔다. <a href="/wiki/1969%EB%85%84" title="1969년">1969년</a> 아우크슈타인은 그루너 + 야의 지분을 4,200만 <a href="/wiki/%EB%8F%85%EC%9D%BC_%EB%A7%88%EB%A5%B4%ED%81%AC" title="독일 마르크">마르크</a>에 모두 인수하였다. 그러나 <a href="/wiki/1971%EB%85%84" title="1971년">1971년</a> 그루너 + 야는 다시 슈피겔의 지분 25%를 사들였다. <a href="/wiki/1974%EB%85%84" title="1974년">1974년</a> 아우크슈타인은 슈피겔의 형태를 <a class="new" href="/w/index.php?title=%EC%82%AC%EC%9B%90%EC%A3%BC%EC%A3%BC%EC%A0%9C&amp;action=edit&amp;redlink=1" title="사원주주제 (없는 문서)">사원주주제</a>로 바꾸어, 슈피겔에 3년 이상 근무한 직원에게 회사 경영에 참여할 수 있게 하였다.</p>
<p>슈피겔은 <a href="/wiki/%EB%AF%B8%EA%B5%AD" title="미국">미국</a>의 <a href="/wiki/%ED%83%80%EC%9E%84_(%EC%9E%A1%EC%A7%80)" title="타임 (잡지)">타임</a>지나 <a href="/wiki/%EB%89%B4%EC%8A%A4%EC%9C%84%ED%81%AC" title="뉴스위크">뉴스위크</a>와 비슷한 편집 스타일을 취하고 있다. 기사의 깊이나 내용 측면에서는 영국의 <a href="/wiki/%EC%9D%B4%EC%BD%94%EB%85%B8%EB%AF%B8%EC%8A%A4%ED%8A%B8" title="이코노미스트">이코노미스트</a>에 비할 수 있다. 독일에서는 그 특유의 문체와 보통 호당 200쪽을 넘는 폭넓은 보도로 정평이 나 있다. 특히 슈피겔은 2:1의 본문 대비 광고의 비율을 유지하고 있다.</p>
