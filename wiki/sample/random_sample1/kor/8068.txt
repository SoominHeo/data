<p><b>슐레스비히</b>(<span style="font-size: smaller;"><a href="/wiki/%EB%8F%85%EC%9D%BC%EC%96%B4" title="독일어">독일어</a>:</span> <span lang="de" xml:lang="de">Schleswig</span>)는 <a href="/wiki/%EB%8F%85%EC%9D%BC" title="독일">독일</a> 북부 <a href="/wiki/%EC%8A%90%EB%A0%88%EC%8A%A4%EB%B9%84%ED%9E%88%ED%99%80%EC%8A%88%ED%83%80%EC%9D%B8_%EC%A3%BC" title="슐레스비히홀슈타인 주">슐레스비히홀슈타인 주</a>에 있는 도시이다. 인구 24,015(2010)<sup class="reference" id="cite_ref-1"><a href="#cite_note-1">[1]</a></sup>.</p>
<p>슐레스비히홀슈타인 주 북부, <a href="/wiki/%ED%94%8C%EB%A0%8C%EC%8A%A4%EB%B6%80%EB%A5%B4%ED%81%AC" title="플렌스부르크">플렌스부르크</a>와 <a href="/wiki/%ED%82%AC_(%EC%8A%90%EB%A0%88%EC%8A%A4%EB%B9%84%ED%9E%88%ED%99%80%EC%8A%88%ED%83%80%EC%9D%B8_%EC%A3%BC)" title="킬 (슐레스비히홀슈타인 주)">킬</a> 사이, <a href="/wiki/%EB%B0%9C%ED%8A%B8_%ED%95%B4" title="발트 해">발트 해</a>에서 깊숙히 들어간 <a class="new" href="/w/index.php?title=%EC%8A%90%EB%9D%BC%EC%9D%B4_%EB%A7%8C&amp;action=edit&amp;redlink=1" title="슐라이 만 (없는 문서)">슐라이 만</a>을 끼고 있다. 북부 독일에서 오래 된 도시 중 하나로, <a href="/wiki/%EB%B0%94%EC%9D%B4%ED%82%B9" title="바이킹">바이킹</a>의 정착지였다가 <a href="/wiki/9%EC%84%B8%EA%B8%B0" title="9세기">9세기</a>부터 알려지기 시작하여 발트 해 일대의 주요 도시가 되었으나, 후에 <a href="/wiki/%EB%A4%BC%EB%B2%A0%ED%81%AC" title="뤼베크">뤼베크</a>에 밀려 쇠퇴했다.</p>
<p><a href="/wiki/10%EC%84%B8%EA%B8%B0" title="10세기">10세기</a>에 <a href="/wiki/%EA%B0%80%ED%86%A8%EB%A6%AD" title="가톨릭">가톨릭</a> 주교관구가 되었고, <a href="/wiki/16%EC%84%B8%EA%B8%B0" title="16세기">16세기</a> 종교개혁으로 <a href="/wiki/%EA%B0%9C%EC%8B%A0%EA%B5%90" title="개신교">개신교</a> 지역이 된 후 <a href="/wiki/%ED%99%80%EC%8A%88%ED%83%80%EC%9D%B8" title="홀슈타인">홀슈타인</a>과 <a class="new" href="/w/index.php?title=%EC%8A%90%EB%A0%88%EC%8A%A4%EB%B9%84%ED%9E%88%EA%B3%A0%ED%8A%B8%EB%A1%9C%ED%94%84_%EA%B0%80%EB%AC%B8&amp;action=edit&amp;redlink=1" title="슐레스비히고트로프 가문 (없는 문서)">슐레스비히고트로프 가문</a>의 중심지가 되었다. <a href="/wiki/1866%EB%85%84" title="1866년">1866년</a> <a href="/wiki/%ED%94%84%EB%A1%9C%EC%9D%B4%EC%84%BC" title="프로이센">프로이센</a>의 새로 형성된 슐레스비히홀슈타인 주의 주도가 되었고, 당시 주의 영역은 현재 <a href="/wiki/%EB%8D%B4%EB%A7%88%ED%81%AC" title="덴마크">덴마크</a>에 속하는 <a class="mw-redirect" href="/wiki/%EB%B6%81%EC%8A%90%EB%A0%88%EC%8A%A4%EB%B9%84%ED%9E%88" title="북슐레스비히">북슐레스비히</a> 지역을 포함하였다. <a href="/wiki/1917%EB%85%84" title="1917년">1917년</a> 주도가 킬로 옮겨졌고, <a href="/wiki/1974%EB%85%84" title="1974년">1974년</a> 이후 <a class="new" href="/w/index.php?title=%EC%8A%90%EB%A0%88%EC%8A%A4%EB%B9%84%ED%9E%88%ED%94%8C%EB%A0%8C%EC%8A%A4%EB%B6%80%EB%A5%B4%ED%81%AC_%EA%B5%B0&amp;action=edit&amp;redlink=1" title="슐레스비히플렌스부르크 군 (없는 문서)">슐레스비히플렌스부르크 군</a>의 군청 소재지가 되고 있다. 알려진 옛 건물로는 <a class="new" href="/w/index.php?title=%EA%B3%A0%ED%8A%B8%EB%A1%9C%ED%94%84_%EC%84%B1&amp;action=edit&amp;redlink=1" title="고트로프 성 (없는 문서)">고트로프 성</a>과 덴마크 <a class="new" href="/w/index.php?title=%ED%94%84%EB%A0%88%EB%8D%B0%EB%A6%AC%ED%81%AC_1%EC%84%B8_(%EB%8D%B4%EB%A7%88%ED%81%AC)&amp;action=edit&amp;redlink=1" title="프레데리크 1세 (덴마크) (없는 문서)">프레데리크 1세</a>의 무덤이 있는 대성당이 있다.</p>
