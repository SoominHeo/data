<p><b>스가야 리사코</b>(菅谷梨沙子, <a href="/wiki/1994%EB%85%84" title="1994년">1994년</a> <a href="/wiki/4%EC%9B%94_4%EC%9D%BC" title="4월 4일">4월 4일</a> - ) 는 <a href="/wiki/%EC%9D%BC%EB%B3%B8" title="일본">일본</a>의 <a href="/wiki/%EC%97%AC%EC%84%B1" title="여성">여성</a> <a class="mw-redirect" href="/wiki/%EC%95%84%EC%9D%B4%EB%8F%8C_%EA%B0%80%EC%88%98" title="아이돌 가수">아이돌</a> <a href="/wiki/%EA%B0%80%EC%88%98" title="가수">가수</a>이다. <a href="/wiki/%EA%B0%80%EB%82%98%EA%B0%80%EC%99%80_%ED%98%84" title="가나가와 현">가나가와 현</a> 출신. 연예사무소 <a href="/wiki/%EC%97%85%ED%94%84%EB%9F%B0%ED%8A%B8_%ED%94%84%EB%A1%9C%EB%AA%A8%EC%85%98" title="업프런트 프로모션">업프런트 프로모션</a> 소속. 현재의 키는 161cm. 애칭은 <b>리샤코</b>, <b>리이짱</b>, <b>리사코</b>, <b>리사티</b>, <b>시이코</b>, <b>리샤마루</b> 등.</p>
<p><a href="/wiki/%EB%B2%A0%EB%A6%AC%EC%A6%88%EC%BD%94%EB%B3%B4" title="베리즈코보">베리즈코보</a>의 최연소 멤버이며, 악곡에서는 메인 보컬과 센터를 맡는 경우가 많다.</p>

<h2><span class="mw-headline" id=".EC.95.BD.EB.A0.A5">약력</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%8A%A4%EA%B0%80%EC%95%BC_%EB%A6%AC%EC%82%AC%EC%BD%94&amp;action=edit&amp;section=1" title="부분 편집: 약력">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>베리즈코보의 메이저 데뷔와 함께 그 성원으로서 <a href="/wiki/2004%EB%85%84" title="2004년">2004년</a>에 데뷔. <a href="/wiki/2006%EB%85%84" title="2006년">2006년</a> <a href="/wiki/10%EC%9B%94" title="10월">10월</a>에 첫 번째 솔로 사진집을 발표. 이 사진집은 수영복 컷도 있으며 <a href="/wiki/%ED%97%AC%EB%A1%9C!_%ED%94%84%EB%A1%9C%EC%A0%9D%ED%8A%B8" title="헬로! 프로젝트">헬로! 프로젝트</a>에서는 사상 최연소 발매이다. <a href="/wiki/2007%EB%85%84" title="2007년">2007년</a> <a href="/wiki/7%EC%9B%94" title="7월">7월</a>에는 새로운 사진집 <a class="new" href="/w/index.php?title=Pure%2B&amp;action=edit&amp;redlink=1" title="Pure+ (없는 문서)">pure+</a>를 발매했었다.</p>
<h2><span class="mw-headline" id=".EC.9D.B8.EB.AC.BC">인물</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%8A%A4%EA%B0%80%EC%95%BC_%EB%A6%AC%EC%82%AC%EC%BD%94&amp;action=edit&amp;section=2" title="부분 편집: 인물">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<ul>
<li><a href="/wiki/2005%EB%85%84" title="2005년">2005년</a> 가을 콘서트 투어《스위치 ON!》에서 의상에 달린 번호는《48》(스=4, 야=8). <a href="/wiki/2006%EB%85%84" title="2006년">2006년</a> 신춘에 개최된《Hello! Project 2006 Winter ~원더풀 하츠~》와 같은해 봄 콘서트 투어《쑥쑥 챔피언!》에서도 같은 번호를 달았다.</li>
<li>친구 <a href="/wiki/%EB%B2%A0%EB%A6%AC%EC%A6%88%EC%BD%94%EB%B3%B4" title="베리즈코보">베리즈코보</a> <a href="/wiki/%EC%8B%9C%EB%AF%B8%EC%A6%88_%EC%82%AC%ED%82%A4" title="시미즈 사키">시미즈 사키</a>(캡틴).<a href="/wiki/%EC%B8%A0%EA%B5%AC%EB%82%98%EA%B0%80_%EB%AA%A8%EB%AA%A8%EC%BD%94" title="츠구나가 모모코">츠구나가 모모코</a>.<a href="/wiki/%EB%82%98%EC%B8%A0%EC%95%BC%ED%82%A4_%EB%AF%B8%EC%95%BC%EB%B9%84" title="나츠야키 미야비">나츠야키 미야비</a>.<a href="/wiki/%E2%84%83-ute" title="℃-ute">℃-ute</a> <a href="/wiki/%EC%8A%A4%EC%A6%88%ED%82%A4_%EC%95%84%EC%9D%B4%EB%A6%AC" title="스즈키 아이리">스즈키 아이리</a></li>
