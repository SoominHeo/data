<p><b>스리아게하라 전투</b>(摺上原の戦い|すりあげはらのたたかい}})는 <a href="/wiki/%EC%9D%BC%EB%B3%B8" title="일본">일본</a> <a class="mw-redirect" href="/wiki/%EC%95%84%EC%A6%88%EC%B9%98%EB%AA%A8%EB%AA%A8%EC%95%BC%EB%A7%88_%EC%8B%9C%EB%8C%80" title="아즈치모모야마 시대">아즈치모모야마 시대</a> 때 <a href="/wiki/%EB%8F%84%ED%98%B8%EC%BF%A0_%EC%A7%80%EB%B0%A9" title="도호쿠 지방">도호쿠 지방</a>의 스리아게하라(摺上原)에서 <a href="/wiki/%EB%8B%A4%ED%85%8C_%EB%A7%88%EC%82%AC%EB%AC%B4%EB%84%A4" title="다테 마사무네">다테 마사무네</a>와 <a class="new" href="/w/index.php?title=%EC%95%84%EC%8B%9C%EB%82%98_%EC%9A%94%EC%8B%9C%ED%9E%88%EB%A1%9C&amp;action=edit&amp;redlink=1" title="아시나 요시히로 (없는 문서)">아시나 요시히로</a>(蘆名義広) 사이에 벌어진 전투이다.</p>

<h2><span class="mw-headline" id=".EB.B0.B0.EA.B2.BD">배경</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%8A%A4%EB%A6%AC%EC%95%84%EA%B2%8C%ED%95%98%EB%9D%BC_%EC%A0%84%ED%88%AC&amp;action=edit&amp;section=1" title="부분 편집: 배경">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>다테 마사무네는 <a href="/wiki/%ED%9E%88%ED%86%A0%ED%86%A0%EB%A6%AC%EB%B0%94%EC%8B%9C_%EC%A0%84%ED%88%AC" title="히토토리바시 전투">히토토리바시 전투</a> 이후, <a class="new" href="/w/index.php?title=%EC%95%84%EC%8B%9C%EB%82%98_%EA%B0%80%EB%AC%B8&amp;action=edit&amp;redlink=1" title="아시나 가문 (없는 문서)">아시나 가문</a>(蘆名氏)과 <a class="new" href="/w/index.php?title=%EC%82%AC%ED%83%80%EC%BC%80_%EA%B0%80%EB%AC%B8&amp;action=edit&amp;redlink=1" title="사타케 가문 (없는 문서)">사타케 가문</a>(佐竹氏)과의 대결 자세를 강화하면서, <a class="new" href="/w/index.php?title=%EB%82%98%EC%B9%B4%EB%8F%84%EC%98%A4%EB%A6%AC&amp;action=edit&amp;redlink=1" title="나카도오리 (없는 문서)">나카도오리</a>(中通り) 지방으로 세력을 확대해 나갔기 때문에, 이에 맞서는 아시나·사타케 가문의 위기감이 고조되고 있었다. 이윽고 다테 마사무네는 아시나 가문의 본거지 <a href="/wiki/%EC%95%84%EC%9D%B4%EC%A6%88" title="아이즈">아이즈</a>(会津)를 빼앗기 위하여 행동을 개시하였다. 단, 이는 <a href="/wiki/%EB%8F%84%EC%9A%94%ED%86%A0%EB%AF%B8_%ED%9E%88%EB%8D%B0%EC%9A%94%EC%8B%9C" title="도요토미 히데요시">도요토미 히데요시</a>의 <a class="new" href="/w/index.php?title=%EC%B4%9D%EB%AC%B4%EC%82%AC%EB%A0%B9&amp;action=edit&amp;redlink=1" title="총무사령 (없는 문서)">총무사령</a>(惣無事令)<sup class="reference" id="cite_ref-1"><a href="#cite_note-1">[1]</a></sup>을 무시하는 행동이었다.</p>
<p>마사무네는 <a class="new" href="/w/index.php?title=%EC%9D%B4%EB%82%98%EC%99%80%EC%8B%9C%EB%A1%9C_%EB%AA%A8%EB%A6%AC%EC%BF%A0%EB%8B%88&amp;action=edit&amp;redlink=1" title="이나와시로 모리쿠니 (없는 문서)">이나와시로 모리쿠니</a>(猪苗代盛国)의 내통으로 <a class="new" href="/w/index.php?title=%EB%AA%A8%ED%86%A0%EB%AF%B8%EC%95%BC_%EC%84%B1&amp;action=edit&amp;redlink=1" title="모토미야 성 (없는 문서)">모토미야 성</a>(本宮城)에서 아이즈로 향하는 진군로를 확보하고 <a class="new" href="/w/index.php?title=%EC%9D%B4%EB%82%98%EC%99%80%EC%8B%9C%EB%A1%9C_%EC%84%B1&amp;action=edit&amp;redlink=1" title="이나와시로 성 (없는 문서)">이나와시로 성</a>(猪苗代城)을 거점으로 하였다. 한편 아시나 요시히로는 <a class="new" href="/w/index.php?title=%EC%8A%A4%EC%B9%B4%EA%B0%80%EC%99%80_%EC%84%B1&amp;action=edit&amp;redlink=1" title="스카가와 성 (없는 문서)">스카가와 성</a>(須賀川城)에서 이나와시로 호 남안으로 진군하여, <a class="new" href="/w/index.php?title=%EA%B5%AC%EB%A1%9C%EA%B0%80%EC%99%80_%EC%84%B1&amp;action=edit&amp;redlink=1" title="구로가와 성 (없는 문서)">구로가와 성</a>(黒川城)에 병사를 집결시켰다. 다카모리 산에 포진한 아시나 군은 민가에 불을 질러 다테 군을 도발하였다. 다테 군은 21,000명, 아시나 군은 18,000명으로 병력상으로는 거의 호각인 상태였다.</p>
<p>그러나 아시나 군에는 <a href="/wiki/1580%EB%85%84" title="1580년">1580년</a>의 <a href="/wiki/%EC%95%84%EC%8B%9C%EB%82%98_%EB%AA%A8%EB%A6%AC%EC%9A%B0%EC%A7%80" title="아시나 모리우지">아시나 모리우지</a>의 사망 이후 주군 가문에 불만을 품은 자<sup class="reference" id="cite_ref-2"><a href="#cite_note-2">[2]</a></sup>, <a class="mw-redirect" href="/wiki/%EB%8B%A4%ED%85%8C_%EA%B0%80%EB%AC%B8" title="다테 가문">다테 가문</a>에 내통한 자, 사타케 가문에 의해 세워진 당주 아시나 요시히로<sup class="reference" id="cite_ref-3"><a href="#cite_note-3">[3]</a></sup>에 대해 불만을 품은 자 등이 있어, 여러가지 의혹으로 단결력이 부족한 상태였다.</p>
<h2><span class="mw-headline" id=".EA.B2.BD.EA.B3.BC">경과</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%8A%A4%EB%A6%AC%EC%95%84%EA%B2%8C%ED%95%98%EB%9D%BC_%EC%A0%84%ED%88%AC&amp;action=edit&amp;section=2" title="부분 편집: 경과">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>전투는 이나와시로 호수의 남안 반다이 산(磐梯山) 기슭의 평원인 스리아게하라에서 벌어졌다. 개전 당시에는 선봉 <a class="new" href="/w/index.php?title=%EB%8F%84%EB%AF%B8%ED%83%80_%EB%8B%A4%EC%B9%B4%EC%9E%90%EB%84%A4&amp;action=edit&amp;redlink=1" title="도미타 다카자네 (없는 문서)">도미타 다카자네</a>(富田隆実)의 활약과 서쪽에서 열풍이 불어와 순풍을 받은 아시나 군이 유리한 상황이었다. 그러나 얼마 안 있어 풍향이 동풍으로 바뀌자 순풍을 받은 다테 군이 역으로 압도적인 우세를 점하여, 아시나 군을 대파하였다. 패주한 아시나 군은 후방의 강이 있어 구로가와 성으로 귀환하는 것을 방해받아 더욱 큰 피해를 입었다. 아시나 요시히로는 본가인 사타케 가문을 의지하여 <a href="/wiki/%ED%9E%88%ED%83%80%EC%B9%98_%EA%B5%AD" title="히타치 국">히타치 국</a>으로 도주하였다.</p>
<h2><span class="mw-headline" id=".EC.98.81.ED.96.A5">영향</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%8A%A4%EB%A6%AC%EC%95%84%EA%B2%8C%ED%95%98%EB%9D%BC_%EC%A0%84%ED%88%AC&amp;action=edit&amp;section=3" title="부분 편집: 영향">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>아시나 가문은 <a class="new" href="/w/index.php?title=%EA%B0%80%EB%82%98%EA%B0%80%EB%AF%B8_%EB%AA%A8%EB%A6%AC%ED%95%98%EB%A3%A8&amp;action=edit&amp;redlink=1" title="가나가미 모리하루 (없는 문서)">가나가미 모리하루</a>(金上盛備), <a class="new" href="/w/index.php?title=%EC%82%AC%EC%84%B8_%EB%8B%A4%EB%84%A4%EC%93%B0%EB%82%98&amp;action=edit&amp;redlink=1" title="사세 다네쓰나 (없는 문서)">사세 다네쓰나</a>(佐瀬種常)등 대부분의 유력 가신을 잃는 막대한 타격을 입었다. 당주 요시히로는 패전에 따른 가신의 배반과 오슈 총무사령(奥州惣無事令)에 크게 신경을 써서 결전을 단행하지 못하고, 본가인 히타치의 <a class="new" href="/w/index.php?title=%EC%82%AC%ED%83%80%EC%BC%80_%EA%B0%80%EB%AC%B8&amp;action=edit&amp;redlink=1" title="사타케 가문 (없는 문서)">사타케 가문</a>에게 몸을 의탁하였다.</p>
<p>다테 마사무네는 남 오슈의 패자가 되어 거성을 아이즈 구로가와 성으로 옮기고, 더욱이 시라카와 유키 씨(白川結城氏)·이시카와 씨(石川氏) 등을 복속시켰다. 그러나 다음해 <a href="/wiki/%EB%8F%84%EC%9A%94%ED%86%A0%EB%AF%B8_%ED%9E%88%EB%8D%B0%EC%9A%94%EC%8B%9C" title="도요토미 히데요시">도요토미 히데요시</a>의 <a class="new" href="/w/index.php?title=%EC%98%A4%EC%8A%88_%EC%8B%9C%EC%98%A4%ED%82%A4&amp;action=edit&amp;redlink=1" title="오슈 시오키 (없는 문서)">오슈 시오키</a>(奥州仕置)에 의하여, 아이즈 공략은 히데요시가 발령한 총무사령을 위반한 것이기 때문에 마사무네는 아이즈 등지를 몰수당했다. 이후 아이즈는 <a href="/wiki/%EA%B0%80%EB%AA%A8_%EC%9A%B0%EC%A7%80%EC%82%AC%ED%86%A0" title="가모 우지사토">가모 우지사토</a>가 영지가 되었다.</p>
<h2><span class="mw-headline" id=".EA.B0.81.EC.A3.BC">각주</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%8A%A4%EB%A6%AC%EC%95%84%EA%B2%8C%ED%95%98%EB%9D%BC_%EC%A0%84%ED%88%AC&amp;action=edit&amp;section=4" title="부분 편집: 각주">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<ol class="references">
<li id="cite_note-1"><span class="mw-cite-backlink"><a href="#cite_ref-1">↑</a></span> <span class="reference-text"><a href="/wiki/1585%EB%85%84" title="1585년">1585년</a> 도요토미 히데요시가 발령한 <a href="/wiki/%EB%8B%A4%EC%9D%B4%EB%AC%98" title="다이묘">다이묘</a> 간의 사적인 전투를 금하는 법령.</span></li>
<li id="cite_note-2"><span class="mw-cite-backlink"><a href="#cite_ref-2">↑</a></span> <span class="reference-text">아시나 모리우지의 아들 아시나 모리오키가 젊은 나이에 요절하여, 모리우지는 인질로 잡고 있던 <a class="new" href="/w/index.php?title=%EB%8B%88%EC%B9%B4%EC%9D%B4%EB%8F%84_%EB%AA%A8%EB%A6%AC%EC%9A%94%EC%8B%9C&amp;action=edit&amp;redlink=1" title="니카이도 모리요시 (없는 문서)">니카이도 모리요시</a>(二階堂盛義)의 아들 <a class="new" href="/w/index.php?title=%EB%8B%88%EC%B9%B4%EC%9D%B4%EB%8F%84_%EB%AA%A8%EB%A6%AC%ED%83%80%EC%B9%B4&amp;action=edit&amp;redlink=1" title="니카이도 모리타카 (없는 문서)">니카이도 모리타카</a>(二階堂盛隆)를 양자로 들여 아시나 가문의 가독을 물려주었고, 모리타카 역시 후사가 요절하여 <a href="/wiki/%EC%82%AC%ED%83%80%EC%BC%80_%EC%9A%94%EC%8B%9C%EC%8B%9C%EA%B2%8C" title="사타케 요시시게">사타케 요시시게</a>의 요시히로를 데릴사위로 들여 가독을 잇게 하였기 때문에, 아시나 가문의 직계가 아닌 주변 다이묘 가문의 양자들이 연이어 아시나 가문의 당주가 되었다. 이는 당연히 모리우지의 대부터 아시나 가문을 섬기던 가신들의 입장에서는 달가운 일이 아니었다.</span></li>
<li id="cite_note-3"><span class="mw-cite-backlink"><a href="#cite_ref-3">↑</a></span> <span class="reference-text">요시히로는 사타케 요시시게의 아들로 다테 마사무네와는 사촌형제 간에 해당하며, 아시나 가문의 당주 모리타카의 딸과 결혼하여 아시나 가문의 가독을 상속하였다.</span></li>
</ol>
