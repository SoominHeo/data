<div class="mw-content-ltr" dir="ltr" id="mw-content-text" lang="ko"><script>function mfTempOpenSection(id){var block=document.getElementById("mf-section-"+id);block.className+=" open-block";block.previousSibling.className+=" open-block";}</script><div class="dablink"><a href="/wiki/%EC%9C%84%ED%82%A4%EB%B0%B1%EA%B3%BC:%EB%8F%99%EC%9D%8C%EC%9D%B4%EC%9D%98%EC%96%B4_%EB%AC%B8%EC%84%9C" title="위키백과:동음이의어 문서"><img alt="" data-file-height="200" data-file-width="260" height="18" src="//upload.wikimedia.org/wikipedia/commons/thumb/4/4a/Disambig_grey.svg/23px-Disambig_grey.svg.png" srcset="//upload.wikimedia.org/wikipedia/commons/thumb/4/4a/Disambig_grey.svg/35px-Disambig_grey.svg.png 1.5x, //upload.wikimedia.org/wikipedia/commons/thumb/4/4a/Disambig_grey.svg/46px-Disambig_grey.svg.png 2x" width="23"/></a> 애니메이션 시리즈에 대해서는 <a href="/wiki/%EA%B0%9C%EA%B5%AC%EC%9F%81%EC%9D%B4_%EC%8A%A4%EB%A8%B8%ED%94%84_(%EC%95%A0%EB%8B%88%EB%A9%94%EC%9D%B4%EC%85%98)" title="개구쟁이 스머프 (애니메이션)">개구쟁이 스머프 (애니메이션)</a> 문서를, 영화에 대해서는 <a href="/wiki/%EA%B0%9C%EA%B5%AC%EC%9F%81%EC%9D%B4_%EC%8A%A4%EB%A8%B8%ED%94%84_(%EC%98%81%ED%99%94)" title="개구쟁이 스머프 (영화)">개구쟁이 스머프 (영화)</a> 문서를 참조하십시오.</div>
<p><b>스머프</b>(<span style="font-size: smaller;"><a href="/wiki/%ED%94%84%EB%9E%91%EC%8A%A4%EC%96%B4" title="프랑스어">프랑스어</a>:</span> <span lang="fr" xml:lang="fr">Les Schtroumpfs</span>, <span lang="en" xml:lang="en">The Smurfs</span>)는 <a href="/wiki/%EB%B2%A8%EA%B8%B0%EC%97%90" title="벨기에">벨기에</a>의 작가인 <a href="/wiki/%ED%8E%98%EC%9A%94" title="페요">페요</a>(Peyo, <small>본명:</small>Pierre Culliford <small>피에르 컬리포드</small>)가 만들어낸 <a href="/wiki/%EB%A7%8C%ED%99%94" title="만화">만화</a> 캐릭터들을 지칭한다. 스머프들은 <a href="/wiki/%EC%9C%A0%EB%9F%BD" title="유럽">유럽</a>의 어느 숲에 살고 있으며 <a href="/wiki/%ED%95%98%EB%8A%98%EC%83%89" title="하늘색">하늘색</a>의 몸 색깔에 하얀 모자와 바지를 입는, 의인화된 작은 캐릭터들로 묘사된다. 원작을 기초로 해서 <a href="/wiki/%EB%AF%B8%EA%B5%AD" title="미국">미국</a>의 <a href="/wiki/%ED%95%B4%EB%82%98_%EB%B0%94%EB%B2%A0%EB%9D%BC" title="해나 바베라">해나 바베라</a>에서 <a href="/wiki/1981%EB%85%84" title="1981년">1981년</a>에 제작한 <a href="/wiki/%ED%85%94%EB%A0%88%EB%B9%84%EC%A0%84" title="텔레비전">텔레비전</a>용 <a href="/wiki/%EC%95%A0%EB%8B%88%EB%A9%94%EC%9D%B4%EC%85%98" title="애니메이션">애니메이션</a> 시리즈로도 유명하다.</p>
<p><a href="/wiki/%EB%8C%80%ED%95%9C%EB%AF%BC%EA%B5%AD" title="대한민국">대한민국</a>에서는 <a class="mw-redirect" href="/wiki/KBS" title="KBS">KBS</a>에서 '개구쟁이 스머프'라는 이름으로 방영한 바 있다. 당시 스머프 만화가 <a href="/wiki/%EA%B3%B5%EC%82%B0%EC%A3%BC%EC%9D%98" title="공산주의">공산주의</a>적인 이상향을 그린다는 이유로 수입이 불허되었으나, <a href="/wiki/%EB%AF%B8%EA%B5%AD" title="미국">미국</a>제작사의 요구로 방영되었다고 한다.<sup class="reference" id="cite_ref-1"><a href="#cite_note-1">[1]</a></sup> <a href="/wiki/2009%EB%85%84" title="2009년">2009년</a>에 애니메이션 채널인 <a class="mw-redirect" href="/wiki/%EB%8B%88%EC%BC%88%EB%A1%9C%EB%94%94%EC%96%B8_%EC%BD%94%EB%A6%AC%EC%95%84" title="니켈로디언 코리아">닉</a>에서 다시 방영했으며 일부 언어 표현과 주인공의 이름이 바뀌었다. 해나 바베라가 워너브라더스에 인수된 이후 벨기에의 배급사에서 소유권을 가지면서 녹음을 다시 했다. <a href="/wiki/2015%EB%85%84" title="2015년">2015년</a>에는 <a href="/wiki/%EB%94%94%EC%A6%88%EB%8B%88_%EC%B1%84%EB%84%90_(%EB%8C%80%ED%95%9C%EB%AF%BC%EA%B5%AD)" title="디즈니 채널 (대한민국)">디즈니 채널</a>에서 다시 방영하였는데 6년전 기존 재녹음된 방식으로 그대로 방영되었다.</p>

<h2><span class="mw-headline" id=".EC.9C.A0.EB.9E.98">유래</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%8A%A4%EB%A8%B8%ED%94%84&amp;action=edit&amp;section=1" title="부분 편집: 유래">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>스머프라는 이름은 <a href="/wiki/%ED%94%84%EB%9E%91%EC%8A%A4%EC%96%B4" title="프랑스어">프랑스어</a> "Schtroumpfs"에서 영어로 번역("The Smurfs")된 번역어이며, 원래의 이름은 페요에 따르면, 벨기에의 만화가인 <a class="new" href="/w/index.php?title=%EC%95%99%EB%93%9C%EB%A0%88_%ED%94%84%EB%9E%91%EC%BA%A5&amp;action=edit&amp;redlink=1" title="앙드레 프랑캥 (없는 문서)">앙드레 프랑캥</a>과 점심 식사를 할 때, '소금'이라는 말이 갑자기 생각나지 않아서 "스머프 좀 건네줘"(passe-moi le schtroumpf, <span lang="en" xml:lang="en">pass me the smurf</span>)라고 했던 그의 말로부터 비롯된 것이라고 한다.</p>
<h2><span class="mw-headline" id=".EC.9D.8C.EB.AA.A8.EB.A1.A0">음모론</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%8A%A4%EB%A8%B8%ED%94%84&amp;action=edit&amp;section=2" title="부분 편집: 음모론">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>스머프가 <a href="/wiki/%EA%B3%B5%EC%82%B0%EC%A3%BC%EC%9D%98" title="공산주의">공산주의</a>적인 이상향 사회를 그리고 있다는 음모론은 <a href="/wiki/1992%EB%85%84" title="1992년">1992년</a> <a href="/wiki/%EC%9C%A0%EC%A6%88%EB%84%B7" title="유즈넷">유즈넷</a> soc.politics에서 처음 제기된 것으로 추정된다. 음모론에서는 다음과 같은 주장을 한다.</p>
<ul>
<li>스머프(Smurf)는 사실 S.M.U.R.F., 즉 "Socialist Men Under Red Father"(붉은 아버지 아래의 <a class="mw-redirect" href="/wiki/%EC%82%AC%ED%9A%8C%EC%A3%BC%EC%9D%98%EC%9E%90" title="사회주의자">사회주의자</a>들) 혹은 "Soviet Men Under Red Father"(붉은 아버지 아래의 소련인들)의 약자다.</li>
<li><a class="new" href="/w/index.php?title=%ED%8C%8C%ED%8C%8C_%EC%8A%A4%EB%A8%B8%ED%94%84&amp;action=edit&amp;redlink=1" title="파파 스머프 (없는 문서)">파파 스머프</a>는 <a href="/wiki/%EC%B9%B4%EB%A5%BC_%EB%A7%88%EB%A5%B4%ED%81%AC%EC%8A%A4" title="카를 마르크스">카를 마르크스</a>와 <a class="new" href="/w/index.php?title=%EB%98%98%EB%98%98%EC%9D%B4_%EC%8A%A4%EB%A8%B8%ED%94%84&amp;action=edit&amp;redlink=1" title="똘똘이 스머프 (없는 문서)">똘똘이 스머프</a>는 <a class="mw-redirect" href="/wiki/%EB%A0%88%EC%98%A8_%ED%8A%B8%EB%A1%9C%EC%B8%A0%ED%82%A4" title="레온 트로츠키">레온 트로츠키</a>와 닮아있고 이는 그 두 인물을 상징한다.</li>
<li>스머프의 캐릭터들은 모두 공통적인 <a href="/wiki/%EB%85%B8%EB%8F%99%EC%9E%90" title="노동자">노동자</a> 복장을 입고 있다.</li>
<li>스머프에 등장하는 <a href="/wiki/%EC%9E%AC%EC%82%B0" title="재산">재산</a>은 모두 공동의 소유이다.</li>
<li>모든 등장인물 뒤에 붙는 "-스머프"란 호칭은 <a class="mw-redirect" href="/wiki/%EB%8F%99%EB%AC%B4" title="동무">동무</a>(comrade)를 연상시킨다.</li>
<li>악역인 가가멜은 <a href="/wiki/%EC%9E%90%EB%B3%B8%EC%A3%BC%EC%9D%98" title="자본주의">자본주의</a>를 상징한다. 그래서 최초 가가멜의 설정을, 가가멜의 직업은 연금술사이며 가가멜이 스머프를 잡아다 황금으로 만들려 한다는 것이지만 나중에 스머프를 잡아서 스프를 끓여먹는 것으로 바꿨다.</li>
<li>스머페트를 가가멜이 만든 후 파파 스머프가 개조한다는 설정은 자본주의 사회의 직업적 윤락여성을 공산주의자들이 교화시켰다는 의미이다.</li>
</ul>
<h2><span class="mw-headline" id=".ED.8A.B9.EC.A7.95">특징</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%8A%A4%EB%A8%B8%ED%94%84&amp;action=edit&amp;section=3" title="부분 편집: 특징">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>스머프의 키는 <a class="mw-redirect" href="/wiki/%EC%82%AC%EA%B3%BC" title="사과">사과</a>열매의 길이와 동일하며 피부는 파란색이다. 또한 엉덩이에 <a href="/wiki/%ED%86%A0%EB%81%BC" title="토끼">토끼</a>와 동일한 모양의 꼬리가 달려 있다.</p>
<h2><span class="mw-headline" id=".EB.93.B1.EC.9E.A5_.EC.9D.B8.EB.AC.BC">등장 인물</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%8A%A4%EB%A8%B8%ED%94%84&amp;action=edit&amp;section=4" title="부분 편집: 등장 인물">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<ul>
<li><b>할아버지 스머프</b>: 나이가 매우 많은 최고령 스머프로, 때때로 파파 스머프에게 조언을 하기도 하는 <a href="/wiki/%EC%98%88%EC%96%B8%EC%9E%90" title="예언자">예언자</a>적 인물이다.</li>
<li><b>파파 스머프</b>: 스머프들을 이끄는 리더로 붉은 <a href="/wiki/%EB%B0%94%EC%A7%80" title="바지">바지</a>를 입은 지도자 스머프이다.</li>
<li><b>주책이 스머프</b>: 언제나 멍하고 말이 느리다. 행동이 둔해서 실수를 많이 한다.</li>
<li><b>스머페트</b>: 여성 스머프로, 본래는 가가멜에 의해 만들어진 검은 머리 스머프로 다른 스머프들을 겨냥한 심술궂은 스머프였으나, 파파 스머프가 깨달을을 주자 아리따운 금발머리 스머프로 변신했다.</li>
<li><b>똘똘이 스머프</b>: <a href="/wiki/%EC%95%88%EA%B2%BD" title="안경">안경</a>을 쓴 스머프. 똑똑하게 문제를 해결한다.</li>
<li><b>투덜이 스머프</b>: 항상 불평 불만이 가득하여 "난 ○○는 싫어!"라고 외치는 스머프.</li>
<li><b>욕심이 스머프</b>: 먹을 것만 생각하는 스머프이다.</li>
<li><b>허영이 스머프</b>: <a class="mw-disambig" href="/wiki/%ED%99%94%EC%9E%A5" title="화장">화장</a> 하기를 좋아하며 <a href="/wiki/%EA%B1%B0%EC%9A%B8" title="거울">거울</a>보는게 취미인 스머프.</li>
<li><b>만능이 스머프</b>: 공구함을 갖고 다니며 일하는 기술자 스머프이다. <a class="mw-redirect" href="/wiki/%ED%95%9C%EA%B5%AD%EB%B0%A9%EC%86%A1" title="한국방송">한국방송</a>에서 방영할 때 이름은 편리.</li>
<li><b>시인 스머프</b>: <a class="mw-disambig" href="/wiki/%EC%8B%9C" title="시">시</a>를 짓는 것을 취미로 하는 스머프이다.</li>
<li><b>하모니 스머프</b> : <a href="/wiki/%EC%95%85%EA%B8%B0" title="악기">악기</a> 연주를 좋아하는 스머프이다. <a class="mw-redirect" href="/wiki/%ED%95%9C%EA%B5%AD%EB%B0%A9%EC%86%A1" title="한국방송">한국방송</a>에서 방영할 때 이름은 조화.</li>
<li><b>익살이 스머프</b>: 장난을 좋아하는 스머프. 상대방에게 <a href="/wiki/%ED%8F%AD%ED%83%84" title="폭탄">폭탄</a>선물을 주기를 좋아한다.</li>
<li><b>근육이 스머프</b>: 건장한 육체미를 자랑하는 스머프로 힘이 좋다. <a class="mw-redirect" href="/wiki/%ED%95%9C%EA%B5%AD%EB%B0%A9%EC%86%A1" title="한국방송">한국방송</a>에서 방영할 때 이름은 덩치.</li>
<li><b>아기 스머프</b>: 말을 잘 하지 못하고 옹알이를 하는 <a href="/wiki/%EC%95%84%EA%B8%B0" title="아기">아기</a> 스머프이다.</li>
<li><b>사세트</b>: 4명의 어린 스머프 중 하나로 스머페트와 같은 여성이다.</li>
<li><b>가가멜</b>(Gargamel): 스머프의 숙적. <a class="new" href="/w/index.php?title=%EB%A7%88%EB%B2%95%EC%82%AC_(%ED%8C%90%ED%83%80%EC%A7%80)&amp;action=edit&amp;redlink=1" title="마법사 (판타지) (없는 문서)">마법사</a>. 스머프를 잡으려고 하나 번번이 실패한다.</li>
<li><b>아즈라엘</b>: 가가멜 마법사가 키우는 <a href="/wiki/%EA%B3%A0%EC%96%91%EC%9D%B4" title="고양이">고양이</a></li>
<li><b>유모 스머프</b>: 아기 스머프들을 돌보는 스머프</li>
</ul>
<h2><span class="mw-headline" id=".EA.B0.9C.EA.B5.AC.EC.9F.81.EC.9D.B4_.EC.8A.A4.EB.A8.B8.ED.94.84_.28The_Smurfs.2C_2011.29">개구쟁이 스머프 (The Smurfs, 2011)</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%8A%A4%EB%A8%B8%ED%94%84&amp;action=edit&amp;section=5" title="부분 편집: 개구쟁이 스머프 (The Smurfs, 2011)">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
