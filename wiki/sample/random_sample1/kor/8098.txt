<p><b>스뫼르고스보르드</b>(<span style="font-size: smaller;"><a href="/wiki/%EC%8A%A4%EC%9B%A8%EB%8D%B4%EC%96%B4" title="스웨덴어">스웨덴어</a>:</span> <span lang="sv" xml:lang="sv">smörgåsbord</span>)는 <a href="/wiki/%EC%8A%A4%EC%9B%A8%EB%8D%B4_%EC%9A%94%EB%A6%AC" title="스웨덴 요리">스웨덴 요리</a>에서 <a class="mw-redirect" href="/wiki/%EB%B6%80%ED%8E%98" title="부페">부페</a> 형식의 식사를 뜻한다. 이는 흔히 <a href="/wiki/%ED%9C%B4%EC%9D%BC" title="휴일">휴일</a>의 <a href="/wiki/%EC%9E%94%EC%B9%98" title="잔치">잔치</a>에서 가족과 손님이 각자 음식을 원하는 만큼 덜어서 먹는다. 식당에서 이 용어는 많은 종류의 작은 접시들을 늘여놓은 부페식 요리로 고정된 금액 안에서 각자가 원하는 만큼 선택해서 먹을 수 있다.</p>
<p>전통적인 스웨덴의 스뫼르고스보르드는 차갑고 뜨거운 음식 둘 다가 포함된다. 관습적으로 차가운 <a href="/wiki/%EC%83%9D%EC%84%A0" title="생선">생선</a> 요리(일반적으로 <a href="/wiki/%EC%B2%AD%EC%96%B4" title="청어">청어</a>, <a href="/wiki/%EC%97%B0%EC%96%B4" title="연어">연어</a>, <a href="/wiki/%EC%9E%A5%EC%96%B4" title="장어">장어</a> 등의 다양한 형식의 음식)로 시작해서, 다른 차가운 음식을 먹다가, 스웨덴식 <a href="/wiki/%EB%AF%B8%ED%8A%B8%EB%B3%BC" title="미트볼">미트볼</a>인 셰불라르(<span lang="sv" xml:lang="sv">köttbullar</span>)와 또 다른 스웨덴의 별식인 <a class="mw-redirect" href="/wiki/%EC%96%80%EC%86%90%EC%9D%98_%EC%9C%A0%ED%98%B9" title="얀손의 유혹">얀손의 유혹</a>(<span lang="sv" xml:lang="sv">Janssons frestelse</span> <small>얀손스 프레스텔세<sup>[<a href="/wiki/%EC%9C%84%ED%82%A4%EB%B0%B1%EA%B3%BC:%EC%8A%A4%EC%9B%A8%EB%8D%B4%EC%96%B4%EC%9D%98_%ED%95%9C%EA%B8%80_%ED%91%9C%EA%B8%B0" title="위키백과:스웨덴어의 한글 표기">*</a>]</sup></small>)과 같은 뜨거운 음식들을 먹는다. <a href="/wiki/%ED%9B%84%EC%8B%9D" title="후식">후식</a>은 스뫼르고스보르드에 포함될 수도 있고, 아닐 수도 있다.</p>

<h2><span class="mw-headline" id=".EC.96.B4.EC.9B.90">어원</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%8A%A4%EB%AB%BC%EB%A5%B4%EA%B3%A0%EC%8A%A4%EB%B3%B4%EB%A5%B4%EB%93%9C&amp;action=edit&amp;section=1" title="부분 편집: 어원">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<div class="thumb tright">
<div class="thumbinner" style="width:222px;"><a class="image" href="/wiki/%ED%8C%8C%EC%9D%BC:Swedish_buffet-Sm%C3%B6rg%C3%A5sbord-02.jpg"><img alt="" class="thumbimage" data-file-height="1920" data-file-width="2560" height="165" src="//upload.wikimedia.org/wikipedia/commons/thumb/e/e9/Swedish_buffet-Sm%C3%B6rg%C3%A5sbord-02.jpg/220px-Swedish_buffet-Sm%C3%B6rg%C3%A5sbord-02.jpg" srcset="//upload.wikimedia.org/wikipedia/commons/thumb/e/e9/Swedish_buffet-Sm%C3%B6rg%C3%A5sbord-02.jpg/330px-Swedish_buffet-Sm%C3%B6rg%C3%A5sbord-02.jpg 1.5x, //upload.wikimedia.org/wikipedia/commons/thumb/e/e9/Swedish_buffet-Sm%C3%B6rg%C3%A5sbord-02.jpg/440px-Swedish_buffet-Sm%C3%B6rg%C3%A5sbord-02.jpg 2x" width="220"/></a>
<div class="thumbcaption">
<div class="magnify"><a class="internal" href="/wiki/%ED%8C%8C%EC%9D%BC:Swedish_buffet-Sm%C3%B6rg%C3%A5sbord-02.jpg" title="실제 크기로"></a></div>
스뫼르고스보르드</div>
</div>
</div>
<p><a href="/wiki/%EC%8A%A4%EC%9B%A8%EB%8D%B4%EC%96%B4" title="스웨덴어">스웨덴어</a>인 스뫼르고스보르드(<span lang="sv" xml:lang="sv">smörgåsbord</span>)는 <a href="/wiki/%EC%83%8C%EB%93%9C%EC%9C%84%EC%B9%98" title="샌드위치">샌드위치</a>를 뜻하는 '스뫼르고스(<span lang="sv" xml:lang="sv">smörgås</span>)'와 <a class="mw-redirect" href="/wiki/%EC%8B%9D%ED%83%81" title="식탁">식탁</a>을 뜻하는 '보르드(<span lang="sv" xml:lang="sv">bord</span>)'로 구성된 단어이다. 그런데 '스뫼르고스(<span lang="sv" xml:lang="sv">smörgås</span>)'는 실상 <a href="/wiki/%EB%B2%84%ED%84%B0" title="버터">버터</a>를 뜻하는 '소뫼르(<span lang="sv" xml:lang="sv">smör</span>)'와 '고스(<span lang="sv" xml:lang="sv">gås</span>)'로 구성되는데, 고스는 예전에 <a href="/wiki/%EA%B1%B0%EC%9C%84" title="거위">거위</a>를 의미했지만, 지금은 <a class="new" href="/w/index.php?title=%ED%83%88%EC%A7%80%EC%9A%B0%EC%9C%A0&amp;action=edit&amp;redlink=1" title="탈지우유 (없는 문서)">탈지우유</a>에 떠 있던 분리된 <a href="/wiki/%EB%B2%84%ED%84%B0" title="버터">버터</a>를 뜻한다.</p>
<p>스웨덴의 <a class="mw-redirect" href="/wiki/%EC%96%B8%EC%96%B4%ED%95%99%EC%9E%90" title="언어학자">언어학자</a>, 카타리나 그륀바움(<span lang="sv" xml:lang="sv">Catharina Grünbaum</span>)에 따르면<sup class="reference" id="cite_ref-1"><a href="#cite_note-1">[1]</a></sup>, '고스(<span lang="sv" xml:lang="sv">gås</span>)'는 <a href="/wiki/%EC%9B%90%EC%8B%AC%EB%B6%84%EB%A6%AC%EA%B8%B0" title="원심분리기">원심분리기</a>로 분리했을 때, <a class="mw-disambig" href="/wiki/%ED%81%AC%EB%A6%BC" title="크림">크림</a> 표면에 형성되며 떠다니던 <a href="/wiki/%EB%B2%84%ED%84%B0" title="버터">버터</a> 조각을 뜻한다고 말한다. 이 조각들이 물 위에서 수영하는 통통한 거위떼와 닮았고, 그만한 크기가 빵위에 놓아서 펴바르기에 알맞았다. 그래서 스뫼르고스는 버터와 빵을 함께 부르기 시작했다. 스웨덴에서 '버터 바른 샌드위치(<span lang="sv" xml:lang="sv">bredda smörgåsar</span> <small>브레다 스뵈르가스<sup>[<a href="/wiki/%EC%9C%84%ED%82%A4%EB%B0%B1%EA%B3%BC:%EC%8A%A4%EC%9B%A8%EB%8D%B4%EC%96%B4%EC%9D%98_%ED%95%9C%EA%B8%80_%ED%91%9C%EA%B8%B0" title="위키백과:스웨덴어의 한글 표기">*</a>]</sup></small>)'라는 용어가 최소한 <a href="/wiki/16%EC%84%B8%EA%B8%B0" title="16세기">16세기</a>부터 사용되었으며, ‘오늘날 버터 바른 빵’을 의미할 뿐만 아니라, ‘버터 바른 샌드위치 사이에 껴있는 음식물’을 뜻한다.</p>

율보르드, <a href="/wiki/%EC%8A%A4%EC%9B%A8%EB%8D%B4" title="스웨덴">스웨덴</a>의 <a class="mw-redirect" href="/wiki/%EC%84%B1%ED%83%84%EC%A0%88" title="성탄절">성탄절</a> <a class="mw-disambig" href="/wiki/%EC%A0%95%EC%B0%AC" title="정찬">정찬</a></div>

<p><b>율보르드</b>(<span lang="sv" xml:lang="sv">julbord</span>)는 <a class="mw-redirect" href="/wiki/%EC%84%B1%ED%83%84%EC%A0%88" title="성탄절">성탄절</a>을 위한 스뫼르고스보르드로, 스웨덴의 성탄절 <a class="mw-disambig" href="/wiki/%EC%A0%95%EC%B0%AC" title="정찬">정찬</a>이다. 이 용어는 스웨덴어에서 성탄절를 뜻하는 율레(<span lang="sv" xml:lang="sv">Yule</span>)의 '율(<span lang="sv" xml:lang="sv">yul</span>)'와 <a class="mw-redirect" href="/wiki/%EC%8B%9D%ED%83%81" title="식탁">식탁</a>을 뜻하는 '보르드(<span lang="sv" xml:lang="sv">bord</span>)'가 합쳐진 말이다. 스웨덴식 율보르드는 전통적으로 <a href="/wiki/12%EC%9B%94" title="12월">12월</a> 초부터 성탄절까지 집이나 식당에서 먹는 부페를 뜻한다.</p>
<p>전통적인 율보르드는 일반적으로 3가지 주요리로 먹는다. 첫 번째 주요리는 가공한 <a href="/wiki/%EC%97%B0%EC%96%B4" title="연어">연어</a>와 절인 <a href="/wiki/%EC%B2%AD%EC%96%B4" title="청어">청어</a>와 <a href="/wiki/%EC%9E%A5%EC%96%B4" title="장어">장어</a>를 다양한 <a class="mw-redirect" href="/wiki/%EC%96%91%EB%85%90" title="양념">양념</a>으로 요리한 음식으로 구성된다. 스웨덴 사람들은 관습적으로 특정 음식은 맛과 <a class="new" href="/w/index.php?title=%EC%8B%9D%EA%B0%90&amp;action=edit&amp;redlink=1" title="식감 (없는 문서)">식감</a>이 잘 어울리는 음식들과 함께 먹는다. 예를 들면, 청어는 일반적으로 삶은 감자와 완숙한 <a href="/wiki/%EB%8B%AC%EA%B1%80" title="달걀">달걀</a>과 함께 먹으며, 종종 향료를 곁들이거나, 곁들이지 않는 <a href="/wiki/%EB%B3%B4%EB%93%9C%EC%B9%B4" title="보드카">보드카</a> 한잔을 같이 마신다. 두 번째 주요리는 <a href="/wiki/%EC%B9%A0%EB%A9%B4%EC%A1%B0%EA%B3%A0%EA%B8%B0" title="칠면조고기">칠면조고기</a>, 구운 <a href="/wiki/%EC%87%A0%EA%B3%A0%EA%B8%B0" title="쇠고기">쇠고기</a>인 <a href="/wiki/%EB%A1%9C%EC%8A%A4%ED%8A%B8_%EB%B9%84%ED%94%84" title="로스트 비프">로스트 비프</a>, 성탄절 용 <a href="/wiki/%ED%96%84" title="햄">햄</a>인 '율스킨카(<span lang="sv" xml:lang="sv">julskinka</span>)' 등 얇게 썬 차가운 고기들로 구성된다. 얇게 썬 <a href="/wiki/%EC%B9%98%EC%A6%88" title="치즈">치즈</a>와 <a href="/wiki/%EC%98%A4%EC%9D%B4" title="오이">오이</a>, <a class="new" href="/w/index.php?title=%EA%B0%84_(%EC%9D%8C%EC%8B%9D)&amp;action=edit&amp;redlink=1" title="간 (음식) (없는 문서)">간</a> <a class="mw-disambig" href="/wiki/%ED%8C%8C%ED%85%8C" title="파테">파테</a>는 종종 딱딱한 <a href="/wiki/%EB%B9%B5" title="빵">빵</a> 위에 얹어서 먹는다. 세 번째 주요리는 스웨덴의 <a href="/wiki/%EB%AF%B8%ED%8A%B8%EB%B3%BC" title="미트볼">미트볼</a>인 '셰ㅅ불라르(<span lang="sv" xml:lang="sv">köttbullar</span>)'와 작은 <a class="mw-redirect" href="/wiki/%EC%86%8C%EC%84%B8%EC%A7%80" title="소세지">소세지</a>인 '프린스코르브(<span lang="sv" xml:lang="sv">prinskorv</span>)', <a href="/wiki/%EC%96%91%EB%B0%B0%EC%B6%94%EB%A7%90%EC%9D%B4" title="양배추말이">양배추말이</a>인 '<a href="/wiki/%EC%BD%9C%EB%8F%8C%EB%A7%88%EB%A5%B4" title="콜돌마르">콜돌마르</a>', <a href="/wiki/%EC%A0%A4%EB%A6%AC" title="젤리">젤리</a>처럼 굳힌 <a class="new" href="/w/index.php?title=%EB%8F%BC%EC%A7%80%EC%A1%B1%EB%B0%9C&amp;action=edit&amp;redlink=1" title="돼지족발 (없는 문서)">돼지족발</a> 요리, 뻑뻑한 <a class="mw-redirect" href="/wiki/%ED%99%94%EC%9D%B4%ED%8A%B8_%EC%86%8C%EC%8A%A4" title="화이트 소스">화이트 소스</a>에 말린 <a class="mw-redirect" href="/wiki/%EB%8C%80%EA%B5%AC_(%EB%8F%99%EB%AC%BC)" title="대구 (동물)">대구</a>를 재구성한 음식인 '<a class="new" href="/w/index.php?title=%EB%A3%A8%ED%8A%B8%ED%94%BC%EC%8A%A4%ED%81%AC&amp;action=edit&amp;redlink=1" title="루트피스크 (없는 문서)">루트피스크</a>', 오븐에 구운 <a href="/wiki/%EB%8F%BC%EC%A7%80%EA%B0%88%EB%B9%84" title="돼지갈비">돼지갈비</a>인 '레브벤스폘(<span lang="sv" xml:lang="sv">revbenspjäll</span>)', '얀손의 유혹' 등 따뜻한 요리로 구성된다.</p>
<p>율보르드에서 <a href="/wiki/%ED%9B%84%EC%8B%9D" title="후식">후식</a>은 <a href="/wiki/%EB%9D%BC%EC%9D%B4%EC%8A%A4_%ED%91%B8%EB%94%A9" title="라이스 푸딩">라이스 푸딩</a>에 <a href="/wiki/%EA%B3%84%ED%94%BC" title="계피">계피</a>가루를 뿌린 '리스그륀스그뢰트(<span lang="sv" xml:lang="sv">risgrynsgröt</span>)'를 포함한다. 전통적으로 <a href="/wiki/%EC%95%84%EB%AA%AC%EB%93%9C" title="아몬드">아몬드</a>가 리스그륀스그뢰트 안에 숨어져 있는데, 발견하는 사람은 작은 상을 받거나, <a class="mw-redirect" href="/wiki/%ED%96%89%EC%9A%B4" title="행운">행운</a>을 가지게 될 것이라고 생각한다. <sup class="정리_필요 noprint">[<a href="/wiki/%EC%9C%84%ED%82%A4%EB%B0%B1%EA%B3%BC:%EC%B6%9C%EC%B2%98_%EB%B0%9D%ED%9E%88%EA%B8%B0" title="위키백과:출처 밝히기">출처 필요</a>]</sup></p>
<h2><span class="mw-headline" id=".EC.98.81.EC.96.B4.EC.97.90.EC.84.9C.EC.9D.98_.EC.93.B0.EC.9E.84">영어에서의 쓰임</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%8A%A4%EB%AB%BC%EB%A5%B4%EA%B3%A0%EC%8A%A4%EB%B3%B4%EB%A5%B4%EB%93%9C&amp;action=edit&amp;section=3" title="부분 편집: 영어에서의 쓰임">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p><a href="/wiki/%EC%98%81%EC%96%B4" title="영어">영어</a>에서 <span lang="sv" xml:lang="sv">smörgåsbord</span>, 또는 종종 '스모르거스보드(<span lang="en" xml:lang="en">smorgasbord</span>)'는 스웨덴 <a href="/wiki/%EC%A0%84%ED%86%B5" title="전통">전통</a>과 관련할 필요없이, 다양한 음식으로 구성된 어떠한 종류의 부페 식사를 뜻한다. 넓은 의미에서 이 단어는 <a class="mw-redirect" href="/wiki/%ED%9B%84%EC%9B%90%EC%9E%90" title="후원자">후원자</a>를 초대해서 그들이 몇가지 유쾌한 것 중에서 원하는 것을 고른다는 상황을 뜻한다. 예를 들어 <a href="/wiki/%EB%8C%80%ED%95%99" title="대학">대학</a> <a href="/wiki/%EC%88%98%EC%97%85" title="수업">수업</a> 또는 <a href="/wiki/%EC%84%9C%EC%A0%90" title="서점">서점</a>의 <a href="/wiki/%EC%B1%85" title="책">책</a>의 스모르거스보드 등에서 쓰인다. 또한 이 단어는 다양한 집합을 뜻할 때도 쓰이며, <span lang="en" xml:lang="en">hodge-podge</span>와 동의어이다.</p>
<p><a href="/wiki/%EC%98%A4%EC%8A%A4%ED%8A%B8%EB%A0%88%EC%9D%BC%EB%A6%AC%EC%95%84" title="오스트레일리아">오스트레일리아</a>에서 <span lang="en" xml:lang="en">smorgasbord</span>는 종종 호주식 부페를 포함하여, 모든 형식의 부페 식사를 뜻한다.</p>
<p><a href="/wiki/%EC%BA%90%EB%82%98%EB%8B%A4" title="캐나다">캐나다</a>에서 '중국식 스뫼르고스보르드(<span lang="en" xml:lang="en">Chinese smorgasbord</span>)'는 '캐나다식 중국 요리'에서 부페 음식을 뜻한다. 이 전통은 뒤에 <a href="/wiki/%EB%B0%B4%EC%BF%A0%EB%B2%84" title="밴쿠버">밴쿠버</a>가 된 <a class="new" href="/w/index.php?title=%EC%BA%90%EC%8A%A4%ED%83%80%EC%9A%B4&amp;action=edit&amp;redlink=1" title="캐스타운 (없는 문서)">캐스타운</a>, <a class="new" href="/w/index.php?title=%EB%B8%8C%EB%A6%AC%ED%8B%B0%EC%8B%9C_%EC%BD%9C%EB%A1%AC%EB%B9%84%EC%95%84&amp;action=edit&amp;redlink=1" title="브리티시 콜롬비아 (없는 문서)">브리티시 콜롬비아</a>에서 유래된 것으로, <a href="/wiki/%EC%8A%A4%EC%B9%B8%EB%94%94%EB%82%98%EB%B9%84%EC%95%84" title="스칸디나비아">스칸디나비아</a>의 <a class="mw-redirect" href="/wiki/%EC%9D%B4%EB%AF%BC" title="이민">이민</a> <a class="new" href="/w/index.php?title=%EC%A0%9C%EB%B6%84%EC%86%8C&amp;action=edit&amp;redlink=1" title="제분소 (없는 문서)">제분소</a> <a href="/wiki/%EB%85%B8%EB%8F%99%EC%9E%90" title="노동자">노동자</a>와 <a class="new" href="/w/index.php?title=%EB%B2%8C%EB%AA%A9%EA%BE%BC&amp;action=edit&amp;redlink=1" title="벌목꾼 (없는 문서)">벌목꾼</a>들이 그들의 <a href="/wiki/%EC%A4%91%EA%B5%AD%EC%9D%B8" title="중국인">중국인</a> <a class="mw-redirect" href="/wiki/%EC%9A%94%EB%A6%AC%EC%82%AC" title="요리사">요리사</a>들에게 <a class="mw-disambig" href="/wiki/%EA%B3%A0%ED%96%A5" title="고향">고향</a>에서처럼 먹던 방식을 <a class="mw-redirect" href="/wiki/%EC%A4%91%EA%B5%AD_%EC%9D%8C%EC%8B%9D" title="중국 음식">중국 음식</a>에 적용시켰다고 한다. <sup class="정리_필요 noprint">[<a href="/wiki/%EC%9C%84%ED%82%A4%EB%B0%B1%EA%B3%BC:%EC%B6%9C%EC%B2%98_%EB%B0%9D%ED%9E%88%EA%B8%B0" title="위키백과:출처 밝히기">출처 필요</a>]</sup></p>
