<p><b>스미다 구</b>(<span style="font-size: smaller;"><a href="/wiki/%EC%9D%BC%EB%B3%B8%EC%96%B4" title="일본어">일본어</a>:</span> <span lang="ja" xml:lang="ja">墨田区, すみだく</span>）는 <a href="/wiki/%EC%9D%BC%EB%B3%B8" title="일본">일본</a> <a href="/wiki/%EB%8F%84%EC%BF%84_%EB%8F%84" title="도쿄 도">도쿄 도</a>에 있는 <a class="mw-redirect" href="/wiki/%EB%8F%84%EC%BF%84_23%EA%B5%AC" title="도쿄 23구">특별구</a>의 하나이다. <a class="new" href="/w/index.php?title=%EC%8B%9C%ED%83%80%EB%A7%88%EC%B9%98&amp;action=edit&amp;redlink=1" title="시타마치 (없는 문서)">시타마치</a>(<span lang="ja" xml:lang="ja">下町</span>)라고 불리는 <a href="/wiki/%EC%A0%9C2%EC%B0%A8_%EC%84%B8%EA%B3%84_%EB%8C%80%EC%A0%84" title="제2차 세계 대전">전쟁</a> 이전부터의 주택 지역이다. 또 프로 <a href="/wiki/%EC%8A%A4%EB%AA%A8" title="스모">스모</a>의 대회가 열리는 <a href="/wiki/%EB%A3%8C%EA%B3%A0%EC%BF%A0_%EA%B5%AD%EA%B8%B0%EA%B4%80" title="료고쿠 국기관">료고쿠 국기관</a>(<span lang="ja" xml:lang="ja">両国国技館</span>)이 있다.</p>
<p>2016년 10월 1일 실시한 인구조사에 의하면 인구는 260,358명으로 집계됐고, 인구 밀도는 18,690km²였다. 면적은 13.77 km²이다.</p>

<h2><span class="mw-headline" id=".EC.A7.80.EB.A6.AC">지리</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%8A%A4%EB%AF%B8%EB%8B%A4_%EA%B5%AC&amp;action=edit&amp;section=1" title="부분 편집: 지리">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>스미다 구는 도쿄의 본토 부분의 북동부에 위치한다. 주요 강으로 <a href="/wiki/%EC%8A%A4%EB%AF%B8%EB%8B%A4_%EA%B0%95" title="스미다 강">스미다 강</a>과 <a class="mw-redirect" href="/wiki/%EC%95%84%EB%9D%BC%EC%B9%B4%EC%99%80_%EA%B0%95_(%EB%8F%84%EC%BF%84_%EB%A7%8C)" title="아라카와 강 (도쿄 만)">아라카와 강</a>이 있으며 구의 경계를 일부를 이룬다. 북쪽으로 <a href="/wiki/%EC%95%84%EB%8B%A4%EC%B9%98_%EA%B5%AC" title="아다치 구">아다치 구</a>, 북서쪽으로 <a href="/wiki/%EC%95%84%EB%9D%BC%EC%B9%B4%EC%99%80_%EA%B5%AC" title="아라카와 구">아라카와 구</a>, 동쪽으로 <a href="/wiki/%EA%B0%80%EC%93%B0%EC%8B%9C%EC%B9%B4_%EA%B5%AC" title="가쓰시카 구">가쓰시카 구</a>, 남동쪽으로 <a href="/wiki/%EC%97%90%EB%8F%84%EA%B0%80%EC%99%80_%EA%B5%AC" title="에도가와 구">에도가와 구</a>, 서쪽으로 <a href="/wiki/%EB%8B%A4%EC%9D%B4%ED%86%A0_%EA%B5%AC" title="다이토 구">다이토 구</a>, 남서쪽으로 <a href="/wiki/%EC%A3%BC%EC%98%A4_%EA%B5%AC_(%EB%8F%84%EC%BF%84_%EB%8F%84)" title="주오 구 (도쿄 도)">주오 구</a>, 남쪽으로 <a href="/wiki/%EA%B3%A0%ED%86%A0_%EA%B5%AC" title="고토 구">고토 구</a>와 접한다.</p>
