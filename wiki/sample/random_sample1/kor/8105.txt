<p><b>스베이든 비외르든손</b>(<span style="font-size: smaller;"><a href="/wiki/%EC%95%84%EC%9D%B4%EC%8A%AC%EB%9E%80%EB%93%9C%EC%96%B4" title="아이슬란드어">아이슬란드어</a>:</span> <span lang="is" xml:lang="is">Sveinn Björnsson</span>, <a href="/wiki/1881%EB%85%84" title="1881년">1881년</a> <a href="/wiki/2%EC%9B%94_27%EC%9D%BC" title="2월 27일">2월 27일</a> ~ <a href="/wiki/1952%EB%85%84" title="1952년">1952년</a> <a href="/wiki/1%EC%9B%94_25%EC%9D%BC" title="1월 25일">1월 25일</a>)은 <a href="/wiki/%EC%95%84%EC%9D%B4%EC%8A%AC%EB%9E%80%EB%93%9C" title="아이슬란드">아이슬란드</a>의 초대 대통령이었다.</p>
<p><a href="/wiki/%EB%8D%B4%EB%A7%88%ED%81%AC" title="덴마크">덴마크</a>의 <a href="/wiki/%EC%BD%94%ED%8E%9C%ED%95%98%EA%B2%90" title="코펜하겐">코펜하겐</a>에서 태어났다. <a href="/wiki/1912%EB%85%84" title="1912년">1912년</a> <a href="/wiki/%EB%A0%88%EC%9D%B4%EC%BA%AC%EB%B9%84%ED%81%AC" title="레이캬비크">레이캬비크</a> 시의원이 되어 <a href="/wiki/1918%EB%85%84" title="1918년">1918년</a>부터 <a href="/wiki/1920%EB%85%84" title="1920년">1920년</a>까지 의회 의장을 맡고 있었다. <a href="/wiki/1914%EB%85%84" title="1914년">1914년</a>부터 <a href="/wiki/1916%EB%85%84" title="1916년">1916년</a>까지, <a href="/wiki/1920%EB%85%84" title="1920년">1920년</a> 국회의원을 맡았으며 <a href="/wiki/1944%EB%85%84" title="1944년">1944년</a> 대통령이 되어 죽을 때 까지 그 권한을 맡았다.</p>
