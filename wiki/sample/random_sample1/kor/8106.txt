<p><b>스베틀라나 레오니도브나 보긴스카야</b>(<span style="font-size: smaller;"><a href="/wiki/%EB%9F%AC%EC%8B%9C%EC%95%84%EC%96%B4" title="러시아어">러시아어</a>:</span> <span lang="ru" xml:lang="ru">Светлана Леонидовна Богинская</span>, <span style="font-size: smaller;"><a href="/wiki/%EB%B2%A8%EB%9D%BC%EB%A3%A8%EC%8A%A4%EC%96%B4" title="벨라루스어">벨라루스어</a>:</span> <span lang="be" xml:lang="be">Святлана Леанідаўна Багінская</span> <small>스뱌틀라나 레아니다위나 바긴스카야</small>, <a href="/wiki/1973%EB%85%84" title="1973년">1973년</a> <a href="/wiki/2%EC%9B%94_9%EC%9D%BC" title="2월 9일">2월 9일</a> ~ )는 <a href="/wiki/%EB%B2%A8%EB%9D%BC%EB%A3%A8%EC%8A%A4" title="벨라루스">벨라루스</a>의 <a href="/wiki/%EC%B2%B4%EC%A1%B0" title="체조">체조</a> 선수로 <a class="mw-redirect" href="/wiki/%EC%86%8C%EB%B9%84%EC%97%90%ED%8A%B8_%EC%97%B0%EB%B0%A9" title="소비에트 연방">소비에트 연방</a>을 대표하여 <a class="mw-redirect" href="/wiki/1988%EB%85%84_%EC%84%9C%EC%9A%B8_%EC%98%AC%EB%A6%BC%ED%94%BD" title="1988년 서울 올림픽">1988년 서울 올림픽</a>에서 4개의 메달을 획득하였다. 그녀의 신장, 발레적 우미 등의 이유로 "벨라루스의 백조", "체조의 여신"으로 불리었다.</p>
<h2><span class="mw-headline" id=".EA.B2.BD.EB.A0.A5">경력</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%8A%A4%EB%B2%A0%ED%8B%80%EB%9D%BC%EB%82%98_%EB%B3%B4%EA%B8%B4%EC%8A%A4%EC%B9%B4%EC%95%BC&amp;action=edit&amp;section=1" title="부분 편집: 경력">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p><a href="/wiki/%EB%B2%A8%EB%9D%BC%EB%A3%A8%EC%8A%A4" title="벨라루스">벨라루스</a> <a href="/wiki/%EB%AF%BC%EC%8A%A4%ED%81%AC" title="민스크">민스크</a>에서 태어나 <a class="mw-redirect" href="/wiki/%ED%94%BC%EA%B2%A8_%EC%8A%A4%EC%BC%80%EC%9D%B4%ED%8A%B8" title="피겨 스케이트">피겨 스케이트</a>를 활약하다가, 6세 때에 체조를 시작하였다. 2년 후, <a href="/wiki/%EB%AA%A8%EC%8A%A4%ED%81%AC%EB%B0%94" title="모스크바">모스크바</a>의 원형호수 체조 센터로 옮겨 전임적으로 훈련하였고, 14세 때에 소비에트 연방 국가 대표에 뽑혔다.</p>
<p><a href="/wiki/1987%EB%85%84" title="1987년">1987년</a> 세계 선수권 대회에서 평균대 운동에서 3위를 시작으로 국제 데뷔를 한 보긴스카야는 소비에트 팀의 최고 체조 선수들 중의 하나가 되어 <a href="/wiki/1988%EB%85%84_%ED%95%98%EA%B3%84_%EC%98%AC%EB%A6%BC%ED%94%BD" title="1988년 하계 올림픽">서울 올림픽</a>에서 2개의 <a href="/wiki/%EA%B8%88%EB%A9%94%EB%8B%AC" title="금메달">금메달</a>(단체, 도마), 1개의 <a href="/wiki/%EC%9D%80%EB%A9%94%EB%8B%AC" title="은메달">은메달</a>(마루 운동), 1개의 <a href="/wiki/%EB%8F%99%EB%A9%94%EB%8B%AC" title="동메달">동메달</a>(개인 종합)을 획득한다. 그 일은 동료 선수 <a href="/wiki/%EC%98%90%EB%A0%88%EB%82%98_%EC%8A%88%EC%8A%88%EB%85%B8%EB%B0%94" title="옐레나 슈슈노바">옐레나 슈슈노바</a>와 <a href="/wiki/%EB%A3%A8%EB%A7%88%EB%8B%88%EC%95%84" title="루마니아">루마니아</a>의 <a class="new" href="/w/index.php?title=%EB%8B%A4%EB%8B%88%EC%97%98%EB%9D%BC_%EC%8B%A4%EB%A6%AC%EB%B0%94%EC%8A%A4&amp;action=edit&amp;redlink=1" title="다니엘라 실리바스 (없는 문서)">다니엘라 실리바스</a> 사이의 투쟁에 회전한 올림픽에서 어린 선수들에 큰 성취였다.</p>
<p>경기가 끝난 3일 후, 그녀의 장기간 코치 류보프 미로마노바가 자살하는 사건이 생겼다.</p>
<p>후에 새 코이 루드밀라 포프코비치 아래에서 훈련한 보긴스카야는 <a href="/wiki/1989%EB%85%84" title="1989년">1989년</a> 세계 선수권 3관왕이 되었다.</p>
<p><a href="/wiki/1990%EB%85%84" title="1990년">1990년</a> <a href="/wiki/%EB%B2%A0%EB%9D%BC_%EC%B0%A8%EC%8A%AC%EB%9D%BC%ED%94%84%EC%8A%A4%EC%B9%B4" title="베라 차슬라프스카">베라 차슬라프스카</a>(1965, 1967), <a href="/wiki/%EB%A5%98%EB%93%9C%EB%B0%80%EB%9D%BC_%ED%88%AC%EB%A6%AC%EC%8A%A4%EC%B2%B4%EB%B0%94" title="류드밀라 투리스체바">류드밀라 투리스체바</a>(1973)에 이어 모든 개인적 종목을 우승하면서 유럽 선수권 대회를 휩쓴 세 번째 여자 선수가 되었다. <a href="/wiki/1991%EB%85%84" title="1991년">1991년</a> 세계 선수권 대회 개인 종합전에서 <a href="/wiki/%EB%AF%B8%EA%B5%AD" title="미국">미국</a>의 <a class="new" href="/w/index.php?title=%ED%82%B4_%EC%B8%A0%EB%A9%94%EC%8A%A4%EC%B9%BC&amp;action=edit&amp;redlink=1" title="킴 츠메스칼 (없는 문서)">킴 츠메스칼</a>에게 패하였으나, 단체전과 평균대 운동에서 금메달을 획득하였다.</p>
<p><a href="/wiki/1992%EB%85%84" title="1992년">1992년</a> 유럽 선수권 대회에서 마루 운동 5위로 실망적인 상연을 가졌으나 아직도 <a href="/wiki/1992%EB%85%84_%ED%95%98%EA%B3%84_%EC%98%AC%EB%A6%BC%ED%94%BD" title="1992년 하계 올림픽">바르셀로나 올림픽</a>에서 개인 종합 우승 후보로 보였다.</p>
<p>체조 팬들은 보긴스카야와 츠메스칼 사이의 결투를 예상하였다. 보긴스카야는 단체전을 2연승하여 자신의 3번째 올림픽 금메달을 땄다. 개인 종합 전에서 그녀는 5위, 츠메스칼은 12위로 왔다. 둘 사이의 대결에 예상 대신 <a class="new" href="/w/index.php?title=%ED%83%80%ED%8B%B0%EC%95%84%EB%82%98_%EA%B5%AC%EC%B6%94&amp;action=edit&amp;redlink=1" title="타티아나 구추 (없는 문서)">타티아나 구추</a>와 <a class="new" href="/w/index.php?title=%EC%84%80%EB%84%8C_%EB%B0%80%EB%9F%AC&amp;action=edit&amp;redlink=1" title="섀넌 밀러 (없는 문서)">섀넌 밀러</a>가 올림픽 역사에서 가장 드라마적인 경쟁들 중의 하나를 마련하였다.</p>
<p>바르셀로나 올림픽 후에 은퇴한 보긴스카야는 <a href="/wiki/1995%EB%85%84" title="1995년">1995년</a> 복귀하기로 결정하였는 데, <a href="/wiki/1994%EB%85%84_%EB%8F%99%EA%B3%84_%EC%98%AC%EB%A6%BC%ED%94%BD" title="1994년 동계 올림픽">1994년 동계 올림픽</a>에 돌아온 <a href="/wiki/%EC%B9%B4%ED%83%80%EB%A6%AC%EB%82%98_%EB%B9%84%ED%8A%B8" title="카타리나 비트">카타리나 비트</a>의 영감을 받았다고 한다.</p>
<p><a href="/wiki/1996%EB%85%84" title="1996년">1996년</a> 유럽 선수권 대회 개인 종합전에서 <a href="/wiki/%EC%9A%B0%ED%81%AC%EB%9D%BC%EC%9D%B4%EB%82%98" title="우크라이나">우크라이나</a>의 <a href="/wiki/%EB%A6%B4%EB%A6%AC%EC%95%BC_%ED%8F%AC%EB%93%9C%EC%BD%94%ED%8C%8C%EC%98%88%EB%B0%94" title="릴리야 포드코파예바">릴리야 포드코파예바</a>에게 밀려 2위를 하였다. <a href="/wiki/1996%EB%85%84_%ED%95%98%EA%B3%84_%EC%98%AC%EB%A6%BC%ED%94%BD" title="1996년 하계 올림픽">애틀랜타 올림픽</a>에 참가하여 단체전에서 벨라루스 팀을 6위를 이끌었고, 개인 종합과 도마 운동에도 나왔으나 그녀의 전성기가 끝나고 말았다.</p>
<p>보긴스카야는 3개의 올림픽에 참가한 여성 체조 선수들 중의 하나이며, 3개의 다른 올림픽 팀 - 소비에트 연방, <a class="mw-redirect" href="/wiki/%EB%8F%85%EB%A6%BD_%EA%B5%AD%EA%B0%80_%EC%97%B0%ED%95%A9" title="독립 국가 연합">독립 국가 연합</a>, 벨라루스를 대표하였다. <a href="/wiki/2005%EB%85%84" title="2005년">2005년</a> 국제 체조 명예의 전당에 헌액되었다.</p>
<p>오늘날 <a href="/wiki/%ED%85%8D%EC%82%AC%EC%8A%A4_%EC%A3%BC" title="텍사스 주">텍사스 주</a> <a href="/wiki/%ED%9C%B4%EC%8A%A4%ED%84%B4" title="휴스턴">휴스턴</a>에 살면서 2개의 <a class="mw-redirect" href="/wiki/%EB%B9%84%EC%A7%80%EB%8B%88%EC%8A%A4" title="비지니스">비지니스</a> - 온라인 체조 의상 소매업과 체조 선수 학생들을 위한 여름 캠프를 운영하고 있다.</p>
