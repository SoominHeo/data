<p><b>스벤예란 에릭손</b>(<span style="font-size: smaller;"><a href="/wiki/%EC%8A%A4%EC%9B%A8%EB%8D%B4%EC%96%B4" title="스웨덴어">스웨덴어</a>:</span> <span lang="sv" xml:lang="sv">Sven-Göran Eriksson</span>, <a class="mw-redirect" href="/wiki/IPA" title="IPA">IPA</a>: <span class="IPA" title="IPA 발음">[svɛn'jœːrɑn 'eːrɪksɔn]</span><span class="unicode audiolink"><a class="internal" href="//upload.wikimedia.org/wikipedia/commons/a/a1/Sv-Sven-G%C3%B6ran_Eriksson.ogg" title="Sv-Sven-Göran Eriksson.ogg">▶</a></span> <span class="metadata audiolinkinfo"><small>(<a href="/wiki/%EC%9C%84%ED%82%A4%EB%B0%B1%EA%B3%BC:%EB%AF%B8%EB%94%94%EC%96%B4_%EB%8F%84%EC%9B%80%EB%A7%90" title="위키백과:미디어 도움말">도움말</a>·<a href="/wiki/%ED%8C%8C%EC%9D%BC:Sv-Sven-G%C3%B6ran_Eriksson.ogg" title="파일:Sv-Sven-Göran Eriksson.ogg">정보</a>)</small></span>, <a href="/wiki/1948%EB%85%84" title="1948년">1948년</a> <a href="/wiki/2%EC%9B%94_5%EC%9D%BC" title="2월 5일">2월 5일</a> ~ ) 또는 <b>스베니스 에릭손</b>(<span style="font-size: smaller;"><a href="/wiki/%EC%8A%A4%EC%9B%A8%EB%8D%B4%EC%96%B4" title="스웨덴어">스웨덴어</a>:</span> <span lang="sv" xml:lang="sv">Svennis Eriksson</span>)은 <a href="/wiki/%EC%8A%A4%EC%9B%A8%EB%8D%B4" title="스웨덴">스웨덴</a>의 <a href="/wiki/%EC%B6%95%EA%B5%AC" title="축구">축구</a> 감독이다. <a href="/wiki/%EC%BD%94%ED%8A%B8%EB%94%94%EB%B6%80%EC%95%84%EB%A5%B4_%EC%B6%95%EA%B5%AC_%EA%B5%AD%EA%B0%80%EB%8C%80%ED%91%9C%ED%8C%80" title="코트디부아르 축구 국가대표팀">코트디부아르 축구 국가대표팀</a>을 맡다가 <a href="/wiki/2010%EB%85%84_FIFA_%EC%9B%94%EB%93%9C%EC%BB%B5" title="2010년 FIFA 월드컵">2010년 FIFA 월드컵</a> 이후 사임하였다. 과거 <a href="/wiki/%EC%8A%A4%EC%9B%A8%EB%8D%B4" title="스웨덴">스웨덴</a>, <a href="/wiki/%ED%8F%AC%EB%A5%B4%ED%88%AC%EA%B0%88" title="포르투갈">포르투갈</a>, <a href="/wiki/%EC%9D%B4%ED%83%88%EB%A6%AC%EC%95%84" title="이탈리아">이탈리아</a> 등 3개국에서 리그 및 컵 <a href="/wiki/%EB%8D%94%EB%B8%94_(%EC%B6%95%EA%B5%AC)" title="더블 (축구)">더블</a>을 기록한 유일한 감독기도 하다.</p>
<p>에릭손은 두 번의 <a href="/wiki/FIFA_%EC%9B%94%EB%93%9C%EC%BB%B5" title="FIFA 월드컵">FIFA 월드컵</a>(<a href="/wiki/2002%EB%85%84_FIFA_%EC%9B%94%EB%93%9C%EC%BB%B5" title="2002년 FIFA 월드컵">2002년</a>, <a href="/wiki/2006%EB%85%84_FIFA_%EC%9B%94%EB%93%9C%EC%BB%B5" title="2006년 FIFA 월드컵">2006년</a>)에서 <a href="/wiki/%EC%9E%89%EA%B8%80%EB%9E%9C%EB%93%9C_%EC%B6%95%EA%B5%AC_%EA%B5%AD%EA%B0%80%EB%8C%80%ED%91%9C%ED%8C%80" title="잉글랜드 축구 국가대표팀">잉글랜드 축구 국가대표팀</a> 감독을 맡았으나(외국인으로는 최초) 팀이 각각 <a href="/wiki/%EB%B8%8C%EB%9D%BC%EC%A7%88_%EC%B6%95%EA%B5%AC_%EA%B5%AD%EA%B0%80%EB%8C%80%ED%91%9C%ED%8C%80" title="브라질 축구 국가대표팀">브라질</a>과 <a href="/wiki/%ED%8F%AC%EB%A5%B4%ED%88%AC%EA%B0%88_%EC%B6%95%EA%B5%AC_%EA%B5%AD%EA%B0%80%EB%8C%80%ED%91%9C%ED%8C%80" title="포르투갈 축구 국가대표팀">포르투갈</a>에게 패하여 8강에서 탈락하자 사임하였다.</p>
<p><a href="/wiki/2007%EB%85%84" title="2007년">2007년</a>부터 <a class="mw-redirect" href="/wiki/%EC%9E%89%EA%B8%80%EB%9E%9C%EB%93%9C_%ED%94%84%EB%A6%AC%EB%AF%B8%EC%96%B4%EB%A6%AC%EA%B7%B8" title="잉글랜드 프리미어리그">잉글랜드 프리미어리그</a>의 <a href="/wiki/%EB%A7%A8%EC%B2%B4%EC%8A%A4%ED%84%B0_%EC%8B%9C%ED%8B%B0_FC" title="맨체스터 시티 FC">맨체스터 시티 FC</a>의 감독을 지냈으나 시즌이 끝난 뒤 팀을 떠나게 되었고 <a href="/wiki/2008%EB%85%84" title="2008년">2008년</a>부터 <a href="/wiki/2009%EB%85%84" title="2009년">2009년</a>까지 <a href="/wiki/%EB%A9%95%EC%8B%9C%EC%BD%94_%EC%B6%95%EA%B5%AC_%EA%B5%AD%EA%B0%80%EB%8C%80%ED%91%9C%ED%8C%80" title="멕시코 축구 국가대표팀">멕시코 축구 국가대표팀</a> 팀을 이끌었으나 성적 부진으로 <a href="/wiki/2010%EB%85%84_FIFA_%EC%9B%94%EB%93%9C%EC%BB%B5_%EC%98%88%EC%84%A0" title="2010년 FIFA 월드컵 예선">2010년 FIFA 월드컵 예선</a> 도중 중도 하차하였다.</p>
<p><a href="/wiki/2010%EB%85%84" title="2010년">2010년</a> <a href="/wiki/%EB%85%B8%EC%B8%A0_%EC%B9%B4%EC%9A%B4%ED%8B%B0_FC" title="노츠 카운티 FC">노츠 카운티 FC</a>의 단장을 맡기도 하였으며 <a href="/wiki/%EC%BD%94%ED%8A%B8%EB%94%94%EB%B6%80%EC%95%84%EB%A5%B4_%EC%B6%95%EA%B5%AC_%EA%B5%AD%EA%B0%80%EB%8C%80%ED%91%9C%ED%8C%80" title="코트디부아르 축구 국가대표팀">코트디부아르 축구 국가대표팀</a> 감독으로 선임되어 <a href="/wiki/2010%EB%85%84_FIFA_%EC%9B%94%EB%93%9C%EC%BB%B5" title="2010년 FIFA 월드컵">2010년 FIFA 월드컵</a>에 나오게 되었다. 하지만 팀이 월드컵 조별 예선에서 탈락하자 그는 월드컵이 끝난 뒤 사임하였다. 이후 잉글랜드 <a href="/wiki/%EB%A0%88%EC%8A%A4%ED%84%B0_%EC%8B%9C%ED%8B%B0_FC" title="레스터 시티 FC">레스터 시티 FC</a> 감독을 거치고 <a href="/wiki/%EC%84%B8%EB%A5%B4%EC%A7%80%EC%9A%B0_%ED%8C%8C%EB%A6%AC%EC%95%84%EC%8A%A4" title="세르지우 파리아스">세르지우 파리아스</a> 후임으로 <a href="/wiki/%EA%B4%91%EC%A0%80%EC%9A%B0_%ED%91%B8%EB%A6%AC" title="광저우 푸리">광저우 푸리</a> 감독에 선임되었다. 하지만 우호적인 협상을 통해 사임하였다.</p>
