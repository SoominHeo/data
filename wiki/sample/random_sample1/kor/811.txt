<div class="mw-content-ltr" dir="ltr" id="mw-content-text" lang="ko"><script>function mfTempOpenSection(id){var block=document.getElementById("mf-section-"+id);block.className+=" open-block";block.previousSibling.className+=" open-block";}</script><p><b>BMW X3</b>는 <a href="/wiki/%EB%8F%85%EC%9D%BC" title="독일">독일</a>의 <a href="/wiki/%EC%9E%90%EB%8F%99%EC%B0%A8" title="자동차">자동차</a> 회사인 <a href="/wiki/BMW" title="BMW">BMW</a>가 생산/판매하는 준중형 크로스오버 <a href="/wiki/%EC%8A%A4%ED%8F%AC%EC%B8%A0_%EC%9C%A0%ED%8B%B8%EB%A6%AC%ED%8B%B0_%EC%9E%90%EB%8F%99%EC%B0%A8" title="스포츠 유틸리티 자동차">SUV</a>이다.</p>

<p><a href="/wiki/2003%EB%85%84_1%EC%9B%94" title="2003년 1월">2003년 1월</a>에 <a href="/wiki/BMW_3_%EC%8B%9C%EB%A6%AC%EC%A6%88" title="BMW 3 시리즈">3 시리즈</a> 플랫폼을 기반으로 한 x액티비티라는 <a href="/wiki/%EC%BD%98%EC%85%89%ED%8A%B8_%EC%B9%B4" title="콘셉트 카">컨셉트 카</a>가 출품되었고, 양산형은 같은 해 <a href="/wiki/9%EC%9B%94" title="9월">9월</a>에 <a href="/wiki/%ED%94%84%EB%9E%91%ED%81%AC%ED%91%B8%EB%A5%B4%ED%8A%B8_%EA%B5%AD%EC%A0%9C_%EB%AA%A8%ED%84%B0%EC%87%BC" title="프랑크푸르트 국제 모터쇼">프랑크푸르트 국제 모터쇼</a>에서 공개 후 출시되었다. 지난 <a href="/wiki/1999%EB%85%84" title="1999년">1999년</a>에 출시된 <a href="/wiki/BMW_X5" title="BMW X5">X5</a>가 성공을 거둔 뒤, 컴팩트 럭셔리 크로스 오버 SUV의 분야에도 진출하기 위해 기획되었다. <a href="/wiki/BMW" title="BMW">BMW</a>는 X 브랜드를 상표 등록했으며, SAV(Sports Activity Vehicle)라는 <a href="/wiki/%EC%8B%A0%EC%A1%B0%EC%96%B4" title="신조어">신조어</a>를 사용함으로써, 이미 출시된 <a href="/wiki/BMW_X5" title="BMW X5">X5</a>과 함께 "BMW가 발안한 완전히 새로운 카테고리 자동차입니다."라고 주장했다. <a href="/wiki/%EB%8C%80%ED%95%9C%EB%AF%BC%EA%B5%AD" title="대한민국">대한민국</a>에서는 <a href="/wiki/2004%EB%85%84_5%EC%9B%94" title="2004년 5월">2004년 5월</a>부터 공식 수입되었고, <a href="/wiki/2005%EB%85%84_11%EC%9B%94" title="2005년 11월">2005년 11월</a>에 M-Sport 패키지를 옵션으로 추가되었다. <a href="/wiki/2006%EB%85%84_9%EC%9B%94" title="2006년 9월">2006년 9월</a>에 페이스 리프트를 거쳐 전조등, 그릴, 후미등, 뒷범퍼, 내부 모습이 조금 바뀌었다. <a href="/wiki/2008%EB%85%84_11%EC%9B%94" title="2008년 11월">2008년 11월</a>에는 2.5si와 3.0si는 각각 xDrive25i, xDrive30i로 개명되었다. 1세대는 <a href="/wiki/%EC%98%A4%EC%8A%A4%ED%8A%B8%EB%A6%AC%EC%95%84" title="오스트리아">오스트리아</a>에서 생산되었다.</p>

<p><a href="/wiki/2010%EB%85%84" title="2010년">2010년</a> <a href="/wiki/7%EC%9B%94_14%EC%9D%BC" title="7월 14일">7월 14일</a>에 2세대(F25) X3가 발표되었고, 같은 해 <a href="/wiki/11%EC%9B%94_20%EC%9D%BC" title="11월 20일">11월 20일</a>에 출시되었다. 이미 하급 차종인 <a href="/wiki/BMW_X1" title="BMW X1">X1</a>이 출시되면서 1세대(E83)에 비해 1세대(E53) <a href="/wiki/BMW_X5" title="BMW X5">X5</a>만큼 커졌다. 또한 주요 생산지는 <a href="/wiki/%EB%AF%B8%EA%B5%AD" title="미국">미국</a>에 있는 <a href="/wiki/%EB%82%A8%EB%B6%80_%EC%BA%98%EB%A6%AC%ED%8F%AC%EB%8B%88%EC%95%84" title="남부 캘리포니아">남부 캘리포니아</a>의 <a class="new" href="/w/index.php?title=%EA%B7%B8%EB%A6%AC%EC%96%B4&amp;action=edit&amp;redlink=1" title="그리어 (없는 문서)">그리어</a>로 옮겨졌다. 초기에는 xDrive28i과 xDrive35i, xDrive20d만 있었지만, <a href="/wiki/2011%EB%85%84" title="2011년">2011년</a>에 xDrive20i, xDrive30d, xDrive35d가 추가되면서 선택의 폭을 넓혔다. <a href="/wiki/2012%EB%85%84" title="2012년">2012년</a> <a href="/wiki/5%EC%9B%94_22%EC%9D%BC" title="5월 22일">5월 22일</a>에 xDrive28i의 엔진이 직렬 6기통 3.0L 자연 흡기 방식에서 직렬 4기통 2.0L 직분사 트윈 스크롤 터보 엔진으로 변경되었다. 또한 모든 등급에 ECO PRO 모드와 퍼포먼스 컨트롤이 추가되었다. 그 해 <a href="/wiki/9%EC%9B%94_24%EC%9D%BC" title="9월 24일">9월 24일</a>에는 클린 디젤 엔진을 탑재한 xDrive20d 블루 퍼포먼스가 추가되었다. <a href="/wiki/2014%EB%85%84_3%EC%9B%94" title="2014년 3월">2014년 3월</a> 제네바 모터쇼에서 페이스 리프트 모델이 발표되었고, 같은 해 <a href="/wiki/5%EC%9B%94" title="5월">5월</a>에 페이스 리프트를 거쳐 헤드라이트에 앞트임이 추가되었다.</p>
