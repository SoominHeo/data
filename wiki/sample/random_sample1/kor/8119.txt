<p><b>스웨덴 왕비 실비아</b> (<span style="font-size: smaller;"><a href="/wiki/%EC%8A%A4%EC%9B%A8%EB%8D%B4%EC%96%B4" title="스웨덴어">스웨덴어</a>:</span> <span lang="sv" xml:lang="sv">Drottning Silvia av Sverige</span>, <a href="/wiki/1943%EB%85%84" title="1943년">1943년</a> <a href="/wiki/12%EC%9B%94_23%EC%9D%BC" title="12월 23일">12월 23일</a>~)는 <a class="mw-redirect" href="/wiki/%EC%8A%A4%EC%9B%A8%EB%8D%B4%EC%9D%98_%EC%B9%BC_16%EC%84%B8_%EA%B5%AC%EC%8A%A4%ED%83%80%ED%94%84" title="스웨덴의 칼 16세 구스타프">스웨덴의 칼 16세 구스타프</a>의 아내로, 현재 <a class="new" href="/w/index.php?title=%EC%8A%A4%EC%9B%A8%EB%8D%B4%EC%9D%98_%EA%B5%B0%EC%A3%BC_%EB%B0%B0%EC%9A%B0%EC%9E%90&amp;action=edit&amp;redlink=1" title="스웨덴의 군주 배우자 (없는 문서)">스웨덴 왕비</a>이다.</p>
<p><a href="/wiki/%EC%B9%BC_16%EC%84%B8_%EA%B5%AC%EC%8A%A4%ED%83%80%ED%94%84" title="칼 16세 구스타프">칼 16세 구스타프</a>와는 <a href="/wiki/1972%EB%85%84_%ED%95%98%EA%B3%84_%EC%98%AC%EB%A6%BC%ED%94%BD" title="1972년 하계 올림픽">1972년 뮌헨 하계 올림픽</a> 때 처음 만나게 되었고, 연인으로 발전하였고 결혼하게 되었다.</p>
<h2><span class="mw-headline" id=".EC.83.9D.EC.95.A0">생애</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%8A%A4%EC%9B%A8%EB%8D%B4_%EC%99%95%EB%B9%84_%EC%8B%A4%EB%B9%84%EC%95%84&amp;action=edit&amp;section=1" title="부분 편집: 생애">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<h3><span class="mw-headline" id=".EA.B2.B0.ED.98.BC_.EC.9D.B4.EC.A0.84">결혼 이전</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%8A%A4%EC%9B%A8%EB%8D%B4_%EC%99%95%EB%B9%84_%EC%8B%A4%EB%B9%84%EC%95%84&amp;action=edit&amp;section=2" title="부분 편집: 결혼 이전">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>그녀는 <a href="/wiki/%ED%95%98%EC%9D%B4%EB%8D%B8%EB%B2%A0%EB%A5%B4%ED%81%AC" title="하이델베르크">하이델베르크</a>에서 독일인 <a class="new" href="/w/index.php?title=%EB%B0%9C%ED%84%B0_%EC%A2%80%EB%A9%80%EB%9D%BC%ED%8A%B8&amp;action=edit&amp;redlink=1" title="발터 좀멀라트 (없는 문서)">발터 좀멀라트</a>와 브라질인인 <a class="new" href="/w/index.php?title=%EC%95%8C%EB%A6%AC%EC%B2%B4_%EC%A2%80%EB%A9%80%EB%9D%BC%ED%8A%B8&amp;action=edit&amp;redlink=1" title="알리체 좀멀라트 (없는 문서)">알리체 좀멀라트</a> 사이에서 막내로 태어났다. 그녀는 7살이 되던 해에 가족들과 함께 <a href="/wiki/%EB%B8%8C%EB%9D%BC%EC%A7%88" title="브라질">브라질</a> <a href="/wiki/%EC%83%81%ED%8C%8C%EC%9A%B8%EB%A3%A8" title="상파울루">상파울루</a>로 건너가서 생활하게 되었으나, <a href="/wiki/1957%EB%85%84" title="1957년">1957년</a> 아버지가 <a href="/wiki/%EC%84%9C%EB%8F%85" title="서독">서독</a>으로 돌아오면서, 다시 독일에서 생활하게 된다. 이후 그녀는 외국어에 소질을 보여 <a href="/wiki/%EC%8A%A4%EC%9B%A8%EB%8D%B4%EC%96%B4" title="스웨덴어">스웨덴어</a>, <a href="/wiki/%EB%8F%85%EC%9D%BC%EC%96%B4" title="독일어">독일어</a>, <a href="/wiki/%ED%8F%AC%EB%A5%B4%ED%88%AC%EA%B0%88%EC%96%B4" title="포르투갈어">포르투갈어</a>, <a href="/wiki/%EC%98%81%EC%96%B4" title="영어">영어</a>, <a href="/wiki/%ED%94%84%EB%9E%91%EC%8A%A4%EC%96%B4" title="프랑스어">프랑스어</a>, <a href="/wiki/%EC%8A%A4%ED%8E%98%EC%9D%B8%EC%96%B4" title="스페인어">스페인어</a>에 능숙하게 되었고, <a href="/wiki/%EB%8F%85%EC%9D%BC" title="독일">독일</a>에 위치한 <a href="/wiki/%EC%95%84%EB%A5%B4%ED%97%A8%ED%8B%B0%EB%82%98" title="아르헨티나">아르헨티나</a> 대사관에서 일하게 되었다.</p>

<p>그녀는 후에 <a class="mw-disambig" href="/wiki/%EC%9D%B8%EC%8A%A4%EB%B8%8C%EB%A3%A8%ED%81%AC_%EC%98%AC%EB%A6%BC%ED%94%BD" title="인스브루크 올림픽">인스브루크 올림픽</a>이나 <a class="mw-redirect" href="/wiki/%EB%AE%8C%ED%97%A8_%EC%98%AC%EB%A6%BC%ED%94%BD" title="뮌헨 올림픽">뮌헨 올림픽</a>등에서 통역을 하기도 하였는데, <a class="mw-redirect" href="/wiki/%EB%AE%8C%ED%97%A8_%EC%98%AC%EB%A6%BC%ED%94%BD" title="뮌헨 올림픽">뮌헨 올림픽</a> 때 실비아는 <a href="/wiki/%EC%B9%BC_16%EC%84%B8_%EA%B5%AC%EC%8A%A4%ED%83%80%ED%94%84" title="칼 16세 구스타프">칼 구스타프 왕세자</a>를 만나는데, 그와 사랑하게 된다. 스웨덴에서는 엄격한 귀천상혼제가 적용되고 있어, 왕자들은 신분이 높은 여성과 결혼하지 않으면 <a href="/wiki/%EC%8A%A4%EC%9B%A8%EB%8D%B4%EC%9D%98_%EC%99%95%EC%9C%84_%EA%B3%84%EC%8A%B9_%EC%88%9C%EC%9C%84" title="스웨덴의 왕위 계승 순위">왕위계승권</a>과 스웨덴의 왕자 칭호를 박탈당하는데, 이로 인하여 그때 스웨덴에는 <a href="/wiki/%EC%B9%BC_16%EC%84%B8_%EA%B5%AC%EC%8A%A4%ED%83%80%ED%94%84" title="칼 16세 구스타프">칼 구스타프 왕세자</a>와 그의 삼촌인 <a class="new" href="/w/index.php?title=%ED%95%A0%EB%9E%80%EB%93%9C_%EA%B3%B5%EC%9E%91_%EB%B2%A0%EB%A5%B4%ED%8B%B8&amp;action=edit&amp;redlink=1" title="할란드 공작 베르틸 (없는 문서)">할란드 공작 베르틸</a> 이외에는 모두 왕자 칭호를 박탈당하거나 사망하여서 왕위계승자가 남아있지 않았다. <a href="/wiki/%EA%B5%AC%EC%8A%A4%ED%83%80%ED%94%84_6%EC%84%B8_%EC%95%84%EB%8F%8C%ED%94%84" title="구스타프 6세 아돌프">구스타프 6세 아돌프</a>의 사망 이후 왕위를 계승하게 된 <a href="/wiki/%EC%B9%BC_16%EC%84%B8_%EA%B5%AC%EC%8A%A4%ED%83%80%ED%94%84" title="칼 16세 구스타프">칼 16세 구스타프</a>의 그녀가 아니면 평생 결혼하지 않겠다고 고집을 부렸고, 결국 의회의 승인을 얻어 실비아와 칼 16세 구스타프는 <a href="/wiki/1976%EB%85%84" title="1976년">1976년</a> <a href="/wiki/6%EC%9B%94_19%EC%9D%BC" title="6월 19일">6월 19일</a> 결혼하게 되었으며, 현재 둘 사이에는 두 명의 공주(<a href="/wiki/%EC%8A%A4%EC%9B%A8%EB%8D%B4_%EC%99%95%EC%84%B8%EB%85%80_%EB%B9%85%ED%86%A0%EB%A6%AC%EC%95%84" title="스웨덴 왕세녀 빅토리아">빅토리아</a>, <a class="new" href="/w/index.php?title=%ED%97%AC%EC%8B%B1%EB%9E%80%EB%93%9C%EC%99%80_%EC%97%90%EC%8A%A4%ED%8A%B8%EB%A6%AD%EB%9E%80%EB%93%9C%EC%9D%98_%EC%97%AC%EA%B3%B5%EC%9E%91_%EB%A7%88%EB%8D%B8%EB%A0%88%EC%9D%B4%EB%84%A4&amp;action=edit&amp;redlink=1" title="헬싱란드와 에스트릭란드의 여공작 마델레이네 (없는 문서)">마델레이네</a>)와 한 명의 왕자(<a class="new" href="/w/index.php?title=%EB%B2%A0%EB%A6%84%EB%9E%80%EB%93%9C_%EA%B3%B5%EC%9E%91_%EC%B9%BC_%ED%95%84%EB%A6%BD&amp;action=edit&amp;redlink=1" title="베름란드 공작 칼 필립 (없는 문서)">칼 필립</a>)을 두고 있다.</p>
