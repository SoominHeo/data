<h1 class="firstHeading" id="firstHeading" lang="ko">BMW X6</h1>
<div class="mw-body-content" id="bodyContent">
<div id="siteSub">위키백과, 우리 모두의 백과사전.</div>
<div id="contentSub"></div>
<div class="mw-jump" id="jump-to-nav">
					이동:					<a href="#mw-head">둘러보기</a>, 					<a href="#p-search">검색</a>
</div>
<div class="mw-content-ltr" dir="ltr" id="mw-content-text" lang="ko"><script>function mfTempOpenSection(id){var block=document.getElementById("mf-section-"+id);block.className+=" open-block";block.previousSibling.className+=" open-block";}</script><p><b>BMW X6</b>는 <a href="/wiki/BMW" title="BMW">BMW</a>가 생산 및 판매하는 대형 크로스오버 <a href="/wiki/%EC%8A%A4%ED%8F%AC%EC%B8%A0_%EC%9C%A0%ED%8B%B8%EB%A6%AC%ED%8B%B0_%EC%9E%90%EB%8F%99%EC%B0%A8" title="스포츠 유틸리티 자동차">SUV</a>이다. 앞모습은 SUV의 거대함이 있지만 뒷모습은 쿠페처럼 전고를 점점 낮게 설계된 것이 특징이다. SUV의 중후함과 쿠페의 날렵한 디자인을 적절히 타협했다.</p>

<p><a href="/wiki/2007%EB%85%84" title="2007년">2007년</a>에 <a href="/wiki/%ED%94%84%EB%9E%91%ED%81%AC%ED%91%B8%EB%A5%B4%ED%8A%B8_%EA%B5%AD%EC%A0%9C_%EB%AA%A8%ED%84%B0%EC%87%BC" title="프랑크푸르트 국제 모터쇼">프랑크푸르트 국제 모터쇼</a>에서 <a class="mw-redirect" href="/wiki/%EC%BB%A8%EC%85%89%ED%8A%B8%EC%B9%B4" title="컨셉트카">컨셉트카</a>인 BMW 컨셉트 X6가 공개되었고, 양산형은 같은 해 <a href="/wiki/12%EC%9B%94_3%EC%9D%BC" title="12월 3일">12월 3일</a>에 생산이 시작되었다. 이후 <a href="/wiki/2008%EB%85%84_1%EC%9B%94" title="2008년 1월">2008년 1월</a>에 <a href="/wiki/%EB%B6%81%EB%AF%B8_%EA%B5%AD%EC%A0%9C_%EC%98%A4%ED%86%A0%EC%87%BC" title="북미 국제 오토쇼">북미 국제 오토쇼</a>를 통해 공식적으로 공개되었고, 같은 해 <a href="/wiki/4%EC%9B%94" title="4월">4월</a>에 판매가 시작되었다. 주행 상황이나 차량 상황에 따라 구동력을 임기응변으로 좌우의 리어 휠간에 배분하는 <a href="/wiki/BMW" title="BMW">BMW</a>의 신기술인 '다이나믹 퍼포먼스 컨트롤(Dynamic Performance Control)'을 처음 장착했다. <a href="/wiki/2009%EB%85%84" title="2009년">2009년</a>에는 고성능 모델인 X6 M이 선보였으며, <a href="/wiki/2010%EB%85%84_4%EC%9B%94" title="2010년 4월">2010년 4월</a>에 <a href="/wiki/%ED%95%98%EC%9D%B4%EB%B8%8C%EB%A6%AC%EB%93%9C_(%EB%8F%99%EB%A0%A5)" title="하이브리드 (동력)">하이브리드</a> 사양인 액티브하이브리드 X6도 선보였다. 출시 당초에 승차인원은 4명이었으나, 반발이 거세지자 <a href="/wiki/2011%EB%85%84" title="2011년">2011년</a> <a href="/wiki/5%EC%9B%94_26%EC%9D%BC" title="5월 26일">5월 26일</a>에 5명으로 바뀌었다. <a href="/wiki/2012%EB%85%84_7%EC%9B%94" title="2012년 7월">2012년 7월</a>에 마이너 체인지를 감행했다.</p>
