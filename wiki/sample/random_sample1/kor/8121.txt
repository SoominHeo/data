<p><b>스웨덴 중앙은행</b>(<span style="font-size: smaller;"><a href="/wiki/%EC%8A%A4%EC%9B%A8%EB%8D%B4%EC%96%B4" title="스웨덴어">스웨덴어</a>:</span> <span lang="sv" xml:lang="sv">Sveriges Riksbank</span>)은 <a href="/wiki/1668%EB%85%84" title="1668년">1668년</a>에 창설된 <a href="/wiki/%EC%8A%A4%EC%9B%A8%EB%8D%B4" title="스웨덴">스웨덴</a>의 <a href="/wiki/%EC%A4%91%EC%95%99%EC%9D%80%ED%96%89" title="중앙은행">중앙은행</a>으로, 세계에서 가장 오래된 중앙은행이다. 흔히 <b>스웨덴 국립은행</b>(<span lang="en" xml:lang="en">Swedish National Bank</span>) 또는 <b>스웨덴 은행</b>(<span lang="en" xml:lang="en">Bank of Sweden</span>)이라고도 한다. 본부는 <a href="/wiki/%EC%8A%A4%ED%86%A1%ED%99%80%EB%A6%84" title="스톡홀름">스톡홀름</a>에 있으며 <a class="mw-redirect" href="/wiki/%EC%9C%A0%EB%9F%BD_%EC%A4%91%EC%95%99%EC%9D%80%ED%96%89_%EC%B2%B4%EC%A0%9C" title="유럽 중앙은행 체제">유럽 중앙은행 체제</a>의 회원으로 활동하고 있다. 스웨덴 중앙은행 소속의 조폐국은 <a href="/wiki/%EC%8A%A4%EC%9B%A8%EB%8D%B4_%ED%81%AC%EB%A1%9C%EB%82%98" title="스웨덴 크로나">스웨덴 크로나</a>의 동전과 지폐를 발행하는 업무를 담당한다.</p>
<p>스웨덴 중앙은행의 전신은 <a href="/wiki/1656%EB%85%84" title="1656년">1656년</a> 요한 팔름스트루흐가 설립한 스톡홀름 은행(<span lang="en" xml:lang="en">Stockholms Banco</span>)으로, 스웨덴 국왕이 관할하던 은행이었다. 이 은행은 <a href="/wiki/1661%EB%85%84" title="1661년">1661년</a> 유럽에서 처음으로 지폐를 발행하였다.</p>
<p><a href="/wiki/1668%EB%85%84" title="1668년">1668년</a> <a href="/wiki/9%EC%9B%94_17%EC%9D%BC" title="9월 17일">9월 17일</a> 스웨덴 의회는 중앙은행을 의회 관할로 하는 내용의 법률을 제정하는 한편 스톡홀름 은행의 업무를 스웨덴 왕립 재정 은행(<span lang="sv" xml:lang="sv">Riksens Ständers Bank</span>)으로 이관하였다. <a href="/wiki/1866%EB%85%84" title="1866년">1866년</a> 스웨덴 중앙은행(<span lang="sv" xml:lang="sv">Sveriges Riksbank</span>)으로 명칭을 변경했으며 <a href="/wiki/1897%EB%85%84" title="1897년">1897년</a> 스웨덴 의회가 스웨덴 중앙은행법을 제정하면서 스웨덴 국립은행은 중앙은행으로서의 역할을 맡게 되었다.</p>
<p>스웨덴 중앙은행은 <a href="/wiki/1968%EB%85%84" title="1968년">1968년</a> 스웨덴 중앙은행 창립 300주년을 맞아 <a href="/wiki/%EB%85%B8%EB%B2%A8_%EA%B2%BD%EC%A0%9C%ED%95%99%EC%83%81" title="노벨 경제학상">알프레드 노벨을 기념하는 스웨덴 중앙은행 경제학상</a>을 신설했다. 노벨 경제학상은 <a href="/wiki/%EC%95%8C%ED%94%84%EB%A0%88%EB%93%9C_%EB%85%B8%EB%B2%A8" title="알프레드 노벨">알프레드 노벨</a>이 사망한 날인 매년 <a href="/wiki/12%EC%9B%94_10%EC%9D%BC" title="12월 10일">12월 10일</a> <a href="/wiki/%EC%8A%A4%ED%86%A1%ED%99%80%EB%A6%84" title="스톡홀름">스톡홀름</a>에서 열린다.</p>
