<p><b>스웨덴의 마르가레타</b>(Margaretha af Sverige, <a href="/wiki/1899%EB%85%84" title="1899년">1899년</a> <a href="/wiki/6%EC%9B%94_25%EC%9D%BC" title="6월 25일">6월 25일</a> ~ <a href="/wiki/1977%EB%85%84" title="1977년">1977년</a> <a href="/wiki/1%EC%9B%94_4%EC%9D%BC" title="1월 4일">1월 4일</a>)는 <a href="/wiki/%EC%8A%A4%EC%9B%A8%EB%8D%B4" title="스웨덴">스웨덴</a>의 왕족으로 <a href="/wiki/%EB%8D%B4%EB%A7%88%ED%81%AC" title="덴마크">덴마크</a> 왕자 <a class="new" href="/w/index.php?title=%EB%8D%B4%EB%A7%88%ED%81%AC_%EC%99%95%EC%9E%90_%EC%95%85%EC%85%80&amp;action=edit&amp;redlink=1" title="덴마크 왕자 악셀 (없는 문서)">악셀</a>의 아내이다. 본명은 <b>마르가레타 소피아 로비사 잉에보리</b>(Margaretha Sofia Lovisa Ingeborg)이다. <a href="/wiki/%EC%98%A4%EC%8A%A4%EC%B9%B4%EB%A5%B4_2%EC%84%B8" title="오스카르 2세">오스카르 2세</a>의 손녀로, 그 아들 베스테르예틀란드 공 칼과 덴마크 공주 <a class="new" href="/w/index.php?title=%EB%8D%B4%EB%A7%88%ED%81%AC%EC%9D%98_%EC%9E%89%EC%97%90%EB%B3%B4%EB%A6%AC&amp;action=edit&amp;redlink=1" title="덴마크의 잉에보리 (없는 문서)">잉에보리</a> 사이에서 장녀로 태어났다. 동생들로는 노르웨이 왕태자비 <a class="new" href="/w/index.php?title=%EC%8A%A4%EC%9B%A8%EB%8D%B4%EC%9D%98_%EB%A7%88%ED%83%80&amp;action=edit&amp;redlink=1" title="스웨덴의 마타 (없는 문서)">마타</a>, 벨기에의 왕비 <a href="/wiki/%EC%8A%A4%EC%9B%A8%EB%8D%B4%EC%9D%98_%EC%95%84%EC%8A%A4%ED%8A%B8%EB%A6%AC%EB%93%9C" title="스웨덴의 아스트리드">아스트리드</a> 등이 있다.</p>
<p><a href="/wiki/1919%EB%85%84" title="1919년">1919년</a> <a href="/wiki/5%EC%9B%94_22%EC%9D%BC" title="5월 22일">5월 22일</a> 어머니의 사촌동생인 덴마크의 악셀 왕자와 결혼했다. 부부 사이는 원만했고 두 아들을 두었다. <a href="/wiki/1964%EB%85%84" title="1964년">1964년</a> 남편과 사별했고 그녀도 <a href="/wiki/1977%EB%85%84" title="1977년">1977년</a> 덴마크에서 죽었다.</p>
