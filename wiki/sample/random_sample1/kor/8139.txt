<p><b>스즈키 아이리</b> (<span style="font-size: smaller;"><a href="/wiki/%EC%9D%BC%EB%B3%B8%EC%96%B4" title="일본어">일본어</a>:</span> <span lang="ja"><span lang="ja"><ruby><rb>鈴木愛理</rb> <rp>(</rp><rt>すずき あいり</rt><rp>)</rp></ruby></span></span>, <a href="/wiki/1994%EB%85%84" title="1994년">1994년</a> <a href="/wiki/4%EC%9B%94_12%EC%9D%BC" title="4월 12일">4월 12일</a> ~ ) 는 <a href="/wiki/%EC%9D%BC%EB%B3%B8" title="일본">일본</a> <a href="/wiki/%EC%A7%80%EB%B0%94_%ED%98%84" title="지바 현">지바 현</a> 출신의 <a href="/wiki/%EC%97%AC%EC%84%B1" title="여성">여성</a> <a class="mw-redirect" href="/wiki/%EC%95%84%EC%9D%B4%EB%8F%8C_%EA%B0%80%EC%88%98" title="아이돌 가수">아이돌</a> <a href="/wiki/%EA%B0%80%EC%88%98" title="가수">가수</a>이다. <a href="/wiki/%E2%84%83-ute" title="℃-ute">큐트</a>의 멤버 이자 <a href="/wiki/Buono!" title="Buono!">Buono!</a>의 멤버 이다. 연예사무소 <a href="/wiki/%EC%97%85%ED%94%84%EB%9F%B0%ED%8A%B8_%ED%94%84%EB%A1%9C%EB%AA%A8%EC%85%98" title="업프런트 프로모션">업프런트 프로모션</a> 소속. 애칭은 <b>오스즈</b>, <b>아이링</b>, <b><a href="/wiki/%EA%B0%93%ED%8C%8C" title="갓파">갓파</a></b> 등이 있다.</p>

<h2><span class="mw-headline" id=".ED.8A.B9.EC.A7.95">특징</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%8A%A4%EC%A6%88%ED%82%A4_%EC%95%84%EC%9D%B4%EB%A6%AC&amp;action=edit&amp;section=1" title="부분 편집: 특징">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<ul>
<li>아버지는 프로 골퍼(<a class="new" href="/w/index.php?title=%EC%8A%A4%EC%A6%88%ED%82%A4_%ED%86%A0%EC%98%A4%EB%A3%A8&amp;action=edit&amp;redlink=1" title="스즈키 토오루 (없는 문서)">스즈키 토오루</a>)이며 어머니는 전직 국가대표 골프 선수(<a class="new" href="/w/index.php?title=%EB%A7%88%EB%A3%A8%EC%95%BC_%EC%BF%84%EC%BD%94&amp;action=edit&amp;redlink=1" title="마루야 쿄코 (없는 문서)">마루야 쿄코</a>)였다. MC 토크에서 <a href="/wiki/%EA%B3%A8%ED%94%84" title="골프">골프</a>에 관련된 화제가 자주 등장한다.</li>
<li>키우고 있는 강아지는 총 6마리이다.</li>
<li>스스로 말하길 “성격은 보기보다 느긋한 편”이라고 한다.</li>
<li>취미는 섬세한 작업을 하는 것, 물건을 만드는 것, 그림을 그리는 것 등이 있다.</li>
<li>평균 수면 시간은 3시간 정도로 기상 시간은 새벽 5시 30분 정도라고 한다(2012년 고교 3학년 재학 시점).</li>
<li>좋아하는 동물은 강아지이다.
<ul>
<li>2009년부터 프로필 상 '인간을 가장 좋아한다'고 표현하고 있다.</li>
</ul>
</li>
<li>좋아하는 계절은 봄이다.</li>
