<h2><span class="mw-headline" id=".EA.B5.90.ED.9A.8C.EC.82.AC">교회사</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%8A%A4%EC%BD%94%ED%8B%80%EB%9E%9C%EB%93%9C_%EC%84%B1%EA%B3%B5%ED%9A%8C&amp;action=edit&amp;section=1" title="부분 편집: 교회사">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p><a href="/wiki/%EC%9E%89%EA%B8%80%EB%9E%9C%EB%93%9C_%EC%99%95%EA%B5%AD" title="잉글랜드 왕국">잉글랜드 왕국</a> <a href="/wiki/%EC%A2%85%EA%B5%90_%EA%B0%9C%ED%98%81" title="종교 개혁">종교 개혁</a> 이래로 <a href="/wiki/%EC%9E%89%EA%B8%80%EB%9E%9C%EB%93%9C_%EC%84%B1%EA%B3%B5%ED%9A%8C" title="잉글랜드 성공회">잉글랜드 성공회</a>에 속해 있었으나, <a class="mw-redirect" href="/wiki/%EC%9E%89%EA%B8%80%EB%9E%9C%EB%93%9C%EC%9D%98_%EC%9C%8C%EB%A6%AC%EC%97%84_3%EC%84%B8" title="잉글랜드의 윌리엄 3세">윌리엄 3세</a>가 <a href="/wiki/%EB%AA%85%EC%98%88%ED%98%81%EB%AA%85" title="명예혁명">반정</a>으로 즉위한 이후 몇 차례의 혼란과 시행착오 끝에 <a href="/wiki/18%EC%84%B8%EA%B8%B0" title="18세기">18세기</a> 말부터 자치적, 독자적인 지역 <a href="/wiki/%EC%84%B1%EA%B3%B5%ED%9A%8C" title="성공회">성공회</a> 교회, 곧 <a href="/wiki/%EA%B4%80%EA%B5%AC" title="관구">관구</a>가 되었다. <a href="/wiki/%EC%8A%A4%EC%BD%94%ED%8B%80%EB%9E%9C%EB%93%9C_%EA%B5%90%ED%9A%8C" title="스코틀랜드 교회">스코틀랜드 교회</a>에 이어 스코틀랜드에서 두 번째로 큰 <a href="/wiki/%EA%B5%90%ED%9A%8C" title="교회">교회</a>이다. 참고로 <a href="/wiki/%EB%AF%B8%EA%B5%AD_%EC%84%B1%EA%B3%B5%ED%9A%8C" title="미국 성공회">미국 성공회</a>를 영어로 Episcopal Church라고 하는 것은 스코틀랜드 성공회의 영향이다. <a class="mw-redirect" href="/wiki/%EB%AF%B8%EA%B5%AD_%EB%8F%85%EB%A6%BD%EC%A0%84%EC%9F%81" title="미국 독립전쟁">미국 독립전쟁</a>으로 미국이 <a href="/wiki/%EC%98%81%EA%B5%AD" title="영국">영국</a>에서 독립하자 <a href="/wiki/%EB%AF%B8%EA%B5%AD_%EC%84%B1%EA%B3%B5%ED%9A%8C" title="미국 성공회">미국 성공회</a>는 <a href="/wiki/%EA%B5%90%ED%9A%8C" title="교회">교회</a>를 지도할 <a href="/wiki/%EC%A3%BC%EA%B5%90" title="주교">주교</a>가 필요했는데,<a class="mw-disambig" href="/wiki/%EC%98%81%EA%B5%AD_%EC%84%B1%EA%B3%B5%ED%9A%8C" title="영국 성공회">영국 성공회</a>에서 주교서품을 주지 않았으므로 스코틀랜드 성공회의 주교로부터 <a class="mw-redirect" href="/wiki/%EC%84%9C%ED%92%88" title="서품">서품</a>을 받아야 했다. 그래서 스코틀랜드 성공회처럼 스스로를 <a href="/wiki/%EC%A3%BC%EA%B5%90" title="주교">주교</a>제 교회(Episcopal Church)라고 부르게 된 것이다.</p>
