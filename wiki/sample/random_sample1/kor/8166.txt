<h1 class="firstHeading" id="firstHeading" lang="ko">스콜라스틱</h1>
<div class="mw-body-content" id="bodyContent">
<div id="siteSub">위키백과, 우리 모두의 백과사전.</div>
<div id="contentSub"></div>
<div class="mw-jump" id="jump-to-nav">
					이동:					<a href="#mw-head">둘러보기</a>, 					<a href="#p-search">검색</a>
</div>
<div class="mw-content-ltr" dir="ltr" id="mw-content-text" lang="ko"><script>function mfTempOpenSection(id){var block=document.getElementById("mf-section-"+id);block.className+=" open-block";block.previousSibling.className+=" open-block";}</script><p><b>스콜라스틱</b>(Scholastic Corporation, <span style="font-size:small"><a href="/wiki/%EB%82%98%EC%8A%A4%EB%8B%A5" title="나스닥">나스닥</a>: <a class="external text" href="http://quotes.nasdaq.com/asp/SummaryQuote.asp?symbol=SCHL&amp;selected=SCHL" rel="nofollow"><b>SCHL</b></a></span>)은 <a href="/wiki/%EB%AF%B8%EA%B5%AD" title="미국">미국</a> <a href="/wiki/%EB%89%B4%EC%9A%95_%EC%A3%BC" title="뉴욕 주">뉴욕 주</a> <a href="/wiki/%EB%89%B4%EC%9A%95" title="뉴욕">뉴욕</a>에 있는 대형 출판사이다.</p>
<p><a href="/wiki/1925%EB%85%84" title="1925년">1925년</a> M. R. "로비" 로빈슨이 <a href="/wiki/%ED%8E%9C%EC%8B%A4%EB%B2%A0%EC%9D%B4%EB%8B%88%EC%95%84_%EC%A3%BC" title="펜실베이니아 주">펜실베이니아 주</a> 피츠버그에서 설립하였으며, <a href="/wiki/1957%EB%85%84" title="1957년">1957년</a>에는 <a href="/wiki/%EC%BA%90%EB%82%98%EB%8B%A4" title="캐나다">캐나다</a> <a href="/wiki/%ED%86%A0%EB%A1%A0%ED%86%A0" title="토론토">토론토</a>에 해외 자회사를 처음 세웠다. 현재 미국 뿐만 아니라, <a href="/wiki/%ED%99%8D%EC%BD%A9" title="홍콩">홍콩</a>, <a class="mw-disambig" href="/wiki/%EC%98%A4%ED%81%B4%EB%9E%9C%EB%93%9C" title="오클랜드">오클랜드</a>, <a href="/wiki/%EB%89%B4%EB%8D%B8%EB%A6%AC" title="뉴델리">뉴델리</a>, <a class="mw-redirect" href="/wiki/%EB%B6%80%EC%97%90%EB%85%B8%EC%8A%A4_%EC%95%84%EC%9D%B4%EB%A0%88%EC%8A%A4" title="부에노스 아이레스">부에노스 아이레스</a>, <a class="mw-redirect" href="/wiki/%EB%A9%95%EC%8B%9C%EC%BD%94_%EC%8B%9C%ED%8B%B0" title="멕시코 시티">멕시코 시티</a>, <a href="/wiki/%EB%9F%B0%EB%8D%98" title="런던">런던</a>, <a href="/wiki/%EC%8B%9C%EB%93%9C%EB%8B%88" title="시드니">시드니</a>에도 지사가 있다.</p>
<p>스콜라스틱은 주로 교육용 서적을 출판한다. 많이 알려진 책 중에서는 The Secrets of Droon, Junie B. Jones, Magic Tree House (<a class="new" href="/w/index.php?title=%EB%A7%88%EB%B2%95%EC%9D%98_%EC%8B%9C%EA%B0%84_%EC%97%AC%ED%96%89&amp;action=edit&amp;redlink=1" title="마법의 시간 여행 (없는 문서)">마법의 시간 여행</a>), Harry Potter (<a class="mw-redirect" href="/wiki/%ED%95%B4%EB%A6%AC%ED%8F%AC%ED%84%B0" title="해리포터">해리포터</a>) 등등이 있다.</p>
