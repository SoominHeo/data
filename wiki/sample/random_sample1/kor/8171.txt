<h2><span class="mw-headline" id=".EB.A9.94.EC.9D.B4.EC.A0.80_.EB.A6.AC.EA.B7.B8_.EC.84.A0.EC.88.98_.EC.8B.9C.EC.A0.88">메이저 리그 선수 시절</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%8A%A4%EC%BD%A7_%ED%94%84%EB%A1%9D%ED%84%B0&amp;action=edit&amp;section=1" title="부분 편집: 메이저 리그 선수 시절">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p><a href="/wiki/1995%EB%85%84" title="1995년">1995년</a> 17라운드로 <a href="/wiki/%EB%89%B4%EC%9A%95_%EB%A9%94%EC%B8%A0" title="뉴욕 메츠">뉴욕 메츠</a>에 지명되었으나, 입단을 거부하고 플로리다주립대학교(Florida State Uniersity)에 진학한다. 이후 <a href="/wiki/1998%EB%85%84" title="1998년">1998년</a> 5라운드로 <a href="/wiki/%EB%A1%9C%EC%8A%A4%EC%95%A4%EC%A0%A4%EB%A0%88%EC%8A%A4_%EB%8B%A4%EC%A0%80%EC%8A%A4" title="로스앤젤레스 다저스">로스앤젤레스 다저스</a>에 입단한다. <a href="/wiki/2003%EB%85%84" title="2003년">2003년</a> <a href="/wiki/7%EC%9B%94_31%EC%9D%BC" title="7월 31일">7월 31일</a> 버버 크로스비를 상대로 트레이드되어 <a href="/wiki/%EB%89%B4%EC%9A%95_%EC%96%91%ED%82%A4%EC%8A%A4" title="뉴욕 양키스">뉴욕 양키스</a>로 이적했다. <a href="/wiki/2006%EB%85%84" title="2006년">2006년</a> 28홀드 기록을 세웠다.</p>
<p><a href="/wiki/2011%EB%85%84" title="2011년">2011년</a> 시즌 이후 FA 자격을 획득하였다.</p>
<h2><span class="mw-headline" id=".ED.95.9C.EA.B5.AD_.ED.94.84.EB.A1.9C_.EC.95.BC.EA.B5.AC_.EC.8B.9C.EC.A0.88">한국 프로 야구 시절</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%8A%A4%EC%BD%A7_%ED%94%84%EB%A1%9D%ED%84%B0&amp;action=edit&amp;section=2" title="부분 편집: 한국 프로 야구 시절">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>2012 시즌 <a href="/wiki/%EB%91%90%EC%82%B0_%EB%B2%A0%EC%96%B4%EC%8A%A4" title="두산 베어스">두산 베어스</a>가 계약금 5만 달러, 연봉 25만 달러에 계약해 <a href="/wiki/%EB%8C%80%ED%95%9C%EB%AF%BC%EA%B5%AD" title="대한민국">대한민국</a> 무대를 밟게 되었다. <a href="/wiki/%EA%B9%80%EC%A7%84%EC%9A%B1" title="김진욱">김진욱</a> 감독의 전력 구상에 따라 마무리 보직을 맡았다. 시즌 초반 엄청난 위력을 보여 주었으나 여름이 오면서 구위가 점점 떨어진 모습을 보여 주자를 루상에 내보내는 일이 많아졌다. <a href="/wiki/2012%EB%85%84" title="2012년">2012년</a> <a href="/wiki/6%EC%9B%94_12%EC%9D%BC" title="6월 12일">6월 12일</a> <a href="/wiki/%EB%A1%AF%EB%8D%B0_%EC%9E%90%EC%9D%B4%EC%96%B8%EC%B8%A0" title="롯데 자이언츠">롯데 자이언츠</a>전에서 11회 말에 마무리로 등판하였으나 한국야구에서 첫 블론세이브를 기록하고, 12회 말에는 <a href="/wiki/%EC%A1%B0%EC%84%B1%ED%99%98_(%EC%95%BC%EA%B5%AC_%EC%84%A0%EC%88%98)" title="조성환 (야구 선수)">조성환</a>에게 밀어내기 볼넷을 주며 한국야구 첫 패전투수가 된다. 그 해 55.1이닝을 던져 4승 4패 35세이브, ERA 1.79를 기록했고, 블론 세이브는 7개 기록했다.</p>
<p><a href="/wiki/2012%EB%85%84" title="2012년">2012년</a> <a href="/wiki/12%EC%9B%94_26%EC%9D%BC" title="12월 26일">12월 26일</a> <a href="/wiki/%EB%91%90%EC%82%B0_%EB%B2%A0%EC%96%B4%EC%8A%A4" title="두산 베어스">두산 베어스</a>는 선발 <a href="/wiki/%ED%88%AC%EC%88%98" title="투수">투수</a> 보강을 위해 프록터와 재계약하지 않기로 했다.<sup class="reference" id="cite_ref-1"><a href="#cite_note-1">[1]</a></sup></p>
<p>이후 <a href="/wiki/%EB%AF%B8%EA%B5%AD" title="미국">미국</a>으로 돌아가 <a href="/wiki/%EC%83%8C%ED%94%84%EB%9E%80%EC%8B%9C%EC%8A%A4%EC%BD%94_%EC%9E%90%EC%9D%B4%EC%96%B8%EC%B8%A0" title="샌프란시스코 자이언츠">샌프란시스코 자이언츠</a>와 마이너 계약을 맺고 재기를 노렸지만, 시범 경기에서 13이닝 동안 ERA 4.15를 기록한 뒤 <a href="/wiki/%EB%B3%BC%ED%8B%B0%EB%AA%A8%EC%96%B4_%EC%98%A4%EB%A6%AC%EC%98%AC%EC%8A%A4" title="볼티모어 오리올스">볼티모어 오리올스</a>에 현금 트레이드되었다.<sup class="reference" id="cite_ref-2"><a href="#cite_note-2">[2]</a></sup> 그러나 <a href="/wiki/%EB%B3%BC%ED%8B%B0%EB%AA%A8%EC%96%B4_%EC%98%A4%EB%A6%AC%EC%98%AC%EC%8A%A4" title="볼티모어 오리올스">볼티모어</a> 산하 <a class="mw-disambig" href="/wiki/%ED%8A%B8%EB%A6%AC%ED%94%8C_A" title="트리플 A">트리플 A</a> 팀인 노포크에서 중간계투로 등판하였지만 7.1이닝 1세이브, 평균자책 8.59에 그쳐 <a href="/wiki/2013%EB%85%84" title="2013년">2013년</a> <a href="/wiki/4%EC%9B%94_28%EC%9D%BC" title="4월 28일">4월 28일</a>에 방출당했고, 현역에서 은퇴하였다.<sup class="reference" id="cite_ref-3"><a href="#cite_note-3">[3]</a></sup></p>
