<h1 class="firstHeading" id="firstHeading" lang="ko">스쿨 타이쿤</h1>
<div class="mw-body-content" id="bodyContent">
<div id="siteSub">위키백과, 우리 모두의 백과사전.</div>
<div id="contentSub"></div>
<div class="mw-jump" id="jump-to-nav">
					이동:					<a href="#mw-head">둘러보기</a>, 					<a href="#p-search">검색</a>
</div>
<div class="mw-content-ltr" dir="ltr" id="mw-content-text" lang="ko"><script>function mfTempOpenSection(id){var block=document.getElementById("mf-section-"+id);block.className+=" open-block";block.previousSibling.className+=" open-block";}</script><p>《<b>스쿨 타이쿤</b>》(School Tycoon)은 캣 데디 게임스사(Cat Daddy Games)가 개발하고 글로벌 스타사(Global Star)가 <a href="/wiki/2004%EB%85%84" title="2004년">2004년</a>에 발매한 학교 시뮬레이션 게임이며 현재 한국어판도 출시되었다. 게임 운영의 통계학 같은 복잡한 요소가 없어서 그런지 다른 <a class="new" href="/w/index.php?title=%ED%83%80%EC%9D%B4%EC%BF%A4_%EA%B2%8C%EC%9E%84&amp;action=edit&amp;redlink=1" title="타이쿤 게임 (없는 문서)">타이쿤 게임</a> 보다 쉽고 간단하다.</p>
<h2><span class="mw-headline" id=".EA.B2.8C.EC.9E.84.EB.B0.A9.EB.B2.95">게임방법</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%8A%A4%EC%BF%A8_%ED%83%80%EC%9D%B4%EC%BF%A4&amp;action=edit&amp;section=1" title="부분 편집: 게임방법">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p><a href="/wiki/%EC%98%81%EC%96%B4" title="영어">영어</a>, <a href="/wiki/%EC%88%98%ED%95%99" title="수학">수학</a>, <a href="/wiki/%EC%82%AC%ED%9A%8C" title="사회">사회</a>, <a class="mw-redirect" href="/wiki/%EB%8F%85%EC%84%9C" title="독서">독서</a>, <a href="/wiki/%EC%83%9D%EB%AC%BC%ED%95%99" title="생물학">생물학</a>, <a href="/wiki/%ED%99%94%ED%95%99" title="화학">화학</a>, <a href="/wiki/%EC%BB%B4%ED%93%A8%ED%84%B0" title="컴퓨터">컴퓨터</a>, <a class="new" href="/w/index.php?title=%EA%B8%B0%EC%88%A0%EA%B0%80%EC%A0%95%ED%95%99&amp;action=edit&amp;redlink=1" title="기술가정학 (없는 문서)">기술가정학</a>, <a href="/wiki/%EB%AF%B8%EC%88%A0" title="미술">미술</a>, <a href="/wiki/%EC%9D%8C%EC%95%85" title="음악">음악</a> 등의 강의실을 지어 놓으면 학생들이 몰려온다. 플레이어는 교장 선생님이 되어 학생들의 이탈을 막기 위해 <a href="/wiki/%EC%B2%B4%EC%9C%A1%EA%B4%80" title="체육관">체육관</a>, <a href="/wiki/%EC%B6%95%EA%B5%AC%EC%9E%A5" title="축구장">축구장</a>, <a class="new" href="/w/index.php?title=%ED%85%8C%EB%8B%88%EC%8A%A4%EC%9E%A5&amp;action=edit&amp;redlink=1" title="테니스장 (없는 문서)">테니스장</a>, <a href="/wiki/%EB%86%8D%EA%B5%AC%EC%9E%A5" title="농구장">농구장</a>, <a href="/wiki/%EC%95%BC%EA%B5%AC%EC%9E%A5" title="야구장">야구장</a>, <a class="new" href="/w/index.php?title=%EB%AF%B8%EC%8B%9D%EC%B6%95%EA%B5%AC%EC%9E%A5&amp;action=edit&amp;redlink=1" title="미식축구장 (없는 문서)">미식축구장</a>, <a class="mw-redirect" href="/wiki/%EC%98%A4%EB%9D%BD%EC%8B%A4" title="오락실">오락실</a>, <a href="/wiki/%EB%A1%A4%EB%9F%AC%EC%BD%94%EC%8A%A4%ED%84%B0" title="롤러코스터">롤러코스터</a> 등 운동과 놀이시설 등을 짓고 <a class="mw-redirect" href="/wiki/%EB%AF%B8%ED%99%94%EC%9B%90" title="미화원">미화원</a> 등을 고용해 학교를 더욱 발전해 나가는 게임이다. 또 불량학생들은 처벌시설에 가둬놓고 거기서 교육시키는 경우도 있다.</p>
<h2><span class="mw-headline" id=".EC.99.B8.EB.B6.80_.EC.97.B0.EA.B2.B0">외부 연결</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%8A%A4%EC%BF%A8_%ED%83%80%EC%9D%B4%EC%BF%A4&amp;action=edit&amp;section=2" title="부분 편집: 외부 연결">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
