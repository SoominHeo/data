<p><b>스타드 드 프랑스</b>(<span style="font-size: smaller;"><a href="/wiki/%ED%94%84%EB%9E%91%EC%8A%A4%EC%96%B4" title="프랑스어">프랑스어</a>:</span> <span lang="fr" xml:lang="fr">Stade de France</span>)는 <a href="/wiki/%ED%94%84%EB%9E%91%EC%8A%A4" title="프랑스">프랑스</a> <a href="/wiki/%EC%83%9D%EB%93%9C%EB%8B%88" title="생드니">생드니</a>에 건설된 국립경기장이다. <a href="/wiki/%ED%94%84%EB%9E%91%EC%8A%A4_%EC%B6%95%EA%B5%AC_%EA%B5%AD%EA%B0%80%EB%8C%80%ED%91%9C%ED%8C%80" title="프랑스 축구 국가대표팀">프랑스 축구 국가대표팀</a>과 <a class="new" href="/w/index.php?title=%ED%94%84%EB%9E%91%EC%8A%A4_%EB%9F%AD%EB%B9%84_%EC%9C%A0%EB%8B%88%EC%98%A8_%EA%B5%AD%EA%B0%80%EB%8C%80%ED%91%9C%ED%8C%80&amp;action=edit&amp;redlink=1" title="프랑스 럭비 유니온 국가대표팀 (없는 문서)">프랑스 럭비 유니온 국가대표팀</a>이 국제 경기를 치를 때에 사용한다. 수용 인원은 81,338명으로, <a href="/wiki/%EC%9C%A0%EB%9F%BD" title="유럽">유럽</a>에서 다섯 번째로 큰 규모이다. <a href="/wiki/1998%EB%85%84_FIFA_%EC%9B%94%EB%93%9C%EC%BB%B5" title="1998년 FIFA 월드컵">1998년 FIFA 월드컵</a>과 <a href="/wiki/2003%EB%85%84_%EC%84%B8%EA%B3%84_%EC%9C%A1%EC%83%81_%EC%84%A0%EC%88%98%EA%B6%8C_%EB%8C%80%ED%9A%8C" title="2003년 세계 육상 선수권 대회">2003년 세계 육상 선수권 대회</a>과 <a href="/wiki/2007%EB%85%84_%EB%9F%AD%EB%B9%84_%EC%9B%94%EB%93%9C%EC%BB%B5" title="2007년 럭비 월드컵">2007년 럭비 월드컵</a>이 개최되었으며, 세계에서 유일하게 <a href="/wiki/FIFA_%EC%9B%94%EB%93%9C%EC%BB%B5" title="FIFA 월드컵">FIFA 월드컵</a>과 <a href="/wiki/%EB%9F%AD%EB%B9%84_%EC%9B%94%EB%93%9C%EC%BB%B5" title="럭비 월드컵">럭비 월드컵</a>의 결승전이 열린 장소이기도 하다. 2004년부터 2006년에 <a href="/wiki/%EB%9F%B0%EB%8D%98" title="런던">런던</a>의 <a class="mw-redirect" href="/wiki/%EC%9B%B8%EB%B8%94%EB%A6%AC_%EA%B2%BD%EA%B8%B0%EC%9E%A5" title="웸블리 경기장">웸블리 경기장</a>으로 옮기기 전까지 매년마다 <a href="/wiki/%EB%AA%A8%ED%84%B0%EC%8A%A4%ED%8F%AC%EC%B8%A0" title="모터스포츠">모터스포츠</a> 행사인 <a class="new" href="/w/index.php?title=%EB%A0%88%EC%9D%B4%EC%8A%A4_%EC%98%A4%EB%B8%8C_%EC%B1%94%ED%94%BC%EC%96%B8%EC%8A%A4&amp;action=edit&amp;redlink=1" title="레이스 오브 챔피언스 (없는 문서)">레이스 오브 챔피언스</a>가 열리기도 하였다. 스타드 드 프랑스는 <a class="mw-redirect" href="/wiki/UEFA_%EA%B2%BD%EA%B8%B0%EC%9E%A5_%EB%93%B1%EA%B8%89" title="UEFA 경기장 등급">UEFA 5성급 경기장</a>에 올라 있다.</p>
<p>스타드 드 프랑스는 <a href="/wiki/%ED%8C%8C%EB%A6%AC_(%ED%94%84%EB%9E%91%EC%8A%A4)" title="파리 (프랑스)">파리</a>의 메인 <a href="/wiki/%EB%9F%AD%EB%B9%84" title="럭비">럭비</a>팀인 <a class="new" href="/w/index.php?title=%EC%8A%A4%ED%83%80%EB%93%9C_%ED%94%84%EB%9E%91%EC%8A%A4_%ED%8C%8C%EB%A6%AC&amp;action=edit&amp;redlink=1" title="스타드 프랑스 파리 (없는 문서)">스타드 프랑스 파리</a>의 두 번째 경기장으로 사용되고 있으며, <a class="new" href="/w/index.php?title=Top_14&amp;action=edit&amp;redlink=1" title="Top 14 (없는 문서)">Top 14</a> <a class="mw-redirect" href="/wiki/%EB%9F%AD%EB%B9%84_%EC%9C%A0%EB%8B%88%EC%98%A8" title="럭비 유니온">럭비 유니온</a> 경기나 <a href="/wiki/%EC%BF%A0%ED%94%84_%EB%93%9C_%ED%94%84%EB%9E%91%EC%8A%A4" title="쿠프 드 프랑스">쿠프 드 프랑스</a>, <a href="/wiki/%EC%BF%A0%ED%94%84_%EB%93%9C_%EB%9D%BC_%EB%A6%AC%EA%B7%B8" title="쿠프 드 라 리그">쿠프 드 라 리그</a>, <a class="mw-redirect" href="/wiki/%EC%BF%A0%ED%94%84_%EA%B0%95%EB%B0%94%EB%A5%B4%EB%8D%B8%EB%9D%BC" title="쿠프 강바르델라">쿠프 강바르델라</a>, <a class="new" href="/w/index.php?title=%EC%B1%8C%EB%A6%B0%EC%A7%80_%EB%93%9C_%ED%94%84%EB%9E%91%EC%8A%A4&amp;action=edit&amp;redlink=1" title="챌린지 드 프랑스 (없는 문서)">챌린지 드 프랑스</a> 결승전 등의 각종 주요 국내 경기가 개최된다. 이 곳에서 두 번의 <a href="/wiki/UEFA_%EC%B1%94%ED%94%BC%EC%96%B8%EC%8A%A4%EB%A6%AC%EA%B7%B8" title="UEFA 챔피언스리그">UEFA 챔피언스리그</a> 결승전 (2000년과 2006년)이 열렸다. 스타드 드 프랑스는 음악 공연장으로 이용되기도 하며, 다른 스포츠 행사가 열리기도 한다. 이 경기장은 스타드 드 프랑스 컨소시엄이 소유하고 있으며, 운영 또한 담당한다.</p>

<h2><span class="mw-headline" id=".EC.97.AD.EC.82.AC">역사</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%8A%A4%ED%83%80%EB%93%9C_%EB%93%9C_%ED%94%84%EB%9E%91%EC%8A%A4&amp;action=edit&amp;section=1" title="부분 편집: 역사">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p><a href="/wiki/1992%EB%85%84" title="1992년">1992년</a> <a href="/wiki/7%EC%9B%94_2%EC%9D%BC" title="7월 2일">7월 2일</a> <a class="mw-redirect" href="/wiki/FIFA" title="FIFA">FIFA</a>로부터 <a href="/wiki/1998%EB%85%84_FIFA_%EC%9B%94%EB%93%9C%EC%BB%B5" title="1998년 FIFA 월드컵">1998년 FIFA 월드컵</a>을 개최할 국가로 <a href="/wiki/%ED%94%84%EB%9E%91%EC%8A%A4" title="프랑스">프랑스</a>가 선택되었고, 이에 프랑스 국립경기장에 대한 논의가 시작되었다. 논의 결과, 정부와 <a href="/wiki/%ED%94%84%EB%9E%91%EC%8A%A4_%EC%B6%95%EA%B5%AC_%ED%98%91%ED%9A%8C" title="프랑스 축구 협회">프랑스 축구 협회</a>는 80,000명 이상의 관중을 수용할 수 있으며 전 좌석으로 구성되어 있는 경기장을 건설하기로 협의하였다. 특별 행사를 목적으로 경기장을 건설하는 것은 <a href="/wiki/%EC%8A%A4%ED%83%80%EB%93%9C_%EC%98%AC%EB%9E%AD%ED%94%BC%ED%81%AC_%EC%9D%B4%EB%B8%8C_%EB%92%A4_%EB%A7%88%EB%88%84%EC%95%84%EB%A5%B4" title="스타드 올랭피크 이브 뒤 마누아르">스타드 올랭피크 이브 뒤 마누아르</a>의 건설 이후 70년 만에 처음이었다. 경기장의 규모와 중요성 때문에, 주 의회는 경기장이 어떻게 건설되고 경비가 어떻게 나가는지 직접 관여하였다. 의회는 경기장을 지을 곳으로 가능한 한 프랑스의 수도인 <a href="/wiki/%ED%8C%8C%EB%A6%AC_(%ED%94%84%EB%9E%91%EC%8A%A4)" title="파리 (프랑스)">파리</a>와 가까운 곳을 물색하였고, 경기장의 건설자와 운영자는 경기장이 완성될 때까지 30개월동안 재정적 지원을 받았다. 경기장의 설계는 CR SCAU 건설의 미셸 마카리, 아이메리크 주블레나, 미셸 레겡발, 클로드 콩슈탄티니로 구성된 단체가 맡게 되었다.</p>
<p>경기장 건설은 정부가 건설 회사로 <a class="new" href="/w/index.php?title=%EB%B6%80%EC%9D%B4%EA%B7%B8&amp;action=edit&amp;redlink=1" title="부이그 (없는 문서)">부이그</a>, <a class="new" href="/w/index.php?title=%EB%92%A4%EB%A9%94%EC%A6%88&amp;action=edit&amp;redlink=1" title="뒤메즈 (없는 문서)">뒤메즈</a>, <a class="new" href="/w/index.php?title=SGE&amp;action=edit&amp;redlink=1" title="SGE (없는 문서)">SGE</a>를 택하면서 본격적으로 시작되었고, <a href="/wiki/1995%EB%85%84" title="1995년">1995년</a> <a href="/wiki/4%EC%9B%94_30%EC%9D%BC" title="4월 30일">4월 30일</a>에 건축을 허가하였다. <a href="/wiki/1995%EB%85%84" title="1995년">1995년</a> <a href="/wiki/5%EC%9B%94_2%EC%9D%BC" title="5월 2일">5월 2일</a> 경기장 공사가 시작돠었고, 31개월만에 완공되었다. 첫 주춧돌을 세우는 일은 공사가 시작된 지 5달 뒤인 <a href="/wiki/9%EC%9B%94_6%EC%9D%BC" title="9월 6일">9월 6일</a>에 하였다. 공사가 시작된 지 1년 뒤에 800,000m² 이상의 토루가 완성되었으며, 180,000 m³의 콘크리트가 부어졌다. 지붕의 설치에 4천 5백만 <a href="/wiki/%EC%9C%A0%EB%A1%9C" title="유로">유로</a>가 들었으며, 이동 플랫폼 또한 1년 뒤에 완성되었다.</p>
<p>건설 기간 동안 경기장은 <b>그랑 스타드</b>(<span style="font-size: smaller;"><a href="/wiki/%ED%94%84%EB%9E%91%EC%8A%A4%EC%96%B4" title="프랑스어">프랑스어</a>:</span> <span lang="fr" xml:lang="fr">Grand Stade, 위대한 경기장이라는 뜻</span>)라는 이름으로 불렸다. <a href="/wiki/1995%EB%85%84" title="1995년">1995년</a> <a href="/wiki/12%EC%9B%94_4%EC%9D%BC" title="12월 4일">12월 4일</a> 체육부에서는 경기장에 붙일 이름을 공모하였고, <a href="/wiki/%ED%94%84%EB%9E%91%EC%8A%A4" title="프랑스">프랑스</a>의 전설적인 축구 선수인 <a href="/wiki/%EB%AF%B8%EC%85%B8_%ED%94%8C%EB%9D%BC%ED%8B%B0%EB%8B%88" title="미셸 플라티니">미셸 플라티니</a>의 제안에 따라 <b>스타드 드 프랑스</b>로 공식적으로 명명되었다. <a href="/wiki/1998%EB%85%84" title="1998년">1998년</a> <a href="/wiki/1%EC%9B%94_28%EC%9D%BC" title="1월 28일">1월 28일</a> 경기장이 개장되어 <a href="/wiki/%ED%94%84%EB%9E%91%EC%8A%A4_%EC%B6%95%EA%B5%AC_%EA%B5%AD%EA%B0%80%EB%8C%80%ED%91%9C%ED%8C%80" title="프랑스 축구 국가대표팀">프랑스</a>와 <a href="/wiki/%EC%8A%A4%ED%8E%98%EC%9D%B8_%EC%B6%95%EA%B5%AC_%EA%B5%AD%EA%B0%80%EB%8C%80%ED%91%9C%ED%8C%80" title="스페인 축구 국가대표팀">스페인</a> 간의 기념 경기가 열였으며, 건설 비용으로 약 2억 9천만 <a href="/wiki/%EC%9C%A0%EB%A1%9C" title="유로">유로</a>가 소요되었다.</p>
<h2><span class="mw-headline" id=".EA.B5.AC.EC.A1.B0_.EB.B0.8F_.EC.8B.9C.EC.84.A4">구조 및 시설</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%8A%A4%ED%83%80%EB%93%9C_%EB%93%9C_%ED%94%84%EB%9E%91%EC%8A%A4&amp;action=edit&amp;section=2" title="부분 편집: 구조 및 시설">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>스타드 드 프랑스는 필요한 경우에 육상 트랙을 사용할 수 있도록 이동 좌석이 설치되어 있다. 경기장은 어떻게 하면 관중이 경기를 더 잘 관전할 수 있을지를 소프트웨어 시뮬레이션의 도움을 받아 설계되었다. <a href="/wiki/%EC%83%9D%EB%93%9C%EB%8B%88" title="생드니">생드니</a>, <a class="new" href="/w/index.php?title=%EC%98%A4%EB%B2%A0%EB%A5%B4%EB%B9%8C%EB%A6%AC%EC%97%90%EB%A5%B4&amp;action=edit&amp;redlink=1" title="오베르빌리에르 (없는 문서)">오베르빌리에르</a>, <a class="mw-redirect" href="/wiki/%EC%83%9D%ED%88%AC%EC%95%99" title="생투앙">생투앙</a>을 가로지르는 지역인 <a class="new" href="/w/index.php?title=%ED%94%8C%EB%9E%AD&amp;action=edit&amp;redlink=1" title="플랭 (없는 문서)">플랭</a>의 발전을 소개하려는 의도 또한 가지고 있으며, 주된 목표는 새로운 주거 지역과 세 지역에 경기장을 세움으로서 지역을 발전시키는 것이다.</p>
