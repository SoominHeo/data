<h2><span class="mw-headline" id=".EC.97.AD.EC.82.AC.EC.99.80_.EB.AA.85.EC.B9.AD">역사와 명칭</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%8A%A4%ED%83%80%EC%9D%BC_%EC%95%84%EC%9D%B4%EC%BD%98_%EC%95%84%EC%8B%9C%EC%95%84&amp;action=edit&amp;section=1" title="부분 편집: 역사와 명칭">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>'스타일 아이콘 어워즈(Style Icon Awards)'라는 이름으로 <a href="/wiki/2008%EB%85%84" title="2008년">2008년</a> <a href="/wiki/10%EC%9B%94_30%EC%9D%BC" title="10월 30일">10월 30일</a> 처음 개최됐다. <a href="/wiki/2009%EB%85%84" title="2009년">2009년</a> <a href="/wiki/11%EC%9B%94_11%EC%9D%BC" title="11월 11일">11월 11일</a> 2회, <a href="/wiki/2010%EB%85%84" title="2010년">2010년</a> <a href="/wiki/11%EC%9B%94_17%EC%9D%BC" title="11월 17일">11월 17일</a> 3회, <a href="/wiki/2011%EB%85%84" title="2011년">2011년</a> <a href="/wiki/11%EC%9B%94_3%EC%9D%BC" title="11월 3일">11월 3일</a> 4회, <a href="/wiki/2012%EB%85%84" title="2012년">2012년</a> <a href="/wiki/10%EC%9B%94_25%EC%9D%BC" title="10월 25일">10월 25일</a> 5회, <a href="/wiki/2013%EB%85%84" title="2013년">2013년</a> <a href="/wiki/10%EC%9B%94_24%EC%9D%BC" title="10월 24일">10월 24일</a> 6회가 개최되었고, <a href="/wiki/2014%EB%85%84" title="2014년">2014년</a> <a href="/wiki/10%EC%9B%94_28%EC%9D%BC" title="10월 28일">10월 28일</a> 7회가 개최됐다. '스타일 아이콘 아시아(Style Icon Asia)'로 이름을 바꾼 후 <a href="/wiki/2016%EB%85%84" title="2016년">2016년</a> <a href="/wiki/3%EC%9B%94_15%EC%9D%BC" title="3월 15일">3월 15일</a> 8회가 페스티벌 형식으로 개최됐다.<sup class="reference" id="cite_ref-1"><a href="#cite_note-1">[1]</a></sup></p>
