<p><b>스탠 게츠</b>(<b>Stan Getz</b>, <a href="/wiki/%EC%95%84%EB%AA%85" title="아명">원명(原名)</a>은 <b>스탠리 게이츠키</b>(<b>Stanley Gayetzky</b>)이며 나중에 <b>스탠리 게츠</b>(<b>Stanley Getz</b>)로 <a href="/wiki/%EA%B0%9C%EB%AA%85" title="개명">개명(改名)</a>, <a href="/wiki/1927%EB%85%84" title="1927년">1927년</a> <a href="/wiki/2%EC%9B%94_2%EC%9D%BC" title="2월 2일">2월 2일</a> ~ <a href="/wiki/1991%EB%85%84" title="1991년">1991년</a> <a href="/wiki/6%EC%9B%94_6%EC%9D%BC" title="6월 6일">6월 6일</a>)는 <a href="/wiki/%EB%AF%B8%EA%B5%AD" title="미국">미국</a>의 <a href="/wiki/%EC%9E%AC%EC%A6%88" title="재즈">재즈</a> <a href="/wiki/%EC%83%89%EC%86%8C%ED%8F%B0" title="색소폰">색소폰 연주가</a> 겸 <a class="mw-redirect" href="/wiki/%EC%98%81%ED%99%94_%EB%B0%B0%EC%9A%B0" title="영화 배우">영화배우</a>이다.</p>
<h2><span class="mw-headline" id=".EC.83.9D.EC.95.A0">생애</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%8A%A4%ED%83%A0_%EA%B2%8C%EC%B8%A0&amp;action=edit&amp;section=1" title="부분 편집: 생애">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>그는 16세 때였던 <a href="/wiki/1943%EB%85%84" title="1943년">1943년</a> <a href="/wiki/%EC%83%89%EC%86%8C%ED%8F%B0" title="색소폰">색소폰 연주가</a> 첫 데뷔하였고 3년 후 <a href="/wiki/1946%EB%85%84" title="1946년">1946년</a> <a href="/wiki/%ED%85%8C%EB%84%88_%EC%83%89%EC%86%8C%ED%8F%B0" title="테너 색소폰">테너 색소폰</a> 연주에도 손을 대었으며 이듬해 <a href="/wiki/1947%EB%85%84" title="1947년">1947년</a> <a href="/wiki/%EB%B0%94%EB%A6%AC%ED%86%A4" title="바리톤">바리톤</a> <a href="/wiki/%EC%83%89%EC%86%8C%ED%8F%B0" title="색소폰">색소폰</a> 연주에도 손을 대어 <a href="/wiki/%EC%83%89%EC%86%8C%ED%8F%B0" title="색소폰">색소폰</a> 연주의 폭을 넓혔다.</p>
<p>선배 재즈 음악가 <a href="/wiki/%EB%B2%A0%EB%8B%88_%EA%B5%BF%EB%A7%A8" title="베니 굿맨">베니 굿맨</a>의 총애를 받기도 한 그는 <a href="/wiki/1955%EB%85%84" title="1955년">1955년</a> <a class="mw-redirect" href="/wiki/%EB%AF%B8%EA%B5%AD_%EC%98%81%ED%99%94" title="미국 영화">미국 영화</a> 《The Benny Goodman story》의 단역 출연을 통하여 <a class="mw-redirect" href="/wiki/%EC%98%81%ED%99%94_%EB%B0%B0%EC%9A%B0" title="영화 배우">영화배우</a> 데뷔하였다.</p>
