<p><b>스테파노 발디니</b>(<span style="font-size: smaller;"><a href="/wiki/%EC%9D%B4%ED%83%88%EB%A6%AC%EC%95%84%EC%96%B4" title="이탈리아어">이탈리아어</a>:</span> <span lang="it" xml:lang="it">Stefano Baldini</span>, <a href="/wiki/1971%EB%85%84" title="1971년">1971년</a> <a href="/wiki/5%EC%9B%94_25%EC%9D%BC" title="5월 25일">5월 25일</a> ~ )는 <a href="/wiki/%EC%9D%B4%ED%83%88%EB%A6%AC%EC%95%84" title="이탈리아">이탈리아</a>의 육상 선수이며, <a href="/wiki/%EB%A7%88%EB%9D%BC%ED%86%A4" title="마라톤">마라톤</a>에 전문적인 그는 <a class="mw-redirect" href="/wiki/2004%EB%85%84_%EC%95%84%ED%85%8C%EB%84%A4_%EC%98%AC%EB%A6%BC%ED%94%BD" title="2004년 아테네 올림픽">2004년 아테네 올림픽</a>에서 <a href="/wiki/%EA%B8%88%EB%A9%94%EB%8B%AC" title="금메달">금메달</a>을 획득했고 유럽 선수권 대회에서 2연패했다.</p>
<p>그는 또한 1996년에 <a class="mw-redirect" href="/wiki/%ED%8C%94%EB%A7%88_%EB%8D%B0_%EB%A7%88%EC%9A%94%EB%A5%B4%EC%B9%B4" title="팔마 데 마요르카">팔마 데 마요르카</a>에서 열린 국제 하프 마라톤 대회에서 세계 챔피언이기도 하였다.</p>
<p>자신이 <a href="/wiki/%EB%9F%B0%EB%8D%98_%EB%A7%88%EB%9D%BC%ED%86%A4" title="런던 마라톤">런던 마라톤</a>을 완료하지 않은 실망적 <a href="/wiki/2007%EB%85%84" title="2007년">2007년</a> 시즌 후에 발디니는 <a href="/wiki/%EB%89%B4%EC%9A%95_%EB%A7%88%EB%9D%BC%ED%86%A4" title="뉴욕 마라톤">뉴욕 마라톤</a>에 달리는 데 선택하여 4위를 하였다. <a class="mw-redirect" href="/wiki/2008%EB%85%84_%EB%B2%A0%EC%9D%B4%EC%A7%95_%EC%98%AC%EB%A6%BC%ED%94%BD" title="2008년 베이징 올림픽">2008년 베이징 올림픽</a>에서 12위를 한 발디니는 <a href="/wiki/2010%EB%85%84" title="2010년">2010년</a> <a href="/wiki/10%EC%9B%94" title="10월">10월</a>에 선수생활을 마감했다.</p>
