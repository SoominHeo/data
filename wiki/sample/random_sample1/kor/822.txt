<h2><span class="mw-headline" id=".EA.B0.9C.EC.9A.94">개요</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Beck_(%EB%A7%8C%ED%99%94)&amp;action=edit&amp;section=1" title="부분 편집: 개요">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p><a href="/wiki/1999%EB%85%84" title="1999년">1999년</a>부터 <a href="/wiki/2008%EB%85%84" title="2008년">2008년</a>까지 월간 소년 매거진(<a href="/wiki/%EA%B3%A0%EB%8B%A8%EC%83%A4" title="고단샤">고단샤</a>)에서 연재되어, <a href="/wiki/2002%EB%85%84" title="2002년">2002년</a>에 제26회 <a href="/wiki/%EA%B3%A0%EB%8B%A8%EC%83%A4_%EB%A7%8C%ED%99%94%EC%83%81" title="고단샤 만화상">고단샤 만화상</a> 소년 부문 수상했다. <a href="/wiki/2004%EB%85%84" title="2004년">2004년</a> <a href="/wiki/10%EC%9B%94" title="10월">10월</a>부터 <a href="/wiki/2005%EB%85%84" title="2005년">2005년</a> <a href="/wiki/3%EC%9B%94" title="3월">3월</a>까지 <a href="/wiki/%EC%95%A0%EB%8B%88%EB%A9%94%EC%9D%B4%EC%85%98" title="애니메이션">애니메이션</a>이 방송되었으며, <a href="/wiki/2010%EB%85%84" title="2010년">2010년</a> <a href="/wiki/9%EC%9B%94" title="9월">9월</a>에는 <a href="/wiki/%EC%B8%A0%EC%B8%A0%EB%AF%B8_%EC%9C%A0%ED%82%A4%ED%9E%88%EC%BD%94" title="츠츠미 유키히코">츠츠미 유키히코</a> 감독에 의해 <a class="mw-redirect" href="/wiki/%EC%8B%A4%EC%82%AC_%EC%98%81%ED%99%94" title="실사 영화">실사 영화</a> "BECK"이 공개되었다. <a href="/wiki/%EB%8C%80%ED%95%9C%EB%AF%BC%EA%B5%AD" title="대한민국">대한민국</a>에서는 같은 해 <a href="/wiki/11%EC%9B%94" title="11월">11월</a> 개봉되었다.</p>
<p>"고릴라맨", "멍청한 녀석"으로 알려진 <a href="/wiki/%EB%A7%8C%ED%99%94%EA%B0%80" title="만화가">만화가</a> <a class="new" href="/w/index.php?title=%ED%95%B4%EB%9F%B4%EB%93%9C_%EC%82%AC%EC%BF%A0%EC%9D%B4%EC%8B%9C&amp;action=edit&amp;redlink=1" title="해럴드 사쿠이시 (없는 문서)">해럴드 사쿠이시</a>에 있어서 첫 <a href="/wiki/%EC%86%8C%EB%85%84_%EB%A7%8C%ED%99%94" title="소년 만화">소년 만화</a>이다. 전 34권과 가이드 북 4권이 발매되어, <a href="/wiki/%EC%9D%BC%EB%B3%B8" title="일본">일본</a>에서 누계 발행 부수는 1200만부를 넘는 대 히트작이다. 단행본 한정판에 부속된 코유키의 피겨에 맞는 스케일로 깁슨 등의 기타, 베이스, 드럼 (제3​​탄)의 소형이 2008년 6월 현재 4탄까지 발매되고 있다.</p>
<p>작품의 세계는 "고릴라맨"이나 "바카이치", "멍청한 녀석"와 같은 세계의 설정을 가지고 있으며, 종종 다른 작품의 등장 인물이 조연으로 등장하고 있다.</p>
