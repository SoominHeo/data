<h2><span class="mw-headline" id=".EC.B4.88.EA.B8.B0_.EA.B2.BD.EB.A0.A5">초기 경력</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%8A%A4%ED%85%8C%ED%8C%90_%EC%9D%8C%EB%B9%84%EC%95%84&amp;action=edit&amp;section=1" title="부분 편집: 초기 경력">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>스테판 음비아는 <a href="/wiki/%EC%B9%B4%EB%A9%94%EB%A3%AC" title="카메룬">카메룬</a>의 <a href="/wiki/%EB%91%90%EC%95%8C%EB%9D%BC" title="두알라">두알라</a>에 위치한 카지 스포츠 아카데미의 유소년팀에 입단하여 <a href="/wiki/%EC%82%AC%EB%AE%88%EC%97%98_%EC%97%90%ED%86%A0" title="사뮈엘 에토">사뮈엘 에토</a>, <a href="/wiki/%EC%9D%B4%EB%93%9C%EB%A6%AC%EC%8A%A4_%EC%B9%B4%EB%A5%BC%EB%A1%9C%EC%8A%A4_%EC%B9%B4%EB%A9%94%EB%8B%88" title="이드리스 카를로스 카메니">이드리스 카를로스 카메니</a>, 에릭 젬바-젬바 등이 밟은 엘리트코스를 따랐다. 2004년, 그는 <a href="/wiki/%ED%94%84%EB%9E%91%EC%8A%A4" title="프랑스">프랑스</a>의 <a href="/wiki/%EC%8A%A4%ED%83%80%EB%93%9C_%EB%A0%8C_FC" title="스타드 렌 FC">스타드 렌 FC</a>로 이적하였다.</p>
<p>2005-06 시즌, 음비아는 <a href="/wiki/%EC%8A%A4%ED%83%80%EB%93%9C_%EB%A0%8C_FC" title="스타드 렌 FC">스타드 렌 FC</a>의 1군으로 승격되었고, 다수의 경기에 출전하였다. 이 <a href="/wiki/%EC%B9%B4%EB%A9%94%EB%A3%AC" title="카메룬">카메룬</a>인은 2006년 11월에 <a href="/wiki/%EC%98%AC%EB%9E%AD%ED%94%BC%ED%81%AC_%EB%A6%AC%EC%98%B9" title="올랭피크 리옹">올랭피크 리옹</a>을 상대로 프로계약 이후 첫 리그골을 득점하였다.</p>
<p>2007년 4월 18일, 음비아는 리옹의 <a href="/wiki/%EA%B3%B5%EA%B2%A9%EC%88%98" title="공격수">스트라이커</a> <a href="/wiki/%EB%B0%80%EB%9E%80_%EB%B0%94%EB%A1%9C%EC%8B%9C" title="밀란 바로시">밀란 바로시</a>가 인종 차별하였다고 고소하였다. 이로 인해, 바로시는 행위에 대해 유죄판결을 받았으나, 인종차별을 하지 않은 것으로 밝혀졌고, 바로시는 3경기 출장이 금지되었다.<sup class="reference" id="cite_ref-1"><a href="#cite_note-1">[1]</a></sup><sup class="reference" id="cite_ref-2"><a href="#cite_note-2">[2]</a></sup></p>
<p>2009년 7월 14일, 음비아는 £10.4M의 가격에 <a href="/wiki/%EC%8A%A4%ED%83%80%EB%93%9C_%EB%A0%8C_FC" title="스타드 렌 FC">스타드 렌 FC</a>에서 <a href="/wiki/%EC%98%AC%EB%9E%AD%ED%94%BC%ED%81%AC_%EB%93%9C_%EB%A7%88%EB%A5%B4%EC%84%B8%EC%9C%A0" title="올랭피크 드 마르세유">올랭피크 드 마르세유</a>로 이적하였다. 2009-10 시즌 중반, 그는 <a href="/wiki/%EC%88%98%EB%B9%84%EC%88%98" title="수비수">센터백</a>으로 전향하여 <a href="/wiki/%EC%88%A0%EB%A0%88%EB%A7%8C_%EB%94%94%EC%95%84%EC%99%80%EB%9D%BC" title="술레만 디아와라">술레만 디아와라</a>와 센터백 듀오를 형성하였다. 그의 중앙 수비수 전향은 성공적이었고, 잔여 시즌동안 그는 이 포지션을 맡았고, 그에 따라 <a href="/wiki/%EB%94%94%EB%94%94%EC%97%90_%EB%8D%B0%EC%83%B9" title="디디에 데샹">디디에 데샹</a>의 주축 선수들 중 하나로 자리잡았다. 또한 마르세유는 2009-10 시즌에 리그 우승을 거두었다.</p>
<p>2012년 8월 30일, 음비아는 <a class="mw-redirect" href="/wiki/%ED%80%B8%EC%A6%88_%ED%8C%8C%ED%81%AC_%EB%A0%88%EC%9D%B8%EC%A0%80%EC%8A%A4_FC" title="퀸즈 파크 레인저스 FC">퀸즈 파크 레인저스 FC</a> 로 이적 완료 후 인터뷰에서 '나는 스코틀랜드의 <a href="/wiki/%EB%A0%88%EC%9D%B8%EC%A0%80%EC%8A%A4_FC" title="레인저스 FC">레인저스 FC</a>의 영입 제의 인줄 알고 받아들였다' 라고 밝혀 팬들의 비난을 받았다.<sup class="reference" id="cite_ref-3"><a href="#cite_note-3">[3]</a></sup></p>
<p>2012-13시즌 <a class="mw-redirect" href="/wiki/%ED%80%B8%EC%A6%88_%ED%8C%8C%ED%81%AC_%EB%A0%88%EC%9D%B8%EC%A0%80%EC%8A%A4_FC" title="퀸즈 파크 레인저스 FC">퀸즈 파크 레인저스 FC</a>가 2부리그로 강등 되면서, 음비아는 <a href="/wiki/%EC%84%B8%EB%B9%84%EC%95%BC_FC" title="세비야 FC">세비야 FC</a>로 임대 이적 하였다.</p>
<h2><span class="mw-headline" id=".EA.B5.AD.EA.B0.80.EB.8C.80.ED.91.9C.ED.8C.80_.EA.B2.BD.EB.A0.A5">국가대표팀 경력</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%8A%A4%ED%85%8C%ED%8C%90_%EC%9D%8C%EB%B9%84%EC%95%84&amp;action=edit&amp;section=2" title="부분 편집: 국가대표팀 경력">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>음비아는 <a href="/wiki/%EC%B9%B4%EB%A9%94%EB%A3%AC_%EC%B6%95%EA%B5%AC_%EA%B5%AD%EA%B0%80%EB%8C%80%ED%91%9C%ED%8C%80" title="카메룬 축구 국가대표팀">카메룬</a>의 청소년 국가대표팀 일원이었다. 그는 <a href="/wiki/%ED%95%80%EB%9E%80%EB%93%9C" title="핀란드">핀란드</a>에서 개최한 2003년 FIFA U-17 월드컵에 참여하였고, 카메룬은 조별 리그에서 탈락하였다.<sup class="reference" id="cite_ref-4"><a href="#cite_note-4">[4]</a></sup></p>
<p>2005년, 그는 <a href="/wiki/%EC%B9%B4%EB%A9%94%EB%A3%AC_%EC%B6%95%EA%B5%AC_%EA%B5%AD%EA%B0%80%EB%8C%80%ED%91%9C%ED%8C%80" title="카메룬 축구 국가대표팀">카메룬 축구 국가대표팀</a> 데뷔전을 치루었다. 2008년에는 <a href="/wiki/%EA%B0%80%EB%82%98" title="가나">가나</a>에서 열린 <a href="/wiki/2008%EB%85%84_%EC%95%84%ED%94%84%EB%A6%AC%EC%B9%B4_%EB%84%A4%EC%9D%B4%EC%85%98%EC%8A%A4%EC%BB%B5" title="2008년 아프리카 네이션스컵">2008년 아프리카 네이션스컵</a>에 참여하였고, 이 대회의 8강인 <a href="/wiki/%ED%8A%80%EB%8B%88%EC%A7%80_%EC%B6%95%EA%B5%AC_%EA%B5%AD%EA%B0%80%EB%8C%80%ED%91%9C%ED%8C%80" title="튀니지 축구 국가대표팀">튀니지</a>전에서 음비아는 2번 골네트를 때렸다.</p>
<p>음비아는 <a class="mw-redirect" href="/wiki/%EB%B2%A0%EC%9D%B4%EC%A7%95" title="베이징">베이징</a>에서 열린 <a href="/wiki/2008%EB%85%84_%ED%95%98%EA%B3%84_%EC%98%AC%EB%A6%BC%ED%94%BD" title="2008년 하계 올림픽">2008년 하계 올림픽</a>에 참여하여, 카메룬의 4경기에 모두 출장하였고, <a href="/wiki/%EC%98%A8%EB%91%90%EB%9D%BC%EC%8A%A4_%EC%B6%95%EA%B5%AC_%EA%B5%AD%EA%B0%80%EB%8C%80%ED%91%9C%ED%8C%80" title="온두라스 축구 국가대표팀">온두라스</a>전에서는 득점도 하였다.<sup class="reference" id="cite_ref-5"><a href="#cite_note-5">[5]</a></sup></p>
<h2><span class="mw-headline" id=".EC.82.AC.EC.83.9D.ED.99.9C">사생활</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%8A%A4%ED%85%8C%ED%8C%90_%EC%9D%8C%EB%B9%84%EC%95%84&amp;action=edit&amp;section=3" title="부분 편집: 사생활">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>스테판의 형은 프랑크 에툰디로, 축구선수이며, <a href="/wiki/%EC%8A%A4%EC%9C%84%EC%8A%A4" title="스위스">스위스</a>에서 활동하고 있다. 스테판 음비아는 <a href="/wiki/%ED%94%84%EB%9E%91%EC%8A%A4" title="프랑스">프랑스</a> 시민권도 가지고 있다.<sup class="reference" id="cite_ref-6"><a href="#cite_note-6">[6]</a></sup></p>
