<p><b>스테판 페트로비치 시파초프</b>(<span style="font-size: smaller;"><a href="/wiki/%EB%9F%AC%EC%8B%9C%EC%95%84%EC%96%B4" title="러시아어">러시아어</a>:</span> <span lang="ru" xml:lang="ru">Степа́н Петро́вич Щипачёв</span>, <span style="font-size: smaller;"><a href="/wiki/%EC%98%81%EC%96%B4" title="영어">영어</a>:</span> <span lang="en" xml:lang="en">Stepan Shchipachev</span>, <a href="/wiki/1898%EB%85%84" title="1898년">1898년</a> <a href="/wiki/12%EC%9B%94_17%EC%9D%BC" title="12월 17일">12월 17일</a> - <a href="/wiki/1980%EB%85%84" title="1980년">1980년</a> <a href="/wiki/1%EC%9B%94_1%EC%9D%BC" title="1월 1일">1월 1일</a>)은 <a href="/wiki/%EC%86%8C%EB%A0%A8" title="소련">소련</a>의 <a href="/wiki/%EC%8B%9C%EC%9D%B8" title="시인">시인</a> 겸 <a class="mw-redirect" href="/wiki/%EC%88%98%ED%95%84%EA%B0%80" title="수필가">수필가</a>이다.</p>

<p>소년 시절이자 <a href="/wiki/%EB%9F%AC%EC%8B%9C%EC%95%84_%EC%A0%9C%EA%B5%AD" title="러시아 제국">러시아 제국</a> 말기 시기인 <a href="/wiki/1913%EB%85%84" title="1913년">1913년</a>에 <a href="/wiki/%EC%8B%9C_(%EB%AC%B8%ED%95%99)" title="시 (문학)">시인</a>으로 첫 등단하였고 <a href="/wiki/1916%EB%85%84" title="1916년">1916년</a> <a href="/wiki/%EC%88%98%ED%95%84" title="수필">수필가</a>로 등단하였다.</p>
<h3><span class="mw-headline" id=".EC.A1.B0.EC.84.A0.EB.AF.BC.EC.A3.BC.EC.A3.BC.EC.9D.98.EC.9D.B8.EB.AF.BC.EA.B3.B5.ED.99.94.EA.B5.AD_.EC.8B.9C.EC.9D.B8_.EA.B9.80.EC.88.9C.EC.84.9D.EA.B3.BC.EC.9D.98_.EB.A7.8C.EB.82.A8">조선민주주의인민공화국 시인 <a href="/wiki/%EA%B9%80%EC%88%9C%EC%84%9D" title="김순석">김순석</a>과의 만남</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%8A%A4%ED%85%8C%ED%8C%90_%ED%8E%98%ED%8A%B8%EB%A1%9C%EB%B9%84%EC%B9%98_%EC%8B%9C%ED%8C%8C%EC%B4%88%ED%94%84&amp;action=edit&amp;section=3" title="부분 편집: 조선민주주의인민공화국 시인 김순석과의 만남">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>북한 시인 <a href="/wiki/%EA%B9%80%EC%88%9C%EC%84%9D" title="김순석">김순석</a>(金淳石)은 휴전 후 1년이 채 지나지 않은 <a href="/wiki/1954%EB%85%84" title="1954년">1954년</a> <a href="/wiki/4%EC%9B%94_26%EC%9D%BC" title="4월 26일">4월 26일</a> <a class="new" href="/w/index.php?title=%EC%86%8C%EB%A0%A8%EC%9E%91%EA%B0%80%EB%8F%99%EB%A7%B9&amp;action=edit&amp;redlink=1" title="소련작가동맹 (없는 문서)">소련작가동맹</a> 초청으로 소련을 방문해 소련작가동맹 쉬빠쵸브와 교류하였고 쉬빠쵸브(시파초프)는 <a href="/wiki/%EA%B9%80%EC%88%9C%EC%84%9D" title="김순석">김순석</a>의 전달을 통하여 <a href="/wiki/%EA%B9%80%EC%9D%BC%EC%84%B1" title="김일성">김일성</a> 조선민주주의인민공화국 국가원수가 친히 써 준 <b>김효열</b>(<b>金孝悅</b>)이라는 <a href="/wiki/%EC%A1%B0%EC%84%A0%EB%AF%BC%EC%A3%BC%EC%A3%BC%EC%9D%98%EC%9D%B8%EB%AF%BC%EA%B3%B5%ED%99%94%EA%B5%AD" title="조선민주주의인민공화국">북조선</a>식 이름을 받았다. <a href="/wiki/%EA%B9%80%EC%88%9C%EC%84%9D" title="김순석">김순석</a>은 <a href="/wiki/%EC%A1%B0%EC%84%A0%EB%AF%BC%EC%A3%BC%EC%A3%BC%EC%9D%98%EC%9D%B8%EB%AF%BC%EA%B3%B5%ED%99%94%EA%B5%AD" title="조선민주주의인민공화국">조선민주주의인민공화국</a>에 돌아와 쉬빠쵸브와 만나서 나눈 담화를 정리해 《서정시에 대한 담화》라는 글로 자신이 편집위원으로 있었던 〈조선 문학〉 1954년 8월호에 발표하기도 하였다. 쉬빠쵸브와의 만남은 김순석의 향토적 미의식이 가장 잘 드러나는 시집 《황금의 땅》에도 영향을 끼쳤다. 북조선에서는 1955년 조선작가동맹출판사가 《쉬빠쵸브 서정시선집》을 펴냈다.</p>
