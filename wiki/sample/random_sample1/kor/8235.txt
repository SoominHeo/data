<h1 class="firstHeading" id="firstHeading" lang="ko">스투를룽 시대</h1>
<div class="mw-body-content" id="bodyContent">
<div id="siteSub">위키백과, 우리 모두의 백과사전.</div>
<div id="contentSub"></div>
<div class="mw-jump" id="jump-to-nav">
					이동:					<a href="#mw-head">둘러보기</a>, 					<a href="#p-search">검색</a>
</div>
<div class="mw-content-ltr" dir="ltr" id="mw-content-text" lang="ko"><script>function mfTempOpenSection(id){var block=document.getElementById("mf-section-"+id);block.className+=" open-block";block.previousSibling.className+=" open-block";}</script><p><b>스투를룽 시대</b>(<span style="font-size: smaller;"><a href="/wiki/%EC%95%84%EC%9D%B4%EC%8A%AC%EB%9E%80%EB%93%9C%EC%96%B4" title="아이슬란드어">아이슬란드어</a>:</span> <span lang="is" xml:lang="is">Sturlungaöld</span>)란 13세기 중반 <a href="/wiki/%EC%95%84%EC%9D%B4%EC%8A%AC%EB%9E%80%EB%93%9C" title="아이슬란드">아이슬란드</a>의 42-44년 정도의 기간을 말한다. 아이슬란드 역사상 가장 유혈낭자하고 폭력적인 시기였다. <a class="new" href="/w/index.php?title=%EC%8A%A4%ED%88%AC%EB%A5%BC%EB%A3%BD_%EC%9D%BC%EC%A1%B1%EC%9D%98_%EC%82%AC%EA%B0%80&amp;action=edit&amp;redlink=1" title="스투를룽 일족의 사가 (없는 문서)">스투를룽 일족의 사가</a>에 그 상세가 기록되어 있다.</p>
<p>스투를룽 시대는 <a href="/wiki/%EA%B3%A0%EB%94%94" title="고디">고디</a>라는 <a href="/wiki/%EA%B5%B0%EC%9E%A5%EA%B5%AD%EA%B0%80" title="군장국가">군장</a>들 사이의 싸움으로 점철된 시대로, 그 중 가장 강력한 군장 씨족이었던 <a class="mw-redirect" href="/wiki/%EC%8A%A4%ED%88%AC%EB%A5%BC%EB%A3%BD_%EC%94%A8%EC%A1%B1" title="스투를룽 씨족">스투를룽 씨족</a>의 이름을 따서 스투를룽 시대라고 부른다. 스투를룽 시대 말엽에 <a href="/wiki/%EC%95%84%EC%9D%B4%EC%8A%AC%EB%9E%80%EB%93%9C_%EC%9E%90%EC%9C%A0%EA%B5%AD" title="아이슬란드 자유국">아이슬란드 자유국</a>이 멸망하고 <a href="/wiki/%EB%85%B8%EB%A5%B4%EC%9B%A8%EC%9D%B4_%EC%84%B8%EC%8A%B5%EC%99%95%EA%B5%AD" title="노르웨이 세습왕국">노르웨이 세습왕국</a>의 봉토로 편입된다.</p>
<p>역사학자들은 대개 1220년을 스투를룽 시대의 첫 해로 잡는데, 1208년의 <a class="new" href="/w/index.php?title=%EB%B9%84%EB%94%94%EB%84%A4%EC%8A%A4_%EC%A0%84%ED%88%AC&amp;action=edit&amp;redlink=1" title="비디네스 전투 (없는 문서)">비디네스 전투</a> 때문에 더 이른 시기로 비정하는 학자도 있다. 이 시기에 서로 권력을 다툰 주요 씨족들과 그 지역기반은 다음과 같다.</p>
<ul>
<li>아르네스팅그(Árnesþing)의 <a class="new" href="/w/index.php?title=%ED%95%98%EC%9A%B0%EC%B9%B4%EB%8B%AC%EB%A3%A8%EB%A5%B4&amp;action=edit&amp;redlink=1" title="하우카달루르 (없는 문서)">하우카달루르 씨족</a>(Haukdælir)</li>
<li>랑가르팅그(Rangárþing)의 <a class="new" href="/w/index.php?title=%EC%98%A4%EB%8B%A4%EB%B2%A0%EB%9E%B4%EB%A5%B4&amp;action=edit&amp;redlink=1" title="오다베랴르 (없는 문서)">오다베랴르 씨족</a>(Oddaverjar)</li>
<li><a class="new" href="/w/index.php?title=%EC%8A%A4%EC%B9%B4%EA%B0%80%ED%91%9C%EB%A5%B4%EB%91%90%EB%A5%B4&amp;action=edit&amp;redlink=1" title="스카가표르두르 (없는 문서)">스카가표르두르</a>(Skagafjörður)의 <a href="/wiki/%EC%95%84%EC%8A%A4%EB%B9%84%EB%A5%B4%EB%8B%9D" title="아스비르닝">아스비르닝 씨족</a>(Ásbirningar)</li>
<li><a class="new" href="/w/index.php?title=%EC%9D%B4%EC%82%AC%ED%91%9C%EB%A5%B4%EB%91%90%EB%A5%B4&amp;action=edit&amp;redlink=1" title="이사표르두르 (없는 문서)">이사표르두르</a>(Ísafjörður)의 <a class="new" href="/w/index.php?title=%EB%B0%94%ED%8A%B8%EB%8B%88%EC%8A%A4%ED%94%BC%EB%A5%B4%EB%94%A9&amp;action=edit&amp;redlink=1" title="바트니스피르딩 (없는 문서)">바트니스피르딩 씨족</a>(Vatnsfirðingar)</li>
<li><a href="/wiki/%EC%99%B8%EC%9D%B4%EC%8A%A4%ED%8A%80%EB%A5%BC%EB%9E%80%EB%93%9C" title="외이스튀를란드">외이스튀를란드</a>(Austurland)의 <a class="new" href="/w/index.php?title=%EC%8A%A4%EB%B9%88%ED%8E%A0%EB%A7%81&amp;action=edit&amp;redlink=1" title="스빈펠링 (없는 문서)">스빈펠링 씨족</a>(Svínfellingar)</li>
<li><a class="new" href="/w/index.php?title=%ED%9D%90%EB%B0%A4%EB%AC%B4%EB%A5%B4&amp;action=edit&amp;redlink=1" title="흐밤무르 (없는 문서)">흐밤무르</a>(Hvammur)의 <a href="/wiki/%EC%8A%A4%ED%88%AC%EB%A5%BC%EB%A3%BD" title="스투를룽">스투를룽 씨족</a>(Sturlungar)</li>
</ul>
<p>아이슬란드의 군장들이 서로 치고받고 싸우는 동안 노르웨이 국왕 <a class="new" href="/w/index.php?title=%ED%95%98%EC%BD%98_%ED%95%98%EC%BD%94%EB%82%98%EB%A5%B4%EC%86%90&amp;action=edit&amp;redlink=1" title="하콘 하코나르손 (없는 문서)">하콘 4세</a>가 아이슬란드에 대한 영향력 확대를 도모했고, 최종적으로 각 군장들이 하나씩 노르웨이에 복속되어 중세 아이슬란드는 멸망했다.</p>
