<p><b>스티븐 "스티브" 맥매너먼</b>(Steven "Steve" McManaman, <a href="/wiki/1972%EB%85%84" title="1972년">1972년</a> <a href="/wiki/2%EC%9B%94_11%EC%9D%BC" title="2월 11일">2월 11일</a> ~ )은 은퇴한 잉글랜드의 전직 <a href="/wiki/%EC%B6%95%EA%B5%AC" title="축구">축구</a> 선수로, 과거 <a href="/wiki/%EB%AF%B8%EB%93%9C%ED%95%84%EB%8D%94" title="미드필더">미드필더</a>, <a class="mw-redirect mw-disambig" href="/wiki/%EC%9C%99%EC%96%B4" title="윙어">윙어</a>, 혹은 <a href="/wiki/%ED%94%8C%EB%A0%88%EC%9D%B4%EB%A9%94%EC%9D%B4%EC%BB%A4" title="플레이메이커">플레이메이커</a>로 뛰었었다. 20세기의 가장 성공적인 두 클럽, <a href="/wiki/%EB%A6%AC%EB%B2%84%ED%92%80_FC" title="리버풀 FC">리버풀 FC</a>, <a href="/wiki/%EB%A0%88%EC%95%8C_%EB%A7%88%EB%93%9C%EB%A6%AC%EB%93%9C_CF" title="레알 마드리드 CF">레알 마드리드 CF</a>과 더불어, 말년에 <a href="/wiki/%EB%A7%A8%EC%B2%B4%EC%8A%A4%ED%84%B0_%EC%8B%9C%ED%8B%B0_FC" title="맨체스터 시티 FC">맨체스터 시티 FC</a>에서 뛰었던 그는 트로피 획득 숫자로는 해외에서 가장 성공한 잉글랜드 선수이다.</p>
<p>맥매너먼은 영국인들 중 최초로 <a href="/wiki/UEFA_%EC%B1%94%ED%94%BC%EC%96%B8%EC%8A%A4%EB%A6%AC%EA%B7%B8" title="UEFA 챔피언스리그">UEFA 챔피언스리그</a>에서 두번의 우승을 거두었으며, 비잉글랜드 클럽에서 챔피언스리그를 우승한 최초의 잉글랜드인 선수이다. 2008년, 그는 영국의 톱10 축구선수 중에서 <a href="/wiki/%EC%BC%80%EB%B9%88_%ED%82%A4%EA%B1%B4" title="케빈 키건">케빈 키건</a>과 <a href="/wiki/%EC%A1%B4_%EC%B0%B0%EC%8A%A4" title="존 찰스">존 찰스</a>에 이어 3위에 랭크되었다.</p>
<p>그의 1990년대 말 <a href="/wiki/%EB%A0%88%EC%95%8C_%EB%A7%88%EB%93%9C%EB%A6%AC%EB%93%9C_CF" title="레알 마드리드 CF">레알 마드리드</a>로의 이적은 <a class="mw-redirect" href="/wiki/%EB%B3%B4%EC%8A%A4%EB%A7%8C_%EB%A3%B0" title="보스만 룰">보스만 룰</a>에 의한 자유 이적이었는데, 이 계약에서 그는 1999년에서 2001년까지 가장 많은 연봉을 받는 영국인 선수가 되었다.</p>
<p>2005년 은퇴 후 그는 <a class="new" href="/w/index.php?title=%EC%84%B8%ED%83%84%ED%83%80_%EC%8A%A4%ED%8F%AC%EC%B8%A0&amp;action=edit&amp;redlink=1" title="세탄타 스포츠 (없는 문서)">세탄타 스포츠</a>의 축구 평론가겸 <a class="mw-redirect" href="/wiki/%EA%B3%A8!_2:_%EA%BF%88%EC%9D%84_%ED%96%A5%ED%95%B4_%EB%9B%B0%EC%96%B4%EB%9D%BC" title="골! 2: 꿈을 향해 뛰어라">골! 2</a>의 공동 프로듀서로 일하였다. 그는 2010년 <a href="/wiki/ESPN" title="ESPN">ESPN</a>의 <a href="/wiki/2010%EB%85%84_FIFA_%EC%9B%94%EB%93%9C%EC%BB%B5" title="2010년 FIFA 월드컵">2010 남아공 월드컵</a>의 분석가로 일하였다. 현재 <a class="new" href="/w/index.php?title=%EC%9D%B4%EC%95%88_%EB%8B%A4%ED%81%AC&amp;action=edit&amp;redlink=1" title="이안 다크 (없는 문서)">이안 다크</a>와 공동으로 영국과 미국의 <a class="mw-redirect" href="/wiki/FA_%ED%94%84%EB%A6%AC%EB%AF%B8%EC%96%B4%EB%A6%AC%EA%B7%B8" title="FA 프리미어리그">FA 프리미어리그</a>, <a class="mw-redirect mw-disambig" href="/wiki/FA_%EC%BB%B5" title="FA 컵">FA 컵</a>, <a href="/wiki/%EB%A9%94%EC%9D%B4%EC%A0%80_%EB%A6%AC%EA%B7%B8_%EC%82%AC%EC%BB%A4" title="메이저 리그 사커">메이저 리그 사커</a>의 분석가로 활동하고 있다.</p>

<h3><span class="mw-headline" id=".EB.A0.88.EC.95.8C_.EB.A7.88.EB.93.9C.EB.A6.AC.EB.93.9C_CF">레알 마드리드 CF</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%8A%A4%ED%8B%B0%EB%B8%8C_%EB%A7%A5%EB%A7%A4%EB%84%88%EB%A8%BC&amp;action=edit&amp;section=3" title="부분 편집: 레알 마드리드 CF">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<h4><span class="mw-headline" id="1999-2000.EC.8B.9C.EC.A6.8C.EC.9D.98_.EC.84.B1.EA.B3.B5">1999-2000시즌의 성공</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%8A%A4%ED%8B%B0%EB%B8%8C_%EB%A7%A5%EB%A7%A4%EB%84%88%EB%A8%BC&amp;action=edit&amp;section=4" title="부분 편집: 1999-2000시즌의 성공">편집</a><span class="mw-editsection-bracket">]</span></span></h4>
<p>1999년 7월 1일, <a href="/wiki/%EB%A6%AC%EB%B2%84%ED%92%80_FC" title="리버풀 FC">리버풀 FC</a>에서 366경기 출장과 66골을 기록한 후, 맥매너먼은 <a href="/wiki/%EA%B1%B0%EC%8A%A4_%ED%9E%88%EB%94%A9%ED%81%AC" title="거스 히딩크">거스 히딩크</a>가 경질되기 전 마지막으로 영입되었다 (또한 <a class="new" href="/w/index.php?title=%EB%A1%9C%EB%A0%8C%EC%86%8C_%EC%82%B0%EC%8A%A4&amp;action=edit&amp;redlink=1" title="로렌소 산스 (없는 문서)">로렌소 산스</a> 회장의 마지막 영입이었다). 맥매너먼이 들어오기 전, 레알마드리드는 심각한 부채문제를 수습하고 있었으며, <a href="/wiki/%EB%9D%BC%EC%9A%B8_%EA%B3%A4%EC%82%B4%EB%A0%88%EC%8A%A4" title="라울 곤살레스">라울 곤살레스</a>는 기자회견에서 다음과 같이 말하였다: "드레싱 룸은 거짓말의 오물통이고, 배신과 속삭임...나는 스티브 맥매너먼과 같은 새 선수들이 이 클럽으로 들어오는 것에 유감한다. 맥매너먼은 세계의 최고 클럽들 중 하나로 들어오는 것은 실수이다." 클럽은 <a href="/wiki/%EC%A1%B4_%ED%86%A0%EC%83%A5" title="존 토샥">존 토샥</a>을 감독으로 새로 내정하였고, 이미 <a href="/wiki/%EB%8B%A4%EB%B3%B4%EB%A5%B4_%EC%8A%88%EC%BC%80%EB%A5%B4" title="다보르 슈케르">다보르 슈케르</a>, <a class="mw-redirect" href="/wiki/%ED%94%84%EB%A0%88%EB%93%9C%EB%9D%BD_%EB%AF%B8%EC%95%BC%ED%86%A0%EB%B9%84%EC%B9%98" title="프레드락 미야토비치">프레드락 미야토비치</a>, <a href="/wiki/%ED%81%AC%EB%A6%AC%EC%8A%A4%ED%8B%B0%EC%95%88_%ED%8C%8C%EB%88%84%EC%B9%98" title="크리스티안 파누치">크리스티안 파누치</a>와 같은 최고 선수를 부채문제를 해결하기 위해 벌써 매각한 상태였다. 맥매너먼이 들어온 직후 <a href="/wiki/%ED%81%B4%EB%9D%BC%EB%A0%8C%EC%84%9C_%EC%84%B8%EC%9D%B4%EB%8F%84%EB%A5%B4%ED%94%84" title="클라렌서 세이도르프">클라렌서 세이도르프</a> 또한 팀을 떠났다. 맥매너먼은 1980년대 <a class="new" href="/w/index.php?title=%EB%A1%9C%EB%A6%AC_%EC%BB%A4%EB%8B%9D%EC%97%84&amp;action=edit&amp;redlink=1" title="로리 커닝엄 (없는 문서)">로리 커닝엄</a>에 이어 레알마드리드의 두 번째 잉글랜드인 선수가 되었다. 그는 또한 <a href="/wiki/%EA%B2%8C%EB%A6%AC_%EB%A6%AC%EB%84%A4%EC%BB%A4" title="게리 리네커">게리 리네커</a>가 <a class="mw-redirect" href="/wiki/%EC%97%90%EB%B2%84%ED%8A%BC_FC" title="에버튼 FC">에버튼 FC</a>에서 <a href="/wiki/FC_%EB%B0%94%EB%A5%B4%EC%85%80%EB%A1%9C%EB%82%98" title="FC 바르셀로나">FC 바르셀로나</a>로 둥지를 옮긴 이후로 가장 주목받은 라 리가의 잉글랜드인 선수가 되었다. 그러한 우려에도 불구하고, 그는 <i>로스 메렝게스</i>를 위해 시즌 초반 경기에서 3골을 넣고 여러 골을 어시스트하며, <a href="/wiki/%EC%97%90%EC%8A%A4%ED%83%80%EB%94%94%EC%98%A4_%EC%82%B0%ED%8B%B0%EC%95%84%EA%B3%A0_%EB%B2%A0%EB%A5%B4%EB%82%98%EB%B2%A0%EC%9A%B0" title="에스타디오 산티아고 베르나베우">산티아고 베르나베우</a>의 팬들의 호응을 얻었다.</p>
<p>1999년 8월 22일 맥매너먼은 <a href="/wiki/RCD_%EB%A7%88%EC%9A%94%EB%A5%B4%EC%B9%B4" title="RCD 마요르카">RCD 마요르카</a>의 2-1로 승리한 <a class="mw-redirect" href="/wiki/%EC%9D%B4%EB%B2%A0%EB%A1%9C%EC%8A%A4%ED%83%80_%EC%97%90%EC%8A%A4%ED%83%80%EB%94%94%EC%98%A4" title="이베로스타 에스타디오">손 모이스 경기장</a> 원정경기에서 레알마드리드에서의 데뷔전을 치렀고, 이 경기에서 <a href="/wiki/%ED%8E%98%EB%A5%B4%EB%82%9C%EB%8F%84_%EB%AA%A8%EB%A6%AC%EC%97%94%ED%85%8C%EC%8A%A4" title="페르난도 모리엔테스">페르난도 모리엔테스</a>가 로스타임에서 헤딩하는 결승골을 어시스트했다. 그는 첫 홈경기로부터 일주일 뒤 첫골을 넣었고, 이 경기는 <a href="/wiki/CD_%EB%88%84%EB%A7%8C%EC%8B%9C%EC%95%84" title="CD 누만시아">CD 누만시아</a>와의 홈 경기였고 팀은 4-1로 승리하였다.</p>
<p>1999년 12월에서 2000년 1월까지, 맥매너먼의 마드리드는 논란의 대상이 되는 <a href="/wiki/%EB%A6%AC%EC%9A%B0%EB%8D%B0%EC%9E%90%EB%84%A4%EC%9D%B4%EB%A3%A8" title="리우데자네이루">리우데자네이루</a>에서 개최한 1대 <a class="new" href="/w/index.php?title=FIFA_%ED%81%B4%EB%9F%BD%EC%9B%94%EB%93%9C%EC%BB%B5&amp;action=edit&amp;redlink=1" title="FIFA 클럽월드컵 (없는 문서)">FIFA 클럽월드컵</a>에참여하였고, 그는 네카사와의 경기에서 인저리 타임 찬스를 날렸고, <a class="mw-redirect" href="/wiki/%ED%8E%98%EB%84%90%ED%8B%B0%ED%82%A5" title="페널티킥">페널티킥</a>을 허용하여 3위로 마감하였다. 그 후 팀은 스페인으로 복귀하였고, 1999년 11월 <a href="/wiki/%EC%A1%B4_%ED%86%A0%EC%83%A5" title="존 토샥">존 토샥</a>을 대신하여 들어온 <a href="/wiki/%EB%B9%84%EC%84%BC%ED%85%8C_%EB%8D%B8_%EB%B3%B4%EC%8A%A4%EC%BC%80" title="비센테 델 보스케">비센테 델 보스케</a>의 새 리더쉽 하에 2000년 챔피언스리그 결승까지 올라왔다. 맥매너먼은 <a href="/wiki/%ED%8E%98%EB%A5%B4%EB%82%9C%EB%8F%84_%EB%A0%88%EB%8F%88%EB%8F%84" title="페르난도 레돈도">페르난도 레돈도</a>와의 좋은 호흡을 구성하며, 여러 경기에서 최우수 선수로 평가받았고, 전 시즌 우승팀 <a href="/wiki/%EB%A7%A8%EC%B2%B4%EC%8A%A4%ED%84%B0_%EC%9C%A0%EB%82%98%EC%9D%B4%ED%8B%B0%EB%93%9C_FC" title="맨체스터 유나이티드 FC">맨체스터 유나이티드 FC</a>와 <a href="/wiki/FC_%EB%B0%94%EC%9D%B4%EC%97%90%EB%A5%B8_%EB%AE%8C%ED%97%A8" title="FC 바이에른 뮌헨">FC 바이에른 뮌헨</a>을 차례로 격파하며 UEFA 챔피언스리그 결승에 진출하였다. 그는 <a href="/wiki/%EC%8A%A4%ED%83%80%EB%93%9C_%EB%93%9C_%ED%94%84%EB%9E%91%EC%8A%A4" title="스타드 드 프랑스">스타드 드 프랑스</a>에서 개최한 <a href="/wiki/2000_UEFA_%EC%B1%94%ED%94%BC%EC%96%B8%EC%8A%A4%EB%A6%AC%EA%B7%B8_%EA%B2%B0%EC%8A%B9%EC%A0%84" title="2000 UEFA 챔피언스리그 결승전">2000년 UEFA 챔피언스리그 결승전</a>에서 최고 정점을 찍었고, 이 경기에서 발리슛을 넣으며 결승에 처음 진출한 <a href="/wiki/%EB%B0%9C%EB%A0%8C%EC%8B%9C%EC%95%84_CF" title="발렌시아 CF">발렌시아 CF</a>를 3-0으로 완파하고 팀의 8번째 빅이어를 들어올리는 1등공신이 되었다. 이 챔피언스리그 우승으로 잉글랜드인으로는 최초로 비잉글랜드 클럽에서 빅이어를 들어올린 첫 번째 선수가 되었다.</p>
<h4><span class="mw-headline" id="2000-01_.EC.8B.9C.EC.A6.8C">2000-01 시즌</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%8A%A4%ED%8B%B0%EB%B8%8C_%EB%A7%A5%EB%A7%A4%EB%84%88%EB%A8%BC&amp;action=edit&amp;section=5" title="부분 편집: 2000-01 시즌">편집</a><span class="mw-editsection-bracket">]</span></span></h4>
<p>유러피언컵 결승에서의 성과로 마드리드에서의 가치있는 선수로 평가받은 이후, <a href="/wiki/%ED%94%8C%EB%A1%9C%EB%A0%8C%ED%8B%B0%EB%85%B8_%ED%8E%98%EB%A0%88%EC%8A%A4" title="플로렌티노 페레스">플로렌티노 페레스</a>가 회장으로 취임하고, <a href="/wiki/%EB%A3%A8%EC%9D%B4%EC%8A%A4_%ED%94%BC%EA%B5%AC" title="루이스 피구">루이스 피구</a>가 <a href="/wiki/FC_%EB%B0%94%EB%A5%B4%EC%85%80%EB%A1%9C%EB%82%98" title="FC 바르셀로나">FC 바르셀로나</a>로부터 클럽의 최고금액으로 이적을 하면서, <a href="/wiki/%ED%8E%98%EB%A5%B4%EB%82%9C%EB%8F%84_%EB%A0%88%EB%8F%88%EB%8F%84" title="페르난도 레돈도">페르난도 레돈도</a>, <a href="/wiki/%EB%8B%88%EC%BD%9C%EB%9D%BC_%EC%95%84%EB%84%AC%EC%B9%B4" title="니콜라 아넬카">니콜라 아넬카</a>, <a href="/wiki/%ED%81%AC%EB%A6%AC%EC%8A%A4%ED%8B%B0%EC%95%99_%EC%B9%B4%EB%9E%91%EB%B5%88" title="크리스티앙 카랑뵈">크리스티앙 카랑뵈</a>가 부채를 갚기 위해 방출되었으며, 2000-01시즌 시작 직전 잉여자원으로 전락되어 매각대상이 되었다. 그는 다른 선수와 다르게 잔류를 원했지만, 델 보스케 감독은 "금년 적은 기회가 주어진다"고 보고되었다. 그리고 그가 새 포지션에 적응을 하지 못하는 경우, 클럽은 그 다음해에 그를 스쿼드 등록을 하지 못하도록 하였다. 부채에 허덕이고 이미 £37M을 피구를 영입하는데 쓴 클럽은 $4.5M의 연봉을 삭감하겠다고 보드진은 민감하게 반응하였다. 그러나, 2004년까지 계약된 그는 다른 팀으로 떠나 주전자리를 차지하는 대신 팀에 잔류하여 주전자리 탈환에 힘썼다.</p>
<p>레알마드리드는 후에 <a href="/wiki/%EB%AF%B8%EB%93%A4%EC%A6%88%EB%B8%8C%EB%9F%AC_FC" title="미들즈브러 FC">미들즈브러 FC</a>의 £11M에 합의하였고 그 뒤 <a href="/wiki/%EC%B2%BC%EC%8B%9C_FC" title="첼시 FC">첼시 FC</a>의 £12M에 <a href="/wiki/%ED%86%A0%EB%A0%88_%EC%95%88%EB%93%9C%EB%A0%88_%ED%94%8C%EB%A1%9C" title="토레 안드레 플로">토레 안드레 플로</a>를 얹힌 이적에도 합의한적 있으나, 그는 강하게 부인하였다. 스카이스포츠는 2000년 8월에 8개의 클럽이 그를 영입하기 위해 레이스에 가담하였으며, <a href="/wiki/%EB%A7%A8%EC%B2%B4%EC%8A%A4%ED%84%B0_%EC%9C%A0%EB%82%98%EC%9D%B4%ED%8B%B0%EB%93%9C_FC" title="맨체스터 유나이티드 FC">맨체스터 유나이티드 FC</a>, <a class="mw-redirect" href="/wiki/%ED%8C%8C%EB%A5%B4%EB%A7%88_FC" title="파르마 FC">파르마 FC</a>, <a href="/wiki/ACF_%ED%94%BC%EC%98%A4%EB%A0%8C%ED%8B%B0%EB%82%98" title="ACF 피오렌티나">ACF 피오렌티나</a>도 포함되어 있었다. 그러나 그는 스페인에서의 성공을 원하였다. 그의 의지는 <a href="/wiki/%EC%97%98_%EB%AC%B8%EB%8F%84" title="엘 문도">엘 문도</a>에서 시행한 투표에서 90%의 팬이 맥매너먼의 잔류를 원한다고 하였으며, <a href="/wiki/%EB%AF%B8%EC%B2%BC_%EC%82%B4%EA%B0%80%EB%8F%84" title="미첼 살가도">미첼 살가도</a>는 맥매너먼이 배신을 당하였다며 감독을 비난하였고, £8M으로 <a href="/wiki/SS_%EB%9D%BC%EC%B9%98%EC%98%A4" title="SS 라치오">SS 라치오</a> 이적까지 제안되었다. 맥매너먼은 2000-01시즌 9월 3-3으로 비긴 <a href="/wiki/%EB%A7%90%EB%9D%BC%EA%B0%80_CF" title="말라가 CF">말라가 CF</a>전에서 시즌 처음으로 교체출장하였다. 맥매너먼은 10월에 감독측을 상대로 승리하였으며, 팀의 경기 2/3을 뛰었고, 시즌 후반에는 주전자리를 찾았고, 잉글랜드인으로는 해외에서 뛴 최고의 선수로써 각인되었다. 맥매너먼은 2번째 시즌에 챔피언스리그와 라리가 2관왕을 노렸지만, 챔피언스리그는 준결승에서 바이에른 뮌헨에게 패하였지만, <a class="mw-redirect" href="/wiki/%EB%8D%B0%ED%8F%AC%EB%A5%B4%ED%8B%B0%EB%B3%B4_%EB%9D%BC_%EC%BD%94%EB%A3%A8%EB%83%90" title="데포르티보 라 코루냐">데포르티보 라 코루냐</a>와 7점 차이로 4년만의 리그 타이틀을 탈환하였고, 팀의 통산 28번째 라리가 우승이었다.</p>
<h4><span class="mw-headline" id=".EA.B0.88.EB.9D.BC.ED.8B.B0.EC.BD.94_.EC.A0.95.EC.B1.85">갈라티코 정책</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%8A%A4%ED%8B%B0%EB%B8%8C_%EB%A7%A5%EB%A7%A4%EB%84%88%EB%A8%BC&amp;action=edit&amp;section=6" title="부분 편집: 갈라티코 정책">편집</a><span class="mw-editsection-bracket">]</span></span></h4>
<p>맥매너먼은 세계 최고의 선수인 <a href="/wiki/%EB%A3%A8%EC%9D%B4%EC%8A%A4_%ED%94%BC%EA%B5%AC" title="루이스 피구">루이스 피구</a>, <a href="/wiki/%EC%A7%80%EB%84%A4%EB%94%98_%EC%A7%80%EB%8B%A8" title="지네딘 지단">지네딘 지단</a>, <a href="/wiki/%ED%98%B8%EB%82%98%EC%9A%B0%EB%91%90" title="호나우두">호나우두</a>가 매년마다 한명씩 들어오며 압박을 받았고, 클럽의 갈라티코 정책으로 그의 경기 출장시간이 매년마다 줄어들었다. 이 당시에는 그의 뚝심과 낙천적인 성격으로 버틸수 있었지만, 출장시간을 줄여야만 했다. 2002년, 맥매너먼은 <a href="/wiki/FC_%EC%9D%B8%ED%85%8C%EB%A5%B4%EB%82%98%EC%B9%98%EC%98%A4%EB%82%A0%EB%A0%88_%EB%B0%80%EB%9D%BC%EB%85%B8" title="FC 인테르나치오날레 밀라노">FC 인테르나치오날레 밀라노</a>에 <a href="/wiki/%ED%98%B8%EB%82%98%EC%9A%B0%EB%91%90" title="호나우두">호나우두</a>와의 스왑딜을 진행할 뻔했다. 그의 뚝심은 <a class="mw-redirect" href="/wiki/%EB%9D%BC%EC%9A%B8_%EA%B3%A4%EC%9E%98%EB%A0%88%EC%8A%A4" title="라울 곤잘레스">라울 곤잘레스</a>, <a href="/wiki/%EC%A7%80%EB%84%A4%EB%94%98_%EC%A7%80%EB%8B%A8" title="지네딘 지단">지네딘 지단</a>, <a href="/wiki/%EA%B5%AC%ED%8B%B0_(%EC%B6%95%EA%B5%AC_%EC%84%A0%EC%88%98)" title="구티 (축구 선수)">구티</a>, <a href="/wiki/%EC%9D%B4%EB%B0%98_%EC%97%98%EA%B2%8C%EB%9D%BC" title="이반 엘게라">이반 엘게라</a>, <a href="/wiki/%EB%A3%A8%EC%9D%B4%EC%8A%A4_%ED%94%BC%EA%B5%AC" title="루이스 피구">루이스 피구</a>, <a href="/wiki/%ED%98%B8%EB%82%98%EC%9A%B0%EB%91%90" title="호나우두">호나우두</a>의 호응을 얻고, 기자회견에서 종종 받쳐주었다. 2001년의 엘 파이스에 의하면, <a href="/wiki/%EB%A0%88%EC%95%8C_%EC%98%A4%EB%B9%84%EC%97%90%EB%8F%84" title="레알 오비에도">레알 오비에도</a>전에서 원더골을 넣자 팬들은 '흰 손수건'을 흔들었다.</p>
<p>여러 스페인의 언론에서는 비록 선수들이 실력에 따라서 차출되기도 하였지만 매출이 떨어진다는 이유로 맥매너먼을 비롯하여 여러 선수들은 마케팅과 예산 정책으로 인한 "희생양"이 되었다고 하였다. 이에 대하여 맥매너먼은 절대로 갈라티코 정책에 대해 혹평하지 않았지만, 그의 자서전인 "엘 마카" (윌리엄 힐 올해의 스포츠 북으로 수상)에, "레알마드리드의 디즈니화"라고 그가 떠날 때에 비난하였다. 이러한 정책은 절대로 클럽이 최고에 도달한 적이 없다고 하였고, 그 의견은 지금까지도 이어지고 있다.</p>
<h4><span class="mw-headline" id="2001-02:_.EB.91.90_.EB.B2.88.EC.A7.B8_.EB.B9.85.EC.9D.B4.EC.96.B4">2001-02: 두 번째 빅이어</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%8A%A4%ED%8B%B0%EB%B8%8C_%EB%A7%A5%EB%A7%A4%EB%84%88%EB%A8%BC&amp;action=edit&amp;section=7" title="부분 편집: 2001-02: 두 번째 빅이어">편집</a><span class="mw-editsection-bracket">]</span></span></h4>
<p>결국 <a href="/wiki/%ED%94%8C%EB%A1%9C%EB%A0%8C%ED%8B%B0%EB%85%B8_%ED%8E%98%EB%A0%88%EC%8A%A4" title="플로렌티노 페레스">플로렌티노 페레스</a> 회장은 "항상 나의 클럽에서 자리를 차지할 수 있는 사람"이라 동의하였고, <a class="mw-redirect" href="/wiki/%EC%9A%94%ED%95%9C_%ED%81%AC%EB%A3%A8%EC%9D%B4%ED%94%84" title="요한 크루이프">요한 크루이프</a>는 맥매너먼을 "모든이들의 피치에서 최고의 파트너"로써 "갈라티코"에서 가장 유용한 선수로 묘사하였다. 이러한 점은 2001/02 시즌 , 레알마드리드의 100주년 기념으로 FIFA 월드 XI와의 경기에서 주장완장을 차고 경기하였다. 그의 레알에서 제2전성기 또한 이 시즌에 이루어졌다. 2002 UEFA 챔피언스리그 준결승에서 <a href="/wiki/FC_%EB%B0%94%EB%A5%B4%EC%85%80%EB%A1%9C%EB%82%98" title="FC 바르셀로나">FC 바르셀로나</a>를 <a class="mw-redirect" href="/wiki/%EB%88%84%EC%BA%84%ED%94%84" title="누캄프">누캄프</a>에서 상대한 <a href="/wiki/2002%EB%85%84" title="2002년">2002년</a> <a href="/wiki/4%EC%9B%94_23%EC%9D%BC" title="4월 23일">4월 23일</a>이었다. 맥매너먼은 후반전에 교체되어 로스타임에 <a class="new" href="/w/index.php?title=%ED%94%8C%EB%9D%BC%EB%B9%84%EC%9A%B0_%EC%BD%98%EC%84%B8%EC%9D%B4%EC%83%81&amp;action=edit&amp;redlink=1" title="플라비우 콘세이상 (없는 문서)">플라비우 콘세이상</a>의 공을 받아, 골키퍼 <a class="new" href="/w/index.php?title=%EB%A1%9C%EB%B2%A0%EB%A5%B4%ED%86%A0_%EB%B3%B4%EB%82%98%EB%85%B8&amp;action=edit&amp;redlink=1" title="로베르토 보나노 (없는 문서)">로베르토 보나노</a> 키를 넘기고 2-0 쐐기골을 박아 1차전에서 유리한 고지를 점령하였다. 이 기념비적인 승리는 레알의 1993년이후 누캄프원정에서의 첫 승리였고, <a href="/wiki/%EA%B8%80%EB%9E%98%EC%8A%A4%EA%B3%A0" title="글래스고">글래스고</a>의 <a class="mw-redirect" href="/wiki/%ED%96%84%EB%93%A0_%ED%8C%8C%ED%81%AC" title="햄든 파크">햄든 파크</a>에서 개최되는 2002년 챔피언스리그 결승에 진출하였다. 결승전에서 <a href="/wiki/%EB%A3%A8%EC%9D%B4%EC%8A%A4_%ED%94%BC%EA%B5%AC" title="루이스 피구">루이스 피구</a>와 교체되어 투입되었고, <a href="/wiki/%EB%B0%94%EC%9D%B4%EC%96%B4_04_%EB%A0%88%EB%B2%84%EC%BF%A0%EC%A0%A0" title="바이어 04 레버쿠젠">바이어 04 레버쿠젠</a>을 상대로 2-1로 승리한 뒤, 챔피언스리그 2번째 메달을 확보하였다.</p>
<h4><span class="mw-headline" id="2002-03:_.EB.A7.88.EC.A7.80.EB.A7.89_.EC.8B.9C.EC.A6.8C">2002-03: 마지막 시즌</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%8A%A4%ED%8B%B0%EB%B8%8C_%EB%A7%A5%EB%A7%A4%EB%84%88%EB%A8%BC&amp;action=edit&amp;section=8" title="부분 편집: 2002-03: 마지막 시즌">편집</a><span class="mw-editsection-bracket">]</span></span></h4>
<p>맥매너먼은 시즌 초반에 경기 막판 교체로 사용되었고, 새로 영입한 호나우두의 골들을 어시스트한 뒤, 간간히 선발로 등장하였다. 그가 선발로 출장한 경기 중 챔피언스리그에서 <a class="mw-redirect" href="/wiki/AEK_%EC%95%84%ED%85%8C%EB%84%A4" title="AEK 아테네">AEK 아테네</a>를 상대로 골을 뽑아내었다. 맥매너먼은 또한 <a class="mw-redirect" href="/wiki/%EC%98%AC%EB%93%9C_%ED%8A%B8%EB%9E%98%ED%8F%AC%EB%93%9C" title="올드 트래포드">올드 트래포드</a>에서 맨체스터 유나이티드에게 4-3으로 패한 명승부에도 선발로 출전하였고, 동료 호나우두는 해트트릭을 기록하였다.</p>
<p>그러나, 불과 21경기에 출장하였고 그 중에 9경기만 선발이고, 라 리가에 15경기 출장하였고, 1군경기와 국제경기 출장이 부족한 맥매너먼이 그의 줄어들은 임무를 가지고도 스페인에 잔류하는 것에 대하여 의혹을 제기하였다. 이러한 점은 잉글랜드의 언론에서, "스페인에 머무는 동안 맥매너먼은 언론에 의해 묻혔다" 라고 보도하였다.</p>
<p>그에도 불구하고, 맥매너먼은 두 번째 리그 메달을 받았고, 일본에서 주최한 <a href="/wiki/%EC%9D%B8%ED%84%B0%EC%BD%98%ED%8B%B0%EB%84%A8%ED%84%B8%EC%BB%B5_(%EC%B6%95%EA%B5%AC)" title="인터콘티넨털컵 (축구)">인터콘티넨털컵</a>의 스쿼드에도 포함되었다. 4년간 그가 클럽에 머물면서 모은 트로피들은 잉글랜드인 선수로는 해외에서 가장 성공한 선수로 평가된다. 더 나아가 2001년 <a class="mw-disambig" href="/wiki/%ED%8F%AC%EB%B3%B4%EC%8A%A4" title="포보스">포보스</a> 지는 세계에서 6번째로 소득이 많은 축구선수로 기록되었고, 약 €15M를 레알마드리드에서의 4년간 얻었다.</p>
<p>같은 잉글랜드인 <a href="/wiki/%EB%8D%B0%EC%9D%B4%EB%B9%84%EB%93%9C_%EB%B2%A0%EC%BB%B4" title="데이비드 베컴">데이비드 베컴</a>의 영입은 맥매너먼이 레알 마드리드에서 짐을 싸도록 만들었다. 델보스케 감독이 리그 29번째 타이틀을 딴지 24시간 이후에도 2003-04의 프리시즌 기간동안 팀에 잔류하였지만, <a href="/wiki/%EC%B9%B4%EB%A5%BC%EB%A3%A8%EC%8A%A4_%EC%BC%80%EC%9D%B4%EB%A1%9C%EC%8A%A4" title="카를루스 케이로스">카를루스 케이로스</a>가 새 감독으로 내정됨에 따라, 맥매너먼은 데이비드 베컴이 정착 성공을 위해 잔류를 원함에도 불구하고 떠났다.</p>
<p>맥 마나만이 클럽을 떠난 지 4년 후, 전 감독 델 보스케는 스페인의 신문, AS 지, 마르카 지를 통해 맥매너먼은 <a href="/wiki/%EC%A0%9C%EB%A0%88%EB%AF%B8_%EC%9D%80%EC%A7%80%ED%83%80%ED%94%84" title="제레미 은지타프">제레미 은지타프</a>와 더불어 그의 스쿼드에서 가장 중요한 선수였으며, "팀을 하나로 융합시켰다"라고 회고하였다.</p>
