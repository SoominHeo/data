<p><b>스티브 지소와의 해저 생활</b>(The Life Aquatic with Steve Zissou)은 2004년에 <a href="/wiki/%EC%9B%A8%EC%8A%A4_%EC%95%A4%EB%8D%94%EC%8A%A8" title="웨스 앤더슨">웨스 앤더슨</a>이 공동 각본과 감독을 맡은 <a href="/wiki/%EC%BD%94%EB%AF%B8%EB%94%94_%EB%93%9C%EB%9D%BC%EB%A7%88" title="코미디 드라마">코미디 드라마</a> 영화이다.<sup class="reference" id="cite_ref-2"><a href="#cite_note-2">[2]</a></sup><sup class="reference" id="cite_ref-3"><a href="#cite_note-3">[3]</a></sup> 앤더슨 감독의 4번째 장편 영화이며, 2004년 크리스마스에 미국에서 개봉했다. 앤더슨과 <a href="/wiki/%EB%85%B8%EC%95%84_%EB%B0%94%EC%9B%80%EB%B0%B1" title="노아 바움백">노아 바움백</a>이 각본을 썼고 <a href="/wiki/%EB%82%98%ED%8F%B4%EB%A6%AC" title="나폴리">나폴리</a>, <a href="/wiki/%ED%8F%B0%EC%B0%A8" title="폰차">폰차</a>, <a href="/wiki/%EC%9D%B4%ED%83%88%EB%A6%AC%EC%95%84_%EB%A6%AC%EB%B9%84%EC%97%90%EB%9D%BC" title="이탈리아 리비에라">이탈리아 리비에라</a>에서 촬영하였다.</p>
<p>영화 배우 <a href="/wiki/%EB%B9%8C_%EB%A8%B8%EB%A6%AC" title="빌 머리">빌 머리</a>는 자신의 파트너를 먹어치운 "표범 <a href="/wiki/%EC%83%81%EC%96%B4" title="상어">상어</a>"(Jaguar shark)에게 철저한 복수를 착수한 괴짜 <a href="/wiki/%ED%95%B4%EC%96%91%ED%95%99" title="해양학">해양학자</a> 지소 역을 맡았다. 지소는 프랑스 출신의 다이빙 개척자 <a href="/wiki/%EC%9E%90%ED%81%AC_%EC%BF%A0%EC%8A%A4%ED%86%A0" title="자크 쿠스토">자크 쿠스토</a>(1910–1997)의 <a href="/wiki/%ED%8C%A8%EB%9F%AC%EB%94%94" title="패러디">패러디</a>이자 <a href="/wiki/%EC%98%A4%EB%A7%88%EC%A3%BC" title="오마주">오마주</a>이며, 이 영화를 헌정한 인물이다. <a href="/wiki/%EC%BC%80%EC%9D%B4%ED%8A%B8_%EB%B8%94%EB%9E%80%EC%B3%87" title="케이트 블란쳇">케이트 블란쳇</a>, <a href="/wiki/%EC%9C%8C%EB%9F%BC_%EB%8D%94%ED%8F%AC" title="윌럼 더포">윌럼 더포</a>, <a href="/wiki/%EB%A7%88%EC%9D%B4%ED%81%B4_%EA%B0%AC%EB%B3%B8" title="마이클 갬본">마이클 갬본</a>, <a href="/wiki/%EC%A0%9C%ED%94%84_%EA%B3%A8%EB%93%9C%EB%B8%94%EB%A3%B8" title="제프 골드블룸">제프 골드블룸</a>, <a href="/wiki/%EC%95%A4%EC%A0%A4%EB%A6%AC%EC%B9%B4_%ED%9C%B4%EC%8A%A4%ED%84%B4" title="앤젤리카 휴스턴">앤젤리카 휴스턴</a>, <a href="/wiki/%EC%98%A4%EC%96%B8_%EC%9C%8C%EC%8A%A8" title="오언 윌슨">오언 윌슨</a>, <a class="new" href="/w/index.php?title=%EC%84%B8%EC%9A%B0_%EC%A1%B0%EB%A5%B4%EC%A7%80&amp;action=edit&amp;redlink=1" title="세우 조르지 (없는 문서)">세우 조르지</a>, <a href="/wiki/%EB%B2%84%EB%93%9C_%EC%BD%94%ED%8A%B8" title="버드 코트">버드 코트</a> 역시도 영화에 출연하였다.</p>
