<p><b>스티븐 존 "스티브" 피넌</b>(Stephen John "Steve" Finnan, <a href="/wiki/1976%EB%85%84" title="1976년">1976년</a> <a href="/wiki/4%EC%9B%94_24%EC%9D%BC" title="4월 24일">4월 24일</a>, <a href="/wiki/%EC%95%84%EC%9D%BC%EB%9E%9C%EB%93%9C" title="아일랜드">아일랜드</a> <a href="/wiki/%EB%A6%AC%EB%A8%B8%EB%A6%AD" title="리머릭">리머릭</a> ~ )은 <a href="/wiki/%EC%95%84%EC%9D%BC%EB%9E%9C%EB%93%9C" title="아일랜드">아일랜드</a>의 축구 선수이다. 피넌은 <a href="/wiki/FIFA_%EC%9B%94%EB%93%9C%EC%BB%B5" title="FIFA 월드컵">FIFA 월드컵</a>, <a href="/wiki/UEFA_%EC%B1%94%ED%94%BC%EC%96%B8%EC%8A%A4%EB%A6%AC%EA%B7%B8" title="UEFA 챔피언스리그">챔피언스리그</a>, <a class="mw-redirect" href="/wiki/UEFA%EC%BB%B5" title="UEFA컵">UEFA컵</a>, 그리고 <a href="/wiki/%EC%9E%89%EA%B8%80%EB%9E%9C%EB%93%9C" title="잉글랜드">잉글랜드</a>의 4개의 디비전과 잉글리시 콘퍼런스를 경험한 선수이다.</p>
