<p><b>스티븐 제라드</b> (<span style="font-size: smaller;"><a href="/wiki/%EC%98%81%EC%96%B4" title="영어">영어</a>:</span> <span lang="en" xml:lang="en">Steven George Gerrard</span>, <a href="/wiki/%EB%8C%80%EC%98%81_%EC%A0%9C%EA%B5%AD_%ED%9B%88%EC%9E%A5" title="대영 제국 훈장">MBE</a>, <a href="/wiki/1980%EB%85%84" title="1980년">1980년</a> <a href="/wiki/5%EC%9B%94_30%EC%9D%BC" title="5월 30일">5월 30일</a> ~ )는 <a href="/wiki/%EC%9E%89%EA%B8%80%EB%9E%9C%EB%93%9C" title="잉글랜드">잉글랜드</a>의 전 <a href="/wiki/%EC%B6%95%EA%B5%AC_%EC%84%A0%EC%88%98" title="축구 선수">축구 선수</a>이다. 그의 포지션은 <a class="mw-redirect" href="/wiki/%EC%A4%91%EC%95%99_%EB%AF%B8%EB%93%9C%ED%95%84%EB%8D%94" title="중앙 미드필더">중앙 미드필더</a>로, <a class="mw-redirect" href="/wiki/%EB%94%A5-%EB%9D%BC%EC%9E%89_%ED%94%8C%EB%A0%88%EC%9D%B4%EB%A9%94%EC%9D%B4%EC%BB%A4" title="딥-라잉 플레이메이커">딥-라잉 플레이메이커</a> 유형의 <a class="mw-redirect" href="/wiki/%EC%88%98%EB%B9%84%ED%98%95_%EB%AF%B8%EB%93%9C%ED%95%84%EB%8D%94" title="수비형 미드필더">수비형 미드필더</a>와 <a class="mw-redirect" href="/wiki/%EA%B3%B5%EA%B2%A9%ED%98%95_%EB%AF%B8%EB%93%9C%ED%95%84%EB%8D%94" title="공격형 미드필더">공격형 미드필더</a>, <a href="/wiki/%EB%AF%B8%EB%93%9C%ED%95%84%EB%8D%94#.EC.9C.99.EC.96.B4" title="미드필더">윙어</a>로도 뛸 수있다.<sup class="reference" id="cite_ref-1"><a href="#cite_note-1">[1]</a></sup><sup class="reference" id="cite_ref-2"><a href="#cite_note-2">[2]</a></sup> <a href="/wiki/1998%EB%85%84" title="1998년">1998년</a> 프로 선수로 데뷔를 하였고, <a href="/wiki/2000%EB%85%84" title="2000년">2000년</a>부터 주전으로 뛰며 <a href="/wiki/2003%EB%85%84" title="2003년">2003년</a>부터 <a href="/wiki/%EB%A6%AC%EB%B2%84%ED%92%80_FC" title="리버풀 FC">리버풀</a>의 주장을 역임하고 있었다. 그의 2회의 FA 컵, 3회의 <a class="mw-redirect" href="/wiki/%ED%92%8B%EB%B3%BC_%EB%A6%AC%EA%B7%B8_%EC%BB%B5" title="풋볼 리그 컵">풋볼 리그 컵</a>, 2회의 <a class="mw-redirect" href="/wiki/%EC%BB%A4%EB%AE%A4%EB%8B%88%ED%8B%B0_%EC%8B%A4%EB%93%9C" title="커뮤니티 실드">커뮤니티 실드</a>, 1회의 <a class="mw-redirect mw-disambig" href="/wiki/%EC%B1%94%ED%94%BC%EC%96%B8%EC%8A%A4_%EB%A6%AC%EA%B7%B8" title="챔피언스 리그">챔피언스 리그</a> , 1회의 <a class="mw-redirect" href="/wiki/UEFA_%EC%BB%B5" title="UEFA 컵">UEFA 컵</a>, 그리고 2회의 <a class="mw-redirect" href="/wiki/%EC%8A%88%ED%8D%BC_%EC%BB%B5" title="슈퍼 컵">슈퍼 컵</a> 우승의 커리어를 보유하고 있다.</p>
<p>제라드는 <a href="/wiki/2000%EB%85%84" title="2000년">2000년</a> <a href="/wiki/%EC%9E%89%EA%B8%80%EB%9E%9C%EB%93%9C_%EC%B6%95%EA%B5%AC_%EA%B5%AD%EA%B0%80%EB%8C%80%ED%91%9C%ED%8C%80" title="잉글랜드 축구 국가대표팀">잉글랜드 대표팀</a>으로 국가대표로 데뷔하였고, <a href="/wiki/UEFA_%EC%9C%A0%EB%A1%9C_2000" title="UEFA 유로 2000">UEFA 유로 2000</a>, <a href="/wiki/UEFA_%EC%9C%A0%EB%A1%9C_2004" title="UEFA 유로 2004">UEFA 유로 2004</a>에 출전뿐만 아니라, <a href="/wiki/2006%EB%85%84_FIFA_%EC%9B%94%EB%93%9C%EC%BB%B5" title="2006년 FIFA 월드컵">2006년 독일 월드컵</a>과 <a href="/wiki/2010%EB%85%84_FIFA_%EC%9B%94%EB%93%9C%EC%BB%B5" title="2010년 FIFA 월드컵">2010년 남아공 월드컵</a>에서 골을 기록하였다. 그는 <a href="/wiki/2010%EB%85%84_FIFA_%EC%9B%94%EB%93%9C%EC%BB%B5" title="2010년 FIFA 월드컵">2010년 남아공 월드컵</a>에 임시 주장을 맡은 이래,<sup class="reference" id="cite_ref-autogenerated1_3-0"><a href="#cite_note-autogenerated1-3">[3]</a></sup> <a href="/wiki/UEFA_%EC%9C%A0%EB%A1%9C_2012" title="UEFA 유로 2012">UEFA 유로 2012</a> 대회를 앞두고 새로 부임한 <a href="/wiki/%EB%A1%9C%EC%9D%B4_%ED%98%B8%EC%A7%80%EC%8A%A8" title="로이 호지슨">로이 호지슨</a> 감독으로부터 주장으로 정식 임명되었다.<sup class="reference" id="cite_ref-4"><a href="#cite_note-4">[4]</a></sup><sup class="reference" id="cite_ref-5"><a href="#cite_note-5">[5]</a></sup> 그는 또한 <a href="/wiki/2014%EB%85%84_FIFA_%EC%9B%94%EB%93%9C%EC%BB%B5" title="2014년 FIFA 월드컵">2014년 브라질 월드컵</a>에서 대표팀의 주장을 역임했다. 제라드는 <a href="/wiki/2012%EB%85%84" title="2012년">2012년</a> <a href="/wiki/%EC%98%81%EA%B5%AD" title="영국">영국</a> 선수로 100경기를 출장한 여섯 번째 선수가 되었다.<sup class="reference" id="cite_ref-6"><a href="#cite_note-6">[6]</a></sup> 2005년, 그는 발롱도르 브론즈상을 수상하였고, 다음 해에는 올해의 UEFA 최우수 선수상을 수상했다.</p>
<p>그는 또한 <a href="/wiki/UEFA_%EC%98%AC%ED%95%B4%EC%9D%98_%ED%8C%80" title="UEFA 올해의 팀">UEFA 올해의 팀</a>과 <a href="/wiki/%EA%B5%AD%EC%A0%9C_%EC%B6%95%EA%B5%AC_%EC%84%A0%EC%88%98_%ED%98%91%ED%9A%8C" title="국제 축구 선수 협회">FIFA 세계 XI 베스트 일레븐</a>, 8회의 PFA 올해의 팀에 선정 되었으며, 2006년 PFA 올해의 선수상과 2009년 올해의 FWA(잉글랜드 축구기자협회) 선수상을 수상했다. 제라드는 현재까지 <a class="mw-redirect mw-disambig" href="/wiki/FA_%EC%BB%B5" title="FA 컵">FA 컵</a> 결승전, <a class="mw-redirect" href="/wiki/%ED%92%8B%EB%B3%BC_%EB%A6%AC%EA%B7%B8_%EC%BB%B5" title="풋볼 리그 컵">풋볼 리그 컵</a> 결승전, <a class="mw-redirect" href="/wiki/UEFA%EC%BB%B5" title="UEFA컵">UEFA컵</a>(현재의 <a href="/wiki/UEFA_%EC%9C%A0%EB%A1%9C%ED%8C%8C%EB%A6%AC%EA%B7%B8" title="UEFA 유로파리그">UEFA 유로파리그</a>)과 <a href="/wiki/UEFA_%EC%B1%94%ED%94%BC%EC%96%B8%EC%8A%A4%EB%A6%AC%EA%B7%B8" title="UEFA 챔피언스리그">UEFA 챔피언스리그</a> 결승전에서 골을 득점한 기록을 보유한 유일한 축구 선수이다.</p>

<h2><span class="mw-headline" id=".EC.84.A0.EC.88.98_.EA.B2.BD.EB.A0.A5">선수 경력</span></h2>
<h3><span class="mw-headline" id=".ED.81.B4.EB.9F.BD">클럽</span></h3>
<p>"리버풀 유소년팀"에서 뛰던 제라드는 <a href="/wiki/%EB%A6%AC%EB%B2%84%ED%92%80_FC" title="리버풀 FC">리버풀</a>과 1997년 11월 5일 프로 계약을 맺고, 성인팀에서 뛰게 되었다. 1군에 합류한 이후 놀라운 퍼포먼스를 보여준 제라드는 어린 나이에도 불구하고 팀의 핵심 멤버로 자리를 잡았다. 1998년 11월 <a href="/wiki/%EC%85%80%ED%83%80_%EB%B9%84%EA%B3%A0" title="셀타 비고">셀타 비고</a>와의 <a class="mw-redirect" href="/wiki/UEFA%EC%BB%B5" title="UEFA컵">UEFA컵</a> 경기에서 데뷔한 제라드는 얼마 뒤인 1997년 11월 29일 <a href="/wiki/%EB%B8%94%EB%9E%99%EB%B2%88_%EB%A1%9C%EB%B2%84%EC%8A%A4_FC" title="블랙번 로버스 FC">블랙번 로버스</a>에 교체 출전하면서 <a href="/wiki/%ED%94%84%EB%A6%AC%EB%AF%B8%EC%96%B4%EB%A6%AC%EA%B7%B8" title="프리미어리그">프리미어리그</a> 데뷔전을 치렀다.</p>
<p>그 후 2000-2001 시즌에는 팀의 3관왕(리그컵, FA컵, UEFA컵)에 큰 공헌을 하며 PFA 올해의 신인선수상에 선정된다. 지난 2003-2004 시즌에는 팀의 주장이었던 <a class="mw-redirect" href="/wiki/%EC%82%AC%EB%AF%B8_%ED%9E%88%ED%94%BC%EC%95%84" title="사미 히피아">사미 히피아</a>로부터 주장 완장을 이어받아, 만 23세의 젊은 나이에 리버풀의 주장이되었다.</p>
<p>제라드는 2004-2005 시즌 <a href="/wiki/%EB%A6%AC%EB%B2%84%ED%92%80_FC" title="리버풀 FC">리버풀</a>의 <a href="/wiki/UEFA_%EC%B1%94%ED%94%BC%EC%96%B8%EC%8A%A4%EB%A6%AC%EA%B7%B8" title="UEFA 챔피언스리그">챔피언스리그</a> 우승에 크게 공헌했다. 조별리그 마지막 경기인 <a class="mw-redirect" href="/wiki/%EC%98%AC%EB%A6%BC%ED%94%BC%EC%95%84%EC%BD%94%EC%8A%A4" title="올림피아코스">올림피아코스</a>와의 시합에서 2골차 이상으로 승리해야 했던 리버풀의 시합이 끝나갈 무렵 2-1로 이기고 있었지만 팀의 목표인 '2골 차 승리'는 힘들어 보였고, 따라서 팀의 16강 진출도 물거품이 되는 것처럼 보였다. 그러나 종료 5분을 남기고 터진 제라드의 대단한 중거리 슛은 팀을 3-1 승리로 이끌었고, 이 시합을 발판으로 삼아 리버풀은 <a class="mw-redirect" href="/wiki/%EB%B0%94%EC%9D%B4%EC%96%B4_%EB%A0%88%EB%B2%84%EC%BF%A0%EC%A0%A0" title="바이어 레버쿠젠">레버쿠젠</a>, <a href="/wiki/%EC%9C%A0%EB%B2%A4%ED%88%AC%EC%8A%A4_FC" title="유벤투스 FC">유벤투스</a>, <a href="/wiki/%EC%B2%BC%EC%8B%9C_FC" title="첼시 FC">첼시</a>, <a href="/wiki/AC_%EB%B0%80%EB%9E%80" title="AC 밀란">AC 밀란</a>을 차례로 꺾고, 2004-2005 시즌 <a class="mw-redirect" href="/wiki/UEFA_%EC%B1%94%ED%94%BC%EC%96%B8%EC%8A%A4_%EB%A6%AC%EA%B7%B8" title="UEFA 챔피언스 리그">UEFA 챔피언스 리그</a> 우승하였다. 밀란과의 챔피언스리그 결승전에서 팀이 전반전에만 3-0으로 뒤지고 있었지만 후반전에 2골을 넣은지 6분만에 동점을 만드는 기염을 토했다. 이는 축구 역사상 최고의 경기로 기록되고 있다.</p>
<p>2007-2008 시즌에는 여느 <a href="/wiki/%EA%B3%B5%EA%B2%A9%EC%88%98" title="공격수">공격수</a> 못지 않은 득점력으로 미드필더 역할 뿐 아니라 강한 공격력 역시 인정 받았다. 이후 2008-2009 시즌에는 <a href="/wiki/UEFA_%EC%B1%94%ED%94%BC%EC%96%B8%EC%8A%A4%EB%A6%AC%EA%B7%B8" title="UEFA 챔피언스리그">UEFA 챔피언스리그</a> PSV와의 조별예선에서 3-1 승리를 거둘때 자신의 리버풀에서의 100번째 골을 터트렸으며 2위로 마친 프리미어리그에서도 31경기 출장 16골이라는 기록을 세웠다.</p>
<p>또한 2009-2010 시즌 <a href="/wiki/%EB%B8%94%EB%9E%99%EB%B2%88_%EB%A1%9C%EB%B2%84%EC%8A%A4_FC" title="블랙번 로버스 FC">블랙번 로버스 FC</a>와의 프리미어리그 15라운드에서 자신의 리버풀에서의 188번째의 출장기록을 세웠다.</p>
<p>이후 제라드는 <a href="/wiki/%ED%94%84%EB%A6%AC%EB%AF%B8%EC%96%B4%EB%A6%AC%EA%B7%B8_2011-12" title="프리미어리그 2011-12">2011-2012 시즌</a> 26R경기 <a href="/wiki/%EB%A8%B8%EC%A7%80%EC%82%AC%EC%9D%B4%EB%93%9C_%EB%8D%94%EB%B9%84" title="머지사이드 더비">머지사이드 더비</a>라 불리는 <a href="/wiki/%EC%97%90%EB%B2%84%ED%84%B4_FC" title="에버턴 FC">에버턴 FC</a>과의 경기에서 선발출전함으로써 그는 <a href="/wiki/%ED%94%84%EB%A6%AC%EB%AF%B8%EC%96%B4%EB%A6%AC%EA%B7%B8" title="프리미어리그">프리미어리그</a> 단일 팀 통산 400경기 출장이라는 대기록을 세웠다.이는 <a href="/wiki/%EB%9D%BC%EC%9D%B4%EC%96%B8_%EA%B8%B1%EC%8A%A4" title="라이언 긱스">라이언 긱스</a>과 <a class="mw-redirect" href="/wiki/%EA%B0%9C%EB%A6%AC_%EB%84%A4%EB%B9%8C" title="개리 네빌">개리 네빌</a>,<a href="/wiki/%ED%8F%B4_%EC%8A%A4%EC%BD%9C%EC%8A%A4" title="폴 스콜스">폴 스콜스</a>라는 몇몇 선수와 또다른 위대한 선수인 <a href="/wiki/%EC%A0%9C%EC%9D%B4%EB%AF%B8_%EC%BA%90%EB%9F%AC%EA%B1%B0" title="제이미 캐러거">제이미 캐러거</a>에 이어 5번째 기록이다. 그는 이 경기에서 해트트릭을 기록하면서 팀의 3-0 대승을 이끌었다. 2013년 10월 <a class="mw-redirect" href="/wiki/%EC%9E%89%EA%B8%80%EB%9E%9C%EB%93%9C_%ED%94%84%EB%A6%AC%EB%AF%B8%EC%96%B4_%EB%A6%AC%EA%B7%B8" title="잉글랜드 프리미어 리그">잉글랜드 프리미어 리그</a> 8라운드 뉴캐슬 전에서 100번째 골을 기록 하였다.</p>
<p>2015년 제라드는 <a class="mw-redirect" href="/wiki/LA_%EA%B0%A4%EB%9F%AD%EC%8B%9C" title="LA 갤럭시">LA 갤럭시</a>로 이적을 발표하였다. 이로써 17년간 리버풀에서의 생활을 마감하였다. (정규 리그를 비롯한 리버풀에서 뛴 모든 경기 통산 708경기 출장, 186 골 득점.)</p>
<h2><span class="mw-headline" id=".EA.B5.AD.EA.B0.80.EB.8C.80.ED.91.9C">국가대표</span></h2>
<p>제라드는 2000년 5월 31일 <a href="/wiki/%EC%9A%B0%ED%81%AC%EB%9D%BC%EC%9D%B4%EB%82%98_%EC%B6%95%EA%B5%AC_%EA%B5%AD%EA%B0%80%EB%8C%80%ED%91%9C%ED%8C%80" title="우크라이나 축구 국가대표팀">우크라이나</a>와의 경기에서 국가대표로서는 처음으로 모습을 드러냈다. 그 해 여름 제라드는 <a href="/wiki/%EC%9E%89%EA%B8%80%EB%9E%9C%EB%93%9C_%EC%B6%95%EA%B5%AC_%EA%B5%AD%EA%B0%80%EB%8C%80%ED%91%9C%ED%8C%80" title="잉글랜드 축구 국가대표팀">잉글랜드</a>의 <a href="/wiki/UEFA_%EC%9C%A0%EB%A1%9C_2000" title="UEFA 유로 2000">UEFA 유로 2000</a> 대표팀 명단에 포함되었다. 그러나 <a href="/wiki/UEFA_%EC%9C%A0%EB%A1%9C_2000" title="UEFA 유로 2000">UEFA 유로 2000</a>에서 <a href="/wiki/%EC%9E%89%EA%B8%80%EB%9E%9C%EB%93%9C_%EC%B6%95%EA%B5%AC_%EA%B5%AD%EA%B0%80%EB%8C%80%ED%91%9C%ED%8C%80" title="잉글랜드 축구 국가대표팀">잉글랜드</a>는 초라한 성적을 거두었고 제라드도 단 한 경기에 교체 출전했을 뿐이었다.</p>
<p>2년 뒤 <a href="/wiki/2006%EB%85%84_FIFA_%EC%9B%94%EB%93%9C%EC%BB%B5" title="2006년 FIFA 월드컵">2006년 FIFA 월드컵</a>에서 제라드는 <a href="/wiki/%ED%8A%B8%EB%A6%AC%EB%8B%88%EB%8B%A4%EB%93%9C_%ED%86%A0%EB%B0%94%EA%B3%A0_%EC%B6%95%EA%B5%AC_%EA%B5%AD%EA%B0%80%EB%8C%80%ED%91%9C%ED%8C%80" title="트리니다드 토바고 축구 국가대표팀">트리니다드 토바고</a>와의 조별리그 2차전에서 1골, <a href="/wiki/%EC%8A%A4%EC%9B%A8%EB%8D%B4_%EC%B6%95%EA%B5%AC_%EA%B5%AD%EA%B0%80%EB%8C%80%ED%91%9C%ED%8C%80" title="스웨덴 축구 국가대표팀">스웨덴</a>과의 조별리그 최종전에서 1골을 넣으면서 좋은 모습을 보여주었다. 그러나 <a href="/wiki/%ED%8F%AC%EB%A5%B4%ED%88%AC%EA%B0%88_%EC%B6%95%EA%B5%AC_%EA%B5%AD%EA%B0%80%EB%8C%80%ED%91%9C%ED%8C%80" title="포르투갈 축구 국가대표팀">포르투갈</a>과의 8강전 승부차기에서 실축하면서 체면을 구겼다.</p>
<p><a href="/wiki/UEFA_%EC%9C%A0%EB%A1%9C_2008" title="UEFA 유로 2008">UEFA 유로 2008</a> 예선에서도 무난한 활약을 하였으나, 잉글랜드가 크로아티아와 러시아에 밀려 본선 진출에 실패하면서 다음 대회를 기약하게 되었다. <a href="/wiki/2010%EB%85%84_FIFA_%EC%9B%94%EB%93%9C%EC%BB%B5" title="2010년 FIFA 월드컵">2010년 FIFA 월드컵</a> 예선을 통과하고 월드컵 본선에서 미국과의 1차전에서 선제골을 넣으면서 팀의 16강 진출을 이끌었다.</p>
<p><a href="/wiki/UEFA_%EC%9C%A0%EB%A1%9C_2012" title="UEFA 유로 2012">UEFA 유로 2012</a>을 앞두고 <a href="/wiki/%EC%9E%89%EA%B8%80%EB%9E%9C%EB%93%9C_%EC%B6%95%EA%B5%AC_%EA%B5%AD%EA%B0%80%EB%8C%80%ED%91%9C%ED%8C%80" title="잉글랜드 축구 국가대표팀">잉글랜드 축구 국가대표팀</a>의 새로운 감독으로 부임한 <a href="/wiki/%EB%A1%9C%EC%9D%B4_%ED%98%B8%EC%A7%80%EC%8A%A8" title="로이 호지슨">로이 호지슨</a>에 의해 주장으로 임명되었으며, <a class="mw-redirect" href="/wiki/%EA%B0%80%EB%A0%88%EC%8A%A4_%EB%B0%B0%EB%A6%AC" title="가레스 배리">가레스 배리</a>, <a class="mw-redirect" href="/wiki/%ED%94%84%EB%9E%AD%ED%81%AC_%EB%9E%A8%ED%8C%8C%EB%93%9C" title="프랭크 램파드">프랭크 램파드</a> 등 주전급 선수들의 부상으로 전력이 크게 약화된 상황에도 불구, 공수 전반에 기여하며 <a href="/wiki/%EC%9E%89%EA%B8%80%EB%9E%9C%EB%93%9C" title="잉글랜드">잉글랜드</a>의 8강 진출을 이끌었다.</p>
<p>그는 <a href="/wiki/%EC%9D%B4%ED%83%88%EB%A6%AC%EC%95%84_%EC%B6%95%EA%B5%AC_%EA%B5%AD%EA%B0%80%EB%8C%80%ED%91%9C%ED%8C%80" title="이탈리아 축구 국가대표팀">이탈리아</a>와의 8강전 승부차기에서 첫번째 키커로 나서서 성공했지만, 이후 각각 세번째, 네번째 키커로 나선 <a href="/wiki/%EC%95%A0%EC%8A%90%EB%A6%AC_%EC%98%81" title="애슐리 영">애슐리 영</a>, <a href="/wiki/%EC%95%A0%EC%8A%90%EB%A6%AC_%EC%BD%9C" title="애슐리 콜">애슐리 콜</a>이 연속으로 실축했고, 이탈리아의 마지막 키커로 나선 <a href="/wiki/%EC%95%8C%EB%A0%88%EC%82%B0%EB%93%9C%EB%A1%9C_%EB%94%94%EC%95%84%EB%A7%8C%ED%8B%B0" title="알레산드로 디아만티">알레산드로 디아만티</a>가 성공하면서 스코어 4대 2로 패한 동시에 또 다시 승부차기 징크스를 이어가게 됐다.</p>
<p><a href="/wiki/2012%EB%85%84" title="2012년">2012년</a> <a href="/wiki/11%EC%9B%94_14%EC%9D%BC" title="11월 14일">11월 14일</a>에서 제라드는 <a href="/wiki/%EC%8A%A4%EC%9B%A8%EB%8D%B4" title="스웨덴">스웨덴</a>과의 경기에서 자신의 <a href="/wiki/%EC%9E%89%EA%B8%80%EB%9E%9C%EB%93%9C_%EC%B6%95%EA%B5%AC_%EA%B5%AD%EA%B0%80%EB%8C%80%ED%91%9C%ED%8C%80" title="잉글랜드 축구 국가대표팀">잉글랜드 축구 국가대표팀</a> 100번째 경기를 치뤘다<sup class="reference" id="cite_ref-7"><a href="#cite_note-7">[7]</a></sup><sup class="reference" id="cite_ref-8"><a href="#cite_note-8">[8]</a></sup></p>
<h2><span class="mw-headline" id=".EA.B0.80.EC.A1.B1">가족</span></h2>
<p><a href="/wiki/2007%EB%85%84" title="2007년">2007년</a> 2살 연하의 모델이자 데일리미러의 칼럼니스트로 활동중인 "알렉스 제라드"(옛성씨: 커란)와 결혼하였다. 자녀는 릴리-엘라 제라드 (Lilly-Ella, <a href="/wiki/2004%EB%85%84" title="2004년">2004년</a> <a href="/wiki/2%EC%9B%94_23%EC%9D%BC" title="2월 23일">2월 23일</a> <a href="/wiki/%EB%A6%AC%EB%B2%84%ED%92%80" title="리버풀">리버풀</a> 출생), 렉시벨 제라드 (Lexie, <a href="/wiki/2006%EB%85%84" title="2006년">2006년</a> <a href="/wiki/5%EC%9B%94_9%EC%9D%BC" title="5월 9일">5월 9일</a> <a href="/wiki/%EB%A6%AC%EB%B2%84%ED%92%80" title="리버풀">리버풀</a> 출생), 루르드 제라드 (Lourdes, <a href="/wiki/2011%EB%85%84" title="2011년">2011년</a> <a href="/wiki/11%EC%9B%94_2%EC%9D%BC" title="11월 2일">11월 2일</a> <a href="/wiki/%EB%A6%AC%EB%B2%84%ED%92%80" title="리버풀">리버풀</a> 출생)으로 3명의 딸이 있다.</p>
