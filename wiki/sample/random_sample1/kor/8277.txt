<p>《<b>스파이더위크가의 비밀</b>》(The Spiderwick Chronicles)은 <a href="/wiki/2008%EB%85%84" title="2008년">2008년</a> 공개된 <a class="mw-redirect" href="/wiki/%EB%AF%B8%EA%B5%AD_%EC%98%81%ED%99%94" title="미국 영화">미국</a>의 판타지 영화이다. <a class="new" href="/w/index.php?title=%ED%86%A0%EB%93%9C_%EB%94%94%ED%84%B8%EB%A6%AC%EC%A7%80&amp;action=edit&amp;redlink=1" title="토드 디털리지 (없는 문서)">토드 디털리지</a>와 <a class="new" href="/w/index.php?title=%ED%99%80%EB%A6%AC_%EB%B8%94%EB%9E%99&amp;action=edit&amp;redlink=1" title="홀리 블랙 (없는 문서)">홀리 블랙</a>이 쓴 <a href="/wiki/%EC%8A%A4%ED%8C%8C%EC%9D%B4%EB%8D%94%EC%9C%84%ED%81%AC%EA%B0%80%EC%9D%98_%EB%B9%84%EB%B0%80" title="스파이더위크가의 비밀">동명의 동화 시리즈</a>를 원작으로 <a href="/wiki/%EB%A7%88%ED%81%AC_%EC%9B%8C%ED%84%B0%EC%8A%A4" title="마크 워터스">마크 워터스</a>가 연출하고 <a href="/wiki/%ED%94%84%EB%A0%88%EB%94%94_%ED%95%98%EC%9D%B4%EB%AA%A8%EC%96%B4" title="프레디 하이모어">프레디 하이모어</a>, <a href="/wiki/%EC%84%B8%EB%9D%BC_%EB%B3%BC%EC%A0%80" title="세라 볼저">세라 볼저</a>, <a href="/wiki/%EB%A9%94%EB%A6%AC%EB%A3%A8%EC%9D%B4%EC%A6%88_%ED%8C%8C%EC%BB%A4" title="메리루이즈 파커">메리루이즈 파커</a>, <a href="/wiki/%EB%A7%88%ED%8B%B4_%EC%87%BC%ED%8A%B8" title="마틴 쇼트">마틴 쇼트</a>, <a href="/wiki/%EB%8B%89_%EB%86%80%ED%8B%B0" title="닉 놀티">닉 놀티</a>, <a href="/wiki/%EC%84%B8%EC%8A%A4_%EB%A1%9C%EA%B1%B4" title="세스 로건">세스 로건</a>이 출연했다.</p>
