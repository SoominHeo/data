<p><b>스파이웨어</b>(Spyware)는 사용자의 동의 없이 설치되어 컴퓨터의 정보를 수집하고 전송하는 <a href="/wiki/%EC%95%85%EC%84%B1_%EC%86%8C%ED%94%84%ED%8A%B8%EC%9B%A8%EC%96%B4" title="악성 소프트웨어">악성 소프트웨어</a>로, 신용 카드와 같은 금융 정보 및 <a href="/wiki/%EC%A3%BC%EB%AF%BC%EB%93%B1%EB%A1%9D%EB%B2%88%ED%98%B8" title="주민등록번호">주민등록번호</a>와 같은 신상정보, <a href="/wiki/%EC%95%94%ED%98%B8" title="암호">암호</a>를 비롯한 각종 정보를 수집한다. 처음에는 미국의 인터넷 전문 광고 회사인 라디에이트가 시작하였으나 그 뒤로 아이디, 암호 등을 알아낼 수 있도록 나쁜 용도로 변형되었다. <a href="/wiki/%EA%B5%AD%EB%A6%BD%EA%B5%AD%EC%96%B4%EC%9B%90" title="국립국어원">국립국어원</a>에서는 스파이웨어를 <b>정보 빼내기 프로그램</b>으로 순화한 바가 있다.</p>

<h2><span class="mw-headline" id=".EC.B9.98.EB.A3.8C_.EB.B0.8F_.EC.98.88.EB.B0.A9">치료 및 예방</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%8A%A4%ED%8C%8C%EC%9D%B4%EC%9B%A8%EC%96%B4&amp;action=edit&amp;section=1" title="부분 편집: 치료 및 예방">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>스파이웨어가 침입하여 더 악화됨에 따라, 수많은 기술들이 이들을 대항하기 위해 뭉쳐왔다. 이러한 것들은 스파이웨어를 제거하고 예방하도록 고안된 프로그램들로, 여러 사용자가 시스템 위에서 스파이웨어가 설치될 기회를 줄이도록 구성되어 있다.</p>
<p>그럼에도 불구하고, 스파이웨어는 금전적인 문제를 남길 수도 있다. 수많은 스파이웨어는 윈도 컴퓨터를 감염시킬 때, 유일한 치료법은 사용자 데이터를 백업하고 <a href="/wiki/%EC%9A%B4%EC%98%81_%EC%B2%B4%EC%A0%9C" title="운영 체제">운영 체제</a>를 완전히 다시 설치하는 것이다.</p>
<h3><span class="mw-headline" id=".EC.95.88.ED.8B.B0.EC.8A.A4.ED.8C.8C.EC.9D.B4.EC.9B.A8.EC.96.B4">안티스파이웨어</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%8A%A4%ED%8C%8C%EC%9D%B4%EC%9B%A8%EC%96%B4&amp;action=edit&amp;section=2" title="부분 편집: 안티스파이웨어">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>많은 프로그래머들과 회사들은 스파이웨어를 제거하고 예방하는 제품들을 공개해 왔다. <a class="new" href="/w/index.php?title=%EB%9D%BC%EB%B0%94%EC%86%8C%ED%94%84%ED%8A%B8_%EC%96%B4%EB%93%9C%EC%96%B4%EC%9B%A8%EC%96%B4_SE&amp;action=edit&amp;redlink=1" title="라바소프트 어드어웨어 SE (없는 문서)">라바소프트 어드어웨어 SE</a>와 <a class="new" href="/w/index.php?title=%ED%8C%A8%ED%8A%B8%EB%A6%AD_%EC%BD%9C%EB%9D%BC%EC%9D%98_%EC%8A%A4%ED%8C%8C%EC%9D%B4%EB%B4%87&amp;action=edit&amp;redlink=1" title="패트릭 콜라의 스파이봇 (없는 문서)">패트릭 콜라의 스파이봇</a>을 예로 들 수 있다. 최근에 마이크로소프트는 거대한 앤티스파이웨어 소프트웨어를 만들었는데, <a class="mw-redirect" href="/wiki/%EC%9C%88%EB%8F%84_XP" title="윈도 XP">윈도 XP</a>와 <a class="mw-redirect" href="/wiki/%EC%9C%88%EB%8F%84_2003" title="윈도 2003">윈도 2003</a> 정품 사용자들은 무료로 내려 받아 사용할 수 있다. 2006년 초에 마이크로소프트는 이 베타 소프트웨어의 이름을 <a class="mw-redirect" href="/wiki/%EC%9C%88%EB%8F%84_%EB%94%94%ED%8E%9C%EB%8D%94" title="윈도 디펜더">윈도 디펜더</a>라고 지었으며, <a href="/wiki/2006%EB%85%84_10%EC%9B%94" title="2006년 10월">2006년 10월</a>에 무료 다운로드 버전이 공개되었다. 마이크로소프트는 대부분의 최신 <a class="mw-disambig" href="/wiki/%EC%9C%88%EB%8F%84" title="윈도">윈도</a>에 이 제품을 탑재하여 판매하고 있다.</p>
<p>잘 알려진 다른 안티스파이웨어는 다음과 같다:</p>
<ul>
<li>PC Tools's Spyware Doctor</li>
<li>Sunbelt Software's Counterspy</li>
<li>Trend Micro's HijackThis</li>
<li>Webroot Software's Spy Sweeper</li>
<li>ParetoLogic's Anti-Spyware and XoftSpy SE</li>
<li>Steve Gibson's Optout</li>
<li>Lavasoft's Ad-aware</li>
</ul>
<h3><span class="mw-headline" id=".EA.B0.80.EC.A7.9C_.EC.95.88.ED.8B.B0.EC.8A.A4.ED.8C.8C.EC.9D.B4.EC.9B.A8.EC.96.B4">가짜 안티스파이웨어</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%8A%A4%ED%8C%8C%EC%9D%B4%EC%9B%A8%EC%96%B4&amp;action=edit&amp;section=3" title="부분 편집: 가짜 안티스파이웨어">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>악독한 프로그래머들은 많은 수의 가짜 앤티스파이웨어 프로그램들을 공개했는데 웹 <a class="mw-redirect" href="/wiki/%EB%B0%B0%EB%84%88_%EA%B4%91%EA%B3%A0" title="배너 광고">배너 광고</a>를 통해 스파이웨어에 감염되었다는 메시지를 사용자에게 경고로 보여 주며, 스파이웨어를 실제로는 제거하지 않는 프로그램을 내려 받게 한다. 더 심각한 경우, 오히려 스파이웨어를 설치해 버리는 경우도 있다. 이런 프로그램들은 보통 인터넷 익스플로러의 취약점(ActiveX)을 이용해 사용자가 모르는 사이에 설치되는 경우가 많다.</p>
