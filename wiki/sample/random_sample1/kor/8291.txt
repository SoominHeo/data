<p><b>스페인 해군</b>(<span style="font-size: smaller;"><a href="/wiki/%EC%8A%A4%ED%8E%98%EC%9D%B8%EC%96%B4" title="스페인어">스페인어</a>:</span> <span lang="es" xml:lang="es">Armada Española</span>)은 <a href="/wiki/%EC%8A%A4%ED%8E%98%EC%9D%B8" title="스페인">스페인</a>의 <a href="/wiki/%EA%B5%B0%EC%82%AC_%EC%A1%B0%EC%A7%81" title="군사 조직">군사 조직</a> 중 하나로써, 영해 방위를 책임지고 있다.</p>
<h2><span class="mw-headline" id=".EC.97.AD.EC.82.AC">역사</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%8A%A4%ED%8E%98%EC%9D%B8_%ED%95%B4%EA%B5%B0&amp;action=edit&amp;section=1" title="부분 편집: 역사">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<table class="notice section" style="text-align: left; margin:0 0 0.5em 0.5em; clear:none; font-size:smaller;">

<td width="auto"><img alt="" data-file-height="204" data-file-width="262" height="19" src="//upload.wikimedia.org/wikipedia/commons/thumb/6/64/Question_book-4.svg/25px-Question_book-4.svg.png" srcset="//upload.wikimedia.org/wikipedia/commons/thumb/6/64/Question_book-4.svg/38px-Question_book-4.svg.png 1.5x, //upload.wikimedia.org/wikipedia/commons/thumb/6/64/Question_book-4.svg/50px-Question_book-4.svg.png 2x" width="25"/> 이 문단의 <b>내용은</b> <b><a href="/wiki/%EC%9C%84%ED%82%A4%EB%B0%B1%EA%B3%BC:%EC%B6%9C%EC%B2%98_%EB%B0%9D%ED%9E%88%EA%B8%B0" title="위키백과:출처 밝히기">출처</a>가 분명하지 않습니다.</b> 지금 바로 이 <span class="plainlinks"><a class="external text" href="//ko.wikipedia.org/w/index.php?title=%EC%8A%A4%ED%8E%98%EC%9D%B8_%ED%95%B4%EA%B5%B0&amp;action=edit">문단을 편집</a></span>하여, 참고하신 문헌이나 <a href="/wiki/%EC%9C%84%ED%82%A4%EB%B0%B1%EA%B3%BC:%EC%8B%A0%EB%A2%B0%ED%95%A0_%EC%88%98_%EC%9E%88%EB%8A%94_%EC%B6%9C%EC%B2%98" title="위키백과:신뢰할 수 있는 출처">신뢰할 수 있는 출처</a>를 각주 등으로 표기해 주세요. <a href="/wiki/%EC%9C%84%ED%82%A4%EB%B0%B1%EA%B3%BC:%ED%99%95%EC%9D%B8_%EA%B0%80%EB%8A%A5" title="위키백과:확인 가능">검증</a>되지 않은 내용은 삭제될 수도 있습니다. 내용에 대한 의견이 있으시다면 <a href="/wiki/%ED%86%A0%EB%A1%A0:%EC%8A%A4%ED%8E%98%EC%9D%B8_%ED%95%B4%EA%B5%B0#.23" title="토론:스페인 해군">토론 문서</a>에서 나누어 주세요. (2013년 12월 28일에 문단의 출처가 요청되었습니다.)</td>

<p>스페인 해군은 <a href="/wiki/13%EC%84%B8%EA%B8%B0" title="13세기">13세기</a>경에 창설되었다. <a class="mw-redirect" href="/wiki/%EB%8C%80%ED%95%AD%ED%95%B4_%EC%8B%9C%EB%8C%80" title="대항해 시대">대항해 시대</a> 때는 <a class="mw-redirect" href="/wiki/%EB%AC%B4%EC%A0%81_%ED%95%A8%EB%8C%80" title="무적 함대">무적 함대</a>로 불릴 정도로 강력했으나, <a href="/wiki/16%EC%84%B8%EA%B8%B0" title="16세기">16세기</a> 말에 <a href="/wiki/%EC%9E%89%EA%B8%80%EB%9E%9C%EB%93%9C_%EC%99%95%EA%B5%AD" title="잉글랜드 왕국">잉글랜드</a>의 <a href="/wiki/%ED%95%B4%EC%A0%81" title="해적">해적</a>인 <a href="/wiki/%ED%94%84%EB%9E%9C%EC%8B%9C%EC%8A%A4_%EB%93%9C%EB%A0%88%EC%9D%B4%ED%81%AC" title="프랜시스 드레이크">프랜시스 드레이크</a>에게 어이없게 패배한 이래, 무적 함대의 자존심에 금이 가기 시작했다. 본디 스페인 해군은 전함 설계등이 영국 해군보다 우수했었으나, 혼란한 정치로 인해 일관성없는 해군 정책등으로 영국 해군에게 우위를 점할수 없었다. 결국, <a href="/wiki/1805%EB%85%84" title="1805년">1805년</a>에 있었던 <a href="/wiki/%ED%8A%B8%EB%9D%BC%ED%8C%94%EA%B0%80%EB%A5%B4_%ED%95%B4%EC%A0%84" title="트라팔가르 해전">트라팔가르</a> 해전 때는 <a href="/wiki/%ED%94%84%EB%9E%91%EC%8A%A4" title="프랑스">프랑스</a> <a href="/wiki/%ED%94%84%EB%9E%91%EC%8A%A4_%ED%95%B4%EA%B5%B0" title="프랑스 해군">해군</a>과 함께 <a href="/wiki/%EC%98%81%EA%B5%AD" title="영국">영국</a> <a class="mw-redirect" href="/wiki/%EC%98%81%EA%B5%AD_%ED%95%B4%EA%B5%B0" title="영국 해군">해군</a>에 맞서 싸웠으나, 패배하였다. 이후 <a href="/wiki/1898%EB%85%84" title="1898년">1898년</a> <a href="/wiki/%EB%AF%B8%EA%B5%AD-%EC%8A%A4%ED%8E%98%EC%9D%B8_%EC%A0%84%EC%9F%81" title="미국-스페인 전쟁">미국-스페인 전쟁</a> 때는 <a href="/wiki/%EB%AF%B8%EA%B5%AD" title="미국">미국</a> <a href="/wiki/%EB%AF%B8%EA%B5%AD_%ED%95%B4%EA%B5%B0" title="미국 해군">해군</a>보다 수적으로 불리하고, 장비 면에서도 열세여서 패배했다(당시 스페인 해군에서는 목조 전함이 많았다).</p>
