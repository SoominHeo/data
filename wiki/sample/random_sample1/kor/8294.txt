<p><b>스페인의 마리아</b>(Maria von Spanien, <a href="/wiki/1528%EB%85%84" title="1528년">1528년</a> <a href="/wiki/6%EC%9B%94_21%EC%9D%BC" title="6월 21일">6월 21일</a> ~ <a href="/wiki/1603%EB%85%84" title="1603년">1603년</a> <a href="/wiki/2%EC%9B%94_26%EC%9D%BC" title="2월 26일">2월 26일</a>)는 <a href="/wiki/%EC%8B%A0%EC%84%B1_%EB%A1%9C%EB%A7%88_%EC%A0%9C%EA%B5%AD%EC%9D%98_%ED%99%A9%EC%A0%9C" title="신성 로마 제국의 황제">신성 로마 제국의 황제</a> <a class="mw-redirect" href="/wiki/%EC%8B%A0%EC%84%B1%EB%A1%9C%EB%A7%88%EC%A0%9C%EA%B5%AD%EC%9D%98_%EB%A7%89%EC%8B%9C%EB%B0%80%EB%A6%AC%EC%95%88_2%EC%84%B8" title="신성로마제국의 막시밀리안 2세">막시밀리안 2세</a>의 황후이다. <a href="/wiki/%EC%8B%A0%EC%84%B1_%EB%A1%9C%EB%A7%88_%EC%A0%9C%EA%B5%AD%EC%9D%98_%ED%99%A9%EC%A0%9C" title="신성 로마 제국의 황제">신성 로마 제국의 황제</a> <a href="/wiki/%EC%B9%B4%EB%A5%BC_5%EC%84%B8" title="카를 5세">카를 5세</a>와 <a class="mw-redirect" href="/wiki/%ED%8F%AC%EB%A5%B4%ED%88%AC%EA%B0%88%EC%9D%98_%EC%9D%B4%EC%82%AC%EB%B2%A8%EB%9D%BC" title="포르투갈의 이사벨라">포르투갈의 이사벨라</a> 사이에서 태어난 장녀이다. <a class="mw-redirect" href="/wiki/%EC%8B%A0%EC%84%B1%EB%A1%9C%EB%A7%88%EC%A0%9C%EA%B5%AD%EC%9D%98_%EB%A3%A8%EB%8F%8C%ED%94%84_2%EC%84%B8" title="신성로마제국의 루돌프 2세">루돌프 2세</a>와 <a href="/wiki/%EB%A7%88%ED%8B%B0%EC%95%84%EC%8A%A4" title="마티아스">마티아스</a>의 어머니이기도 하다. 남편이 죽은 뒤에는 <a href="/wiki/%EC%8A%A4%ED%8E%98%EC%9D%B8" title="스페인">스페인</a>으로 돌아와 그곳에서 죽었다.</p>
