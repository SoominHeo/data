<p><b>람베르토 2세</b>(Lambert II of Spoleto, <a href="/wiki/870%EB%85%84" title="870년">870년</a>/<a href="/wiki/875%EB%85%84" title="875년">875년</a>경 - <a href="/wiki/898%EB%85%84" title="898년">898년</a> <a href="/wiki/10%EC%9B%94_15%EC%9D%BC" title="10월 15일">10월 15일</a>)는 <a href="/wiki/%EC%B9%B4%EB%A1%A4%EB%A7%81%EA%B1%B0_%EC%99%95%EC%A1%B0" title="카롤링거 왕조">카롤링거 왕조</a> 출신의 <a href="/wiki/%EC%8A%A4%ED%8F%B4%EB%A0%88%ED%86%A0" title="스폴레토">스폴레토</a> <a class="mw-disambig" href="/wiki/%EA%B3%B5%EC%9E%91" title="공작">공작</a>, <a href="/wiki/%EC%9D%B4%ED%83%88%EB%A6%AC%EC%95%84%EC%9D%98_%EA%B5%B0%EC%A3%BC" title="이탈리아의 군주">이탈리아의 군주</a>이다. <a href="/wiki/%EC%8A%A4%ED%8F%B4%EB%A0%88%ED%86%A0" title="스폴레토">스폴레토</a> <a class="mw-disambig" href="/wiki/%EA%B3%B5%EC%9E%91" title="공작">공작</a> <a class="mw-redirect" href="/wiki/%EA%B5%AC%EC%9D%B4%EB%8F%84_3%EC%84%B8" title="구이도 3세">구이도 3세</a>의 아들이며 <a href="/wiki/891%EB%85%84" title="891년">891년</a> <a href="/wiki/5%EC%9B%94" title="5월">5월</a> 부왕에 의해 공동 황제로 선포되었다. 그러나 <a href="/wiki/%EA%B5%90%ED%99%A9_%ED%8F%AC%EB%A5%B4%EB%AA%A8%EC%86%8C" title="교황 포르모소">교황 포르모소</a>는 그에게 왕관 수여를 거절했고, 동프랑크의 아르눌프에게 도움을 청하여 이후 <a href="/wiki/%EC%9D%B4%ED%83%88%EB%A6%AC%EC%95%84" title="이탈리아">이탈리아</a>를 원정한 아르눌프와 여러 번 교전하였다. <a href="/wiki/896%EB%85%84" title="896년">896년</a> <a href="/wiki/%EA%B5%90%ED%99%A9_%ED%8F%AC%EB%A5%B4%EB%AA%A8%EC%86%8C" title="교황 포르모소">교황 포르모소</a>에 의해 황제위를 박탈당하자 분개한 그는 <a href="/wiki/%EA%B5%90%ED%99%A9_%ED%8F%AC%EB%A5%B4%EB%AA%A8%EC%86%8C" title="교황 포르모소">포르모소</a> 사후 자신들의 측근들을 시켜 포르모소의 묘소를 파헤쳐 시신을 상대로 재판을 하였다.</p>
<p>재위기간 내내 <a href="/wiki/%EA%B5%90%ED%99%A9_%ED%8F%AC%EB%A5%B4%EB%AA%A8%EC%86%8C" title="교황 포르모소">교황 포르모소</a>의 구원 요청으로 이탈리아 원정을 한 <a href="/wiki/%EC%95%84%EB%A5%B4%EB%88%8C%ED%94%84_%ED%8F%B0_%EC%BC%80%EB%A5%B8%ED%85%90" title="아르눌프 폰 케른텐">아르눌프</a>와 교전했고, 이탈리아 내부에서도 <a href="/wiki/%EB%B2%A0%EB%A0%8C%EA%B0%80%EB%A6%AC%EC%98%A4_1%EC%84%B8" title="베렌가리오 1세">베렌가리오 1세</a>와 제관 및 이탈리아 왕관을 놓고 경합하였다. <a href="/wiki/896%EB%85%84" title="896년">896년</a> <a href="/wiki/2%EC%9B%94_22%EC%9D%BC" title="2월 22일">2월 22일</a> <a href="/wiki/%EA%B5%90%ED%99%A9_%ED%8F%AC%EB%A5%B4%EB%AA%A8%EC%86%8C" title="교황 포르모소">교황 포르모소</a>에 의해 폐위당했으나 <a href="/wiki/898%EB%85%84" title="898년">898년</a>경까지 이탈리아의 실질적인 통치자로 군림하였다. 병을 앓던 아르눌프가 <a href="/wiki/%EC%9D%B4%ED%83%88%EB%A6%AC%EC%95%84" title="이탈리아">이탈리아</a>에 오래 체류하지 못하고 <a href="/wiki/%EB%8F%99%ED%94%84%EB%9E%91%ED%81%AC_%EC%99%95%EA%B5%AD" title="동프랑크 왕국">동프랑크 왕국</a>으로 회군하자, 람베르토는 <a href="/wiki/%EC%95%84%EB%A5%B4%EB%88%8C%ED%94%84_%ED%8F%B0_%EC%BC%80%EB%A5%B8%ED%85%90" title="아르눌프 폰 케른텐">아르눌프</a> 제관 수여의 무효화를 주장했지만 <a href="/wiki/898%EB%85%84" title="898년">898년</a> <a href="/wiki/10%EC%9B%94_15%EC%9D%BC" title="10월 15일">10월 15일</a> <a class="new" href="/w/index.php?title=%ED%94%BC%EB%8D%B4%EC%B0%A8&amp;action=edit&amp;redlink=1" title="피덴차 (없는 문서)">피덴차</a> 근처에서 사냥 도중 암살당했다. 그는 <a href="/wiki/%EC%9D%B4%ED%83%88%EB%A6%AC%EC%95%84" title="이탈리아">이탈리아</a>에서 <a class="mw-redirect" href="/wiki/%EC%B9%B4%EB%A1%A4%EB%A7%81%EA%B1%B0_%EC%99%95%EA%B0%80" title="카롤링거 왕가">카롤링거 왕가</a>의 카피툴라리아(Capitulary) 전통을 존중한 마지막 군주이기도 하다.</p>

<p>람베르토 2세는 <a class="mw-redirect" href="/wiki/%EA%B5%AC%EC%9D%B4%EB%8F%84_3%EC%84%B8" title="구이도 3세">구이도 3세</a>와 <a href="/wiki/%EB%B2%A0%EB%84%A4%EB%B2%A4%ED%86%A0" title="베네벤토">베네벤토</a> <a class="mw-disambig" href="/wiki/%EA%B3%B5%EC%9E%91" title="공작">공작</a> <a class="new" href="/w/index.php?title=%EC%95%84%EB%8D%B8%ED%9E%88%EC%8A%A4&amp;action=edit&amp;redlink=1" title="아델히스 (없는 문서)">아델히스</a>의 딸 <a href="/wiki/%EB%B2%A0%EB%84%A4%EB%B2%A4%ED%86%A0" title="베네벤토">베네벤토</a>의 아겔트루드의 아들이었다. 그의 증조모 <a class="new" href="/w/index.php?title=%EB%A1%AC%EB%B0%94%EB%A5%B4%EB%94%94%EC%95%84%EC%9D%98_%EC%95%84%EB%8D%B8%EB%9D%BC%EC%9D%B4%EB%93%9C&amp;action=edit&amp;redlink=1" title="롬바르디아의 아델라이드 (없는 문서)">롬바르디아의 아델라이드</a>는 <a class="mw-redirect" href="/wiki/%EC%83%A4%EB%A5%BC%EB%A7%88%EB%89%B4" title="샤를마뉴">샤를마뉴</a>의 아들 <a href="/wiki/%ED%94%BC%ED%94%BC%EB%85%B8_%EC%B9%B4%EB%A5%BC%EB%A1%9C%EB%A7%8C" title="피피노 카를로만">피피노 카를로만</a>의 딸이었다. 또한 작은할아버지 <a class="new" href="/w/index.php?title=%EB%82%AD%ED%8A%B8%EC%9D%98_%EB%9E%8C%EB%B2%A0%EB%A5%B4%ED%86%A0_2%EC%84%B8&amp;action=edit&amp;redlink=1" title="낭트의 람베르토 2세 (없는 문서)">낭트의 람베르토 2세</a>의 부인 로틸데는 <a href="/wiki/%EC%A4%91%ED%94%84%EB%9E%91%ED%81%AC_%EC%99%95%EA%B5%AD" title="중프랑크 왕국">중프랑크 왕국</a>의 황제 <a href="/wiki/%EB%A1%9C%ED%83%80%EB%A5%B4_1%EC%84%B8" title="로타르 1세">로타르 1세</a>의 딸이었다. 그는 <a class="mw-redirect" href="/wiki/%EC%83%A4%EB%A5%BC%EB%A7%88%EB%89%B4" title="샤를마뉴">샤를마뉴</a>의 아들 <a href="/wiki/%ED%94%BC%ED%94%BC%EB%85%B8_%EC%B9%B4%EB%A5%BC%EB%A1%9C%EB%A7%8C" title="피피노 카를로만">피피노 카를로만</a>의 증외고손에 해당되어 계승권에서 멀었지만, 아버지 <a href="/wiki/%EC%8A%A4%ED%8F%B4%EB%A0%88%ED%86%A0%EC%9D%98_%EA%B7%80%EB%8F%84_3%EC%84%B8" title="스폴레토의 귀도 3세">구이도</a>가 <a href="/wiki/888%EB%85%84" title="888년">888년</a> <a href="/wiki/12%EC%9B%94" title="12월">12월</a> <a href="/wiki/%EC%9D%B4%ED%83%88%EB%A6%AC%EC%95%84" title="이탈리아">이탈리아</a>의 귀족들로부터 왕으로 선출되고, 신성 로마 제국 황제관을 수여받으면서 공동황제이자 황위 계승자가 되었다.</p>
<p><a href="/wiki/889%EB%85%84" title="889년">889년</a> 아버지 <a href="/wiki/%EC%8A%A4%ED%8F%B4%EB%A0%88%ED%86%A0%EC%9D%98_%EA%B7%80%EB%8F%84_3%EC%84%B8" title="스폴레토의 귀도 3세">구이도 3세</a>는 <a href="/wiki/%EC%9D%B4%ED%83%88%EB%A6%AC%EC%95%84" title="이탈리아">이탈리아</a> <a href="/wiki/%ED%8C%8C%EB%B9%84%EC%95%84" title="파비아">파비아</a>에서 귀족들에 의해 이탈리아의 왕으로 선출되었다. 왕세자가 된 람베르토는 <a href="/wiki/889%EB%85%84" title="889년">889년</a> <a href="/wiki/5%EC%9B%94" title="5월">5월</a> 제국의 섭정이자 이탈리아 왕국 섭정으로 임명된다. 왕권을 불안히 여긴 부왕 구이도는 교황에게 아들 람베르토의 축성과 제관 수여를 계속 요청했지만 번번히 거절당한다. <a href="/wiki/891%EB%85%84" title="891년">891년</a> <a href="/wiki/5%EC%9B%94" title="5월">5월</a> 부왕 <a href="/wiki/%EC%8A%A4%ED%8F%B4%EB%A0%88%ED%86%A0%EC%9D%98_%EA%B7%80%EB%8F%84_3%EC%84%B8" title="스폴레토의 귀도 3세">구이도 3세</a>에 의해 <a href="/wiki/%ED%8C%8C%EB%B9%84%EC%95%84" title="파비아">파비아</a>에서 공동 황제로 선포되었다. <a href="/wiki/%EA%B5%90%ED%99%A9_%EC%8A%A4%ED%85%8C%ED%8C%8C%EB%85%B8_5%EC%84%B8" title="교황 스테파노 5세">교황 스테파노 5세</a>에게 축성받고 바로 이탈리아의 왕이 되었다. 그러나 같은 해 <a href="/wiki/%EA%B5%90%ED%99%A9_%EC%8A%A4%ED%85%8C%ED%8C%8C%EB%85%B8_5%EC%84%B8" title="교황 스테파노 5세">교황 스테파노 5세</a>가 죽고 포르모소가 새 교황이 되자 람베르토는 <a href="/wiki/892%EB%85%84" title="892년">892년</a> <a href="/wiki/4%EC%9B%94_30%EC%9D%BC" title="4월 30일">4월 30일</a> 아버지 구이도 3세와 함께 <a href="/wiki/%EB%9D%BC%EB%B2%A4%EB%82%98" title="라벤나">라벤나</a>로 가서 제관을 요구했지만 <a href="/wiki/%EA%B5%90%ED%99%A9_%ED%8F%AC%EB%A5%B4%EB%AA%A8%EC%86%8C" title="교황 포르모소">교황 포르모소</a>는 제관 수여를 거절했다. 람베르토와 그의 아버지 구이도 3세는 교황에게 <a href="/wiki/%ED%94%BC%ED%95%80%EC%9D%98_%EA%B8%B0%EC%A6%9D" title="피핀의 기증">피핀의 기증</a>과 같은 형태로 영토를 헌납하겠다고 약속했지만, <a href="/wiki/%EA%B5%90%ED%99%A9_%ED%8F%AC%EB%A5%B4%EB%AA%A8%EC%86%8C" title="교황 포르모소">교황 포르모소</a>에게 거절당했다.</p>
<h3><span class="mw-headline" id=".EC.A0.9C.EA.B4.80_.EC.88.98.EC.97.AC_.EB.AC.B8.EC.A0.9C">제관 수여 문제</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%8A%A4%ED%8F%B4%EB%A0%88%ED%86%A0%EC%9D%98_%EB%9E%8C%EB%B2%A0%EB%A5%B4%ED%86%A0_2%EC%84%B8&amp;action=edit&amp;section=3" title="부분 편집: 제관 수여 문제">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p><a href="/wiki/892%EB%85%84" title="892년">892년</a> <a href="/wiki/4%EC%9B%94_30%EC%9D%BC" title="4월 30일">4월 30일</a>과 <a href="/wiki/8%EC%9B%94_30%EC%9D%BC" title="8월 30일">8월 30일</a>, 그리고 <a href="/wiki/893%EB%85%84" title="893년">893년</a> 초, 귀도 3세는 <a href="/wiki/%EB%9D%BC%EB%B2%A4%EB%82%98" title="라벤나">라벤나</a>를 방문하여 <a href="/wiki/%EA%B5%90%ED%99%A9_%ED%8F%AC%EB%A5%B4%EB%AA%A8%EC%86%8C" title="교황 포르모소">교황 포르모소</a>에게 아들 람베르토를 공동 황제로 인정해줄 것을 요청했으나 거절당했다. 귀도는 군사적 압력을 행사하여 <a href="/wiki/%EA%B5%90%ED%99%A9_%ED%8F%AC%EB%A5%B4%EB%AA%A8%EC%86%8C" title="교황 포르모소">교황 포르모소</a>에게 아들 람베르토의 공동 황제를 추인하도록 강요했다. 귀도는 황제나 왕의 아들을 공동 통치자로 승인하는 <a href="/wiki/%EB%B2%95" title="법">법</a>을 제정하려 했지만 실패했다. <a href="/wiki/%EA%B5%90%ED%99%A9_%ED%8F%AC%EB%A5%B4%EB%AA%A8%EC%86%8C" title="교황 포르모소">교황 포르모소</a>는 비밀리에 <a href="/wiki/%EC%95%84%EB%A5%B4%EB%88%8C%ED%94%84_%ED%8F%B0_%EC%BC%80%EB%A5%B8%ED%85%90" title="아르눌프 폰 케른텐">아르눌프</a>에게 사람을 보내 도움을 호소하였다.</p>
<p><a href="/wiki/893%EB%85%84" title="893년">893년</a> <a href="/wiki/%EA%B5%90%ED%99%A9_%ED%8F%AC%EB%A5%B4%EB%AA%A8%EC%86%8C" title="교황 포르모소">교황 포르모소</a>는 <a class="mw-redirect" href="/wiki/%EB%8F%99%ED%94%84%EB%9E%91%ED%81%AC" title="동프랑크">동프랑크</a>의 <a href="/wiki/%EB%A0%88%EA%B2%90%EC%8A%A4%EB%B6%80%EB%A5%B4%ED%81%AC" title="레겐스부르크">레겐스부르크</a>로 사절을 보내 <a href="/wiki/%EB%8F%99%ED%94%84%EB%9E%91%ED%81%AC_%EC%99%95%EA%B5%AD" title="동프랑크 왕국">동프랑크의 국왕</a> <a href="/wiki/%EC%95%84%EB%A5%B4%EB%88%8C%ED%94%84_%ED%8F%B0_%EC%BC%80%EB%A5%B8%ED%85%90" title="아르눌프 폰 케른텐">아르눌프</a>에게 <a href="/wiki/%EC%9D%B4%ED%83%88%EB%A6%AC%EC%95%84" title="이탈리아">이탈리아</a>를 해방시키고 <a href="/wiki/%EB%A1%9C%EB%A7%88" title="로마">로마</a>를 구해 달라며 군사를 요청했고, <a href="/wiki/894%EB%85%84" title="894년">894년</a> 초 아르눌프는 자신의 서자 <a href="/wiki/%EC%B8%A0%EB%B2%A4%ED%8B%B0%EB%B3%BC%ED%8A%B8" title="츠벤티볼트">츠벤티볼트</a>와 함께 <a class="mw-redirect" href="/wiki/%EB%B0%94%EC%9D%B4%EC%97%90%EB%A5%B8" title="바이에른">바이에른</a>의 군대를 이끌고 <a href="/wiki/%EC%95%8C%ED%94%84%EC%8A%A4_%EC%82%B0%EB%A7%A5" title="알프스 산맥">알프스 산맥</a>을 넘어와 이탈리아를 공격했고, <a href="/wiki/%ED%94%84%EB%A6%AC%EC%9A%B8%EB%A6%AC" title="프리울리">프리울리</a>의 <a class="mw-redirect" href="/wiki/%EB%B2%A0%EB%A0%9D%EA%B0%80%EB%A6%AC%EC%98%A4_1%EC%84%B8" title="베렝가리오 1세">베렝가리오 1세</a>가 이들에게 합류했다.</p>
<h3><span class="mw-headline" id=".EB.8F.99.ED.94.84.EB.9E.91.ED.81.AC_.EA.B5.B0.EB.8C.80.EC.99.80.EC.9D.98_.EC.A0.84.EC.9F.81">동프랑크 군대와의 전쟁</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%8A%A4%ED%8F%B4%EB%A0%88%ED%86%A0%EC%9D%98_%EB%9E%8C%EB%B2%A0%EB%A5%B4%ED%86%A0_2%EC%84%B8&amp;action=edit&amp;section=4" title="부분 편집: 동프랑크 군대와의 전쟁">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>아버지 구이도와 함께 아르눌프, 츠벤티볼트, 베렝가리오 1세의 연합군에 맞서 싸웠으나 패배했다. 그러나 이들은 곧 아버지 구이도의 뇌물로 그해 가을 동프랑크로 떠나보냈다. <a href="/wiki/894%EB%85%84" title="894년">894년</a> <a href="/wiki/11%EC%9B%94" title="11월">11월</a> 공동통치자였던 아버지 구이도가 병사하자 유일한 황제이자, 이탈리아의 실질적 통치자가 되었다. 아버지 구이도 3세가 죽자 마자 아버지 구이도의 경쟁자였던 <a href="/wiki/%EB%B2%A0%EB%A0%8C%EA%B0%80%EB%A6%AC%EC%98%A4_1%EC%84%B8" title="베렌가리오 1세">베렌가리오 1세</a>는 다시 <a class="mw-redirect" href="/wiki/%EB%A1%AC%EB%B0%94%EB%A5%B4%EB%94%94%EC%95%84" title="롬바르디아">롬바르디아</a>의 수도 <a href="/wiki/%ED%8C%8C%EB%B9%84%EC%95%84" title="파비아">파비아</a>를 점령했지만 람베르토는 자신의 모후와 측근들의 강력한 지원으로 베렌가리오 군대를 다시 쫓아낼 수 있었다.</p>
<p>그러나 <a href="/wiki/894%EB%85%84" title="894년">894년</a> 초부터 아르눌프는 개인적으로 군대를 이끌고 <a href="/wiki/%EC%95%8C%ED%94%84%EC%8A%A4_%EC%82%B0%EB%A7%A5" title="알프스 산맥">알프스 산맥</a>을 넘어 이탈리아 북부의 <a href="/wiki/%ED%8F%AC_%EA%B0%95" title="포 강">포 강</a>까지 공략하였다. 람베르토는 아르눌프의 군대를 상대해야 했다. 그는 나이가 젊었지만 어머니 아겔트루드가 섭정 노릇을 하였다. 아겔트루드는 철저한 반(反) 독일주의자이기도 했다. 람베르토와 아겔트루드는 교황의 승인을 얻기 위해 <a href="/wiki/%EB%A1%9C%EB%A7%88" title="로마">로마</a>로 갔다. 람베르토 모자가 로마로 갈 무렵 베렌가리오가 <a href="/wiki/%ED%8C%8C%EB%B9%84%EC%95%84" title="파비아">파비아</a>를 점유한다. 람베르토는 다시 <a href="/wiki/%EA%B5%90%ED%99%A9_%ED%8F%AC%EB%A5%B4%EB%AA%A8%EC%86%8C" title="교황 포르모소">교황 포르모소</a>에게 신성 로마 제국 황제관을 요구하지만 포르모소는 제관을 아르눌프에게 수여했고, 람베르토 일행을 <a class="new" href="/w/index.php?title=%EC%82%B0_%ED%83%84%EC%A0%A4%EB%A1%9C_%EC%84%B1&amp;action=edit&amp;redlink=1" title="산 탄젤로 성 (없는 문서)">산 탄젤로 성</a>에 감금해버린다.</p>
<p>그는 집권기간 내내 <a href="/wiki/%EC%9D%B4%ED%83%88%EB%A6%AC%EC%95%84" title="이탈리아">이탈리아</a>를 공략하려는 <a href="/wiki/%EC%95%84%EB%A5%B4%EB%88%8C%ED%94%84_%ED%8F%B0_%EC%BC%80%EB%A5%B8%ED%85%90" title="아르눌프 폰 케른텐">아르눌프</a>와 <a class="mw-redirect" href="/wiki/%EB%B2%A0%EB%A0%9D%EA%B0%80%EB%A6%AC%EC%98%A4_1%EC%84%B8" title="베렝가리오 1세">베렝가리오 1세</a>와 전쟁하였다. <a class="mw-redirect" href="/wiki/%ED%86%A0%EC%8A%A4%EC%B9%B4%EB%82%98" title="토스카나">토스카나</a>의 <a class="new" href="/w/index.php?title=%EC%95%84%EB%8B%AC%EB%B2%A0%EB%A5%B4%ED%86%A0_2%EC%84%B8_(%ED%86%A0%EC%8A%A4%EC%B9%B4%EB%82%98)&amp;action=edit&amp;redlink=1" title="아달베르토 2세 (토스카나) (없는 문서)">아달베르토 2세</a>는 <a href="/wiki/%ED%8C%8C%EB%B9%84%EC%95%84" title="파비아">파비아</a>에 있으면서 <a class="mw-redirect" href="/wiki/%EB%B2%A0%EB%A0%9D%EA%B0%80%EB%A6%AC%EC%98%A4_1%EC%84%B8" title="베렝가리오 1세">베렝가리오 1세</a> 군대를 공격하는 저격수 역할을 수행하였다. 그러나 <a href="/wiki/896%EB%85%84" title="896년">896년</a> 무렵에는 람베르토를 배신하고 아르눌프의 편에 서게 된다. <a href="/wiki/893%EB%85%84" title="893년">893년</a>부터 <a href="/wiki/%EC%95%84%EB%A5%B4%EB%88%8C%ED%94%84_%ED%8F%B0_%EC%BC%80%EB%A5%B8%ED%85%90" title="아르눌프 폰 케른텐">아르눌프</a>가 군사를 이끌고 오자, 람베르토는 선봉으로 <a href="/wiki/%EC%95%84%EB%A5%B4%EB%88%8C%ED%94%84_%ED%8F%B0_%EC%BC%80%EB%A5%B8%ED%85%90" title="아르눌프 폰 케른텐">아르눌프</a> 군과 맞서 싸우기도 했다. <a href="/wiki/894%EB%85%84" title="894년">894년</a> 부왕 <a class="mw-redirect" href="/wiki/%EA%B5%AC%EC%9D%B4%EB%8F%84_3%EC%84%B8" title="구이도 3세">구이도 3세</a>가 죽자 <a class="mw-redirect" href="/wiki/%EC%8B%A0%EC%84%B1%EB%A1%9C%EB%A7%88%EC%A0%9C%EA%B5%AD" title="신성로마제국">신성로마제국</a>의 황위를 계승하였다. 그러나 <a href="/wiki/%EC%95%84%EB%A5%B4%EB%88%8C%ED%94%84_%ED%8F%B0_%EC%BC%80%EB%A5%B8%ED%85%90" title="아르눌프 폰 케른텐">아르눌프</a>의 침공은 계속되었고, <a href="/wiki/%EA%B5%90%ED%99%A9_%ED%8F%AC%EB%A5%B4%EB%AA%A8%EC%86%8C" title="교황 포르모소">교황 포르모소</a>에 의해 <a href="/wiki/896%EB%85%84" title="896년">896년</a> <a href="/wiki/2%EC%9B%94_21%EC%9D%BC" title="2월 21일">2월 21일</a> 황제위를 박탈당했다.</p>
<h3><span class="mw-headline" id=".EC.8A.B9.EC.9D.B8_.EA.B1.B0.EB.B6.80">승인 거부</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%8A%A4%ED%8F%B4%EB%A0%88%ED%86%A0%EC%9D%98_%EB%9E%8C%EB%B2%A0%EB%A5%B4%ED%86%A0_2%EC%84%B8&amp;action=edit&amp;section=5" title="부분 편집: 승인 거부">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p><a href="/wiki/895%EB%85%84" title="895년">895년</a> <a href="/wiki/1%EC%9B%94" title="1월">1월</a> 람베르토는 자신의 수도로 되돌아올 수 있었다. 같은 해에 그의 형제 <a class="new" href="/w/index.php?title=%EA%B5%AC%EC%9D%B4%EB%8F%84_4%EC%84%B8&amp;action=edit&amp;redlink=1" title="구이도 4세 (없는 문서)">구이도 4세</a>는 <a class="mw-redirect" href="/wiki/%EB%B9%84%EC%9E%94%ED%8B%B4_%EC%A0%9C%EA%B5%AD" title="비잔틴 제국">비잔틴 제국</a>에 도움을 얻어 <a href="/wiki/%EC%8A%A4%ED%8F%B4%EB%A0%88%ED%86%A0" title="스폴레토">스폴레토</a> 지역을 점령했다. 람베르토는 정식 황제로 승인해줄 것을 촉구했고, <a href="/wiki/%EB%9E%AD%EC%8A%A4" title="랭스">랭스</a>의 풀크 주교는 침묵했다. <a href="/wiki/895%EB%85%84" title="895년">895년</a> <a href="/wiki/9%EC%9B%94" title="9월">9월</a> 교황의 사절은 다시 아르눌프에게 도움을 청하기 위해 <a href="/wiki/%EB%A0%88%EA%B2%90%EC%8A%A4%EB%B6%80%EB%A5%B4%ED%81%AC" title="레겐스부르크">레겐스부르크</a>에 도착했다. <a href="/wiki/10%EC%9B%94" title="10월">10월</a> 아르눌프는 두 번째 이탈리아 공략에 착수했다. 람베르토 2세는 아르눌프 군과의 전투를 회피하였고, 아르눌프는 토스카나의 귀족들 사이에서 지지를 모으고 있었다. 이때 람베르토 2세의 측근이었던 토스카나의 아달베르토 역시 아르눌프의 편에 가담했다. 이들은 람베르토와 아겔트루드를 찾아내려 했고, <a href="/wiki/896%EB%85%84" title="896년">896년</a> <a href="/wiki/2%EC%9B%94_21%EC%9D%BC" title="2월 21일">2월 21일</a>에는 로마를 점령했다. 바로 <a href="/wiki/%EA%B5%90%ED%99%A9_%ED%8F%AC%EB%A5%B4%EB%AA%A8%EC%86%8C" title="교황 포르모소">포르모소</a>에 의해 제관은 <a href="/wiki/%EC%95%84%EB%A5%B4%EB%88%8C%ED%94%84_%ED%8F%B0_%EC%BC%80%EB%A5%B8%ED%85%90" title="아르눌프 폰 케른텐">아르눌프</a>에게 수여되었고, 람베르토는 황제직에서 면직이 선언되었다. 람베르토는 어머니 아겔트루드와 함께 <a href="/wiki/%EC%8A%A4%ED%8F%B4%EB%A0%88%ED%86%A0" title="스폴레토">스폴레토</a>로 도주했다. 하지만 <a href="/wiki/%EC%95%84%EB%A5%B4%EB%88%8C%ED%94%84_%ED%8F%B0_%EC%BC%80%EB%A5%B8%ED%85%90" title="아르눌프 폰 케른텐">아르눌프</a>가 <a href="/wiki/%EB%87%8C%EC%A1%B8%EC%A4%91" title="뇌졸중">뇌졸중</a>과 <a href="/wiki/%EC%A4%91%ED%92%8D" title="중풍">중풍</a>을 앓고 있었으므로 <a href="/wiki/%EC%95%84%EB%A5%B4%EB%88%8C%ED%94%84_%ED%8F%B0_%EC%BC%80%EB%A5%B8%ED%85%90" title="아르눌프 폰 케른텐">아르눌프</a>의 군대는 곧 <a class="mw-redirect" href="/wiki/%EB%8F%99%ED%94%84%EB%9E%91%ED%81%AC" title="동프랑크">동프랑크</a>로 되돌아갔고, 람베르토 2세는 곧 풀려났다. <a href="/wiki/%EC%9D%B4%ED%83%88%EB%A6%AC%EC%95%84" title="이탈리아">이탈리아</a>는 사실상 람베르토가 통치하고 있었다.</p>
<p>아르눌프가 <a href="/wiki/%EC%A4%91%ED%92%8D" title="중풍">중풍</a>과 <a href="/wiki/%EB%87%8C%EC%A1%B8%EC%A4%91" title="뇌졸중">뇌졸중</a>이 발병해서 다시 <a class="mw-redirect" href="/wiki/%EB%8F%99%ED%94%84%EB%9E%91%ED%81%AC" title="동프랑크">동프랑크</a>로 돌아가자 이탈리아의 북동부 지역을 제외한 나머지 영역은 람베르토의 영향력 하에 있었다. 람베르토는 이탈리아 중부 지역을 오가며 이탈리아의 통치자임을 과시했다. 그는 <a href="/wiki/%ED%8C%8C%EB%B9%84%EC%95%84" title="파비아">파비아</a>를 재탈환하여 마기눌프 등을 참수하였고, 동시에 <a href="/wiki/10%EC%9B%94" title="10월">10월</a>과 <a href="/wiki/11%EC%9B%94" title="11월">11월</a> <a class="mw-redirect" href="/wiki/%EB%B2%A0%EB%A0%9D%EA%B0%80%EB%A6%AC%EC%98%A4_1%EC%84%B8" title="베렝가리오 1세">베렝가리오 1세</a>에게 사람을 보내, 베렝가리오가 차지하고 있던 영역과 자신이 차지한 파비아 이남 지역에 경계선을 설정하고 이탈리아를 분할하여 서로의 주권을 인정하기로 하고, <a href="/wiki/%ED%8F%AC_%EA%B0%95" title="포 강">포 강</a>에서 휴전을 선언하였다. 람베르토 2세와 <a class="mw-redirect" href="/wiki/%EB%B2%A0%EB%A0%9D%EA%B0%80%EB%A6%AC%EC%98%A4_1%EC%84%B8" title="베렝가리오 1세">베렝가리오 1세</a>는 또한 <a class="new" href="/w/index.php?title=%EB%B2%A0%EB%A5%B4%EA%B0%80%EB%AA%AC&amp;action=edit&amp;redlink=1" title="베르가몬 (없는 문서)">베르가몬</a>을 공유하기로 약속하고 <a href="/wiki/%EB%B2%A0%EB%A0%8C%EA%B0%80%EB%A6%AC%EC%98%A4_1%EC%84%B8" title="베렌가리오 1세">베렌가리오 1세</a>의 딸 기셀라와 결혼하기로 정략결혼동맹 약속을 체결하였다. 그러나 람베르토와 베렝가리오 1세의 일시적인 휴전과 정략결혼 동맹계획은 깨졌는데, <a href="/wiki/%ED%81%AC%EB%A0%88%EB%AA%A8%EB%82%98" title="크레모나">크레모나</a>의 작가 <a class="new" href="/w/index.php?title=%EB%A6%AC%EC%9A%B0%ED%8A%B8%ED%94%84%EB%9E%80%ED%8A%B8_(%ED%81%AC%EB%A0%88%EB%AA%A8%EB%82%98)&amp;action=edit&amp;redlink=1" title="리우트프란트 (크레모나) (없는 문서)">리우르프란트</a>는 이탈리아가 두 왕으로 인해 고통을 받는다고 지적했다. 이는 이런 저런 말을 낳다가 람베르토와 베렝가리오 1세의 동맹관계는 깨지게 된다.</p>
<h3><span class="mw-headline" id=".EC.8B.9C.EC.8B.A0_.EC.9E.AC.ED.8C.90">시신 재판</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%8A%A4%ED%8F%B4%EB%A0%88%ED%86%A0%EC%9D%98_%EB%9E%8C%EB%B2%A0%EB%A5%B4%ED%86%A0_2%EC%84%B8&amp;action=edit&amp;section=6" title="부분 편집: 시신 재판">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p><a href="/wiki/896%EB%85%84" title="896년">896년</a> 람베르토는 다시 <a href="/wiki/%EA%B5%90%ED%99%A9_%ED%8F%AC%EB%A5%B4%EB%AA%A8%EC%86%8C" title="교황 포르모소">교황 포르모소</a>를 찾아갔지만 그해 <a href="/wiki/4%EC%9B%94_4%EC%9D%BC" title="4월 4일">4월 4일</a> <a href="/wiki/%EA%B5%90%ED%99%A9_%ED%8F%AC%EB%A5%B4%EB%AA%A8%EC%86%8C" title="교황 포르모소">포르모소</a>는 병으로 죽었다. 황제관 박탈에 분개한 그는 <a href="/wiki/%EA%B5%90%ED%99%A9_%ED%8F%AC%EB%A5%B4%EB%AA%A8%EC%86%8C" title="교황 포르모소">포르모소</a> 사후 자신들의 측근들을 시켜 포르모수스의 묘소를 파헤치기도 했다. 한편 람베르토는 아르눌프에게 협력했던 밀라노의 백작 마기눌프 등을 <a href="/wiki/%ED%8C%8C%EB%B9%84%EC%95%84" title="파비아">파비아</a>에서 참수형에 처하는 등의 조치를 취하여 지지층의 이탈을 막기도 했다. 이어 <a href="/wiki/%EA%B5%90%ED%99%A9_%EC%8A%A4%ED%85%8C%ED%8C%8C%EB%85%B8_6%EC%84%B8" title="교황 스테파노 6세">교황 스테파노 6세</a>를 사주하여 죽은 <a href="/wiki/%EA%B5%90%ED%99%A9_%ED%8F%AC%EB%A5%B4%EB%AA%A8%EC%86%8C" title="교황 포르모소">포르모소</a>를 재판하는 이벤트를 벌이기도 했다. 죽은 <a href="/wiki/%EA%B5%90%ED%99%A9_%ED%8F%AC%EB%A5%B4%EB%AA%A8%EC%86%8C" title="교황 포르모소">교황 포르모소</a>의 석관에서 시신을 꺼낸 뒤, 교황의 옷을 입히고 재판을 하였다. 그리고 심판관들이 죄를 물으면 포르모소 시신의 뒤에 있는 인물이 시신의 손가락을 움직이며 죄를 자복한다고 하는 식의 재판이었다. 이 재판에서 <a href="/wiki/%EA%B5%90%ED%99%A9_%EC%8A%A4%ED%85%8C%ED%8C%8C%EB%85%B8_6%EC%84%B8" title="교황 스테파노 6세">교황 스테파노 6세</a>는 <a href="/wiki/%EA%B5%90%ED%99%A9_%ED%8F%AC%EB%A5%B4%EB%AA%A8%EC%86%8C" title="교황 포르모소">교황 포르모소</a>의 교황 선출은 무효라는 판결을 내린다.</p>
<p>람베르토 2세와 <a class="new" href="/w/index.php?title=%EA%B5%90%ED%99%A9_%EC%8A%A4%ED%85%8C%ED%8C%8C%EB%88%84%EC%8A%A4_6%EC%84%B8&amp;action=edit&amp;redlink=1" title="교황 스테파누스 6세 (없는 문서)">교황 스테파누스 6세</a>는 <a href="/wiki/%EA%B5%90%ED%99%A9_%ED%8F%AC%EB%A5%B4%EB%AA%A8%EC%86%8C" title="교황 포르모소">교황 포르모소</a>의 시체에 교황 옷을 입히고 교황자리에 앉힌 뒤 <a href="/wiki/%EA%B5%90%ED%99%A9_%ED%8F%AC%EB%A5%B4%EB%AA%A8%EC%86%8C" title="교황 포르모소">교황 포르모소</a>의 교황직을 폐하고, 그가 시행한 교회 훈령과 포르모소가 생전 수여한 성직자 임명을 전부 무효라고 선언했다. 곧이어 스테파누스는 포르모소의 시체를 <a href="/wiki/%EB%A1%9C%EB%A7%88" title="로마">로마</a>시내 거리로 끌고 다녀 조리돌림을 한 뒤 <a href="/wiki/%ED%85%8C%EB%B2%A0%EB%A0%88_%EA%B0%95" title="테베레 강">테베레 강</a>에 던져 빠뜨리도록 명령함으로써 그 재판을 끝냈다. 그러나 <a href="/wiki/898%EB%85%84" title="898년">898년</a> <a href="/wiki/1%EC%9B%94" title="1월">1월</a> 정변이 발생해 <a href="/wiki/%EA%B5%90%ED%99%A9_%EC%8A%A4%ED%85%8C%ED%8C%8C%EB%85%B8_6%EC%84%B8" title="교황 스테파노 6세">교황 스테파노 6세</a>가 퇴출당하고 <a class="new" href="/w/index.php?title=%EC%9A%94%ED%95%9C%EB%84%A4%EC%8A%A4_9%EC%84%B8&amp;action=edit&amp;redlink=1" title="요한네스 9세 (없는 문서)">요한네스 9세</a>가 <a href="/wiki/%EA%B5%90%ED%99%A9" title="교황">교황</a>으로 즉위하면서 판결은 뒤집힌다. <a class="new" href="/w/index.php?title=%EA%B5%90%ED%99%A9_%EC%9A%94%ED%95%9C%EB%84%A4%EC%8A%A4_9%EC%84%B8&amp;action=edit&amp;redlink=1" title="교황 요한네스 9세 (없는 문서)">교황 요한네스 9세</a>는 즉위 직후 <a href="/wiki/%EB%A1%9C%EB%A7%88" title="로마">로마</a>, <a href="/wiki/%EB%9D%BC%EB%B2%A4%EB%82%98" title="라벤나">라벤나</a>에서 공의회 소집을 명하고 <a href="/wiki/%EA%B5%90%ED%99%A9_%ED%8F%AC%EB%A5%B4%EB%AA%A8%EC%86%8C" title="교황 포르모소">교황 포르모소</a>를 복권시켰으며, <a href="/wiki/%EA%B5%90%ED%99%A9_%EC%8A%A4%ED%85%8C%ED%8C%8C%EB%85%B8_6%EC%84%B8" title="교황 스테파노 6세">교황 스테파노 6세</a>는 <a href="/wiki/%EA%B5%90%ED%99%A9_%ED%8F%AC%EB%A5%B4%EB%AA%A8%EC%86%8C" title="교황 포르모소">교황 포르모소</a>의 시체를 재판하여 그의 교황선출과 정책이 무효라고 선언한 것을 다시 뒤집었다. <a class="new" href="/w/index.php?title=%EA%B5%90%ED%99%A9_%EC%9A%94%ED%95%9C%EB%84%A4%EC%8A%A4_9%EC%84%B8&amp;action=edit&amp;redlink=1" title="교황 요한네스 9세 (없는 문서)">교황 요한네스 9세</a>가 소집한 공의회의 성직자들은 스테파노의 재판과 성직자 회의를 불법으로 규정하고 그 교령들을 파기했다. 또한 스테파누스와 그 일파에게 폐위당한 성직자들을 복권시켰다.</p>
<p><a href="/wiki/898%EB%85%84" title="898년">898년</a> <a href="/wiki/2%EC%9B%94" title="2월">2월</a> 람베르토 2세는 <a href="/wiki/%EB%9D%BC%EB%B2%A4%EB%82%98" title="라벤나">라벤나</a>에서 제국 의회를 소집, 70명의 주교단과 만나 <a href="/wiki/891%EB%85%84" title="891년">891년</a>의 자신의 즉위의 정당성을 강조하고, 아르눌프의 대관식 무효와 람베르토 2세의 제국 황제 지위의 적법성의 동의를 구하였다.</p>
<h3><span class="mw-headline" id=".EC.83.9D.EC.95.A0_.ED.9B.84.EB.B0.98">생애 후반</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%8A%A4%ED%8F%B4%EB%A0%88%ED%86%A0%EC%9D%98_%EB%9E%8C%EB%B2%A0%EB%A5%B4%ED%86%A0_2%EC%84%B8&amp;action=edit&amp;section=7" title="부분 편집: 생애 후반">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>람베르토 2세는 아버지 구이도 3세의 "새로운 프랑크 왕국(Renovatio Francorum)"이라는 정책을 계승하면서, 동시에 선대 프랑크 왕들의 법령 역시 계승하였다. 람베르토 2세는 또 <a href="/wiki/824%EB%85%84" title="824년">824년</a>의 프랑크 황제 로타르 1세의 로마 헌장(Constitutio Romana) 부활을 선언했는데, <a class="new" href="/w/index.php?title=%EA%B5%90%ED%99%A9_%EC%9A%94%ED%95%9C%EB%84%A4%EC%8A%A4_9%EC%84%B8&amp;action=edit&amp;redlink=1" title="교황 요한네스 9세 (없는 문서)">교황 요한네스 9세</a>는 로타르의 로마 헌장을 다시 승인하여, 교황 선출 때 황제의 사절을 의무적으로 참석하도록 확정하였다. <a href="/wiki/898%EB%85%84" title="898년">898년</a> 초, 람베르토 2세는 아버지 구이도의 정책을 명문화한 우티넨의 로마법(Lex Romana Utinensis)을 제정하였다. <a href="/wiki/895%EB%85%84" title="895년">895년</a> <a href="/wiki/5%EC%9B%94" title="5월">5월</a> 아르눌프의 황제관 수여 무효를 선언했다. 그러나 이탈리아 북동부는 여전히 <a class="mw-redirect" href="/wiki/%EB%B2%A0%EB%A0%9D%EA%B0%80%EB%A6%AC%EC%98%A4_1%EC%84%B8" title="베렝가리오 1세">베렝가리오 1세</a>의 세력이 강건했고, 그는 <a href="/wiki/%ED%86%A0%EC%8A%A4%EC%B9%B4" title="토스카">토스카</a>나의 반항적인 아달베르토를 상대해야 했다.</p>
<p><a href="/wiki/898%EB%85%84" title="898년">898년</a> 초가을, 그는 <a class="mw-redirect" href="/wiki/%EB%B2%A0%EB%A0%9D%EA%B0%80%EB%A6%AC%EC%98%A4_1%EC%84%B8" title="베렝가리오 1세">베렝가리오 1세</a>와 교전했지만 패하였다. 그러나 베렝가리오는 파비아 남쪽으로 내려오지 못했고 여전히 이탈리아 중남부는 람베르토의 영향력 하에 있었다. <a href="/wiki/898%EB%85%84" title="898년">898년</a> <a href="/wiki/10%EC%9B%94_15%EC%9D%BC" title="10월 15일">10월 15일</a> 사냥을 위해 <a href="/wiki/%ED%8C%8C%EB%B9%84%EC%95%84" title="파비아">파비아</a>에 갔다가 <a href="/wiki/%EB%B0%80%EB%9D%BC%EB%85%B8" title="밀라노">밀라노</a> 남부의 <a class="mw-disambig" href="/wiki/%EB%A7%88%EB%A0%9D%EA%B3%A0" title="마렝고">마렝고</a>로 가서 사냥하였다. 사냥 도중 그는 <a class="new" href="/w/index.php?title=%ED%94%BC%EB%8D%B4%EC%B0%A8&amp;action=edit&amp;redlink=1" title="피덴차 (없는 문서)">피덴차</a>의 <a class="new" href="/w/index.php?title=%EB%B3%B4%EB%A5%B4%EB%8F%84_%EC%82%B0_%EB%8F%84%EB%AF%B8%EB%85%B8&amp;action=edit&amp;redlink=1" title="보르도 산 도미노 (없는 문서)">보르도 산 도미노</a>(Borgo San Donnino)에서 경쟁자가 보낸 자객에 의해 말 위에서 떨어져 낙마한 뒤 암살되었다. <a href="/wiki/%ED%81%AC%EB%A0%88%EB%AA%A8%EB%82%98" title="크레모나">크레모나</a>의 주교 <a class="new" href="/w/index.php?title=%EB%A6%AC%EC%9A%B0%ED%8A%B8%ED%94%84%EB%9E%80%ED%8A%B8_(%ED%81%AC%EB%A0%88%EB%AA%A8%EB%82%98)&amp;action=edit&amp;redlink=1" title="리우트프란트 (크레모나) (없는 문서)">리우르프란트</a>는 <a class="new" href="/w/index.php?title=%EB%A7%88%EA%B8%B0%EB%88%8C%ED%94%84&amp;action=edit&amp;redlink=1" title="마기눌프 (없는 문서)">마기눌프</a>의 아들 우고를 암살자로 지목했다.</p>
<h3><span class="mw-headline" id=".EC.82.AC.ED.9B.84">사후</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%8A%A4%ED%8F%B4%EB%A0%88%ED%86%A0%EC%9D%98_%EB%9E%8C%EB%B2%A0%EB%A5%B4%ED%86%A0_2%EC%84%B8&amp;action=edit&amp;section=8" title="부분 편집: 사후">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>그의 시신은 곧 수습되어 <a href="/wiki/%ED%8C%8C%EB%B9%84%EC%95%84" title="파비아">파비아</a>에서 <a href="/wiki/%ED%94%BC%EC%95%84%EC%B2%B8%EC%B0%A8" title="피아첸차">피아첸차</a>로 운구되어 팔라센티(Placentiae)에 매장되었다. 그는 아버지 <a class="mw-redirect" href="/wiki/%EA%B5%AC%EC%9D%B4%EB%8F%84_3%EC%84%B8" title="구이도 3세">구이도 3세</a>와 함께 혼란에 빠진 틈을 타서 <a href="/wiki/%EC%9D%B4%ED%83%88%EB%A6%AC%EC%95%84%EC%9D%98_%EA%B5%B0%EC%A3%BC" title="이탈리아의 군주">이탈리아의 왕위</a>와 <a class="mw-redirect" href="/wiki/%EC%8B%A0%EC%84%B1%EB%A1%9C%EB%A7%88%ED%99%A9%EC%A0%9C" title="신성로마황제">신성로마황제</a>의 자리를 차지했다. 그러나 이는 곧 많은 제후들의 반발을 샀고, 많은 <a href="/wiki/%EC%9D%B4%ED%83%88%EB%A6%AC%EC%95%84" title="이탈리아">이탈리아</a>와 <a href="/wiki/%EB%8F%85%EC%9D%BC" title="독일">독일</a>의 제후들이 <a href="/wiki/%EC%8B%A0%EC%84%B1_%EB%A1%9C%EB%A7%88_%EC%A0%9C%EA%B5%AD" title="신성 로마 제국">신성 로마 제국</a> 황제위를 넘보는 원인이 되었다.</p>
<p>그의 죽음을 기점으로 이탈리아는 여러 통치자들이 난립하게 되었다. 람베르토에게는 남자 후계자가 없었으므로, <a class="new" href="/w/index.php?title=%EC%8A%A4%ED%8F%B4%EB%A0%88%ED%86%A0_%EA%B3%B5%EC%9E%91&amp;action=edit&amp;redlink=1" title="스폴레토 공작 (없는 문서)">스폴레토 공작</a>위는 그의 동생인 <a class="new" href="/w/index.php?title=%EC%8A%A4%ED%8F%B4%EB%A0%88%ED%86%A0%EC%9D%98_%EA%B7%80%EB%8F%84_4%EC%84%B8&amp;action=edit&amp;redlink=1" title="스폴레토의 귀도 4세 (없는 문서)">귀도 4세</a>가 계승하였다. 그러나 귀도 4세는 <a class="mw-redirect" href="/wiki/%EB%8F%99%ED%94%84%EB%9E%91%ED%81%AC" title="동프랑크">동프랑크</a>의 귀족가 출신인 <a class="new" href="/w/index.php?title=%EC%95%8C%EB%B2%A0%EB%A6%AC%ED%81%AC_1%EC%84%B8&amp;action=edit&amp;redlink=1" title="알베리크 1세 (없는 문서)">알베리크 1세</a>(Alberic I)에게 살해당하고 이후 스폴레토 공작직과 이탈리아 남부지역의 패권은 알베리크 가문과 <a href="/wiki/%EB%A7%88%EB%A1%9C%EC%B9%98%EC%95%84" title="마로치아">마로치아</a> 가문이 차지하게 된다. 한편 람베르토가 죽자마자 수일 안에 <a href="/wiki/%ED%94%84%EB%A6%AC%EC%9A%B8%EB%A6%AC" title="프리울리">프리울리</a>의 <a class="mw-redirect" href="/wiki/%EB%B2%A0%EB%A0%9D%EA%B0%80%EB%A6%AC%EC%98%A4_1%EC%84%B8" title="베렝가리오 1세">베렝가리오 1세</a>가 <a href="/wiki/%ED%8C%8C%EB%B9%84%EC%95%84" title="파비아">파비아</a> 이남 지역을 점령하였다. 이탈리아 북부는 베렝가리오 1세와 그의 경쟁자들이 난립하여 혼란이 발생하였다.</p>
