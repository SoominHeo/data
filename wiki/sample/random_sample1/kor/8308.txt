<p><b>스푸트니크 1호</b>(<span style="font-size: smaller;"><a href="/wiki/%EB%9F%AC%EC%8B%9C%EC%95%84%EC%96%B4" title="러시아어">러시아어</a>:</span> <span lang="ru" xml:lang="ru">Спутник-1</span>)는 <a href="/wiki/%EC%86%8C%EB%A0%A8" title="소련">소련</a>이 <a href="/wiki/1957%EB%85%84" title="1957년">1957년</a> <a href="/wiki/10%EC%9B%94_4%EC%9D%BC" title="10월 4일">10월 4일</a>에 타원형의 <a href="/wiki/%EC%A7%80%EA%B5%AC_%EC%A0%80%EA%B6%A4%EB%8F%84" title="지구 저궤도">지구 저궤도</a>로 발사한 최초의 <a href="/wiki/%EC%9D%B8%EA%B3%B5%EC%9C%84%EC%84%B1" title="인공위성">인공위성</a>이다. 스푸트니크 1호의 성공은 미국의 <a class="mw-redirect" href="/wiki/%EC%8A%A4%ED%91%B8%ED%8A%B8%EB%8B%88%ED%81%AC_%EC%87%BC%ED%81%AC" title="스푸트니크 쇼크">스푸트니크 쇼크</a>를 촉발시켰고, <a class="new" href="/w/index.php?title=%EC%9A%B0%EC%A3%BC%EC%8B%9C%EB%8C%80&amp;action=edit&amp;redlink=1" title="우주시대 (없는 문서)">우주시대</a>와 <a class="mw-redirect" href="/wiki/%EC%9A%B0%EC%A3%BC%EA%B2%BD%EC%9F%81" title="우주경쟁">우주경쟁</a>의 방아쇠를 당겼으며, <a href="/wiki/%EB%83%89%EC%A0%84" title="냉전">냉전</a>기의 정치 상황에도 영향을 주었다. 스푸트니크 1호의 발사는 새로운 정치, 군사, 기술 및 과학발전을 이끌었다.<sup class="reference" id="cite_ref-test_1-0"><a href="#cite_note-test-1">[1]</a></sup><sup class="reference" id="cite_ref-2"><a href="#cite_note-2">[2]</a></sup></p>
<p>스푸트니크 1호는 과학적으로 중요한데, 위성의 궤도에 영향을 미치는 <a class="new" href="/w/index.php?title=%EA%B3%B5%EA%B8%B0%EC%A0%80%ED%95%AD&amp;action=edit&amp;redlink=1" title="공기저항 (없는 문서)">공기저항</a>으로 <a href="/wiki/%EC%A7%80%EA%B5%AC_%EB%8C%80%EA%B8%B0%EA%B6%8C" title="지구 대기권">지구 대기권</a>의 밀도를 추측할 수 있었고, 무선신호의 전파는 <a href="/wiki/%EC%A0%84%EB%A6%AC%EC%B8%B5" title="전리층">전리층</a>에 대한 정보를 가져다주었다.</p>
<p>발사는 <a class="new" href="/w/index.php?title=%EA%B5%AD%EC%A0%9C%EC%A7%80%EA%B5%AC%EB%AC%BC%EB%A6%AC%EA%B4%80%EC%B8%A1%EB%85%84&amp;action=edit&amp;redlink=1" title="국제지구물리관측년 (없는 문서)">국제지구물리관측년</a>인 <a href="/wiki/1957%EB%85%84" title="1957년">1957년</a> <a href="/wiki/10%EC%9B%94_4%EC%9D%BC" title="10월 4일">10월 4일</a> 19시 28분 34초(<a href="/wiki/%ED%98%91%EC%A0%95_%EC%84%B8%EA%B3%84%EC%8B%9C" title="협정 세계시">국제 표준시(UTC)</a> 기준)에 <a href="/wiki/%EC%B9%B4%EC%9E%90%ED%9D%90_%EC%86%8C%EB%B9%84%EC%97%90%ED%8A%B8_%EC%82%AC%ED%9A%8C%EC%A3%BC%EC%9D%98_%EA%B3%B5%ED%99%94%EA%B5%AD" title="카자흐 소비에트 사회주의 공화국">카자흐 소비에트 사회주의 공화국</a>의 <a href="/wiki/%EB%B0%94%EC%9D%B4%EC%BD%94%EB%88%84%EB%A5%B4_%EC%9A%B0%EC%A3%BC_%EA%B8%B0%EC%A7%80" title="바이코누르 우주 기지">바이코누르 우주 기지</a>에서 행해졌다. 위성은 29,000km/h의 속도로 비행했으며, 궤도를 한 번 도는데 96.2분 소요되었다. 위성은 전 세계의 <a href="/wiki/%EC%95%84%EB%A7%88%EC%B6%94%EC%96%B4_%EB%AC%B4%EC%84%A0" title="아마추어 무선">아마추어 무선</a> 통신가에 의해 20.005와 40.002<a class="mw-redirect" href="/wiki/MHz" title="MHz">MHz</a><sup class="reference" id="cite_ref-3"><a href="#cite_note-3">[3]</a></sup> 대역에서 관측되었다.<sup class="reference" id="cite_ref-4"><a href="#cite_note-4">[4]</a></sup> 1957년 10월 26일에 송신기의 배터리가 다 떨어질 때까지 신호는 22일동안 계속되었다.<sup class="reference" id="cite_ref-5"><a href="#cite_note-5">[5]</a></sup> 스푸트니크 1호는 3개월동안 약 6천만km를 비행한 뒤, 1958년 1월 4일에 대기권에 재진입하여 불타버렸다.<sup class="reference" id="cite_ref-6"><a href="#cite_note-6">[6]</a></sup></p>

<h2><span class="mw-headline" id=".EB.B0.9C.EC.82.AC_.EC.9D.B4.EC.A0.84">발사 이전</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%8A%A4%ED%91%B8%ED%8A%B8%EB%8B%88%ED%81%AC_1%ED%98%B8&amp;action=edit&amp;section=1" title="부분 편집: 발사 이전">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<h3><span class="mw-headline" id=".EC.9C.84.EC.84.B1_.EC.A0.9C.EC.9E.91_.ED.94.84.EB.A1.9C.EC.A0.9D.ED.8A.B8">위성 제작 프로젝트</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%8A%A4%ED%91%B8%ED%8A%B8%EB%8B%88%ED%81%AC_1%ED%98%B8&amp;action=edit&amp;section=2" title="부분 편집: 위성 제작 프로젝트">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>스푸트니크 1호의 역사는, 1954년 12월 17일에 <a href="/wiki/%EC%84%B8%EB%A5%B4%EA%B2%8C%EC%9D%B4_%EC%BD%94%EB%A1%A4%EB%A3%8C%ED%94%84" title="세르게이 코롤료프">세르게이 코롤료프</a>가 <a class="new" href="/w/index.php?title=%EB%94%94%EB%AF%B8%ED%8A%B8%EB%A6%AC_%EC%95%88%ED%86%A0%EB%8B%88%EC%98%A4%EC%9A%B0&amp;action=edit&amp;redlink=1" title="디미트리 안토니오우 (없는 문서)">디미트리 안토니오우</a> 국방산업장관에게 지구궤도를 도는 인공위성의 개발을 제안하는 연설에서 시작했다. 코롤료프는 또한 유스티노프에게 해외 유사 프로젝트의 개요에 대한 <a class="new" href="/w/index.php?title=Mikhail_Tikhonravov&amp;action=edit&amp;redlink=1" title="Mikhail Tikhonravov (없는 문서)">Mikhail Tikhonravov</a>의 보고서를 전달했다.<sup class="reference" id="cite_ref-7"><a href="#cite_note-7">[7]</a></sup> Tikhonravov는 1951년에 로켓 개발에서 인공위성 발사는 피할 수 없는 단계임을 강조했으며, 그 이후 행성간 커뮤니케이션이 가능하리라고 했다.<sup class="reference" id="cite_ref-ZE_8-0"><a href="#cite_note-ZE-8">[8]</a></sup> 1955년 7월 29일에 미국의 대통령 <a href="/wiki/%EB%93%9C%EC%99%80%EC%9D%B4%ED%8A%B8_D._%EC%95%84%EC%9D%B4%EC%A0%A0%ED%95%98%EC%9B%8C" title="드와이트 D. 아이젠하워">드와이트 D. 아이젠하워</a>가 미국이 국제지구물리관측년동안 인공위성을 발사할 것이라고 발표했다.<sup class="reference" id="cite_ref-9"><a href="#cite_note-9">[9]</a></sup> 이후 8월 8일에 <a class="mw-redirect" href="/wiki/%EC%86%8C%EB%B9%84%EC%97%90%ED%8A%B8_%EC%97%B0%EB%B0%A9_%EA%B3%B5%EC%82%B0%EB%8B%B9_%EC%A0%95%EC%B9%98%EA%B5%AD" title="소비에트 연방 공산당 정치국">소비에트 연방 공산당 정치국</a>은 인공위성을 만드는 계획을 승인했다.<sup class="reference" id="cite_ref-10"><a href="#cite_note-10">[10]</a></sup> 8월 30일 <a class="mw-redirect" href="/wiki/R-7_%EB%A1%9C%EC%BC%93" title="R-7 로켓">R-7 로켓</a> 시험 발사의 책임자인 <a class="new" href="/w/index.php?title=Vasily_Ryabikov&amp;action=edit&amp;redlink=1" title="Vasily Ryabikov (없는 문서)">Vasily Ryabikov</a>와의 회의에서, 코롤료프는 달궤도로의 우주비행에 대한 계산 데이터를 제시했다. 그들은 인공위성 발사를 위한 R-7 로켓의 3단계 버전을 개발하기로 결정했다.<sup class="reference" id="cite_ref-11"><a href="#cite_note-11">[11]</a></sup></p>
<p><br/>
"스푸트니크(Спутник)"란 러시아어로 "여행의 동반자"란 뜻이다. 스푸트니크 1호는 <a href="/wiki/%EB%9F%AC%EC%8B%9C%EC%95%84" title="러시아">러시아</a> 우주계획의 선구자인 <a class="mw-redirect" href="/wiki/%EC%BD%98%EC%8A%A4%ED%83%84%ED%8B%B4_%EC%B9%98%EC%98%AC%EC%BD%94%ED%94%84%EC%8A%A4%ED%82%A4" title="콘스탄틴 치올코프스키">콘스탄틴 치올코프스키</a>의 탄생 100주년과 <a class="new" href="/w/index.php?title=%EA%B5%AD%EC%A0%9C_%EC%A7%80%EA%B5%AC_%EA%B4%80%EC%B8%A1%EB%85%84&amp;action=edit&amp;redlink=1" title="국제 지구 관측년 (없는 문서)">국제 지구 관측년</a>(1957년 7월 1일∼1958년 12월 31일까지의 18개월간의 기간)의 기간에 맞추어 발사되었다.</p>
<p>위성의 외형은, 직경 58cm의 <a href="/wiki/%EC%95%8C%EB%A3%A8%EB%AF%B8%EB%8A%84" title="알루미늄">알루미늄</a>제 구이며 거기에 4개의 안테나가 한 방향으로 붙어 있는 것이었다. 중량은 83.6kg이다. 20MHz와 40MHz 짜리 2개의 송신기(출력 1W)를 탑재하고 있어, 위성의 온도 정보를 0.3초마다 발신했다.</p>
<p>발사에 사용된 로켓은 <a class="mw-redirect" href="/wiki/R-7_%EB%A1%9C%EC%BC%93" title="R-7 로켓">R-7 로켓</a>이다. 위성의 궤도는 원지점 약 950km, 근지점 약 230km, 궤도 경사각 65°의 타원 궤도이며, <a class="new" href="/w/index.php?title=%EA%B6%A4%EB%8F%84_%EC%A3%BC%EA%B8%B0&amp;action=edit&amp;redlink=1" title="궤도 주기 (없는 문서)">궤도 주기</a>는 96.2분이었다.</p>
