<p><b>Btrfs</b>(B-tree file system 또는 Butter file system<sup class="reference" id="cite_ref-10"><a href="#cite_note-10">[9]</a></sup>, Better F S<sup class="reference" id="cite_ref-CM090622_9-1"><a href="#cite_note-CM090622-9">[8]</a></sup>)는 <a href="/wiki/%ED%8C%8C%EC%9D%BC_%EC%8B%9C%EC%8A%A4%ED%85%9C" title="파일 시스템">파일 시스템</a> 가운데 하나로 현재 <a href="/wiki/%ED%8E%98%EC%9D%B4%EC%8A%A4%EB%B6%81" title="페이스북">페이스북</a>의 <a class="new" href="/w/index.php?title=%ED%81%AC%EB%A6%AC%EC%8A%A4_%EB%A9%94%EC%9D%B4%EC%8A%A8&amp;action=edit&amp;redlink=1" title="크리스 메이슨 (없는 문서)">크리스 메이슨</a>이 개발을 지휘하고 있다. 꽤 안정화되어 시험적으로 사용하는곳들이 생기고 있다.</p>
<p><a href="/wiki/GNU_%EC%9D%BC%EB%B0%98_%EA%B3%B5%EC%A4%91_%EC%82%AC%EC%9A%A9_%ED%97%88%EA%B0%80%EC%84%9C" title="GNU 일반 공중 사용 허가서">GNU 일반 공중 사용 허가서</a>를 따르고 있으며 2014년 2월 5일 현재 최신판은 <b>3.13</b>(<a href="/wiki/%EB%A6%AC%EB%88%85%EC%8A%A4" title="리눅스">리눅스</a> 커널 2.6.32 이후 릴리즈된 커널에 기본으로 포함)이다.</p>

<h2><span class="mw-headline" id=".EC.97.AD.EC.82.AC">역사</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Btrfs&amp;action=edit&amp;section=1" title="부분 편집: 역사">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>Btrfs의 핵심 데이터 구조인 카피온라이트 <a href="/wiki/B_%ED%8A%B8%EB%A6%AC" title="B 트리">B 트리</a>(copy-on-write B-tree)는 본래 IBM의 연구원 Ohad Rodeh이 <a class="new" href="/w/index.php?title=USENIX&amp;action=edit&amp;redlink=1" title="USENIX (없는 문서)">USENIX</a> 2007의 발표에서 제안하였다.<sup class="reference" id="cite_ref-rodeh-1_11-0"><a href="#cite_note-rodeh-1-11">[10]</a></sup></p>
<h2><span class="mw-headline" id=".EA.B8.B0.EB.8A.A5">기능</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Btrfs&amp;action=edit&amp;section=2" title="부분 편집: 기능">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>Btrfs가 가질 주기능은 다음과 같다.</p>
<ul>
<li>동적 <a href="/wiki/%EC%95%84%EC%9D%B4%EB%85%B8%EB%93%9C" title="아이노드">아이노드</a> 할당</li>
<li>기록 가능 스냅샷, 스냅샷에 대한 스냅샷</li>
<li>하위 볼륨</li>
<li>오브젝트 차원에서의 미러링 및 스트리핑</li>
<li><a href="/wiki/Zlib" title="Zlib">zlib</a>을 통한 자체 압축</li>
<li>온라인 및 오프라인 파일 시스템 검사</li>
<li><a href="/wiki/Ext3" title="Ext3">ext3</a> ↔ btrfs 상호간 변환</li>
<li><a href="/wiki/%EC%86%94%EB%A6%AC%EB%93%9C_%EC%8A%A4%ED%85%8C%EC%9D%B4%ED%8A%B8_%EB%93%9C%EB%9D%BC%EC%9D%B4%EB%B8%8C" title="솔리드 스테이트 드라이브">솔리드 스테이트 드라이브</a> 최적화 모드</li>
<li>온라인 <a href="/wiki/%EB%8B%A8%ED%8E%B8%ED%99%94_%EC%A0%9C%EA%B1%B0" title="단편화 제거">단편화 제거</a></li>
<li>시드 디바이스</li>
</ul>
<h2><span class="mw-headline" id=".EA.B0.81.EC.A3.BC">각주</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Btrfs&amp;action=edit&amp;section=3" title="부분 편집: 각주">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
