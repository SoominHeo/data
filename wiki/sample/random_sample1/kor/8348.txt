<p><b>시가 현</b>(<span style="font-size: smaller;"><a href="/wiki/%EC%9D%BC%EB%B3%B8%EC%96%B4" title="일본어">일본어</a>:</span> <span lang="ja" xml:lang="ja">滋賀県</span>)은 <a href="/wiki/%EC%9D%BC%EB%B3%B8" title="일본">일본</a>의 <a href="/wiki/%EA%B8%B4%ED%82%A4_%EC%A7%80%EB%B0%A9" title="긴키 지방">긴키 지방</a>에 있는 <a href="/wiki/%EC%9D%BC%EB%B3%B8%EC%9D%98_%ED%96%89%EC%A0%95_%EA%B5%AC%EC%97%AD" title="일본의 행정 구역">현</a>. <a href="/wiki/%EA%B5%90%ED%86%A0_%EB%B6%80" title="교토 부">교토 부</a>, <a href="/wiki/%EB%AF%B8%EC%97%90_%ED%98%84" title="미에 현">미에 현</a>, <a href="/wiki/%EA%B8%B0%ED%9B%84_%ED%98%84" title="기후 현">기후 현</a>, <a href="/wiki/%ED%9B%84%EC%BF%A0%EC%9D%B4_%ED%98%84" title="후쿠이 현">후쿠이 현</a>에 둘러싸여, 중앙에는 <a href="/wiki/%EB%B9%84%EC%99%80_%ED%98%B8" title="비와 호">비와 호</a>(<span style="font-size: smaller;"><a href="/wiki/%EC%9D%BC%EB%B3%B8%EC%96%B4" title="일본어">일본어</a>:</span> <span lang="ja" xml:lang="ja">琵琶湖</span>)가 있다. 현청소재지는 <a href="/wiki/%EC%98%A4%EC%93%B0_%EC%8B%9C" title="오쓰 시">오쓰 시</a>이다.</p>

<h2><span class="mw-headline" id=".EC.97.AD.EC.82.AC">역사</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%8B%9C%EA%B0%80_%ED%98%84&amp;action=edit&amp;section=1" title="부분 편집: 역사">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>시가 현은 도도부현 체제가 성립되기 이전에 <a href="/wiki/%EC%98%A4%EB%AF%B8_%EA%B5%AD" title="오미 국">오미 국</a>으로 알려져있었다.</p>
<h2><span class="mw-headline" id=".EC.A7.80.EB.A6.AC">지리</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%8B%9C%EA%B0%80_%ED%98%84&amp;action=edit&amp;section=2" title="부분 편집: 지리">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<div class="thumb tleft">
<div class="thumbinner" style="width:177px;"><a class="image" href="/wiki/%ED%8C%8C%EC%9D%BC:Lake_biwa.jpg"><img alt="" class="thumbimage" data-file-height="1399" data-file-width="1024" height="239" src="//upload.wikimedia.org/wikipedia/commons/thumb/d/df/Lake_biwa.jpg/175px-Lake_biwa.jpg" srcset="//upload.wikimedia.org/wikipedia/commons/thumb/d/df/Lake_biwa.jpg/263px-Lake_biwa.jpg 1.5x, //upload.wikimedia.org/wikipedia/commons/thumb/d/df/Lake_biwa.jpg/350px-Lake_biwa.jpg 2x" width="175"/></a>
<div class="thumbcaption">
<div class="magnify"><a class="internal" href="/wiki/%ED%8C%8C%EC%9D%BC:Lake_biwa.jpg" title="실제 크기로"></a></div>
비와 호 주변의 위성 사진</div>
</div>
</div>
<p>시가 현은 북쪽으로 <a href="/wiki/%ED%9B%84%EC%BF%A0%EC%9D%B4_%ED%98%84" title="후쿠이 현">후쿠이 현</a>, 동쪽으로 <a href="/wiki/%EA%B8%B0%ED%9B%84_%ED%98%84" title="기후 현">기후 현</a>, 남동쪽으로 <a href="/wiki/%EB%AF%B8%EC%97%90_%ED%98%84" title="미에 현">미에 현</a>, 서쪽으로 <a href="/wiki/%EA%B5%90%ED%86%A0_%EB%B6%80" title="교토 부">교토 부</a>와 접한다. 현은 네 개의 다른 지역인 고호쿠(호수의 북쪽), 고세이(호수의 서쪽), 고토(호수의 동쪽), 고난(호수의 남쪽)을 포함한다.</p>
<p>일본에서 가장 큰 호수인 <a href="/wiki/%EB%B9%84%EC%99%80_%ED%98%B8" title="비와 호">비와 호</a>는 시가 현의 중앙에 위치한다. 호수는 시가 현 총면적의 6분의 1을 차지한다. 현은 서쪽으로 <a class="new" href="/w/index.php?title=%ED%9E%88%EB%9D%BC_%EC%82%B0%EC%A7%80&amp;action=edit&amp;redlink=1" title="히라 산지 (없는 문서)">히라 산지</a>, 동쪽으로 <a class="new" href="/w/index.php?title=%EC%9D%B4%EB%B6%80%ED%82%A4_%EC%82%B0&amp;action=edit&amp;redlink=1" title="이부키 산 (없는 문서)">이부키 산</a>, 남동쪽으로 <a class="new" href="/w/index.php?title=%EC%8A%A4%EC%A6%88%EC%B9%B4_%EC%82%B0%EB%A7%A5&amp;action=edit&amp;redlink=1" title="스즈카 산맥 (없는 문서)">스즈카 산맥</a>으로 둘러싸여 있다. 시가 현 북쪽은 남쪽에 비해 대체적으로 춥고 눈이 많이 온다.</p>
<p><a class="mw-redirect" href="/wiki/%EC%9A%94%EB%8F%84%EA%B0%80%EC%99%80_%EA%B0%95" title="요도가와 강">요도가와 강</a>이 비와 호에서 발원해 <a href="/wiki/%EA%B5%90%ED%86%A0_%EC%8B%9C" title="교토 시">교토 시</a>를 거쳐 <a href="/wiki/%EC%98%A4%EC%82%AC%EC%B9%B4_%EB%A7%8C" title="오사카 만">오사카 만</a>으로 흘러든다. 이것은 호수의 밖으로 흐르는 유일한 강이다.</p>

<h2><span class="mw-headline" id=".EA.B2.BD.EC.A0.9C">경제</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%8B%9C%EA%B0%80_%ED%98%84&amp;action=edit&amp;section=6" title="부분 편집: 경제">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>중세 시대 이래로 많은 시가인들은 일본의 상업 부문에 활발히 진출했고 <b>오미 상인</b>(近江商人)으로 불렸다. 예를 들어 <a class="new" href="/w/index.php?title=%EC%9D%BC%EB%B3%B8%EC%83%9D%EB%AA%85&amp;action=edit&amp;redlink=1" title="일본생명 (없는 문서)">일본생명</a>, <a class="new" href="/w/index.php?title=%EC%9D%B4%ED%86%A0%EC%B6%94&amp;action=edit&amp;redlink=1" title="이토추 (없는 문서)">이토추</a>, <a class="new" href="/w/index.php?title=%EB%A7%88%EB%A3%A8%EB%B2%A0%EB%8B%88&amp;action=edit&amp;redlink=1" title="마루베니 (없는 문서)">마루베니</a>, <a href="/wiki/%EB%8B%A4%EC%B9%B4%EC%8B%9C%EB%A7%88%EC%95%BC" title="다카시마야">다카시마야</a>, <a href="/wiki/%EC%99%80%EC%BD%94%EB%A3%A8" title="와코루">와코루</a>, <a href="/wiki/%EC%96%80%EB%A7%88" title="얀마">얀마</a>는 시가인들에 의해 세워졌다. 오미 상인들은 종종 시기하는 다른 상인들로부터 <b>오미 도둑</b>(近江泥棒)으로 불리기도 했다.</p>
<p>현재 시가 현의 주요 산업은 공업이다. IBM 재팬, 캐논, 얀마 디젤, <a class="mw-disambig" href="/wiki/%EB%AF%B8%EC%93%B0%EB%B9%84%EC%8B%9C" title="미쓰비시">미쓰비시</a>, <a class="new" href="/w/index.php?title=%EB%8F%84%EB%A0%88&amp;action=edit&amp;redlink=1" title="도레 (없는 문서)">도레</a>와 같은 수많은 주요 회사들이 시가 현에 공장을 두고 있다. 2006년 <a href="/wiki/%EC%9D%BC%EB%B3%B8_%EB%82%B4%EA%B0%81%EB%B6%80" title="일본 내각부">일본 내각부</a>의 통계에 따르면 2차 산업은 시가 현 총 생산의 46.7%를 차지해 일본에서 그 비율이 가장 높았다.</p>
<h2><span class="mw-headline" id=".EC.9D.B8.EA.B5.AC">인구</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%8B%9C%EA%B0%80_%ED%98%84&amp;action=edit&amp;section=7" title="부분 편집: 인구">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>인구는 <a href="/wiki/%EB%B9%84%EC%99%80_%ED%98%B8" title="비와 호">비와 호</a>의 남안의 <a href="/wiki/%EC%98%A4%EC%93%B0_%EC%8B%9C" title="오쓰 시">오쓰 시</a>와 호수 동안에 집중 되어 있다. <a href="/wiki/%EA%B5%AC%EC%82%AC%EC%93%B0_%EC%8B%9C" title="구사쓰 시">구사쓰 시</a>, <a href="/wiki/%EB%AA%A8%EB%A6%AC%EC%95%BC%EB%A7%88_%EC%8B%9C" title="모리야마 시">모리야마 시</a>와 같은 동안의 도시들은 <a href="/wiki/%EA%B5%90%ED%86%A0_%EC%8B%9C" title="교토 시">교토 시</a>의 통근권 내에 있다. 최근에 많은 일본계 브라질인들이 주변의 공장들에서 일하기위해 시가 현에 정착했다. 호수의 서안은 더 전원적이고 백사장이 있는 휴양지이다.</p>
