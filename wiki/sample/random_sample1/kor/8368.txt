<p><b>시니샤 미하일로비치</b>(<span style="font-size: smaller;"><a href="/wiki/%EC%84%B8%EB%A5%B4%EB%B9%84%EC%95%84%EC%96%B4" title="세르비아어">세르비아어</a>:</span> <span lang="sr" xml:lang="sr">Синиша Михајловић</span>, <a href="/wiki/1969%EB%85%84" title="1969년">1969년</a> <a href="/wiki/2%EC%9B%94_20%EC%9D%BC" title="2월 20일">2월 20일</a>, <a class="mw-redirect" href="/wiki/%EC%9C%A0%EA%B3%A0%EC%8A%AC%EB%9D%BC%EB%B9%84%EC%95%84_%EC%82%AC%ED%9A%8C%EC%A3%BC%EC%9D%98_%EC%97%B0%EB%B0%A9_%EA%B3%B5%ED%99%94%EA%B5%AD" title="유고슬라비아 사회주의 연방 공화국">유고슬라비아</a> <a href="/wiki/%ED%81%AC%EB%A1%9C%EC%95%84%ED%8B%B0%EC%95%84" title="크로아티아">크로아티아</a> <a href="/wiki/%EB%B6%80%EC%BD%94%EB%B0%94%EB%A5%B4" title="부코바르">부코바르</a> ~ )는 과거 <a href="/wiki/%EC%84%B8%EB%A5%B4%EB%B9%84%EC%95%84" title="세르비아">세르비아</a>의 <a href="/wiki/%EC%B6%95%EA%B5%AC" title="축구">축구</a> 선수이자 현 지도자이다.</p>
<p>그는 <a href="/wiki/%EB%B6%80%EC%BD%94%EB%B0%94%EB%A5%B4" title="부코바르">부코바르</a>에서 태어나, 보로보 나세례 자치구에서 성장했다.</p>
<p>미하일로비치는 세르비아와 <a href="/wiki/%EC%9D%B4%ED%83%88%EB%A6%AC%EC%95%84" title="이탈리아">이탈리아</a>에서 선수로 활약했으며. <a href="/wiki/2004%EB%85%84" title="2004년">2004년</a> 여름, 그는 라치오를 떠나 그의 친구 <a href="/wiki/%EB%A1%9C%EB%B2%A0%EB%A5%B4%ED%86%A0_%EB%A7%8C%EC%B9%98%EB%8B%88" title="로베르토 만치니">로베르토 만치니</a>가 감독으로 있는 <a href="/wiki/FC_%EC%9D%B8%ED%85%8C%EB%A5%B4%EB%82%98%EC%B9%98%EC%98%A4%EB%82%A0%EB%A0%88_%EB%B0%80%EB%9D%BC%EB%85%B8" title="FC 인테르나치오날레 밀라노">FC 인테르나치오날레 밀라노</a>로 이적하였다.</p>
<p>선수 은퇴 후에는 인테르의 수석 코치로 2년간 있다가, <a href="/wiki/2008%EB%85%84" title="2008년">2008년</a>부터 <a href="/wiki/2009%EB%85%84" title="2009년">2009년</a>까지는 <a class="mw-redirect" href="/wiki/%EB%B3%BC%EB%A1%9C%EB%83%90_FC" title="볼로냐 FC">볼로냐 FC</a>의 감독을 맡았으며, <a href="/wiki/2009%EB%85%84" title="2009년">2009년</a> <a href="/wiki/12%EC%9B%94" title="12월">12월</a>부터 <a href="/wiki/%EC%B9%BC%EC%B4%88_%EC%B9%B4%ED%83%80%EB%8B%88%EC%95%84" title="칼초 카타니아">칼초 카타니아</a>의 감독을 맡았다.</p>
<p><a href="/wiki/2010%EB%85%84" title="2010년">2010년</a> <a href="/wiki/6%EC%9B%94_4%EC%9D%BC" title="6월 4일">6월 4일</a>부터 <a href="/wiki/ACF_%ED%94%BC%EC%98%A4%EB%A0%8C%ED%8B%B0%EB%82%98" title="ACF 피오렌티나">ACF 피오렌티나</a>의 감독이 되어 1시즌간 활동하였다.</p>
<p>그리고 2년간 쉬었다가 2012년 조국 세르비아의 감독이 되었다.</p>
<p>2013년부터 2015년까지 <a href="/wiki/UC_%EC%82%BC%ED%94%84%EB%8F%84%EB%A6%AC%EC%95%84" title="UC 삼프도리아">UC 삼프도리아</a>의 감독이었다.</p>
