<p><b>시드니 풋볼 스타디움</b>(<span lang="en" xml:lang="en">Sydney Football Stadium</span> 줄여서 SFS)은 2002년부터 2007년까지는 오지 스타디움으로 알려졌으며<sup class="reference" id="cite_ref-1"><a href="#cite_note-1">[1]</a></sup> <a href="/wiki/%EC%98%A4%EC%8A%A4%ED%8A%B8%EB%A0%88%EC%9D%BC%EB%A6%AC%EC%95%84" title="오스트레일리아">오스트레일리아</a> <a href="/wiki/%EB%89%B4%EC%82%AC%EC%9A%B0%EC%8A%A4%EC%9B%A8%EC%9D%BC%EC%8A%A4_%EC%A3%BC" title="뉴사우스웨일스 주">뉴사우스웨일스 주</a> <a href="/wiki/%EC%8B%9C%EB%93%9C%EB%8B%88" title="시드니">시드니</a>의 <a class="new" href="/w/index.php?title=%EB%AC%B4%EC%96%B4_%ED%8C%8C%ED%81%AC&amp;action=edit&amp;redlink=1" title="무어 파크 (없는 문서)">무어 파크</a>에 위치한 경기장이다.</p>
<p><a href="/wiki/1998%EB%85%84" title="1998년">1998년</a> <a href="/wiki/%EB%9F%AD%EB%B9%84_%EB%A6%AC%EA%B7%B8" title="럭비 리그">럭비 리그</a> 경기를 치르기 위해 <a href="/wiki/%EC%A7%81%EC%82%AC%EA%B0%81%ED%98%95" title="직사각형">직사각형</a> 형태의 경기장이 건설되었으며 현재는 <a href="/wiki/%EC%98%A4%EC%8A%A4%ED%8A%B8%EB%A0%88%EC%9D%BC%EB%A6%AC%EC%95%84" title="오스트레일리아">오스트레일리아</a> 국내외의 <a href="/wiki/%EC%B6%95%EA%B5%AC" title="축구">축구</a>와 <a class="mw-redirect" href="/wiki/%EB%9F%AD%EB%B9%84_%EC%9C%A0%EB%8B%88%EC%98%A8" title="럭비 유니온">럭비 유니온</a> 경기도 이 곳에서 벌어진다.</p>
<p>이 경기장을 홈 경기장으로 사용하고 있는 팀은 다음과 같다.</p>
<ul>
<li>시드니 루스터스 - 오스트레일리아 <a href="/wiki/%EB%82%B4%EC%85%94%EB%84%90_%EB%9F%AD%EB%B9%84_%EB%A6%AC%EA%B7%B8" title="내셔널 럭비 리그">내셔널 럭비 리그</a> (NRL)</li>
<li>뉴사우스웨일스 와라타스 - <a class="new" href="/w/index.php?title=%EB%9F%AD%EB%B9%84_%EC%9C%A0%EB%8B%88%EC%98%A8_%EC%8A%88%ED%8D%BC_14&amp;action=edit&amp;redlink=1" title="럭비 유니온 슈퍼 14 (없는 문서)">럭비 유니온 슈퍼 14</a></li>
<li><a href="/wiki/%EC%8B%9C%EB%93%9C%EB%8B%88_FC" title="시드니 FC">시드니 FC</a> - 오스트레일리아 현대 <a class="mw-redirect" href="/wiki/A-%EB%A6%AC%EA%B7%B8" title="A-리그">A-리그</a></li>
</ul>
<p>시드니 풋볼 스타디움에서는 보통 <a href="/wiki/%EB%82%B4%EC%85%94%EB%84%90_%EB%9F%AD%EB%B9%84_%EB%A6%AC%EA%B7%B8" title="내셔널 럭비 리그">내셔널 럭비 리그</a> (NRL)의 준 결승전과 예선 라운드 결승전, 시즌이 시작하기 전 사우스 시드니와 세인트 조지 드래곤즈의 채리티 쉴드 경기가 열린다. 과거에는 럭비 리그의 그랜드 파이널 경기가 열렸었다.</p>
<p><a href="/wiki/1999%EB%85%84" title="1999년">1999년</a> <a href="/wiki/2000%EB%85%84_%ED%95%98%EA%B3%84_%EC%98%AC%EB%A6%BC%ED%94%BD" title="2000년 하계 올림픽">시드니 올림픽</a>을 개최하기 위해 건설된 <a class="mw-redirect" href="/wiki/%EC%8B%9C%EB%93%9C%EB%8B%88_%EC%98%AC%EB%A6%BC%ED%94%BD_%EA%B2%BD%EA%B8%B0%EC%9E%A5" title="시드니 올림픽 경기장">시드니 올림픽 경기장</a>이 완공되기 전까지 이 경기장에서는 스테이트 오브 오리진 경기나 국제 럭비 경기 등 <a href="/wiki/%EC%8B%9C%EB%93%9C%EB%8B%88" title="시드니">시드니</a>에서 열린 주요 메이저 대회가 벌어진 장소였다.</p>
<p>2012년 3월 호주 알리안츠와 시드니 스타디움의 운영 기관인 Sydney Cricket Ground Trust가 맺은 파트너십의 결과로 '알리안츠 스타디움'으로 명칭을 변경하였다.</p>

<h2><span class="mw-headline" id=".EC.97.AD.EC.82.AC">역사</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%8B%9C%EB%93%9C%EB%8B%88_%ED%92%8B%EB%B3%BC_%EC%8A%A4%ED%83%80%EB%94%94%EC%9B%80&amp;action=edit&amp;section=1" title="부분 편집: 역사">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>1988년 경기장이 만들어지기 전까지 주요 경기가 열렸던 <a href="/wiki/%EC%8B%9C%EB%93%9C%EB%8B%88_%ED%81%AC%EB%A6%AC%EC%BC%93_%EA%B7%B8%EB%9D%BC%EC%9A%B4%EB%93%9C" title="시드니 크리켓 그라운드">시드니 크리켓 그라운드</a>(SCG)는 <a href="/wiki/%ED%83%80%EC%9B%90" title="타원">타원형</a>의 경기장이어서 <a href="/wiki/%ED%81%AC%EB%A6%AC%EC%BC%93" title="크리켓">크리켓</a>과 <a class="mw-redirect" href="/wiki/%EC%98%A4%EC%A7%80_%ED%92%8B%EB%B3%BC" title="오지 풋볼">오지 풋볼</a> 이외의 다른 스포츠 경기가 열리기에 적합하지 못했다. 이러한 이유에서 <a href="/wiki/1986%EB%85%84" title="1986년">1986년</a> 시드니 크리켓 그라운드 트러스트 사(SCG Trust)에 의해 <a class="new" href="/w/index.php?title=%EB%AC%B4%EC%96%B4_%ED%8C%8C%ED%81%AC&amp;action=edit&amp;redlink=1" title="무어 파크 (없는 문서)">무어 파크</a> 지역 내, SCG 근처에 있던 과거 <a class="new" href="/w/index.php?title=%EC%8B%9C%EB%93%9C%EB%8B%88_%EC%8A%A4%ED%8F%AC%EC%B8%A0_%EA%B7%B8%EB%9D%BC%EC%9A%B4%EB%93%9C&amp;action=edit&amp;redlink=1" title="시드니 스포츠 그라운드 (없는 문서)">시드니 스포츠 그라운드</a> 자리에 시드니 풋볼 스타디움이 세워졌다.</p>
<p>개장 초기 이 경기장의 수용 규모는 41,159석으로 건축 되었으나 <a href="/wiki/2007%EB%85%84" title="2007년">2007년</a> 확장 공사를 통해 45,500석까지 그 규모를 늘렸고 <a href="/wiki/1993%EB%85%84" title="1993년">1993년</a> <a href="/wiki/%EC%98%A4%EC%8A%A4%ED%8A%B8%EB%A0%88%EC%9D%BC%EB%A6%AC%EC%95%84" title="오스트레일리아">오스트레일리아</a>와 <a href="/wiki/%EC%95%84%EB%A5%B4%ED%97%A8%ED%8B%B0%EB%82%98" title="아르헨티나">아르헨티나</a>의 <a href="/wiki/1994%EB%85%84_FIFA_%EC%9B%94%EB%93%9C%EC%BB%B5" title="1994년 FIFA 월드컵">미국 월드컵</a> 최종 예선 경기에서 43,967 명의 관중이 입장하여 이 경기장에 모인 가장 많은 관중 기록으로 남아있다.</p>
<p><a href="/wiki/2002%EB%85%84" title="2002년">2002년</a> 시드니 풋볼 스타디움은 오지 홈 론 사와의 5년(추후 협상을 통해 +5년 가능)간의 스폰서 계약을 통해 <b>오지 스타디움</b> (Aussie Stadium)으로 개명되었고 <a href="/wiki/2007%EB%85%84" title="2007년">2007년</a> <a href="/wiki/7%EC%9B%94_7%EC%9D%BC" title="7월 7일">7월 7일</a> 스폰서 계약 연장에 실패하며 원래의 명칭인 시드니 풋볼 스타디움이란 명칭을 다시 사용하게 되었다.</p>
<p><a href="/wiki/2003%EB%85%84" title="2003년">2003년</a> 이 경기장에서는 <a href="/wiki/%EB%9F%AD%EB%B9%84_%EC%9B%94%EB%93%9C%EC%BB%B5" title="럭비 월드컵">럭비 월드컵</a>; <a href="/wiki/%EC%95%84%EC%9D%BC%EB%9E%9C%EB%93%9C" title="아일랜드">아일랜드</a> 대 <a href="/wiki/%EB%82%98%EB%AF%B8%EB%B9%84%EC%95%84" title="나미비아">나미비아</a>, <a href="/wiki/%EC%95%84%EB%A5%B4%ED%97%A8%ED%8B%B0%EB%82%98" title="아르헨티나">아르헨티나</a> 대 <a href="/wiki/%EB%A3%A8%EB%A7%88%EB%8B%88%EC%95%84" title="루마니아">루마니아</a>, <a href="/wiki/%EC%8A%A4%EC%BD%94%ED%8B%80%EB%9E%9C%EB%93%9C" title="스코틀랜드">스코틀랜드</a> 대 <a href="/wiki/%ED%94%BC%EC%A7%80" title="피지">피지</a>, <a href="/wiki/%EB%82%A8%EC%95%84%ED%94%84%EB%A6%AC%EC%B9%B4_%EA%B3%B5%ED%99%94%EA%B5%AD" title="남아프리카 공화국">남아프리카 공화국</a> 대 <a class="mw-redirect" href="/wiki/%EA%B7%B8%EB%A3%A8%EC%A7%80%EC%95%BC" title="그루지야">그루지야</a>, <a href="/wiki/%EC%A1%B0%EC%A7%80%EC%95%84_(%EA%B5%AD%EA%B0%80)" title="조지아 (국가)">조지아</a> 대 <a href="/wiki/%EC%9A%B0%EB%A3%A8%EA%B3%BC%EC%9D%B4" title="우루과이">우루과이</a>의 경기가 열렸으며 <a href="/wiki/%EC%A1%B0%EC%A7%80%EC%95%84_(%EA%B5%AD%EA%B0%80)" title="조지아 (국가)">조지아</a>와 <a href="/wiki/%EC%9A%B0%EB%A3%A8%EA%B3%BC%EC%9D%B4" title="우루과이">우루과이</a>의 마지막 경기에서는 양팀의 저명성이 떨어지는 조건에도 불구하고 28,576 명의 관중이 입장해 경기를 관람하였다.</p>
<p>시드니 풋볼 스타디움에서는 <a href="/wiki/%EC%98%A4%EC%8A%A4%ED%8A%B8%EB%A0%88%EC%9D%BC%EB%A6%AC%EC%95%84" title="오스트레일리아">오스트레일리아</a> 국내의 주요 경기가 열렸는데 <a href="/wiki/%EC%BA%94%EB%B2%84%EB%9D%BC" title="캔버라">캔버라</a> 레이더스와 발메인 타이거즈의 <a href="/wiki/1989%EB%85%84" title="1989년">1989년</a> 내셔널 럭비 리그 그랜드 파이널 경기, 팬리스와 캔버라의 <a href="/wiki/1991%EB%85%84" title="1991년">1991년</a> 내셔널 럭비 리그 그랜드 파이널 경기, 뉴캐슬과 맨리의 <a href="/wiki/1997%EB%85%84" title="1997년">1997년</a> <a class="new" href="/w/index.php?title=%EC%98%A4%EC%8A%A4%ED%8A%B8%EB%A0%88%EC%9D%BC%EB%A6%AC%EC%95%88_%EB%9F%AD%EB%B9%84_%EB%A6%AC%EA%B7%B8&amp;action=edit&amp;redlink=1" title="오스트레일리안 럭비 리그 (없는 문서)">오스트레일리안 럭비 리그</a>(ARL) 그랜드 파이널 등의 경기가 열렸다.</p>
<p>2012년 3월 독일의 알리안츠 그룹과 파트너십을 통하여 '알리안츠 스타디움'으로 이름을 바꿨다. 이는 호주 알리안츠와 시드니 스타디움의 운영 기관인 Sydney Cricket Ground Trust가 맺은 파트너십의 결과다.</p>
<h2><span class="mw-headline" id=".EC.A3.BC.EC.9A.94_.EC.82.AC.EA.B1.B4">주요 사건</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%8B%9C%EB%93%9C%EB%8B%88_%ED%92%8B%EB%B3%BC_%EC%8A%A4%ED%83%80%EB%94%94%EC%9B%80&amp;action=edit&amp;section=2" title="부분 편집: 주요 사건">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<ul>
<li><a href="/wiki/1994%EB%85%84" title="1994년">1994년</a> <a class="new" href="/w/index.php?title=%EB%B8%94%EB%A0%88%EB%94%94%EC%86%94_%EC%BB%B5&amp;action=edit&amp;redlink=1" title="블레디솔 컵 (없는 문서)">블레디솔 컵</a> 럭비 대회에서 <a class="new" href="/w/index.php?title=%EC%A1%B0%EC%A7%80_%EA%B7%B8%EB%A0%88%EA%B1%B4&amp;action=edit&amp;redlink=1" title="조지 그레건 (없는 문서)">조지 그레건</a>의 태클로 <a class="mw-redirect" href="/wiki/%EC%9B%94%EB%9F%AC%EB%B9%84%EC%8A%A4" title="월러비스">오스트레일리아</a>가 <a class="mw-redirect" href="/wiki/%EC%98%AC%EB%B8%94%EB%9E%99%EC%8A%A4" title="올블랙스">뉴질랜드</a>에 승리하였다.</li>
<li><a href="/wiki/1993%EB%85%84" title="1993년">1993년</a> <a href="/wiki/%EC%95%84%EB%A5%B4%ED%97%A8%ED%8B%B0%EB%82%98_%EC%B6%95%EA%B5%AC_%EA%B5%AD%EA%B0%80%EB%8C%80%ED%91%9C%ED%8C%80" title="아르헨티나 축구 국가대표팀">아르헨티나</a> 대 <a href="/wiki/%EC%98%A4%EC%8A%A4%ED%8A%B8%EB%A0%88%EC%9D%BC%EB%A6%AC%EC%95%84_%EC%B6%95%EA%B5%AC_%EA%B5%AD%EA%B0%80%EB%8C%80%ED%91%9C%ED%8C%80" title="오스트레일리아 축구 국가대표팀">오스트레일리아</a>의 <a href="/wiki/1994%EB%85%84_FIFA_%EC%9B%94%EB%93%9C%EC%BB%B5" title="1994년 FIFA 월드컵">미국 월드컵</a> 최종 예선에서 <a href="/wiki/%EB%94%94%EC%97%90%EA%B3%A0_%EB%A7%88%EB%9D%BC%EB%8F%84%EB%82%98" title="디에고 마라도나">디에고 마라도나</a>가 선수로 뛰었다.</li>
<li><a href="/wiki/2006%EB%85%84" title="2006년">2006년</a> <a href="/wiki/5%EC%9B%94_17%EC%9D%BC" title="5월 17일">5월 17일</a> <a href="/wiki/%EB%8C%80%EB%8B%88_%EA%B7%B8%EB%A6%B0" title="대니 그린">대니 그린</a>과 <a class="new" href="/w/index.php?title=%EC%95%A4%EC%84%9C%EB%8B%88_%EB%A8%BC%EB%8B%A4%EC%9D%B8&amp;action=edit&amp;redlink=1" title="앤서니 먼다인 (없는 문서)">앤서니 먼다인</a>의 <a class="mw-redirect" href="/wiki/%EB%B3%B5%EC%8B%B1" title="복싱">복싱</a> 경기가 열렸다.</li>
<li><a href="/wiki/2006%EB%85%84" title="2006년">2006년</a> <a href="/wiki/11%EC%9B%94_25%EC%9D%BC" title="11월 25일">11월 25일</a> 럭비 리그 트라이 네이션스 대회의 그랜드 파이널 경기가 열렸다.</li>
<li><a href="/wiki/2007%EB%85%84" title="2007년">2007년</a> 초, 2번째 전광판을 설치하고 경기장 규모를 45,500석으로 확장하였다.</li>
<li><a href="/wiki/2007%EB%85%84" title="2007년">2007년</a> <a href="/wiki/7%EC%9B%94_7%EC%9D%BC" title="7월 7일">7월 7일</a> 환경 콘서트인 <a href="/wiki/%EB%9D%BC%EC%9D%B4%EB%B8%8C_%EC%96%B4%EC%8A%A4" title="라이브 어스">라이브 어스 콘서트</a>가 열렸다.</li>
<li><a href="/wiki/2012%EB%85%84" title="2012년">2012년</a> <a href="/wiki/3%EC%9B%94" title="3월">3월</a> 알리안츠그룹과의 파트너십에 의해 <a class="new" href="/w/index.php?title=%EC%95%8C%EB%A6%AC%EC%95%88%EC%B8%A0_%EC%8A%A4%ED%83%80%EB%94%94%EC%9B%80&amp;action=edit&amp;redlink=1" title="알리안츠 스타디움 (없는 문서)">알리안츠 스타디움</a>으로 명칭 변경하였다.</li>
</ul>
