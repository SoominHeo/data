<p><b>시라쿠사</b>(<span style="font-size: smaller;"><a href="/wiki/%EC%9D%B4%ED%83%88%EB%A6%AC%EC%95%84%EC%96%B4" title="이탈리아어">이탈리아어</a>:</span> <span lang="it" xml:lang="it">Siracusa</span> <small>시라쿠자<sup>[<a href="/wiki/%EC%9C%84%ED%82%A4%EB%B0%B1%EA%B3%BC:%EC%9D%B4%ED%83%88%EB%A6%AC%EC%95%84%EC%96%B4%EC%9D%98_%ED%95%9C%EA%B8%80_%ED%91%9C%EA%B8%B0" title="위키백과:이탈리아어의 한글 표기">*</a>]</sup></small>, <span style="font-size: smaller;"><a href="/wiki/%EA%B7%B8%EB%A6%AC%EC%8A%A4%EC%96%B4" title="그리스어">그리스어</a>:</span> <span lang="el" xml:lang="el">Συρακούσαι <small>또는</small> Συρακούσα</span> <small>시라쿠사이(<a href="/wiki/%EA%B3%A0%EB%8C%80_%EA%B7%B8%EB%A6%AC%EC%8A%A4%EC%96%B4" title="고대 그리스어">고대</a>), 시라쿠사<sup>[<a href="/wiki/%EC%9C%84%ED%82%A4%EB%B0%B1%EA%B3%BC:%EC%99%B8%EB%9E%98%EC%96%B4_%ED%91%9C%EA%B8%B0%EB%B2%95/%EA%B7%B8%EB%A6%AC%EC%8A%A4%EC%96%B4_%EC%8B%9C%EC%95%88" title="위키백과:외래어 표기법/그리스어 시안">*</a>]</sup></small>, <span style="font-size: smaller;"><a href="/wiki/%EB%9D%BC%ED%8B%B4%EC%96%B4" title="라틴어">라틴어</a>:</span> <span lang="la" xml:lang="la">Syracusa</span> <small>시라쿠사<sup>[<a href="/wiki/%EC%9C%84%ED%82%A4%EB%B0%B1%EA%B3%BC:%EC%99%B8%EB%9E%98%EC%96%B4_%ED%91%9C%EA%B8%B0%EB%B2%95/%EB%9D%BC%ED%8B%B4%EC%96%B4_%ED%91%9C%EA%B8%B0_%EC%9B%90%EC%B9%99" title="위키백과:외래어 표기법/라틴어 표기 원칙">*</a>]</sup></small>)는 <a href="/wiki/%EC%9D%B4%ED%83%88%EB%A6%AC%EC%95%84" title="이탈리아">이탈리아</a> <a href="/wiki/%EC%8B%9C%EC%B9%A0%EB%A6%AC%EC%95%84" title="시칠리아">시칠리아 섬</a>의 남동쪽 해안에 있는 도시이다. 인구는 123,408명 (2011년)이며 <a class="new" href="/w/index.php?title=%EC%8B%9C%EB%9D%BC%EC%BF%A0%EC%82%AC_%EC%A3%BC&amp;action=edit&amp;redlink=1" title="시라쿠사 주 (없는 문서)">시라쿠사 주</a>의 주도이다. <a href="/wiki/%EB%A7%88%EB%A5%B4%EC%BF%A0%EC%8A%A4_%ED%88%B4%EB%A6%AC%EC%9A%B0%EC%8A%A4_%ED%82%A4%EC%BC%80%EB%A1%9C" title="마르쿠스 툴리우스 키케로">키케로</a>가 “가장 위대한 그리스 도시이며 가장 아름답다”라고 묘사한 바 있는 시라쿠사의 구시가는 <a class="mw-redirect" href="/wiki/%EA%B5%AD%EC%A0%9C_%EC%97%B0%ED%95%A9_%EA%B5%90%EC%9C%A1_%EA%B3%BC%ED%95%99_%EB%AC%B8%ED%99%94_%EA%B8%B0%EA%B5%AC" title="국제 연합 교육 과학 문화 기구">유네스코</a> 지정 <a href="/wiki/%EC%84%B8%EA%B3%84%EC%9C%A0%EC%82%B0" title="세계유산">세계유산</a>이다.</p>
<p><a href="/wiki/%EA%B7%B8%EB%A6%AC%EC%8A%A4" title="그리스">그리스</a> 시대에 건설된 옛 도시로, 구시가는 <a href="/wiki/%EC%8B%9C%EC%B9%A0%EB%A6%AC%EC%95%84" title="시칠리아">시칠리아 섬</a>과 좁은 해협으로 격리되어 있는 작은 섬인 <a class="new" href="/w/index.php?title=%EC%98%A4%EB%A5%B4%ED%8B%B0%EC%9E%90_%EC%84%AC&amp;action=edit&amp;redlink=1" title="오르티자 섬 (없는 문서)">오르티자 섬</a>(<span style="font-size: smaller;"><a href="/wiki/%EC%9D%B4%ED%83%88%EB%A6%AC%EC%95%84%EC%96%B4" title="이탈리아어">이탈리아어</a>:</span> <span lang="it" xml:lang="it">Ortigia</span>)에 있으나, 신시가는 시칠리아 섬까지 펼쳐져 있다. <a href="/wiki/%EB%B9%84%EB%88%84" title="비누">비누</a>·<a href="/wiki/%EC%8B%9C%EB%A9%98%ED%8A%B8" title="시멘트">시멘트</a>·<a href="/wiki/%EC%98%AC%EB%A6%AC%EB%B8%8C%EA%B8%B0%EB%A6%84" title="올리브기름">올리브유</a>·<a href="/wiki/%ED%8F%AC%EB%8F%84%EC%A3%BC" title="포도주">포도주</a>의 제조 및 어업의 중심지로 제염업이 성하다. <a href="/wiki/%EC%95%84%ED%8F%B4%EB%A1%A0" title="아폴론">아폴로</a> 신전, 그리스식 극장 등 <a href="/wiki/%EA%B3%A0%EB%8C%80_%EA%B7%B8%EB%A6%AC%EC%8A%A4" title="고대 그리스">고대 그리스</a>·<a href="/wiki/%EA%B3%A0%EB%8C%80_%EB%A1%9C%EB%A7%88" title="고대 로마">로마</a> 시대의 유적이 많으며 <a href="/wiki/%EA%B4%80%EA%B4%91" title="관광">관광</a>지로서도 유명하다.</p>
<h2><span class="mw-headline" id=".EC.97.AD.EC.82.AC">역사</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%8B%9C%EB%9D%BC%EC%BF%A0%EC%82%AC_(%EC%8B%9C%EC%B9%A0%EB%A6%AC%EC%95%84)&amp;action=edit&amp;section=1" title="부분 편집: 역사">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>오늘날의 시라쿠사에 해당하는 지역은 매우 오래전부터 사람들이 정착하였다. 이 지역의 마을들은 <a href="/wiki/%EB%AF%B8%EC%BC%80%EB%84%A4" title="미케네">미케네</a> 시대 그리스와도 이미 관계를 가졌다. <a href="/wiki/%EA%B8%B0%EC%9B%90%EC%A0%84_8%EC%84%B8%EA%B8%B0" title="기원전 8세기">기원전 734년</a> 혹은 <a href="/wiki/%EA%B8%B0%EC%9B%90%EC%A0%84_8%EC%84%B8%EA%B8%B0" title="기원전 8세기">733년</a> 아르키아스가 이끄는 <a href="/wiki/%EC%BD%94%EB%A6%B0%ED%86%A0%EC%8A%A4" title="코린토스">코린토스</a> 출신 <a href="/wiki/%EA%B7%B8%EB%A6%AC%EC%8A%A4%EC%9D%B8" title="그리스인">그리스인</a>들이 오르티자 섬에 시칠리아섬 최초의 그리스 식민시 시라쿠사이(<span style="font-size: smaller;"><a href="/wiki/%EA%B3%A0%EB%8C%80_%EA%B7%B8%EB%A6%AC%EC%8A%A4%EC%96%B4" title="고대 그리스어">고대 그리스어</a>:</span> <span lang="grc" xml:lang="grc">Συρακούσαι</span>)를 건설하여, 선주민을 <a class="mw-redirect" href="/wiki/%EB%86%8D%EB%85%B8" title="농노">농노</a>(農奴)로 삼은 뒤 지주가 귀족이 되어 정권을 장악하였다. 그들은 오래지 않아 농업과 무역으로 번영을 이루었으며, <a href="/wiki/%EA%B8%B0%EC%9B%90%EC%A0%84_480%EB%85%84" title="기원전 480년">기원전 480년</a> <a href="/wiki/%EC%B0%B8%EC%A3%BC" title="참주">참주</a>(僭主) <a class="new" href="/w/index.php?title=%EA%B2%94%EB%A1%A0&amp;action=edit&amp;redlink=1" title="겔론 (없는 문서)">겔론</a>이 <a href="/wiki/%ED%9E%88%EB%A9%94%EB%9D%BC" title="히메라">히메라</a> 전투에서 <a href="/wiki/%EC%B9%B4%EB%A5%B4%ED%83%80%EA%B3%A0" title="카르타고">카르타고</a> 군의 침입을 격퇴한 이후 서쪽의 그리스 세계에서 주도권을 굳히고, 그 세력을 신장하였다. <a href="/wiki/%EA%B8%B0%EC%9B%90%EC%A0%84_3%EC%84%B8%EA%B8%B0" title="기원전 3세기">기원전 3세기</a> 후반부터는 친(親)로마파와 친카르타고파의 분쟁이 일어나자 이에 편승하여 로마가 간섭하게 되어 <a href="/wiki/%EA%B8%B0%EC%9B%90%EC%A0%84_211%EB%85%84" title="기원전 211년">기원전 211년</a> <a href="/wiki/%EC%95%84%EB%A5%B4%ED%82%A4%EB%A9%94%EB%8D%B0%EC%8A%A4" title="아르키메데스">아르키메데스</a> 등의 방위전(防衛戰)도 보람 없이 <a href="/wiki/%EB%A1%9C%EB%A7%88_%EA%B3%B5%ED%99%94%EC%A0%95" title="로마 공화정">로마 공화정</a>에게 굴복하였으나, 이후 로마의 시칠리아 섬 통치관의 주재지로서 번창하였다.</p>
<p><a href="/wiki/286%EB%85%84" title="286년">286년</a> 무렵에 <a href="/wiki/%ED%94%84%EB%9E%91%ED%81%AC_%EC%99%95%EA%B5%AD" title="프랑크 왕국">프랑크</a> 해적의 침입으로 황폐해졌으나 이른바 민족 대이동의 물결에는 휩쓸리지 않았고 5세기에는 <a href="/wiki/%EC%95%84%ED%94%84%EB%A6%AC%EC%B9%B4" title="아프리카">아프리카</a>로부터 <a href="/wiki/%EB%B0%98%EB%8B%AC%EC%A1%B1" title="반달족">반달족</a>의 침략을 받았다. <a href="/wiki/6%EC%84%B8%EA%B8%B0" title="6세기">6세기</a>에는 <a href="/wiki/%EA%B7%B8%EB%A6%AC%EC%8A%A4%EC%9D%B8" title="그리스인">그리스인</a>도 이주했으며, <a href="/wiki/7%EC%84%B8%EA%B8%B0" title="7세기">7세기</a>에는 한때 <a href="/wiki/%EB%B9%84%EC%9E%94%ED%8B%B0%EC%9B%80_%EC%A0%9C%EA%B5%AD" title="비잔티움 제국">비잔티움 제국</a>에 소속되었다. 이 무렵부터 <a href="/wiki/%EC%82%AC%EB%9D%BC%EC%84%BC" title="사라센">사라센</a> 세력이 아프리카로 뻗어 시라쿠사는 기독교 세계의 방위선의 일환을 이루었으나 <a href="/wiki/878%EB%85%84" title="878년">878년</a> 결국 <a href="/wiki/%EC%82%AC%EB%9D%BC%EC%84%BC" title="사라센">사라센</a>의 지배에 들어갔으며 다시 11세기에는 <a class="mw-redirect" href="/wiki/%EB%85%B8%EB%A5%B4%EB%A7%8C%EC%A1%B1" title="노르만족">노르만족</a>의 침입을 받아 비잔티움·사라센·노르만의 문화가 혼재하기에 이르렀다. 그 후 <a class="mw-redirect" href="/wiki/%ED%98%B8%EC%97%94%EC%8A%88%ED%83%80%EC%9A%B0%ED%8E%9C_%EC%99%95%EA%B0%80" title="호엔슈타우펜 왕가">호엔슈타우펜 왕가</a>와 <a class="new" href="/w/index.php?title=%EC%95%84%EB%9D%BC%EA%B3%A4_%EC%99%95%EA%B0%80&amp;action=edit&amp;redlink=1" title="아라곤 왕가 (없는 문서)">아라곤 왕가</a> 등의 지배를 받아오다가 <a href="/wiki/19%EC%84%B8%EA%B8%B0" title="19세기">19세기</a> 후반 <a href="/wiki/%EC%9D%B4%ED%83%88%EB%A6%AC%EC%95%84_%EC%99%95%EA%B5%AD_(1861%EB%85%84_~_1946%EB%85%84)" title="이탈리아 왕국 (1861년 ~ 1946년)">이탈리아 왕국</a>에 통합되었다. <a href="/wiki/%EC%A0%9C2%EC%B0%A8_%EC%84%B8%EA%B3%84_%EB%8C%80%EC%A0%84" title="제2차 세계 대전">제2차 세계 대전</a> 중에는 연합군의 폭격을 받았으며, <a href="/wiki/1943%EB%85%84" title="1943년">1943년</a>에는 <a href="/wiki/%EB%AF%B8%EA%B5%AD" title="미국">미국</a>·<a href="/wiki/%EC%98%81%EA%B5%AD" title="영국">영국</a>군에게 점령되었다.</p>
