<h2><span class="mw-headline" id=".EA.B0.9C.EC.9A.94">개요</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%8B%9C%EB%A1%9C%ED%83%80_%EC%9C%A0&amp;action=edit&amp;section=1" title="부분 편집: 개요">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>노래부르는 것을 좋아했던 유년기 때부터 자연스럽게 연예계에 관심을 가지게 되어 "텔레비전에 나오는 사람이 될래"라고 생각했었으나, 신장 및 외국인 같은 외모 때문에 역할에 제약을 많이 받자 회의를 느꼈다.<sup class="reference" id="cite_ref-2"><a href="#cite_note-2">[2]</a></sup></p>
<p><a href="/wiki/2003%EB%85%84" title="2003년">2003년</a>, <a href="/wiki/%EB%AE%A4%EC%A7%80%EC%BB%AC" title="뮤지컬">뮤지컬</a> 《<a href="/wiki/%EB%8B%AC%EC%9D%98_%EC%9A%94%EC%A0%95_%EC%84%B8%EC%9D%BC%EB%9F%AC%EB%AC%B8#.EB.AE.A4.EC.A7.80.EC.BB.AC" title="달의 요정 세일러문">세일러문</a>》의 '<a href="/wiki/%EC%B9%98%EB%B0%94_%EB%A7%88%EB%AA%A8%EB%A3%A8" title="치바 마모루">치바 마모루/턱시도 가면</a>'역으로 데뷔. 《<a href="/wiki/%ED%85%8C%EB%8B%88%EC%8A%A4%EC%9D%98_%EC%99%95%EC%9E%90_(%EB%AE%A4%EC%A7%80%EC%BB%AC)" title="테니스의 왕자 (뮤지컬)">뮤지컬 테니스의 왕자</a>》(2005년)나, 《<a class="new" href="/w/index.php?title=%EC%8A%A4%EC%9C%84%EB%8B%88%ED%86%A0%EB%93%9C&amp;action=edit&amp;redlink=1" title="스위니토드 (없는 문서)">스위니토드</a>》(2007년) 등으로 무대 경험을 쌓았다.</p>
<p><a href="/wiki/2004%EB%85%84" title="2004년">2004년</a> <a href="/wiki/10%EC%9B%94" title="10월">10월</a>, <a class="mw-redirect" href="/wiki/D-BOYS" title="D-BOYS">D-BOYS</a> 가입.</p>
<p><a href="/wiki/2006%EB%85%84" title="2006년">2006년</a>, 《<a class="new" href="/w/index.php?title=%EC%A4%80_%ED%94%84%EB%9D%BC%EC%9D%B4%EB%93%9C&amp;action=edit&amp;redlink=1" title="준 프라이드 (없는 문서)">준 프라이드</a>》로 첫 영화 주연을 맡았으며, 영화 외에도 《<a class="mw-redirect" href="/wiki/%ED%8C%8C%EA%B2%AC%EC%9D%98_%ED%92%88%EA%B2%A9" title="파견의 품격">파견의 품격</a>》, 《<a href="/wiki/%EC%95%84%EB%A6%84%EB%8B%A4%EC%9A%B4_%EA%B7%B8%EB%8C%80%EC%97%90%EA%B2%8C~%EB%AF%B8%EB%82%A8%E2%99%82%ED%8C%8C%EB%9D%BC%EB%8B%A4%EC%9D%B4%EC%8A%A4~" title="아름다운 그대에게~미남♂파라다이스~">아름다운 그대에게~미남♂파라다이스~</a>》(이상 2007년), 《<a class="new" href="/w/index.php?title=%EA%B5%90%EC%84%AD%EC%9D%B8&amp;action=edit&amp;redlink=1" title="교섭인 (없는 문서)">교섭인~THE NEGOTIATOR~</a>》(2008년) 등의 드라마에 출연하며 활발하게 활동했다.</p>
<p><a href="/wiki/2008%EB%85%84" title="2008년">2008년</a> <a href="/wiki/7%EC%9B%94" title="7월">7월</a>, 《D-BOYS STAGE vol.2 라스트게임 ~최후의 조경전~》을 끝으로 D-BOYS 졸업.</p>
<p><a href="/wiki/2009%EB%85%84" title="2009년">2009년</a>, <a class="mw-redirect" href="/wiki/%EB%8C%80%ED%95%98%EB%93%9C%EB%9D%BC%EB%A7%88" title="대하드라마">대하드라마</a> 《<a href="/wiki/%EC%B2%9C%EC%A7%80%EC%9D%B8_(%EB%93%9C%EB%9D%BC%EB%A7%88)" title="천지인 (드라마)">천지인</a>》(<a class="mw-redirect" href="/wiki/%EC%9D%BC%EB%B3%B8_%EB%B0%A9%EC%86%A1_%ED%98%91%ED%9A%8C" title="일본 방송 협회">NHK</a>)의 <a href="/wiki/%EC%82%AC%EB%82%98%EB%8B%A4_%EB%85%B8%EB%B6%80%EC%8B%9C%EA%B2%8C" title="사나다 노부시게">사나다 유키무라</a> 역을 맡으며 처음으로 시대극에 출연. 이 역할을 계기로 《<a href="/wiki/NHK_%ED%99%8D%EB%B0%B1%EA%B0%80%ED%95%A9%EC%A0%84" title="NHK 홍백가합전">제60회 NHK 홍백가합전</a>》의 게스트 심사원으로 발탁되었다.</p>
<h2><span class="mw-headline" id=".EC.9D.B8.EB.AC.BC">인물</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%8B%9C%EB%A1%9C%ED%83%80_%EC%9C%A0&amp;action=edit&amp;section=2" title="부분 편집: 인물">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>아버지는 <a href="/wiki/%EC%9D%BC%EB%B3%B8%EC%9D%B8" title="일본인">일본인</a>으로 <a class="mw-redirect" href="/wiki/%EB%8B%88%ED%98%BC_TV_%EB%B0%A9%EC%86%A1%EB%A7%9D" title="니혼 TV 방송망">NTV</a>에서 구성을 담당하는 방송작가 시로타 미쓰오이며,<sup class="reference" id="cite_ref-3"><a href="#cite_note-3">[3]</a></sup> 어머니는 <a href="/wiki/%EC%8A%A4%ED%8E%98%EC%9D%B8" title="스페인">스페인</a>인이다. 형제는 형 2명, 누나 1명, 여동생 1명인 5형제이며, 시로타는 넷째이다. 둘째 형은 <a class="mw-disambig" href="/wiki/%EB%AA%A8%EB%8D%B8" title="모델">모델</a>, <a href="/wiki/%EB%94%94%EC%9E%90%EC%9D%B4%EB%84%88" title="디자이너">디자이너</a>, <a href="/wiki/%EA%B0%80%EC%88%98" title="가수">가수</a>로 활동하고 있는 시로타 준. 형이랑 얼굴이 똑같이 생겼다.</p>
<p>취미는 <a href="/wiki/%EC%9E%91%EC%82%AC" title="작사">작사</a>, <a href="/wiki/%EC%9E%91%EA%B3%A1" title="작곡">작곡</a>.<sup class="reference" id="cite_ref-talentmeikan_4-0"><a href="#cite_note-talentmeikan-4">[4]</a></sup></p>
<p>특기는 <a href="/wiki/%EC%8A%A4%ED%8E%98%EC%9D%B8%EC%96%B4" title="스페인어">스페인어</a>와 <a href="/wiki/%ED%94%BC%EC%95%84%EB%85%B8" title="피아노">피아노</a>, <a href="/wiki/%EC%83%89%EC%86%8C%ED%8F%B0" title="색소폰">색소폰</a>, <a href="/wiki/%EA%B8%B0%ED%83%80" title="기타">기타</a> 등 악기 연주.<sup class="reference" id="cite_ref-talentmeikan_4-1"><a href="#cite_note-talentmeikan-4">[4]</a></sup></p>
<p>좋아하는 운동은 <a href="/wiki/%EC%B6%95%EA%B5%AC" title="축구">축구</a>, <a href="/wiki/%EC%88%98%EC%98%81" title="수영">수영</a>, <a href="/wiki/%EB%B3%BC%EB%A7%81" title="볼링">볼링</a>.<sup class="reference" id="cite_ref-5"><a href="#cite_note-5">[5]</a></sup></p>
<p><a href="/wiki/%EC%95%BC%EB%A7%88%EC%8B%9C%ED%83%80_%ED%86%A0%EB%AA%A8%ED%9E%88%EC%82%AC" title="야마시타 토모히사">야마시타 토모히사</a>와는 호리코시 고등학교 동창생. 가족들끼리 교류하는 사이여서 사적으로도 매우 친하다.</p>
<p>친구는 아카니시군단 ( <a href="/wiki/%EC%95%84%EC%B9%B4%EB%8B%88%EC%8B%9C_%EC%A7%84" title="아카니시 진">아카니시 진</a>, <a href="/wiki/%EC%95%BC%EB%A7%88%EC%8B%9C%ED%83%80_%ED%86%A0%EB%AA%A8%ED%9E%88%EC%82%AC" title="야마시타 토모히사">야마시타 토모히사</a> ) 이다.</p>
<p>형인 시로타 준과는 얼굴이 똑같이 생겼다. 두사람은 모두 <a href="/wiki/%EC%9D%B8%EC%8A%A4%ED%83%80%EA%B7%B8%EB%9E%A8" title="인스타그램">인스타그램</a> 과 <a href="/wiki/%ED%8A%B8%EC%9C%84%ED%84%B0" title="트위터">트위터</a> 를 하고 있다.</p>
