<p><b>시만텍</b>(Symantec, <span style="font-size:small"><a href="/wiki/%EB%82%98%EC%8A%A4%EB%8B%A5" title="나스닥">나스닥</a>: <a class="external text" href="http://quotes.nasdaq.com/asp/SummaryQuote.asp?symbol=SYMC&amp;selected=SYMC" rel="nofollow"><b>SYMC</b></a></span>)은 1982년에 <a class="new" href="/w/index.php?title=%EA%B2%8C%EB%A6%AC_%ED%97%A8%EB%93%9C%EB%A6%AD%EC%8A%A4&amp;action=edit&amp;redlink=1" title="게리 헨드릭스 (없는 문서)">게리 헨드릭스</a>가 <a class="mw-redirect" href="/wiki/%EA%B5%AD%EB%A6%BD%EA%B3%BC%ED%95%99%EC%9E%AC%EB%8B%A8" title="국립과학재단">국립과학재단</a>의 보조금으로 설립한 회사이다. 시만텍은 원래 데이터베이스 프로그램을 비롯한 <a class="mw-redirect" href="/wiki/%EC%9D%B8%EA%B3%B5_%EC%A7%80%EB%8A%A5" title="인공 지능">인공 지능</a> 관련 프로젝트에 초점을 두었었다. 헨드릭스는 몇 명의 <a class="mw-redirect" href="/wiki/%EC%8A%A4%ED%83%A0%ED%8F%AC%EB%93%9C_%EB%8C%80%ED%95%99%EA%B5%90" title="스탠포드 대학교">스탠포드 대학교</a> <a href="/wiki/%EC%9E%90%EC%97%B0_%EC%96%B8%EC%96%B4_%EC%B2%98%EB%A6%AC" title="자연 언어 처리">자연어 처리</a> 연구원들을 회사의 첫 직원들로 고용하였다. 1984년에 시만텍은 다른 매우 작은 컴퓨터 소프트웨어 시발 업체인, <a class="new" href="/w/index.php?title=%EA%B3%A0%EB%93%A0_%EC%9C%A0%EB%B1%85%EC%8A%A4&amp;action=edit&amp;redlink=1" title="고든 유뱅스 (없는 문서)">고든 유뱅스</a>가 이끌었던 C&amp;E 소프트웨어(데니스 콜먼과 고든 유뱅스이 창립한 회사)에 인수되었다. 이렇게 병합된 회사는 시만텍이라는 이름을 그대로 갖고 있으면서 유뱅스는 <a href="/wiki/%EC%B5%9C%EA%B3%A0%EA%B2%BD%EC%98%81%EC%9E%90" title="최고경영자">최고경영 책임자</a>가 되었다. 자사의 첫 제품 <a class="new" href="/w/index.php?title=Q%26A_(%EC%86%8C%ED%94%84%ED%8A%B8%EC%9B%A8%EC%96%B4)&amp;action=edit&amp;redlink=1" title="Q&amp;A (소프트웨어) (없는 문서)">Q&amp;A</a>는 1985년에 출시되었다. Q&amp;A는 <a href="/wiki/%EB%8D%B0%EC%9D%B4%ED%84%B0%EB%B2%A0%EC%9D%B4%EC%8A%A4_%EA%B4%80%EB%A6%AC_%EC%8B%9C%EC%8A%A4%ED%85%9C" title="데이터베이스 관리 시스템">데이터베이스 관리</a> 기능을 제공하였고 그 안에 워드 처리 프로그램을 덩달아 추가하였다. 1990년 8월에 시만텍은 <a href="/wiki/%EB%8F%84%EC%8A%A4" title="도스">도스</a>용 응용 프로그램들을 개발한 <a class="new" href="/w/index.php?title=%ED%94%BC%ED%84%B0_%EB%85%B8%ED%84%B4_%EC%BB%B4%ED%93%A8%ED%8C%85&amp;action=edit&amp;redlink=1" title="피터 노턴 컴퓨팅 (없는 문서)">피터 노턴 컴퓨팅</a>을 사들였다. 시만텍의 소비자용 바이러스 검사 소프트웨어와 데이터 관리 유틸리티들은 현재까지도 <a class="new" href="/w/index.php?title=%ED%94%BC%ED%84%B0_%EB%85%B8%ED%84%B4&amp;action=edit&amp;redlink=1" title="피터 노턴 (없는 문서)">피터 노턴</a>의 이름으로 시장에 출시되고 있다.</p>
<p>2008년 11월 18일, 회장 겸 최고경영자 (CEO)인 존 톰슨이 회계연도 2009년이 종료되는 2009년 3월 CEO에서 사퇴할 것이라고 밝혔다. 시만텍 이사회는 현 시만텍 최고운영책임자(COO)인 엔리크 살렘을 2009년 4월 취임할 사장 겸 CEO로 지명했고,<sup class="reference" id="cite_ref-1"><a href="#cite_note-1">[1]</a></sup> 2009년 4월 이사회는 그를 CEO로 선임하였다.<sup class="reference" id="cite_ref-2"><a href="#cite_note-2">[2]</a></sup></p>
