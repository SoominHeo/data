<h2><span class="mw-headline" id=".EA.B5.AC.EC.84.B1.EC.9B.90">구성원</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=CAPSULE&amp;action=edit&amp;section=1" title="부분 편집: 구성원">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>두 사람 모두 <a href="/wiki/%EC%9D%B4%EC%8B%9C%EC%B9%B4%EC%99%80_%ED%98%84" title="이시카와 현">이시카와 현</a> <a href="/wiki/%EA%B0%80%EB%82%98%EC%9E%90%EC%99%80_%EC%8B%9C" title="가나자와 시">가나자와 시</a> 출신이다.</p>
<ul>
<li><a href="/wiki/%EB%82%98%EC%B9%B4%ED%83%80_%EC%95%BC%EC%8A%A4%ED%83%80%EC%B9%B4" title="나카타 야스타카">나카타 야스타카</a>: 작사, 작곡, 편곡, 프로듀스</li>
<li><a href="/wiki/%EA%B3%A0%EC%8B%9C%EC%A7%80%EB%A7%88_%EB%8F%84%EC%8B%9C%EC%BD%94" title="고시지마 도시코">고시지마 도시코</a>: 보컬</li>
</ul>
<h2><span class="mw-headline" id=".EA.B0.9C.EC.9A.94">개요</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=CAPSULE&amp;action=edit&amp;section=2" title="부분 편집: 개요">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>본래 고등학교 시절의 동창이었던 <a href="/wiki/%EB%82%98%EC%B9%B4%ED%83%80_%EC%95%BC%EC%8A%A4%ED%83%80%EC%B9%B4" title="나카타 야스타카">나카타 야스타카</a>와 <a href="/wiki/%EA%B3%A0%EC%8B%9C%EC%A7%80%EB%A7%88_%EB%8F%84%EC%8B%9C%EC%BD%94" title="고시지마 도시코">고시지마 도시코</a>는 <a href="/wiki/1997%EB%85%84" title="1997년">1997년</a>에 capsule을 결성하였고, 그들이 성인이 된 이후인 <a href="/wiki/2001%EB%85%84" title="2001년">2001년</a>에 정식으로 데뷔했다. <a href="/wiki/2006%EB%85%84" title="2006년">2006년</a>과 <a href="/wiki/2007%EB%85%84" title="2007년">2007년</a> 사이에 capsule의 음악적 성향은 크게 변하여, 기존의 복고적인 색채를 띤 <a class="new" href="/w/index.php?title=%ED%94%84%EB%A0%8C%EC%B9%98_%ED%8C%9D&amp;action=edit&amp;redlink=1" title="프렌치 팝 (없는 문서)">프렌치 팝</a> 스타일에서 <a href="/wiki/%ED%95%98%EC%9A%B0%EC%8A%A4_%EC%9D%8C%EC%95%85" title="하우스 음악">하우스</a>로 바뀌었고, 이에 따라 공연이나, 뮤직비디오 및 음반의 표지 사진을 촬영할 때 입는 의상에도 상당한 변화가 있었다. <a href="/wiki/%EB%82%98%EC%B9%B4%ED%83%80_%EC%95%BC%EC%8A%A4%ED%83%80%EC%B9%B4" title="나카타 야스타카">나카타 야스타카</a>는 <a class="new" href="/w/index.php?title=MEG&amp;action=edit&amp;redlink=1" title="MEG (없는 문서)">MEG</a>, <a href="/wiki/SMAP" title="SMAP">SMAP</a>, <a href="/wiki/M-flo" title="M-flo">m-flo</a>, <a class="mw-redirect" href="/wiki/%EC%95%84%EC%98%A4%EC%95%BC%EB%A7%88_%ED%85%8C%EB%A3%A8%EB%A7%88" title="아오야마 테루마">아오야마 테루마</a>, <a href="/wiki/%EC%8A%A4%EC%A6%88%ED%82%A4_%EC%95%84%EB%AF%B8" title="스즈키 아미">스즈키 아미</a>, <a href="/wiki/%EB%A6%AC%EC%95%84_%EB%94%94%EC%A1%B4" title="리아 디존">리아 디존</a>등 여러 아티스트에게 노래를 제공하거나 리믹서로 참가하고있다. 또, 프로듀스 하고 있는 아이돌 그룹인 <a class="mw-redirect" href="/wiki/%ED%8D%BC%ED%93%B8_(%EC%9D%8C%EC%95%85_%EA%B7%B8%EB%A3%B9)" title="퍼퓸 (음악 그룹)">퍼퓸</a>이 앨범 《GAME》이나 싱글 《love the world》로 <a class="mw-redirect" href="/wiki/%EC%98%A4%EB%A6%AC%EC%BD%98_%EC%B0%A8%ED%8A%B8" title="오리콘 차트">오리콘 차트</a> 1위를 차지하여 주목을 받고있다.</p>
<h2><span class="mw-headline" id=".EC.9D.8C.EB.B0.98_.EB.AA.A9.EB.A1.9D">음반 목록</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=CAPSULE&amp;action=edit&amp;section=3" title="부분 편집: 음반 목록">편집</a><span class="mw-editsection-bracket">]</span></span></h2>

<p><a href="/wiki/2008%EB%85%84" title="2008년">2008년</a> <a href="/wiki/3%EC%9B%94_1%EC%9D%BC" title="3월 1일">3월 1일</a>에는 <a href="/wiki/%EB%82%98%EC%B9%B4%ED%83%80_%EC%95%BC%EC%8A%A4%ED%83%80%EC%B9%B4" title="나카타 야스타카">나카타 야스타카</a>가 <a href="/wiki/%ED%81%B4%EB%9E%98%EC%A7%80%EC%BD%B0%EC%9D%B4_%ED%94%84%EB%A1%9C%EC%A0%9D%ED%8A%B8" title="클래지콰이 프로젝트">클래지콰이 프로젝트</a>의 ROBOTICA 파티 공연에 게스트로 참가했으며<sup class="reference" id="cite_ref-1"><a href="#cite_note-1">[1]</a></sup>, <a href="/wiki/2009%EB%85%84" title="2009년">2009년</a> <a class="new" href="/w/index.php?title=3%EC%9B%94_14&amp;action=edit&amp;redlink=1" title="3월 14 (없는 문서)">3월 14</a>일에는 <a href="/wiki/%ED%99%8D%EC%9D%B5%EB%8C%80%ED%95%99%EA%B5%90" title="홍익대학교">홍익대학교</a> 근교의 <a href="/wiki/%ED%81%B4%EB%9F%BD_(%EA%B3%B5%EC%97%B0)" title="클럽 (공연)">클럽</a>인 <a class="new" href="/w/index.php?title=%EB%B2%A8%EB%B2%B3%EB%B0%94%EB%82%98%EB%82%98&amp;action=edit&amp;redlink=1" title="벨벳바나나 (없는 문서)">벨벳바나나</a>에서 첫 내한 공연을 펼쳤다<sup class="reference" id="cite_ref-2"><a href="#cite_note-2">[2]</a></sup>.</p>
<p>2012년 UMF KOREA에서 capsule이 게스트로 확정돼 내한 공연을 펼친 바 있다.</p>
