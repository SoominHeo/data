<p><b>라누스</b> (<span lang="es" xml:lang="es">Club Atlético Lanús</span>)는 <a href="/wiki/%EC%95%84%EB%A5%B4%ED%97%A8%ED%8B%B0%EB%82%98" title="아르헨티나">아르헨티나</a> <a class="mw-redirect" href="/wiki/%EB%B6%80%EC%97%90%EB%85%B8%EC%8A%A4_%EC%95%84%EC%9D%B4%EB%A0%88%EC%8A%A4" title="부에노스 아이레스">부에노스 아이레스</a>의 <a href="/wiki/%EB%9D%BC%EB%88%84%EC%8A%A4" title="라누스">라누스</a> 지역을 근거로 하는 <a href="/wiki/%EC%B6%95%EA%B5%AC" title="축구">축구</a><a class="mw-redirect" href="/wiki/%ED%81%B4%EB%9F%BD" title="클럽">클럽</a>이다. <a href="/wiki/1915%EB%85%84" title="1915년">1915년</a> <a href="/wiki/1%EC%9B%94_3%EC%9D%BC" title="1월 3일">1월 3일</a> 설립되었다.</p>
<p>수차례 <a href="/wiki/%EC%95%84%EB%A5%B4%ED%97%A8%ED%8B%B0%EB%82%98_%ED%94%84%EB%A6%AC%EB%A9%94%EB%9D%BC_%EB%94%94%EB%B9%84%EC%8B%9C%EC%98%A8" title="아르헨티나 프리메라 디비시온">1부 리그</a>와 <a href="/wiki/%ED%94%84%EB%A6%AC%EB%A9%94%EB%9D%BC_B_%EB%82%98%EC%8B%9C%EC%98%A4%EB%82%A0" title="프리메라 B 나시오날">2부 리그</a>를 전전하던 팀이었으나, <a href="/wiki/1996%EB%85%84" title="1996년">1996년</a> <a href="/wiki/%EC%BD%94%ED%8C%8C_CONMEBOL" title="코파 CONMEBOL">코파 CONMEBOL</a>을 우승하고, <a href="/wiki/2007%EB%85%84" title="2007년">2007년</a>에는 클럽 역사상 가장 화려한 성과를 거두면서 창단 92년 만에 프리메라 디비시온(<a class="mw-redirect" href="/wiki/%EC%95%84%ED%8E%98%EB%A5%B4%ED%88%AC%EB%9D%BC" title="아페르투라">아페르투라</a>) 우승을 차지했다. 아페르투라 기간중 <a class="new" href="/w/index.php?title=%ED%98%B8%EC%84%B8_%EC%82%B0%EB%93%9C&amp;action=edit&amp;redlink=1" title="호세 산드 (없는 문서)">호세 산드</a>(José Sand)는 15 경기에서 15골을 넣으면서 팀의 영웅이 되었다.</p>
<p>라누스는 <a href="/wiki/1978%EB%85%84" title="1978년">1978년</a> 약 200만 달러의 채무로 파산 직전에 이르렀으나 <a class="new" href="/w/index.php?title=%EC%86%8C%EC%8B%9C%EC%98%A4&amp;action=edit&amp;redlink=1" title="소시오 (없는 문서)">소시오</a>들의 원조로 채무관계를 청산하고 2부 리그로 복귀한 역사가 있다. 라누스 소시오들의 활약은 인상적이어서 당시 클럽 소시오의 숫자는 2 천여 명이었으나 <a href="/wiki/1981%EB%85%84" title="1981년">1981년</a>에는 만 여명을 넘겼다.</p>
<p>라누스는 축구팀 외에 아마추어 <a href="/wiki/%EB%86%8D%EA%B5%AC" title="농구">농구</a>, <a href="/wiki/%EB%B0%B0%EA%B5%AC" title="배구">배구</a>, <a href="/wiki/%ED%95%A9%EA%B8%B0%EB%8F%84" title="합기도">합기도</a>, <a href="/wiki/%ED%95%B8%EB%93%9C%EB%B3%BC" title="핸드볼">핸드볼</a>, <a href="/wiki/%ED%83%9C%EA%B6%8C%EB%8F%84" title="태권도">태권도</a> 팀 등도 소유하고 있다.</p>
