<p><b>시사껫 주</b>(<span style="font-size: smaller;"><a href="/wiki/%ED%83%9C%EA%B5%AD%EC%96%B4" title="태국어">태국어</a>:</span> <span lang="th" xml:lang="th">ศรีสะเกษ</span>)는 <a href="/wiki/%ED%83%80%EC%9D%B4" title="타이">타이</a> <a href="/wiki/%EC%9D%B4%EC%82%B0_(%ED%83%80%EC%9D%B4)" title="이산 (타이)">북동부</a>에 있는 주(창왓)의 하나이다. 이웃한 주는 서쪽부터 시계 방향으로 <a href="/wiki/%EC%88%98%EB%A6%B0_%EC%A3%BC" title="수린 주">수린 주</a>, <a href="/wiki/%EB%A1%9C%EC%9D%B4%EC%97%A3_%EC%A3%BC" title="로이엣 주">로이엣 주</a>, <a href="/wiki/%EC%95%BC%EC%86%8C%ED%86%A4_%EC%A3%BC" title="야소톤 주">야소톤 주</a>, <a href="/wiki/%EC%9A%B0%EB%B3%B8%EB%9E%8F%EC%B0%A8%ED%83%80%EB%8B%88_%EC%A3%BC" title="우본랏차타니 주">우본랏차타니 주</a>가 있다. 남쪽으로 <a href="/wiki/%EC%BA%84%EB%B3%B4%EB%94%94%EC%95%84" title="캄보디아">캄보디아</a>의 <a class="new" href="/w/index.php?title=%EC%98%A4%EB%8B%A4%EB%A9%98%EC%B2%B4%EC%9D%B4_%EC%A3%BC&amp;action=edit&amp;redlink=1" title="오다멘체이 주 (없는 문서)">오다멘체이 주</a>, <a class="new" href="/w/index.php?title=%ED%94%84%EB%A0%88%EB%B9%84%ED%97%A4%EC%95%84_%EC%A3%BC&amp;action=edit&amp;redlink=1" title="프레비헤아 주 (없는 문서)">프레비헤아 주</a>와 접한다.</p>

<h2><span class="mw-headline" id=".EC.A7.80.EB.A6.AC">지리</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%8B%9C%EC%82%AC%EA%BB%AB_%EC%A3%BC&amp;action=edit&amp;section=1" title="부분 편집: 지리">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>주는 <a href="/wiki/%EB%A9%94%EC%BD%A9_%EA%B0%95" title="메콩 강">메콩 강</a>의 지류인 <a href="/wiki/%EB%AC%B8_%EA%B0%95" title="문 강">문 강</a>의 계곡에 위치한다. 주 남쪽의 <a class="new" href="/w/index.php?title=%EB%8B%B9%EB%A0%89_%EC%82%B0%EB%A7%A5&amp;action=edit&amp;redlink=1" title="당렉 산맥 (없는 문서)">당렉 산맥</a>이 <a href="/wiki/%EC%BA%84%EB%B3%B4%EB%94%94%EC%95%84" title="캄보디아">캄보디아</a>와의 국경을 형성한다.</p>
<h2><span class="mw-headline" id=".EC.97.AD.EC.82.AC">역사</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%8B%9C%EC%82%AC%EA%BB%AB_%EC%A3%BC&amp;action=edit&amp;section=2" title="부분 편집: 역사">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>주내에서 발견되는 많은 크메르 유적들은 적어도 12세기에 이곳이 크메르 제국에게 중요한 곳이었음을 보여준다. 지역의 구전에 따르면 이곳은 시나꼰람두안으로 불렸다. <a class="mw-redirect" href="/wiki/%EC%95%84%EC%9C%A0%ED%83%80%EC%95%BC_%EC%99%95%EA%B5%AD" title="아유타야 왕국">아유타야 왕국</a>의 <a class="new" href="/w/index.php?title=%EB%B3%B4%EB%A1%9C%EB%A7%88%EB%9E%8F%EC%B0%A8_3%EC%84%B8&amp;action=edit&amp;redlink=1" title="보로마랏차 3세 (없는 문서)">보로마랏차 3세</a> 왕의 치세 때인 15세기 후반에 이곳에 읍이 세워진 후에는 쿠칸으로 불렸다. <a href="/wiki/%EB%9D%BC%EC%98%A4%EC%A1%B1" title="라오족">라오족</a>이 주의 북부에 정착하면서 1786년에 쿠칸에 속하는 시사껫 읍이 형성되었다. 1904년에 원래의 읍은 후아이누아로 명명되었고 시사껫은 쿠칸으로 개칭되었다. 1912년에는 이 지역의 대부분을 관리했을 것으로 추정되는 몬톤 우돈타니가 설치되었다. 1933년에 몬톤 체제가 폐지되면서 쿤칸 주는 방콕의 직할이 되었다. 읍과 주의 이름은 1938년에 시사껫으로 복구되었다.</p>
<h2><span class="mw-headline" id=".EC.9D.B8.EA.B5.AC">인구</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%8B%9C%EC%82%AC%EA%BB%AB_%EC%A3%BC&amp;action=edit&amp;section=3" title="부분 편집: 인구">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>2000년의 인구 조사에서 <a href="/wiki/%ED%81%AC%EB%A9%94%EB%A5%B4%EC%96%B4" title="크메르어">크메르어</a>를 사용할 수 있는 인구는 26.2%로 보고되었다. 이것은 1990년의 인구 조사 때의 30.2%에 비하면 줄어든 것이다.</p>
<h2><span class="mw-headline" id=".ED.96.89.EC.A0.95_.EA.B5.AC.EC.97.AD">행정 구역</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%8B%9C%EC%82%AC%EA%BB%AB_%EC%A3%BC&amp;action=edit&amp;section=4" title="부분 편집: 행정 구역">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>주에는 22개의 <a href="/wiki/%EC%95%94%ED%94%84" title="암프">암프</a>(군)와 더 세부 행정구역인 206개의 <a href="/wiki/%EB%95%80%EB%B3%B8" title="땀본">땀본</a> 그리고 2411개의 <a class="mw-redirect" href="/wiki/%EB%AC%B4%EB%B0%98" title="무반">무반</a>이 있다.</p>
