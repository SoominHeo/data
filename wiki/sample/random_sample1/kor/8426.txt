<p><b>시솽반나 다이족 자치주</b>(<a class="new" href="/w/index.php?title=%ED%83%80%EC%9D%B4%EB%A3%A8%EC%96%B4&amp;action=edit&amp;redlink=1" title="타이루어 (없는 문서)">타이루어</a>: <a class="image" href="/wiki/%ED%8C%8C%EC%9D%BC:Xishuangbanna.svg"><img alt="Xishuangbanna.svg" data-file-height="99" data-file-width="365" height="16" src="//upload.wikimedia.org/wikipedia/commons/thumb/a/ac/Xishuangbanna.svg/60px-Xishuangbanna.svg.png" srcset="//upload.wikimedia.org/wikipedia/commons/thumb/a/ac/Xishuangbanna.svg/90px-Xishuangbanna.svg.png 1.5x, //upload.wikimedia.org/wikipedia/commons/thumb/a/ac/Xishuangbanna.svg/120px-Xishuangbanna.svg.png 2x" width="60"/></a>, sípsɔ́ngpǎnnǎ, <span style="font-size: smaller;"><a href="/wiki/%ED%83%9C%EA%B5%AD%EC%96%B4" title="태국어">태국어</a>:</span> <span lang="th" xml:lang="th">สิบสองปันนา</span>, <small><a href="/wiki/%EC%A4%91%EA%B5%AD%EC%96%B4" title="중국어">중국어</a></small> <small><a href="/wiki/%EA%B0%84%EC%B2%B4%EC%9E%90" title="간체자">간체</a>:</small> <span lang="zh-hans" xml:lang="zh-hans">西双版納傣族自治州</span>, <small><a href="/wiki/%ED%95%9C%EC%96%B4_%EB%B3%91%EC%9D%8C" title="한어 병음">병음</a>:</small> <font face="Lucida Sans Unicode">Xīshuāngbǎnnà dǎizú Zìzhìzhōu</font>, <a class="new" href="/w/index.php?title=%EC%95%84%EC%B9%B4%EC%96%B4&amp;action=edit&amp;redlink=1" title="아카어 (없는 문서)">아카어</a>: Siˇsawˇpaˆna, <a class="new" href="/w/index.php?title=%ED%95%98%EB%8B%88%EC%96%B4&amp;action=edit&amp;redlink=1" title="하니어 (없는 문서)">하니어</a>: Xisual banaq)는 <a href="/wiki/%EC%9C%88%EB%82%9C_%EC%84%B1" title="윈난 성">윈난 성</a> 최남단에 위치한 <a href="/wiki/%EB%8B%A4%EC%9D%B4%EC%A1%B1" title="다이족">다이족</a> <a href="/wiki/%EC%9E%90%EC%B9%98" title="자치">자치</a> 지역이다. 중국어의 시솽반나(西双版納)라는 표기는 <a class="mw-redirect" href="/wiki/%ED%83%80%EC%9D%B4%EC%96%B4_(%EC%96%B8%EC%96%B4)" title="타이어 (언어)">타이어</a> "<a class="mw-redirect" href="/wiki/%EC%8B%AD%EC%86%A1%EB%B9%A4%EB%82%98" title="십송빤나">십송빤나</a>"(สิบสองปันนา)를 음차한 표기이다.</p>
<p>면적은 19700 평방km, 인구는 90만 명이다. 중심지는 <a href="/wiki/%EC%A7%95%ED%9B%99_%EC%8B%9C" title="징훙 시">징훙</a>이다. 이곳에는 예부터 소수민족인 <a class="new" href="/w/index.php?title=%ED%83%80%EC%9D%B4%EB%A3%A8%EC%A1%B1&amp;action=edit&amp;redlink=1" title="타이루족 (없는 문서)">타이루족</a>이 거주했으며 <a href="/wiki/%EC%BD%94%EB%81%BC%EB%A6%AC" title="코끼리">코끼리</a>가 서식하는 곳이다. 이곳은 <a class="mw-redirect" href="/wiki/%EC%86%8C%EC%8A%B9_%EB%B6%88%EA%B5%90" title="소승 불교">소승 불교</a>가 번성하는 등 <a href="/wiki/%EB%8F%99%EB%82%A8%EC%95%84%EC%8B%9C%EC%95%84" title="동남아시아">동남아시아</a>적인 광경이 펼쳐지는 곳이다. 근년에는 관광지로 발전했다.</p>

<h2><span class="mw-headline" id=".EC.A7.80.EB.A6.AC">지리</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%8B%9C%EC%86%BD%EB%B0%98%EB%82%98_%EB%8B%A4%EC%9D%B4%EC%A1%B1_%EC%9E%90%EC%B9%98%EC%A3%BC&amp;action=edit&amp;section=1" title="부분 편집: 지리">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>남쪽은 <a href="/wiki/%EB%9D%BC%EC%98%A4%EC%8A%A4" title="라오스">라오스</a>, <a href="/wiki/%EB%AF%B8%EC%96%80%EB%A7%88" title="미얀마">미얀마</a>로 966킬로미터에 이르는 국경선에 접해 있다. <a href="/wiki/%EB%A9%94%EC%BD%A9_%EA%B0%95" title="메콩 강">메콩 강</a>이 남쪽으로 흐르고 메콩 수운으로 <a href="/wiki/%EB%9D%BC%EC%98%A4%EC%8A%A4" title="라오스">라오스</a>나 <a href="/wiki/%ED%83%80%EC%9D%B4" title="타이">타이</a>와 연결되어 있다.</p>
<h2><span class="mw-headline" id=".ED.96.89.EC.A0.95_.EA.B5.AC.EC.97.AD">행정 구역</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%8B%9C%EC%86%BD%EB%B0%98%EB%82%98_%EB%8B%A4%EC%9D%B4%EC%A1%B1_%EC%9E%90%EC%B9%98%EC%A3%BC&amp;action=edit&amp;section=2" title="부분 편집: 행정 구역">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>1개의 현급시, 2개의 현을 관할한다.</p>
<ul>
<li><a href="/wiki/%ED%98%84%EA%B8%89%EC%8B%9C" title="현급시">현급시</a>
<ul>
<li><a href="/wiki/%EC%A7%95%ED%9B%99_%EC%8B%9C" title="징훙 시">징훙 시</a>(景洪市)</li>
