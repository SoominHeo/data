<h1 class="firstHeading" id="firstHeading" lang="ko">시야 번</h1>

<div class="mw-content-ltr" dir="ltr" id="mw-content-text" lang="ko"><script>function mfTempOpenSection(id){var block=document.getElementById("mf-section-"+id);block.className+=" open-block";block.previousSibling.className+=" open-block";}</script><p><b>시야 번</b>(<span style="font-size: smaller;"><a href="/wiki/%EC%9D%BC%EB%B3%B8%EC%96%B4" title="일본어">일본어</a>:</span> <span lang="ja" xml:lang="ja">椎谷藩</span> <small>시야한<sup>[<a href="/wiki/%EC%9C%84%ED%82%A4%EB%B0%B1%EA%B3%BC:%EC%9D%BC%EB%B3%B8%EC%96%B4%EC%9D%98_%ED%95%9C%EA%B8%80_%ED%91%9C%EA%B8%B0" title="위키백과:일본어의 한글 표기">*</a>]</sup></small>)은 <a href="/wiki/%EC%9D%BC%EB%B3%B8" title="일본">일본</a> <a href="/wiki/%EC%97%90%EB%8F%84_%EC%8B%9C%EB%8C%80" title="에도 시대">에도 시대</a>에 있던 <a class="mw-redirect" href="/wiki/%EC%9D%BC%EB%B3%B8%EC%9D%98_%EB%B2%88" title="일본의 번">번</a>으로, <a href="/wiki/%EC%97%90%EC%B9%98%EA%B3%A0_%EA%B5%AD" title="에치고 국">에치고 국</a> <a href="/wiki/%EA%B0%80%EB%A6%AC%EC%99%80_%EA%B5%B0" title="가리와 군">가리와 군</a> 시야 일대를 지배하였다. 번청은 <a class="new" href="/w/index.php?title=%EC%8B%9C%EC%95%BC_%EC%A7%84%EC%95%BC&amp;action=edit&amp;redlink=1" title="시야 진야 (없는 문서)">시야 진야</a>이다.</p>
<h2><span class="mw-headline" id=".EC.97.AD.EC.82.AC">역사</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%8B%9C%EC%95%BC_%EB%B2%88&amp;action=edit&amp;section=1" title="부분 편집: 역사">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>번조(藩祖)는 <a class="new" href="/w/index.php?title=%ED%98%B8%EB%A6%AC_%EB%82%98%EC%98%A4%EB%A7%88%EC%82%AC&amp;action=edit&amp;redlink=1" title="호리 나오마사 (없는 문서)">호리 나오마사</a>의 넷째 아들 <a class="new" href="/w/index.php?title=%ED%98%B8%EB%A6%AC_%EB%82%98%EC%98%A4%EC%9C%A0%ED%82%A4&amp;action=edit&amp;redlink=1" title="호리 나오유키 (없는 문서)">호리 나오유키</a>이다. <a href="/wiki/%EA%B2%90%EB%82%98" title="겐나">겐나</a> 2년(<a href="/wiki/1616%EB%85%84" title="1616년">1616년</a>) 7월, 나오유키는 <a href="/wiki/%EC%98%A4%EC%82%AC%EC%B9%B4_%EC%A0%84%ED%88%AC" title="오사카 전투">오사카 전투</a>에서 군공을 세워, 에치고노쿠니 누타리 군(<span lang="ja" xml:lang="ja">沼垂郡</span>)의 5,500석 영지를 받아, 시야(<span lang="ja" xml:lang="ja">椎谷</span>)에 <a href="/wiki/%EC%A7%84%EC%95%BC" title="진야">진야</a>를 설치하였다. 나오유키의 뒤를 이은 <a class="new" href="/w/index.php?title=%ED%98%B8%EB%A6%AC_%EB%82%98%EC%98%A4%EC%B9%B4%EA%B2%8C&amp;action=edit&amp;redlink=1" title="호리 나오카게 (없는 문서)">호리 나오카게</a> 때, <a class="mw-redirect" href="/wiki/%EA%B0%84%ED%86%A0" title="간토">간토</a>의 땅 1만 석을 더 받아 <a href="/wiki/%EB%8B%A4%EC%9D%B4%EB%AC%98" title="다이묘">다이묘</a>가 되었다. <a href="/wiki/%EA%B2%90%EB%A1%9C%EC%BF%A0" title="겐로쿠">겐로쿠</a> 11년(<a href="/wiki/1698%EB%85%84" title="1698년">1698년</a>) 가문의 4대 당주 <a class="new" href="/w/index.php?title=%ED%98%B8%EB%A6%AC_%EB%82%98%EC%98%A4%EC%82%AC%EB%8B%A4_(1665%EB%85%84)&amp;action=edit&amp;redlink=1" title="호리 나오사다 (1665년) (없는 문서)">호리 나오사다</a>가 에치고의 누타리, 미시마(<span lang="ja" xml:lang="ja">三島</span>), 간바라(<span lang="ja" xml:lang="ja">蒲原</span>) 3개 군 안의 1만 석 영지를 다스리면서, 시야에 거주하였던 것이 정식적인 시야 번의 시작이다. 역대 번주들은 에도에 정부(定府)하는 것이 상례였다.</p>
<p>시야 번의 8대 번주 <a class="new" href="/w/index.php?title=%ED%98%B8%EB%A6%AC_%EC%95%84%ED%82%A4%ED%86%A0%EB%AA%A8&amp;action=edit&amp;redlink=1" title="호리 아키토모 (없는 문서)">호리 아키토모</a> 때가 되자 재정이 궁핍해졌다. 이 때문에 영지의 백성들에게 엄혹한 세금들을 징수하였다. 그렇지만 아키토모는 병약하여, 분가인 <a class="new" href="/w/index.php?title=%ED%98%B8%EB%A6%AC_%EB%82%98%EC%98%A4%EB%AA%A8%ED%86%A0&amp;action=edit&amp;redlink=1" title="호리 나오모토 (없는 문서)">호리 나오모토</a>가 실제 정무를 대행했다. 나오모토는 <a href="/wiki/%EB%8D%B4%EB%A9%94%EC%9D%B4_%EB%8C%80%EA%B8%B0%EA%B7%BC" title="덴메이 대기근">덴메이 대기근</a>이 일어남에 따른 쌀 가격의 앙등을 배경으로, 구라마이(<span lang="ja" xml:lang="ja">蔵米</span>)를 경쟁입찰에 내걸었다. 이에 격분한 농민들이 소동을 일으켜, <a class="new" href="/w/index.php?title=%EB%8D%B4%EB%A9%94%EC%9D%B4_%EC%9D%98%EB%AF%BC_%EC%82%AC%EA%B1%B4&amp;action=edit&amp;redlink=1" title="덴메이 의민 사건 (없는 문서)">덴메이 의민 사건</a>으로 발전하였다. 이 사건은 수년간에 걸쳐 농민의 직소와 번의 탄압이 반복되고, 결국에는 막부의 판정을 받기에 이르렀다. 막부는 5차례에 걸친 평정의 결과, <a href="/wiki/%EA%B0%84%EC%84%B8%EC%9D%B4" title="간세이">간세이</a> 4년(<a href="/wiki/1792%EB%85%84" title="1792년">1792년</a>)에 번주 아키토모는 은거하고 <a class="mw-redirect" href="/wiki/%EB%AF%B8%EC%B9%B4%EC%99%80%EB%85%B8%EC%BF%A0%EB%8B%88" title="미카와노쿠니">미카와노쿠니</a> <a class="new" href="/w/index.php?title=%EB%8B%88%EC%8B%9C%EC%98%A4_%EB%B2%88&amp;action=edit&amp;redlink=1" title="니시오 번 (없는 문서)">니시오 번</a>주 <a href="/wiki/%EB%A7%88%EC%93%B0%EB%8B%A4%EC%9D%B4%EB%9D%BC_%EB%85%B8%EB%A6%AC%EC%8A%A4%EC%BC%80" title="마쓰다이라 노리스케">마쓰다이라 노리스케</a>의 일곱 번째 아들(<a class="new" href="/w/index.php?title=%ED%98%B8%EB%A6%AC_%EB%82%98%EC%98%A4%EB%85%B8%EB%A6%AC&amp;action=edit&amp;redlink=1" title="호리 나오노리 (없는 문서)">호리 나오노리</a>)을 후계자로 옹립할 것, 그리고 전체 영지의 절반인 5천 석을 타지와 교체하라는 처분이 내려졌다. 이때 <a class="mw-redirect" href="/wiki/%EC%8B%9C%EB%82%98%EB%85%B8%EB%85%B8%EC%BF%A0%EB%8B%88" title="시나노노쿠니">시나노노쿠니</a> 다카이 군(<span lang="ja" xml:lang="ja">高井郡</span>) 로쿠카와 촌(<span lang="ja" xml:lang="ja">六川村</span>)에 로쿠카와 진야가 설치되었다.</p>
<p>그 후 번내에서는 번 통치 개혁도 시행되었다. 하지만 <a href="/wiki/%EA%B0%80%EC%97%90%EC%9D%B4" title="가에이">가에이</a> 4년(<a href="/wiki/1851%EB%85%84" title="1851년">1851년</a>), 가로인 <a class="new" href="/w/index.php?title=%EC%82%AC%EC%9D%B4%ED%86%A0_%EC%95%BC%EC%98%A4%EC%8B%9C%EB%A1%9C&amp;action=edit&amp;redlink=1" title="사이토 야오시로 (없는 문서)">사이토 야오시로</a>에 의한 번주 <a class="new" href="/w/index.php?title=%ED%98%B8%EB%A6%AC_%EC%9C%A0%ED%82%A4%ED%86%A0%EC%8B%9C&amp;action=edit&amp;redlink=1" title="호리 유키토시 (없는 문서)">호리 유키토시</a> 암살 미수 사건이 일어나는 등, 번 통치는 늘 불안정한 상태였다. <a class="mw-disambig" href="/wiki/%EB%A9%94%EC%9D%B4%EC%A7%80" title="메이지">메이지</a> 원년(<a href="/wiki/1868%EB%85%84" title="1868년">1868년</a>) <a href="/wiki/%EB%B3%B4%EC%8B%A0_%EC%A0%84%EC%9F%81" title="보신 전쟁">보신 전쟁</a>에서 시야 번은 <a href="/wiki/%EB%AF%B8%ED%86%A0_%EB%B2%88" title="미토 번">미토 번</a>과 <a href="/wiki/%EC%82%AC%EC%93%B0%EB%A7%88_%EB%B2%88" title="사쓰마 번">사쓰마 번</a>, <a href="/wiki/%EC%A1%B0%EC%8A%88_%EB%B2%88" title="조슈 번">조슈 번</a>에 의한 격전의 무대가 되었고, 전쟁으로 인한 심대한 피해를 입었다. 이듬해, 번주 <a class="new" href="/w/index.php?title=%ED%98%B8%EB%A6%AC_%EC%9C%A0%ED%82%A4%EC%9A%94%EC%8B%9C&amp;action=edit&amp;redlink=1" title="호리 유키요시 (없는 문서)">호리 유키요시</a>는 <a href="/wiki/%ED%8C%90%EC%A0%81%EB%B4%89%ED%99%98" title="판적봉환">판적봉환</a>에 의해 시야 번지사가 되었다. 그리고 메이지 4년(<a href="/wiki/1871%EB%85%84" title="1871년">1871년</a>) <a href="/wiki/%ED%8F%90%EB%B2%88%EC%B9%98%ED%98%84" title="폐번치현">폐번치현</a>에 의해 시야 번은 폐지되었고, 시야 현이 설치되었다.</p>
