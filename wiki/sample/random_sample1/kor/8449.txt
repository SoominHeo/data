<p><b>시오야 역</b>(<span style="font-size: smaller;"><a href="/wiki/%EC%9D%BC%EB%B3%B8%EC%96%B4" title="일본어">일본어</a>:</span> <span lang="ja"><span lang="ja"><ruby><rb>塩屋駅</rb> <rp>(</rp><rt>しおやえき</rt><rp>)</rp></ruby></span></span> <small>시오야에키<sup>[<a href="/wiki/%EC%9C%84%ED%82%A4%EB%B0%B1%EA%B3%BC:%EC%9D%BC%EB%B3%B8%EC%96%B4%EC%9D%98_%ED%95%9C%EA%B8%80_%ED%91%9C%EA%B8%B0" title="위키백과:일본어의 한글 표기">*</a>]</sup></small>)은 <a href="/wiki/%EC%9D%BC%EB%B3%B8" title="일본">일본</a> <a href="/wiki/%ED%9A%A8%EA%B3%A0_%ED%98%84" title="효고 현">효고 현</a> <a href="/wiki/%EA%B3%A0%EB%B2%A0_%EC%8B%9C" title="고베 시">고베 시</a> <a href="/wiki/%EB%8B%A4%EB%A3%A8%EB%AF%B8_%EA%B5%AC" title="다루미 구">다루미 구</a>에 있는 <a class="mw-redirect" href="/wiki/%EC%84%9C%EC%9D%BC%EB%B3%B8_%EC%97%AC%EA%B0%9D%EC%B2%A0%EB%8F%84" title="서일본 여객철도">서일본 여객철도</a> <a href="/wiki/%EC%B2%A0%EB%8F%84%EC%97%AD" title="철도역">역</a>이다.</p>
<h2><span class="mw-headline" id=".EC.97.AD.EC.82.AC">역사</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%8B%9C%EC%98%A4%EC%95%BC_%EC%97%AD_(%ED%9A%A8%EA%B3%A0_%ED%98%84)&amp;action=edit&amp;section=1" title="부분 편집: 역사">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>내측선에 있는 상대식 승강장으로 2면 2선의 고가역이다. 홈에 연결된 계단은 상행 방면으로 2개, 하행 방면으로 한 개이다. 자동개찰기가 3대 설치되어 있다. 현재 시오야 역에는 <a href="/wiki/%EC%97%98%EB%A6%AC%EB%B2%A0%EC%9D%B4%ED%84%B0" title="엘리베이터">엘리베이터</a>가 설치되어 있지 않아 설치가 예정되어 있다. <a class="new" href="/w/index.php?title=%EC%84%9C%EC%9D%BC%EB%B3%B8_%EC%97%AC%EA%B0%9D%EC%B2%A0%EB%8F%84_%EA%B5%90%ED%86%B5_%EC%84%9C%EB%B9%84%EC%8A%A4&amp;action=edit&amp;redlink=1" title="서일본 여객철도 교통 서비스 (없는 문서)">서일본 여객철도 교통 서비스</a>에 의한 위탁 영업역으로 일부 시간대에는 무인으로 운영된다.</p>
<p><a href="/wiki/%EC%96%B4%EB%B2%88_%EB%84%A4%ED%8A%B8%EC%9B%8C%ED%81%AC" title="어번 네트워크">어번 네트워크</a> 구역에 속해 있어 <a class="mw-redirect" href="/wiki/%EC%9D%B4%EC%BD%94%EC%B9%B4" title="이코카">이코카</a> 및 제휴된 <a class="mw-redirect" href="/wiki/IC_%EC%B9%B4%EB%93%9C" title="IC 카드">IC 카드</a>의 사용이 가능하다.</p>

<h2><span class="mw-headline" id=".EC.97.B0.ED.91.9C">연표</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%8B%9C%EC%98%A4%EC%95%BC_%EC%97%AD_(%ED%9A%A8%EA%B3%A0_%ED%98%84)&amp;action=edit&amp;section=2" title="부분 편집: 연표">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<ul>
<li><a href="/wiki/1896%EB%85%84" title="1896년">1896년</a> <a href="/wiki/7%EC%9B%94_1%EC%9D%BC" title="7월 1일">7월 1일</a>: <a class="new" href="/w/index.php?title=%EC%82%B0%EC%9A%94_%EC%B2%A0%EB%8F%84&amp;action=edit&amp;redlink=1" title="산요 철도 (없는 문서)">산요 철도</a>의 시오야 가정차장으로서 스마 ~ 다루미 간에 신설개업. 여객 및 화물 취급을 개시하였다.</li>
<li><a href="/wiki/1906%EB%85%84" title="1906년">1906년</a> <a href="/wiki/12%EC%9B%94_1%EC%9D%BC" title="12월 1일">12월 1일</a>: 산요 철도의 국유화에 따라 <a href="/wiki/%EC%9D%BC%EB%B3%B8%EA%B5%AD%EC%9C%A0%EC%B2%A0%EB%8F%84" title="일본국유철도">일본국유철도</a>로 이관. 동시에 역으로 승격하였다.</li>
<li><a href="/wiki/1909%EB%85%84" title="1909년">1909년</a> <a href="/wiki/10%EC%9B%94_12%EC%9D%BC" title="10월 12일">10월 12일</a>: 선로 명칭 제정에 따라 <a href="/wiki/%EC%82%B0%EC%9A%94_%EB%B3%B8%EC%84%A0" title="산요 본선">산요 본선</a>의 일부가 되었다.</li>
