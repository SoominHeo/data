<h2><span class="mw-headline" id=".EC.9D.B8.EB.AC.BC">인물</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%8B%9C%EC%98%A4%ED%83%80%EB%8B%88_%EA%B0%80%EC%A6%88%ED%9E%88%EC%BD%94&amp;action=edit&amp;section=1" title="부분 편집: 인물">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<h3><span class="mw-headline" id=".EC.84.A0.EC.88.98_.EC.8B.9C.EC.A0.88">선수 시절</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%8B%9C%EC%98%A4%ED%83%80%EB%8B%88_%EA%B0%80%EC%A6%88%ED%9E%88%EC%BD%94&amp;action=edit&amp;section=2" title="부분 편집: 선수 시절">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>1992년, 신코가쿠엔 신코 고등학교 시절에 포수로서 <a class="mw-redirect" href="/wiki/%EC%9D%BC%EB%B3%B8_%EC%A0%84%EA%B5%AD_%EA%B3%A0%EB%93%B1%ED%95%99%EA%B5%90_%EC%95%BC%EA%B5%AC_%EC%84%A0%EC%88%98%EA%B6%8C_%EB%8C%80%ED%9A%8C" title="일본 전국 고등학교 야구 선수권 대회">고시엔 대회</a>에 출전(제74회 전국 고등학교 야구 선수권 대회), 같은 해 가을에 열린 프로 야구 드래프트 회의에서 <a href="/wiki/%ED%95%9C%EC%8B%A0_%ED%83%80%EC%9D%B4%EA%B1%B0%EC%8A%A4" title="한신 타이거스">한신 타이거스</a>로부터 6순위로 지명을 받고 입단했다. 입단 당시에는 “계약금으로 어머니의 무덤을 짓겠다”라고 말했다.</p>
<p>당시 한신의 포수에는 <a href="/wiki/%EC%95%BC%EB%A7%88%EB%8B%A4_%EA%B0%80%EC%93%B0%ED%9E%88%EC%BD%94" title="야마다 가쓰히코">야마다 가쓰히코</a>, <a href="/wiki/%EA%B8%B0%EB%8F%84_%EA%B0%80%EC%93%B0%ED%9E%88%EC%BD%94" title="기도 가쓰히코">기도 가쓰히코</a>, <a href="/wiki/%EC%84%B8%ED%82%A4%EC%B9%B4%EC%99%80_%EA%B3%A0%EC%9D%B4%EC%B9%98" title="세키카와 고이치">세키카와 고이치</a> 등이 활약하고 있어 시오타니는 포수로서의 출전 기회를 받지 못하는 등 2군에서의 선수 생활이 계속되고 있었다. 1996년 10월 9일의 시즌 최종전이 되는 <a href="/wiki/%EC%A3%BC%EB%8B%88%EC%B9%98_%EB%93%9C%EB%9E%98%EA%B1%B4%EC%8A%A4" title="주니치 드래건스">주니치 드래건스</a>전에서 1회말에 <a href="/wiki/%EB%82%98%EC%B9%B4%EB%8B%88%EC%8B%9C_%EA%B8%B0%EC%9A%94%EC%98%A4%ED%82%A4" title="나카니시 기요오키">나카니시 기요오키</a>의 대타로 기용되어<sup class="reference" id="cite_ref-1"><a href="#cite_note-1">[1]</a></sup> 프로 첫 홈런이 되는 만루 홈런을 때려내 한신은 일본 프로 야구 사상 최초가 되는 ‘1이닝 만루 홈런 2개’를 기록했다(같은 이닝에 1개째가 되는 만루 홈런을 친 것은 <a href="/wiki/%EC%8B%A0%EC%A1%B0_%EC%93%B0%EC%9A%94%EC%8B%9C" title="신조 쓰요시">신조 쓰요시</a>였다).</p>
<p>시오타니가 입단한 이후에도 같은 포수인 <a href="/wiki/%EC%95%BC%EB%85%B8_%EC%95%84%ED%82%A4%ED%9E%88%EB%A1%9C" title="야노 아키히로">야노 아키히로</a>나 <a class="new" href="/w/index.php?title=%EC%9A%94%EC%8B%9C%EB%AA%A8%ED%86%A0_%EB%A3%8C&amp;action=edit&amp;redlink=1" title="요시모토 료 (없는 문서)">요시모토 료</a>가 한신에 이적해 왔기 때문에 1998년에 출전 기회를 늘리기 위해 <a href="/wiki/%EB%82%B4%EC%95%BC%EC%88%98" title="내야수">내야수</a>(주로 <a href="/wiki/3%EB%A3%A8%EC%88%98" title="3루수">3루수</a>)로 전향하는 등 타격이 안정되면서 가끔 정규 경기에 출전하게 되었지만 당시 <a href="/wiki/%EB%85%B8%EB%AC%B4%EB%9D%BC_%EA%B0%80%EC%93%B0%EC%95%BC" title="노무라 가쓰야">노무라 가쓰야</a> 감독에 의한 아들 <a class="new" href="/w/index.php?title=%EB%85%B8%EB%AC%B4%EB%9D%BC_%EA%B0%80%EC%93%B0%EB%85%B8%EB%A6%AC&amp;action=edit&amp;redlink=1" title="노무라 가쓰노리 (없는 문서)">가쓰노리</a>의 기용이나 <a class="new" href="/w/index.php?title=%ED%95%98%EB%A7%88%EB%82%98%EC%B9%B4_%EC%98%A4%EC%82%AC%EB%AC%B4&amp;action=edit&amp;redlink=1" title="하마나카 오사무 (없는 문서)">하마나카 오사무</a>를 시작으로 하는 젊은 야수들의 활약에 의해 타석에 들어설 기회가 적었다. 2001년 시즌 종료 후 <a class="new" href="/w/index.php?title=%EC%82%AC%EC%9D%B4%ED%86%A0_%ED%9E%88%EB%8D%B0%EB%AF%B8%EC%93%B0&amp;action=edit&amp;redlink=1" title="사이토 히데미쓰 (없는 문서)">사이토 히데미쓰</a>와의 맞트레이드로 <a href="/wiki/%EC%98%A4%EB%A6%AD%EC%8A%A4_%EB%B2%84%ED%8E%84%EB%A1%9C%EC%8A%A4" title="오릭스 버펄로스">오릭스 블루웨이브</a>에 이적했고 오릭스에 이적할 당시의 기자 회견에서는 “(한신에서)방출되어서 아까웠다고 생각하게 하고 싶다”, “되돌아보고 싶다” 등의 발언을 했다.</p>
<p>오릭스에서는 2003년에 본인으로서는 처음으로 규정 타석을 채우는 것과 동시에 올스타전에 출전을 하는 등 홈런 8개, 타점 46, 도루 7개, 타율 3할 7리의 성적을 남겼다. 다음해인 2004년에는 홈런 9개, 타점 48, 도루 2개, 타율 2할 6푼 9리의 성적을 남긴다. 2004년 시즌 종료 후 분배 드래프트에 의해 계속 오릭스의 선수가 되었지만 2005년에 16경기에만 출전하면서 시즌 종료 후 방출되었다.</p>
<p>2006년에는 <a href="/wiki/KBO_%EB%A6%AC%EA%B7%B8" title="KBO 리그">KBO 리그</a> 팀인 <a href="/wiki/SK_%EC%99%80%EC%9D%B4%EB%B2%88%EC%8A%A4" title="SK 와이번스">SK 와이번스</a>에 입단하여 계약금 500만엔, 연봉 3,000만 엔에 계약을 맺으면서 KBO에서는 <a href="/wiki/%EC%9D%B4%EB%A6%AC%ED%82%A4_%EC%82%AC%ED%86%A0%EC%8B%9C" title="이리키 사토시">이리키 사토시</a>에 이은 세 번째이며 야수로서는 처음으로 일본인 선수(한국명으로의 등록명이 아닌 선수)가 되었다. 그러나 시즌 개막 이후 SK 와이번스의 주전 <a href="/wiki/3%EB%A3%A8%EC%88%98" title="3루수">3루수</a>를 차지하며 성공한 외국인 선수로서 뒤를 밟아가던 도중 같은 해 5월 9일 <a href="/wiki/KIA_%ED%83%80%EC%9D%B4%EA%B1%B0%EC%A6%88" title="KIA 타이거즈">KIA 타이거즈</a>전에서 <a href="/wiki/%EC%9E%A5%EB%AC%B8%EC%84%9D" title="장문석">장문석</a>이 던진 투구에 손가락에 공을 맞아 손가락 골절 때문에 전치 8주의 진단을 받고 방출당했다. 2006년 시즌 종료 후 은퇴를 했다.</p>
<h3><span class="mw-headline" id=".EA.B7.B8_.ED.9B.84">그 후</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%8B%9C%EC%98%A4%ED%83%80%EB%8B%88_%EA%B0%80%EC%A6%88%ED%9E%88%EC%BD%94&amp;action=edit&amp;section=3" title="부분 편집: 그 후">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>2010년, <a href="/wiki/%EB%B2%A0%EC%9D%B4%EC%8A%A4%EB%B3%BC_%EC%B1%8C%EB%A6%B0%EC%A7%80_%EB%A6%AC%EA%B7%B8" title="베이스볼 챌린지 리그">베이스볼 챌린지 리그</a>의 <a class="new" href="/w/index.php?title=%ED%9B%84%EC%BF%A0%EC%9D%B4_%EB%AF%B8%EB%9D%BC%ED%81%B4_%EC%97%98%EB%A6%AC%ED%8E%80%EC%B8%A0&amp;action=edit&amp;redlink=1" title="후쿠이 미라클 엘리펀츠 (없는 문서)">후쿠이 미라클 엘리펀츠</a>의 타격·수비 코치에 취임했으나 개막 전날인 4월 2일에 개인적인 사정을 이유로 돌연히 사임했다. 16일 일본 스포츠전문지 닛칸스포츠에 따르면 효고현 서부경찰서는 전 한신 타이거즈에서 선수였던 시오타니를 사기 혐의로 체포했다. 시오타니는 "음식점의 매출 권리를 양도해주겠다"며 지인의 회사 임원한테 사기를 부리고 현금 550만 엔(한화 약 7400만원)을 횡령한 혐의를 받고 있다. 경찰에 따르면 시오타니는 "사업비와 생활비가 필요했다"고 진술, 혐의를 인정했다.</p>
