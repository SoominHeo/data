<p><b>시카고 파이어</b>(Chicago Fire)는 <a href="/wiki/%EB%A9%94%EC%9D%B4%EC%A0%80_%EB%A6%AC%EA%B7%B8_%EC%82%AC%EC%BB%A4" title="메이저 리그 사커">메이저 리그 사커</a>에 소속된 <a href="/wiki/%EB%AF%B8%EA%B5%AD" title="미국">미국</a>의 프로축구단으로 <a href="/wiki/%EC%9D%BC%EB%A6%AC%EB%85%B8%EC%9D%B4_%EC%A3%BC" title="일리노이 주">일리노이 주</a> <a href="/wiki/%EC%8B%9C%EC%B9%B4%EA%B3%A0" title="시카고">시카고</a>를 연고지로 하고 있다. 클럽 명칭에서 <b>"파이어(Fire)"</b>는 <a href="/wiki/1871%EB%85%84" title="1871년">1871년</a>에 발생한 <a href="/wiki/%EC%8B%9C%EC%B9%B4%EA%B3%A0_%EB%8C%80%ED%99%94%EC%9E%AC" title="시카고 대화재">시카고 대화재</a>와 관련된 <a href="/wiki/%EC%8B%9C%EC%B9%B4%EA%B3%A0" title="시카고">시카고</a>의 지역 연고성을 상징하며, <a href="/wiki/1997%EB%85%84" title="1997년">1997년</a>에 창단되었다.</p>
<p>홈구장은 <a href="/wiki/%EB%8F%84%EC%9A%94%ED%83%80_%ED%8C%8C%ED%81%AC" title="도요타 파크">도요타 파크</a> <a class="mw-redirect" href="/wiki/%EC%B6%95%EA%B5%AC%EC%A0%84%EC%9A%A9%EA%B5%AC%EC%9E%A5" title="축구전용구장">축구전용구장</a>이며 <a href="/wiki/%EC%9D%BC%EB%A6%AC%EB%85%B8%EC%9D%B4_%EC%A3%BC" title="일리노이 주">일리노이 주</a> <a href="/wiki/%EB%B8%8C%EB%A6%AC%EC%A7%80%EB%B7%B0" title="브리지뷰">브리지뷰</a>에 위치해 있다. 2군팀으로 시카고 파이어 프리미어를 두고 있다.</p>

<p><b>시카고 파이어 프리미어</b>는 1998년 <b>시카고 소커즈</b>라는 이름으로 창단된 <strong class="selflink">시카고 파이어</strong>의 2군 팀이다. 미국 <a href="/wiki/%EC%9D%BC%EB%A6%AC%EB%85%B8%EC%9D%B4_%EC%A3%BC" title="일리노이 주">일리노이 주</a> <a href="/wiki/%EC%8B%9C%EC%B9%B4%EA%B3%A0" title="시카고">시카고</a>를 연고지로 하고 있으며 홈구장은 <a href="/wiki/%EB%8F%84%EC%9A%94%ED%83%80_%ED%8C%8C%ED%81%AC" title="도요타 파크">도요타 파크</a>를 사용한다. 현재 <a class="mw-redirect" href="/wiki/USL_%ED%94%84%EB%A6%AC%EB%AF%B8%EC%96%B4_%EB%94%94%EB%B2%A8%EB%A1%9C%ED%94%84%EB%A8%BC%ED%8A%B8_%EB%A6%AC%EA%B7%B8" title="USL 프리미어 디벨로프먼트 리그">USL 프리미어 디벨로프먼트 리그</a>에 참가하고 있으며 2009년 시즌에는 <a class="new" href="/w/index.php?title=%EB%B2%A4%EC%B6%94%EB%9D%BC_%EC%B9%B4%EC%9A%B4%ED%8B%B0_%ED%93%A8%EC%A0%84&amp;action=edit&amp;redlink=1" title="벤추라 카운티 퓨전 (없는 문서)">벤추라 카운티 퓨전</a>에게 패해 준우승을 차지하였다.</p>
