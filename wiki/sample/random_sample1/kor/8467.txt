<p><b>시코쓰토야 국립공원</b>(支笏洞爺国立公園)은 <a href="/wiki/%EC%9D%BC%EB%B3%B8" title="일본">일본</a> <a href="/wiki/%ED%99%8B%EC%B9%B4%EC%9D%B4%EB%8F%84" title="홋카이도">홋카이도</a> 서부에 있는 국립공원이다. 호수와 화산, 온천이 많은 것이 특징이다. <a href="/wiki/1949%EB%85%84" title="1949년">1949년</a> <a href="/wiki/5%EC%9B%94_16%EC%9D%BC" title="5월 16일">5월 16일</a>에 국립공원으로 지정되었다. 면적은 994.73km²이다.</p>
<p>크게 5개의 지역으로 나눌 수 있다.</p>
<ul>
<li>요테이 산 : <a href="/wiki/%EC%9A%94%ED%85%8C%EC%9D%B4_%EC%82%B0" title="요테이 산">요테이 산</a></li>
<li>도야 호 : <a href="/wiki/%EB%8F%84%EC%95%BC_%ED%98%B8" title="도야 호">도야 호</a>, <a href="/wiki/%EC%9A%B0%EC%8A%A4_%EC%82%B0" title="우스 산">우스 산</a>, <a href="/wiki/%EC%87%BC%EC%99%80%EC%8B%A0_%EC%82%B0" title="쇼와신 산">쇼와신 산</a></li>
<li>노보리베쓰 : 노보리베쓰 온천, 오로후레 고개, <a class="new" href="/w/index.php?title=%EA%B5%BF%ED%83%80%EB%9D%BC_%ED%98%B8&amp;action=edit&amp;redlink=1" title="굿타라 호 (없는 문서)">굿타라 호</a></li>
<li>시코쓰 호 : <a href="/wiki/%EC%8B%9C%EC%BD%94%EC%93%B0_%ED%98%B8" title="시코쓰 호">시코쓰 호</a>, <a href="/wiki/%EB%8B%A4%EB%A3%A8%EB%A7%88%EC%97%90_%EC%82%B0" title="다루마에 산">다루마에 산</a>, <a class="new" href="/w/index.php?title=%ED%9B%97%ED%91%B8%EC%8B%9C_%EC%82%B0&amp;action=edit&amp;redlink=1" title="훗푸시 산 (없는 문서)">훗푸시 산</a>, <a class="new" href="/w/index.php?title=%EC%97%90%EB%8B%88%EC%99%80_%EC%82%B0&amp;action=edit&amp;redlink=1" title="에니와 산 (없는 문서)">에니와 산</a></li>
<li>조잔 계곡 : 조잔 계곡, 도요히라 협곡, 나카야마 고개</li>
</ul>
