<h1 class="firstHeading" id="firstHeading" lang="ko">시큐어 카피</h1>
<div class="mw-body-content" id="bodyContent">
<div id="siteSub">위키백과, 우리 모두의 백과사전.</div>
<div id="contentSub"></div>
<div class="mw-jump" id="jump-to-nav">
					이동:					<a href="#mw-head">둘러보기</a>, 					<a href="#p-search">검색</a>
</div>
<div class="mw-content-ltr" dir="ltr" id="mw-content-text" lang="ko"><script>function mfTempOpenSection(id){var block=document.getElementById("mf-section-"+id);block.className+=" open-block";block.previousSibling.className+=" open-block";}</script><p><b>시큐어 카피</b>(Secure copy) 또는 <b>SCP</b>는 로컬 호스트와 원격 <a href="/wiki/%EC%84%9C%EB%B2%84" title="서버">호스트</a> 간 또는 두 개의 원격 호스트 간에 <a href="/wiki/%EC%BB%B4%ED%93%A8%ED%84%B0_%ED%8C%8C%EC%9D%BC" title="컴퓨터 파일">컴퓨터 파일</a>을 안전하게 전송하는 수단이다. <a href="/wiki/%EC%8B%9C%ED%81%90%EC%96%B4_%EC%85%B8" title="시큐어 셸">시큐어 셸</a>(SSH) 프로토콜 기반이다. SCP라는 용어는 SCP 프로토콜 또는 SCP 프로그램 중 하나를 가리킬 수 있다.</p>
<h2><span class="mw-headline" id="SCP_.ED.94.84.EB.A1.9C.ED.86.A0.EC.BD.9C">SCP 프로토콜</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%8B%9C%ED%81%90%EC%96%B4_%EC%B9%B4%ED%94%BC&amp;action=edit&amp;section=1" title="부분 편집: SCP 프로토콜">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p><b>SCP 프로토콜</b>은 <a href="/wiki/BSD" title="BSD">BSD</a> <a href="/wiki/Rlogin" title="Rlogin">RCP 프로토콜</a> 기반의 <a href="/wiki/%ED%86%B5%EC%8B%A0_%ED%94%84%EB%A1%9C%ED%86%A0%EC%BD%9C" title="통신 프로토콜">통신 프로토콜</a>로서<sup class="reference" id="cite_ref-1"><a href="#cite_note-1">[1]</a></sup>, 네트워크 상의 호스트 간 <a class="new" href="/w/index.php?title=%ED%8C%8C%EC%9D%BC_%EC%A0%84%EC%86%A1&amp;action=edit&amp;redlink=1" title="파일 전송 (없는 문서)">파일 전송</a>을 지원한다.</p>
<h2><span class="mw-headline" id="SCP_.ED.94.84.EB.A1.9C.EA.B7.B8.EB.9E.A8">SCP 프로그램</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%8B%9C%ED%81%90%EC%96%B4_%EC%B9%B4%ED%94%BC&amp;action=edit&amp;section=2" title="부분 편집: SCP 프로그램">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p><b>SCP 프로그램</b>은 서버 데몬이나 클라이언트로서 SCP 프로토콜을 구현하는 소프트웨어 도구로서, 안전한 복사를 수행하는 프로그램이다. SCP 서버 프로그램은 일반적으로 SCP 클라이언트와 동일하다.</p>
<p>일반적으로 <tt>scp</tt> 프로그램<sup class="reference" id="cite_ref-2"><a href="#cite_note-2">[2]</a></sup>의 문법은 <a href="/wiki/Cp_(%EC%9C%A0%EB%8B%89%EC%8A%A4)" title="Cp (유닉스)">cp</a>의 문법과 비슷하다.</p>
<p>파일을 호스트로 복사:</p>
<pre>
scp 원본파일 사용자@호스트주소:디렉터리/대상파일
</pre>
<p>파일을 호스트로부터 복사:</p>
<pre>
scp 사용자@호스트주소:디렉터리/원본파일 대상파일
scp -r 사용자@호스트주소:디렉터리/원본폴더 대상폴더
</pre>
<h2><span class="mw-headline" id=".EA.B0.81.EC.A3.BC">각주</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%8B%9C%ED%81%90%EC%96%B4_%EC%B9%B4%ED%94%BC&amp;action=edit&amp;section=3" title="부분 편집: 각주">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
