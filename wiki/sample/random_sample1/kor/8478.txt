<h1 class="firstHeading" id="firstHeading" lang="ko">시흥군</h1>
<div class="mw-body-content" id="bodyContent">
<div id="siteSub">위키백과, 우리 모두의 백과사전.</div>
<div id="contentSub"></div>
<div class="mw-jump" id="jump-to-nav">
					이동:					<a href="#mw-head">둘러보기</a>, 					<a href="#p-search">검색</a>
</div>
<div class="mw-content-ltr" dir="ltr" id="mw-content-text" lang="ko"><script>function mfTempOpenSection(id){var block=document.getElementById("mf-section-"+id);block.className+=" open-block";block.previousSibling.className+=" open-block";}</script><p><b>시흥군</b>(始興郡)은 <a href="/wiki/1895%EB%85%84" title="1895년">1895년</a>부터 <a href="/wiki/1988%EB%85%84" title="1988년">1988년</a>까지 존재했던 <a href="/wiki/%EA%B2%BD%EA%B8%B0%EB%8F%84" title="경기도">경기도</a>의 <a href="/wiki/%ED%96%89%EC%A0%95_%EA%B5%AC%EC%97%AD" title="행정 구역">행정 구역</a>이다. 원래의 시흥군은 현재의 <a href="/wiki/%EC%84%9C%EC%9A%B8%ED%8A%B9%EB%B3%84%EC%8B%9C" title="서울특별시">서울특별시</a> <a href="/wiki/%EA%B8%88%EC%B2%9C%EA%B5%AC" title="금천구">금천구</a> <a href="/wiki/%EC%8B%9C%ED%9D%A5%EB%8F%99_(%EC%84%9C%EC%9A%B8)" title="시흥동 (서울)">시흥동</a>에 군청을 두고, 서울특별시 <a href="/wiki/%EA%B8%88%EC%B2%9C%EA%B5%AC" title="금천구">금천구</a>를 중심으로 <a href="/wiki/%EC%98%81%EB%93%B1%ED%8F%AC%EA%B5%AC" title="영등포구">영등포구</a>(<a class="mw-redirect" href="/wiki/%EC%96%91%ED%99%94%EB%8F%99" title="양화동">양화동</a>, <a class="mw-redirect" href="/wiki/%EC%97%AC%EC%9D%98%EB%8F%84%EB%8F%99" title="여의도동">여의도동</a> 제외) · <a href="/wiki/%EA%B4%80%EC%95%85%EA%B5%AC" title="관악구">관악구</a>(<a class="mw-redirect" href="/wiki/%EB%82%A8%ED%98%84%EB%8F%99_(%EA%B4%80%EC%95%85%EA%B5%AC)" title="남현동 (관악구)">남현동</a> 제외) · <a href="/wiki/%EB%8F%99%EC%9E%91%EA%B5%AC" title="동작구">동작구</a> 일부(<a href="/wiki/%EC%8B%A0%EB%8C%80%EB%B0%A9%EB%8F%99" title="신대방동">신대방동</a>, <a class="mw-disambig" href="/wiki/%EB%8C%80%EB%B0%A9%EB%8F%99" title="대방동">대방동</a>, <a class="mw-disambig" href="/wiki/%EC%83%81%EB%8F%84%EB%8F%99" title="상도동">상도동</a>) · <a href="/wiki/%EA%B5%AC%EB%A1%9C%EA%B5%AC" title="구로구">구로구</a> 동부(<a href="/wiki/%EC%95%88%EC%96%91%EC%B2%9C" title="안양천">안양천</a> 동쪽)와 경기도 <a href="/wiki/%EA%B4%91%EB%AA%85%EC%8B%9C" title="광명시">광명시</a>(옥길동 제외) · <a href="/wiki/%EC%95%88%EC%96%91%EC%8B%9C" title="안양시">안양시</a> 일부(석수동, 박달동)를 관할구역으로 하였다.</p>
<p><a href="/wiki/1914%EB%85%84" title="1914년">1914년</a> <a href="/wiki/4%EC%9B%94_1%EC%9D%BC" title="4월 1일">4월 1일</a>, 일제의 행정구역 통합으로 <a href="/wiki/%EA%B3%BC%EC%B2%9C%EA%B5%B0" title="과천군">과천군</a>(현 <a href="/wiki/%EA%B3%BC%EC%B2%9C%EC%8B%9C" title="과천시">과천시</a>·<a href="/wiki/%EC%95%88%EC%96%91%EC%8B%9C" title="안양시">안양시</a>·<a href="/wiki/%EA%B5%B0%ED%8F%AC%EC%8B%9C" title="군포시">군포시</a> 및 <a class="mw-redirect" href="/wiki/%EC%84%9C%EC%9A%B8" title="서울">서울</a> <a href="/wiki/%EB%8F%99%EC%9E%91%EA%B5%AC" title="동작구">동작구</a>·<a href="/wiki/%EC%84%9C%EC%B4%88%EA%B5%AC" title="서초구">서초구</a>)과 <a href="/wiki/%EC%95%88%EC%82%B0%EA%B5%B0" title="안산군">안산군</a>(현 <a href="/wiki/%EC%95%88%EC%82%B0%EC%8B%9C" title="안산시">안산시</a>와 <a href="/wiki/%EC%8B%9C%ED%9D%A5%EC%8B%9C" title="시흥시">시흥시</a> 남부)이 시흥군으로 통합되었다. 한편 지금의 시흥시 북부 지역을 이루는 인천부 신현면·전반면·황등천면은 <a href="/wiki/%EB%B6%80%EC%B2%9C%EA%B5%B0" title="부천군">부천군</a>으로 통합되어 부천군 소래면이 되었다.</p>
<p>시흥군은 <a href="/wiki/1989%EB%85%84" title="1989년">1989년</a> <a href="/wiki/1%EC%9B%94_1%EC%9D%BC" title="1월 1일">1월 1일</a> <a href="/wiki/%EC%8B%9C%ED%9D%A5%EC%8B%9C" title="시흥시">시흥시</a>, <a href="/wiki/%EA%B5%B0%ED%8F%AC%EC%8B%9C" title="군포시">군포시</a>, <a href="/wiki/%EC%9D%98%EC%99%95%EC%8B%9C" title="의왕시">의왕시</a> 등 3개시로 분리·승격하면서 폐지되었다. 현재의 <a href="/wiki/%EC%8B%9C%ED%9D%A5%EC%8B%9C" title="시흥시">시흥시</a> 지역은 구한말까지 원래 <a href="/wiki/%EC%95%88%EC%82%B0%EA%B5%B0" title="안산군">안산군</a>과 <a class="mw-disambig" href="/wiki/%EC%9D%B8%EC%B2%9C%EB%B6%80" title="인천부">인천부</a>(일제강점기 이후 <a href="/wiki/%EB%B6%80%EC%B2%9C%EA%B5%B0" title="부천군">부천군</a>)에 속했던 지역으로 결과적으로 시흥이라는 명칭만 이어질 뿐 <a class="mw-redirect" href="/wiki/%EA%B5%AC%ED%95%9C%EB%A7%90" title="구한말">대한제국 시기</a>의 시흥군 지역과는 다른 곳이 되었다.</p>

<h2><span class="mw-headline" id=".EC.9C.A0.EB.9E.98">유래</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%8B%9C%ED%9D%A5%EA%B5%B0&amp;action=edit&amp;section=1" title="부분 편집: 유래">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>시흥의 첫이름은 잉벌노(仍伐奴)이고 나중에 곡양(穀壤)이 되었는데, 이 둘은 사실상 같은 표기이다. 잉벌(仍伐)과 곡(穀)은 고대에는 다 같이 "늠"으로 읽히고, 노(奴)와 양(壤)은 고대에는 "나"나 "내"로 읽혔기 때문에 순우리말로는 "늠내"가 된다. 이 늠내는 "뻗어나가는 땅"이라는 의미도 있어서 현재의 "시흥"이라는 이름과도 상통한다.<sup class="reference" id="cite_ref-1"><a href="#cite_note-1">[1]</a></sup></p>

title="관악구">관악구</a>, <a href="/wiki/%EB%8F%99%EC%9E%91%EA%B5%AC" title="동작구">동작구</a> 일부, <a href="/wiki/%EA%B8%88%EC%B2%9C%EA%B5%AC" title="금천구">금천구</a>, <a href="/wiki/%EA%B4%91%EB%AA%85%EC%8B%9C" title="광명시">광명시</a> 및 <a href="/wiki/%EC%95%88%EC%96%91%EC%8B%9C" title="안양시">안양시</a> 일부 지역은 <a href="/wiki/%EA%B3%A0%EA%B5%AC%EB%A0%A4" title="고구려">고구려</a> 때 <b>잉벌노현</b>(仍伐奴縣)이었던 것<sup class="reference" id="cite_ref-9"><a href="#cite_note-9">[9]</a></sup>을 <a href="/wiki/%EC%8B%A0%EB%9D%BC" title="신라">신라</a> <a class="mw-redirect" href="/wiki/%EC%8B%A0%EB%9D%BC_%EA%B2%BD%EB%8D%95%EC%99%95" title="신라 경덕왕">경덕왕</a> 때 <b>곡양</b>(穀壤)<sup class="reference" id="cite_ref-10"><a href="#cite_note-10">[10]</a></sup>으로 고쳐서 <a class="mw-redirect" href="/wiki/%EC%9C%A8%EC%A7%84%EA%B5%B0" title="율진군">율진군</a>(栗津郡) 속현이 되었다. <a href="/wiki/%EA%B3%A0%EB%A0%A4" title="고려">고려</a> 초에 <b>금주</b>(衿州, '衿'을 '黔'이라고 쓴 기록도 있다)라고 고쳤고, <a href="/wiki/%EA%B3%A0%EB%A0%A4_%EC%84%B1%EC%A2%85" title="고려 성종">성종</a>이 단련사(團練使)를 설치하였는데 <a href="/wiki/%EA%B3%A0%EB%A0%A4_%EB%AA%A9%EC%A2%85" title="고려 목종">목종</a>이 폐지하였다. <a href="/wiki/%EA%B3%A0%EB%A0%A4_%ED%98%84%EC%A2%85" title="고려 현종">현종</a> 9년(1017년)에 수주(樹州)에 예속되었고, <a href="/wiki/%EA%B3%A0%EB%A0%A4_%EB%AA%85%EC%A2%85" title="고려 명종">명종</a>이 비로소 감무를 설치하였다. <a href="/wiki/%EC%A1%B0%EC%84%A0" title="조선">조선</a> 시대에는 <a href="/wiki/%EC%A1%B0%EC%84%A0_%ED%83%9C%EC%A2%85" title="조선 태종">태종</a> 14년(1413년)에 과천(오늘날의 <a href="/wiki/%EB%8F%99%EC%9E%91%EA%B5%AC" title="동작구">동작구</a>, <a href="/wiki/%EC%84%9C%EC%B4%88%EA%B5%AC" title="서초구">서초구</a>, <a href="/wiki/%EC%95%88%EC%96%91%EC%8B%9C" title="안양시">안양시</a>, <a href="/wiki/%EA%B3%BC%EC%B2%9C%EC%8B%9C" title="과천시">과천시</a>, <a href="/wiki/%EA%B5%B0%ED%8F%AC%EC%8B%9C" title="군포시">군포시</a> 지역)과 병합하여 금과현(衿果縣)을 만들었다가 두어 달 만에 혁파하고, 또 양천(오늘날의 <a href="/wiki/%EA%B0%95%EC%84%9C%EA%B5%AC_(%EC%84%9C%EC%9A%B8%ED%8A%B9%EB%B3%84%EC%8B%9C)" title="강서구 (서울특별시)">강서구</a>, <a href="/wiki/%EC%96%91%EC%B2%9C%EA%B5%AC" title="양천구">양천구</a> 지역)과 병합하여 금양현(衿陽縣)으로 만들었는데 한 해 만에 혁파하였다. 태종 16년(1415년)에 <b>금천현</b>(衿川縣)으로 고쳐서 현감을 두었고, <a href="/wiki/%EC%A1%B0%EC%84%A0_%EC%84%B8%EC%A1%B0" title="조선 세조">세조</a> 때에 과천과 병합했다가 얼마 못 가서 복구하였다.<sup class="reference" id="cite_ref-11"><a href="#cite_note-11">[11]</a></sup> 이후에는 금천(衿川)이라고 하였는데, 금천현은 현내면, 동면, 서면, 남면, 상북면, 하북면으로 이루어져 있었고, 동면에 상도리, 봉천리, 신림리, 난곡리, 문교리 등이, 상북면에 당산리, 양평리, 선유봉리, 하북면에 방학호리, 고사리 등이 속해 있었다.<sup class="reference" id="cite_ref-12"><a href="#cite_note-12">[12]</a></sup> 참고로 현내면은 지금의 금천구, 상북면과 하북면은 지금의 영등포구, 동면은 지금의 관악구와 동작구 일부, 남면은 지금의 광명시 서부, 서면은 지금의 광명시 동부에 해당한다. <a href="/wiki/1795%EB%85%84" title="1795년">1795년</a>(정조 19)에 금천현이 <b>시흥현</b>(始興縣)으로 개칭되었다. <a href="/wiki/1895%EB%85%84" title="1895년">1895년</a>(고종 32년) 윤 5월 1일, <a class="mw-redirect" href="/wiki/23%EB%B6%80%EC%A0%9C" title="23부제">23부제</a>의 실시로 시흥현이 <b>시흥군</b>으로 개편되어 <a class="mw-disambig" href="/wiki/%EC%9D%B8%EC%B2%9C%EB%B6%80" title="인천부">인천부</a>에 속하였고,<sup class="reference" id="cite_ref-13"><a href="#cite_note-13">[13]</a></sup> 이듬해 <a class="mw-redirect" href="/wiki/13%EB%8F%84%EC%A0%9C" title="13도제">13도제</a>의 실시로 <a href="/wiki/%EB%8F%84_(%ED%96%89%EC%A0%95_%EA%B5%AC%EC%97%AD)" title="도 (행정 구역)">도</a>(道)가 부활해 다시 경기도에 속하게 되었다.</p>
<p>한편 오늘날의 <a href="/wiki/%EC%8B%9C%ED%9D%A5%EC%8B%9C" title="시흥시">시흥시</a> 남부 지역은 현재의 <a href="/wiki/%EC%95%88%EC%82%B0%EC%8B%9C" title="안산시">안산시</a> 지역과 마찬가지로 삼국시대에는 처음 장항구현(獐項口縣)현으로 명명되었다가 통일신라 시기인 757년에 장구군(獐口郡)으로 개칭되었고, 940년에 <a href="/wiki/%EC%95%88%EC%82%B0%EA%B5%B0" title="안산군">안산군</a>(安山郡)이 되었다. 지금의 <a href="/wiki/%EC%8B%9C%ED%9D%A5%EC%8B%9C" title="시흥시">시흥시</a> 북부 지역은 현재의 <a href="/wiki/%EC%9D%B8%EC%B2%9C%EA%B4%91%EC%97%AD%EC%8B%9C" title="인천광역시">인천광역시</a> 남부 지역과 마찬가지로 삼국시대에는 처음 미추홀(彌鄒忽)로 명명되었다가 매소홀현(買召忽縣)이 되었고 통일신라 시기인 757년에 소성현(邵城縣)으로 개칭된 후 율진군(栗津郡; 지금의 <a href="/wiki/%EA%B3%BC%EC%B2%9C%EC%8B%9C" title="과천시">과천시</a>)의 영현(領縣)이 되었다가 <a href="/wiki/1018%EB%85%84" title="1018년">1018년</a>에 수주(樹州; 지금의 <a href="/wiki/%EB%B6%80%ED%8F%89%EA%B5%AC" title="부평구">부평구</a>)의 속현이 되었다. <a href="/wiki/1105%EB%85%84" title="1105년">1105년</a> 숙종의 어머니인 <a href="/wiki/%EC%9D%B8%EC%98%88%EC%99%95%ED%9B%84" title="인예왕후">인예왕후</a> 이 씨의 내향(內鄕)이라 하여 경원부(慶源府)로 승격되어 수주(樹州)에서 분리되었다. <a href="/wiki/%EA%B3%A0%EB%A0%A4_%EC%9D%B8%EC%A2%85" title="고려 인종">고려 인종</a>때에 인종 왕해의 어머니인 <a href="/wiki/%EC%88%9C%EB%8D%95%EC%99%95%ED%9B%84" title="순덕왕후">순덕왕후</a> 이 씨의 내향이라 하여 인주(仁州)가 되었다가 경원부와 인주군으로 차례대로 이름이 변경되었다. 1413년 행정구역 개편으로 인주군이 <a class="mw-redirect" href="/wiki/%EC%9D%B8%EC%B2%9C%EA%B5%B0" title="인천군">인천군</a>(仁川郡)으로 개명되었고, 1460년에 인천군이 인천도호부(仁川都護府)로 승격하였다.</p>
<p>현재 <a href="/wiki/%EC%84%9C%EC%9A%B8%ED%8A%B9%EB%B3%84%EC%8B%9C" title="서울특별시">서울특별시</a> <b><a href="/wiki/%EA%B8%88%EC%B2%9C%EA%B5%AC" title="금천구">금천구</a></b> 일대는 <a href="/wiki/1963%EB%85%84" title="1963년">1963년</a>까지 <a href="/wiki/%EA%B2%BD%EA%B8%B0%EB%8F%84" title="경기도">경기도</a> 시흥군 동면에 속했던 지역으로, <a href="/wiki/%EC%A1%B0%EC%84%A0" title="조선">조선</a>시대에는 관아가 위치해 금천현의 중심지였고, <a href="/wiki/1910%EB%85%84" title="1910년">1910년</a>까지 시흥군청이 위치했다. <a href="/wiki/1911%EB%85%84" title="1911년">1911년</a>, <b><a href="/wiki/%EC%98%81%EB%93%B1%ED%8F%AC%EA%B5%AC" title="영등포구">영등포</a></b>로 군청이 옮겨지면서 영등포가 시흥군의 중심지가 되었다. 시흥군의 중심지 기능을 잃은 시흥리 지역은 <a href="/wiki/1963%EB%85%84" title="1963년">1963년</a>에 <a href="/wiki/%EC%84%9C%EC%9A%B8%ED%8A%B9%EB%B3%84%EC%8B%9C" title="서울특별시">서울특별시</a>에 편입되었으며, 과거 이 곳이 시흥군의 중심지였다는 사실은 <a href="/wiki/%EC%8B%9C%ED%9D%A5%EB%8F%99_(%EC%84%9C%EC%9A%B8)" title="시흥동 (서울)">시흥동</a>이라는 지명에 남아 있다. 반면, 현재 <a href="/wiki/%EC%8B%9C%ED%9D%A5%EC%8B%9C" title="시흥시">시흥시</a>의 행정구역명은 서울특별시 <a href="/wiki/%EA%B8%88%EC%B2%9C%EA%B5%AC" title="금천구">금천구</a> 지역에서 따온 '시흥'이라는 지명에서 유래했지만 원래의 시흥(현재의 영등포, 구로, 관악, 금천, 광명)과는 행정구역 및 생활권이 다른 지역이다.<sup class="reference" id="cite_ref-14"><a href="#cite_note-14">[14]</a></sup><sup class="reference" id="cite_ref-15"><a href="#cite_note-15">[15]</a></sup></p>
<p><a href="/wiki/1936%EB%85%84" title="1936년">1936년</a>, 영등포읍이 <a href="/wiki/%EA%B2%BD%EC%84%B1%EB%B6%80" title="경성부">경성부</a>에 편입되면서 <a href="/wiki/%EC%95%88%EC%96%91%EC%8B%9C" title="안양시">안양</a>이 시흥군의 중심지가 되었고, <a href="/wiki/1947%EB%85%84" title="1947년">1947년</a>에는 <b>안양</b>으로 군청이 이전하였다. <a href="/wiki/1973%EB%85%84" title="1973년">1973년</a> <a href="/wiki/7%EC%9B%94_1%EC%9D%BC" title="7월 1일">7월 1일</a> 안양읍이 <a href="/wiki/%EC%95%88%EC%96%91%EC%8B%9C" title="안양시">안양시</a>로 승격하였고, <a href="/wiki/%EB%B6%80%EC%B2%9C%EA%B5%B0" title="부천군">부천군</a>이 폐지되면서 부천군 <a href="/wiki/%EC%86%8C%EB%9E%98%EC%9D%8D" title="소래읍">소래면</a>이 시흥군으로 편입되었다.</p>
<p>1970년대부터 1980대까지 시흥군 지역은 <a href="/wiki/%EC%84%9C%EC%9A%B8%ED%8A%B9%EB%B3%84%EC%8B%9C" title="서울특별시">서울</a>, <a href="/wiki/%EC%9D%B8%EC%B2%9C%EA%B4%91%EC%97%AD%EC%8B%9C" title="인천광역시">인천</a>, <a href="/wiki/%EB%B6%80%EC%B2%9C%EC%8B%9C" title="부천시">부천</a>, <a href="/wiki/%EC%88%98%EC%9B%90%EC%8B%9C" title="수원시">수원</a> 등 인근 도시와 연결되는 도로망이 발달한 전형적 전원형 도시로 발전하면서 거듭 행정구역이 개편되었는데, 급격한 산업화와 인구 유입으로 <a href="/wiki/%EC%95%88%EC%96%91%EC%8B%9C" title="안양시">안양시</a>를 시작으로 <a href="/wiki/%EA%B4%91%EB%AA%85%EC%8B%9C" title="광명시">광명시</a>(<a href="/wiki/1981%EB%85%84" title="1981년">1981년</a>), <a href="/wiki/%EC%95%88%EC%82%B0%EC%8B%9C" title="안산시">안산시</a>(<a href="/wiki/1986%EB%85%84" title="1986년">1986년</a>), <a href="/wiki/%EA%B3%BC%EC%B2%9C%EC%8B%9C" title="과천시">과천시</a>(<a href="/wiki/1986%EB%85%84" title="1986년">1986년</a>)가 시로 승격해 분리되었다. 1980년 소래면이 읍으로 승격되어, 시흥군은 <a href="/wiki/%EC%95%88%EC%96%91%EC%8B%9C" title="안양시">안양시</a>와 연계되어 발전한 동부(<a class="mw-redirect" href="/wiki/%EA%B5%B0%ED%8F%AC%EC%9D%8D" title="군포읍">군포읍</a>·<a class="mw-redirect" href="/wiki/%EC%9D%98%EC%99%95%EC%9D%8D" title="의왕읍">의왕읍</a>), 수인산업도로 등으로 도로교통이 발달한 서북부(<a href="/wiki/%EC%86%8C%EB%9E%98%EC%9D%8D" title="소래읍">소래읍</a>), <a class="mw-redirect" href="/wiki/%EB%B0%98%EC%9B%94%EA%B3%B5%EB%8B%A8" title="반월공단">반월공단</a> 등 계획도시로 건설된 서남부(현 <a href="/wiki/%EC%95%88%EC%82%B0%EC%8B%9C" title="안산시">안산시</a>)가 각각 독자적인 도시로 성장하였다.</p>
<p><a href="/wiki/1986%EB%85%84" title="1986년">1986년</a>에 <a href="/wiki/%EC%95%88%EC%82%B0%EC%8B%9C" title="안산시">안산시</a>가 설치된 이후, 시흥군을 폐지하고 시로 승격시키는 방안으로는 <a class="mw-redirect" href="/wiki/%EC%9D%98%EC%99%95%EC%9D%8D" title="의왕읍">의왕읍</a>의 일부(내손·포일·청계·학의리)를 <a href="/wiki/%EC%95%88%EC%96%91%EC%8B%9C" title="안양시">안양시</a>에, 수암면의 일부(수암·장상·장하·화정리)와 군자면의 일부(정왕·죽율·군자·거모리)를 <a href="/wiki/%EC%95%88%EC%82%B0%EC%8B%9C" title="안산시">안산시</a>에 편입시키는 것을 전제로 <a class="mw-redirect" href="/wiki/%EA%B5%B0%ED%8F%AC%EC%9D%8D" title="군포읍">군포읍</a>과 <a class="mw-redirect" href="/wiki/%EC%9D%98%EC%99%95%EC%9D%8D" title="의왕읍">의왕읍</a>을 합쳐 <a href="/wiki/%EC%8B%9C%ED%9D%A5%EC%8B%9C" title="시흥시">시흥시</a>를, <a href="/wiki/%EC%86%8C%EB%9E%98%EC%9D%8D" title="소래읍">소래읍</a>에 수암면과 군자면을 합쳐 <a href="/wiki/%EC%86%8C%EB%9E%98%EC%9D%8D" title="소래읍">소래시</a>를 설치하는 <b>2개(시흥·소래)시 분리안</b>과 군포읍, 의왕읍을 각각 <a href="/wiki/%EA%B5%B0%ED%8F%AC%EC%8B%9C" title="군포시">군포시</a>, <a href="/wiki/%EC%9D%98%EC%99%95%EC%8B%9C" title="의왕시">의왕시</a>로 승격시키고 나머지 지역(<a href="/wiki/%EC%86%8C%EB%9E%98%EC%9D%8D" title="소래읍">소래읍</a>·수암면·군자면)을 <a href="/wiki/%EC%8B%9C%ED%9D%A5%EC%8B%9C" title="시흥시">시흥시</a>로 승격시키는 <b>3개(시흥·군포·의왕)시 분리안</b>이 논의되었는데, 행정관료들이 선호한 3개시 분리·승격안이 채택되었다.<sup class="reference" id="cite_ref-16"><a href="#cite_note-16">[16]</a></sup></p>
<p><a href="/wiki/1989%EB%85%84" title="1989년">1989년</a> <a href="/wiki/1%EC%9B%94_1%EC%9D%BC" title="1월 1일">1월 1일</a>에 시흥군이 폐지되고, <a href="/wiki/%EC%8B%9C%ED%9D%A5%EC%8B%9C" title="시흥시">시흥시</a>(<a href="/wiki/%EC%86%8C%EB%9E%98%EC%9D%8D" title="소래읍">소래읍</a>·군자면·수암면 일원), <a href="/wiki/%EA%B5%B0%ED%8F%AC%EC%8B%9C" title="군포시">군포시</a>(군포읍 일원) 및 <a href="/wiki/%EC%9D%98%EC%99%95%EC%8B%9C" title="의왕시">의왕시</a>(의왕읍 일원)가 설치되었다. 현재의 시흥시 북부(옛 <a href="/wiki/%EC%86%8C%EB%9E%98%EC%9D%8D" title="소래읍">소래읍</a>)는 <a href="/wiki/1973%EB%85%84" title="1973년">1973년</a> <a href="/wiki/7%EC%9B%94_1%EC%9D%BC" title="7월 1일">7월 1일</a>에 <a href="/wiki/%EB%B6%80%EC%B2%9C%EA%B5%B0" title="부천군">부천군</a>이 폐지되면서 시흥군에 편입되었고 남부(옛 군자면·수암면)는 <a href="/wiki/%EC%95%88%EC%82%B0%EC%8B%9C" title="안산시">안산시</a>와 밀착해 성장했기 때문에 시 승격 당시에 두 지역은 교통과 생활권이 사실상 분리되어 있었고,<sup class="reference" id="cite_ref-17"><a href="#cite_note-17">[17]</a></sup> 그러한 영향은 현재까지도 남아 있다.</p>

<p>※ 1 : 구한말의 광주군과 인천부 지역은 1914년부터 1989년까지의 기간 동안 시흥군에 속한 적 있는 지역만 추려서 위 도표에 정리하였음. 나머지 광주군 지역은 현재 서울특별시 강동권(강남구·송파구·강동구), 경기도 광주시·성남시·하남시이며, 나머지 인천부 지역은 현재 인천광역시 남부임.</p>
<p>※ 2 : 구한말의 안산군 지역 중에서 1914년 부군면 통폐합 당시에 수원군(이후 화성군) 반월면으로 이관된 지역은 위 도표에서 제외하였음. 해당 지역은 시흥군에 속한 적이 없으며, 후일에 안산시 및 군포시에 편입됨.</p>
<h2><span class="mw-headline" id=".EA.B5.90.ED.86.B5">교통</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%8B%9C%ED%9D%A5%EA%B5%B0&amp;action=edit&amp;section=6" title="부분 편집: 교통">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p><a href="/wiki/1899%EB%85%84" title="1899년">1899년</a>, <a href="/wiki/%EA%B3%BC%EC%B2%9C%EA%B5%B0" title="과천군">과천군</a> 하북면 노량진리에 지어진 <a href="/wiki/%EB%85%B8%EB%9F%89%EC%A7%84%EC%97%AD" title="노량진역">노량진역</a>에서 <a href="/wiki/%EC%A0%9C%EB%AC%BC%ED%8F%AC%EC%97%AD" title="제물포역">제물포역</a>을 잇는 <a href="/wiki/%EA%B2%BD%EC%9D%B8%EC%84%A0" title="경인선">경인선</a>이 개통되었다. 이 때, 노량진역 이외에도 <a href="/wiki/%EC%98%81%EB%93%B1%ED%8F%AC%EC%97%AD" title="영등포역">영등포역</a>이 영업을 개시하였다. <a href="/wiki/1901%EB%85%84" title="1901년">1901년</a> <a href="/wiki/8%EC%9B%94_21%EC%9D%BC" title="8월 21일">8월 21일</a>, <a href="/wiki/%EC%98%81%EB%93%B1%ED%8F%AC%EC%97%AD" title="영등포역">영등포역</a>에서 <a href="/wiki/%EA%B2%BD%EB%B6%80%EC%84%A0" title="경부선">경부선</a> 착공식이 거행되었고 <a href="/wiki/1905%EB%85%84" title="1905년">1905년</a> 개통되었는데, 경부선은 시흥군의 <a href="/wiki/%EA%B8%88%EC%B2%9C%EA%B5%AC%EC%B2%AD%EC%97%AD" title="금천구청역">시흥역</a>, <a href="/wiki/%EC%95%88%EC%96%91%EC%97%AD" title="안양역">안양역</a>, <a href="/wiki/%EA%B5%B0%ED%8F%AC%EC%97%AD" title="군포역">군포역</a>을 거쳐 <a href="/wiki/%EC%88%98%EC%9B%90%EC%97%AD" title="수원역">수원역</a>으로 노선이 이어졌다. 경부선이 부설되고 <a href="/wiki/1911%EB%85%84" title="1911년">1911년</a>에 시흥군청이 영등포리로 이전하면서 한촌(閑村)이었던 영등포리가 교통, 상업의 중심지로 급속도로 발전하기 시작하였다. 그 밖에 시흥군을 거치던 노선으로는 <a href="/wiki/%EC%88%98%EC%9D%B8%EC%84%A0" title="수인선">수인선</a>이 있었다. <a href="/wiki/1977%EB%85%84" title="1977년">1977년</a>에는 수인산업도로가 개통되어 시흥군의 주요도로로써 인근 도시들을 연결하는 중심역할을 했다.</p>
<h2><span class="mw-headline" id=".EC.82.B0.EC.97.85">산업</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%8B%9C%ED%9D%A5%EA%B5%B0&amp;action=edit&amp;section=7" title="부분 편집: 산업">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>취락의 형성이 극히 미미했던 영등포 지역이 번잡해지기 시작한 것은 1910년대부터였다. 조선 피혁 주식회사, 경성 방직 회사 등이 이 지역에 설립되면서 곧이어 많은 공장들이 가동했다. 당시 영등포 지역에 공장이 집중적으로 설립된 것은 본래 이 지역이 <a href="/wiki/%ED%95%9C%EA%B0%95" title="한강">한강</a>과 <a href="/wiki/%EC%95%88%EC%96%91%EC%B2%9C" title="안양천">안양천</a>으로 둘러싸인 <a href="/wiki/%EB%B2%94%EB%9E%8C%EC%9B%90" title="범람원">범람원</a>으로서 염가로 토지를 매입할 수 있었고, 경인선과 경부선이 개통되면서 철도교통이 편리했기 때문이다.<sup class="reference" id="cite_ref-45"><a href="#cite_note-45">[45]</a></sup></p>
