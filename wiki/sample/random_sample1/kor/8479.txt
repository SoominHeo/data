<p><b>시흥시</b>(始興市)는 <a href="/wiki/%EA%B2%BD%EA%B8%B0%EB%8F%84" title="경기도">경기도</a> 서부에 있는 <a href="/wiki/%EC%8B%9C_(%ED%96%89%EC%A0%95_%EA%B5%AC%EC%97%AD)" title="시 (행정 구역)">시</a>이다. 동쪽은 <a href="/wiki/%EA%B4%91%EB%AA%85%EC%8B%9C" title="광명시">광명시</a>, <a href="/wiki/%EC%95%88%EC%96%91%EC%8B%9C" title="안양시">안양시</a>, 서쪽은 <a href="/wiki/%EC%9D%B8%EC%B2%9C%EA%B4%91%EC%97%AD%EC%8B%9C" title="인천광역시">인천광역시</a> <a href="/wiki/%EB%82%A8%EB%8F%99%EA%B5%AC" title="남동구">남동구</a>, 남쪽은 <a href="/wiki/%ED%99%94%EC%84%B1%EC%8B%9C" title="화성시">화성시</a>, <a href="/wiki/%EC%95%88%EC%82%B0%EC%8B%9C" title="안산시">안산시</a>, 북쪽은 <a href="/wiki/%EB%B6%80%EC%B2%9C%EC%8B%9C" title="부천시">부천시</a>와 접하며, 남서쪽에 <a href="/wiki/%EC%8B%9C%ED%99%94%ED%98%B8" title="시화호">시화호</a>를 사이에 두고 <a href="/wiki/%ED%99%A9%ED%95%B4" title="황해">황해</a> <a href="/wiki/%EA%B2%BD%EA%B8%B0%EB%A7%8C" title="경기만">경기만</a> 일대와 접한다.</p>

<p><a class="mw-redirect" href="/wiki/%EC%82%BC%EA%B5%AD%EC%8B%9C%EB%8C%80" title="삼국시대">삼국시대</a> 초기 <a href="/wiki/%EB%B0%B1%EC%A0%9C" title="백제">백제</a>의 영역이었다. <a href="/wiki/475%EB%85%84" title="475년">475년</a>(<a href="/wiki/%EC%9E%A5%EC%88%98%EC%99%95" title="장수왕">장수왕</a> 63년) <a href="/wiki/%EA%B3%A0%EA%B5%AC%EB%A0%A4" title="고구려">고구려</a>의 남진으로 고구려의 영역에 편입되었다. 당시 시흥시 남부는 '장항구현(獐項口縣)'(지금의 <a href="/wiki/%EC%95%88%EC%82%B0%EC%8B%9C" title="안산시">안산시</a>), 북부는 '매소홀현(買召忽縣)'(지금의 <a href="/wiki/%EC%9D%B8%EC%B2%9C%EA%B4%91%EC%97%AD%EC%8B%9C" title="인천광역시">인천광역시</a>)이란 이름으로 기록되어 있다. 매소홀현과 장항구현은 <a href="/wiki/551%EB%85%84" title="551년">551년</a>(<a class="mw-redirect" href="/wiki/%EB%B0%B1%EC%A0%9C_%EC%84%B1%EC%99%95" title="백제 성왕">백제 성왕</a> 29년)에 <a href="/wiki/%EB%B0%B1%EC%A0%9C" title="백제">백제</a>가 지배력을 회복하면서 다시 <a href="/wiki/%EB%B0%B1%EC%A0%9C" title="백제">백제</a>의 영역으로 환원되으나, <a href="/wiki/553%EB%85%84" title="553년">553년</a>(<a class="mw-redirect" href="/wiki/%EC%8B%A0%EB%9D%BC_%EC%A7%84%ED%9D%A5%EC%99%95" title="신라 진흥왕">신라 진흥왕</a> 14년) <a href="/wiki/%EC%8B%A0%EB%9D%BC" title="신라">신라</a>가 한강 유역을 점령함으로써 신라의 영역이 되었다. <a href="/wiki/%EA%B2%BD%EB%8D%95%EC%99%95" title="경덕왕">경덕왕</a> 16년(757년) 기존의 지명을 중국식 한자 지명으로 바꾸면서 장항구현은 '장구군(獐口郡)'으로 승격하고 매소홀현은 '소성현(邵城縣)'으로 개칭되어 율진군(栗津郡)에 속하게 되었다. <a href="/wiki/%EA%B3%A0%EB%A0%A4" title="고려">고려</a> 때는 시흥시 남부인 장구군이 <a class="mw-redirect" href="/wiki/%EC%99%95%EA%B1%B4" title="왕건">태조</a> 23년(<a href="/wiki/940%EB%85%84" title="940년">940년</a>) '안산현(安山縣)'으로 개칭되었다가 <a class="mw-redirect" href="/wiki/%EC%B6%A9%EB%A0%AC%EC%99%95" title="충렬왕">충렬왕</a> 34년(1308년)에 '안산군(安山郡)'으로 승격하였고, 시흥시 북부인 소성현은 경원부에 속하였다.</p>
<p>현재의 시흥시는 조선시대에 <a class="mw-redirect" href="/wiki/%EC%9D%B8%EC%B2%9C%EB%8F%84%ED%98%B8%EB%B6%80" title="인천도호부">인천도호부</a>(옛 <a href="/wiki/%EC%86%8C%EB%9E%98%EC%9D%8D" title="소래읍">소래읍</a>인 북부)와 <a href="/wiki/%EC%95%88%EC%82%B0%EA%B5%B0" title="안산군">안산군</a>(옛 수암면·군자면인 남부)에 속해 있었으며, 대야동·신천동·은행동·신현동·매화동·과림동 등 시흥시 북부는 옛 소래읍 지역으로 1914년 부군면 통합으로 지금의 인천광역시 남부 상당부분 지역(옛 인천부 외곽지역)과 같이 경기도 <a href="/wiki/%EB%B6%80%EC%B2%9C%EA%B5%B0" title="부천군">부천군</a>에 속해 있다가 소래읍 지역이 <a href="/wiki/1973%EB%85%84" title="1973년">1973년</a> <a href="/wiki/7%EC%9B%94_1%EC%9D%BC" title="7월 1일">7월 1일</a> <a href="/wiki/%EC%8B%9C%ED%9D%A5%EA%B5%B0" title="시흥군">시흥군</a>에 편입되었다. 시흥시 남부(정왕동·연성동·군자동·거모동·목감동 일대)는 1914년 부군면 통합으로 안산군 지역이 시흥군에 통폐합되어 시흥군 수암면·군자면이 되었다.</p>
<p>현재의 시흥시는 1914년 <a href="/wiki/%EC%A1%B0%EC%84%A0%EC%B4%9D%EB%8F%85%EB%B6%80" title="조선총독부">조선총독부</a>의 <a href="/wiki/%EB%B6%80%EA%B5%B0%EB%A9%B4_%ED%86%B5%ED%8F%90%ED%95%A9" title="부군면 통폐합">부군면 통폐합</a> 이전의 전통적·역사적인 시흥(현재의 <a href="/wiki/%EC%84%9C%EC%9A%B8%ED%8A%B9%EB%B3%84%EC%8B%9C" title="서울특별시">서울특별시</a> <a href="/wiki/%EA%B8%88%EC%B2%9C%EA%B5%AC" title="금천구">금천구</a>·<a href="/wiki/%EC%98%81%EB%93%B1%ED%8F%AC%EA%B5%AC" title="영등포구">영등포구</a>·<a href="/wiki/%EA%B4%80%EC%95%85%EA%B5%AC" title="관악구">관악구</a> 및 <a href="/wiki/%EB%8F%99%EC%9E%91%EA%B5%AC" title="동작구">동작구</a> 일부, <a href="/wiki/%EA%B5%AC%EB%A1%9C%EA%B5%AC" title="구로구">구로구</a> 동부, <a href="/wiki/%EA%B2%BD%EA%B8%B0%EB%8F%84" title="경기도">경기도</a> <a href="/wiki/%EA%B4%91%EB%AA%85%EC%8B%9C" title="광명시">광명시</a>)과는 원래는 무관한 지역이다. 그러나 일제시대의 행정개편때 안산군, 과천군, 원 시흥군이 <a href="/wiki/%EC%8B%9C%ED%9D%A5%EA%B5%B0" title="시흥군">시흥군</a>으로 같은 행정구역으로 통합되었고, 그 후에는 시흥군 중심지역 였던 영등포, 현재의 서울 <a href="/wiki/%EC%8B%9C%ED%9D%A5%EB%8F%99_(%EC%84%9C%EC%9A%B8)" title="시흥동 (서울)">시흥동</a> 지역 일대가 서울특별시로 편입이 되고, 많은 지역들이 도시로 발전 함에 따라 서울로 편입 되거나, 광명, 안양, 과천, 안산, 군포, 의왕 등으로 각각 행정구역으로 분리되고 본래 인천도호부의 일부였던 부천군 소래면이 1973년 부천군 소사읍의 부천시 승격으로 시흥군에 편입됨에 따라 시흥이라는 명칭만 이어질뿐 옛 시흥 지역인 서울 시흥동 일대 <a href="/wiki/%EA%B8%88%EC%B2%9C%EA%B5%AC" title="금천구">금천구</a>와는 지금은 전혀 다른 지역이 되었다.</p>

<h2><span class="mw-headline" id=".EC.A7.80.EB.A6.AC">지리</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%8B%9C%ED%9D%A5%EC%8B%9C&amp;action=edit&amp;section=2" title="부분 편집: 지리">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>시흥시는 <a href="/wiki/%EA%B2%BD%EA%B8%B0%EB%8F%84" title="경기도">경기도</a> 중서부에 위치한 시이다. 북서쪽으로는 <a href="/wiki/%EC%9D%B8%EC%B2%9C%EA%B4%91%EC%97%AD%EC%8B%9C" title="인천광역시">인천광역시</a>, 북쪽으로는 <a href="/wiki/%EB%B6%80%EC%B2%9C%EC%8B%9C" title="부천시">부천시</a>, 북동쪽으로는 <a href="/wiki/%EA%B4%91%EB%AA%85%EC%8B%9C" title="광명시">광명시</a>, 동쪽으로는 <a href="/wiki/%EC%95%88%EC%96%91%EC%8B%9C" title="안양시">안양시</a>, 남쪽으로는 <a href="/wiki/%EC%95%88%EC%82%B0%EC%8B%9C" title="안산시">안산시</a>와 접하고 서쪽으로는 <a class="mw-redirect" href="/wiki/%EC%84%9C%ED%95%B4" title="서해">서해</a>와 면한다. 시흥시의 면적은 135.01 km²이며, 동서간 17.5 km, 남북간 17.4 km이다.</p>
<h2><span class="mw-headline" id=".ED.96.89.EC.A0.95_.EA.B5.AC.EC.97.AD">행정 구역</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%8B%9C%ED%9D%A5%EC%8B%9C&amp;action=edit&amp;section=3" title="부분 편집: 행정 구역">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<div class="rellink relarticle mainarticle"><img alt="" data-file-height="128" data-file-width="128" height="16" src="//upload.wikimedia.org/wikipedia/commons/thumb/e/ec/Crystal_Clear_app_xmag.svg/16px-Crystal_Clear_app_xmag.svg.png" srcset="//upload.wikimedia.org/wikipedia/commons/thumb/e/ec/Crystal_Clear_app_xmag.svg/24px-Crystal_Clear_app_xmag.svg.png 1.5x, //upload.wikimedia.org/wikipedia/commons/thumb/e/ec/Crystal_Clear_app_xmag.svg/32px-Crystal_Clear_app_xmag.svg.png 2x" width="16"/> 이 부분의 본문은 <a href="/wiki/%EC%8B%9C%ED%9D%A5%EC%8B%9C%EC%9D%98_%ED%96%89%EC%A0%95_%EA%B5%AC%EC%97%AD" title="시흥시의 행정 구역">시흥시의 행정 구역</a>입니다.</div>
<p><b>시흥시의 행정 구역</b>은 17개 <a class="mw-redirect" href="/wiki/%ED%96%89%EC%A0%95%EB%8F%99" title="행정동">행정동</a>, 30개 <a href="/wiki/%EB%B2%95%EC%A0%95%EB%8F%99" title="법정동">법정동</a>으로 구성되어 있다. 시흥시의 인구는 2016년 9월 말을 기준으로, 40만17명, 15만8729 세대이며, 면적은 135.78 km<sup>2</sup>이다. 전년도 인구대비 증가율은 0.44%이다. 시흥시의 인구는 감소세를 보이다가 최근들어 다시 증가하고 있는데 정왕동 배곧신도시의 영향으로 보인다.<sup class="reference" id="cite_ref-11"><a href="#cite_note-11">[11]</a></sup></p>

<h2><span class="mw-headline" id=".EC.82.B0.EC.97.85">산업</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%8B%9C%ED%9D%A5%EC%8B%9C&amp;action=edit&amp;section=5" title="부분 편집: 산업">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<h3><span class="mw-headline" id=".EC.A7.80.EC.97.AD.EB.82.B4_.EC.B4.9D.EC.83.9D.EC.82.B0">지역내 총생산</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%8B%9C%ED%9D%A5%EC%8B%9C&amp;action=edit&amp;section=6" title="부분 편집: 지역내 총생산">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>시흥시의 2012년 지역내 총생산은 26조889억원으로 경기도 지역내 총생산의 3.6%를 차지하고 있다. 이중 농림산업(1차 산업)은 414억원으로 비중이 낮고 광업 및 제조업(2차 산업)은 16조7496억원으로 64.2%의 비중으로 차지하고 상업 및 서비스업(3차 산업)은 9조2979억원으로 35.6%의 비중을 차지한다. 3차 산업 부문에서는 도소매업(6.6%)과 건설업(4.0%), 부동산업 및 임대업(3.3%)과 사업서비스업(3.2%)이 큰 비중을 차지하고 있다. <sup class="reference" id="cite_ref-12"><a href="#cite_note-12">[12]</a></sup></p>
<h3><span class="mw-headline" id=".EC.82.B0.EC.97.85.EB.B3.84_.EC.A2.85.EC.82.AC.EC.9E.90_.ED.98.84.ED.99.A9">산업별 종사자 현황</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%8B%9C%ED%9D%A5%EC%8B%9C&amp;action=edit&amp;section=7" title="부분 편집: 산업별 종사자 현황">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>2014년 시흥시 산업의 총종사자 수는 185,032명으로 경기도 총종사자 수의 4.1%를 차지하고 있다. 이중 농림어업(1차 산업)은 17명으로 비중이 낮고 광업 및 제조업(2차 산업)은 89,554명으로 48.4%의 비중으로 차지하고 상업 및 서비스업(3차 산업)은 95,461명으로 51.6%의 비중을 차지한다. 2차 산업은 경기도 전체의 비중(27.1%)보다 높고 3차 산업은 경기도 전체 비중(72.9%)보다 낮다. 3차 산업 부문에서는 도소매업(13.9%) 숙박 및 음식업(7.6%) 교육서비스업(5.2%) 보건업 및 사회복지서비스업(4.4%)이 큰 비중을 차지하고 있다. <sup class="reference" id="cite_ref-13"><a href="#cite_note-13">[13]</a></sup></p>
<h2><span class="mw-headline" id=".EB.AC.B8.ED.99.94.C2.B7.EA.B4.80.EA.B4.91">문화·관광</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%8B%9C%ED%9D%A5%EC%8B%9C&amp;action=edit&amp;section=8" title="부분 편집: 문화·관광">편집</a><span class="mw-editsection-bracket">]</span></span></h2>

<ul>
<li><b>시흥갯골생태공원</b></li>
</ul>
<p>세계에서도 희귀한 <a href="/wiki/%EA%B0%9C%ED%8E%84" title="개펄">내만갯골</a>이 있는 곳으로, 이 갯골은 하루 두 번, 썰물 때 그 모습을 드러내는데, 그 모습이 마치 살아 움직이는 뱀의 모습을 닮았다 해서 사행성(巳行性) 갯골이라고도 불린다.<sup class="reference" id="cite_ref-14"><a href="#cite_note-14">[14]</a></sup></p>
<ul>
<li><b>시흥갯골축제</b></li>
</ul>
<p>매년 8월 중순에 시흥갯골생태공원에서 열리며 갯골생태체험, 연꽃 페스티벌, 문화행사 등을 즐길 수 있다.<sup class="reference" id="cite_ref-15"><a href="#cite_note-15">[15]</a></sup></p>
<ul>
<li><b>오이도</b></li>
</ul>
<p>1922년 일제가 염전을 만들기 위해 안산간 제방을 쌓으면서 육지와 연결되었다. 현재는 섬 서쪽에 해안을 매립, 이주단지가 조성되어 새로운 삶의 보금자리가 형성되었다. 이곳에는 시흥시의 향토음식인 오이도 굴회덮밥을 비롯한 많은 식당과 횟집이 들어서 있어 미각을 한층 돋우고 있다.<sup class="reference" id="cite_ref-16"><a href="#cite_note-16">[16]</a></sup></p>
<ul>
<li><b>월곶포구</b></li>
</ul>
<p>조선시대에는 수군만호가 설치되었고, 1991년까지는 바다와 자원의 보고라고 불리는 갯벌이었다. 이곳을 시흥시가 매립하여 면적 564,938㎡(170,893평)에 횟집과 어물전 230여 곳을 비롯하여 각종 위락시설을 조성하였다. 신선한 자연산 활어회 등 많은 횟집마다 독특한 분위기가 있으며 월곶을 찾는 사람들이 언제라도 싱싱한 횟감을 사갈 수 있는 어시장이 자리 잡고 있다.<sup class="reference" id="cite_ref-17"><a href="#cite_note-17">[17]</a></sup></p>
