<p><b>식각</b>(蝕刻, <span lang="en" xml:lang="en">etching</span> <small>에칭<sup>[<a href="/wiki/%EC%9C%84%ED%82%A4%EB%B0%B1%EA%B3%BC:%EC%98%81%EC%96%B4%EC%9D%98_%ED%95%9C%EA%B8%80_%ED%91%9C%EA%B8%B0" title="위키백과:영어의 한글 표기">*</a>]</sup></small>)은 <a href="/wiki/%ED%99%94%ED%95%99" title="화학">화학</a><a class="new" href="/w/index.php?title=%EC%95%BD%ED%92%88&amp;action=edit&amp;redlink=1" title="약품 (없는 문서)">약품</a>의 <a href="/wiki/%EB%B6%80%EC%8B%9D" title="부식">부식</a>작용을 응용한 <a class="new" href="/w/index.php?title=%EC%86%8C%ED%98%95&amp;action=edit&amp;redlink=1" title="소형 (없는 문서)">소형</a>(塑型)이나 <a class="new" href="/w/index.php?title=%ED%91%9C%EB%A9%B4%EA%B0%80%EA%B3%B5&amp;action=edit&amp;redlink=1" title="표면가공 (없는 문서)">표면가공</a>의 방법이다. 사용하는 소재에서 필요한 부위만 방식(防蝕) 처리를 한 후 부식시켜서 불필요한 부분을 제거하여 원하는 모양을 얻는다.</p>
<p>동판에 의한 <a href="/wiki/%ED%8C%90%ED%99%94" title="판화">판화</a>, <a href="/wiki/%EC%9D%B8%EC%87%84" title="인쇄">인쇄</a>기술로 발전해온 역사가 길기때문에 구리나 아연같은 금속가공으로 사용되는 것이 많지만, 부식성만 있다면 다양한 소재의 소형, 표면가공에 응용이 가능하다.</p>

<h2><span class="mw-headline" id=".ED.8C.90.ED.99.94.2C_.EC.9D.B8.EC.87.84">판화, 인쇄</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%8B%9D%EA%B0%81&amp;action=edit&amp;section=1" title="부분 편집: 판화, 인쇄">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>방식처리를 가한 <a href="/wiki/%EA%B5%AC%EB%A6%AC" title="구리">구리</a>판의 표면을 바늘로 깎아서 부식하여 <a class="mw-redirect" href="/wiki/%EC%9D%8C%EA%B0%81" title="음각">음각</a>을 얻는 데 사용된다. 부식작용을 통하여 간접적으로 판을 가공하기 때문에 음각화기법에서 <a href="/wiki/%ED%8C%90%ED%99%94" title="판화">간접법</a>에 분류된다. 직접 구리판에 선을 조각하는 <a href="/wiki/%ED%8C%90%ED%99%94" title="판화">직접법</a>보다 선을 뜻대로 그리기 쉽다. 아주 정교하고 세밀한 묘사가 가능하기 때문에 <a href="/wiki/%EC%A7%80%ED%8F%90" title="지폐">지폐</a>의 자작에 사용된다. 자세한 것은 <a href="/wiki/%ED%8C%90%ED%99%94" title="판화">판화</a>를 참조하세요</p>
<h2><span class="mw-headline" id=".EA.B8.88.EC.86.8D.EA.B0.80.EA.B3.B5">금속가공</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%8B%9D%EA%B0%81&amp;action=edit&amp;section=2" title="부분 편집: 금속가공">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p><a class="new" href="/w/index.php?title=%ED%94%84%EB%A0%88%EC%8A%A4_%EA%B0%80%EA%B3%B5&amp;action=edit&amp;redlink=1" title="프레스 가공 (없는 문서)">프레스 가공</a>에서 어렵고 복잡한 가공을 위해 식각이 응용되고 있다. <a class="mw-redirect" href="/wiki/%EC%A7%91%EC%A0%81%ED%9A%8C%EB%A1%9C" title="집적회로">집적회로</a>의 리드 프레임, 전기면도기의 망, 컬러 <a href="/wiki/%EC%9D%8C%EA%B7%B9%EC%84%A0%EA%B4%80" title="음극선관">음극선관</a>의 <a class="new" href="/w/index.php?title=%EC%89%90%EB%8F%84%EC%9A%B0%EB%A7%88%EC%8A%A4%ED%81%AC&amp;action=edit&amp;redlink=1" title="쉐도우마스크 (없는 문서)">쉐도우마스크</a>같은 수십 ~ 수백 μm 두께의 금속판재부품을 제조하는 기술도 있다. 이 방법으로 제작된 모형의 부품을 <a class="new" href="/w/index.php?title=%EC%8B%9D%EA%B0%81%EB%B6%80%ED%92%88&amp;action=edit&amp;redlink=1" title="식각부품 (없는 문서)">식각부품</a>이라고 불린다.</p>
<h2><span class="mw-headline" id=".EB.B0.98.EB.8F.84.EC.B2.B4.EA.B3.B5.ED.95.99">반도체공학</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%8B%9D%EA%B0%81&amp;action=edit&amp;section=3" title="부분 편집: 반도체공학">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p><a class="new" href="/w/index.php?title=%EB%B0%98%EB%8F%84%EC%B2%B4%EA%B3%B5%ED%95%99&amp;action=edit&amp;redlink=1" title="반도체공학 (없는 문서)">반도체공학</a> 분야는 <a href="/wiki/%EC%9B%A8%EC%9D%B4%ED%8D%BC" title="웨이퍼">웨이퍼</a>의 반도체 박막을 형상가공하는 기술에 응용되고 있다. 반도체 웨이퍼에 산화박막을 형성해서 <a class="new" href="/w/index.php?title=%ED%8F%AC%ED%86%A0%EB%A0%88%EC%A7%80%EC%8A%A4%ED%8A%B8&amp;action=edit&amp;redlink=1" title="포토레지스트 (없는 문서)">포토레지스트</a>(<span lang="en" xml:lang="en">photoresist</span>)로 패턴을 형성한 후 식각으로 불필요한 박막을 제거한다. 식각 기법은 <a href="/wiki/%ED%94%8C%EB%A3%A8%EC%98%A4%EB%A6%B0%ED%99%94_%EC%88%98%EC%86%8C%EC%82%B0" title="플루오린화 수소산">불산</a> 액체를 사용하는 <a href="/wiki/%EC%8A%B5%EC%8B%9D_%EC%8B%9D%EA%B0%81" title="습식 식각">습식 식각</a>(웨트 에칭)과 <a class="new" href="/w/index.php?title=4%EB%B6%88%ED%99%94_%EB%A9%94%ED%83%84&amp;action=edit&amp;redlink=1" title="4불화 메탄 (없는 문서)">4불화 메탄</a>(<span lang="en" xml:lang="en">tetrafluoromethane</span>) 가스를 사용하는 <a href="/wiki/%EA%B1%B4%EC%8B%9D_%EC%8B%9D%EA%B0%81" title="건식 식각">건식 식각</a>(드라이 에칭)이 있다.</p>
<p>동일하게 <a href="/wiki/%EC%9D%B8%EC%87%84_%ED%9A%8C%EB%A1%9C_%EA%B8%B0%ED%8C%90" title="인쇄 회로 기판">인쇄 회로 기판</a>의 배선 형성을 위해 도체(동박)을 제거하기 위한 공정에도 이용된다. 식각액으로는 <a class="mw-redirect mw-disambig" href="/wiki/%EC%97%BC%ED%99%94%EC%B2%A0" title="염화철">염화철</a>이 쓰인다.</p>
<h2><span class="mw-headline" id=".EA.B8.B0.ED.83.80">기타</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%8B%9D%EA%B0%81&amp;action=edit&amp;section=4" title="부분 편집: 기타">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p><a href="/wiki/%EC%9C%A0%EB%A6%AC" title="유리">유리</a>의 장식방법으로 <a class="new" href="/w/index.php?title=%EC%9C%A0%EB%A6%AC_%EC%8B%9D%EA%B0%81&amp;action=edit&amp;redlink=1" title="유리 식각 (없는 문서)">유리 식각</a>이 있다.</p>
