<p><b>식물</b>(植物)은 분류학적으로 <b>식물계</b>에 속하는 <a href="/wiki/%EC%83%9D%EB%AC%BC" title="생물">생물</a>을 말한다. <a href="/wiki/%EC%83%9D%EB%AC%BC" title="생물">생물</a>을 나누는 주요 분류의 하나로, <a href="/wiki/%EB%82%98%EB%AC%B4" title="나무">나무</a>, <a href="/wiki/%ED%92%80" title="풀">풀</a> 등이 여기에 속한다. 넓게는 <a href="/wiki/%EB%8F%99%EB%AC%BC" title="동물">동물</a>이 아닌 것을 모두 가리킨다. 보통 <a href="/wiki/%EA%B4%91%ED%95%A9%EC%84%B1" title="광합성">광합성</a>을 하여 녹말 등의 광합성을 만드나, 일부 <a class="mw-redirect" href="/wiki/%EA%B8%B0%EC%83%9D%EC%8B%9D%EB%AC%BC" title="기생식물">기생식물</a>이랑 <a class="new" href="/w/index.php?title=%EA%B3%B5%EC%83%9D%EC%8B%9D%EB%AC%BC&amp;action=edit&amp;redlink=1" title="공생식물 (없는 문서)">공생식물</a>처럼, 엽록소를 잃고, 대신 직접 포식하거나, 기생, 공생 등으로 양분을 얻는 종도 있다. 원래 식물은 운동성이 거의 없으나, <a href="/wiki/%ED%8C%8C%EB%A6%AC%EC%A7%80%EC%98%A5" title="파리지옥">파리지옥</a>, <a class="mw-redirect" href="/wiki/%EC%8B%A0%EA%B2%BD%EC%B4%88" title="신경초">신경초</a>, <a class="new" href="/w/index.php?title=%EB%AC%B4%EC%B4%88&amp;action=edit&amp;redlink=1" title="무초 (없는 문서)">무초</a>처럼 민첩한 운동을 하는 식물도 몇 종 있다.</p>

<p><a href="/wiki/%EC%95%84%EB%A6%AC%EC%8A%A4%ED%86%A0%ED%85%94%EB%A0%88%EC%8A%A4" title="아리스토텔레스">아리스토텔레스</a>는 모든 생물을 식물(일반적으로 움직이지 않는)과 동물(가끔씩 먹이를 찾으러 움직이는)로 구분하였다. <a href="/wiki/%EC%B9%BC_%ED%8F%B0_%EB%A6%B0%EB%84%A4" title="칼 폰 린네">칼 폰 린네</a> 시스템에서는 식물<a href="/wiki/%EA%B3%84_(%EC%83%9D%EB%AC%BC%ED%95%99)" title="계 (생물학)">계</a>와 <a class="mw-redirect" href="/wiki/%EB%8F%99%EB%AC%BC%EA%B3%84" title="동물계">동물계</a>가 되었다. <a href="/wiki/%EA%B7%A0%EA%B3%84" title="균계">균</a>과 일부 <a href="/wiki/%EC%A1%B0%EB%A5%98_(%EC%88%98%EC%83%9D_%EC%83%9D%EB%AC%BC)" title="조류 (수생 생물)">조류</a>(藻類) 그룹은 새로운 식물계에서 제거되었다. 그러나 이들은 여전히 여러 환경에서 식물이라 불리기도 한다.</p>

<h3><span class="mw-headline" id=".EC.A1.B0.EB.A5.98">조류</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%8B%9D%EB%AC%BC&amp;action=edit&amp;section=2" title="부분 편집: 조류">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<div class="rellink relarticle mainarticle"><img alt="" data-file-height="128" data-file-width="128" height="16" src="//upload.wikimedia.org/wikipedia/commons/thumb/e/ec/Crystal_Clear_app_xmag.svg/16px-Crystal_Clear_app_xmag.svg.png" srcset="//upload.wikimedia.org/wikipedia/commons/thumb/e/ec/Crystal_Clear_app_xmag.svg/24px-Crystal_Clear_app_xmag.svg.png 1.5x, //upload.wikimedia.org/wikipedia/commons/thumb/e/ec/Crystal_Clear_app_xmag.svg/32px-Crystal_Clear_app_xmag.svg.png 2x" width="16"/> 이 부분의 본문은 <a href="/wiki/%EC%A1%B0%EB%A5%98_(%EC%88%98%EC%83%9D_%EC%83%9D%EB%AC%BC)" title="조류 (수생 생물)">조류 (수생 생물)</a>입니다.</div>
<p>대부분의 <a href="/wiki/%EC%A1%B0%EB%A5%98_(%EC%88%98%EC%83%9D_%EC%83%9D%EB%AC%BC)" title="조류 (수생 생물)">조류</a>(藻類)는 더 이상 식물계에 속하지 않는다.<sup class="reference" id="cite_ref-2"><a href="#cite_note-2">[2]</a></sup><sup class="reference" id="cite_ref-3"><a href="#cite_note-3">[3]</a></sup> 조류는 광합성을 통하여 에너지를 만드는 여러 그룹의 유기체들로 이루어져 있으며 이들은 각기 광합성을 띠지 않는 원종(原種)으로부터 독립적으로 일어난다. 조류 가운데 가장 눈에 띄는 것이 다세포 조류인 <a href="/wiki/%EB%B0%94%EB%8B%B7%EB%A7%90" title="바닷말">바닷말</a>인데 토양 식물과 매우 닮았지만 <a href="/wiki/%EB%85%B9%EC%A1%B0%EB%A5%98" title="녹조류">녹조류</a>, <a href="/wiki/%ED%99%8D%EC%A1%B0%EC%8B%9D%EB%AC%BC" title="홍조식물">홍조류</a>, <a href="/wiki/%EA%B0%88%EC%A1%B0%EB%A5%98" title="갈조류">갈조류</a>로 분류된다. 이러한 각 조류 그룹은 다양하고도 미시적인 단세포 유기체를 포함하기도 한다.</p>
<h3><span class="mw-headline" id=".EA.B7.A0">균</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%8B%9D%EB%AC%BC&amp;action=edit&amp;section=3" title="부분 편집: 균">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<div class="rellink relarticle mainarticle"><img alt="" data-file-height="128" data-file-width="128" height="16" src="//upload.wikimedia.org/wikipedia/commons/thumb/e/ec/Crystal_Clear_app_xmag.svg/16px-Crystal_Clear_app_xmag.svg.png" srcset="//upload.wikimedia.org/wikipedia/commons/thumb/e/ec/Crystal_Clear_app_xmag.svg/24px-Crystal_Clear_app_xmag.svg.png 1.5x, //upload.wikimedia.org/wikipedia/commons/thumb/e/ec/Crystal_Clear_app_xmag.svg/32px-Crystal_Clear_app_xmag.svg.png 2x" width="16"/> 이 부분의 본문은 <a href="/wiki/%EA%B7%A0%EA%B3%84" title="균계">균계</a>입니다.</div>
<p><a href="/wiki/%EA%B7%A0%EA%B3%84" title="균계">균</a>의 분류는 최근의 생물 역사에 이르기까지 논란이 일었다. 이들은 분명히 동물이 아니었기에 <a href="/wiki/%EC%B9%BC_%ED%8F%B0_%EB%A6%B0%EB%84%A4" title="칼 폰 린네">린네</a>의 원래 분류는 균을 식물계 안으로 넣었다. 나중에 <a href="/wiki/%EB%AF%B8%EC%83%9D%EB%AC%BC%ED%95%99" title="미생물학">미생물학</a>이 발전하자 19세기 <a href="/wiki/%EC%97%90%EB%A5%B8%EC%8A%A4%ED%8A%B8_%ED%97%A4%EC%BC%88" title="에른스트 헤켈">에른스트 헤켈</a>은 새롭게 발견한 미생물을 분류할 필요를 느꼈다. 이를 동물계에 넣지 않고 대신 <a href="/wiki/%EC%9B%90%EC%83%9D%EC%83%9D%EB%AC%BC" title="원생생물">원생생물계</a>라는 새로운 계를 만들었는데 균이 식물계에 들어가는 것이 적절한지 원생생물계로 다시 분류되는 것이 맞는지 논란이 일었다. 헤켈은 스스로 이를 결정하는 것이 어렵다는 것을 알았다. 1969년에 <a href="/wiki/%EB%A1%9C%EB%B2%84%ED%8A%B8_%ED%9C%98%ED%83%9C%EC%BB%A4" title="로버트 휘태커">로버트 휘태커</a>는 균계를 만들 것을 제안하였다.</p>
<p>이러한 휘태커의 재분류는 균계와 식물계의 영양분이 본질적으로 다르다는 것에 기반을 둔다. 광합성을 통하여 탄소를 얻음으로써 <a href="/wiki/%EC%9E%90%EA%B0%80%EC%98%81%EC%96%91%EC%83%9D%EB%AC%BC" title="자가영양생물">스스로 영양분을 얻는</a> 다세포 <a href="/wiki/%EA%B4%91%EC%98%81%EC%96%91%EC%83%9D%EB%AC%BC" title="광영양생물">광영양생물</a>인 식물과는 달리 균은 일반적으로 주변 물질을 분해하거나 흡수하여 탄소를 얻는 <a href="/wiki/%EC%A2%85%EC%86%8D%EC%98%81%EC%96%91%EC%83%9D%EB%AC%BC" title="종속영양생물">종속영양생물</a>(영양소를 다른 생물이 만든 유기물에 의존하는 생물)이거나 다세포 <a class="new" href="/w/index.php?title=%EB%B6%80%EC%83%9D_%EC%83%9D%EB%AC%BC&amp;action=edit&amp;redlink=1" title="부생 생물 (없는 문서)">부생 생물</a>이다.</p>
<h2><span class="mw-headline" id=".EB.B6.84.EB.A5.98">분류</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%8B%9D%EB%AC%BC&amp;action=edit&amp;section=4" title="부분 편집: 분류">편집</a><span class="mw-editsection-bracket">]</span></span></h2>

<h2><span class="mw-headline" id=".EC.83.9D.EB.A6.AC.ED.95.99">생리학</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%8B%9D%EB%AC%BC&amp;action=edit&amp;section=5" title="부분 편집: 생리학">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<div class="rellink relarticle mainarticle"><img alt="" data-file-height="128" data-file-width="128" height="16" src="//upload.wikimedia.org/wikipedia/commons/thumb/e/ec/Crystal_Clear_app_xmag.svg/16px-Crystal_Clear_app_xmag.svg.png" srcset="//upload.wikimedia.org/wikipedia/commons/thumb/e/ec/Crystal_Clear_app_xmag.svg/24px-Crystal_Clear_app_xmag.svg.png 1.5x, //upload.wikimedia.org/wikipedia/commons/thumb/e/ec/Crystal_Clear_app_xmag.svg/32px-Crystal_Clear_app_xmag.svg.png 2x" width="16"/> 이 부분의 본문은 <a href="/wiki/%EC%8B%9D%EB%AC%BC%EC%83%9D%EB%A6%AC%ED%95%99" title="식물생리학">식물생리학</a>입니다.</div>
<h3><span class="mw-headline" id=".EA.B4.91.ED.95.A9.EC.84.B1">광합성</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%8B%9D%EB%AC%BC&amp;action=edit&amp;section=6" title="부분 편집: 광합성">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<div class="rellink relarticle mainarticle"><img alt="" data-file-height="128" data-file-width="128" height="16" src="//upload.wikimedia.org/wikipedia/commons/thumb/e/ec/Crystal_Clear_app_xmag.svg/16px-Crystal_Clear_app_xmag.svg.png" srcset="//upload.wikimedia.org/wikipedia/commons/thumb/e/ec/Crystal_Clear_app_xmag.svg/24px-Crystal_Clear_app_xmag.svg.png 1.5x, //upload.wikimedia.org/wikipedia/commons/thumb/e/ec/Crystal_Clear_app_xmag.svg/32px-Crystal_Clear_app_xmag.svg.png 2x" width="16"/> 이 부분의 본문은 <a href="/wiki/%EA%B4%91%ED%95%A9%EC%84%B1" title="광합성">광합성</a>, <a class="new" href="/w/index.php?title=%EC%83%9D%EB%AC%BC_%EC%83%89%EC%86%8C&amp;action=edit&amp;redlink=1" title="생물 색소 (없는 문서)">생물 색소</a>입니다.</div>
<p>식물은 <a href="/wiki/%EB%B9%9B" title="빛">빛</a>으로부터 얻은 에너지를 이용하여 자기만의 먹이 분자를 만드는 <a href="/wiki/%EA%B4%91%ED%95%A9%EC%84%B1" title="광합성">광합성</a>을 한다.</p>
<h3><span class="mw-headline" id=".EB.A9.B4.EC.97.AD_.EC.B2.B4.EA.B3.84">면역 체계</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%8B%9D%EB%AC%BC&amp;action=edit&amp;section=7" title="부분 편집: 면역 체계">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>마치 신경계처럼 식물은 입사광의 세기와 질에 대한 체계 정보를 주고받는다. 한 잎에 화학 반응을 자극하는 입사광은 <a href="/wiki/%EA%B4%80%EB%8B%A4%EB%B0%9C" title="관다발">유관속초세포</a>를 통하여 식물 전체에 연쇄적인 신호 반응을 일으킨다.</p>
<h3><span class="mw-headline" id=".EB.82.B4.EB.B6.80_.EB.B6.84.EB.B0.B0">내부 분배</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%8B%9D%EB%AC%BC&amp;action=edit&amp;section=8" title="부분 편집: 내부 분배">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<div class="rellink relarticle mainarticle"><img alt="" data-file-height="128" data-file-width="128" height="16" src="//upload.wikimedia.org/wikipedia/commons/thumb/e/ec/Crystal_Clear_app_xmag.svg/16px-Crystal_Clear_app_xmag.svg.png" srcset="//upload.wikimedia.org/wikipedia/commons/thumb/e/ec/Crystal_Clear_app_xmag.svg/24px-Crystal_Clear_app_xmag.svg.png 1.5x, //upload.wikimedia.org/wikipedia/commons/thumb/e/ec/Crystal_Clear_app_xmag.svg/32px-Crystal_Clear_app_xmag.svg.png 2x" width="16"/> 이 부분의 본문은 <a href="/wiki/%EA%B4%80%EB%8B%A4%EB%B0%9C" title="관다발">관다발</a>입니다.</div>
<p><a href="/wiki/%EA%B4%80%EB%8B%A4%EB%B0%9C%EC%8B%9D%EB%AC%BC" title="관다발식물">관다발식물</a>은 <a href="/wiki/%EB%AC%BC%EA%B4%80%EB%B6%80" title="물관부">물관부</a>(목질부)와 <a href="/wiki/%EC%B2%B4%EA%B4%80%EB%B6%80" title="체관부">체관부</a>로 불리는 특별한 구조를 통하여 다른 부위끼리 영양분을 전달하는 식물과는 다르다. 또, 이들은 물과 무기물을 받아들이는 <a href="/wiki/%EB%BF%8C%EB%A6%AC" title="뿌리">뿌리</a>가 있다. 물관부는 물과 무기질을 뿌리로부터 식물 끝으로까지 옮겨 주며 체관부는 잎이 만든 당분과 영양분을 뿌리에 전달한다.<sup class="reference" id="cite_ref-13"><a href="#cite_note-13">[13]</a></sup></p>
<h2><span class="mw-headline" id=".EC.83.9D.ED.83.9C.ED.95.99">생태학</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%8B%9D%EB%AC%BC&amp;action=edit&amp;section=9" title="부분 편집: 생태학">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<div class="rellink relarticle mainarticle"><img alt="" data-file-height="128" data-file-width="128" height="16" src="//upload.wikimedia.org/wikipedia/commons/thumb/e/ec/Crystal_Clear_app_xmag.svg/16px-Crystal_Clear_app_xmag.svg.png" srcset="//upload.wikimedia.org/wikipedia/commons/thumb/e/ec/Crystal_Clear_app_xmag.svg/24px-Crystal_Clear_app_xmag.svg.png 1.5x, //upload.wikimedia.org/wikipedia/commons/thumb/e/ec/Crystal_Clear_app_xmag.svg/32px-Crystal_Clear_app_xmag.svg.png 2x" width="16"/> 이 부분의 본문은 <a href="/wiki/%EC%8B%9D%EB%AC%BC%EC%83%9D%ED%83%9C%ED%95%99" title="식물생태학">식물생태학</a>입니다.</div>
<p>육지 식물과 조류가 행하는 광합성은 거의 모든 생태계에 궁극의 에너지 원천이자 유기 물질이다. 광합성은 모든 지구의 대기 성분을 빠르게 바꾸었으며 그 결과 <a href="/wiki/%EC%82%B0%EC%86%8C" title="산소">산소</a>가 21%를 차지한다. 동물과 다른 대부분의 생물은 산소에 의지하여 <a class="new" href="/w/index.php?title=%ED%98%B8%EA%B8%B0%EC%84%B1_%EC%83%9D%EB%AC%BC&amp;action=edit&amp;redlink=1" title="호기성 생물 (없는 문서)">호흡</a>을 한다. 그렇지 않은 생물들은 상대적으로 드문 <a href="/wiki/%EC%A0%80%EC%82%B0%EC%86%8C%EC%A6%9D" title="저산소증">저산소 환경</a>에 제한을 받으며 살아간다. 식물들은 대부분의 토양 생태계의 <a href="/wiki/%EC%9E%90%EA%B0%80%EC%98%81%EC%96%91%EC%83%9D%EB%AC%BC" title="자가영양생물">주된 생산물</a>이며 이러한 생태계에서 <a href="/wiki/%EB%A8%B9%EC%9D%B4_%EC%82%AC%EC%8A%AC" title="먹이 사슬">먹이 사슬</a>의 토대를 이룬다. 수많은 동물들은 산소와 먹이뿐 아니라 은신처를 위하여 식물에 의지한다.</p>
<p>토양 식물은 <a href="/wiki/%EB%AC%BC%EC%9D%98_%EC%88%9C%ED%99%98" title="물의 순환">물의 순환</a>과 다른 몇 가지 <a href="/wiki/%EC%83%9D%EB%AC%BC%EC%A7%80%EA%B5%AC%ED%99%94%ED%95%99%EC%A0%81_%EC%88%9C%ED%99%98" title="생물지구화학적 순환">생물지구화학적 순환</a>의 주된 요소이다. 일부 식물은 <a class="mw-redirect" href="/wiki/%EC%A7%88%EC%86%8C%EA%B3%A0%EC%A0%95" title="질소고정">질소고정</a> 박테리아와 <a href="/wiki/%EA%B3%B5%EC%A7%84%ED%99%94" title="공진화">공진화</a>해왔는데 이는 식물을 <a href="/wiki/%EC%A7%88%EC%86%8C%EC%9D%98_%EC%88%9C%ED%99%98" title="질소의 순환">질소의 순환</a>의 중요한 일부로 만들어 놓았다. 식물의 뿌리는 <a class="mw-redirect" href="/wiki/%ED%86%A0%EC%96%91" title="토양">토양</a> 발달과 <a class="new" href="/w/index.php?title=%ED%86%A0%EC%96%91_%EC%B9%A8%EC%8B%9D&amp;action=edit&amp;redlink=1" title="토양 침식 (없는 문서)">토양 침식</a> 방지에 중요한 역할을 한다.</p>
<h2><span class="mw-headline" id=".EC.A4.91.EC.9A.94.EC.84.B1">중요성</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%8B%9D%EB%AC%BC&amp;action=edit&amp;section=10" title="부분 편집: 중요성">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>사람들이 식물을 연구하면서 경제식물학이나 <a class="new" href="/w/index.php?title=%EB%AF%BC%EC%A1%B1%EC%8B%9D%EB%AC%BC%ED%95%99&amp;action=edit&amp;redlink=1" title="민족식물학 (없는 문서)">민족식물학</a>이라는 용어가 생겨나기 시작하였다. 어떠한 사람들은 경제식물학이 현대의 경작 식물에 초점을 두는 반면 민족식물학이 토착인이 경작하고 사용하는 토착 식물에 초점을 둔다고 여긴다. 인간이 식물을 경작하는 일은 문명의 토대인 <a href="/wiki/%EB%86%8D%EC%97%85" title="농업">농업</a>의 일부로 되어 있다. 식물 경작은 <a class="new" href="/w/index.php?title=%EB%86%8D%EC%97%85_%EA%B2%BD%EC%A0%9C%ED%95%99&amp;action=edit&amp;redlink=1" title="농업 경제학 (없는 문서)">농업 경제학</a>, <a href="/wiki/%EC%9B%90%EC%98%88" title="원예">원예</a>, <a href="/wiki/%EC%9E%84%EC%97%85" title="임업">임업</a>으로 나뉜다.</p>

<li>과학적이고 문화적인 이용</li>
