<p><b>신경주역</b>(新慶州驛)은 <a href="/wiki/%EB%8C%80%ED%95%9C%EB%AF%BC%EA%B5%AD" title="대한민국">대한민국</a> <a href="/wiki/%EA%B2%BD%EC%83%81%EB%B6%81%EB%8F%84" title="경상북도">경상북도</a> <a href="/wiki/%EA%B2%BD%EC%A3%BC%EC%8B%9C" title="경주시">경주시</a> <a href="/wiki/%EA%B1%B4%EC%B2%9C%EC%9D%8D" title="건천읍">건천읍</a> 화천리에 있는 <a href="/wiki/%EA%B2%BD%EB%B6%80%EA%B3%A0%EC%86%8D%EC%B2%A0%EB%8F%84" title="경부고속철도">KTX</a>의 <a href="/wiki/%EC%B2%A0%EB%8F%84%EC%97%AD" title="철도역">철도역</a>이다. <a href="/wiki/2010%EB%85%84" title="2010년">2010년</a> <a href="/wiki/11%EC%9B%94_1%EC%9D%BC" title="11월 1일">11월 1일</a> <a href="/wiki/%EA%B2%BD%EB%B6%80%EA%B3%A0%EC%86%8D%EC%B2%A0%EB%8F%84" title="경부고속철도">경부고속철도</a> 2단계 구간이 완공되면서 영업을 개시하였으며, <a href="/wiki/%ED%95%9C%EA%B5%AD%EA%B3%A0%EC%86%8D%EC%B2%A0%EB%8F%84" title="한국고속철도">KTX</a> 열차가 정차하게 되었다.</p>

<h2><span class="mw-headline" id=".EC.A0.84.EA.B4.91.ED.8C.90_.ED.91.9C.EC.8B.9C">전광판 표시</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%8B%A0%EA%B2%BD%EC%A3%BC%EC%97%AD&amp;action=edit&amp;section=1" title="부분 편집: 전광판 표시">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<ul>
<li>부산발 서울행 KTX 열차가 <a href="/wiki/%EC%98%81%EB%93%B1%ED%8F%AC%EC%97%AD" title="영등포역">영등포역</a>, <a href="/wiki/%EC%88%98%EC%9B%90%EC%97%AD" title="수원역">수원역</a>을 경유하면 전광판과 행선지에 <a href="/wiki/%EC%84%9C%EC%9A%B8%EC%97%AD" title="서울역">서울</a>·<a href="/wiki/%EC%88%98%EC%9B%90%EC%97%AD" title="수원역">수원</a>·<a href="/wiki/%EC%98%81%EB%93%B1%ED%8F%AC%EC%97%AD" title="영등포역">영등포</a>라고 표시한다.</li>
</ul>
<h2><span class="mw-headline" id=".EA.B0.9C.EC.9A.94">개요</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%8B%A0%EA%B2%BD%EC%A3%BC%EC%97%AD&amp;action=edit&amp;section=2" title="부분 편집: 개요">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p><a href="/wiki/%EA%B2%BD%EB%B6%80%EA%B3%A0%EC%86%8D%EC%B2%A0%EB%8F%84" title="경부고속철도">경부고속철도</a>를 계획하던 <a href="/wiki/1990%EB%85%84" title="1990년">1990년</a> <a href="/wiki/6%EC%9B%94" title="6월">6월</a>, 동대구-경주-부산 구간을 확정하고 배후 지역으로 <a href="/wiki/%EC%98%81%EC%B2%9C%EC%8B%9C" title="영천시">영천시</a>, <a href="/wiki/%EA%B2%BD%EC%A3%BC%EC%8B%9C" title="경주시">경주시</a>, <a href="/wiki/%ED%8F%AC%ED%95%AD%EC%8B%9C" title="포항시">포항시</a>, <a href="/wiki/%EC%9A%B8%EC%82%B0%EA%B4%91%EC%97%AD%EC%8B%9C" title="울산광역시">울산광역시</a>가 거론되었다. 그 중 “<a href="/wiki/%EB%8C%80%ED%95%9C%EB%AF%BC%EA%B5%AD" title="대한민국">대한민국</a>을 대표하는 역사적 도시”, “<a href="/wiki/%EB%8C%80%ED%95%9C%EB%AF%BC%EA%B5%AD" title="대한민국">대한민국</a>의 관광 대표 도시”로서 <a href="/wiki/%EA%B2%BD%EC%A3%BC%EC%8B%9C" title="경주시">경주시</a>의 중요성이 부각되었고 <a href="/wiki/%ED%8F%AC%ED%95%AD%EC%8B%9C" title="포항시">포항시</a>에는 <a href="/wiki/%ED%8F%AC%ED%95%AD%EA%B3%B5%ED%95%AD" title="포항공항">포항공항</a>, <a href="/wiki/%EC%9A%B8%EC%82%B0%EA%B4%91%EC%97%AD%EC%8B%9C" title="울산광역시">울산광역시</a>는 <a href="/wiki/%EC%9A%B8%EC%82%B0%EA%B3%B5%ED%95%AD" title="울산공항">울산공항</a>이 있는데 반해 <a href="/wiki/%EA%B2%BD%EC%A3%BC%EC%8B%9C" title="경주시">경주시</a>는 공항이 없었고, 관광 도시 접근 편의성이 중요해 신경주역 신설이 확정되었다.<sup class="reference" id="cite_ref-1"><a href="#cite_note-1">[1]</a></sup> 이후 신경주역을 동남권 대표 정차역으로 천명하였으나, <a href="/wiki/%EB%85%B8%EB%AC%B4%ED%98%84_%EC%A0%95%EB%B6%80" title="노무현 정부">노무현 정부</a> 시절 <a href="/wiki/%EC%9A%B8%EC%82%B0%EA%B4%91%EC%97%AD%EC%8B%9C" title="울산광역시">울산광역시</a>에서 <a href="/wiki/%EA%B4%91%EC%97%AD%EC%8B%9C" title="광역시">광역시</a>의 위상을 들어 <a href="/wiki/%EC%9A%B8%EC%82%B0%EC%97%AD" title="울산역">울산역</a> 신설을 요구하였다<sup class="reference" id="cite_ref-2"><a href="#cite_note-2">[2]</a></sup>. <a href="/wiki/2004%EB%85%84" title="2004년">2004년</a> <a class="mw-redirect" href="/wiki/%EA%B1%B4%EC%84%A4%EA%B5%90%ED%86%B5%EB%B6%80" title="건설교통부">건설교통부</a>에서 받아들이고, <a href="/wiki/%EA%B2%BD%EB%B6%80%EA%B3%A0%EC%86%8D%EC%B2%A0%EB%8F%84" title="경부고속철도">경부고속철도</a> 2단계 구간을 수정해서 <a href="/wiki/%EC%98%A4%EC%86%A1%EC%97%AD" title="오송역">오송역</a>, <a id="cite_ref-3"><a href="#cite_note-3">[3]</a></sup></p>
<p>향후 <a href="/wiki/%EC%A4%91%EC%95%99%EC%84%A0" title="중앙선">중앙선</a>, <a href="/wiki/%EB%8F%99%ED%95%B4%EC%84%A0" title="동해선">동해선</a>이 이설되면 기존 <a href="/wiki/%EA%B2%BD%EC%A3%BC%EC%97%AD" title="경주역">경주역</a>과 통합해 <a href="/wiki/%EB%AC%B4%EA%B6%81%ED%99%94%ED%98%B8" title="무궁화호">무궁화호</a>도 취급할 예정이며, 승강장에도 <a href="/wiki/%EB%8F%99%ED%95%B4%EC%84%A0" title="동해선">동해선</a> 공사 표지판이 있다</p>
