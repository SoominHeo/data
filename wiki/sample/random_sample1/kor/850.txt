<p><b>고도이크루스 안토니오 톰바</b>(<span lang="es" xml:lang="es">Club Deportivo Godoy Cruz Antonio Tomba</span>)는 <a href="/wiki/%EC%95%84%EB%A5%B4%ED%97%A8%ED%8B%B0%EB%82%98" title="아르헨티나">아르헨티나</a> <a href="/wiki/%EB%A9%98%EB%8F%84%EC%82%AC_%EC%A3%BC" title="멘도사 주">멘도사 주</a> <a href="/wiki/%EA%B3%A0%EB%8F%84%EC%9D%B4%ED%81%AC%EB%A3%A8%EC%8A%A4" title="고도이크루스">고도이크루스</a>를 연고로 하는 <a href="/wiki/%EC%8A%A4%ED%8F%AC%EC%B8%A0" title="스포츠">스포츠</a> 클럽이다. 특히 <a href="/wiki/%EC%B6%95%EA%B5%AC" title="축구">축구</a>로 유명하며, 최근에 <a href="/wiki/%EC%95%84%EB%A5%B4%ED%97%A8%ED%8B%B0%EB%82%98_%ED%94%84%EB%A6%AC%EB%A9%94%EB%9D%BC_%EB%94%94%EB%B9%84%EC%8B%9C%EC%98%A8" title="아르헨티나 프리메라 디비시온">아르헨티나 프리메라 디비시온</a>에 승격했다.</p>
<p><a href="/wiki/1921%EB%85%84" title="1921년">1921년</a> <a href="/wiki/6%EC%9B%94_21%EC%9D%BC" title="6월 21일">6월 21일</a> 스포르티보 고도이크루스(Sportivo Godoy Cruz)라는 이름으로 처음 만들어져 <a href="/wiki/1930%EB%85%84" title="1930년">1930년</a> <a href="/wiki/4%EC%9B%94_25%EC%9D%BC" title="4월 25일">4월 25일</a> 데포르티보 보데가 안토니오 톰바(Deportivo Bodega Antonio Tomba)와 합병하면서 지금의 이름을 갖게 되었다. <a href="/wiki/1959%EB%85%84" title="1959년">1959년</a>에는 14,000 명을 수용하는 전용 경기장인 펠리시아노 감바르테 경기장(Estadio Feliciano Gambarte)이 완공되었다. 프리메라 디비시온에 승격한 이후부터는 <a class="mw-redirect" href="/wiki/%EB%A7%90%EB%B9%84%EB%82%98%EC%8A%A4_%EC%95%84%EB%A5%B4%ED%97%A8%ED%8B%B0%EB%82%98%EC%8A%A4_%EA%B2%BD%EA%B8%B0%EC%9E%A5" title="말비나스 아르헨티나스 경기장">말비나스 아르헨티나스 경기장</a>(구 멘도사 시립 경기장)을 사용하고 있다.</p>
<p>팀의 별명인 엘 톰바(El Tomba)와 보데게로(Bodeguero)는 합병된 클럽의 이름(데포르티보 보데가 안토니오 톰바)에서 유래한 것이다. 이전에 쓰던 경기장 옆에 철길이 생긴 이후에는 엘 엑스프레소라는 별명이 추가되었다.</p>
<p>전국 리그에 참여하기 전까지 고도이크루스는 멘도사 주 리그에서 활약하였고, 1944, 1947, 1950, 1951, 1954, 1968, 그리고 1989년과 1990년 우승하였고 지금은 폐지된 토르네오 델 인테리오르(Torneo del Interior, 지금의 <a class="new" href="/w/index.php?title=%ED%86%A0%EB%A5%B4%EB%84%A4%EC%98%A4_%EC%95%84%EB%A5%B4%ED%97%A8%ED%8B%B0%EB%85%B8_C&amp;action=edit&amp;redlink=1" title="토르네오 아르헨티노 C (없는 문서)">토르네오 아르헨티노 C</a>와는 다름)에 진출하였다. <a href="/wiki/1994%EB%85%84" title="1994년">1994년</a> 토르네오 델 인테리오르에서 우승한 고도이크루스는 <a href="/wiki/%ED%94%84%EB%A6%AC%EB%A9%94%EB%9D%BC_B_%EB%82%98%EC%8B%9C%EC%98%A4%EB%82%A0" title="프리메라 B 나시오날">나시오날 B</a>에 승격했고, <a href="/wiki/2006%EB%85%84" title="2006년">2006년</a>에는 처음으로 프리메라 디비시온에 진입하였다. 이듬해 프로모시온에서 <a href="/wiki/CA_%EC%9A%B0%EB%9D%BC%EC%B9%B8" title="CA 우라칸">우라칸</a>에 패하면서 다시 강등되었지만 2007/08 시즌 프리메라 B 나시오날 준우승을 차지하면서 프리메라 디비시온에 재진입하였다.</p>
