<p><b>신리후 역</b>(<span style="font-size: smaller;"><a href="/wiki/%EC%9D%BC%EB%B3%B8%EC%96%B4" title="일본어">일본어</a>:</span> <span lang="ja" xml:lang="ja">新利府駅</span>)은 <a href="/wiki/%EC%9D%BC%EB%B3%B8" title="일본">일본</a> <a href="/wiki/%EB%AF%B8%EC%95%BC%EA%B8%B0_%ED%98%84" title="미야기 현">미야기 현</a> <a href="/wiki/%EB%AF%B8%EC%95%BC%EA%B8%B0_%EA%B5%B0" title="미야기 군">미야기 군</a> <a href="/wiki/%EB%A6%AC%ED%9B%84_%EC%A0%95" title="리후 정">리후 정</a>에 있는 <a class="mw-redirect" href="/wiki/%EB%8F%99%EC%9D%BC%EB%B3%B8_%EC%97%AC%EA%B0%9D%EC%B2%A0%EB%8F%84" title="동일본 여객철도">동일본 여객철도</a> <a href="/wiki/%EB%8F%84%ED%98%B8%EC%BF%A0_%EB%B3%B8%EC%84%A0" title="도호쿠 본선">도호쿠 본선</a> (리후 지선)의 <a href="/wiki/%EC%B2%A0%EB%8F%84%EC%97%AD" title="철도역">역</a>이다. <a class="mw-redirect" href="/wiki/%EB%8B%A8%EC%84%A0_%EC%8A%B9%EA%B0%95%EC%9E%A5" title="단선 승강장">단선 승강장</a> 1면 1선의 구조를 지닌 <a href="/wiki/%EC%B2%A0%EB%8F%84%EC%97%AD" title="철도역">지상역</a>이며, <a href="/wiki/%EC%9D%B4%EC%99%80%ED%82%A4%EB%A6%AC_%EC%97%AD" title="이와키리 역">이와키리 역</a>이 이 역을 관리하고 있다. JR 동일본 관내의 <a href="/wiki/%EB%8F%84%ED%98%B8%EC%BF%A0_%EC%8B%A0%EC%B9%B8%EC%84%BC" title="도호쿠 신칸센">도호쿠</a> · <a href="/wiki/%EC%A1%B0%EC%97%90%EC%93%B0_%EC%8B%A0%EC%B9%B8%EC%84%BC" title="조에쓰 신칸센">조에쓰</a> · <a href="/wiki/%EC%95%BC%EB%A7%88%EA%B0%80%ED%83%80_%EC%8B%A0%EC%B9%B8%EC%84%BC" title="야마가타 신칸센">야마가타</a> · <a href="/wiki/%EC%95%84%ED%82%A4%ED%83%80_%EC%8B%A0%EC%B9%B8%EC%84%BC" title="아키타 신칸센">아키타</a> · <a href="/wiki/%ED%98%B8%EC%BF%A0%EB%A6%AC%EC%BF%A0_%EC%8B%A0%EC%B9%B8%EC%84%BC" title="호쿠리쿠 신칸센">호쿠리쿠 신칸센</a>의 차량기지인 신칸센 종합 차량센터로 병설되어 있다. 도호쿠 신칸센 개업에 의해, 신칸센 종합 차량센터 근무자의 편을 고려해서 설치되었던 역이다.</p>
