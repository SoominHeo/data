<p><b>신비</b>(SHINVI)는 <a href="/wiki/%EB%8C%80%ED%95%9C%EB%AF%BC%EA%B5%AD" title="대한민국">대한민국</a>의 <a class="mw-redirect" href="/wiki/%EC%9D%8C%EC%95%85_%EA%B7%B8%EB%A3%B9" title="음악 그룹">음악 그룹</a>이다. 그룹의 뜻은 "높이 비상하라"라는 뜻을 가지고 있다. 데뷔 당시 <a href="/wiki/%EB%B0%80%ED%81%AC_(%EC%9D%8C%EC%95%85_%EA%B7%B8%EB%A3%B9)" title="밀크 (음악 그룹)">밀크</a>와 제2의 <a href="/wiki/S.E.S." title="S.E.S.">S.E.S.</a>로 불리었다. 1집 타이틀곡인 To My Friend 뮤직비디오에는 <a href="/wiki/%EC%8B%A0%ED%99%94" title="신화">신화</a>의 <a href="/wiki/%EC%A0%84%EC%A7%84_(%EA%B0%80%EC%88%98)" title="전진 (가수)">전진</a>이 참여해서 화제가 되기도 하였다. 후속곡인 "Darling"은 <a href="/wiki/2002%EB%85%84" title="2002년">2002년</a>에 출시된 게임인 "샤이닝 로어"에 삽입되기도 하였다. <a href="/wiki/SM_%EC%97%94%ED%84%B0%ED%85%8C%EC%9D%B8%EB%A8%BC%ED%8A%B8" title="SM 엔터테인먼트">SM 엔터테인먼트</a> 소속으로 아직 공식발표는 없지만 1집을 마친 후 사실상 소리소문없이 해체했다. 해체 후 <a href="/wiki/%EB%8B%88%EB%AA%A8_(%EA%B0%80%EC%88%98)" title="니모 (가수)">오상은</a>만 현재 현역으로 활동 중이며, 소속사를 옮긴후 니모라는 이름으로 <a href="/wiki/2007%EB%85%84" title="2007년">2007년</a> 솔로 앨범을 발표하였다. 유수진은 연예계에서 완전히 은퇴했으며, 유나는 <a href="/wiki/2005%EB%85%84" title="2005년">2005년</a> <a href="/wiki/%EA%B3%A0%EB%A0%A4%EB%8C%80%ED%95%99%EA%B5%90" title="고려대학교">고려대학교</a> 정외과에 입학한 후 <a href="/wiki/2010%EB%85%84" title="2010년">2010년</a> 사법시험에 합격하여 현재는 변호사로 활동 중이다.</p>
