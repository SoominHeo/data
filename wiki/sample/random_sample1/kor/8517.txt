<p><b>신성</b>(新星)은 <a href="/wiki/%EB%B0%B1%EC%83%89%EC%99%9C%EC%84%B1" title="백색왜성">백색왜성</a>으로 <a href="/wiki/%EC%88%98%EC%86%8C" title="수소">수소</a>를 비롯한 물질이 유입되면서 급격한 핵 반응이 일어남에 따라 별이 밝아지는 현상을 일컫는다. 신성이라는 이름도 하늘에 새로이 밝은 별이 나타났다는 것을 뜻하는 말이다.</p>

<h2><span class="mw-headline" id=".ED.98.95.EC.84.B1">형성</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%8B%A0%EC%84%B1_(%EC%B2%9C%EC%B2%B4)&amp;action=edit&amp;section=1" title="부분 편집: 형성">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p><a href="/wiki/%EB%B0%B1%EC%83%89%EC%99%9C%EC%84%B1" title="백색왜성">백색왜성</a>이 <a class="mw-redirect" href="/wiki/%EB%A1%9C%EC%8B%9C%EC%97%BD" title="로시엽">로시엽</a>을 넘어서는 인접한 <a class="mw-redirect" href="/wiki/%EB%8F%99%EB%B0%98%EC%84%B1" title="동반성">동반성</a>을 지니고 있다면, 백색왜성은 동반성의 외부대기로부터 지속적으로 기체를 흡수하게 된다. 동반성은 <a class="mw-redirect" href="/wiki/%EC%A3%BC%EA%B3%84%EC%97%B4%EC%84%B1" title="주계열성">주계열성</a>일수도 혹은 <a href="/wiki/%EC%A0%81%EC%83%89%EA%B1%B0%EC%84%B1" title="적색거성">적색거성</a>으로 팽창하는 별일 수도 있다. 흡수된 기체는 대개 <a href="/wiki/%EC%88%98%EC%86%8C" title="수소">수소</a>나 <a href="/wiki/%ED%97%AC%EB%A5%A8" title="헬륨">헬륨</a>이며, 기체는 그 뒤에 백색왜성의 높은 <a href="/wiki/%EC%A4%91%EB%A0%A5" title="중력">중력</a>으로 인해 압축되고 가열된다. 가열된 수소는 백색왜성의 표면에서 <a href="/wiki/CNO_%EC%88%9C%ED%99%98" title="CNO 순환">CNO 순환</a>을 통해 안정적으로 <a href="/wiki/%ED%95%B5%EC%9C%B5%ED%95%A9" title="핵융합">핵융합</a>을 일으킨다. 하지만, 흡수되거나 생성된 헬륨이 핵융합하여 무거운 <a class="mw-redirect" href="/wiki/%ED%99%94%ED%95%99_%EC%9B%90%EC%86%8C" title="화학 원소">원소</a>를 생성하기 위해서는 보다 높은 온도와 압력이 필요하며, 적어도 2000만 <a href="/wiki/%EC%BC%88%EB%B9%88" title="켈빈">켈빈</a>이 넘어서야 <a href="/wiki/%ED%97%AC%EB%A5%A8_%ED%95%B5%EC%9C%B5%ED%95%A9" title="헬륨 핵융합">헬륨 핵융합</a>을 통해 헬륨을 소모할 수 있다.</p>
<p>이러한 핵융합 과정에서 발생하는 엄청난 에너지는 백색왜성의 표면으로부터 기체를 날려보내며, 별을 극도로 밝게 빛나게 한다. 최대 밝기가 되기까지의 시간은 길 수도 짧을 수도 있지만, 최대 밝기를 지나서는 서서히 어두워지게 된다. <sup class="reference" id="cite_ref-1"><a href="#cite_note-1">[1]</a></sup></p>
<p>하지만, 이러한 격렬한 반응 중에도 실제 분출되는 물질의 양은 대개 <a href="/wiki/%ED%83%9C%EC%96%91" title="태양">태양</a> 질량의 1/10,000 정도에 지나지 않으며, 이는 백색왜성의 질량에 비추어볼 때 매우 미미한 수준이다. 게다가 흡수된 질량의 단 5%만이 핵융합을 통해 에너지를 공급한다. 하지만 그 정도의 핵융합만으로도 분출물을 초당 수천 킬로미터의 속도로 가속시키며, 또한 신성의 <a href="/wiki/%EA%B4%91%EB%8F%84" title="광도">광도</a>를 적게는 태양의 수 배에서 많게는 50,000-100,000배까지 높이기에 충분하다. <sup class="reference" id="cite_ref-2"><a href="#cite_note-2">[2]</a></sup></p>
<p><a href="/wiki/%EB%B0%B1%EC%83%89%EC%99%9C%EC%84%B1" title="백색왜성">백색왜성</a>은 수소만 유입된다면, 몇 번씩도 신성이 될 수 있다. 이렇게 반복적으로 신성이 되는 별을 반복신성이라고 하며, 대표적인 예로는 <a class="mw-redirect" href="/wiki/%EB%B1%80%EC%A3%BC%EC%9D%B8%EC%9E%90%EB%A6%AC_RS" title="뱀주인자리 RS">뱀주인자리 RS</a>가 있다. 이 별은 <a href="/wiki/1898%EB%85%84" title="1898년">1898년</a>, <a href="/wiki/1933%EB%85%84" title="1933년">1933년</a>, <a href="/wiki/1958%EB%85%84" title="1958년">1958년</a>, <a href="/wiki/1967%EB%85%84" title="1967년">1967년</a>, <a href="/wiki/1985%EB%85%84" title="1985년">1985년</a>, <a href="/wiki/2006%EB%85%84" title="2006년">2006년</a>, 총 6회의 신성 현상을 나타내었다. 하지만, 백색왜성의 물질이 바닥나거나, 백색왜성이 <a href="/wiki/%EC%A4%91%EC%84%B1%EC%9E%90%EB%B3%84" title="중성자별">중성자별</a>로 붕괴한다면, 혹은 백색왜성이 Ia 형 <a href="/wiki/%EC%B4%88%EC%8B%A0%EC%84%B1" title="초신성">초신성</a>으로 폭발하게 된다면 이 과정은 끝이나게 된다.</p>
<p>가끔 일부 신성은 매우 가까우면서 밝아서 맨눈으로도 관측가능하다. 최근의 현상으로는 <a href="/wiki/1975%EB%85%84" title="1975년">1975년</a> <a href="/wiki/8%EC%9B%94_29%EC%9D%BC" title="8월 29일">8월 29일</a>의 <a class="new" href="/w/index.php?title=%EB%B0%B1%EC%A1%B0%EC%9E%90%EB%A6%AC_V1500&amp;action=edit&amp;redlink=1" title="백조자리 V1500 (없는 문서)">백조자리 V1500</a>이 있다. 이 신성은 <a href="/wiki/%EB%B0%B1%EC%A1%B0%EC%9E%90%EB%A6%AC" title="백조자리">백조자리</a>에서 관측되었으며, <a href="/wiki/%EB%8D%B0%EB%84%A4%EB%B8%8C" title="데네브">데네브</a>에 필적하는 <a class="mw-redirect" href="/wiki/%EA%B2%89%EB%B3%B4%EA%B8%B0_%EB%93%B1%EA%B8%89" title="겉보기 등급">겉보기 등급</a> 2.0 정도까지 이르렀다. 다른 최근의 예로는 <a href="/wiki/1992%EB%85%84" title="1992년">1992년</a>의 <a class="new" href="/w/index.php?title=%EB%B0%B1%EC%A1%B0%EC%9E%90%EB%A6%AC_V1974&amp;action=edit&amp;redlink=1" title="백조자리 V1974 (없는 문서)">백조자리 V1974</a>도 있지만, 이는 이전 신성에 비해 훨씬 어두웠다.</p>
<h2><span class="mw-headline" id=".EB.B0.9C.EC.83.9D_.EB.B9.88.EB.8F.84.EC.99.80_.EC.A4.91.EC.9A.94.EC.84.B1">발생 빈도와 중요성</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%8B%A0%EC%84%B1_(%EC%B2%9C%EC%B2%B4)&amp;action=edit&amp;section=2" title="부분 편집: 발생 빈도와 중요성">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>천문학자들은 <a class="mw-redirect" href="/wiki/%EC%9A%B0%EB%A6%AC_%EC%9D%80%ED%95%98" title="우리 은하">우리 은하</a>에서 연간 20내지 60개의 신성 현상이 있을 것이라고 추정한다. 하지만 실제 관측되는 수는 이에 훨씬 못 미치며, 이는 대개 너무 먼 거리나 관측 위치를 제대로 찾지 못한 것에 기인한다.<sup class="reference" id="cite_ref-3"><a href="#cite_note-3">[3]</a></sup> 한편 이웃하는 <a href="/wiki/%EC%95%88%EB%93%9C%EB%A1%9C%EB%A9%94%EB%8B%A4_%EC%9D%80%ED%95%98" title="안드로메다 은하">안드로메다 은하</a>에서 발견되는 신성은 우리 은하에서 발견되는 신성의 대략 1/2 내지는 1/3 수준이다.<sup class="reference" id="cite_ref-4"><a href="#cite_note-4">[4]</a></sup></p>
<p>신성의 분출물을 <a href="/wiki/%EB%B6%84%EA%B4%91%ED%95%99" title="분광학">분광학</a>을 통해 관측한 결과, 분출물은 주로 <a href="/wiki/%ED%97%AC%EB%A5%A8" title="헬륨">헬륨</a>, <a href="/wiki/%ED%83%84%EC%86%8C" title="탄소">탄소</a>, <a href="/wiki/%EC%A7%88%EC%86%8C" title="질소">질소</a>, <a href="/wiki/%EC%82%B0%EC%86%8C" title="산소">산소</a>, <a href="/wiki/%EB%84%A4%EC%98%A8" title="네온">네온</a>, <a href="/wiki/%EB%A7%88%EA%B7%B8%EB%84%A4%EC%8A%98" title="마그네슘">마그네슘</a>과 같은 원소를 포함하는 것으로 밝혀졌다. 하지만, 신성이 이러한 분출물을 통해 <a href="/wiki/%EC%84%B1%EA%B0%84%EB%A7%A4%EC%A7%88" title="성간매질">성간매질</a>의 농도를 증가시키는 정도는 다른 우주현상에 비해서는 적은 수준이다. <a href="/wiki/%EC%B4%88%EC%8B%A0%EC%84%B1" title="초신성">초신성</a>에 비해서는 1/50 정도이며, <a href="/wiki/%EC%A0%81%EC%83%89%EA%B1%B0%EC%84%B1" title="적색거성">적색거성</a>이나 <a href="/wiki/%EC%B4%88%EA%B1%B0%EC%84%B1" title="초거성">초거성</a>에 비해서는 1/200 정도에 지나지 않는다.</p>
<p><a class="mw-redirect" href="/wiki/%EB%B1%80%EC%A3%BC%EC%9D%B8%EC%9E%90%EB%A6%AC_RS" title="뱀주인자리 RS">뱀주인자리 RS</a>와 같은 반복신성은 드물다. 하지만 천문학자들은, 전부는 아니라도 대부분이 (비록 1,000년에서 100,000년 정도의 주기를 가질지라도) 반복신성일 것이라고 생각한다. <sup class="reference" id="cite_ref-5"><a href="#cite_note-5">[5]</a></sup> 반복신성의 주기는 백색왜성의 흡수율보다는 백색왜성 자체의 질량에 보다 의존한다. 질량이 높은 백색왜성의 경우, 보다 적은 물질로도 충분한 핵융합을 일으킬 수 있으며, 따라서 질량이 낮은 백색왜성보다 더욱 잦은 신성 현상을 일으킬 수 있게 된다.</p>
