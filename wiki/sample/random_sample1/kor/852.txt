<p><b>클루브 데포르티보 엘 나시오날</b> (Club Deportivo El Nacional) 은 <a href="/wiki/%ED%82%A4%ED%86%A0" title="키토">키토</a>를 연고로 하는 <a href="/wiki/%EC%97%90%EC%BD%B0%EB%8F%84%EB%A5%B4" title="에콰도르">에콰도르</a>의 스포츠 클럽으로, 프로 <a href="/wiki/%EC%B6%95%EA%B5%AC" title="축구">축구</a>팀으로 잘 알려져 있다. 이 팀은 현재 국내 1부 리그인 <a class="new" href="/w/index.php?title=%EC%97%90%EC%BD%B0%EB%8F%84%EB%A5%B4_%EC%84%B8%EB%A6%AC%EC%97%90_A&amp;action=edit&amp;redlink=1" title="에콰도르 세리에 A (없는 문서)">에콰도르 세리에 A</a>에 소속되어 있다.</p>
<p>엘 나시오날은 13차례 리그 우승 (<a href="/wiki/%EB%B0%94%EB%A5%B4%EC%85%80%EB%A1%9C%EB%82%98_SC" title="바르셀로나 SC">바르셀로나 스포르팅</a>보다 1회 적음) 을 거두었다. 이 클럽은 국내 리그 3연패를 경험해 본 유일한 팀이며, 이 업적을 1976-78, 그리고 1982-84에 달성하였다. 이 클럽은 <a href="/wiki/%EC%BD%94%ED%8C%8C_%EB%A6%AC%EB%B2%A0%EB%A5%B4%ED%83%80%EB%8F%84%EB%A0%88%EC%8A%A4" title="코파 리베르타도레스">코파 리베르타도레스</a>에 22번 참가했는데, 이는 에콰도르 클럽 역사상 최다 기록이다. 이 팀이 대륙컵에서 낸 최고 성적은 <a href="/wiki/%EC%BD%94%ED%8C%8C_%EB%A6%AC%EB%B2%A0%EB%A5%B4%ED%83%80%EB%8F%84%EB%A0%88%EC%8A%A4_1985" title="코파 리베르타도레스 1985">1985 시즌</a>의 4강이었다.</p>
<p>엘 나시오날은 1964년 6월 1일에 창단되어 <a class="new" href="/w/index.php?title=%EC%97%90%EC%BD%B0%EB%8F%84%EB%A5%B4_%EA%B5%B0%EB%8C%80&amp;action=edit&amp;redlink=1" title="에콰도르 군대 (없는 문서)">에콰도르 군대</a>가 2013년에 첫 민주적 선거가 있기 전까지 관할했었다. 이 클럽은 지금까지도 에콰도르인 축구 선수만 기용하는 전통을 지니고 있는데, 이로 인해 엘 나시오날은 <i>Puros Criollos</i> (순수 크리올) 이라는 별명을 얻었다. 라이벌 구단으로는 같은 연고지의 <a class="mw-redirect" href="/wiki/%EB%A6%AC%EA%B0%80_%EB%8D%B0%ED%8F%AC%EB%A5%B4%ED%8B%B0%EB%B0%94_%EC%9A%B0%EB%8B%88%EB%B2%A0%EB%A5%B4%EC%8B%9C%ED%83%80%EB%A6%AC%EC%95%84_%EB%8D%B0_%ED%82%A4%ED%86%A0" title="리가 데포르티바 우니베르시타리아 데 키토">LDU 키토</a>, <a href="/wiki/%EB%8D%B0%ED%8F%AC%EB%A5%B4%ED%8B%B0%EB%B3%B4_%ED%82%A4%ED%86%A0" title="데포르티보 키토">데포르티보 키토</a>, 그리고 과야킬 연고의 <a href="/wiki/%EB%B0%94%EB%A5%B4%EC%85%80%EB%A1%9C%EB%82%98_SC" title="바르셀로나 SC">바르셀로나 스포르팅</a>과 <a href="/wiki/CS_%EC%97%90%EB%A9%9C%EB%A0%88%ED%81%AC" title="CS 에멜레크">에멜레크</a>가 있었다. 현재 <a class="new" href="/w/index.php?title=%EC%97%90%EC%8A%A4%ED%83%80%EB%94%94%EC%98%A4_%EC%98%AC%EB%A6%BC%ED%94%BC%EC%BD%94_%EC%95%84%ED%83%80%EC%99%88%ED%8C%8C&amp;action=edit&amp;redlink=1" title="에스타디오 올림피코 아타왈파 (없는 문서)">에스타디오 올림피코 아타왈파</a>를 홈구장으로 쓰고 있다.</p>
<p>엘 나시오날이 참가하는 다른 스포츠 부서로는 <a href="/wiki/%EC%B2%B4%EC%A1%B0" title="체조">체조</a>, <a href="/wiki/%ED%83%81%EA%B5%AC" title="탁구">탁구</a>, <a class="new" href="/w/index.php?title=%EC%97%90%EC%BD%B0%EB%B0%9C%EB%A6%AC&amp;action=edit&amp;redlink=1" title="에콰발리 (없는 문서)">에콰발리</a>, <a href="/wiki/%EC%B2%B4%EC%8A%A4" title="체스">체스</a>, 그리고 <a class="mw-redirect" href="/wiki/%EC%82%AC%EA%B2%A9_%EA%B2%BD%EA%B8%B0" title="사격 경기">사격</a>이 있다.</p>
