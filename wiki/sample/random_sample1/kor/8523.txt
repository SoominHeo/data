<p><b>신시내티 레즈</b>(<span style="font-size: smaller;"><a href="/wiki/%EC%98%81%EC%96%B4" title="영어">영어</a>:</span> <span lang="en" xml:lang="en">Cincinnati Reds</span>)는 <a href="/wiki/%EB%AF%B8%EA%B5%AD" title="미국">미국</a> <a href="/wiki/%EC%98%A4%ED%95%98%EC%9D%B4%EC%98%A4_%EC%A3%BC" title="오하이오 주">오하이오 주</a> <a href="/wiki/%EC%8B%A0%EC%8B%9C%EB%82%B4%ED%8B%B0" title="신시내티">신시내티</a>를 연고지로 하는 <a href="/wiki/%ED%94%84%EB%A1%9C_%EC%95%BC%EA%B5%AC" title="프로 야구">프로 야구 팀</a>이다. <a href="/wiki/%EB%A9%94%EC%9D%B4%EC%A0%80_%EB%A6%AC%EA%B7%B8_%EB%B2%A0%EC%9D%B4%EC%8A%A4%EB%B3%BC" title="메이저 리그 베이스볼">메이저 리그</a> <a class="mw-redirect" href="/wiki/%EB%82%B4%EC%85%94%EB%84%90_%EB%A6%AC%EA%B7%B8_%EC%A4%91%EB%B6%80_%EC%A7%80%EA%B5%AC" title="내셔널 리그 중부 지구">내셔널 리그 중부 지구</a> 소속이다.</p>
<p><a href="/wiki/1869%EB%85%84" title="1869년">1869년</a>에 창단한 세계 최초의 프로 야구 팀 <b>신시내티 레드스타킹스</b>(Red Stockings)를 모태로 한다. <a href="/wiki/1958%EB%85%84" title="1958년">1958년</a>부터 현대의 이름을 사용하고 있다. <a href="/wiki/%ED%88%AC%EC%88%98" title="투수">투수</a> <a class="mw-redirect" href="/wiki/%EA%B9%80%EC%84%A0%EC%9A%B0_(1977%EB%85%84)" title="김선우 (1977년)">김선우</a>가 <a href="/wiki/2006%EB%85%84" title="2006년">2006년</a>에, <a href="/wiki/%EB%B4%89%EC%A4%91%EA%B7%BC" title="봉중근">봉중근</a>(<a href="/wiki/LG_%ED%8A%B8%EC%9C%88%EC%8A%A4" title="LG 트윈스">LG 트윈스</a>)이 2004년부터 2006년까지 뛰었던 팀이다.</p>
<p><a href="/wiki/2013%EB%85%84" title="2013년">2013년</a>, <a href="/wiki/%EC%99%B8%EC%95%BC%EC%88%98" title="외야수">외야수</a> <a href="/wiki/%EC%B6%94%EC%8B%A0%EC%88%98" title="추신수">추신수</a>가 같은 주 내의 팀인 <a href="/wiki/%ED%81%B4%EB%A6%AC%EB%B8%94%EB%9E%9C%EB%93%9C_%EC%9D%B8%EB%94%94%EC%96%B8%EC%8A%A4" title="클리블랜드 인디언스">클리블랜드 인디언스</a>에서 트레이드로 이적하여 몸담았었다. <a href="/wiki/2014%EB%85%84" title="2014년">2014년</a>, <a href="/wiki/%EC%99%B8%EC%95%BC%EC%88%98" title="외야수">외야수</a> <a href="/wiki/%EC%B6%94%EC%8B%A0%EC%88%98" title="추신수">추신수</a>가 AL 서부리그 <a href="/wiki/%ED%85%8D%EC%82%AC%EC%8A%A4_%EB%A0%88%EC%9D%B8%EC%A0%80%EC%8A%A4" title="텍사스 레인저스">텍사스 레인저스</a>로 FA 이적하였다.</p>
