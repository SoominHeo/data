<p><b>신윤복필 풍속도 화첩</b>(申潤福筆 風俗圖 畵帖)은 <a class="mw-redirect" href="/wiki/%EC%A1%B0%EC%84%A0%EC%8B%9C%EB%8C%80" title="조선시대">조선시대</a> 후기인 <a href="/wiki/18%EC%84%B8%EA%B8%B0" title="18세기">18세기</a> 말에 혜원 <a href="/wiki/%EC%8B%A0%EC%9C%A4%EB%B3%B5" title="신윤복">신윤복</a>이 그린 풍속화를 엮은 연작 화첩으로, 조선시대의 대표적인 화첩으로 여겨진다. <b>혜원풍속도</b> (蕙園風俗圖) 혹은 <b>혜원풍속도첩</b> (蕙園風俗圖帖)이라고도 부른다.</p>
<p>이 화첩은 신윤복의 호인 혜원을 따서 붙여졌으며, 총 30장의 작품으로 구성되어있다. <a href="/wiki/%EC%9D%BC%EB%B3%B8" title="일본">일본</a>으로 유출된 것을 <a href="/wiki/1930%EB%85%84" title="1930년">1930년</a> <a href="/wiki/%EA%B0%84%EC%86%A1%EB%AF%B8%EC%88%A0%EA%B4%80" title="간송미술관">간송미술관</a>의 설립자인 간송 <a href="/wiki/%EC%A0%84%ED%98%95%ED%95%84" title="전형필">전형필</a>이 일본 <a href="/wiki/%EC%98%A4%EC%82%AC%EC%B9%B4_%EC%8B%9C" title="오사카 시">오사카</a>의 한 고미술상에게서 사들여 새로 표구를 했다. 전형필과 함께 문화재 유출을 막는 데 큰 역할을 담당한 <a href="/wiki/%EC%98%A4%EC%84%B8%EC%B0%BD" title="오세창">오세창</a>이 <a class="new" href="/w/index.php?title=%ED%91%9C%EC%A0%9C&amp;action=edit&amp;redlink=1" title="표제 (없는 문서)">표제</a>와 <a class="new" href="/w/index.php?title=%EB%B0%9C%EB%AC%B8&amp;action=edit&amp;redlink=1" title="발문 (없는 문서)">발문</a>을 썼다. 이 작품은 각기 가로 28㎝, 세로 35㎝이며 <a href="/wiki/%ED%95%9C%EC%A7%80" title="한지">한지</a>에 그림을 그리고, 짤막한 글과 함께 <a href="/wiki/%EB%82%99%EA%B4%80" title="낙관">낙관</a>이 곁들여져있다. 주로 <a href="/wiki/%ED%95%9C%EB%9F%89" title="한량">한량</a>과 <a href="/wiki/%EA%B8%B0%EC%83%9D" title="기생">기생</a>의 모습을 솔직하게 그려냈으며, 18세기 말의 풍속과 복식을 엿볼 수 있다. 이 작품은 <a href="/wiki/%EB%8C%80%ED%95%9C%EB%AF%BC%EA%B5%AD" title="대한민국">대한민국</a>에서 <a href="/wiki/%EA%B5%AD%EB%B3%B4" title="국보">국보</a> 제 135호로 지정되었으며, 현재 <a class="mw-redirect" href="/wiki/%EC%84%9C%EC%9A%B8_%EC%84%B1%EB%B6%81%EA%B5%AC" title="서울 성북구">서울 성북구</a> 간송미술관에서 소장하고 있다.</p>
