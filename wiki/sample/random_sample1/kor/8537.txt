<p><b>신이문역</b>(新里門驛)은 <a href="/wiki/%EC%84%9C%EC%9A%B8%ED%8A%B9%EB%B3%84%EC%8B%9C" title="서울특별시">서울특별시</a> <a href="/wiki/%EB%8F%99%EB%8C%80%EB%AC%B8%EA%B5%AC" title="동대문구">동대문구</a> <a href="/wiki/%EC%9D%B4%EB%AC%B8%EB%8F%99" title="이문동">이문동</a>에 있는 <a href="/wiki/%EA%B2%BD%EC%9B%90%EC%84%A0" title="경원선">경원선</a>의 <a href="/wiki/%EC%A0%84%EC%B2%A0%EC%97%AD" title="전철역">전철역</a>이다. 부역명은 <b>한국예술종합학교</b>(韓國藝術綜合學校)로, 인근에 <a href="/wiki/%ED%95%9C%EA%B5%AD%EC%98%88%EC%88%A0%EC%A2%85%ED%95%A9%ED%95%99%EA%B5%90" title="한국예술종합학교">한국예술종합학교</a>, <a href="/wiki/%EC%9D%B4%EB%AC%B8%EC%B0%A8%EB%9F%89%EC%82%AC%EC%97%85%EC%86%8C" title="이문차량사업소">이문차량사업소</a>가 있다.</p>

<h2><span class="mw-headline" id=".EC.97.AD.EB.AA.85.EC.97.90_.EA.B4.80.ED.95.9C_.EC.9D.B4.EC.95.BC.EA.B8.B0">역명에 관한 이야기</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%8B%A0%EC%9D%B4%EB%AC%B8%EC%97%AD&amp;action=edit&amp;section=1" title="부분 편집: 역명에 관한 이야기">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>역이 <a href="/wiki/%EC%9D%B4%EB%AC%B8%EB%8F%99" title="이문동">이문동</a>에 있으나 굳이 <b>신이문</b>이라는 역명을 사용하는 이유는 <a href="/wiki/2004%EB%85%84" title="2004년">2004년</a> <a href="/wiki/7%EC%9B%94_5%EC%9D%BC" title="7월 5일">7월 5일</a>에 <a href="/wiki/%EC%9D%B4%EB%AC%B8%EC%B0%A8%EB%9F%89%EC%82%AC%EC%97%85%EC%86%8C" title="이문차량사업소">이문차량사업소</a> 건설 관계로 <a href="/wiki/%ED%8F%90%EC%97%AD" title="폐역">폐역</a>된 <a href="/wiki/%EB%A7%9D%EC%9A%B0%EC%84%A0" title="망우선">망우선</a> <a class="mw-redirect" href="/wiki/%EC%9D%B4%EB%AC%B8%EC%97%AD" title="이문역">이문역</a>과의 명칭 중복에 따른 혼란을 피하기 위해서이다.</p>
