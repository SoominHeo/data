<p><b>신일수</b>(申一守, <a href="/wiki/1994%EB%85%84" title="1994년">1994년</a> <a href="/wiki/9%EC%9B%94_4%EC%9D%BC" title="9월 4일">9월 4일</a> ~ )은 <a href="/wiki/%EB%8C%80%ED%95%9C%EB%AF%BC%EA%B5%AD" title="대한민국">대한민국</a>의 <a href="/wiki/%EC%B6%95%EA%B5%AC" title="축구">축구</a> 선수이며 포지션은 <a class="mw-redirect" href="/wiki/%EC%88%98%EB%B9%84%ED%98%95_%EB%AF%B8%EB%93%9C%ED%95%84%EB%8D%94" title="수비형 미드필더">수비형 미드필더</a>이다. 현재 <a href="/wiki/%EB%8C%80%ED%95%9C%EB%AF%BC%EA%B5%AD" title="대한민국">대한민국</a> <a href="/wiki/K%EB%A6%AC%EA%B7%B8_%EC%B1%8C%EB%A6%B0%EC%A7%80" title="K리그 챌린지">K리그 챌린지</a> <a href="/wiki/%EC%84%9C%EC%9A%B8_%EC%9D%B4%EB%9E%9C%EB%93%9C_FC" title="서울 이랜드 FC">서울 이랜드 FC</a>에서 활약하고 있다.</p>

<h2><span class="mw-headline" id=".EC.B6.95.EA.B5.AC.EC.9D.B8_.EA.B2.BD.EB.A0.A5">축구인 경력</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%8B%A0%EC%9D%BC%EC%88%98&amp;action=edit&amp;section=1" title="부분 편집: 축구인 경력">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<h3><span class="mw-headline" id=".EA.B3.A0.EB.A0.A4.EB.8C.80.ED.95.99.EA.B5.90">고려대학교</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%8B%A0%EC%9D%BC%EC%88%98&amp;action=edit&amp;section=2" title="부분 편집: 고려대학교">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p><a href="/wiki/%EA%B3%A0%EB%A0%A4%EB%8C%80%ED%95%99%EA%B5%90_%EC%B6%95%EA%B5%AC%EB%B6%80" title="고려대학교 축구부">고려대학교</a>에서 2014년 <a class="new" href="/w/index.php?title=%EC%B6%98%EA%B3%84%EB%8C%80%ED%95%99%EC%97%B0%EB%A7%B9%EC%A0%84&amp;action=edit&amp;redlink=1" title="춘계대학연맹전 (없는 문서)">춘계대학연맹전</a>, <a href="/wiki/U%EB%A6%AC%EA%B7%B8" title="U리그">U리그</a> 4권역 대회 등에서 우승컵을 들어올렸다. 이를 바탕으로 <a href="/wiki/%EB%8C%80%ED%95%9C%EB%AF%BC%EA%B5%AD_U-23_%EC%B6%95%EA%B5%AC_%EA%B5%AD%EA%B0%80%EB%8C%80%ED%91%9C%ED%8C%80" title="대한민국 U-23 축구 국가대표팀">대한민국 U-23 축구 국가대표팀</a>에 발탁되어 툴롱컵에 참가하였다. <sup class="reference" id="cite_ref-1"><a href="#cite_note-1">[1]</a></sup> 2학년을 마친 뒤 프로팀을 알아보기 시작하였고, <a href="/wiki/%EC%84%9C%EC%9A%B8_%EC%9D%B4%EB%9E%9C%EB%93%9C_FC" title="서울 이랜드 FC">서울 이랜드 FC</a>로 입단하였다.</p>
