<p><b>신정왕후</b>(神貞王后, <a href="/wiki/1809%EB%85%84" title="1809년">1809년</a> <a href="/wiki/1%EC%9B%94_21%EC%9D%BC" title="1월 21일">1월 21일</a>/<a href="/wiki/1808%EB%85%84" title="1808년">1808년</a> <a href="/wiki/%EC%9D%8C%EB%A0%A5_12%EC%9B%94_6%EC%9D%BC" title="음력 12월 6일">음력 12월 6일</a> ~ <a href="/wiki/1890%EB%85%84" title="1890년">1890년</a> <a href="/wiki/6%EC%9B%94_4%EC%9D%BC" title="6월 4일">6월 4일</a>/<a href="/wiki/%EC%9D%8C%EB%A0%A5_4%EC%9B%94_17%EC%9D%BC" title="음력 4월 17일">음력 4월 17일</a>)는 <a href="/wiki/%EC%A1%B0%EC%84%A0" title="조선">조선</a>의 추존왕 <a href="/wiki/%EC%A1%B0%EC%84%A0_%EB%AC%B8%EC%A1%B0" title="조선 문조">문조</a>의 아내이며 <a href="/wiki/%EC%A1%B0%EC%84%A0_%ED%97%8C%EC%A2%85" title="조선 헌종">헌종</a>의 어머니이다. 정식시호는 <b>효유헌성선경정인자혜홍덕순화문광원성숙렬명수협천융목수령희강현정휘안흠륜홍경태운창복희상의모예헌돈장계지경훈철범신정왕후</b>(孝裕獻聖宣敬正仁慈惠弘德純化文光元成肅烈明粹協天隆穆壽寧禧康顯定徽安欽倫洪慶泰運昌福熙祥懿謨睿憲敦章啓祉景勳哲範神貞王后)이다. 정식 시호가 56자로 조선의 왕후 가운데 가장 긴 시호를 가지고 있는 왕후이기도 하다. 풍은부원군(豊恩府院君) <a href="/wiki/%EC%A1%B0%EB%A7%8C%EC%98%81" title="조만영">조만영</a>과 덕안부부인(德安府夫人) 송씨(宋氏) 사이에서 태어났다.</p>

<p>1809년 1월 21일(음력 1808년 12월 6일) 탄생 하여. <a href="/wiki/1819%EB%85%84" title="1819년">1819년</a>, <a href="/wiki/%EC%99%95%EC%84%B8%EC%9E%90" title="왕세자">왕세자</a>였던 <a class="mw-redirect" href="/wiki/%ED%9A%A8%EB%AA%85%EC%84%B8%EC%9E%90" title="효명세자">효명세자</a>와 혼례를 올리고 <a class="mw-redirect" href="/wiki/%EC%84%B8%EC%9E%90%EB%B9%88" title="세자빈">세자빈</a>이 되었다.<sup class="reference" id="cite_ref-1"><a href="#cite_note-1">[1]</a></sup> <a href="/wiki/1827%EB%85%84" title="1827년">1827년</a> 원손(훗날의 헌종)을 낳았다.<sup class="reference" id="cite_ref-2"><a href="#cite_note-2">[2]</a></sup> 그러나 <a href="/wiki/1830%EB%85%84" title="1830년">1830년</a>에 남편인 효명세자가 <a href="/wiki/%EB%8C%80%EB%A6%AC%EC%B2%AD%EC%A0%95" title="대리청정">대리청정</a>을 한 지 4년만에 요절하였다. <a href="/wiki/1834%EB%85%84" title="1834년">1834년</a>에 시아버지인 <a href="/wiki/%EC%A1%B0%EC%84%A0_%EC%88%9C%EC%A1%B0" title="조선 순조">순조</a>가 승하하고 아들 <a href="/wiki/%EC%A1%B0%EC%84%A0_%ED%97%8C%EC%A2%85" title="조선 헌종">헌종</a>이 왕위에 오르자 <a href="/wiki/%EC%99%95%EB%8C%80%EB%B9%84" title="왕대비">왕대비</a>가 되었다. 그러나 <a href="/wiki/1849%EB%85%84" title="1849년">1849년</a>, 헌종이 후사 없이 승하하자 헌종의 할머니이자 선왕 순조의 정비였던 <a href="/wiki/%EC%88%9C%EC%9B%90%EC%99%95%ED%9B%84" title="순원왕후">순원왕후</a>는 친가인 <a href="/wiki/%EC%95%88%EB%8F%99_%EA%B9%80%EC%94%A8" title="안동 김씨">안동 김씨</a> 세력과 결탁하여 <a class="mw-redirect" href="/wiki/%EC%A0%84%EA%B3%84%EA%B5%B0" title="전계군">전계군</a>(全溪君)의 아들 <a href="/wiki/%EC%A1%B0%EC%84%A0_%EC%B2%A0%EC%A2%85" title="조선 철종">철종</a>을 옹립하여 외가인 안동 김씨의 세도를 계속 이어나갔다.</p>
<h3><span class="mw-headline" id=".EC.99.95.EC.8B.A4_.EC.88.98.EC.9E.A5">왕실 수장</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%8B%A0%EC%A0%95%EC%99%95%ED%9B%84_(%EC%A1%B0%EC%84%A0)&amp;action=edit&amp;section=3" title="부분 편집: 왕실 수장">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p><a href="/wiki/1857%EB%85%84" title="1857년">1857년</a>, 시어머니인 <a href="/wiki/%EC%88%9C%EC%9B%90%EC%99%95%ED%9B%84" title="순원왕후">순원왕후</a>가 승하하자 <a href="/wiki/%EB%8C%80%EC%99%95%EB%8C%80%EB%B9%84" title="대왕대비">대왕대비</a>가 되었으며 <a href="/wiki/1863%EB%85%84" title="1863년">1863년</a>에 철종이 또다시 후사 없이 승하하자 당시 대왕대비로서 왕실의 최고어른이었던 신정왕후는 <a href="/wiki/%ED%9D%A5%EC%84%A0%EB%8C%80%EC%9B%90%EA%B5%B0" title="흥선대원군">흥선군</a>과 사전 모의하여 흥선군의 둘째아들인 <a href="/wiki/%EB%8C%80%ED%95%9C%EC%A0%9C%EA%B5%AD_%EA%B3%A0%EC%A2%85" title="대한제국 고종">이재황</a>(李載晃)을 익종과 자신의 양자로 입적시켜 왕위에 옹립하였는데 그가 조선의 26대 왕인 고종이다.<sup class="reference" id="cite_ref-3"><a href="#cite_note-3">[3]</a></sup></p>
<p>이후 신정왕후 자신은 고종의 법적인 어머니로서 <a href="/wiki/%EC%88%98%EB%A0%B4%EC%B2%AD%EC%A0%95" title="수렴청정">수렴청정</a>을 하였으나 얼마 되지 않아 고종의 친부로서 대원군에 책봉된 흥선대원군에게 전권을 주고 물러났다. 한편 안동 김씨 가문에 빼앗겼던 세력을 되찾아오고자 했던 신정왕후는 고종의 비를 자신의 친정 일가인 <a class="new" href="/w/index.php?title=%EC%A1%B0%EB%A9%B4%ED%98%B8&amp;action=edit&amp;redlink=1" title="조면호 (없는 문서)">조면호</a>의 딸로 정할 계획을 세웠다. 그러나 이 역시 흥선대원군의 반대로 취소해야 했다.</p>
<p><a href="/wiki/1890%EB%85%84" title="1890년">1890년</a> 6월 4일(음력 4월 17일), 83세의 나이로 <a href="/wiki/%EA%B2%BD%EB%B3%B5%EA%B6%81" title="경복궁">경복궁</a> <a class="new" href="/w/index.php?title=%ED%9D%A5%EB%B3%B5%EC%A0%84&amp;action=edit&amp;redlink=1" title="흥복전 (없는 문서)">흥복전</a>(興福殿)에서 승하하였으며 능은 <a href="/wiki/%EA%B2%BD%EA%B8%B0%EB%8F%84" title="경기도">경기도</a> <a href="/wiki/%EA%B5%AC%EB%A6%AC%EC%8B%9C" title="구리시">구리시</a> <a href="/wiki/%EB%8F%99%EA%B5%AC%EB%A6%89" title="동구릉">동구릉</a> 내에 남편 문조와 합장되어 있는 <b>수릉</b>(綏陵)이다. 승하한 지 9년 뒤인 <a href="/wiki/%EA%B4%91%EB%AC%B4_(%EC%97%B0%ED%98%B8)" title="광무 (연호)">광무</a>(光武) <a href="/wiki/3%EB%85%84" title="3년">3년</a>, 남편 효명세자가 문조익황제(文祖翼皇帝)로 추존되자 함께 <b>신정익황후</b>(神貞翼皇后)로 추존되었다.</p>
<p>한편 <a href="/wiki/%EA%B3%A0%EA%B5%AC%EB%A7%88" title="고구마">고구마</a>를 <a href="/wiki/%EC%A1%B0%EC%84%A0" title="조선">조선</a>에 들여온 <a href="/wiki/%EC%9D%B4%EC%A1%B0" title="이조">이조</a><a href="/wiki/%ED%8C%90%EC%84%9C" title="판서">판서</a> <a href="/wiki/%EC%A1%B0%EC%97%84_(%EC%A1%B0%EC%84%A0)" title="조엄 (조선)">조엄</a>은 조대비의 증조할아버지였다.</p>
