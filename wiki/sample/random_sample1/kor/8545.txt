<p><b>신정자</b>(<a href="/wiki/1980%EB%85%84" title="1980년">1980년</a> <a href="/wiki/12%EC%9B%94_11%EC%9D%BC" title="12월 11일">12월 11일</a> ~ )은 <a href="/wiki/%EB%8C%80%ED%95%9C%EB%AF%BC%EA%B5%AD" title="대한민국">대한민국</a>의 전직 농구 선수이다.</p>
<h2><span class="mw-headline" id=".EC.83.9D.EC.95.A0">생애</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%8B%A0%EC%A0%95%EC%9E%90&amp;action=edit&amp;section=1" title="부분 편집: 생애">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>마산여자중학교와 <a href="/wiki/%EB%A7%88%EC%82%B0%EC%97%AC%EC%9E%90%EA%B3%A0%EB%93%B1%ED%95%99%EA%B5%90" title="마산여자고등학교">마산여자고등학교</a>를 졸업하고 <a href="/wiki/1999%EB%85%84" title="1999년">1999년</a> <a class="mw-redirect" href="/wiki/%EC%B2%AD%EC%A3%BC_KB%EA%B5%AD%EB%AF%BC%EC%9D%80%ED%96%89_%EC%8A%A4%ED%83%80%EC%A6%88" title="청주 KB국민은행 스타즈">청주 KB국민은행 스타즈</a>에 입단했다. <a class="mw-redirect" href="/wiki/%EC%B2%AD%EC%A3%BC_KB%EA%B5%AD%EB%AF%BC%EC%9D%80%ED%96%89_%EC%8A%A4%ED%83%80%EC%A6%88" title="청주 KB국민은행 스타즈">청주 KB국민은행</a> 시절에는 벤치 멤버였으나 <a href="/wiki/2006%EB%85%84" title="2006년">2006년</a>에 <a href="/wiki/%EA%B5%AC%EB%A6%AC_KDB%EC%83%9D%EB%AA%85_%EC%9C%84%EB%84%88%EC%8A%A4" title="구리 KDB생명 위너스">구리 KDB생명 위너스</a>로 이적한 뒤 기량이 만개하기 시작했다. 탁월한 리바운드 능력을 갖춘데다가 때로는 경기 운영까지 도맡은 에이스로 성장했다. 2011~2012 시즌에 정규리그 최우수선수로 선정되었고 2012~2013 시즌 초반에는 3경기 연속 트리플 크라운이라는 대기록을 세웠다. 통산 리바운드 부문에서도 4502개로 압도적 1위에 이름을 올리며 '미녀 리바운더'로 불렸다. 국가대표에서도 <a class="mw-redirect" href="/wiki/2006%EB%85%84_%EC%95%84%EC%8B%9C%EC%95%88%EA%B2%8C%EC%9E%84" title="2006년 아시안게임">2006년 아시안게임</a>을 기점으로 3회 연속 아시안게임에 출전했고 인천에서 열린 <a class="mw-redirect" href="/wiki/2014%EB%85%84_%EC%95%84%EC%8B%9C%EC%95%88%EA%B2%8C%EC%9E%84" title="2014년 아시안게임">2014년 아시안게임</a>에서 <a href="/wiki/%ED%95%98%EC%9D%80%EC%A3%BC" title="하은주">하은주</a>와 함께 금메달을 목에 걸었다. 이 밖에도 <a href="/wiki/%ED%95%98%EA%B3%84_%EC%98%AC%EB%A6%BC%ED%94%BD" title="하계 올림픽">하계 올림픽</a>, <a href="/wiki/%ED%95%98%EA%B3%84_%EC%98%AC%EB%A6%BC%ED%94%BD" title="하계 올림픽">하계 올림픽</a> 여자 농구 최종 예선, 아시아 여자농구선수권대회 등 권위있는 대회마다 대표팀의 부름을 받았다. 그러나 프로에서는 우승과는 거리가 멀었다. 1999년 프로 데뷔 이후 여름리그, 겨울리그를 포함해 25시즌, 586경기(통산 출전 경기 횟수 1위)를 뛰었지만 우승을 한번도 하지 못했다. 전성기를 누렸던 <a href="/wiki/%EA%B5%AC%EB%A6%AC_KDB%EC%83%9D%EB%AA%85_%EC%9C%84%EB%84%88%EC%8A%A4" title="구리 KDB생명 위너스">구리 KDB생명</a> 시절에는 2010~2011 시즌 챔피언결정전에 진출했으나 <a href="/wiki/%EC%9D%B8%EC%B2%9C_%EC%8B%A0%ED%95%9C%EC%9D%80%ED%96%89_%EC%97%90%EC%8A%A4%EB%B2%84%EB%93%9C" title="인천 신한은행 에스버드">인천 신한은행 에스버드</a>에 기세에 밀리며 준우승에 만족해야했다. 2014~2015 시즌 중반에 <a href="/wiki/%EC%9D%B8%EC%B2%9C_%EC%8B%A0%ED%95%9C%EC%9D%80%ED%96%89_%EC%97%90%EC%8A%A4%EB%B2%84%EB%93%9C" title="인천 신한은행 에스버드">인천 신한은행</a>으로 이적했으나 <a href="/wiki/%EC%95%84%EC%82%B0_%EC%9A%B0%EB%A6%AC%EC%9D%80%ED%96%89_%EC%9C%84%EB%B9%84" title="아산 우리은행 위비">아산 우리은행 위비</a>에게 타이틀을 내준 후 신한은행은 더 이상 전성기 때의 위력이 나오지 않았고 결국 신한은행에서도 우승의 숙원을 풀지 못했다. 통산 출전 경기 부문 10위 안에 든 선수 가운데 유일하게 프로에서 단 한 차례도 우승을 못한 선수로 남게 되었다. <a href="/wiki/2016%EB%85%84" title="2016년">2016년</a> <a href="/wiki/4%EC%9B%94_5%EC%9D%BC" title="4월 5일">4월 5일</a>에 은퇴를 선언하면서 17년간의 현역 선수 생활을
