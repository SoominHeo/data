<p><b>신축조약(베이징 의정서)</b>(辛丑條約)은 1900년 8월 8개국 열강 연합군이 <a href="/wiki/%EC%9D%98%ED%99%94%EB%8B%A8_%EC%9A%B4%EB%8F%99" title="의화단 운동">의화단 운동</a>을 진압하고, <a href="/wiki/%EC%B2%AD%EB%82%98%EB%9D%BC" title="청나라">청나라</a> 베이징을 점령한 다음 이듬해 <a href="/wiki/1901%EB%85%84" title="1901년">1901년</a> <a href="/wiki/9%EC%9B%94_7%EC%9D%BC" title="9월 7일">9월 7일</a> 열강 세력이 <a href="/wiki/%EC%B2%AD%EB%82%98%EB%9D%BC" title="청나라">청나라</a> 정부를 압박하여 체결한 <a href="/wiki/%EB%B6%88%ED%8F%89%EB%93%B1_%EC%A1%B0%EC%95%BD" title="불평등 조약">불평등 조약</a>을 말한다. <b>베이징 의정서</b>라고도 한다. 이 조약으로 거액의 배상금을 지불하고, 수도 <a class="mw-redirect" href="/wiki/%EB%B2%A0%EC%9D%B4%EC%A7%95" title="베이징">베이징</a>에 열강 군대가 주둔한 것을 허락하였다.</p>
<h2><span class="mw-headline" id=".EA.B0.9C.EC.9A.94">개요</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%8B%A0%EC%B6%95%EC%A1%B0%EC%95%BD&amp;action=edit&amp;section=1" title="부분 편집: 개요">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p><a href="/wiki/%EC%B2%AD%EB%82%98%EB%9D%BC" title="청나라">청나라</a> 정부측 대표인 <a href="/wiki/%EC%9D%B4%ED%99%8D%EC%9E%A5" title="이홍장">이홍장</a>(李鴻章)과 <a href="/wiki/%EB%8C%80%EC%98%81_%EC%A0%9C%EA%B5%AD" title="대영 제국">대영 제국</a>·<a href="/wiki/%EB%8F%85%EC%9D%BC_%EC%A0%9C%EA%B5%AD" title="독일 제국">독일 제국</a>·<a href="/wiki/%EC%9D%BC%EB%B3%B8_%EC%A0%9C%EA%B5%AD" title="일본 제국">일본 제국</a>·<a href="/wiki/%EB%AF%B8%EA%B5%AD" title="미국">미국</a>·<a href="/wiki/%EB%9F%AC%EC%8B%9C%EC%95%84_%EC%A0%9C%EA%B5%AD" title="러시아 제국">러시아 제국</a>·<a href="/wiki/%EC%98%A4%EC%8A%A4%ED%8A%B8%EB%A6%AC%EC%95%84-%ED%97%9D%EA%B0%80%EB%A6%AC_%EC%A0%9C%EA%B5%AD" title="오스트리아-헝가리 제국">오스트리아-헝가리 제국</a>·<a href="/wiki/%ED%94%84%EB%9E%91%EC%8A%A4_%EC%A0%9C3%EA%B3%B5%ED%99%94%EA%B5%AD" title="프랑스 제3공화국">프랑스</a>·<a href="/wiki/%EC%9D%B4%ED%83%88%EB%A6%AC%EC%95%84_%EC%99%95%EA%B5%AD_(1861%EB%85%84_~_1946%EB%85%84)" title="이탈리아 왕국 (1861년 ~ 1946년)">이탈리아 왕국</a>·<a href="/wiki/%EB%84%A4%EB%8D%9C%EB%9E%80%EB%93%9C" title="네덜란드">네덜란드</a>·<a href="/wiki/%EC%8A%A4%ED%8E%98%EC%9D%B8" title="스페인">스페인</a>·<a href="/wiki/%EB%B2%A8%EA%B8%B0%EC%97%90" title="벨기에">벨기에</a> 11개국 대표가 <a href="/wiki/%EC%9D%98%ED%99%94%EB%8B%A8_%EC%9A%B4%EB%8F%99" title="의화단 운동">의화단 운동</a> 진압 이후 <a href="/wiki/1901%EB%85%84" title="1901년">1901년</a> <a href="/wiki/9%EC%9B%94_7%EC%9D%BC" title="9월 7일">9월 7일</a> <a class="mw-redirect" href="/wiki/%EB%B2%A0%EC%9D%B4%EC%A7%95" title="베이징">베이징</a>에서 체결하였다. 이 조약으로 중국에 대한 서구 열강의 통치를 강화시켰고, 청나라 조정은 이를 반대할 힘이 없었다. 총 12조이고 부칙은 19항이며, 주요 내용은 다음과 같다.</p>
<blockquote class="toccolours" style="float:none; padding: 10px 15px 10px 15px; display:table;">
<ul>
<li>청나라는 배상금을 연리 4%의 이자로 6,750만 파운드, 즉 45억 냥(거액의 배상금)을 39년내(<a href="/wiki/1940%EB%85%84" title="1940년">1940년</a>까지)에 지불하며 관세, 염세 등을 저당으로 잡는다.</li>
<li>대고포대를 해체하고 <a class="mw-redirect" href="/wiki/%EB%B2%A0%EC%9D%B4%EC%A7%95" title="베이징">베이징</a>에 공사관 구역을 설정하며 <a class="mw-redirect" href="/wiki/%EB%B2%A0%EC%9D%B4%EC%A7%95" title="베이징">베이징</a>과 산해관 사이 철도연변에 열강 군대가 주둔한다.</li>
<li>청나라는 중국인의 외세 배척 운동을 철저하게 탄압해야 하며 만일 지방관이 진압하지 못하면 즉시 파면하고 영구히 등용하지 말아야 한다.</li>
<li>청나라는 총리아문 대신 외교부를 설치해 6부의 위에 둔다.</li>
<li>기왕의 통상조약에 대해 열강이 요구하면 <a href="/wiki/%EC%B2%AD%EB%82%98%EB%9D%BC" title="청나라">청나라</a>가 이에 협상해 수정할 수 있다.</li>
<li>청나라는 황제와 대신을 <a href="/wiki/%EB%8F%85%EC%9D%BC_%EC%A0%9C%EA%B5%AD" title="독일 제국">독일 제국</a>과 <a href="/wiki/%EC%9D%BC%EB%B3%B8_%EC%A0%9C%EA%B5%AD" title="일본 제국">일본 제국</a>에 파견하여 독일 공사와 일본 서기관의 피살 사건에 대하여 사죄한다.</li>
</ul>
