<p>《<b>신학 대전</b>》(神學大全, <span style="font-size: smaller;"><a href="/wiki/%EC%98%81%EC%96%B4" title="영어">영어</a>:</span> <span lang="en" xml:lang="en">Summa Theologica, Summa Theologica, Summa</span>)(1265-1273)은 <a href="/wiki/%EC%A4%91%EC%84%B8" title="중세">중세</a>의 <a class="mw-redirect" href="/wiki/%EC%8A%A4%EC%BD%9C%EB%9D%BC" title="스콜라">스콜라</a> 학파였던 <a href="/wiki/%EA%B8%B0%EB%8F%85%EA%B5%90" title="기독교">기독교</a> 신학자 <a href="/wiki/%ED%86%A0%EB%A7%88%EC%8A%A4_%EC%95%84%ED%80%B4%EB%82%98%EC%8A%A4" title="토마스 아퀴나스">토마스 아퀴나스</a>의 대표적 저서이다.</p>
<p>숨마(Summa, 大全)라는 명칭은 <a href="/wiki/12%EC%84%B8%EA%B8%B0" title="12세기">12세기</a>의 <a class="mw-redirect" href="/wiki/%EC%8A%A4%EC%BD%9C%EB%9D%BC" title="스콜라">스콜라</a> 용어에서 시작된 것으로, <a href="/wiki/%EA%B8%B0%EB%8F%85%EA%B5%90" title="기독교">기독교</a>의 진리를 제시할 목적으로 편찬된 여러 학설의 전체적·체계적 집대성을 말한다. <a class="new" href="/w/index.php?title=%EC%84%B1_%EB%B9%85%ED%86%A0%EB%A5%B4%EC%9D%98_%ED%9B%84%EA%B3%A0&amp;action=edit&amp;redlink=1" title="성 빅토르의 후고 (없는 문서)">성 빅토르의 후고</a>에 의한 숨마가 최초의 전형이다. 토마스 아퀴나스의 《신학 대전》은 가장 완성된 대표적 작품이다. <a class="new" href="/w/index.php?title=%ED%8E%98%ED%88%AC%EB%A5%B4%EC%8A%A4_%EB%A1%AC%EB%B0%94%EB%A5%B4%EB%91%90%EC%8A%A4&amp;action=edit&amp;redlink=1" title="페투르스 롬바르두스 (없는 문서)">페투르스 롬바르두스</a>(1164 사망)에서 전형을 볼 수 있는 《신학명제집(神學命題集)》의 방법(성서나 교부 등의 저서 중에서 신학관계의 중요한 견해(센텐티아)를 모아서 변증적으로 처리하는 것)을 더욱 발전시켜, 철학·논리·정치·경제 등 인간에 관련되는 모든 지식을 유기적으로 구성하고 <a class="mw-redirect" href="/wiki/%EC%8B%A0%EC%95%99" title="신앙">신앙</a>의 빛에 비추어 학문적으로 체계화한 종합적 노작이다. 《신학 대전》은 <a href="/wiki/%EC%A4%91%EC%84%B8_%EB%8C%80%ED%95%99" title="중세 대학">중세 대학</a>의 교육활동에서 생긴 것으로, <a href="/wiki/%EC%8B%A0%ED%95%99" title="신학">신학</a>에 요구되는 최소한의 내용을 교육적 효과가 있도록 집약한 것이다.</p>
<p>13세기의 숨마는 다음의 세 목적을 갖고 있다.</p>
<ol>
<li>주어진 학문영역에 관한 지식의 에센스를 집약한다.</li>
<li>단편적 분석을 넘어 여러 대상을 유기적으로 체계화한다.</li>
<li>학생의 교육에 그 목적을 적응시킨다.</li>
</ol>
<p>이러한 목적을 갖고서 현실생활을 긍정한 이 신학적 사상 체계는 중세 성기(盛期)에 국가까지도 복종시키는 세속적 권력을 가졌던 <a class="mw-redirect" href="/wiki/%EB%A1%9C%EB%A7%88_%EA%B5%90%ED%9A%8C" title="로마 교회">로마 교회</a>의 정신적 기반이 되었다. 토마스 아퀴나스의 《신학 대전》은 3부로 구성되며 6백여 문제, 3천여 항목을 포함하고 있다. 제1부(<a href="/wiki/%EC%8B%A0" title="신">신</a>)와 제2부(<a class="mw-redirect" href="/wiki/%EC%9D%B8%EA%B0%84" title="인간">인간</a>)는 <a href="/wiki/%EC%9D%B4%ED%83%88%EB%A6%AC%EC%95%84" title="이탈리아">이탈리아</a>에서의 교수시대(1265-68)와 <a class="mw-disambig" href="/wiki/%ED%8C%8C%EB%A6%AC" title="파리">파리</a> 체재시대(1269-72)에 완성, 제3부(<a href="/wiki/%EA%B7%B8%EB%A6%AC%EC%8A%A4%EB%8F%84" title="그리스도">그리스도</a>)는 다시 이탈리아에 돌아와 살던 시대(1272-73)에 썼으나, 미완성인 채 병사하여 제자인 <a class="new" href="/w/index.php?title=%ED%94%BC%ED%8E%98%EB%A5%B4%EB%85%B8%EC%9D%98_%EB%A0%88%EA%B8%B0%EB%82%A0%EB%8F%84&amp;action=edit&amp;redlink=1" title="피페르노의 레기날도 (없는 문서)">피페르노의 레기날도</a>가 '보유(補遺)'로 완결시켰다. 토마스 아퀴나스는 각부에 많은 문제를 세우고, 각 문제를 몇 개의 항목으로 나누었다. 또 각 항목을 이론(異論), 반대이론(反對異論), 주문(主文), 이론해답(異論解答)으로 구성하였다. 곧 스콜라학의 논증법(論證法)에 따라 우선 자기의 주장과 다른 주장을 말하고, 다음에는 이에 반대하는 주장을 말하고, 끝으로 자기의 주장을 말하여 그 이유를 설명한다. 내용을 보면 <a href="/wiki/%EC%95%84%EB%A6%AC%EC%8A%A4%ED%86%A0%ED%85%94%EB%A0%88%EC%8A%A4" title="아리스토텔레스">아리스토텔레스</a> 등의 그리스 사상과 <a href="/wiki/%EA%B5%90%EB%B6%80" title="교부">교부</a>들의 기독교 사상을 종합한 대작이다.</p>
