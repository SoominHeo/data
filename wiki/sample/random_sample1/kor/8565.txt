<p><b>신현준</b>(申鉉俊, <a href="/wiki/1915%EB%85%84" title="1915년">1915년</a> <a href="/wiki/10%EC%9B%94_23%EC%9D%BC" title="10월 23일">10월 23일</a> ~ <a href="/wiki/2007%EB%85%84" title="2007년">2007년</a> <a href="/wiki/10%EC%9B%94_15%EC%9D%BC" title="10월 15일">10월 15일</a><sup class="reference" id="cite_ref-1"><a href="#cite_note-1">[1]</a></sup>)은 <a href="/wiki/%EB%8C%80%ED%95%9C%EB%AF%BC%EA%B5%AD" title="대한민국">대한민국</a>의 <a href="/wiki/%EA%B5%B0%EC%9D%B8" title="군인">군인</a>, <a href="/wiki/%EC%A0%95%EC%B9%98%EA%B0%80" title="정치가">정치가</a>, <a href="/wiki/%EC%99%B8%EA%B5%90%EA%B4%80" title="외교관">외교관</a>이다. 본관은 <a href="/wiki/%ED%8F%89%EC%82%B0_%EC%8B%A0%EC%94%A8" title="평산 신씨">평산</a>(平山)이다.</p>
<p><a href="/wiki/%EC%9D%BC%EC%A0%9C_%EA%B0%95%EC%A0%90%EA%B8%B0" title="일제 강점기">일제 강점기</a>에 <a href="/wiki/%EB%A7%8C%EC%A3%BC%EA%B5%AD%EA%B5%B0" title="만주국군">만주국군</a> 장교였으며, <a href="/wiki/1949%EB%85%84" title="1949년">1949년</a>부터 <a href="/wiki/1953%EB%85%84" title="1953년">1953년</a>까지 <a href="/wiki/%EB%8C%80%ED%95%9C%EB%AF%BC%EA%B5%AD_%ED%95%B4%EB%B3%91%EB%8C%80" title="대한민국 해병대">대한민국 해병대</a>의 초대 사령관을 역임하였다.</p>

<h2><span class="mw-headline" id=".EC.83.9D.EC.95.A0">생애</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%8B%A0%ED%98%84%EC%A4%80_(%EA%B5%B0%EC%9D%B8)&amp;action=edit&amp;section=1" title="부분 편집: 생애">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>원적지는 <a href="/wiki/%EA%B2%BD%EC%83%81%EB%B6%81%EB%8F%84" title="경상북도">경상북도</a> <a href="/wiki/%EA%B9%80%EC%B2%9C%EC%8B%9C" title="김천시">김천군</a>이며 역시 <a href="/wiki/%EA%B2%BD%EC%83%81%EB%B6%81%EB%8F%84" title="경상북도">경상북도</a> <a href="/wiki/%EA%B9%80%EC%B2%9C%EC%8B%9C" title="김천시">김천군</a> 출생이다. 부모를 따라 일찍 <a href="/wiki/%EB%91%A5%EB%B2%A0%EC%9D%B4" title="둥베이">만주</a>로 이주하여 <a href="/wiki/%ED%95%98%EC%96%BC%EB%B9%88_%EC%8B%9C" title="하얼빈 시">하얼빈</a>에서 소학교를 졸업한 뒤 <a href="/wiki/%EC%9D%BC%EB%B3%B8" title="일본">일본</a>이 만주에 세운 <a class="mw-redirect" href="/wiki/%EB%B4%89%EC%B2%9C%EA%B5%B0%EA%B4%80%ED%95%99%EA%B5%90" title="봉천군관학교">봉천군관학교</a>에 <a href="/wiki/1936%EB%85%84" title="1936년">1936년</a> 4월 입학하였다. 졸업한 후에는 <a href="/wiki/%EB%A7%8C%EC%A3%BC%EA%B5%AD%EA%B5%B0" title="만주국군">만주국군</a> 대위로 근무했으며, 이때 <a href="/wiki/%EB%B0%95%EC%A0%95%ED%9D%AC" title="박정희">박정희</a>보다 상관으로 같은 부대에 있었다.</p>
<p>광복 후 잠시 <a class="mw-redirect" href="/wiki/%EA%B4%91%EB%B3%B5%EA%B5%B0" title="광복군">광복군</a>에 편입되었다가 귀국하여 <a href="/wiki/1946%EB%85%84" title="1946년">1946년</a> <a class="mw-redirect" href="/wiki/%EC%A1%B0%EC%84%A0%ED%95%B4%EC%95%88%EA%B2%BD%EB%B9%84%EB%8C%80" title="조선해안경비대">조선해안경비대</a>에 입대한 뒤 <a href="/wiki/%EB%8C%80%ED%95%9C%EB%AF%BC%EA%B5%AD_%ED%95%B4%EA%B5%B0" title="대한민국 해군">해군</a> 중위로 임관하였다. 같은 해 인천기지사령관을 거쳐 <a href="/wiki/1948%EB%85%84" title="1948년">1948년</a>에는 진해특설기지 참모장에 임명되었으며, 1949년 <a href="/wiki/4%EC%9B%94_15%EC%9D%BC" title="4월 15일">4월 15일</a>에 약 380명의 병력으로 해병대가 창설될 때 초대사령관을 맡았다. 당시 계급은 중령이었다. 신현준은 박정희를 비롯하여, <a href="/wiki/%EC%A0%95%EC%9D%BC%EA%B6%8C" title="정일권">정일권</a>, <a href="/wiki/%EB%B0%B1%EC%84%A0%EC%97%BD" title="백선엽">백선엽</a>, <a href="/wiki/%EC%9D%B4%ED%95%9C%EB%A6%BC" title="이한림">이한림</a>, <a href="/wiki/%EA%B9%80%EC%84%9D%EB%B2%94_(%EA%B5%B0%EC%9D%B8)" title="김석범 (군인)">김석범</a>, <a href="/wiki/%EC%9B%90%EC%9A%A9%EB%8D%95" title="원용덕">원용덕</a>, <a href="/wiki/%EA%B9%80%EC%B0%BD%EB%A3%A1" title="김창룡">김창룡</a> 등과 함께 국군 내의 대표적인 만주군 인맥이었다<sup class="reference" id="cite_ref-2"><a href="#cite_note-2">[2]</a></sup>.</p>
<p><a href="/wiki/%ED%95%9C%EA%B5%AD_%EC%A0%84%EC%9F%81" title="한국 전쟁">한국 전쟁</a> 중 <a href="/wiki/%EC%9D%B8%EC%B2%9C_%EC%83%81%EB%A5%99_%EC%9E%91%EC%A0%84" title="인천 상륙 작전">인천 상륙 작전</a>과 도솔산 전투 등에서 해병대가 전과를 올리는 데 공을 세웠고, 1953년 해병대 사령관 임기를 마친 뒤 새로 창설된 해병대 제1여단(현 해병대 제1상륙사단)의 초대 여단장을 맡았다.</p>
<p><a href="/wiki/%EB%8C%80%ED%95%9C%EB%AF%BC%EA%B5%AD_%EA%B5%AD%EB%B0%A9%EB%B6%80" title="대한민국 국방부">국방부</a> 차관보를 거친 뒤 <a href="/wiki/1961%EB%85%84" title="1961년">1961년</a> 해병 중장으로 예편하였으며, 이후 초대 <a href="/wiki/%EB%AA%A8%EB%A1%9C%EC%BD%94" title="모로코">모로코</a> 대사와 초대 <a class="mw-redirect" href="/wiki/%EB%B0%94%ED%8B%B0%EC%B9%B8" title="바티칸">바티칸</a> 대사를 지냈다. 제5대 <a class="mw-redirect" href="/wiki/%EC%84%B8%EA%B3%84%EB%B0%98%EA%B3%B5%EC%97%B0%EB%A7%B9" title="세계반공연맹">세계반공연맹</a> 사무총장도 역임했다.</p>
<p>만주군 장교 근무 경력으로 인해 <a href="/wiki/2008%EB%85%84" title="2008년">2008년</a> <a href="/wiki/%EB%AF%BC%EC%A1%B1%EB%AC%B8%EC%A0%9C%EC%97%B0%EA%B5%AC%EC%86%8C" title="민족문제연구소">민족문제연구소</a>에서 <a href="/wiki/%EC%B9%9C%EC%9D%BC%EC%9D%B8%EB%AA%85%EC%82%AC%EC%A0%84" title="친일인명사전">친일인명사전</a>에 수록하기 위해 정리한 <a class="mw-redirect" href="/wiki/%EB%AF%BC%EC%A1%B1%EB%AC%B8%EC%A0%9C%EC%97%B0%EA%B5%AC%EC%86%8C%EC%9D%98_%EC%B9%9C%EC%9D%BC%EC%9D%B8%EB%AA%85%EC%82%AC%EC%A0%84_%EC%88%98%EB%A1%9D%EC%98%88%EC%A0%95%EC%9E%90_%EB%AA%85%EB%8B%A8" title="민족문제연구소의 친일인명사전 수록예정자 명단">민족문제연구소의 친일인명사전 수록예정자 명단</a> 중 군 부문에 포함되었으며 <a href="/wiki/2009%EB%85%84" title="2009년">2009년</a> <a href="/wiki/%EB%8C%80%ED%95%9C%EB%AF%BC%EA%B5%AD_%EC%B9%9C%EC%9D%BC%EB%B0%98%EB%AF%BC%EC%A1%B1%ED%96%89%EC%9C%84%EC%A7%84%EC%83%81%EA%B7%9C%EB%AA%85%EC%9C%84%EC%9B%90%ED%9A%8C" title="대한민국 친일반민족행위진상규명위원회">친일반민족행위진상규명위원회</a>가 발표한 <a href="/wiki/%EC%B9%9C%EC%9D%BC%EB%B0%98%EB%AF%BC%EC%A1%B1%ED%96%89%EC%9C%84_705%EC%9D%B8_%EB%AA%85%EB%8B%A8" title="친일반민족행위 705인 명단">친일반민족행위 705인 명단</a>에도 포함되었다.</p>
<p>일제 강점기 동안의 박정희의 행적을 둘러싼 논란 중 박정희가 비밀리에 광복군 활동을 했다는 설이 있으나, 박정희와 함께 근무했던 신현준이 광복군의 존재 자체를 광복 후 <a class="mw-redirect" href="/wiki/%EB%B2%A0%EC%9D%B4%EC%A7%95" title="베이징">베이징</a>으로 이동할 때까지 전혀 모르고 있었다고 증언한 것이 이러한 주장에 대한 중요한 반박의 근거로 사용되고 있다.<sup class="reference" id="cite_ref-3"><a href="#cite_note-3">[3]</a></sup></p>
