<p><b>신현준</b>(申鉉濬, <a href="/wiki/1968%EB%85%84" title="1968년">1968년</a> <a href="/wiki/10%EC%9B%94_28%EC%9D%BC" title="10월 28일">10월 28일</a> - )은 <a href="/wiki/%EB%8C%80%ED%95%9C%EB%AF%BC%EA%B5%AD" title="대한민국">대한민국</a>의 <a href="/wiki/%EB%B0%B0%EC%9A%B0" title="배우">배우</a>이다. 대표작으로 <a href="/wiki/%EC%98%81%ED%99%94" title="영화">영화</a> 《<a href="/wiki/%EC%9E%A5%EA%B5%B0%EC%9D%98_%EC%95%84%EB%93%A4" title="장군의 아들">장군의 아들</a>》, 《<a href="/wiki/%EB%8B%AC%EB%A7%88%EC%95%BC,_%EC%84%9C%EC%9A%B8_%EA%B0%80%EC%9E%90" title="달마야, 서울 가자">달마야, 서울 가자</a>》, <a href="/wiki/%ED%85%94%EB%A0%88%EB%B9%84%EC%A0%84_%EB%93%9C%EB%9D%BC%EB%A7%88" title="텔레비전 드라마">텔레비전 드라마</a> 《<a href="/wiki/%EC%B9%B4%EC%9D%B8%EA%B3%BC_%EC%95%84%EB%B2%A8_(2009%EB%85%84_%EB%93%9C%EB%9D%BC%EB%A7%88)" title="카인과 아벨 (2009년 드라마)">카인과 아벨</a>》, 《<a href="/wiki/%EB%B0%94%EB%9E%8C%EC%9D%98_%EC%95%84%EB%93%A4" title="바람의 아들">바람의 아들</a>》 등이 있다.</p>

<p>신현준은 <a href="/wiki/1968%EB%85%84" title="1968년">1968년</a> <a href="/wiki/10%EC%9B%94_28%EC%9D%BC" title="10월 28일">10월 28일</a> 태어나 <a class="mw-redirect" href="/wiki/%EC%97%AC%EC%9D%98%EB%8F%84%EC%B4%88%EB%93%B1%ED%95%99%EA%B5%90" title="여의도초등학교">서울여의도초등학교</a>, <a href="/wiki/%EC%97%AC%EC%9D%98%EB%8F%84%EC%A4%91%ED%95%99%EA%B5%90" title="여의도중학교">서울여의도중학교</a>, <a href="/wiki/%EC%97%AC%EC%9D%98%EB%8F%84%EA%B3%A0%EB%93%B1%ED%95%99%EA%B5%90" title="여의도고등학교">서울 여의도고등학교</a>를 졸업하고 <a href="/wiki/%EC%97%B0%EC%84%B8%EB%8C%80%ED%95%99%EA%B5%90" title="연세대학교">연세대학교</a> 교육과학대학 체육교육학과에서 학사를 취득했으며, <a href="/wiki/1990%EB%85%84" title="1990년">1990년</a> 영화 <a href="/wiki/%EC%9E%A5%EA%B5%B0%EC%9D%98_%EC%95%84%EB%93%A4" title="장군의 아들">장군의 아들</a>로 데뷔하였다. 소속사는 HJ 아티스츠로 되어 있으며 2010년 3월부터 서울 <a class="mw-redirect" href="/wiki/%EC%9D%B8%EB%8D%95%EB%8C%80%ED%95%99" title="인덕대학">인덕대학</a>에서 방송연예학과 전임교수 발탁이 되면서 연기활동과 더불어 후학 양성을 하고 있다. <a href="/wiki/2013%EB%85%84" title="2013년">2013년</a> <a href="/wiki/5%EC%9B%94_26%EC%9D%BC" title="5월 26일">5월 26일</a> 12살 연하의 신부 김경미씨와 결혼식을 올렸다. 2016년 4월 4일에는 아들이 태어났다.</p>
