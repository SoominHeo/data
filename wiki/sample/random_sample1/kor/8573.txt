<h1 class="firstHeading" id="firstHeading" lang="ko">실리카 젤</h1>

<p><b>실리카 젤</b>(Silica gel)은 <a class="mw-redirect" href="/wiki/%EA%B7%9C%EC%82%B0%EB%82%98%ED%8A%B8%EB%A5%A8" title="규산나트륨">규산나트륨</a>의 <a href="/wiki/%EC%88%98%EC%9A%A9%EC%95%A1" title="수용액">수용액</a>을 <a href="/wiki/%EC%82%B0" title="산">산</a>으로 처리하여 만들어지는, <a href="/wiki/%EA%B7%9C%EC%86%8C" title="규소">규소</a>와 <a href="/wiki/%EC%82%B0%EC%86%8C" title="산소">산소</a>가 주 성분인 투명한 낱알 모양의 다공성 물질이다.</p>
<p>다공질 구조로 인해 <a href="/wiki/%EA%B2%89%EB%84%93%EC%9D%B4" title="겉넓이">겉넓이</a>가 약 800m²/g로 매우 넓어 수분을 흡수하는 효과가 뛰어나고 인체에 독성이 없어 주로 식품의 포장용 제습제로 사용되며, <a class="new" href="/w/index.php?title=%EC%9C%A0%EA%B8%B0_%EC%9A%A9%EB%A7%A4&amp;action=edit&amp;redlink=1" title="유기 용매 (없는 문서)">유기 용매</a>를 이용한 <a href="/wiki/%ED%81%AC%EB%A1%9C%EB%A7%88%ED%86%A0%EA%B7%B8%EB%9E%98%ED%94%BC" title="크로마토그래피">크로마토그래피</a>의 흡착제나 촉매의 운반체로 사용되기도 한다.</p>
<p>흡습제로 사용 할 경우에는 실리카 젤의 <a href="/wiki/%EC%8A%B5%EB%8F%84" title="습도">습도</a>를 나타내기 위해서 일부 알갱이를 <a class="new" href="/w/index.php?title=%EC%97%BC%ED%99%94_%EC%BD%94%EB%B0%9C%ED%8A%B8&amp;action=edit&amp;redlink=1" title="염화 코발트 (없는 문서)">염화 코발트</a> 또는 <a class="new" href="/w/index.php?title=%EB%A9%94%ED%8B%B8_%EB%B0%94%EC%9D%B4%EC%98%AC%EB%A0%9B&amp;action=edit&amp;redlink=1" title="메틸 바이올렛 (없는 문서)">메틸 바이올렛</a>등으로 염색한다. 염색된 실리카 젤이 수분을 흡수하면 염화 코발트의 경우 <a class="mw-redirect" href="/wiki/%ED%8C%8C%EB%9E%80%EC%83%89" title="파란색">파란색</a>에서 <a class="mw-redirect" href="/wiki/%EB%B6%84%ED%99%8D%EC%83%89" title="분홍색">분홍색</a>으로, 메틸 바이올렛의 경우 <a class="mw-redirect" href="/wiki/%EC%A3%BC%ED%99%A9%EC%83%89" title="주황색">주황색</a>에서 <a class="mw-redirect" href="/wiki/%EB%85%B9%EC%83%89" title="녹색">녹색</a>으로 색상이 변해 수분을 검출할 수 있다. 습기를 머금어 색이 변한 실리카 젤은 가열로 건조해서 다시 사용할 수 있다.</p>
<h2><span class="mw-headline" id=".EC.9C.84.ED.97.98.EC.84.B1">위험성</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%8B%A4%EB%A6%AC%EC%B9%B4_%EC%A0%A4&amp;action=edit&amp;section=1" title="부분 편집: 위험성">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>일반적으로 실리카 젤은 무독성, 불연성의 안전한 물질로 널리 알려져 있다. 하지만, 공업용 실리카 젤<sup class="reference" id="cite_ref-1"><a href="#cite_note-1">[1]</a></sup>에서 발생하는 규산 먼지는 눈과 피부와 호흡기에 염증을 일으키고 심각할 경우에는 <a href="/wiki/%EA%B7%9C%ED%8F%90%EC%A6%9D" title="규폐증">규폐증</a>을 일으킬 수 있으므로, 취급할 때 주의해야 한다.<sup class="reference" id="cite_ref-2"><a href="#cite_note-2">[2]</a></sup> 실리카 젤 속의 습도를 나타내기 위해 들어있는 파란색 또는 붉은색으로 변하는 실리카 젤에 염색된 염화 코발트는 독성이 있으며, 발암물질로 널리 알려져 있다. 실리카 젤을 재사용하기위해 가열할 때 염색되어있던 염화 코발트가 기화하여 독성 염화코발트기체를 형성할 수있다.</p>
