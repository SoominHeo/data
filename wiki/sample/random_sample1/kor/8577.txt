<p><b>실버 서비스</b>(<span style="font-size: smaller;"><a href="/wiki/%EC%98%81%EC%96%B4" title="영어">영어</a>:</span> <span lang="en" xml:lang="en">Silver Service</span>)는 <a href="/wiki/%EC%8B%A4%EB%B2%84_%EB%A9%94%ED%85%8C%EC%98%A4" title="실버 메테오">실버 메테오</a>와 <a href="/wiki/%EC%8B%A4%EB%B2%84_%EC%8A%A4%ED%83%80_(%EC%95%94%ED%8A%B8%EB%9E%99_%EC%97%B4%EC%B0%A8)" title="실버 스타 (암트랙 열차)">실버 스타</a>의 2개 노선을 통합한 명칭으로 <a href="/wiki/%EB%89%B4%EC%9A%95" title="뉴욕">뉴욕</a>과 <a href="/wiki/%EB%A7%88%EC%9D%B4%EC%95%A0%EB%AF%B8" title="마이애미">마이애미</a>를 잇는 철도이다.<sup class="reference" id="cite_ref-1"><a href="#cite_note-1">[1]</a></sup></p>
<p>실버 서비스는 <a class="new" href="/w/index.php?title=%EB%A1%9D%ED%82%A4%EB%A7%88%EC%9A%B4%ED%8A%B8_(%EB%85%B8%EC%8A%A4%EC%BA%90%EB%A1%A4%EB%9D%BC%EC%9D%B4%EB%82%98_%EC%A3%BC)&amp;action=edit&amp;redlink=1" title="록키마운트 (노스캐롤라이나 주) (없는 문서)">록키마운트</a>, <a class="new" href="/w/index.php?title=%EC%84%9C%EB%B0%B0%EB%84%88_(%ED%94%8C%EB%A1%9C%EB%A6%AC%EB%8B%A4_%EC%A3%BC)&amp;action=edit&amp;redlink=1" title="서배너 (플로리다 주) (없는 문서)">서배너</a>에서 분기한다. <a href="/wiki/%EC%8B%A4%EB%B2%84_%EB%A9%94%ED%85%8C%EC%98%A4" title="실버 메테오">실버 메테오</a> 노선의 경우 <a href="/wiki/%ED%8E%98%EC%9D%B4%EC%97%A3%EB%B9%8C_(%EB%85%B8%EC%8A%A4%EC%BA%90%EB%A1%A4%EB%9D%BC%EC%9D%B4%EB%82%98_%EC%A3%BC)" title="페이엣빌 (노스캐롤라이나 주)">페이엣빌</a>, <a href="/wiki/%ED%94%8C%EB%A1%9C%EB%A0%8C%EC%8A%A4_(%EC%82%AC%EC%9A%B0%EC%8A%A4%EC%BA%90%EB%A1%A4%EB%9D%BC%EC%9D%B4%EB%82%98_%EC%A3%BC)" title="플로렌스 (사우스캐롤라이나 주)">플로렌스</a>, <a href="/wiki/%EC%B0%B0%EC%8A%A4%ED%84%B4_(%EC%82%AC%EC%9A%B0%EC%8A%A4%EC%BA%90%EB%A1%A4%EB%9D%BC%EC%9D%B4%EB%82%98_%EC%A3%BC)" title="찰스턴 (사우스캐롤라이나 주)">찰스턴</a>을 경유하는 반면 <a href="/wiki/%EC%8B%A4%EB%B2%84_%EC%8A%A4%ED%83%80_(%EC%95%94%ED%8A%B8%EB%9E%99_%EC%97%B4%EC%B0%A8)" title="실버 스타 (암트랙 열차)">실버 스타</a>의 경우 <a href="/wiki/%EB%A1%A4%EB%A6%AC_(%EB%85%B8%EC%8A%A4%EC%BA%90%EB%A1%A4%EB%9D%BC%EC%9D%B4%EB%82%98_%EC%A3%BC)" title="롤리 (노스캐롤라이나 주)">롤리</a>, <a class="mw-redirect" href="/wiki/%EC%BA%90%EB%A6%AC_(%EB%85%B8%EC%8A%A4%EC%BA%90%EB%A1%A4%EB%9D%BC%EC%9D%B4%EB%82%98_%EC%A3%BC)" title="캐리 (노스캐롤라이나 주)">캐리</a>, <a href="/wiki/%EC%BA%A0%EB%8D%98_(%EC%82%AC%EC%9A%B0%EC%8A%A4%EC%BA%90%EB%A1%A4%EB%9D%BC%EC%9D%B4%EB%82%98_%EC%A3%BC)" title="캠던 (사우스캐롤라이나 주)">캠던</a>, <a href="/wiki/%EC%BB%AC%EB%9F%BC%EB%B9%84%EC%95%84_(%EC%82%AC%EC%9A%B0%EC%8A%A4%EC%BA%90%EB%A1%A4%EB%9D%BC%EC%9D%B4%EB%82%98_%EC%A3%BC)" title="컬럼비아 (사우스캐롤라이나 주)">컬럼비아</a>를 경유한다. 또한 실버 스타 노선의 경우 <a href="/wiki/%ED%83%AC%ED%8C%8C" title="탬파">탬파</a>를 경유한다.</p>
<p><a href="/wiki/1996%EB%85%84" title="1996년">1996년</a>부터 <a href="/wiki/2002%EB%85%84" title="2002년">2002년</a>까지 <a href="/wiki/%EC%95%94%ED%8A%B8%EB%9E%99" title="암트랙">암트랙</a>이 세 번째로 <a href="/wiki/%ED%8C%94%EB%A9%94%ED%86%A0_(%EC%97%B4%EC%B0%A8)" title="팔메토 (열차)">실버 팜</a> 열차를 운행했다.<sup class="reference" id="cite_ref-2"><a href="#cite_note-2">[2]</a></sup></p>
<p><a href="/wiki/1970%EB%85%84" title="1970년">1970년</a>대에 <a href="/wiki/%EC%95%94%ED%8A%B8%EB%9E%99" title="암트랙">암트랙</a>은 <b>플로리다 플레트</b>(<span style="font-size: smaller;"><a href="/wiki/%EC%98%81%EC%96%B4" title="영어">영어</a>:</span> <span lang="en" xml:lang="en">Florida Fleet</span>)란 브랜드를 사용했다. <a href="/wiki/%EC%8B%A4%EB%B2%84_%EB%A9%94%ED%85%8C%EC%98%A4" title="실버 메테오">실버 메테오</a>와 <a href="/wiki/%EC%8B%A4%EB%B2%84_%EC%8A%A4%ED%83%80_(%EC%95%94%ED%8A%B8%EB%9E%99_%EC%97%B4%EC%B0%A8)" title="실버 스타 (암트랙 열차)">실버 스타</a>를 비롯해 현재 없어진 <a class="new" href="/w/index.php?title=%EC%B1%94%ED%94%BC%EC%96%B8_(%EC%97%B4%EC%B0%A8)&amp;action=edit&amp;redlink=1" title="챔피언 (열차) (없는 문서)">챔피언</a>이란 열차도 있다.<sup class="reference" id="cite_ref-3"><a href="#cite_note-3">[3]</a></sup></p>
