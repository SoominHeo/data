<p><b>CJ-10</b>은 <a href="/wiki/%EC%A4%91%ED%99%94%EC%9D%B8%EB%AF%BC%EA%B3%B5%ED%99%94%EA%B5%AD" title="중화인민공화국">중화인민공화국</a>의 순항미사일이다. <b>창젠-10</b>의 영어 약자이다.</p>
<p><a href="/wiki/%ED%84%B0%EB%B3%B4%ED%8C%AC" title="터보팬">터보팬</a> 엔진을 사용하는 아음속 순항미사일로서, 속도 마하 0.75, 사거리 1,500 km이다. CJ-10 3발이 장전된 8륜 구동 차량에서 발사되며, Tu-16을 중국에서 라이센스 생산한 H-6K 폭격기에서 CJ-10A 6발이 발사된다. H-6K 폭격기에서 발사될 경우 사거리가 3,000 km에 달하여, 하와이 미군을 공격할 수 있다.</p>
<p>2009년 10월 1일 베이징 천안문 광장에서 거행된 건국 60주년 기념 열병식에서 3개의 연속 발사관으로 이뤄져 있으며 사거리가 1500 km인 크루즈 미사일 '창젠(長劍)-10'을 선보였다. 예전에는 공개된 적이 없다.<sup class="reference" id="cite_ref-1"><a href="#cite_note-1">[1]</a></sup></p>
<p>2010년 중국은 <a class="mw-redirect" href="/wiki/%EB%A5%98%EC%A0%80%EC%9A%B0" title="류저우">류저우</a>에 <a class="mw-redirect" href="/wiki/%EB%8C%80%EB%A7%8C" title="대만">대만</a>, <a href="/wiki/%EB%B2%A0%ED%8A%B8%EB%82%A8" title="베트남">베트남</a>, <a class="mw-disambig" href="/wiki/%EC%98%A4%ED%82%A4%EB%82%98%EC%99%80" title="오키나와">오키나와</a>, <a href="/wiki/%EC%9D%B8%EB%8F%84" title="인도">인도</a> 북부를 겨냥한 순항미사일 창젠(長劍)-10을 배치한 것으로 보인다고 보도되었다.<sup class="reference" id="cite_ref-2"><a href="#cite_note-2">[2]</a></sup></p>
<p>중국에는 장거리 순항미사일로 창젠-10 이외에도 <a href="/wiki/DH-10" title="DH-10">DH-10</a>, <a class="new" href="/w/index.php?title=CF-2&amp;action=edit&amp;redlink=1" title="CF-2 (없는 문서)">CF-2</a>, <a class="new" href="/w/index.php?title=CF-1&amp;action=edit&amp;redlink=1" title="CF-1 (없는 문서)">CF-1</a>, <a href="/wiki/HN-3" title="HN-3">HN-3</a>, <a class="mw-redirect" href="/wiki/HN-2" title="HN-2">HN-2</a>, <a class="mw-redirect" href="/wiki/HN-1" title="HN-1">HN-1</a> 장거리 순항미사일이 있다.</p>
<h2><span class="mw-headline" id=".EA.B0.9C.EB.B0.9C">개발</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=CJ-10&amp;action=edit&amp;section=1" title="부분 편집: 개발">원본 편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>CJ-10 미사일은 이전의 <a href="/wiki/HN-3" title="HN-3">HN-3</a> 미사일을 개량한 것이다. 새로운 디자인에는 우크라이나의 협조로, 소련의 <a href="/wiki/Kh-55" title="Kh-55">Kh-55</a> 순항미사일이 참고되었다고 한다.</p>
<p>중국은 파키스탄과 아프가니스탄에서 불발탄인 미국의 <a class="mw-redirect" href="/wiki/%ED%86%A0%EB%A7%88%ED%98%B8%ED%81%AC_%EB%AF%B8%EC%82%AC%EC%9D%BC" title="토마호크 미사일">토마호크 미사일</a> 다수를 획득했다. 당시 미국은 1999년 탈레반에 대한 공격을 가했었다. CJ-10 미사일 개발에 이용되었을 것으로 보고 있다.</p>
<p>영국 해군 <a class="new" href="/w/index.php?title=%EC%9D%B4%EC%95%88_%EC%9D%B4%EC%8A%A4%ED%86%A4&amp;action=edit&amp;redlink=1" title="이안 이스톤 (없는 문서)">이안 이스톤</a>(<a class="extiw" href="https://en.wikipedia.org/wiki/Ian_Easton" title="en:Ian Easton">en:Ian Easton</a>)은 CJ-10는 <a class="mw-redirect" href="/wiki/HN-2" title="HN-2">HN-2</a>와 동일하고, <a href="/wiki/HN-3" title="HN-3">HN-3</a>는 <a href="/wiki/DH-10" title="DH-10">DH-10A</a>와 동일하다고 믿는다.</p>
