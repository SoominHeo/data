<p><b>실비 바르탕</b>(<a href="/wiki/%EB%B6%88%EA%B0%80%EB%A6%AC%EC%95%84%EC%96%B4" title="불가리아어">불가리아어</a> : Силви Вартан, <a href="/wiki/%ED%94%84%EB%9E%91%EC%8A%A4%EC%96%B4" title="프랑스어">프랑스어</a> : Sylvie Vartan, <a href="/wiki/1944%EB%85%84" title="1944년">1944년</a> <a href="/wiki/8%EC%9B%94_15%EC%9D%BC" title="8월 15일">8월 15일</a> ~ )은 <a href="/wiki/%ED%94%84%EB%9E%91%EC%8A%A4" title="프랑스">프랑스</a>의 <a href="/wiki/%EA%B0%80%EC%88%98" title="가수">가수</a> 겸 <a class="mw-redirect" href="/wiki/%EC%98%81%ED%99%94_%EB%B0%B0%EC%9A%B0" title="영화 배우">영화배우</a>이며, 프랑스의 1세대 락(rock) 가수 중 하나이다. 또한 프랑스 내에서 음반이 가장 많린 가수이기도 하다.</p>

<h2><span class="mw-headline" id=".EC.83.9D.EC.95.A0">생애</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%8B%A4%EB%B9%84_%EB%B0%94%EB%A5%B4%ED%83%95&amp;action=edit&amp;section=1" title="부분 편집: 생애">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p><a href="/wiki/%EB%B6%88%EA%B0%80%EB%A6%AC%EC%95%84" title="불가리아">불가리아</a>의 <a href="/wiki/%EC%86%8C%ED%94%BC%EC%95%84" title="소피아">소피아</a>에서 외무공무원의 딸로 태어났으며, 얼마 못 가 불가리아가 <a href="/wiki/%EA%B3%B5%EC%82%B0%EC%A3%BC%EC%9D%98" title="공산주의">공산주의</a>의 억압을 받고, <a href="/wiki/%EA%B5%B0%EC%A3%BC%EC%A0%9C" title="군주제">군주제</a>가 폐지됨으로써 가족과 함께 프랑스로 망명해야 했다. <a href="/wiki/1960%EB%85%84" title="1960년">1960년</a> <a href="/wiki/%EA%B0%80%EC%88%98" title="가수">가수</a>로 첫 데뷔하였고 그녀가 <a href="/wiki/1968%EB%85%84" title="1968년">1968년</a>에 발표한 《La Maritza (마리차 강의 노래)》의 가사에서 정치적 이유로 조국을 등져야 했던 어린 시절의 회한이 드러나 있다. 그 외에도 <a href="/wiki/%ED%8A%80%EB%8B%88%EC%A7%80" title="튀니지">튀니지</a>에서 출생한 <a href="/wiki/%ED%94%84%EB%9E%91%EC%8A%A4" title="프랑스">프랑스</a>의 <a href="/wiki/%ED%94%BC%EC%95%84%EB%85%B8_%EC%97%B0%EC%A3%BC%EC%9E%90" title="피아노 연주자">피아노 연주자</a> 겸 <a href="/wiki/%EC%9E%91%EA%B3%A1%EA%B0%80" title="작곡가">작곡가</a> 미셸 로랑(Michel Laurent, 1944년 12월 24일 ~ )으로부터 《La leine de Saba(시바의 여왕)》이라는 곡을 받아 불러 히트하였다. 바르탕은 가장 많이 음반이 팔린 가수이자, <a class="new" href="/w/index.php?title=%EC%98%88%EC%98%88&amp;action=edit&amp;redlink=1" title="예예 (없는 문서)">예예</a>(yé-yé) 스타일의 작가로 활동하고 있으며, 예예 스타일의 작가 중에서 가장 강렬하고 매혹적인 것으로 여겨진다. 그녀의 공연은 때때로 공을 많이 들인 것으로 인기를 끈다. 그녀는 프랑스와 <a href="/wiki/%EC%9D%B4%ED%83%88%EB%A6%AC%EC%95%84" title="이탈리아">이탈리아</a> 텔레비전에서 많이 출연하고 있다.</p>
