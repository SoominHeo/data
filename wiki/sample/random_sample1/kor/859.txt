<p><b>씨제이이앤엠 주식회사</b>(<span style="font-size: smaller;"><a href="/wiki/%EC%98%81%EC%96%B4" title="영어">영어</a>:</span> <span lang="en">CJ E&amp;M Corporation</span>, 상표 : <b>CJ E&amp;M</b>)는 <a href="/wiki/%EB%8C%80%ED%95%9C%EB%AF%BC%EA%B5%AD" title="대한민국">대한민국</a>의 종합 콘텐츠 기업으로 <a href="/wiki/CJ%EA%B7%B8%EB%A3%B9" title="CJ그룹">CJ그룹</a>의 계열사이다.</p>
<p><a href="/wiki/2010%EB%85%84" title="2010년">2010년</a> <a href="/wiki/9%EC%9B%94_15%EC%9D%BC" title="9월 15일">9월 15일</a> <a href="/wiki/CJ%EC%98%A4%EC%87%BC%ED%95%91" title="CJ오쇼핑">㈜씨제이오쇼핑</a>의 미디어사업부문을 인적분할하여 <b>㈜오미디어홀딩스</b>(<span style="font-size: smaller;"><a href="/wiki/%EC%98%81%EC%96%B4" title="영어">영어</a>:</span> <span lang="en">O Media Holdings Corporation</span>)로 설립하였다. 같은 해 10월 15일 한국거래소 코스닥증권시장에 상장하였으며 2011년 1월 20일에 현재의 씨제이이앤엠 주식회사로 상호 변경하였다. <a href="/wiki/2011%EB%85%84" title="2011년">2011년</a> <a href="/wiki/3%EC%9B%94_1%EC%9D%BC" title="3월 1일">3월 1일</a> CJ그룹 내 미디어 계열 5개사 <a href="/wiki/%EC%98%A8%EB%AF%B8%EB%94%94%EC%96%B4" title="온미디어">㈜온미디어</a>, <a href="/wiki/CJ%EB%AF%B8%EB%94%94%EC%96%B4" title="CJ미디어">씨제이미디어㈜</a>, <a href="/wiki/CJ%EC%9D%B8%ED%84%B0%EB%84%B7" title="CJ인터넷">씨제이인터넷㈜</a>, <a href="/wiki/%EC%97%A0%EB%84%B7%EB%AF%B8%EB%94%94%EC%96%B4" title="엠넷미디어">엠넷미디어㈜</a>, <a href="/wiki/CJ%EC%97%94%ED%84%B0%ED%85%8C%EC%9D%B8%EB%A8%BC%ED%8A%B8" title="CJ엔터테인먼트">씨제이엔터테인먼트㈜</a>를 인수 합병하였다. 합병 이후 방송, 게임, 영화, 음악/공연/온라인 4개 사업부문으로 편성하여 종합적인 미디어, 엔터테인먼트 사업을 영위하고 있다.</p>
<p>대표이사 2명과 사외이사 3명, <a href="/wiki/%EC%9D%B4%EB%AF%B8%EA%B2%BD_(1958%EB%85%84)" title="이미경 (1958년)">이미경</a> <a href="/wiki/CJ_(%EA%B8%B0%EC%97%85)" title="CJ (기업)">씨제이㈜</a> 부회장을 포함한 미등기임원 25명이 재직중이며 CJ E&amp;M 총괄부회장직을 맡고 있는 CJ 이미경이 경영에 상당한 영향력을 행사며 사실상 총괄<sup class="reference" id="cite_ref-1"><a href="#cite_note-1">[1]</a></sup>하고 있다.</p>

<p>2011년 3월 <a href="/wiki/CJ%EA%B7%B8%EB%A3%B9" title="CJ그룹">CJ그룹</a>의 미디어 계열 5개사를 흡수 합병한 이후 방송, 게임, 영화, 음악/공연/온라인 사업부문으로 편성하였다.</p>

<h3><span class="mw-headline" id=".EC.98.81.ED.99.94">영화</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=CJ_E%26M&amp;action=edit&amp;section=3" title="부분 편집: 영화">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
