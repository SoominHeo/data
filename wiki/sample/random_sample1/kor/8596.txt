<h1 class="firstHeading" id="firstHeading" lang="ko">심신 문제</h1>
<div class="mw-body-content" id="bodyContent">
<div id="siteSub">위키백과, 우리 모두의 백과사전.</div>
<div id="contentSub"></div>
<div class="mw-jump" id="jump-to-nav">
					이동:					<a href="#mw-head">둘러보기</a>, 					<a href="#p-search">검색</a>
</div>
<div class="mw-content-ltr" dir="ltr" id="mw-content-text" lang="ko"><script>function mfTempOpenSection(id){var block=document.getElementById("mf-section-"+id);block.className+=" open-block";block.previousSibling.className+=" open-block";}</script><p><b>심신 문제</b>(心身問題)에 관해서는 <a href="/wiki/%EA%B3%A0%EB%8C%80_%EA%B7%B8%EB%A6%AC%EC%8A%A4" title="고대 그리스">그리스</a> 이후에 여러 가지 사고방식이 있었다. 원래의 그리스 사상에서는 <a href="/wiki/%EB%A7%88%EC%9D%8C" title="마음">마음</a>(<a href="/wiki/%EC%98%81%ED%98%BC" title="영혼">영혼</a>)은 그림자처럼 엷은 것이라 생각했다. 그런데 북방에서 들어온 <a href="/wiki/%EB%94%94%EC%98%A4%EB%8B%88%EC%86%8C%EC%8A%A4" title="디오니소스">디오니소스</a>의 생각으로는 영혼이 독립된 것이라 했다. 이 생각은 <a href="/wiki/%ED%94%BC%ED%83%80%EA%B3%A0%EB%9D%BC%EC%8A%A4" title="피타고라스">피타고라스</a>를 거쳐 <a href="/wiki/%ED%94%8C%EB%9D%BC%ED%86%A4" title="플라톤">플라톤</a>에 이르렀다.</p>
<p><a href="/wiki/%EC%A4%91%EC%84%B8" title="중세">중세</a>에는 <a class="mw-redirect" href="/wiki/%EC%8B%A0%EC%B2%B4" title="신체">신체</a>가 악의 근원이라고 생각했다. <a href="/wiki/%EA%B7%BC%EC%84%B8" title="근세">근세</a>에 들어와서 심신문제를 중심 문제로 생각한 사람은 <a class="mw-redirect" href="/wiki/%EB%8D%B0%EC%B9%B4%EB%A5%B4%ED%8A%B8" title="데카르트">데카르트</a>이다. 그는 <a href="/wiki/%EC%A0%95%EC%8B%A0" title="정신">정신</a>과 <a href="/wiki/%EB%AC%BC%EC%B2%B4" title="물체">물체</a>는 상호간에 독립한다는 <a href="/wiki/%EC%9D%B4%EC%9B%90%EB%A1%A0" title="이원론">이원론</a>을 수립했다. <a class="mw-redirect" href="/wiki/%EC%9E%90%EC%97%B0%EA%B3%84" title="자연계">자연계</a>는 연장(延長)과 운동으로 보아 완전히 기계적으로 설명되고 사고를 본성으로 삼는 정신과 교섭이 없다. 이 때문에 심신문제의 설명이 곤란해졌다. <a class="mw-redirect" href="/wiki/%EC%8A%A4%ED%94%BC%EB%85%B8%EC%9E%90" title="스피노자">스피노자</a>는 이 곤란을 해결하기 위해 심신을 실체의 표리라 생각했는데, 이는 일종의 이면설(二面說)이라고 할 수 있을 것이다.</p>
