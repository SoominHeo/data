<p><b>Ogg</b>(오그)는 <a href="/wiki/%EB%A9%80%ED%8B%B0%EB%AF%B8%EB%94%94%EC%96%B4" title="멀티미디어">멀티미디어</a> <a class="mw-redirect" href="/wiki/%EC%BB%A8%ED%85%8C%EC%9D%B4%EB%84%88_%ED%8F%AC%EB%A7%B7" title="컨테이너 포맷">컨테이너 포맷</a>이다. <a class="new" href="/w/index.php?title=%EC%86%8C%ED%94%84%ED%8A%B8%EC%9B%A8%EC%96%B4_%ED%8A%B9%ED%97%88%EA%B6%8C&amp;action=edit&amp;redlink=1" title="소프트웨어 특허권 (없는 문서)">특허권</a>으로 보호되지 않는 <a class="mw-redirect" href="/wiki/%EC%98%A4%ED%94%88_%ED%91%9C%EC%A4%80" title="오픈 표준">오픈 표준</a> 파일 형식으로 <a href="/wiki/%EB%A9%80%ED%8B%B0%EB%AF%B8%EB%94%94%EC%96%B4" title="멀티미디어">멀티미디어</a> <a class="new" href="/w/index.php?title=%EB%B9%84%ED%8A%B8%EC%8A%A4%ED%8A%B8%EB%A6%BC_%ED%98%95%EC%8B%9D&amp;action=edit&amp;redlink=1" title="비트스트림 형식 (없는 문서)">비트스트림</a>을 효율적으로 <a class="mw-redirect" href="/wiki/%EC%8A%A4%ED%8A%B8%EB%A6%AC%EB%B0%8D_%EB%AF%B8%EB%94%94%EC%96%B4" title="스트리밍 미디어">전송</a>하고 처리할 수 있게 하기 위해 <a href="/wiki/Xiph.Org_%EC%9E%AC%EB%8B%A8" title="Xiph.Org 재단">Xiph.Org 재단</a>에서 개발한 것이다. 보통 Ogg 스트림에는 Ogg <a class="mw-redirect" href="/wiki/%EB%B3%B4%EB%B9%84%EC%8A%A4" title="보비스">보비스</a> <a class="new" href="/w/index.php?title=%EC%86%8C%EB%A6%AC_%ED%8C%8C%EC%9D%BC_%ED%98%95%EC%8B%9D&amp;action=edit&amp;redlink=1" title="소리 파일 형식 (없는 문서)">소리 파일 형식</a>을 사용한 <a href="/wiki/%EC%86%8C%EB%A6%AC" title="소리">오디오</a>가 많이 담기긴 하지만, 이외에도 Ogg <a href="/wiki/%ED%85%8C%EC%98%A4%EB%9D%BC" title="테오라">테오라</a>와 같은 다른 형식도 들어 갈 수 있다.</p>
<p>Ogg 형식 파일에는 오디오와 <a href="/wiki/%EB%B9%84%EB%94%94%EC%98%A4" title="비디오">비디오</a>, 문자열 (예: 자막) 용의 몇 가지의 서로 다른 오픈 소스 <a href="/wiki/%EC%BD%94%EB%8D%B1" title="코덱">코덱</a>으로 저장된 정보를 담을 수 있다. .ogg라는 확장자의 파일은 앞서 언급한 몇 가지의 Ogg 미디어 파일 형식들 중 하나에 해당할 수 있다. 또한 무료로 사용할 수 있는 파일 형식이기 때문에, 여러 무료 및 상용 <a href="/wiki/%EB%AF%B8%EB%94%94%EC%96%B4_%ED%94%8C%EB%A0%88%EC%9D%B4%EC%96%B4" title="미디어 플레이어">미디어 플레이어</a>는 물론 다양한 <a href="/wiki/%ED%8F%AC%ED%84%B0%EB%B8%94_%EB%AF%B8%EB%94%94%EC%96%B4_%ED%94%8C%EB%A0%88%EC%9D%B4%EC%96%B4" title="포터블 미디어 플레이어">포터블 미디어 플레이어</a>에 Ogg의 여러 코덱을 내장해 사용될 수 있다.</p>
<p>현재 Ogg를 기본적으로 사용하는 프로젝트로는 <a class="mw-redirect" href="/wiki/%EB%B3%B4%EB%B9%84%EC%8A%A4" title="보비스">보비스</a>, 스픽스, 테오라등이 있고, FLAC 프로젝트도 디랙 프로젝트의 하부 프로젝트인 슈뢰딩거 프로젝트도 Ogg 포맷의 사용을 지원한다.</p>
<p>2007년 이후부터 Xiph.org 재단은 .ogg 확장자를 Ogg 보비스 오디오 파일에만 사용할 것을 권고하고 있다. 그리고 Xiph 재단은 .oga, .ogv, .ogx와 같은 새로운 파일 확장자를 만들기로 하였다. .oga는 오디오만 담고 있는 파일, .ogv는 소리를 포함하지 않는 비디오, .ogx는 멀티플렉싱 기술이 적용된 Ogg에 사용된다.<sup class="reference" id="cite_ref-1"><a href="#cite_note-1">[1]</a></sup></p>
<h2><span class="mw-headline" id=".EC.98.A4.EA.B7.B8_.EC.BD.94.EB.8D.B1">오그 코덱</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Ogg&amp;action=edit&amp;section=1" title="부분 편집: 오그 코덱">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>Ogg는 <a class="mw-redirect" href="/wiki/%EC%BB%A8%ED%85%8C%EC%9D%B4%EB%84%88_%ED%8C%8C%EC%9D%BC" title="컨테이너 파일">컨테이너 파일</a>에 불과하다. 코덱으로 인코딩된 실제 오디오 데이터 또는 비디오 데이터는 Ogg 컨테이너 안에 저장되어 있다.</p>
<p>오그는 컨테이너 포맷이므로 다양한 포맷으로 오디오와 비디오를 담을 수 있다. 그러나 오그는 의도적으로 또는 보통 다음과 같은 <a class="mw-redirect" href="/wiki/Xiph.org" title="Xiph.org">Xiph.org</a> 무료 코덱을 이용한다.</p>
