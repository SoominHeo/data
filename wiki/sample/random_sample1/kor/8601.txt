<p><b>심은경</b>(沈恩敬, <a href="/wiki/1994%EB%85%84" title="1994년">1994년</a> <a href="/wiki/5%EC%9B%94_31%EC%9D%BC" title="5월 31일">5월 31일</a> ~ )은 <a href="/wiki/%EB%8C%80%ED%95%9C%EB%AF%BC%EA%B5%AD" title="대한민국">대한민국</a>의 <a href="/wiki/%EB%B0%B0%EC%9A%B0" title="배우">배우</a>이다.그녀는 2003년 <a class="mw-redirect" href="/wiki/MBC" title="MBC">MBC</a> 드라마 《<a href="/wiki/%EB%8C%80%EC%9E%A5%EA%B8%88" title="대장금">대장금</a>》으로 데뷔하였다. 2004년 MBC 드라마 《결혼하고 싶은 여자》에서 명세빈의 아역으로 본격 연기자로 입문해 데뷔했고 이후 영화 《헨젤과 그레텔》(2007), 《<a href="/wiki/%EB%B6%88%EC%8B%A0%EC%A7%80%EC%98%A5_(%EC%98%81%ED%99%94)" title="불신지옥 (영화)">불신지옥</a>》(2009), 《<a href="/wiki/%ED%80%B4%EC%A6%88%EC%99%95" title="퀴즈왕">퀴즈왕</a>》(2010) 등을 거치며 성장했다. 심은경은 또한 《<a href="/wiki/%EC%8D%A8%EB%8B%88_(%EC%98%81%ED%99%94)" title="써니 (영화)">써니</a>》(2011), 《<a href="/wiki/%EC%88%98%EC%83%81%ED%95%9C_%EA%B7%B8%EB%85%80" title="수상한 그녀">수상한 그녀</a>》(2014)에서 주연을 맡아 충무로 '최연소 흥행퀸'으로 거듭났다.<sup class="reference" id="cite_ref-name_1-0"><a href="#cite_note-name-1">[1]</a></sup><sup class="reference" id="cite_ref-glamorous_2-0"><a href="#cite_note-glamorous-2">[2]</a></sup></p>

<p>심은경은 <a href="/wiki/2003%EB%85%84" title="2003년">2003년</a> MBC 드라마 《<a href="/wiki/%EB%8C%80%EC%9E%A5%EA%B8%88" title="대장금">대장금</a>》에서 생각시(어린 궁녀) 중 한명인 단역으로 연기를 시작해, <a href="/wiki/2004%EB%85%84" title="2004년">2004년</a> 드라마 《<a href="/wiki/%EA%B2%B0%ED%98%BC%ED%95%98%EA%B3%A0_%EC%8B%B6%EC%9D%80_%EC%97%AC%EC%9E%90" title="결혼하고 싶은 여자">결혼하고 싶은 여자</a>》와 영화 《<a class="mw-redirect" href="/wiki/%EB%8F%84%EB%A7%88_%EC%95%88%EC%A4%91%EA%B7%BC_(%EC%98%81%ED%99%94)" title="도마 안중근 (영화)">도마 안중근</a>》으로 처음 공식적으로 데뷔했다. 2004년 드라마 《<a href="/wiki/%EB%8B%A8%ED%8C%A5%EB%B9%B5_(%EB%93%9C%EB%9D%BC%EB%A7%88)" title="단팥빵 (드라마)">단팥빵</a>》에서 보여준 '헥토파스칼 킥'은 후일 끊임없이 자주 회자되고 있다. 심은경은 <a href="/wiki/2006%EB%85%84" title="2006년">2006년</a> 드라마 《<a class="mw-redirect" href="/wiki/%ED%99%A9%EC%A7%84%EC%9D%B4_(%EB%93%9C%EB%9D%BC%EB%A7%88)" title="황진이 (드라마)">황진이</a>》에서 극 초반의 인상적인 연기로 주목을 받았고, 이듬해 《<a href="/wiki/%ED%83%9C%EC%99%95%EC%82%AC%EC%8B%A0%EA%B8%B0" title="태왕사신기">태왕사신기</a>》, 《<a href="/wiki/%ED%83%9C%EC%96%91%EC%9D%98_%EC%97%AC%EC%9E%90" title="태양의 여자">태양의 여자</a>》등 의 많은 드라마에서 아역배우로 연기를 인정 받았으며, <a href="/wiki/2007%EB%85%84" title="2007년">2007년</a> 영화 《<a href="/wiki/%ED%97%A8%EC%A0%A4%EA%B3%BC_%EA%B7%B8%EB%A0%88%ED%85%94_(2007%EB%85%84_%EC%98%81%ED%99%94)" title="헨젤과 그레텔 (2007년 영화)">헨젤과 그레텔</a>》을 통해 스크린에서도 영향력을 넓히기 시작하였다.</p>
<p><a href="/wiki/2010%EB%85%84" title="2010년">2010년</a> 9월, <a href="/wiki/%EB%AF%B8%EA%B5%AD" title="미국">미국</a> <a href="/wiki/%EB%89%B4%EC%9A%95_%EC%A3%BC" title="뉴욕 주">뉴욕 주</a> <a class="mw-redirect" href="/wiki/%EB%89%B4%EC%9A%95%EC%8B%9C" title="뉴욕시">뉴욕시</a>에 위치한 <a class="extiw" href="https://en.wikipedia.org/wiki/Professional_Children%27s_School" title="en:Professional Children's School">프로페셔널 칠드런 스쿨</a> 유학 과정에서<sup class="reference" id="cite_ref-4"><a href="#cite_note-4">[4]</a></sup> <a href="/wiki/2011%EB%85%84" title="2011년">2011년</a> 영화 《<a href="/wiki/%EC%8D%A8%EB%8B%88_(%EC%98%81%ED%99%94)" title="써니 (영화)">써니</a>》, <a href="/wiki/2012%EB%85%84" title="2012년">2012년</a> 《<a href="/wiki/%EA%B4%91%ED%95%B4,_%EC%99%95%EC%9D%B4_%EB%90%9C_%EB%82%A8%EC%9E%90" title="광해, 왕이 된 남자">광해, 왕이 된 남자</a>》등의 화제작에 출연하였다. <a href="/wiki/2013%EB%85%84" title="2013년">2013년</a> 6월, 유학을 마친 후 성인 역으로 처음 출연한 <a href="/wiki/2014%EB%85%84" title="2014년">2014년</a> 영화 《<a href="/wiki/%EC%88%98%EC%83%81%ED%95%9C_%EA%B7%B8%EB%85%80" title="수상한 그녀">수상한 그녀</a>》를 통해 차세대 20대 배우로서 가능성과 연기력을 인정받은 뒤 <a href="/wiki/%EC%9D%BC%EB%B3%B8_%EB%A7%8C%ED%99%94" title="일본 만화">일본 만화</a>와 드라마 원작의 드라마 《<a href="/wiki/%EB%82%B4%EC%9D%BC%EB%8F%84_%EC%B9%B8%ED%83%80%EB%B9%8C%EB%A0%88" title="내일도 칸타빌레">내일도 칸타빌레</a>》에 캐스팅 되어 열연을 펼쳤다. 또한 《<a href="/wiki/%EC%88%98%EC%83%81%ED%95%9C_%EA%B7%B8%EB%85%80" title="수상한 그녀">수상한 그녀</a>》로 제23회 부일영화상 여우주연상, 제19회 춘사영화상 여우주연상, 제50회 백상예술대상 영화부문 여자 최우수 연기상 등 다수의 상을 수상하며 배우로서의 연기력을 인정받았다.</p>
