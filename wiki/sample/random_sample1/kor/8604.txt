<p><b>심천역</b>(深川驛)은 <a href="/wiki/%EB%8C%80%ED%95%9C%EB%AF%BC%EA%B5%AD" title="대한민국">대한민국</a> <a href="/wiki/%EC%B6%A9%EC%B2%AD%EB%B6%81%EB%8F%84" title="충청북도">충청북도</a> <a href="/wiki/%EC%98%81%EB%8F%99%EA%B5%B0" title="영동군">영동군</a> <a href="/wiki/%EC%8B%AC%EC%B2%9C%EB%A9%B4" title="심천면">심천면</a>에 있는 <a href="/wiki/%EA%B2%BD%EB%B6%80%EC%84%A0" title="경부선">경부선</a>의 <a href="/wiki/%EC%B2%A0%EB%8F%84%EC%97%AD" title="철도역">철도역</a>이다. <a href="/wiki/1934%EB%85%84" title="1934년">1934년</a> <a class="mw-redirect" href="/wiki/9%EC%9B%94_31%EC%9D%BC" title="9월 31일">9월 31일</a>에 <a href="/wiki/%EA%B2%BD%EB%B6%80%EC%84%A0" title="경부선">경부선</a> 복선화 공사에 의해 현재의 역사가 신축되었으며, 원형이 잘 보존되어 있어 건축적, 철도사적 가치가 큰 것으로 평가되어 <a href="/wiki/2006%EB%85%84" title="2006년">2006년</a> <a href="/wiki/12%EC%9B%94_4%EC%9D%BC" title="12월 4일">12월 4일</a>에 <a class="mw-redirect" href="/wiki/%EB%93%B1%EB%A1%9D%EB%AC%B8%ED%99%94%EC%9E%AC" title="등록문화재">등록문화재</a>로 지정되었다.</p>
