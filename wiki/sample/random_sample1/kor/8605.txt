<p><b>심청가</b>(沈淸歌)는 <a href="/wiki/%ED%8C%90%EC%86%8C%EB%A6%AC" title="판소리">판소리</a> 다섯 마당 중의 하나로서 <a href="/wiki/%EC%8B%AC%EC%B2%AD%EC%A0%84" title="심청전">심청전</a>의 내용을 판소리로 만든 것이다. 주로 애절하고 슬픈 대목이 많아서 골계미가 적다. 과거에는 너무 슬픈 소리라 하여 높게 치지 않았으나 근래에는 《<a href="/wiki/%EC%B6%98%ED%96%A5%EA%B0%80" title="춘향가">춘향가</a>》 다음으로 예술성이 높은 작품으로 평가된다. 심봉사의 딸 심청은 어려서 친모를 잃었으나 효성이 지극하여 아버지를 봉양하다 그 눈을 띄우기 위해 스스로 몸이 팔려 인당수에 떨어지나 다시 환생하여 황후가 되고 심봉사는 그 효성에 눈을 뜨게 된다는 심청전의 줄거리이다.</p>

<h2><span class="mw-headline" id=".EA.B5.AC.EC.84.B1">구성</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%8B%AC%EC%B2%AD%EA%B0%80&amp;action=edit&amp;section=1" title="부분 편집: 구성">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>① 초앞(廳－초비두)<sup class="reference" id="cite_ref-1"><a href="#cite_note-1">[1]</a></sup> ② 심청 탄생 ③ 곽씨 부인 죽는 대목 ④ 곽씨 부인 장사 ⑤ 심청 효행 ⑥ 장 승상 부인 만나는 대목 ⑦ 심봉사 화주승 만나는 대목 ⑧ 남경 선인 만나는 대목 ⑨ 인당수 들어가는 대목 ⑩ 심청 환생 ⑪ 뺑파 과장 ⑫ 황성 올라가는 대목 ⑬ 맹인 잔치 ⑭ 뒤풀이</p>
<h2><span class="mw-headline" id=".EC.9D.8C.EC.95.85.EC.A0.81_.ED.8A.B9.EC.A7.95.EA.B3.BC_.EB.88.88.EB.8C.80.EB.AA.A9">음악적 특징과 눈대목</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%8B%AC%EC%B2%AD%EA%B0%80&amp;action=edit&amp;section=2" title="부분 편집: 음악적 특징과 눈대목">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>《심청가》는 판소리 다섯 바탕 가운데 가장 애절한 정서를 기저에 두고 있는 작품이다. 물론 대목에 따라서 소소한 웃음거리<sup class="reference" id="cite_ref-2"><a href="#cite_note-2">[2]</a></sup>를 제공하기도 하지만, 기본적으로는 비장미가 넘치는 작품이다. 때문에 화평한 <a href="/wiki/%ED%8F%89%EC%A1%B0" title="평조">평조</a>(平調) 보다는 유장한 <a href="/wiki/%EC%9A%B0%EC%A1%B0_(%ED%8C%90%EC%86%8C%EB%A6%AC)" title="우조 (판소리)">우조</a>(羽調), 우렁찬 우조보다는 슬픈 <a href="/wiki/%EA%B3%84%EB%A9%B4%EC%A1%B0" title="계면조">계면조</a>(界面調)를 사용한다. 고제(古制)일수록 우조의 비중이 다소 높으며 신제(新制)일수록 계면조와 평조의 비율이 높아진다.</p>
<p>《심청가》 가운데 목기교나 부침새 면에서 가장 치밀하고 탁월하게 구성되어 있는 대표적인 눈대목은 다음과 같다.</p>
<ul>
<li>곽씨 부인 유언 (계면조-진양)</li>
<li>상여소리 (계면조-중모리, 중중모리)</li>
<li>주과포혜 (계면조-진양)<sup class="reference" id="cite_ref-3"><a href="#cite_note-3">[3]</a></sup></li>
<li>시비따라 가는 대목 (우조-진양)</li>
<li>범피중류 (우조-진양)</li>
<li>화초타령 (평조-중중모리)<sup class="reference" id="cite_ref-4"><a href="#cite_note-4">[4]</a></sup></li>
<li>추월만정 (계면조-진양)</li>
<li>방아타령 (평조-중중모리, 자진모리)<sup class="reference" id="cite_ref-5"><a href="#cite_note-5">[5]</a></sup></li>
</ul>
<h2><span class="mw-headline" id=".E3.80.8A.EC.8B.AC.EC.B2.AD.EA.B0.80.E3.80.8B.EC.9D.98_.EB.AA.85.EC.B0.BD.EA.B3.BC_.EC.A0.84.EC.8A.B9_.ED.98.84.ED.99.A9">《심청가》의 명창과 전승 현황</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%8B%AC%EC%B2%AD%EA%B0%80&amp;action=edit&amp;section=3" title="부분 편집: 《심청가》의 명창과 전승 현황">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>《심청가》를 잘했고, 더늠을 남긴 명창으로는 <a href="/wiki/%EC%86%A1%EA%B4%91%EB%A1%9D" title="송광록">송광록</a>(宋光綠), <a href="/wiki/%EC%A0%84%EB%8F%84%EC%84%B1" title="전도성">전도성</a>(全道成), <a class="new" href="/w/index.php?title=%EA%B9%80%EC%B1%84%EB%A7%8C&amp;action=edit&amp;redlink=1" title="김채만 (없는 문서)">김채만</a>(金采萬), <a class="mw-disambig" href="/wiki/%EB%B0%95%EA%B8%B0%ED%99%8D" title="박기홍">박기홍</a>(朴基洪), <a href="/wiki/%EC%A0%95%EC%B0%BD%EC%97%85" title="정창업">정창업</a>(丁昌業), <a href="/wiki/%EA%B9%80%EC%B0%BD%EB%A3%A1_(%EA%B5%AD%EC%95%85%EC%9D%B8)" title="김창룡 (국악인)">김창룡</a>(金昌龍) 등을 들 수 있다. 송광록과 전도성의 더늠은 범피중류, 김채만의 더늠은 초앞, 박기홍과 김창룡의 더늠은 화초타령<sup class="reference" id="cite_ref-6"><a href="#cite_note-6">[6]</a></sup>, 정창업의 더늠은 중타령 대목이다. 이 외에도 김창룡의 동생 김창진(金昌鎭)이 또한 《심청가》를 잘했고, <a href="/wiki/%EC%86%A1%EB%A7%8C%EA%B0%91" title="송만갑">송만갑</a>(宋萬甲)도 잘했다고 하며, <a href="/wiki/%EC%9D%B4%EB%82%A0%EC%B9%98" title="이날치">이날치</a>(李捺治)와 박유전(朴裕全)이 또한 심청가를 잘했다. 이들 이후로 여류명창 <a href="/wiki/%EC%9D%B4%ED%99%94%EC%A4%91%EC%84%A0" title="이화중선">이화중선</a>(李花仲仙)은 《심청가》가운데 추월만정 대목으로 공전의 빅히트를 기록했다. 이 외에도 박유전의 지도를 받은 <a href="/wiki/%EC%A0%95%EC%9E%AC%EA%B7%BC_(%EA%B5%AD%EC%95%85%EC%9D%B8)" title="정재근 (국악인)">정재근</a>(鄭在根)과 그 조카 <a href="/wiki/%EC%A0%95%EC%9D%91%EB%AF%BC" title="정응민">정응민</a>(鄭應珉), 김채만의 지도를 받은 박동실(朴東實)이 또한 심청가를 잘했다.</p>
<p>이러한 명창들의 계통을 중심으로 근래에 전승되어 전판의 면모를 확인할 수 있는 《심청가》의 유파는 다음과 같이 5개 정도의 유파가 있다.</p>
<ul>
<li>정응민 바디 : 박유전 → 정재근 → <b>정응민</b> → <b>정권진</b> → 성우향, <b>조상현</b>, <b>성창순</b> : 정권진, 조상현, 성창순 중요무형문화재 지정.</li>
<li>박동실 바디 : 이날치 → 김채만 → <b>박동실</b> → 한애순, 장월중선, 한승호<sup class="reference" id="cite_ref-7"><a href="#cite_note-7">[7]</a></sup> : 본래 처음 중요무형문화재로 지정되었으나 예능보유자는 지정되지 않았고, 한애순과 장월중선, 장월중선의 딸 정순임은 지방문화재 예능보유자이다.</li>
<li>김연수 바디 : 송만갑, 정응민 → <b>김연수</b> → 오정숙 → 이일주</li>
<li>김소희 바디 : 송만갑, 박동실, 정응민 → <b>김소희</b> → <b>신영희</b>, 안숙선, 이명희</li>
<li>박동진 바디 : 김정근 → 김창진 → <b>박동진</b> → 김양숙 : 현재 연창되지 않음.</li>
</ul>
