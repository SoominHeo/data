<p><b>CLC</b>(씨엘씨)는 <a href="/wiki/%EB%8C%80%ED%95%9C%EB%AF%BC%EA%B5%AD" title="대한민국">대한민국</a>의 7인조 <a href="/wiki/%EA%B1%B8_%EA%B7%B8%EB%A3%B9" title="걸 그룹">걸 그룹</a>으로, <a href="/wiki/%ED%81%90%EB%B8%8C_%EC%97%94%ED%84%B0%ED%85%8C%EC%9D%B8%EB%A8%BC%ED%8A%B8" title="큐브 엔터테인먼트">큐브 엔터테인먼트</a> 소속이다. CLC는 Crystal Clear의 약자로 수정처럼 맑게, 수정처럼 영원히 변치 않는 매력을 지니겠다는 의미를 담고 있다. <a href="/wiki/2015%EB%85%84" title="2015년">2015년</a> <a href="/wiki/3%EC%9B%94_18%EC%9D%BC" title="3월 18일">3월 18일</a> 데뷔 쇼케이스를 통해 데뷔무대를 가졌다. <a href="/wiki/3%EC%9B%94_19%EC%9D%BC" title="3월 19일">3월 19일</a> 앨범 《첫사랑》을 발매하였고, 이 날 <a href="/wiki/%EC%97%A0%EC%B9%B4%EC%9A%B4%ED%8A%B8%EB%8B%A4%EC%9A%B4" title="엠카운트다운">엠카운트다운</a>을 시작으로 본격적인 방송 활동을 시작하였다.</p>

<h2><span class="mw-headline" id=".ED.99.9C.EB.8F.99">활동</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=CLC_(%EC%9D%8C%EC%95%85_%EA%B7%B8%EB%A3%B9)&amp;action=edit&amp;section=1" title="부분 편집: 활동">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<h3><span class="mw-headline" id=".EC.B2.AB.EC.82.AC.EB.9E.91_.282015.03.18_.7E_2015.04.15.29">첫사랑 (2015.03.18 ~ 2015.04.15)</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=CLC_(%EC%9D%8C%EC%95%85_%EA%B7%B8%EB%A3%B9)&amp;action=edit&amp;section=2" title="부분 편집: 첫사랑 (2015.03.18 ~ 2015.04.15)">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>CLC는 <a href="/wiki/2015%EB%85%84" title="2015년">2015년</a> <a href="/wiki/3%EC%9B%94_18%EC%9D%BC" title="3월 18일">3월 18일</a>에 뮤직비디오와 음원을 공개하고, 광장동 AX홀에서 데뷔 쇼케이스를 열었다. <a href="/wiki/3%EC%9B%94_19%EC%9D%BC" title="3월 19일">3월 19일</a> 엠카운트다운에서 《<a class="new" href="/w/index.php?title=Pepe&amp;action=edit&amp;redlink=1" title="Pepe (없는 문서)">Pepe</a>》 데뷔 무대를 가졌고, <a href="/wiki/4%EC%9B%94_15%EC%9D%BC" title="4월 15일">4월 15일</a> 후속곡 앨범으로 활동하기 위해 《<a class="new" href="/w/index.php?title=Pepe&amp;action=edit&amp;redlink=1" title="Pepe (없는 문서)">Pepe</a>》활동을 마무리하였다.</p>
<h3><span class="mw-headline" id="Eighteen_.282015.4.16_.7E_2015.5.3.29">Eighteen (2015.4.16 ~ 2015.5.3)</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=CLC_(%EC%9D%8C%EC%95%85_%EA%B7%B8%EB%A3%B9)&amp;action=edit&amp;section=3" title="부분 편집: Eighteen (2015.4.16 ~ 2015.5.3)">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p><a href="/wiki/2015%EB%85%84" title="2015년">2015년</a> <a href="/wiki/4%EC%9B%94_16%EC%9D%BC" title="4월 16일">4월 16일</a> 《<a class="new" href="/w/index.php?title=Eighteen&amp;action=edit&amp;redlink=1" title="Eighteen (없는 문서)">Eighteen</a>》 앨범을 내면서 《<a class="new" href="/w/index.php?title=Eighteen&amp;action=edit&amp;redlink=1" title="Eighteen (없는 문서)">Eighteen</a>》으로 후속곡 활동을 이어갔다. 그리고 <a href="/wiki/5%EC%9B%94_3%EC%9D%BC" title="5월 3일">5월 3일</a> 두번째 미니앨범을 준비하기 위해 활동을 마무리하였다.</p>
<h3><span class="mw-headline" id="QUESTION_.282015.05.28_.7E_2015.06.28.29">QUESTION (2015.05.28 ~ 2015.06.28)</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=CLC_(%EC%9D%8C%EC%95%85_%EA%B7%B8%EB%A3%B9)&amp;action=edit&amp;section=4" title="부분 편집: QUESTION (2015.05.28 ~ 2015.06.28)">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p><a href="/wiki/2015%EB%85%84" title="2015년">2015년</a> <a href="/wiki/5%EC%9B%94_20%EC%9D%BC" title="5월 20일">5월 20일</a>부터 공식 트위터에 티저 이미지를 하나씩 공개하였고, 티저 영상과 수록곡 영상은 <a class="new" href="/w/index.php?title=5%EC%9B%94_25&amp;action=edit&amp;redlink=1" title="5월 25 (없는 문서)">5월 25</a>일에 공개했다. 뮤직비디오와 음원은 <a href="/wiki/5%EC%9B%94_28%EC%9D%BC" title="5월 28일">5월 28일</a> 정오에 본격적으로 공개하였고, 이 날 <a href="/wiki/%EC%97%A0%EC%B9%B4%EC%9A%B4%ED%8A%B8%EB%8B%A4%EC%9A%B4" title="엠카운트다운">엠카운트다운</a>에서 타이틀곡인 《<a class="new" href="/w/index.php?title=%EA%B6%81%EA%B8%88%ED%95%B4&amp;action=edit&amp;redlink=1" title="궁금해 (없는 문서)">궁금해</a>》로 컴백 무대를 처음으로 활동을 시작하였다.</p>
<h3><span class="mw-headline" id="Refresh_.282016.03.03_.7E_2016.03.27.29">Refresh (2016.03.03 ~ 2016.03.27)</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=CLC_(%EC%9D%8C%EC%95%85_%EA%B7%B8%EB%A3%B9)&amp;action=edit&amp;section=5" title="부분 편집: Refresh (2016.03.03 ~ 2016.03.27)">편집</a><span class="mw-editsection-bracket">]</span></span></h3>

<p>9개월만에 새롭게 컴백을 앞두고 <a href="/wiki/PRODUCE_101" title="PRODUCE 101">PRODUCE 101</a>에 출연중인 권은빈과 <a href="/wiki/%ED%99%8D%EC%BD%A9" title="홍콩">홍콩</a> 출신의 엘키를 영입하여 멤버를 새로 개편하였다. 2월 26일부터 공식 트위터를 통해 티저이미지를 공개하였고, 2월 28일에는 티저 영상과 수록곡 영상을 공개했다. 2월 29일 자정에 음원과 뮤직비디오 짧은 버전이 공개되었고 3월 2일 쇼챔피언에서 컴백을 가질것이라고 알려진 것과는 달리 그 다음 날인 3일 엠카운트다운에서 타이틀곡인 《<a class="new" href="/w/index.php?title=%EC%98%88%EB%BB%90%EC%A7%80%EA%B2%8C&amp;action=edit&amp;redlink=1" title="예뻐지게 (없는 문서)">예뻐지게</a>》로 최초 컴백 무대를 가졌다. 하지만 <a class="mw-redirect" href="/wiki/%EA%B6%8C%EC%9D%80%EB%B9%88" title="권은빈">권은빈</a>은 <a href="/wiki/PRODUCE_101" title="PRODUCE 101">PRODUCE 101</a>에 출연하는 도중에 타 프로그램에 출연이 불가하다는 조항에 따라 탈락할 경우에는 CLC로 활동하며, 최종까지 살아남는다면 <a href="/wiki/PRODUCE_101" title="PRODUCE 101">PRODUCE 101</a> 활동을 모두 마친 후 CLC로 활동하게 된다. 3월 4일 <a class="mw-redirect" href="/wiki/%EC%8A%88%ED%8D%BC%EC%A3%BC%EB%8B%88%EC%96%B4%EC%9D%98_%ED%82%A4%EC%8A%A4_%EB%8D%94_%EB%9D%BC%EB%94%94%EC%98%A4" title="슈퍼주니어의 키스 더 라디오">슈퍼주니어의 키스 더 라디오</a>에 출연해 오승희에서 장승연으로 리더가 교체되었음을 밝혔고, 3월 27일에 방송된 SBS 인기가요를 마지막으로 예뻐지게 활동을 마감하였다.</p>
<h3><span class="mw-headline" id="NU.CLEAR_.282016.05.30_.7E_2016.07.08.29">NU.CLEAR (2016.05.30 ~ 2016.07.08)</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=CLC_(%EC%9D%8C%EC%95%85_%EA%B7%B8%EB%A3%B9)&amp;action=edit&amp;section=6" title="부분 편집: NU.CLEAR (2016.05.30 ~ 2016.07.08)">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>CLC는 5월 30일에 쇼케이스를 열고 2개월만에 네번째 미니앨범《<a class="new" href="/w/index.php?title=NU.CLEAR&amp;action=edit&amp;redlink=1" title="NU.CLEAR (없는 문서)">NU.CLEAR</a>》을 발매하였고, <a href="/wiki/PRODUCE_101" title="PRODUCE 101">PRODUCE 101</a>에 출연으로 인해 CLC 활동을 하지 못했던 권은빈의 합류 이후의 첫 번째 발매앨범으로, <a href="/wiki/6%EC%9B%94_1%EC%9D%BC" title="6월 1일">6월 1일</a> <a class="mw-redirect" href="/wiki/%EC%87%BC%EC%B1%94%ED%94%BC%EC%96%B8" title="쇼챔피언">쇼챔피언</a>으로 컴백하였다. 7월 8일 KBS 뮤직뱅크를 마지막으로 아니야 활동을 마감하였다.</p>
<h3><span class="mw-headline" id="CRYSTYLE_.282017.01.17_.7E.29">CRYSTYLE (2017.01.17 ~)</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=CLC_(%EC%9D%8C%EC%95%85_%EA%B7%B8%EB%A3%B9)&amp;action=edit&amp;section=7" title="부분 편집: CRYSTYLE (2017.01.17 ~)">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>CLC는 1월 17일에 쇼케이스를 열고 8개월만에 다섯번 째 미니앨범 《<a class="new" href="/w/index.php?title=CRYSTYLE&amp;action=edit&amp;redlink=1" title="CRYSTYLE (없는 문서)">CRYSTYLE</a>》을 발매하였다. 그 다음날 <a class="mw-redirect" href="/wiki/%EC%87%BC%EC%B1%94%ED%94%BC%EC%96%B8" title="쇼챔피언">쇼챔피언</a>을 통해서 최초로 타이틀곡 도깨비의 컴백무대를 가졌다.</p>
