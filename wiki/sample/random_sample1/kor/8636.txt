클래스(W124)</a>의 플랫폼을 활용하여 개발하였다. <a class="mw-redirect" href="/wiki/SUV" title="SUV">SUV</a>와 <a href="/wiki/%ED%8A%B8%EB%9F%AD" title="트럭">트럭</a>만 만들던 <a href="/wiki/%EC%8C%8D%EC%9A%A9%EC%9E%90%EB%8F%99%EC%B0%A8" title="쌍용자동차">쌍용자동차</a>의 첫 번째 승용차로, <a href="/wiki/1997%EB%85%84" title="1997년">1997년</a> <a href="/wiki/10%EC%9B%94_14%EC%9D%BC" title="10월 14일">10월 14일</a>에 출시되었다. <a href="/wiki/%EB%A9%94%EB%A5%B4%EC%84%B8%EB%8D%B0%EC%8A%A4-%EB%B2%A4%EC%B8%A0" title="메르세데스-벤츠">메르세데스-벤츠</a>의 수석 디자이너인 갈리첸 도르프가 디자인 개발에 참여하여 <a href="/wiki/%EB%A9%94%EB%A5%B4%EC%84%B8%EB%8D%B0%EC%8A%A4-%EB%B2%A4%EC%B8%A0" title="메르세데스-벤츠">메르세데스-벤츠</a>의 승용차와 흡사하고, 대형차임에도 다이내믹한 모습이 특징이다. 이를 견제한 <a href="/wiki/%EB%A9%94%EB%A5%B4%EC%84%B8%EB%8D%B0%EC%8A%A4-%EB%B2%A4%EC%B8%A0" title="메르세데스-벤츠">메르세데스-벤츠</a>가 체어맨의 익스테리어 변경과 선진국 시장에 대한 수출 자제 등을 요청하기도 하였다.<sup class="reference" id="cite_ref-1"><a href="#cite_note-1">[1]</a></sup> <a href="/wiki/%EB%8C%80%ED%95%9C%EB%AF%BC%EA%B5%AD" title="대한민국">대한민국</a>산 승용차 최초로 5단 자동변속기와 연비창이 표시되는 트립 컴퓨터를 적용하였다. 당시 경쟁 차종이던 <a href="/wiki/%ED%98%84%EB%8C%80_%EB%8B%A4%EC%9D%B4%EB%84%88%EC%8A%A4%ED%8B%B0" title="현대 다이너스티">현대 다이너스티</a>의 <a href="/wiki/%EB%A6%AC%EB%AC%B4%EC%A7%84" title="리무진">리무진</a> 사양이 뒷문을 늘린 것과는 달리, 체어맨의 <a href="/wiki/%EB%A6%AC%EB%AC%B4%EC%A7%84" title="리무진">리무진</a> 사양은 B 필러를 늘렸다. <a href="/wiki/1998%EB%85%84" title="1998년">1998년</a>에는 <a href="/wiki/%ED%95%9C%EA%B5%ADGM" title="한국GM">대우자동차(한국GM의 전신)</a>가 <a href="/wiki/%EC%8C%8D%EC%9A%A9%EC%9E%90%EB%8F%99%EC%B0%A8" title="쌍용자동차">쌍용자동차</a>를 인수하여 당시 <a href="/wiki/%ED%95%9C%EA%B5%ADGM" title="한국GM">대우자동차</a>의 패밀리 룩이던 3분할 라디에이터 그릴이 적용되었으나, 차체 디자인과 어울리지 못하다는 혹평을 받았다. <a href="/wiki/2000%EB%85%84" title="2000년">2000년</a>에 <a href="/wiki/%EC%8C%8D%EC%9A%A9%EC%9E%90%EB%8F%99%EC%B0%A8" title="쌍용자동차">쌍용자동차</a>가 <a href="/wiki/%ED%95%9C%EA%B5%ADGM" title="한국GM">대우자동차</a>에서 분리되며, 기존의 <a href="/wiki/%EC%8C%8D%EC%9A%A9%EC%9E%90%EB%8F%99%EC%B0%A8" title="쌍용자동차">쌍용자동차</a> 엠블럼이 들어간 라디에이터 그릴이 적용되었다. <a href="/wiki/1999%EB%85%84_4%EC%9B%94" title="1999년 4월">1999년 4월</a>에 <a href="/wiki/%EC%98%81%EA%B5%AD" title="영국">영국</a>의 여왕인 <a href="/wiki/%EC%97%98%EB%A6%AC%EC%9E%90%EB%B2%A0%EC%8A%A4_2%EC%84%B8" title="엘리자베스 2세">엘리자베스 2세</a>가 <a href="/wiki/%EB%8C%80%ED%95%9C%EB%AF%BC%EA%B5%AD" title="대한민국">대한민국</a>을 방문하였을 때 의전용 차량으로 채택되기도 하였다.<sup class="reference" id="cite_ref-2"><a href="#cite_note-2">[2]</a></sup> <a href="/wiki/2000%EB%85%84_3%EC%9B%94" title="2000년 3월">2000년 3월</a>에는 직렬 6기통 2.8ℓ 엔진과 직렬 6기통 3.2ℓ 엔진 외에 직렬 4기통 2.3ℓ 엔진을 얹은 보급형 트림인 CM400이 추가되었다.<sup class="reference" id="cite_ref-3"><a href="#cite_note-3">[3]</a></sup> <a href="/wiki/2001%EB%85%84_4%EC%9B%94" title="2001년 4월">2001년 4월</a>에는 세로 바가 적용된 신규 라디에이터 그릴, 우적 감시 센서 내장 와이퍼, 오토 라이트 컨트롤 등이 적용되었다.<sup class="reference" id="cite_ref-4"><a href="#cite_note-4">[4]</a></sup></p>

<p>출시 6년 만인 <a href="/wiki/2003%EB%85%84" title="2003년">2003년</a> <a href="/wiki/9%EC%9B%94_25%EC%9D%BC" title="9월 25일">9월 25일</a>에 출시되었다. 페이스 리프트를 거쳐 익스테리어는 다이내믹한 모습을 유지함과 동시에 보수적인 느낌을 가미하였다. 아날로그 계기판이 디지털 계기판으로 바뀌는 등 인테리어를 고급스럽게 다듬었다. 또한 DVD 형태의 내비게이션을 <a href="/wiki/%EB%8C%80%ED%95%9C%EB%AF%BC%EA%B5%AD" title="대한민국">대한민국</a>산 차 최초로 갖추었다. 기존 체어맨의 직렬 4기통 2.3ℓ 엔진에는 4단 자동변속기도 조합하였으나, 뉴 체어맨부터는 4단 자동변속기를 제외하여 5단 자동변속기만 남게 되었다. <a href="/wiki/2005%EB%85%84_3%EC%9B%94" title="2005년 3월">2005년 3월</a>에는 타이어 공기압 경보 장치(TPMS), 전자 제어 에어 서스펜션(EAS), 전자동 파킹 브레이크(EPB)가 적용된 뉴 테크 트림이 선보였다.<sup class="reference" id="cite_ref-5"><a href="#cite_note-5">[5]</a></sup> <a href="/wiki/2006%EB%85%84_1%EC%9B%94" title="2006년 1월">2006년 1월</a>에는 판매량이 부진한 직렬 4기통 2.3ℓ 엔진이 단종되었다. <a href="/wiki/2006%EB%85%84_6%EC%9B%94" title="2006년 6월">2006년 6월</a>에 직렬 6기통 3.6ℓ 엔진과 차선 이탈 경고 시스템(LDWS) 등이 추가되었다.<sup class="reference" id="cite_ref-6"><a href="#cite_note-6">[6]</a></sup></p>

<div class="thumb tright">
<div class="thumbinner" style="width:252px;"><a class="image" href="/wiki/%ED%8C%8C%EC%9D%BC:20101003_ssangyong_chairman_h_01.jpg"><img alt="" class="thumbimage" data-file-height="660" data-file-width="1450" height="114" src="//upload.wikimedia.org/wikipedia/commons/thumb/6/62/20101003_ssangyong_chairman_h_01.jpg/250px-20101003_ssangyong_chairman_h_01.jpg" srcset="//upload.wikimedia.org/wikipedia/commons/thumb/6/62/20101003_ssangyong_chairman_h_01.jpg/375px-20101003_ssangyong_chairman_h_01.jpg 1.5x, //upload.wikimedia.org/wikipedia/commons/thumb/6/62/20101003_ssangyong_chairman_h_01.jpg/500px-20101003_ssangyong_chairman_h_01.jpg 2x" width="250"/></a>

<div class="thumb tright">
<div class="thumbinner" style="width:252px;"><a class="image" href="/wiki/%ED%8C%8C%EC%9D%BC:20101003_ssangyong_chairman_h_02.jpg"><img alt="" class="thumbimage" data-file-height="660" data-file-width="1450" height="114" src="//upload.wikimedia.org/wikipedia/commons/thumb/6/6c/20101003_ssangyong_chairman_h_02.jpg/250px-20101003_ssangyong_chairman_h_02.jpg" srcset="//upload.wikimedia.org/wikipedia/commons/thumb/6/6c/20101003_ssangyong_chairman_h_02.jpg/375px-20101003_ssangyong_chairman_h_02.jpg 1.5x, //upload.wikimedia.org/wikipedia/commons/thumb/6/6c/20101003_ssangyong_chairman_h_02.jpg/500px-20101003_ssangyong_chairman_h_02.jpg 2x" width="250"/></a>

<p><a href="/wiki/2008%EB%85%84" title="2008년">2008년</a> <a href="/wiki/1%EC%9B%94_3%EC%9D%BC" title="1월 3일">1월 3일</a>에 기존 뉴 체어맨이 체어맨 H로 차명이 변경되었다. H는 하이 오너를 의미하며, 기존 뉴 체어맨에 비해 라인업이 축소되었다. 원래대로라면 뉴 체어맨은 체어맨 W가 출시됨과 동시에 단종되어야 하나, <a href="/wiki/%EC%8C%8D%EC%9A%A9%EC%9E%90%EB%8F%99%EC%B0%A8" title="쌍용자동차">쌍용자동차</a>의 자금 사정상 신차 개발이 제대로 진행되지 못하여 어쩔 수 없이 기존 뉴 체어맨의 편의 사양 및 안전 사양을 대폭 삭제해 가격을 낮춰 체어맨 W와 더불어 선택의 폭을 넓힌 것이 체어맨 H이다. 이 때문에 기존 뉴 체어맨 오너들의 반발을 사기도 했다. 이에 따라 경쟁 차종이 <a href="/wiki/%ED%98%84%EB%8C%80_%EC%97%90%EC%BF%A0%EC%8A%A4" title="현대 에쿠스">현대 에쿠스</a> 에서 <a href="/wiki/%ED%98%84%EB%8C%80_%EC%A0%9C%EB%84%A4%EC%8B%9C%EC%8A%A4" title="현대 제네시스">현대 제네시스</a>, <a href="/wiki/%EA%B8%B0%EC%95%84_%EC%98%A4%ED%94%BC%EB%9F%AC%EC%8A%A4" title="기아 오피러스">기아 오피러스</a>, <a href="/wiki/GM%EB%8C%80%EC%9A%B0_%EB%B2%A0%EB%A6%AC%ED%83%80%EC%8A%A4" title="GM대우 베리타스">GM대우 베리타스</a> 등으로 바뀌었다.</p>

<p>신선함 유지와 상품성 강화를 위해 다시 한번 페이스 리프트를 거쳤고, 인테리어는 대시보드를 바꾸었다. <a href="/wiki/2011%EB%85%84_3%EC%9B%94" title="2011년 3월">2011년 3월</a>에 열린 <a class="mw-redirect" href="/wiki/%EC%84%9C%EC%9A%B8%EB%AA%A8%ED%84%B0%EC%87%BC" title="서울모터쇼">서울모터쇼</a>에 처음 공개되었으며, 같은 해 <a href="/wiki/5%EC%9B%94_27%EC%9D%BC" title="5월 27일">5월 27일</a>에 출시되었다. 뉴 클래식이라는 서브 네임이 더해졌으며, 하이빔과 로우빔이 결합된 LED 라이트 가이드 내장 헤드 램프, USB 단자, 파워 아울렛, 3.5인치 LCD 내장 슈퍼비전 클러스터 등이 적용되었다. <a href="/wiki/2012%EB%85%84_10%EC%9B%94" title="2012년 10월">2012년 10월</a>에는 하만 그룹의 프리미엄 사운드 시스템과 뒷 좌석 센터 3점식 시트 벨트 등이 적용되었다.<sup class="reference" id="cite_ref-7"><a href="#cite_note-7">[7]</a></sup> 이러한 노력에도 불구하고, 계속되는 저조한 판매 실적에 따라 출시된 지 약 4년 만인 <a href="/wiki/2014%EB%85%84_12%EC%9B%94" title="2014년 12월">2014년 12월</a>에 단종되었다. 페이스 리프트만 거치면서 14년 이상 생산된 장수 차종이었다.</p>

<p><a href="/wiki/2007%EB%85%84_4%EC%9B%94" title="2007년 4월">2007년 4월</a>에 개최된 <a class="mw-redirect" href="/wiki/%EC%84%9C%EC%9A%B8%EB%AA%A8%ED%84%B0%EC%87%BC" title="서울모터쇼">서울모터쇼</a>에서 WZ라는 이름의 컨셉트 카로 처음 선보였고,<sup class="reference" id="cite_ref-8"><a href="#cite_note-8">[8]</a></sup> 양산형은 <a href="/wiki/2008%EB%85%84" title="2008년">2008년</a> <a href="/wiki/2%EC%9B%94_27%EC%9D%BC" title="2월 27일">2월 27일</a>에 출시되었다. <a href="/wiki/%EB%8C%80%ED%95%9C%EB%AF%BC%EA%B5%AD" title="대한민국">대한민국</a>산 승용차 최초로 V형 8기통 5.0ℓ 엔진과 7단 <a href="/wiki/%EC%9E%90%EB%8F%99%EB%B3%80%EC%86%8D%EA%B8%B0" title="자동변속기">자동변속기</a>(<a href="/wiki/%EB%A9%94%EB%A5%B4%EC%84%B8%EB%8D%B0%EC%8A%A4-%EB%B2%A4%EC%B8%A0" title="메르세데스-벤츠">메르세데스-벤츠</a>제), 앞좌석 무릎 에어백 등이 적용되어 <a href="/wiki/%EB%8C%80%ED%95%9C%EB%AF%BC%EA%B5%AD" title="대한민국">대한민국</a>산 자동차 중 최다인 10개의 에어백(뒷좌석 사이드 에어백이 적용시)이 장착되었다. <a href="/wiki/%ED%9B%84%EB%A5%9C%EA%B5%AC%EB%8F%99" title="후륜구동">후륜구동</a> 방식이 기본이나, 직렬 6기통 3.6ℓ 엔진에 한해 <a href="/wiki/%EB%8C%80%ED%95%9C%EB%AF%BC%EA%B5%AD" title="대한민국">대한민국</a>산 승용차 최초로 4 트로닉이라고 명명된 <a href="/wiki/4%EB%A5%9C%EA%B5%AC%EB%8F%99" title="4륜구동">4륜구동</a>도 추가되었다. <a href="/wiki/2008%EB%85%84_8%EC%9B%94" title="2008년 8월">2008년 8월</a>에는 직렬 6기통 3.2ℓ 엔진이 추가되었고, <a href="/wiki/2009%EB%85%84_9%EC%9B%94" title="2009년 9월">2009년 9월</a>에는 인테리어 색상에 회색을 적용한 그레이 에디션 트림이 추가되었다.</p>

<p><a href="/wiki/2011%EB%85%84" title="2011년">2011년</a> <a href="/wiki/7%EC%9B%94_6%EC%9D%BC" title="7월 6일">7월 6일</a>에 출시되었다. 기존 체어맨 W의 프리미엄 이미지를 계승함과 동시에 페이스 리프트를 거쳐 현대적인 감각을 가미하였다. 자연광에 가까운 색도와 우수한 조사 성능 및 수명을 자랑하는 오토 레벨링 HID 헤드 램프, 고휘도 LED 방향 지시등, 범퍼 일체형 듀얼 머플러, 차량 이미지가 추가되어 직관성을 높인 트립 컴퓨터, 이중 접합 차음 유리 등이 새로 적용되었다. 직렬 6기통 3.6ℓ 엔진에 한해 선택할 수 있던 <a href="/wiki/4%EB%A5%9C%EA%B5%AC%EB%8F%99" title="4륜구동">4륜구동</a>(4 트로닉) 시스템을 직렬 6기통 3.2ℓ 엔진에서도 선택할 수 있도록 하여 선택의 폭을 넓혔다. <a href="/wiki/2013%EB%85%84_4%EC%9B%94" title="2013년 4월">2013년 4월</a>에 개최된 <a href="/wiki/%EC%84%9C%EC%9A%B8_%EB%AA%A8%ED%84%B0%EC%87%BC" title="서울 모터쇼">서울 모터쇼</a>에서 2열 2인승 시트와 <a href="/wiki/%EC%8A%A4%EC%BD%94%ED%8B%80%EB%9E%9C%EB%93%9C" title="스코틀랜드">스코틀랜드</a> 브릿지 오브 웨어 사의 가죽 시트가 적용된 서미트 트림(<a href="/wiki/%EB%A6%AC%EB%AC%B4%EC%A7%84" title="리무진">리무진</a>)과 바우 에디션 트림(<a href="/wiki/%EC%84%B8%EB%8B%A8" title="세단">세단</a>)이 추가되었다. <a href="/wiki/2014%EB%85%84_6%EC%9B%94" title="2014년 6월">2014년 6월</a>에 출시된 <a href="/wiki/2015%EB%85%84" title="2015년">2015년</a>형은 세이프티 전방 카메라가 모든 트림에 신규 적용되고, <a href="/wiki/4%EB%A5%9C%EA%B5%AC%EB%8F%99" title="4륜구동">4륜구동</a>(4 트로닉) 시스템을 CW600(직렬 6기통 3.2ℓ 엔진)과 CW700(직렬 6기통 3.6ℓ 엔진)의 모든 트림에서 선택할 수 있도록 하였다. 2열 USB 충전기와 앞 좌석 3단 통풍 시트, 세로형 라디에이터 그릴도 모든 트림에 확대 적용되었다. 새롭게 디자인된 19인치 하이퍼 실버 스퍼터링 휠이 적용되었으며, 19인치 다이아몬드 커팅 휠이 새로 선보였다. 같은 해 <a href="/wiki/7%EC%9B%94" title="7월">7월</a>에는 <a href="/wiki/%EB%8C%80%ED%95%9C%EB%AF%BC%EA%B5%AD" title="대한민국">대한민국</a>산 <a href="/wiki/%EB%A6%AC%EB%AC%B4%EC%A7%84" title="리무진">리무진</a> 최초로 <a href="/wiki/4%EB%A5%9C%EA%B5%AC%EB%8F%99" title="4륜구동">4륜구동</a>(4 트로닉) 시스템을 CW700L(직렬 6기통 3.6ℓ 엔진) 트림에서 선택할 수 있도록 하였다. <a href="/wiki/2016%EB%85%84_2%EC%9B%94" title="2016년 2월">2016년 2월</a>에는 상품성이 개선된 체어맨 W 카이저가 선보였는데, <a href="/wiki/%EB%8F%85%EC%9D%BC%EC%96%B4" title="독일어">독일어</a>로 <a href="/wiki/%ED%99%A9%EC%A0%9C" title="황제">황제</a>를 뜻하는 카이저라는 새로운 브랜드명을 채택함으로써 기존 엠블럼에서 황제의 상징인 독수리를 형상화한 카이저 엠블럼으로 교체되었다. 시트와 도어 트림에 퀼팅 패턴이 적용되었고, 대시보드에 블랙 글로시 타입의 새로운 우드 그레인과 골드 컬러 몰딩이 적용되어 고급감을 높였다. 전동식 세이프티 파워 트렁크를 비롯하여 19인치 스퍼터링 휠과 액티브 크루즈 컨트롤 등이 확대 적용되었다.</p>
