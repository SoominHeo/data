<p><b>CMOS</b>(Complementary metal–oxide–semiconductor, 시모스)는 <a href="/wiki/%EC%A7%91%EC%A0%81_%ED%9A%8C%EB%A1%9C" title="집적 회로">집적 회로</a>의 한 종류로, <a href="/wiki/%EB%A7%88%EC%9D%B4%ED%81%AC%EB%A1%9C%ED%94%84%EB%A1%9C%EC%84%B8%EC%84%9C" title="마이크로프로세서">마이크로프로세서</a>나 <a class="mw-redirect" href="/wiki/%EC%97%90%EC%8A%A4%EB%9E%A8" title="에스램">SRAM</a> 등의 <a href="/wiki/%EB%94%94%EC%A7%80%ED%84%B8_%ED%9A%8C%EB%A1%9C" title="디지털 회로">디지털 회로</a>를 구성하는 데에 이용된다. <b>상보성 금속 산화막 반도체</b>(相補性 金屬 酸化膜 半導體)라는 용어도 통용된다.</p>

<h2><span class="mw-headline" id=".EA.B0.9C.EC.9A.94">개요</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=CMOS&amp;action=edit&amp;section=1" title="부분 편집: 개요">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>P 채널과 N 채널의 <a class="mw-redirect" href="/wiki/%EC%A0%84%EA%B3%84%ED%9A%A8%EA%B3%BC_%ED%8A%B8%EB%9E%9C%EC%A7%80%EC%8A%A4%ED%84%B0" title="전계효과 트랜지스터">MOSFET</a>를 전원 전압 간에 직렬로 구성하고 입력은 두가지 MOSFET의 게이트에 같이 연결하고 출력은 두가지 MOSFET 드레인 사이에 연결한 <a href="/wiki/%EC%A7%91%EC%A0%81_%ED%9A%8C%EB%A1%9C" title="집적 회로">집적 회로</a>의 구조이다. 이 경우 각 MOSFET는 스위치로 간주될 수 있으며 같은 입력신호에 대하여 P 채널과 N 채널이 서로 반대로 동작하므로 전원전압과 접지사이에 기본적으로 흐르는 <a class="new" href="/w/index.php?title=%EB%B8%94%EB%A6%AC%EB%94%A9_%EC%A0%84%EB%A5%98&amp;action=edit&amp;redlink=1" title="블리딩 전류 (없는 문서)">bleeding current</a>가 거의 없어지므로 <a href="/wiki/%ED%8A%B8%EB%9E%9C%EC%A7%80%EC%8A%A4%ED%84%B0-%ED%8A%B8%EB%9E%9C%EC%A7%80%EC%8A%A4%ED%84%B0_%EB%85%BC%EB%A6%AC" title="트랜지스터-트랜지스터 논리">TTL 논리 소자</a>에 비해 소비 전력이 적은 <a href="/wiki/%EB%85%BC%EB%A6%AC_%ED%9A%8C%EB%A1%9C" title="논리 회로">논리 회로</a>를 구현할 수 있고, 부하를 면적을 많이 차지하는 저항이 아닌 MOSFET를 사용하므로 집적도를 향상시킬 수 있다.</p>
<p>MOSFET의 동작 영역에서 직류 전달 특성은 선형 영역에서 출력 전압이 입력 전압과 거의 같고, 포화 영역에서 출력 전압은 게이트 전압에서 「문턱 전압」을 뺀 값이 된다. P-MOSFET가 포화 영역일 때 N-MOSFET는 선형 영역이고, N-MOSFET가 포화 영역일 때 P-MOSFET는 선형 영역이다. 시모스의 동작 영역의 대부분은 선형 영역이다. 엄밀하게 양자의 「문턱 전압」이 겹치는 영역이 존재하므로, 사용하지 않는 입력 단자는 문턱 전압 영역에 들어가지 않도록 풀업 또는 풀다운에 연결해 주는 것이 좋다.</p>
<p>시모스 구조에서 게이트 전압에 입력되는 제어 펄스를 "1"에서 "0"으로 변경했을 경우에 노이즈 없이 이전의 출력을 할 수 있고, "0"에서 "1"로 변경했을 경우 역시 노이즈 없이 입력 신호를 출력할 수 있다.</p>
<p>시모스 구조의 논리 회로는 전원 전압을 낮게 하면 소비 전력이 적은 반면, 전달 지연 시간이 커지는 특성을 가지고 있다. 제조 프로세서가 개선되어 낮은 전압의 동작과 고속 동작을 할 수 있게 되었다.</p>
<p>1990년대가 되면서 <a href="/wiki/%EB%B0%98%EB%8F%84%EC%B2%B4" title="반도체">반도체 메모리</a>나 <a href="/wiki/%EB%A7%88%EC%9D%B4%ED%81%AC%EB%A1%9C%ED%94%84%EB%A1%9C%EC%84%B8%EC%84%9C" title="마이크로프로세서">마이크로프로세서</a>의 논리 <a href="/wiki/%EC%A7%91%EC%A0%81_%ED%9A%8C%EB%A1%9C" title="집적 회로">IC</a>는 대부분 시모스 구조가 되었으며 , 작은 규모의 <a class="new" href="/w/index.php?title=%EC%A0%84%EC%9B%90_%ED%9A%8C%EB%A1%9C&amp;action=edit&amp;redlink=1" title="전원 회로 (없는 문서)">전원 회로</a>, <a href="/wiki/%EC%95%84%EB%82%A0%EB%A1%9C%EA%B7%B8-%EB%94%94%EC%A7%80%ED%84%B8_%EB%B3%80%ED%99%98%ED%9A%8C%EB%A1%9C" title="아날로그-디지털 변환회로">아날로그-디지털 변환회로</a>, <a href="/wiki/%EB%94%94%EC%A7%80%ED%84%B8-%EC%95%84%EB%82%A0%EB%A1%9C%EA%B7%B8_%EB%B3%80%ED%99%98%ED%9A%8C%EB%A1%9C" title="디지털-아날로그 변환회로">디지털-아날로그 변환회로</a> 등이 포함되어 제작하기 시작하였다.</p>
<h2><span class="mw-headline" id=".EC.8B.9C.EB.AA.A8.EC.8A.A4_.EB.85.BC.EB.A6.AC_.ED.91.9C.EC.A4.80_IC">시모스 <a class="new" href="/w/index.php?title=%EB%85%BC%EB%A6%AC_%ED%91%9C%EC%A4%80_IC&amp;action=edit&amp;redlink=1" title="논리 표준 IC (없는 문서)">논리 표준 IC</a></span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=CMOS&amp;action=edit&amp;section=2" title="부분 편집: 시모스 논리 표준 IC">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>단일 전원으로 시모스 수준의 입출력 인터페이스로 통일된 집적회로이다. (74HCT나 74ACT처럼 입력 논리 수준을 <a href="/wiki/%ED%8A%B8%EB%9E%9C%EC%A7%80%EC%8A%A4%ED%84%B0-%ED%8A%B8%EB%9E%9C%EC%A7%80%EC%8A%A4%ED%84%B0_%EB%85%BC%EB%A6%AC" title="트랜지스터-트랜지스터 논리">TTL</a>에 맞춘 형태도 있음.)</p>

<h3><span class="mw-headline" id=".EC.8B.9C.EB.AA.A8.EC.8A.A4_.EC.9E.85.EC.B6.9C.EB.A0.A5_.EC.A0.84.EC.95.95_.28V.29">시모스 입출력 전압 (V)</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=CMOS&amp;action=edit&amp;section=3" title="부분 편집: 시모스 입출력 전압 (V)">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<ul>
<li>Hi 레벨 입력 전압 : 0.7×Vcc</li>
<li>Low 레벨 입력 전압 : 0.2×Vcc</li>

<p>Vcc : 전원 전압</p>
<h2><span class="mw-headline" id=".EA.B7.B8_.EB.B0.96.EC.9D.98_.EC.98.88">그 밖의 예</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=CMOS&amp;action=edit&amp;section=4" title="부분 편집: 그 밖의 예">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<ul>
<li>이미지 장치 분야에서는 <a class="mw-redirect" href="/wiki/%EC%8B%9C%EB%AA%A8%EC%8A%A4_%EC%9D%B4%EB%AF%B8%EC%A7%80_%EC%84%BC%EC%84%9C" title="시모스 이미지 센서">시모스 이미지 센서</a>를 줄여서 시모스라고 하는 경우가 있다. 기존에 대중화된 CCD를 대체하게 되었다.</li>
<li><a href="/wiki/%EA%B0%9C%EC%9D%B8%EC%9A%A9_%EC%BB%B4%ED%93%A8%ED%84%B0" title="개인용 컴퓨터">개인용 컴퓨터</a>나 <a href="/wiki/%EC%9B%8C%ED%81%AC%EC%8A%A4%ED%85%8C%EC%9D%B4%EC%85%98" title="워크스테이션">워크스테이션</a>과 같은 작은 규모의 컴퓨터 세계에서 현재 시간이나 하드웨어 정보(<a class="mw-redirect" href="/wiki/BIOS" title="BIOS">BIOS</a>라고 함)를 보관하고 유지하기 위해 쓰이는 비휘발성 메모리를 시모스 롬, 또는 단순히 시모스라고 하는 경우도 있고 보관되어 있는 데이터 자체를 시모스라고 하기도 한다. 예를 들어 「메인보드가 동작하지 않을 때 CMOS를 초기화한다」라는 표현이 사용된다.</li>
</ul>
