<p><b>쑤저우</b>(<span style="font-size: smaller;"><a href="/wiki/%ED%95%9C%EA%B5%AD%EC%96%B4" title="한국어">한국어</a>:</span> <span lang="ko" xml:lang="ko">숙주</span>, <small><a href="/wiki/%EC%A4%91%EA%B5%AD%EC%96%B4" title="중국어">중국어</a></small><small>:</small> <span lang="zh" xml:lang="zh">宿州</span>, <small><a href="/wiki/%ED%95%9C%EC%96%B4_%EB%B3%91%EC%9D%8C" title="한어 병음">병음</a>:</small> <font face="Lucida Sans Unicode">Sùzhōu</font>)는 <a href="/wiki/%EC%95%88%ED%9B%84%EC%9D%B4_%EC%84%B1" title="안후이 성">안후이 성</a> 동쪽에 위치한 지급시이다. 서쪽으로 <a class="mw-redirect" href="/wiki/%ED%99%94%EC%9D%B4%EB%B2%A0%EC%9D%B4" title="화이베이">화이베이</a>, 남쪽으로 <a class="mw-redirect" href="/wiki/%EB%B2%99%EB%B6%80" title="벙부">벙부</a>와 접하고 다른 부분은 <a href="/wiki/%EC%9E%A5%EC%91%A4_%EC%84%B1" title="장쑤 성">장쑤 성</a>과 접한다.</p>
<p>해외의 관광객들은 자주 이 도시를 쑤저우라는 이름을 가진 다른 도시와 혼동한다. 장쑤 성에 위치한 <a class="mw-redirect" href="/wiki/%EC%91%A4%EC%A0%80%EC%9A%B0" title="쑤저우">쑤저우</a>는 그러나 음조가 다르다. 안후이의 쑤저우는 [sù zhōu]로 발음되는 반면, 장쑤의 쑤저우는 [sū zhōu]로 발음된다.</p>
<h2><span class="mw-headline" id=".EC.A7.80.EB.A6.AC">지리</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%91%A4%EC%A0%80%EC%9A%B0_%EC%8B%9C_(%EC%95%88%ED%9B%84%EC%9D%B4_%EC%84%B1)&amp;action=edit&amp;section=1" title="부분 편집: 지리">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>안후이 성의 최북부에 위치하고 <a href="/wiki/%EB%B2%99%EB%B6%80_%EC%8B%9C" title="벙부 시">벙부 시</a>, <a href="/wiki/%ED%99%94%EC%9D%B4%EB%B2%A0%EC%9D%B4_%EC%8B%9C" title="화이베이 시">화이베이 시</a>, <a href="/wiki/%EC%9E%A5%EC%91%A4_%EC%84%B1" title="장쑤 성">장쑤 성</a> <a href="/wiki/%EC%89%AC%EC%A0%80%EC%9A%B0_%EC%8B%9C" title="쉬저우 시">쉬저우 시</a>, <a href="/wiki/%EC%82%B0%EB%91%A5_%EC%84%B1" title="산둥 성">산둥 성</a> <a href="/wiki/%ED%97%88%EC%A9%8C_%EC%8B%9C" title="허쩌 시">허쩌 시</a>, <a href="/wiki/%ED%97%88%EB%82%9C_%EC%84%B1" title="허난 성">허난 성</a> <a href="/wiki/%EC%83%81%EC%B6%94_%EC%8B%9C" title="상추 시">상추 시</a>에 접한다. 화베이 평원의 동서 184.6km, 남북 151.2km의 범위를 차지하고 있다. 총 면적 중 평야는 8,890km²로 전체의 91%를 차지하고 나머지는 약간의 구릉과 대지이다.</p>
<p>시내에는 황하 수계와 화이허 수계에 속하는 70개 정도의 하천이 흐른다.</p>
<p>쑤저우 시의 기후는 온대습윤계절풍기후로 사계절이 분명하고 일조량도 충분하다. 연평균 기온은 14.3도, 무상일수는 약 210일, 강수량은 약 880mm이다.</p>
<p>시내는 석탄과 석유 자원이 있고 황커우 유전의 매장량은 20억 톤 내외로 전망된다. 그 외에도 철·금·구리 등의 금속 자원, 석회석·백운석·자토 등의 비금속 자원도 풍부하다.</p>
<h2><span class="mw-headline" id=".EC.97.AD.EC.82.AC">역사</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%91%A4%EC%A0%80%EC%9A%B0_%EC%8B%9C_(%EC%95%88%ED%9B%84%EC%9D%B4_%EC%84%B1)&amp;action=edit&amp;section=2" title="부분 편집: 역사">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>쑤저우의 역사는 오래돼서, 진대·한대에 쑤저우는 벌써 주자회취, 구주통구지지(舟車會聚，九州通衢之地) - 배나 수레가 모여 구주 각지로 통하는 땅 - 으로 불리고 있었다. 수대에는 황하와 회하를 연결하는 운하인 통제거가 완성되면서 쑤저우는 남북을 연결하는 군사의 요충지가 되었다.</p>
<p>진말부터 한초에 걸쳐 발생한 중국 최초의 대규모 농민 봉기인 <a href="/wiki/%EC%A7%84%EC%8A%B9%C2%B7%EC%98%A4%EA%B4%91%EC%9D%98_%EB%82%9C" title="진승·오광의 난">진승·오광의 난</a>은 현재의 쑤저우 시에서 시작되었다. 또 <a href="/wiki/%EC%B4%88%ED%95%9C%EC%A0%84%EC%9F%81" title="초한전쟁">초한전쟁</a> 마지막의 <a href="/wiki/%ED%95%B4%ED%95%98_%EC%A0%84%ED%88%AC" title="해하 전투">해하 전투</a>가 일어난 곳도 쑤저우 근처였다. 1948년 11월 6일부터 1949년 1월 6일에 걸쳐 <a href="/wiki/%EA%B5%AD%EA%B3%B5_%EB%82%B4%EC%A0%84" title="국공 내전">국공 내전</a>의 중요한 전투인 회하 전투의 주요 전장이 되어 <a href="/wiki/%EB%8D%A9%EC%83%A4%EC%98%A4%ED%95%91" title="덩샤오핑">덩샤오핑</a>, <a href="/wiki/%EB%A5%98%EB%B3%B4%EC%B2%AD" title="류보청">류보청</a>, <a class="mw-disambig" href="/wiki/%EC%B2%9C%EC%9D%B4" title="천이">천이</a> 등이 이곳에서 싸웠다.</p>
<p><a href="/wiki/%EC%A3%BC%EB%82%98%EB%9D%BC" title="주나라">주나라</a> 때는 숙나라가 있었고 진대에는 현이 두어졌다. 후한대에는 패국(현재의 <a class="mw-disambig" href="/wiki/%ED%8E%98%EC%9D%B4_%ED%98%84" title="페이 현">페이 현</a>)에 속했고 <a class="mw-redirect" href="/wiki/%EC%82%BC%EA%B5%AD%EC%8B%9C%EB%8C%80_(%EC%A4%91%EA%B5%AD)" title="삼국시대 (중국)">삼국시대</a>에는 <a href="/wiki/%EC%9C%84_(%EC%82%BC%EA%B5%AD)" title="위 (삼국)">위</a>의 예주의 군에 속했다. 809년, <a href="/wiki/%EB%8B%B9%EB%82%98%EB%9D%BC" title="당나라">당나라</a>에 의해 숙주가 놓여졌다. 용교에 숙주의 치소가 놓여졌고 부리, 홍, 기, 임환의 4현을 관할했다.</p>
<p>1912년에 쑤저우는 쑤셴으로 고쳐졌다. 1979년에 쑤저우 시가 설치되었고 1992년에 쑤 현을 병합했으며 1998년 12월 6일에 국무원은 쑤셴 지구 및 쑤저우 시를 통폐합해 지급시인 쑤저우 시를 만들었다.</p>
