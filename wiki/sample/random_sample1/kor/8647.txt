<p><b>쑨지하이</b> (<small><a href="/wiki/%EC%A4%91%EA%B5%AD%EC%96%B4" title="중국어">중국어</a></small> <small><a href="/wiki/%EA%B0%84%EC%B2%B4%EC%9E%90" title="간체자">간체</a>:</small> <span lang="zh-hans" xml:lang="zh-hans">孙继海</span>, <small><a href="/wiki/%EC%A0%95%EC%B2%B4%EC%9E%90" title="정체자">정체</a>:</small> <span lang="zh-hant" xml:lang="zh-hant">孫繼海</span>, <small><a href="/wiki/%ED%95%9C%EC%96%B4_%EB%B3%91%EC%9D%8C" title="한어 병음">병음</a>:</small> <font face="Lucida Sans Unicode">Sūn Jìhǎi</font>, <a href="/wiki/1977%EB%85%84" title="1977년">1977년</a> <a href="/wiki/9%EC%9B%94_30%EC%9D%BC" title="9월 30일">9월 30일</a> ~ )은 <a href="/wiki/%EC%A4%91%ED%99%94%EC%9D%B8%EB%AF%BC%EA%B3%B5%ED%99%94%EA%B5%AD" title="중화인민공화국">중화인민공화국</a>의 축구 선수이다. 포지션은 오른쪽 <a class="mw-redirect" href="/wiki/%ED%92%80%EB%B0%B1" title="풀백">풀백</a>이며, 현재 <a href="/wiki/%EC%A4%91%EA%B5%AD_%EC%8A%88%ED%8D%BC%EB%A6%AC%EA%B7%B8" title="중국 슈퍼리그">중국 슈퍼리그</a>의 <a class="mw-redirect" href="/wiki/%EC%82%B0%EC%8B%9C_%EC%A4%91%EC%A0%A0_%EC%B0%AC%EB%B0%94" title="산시 중젠 찬바">산시 중젠 찬바</a>에서 뛰고 있다. 그는 <a href="/wiki/%EC%A4%91%ED%99%94%EC%9D%B8%EB%AF%BC%EA%B3%B5%ED%99%94%EA%B5%AD_%EC%B6%95%EA%B5%AC_%EA%B5%AD%EA%B0%80%EB%8C%80%ED%91%9C%ED%8C%80" title="중화인민공화국 축구 국가대표팀">중화인민공화국 축구 국가대표팀</a>의 일원이기도 하다.</p>

href="/wiki/%ED%81%AC%EB%A6%AC%EC%8A%A4%ED%84%B8_%ED%8C%B0%EB%A6%AC%EC%8A%A4_FC" title="크리스털 팰리스 FC">크리스털 팰리스 FC</a>로 이적하였다. 하지만 인상적인 모습을 보이지 못하며 <a href="/wiki/1999%EB%85%84" title="1999년">1999년</a> 팀명이 변경된 친정팀 <a href="/wiki/%EB%8B%A4%EB%A1%84_%EC%8A%A4%EB%8D%94" title="다롄 스더">다롄 스더</a>로 복귀하였고, <a href="/wiki/2002%EB%85%84" title="2002년">2002년</a> <a class="mw-redirect" href="/wiki/%EC%9E%89%EA%B8%80%EB%9E%9C%EB%93%9C_%ED%94%84%EB%A6%AC%EB%AF%B8%EC%96%B4%EB%A6%AC%EA%B7%B8" title="잉글랜드 프리미어리그">잉글랜드 프리미어리그</a>의 <a href="/wiki/UEFA%EC%BB%B5_2003-04" title="UEFA컵 2003-04">UEFA컵 2003-04</a> 시즌에서는 <a href="/wiki/%EB%8D%94_%EB%89%B4_%EC%84%B8%EC%9D%B8%EC%B8%A0_FC" title="더 뉴 세인츠 FC">토탈 네트워크 솔루션</a>을 상대로 <a href="/wiki/%EC%A4%91%ED%99%94%EC%9D%B8%EB%AF%BC%EA%B3%B5%ED%99%94%EA%B5%AD" title="중화인민공화국">중화인민공화국</a> 출신으로는 처음으로 유럽 대회에서의 골을 기록하였다. 좋은 모습을 보인 뒤 <a href="/wiki/2008%EB%85%84" title="2008년">2008년</a> <a href="/wiki/7%EC%9B%94_2%EC%9D%BC" title="7월 2일">7월 2일</a> <a href="/wiki/%ED%92%8B%EB%B3%BC_%EB%A6%AC%EA%B7%B8_%EC%B1%94%ED%94%BC%EC%96%B8%EC%8B%AD" title="풋볼 리그 챔피언십">풋볼 리그 챔피언십</a>의 <a href="/wiki/%EC%85%B0%ED%95%84%EB%93%9C_%EC%9C%A0%EB%82%98%EC%9D%B4%ED%8B%B0%EB%93%9C_FC" title="셰필드 유나이티드 FC">셰필드 유나이티드 FC</a>로 이적하였고, <a href="/wiki/2009%EB%85%84" title="2009년">2009년</a> <a href="/wiki/%EC%A4%91%EA%B5%AD_%EC%8A%88%ED%8D%BC%EB%A6%AC%EA%B7%B8" title="중국 슈퍼리그">중국 슈퍼리그</a>의 <a class="mw-redirect" href="/wiki/%EC%B2%AD%EB%91%90_%EC%85%B0%ED%8E%98%EC%9D%B4%EB%A1%84" title="청두 셰페이롄">청두 블레이즈</a>로 이적하여 <a href="/wiki/%EC%A4%91%ED%99%94%EC%9D%B8%EB%AF%BC%EA%B3%B5%ED%99%94%EA%B5%AD" title="중화인민공화국">중화인민공화국</a> 무대에 복귀하였다. <a href="/wiki/2010%EB%85%84" title="2010년">2010년</a> <a class="mw-redirect" href="/wiki/%EC%82%B0%EC%8B%9C_%EC%A4%91%EC%A0%A0_%EC%B0%AC%EB%B0%94" title="산시 중젠 찬바">산시 중젠 찬바</a>로 다시 이적하였다.</p>
<h3><span class="mw-headline" id=".EA.B5.AD.EA.B0.80.EB.8C.80.ED.91.9C.ED.8C.80_.EA.B2.BD.EB.A0.A5">국가대표팀 경력</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%91%A8%EC%A7%80%ED%95%98%EC%9D%B4&amp;action=edit&amp;section=3" title="부분 편집: 국가대표팀 경력">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>쑨지하이는 <a href="/wiki/1996%EB%85%84" title="1996년">1996년</a> <a href="/wiki/12%EC%9B%94_6%EC%9D%BC" title="12월 6일">12월 6일</a> <a href="/wiki/%EC%9A%B0%EC%A6%88%EB%B2%A0%ED%82%A4%EC%8A%A4%ED%83%84_%EC%B6%95%EA%B5%AC_%EA%B5%AD%EA%B0%80%EB%8C%80%ED%91%9C%ED%8C%80" title="우즈베키스탄 축구 국가대표팀">우즈베키스탄</a>과의 경기로 <a href="/wiki/%EC%A4%91%ED%99%94%EC%9D%B8%EB%AF%BC%EA%B3%B5%ED%99%94%EA%B5%AD_%EC%B6%95%EA%B5%AC_%EA%B5%AD%EA%B0%80%EB%8C%80%ED%91%9C%ED%8C%80" title="중화인민공화국 축구 국가대표팀">국가대표팀</a>에 데뷔하였고, <a href="/wiki/%EB%B3%B4%EB%9D%BC_%EB%B0%80%EB%A3%A8%ED%8B%B0%EB%85%B8%EB%B9%84%EC%B9%98" title="보라 밀루티노비치">보라 밀루티노비치</a> 감독의 신임을 얻어 <a href="/wiki/2002%EB%85%84_FIFA_%EC%9B%94%EB%93%9C%EC%BB%B5" title="2002년 FIFA 월드컵">2002년 FIFA 월드컵</a>에서도 참가하였다. 하지만 <a href="/wiki/%EC%BD%94%EC%8A%A4%ED%83%80%EB%A6%AC%EC%B9%B4_%EC%B6%95%EA%B5%AC_%EA%B5%AD%EA%B0%80%EB%8C%80%ED%91%9C%ED%8C%80" title="코스타리카 축구 국가대표팀">코스타리카</a>와의 조별 예선 첫 경기에서 17분 <a class="new" href="/w/index.php?title=%EB%A7%88%EC%9A%B0%EB%A6%AC%EC%8B%9C%EC%98%A4_%EC%86%94%EB%A6%AC%EC%8A%A4&amp;action=edit&amp;redlink=1" title="마우리시오 솔리스 (없는 문서)">마우리시오 솔리스</a>에게 발목 부상을 당하여 25분 교체되었고, 이후 남은 두 경기에 출전하지 못하였다.</p>
