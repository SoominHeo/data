<p><b>쓰부라야 고키치</b>(<span style="font-size: smaller;"><a href="/wiki/%EC%9D%BC%EB%B3%B8%EC%96%B4" title="일본어">일본어</a>:</span> <span lang="ja"><span lang="ja"><ruby><rb>円谷 幸吉</rb> <rp>(</rp><rt>つぶらや こうきち</rt><rp>)</rp></ruby></span></span>, <a href="/wiki/1940%EB%85%84" title="1940년">1940년</a> <a href="/wiki/5%EC%9B%94_13%EC%9D%BC" title="5월 13일">5월 13일</a> ~ <a href="/wiki/1968%EB%85%84" title="1968년">1968년</a> <a href="/wiki/1%EC%9B%94_9%EC%9D%BC" title="1월 9일">1월 9일</a>)는 주로 <a href="/wiki/%EB%A7%88%EB%9D%BC%ED%86%A4" title="마라톤">마라톤</a> 선수로 활약한 <a href="/wiki/%EC%9D%BC%EB%B3%B8" title="일본">일본</a>의 <a class="mw-redirect" href="/wiki/%EC%9C%A1%EC%83%81" title="육상">육상</a> 선수였다. 그는 <a href="/wiki/%EC%9C%A1%EC%83%81%EC%9E%90%EC%9C%84%EB%8C%80" title="육상자위대">육상자위대</a>의 <a href="/wiki/%EC%A4%91%EC%9C%84" title="중위">중위</a>이기도 하였다.</p>
<h2><span class="mw-headline" id=".EC.83.9D.EC.95.A0">생애</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%93%B0%EB%B6%80%EB%9D%BC%EC%95%BC_%EA%B3%A0%ED%82%A4%EC%B9%98&amp;action=edit&amp;section=1" title="부분 편집: 생애">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>본명은 <b>쓰무라야 고키치</b>로 <a href="/wiki/%ED%9B%84%EC%BF%A0%EC%8B%9C%EB%A7%88_%ED%98%84" title="후쿠시마 현">후쿠시마 현</a> <a href="/wiki/%EC%8A%A4%EC%B9%B4%EA%B0%80%EC%99%80_%EC%8B%9C" title="스카가와 시">스카가와 시</a>에서 태어난 쓰부라야는 <a class="mw-redirect" href="/wiki/1964%EB%85%84_%EB%8F%84%EC%BF%84_%EC%98%AC%EB%A6%BC%ED%94%BD" title="1964년 도쿄 올림픽">1964년 도쿄 올림픽</a>에 나가 10000m 종목에서 6위를 하였다. 마라톤에 나간 그는 경주에서 선두로 달렸으나 마지막 100m 안에서 <a href="/wiki/%EC%98%81%EA%B5%AD" title="영국">영국</a>의 <a class="new" href="/w/index.php?title=%EB%B2%A0%EC%9D%B4%EC%A7%88_%ED%9E%88%ED%8B%80%EB%A6%AC&amp;action=edit&amp;redlink=1" title="베이질 히틀리 (없는 문서)">베이질 히틀리</a>에 의하여 따라잡혀 3위로 들어왔다. 히틀리에게 패배에 의하여 굴욕을 느낀 쓰부라야는 동료 마라톤 선수 <a class="new" href="/w/index.php?title=%EA%B8%B0%EB%AF%B8%ED%95%98%EB%9D%BC_%EA%B2%90%EC%A7%80&amp;action=edit&amp;redlink=1" title="기미하라 겐지 (없는 문서)">기미하라 겐지</a>에게 이렇게 말하였다.</p>
<p>"난 일본 국민들의 앞에서 용서할 수 없는 큰 실수를 저질렀다. 난 <a href="/wiki/%EB%A9%95%EC%8B%9C%EC%BD%94" title="멕시코">멕시코</a>에서 열리는 다음 올림픽에서 달리고 <a class="mw-redirect" href="/wiki/%EC%9D%BC%EC%9E%A5%EA%B8%B0" title="일장기">일장기</a>를 올리면서 보상을 만들어야 해."</p>
<p>도쿄 올림픽이 끝나자마자, 그는 요통을 겪었다. <a href="/wiki/1968%EB%85%84" title="1968년">1968년</a> <a href="/wiki/1%EC%9B%94_9%EC%9D%BC" title="1월 9일">1월 9일</a> 멕시코시티 올림픽을 위한 자신의 훈련 기간 동안에 자신이 머물던 기숙사에서 자신의 <a href="/wiki/%EC%86%90%EB%AA%A9" title="손목">손목</a>을 베어 자살하였다.</p>
<p>그의 자살 각서에는 이렇게 쓰여있었다. "난 더 이상 달리는 데 너무나 지쳤다. 날 용서해라. 난 나의 부모님에게 근심과 걱정을 시켜 미안하지만, 이것이 최고를 위해서이다. 나에게 해 준 모든 것에 대단히 감사하다." 그는 27세였다.</p>
