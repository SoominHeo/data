<p><b>쓰시마 공항</b>(<span style="font-size: smaller;"><a href="/wiki/%EC%9D%BC%EB%B3%B8%EC%96%B4" title="일본어">일본어</a>:</span> <span lang="ja"><span lang="ja"><ruby><rb>対馬空港</rb> <rp>(</rp><rt>つしまくうこう</rt><rp>)</rp></ruby></span></span>)은 <a href="/wiki/%EC%9D%BC%EB%B3%B8" title="일본">일본</a> <a href="/wiki/%EB%82%98%EA%B0%80%EC%82%AC%ED%82%A4_%ED%98%84" title="나가사키 현">나가사키 현</a> <a class="mw-disambig" href="/wiki/%EC%93%B0%EC%8B%9C%EB%A7%88_%EC%8B%9C" title="쓰시마 시">쓰시마 시</a>에 있는 <a href="/wiki/%EA%B3%B5%ED%95%AD" title="공항">공항</a>이다.</p>
<p><a href="/wiki/1975%EB%85%84" title="1975년">1975년</a> <a href="/wiki/%ED%99%9C%EC%A3%BC%EB%A1%9C" title="활주로">활주로</a> 1,500m를 사용하는 공항으로 개항하여, <a href="/wiki/%EC%A0%84%EC%9D%BC%EB%B3%B8%EA%B3%B5%EC%88%98" title="전일본공수">전일본공수</a>가 처음으로 <a href="/wiki/%ED%9B%84%EC%BF%A0%EC%98%A4%EC%B9%B4_%EA%B3%B5%ED%95%AD" title="후쿠오카 공항">후쿠오카 공항</a>과 쓰시마 공항을 잇는 항공 노선을 취항했다. <a href="/wiki/1983%EB%85%84" title="1983년">1983년</a> 활주로를 1,900m 규격으로 연장했다. 한국 노선으로는 <a href="/wiki/2009%EB%85%84" title="2009년">2009년</a>부터 <a href="/wiki/%EB%8C%80%ED%95%9C%EB%AF%BC%EA%B5%AD" title="대한민국">대한민국</a>의 <a href="/wiki/%EC%BD%94%EB%A6%AC%EC%95%84_%EC%9D%B5%EC%8A%A4%ED%94%84%EB%A0%88%EC%8A%A4_%EC%97%90%EC%96%B4" title="코리아 익스프레스 에어">코리아 익스프레스 에어</a>에서 <a href="/wiki/%EA%B9%80%ED%8F%AC%EA%B5%AD%EC%A0%9C%EA%B3%B5%ED%95%AD" title="김포국제공항">김포국제공항</a> 및 <a href="/wiki/%EA%B9%80%ED%95%B4%EA%B5%AD%EC%A0%9C%EA%B3%B5%ED%95%AD" title="김해국제공항">김해국제공항</a>에서 출발하는 국제 노선을 운영하기도 하였으나 현재는 종료된 상태이다.</p>
