<p><b>루이즈 캐롤라인 앨버타</b>, 아가일 공작 부인(Louise Caroline Alberta, Duchess of Argyll, <a href="/wiki/1848%EB%85%84" title="1848년">1848년</a> <a href="/wiki/3%EC%9B%94_18%EC%9D%BC" title="3월 18일">3월 18일</a> ~ <a href="/wiki/1939%EB%85%84" title="1939년">1939년</a> <a href="/wiki/12%EC%9B%94_3%EC%9D%BC" title="12월 3일">12월 3일</a>)은 <a href="/wiki/%EC%98%81%EA%B5%AD" title="영국">영국</a>의 공주로 <a href="/wiki/%EB%B9%85%ED%86%A0%EB%A6%AC%EC%95%84_(%EC%98%81%EA%B5%AD)" title="빅토리아 (영국)">빅토리아 여왕</a>과 <a href="/wiki/%EC%9E%91%EC%84%BC%EC%BD%94%EB%B6%80%EB%A5%B4%ED%81%AC%EA%B3%A0%ED%83%80%EC%9D%98_%EC%95%A8%EB%B2%84%ED%8A%B8" title="작센코부르크고타의 앨버트">앨버트 공</a>의 넷째딸이다. 형제로 프로이센 왕비 <a href="/wiki/%ED%94%84%EB%A6%B0%EC%84%B8%EC%8A%A4_%EB%A1%9C%EC%97%B4_%EB%B9%85%ED%86%A0%EB%A6%AC%EC%95%84" title="프린세스 로열 빅토리아">빅토리아</a>, 헤센 대공비 <a href="/wiki/%EC%98%81%EA%B5%AD%EC%9D%98_%EC%95%8C%EB%A6%AC%EC%B2%B4" title="영국의 알리체">앨리스</a>, 영국 군주 <a href="/wiki/%EC%97%90%EB%93%9C%EC%9B%8C%EB%93%9C_7%EC%84%B8" title="에드워드 7세">에드워드 7세</a>, 코넛 공작 <a class="new" href="/w/index.php?title=%EC%BD%94%EB%84%9B_%EA%B3%B5%EC%9E%91_%EC%95%84%EC%84%9C&amp;action=edit&amp;redlink=1" title="코넛 공작 아서 (없는 문서)">아서</a> 등이 있다. <a href="/wiki/1871%EB%85%84" title="1871년">1871년</a> <a href="/wiki/3%EC%9B%94_21%EC%9D%BC" title="3월 21일">3월 21일</a> <a href="/wiki/%EC%9C%88%EC%A0%80_%EC%84%B1" title="윈저 성">윈저 성</a>에서 아가일 공작의 아들 론 후작 <a href="/wiki/%EC%A0%9C9%EB%8C%80_%EC%95%84%EA%B0%80%EC%9D%BC_%EA%B3%B5%EC%9E%91_%EC%A1%B4_%EC%BA%A0%EB%B2%A8" title="제9대 아가일 공작 존 캠벨">존</a>과 결혼했다. <a href="/wiki/1900%EB%85%84" title="1900년">1900년</a> 시아버지 아가일 공작이 죽고 남편이 공작위를 계승하여 아가일 공작 부인이 되었다. 남편과의 사이에 자식은 없었다.</p>
