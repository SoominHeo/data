<p><b>아나톨리 안드레예비치 바니솁스키</b>(<span style="font-size: smaller;"><a href="/wiki/%EB%9F%AC%EC%8B%9C%EC%95%84%EC%96%B4" title="러시아어">러시아어</a>:</span> <span lang="ru" xml:lang="ru">Анатолий Андреевич Банишевский</span>, <span style="font-size: smaller;"><a href="/wiki/%EC%95%84%EC%A0%9C%EB%A5%B4%EB%B0%94%EC%9D%B4%EC%9E%94%EC%96%B4" title="아제르바이잔어">아제르바이잔어</a>:</span> <span lang="az" xml:lang="az">Anatoli Andreyeviç Banişevski</span>, <a href="/wiki/1946%EB%85%84" title="1946년">1946년</a> <a href="/wiki/2%EC%9B%94_23%EC%9D%BC" title="2월 23일">2월 23일</a> ~ <a href="/wiki/1997%EB%85%84" title="1997년">1997년</a> <a href="/wiki/12%EC%9B%94_10%EC%9D%BC" title="12월 10일">12월 10일</a>)는 <a href="/wiki/%EC%95%84%EC%A0%9C%EB%A5%B4%EB%B0%94%EC%9D%B4%EC%9E%94" title="아제르바이잔">아제르바이잔</a>의 전직 축구 선수로, 포지션은 <a href="/wiki/%EA%B3%B5%EA%B2%A9%EC%88%98" title="공격수">스트라이커</a>였다.</p>
<p><a class="mw-redirect" href="/wiki/%EC%86%8C%EB%B9%84%EC%97%90%ED%8A%B8_%EC%97%B0%EB%B0%A9_%EC%B6%95%EA%B5%AC_%EA%B5%AD%EA%B0%80%EB%8C%80%ED%91%9C%ED%8C%80" title="소비에트 연방 축구 국가대표팀">소비에트 연방 대표팀</a>에서 51경기에 출전하여 19득점을 기록했다. <a href="/wiki/1966%EB%85%84_FIFA_%EC%9B%94%EB%93%9C%EC%BB%B5" title="1966년 FIFA 월드컵">1966년 FIFA 월드컵</a>과 <a class="mw-redirect" href="/wiki/1968%EB%85%84_%EC%9C%A0%EB%9F%BD%EC%B6%95%EA%B5%AC%EC%84%A0%EC%88%98%EA%B6%8C%EB%8C%80%ED%9A%8C" title="1968년 유럽축구선수권대회">1968년 유럽축구선수권대회</a>에서 소비에트 연방이 4위를 차지하는 데에 기여했고 <a class="mw-redirect" href="/wiki/1972%EB%85%84_%EC%9C%A0%EB%9F%BD%EC%B6%95%EA%B5%AC%EC%84%A0%EC%88%98%EA%B6%8C%EB%8C%80%ED%9A%8C" title="1972년 유럽축구선수권대회">1972년 유럽축구선수권대회</a>에서 소비에트 연방을 준우승으로 이끄는 데에 기여했다. <a class="mw-redirect" href="/wiki/%EB%84%A4%ED%94%84%EC%B9%98_%EB%B0%94%EC%BF%A0_PFC" title="네프치 바쿠 PFC">네프치 바쿠 PFC</a> 소속으로 있는 동안 <a class="mw-redirect" href="/wiki/%EC%86%8C%EB%B9%84%EC%97%90%ED%8A%B8_%ED%86%B1_%EB%A6%AC%EA%B7%B8" title="소비에트 톱 리그">소비에트 톱 리그</a>에서 136득점을 기록했으며 세 차례(1966년, 1967년, 1978년)나 아제르바이잔 올해의 축구 선수로 선정되기도 했다.</p>
<p><a href="/wiki/2003%EB%85%84" title="2003년">2003년</a> <a class="new" href="/w/index.php?title=%EC%95%84%EC%A0%9C%EB%A5%B4%EB%B0%94%EC%9D%B4%EC%9E%94_%EC%B6%95%EA%B5%AC_%ED%98%91%ED%9A%8C&amp;action=edit&amp;redlink=1" title="아제르바이잔 축구 협회 (없는 문서)">아제르바이잔 축구 협회</a>로부터 <a href="/wiki/UEFA_%EC%B0%BD%EB%A6%BD_50%EC%A3%BC%EB%85%84_%EA%B8%B0%EB%85%90%EC%83%81" title="UEFA 창립 50주년 기념상">UEFA 창립 50주년 기념상</a> 수상자로 선정되었다.</p>
