아나톨리 솝차크의 얼굴이 그려진 우표</div>
</div>
</div>
<p><b>아나톨리 알렉산드로비치 솝차크</b>(<span style="font-size: smaller;"><a href="/wiki/%EB%9F%AC%EC%8B%9C%EC%95%84%EC%96%B4" title="러시아어">러시아어</a>:</span> <span lang="ru" xml:lang="ru">Анато́лий Алекса́ндрович Собча́к</span>, <a href="/wiki/1937%EB%85%84" title="1937년">1937년</a> <a href="/wiki/8%EC%9B%94_10%EC%9D%BC" title="8월 10일">8월 10일</a> ~ <a href="/wiki/2000%EB%85%84" title="2000년">2000년</a> <a href="/wiki/2%EC%9B%94_20%EC%9D%BC" title="2월 20일">2월 20일</a>)는 <a href="/wiki/%EB%9F%AC%EC%8B%9C%EC%95%84" title="러시아">러시아</a>의 정치인이자 <a href="/wiki/%EB%9F%AC%EC%8B%9C%EC%95%84_%ED%97%8C%EB%B2%95" title="러시아 헌법">러시아 연방 헌법</a>을 작성한 사람이다. 그는 민주적으로 선출된 제1대 <a href="/wiki/%EC%83%81%ED%8A%B8%ED%8E%98%ED%85%8C%EB%A5%B4%EB%B6%80%EB%A5%B4%ED%81%AC" title="상트페테르부르크">상트페테르부르크</a> 시장이며 <a href="/wiki/%EB%B8%94%EB%9D%BC%EB%94%94%EB%AF%B8%EB%A5%B4_%ED%91%B8%ED%8B%B4" title="블라디미르 푸틴">블라디미르 푸틴</a>과 <a href="/wiki/%EB%93%9C%EB%AF%B8%ED%8A%B8%EB%A6%AC_%EB%A9%94%EB%93%9C%EB%B2%A0%EB%8D%B0%ED%94%84" title="드미트리 메드베데프">드미트리 메드베데프</a>의 스승이기도 하다.</p>
<p><a href="/wiki/1937%EB%85%84" title="1937년">1937년</a> <a class="mw-redirect" href="/wiki/%EC%9E%90%EB%B0%94%EC%9D%B4%EC%B9%BC%EC%8A%A4%ED%82%A4_%EC%A7%80%EB%B0%A9" title="자바이칼스키 지방">자바이칼스키 지방</a> <a href="/wiki/%EC%B9%98%ED%83%80_(%EB%8F%84%EC%8B%9C)" title="치타 (도시)">치타</a>에서 네 명의 형제들 가운데 장남으로 태어났다. 아버지 알렉산드르 안토노비치는 <a href="/wiki/%EC%B2%A0%EB%8F%84" title="철도">철도</a> <a href="/wiki/%EA%B8%B0%EC%88%A0%EC%9E%90" title="기술자">기술자</a>였고 어머니 나데즈다 안드레예브나 리트비노바는 회계사였다. <a href="/wiki/1939%EB%85%84" title="1939년">1939년</a> 솝차크의 가족들은 <a href="/wiki/%EC%9A%B0%EC%A6%88%EB%B2%A0%ED%82%A4%EC%8A%A4%ED%83%84" title="우즈베키스탄">우즈베키스탄</a>으로 이주했으며 솝차크는 <a href="/wiki/1953%EB%85%84" title="1953년">1953년</a>까지 거주했다. 그곳에서 그는 <a href="/wiki/%EC%8A%A4%ED%83%80%EB%B8%8C%EB%A1%9C%ED%8F%B4" title="스타브로폴">스타브로폴</a> 법학 대학교을 다녔지만 <a href="/wiki/1954%EB%85%84" title="1954년">1954년</a> <a href="/wiki/%EC%83%81%ED%8A%B8%ED%8E%98%ED%85%8C%EB%A5%B4%EB%B6%80%EB%A5%B4%ED%81%AC_%EB%8C%80%ED%95%99%EA%B5%90" title="상트페테르부르크 대학교">상트페테르부르크 대학교</a>로 전학을 갔다. <a href="/wiki/1958%EB%85%84" title="1958년">1958년</a>에는 논나 간주크와 결혼했고 <a href="/wiki/1980%EB%85%84" title="1980년">1980년</a>에는 류드밀라 나루소바와 결혼했다.</p>
<p>2000년 2월 20일 대통령 선거에 출마한 블라디미르 푸틴을 지원하기 위한 활동을 벌이던 도중에 <a href="/wiki/%EC%B9%BC%EB%A6%AC%EB%8B%8C%EA%B7%B8%EB%9D%BC%EB%93%9C_%EC%A3%BC" title="칼리닌그라드 주">칼리닌그라드 주</a> <a href="/wiki/%EC%8A%A4%EB%B2%A0%ED%8B%80%EB%A1%9C%EA%B3%A0%EB%A5%B4%EC%8A%A4%ED%81%AC_(%EC%B9%BC%EB%A6%AC%EB%8B%8C%EA%B7%B8%EB%9D%BC%EB%93%9C_%EC%A3%BC)" title="스베틀로고르스크 (칼리닌그라드 주)">스베틀로고르스크</a>에서 심장마비로 사망했다. 그의 딸인 마리아 솝차크는 <a href="/wiki/%EC%83%81%ED%8A%B8%ED%8E%98%ED%85%8C%EB%A5%B4%EB%B6%80%EB%A5%B4%ED%81%AC" title="상트페테르부르크">상트페테르부르크</a>에서 <a href="/wiki/%EB%B3%80%ED%98%B8%EC%82%AC" title="변호사">변호사</a>로 활동하고 있다.</p>
