<p>《<b>아돌프에게 고한다</b>》(<span style="font-size: smaller;"><a href="/wiki/%EC%9D%BC%EB%B3%B8%EC%96%B4" title="일본어">일본어</a>:</span> <span lang="ja" xml:lang="ja">アドルフに告ぐ</span> <small>아도루푸니 쓰구<sup>[<a href="/wiki/%EC%9C%84%ED%82%A4%EB%B0%B1%EA%B3%BC:%EC%9D%BC%EB%B3%B8%EC%96%B4%EC%9D%98_%ED%95%9C%EA%B8%80_%ED%91%9C%EA%B8%B0" title="위키백과:일본어의 한글 표기">*</a>]</sup></small>)는 <a class="mw-redirect" href="/wiki/%ED%85%8C%EC%A6%88%EC%B9%B4_%EC%98%A4%EC%82%AC%EB%AC%B4" title="테즈카 오사무">테즈카 오사무</a>가 창작한 <a href="/wiki/%EB%A7%8C%ED%99%94" title="만화">만화</a>이다. 1986년에 <a href="/wiki/%EA%B3%A0%EB%8B%A8%EC%83%A4_%EB%A7%8C%ED%99%94%EC%83%81" title="고단샤 만화상">고단샤 만화상</a>을 수상하였다. 흔히 악을 응징한다면서 폭력을 정당화하지만, 폭력은 또 다른 미움과 폭력을 만들어낼 뿐이라는 비폭력사상이 주제이다. 만화는 <a href="/wiki/%EC%A0%9C2%EC%B0%A8_%EC%84%B8%EA%B3%84_%EB%8C%80%EC%A0%84" title="제2차 세계 대전">제2차 세계 대전</a> 이전의 세계를 배경으로 "아돌프"(Adolf)라는 이름을 가진 세 명의 남자를 따라가면서 진행된다. 첫 번째 아돌프는 일본에 거주하고 있는 <a class="mw-redirect" href="/wiki/%EC%95%84%EC%8A%88%EC%BC%80%EB%82%98%EC%A7%90" title="아슈케나짐">아슈케나짐</a> 아돌프 카밀이며, 두 번째 아돌프는 아돌프 카우프만, 마지막 아돌프는 <a href="/wiki/%EB%82%98%EC%B9%98_%EB%8F%85%EC%9D%BC" title="나치 독일">나치 독일</a>의 독재자 <a href="/wiki/%EC%95%84%EB%8F%8C%ED%94%84_%ED%9E%88%ED%8B%80%EB%9F%AC" title="아돌프 히틀러">아돌프 히틀러</a>이다.</p>
<h2><span class="mw-headline" id=".EC.A4.84.EA.B1.B0.EB.A6.AC">줄거리</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%95%84%EB%8F%8C%ED%94%84%EC%97%90%EA%B2%8C_%EA%B3%A0%ED%95%9C%EB%8B%A4&amp;action=edit&amp;section=1" title="부분 편집: 줄거리">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>《아돌프에게 고한다》는 1936년, 일본의 기자 도게 소헤이(峠草平)가 <a class="mw-redirect" href="/wiki/1936%EB%85%84_%EB%B2%A0%EB%A5%BC%EB%A6%B0_%EC%98%AC%EB%A6%BC%ED%94%BD" title="1936년 베를린 올림픽">베를린 올림픽</a>을 취재하러 가면서 시작된다. 독일에 도착한 도게는 유학중인 남동생이 살해당했다는 것을 알게 된다. 거기다 마치 동생이 원래 존재하지 않았다는듯 모든 흔적과 증거가 지워져 있었다. 도게는 동생이 일본에 부친 아돌프 히틀러와 관련된 정보가 담긴 편지 때문에 살해당했다는 것을 알게 된다. 이 정보는 제3제국을 무너뜨릴 수 있는 치명적인 정보였다.</p>
<h2><span class="mw-headline" id=".EB.93.B1.EC.9E.A5.EC.9D.B8.EB.AC.BC">등장인물</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%95%84%EB%8F%8C%ED%94%84%EC%97%90%EA%B2%8C_%EA%B3%A0%ED%95%9C%EB%8B%A4&amp;action=edit&amp;section=2" title="부분 편집: 등장인물">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<ul>
<li><b><a href="/wiki/%EC%95%84%EB%8F%8C%ED%94%84_%ED%9E%88%ED%8B%80%EB%9F%AC" title="아돌프 히틀러">아돌프 히틀러</a></b>: 독일의 독재자.</li>
<li><b>아돌프 카우프만</b>: 독일인 부친과 일본인 모친의 혼혈. 처음에는 나치즘에 관심이 없었으나 <a class="mw-redirect" href="/wiki/%ED%9E%88%ED%8B%80%EB%9F%AC_%EC%9C%A0%EA%B2%90%ED%8A%B8" title="히틀러 유겐트">유겐트</a>를 거쳐 <a class="new" href="/w/index.php?title=%EB%B3%B4%EC%95%88%EB%8C%80&amp;action=edit&amp;redlink=1" title="보안대 (없는 문서)">보안대</a> 대원, <a href="/wiki/%EA%B2%8C%EC%8A%88%ED%83%80%ED%8F%AC" title="게슈타포">게슈타포</a>가 되어 유대인을 탄압한다.</li>
<li><b>아돌프 카밀</b>: <a href="/wiki/%EC%9D%BC%EB%B3%B8" title="일본">일본</a>에 살고 있는 독일계 유대인. 아돌프 카우프만과 어릴 적 친구였으나, 역사의 흐름에 따라 반목하게 된다. 자신의 부친이 <a class="mw-redirect" href="/wiki/%EB%82%98%EC%B9%98%EB%8F%85%EC%9D%BC" title="나치독일">나치독일</a>로부터 죽임당한 것처럼 자신도 <a href="/wiki/%ED%8C%94%EB%A0%88%EC%8A%A4%ED%83%80%EC%9D%B8" title="팔레스타인">팔레스타인</a> 사람들에게 <a href="/wiki/%ED%8F%AD%EB%A0%A5" title="폭력">폭력</a>을 휘두른다. 부모님과 같이 <a href="/wiki/%EA%B5%90%ED%86%A0" title="교토">교토</a> <a href="/wiki/%EC%9C%A0%EB%8C%80%EC%9D%B8" title="유대인">유대인</a> 공동체에서 살았는데, <a class="mw-redirect" href="/wiki/%EB%82%98%EC%B9%98%EB%8F%85%EC%9D%BC" title="나치독일">나치독일</a>과 <a class="mw-redirect" href="/wiki/%EC%97%B0%ED%95%A9%EA%B5%B0" title="연합군">연합군</a>의 폭력으로 약혼자와 부모님이 희생된다.</li>
<li><b>도게 소헤이</b>: 일본인 <a class="mw-disambig" href="/wiki/%EA%B8%B0%EC%9E%90" title="기자">기자</a>. <a href="/wiki/1936%EB%85%84" title="1936년">1936년</a> <a class="mw-redirect" href="/wiki/%EB%B2%A0%EB%A5%BC%EB%A6%B0_%EC%98%AC%EB%A6%BC%ED%94%BD" title="베를린 올림픽">베를린 올림픽</a>을 취재하러 갔다가 동생이 살해당하자 그 비밀을 파헤치던 도중, <a href="/wiki/%EC%A0%9C2%EC%B0%A8_%EC%84%B8%EA%B3%84_%EB%8C%80%EC%A0%84" title="제2차 세계 대전">제2차 세계 대전</a>에 휘말리게 된다. 아돌프 카우프만의 어머니와 재혼하여 끝까지 사랑한다. 연합군의 공습으로 청각<a href="/wiki/%EC%9E%A5%EC%95%A0%EC%9D%B8" title="장애인">장애인</a>이 되는데, 다른 분들은 더 심한 <a class="mw-redirect" href="/wiki/%EC%9E%A5%EC%95%A0" title="장애">장애</a>를 갖고 있는데 자신은 그렇지 않으니 다행이라고 생각함으로써 장애를 받아들인다.</li>
<li><b><a href="/wiki/%EB%A6%AC%ED%95%98%EB%A5%B4%ED%8A%B8_%EC%A1%B0%EB%A5%B4%EA%B2%8C" title="리하르트 조르게">리하르트 조르게</a></b>: <a href="/wiki/%EC%9D%BC%EB%B3%B8_%EC%A0%9C%EA%B5%AD" title="일본 제국">일본 제국</a> 내에 암약하고 있던 <a href="/wiki/%EC%86%8C%EB%A0%A8" title="소련">소련</a>의 <a class="mw-redirect" href="/wiki/%EC%8A%A4%ED%8C%8C%EC%9D%B4" title="스파이">스파이</a>. 조르게의 스파이단이 추출해낸 정보는 이야기의 마지막 순간까지 결정적 역할을 한다.</li>
<li><b><a href="/wiki/%EC%97%90%EB%A5%B4%EB%B9%88_%EB%A1%AC%EB%A9%9C" title="에르빈 롬멜">에르빈 롬멜</a></b>: 아프리카 군단을 지휘한 독일의 국민적 영웅. <a class="mw-redirect" href="/wiki/%ED%9E%88%ED%8B%80%EB%9F%AC_%EC%95%94%EC%82%B4_%EB%AF%B8%EC%88%98_%EC%82%AC%EA%B1%B4" title="히틀러 암살 미수 사건">히틀러 암살 미수 사건</a>에 연루되어 자살을 빙자해 친위대에게 살해당한다.</li>
<li><b><a href="/wiki/%EB%B9%8C%ED%97%AC%EB%A6%84_%EC%B9%B4%EC%9D%B4%ED%85%94" title="빌헬름 카이텔">빌헬름 카이텔</a></b></li>
<li><b><a href="/wiki/%EC%9A%94%EC%A0%9C%ED%94%84_%EA%B4%B4%EB%B2%A8%EC%8A%A4" title="요제프 괴벨스">요제프 괴벨스</a></b></li>
<li><b><a href="/wiki/%EC%95%84%EB%8F%8C%ED%94%84_%EC%95%84%EC%9D%B4%ED%9E%88%EB%A7%8C" title="아돌프 아이히만">아돌프 아이히만</a></b>: 유대인 학살의 실무 책임자. 아돌프 카우프만에게 이송되는 유대인들을 중간에 죽여 버리라고 사주한다.</li>
<li><b><a href="/wiki/%EC%95%84%EC%84%B8%ED%8B%B8%EB%A0%8C_%EB%9E%A8%ED%94%84" title="아세틸렌 램프">아세틸렌 램프</a></b>: 데즈카 오사무의 간판 캐릭터. 게슈타포 요원으로 등장해 주인공들을 괴롭힌다.</li>
<li><b><a class="mw-redirect" href="/wiki/%ED%96%84_%EC%97%90%EA%B7%B8" title="햄 에그">햄 에그</a></b></li>
