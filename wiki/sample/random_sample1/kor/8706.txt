<p><b>탱고</b>(Tango)는 <a href="/wiki/%EC%95%84%EB%A5%B4%ED%97%A8%ED%8B%B0%EB%82%98" title="아르헨티나">아르헨티나</a>가 개최한 <a href="/wiki/1978%EB%85%84_FIFA_%EC%9B%94%EB%93%9C%EC%BB%B5" title="1978년 FIFA 월드컵">1978년 FIFA 월드컵</a>의 공인구로, <a href="/wiki/%EC%95%84%EB%94%94%EB%8B%A4%EC%8A%A4" title="아디다스">아디다스</a>가 제작하였다. 이 때부터 <a href="/wiki/FIFA_%EC%9B%94%EB%93%9C%EC%BB%B5_%EA%B3%B5%EC%9D%B8%EA%B5%AC" title="FIFA 월드컵 공인구">월드컵 공인구</a>는 <a href="/wiki/1998%EB%85%84_FIFA_%EC%9B%94%EB%93%9C%EC%BB%B5" title="1998년 FIFA 월드컵">1998년 FIFA 월드컵</a> 공인구인 <a class="mw-redirect" href="/wiki/%ED%8A%B8%EB%A6%AC%EC%BD%9C%EB%A1%9C" title="트리콜로">트리콜로</a>까지 탱고형 디자인을 유지했다.</p>
<p>대회마다 아래처럼 세부적으로 이름을 달리 명명하였다.</p>
<ul>
<li><a href="/wiki/1978%EB%85%84_FIFA_%EC%9B%94%EB%93%9C%EC%BB%B5" title="1978년 FIFA 월드컵">1978년 FIFA 월드컵</a>: 아디다스 탱고 더래스트</li>
<li><a href="/wiki/UEFA_%EC%9C%A0%EB%A1%9C_1980" title="UEFA 유로 1980">1980 유럽축구선수권대회</a>: 아디다스 탱고 리버플레이트</li>
<li><a href="/wiki/1982%EB%85%84_FIFA_%EC%9B%94%EB%93%9C%EC%BB%B5" title="1982년 FIFA 월드컵">1982년 FIFA 월드컵</a>:아디다스 탱고 에스파냐</li>
<li><a href="/wiki/1988%EB%85%84_%ED%95%98%EA%B3%84_%EC%98%AC%EB%A6%BC%ED%94%BD" title="1988년 하계 올림픽">1988년 하계 올림픽</a>: 아디다스 탱고 서울</li>
</ul>
<p>이 디자인은 <a class="mw-redirect" href="/wiki/%ED%85%94%EC%8A%A4%ED%83%80_(%EC%B6%95%EA%B5%AC%EA%B3%B5)" title="텔스타 (축구공)">텔스타</a>와 함께 대표적인 축구공 디자인이며, <a href="/wiki/2012%EB%85%84" title="2012년">2012년</a> <a href="/wiki/UEFA_%EC%9C%A0%EB%A1%9C_2012" title="UEFA 유로 2012">UEFA 유로 2012</a>의 공인구로 제작되어 새로운 버전인 <a href="/wiki/%EC%95%84%EB%94%94%EB%8B%A4%EC%8A%A4_%ED%83%B1%EA%B3%A0_12" title="아디다스 탱고 12">아디다스 탱고 12</a>가 탄생하였다.</p>
