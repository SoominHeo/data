<p><b>아라곤 연합왕국</b>(<span style="font-size: smaller;"><a href="/wiki/%EC%95%84%EB%9D%BC%EA%B3%A4%EC%96%B4" title="아라곤어">아라곤어</a>:</span> <span lang="an" xml:lang="an">Corona d'Aragón</span>, <span style="font-size: smaller;"><a href="/wiki/%EC%B9%B4%ED%83%88%EB%A3%A8%EB%83%90%EC%96%B4" title="카탈루냐어">카탈루냐어</a>:</span> <span lang="ca" xml:lang="ca">Corona d'Aragó</span>, <span style="font-size: smaller;"><a href="/wiki/%EB%9D%BC%ED%8B%B4%EC%96%B4" title="라틴어">라틴어</a>:</span> <span lang="la" xml:lang="la">Corona Aragonum</span>, <span style="font-size: smaller;"><a href="/wiki/%EC%8A%A4%ED%8E%98%EC%9D%B8%EC%96%B4" title="스페인어">스페인어</a>:</span> <span lang="es" xml:lang="es">Corona de Aragón</span>)은 <a href="/wiki/%EC%95%84%EB%9D%BC%EA%B3%A4%EC%9D%98_%EA%B5%B0%EC%A3%BC" title="아라곤의 군주">아라곤의 군주</a>가 지배하는 국가들의 <a class="mw-redirect" href="/wiki/%EB%8F%99%EA%B5%B0_%EC%97%B0%ED%95%A9" title="동군 연합">동군 연합</a>으로 이뤄진 왕국이다. 14세기와 15세기에 전성기를 맞은 아라곤 연합왕국은 오늘날의 <a href="/wiki/%EC%8A%A4%ED%8E%98%EC%9D%B8" title="스페인">스페인</a> 동부와 <a href="/wiki/%ED%94%84%EB%9E%91%EC%8A%A4" title="프랑스">프랑스</a> 남서부, 그리고 그리스까지 걸친 지중해의 주요 섬으로 해양 왕국이었다. 각 왕국들은 같은 왕이 지배한다는 것 외에는 정치적으로 통일되어 있지 않았다. 현대적 의미에서 아라곤 연합왕국은 단일한 국가라기보다는 <a href="/wiki/%EC%97%B0%EB%B0%A9%EC%A0%9C" title="연방제">연방제</a>의 형태를 갖고 있었으며, 이런 의미에서 아라곤 연합왕국은 그 일부인 <a href="/wiki/%EC%95%84%EB%9D%BC%EA%B3%A4_%EC%99%95%EA%B5%AD" title="아라곤 왕국">아라곤 왕국</a>과는 구별되어야 한다.</p>
<p>1469년 <a href="/wiki/%EA%B0%80%ED%86%A8%EB%A6%AD_%EA%B5%B0%EC%A3%BC" title="가톨릭 군주">가톨릭 군주</a>들의 결혼으로 아라곤 연합왕국과 <a href="/wiki/%EC%B9%B4%EC%8A%A4%ED%8B%B0%EC%95%BC_%EC%99%95%EA%B5%AD" title="카스티야 왕국">카스티야 왕국</a>이 결합하여 <a class="mw-redirect" href="/wiki/%EC%8A%A4%ED%8E%98%EC%9D%B8_%EC%99%95%EA%B5%AD" title="스페인 왕국">스페인 왕국</a>의 토대를 쌓았다. 이후 아라곤 연합왕국은 <a class="mw-redirect" href="/wiki/%ED%95%A9%EC%8A%A4%EB%B6%80%EB%A5%B4%ED%81%AC_%EC%99%95%EA%B0%80" title="합스부르크 왕가">합스부르크 왕가</a>가 세습하게 되어, 1716년 <a href="/wiki/%EC%8A%A4%ED%8E%98%EC%9D%B8_%EC%99%95%EC%9C%84_%EA%B3%84%EC%8A%B9_%EC%A0%84%EC%9F%81" title="스페인 왕위 계승 전쟁">스페인 왕위 계승 전쟁</a>의 결과로 아라곤 연합왕국이 해체될 때까지 이어졌다.</p>

<h2><span class="mw-headline" id=".EB.B0.B0.EA.B2.BD">배경</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%95%84%EB%9D%BC%EA%B3%A4_%EC%97%B0%ED%95%A9%EC%99%95%EA%B5%AD&amp;action=edit&amp;section=1" title="부분 편집: 배경">원본 편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>아라곤 연합 왕국의 정치적 수도는 <a href="/wiki/%EC%82%AC%EB%9D%BC%EA%B3%A0%EC%82%AC" title="사라고사">사라고사</a>로, 아라곤의 왕은 이 도시의 라 세오 대성당에서 대관식을 치렀다. 아라곤의 경제 중심지는 <a href="/wiki/%EB%B0%94%EB%A5%B4%EC%85%80%EB%A1%9C%EB%82%98" title="바르셀로나">바르셀로나</a>와 <a href="/wiki/%EB%B0%9C%EB%A0%8C%EC%8B%9C%EC%95%84" title="발렌시아">발렌시아</a>였다. <a href="/wiki/%ED%8C%94%EB%A7%88%EB%8D%B0%EB%A7%88%EC%9A%94%EB%A5%B4%EC%B9%B4" title="팔마데마요르카">팔마데마요르카</a> 또한 중요한 항구도시였다.</p>
<p>아라곤 연합 왕국은 최종적으로 <a href="/wiki/%EC%95%84%EB%9D%BC%EA%B3%A4_%EC%99%95%EA%B5%AD" title="아라곤 왕국">아라곤 왕국</a>, 바르셀로나 백작령, 발렌시아 왕국, 마요르카 왕국, <a href="/wiki/%EC%8B%9C%EC%B9%A0%EB%A6%AC%EC%95%84_%EC%99%95%EA%B5%AD" title="시칠리아 왕국">시칠리아 왕국</a>, <a href="/wiki/%EB%AA%B0%ED%83%80" title="몰타">몰타</a>, <a href="/wiki/%EB%82%98%ED%8F%B4%EB%A6%AC_%EC%99%95%EA%B5%AD" title="나폴리 왕국">나폴리 왕국</a>, <a href="/wiki/%EC%82%AC%EB%A5%B4%EB%8D%B0%EB%83%90_%EC%99%95%EA%B5%AD" title="사르데냐 왕국">사르데냐 왕국</a>을 포함하게 되었다. 또한 <a href="/wiki/%EB%AA%BD%ED%8E%A0%EB%A6%AC%EC%97%90" title="몽펠리에">몽펠리에</a>, <a href="/wiki/%ED%94%84%EB%A1%9C%EB%B0%A9%EC%8A%A4" title="프로방스">프로방스</a>, <a href="/wiki/%EC%BD%94%EB%A5%B4%EC%8B%9C%EC%B9%B4" title="코르시카">코르시카</a>, 네오파트리아 공국, 아테네 공국을 잠시동안 지배하였다.</p>
<p>오늘날의 <a href="/wiki/%EC%8A%A4%ED%8E%98%EC%9D%B8" title="스페인">스페인</a>과 <a href="/wiki/%ED%8F%AC%EB%A5%B4%ED%88%AC%EA%B0%88" title="포르투갈">포르투갈</a> 지역은 722년부터 "<a class="mw-redirect" href="/wiki/%EB%A0%88%EC%BD%98%ED%82%A4%EC%8A%A4%ED%83%80" title="레콘키스타">레콘키스타</a>"라 불리는 주기적인 전투와 함께 <a href="/wiki/%EC%A4%91%EC%84%B8" title="중세">중세</a>를 보냈다. 북부의 기독교 왕국들은 <a href="/wiki/%EC%95%8C%EC%95%88%EB%8B%AC%EB%A3%A8%EC%8A%A4" title="알안달루스">알안달루스</a>의 <a class="mw-redirect" href="/wiki/%EC%9D%B4%EC%8A%AC%EB%9E%8C" title="이슬람">이슬람</a> 소왕국들과 싸우고, 때로는 서로 싸우기도 했다.</p>
<p>중세 후기에 들어서 남쪽으로 확장을 계속한 아라곤 연합 왕국과 <a href="/wiki/%EB%AC%B4%EB%A5%B4%EC%8B%9C%EC%95%84" title="무르시아">무르시아</a> 동쪽으로 확장한 카스티야 연합 왕국이 서로 만나게 되었다. 이후 아라곤 연합 왕국은 그리스에서 <a class="mw-redirect" href="/wiki/%EB%B0%94%EB%A5%B4%EB%B0%94%EB%A6%AC_%ED%95%B4%EC%95%88" title="바르바리 해안">바르바리 해안</a>에 걸친 지중해 방향으로 힘을 기울였다. 아라곤의 지중해 제국 건설 과정에서, 각 연합 왕국에서 온 "알모가바르"라는 용병들이 참여하였다.</p>
<p>어떤 이들은 아라곤 연합 왕국을 수백년동안 지중해를 지배한 해양 제국으로 간주한다. 예를 들어 <a href="/wiki/%EC%B9%B4%ED%83%88%EB%A3%A8%EB%83%90%EC%96%B4" title="카탈루냐어">카탈루냐어</a>로 쓰인 《바다의 영사》는 세계 최초로 편찬된 해양법 법전이다. 아라곤의 전성기에는 실제로 유럽의 주요 강대국 가운데 하나였다.</p>
<p>그러나 서로 다른 왕국이 국왕을 통해서만 정치적 연합을 이루었으므로, 동시대인 Marqués de Lozoya는 아라곤 연합 왕국이 연방제 국가일 뿐 중앙집권화된 왕국이 아니며, 제국은 더더욱 아니라고 서술했다.<sup class="reference" id="cite_ref-1"><a href="#cite_note-1">[1]</a></sup> 아라곤을 제국으로 언급하는 당시의 공식 문서도 존재하지 않으며, 자치 국가들의 연합으로 간주되었다.</p>
<h2><span class="mw-headline" id=".EC.97.AD.EC.82.AC">역사</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%95%84%EB%9D%BC%EA%B3%A4_%EC%97%B0%ED%95%A9%EC%99%95%EA%B5%AD&amp;action=edit&amp;section=2" title="부분 편집: 역사">원본 편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<h3><span class="mw-headline" id=".EA.B8.B0.EC.9B.90">기원</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%95%84%EB%9D%BC%EA%B3%A4_%EC%97%B0%ED%95%A9%EC%99%95%EA%B5%AD&amp;action=edit&amp;section=3" title="부분 편집: 기원">원본 편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<div class="thumb tright">
<div class="thumbinner" style="width:222px;"><a class="image" href="/wiki/%ED%8C%8C%EC%9D%BC:Petronila_Ramon_Berenguer.jpg"><img alt="" class="thumbimage" data-file-height="856" data-file-width="772" height="244" src="//upload.wikimedia.org/wikipedia/commons/thumb/5/56/Petronila_Ramon_Berenguer.jpg/220px-Petronila_Ramon_Berenguer.jpg" srcset="//upload.wikimedia.org/wikipedia/commons/thumb/5/56/Petronila_Ramon_Berenguer.jpg/330px-Petronila_Ramon_Berenguer.jpg 1.5x, //upload.wikimedia.org/wikipedia/commons/thumb/5/56/Petronila_Ramon_Berenguer.jpg/440px-Petronila_Ramon_Berenguer.jpg 2x" width="220"/></a>

16세기 회화에 묘사된 아라곤 여왕 페트로넬라와 바르셀로나 후작 라몬 베렝게르 4세</div>

<p>아라곤 연합 왕국은 1137년 아라곤 여왕 페트로넬라와 바르셀로나 백작 라몬 베렝게르 4세의 결혼으로 <a href="/wiki/%EC%95%84%EB%9D%BC%EA%B3%A4_%EC%99%95%EA%B5%AD" title="아라곤 왕국">아라곤 왕국</a>과 바르셀로나 백작령(프로방스 후작령 포함)이 연합하면서 시작되었다.<sup class="reference" id="cite_ref-bisson_31_2-0"><a href="#cite_note-bisson_31-2">[2]</a></sup><sup class="reference" id="cite_ref-3"><a href="#cite_note-3">[3]</a></sup> 두 사람 사이에 낳은 아들 <a class="new" href="/w/index.php?title=%EC%95%8C%ED%8F%B0%EC%86%8C_2%EC%84%B8_(%EC%95%84%EB%9D%BC%EA%B3%A4)&amp;action=edit&amp;redlink=1" title="알폰소 2세 (아라곤) (없는 문서)">알폰소 2세</a>가 1162년 왕위에 오르면서 두 왕국의 군주가 되었다. 연합 후에도 양국의 통치기구와 의회는 그대로 유지되었다. 바르셀로나 백작령이 더 부유했으나, 아라곤 왕가가 "에스파냐의 황제"라 불린 나바라의 <a class="new" href="/w/index.php?title=%EC%82%B0%EC%B4%88_3%EC%84%B8_(%EB%82%98%EB%B0%94%EB%9D%BC)&amp;action=edit&amp;redlink=1" title="산초 3세 (나바라) (없는 문서)">산초 3세</a>의 후손으로 더 높은 작위를 가졌으므로 연합 왕국의 이름은 아라곤이 되었다.<sup class="reference" id="cite_ref-4"><a href="#cite_note-4">[4]</a></sup> 또한 군사적 재능이 형편없어 "수도승"이란 별명으로 불린 페트로넬라 여왕의 아버지 라미로 2세가 딸을 바르셀로나 백작 라몬 베렝게르 4세와 혼인시키기로 결정하면서, 자신을 "나의 왕이자 주군이시며 아버지"라고 부르도록 요구하였다. 따라서 라몬 백작은 형식적으로 아라곤 왕가의 일원이 되어 아라곤 공작의 작위를 받았다.</p>
<h3><span class="mw-headline" id=".EC.98.81.ED.86.A0.ED.99.95.EC.9E.A5">영토확장</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%95%84%EB%9D%BC%EA%B3%A4_%EC%97%B0%ED%95%A9%EC%99%95%EA%B5%AD&amp;action=edit&amp;section=4" title="부분 편집: 영토확장">원본 편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>알폰소 2세는 발렌시아를 정복할 기회를 노렸으나, 나바라의 <a class="new" href="/w/index.php?title=%EC%82%B0%EC%B4%88_6%EC%84%B8_(%EB%82%98%EB%B0%94%EB%9D%BC)&amp;action=edit&amp;redlink=1" title="산초 6세 (나바라) (없는 문서)">산초 6세</a>의 침공으로 야심을 버려야 했다. 알폰소 2세는 전선의 안정을 꾀하기 위해 카스티야의 <a class="new" href="/w/index.php?title=%EC%95%8C%ED%8F%B0%EC%86%8C_8%EC%84%B8_(%EC%B9%B4%EC%8A%A4%ED%8B%B0%EC%95%BC)&amp;action=edit&amp;redlink=1" title="알폰소 8세 (카스티야) (없는 문서)">알폰소 8세</a>와 카졸라 협정을 맺어, 각자 점령할 수 있는 이슬람 영토의 한계를 정하고, 카스티야가 무르시아를 점령하는 대신 발렌시아는 아라곤 왕국이 갖기로 협정을 맺었다.</p>
<p>9세기 이후 아키텐 공작, 푸아 후작, 툴루즈 후작과 <a href="/wiki/%EC%95%84%EB%9D%BC%EA%B3%A4%EC%9D%98_%EA%B5%B0%EC%A3%BC" title="아라곤의 군주">아라곤의 군주</a>들은 <a class="mw-redirect" href="/wiki/%EC%98%A4%EC%8B%9C%ED%83%80%EB%8B%88%EC%95%84" title="오시타니아">오시타니아</a>의 여러 지역들을 지배하기 위해 경쟁하였다. <a href="/wiki/%EC%B9%B4%ED%83%80%EB%A6%AC%ED%8C%8C" title="카타리파">카타리파</a> 또는 알비파라 불린 기독교 교파가 로마 가톨릭에 대항해 반란을 일으켜, 아라곤은 프랑스 남부에서의 영향력을 잃었다. <a href="/wiki/%EA%B5%90%ED%99%A9_%EC%9D%B8%EB%85%B8%EC%B2%B8%EC%8B%9C%EC%98%A4_3%EC%84%B8" title="교황 인노첸시오 3세">교황 인노첸시오 3세</a>의 요청을 받아 <a href="/wiki/%EC%95%8C%EB%B9%84_%EC%8B%AD%EC%9E%90%EA%B5%B0" title="알비 십자군">알비 십자군</a>을 일으킨 프랑스의 <a href="/wiki/%ED%95%84%EB%A6%AC%ED%94%84_2%EC%84%B8_(%ED%94%84%EB%9E%91%EC%8A%A4)" title="필리프 2세 (프랑스)">필리프 2세</a>는 오시타니아 지방을 완전히 점령하여, 프랑스 북부의 <a href="/wiki/%EC%B9%B4%ED%8E%98_%EC%99%95%EC%A1%B0" title="카페 왕조">카페 왕조</a>가 이 지역을 지배하게 되었다.</p>
<p>아라곤의 <a class="new" href="/w/index.php?title=%ED%8E%98%EB%93%9C%EB%A1%9C_2%EC%84%B8_(%EC%95%84%EB%9D%BC%EA%B3%A4)&amp;action=edit&amp;redlink=1" title="페드로 2세 (아라곤) (없는 문서)">페드로 2세</a>가 1212년 가을 라스 나바스 데 톨로사 전투에서 이슬람군을 물리치고 돌아왔을 때, 레스터 백작 시몽 드 몽포르는 <a href="/wiki/%ED%88%B4%EB%A3%A8%EC%A6%88" title="툴루즈">툴루즈</a>를 점령하고 페드로 2세의 매형이자 신하인 툴루즈 후작 라몬 6세를 추방했다. 군대를 이끌고 <a href="/wiki/%ED%94%BC%EB%A0%88%EB%84%A4_%EC%82%B0%EB%A7%A5" title="피레네 산맥">피레네 산맥</a>을 넘어 진군한 페드로는 뮤헤에서 라몬 6세의 군대와 합류하여 시몽 드 몽포르와 맞섰다.</p>
<p>1213년 9월 12일 벌어진 뮤헤 전투에서 페드로는 만용을 부리다 전사하고, 아라곤 군대는 대패하였다. 이 전투의 결과로 아라곤 연합 왕국은 프랑스 왕국의 오시타니아 지배를 인정해야 했다.</p>
<p>13세기 아라곤의 <a href="/wiki/%ED%95%98%EC%9D%B4%EB%A9%94_1%EC%84%B8" title="하이메 1세">하이메 1세</a>는 영토 확장 전쟁을 개시하여 <a href="/wiki/%EB%A7%88%EC%9A%94%EB%A5%B4%EC%B9%B4_%EC%84%AC" title="마요르카 섬">마요르카 섬</a>을 합병하고 발렌시아 왕국을 절반 이상 차지하였다. 마요르카의 하우메 2세가 지배하고 있던 세르다냐, 루시용, <a href="/wiki/%EB%AA%BD%ED%8E%A0%EB%A6%AC%EC%97%90" title="몽펠리에">몽펠리에</a>는 이때부터 사실상 아라곤에 합병되었고, 발렌시아는 기존의 통치기구를 그대로 유지한 채 연합 왕국의 세 번째 구성원이 되었다. 1258년 아라곤과 프랑스는 협정을 맺어, 아라곤은 피레네 이북에서의 영향력을 포기하고 프랑스는 카탈루냐에 대한 영유권을 포기하기로 하였다.<sup class="reference" id="cite_ref-chaytor_james_5-0"><a href="#cite_note-chaytor_james-5">[5]</a></sup> 1266년 1월 하이메 1세는 무르시아를 점령하고 자신의 카탈루냐 백성들을 이주시킨 후, 카졸라 협정에 따라 카스티야에 넘겼다.<sup class="reference" id="cite_ref-bisson_67_6-0"><a href="#cite_note-bisson_67-6">[6]</a></sup></p>
<p>1282년 <a href="/wiki/%EC%8B%9C%EC%B9%A0%EB%A6%AC%EC%95%84" title="시칠리아">시칠리아</a>인들이 앙주 왕가에 <a href="/wiki/%EC%8B%9C%EC%B9%A0%EB%A6%AC%EC%95%84_%EB%A7%8C%EC%A2%85_%EC%82%AC%EA%B1%B4" title="시칠리아 만종 사건">반란</a>을 일으켜 주둔군을 학살하는 사건이 일어났다. 아라곤의 <a href="/wiki/%ED%8E%98%EB%93%9C%EB%A1%9C_3%EC%84%B8_(%EC%95%84%EB%9D%BC%EA%B3%A4)" title="페드로 3세 (아라곤)">페드로 3세</a>는 시칠리아인들의 요청을 받고 성대한 환영을 받으며 시칠리아에 상륙했다. 이 사건에 분노한 <a href="/wiki/%EA%B5%90%ED%99%A9_%EB%A7%88%EB%A5%B4%ED%8B%B0%EB%85%B8_4%EC%84%B8" title="교황 마르티노 4세">교황 마르티노 4세</a>는 시칠리아에 성무금지 조치를 내리고, 페드로 3세를 파문했다.</p>
<p>아라곤의 <a class="new" href="/w/index.php?title=%ED%95%98%EC%9D%B4%EB%A9%94_2%EC%84%B8_(%EC%95%84%EB%9D%BC%EA%B3%A4)&amp;action=edit&amp;redlink=1" title="하이메 2세 (아라곤) (없는 문서)">하이메 2세</a>는 발렌시아를 완전히 점령하여, 아라곤을 유럽의 강대국으로 끌어올렸다.</p>
<p>1297년 <a href="/wiki/%EA%B5%90%ED%99%A9_%EB%B3%B4%EB%8B%88%ED%8C%8C%EC%8B%9C%EC%98%A4_8%EC%84%B8" title="교황 보니파시오 8세">교황 보니파시오 8세</a>는 사르데냐-코르시카 왕국을 하이메 2세에게 주어 연합 왕국에 추가시켰으나, 이 연합은 1세기도 유지되지 못했다. 1381년 아라곤의 <a class="new" href="/w/index.php?title=%ED%8E%98%EB%93%9C%EB%A1%9C_4%EC%84%B8_(%EC%95%84%EB%9D%BC%EA%B3%A4)&amp;action=edit&amp;redlink=1" title="페드로 4세 (아라곤) (없는 문서)">페드로 4세</a>와 시칠리아의 마리아 여왕의 결혼으로 <a href="/wiki/%EC%8B%9C%EC%B9%A0%EB%A6%AC%EC%95%84_%EC%99%95%EA%B5%AD" title="시칠리아 왕국">시칠리아 왕국</a>, 아테네 공국, 네오파트리아 공국이 연합에 추가되었다. 그러나 1388년 네리오 1세 아치아이올리에게 그리스 영토를 잃었다. 1422년 <a class="new" href="/w/index.php?title=%EC%95%8C%ED%8F%B0%EC%86%8C_5%EC%84%B8_(%EC%95%84%EB%9D%BC%EA%B3%A4)&amp;action=edit&amp;redlink=1" title="알폰소 5세 (아라곤) (없는 문서)">알폰소 5세</a>는 <a href="/wiki/%EB%82%98%ED%8F%B4%EB%A6%AC_%EC%99%95%EA%B5%AD" title="나폴리 왕국">나폴리 왕국</a>을 점령했다.</p>
<p>중세 사회에서 근대 초기의 절대 왕정으로 전이하는 다른 왕국과 마찬가지로, 아라곤 국왕은 귀족들과 끊임없이 세력다툼을 하였다. 아라곤 국왕은 <a href="/wiki/%EB%AC%B4%EC%96%B4%EC%9D%B8" title="무어인">무어인</a>들에게서 빼앗은 발렌시아, 마요르카 등의 도시를 귀족들에게 나눠주는 대신, 일종의 자치 기구인 "푸에로"(<span style="font-size: smaller;"><a href="/wiki/%EC%8A%A4%ED%8E%98%EC%9D%B8%EC%96%B4" title="스페인어">스페인어</a>:</span> <span lang="es" xml:lang="es">fuero</span>, <span style="font-size: smaller;"><a href="/wiki/%EC%B9%B4%ED%83%88%EB%A3%A8%EB%83%90%EC%96%B4" title="카탈루냐어">카탈루냐어</a>:</span> <span lang="ca" xml:lang="ca">furs</span>)를 설치하여 귀족들의 권력을 제한했다. 이웃의 <a href="/wiki/%EC%B9%B4%EC%8A%A4%ED%8B%B0%EC%95%BC_%EC%99%95%EA%B5%AD" title="카스티야 왕국">카스티야 왕국</a> 또한 마찬가지로 <a class="mw-redirect" href="/wiki/%EB%A0%88%EC%BD%98%ED%82%A4%EC%8A%A4%ED%83%80" title="레콘키스타">레콘키스타</a>에서 획득한 도시를 봉건 귀족의 영지로 주는 대신 자치권을 부여했다.</p>

<p>1410년 <a class="new" href="/w/index.php?title=%EC%95%84%EB%9D%BC%EA%B3%A4%EC%9D%98_%EB%A7%88%EB%A5%B4%ED%8B%B0%EB%85%B8_1%EC%84%B8&amp;action=edit&amp;redlink=1" title="아라곤의 마르티노 1세 (없는 문서)">아라곤의 마르티노 1세</a>가 후사를 남기지 못하고 죽었다. 따라서 아라곤, 발렌시아, 바르셀로나의 대표가 모여 난투극으로 이어진 격렬한 논쟁 끝에 <a class="new" href="/w/index.php?title=%ED%8E%98%EB%93%9C%EB%A1%9C_4%EC%84%B8_(%EC%95%84%EB%9D%BC%EA%B3%A4)&amp;action=edit&amp;redlink=1" title="페드로 4세 (아라곤) (없는 문서)">페드로 4세</a>의 외손자이자 카스티야의 왕족인 <a class="new" href="/w/index.php?title=%ED%8E%98%EB%A5%B4%EB%82%9C%EB%8F%84_1%EC%84%B8_(%EC%95%84%EB%9D%BC%EA%B3%A4)&amp;action=edit&amp;redlink=1" title="페르난도 1세 (아라곤) (없는 문서)">페르난도 엘 데 안테케라</a>가 추대되어 페르난도 1세로 왕위에 올랐다.</p>
<p>페르난도 1세의 손자 <a href="/wiki/%ED%8E%98%EB%A5%B4%EB%82%9C%EB%8F%84_2%EC%84%B8_(%EC%95%84%EB%9D%BC%EA%B3%A4)" title="페르난도 2세 (아라곤)">페르난도 2세</a>는 프랑스와 나바라에 빼앗긴 카탈란 북부의 루시용과 세르다냐 지방을 다시 빼앗아 왔다.</p>
<p>1469년 페르난도 2세는 카스티야 <a class="new" href="/w/index.php?title=%EC%97%94%EB%A6%AC%EC%BC%80_4%EC%84%B8_(%EC%B9%B4%EC%8A%A4%ED%8B%B0%EC%95%BC)&amp;action=edit&amp;redlink=1" title="엔리케 4세 (카스티야) (없는 문서)">엔리케 4세</a>의 이복누이이자 나중에 카스티야와 레온 왕국의 여왕이 된 <a href="/wiki/%EC%9D%B4%EC%82%AC%EB%B2%A8_1%EC%84%B8" title="이사벨 1세">이사벨</a> 왕녀와 결혼식을 올렸다. 그들의 결혼으로 이루어진 두 왕국의 연합은 장차 <a class="mw-redirect" href="/wiki/%EC%8A%A4%ED%8E%98%EC%9D%B8_%EC%99%95%EA%B5%AD" title="스페인 왕국">스페인 왕국</a>의 기반이 되었다.<sup class="reference" id="cite_ref-7"><a href="#cite_note-7">[7]</a></sup><sup class="reference" id="cite_ref-8"><a href="#cite_note-8">[8]</a></sup><sup class="reference" id="cite_ref-9"><a href="#cite_note-9">[9]</a></sup> 이 시점에서 아직 카스티야와 아라곤 연합 왕국은 각각 기존의 통치기구를 유지한 채 독립적인 왕국으로 남았다. 신성 로마 제국의 <a href="/wiki/%EC%B9%B4%EB%A5%BC_5%EC%84%B8" title="카를 5세">카를 5세</a>이기도 했던 카를로스 1세는 1516년 포르투갈과 <a href="/wiki/%EC%95%8C%EA%B0%80%EB%A5%B4%EB%B8%8C_%EC%A7%80%EB%B0%A9" title="알가르브 지방">알가르브</a>를 제외한 <a href="/wiki/%EC%9D%B4%EB%B2%A0%EB%A6%AC%EC%95%84_%EB%B0%98%EB%8F%84" title="이베리아 반도">이베리아 반도</a>의 왕국들을 통합하여 단일한 국가로서의 스페인을 완성하였다.</p>
<h3><span class="mw-headline" id=".ED.95.B4.EC.B2.B4">해체</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%95%84%EB%9D%BC%EA%B3%A4_%EC%97%B0%ED%95%A9%EC%99%95%EA%B5%AD&amp;action=edit&amp;section=6" title="부분 편집: 해체">원본 편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>발렌시아, 마요르카, 시칠리아의 점령이 가져온 급격한 인구증가는 아라곤 연합 왕국의 전성기를 가져온 한편, 사회적 갈등도 불러왔다.<sup class="reference" id="cite_ref-bisson_188_189_10-0"><a href="#cite_note-bisson_188_189-10">[10]</a></sup> 이후 아라곤의 경제력과 군사력은 성장을 멈추고, 아라곤 왕국의 권력은 차츰 새로운 스페인 연합 왕국으로 이동했다. 아라곤은 1659년 루시용을 잃고, 1707년-1716년 미노르카 및 이탈리아 영토들을 잃었으며, 스페인 궁정의 정책으로 정부의 공용어 또한 <a class="mw-redirect" href="/wiki/%EC%B9%B4%EC%8A%A4%ED%8B%B0%EC%95%BC%EC%96%B4" title="카스티야어">카스티야어</a>로 변경되었다.</p>
<p><a href="/wiki/%EC%8A%A4%ED%8E%98%EC%9D%B8_%EC%99%95%EC%9C%84_%EA%B3%84%EC%8A%B9_%EC%A0%84%EC%9F%81" title="스페인 왕위 계승 전쟁">스페인 왕위 계승 전쟁</a> 직후인 1716년, 스페인의 <a href="/wiki/%ED%8E%A0%EB%A6%AC%ED%8E%98_5%EC%84%B8" title="펠리페 5세">펠리페 5세</a>는 아라곤 연합 왕국과 그 통치기구들을 폐지하고 모든 영토를 카스티야 왕국에 흡수하여, <a class="mw-redirect" href="/wiki/%EB%B6%80%EB%A5%B4%EB%B4%89_%EC%99%95%EA%B0%80" title="부르봉 왕가">부르봉 왕가</a>가 지배하는 단일한 전제주의 왕국으로서의 스페인 통합을 완료했다.<sup class="reference" id="cite_ref-bisson_189_11-0"><a href="#cite_note-bisson_189-11">[11]</a></sup></p>
<h2><span class="mw-headline" id=".EC.88.98.EB.8F.84">수도</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%95%84%EB%9D%BC%EA%B3%A4_%EC%97%B0%ED%95%A9%EC%99%95%EA%B5%AD&amp;action=edit&amp;section=7" title="부분 편집: 수도">원본 편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>알폰소 5세와 페르난도 2세가 <a href="/wiki/%EB%82%98%ED%8F%B4%EB%A6%AC" title="나폴리">나폴리</a>에 수도를 정한 뒤, 15세기와 16세기에 걸쳐 아라곤 연합 왕국의 실질적 수도는 나폴리였다. 특히 알폰소 5세는 나폴리를 수도로 삼기 위해 큰 돈을 들여 나폴리의 궁정을 아름답게 꾸몄다.<sup class="reference" id="cite_ref-12"><a href="#cite_note-12">[12]</a></sup> 이후 펠리페 2세 시대까지 아라곤 궁정은 고정된 도시에 있지 않고 이동하였다.<sup class="reference" id="cite_ref-uib_13-0"><a href="#cite_note-uib-13">[13]</a></sup> 아라곤 왕들은 사라고사의 라 세오 대성당에서 대관식을 올렸으므로 사라고사를 상징적 수도로 볼 수는 있으나, 경제적 행정적 수도는 아니었다.</p>
