<p><b>아라곤의 캐서린</b>(<span lang="en" xml:lang="en"><i>Catherine of Aragon</i></span>/<span style="font-size: smaller;"><a href="/wiki/%EC%8A%A4%ED%8E%98%EC%9D%B8%EC%96%B4" title="스페인어">스페인어</a>:</span> <span lang="es" xml:lang="es"><i>Catalina de Aragón</i></span>, <a href="/wiki/1485%EB%85%84" title="1485년">1485년</a> <a href="/wiki/12%EC%9B%94_16%EC%9D%BC" title="12월 16일">12월 16일</a> ~ <a href="/wiki/1536%EB%85%84" title="1536년">1536년</a> <a href="/wiki/1%EC%9B%94_7%EC%9D%BC" title="1월 7일">1월 7일</a>)은 <a class="mw-redirect" href="/wiki/%EC%95%84%EB%9D%BC%EA%B3%A4%EC%9D%98_%ED%8E%98%EB%A5%B4%EB%82%9C%EB%8F%84_2%EC%84%B8" title="아라곤의 페르난도 2세">아라곤의 페르난도 2세</a>와 <a class="mw-redirect" href="/wiki/%EC%B9%B4%EC%8A%A4%ED%8B%B0%EC%95%BC%EC%9D%98_%EC%9D%B4%EC%82%AC%EB%B2%A8_1%EC%84%B8" title="카스티야의 이사벨 1세">카스티야의 이사벨 1세</a>의 넷째딸로 <a class="mw-redirect" href="/wiki/%EC%9E%89%EA%B8%80%EB%9E%9C%EB%93%9C%EC%9D%98_%ED%97%A8%EB%A6%AC_8%EC%84%B8" title="잉글랜드의 헨리 8세">잉글랜드의 헨리 8세</a>의 정비이자 <a class="mw-redirect" href="/wiki/%EC%9E%89%EA%B8%80%EB%9E%9C%EB%93%9C%EC%9D%98_%EB%A9%94%EB%A6%AC_1%EC%84%B8" title="잉글랜드의 메리 1세">잉글랜드의 메리 1세</a>의 어머니이다.</p>
<h2><span class="mw-headline" id=".EC.83.9D.EC.95.A0">생애</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%95%84%EB%9D%BC%EA%B3%A4%EC%9D%98_%EC%BA%90%EC%84%9C%EB%A6%B0&amp;action=edit&amp;section=1" title="부분 편집: 생애">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>캐서린은 여왕인 어머니 <a class="mw-redirect" href="/wiki/%EC%B9%B4%EC%8A%A4%ED%8B%B0%EC%95%BC%EC%9D%98_%EC%9D%B4%EC%82%AC%EB%B2%A8_1%EC%84%B8" title="카스티야의 이사벨 1세">카스티야의 이사벨 1세</a>의 배려로 어렸을 때부터 남자 왕족에 뒤지지 않는 교육을 받아 역사, 법, 정치, 고전 이외에도 라틴어, 프랑스어, 그리스어 등을 익혔다. 당대 기록에 따르면 캐서린은 키는 작았지만 붉은 빛이 도는 금발에 희고 홍조가 도는 피부, 그리고 통통한 체형의 복스러운 미인으로, 신앙심 깊고 겸손하며 기품 있는 여인으로 여겨졌다.</p>
<p><a href="/wiki/1501%EB%85%84" title="1501년">1501년</a> 캐서린은 잉글랜드의 <a href="/wiki/%ED%97%A8%EB%A6%AC_7%EC%84%B8" title="헨리 7세">헨리 7세</a>의 장남 <a href="/wiki/%EC%9B%A8%EC%9D%BC%EC%8A%A4_%EA%B3%B5" title="웨일스 공">웨일스 공</a> <a href="/wiki/%EC%95%84%EC%84%9C_%ED%8A%9C%EB%8D%94" title="아서 튜더">아서</a>와 정략결혼을 했다. 그러나 얼마 지나지 않아 둘은 혼례도 제대로 치르지 못한 채 심한 병에 걸렸다. 캐서린은 가까스로 회복했으나 아서는 살아남지 못했다. <a href="/wiki/%EA%B5%90%ED%99%A9_%EC%9C%A8%EB%A6%AC%EC%98%A4_2%EC%84%B8" title="교황 율리오 2세">교황 율리오 2세</a>는 캐서린이 아직 처녀임을 인정해 이전의 혼인을 무효화시켰고, 헨리 7세의 둘째 아들인 <a href="/wiki/%ED%97%A8%EB%A6%AC_8%EC%84%B8" title="헨리 8세">헨리</a>와 약혼했다.</p>
<p>그러나 헨리 7세와 <a class="mw-redirect" href="/wiki/%EC%95%84%EB%9D%BC%EA%B3%A4%EC%9D%98_%ED%8E%98%EB%A5%B4%EB%82%9C%EB%8F%84_2%EC%84%B8" title="아라곤의 페르난도 2세">페르난도 2세</a> 사이 캐서린의 지참금에 관련된 분쟁이 일어난데다 왕비를 잃은 헨리 7세가 정치적 욕심으로 혼인을 차일피일 미루기 시작했다. 입지가 곤란해진 캐서린은 낯선 나라에서 지독한 가난과 외로움에 시달렸다. 캐서린은 음식을 마련하기도 힘들다며 몇번이고 시아버지 헨리 7세와 페르난도 2세에게 편지를 보냈지만 캐서린의 형편은 나아지지 않았다. 거의 8년의 공백기가 흐른 후 헨리 7세가 갑작스러운 병으로 세상을 뜨자, 그의 둘째 아들이자 왕위 계승자가 된 당시 20세의 헨리 8세는 캐서린과 마침내 결혼식을 올렸다.</p>
<p>헨리 8세와 캐서린의 금슬은 좋았고, 두 사람은 깊이 사랑했다. 그러나 캐서린이 <a href="/wiki/1516%EB%85%84" title="1516년">1516년</a> 메리 공주를 낳고 나서 유산을 거듭하다 <a href="/wiki/%ED%8F%90%EA%B2%BD%EA%B8%B0" title="폐경기">폐경기</a>를 맞자 <a href="/wiki/%ED%97%A8%EB%A6%AC_8%EC%84%B8" title="헨리 8세">헨리 8세</a>는 왕위 후계자, 즉 아들이 없음을 걱정하면서 점점 부부 사이가 틀어지기 시작했다. 헨리 8세는 이미 정부인 베시 블라운트(Bessie Blount)와의 사이에 아들 헨리 피츠로이를 두어 자신에게는 문제가 없음을 증명했고, 불임은 무조건 여성의 책임이라고 여겼던 당시 관습에 따라 그 책임을 캐서린에게 떠넘기기 시작했다. 그 배경에는 아직 역사가 짧은 튜더 왕가를 이어가야 한다는 중압감도 있었다.</p>

<p>캐서린의 시녀 <a href="/wiki/%EC%95%A4_%EB%B6%88%EB%A6%B0" title="앤 불린">앤 불린</a>과 사랑에 빠진 헨리 8세는 결국 <a href="/wiki/1527%EB%85%84" title="1527년">1527년</a> <a href="/wiki/%EA%B5%90%ED%99%A9_%ED%81%B4%EB%A0%88%EB%A9%98%EC%8A%A4_7%EC%84%B8" title="교황 클레멘스 7세">교황 클레멘스 7세</a>에게 결혼을 무효화해달라고 요청했다. 캐서린이 자신의 형과 혼인한 적이 있었기 때문에 처음부터 <a href="/wiki/%EC%84%B1%EA%B2%BD" title="성경">성경</a>의 가르침에 어긋난 결혼이었다는 것이 표면상의 이유였다. 헨리 8세는 또한 전에 선언했던 것과는 달리 말을 바꾸어서 캐서린이 자신과 결혼할 때 이미 처녀가 아니었다고 주장했다. 헨리 8세는 캐서린에게도 결혼이 무효임을 인정하고 수도원에 들어가 조용한 삶을 보낼 것을 권유했지만, 캐서린은 의외로 단호하게 거절했다. 두 왕국을 다스리는 왕과 여왕의 딸이었고 어렸을 때 잉글랜드의 왕자비로 내정되었던 캐서린에게는 왕좌 위가 아닌 삶은 상상도 할 수 없는 것이었다.</p>
<p>캐서린의 조카인 <a class="mw-redirect" href="/wiki/%EC%8B%A0%EC%84%B1_%EB%A1%9C%EB%A7%88_%EC%A0%9C%EA%B5%AD%EC%9D%98_%EC%B9%B4%EB%A5%BC_5%EC%84%B8" title="신성 로마 제국의 카를 5세">신성 로마 제국의 카를 5세</a>의 심기를 거스를 수 없던 교황은 결혼 무효화를 거절했다. 헨리 8세는 교황과의 오랜 첨예한 대립 끝에 <a class="mw-disambig" href="/wiki/%EC%98%81%EA%B5%AD_%EC%84%B1%EA%B3%B5%ED%9A%8C" title="영국 성공회">성공회</a>를 건립하고 스스로 성공회의 우두머리가 되었다.<a href="/wiki/1531%EB%85%84" title="1531년">1531년</a> 헨리 8세는 캐서린과 별거에 들어갔다. <a href="/wiki/1533%EB%85%84" title="1533년">1533년</a> 헨리 8세와 앤 불린의 결혼식이 거행되었다. 캐서린은 딸 <a class="mw-redirect" href="/wiki/%EC%9E%89%EA%B8%80%EB%9E%9C%EB%93%9C%EC%9D%98_%EB%A9%94%EB%A6%AC_1%EC%84%B8" title="잉글랜드의 메리 1세">메리</a>와 헤어져 변경으로 내쫓겼다. 신실하고 위엄있는 왕비로 백성들의 존경을 받고 있던 그녀는 사람들의 동정의 대상이 되었다.</p>
<p><a href="/wiki/1536%EB%85%84" title="1536년">1536년</a> 암으로 세상을 떠나기 전까지 캐서린은 이 일방적인 이혼을 인정하지 않았다. 그녀는 앤 불린이 대관식을 치르고 나서도 문서나 편지에도 왕비 캐서린(<i>Catherine the Queen</i>)이라고 서명했다. 말년에 앤 불린과 헨리 8세의 사이가 급속도로 틀어지자 캐서린은 왕이 자신을 다시 받아들일 것이라는 희망을 버리지 못했다.</p>
<p>독실한 <a class="mw-redirect" href="/wiki/%EB%A1%9C%EB%A7%88_%EA%B0%80%ED%86%A8%EB%A6%AD" title="로마 가톨릭">로마 가톨릭</a> 신자였던 캐서린은 딸 <a href="/wiki/%EB%A9%94%EB%A6%AC_1%EC%84%B8_(%EC%9E%89%EA%B8%80%EB%9E%9C%EB%93%9C)" title="메리 1세 (잉글랜드)">메리 1세</a>에게 지대한 영향을 미쳤다.</p>
