<h2><span class="mw-headline" id=".EA.B0.9C.EC.9A.94">개요</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%95%84%EB%9E%8D_%EC%A0%9C%EA%B5%AD&amp;action=edit&amp;section=1" title="부분 편집: 개요">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>일반적으로 다음과 같은 <a href="/wiki/%EC%B9%BC%EB%A6%AC%ED%8C%8C" title="칼리파">칼리파</a>가 이끄는 시기(칼리파조)를 가리킨다. 칼리파는 <a href="/wiki/%EB%AC%B4%ED%95%A8%EB%A7%88%EB%93%9C" title="무함마드">무함마드</a>의 정통성을 이어받은 정치 지도자이다.</p>
<ul>
<li><a class="mw-redirect" href="/wiki/%EC%A0%95%ED%86%B5_%EC%B9%BC%EB%A6%AC%ED%8C%8C_%EC%8B%9C%EB%8C%80" title="정통 칼리파 시대">정통 칼리파 시대</a> (632년 ~ 661년)</li>
<li><a href="/wiki/%EC%9A%B0%EB%A7%88%EC%9D%B4%EC%95%BC_%EC%99%95%EC%A1%B0" title="우마이야 왕조">우마이야 왕조</a> (661년 ~ 750년) - 라시둔 칼리파의 계승자
<ul>
<li><a href="/wiki/%EC%95%8C%EC%95%88%EB%8B%AC%EB%A3%A8%EC%8A%A4" title="알안달루스">이슬람화 된 스페인</a>의 우마이야 토후국 (750년 ~ 929년)</li>
<li>이슬람화 된 스페인의 우마이야 <a href="/wiki/%ED%9B%84%EC%9A%B0%EB%A7%88%EC%9D%B4%EC%95%BC_%EC%99%95%EC%A1%B0" title="후우마이야 왕조">후우마이야 왕조</a>( (929년 ~ 1031년)</li>
</ul>
</li>
<li><a href="/wiki/%EC%95%84%EB%B0%94%EC%8A%A4_%EC%99%95%EC%A1%B0" title="아바스 왕조">아바스 왕조</a> (750년 ~ 1258년) - 우마이야 칼리파의 계승자</li>

<h2><span class="mw-headline" id=".EC.A0.95.ED.86.B5_.EC.B9.BC.EB.A6.AC.ED.8C.8C_.EC.8B.9C.EB.8C.80">정통 칼리파 시대</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%95%84%EB%9E%8D_%EC%A0%9C%EA%B5%AD&amp;action=edit&amp;section=2" title="부분 편집: 정통 칼리파 시대">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<div class="rellink relarticle mainarticle"><img alt="" data-file-height="128" data-file-width="128" height="16" src="//upload.wikimedia.org/wikipedia/commons/thumb/e/ec/Crystal_Clear_app_xmag.svg/16px-Crystal_Clear_app_xmag.svg.png" srcset="//upload.wikimedia.org/wikipedia/commons/thumb/e/ec/Crystal_Clear_app_xmag.svg/24px-Crystal_Clear_app_xmag.svg.png 1.5x, //upload.wikimedia.org/wikipedia/commons/thumb/e/ec/Crystal_Clear_app_xmag.svg/32px-Crystal_Clear_app_xmag.svg.png 2x" width="16"/> 이 부분의 본문은 <a class="mw-redirect" href="/wiki/%EC%A0%95%ED%86%B5_%EC%B9%BC%EB%A6%AC%ED%8C%8C_%EC%8B%9C%EB%8C%80" title="정통 칼리파 시대">정통 칼리파 시대</a>입니다.</div>
<p>정통 칼리파국은 무함마드의 직접적인 후계자로서 <a href="/wiki/632%EB%85%84" title="632년">632년</a>부터 <a href="/wiki/661%EB%85%84" title="661년">661년</a>까지 <a class="mw-redirect" href="/wiki/%EC%95%84%EB%9E%8D_%EC%84%B8%EA%B3%84" title="아랍 세계">아랍 세계</a>를 지배했다. 네 명의 지도자가 다스린 정통 칼리파국에서 이슬람은 중동 지역의 가장 강력한 세력 중 하나로 성장했다. 라쉬둔은 이전에 이 지역을 지배했던 <a href="/wiki/%ED%8E%98%EB%A5%B4%EC%8B%9C%EC%95%84_%EC%A0%9C%EA%B5%AD" title="페르시아 제국">페르시아 제국</a>과 <a href="/wiki/%EB%B9%84%EC%9E%94%ED%8B%B0%EC%9B%80_%EC%A0%9C%EA%B5%AD" title="비잔티움 제국">비잔티움 제국</a>과는 달리 세금을 줄이고 지역의 자치권과 종교의 자유를 제공하여 지지를 받았다.</p>
<h2><span class="mw-headline" id=".EC.9A.B0.EB.A7.88.EC.9D.B4.EC.95.BC_.EC.99.95.EC.A1.B0">우마이야 왕조</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%95%84%EB%9E%8D_%EC%A0%9C%EA%B5%AD&amp;action=edit&amp;section=3" title="부분 편집: 우마이야 왕조">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<div class="rellink relarticle mainarticle"><img alt="" data-file-height="128" data-file-width="128" height="16" src="//upload.wikimedia.org/wikipedia/commons/thumb/e/ec/Crystal_Clear_app_xmag.svg/16px-Crystal_Clear_app_xmag.svg.png" srcset="//upload.wikimedia.org/wikipedia/commons/thumb/e/ec/Crystal_Clear_app_xmag.svg/24px-Crystal_Clear_app_xmag.svg.png 1.5x, //upload.wikimedia.org/wikipedia/commons/thumb/e/ec/Crystal_Clear_app_xmag.svg/32px-Crystal_Clear_app_xmag.svg.png 2x" width="16"/> 이 부분의 본문은 <a href="/wiki/%EC%9A%B0%EB%A7%88%EC%9D%B4%EC%95%BC_%EC%99%95%EC%A1%B0" title="우마이야 왕조">우마이야 왕조</a>입니다.</div>
<p><a href="/wiki/660%EB%85%84" title="660년">660년</a>부터 <a href="/wiki/750%EB%85%84" title="750년">750년</a>까지 존속했던 칼리파 국가이다. 수도는 현 <a href="/wiki/%EC%8B%9C%EB%A6%AC%EC%95%84" title="시리아">시리아</a>의 수도인 <a href="/wiki/%EB%8B%A4%EB%A7%88%EC%8A%A4%EC%BF%A0%EC%8A%A4" title="다마스쿠스">다마스쿠스</a>였다. 시조인 우마야의 가계는 무함마드와 같은 조상을 가진 것으로 전승된다.</p>
<h2><span class="mw-headline" id=".EC.95.84.EB.B0.94.EC.8A.A4_.EC.99.95.EC.A1.B0">아바스 왕조</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%95%84%EB%9E%8D_%EC%A0%9C%EA%B5%AD&amp;action=edit&amp;section=4" title="부분 편집: 아바스 왕조">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<div class="rellink relarticle mainarticle"><img alt="" data-file-height="128" data-file-width="128" height="16" src="//upload.wikimedia.org/wikipedia/commons/thumb/e/ec/Crystal_Clear_app_xmag.svg/16px-Crystal_Clear_app_xmag.svg.png" srcset="//upload.wikimedia.org/wikipedia/commons/thumb/e/ec/Crystal_Clear_app_xmag.svg/24px-Crystal_Clear_app_xmag.svg.png 1.5x, //upload.wikimedia.org/wikipedia/commons/thumb/e/ec/Crystal_Clear_app_xmag.svg/32px-Crystal_Clear_app_xmag.svg.png 2x" width="16"/> 이 부분의 본문은 <a href="/wiki/%EC%95%84%EB%B0%94%EC%8A%A4_%EC%99%95%EC%A1%B0" title="아바스 왕조">아바스 왕조</a>입니다.</div>
<p>우마이야 왕조를 무너뜨리고 세워진 칼리파 국가로써 <a href="/wiki/750%EB%85%84" title="750년">750년</a>부터 <a href="/wiki/1258%EB%85%84" title="1258년">1258년</a>까지 존속했다. 수도는 <a href="/wiki/%EB%B0%94%EA%B7%B8%EB%8B%A4%EB%93%9C" title="바그다드">바그다드</a>였다가 <a href="/wiki/%EB%AA%BD%EA%B3%A8%EC%A1%B1" title="몽골족">몽골족</a>에게 함락되자 <a href="/wiki/%EC%B9%B4%EC%9D%B4%EB%A1%9C" title="카이로">카이로</a>로 천도했다. 8세기 중반 바그다드에 수도를 정하면서 아랍 제국은 <a class="new" href="/w/index.php?title=%EC%9D%B4%EC%8A%AC%EB%9E%8C_%ED%99%A9%EA%B8%88%EC%8B%9C%EB%8C%80&amp;action=edit&amp;redlink=1" title="이슬람 황금시대 (없는 문서)">이슬람 황금시대</a>를 맞게 되었다.</p>
<h2><span class="mw-headline" id=".ED.8C.8C.ED.8B.B0.EB.A7.88_.EC.99.95.EC.A1.B0">파티마 왕조</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%95%84%EB%9E%8D_%EC%A0%9C%EA%B5%AD&amp;action=edit&amp;section=5" title="부분 편집: 파티마 왕조">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<div class="rellink relarticle mainarticle"><img alt="" data-file-height="128" data-file-width="128" height="16" src="//upload.wikimedia.org/wikipedia/commons/thumb/e/ec/Crystal_Clear_app_xmag.svg/16px-Crystal_Clear_app_xmag.svg.png" srcset="//upload.wikimedia.org/wikipedia/commons/thumb/e/ec/Crystal_Clear_app_xmag.svg/24px-Crystal_Clear_app_xmag.svg.png 1.5x, //upload.wikimedia.org/wikipedia/commons/thumb/e/ec/Crystal_Clear_app_xmag.svg/32px-Crystal_Clear_app_xmag.svg.png 2x" width="16"/> 이 부분의 본문은 <a href="/wiki/%ED%8C%8C%ED%8B%B0%EB%A7%88_%EC%99%95%EC%A1%B0" title="파티마 왕조">파티마 왕조</a>입니다.</div>
<p><a href="/wiki/%EC%8B%9C%EC%95%84%ED%8C%8C" title="시아파">시아파</a>의 일족이자 무함마드의 딸 파티마의 후손이라 주장하는 <a href="/wiki/%EC%9D%B4%EC%8A%A4%EB%A7%88%EC%9D%BC%ED%8C%8C" title="이스마일파">이스마일파</a>가 세워 <a href="/wiki/909%EB%85%84" title="909년">909년</a>부터 <a href="/wiki/1171%EB%85%84" title="1171년">1171년</a>까지 존속했다. 수도는 초기에는 <a class="mw-redirect" href="/wiki/%EB%A7%88%EB%94%94%EC%95%BC" title="마디야">마디야</a>에 있다가 카이로로 이전했다. <a href="/wiki/%EC%8B%AD%EC%9E%90%EA%B5%B0" title="십자군">십자군</a>을 몰아내고 <a href="/wiki/%EC%95%84%EC%9D%B4%EC%9C%A0%EB%B8%8C_%EC%88%A0%ED%83%84%EA%B5%AD" title="아이유브 술탄국">아이유브 술탄국</a>을 세운 <a href="/wiki/%EC%88%98%EB%8B%88%ED%8C%8C" title="수니파">수니파</a>의 <a class="mw-redirect" href="/wiki/%EC%82%B4%EB%9D%BC%ED%9D%90_%EC%95%97_%EB%94%98" title="살라흐 앗 딘">살라흐 앗 딘</a>에 의해 멸망했다. 수도는 <a href="/wiki/%EB%A7%88%EB%94%94%EC%95%84" title="마디아">마디아</a>, <a class="new" href="/w/index.php?title=%EC%95%8C_%EB%A7%8C%EC%88%98%EB%A6%AC%EC%95%84&amp;action=edit&amp;redlink=1" title="알 만수리아 (없는 문서)">알 만수리아</a>, <a href="/wiki/%EC%B9%B4%EC%9D%B4%EB%A1%9C" title="카이로">카이로</a> 순서이다. 파티마 칼리파조를 멸망시킨 아이유브 술탄국의 군주는 "칼리파"가 아닌 "<a href="/wiki/%EC%88%A0%ED%83%84" title="술탄">술탄</a>"을 칭했으며, 아이유브 술탄국을 대체 및 계승한 <a href="/wiki/%EB%A7%98%EB%A3%A8%ED%81%AC_%EC%88%A0%ED%83%84%EA%B5%AD" title="맘루크 술탄국">맘루크 술탄국</a> 역시 그러했다.</p>
