<p><br/>
<b>아루나찰프라데시 주</b>(<span style="font-size: smaller;"><a href="/wiki/%EC%98%81%EC%96%B4" title="영어">영어</a>:</span> <span lang="en">Arunachal Pradesh</span>, <span style="font-size: smaller;"><a href="/wiki/%ED%9E%8C%EB%94%94%EC%96%B4" title="힌디어">힌디어</a>:</span> <span lang="hi">अरुणाचल प्रदेश</span>)는 <a href="/wiki/%EC%9D%B8%EB%8F%84" title="인도">인도</a> 북동부에 위치한 주로 주도는 <a href="/wiki/%EC%9D%B4%ED%83%80%EB%82%98%EA%B0%80%EB%A5%B4" title="이타나가르">이타나가르</a>이며 면적은 83,743㎢, 인구는 965,000명(<a href="/wiki/1994%EB%85%84" title="1994년">1994년</a> 기준)이다. <a href="/wiki/%EC%98%81%EC%96%B4" title="영어">영어</a>와 <a href="/wiki/%ED%9E%8C%EB%94%94%EC%96%B4" title="힌디어">힌디어</a>를 공용어로 사용한다.</p>
<p><a href="/wiki/1987%EB%85%84" title="1987년">1987년</a> <a href="/wiki/2%EC%9B%94_20%EC%9D%BC" title="2월 20일">2월 20일</a> 주로 승격되었다. 남쪽으로는 <a href="/wiki/%EC%95%84%EC%82%BC_%EC%A3%BC" title="아삼 주">아삼 주</a>, <a href="/wiki/%EB%82%98%EA%B0%88%EB%9E%9C%EB%93%9C_%EC%A3%BC" title="나갈랜드 주">나갈랜드 주</a>와 접하며 서쪽으로는 <a href="/wiki/%EB%B6%80%ED%83%84" title="부탄">부탄</a>, 동쪽으로는 <a href="/wiki/%EB%AF%B8%EC%96%80%EB%A7%88" title="미얀마">미얀마</a>, 북쪽으로는 <a href="/wiki/%EC%A4%91%ED%99%94%EC%9D%B8%EB%AF%BC%EA%B3%B5%ED%99%94%EA%B5%AD" title="중화인민공화국">중화인민공화국</a>과 국경을 접한다.</p>

<h2><span class="mw-headline" id=".EA.B5.AD.EA.B2.BD.EC.84.A0_.EB.B6.84.EC.9F.81">국경선 분쟁</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%95%84%EB%A3%A8%EB%82%98%EC%B0%B0%ED%94%84%EB%9D%BC%EB%8D%B0%EC%8B%9C_%EC%A3%BC&amp;action=edit&amp;section=1" title="부분 편집: 국경선 분쟁">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>인도 정부가 실효적 지배중이지만, 중국 정부는 자국영토라고 주장하며 <b>남티베트</b>, <b>장난(藏南)</b>이라고 부른다. 서부의 <a href="/wiki/%EC%B9%B4%EC%8A%88%EB%AF%B8%EB%A5%B4" title="카슈미르">카슈미르</a>, 동부의 아루나찰프라데시는 중국과 인도의 대표적인 국경선 분쟁지역이다.</p>
<h2><span class="mw-headline" id=".EB.B8.8C.EB.9D.BC.EB.AA.A8.EC.8A.A4_.EB.AF.B8.EC.82.AC.EC.9D.BC">브라모스 미사일</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%95%84%EB%A3%A8%EB%82%98%EC%B0%B0%ED%94%84%EB%9D%BC%EB%8D%B0%EC%8B%9C_%EC%A3%BC&amp;action=edit&amp;section=2" title="부분 편집: 브라모스 미사일">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>2016년 7월 3일, <a href="/wiki/%EB%82%98%EB%A0%8C%EB%93%9C%EB%9D%BC_%EB%AA%A8%EB%94%94" title="나렌드라 모디">나렌드라 모디</a> 총리가 주재한 <a class="new" href="/w/index.php?title=%EC%9D%B8%EB%8F%84_%EB%82%B4%EA%B0%81%EC%95%88%EB%B3%B4%EC%9C%84%EC%9B%90%ED%9A%8C&amp;action=edit&amp;redlink=1" title="인도 내각안보위원회 (없는 문서)">인도 내각안보위원회</a> 회의에서 430억 루피(7,177억 원)를 들여 중국과 접한 동북부 아루나찰 프라데시 주에 <a class="mw-redirect" href="/wiki/%EB%B8%8C%EB%9D%BC%EB%AA%A8%EC%8A%A4_%EB%AF%B8%EC%82%AC%EC%9D%BC" title="브라모스 미사일">브라모스 미사일</a> 연대를 배치하기로 결정했다. 이 연대는 100기의 <a class="mw-redirect" href="/wiki/%EB%B8%8C%EB%9D%BC%EB%AA%A8%EC%8A%A4_%EB%AF%B8%EC%82%AC%EC%9D%BC" title="브라모스 미사일">브라모스 미사일</a>과 4대의 이동식 발사대, 대형 트럭, 이동 지휘부로 구성될 것이다.<sup class="reference" id="cite_ref-1"><a href="#cite_note-1">[1]</a></sup></p>
