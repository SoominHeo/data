<p><b>아르투어 자이스잉크바르트</b>(<span lang="de" xml:lang="de">Arthur Seyß-Inquart</span>, <a href="/wiki/1892%EB%85%84" title="1892년">1892년</a> <a href="/wiki/7%EC%9B%94_22%EC%9D%BC" title="7월 22일">7월 22일</a> ~ <a href="/wiki/1946%EB%85%84" title="1946년">1946년</a> <a href="/wiki/10%EC%9B%94_16%EC%9D%BC" title="10월 16일">10월 16일</a>)는 <a href="/wiki/%EC%98%A4%EC%8A%A4%ED%8A%B8%EB%A6%AC%EC%95%84" title="오스트리아">오스트리아</a>의 정치가, 법률가로, <a class="mw-redirect" href="/wiki/%EB%AF%BC%EC%A1%B1%EC%82%AC%ED%9A%8C%EC%A3%BC%EC%9D%98_%EB%8F%85%EC%9D%BC_%EB%85%B8%EB%8F%99%EC%9E%90%EB%8B%B9" title="민족사회주의 독일 노동자당">나치당</a> 당원이기도 하다.</p>
<p><a href="/wiki/%EB%AA%A8%EB%9D%BC%EB%B0%94" title="모라바">모라바</a> 지방(출생 시에는 <a href="/wiki/%EC%98%A4%EC%8A%A4%ED%8A%B8%EB%A6%AC%EC%95%84-%ED%97%9D%EA%B0%80%EB%A6%AC_%EC%A0%9C%EA%B5%AD" title="오스트리아-헝가리 제국">오스트리아-헝가리 제국</a> 영토)에서 아르투르 차이티흐(Arthur Zajtich)라는 이름으로 태어났고, <a href="/wiki/1907%EB%85%84" title="1907년">1907년</a> <a href="/wiki/%EB%B9%88" title="빈">빈</a>으로 이주한 뒤 독일식 성인 자이스잉크바르트로 고쳤다. <a class="mw-redirect" href="/wiki/%EB%B9%88_%EB%8C%80%ED%95%99" title="빈 대학">빈 대학</a>에서 법률을 전공했고, <a class="mw-redirect" href="/wiki/1%EC%B0%A8_%EC%84%B8%EA%B3%84%EB%8C%80%EC%A0%84" title="1차 세계대전">1차대전</a> 중에는 오스트리아군에 입대해 전쟁에 참가했다.</p>
<p>종전 후 변호사로 활동했고, <a href="/wiki/%EC%98%A4%EC%8A%A4%ED%8A%B8%EB%A6%AC%EC%95%84" title="오스트리아">오스트리아</a>의 <a href="/wiki/%EA%B7%B9%EC%9A%B0" title="극우">극우</a> <a href="/wiki/%EB%AF%BC%EC%A1%B1%EC%A3%BC%EC%9D%98" title="민족주의">민족주의</a> 정당인 <a href="/wiki/%EC%A1%B0%EA%B5%AD%EC%A0%84%EC%84%A0_(%EC%98%A4%EC%8A%A4%ED%8A%B8%EB%A6%AC%EC%95%84)" title="조국전선 (오스트리아)">조국전선</a>과 긴밀한 관계를 맺으면서 <a href="/wiki/1933%EB%85%84" title="1933년">1933년</a>에는 <a href="/wiki/%EC%97%A5%EA%B2%94%EB%B2%A0%EB%A5%B4%ED%8A%B8_%EB%8F%8C%ED%91%B8%EC%8A%A4" title="엥겔베르트 돌푸스">엥겔베르트 돌푸스</a> 내각에 입각하기도 했다. 돌푸스가 암살된 뒤에도 정계에 남아있었으나, <a href="/wiki/%EB%8F%85%EC%9D%BC" title="독일">독일</a>의 오스트리아 병합을 위해 암약하고 있던 나치당과도 막후 관계를 맺고 있었다.</p>
<p><a href="/wiki/1938%EB%85%84" title="1938년">1938년</a> <a href="/wiki/3%EC%9B%94_13%EC%9D%BC" title="3월 13일">3월 13일</a>에 오스트리아가 <a href="/wiki/%EB%82%98%EC%B9%98_%EB%8F%85%EC%9D%BC" title="나치 독일">나치 독일</a>에 병합되어 <a class="new" href="/w/index.php?title=%EC%98%A4%EC%8A%A4%ED%8A%B8%EB%A7%88%EB%A5%B4%ED%81%AC_%EC%A3%BC&amp;action=edit&amp;redlink=1" title="오스트마르크 주 (없는 문서)">오스트마르크 주</a>로 격하되면서 정식으로 나치당에 입당했고, <a href="/wiki/%EC%95%84%EB%8F%8C%ED%94%84_%ED%9E%88%ED%8B%80%EB%9F%AC" title="아돌프 히틀러">히틀러</a>에 의해 오스트마르크의 제국 관리인으로 입각해 합병과 관련된 법률의 검토와 제정, 처리를 담당했다.</p>
<p><a href="/wiki/1940%EB%85%84" title="1940년">1940년</a> <a href="/wiki/5%EC%9B%94" title="5월">5월</a> 독일군이 <a href="/wiki/%EB%84%A4%EB%8D%9C%EB%9E%80%EB%93%9C" title="네덜란드">네덜란드</a>를 점령한 후 제국 전권위원 자격으로 파견되었고, 전쟁 물자 수탈과 <a href="/wiki/%EC%9C%A0%EB%8C%80%EC%9D%B8" title="유대인">유대인</a>이나 정치범 등 '민족의 적' 에 대한 대대적인 검거와 분류 작업, 처벌, <a href="/wiki/%EA%B0%95%EC%A0%9C_%EC%88%98%EC%9A%A9%EC%86%8C" title="강제 수용소">강제 수용소</a> 운영 등에 깊이 관여해 '네덜란드의 도살자' 라는 악명높은 별명으로 불리게 되었다.</p>
<p><a href="/wiki/1944%EB%85%84" title="1944년">1944년</a> <a href="/wiki/%EB%85%B8%EB%A5%B4%EB%A7%9D%EB%94%94_%EC%83%81%EB%A5%99_%EC%9E%91%EC%A0%84" title="노르망디 상륙 작전">노르망디 상륙 작전</a>의 성공으로 네덜란드에 대한 연합군의 공세가 시작된 뒤에는 히틀러가 내세운 초토화 정책에 따라 네덜란드 각지의 항구나 제방에 대한 파괴 명령을 내려 공세 지연을 꾀했고, 연합군과 대치 중에는 휴전 회담에 독일 대표로 참석하기도 했다.</p>
<p>히틀러 자살 후 <a href="/wiki/%EC%B9%B4%EB%A5%BC_%EB%90%98%EB%8B%88%EC%B8%A0" title="카를 되니츠">카를 되니츠</a> 휘하의 내각에서는 <a href="/wiki/%EC%9A%94%EC%95%84%ED%9E%98_%ED%8F%B0_%EB%A6%AC%EB%B2%A4%ED%8A%B8%EB%A1%9C%ED%94%84" title="요아힘 폰 리벤트로프">요아힘 폰 리벤트로프</a>의 후임으로 외무 장관이 되었으나, 3일 만에 물러났다. 종전 직전 도피처였던 <a href="/wiki/%ED%95%A8%EB%B6%80%EB%A5%B4%ED%81%AC" title="함부르크">함부르크</a>에서 체포되었고, <a href="/wiki/%EB%89%98%EB%A5%B8%EB%B2%A0%EB%A5%B4%ED%81%AC_%EC%9E%AC%ED%8C%90" title="뉘른베르크 재판">뉘른베르크 재판</a>에서 주요 기소 항목 4개 중 3개 부문의 유죄가 입증되어 <a href="/wiki/%EC%82%AC%ED%98%95" title="사형">사형</a> 판결을 받아 처형되었다.</p>
