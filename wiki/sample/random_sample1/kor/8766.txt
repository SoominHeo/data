<div class="mw-content-ltr" dir="ltr" id="mw-content-text" lang="ko"><script>function mfTempOpenSection(id){var block=document.getElementById("mf-section-"+id);block.className+=" open-block";block.previousSibling.className+=" open-block";}</script><p><b>아르헨티나 축구 협회</b>(<span lang="es" xml:lang="es">Asociación del Fútbol Argentino</span>)는 <a href="/wiki/%EC%95%84%EB%A5%B4%ED%97%A8%ED%8B%B0%EB%82%98" title="아르헨티나">아르헨티나</a>의 <a href="/wiki/%EC%B6%95%EA%B5%AC_%ED%98%91%ED%9A%8C" title="축구 협회">축구 협회</a>로써 <a href="/wiki/%EB%B6%80%EC%97%90%EB%85%B8%EC%8A%A4%EC%95%84%EC%9D%B4%EB%A0%88%EC%8A%A4" title="부에노스아이레스">부에노스아이레스</a>에 본부를 두고 있다. 협회는 <a href="/wiki/%EC%95%84%EB%A5%B4%ED%97%A8%ED%8B%B0%EB%82%98_%ED%94%84%EB%A6%AC%EB%A9%94%EB%9D%BC_%EB%94%94%EB%B9%84%EC%8B%9C%EC%98%A8" title="아르헨티나 프리메라 디비시온">아르헨티나 프리메라 디비시온</a>과 <a href="/wiki/%ED%94%84%EB%A6%AC%EB%A9%94%EB%9D%BC_B_%EB%82%98%EC%8B%9C%EC%98%A4%EB%82%A0" title="프리메라 B 나시오날">프리메라 B 나시오날</a>을 비롯한 <a href="/wiki/%EC%95%84%EB%A5%B4%ED%97%A8%ED%8B%B0%EB%82%98_%EC%B6%95%EA%B5%AC_%EB%A6%AC%EA%B7%B8_%EC%8B%9C%EC%8A%A4%ED%85%9C" title="아르헨티나 축구 리그 시스템">아르헨티나 축구 리그 시스템</a> 전체를 운영하고 있으며, 각급 <a href="/wiki/%EC%95%84%EB%A5%B4%ED%97%A8%ED%8B%B0%EB%82%98_%EC%B6%95%EA%B5%AC_%EA%B5%AD%EA%B0%80%EB%8C%80%ED%91%9C%ED%8C%80" title="아르헨티나 축구 국가대표팀">국가대표팀</a>과 <a href="/wiki/%ED%92%8B%EC%82%B4" title="풋살">풋살</a>팀에 대해서도 관여하고 있다.</p>

<h2><span class="mw-headline" id=".EC.97.AD.EC.82.AC">역사</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%95%84%EB%A5%B4%ED%97%A8%ED%8B%B0%EB%82%98_%EC%B6%95%EA%B5%AC_%ED%98%91%ED%9A%8C&amp;action=edit&amp;section=1" title="부분 편집: 역사">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p><b>아르헨티나 축구 리그</b>(<span lang="en" xml:lang="en">The Argentine Association Football League</span>)는 <a class="mw-redirect" href="/wiki/CONMEBOL" title="CONMEBOL">남아메리카</a>에서 가장 오래된, 그리고 영국을 제외하고는 세계에서 가장 오래된 축구 관련 조직 중 하나로써 <a href="/wiki/1893%EB%85%84" title="1893년">1893년</a> 설립되었다. <a href="/wiki/1903%EB%85%84" title="1903년">1903년</a>에 <b>아르헨티나 축구협회</b>(<span lang="en" xml:lang="en">Argentine Football Association</span>)로 이름을 바꾸었고, <a href="/wiki/1912%EB%85%84" title="1912년">1912년</a>에는 공식 명칭을 <a href="/wiki/%EC%8A%A4%ED%8E%98%EC%9D%B8%EC%96%B4" title="스페인어">스페인어</a> <span lang="es" xml:lang="es">Asociación Argentina de Football</span>로 개칭하였다.</p>
<p><a href="/wiki/1912%EB%85%84" title="1912년">1912년</a>에는 <b>아르헨티나 축구연맹</b>(<span lang="es" xml:lang="es">Federación Argentina de Football</span>, FAF)이, <a href="/wiki/1919%EB%85%84" title="1919년">1919년</a>에는 <b>아마추어 축구협회</b>(<span lang="es" xml:lang="es">Asociación Amateurs de Football</span>, AAF)가 생겨 경쟁하기도 했지만, <a href="/wiki/1926%EB%85%84" title="1926년">1926년</a> 다시 축구협회로 통합되었다.</p>
<p><a href="/wiki/1926%EB%85%84" title="1926년">1926년</a>에는 이름을 다시 <b>아르헨티나 아마추어 축구협회</b>(<span lang="es" xml:lang="es">Asociación Amateur de Argentina de Football</span>)로 바꾸었지만, <a href="/wiki/1931%EB%85%84" title="1931년">1931년</a> 프로리그가 출범되면서 협회 명칭은 <b>아마추어/프로 축구협회</b>(<span lang="es" xml:lang="es">Asociación de Football Amateur y Profesionales</span>)로 또 바뀌었다.</p>
<p>프로리그 첫 경기는 <a href="/wiki/1931%EB%85%84" title="1931년">1931년</a> <a href="/wiki/5%EC%9B%94_31%EC%9D%BC" title="5월 31일">5월 31일</a> 열렸다.<sup class="reference" id="cite_ref-1"><a href="#cite_note-1">[1]</a></sup> 이때 출범한 출범한 <b>아르헨티나 축구 리그</b>(<span lang="es" xml:lang="es">Liga Argentina de Football</span>)는 <a href="/wiki/1934%EB%85%84" title="1934년">1934년</a> <a href="/wiki/11%EC%9B%94_3%EC%9D%BC" title="11월 3일">11월 3일</a> 아마추어/프로 축구협회와 통합하면서 현재의 <b>아르헨티나 축구협회</b> (<span lang="es" xml:lang="es">Asociación de Fútbol Argentino</span>)가 되었다.</p>
<h3><span class="mw-headline" id="2030.EB.85.84_FIFA_.EC.9B.94.EB.93.9C.EC.BB.B5_.EC.9C.A0.EC.B9.98">2030년 FIFA 월드컵 유치</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%95%84%EB%A5%B4%ED%97%A8%ED%8B%B0%EB%82%98_%EC%B6%95%EA%B5%AC_%ED%98%91%ED%9A%8C&amp;action=edit&amp;section=2" title="부분 편집: 2030년 FIFA 월드컵 유치">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>첫 <a href="/wiki/FIFA_%EC%9B%94%EB%93%9C%EC%BB%B5" title="FIFA 월드컵">FIFA 월드컵</a>인 <a href="/wiki/1930%EB%85%84_FIFA_%EC%9B%94%EB%93%9C%EC%BB%B5" title="1930년 FIFA 월드컵">1930년 FIFA 월드컵</a>에서 결승에 오른 바 있는 아르헨티나와 우루과이는 첫 FIFA 월드컵 100주년이 되는 2030년 FIFA 월드컵 공동개최를 유치하고 있으며, 많은 지지를 받고 있다.<sup class="reference" id="cite_ref-2"><a href="#cite_note-2">[2]</a></sup></p>
