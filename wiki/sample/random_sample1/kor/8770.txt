<p><b>아르헨티나개미</b>(Linepithema humile, 구 Iridomyrmex humilis)는 색이 어두운 <a href="/wiki/%EA%B0%9C%EB%AF%B8" title="개미">개미</a>로 원래 <a class="new" href="/w/index.php?title=%ED%8C%8C%EB%82%98%EB%A7%88_%EA%B0%95&amp;action=edit&amp;redlink=1" title="파나마 강 (없는 문서)">파나마 강</a>의 주변에 살았다. 아르헨티나개미는 <a class="new" href="/w/index.php?title=%EC%B9%A8%EC%9E%85%EC%A2%85&amp;action=edit&amp;redlink=1" title="침입종 (없는 문서)">침입종</a>으로 <a href="/wiki/%EC%A7%80%EC%A4%91%ED%95%B4" title="지중해">지중해</a> 주변의 여러 지점에 정착하였다.<sup class="reference" id="cite_ref-1"><a href="#cite_note-1">[1]</a></sup>. 인간 때문에 <a href="/wiki/%EB%89%B4%EC%A7%88%EB%9E%9C%EB%93%9C" title="뉴질랜드">뉴질랜드</a>, <a href="/wiki/%EC%9D%BC%EB%B3%B8" title="일본">일본</a>, <a href="/wiki/%EC%9D%B4%EC%8A%A4%ED%84%B0_%EC%84%AC" title="이스터 섬">이스터 섬</a>, <a href="/wiki/%EC%98%A4%EC%8A%A4%ED%8A%B8%EB%A0%88%EC%9D%BC%EB%A6%AC%EC%95%84" title="오스트레일리아">오스트레일리아</a>, <a href="/wiki/%ED%95%98%EC%99%80%EC%9D%B4_%EC%A0%9C%EB%8F%84" title="하와이 제도">하와이</a>, <a href="/wiki/%EC%9C%A0%EB%9F%BD" title="유럽">유럽</a>, <a href="/wiki/%EB%AF%B8%EA%B5%AD" title="미국">미국</a> 등 여러 지역으로 퍼졌다.</p>

<h2><span class="mw-headline" id=".EC.84.A4.EB.AA.85">설명</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%95%84%EB%A5%B4%ED%97%A8%ED%8B%B0%EB%82%98%EA%B0%9C%EB%AF%B8&amp;action=edit&amp;section=1" title="부분 편집: 설명">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>아르헨티나개미의 일개미는 길이가 약3 mm 이고 1 mm정도의 틈새는 쉽게 빠져나온다. 여왕 개미의 크기는 일개미들의 2~4배 정도 크다. 아르헨티나개미는 땅 속, 콘크리트 벽의 틈새, 버려진 목재의 틈, 그리고 인간의 집에도 집을 짓는다. 독일의 곤충학자 <a class="new" href="/w/index.php?title=%EA%B5%AC%EC%8A%A4%ED%83%80%ED%94%84_%EB%A7%88%EC%9D%B4%EC%96%B4&amp;action=edit&amp;redlink=1" title="구스타프 마이어 (없는 문서)">구스타프 마이어</a>(Gustav Mayr)는 최초로 아르헨티나개미를 <a href="/wiki/%EC%95%84%EB%A5%B4%ED%97%A8%ED%8B%B0%EB%82%98" title="아르헨티나">아르헨티나</a>의 <a class="mw-redirect" href="/wiki/%EB%B6%80%EC%97%90%EB%85%B8%EC%8A%A4_%EC%95%84%EC%9D%B4%EB%A0%88%EC%8A%A4" title="부에노스 아이레스">부에노스 아이레스</a>에서 발견하였다. 아르헨티나개미는 임시로 <a href="/wiki/%ED%9D%B0%EB%B0%9C%EB%A7%88%EB%94%94%EA%B0%9C%EB%AF%B8%EC%86%8D" title="흰발마디개미속">흰발마디개미속</a>(Iridomyrmex)으로 분류되었다가, 1990년대 초반에 <a class="new" href="/w/index.php?title=%EB%A6%AC%EB%84%A4%ED%94%BC%ED%85%8C%EB%A7%88%EC%86%8D&amp;action=edit&amp;redlink=1" title="리네피테마속 (없는 문서)">리네피테마속</a>(Linepithema)로 최종 분류되었다.</p>
<h2><span class="mw-headline" id=".EB.B6.84.ED.8F.AC">분포</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%95%84%EB%A5%B4%ED%97%A8%ED%8B%B0%EB%82%98%EA%B0%9C%EB%AF%B8&amp;action=edit&amp;section=2" title="부분 편집: 분포">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>아르헨티나개미의 원래 <a class="new" href="/w/index.php?title=%ED%8C%8C%EB%82%98%EB%A7%88_%EA%B0%95&amp;action=edit&amp;redlink=1" title="파나마 강 (없는 문서)">파나마 강</a>의 주변 지역에만 분포하고 있었다. 최근에는 <a href="/wiki/%EC%95%84%EB%A5%B4%ED%97%A8%ED%8B%B0%EB%82%98" title="아르헨티나">아르헨티나</a>, <a href="/wiki/%EB%B8%8C%EB%9D%BC%EC%A7%88" title="브라질">브라질</a>, <a href="/wiki/%EC%B9%A0%EB%A0%88" title="칠레">칠레</a>, <a href="/wiki/%EC%BD%9C%EB%A1%AC%EB%B9%84%EC%95%84" title="콜롬비아">콜롬비아</a>, <a href="/wiki/%EC%97%90%EC%BD%B0%EB%8F%84%EB%A5%B4" title="에콰도르">에콰도르</a>, 그리고 <a href="/wiki/%ED%8E%98%EB%A3%A8" title="페루">페루</a>에 퍼졌다.<sup class="reference" id="cite_ref-2"><a href="#cite_note-2">[2]</a></sup> 이 나라들 외에도 아르헨티나개미는 최소한 15개국에 퍼졌으며, 남극 이외의 모든 대륙에 침입하였고 많은 섬에도 전파되었다.<sup class="reference" id="cite_ref-3"><a href="#cite_note-3">[3]</a></sup></p>
<h2><span class="mw-headline" id=".ED.96.89.EB.8F.99">행동</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%95%84%EB%A5%B4%ED%97%A8%ED%8B%B0%EB%82%98%EA%B0%9C%EB%AF%B8&amp;action=edit&amp;section=3" title="부분 편집: 행동">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>아르헨티나개미는 다른 개미와 달리, 서식지가 달라도 서로 싸우지 않는다. 그들의 유전자가 서로 너무 비슷하기 때문에 아르헨티나개미가 다른 아르헨티나개미의 집에 들어가도 공격받지 않는다.<sup class="reference" id="cite_ref-4"><a href="#cite_note-4">[4]</a></sup> 이와 반대로, 원산지의 아르헨티나개미는 유전적으로 차이가 있어 서로 싸우며, 새롭게 퍼진 지역보다 작은 집을 짓는다. <a href="/wiki/%EB%82%A8%EC%95%84%EB%A9%94%EB%A6%AC%EC%B9%B4" title="남아메리카">남아메리카</a>에 사는 아르헨티나개미는 여러 다른 개미들과 공생하며 새로 퍼진 지역보다 낮은 밀도를 유지한다.<sup class="reference" id="cite_ref-5"><a href="#cite_note-5">[5]</a></sup></p>
<h2><span class="mw-headline" id=".EC.83.9D.EC.8B.9D">생식</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%95%84%EB%A5%B4%ED%97%A8%ED%8B%B0%EB%82%98%EA%B0%9C%EB%AF%B8&amp;action=edit&amp;section=4" title="부분 편집: 생식">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>아르헨티나개미의 일개미들은 다른 개미들의 일개미들과 마찬가지로 생식 행위를 할 수 없다. 그러나 알을 어떻게 낳아야 할지 판단하여 여왕개미에게 명령을 내리는 것으로 보인다. 또한 수개미의 개체 수는 애벌레에게 주는 먹이의 양에 따라 결정되는 것으로 보인다.<sup class="reference" id="cite_ref-6"><a href="#cite_note-6">[6]</a></sup> 아르헨티나개미는 다른 개미들과 달리 <a href="/wiki/%ED%98%BC%EC%9D%B8_%EB%B9%84%ED%96%89" title="혼인 비행">혼인 비행</a>을 거의 하지 않는다. 그 대신 수정된 암개미들이 원래의 집에서 일개미 몇 마리와 함께 집에서 어느 정도 떨어진 곳으로 가서 새로운 집을 짓는다.</p>
<h2><span class="mw-headline" id=".EC.98.81.ED.96.A5">영향</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%95%84%EB%A5%B4%ED%97%A8%ED%8B%B0%EB%82%98%EA%B0%9C%EB%AF%B8&amp;action=edit&amp;section=5" title="부분 편집: 영향">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>아르헨티나개미는 세계 100대 침입종에 속한다.<sup class="reference" id="cite_ref-7"><a href="#cite_note-7">[7]</a></sup> 새로 퍼진 지역에서 아르헨티나개미는 토종 개미종을 대부분 또는 모두 멸종시켜 생태계를 위협할 수 있다. 예를 들어 <a class="mw-redirect" href="/wiki/%EC%BA%98%EB%A6%AC%ED%8F%AC%EB%8B%88%EC%95%84" title="캘리포니아">캘리포니아</a>의 <a class="new" href="/w/index.php?title=%ED%94%84%EB%A6%AC%EB%85%B8%EC%86%8C%EB%A7%88%EC%86%8D&amp;action=edit&amp;redlink=1" title="프리노소마속 (없는 문서)">프리노소마속</a>(Phrynosoma)에 속하는 도마뱀의 수가 최근 줄어들고 있는데, 이는 도마뱀이 먹는 토종 개미의 수가 아르헨티나개미 때문에 줄어든 탓으로 보인다.</p>
