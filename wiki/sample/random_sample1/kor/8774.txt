<h2><span class="mw-headline" id=".EC.BD.9C.EB.9F.BC.EB.B2.84.EC.8A.A4_.EC.9D.B4.EC.A0.84_.EC.8B.9C.EB.8C.80">콜럼버스 이전 시대</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%95%84%EB%A5%B4%ED%97%A8%ED%8B%B0%EB%82%98%EC%9D%98_%EC%97%AD%EC%82%AC&amp;action=edit&amp;section=1" title="부분 편집: 콜럼버스 이전 시대">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p><a href="/wiki/%EC%95%84%EB%A5%B4%ED%97%A8%ED%8B%B0%EB%82%98" title="아르헨티나">아르헨티나</a>에 최초로 인간이 정주한 증거는 <a href="/wiki/%ED%8C%8C%ED%83%80%EA%B3%A0%EB%8B%88%EC%95%84" title="파타고니아">파타고니아</a>에서 발견되며 기원전 11000여년으로 거슬러 올라간다. 이 지역에는 국가 없이 <a href="/wiki/%EC%9D%B8%EB%94%94%EC%98%A4" title="인디오">인디오</a>들이 부족을 이루며 살고 있었는데, 15세기 초반부터 <a class="new" href="/w/index.php?title=%ED%8C%8C%EC%B0%A8%EC%BF%A0%ED%85%8D&amp;action=edit&amp;redlink=1" title="파차쿠텍 (없는 문서)">파차쿠텍</a> 왕이 통치하던 <a href="/wiki/%EC%9E%89%EC%B9%B4_%EC%A0%9C%EA%B5%AD" title="잉카 제국">잉카 제국</a>은 <a href="/wiki/1480%EB%85%84" title="1480년">1480년</a>에 공격을 시작하여 오늘날의 아르헨티나 북서부를 정복해서 <a class="new" href="/w/index.php?title=%EC%BD%94%EC%95%BC%EC%88%98%EC%9C%A0&amp;action=edit&amp;redlink=1" title="코야수유 (없는 문서)">코야수유</a> 지역에 포함시켰다. <a class="mw-redirect" href="/wiki/%EA%B3%BC%EB%9D%BC%EB%8B%88%EC%A1%B1" title="과라니족">과라니족</a>들은 <a class="mw-disambig" href="/wiki/%EC%9C%A0%EC%B9%B4" title="유카">유카</a>, <a href="/wiki/%EA%B3%A0%EA%B5%AC%EB%A7%88" title="고구마">고구마</a>, <a class="new" href="/w/index.php?title=%EC%98%88%EB%A5%B4%EB%B0%94_%EB%A7%88%ED%85%8C&amp;action=edit&amp;redlink=1" title="예르바 마테 (없는 문서)">예르바 마테</a> <a href="/wiki/%EB%86%8D%EC%97%85" title="농업">농업</a>을 발전시켰다. 중부와 남부 지역(<a href="/wiki/%ED%8C%9C%ED%8C%8C%EC%8A%A4" title="팜파스">팜파스</a>와 파타고니아)는 유목민들이 지배했으며 <a href="/wiki/17%EC%84%B8%EA%B8%B0" title="17세기">17세기</a>에 <a href="/wiki/%EB%A7%88%ED%91%B8%EC%B2%B4" title="마푸체">마푸체</a>인들이 통일시켰다.</p>
<h2><span class="mw-headline" id=".EC.8B.9D.EB.AF.BC_.EC.8B.9C.EB.8C.80">식민 시대</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%95%84%EB%A5%B4%ED%97%A8%ED%8B%B0%EB%82%98%EC%9D%98_%EC%97%AD%EC%82%AC&amp;action=edit&amp;section=2" title="부분 편집: 식민 시대">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<div class="thumb tleft">
<div class="thumbinner" style="width:172px;"><a class="image" href="/wiki/%ED%8C%8C%EC%9D%BC:Jos%C3%A9_de_San_Mart%C3%ADn_(retrato,_c.1828).jpg"><img alt="" class="thumbimage" data-file-height="1573" data-file-width="1199" height="223" src="//upload.wikimedia.org/wikipedia/commons/thumb/d/d1/Jos%C3%A9_de_San_Mart%C3%ADn_%28retrato%2C_c.1828%29.jpg/170px-Jos%C3%A9_de_San_Mart%C3%ADn_%28retrato%2C_c.1828%29.jpg" srcset="//upload.wikimedia.org/wikipedia/commons/thumb/d/d1/Jos%C3%A9_de_San_Mart%C3%ADn_%28retrato%2C_c.1828%29.jpg/255px-Jos%C3%A9_de_San_Mart%C3%ADn_%28retrato%2C_c.1828%29.jpg 1.5x, //upload.wikimedia.org/wikipedia/commons/thumb/d/d1/Jos%C3%A9_de_San_Mart%C3%ADn_%28retrato%2C_c.1828%29.jpg/340px-Jos%C3%A9_de_San_Mart%C3%ADn_%28retrato%2C_c.1828%29.jpg 2x" width="170"/></a>
<div class="thumbcaption">
<div class="magnify"><a class="internal" href="/wiki/%ED%8C%8C%EC%9D%BC:Jos%C3%A9_de_San_Mart%C3%ADn_(retrato,_c.1828).jpg" title="실제 크기로"></a></div>
호세 데 산 마르틴 장군. 아르헨티나와 <a href="/wiki/%EC%B9%A0%EB%A0%88" title="칠레">칠레</a>, <a href="/wiki/%ED%8E%98%EB%A3%A8" title="페루">페루</a>의 해방자</div>
</div>
</div>
<p><a href="/wiki/1516%EB%85%84" title="1516년">1516년</a> 이 땅에 <a href="/wiki/%EC%9C%A0%EB%9F%BD%EC%9D%B8" title="유럽인">유럽인</a>들이 처음으로 들어왔다. <a href="/wiki/%EC%8A%A4%ED%8E%98%EC%9D%B8" title="스페인">스페인</a>은 <a href="/wiki/1580%EB%85%84" title="1580년">1580년</a> <a href="/wiki/%EB%B6%80%EC%97%90%EB%85%B8%EC%8A%A4%EC%95%84%EC%9D%B4%EB%A0%88%EC%8A%A4" title="부에노스아이레스">부에노스아이레스</a>에 영구 <a href="/wiki/%EC%8B%9D%EB%AF%BC%EC%A7%80" title="식민지">식민지</a>를 건설했으며, <a href="/wiki/1776%EB%85%84" title="1776년">1776년</a> <a class="mw-redirect" href="/wiki/%EB%A6%AC%EC%98%A4%EB%8D%B0%EB%9D%BC%ED%94%8C%EB%9D%BC%ED%83%80" title="리오데라플라타">리오데라플라타</a> 부왕령을 설치했다. (이 영향으로 <a href="/wiki/%EC%8A%A4%ED%8E%98%EC%9D%B8%EC%96%B4" title="스페인어">스페인어</a>가 아르헨티나의 <a href="/wiki/%EA%B3%B5%EC%9A%A9%EC%96%B4" title="공용어">공용어</a>로 쓰이고 있고 <a class="mw-redirect" href="/wiki/%EB%A1%9C%EB%A7%88_%EA%B0%80%ED%86%A8%EB%A6%AD" title="로마 가톨릭">로마 가톨릭</a>이 주요 <a href="/wiki/%EC%A2%85%EA%B5%90" title="종교">종교</a>이다.) 이 지역은 주로 스페인 이민자들과 <a class="mw-redirect" href="/wiki/%ED%81%AC%EB%A6%AC%EC%98%A4%EC%9A%94" title="크리오요">크리오요</a>라는 그들의 후손과 토착민과 <a href="/wiki/%EC%95%84%ED%94%84%EB%A6%AC%EC%B9%B4" title="아프리카">아프리카</a> 노예의 후예들의 나라였다. 식민 시대 정착민 중 1/3은 <a class="mw-redirect" href="/wiki/%EB%B6%80%EC%97%90%EB%85%B8%EC%8A%A4_%EC%95%84%EC%9D%B4%EB%A0%88%EC%8A%A4" title="부에노스 아이레스">부에노스 아이레스</a>나 다른 <a href="/wiki/%EB%8F%84%EC%8B%9C" title="도시">도시</a>에 모여 살았으며, 나머지는 <a href="/wiki/%EA%B0%80%EC%9A%B0%EC%B4%88" title="가우초">가우초</a>라 하여 <i>팜파스</i>에서 살았다. 토착민들은 아르헨티나에서 그 밖의 다른 지역에서 살았다. <a href="/wiki/%EC%98%81%EA%B5%AD" title="영국">영국</a>이 <a href="/wiki/1806%EB%85%84" title="1806년">1806</a>~<a href="/wiki/1807%EB%85%84" title="1807년">1807년</a>에 부에노스 아이레스에 두 번 침공했으나 크리요오들은 그들을 물리쳤다.</p>
<h2><span class="mw-headline" id=".EB.8F.85.EB.A6.BD">독립</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%95%84%EB%A5%B4%ED%97%A8%ED%8B%B0%EB%82%98%EC%9D%98_%EC%97%AD%EC%82%AC&amp;action=edit&amp;section=3" title="부분 편집: 독립">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p><a href="/wiki/1810%EB%85%84" title="1810년">1810년</a> <a href="/wiki/5%EC%9B%94_25%EC%9D%BC" title="5월 25일">5월 25일</a> <a class="new" href="/w/index.php?title=%ED%8E%98%EB%A5%B4%EB%94%94%EB%82%9C%EB%8F%84_7%EC%84%B8&amp;action=edit&amp;redlink=1" title="페르디난도 7세 (없는 문서)">페르디난도 7세</a>가 <a class="mw-redirect" href="/wiki/%EB%82%98%ED%8F%B4%EB%A0%88%EC%98%B9" title="나폴레옹">나폴레옹</a>의 침공으로 쫓겨났단 소문이 사실로 드러나면서 부에노스 아이레스 시민들은 <a class="new" href="/w/index.php?title=1%EC%B0%A8_%EC%9D%98%ED%9A%8C&amp;action=edit&amp;redlink=1" title="1차 의회 (없는 문서)">1차 의회</a>(First Government Junta, <a class="new" href="/w/index.php?title=%EC%95%84%EB%A5%B4%ED%97%A8%ED%8B%B0%EB%82%98_5%EC%9B%94_%ED%98%81%EB%AA%85&amp;action=edit&amp;redlink=1" title="아르헨티나 5월 혁명 (없는 문서)">5월 혁명</a>)을 조직했다. 현재의 아르헨티나를 이루게 되는 두 나라가 생겼는데, <a class="new" href="/w/index.php?title=%EB%82%A8%EC%95%84%EB%A9%94%EB%A6%AC%EC%B9%B4_%ED%95%A9%EC%A3%BC%EA%B5%AD&amp;action=edit&amp;redlink=1" title="남아메리카 합주국 (없는 문서)">남아메리카 합주국</a>(1810년)과 <a class="new" href="/w/index.php?title=%EC%9E%90%EC%9C%A0_%EC%97%B0%EB%A7%B9&amp;action=edit&amp;redlink=1" title="자유 연맹 (없는 문서)">자유 연맹</a>(1815년)이었다. 다른 주들은 자치론자와 중앙집권론자들의 견해 차로 통합 국가에 귀속되는데 시간이 걸렸다. <a href="/wiki/%ED%8C%8C%EB%9D%BC%EA%B3%BC%EC%9D%B4" title="파라과이">파라과이</a>는 <a href="/wiki/1811%EB%85%84" title="1811년">1811년</a> 독립을 선언하며 분리했다.</p>

<p><a class="mw-redirect" href="/wiki/%ED%98%B8%EC%84%B8_%EB%8D%B0_%EC%82%B0_%EB%A7%88%EB%A5%B4%ED%8B%B4" title="호세 데 산 마르틴">호세 데 산 마르틴</a>이 <a href="/wiki/1814%EB%85%84" title="1814년">1814년</a>에서 <a href="/wiki/1817%EB%85%84" title="1817년">1817년</a> 사이에 군사 작전을 지휘하면서 점차 독립은 현실이 되었다. 아르헨티나 사람들은 산 마르틴을 국가 독립 영웅으로 추앙하고 있다. 산 마르틴과 그의 군대는 1817년 안데스를 넘어 <a href="/wiki/%EC%B9%A0%EB%A0%88" title="칠레">칠레</a>와 <a href="/wiki/%ED%8E%98%EB%A3%A8" title="페루">페루</a>의 왕당파를 무찔러 독립을 공고하게 다졌다. <a href="/wiki/1816%EB%85%84" title="1816년">1816년</a> <a href="/wiki/7%EC%9B%94_9%EC%9D%BC" title="7월 9일">7월 9일</a> <a class="new" href="/w/index.php?title=%ED%88%AC%EC%BF%A0%EB%A7%8C_%EC%9D%98%ED%9A%8C&amp;action=edit&amp;redlink=1" title="투쿠만 의회 (없는 문서)">투쿠만 의회</a>가 소집되어 <a href="/wiki/%EC%8A%A4%ED%8E%98%EC%9D%B8" title="스페인">스페인</a>에서 공식적으로 독립을 선언했다. <a href="/wiki/1820%EB%85%84" title="1820년">1820년</a> 자유 연맹(Liga Federal)은 남아메리카 통합주 군대와 <a href="/wiki/%EB%B8%8C%EB%9D%BC%EC%A7%88" title="브라질">브라질</a>에서 온 <a href="/wiki/%ED%8F%AC%EB%A5%B4%ED%88%AC%EA%B0%88" title="포르투갈">포르투갈</a> 군대의 공격으로 무너졌으며, 이 주는 남아메리카 통합주로 귀속되었다. <a href="/wiki/1825%EB%85%84" title="1825년">1825년</a> <a href="/wiki/%EB%B3%BC%EB%A6%AC%EB%B9%84%EC%95%84" title="볼리비아">볼리비아</a>가 독립을 선언했고, <a href="/wiki/1826%EB%85%84" title="1826년">1826년</a>에 아르헨티나는 브라질로부터 독립을 시도하는 시스플라티나 주를 적극적으로 지지한 결과 당시 브라질의 속주였던 시스플라티나 주의 독립을 막으려는 브라질과 무력으로 충돌하여 전쟁을 하기에 이르렀다. 이것이 <a href="/wiki/%EC%95%84%EB%A5%B4%ED%97%A8%ED%8B%B0%EB%82%98-%EB%B8%8C%EB%9D%BC%EC%A7%88_%EC%A0%84%EC%9F%81" title="아르헨티나-브라질 전쟁">아르헨티나-브라질 전쟁</a>이었다. <a href="/wiki/1828%EB%85%84" title="1828년">1828년</a>에는 <a href="/wiki/%EC%95%84%EB%A5%B4%ED%97%A8%ED%8B%B0%EB%82%98-%EB%B8%8C%EB%9D%BC%EC%A7%88_%EC%A0%84%EC%9F%81" title="아르헨티나-브라질 전쟁">아르헨티나-브라질 전쟁</a>이 정전을 맺고 그 결과 시스플라티나 주가 <a href="/wiki/%EC%9A%B0%EB%A3%A8%EA%B3%BC%EC%9D%B4" title="우루과이">우루과이</a>라는 이름으로 독립했다. 이 정전으로 부에노스 아이레스 주 총독 <a href="/wiki/%ED%9B%84%EC%95%88_%EB%A7%88%EB%88%84%EC%97%98_%EB%8D%B0_%EB%A1%9C%EC%82%AC%EC%8A%A4" title="후안 마누엘 데 로사스">후안 마누엘 데 로사스</a>가 부상했는데, <a class="mw-redirect" href="/wiki/%EC%97%B0%EB%B0%A9%EC%A3%BC%EC%9D%98%EC%9E%90" title="연방주의자">연방주의자</a>인 그는 공포 정치를 펴고 불안한 연맹을 계속 유지했다.</p>

<p><a href="/wiki/1852%EB%85%84" title="1852년">1852년</a> 로사스 총독이 축출되자 <a class="new" href="/w/index.php?title=%EC%A4%91%EC%95%99%EC%A7%91%EA%B6%8C%EB%8B%B9&amp;action=edit&amp;redlink=1" title="중앙집권당 (없는 문서)">중앙집권당</a>(Unitarios)과 <a class="mw-redirect" href="/wiki/%EC%97%B0%EB%B0%A9%EC%A3%BC%EC%9D%98%EC%9E%90" title="연방주의자">연방주의자</a>(Federales) 사이에 치열한 싸움이 벌어졌는데 이는 결과적으로 미래의 내분을 미리 막게 된다. <a href="/wiki/1853%EB%85%84" title="1853년">1853년</a> <a href="/wiki/%ED%97%8C%EB%B2%95" title="헌법">헌법</a>이 통과된다. 이것은 법학자 <a class="new" href="/w/index.php?title=%ED%9B%84%EC%95%88_%EB%B0%94%EC%9A%B0%ED%8B%B0%EC%8A%A4%ED%83%80_%EC%95%8C%EB%B2%A0%EB%A5%B4%EB%94%94&amp;action=edit&amp;redlink=1" title="후안 바우티스타 알베르디 (없는 문서)">후안 바우티스타 알베르디</a>가 초안을 썼으며, <a class="mw-redirect" href="/wiki/%ED%94%84%EB%9E%80%EC%B2%B4%EC%8A%A4%EC%BD%94%ED%9A%8C" title="프란체스코회">프란체스코회</a> 수사 <a class="new" href="/w/index.php?title=%EB%A7%88%EB%A9%94%EB%A5%B4%ED%86%A0_%EC%97%90%EB%A5%B4%ED%82%A4%EC%9A%B0&amp;action=edit&amp;redlink=1" title="마메르토 에르키우 (없는 문서)">마메르토 에르키우</a>가 설교를 통해 헌법과 국가 통합을 변호했으며, 처음 몇 년간 어려운 시기를 맞았으나, <a href="/wiki/1865%EB%85%84" title="1865년">1865년</a> 전쟁으로 국가 통합이 강화되는 결과를 낳았다. 대신 그 전쟁으로 <a href="/wiki/%ED%8C%8C%EB%9D%BC%EA%B3%BC%EC%9D%B4" title="파라과이">파라과이</a>를 멸망 위기까지 몰고 갔다.</p>
<h2><span class="mw-headline" id=".ED.98.84.EB.8C.80_.EC.95.84.EB.A5.B4.ED.97.A8.ED.8B.B0.EB.82.98.EC.9D.98_.EB.93.B1.EC.9E.A5">현대 아르헨티나의 등장</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%95%84%EB%A5%B4%ED%97%A8%ED%8B%B0%EB%82%98%EC%9D%98_%EC%97%AD%EC%82%AC&amp;action=edit&amp;section=4" title="부분 편집: 현대 아르헨티나의 등장">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p><a href="/wiki/1870%EB%85%84%EB%8C%80" title="1870년대">1870년대</a>부터 해외 투자와 <a class="mw-redirect" href="/wiki/%EC%9D%B4%EB%AF%BC" title="이민">이민</a>이 밀려 들어오면서 <a href="/wiki/%EB%86%8D%EC%97%85" title="농업">농업</a>은 근대적으로 발전했고 아르헨티나 사회와 경제 새롭게 재편되어 국가 통합이 강화되었다. 그러나 1870년대 "<a class="new" href="/w/index.php?title=%EC%82%AC%EB%A7%89%EC%9D%98_%EC%A0%95%EB%B3%B5&amp;action=edit&amp;redlink=1" title="사막의 정복 (없는 문서)">사막의 정복</a>" 남부 팜파스와 파타고니아의 토착 부족들을 탄압하여 1,300여명의 토착민이 죽었다.</p>
<p><a href="/wiki/1880%EB%85%84" title="1880년">1880년</a>과 <a href="/wiki/1929%EB%85%84" title="1929년">1929년</a> 사이 아르헨티나는 경제 성장을 계속하여 당시 세계적으로 부유한 10개국 중 한 곳이었으며 농업 수출 경제로 이익을 보았다. 이민이 들고 사망률이 떨어지면서 아르헨티나 인구는 5배로 늘었고, 경제는 15배로 확대되었다.</p>
<p><a class="new" href="/w/index.php?title=%EA%B5%AD%EB%AF%BC%EC%9E%90%EC%B9%98%EB%8B%B9&amp;action=edit&amp;redlink=1" title="국민자치당 (없는 문서)">국민자치당</a>은 비민주적으로 아르헨티나 정치를 지배했으나 <a href="/wiki/1912%EB%85%84" title="1912년">1912년</a> <a class="new" href="/w/index.php?title=%EB%A1%9C%EC%BC%80_%EC%82%AC%EC%97%94%EC%8A%A4_%ED%8E%98%EB%83%90&amp;action=edit&amp;redlink=1" title="로케 사엔스 페냐 (없는 문서)">로케 사엔스 페냐</a> 대통령이 남성 <a class="mw-redirect" href="/wiki/%EB%B3%B4%ED%86%B5_%EC%84%A0%EA%B1%B0" title="보통 선거">보통 선거</a>와 <a href="/wiki/%EB%B9%84%EB%B0%80_%EC%84%A0%EA%B1%B0" title="비밀 선거">비밀 선거</a>를 법제화했다. 그리하여 그들의 전통적인 정적인 중앙집권론자 <a class="new" href="/w/index.php?title=%EA%B8%89%EC%A7%84_%EC%8B%9C%EB%AF%BC_%EC%97%B0%EB%A7%B9&amp;action=edit&amp;redlink=1" title="급진 시민 연맹 (없는 문서)">급진 시민 연맹</a>이 <a href="/wiki/1916%EB%85%84" title="1916년">1916년</a> 아르헨티나 첫 자유 선거에서 승리했다. <a class="new" href="/w/index.php?title=%EC%9D%B4%ED%8F%B4%EB%A6%AC%ED%86%A0_%EC%9D%B4%EB%A6%AC%EA%B3%A0%EC%98%8C&amp;action=edit&amp;redlink=1" title="이폴리토 이리고옌 (없는 문서)">이폴리토 이리고옌</a> 대통령은 사회 경제 개혁을 추진하고 가족농과 소상인 지원을 확대했다. 그러나 <a href="/wiki/%EC%84%B8%EA%B3%84" title="세계">세계</a> <a href="/wiki/%EB%8C%80%EA%B3%B5%ED%99%A9" title="대공황">대공황</a>의 여파로 <a href="/wiki/1930%EB%85%84" title="1930년">1930년</a> 군부는 그를 축출했다. 그리하여 십 여년간 보수주의자의 지배가 시작되었으며 경제 정책을 보호주의로 선회했다. 아르헨티나는 <a class="mw-redirect" href="/wiki/%EC%A0%9C1%EC%B0%A8_%EC%84%B8%EA%B3%84%EB%8C%80%EC%A0%84" title="제1차 세계대전">제1차 세계대전</a> 때 중립을 지켰고, <a class="mw-redirect" href="/wiki/%EC%A0%9C2%EC%B0%A8_%EC%84%B8%EA%B3%84%EB%8C%80%EC%A0%84" title="제2차 세계대전">제2차 세계대전</a> 때도 대체로 그런 태도를 보였으며, 연합국의 식량 공급처가 되었다.</p>
<h2><span class="mw-headline" id=".ED.8E.98.EB.A1.A0.EC.97.90.EC.84.9C_.EA.B5.B0.EC.82.AC.EB.8F.85.EC.9E.AC.EA.B9.8C.EC.A7.80">페론에서 군사독재까지</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%95%84%EB%A5%B4%ED%97%A8%ED%8B%B0%EB%82%98%EC%9D%98_%EC%97%AD%EC%82%AC&amp;action=edit&amp;section=5" title="부분 편집: 페론에서 군사독재까지">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<h3><span class="mw-headline" id=".ED.8E.98.EB.A1.A0.EC.A0.95.EB.B6.80">페론정부</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%95%84%EB%A5%B4%ED%97%A8%ED%8B%B0%EB%82%98%EC%9D%98_%EC%97%AD%EC%82%AC&amp;action=edit&amp;section=6" title="부분 편집: 페론정부">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p><a href="/wiki/1946%EB%85%84" title="1946년">1946년</a> <a class="mw-redirect" href="/wiki/%ED%9B%84%EC%95%88_%ED%8E%98%EB%A1%A0" title="후안 페론">후안 페론</a>이 대통령이 되면서 정치적 변화가 일어났다 그는 <a href="/wiki/%EC%82%AC%ED%9A%8C" title="사회">사회</a> 및 <a href="/wiki/%EA%B5%90%EC%9C%A1" title="교육">교육</a> 정책을 폈으며 노동자 계급의 힘을 강화하고 노조 노동자수를 늘렸다. 그의 아내 <a href="/wiki/%EC%97%90%EB%B0%94_%ED%8E%98%EB%A1%A0" title="에바 페론">에바 페론</a>("에비타"로 잘 알려졌다.)은 페론의 두 번째 임기까지 영부인으로 중요한 역할을 했다. 그녀는 페론의 성공으로 노동 계급 사이에 거물이었다. <a href="/wiki/1947%EB%85%84" title="1947년">1947년</a> 그녀는 <a class="new" href="/w/index.php?title=%EC%97%90%EB%B0%94_%ED%8E%98%EB%A1%A0_%EC%9E%AC%EB%8B%A8&amp;action=edit&amp;redlink=1" title="에바 페론 재단 (없는 문서)">에바 페론 재단</a>을 설립하여 사회 복지를 실시했다.</p>

<p>이것은 아르헨티나 최초로 정부가 빈민 구제에 관심을 두게 된 사건이었으며, 기존의 과두정치와는 양립할 수 없는 것이었다. 에비타는 무뚝뚝한 페론과 그의 지지자 사이의 "<a href="/wiki/%EC%82%AC%EB%9E%91" title="사랑">사랑</a>의 다리"로 여겨졌다. 그녀는 여성 참정권을 얻어냈으며, <a class="new" href="/w/index.php?title=%ED%8E%98%EB%A1%A0%EC%A3%BC%EC%9D%98_%EC%97%AC%EC%84%B1%EB%8B%B9&amp;action=edit&amp;redlink=1" title="페론주의 여성당 (없는 문서)">페론주의 여성당</a>을 창립하기도 했다. 페론의 두 번째 임기까지 그의 경제학자들은 <a href="/wiki/%EC%82%B0%EC%97%85" title="산업">산업</a>과 도시 발전을 장려했다. <a href="/wiki/1952%EB%85%84" title="1952년">1952년</a> 에비타가 33살의 나이로 죽자 페론 행정부는 점차 아르헨티나의 전통교회인 <a href="/wiki/%EA%B0%80%ED%86%A8%EB%A6%AD" title="가톨릭">가톨릭</a>과 자신의 정치운동 사이의 갈등으로 곤란에 처했다. <a href="/wiki/1955%EB%85%84" title="1955년">1955년</a> <a href="/wiki/%EC%BF%A0%EB%8D%B0%ED%83%80" title="쿠데타">쿠데타</a>가 일어나 <a href="/wiki/%EC%8A%A4%ED%8E%98%EC%9D%B8" title="스페인">스페인</a> 으로 망명했다.</p>

<h3><span class="mw-headline" id=".EC.9D.BC.EB.A6.AC.EC.95.84">일리아</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%95%84%EB%A5%B4%ED%97%A8%ED%8B%B0%EB%82%98%EC%9D%98_%EC%97%AD%EC%82%AC&amp;action=edit&amp;section=8" title="부분 편집: 일리아">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p><a href="/wiki/1963%EB%85%84" title="1963년">1963년</a> <a class="new" href="/w/index.php?title=%EC%95%84%EB%A5%B4%ED%88%AC%EB%A1%9C_%EC%9D%BC%EB%A6%AC%EC%95%84&amp;action=edit&amp;redlink=1" title="아르투로 일리아 (없는 문서)">아르투로 일리아</a>가 대통령에 선출되어 확장정책을 폈다. 경제적 번영에도 불구하고 그가 페론주의자들을 정계에 끌어들이려 하자 군부는 <a href="/wiki/1966%EB%85%84" title="1966년">1966년</a> 쿠데타로 대통령을 축출했다.</p>
<h3><span class="mw-headline" id=".ED.8E.98.EB.A1.A0.EC.9D.98_.EC.9E.AC.EC.A7.91.EA.B6.8C.EA.B3.BC_.EA.B5.B0.EC.82.AC.EB.8F.85.EC.9E.AC">페론의 재집권과 군사독재</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%95%84%EB%A5%B4%ED%97%A8%ED%8B%B0%EB%82%98%EC%9D%98_%EC%97%AD%EC%82%AC&amp;action=edit&amp;section=9" title="부분 편집: 페론의 재집권과 군사독재">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>억압적인 군부독재정권은 국내 개발과 공공 사업에 대한 투자를 장려했다. <a href="/wiki/1975%EB%85%84" title="1975년">1975년</a>에 경제는 매우 발전했고 빈곤은 7%나 떨어져 낮은 수준이었다. 부분적으로 군부 정권의 억압 때문에, 정치적 소요가 확됐었으며, 망명지에서 페론은 능란하게 학생과 지식인으로 구성된 시위대를 동원해서 결국 군사 정권은 <a href="/wiki/1973%EB%85%84" title="1973년">1973년</a> 자유 선거를 요구하게 되었으며, 페론은 스페인에서 다시 돌아와 대통령이 되었다. 그러나 페론은 <a href="/wiki/1974%EB%85%84" title="1974년">1974년</a> 7월에 죽고, 그의 셋째 부인이자 부통령 <a href="/wiki/%EC%9D%B4%EC%82%AC%EB%B2%A8_%ED%8E%98%EB%A1%A0" title="이사벨 페론">이사벨 페론</a>은 남편의 뒤를 승계했다. 이사벨 페론은 페론주의자 파벌 사이의 타협으로 선택된 것이었다. 좌우익 극단주의자 사이의 폭력 사태와 재정위기가 일어나자 <a href="/wiki/1976%EB%85%84" title="1976년">1976년</a> <a href="/wiki/3%EC%9B%94_24%EC%9D%BC" title="3월 24일">3월 24일</a> <a href="/wiki/1976%EB%85%84_%EC%95%84%EB%A5%B4%ED%97%A8%ED%8B%B0%EB%82%98_%EC%BF%A0%EB%8D%B0%ED%83%80" title="1976년 아르헨티나 쿠데타">군사 쿠데타</a>로 이사벨도 축출됐다.</p>

군사정부(PRN) 통치 기간 동안 아르헨티나의 빈곤율(Porcentaje de la población)이 폭증했다. - 아르헨티나 국립통계센서스연구소(Indec)</div>

<p>군사독재정권은 스스로를 <a class="mw-redirect" href="/wiki/%EA%B5%AD%EA%B0%80_%EC%9E%AC%EC%A1%B0%EC%A7%81_%EA%B3%BC%EC%A0%95" title="국가 재조직 과정">국가 재조직 과정</a>이라고 불렀는데, 즉시 잔인하고 불법적인 수단을 동원하여 반대파와 좌파를 탄압(일명 <a href="/wiki/%EB%8D%94%EB%9F%AC%EC%9A%B4_%EC%A0%84%EC%9F%81" title="더러운 전쟁">더러운 전쟁</a>)하는 독재를 하였는데, 군사독재정권 당시 아르헨티나 비밀정보국은 칠레 비밀경찰 및 미국 <a href="/wiki/%EC%A4%91%EC%95%99%EC%A0%95%EB%B3%B4%EA%B5%AD" title="중앙정보국">중앙정보국</a>과 협조하여 <a href="/wiki/%EC%BD%98%EB%8F%84%EB%A5%B4_%EC%9E%91%EC%A0%84" title="콘도르 작전">콘도르 작전</a>을 수행해 수천명의 반체제 "행방불명자"가 생겼다. 더러운 전쟁에 낀 많은 군사 독재자들의 <a href="/wiki/%EB%AF%B8%EA%B5%AD" title="미국">미국</a>의 지원을 받은 자들이었으며, 그들 중에는 아르헨티나 독재자 <a class="mw-redirect" href="/wiki/%EB%A1%9C%EB%B2%A0%EB%A5%B4%ED%86%A0_%EB%B9%84%EC%98%AC%EB%9D%BC" title="로베르토 비올라">로베르토 비올라</a>와 <a href="/wiki/%EB%A0%88%EC%98%A4%ED%8F%B4%EB%8F%84_%EA%B0%88%ED%8B%B0%EC%97%90%EB%A6%AC" title="레오폴도 갈티에리">레오폴도 갈티에리</a>도 있었다. 군사독재정권은 처음에 얼마간 안정적이었고 여러 중요한 공공 사업을 벌였으나, 잦은 임금 동결과 <a href="/wiki/%EA%B8%88%EC%9C%B5" title="금융">금융</a> 규제 철폐로 생활 수준이 급격히 떨어지고 외채가 늘어났다.</p>
<p>전례없는 부패와 더불어 산업 위축, 페소화 붕괴, 실질 이자율 하락이 겹치면서, 민중들은 군사독재정권의 <a href="/wiki/%EC%9D%B8%EA%B6%8C" title="인권">인권</a> 유린에 맞섰고 결국 <a href="/wiki/1982%EB%85%84" title="1982년">1982년</a> 군사 정부가 <a href="/wiki/%EC%98%81%EA%B5%AD" title="영국">영국</a>과 <a href="/wiki/%ED%8F%AC%ED%81%B4%EB%9E%9C%EB%93%9C_%EC%A0%84%EC%9F%81" title="포클랜드 전쟁">포클랜드 전쟁</a>을 벌이다 패하자 신뢰를 잃고 <a href="/wiki/1983%EB%85%84" title="1983년">1983년</a> 자유 선거를 치르게 된다.</p>
<h2><span class="mw-headline" id=".EB.AF.BC.EC.A3.BC.EC.A3.BC.EC.9D.98">민주주의</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%95%84%EB%A5%B4%ED%97%A8%ED%8B%B0%EB%82%98%EC%9D%98_%EC%97%AD%EC%82%AC&amp;action=edit&amp;section=10" title="부분 편집: 민주주의">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<h3><span class="mw-headline" id=".EC.95.8C.ED.8F.B0.EC.8B.A0">알폰신</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%95%84%EB%A5%B4%ED%97%A8%ED%8B%B0%EB%82%98%EC%9D%98_%EC%97%AD%EC%82%AC&amp;action=edit&amp;section=11" title="부분 편집: 알폰신">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p><a href="/wiki/%EB%9D%BC%EC%9A%B8_%EC%95%8C%ED%8F%B0%EC%8B%A0" title="라울 알폰신">라울 알폰신</a> 정부는 "행방불명자"를 찾는 작업에 착수했으며, <a href="/wiki/%EA%B5%B0%EB%8C%80" title="군대">군대</a>에 대한 민간의 통제 및 민주적인 통합 조직을 세웠다. 군사독재정권 출신자들은 고발되어 <a href="/wiki/%EC%A2%85%EC%8B%A0%ED%98%95" title="종신형">종신형</a>을 선고받는 처벌을 받았다. 그러나 과거 정권이 떠안은 외채는 그대로 남아 <a class="mw-redirect" href="/wiki/%EA%B5%AD%EC%A0%9C%ED%86%B5%ED%99%94%EA%B8%B0%EA%B8%88" title="국제통화기금">국제통화기금</a>과 개인 채권자들은 아르헨티나 경제에 몇몇 조건을 내렸으며, 국가는 우선 공공 사업과 국내 신용을 희생하고 외채를 갚아야 했다. 알폰신은 경제 악화를 해결하지 못하여 대중의 지지를 잃었다. 뒤이어 갑자기 닥친 <a href="/wiki/1989%EB%85%84" title="1989년">1989년</a> 통화 위기로 물가는 15배나 껑충 뛰었으며, 그는 5개월 일찍 대통령직에서 사임했다.</p>
<h3><span class="mw-headline" id=".EB.A9.94.EB.84.B4">메넴</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%95%84%EB%A5%B4%ED%97%A8%ED%8B%B0%EB%82%98%EC%9D%98_%EC%97%AD%EC%82%AC&amp;action=edit&amp;section=12" title="부분 편집: 메넴">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>새로 대통령에 선출된 <a href="/wiki/%EC%B9%B4%EB%A5%BC%EB%A1%9C%EC%8A%A4_%EB%A9%94%EB%84%B4" title="카를로스 메넴">카를로스 메넴</a>은 사유화 정책을 펴기 시작했으며, <a href="/wiki/1990%EB%85%84" title="1990년">1990년</a>에 두 번째 <a class="mw-redirect" href="/wiki/%ED%95%98%EC%9D%B4%ED%8D%BC%EC%9D%B8%ED%94%8C%EB%A0%88%EC%9D%B4%EC%85%98" title="하이퍼인플레이션">하이퍼인플레이션</a> 이후 경제학자 <a class="new" href="/w/index.php?title=%EB%8F%84%EB%B0%8D%EA%B3%A0_%EC%B9%B4%EB%B0%94%EC%9A%94&amp;action=edit&amp;redlink=1" title="도밍고 카바요 (없는 문서)">도밍고 카바요</a>를 기용하여 <a href="/wiki/1991%EB%85%84" title="1991년">1991년</a> 페소화 환율을 달러화에 고정시켰으며, 시장 기반의 정책을 추진해 사유화 정책에 박차를 가하는 동시에 무역 장벽과 기업 규제를 철폐했다. 이런 개혁 덕분에 <a href="/wiki/1990%EB%85%84%EB%8C%80" title="1990년대">1990년대</a> 거의 내내 물가 안정을 이루면서 투자와 성장 확대를 이끌었다. 그러나 페소화는 시장에 달러화가 밀려들어와 제 가치를 유지하지 못했으며, 외채가 다시 늘어나는 결과를 낳았다. <a href="/wiki/1998%EB%85%84" title="1998년">1998년</a> 일련의 국제 금융 위기가 닥치고 고정 페소화가 지나치게 고평가되면서 점차 경제 위기로 빠져들었다. 1990년대에 팽배하던 경제 안정에 대한 기대는 갑자기 무너졌으며, 그의 임기말인 <a href="/wiki/1999%EB%85%84" title="1999년">1999년</a> 누적된 문제들과 부패 보고로 메넴은 인기를 잃었다.</p>

<h3><span class="mw-headline" id=".EB.A3.A8.EC.95.84">루아</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%95%84%EB%A5%B4%ED%97%A8%ED%8B%B0%EB%82%98%EC%9D%98_%EC%97%AD%EC%82%AC&amp;action=edit&amp;section=13" title="부분 편집: 루아">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>후임 대통령 <a href="/wiki/%ED%8E%98%EB%A5%B4%EB%82%9C%EB%8F%84_%EB%8D%B0_%EB%9D%BC_%EB%A3%A8%EC%95%84" title="페르난도 데 라 루아">페르난도 데 라 루아</a>는 카를로스 메넴정권의 만성적인 재정 적자와 악화된 <a href="/wiki/%EC%88%98%EC%B6%9C" title="수출">수출</a> 경쟁력을 물려받았다. 정치 연합은 균열을 보이기 시작했고, 카바요가 다시 경제 장관으로 돌아오자 이는 금융 파생상품 시장에 의한 위기로 해석되었다. 이에 맞불로 카바요는 결국 자본 이탈 흐름을 막고 긴급한 채무 위기를 막을 수밖에(은행 계좌를 동결하며 정점을 이루었다) 없었다. 그러나 대중의 불만은 계속되었고 <a href="/wiki/2001%EB%85%84" title="2001년">2001년</a> <a href="/wiki/12%EC%9B%94_20%EC%9D%BC" title="12월 20일">12월 20일</a> 아르헨티나는 <a href="/wiki/1890%EB%85%84" title="1890년">1890년</a> 대불황 이래 최악의 사회 및 경제 위기에 봉착했다. 거리에선 폭력 <a href="/wiki/%EC%8B%9C%EC%9C%84" title="시위">시위</a>가 일어났고 <a href="/wiki/%EA%B2%BD%EC%B0%B0" title="경찰">경찰</a>과 충돌하여 사망자가 발생했다. 시위 군중 사이로 점차 "모두 꺼져라"라고 외쳤으며, 결국 데 라 루아 대통령도 사임하게 된다.</p>
<h3><span class="mw-headline" id=".EB.91.90.EC.95.8C.EB.8D.B0">두알데</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%95%84%EB%A5%B4%ED%97%A8%ED%8B%B0%EB%82%98%EC%9D%98_%EC%97%AD%EC%82%AC&amp;action=edit&amp;section=14" title="부분 편집: 두알데">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>아르헨티나는 2주 동안 세 명의 대통령을 갈아치웠으며, <a href="/wiki/2002%EB%85%84" title="2002년">2002년</a> <a href="/wiki/1%EC%9B%94_2%EC%9D%BC" title="1월 2일">1월 2일</a> 입법 의회가 <a href="/wiki/%EC%97%90%EB%91%90%EC%95%84%EB%A5%B4%EB%8F%84_%EB%91%90%EC%95%8C%EB%8D%B0" title="에두아르도 두알데">에두아르도 두알데</a>를 임시 대통령으로 임명하면서 극에 달했다. 아르헨티나는 해외 채무에 대해 <a class="mw-redirect" href="/wiki/%EC%B1%84%EB%AC%B4_%EB%B6%88%EC%9D%B4%ED%96%89" title="채무 불이행">채무 불이행</a>(default)를 선언했으며, 11년간 지속된 페소화의 달러화 연동은 폐지되었고, 이로써 페소화가 크게 절하되고 <a href="/wiki/%EC%9D%B8%ED%94%8C%EB%A0%88%EC%9D%B4%EC%85%98" title="인플레이션">인플레이션</a>을 막았다. 중도 좌파 성향의 페론주의자 두알데는 2002년 말까지 실업률이 25%에 이르고 실질 임금이 지난 60년간 최저치를 보이면서 금융 및 사회경제 위기에 봉착했다. 이 위기로 대중은 정치인과 제도를 더욱 불신하게 되었다. 이듬해 시위가 득세했으며, 경제는 2002년 말부터 안정되기 시작했으며 그해 12월 <a href="/wiki/%EC%9D%80%ED%96%89" title="은행">은행</a> 인출 제한이 해제되었다.</p>
<h3><span class="mw-headline" id=".ED.82.A4.EB.A5.B4.EC.B8.A0.EB.84.A4.EB.A5.B4">키르츠네르</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%95%84%EB%A5%B4%ED%97%A8%ED%8B%B0%EB%82%98%EC%9D%98_%EC%97%AD%EC%82%AC&amp;action=edit&amp;section=15" title="부분 편집: 키르츠네르">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<div class="thumb tleft">
<div class="thumbinner" style="width:172px;"><a class="image" href="/wiki/%ED%8C%8C%EC%9D%BC:Cristina_con_baston_y_banda.jpg"><img alt="" class="thumbimage" data-file-height="2200" data-file-width="1502" height="249" src="//upload.wikimedia.org/wikipedia/commons/thumb/3/39/Cristina_con_baston_y_banda.jpg/170px-Cristina_con_baston_y_banda.jpg" srcset="//upload.wikimedia.org/wikipedia/commons/thumb/3/39/Cristina_con_baston_y_banda.jpg/255px-Cristina_con_baston_y_banda.jpg 1.5x, //upload.wikimedia.org/wikipedia/commons/thumb/3/39/Cristina_con_baston_y_banda.jpg/340px-Cristina_con_baston_y_banda.jpg 2x" width="170"/></a>

현 대통령 <a href="/wiki/%ED%81%AC%EB%A6%AC%EC%8A%A4%ED%8B%B0%EB%82%98_%ED%8E%98%EB%A5%B4%EB%82%9C%EB%8D%B0%EC%8A%A4" title="크리스티나 페르난데스">크리스티나 페르난데스 데 키르츠네르</a>. <a href="/wiki/2007%EB%85%84" title="2007년">2007년</a> 12월에 취임했다.</div>

<p><a href="/wiki/2003%EB%85%84" title="2003년">2003년</a> 5월 <a href="/wiki/%EC%82%AC%ED%9A%8C%EB%AF%BC%EC%A3%BC%EC%A3%BC%EC%9D%98" title="사회민주주의">사회민주주의</a> <a href="/wiki/%ED%8E%98%EB%A1%A0%EC%A3%BC%EC%9D%98" title="페론주의">페론주의</a>자 <a class="mw-redirect" href="/wiki/%EB%84%A4%EC%8A%A4%ED%86%A0%EB%A5%B4_%ED%82%A4%EB%A5%B4%EC%B8%A0%EB%84%A4%EB%A5%B4" title="네스토르 키르츠네르">네스토르 키르츠네르</a> 산타크루스 주지사가 대통령에 선출됐다. 환율의 평가절하 정책 덕택에 정부는 산업 부흥했으며, <a class="new" href="/w/index.php?title=%EC%88%98%EC%9E%85%EB%8C%80%EC%B2%B4&amp;action=edit&amp;redlink=1" title="수입대체 (없는 문서)">수입대체</a>(import substitution)와 수출 증가에 기반을 둔 새 정책을 실행했고, 재정과 무역 면에서 안정적으로 흑자를 보였다. 키르츠네르 시기에 아르헨티나는 대부분의 채권에 높은 할인율(약 66%)로 채무 재조정을 했다. 또한 2005년 <a class="mw-redirect" href="/wiki/%EA%B5%AD%EC%A0%9C%ED%86%B5%ED%99%94%EA%B8%B0%EA%B8%88" title="국제통화기금">국제통화기금</a>에 대한 모든 외채를 갚았다. 공익 사업을 재조정하고 90년대 민영화된 기업 중 일부을 국유화했다. 그의 경제부 장관인 <a class="new" href="/w/index.php?title=%EB%A1%9C%EB%B2%A0%EB%A5%B4%ED%86%A0_%EB%9D%BC%EB%B0%94%EB%83%90&amp;action=edit&amp;redlink=1" title="로베르토 라바냐 (없는 문서)">로베르토 라바냐</a>는 강력한 소득 증진 정책과 공공 사업 투자를 추진했다. 그리하여 아르헨티나는 경제위기에서 벗어나 경제 성장을 계속 하게 되었다.</p>
<p>네스토르 키르츠네르가 2007년 대선에 출마하지 않는 대신, 그의 부인인 <a href="/wiki/%ED%81%AC%EB%A6%AC%EC%8A%A4%ED%8B%B0%EB%82%98_%ED%8E%98%EB%A5%B4%EB%82%9C%EB%8D%B0%EC%8A%A4" title="크리스티나 페르난데스">크리스티나 페르난데스</a> 상원의원이 출마하여 당선됐다. 페르난데스는 아르헨티나 역사상 직접 선거로 뽑힌 첫 번째 <a href="/wiki/%EC%97%AC%EC%84%B1" title="여성">여성</a> 대통령이 됐다. 여당인 <a href="/wiki/%EC%8A%B9%EB%A6%AC%EB%A5%BC_%EC%9C%84%ED%95%9C_%EC%A0%84%EC%84%A0" title="승리를 위한 전선">승리를 위한 전선</a>은 대통령 선거와 함께 치뤄진 총선과 지방선거에 승리해서 의회를 다수를 차지했다. 페르난데스 대통령은 <a href="/wiki/%EB%86%8D%EC%97%85" title="농업">농업</a> 수출세 인상 계획을 펴려고 하자, <a href="/wiki/2008%EB%85%84" title="2008년">2008년</a> 3월부터 7월까지 농업인들이 대규모 시위를 벌이며 직장폐쇄로 반발했다. <a href="/wiki/2008%EB%85%84" title="2008년">2008년</a> <a href="/wiki/7%EC%9B%94_16%EC%9D%BC" title="7월 16일">7월 16일</a> 부통령 <a href="/wiki/%ED%9B%8C%EB%A6%AC%EC%98%A4_%EC%BD%94%EB%B3%B4%EC%8A%A4" title="훌리오 코보스">훌리오 코보스</a>는 의회의 가부 동수(同數) 결과에서 상원의장으로서 여당에 반대표를 던져서 대통령의 정책을 좌절시켰다. 2008년 10월 세계 금융 위기가 닥치자 크리스티나 페르난데스 대통령은 남편의 정책대로 국가가 직접 경제에 개입했다. 2009년 6월 총선에서 집권연합 <a href="/wiki/%EC%8A%B9%EB%A6%AC%EB%A5%BC_%EC%9C%84%ED%95%9C_%EC%A0%84%EC%84%A0" title="승리를 위한 전선">승리를 위한 전선</a>이 패해 과반수를 잃었다. 2009년 10월 아르헨티나 상하원은 야당의 반대에도 불구하고 1개 업체가 소유할 수 있는 방송 면허의 수를 제한하는 것을 주요 내용으로 하는 미디어 개편법안을 통과시켰다. 이 미디어 개편법안에 대해 정부와 대립각을 세워온 <a href="/wiki/%ED%81%B4%EB%9D%BC%EB%A6%B0" title="클라린">클라린</a>을 겨냥했다는 평가도 있다.<sup class="reference" id="cite_ref-1"><a href="#cite_note-1">[1]</a></sup></p>
