<h2><span class="mw-headline" id=".EC.95.84.EB.A6.AC.EC.95.84.EC.9D.B8.EB.93.A4.EC.9D.98_.EA.B8.B0.EC.9B.90">아리아인들의 기원</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%95%84%EB%A6%AC%EC%95%84%EC%9D%B8&amp;action=edit&amp;section=1" title="부분 편집: 아리아인들의 기원">원본 편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>아리아인들의 기원에 관하여 많은 학자들이 연구한 바에 의하면 <a href="/wiki/%EB%A9%94%EC%86%8C%ED%8F%AC%ED%83%80%EB%AF%B8%EC%95%84" title="메소포타미아">메소포타미아</a>의 여러 도시에 아리안들이 건국한 시점으로 미루어 기원전 2400년 - 2000년전으로 추정하고 있으나 쓰나미 처럼 카스피해, 발칸반도로 남하 해 온 그리스 북부종족인 '빈카문화권'의 사람들에 의한 남진,동남진하였다고 주장한 '쿠르간 가설'에 의하여 기원전 40세기 이전으로 보는 학자들도 있다.</p>
<p>이미 17세기말 언어학자들이 연구한 결과에 의하면 <a href="/wiki/%EB%9D%BC%ED%8B%B4%EC%96%B4" title="라틴어">라틴어</a>, <a href="/wiki/%EA%B7%B8%EB%A6%AC%EC%8A%A4%EC%96%B4" title="그리스어">그리스어</a>, <a href="/wiki/%EA%B3%A0%EB%8C%80_%ED%8E%98%EB%A5%B4%EC%8B%9C%EC%95%84%EC%96%B4" title="고대 페르시아어">고대 페르시아어</a>와 현대 유럽 언어들 사이의 유사성에 주목했는데 이 언어들의 공통된 조상이 <a href="/wiki/%EC%8A%A4%ED%82%A4%ED%83%80%EC%9D%B4%EC%A1%B1" title="스키타이족">스키타이족</a>의 언어라고 생각했다.</p>
<p>인도의 <a href="/wiki/%EC%82%B0%EC%8A%A4%ED%81%AC%EB%A6%AC%ED%8A%B8%EC%96%B4" title="산스크리트어">산스크리트어</a>가 라틴어와 <a href="/wiki/%EA%B3%A0%EB%8C%80_%EA%B7%B8%EB%A6%AC%EC%8A%A4%EC%96%B4" title="고대 그리스어">고대 그리스어</a>와 밀접한 관계가 있다는 것을 발견한 <a href="/wiki/18%EC%84%B8%EA%B8%B0" title="18세기">18세기</a>의 <a href="/wiki/%EC%98%81%EA%B5%AD%EC%9D%B8" title="영국인">영국인</a> <a href="/wiki/%EC%9C%8C%EB%A6%AC%EC%97%84_%EC%A1%B4%EC%8A%A4_(%EB%AC%B8%ED%97%8C%ED%95%99%EC%9E%90)" title="윌리엄 존스 (문헌학자)">윌리엄 존스</a>에 뒤이어 '인도유럽어족' 이라는 용어를 영국의 토마스 영이 처음으로 사용했으며 그는 "하나의 발상지에서 나온 단일민족이 이웃민족들을 잇달아 침략하여 자기네 언어를 전파했다."라는 학설을 발표했다.</p>
<p>인도유럽어족에게는 하위 분류로서 10개의 언어의 파가 있는데 <a href="/wiki/%EC%95%84%EB%82%98%ED%86%A8%EB%A6%AC%EC%95%84%EC%96%B4%ED%8C%8C" title="아나톨리아어파">아나톨리아어파</a>, <a href="/wiki/%EB%B0%9C%ED%8A%B8%EC%96%B4%ED%8C%8C" title="발트어파">발트어파</a>, <a href="/wiki/%EC%8A%AC%EB%9D%BC%EB%B8%8C%EC%96%B4%ED%8C%8C" title="슬라브어파">슬라브어파</a>, <a href="/wiki/%EC%9D%B8%EB%8F%84%EC%9D%B4%EB%9E%80%EC%96%B4%ED%8C%8C" title="인도이란어파">인도이란어파</a>, <a href="/wiki/%EC%95%84%EB%A5%B4%EB%A9%94%EB%8B%88%EC%95%84%EC%96%B4" title="아르메니아어">아르메니아어파</a>, <a href="/wiki/%EC%95%8C%EB%B0%94%EB%8B%88%EC%95%84%EC%96%B4" title="알바니아어">알바니아어파</a>, <a href="/wiki/%EA%B2%8C%EB%A5%B4%EB%A7%8C%EC%96%B4%ED%8C%8C" title="게르만어파">게르만어파</a>, <a href="/wiki/%EA%B7%B8%EB%A6%AC%EC%8A%A4%EC%96%B4" title="그리스어">그리스어파</a>, <a href="/wiki/%EC%9D%B4%ED%83%88%EB%A6%AC%EC%95%84%EC%96%B4%ED%8C%8C" title="이탈리아어파">이탈리아어파</a>, <a href="/wiki/%EC%BC%88%ED%8A%B8%EC%96%B4%ED%8C%8C" title="켈트어파">켈트어파</a>, <a href="/wiki/%ED%86%A0%EC%B9%B4%EB%A6%AC%EC%95%84%EC%96%B4" title="토카리아어">토카리아어파</a>가 있다.</p>
<p>아리아라는 것은 산스크리트어의 <b>아리야</b>(<span lang="sa" xml:lang="sa">आर्य</span>, <span class="IPA" title="IPA 발음">[ārya]</span>)에서 유래한 것으로 ‘고귀한’이란 의미로서 그들이 자칭하는 말이다. <a href="/wiki/%EC%95%84%EB%B2%A0%EC%8A%A4%ED%83%80%EC%96%B4" title="아베스타어">아베스타어</a>로 <b>아이리야</b>이며 고대 페르시아어의 <b>아리야</b>이다.</p>
<p>북방의 <a class="mw-disambig" href="/wiki/%EC%8A%A4%ED%85%9D" title="스텝">스텝</a> 지방과 <a href="/wiki/%EC%BA%85%EC%B9%B4%EC%8A%A4" title="캅카스">캅카스</a> 지역 등에서 유목 생활을 하고 있었으나, 기원전 2400년-2000년경, 남방·서방에 이동하여, 어떤 무리는 인도에 들어가서 인도의 <a href="/wiki/%EB%B2%A0%EB%8B%A4_%EC%8B%9C%EB%8C%80" title="베다 시대">베다 문화 시대</a>를 만들었고(인도·아리아人), 어떤 무리는 <a href="/wiki/%EC%A4%91%EC%95%99%EC%95%84%EC%8B%9C%EC%95%84" title="중앙아시아">중앙아시아</a>의 <a href="/wiki/%EC%9D%B4%EB%9E%80_%EA%B3%A0%EC%9B%90" title="이란 고원">이란 고원</a>에 정착하여 <a href="/wiki/%EC%86%8C%EA%B7%B8%EB%93%9C%EC%9D%B8" title="소그드인">소그드인</a>, <a class="mw-redirect" href="/wiki/%EB%A9%94%EB%94%94%EC%95%84%EC%9D%B8" title="메디아인">메디아인</a>은 고대 <a class="mw-redirect" href="/wiki/%EC%9D%B4%EB%9E%80%EC%9D%B8" title="이란인">이란인</a>의 효시가 되었다.(이란人). 또한 서진을 계속한 아리아인들은 <a href="/wiki/%EC%BC%88%ED%8A%B8%EC%A1%B1" title="켈트족">켈트족</a>(<a href="/wiki/%EA%B0%88%EB%A6%AC%EC%95%84" title="갈리아">갈리아</a>의 제부족: 36개 씨족, 부족), 일리리아족, 원시 <a href="/wiki/%EC%8A%AC%EB%9D%BC%EB%B8%8C%EC%A1%B1" title="슬라브족">슬라브족</a>을 중심으로 하여 <a href="/wiki/%EB%8F%99%EC%9C%A0%EB%9F%BD" title="동유럽">동유럽</a>과 <a href="/wiki/%EC%84%9C%EC%9C%A0%EB%9F%BD" title="서유럽">서유럽</a>으로 진출하게 되었다. 학자들마다 그 기원이 다르지만 공통적으로 기원전 25세기에서 20세기로 보고 있다.</p>
<h2><span class="mw-headline" id=".EC.95.84.EB.A6.AC.EC.95.84.EC.9D.B8.EB.93.A4.EC.9D.98_.ED.8C.BD.EC.B0.BD.EA.B3.BC_.EC.97.AD.EC.82.AC">아리아인들의 팽창과 역사</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%95%84%EB%A6%AC%EC%95%84%EC%9D%B8&amp;action=edit&amp;section=2" title="부분 편집: 아리아인들의 팽창과 역사">원본 편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>켈트족은 나중에 <a class="mw-redirect" href="/wiki/%EC%9D%B4%EB%B2%A0%EB%A6%AC%EC%95%84%EC%9D%B8" title="이베리아인">이베리아인</a>(<a href="/wiki/%EC%8A%A4%ED%8E%98%EC%9D%B8" title="스페인">스페인</a>)과 동화되고, <a href="/wiki/%EC%8A%A4%EC%9C%84%EC%8A%A4" title="스위스">스위스</a>, <a href="/wiki/%EC%98%A4%EC%8A%A4%ED%8A%B8%EB%A6%AC%EC%95%84" title="오스트리아">오스트리아</a>, <a href="/wiki/%ED%94%84%EB%9E%91%EC%8A%A4" title="프랑스">프랑스</a>의 <a class="mw-redirect" href="/wiki/%ED%94%84%EB%9E%91%ED%81%AC%EC%A1%B1" title="프랑크족">프랑크족</a> 등으로 분화, 동화되어 갔다. 지금도 프랑스의 <a class="new" href="/w/index.php?title=%EB%B8%8C%EB%A5%B4%ED%83%80%EB%89%B4_%EC%A7%80%EB%B0%A9&amp;action=edit&amp;redlink=1" title="브르타뉴 지방 (없는 문서)">브르타뉴 지방</a>에선 켈트어의 일파인 <a href="/wiki/%EB%B8%8C%EB%A5%B4%ED%83%80%EB%89%B4%EC%96%B4" title="브르타뉴어">브르타뉴어</a>를 사용하고 있고, <a href="/wiki/%EC%98%81%EA%B5%AD" title="영국">영국</a>으로부터 분리 독립을 원하는 <a href="/wiki/%EC%8A%A4%EC%BD%94%ED%8B%80%EB%9E%9C%EB%93%9C" title="스코틀랜드">스코틀랜드</a>가 켈트족의 후예이다. 또한 <a href="/wiki/%EC%9B%A8%EC%9D%BC%EC%8A%A4" title="웨일스">웨일스</a>과 <a href="/wiki/%EC%95%84%EC%9D%BC%EB%9E%9C%EB%93%9C" title="아일랜드">아일랜드</a>, <a href="/wiki/%EB%A7%A8_%EC%84%AC" title="맨 섬">맨 섬</a>도 켈트족의 후예이다. 아리아인들은 대다수가 목축을 하는 민족으로 <a href="/wiki/%EC%9A%B0%EB%9E%84_%EC%82%B0%EB%A7%A5" title="우랄 산맥">우랄 산맥</a>을 넘어 온 서방 스키타이로 이해를 해야 한다. 또한 <a href="/wiki/%EB%B0%9C%EC%B9%B8_%EB%B0%98%EB%8F%84" title="발칸 반도">발칸 반도</a>의 <a href="/wiki/%ED%8A%B8%EB%9D%BC%ED%82%A4%EC%95%84" title="트라키아">트라키아</a>와 아리안계의 <a class="new" href="/w/index.php?title=%EC%9D%BC%EB%A6%AC%EB%A6%AC%EC%95%84%EC%9D%B8&amp;action=edit&amp;redlink=1" title="일리리아인 (없는 문서)">일리리아인</a>은 <a href="/wiki/%EC%9D%BC%EB%A6%AC%EB%A6%AC%EC%95%84" title="일리리아">일리리아</a> 제국을 건설하였다. 기원전 3세기경의 <a href="/wiki/%EC%95%8C%EB%A0%89%EC%82%B0%EB%93%9C%EB%A1%9C%EC%8A%A4_3%EC%84%B8_%EB%A9%94%EA%B0%80%EC%8A%A4" title="알렉산드로스 3세 메가스">알렉산드로스 3세 메가스</a>는 <a href="/wiki/%EB%A7%88%EC%BC%80%EB%8F%84%EB%8B%88%EC%95%84_%EC%99%95%EA%B5%AD%EC%9D%98_%EA%B5%B0%EC%A3%BC" title="마케도니아 왕국의 군주">마케도니아 왕</a>으로 유럽 정복과 고대 <a href="/wiki/%ED%8E%98%EB%A5%B4%EC%8B%9C%EC%95%84_%EC%A0%9C%EA%B5%AD" title="페르시아 제국">페르시아 제국</a>인 <a href="/wiki/%EC%95%84%EC%BC%80%EB%A9%94%EB%84%A4%EC%8A%A4_%EC%A0%9C%EA%B5%AD" title="아케메네스 제국">아케메네스 제국</a>의 영토를 전부 차지하고 <a href="/wiki/%EC%9D%B4%EC%A7%91%ED%8A%B8" title="이집트">이집트</a> 정복과 <a href="/wiki/%EB%B6%81%EC%9D%B8%EB%8F%84" title="북인도">북인도</a>까지 점령하는 소위 '동방원정'을 하기도 하였다. 알렉산드로스 대왕이 있기 전 그리스의 침공이 있을 때 일리리아 제국은 항거하였으며, 트라키아도 그리스와 항쟁을 하였으나, 정작 <a href="/wiki/%EB%A7%88%EC%BC%80%EB%8F%84%EB%8B%88%EC%95%84_%EC%99%95%EA%B5%AD" title="마케도니아 왕국">마케도니아</a>는 전쟁 한번 해 보지 않고 복속된 적이 있기 때문이다. 이런 이유로 마케도니아는 약소국이었는데 일리리아(아리안족의 국가)와 아리안계의 <a href="/wiki/%ED%8A%B8%EB%9D%BC%ED%82%A4%EC%95%84%EC%9D%B8" title="트라키아인">트라키아인</a>이 합쳐서 거대한 마케도니아로 탄생되었다는 학설에 믿음이 가는 것이다.</p>
<p>아리안인들은 유럽의 주요 민족들의 조상이다. 오늘날 <a class="mw-redirect" href="/wiki/%EC%83%89%EC%8A%A8%EC%A1%B1" title="색슨족">색슨족</a>, <a href="/wiki/%EA%B2%8C%EB%A5%B4%EB%A7%8C%EC%A1%B1" title="게르만족">게르만족</a>, <a href="/wiki/%EC%8A%AC%EB%9D%BC%EB%B8%8C%EC%A1%B1" title="슬라브족">슬라브족</a> 등은 아리아인들의 제 부족에서 갈리어 나온 것으로 유추 해야 한다. 스키타이의 고향은 <a href="/wiki/%EA%B7%B8%EB%A6%AC%EC%8A%A4_%EC%8B%A0%ED%99%94" title="그리스 신화">그리스 신화</a>에 <a href="/wiki/%EC%9A%B0%ED%81%AC%EB%9D%BC%EC%9D%B4%EB%82%98" title="우크라이나">우크라이나</a>의 <a href="/wiki/%EB%93%9C%EB%84%A4%ED%94%84%EB%A5%B4_%EA%B0%95" title="드네프르 강">드네프르 강</a>변으로 보고 있으며 그들의 신은 '해신' 오누이라고 전해지고 있다. 그래서 인도의 쌍어문과 물고기숭배 사상이 우연이 아닌 것이며 인도 베다 시대에 아리안에 의해 전래된 것으로 보고 있다. 이것은 인도의 물고기, 쌍어문 숭배사상은 바로 <a class="new" href="/w/index.php?title=%EC%8A%A4%ED%82%A4%ED%83%80%EC%9D%B4_%EC%8B%A0%ED%99%94&amp;action=edit&amp;redlink=1" title="스키타이 신화 (없는 문서)">스키타이 신화</a>를 일컫는 것이며 아리안이 서방 스키타이란 것을 간접적으로 증명해 주는 것이다.</p>
<p>아리안의 역사는 바로 우랄 산맥을 넘어 온 서방 스키타이로 이해를 해야 하며 이러한 아리안의 부족들은 동부 유럽의 제국가를 형성하였으며 나아가 서부 유럽 국가를 형성하는데 많은 역할을 하였다고 보는 것이다. 기원전 3세기경의 알렉산드로스 3세 메가스의 동방원정 이후로 아리안의 한 부족인 <a href="/wiki/%EC%9B%94%EC%A7%80" title="월지">월지족</a>(대월지국)이 서기 30년경에 쿠줄라카드피세스(구취각)에 의해 <a class="mw-redirect" href="/wiki/%EC%BF%A0%EC%83%A8_%EC%99%95%EC%A1%B0" title="쿠샨 왕조">쿠샨 왕조</a>를 건국하게 되며 <a href="/wiki/%EC%86%8C%EA%B7%B8%EB%94%94%EC%95%84%EB%82%98" title="소그디아나">소그디아나</a>와 <a href="/wiki/%EB%B0%95%ED%8A%B8%EB%A6%AC%EC%95%84" title="박트리아">박트리아</a>를 정복하고 나아가 북인도를 점령(서기 78-226년)하였다. 중앙아시아의 <a href="/wiki/%ED%83%80%EB%A6%BC_%EB%B6%84%EC%A7%80" title="타림 분지">타림 분지</a> 지배하는 등 대제국을 과시하게 되지만, 같은 부족인 키타리테와 <a href="/wiki/%EC%97%90%ED%94%84%ED%83%88" title="에프탈">에프탈</a>(백훈족)에게 멸망당한다..</p>
<h2><span class="mw-headline" id=".E3.80.8A.EC.95.84.EB.A6.AC.EC.95.88.EC.9D.98_.EC.A3.BC.EC.9A.94_.EB.B6.80.EC.A1.B1.E3.80.8B">《아리안의 주요 부족》</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%95%84%EB%A6%AC%EC%95%84%EC%9D%B8&amp;action=edit&amp;section=3" title="부분 편집: 《아리안의 주요 부족》">원본 편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<ul>
<li><i><b>아리안계 스키타이족의 분류</b></i>
<ul>
<li>아리아인: 크게 <a href="/wiki/%EC%9D%B8%EB%8F%84%EC%95%84%EB%A6%AC%EC%95%84%EC%96%B4%EA%B5%B0" title="인도아리아어군">인도아리아</a>, 중앙아시아계 아리아, 중동계아리아, 유럽계 아리아로 구분하며 이 종족은 나중에 많은 부족이나 씨족으로 분화되어 나가며 동화되기도 하였다.</li>
<li><a class="new" href="/w/index.php?title=%ED%82%B4%EB%A9%94%EB%A5%B4%EC%A1%B1&amp;action=edit&amp;redlink=1" title="킴메르족 (없는 문서)">킴메르족</a>: 캅카스와 <a href="/wiki/%ED%9D%91%ED%95%B4" title="흑해">흑해</a>의 북쪽 영역에 거주하였는데 지금 우크라이나와 <a href="/wiki/%EB%9F%AC%EC%8B%9C%EC%95%84" title="러시아">러시아</a> 남부이다. 같은 인도유럽어족으로 이란의 지배계층으로 보고 있다.</li>
<li><a class="mw-redirect" href="/wiki/%EB%A9%94%EB%94%94%EC%95%84%EC%9D%B8" title="메디아인">메디아인</a>: 이란의 북서부에 있었던 고대 이란인의 호칭으로 현재 케르만샤 일부와 <a href="/wiki/%EC%95%84%EC%A0%9C%EB%A5%B4%EB%B0%94%EC%9D%B4%EC%9E%94" title="아제르바이잔">아제르바이잔</a>, 하메단, <a href="/wiki/%ED%85%8C%ED%97%A4%EB%9E%80" title="테헤란">테헤란</a>, <a href="/wiki/%EC%BF%A0%EB%A5%B4%EB%94%94%EC%8A%A4%ED%83%84" title="쿠르디스탄">쿠르디스탄</a> 지방에 걸쳐 살았다. 이 메디아라는 이름은 고대 그리스인들이 이 지역을 메디아 또는 메데아로 불렀기 때문에 붙여진 이름으로 <a class="mw-redirect" href="/wiki/%EA%B5%AC%EC%95%BD%EC%84%B1%EC%84%9C" title="구약성서">구약성서</a>의 <a href="/wiki/%EB%8B%A4%EB%8B%88%EC%97%98%EC%84%9C" title="다니엘서">다니엘서</a>에는 '메데'로 불린다.</li>
<li><a class="new" href="/w/index.php?title=%EB%A6%AC%EB%94%94%EC%95%84%EC%9D%B8&amp;action=edit&amp;redlink=1" title="리디아인 (없는 문서)">리디아인</a>: 고대의 서부 <a href="/wiki/%EC%95%84%EB%82%98%ED%86%A8%EB%A6%AC%EC%95%84" title="아나톨리아">아나톨리아</a>(소아시아)지방을 다스린 민족으로 현 <a href="/wiki/%ED%84%B0%ED%82%A4" title="터키">터키</a>의 <a href="/wiki/%EC%9D%B4%EC%A6%88%EB%AF%B8%EB%A5%B4" title="이즈미르">이즈미르</a>, <a href="/wiki/%EB%A7%88%EB%8B%88%EC%82%AC_%EC%A3%BC" title="마니사 주">마니사 주</a>에 해당된다. 인도유럽어족으로 '<a href="/wiki/%EB%A3%A8%EB%B9%84%EC%95%84%EC%96%B4" title="루비아어">루비아어</a>'와 '<a href="/wiki/%ED%9E%88%ED%83%80%EC%9D%B4%ED%8A%B8%EC%96%B4" title="히타이트어">히타이트어</a>'와 관련되어 있다.</li>
<li><a href="/wiki/%EC%B9%B4%EC%8B%9C%ED%8A%B8%EC%9D%B8" title="카시트인">카시트인</a>: 이란 서부 로레스탄의 <a href="/wiki/%EC%9E%90%EA%B7%B8%EB%A1%9C%EC%8A%A4_%EC%82%B0%EB%A7%A5" title="자그로스 산맥">자그로스 산맥</a> 내에 위치하였던 것으로 추정되며 메소포타미아로 이동하여 기원전 1475년 <a href="/wiki/%EB%B0%94%EB%B9%8C%EB%A1%9C%EB%8B%88%EC%95%84" title="바빌로니아">바빌론 왕국</a>의 남부를 점유하였다.</li>
<li>트라키아인: 인도유럽어족으로 <a class="new" href="/w/index.php?title=%ED%8A%B8%EB%9D%BC%ED%82%A4%EC%95%84%EC%96%B4&amp;action=edit&amp;redlink=1" title="트라키아어 (없는 문서)">트라키아어</a>를 사용했다. 그들은 유럽 중앙과 남동부의 넓은 지역에 걸쳐 살았던 종족으로 그리스의 '<a href="/wiki/%EC%9D%BC%EB%A6%AC%EC%95%84%EC%8A%A4" title="일리아스">일리아스</a>'에서 첫 기록으로 나타나는데 그들은 <a class="mw-redirect" href="/wiki/%ED%8A%B8%EB%A1%9C%EC%9D%B4_%EC%A0%84%EC%9F%81" title="트로이 전쟁">트로이 전쟁</a>에서 그리스에 맞섰던 <a class="mw-redirect" href="/wiki/%ED%8A%B8%EB%A1%9C%EC%9D%B4" title="트로이">트로이</a>의 동맹으로 묘사되고 있다.</li>
<li><a class="mw-redirect" href="/wiki/%EB%8F%84%EB%A6%AC%EC%95%84%EC%9D%B8" title="도리아인">도리아인</a>: 발칸 반도로 남하하여 <a href="/wiki/%EC%8A%A4%ED%8C%8C%EB%A5%B4%ED%83%80" title="스파르타">스파르타</a>, <a href="/wiki/%EC%BD%94%EB%A6%B0%ED%86%A0%EC%8A%A4" title="코린토스">코린토스</a> 등의 <a href="/wiki/%ED%8F%B4%EB%A6%AC%EC%8A%A4" title="폴리스">폴리스</a>를 건설한 종족이다.</li>
<li><a class="new" href="/w/index.php?title=%EC%86%8C%EC%8A%A4%ED%82%A4%EB%8B%A4%EC%9D%B4%EC%9D%B8&amp;action=edit&amp;redlink=1" title="소스키다이인 (없는 문서)">소스키다이인</a>: 고대에 북서쪽으로는 <a class="mw-redirect" href="/wiki/%EB%8B%A4%EB%89%B4%EB%B8%8C_%EA%B0%95" title="다뉴브 강">다뉴브 강</a>으로 동으로는 흑해에 둘러 쌓인 영역이다. 지금의 '<a class="new" href="/w/index.php?title=%EB%8F%84%EB%B6%80%EB%A5%B4%EC%9E%90&amp;action=edit&amp;redlink=1" title="도부르자 (없는 문서)">도부르자</a>'에 해당되며 대부분 <a href="/wiki/%EB%A3%A8%EB%A7%88%EB%8B%88%EC%95%84" title="루마니아">루마니아</a>에 속하며 <a href="/wiki/%EB%B6%88%EA%B0%80%EB%A6%AC%EC%95%84" title="불가리아">불가리아</a> 지역도 있다.</li>
<li>일리리아인: 현재의 <a href="/wiki/%EC%95%8C%EB%B0%94%EB%8B%88%EC%95%84%EC%9D%B8" title="알바니아인">알바니아인</a>의 조상으로서, 발칸 반도의 서부에 일리리아 제국을 고대 마케도니아 옆에 건국하였다. 후에 마케도니아에 복속된다.</li>
<li>켈트족: 인도유럽어족의 켈트어파를 쓰는 아리아족으로 프랑스, <a href="/wiki/%EB%8F%85%EC%9D%BC" title="독일">독일</a>, 스위스, <a href="/wiki/%EC%95%8C%ED%94%84%EC%8A%A4_%EC%82%B0%EB%A7%A5" title="알프스 산맥">알프스 산맥</a> 주변에 거주한 종족으로 켈트어파는 인도유럽어족의 한 파로 <a href="/wiki/%EA%B8%B0%EC%9B%90%EC%A0%84_10%EC%84%B8%EA%B8%B0" title="기원전 10세기">기원전 10세기</a>경엔 거의 유럽 대륙 전역에 걸쳐서 사용되었다. 그후 켈트족은 기원전 390년 <a href="/wiki/%EA%B3%A0%EB%8C%80_%EB%A1%9C%EB%A7%88" title="고대 로마">로마</a>를 공략하고 전 유럽 국가를 공략하려고 위협하였는데 그들은 아나톨리아(소아시아)를 정복하여 그곳에서 영구 거주지를 만들었는데 후에 '갈라티아인'으로 불리게 되었다. 후에 <a href="/wiki/%EC%98%A4%EC%8A%A4%ED%8A%B8%EB%A6%AC%EC%95%84" title="오스트리아">오스트리아</a>, <a href="/wiki/%EC%8A%A4%EC%9C%84%EC%8A%A4" title="스위스">스위스</a>, <a href="/wiki/%EB%8F%85%EC%9D%BC" title="독일">독일</a> 등으로 이주하였고 일부는 게르만족의 <a class="mw-redirect" href="/wiki/%ED%94%84%EB%9E%91%ED%81%AC%EC%A1%B1" title="프랑크족">프랑크족</a>으로 동화되었다. 일부는 <a href="/wiki/%EC%9E%89%EA%B8%80%EB%9E%9C%EB%93%9C" title="잉글랜드">잉글랜드</a>에 동화되고 이베리아인(스페인)과 동화되기도 하였다.</li>
<li><a href="/wiki/%EA%B0%88%EB%A6%AC%EC%95%84%EC%9D%B8" title="갈리아인">갈리아인</a>: 갈리아의 원 거주민은 켈트족으로 갈리아라는 이름 자체가 켈타이(celtae)를 라틴어식으로 옮긴것이다. 갈리아의 켈트족은 증명된 바로는 35개 부족으로 나뉘어 있었는데 <a href="/wiki/%EC%9C%A8%EB%A6%AC%EC%9A%B0%EC%8A%A4_%EC%B9%B4%EC%9D%B4%EC%82%AC%EB%A5%B4" title="율리우스 카이사르">율리우스 카이사르</a>의 "갈리아전쟁기"에 상세히 기록되어 있다.</li>
<li><a href="/wiki/%EC%95%8C%EB%9D%BC%EB%8B%88%EC%A1%B1" title="알라니족">알라니족</a>: 흑해 연안 북동쪽의 스탭 지역에서 일어난 이란계 유목민이다 게르만계의 <a href="/wiki/%EB%B0%98%EB%8B%AC%EC%A1%B1" title="반달족">반달족</a>이 <a href="/wiki/%EB%B6%81%EC%95%84%ED%94%84%EB%A6%AC%EC%B9%B4" title="북아프리카">북아프리카</a>로 이주 할 때 대부분 같이 이주하였고, 일부는 현재의 <a href="/wiki/%ED%8F%B4%EB%9E%80%EB%93%9C" title="폴란드">폴란드</a> 지방에서 슬라브족과 동화되었다.</li>
<li><a href="/wiki/%EC%98%A4%EC%84%B8%ED%8A%B8%EC%9D%B8" title="오세트인">오세트인</a>: 캅카스 지방의 산악지대에 살았던 이란계의 종족으로 고대 동유럽에서 활동한 '알란족'의 후예로 현재 러시아의 <a href="/wiki/%EC%84%B8%EB%B2%A0%EB%A1%9C%EC%98%A4%EC%84%B8%ED%8B%B0%EC%95%BC_%EA%B3%B5%ED%99%94%EA%B5%AD" title="세베로오세티야 공화국">세베로오세티야 공화국</a>과 조지아의 <a href="/wiki/%EB%82%A8%EC%98%A4%EC%84%B8%ED%8B%B0%EC%95%BC" title="남오세티야">남오세티야</a>에 거주한다.</li>
<li>슬라브인: <a class="mw-redirect" href="/wiki/%EB%8F%99%EC%8A%AC%EB%9D%BC%EB%B8%8C%EC%9D%B8" title="동슬라브인">동슬라브인</a>(러시아, 우크라이나)과 <a class="mw-redirect" href="/wiki/%EC%84%9C%EC%8A%AC%EB%9D%BC%EB%B8%8C%EC%9D%B8" title="서슬라브인">서슬라브인</a>(폴란드, <a href="/wiki/%EC%B2%B4%EC%BD%94" title="체코">체코</a>, <a href="/wiki/%EC%8A%AC%EB%A1%9C%EB%B0%94%ED%82%A4%EC%95%84" title="슬로바키아">슬로바키아</a>) 그리고 <a class="mw-redirect" href="/wiki/%EB%82%A8%EC%8A%AC%EB%9D%BC%EB%B8%8C%EC%9D%B8" title="남슬라브인">남슬라브인</a>(<a href="/wiki/%EC%8A%AC%EB%A1%9C%EB%B2%A0%EB%8B%88%EC%95%84" title="슬로베니아">슬로베니아</a>, <a href="/wiki/%EC%84%B8%EB%A5%B4%EB%B9%84%EC%95%84" title="세르비아">세르비아</a>, <a href="/wiki/%ED%81%AC%EB%A1%9C%EC%95%84%ED%8B%B0%EC%95%84" title="크로아티아">크로아티아</a>, <a href="/wiki/%EB%B6%88%EA%B0%80%EB%A6%AC%EC%95%84" title="불가리아">불가리아</a>, <a href="/wiki/%EB%B3%B4%EC%8A%A4%EB%8B%88%EC%95%84" title="보스니아">보스니아</a>, <a class="mw-disambig" href="/wiki/%EB%A7%88%EC%BC%80%EB%8F%84%EB%8B%88%EC%95%84" title="마케도니아">마케도니아</a>, <a href="/wiki/%EB%AA%AC%ED%85%8C%EB%84%A4%EA%B7%B8%EB%A1%9C" title="몬테네그로">몬테네그로</a>)으로 나뉜다. 고대의 슬라브족은 '<a href="/wiki/%EC%B9%B4%EB%A5%B4%ED%8C%8C%ED%8B%B0%EC%95%84_%EC%82%B0%EB%A7%A5" title="카르파티아 산맥">카르파티아 산맥</a>' 북부에서 <a href="/wiki/%EB%93%9C%EB%84%A4%ED%94%84%EB%A5%B4_%EA%B0%95" title="드네프르 강">드네프르 강</a> 유역에 걸치는 광범위한 지역에 동서남북으로 흩어져 살다 발칸의 전지역으로 퍼졌다.</li>
<li><a href="/wiki/%EC%95%84%EB%B0%94%EB%A5%B4%EC%A1%B1" title="아바르족">아바르족</a>: 서기 6-9세기에 중앙아시아에 존속했던 종족으로 후에 <a href="/wiki/%ED%94%84%EB%9E%91%ED%81%AC_%EC%99%95%EA%B5%AD" title="프랑크 왕국">프랑크 왕국</a>에 의해 멸망한다.</li>
</ul>
</li>

<h2><span class="mw-headline" id=".EC.9C.A0.EC.A0.84.EC.9E.90">유전자</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%95%84%EB%A6%AC%EC%95%84%EC%9D%B8&amp;action=edit&amp;section=4" title="부분 편집: 유전자">원본 편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>인도-이란의 아리아인들은 <a class="new" href="/w/index.php?title=%ED%95%98%ED%94%8C%EB%A1%9C%EA%B7%B8%EB%A3%B9_R1a_(Y-DNA)&amp;action=edit&amp;redlink=1" title="하플로그룹 R1a (Y-DNA) (없는 문서)">하플로그룹 R1a (Y-DNA)</a>가 높은 빈도을 보인다. 인도 북부에서는 48.9%, <a href="/wiki/%ED%8C%8C%EC%8A%88%ED%88%B0%EC%9D%B8" title="파슈툰인">파슈툰인</a>에게 51%, <a href="/wiki/%ED%83%80%EC%A7%80%ED%81%AC%EC%9D%B8" title="타지크인">타지크인</a>에 44.7% 보인다.</p>
