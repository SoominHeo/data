<p><b>아마록</b>(Amarok, 실제 발음은 /ˈæməɹɑk/<sup class="reference" id="cite_ref-2"><a href="#cite_note-2">[2]</a></sup>)은 <a href="/wiki/%EB%A6%AC%EB%88%85%EC%8A%A4" title="리눅스">리눅스</a>와 <a href="/wiki/%EC%9C%A0%EB%8B%89%EC%8A%A4_%EA%B3%84%EC%97%B4" title="유닉스 계열">유닉스 계열</a> 프로그램을 위한 <a href="/wiki/%EC%9E%90%EC%9C%A0_%EC%86%8C%ED%94%84%ED%8A%B8%EC%9B%A8%EC%96%B4" title="자유 소프트웨어">자유 소프트웨어</a> 음악 플레이어이다. <a href="/wiki/KDE" title="KDE">KDE</a>의 코어 구성 요소를 이용하지만 KDE 발표 수명과는 관련 없이 독립적으로 발표되었다.</p>
<p>원래 이름은 <b>amaroK</b>이지만, 2006년에 Amarok으로 개정되었다.</p>

<h2><span class="mw-headline" id=".EA.B8.B0.EB.8A.A5">기능</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%95%84%EB%A7%88%EB%A1%9D&amp;action=edit&amp;section=1" title="부분 편집: 기능">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<ul>
<li>다양한 포맷의 미디어 파일을 재생한다.</li>
<li>디지털 음악 파일에 태그를 넣는다.</li>
<li>커버 아트를 특정한 앨범과 연결하고 <a class="mw-redirect" href="/wiki/%EC%95%84%EB%A7%88%EC%A1%B4_%EB%8B%B7%EC%BB%B4" title="아마존 닷컴">아마존</a>으로부터 커버 정보를 받는다.</li>
<li>스마트 재생 목록과 동적 재생 목록 등의 재생 목록을 만들고 편집한다.</li>
<li>위키백과로부터 작곡가 정보를 내려 받아 보여 주고, 가사를 받는다.</li>
<li><a class="mw-redirect" href="/wiki/%EB%9D%BC%EC%8A%A4%ED%8A%B8_%EC%97%90%ED%94%84%EC%97%A0" title="라스트 에프엠">라스트 에프엠</a>을 지원한다.</li>
<li>jamendo.com, magnatune.com에서 <a href="/wiki/%EB%8F%85%EB%A6%BD_%EC%9D%8C%EC%95%85" title="독립 음악">독립 음악</a> DB를 받아, 아마록 내에서 실행하고, 다운로드 받을 수 있다.</li>
<li>mp3tunes와 동기화가 가능하다.</li>
<li><a href="/wiki/%ED%8C%9F%EC%BA%90%EC%8A%A4%ED%8A%B8" title="팟캐스트">팟캐스트</a>리스트에서 여러 <a href="/wiki/%ED%8C%9F%EC%BA%90%EC%8A%A4%ED%8A%B8" title="팟캐스트">팟캐스트</a>들을 선택해 들을 수 있다.</li>
</ul>
