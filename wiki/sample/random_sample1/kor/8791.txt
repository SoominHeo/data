<h1 class="firstHeading" id="firstHeading" lang="ko">아마시아 강화</h1>
<div class="mw-body-content" id="bodyContent">
<div id="siteSub">위키백과, 우리 모두의 백과사전.</div>
<div id="contentSub"></div>
<div class="mw-jump" id="jump-to-nav">
					이동:					<a href="#mw-head">둘러보기</a>, 					<a href="#p-search">검색</a>
</div>
<div class="mw-content-ltr" dir="ltr" id="mw-content-text" lang="ko"><script>function mfTempOpenSection(id){var block=document.getElementById("mf-section-"+id);block.className+=" open-block";block.previousSibling.className+=" open-block";}</script><p><b>아마시아 강화</b>(<span style="font-size: smaller;"><a href="/wiki/%ED%8E%98%EB%A5%B4%EC%8B%9C%EC%95%84%EC%96%B4" title="페르시아어">페르시아어</a>:</span> <span lang="fa" xml:lang="fa">پیمان آماسیه‎‎</span>, <span style="font-size: smaller;"><a href="/wiki/%ED%84%B0%ED%82%A4%EC%96%B4" title="터키어">터키어</a>:</span> <span lang="tr" xml:lang="tr">Amasya Antlaşması</span>)는 1555년 5월 29일, <a href="/wiki/%EC%82%AC%ED%8C%8C%EB%B9%84_%EC%99%95%EC%A1%B0" title="사파비 왕조">사파비 왕조</a>의 <a class="new" href="/w/index.php?title=%ED%83%80%ED%9D%90%EB%A7%88%EC%8A%A4%ED%94%84_1%EC%84%B8&amp;action=edit&amp;redlink=1" title="타흐마스프 1세 (없는 문서)">타흐마스프 1세</a>와 <a href="/wiki/%EC%98%A4%EC%8A%A4%EB%A7%8C_%EC%A0%9C%EA%B5%AD" title="오스만 제국">오스만 제국</a>의 <a class="mw-redirect" href="/wiki/%EC%8A%90%EB%A0%88%EC%9D%B4%EB%A7%8C_1%EC%84%B8" title="슐레이만 1세">슐레이만 1세</a> 사이에 체결된 <a class="new" href="/w/index.php?title=%EC%98%A4%EC%8A%A4%EB%A7%8C-%EC%82%AC%ED%8C%8C%EB%B9%84_%EC%A0%84%EC%9F%81_(1532%EB%85%84-1955%EB%85%84)&amp;action=edit&amp;redlink=1" title="오스만-사파비 전쟁 (1532년-1955년) (없는 문서)">제1차 오스만-사파비 전쟁</a>의 강화 조약이다. 조약의 명칭은 <a href="/wiki/%EC%95%84%EB%A7%88%EC%8B%9C%EC%95%84" title="아마시아">아마시아</a>에서 체결된 데 따른 것이다.</p>
<h2><span class="mw-headline" id=".EB.82.B4.EC.9A.A9">내용</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%95%84%EB%A7%88%EC%8B%9C%EC%95%84_%EA%B0%95%ED%99%94&amp;action=edit&amp;section=1" title="부분 편집: 내용">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>조약은 이란과 오스만 제국 사이의 국경을 획정하고 평화는 이후 20년간 계속되었다. 조약에 의해 <a href="/wiki/%EC%95%84%EB%A5%B4%EB%A9%94%EB%8B%88%EC%95%84" title="아르메니아">아르메니아</a>와 <a href="/wiki/%EC%A1%B0%EC%A7%80%EC%95%84_(%EA%B5%AD%EA%B0%80)" title="조지아 (국가)">조지아</a>는 동서로 등분되어 <a class="new" href="/w/index.php?title=%EC%84%9C%EB%B6%80_%EC%95%84%EB%A5%B4%EB%A9%94%EB%8B%88%EC%95%84&amp;action=edit&amp;redlink=1" title="서부 아르메니아 (없는 문서)">서부 아르메니아</a>, 서부 <a href="/wiki/%EC%BF%A0%EB%A5%B4%EB%94%94%EC%8A%A4%ED%83%84" title="쿠르디스탄">쿠르디스탄</a>, 서부 조지아 (<a href="/wiki/%EB%A9%94%EC%8A%A4%EC%BC%80%ED%8B%B0" title="메스케티">메스케티</a> 서부 포함)는 오스만 령, <a class="new" href="/w/index.php?title=%EB%8F%99%EB%B6%80_%EC%95%84%EB%A5%B4%EB%A9%94%EB%8B%88%EC%95%84&amp;action=edit&amp;redlink=1" title="동부 아르메니아 (없는 문서)">동부 아르메니아</a>, 동부 쿠르디스탄, 동부 조지아 (메스케티 동부 포함)를 이란 령으로 한다.<sup class="reference" id="cite_ref-1"><a href="#cite_note-1">[1]</a></sup> 오스만 제국이 <a href="/wiki/%EB%B0%94%EA%B7%B8%EB%8B%A4%EB%93%9C" title="바그다드">바그다드</a>를 포함한 <a href="/wiki/%EC%9D%B4%EB%9D%BC%ED%81%AC" title="이라크">이라크</a> 대부분을 획득하고 <a href="/wiki/%ED%8E%98%EB%A5%B4%EC%8B%9C%EC%95%84_%EB%A7%8C" title="페르시아 만">페르시아 만</a> 출구를 얻었으며, 이란은 전쟁에 영유하고 있는 전 수도 <a href="/wiki/%ED%83%80%EB%B8%8C%EB%A6%AC%EC%A6%88" title="타브리즈">타브리즈</a>와 북서부 <a class="mw-redirect" href="/wiki/%EC%BD%94%EC%B9%B4%EC%84%9C%EC%8A%A4" title="코카서스">코카서스</a> 지방 등 (<a class="mw-redirect" href="/wiki/%EB%8B%A4%EA%B2%8C%EC%8A%A4%ED%83%84" title="다게스탄">다게스탄</a>과 <a href="/wiki/%EC%95%84%EC%A0%9C%EB%A5%B4%EB%B0%94%EC%9D%B4%EC%9E%94" title="아제르바이잔">아제르바이잔</a> 전역 포함) 모두 유지했다.<sup class="reference" id="cite_ref-2"><a href="#cite_note-2">[2]</a></sup><sup class="reference" id="cite_ref-3"><a href="#cite_note-3">[3]</a></sup><sup class="reference" id="cite_ref-4"><a href="#cite_note-4">[4]</a></sup> 이를 통해 양국의 국경선은 동서 조지아를 나누며 <a href="/wiki/%EB%A7%90%EB%A6%AC%EC%BA%85%EC%B9%B4%EC%8A%A4_%EC%82%B0%EB%A7%A5" title="말리캅카스 산맥">말리캅카스 산맥</a>에서 아르메니아, <a href="/wiki/%EC%9E%90%EA%B7%B8%EB%A1%9C%EC%8A%A4_%EC%82%B0%EB%A7%A5" title="자그로스 산맥">자그로스 산맥</a>의 서쪽 기슭을 통과하여 페르시아 만까지 계속된다.</p>
<p><a href="/wiki/%EC%97%90%EB%A5%B4%EC%A3%BC%EB%A3%B8" title="에르주룸">에르주룸</a>, <a class="new" href="/w/index.php?title=%EC%83%A4%ED%9D%90%EB%A6%AC%EC%A1%B0%EB%A5%B4&amp;action=edit&amp;redlink=1" title="샤흐리조르 (없는 문서)">샤흐리조르</a>, <a href="/wiki/%EB%B0%98_(%ED%84%B0%ED%82%A4)" title="반 (터키)">반</a> 등 <a class="new" href="/w/index.php?title=%EB%8F%99%EB%B6%80_%EC%95%84%EB%82%98%ED%86%A8%EB%A6%AC%EC%95%84_%EC%A7%80%EB%B0%A9&amp;action=edit&amp;redlink=1" title="동부 아나톨리아 지방 (없는 문서)">동부 아나톨리아 지방</a>의 일부 도시는 완충 지대에 지정되었다.<sup class="reference" id="cite_ref-5"><a href="#cite_note-5">[5]</a></sup> <a href="/wiki/%EC%B9%B4%EB%A5%B4%EC%8A%A4" title="카르스">카르스</a>는 중립을 선언하였지만, 요새가 파괴되었다.<sup class="reference" id="cite_ref-6"><a href="#cite_note-6">[6]</a></sup><sup class="reference" id="cite_ref-7"><a href="#cite_note-7">[7]</a></sup></p>
<p>오스만 제국은 또한 페르시아 순례자에게 <a href="/wiki/%EB%A9%94%EC%B9%B4" title="메카">메카</a>와 <a href="/wiki/%EB%A9%94%EB%94%94%EB%82%98" title="메디나">메디나</a> 등 성지과 이라크의 시아파 순례에 통행을 허용했다.<sup class="reference" id="cite_ref-8"><a href="#cite_note-8">[8]</a></sup></p>
<p>최종적으로 코카서스 지방의 분할과 <a href="/wiki/%EB%A9%94%EC%86%8C%ED%8F%AC%ED%83%80%EB%AF%B8%EC%95%84" title="메소포타미아">메소포타미아</a>의 오스만에 할양에 대해서는 1639년 맺어진 평화 조약인 <a class="new" href="/w/index.php?title=%EC%A3%BC%ED%95%98%EB%B8%8C_%EC%A1%B0%EC%95%BD&amp;action=edit&amp;redlink=1" title="주하브 조약 (없는 문서)">주하브 조약</a>으로 미루어졌다.<sup class="reference" id="cite_ref-slovari33200_9-0"><a href="#cite_note-slovari33200-9">[9]</a></sup></p>
<p>이 조약은 또 다른 의무를 부과했다. 3대째까지의 <a href="/wiki/%EC%A0%95%ED%86%B5_%EC%B9%BC%EB%A6%AC%ED%8C%8C" title="정통 칼리파">정통 칼리파</a>,<sup class="reference" id="cite_ref-10"><a href="#cite_note-10">[10]</a></sup> <a href="/wiki/%EC%95%84%EC%9D%B4%EC%83%A4" title="아이샤">아이샤</a>와 <a href="/wiki/%EC%82%AC%ED%95%98%EB%B0%94" title="사하바">사하바</a>에 대한 저주를 중지하는 의무를 부과했다. 이 의무는 모든 오스만과 사파비 간의 조약에서 부과되었지만,<sup class="reference" id="cite_ref-11"><a href="#cite_note-11">[11]</a></sup> <a href="/wiki/%EC%8B%9C%EC%95%84%ED%8C%8C" title="시아파">시아파</a>인 사파비 왕조와 타흐마스프 1세에게는 굴욕적이었다.<sup class="reference" id="cite_ref-12"><a href="#cite_note-12">[12]</a></sup></p>
<h2><span class="mw-headline" id=".EA.B0.81.EC.A3.BC">각주</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%95%84%EB%A7%88%EC%8B%9C%EC%95%84_%EA%B0%95%ED%99%94&amp;action=edit&amp;section=2" title="부분 편집: 각주">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<ol class="references">
<li id="cite_note-1"><span class="mw-cite-backlink"><a href="#cite_ref-1">↑</a></span> <span class="reference-text"><cite class="citation book">Mikaberidze, Alexander (2015). 《Historical Dictionary of Georgia》 2판.
