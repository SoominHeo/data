<p><b>아말렉</b>은 <a class="mw-redirect" href="/wiki/%EA%B5%AC%EC%95%BD%EC%84%B1%EC%84%9C" title="구약성서">구약성서</a>에 나오는 고대 민족으로 현 <a href="/wiki/%EC%9D%B4%EC%8A%A4%EB%9D%BC%EC%97%98" title="이스라엘">이스라엘</a> 남쪽 주변 국가들에 살았다. 이 민족 역시 <a class="mw-redirect" href="/wiki/%EC%9D%B4%EC%8A%A4%EB%9D%BC%EC%97%98_%EB%AF%BC%EC%A1%B1" title="이스라엘 민족">이스라엘 민족</a>을 오랫동안 괴롭힌 것으로 알려져 있다.</p>
<p>아말렉의 시조는 <a href="/wiki/%EC%95%84%EB%B8%8C%EB%9D%BC%ED%95%A8" title="아브라함">아브라함</a>의 손자이자 이삭의 아들 <a href="/wiki/%EC%95%BC%EA%B3%B1" title="야곱">야곱</a>의 형으로 죽 한그릇으로 장자의 권리를 팔았던 <a class="new" href="/w/index.php?title=%EC%97%90%EC%84%9C&amp;action=edit&amp;redlink=1" title="에서 (없는 문서)">에서</a>이다. 처음에는 형제처럼 지냈으나 <a href="/wiki/%EC%95%BC%EA%B3%B1" title="야곱">야곱</a>과 형제들, 이스라엘 백성들이 <a href="/wiki/%EC%9A%94%EC%85%89" title="요셉">요셉</a>을 따라 <a href="/wiki/%EC%9D%B4%EC%A7%91%ED%8A%B8" title="이집트">이집트</a>로 가고 아말렉인들은 <a href="/wiki/%EA%B0%80%EB%82%98%EC%95%88" title="가나안">가나안</a> 남쪽 지방에 살기 시작했다.</p>
<p>이후 <a href="/wiki/%EB%AA%A8%EC%84%B8" title="모세">모세</a>가 이끄는 <a class="mw-redirect" href="/wiki/%EC%9D%B4%EC%8A%A4%EB%9D%BC%EC%97%98_%EB%AF%BC%EC%A1%B1" title="이스라엘 민족">이스라엘 민족</a>이 <a href="/wiki/%EA%B0%80%EB%82%98%EC%95%88" title="가나안">가나안</a>에 들어갈 때 이를 방해하면서 적대 관계가 되었고 <a class="new" href="/w/index.php?title=%EB%A5%B4%EB%B9%84%EB%94%A4_%EC%A0%84%ED%88%AC&amp;action=edit&amp;redlink=1" title="르비딤 전투 (없는 문서)">르비딤 전투</a>에서 패했다가 후에 가나안으로 쳐들어오는 이스라엘군을 가나안인들과 함께 연합해 무찔렀다.</p>
<p><a href="/wiki/%ED%8C%90%EA%B4%80%EA%B8%B0" title="판관기">판관기</a> 시기 <a href="/wiki/%EC%9D%B4%EC%8A%A4%EB%9D%BC%EC%97%98" title="이스라엘">이스라엘</a>을 침공해 잠시 다스리기도 했다. 하지만 <a href="/wiki/%EC%97%B0%ED%95%A9_%EC%9D%B4%EC%8A%A4%EB%9D%BC%EC%97%98_%EC%99%95%EA%B5%AD" title="연합 이스라엘 왕국">연합 이스라엘 왕국</a>이 설립되고 왕 <a href="/wiki/%EC%82%AC%EC%9A%B8" title="사울">사울</a>이 21만 대군을 이끌고 아말렉을 공격해 아말렉의 왕 <a class="new" href="/w/index.php?title=%EC%95%84%EA%B0%81&amp;action=edit&amp;redlink=1" title="아각 (없는 문서)">아각</a>을 처형하고 아말렉인들은 <a href="/wiki/%EC%9D%B4%EC%A7%91%ED%8A%B8" title="이집트">이집트</a>로 내쫓았다.</p>
<p>그리고 계속 가나안 남쪽 지방에 남아있던 아말렉의 잔당들은 <a class="mw-redirect" href="/wiki/%EC%8B%9C%EA%B8%80%EB%9D%BD_%EC%A0%84%ED%88%AC" title="시글락 전투">시글락 전투</a>에서 <a href="/wiki/%EB%8B%A4%EC%9C%97" title="다윗">다윗</a>에 의해 모두 토벌당했다. 그 이후 <a href="/wiki/%EC%82%AC%EB%AC%B4%EC%97%98" title="사무엘">사무엘</a>에게 멸망의 저주를 받아 <a href="/wiki/%EC%9C%A0%EB%8B%A4_%EC%99%95%EA%B5%AD" title="유다 왕국">유다 왕국</a>의 <a class="mw-redirect" href="/wiki/%ED%9E%88%EC%8A%A4%EA%B8%B0%EC%95%BC" title="히스기야">히스기야</a> 시대에 전멸했다.</p>
