<p><b>아메리스타 에어 카고</b>(<span style="font-size: smaller;"><a href="/wiki/%EC%98%81%EC%96%B4" title="영어">영어</a>:</span> <span lang="en" xml:lang="en">Ameristar Air Cargo</span>)는 <a href="/wiki/%EB%AF%B8%EA%B5%AD" title="미국">미국</a>의 화물 항공사로 본사는 <a href="/wiki/%EB%AF%B8%EA%B5%AD" title="미국">미국</a> <a href="/wiki/%ED%85%8D%EC%82%AC%EC%8A%A4_%EC%A3%BC" title="텍사스 주">텍사스 주</a> <a href="/wiki/%EB%8C%88%EB%9F%AC%EC%8A%A4" title="댈러스">댈러스</a>에 위치해 있으며 <a href="/wiki/2000%EB%85%84" title="2000년">2000년</a>에 설립했다. <a href="/wiki/%ED%97%88%EB%B8%8C_%EA%B3%B5%ED%95%AD" title="허브 공항">허브 공항</a>으로 <a class="new" href="/w/index.php?title=%EC%95%A0%EB%94%94%EC%8A%A8_%EA%B3%B5%ED%95%AD&amp;action=edit&amp;redlink=1" title="애디슨 공항 (없는 문서)">애디슨 공항</a>이 있으며 그 밖의 <a href="/wiki/%ED%97%88%EB%B8%8C_%EA%B3%B5%ED%95%AD" title="허브 공항">허브 공항</a>으로 <a class="new" href="/w/index.php?title=%EC%9B%94%EB%A1%9C%EC%9A%B0%EB%9F%B0_%EA%B3%B5%ED%95%AD&amp;action=edit&amp;redlink=1" title="월로우런 공항 (없는 문서)">월로우런 공항</a>과 <a class="new" href="/w/index.php?title=%EC%97%98%ED%8C%8C%EC%86%8C_%EA%B5%AD%EC%A0%9C%EA%B3%B5%ED%95%AD&amp;action=edit&amp;redlink=1" title="엘파소 국제공항 (없는 문서)">엘파소 국제공항</a>이 있다.<sup class="reference" id="cite_ref-FI_1-0"><a href="#cite_note-FI-1">[1]</a></sup></p>

<h2><span class="mw-headline" id=".EC.97.AD.EC.82.AC">역사</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%95%84%EB%A9%94%EB%A6%AC%EC%8A%A4%ED%83%80_%EC%97%90%EC%96%B4_%EC%B9%B4%EA%B3%A0&amp;action=edit&amp;section=1" title="부분 편집: 역사">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>항공사는 <a href="/wiki/2000%EB%85%84" title="2000년">2000년</a>에 설립 했으며 <a href="/wiki/2005%EB%85%84" title="2005년">2005년</a> <a href="/wiki/9%EC%9B%94" title="9월">9월</a>에 <a href="/wiki/%EB%B3%B4%EC%9E%89_737" title="보잉 737">보잉 737-200</a>을 도입 했으며 <a href="/wiki/2008%EB%85%84" title="2008년">2008년</a>에 <a href="/wiki/%EC%98%81%ED%99%94" title="영화">영화</a>에 출연한 적이 있다. 여객의 경우 스포츠와 엔터테인먼트 업계의 유명 인사, 대학 체육팀, 높은 자산 개인을 포함한다.</p>
