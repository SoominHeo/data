<p><b>아베 신조</b>(<span style="font-size: smaller;"><a href="/wiki/%EC%9D%BC%EB%B3%B8%EC%96%B4" title="일본어">일본어</a>:</span> <span lang="ja"><span lang="ja"><ruby><rb>安倍 晋三</rb> <rp>(</rp><rt>あべ しんぞう</rt><rp>)</rp></ruby></span></span>, <a href="/wiki/1954%EB%85%84" title="1954년">1954년</a> <a href="/wiki/9%EC%9B%94_21%EC%9D%BC" title="9월 21일">9월 21일</a> ~ )는 <a href="/wiki/%EC%9D%BC%EB%B3%B8" title="일본">일본</a>의 정치인이다. <a href="/wiki/%EC%9E%90%EC%9C%A0%EB%AF%BC%EC%A3%BC%EB%8B%B9_(%EC%9D%BC%EB%B3%B8)" title="자유민주당 (일본)">자유민주당</a> 소속의 <a href="/wiki/%EC%A4%91%EC%9D%98%EC%9B%90" title="중의원">중의원</a> 의원이자 제90대·96대·97대 <a href="/wiki/%EC%9D%BC%EB%B3%B8%EC%9D%98_%EB%82%B4%EA%B0%81%EC%B4%9D%EB%A6%AC%EB%8C%80%EC%8B%A0" title="일본의 내각총리대신">일본의 내각총리대신</a>으로,<sup class="reference" id="cite_ref-1"><a href="#cite_note-1">[1]</a></sup> 제21대 자유민주당 총재이다.</p>
<p>총리가 되기 전에는 제37대 자유민주당 간사장을 지냈으며, 제72대 내각관방장관, 제21대 자유민주당 총재를 지냈다. 2012년 12월 총선에서 다시 승리하여,<sup class="reference" id="cite_ref-2"><a href="#cite_note-2">[2]</a></sup> 5년 3개월만에 다시 총리로 올랐다.</p>
<p>2014년 12월 총선에서 또 다시 승리하여, 97대 총리가 되었다.</p>
<p>일본 경제의 고질적인 문제였던 <a href="/wiki/%EB%94%94%ED%94%8C%EB%A0%88%EC%9D%B4%EC%85%98" title="디플레이션">디플레이션</a> 해결을 위해 집권하면서부터 <a href="/wiki/%EC%95%84%EB%B2%A0%EB%85%B8%EB%AF%B9%EC%8A%A4" title="아베노믹스">아베노믹스</a>로 불리는 강력한 양적완화 정책을 실시하였으며, 이로 인해 일본 증시에 활력이 붙고 일본 대기업들의 수출 경쟁력 증가 등의 효과를 거둔 한편, 수입 원자재 가격 상승 및 재정 부채 증가 등 부작용도 겪고 있다.<sup class="reference" id="cite_ref-3"><a href="#cite_note-3">[3]</a></sup> 또한 이로 인해 엔화 가치가 떨어져 대한민국 기업들의 수출 경쟁력에도 많은 영향을 끼치고 있다.</p>
<p>외교적으로는 미일 동맹을 최우선시하는 등 대미 관계에 중점을 두고 있다. 군사적으로는 집단적 자위권을 추진, 전후 전쟁을 금지한 평화헌법을 개정하여 일본을 전쟁을 할 수 있는 '보통국가'로 만드는 일을 추진하고 있다. 중국과 한반도에 대한 과거의 식민지 침략에 대해서는 “침략의 정의는 학계에서도, 국제적으로도 정해지지 않았다”고 주장하는 등 극우 정치가의 행보를 보여 <a href="/wiki/%EC%A0%9C2%EC%B0%A8_%EC%84%B8%EA%B3%84_%EB%8C%80%EC%A0%84" title="제2차 세계 대전">제2차 세계 대전</a> 때 피해를 입은 유럽 국가, 중국, 한국의 비판을 샀다.<sup class="reference" id="cite_ref-4"><a href="#cite_note-4">[4]</a></sup></p>

<h2><span class="mw-headline" id=".EA.B0.9C.EC.9A.94">개요</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%95%84%EB%B2%A0_%EC%8B%A0%EC%A1%B0&amp;action=edit&amp;section=1" title="부분 편집: 개요">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>아베의 집안에는 정치가를 지낸 이들이 많으며, 조부와 증조부도 수상을 지낸 세습 정치 가문으로 잘 알려져 있다.<sup class="reference" id="cite_ref-5"><a href="#cite_note-5">[5]</a></sup> 조부는 중의원 의원을 지낸 <a class="new" href="/w/index.php?title=%EC%95%84%EB%B2%A0_%EA%B0%84&amp;action=edit&amp;redlink=1" title="아베 간 (없는 문서)">아베 간</a>(安倍寬)이고, 외조부는 제56·57대 <a class="mw-redirect" href="/wiki/%EC%9D%BC%EB%B3%B8%EC%9D%98_%EC%B4%9D%EB%A6%AC" title="일본의 총리">총리</a>를 역임한 <a href="/wiki/%EA%B8%B0%EC%8B%9C_%EB%85%B8%EB%B6%80%EC%8A%A4%EC%BC%80" title="기시 노부스케">기시 노부스케</a>(岸信介)이다. 아버지는 <a href="/wiki/%EC%9D%BC%EB%B3%B8%EC%9D%98_%EC%99%B8%EB%AC%B4%EB%8C%80%EC%8B%A0" title="일본의 외무대신">일본의 외무대신</a>을 지낸 <a href="/wiki/%EC%95%84%EB%B2%A0_%EC%8B%A0%ED%83%80%EB%A1%9C" title="아베 신타로">아베 신타로</a>(安倍晋太郎), 남동생은 <a href="/wiki/%EC%B0%B8%EC%9D%98%EC%9B%90_(%EC%9D%BC%EB%B3%B8)" title="참의원 (일본)">참의원</a>인 <a class="new" href="/w/index.php?title=%EA%B8%B0%EC%8B%9C_%EB%85%B8%EB%B6%80%EC%98%A4&amp;action=edit&amp;redlink=1" title="기시 노부오 (없는 문서)">기시 노부오</a>(岸信夫)이다. 부인은 <a href="/wiki/%EB%AA%A8%EB%A6%AC%EB%82%98%EA%B0%80_%EC%A0%9C%EA%B3%BC" title="모리나가 제과">모리나가 제과</a> 사장 <a class="new" href="/w/index.php?title=%EB%A7%88%EC%93%B0%EC%9E%90%ED%82%A4_%EC%95%84%ED%82%A4%EC%98%A4&amp;action=edit&amp;redlink=1" title="마쓰자키 아키오 (없는 문서)">마쓰자키 아키오</a>의 딸 <a href="/wiki/%EC%95%84%EB%B2%A0_%EC%95%84%ED%82%A4%EC%97%90" title="아베 아키에">아키에</a>이다.</p>
<p>대학 졸업 후 고베 제강소의 직원으로 일했고, 외무장관의 비서관을 지냈다. 내각관방장관, 내각관방부장관, <a class="mw-disambig" href="/wiki/%EC%9E%90%EC%9C%A0%EB%AF%BC%EC%A3%BC%EB%8B%B9" title="자유민주당">자유민주당</a> 간사장과 총재를 지냈으며, <a href="/wiki/2006%EB%85%84" title="2006년">2006년</a> 총리로 임명되었으나 이듬 해 사임했다. <a href="/wiki/2012%EB%85%84" title="2012년">2012년</a> 다시 <a class="mw-redirect" href="/wiki/%EC%9D%BC%EB%B3%B8_%EB%82%B4%EA%B0%81%EC%B4%9D%EB%A6%AC%EB%8C%80%EC%8B%A0" title="일본 내각총리대신">일본 내각총리대신</a>직에 복귀했다. 2013년 <a href="/wiki/%EC%95%BC%EC%8A%A4%EC%BF%A0%EB%8B%88_%EC%8B%A0%EC%82%AC" title="야스쿠니 신사">야스쿠니 신사</a> 참배로 <a href="/wiki/%EB%8C%80%ED%95%9C%EB%AF%BC%EA%B5%AD" title="대한민국">대한민국</a>과 <a href="/wiki/%EC%A4%91%EA%B5%AD" title="중국">중국</a>으로부터 많은 비판을 받았다.</p>
<h2><span class="mw-headline" id=".EA.B2.BD.EB.A0.A5">경력</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%95%84%EB%B2%A0_%EC%8B%A0%EC%A1%B0&amp;action=edit&amp;section=2" title="부분 편집: 경력">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p><a href="/wiki/1954%EB%85%84" title="1954년">1954년</a> <a href="/wiki/9%EC%9B%94_21%EC%9D%BC" title="9월 21일">9월 21일</a>, 당시 <a href="/wiki/%EB%A7%88%EC%9D%B4%EB%8B%88%EC%B9%98_%EC%8B%A0%EB%AC%B8" title="마이니치 신문">마이니치 신문</a>의 기자였던 <a href="/wiki/%EC%95%84%EB%B2%A0_%EC%8B%A0%ED%83%80%EB%A1%9C" title="아베 신타로">아베 신타로</a>와 그 아내 요코(洋子)의 차남으로 <a href="/wiki/%EB%8F%84%EC%BF%84" title="도쿄">도쿄</a>에서 태어났다.</p>
<p><a href="/wiki/%EA%B8%B0%EC%8B%9C_%EB%85%B8%EB%B6%80%EC%8A%A4%EC%BC%80" title="기시 노부스케">기시 노부스케</a>가 입학하도록 주장한<sup class="reference" id="cite_ref-aksk_6-0"><a href="#cite_note-aksk-6">[6]</a></sup> 세이케이 계열에서 초등학교부터 고등학교를 거쳐 <a class="new" href="/w/index.php?title=%EC%84%B8%EC%9D%B4%EC%BC%80%EC%9D%B4_%EB%8C%80%ED%95%99&amp;action=edit&amp;redlink=1" title="세이케이 대학 (없는 문서)">세이케이 대학</a> 법학부 정치학과를 졸업했다.</p>
<p>아버지인 <a href="/wiki/%EC%95%84%EB%B2%A0_%EC%8B%A0%ED%83%80%EB%A1%9C" title="아베 신타로">아베 신타로</a>를 비롯해 <a class="new" href="/w/index.php?title=%EC%95%84%EB%B2%A0_%EA%B0%84&amp;action=edit&amp;redlink=1" title="아베 간 (없는 문서)">아베 간</a>, <a href="/wiki/%EA%B8%B0%EC%8B%9C_%EB%85%B8%EB%B6%80%EC%8A%A4%EC%BC%80" title="기시 노부스케">기시 노부스케</a>, <a href="/wiki/%EC%82%AC%ED%86%A0_%EC%97%90%EC%9D%B4%EC%82%AC%EC%BF%A0" title="사토 에이사쿠">사토 에이사쿠</a>, 가정교사였던 <a class="new" href="/w/index.php?title=%ED%9E%88%EB%9D%BC%EC%82%AC%EC%99%80_%EA%B0%80%EC%93%B0%EC%97%90%EC%9D%B4&amp;action=edit&amp;redlink=1" title="히라사와 가쓰에이 (없는 문서)">히라사와 가쓰에이</a>, 친척들까지 <a href="/wiki/%EB%8F%84%EC%BF%84_%EB%8C%80%ED%95%99" title="도쿄 대학">도쿄 대학</a> 출신이 많았지만 아베 신조는 입시를 통과한 것이 아닌 에스컬레이터식으로 세이케이 대학으로 진학했다.</p>
<h3><span class="mw-headline" id=".EC.A0.95.EA.B3.84_.EC.9E.85.EB.AC.B8">정계 입문</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%95%84%EB%B2%A0_%EC%8B%A0%EC%A1%B0&amp;action=edit&amp;section=3" title="부분 편집: 정계 입문">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>졸업이 후 얼마 간 미국으로 건너가 영어를 배우고, <a href="/wiki/1979%EB%85%84" title="1979년">1979년</a> 4월에 귀국하여 <a href="/wiki/%EA%B3%A0%EB%B2%A0_%EC%A0%9C%EA%B0%95" title="고베 제강">고베 제강</a>에 입사했다. 3년 뒤인 <a href="/wiki/1982%EB%85%84" title="1982년">1982년</a>에는 아버지이자 <a href="/wiki/%EC%9D%BC%EB%B3%B8%EC%9D%98_%EC%99%B8%EB%AC%B4%EB%8C%80%EC%8B%A0" title="일본의 외무대신">외무대신</a>이던 <a href="/wiki/%EC%95%84%EB%B2%A0_%EC%8B%A0%ED%83%80%EB%A1%9C" title="아베 신타로">아베 신타로</a>의 비서로 취임하고, 각국의 수뇌와의 회담에 동석하는 등 후계자 수업을 진행해갔다. <a href="/wiki/1987%EB%85%84" title="1987년">1987년</a>에는 <a href="/wiki/%ED%9B%84%EC%BF%A0%EB%8B%A4_%EB%8B%A4%EC%BC%80%EC%98%A4" title="후쿠다 다케오">후쿠다 다케오</a> 전 총리의 주선으로 <a href="/wiki/%EB%AA%A8%EB%A6%AC%EB%82%98%EA%B0%80_%EC%A0%9C%EA%B3%BC" title="모리나가 제과">모리나가 제과</a> 사장 <a class="new" href="/w/index.php?title=%EB%A7%88%EC%93%B0%EC%9E%90%ED%82%A4_%EC%95%84%ED%82%A4%EC%98%A4&amp;action=edit&amp;redlink=1" title="마쓰자키 아키오 (없는 문서)">마쓰자키 아키오</a>의 딸이던 <a href="/wiki/%EC%95%84%EB%B2%A0_%EC%95%84%ED%82%A4%EC%97%90" title="아베 아키에">아키에</a>와 결혼했다. <a href="/wiki/1991%EB%85%84" title="1991년">1991년</a>에 총리 후보로 유력하게 거론되었던 아버지 아베 신타로가 급사하면서, <a href="/wiki/1993%EB%85%84" title="1993년">1993년</a> <a href="/wiki/%EC%A0%9C40%ED%9A%8C_%EC%9D%BC%EB%B3%B8_%EC%A4%91%EC%9D%98%EC%9B%90_%EC%9D%98%EC%9B%90_%EC%B4%9D%EC%84%A0%EA%B1%B0" title="제40회 일본 중의원 의원 총선거">제40회 일본 중의원 의원 총선거</a>에 사망한 아버지의 지역구였던 야마구치 1구(당시)에서 출마해 처음으로 당선되었다.</p>
<h3><span class="mw-headline" id=".EC.A4.91.EC.9D.98.EC.9B.90_.EC.9D.98.EC.9B.90">중의원 의원</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%95%84%EB%B2%A0_%EC%8B%A0%EC%A1%B0&amp;action=edit&amp;section=4" title="부분 편집: 중의원 의원">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>소속 파벌의 영수였던 <a href="/wiki/%EB%AA%A8%EB%A6%AC_%EC%9A%94%EC%8B%9C%EB%A1%9C" title="모리 요시로">모리 요시로</a>가 수상으로 조각한 <a href="/wiki/2000%EB%85%84" title="2000년">2000년</a>의 <a href="/wiki/%EC%A0%9C2%EC%B0%A8_%EB%AA%A8%EB%A6%AC_%EB%82%B4%EA%B0%81_%EA%B0%9C%EC%A1%B0%EB%82%B4%EA%B0%81_(%EC%A4%91%EC%95%99_%EC%84%B1%EC%B2%AD_%EA%B0%9C%ED%8E%B8_%EC%A0%84)" title="제2차 모리 내각 개조내각 (중앙 성청 개편 전)">제2차 모리 내각 개조내각</a>에서, 나중에 총리가 되는 <a href="/wiki/%EA%B3%A0%EC%9D%B4%EC%A6%88%EB%AF%B8_%EC%A4%80%EC%9D%B4%EC%B9%98%EB%A1%9C" title="고이즈미 준이치로">고이즈미 준이치로</a>의 추천으로<sup class="reference" id="cite_ref-naz2_7-0"><a href="#cite_note-naz2-7">[7]</a></sup> <a href="/wiki/%EC%9D%BC%EB%B3%B8_%EB%82%B4%EA%B0%81%EA%B4%80%EB%B0%A9" title="일본 내각관방">내각관방</a> 부장관에 취임한다. 모리 내각의 뒤를 이어 출범한 <a href="/wiki/2001%EB%85%84" title="2001년">2001년</a>의 <a href="/wiki/%EC%A0%9C1%EC%B0%A8_%EA%B3%A0%EC%9D%B4%EC%A6%88%EB%AF%B8_%EB%82%B4%EA%B0%81" title="제1차 고이즈미 내각">제1차 고이즈미 내각</a>에서도 유임되었다.</p>

<p><a href="/wiki/2002%EB%85%84" title="2002년">2002년</a>의 고이즈미 총리의 <a href="/wiki/%EC%A1%B0%EC%84%A0%EB%AF%BC%EC%A3%BC%EC%A3%BC%EC%9D%98%EC%9D%B8%EB%AF%BC%EA%B3%B5%ED%99%94%EA%B5%AD" title="조선민주주의인민공화국">북한</a> 방문을 수행하고, 고이즈미 총리와 <a href="/wiki/%EA%B9%80%EC%A0%95%EC%9D%BC" title="김정일">김정일</a> 국방위원장과의 정상회담에서 ‘안이한 타협은 안된다’며 강경론을 주창했다. 이후 납치 피해자 5명은 귀국했지만, 일본 내에서는 납치문제의 타협에 난항을 거듭했다. 이후 아베는 <a class="new" href="/w/index.php?title=%EB%82%98%EC%B9%B4%EC%95%BC%EB%A7%88_%EA%B5%90%EC%BD%94&amp;action=edit&amp;redlink=1" title="나카야마 교코 (없는 문서)">나카야마 교코</a>와 함께 북한에 대한 경제제재와 무력행사를 포함한 강경 노선을 고집했다. 대화 등의 신중론을 취한 의원들이 많던 가운데, 언론은 아베를 두고 ‘의연한 태도를 취하고 있다’고 미화하면서 이후 인기가 상승했다.</p>
<p><a href="/wiki/2003%EB%85%84" title="2003년">2003년</a> 9월에는 고이즈미 총리에 의해 <a href="/wiki/%EC%9E%90%EC%9C%A0%EB%AF%BC%EC%A3%BC%EB%8B%B9_(%EC%9D%BC%EB%B3%B8)" title="자유민주당 (일본)">자민당</a>의 <a class="new" href="/w/index.php?title=%EA%B0%84%EC%82%AC%EC%9E%A5_(%EC%9D%BC%EB%B3%B8,_%EC%A0%95%EB%8B%B9)&amp;action=edit&amp;redlink=1" title="간사장 (일본, 정당) (없는 문서)">간사장</a>으로 발탁되었다. 3선의 <a class="mw-redirect" href="/wiki/%EA%B5%AD%EB%AC%B4%EB%8C%80%EC%8B%A0_(%EC%9D%BC%EB%B3%B8)" title="국무대신 (일본)">국무대신</a> 경력도 없는 젊은 의원이 간사장에 취임하는 것은 전례 없는 일이었지만, 11월의 <a href="/wiki/%EC%A0%9C43%ED%9A%8C_%EC%9D%BC%EB%B3%B8_%EC%A4%91%EC%9D%98%EC%9B%90_%EC%9D%98%EC%9B%90_%EC%B4%9D%EC%84%A0%EA%B1%B0" title="제43회 일본 중의원 의원 총선거">총선거</a>를 앞두고 아베의 ‘인기’를 필요로 했던 것이라고 추측된다. 이후 자민당은 총선거에서 절대 안정 다수를 확보하는 데 성공했다.</p>
<p>간사장으로는 자민당 내에서 지속적으로 지급되던 떡값이나 얼음값(파벌의 영수가 소속원에게 지급하는 활동자금)을 폐지하고, 자민당 후보자의 공모제를 일부 도입하는 등 당내의 제도 개혁을 추진했다. 그러나 <a href="/wiki/2004%EB%85%84" title="2004년">2004년</a>의 <a href="/wiki/%EC%A0%9C20%ED%9A%8C_%EC%9D%BC%EB%B3%B8_%EC%B0%B8%EC%9D%98%EC%9B%90_%EC%9D%98%EC%9B%90_%ED%86%B5%EC%83%81%EC%84%A0%EA%B1%B0" title="제20회 일본 참의원 의원 통상선거">참의원 선거</a>에서는 목표치였던 51석의 확보에 실패하면 ‘가장 무거운 책임을 지는 방향으로 한다’고 사임을 시사했고, 결과는 49석을 확보하면서 곧 사직했다. 같은해 9월부터는 후임 간사장이었던 <a class="new" href="/w/index.php?title=%EB%8B%A4%EC%BC%80%EB%B2%A0_%EC%93%B0%ED%86%A0%EB%AC%B4&amp;action=edit&amp;redlink=1" title="다케베 쓰토무 (없는 문서)">다케베 쓰토무</a>의 강한 요청으로 간사장대리에 취임했는데, 간사장 역임자가 간사장대리라는 낮은 직위로 가는 것도 이례적인 일이었다. 이후 간사장대리로 맞이한 <a href="/wiki/%EC%A0%9C44%ED%9A%8C_%EC%9D%BC%EB%B3%B8_%EC%A4%91%EC%9D%98%EC%9B%90_%EC%9D%98%EC%9B%90_%EC%B4%9D%EC%84%A0%EA%B1%B0" title="제44회 일본 중의원 의원 총선거">2005년 중의원 선거</a>에서는 여러 선거구에서 상대방의 낙선을 노린 고이즈미의 자객을 공천했다.</p>
<p>고이즈미 정권 말부터 ‘포스트 고이즈미’의 가장 강력한 후보로 일컬어졌고, <a href="/wiki/2005%EB%85%84" title="2005년">2005년</a> <a href="/wiki/10%EC%9B%94_31%EC%9D%BC" title="10월 31일">10월 31일</a>에 발족한 <a class="mw-redirect" href="/wiki/%EC%A0%9C3%EC%B0%A8_%EA%B3%A0%EC%9D%B4%EC%A6%88%EB%AF%B8_%EB%82%B4%EA%B0%81_%EA%B0%9C%EC%A1%B0%EB%82%B4%EA%B0%81" title="제3차 고이즈미 내각 개조내각">제3차 고이즈미 내각 개조내각</a>에서는 <a class="mw-redirect" href="/wiki/%EB%82%B4%EA%B0%81%EA%B4%80%EB%B0%A9%EC%9E%A5%EA%B4%80" title="내각관방장관">내각관방장관</a>에 취임한다.</p>
<p><a href="/wiki/2006%EB%85%84" title="2006년">2006년</a> <a href="/wiki/9%EC%9B%94_1%EC%9D%BC" title="9월 1일">9월 1일</a>에는 자민당 총재선거에 출마할 것을 발표하고, <a href="/wiki/%EC%9D%BC%EB%B3%B8%EA%B5%AD_%ED%97%8C%EB%B2%95" title="일본국 헌법">일본국 헌법</a>의 개정과 교육 개혁, 재정 건전화를 전면에 내세웠다. 이후 고이즈미 총리의 임기가 끝나고, <a href="/wiki/2006%EB%85%84" title="2006년">2006년</a> <a href="/wiki/9%EC%9B%94_20%EC%9D%BC" title="9월 20일">9월 20일</a> 실시된 총재 선거 투표에서 전체 703표(국회의원 403, 당원 300표) 가운데 464표를 얻어 경쟁 후보인 <a href="/wiki/%EC%95%84%EC%86%8C_%EB%8B%A4%EB%A1%9C" title="아소 다로">아소 다로</a> 외상과 <a href="/wiki/%EB%8B%A4%EB%8B%88%EA%B0%80%ED%82%A4_%EC%82%AC%EB%8B%A4%EC%B9%B4%EC%A6%88" title="다니가키 사다카즈">다니가키 사다카즈</a> 재무상을 큰 표차로 물리치고 <a class="mw-redirect" href="/wiki/%EC%9D%BC%EB%B3%B8_%EC%9E%90%EC%9C%A0%EB%AF%BC%EC%A3%BC%EB%8B%B9" title="일본 자유민주당">일본 자유민주당</a> (자민당)의 21대 총재로 선출되어, <a href="/wiki/9%EC%9B%94_26%EC%9D%BC" title="9월 26일">9월 26일</a> 국회 중.참 양원 본회의에서 제90대 총리로 지명되었다. 아베는 첫 전후 세대 총리로, <a href="/wiki/%EC%A0%9C2%EC%B0%A8_%EC%84%B8%EA%B3%84_%EB%8C%80%EC%A0%84" title="제2차 세계 대전">제2차 세계 대전</a> 이후의 총리 중에서 가장 젊다.</p>

<p>총리에 취임한 그는 <a href="/wiki/%EA%B3%A0%EC%9D%B4%EC%A6%88%EB%AF%B8_%EC%A4%80%EC%9D%B4%EC%B9%98%EB%A1%9C" title="고이즈미 준이치로">고이즈미</a> 총리의 <a href="/wiki/%EC%95%BC%EC%8A%A4%EC%BF%A0%EB%8B%88_%EC%8B%A0%EC%82%AC" title="야스쿠니 신사">야스쿠니 신사</a> 참배 문제로 중단되었던 <a href="/wiki/%EC%A4%91%ED%99%94%EC%9D%B8%EB%AF%BC%EA%B3%B5%ED%99%94%EA%B5%AD" title="중화인민공화국">중국</a>과 <a href="/wiki/%EB%8C%80%ED%95%9C%EB%AF%BC%EA%B5%AD" title="대한민국">한국</a>에 방문할 것을 발표하고, <a href="/wiki/10%EC%9B%94_8%EC%9D%BC" title="10월 8일">10월 8일</a>에는 <a class="mw-redirect" href="/wiki/%EB%B2%A0%EC%9D%B4%EC%A7%95" title="베이징">베이징</a>에서 <a href="/wiki/%ED%9B%84%EC%A7%84%ED%83%80%EC%98%A4" title="후진타오">후진타오</a> 국가주석을, 9일에는 <a class="mw-redirect" href="/wiki/%EC%84%9C%EC%9A%B8" title="서울">서울</a>에서 <a href="/wiki/%EB%85%B8%EB%AC%B4%ED%98%84" title="노무현">노무현</a> 대통령과 회담을 가져 양국과의 관계를 개선하고자 시도했다.</p>
<p>그러나 그날 <a href="/wiki/%EC%A1%B0%EC%84%A0%EB%AF%BC%EC%A3%BC%EC%A3%BC%EC%9D%98%EC%9D%B8%EB%AF%BC%EA%B3%B5%ED%99%94%EA%B5%AD" title="조선민주주의인민공화국">북한</a>이 <a class="mw-redirect" href="/wiki/%EC%A1%B0%EC%84%A0%EB%AF%BC%EC%A3%BC%EC%A3%BC%EC%9D%98%EC%9D%B8%EB%AF%BC%EA%B3%B5%ED%99%94%EA%B5%AD_%ED%95%B5%EB%AC%B8%EC%A0%9C" title="조선민주주의인민공화국 핵문제">핵실험</a>을 실시하자 그는 “(핵실험은) 일본의 안전보장에 대한 중대한 도전”이라고 평하고, <a class="mw-redirect" href="/wiki/%EA%B5%AD%EC%A0%9C%EC%97%B0%ED%95%A9" title="국제연합">국제연합</a>의 제제결의보다 엄격한 경제제재 조치를 취했다. 이후 12월에는 교육기본법의 개정과 <a href="/wiki/%EC%9D%BC%EB%B3%B8_%EB%B0%A9%EC%9C%84%EC%84%B1" title="일본 방위성">방위청</a>의 성 승격을 강행했다. 같은 달 혼마 세제회장이 공무원 기숙사의 애인 문제로 사임하고, <a class="new" href="/w/index.php?title=%EC%82%AC%EB%8B%A4_%EA%B2%90%EC%9D%B4%EC%B9%98%EB%A1%9C&amp;action=edit&amp;redlink=1" title="사다 겐이치로 (없는 문서)">사다 겐이치로</a> 행정개혁대신이 사무소 운영비 계상 문제로 사임했다.</p>
<p><a href="/wiki/2007%EB%85%84" title="2007년">2007년</a>에 열린 제166회 통상국회에서는 많은 주요 법안들을 강행 표결하여 잇달아 통과시켰다.</p>

<p>5월 초순에는 처음으로 <a href="/wiki/%EB%AF%B8%EA%B5%AD" title="미국">미국</a>을 방문하고, 고이즈미 내각 이후 미일관계가 지속적으로 굳건한 것을 어필했다. 같은 달, 이전부터 여러가지 의혹이 제기된 <a href="/wiki/%EB%A7%88%EC%93%B0%EC%98%A4%EC%B9%B4_%EB%8F%84%EC%8B%9C%EC%B9%B4%EC%93%B0" title="마쓰오카 도시카쓰">마쓰오카 도시카쓰</a> <a href="/wiki/%EC%9D%BC%EB%B3%B8%EC%9D%98_%EB%86%8D%EB%A6%BC%EC%88%98%EC%82%B0%EB%8C%80%EC%8B%A0" title="일본의 농림수산대신">농림수산상</a>이 자살하고, 연금 기록 문제도 수면 위로 떠올랐다. 이런 와중에 6월초의 지지율이 고이즈미 정권 이래 최저라는 보도가 이어졌다. 이후 <a href="/wiki/%EA%B7%9C%EB%A7%88_%ED%9B%84%EB%AF%B8%EC%98%A4" title="규마 후미오">규마 후미오</a> <a href="/wiki/%EC%9D%BC%EB%B3%B8%EC%9D%98_%EB%B0%A9%EC%9C%84%EB%8C%80%EC%8B%A0" title="일본의 방위대신">방위상</a>이 <a href="/wiki/%ED%9E%88%EB%A1%9C%EC%8B%9C%EB%A7%88_%EC%8B%9C" title="히로시마 시">히로시마</a>와 <a href="/wiki/%EB%82%98%EA%B0%80%EC%82%AC%ED%82%A4_%EC%8B%9C" title="나가사키 시">나가사키</a>의 원폭 투하를 두고 ‘어쩔 수 없다’고 발언하는 일까지 일어나자, 처음에는 문제가 없다는 자세를 취한 아베 총리는 이후 비판의 목소리가 높아지면서 규마 방위상에 대해 주의를 부탁했다. 다음날에는 규마 방위상이 사임했다.</p>
<p><a href="/wiki/%EC%A0%9C21%ED%9A%8C_%EC%9D%BC%EB%B3%B8_%EC%B0%B8%EC%9D%98%EC%9B%90_%EC%9D%98%EC%9B%90_%ED%86%B5%EC%83%81%EC%84%A0%EA%B1%B0" title="제21회 일본 참의원 의원 통상선거">제21회 일본 참의원 의원 통상선거</a>를 앞두고는 자살한 마쓰오카의 후임 <a class="new" href="/w/index.php?title=%EC%95%84%EC%B9%B4%EA%B8%B0_%EB%85%B8%EB%A6%AC%ED%9E%88%EC%BD%94&amp;action=edit&amp;redlink=1" title="아카기 노리히코 (없는 문서)">아카기 노리히코</a> 농림수산상의 사무소 비용 문제가 제기되었다. 이어 <a href="/wiki/%EB%8B%88%EA%B0%80%ED%83%80_%ED%98%84_%EC%A3%BC%EC%97%90%EC%93%B0_%EC%98%A4%ED%82%A4_%EC%A7%80%EC%A7%84" title="니가타 현 주에쓰 오키 지진">니가타 현 주에쓰 오키 지진</a>이 발생하면서 유세를 중단하고 현지로 향했다. 선거에서는 연금 문제의 해결과, ‘개혁할 수 없는 야당보다, 책임정당 자민당이 개혁의 실행력을 가지고 있다’고 호소했지만 결과는 연립 정권의 <a class="mw-redirect" href="/wiki/%EA%B3%B5%EB%AA%85%EB%8B%B9_(%EC%9D%BC%EB%B3%B8)" title="공명당 (일본)">공명당</a>과 합쳐도 과반수가 되지 않는 역사적인 대패를 맞았다.</p>
<p><a href="/wiki/7%EC%9B%94_30%EC%9D%BC" title="7월 30일">7월 30일</a> 오전, 그는 “유감스러운 결과가 되었다. 지지해 준 사람에게는 미안하다. 국민의 목소리를 받아들이면서 앞으로도 책임을 수행해 가겠다”고 정권을 계속할 방침을 발표했다. 이어 <a href="/wiki/%EB%82%98%EC%B9%B4%EA%B0%80%EC%99%80_%ED%9E%88%EB%8D%B0%EB%82%98%EC%98%A4" title="나카가와 히데나오">나카가와 히데나오</a> <a href="/wiki/%EC%9E%90%EC%9C%A0%EB%AF%BC%EC%A3%BC%EB%8B%B9_(%EC%9D%BC%EB%B3%B8)" title="자유민주당 (일본)">자민당</a> <a class="new" href="/w/index.php?title=%EA%B0%84%EC%82%AC%EC%9E%A5_(%EC%9D%BC%EB%B3%B8,_%EC%A0%95%EB%8B%B9)&amp;action=edit&amp;redlink=1" title="간사장 (일본, 정당) (없는 문서)">간사장</a>은 “총재·총리의 결의를 일원이 되어 지지하고 진행하자”고 발표해, 아베 총리를 지지하는 방침을 결정했다.<sup class="reference" id="cite_ref-8"><a href="#cite_note-8">[8]</a></sup></p>
<p>그러나 새로 조직한 <a href="/wiki/%EC%A0%9C1%EC%B0%A8_%EC%95%84%EB%B2%A0_%EC%8B%A0%EC%A1%B0_%EB%82%B4%EA%B0%81_(%EA%B0%9C%EC%A1%B0)" title="제1차 아베 신조 내각 (개조)">제1차 아베 신조 개조내각</a>이 출범하자마자 각료의 문제가 제기되면서 구심력을 잃자, 그는 <a href="/wiki/9%EC%9B%94_9%EC%9D%BC" title="9월 9일">9월 9일</a>에 <a href="/wiki/%EC%98%A4%EC%8A%A4%ED%8A%B8%EB%A0%88%EC%9D%BC%EB%A6%AC%EC%95%84" title="오스트레일리아">오스트레일리아</a>의 <a href="/wiki/%EC%8B%9C%EB%93%9C%EB%8B%88" title="시드니">시드니</a>에서 개최된 <a class="mw-redirect" href="/wiki/APEC" title="APEC">APEC</a> 수뇌회의의 종료에 즈음해 열린 기자회견에서 ‘<a href="/wiki/9%EC%9B%94_10%EC%9D%BC" title="9월 10일">9월 10일</a>부터 열리는 임시국회에서 테러 특별조치법을 연장할 수 없을 때에는 <b>내각총사직</b>’을 한다고 공언했다.</p>
<h4><span class="mw-headline" id=".EA.B0.91.EC.9E.91.EC.8A.A4.EB.9F.B0_.EC.82.AC.EC.9D.98_.ED.91.9C.EB.AA.85">갑작스런 사의 표명</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%95%84%EB%B2%A0_%EC%8B%A0%EC%A1%B0&amp;action=edit&amp;section=6" title="부분 편집: 갑작스런 사의 표명">편집</a><span class="mw-editsection-bracket">]</span></span></h4>
<p>그러나 <a href="/wiki/2007%EB%85%84" title="2007년">2007년</a> <a href="/wiki/9%EC%9B%94_12%EC%9D%BC" title="9월 12일">9월 12일</a> 오후 2시, “내각총리대신에서 물러난다”는 내용의 내각총사직을 표명한 기자회견을 가졌으며, 이로 인해 같은 날로 예정되었던 대표 질문이 불가능해졌다.</p>
<p>그는 사직의 이유로 “테러특별조치법의 재연장에 대해 논의하기 위해 <a class="mw-redirect" href="/wiki/%EB%AF%BC%EC%A3%BC%EB%8B%B9_(%EC%9D%BC%EB%B3%B8)" title="민주당 (일본)">민주당</a>의 <a href="/wiki/%EC%98%A4%EC%9E%90%EC%99%80_%EC%9D%B4%EC%B9%98%EB%A1%9C" title="오자와 이치로">오자와</a> 대표와의 회담을 타진했지만, 사실상 거절당했고, 이대로 총리직을 계속 수행하는 것보다는 새 총리가 수행하는 것이 좋을 것이라고 판단했다.”였으며, “내가 총리인 것이 장애가 되고 있다”고 발언했다(다만 언급된 오자와 대표는 기자회견에서 “타진을 받은 것은 단 한 차례도 없었다”고 부정). 자신의 병이 악화되어 몸상태가 좋지 못하다는 이유도 있었다고 <a href="/wiki/%EC%9A%94%EC%82%AC%EB%85%B8_%EA%B0%80%EC%98%A4%EB%A3%A8" title="요사노 가오루">요사노 가오루</a> 내각관방장관이 회견을 통해 밝히기도 했다.</p>
<p>그러나 임시국회가 개막해 내정과 외교에 중요 과제가 산적한 상황에서, 국회에서 소신을 표명한 이틀 만에 물러난다는 뜻을 발표한 것은 무책임하다는 의견이 잇따랐다. 한편 <a href="/wiki/%EC%95%84%EC%86%8C_%EB%8B%A4%EB%A1%9C" title="아소 다로">아소 다로</a> 간사장은 같은 날의 회견에서 기자의 “총리는 언제 사임을 결심한 것인가”라는 질문에서 “이틀 정도 전이라고 하기도 그렇고, 어제라고 하기도 그렇고…”라고 발언해 이틀 전쯤에는 이미 아베 총리가 조만간 사임할 것이라는 사실을 알고 있었음을 밝혔다.</p>
<p>사의를 표한 다음 날인 <a href="/wiki/9%EC%9B%94_13%EC%9D%BC" title="9월 13일">9월 13일</a>에는 <a href="/wiki/%EA%B2%8C%EC%9D%B4%EC%98%A4%EA%B8%B0%EC%A3%BC%EC%BF%A0_%EB%8C%80%ED%95%99" title="게이오기주쿠 대학">게이오기주쿠 대학</a>병원에 입원하고, 검사 결과 위장 기능 이상의 소견이 보이며 상당히 쇠약해진 상태라고 의사단이 발표했다.</p>
<h3><span class="mw-headline" id=".EC.9E.AC.EC.A7.91.EA.B6.8C">재집권</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%95%84%EB%B2%A0_%EC%8B%A0%EC%A1%B0&amp;action=edit&amp;section=7" title="부분 편집: 재집권">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<div class="rellink boilerplate seealso"><img alt="" data-file-height="128" data-file-width="128" height="16" src="//upload.wikimedia.org/wikipedia/commons/thumb/e/ec/Crystal_Clear_app_xmag.svg/16px-Crystal_Clear_app_xmag.svg.png" srcset="//upload.wikimedia.org/wikipedia/commons/thumb/e/ec/Crystal_Clear_app_xmag.svg/24px-Crystal_Clear_app_xmag.svg.png 1.5x, //upload.wikimedia.org/wikipedia/commons/thumb/e/ec/Crystal_Clear_app_xmag.svg/32px-Crystal_Clear_app_xmag.svg.png 2x" width="16"/> <a href="/wiki/%EC%A0%9C2%EC%B0%A8_%EC%95%84%EB%B2%A0_%EC%8B%A0%EC%A1%B0_%EB%82%B4%EA%B0%81" title="제2차 아베 신조 내각">제2차 아베 신조 내각</a>, <a href="/wiki/%EC%A0%9C2%EC%B0%A8_%EC%95%84%EB%B2%A0_%EC%8B%A0%EC%A1%B0_%EB%82%B4%EA%B0%81_(%EA%B0%9C%EC%A1%B0)" title="제2차 아베 신조 내각 (개조)">제2차 아베 신조 내각 (개조)</a> 문서를 참고하십시오.</div>
<p>2012년 12월에 자유민주당의 총재로서 임했다 열린 총선거에서 재선하였으며,<sup class="reference" id="cite_ref-9"><a href="#cite_note-9">[9]</a></sup> 12월 26일 다시 총리직에 복귀했다. 중국과 한국에서는 그의 이미지가 극우 정치인으로 대단히 안 좋지만, 일본 내에서는 인덕 넘치는 보수 정치인의 이미지를 갖추고 국민의 높은 지지를 받고 있다. 2014년 9월 3일에 성립한 제2차 아베 신조 내각 개조내각은 총리를 포함한 각료의 15명이 <a href="/wiki/%EC%9D%BC%EB%B3%B8%ED%9A%8C%EC%9D%98" title="일본회의">일본회의</a> 회원들이다.<sup class="reference" id="cite_ref-.EC.A4.91.EC.95.99.EC.9D.BC.EB.B3.B420140906_10-0"><a href="#cite_note-.EC.A4.91.EC.95.99.EC.9D.BC.EB.B3.B420140906-10">[10]</a></sup></p>
<h3><span class="mw-headline" id=".EC.95.BC.EC.8A.A4.EC.BF.A0.EB.8B.88_.EC.8B.A0.EC.82.AC_.EC.A0.84.EA.B2.A9_.EC.B0.B8.EB.B0.B0">야스쿠니 신사 전격 참배</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%95%84%EB%B2%A0_%EC%8B%A0%EC%A1%B0&amp;action=edit&amp;section=8" title="부분 편집: 야스쿠니 신사 전격 참배">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>그가 총리직에 복귀한 지 1년이 되는 <a href="/wiki/2013%EB%85%84" title="2013년">2013년</a> <a href="/wiki/12%EC%9B%94_26%EC%9D%BC" title="12월 26일">12월 26일</a>, 선거로 공약한 대로, 그는 <a href="/wiki/%EC%95%BC%EC%8A%A4%EC%BF%A0%EB%8B%88_%EC%8B%A0%EC%82%AC" title="야스쿠니 신사">야스쿠니 신사</a> 참배를 행했다. 참배 후 그는 “일본을 위해 귀중한 생명을 희생한 영령에게 존숭의 뜻을 표했다.”라고 말한데 이어 “중국인들 및 한국인들의 기분을 상하게 할 생각은 털끝만큼도 없다.”고 주장하기까지 했다. 국가를 위해서 순사한 사람들의 영령을 위로하는 것은, 국가지도자로서 당연한 책무라는 것이 그의 주장이었다. 앞서 그는 1차 아베 내각 시절, 야스쿠니를 참배하지 못한 것에 대해 “통한의 극치”라고 밝힌 바 있다. <sup class="reference" id="cite_ref-11"><a href="#cite_note-11">[11]</a></sup></p>
<h2><span class="mw-headline" id=".EA.B0.80.EC.A1.B1_.EA.B4.80.EA.B3.84_.EB.B0.8F_.EA.B8.B0.ED.83.80">가족 관계 및 기타</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%95%84%EB%B2%A0_%EC%8B%A0%EC%A1%B0&amp;action=edit&amp;section=9" title="부분 편집: 가족 관계 및 기타">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p><a href="/wiki/1987%EB%85%84" title="1987년">1987년</a> <a class="mw-redirect" href="/wiki/%EB%AA%A8%EB%A6%AC%EB%82%98%EA%B0%80" title="모리나가">모리나가</a>(<span style="font-size: smaller;"><a href="/wiki/%EC%9D%BC%EB%B3%B8%EC%96%B4" title="일본어">일본어</a>:</span> <span lang="ja">森永</span>)제과 사장의 큰딸 아키에(<span style="font-size: smaller;"><a href="/wiki/%EC%9D%BC%EB%B3%B8%EC%96%B4" title="일본어">일본어</a>:</span> <span lang="ja">昭恵</span>)와 결혼하였다. 부인 아키에는 열렬한 한류 마니아로, 겨울연가에 나온 고(故) <a href="/wiki/%EB%B0%95%EC%9A%A9%ED%95%98" title="박용하">박용하</a>의 팬이다.<sup class="reference" id="cite_ref-yakiniku_12-0"><a href="#cite_note-yakiniku-12">[12]</a></sup> 아베 부부는 슬하에 자녀를 두지 않았는데, 아키에 여사는 2006년 11월 <a class="mw-redirect" href="/wiki/%EB%AC%B8%EC%98%88%EC%B6%98%EC%B6%94" title="문예춘추">문예춘추</a>에 기고한 수기에서 자신이 <a href="/wiki/%EB%B6%88%EC%9E%84" title="불임">불임</a>이라는 사실을 언론에 공개했다.<sup class="reference" id="cite_ref-13"><a href="#cite_note-13">[13]</a></sup></p>
<p>아베 신조는 우익 정치인으로서의 행보 때문에 한일 관계를 파탄으로 이끌었다는 인식이 강하지만, 한편으로는 관계 개선을 위한 행보를 보이기도 했다. 그는 인터넷 프로필에 가장 좋아하는 음식을 야키니쿠(燒肉)라고 적는가 하면, 2013년 BS아사히와의 인터뷰 및 산케이신문과의 인터뷰를 야키니쿠 식당에서 갖는 등 한국과의 관계를 의식한 행보를 보이기도 했다. 이는 별것 아닌 것처럼 보일지 모르지만, 일본에서는 야키니쿠가 한국 음식으로 인식되어 있기 때문에, <a href="/wiki/%EB%B0%95%EA%B7%BC%ED%98%9C" title="박근혜">박근혜</a> 대통령이 일식당에서 인터뷰를 하는 것과도 같은 의도적인 행동이라는 평가다.<sup class="reference" id="cite_ref-yakiniku_12-1"><a href="#cite_note-yakiniku-12">[12]</a></sup></p>
<p>때로는 자신의 우익 이미지를 보완하기 위해, 한류 팬인 자신의 아내를 외교 행보에 활용하여 친한(親韓) 이미지를 부각시키도 했다.<sup class="reference" id="cite_ref-yakiniku_12-2"><a href="#cite_note-yakiniku-12">[12]</a></sup> 아베는 2014년 박근혜 대통령과 만난 자리에서 “박근혜 대통령님, 오늘 만나서 반갑습니다”라고 서툰 한국말로 인사하며 성의를 보였으나, <a class="mw-redirect" href="/wiki/%EA%B3%A0%EB%85%B8_%EB%8B%B4%ED%99%94" title="고노 담화">고노 담화</a>의 수정 시도 및 <a href="/wiki/%EC%95%BC%EC%8A%A4%EC%BF%A0%EB%8B%88_%EC%8B%A0%EC%82%AC_%EC%B0%B8%EB%B0%B0_%EC%82%AC%EA%B1%B4" title="야스쿠니 신사 참배 사건">야스쿠니 신사 참배</a> 등으로 인한 최악으로 치달은 한일관계 때문에 <a href="/wiki/%EB%B0%95%EA%B7%BC%ED%98%9C" title="박근혜">박근혜</a> 대통령에게 완전히 무시당하는 모습을 보이기도 했다.<sup class="reference" id="cite_ref-14"><a href="#cite_note-14">[14]</a></sup></p>
