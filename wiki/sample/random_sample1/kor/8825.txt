<p><b>아베디 아유 펠레</b> (<span style="font-size: smaller;"><a href="/wiki/%EC%98%81%EC%96%B4" title="영어">영어</a>:</span> <span lang="en" xml:lang="en">Abédi Ayew Pelé</span>, <a href="/wiki/1964%EB%85%84" title="1964년">1964년</a> <a href="/wiki/11%EC%9B%94_5%EC%9D%BC" title="11월 5일">11월 5일</a> ~ )는 <a href="/wiki/%EA%B0%80%EB%82%98" title="가나">가나</a>의 전직 <a href="/wiki/%EC%B6%95%EA%B5%AC" title="축구">축구</a> 선수로, 포지션은 <a href="/wiki/%EB%AF%B8%EB%93%9C%ED%95%84%EB%8D%94" title="미드필더">윙어</a>와 <a class="mw-disambig" href="/wiki/%EC%8A%A4%ED%8A%B8%EB%9D%BC%EC%9D%B4%EC%BB%A4" title="스트라이커">스트라이커</a>였다.</p>
<p><a class="new" href="/w/index.php?title=%EB%A0%88%EC%95%8C_%ED%83%80%EB%A7%90%EB%A0%88_%EC%9C%A0%EB%82%98%EC%9D%B4%ED%8B%B0%EB%93%9C&amp;action=edit&amp;redlink=1" title="레알 타말레 유나이티드 (없는 문서)">레알 타말레 유나이티드</a> (1978년~1982년, 1985년) 소속으로 데뷔했으며 <a href="/wiki/%EC%B9%B4%ED%83%80%EB%A5%B4" title="카타르">카타르</a>의 <a class="mw-redirect" href="/wiki/%EC%95%8C%EC%82%AC%EB%93%9C_%EC%8A%A4%ED%8F%AC%EC%B8%A0_%ED%81%B4%EB%9F%BD" title="알사드 스포츠 클럽">알사드</a>(1982년~1983년)와 <a href="/wiki/%EC%8A%A4%EC%9C%84%EC%8A%A4" title="스위스">스위스</a>의 <a href="/wiki/FC_%EC%B7%A8%EB%A6%AC%ED%9E%88" title="FC 취리히">FC 취리히</a>(1983년~1984년), <a href="/wiki/%EB%B2%A0%EB%83%89" title="베냉">베냉</a>의 <a class="new" href="/w/index.php?title=AS_%EB%93%9C%EB%9D%BC%EA%B3%B5_FC_%EB%93%9C_%EB%A3%A8%EC%97%90%EB%A9%94&amp;action=edit&amp;redlink=1" title="AS 드라공 FC 드 루에메 (없는 문서)">AS 드라공 FC 드 루에메</a>(1984년), <a href="/wiki/%ED%94%84%EB%9E%91%EC%8A%A4" title="프랑스">프랑스</a>의 <a href="/wiki/%EC%83%A4%EB%AC%B4%EC%95%84_%EB%8B%88%EC%98%A4%EB%A5%B4_FC" title="샤무아 니오르 FC">샤무아 니오르</a>(1986년~1987년)와 <a href="/wiki/FC_%EB%AE%90%EB%A3%A8%EC%A6%88" title="FC 뮐루즈">FC 뮐루즈</a>(1987년) 소속으로 뛰었다. <a href="/wiki/1987%EB%85%84" title="1987년">1987년</a> <a href="/wiki/%EC%98%AC%EB%9E%AD%ED%94%BC%ED%81%AC_%EB%93%9C_%EB%A7%88%EB%A5%B4%EC%84%B8%EC%9C%A0" title="올랭피크 드 마르세유">올랭피크 드 마르세유</a>로 이적했고 <a href="/wiki/1988%EB%85%84" title="1988년">1988년</a>부터 <a href="/wiki/1990%EB%85%84" title="1990년">1990년</a>까지 <a href="/wiki/%EB%A6%B4_OSC" title="릴 OSC">릴 OSC</a> 소속으로 뛰었다. <a href="/wiki/1990%EB%85%84" title="1990년">1990년</a> 올랭피크 드 마르세유로 복귀했으며 팀의 <a href="/wiki/%EC%9C%A0%EB%9F%AC%ED%94%BC%EC%96%B8%EC%BB%B5_1990-91" title="유러피언컵 1990-91">유러피언컵 1990-91</a> 준우승과 <a href="/wiki/UEFA_%EC%B1%94%ED%94%BC%EC%96%B8%EC%8A%A4%EB%A6%AC%EA%B7%B8_1992-93" title="UEFA 챔피언스리그 1992-93">UEFA 챔피언스리그 1992-93</a> 우승을 이끌었다. 프랑스의 <a href="/wiki/%EC%98%AC%EB%9E%AD%ED%94%BC%ED%81%AC_%EB%A6%AC%EC%98%B9" title="올랭피크 리옹">올랭피크 리옹</a>(1993년~1994년)과 <a href="/wiki/%EC%9D%B4%ED%83%88%EB%A6%AC%EC%95%84" title="이탈리아">이탈리아</a>의 <a href="/wiki/%ED%86%A0%EB%A6%AC%EB%85%B8_FC" title="토리노 FC">토리노 FC</a>(1994년~1996년), <a href="/wiki/%EB%8F%85%EC%9D%BC" title="독일">독일</a>의 <a href="/wiki/TSV_1860_%EB%AE%8C%ED%97%A8" title="TSV 1860 뮌헨">TSV 1860 뮌헨</a>(1996년~1998년), <a href="/wiki/%EC%95%84%EB%9E%8D%EC%97%90%EB%AF%B8%EB%A6%AC%ED%8A%B8" title="아랍에미리트">아랍에미리트</a>의 <a href="/wiki/%EC%95%8C%EC%95%84%EC%9D%B8_FC" title="알아인 FC">알아인 FC</a>(1998년~2000년) 소속으로 뛰었으며 <a href="/wiki/2000%EB%85%84" title="2000년">2000년</a> 현역 은퇴를 선언했다.</p>
<p><a href="/wiki/1981%EB%85%84" title="1981년">1981년</a>부터 <a href="/wiki/1998%EB%85%84" title="1998년">1998년</a>까지 <a href="/wiki/%EA%B0%80%EB%82%98_%EC%B6%95%EA%B5%AC_%EA%B5%AD%EA%B0%80%EB%8C%80%ED%91%9C%ED%8C%80" title="가나 축구 국가대표팀">가나 대표팀</a>에서 73경기에 출전했고 33득점을 기록했으며 <a href="/wiki/1982%EB%85%84" title="1982년">1982년</a> 가나의 <a href="/wiki/%EC%95%84%ED%94%84%EB%A6%AC%EC%B9%B4_%EB%84%A4%EC%9D%B4%EC%85%98%EC%8A%A4%EC%BB%B5" title="아프리카 네이션스컵">아프리카 네이션스컵</a> 우승에 공헌했다. <a href="/wiki/1991%EB%85%84" title="1991년">1991년</a>과 <a href="/wiki/1992%EB%85%84" title="1992년">1992년</a>, <a href="/wiki/1993%EB%85%84" title="1993년">1993년</a> 세 차례나 <a href="/wiki/%EC%95%84%ED%94%84%EB%A6%AC%EC%B9%B4_%EC%98%AC%ED%95%B4%EC%9D%98_%EC%B6%95%EA%B5%AC_%EC%84%A0%EC%88%98" title="아프리카 올해의 축구 선수">아프리카 올해의 축구 선수</a>로 선정되었으며 <a href="/wiki/2004%EB%85%84" title="2004년">2004년</a> 3월에는 <a href="/wiki/%ED%8E%A0%EB%A0%88" title="펠레">펠레</a>가 선정한 현존하는 최고의 축구 선수 목록인 <a href="/wiki/FIFA_100" title="FIFA 100">FIFA 100</a>에 선정되기도 했다. 그의 아들인 <a href="/wiki/%EC%95%95%EB%91%98_%EB%9D%BC%ED%9E%98_%EC%95%84%EC%9C%A0" title="압둘 라힘 아유">압둘 라힘</a>과 <a href="/wiki/%EC%95%99%EB%93%9C%EB%A0%88_%EC%95%84%EC%9C%A0" title="앙드레 아유">앙드레</a>, <a href="/wiki/%EC%A1%B0%EB%A5%B4%EB%8B%B9_%EC%95%84%EC%9C%A0" title="조르당 아유">조르당</a> 모두 축구 선수로 뛰고 있다.</p>
