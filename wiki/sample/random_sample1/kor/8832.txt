<p><b>아사노 나가아키라</b>(浅野長晟, <a class="mw-disambig" href="/wiki/%EB%8D%B4%EC%87%BC" title="덴쇼">덴쇼</a> 14년 <a href="/wiki/%EC%9D%8C%EB%A0%A5_1%EC%9B%94_28%EC%9D%BC" title="음력 1월 28일">음력 1월 28일</a>(<a href="/wiki/1586%EB%85%84" title="1586년">1586년</a> <a href="/wiki/3%EC%9B%94_18%EC%9D%BC" title="3월 18일">3월 18일</a>) ~ <a href="/wiki/%EA%B0%84%EC%97%90%EC%9D%B4" title="간에이">간에이</a> 9년 <a href="/wiki/%EC%9D%8C%EB%A0%A5_9%EC%9B%94_3%EC%9D%BC" title="음력 9월 3일">음력 9월 3일</a>(<a href="/wiki/1632%EB%85%84" title="1632년">1632년</a> <a href="/wiki/10%EC%9B%94_16%EC%9D%BC" title="10월 16일">10월 16일</a>))는 <a class="mw-redirect" href="/wiki/%EC%95%84%EC%A6%88%EC%B9%98%EB%AA%A8%EB%AA%A8%EC%95%BC%EB%A7%88_%EC%8B%9C%EB%8C%80" title="아즈치모모야마 시대">아즈치모모야마 시대</a>에서 <a href="/wiki/%EC%97%90%EB%8F%84_%EC%8B%9C%EB%8C%80" title="에도 시대">에도 시대</a>에 걸쳐 산 <a href="/wiki/%EB%8B%A4%EC%9D%B4%EB%AC%98" title="다이묘">다이묘</a>이다.</p>
<h2><span class="mw-headline" id=".EC.83.9D.EC.95.A0">생애</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%95%84%EC%82%AC%EB%85%B8_%EB%82%98%EA%B0%80%EC%95%84%ED%82%A4%EB%9D%BC&amp;action=edit&amp;section=1" title="부분 편집: 생애">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p><a href="/wiki/%EC%98%A4%EB%AF%B8_%EA%B5%AD" title="오미 국">오미</a> 사카모토(현 <a href="/wiki/%EC%8B%9C%EA%B0%80_%ED%98%84" title="시가 현">시가 현</a> <a href="/wiki/%EC%98%A4%EC%93%B0_%EC%8B%9C" title="오쓰 시">오쓰 시</a>)에서 태어났다. <a href="/wiki/%EB%B6%84%EB%A1%9C%EC%BF%A0" title="분로쿠">분로쿠</a> 3년(<a href="/wiki/1594%EB%85%84" title="1594년">1594년</a>)부터 <a href="/wiki/%EB%8F%84%EC%9A%94%ED%86%A0%EB%AF%B8_%ED%9E%88%EB%8D%B0%EC%9A%94%EC%8B%9C" title="도요토미 히데요시">도요토미 히데요시</a>의 가신으로 임관해 3000석을 받았다. <a href="/wiki/%EA%B2%8C%EC%9D%B4%EC%B4%88" title="게이초">게이초</a> 5년(<a href="/wiki/1600%EB%85%84" title="1600년">1600년</a>) <a href="/wiki/%EC%84%B8%ED%82%A4%EA%B0%80%ED%95%98%EB%9D%BC_%EC%A0%84%ED%88%AC" title="세키가하라 전투">세키가하라 전투</a> 후 정권을 장악한 <a href="/wiki/%EB%8F%84%EC%BF%A0%EA%B0%80%EC%99%80_%EC%9D%B4%EC%97%90%EC%95%BC%EC%8A%A4" title="도쿠가와 이에야스">도쿠가와 이에야스</a> 밑에 들어가 <a href="/wiki/%EB%B9%97%EC%B6%94_%EA%B5%AD" title="빗추 국">빗추</a> <a class="new" href="/w/index.php?title=%EC%95%84%EC%8B%9C%EB%AA%A8%EB%A6%AC_%EB%B2%88&amp;action=edit&amp;redlink=1" title="아시모리 번 (없는 문서)">아시모리 번</a>의 2만 4000석을 받는다. 게이초 18년(<a href="/wiki/1613%EB%85%84" title="1613년">1613년</a>), 형 <a href="/wiki/%EC%95%84%EC%82%AC%EB%85%B8_%EC%9A%94%EC%8B%9C%EB%82%98%EA%B0%80" title="아사노 요시나가">요시나가</a>가 후사를 두지 못하고 병사하자 <a href="/wiki/%EA%B8%B0%EC%9D%B4_%EA%B5%AD" title="기이 국">기이</a> <a href="/wiki/%EA%B8%B0%EC%8A%88_%EB%B2%88" title="기슈 번">기슈 번</a>을 물려받아 <a href="/wiki/%EC%99%80%EC%B9%B4%EC%95%BC%EB%A7%88_%EC%84%B1" title="와카야마 성">와카야마 성</a>의 주인이 된다.</p>
<p><a href="/wiki/%EC%98%A4%EC%82%AC%EC%B9%B4_%EC%A0%84%ED%88%AC" title="오사카 전투">오사카 전투</a>에서 도쿠가와 측으로 참가했다. 오사카 여름의 진에서는 적장 <a class="new" href="/w/index.php?title=%EB%B0%98_%EB%82%98%EC%98%A4%EC%9C%A0%ED%82%A4&amp;action=edit&amp;redlink=1" title="반 나오유키 (없는 문서)">반 나오유키</a>를 사살하는 공을 세운다. <a href="/wiki/1619%EB%85%84" title="1619년">1619년</a> <a href="/wiki/%ED%9B%84%EC%BF%A0%EC%8B%9C%EB%A7%88_%EB%A7%88%EC%82%AC%EB%85%B8%EB%A6%AC" title="후쿠시마 마사노리">후쿠시마 마사노리</a>의 영지가 감봉되어 이봉되자, 그 영지인 <a href="/wiki/%EC%95%84%ED%82%A4_%EA%B5%AD" title="아키 국">아키</a> <a href="/wiki/%ED%9E%88%EB%A1%9C%EC%8B%9C%EB%A7%88_%EB%B2%88" title="히로시마 번">히로시마 번</a> 42만석으로 가증 이봉된다. <a href="/wiki/1632%EB%85%84" title="1632년">1632년</a>에 사망한다. 향년 48세.</p>
