<p><b>아서 드루리</b>(Arthur Drewry, <a href="/wiki/%EB%8C%80%EC%98%81_%EC%A0%9C%EA%B5%AD_%ED%9B%88%EC%9E%A5" title="대영 제국 훈장">CBE</a>, <a href="/wiki/1891%EB%85%84" title="1891년">1891년</a> <a href="/wiki/3%EC%9B%94_3%EC%9D%BC" title="3월 3일">3월 3일</a>~<a href="/wiki/1961%EB%85%84" title="1961년">1961년</a> <a href="/wiki/3%EC%9B%94_25%EC%9D%BC" title="3월 25일">3월 25일</a>)는 <a href="/wiki/%EC%98%81%EA%B5%AD" title="영국">영국</a>의 스포츠인으로, 제5대 <a class="mw-redirect" href="/wiki/%EA%B5%AD%EC%A0%9C_%EC%B6%95%EA%B5%AC_%EC%97%B0%EB%A7%B9" title="국제 축구 연맹">국제 축구 연맹</a>(FIFA) 회장을 지냈다.</p>
<h2><span class="mw-headline" id=".EC.9D.BC.EC.83.9D.EA.B3.BC_.EA.B2.BD.EB.A0.A5">일생과 경력</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%95%84%EC%84%9C_%EB%93%9C%EB%A3%A8%EB%A6%AC&amp;action=edit&amp;section=1" title="부분 편집: 일생과 경력">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p><a href="/wiki/1955%EB%85%84" title="1955년">1955년</a>부터 <a href="/wiki/1961%EB%85%84" title="1961년">1961년</a>까지 <a class="mw-redirect" href="/wiki/%EC%9E%89%EA%B8%80%EB%9E%9C%EB%93%9C_%EC%B6%95%EA%B5%AC_%ED%98%91%ED%9A%8C" title="잉글랜드 축구 협회">잉글랜드 축구 협회</a>의 회장으로 있었으며, <a href="/wiki/%ED%92%8B%EB%B3%BC_%EB%A6%AC%EA%B7%B8" title="풋볼 리그">풋볼 리그</a>의 회장과 <a href="/wiki/%EA%B7%B8%EB%A6%BC%EC%A6%88%EB%B9%84_%ED%83%80%EC%9A%B4_FC" title="그림즈비 타운 FC">그림즈비 타운 FC</a>의 구단주를 지냈다.</p>
<p><a href="/wiki/1955%EB%85%84" title="1955년">1955년</a> <a href="/wiki/6%EC%9B%94_7%EC%9D%BC" title="6월 7일">6월 7일</a>, <a href="/wiki/%EB%A6%AC%EC%8A%A4%EB%B3%B8" title="리스본">리스본</a>에서 열린 국제 축구 연맹 회의에서 회장에 당선되었다.</p>
