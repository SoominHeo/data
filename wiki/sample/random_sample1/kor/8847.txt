<p><b>아서 케일리</b>(<span style="font-size: smaller;"><a href="/wiki/%EC%98%81%EC%96%B4" title="영어">영어</a>:</span> <span lang="en" xml:lang="en">Arthur Cayley</span> <small><a href="/wiki/%EC%9C%84%ED%82%A4%EB%B0%B1%EA%B3%BC:IPA" title="위키백과:IPA">IPA</a>: </small><span class="IPA" title="IPA 발음">[ˈɑː(ɹ)θə(ɹ) ˈkeɪli]</span> <a class="mw-disambig" href="/wiki/FRS" title="FRS">FRS</a>, <a href="/wiki/1821%EB%85%84" title="1821년">1821년</a> <a href="/wiki/8%EC%9B%94_16%EC%9D%BC" title="8월 16일">8월 16일</a>~<a href="/wiki/1895%EB%85%84" title="1895년">1895년</a> <a href="/wiki/1%EC%9B%94_26%EC%9D%BC" title="1월 26일">1월 26일</a>)는 <a href="/wiki/%EC%98%81%EA%B5%AD" title="영국">영국</a>의 <a href="/wiki/%EB%B2%95%EB%A5%A0%EA%B0%80" title="법률가">법률가</a>이자 <a href="/wiki/%EC%88%98%ED%95%99%EC%9E%90" title="수학자">수학자</a>이다. 그는 대학졸업후 25세에 14년동안 법률가로서 활동하였다.<sup class="reference" id="cite_ref-1"><a href="#cite_note-1">[1]</a></sup> 그후 40대이후에 드디어 본격적으로 전문수학자로서 활동하기 시작했다.</p>
<p><a class="mw-redirect" href="/wiki/%EC%83%81%EB%8C%80%EC%84%B1_%EC%9D%B4%EB%A1%A0" title="상대성 이론">상대성 이론</a>에서 <a href="/wiki/4%EC%B0%A8%EC%9B%90" title="4차원">4차원</a>의 개념을 뚜렷하게 만들고, 기하 공간이 <a class="mw-disambig" href="/wiki/%EC%A0%90" title="점">점</a>과 <a class="mw-disambig" href="/wiki/%EC%84%A0" title="선">선</a>으로만 이루어진다고 한정하는 것을 벗어나게 했다. 또 행렬의 대수를 발전시켰고, <a href="/wiki/%EA%B8%B0%ED%95%98%ED%95%99" title="기하학">기하학</a>에서도 많은 업적을 쌓았다. <a href="/wiki/1884%EB%85%84" title="1884년">1884년</a> 드 모르간상을 수상했다.</p>
<p><a href="/wiki/%EC%9C%8C%EB%A6%AC%EC%97%84_%EB%A1%9C%EC%96%B8_%ED%95%B4%EB%B0%80%ED%84%B4" title="윌리엄 로언 해밀턴">윌리엄 로언 해밀턴</a>의 제자로서 해밀턴과 함께 <a href="/wiki/%EC%BC%80%EC%9D%BC%EB%A6%AC-%ED%95%B4%EB%B0%80%ED%84%B4_%EC%A0%95%EB%A6%AC" title="케일리-해밀턴 정리">케일리-해밀턴 정리</a>를 발견하고, 해밀턴의 <a href="/wiki/%EC%82%AC%EC%9B%90%EC%88%98" title="사원수">사원수</a>를 발전시켜서 <a href="/wiki/%ED%8C%94%EC%9B%90%EC%88%98" title="팔원수">팔원수</a>를 고안하기도 했다. 윌리엄 로언 해밀턴의 친구인 아일랜드의 수학자 존 토머스 그레이브스(영어: John Thomas Graves, 1806~1870)는 사원수를 확장하려는 시도 끝에 <a href="/wiki/%EB%8D%B0%EA%B2%90%EC%9D%98_%EC%97%AC%EB%8D%9F_%EC%A0%9C%EA%B3%B1%EC%88%98_%ED%95%AD%EB%93%B1%EC%8B%9D" title="데겐의 여덟 제곱수 항등식">데겐의 여덟 제곱수 항등식</a>을 재발견하였고, 이를 기반으로 한 팔원수를 고안하였다.<sup class="reference" id="cite_ref-2"><a href="#cite_note-2">[2]</a></sup> 그러나 아서 케일리는 1845년에 이와는 독립적으로 팔원수를 발견하여 발표하였다.<sup class="reference" id="cite_ref-3"><a href="#cite_note-3">[3]</a></sup></p>
<p>아서 케일리의 팔원수는 <a href="/wiki/%EC%82%AC%EC%9B%90%EC%88%98" title="사원수">사원수</a> 대수에 <a href="/wiki/%EC%BC%80%EC%9D%BC%EB%A6%AC-%EB%94%95%EC%8A%A8_%EA%B5%AC%EC%84%B1" title="케일리-딕슨 구성">케일리-딕슨 구성</a>을 가하여 얻어진다.<sup class="reference" id="cite_ref-4"><a href="#cite_note-4">[4]</a></sup></p>
<p><a href="/wiki/%EC%A0%9C%EC%9E%84%EC%8A%A4_%EC%A1%B0%EC%A7%80%ED%94%84_%EC%8B%A4%EB%B2%A0%EC%8A%A4%ED%84%B0" title="제임스 조지프 실베스터">제임스 조지프 실베스터</a>와는 친구로, 실베스터가 런던을 방문할때면, 함께 <a class="extiw" href="https://en.wikipedia.org/wiki/Lincoln%27s_Inn" title="w:Lincoln's Inn">Lincoln's Inn, London</a>(런던법학원,영국법조원 비영리협회)에서 자주 거닐며, <a class="extiw" href="https://en.wikipedia.org/wiki/Invariant_theory" title="w:Invariant theory">불변식론</a>을 논의하곤 했다. <sup class="reference" id="cite_ref-5"><a href="#cite_note-5">[5]</a></sup></p>
<p><a href="/wiki/%EB%8B%A4%EB%B9%84%ED%8A%B8_%ED%9E%90%EB%B2%A0%EB%A5%B4%ED%8A%B8" title="다비트 힐베르트">다비트 힐베르트</a>가 힐베르트의 불변식론을 전개할때 케일리의 <a class="extiw" href="https://en.wikipedia.org/wiki/Cayley%27s_%CE%A9_process" title="w:Cayley's Ω process">오메가 프로세스</a>를 사용하였다.<sup class="reference" id="cite_ref-6"><a href="#cite_note-6">[6]</a></sup></p>
