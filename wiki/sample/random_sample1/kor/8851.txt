<p><b>아셀라 익스프레스</b>(<span style="font-size: smaller;"><a href="/wiki/%EC%98%81%EC%96%B4" title="영어">영어</a>:</span> <span lang="en" xml:lang="en">Acela Express</span>, <span style="font-size: smaller;"><a href="/wiki/%EC%98%81%EC%96%B4" title="영어">영어</a>:</span> <span lang="en" xml:lang="en">Acela</span>)는 <a href="/wiki/%EC%95%94%ED%8A%B8%EB%9E%99" title="암트랙">암트랙</a>이 <a class="mw-redirect" href="/wiki/%EC%9B%8C%EC%8B%B1%ED%84%B4_D.C" title="워싱턴 D.C">워싱턴 D.C</a>부터 <a href="/wiki/%EB%B3%B4%EC%8A%A4%ED%84%B4" title="보스턴">보스턴</a>까지 운영하는 고속 틸팅열차이다. 차체설계는 곡선궤도에서 열차를 기울게 해 속도를 유지할 수 있게 한다. <a class="mw-redirect" href="/wiki/%EB%B6%81%EB%AF%B8" title="북미">북미</a>에서 운행하는 유일한 <a href="/wiki/%EA%B3%A0%EC%86%8D%EC%B2%A0%EB%8F%84" title="고속철도">고속철도</a>로 북동 지역의 수송량을 적절히 감당하고 있다. 속도는 선로 상태에 따라 120km/h에서 최고 241km/h에 달한다. 하지만 아직 평균속도는 지구상 다른 고속철도에 비해 현저히 느린 편이다. 이러한 한계점에도 불구하고 국제유가 상승에 의한 항공료상승, 인근지역의 만성적인 교통정체, 그리고 강화된 공항보안검색은 아셀라 익스프레스의 경쟁력을 상승시켰다.</p>

<h2><span class="mw-headline" id=".EA.B0.9C.EC.9A.94">개요</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%95%84%EC%85%80%EB%9D%BC_%EC%9D%B5%EC%8A%A4%ED%94%84%EB%A0%88%EC%8A%A4&amp;action=edit&amp;section=1" title="부분 편집: 개요">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>현재 <a href="/wiki/%EC%9E%90%EB%8F%99%EC%B0%A8" title="자동차">자동차</a> 대국인 <a href="/wiki/%EB%AF%B8%EA%B5%AD" title="미국">미국</a>에서 <a href="/wiki/%EC%B2%A0%EB%8F%84" title="철도">철도</a>의 여객은 대도시 근교 구간을 제외하고 이용자가 거의 전무한 대신 도시 간의 이동은 주로 <a href="/wiki/%EB%B9%84%ED%96%89%EA%B8%B0" title="비행기">비행기</a>가 운항되고 있다. <a href="/wiki/%EC%95%94%ED%8A%B8%EB%9E%99" title="암트랙">암트랙</a>은 <a href="/wiki/%EB%AF%B8%EA%B5%AD" title="미국">미국</a> 동부를 중심으로 노선을 가지고 지금까지 <a href="/wiki/%EB%89%B4%EC%9A%95" title="뉴욕">뉴욕</a> ~ <a href="/wiki/%EC%9B%8C%EC%8B%B1%ED%84%B4_D.C." title="워싱턴 D.C.">워싱턴 D.C.</a> 사이를 잇는 <a href="/wiki/%EB%A9%94%ED%8A%B8%EB%A1%9C_%EB%9D%BC%EC%9D%B4%EB%84%88" title="메트로 라이너">메트로 라이너</a><sup class="reference" id="cite_ref-1"><a href="#cite_note-1">[1]</a></sup>를 시작한 특급 열차를 운행하고 있었지만, 비즈니스 여행객의 이용객을 제외하면 사실상 침체에 시달리고 있었다. 기사 회생을 노리고 <a href="/wiki/%EB%AF%B8%EA%B5%AD" title="미국">미국</a>에서 가장 인구가 밀집한 지역을 연결하는 <a href="/wiki/%EB%85%B8%EC%8A%A4%EC%9D%B4%EC%8A%A4%ED%8A%B8_%EC%BD%94%EB%A6%AC%EB%8D%94" title="노스이스트 코리더">노스이스트 코리더</a> 구간인 <a href="/wiki/%EB%B3%B4%EC%8A%A4%ED%84%B4" title="보스턴">보스턴</a> ~ <a href="/wiki/%EB%89%B4%EC%9A%95" title="뉴욕">뉴욕</a> ~ <a href="/wiki/%EC%9B%8C%EC%8B%B1%ED%84%B4_D.C." title="워싱턴 D.C.">워싱턴 D.C.</a> 사이에 <a class="mw-redirect" href="/wiki/%EA%B3%A0%EC%86%8D%EC%97%B4%EC%B0%A8" title="고속열차">고속열차</a>를 운행할 계획을 세웠는데 <a href="/wiki/2000%EB%85%84" title="2000년">2000년</a>에 시작한 아셀라 익스프레스로 틸팅 기능을 한 열차로 운행하고 있다.</p>

<h2><span class="mw-headline" id=".EC.A0.91.EA.B0.9D_.EC.84.A4.EB.B9.84">접객 설비</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%95%84%EC%85%80%EB%9D%BC_%EC%9D%B5%EC%8A%A4%ED%94%84%EB%A0%88%EC%8A%A4&amp;action=edit&amp;section=3" title="부분 편집: 접객 설비">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>좌석은 퍼스트 클래스와 비즈니스 클래스만 되어 있고 다른 열차의 일반석과 가격과 서비스에서 차이를 내고있다. 퍼스트 클래스는 다른 암트랙 의 침대차뿐만 아니라 식사와 술을 포함한 음료가 서비스가 된다. 아세라 차내 판매 한정 샌드위치도 인기가 높다. 전 열차가 기본적으로 비즈니스 석 이상으로만 좌석이 편성되어 있다. 비즈니스 석은 2*2 배열로 되어 있다. 1등석(퍼스트석)은 2*1 배열이며, 운임은 비즈니스석보다 1.5배 정도 비싸다. 6호차에는 별도의 스낵 바가 편성되어 있다.</p>
<h2><span class="mw-headline" id=".EC.B0.A8.EB.9F.89">차량</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%95%84%EC%85%80%EB%9D%BC_%EC%9D%B5%EC%8A%A4%ED%94%84%EB%A0%88%EC%8A%A4&amp;action=edit&amp;section=4" title="부분 편집: 차량">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>차량은 <a href="/wiki/%ED%94%84%EB%9E%91%EC%8A%A4" title="프랑스">프랑스</a>의 <a href="/wiki/TGV" title="TGV">TGV</a>, <a href="/wiki/%EB%8F%85%EC%9D%BC" title="독일">독일</a>의 <a class="mw-disambig" href="/wiki/ICE" title="ICE">ICE</a>, <a href="/wiki/%EC%8A%A4%ED%8E%98%EC%9D%B8" title="스페인">스페인</a>의 <a href="/wiki/AVE" title="AVE">AVE</a>, <a href="/wiki/%EC%9D%B4%ED%83%88%EB%A6%AC%EC%95%84" title="이탈리아">이탈리아</a>의 <a href="/wiki/%ED%8E%9C%EB%8F%8C%EB%A6%AC%EB%85%B8" title="펜돌리노">펜돌리노</a> <a href="/wiki/%EC%8A%A4%EC%9B%A8%EB%8D%B4" title="스웨덴">스웨덴</a>의 <a class="mw-redirect" href="/wiki/X2000" title="X2000">X2000</a> 모델로 대부분 <a href="/wiki/%EC%9C%A0%EB%9F%BD" title="유럽">유럽</a>의 고속 열차가 후보에 올랐고, <a class="mw-disambig" href="/wiki/ICE" title="ICE">ICE</a>나 <a class="mw-redirect" href="/wiki/X2000" title="X2000">X2000</a>는 실제로 시운전까지 갔지만 최종 선정 결과 <a href="/wiki/TGV" title="TGV">TGV</a>의 기술을 기반으로 한 차량을 통해 도입되었다. 해안을 달리는 특성 때문에 차체는 스테인리스 무도장을 적용했다.</p>
<p>고속열차 전용 시설이 없는 대신 재래선을 달린다. <a href="/wiki/%EB%89%B4%EC%9A%95" title="뉴욕">뉴욕</a> ~ <a href="/wiki/%ED%95%84%EB%9D%BC%EB%8D%B8%ED%94%BC%EC%95%84" title="필라델피아">필라델피아</a> ~ <a href="/wiki/%EC%9B%8C%EC%8B%B1%ED%84%B4_D.C." title="워싱턴 D.C.">워싱턴 D.C.</a> 운행하는 동안 이전 시설이 남아있는 구간도 많기 때문에 2시간 47분이 소요된다. 한편 <a href="/wiki/%EB%B3%B4%EC%8A%A4%ED%84%B4" title="보스턴">보스턴</a> ~ <a href="/wiki/%EB%89%B4%ED%97%A4%EC%9D%B4%EB%B8%90" title="뉴헤이븐">뉴헤이븐</a> 구간에 고속 운전 대응으로 최고 속도를 도달하는 구간도 있지만, <a href="/wiki/%EB%89%B4%ED%97%A4%EC%9D%B4%EB%B8%90" title="뉴헤이븐">뉴헤이븐</a> ~ <a href="/wiki/%EB%89%B4%EC%9A%95" title="뉴욕">뉴욕</a> 사이에 느린 통근 열차와 같은 선로를 달리기 위해 속도를 줄여 가장 빠른 시간인 3시간 23분이 소요된다.</p>
<p>또한 아셀라 익스프레스의 운전 개시 후 운행했던 <a href="/wiki/%EB%A9%94%ED%8A%B8%EB%A1%9C_%EB%9D%BC%EC%9D%B4%EB%84%88" title="메트로 라이너">메트로 라이너</a>는 <a href="/wiki/2006%EB%85%84" title="2006년">2006년</a> <a href="/wiki/10%EC%9B%94_27%EC%9D%BC" title="10월 27일">10월 27일</a> 마지막 운행을 가지고 현역에서 퇴역했다. 덧붙여서 만년 메트로 라이너는 평일 1일 1회 왕복으로 운행되고 있었다.</p>
<h2><span class="mw-headline" id=".EB.AC.B8.EC.A0.9C.EC.A0.90">문제점</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%95%84%EC%85%80%EB%9D%BC_%EC%9D%B5%EC%8A%A4%ED%94%84%EB%A0%88%EC%8A%A4&amp;action=edit&amp;section=5" title="부분 편집: 문제점">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>아셀라 익스프레스는 매년 <a href="/wiki/%EB%AF%B8%EA%B5%AD" title="미국">미국</a> 의회의 비난의 대상이 되고 있다. <a href="/wiki/%EC%95%94%ED%8A%B8%EB%9E%99" title="암트랙">암트랙</a> 기사 회생을 위해 비장의 카드인 아세라 익스프레스를 사용 했지만 몇 가지 문제가 생기고 있다.</p>
<dl>
<dt>소요 시간</dt>
<dd>이 열차는 고속선이 아닌 재래선을 달리는 고속열차로 선형의 영향으로 속도에 한계가 있다.</dd>
<dt>고장</dt>
<dd><a href="/wiki/%EB%AF%B8%EA%B5%AD" title="미국">미국</a>에서 유일되는 고속열차이므로, 부품의 균열 등에 의해 종종 모든 열차가 원인 규명을 위해 운행이 중단된 적이 있다.</dd>
<dd><a href="/wiki/2002%EB%85%84" title="2002년">2002년</a> <a href="/wiki/8%EC%9B%94_15%EC%9D%BC" title="8월 15일">8월 15일</a> 좌우 흔들림 흡수 장치에 균열이 발견 되면서 전체 차량에 무기한 운행 정지했다. <a href="/wiki/2005%EB%85%84" title="2005년">2005년</a> <a href="/wiki/4%EC%9B%94_15%EC%9D%BC" title="4월 15일">4월 15일</a> 브레이크 디스크에 균열이 발견된 운행이 중단 이후 두 사례도 원인 규명 후 운전을 재개했다.</dd>
</dl>
