<p><b>아스마라</b>(<span style="font-size: smaller;"><a href="/wiki/%ED%8B%B0%EA%B7%B8%EB%A6%AC%EB%83%90%EC%96%B4" title="티그리냐어">티그리냐어</a>:</span> <span lang="ti" xml:lang="ti">ኣስመራ</span>, <i>Asmera</i>)는 <a href="/wiki/%EC%97%90%EB%A6%AC%ED%8A%B8%EB%A0%88%EC%95%84" title="에리트레아">에리트레아</a>의 <a href="/wiki/%EC%88%98%EB%8F%84" title="수도">수도</a>이다. 인구는 2009년 기준 64만 9천 명이다. 고도 2,325m에 위치한 고원 도시(高原都市)이다.</p>

<h2><span class="mw-headline" id=".EC.97.AD.EC.82.AC">역사</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%95%84%EC%8A%A4%EB%A7%88%EB%9D%BC&amp;action=edit&amp;section=1" title="부분 편집: 역사">원본 편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>도시의 역사는 기원전으로 거슬러올라갈 수 있다. 그러나 19세기말 <a href="/wiki/%EC%9D%B4%ED%83%88%EB%A6%AC%EC%95%84" title="이탈리아">이탈리아</a>의 세력이 진출하고 1897년 <a href="/wiki/%EC%9D%B4%ED%83%88%EB%A6%AC%EC%95%84%EB%A0%B9_%EC%97%90%EB%A6%AC%ED%8A%B8%EB%A0%88%EC%95%84" title="이탈리아령 에리트레아">이탈리아령 에리트레아</a>의 수도가 <a href="/wiki/%EB%A7%88%EC%82%AC%EC%99%80" title="마사와">마사와</a>에서 이 곳으로 옮겨오면서 급속히 발전하였다. <a href="/wiki/1913%EB%85%84" title="1913년">1913년</a>과 <a href="/wiki/1915%EB%85%84" title="1915년">1915년</a>의 두 차례 지진은 경미한 피해만을 주었고<sup class="reference" id="cite_ref-1"><a href="#cite_note-1">[1]</a></sup> 1930년대말 <a class="mw-redirect" href="/wiki/%EB%AC%B4%EC%86%94%EB%A6%AC%EB%8B%88" title="무솔리니">무솔리니</a> 치하의 이탈리아는 아스마라에 새로운 건물들을 연이어 건축하여 도시의 모습을 바꾸어놓았다. 아스마라는 <i>Piccola Roma</i>(작은 로마)라는 별명을 갖게 되었다.<sup class="reference" id="cite_ref-2"><a href="#cite_note-2">[2]</a></sup> 이탈리아에서 온 정착민들도 늘어났으며, 1939년 이탈리아의 통계에 의하면 도시 주민 9만 8천 명 가운데 5만 3천 명이 <a href="/wiki/%EC%9D%B4%ED%83%88%EB%A6%AC%EC%95%84%EC%9D%B8" title="이탈리아인">이탈리아인</a>이었다. 당시 에리트레아에 거주하던 이탈리아인이 7만 5천 명임을 고려하면, 아스마라는 에리트레아에 거주하던 이탈리아인이 중점적으로 몰려 있는 지역이었다.<sup class="reference" id="cite_ref-3"><a href="#cite_note-3">[3]</a></sup> <a href="/wiki/1941%EB%85%84" title="1941년">1941년</a> 영국군에 의해 이탈리아 세력은 물러나게 되었고, 1952년 연방제를 한다는 조건 아래 <a href="/wiki/%EC%97%90%ED%8B%B0%EC%98%A4%ED%94%BC%EC%95%84_%EC%A0%9C%EA%B5%AD" title="에티오피아 제국">에티오피아 제국</a>의 도시가 되었다. 그러나 에티오피아가 연방제 약속을 어기자 1961년부터 독립전쟁이 일어났고, 1991년에 독립 에리트레아의 수도가 되었다.</p>
<h2><span class="mw-headline" id=".EC.A3.BC.EB.AF.BC">주민</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%95%84%EC%8A%A4%EB%A7%88%EB%9D%BC&amp;action=edit&amp;section=2" title="부분 편집: 주민">원본 편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>아스마라는 에리트레아 최대도시이다. 티그리냐족이 시민의 77%를 차지하며, 그 다음은 티그레족(15%)이다. <a href="/wiki/%ED%8B%B0%EA%B7%B8%EB%A6%AC%EB%83%90%EC%96%B4" title="티그리냐어">티그리냐어</a>가 주로 쓰이며, <a href="/wiki/%EC%95%84%EB%9E%8D%EC%96%B4" title="아랍어">아랍어</a>, <a href="/wiki/%EC%9D%B4%ED%83%88%EB%A6%AC%EC%95%84%EC%96%B4" title="이탈리아어">이탈리아어</a>, <a href="/wiki/%EC%98%81%EC%96%B4" title="영어">영어</a>도 널리 쓰이고 이해된다. 종교는 <a class="new" href="/w/index.php?title=%EC%97%90%EB%A6%AC%ED%8A%B8%EB%A0%88%EC%95%84_%EC%A0%95%EA%B5%90%ED%9A%8C&amp;action=edit&amp;redlink=1" title="에리트레아 정교회 (없는 문서)">에리트레아 정교회</a> 신자가 시민의 60%이며, <a class="mw-redirect" href="/wiki/%EB%A1%9C%EB%A7%88_%EA%B0%80%ED%86%A8%EB%A6%AD" title="로마 가톨릭">로마 가톨릭</a> 신자 15%, <a href="/wiki/%EC%88%98%EB%8B%88%ED%8C%8C" title="수니파">수니파</a> <a class="mw-redirect" href="/wiki/%EC%9D%B4%EC%8A%AC%EB%9E%8C" title="이슬람">이슬람</a>교도가 25%이다.</p>
<h2><span class="mw-headline" id=".EC.82.B0.EC.97.85">산업</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%95%84%EC%8A%A4%EB%A7%88%EB%9D%BC&amp;action=edit&amp;section=3" title="부분 편집: 산업">원본 편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>주된 생산품은 <a href="/wiki/%EC%A7%81%EB%AC%BC" title="직물">직물</a>, <a href="/wiki/%EC%98%B7" title="옷">의류</a>, <a href="/wiki/%EB%A7%A5%EC%A3%BC" title="맥주">맥주</a>, <a href="/wiki/%EA%B5%AC%EB%91%90" title="구두">구두</a>이다.</p>
<h2><span class="mw-headline" id=".EA.B5.90.ED.86.B5">교통</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%95%84%EC%8A%A4%EB%A7%88%EB%9D%BC&amp;action=edit&amp;section=4" title="부분 편집: 교통">원본 편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>아스마라에는 <a href="/wiki/%EC%95%84%EC%8A%A4%EB%A7%88%EB%9D%BC_%EA%B5%AD%EC%A0%9C%EA%B3%B5%ED%95%AD" title="아스마라 국제공항">아스마라 국제 공항</a>이 있고, 에리트레아 제2의 도시 <a href="/wiki/%EB%A7%88%EC%82%AC%EC%99%80" title="마사와">마사와</a>와는 <a class="new" href="/w/index.php?title=%EC%97%90%EB%A6%AC%ED%8A%B8%EB%A0%88%EC%95%84_%EC%B2%A0%EB%8F%84&amp;action=edit&amp;redlink=1" title="에리트레아 철도 (없는 문서)">에리트레아 철도</a>로 연결되고 있다.</p>
