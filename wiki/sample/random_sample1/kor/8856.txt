<p><b>아스벨 키프루토 키프로프</b>(Asbel Kipruto Kiprop, <a href="/wiki/1989%EB%85%84" title="1989년">1989년</a> <a href="/wiki/6%EC%9B%94_30%EC%9D%BC" title="6월 30일">6월 30일</a> ~ )는 <a href="/wiki/%EC%BC%80%EB%83%90" title="케냐">케냐</a>의 중거리 육상 선수로 키는 188cm, 몸무게는 62kg이며 코치는 지미 뷰타(Jimmy Beauttah)이다. 그는 <a href="/wiki/%EC%97%98%EB%8F%84%EB%A0%88%ED%8A%B8" title="엘도레트">엘도레트</a> 근처의 캅틴가(Kaptinga) 마을에서 자랐으며, 데이빗 케베네이(David Kebenei)와 줄리아 케베네이 사이에서 태어났다. 그의 아버지인 데이빗 케베네이도 육상 선수로 1987년 아프리칸 게임에 참가해 4위를 차지했다. 키프로프는 13살 때 육상을 시작했으며, 훈련에 집중하기 위해 고등학교를 나왔다. 이후 엘도레트의 <a class="new" href="/w/index.php?title=%ED%82%B5%EC%B4%88%EA%B2%8C_%EC%BC%80%EC%9D%B4%EB%85%B8&amp;action=edit&amp;redlink=1" title="킵초게 케이노 (없는 문서)">킵초게 케이노</a> 캠프로 들어가 훈련했으나, <a href="/wiki/2009%EB%85%84" title="2009년">2009년</a> 룰을 어기고 여자친구를 데려왔다는 이유로 퇴출당했다.<sup class="reference" id="cite_ref-1"><a href="#cite_note-1">[1]</a></sup> 그의 동생인 빅터 킵치르킵치르 케베네이(Victor Kipchirchir Kebenei) 역시 1500m 선수이다.<sup class="reference" id="cite_ref-2"><a href="#cite_note-2">[2]</a></sup></p>

<h2><span class="mw-headline" id=".EA.B2.BD.EB.A0.A5">경력</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%95%84%EC%8A%A4%EB%B2%A8_%ED%82%A4%ED%94%84%EB%A3%A8%ED%86%A0_%ED%82%A4%ED%94%84%EB%A1%9C%ED%94%84&amp;action=edit&amp;section=1" title="부분 편집: 경력">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p><a href="/wiki/2007%EB%85%84" title="2007년">2007년</a>, 아프리칸 게임 1500m에서 금메달을 획득했으며 <a class="mw-redirect" href="/wiki/2007%EB%85%84_%EC%84%B8%EA%B3%84%EC%9C%A1%EC%83%81%EC%84%A0%EC%88%98%EA%B6%8C%EB%8C%80%ED%9A%8C" title="2007년 세계육상선수권대회">오사카 세계선수권대회</a>에서도 3분 35초 24의 개인 기록으로 4위를 차지해 화려하게 국제 무대에 데뷔했다. <a href="/wiki/2008%EB%85%84" title="2008년">2008년</a> <a href="/wiki/%EC%97%90%ED%8B%B0%EC%98%A4%ED%94%BC%EC%95%84" title="에티오피아">에티오피아</a> <a href="/wiki/%EC%95%84%EB%94%94%EC%8A%A4%EC%95%84%EB%B0%94%EB%B0%94" title="아디스아바바">아디스아바바</a>에서 열린 아프리카 육상 선수권대회에서 800m 3위, 1500m 4위를 기록했으며, 첫 올림픽인 <a href="/wiki/2008%EB%85%84_%ED%95%98%EA%B3%84_%EC%98%AC%EB%A6%BC%ED%94%BD" title="2008년 하계 올림픽">2008년 하계 올림픽</a> 1500m에서 은메달을 획득했다. 이 여세를 몰아 2008년 세계 육상 파이널에서도 2위를 기록했다. 다음 해인 <a href="/wiki/2009%EB%85%84" title="2009년">2009년</a>엔 <a href="/wiki/%EB%A1%9C%EB%A7%88" title="로마">로마</a>에서 3분 31초 20의 개인 기록을 세우며 상승세를 탔지만, <a href="/wiki/2009%EB%85%84_%EC%84%B8%EA%B3%84_%EC%9C%A1%EC%83%81_%EC%84%A0%EC%88%98%EA%B6%8C_%EB%8C%80%ED%9A%8C" title="2009년 세계 육상 선수권 대회">세계 육상 선수권 대회</a>에선 아쉽게 4위에 그쳤다.</p>
