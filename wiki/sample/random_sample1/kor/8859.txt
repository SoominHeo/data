<p><b>아스카니오 마리아 스포르차 비스콘티</b>(<span style="font-size: smaller;"><a href="/wiki/%EC%9D%B4%ED%83%88%EB%A6%AC%EC%95%84%EC%96%B4" title="이탈리아어">이탈리아어</a>:</span> <span lang="it" xml:lang="it">Ascanio Maria Sforza Visconti</span>, <a href="/wiki/1455%EB%85%84" title="1455년">1455년</a> <a href="/wiki/3%EC%9B%94_3%EC%9D%BC" title="3월 3일">3월 3일</a> - <a href="/wiki/1505%EB%85%84" title="1505년">1505년</a> <a href="/wiki/5%EC%9B%94_28%EC%9D%BC" title="5월 28일">5월 28일</a>)는 <a href="/wiki/%EB%A1%9C%EB%A7%88_%EA%B0%80%ED%86%A8%EB%A6%AD%EA%B5%90%ED%9A%8C" title="로마 가톨릭교회">로마 가톨릭교회</a>의 이탈리아인 <a href="/wiki/%EC%B6%94%EA%B8%B0%EA%B2%BD" title="추기경">추기경</a>이며, 일반적으로 로드리고 보르자가 <a href="/wiki/%EA%B5%90%ED%99%A9_%EC%95%8C%EB%A0%89%EC%82%B0%EB%8D%B0%EB%A5%B4_6%EC%84%B8" title="교황 알렉산데르 6세">교황 알렉산데르 6세</a>로 선출되었을 당시 주요 역할을 한 노련한 외교가로 알려졌다.</p>

<h2><span class="mw-headline" id=".EC.9D.BC.EB.8C.80.EA.B8.B0">일대기</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%95%84%EC%8A%A4%EC%B9%B4%EB%8B%88%EC%98%A4_%EC%8A%A4%ED%8F%AC%EB%A5%B4%EC%B0%A8&amp;action=edit&amp;section=1" title="부분 편집: 일대기">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<h3><span class="mw-headline" id=".EC.B4.88.EB.85.84_.EC.8B.9C.EC.A0.88">초년 시절</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%95%84%EC%8A%A4%EC%B9%B4%EB%8B%88%EC%98%A4_%EC%8A%A4%ED%8F%AC%EB%A5%B4%EC%B0%A8&amp;action=edit&amp;section=2" title="부분 편집: 초년 시절">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>아스카니오 스포르차는 <a href="/wiki/%EB%A1%AC%EB%B0%94%EB%A5%B4%EB%94%94%EC%95%84_%EC%A3%BC" title="롬바르디아 주">롬바르디아 주</a>의 <a href="/wiki/%ED%81%AC%EB%A0%88%EB%AA%A8%EB%82%98" title="크레모나">크레모나</a>에서 태어났다. 그의 아버지는 <a class="mw-redirect" href="/wiki/%EB%B0%80%EB%9D%BC%EB%85%B8_%EA%B3%B5%EC%9E%91" title="밀라노 공작">밀라노 공작</a> <a class="mw-redirect" href="/wiki/%ED%94%84%EB%9E%80%EC%B2%B4%EC%8A%A4%EC%BD%94_%EC%8A%A4%ED%8F%AC%EB%A5%B4%EC%B0%A8" title="프란체스코 스포르차">프란체스코 스포르차</a>이며, 어머니는 <a class="new" href="/w/index.php?title=%EB%B9%84%EC%95%88%EC%B9%B4_%EB%A7%88%EB%A6%AC%EC%95%84_%EB%B9%84%EC%8A%A4%EC%BD%98%ED%8B%B0&amp;action=edit&amp;redlink=1" title="비안카 마리아 비스콘티 (없는 문서)">비안카 마리아 비스콘티</a>이다. 또한, 두 명의 밀라노 공작 <a href="/wiki/%EA%B0%88%EB%A0%88%EC%95%84%EC%B4%88_%EB%A7%88%EB%A6%AC%EC%95%84_%EC%8A%A4%ED%8F%AC%EB%A5%B4%EC%B0%A8" title="갈레아초 마리아 스포르차">갈레아초 마리아 스포르차</a>와 <a href="/wiki/%EB%A3%A8%EB%8F%84%EB%B9%84%EC%BD%94_%EC%8A%A4%ED%8F%AC%EB%A5%B4%EC%B0%A8" title="루도비코 스포르차">루도비코 스포르차</a>는 그와 형제지간이다. <a class="new" href="/w/index.php?title=%ED%94%BC%EB%A0%88%ED%8F%AC&amp;action=edit&amp;redlink=1" title="피레포 (없는 문서)">피레포</a>는 아스카니오에게 정치와 문학을 가르친 교사이다. 그 밖에 집안에서 추기경이 된 사람들로는 <a class="new" href="/w/index.php?title=%EC%95%84%EC%8A%A4%EC%B9%B4%EB%8B%88%EC%98%A4_%EC%8A%A4%ED%8F%AC%EB%A5%B4%EC%B0%A8_%EB%94%94_%EC%82%B0%ED%83%80_%ED%94%BC%EC%98%A4%EB%9D%BC&amp;action=edit&amp;redlink=1" title="아스카니오 스포르차 디 산타 피오라 (없는 문서)">아스카니오 스포르차 디 산타 피오라</a>(1534), 알레산드로 스포르차(1565), 프란체스코 스포르차(1508), 페데리코 스포르차(1645) 등이 있다.</p>
<p>10살 때, 아스카니오는 일시적 성직급을 받는 <a class="mw-redirect" href="/wiki/%EC%95%84%EB%B9%A0%EC%8A%A4" title="아빠스">아빠스</a>로 임명되었다. 청년 시절에 아스카니오는 <a class="new" href="/w/index.php?title=%EA%B8%B0%EC%9A%A4_%EB%8D%B0%EC%8A%A4%ED%88%AC%ED%88%AC%EB%B9%8C&amp;action=edit&amp;redlink=1" title="기욤 데스투투빌 (없는 문서)">기욤 데스투투빌</a>로부터 추기경 자리를 약속받았는데, 당시 기욤은 1471년 자신의 교황좌 착좌를 위해 갈레아초 마리아 스포르차의 지지를 얻고자 그러한 약속을 한 것이었다. 최종적으로 <a href="/wiki/%EA%B5%90%ED%99%A9%EA%B4%80" title="교황관">교황관</a>이 프란체스코 델라 로베레(<a href="/wiki/%EA%B5%90%ED%99%A9_%EC%8B%9D%EC%8A%A4%ED%86%A0_4%EC%84%B8" title="교황 식스토 4세">교황 식스토 4세</a>)에게 주어지면서, 아스카니오의 추기경 서임은 지연되었다.</p>
<h3><span class="mw-headline" id=".EC.A3.BC.EA.B5.90_.EC.8B.9C.EC.A0.88">주교 시절</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%95%84%EC%8A%A4%EC%B9%B4%EB%8B%88%EC%98%A4_%EC%8A%A4%ED%8F%AC%EB%A5%B4%EC%B0%A8&amp;action=edit&amp;section=3" title="부분 편집: 주교 시절">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>아스카니오는 1479년 9월 <a class="new" href="/w/index.php?title=%ED%8C%8C%EB%B9%84%EC%95%84_%EA%B5%90%EA%B5%AC%EC%9E%A5&amp;action=edit&amp;redlink=1" title="파비아 교구장 (없는 문서)">파비아 교구장</a>으로 서품받았으며 이후 평생 교구장직을 유지하였다. 1484년 아스카니오를 대신해서 <a href="/wiki/%EB%A3%A8%EB%8F%84%EB%B9%84%EC%BD%94_%EC%8A%A4%ED%8F%AC%EB%A5%B4%EC%B0%A8" title="루도비코 스포르차">루도비코 스포르차</a>가 아스카니오의 추기경단 가입을 요구하는 크레모나 의회에 참석하였다.</p>
<h3><span class="mw-headline" id=".EC.B6.94.EA.B8.B0.EA.B2.BD_.EC.8B.9C.EC.A0.88">추기경 시절</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%95%84%EC%8A%A4%EC%B9%B4%EB%8B%88%EC%98%A4_%EC%8A%A4%ED%8F%AC%EB%A5%B4%EC%B0%A8&amp;action=edit&amp;section=4" title="부분 편집: 추기경 시절">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>1484년 3월 17일, 교황 식스토 4세는 아스카니오를 부제급 추기경에 서임하였다. 같은 해 8월 23일, 식스토 4세의 선종한 지 며칠이 지나서 아스카니오는 로마에 들어왔다. 공식 서임식은 열리지 않았으며 일부 추기경들은 곧 있을 콘클라베에 그가 참여하는 문제에 대해 강하게 반대의사를 제기하였다. 하지만, <a href="/wiki/%EA%B5%90%ED%99%A9_%EC%95%8C%EB%A0%89%EC%82%B0%EB%8D%B0%EB%A5%B4_6%EC%84%B8" title="교황 알렉산데르 6세">로드리고 보르자</a> 추기경의 개입으로 말미암아 아스카니오는 추기경으로서의 권한을 온전히 받게 되었고, 조반니 치보가 <a href="/wiki/%EA%B5%90%ED%99%A9_%EC%9D%B8%EB%85%B8%EC%B2%B8%EC%8B%9C%EC%98%A4_8%EC%84%B8" title="교황 인노첸시오 8세">교황 인노첸시오 8세</a>로 선출된 교황 선거에도 참여하였다.</p>
<p>그의 주요 업적은 <a class="mw-redirect" href="/wiki/%EB%82%98%ED%8F%B4%EB%A6%AC%EC%9D%98_%ED%8E%98%EB%A5%B4%EB%94%94%EB%82%9C%EB%8F%84_1%EC%84%B8" title="나폴리의 페르디난도 1세">나폴리의 페르디난도 1세</a>와 스포르차 일족 간에 화해를 성립시킨 것이다. 1486년 3월, 아스카니오는 로마 주재 프랑스 대사인 <a class="new" href="/w/index.php?title=%EC%9E%A5_%EB%B0%9C%EB%A4%BC&amp;action=edit&amp;redlink=1" title="장 발뤼 (없는 문서)">장 발뤼</a> 추기경과 대립하였다. 장 발뤼 추기경은 <a href="/wiki/%EA%B5%90%ED%99%A9" title="교황">교황</a>이 <a class="mw-redirect" href="/wiki/%EC%95%99%EC%A3%BC%EC%9D%98_%EB%A5%B4%EB%84%A4" title="앙주의 르네">르네 드 앙주</a>에게 <a href="/wiki/%EB%82%98%ED%8F%B4%EB%A6%AC_%EC%99%95%EA%B5%AD" title="나폴리 왕국">나폴리</a> 왕위에 대한 권리를 되찾아 주어야 한다고 주장하였다. 보통은 휘하 성직자들의 일에 간섭하기를 꺼렸던 교황 인노첸시오 8세도 두 사람의 다툼이 너무 심해지자 그만 싸움을 멈추라고 지시하였다.</p>
<p>아스카니오는 <a href="/wiki/1486%EB%85%84" title="1486년">1486년</a> <a href="/wiki/7%EC%9B%94_28%EC%9D%BC" title="7월 28일">7월 28일</a>에 <a href="/wiki/%ED%81%AC%EB%A0%88%EB%AA%A8%EB%82%98" title="크레모나">크레모나</a> 교구의 행정관으로 임명되어 그 직위를 죽을 때까지 유지하였다. 1487년에는 <a href="/wiki/%ED%8E%98%EC%82%AC%EB%A1%9C" title="페사로">페사로</a> 교구의 행정관을 겸임하여 1491년 5월까지 유지하였다.</p>
<p>아스카니오는 <a href="/wiki/%EB%82%98%ED%8F%B4%EB%A6%AC_%EC%99%95%EA%B5%AD" title="나폴리 왕국">나폴리</a>와 <a href="/wiki/%EB%B0%80%EB%9D%BC%EB%85%B8_%EA%B3%B5%EA%B5%AD" title="밀라노 공국">밀라노</a>의 동맹을 체결하려고 노력한 결과, 1492년 5월 페란테의 손자인 카푸아의 페르디난도와 <a class="new" href="/w/index.php?title=%ED%8A%B8%EB%9D%BC%EC%8A%A4%ED%85%8C%EB%B2%A0%EB%A0%88&amp;action=edit&amp;redlink=1" title="트라스테베레 (없는 문서)">트라스테베레</a>에 있는 그의 궁전에서 접견할 수 있었다. 나폴리 왕자를 주빈으로 해서 열린 만찬은 너무나도 호사스러웠는데, <a class="new" href="/w/index.php?title=%EC%8A%A4%ED%85%8C%ED%8C%8C%EB%AA%A8_%EC%9D%B8%ED%8E%98%EC%88%98%EB%9D%BC&amp;action=edit&amp;redlink=1" title="스테파모 인페수라 (없는 문서)">스테파모 인페수라</a>에 따르면, 자신이 그 만찬의 화려함에 대해 아무리 말해봤자 아무도 자기 말을 믿지 않으리라고 할 정도였다고 한다.</p>
<h3><span class="mw-headline" id=".EA.B5.90.ED.99.A9.EC.B2.AD_.EB.B6.80.EC.83.81.EC.84.9C.EC.9B.90.EC.9E.A5">교황청 부상서원장</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%95%84%EC%8A%A4%EC%B9%B4%EB%8B%88%EC%98%A4_%EC%8A%A4%ED%8F%AC%EB%A5%B4%EC%B0%A8&amp;action=edit&amp;section=5" title="부분 편집: 교황청 부상서원장">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p><a href="/wiki/1492%EB%85%84" title="1492년">1492년</a> <a href="/wiki/8%EC%9B%94" title="8월">8월</a>에 열린 콘클라베에서 아스카니오는 자신의 피선이 무위로 돌아가자 로드리고 보르자와 제휴하여 그에게 표를 던지겠다고 약속하고, 그 대가로 교황청 부상서원장 직위를 약속받았다. 아스카니오의 능수능란한 말솜씨는 다른 추기경들을 포섭하여 로드리고가 교황에 선출되는 데 도움이 되었다. 로드리고는 <a href="/wiki/%EA%B5%90%ED%99%A9_%EC%95%8C%EB%A0%89%EC%82%B0%EB%8D%B0%EB%A5%B4_6%EC%84%B8" title="교황 알렉산데르 6세">알렉산데르 6세 교황</a>이 되고 나서, 아스카니오를 교황청 부상서원장에 임명하여 그를 사실상 교황령의 총리로 만들었다. <a href="/wiki/1492%EB%85%84" title="1492년">1492년</a> <a href="/wiki/8%EC%9B%94_31%EC%9D%BC" title="8월 31일">8월 31일</a>에는 에게르 수도대주교좌의 행정관으로 임명되어 1497년 6월까지 유지하였다. 1493년, 아스카니오는 자기 집안과 교황의 집안 사이의 관계를 돈독히 하고자 자신의 조카인 페사로 백작 조반니 스포르차에게 교황의 사생아인 <a href="/wiki/%EB%A3%A8%ED%81%AC%EB%A0%88%EC%B9%98%EC%95%84_%EB%B3%B4%EB%A5%B4%EC%9E%90" title="루크레치아 보르자">루크레치아 보르자</a>와의 혼인을 주선하였다.</p>
<h3><span class="mw-headline" id=".ED.94.84.EB.9E.91.EC.8A.A4.EC.9D.98_.EC.B9.A8.EA.B3.B5">프랑스의 침공</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%95%84%EC%8A%A4%EC%B9%B4%EB%8B%88%EC%98%A4_%EC%8A%A4%ED%8F%AC%EB%A5%B4%EC%B0%A8&amp;action=edit&amp;section=6" title="부분 편집: 프랑스의 침공">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>아스카니오 추기경은 1494년 엘르네 교구를 할당받아 1495년 5월까지 사목하였다. 아스카니오와 알렉산데르 6세의 관계는 1494년 9월 <a href="/wiki/%ED%94%84%EB%9E%91%EC%8A%A4" title="프랑스">프랑스</a>가 <a href="/wiki/%EC%9D%B4%ED%83%88%EB%A6%AC%EC%95%84" title="이탈리아">이탈리아</a>를 침공하면서 교착 상태가 되었다. <a href="/wiki/%EA%B5%90%ED%99%A9_%EC%9C%A8%EB%A6%AC%EC%98%A4_2%EC%84%B8" title="교황 율리오 2세">줄리아노 델라 로베레 추기경</a>이 자신에게 대항하려 한다는 음모를 알아차린 알렉산데르 6세는 프랑스에 저항하기로 하였다. 루도비코 스포르차는 비밀리에 <a class="mw-redirect" href="/wiki/%ED%94%84%EB%9E%91%EC%8A%A4%EC%9D%98_%EC%83%A4%EB%A5%BC_8%EC%84%B8" title="프랑스의 샤를 8세">프랑스의 샤를 8세</a> 국왕과 동맹을 맺었으며, 아스카니오는 몇몇 추기경과 함께 교황을 배신하여 델라 로베레 추기경의 영향을 받아 그에게 퇴위를 요구하였다. 교황이 프랑스 국왕을 상대로 승리를 거두자, 프랑스에 버림받은 밀라노 세력과 아스카니오는 다시 한 번 바티칸에 받아들여졌다. 아스카니오는 그러나 예전처럼 교황에게 영향력을 행사하지는 못하였다. 1497년에 교황의 사생아인 <a href="/wiki/%ED%9B%84%EC%95%88_%EB%B3%B4%EB%A5%B4%EC%9E%90" title="후안 보르자">후안 보르자</a>가 칼에 찔려 살해당한 사건이 일어나자, 뒤이어 열린 추기경회의에서 아스카니오가 의심을 받아 살인 혐의로 기소되었다. 그러나 곧바로 교황은 아스카니오의 결백을 공표했다.</p>
<p>프랑스가 이탈리아를 다시 침공했을 때, 아스카니오는 루도비코 스포르차의 몰락과 구금(1500년)을 그저 무력하게 지켜보았다. <a href="/wiki/1500%EB%85%84" title="1500년">1500년</a> <a href="/wiki/6%EC%9B%94_15%EC%9D%BC" title="6월 15일">6월 15일</a>, 아스카니오는 프랑스에 호송되어 <a href="/wiki/%EB%A6%AC%EC%98%B9" title="리옹">리옹</a>에 구금되었다가 나중에 <a class="new" href="/w/index.php?title=%ED%88%AC%EB%A5%B4%EB%93%9C%EB%B6%80%EB%A5%B4%EC%A3%BC&amp;action=edit&amp;redlink=1" title="투르드부르주 (없는 문서)">투르드부르주</a>로 옮겨졌다. 1월 3일, 아스카니오는 프랑스 국왕의 허가 없이는 프랑스 땅을 떠나지 않겠다는 약속을 하고 석방되었다. 그는 1503년 9월 열린 콘클라베에 참석하여 줄리아노 델라 로베레와 프랑스 세력의 지지를 받은 <a class="new" href="/w/index.php?title=%EC%A1%B0%EB%A5%B4%EC%A3%BC_%EC%95%99%EB%B6%80%EC%95%84%EC%A6%88&amp;action=edit&amp;redlink=1" title="조르주 앙부아즈 (없는 문서)">조르주 앙부아즈</a> 추기경에 맞서 싸워 알렉산데르 6세의 후임자가 되려고 헛수고하였다. <a href="/wiki/%EA%B5%90%ED%99%A9_%EB%B9%84%EC%98%A4_3%EC%84%B8" title="교황 비오 3세">교황 비오 3세</a>(프란체스코 피콜로미니)가 즉위한 지 26일 만에 선종하자, 아스카니오 추기경은 1503년 10월 교황 선출을 위해 열린 콘클라베에 다시 참석하였으나 줄리아노 델라 로베레(<a href="/wiki/%EA%B5%90%ED%99%A9_%EC%9C%A8%EB%A6%AC%EC%98%A4_2%EC%84%B8" title="교황 율리오 2세">교황 율리오 2세</a>)에게 패배하였다. 좌절을 연거푸 경험한 아스카니오는 <a href="/wiki/1505%EB%85%84" title="1505년">1505년</a> <a href="/wiki/5%EC%9B%94_25%EC%9D%BC" title="5월 25일">5월 25일</a> 쉰 살의 나이로 로마에서 선종하였다. 율리오 2세는 <a href="/wiki/%EC%82%B0%ED%83%80_%EB%A7%88%EB%A6%AC%EC%95%84_%EB%8D%B8_%ED%8F%AC%ED%8F%B4%EB%A1%9C_%EC%84%B1%EB%8B%B9" title="산타 마리아 델 포폴로 성당">산타 마리아 델 포폴로 성당</a> 내 마조레 경당에 아스카니오 추기경의 무덤을 만들라고 지시하였다.</p>
