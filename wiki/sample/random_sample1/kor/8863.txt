<div class="mw-content-ltr" dir="ltr" id="mw-content-text" lang="ko"><script>function mfTempOpenSection(id){var block=document.getElementById("mf-section-"+id);block.className+=" open-block";block.previousSibling.className+=" open-block";}</script><p><b>아스터</b>(aster)는 유럽에서 제작한 단거리와 장거리의 대공 미사일으로 유럽판 <a class="mw-redirect mw-disambig" href="/wiki/%EC%8A%A4%ED%83%A0%EB%8B%A4%EB%93%9C_%EB%AF%B8%EC%82%AC%EC%9D%BC" title="스탠다드 미사일">스탠다드 미사일</a>이다. MBDA 이태리 66% 프랑스 탈레스 33%의 지분 참여로 개발하여 2001년 실전배치되었다.</p>

<h2><span class="mw-headline" id=".EC.97.AD.EC.82.AC">역사</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%95%84%EC%8A%A4%ED%84%B0_(%EC%9C%A0%EB%8F%84%ED%83%84)&amp;action=edit&amp;section=1" title="부분 편집: 역사">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>1980년대 <a href="/wiki/%ED%94%84%EB%9E%91%EC%8A%A4" title="프랑스">프랑스</a>와 <a href="/wiki/%EC%9D%B4%ED%83%88%EB%A6%AC%EC%95%84" title="이탈리아">이탈리아</a>에 실전배치된 것은 주로 프랑스의 <a class="mw-redirect" href="/wiki/%ED%81%AC%EB%A1%9C%ED%83%88_%EB%AF%B8%EC%82%AC%EC%9D%BC" title="크로탈 미사일">크로탈 미사일</a>, 이탈리아의 <a class="new" href="/w/index.php?title=%EC%95%A0%EC%8A%A4%ED%94%BC%EB%8D%B0&amp;action=edit&amp;redlink=1" title="애스피데 (없는 문서)">애스피데</a>, 미국의 <a class="mw-redirect" href="/wiki/RIM-7_%EC%8B%9C_%EC%8A%A4%ED%8C%A8%EB%A1%9C%EC%9A%B0" title="RIM-7 시 스패로우">RIM-7 시 스패로우</a>와 같은 단거리 대공 미사일이었다. 이 미사일들은 사정거리가 겨우 10여 킬로미터 정도였기 때문에 2000년 프랑스와 이탈리아는 국산 중거리 장거리 지대공 미사일을 개발하기로 결정했다. 이 두 나라는 미국의 <a class="mw-redirect mw-disambig" href="/wiki/%EC%8A%A4%ED%83%A0%EB%8B%A4%EB%93%9C_%EB%AF%B8%EC%82%AC%EC%9D%BC" title="스탠다드 미사일">스탠다드 미사일</a> 또는 영국의 <a class="new" href="/w/index.php?title=%EC%8B%9C_%EB%8B%A4%ED%8A%B8&amp;action=edit&amp;redlink=1" title="시 다트 (없는 문서)">시 다트</a> 미사일와 동급이며, <a href="/wiki/%EC%9D%B8%EB%8F%84" title="인도">인도</a>와 <a href="/wiki/%EB%9F%AC%EC%8B%9C%EC%95%84" title="러시아">러시아</a>가 합작 개발한 <a class="new" href="/w/index.php?title=%EB%B8%8C%EB%9D%BC%EB%AA%A8%EC%8A%A4&amp;action=edit&amp;redlink=1" title="브라모스 (없는 문서)">브라모스</a> 초음속 대함미사일을 요격할 미사일이 필요했다.</p>
<h2><span class="mw-headline" id="SAMP.2FT">SAMP/T</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%95%84%EC%8A%A4%ED%84%B0_(%EC%9C%A0%EB%8F%84%ED%83%84)&amp;action=edit&amp;section=2" title="부분 편집: SAMP/T">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<div class="thumb tright">
<div class="thumbinner" style="width:222px;"><a class="image" href="/wiki/%ED%8C%8C%EC%9D%BC:Aster_30_system.jpg"><img alt="" class="thumbimage" data-file-height="2592" data-file-width="3872" height="147" src="//upload.wikimedia.org/wikipedia/commons/thumb/a/ae/Aster_30_system.jpg/220px-Aster_30_system.jpg" srcset="//upload.wikimedia.org/wikipedia/commons/thumb/a/ae/Aster_30_system.jpg/330px-Aster_30_system.jpg 1.5x, //upload.wikimedia.org/wikipedia/commons/thumb/a/ae/Aster_30_system.jpg/440px-Aster_30_system.jpg 2x" width="220"/></a>
<div class="thumbcaption">
<div class="magnify"><a class="internal" href="/wiki/%ED%8C%8C%EC%9D%BC:Aster_30_system.jpg" title="실제 크기로"></a></div>
지상 발사형 아스터 미사일인 SAMP/T</div>

<p>아스터 30을 지대공 미사일로도 사용한다. <b>SAMP/T</b> (Surface-to-Air Missile Platform/Terrain)이라고 부른다. 사거리 600 km인 단거리 탄도 미사일을 요격할 수 있다. 프랑스 국방부는 육군용 6개, 공군용 6개를 주문했다가 2012년 10개로 줄였으며, 모두 <a href="/wiki/%ED%94%84%EB%9E%91%EC%8A%A4_%EA%B3%B5%EA%B5%B0" title="프랑스 공군">프랑스 공군</a>에 배치하기로 했다. <a href="/wiki/%EC%9D%B4%ED%83%88%EB%A6%AC%EC%95%84_%EC%9C%A1%EA%B5%B0" title="이탈리아 육군">이탈리아 육군</a>도 6개를 주문했다. <a href="/wiki/%EC%8B%B1%EA%B0%80%ED%8F%AC%EB%A5%B4_%EA%B3%B5%EA%B5%B0" title="싱가포르 공군">싱가포르 공군</a>이 <a class="mw-redirect" href="/wiki/%ED%98%B8%ED%81%AC_%EB%AF%B8%EC%82%AC%EC%9D%BC" title="호크 미사일">호크 미사일</a>을 대체하기 위해 몇대를 주문했다.</p>
