<p><b>아스피린</b>(Aspirin) 또는 <b>아세틸 살리실산</b>(<span lang="en" xml:lang="en">acetylsalicylic acid, ASA</span>)은 <a href="/wiki/%EC%82%B4%EB%A6%AC%EC%8B%A4%EC%82%B0" title="살리실산">살리실산염</a> 의약품이다. <a href="/wiki/%EC%A7%84%ED%86%B5%EC%A0%9C" title="진통제">진통제</a>, <a class="new" href="/w/index.php?title=%ED%95%B4%EC%97%B4%EC%A0%9C&amp;action=edit&amp;redlink=1" title="해열제 (없는 문서)">해열제</a>로 쓰고, 혈중 농도를 낮추어 <a class="mw-redirect" href="/wiki/%EC%8B%AC%ED%98%88%EA%B4%80%EC%A7%88%ED%99%98" title="심혈관질환">심혈관질환</a>이나 <a class="mw-disambig" href="/wiki/%EC%8B%AC%EC%9E%A5%EB%A7%88%EB%B9%84" title="심장마비">심장마비</a> 예방약으로 장기간 쓴다. 반감기는 300~650 mg 일 경우 3.1 - 3.2시간, 1g일 경우 6시간, 2g일 경우 9시간이다.</p>
<p>‘아스피린’은 <a href="/wiki/%EB%B0%94%EC%9D%B4%EC%97%98_(%EA%B8%B0%EC%97%85)" title="바이엘 (기업)">바이엘사</a>의 상표명이지만, 몇몇 나라에서는 아스피린을 아세틸살리실산이라는 물질명으로 부르기도 한다. 아스피린은 <a class="new" href="/w/index.php?title=%ED%94%84%EB%A1%9C%ED%8A%B8%EB%A1%AC%EB%B9%88&amp;action=edit&amp;redlink=1" title="프로트롬빈 (없는 문서)">프로트롬빈</a> 생성 억제를 통한 혈소판 제거 반응을 하기 때문에 항응고 작용을 하여 혈전을 예방 할 수 있다. 이 때문에 급성 심장마비 혹은 혈전 용해에도 사용한다. 그러나 통증 억제를 위한 복용 시 이는 과다 출혈과 같은 부작용을 일으킬 수 있다.</p>
<div style="clear: both;"></div>
<h2><span class="mw-headline" id=".EC.97.AD.EC.82.AC">역사</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%95%84%EC%8A%A4%ED%94%BC%EB%A6%B0&amp;action=edit&amp;section=1" title="부분 편집: 역사">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p><a href="/wiki/%EB%B2%84%EB%93%9C%EB%82%98%EB%AC%B4" title="버드나무">버드나무</a> 껍질에 함유된 <a href="/wiki/%EC%82%B4%EB%A6%AC%EC%8B%A4%EC%82%B0" title="살리실산">살리실산</a>이라는 물질에서 비롯된 아스피린은 기원전(BC) 1천500년쯤 고대 <a href="/wiki/%EC%9D%B4%EC%A7%91%ED%8A%B8" title="이집트">이집트</a>에서 작성된 <a href="/wiki/%ED%8C%8C%ED%94%BC%EB%A3%A8%EC%8A%A4" title="파피루스">파피루스</a>에서 언급된다. 아스피린하면 바이엘을 연상하지만 최초의 탄생역사는 기원전으로 올라가고 현대의약으로 특허를 최초로 등록한 독일의 바이엘AG사가 베를린의 제국 특허국에 자사의 상표를 등록한 날로부터 약 110년이 되었다. 그동안 아스피린은 세계에서 가장 생명이 길고 가장 놀라운 약임이 입증되었다. BC 400년쯤에는 '의학의 아버지'로 불리는 그리스의 <a href="/wiki/%ED%9E%88%ED%8F%AC%ED%81%AC%EB%9D%BC%ED%85%8C%EC%8A%A4" title="히포크라테스">히포크라테스</a>가 사용했다는 기록이 있다. 그 후 2천여 년이 지나 영국에서 <a class="mw-disambig" href="/wiki/%EC%97%90%EB%93%9C%EC%9B%8C%EB%93%9C_%EC%8A%A4%ED%86%A4" title="에드워드 스톤">에드워드 스톤</a>이라는 성직자가 백버드나무 껍질 즙을 열이 있는 사람 50명에게 먹여 해열작용을 확인했다. 그는 이 사실을 1763년 <a class="new" href="/w/index.php?title=%EB%9F%B0%EB%8D%98_%EC%99%95%EB%A6%BD%ED%95%99%ED%9A%8C&amp;action=edit&amp;redlink=1" title="런던 왕립학회 (없는 문서)">런던 왕립학회</a>에서 발표했다. 약 60년 후에 이탈리아 화학자 피리아는 버드나무 껍질에서 약효의 주성분인 <a href="/wiki/%EC%82%B4%EB%A6%AC%EC%8B%A0" title="살리신">살리신</a>을 분리했다. 그 뒤 몇 단계 화학 반응을 거쳐 아스피린의 모체인 <a class="new" href="/w/index.php?title=%EC%8B%A4%EB%A6%AC%EC%8B%A4%EC%82%B0&amp;action=edit&amp;redlink=1" title="실리실산 (없는 문서)">실리실산</a>을 얻었다.<sup class="reference" id="cite_ref-1"><a href="#cite_note-1">[1]</a></sup> 살리실산은 의학적인 효과가 있었지만 <a class="new" href="/w/index.php?title=%EC%9C%84%EB%B2%BD&amp;action=edit&amp;redlink=1" title="위벽 (없는 문서)">위벽</a>을 자극하며 <a href="/wiki/%EC%84%A4%EC%82%AC" title="설사">설사</a>를 일으키고, 많이 먹을 경우 죽는 경우도 있었다. 1897년 독일 <a href="/wiki/%EB%B0%94%EC%9D%B4%EC%97%98_(%EA%B8%B0%EC%97%85)" title="바이엘 (기업)">바이엘</a> 사의 연구원 <a href="/wiki/%ED%8E%A0%EB%A6%AD%EC%8A%A4_%ED%98%B8%ED%94%84%EB%A7%8C" title="펠릭스 호프만">펠릭스 호프만</a><sup class="reference" id="cite_ref-2"><a href="#cite_note-2">[2]</a></sup> 은 <a href="/wiki/%EC%82%B4%EB%A6%AC%EC%8B%A4%EC%82%B0" title="살리실산">살리실산</a>의 <a class="mw-redirect" href="/wiki/%ED%9E%88%EB%93%9C%EB%A1%9D%EC%8B%9C%EA%B8%B0" title="히드록시기">히드록시기</a>를 <a class="new" href="/w/index.php?title=%EC%95%84%EC%84%B8%ED%8B%B8%EA%B8%B0&amp;action=edit&amp;redlink=1" title="아세틸기 (없는 문서)">아세틸기</a>와 <a class="mw-redirect" href="/wiki/%EC%97%90%EC%8A%A4%ED%85%8C%EB%A5%B4%ED%99%94" title="에스테르화">에스테르화</a> 반응을 시켜 아스피린을 만들었는데 <a href="/wiki/%EC%82%B4%EB%A6%AC%EC%8B%A4%EC%82%B0" title="살리실산">살리실산</a>의 부작용이 크게 줄어들었다. 이는 최초의 <a class="new" href="/w/index.php?title=%ED%95%A9%EC%84%B1_%EC%9D%98%EC%95%BD%ED%92%88&amp;action=edit&amp;redlink=1" title="합성 의약품 (없는 문서)">합성 의약품</a>이다.</p>
<h2><span class="mw-headline" id=".EA.B0.99.EC.9D.B4_.EB.B3.B4.EA.B8.B0">같이 보기</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%95%84%EC%8A%A4%ED%94%BC%EB%A6%B0&amp;action=edit&amp;section=2" title="부분 편집: 같이 보기">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
