<h1 class="firstHeading" id="firstHeading" lang="ko">아시아 태평양 이론물리센터</h1>

<p><br/>
<b>아시아 태평양 이론물리센터</b>(Asia Pacific Center for Theoretical Physics)는 이론물리학 연구 선도, 국제공동연구 촉진, 아시아 태평양 지역 젊은 과학자 연수, 아시아 태평양 지역의 회원국과 그 외 지역 물리학자들 사이의 국제 협력 증진을 위하여 <a href="/wiki/1996%EB%85%84" title="1996년">1996년</a> 6월 <a href="/wiki/%EB%8C%80%ED%95%9C%EB%AF%BC%EA%B5%AD" title="대한민국">대한민국</a>에 설립되었다. 국제비정부기구(INGO)로서 현재 회원국은 <a class="mw-redirect" href="/wiki/%ED%98%B8%EC%A3%BC" title="호주">호주</a>, [|중국]], <a href="/wiki/%EC%9D%BC%EB%B3%B8" title="일본">일본</a>, <a href="/wiki/%EB%8C%80%ED%95%9C%EB%AF%BC%EA%B5%AD" title="대한민국">대한민국</a>, <a href="/wiki/%EB%A7%90%EB%A0%88%EC%9D%B4%EC%8B%9C%EC%95%84" title="말레이시아">말레이시아</a>, <a href="/wiki/%ED%95%84%EB%A6%AC%ED%95%80" title="필리핀">필리핀</a>, <a href="/wiki/%EC%8B%B1%EA%B0%80%ED%8F%AC%EB%A5%B4" title="싱가포르">싱가포르</a>, [대만]], <a class="mw-redirect" href="/wiki/%ED%83%9C%EA%B5%AD" title="태국">태국</a>, <a href="/wiki/%EB%B2%A0%ED%8A%B8%EB%82%A8" title="베트남">베트남</a>, <a href="/wiki/%EB%9D%BC%EC%98%A4%EC%8A%A4" title="라오스">라오스</a>, <a href="/wiki/%EB%AA%BD%EA%B3%A8" title="몽골">몽골</a>, <a href="/wiki/%EC%9D%B8%EB%8F%84" title="인도">인도</a>, <a href="/wiki/%EC%9A%B0%EC%A6%88%EB%B2%A0%ED%82%A4%EC%8A%A4%ED%83%84" title="우즈베키스탄">우즈베키스탄</a>, <a href="/wiki/%EC%B9%B4%EC%9E%90%ED%9D%90%EC%8A%A4%ED%83%84" title="카자흐스탄">카자흐스탄</a>, <a href="/wiki/%EC%BA%90%EB%82%98%EB%8B%A4" title="캐나다">캐나다</a>(16개국)이다.</p>

<h2><span class="mw-headline" id=".ED.95.99.EC.88.A0_.ED.99.9C.EB.8F.99">학술 활동</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%95%84%EC%8B%9C%EC%95%84_%ED%83%9C%ED%8F%89%EC%96%91_%EC%9D%B4%EB%A1%A0%EB%AC%BC%EB%A6%AC%EC%84%BC%ED%84%B0&amp;action=edit&amp;section=2" title="부분 편집: 학술 활동">편집</a><span class="mw-editsection-bracket">]</span></span></h2>

<li>Conferences and Workshops</li>

</li>

<ul>

<li>프로그램 구성

</ul>
