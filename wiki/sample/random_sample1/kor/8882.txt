<p><b>아시아린상</b>(Asiatic linsang)은 <a href="/wiki/%EC%8B%9D%EC%9C%A1%EB%AA%A9" title="식육목">식육목</a>의 <b>아시아린상과</b>(Prionodontidae)와 <b>아시아린상속</b>(<i>Prionodon</i>)에 속하는 2종으로 이루어진 <a href="/wiki/%EB%8F%99%EB%82%A8%EC%95%84%EC%8B%9C%EC%95%84" title="동남아시아">동남아시아</a> 토착종의 총칭이다. 2종은 <a href="/wiki/%EC%A4%84%EB%AC%B4%EB%8A%AC%EB%A6%B0%EC%83%81" title="줄무늬린상">줄무늬린상</a>(<i>Prionodon linsang</i>)과 <a href="/wiki/%EC%A0%90%EB%B0%95%EC%9D%B4%EB%A6%B0%EC%83%81" title="점박이린상">점박이린상</a>(<i>Prionodon pardicolor</i>)이다.<sup class="reference" id="cite_ref-Pocock1939_4-0"><a href="#cite_note-Pocock1939-4">[4]</a></sup> 아시아린상속(<i>Prionodon</i>)은 <a href="/wiki/%EA%B3%A0%EC%96%91%EC%9D%B4%EC%95%84%EB%AA%A9" title="고양이아목">고양이아목</a>의 기저 분류군으로 <a href="/wiki/%EA%B3%A0%EC%96%91%EC%9D%B4%EA%B3%BC" title="고양이과">고양이과</a>와 자매군 관계를 갖고 있다.<sup class="reference" id="cite_ref-5"><a href="#cite_note-5">[5]</a></sup></p>
<h2><span class="mw-headline" id=".EB.B6.84.EB.A5.98.EC.9D.98_.EC.97.AD.EC.82.AC">분류의 역사</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%95%84%EC%8B%9C%EC%95%84%EB%A6%B0%EC%83%81&amp;action=edit&amp;section=1" title="부분 편집: 분류의 역사">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>아시아린상속(<i>Prionodon</i>)은 1822년 호스필드(Thomas Horsfield)가 <a href="/wiki/%EC%9E%90%EC%99%80_%EC%84%AC" title="자와 섬">자와 섬</a>의 린상을 토대로 처음 기술했다. 처음에는 두 속, <a href="/wiki/%EC%82%AC%ED%96%A5%EA%B3%A0%EC%96%91%EC%9D%B4%EC%86%8D" title="사향고양이속">사향고양이속</a>(<i>Viverra</i>)과 <a href="/wiki/%EA%B3%A0%EC%96%91%EC%9D%B4%EC%86%8D" title="고양이속">고양이속</a>(<i>Felis</i>)의 유사성때문에 <b>프리오노돈과</b>(Prionodontidae) 아래에 이 속을 분류했다.<sup class="reference" id="cite_ref-Horsfield_2-1"><a href="#cite_note-Horsfield-2">[2]</a></sup> 1864년, <a href="/wiki/%EC%A1%B4_%EC%97%90%EB%93%9C%EC%9B%8C%EB%93%9C_%EA%B7%B8%EB%A0%88%EC%9D%B4" title="존 에드워드 그레이">존 에드워드 그레이</a>는 아시아린상속(<i>Prionodon</i>)과 <a href="/wiki/%EC%95%84%ED%94%84%EB%A6%AC%EC%B9%B4%EB%A6%B0%EC%83%81%EC%86%8D" title="아프리카린상속">아프리카린상속</a>(<i>Poiana</i>)을 포함하는 <b>프리오노돈아과</b>(Prionodontinae)를 명명했다.<sup class="reference" id="cite_ref-Gray1864_3-1"><a href="#cite_note-Gray1864-3">[3]</a></sup> 포콕(Reginald Innes Pocock)은 처음에 그레이의 분류를 이어받았지만, <a href="/wiki/%EC%95%84%ED%94%84%EB%A6%AC%EC%B9%B4%EB%A6%B0%EC%83%81%EC%86%8D" title="아프리카린상속">아프리카린상</a>의 사향샘의 존재는 <a href="/wiki/%EC%A0%9C%EB%84%B7%EC%86%8D" title="제넷속">제넷고양이</a>의 특수한 형태처럼 일시적으로 유발된 것이며, 프리오노돈속에 대한 유사성은 적응된 것으로 추정했다.<sup class="reference" id="cite_ref-Pocock1939_4-1"><a href="#cite_note-Pocock1939-4">[4]</a></sup> <a href="/wiki/%EC%82%AC%ED%96%A5%EA%B3%A0%EC%96%91%EC%9D%B4%EC%95%84%EA%B3%BC" title="사향고양이아과">사향고양이아과</a>의 13종과 <a href="/wiki/%EC%95%84%EC%8B%9C%EC%95%84%EC%82%AC%ED%96%A5%EA%B3%A0%EC%96%91%EC%9D%B4%EC%86%8D" title="아시아사향고양이속">아시아사향고양이속</a>과 <a href="/wiki/%ED%9D%B0%EC%BD%94%EC%82%AC%ED%96%A5%EA%B3%A0%EC%96%91%EC%9D%B4" title="흰코사향고양이">흰코사향고양이속</a> 그리고 <a href="/wiki/%EC%A4%84%EB%AC%B4%EB%8A%AC%EC%82%AC%ED%96%A5%EA%B3%A0%EC%96%91%EC%9D%B4%EC%95%84%EA%B3%BC" title="줄무늬사향고양이아과">줄무늬사향고양이아과</a>를 대표하는 3종을 포함한 <a href="/wiki/%EC%8B%9D%EC%9C%A1%EB%AA%A9" title="식육목">식육목</a> 29종의 DNA 분석 결과에 의해, 아프리카린상속과 제넷고양이속이 자매군 관계를 갖는다는 포콕의 추정이 사실로 확인되었다. 프리오노돈속은 <a href="/wiki/%EA%B3%A0%EC%96%91%EC%9D%B4%EA%B3%BC" title="고양이과">고양이과</a>의 자매군이자 프리오노돈과의 단형속으로 강하게 지지되고 제안되었다.<sup class="reference" id="cite_ref-rspb2003_6-0"><a href="#cite_note-rspb2003-6">[6]</a></sup></p>
<h2><span class="mw-headline" id=".EA.B3.84.ED.86.B5_.EB.B6.84.EB.A5.98">계통 분류</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%95%84%EC%8B%9C%EC%95%84%EB%A6%B0%EC%83%81&amp;action=edit&amp;section=2" title="부분 편집: 계통 분류">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>다음은 식육목의 계통 분류이다.<sup class="reference" id="cite_ref-Eizirik.2C_E._2010_7-0"><a href="#cite_note-Eizirik.2C_E._2010-7">[7]</a></sup></p>
