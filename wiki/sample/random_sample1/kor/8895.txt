href="#cite_note-mw112-1">[1]</a></sup> 다만 콘스탄티노폴리스가 라틴 제국에 의해서 점령된 1204년부터 1261년까지는 로마 가톨릭교회의 성당으로 개조됐다. <a href="/wiki/1453%EB%85%84" title="1453년">1453년</a> <a href="/wiki/5%EC%9B%94_29%EC%9D%BC" title="5월 29일">5월 29일</a>부터 1931년까지는 모스크로 사용되었고, <a href="/wiki/1935%EB%85%84" title="1935년">1935년</a>에 박물관으로 다시 개장했다.<sup class="reference" id="cite_ref-2"><a href="#cite_note-2">[2]</a></sup></p>
<p>현재까지 남아있는 <a href="/wiki/%EB%B9%84%EC%9E%94%ED%8B%B0%EC%9B%80" title="비잔티움">비잔티움</a> 건축의 대표작으로 세계에서 몇 손가락 안에 꼽히는 건축물로 여겨지고 있다. 비잔티움 제국의 건물이라고 하여, <a href="/wiki/%EA%B8%B0%EB%8F%85%EA%B5%90" title="기독교">기독교</a>의 문화유산으로만 생각할 수도 있으나, <a href="/wiki/%EC%9D%B4%EC%8A%AC%EB%9E%8C%EA%B5%90" title="이슬람교">이슬람교</a>와도 관련이 크며, 500년 가까이 <a href="/wiki/%EC%9D%B4%EC%8A%AC%EB%9E%8C%EA%B5%90" title="이슬람교">이슬람교</a> 신자들의 예배당으로 사용되었다. 성당 옆에 있는 4개의 탑들은 <a href="/wiki/%EB%AF%B8%EB%82%98%EB%A0%88%ED%8A%B8" title="미나레트">미나레트</a>라고 부른다.</p>

<h2><span class="mw-headline" id=".EC.97.AD.EC.82.AC">역사</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%95%84%EC%95%BC_%EC%86%8C%ED%94%BC%EC%95%84&amp;action=edit&amp;section=1" title="부분 편집: 역사">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<h3><span class="mw-headline" id=".EB.8F.99.EB.A1.9C.EB.A7.88_.EC.A0.9C.EA.B5.AD_.EC.8B.9C.EB.8C.80">동로마 제국 시대</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%95%84%EC%95%BC_%EC%86%8C%ED%94%BC%EC%95%84&amp;action=edit&amp;section=2" title="부분 편집: 동로마 제국 시대">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>하기아 소피아는 원래 <a class="mw-redirect" href="/wiki/%EC%A0%95%EA%B5%90%ED%9A%8C" title="정교회">정교회</a> 대성당이며, 교회는 <a href="/wiki/360%EB%85%84" title="360년">360년</a>, 이스탄불이 <a href="/wiki/%EB%A1%9C%EB%A7%88_%EC%A0%9C%EA%B5%AD" title="로마 제국">로마 제국</a>의 새로운 수도로서 <a href="/wiki/%EC%BD%98%EC%8A%A4%ED%83%84%ED%8B%B0%EB%85%B8%ED%8F%B4%EB%A6%AC%EC%8A%A4" title="콘스탄티노폴리스">콘스탄티노폴리스</a>로 불렸을 때, <a href="/wiki/%EC%BD%98%EC%8A%A4%ED%83%84%ED%8B%B0%EB%88%84%EC%8A%A4_1%EC%84%B8" title="콘스탄티누스 1세">콘스탄티누스 1세</a>의 아들 <a href="/wiki/%EC%BD%98%EC%8A%A4%ED%83%84%ED%8B%B0%EC%9A%B0%EC%8A%A4_2%EC%84%B8" title="콘스탄티우스 2세">콘스탄티우스 2세</a>에 의해서 처음으로 건립되었다. 이 정교회 대성당이 이때부터 하기아 소피아로 불리게 되었는지, 훗날의 재건 때부터 하기아 소피아로 명명되었는지는 불확실하다. 최초의 정교회 대성당은 목조 지붕을 가진 <a class="new" href="/w/index.php?title=%EB%B0%94%EC%8B%A4%EB%A6%AC%ED%81%AC&amp;action=edit&amp;redlink=1" title="바실리크 (없는 문서)">바실리크</a>였던 것 같지만, 오늘날에는 그 위치밖에 알려지지 않았다.</p>
<p><a href="/wiki/404%EB%85%84" title="404년">404년</a>에 <a href="/wiki/%EC%BD%98%EC%8A%A4%ED%83%84%ED%8B%B0%EB%85%B8%ED%8F%B4%EB%A6%AC%EC%8A%A4_%EC%B4%9D%EB%8C%80%EC%A3%BC%EA%B5%90" title="콘스탄티노폴리스 총대주교">콘스탄티노폴리스 총대주교</a> <a href="/wiki/%EC%9A%94%ED%95%9C%EB%84%A4%EC%8A%A4_%ED%81%AC%EB%A6%AC%EC%86%8C%EC%8A%A4%ED%86%A0%EB%AC%B4%EC%8A%A4" title="요한네스 크리소스토무스">요한네스 크리소스토무스</a>(성 크리소스톰)를 추방시키는데 수반된 난동으로 교회가 소실되어, <a href="/wiki/%ED%85%8C%EC%98%A4%EB%8F%84%EC%8B%9C%EC%9A%B0%EC%8A%A4_2%EC%84%B8" title="테오도시우스 2세">테오도시우스 2세</a>에 의해 재건,<a href="/wiki/415%EB%85%84" title="415년">415년</a>에 축성되었다. 이 대성당도 현재의 것과는 완전히 다른 바실리카이며, 현재에도 일렬의 원주와 주기, 장식된 대들보가 남아 있다. 그러나, 이 정교회 대성당도 <a href="/wiki/532%EB%85%84" title="532년">532년</a> <a href="/wiki/1%EC%9B%94" title="1월">1월</a> 13-14일 <a class="mw-redirect" href="/wiki/%EB%8B%88%EC%B9%B4%EC%9D%98_%EB%B0%98%EB%9E%80" title="니카의 반란">니카의 반란</a> 도중 일어났던 대화재로, 황궁의 일부나 <a class="new" href="/w/index.php?title=%EC%84%B1_%EC%9D%B4%EB%A0%88%EB%84%A4_%EC%84%B1%EB%8B%B9&amp;action=edit&amp;redlink=1" title="성 이레네 성당 (없는 문서)">성 이레네 성당</a>과 함께 다시 소실해 버린다.</p>
<p>두 번의 소실을 겪은 후, <a href="/wiki/%EC%9C%A0%EC%8A%A4%ED%8B%B0%EB%8B%88%EC%95%84%EB%88%84%EC%8A%A4_1%EC%84%B8" title="유스티니아누스 1세">유스티니아누스 1세</a>는 즉위후 소피아 정교회 성당의 재건을 결정하여, 그 설계를 <a class="new" href="/w/index.php?title=%EC%95%88%ED%85%8C%EB%AF%B8%EC%98%A4%EC%8A%A4&amp;action=edit&amp;redlink=1" title="안테미오스 (없는 문서)">안테미오스</a>와 <a class="new" href="/w/index.php?title=%EC%9D%B4%EC%8B%9C%EB%8F%84%EB%A1%9C%EC%8A%A4&amp;action=edit&amp;redlink=1" title="이시도로스 (없는 문서)">이시도로스</a>에게 맡겼다. 공사는 5년 11개월이라고 하는 단기간에 종료되어, <a href="/wiki/537%EB%85%84" title="537년">537년</a> <a href="/wiki/12%EC%9B%94_27%EC%9D%BC" title="12월 27일">12월 27일</a> 유스티니아누스 1세를 맞이하고 <a href="/wiki/%EC%BD%98%EC%8A%A4%ED%83%84%ED%8B%B0%EB%85%B8%ED%8F%B4%EB%A6%AC%EC%8A%A4" title="콘스탄티노폴리스">콘스탄티노폴리스</a> 총대주교인 <a class="new" href="/w/index.php?title=%EB%A9%94%EB%82%98%EC%8A%A4&amp;action=edit&amp;redlink=1" title="메나스 (없는 문서)">메나스</a> <a href="/wiki/%EC%B4%9D%EB%8C%80%EC%A3%BC%EA%B5%90" title="총대주교">총대주교</a>에 의한 헌당식을 거행했다. 이때 유스티니아누스 1세는, 고대 이스라엘 왕국의 왕 <a class="mw-redirect" href="/wiki/%EC%86%94%EB%A1%9C%EB%AA%AC_%EC%99%95" title="솔로몬 왕">솔로몬</a>의 신전을 능가하는 교회를 세웠다는 생각을 하여 “솔로몬이여, 내가 그대에게 승리했도다!” 라고 외쳤다고 전해지고 있다.</p>
<p><a href="/wiki/%EC%9C%A0%EC%8A%A4%ED%8B%B0%EB%8B%88%EC%95%84%EB%88%84%EC%8A%A4_1%EC%84%B8" title="유스티니아누스 1세">유스티니아누스 1세</a>에 의해서 재건된 하기아 소피아 대성당은, 콘스탄티노폴리스 총대주교구의 소재지로서 <a href="/wiki/%EB%8F%99%EB%B0%A9_%EC%A0%95%EA%B5%90%ED%9A%8C" title="동방 정교회">동방 정교회</a> 제일의 격식을 자랑하였으며, 또 <a class="mw-redirect" href="/wiki/%EB%8F%99%EB%A1%9C%EB%A7%88_%EC%A0%9C%EA%B5%AD" title="동로마 제국">동로마 제국</a>의 황제의 사당으로서 이용되었다. 콘스탄티노폴리스를 방문한 순례자들의 기록을 보면, 대성당 안에는 현재는 없어진 시설이나 성유물이 있었던 것을 알 수 있다. <a href="/wiki/14%EC%84%B8%EA%B8%B0" title="14세기">14세기</a>에 콘스탄티노폴리스를 방문한 <a href="/wiki/%EB%9F%AC%EC%8B%9C%EC%95%84%EC%9D%B8" title="러시아인">러시아인</a> <a href="/wiki/%EC%8A%A4%EB%AA%B0%EB%A0%8C%EC%8A%A4%ED%81%AC" title="스몰렌스크">스몰렌스크</a>의 <a class="new" href="/w/index.php?title=%EC%9D%B4%EA%B7%B8%EB%82%98%ED%8B%B0%EC%98%A4%EC%8A%A4&amp;action=edit&amp;redlink=1" title="이그나티오스 (없는 문서)">이그나티오스</a>의 기록에서는, 대성당 내부에는 많은 예배당이 설치되어 있으며 <a href="/wiki/%EB%85%B8%EC%95%84" title="노아">노아</a>의 방주의 문이나 성 십자가<sup class="reference" id="cite_ref-3"><a href="#cite_note-3">[3]</a></sup>, <a href="/wiki/%EC%95%84%EB%B8%8C%EB%9D%BC%ED%95%A8" title="아브라함">아브라함</a>의 테이블 등 많은 성유물들이 안치되어 있었다. 또, 이 시대에는 근처에 총대주교구의 자택이 병설되어 있어, 현재는 출입구가 되어 있는 부분은 총대주교 자택으로 통하는 통로로 이용되고 있었다.</p>
<h3><span class="mw-headline" id=".EC.98.A4.EC.8A.A4.EB.A7.8C_.EC.A0.9C.EA.B5.AD_.EC.8B.9C.EB.8C.80">오스만 제국 시대</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%95%84%EC%95%BC_%EC%86%8C%ED%94%BC%EC%95%84&amp;action=edit&amp;section=3" title="부분 편집: 오스만 제국 시대">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p><a href="/wiki/1453%EB%85%84" title="1453년">1453년</a> <a href="/wiki/5%EC%9B%94_29%EC%9D%BC" title="5월 29일">5월 29일</a>, <a href="/wiki/%EC%BD%98%EC%8A%A4%ED%83%84%ED%8B%B0%EB%85%B8%ED%8F%B4%EB%A6%AC%EC%8A%A4" title="콘스탄티노폴리스">콘스탄티노폴리스</a>를 점거한(<a href="/wiki/%EC%BD%98%EC%8A%A4%ED%83%84%ED%8B%B0%EB%85%B8%ED%8F%B4%EB%A6%AC%EC%8A%A4%EC%9D%98_%ED%95%A8%EB%9D%BD" title="콘스탄티노폴리스의 함락">콘스탄티노폴리스의 함락</a>) <a href="/wiki/%EC%98%A4%EC%8A%A4%EB%A7%8C_%EC%A0%9C%EA%B5%AD" title="오스만 제국">오스만 제국</a>의 <a href="/wiki/%EC%88%A0%ED%83%84" title="술탄">술탄</a> <a class="mw-redirect" href="/wiki/%EB%A9%94%ED%9D%90%EB%A9%94%EB%93%9C_2%EC%84%B8" title="메흐메드 2세">메흐메드 2세</a>는, 그 날 오후에 도시로 입성하자마자 하기아 소피아 대성당으로 향했다. 그는 “<a class="mw-redirect" href="/wiki/%EA%B7%B8%EB%A6%AC%EC%8A%A4%EB%8F%84%EC%9D%B8" title="그리스도인">그리스도인</a>들이 믿는 <a href="/wiki/%ED%95%98%EB%8A%90%EB%8B%98" title="하느님">하느님</a>은 없고, <a href="/wiki/%EC%95%8C%EB%9D%BC" title="알라">알라</a>만 존재한다.” 고 외치면서 영토 확장 목적의 달성을 기념하기 위해 대성당의 흙을 자신의 머리에 뿌리고, 콘스탄티노폴리스 총대주교로부터 이 대성당을 몰수, <a href="/wiki/%EB%AA%A8%EC%8A%A4%ED%81%AC" title="모스크">모스크</a>로 사용할 것을 선언하였다. 이때에 대성당과 연결되어 있는 총대주교 자택의 통로는 파괴되고 대성당 내부는 <a href="/wiki/%EC%8B%AD%EC%9E%90%EA%B0%80" title="십자가">십자가</a>가 떼어지고 <a class="mw-redirect" href="/wiki/%EC%84%B1%ED%99%94%EC%83%81" title="성화상">성화</a>(이콘)는 석회칠로 덮어지고 <a href="/wiki/%EB%A9%94%EC%B9%B4" title="메카">메카</a>의 방향을 나타내는 <a href="/wiki/%EB%AF%B8%ED%9D%90%EB%9D%BC%EB%B8%8C" title="미흐라브">미흐라브</a>가 더해졌다. 그 후, 네 개의 미나렛(첨탑)이 증축되어 교회 내에는 민발로 불리는 설교 단상도 장착되었다. 아야 소피아 자미로 불리게 된 이 정교회 대성당은 토프카프 궁전 쪽에 위치해 있어, 오스만 제국의 술탄이 매주 금요일 예배마다 방문하게 되어 오스만 제국에서 가장 격식 높은 모스크 중 하나로 여겨지게 되었다.</p>
<h3><span class="mw-headline" id=".ED.84.B0.ED.82.A4_.EA.B3.B5.ED.99.94.EA.B5.AD">터키 공화국</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%95%84%EC%95%BC_%EC%86%8C%ED%94%BC%EC%95%84&amp;action=edit&amp;section=4" title="부분 편집: 터키 공화국">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p><a href="/wiki/1923%EB%85%84" title="1923년">1923년</a> 오스만 제정이 무너지고 <a href="/wiki/%ED%84%B0%ED%82%A4" title="터키">터키 공화국</a>이 수립되었을 때 <a href="/wiki/%EA%B7%B8%EB%A6%AC%EC%8A%A4" title="그리스">그리스</a>를 중심으로 유럽 각국은 하기아 소피아의 반환과 종교적 복원을 강력하게 요구했다. 터키 정부는 하기아 소피아를 인류 모두의 공동유산인 박물관으로 지정하고 아야 소피아 박물관(Ayasofya Müzesi)으로 개조해 그 안에서 <a href="/wiki/%EA%B8%B0%EB%8F%85%EA%B5%90" title="기독교">기독교</a>든 <a class="mw-redirect" href="/wiki/%EC%9D%B4%EC%8A%AC%EB%9E%8C" title="이슬람">이슬람</a>이든 종교적 행위를 일절 금지했다. 하지만 <a href="/wiki/2013%EB%85%84" title="2013년">2013년</a> <a href="/wiki/11%EC%9B%94_13%EC%9D%BC" title="11월 13일">11월 13일</a>에 아른츠 부총리가 아야 소피아를 모스크로 다시 바뀌기를 원한다는 발언을 함으로써 논란에 휩싸이고 있다.<sup class="reference" id="cite_ref-4"><a href="#cite_note-4">[4]</a></sup></p>
<h2><span class="mw-headline" id=".EA.B1.B4.EC.B6.95">건축</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%95%84%EC%95%BC_%EC%86%8C%ED%94%BC%EC%95%84&amp;action=edit&amp;section=5" title="부분 편집: 건축">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<div class="thumb tright">
<div class="thumbinner" style="width:222px;"><a class="image" href="/wiki/%ED%8C%8C%EC%9D%BC:Hagia-Sophia-Laengsschnitt.jpg"><img alt="" class="thumbimage" data-file-height="1912" data-file-width="3996" height="105" src="//upload.wikimedia.org/wikipedia/commons/thumb/4/45/Hagia-Sophia-Laengsschnitt.jpg/220px-Hagia-Sophia-Laengsschnitt.jpg" srcset="//upload.wikimedia.org/wikipedia/commons/thumb/4/45/Hagia-Sophia-Laengsschnitt.jpg/330px-Hagia-Sophia-Laengsschnitt.jpg 1.5x, //upload.wikimedia.org/wikipedia/commons/thumb/4/45/Hagia-Sophia-Laengsschnitt.jpg/440px-Hagia-Sophia-Laengsschnitt.jpg 2x" width="220"/></a>
<div class="thumbcaption">

아야 소피아의 단면도</div>
</div>
</div>
<p>아야 소피아는 세계에서 가장 손 꼽히는 <a class="mw-redirect" href="/wiki/%EB%B9%84%EC%9E%94%ED%8B%B4_%EC%A0%9C%EA%B5%AD" title="비잔틴 제국">비잔틴 제국</a>의 건축물 중 하나이다. 아야 소피아 내부는 <a href="/wiki/%EB%AA%A8%EC%9E%90%EC%9D%B4%ED%81%AC" title="모자이크">모자이크</a>와 <a class="mw-redirect" href="/wiki/%EB%8C%80%EB%A6%AC%EC%84%9D" title="대리석">대리석</a> 기둥으로 장식되어 있고, 대단히 높은 예술적 가치를 갖고 있다. 아야 소피아의 아름다움을 자랑스러워한 <a href="/wiki/%EC%9C%A0%EC%8A%A4%ED%8B%B0%EB%8B%88%EC%95%84%EB%88%84%EC%8A%A4_1%EC%84%B8" title="유스티니아누스 1세">유스티니아누스 1세</a>는 "<a href="/wiki/%EC%86%94%EB%A1%9C%EB%AA%AC" title="솔로몬">솔로몬</a>, 내가 그대를 이겼다."(Νενίκηκά σε Σολομών)라는 유명한 말로 아야 소피아의 아름다움을 표현했다.</p>
