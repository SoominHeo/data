<p><b>아오키 가즈오</b> (青木 一男, あおき かずお, <a href="/wiki/1889%EB%85%84" title="1889년">1889년</a> <a href="/wiki/11%EC%9B%94_28%EC%9D%BC" title="11월 28일">11월 28일</a> ~ <a href="/wiki/1982%EB%85%84" title="1982년">1982년</a> <a href="/wiki/6%EC%9B%94_25%EC%9D%BC" title="6월 25일">6월 25일</a>) 는 <a href="/wiki/%EC%9D%BC%EB%B3%B8" title="일본">일본</a>의 <a class="mw-redirect" href="/wiki/%EC%A0%95%EC%B9%98%EC%9D%B8" title="정치인">정치인</a>으로, <a href="/wiki/1939%EB%85%84" title="1939년">1939년</a> <a href="/wiki/8%EC%9B%94_30%EC%9D%BC" title="8월 30일">8월 30일</a>부터 <a href="/wiki/1940%EB%85%84" title="1940년">1940년</a> <a href="/wiki/1%EC%9B%94_16%EC%9D%BC" title="1월 16일">1월 16일</a>까지 <a href="/wiki/%EC%95%84%EB%B2%A0_%EB%85%B8%EB%B6%80%EC%9C%A0%ED%82%A4" title="아베 노부유키">아베 노부유키</a> 총리 내각에서 대장성 대신, <a href="/wiki/1942%EB%85%84" title="1942년">1942년</a>부터 <a href="/wiki/1944%EB%85%84" title="1944년">1944년</a>까지 대동아성 대신을 지냈다. 사위 가토 이치로(加藤一郎) 는 전 <a class="mw-redirect" href="/wiki/%EB%8F%84%EC%BF%84_%EB%8C%80%ED%95%99%EA%B5%90" title="도쿄 대학교">도쿄 대학교</a> 총장이며, 손녀이자 가토 이치로의 딸인 고미야마 요코(小宮山洋子)는 일본 국회의원으로 활동하고 있다.</p>
<p>아오키는 1889년, <a href="/wiki/%EB%82%98%EA%B0%80%EB%85%B8_%ED%98%84" title="나가노 현">나가노 현</a>의 농가에서 태어나, <a href="/wiki/1916%EB%85%84" title="1916년">1916년</a> <a class="mw-redirect" href="/wiki/%EB%8F%84%EC%BF%84_%EC%A0%9C%EA%B5%AD%EB%8C%80%ED%95%99" title="도쿄 제국대학">도쿄 제국대학</a> 법학부를 졸업하고 <a class="mw-redirect" href="/wiki/%EC%9D%BC%EB%B3%B8_%EB%8C%80%EC%9E%A5%EC%84%B1" title="일본 대장성">대장성</a>에 들어갔다.</p>
<p>여러 직책을 거쳐 1934년, 이재국장이 되었으며, <a href="/wiki/%EA%B0%80%EC%95%BC_%EC%98%A4%ED%82%A4%EB%85%B8%EB%A6%AC" title="가야 오키노리">가야 오키노리</a>, 이시와타 소타로(石渡荘太郎)와 함께 대장성의 특출한 세 인물이라 불렸다. 1937년 <a href="/wiki/%EA%B3%A0%EB%85%B8%EC%97%90_%ED%9B%84%EB%AF%B8%EB%A7%88%EB%A1%9C" title="고노에 후미마로">고노에 후미마로</a> 총리의 요청에 따라 기획원에 창설에 참여해 차장에 취임하였고, 1939년, 총재 취임과 동시에, 귀족원 의원에 당선되었다.</p>
<p>1939년, <a href="/wiki/%EC%95%84%EB%B2%A0_%EB%85%B8%EB%B6%80%EC%9C%A0%ED%82%A4" title="아베 노부유키">아베 노부유키</a> 내각에 대장 대신 겸 기획원 총재로 첫 입각하였고, <a href="/wiki/%EC%99%95%EC%A7%95%EC%9B%A8%EC%9D%B4" title="왕징웨이">왕징웨이</a> 정권에의 특파 대사 고문으로서 <a class="mw-redirect" href="/wiki/%EB%82%9C%EC%A7%95" title="난징">난징</a>에 부임해 경제 정책을 지도하였다. 1942년, <a href="/wiki/%EB%8F%84%EC%A1%B0_%ED%9E%88%EB%8D%B0%ED%82%A4" title="도조 히데키">도조 히데키</a> 내각에서 초대 대동아 대신에 임명되어, 대동아 회의 등의 시책을 전개하였다.</p>
<p>전후 <a href="/wiki/A%EA%B8%89_%EC%A0%84%EB%B2%94" title="A급 전범">A급 전범</a> 용의자로 수용되었으나, 1948년에 석방되었다. 석방된 후에는 변호사로 일했으며, 추방 해제 후인 1953년, 제3회 참의원 의원 통상 선거에 자유당 소속으로 전국구에서 당선되어, 이후 네 차례 당선되었다.</p>
<p>1960년, 지바 사부로 (千葉三郎), 기무라 도쿠타로 (木村篤太郎) 등과 우파 집단 "소심회" (素心会) 를 결성하여, 친 <a href="/wiki/%ED%83%80%EC%9D%B4%EC%99%84" title="타이완">타이완</a>파의 입장에서 일본과 중국 간의 국교 회복에 크게 반대하였다. 1968년, 나가노 현 제2의 민간 방송 회사 <a href="/wiki/%EB%82%98%EA%B0%80%EB%85%B8_%EB%B0%A9%EC%86%A1" title="나가노 방송">나가노 방송</a>이 설립되면서 초대 회장이 되었고, 1980년 은퇴할 때까지 이 지위에 있었다.</p>
