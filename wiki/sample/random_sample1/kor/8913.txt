<p><b>아우렐 스타인</b> 경(<span lang="en" xml:lang="en">Sir Marc Aurel Stein</span>, <a class="extiw" href="https://en.wikipedia.org/wiki/Order_of_the_Indian_Empire" title="en:Order of the Indian Empire">KCIE</a>, <a href="/wiki/1862%EB%85%84" title="1862년">1862년</a> <a href="/wiki/11%EC%9B%94_26%EC%9D%BC" title="11월 26일">11월 26일</a>~<a href="/wiki/1943%EB%85%84" title="1943년">1943년</a> <a href="/wiki/10%EC%9B%94_26%EC%9D%BC" title="10월 26일">10월 26일</a>)은 <a href="/wiki/%ED%97%9D%EA%B0%80%EB%A6%AC" title="헝가리">헝가리</a>에서 태어난 <a href="/wiki/%EC%98%81%EA%B5%AD" title="영국">영국</a>의 <a href="/wiki/%ED%83%90%ED%97%98" title="탐험">탐험가</a>이다.</p>
<p>그는 영국과 인도 정부의 지원으로 <a href="/wiki/%EC%A4%91%EC%95%99%EC%95%84%EC%8B%9C%EC%95%84" title="중앙아시아">중앙아시아</a>를 세 번 탐사하여 많은 유물을 발굴했으며, 중국 <a href="/wiki/%EA%B0%84%EC%91%A4_%EC%84%B1" title="간쑤 성">간쑤 성</a> <a href="/wiki/%EB%91%94%ED%99%A9_%EC%8B%9C" title="둔황 시">둔황</a>의 <a href="/wiki/%EB%A7%89%EA%B3%A0%EA%B5%B4" title="막고굴">막고굴</a> 제17굴 장경동의 유물을 연구해 이를 유럽의 학계에 알려 '<a class="new" href="/w/index.php?title=%EB%8F%88%ED%99%A9%ED%95%99&amp;action=edit&amp;redlink=1" title="돈황학 (없는 문서)">돈황학</a>(敦煌學)'을 정립한 인물이기도 하다.</p>

<h2><span class="mw-headline" id=".EC.83.9D.EC.95.A0">생애</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%95%84%EC%9A%B0%EB%A0%90_%EC%8A%A4%ED%83%80%EC%9D%B8&amp;action=edit&amp;section=1" title="부분 편집: 생애">원본 편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<div class="thumb tright">
<div class="thumbinner" style="width:222px;"><a class="image" href="/wiki/%ED%8C%8C%EC%9D%BC:Aur%C3%A9l_Stein.jpg"><img alt="" class="thumbimage" data-file-height="546" data-file-width="711" height="169" src="//upload.wikimedia.org/wikipedia/commons/thumb/f/f2/Aur%C3%A9l_Stein.jpg/220px-Aur%C3%A9l_Stein.jpg" srcset="//upload.wikimedia.org/wikipedia/commons/thumb/f/f2/Aur%C3%A9l_Stein.jpg/330px-Aur%C3%A9l_Stein.jpg 1.5x, //upload.wikimedia.org/wikipedia/commons/thumb/f/f2/Aur%C3%A9l_Stein.jpg/440px-Aur%C3%A9l_Stein.jpg 2x" width="220"/></a>
<div class="thumbcaption">
<div class="magnify"><a class="internal" href="/wiki/%ED%8C%8C%EC%9D%BC:Aur%C3%A9l_Stein.jpg" title="실제 크기로"></a></div>
<a href="/wiki/%EB%91%90%EB%9D%BC_%EC%97%90%EC%9A%B0%EB%A1%9C%ED%8F%AC%EC%8A%A4" title="두라 에우로포스">두라 에우로포스</a>에서의 아우렐 스타인(1929년)</div>
</div>
</div>
<p>헝가리의 <a href="/wiki/%EB%B6%80%EB%8B%A4%ED%8E%98%EC%8A%A4%ED%8A%B8" title="부다페스트">부다페스트</a>에서 태어나, 영국에 귀화하였다. 독일과 영국의 여러 대학에서 이란학과 인도학을 전공하고, <a href="/wiki/1888%EB%85%84" title="1888년">1888년</a> 인도 <a href="/wiki/%EB%9D%BC%ED%98%B8%EB%A5%B4" title="라호르">라호르</a> 동양 학교(Oriental College, Lahore) 교장이 되었다.</p>
<p>그는 인도 정청(政廳) 고고학 조사부에 근무했는데, <a href="/wiki/1900%EB%85%84" title="1900년">1900년</a>~<a href="/wiki/1901%EB%85%84" title="1901년">1901년</a> 동 조사부의 파견으로 <a href="/wiki/%EC%8B%A0%EC%9E%A5_%EC%9C%84%EA%B5%AC%EB%A5%B4_%EC%9E%90%EC%B9%98%EA%B5%AC" title="신장 위구르 자치구">신장 지역</a> 각지를 탐험하고, 2번째 탐험 때에는 중국 둔황에서 많은 고문서와 회화를 발견하였다. 3번째 탐험 때는 하라 호토(Khara-Khoto) 폐허와 <a href="/wiki/%ED%88%AC%EB%A3%A8%ED%8C%90_%EB%B6%84%EC%A7%80" title="투루판 분지">투루판 분지</a>에 있는 옛 무덤을 발견하였다. 많은 미술품과 고고학적인 유물 자료를 발견하여, 이것의 상세한 보고서 및 연구 저서로 중앙아시아 연구에 크게 이바지 하였다. 또, 서역을 휩쓴 <a href="/wiki/%EA%B3%A0%EA%B5%AC%EB%A0%A4" title="고구려">고구려</a> 유민 <a href="/wiki/%EA%B3%A0%EC%84%A0%EC%A7%80" title="고선지">고선지</a> 장군의 유적도 발견하였다. 저서로 《동남 이란의 고고학적 조사》가 있다.</p>
<h2><span class="mw-headline" id=".EC.97.85.EC.A0.81">업적</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%95%84%EC%9A%B0%EB%A0%90_%EC%8A%A4%ED%83%80%EC%9D%B8&amp;action=edit&amp;section=2" title="부분 편집: 업적">원본 편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p><a href="/wiki/1904%EB%85%84" title="1904년">1904년</a> 당시 막고굴을 관리하고 있던 태청궁의 도사 <a href="/wiki/%EC%99%95%EC%9C%84%EC%95%88%EB%A3%A8" title="왕위안루">왕위안루</a>(왕원록)가 막고굴 제16굴에서 숨겨진 작은 방을 발견하게 된다. 이것이 제17굴로 알려진 장경동이며, 이곳에서 약 5만점의 유물을 발견하였다. 당시의 서구 열강의 각축과 내부 분란, 부패 등으로 국력이 피폐해진 <a href="/wiki/%EC%B2%AD%EB%82%98%EB%9D%BC" title="청나라">청나라</a> 정부와 지방 관리들은 왕위안루의 탐사 활동에 대해 무관심했다. 왕위안루는 고문의 가치를 잘 알고 있던 사람으로 유물 보존을 위해 엄청난 발품과 노력을 기울였으나 어떤 관리도 인적으로나 재정적으로나 지원을 해주지 않았다.</p>
<p>장경동의 보물에 대한 소문을 들은 아우렐 스타인은 <a href="/wiki/1907%EB%85%84" title="1907년">1907년</a> 실의에 빠진 왕위안루에게 재정 보조금을 지급하고, 약 7,000점의 유물을 영국으로 보냈다. 이것이 바로 유럽에 '둔황'의 '막고굴'의 존재를 널리 알린 계기가 되었으며, 둔황의 유물과 배경에 대해 활발한 연구가 시작된 계기이다.</p>
<p>무엇보다도 스타인이 세운 가장 큰 업적 중 하나는 <a href="/wiki/8%EC%84%B8%EA%B8%B0" title="8세기">8세기</a>의 고구려계 <a href="/wiki/%EB%8B%B9%EB%82%98%EB%9D%BC" title="당나라">당나라</a> 명장 고선지의 업적을 세상에 알린 것이라 할 수 있다.</p>

<p><a href="/wiki/1908%EB%85%84" title="1908년">1908년</a> 프랑스의 중국학자 <a href="/wiki/%ED%8F%B4_%ED%8E%A0%EB%A6%AC%EC%98%A4" title="폴 펠리오">폴 펠리오</a>도 비슷한 금액을 지불하고, 아우렐 스타인이 가져갔던 분량만큼의 유물을 프랑스로 보내게 된다. 이후 일본의 승려 <a href="/wiki/%EC%98%A4%ED%83%80%EB%8B%88_%EA%B3%A0%EC%A6%88%EC%9D%B4" title="오타니 고즈이">오타니 고즈이</a>도 5,000점의 유물을 일본으로 가져갔으며, 미국인 <a class="new" href="/w/index.php?title=%EB%9E%AD%EB%8D%98_%EC%9C%84%EB%84%88&amp;action=edit&amp;redlink=1" title="랭던 위너 (없는 문서)">랭던 위너</a><sup class="정리_필요 noprint">[<a href="/wiki/%EC%9C%84%ED%82%A4%EB%B0%B1%EA%B3%BC:%EC%B6%9C%EC%B2%98_%EB%B0%9D%ED%9E%88%EA%B8%B0" title="위키백과:출처 밝히기">출처 필요</a>]</sup> 등도 막고굴의 유물을 모국으로 가져가게 되었다. 폴 펠리오가 가져갔던 유물 중에는 <a href="/wiki/%ED%98%9C%EC%B4%88" title="혜초">혜초</a>의 <a href="/wiki/%EC%99%95%EC%98%A4%EC%B2%9C%EC%B6%95%EA%B5%AD%EC%A0%84" title="왕오천축국전">왕오천축국전</a>의 필사본이 있었다. 오타니가 일본으로 유출한 유물 중에는 일부가 완전히 유출되지 못하고 <a href="/wiki/%EC%A1%B0%EC%84%A0%EC%B4%9D%EB%8F%85%EB%B6%80" title="조선총독부">조선총독부</a>에 기증되어, 현재는 대한민국 <a href="/wiki/%EA%B5%AD%EB%A6%BD%EC%A4%91%EC%95%99%EB%B0%95%EB%AC%BC%EA%B4%80" title="국립중앙박물관">국립중앙박물관</a>(307호)에 전시되어 있다.</p>
<p>당시에 이런 대량의 유물 유출을 두고 중국은 '유물 도둑'이라며 분개하지만, 중국은 당시 왕위안루가 유물 보존을 위해 400킬로미터를 여행하며 지역의 관리들과 교섭·담판을 하며 다녔던 노력과 <a href="/wiki/%EB%AC%B8%ED%99%94_%EB%8C%80%ED%98%81%EB%AA%85" title="문화 대혁명">문화 대혁명</a> 당시 둔황의 상당수 유물들이 <a href="/wiki/%ED%99%8D%EC%9C%84%EB%B3%91" title="홍위병">홍위병</a>들에게 파손되었던 역사에 대해서는 말하지 않고 있다. 현재도 중국 측은 국립중앙박물관에 남은 유물을 반환하라는 요구를 계속하고 있다.</p>
