<p><b>아우렐리우 페르낭데즈 미게우</b>(<span style="font-size: smaller;"><a href="/wiki/%ED%8F%AC%EB%A5%B4%ED%88%AC%EA%B0%88%EC%96%B4" title="포르투갈어">포르투갈어</a>:</span> <span lang="pt" xml:lang="pt">Aurélio Fernández Miguel</span>, <a href="/wiki/1964%EB%85%84" title="1964년">1964년</a> <a href="/wiki/3%EC%9B%94_10%EC%9D%BC" title="3월 10일">3월 10일</a> ~ )은 전 <a href="/wiki/%EB%B8%8C%EB%9D%BC%EC%A7%88" title="브라질">브라질</a>의 <a href="/wiki/%EC%9C%A0%EB%8F%84" title="유도">유도</a> 선수로 라이트헤비급에서 활약하면서 <a class="mw-redirect" href="/wiki/1988%EB%85%84_%EC%84%9C%EC%9A%B8_%EC%98%AC%EB%A6%BC%ED%94%BD" title="1988년 서울 올림픽">1988년 서울 올림픽</a> <a href="/wiki/%EA%B8%88%EB%A9%94%EB%8B%AC" title="금메달">금메달</a>리스트이다. 후에 정계에 입문하였다.</p>
<h2><span class="mw-headline" id=".EC.84.A0.EC.88.98_.EA.B2.BD.EB.A0.A5">선수 경력</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%95%84%EC%9A%B0%EB%A0%90%EB%A6%AC%EC%9A%B0_%EB%AF%B8%EA%B2%8C%EC%9A%B0&amp;action=edit&amp;section=1" title="부분 편집: 선수 경력">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p><a href="/wiki/%EC%83%81%ED%8C%8C%EC%9A%B8%EB%A3%A8" title="상파울루">상파울루</a>에서 태어났다. <a href="/wiki/%EA%B8%B0%EA%B4%80%EC%A7%80%EC%97%BC" title="기관지염">기관지염</a>과 부친의 주장으로 인하여 미게우는 4세 때부터 유도 훈련을 시작하였다. 시초에 유도를 싫어한 그는 어린이로서 경연과 토너먼트의 난폭함에 겁에 질렸다고 한다. 시간이 지나면서 스포츠를 좋아하게 된 미게우는 <a href="/wiki/1972%EB%85%84" title="1972년">1972년</a> 첫 타이틀을 우승하였다. 그러고나서 <a href="/wiki/%EC%83%81%ED%8C%8C%EC%9A%B8%EB%A3%A8_%EC%A3%BC" title="상파울루 주">상파울루 주</a> 토너먼트를 많이 이기고 <a href="/wiki/1980%EB%85%84" title="1980년">1980년</a>으로 봐서 그는 주에서 최고 유도 선수로서 인정을 받았다.</p>
<p>후에 미게우는 국제적으로 나가기 시작하였고 <a href="/wiki/1983%EB%85%84" title="1983년">1983년</a> <a href="/wiki/%EC%B9%B4%EB%9D%BC%EC%B9%B4%EC%8A%A4" title="카라카스">카라카스</a>에서 열린 <a href="/wiki/%ED%8C%AC%EC%95%84%EB%A9%94%EB%A6%AC%EC%B9%B8_%EA%B2%8C%EC%9E%84" title="팬아메리칸 게임">팬아메리칸 게임</a>에서 <a href="/wiki/%EC%9D%80%EB%A9%94%EB%8B%AC" title="은메달">은메달</a>을 획득하였다. 4년 후의 팬아메리칸 게임에서는 <a href="/wiki/%EA%B8%88%EB%A9%94%EB%8B%AC" title="금메달">금메달</a>을 따고, 세계 선수권에서 <a href="/wiki/%EB%8F%99%EB%A9%94%EB%8B%AC" title="동메달">동메달</a>을 따기도 하였다.</p>
<p><a class="mw-redirect" href="/wiki/1988%EB%85%84_%EC%84%9C%EC%9A%B8_%EC%98%AC%EB%A6%BC%ED%94%BD" title="1988년 서울 올림픽">1988년 서울 올림픽</a>에서 <a href="/wiki/%EC%84%9C%EB%8F%85" title="서독">서독</a>의 <a class="new" href="/w/index.php?title=%EB%A7%88%EB%A5%B4%ED%81%AC_%EB%A7%88%EC%9D%BC%EB%A7%81&amp;action=edit&amp;redlink=1" title="마르크 마일링 (없는 문서)">마르크 마일링</a>을 꺾고 우승하면서 올림픽 챔피언이 되었다. 4년 후, <a href="/wiki/1992%EB%85%84_%ED%95%98%EA%B3%84_%EC%98%AC%EB%A6%BC%ED%94%BD" title="1992년 하계 올림픽">바르셀로나 올림픽</a>에서 9위를 하고 이듬해 세계 선수권에서 <a href="/wiki/%ED%97%9D%EA%B0%80%EB%A6%AC" title="헝가리">헝가리</a>의 <a class="new" href="/w/index.php?title=%EC%BD%94%EB%B0%94%EC%B8%A0_%EC%95%88%ED%83%88&amp;action=edit&amp;redlink=1" title="코바츠 안탈 (없는 문서)">코바츠 안탈</a>에게 패하여 은메달을 땄다.</p>
<p><a class="mw-redirect" href="/wiki/1996%EB%85%84_%EC%95%A0%ED%8B%80%EB%9E%9C%ED%83%80_%EC%98%AC%EB%A6%BC%ED%94%BD" title="1996년 애틀랜타 올림픽">1996년 애틀랜타 올림픽</a>에서 동메달을 따고 이듬해 세계 선수권에서 은메달로 선수 경력을 마쳤다.</p>
<h2><span class="mw-headline" id=".EC.A0.95.EC.B9.98_.EA.B2.BD.EB.A0.A5">정치 경력</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%95%84%EC%9A%B0%EB%A0%90%EB%A6%AC%EC%9A%B0_%EB%AF%B8%EA%B2%8C%EC%9A%B0&amp;action=edit&amp;section=2" title="부분 편집: 정치 경력">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p><a href="/wiki/2004%EB%85%84" title="2004년">2004년</a> <a href="/wiki/10%EC%9B%94" title="10월">10월</a> 성공적으로 상파울루의 시의회를 위하여 입후보하였으며, 4년 후 공화당의 기치 아래 두번째 임기를 위하여 재선되었다.</p>
