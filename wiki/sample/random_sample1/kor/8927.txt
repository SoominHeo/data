<p>《<b>아이돌 스타 올림픽</b>》은 <a href="/wiki/2012%EB%85%84_%ED%95%98%EA%B3%84_%EC%98%AC%EB%A6%BC%ED%94%BD" title="2012년 하계 올림픽">2012년 런던 올림픽</a>의 개최를 기념하여 <a href="/wiki/2012%EB%85%84" title="2012년">2012년</a> <a href="/wiki/7%EC%9B%94_25%EC%9D%BC" title="7월 25일">7월 25일</a>과 <a href="/wiki/7%EC%9B%94_26%EC%9D%BC" title="7월 26일">7월 26일</a>에 방송한 <a href="/wiki/%EB%AC%B8%ED%99%94%EB%B0%A9%EC%86%A1" title="문화방송">MBC</a>의 예능 프로그램이다. <a href="/wiki/%EC%84%9C%EC%9A%B8%ED%8A%B9%EB%B3%84%EC%8B%9C" title="서울특별시">서울</a> <a class="mw-redirect" href="/wiki/%EB%AA%A9%EB%8F%99_%EC%A3%BC%EA%B2%BD%EA%B8%B0%EC%9E%A5" title="목동 주경기장">목동 종합운동장</a>에서 <a class="mw-redirect" href="/wiki/%EC%9C%A1%EC%83%81" title="육상">육상</a> 종목, <a href="/wiki/%EC%9A%A9%EC%9D%B8%EB%8C%80%ED%95%99%EA%B5%90" title="용인대학교">용인대학교</a> 종합 체육관에서 <a href="/wiki/%ED%8E%9C%EC%8B%B1" title="펜싱">펜싱</a> 샤브르와 <a href="/wiki/%ED%83%81%EA%B5%AC" title="탁구">탁구</a> 남녀 혼합 복식, <a href="/wiki/%EC%96%91%EA%B6%81" title="양궁">양궁</a> 단체전 경기가 열렸다.</p>
