<p><b>아이란</b>(Ayran)은 <a href="/wiki/%EC%86%8C%EA%B8%88" title="소금">소금</a>이 곁들여진 차가운 <a href="/wiki/%EC%9A%94%EA%B5%AC%EB%A5%B4%ED%8A%B8" title="요구르트">요구르트</a> 음료이다. <a href="/wiki/%ED%84%B0%ED%82%A4" title="터키">터키</a>의 전체 부분에서 유명하며 터키 이외에도 <a href="/wiki/%EC%95%84%EB%A5%B4%EB%A9%94%EB%8B%88%EC%95%84" title="아르메니아">아르메니아</a> (<i>탄</i>으로 불린다.), <a href="/wiki/%EC%95%84%EC%A0%9C%EB%A5%B4%EB%B0%94%EC%9D%B4%EC%9E%94" title="아제르바이잔">아제르바이잔</a>, <a href="/wiki/%EC%9D%B4%EB%9E%80" title="이란">이란</a>, <a href="/wiki/%EB%B0%9C%EC%B9%B8_%EB%B0%98%EB%8F%84" title="발칸 반도">발칸 반도</a>, <a href="/wiki/%EC%B9%B4%EC%9E%90%ED%9D%90%EC%8A%A4%ED%83%84" title="카자흐스탄">카자흐스탄</a>, <a href="/wiki/%ED%82%A4%EB%A5%B4%EA%B8%B0%EC%8A%A4%EC%8A%A4%ED%83%84" title="키르기스스탄">키르기스스탄</a>, <a href="/wiki/%EB%A0%88%EB%B0%94%EB%85%BC" title="레바논">레바논</a>, <a href="/wiki/%EC%8B%9C%EB%A6%AC%EC%95%84" title="시리아">시리아</a>, <a href="/wiki/%ED%82%A4%ED%94%84%EB%A1%9C%EC%8A%A4" title="키프로스">키프로스</a>, <a href="/wiki/%EC%BA%85%EC%B9%B4%EC%8A%A4" title="캅카스">캅카스</a> 지방에서도 찾아볼 수 있다. <a href="/wiki/%EB%AC%BC" title="물">물</a>과 <a href="/wiki/%EC%9A%94%EA%B5%AC%EB%A5%B4%ED%8A%B8" title="요구르트">요구르트</a>가 주 재료로 들어가며 보관할 때는 <a href="/wiki/%EB%83%89%EC%9E%A5%EA%B3%A0" title="냉장고">냉장고</a>에 보관되어 서빙 온도 또한 차갑다. 여름 동안에는 <a href="/wiki/%EA%B7%B8%EB%A6%B4" title="그릴">그릴</a>에 구운 고기나 쌀과 같이 제공되기도 한다.</p>
