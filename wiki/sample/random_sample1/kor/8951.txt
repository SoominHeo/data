<h3><span class="mw-headline" id=".EC.8B.A4.EC.A0.9C_.EC.9D.B8.EB.AC.BC">실제 인물</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%95%84%EC%9D%B4%EC%9E%91&amp;action=edit&amp;section=2" title="부분 편집: 실제 인물">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<ul>
<li><a href="/wiki/%EC%95%84%EC%9D%B4%EC%9E%91_%EB%89%B4%ED%84%B4" title="아이작 뉴턴">아이작 뉴턴</a>은 <a href="/wiki/%EC%98%81%EA%B5%AD" title="영국">영국</a>의 <a href="/wiki/%EB%AC%BC%EB%A6%AC%ED%95%99%EC%9E%90" title="물리학자">물리학자</a>, <a href="/wiki/%EC%88%98%ED%95%99%EC%9E%90" title="수학자">수학자</a>, <a href="/wiki/%EC%B2%9C%EB%AC%B8%ED%95%99%EC%9E%90" title="천문학자">천문학자</a>, <a class="mw-redirect" href="/wiki/%EA%B4%91%ED%95%99%EC%9E%90" title="광학자">광학자</a>, <a class="new" href="/w/index.php?title=%EC%9E%90%EC%97%B0%EC%B2%A0%ED%95%99%EC%9E%90&amp;action=edit&amp;redlink=1" title="자연철학자 (없는 문서)">자연철학자</a>이자 <a class="mw-redirect" href="/wiki/%EC%97%B0%EA%B8%88%EC%88%A0%EC%82%AC" title="연금술사">연금술사</a>, <a class="mw-redirect" href="/wiki/%EC%8B%A0%ED%95%99%EC%9E%90" title="신학자">신학자</a>이다.</li>
<li><a class="mw-redirect" href="/wiki/%EC%95%84%EC%9D%B4%EC%9E%91_%ED%97%A4%EC%9D%B4%EC%A6%88" title="아이작 헤이즈">아이작 헤이즈</a>는 <a href="/wiki/%EB%AF%B8%EA%B5%AD" title="미국">미국</a>의 <a class="mw-disambig" href="/wiki/%EC%86%8C%EC%9A%B8" title="소울">소울</a>, <a href="/wiki/%ED%8E%91%ED%81%AC" title="펑크">펑크</a> <a href="/wiki/%EC%8B%B1%EC%96%B4%EC%86%A1%EB%9D%BC%EC%9D%B4%ED%84%B0" title="싱어송라이터">가수 겸 작곡가</a>, 레코드 제작자, 편곡자, <a href="/wiki/%EC%9E%91%EA%B3%A1%EA%B0%80" title="작곡가">작곡가</a>, <a href="/wiki/%EB%B0%B0%EC%9A%B0" title="배우">배우</a>이다.</li>
<li><a href="/wiki/%EC%95%84%EC%9D%B4%EC%9E%91_%EC%95%84%EC%8B%9C%EB%AA%A8%ED%94%84" title="아이작 아시모프">아이작 아시모프</a>는 <a href="/wiki/%EB%9F%AC%EC%8B%9C%EC%95%84" title="러시아">러시아</a> 태생의 <a href="/wiki/%EB%AF%B8%EA%B5%AD" title="미국">미국</a> <a class="mw-redirect" href="/wiki/%EA%B3%BC%ED%95%99_%EC%86%8C%EC%84%A4" title="과학 소설">과학 소설가</a>이자 <a href="/wiki/%EC%9E%91%EA%B0%80" title="작가">저술가</a>이다.</li>
<li><a href="/wiki/%EC%95%84%EC%9D%B4%EC%9E%91_%ED%97%B4%ED%94%84%EC%8A%A4%ED%85%8C%EB%93%9C%EB%9D%BC%EC%9D%B4%ED%8A%B8" title="아이작 헴프스테드라이트">아이작 헴프스테드라이트</a>는 <a href="/wiki/%EC%9E%89%EA%B8%80%EB%9E%9C%EB%93%9C" title="잉글랜드">잉글랜드</a>의 <a href="/wiki/%EB%B0%B0%EC%9A%B0" title="배우">배우</a>이다.</li>
<li><a href="/wiki/%EC%95%84%EC%9D%B4%EC%9E%91_%EC%8A%A4%ED%84%B4" title="아이작 스턴">아이작 스턴</a>은 <a href="/wiki/%EB%AF%B8%EA%B5%AD" title="미국">미국</a>의 <a class="mw-redirect" href="/wiki/%EB%B0%94%EC%9D%B4%EC%98%AC%EB%A6%AC%EB%8B%88%EC%8A%A4%ED%8A%B8" title="바이올리니스트">바이올리니스트</a>이다.</li>
<li><a href="/wiki/%EC%95%84%EC%9D%B4%EC%9E%91_%ED%97%90" title="아이작 헐">아이작 헐</a>은 미국 해군 사령관이다.</li>
<li><a href="/wiki/%EC%95%84%EC%9D%B4%EC%9E%91_%EB%A9%94%EB%A6%BF_%EC%8B%B1%EC%96%B4" title="아이작 메릿 싱어">아이작 메릿 싱어</a>는 <a href="/wiki/%EB%AF%B8%EA%B5%AD" title="미국">미국</a>의 <a href="/wiki/%EB%B0%9C%EB%AA%85%EA%B0%80" title="발명가">발명가</a>이다.</li>
<li><a href="/wiki/%EC%95%84%EC%9D%B4%EC%9E%91_%EB%B8%8C%EB%A1%9D" title="아이작 브록">아이작 브록</a>은 <a href="/wiki/%EB%AF%B8%EA%B5%AD" title="미국">미국</a>의 록 밴드 <a href="/wiki/%EB%AA%A8%EB%94%94%EC%8A%A4%ED%8A%B8_%EB%A7%88%EC%9A%B0%EC%8A%A4" title="모디스트 마우스">모디스트 마우스</a>의 <a href="/wiki/%EA%B0%80%EC%88%98" title="가수">리드 보컬</a>이자 <a class="mw-redirect" href="/wiki/%EA%B8%B0%ED%83%80%EB%A6%AC%EC%8A%A4%ED%8A%B8" title="기타리스트">기타리스트</a>, <a href="/wiki/%EC%9E%91%EA%B3%A1%EA%B0%80" title="작곡가">작곡가</a>이다.</li>
<li><a href="/wiki/%EC%95%84%EC%9D%B4%EC%9E%91_%EC%85%B8%EB%B9%84" title="아이작 셸비">아이작 셸비</a>은 미국 켄터키 주의 초대 및 5대 주지사이자 버지니아 주와 노스캐롤라이나 주의 각 주 의회 의원이다.</li>
<li><a href="/wiki/%EC%95%84%EC%9D%B4%EC%9E%91_%EB%B2%84%ED%8A%B8" title="아이작 버트">아이작 버트</a>는 <a class="new" href="/w/index.php?title=%EC%9E%90%EC%B9%98_%EC%97%B0%EB%A7%B9&amp;action=edit&amp;redlink=1" title="자치 연맹 (없는 문서)">자치 연맹</a>(후에 <a class="new" href="/w/index.php?title=%EC%95%84%EC%9D%BC%EB%9E%9C%EB%93%9C_%EC%9D%98%ED%9A%8C%EB%8B%B9&amp;action=edit&amp;redlink=1" title="아일랜드 의회당 (없는 문서)">아일랜드 의회당</a>으로 개명)의 창립자이자 초대 지도자이다.</li>
<li><a href="/wiki/%EC%95%84%EC%9D%B4%EC%9E%91_%EC%95%84%EC%BF%A0%EB%83%90" title="아이작 아쿠냐">아이작 아쿠냐</a>는 <a href="/wiki/%EB%AF%B8%EA%B5%AD" title="미국">미국</a>의 축구 선수이다.</li>
<li><a href="/wiki/%EC%95%84%EC%9D%B4%EC%9E%91_%EB%8D%94%EC%8A%A4%ED%8A%B8" title="아이작 더스트">아이작 더스트</a>는 <a href="/wiki/%EB%AF%B8%EA%B5%AD" title="미국">미국</a> 출신의 <a class="mw-redirect" href="/wiki/%EB%B0%A9%EC%86%A1%EC%9D%B8" title="방송인">방송인</a>이자 <a href="/wiki/%EC%98%81%EC%96%B4" title="영어">영어</a> <a href="/wiki/%EA%B0%95%EC%82%AC" title="강사">강사</a>이다.</li>
<li><a href="/wiki/%EC%95%84%EC%9D%B4%EC%9E%91_(%EB%B2%A0%ED%8A%B8%EB%82%A8_%EA%B0%80%EC%88%98)" title="아이작 (베트남 가수)">아이작 (베트남 가수)</a></li>
