<p><b>아이즈미사토 정</b>(<span style="font-size: smaller;"><a href="/wiki/%EC%9D%BC%EB%B3%B8%EC%96%B4" title="일본어">일본어</a>:</span> <span lang="ja" xml:lang="ja">会津美里町</span>)은 <a href="/wiki/%EC%9D%BC%EB%B3%B8" title="일본">일본</a> <a href="/wiki/%ED%9B%84%EC%BF%A0%EC%8B%9C%EB%A7%88_%ED%98%84" title="후쿠시마 현">후쿠시마 현</a> <a href="/wiki/%EC%98%A4%EB%88%84%EB%A7%88_%EA%B5%B0" title="오누마 군">오누마 군</a>의 정이다. 면적은 276.37km²이고 인구는 <a href="/wiki/2010%EB%85%84" title="2010년">2010년</a> <a href="/wiki/1%EC%9B%94_1%EC%9D%BC" title="1월 1일">1월 1일</a> 기준으로 23,168명이다.</p>

<h2><span class="mw-headline" id=".EC.A7.80.EB.A6.AC">지리</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%95%84%EC%9D%B4%EC%A6%88%EB%AF%B8%EC%82%AC%ED%86%A0_%EC%A0%95&amp;action=edit&amp;section=1" title="부분 편집: 지리">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>후쿠시마 현 서부에 있는 아이즈 분지의 남서부를 차지한다. 중앙을 미야카와 강이 남서쪽에서 북동쪽으로 흐르고 <a href="/wiki/%EC%95%84%EC%9D%B4%EC%A6%88%EC%99%80%EC%B9%B4%EB%A7%88%EC%93%B0_%EC%8B%9C" title="아이즈와카마쓰 시">아이즈와카마쓰 시</a>와의 경계 부근에서는 북쪽으로 흐름을 바꾸어 이윽고 <a href="/wiki/%EC%95%84%EA%B0%80%EB%85%B8_%EA%B0%95" title="아가노 강">아가노 강</a>과 합류한다. 북부·동부는 분지의 평야부(아이즈다이라)가 펼쳐져 있고 <a href="/wiki/%EA%B3%A0%EC%8B%9C%ED%9E%88%EC%B9%B4%EB%A6%AC" title="고시히카리">고시히카리</a>, 히토메보레 등을 중심으로 한 벼농사가 번성한다. 남부·서부는 산간지이고 아이즈 고원으로 연결된다. 산기슭에서는 <a href="/wiki/%ED%8F%AC%EB%8F%84" title="포도">포도</a>, <a href="/wiki/%EC%9D%B8%EC%82%BC" title="인삼">인삼</a>, <a href="/wiki/%EB%A9%94%EB%B0%80" title="메밀">메밀</a>, <a class="mw-redirect" href="/wiki/%EC%95%BC%EC%B1%84" title="야채">야채</a>류 등이 재배되고 있다.</p>
