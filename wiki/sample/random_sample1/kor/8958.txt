<p><b>아이튠스</b>(<span lang="en" xml:lang="en">iTunes</span>, <span style="font-size: small;"><a href="/wiki/%EC%99%B8%EA%B5%AD%EC%96%B4%EC%9D%98_%ED%95%9C%EA%B8%80_%ED%91%9C%EA%B8%B0" title="외국어의 한글 표기">통용</a>:</span> 아이튠즈)는 <a href="/wiki/%EC%95%A0%ED%94%8C_(%EA%B8%B0%EC%97%85)" title="애플 (기업)">애플</a>사가 만든 멀티미디어 플레이어 및 <a href="/wiki/%EC%95%84%EC%9D%B4%ED%8C%9F" title="아이팟">아이팟</a>용 동기화 프로그램이다. 아이튠스는 컴퓨터 내의 <a href="/wiki/%EC%9D%8C%EC%95%85" title="음악">음악</a>과 <a href="/wiki/%EB%8F%99%EC%98%81%EC%83%81" title="동영상">동영상</a>을 관리하고, <a class="mw-redirect" href="/wiki/%EC%95%84%EC%9D%B4%ED%8A%A0%EC%A6%88_%EC%8A%A4%ED%86%A0%EC%96%B4" title="아이튠즈 스토어">아이튠즈 스토어</a>와 <a class="mw-redirect" href="/wiki/%EC%95%B1%EC%8A%A4%ED%86%A0%EC%96%B4" title="앱스토어">앱스토어</a>에 접속하여 음악이나 <a href="/wiki/%EB%AE%A4%EC%A7%81_%EB%B9%84%EB%94%94%EC%98%A4" title="뮤직 비디오">뮤직 비디오</a>, <a href="/wiki/%EC%98%81%ED%99%94" title="영화">영화</a>, <a href="/wiki/%EC%9D%91%EC%9A%A9_%EC%86%8C%ED%94%84%ED%8A%B8%EC%9B%A8%EC%96%B4" title="응용 소프트웨어">앱</a> 등을 구매할 수 있으며, <a href="/wiki/%EC%95%84%EC%9D%B4%ED%8C%9F" title="아이팟">아이팟</a>과 <a href="/wiki/%EC%95%A0%ED%94%8C_TV" title="애플 TV">애플 TV</a>, <a class="mw-redirect" href="/wiki/%EB%AA%A8%ED%86%A0%EB%A1%9C%EB%9D%BC" title="모토로라">모토로라</a> ROKR, <a href="/wiki/%EC%95%84%EC%9D%B4%ED%8F%B0" title="아이폰">아이폰</a>,<a href="/wiki/%EC%95%84%EC%9D%B4%ED%8C%A8%EB%93%9C" title="아이패드">아이패드</a> 등 아이튠즈와 연동되는 장치에 데이터를 전송하고 관리하며 <a class="mw-redirect" href="/wiki/IOS_(%EC%95%A0%ED%94%8C)" title="IOS (애플)">iOS</a>를 업그레이드할 수 있는 기능을 갖고 있다. 이외에도 음악 CD의 정보를 내려받아 리핑할 수 있는 기능을 가지고 있다. 아이튠스는 <a class="mw-redirect" href="/wiki/OS_X" title="OS X">OS X</a>용과 <a class="mw-redirect" href="/wiki/%EB%A7%88%EC%9D%B4%ED%81%AC%EB%A1%9C%EC%86%8C%ED%94%84%ED%8A%B8_%EC%9C%88%EB%8F%84" title="마이크로소프트 윈도">윈도</a>용 버전이 있고 <a href="/wiki/%EC%95%A0%ED%94%8C_(%EA%B8%B0%EC%97%85)" title="애플 (기업)">애플</a> 웹사이트에서 무료로 다운로드받을 수 있다. <a class="mw-redirect" href="/wiki/%EB%A7%88%EC%9D%B4%ED%81%AC%EB%A1%9C%EC%86%8C%ED%94%84%ED%8A%B8_%EC%9C%88%EB%8F%84" title="마이크로소프트 윈도">윈도</a>용은 32비트와 64비트가 따로 있다.</p>
<p>아이튠스는 <a href="/wiki/ID3" title="ID3">ID 태그</a>를 기반으로 음악과 동영상을 관리한다. <a class="mw-redirect" href="/wiki/%ED%8C%9F%EC%BA%90%EC%8A%A4%ED%8C%85" title="팟캐스팅">팟캐스트</a>를 구독하고 관리할 수 있으며 자신의 보관함을 네트워크 상의 다른 아이튠스 사용자와 공유할 수 있다. 아이튠스와 동기화할 수 있는 장비의 대수에는 제한이 없고 음악 파일 역시 회수 제한 없이 음악 시디로 복사할 수 있다. 아이튠스 7에는 앨범 사진을 보면서 빠르게 음악을 검색할 수 있는 <a href="/wiki/%EC%BB%A4%EB%B2%84_%ED%94%8C%EB%A1%9C" title="커버 플로">커버 플로</a> 기능과 아이튠스 스토어에서 구입한 파일을 같은 아이튠스 스토어 계정을 사용하는 컴퓨터로 전송하는 양방향 전송 기능이 추가되었다.</p>
<p><a class="mw-redirect" href="/wiki/%EB%A7%88%EC%9D%B4%ED%81%AC%EB%A1%9C%EC%86%8C%ED%94%84%ED%8A%B8_%EC%9C%88%EB%8F%84" title="마이크로소프트 윈도">윈도</a>용 초기 버전들에서는 맥 OS판에서와 비교하여 구동 속도가 느린 편이 었지만 버전이 올라가면서 속도는 크게 향상되었으며, <a class="mw-redirect" href="/wiki/%ED%80%B5%ED%83%80%EC%9E%84_%ED%94%8C%EB%A0%88%EC%9D%B4%EC%96%B4" title="퀵타임 플레이어">퀵타임 플레이어</a> 및 <a href="/wiki/%EC%82%AC%ED%8C%8C%EB%A6%AC_(%EC%9B%B9_%EB%B8%8C%EB%9D%BC%EC%9A%B0%EC%A0%80)" title="사파리 (웹 브라우저)">사파리</a> 등의 프로그램과 더 이상 함께 설치되지 않고 독립적으로 설치할 수 있게 되었다.</p>
<p>음악을 다운로드받을 때, mp3 포맷이 아닌 m4a 포맷으로 나온다. 별도의 mp3 변환기를 이용하여 mp3으로 변환할 수 있다.</p>
<p>음악을 다운로드하고 싶으면, <a href="/wiki/%EB%8C%80%ED%95%9C%EB%AF%BC%EA%B5%AD" title="대한민국">대한민국</a> 계정이 아닌 아닌 <a href="/wiki/%EB%AF%B8%EA%B5%AD" title="미국">미국</a> 계정으로 아이튠스에 로그인해서 이용해야 한다. <a href="/wiki/%EB%8C%80%ED%95%9C%EB%AF%BC%EA%B5%AD" title="대한민국">대한민국</a> 아이튠스에서는 음악다운 지원을 하지 않기 때문이다. 해외신판이 되는 결제용 카드를 Profile에서 등록해야 하며 <a class="mw-redirect" href="/wiki/%EB%A7%88%EC%8A%A4%ED%83%80%EC%B9%B4%EB%93%9C" title="마스타카드">마스타카드</a>, <a href="/wiki/%EB%B9%84%EC%9E%90%EC%B9%B4%EB%93%9C" title="비자카드">비자카드</a>가 달린 카드 중에서도 Profile에서 결제용으로 등록되지 않는 경우가 간혹 있기 때문에 확인해야 한다.</p>
