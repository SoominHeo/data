<p><b>아이팟리눅스</b>(iPodLinux)는 <a href="/wiki/%EC%95%A0%ED%94%8C_(%EA%B8%B0%EC%97%85)" title="애플 (기업)">애플</a>의 <a href="/wiki/%EC%95%84%EC%9D%B4%ED%8C%9F" title="아이팟">아이팟</a>에서 동작하도록 설계된 <a class="new" href="/w/index.php?title=%CE%9CClinux&amp;action=edit&amp;redlink=1" title="ΜClinux (없는 문서)">μClinux</a> 기반의 <a href="/wiki/%EB%A6%AC%EB%88%85%EC%8A%A4_%EB%B0%B0%ED%8F%AC%ED%8C%90" title="리눅스 배포판">리눅스 배포판</a>이다. 아이팟리눅스 <a href="/wiki/%EB%A6%AC%EB%88%85%EC%8A%A4_%EC%BB%A4%EB%84%90" title="리눅스 커널">커널</a>이 <a href="/wiki/%EB%B6%80%ED%8C%85" title="부팅">시동</a>되면 애플의 아이팟 운영 체제로 대체하여 자동으로 팟질라(Podzilla, <a class="new" href="/w/index.php?title=%EB%B9%84%EB%94%94%EC%98%A4_%ED%94%8C%EB%A0%88%EC%9D%B4%EC%96%B4&amp;action=edit&amp;redlink=1" title="비디오 플레이어 (없는 문서)">비디오 플레이어</a>, <a href="/wiki/%EC%9D%B4%EB%AF%B8%EC%A7%80_%EB%B7%B0%EC%96%B4" title="이미지 뷰어">이미지 뷰어</a>, <a href="/wiki/%EB%AA%85%EB%A0%B9_%EC%A4%84_%EC%9D%B8%ED%84%B0%ED%8E%98%EC%9D%B4%EC%8A%A4" title="명령 줄 인터페이스">명령 셸</a>, <a href="/wiki/%EB%B9%84%EB%94%94%EC%98%A4_%EA%B2%8C%EC%9E%84" title="비디오 게임">오락</a>, <a href="/wiki/%EB%B9%84%EB%94%94%EC%98%A4_%EA%B2%8C%EC%9E%84_%EC%BD%98%EC%86%94" title="비디오 게임 콘솔">비디오 게임 콘솔</a>용 <a href="/wiki/%EC%97%90%EB%AE%AC%EB%A0%88%EC%9D%B4%ED%84%B0" title="에뮬레이터">에뮬레이터</a>, <a class="mw-redirect" href="/wiki/%ED%94%84%EB%A1%9C%EA%B7%B8%EB%9E%98%EB%B0%8D" title="프로그래밍">프로그래밍</a> 데모, 기타 실험적이거나 미완성된 <a href="/wiki/%EC%86%8C%ED%94%84%ED%8A%B8%EC%9B%A8%EC%96%B4" title="소프트웨어">소프트웨어</a>가 포함된 <a href="/wiki/%EA%B7%B8%EB%9E%98%ED%94%BD_%EC%82%AC%EC%9A%A9%EC%9E%90_%EC%9D%B8%ED%84%B0%ED%8E%98%EC%9D%B4%EC%8A%A4" title="그래픽 사용자 인터페이스">GUI</a> 및 런처)를 호출한다.</p>
<p>2011년 기준으로 이 프로젝트는 여러 해에 걸쳐 활성화되지 못했으며 해당 웹사이트 또한 더 이상 관리되지 않고 있다. 애플 아이팟의 오픈 소스 소프트웨어의 차후 개발은 <a href="/wiki/%EB%9D%BD%EB%B0%95%EC%8A%A4" title="락박스">락박스 프로젝트</a>와 <a class="external text" href="http://freemyipod.org/" rel="nofollow">FreeMyIpod.org</a>에서 계속되고 있다.</p>
