<p><b>아이폰 OS 3</b> (iPhone OS 3), 후에 변경된 이름으로 <b>iOS 3</b>는 <a href="/wiki/%EC%95%A0%ED%94%8C_(%EA%B8%B0%EC%97%85)" title="애플 (기업)">애플</a>이 개발하여 <a href="/wiki/%EC%95%84%EC%9D%B4%ED%8F%B0_(1%EC%84%B8%EB%8C%80)" title="아이폰 (1세대)">아이폰 (1세대)</a>, <a href="/wiki/%EC%95%84%EC%9D%B4%ED%8F%B0_3G" title="아이폰 3G">아이폰 3G</a>, <a href="/wiki/%EC%95%84%EC%9D%B4%ED%8F%B0_3GS" title="아이폰 3GS">아이폰 3GS</a>, <a class="new" href="/w/index.php?title=%EC%95%84%EC%9D%B4%ED%8C%9F_%ED%84%B0%EC%B9%98_(1%EC%84%B8%EB%8C%80)&amp;action=edit&amp;redlink=1" title="아이팟 터치 (1세대) (없는 문서)">아이팟 터치 (1세대)</a>, <a class="new" href="/w/index.php?title=%EC%95%84%EC%9D%B4%ED%8C%9F_%ED%84%B0%EC%B9%98_(2%EC%84%B8%EB%8C%80)&amp;action=edit&amp;redlink=1" title="아이팟 터치 (2세대) (없는 문서)">아이팟 터치 (2세대)</a>, <a class="new" href="/w/index.php?title=%EC%95%84%EC%9D%B4%ED%8C%9F_%ED%84%B0%EC%B9%98_(3%EC%84%B8%EB%8C%80)&amp;action=edit&amp;redlink=1" title="아이팟 터치 (3세대) (없는 문서)">아이팟 터치 (3세대)</a>, <a href="/wiki/%EC%95%84%EC%9D%B4%ED%8C%A8%EB%93%9C_(1%EC%84%B8%EB%8C%80)" title="아이패드 (1세대)">아이패드 (1세대)</a> 등에 탑재되었던 <a class="mw-redirect" href="/wiki/IOS_(%EC%95%A0%ED%94%8C)" title="IOS (애플)">iOS</a> 계열의 <a href="/wiki/%EC%8A%A4%EB%A7%88%ED%8A%B8%ED%8F%B0" title="스마트폰">스마트폰</a> <a href="/wiki/%EC%9A%B4%EC%98%81_%EC%B2%B4%EC%A0%9C" title="운영 체제">운영 체제</a> 이다. <a href="/wiki/%EC%95%84%EC%9D%B4%ED%8F%B0_OS_2" title="아이폰 OS 2">아이폰 OS 2</a>까지 <a href="/wiki/%EC%95%84%EC%9D%B4%ED%8F%B0" title="아이폰">아이폰</a>에만 탑재되어 <a class="mw-redirect" href="/wiki/%EC%95%84%EC%9D%B4%ED%8F%B0_OS" title="아이폰 OS">아이폰 OS</a>라 불렸으나, 아이폰 OS 3부터 <a href="/wiki/%EC%95%84%EC%9D%B4%ED%8C%A8%EB%93%9C" title="아이패드">아이패드</a> 에도 탑재되어, 이름을 iOS 3으로 변경하였다. 후속 버전은 <a href="/wiki/IOS_4" title="IOS 4">iOS 4</a>이다.</p>
<h2><span class="mw-headline" id=".EC.83.88.EB.A1.9C.EC.9A.B4_.EA.B8.B0.EB.8A.A5">새로운 기능</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%95%84%EC%9D%B4%ED%8F%B0_OS_3&amp;action=edit&amp;section=1" title="부분 편집: 새로운 기능">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>오려내기, 복사하기, 붙이기, 스포트라이트, 인앱 결제 등의 기능이 아이폰 OS 3에서 새로이 추가되었다.</p>
<h2><span class="mw-headline" id=".EB.AC.B8.EC.A0.9C">문제</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%95%84%EC%9D%B4%ED%8F%B0_OS_3&amp;action=edit&amp;section=2" title="부분 편집: 문제">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>몇몇 <a href="/wiki/%EC%95%84%EC%9D%B4%ED%8F%B0" title="아이폰">아이폰</a>과 <a href="/wiki/%EC%95%84%EC%9D%B4%ED%8C%9F_%ED%84%B0%EC%B9%98" title="아이팟 터치">아이팟 터치</a> 사용자들이 기기가 잠잔 상태에서 깨우기 버튼을 눌러도 기기가 켜지지 않는 문제를 보고하였으며, 애플이 출시한 업데이트 버전인 아이폰 OS 3.1.2에서 이 문제가 해결되었다. 몇몇 <a href="/wiki/%EC%95%84%EC%9D%B4%ED%8C%A8%EB%93%9C" title="아이패드">아이패드</a> 사용자들은 와이파이 문제 등을 보고하였으며, 역시 추후에 출시된 업데이트 버전인 iOS 3.2.1에서 해결되었다.</p>
