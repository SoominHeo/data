<h1 class="firstHeading" id="firstHeading" lang="ko">아자이 칸</h1>
<div class="mw-body-content" id="bodyContent">
<div id="siteSub">위키백과, 우리 모두의 백과사전.</div>
<div id="contentSub"></div>
<div class="mw-jump" id="jump-to-nav">
					이동:					<a href="#mw-head">둘러보기</a>, 					<a href="#p-search">검색</a>
</div>
<div class="mw-content-ltr" dir="ltr" id="mw-content-text" lang="ko"><script>function mfTempOpenSection(id){var block=document.getElementById("mf-section-"+id);block.className+=" open-block";block.previousSibling.className+=" open-block";}</script><p><b>아자이 칸</b> 또는 <b>아다이 칸</b> (阿岱汗, <a href="/wiki/1390%EB%85%84" title="1390년">1390년</a> ~ <a href="/wiki/1438%EB%85%84" title="1438년">1438년</a>)은 <a href="/wiki/%EB%B6%81%EC%9B%90" title="북원">북원</a>의 대칸(재위:<a href="/wiki/1425%EB%85%84" title="1425년">1425년</a> ~ <a href="/wiki/1428%EB%85%84" title="1428년">1428년</a>)이었다.</p>
<p>그는 <a class="mw-redirect" href="/wiki/%EC%9A%B0%EA%B5%AC%EB%8D%B0%EC%9D%B4" title="우구데이">우구데이</a> 가문의 <a href="/wiki/%EC%9A%B8%EB%A3%A8%EA%B7%B8_%ED%85%8C%EB%AC%B4%EB%A5%B4_%EC%B9%B8" title="울루그 테무르 칸">울루그 테무르 칸</a>의 아들이었다. 오랫동안 전쟁을 치러 <a href="/wiki/%EB%AA%BD%EA%B3%A8" title="몽골">몽골</a>을 사실상 통일하였고 <a href="/wiki/1425%EB%85%84" title="1425년">1425년</a> 에세크 칸이 죽자 <a href="/wiki/%EC%98%A4%EC%84%B8%ED%8A%B8%EC%9D%B8" title="오세트인">아스트</a> 부의 족장 아루쿠타이(阿魯台)에 의해 대칸으로 옹립되었다.</p>
<p>하지만 이에 위협을 느낀 <a href="/wiki/%EB%AA%85%EB%82%98%EB%9D%BC" title="명나라">명나라</a>가 <a href="/wiki/%EC%98%A4%EC%9D%B4%EB%9D%BC%ED%8A%B8" title="오이라트">오이라트</a>를 지원하였고 <a href="/wiki/1422%EB%85%84" title="1422년">1422년</a> ~ <a href="/wiki/1423%EB%85%84" title="1423년">1423년</a>의 두 차례의 전쟁에서 영토를 모두 잃었고 <a href="/wiki/1428%EB%85%84" title="1428년">1428년</a> 토곤과 손을 잡은 쿠빌라이 가문의 <a href="/wiki/%ED%83%80%EC%9D%B4%EC%86%90_%EC%B9%B8" title="타이손 칸">토크토아부카</a>에게 살해당했다.</p>
<p>이후 오이라트는 <a href="/wiki/1431%EB%85%84" title="1431년">1431년</a> 케룰렌 유역을 병합하여 모든 몽골 초원을 손에 넣게 되었고 허수아비 대칸인 <a href="/wiki/%ED%83%80%EC%9D%B4%EC%86%90_%EC%B9%B8" title="타이손 칸">타이손 칸</a>을 북원의 대칸으로 내세웠다.</p>
