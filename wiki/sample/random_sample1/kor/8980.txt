<p><b>아제르 상귀니스 전투</b>는 <b>피의 벌판 전투</b>, <b>사르마다 전투</b>, <b>발랏 전투</b>로도 알려져 있으며 <a class="new" href="/w/index.php?title=%EC%82%B4%EB%A0%88%EB%A5%B4%EB%85%B8%EC%9D%98_%EB%A1%9C%EC%A0%9C%EB%A5%B4&amp;action=edit&amp;redlink=1" title="살레르노의 로제르 (없는 문서)">살레르노의 로제르</a>가 지휘하는 <a href="/wiki/%EC%95%88%ED%8B%B0%EC%98%A4%ED%82%A4%EC%95%84_%EA%B3%B5%EA%B5%AD" title="안티오키아 공국">안티오키아 공국</a>의 십자군 부대가 <a href="/wiki/%EC%95%8C%EB%A0%88%ED%8F%AC" title="알레포">알레포</a>의 <a class="new" href="/w/index.php?title=%EC%95%84%EB%A5%B4%ED%88%AC%ED%81%AC_%EC%99%95%EC%A1%B0&amp;action=edit&amp;redlink=1" title="아르투크 왕조 (없는 문서)">오르토퀴드 왕가</a>의 지배자 <a class="new" href="/w/index.php?title=%EC%9D%BC%EA%B0%80%EC%A7%80&amp;action=edit&amp;redlink=1" title="일가지 (없는 문서)">마르딘의 일가지</a>가 지휘하는 부대에게 <a href="/wiki/1119%EB%85%84" title="1119년">1119년</a> <a href="/wiki/7%EC%9B%94_28%EC%9D%BC" title="7월 28일">7월 28일</a> 궤멸한 사건이다.</p>

<h2><span class="mw-headline" id=".EB.B0.B0.EA.B2.BD">배경</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%95%84%EC%A0%9C%EB%A5%B4_%EC%83%81%EA%B7%80%EB%8B%88%EC%8A%A4_%EC%A0%84%ED%88%AC&amp;action=edit&amp;section=1" title="부분 편집: 배경">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p><a href="/wiki/%EC%95%88%ED%8B%B0%EC%98%A4%ED%82%A4%EC%95%84_%EA%B3%B5%EA%B5%AD" title="안티오키아 공국">안티오키아 공국</a>과 다른 <a href="/wiki/%EC%8B%AD%EC%9E%90%EA%B5%B0_%EA%B5%AD%EA%B0%80" title="십자군 국가">십자군 국가</a>들은 알레포와 <a href="/wiki/%EB%AA%A8%EC%88%A0" title="모술">모술</a>에 거점을 둔 북 <a href="/wiki/%EC%8B%9C%EB%A6%AC%EC%95%84" title="시리아">시리아</a>와 <a class="mw-redirect mw-disambig" href="/wiki/%EC%9E%90%EC%A7%80%EB%9D%BC" title="자지라">자지라</a>를 지배하는 이슬람 국가들과 전쟁을 계속하고 있었다. <a href="/wiki/1113%EB%85%84" title="1113년">1113년</a> <a class="new" href="/w/index.php?title=%EC%95%8C%EB%A0%88%ED%8F%AC%EC%9D%98_%EB%A6%AC%EB%93%9C%EC%99%84&amp;action=edit&amp;redlink=1" title="알레포의 리드완 (없는 문서)">알레포의 리드완</a>이 사망하고 몇 년간의 평화가 계속되었다. 그러나 <a href="/wiki/%EB%B3%B4%EC%97%90%EB%AA%BD_2%EC%84%B8" title="보에몽 2세">보에몽 2세</a>의 섭정으로써 안티오키아 공국을 다스리던 <a class="new" href="/w/index.php?title=%EC%82%B4%EB%A0%88%EB%A5%B4%EB%85%B8%EC%9D%98_%EB%A1%9C%EC%A0%9C%EB%A5%B4&amp;action=edit&amp;redlink=1" title="살레르노의 로제르 (없는 문서)">살레르노의 로제르</a>는 리드완의 죽음으로 유발된 유리한 상황을 이용하려 하지 않았다. <a href="/wiki/%EC%97%90%EB%8D%B0%EC%82%AC_%EB%B0%B1%EA%B5%AD" title="에데사 백국">에데사 백작</a> <a href="/wiki/%EB%B3%B4%EB%91%90%EC%95%B5_2%EC%84%B8_(%EC%98%88%EB%A3%A8%EC%82%B4%EB%A0%98)" title="보두앵 2세 (예루살렘)">보두앵 2세</a>와 <a href="/wiki/%ED%8A%B8%EB%A6%AC%ED%8F%B4%EB%A6%AC_%EB%B0%B1%EA%B5%AD" title="트리폴리 백국">트리폴리 백작</a> <a class="new" href="/w/index.php?title=%ED%8A%B8%EB%A6%AC%ED%8F%B4%EB%A6%AC%EC%9D%98_%ED%8F%B0%EC%8A%A4&amp;action=edit&amp;redlink=1" title="트리폴리의 폰스 (없는 문서)">폰스</a> 역시 마찬가지로 자기 자신들의 당면한 이익에만 집중하여 알레포의 무슬림 세력에 대항하여 로제르와 동맹을 맺으려 하지 않았다. 1115년 로제르는 <a class="new" href="/w/index.php?title=%EC%82%AC%EB%A5%B4%EB%AF%BC_%EC%A0%84%ED%88%AC&amp;action=edit&amp;redlink=1" title="사르민 전투 (없는 문서)">사르민 전투</a>에서 부르스크 빈 부르스크가 지휘하는 <a href="/wiki/%EC%85%80%EC%A3%BC%ED%81%AC" title="셀주크">셀주크</a> 군의 공격을 격퇴하였다.</p>
<p><a href="/wiki/1117%EB%85%84" title="1117년">1117년</a> 알레포는 오르토퀴드 가문의 <a href="/wiki/%EC%95%84%ED%83%80%EB%B2%A0%EA%B7%B8" title="아타베그">아타베그</a> <a class="new" href="/w/index.php?title=%EC%9D%BC%EA%B0%80%EC%A7%80&amp;action=edit&amp;redlink=1" title="일가지 (없는 문서)">일가지</a>의 지배하에 들게 되었다. <a href="/wiki/1118%EB%85%84" title="1118년">1118년</a> 로제르는 <a class="new" href="/w/index.php?title=%EC%95%84%EC%9E%90%EC%A6%88&amp;action=edit&amp;redlink=1" title="아자즈 (없는 문서)">아자즈</a>를 함락함에 따라 알레포는 십자군의 공격에 노출되었다. 이에대하여 일가지는 1119년 안티오키아 공국을 공격하여 대응했다. 로제르는 <a href="/wiki/%EC%95%88%ED%8B%B0%EC%98%A4%ED%82%A4%EC%95%84_%EC%B4%9D%EB%8C%80%EC%A3%BC%EA%B5%90" title="안티오키아 총대주교">안티오키아의 라틴 총대주교</a> <a class="new" href="/w/index.php?title=%EB%B0%9C%EB%A0%8C%EC%8A%A4%EC%9D%98_%EB%B2%A0%EB%A5%B4%EB%82%98%EB%A5%B4%EB%8F%84&amp;action=edit&amp;redlink=1" title="발렌스의 베르나르도 (없는 문서)">발렌스의 베르나르도</a>와 함께 아르타에서 출진하였다. 베르나르도 총대주교는 아르타가 방어가 충실하고 안티오키아에서 가까우며, 로제르의 군대가 아르타에 있는 한 일가지의 군대는 감히 안티오키아를 향하여 진군하지 못할 것이라고 로제르에게 조언하였다. 베르나르도 총대주교는 또 로제르에게 <a href="/wiki/%EC%98%88%EB%A3%A8%EC%82%B4%EB%A0%98_%EC%99%95%EA%B5%AD" title="예루살렘 왕국">예루살렘 왕국</a>의 왕 보두앵과 폰스에게 지원을 요청하라고 조언하였다. 그러나 로제르는 지원군의 도착을 기다리지 않았다. 로제르는 일가지가 알아타리브 항구를 공략하는 동안 사르마다의 도로에 진영을 쳤다. 뷰에 퐁의 로베르 휘하의 소규모 군세가 공성을 풀기위해 출진했고, 일가지는 투르크군의 전형적인 작전을 다시한번 시도하여 거짓퇴각을 개시했다. 로베르의 군세는 항구에서 투르크군이 있는 곳으로 유인당했고, 매복공격을 받았다.</p>
<h2><span class="mw-headline" id=".EC.A0.84.ED.88.AC">전투</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%95%84%EC%A0%9C%EB%A5%B4_%EC%83%81%EA%B7%80%EB%8B%88%EC%8A%A4_%EC%A0%84%ED%88%AC&amp;action=edit&amp;section=2" title="부분 편집: 전투">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>일가지는 <a href="/wiki/%EB%8B%A4%EB%A7%88%EC%8A%A4%EC%BF%A0%EC%8A%A4" title="다마스쿠스">다마스쿠스</a>를 지배하는 <a class="new" href="/w/index.php?title=%EB%B6%80%EB%A6%AC%EB%93%9C_%EC%99%95%EC%A1%B0&amp;action=edit&amp;redlink=1" title="부리드 왕조 (없는 문서)">부리드</a> 가문의 에미르 톡테인의 원군을 기다리고 있었으나, 일가지 역시 원군을 기다리는 것에 넌더리를 내고 있었다. 별로 사용되지 않는 도로를 이용하여 일가지의 군대는 <a href="/wiki/6%EC%9B%94_27%EC%9D%BC" title="6월 27일">6월 27일</a> 밤동안 재빠르게 로제르의 진영을 포위하였다. 로제르는 부주의하게도 퇴로가 별로 없는 급경사의 나무로 가득찬 계곡에 진영을 설치했다. 로제르의 군대는 700명의 기사와 기마궁수인 <a class="new" href="/w/index.php?title=%ED%88%AC%EB%A5%B4%EC%BD%94%ED%8F%B4%EB%A0%88%EC%8A%A4&amp;action=edit&amp;redlink=1" title="투르코폴레스 (없는 문서)">투르코폴레스</a>를 포함한 3,000명의 일반병으로 구성되어 있었으며 기습을 당하자 급하게 다섯 개의 부대로 편성하였다. 십자군은 끝부분이 이슬람군과 가장 멀리 떨어진 V자 형태로 진을 쳤다. 좌익에서 우익에 이르는 각 부대는 생 로의 로베르, 로제르 공, 프레넬의 기, 수도사 고프레이와 피에르가 지휘를 맡았다. 그러는 동안 로제르는 레나드 멘소에르 휘하의 여섯 번째 부대에게 후방을 지킬 것을 명했다.<sup class="reference" id="cite_ref-1"><a href="#cite_note-1">[1]</a></sup> 이슬람 군대가 대기하고 있을 때 카디(이슬람 재판관) <a class="new" href="/w/index.php?title=%EC%95%84%EB%B6%80_%EC%95%8C%ED%8C%8C%EB%93%A4_%EC%9D%B4%EB%B8%90_%EC%95%8C%EC%B9%B4%EC%8B%9C%EC%83%A4%EB%B8%8C&amp;action=edit&amp;redlink=1" title="아부 알파들 이븐 알카시샤브 (없는 문서)">아부 알파들 이븐 알카시샤브</a>는 머리에는 법률가의 터번을 썼으나 손으로는 창을 휘두르며 군대의 앞으로 말을 달려나왔다. 동시대 알레포의 역사학자 카말 아드 딘에 따르면, 이슬람군은 처음에는 조심스럽게 카쉬샤브의 연설을 들었으나, <a href="/wiki/%EC%A7%80%ED%95%98%EB%93%9C" title="지하드">지하드</a>에 참여하는 전사들의 의무와 이익을 환기하는 카쉬샤브의 열정적인 연설이 끝나자, 고무된 이슬람 병사들은 격정적으로 전투에 참여했다고 한다. <a href="/wiki/7%EC%9B%94_28%EC%9D%BC" title="7월 28일">7월 28일</a> 아침 기사들의 앞에 서 있던 안티오키아 보병대와 투르크 궁병대 사이의 궁시의 교환으로 시작되었다. 처음 십자군은 우익의 피에르와 수도사 고프레이가 지휘하는 부대가 공격을 개시하여 그들의 상대를 격파했을 때 전투를 유리하게 진행해 나갔다. 프레넬의 기가 지휘하는 중앙부대도 역시 전투를 유리하게 진행해 나갔다. 그러나 전투는 좌익에서의 싸움에서 순식간에 결판이 나버렸다.<sup class="reference" id="cite_ref-2"><a href="#cite_note-2">[2]</a></sup> 생 로의 로베르와 <a class="new" href="/w/index.php?title=%ED%88%AC%EB%A5%B4%EC%BD%94%ED%8F%B4%EB%A0%88%EC%8A%A4&amp;action=edit&amp;redlink=1" title="투르코폴레스 (없는 문서)">투르코폴레스</a>들은 적에게 격파당해 로제르의 부대쪽으로 도망쳤고, 로제르 부대를 혼란시켰다. 또 안티오크의 기사와 보병대들의 얼굴에 먼지를 실은 북풍이 불어와 시야를 가려 이들을 더욱 더 혼란시켰다. 곧 투르크군의 양익은 십자군을 포위하였다. 전투 중에 로제르는 자신의 군기로 쓰이던 거대하고 보석이 아로새겨진 십자가 깃발 아래에서 얼굴에 칼을 맞고 전사하였다. 나머지 십자군 병사들도 살해당하거나 사로잡혔다. 오직 두 명의 기사들만이 살아남았다. 레나드 맨소에르는 보두앵 왕을 기다리기 위해 사르마다 항에 피난하였으나, 후에 일가지에 의해 포로로 잡혔다. 다른 포로들 중에는 나중에 이 전투에 대하여 기록하게 되는 <a class="new" href="/w/index.php?title=%EB%8C%80%EB%B2%95%EA%B4%80_%EC%9B%94%ED%84%B0&amp;action=edit&amp;redlink=1" title="대법관 월터 (없는 문서)">대법관 월터</a>와 같은 이들이 있었다. 학살에 가까운 피해를 입은 십자군의 패배는 이 전투로 하여금 <a href="/wiki/%EB%9D%BC%ED%8B%B4%EC%96%B4" title="라틴어">라틴어</a>로 "피의 벌판"을 뜻하는 <i>아제르 상귀니스</i>라는 이름을 얻게 하였다.</p>
<h2><span class="mw-headline" id=".EC.98.81.ED.96.A5">영향</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%95%84%EC%A0%9C%EB%A5%B4_%EC%83%81%EA%B7%80%EB%8B%88%EC%8A%A4_%EC%A0%84%ED%88%AC&amp;action=edit&amp;section=3" title="부분 편집: 영향">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>이 전투는 무슬림이 <a class="mw-redirect" href="/wiki/%EC%85%80%EC%A3%BC%ED%81%AC_%ED%88%AC%EB%A5%B4%ED%81%AC" title="셀주크 투르크">셀주크 투르크</a>의 도움 없이 십자군을 격퇴할 수 있다는 것을 증명하였다. 그러나 일가지는 얼마 안 있어 주연에 빠져 안티오키아로 진군하지 않았고,<sup class="reference" id="cite_ref-autogenerated1_3-0"><a href="#cite_note-autogenerated1-3">[3]</a></sup> 이에 베르나르도 총대주교는 이 기회를 이용하여 그가 할 수 있는 모든 것을 다하여 방어수단을 구축하였다. 그러나 상황이 이러했음에도 불구하고 안티오키아 공국은 많은 병사들을 잃었고 아타리브, 제르다나, 사르민, <a class="new" href="/w/index.php?title=%EB%A7%88%EC%95%84%EB%9D%BC%ED%8A%B8_%EC%95%8C%EB%88%84%EB%A7%8C&amp;action=edit&amp;redlink=1" title="마아라트 알누만 (없는 문서)">마아라트 알누만</a>과 카프르 타브와 같은 영토들은 급속도로 무슬림에게 함락 당했다.<sup class="reference" id="cite_ref-autogenerated1_3-1"><a href="#cite_note-autogenerated1-3">[3]</a></sup> 일가지는 <a href="/wiki/8%EC%9B%94_14%EC%9D%BC" title="8월 14일">8월 14일</a> <a class="mw-redirect" href="/wiki/%EC%98%88%EB%A3%A8%EC%82%B4%EB%A0%98%EC%9D%98_%EB%B3%B4%EB%91%90%EC%95%B5_2%EC%84%B8" title="예루살렘의 보두앵 2세">예루살렘의 보두앵 2세</a>와 폰스 백작에게 <a class="new" href="/w/index.php?title=%ED%95%98%EB%B8%8C_%EC%A0%84%ED%88%AC&amp;action=edit&amp;redlink=1" title="하브 전투 (없는 문서)">하브 전투</a>에서 패퇴하였고, 보두앵은 안티오키아의 섭정직을 차지하였다. 보두앵은 무슬림 손에 떨어진 마을들을 일부 탈환하였다. 그럼에도 불구하고 아제르 상귀니스 전투는 안티오키아 공국을 약화시켰고, 다음에 찾아올 10년 동안 계속된 무슬림의 공격을 불러들였다. 그리고 이러한 결과로 인해 안티오키아 공격은 결과적으로 <a href="/wiki/%EB%B9%84%EC%9E%94%ED%8B%B0%EC%9B%80_%EC%A0%9C%EA%B5%AD" title="비잔티움 제국">비잔티움 제국</a>의 영향력 하에 편입된다. 이후 십자군은 <a href="/wiki/1125%EB%85%84" title="1125년">1125년</a>에야 <a class="new" href="/w/index.php?title=%EC%95%84%EC%9E%90%EC%A6%88_%EC%A0%84%ED%88%AC&amp;action=edit&amp;redlink=1" title="아자즈 전투 (없는 문서)">아자즈 전투</a>에서 승리함에 따라 시리아에 대한 영향력을 되찾는다. <i>아제르 상귀니스</i>라는 라틴어는 <a
