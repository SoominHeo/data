<h2><span class="mw-headline" id=".EC.97.AD.EC.82.AC">역사</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%95%84%EC%A0%9C%EB%A5%B4%EB%B0%94%EC%9D%B4%EC%9E%94_%ED%95%AD%EA%B3%B5&amp;action=edit&amp;section=1" title="부분 편집: 역사">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p><a href="/wiki/1991%EB%85%84" title="1991년">1991년</a> <a href="/wiki/%EC%86%8C%EB%A0%A8" title="소련">소련</a>의 붕괴와 동시에 다음 해인 <a href="/wiki/1992%EB%85%84" title="1992년">1992년</a> <a href="/wiki/4%EC%9B%94" title="4월">4월</a>에 설립했다. <a href="/wiki/1994%EB%85%84" title="1994년">1994년</a>에 최초의 국제선인 <a href="/wiki/%EB%91%90%EB%B0%94%EC%9D%B4" title="두바이">두바이</a>와 <a href="/wiki/%EC%9D%B4%EC%8A%A4%ED%83%84%EB%B6%88" title="이스탄불">이스탄불</a>에 취항 했으며 <a href="/wiki/2000%EB%85%84" title="2000년">2000년</a>에 최초로 <a href="/wiki/%EB%B3%B4%EC%9E%89_757" title="보잉 757">보잉 757</a> 기종을 도입했다. <a href="/wiki/2005%EB%85%84" title="2005년">2005년</a>에 <a href="/wiki/%EB%B0%94%EC%BF%A0" title="바쿠">바쿠</a>에 위치한 화물터미널이 완료되었고 <a href="/wiki/2010%EB%85%84" title="2010년">2010년</a>에 <a href="/wiki/%EB%B3%B4%EC%9E%89_737" title="보잉 737">보잉 737</a>과 <a href="/wiki/%EB%B3%B4%EC%9E%89_767" title="보잉 767">보잉 767</a> 기종을 주문했다.<sup class="reference" id="cite_ref-2"><a href="#cite_note-2">[2]</a></sup> 같은 해 <a href="/wiki/12%EC%9B%94" title="12월">12월</a>에 <a href="/wiki/%ED%88%AC%ED%8F%B4%EB%A0%88%ED%94%84_Tu-134" title="투폴레프 Tu-134">투폴레프 Tu-134</a>, <a href="/wiki/%ED%88%AC%ED%8F%B4%EB%A0%88%ED%94%84_Tu-154" title="투폴레프 Tu-154">투폴레프 Tu-154</a> 기종을 퇴역에 이어 <a href="/wiki/%EB%B3%B4%EC%9E%89_737" title="보잉 737">보잉 737</a> 기종이 주문을 취소했다.<sup class="reference" id="cite_ref-3"><a href="#cite_note-3">[3]</a></sup> 한편 <a href="/wiki/2014%EB%85%84" title="2014년">2014년</a>에 <a href="/wiki/%EB%AF%B8%EA%B5%AD" title="미국">미국</a>에 취항하기 위해 <a href="/wiki/%EB%B3%B4%EC%9E%89_787_%EB%93%9C%EB%A6%BC%EB%9D%BC%EC%9D%B4%EB%84%88" title="보잉 787 드림라이너">보잉 787 드림라이너</a>를 주문 했으며 이 기종이 도입되면 <a href="/wiki/%EB%AF%B8%EA%B5%AD" title="미국">미국</a> 국제선 노선에 취항될 예정이다.</p>
<h2><span class="mw-headline" id=".EC.9A.B4.ED.95.AD_.EB.85.B8.EC.84.A0">운항 노선</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%95%84%EC%A0%9C%EB%A5%B4%EB%B0%94%EC%9D%B4%EC%9E%94_%ED%95%AD%EA%B3%B5&amp;action=edit&amp;section=2" title="부분 편집: 운항 노선">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
