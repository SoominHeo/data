<h1 class="firstHeading" id="firstHeading" lang="ko">아즈마야 대수</h1>
<div class="mw-body-content" id="bodyContent">
<div id="siteSub">위키백과, 우리 모두의 백과사전.</div>
<div id="contentSub"></div>
<div class="mw-jump" id="jump-to-nav">
					이동:					<a href="#mw-head">둘러보기</a>, 					<a href="#p-search">검색</a>
</div>
<div class="mw-content-ltr" dir="ltr" id="mw-content-text" lang="ko"><script>function mfTempOpenSection(id){var block=document.getElementById("mf-section-"+id);block.className+=" open-block";block.previousSibling.className+=" open-block";}</script><p><a href="/wiki/%ED%99%98%EB%A1%A0" title="환론">환론</a>과 <a href="/wiki/%EB%8C%80%EC%88%98%EC%A0%81_%EC%88%98%EB%A1%A0" title="대수적 수론">대수적 수론</a>과 <a href="/wiki/%EB%8C%80%EC%88%98%EA%B8%B0%ED%95%98%ED%95%99" title="대수기하학">대수기하학</a>에서, <b>아즈마야 대수</b>([東屋]代數, <span style="font-size: smaller;"><a href="/wiki/%EC%98%81%EC%96%B4" title="영어">영어</a>:</span> <span lang="en" xml:lang="en">Azumaya algebra</span>)는 <a href="/wiki/%EA%B0%80%ED%99%98%ED%99%98" title="가환환">가환환</a> 또는 <a href="/wiki/%EC%8A%A4%ED%82%B4_(%EC%88%98%ED%95%99)" title="스킴 (수학)">스킴</a> 위의 <a class="mw-redirect" href="/wiki/%EB%8B%A8%EC%9C%84_%EA%B2%B0%ED%95%A9_%EB%8C%80%EC%88%98" title="단위 결합 대수">단위 결합 대수</a> 가운데, <a href="/wiki/%EC%9E%90%EB%A6%AC%EC%8A%A4%ED%82%A4_%EC%9C%84%EC%83%81" title="자리스키 위상">자리스키 위상</a>에서 각 <a href="/wiki/%EC%A4%84%EA%B8%B0_(%EC%88%98%ED%95%99)" title="줄기 (수학)">줄기</a>가 유한 차원 <a href="/wiki/%EC%9E%90%EC%9C%A0_%EA%B0%80%EA%B5%B0" title="자유 가군">자유 가군</a>이며, 줄기의 포락 대수가 행렬환과 동형인 것이다. <a href="/wiki/%EB%8C%80%EC%88%98%EA%B8%B0%ED%95%98%ED%95%99" title="대수기하학">대수기하학</a>적으로, 아즈마야 대수는 올이 <a href="/wiki/%EC%82%AC%EC%98%81_%EA%B3%B5%EA%B0%84" title="사영 공간">사영 공간</a>인 <a href="/wiki/%EC%98%AC%EB%8B%A4%EB%B0%9C" title="올다발">올다발</a>에 해당한다.</p>
<p>아즈마야 대수들의 동치류는 <b>브라우어 군</b>(Brauer群, <span style="font-size: smaller;"><a href="/wiki/%EC%98%81%EC%96%B4" title="영어">영어</a>:</span> <span lang="en" xml:lang="en">Brauer group</span>)이라는 <a href="/wiki/%EA%B5%B0_(%EC%88%98%ED%95%99)" title="군 (수학)">군</a>을 정의한다. 이는 <a href="/wiki/%EB%B2%A1%ED%84%B0_%EB%8B%A4%EB%B0%9C" title="벡터 다발">벡터 다발</a>의 동치류들이 <a href="/wiki/K%EC%9D%B4%EB%A1%A0" title="K이론">K군</a>을 정의하는 것과 마찬가지다.</p>

</math></span><img alt="{\displaystyle \operatorname {Br} (\mathbb {R} )\cong \mathbb {Z} /2}" aria-hidden="true" class="mwe-math-fallback-image-inline" src="https://wikimedia.org/api/rest_v1/media/math/render/svg/bf2bcca0c0dd61d515ac6142aa1c1307ef70720a" style="vertical-align: -0.838ex; width:13.112ex; height:2.843ex;"/></span>을 계산하였다 (프로베니우스 정리).<sup class="reference" id="cite_ref-3"><a href="#cite_note-3">[3]</a></sup> <a href="/wiki/%EC%A1%B0%EC%A7%80%ED%94%84_%EC%9B%A8%EB%8D%94%EB%B2%88" title="조지프 웨더번">조지프 웨더번</a>은 1905년에 (현대적 용어로) <a href="/wiki/%EC%9C%A0%ED%95%9C%EC%B2%B4" title="유한체">유한체</a>의 브라우어 군이 자명하다는 것을 증명하였다 (웨더번 소정리).<sup class="reference" id="cite_ref-4"><a href="#cite_note-4">[4]</a></sup> 그러나 웨더번의 첫 증명은 약간의 결함이 있었으며, <a href="/wiki/%EB%A0%88%EB%84%88%EB%93%9C_%EC%9C%A0%EC%A7%84_%EB%94%95%EC%8A%A8" title="레너드 유진 딕슨">레너드 유진 딕슨</a>이 최초로 올바른 증명을 발표하였다.<sup class="reference" id="cite_ref-5"><a href="#cite_note-5">[5]</a></sup></p>
<p>브라우어 군은 <a href="/wiki/%EB%A6%AC%ED%95%98%EB%A5%B4%ED%8A%B8_%EB%B8%8C%EB%9D%BC%EC%9A%B0%EC%96%B4" title="리하르트 브라우어">리하르트 브라우어</a>가 1932년에 정의하였다.<sup class="reference" id="cite_ref-6"><a href="#cite_note-6">[6]</a></sup></p>
<p><a href="/wiki/%EC%95%84%EC%A6%88%EB%A7%88%EC%95%BC_%EA%B3%A0%EB%A1%9C" title="아즈마야 고로">아즈마야 고로</a>가 1951년에 "고유 극대 중심 대수"(<span style="font-size: smaller;"><a href="/wiki/%EC%98%81%EC%96%B4" title="영어">영어</a>:</span> <span lang="en" xml:lang="en">proper maximally central algebra</span>)라는 이름으로 도입하였다.<sup class="reference" id="cite_ref-7"><a href="#cite_note-7">[7]</a></sup><span class="reference plainlinksneverexpand" style="white-space:nowrap"><sup>:128</sup></span><sup class="reference" id="cite_ref-Millar_2-2"><a href="#cite_note-Millar-2">[2]</a></sup><span class="reference plainlinksneverexpand" style="white-space:nowrap"><sup>:§1.6</sup></span> (엄밀히 말해, 아즈마야는 이 용어를 정의할 때 <a href="/wiki/%EC%9E%90%EC%9C%A0_%EA%B0%80%EA%B5%B0" title="자유 가군">자유 가군</a>이어야 한다는 조건을 추가하였다.) 이후 1964~1965년 <a href="/wiki/%EB%8B%88%EC%BD%9C%EB%9D%BC_%EB%B6%80%EB%A5%B4%EB%B0%94%ED%82%A4" title="니콜라 부르바키">니콜라 부르바키</a> 세미나에서 <a href="/wiki/%EC%95%8C%EB%A0%89%EC%82%B0%EB%8D%94_%EA%B7%B8%EB%A1%9C%ED%85%90%EB%94%94%ED%81%AC" title="알렉산더 그로텐디크">알렉산더 그로텐디크</a>가 그 정의를 일반화하여 스킴 위의 아즈마야 대수를 정의하였고, "아즈마야 대수"라는 용어를 도입하였다.<sup class="reference" id="cite_ref-8"><a href="#cite_note-8">[8]</a></sup><sup class="reference" id="cite_ref-9"><a href="#cite_note-9">[9]</a></sup></p>
