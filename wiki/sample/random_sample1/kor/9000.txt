<p><b>아카시 성</b>(<span style="font-size: smaller;"><a href="/wiki/%EC%9D%BC%EB%B3%B8%EC%96%B4" title="일본어">일본어</a>:</span> <span lang="ja"><span lang="ja"><ruby><rb>明石城</rb> <rp>(</rp><rt>あかしじょう</rt><rp>)</rp></ruby></span></span>)은 <a href="/wiki/%ED%9A%A8%EA%B3%A0_%ED%98%84" title="효고 현">효고 현</a> <a href="/wiki/%EC%95%84%EC%B9%B4%EC%8B%9C_%EC%8B%9C" title="아카시 시">아카시 시</a>에 있는 연곽제곽혼합식 평산성이다. <a href="/wiki/%EC%97%90%EB%8F%84_%EC%8B%9C%EB%8C%80" title="에도 시대">에도 시대</a> <a href="/wiki/%EC%95%84%EC%B9%B4%EC%8B%9C_%EB%B2%88" title="아카시 번">아카시 번</a>의 번청으로 사용되었다. 기하루 성/기슌 성(喜春城), 긴코 성(錦江城)이라는 이명도 있다. 축성에 있어서 무예가 <a href="/wiki/%EB%AF%B8%EC%95%BC%EB%AA%A8%ED%86%A0_%EB%AC%B4%EC%82%AC%EC%8B%9C" title="미야모토 무사시">미야모토 무사시</a>가 관련되어 있는 것으로 유명하다.</p>

<h2><span class="mw-headline" id=".EA.B0.9C.EC.9A.94">개요</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%95%84%EC%B9%B4%EC%8B%9C_%EC%84%B1&amp;action=edit&amp;section=1" title="부분 편집: 개요">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p><a href="/wiki/JR" title="JR">JR</a> <a href="/wiki/%EC%95%84%EC%B9%B4%EC%8B%9C_%EC%97%AD" title="아카시 역">아카시 역</a> 북쪽에 위치해 있어 역 앞에서 성을 조망할 수 있다. 성의 배치는 연곽식과 제곽식이 혼합된 양식을 취했으며 아카마쓰 산을 중심으로 성을 조성한 평산성이다. 성의 중앙에 <a href="/wiki/%ED%98%BC%EB%A7%88%EB%A3%A8" title="혼마루">혼마루</a>를 두었고, 혼마루 동쪽에는 <a class="mw-redirect" href="/wiki/%EB%8B%88%EB%85%B8%EB%A7%88%EB%A3%A8" title="니노마루">니노마루</a>, 그 동쪽에는 히가시노마루를 두었다. 또, 남쪽에는 <a class="mw-redirect" href="/wiki/%EC%82%B0%EB%85%B8%EB%A7%88%EB%A3%A8" title="산노마루">산노마루</a>와 서쪽에는 이나리마루를 배치했다. 혼마루에 천수대를 조성해 두었지만, <a class="mw-disambig" href="/wiki/%EC%B2%9C%EC%88%98" title="천수">천수</a>는 세워지지 않고, 네 모퉁이에 <a class="mw-redirect" href="/wiki/%EC%95%BC%EA%B5%AC%EB%9D%BC" title="야구라">망루</a>를 배치했다. 그 중 남동쪽과 남서쪽에 다쓰미 망루, 히쓰지사루 망루만 남아 국가 중요문화재로 지정되어 있다. 성터는 <a class="new" href="/w/index.php?title=%EC%95%84%EC%B9%B4%EC%8B%9C_%EA%B3%B5%EC%9B%90&amp;action=edit&amp;redlink=1" title="아카시 공원 (없는 문서)">아카시 공원</a>으로 정비되어 있으며, <a href="/wiki/%EB%B2%9A%EB%82%98%EB%AC%B4" title="벚나무">벚꽃</a>의 명소로 유명해 <a class="new" href="/w/index.php?title=%EC%9D%BC%EB%B3%B8_%EB%B2%9A%EA%BD%83_%EB%AA%85%EC%86%8C_100%EC%84%A0&amp;action=edit&amp;redlink=1" title="일본 벚꽃 명소 100선 (없는 문서)">일본 벚꽃 명소 100선</a>에 올라있다. 성의 서쪽에 흐르는 <a class="new" href="/w/index.php?title=%EC%95%84%EC%B9%B4%EC%8B%9C_%EA%B0%95&amp;action=edit&amp;redlink=1" title="아카시 강 (없는 문서)">아카시 강</a>을 천연의 <a href="/wiki/%ED%95%B4%EC%9E%90" title="해자">해자</a>로 삼고, 남쪽은 <a href="/wiki/%EC%9A%B4%ED%95%98" title="운하">운하</a>를 파 해자 겸 <a href="/wiki/%ED%95%AD%EA%B5%AC" title="항구">항구</a>로 삼았다.</p>

<p><a href="/wiki/1617%EB%85%84" title="1617년">1617년</a> <a class="mw-redirect" href="/wiki/%ED%95%98%EB%A6%AC%EB%A7%88%EB%85%B8%EC%BF%A0%EB%8B%88" title="하리마노쿠니">하리마 국</a> <a class="mw-redirect" href="/wiki/%EC%9D%B4%EB%82%98%EB%B0%94%EB%85%B8%EC%BF%A0%EB%8B%88" title="이나바노쿠니">이나바 국</a> <a href="/wiki/%EB%8F%97%ED%86%A0%EB%A6%AC_%EB%B2%88" title="돗토리 번">돗토리 번</a>으로 이봉되자, href="/w/index.php?title=%EB%A7%88%EC%93%B0%EB%AA%A8%ED%86%A0_%EB%B2%88&amp;action=edit&amp;redlink=1" title="마쓰모토 번 (없는 문서)">마쓰모토 번</a>의 <a href="/wiki/%EC%98%A4%EA%B0%80%EC%82%AC%EC%99%80%EB%9D%BC_%EB%8B%A4%EB%8B%A4%EC%9E%90%EB%84%A4" title="오가사와라 다다자네">오가사와라 다다자네</a>가 10만 석에 입봉되어 <a class="new" class="new" href="/w/index.php?title=%EC%9D%BC%EA%B5%AD%EC%9D%BC%EC%84%B1%EB%A0%B9&amp;action=edit&amp;redlink=1" title="일국일성령 (없는 문서)">일국일성령</a>에 따라 후나게 성은 이미 파각되어 <a href="/wiki/%EC%A7%84%EC%95%BC" title="진야">진야</a> 규모에 불과했다. 같은 해에 2대 <a class="mw-redirect" href="/wiki/%EC%87%BC%EA%B5%B0" title="쇼군">쇼군</a> <a href="/wiki/%EB%8F%84%EC%BF%A0%EA%B0%80%EC%99%80_%ED%9E%88%EB%8D%B0%ED%83%80%EB%8B%A4" title="도쿠가와 히데타다">도쿠가와 히데타다</a>가 아카시 성 축성을 명해 <a href="/wiki/1619%EB%85%84" title="1619년">1619년</a> 완공되었다. 축성에 관해서는 <a href="/wiki/%ED%9B%84%EC%8B%9C%EB%AF%B8_%EC%84%B1" title="후시미 성">후시미 성</a>, <a href="/wiki/%EB%AF%B8%ED%82%A4_%EC%84%B1" title="미키 성">미키 성</a>, 후나게 성의 건조물을 부재로 사용했으며, 축성 목적은 서국<sup class="reference" id="cite_ref-1"><a href="#cite_note-1">[1]</a></sup> <a href="/wiki/%EB%8B%A4%EC%9D%B4%EB%AC%98" title="다이묘">다이묘</a>의 견제하기 위함이다. 이때 천수대는 조성되었지만, 천수는 세워지지 않았다.</p>
<p>성의 남쪽 습지 일대는 메워져 시가지로 조성되었다. 이때 도시계획에 참여한 인물이 검호로 익히 알려진 <a href="/wiki/%EB%AF%B8%EC%95%BC%EB%AA%A8%ED%86%A0_%EB%AC%B4%EC%82%AC%EC%8B%9C" title="미야모토 무사시">미야모토 무사시</a>라고 한다.</p>
<p><a href="/wiki/1632%EB%85%84" title="1632년">1632년</a> 번주 오가사와라 다다자네는 <a class="mw-redirect" href="/wiki/%EB%B6%80%EC%A0%A0%EB%85%B8%EC%BF%A0%EB%8B%88" title="부젠노쿠니">부젠 국</a> <a href="/wiki/%EA%B3%A0%EC%BF%A0%EB%9D%BC_%EB%B2%88" title="고쿠라 번">고쿠라 번</a>으로 이봉됨에 따라 <a href="/wiki/1633%EB%85%84" title="1633년">1633년</a> 도다 마쓰다이라 가문의 <a class="mw-disambig" href="/wiki/%EB%A7%88%EC%93%B0%EB%8B%A4%EC%9D%B4%EB%9D%BC_%EC%95%BC%EC%8A%A4%EB%82%98%EC%98%A4" title="마쓰다이라 야스나오">마쓰다이라 야스나오</a>가 시나노 국 마쓰모토 번에서 하리마 국 아카시 번 7만 석에 이봉되어 입성한다. 이후, 오쿠보 가문(<a href="/wiki/1639%EB%85%84" title="1639년">1639년</a> ~ <a href="/wiki/1649%EB%85%84" title="1649년">1649년</a>), 후지이 마쓰다이라 가문(1649년 ~ <a href="/wiki/1679%EB%85%84" title="1679년">1679년</a>), 혼다 가문(1679년 ~ <a href="/wiki/1682%EB%85%84" title="1682년">1682년</a>)이 성주로 부임한 <a href="/wiki/%ED%9B%84%EB%8B%A4%EC%9D%B4_%EB%8B%A4%EC%9D%B4%EB%AC%98" title="후다이 다이묘">후다이 다이묘</a> 가문의 거성이었다.</p>
<p>1682년 <a class="mw-disambig" href="/wiki/%ED%98%BC%EB%8B%A4_%EB%A7%88%EC%82%AC%ED%86%A0%EC%8B%9C" title="혼다 마사토시">혼다 마사토시</a>가 악정을 이유로 <a class="mw-redirect" href="/wiki/%EB%AC%B4%EC%93%B0%EB%85%B8%EC%BF%A0%EB%8B%88" title="무쓰노쿠니">무쓰 국</a> <a href="/wiki/%EC%98%A4%EC%BF%A0%EB%B3%B4_%EB%B2%88" title="오쿠보 번">오쿠보 번</a> 1만 석에 감봉되어 이봉되었고, 대신 <a class="mw-redirect" href="/wiki/%EC%97%90%EC%B9%98%EC%A0%A0%EB%85%B8%EC%BF%A0%EB%8B%88" title="에치젠노쿠니">에치젠 국</a> <a class="new" href="/w/index.php?title=%EC%98%A4%EB%85%B8_%EB%B2%88&amp;action=edit&amp;redlink=1" title="오노 번 (없는 문서)">오노 번</a>에서 <a href="/wiki/%EB%A7%88%EC%93%B0%EB%8B%A4%EC%9D%B4%EB%9D%BC_%EB%82%98%EC%98%A4%EC%95%84%ED%82%A4%EB%9D%BC" title="마쓰다이라 나오아키라">마쓰다이라 나오아키라</a>가 번주로 부임해 입성하였다. 이후, <a href="/wiki/%EC%8B%A0%ED%8C%90_%EB%8B%A4%EC%9D%B4%EB%AC%98" title="신판 다이묘">신판 다이묘</a> 에치젠 마쓰다이라 가문의 거성으로 <a href="/wiki/%EB%A9%94%EC%9D%B4%EC%A7%80_%EC%9C%A0%EC%8B%A0" title="메이지 유신">메이지 유신</a>을 맞이한다.</p>
<p>아카시 성은 에치젠 마쓰다이라 가문의 2번주 <a class="new" href="/w/index.php?title=%EB%A7%88%EC%93%B0%EB%8B%A4%EC%9D%B4%EB%9D%BC_%EB%82%98%EC%98%A4%EC%93%B0%EB%84%A4&amp;action=edit&amp;redlink=1" title="마쓰다이라 나오쓰네 (없는 문서)">마쓰다이라 나오쓰네</a> 때인 <a href="/wiki/1739%EB%85%84" title="1739년">1739년</a> 대대적으로 수리되었다.</p>
