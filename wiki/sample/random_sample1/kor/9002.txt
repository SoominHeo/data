<p><b>아칸 호</b>(<span style="font-size: smaller;"><a href="/wiki/%EC%9D%BC%EB%B3%B8%EC%96%B4" title="일본어">일본어</a>:</span> <span lang="ja"><span lang="ja"><ruby><rb>阿寒湖</rb> <rp>(</rp><rt>あかんこ</rt><rp>)</rp></ruby></span></span> <small>아칸코<sup>[<a href="/wiki/%EC%9C%84%ED%82%A4%EB%B0%B1%EA%B3%BC:%EC%9D%BC%EB%B3%B8%EC%96%B4%EC%9D%98_%ED%95%9C%EA%B8%80_%ED%91%9C%EA%B8%B0" title="위키백과:일본어의 한글 표기">*</a>]</sup></small>)는 <a href="/wiki/%ED%99%8B%EC%B9%B4%EC%9D%B4%EB%8F%84" title="홋카이도">홋카이도</a> 동부의 <a href="/wiki/%EA%B5%AC%EC%8B%9C%EB%A1%9C_%EC%8B%9C" title="구시로 시">구시로 시</a>에 있는 <a href="/wiki/%ED%98%B8%EC%88%98" title="호수">호수</a>이다. 호수 전체가 <a href="/wiki/%EC%95%84%EC%B9%B8_%EA%B5%AD%EB%A6%BD%EA%B3%B5%EC%9B%90" title="아칸 국립공원">아칸 국립공원</a>에 속해 홋카이도 동부의 대표적 관광지이다. 2005년 11월 <a href="/wiki/%EB%9E%8C%EC%82%AC%EB%A5%B4_%ED%98%91%EC%95%BD" title="람사르 협약">람사르 협약</a>에 등록된 <a href="/wiki/%EC%8A%B5%EC%A7%80" title="습지">습지</a> 중 한 곳이다.</p>
<h2><span class="mw-headline" id=".EC.A7.80.EB.A6.AC">지리</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%95%84%EC%B9%B8_%ED%98%B8&amp;action=edit&amp;section=1" title="부분 편집: 지리">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>홋카이도 동부의 구시로 시 북부에 위치해 있으며, 호수 동쪽에는 <a class="new" href="/w/index.php?title=%EC%98%A4%EC%95%84%EC%B9%B8_%EC%82%B0&amp;action=edit&amp;redlink=1" title="오아칸 산 (없는 문서)">오아칸 산</a>이 입지해 있다. 호수는 오아칸 산의 <a class="mw-disambig" href="/wiki/%EB%B6%84%ED%99%94" title="분화">분화</a>로 생성된 <a href="/wiki/%ED%8F%90%EC%83%89%ED%98%B8" title="폐색호">폐색호</a>이다. 호수의 둘레는 30km, 면적은 13.28㎢, 최대수심은 45m에 이른다. 섬으로는 오시마 섬, 고시마 섬, 주루이 섬, 야이타이 섬이 떠있다. 그중 주루이 섬에는 천연기념물인 <a href="/wiki/%EB%A7%88%EB%A6%AC%EB%AA%A8" title="마리모">마리모</a> 전시관람 센터가 입지해 있다.</p>
<ul>
<li>유입하천 : 이베시베쓰 강, 기네탄베쓰 강, 주루이 강, 본추루이 강, 시리코마베쓰 강 등</li>
<li>유출하천 : <a class="new" href="/w/index.php?title=%EC%95%84%EC%B9%B8_%EA%B0%95&amp;action=edit&amp;redlink=1" title="아칸 강 (없는 문서)">아칸 강</a></li>
</ul>
