<p><b>아커쑤 시</b>(<span style="font-size: smaller;"><a href="/wiki/%EC%9C%84%EA%B5%AC%EB%A5%B4%EC%96%B4" title="위구르어">위구르어</a>:</span> <span lang="ug" xml:lang="ug">ئاقسۇ شەھىرى</span> <small>아크수 시</small>, <small><a href="/wiki/%EC%A4%91%EA%B5%AD%EC%96%B4" title="중국어">중국어</a></small> <small><a href="/wiki/%EA%B0%84%EC%B2%B4%EC%9E%90" title="간체자">간체</a>:</small> <span lang="zh-hans" xml:lang="zh-hans">阿克苏市</span>, <small><a href="/wiki/%EC%A0%95%EC%B2%B4%EC%9E%90" title="정체자">정체</a>:</small> <span lang="zh-hant" xml:lang="zh-hant">阿克蘇市</span>, <small><a href="/wiki/%ED%95%9C%EC%96%B4_%EB%B3%91%EC%9D%8C" title="한어 병음">병음</a>:</small> <font face="Lucida Sans Unicode">Ākèsū Shì</font>)는 중국 신장의 <a href="/wiki/%ED%98%84%EA%B8%89%EC%8B%9C" title="현급시">현급시</a>이며 <a href="/wiki/%EC%95%84%EC%BB%A4%EC%91%A4_%EC%A7%80%EA%B5%AC" title="아커쑤 지구">아커쑤 지구</a>의 수도이다. 이름 아크수는 위구르어로 '흰 물'을 의미하며 도시와 <a class="new" href="/w/index.php?title=%EC%95%84%EC%BB%A4%EC%91%A4%ED%97%88&amp;action=edit&amp;redlink=1" title="아커쑤허 (없는 문서)">아커쑤허</a>를 모두 나타낸다.</p>

<h2><span class="mw-headline" id=".EC.A7.80.EB.A6.AC">지리</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%95%84%EC%BB%A4%EC%91%A4_%EC%8B%9C&amp;action=edit&amp;section=1" title="부분 편집: 지리">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>도시는 <a href="/wiki/%ED%86%88%EC%82%B0_%EC%82%B0%EB%A7%A5" title="톈산 산맥">톈산 산맥</a>의 남쪽기슭에 위치해 있으며 그곳은 북쪽 <a class="mw-redirect" href="/wiki/%EC%8B%A4%ED%81%AC%EB%A1%9C%EB%93%9C" title="실크로드">실크로드</a> 상의 중요한 휴식처이며 <a href="/wiki/%ED%83%80%EB%A6%BC_%EB%B6%84%EC%A7%80" title="타림 분지">타림 분지</a>내의 <a href="/wiki/%ED%83%80%ED%81%B4%EB%9D%BC%EB%A7%88%EC%B9%B8_%EC%82%AC%EB%A7%89" title="타클라마칸 사막">타클라마칸 사막</a>의 북쪽이며 <a class="mw-redirect" href="/wiki/%EC%BF%A0%EC%B0%A8" title="쿠차">쿠차</a>와 <a class="mw-redirect" href="/wiki/%EC%B9%B4%EC%8A%88%EA%B0%80%EB%A5%B4" title="카슈가르">카슈가르</a> 사이에 위치한다. 시의 해발고도는 1114.8m이다.</p>
<p>연평균 기온은 9.8℃이고 연평균 강수량은 62mm이다.</p>
<h2><span class="mw-headline" id=".EC.97.AD.EC.82.AC">역사</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%95%84%EC%BB%A4%EC%91%A4_%EC%8B%9C&amp;action=edit&amp;section=2" title="부분 편집: 역사">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>적어도 <a href="/wiki/%EC%A0%84%ED%95%9C" title="전한">전한</a>(<a href="/wiki/%EA%B8%B0%EC%9B%90%EC%A0%84_125%EB%85%84" title="기원전 125년">기원전 125년</a> ~ <a href="/wiki/23%EB%85%84" title="23년">23년</a>)때부터 <a href="/wiki/%EB%8B%B9%EB%82%98%EB%9D%BC" title="당나라">당나라</a>(<a href="/wiki/618%EB%85%84" title="618년">618년</a> ~ <a href="/wiki/907%EB%85%84" title="907년">907년</a>)때까지 아크수는 구모(姑墨)로 알려져 있었다. 불교에서는 그곳이 바루카로 알려져 있다.</p>
<p>전한 시대에는 이곳이 왕국으로 기술되어 있다. 3,500가구와 24,500명의 인구를 가졌고 4,500명이 무기를 들 수 있었다고 기록되어 있다. 또한 그곳에서는 구리, 철과 웅황을 생산하였다고 한다.</p>
<p>중국의 삼장(현장)법사가 이 왕국을 방문하였고 발루카라고 하였다. 그는 아크수에 수십 개의 사원과 1,000명 이상의 승려가 있었고 이 왕국의 반경은 동에서 서로 600리에 걸치며, 남북으로는 300리에 달한다고 기술하고 있다.</p>
<p>7세기, 8세기, 9세기에 이 지역의 지배권을 놓고 중국의 <a href="/wiki/%EB%8B%B9%EB%82%98%EB%9D%BC" title="당나라">당나라</a>, 티베트의 <a href="/wiki/%ED%86%A0%EB%B2%88%EA%B5%AD" title="토번국">토번국</a>, <a class="mw-redirect" href="/wiki/%EC%9C%84%EA%B5%AC%EB%A5%B4_%EC%A0%9C%EA%B5%AD" title="위구르 제국">위구르 제국</a>이 경쟁하였고 도시의 지배권은 빈번하게 바뀌었다. 티베트인들은 670년에 아크수를 차지하였으나 당나라는 692년에 이 지역을 재정복하였다. 티베트인들은 720년대에 타림 분지를 탈환하였고 당나라는 740년대에 이 지역을 다시 합병하였다. <a href="/wiki/%ED%83%88%EB%9D%BC%EC%8A%A4_%EC%A0%84%ED%88%AC" title="탈라스 전투">탈라스 전투</a>는 이 지역의 중국군을 점차 철수하게 만들었고 위구르와 티베트가 이 지역을 놓고 경쟁하게 되었다.</p>
<p>아크수는 비록 쓰이는 언어가 표준 <a href="/wiki/%ED%86%A0%EC%B9%B4%EB%A6%AC%EC%95%84%EC%96%B4" title="토카리아어">토카리아어</a>와 조금 다르지만 <a class="mw-redirect" href="/wiki/%EC%BF%A0%EC%B0%A8" title="쿠차">쿠차</a>와 강한 연결성을 갖고 있다. 타림 북쪽 <a class="mw-redirect" href="/wiki/%EC%8B%A4%ED%81%AC%EB%A1%9C%EB%93%9C" title="실크로드">실크로드</a>와 비옥한 <a class="mw-redirect" href="/wiki/%EC%9D%B4%EB%A6%AC_%EA%B0%95" title="이리 강">이리 강</a> 계곡의 북쪽 길의 무역로의 교차점에 위치한다.</p>
<p>1220년 경에 아크수는 <a class="new" href="/w/index.php?title=%EB%A7%8C%EA%B0%80%EB%9D%BC%EC%9D%B4_%EC%99%95%EA%B5%AD&amp;action=edit&amp;redlink=1" title="만가라이 왕국 (없는 문서)">만가라이 왕국</a>의 수도가 되었다.</p>
<p><a class="new" href="/w/index.php?title=%ED%94%84%EB%9E%9C%EC%8B%9C%EC%8A%A4_%EC%98%81%ED%97%88%EC%8A%A4%EB%B0%B4%EB%93%9C&amp;action=edit&amp;redlink=1" title="프랜시스 영허스밴드 (없는 문서)">프랜시스 영허스밴드</a>는 <a class="mw-redirect" href="/wiki/%EB%B2%A0%EC%9D%B4%EC%A7%95" title="베이징">베이징</a>에서 <a href="/wiki/%EC%9D%B8%EB%8F%84" title="인도">인도</a>로 가는 여행길에 아크수를 방문했다. 그는 이곳을 베이징으로 가는 길에 본 가장 큰 마을로 묘사했다. 지역의 인구는 약 2만 명이고 2천 명의 군인이 주둔하고 있었다. "이곳에는 커다란 시장과 몇 개의 여관이 있다 - 몇몇 여행객과 상인들은 이곳에 오래 머물며 물건을 팔기를 바란다"</p>
<h2><span class="mw-headline" id=".EC.9D.B8.EA.B5.AC">인구</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%95%84%EC%BB%A4%EC%91%A4_%EC%8B%9C&amp;action=edit&amp;section=3" title="부분 편집: 인구">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p><a href="/wiki/2003%EB%85%84" title="2003년">2003년</a> 기준으로 인구가 572,694명으로 주로 <a href="/wiki/%ED%95%9C%EC%A1%B1" title="한족">한족</a>이며 도시 자체의 인구는 362,300명이다. <a href="/wiki/%ED%9B%84%EC%9D%B4%EC%A1%B1" title="후이족">후이족</a>, <a class="mw-redirect" href="/wiki/%ED%82%A4%EB%A5%B4%EA%B8%B0%EC%8A%A4%EC%A1%B1" title="키르기스족">키르기스족</a>, <a href="/wiki/%EB%AA%BD%EA%B3%A8%EC%A1%B1_(%EC%A4%91%EA%B5%AD)" title="몽골족 (중국)">몽골족</a>, <a class="mw-redirect" href="/wiki/%EC%B9%B4%EC%9E%90%ED%9D%90%EC%A1%B1" title="카자흐족">카자흐족</a> 등의 소수민족이 45%를 차지한다.</p>
<h2><span class="mw-headline" id=".EA.B2.BD.EC.A0.9C">경제</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%95%84%EC%BB%A4%EC%91%A4_%EC%8B%9C&amp;action=edit&amp;section=4" title="부분 편집: 경제">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>아크수의 경제는 대개 농업이며 목화를 심는다. 또한 곡식과 과일, 기름, 사탕무 등을 생산한다. 산업은 주로 직물과 시멘트 화학산업으로 구성된다.</p>
<h2><span class="mw-headline" id=".EC.97.B0.EB.8C.80_.EA.B8.B0.EB.A1.9D">연대 기록</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%95%84%EC%BB%A4%EC%91%A4_%EC%8B%9C&amp;action=edit&amp;section=5" title="부분 편집: 연대 기록">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<ul>
<li><a href="/wiki/630%EB%85%84" title="630년">630년</a>: <a href="/wiki/%EC%82%BC%EC%9E%A5" title="삼장">삼장</a>(현장) 아커쑤를 방문하다.</li>
</ul>
<h2><span class="mw-headline" id=".EC.A0.91.EA.B2.BD">접경</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%95%84%EC%BB%A4%EC%91%A4_%EC%8B%9C&amp;action=edit&amp;section=6" title="부분 편집: 접경">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>아크수는 남서로 <a class="mw-redirect" href="/wiki/%EC%B9%B4%EC%8A%88%EA%B0%80%EB%A5%B4" title="카슈가르">카슈가르</a>와 <a class="mw-redirect" href="/wiki/%EC%BF%A0%EC%B0%A8" title="쿠차">쿠차</a>와 맞닿아 있으며, <a class="new" href="/w/index.php?title=%EC%B9%B4%EB%9D%BC%EC%82%AC%EB%A5%B4&amp;action=edit&amp;redlink=1" title="카라사르 (없는 문서)">카라사르</a>와 <a class="mw-redirect" href="/wiki/%ED%88%AC%EB%A5%B4%ED%8C%90" title="투르판">투르판</a>이 동쪽으로 접경해있다. 사막 건너 남쪽은 호탄과 맞닿아 있다.</p>
<h2><span class="mw-headline" id=".ED.96.89.EC.A0.95_.EA.B5.AC.EC.97.AD">행정 구역</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%95%84%EC%BB%A4%EC%91%A4_%EC%8B%9C&amp;action=edit&amp;section=7" title="부분 편집: 행정 구역">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
