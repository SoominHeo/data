<p><b>아크플라이</b>(<span style="font-size: smaller;"><a href="/wiki/%EC%98%81%EC%96%B4" title="영어">영어</a>:</span> <span lang="en" xml:lang="en">Arkefly</span>)은 <a href="/wiki/%EB%84%A4%EB%8D%9C%EB%9E%80%EB%93%9C" title="네덜란드">네덜란드</a>의 지역 항공사로 총 70개 노선을 취항하고 있다. 본사는 <a href="/wiki/%EB%84%A4%EB%8D%9C%EB%9E%80%EB%93%9C" title="네덜란드">네덜란드</a> <a href="/wiki/%EC%95%94%EC%8A%A4%ED%85%8C%EB%A5%B4%EB%8B%B4" title="암스테르담">암스테르담</a>에 위치해 있다. 또한 사용하고 있는 <a href="/wiki/%ED%97%88%EB%B8%8C_%EA%B3%B5%ED%95%AD" title="허브 공항">허브 공항</a>으로 <a class="mw-redirect" href="/wiki/%EC%8A%A4%ED%9E%88%ED%8F%B4_%EA%B5%AD%EC%A0%9C%EA%B3%B5%ED%95%AD" title="스히폴 국제공항">스히폴 국제공항</a>이 있으며 계열사는 <a href="/wiki/%EB%B2%A8%EA%B8%B0%EC%97%90" title="벨기에">벨기에</a>의 항공사인 <a class="new" href="/w/index.php?title=TUI_%ED%8A%B8%EB%9F%AC%EB%B8%94&amp;action=edit&amp;redlink=1" title="TUI 트러블 (없는 문서)">TUI 트러블</a>이 있다.</p>

<h2><span class="mw-headline" id=".EC.97.AD.EC.82.AC">역사</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%95%84%ED%81%AC%ED%94%8C%EB%9D%BC%EC%9D%B4&amp;action=edit&amp;section=1" title="부분 편집: 역사">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p><a href="/wiki/1981%EB%85%84" title="1981년">1981년</a> 에어 네덜란드로 설립했다. <a href="/wiki/2000%EB%85%84%EB%8C%80" title="2000년대">2000년대</a> 들면서 재정난에 빠지면서 <a href="/wiki/2004%EB%85%84" title="2004년">2004년</a>에 XL 에이비에이션 그룹에 인수했다. 하지만 <a href="/wiki/2005%EB%85%84" title="2005년">2005년</a>에 XL 에이비에이션 그룹이 파산되면서 <a href="/wiki/%EB%B2%A8%EA%B8%B0%EC%97%90" title="벨기에">벨기에</a>의 항공사인 <a class="mw-redirect" href="/wiki/TUI_%ED%95%AD%EA%B3%B5" title="TUI 항공">TUI 항공</a>이 인수해 현재의 명칭으로 변경하고 <a href="/wiki/2005%EB%85%84" title="2005년">2005년</a> <a href="/wiki/9%EC%9B%94" title="9월">9월</a> 운항을 재개했다. <a class="mw-redirect" href="/wiki/TUI_%ED%95%AD%EA%B3%B5" title="TUI 항공">TUI 항공</a>의 자회사로 <a href="/wiki/%EB%84%A4%EB%8D%9C%EB%9E%80%EB%93%9C" title="네덜란드">네덜란드</a> 최대의 여행사인 아크 여행사에 전세기를 많이 제공하고 있다.</p>
