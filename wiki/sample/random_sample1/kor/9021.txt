<p><a href="/wiki/1997%EB%85%84" title="1997년">1997년</a> <a href="/wiki/7%EC%9B%94_3%EC%9D%BC" title="7월 3일">7월 3일</a>, <a href="/wiki/%EC%8B%9C%EC%95%A0%ED%8B%80_%EB%A7%A4%EB%A6%AC%EB%84%88%EC%8A%A4" title="시애틀 매리너스">시애틀 매리너스</a>와 입단 계약을 맺는다. 이후 <a href="/wiki/2002%EB%85%84" title="2002년">2002년</a> <a href="/wiki/12%EC%9B%94_16%EC%9D%BC" title="12월 16일">12월 16일</a> 룰 5 드래프트를 통해 <a class="mw-redirect" href="/wiki/%ED%86%A0%EB%A1%A0%ED%86%A0_%EB%B8%94%EB%A3%A8_%EC%A0%9C%EC%9D%B4%EC%8A%A4" title="토론토 블루 제이스">토론토 블루 제이스</a>로 이적하였고, <a href="/wiki/2003%EB%85%84" title="2003년">2003년</a> <a href="/wiki/4%EC%9B%94_2%EC%9D%BC" title="4월 2일">4월 2일</a> <a href="/wiki/%EB%A9%94%EC%9D%B4%EC%A0%80_%EB%A6%AC%EA%B7%B8_%EB%B2%A0%EC%9D%B4%EC%8A%A4%EB%B3%BC" title="메이저 리그 베이스볼">메이저 리그</a> 데뷔 경기를 갖는다. <a href="/wiki/2003%EB%85%84" title="2003년">2003년</a> 시즌 총 72번 등판하여 73.2이닝 동안 3.42의 평균 자책점을 기록한다.</p>
<p><a href="/wiki/2004%EB%85%84" title="2004년">2004년</a> 시즌 18경기에 등판하여 21이닝을 투구하였다. 이후 <a href="/wiki/%EB%A1%9C%EC%8A%A4%EC%95%A4%EC%A0%A4%EB%A0%88%EC%8A%A4_%EB%8B%A4%EC%A0%80%EC%8A%A4" title="로스앤젤레스 다저스">로스앤젤레스 다저스</a>와 <a href="/wiki/%ED%95%84%EB%9D%BC%EB%8D%B8%ED%94%BC%EC%95%84_%ED%95%84%EB%A6%AC%EC%8A%A4" title="필라델피아 필리스">필라델피아 필리스</a> 산하 <a href="/wiki/%EB%A7%88%EC%9D%B4%EB%84%88_%EB%A6%AC%EA%B7%B8_%EB%B2%A0%EC%9D%B4%EC%8A%A4%EB%B3%BC" title="마이너 리그 베이스볼">마이너 리그</a> 팀에서 활동하게 되고, <a href="/wiki/2007%EB%85%84" title="2007년">2007년</a> 시즌 전 <a href="/wiki/%EB%94%94%ED%8A%B8%EB%A1%9C%EC%9D%B4%ED%8A%B8_%ED%83%80%EC%9D%B4%EA%B1%B0%EC%8A%A4" title="디트로이트 타이거스">디트로이트 타이거스</a>와 <a href="/wiki/%EB%A7%88%EC%9D%B4%EB%84%88_%EB%A6%AC%EA%B7%B8_%EB%B2%A0%EC%9D%B4%EC%8A%A4%EB%B3%BC" title="마이너 리그 베이스볼">마이너</a> 계약을 체결한다.</p>
<p><a href="/wiki/2008%EB%85%84" title="2008년">2008년</a> 시즌 <a href="/wiki/%EB%A9%94%EC%9D%B4%EC%A0%80_%EB%A6%AC%EA%B7%B8_%EB%B2%A0%EC%9D%B4%EC%8A%A4%EB%B3%BC" title="메이저 리그 베이스볼">메이저 리그</a> <a href="/wiki/%EB%94%94%ED%8A%B8%EB%A1%9C%EC%9D%B4%ED%8A%B8_%ED%83%80%EC%9D%B4%EA%B1%B0%EC%8A%A4" title="디트로이트 타이거스">디트로이트 타이거스</a>에서 활동했지만 페르난도 로드니 선수가 부상자 명단에서 <a href="/wiki/%EB%A9%94%EC%9D%B4%EC%A0%80_%EB%A6%AC%EA%B7%B8_%EB%B2%A0%EC%9D%B4%EC%8A%A4%EB%B3%BC" title="메이저 리그 베이스볼">메이저 리그</a> 로스터로 복귀하자 로페즈는 <a href="/wiki/%EB%A7%88%EC%9D%B4%EB%84%88_%EB%A6%AC%EA%B7%B8_%EB%B2%A0%EC%9D%B4%EC%8A%A4%EB%B3%BC" title="마이너 리그 베이스볼">마이너 리그</a>로 내려가게 되고 이후 <a href="/wiki/7%EC%9B%94_2%EC%9D%BC" title="7월 2일">7월 2일</a> 다시 <a href="/wiki/%EB%A9%94%EC%9D%B4%EC%A0%80_%EB%A6%AC%EA%B7%B8_%EB%B2%A0%EC%9D%B4%EC%8A%A4%EB%B3%BC" title="메이저 리그 베이스볼">메이저 리그</a>로 올라오게 된다. <a href="/wiki/2008%EB%85%84" title="2008년">2008년</a> 시즌 평균 자책점은 상대적으로 낮은 3.55였지만 승계 주자 실점이 <a href="/wiki/%EB%A9%94%EC%9D%B4%EC%A0%80_%EB%A6%AC%EA%B7%B8_%EB%B2%A0%EC%9D%B4%EC%8A%A4%EB%B3%BC" title="메이저 리그 베이스볼">메이저 리그</a>에서 6번째로 높은 40.9%였다. 시즌 종료 후 <a href="/wiki/%EB%94%94%ED%8A%B8%EB%A1%9C%EC%9D%B4%ED%8A%B8_%ED%83%80%EC%9D%B4%EA%B1%B0%EC%8A%A4" title="디트로이트 타이거스">디트로이트 타이거스</a> 팀은 연봉 조정 신청을 거부함에 따라 자유 계약 선수가 되었다.</p>
<h2><span class="mw-headline" id="KBO_.EB.A6.AC.EA.B7.B8_.EC.8B.9C.EC.A0.88"><a href="/wiki/KBO_%EB%A6%AC%EA%B7%B8" title="KBO 리그">KBO 리그</a> 시절</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%95%84%ED%82%AC%EB%A6%AC%EB%85%B8_%EB%A1%9C%ED%8E%98%EC%8A%A4&amp;action=edit&amp;section=2" title="부분 편집: KBO 리그 시절">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<h3><span class="mw-headline" id="KIA_.ED.83.80.EC.9D.B4.EA.B1.B0.EC.A6.88"><a href="/wiki/KIA_%ED%83%80%EC%9D%B4%EA%B1%B0%EC%A6%88" title="KIA 타이거즈">KIA 타이거즈</a></span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%95%84%ED%82%AC%EB%A6%AC%EB%85%B8_%EB%A1%9C%ED%8E%98%EC%8A%A4&amp;action=edit&amp;section=3" title="부분 편집: KIA 타이거즈">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p><a href="/wiki/2009%EB%85%84" title="2009년">2009년</a> <a href="/wiki/1%EC%9B%94_14%EC%9D%BC" title="1월 14일">1월 14일</a> 계약금 5만 달러, 연봉 25만 달러의 조건으로 <a href="/wiki/KIA_%ED%83%80%EC%9D%B4%EA%B1%B0%EC%A6%88" title="KIA 타이거즈">KIA 타이거즈</a>와 입단 계약을 맺었다.<sup class="reference" id="cite_ref-1"><a href="#cite_note-1">[1]</a></sup> 광주에서 열렸던 <a href="/wiki/2009%EB%85%84_%ED%95%9C%EA%B5%AD%EC%8B%9C%EB%A6%AC%EC%A6%88" title="2009년 한국시리즈">2009년 한국시리즈</a> 1차전에서 선발 등판하여 완벽투를 보여 주었고, 5차전에서 완봉승을 거두면서 <a href="/wiki/KIA_%ED%83%80%EC%9D%B4%EA%B1%B0%EC%A6%88" title="KIA 타이거즈">KIA 타이거즈</a>의 <a class="mw-redirect" href="/wiki/%ED%95%9C%EA%B5%AD%EC%8B%9C%EB%A6%AC%EC%A6%88" title="한국시리즈">한국시리즈</a> 우승에 견인차 역할을 하였다. 5차전의 완봉승에 이은 피로에도 불구하고 7차전 8회 초에 본인 스스로 중간계투를 자원하여 이닝을 마무리했다. 3경기 동안 2승 1.53을 기록하여 <a href="/wiki/KIA_%ED%83%80%EC%9D%B4%EA%B1%B0%EC%A6%88" title="KIA 타이거즈">KIA 타이거즈</a>의 <a class="mw-redirect" href="/wiki/%ED%95%9C%EA%B5%AD%EC%8B%9C%EB%A6%AC%EC%A6%88" title="한국시리즈">한국시리즈</a> V10을 이루는데 큰 기여를 하는 모습을 보여 주었고, <a href="/wiki/KIA_%ED%83%80%EC%9D%B4%EA%B1%B0%EC%A6%88" title="KIA 타이거즈">KIA 타이거즈</a> 팬들은 그의 헌신적인 행동에 크게 감명받았다.</p>
<p>골든글러브 <a href="/wiki/%ED%88%AC%EC%88%98" title="투수">투수</a> 부문에 선정되었다.<sup class="reference" id="cite_ref-2"><a href="#cite_note-2">[2]</a></sup> <a href="/wiki/2009%EB%85%84" title="2009년">2009년</a> <a href="/wiki/12%EC%9B%94_2%EC%9D%BC" title="12월 2일">12월 2일</a> <a href="/wiki/KIA_%ED%83%80%EC%9D%B4%EA%B1%B0%EC%A6%88" title="KIA 타이거즈">KIA 타이거즈</a>와 계약금 5만 달러, 연봉 30만 달러, 총액 35만 달러로 재계약하여 <a href="/wiki/2010%EB%85%84" title="2010년">2010년</a>에도 <a href="/wiki/KIA_%ED%83%80%EC%9D%B4%EA%B1%B0%EC%A6%88" title="KIA 타이거즈">KIA 타이거즈</a>에서 활동했다.</p>
<p>그러나 2년차 징크스 때문인지 <a href="/wiki/2010%EB%85%84" title="2010년">2010년</a>에는 5승 10패 1세이브 방어율 4.46로 부진하였고, 게다가 다혈질적인 성격이라 리드 상황에서 내려가고 나서 구원 투수들이 동점을 내주면 <a href="/wiki/%EB%8D%95%EC%95%84%EC%9B%83" title="덕아웃">덕아웃</a>에서 화풀이를 하는 모습을 자주 보여 주었다. 먼저 <a href="/wiki/4%EC%9B%94_23%EC%9D%BC" title="4월 23일">4월 23일</a> <a href="/wiki/%EB%84%A5%EC%84%BC_%ED%9E%88%EC%96%B4%EB%A1%9C%EC%A6%88" title="넥센 히어로즈">넥센 히어로즈</a>전에서 8회말 1아웃까지 3-2로 <a href="/wiki/KIA_%ED%83%80%EC%9D%B4%EA%B1%B0%EC%A6%88" title="KIA 타이거즈">KIA</a>가 이기고 있는 상황에서 5-4-3으로 이어지는 완벽한 병살 플레이를 만들 수 있었는데, 1루 주자는 잡았으나 2루수 <a href="/wiki/%EC%95%88%EC%B9%98%ED%99%8D" title="안치홍">안치홍</a>의 악송구로 인해 동점이 되어 버렸고, 결국 강판당하고 나서 <a href="/wiki/%EB%8D%95%EC%95%84%EC%9B%83" title="덕아웃">덕아웃</a>에서 글러브를 집어던지고 쓰레기통을 걷어차고 라커룸으로 들어가 버렸다.</p>
<p><a href="/wiki/6%EC%9B%94_30%EC%9D%BC" title="6월 30일">6월 30일</a> <a href="/wiki/SK_%EC%99%80%EC%9D%B4%EB%B2%88%EC%8A%A4" title="SK 와이번스">SK 와이번스</a>전에서도 7이닝 2실점으로 퀄리티 스타트를 보여주었으나 8회초에 <a class="mw-redirect" href="/wiki/%EA%B9%80%ED%9D%AC%EA%B1%B8" title="김희걸">김희걸</a>이 나와서 안타를 계속 맞았으며, 게다가 3루수 <a href="/wiki/%EC%9D%B4%ED%98%84%EA%B3%A4" title="이현곤">이현곤</a>의 실책까지 겹치면서 만루가 되어 버렸고, <a href="/wiki/%EC%A1%B0%EB%B2%94%ED%98%84" title="조범현">조범현</a> 감독은 <a href="/wiki/%EC%9C%A0%EB%8F%99%ED%9B%88" title="유동훈">유동훈</a>을 투입했으나 <a href="/wiki/%EC%A1%B0%EB%8F%99%ED%99%94" title="조동화">조동화</a>에게 내야 €나타낸 <a href="/wiki/%EC%A0%95%EA%B7%BC%EC%9A%B0" title="정근우">정근우</a>에게마저도 2타점 적시타를 맞으면서 <a href="/wiki/%EC%9C%A0%EB%8F%99%ED%9B%88" title="유동훈">유동훈</a>의 블론 세이브와 동시에 자신의 2승도 날아가 버리자, 결국 화를 참지 못하고 덕아웃에서 의자를 집어던져 버렸다. 이 장면은 <a class="mw-redirect" href="/wiki/MBC_%EC%8A%A4%ED%8F%AC%EC%B8%A0%2B" title="MBC 스포츠+">MBC 스포츠+</a>을 통해 전국에 방송되면서 큰 충격을 주었고, 로페즈는 갖은 비난과 더불어 팀내 자체징계로 벌금 500만원과 엄중 경고를 받아야 했다. 하지만 후반기부터는 점차 안정적이었던 <a href="/wiki/2009%EB%85%84" title="2009년">2009년</a>의 모습을 되찾아 3승을 따 냈고, 오히려 자신보다 성적이 좋았던 <a href="/wiki/%EB%A1%9C%EB%A7%8C_%EC%BD%9C%EB%A1%A0" title="로만 콜론">콜론</a>이 로페즈보다 떨어지는 이닝 소화 능력으로 인해 재계약에 실패했다. 로페즈는 시즌 막판 제 페이스를 찾게 되자 <a href="/wiki/KIA_%ED%83%80%EC%9D%B4%EA%B1%B0%EC%A6%88" title="KIA 타이거즈">KIA 타이거즈</a> 구단으로부터 경기 중 감정을 드러내지 말 것, 윈터 리그에 뛰지 말 것 등의 조건으로 재계약했다. <a href="/wiki/2011%EB%85%84" title="2011년">2011년</a> 전반기에 자신의 페이스를 되찾았으나, 후반기 옆구리 통증 후유증으로 제 실력을 보여주지 못했고, 11승 9패를 기록했다. <a href="/wiki/%EC%84%A0%EB%8F%99%EC%97%B4" title="선동열">선동열</a> 감독 부임 이후 <a href="/wiki/2012%EB%85%84" title="2012년">2012년</a> 시즌 전 조건 없이 방출되어 국내 타 구단으로 이적이 가능하게 되었다. 이후 <a href="/wiki/SK_%EC%99%80%EC%9D%B4%EB%B2%88%EC%8A%A4" title="SK 와이번스">SK 와이번스</a>와 총액 40만 달러를 받고 이적하게 되어 한국 프로 야구에서 계속 뛸 수 있게 되었다.<sup class="reference" id="cite_ref-3"><a href="#cite_note-3">[3]</a></sup></p>
<h3><span class="mw-headline" id="SK_.EC.99.80.EC.9D.B4.EB.B2.88.EC.8A.A4"><a href="/wiki/SK_%EC%99%80%EC%9D%B4%EB%B2%88%EC%8A%A4" title="SK 와이번스">SK 와이번스</a></span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%95%84%ED%82%AC%EB%A6%AC%EB%85%B8_%EB%A1%9C%ED%8E%98%EC%8A%A4&amp;action=edit&amp;section=4" title="부분 편집: SK 와이번스">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p><a href="/wiki/2012%EB%85%84" title="2012년">2012년</a> KIA에서 재계약을 포기한 로페즈는 총액 40만 달러를 받고 <a href="/wiki/SK_%EC%99%80%EC%9D%B4%EB%B2%88%EC%8A%A4" title="SK 와이번스">SK 와이번스</a>로 이적하게 되었다. 하지만 그 해 어깨 부상이 심화되어 2군에 자주 갔다 오게 되었고, 그 해 <a href="/wiki/6%EC%9B%94_5%EC%9D%BC" title="6월 5일">6월 5일</a> 잠실 <a href="/wiki/%EB%91%90%EC%82%B0_%EB%B2%A0%EC%96%B4%EC%8A%A4" title="두산 베어스">두산 베어스</a>와의 선발 등판 경기를 마지막으로 퇴출당하면서 한국 프로 야구를 떠나게 된다. 두산전 선발승으로 유종의 미를 거두고 떠났다.</p>
