<h1 class="firstHeading" id="firstHeading" lang="ko">아타리 7800</h1>
<div class="mw-body-content" id="bodyContent">
<div id="siteSub">위키백과, 우리 모두의 백과사전.</div>

<p><b>아타리 7800</b>(Atari 7800)은 1986년 6월 <a href="/wiki/%EC%95%84%ED%83%80%EB%A6%AC" title="아타리">아타리</a>에서 발매한 <a href="/wiki/%EA%B0%80%EC%A0%95%EC%9A%A9_%EA%B2%8C%EC%9E%84%EA%B8%B0" title="가정용 게임기">가정용 게임기</a>로 실패한 <a href="/wiki/%EC%95%84%ED%83%80%EB%A6%AC_5200" title="아타리 5200">아타리 5200</a>을 대체하여 <a href="/wiki/%EB%8B%8C%ED%85%90%EB%8F%84" title="닌텐도">닌텐도</a>와 <a class="mw-redirect" href="/wiki/%EC%84%B8%EA%B0%80" title="세가">세가</a>로부터 시장 1위를 탈환하기 위해 투입되었다.</p>
<p><a href="/wiki/%EC%95%84%ED%83%80%EB%A6%AC_5200" title="아타리 5200">아타리 5200</a>의 단점을 보완해 심플한 <a href="/wiki/%EB%94%94%EC%A7%80%ED%84%B8" title="디지털">디지털</a> <a href="/wiki/%EC%A1%B0%EC%9D%B4%EC%8A%A4%ED%8B%B1" title="조이스틱">조이스틱</a>과 <a href="/wiki/%EC%95%84%ED%83%80%EB%A6%AC_2600" title="아타리 2600">아타리 2600</a>과의 호환성을 확보했고 당시로서는 적당한 가격인 140<a href="/wiki/%EB%8B%AC%EB%9F%AC" title="달러">달러</a>에 출시되었다.</p>
<h2><span class="mw-headline" id=".EC.84.B8.EB.B6.80.EC.82.AC.ED.95.AD">세부사항</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%95%84%ED%83%80%EB%A6%AC_7800&amp;action=edit&amp;section=1" title="부분 편집: 세부사항">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>아타리 7800은 1984년 6월 남부 <a class="mw-redirect" href="/wiki/%EC%BA%98%EB%A6%AC%ED%8F%AC%EB%8B%88%EC%95%84" title="캘리포니아">캘리포니아</a>에서 시험 판매하였으나 한달후 워너 커뮤니케이션에서 아타리를 인수한 Jack Tramiel은 새로운 16비트 컴퓨터 개발에 집중하기 위해 모든 계획을 연기하였다. 1986년 7800은 다시 판매되었으나 시장은 이미 닌텐도의 <a href="/wiki/%ED%8C%A8%EB%B0%80%EB%A6%AC_%EC%BB%B4%ED%93%A8%ED%84%B0" title="패밀리 컴퓨터">패밀리 컴퓨터</a>(NES)가 90%를 차지하고 있었고 나머지는 세가의 <a class="mw-redirect" href="/wiki/%EB%A7%88%EC%8A%A4%ED%84%B0_%EC%8B%9C%EC%8A%A4%ED%85%9C" title="마스터 시스템">마스터 시스템</a>이 점유하고 있었다.</p>
<p>7800은 <a href="/wiki/%EC%95%84%ED%83%80%EB%A6%AC_2600" title="아타리 2600">아타리 2600</a>과 동일한 칩을 사용하여 호환성을 확보했다. 7800 모드에서 <a href="/wiki/%EC%95%84%ED%83%80%EB%A6%AC_2600" title="아타리 2600">아타리 2600</a> 칩은 사운드와 컨트롤러 인터페이스만을 담당하며 그래픽은 마리아 <a href="/wiki/%EA%B7%B8%EB%9E%98%ED%94%BD_%EC%B2%98%EB%A6%AC_%EC%9E%A5%EC%B9%98" title="그래픽 처리 장치">GPU</a>를 이용한다. 그리고 볼 블레이저(Ballblazer)나 코만도(Commando)게임 같이 게임 카트리지 안에 <a href="/wiki/%EC%95%84%ED%83%80%EB%A6%AC_%ED%8F%AC%ED%82%A4" title="아타리 포키">포키 오디오 칩</a>을 내장하여 사운드 효과를 향상할 수 있었다.</p>
<p><a href="/wiki/NTSC" title="NTSC">NTSC</a> 7800 <a href="/wiki/%EB%B0%94%EC%9D%B4%EC%98%A4%EC%8A%A4" title="바이오스">바이오스</a>는 디지털 서명을 생성하고 카트리지에 저장된 서명에 비교하여 2600과 7800 카트리지를 구별하며 아타리에게 인증된 개발자 만이 게임을 제작할 수 있게 만들었다. <a href="/wiki/PAL" title="PAL">PAL</a>방식의 7800은 디지털 서명 코드가 없으며 2600 카트리지를 구별하는 데 다양한 방법을 사용했다.</p>
<p>7800은 <a class="mw-redirect" href="/wiki/%EC%95%84%ED%83%80%EB%A6%AC_%EC%87%BC%ED%81%AC" title="아타리 쇼크">아타리 쇼크</a>이후 판매된 다른 아타리 게임기와 마찬가지로 극심한 소프트웨어 부족에 직면했다. 이전 게임기의 게임들과 마찬가지로 아타리가 발매한 소수의 게임 중 대부분이 투박하고 내용이 부실했으며 <a class="new" href="/w/index.php?title=%EC%95%B1%EC%86%94%EB%A3%A8%ED%8A%B8&amp;action=edit&amp;redlink=1" title="앱솔루트 (없는 문서)">앱솔루트</a>(Absolute)와 <a href="/wiki/%EC%95%A1%ED%8B%B0%EB%B9%84%EC%A0%84" title="액티비전">액티비전</a>(Activision), <a class="new" href="/w/index.php?title=%ED%94%84%EB%A1%9C%EA%B3%A0&amp;action=edit&amp;redlink=1" title="프로고 (없는 문서)">프로고</a>(Froggo)를 제외한 아타리의 다른 서드 파티 개발사들은 닌텐도의 독점 계약에 묶여 다른 게임기용 게임을 개발할 수 없었다.</p>
<p><a href="/wiki/%EC%95%84%ED%83%80%EB%A6%AC" title="아타리">아타리</a>는 <a href="/wiki/%EB%8B%8C%ED%85%90%EB%8F%84" title="닌텐도">닌텐도</a>, <a class="mw-redirect" href="/wiki/%EC%84%B8%EA%B0%80" title="세가">세가</a>와의 벌어진 격차를 줄이지 못했고 1992년 1월 1일 공식적으로 아타리 7800을 포함해 <a href="/wiki/%EC%95%84%ED%83%80%EB%A6%AC_2600" title="아타리 2600">아타리 2600</a>, <a class="new" href="/w/index.php?title=%EC%95%84%ED%83%80%EB%A6%AC_8%EB%B9%84%ED%8A%B8_%EC%BB%B4%ED%93%A8%ED%84%B0&amp;action=edit&amp;redlink=1" title="아타리 8비트 컴퓨터 (없는 문서)">아타리 8비트 컴퓨터</a>, <a class="new" href="/w/index.php?title=%EC%95%84%ED%83%80%EB%A6%AC_XE_%EA%B2%8C%EC%9E%84%EC%8B%9C%EC%8A%A4%ED%85%9C&amp;action=edit&amp;redlink=1" title="아타리 XE 게임시스템 (없는 문서)">아타리 XE 게임시스템</a>을 단종시켰다.</p>
<h2><span class="mw-headline" id=".EC.82.AC.EC.96.91">사양</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%95%84%ED%83%80%EB%A6%AC_7800&amp;action=edit&amp;section=2" title="부분 편집: 사양">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
