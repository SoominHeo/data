<p>﻿ <b>페르가몬 왕국</b> 또는 <b>아탈로스 왕조</b>는 <a class="mw-redirect" href="/wiki/%EC%95%8C%EB%A0%89%EC%82%B0%EB%93%9C%EB%A1%9C%EC%8A%A4_%EB%8C%80%EC%99%95" title="알렉산드로스 대왕">알렉산드로스 대왕</a>의 장군중 한명이였던 <a href="/wiki/%EB%A6%AC%EC%8B%9C%EB%A7%88%EC%BD%94%EC%8A%A4" title="리시마코스">리시마코스</a>의 사망이후에 <a href="/wiki/%ED%8E%98%EB%A5%B4%EA%B0%80%EB%AA%AC" title="페르가몬">페르가몬</a>을 지배했던 <a class="mw-redirect" href="/wiki/%ED%97%AC%EB%A0%88%EB%8B%88%EC%A6%98_%EC%8B%9C%EB%8C%80" title="헬레니즘 시대">헬레니즘</a> 왕조이다. 페르가문 왕국은 리시마코스의 제국이 붕괴된 이후에 생겨난 <a class="mw-redirect" href="/wiki/%EC%9E%94%EB%A5%98%EA%B5%AD" title="잔류국">잔류국</a>였다. 리시마코스의 관료 중 하나였던 <a href="/wiki/%ED%95%84%EB%A0%88%ED%83%80%EC%9D%B4%EB%A1%9C%EC%8A%A4" title="필레타이로스">필레타이로스</a>가 기원전 282년에 페르가몬의 통치권을 차지했다. 이후에 에우메네스 1세는 그의 아버지의 유래에 따라, 도시에서 왕국으로 확장시켰다. <a class="new" href="/w/index.php?title=%EC%95%84%ED%83%88%EB%A1%9C%EC%8A%A4_1%EC%84%B8&amp;action=edit&amp;redlink=1" title="아탈로스 1세 (없는 문서)">아탈로스 1세</a>는 <a href="/wiki/%EA%B0%88%EB%9D%BC%ED%8B%B0%EC%95%84" title="갈라티아">갈라티아인</a>들을 상대로 승리를 거둔후, 기원전 230년대에 스스로를 왕으로 선언하였다. 아탈리드 왕조는 <a class="new" href="/w/index.php?title=%EC%95%84%ED%83%88%EB%A1%9C%EC%8A%A4_3%EC%84%B8&amp;action=edit&amp;redlink=1" title="아탈로스 3세 (없는 문서)">아탈로스 3세</a>까지 페르가몬을 통치했으며, 왕위 계승의 위험을 피하기 위해 유언에 따라 기원전 133년에 <a class="mw-disambig" href="/wiki/%EB%A1%9C%EB%A7%88_%EA%B3%B5%ED%99%94%EA%B5%AD" title="로마 공화국">로마 공화국</a>에 귀속되었다.</p>
<p><a class="new" href="/w/index.php?title=%ED%8E%98%EB%A5%B4%EA%B0%80%EB%AA%AC_%EB%8C%80%EC%A0%9C%EB%8B%A8&amp;action=edit&amp;redlink=1" title="페르가몬 대제단 (없는 문서)">페르가몬 대제단</a>의 내부에는 <a href="/wiki/%ED%97%A4%EB%9D%BC%ED%81%B4%EB%A0%88%EC%8A%A4" title="헤라클레스">헤라클레스</a>의 아들이자 아탈로스 왕조 통치시기와 관련되어 올림피아인들의 후손이라는 주장에 활용됐던 <a class="new" href="/w/index.php?title=%ED%85%94%EB%A0%88%ED%8F%AC%EC%8A%A4&amp;action=edit&amp;redlink=1" title="텔레포스 (없는 문서)">텔레포스</a>의 삶이 묘사된 <a class="new" href="/w/index.php?title=%ED%94%84%EB%A6%AC%EC%A6%88_(%EA%B1%B4%EC%B6%95)&amp;action=edit&amp;redlink=1" title="프리즈 (건축) (없는 문서)">프리즈</a>가 있다.</p>
