<h1 class="firstHeading" id="firstHeading" lang="ko">아테네 론 테니스 클럽</h1>
<div class="mw-body-content" id="bodyContent">

<div class="mw-content-ltr" dir="ltr" id="mw-content-text" lang="ko"><script>function mfTempOpenSection(id){var block=document.getElementById("mf-section-"+id);block.className+=" open-block";block.previousSibling.className+=" open-block";}</script><p><b>아테네 론 테니스 클럽</b>(<span lang="el" xml:lang="el">Όμιλος Αντισφαίρισης Αθηνών in greco</span>)은 <a href="/wiki/1895%EB%85%84" title="1895년">1895년</a>에 만들어진 <a href="/wiki/%EA%B7%B8%EB%A6%AC%EC%8A%A4" title="그리스">그리스</a> <a href="/wiki/%EC%95%84%ED%85%8C%EB%84%A4" title="아테네">아테네</a>에 있는 <a href="/wiki/%ED%85%8C%EB%8B%88%EC%8A%A4" title="테니스">테니스</a> 경기장이다. <a href="/wiki/1896%EB%85%84_%ED%95%98%EA%B3%84_%EC%98%AC%EB%A6%BC%ED%94%BD" title="1896년 하계 올림픽">1896년 하계 올림픽</a>에서 <a href="/wiki/1896%EB%85%84_%ED%95%98%EA%B3%84_%EC%98%AC%EB%A6%BC%ED%94%BD_%ED%85%8C%EB%8B%88%EC%8A%A4" title="1896년 하계 올림픽 테니스">테니스</a>와 <a href="/wiki/1896%EB%85%84_%ED%95%98%EA%B3%84_%EC%98%AC%EB%A6%BC%ED%94%BD_%EC%82%AC%EC%9D%B4%ED%81%B4" title="1896년 하계 올림픽 사이클">사이클</a> 경기가 열렸던 곳이다. 이 경기장은 <a href="/wiki/1906%EB%85%84_%EC%A4%91%EA%B0%84_%EC%98%AC%EB%A6%BC%ED%94%BD" title="1906년 중간 올림픽">1906년 중간 올림픽</a> 때도 쓰였다.</p>
<p>아테네 론 테니스 클럽은 <a href="/wiki/%EC%A0%9C%EC%9A%B0%EC%8A%A4_%EC%8B%A0%EC%A0%84" title="제우스 신전">제우스 신전</a>과 <a href="/wiki/%EC%95%84%ED%81%AC%EB%A1%9C%ED%8F%B4%EB%A6%AC%EC%8A%A4" title="아크로폴리스">아크로폴리스</a> 사이에 위치해있으며 <a href="/wiki/%ED%8C%8C%EB%82%98%ED%8B%B0%EB%82%98%EC%9D%B4%EC%BD%94_%EA%B2%BD%EA%B8%B0%EC%9E%A5" title="파나티나이코 경기장">파나티나이코 경기장</a> 근처에 있는 동시에 <a class="mw-redirect" href="/wiki/%EC%9E%90%ED%94%BC%EC%98%A8" title="자피온">자피온</a> 건너편에 있다.</p>
