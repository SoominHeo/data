<p><b>아트 스피겔먼</b>(<span lang="en" xml:lang="en">Art Spiegelman</span>, <a href="/wiki/1948%EB%85%84" title="1948년">1948년</a> <a href="/wiki/2%EC%9B%94_15%EC%9D%BC" title="2월 15일">2월 15일</a> ~ )은 <a href="/wiki/%EB%AF%B8%EA%B5%AD" title="미국">미국</a>의 <a href="/wiki/%EB%A7%8C%ED%99%94%EA%B0%80" title="만화가">만화가</a>, 편집자이다. 자신의 아버지인 블라덱 스피겔먼의 젊은시절 경험담이며 실화인 <a href="/wiki/%ED%93%B0%EB%A6%AC%EC%B2%98%EC%83%81" title="퓰리처상">퓰리처상</a> 수상작 <a href="/wiki/%EC%A5%90_(%EB%A7%8C%ED%99%94)" title="쥐 (만화)">쥐</a>로 잘 알려져 있다.</p>
<h2><span class="mw-headline" id=".EB.8B.A4.EB.A5.B8_.EB.A7.A4.EC.B2.B4.EC.97.90_.EB.93.B1.EC.9E.A5.ED.95.9C_.EA.B8.B0.EB.A1.9D">다른 매체에 등장한 기록</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%95%84%ED%8A%B8_%EC%8A%A4%ED%94%BC%EA%B2%94%EB%A8%BC&amp;action=edit&amp;section=1" title="부분 편집: 다른 매체에 등장한 기록">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>스피겔먼은 2007년 <a href="/wiki/%EC%8B%AC%EC%8A%A8_%EA%B0%80%EC%A1%B1" title="심슨 가족">심슨 가족</a>의 "Husbands and Knives" 편에 배우 <a href="/wiki/%EC%9E%AD_%EB%B8%94%EB%9E%99" title="잭 블랙">잭 블랙</a>, 다른 만화가들 <a href="/wiki/%EC%95%A8%EB%9F%B0_%EB%AC%B4%EC%96%B4" title="앨런 무어">앨런 무어</a>, <a class="new" href="/w/index.php?title=Dan_Clowes&amp;action=edit&amp;redlink=1" title="Dan Clowes (없는 문서)">Dan Clowes</a> 과 같이 출연했다.</p>
