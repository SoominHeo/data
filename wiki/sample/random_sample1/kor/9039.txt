<p><b>아티스</b>(<span style="font-size: smaller;"><a href="/wiki/%EC%98%81%EC%96%B4" title="영어">영어</a>:</span> <span lang="en" xml:lang="en">Attis</span>, <span style="font-size: smaller;"><a href="/wiki/%EA%B3%A0%EB%8C%80_%EA%B7%B8%EB%A6%AC%EC%8A%A4%EC%96%B4" title="고대 그리스어">고대 그리스어</a>:</span> <span lang="grc" xml:lang="grc">Ἄττις/Ἄττης</span>)는 <a href="/wiki/%EC%95%84%EB%82%98%ED%86%A8%EB%A6%AC%EC%95%84" title="아나톨리아">아나톨리아</a>의 <a href="/wiki/%ED%94%84%EB%A6%AC%EA%B8%B0%EC%95%84" title="프리기아">프리기아</a>를 기원으로 하는 <a href="/wiki/%EC%A3%BD%EC%9D%8C" title="죽음">죽음</a>과 <a class="mw-disambig" href="/wiki/%EB%B6%80%ED%99%9C" title="부활">부활</a>의 <a href="/wiki/%EC%8B%A0" title="신">신</a>이며 <a href="/wiki/%EC%A0%9C%EC%9A%B0%EC%8A%A4" title="제우스">제우스</a>와 <a href="/wiki/%EA%B0%80%EC%9D%B4%EC%95%84" title="가이아">가이아</a>의 손자이자 <a href="/wiki/%EB%A7%88%EB%84%A4%EC%8A%A4" title="마네스">마네스</a>의 아들이기도 하다. 모신인 <a href="/wiki/%ED%82%A4%EB%B2%A8%EB%A0%88" title="키벨레">키벨레</a>와 함께 <a href="/wiki/%EA%B3%A0%EB%8C%80_%EA%B7%B8%EB%A6%AC%EC%8A%A4" title="고대 그리스">고대 그리스</a>와 <a href="/wiki/%EA%B3%A0%EB%8C%80_%EB%A1%9C%EB%A7%88" title="고대 로마">고대 로마</a> 등에 전파되었다.</p>

<h2><span class="mw-headline" id=".EA.B0.9C.EC.9A.94">개요</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%95%84%ED%8B%B0%EC%8A%A4&amp;action=edit&amp;section=1" title="부분 편집: 개요">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>아티스는 <a class="mw-redirect" href="/wiki/%EB%8C%80%EC%A7%80%EB%AA%A8" title="대지모">대지모</a>신으로 알려진 프리기아의 여신 키벨레의 아들이며 애인이기도 하다. 일설에 따르면 <a href="/wiki/%EC%A0%9C%EC%9A%B0%EC%8A%A4" title="제우스">제우스</a>와 <a href="/wiki/%EA%B0%80%EC%9D%B4%EC%95%84" title="가이아">가이아</a>의 아들 <a href="/wiki/%EB%A7%88%EB%84%A4%EC%8A%A4" title="마네스">마네스</a>가 그의 아버지라고 한다. 그러나 키벨레는 아티스가 다른 인간의 여성과 사랑에 빠지자 그를 미쳐 자살하게 만들었다. 이후 아티스는 나무의 형태로 부활하게 된다.</p>
<h2><span class="mw-headline" id=".EC.A0.84.EC.84.A4">전설</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%95%84%ED%8B%B0%EC%8A%A4&amp;action=edit&amp;section=2" title="부분 편집: 전설">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<h3><span class="mw-headline" id=".EA.B8.B0.EC.9B.90">기원</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%95%84%ED%8B%B0%EC%8A%A4&amp;action=edit&amp;section=3" title="부분 편집: 기원">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>아티스는 본래 프리기아의 상업도시 <a class="new" href="/w/index.php?title=%ED%8E%AB%EC%82%AC%EB%88%84%EC%8A%A4&amp;action=edit&amp;redlink=1" title="펫사누스 (없는 문서)">펫사누스</a>(<a class="extiw" href="https://en.wikipedia.org/wiki/Pessinus" title="en:Pessinus">en:Pessinus</a>)와 관련된 <a href="/wiki/%EB%B0%98%EC%8B%A0" title="반신">반신</a>이었다. 그 도시에 바람이 불어 오는 쪽에는 <a class="new" href="/w/index.php?title=%EC%95%84%EA%B7%B8%EB%94%94%EC%8A%A4%ED%8B%B0%EC%8A%A4&amp;action=edit&amp;redlink=1" title="아그디스티스 (없는 문서)">아그디스티스</a>(<a class="extiw" href="https://en.wikipedia.org/wiki/Agdistis" title="en:Agdistis">en:Agdistis</a>)<sup class="reference" id="cite_ref-1"><a href="#cite_note-1">[1]</a></sup> 산이 있어, 그 산은 이전부터 <a href="/wiki/%EC%95%85%EB%A7%88" title="악마">데몬</a>의 체현이라고 여겨지고 있었다.</p>
<p>아티스가 아그디스티스(키벨레)로부터 유래한다는 이야기를 들은 것은 여행자 <a class="mw-disambig" href="/wiki/%ED%8C%8C%EC%9A%B0%EC%82%AC%EB%8B%88%EC%95%84%EC%8A%A4" title="파우사니아스">파우사니아스</a>(<a class="extiw" href="https://en.wikipedia.org/wiki/Pausanias" title="en:Pausanias">en:Pausanias</a>)였다. 그가 전해 들은 이야기에 따르면, 아그디스티스는 본래 <a class="mw-redirect" href="/wiki/%EB%B0%98%EC%9D%8C%EC%96%91" title="반음양">반음양</a>이었다. 그러나 <a href="/wiki/%EC%98%AC%EB%A6%BC%ED%91%B8%EC%8A%A4" title="올림푸스">올림푸스</a>의 신들은 아그디스티스를 두려워해 그의 남성기를 절제해 버렸다. 이윽고 지상에 버려진 성기에선 한 그루의 <a href="/wiki/%EC%95%84%EB%AA%AC%EB%93%9C" title="아몬드">아몬드</a> 나무가 자라났다. 그 나무가 여물었을 때, <a class="new" href="/w/index.php?title=%EC%83%81%EA%B0%80%EB%A6%AC%EC%98%A4%EC%8A%A4&amp;action=edit&amp;redlink=1" title="상가리오스 (없는 문서)">상가리오스</a>(<a class="extiw" href="https://en.wikipedia.org/wiki/Sangarios" title="en:Sangarios">en:Sangarios</a>)<sup class="reference" id="cite_ref-2"><a href="#cite_note-2">[2]</a></sup> 강의 딸 나나가 그 씨앗을 가슴에 숨겼다. 얼마 지나지 않아 씨앗은 사라져 그녀의 몸 속으로 흡수되었다. 그리고 사내아이가 태어났지만, 곧 들판에 버려지고 그를 수컷의 <a class="mw-disambig" href="/wiki/%EC%97%BC%EC%86%8C" title="염소">염소</a>들이 키우게 되었다.</p>
<h3><span class="mw-headline" id=".EC.A3.BD.EC.9D.8C.EA.B3.BC_.EB.B6.80.ED.99.9C">죽음과 부활</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%95%84%ED%8B%B0%EC%8A%A4&amp;action=edit&amp;section=4" title="부분 편집: 죽음과 부활">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>아티스는 자라나면서 장발의 미남자가 되었다. 이윽고 아그디스티스(키벨레)는 아티스와 사랑에 빠졌다. 그러나 아티스의 수양부모는 그를 펫사누스에 보내 그곳의 왕녀와 결혼시키려고 하였다. 이것에 질투를 느낀 아그디스티스는 초절적인 힘을 발휘해 아티스의 정신을 분열시키고 스스로 거세하여 죽게 만들었다. 하지만 펫사누스의 왕도 그것을 모방하여 스스로를 거세하고 키벨레에 몸을 바치는 <a class="new" href="/w/index.php?title=%EC%BD%94%EB%A5%98%EB%B0%98%ED%85%8C%EC%8A%A4&amp;action=edit&amp;redlink=1" title="코류반테스 (없는 문서)">코류반테스</a><sup class="reference" id="cite_ref-3"><a href="#cite_note-3">[3]</a></sup> 의 선례가 되었다. 이윽고 아그디스티스는 아티스를 죽인 것을 후회하여 그의 신체를 쇠약해지지 않는 <a href="/wiki/%EC%A0%84%EB%82%98%EB%AC%B4" title="전나무">전나무</a>로 바꾸어 버렸다.<sup class="reference" id="cite_ref-4"><a href="#cite_note-4">[4]</a></sup></p>
<h3><span class="mw-headline" id=".EB.8F.BC.EC.A7.80.EC.99.80_.EA.B4.80.EB.A0.A8">돼지와 관련</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%95%84%ED%8B%B0%EC%8A%A4&amp;action=edit&amp;section=5" title="부분 편집: 돼지와 관련">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>이웃의 <a href="/wiki/%EB%A6%AC%EB%94%94%EC%95%84" title="리디아">리디아</a>가 프리기아를 지배하게 되자, 키벨레와 아티스의 신앙이 보다 널리 퍼지게 되었다. 이것은 <a href="/wiki/%EC%A0%9C%EC%9A%B0%EC%8A%A4" title="제우스">제우스</a>의 질투를 불렀다. 제우스는 리디아의 작물을 파괴할 수 있도록 한 필의 <a href="/wiki/%EB%8F%BC%EC%A7%80" title="돼지">수퇘지</a>를 풀었다. 게다가 수퇘지들은 아티스는 물론 몇몇의 리디아인들을 살해했다. 파우사니아스는 이것과 펫사누스의 코류반테스들이 돼지를 먹지 않는 습관을 연관시켰다. 신화의 이러한 부분은, 리디아의 코류반테스들이 돼지를 먹지 않는 금기를 설명하기 위해 생각해 낸 것이라고 여긴다.</p>
<h2><span class="mw-headline" id=".EC.98.81.ED.96.A5">영향</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%95%84%ED%8B%B0%EC%8A%A4&amp;action=edit&amp;section=6" title="부분 편집: 영향">편집</a><span class="mw-editsection-bracket">]</span></span></h2>

<li>모셀(mosel)이 복원한 아티스의 상은 아나톨리아의 신들이 입는 전형적인 의복을 갖추고 <a class="mw-redirect" href="/wiki/%ED%94%84%EB%A4%BC%EA%B8%B0%EC%95%84_%EB%AA%A8%EC%9E%90" title="프뤼기아 모자">프리기아 모자</a>를 쓰고있다.(우측사진)</li>
