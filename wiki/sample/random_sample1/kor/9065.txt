<p><b>아프리카폐어속</b>(<i>Protopterus</i>)은 <a href="/wiki/%EC%95%84%ED%94%84%EB%A6%AC%EC%B9%B4" title="아프리카">아프리카</a>에서 발견되는 <a href="/wiki/%ED%8F%90%EC%96%B4%EB%A5%98" title="폐어류">폐어류</a> 속의 하나이다. <b>아프리카폐어과</b>(Protopteridae)의 유일속으로 4종으로 이루어져 있다.</p>
<h2><span class="mw-headline" id=".ED.95.98.EC.9C.84_.EC.A2.85">하위 종</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%95%84%ED%94%84%EB%A6%AC%EC%B9%B4%ED%8F%90%EC%96%B4%EC%86%8D&amp;action=edit&amp;section=1" title="부분 편집: 하위 종">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>4종으로 이루어져 있다.<sup class="reference" id="cite_ref-FB_1-0"><a href="#cite_note-FB-1">[1]</a></sup></p>
<ul>
<li><a href="/wiki/%ED%91%9C%EB%B2%94%ED%8F%90%EC%96%B4" title="표범폐어">표범폐어</a> 또는 <a class="mw-redirect" href="/wiki/%ED%94%84%EB%A1%9C%ED%86%B1%ED%85%8C%EB%A3%A8%EC%8A%A4_%EC%97%90%EC%9D%B4%ED%8B%B0%EC%98%A4%ED%94%BC%EC%BF%A0%EC%8A%A4" title="프로톱테루스 에이티오피쿠스">프로톱테루스 에이티오피쿠스</a> (<i>Protopterus aethiopicus</i>)
<ul>
<li><i>Protopterus aethiopicus aethiopicus</i> <small>Heckel, 1851</small></li>
<li><i>Protopterus aethiopicus congicus</i> <small>Poll, 1961</small></li>
<li><i>Protopterus aethiopicus mesmaekersi</i> <small>Poll, 1961</small></li>
</ul>
</li>
<li><a href="/wiki/%EC%84%9C%EC%95%84%ED%94%84%EB%A6%AC%EC%B9%B4%ED%8F%90%EC%96%B4" title="서아프리카폐어">서아프리카폐어</a> 또는 <a class="mw-redirect" href="/wiki/%ED%94%84%EB%A1%9C%ED%86%B1%ED%85%8C%EB%A3%A8%EC%8A%A4_%EC%95%88%EB%84%A5%ED%85%90%EC%8A%A4" title="프로톱테루스 안넥텐스">프로톱테루스 안넥텐스</a> (<i>Protopterus annectens</i>)
<ul>
<li><i>Protopterus annectens annectens</i> <small>(Owen, 1839)</small></li>
<li><i>Protopterus annectens brieni</i> <small>Poll, 1961</small> 남방폐어</li>
</ul>
</li>
<li><a href="/wiki/%EB%8F%99%EC%95%84%ED%94%84%EB%A6%AC%EC%B9%B4%ED%8F%90%EC%96%B4" title="동아프리카폐어">동아프리카폐어</a> 또는 <a class="mw-redirect" href="/wiki/%ED%94%84%EB%A1%9C%ED%86%B1%ED%85%8C%EB%A3%A8%EC%8A%A4_%EC%95%94%ED%94%BC%EB%B9%84%EC%9A%B0%EC%8A%A4" title="프로톱테루스 암피비우스">프로톱테루스 암피비우스</a> (<i>Protopterus amphibius</i>)</li>
<li><a href="/wiki/%EB%B0%98%EC%A0%90%EC%95%84%ED%94%84%EB%A6%AC%EC%B9%B4%ED%8F%90%EC%96%B4" title="반점아프리카폐어">반점아프리카폐어</a> 또는 <a class="mw-redirect" href="/wiki/%ED%94%84%EB%A1%9C%ED%86%B1%ED%85%8C%EB%A3%A8%EC%8A%A4_%EB%8F%8C%EB%A1%9C%EC%9D%B4" title="프로톱테루스 돌로이">프로톱테루스 돌로이</a> (<i>Protopterus dolloi</i>)</li>
