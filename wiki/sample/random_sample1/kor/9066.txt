<p><b>아프리키야 항공</b>(<span style="font-size: smaller;"><a href="/wiki/%EC%98%81%EC%96%B4" title="영어">영어</a>:</span> <span lang="en" xml:lang="en">Afriqiyah Airways</span>, <span style="font-size: smaller;"><a href="/wiki/%EC%95%84%EB%9E%8D%EC%96%B4" title="아랍어">아랍어</a>:</span> <span lang="ar" xml:lang="ar">الخطوط الجوية الأفريقية</span>)은 <a href="/wiki/%EB%A6%AC%EB%B9%84%EC%95%84" title="리비아">리비아</a>의 <a href="/wiki/%ED%95%AD%EA%B3%B5%EC%82%AC" title="항공사">항공사</a>이다. 수도 <a class="mw-disambig" href="/wiki/%ED%8A%B8%EB%A6%AC%ED%8F%B4%EB%A6%AC" title="트리폴리">트리폴리</a>에 본사가 있으며 중심(허브)공항인 <a href="/wiki/%ED%8A%B8%EB%A6%AC%ED%8F%B4%EB%A6%AC_%EA%B5%AD%EC%A0%9C%EA%B3%B5%ED%95%AD" title="트리폴리 국제공항">트리폴리 국제공항</a>을 통해 <a href="/wiki/%EC%95%84%ED%94%84%EB%A6%AC%EC%B9%B4" title="아프리카">아프리카</a>, <a href="/wiki/%EC%9C%A0%EB%9F%BD" title="유럽">유럽</a>, <a href="/wiki/%EC%95%84%EC%8B%9C%EC%95%84" title="아시아">아시아</a>와 <a href="/wiki/%EC%A4%91%EB%8F%99" title="중동">중동</a>에 취항하고 있다. <i>아프리키야</i>라는 이름은 아랍어로 아프리카를 뜻한다. 현재 항공 연합은 가입하지 않았다. 아프리키야 항공은 <a href="/wiki/%EA%B5%AD%EC%A0%9C_%ED%95%AD%EA%B3%B5_%EC%9A%B4%EC%86%A1_%ED%98%91%ED%9A%8C" title="국제 항공 운송 협회">국제 항공 운송 협회</a>에 가입했다.</p>

<h2><span class="mw-headline" id=".EA.B0.9C.EC.9A.94">개요</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%95%84%ED%94%84%EB%A6%AC%ED%82%A4%EC%95%BC_%ED%95%AD%EA%B3%B5&amp;action=edit&amp;section=1" title="부분 편집: 개요">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<h3><span class="mw-headline" id=".EA.B5.AD.EC.98.81_.ED.95.AD.EA.B3.B5">국영 항공</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%95%84%ED%94%84%EB%A6%AC%ED%82%A4%EC%95%BC_%ED%95%AD%EA%B3%B5&amp;action=edit&amp;section=2" title="부분 편집: 국영 항공">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p><a href="/wiki/2001%EB%85%84" title="2001년">2001년</a> <a href="/wiki/%EB%A6%AC%EB%B9%84%EC%95%84" title="리비아">리비아 정부</a>가 100% 주주인 국유 기업으로 설립했다. 아랍 항공기구(<span style="font-size: smaller;"><a href="/wiki/%EC%98%81%EC%96%B4" title="영어">영어</a>:</span> <span lang="en" xml:lang="en">Arab Air Carriers Organization</span>)의 일원으로 현재 <a href="/wiki/%EB%A6%AC%EB%B9%84%EC%95%84" title="리비아">리비아</a> 국내선과 <a href="/wiki/%EC%9C%A0%EB%9F%BD" title="유럽">유럽</a>, <a href="/wiki/%EC%95%84%ED%94%84%EB%A6%AC%EC%B9%B4" title="아프리카">아프리카</a> 각지에 정기 노선을 운항하고 있다.</p>
<h3><span class="mw-headline" id=".EB.82.AE.EC.9D.80_.EC.84.9C.EB.B9.84.EC.8A.A4_.ED.8F.89.EA.B0.80">낮은 서비스 평가</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%95%84%ED%94%84%EB%A6%AC%ED%82%A4%EC%95%BC_%ED%95%AD%EA%B3%B5&amp;action=edit&amp;section=3" title="부분 편집: 낮은 서비스 평가">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p><a href="/wiki/%EC%9C%A0%EB%9F%BD" title="유럽">유럽</a>의 <a href="/wiki/%EC%97%90%EC%96%B4%EB%B2%84%EC%8A%A4" title="에어버스">에어버스</a>의 최신 기종으로 많이 운항하고 있으며, 근거리 국제선의 <a href="/wiki/%EC%97%90%EC%96%B4%EB%B2%84%EC%8A%A4_A320" title="에어버스 A320">에어버스 A320</a> 항공기의 이코노미 클래스에 개인용 TV가 장착되는 등 하드웨어 측면은 충실하고 있지만, 그 서비스에 대해 평가는 낮고, <a href="/wiki/%EC%98%81%EA%B5%AD" title="영국">영국</a> 항공 조사 기관 <a href="/wiki/%EC%8A%A4%EC%B9%B4%EC%9D%B4%ED%8A%B8%EB%9E%99%EC%8A%A4" title="스카이트랙스">스카이트랙스</a>에 평가는 최저 등급인 1등급으로 평가받고 있다.</p>
<h2><span class="mw-headline" id=".EC.9A.B4.ED.95.AD_.EB.85.B8.EC.84.A0">운항 노선</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%95%84%ED%94%84%EB%A6%AC%ED%82%A4%EC%95%BC_%ED%95%AD%EA%B3%B5&amp;action=edit&amp;section=4" title="부분 편집: 운항 노선">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<div class="rellink relarticle mainarticle"><img alt="" data-file-height="128" data-file-width="128" height="16" src="//upload.wikimedia.org/wikipedia/commons/thumb/e/ec/Crystal_Clear_app_xmag.svg/16px-Crystal_Clear_app_xmag.svg.png" srcset="//upload.wikimedia.org/wikipedia/commons/thumb/e/ec/Crystal_Clear_app_xmag.svg/24px-Crystal_Clear_app_xmag.svg.png 1.5x, //upload.wikimedia.org/wikipedia/commons/thumb/e/ec/Crystal_Clear_app_xmag.svg/32px-Crystal_Clear_app_xmag.svg.png 2x" width="16"/> 이 부분의 본문은 <a href="/wiki/%EC%95%84%ED%94%84%EB%A6%AC%ED%82%A4%EC%95%BC_%ED%95%AD%EA%B3%B5%EC%9D%98_%EC%9A%B4%ED%95%AD_%EB%85%B8%EC%84%A0" title="아프리키야 항공의 운항 노선">아프리키야 항공의 운항 노선</a>입니다.</div>
<ul>
<li><a href="/wiki/2011%EB%85%84" title="2011년">2011년</a> <a href="/wiki/11%EC%9B%94" title="11월">11월</a> 현재 아프리키야 항공은 <a href="/wiki/%EC%95%84%ED%94%84%EB%A6%AC%EC%B9%B4" title="아프리카">아프리카</a>, <a href="/wiki/%EC%95%84%EC%8B%9C%EC%95%84" title="아시아">아시아</a> 그리고 <a href="/wiki/%EC%9C%A0%EB%9F%BD" title="유럽">유럽</a>의 여러 도시들로 취항하고 있다.</li>
</ul>

<h2><span class="mw-headline" id=".EC.82.AC.EA.B1.B4_.EB.B0.8F_.EC.82.AC.EA.B3.A0">사건 및 사고</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%95%84%ED%94%84%EB%A6%AC%ED%82%A4%EC%95%BC_%ED%95%AD%EA%B3%B5&amp;action=edit&amp;section=8" title="부분 편집: 사건 및 사고">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<ul>
<li><a href="/wiki/2010%EB%85%84" title="2010년">2010년</a> <a href="/wiki/6%EC%9B%94_12%EC%9D%BC" title="6월 12일">6월 12일</a> <a class="new" href="/w/index.php?title=%EC%95%84%ED%94%84%EB%A6%AC%ED%82%A4%EC%95%84%ED%95%AD%EA%B3%B5_711%ED%8E%B8&amp;action=edit&amp;redlink=1" title="아프리키아항공 711편 (없는 문서)">아프리키아항공 711편</a> 여객기가 <a href="/wiki/%EB%82%A8%EC%95%84%ED%94%84%EB%A6%AC%EC%B9%B4_%EA%B3%B5%ED%99%94%EA%B5%AD" title="남아프리카 공화국">남아프리카 공화국</a>의 <a class="mw-redirect" href="/wiki/%EC%9A%94%ED%95%98%EB%84%A4%EC%8A%A4%EB%B2%84%EA%B7%B8_%EA%B5%AD%EC%A0%9C%EA%B3%B5%ED%95%AD" title="요하네스버그 국제공항">요하네스버그 국제공항</a>을 출발하여 리비아 트리폴리 공항에 착륙하던 중 일어난 사고이며, 본래 항공기는 <a href="/wiki/%EB%9F%B0%EB%8D%98_%EA%B0%9C%ED%8A%B8%EC%9C%85_%EA%B3%B5%ED%95%AD" title="런던 개트윅 공항">런던 개트윅 공항</a>으로 가던 중이였다. 이 사고로 <a href="/wiki/%EB%84%A4%EB%8D%9C%EB%9E%80%EB%93%9C" title="네덜란드">네덜란드</a>의 소년 1명을 제외한 모든 탑승객이 생명을 잃었다.</li>
</ul>
