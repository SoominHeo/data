<p><b>아피아니</b>(Appiani) 또는 <b>아피아노</b>(Appiano), <b>다피아노</b>(d'Appiano) 가문은 오늘날 <a class="new" href="/w/index.php?title=%ED%8F%B0%EC%82%AC%EC%BD%94&amp;action=edit&amp;redlink=1" title="폰사코 (없는 문서)">폰사코</a>의 라 피에페(La Pieve)에서 확인된 오늘날에는 사라진 <a href="/wiki/%EC%A7%80%EB%AA%85%ED%95%99" title="지명학">지명학</a>인 알피아노(Al Piano) 또는 아피아노(Appiano)에서 기원한 이탈리아의 귀족 가문이다. 그들은 15세기 초에서 1628년까지 <a href="/wiki/%ED%94%BC%EC%98%B4%EB%B9%84%EB%85%B8_%EB%8C%80%EA%B3%B5%EA%B5%AD" title="피옴비노 대공국">피옴비노 대공국</a>을 통치했다.</p>
<h2><span class="mw-headline" id=".EC.97.AD.EC.82.AC">역사</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%95%84%ED%94%BC%EC%95%84%EB%8B%88_%EA%B0%80&amp;action=edit&amp;section=1" title="부분 편집: 역사">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>아피아니 가문 중 처음으로 알려진 이는 1200년에서 1255년 사이에 살았던 공증인 구아르니토 다피아노(Guarnito d'Appiano)이며, 그의 아들 야코포(Jacopo, <a href="/wiki/%ED%94%BC%EC%82%AC" title="피사">피사</a>에서 흥함. 1230년-1290년) 또한 공증인이였고, 게다가 구아르니토의 손자 벤베누토(Benvenuto)는 피사 공증인 조합의 조합장이 되었다. 또한 공증인이였던 그의 증손자 반니(Vanni)는 피사의 안치아노(Anziano, 영사)가 그 이후에는 1347년에 <a href="/wiki/%EB%A3%A8%EC%B9%B4_%EA%B3%B5%ED%99%94%EA%B5%AD" title="루카 공화국">루카</a>의 장관이 되었다. 하지만 그는 1355년 5월 피사에서 처형을 당하고 만다. 반니의 아들 야코포(Jacopo, 1322년 - 1398년)는 <a href="/wiki/%ED%94%BC%EC%82%AC_%EA%B3%B5%ED%99%94%EA%B5%AD" title="피사 공화국">피사 공화국</a>의 장관이자 <a class="new" href="/w/index.php?title=%EB%8D%B8%EB%9D%BC_%EA%B2%8C%EB%9D%BC%EB%A5%B4%EB%8D%B0%EC%8A%A4%EC%B9%B4_%EA%B0%80%EB%AC%B8&amp;action=edit&amp;redlink=1" title="델라 게라르데스카 가문 (없는 문서)">델라 게라르데스카 가문</a>과 함께 라스판티당(Raspanti)의 수장이 되었다. 1392년, 피에트로 감바코르티(Pietro Gambacorti)와 그의 아들들의 암살 후, 그는 10년간 피사의 사실상의 군주가 되었다.</p>
<p>그는 1398년 아들인 <a class="new" href="/w/index.php?title=%EA%B2%8C%EB%9D%BC%EB%A5%B4%EB%8F%84_%EC%95%84%ED%94%BC%EC%95%84%EB%8B%88&amp;action=edit&amp;redlink=1" title="게라르도 아피아니 (없는 문서)">게라르도</a>에게 자리를 물려주었고, 1년 뒤 게라르도는 피사의 주권을 <a href="/wiki/%EB%B0%80%EB%9D%BC%EB%85%B8_%EA%B3%B5%EA%B5%AD" title="밀라노 공국">밀라노의 군주</a> <a href="/wiki/%EC%9E%94_%EA%B0%88%EB%A0%88%EC%95%84%EC%B4%88_%EB%B9%84%EC%8A%A4%EC%BD%98%ED%8B%B0" title="잔 갈레아초 비스콘티">잔 갈레아초 비스콘티</a>에게 넘기게된다. 하지만 잔 갈레아초가 사망한 뒤, 피사는 <a class="new" href="/w/index.php?title=%EA%B0%80%EB%B8%8C%EB%A6%AC%EC%97%98%EB%A0%88_%EB%A7%88%EB%A6%AC%EC%95%84_%EB%B9%84%EC%8A%A4%EC%BD%98%ED%8B%B0&amp;action=edit&amp;redlink=1" title="가브리엘레 마리아 비스콘티 (없는 문서)">가브리엘레 마리아 비스콘티</a>에 의해 <a href="/wiki/%ED%94%BC%EB%A0%8C%EC%B2%B4" title="피렌체">피렌체</a>에 팔리게 되었으며, 한편 아피아니 가문은 <a href="/wiki/%ED%94%BC%EC%98%B4%EB%B9%84%EB%85%B8" title="피옴비노">피옴비노</a>, <a href="/wiki/%EC%88%98%EB%B2%A0%EB%A0%88%ED%86%A0" title="수베레토">수베레토</a>, 부리아노(Buriano), <a class="new" href="/w/index.php?title=%EC%8A%A4%EC%B9%BC%EB%A6%AC%EB%85%B8&amp;action=edit&amp;redlink=1" title="스칼리노 (없는 문서)">스칼리노</a>, <a class="new" href="/w/index.php?title=%EB%B9%84%EB%83%98%EB%A0%88&amp;action=edit&amp;redlink=1" title="비냘레 (없는 문서)">비냘레</a>, <a class="new" href="/w/index.php?title=%ED%8F%AC%ED%92%80%EB%A1%9C%EB%8B%88%EC%95%84&amp;action=edit&amp;redlink=1" title="포풀로니아 (없는 문서)">포풀로니아</a>, <a href="/wiki/%EC%97%98%EB%B0%94_%EC%84%AC" title="엘바 섬">엘바</a>, <a class="new" href="/w/index.php?title=%ED%94%BC%EC%95%84%EB%85%B8%EC%82%AC&amp;action=edit&amp;redlink=1" title="피아노사 (없는 문서)">피아노사</a>, <a class="mw-disambig" href="/wiki/%EB%AA%AC%ED%85%8C%ED%81%AC%EB%A6%AC%EC%8A%A4%ED%86%A0" title="몬테크리스토">몬테크리스토</a>에 대한 지배권을 유지했다. 게라르도는 1404년에 그의 아들 <a href="/wiki/%EC%9D%B4%EC%95%84%EC%BD%94%ED%8F%AC_2%EC%84%B8_%EC%95%84%ED%94%BC%EC%95%84%EB%8B%88" title="이아코포 2세 아피아니">이아코포 2세</a>에게 물려주었는데, 게라르도가 사망 후(1441년), 이아코포의 어머니이자 섭정인 <a href="/wiki/%ED%8C%8C%EC%98%AC%EB%9D%BC_%EC%BD%9C%EB%A1%A0%EB%82%98" title="파올라 콜론나">파올라 콜론나</a>는 피옴비노를 위해 <a class="new" href="/w/index.php?title=%EC%97%90%EB%A7%88%EB%88%84%EC%97%98%EB%A0%88_%EC%95%84%ED%94%BC%EC%95%84%EB%8B%88&amp;action=edit&amp;redlink=1" title="에마누엘레 아피아니 (없는 문서)">에마누엘레 아피아니</a>를 격퇴시켜냈다. 1445년에 파올라 콜론나가 사망하고, 통치권은 이아코포 2세의 자매인 <a href="/wiki/%EC%B9%B4%ED%85%8C%EB%A6%AC%EB%82%98_%EC%95%84%ED%94%BC%EC%95%84%EB%8B%88" title="카테리나 아피아니">카테리나</a>에게 넘어갔지만, 그럼에도 실질적인 지배권은 그녀의 남편 <a class="new" href="/w/index.php?title=%EB%A6%AC%EB%82%A0%EB%8F%84_%EC%98%A4%EB%A5%B4%EC%8B%9C%EB%8B%88&amp;action=edit&amp;redlink=1" title="리날도 오르시니 (없는 문서)">리날도 오르시니</a>가 쥐고 있었다. 1450년에 리날도가 1451년에는 카테리나가 역병으로 사망하고, 에마누엘레는 피옴비노인들을 통해 피옴비노의 군주로 선출되었다.</p>
<p>1509년 <a href="/wiki/%EC%9D%B4%EC%95%84%EC%BD%94%ED%8F%AC_4%EC%84%B8_%EC%95%84%ED%94%BC%EC%95%84%EB%8B%88" title="이아코포 4세 아피아니">이아코포 4세</a>가 합스부르크 가문의 <a href="/wiki/%EB%A7%89%EC%8B%9C%EB%B0%80%EB%A6%AC%EC%95%88_1%EC%84%B8_(%EC%8B%A0%EC%84%B1_%EB%A1%9C%EB%A7%88_%EC%A0%9C%EA%B5%AD)" title="막시밀리안 1세 (신성 로마 제국)">막시밀리안 1세</a>를 통해 피옴비노 대공 작위를 획득했다. 그의 아들 <a class="new" href="/w/index.php?title=%EC%9D%B4%EC%95%84%EC%BD%94%ED%8F%AC_5%EC%84%B8_%EC%95%84%ED%94%BC%EC%95%84%EB%8B%88&amp;action=edit&amp;redlink=1" title="이아코포 5세 아피아니 (없는 문서)">이아코포 5세</a>는 <a href="/wiki/%EB%A9%94%EB%94%94%EC%B9%98_%EA%B0%80%EB%AC%B8" title="메디치 가문">메디치 가문</a>의 교황들인 <a href="/wiki/%EA%B5%90%ED%99%A9_%EB%A0%88%EC%98%A4_10%EC%84%B8" title="교황 레오 10세">교황 레오 10세</a>와 <a href="/wiki/%EA%B5%90%ED%99%A9_%ED%81%B4%EB%A0%88%EB%A9%98%EC%8A%A4_7%EC%84%B8" title="교황 클레멘스 7세">교황 클레멘스 7세</a>가 사망한 후, <a href="/wiki/%EB%A1%9C%EB%A0%8C%EC%B4%88_%EB%8D%B0_%EB%A9%94%EB%94%94%EC%B9%98" title="로렌초 데 메디치">로렌초 데 메디치</a>의 후손들인 두 자매와 혼인했다. 하지만 그의 아들 <a class="new" href="/w/index.php?title=%EC%9D%B4%EC%95%84%EC%BD%94%ED%8F%AC_6%EC%84%B8_%EC%95%84%ED%94%BC%EC%95%84%EB%8B%88&amp;action=edit&amp;redlink=1" title="이아코포 6세 아피아니 (없는 문서)">이아코포 6세</a>는 1548년 <a href="/wiki/%EC%B9%B4%EB%A5%BC_5%EC%84%B8" title="카를 5세">카를 5세</a>에 의해 피옴비노에서 쫒겨나고 그의 영지를 <a href="/wiki/%EC%BD%94%EC%8B%9C%EB%AA%A8_1%EC%84%B8_%EB%8D%B0_%EB%A9%94%EB%94%94%EC%B9%98" title="코시모 1세 데 메디치">코시모 1세 데 메디치</a>에게 맡겨버렸다. 이아코포는 1559년에 주권을 회복하였지만, 3년만에 폭동을 일으킨 시민들이 토스카나 대공국의 함대의 사령관으로 남아있던 <a class="new" href="/w/index.php?title=%EC%95%8C%EB%A0%88%EC%82%B0%EB%93%9C%EB%A1%9C_%EC%95%84%ED%94%BC%EC%95%84%EB%8B%88&amp;action=edit&amp;redlink=1" title="알레산드로 아피아니 (없는 문서)">알레산드로</a>와 싸우도록 강요하였다. 알레산드로는 황제로부터 통치의 합법화를 얻어냈지만, 통치 4년만에 대중들에 의해 살해당하고 만다.</p>

<p><a class="new" href="/w/index.php?title=%EC%9D%B4%EC%95%84%EC%BD%94%ED%8F%AC_7%EC%84%B8_%EC%95%84%ED%94%BC%EC%95%84%EB%8B%88&amp;action=edit&amp;redlink=1" title="이아코포 7세 아피아니 (없는 문서)">이아코포 7세</a>는 신성 로마 제국 황제 <a href="/wiki/%EB%A3%A8%EB%8F%8C%ED%94%84_2%EC%84%B8" title="루돌프 2세">루돌프 2세</a>에게서 공작 자위를 얻어, 알레산드로의 자리를 이어받는다. 이아코포가 후계자를 두지 못하고 사망하자, 피옴비노 대공국은 친척에게 잠시 통치권이 넘어갔었지만, 실질적으로는 <a class="new" href="/w/index.php?title=%EC%9D%B4%EC%82%AC%EB%B2%A8%EB%9D%BC_%EC%95%84%ED%94%BC%EC%95%84%EB%8B%88&amp;action=edit&amp;redlink=1" title="이사벨라 아피아니 (없는 문서)">이사벨라 아피아니</a>가 추방당하는 1628년까지 통치하였다가, 결국에는 1634년 신성 로마 제국 황제 <a href="/wiki/%ED%8E%98%EB%A5%B4%EB%94%94%EB%82%9C%ED%8A%B8_2%EC%84%B8" title="페르디난트 2세">페르디난트 2세</a>가 피옴비노를 이사벨라의 딸과 결혼한 <a class="new" href="/w/index.php?title=%EB%8B%88%EC%BD%9C%EB%A1%9C_%EB%A3%A8%EB%8F%84%EB%B9%84%EC%8B%9C&amp;action=edit&amp;redlink=1" title="니콜로 루도비시 (없는 문서)">니콜로 루도비시</a>에게 위임한다.</p>
