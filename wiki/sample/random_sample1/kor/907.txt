<p><b>디제이맥스 포터블 블랙 스퀘어</b>(DJMAX Portable BLACK SQUARE, DMP BS)는 <a href="/wiki/%EB%8C%80%ED%95%9C%EB%AF%BC%EA%B5%AD" title="대한민국">대한민국</a>의 게임 개발사인 <a href="/wiki/%ED%8E%9C%ED%83%80%EB%B9%84%EC%A0%84" title="펜타비전">펜타비전</a>에서 개발한 <a class="mw-redirect" href="/wiki/%EC%9D%8C%EC%95%85_%EA%B2%8C%EC%9E%84" title="음악 게임">음악 게임</a>이다. 메트로 프로젝트의 두 번째 작품중 두 번째 플랫폼으로 <a href="/wiki/%EC%98%A8%EB%9D%BC%EC%9D%B8_%EA%B2%8C%EC%9E%84" title="온라인 게임">온라인 게임</a>으로 만들어진 <a href="/wiki/%EB%94%94%EC%A0%9C%EC%9D%B4%EB%A7%A5%EC%8A%A4" title="디제이맥스">디제이맥스</a>를 <a href="/wiki/%ED%94%8C%EB%A0%88%EC%9D%B4%EC%8A%A4%ED%85%8C%EC%9D%B4%EC%85%98_%ED%8F%AC%ED%84%B0%EB%B8%94" title="플레이스테이션 포터블">플레이스테이션 포터블</a>용으로 이식한 <a class="mw-redirect" href="/wiki/%EB%94%94%EC%A0%9C%EC%9D%B4%EB%A7%A5%EC%8A%A4_%ED%8F%AC%ED%84%B0%EB%B8%94_2" title="디제이맥스 포터블 2">디제이맥스 포터블 2</a>의 후속작이다. 전작인 DMP2의 특징을 강화시킨 것을 내세우는 실질적인 DMP3라고 할 수 있는 후속작. <a href="/wiki/2008%EB%85%84" title="2008년">2008년</a> <a href="/wiki/9%EC%9B%94_2%EC%9D%BC" title="9월 2일">9월 2일</a> PV영상과 정보가 공개되었다. 대한민국에서 11월중 발매를 목표로 하였으나 <a class="mw-redirect" href="/wiki/%EB%94%94%EC%A0%9C%EC%9D%B4%EB%A7%A5%EC%8A%A4_%ED%8F%AC%ED%84%B0%EB%B8%94_%ED%81%B4%EB%9E%98%EC%A7%80%EC%BD%B0%EC%9D%B4_%EC%97%90%EB%94%94%EC%85%98" title="디제이맥스 포터블 클래지콰이 에디션">디제이맥스 포터블 클래지콰이 에디션</a>에서 많은 버그가 발견되어 이를 DMP BS에서 수정하기 위해 12월로 발매가 연기되었으며 12월 24일 대한민국에서 정식발매되었다.</p>

<h2><span class="mw-headline" id=".EB.94.94.EC.A0.9C.EC.9D.B4.EB.A7.A5.EC.8A.A4_.ED.8F.AC.ED.84.B0.EB.B8.942.EC.99.80.EC.9D.98_.EC.B0.A8.EC.9D.B4.EC.A0.90">디제이맥스 포터블2와의 차이점</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=DJMAX_Portable_BLACK_SQUARE&amp;action=edit&amp;section=2" title="부분 편집: 디제이맥스 포터블2와의 차이점">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>이전버전에서의 기능을 향상시켰으며, 익숙한 작곡자들이 이번 시리즈에서도 참여한다. 전작에서 추가/수정된 기능은 아래와 같다.</p>
<ul>

<li>8b모드가 6bFX모드로 이름이 변경됨</li>
<li>피버기능의 향상(5배와 7배를 자유롭게 선택 가능, Auto 5x 추가)</li>

<ul>
<li>2008년 <a href="/wiki/9%EC%9B%94_2%EC%9D%BC" title="9월 2일">9월 2일</a> - 루리웹 홈페이지를 통해 신작 메트로 프로젝트 <a class="mw-redirect" href="/wiki/%EB%94%94%EC%A0%9C%EC%9D%B4%EB%A7%A5%EC%8A%A4_%ED%8F%AC%ED%84%B0%EB%B8%94_%ED%81%B4%EB%9E%98%EC%A7%80%EC%BD%B0%EC%9D%B4_%EC%97%90%EB%94%94%EC%85%98" title="디제이맥스 포터블 클래지콰이 에디션">디제이맥스 포터블 클래지콰이 에디션</a>, 디제이맥스 포터블 블랙 스퀘어 PV 영상 공개</li>
<li>2008년 <a href="/wiki/11%EC%9B%94_13%EC%9D%BC" title="11월 13일">11월 13일</a> - <a class="mw-redirect" href="/wiki/%EB%94%94%EC%A0%9C%EC%9D%B4%EB%A7%A5%EC%8A%A4_%ED%8F%AC%ED%84%B0%EB%B8%94_%ED%81%B4%EB%9E%98%EC%A7%80%EC%BD%B0%EC%9D%B4_%EC%97%90%EB%94%94%EC%85%98" title="디제이맥스 포터블 클래지콰이 에디션">디제이맥스 포터블 클래지콰이 에디션</a> 정품 유저를 대상으로 FGT(Focus Group Test)실시</li>
<li>2008년 <a href="/wiki/12%EC%9B%94_2%EC%9D%BC" title="12월 2일">12월 2일</a> - 디제이맥스 포터블 블랙 스퀘어의 정식 정보가 공개되었다. <a class="external autonumber" href="http://ruliweb.empas.com/data/rulinews/read.htm?num=16628" rel="nofollow">[1]</a></li>

<li>2008년 <a href="/wiki/12%EC%9B%94_6%EC%9D%BC" title="12월 6일">12월 6일</a> - 디제이맥스 포터블 블랙 스퀘어의 한정판인 디제이맥스 포터블 블랙 스퀘어 ~Quattra~에 대한 예약 판매가 시행되었다.</li>
<li>2008년 <a href="/wiki/12%EC%9B%94_24%EC%9D%BC" title="12월 24일">12월 24일</a> - 디제이맥스 포터블 블랙 스퀘어가 발매되었다.</li>
<li>2008년 <a href="/wiki/12%EC%9B%94_26%EC%9D%BC" title="12월 26일">12월 26일</a> - 디제이맥스 포터블 블랙 스퀘어의 랭킹팩과 링크팩이 공개되었다.</li>
<li>2010년 <a href="/wiki/7%EC%9B%94_16%EC%9D%BC" title="7월 16일">7월 16일</a> - 디제이맥스 테크니카1의 플래티넘 크루 서비스가 종료되면서, 랭킹 서비스와 랭킹팩, 링크팩의 다운로드 서비스가 중단되었다.</li>
<li>2010년 <a href="/wiki/9%EC%9B%94" title="9월">9월</a> - 디제이맥스 포터블 클래지콰이 에디션과 디제이맥스 포터블 블랙스퀘어를 위한 플래티넘 크루 서비스가 재개되었다. <a class="external autonumber" href="http://djmax.platinumcrew.co.kr" rel="nofollow">[3]</a></li>
</ul>
