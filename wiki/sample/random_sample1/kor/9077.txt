<p><b>악셀 로랑 엔젤 랑베르 위첼</b> (<span style="font-size: smaller;"><a href="/wiki/%ED%94%84%EB%9E%91%EC%8A%A4%EC%96%B4" title="프랑스어">프랑스어</a>:</span> <span lang="fr" xml:lang="fr">Axel Laurent Angel Lambert Witsel</span>, <a href="/wiki/1989%EB%85%84" title="1989년">1989년</a> <a href="/wiki/1%EC%9B%94_12%EC%9D%BC" title="1월 12일">1월 12일</a>, <a href="/wiki/%EB%A6%AC%EC%97%90%EC%A3%BC" title="리에주">리에주</a> ~ )는 <a href="/wiki/%EB%B2%A8%EA%B8%B0%EC%97%90" title="벨기에">벨기에</a>의 축구 선수로, 현재 <a href="/wiki/%EC%A4%91%ED%99%94%EC%9D%B8%EB%AF%BC%EA%B3%B5%ED%99%94%EA%B5%AD" title="중화인민공화국">중화인민공화국</a>의 <a href="/wiki/%ED%86%88%EC%A7%84_%EC%B7%A8%EC%95%88%EC%A0%A0" title="톈진 취안젠">톈진 취안젠</a> 소속이다. 포지션은 <a href="/wiki/%EB%AF%B8%EB%93%9C%ED%95%84%EB%8D%94" title="미드필더">미드필더</a>이며, 그의 아버지는 <a href="/wiki/%EB%A7%88%EB%A5%B4%ED%8B%B0%EB%8B%88%ED%81%AC" title="마르티니크">마르티니크</a> 뿌리를 두고 있는 전 축구 선수이다.</p>
<p>비첼은 <a href="/wiki/2006%EB%85%84" title="2006년">2006년</a> <a href="/wiki/9%EC%9B%94_17%EC%9D%BC" title="9월 17일">9월 17일</a> <a href="/wiki/FC_%EB%B8%8C%EB%A4%BC%EC%85%80" title="FC 브뤼셀">FC 브뤼셀</a>과의 경기에서 89분, <a class="mw-redirect" href="/wiki/%EC%8A%A4%ED%85%8C%EB%B2%88_%EB%8D%B0%ED%8C%8C%EC%9A%B0%EB%A5%B4" title="스테번 데파우르">스테번 데파우르</a>와 교체되면서 17세의 나이에 데뷔전을 치렀다.</p>
<p>그는 2007-08 시즌, 18세의 어린 나이에 팀의 벨기에 리그 우승을 이끈 핵심 선수였으며, 일 년 동안 다른 벨기에의 재능들인 <a href="/wiki/%EB%A7%88%EB%A3%A8%EC%95%88_%ED%8E%A0%EB%9D%BC%EC%9D%B4%EB%8B%88" title="마루안 펠라이니">마루안 펠라이니</a>, <a class="new" href="/w/index.php?title=%EC%8A%A4%ED%8B%B0%EB%B8%90_%EB%8D%B0%ED%91%B8%EB%A5%B4&amp;action=edit&amp;redlink=1" title="스티븐 데푸르 (없는 문서)">스티븐 데푸르</a>와 함께 굉장히 어린 중원을 형성하였다. 다음 시즌에는 챔피언십 플레이오프에서 페널티킥으로 결승골을 성공시키며 팀의 리그 우승을 도왔다.</p>
<p><a href="/wiki/2009%EB%85%84" title="2009년">2009년</a> <a href="/wiki/8%EC%9B%94_30%EC%9D%BC" title="8월 30일">8월 30일</a>, 비첼은 <a href="/wiki/RSC_%EC%95%88%EB%8D%B0%EB%A5%BC%EB%A0%88%ED%9D%90%ED%8A%B8" title="RSC 안데를레흐트">안데를레흐트</a>와 스탕다르의 경기 도중, 폴란드의 <a href="/wiki/%EB%A7%88%EB%A5%B4%EC%B9%9C_%EB%B0%94%EC%8B%A4%EB%A0%88%ED%94%84%EC%8A%A4%ED%82%A4" title="마르친 바실레프스키">마르친 바실레프스키</a> (Marcin Wasilewski)의 다리에 위험한 태클로 치명적인 부상을 입혀 레드 카드를 받았으나, 주심의 판단에 항의하는 모습을 보였다. 그는 곧 사과했지만, 이 일로 인해 안데를레흐트, 폴란드 팬들과 언론 매체의 큰 비난의 대상이 되었다.</p>
<p>또한 그는 <a href="/wiki/2014%EB%85%84_FIFA_%EC%9B%94%EB%93%9C%EC%BB%B5" title="2014년 FIFA 월드컵">2014년 브라질월드컵</a>에 출전하여, 팀이 8강에 오르는데 큰 활약을 하였다.</p>
