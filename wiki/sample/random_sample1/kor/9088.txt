<h2><span class="mw-headline" id=".EC.84.B1.EC.9E.A5.EA.B3.BC.EC.A0.95">성장과정</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%95%88%EB%82%B4%EC%83%81&amp;action=edit&amp;section=1" title="부분 편집: 성장과정">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>안내상의 <a href="/wiki/%EC%95%84%EB%B2%84%EC%A7%80" title="아버지">아버지</a>가 <a href="/wiki/%EB%8C%80%EA%B5%AC%EA%B4%91%EC%97%AD%EC%8B%9C" title="대구광역시">대구광역시</a>에서 염색 공장을 하다가 <a class="mw-redirect" href="/wiki/%EB%B3%B4%EC%A6%9D" title="보증">보증</a>을 잘못 서서 집이 망해 서울로 상경하여 <a href="/wiki/%EC%B2%AD%EA%B3%84%EC%B2%9C" title="청계천">청계천</a> <a href="/wiki/%ED%8C%90%EC%9E%90%EC%B4%8C" title="판자촌">판자촌</a>에서 서울 생활을 시작했다. 초등학교 1학년 때부터 수업도 들어가지 않고 학우들을 괴롭히는 등, 초등학교 3학년 부터는 <a href="/wiki/%EB%8B%B4%EB%B0%B0" title="담배">담배</a>를 피우고 <a class="mw-redirect" href="/wiki/%EB%8F%84%EB%91%91%EC%A7%88" title="도둑질">도둑질</a>을 했다. 이 때 안내상을 지도하던 선생님이 <a href="/wiki/%EC%95%84%EC%9D%B4%EC%8A%A4%ED%81%AC%EB%A6%BC" title="아이스크림">아이스크림</a>을 사준다고 하여 <a href="/wiki/%EA%B0%9C%EC%8B%A0%EA%B5%90" title="개신교">개신교</a> <a href="/wiki/%EA%B5%90%ED%9A%8C" title="교회">교회</a>를 간 것이 계기가 되어 종교에 귀의한 후에 악행을 멈추었다고 회고했다<sup class="reference" id="cite_ref-1"><a href="#cite_note-1">[1]</a></sup><sup class="reference" id="cite_ref-2"><a href="#cite_note-2">[2]</a></sup>.</p>
<p><a href="/wiki/%EC%97%B0%EC%84%B8%EB%8C%80%ED%95%99%EA%B5%90" title="연세대학교">연세대학교</a> 신학과에 입학하였으며, 재학 중 학생운동을 주도했다. <a href="/wiki/1988%EB%85%84" title="1988년">1988년</a> <a href="/wiki/2%EC%9B%94_26%EC%9D%BC" title="2월 26일">2월 26일</a> <a href="/wiki/%EC%A0%84%EB%9D%BC%EB%82%A8%EB%8F%84" title="전라남도">전라남도</a> <a href="/wiki/%EA%B4%91%EC%A3%BC%EA%B4%91%EC%97%AD%EC%8B%9C" title="광주광역시">광주시</a> (현재 광주광역시) <a href="/wiki/%EB%AF%B8%EA%B5%AD" title="미국">미국</a> 문화원 내 도서관 진열장에 사제 시한폭탄을 설치한 혐의가 드러나 총포 도검 화약류 단속법 위반, 폭력행위 등 처벌에 관한 법률위반, <a href="/wiki/%EA%B5%AD%EA%B0%80%EB%B3%B4%EC%95%88%EB%B2%95" title="국가보안법">국가보안법</a> 위반혐의로 구속됐다. 8개월 동안 수감된 후 집행유예로 석방되었다.</p>
<p>출소 후에는 학생운동을 그만두고 연극 쪽으로 발길을 돌렸다<sup class="reference" id="cite_ref-3"><a href="#cite_note-3">[3]</a></sup>.</p>
<h2><span class="mw-headline" id=".ED.8A.B9.EC.9D.B4.EC.82.AC.ED.95.AD">특이사항</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%95%88%EB%82%B4%EC%83%81&amp;action=edit&amp;section=2" title="부분 편집: 특이사항">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<ul>
<li>배우 <a href="/wiki/%EC%84%A4%EA%B2%BD%EA%B5%AC" title="설경구">설경구</a>의 전처는 안내상의 여동생이다. 또한 <a href="/wiki/%ED%95%9C%EC%96%91%EB%8C%80%ED%95%99%EA%B5%90" title="한양대학교">한양대학교</a> 연극영화학과 출신인 설경구의 소개를 받아 극단 한양레퍼토리에 들어갔고, <a href="/wiki/%EC%9D%B4%EB%AC%B8%EC%8B%9D" title="이문식">이문식</a>, <a href="/wiki/%EA%B6%8C%ED%95%B4%ED%9A%A8" title="권해효">권해효</a>, <a href="/wiki/%EC%9C%A0%EC%98%A4%EC%84%B1" title="유오성">유오성</a> 등이 당시 연극을 함께한 동료 배우들이다.<sup class="reference" id="cite_ref-4"><a href="#cite_note-4">[4]</a></sup></li>
<li>고교 선배로는 배우 <a href="/wiki/%EC%B5%9C%EC%88%98%EC%A2%85" title="최수종">최수종</a> 등이 있으며 대학 동창으로는 배우 <a href="/wiki/%EC%9A%B0%ED%98%84_(%EB%B0%B0%EC%9A%B0)" title="우현 (배우)">우현</a>, <a href="/wiki/%EC%9D%B4%EB%8C%80%EC%97%B0" title="이대연">이대연</a>, 영어 강사 <a href="/wiki/%EB%AC%B8%EB%8B%A8%EC%97%B4" title="문단열">문단열</a> 등이 있다.</li>
</ul>
<h2><span class="mw-headline" id=".EC.B6.9C.EC.97.B0.EC.9E.91">출연작</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%95%88%EB%82%B4%EC%83%81&amp;action=edit&amp;section=3" title="부분 편집: 출연작">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<h3><span class="mw-headline" id=".EC.98.81.ED.99.94">영화</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%95%88%EB%82%B4%EC%83%81&amp;action=edit&amp;section=4" title="부분 편집: 영화">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
