<p><b>안달루시아 구어체 아랍어</b>는 <a href="/wiki/%EC%95%8C%EC%95%88%EB%8B%AC%EB%A3%A8%EC%8A%A4" title="알안달루스">알안달루스 지역</a>에서 사용되었던 <a href="/wiki/%EC%95%84%EB%9E%8D%EC%96%B4" title="아랍어">아랍어</a>의 방언이었다. 오늘날 <a class="new" href="/w/index.php?title=%EC%95%88%EB%8B%AC%EB%A3%A8%EC%8B%9C%EC%95%84%EC%9D%98_%EC%9D%8C%EC%95%85&amp;action=edit&amp;redlink=1" title="안달루시아의 음악 (없는 문서)">안달루시아 지방의 음악</a>에 영향을 미쳤으나, <a href="/wiki/%EC%8A%A4%ED%8E%98%EC%9D%B8" title="스페인">스페인</a>에서는 일찍이 <a class="mw-redirect" href="/wiki/%EC%82%AC%EB%A9%B8%EB%90%9C_%EC%96%B8%EC%96%B4" title="사멸된 언어">사멸</a>되었다. 하지만 지금도 <a href="/wiki/%ED%85%8C%ED%88%AC%EC%95%88" title="테투안">테투안</a>, <a href="/wiki/%ED%8E%98%EC%8A%A4" title="페스">페스</a>, <a href="/wiki/%EB%9D%BC%EB%B0%94%ED%8A%B8" title="라바트">라바트</a>, <a href="/wiki/%ED%83%95%ED%97%A4%EB%A5%B4" title="탕헤르">탕헤르</a>와 <a class="new" href="/w/index.php?title=%EC%85%B0%EB%A5%B4%EC%85%B8&amp;action=edit&amp;redlink=1" title="셰르셸 (없는 문서)">셰르셸</a>에서는 조금씩 사용된다.</p>
<p>이외에 <a href="/wiki/%EB%AA%A8%EC%9E%90%EB%9D%BC%EB%B8%8C%EC%96%B4" title="모자라브어">모자라브어</a>, <a href="/wiki/%EC%8A%A4%ED%8E%98%EC%9D%B8%EC%96%B4" title="스페인어">스페인어</a>, <a href="/wiki/%EC%B9%B4%ED%83%88%EB%A3%A8%EB%83%90%EC%96%B4" title="카탈루냐어">카탈루냐어</a>, <a href="/wiki/%ED%8F%AC%EB%A5%B4%ED%88%AC%EA%B0%88%EC%96%B4" title="포르투갈어">포르투갈어</a>, <a class="new" href="/w/index.php?title=%EB%AA%A8%EB%A1%9C%EC%BD%94_%EA%B5%AC%EC%96%B4%EC%B2%B4_%EC%95%84%EB%9E%8D%EC%96%B4&amp;action=edit&amp;redlink=1" title="모로코 구어체 아랍어 (없는 문서)">모로코 구어체 아랍어</a> 등에서 차용한 어휘가 많다.</p>
