<p><b>안덕해</b>(<small><a href="/wiki/%EC%A4%91%EA%B5%AD%EC%96%B4" title="중국어">중국어</a></small> <small><a href="/wiki/%EC%A0%95%EC%B2%B4%EC%9E%90" title="정체자">정체</a>:</small> <span lang="zh-hant" xml:lang="zh-hant">安德海</span>,  <small>안더하이<sup>[<a href="/wiki/%EC%9C%84%ED%82%A4%EB%B0%B1%EA%B3%BC:%EC%A4%91%EA%B5%AD%EC%96%B4%EC%9D%98_%ED%95%9C%EA%B8%80_%ED%91%9C%EA%B8%B0" title="위키백과:중국어의 한글 표기">*</a>]</sup></small>, 안더하이, <a href="/wiki/1844%EB%85%84" title="1844년">1844년</a> ~ <a href="/wiki/1869%EB%85%84" title="1869년">1869년</a> <a href="/wiki/9%EC%9B%94_12%EC%9D%BC" title="9월 12일">9월 12일</a>)은 <a href="/wiki/%EC%A4%91%EA%B5%AD" title="중국">중국</a> <a href="/wiki/%EC%B2%AD%EB%82%98%EB%9D%BC" title="청나라">청나라</a>의 <a href="/wiki/%ED%99%98%EA%B4%80" title="환관">환관(宦官)</a>이다.</p>
<h2><span class="mw-headline" id=".EC.83.9D.EC.95.A0">생애</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%95%88%EB%8D%95%ED%95%B4&amp;action=edit&amp;section=1" title="부분 편집: 생애">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p><a href="/wiki/1849%EB%85%84" title="1849년">1849년</a> <a href="/wiki/%ED%99%98%EA%B4%80" title="환관">환관</a>(宦官)으로 첫 입문하여 이후 <a href="/wiki/%ED%95%A8%ED%92%8D%EC%A0%9C" title="함풍제">함풍제</a>(咸豊帝)와 그의 <a href="/wiki/%ED%9B%84%EA%B6%81" title="후궁">후궁</a> <a href="/wiki/%EC%84%9C%ED%83%9C%ED%9B%84" title="서태후">서태후</a>(西太后)를 모두 주인이자 주군으로 섬기었으나 <a href="/wiki/1869%EB%85%84" title="1869년">1869년</a> 독직죄가 적용되어 끝끝내 <a href="/wiki/%EC%84%9C%ED%83%9C%ED%9B%84" title="서태후">서태후</a>의 압력으로 파직에 처해지고 <a href="/wiki/%EC%B2%AD%EB%82%98%EB%9D%BC" title="청나라">청나라</a> <a href="/wiki/%EC%82%B0%EB%91%A5_%EC%84%B1" title="산둥 성">산둥 성</a> <a href="/wiki/%EC%A7%80%EB%82%9C_%EC%8B%9C" title="지난 시">지난</a>에 유배되어 그곳에서 <a href="/wiki/%EC%B0%B8%EC%88%98" title="참수">참수</a> <a href="/wiki/%EC%82%AC%ED%98%95" title="사형">사형</a> 집행되었다.</p>
<h2><span class="mw-headline" id=".EB.AC.B8.ED.95.99_.EC.9E.91.ED.92.88.EC.97.90.EC.84.9C_.EB.AC.98.EC.82.AC_.ED.91.9C.ED.98.84.EB.90.9C_.EC.95.88.EB.8D.95.ED.95.B4">문학 작품에서 묘사 표현된 안덕해</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%95%88%EB%8D%95%ED%95%B4&amp;action=edit&amp;section=2" title="부분 편집: 문학 작품에서 묘사 표현된 안덕해">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p><a href="/wiki/%EC%95%84%EC%82%AC%EB%8B%A4_%EC%A7%80%EB%A1%9C" title="아사다 지로">아사다 지로</a>의 장편역사소설 《창궁의 묘성》에서 주인공 이춘운의 스승으로 등장한다. 소설 속에서 안덕해는 사형 직전에 뇌물을 써서 간신히 목숨을 건진 뒤에 환관들이 모여 사는 곳에서 구걸을 하며 사는 것으로 나온다.</p>
