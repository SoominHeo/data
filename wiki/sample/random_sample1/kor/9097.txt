<p><b>안데르 에레라 아구에라</b> (<span style="font-size: smaller;"><a href="/wiki/%EC%8A%A4%ED%8E%98%EC%9D%B8%EC%96%B4" title="스페인어">스페인어</a>:</span> <span lang="es" xml:lang="es">Ander Herrera Agüera</span>, <a href="/wiki/1989%EB%85%84" title="1989년">1989년</a> <a href="/wiki/8%EC%9B%94_14%EC%9D%BC" title="8월 14일">8월 14일</a> ~ )는 <a href="/wiki/%EC%8A%A4%ED%8E%98%EC%9D%B8" title="스페인">스페인</a>의 <a href="/wiki/%EC%B6%95%EA%B5%AC_%EC%84%A0%EC%88%98" title="축구 선수">축구 선수</a>로, 현재 <a class="mw-redirect" href="/wiki/%EB%A7%A8%EC%B2%B4%EC%8A%A4%ED%84%B0_%EC%9C%A0%EB%82%98%EC%9D%B4%ED%8B%B0%EB%93%9C" title="맨체스터 유나이티드">맨체스터 유나이티드</a>에서 <a class="mw-redirect" href="/wiki/%EC%A4%91%EC%95%99_%EB%AF%B8%EB%93%9C%ED%95%84%EB%8D%94" title="중앙 미드필더">중앙 미드필더</a>로 뛰고 있으며 장차 <a href="/wiki/%ED%8F%B4_%EC%8A%A4%EC%BD%9C%EC%8A%A4" title="폴 스콜스">폴 스콜스</a>의 후계자로 많은 기대를 모으고 있다.</p>
<p>에레라는 <a href="/wiki/2014%EB%85%84" title="2014년">2014년</a> <a href="/wiki/7%EC%9B%94_24%EC%9D%BC" title="7월 24일">7월 24일</a>에 열린 <a class="mw-redirect" href="/wiki/LA_%EA%B0%A4%EB%9F%AD%EC%8B%9C" title="LA 갤럭시">LA 갤럭시</a>와의 프리시즌 친선 경기에 출전해 데뷔전을 치렀는데, 중원에서 맹활약을 펼치며 경기 최우수 선수로 선정됐다.</p>
<p>이날 <a href="/wiki/%EB%A7%A8%EC%B2%B4%EC%8A%A4%ED%84%B0_%EC%9C%A0%EB%82%98%EC%9D%B4%ED%8B%B0%EB%93%9C_FC" title="맨체스터 유나이티드 FC">맨체스터 유나이티드</a>는 <a class="mw-redirect" href="/wiki/LA_%EA%B0%A4%EB%9F%AD%EC%8B%9C" title="LA 갤럭시">LA 갤럭시</a>와의 친선 경기서 7-0 완승을 거뒀다.</p>
<p>그리고 <a href="/wiki/2014%EB%85%84" title="2014년">2014년</a> <a href="/wiki/8%EC%9B%94_16%EC%9D%BC" title="8월 16일">8월 16일</a>, <a href="/wiki/%EC%8A%A4%EC%99%84%EC%A7%80_%EC%8B%9C%ED%8B%B0_AFC" title="스완지 시티 AFC">스완지 시티</a>와 개막전에서 공식 리그 데뷔전을 치렀으며, <a href="/wiki/9%EC%9B%94_15%EC%9D%BC" title="9월 15일">9월 15일</a>에는 <a href="/wiki/%ED%80%B8%EC%8A%A4_%ED%8C%8C%ED%81%AC_%EB%A0%88%EC%9D%B8%EC%A0%80%EC%8A%A4_FC" title="퀸스 파크 레인저스 FC">퀸스 파크 레인저스</a>와의 홈 경기에서 EPL 데뷔골을 기록하였다.</p>
<p>2016년 10월 3일(한국시각)에는 <a href="/wiki/%ED%95%98%EB%B9%84_%EB%A7%88%EB%A5%B4%ED%8B%B0%EB%84%A4%EC%8A%A4" title="하비 마르티네스">하비 마르티네스</a>가 부상으로 스페인 국가대표팀에서 이탈하자, 그를 대신하여 에레라가 대표팀에 합류하였다. 이로써 에레라는 이번 스페인 국가대표팀에 처음으로 발탁되는 영광을 안았다.</p>
<p>2016년 10월 18일(한국시각)에는 <a href="/wiki/%EB%85%B8%EC%8A%A4%EC%9B%A8%EC%8A%A4%ED%8A%B8_%EB%8D%94%EB%B9%84" title="노스웨스트 더비">노스웨스트 더비</a>에서 11번의 가로채기와 3차례의 걷어내기 등으로 수비적인 임무를 완벽하게 수행하였으며 펑점 9.69점을 받았다.</p>
<p>2016년 11월 16일(한국시각)에는 스페인과 잉글랜드와의 평가전에 후반 교체투입 되어 A매치 데뷔전을 치루었다. 경기는 2:2 무승부로 종료되었다.</p>
