<p><b>안드레 미야토비치</b> (Andre Mijatović, <a href="/wiki/1979%EB%85%84" title="1979년">1979년</a> <a href="/wiki/12%EC%9B%94_3%EC%9D%BC" title="12월 3일">12월 3일</a>, <a href="/wiki/%EB%A6%AC%EC%98%88%EC%B9%B4" title="리예카">리예카</a> ~) 는 <a href="/wiki/%ED%81%AC%EB%A1%9C%EC%95%84%ED%8B%B0%EC%95%84" title="크로아티아">크로아티아</a>의 축구 선수로, 현재 <a class="mw-redirect" href="/wiki/2._%ED%91%B8%EC%8A%A4%EB%B0%9C-%EB%B6%84%EB%8D%B0%EC%8A%A4%EB%A6%AC%EA%B0%80" title="2. 푸스발-분데스리가">2. 푸스발-분데스리가</a>의 <a href="/wiki/FC_%EC%9E%89%EA%B3%A8%EC%8A%88%ED%83%80%ED%8A%B8_04" title="FC 잉골슈타트 04">FC 잉골슈타트 04</a>에서 <a href="/wiki/%EC%88%98%EB%B9%84%EC%88%98" title="수비수">수비수</a>로 활약하고 있다.</p>
<p>그는 <a href="/wiki/%EB%A6%AC%EC%98%88%EC%B9%B4" title="리예카">리예카</a> 출신으로, 지역 클럽 HNK 리예카에서 축구를 시작하였고, <a href="/wiki/1998%EB%85%84" title="1998년">1998년</a> 팀의 1군에 합류하였다. 그는 5시즌동안 클럽과 함께하였으며, 5시즌 모두 주전으로 뛰며, <a href="/wiki/%ED%94%84%EB%A5%B4%EB%B0%94_HNL" title="프르바 HNL">리그</a>에서 총 122경기에 출장하여 10골을 득점하였다. 그는 또한 <a href="/wiki/1999%EB%85%84" title="1999년">1999년</a> <a href="/wiki/UEFA_%EC%B1%94%ED%94%BC%EC%96%B8%EC%8A%A4%EB%A6%AC%EA%B7%B8" title="UEFA 챔피언스리그">UEFA 챔피언스리그</a>와 2000년대 초반 <a class="mw-redirect" href="/wiki/UEFA_%EC%BB%B5" title="UEFA 컵">UEFA 컵</a>에 출전하여 10경기 출장, 3골을 기록하였다.</p>
<p>그가 리예카에 있는 동안, 미야토비치는 <a href="/wiki/%ED%81%AC%EB%A1%9C%EC%95%84%ED%8B%B0%EC%95%84_%EC%B6%95%EA%B5%AC_%EA%B5%AD%EA%B0%80%EB%8C%80%ED%91%9C%ED%8C%80" title="크로아티아 축구 국가대표팀">크로아티아 U-21 국가대표팀</a>에 발탁되었고, <a href="/wiki/1999%EB%85%84" title="1999년">1999년</a> <a href="/wiki/4%EC%9B%94" title="4월">4월</a>과 <a href="/wiki/2001%EB%85%84" title="2001년">2001년</a> <a href="/wiki/6%EC%9B%94" title="6월">6월</a> 사이, U-21 대표팀 소속으로 10경기 출장하였다. 그는 <a href="/wiki/1999%EB%85%84" title="1999년">1999년</a>에 <a href="/wiki/%EB%82%98%EC%9D%B4%EC%A7%80%EB%A6%AC%EC%95%84" title="나이지리아">나이지리아</a>에서 열린 청소년 챔피언쉽에도 출전하여, 4경기에 풀타임으로 출장하였다. 크로아티아는 16강에서 탈락하였다. 그는 2002년 UEFA U-21 챔피언쉽에도 참여하여 4경기 출장하였다.</p>
<p><a href="/wiki/2003%EB%85%84" title="2003년">2003년</a> 여름, 미야토비치는 리예카를 떠나 <a class="mw-redirect" href="/wiki/NK_%EB%94%94%EB%82%98%EB%AA%A8_%EC%9E%90%EA%B7%B8%EB%A0%88%EB%B8%8C" title="NK 디나모 자그레브">NK 디나모 자그레브</a>로 이적하여 2시즌동안 주전으로 활약하며, 리그에서 51경기에 출장하여 1골을 기록하였다. 그는 또한 <a href="/wiki/2004%EB%85%84" title="2004년">2004년</a>에 디나모 소속으로 크로아티아 컵을 우승하였고, 이어지는 시즌에는 디나모 소속으로 <a class="mw-redirect" href="/wiki/UEFA_%EC%BB%B5" title="UEFA 컵">UEFA 컵</a>에 참가하여, 총 8경기에 출장하여, <a href="/wiki/2004%EB%85%84" title="2004년">2004년</a> <a href="/wiki/11%EC%9B%94_4%EC%9D%BC" title="11월 4일">11월 4일</a> KSK 베버렌전에서 득점하였고, 팀은 6-1로 승리하였다.</p>
<p><a href="/wiki/2005%EB%85%84" title="2005년">2005년</a> 여름, 미야토비치는 디나모를 떠나 <a href="/wiki/%EB%8F%85%EC%9D%BC" title="독일">독일</a>의 <a class="mw-redirect" href="/wiki/2._%ED%91%B8%EC%8A%A4%EB%B3%BC-%EB%B6%84%EB%8D%B0%EC%8A%A4%EB%A6%AC%EA%B0%80" title="2. 푸스볼-분데스리가">2 분데스리가</a>에 속해 있는 <a href="/wiki/SpVgg_%EA%B7%B8%EB%A1%9C%EC%9D%B4%ED%84%B0_%ED%93%8C%EB%A5%B4%ED%8A%B8" title="SpVgg 그로이터 퓌르트">SpVgg 그로이터 퓌르트</a>로 이적하여 팀의 주전이 되었다. 그의 <a class="mw-redirect" href="/wiki/2._%ED%91%B8%EC%8A%A4%EB%B3%BC-%EB%B6%84%EB%8D%B0%EC%8A%A4%EB%A6%AC%EA%B0%80" title="2. 푸스볼-분데스리가">2 분데스리가</a> 데뷔전은 <a href="/wiki/2005%EB%85%84" title="2005년">2005년</a> <a href="/wiki/8%EC%9B%94_6%EC%9D%BC" title="8월 6일">8월 6일</a>, <a class="mw-redirect" href="/wiki/%EB%94%94%EB%82%98%EB%AA%A8_%EB%93%9C%EB%A0%88%EC%8A%A4%EB%8D%B4" title="디나모 드레스덴">디나모 드레스덴</a> 전이었지만, 경고누적으로 퇴장당하면서 씁쓸하게 데뷔전을 치루었다. 그는 <a class="mw-redirect" href="/wiki/2._%ED%91%B8%EC%8A%A4%EB%B3%BC-%EB%B6%84%EB%8D%B0%EC%8A%A4%EB%A6%AC%EA%B0%80" title="2. 푸스볼-분데스리가">2 분데스리가</a>의 첫 시즌에 34경기 중 30경기에 출장하였으며, 팀을 위해 리그에서 3골을 넣었다. 이들 중 첫 번째 골은 <a href="/wiki/2006%EB%85%84" title="2006년">2006년</a> <a href="/wiki/3%EC%9B%94_17%EC%9D%BC" title="3월 17일">3월 17일</a>, 2-2 무승부로 종료된 <a href="/wiki/SV_%EB%B0%94%EC%BB%A4_%EB%B6%80%EB%A5%B4%ED%81%AC%ED%95%98%EC%9A%B0%EC%A0%A0" title="SV 바커 부르크하우젠">SV 바커 부르크하우젠</a> 전이었다.</p>
<p><a href="/wiki/2007%EB%85%84" title="2007년">2007년</a> <a href="/wiki/7%EC%9B%94" title="7월">7월</a>, 그는 <a class="mw-redirect" href="/wiki/%ED%91%B8%EC%8A%A4%EB%B3%BC-%EB%B6%84%EB%8D%B0%EC%8A%A4%EB%A6%AC%EA%B0%80" title="푸스볼-분데스리가">1 분데스리가</a>에 속해 있던 <a href="/wiki/%EC%95%84%EB%A5%B4%EB%AF%B8%EB%8B%88%EC%95%84_%EB%B9%8C%EB%A0%88%ED%8E%A0%ED%8A%B8" title="아르미니아 빌레펠트">아르미니아 빌레펠트</a>로 이적하여, 팀의 주전이 되었다. 빌레펠트와 3년을 보낸 후, 그는 아르미니아를 떠나 2009-10 시즌 종료후 강등된 <a class="mw-redirect" href="/wiki/%ED%97%A4%EB%A5%B4%ED%83%80_BSC_%EB%B2%A0%EB%A5%BC%EB%A6%B0" title="헤르타 BSC 베를린">헤르타 BSC 베를린</a>으로 이적하였다.</p>
