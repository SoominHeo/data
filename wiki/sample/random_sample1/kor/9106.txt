<p><b>안드레아스 힝켈</b> (Andreas Hinkel, <a href="/wiki/1982%EB%85%84" title="1982년">1982년</a> <a href="/wiki/3%EC%9B%94_26%EC%9D%BC" title="3월 26일">3월 26일</a>, <a href="/wiki/%EB%B0%94%EB%8D%B4%EB%B7%94%EB%A5%B4%ED%85%9C%EB%B2%A0%EB%A5%B4%ED%81%AC_%EC%A3%BC" title="바덴뷔르템베르크 주">바덴뷔르템베르크 주</a>, <a class="new" href="/w/index.php?title=%EB%B0%94%ED%81%AC%EB%82%AD&amp;action=edit&amp;redlink=1" title="바크낭 (없는 문서)">바크낭</a> ~ ) 은 은퇴한 <a href="/wiki/%EB%8F%85%EC%9D%BC" title="독일">독일</a>의 <a href="/wiki/%EC%B6%95%EA%B5%AC" title="축구">축구</a> 선수이다. 그는 현역 시절 <a href="/wiki/%EC%88%98%EB%B9%84%EC%88%98" title="수비수">라이트 백</a>으로 활약하였으며, 21차례 <a href="/wiki/%EB%8F%85%EC%9D%BC_%EC%B6%95%EA%B5%AC_%EA%B5%AD%EA%B0%80%EB%8C%80%ED%91%9C%ED%8C%80" title="독일 축구 국가대표팀">독일 축구 국가대표팀</a> 경기에 출전하였다.</p>
<p>그는 <a href="/wiki/%ED%91%B8%EC%8A%A4%EB%B0%9C-%EB%B6%84%EB%8D%B0%EC%8A%A4%EB%A6%AC%EA%B0%80" title="푸스발-분데스리가">푸스발-분데스리가</a>의 <a href="/wiki/VfB_%EC%8A%88%ED%88%AC%ED%8A%B8%EA%B0%80%EB%A5%B4%ED%8A%B8" title="VfB 슈투트가르트">VfB 슈투트가르트</a> 소속으로 2000년에 데뷔하였고, 2006년에는 <a class="mw-redirect" href="/wiki/%ED%94%84%EB%A6%AC%EB%A9%94%EB%9D%BC%EB%A6%AC%EA%B0%80" title="프리메라리가">라 리가</a>의 <a href="/wiki/%EC%84%B8%EB%B9%84%EC%95%BC_FC" title="세비야 FC">세비야 FC</a>로 이적하였다. 2008년 1월 그는 <a href="/wiki/%EC%8A%A4%EC%BD%94%ED%8B%80%EB%9E%9C%EB%93%9C_%ED%94%84%EB%A6%AC%EB%AF%B8%EC%96%B4%EB%A6%AC%EA%B7%B8" title="스코틀랜드 프리미어리그">스코틀랜드 프리미어리그</a>의 <a href="/wiki/%EC%85%80%ED%8B%B1_FC" title="셀틱 FC">셀틱 FC</a>와 £1.9M의 계약을 체결하고 팀의 일원이 되었다. 2011년 여름, 그는 <a href="/wiki/%EC%85%80%ED%8B%B1_FC" title="셀틱 FC">셀틱 FC</a>와의 계약이 만료된 후, <a href="/wiki/%ED%91%B8%EC%8A%A4%EB%B0%9C-%EB%B6%84%EB%8D%B0%EC%8A%A4%EB%A6%AC%EA%B0%80" title="푸스발-분데스리가">푸스발-분데스리가</a>의 <a href="/wiki/SC_%ED%94%84%EB%9D%BC%EC%9D%B4%EB%B6%80%EB%A5%B4%ED%81%AC" title="SC 프라이부르크">SC 프라이부르크</a>에 입단하였다. 그는 십자인대 파열로 2010-11 시즌을 통째로 결정하였다.</p>

<h2><span class="mw-headline" id=".ED.81.B4.EB.9F.BD_.EA.B2.BD.EB.A0.A5">클럽 경력</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%95%88%EB%93%9C%EB%A0%88%EC%95%84%EC%8A%A4_%ED%9E%9D%EC%BC%88&amp;action=edit&amp;section=1" title="부분 편집: 클럽 경력">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<h3><span class="mw-headline" id="VfB_.EC.8A.88.ED.88.AC.ED.8A.B8.EA.B0.80.EB.A5.B4.ED.8A.B8">VfB 슈투트가르트</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%95%88%EB%93%9C%EB%A0%88%EC%95%84%EC%8A%A4_%ED%9E%9D%EC%BC%88&amp;action=edit&amp;section=2" title="부분 편집: VfB 슈투트가르트">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p><a href="/wiki/%ED%91%B8%EC%8A%A4%EB%B0%9C-%EB%B6%84%EB%8D%B0%EC%8A%A4%EB%A6%AC%EA%B0%80_2002-03" title="푸스발-분데스리가 2002-03">2002-03</a> 시즌, <a href="/wiki/VfB_%EC%8A%88%ED%88%AC%ED%8A%B8%EA%B0%80%EB%A5%B4%ED%8A%B8" title="VfB 슈투트가르트">VfB 슈투트가르트</a>가 리그 준우승을 거둔 해에, 힝켈은 주전을 꿰찼고, 이듬해 <a href="/wiki/UEFA_%EC%B1%94%ED%94%BC%EC%96%B8%EC%8A%A4%EB%A6%AC%EA%B7%B8" title="UEFA 챔피언스리그">UEFA 챔피언스리그</a>에서 프로생활 시작 후로는 첫 출전을 경험하였다. 2004년 3월, 그러나, 그는 <a href="/wiki/%EB%AC%B4%EB%A6%8E" title="무릎">무릎</a> <a href="/wiki/%EC%9D%B8%EB%8C%80" title="인대">인대</a> 파열로 아웃되었다.</p>
<h3><span class="mw-headline" id=".EC.84.B8.EB.B9.84.EC.95.BC_FC">세비야 FC</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%95%88%EB%93%9C%EB%A0%88%EC%95%84%EC%8A%A4_%ED%9E%9D%EC%BC%88&amp;action=edit&amp;section=3" title="부분 편집: 세비야 FC">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>2006년 6월 23일, 힝켈은 <a href="/wiki/UEFA_%EC%9C%A0%EB%A1%9C%ED%8C%8C%EB%A6%AC%EA%B7%B8" title="UEFA 유로파리그">UEFA컵</a> 우승을 거둔 <a href="/wiki/%EC%84%B8%EB%B9%84%EC%95%BC_FC" title="세비야 FC">세비야 FC</a>와 계약을 체결하였다.<sup class="reference" id="cite_ref-1"><a href="#cite_note-1">[1]</a></sup> 이적료는 €4M으로 책정되었다. 그러나, 그는 세비야의 주전 라이트백이었던 <a class="mw-redirect" href="/wiki/%EB%8B%A4%EB%8B%88%EC%97%90%EC%9A%B0_%EC%95%84%EC%9A%B0%EB%B2%A0%EC%8A%A4" title="다니에우 아우베스">다니에우 아우베스</a>에 밀려 많은 출장을 기록하지 못하였다.<sup class="reference" id="cite_ref-CelticSign_2-0"><a href="#cite_note-CelticSign-2">[2]</a></sup></p>
<h3><span class="mw-headline" id=".EC.85.80.ED.8B.B1_FC">셀틱 FC</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%95%88%EB%93%9C%EB%A0%88%EC%95%84%EC%8A%A4_%ED%9E%9D%EC%BC%88&amp;action=edit&amp;section=4" title="부분 편집: 셀틱 FC">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>2008년 1월 4일, 힌켈은 <a href="/wiki/%EC%85%80%ED%8B%B1_FC" title="셀틱 FC">셀틱 FC</a>에 £1.9M의 계약을 체결하였다.<sup class="reference" id="cite_ref-CelticSign_2-1"><a href="#cite_note-CelticSign-2">[2]</a></sup> 힝켈은 계약 8일 후에 3-0으로 승리한 <a href="/wiki/%EC%8A%A4%ED%84%B8%EB%A7%81_%EC%95%A8%EB%B9%84%EC%96%B8_FC" title="스털링 앨비언 FC">스털링 앨비언 FC</a>와의 <a href="/wiki/%EC%8A%A4%EC%BD%94%ED%8B%80%EB%9E%9C%EB%93%9C_%EC%BB%B5" title="스코틀랜드 컵">스코틀랜드 컵</a>에서 스코틀랜드 무대 데뷔를 하였고, 그로부터 1달 뒤에는 3-0으로 이긴 <a href="/wiki/%ED%95%98%ED%8A%B8_%EC%98%A4%EB%B8%8C_%EB%AF%B8%EB%93%A4%EB%A1%9C%EB%94%94%EC%96%B8_FC" title="하트 오브 미들로디언 FC">하트 오브 미들로디언 FC</a>과의 <a href="/wiki/%EC%85%80%ED%8B%B1_%ED%8C%8C%ED%81%AC" title="셀틱 파크">셀틱 파크</a> 홈경기에서 셀틱 첫골을 득점하였다.<sup class="reference" id="cite_ref-3"><a href="#cite_note-3">[3]</a></sup> 팀은 SPL 타이틀을 마지막 매치데이에 극적으로 가져갔고, 힝켈은 자신의 프로 데뷔 이후로는 처음으로 리그 우승컵을 획득하였다.</p>
<p>그다음 시즌, 힝켈은 <a href="/wiki/%EB%A0%88%EC%9D%B8%EC%A0%80%EC%8A%A4_FC" title="레인저스 FC">레인저스 FC</a>와의 <a href="/wiki/%EC%8A%A4%EC%BD%94%ED%8B%80%EB%9E%9C%EB%93%9C_%EC%BB%B5" title="스코틀랜드 컵">스코틀랜드 컵</a> 결승에서 2-0 승리를 도우며, 팀의 컵 타이틀 획득에 일조하였다.<sup class="reference" id="cite_ref-4"><a href="#cite_note-4">[4]</a></sup></p>
<p>2009년 8월 5일, 힝켈은 러시아 클럽 <a href="/wiki/FC_%EB%94%94%EB%82%98%EB%AA%A8_%EB%AA%A8%EC%8A%A4%ED%81%AC%EB%B0%94" title="FC 디나모 모스크바">FC 디나모 모스크바</a>와의 <a href="/wiki/UEFA_%EC%B1%94%ED%94%BC%EC%96%B8%EC%8A%A4%EB%A6%AC%EA%B7%B8" title="UEFA 챔피언스리그">UEFA 챔피언스리그</a> 3차 예선전에서 중요한 역할을 맡았다. 경기는 힝켈의 <a href="/wiki/%EC%8A%A4%EC%BD%A7_%EB%A7%A5%EB%8F%84%EB%84%90%EB%93%9C" title="스콧 맥도널드">스콧 맥도널드</a> 선제골 어시스트와, <a class="mw-redirect" href="/wiki/%EC%9A%94%EB%A5%B4%EA%B3%A0%EC%8A%A4_%EC%82%AC%EB%A7%88%EB%9D%BC%EC%8A%A4" title="요르고스 사마라스">요르고스 사마라스</a>의 슛이 결승골로 연결되며 셀틱의 2-0 승리로 종료되었다.<sup class="reference" id="cite_ref-5"><a href="#cite_note-5">[5]</a></sup> 힝켈은 디나모의 유효 슈팅을 두차례나 걷어내기도 하였다. 힝켈은 팀에 있어서는 중요한 존재로 누구도 그의 실력에 이의를 달지 않았으며,<sup class="reference" id="cite_ref-6"><a href="#cite_note-6">[6]</a></sup> 이는 리그 29경기 출장을 통해 증명되었다.<sup class="reference" id="cite_ref-7"><a href="#cite_note-7">[7]</a></sup></p>
<p>2010년 8월 14일, 2010-11 시즌의 첫 경기에서, 힝켈은 새로 영입된 <a href="/wiki/%EC%B0%A8%EB%91%90%EB%A6%AC" title="차두리">차두리</a>에 밀렸다. 그 다음주, 힝켈은 십자인대 부상으로 9달 아웃 판정이 났다.<sup class="reference" id="cite_ref-8"><a href="#cite_note-8">[8]</a></sup> 그는 다음해 1월에 훈련에 복귀하였다.<sup class="reference" id="cite_ref-9"><a href="#cite_note-9">[9]</a></sup> 2011년 7월 1일, <a href="/wiki/%EC%85%80%ED%8B%B1_FC" title="셀틱 FC">셀틱 FC</a>와의 계약이 만료된 힝켈은 클럽을 떠났다.</p>
<h3><span class="mw-headline" id="SC_.ED.94.84.EB.9D.BC.EC.9D.B4.EB.B6.80.EB.A5.B4.ED.81.AC">SC 프라이부르크</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%95%88%EB%93%9C%EB%A0%88%EC%95%84%EC%8A%A4_%ED%9E%9D%EC%BC%88&amp;action=edit&amp;section=5" title="부분 편집: SC 프라이부르크">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>2011년 10월 6일, <a href="/wiki/SC_%ED%94%84%EB%9D%BC%EC%9D%B4%EB%B6%80%EB%A5%B4%ED%81%AC" title="SC 프라이부르크">SC 프라이부르크</a>는 힝켈을 자유계약으로 영입하였다.</p>
<p>2012년 9월 10일, 힝켈은 돌연 은퇴를 선언하였다.<sup class="reference" id="cite_ref-10"><a href="#cite_note-10">[10]</a></sup></p>
<h2><span class="mw-headline" id=".EA.B5.AD.EA.B0.80.EB.8C.80.ED.91.9C.ED.8C.80_.EA.B2.BD.EB.A0.A5">국가대표팀 경력</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%95%88%EB%93%9C%EB%A0%88%EC%95%84%EC%8A%A4_%ED%9E%9D%EC%BC%88&amp;action=edit&amp;section=6" title="부분 편집: 국가대표팀 경력">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>힝켈은 2004년 <a href="/wiki/%EC%84%B8%EB%A5%B4%EB%B9%84%EC%95%84_%EB%AA%AC%ED%85%8C%EB%84%A4%EA%B7%B8%EB%A1%9C_%EC%B6%95%EA%B5%AC_%EA%B5%AD%EA%B0%80%EB%8C%80%ED%91%9C%ED%8C%80" title="세르비아 몬테네그로 축구 국가대표팀">세르비아 몬테네그로</a>전을 시작으로 21차례 <a href="/wiki/%EB%8F%85%EC%9D%BC_%EC%B6%95%EA%B5%AC_%EA%B5%AD%EA%B0%80%EB%8C%80%ED%91%9C%ED%8C%80" title="독일 축구 국가대표팀">독일 축구 국가대표팀</a> 경기에 출전하였다.<sup class="reference" id="cite_ref-11"><a href="#cite_note-11">[11]</a></sup> 그는 무릎 부상으로부터 복귀한 뒤 즉시 <a href="/wiki/UEFA_%EC%9C%A0%EB%A1%9C_2004" title="UEFA 유로 2004">UEFA 유로 2004</a>에 출전하는 독일 대표팀 일원이 되었으나 <a href="/wiki/2006%EB%85%84_FIFA_%EC%9B%94%EB%93%9C%EC%BB%B5" title="2006년 FIFA 월드컵">2006년 FIFA 월드컵</a>에는 출전하지 못하였다.<sup class="reference" id="cite_ref-FIFA_player_12-0"><a href="#cite_note-FIFA_player-12">[12]</a></sup> 힝켈은 주전 라이트 백 <a href="/wiki/%ED%95%84%EB%A6%AC%ED%94%84_%EB%9E%8C" title="필리프 람">필리프 람</a>에 밀린 백업 전력이었다.<sup class="reference" id="cite_ref-Players_Info_Hinkel_13-0"><a href="#cite_note-Players_Info_Hinkel-13">[13]</a></sup> 그는 <a href="/wiki/UEFA_%EC%9C%A0%EB%A1%9C_2008" title="UEFA 유로 2008">UEFA 유로 2008</a><sup class="reference" id="cite_ref-Players_Info_Hinkel_13-1"><a href="#cite_note-Players_Info_Hinkel-13">[13]</a></sup> 과 <a href="/wiki/2010%EB%85%84_FIFA_%EC%9B%94%EB%93%9C%EC%BB%B5" title="2010년 FIFA 월드컵">2010년 FIFA 월드컵</a>에도 최종 엔트리에 들지 못하였다.<sup class="reference" id="cite_ref-FIFA_player_12-1"><a href="#cite_note-FIFA_player-12">[12]</a></sup></p>
