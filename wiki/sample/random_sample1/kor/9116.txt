href="/wiki/%EB%8B%A8%EB%AC%B8_%EB%A9%94%EC%8B%9C%EC%A7%80_%EC%84%9C%EB%B9%84%EC%8A%A4" title="단문 메시지 서비스">단문 메시지 서비스</a>(SMS), <a href="/wiki/%EB%A9%80%ED%8B%B0%EB%AF%B8%EB%94%94%EC%96%B4_%EB%A9%94%EC%8B%9C%EC%A7%80_%EC%84%9C%EB%B9%84%EC%8A%A4" title="멀티미디어 메시지 서비스">멀티미디어 메시지 서비스</a>(MMS)등)을 포함하고 있는 <a href="/wiki/%EC%86%8C%ED%94%84%ED%8A%B8%EC%9B%A8%EC%96%B4" title="소프트웨어">소프트웨어</a> <a href="/wiki/%EC%8A%A4%ED%83%9D" title="스택">스택</a>이자 <a href="/wiki/%EB%AA%A8%EB%B0%94%EC%9D%BC_%EC%9A%B4%EC%98%81_%EC%B2%B4%EC%A0%9C" title="모바일 운영 체제">모바일 운영 체제</a>이다. <b>안드로이드</b>는 개발자들이 <a href="/wiki/%EC%9E%90%EB%B0%94_(%ED%94%84%EB%A1%9C%EA%B7%B8%EB%9E%98%EB%B0%8D_%EC%96%B8%EC%96%B4)" title="자바 (프로그래밍 언어)">자바</a> 언어로 응용 프로그램을 작성할 수 있게 하였으며, <a class="mw-redirect" href="/wiki/%EC%BB%B4%ED%8C%8C%EC%9D%BC" title="컴파일">컴파일</a>된 <a href="/wiki/%EB%B0%94%EC%9D%B4%ED%8A%B8%EC%BD%94%EB%93%9C" title="바이트코드">바이트코드</a>를 구동할 수 있는 <a href="/wiki/%EB%9F%B0%ED%83%80%EC%9E%84_%EB%9D%BC%EC%9D%B4%EB%B8%8C%EB%9F%AC%EB%A6%AC" title="런타임 라이브러리">런타임 라이브러리</a>를 제공한다. 또한 안드로이드 <a href="/wiki/%EC%86%8C%ED%94%84%ED%8A%B8%EC%9B%A8%EC%96%B4_%EA%B0%9C%EB%B0%9C_%ED%82%A4%ED%8A%B8" title="소프트웨어 개발 키트">소프트웨어 개발 키트</a>(SDK)를 통해 응용 프로그램을 개발하기 위해 필요한 각종 도구들과 <a href="/wiki/API" title="API">API</a>를 제공한다.</p>
<p>안드로이드는 <a href="/wiki/%EB%A6%AC%EB%88%85%EC%8A%A4_%EC%BB%A4%EB%84%90" title="리눅스 커널">리눅스 커널</a> 위에서 동작하며, 다양한 안드로이드 시스템 구성 요소에서 사용되는 C/C++ <a class="mw-disambig" href="/wiki/%EB%9D%BC%EC%9D%B4%EB%B8%8C%EB%9F%AC%EB%A6%AC" title="라이브러리">라이브러리</a>들을 포함하고 있다. 안드로이드는 기존의 <a href="/wiki/%EC%9E%90%EB%B0%94_%EA%B0%80%EC%83%81_%EB%A8%B8%EC%8B%A0" title="자바 가상 머신">자바 가상 머신</a>과는 다른 <a href="/wiki/%EA%B0%80%EC%83%81_%EB%A8%B8%EC%8B%A0" title="가상 머신">가상 머신</a>인 <a class="mw-redirect" href="/wiki/%EB%8B%AC%EB%B9%85_%EA%B0%80%EC%83%81_%EB%A8%B8%EC%8B%A0" title="달빅 가상 머신">달빅 가상 머신</a>을 통해 자바로 작성된 응용 프로그램을 별도의 <a href="/wiki/%ED%94%84%EB%A1%9C%EC%84%B8%EC%8A%A4" title="프로세스">프로세스</a>에서 실행하는 구조로 되어 있다.</p>
<p><a href="/wiki/2005%EB%85%84" title="2005년">2005년</a>에 안드로이드 사를 <a href="/wiki/%EA%B5%AC%EA%B8%80" title="구글">구글</a>에서 인수한 후 <a href="/wiki/2007%EB%85%84" title="2007년">2007년</a> <a href="/wiki/11%EC%9B%94" title="11월">11월</a>에 안드로이드 플랫폼을 <a class="mw-redirect" href="/wiki/%ED%9C%B4%EB%8C%80%EC%9A%A9_%EC%9E%A5%EC%B9%98" title="휴대용 장치">휴대용 장치</a> 운영 체제로서 무료 공개한다고 발표한 후 48개의 하드웨어, 소프트웨어, 통신 회사가 모여 만든 <a href="/wiki/%EC%98%A4%ED%94%88_%ED%95%B8%EB%93%9C%EC%85%8B_%EC%96%BC%EB%9D%BC%EC%9D%B4%EC%96%B8%EC%8A%A4" title="오픈 핸드셋 얼라이언스">오픈 핸드셋 얼라이언스</a>(Open Handset Aliance, OHA)에서 공개 표준을 위해 개발하고 있다. 구글은 안드로이드의 모든 소스 코드를 <a href="/wiki/%EC%98%A4%ED%94%88_%EC%86%8C%EC%8A%A4" title="오픈 소스">오픈 소스</a> <a class="mw-redirect" href="/wiki/%EB%9D%BC%EC%9D%B4%EC%84%A0%EC%8A%A4" title="라이선스">라이선스</a>인 <a href="/wiki/%EC%95%84%ED%8C%8C%EC%B9%98_%EB%9D%BC%EC%9D%B4%EC%84%A0%EC%8A%A4" title="아파치 라이선스">아파치 v2 라이선스</a>로 배포하고 있어 기업이나 사용자는 각자 안드로이드 프로그램을 독자적으로 개발을 해서 탑재할 수 있다. 또한 응용 프로그램을 사고 팔 수 있는 <a href="/wiki/%EA%B5%AC%EA%B8%80_%ED%94%8C%EB%A0%88%EC%9D%B4" title="구글 플레이">구글 플레이</a>를 제공하고 있으며, 이와 동시에 각 제조사 혹은 통신사별 응용 프로그램 마켓이 함께 운영되고 있다. <a href="/wiki/%EA%B5%AC%EA%B8%80_%ED%94%8C%EB%A0%88%EC%9D%B4" title="구글 플레이">구글 플레이</a>에서는 유료 및 무료 응용 프로그램이 제공되고 있다.</p>

<p><a href="/wiki/2005%EB%85%84" title="2005년">2005년</a> <a href="/wiki/7%EC%9B%94" title="7월">7월</a>에 구글은 미국 <a href="/wiki/%EC%BA%98%EB%A6%AC%ED%8F%AC%EB%8B%88%EC%95%84_%EC%A3%BC" title="캘리포니아 주">캘리포니아 주</a>의 <a class="mw-redirect" href="/wiki/%ED%8C%94%EB%A1%9C%EC%95%8C%ED%86%A0" title="팔로알토">팔로알토</a>에 위치한 작은 안드로이드사를 인수하였다. 안드로이드사는 <a href="/wiki/%EC%95%A4%EB%94%94_%EB%A3%A8%EB%B9%88" title="앤디 루빈">앤디 루빈</a>이 세운 업체이다.</p>
<p><a href="/wiki/2007%EB%85%84" title="2007년">2007년</a> <a href="/wiki/11%EC%9B%94_5%EC%9D%BC" title="11월 5일">11월 5일</a>에 <a href="/wiki/%ED%85%8D%EC%82%AC%EC%8A%A4_%EC%9D%B8%EC%8A%A4%ED%8A%B8%EB%A3%A8%EB%A8%BC%ED%8A%B8" title="텍사스 인스트루먼트">텍사스 인스트루먼트</a>, <a class="mw-redirect" href="/wiki/%EB%B8%8C%EB%A1%9C%EB%93%9C%EC%BB%B4_%EC%BD%94%ED%8D%BC%EB%A0%88%EC%9D%B4%EC%85%98" title="브로드컴 코퍼레이션">브로드컴 코퍼레이션</a>, <a href="/wiki/%EA%B5%AC%EA%B8%80" title="구글">구글</a>, <a href="/wiki/HTC_(%EA%B8%B0%EC%97%85)" title="HTC (기업)">HTC</a>, <a href="/wiki/%EC%9D%B8%ED%85%94" title="인텔">인텔</a>, <a href="/wiki/LG%EC%A0%84%EC%9E%90" title="LG전자">LG전자</a>, <a href="/wiki/%EB%A7%88%EB%B2%A8_%ED%85%8C%ED%81%AC%EB%86%80%EB%A1%9C%EC%A7%80_%EA%B7%B8%EB%A3%B9" title="마벨 테크놀로지 그룹">마벨 테크놀로지 그룹</a>, <a class="mw-redirect" href="/wiki/%EB%AA%A8%ED%86%A0%EB%A1%9C%EB%9D%BC" title="모토로라">모토로라</a>, <a href="/wiki/%EC%97%94%EB%B9%84%EB%94%94%EC%95%84" title="엔비디아">엔비디아</a>, <a href="/wiki/%ED%80%84%EC%BB%B4" title="퀄컴">퀄컴</a>, <a href="/wiki/%EC%82%BC%EC%84%B1%EC%A0%84%EC%9E%90" title="삼성전자">삼성전자</a>, <a class="mw-redirect" href="/wiki/%EC%8A%A4%ED%94%84%EB%A6%B0%ED%8A%B8_%EB%84%A5%EC%8A%A4%ED%85%94" title="스프린트 넥스텔">스프린트 넥스텔</a>, <a class="mw-redirect" href="/wiki/T%EB%AA%A8%EB%B0%94%EC%9D%BC" title="T모바일">T-모바일</a>의 몇몇 회사로 구성된 <a class="mw-redirect" href="/wiki/%EC%BB%A8%EC%86%8C%EC%8B%9C%EC%97%84" title="컨소시엄">컨소시엄</a>인 <a href="/wiki/%EC%98%A4%ED%94%88_%ED%95%B8%EB%93%9C%EC%85%8B_%EC%96%BC%EB%9D%BC%EC%9D%B4%EC%96%B8%EC%8A%A4" title="오픈 핸드셋 얼라이언스">오픈 핸드셋 얼라이언스</a>(OHA)가 모바일 기기의 공개 표준을 개발하는 것을 목표로 결성되었다.<sup class="reference" id="cite_ref-AndroidAnnouncement_1-0"><a href="#cite_note-AndroidAnnouncement-1">[1]</a></sup> 또한 OHA는 리눅스 커널 2.6에서 빌드된 그들의 첫 번째 모바일 기기 플랫폼 결과물인 안드로이드를 발표하였다.</p>
<p><a href="/wiki/2008%EB%85%84" title="2008년">2008년</a> <a href="/wiki/10%EC%9B%94_21%EC%9D%BC" title="10월 21일">10월 21일</a>에 안드로이드가 오픈 소스로 선언되었다. 구글은 네트워크와 텔레폰 스택을 포함하는 완전한 소스 코드를 <a href="/wiki/%EC%95%84%ED%8C%8C%EC%B9%98_%EB%9D%BC%EC%9D%B4%EC%84%A0%EC%8A%A4" title="아파치 라이선스">아파치 라이선스</a>로 공개하였다.</p>
<p>2008년 <a href="/wiki/12%EC%9B%94_9%EC%9D%BC" title="12월 9일">12월 9일</a>에 <a href="/wiki/ARM_%ED%99%80%EB%94%A9%EC%8A%A4" title="ARM 홀딩스">ARM 홀딩스</a>, 아세로스(Atheros Communications), <a href="/wiki/%EC%97%90%EC%9D%B4%EC%88%98%EC%8A%A4" title="에이수스">에이수스</a>, <a href="/wiki/%EA%B0%80%EB%A5%B4%EB%AF%BC" title="가르민">가르민</a>, <a href="/wiki/%EC%86%8C%ED%94%84%ED%8A%B8%EB%B1%85%ED%81%AC" title="소프트뱅크">소프트뱅크</a>, <a class="mw-redirect" href="/wiki/%EC%86%8C%EB%8B%88_%EC%97%90%EB%A6%AD%EC%8A%A8" title="소니 에릭슨">소니 에릭슨</a>, <a href="/wiki/%EB%8F%84%EC%8B%9C%EB%B0%94" title="도시바">도시바</a>, <a href="/wiki/%EB%B3%B4%EB%8B%A4%ED%8F%B0" title="보다폰">보다폰</a>으로 구성된 새로운 14개의 멤버가 안드로이드 프로젝트에 참여하였다.</p>
<p><a href="/wiki/2013%EB%85%84" title="2013년">2013년</a> <a href="/wiki/12%EC%9B%94_12%EC%9D%BC" title="12월 12일">12월 12일</a> 그러나 현재 API만 완전한 공개소스인 상태이며 VM에대한 소스는 공개하지 않고 있다. 그런점에서 완전한 오픈소스 스마트폰 운영체제라고 할수는 없다.</p>

<p><a href="/wiki/2014%EB%85%84" title="2014년">2014년</a> <a href="/wiki/6%EC%9B%94_25%EC%9D%BC" title="6월 25일">6월 25일</a>, 구글은 <a href="/wiki/%EA%B5%AC%EA%B8%80_I/O" title="구글 I/O">구글 I/O</a>에서 다양한 안드로이드 구축을 통한 <a class="mw-redirect" href="/wiki/%EC%82%AC%EB%AC%BC%EC%9D%B8%ED%84%B0%EB%84%B7" title="사물인터넷">사물인터넷</a>(IoT) 전략을 공개하였다.<sup class="reference" id="cite_ref-2"><a href="#cite_note-2">[2]</a></sup> 안드로이드 웨어는 미리 공개되었으며, 웨어를 제외한 나머지는 구글 I/O에서 공개되었다.</p>
<p>“일상의 전자 기기부터 농부가 사용하는 장비, 버스까지 인터넷에 연결된다면 사람들의 삶은 더 크게 변할 것이다.”<a href="/wiki/2015%EB%85%84" title="2015년">2015년</a> <a href="/wiki/4%EC%9B%94_28%EC%9D%BC" title="4월 28일">4월 28일</a>(현지시각)에 미국 샌프란시스코에서 열린 <a href="/wiki/%EA%B5%AC%EA%B8%80_I/O" title="구글 I/O">구글 I/O</a>에서 구글의 선다 피차이 제품총괄 수석부사장이 <a class="mw-redirect" href="/wiki/%EC%82%AC%EB%AC%BC%EC%9D%B8%ED%84%B0%EB%84%B7" title="사물인터넷">사물인터넷</a>(IoT) 운영체제 ‘브릴로(Brillo)’를 발표하며 한 말이다.<sup class="reference" id="cite_ref-3"><a href="#cite_note-3">[3]</a></sup></p>

(안드로이드 4.4W)</li>

<h2><span class="mw-headline" id=".EB.B2.84.EC.A0.84">버전</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%95%88%EB%93%9C%EB%A1%9C%EC%9D%B4%EB%93%9C_(%EC%9A%B4%EC%98%81_%EC%B2%B4%EC%A0%9C)&amp;action=edit&amp;section=4" title="부분 편집: 버전">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<div class="rellink relarticle mainarticle"><img alt="" data-file-height="128" data-file-width="128" height="16" src="//upload.wikimedia.org/wikipedia/commons/thumb/e/ec/Crystal_Clear_app_xmag.svg/16px-Crystal_Clear_app_xmag.svg.png" srcset="//upload.wikimedia.org/wikipedia/commons/thumb/e/ec/Crystal_Clear_app_xmag.svg/24px-Crystal_Clear_app_xmag.svg.png 1.5x, //upload.wikimedia.org/wikipedia/commons/thumb/e/ec/Crystal_Clear_app_xmag.svg/32px-Crystal_Clear_app_xmag.svg.png 2x" width="16"/> 이 부분의 본문은 <a href="/wiki/%EC%95%88%EB%93%9C%EB%A1%9C%EC%9D%B4%EB%93%9C_%EB%B2%84%EC%A0%84_%EC%97%AD%EC%82%AC" title="안드로이드 버전 역사">안드로이드 버전 역사</a>입니다.</div>
<p>안드로이드는 6개월~12개월에 한번씩 업데이트를 진행하며, 업데이트의 내용은 주로 오류 수정이나 새로운 기능 추가로 구성된다. 안드로이드의 각 버전들은 알파벳 첫 글자를 오름차순에 맞춘 음식 이름(디저트)을 코드명으로 삼고 있다. 버전은 1.0, 2.0, 4.0, 5.0, 6.0, 7.0...으로 올라가며, 앞자리의 숫자가 올라가면 대규모 업데이트(메이저 업데이트), 소수점 아래 숫자가 올라가면 소규모 업데이트(마이너 업데이트)로 판명한다.</p>
<p>이전 버전을 사용하는 기기들은 바로 업데이트를 제공받을 수 없으며, 기기 제조사에서 구글의 소스를 받아 직접 작업후 배포해야 하므로 가끔씩 하드웨어 성능의 한계로 업그레이드가 취소되기도 한다.<sup class="reference" id="cite_ref-7"><a href="#cite_note-7">[7]</a></sup></p>

<h3><span class="mw-headline" id=".EB.8C.80.ED.95.9C.EB.AF.BC.EA.B5.AD">대한민국</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%95%88%EB%93%9C%EB%A1%9C%EC%9D%B4%EB%93%9C_(%EC%9A%B4%EC%98%81_%EC%B2%B4%EC%A0%9C)&amp;action=edit&amp;section=8" title="부분 편집: 대한민국">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<ul>
<li><a href="/wiki/2014%EB%85%84" title="2014년">2014년</a> <a href="/wiki/1%EC%9B%94_21%EC%9D%BC" title="1월 21일">1월 21일</a> <a href="/wiki/%EB%AF%B8%EA%B5%AD" title="미국">미국</a>의 시장조사업체 스트래티지애널리틱스(SA)는 세계 88개국을 대상으로 운영체제(OS)별 스마트폰 사용자 수(IB; Installed Base)와 점유율을 조사하였다.</li>
<li>결과 대한민국의 안드로이드 스마트폰 사용자 점유율이 93.4%로 세계에서 가장 높았으며. 이는 전 세계 시장 전체의 안드로이드 스마트폰 사용자 수 비율인 67.5%와 견줘 무려 25.9%포인트 높은 수준이다. 또한 조사 대상국 중 안드로이드의 비중이 90%를 넘는 유일한 나라이다<sup class="reference" id="cite_ref-12"><a href="#cite_note-12">[12]</a></sup>.</li>
</ul>
