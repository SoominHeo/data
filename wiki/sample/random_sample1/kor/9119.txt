<p><b>안드로이드 진저브레드</b>는 <a href="/wiki/2010%EB%85%84" title="2010년">2010년</a> <a href="/wiki/12%EC%9B%94" title="12월">12월</a>에 출시한 <a href="/wiki/%EA%B7%BC%EA%B1%B0%EB%A6%AC_%EB%AC%B4%EC%84%A0_%ED%86%B5%EC%8B%A0" title="근거리 무선 통신">근거리 무선 통신</a> 기능을 첫 지원하였던 <a href="/wiki/%EA%B5%AC%EA%B8%80" title="구글">구글</a>의 <a href="/wiki/%EC%95%88%EB%93%9C%EB%A1%9C%EC%9D%B4%EB%93%9C_(%EC%9A%B4%EC%98%81_%EC%B2%B4%EC%A0%9C)" title="안드로이드 (운영 체제)">안드로이드 (운영 체제)</a> 중 일부이다. 진저브레드의 <a href="/wiki/%EC%82%AC%EC%9A%A9%EC%9E%90_%EC%9D%B8%ED%84%B0%ED%8E%98%EC%9D%B4%EC%8A%A4" title="사용자 인터페이스">사용자 인터페이스</a> 는 쉽게 만들고 마스터하고, 사용하는데에 빠르고, 더 나은 전원 효율성 등에 중점을 두어 만들어졌다. 진저브레드는 2.6.35 버전의 <a href="/wiki/%EB%A6%AC%EB%88%85%EC%8A%A4_%EC%BB%A4%EB%84%90" title="리눅스 커널">리눅스 커널</a>을 사용한다.</p>

<li>다운로드 관리: 다운로드 관리 애플리케이션이 강화되어 브라우저, 메일 등에서 다운로드 받은 파일을 쉽게 찾을 수 있게 배려</li>

<li>게임을 위한 성능 향상
<ul>
<li><a class="mw-redirect" href="/wiki/%EB%8B%AC%EB%B9%85_%EA%B0%80%EC%83%81_%EB%A8%B8%EC%8B%A0" title="달빅 가상 머신">달빅 가상 머신</a>의 병행 가베지 컬렉터 도입</li>

<li>NDK 향상</li>

<li>다수 카메라 제어 가능하도록 카메라 API 향상</li>
<li>오픈코어(<span style="font-size: smaller;"><a href="/wiki/%EC%98%81%EC%96%B4" title="영어">영어</a>:</span> <span lang="en" xml:lang="en">OpenCore</span>)를 대체하는 새 미디어 프레임워크: 기존 코덱/컨테이너 지원 유지</li>
