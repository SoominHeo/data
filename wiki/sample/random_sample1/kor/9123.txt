<p><b>안드리스 베르진시</b>(<span style="font-size: smaller;"><a href="/wiki/%EB%9D%BC%ED%8A%B8%EB%B9%84%EC%95%84%EC%96%B4" title="라트비아어">라트비아어</a>:</span> <span lang="lv" xml:lang="lv">Andris Bērziņš</span>, <a href="/wiki/1944%EB%85%84" title="1944년">1944년</a> <a href="/wiki/12%EC%9B%94_10%EC%9D%BC" title="12월 10일">12월 10일</a> ~ )는 전 <a href="/wiki/%EB%9D%BC%ED%8A%B8%EB%B9%84%EC%95%84" title="라트비아">라트비아</a>의 <a href="/wiki/%EB%9D%BC%ED%8A%B8%EB%B9%84%EC%95%84%EC%9D%98_%EB%8C%80%ED%86%B5%EB%A0%B9" title="라트비아의 대통령">대통령</a>으로, <a href="/wiki/2011%EB%85%84" title="2011년">2011년</a> <a href="/wiki/6%EC%9B%94_2%EC%9D%BC" title="6월 2일">6월 2일</a> 대통령 선거에 출마하여 승리했다.</p>
<h2><span class="mw-headline" id=".EC.9D.BC.EC.83.9D">일생</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%95%88%EB%93%9C%EB%A6%AC%EC%8A%A4_%EB%B2%A0%EB%A5%B4%EC%A7%84%EC%8B%9C_(%EB%9D%BC%ED%8A%B8%EB%B9%84%EC%95%84%EC%9D%98_%EB%8C%80%ED%86%B5%EB%A0%B9)&amp;action=edit&amp;section=1" title="부분 편집: 일생">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>안드리스 베르진시는 <a href="/wiki/1944%EB%85%84" title="1944년">1944년</a> <a href="/wiki/12%EC%9B%94_10%EC%9D%BC" title="12월 10일">12월 10일</a> 당시 <a class="mw-redirect" href="/wiki/%EC%86%8C%EB%B9%84%EC%97%90%ED%8A%B8_%EC%97%B0%EB%B0%A9" title="소비에트 연방">소비에트 연방</a>의 지배를 받던 <a href="/wiki/%EB%9D%BC%ED%8A%B8%EB%B9%84%EC%95%84_%EC%86%8C%EB%B9%84%EC%97%90%ED%8A%B8_%EC%82%AC%ED%9A%8C%EC%A3%BC%EC%9D%98_%EA%B3%B5%ED%99%94%EA%B5%AD" title="라트비아 소비에트 사회주의 공화국">라트비아</a>의 <a class="new" href="/w/index.php?title=%EB%8B%88%ED%83%80%EC%9A%B0%EB%A0%88&amp;action=edit&amp;redlink=1" title="니타우레 (없는 문서)">니타우레</a>에서 태어났다. <a href="/wiki/1958%EB%85%84" title="1958년">1958년</a> 그는 초등학교를 졸업하고 <a href="/wiki/1962%EB%85%84" title="1962년">1962년</a>까지 <a href="/wiki/%EC%8B%9C%EA%B5%B4%EB%8B%A4" title="시굴다">시굴다</a>에 있는 중학교에 다녔다. 그 후 그는 <a href="/wiki/%EB%A6%AC%EA%B0%80_%EA%B8%B0%EC%88%A0%EB%8C%80%ED%95%99%EA%B5%90" title="리가 기술대학교">리가 기술대학교</a>를 졸업하였고, 후에는 <a class="new" href="/w/index.php?title=%EC%9D%BC%EB%A0%89%ED%8A%B8%EB%A1%A0%EC%8A%A4_%EA%B3%B5%EC%9E%A5&amp;action=edit&amp;redlink=1" title="일렉트론스 공장 (없는 문서)">일렉트론스 공장</a>의 라디오 엔지니어가 되었다. <a href="/wiki/1988%EB%85%84" title="1988년">1988년</a> 그는 <a href="/wiki/%EB%9D%BC%ED%8A%B8%EB%B9%84%EC%95%84" title="라트비아">라트비아</a>의 주지사로 일했으며 <a class="new" href="/w/index.php?title=%EB%9D%BC%ED%8A%B8%EB%B9%84%EC%95%84_%EB%8C%80%ED%95%99%EA%B5%90&amp;action=edit&amp;redlink=1" title="라트비아 대학교 (없는 문서)">라트비아 대학교</a>를 졸업하였다.</p>
<p><a href="/wiki/1989%EB%85%84" title="1989년">1989년</a> 베르진시는 <a href="/wiki/%EB%B0%9C%EB%AF%B8%EC%97%90%EB%9D%BC" title="발미에라">발미에라</a> 지구 협의회 의장으로 선출된다. <a href="/wiki/1990%EB%85%84" title="1990년">1990년</a> 그는 <a href="/wiki/%EB%B0%9C%EB%AF%B8%EC%97%90%EB%9D%BC" title="발미에라">발미에라</a> 지구 협의회의 최고의장으로 선출되었다. <a href="/wiki/1990%EB%85%84" title="1990년">1990년</a> <a href="/wiki/5%EC%9B%94_4%EC%9D%BC" title="5월 4일">5월 4일</a>, 그는 <a class="new" href="/w/index.php?title=%EB%9D%BC%ED%8A%B8%EB%B9%84%EC%95%84_%EB%8F%85%EB%A6%BD_%EC%84%A0%EC%96%B8%EC%84%9C&amp;action=edit&amp;redlink=1" title="라트비아 독립 선언서 (없는 문서)">라트비아 독립 선언서</a>를 복원하는 데 찬성하게 된다.</p>
<p><a href="/wiki/1993%EB%85%84" title="1993년">1993년</a> 그는 <a class="new" href="/w/index.php?title=%EB%9D%BC%ED%8A%B8%EB%B9%84%EC%95%84_%EC%9D%80%ED%96%89&amp;action=edit&amp;redlink=1" title="라트비아 은행 (없는 문서)">라트비아 은행</a>에서 일하게 되며 <a class="new" href="/w/index.php?title=SEB_%EC%9C%A0%EB%8B%88%EB%B0%A9%EC%B9%B4&amp;action=edit&amp;redlink=1" title="SEB 유니방카 (없는 문서)">SEB 유니방카</a>의 행장으로 선출된다. <a href="/wiki/2005%EB%85%84" title="2005년">2005년</a> 그는 <a class="new" href="/w/index.php?title=%EB%85%B9%EC%83%89%EB%86%8D%EA%B2%BD%EC%97%B0%ED%95%A9&amp;action=edit&amp;redlink=1" title="녹색농경연합 (없는 문서)">녹색농경연합</a> 의원으로서 최초의 <a href="/wiki/%EB%A6%AC%EA%B0%80" title="리가">리가</a> 시장 선거에 나갔지만 낙선된다. 그러나 제8대 라트비아 대통령 선거에서 승리, <a href="/wiki/2011%EB%85%84" title="2011년">2011년</a> <a href="/wiki/7%EC%9B%94_8%EC%9D%BC" title="7월 8일">7월 8일</a> 대통령이 된다.</p>
