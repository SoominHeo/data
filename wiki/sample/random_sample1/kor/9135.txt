<p><b>안심(혁신도시·첨복단지)역</b>(安心(革新都市·尖複團地)驛)은 <a href="/wiki/%EB%8C%80%ED%95%9C%EB%AF%BC%EA%B5%AD" title="대한민국">대한민국</a> <a href="/wiki/%EB%8C%80%EA%B5%AC%EA%B4%91%EC%97%AD%EC%8B%9C" title="대구광역시">대구광역시</a> <a href="/wiki/%EB%8F%99%EA%B5%AC_(%EB%8C%80%EA%B5%AC%EA%B4%91%EC%97%AD%EC%8B%9C)" title="동구 (대구광역시)">동구</a> <a href="/wiki/%EC%95%88%EC%8B%AC%EB%A1%9C_(%EB%8C%80%EA%B5%AC)" title="안심로 (대구)">안심로</a> 송정삼거리에 있는 <a href="/wiki/%EB%8C%80%EA%B5%AC_%EB%8F%84%EC%8B%9C%EC%B2%A0%EB%8F%84_1%ED%98%B8%EC%84%A0" title="대구 도시철도 1호선">대구 도시철도 1호선</a>의 <a href="/wiki/%EC%A0%84%EC%B2%A0%EC%97%AD" title="전철역">지하철역</a>이다. 현재 <a href="/wiki/%EB%8C%80%EA%B5%AC_%EB%8F%84%EC%8B%9C%EC%B2%A0%EB%8F%84_1%ED%98%B8%EC%84%A0" title="대구 도시철도 1호선">대구 도시철도 1호선</a>의 종점으로, 인근에 <a href="/wiki/%EC%95%88%EC%8B%AC%EC%B0%A8%EB%9F%89%EC%82%AC%EC%97%85%EC%86%8C" title="안심차량사업소">안심차량사업소</a>가 소재하고 있다.</p>

<h2><span class="mw-headline" id=".ED.8A.B9.EC.A7.95">특징</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%95%88%EC%8B%AC%EC%97%AD&amp;action=edit&amp;section=1" title="부분 편집: 특징">원본 편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p><a href="/wiki/%EC%84%9C%EC%9A%B8_%EC%A7%80%ED%95%98%EC%B2%A0_8%ED%98%B8%EC%84%A0" title="서울 지하철 8호선">서울 지하철 8호선</a> <a href="/wiki/%EB%B3%B5%EC%A0%95%EC%97%AD" title="복정역">복정역</a>과 <a href="/wiki/%EC%88%98%EB%8F%84%EA%B6%8C_%EC%A0%84%EC%B2%A0" title="수도권 전철">수도권 전철</a> <a href="/wiki/%EC%88%98%EC%9D%B8%EC%84%A0" title="수인선">수인선</a> <a href="/wiki/%EC%9D%B8%EC%B2%9C%EC%97%AD" title="인천역">인천역</a>처럼 <a href="/wiki/%EB%8C%80%EA%B5%AC_%EB%8F%84%EC%8B%9C%EC%B2%A0%EB%8F%84_1%ED%98%B8%EC%84%A0" title="대구 도시철도 1호선">대구 도시철도 1호선</a>에서 유일무이하게 섬식 승강장 구조로 되어 있다. 현재는 1호선의 종점이나, 향후 이 역을 거쳐서 <a href="/wiki/%ED%95%98%EC%96%91%EC%97%AD" title="하양역">하양역</a>까지 연장되는 방안이 최종 확정되었다.<sup class="reference" id="cite_ref-1"><a href="#cite_note-1">[1]</a></sup> <a href="/wiki/%EA%B2%BD%EC%83%81%EB%B6%81%EB%8F%84" title="경상북도">경상북도</a> <a href="/wiki/%EA%B2%BD%EC%82%B0%EC%8B%9C" title="경산시">경산시</a> <a href="/wiki/%ED%95%98%EC%96%91%EC%9D%8D" title="하양읍">하양읍</a>으로 가는 길목에 있다. <a href="/wiki/%EB%8C%80%EA%B5%AC_%EB%8F%84%EC%8B%9C%EC%B2%A0%EB%8F%84_2%ED%98%B8%EC%84%A0" title="대구 도시철도 2호선">2호선</a> 개통 전까지는 대구 도시철도의 유일한 섬식 승강장 역이었다.</p>
<p>역 바로 앞은 아니지만 동구 숙천동과의 경계에 있는 북경산 지역 및 영천 지역으로 가는 환승 거점의 역할을 하고 있어서 이용률이 있는 편이다.</p>
<h2><span class="mw-headline" id=".EC.97.AD.EB.AA.85.EC.9D.98_.EC.9C.A0.EB.9E.98">역명의 유래</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%95%88%EC%8B%AC%EC%97%AD&amp;action=edit&amp;section=2" title="부분 편집: 역명의 유래">원본 편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p><a href="/wiki/%EC%8B%A0%EB%9D%BC" title="신라">신라</a> 말기, <a href="/wiki/%ED%9B%84%EB%B0%B1%EC%A0%9C" title="후백제">후백제</a>의 <a href="/wiki/%EA%B2%AC%ED%9B%A4" title="견훤">견훤</a>과 맞붙었던 <a href="/wiki/%EA%B3%B5%EC%82%B0_%EB%8F%99%EC%88%98_%EC%A0%84%ED%88%AC" title="공산 동수 전투">공산 전투</a>에서 패전해, <a href="/wiki/%EC%8B%A0%EC%88%AD%EA%B2%B8" title="신숭겸">신숭겸</a> 등의 부하들을 잃고 도망쳤던 <a href="/wiki/%EA%B3%A0%EB%A0%A4" title="고려">고려</a>의 <a class="mw-redirect" href="/wiki/%EC%99%95%EA%B1%B4" title="왕건">왕건</a>이 이곳에 다다라서야 안심했다는 것에서 유래하였다.<sup class="reference" id="cite_ref-2"><a href="#cite_note-2">[2]</a></sup> 참고로 왕건이 견훤과의 전투에서 패전한 곳이 <a href="/wiki/%EA%B4%91%EB%82%A8%EC%9E%90%EB%8F%99%EC%B0%A8" title="광남자동차">광남자동차</a> 차고지가 있는 <a href="/wiki/%EB%B6%88%EB%A1%9C%EB%B4%89%EB%AC%B4%EB%8F%99" title="불로봉무동">봉무동</a>의 파군재삼거리다.<sup class="reference" id="cite_ref-3"><a href="#cite_note-3">[3]</a></sup> 안심은 원래 대구가 아닌 경산군 안심읍이었다가 대구시 안심동으로 편입된 지역이다. 안심역이 있는 송정삼거리에서 반야월 구길과 신길이 나뉘며, <a href="/wiki/%EB%8C%80%EA%B5%AC_%EB%8F%84%EC%8B%9C%EC%B2%A0%EB%8F%84_1%ED%98%B8%EC%84%A0" title="대구 도시철도 1호선">대구 도시철도 1호선</a>은 반야월 신길로 운행한다. <a href="/wiki/2014%EB%85%84" title="2014년">2014년</a> <a href="/wiki/2%EC%9B%94_28%EC%9D%BC" title="2월 28일">2월 28일</a> <b>안심역</b>에서 <b>안심(혁신도시·첨복단지)역</b>으로 역명이 변경되었다.<sup class="reference" id="cite_ref-4"><a href="#cite_note-4">[4]</a></sup> 첨복단지는 첨단의료복합단지의 줄임말이다.</p>

<h2><span class="mw-headline" id=".EC.97.AD_.EA.B5.AC.EC.A1.B0">역 구조</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%95%88%EC%8B%AC%EC%97%AD&amp;action=edit&amp;section=4" title="부분 편집: 역 구조">원본 편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>1면 2선의 <a class="mw-redirect" href="/wiki/%EC%84%AC%EC%8B%9D_%EC%8A%B9%EA%B0%95%EC%9E%A5" title="섬식 승강장">섬식 승강장</a>이 있는 지하역이다. <a href="/wiki/%EC%8A%A4%ED%81%AC%EB%A6%B0%EB%8F%84%EC%96%B4" title="스크린도어">스크린도어</a>가 설치되어 있다.</p>
<p>현재 <a href="/wiki/%EB%8C%80%EA%B5%AC_%EB%8F%84%EC%8B%9C%EC%B2%A0%EB%8F%84_1%ED%98%B8%EC%84%A0" title="대구 도시철도 1호선">대구 도시철도 1호선</a>의 32개 역들 중 유일무이 하게 섬식 승강장 구조를 갖추고 있는 역이다.</p>
