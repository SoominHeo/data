<p><b>안양 KGC인삼공사 프로농구단</b>(Anyang KGC Ginseng Corporation Pro Basketball Club)은 <a href="/wiki/%EA%B2%BD%EA%B8%B0%EB%8F%84" title="경기도">경기도</a> <a href="/wiki/%EC%95%88%EC%96%91%EC%8B%9C" title="안양시">안양시</a>를 연고로 하는 <a href="/wiki/%EB%8C%80%ED%95%9C%EB%AF%BC%EA%B5%AD" title="대한민국">대한민국</a>의 프로 <a href="/wiki/%EB%86%8D%EA%B5%AC" title="농구">농구</a>단이다. 전신 구단으로는 안양 SBS 스타즈로 <a href="/wiki/1992%EB%85%84" title="1992년">1992년</a>부터 존재해 왔다. 2008-09 시즌에는 <a href="/wiki/%EC%9D%B8%EC%B2%9C_%EC%A0%84%EC%9E%90%EB%9E%9C%EB%93%9C_%EC%97%98%EB%A6%AC%ED%8E%80%EC%B8%A0" title="인천 전자랜드 엘리펀츠">인천 전자랜드 엘리펀츠</a>와 <a href="/wiki/%EC%B0%BD%EC%9B%90_LG_%EC%84%B8%EC%9D%B4%EC%BB%A4%EC%8A%A4" title="창원 LG 세이커스">창원 LG 세이커스</a>와의 경쟁에서 밀려나며 7위로 플레이오프에 탈락했다. <a href="/wiki/2010%EB%85%84" title="2010년">2010년</a> 모기업 <a href="/wiki/KT%26G" title="KT&amp;G">KT&amp;G</a>가 이사회 결의를 통해 자회사인 <a href="/wiki/%ED%95%9C%EA%B5%AD%EC%9D%B8%EC%82%BC%EA%B3%B5%EC%82%AC" title="한국인삼공사">한국인삼공사</a>에게 스포츠단 지분 100%를 옮기면서 팀명과 엠블렘을 변경하여 2010-11 시즌부터 참가했다. 2011-12 시즌에는 신인 <a href="/wiki/%EC%98%A4%EC%84%B8%EA%B7%BC" title="오세근">오세근</a>의 활약으로 당해 시즌 정규리그 1위팀이었던 <a href="/wiki/%EC%9B%90%EC%A3%BC_%EB%8F%99%EB%B6%80_%ED%94%84%EB%A1%9C%EB%AF%B8" title="원주 동부 프로미">원주 동부 프로미</a>와의 <a href="/wiki/KBL_%EC%B1%94%ED%94%BC%EC%96%B8_%EA%B2%B0%EC%A0%95%EC%A0%84" title="KBL 챔피언 결정전">KBL 챔피언 결정전</a>에서 6차전까지 가는 접전 끝에 시리즈 전적 4-2으로 승리하고 창단 첫 우승을 차지하였다.<sup class="reference" id="cite_ref-1"><a href="#cite_note-1">[1]</a></sup></p>

<h3><span class="mw-headline" id="1997-2005_.EC.8B.9C.EC.A6.8C_.28.EC.95.88.EC.96.91_SBS_.EC.8A.A4.ED.83.80.EC.A6.88_.EC.8B.9C.EC.A0.88.29">1997-2005 시즌 (안양 SBS 스타즈 시절)</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%95%88%EC%96%91_KGC%EC%9D%B8%EC%82%BC%EA%B3%B5%EC%82%AC_%EB%86%8D%EA%B5%AC%EB%8B%A8&amp;action=edit&amp;section=2" title="부분 편집: 1997-2005 시즌 (안양 SBS 스타즈 시절)">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<ul>
<li>1997 시즌 8개팀 중 2위를 기록해서 4강 플레이오프에 직행했다. 첫 시즌 <a href="/wiki/%EA%B9%80%EB%8F%99%EA%B4%91" title="김동광">김동광</a> 감독은 4강 플레이오프에서의 챔피언결정전 진출 실패(원주 나래 해커스 4:1 안양 SBS 스타즈)로 인해 사퇴했다.</li>
</ul>
<ul>
<li>1997-98, 1998-99 시즌은 각각 8위, 7위로 플레이오프 진출도 하지 못했다. 당시 <a href="/wiki/%EA%B0%95%EC%A0%95%EC%88%98" title="강정수">강정수</a> 감독이 지휘하였으나 그도 역시 성적 부진으로 인해서 <a href="/wiki/1999%EB%85%84_4%EC%9B%94" title="1999년 4월">1999년 4월</a> 사퇴했다.</li>
</ul>
<ul>
<li>1999-2000 시즌 <a href="/wiki/%EA%B9%80%EC%9D%B8%EA%B1%B4" title="김인건">김인건</a> 감독이 취임하였다. 그는 1999-2000 시즌의 안양 SBS 스타즈를 5위로 올려놓았다. 6강 플레이 오프에서 원주 삼보 엑서스를 만나 3대1로 4강 플레이 오프에 진출했으나 대전 현대 다이넷에 3대 0으로 패배했다. 2000-01 시즌에는 안양 SBS 스타즈는 4위까지 올라서 6강에서 인천 신세기 빅스를 이기고 4강에 진입했으나 역시 4강 플레이오프에서 <a href="/wiki/%EC%84%9C%EC%9A%B8_%EC%82%BC%EC%84%B1_%EC%8D%AC%EB%8D%94%EC%8A%A4" title="서울 삼성 썬더스">서울 삼성 썬더스</a>에 1:3의 참패를 당했다. 2001-02 시즌에는 6위를 차지하고 6강 플레이오프에 진입했으나 <a href="/wiki/%EC%A0%84%EC%A3%BC_KCC_%EC%9D%B4%EC%A7%80%EC%8A%A4" title="전주 KCC 이지스">전주 KCC 이지스</a>에 완패했다. 그 시즌을 끝으로 <a href="/wiki/%EA%B9%80%EC%9D%B8%EA%B1%B4" title="김인건">김인건</a> 감독이 사퇴한다.</li>
</ul>
<ul>
<li><a class="new" href="/w/index.php?title=%EC%A0%95%EB%8D%95%ED%99%94&amp;action=edit&amp;redlink=1" title="정덕화 (없는 문서)">정덕화</a> 감독의 체제로 새로운 안양 SBS 스타즈가 탄생했다. 그러나 안양 SBS 스타즈는 초반 2위까지 오르다가 하락해서 7위를 지키는 듯 하다가 막판에 8위로 떨어지면서 2002-03 시즌은 실패했다. 2003-04 시즌도 재정비를 다짐했으나 <a href="/wiki/%EC%A0%84%EC%A3%BC_KCC_%EC%9D%B4%EC%A7%80%EC%8A%A4" title="전주 KCC 이지스">전주 KCC 이지스</a>와의 몰수게임이 벌어지고 그 후 7연패까지 빠지는 등 고전하다가 9위로 시즌을 마감한다. <a href="/wiki/2004%EB%85%84_5%EC%9B%94" title="2004년 5월">2004년 5월</a> 정덕화 감독이 사퇴했다.</li>
</ul>
<ul>
<li>그 뒤, <a href="/wiki/%EA%B9%80%EB%8F%99%EA%B4%91" title="김동광">김동광</a> 감독이 다시 취임하였다. 2004-05 시즌은 최고의 전성기였다. 초반에는 개막 3연패 등 부진을 면하지 못했다. SBS는 올스타전 직전에 7위를 기록하고 이 시즌도 그렇게 마감하는 듯 했으나 <a href="/wiki/2005%EB%85%84" title="2005년">2005년</a> <a href="/wiki/2%EC%9B%94_5%EC%9D%BC" title="2월 5일">2월 5일</a>부터 조 번의 대체 용병으로 등장한 <a class="new" href="/w/index.php?title=%EB%8B%A8%ED%85%8C_%EC%A1%B4%EC%8A%A4&amp;action=edit&amp;redlink=1" title="단테 존스 (없는 문서)">단테 존스</a>에 힘입어서 안양 SBS 스타즈가 15연승을 하는 등 3위까지 올랐다. 그러나 상대 전적에서 3승 3패로 팽팽했던 <a class="mw-redirect" href="/wiki/%EA%B3%A0%EC%96%91_%EC%98%A4%EB%A6%AC%EC%98%A8%EC%8A%A4" title="고양 오리온스">대구 오리온스</a>를 꺾고 4강을 오르고 <a href="/wiki/%EC%A0%84%EC%A3%BC_KCC_%EC%9D%B4%EC%A7%80%EC%8A%A4" title="전주 KCC 이지스">전주 KCC 이지스</a>와 원주 TG삼보 엑서스까지 꺾는 듯 하더니 <a href="/wiki/%EC%A0%84%EC%A3%BC_KCC_%EC%9D%B4%EC%A7%80%EC%8A%A4" title="전주 KCC 이지스">전주 KCC 이지스</a>전에서 멈춰 섰다. 이 시즌을 끝으로 모기업 <a class="mw-disambig" href="/wiki/SBS" title="SBS">SBS</a>는 <a href="/wiki/2005%EB%85%84" title="2005년">2005년</a> <a href="/wiki/8%EC%9B%94_17%EC%9D%BC" title="8월 17일">8월 17일</a> 인수 대금 27억원에 <a href="/wiki/KT%26G" title="KT&amp;G">KT&amp;G</a>로 농구단을 매각하게 되어 안양 SBS 스타즈는 역사 속으로 사라지고 안양 KT&amp;G 카이츠라는 팀명으로 재창단하였다.<sup class="reference" id="cite_ref-2"><a href="#cite_note-2">[2]</a></sup></li>
</ul>
<h3><span class="mw-headline" id="2005.EB.85.84_.EC.95.88.EC.96.91_KT.26G_.EC.B9.B4.EC.9D.B4.EC.B8.A0_.EC.B0.BD.EB.8B.A8">2005년 안양 KT&amp;G 카이츠 창단</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%95%88%EC%96%91_KGC%EC%9D%B8%EC%82%BC%EA%B3%B5%EC%82%AC_%EB%86%8D%EA%B5%AC%EB%8B%A8&amp;action=edit&amp;section=3" title="부분 편집: 2005년 안양 KT&amp;G 카이츠 창단">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<ul>
<li>2005-06 시즌 9할대 승률 1위를 예상하는 자도 적지 않았다. 그러나 3라운드의 부진을 결국 떨치지 못하고 승률 5할을 기록하고도 플레이오프 진출에 실패하였다.</li>
</ul>
<h4><span class="mw-headline" id="2005-06_.EC.8B.9C.EC.A6.8C_.EC.95.88.EC.96.91_KT.26G_.EC.B9.B4.EC.9D.B4.EC.B8.A0.EC.9D.98_3.EB.9D.BC.EC.9A.B4.EB.93.9C_.EB.B6.80.EC.A7.84.EC.9D.98_.EC.9D.B4.EC.9C.A0">2005-06 시즌 안양 KT&amp;G 카이츠의 3라운드 부진의 이유</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%95%88%EC%96%91_KGC%EC%9D%B8%EC%82%BC%EA%B3%B5%EC%82%AC_%EB%86%8D%EA%B5%AC%EB%8B%A8&amp;action=edit&amp;section=4" title="부분 편집: 2005-06 시즌 안양 KT&amp;G 카이츠의 3라운드 부진의 이유">편집</a><span class="mw-editsection-bracket">]</span></span></h4>
<p>당시의 용병은 허브 래미자냐와 <a class="new" href="/w/index.php?title=%EB%8B%A8%ED%85%8C_%EC%A1%B4%EC%8A%A4&amp;action=edit&amp;redlink=1" title="단테 존스 (없는 문서)">단테 존스</a>였다. 래미자냐는 100kg도 되지 않아서 거구들에게 밀렸다. 게다가 단테 존스가 점점 개인적으로 변해가고 있었다. 그런 가운데에서 <a href="/wiki/2005%EB%85%84" title="2005년">2005년</a> <a href="/wiki/12%EC%9B%94_3%EC%9D%BC" title="12월 3일">12월 3일</a> 꼴찌 팀이었던 <a class="mw-redirect" href="/wiki/%EC%9D%B8%EC%B2%9C_%EC%A0%84%EC%9E%90%EB%9E%9C%EB%93%9C_%EB%B8%94%EB%9E%99%EC%8A%AC%EB%9E%98%EB%A8%B8" title="인천 전자랜드 블랙슬래머">인천 전자랜드 블랙슬래머</a>에 대패한 것을 시작으로 안양 KT&amp;G 카이츠의 폭락은 시작되었다. <a href="/wiki/12%EC%9B%94_17%EC%9D%BC" title="12월 17일">12월 17일</a> <a href="/wiki/%EC%A0%84%EC%A3%BC_KCC_%EC%9D%B4%EC%A7%80%EC%8A%A4" title="전주 KCC 이지스">전주 KCC 이지스</a>에 승리를 거둔 뒤 6연패에 빠졌다. 급기야는 인천 전자랜드 블랙슬래머에 밀려 꼴찌된다는 말이 나올 정도였다. 그나마 후반에 분발해서 5할 승률을 기록하면서 다음 시즌을 기약할 수 있었다.</p>
<h3><span class="mw-headline" id="2006-07_.EC.8B.9C.EC.A6.8C.2C_2007-08.EC.8B.9C.EC.A6.8C">2006-07 시즌, 2007-08시즌</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%95%88%EC%96%91_KGC%EC%9D%B8%EC%82%BC%EA%B3%B5%EC%82%AC_%EB%86%8D%EA%B5%AC%EB%8B%A8&amp;action=edit&amp;section=5" title="부분 편집: 2006-07 시즌, 2007-08시즌">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>2006-07 시즌 개막후 16번째 경기까지 6승 10패라는 실망스런 성적으로 인해 김동광 감독은 안양 KT&amp;G 카이츠에서 사퇴한다. 이후 <a href="/wiki/%EA%B9%80%EC%83%81%EC%8B%9D_(%EB%86%8D%EA%B5%AC%EC%9D%B8)" title="김상식 (농구인)">김상식</a> 감독의 대행체제로 가다 <a href="/wiki/%EC%9C%A0%EB%8F%84%ED%9B%88" title="유도훈">유도훈</a> 감독이 취임한다. 안양 KT&amp;G 카이츠는 점점 상승세로 돌아서더니 결국 SK와 동부 전자랜드를 따돌리고 6위로 플레이 오프를 진출한다. 그러나 <a class="mw-redirect" href="/wiki/%EB%B6%80%EC%82%B0_KTF_%EB%A7%A4%EC%A7%81%EC%9C%99%EC%8A%A4" title="부산 KTF 매직윙스">부산 KTF 매직윙스</a>의 벽에 막혔다.</p>
<p>2007-08 시즌에는 <a class="new" href="/w/index.php?title=%EB%8B%A8%ED%85%8C_%EC%A1%B4%EC%8A%A4&amp;action=edit&amp;redlink=1" title="단테 존스 (없는 문서)">단테 존스</a>도 떠났다. KT&amp;G는 그렇게 하위권 시즌이 예상되었으나 4위, 4강 진출을 일구어 냈다. 6강 플레이오프에서 <a href="/wiki/%EB%B0%A9%EC%84%B1%EC%9C%A4" title="방성윤">방성윤</a>과 <a href="/wiki/%EA%B9%80%ED%83%9C%EC%88%A0" title="김태술">김태술</a>이 이끄는 <a href="/wiki/%EC%84%9C%EC%9A%B8_SK_%EB%82%98%EC%9D%B4%EC%B8%A0" title="서울 SK 나이츠">서울 SK 나이츠</a>를 격파하고, 4강에서 <a href="/wiki/%EA%B9%80%EC%A3%BC%EC%84%B1_(%EB%86%8D%EA%B5%AC_%EC%84%A0%EC%88%98)" title="김주성 (농구 선수)">김주성</a>이 이끄는 <a href="/wiki/%EC%9B%90%EC%A3%BC_%EB%8F%99%EB%B6%80_%ED%94%84%EB%A1%9C%EB%AF%B8" title="원주 동부 프로미">원주 동부 프로미</a>에 무릎을 꿇는다. 4강 플레이오프 직전 팀의 핵심 식스맨인 <a class="new" href="/w/index.php?title=%EA%B9%80%EC%9D%BC%EB%91%90_(%EB%86%8D%EA%B5%AC_%EC%84%A0%EC%88%98)&amp;action=edit&amp;redlink=1" title="김일두 (농구 선수) (없는 문서)">김일두</a>가 연습중에 부상을 당했는데, 후에 <a href="/wiki/%EC%A3%BC%ED%9D%AC%EC%A0%95" title="주희정">주희정</a>은 '김일두가 부상을 당하지 않았으면 플레이오프의 판도는 바뀌었을 것이다.' 라고 말했다. 당시 외국인 선수는 마퀸 챈들러와 TJ 커밍스였으며 거의 풀타임을 소화하는 주희정을 중심으로 안양 KT&amp;G 카이츠는 속공과 조직력으로 중상위권의 성적을 일구어냈다. <a href="/wiki/%EC%96%91%ED%9D%AC%EC%A2%85" title="양희종">양희종</a> 등 신인도 많이 배출되었다.</p>
<h3><span class="mw-headline" id="2008-09.EC.8B.9C.EC.A6.8C">2008-09시즌</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%95%88%EC%96%91_KGC%EC%9D%B8%EC%82%BC%EA%B3%B5%EC%82%AC_%EB%86%8D%EA%B5%AC%EB%8B%A8&amp;action=edit&amp;section=6" title="부분 편집: 2008-09시즌">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>시즌 전 <a href="/wiki/%EC%9C%A0%EB%8F%84%ED%9B%88" title="유도훈">유도훈</a> 감독이 갑작스럽게 사퇴하면서, <a href="/wiki/%EC%9D%B4%EC%83%81%EB%B2%94_(%EB%86%8D%EA%B5%AC%EC%9D%B8)" title="이상범 (농구인)">이상범</a> 코치가 감독대행 체제로 되었다. 당시의 베스트5는 <a href="/wiki/%EC%A3%BC%ED%9D%AC%EC%A0%95" title="주희정">주희정</a>, <a href="/wiki/%ED%99%A9%EC%A7%84%EC%9B%90" title="황진원">황진원</a>, <a href="/wiki/%EC%96%91%ED%9D%AC%EC%A2%85" title="양희종">양희종</a>, 마퀸 챈들러, 캘빈 워너였으며 김일두, <a class="new" href="/w/index.php?title=%EC%8B%A0%EC%A0%9C%EB%A1%9D&amp;action=edit&amp;redlink=1" title="신제록 (없는 문서)">신제록</a> 등이 식스맨으로 활약했다. 시즌 초반에 상위권까지 갔으나, 캘빈 워너의 장기 부상과 대체 용병들의 실망스러운 플레이로 중하위권으로 추락하게 된다. 캘빈 워너의 복귀로 상위권으로 올라갈 것이라 예상되었으나, 캘빈 워너의 마약 혐의로 인해 결국 그 공백을 매우지 못하고 29승 25패, 7위로 시즌을 마감하게 된다.</p>
<p>시즌 전 약체로 분류되었던 안양 KT&amp;G 카이츠를 이끌며 득점, 어시스트, 리바운드 에서 뛰어난 활약을 보인 <a href="/wiki/%EC%A3%BC%ED%9D%AC%EC%A0%95" title="주희정">주희정</a>은 시즌\</p>
<h3><span class="mw-headline" id="2009-10_.EC.8B.9C.EC.A6.8C">2009-10 시즌</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%95%88%EC%96%91_KGC%EC%9D%B8%EC%82%BC%EA%B3%B5%EC%82%AC_%EB%86%8D%EA%B5%AC%EB%8B%A8&amp;action=edit&amp;section=7" title="부분 편집: 2009-10 시즌">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>2008-09 시즌이 끝난 뒤 <a href="/wiki/%EC%A3%BC%ED%9D%AC%EC%A0%95" title="주희정">주희정</a>을 <a href="/wiki/%EC%84%9C%EC%9A%B8_SK_%EB%82%98%EC%9D%B4%EC%B8%A0" title="서울 SK 나이츠">서울 SK 나이츠</a>로부터 <a href="/wiki/%EA%B9%80%ED%83%9C%EC%88%A0" title="김태술">김태술</a>과 <a href="/wiki/%EA%B9%80%EC%A2%85%ED%95%99" title="김종학">김종학</a>을 받는 조건으로 트레이드를 단행한다. 그 후 양희종, 김태술, 김일두, 신제록 등 주전급 선수 대거를 군 입대를 시킨뒤 리빌딩의 시작하였다.</p>
<p><a href="/wiki/2009%EB%85%84" title="2009년">2009년</a> <a href="/wiki/10%EC%9B%94_18%EC%9D%BC" title="10월 18일">10월 18일</a> 개막전 <a href="/wiki/%EC%B0%BD%EC%9B%90_LG_%EC%84%B8%EC%9D%B4%EC%BB%A4%EC%8A%A4" title="창원 LG 세이커스">창원 LG 세이커스</a>전부터 삐걱대더니, <a href="/wiki/10%EC%9B%94_21%EC%9D%BC" title="10월 21일">10월 21일</a>에 <a href="/wiki/%EC%9D%B8%EC%B2%9C_%EC%A0%84%EC%9E%90%EB%9E%9C%EB%93%9C_%EC%97%98%EB%A6%AC%ED%8E%80%EC%B8%A0" title="인천 전자랜드 엘리펀츠">인천 전자랜드 엘리펀츠</a>를 상대로 승리하게 되나, 이후 5연패를 한다. <a href="/wiki/11%EC%9B%94_12%EC%9D%BC" title="11월 12일">11월 12일</a>엔 <a href="/wiki/%EC%9D%B4%ED%98%84%ED%98%B8_(%EB%86%8D%EA%B5%AC_%EC%84%A0%EC%88%98)" title="이현호 (농구 선수)">이현호</a>, <a class="mw-disambig" href="/wiki/%EC%9D%B4%EC%83%81%EC%A4%80" title="이상준">이상준</a>, 라샤드 벨을 인천 전자랜드 엘리펀츠로 보내고 <a href="/wiki/%EA%B9%80%EC%84%B1%EC%B2%A0_(%EB%86%8D%EA%B5%AC%EC%9D%B8)" title="김성철 (농구인)">김성철</a>, 크리스 다니엘스를 받아오는 2대3 트레이드를 단행한다. <a href="/wiki/2009%EB%85%84" title="2009년">2009년</a> <a href="/wiki/12%EC%9B%94_11%EC%9D%BC" title="12월 11일">12월 11일</a>에는 나이젤 딕슨을 <a class="mw-redirect mw-disambig" href="/wiki/%EB%B6%80%EC%82%B0_KT_%EC%86%8C%EB%8B%89%EB%B6%90" title="부산 KT 소닉붐">부산 KT 소닉붐</a>으로 보내고 도널드 리틀과 <a href="/wiki/2010%EB%85%84" title="2010년">2010년</a> 신인 드래프트 1라운드 지명권을 받는 조건으로 트레이드를 하게된다, 나이젤 딕슨 트레이드로 받아온 신인선수 1라운드 지명권은 2010년 드래프트에서 KBL 사상 최초로 전체 1순위, 2순위 지명권을 받게되는 행운을 얻게 된다. 이 드래프트에서 1순위로 <a href="/wiki/%EA%B2%BD%ED%9D%AC%EB%8C%80%ED%95%99%EA%B5%90" title="경희대학교">경희대학교</a>의 <a href="/wiki/%EB%B0%95%EC%B0%AC%ED%9D%AC_(%EB%86%8D%EA%B5%AC_%EC%84%A0%EC%88%98)" title="박찬희 (농구 선수)">박찬희</a>를 지명하였고 2순위로 <a href="/wiki/%EC%97%B0%EC%84%B8%EB%8C%80%ED%95%99%EA%B5%90" title="연세대학교">연세대학교</a>의 이정현을 지명하게 된다. 그 후 부진에 시달리다가 <a href="/wiki/12%EC%9B%94_29%EC%9D%BC" title="12월 29일">12월 29일</a>에 가까스로 2연승을 한다. 안양 KT&amp;G 카이츠는 16승 38패 8위로 시즌을 마감하였다. 이 시즌을 끝으로 안양 KT&amp;G 카이츠는 모기업 변경으로 안양 한국인삼공사 농구단으로 팀명을 변경했다.</p>
<h3><span class="mw-headline" id="2010-11_.EC.8B.9C.EC.A6.8C">2010-11 시즌</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%95%88%EC%96%91_KGC%EC%9D%B8%EC%82%BC%EA%B3%B5%EC%82%AC_%EB%86%8D%EA%B5%AC%EB%8B%A8&amp;action=edit&amp;section=8" title="부분 편집: 2010-11 시즌">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>안양 한국인삼공사 프로농구단으로 팀명을 변경한 뒤 시범경기에서 <a href="/wiki/10%EC%9B%94_4%EC%9D%BC" title="10월 4일">10월 4일</a> <a href="/wiki/%EC%A0%84%EC%A3%BC_KCC_%EC%9D%B4%EC%A7%80%EC%8A%A4" title="전주 KCC 이지스">전주 KCC 이지스</a>에 패하고, <a href="/wiki/10%EC%9B%94_6%EC%9D%BC" title="10월 6일">10월 6일</a> <a href="/wiki/%EC%84%9C%EC%9A%B8_%EC%82%BC%EC%84%B1_%EC%8D%AC%EB%8D%94%EC%8A%A4" title="서울 삼성 썬더스">서울 삼성 썬더스</a>를 이겼다. 리빌딩이 한창 진행 중인 팀이었기 때문에 팀 성적은 좋지 못했다, 결국 16승 38패를 거두며 9위로 2010-11 시즌을 마무리지었다.</p>
<h3><span class="mw-headline" id="2011-12_.EC.8B.9C.EC.A6.8C">2011-12 시즌</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%95%88%EC%96%91_KGC%EC%9D%B8%EC%82%BC%EA%B3%B5%EC%82%AC_%EB%86%8D%EA%B5%AC%EB%8B%A8&amp;action=edit&amp;section=9" title="부분 편집: 2011-12 시즌">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>팀명을 한 시즌만에 안양 KGC인삼공사로 교체하고, 기존의 <a href="/wiki/%EB%B0%95%EC%B0%AC%ED%9D%AC_(%EB%86%8D%EA%B5%AC_%EC%84%A0%EC%88%98)" title="박찬희 (농구 선수)">박찬희</a>, <a class="mw-disambig" href="/wiki/%EC%9D%B4%EC%A0%95%ED%98%84" title="이정현">이정현</a>과 함께 군에서 전역한 <a href="/wiki/%EC%96%91%ED%9D%AC%EC%A2%85" title="양희종">양희종</a>, 김일두, <a href="/wiki/%EA%B9%80%ED%83%9C%EC%88%A0" title="김태술">김태술</a>과 드래프트 1순위의 <a href="/wiki/%EC%98%A4%EC%84%B8%EA%B7%BC" title="오세근">오세근</a>을 영입하여 빠른 농구를 앞세워 정규리그에서 <a href="/wiki/1997%EB%85%84" title="1997년">1997년</a> 원년 이후 15년 만에 정규리그 준우승을 하였고, 4강 플레이오프에서 정규리그 3위 <a class="mw-redirect mw-disambig" href="/wiki/%EB%B6%80%EC%82%B0_KT_%EC%86%8C%EB%8B%89%EB%B6%90" title="부산 KT 소닉붐">부산 KT 소닉붐</a>을 이기며 <a href="/wiki/KBL_%EC%B1%94%ED%94%BC%EC%96%B8_%EA%B2%B0%EC%A0%95%EC%A0%84" title="KBL 챔피언 결정전">KBL 챔피언 결정전</a>에 진출하였다. 당시 2006-07 시즌부터 2위팀은 3위팀에게 발목이 잡혔는데, KGC는 6년 만에 2위 팀의 챔프전 진출 및 팀 창단 첫 진출을 하였다. 챔피언 결정전에서 정규리그 우승팀인 <a href="/wiki/%EC%9B%90%EC%A3%BC_%EB%8F%99%EB%B6%80_%ED%94%84%EB%A1%9C%EB%AF%B8" title="원주 동부 프로미">원주 동부 프로미</a>었다 첫판에는 패했으나 2차전 승리 3차전에는 접전끝에 1점차로 패했고 나머지 경기는 전승으로 4승 2패의 성적으로 팀 창단 첫 우승을 하였다. <a href="/wiki/%EC%98%A4%EC%84%B8%EA%B7%BC" title="오세근">오세근</a>은 플레이오프 MVP를 받았으며, SBS 시절부터 한 팀에 몸담았던 프랜차이즈 스타 출신 <a href="/wiki/%EC%9D%B4%EC%83%81%EB%B2%94_(%EB%86%8D%EA%B5%AC%EC%9D%B8)" title="이상범 (농구인)">이상범</a> 감독은 첫 우승을 안았다.</p>
<h3><span class="mw-headline" id="2012-13_.EC.8B.9C.EC.A6.8C">2012-13 시즌</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%95%88%EC%96%91_KGC%EC%9D%B8%EC%82%BC%EA%B3%B5%EC%82%AC_%EB%86%8D%EA%B5%AC%EB%8B%A8&amp;action=edit&amp;section=10" title="부분 편집: 2012-13 시즌">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>안양 KGC인삼공사의 첫 우승을 견인했던 <a href="/wiki/%EC%98%A4%EC%84%B8%EA%B7%BC" title="오세근">오세근</a>이 시즌 내내 출전하지 못했고, 김일두와 <a class="mw-disambig" href="/wiki/%EC%9D%B4%EC%A0%95%ED%98%84" title="이정현">이정현</a>까지 부상으로 신음하면서 6연패도 하는 등 시즌 내내 중위권에 머물렀다. 하지만 용병 <a class="new" href="/w/index.php?title=%ED%82%A4%EB%B8%8C%EC%9B%A8_%ED%8A%B8%EB%A6%BC&amp;action=edit&amp;redlink=1" title="키브웨 트림 (없는 문서)">키브웨 트림</a>과 <a href="/wiki/%ED%9B%84%EC%95%88_%ED%8C%8C%ED%8B%B8%EB%A1%9C" title="후안 파틸로">후안 파틸로</a>가 분전하고 <a href="/wiki/%EC%A0%95%ED%9C%98%EB%9F%89" title="정휘량">정휘량</a>, <a class="new" href="/w/index.php?title=%EC%B5%9C%ED%98%84%EB%AF%BC&amp;action=edit&amp;redlink=1" title="최현민 (없는 문서)">최현민</a> 등의 활약, 그리고 근성있는 플레이가 돋보이면서 중위권에서 버텨 나갔고, 결국 4위로 6강 플레이오프에 진출하였다. 6강 플레이오프에서는 <a class="mw-redirect" href="/wiki/%EA%B3%A0%EC%96%91_%EC%98%A4%EB%A6%AC%EC%98%A8%EC%8A%A4" title="고양 오리온스">고양 오리온스</a>를 상대해서 3승 2패로 4강 플레이오프에서 <a href="/wiki/%EC%84%9C%EC%9A%B8_SK_%EB%82%98%EC%9D%B4%EC%B8%A0" title="서울 SK 나이츠">서울 SK 나이츠</a>와 맞대결을 펼쳤는데, <a href="/wiki/%EC%9E%A0%EC%8B%A4%ED%95%99%EC%83%9D%EC%B2%B4%EC%9C%A1%EA%B4%80" title="잠실학생체육관">잠실학생체육관</a>에서는 1승 1패를 거두었지만 <a href="/wiki/%EC%95%88%EC%96%91%EC%B2%B4%EC%9C%A1%EA%B4%80" title="안양체육관">안방</a>에서 2패를 당하면서 시즌을 마감하였다.</p>
<h3><span class="mw-headline" id="2013-14_.EC.8B.9C.EC.A6.8C">2013-14 시즌</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%95%88%EC%96%91_KGC%EC%9D%B8%EC%82%BC%EA%B3%B5%EC%82%AC_%EB%86%8D%EA%B5%AC%EB%8B%A8&amp;action=edit&amp;section=11" title="부분 편집: 2013-14 시즌">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p><a href="/wiki/%EC%98%A4%EC%84%B8%EA%B7%BC" title="오세근">오세근</a>이 돌아왔지만 외국인 농사를 망쳤고, <a href="/wiki/%EA%B9%80%ED%83%9C%EC%88%A0" title="김태술">김태술</a>등이 부상으로 신음하면서 12월 중순에는 8연패에 빠지면서 10위까지 추락했다. 1월에 약간 분전하는 듯 하지만 2월에 다시 부진하면서 결국 <a href="/wiki/%EC%9D%B4%EC%83%81%EB%B2%94_(%EB%86%8D%EA%B5%AC%EC%9D%B8)" title="이상범 (농구인)">이상범</a> 감독이 사퇴하고 이동남 감독대행으로 교체하는 등 9위로 시즌을 마쳤다.</p>
<p>시즌 후 <a href="/wiki/%EA%B9%80%ED%83%9C%EC%88%A0" title="김태술">김태술</a>이<a href="/wiki/%EA%B0%95%EB%B3%91%ED%98%84" title="강병현">강병현</a>,<a class="new" href="/w/index.php?title=%EC%9E%A5%EB%AF%BC%EA%B5%AD&amp;action=edit&amp;redlink=1" title="장민국 (없는 문서)">장민국</a>과 트레이드됐고, 김일두가<a class="new" href="/w/index.php?title=%ED%95%98%EC%9E%AC%ED%95%84&amp;action=edit&amp;redlink=1" title="하재필 (없는 문서)">하재필</a>과 트레이드됐다.</p>
<h3><span class="mw-headline" id="2014-15_.EC.8B.9C.EC.A6.8C">2014-15 시즌</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%95%88%EC%96%91_KGC%EC%9D%B8%EC%82%BC%EA%B3%B5%EC%82%AC_%EB%86%8D%EA%B5%AC%EB%8B%A8&amp;action=edit&amp;section=12" title="부분 편집: 2014-15 시즌">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>시즌 후 전창진 감독을 새로운 사령탑으로 영입하였고 김승기 손규완 코치도 전창진 감독을 따라 안양KGC인삼공사로 이적하였다. 서울 삼성의 유성호를 장민국과 트레이드를 통해 영입하였다. 그러나 그 와중에 <a href="/wiki/%EC%A0%84%EC%B0%BD%EC%A7%84" title="전창진">전창진</a> 감독이 2015년 승부조작에 연루되어 <a class="mw-disambig" href="/wiki/KBL" title="KBL">KBL</a>에서 영구제명을 당하였고, 이후 <a href="/wiki/%EC%98%A4%EC%84%B8%EA%B7%BC" title="오세근">오세근</a>과 <a class="new" href="/w/index.php?title=%EC%A0%84%EC%84%B1%ED%98%84&amp;action=edit&amp;redlink=1" title="전성현 (없는 문서)">전성현</a>이 도박 사건에 연루되는 등 큰 악재를 맞았으나 <a href="/wiki/%EC%A0%84%EC%B0%BD%EC%A7%84" title="전창진">전창진</a>을 대신하여 <a href="/wiki/%EA%B9%80%EC%8A%B9%EA%B8%B0_(%EB%86%8D%EA%B5%AC%EC%9D%B8)" title="김승기 (농구인)">김승기</a> 수석코치가 감독대행이 되었다. 김승기 대행은 개막 4연패 이후 <a href="/wiki/%EC%9B%90%EC%A3%BC_%EB%8F%99%EB%B6%80_%ED%94%84%EB%A1%9C%EB%AF%B8" title="원주 동부 프로미">동부</a>전 원정경기를 1점차 승리로 장식하며 빠르게 팀을 수습하였고, 이후 최근 11경기에서 7승 4패로 드래프트가 열리기 전 16경기에서 8승 8패, 소기의 목적을 달성하였다. <a href="/wiki/10%EC%9B%94_26%EC%9D%BC" title="10월 26일">10월 26일</a> 열린 드래프트에서 1라운드 1순위 <a href="/wiki/%EB%AC%B8%EC%84%B1%EA%B3%A4" title="문성곤">문성곤</a>을 지명하여 기존 <a class="mw-disambig" href="/wiki/%EB%B0%95%EC%B0%AC%ED%9D%AC" title="박찬희">박찬희</a>, <a class="new" href="/w/index.php?title=%EC%9D%B4%EC%A0%95%ED%98%84_(%EB%86%8D%EA%B5%AC_%EC%84%A0%EC%88%98)&amp;action=edit&amp;redlink=1" title="이정현 (농구 선수) (없는 문서)">이정현</a>, <a href="/wiki/%EA%B0%95%EB%B3%91%ED%98%84" title="강병현">강병현</a> 등과 함께 <a href="/wiki/%EC%98%A4%EC%84%B8%EA%B7%BC" title="오세근">오세근</a>, <a class="new" href="/w/index.php?title=%EC%A0%84%EC%84%B1%ED%98%84&amp;action=edit&amp;redlink=1" title="전성현 (없는 문서)">전성현</a>의 공백에 대한 불안감을 해소하였다. <a href="/wiki/2015%EB%85%84" title="2015년">2015년</a> <a href="/wiki/12%EC%9B%94_31%EC%9D%BC" title="12월 31일">12월 31일</a> <a href="/wiki/%EA%B9%80%EC%8A%B9%EA%B8%B0_(%EB%86%8D%EA%B5%AC%EC%9D%B8)" title="김승기 (농구인)">김승기</a>가 정식 감독으로 승격된다.</p>
<p>2016-17 시즌을 앞두고 박찬희가 한희원과의 트레이드로 <a href="/wiki/%EC%9D%B8%EC%B2%9C_%EC%A0%84%EC%9E%90%EB%9E%9C%EB%93%9C_%EC%97%98%EB%A6%AC%ED%8E%80%EC%B8%A0" title="인천 전자랜드 엘리펀츠">인천 전자랜드 엘리펀츠</a>로 이적하였다.</p>
