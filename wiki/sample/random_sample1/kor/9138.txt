<p><a href="/wiki/1995%EB%85%84" title="1995년">1995년</a> <a class="mw-redirect" href="/wiki/%EC%97%B0%EA%B7%B9%EB%B0%B0%EC%9A%B0" title="연극배우">연극배우</a>로 처음 데뷔하였다. <a href="/wiki/1996%EB%85%84" title="1996년">1996년</a> <a class="mw-redirect" href="/wiki/MBC_%EB%AC%B8%ED%99%94%EB%B0%A9%EC%86%A1" title="MBC 문화방송">MBC 문화방송</a> 공채 탤런트 25기를 통해서 정식으로 데뷔하였으며 <a href="/wiki/2000%EB%85%84" title="2000년">2000년</a> <a href="/wiki/%EC%98%81%ED%99%94" title="영화">영화</a>《찍히면 죽는다》의 주연을 통해 <a href="/wiki/%EC%98%81%ED%99%94%EB%B0%B0%EC%9A%B0" title="영화배우">영화배우</a>로 데뷔하였고 이듬해 <a href="/wiki/2001%EB%85%84" title="2001년">2001년</a> <a class="mw-redirect" href="/wiki/%EB%AE%A4%EC%A7%80%EC%BB%AC_%EB%B0%B0%EC%9A%B0" title="뮤지컬 배우">뮤지컬 배우</a>로 데뷔하였다.</p>
<p><a href="/wiki/2007%EB%85%84" title="2007년">2007년</a>부터는 대학교에서 강의를 하기도 하였다.</p>
<p><a href="/wiki/2008%EB%85%84" title="2008년">2008년</a> <a href="/wiki/8%EC%9B%94_25%EC%9D%BC" title="8월 25일">8월 25일</a> <a href="/wiki/%EC%9E%90%EC%82%B4" title="자살">자살</a>로 인해 세상을 떠났다.<sup class="reference" id="cite_ref-commit_suicide_2-0"><a href="#cite_note-commit_suicide-2">[2]</a></sup></p>
<p><a href="/wiki/%EB%8C%80%EC%9B%90%EC%99%B8%EA%B5%AD%EC%96%B4%EA%B3%A0%EB%93%B1%ED%95%99%EA%B5%90" title="대원외국어고등학교">대원외국어고등학교</a>를 거쳐 <a href="/wiki/%EC%84%9C%EC%9A%B8%EB%8C%80%ED%95%99%EA%B5%90" title="서울대학교">서울대학교</a> 공예과를 졸업하였고 뷰티유 엔터테인먼트 대표를 역임하였다. <a href="/wiki/2007%EB%85%84" title="2007년">2007년</a> <a href="/wiki/3%EC%9B%94" title="3월">3월</a>부터는 <a class="mw-redirect" href="/wiki/%EC%A3%BC%EC%84%B1%EB%8C%80%ED%95%99" title="주성대학">주성대학</a> 방송연기영상과 겸임교수로 강단에 섰고,<sup class="reference" id="cite_ref-.EB.A8.B8.EB.8B.88.ED.88.AC.EB.8D.B0.EC.9D.B4_3-0"><a href="#cite_note-.EB.A8.B8.EB.8B.88.ED.88.AC.EB.8D.B0.EC.9D.B4-3">[3]</a></sup> <a href="/wiki/2007%EB%85%84" title="2007년">2007년</a> <a href="/wiki/11%EC%9B%94" title="11월">11월</a> <a class="mw-redirect" href="/wiki/%EC%BD%94%EB%AF%B8%EB%94%94%EC%96%B8" title="코미디언">코미디언</a> <a href="/wiki/%EC%A0%95%EC%84%A0%ED%9D%AC" title="정선희">정선희</a>와 결혼하였다.<sup class="reference" id="cite_ref-.EB.A8.B8.EB.8B.88.ED.88.AC.EB.8D.B0.EC.9D.B4_3-1"><a href="#cite_note-.EB.A8.B8.EB.8B.88.ED.88.AC.EB.8D.B0.EC.9D.B4-3">[3]</a></sup></p>
<p>그는 사업을 진행하고 있던 <a href="/wiki/2008%EB%85%84" title="2008년">2008년</a> <a href="/wiki/5%EC%9B%94" title="5월">5월</a> 말 경, 정선희 발언 파문 이후로 항의전화, 불매운동, 악플 게시물이 쏟아져 한 달 정도 방송을 중단했다. 그 후 6월 말 판매를 재개했을 때의 매출은 평소의 80%정도였다. <a href="/wiki/9%EC%9B%94_8%EC%9D%BC" title="9월 8일">9월 8일</a>, <a href="/wiki/%EC%84%9C%EC%9A%B8%ED%8A%B9%EB%B3%84%EC%8B%9C" title="서울특별시">서울특별시</a> <a href="/wiki/%EB%85%B8%EC%9B%90%EA%B5%AC" title="노원구">노원구</a> 하계동의 한 주택가 골목에 주차된 자신의 차 안에서 자살한 상태로 발견되었다. 정확한 사망일시는 8월 25일.</p>
<h2><span class="mw-headline" id=".EC.82.AC.EB.A7.9D.EC.97.90_.EA.B4.80.ED.95.9C_.EC.9D.B4.EC.95.BC.EA.B8.B0">사망에 관한 이야기</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%95%88%EC%9E%AC%ED%99%98_(%EB%B0%B0%EC%9A%B0)&amp;action=edit&amp;section=2" title="부분 편집: 사망에 관한 이야기">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>정확한 사망일은 8월 25일. 경찰은 우선 타살 가능성은 없다고 보고, 그가 숨진 채 발견되었던 자가용 차량(<a href="/wiki/%EA%B8%B0%EC%95%84_%EC%B9%B4%EB%8B%88%EB%B0%9C" title="기아 카니발">카니발</a>)내에 연소된 <a href="/wiki/%EC%97%B0%ED%83%84" title="연탄">연탄</a> 2개와 유서를 통해 <a class="mw-redirect" href="/wiki/%EC%9D%BC%EC%82%B0%ED%99%94%ED%83%84%EC%86%8C" title="일산화탄소">일산화탄소</a> 가스 중독에 의한 자살로 보고 있다.<sup class="reference" id="cite_ref-4"><a href="#cite_note-4">[4]</a></sup> 죽기 전, '국민 여러분 선희에게 잘 해주세요'란 당부와 먼저 세상을 뜨는 자신을 용서해달라며 부모님에게 전한 글을 남겼다 한다.<sup class="reference" id="cite_ref-5"><a href="#cite_note-5">[5]</a></sup> <a href="/wiki/9%EC%9B%94_24%EC%9D%BC" title="9월 24일">9월 24일</a> <a class="mw-redirect" href="/wiki/%EA%B5%AD%EB%A6%BD%EA%B3%BC%ED%95%99%EC%88%98%EC%82%AC%EC%97%B0%EA%B5%AC%EC%86%8C" title="국립과학수사연구소">국립과학수사연구소</a>의 감식 결과, 유서의 필적은 안재환 본인의 것으로 확인되었다.</p>
<h3><span class="mw-headline" id=".EC.95.85.ED.94.8C">악플</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%95%88%EC%9E%AC%ED%99%98_(%EB%B0%B0%EC%9A%B0)&amp;action=edit&amp;section=3" title="부분 편집: 악플">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p><a class="mw-redirect" href="/wiki/%EC%95%85%ED%94%8C" title="악플">악플</a>과 불매운동 등이 그의 죽음의 원인을 제공했다는 견해도 있다. 정선희 발언 파문 이후 안재환의 <a href="/wiki/%EC%8B%B8%EC%9D%B4%EC%9B%94%EB%93%9C" title="싸이월드">싸이월드</a> 미니홈피에 악플이 많이 달렸던 것이 사실이다. 그의 사망과 관련한 기사들에도 악플이 많이 달렸고, 포털사이트 <a href="/wiki/%EB%84%A4%EC%9D%B4%EB%B2%84" title="네이버">네이버</a>에서는 댓글기능을 일시적으로 정지시키기도 했다.<sup class="reference" id="cite_ref-6"><a href="#cite_note-6">[6]</a></sup> 이러한 악플이 사회적 문제가 되고 있다는 지적이 있으며<sup class="reference" id="cite_ref-7"><a href="#cite_note-7">[7]</a></sup><sup class="reference" id="cite_ref-8"><a href="#cite_note-8">[8]</a></sup>, <a href="/wiki/%EC%A0%84%EC%97%AC%EC%98%A5" title="전여옥">전여옥</a> 등은 그의 홈페이지에 달린 악플에 대하여 비판하였다.<sup class="reference" id="cite_ref-9"><a href="#cite_note-9">[9]</a></sup><sup class="reference" id="cite_ref-10"><a href="#cite_note-10">[10]</a></sup></p>
<h3><span class="mw-headline" id=".ED.83.80.EC.82.B4_.EC.A3.BC.EC.9E.A5">타살 주장</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%95%88%EC%9E%AC%ED%99%98_(%EB%B0%B0%EC%9A%B0)&amp;action=edit&amp;section=4" title="부분 편집: 타살 주장">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>유가족 등 일각에서는 타살 의혹을 제기하고 있다.<sup class="reference" id="cite_ref-commit_suicide_2-1"><a href="#cite_note-commit_suicide-2">[2]</a></sup> 안재환의 셋째 누나 안미선은 <a href="/wiki/%EC%A0%95%EC%84%A0%ED%9D%AC" title="정선희">정선희</a>가 동생과 함께 납치되었다가 풀려났다는 얘기를 정선희로부터 들었다고 주장했고, 시신 발견 3일 전에 받은 문자 메시지를 공개했다. 또한, 정선희가 범인의 얼굴을 봤을 것이며<sup class="reference" id="cite_ref-sbs_26_11-0"><a href="#cite_note-sbs_26-11">[11]</a></sup> 안재환의 죽음에 대해 많은 것을 알고 있을 것이라고<sup class="reference" id="cite_ref-12"><a href="#cite_note-12">[12]</a></sup> 주장하며 경찰에 안재환의 아내인 정선희의 출국금지를 요청했다. 한편, 경찰은 정선희의 출국금지요청을 할 상황은 아니라는 입장이다.<sup class="reference" id="cite_ref-sbs_26_11-1"><a href="#cite_note-sbs_26-11">[11]</a></sup></p>
