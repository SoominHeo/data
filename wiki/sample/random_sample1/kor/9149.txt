<p><b>안토니오 나엘손 마티아스</b>(<span style="font-size: smaller;"><a href="/wiki/%EC%8A%A4%ED%8E%98%EC%9D%B8%EC%96%B4" title="스페인어">스페인어</a>:</span> <span lang="es" xml:lang="es">Antônio Naelson Matias</span>, <a href="/wiki/1976%EB%85%84" title="1976년">1976년</a> <a href="/wiki/2%EC%9B%94_23%EC%9D%BC" title="2월 23일">2월 23일</a> ~ )는 흔히 <b>시나</b>, <b>시냐</b>(Sinha, Zinha)라는 닉네임으로 알려져 있는 <a href="/wiki/%EB%A9%95%EC%8B%9C%EC%BD%94" title="멕시코">멕시코</a>의 <a href="/wiki/%EC%B6%95%EA%B5%AC" title="축구">축구</a> 선수로서, 포지션은 <a class="mw-redirect" href="/wiki/%EA%B3%B5%EA%B2%A9%ED%98%95_%EB%AF%B8%EB%93%9C%ED%95%84%EB%8D%94" title="공격형 미드필더">공격형 미드필더</a> 및 <a href="/wiki/%EB%AF%B8%EB%93%9C%ED%95%84%EB%8D%94#.EC.A4.91.EC.95.99_.EB.AF.B8.EB.93.9C.ED.95.84.EB.8D.94" title="미드필더">중앙 미드필더</a> 이다. 현재 <a href="/wiki/%EB%8D%B0%ED%8F%AC%EB%A5%B4%ED%8B%B0%EB%B3%B4_%ED%86%A8%EB%A3%A8%EC%B9%B4_FC" title="데포르티보 톨루카 FC">데포르티보 톨루카 FC</a>에서 활약하고 있다.</p>
<h2><span class="mw-headline" id=".EC.84.A0.EC.88.98_.EA.B2.BD.EB.A0.A5">선수 경력</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%95%88%ED%86%A0%EB%8B%88%EC%98%A4_%EB%82%98%EC%97%98%EC%86%90&amp;action=edit&amp;section=1" title="부분 편집: 선수 경력">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<h3><span class="mw-headline" id=".ED.81.B4.EB.9F.BD_.EA.B2.BD.EB.A0.A5">클럽 경력</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%95%88%ED%86%A0%EB%8B%88%EC%98%A4_%EB%82%98%EC%97%98%EC%86%90&amp;action=edit&amp;section=2" title="부분 편집: 클럽 경력">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p><a href="/wiki/1995%EB%85%84" title="1995년">1995년</a> <a href="/wiki/%EB%B8%8C%EB%9D%BC%EC%A7%88" title="브라질">브라질</a>의 <a class="mw-redirect" href="/wiki/%EC%95%84%EB%A9%94%EB%A6%AC%EC%B9%B4_%EC%A7%80_%EB%82%98%ED%83%80%EC%9A%B0" title="아메리카 지 나타우">아메리카 지 나타우</a>에서 프로 무대에 데뷔하였으며 <a href="/wiki/1998%EB%85%84" title="1998년">1998년</a> <a href="/wiki/CF_%EB%AA%AC%ED%85%8C%EB%A0%88%EC%9D%B4" title="CF 몬테레이">CF 몬테레이</a>로 이적하여 <a href="/wiki/%EB%A9%95%EC%8B%9C%EC%BD%94" title="멕시코">멕시코</a>에 둥지를 틀었고, 이듬 해 <a href="/wiki/%EB%8D%B0%ED%8F%AC%EB%A5%B4%ED%8B%B0%EB%B3%B4_%ED%86%A8%EB%A3%A8%EC%B9%B4_FC" title="데포르티보 톨루카 FC">데포르티보 톨루카 FC</a>로 이적하였다. 톨루카에서 오랜 기간 활약하며 5번의 리그 우승을 이끌어 팀의 상징적인 선수로 꼽힌다.</p>
