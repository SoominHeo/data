<p><b>안토니오 데 레이바</b>(Antonio de Leyva, 1480년–1536년)는 <a href="/wiki/%EC%9D%B4%ED%83%88%EB%A6%AC%EC%95%84_%EC%A0%84%EC%9F%81" title="이탈리아 전쟁">이탈리아 전쟁</a> 기간의 <a href="/wiki/%ED%95%A9%EC%8A%A4%EB%B6%80%EB%A5%B4%ED%81%AC_%EC%8A%A4%ED%8E%98%EC%9D%B8" title="합스부르크 스페인">스페인</a>의 장군이다. <a class="new" href="/w/index.php?title=%EC%9D%B4%ED%83%88%EB%A6%AC%EC%95%84_%EC%A0%84%EC%9F%81_(1521-26%EB%85%84)&amp;action=edit&amp;redlink=1" title="이탈리아 전쟁 (1521-26년) (없는 문서)">1521-26년 이탈리아 전쟁</a> 동안 그는 <a href="/wiki/%ED%94%84%EB%9E%91%EC%88%98%EC%95%84_1%EC%84%B8_(%ED%94%84%EB%9E%91%EC%8A%A4)" title="프랑수아 1세 (프랑스)">프랑수아 1세</a>의 파비아 공성전에서 파비아를 방어했었으며, 1525년 <a href="/wiki/%ED%8C%8C%EB%B9%84%EC%95%84_%EC%A0%84%ED%88%AC" title="파비아 전투">파비아 전투</a>에 참여했었다. <a class="new" href="/w/index.php?title=%ED%8E%98%EB%A5%B4%EB%82%9C%EB%8F%84_%EB%8D%B0_%EC%95%84%EB%B0%9C%EB%A1%9C%EC%8A%A4&amp;action=edit&amp;redlink=1" title="페르난도 데 아발로스 (없는 문서)">페르난도 데 아발로스</a> 사망 후, 그는 <a href="/wiki/%EC%BD%94%EB%83%91_%EB%8F%99%EB%A7%B9_%EC%A0%84%EC%9F%81" title="코냑 동맹 전쟁">코냑 동맹 전쟁</a>에서 이탈리아내 더많은 지휘권을 지니게 됐고 후에는 <a href="/wiki/%ED%94%84%EB%A1%9C%EB%B0%A9%EC%8A%A4" title="프로방스">프로방스</a> 침공을 시도한후 사망하고 말았다.</p>
<h2><span class="mw-headline" id=".EC.83.9D.EC.95.A0">생애</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%95%88%ED%86%A0%EB%8B%88%EC%98%A4_%EB%8D%B0_%EB%A0%88%EC%9D%B4%EB%B0%94&amp;action=edit&amp;section=1" title="부분 편집: 생애">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>나바라에 속하는 가문 출신인 그는 <a href="/wiki/%EA%B7%B8%EB%9D%BC%EB%82%98%EB%8B%A4" title="그라나다">그라나다</a>에서 발생한 <a class="new" href="/w/index.php?title=%EB%AC%B4%EB%8D%B0%ED%95%98%EB%A5%B4&amp;action=edit&amp;redlink=1" title="무데하르 (없는 문서)">무데하르</a>들의 반란을 진합하는데 어려움을 겪은 <a class="new" href="/w/index.php?title=%EC%95%8C%ED%91%B8%ED%95%98%EB%9D%BC%EC%8A%A4&amp;action=edit&amp;redlink=1" title="알푸하라스 (없는 문서)">알푸하라스</a>에서 군사적 데뷔를 하였고 이후 <i><a href="/wiki/%EA%B3%A4%EC%82%B4%EB%A1%9C_%ED%8E%98%EB%A5%B4%EB%82%9C%EB%8D%B0%EC%8A%A4_%EB%8D%B0_%EC%BD%94%EB%A5%B4%EB%8F%84%EB%B0%94" title="곤살로 페르난데스 데 코르도바">그란 카피탄</a></i> 하 이탈리아내에서 복무했었다(1503년–1504년). 그는 <a href="/wiki/%EC%9D%B4%ED%83%88%EB%A6%AC%EC%95%84_%EC%A0%84%EC%9F%81" title="이탈리아 전쟁">이탈리아 전쟁</a>에서 중요한 부분을 차지했고 <a href="/wiki/%EB%9D%BC%EB%B2%A4%EB%82%98_%EC%A0%84%ED%88%AC_(1512%EB%85%84)" title="라벤나 전투 (1512년)">라벤나 전투</a>(1512년)에서 부상을 입었다. 이후 <a class="new" href="/w/index.php?title=%ED%8E%98%EB%A5%B4%EB%82%9C%EB%8F%84_%EB%8D%B0_%EC%95%84%EB%B0%9C%EB%A1%9C%EC%8A%A4&amp;action=edit&amp;redlink=1" title="페르난도 데 아발로스 (없는 문서)">페스카라 후작</a> 지휘하에서 그는 밀라노 인근과 1524년에 불운했던 <a href="/wiki/%ED%94%84%EB%A1%9C%EB%B0%A9%EC%8A%A4" title="프로방스">프로방스</a> 원정에서 전투를 벌였다.</p>
<p>프로방스 원정 실패 후, 그는 <a href="/wiki/%ED%8C%8C%EB%B9%84%EC%95%84" title="파비아">파비아</a> 수비대 사령관 직을 맡았고 여기에서 <a href="/wiki/%ED%94%84%EB%9E%91%EC%88%98%EC%95%84_1%EC%84%B8_(%ED%94%84%EB%9E%91%EC%8A%A4)" title="프랑수아 1세 (프랑스)">프랑수아 1세</a>가 이끈 프랑스군의 긴 공성을 한결같이 막아내며, 스페인군과 황제군의 합류 시간을 벌었고 그 유명한 <a href="/wiki/%ED%8C%8C%EB%B9%84%EC%95%84_%EC%A0%84%ED%88%AC" title="파비아 전투">파비아 전투</a>에서 승리를 거뒀다.</p>
<p>1525년 그는 밀라노 공국의 황제군 총사령관 자리를 페스카라 후작에게서 이어받았다. 마지막 <a href="/wiki/%EC%8A%A4%ED%8F%AC%EB%A5%B4%EC%B0%A8_%EA%B0%80%EB%AC%B8" title="스포르차 가문">스포르차 가문</a> 공작인 <a class="new" href="/w/index.php?title=%ED%94%84%EB%9E%80%EC%B2%B4%EC%8A%A4%EC%BD%94_2%EC%84%B8_%EC%8A%A4%ED%8F%AC%EB%A5%B4%EC%B0%A8&amp;action=edit&amp;redlink=1" title="프란체스코 2세 스포르차 (없는 문서)">프란체스코 2세</a>가 1535년에 사망하자, 그는 <a href="/wiki/%EC%B9%B4%EB%A5%BC_5%EC%84%B8" title="카를 5세">카를 5세</a>에 의해 밀라노 총독으로 임명되었다.</p>
<p>레이바의 공적을 검토하는 동안, 카를 5세는 일반적인 미늘창병의 옷차림을 하고 나타나 "안토니오 데 레이바의 용감한 테르시오 병사, 카를로스 데 간테"로서 테르시오들의 명부에 기록해달라고 요청했다고 한다.</p>
<p>그는 1536년 프로방스 원정 중에 <a href="/wiki/%ED%86%B5%ED%92%8D" title="통풍">통풍</a>으로 사망했지만, 오늘날 파괴된 밀라노의 산 디오니지(San Dionigi) 교회에 묻혔다.</p>
<p>그의 후손들은 밀라노에서 거주하고 있으며, 가장 중요한 가문 중 하나가 되었다. 그의 후손 중 한 명은 악명높은 <a class="new" href="/w/index.php?title=%EB%A7%88%EB%A6%AC%EC%95%84%EB%82%98_%EB%8D%B0_%EB%A0%88%EB%B9%84%EC%95%84_%EC%9D%B4_%EB%A7%88%EB%A6%AC%EB%85%B8&amp;action=edit&amp;redlink=1" title="마리아나 데 레비아 이 마리노 (없는 문서)">마리아나 데 레비아 이 마리노</a>이다.</p>
