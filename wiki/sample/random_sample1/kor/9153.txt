<p><b>안토니오 베스푸시오 리베르티</b>(<span style="font-size: smaller;"><a href="/wiki/%EC%8A%A4%ED%8E%98%EC%9D%B8%EC%96%B4" title="스페인어">스페인어</a>:</span> <span lang="es" xml:lang="es">Antonio Vespucio Liberti</span>, <a href="/wiki/1902%EB%85%84" title="1902년">1902년</a> ~ <a href="/wiki/1978%EB%85%84" title="1978년">1978년</a> <a href="/wiki/11%EC%9B%94_28%EC%9D%BC" title="11월 28일">11월 28일</a>)은 <a href="/wiki/%EB%A6%AC%EB%B2%84_%ED%94%8C%EB%A0%88%EC%9D%B4%ED%8A%B8" title="리버 플레이트">리버 플레이트</a>의 구단주였으며, 홈 구장인 <a class="mw-redirect" href="/wiki/%EC%95%88%ED%86%A0%EB%8B%88%EC%98%A4_%EB%A6%AC%EB%B2%A0%EB%A5%B4%ED%8B%B0_%EA%B8%B0%EB%85%90_%EA%B2%BD%EA%B8%B0%EC%9E%A5" title="안토니오 리베르티 기념 경기장">안토니오 리베르티 기념 경기장</a>의 이름은 그의 이름을 딴 것이다.</p>
<p>리베르티는 여러차례 구단주를 역임했는데, <a href="/wiki/1933%EB%85%84" title="1933년">1933년</a> ~ <a href="/wiki/1935%EB%85%84" title="1935년">1935년</a>부터 <a href="/wiki/1939%EB%85%84" title="1939년">1939년</a>, <a href="/wiki/1943%EB%85%84" title="1943년">1943년</a> ~ <a href="/wiki/1952%EB%85%84" title="1952년">1952년</a>, <a href="/wiki/1960%EB%85%84" title="1960년">1960년</a> ~ <a href="/wiki/1964%EB%85%84" title="1964년">1964년</a>, 그리고 <a href="/wiki/1966%EB%85%84" title="1966년">1966년</a> ~ <a href="/wiki/1969%EB%85%84" title="1969년">1969년</a>까지 많은 해를 리버 플레이트와 함께 보냈다. 리버 플레이트는 그의 업적을 기려 그들의 홈 구장에 그의 이름을 붙였다.</p>
