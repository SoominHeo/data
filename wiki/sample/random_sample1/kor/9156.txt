<h2><span class="mw-headline" id=".ED.81.B4.EB.9F.BD_.EA.B2.BD.EB.A0.A5">클럽 경력</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%95%88%ED%86%A0%EB%8B%88%EC%9A%B0_%EC%95%8C%EB%B2%A0%EB%A5%B4%ED%88%AC_%EB%B0%94%EC%8A%A4%ED%88%AC%EC%8A%A4_%ED%95%8C%ED%8C%8C%EB%A0%90&amp;action=edit&amp;section=1" title="부분 편집: 클럽 경력">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>베투는 <a href="/wiki/%EB%A6%AC%EC%8A%A4%EB%B3%B8" title="리스본">리스본</a> 출신이다. 지역의 <a href="/wiki/%EC%8A%A4%ED%8F%AC%EB%A5%B4%ED%8C%85_CP" title="스포르팅 CP">스포르팅 리스본</a>의 유소년 팀에서 실패를 거둔 후, (그는 1군에서 단 한시즌을 보냈는데, 그 때 베투는 써드 골키퍼였다.) 인근의 <a class="new" href="/w/index.php?title=%EC%B9%B4%EC%82%AC_%ED%94%BC%EC%95%84_AC&amp;action=edit&amp;redlink=1" title="카사 피아 AC (없는 문서)">카사 피아</a>로 임대되었고, 2004년 6월에 방출당했다.</p>
<p><a href="/wiki/%EC%84%B8%EA%B5%B0%EB%8B%A4%EB%A6%AC%EA%B0%80" title="세군다리가">2부 리그</a> 클럽인 <a href="/wiki/GD_%EC%83%A4%EB%B2%A0%EC%8A%A4" title="GD 샤베스">샤베스</a>와 <a class="new" href="/w/index.php?title=FC_%EB%A7%88%EB%A5%B4%EC%BF%A0&amp;action=edit&amp;redlink=1" title="FC 마르쿠 (없는 문서)">마르쿠</a>에서 한 시즌씩 보냈던 베투는 <a class="new" href="/w/index.php?title=%EC%84%B8%EA%B5%B0%EB%8B%A4%EB%A6%AC%EA%B0%80_2006-07&amp;action=edit&amp;redlink=1" title="세군다리가 2006-07 (없는 문서)">2006-07 시즌</a>에 <a href="/wiki/%EB%A0%88%EC%9D%B4%EC%88%91%EC%9D%B4%EC%8A%A4_SC" title="레이숑이스 SC">레이숑이스</a>에 합류해 <a class="new" href="/w/index.php?title=%EB%A7%88%ED%88%AC%EC%8B%9C%EB%89%B4%EC%8A%A4&amp;action=edit&amp;redlink=1" title="마투시뉴스 (없는 문서)">마투시뉴스</a> 연고의 클럽이 <a href="/wiki/%ED%94%84%EB%A6%AC%EB%A9%94%EC%9D%B4%EB%9D%BC%EB%A6%AC%EA%B0%80" title="프리메이라리가">1부 리그</a>로 승격되게 도왔고, 이 곳에서의 3시즌 동안 단 여섯번의 경기에 빠졌다.</p>
<p>2009년 3월, <a class="new" href="/w/index.php?title=%ED%94%84%EB%A6%AC%EB%A9%94%EC%9D%B4%EB%9D%BC%EB%A6%AC%EA%B0%80_2008-09&amp;action=edit&amp;redlink=1" title="프리메이라리가 2008-09 (없는 문서)">2008-09 시즌</a> 종료 전, 베투는 리그 강호 <a href="/wiki/FC_%ED%8F%AC%EB%A5%B4%ED%88%AC" title="FC 포르투">포르투</a>로 이적하기로 합의를 보았는데, 계약기간은 4년으로 <a href="/wiki/%EC%9C%A0%EB%A1%9C" title="유로">€</a>750,000에 체결되어 6월에 발효되었다.<sup class="reference" id="cite_ref-BETO_1-0"><a href="#cite_note-BETO-1">[1]</a></sup> 그는 두 시즌동안 <a href="/wiki/%EB%B8%8C%EB%9D%BC%EC%A7%88" title="브라질">브라질</a>의 <a href="/wiki/%EC%97%90%EC%9A%B0%ED%86%B5_%EB%8B%A4_%EC%8B%9C%EC%9A%B0%EB%B0%94_%EC%95%84%EB%A3%A8%EB%8B%A4" title="에우통 다 시우바 아루다">에우통</a> 골키퍼의 백업으로 활약했는데, 출전 기회는 <a href="/wiki/%ED%83%80%EC%82%AC_%EB%93%9C_%ED%8F%AC%EB%A5%B4%ED%88%AC%EA%B0%88" title="타사 드 포르투갈">타사 드 포르투갈</a> 경기로 한정되었다. 2011년 5월 22일, <a href="/wiki/%EB%A6%AC%EC%8A%A4%EB%B3%B8" title="리스본">리스본</a>에서 열린 <a class="new" href="/w/index.php?title=2011_%ED%83%80%EC%82%AC_%EB%93%9C_%ED%8F%AC%EB%A5%B4%ED%88%AC%EA%B0%88_%EA%B2%B0%EC%8A%B9%EC%A0%84&amp;action=edit&amp;redlink=1" title="2011 타사 드 포르투갈 결승전 (없는 문서)">타사 드 포르투갈</a> 2-2로 비긴 상황에 <a class="mw-redirect" href="/wiki/%ED%8E%98%EB%84%90%ED%8B%B0%ED%82%A5" title="페널티킥">페널티킥</a>을 선방해내며, 최종 6-2 승리를 이끌어냈다.<sup class="reference" id="cite_ref-2"><a href="#cite_note-2">[2]</a></sup></p>
<p><a class="new" href="/w/index.php?title=%EB%A6%AC%EA%B0%80_I_2011-12&amp;action=edit&amp;redlink=1" title="리가 I 2011-12 (없는 문서)">2011-12 시즌</a>, 베투는 루마니아의 <a href="/wiki/CFR_%ED%81%B4%EB%A3%A8%EC%A7%80" title="CFR 클루지">클루지</a>로 임대되었는데,<sup class="reference" id="cite_ref-3"><a href="#cite_note-3">[3]</a></sup> 그는 팀이 우승을 거둔 이 시즌에 단 7번의 <a href="/wiki/%EB%A6%AC%EA%B0%80_I" title="리가 I">리가 I</a> 경기에 결장했다. 포르투로부터 방출된 그는 이어지는 여름에 모국으로 복귀해 <a href="/wiki/SC_%EB%B8%8C%EB%9D%BC%EA%B0%80" title="SC 브라가">브라가</a>와 2년 계약을 체결했다.<sup class="reference" id="cite_ref-4"><a href="#cite_note-4">[4]</a></sup> 그러나, 다음 <a class="new" href="/w/index.php?title=%EC%9D%B4%EC%A0%81%EC%8B%9C%EC%9E%A5&amp;action=edit&amp;redlink=1" title="이적시장 (없는 문서)">이적시장</a>에서, 그는 <a class="mw-disambig" href="/wiki/%EB%94%94%EC%97%90%EA%B3%A0_%EB%A1%9C%ED%8E%98%EC%8A%A4" title="디에고 로페스">디에고 로페스</a>를 <a href="/wiki/%EB%A0%88%EC%95%8C_%EB%A7%88%EB%93%9C%EB%A6%AC%EB%93%9C_CF" title="레알 마드리드 CF">레알 마드리드</a>로 보내면서 급히 이 포지션을 수혈할 필요가 있는 <a href="/wiki/%EC%8A%A4%ED%8E%98%EC%9D%B8" title="스페인">스페인</a>의 <a href="/wiki/%EC%84%B8%EB%B9%84%EC%95%BC_FC" title="세비야 FC">세비야</a>로 6월까지 임대되었고, 이후로의 완전 이적 가능성이 있었다.<sup class="reference" id="cite_ref-5"><a href="#cite_note-5">[5]</a></sup></p>
<p><a class="new" href="/w/index.php?title=%EC%95%88%EB%93%9C%EB%A0%88%EC%8A%A4_%ED%8C%94%EB%A1%9C%ED%94%84&amp;action=edit&amp;redlink=1" title="안드레스 팔로프 (없는 문서)">안드레스 팔로프</a>의 부상으로 기회를 잡은 베투는 이적 하루만에 <a href="/wiki/%EC%95%88%EB%8B%AC%EB%A3%A8%EC%8B%9C%EC%95%84_%EC%A7%80%EB%B0%A9" title="안달루시아 지방">안달루시아</a> 연고의 클럽 소속으로 데뷔하였으나, <a href="/wiki/%EC%95%84%ED%8B%80%EB%A0%88%ED%8B%B0%EC%BD%94_%EB%A7%88%EB%93%9C%EB%A6%AC%EB%93%9C" title="아틀레티코 마드리드">아틀레티코 마드리드</a>와의 <a class="new" href="/w/index.php?title=%EC%BD%94%ED%8C%8C_%EB%8D%B8_%EB%A0%88%EC%9D%B4_2012-13&amp;action=edit&amp;redlink=1" title="코파 델 레이 2012-13 (없는 문서)">2012-13 시즌</a> <a href="/wiki/%EC%BD%94%ED%8C%8C_%EB%8D%B8_%EB%A0%88%EC%9D%B4" title="코파 델 레이">코파 델 레이</a> 준결승 1차전 원정 경기에서 1-2로 패하였고, 이 중 막판 8분을 9명으로 경기에 임했었다.<sup class="reference" id="cite_ref-6"><a href="#cite_note-6">[6]</a></sup> 그의 첫 <a class="mw-redirect" href="/wiki/%ED%94%84%EB%A6%AC%EB%A9%94%EB%9D%BC%EB%A6%AC%EA%B0%80" title="프리메라리가">라 리가</a> 경기는 2013년 2월 3일, 2-1로 이긴 <a href="/wiki/%EB%9D%BC%EC%9A%94_%EB%B0%94%EC%98%88%EC%B9%B4%EB%85%B8" title="라요 바예카노">라요 바예카노</a>와의 홈경기였다.<sup class="reference" id="cite_ref-7"><a href="#cite_note-7">[7]</a></sup></p>
<p>베투는 이후 영구 이적을 확정지었고,<sup class="reference" id="cite_ref-8"><a href="#cite_note-8">[8]</a></sup> <a href="/wiki/%EC%9A%B0%EB%82%98%EC%9D%B4_%EC%97%90%EB%A9%94%EB%A6%AC" title="우나이 에메리">우나이 에메리</a> 감독 하에 주전이 되었다.<sup class="reference" id="cite_ref-9"><a href="#cite_note-9">[9]</a></sup> 그는 <a class="new" href="/w/index.php?title=%ED%94%84%EB%A6%AC%EB%A9%94%EB%9D%BC%EB%A6%AC%EA%B0%80_2013-14&amp;action=edit&amp;redlink=1" title="프리메라리가 2013-14 (없는 문서)">2013-14 시즌</a>을 통틀어 총 43경기에 나왔고, 이 중 <a href="/wiki/UEFA_%EC%9C%A0%EB%A1%9C%ED%8C%8C%EB%A6%AC%EA%B7%B8_2013-14" title="UEFA 유로파리그 2013-14">우승</a>을 거둔 <a href="/wiki/UEFA_%EC%9C%A0%EB%A1%9C%ED%8C%8C%EB%A6%AC%EA%B7%B8" title="UEFA 유로파리그">UEFA 유로파리그</a> 결승전에서 <a href="/wiki/SL_%EB%B2%A4%ED%94%BC%EC%B9%B4" title="SL 벤피카">벤피카</a>를 상대해 <a href="/wiki/%EC%8A%B9%EB%B6%80%EC%B0%A8%EA%B8%B0" title="승부차기">승부차기</a>에서 <a href="/wiki/%EC%98%A4%EC%8A%A4%EC%B9%B4%EB%A5%B4_%EC%B9%B4%EB%A5%B4%EB%8F%84%EC%86%8C" title="오스카르 카르도소">오스카르 카르도소</a>와 <a href="/wiki/%EB%A1%9C%EB%93%9C%EB%A6%AC%EA%B3%A0_%EB%AA%A8%EB%A0%88%EB%85%B8_%EB%A7%88%EC%B0%A8%EB%8F%84" title="로드리고 모레노 마차도">로드리고</a>의 두 차례 킥을 막아냈다.<sup class="reference" id="cite_ref-10"><a href="#cite_note-10">[10]</a></sup></p>
<h2><span class="mw-headline" id=".EA.B5.AD.EA.B0.80.EB.8C.80.ED.91.9C.ED.8C.80_.EA.B2.BD.EB.A0.A5">국가대표팀 경력</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%95%88%ED%86%A0%EB%8B%88%EC%9A%B0_%EC%95%8C%EB%B2%A0%EB%A5%B4%ED%88%AC_%EB%B0%94%EC%8A%A4%ED%88%AC%EC%8A%A4_%ED%95%8C%ED%8C%8C%EB%A0%90&amp;action=edit&amp;section=2" title="부분 편집: 국가대표팀 경력">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>2009년 6월 10일, 베투는 무득점으로 비긴 <a href="/wiki/%EC%97%90%EC%8A%A4%ED%86%A0%EB%8B%88%EC%95%84_%EC%B6%95%EA%B5%AC_%EA%B5%AD%EA%B0%80%EB%8C%80%ED%91%9C%ED%8C%80" title="에스토니아 축구 국가대표팀">에스토니아</a>와의 <a class="mw-redirect" href="/wiki/%EC%B9%9C%EC%84%A0%EA%B2%BD%EA%B8%B0" title="친선경기">친선경기</a>에서 <a href="/wiki/%ED%8F%AC%EB%A5%B4%ED%88%AC%EA%B0%88_%EC%B6%95%EA%B5%AC_%EA%B5%AD%EA%B0%80%EB%8C%80%ED%91%9C%ED%8C%80" title="포르투갈 축구 국가대표팀">포르투갈 국가대표팀</a> 데뷔전을 치렀다.<sup class="reference" id="cite_ref-BETO_1-1"><a href="#cite_note-BETO-1">[1]</a></sup></p>
<p><a class="new" href="/w/index.php?title=%ED%94%84%EB%A6%AC%EB%A9%94%EC%9D%B4%EB%9D%BC%EB%A6%AC%EA%B0%80_2009-10&amp;action=edit&amp;redlink=1" title="프리메이라리가 2009-10 (없는 문서)">2009-10 시즌</a>에 포르투에서의 시즌을 선발로 마친 그는 <a class="mw-redirect" href="/wiki/%EC%B9%B4%EB%A5%BC%EB%A3%A8%EC%8A%A4_%EC%BC%80%EC%9D%B4%EB%A3%A8%EC%8A%A4" title="카를루스 케이루스">카를루스 케이루스</a> 감독에 의해 <a href="/wiki/2010%EB%85%84_FIFA_%EC%9B%94%EB%93%9C%EC%BB%B5" title="2010년 FIFA 월드컵">2010년 FIFA 월드컵</a> 24인 예비 엔트리에 들어갔고,<sup class="reference" id="cite_ref-11"><a href="#cite_note-11">[11]</a></sup> 남아프리카 공화국에서 열린 본선행 최종 엔트리에 들었으나, 벤치만 지켰다. 그는 브라질에서 열리는 <a href="/wiki/2014%EB%85%84_FIFA_%EC%9B%94%EB%93%9C%EC%BB%B5" title="2014년 FIFA 월드컵">4년 후 대회</a>에서도 명단에 들었고,<sup class="reference" id="cite_ref-12"><a href="#cite_note-12">[12]</a></sup> 6월 22일, <a href="/wiki/%EB%AF%B8%EA%B5%AD_%EC%B6%95%EA%B5%AC_%EA%B5%AD%EA%B0%80%EB%8C%80%ED%91%9C%ED%8C%80" title="미국 축구 국가대표팀">미국</a>과의 2차전에서, 앞서 <a href="/wiki/%EB%8F%85%EC%9D%BC_%EC%B6%95%EA%B5%AC_%EA%B5%AD%EA%B0%80%EB%8C%80%ED%91%9C%ED%8C%80" title="독일 축구 국가대표팀">독일</a>전에서 부상당한 <a href="/wiki/%ED%9B%84%EC%9D%B4_%ED%8C%8C%ED%8A%B8%EB%A6%AC%EC%8B%9C%EC%9A%B0" title="후이 파트리시우">후이 파트리시우</a>를 대신해 <a href="/wiki/FIFA_%EC%9B%94%EB%93%9C%EC%BB%B5" title="FIFA 월드컵">대회</a> 첫 경기를 나갔다.<sup class="reference" id="cite_ref-13"><a href="#cite_note-13">[13]</a></sup></p>
