<p><b>알가르브 지방</b>(<span style="font-size: smaller;"><a href="/wiki/%ED%8F%AC%EB%A5%B4%ED%88%AC%EA%B0%88%EC%96%B4" title="포르투갈어">포르투갈어</a>:</span> <span lang="pt" xml:lang="pt">Algarve</span>)은 <a href="/wiki/%ED%8F%AC%EB%A5%B4%ED%88%AC%EA%B0%88" title="포르투갈">포르투갈</a> 남부 <a href="/wiki/%ED%8C%8C%EB%A3%A8_%ED%98%84" title="파루 현">파루 현</a>을 중심으로 한 지방을 통칭하는 지명으로, 면적은 4,960㎢이다. 그리니치 천문대 기준으로 UTC + 0이며 인구는 426,386명(2007년 기준)이며 인구 밀도는 86명/㎢(2007년 기준)이다</p>

<h2><span class="mw-headline" id=".EC.A7.80.EB.A6.AC">지리</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%95%8C%EA%B0%80%EB%A5%B4%EB%B8%8C_%EC%A7%80%EB%B0%A9&amp;action=edit&amp;section=1" title="부분 편집: 지리">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>알가르브는 알가르브(Al-Gharb: الغرب 아랍어로 서쪽)란 단어를 따 만든 땅으로 <a href="/wiki/%ED%8F%AC%EB%A5%B4%ED%88%AC%EA%B0%88" title="포르투갈">포르투갈</a>의 최남단에 있는 해안가이다. 바다와 인접해 있기 때문에 겨울에는 따뜻하고 여름에는 신선한 기후를 나타내며, 해안에는 경치가 아름다운 곳이 많다.</p>
<h2><span class="mw-headline" id=".EC.97.AD.EC.82.AC">역사</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%95%8C%EA%B0%80%EB%A5%B4%EB%B8%8C_%EC%A7%80%EB%B0%A9&amp;action=edit&amp;section=2" title="부분 편집: 역사">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>역사상에서는 알가르브의 일부가 고대 <a href="/wiki/%EC%95%84%ED%8B%80%EB%9E%80%ED%8B%B0%EC%8A%A4" title="아틀란티스">아틀란티스</a>의 후계자인 <a class="new" href="/w/index.php?title=%ED%83%80%EB%A5%B4%ED%85%8C%EC%86%8C%EC%8A%A4_%EC%99%95%EA%B5%AD&amp;action=edit&amp;redlink=1" title="타르테소스 왕국 (없는 문서)">타르테소스 왕국</a>의 기원지에 속해있었다. 역사의 아버지 <a href="/wiki/%ED%97%A4%EB%A1%9C%EB%8F%84%ED%86%A0%EC%8A%A4" title="헤로도토스">헤로도토스</a>, <a href="/wiki/%EC%8A%A4%ED%8A%B8%EB%9D%BC%EB%B3%B8" title="스트라본">스트라본</a>(Strabōn)등이 알가르베를 언급했으며 로마인에게 정복되기 전까진 <a class="mw-redirect" href="/wiki/%ED%8E%98%EB%8B%88%ED%82%A4%EC%95%84%EC%9D%B8" title="페니키아인">페니키아인</a>이 정착하여 살았다. 카르타고의 지배지에서 <a href="/wiki/%ED%8F%AC%EC%97%90%EB%8B%88_%EC%A0%84%EC%9F%81" title="포에니 전쟁">포에니 전쟁</a>으로 인해 알가르브 지방을 비롯한 <a href="/wiki/%ED%9E%88%EC%8A%A4%ED%8C%8C%EB%8B%88%EC%95%84" title="히스파니아">히스파니아</a>는 <a href="/wiki/%EB%A1%9C%EB%A7%88_%EC%A0%9C%EA%B5%AD" title="로마 제국">로마 제국</a>의 속령이 되었고 5세기경 <a class="mw-redirect" href="/wiki/%EC%84%9C%EA%B3%A0%ED%8A%B8%EC%A1%B1" title="서고트족">서고트족</a>이 알가르브를 점령하였으며 약 250년간 통치하였다. 그러다 8세기 초 이베리아 반도의 이슬람화가 되면서 무슬림의 통치를 받았으며, 마지막으로 <a href="/wiki/1189%EB%85%84" title="1189년">1189년</a> <a href="/wiki/%EB%A0%88%EC%BD%A9%ED%82%A4%EC%8A%A4%ED%83%80" title="레콩키스타">레콩키스타</a>(국토 재정복 운동)으로 인해 <a href="/wiki/%ED%8F%AC%EB%A5%B4%ED%88%AC%EA%B0%88_%EC%99%95%EA%B5%AD" title="포르투갈 왕국">포르투갈 왕국</a>에 최종 합병되었다.</p>
<h2><span class="mw-headline" id=".EC.83.9D.ED.99.9C">생활</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%95%8C%EA%B0%80%EB%A5%B4%EB%B8%8C_%EC%A7%80%EB%B0%A9&amp;action=edit&amp;section=3" title="부분 편집: 생활">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>포르투갈의 유명한 역사적 인물인 <a class="mw-redirect" href="/wiki/%ED%95%AD%ED%95%B4%EC%99%95%EC%9E%90_%EC%97%94%ED%9E%88%ED%81%AC" title="항해왕자 엔히크">항해왕자 엔히크</a>가 1418년경 알가르브의 사그르스 곶에 항해학교를 설립하였으며 알가르브 지방의 중심 도시인 <a href="/wiki/%ED%8C%8C%EB%A3%A8_(%EB%8F%84%EC%8B%9C)" title="파루 (도시)">파루</a>에 <a class="new" href="/w/index.php?title=%EC%95%8C%EA%B0%80%EB%A5%B4%EB%B8%8C_%EB%8C%80%ED%95%99%EA%B5%90&amp;action=edit&amp;redlink=1" title="알가르브 대학교 (없는 문서)">알가르브 대학교</a>가 있다. 포르투갈의 국토회복운동(<a href="/wiki/%EB%A0%88%EC%BD%A9%ED%82%A4%EC%8A%A4%ED%83%80" title="레콩키스타">레콩키스타</a>) 중 가장 마지막에 마지막에 점령한 곳이라 건축, 의상, 언어 등 문화적으로 오랫동안 이 지방을 점령한 <a href="/wiki/%EB%AC%B4%EC%96%B4%EC%9D%B8" title="무어인">무어인</a>의 영향이 많이 남아 있으며 주요 산업은 농업과 어업이 지방 경제의 주축이다. 하지만 최근에는 <a href="/wiki/%EB%8F%85%EC%9D%BC" title="독일">독일</a>, <a href="/wiki/%EC%98%81%EA%B5%AD" title="영국">영국</a>, <a href="/wiki/%EB%84%A4%EB%8D%9C%EB%9E%80%EB%93%9C" title="네덜란드">네덜란드</a> 등에서 일상에서 탈출하여 삶의 질을 높이기 위해 찾아온 사람들이나 아름다운 해변가로 인해 리조트가 많이 세워져 관광업과 관련된 서비스 등이 각광받고 있다.</p>
