<p><b>알갱이</b> 또는 <b>과립</b>(顆粒)은 입자가 서로 자극을 줄 때마다 에너지를 잃음으로써 구성되는 것으로(가장 흔한 예로 알갱이가 충둘할 때 발생하는 <a href="/wiki/%EB%A7%88%EC%B0%B0%EB%A0%A5" title="마찰력">마찰력</a>을 수 있음<sup class="reference" id="cite_ref-1"><a href="#cite_note-1">[1]</a></sup>), 육안으로 보이는 <a href="/wiki/%EA%B3%A0%EC%B2%B4" title="고체">고체</a> 입자의 덩어리이다.</p>
<p>알갱이 물질들의 예로는 <a href="/wiki/%EB%88%88_(%EB%82%A0%EC%94%A8)" title="눈 (날씨)">눈</a>, <a href="/wiki/%EA%B2%AC%EA%B3%BC" title="견과">견과류</a>, <a href="/wiki/%EC%84%9D%ED%83%84" title="석탄">석탄</a>, <a href="/wiki/%EB%AA%A8%EB%9E%98" title="모래">모래</a>, <a href="/wiki/%EC%8C%80" title="쌀">쌀</a>, <a href="/wiki/%EC%BB%A4%ED%94%BC" title="커피">커피</a>, <a href="/wiki/%EC%BD%98%ED%94%8C%EB%A0%88%EC%9D%B4%ED%81%AC" title="콘플레이크">콘플레이크</a>, <a href="/wiki/%EB%B9%84%EB%A3%8C" title="비료">비료</a>, <a class="new" href="/w/index.php?title=%EB%B3%BC_%EB%B2%A0%EC%96%B4%EB%A7%81&amp;action=edit&amp;redlink=1" title="볼 베어링 (없는 문서)">볼 베어링</a> 등이 있다. <a href="/wiki/%EA%B0%80%EB%A3%A8" title="가루">가루</a>는 입자 크기가 작기 때문에 특별한 종류의 알갱이 물질로 간주된다.</p>
