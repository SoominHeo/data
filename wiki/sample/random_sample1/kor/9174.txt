<p><b>알기르다스 미콜라스 브라자우스카스</b>(<span style="font-size: smaller;"><a href="/wiki/%EB%A6%AC%ED%88%AC%EC%95%84%EB%8B%88%EC%95%84%EC%96%B4" title="리투아니아어">리투아니아어</a>:</span> <span lang="lt" xml:lang="lt">Algirdas Mykolas Brazauskas</span>, <a href="/wiki/1932%EB%85%84" title="1932년">1932년</a> <a href="/wiki/9%EC%9B%94_22%EC%9D%BC" title="9월 22일">9월 22일</a> ~ <a href="/wiki/2010%EB%85%84" title="2010년">2010년</a> <a href="/wiki/6%EC%9B%94_26%EC%9D%BC" title="6월 26일">6월 26일</a><sup class="reference" id="cite_ref-brazauskas_1-0"><a href="#cite_note-brazauskas-1">[1]</a></sup>)는 <a href="/wiki/%EB%A6%AC%ED%88%AC%EC%95%84%EB%8B%88%EC%95%84" title="리투아니아">리투아니아</a>의 제5대 대통령이었다.</p>
<h2><span class="mw-headline" id=".EC.9D.BC.EC.83.9D">일생</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%95%8C%EA%B8%B0%EB%A5%B4%EB%8B%A4%EC%8A%A4_%EB%B8%8C%EB%9D%BC%EC%9E%90%EC%9A%B0%EC%8A%A4%EC%B9%B4%EC%8A%A4&amp;action=edit&amp;section=1" title="부분 편집: 일생">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>알기르다스 브라자우스카스는 <a href="/wiki/1932%EB%85%84" title="1932년">1932년</a> <a href="/wiki/%EB%A6%AC%ED%88%AC%EC%95%84%EB%8B%88%EC%95%84" title="리투아니아">리투아니아</a> 북부의 <a class="new" href="/w/index.php?title=%EB%A1%9C%ED%82%A4%EC%8A%88%ED%82%A4%EC%8A%A4&amp;action=edit&amp;redlink=1" title="로키슈키스 (없는 문서)">로키슈키스</a>에서 태어났다. <a href="/wiki/1952%EB%85%84" title="1952년">1952년</a> <a class="new" href="/w/index.php?title=%EC%B9%B4%EC%9A%B0%EB%82%98%EC%8A%A4_%EA%B3%B5%EA%B3%BC%EB%8C%80%ED%95%99&amp;action=edit&amp;redlink=1" title="카우나스 공과대학 (없는 문서)">카우나스 공과대학</a>을 졸업하였다. <a class="new" href="/w/index.php?title=%EB%A6%AC%ED%88%AC%EC%95%84%EB%8B%88%EC%95%84_%EA%B3%B5%EC%82%B0%EB%8B%B9&amp;action=edit&amp;redlink=1" title="리투아니아 공산당 (없는 문서)">리투아니아 공산당</a>의 당원으로서 일하게 된 것을 시작으로 <a href="/wiki/1965%EB%85%84" title="1965년">1965년</a>에는 건축 자재 산업 장관, <a href="/wiki/1967%EB%85%84" title="1967년">1967년</a> 국가 계획 위원회 부의장, <a href="/wiki/1977%EB%85%84" title="1977년">1977년</a> <a class="new" href="/w/index.php?title=%EB%A6%AC%ED%88%AC%EC%95%84%EB%8B%88%EC%95%84_%EA%B3%B5%EC%82%B0%EB%8B%B9&amp;action=edit&amp;redlink=1" title="리투아니아 공산당 (없는 문서)">리투아니아 공산당</a> 중앙위원회 서기장 등 당과 정부의 요직을 역임했다.</p>
<p><a href="/wiki/%EB%AF%B8%ED%95%98%EC%9D%BC_%EA%B3%A0%EB%A5%B4%EB%B0%94%EC%B4%88%ED%94%84" title="미하일 고르바초프">미하일 고르바초프</a>가 <a href="/wiki/%ED%8E%98%EB%A0%88%EC%8A%A4%ED%8A%B8%EB%A1%9C%EC%9D%B4%EC%B9%B4" title="페레스트로이카">페레스트로이카</a>를 실시하자 <a href="/wiki/%EB%A6%AC%ED%88%AC%EC%95%84%EB%8B%88%EC%95%84" title="리투아니아">리투아니아</a>를 포함한 <a href="/wiki/%EB%B0%9C%ED%8A%B8_3%EA%B5%AD" title="발트 3국">발트 3국</a>에서는 <a href="/wiki/%EC%86%8C%EB%A0%A8" title="소련">소련</a>으로부터 독립하려는 시도를 하였으며, 브라자우스카스는 <a class="new" href="/w/index.php?title=%EB%A6%AC%ED%88%AC%EC%95%84%EB%8B%88%EC%95%84_%EA%B3%B5%EC%82%B0%EB%8B%B9&amp;action=edit&amp;redlink=1" title="리투아니아 공산당 (없는 문서)">리투아니아 공산당</a>의 독립파의 대표로서 활동하였다. <a href="/wiki/1988%EB%85%84" title="1988년">1988년</a> <a href="/wiki/10%EC%9B%94" title="10월">10월</a> <a class="new" href="/w/index.php?title=%EB%A6%AC%ED%88%AC%EC%95%84%EB%8B%88%EC%95%84_%EA%B3%B5%EC%82%B0%EB%8B%B9&amp;action=edit&amp;redlink=1" title="리투아니아 공산당 (없는 문서)">리투아니아 공산당</a> 제1서기로 취임하였다. 제1서기로서의 브라자우스카스는 공산당의 일당 독재나 <a href="/wiki/%EB%AF%BC%EC%A3%BC%EC%A7%91%EC%A4%91%EC%A0%9C" title="민주집중제">민주집중제</a>의 포기를 표명하는 것과 동시에 <a href="/wiki/%EC%86%8C%EB%A0%A8_%EA%B3%B5%EC%82%B0%EB%8B%B9" title="소련 공산당">소련 공산당</a>으로부터의 분리를 추진했다.</p>
<p><a href="/wiki/%EB%A6%AC%ED%88%AC%EC%95%84%EB%8B%88%EC%95%84" title="리투아니아">리투아니아</a>가 독립한 후, <a class="new" href="/w/index.php?title=%EB%A6%AC%ED%88%AC%EC%95%84%EB%8B%88%EC%95%84_%EA%B3%B5%EC%82%B0%EB%8B%B9&amp;action=edit&amp;redlink=1" title="리투아니아 공산당 (없는 문서)">리투아니아 공산당</a>의 독립파가 개명한 <a class="new" href="/w/index.php?title=%EB%AF%BC%EC%A3%BC%EB%85%B8%EB%8F%99%EB%8B%B9_(%EB%A6%AC%ED%88%AC%EC%95%84%EB%8B%88%EC%95%84)&amp;action=edit&amp;redlink=1" title="민주노동당 (리투아니아) (없는 문서)">민주노동당 (리투아니아)</a>의 당원으로 취임한다. 또한 <a href="/wiki/1990%EB%85%84" title="1990년">1990년</a>에는 부총리, <a href="/wiki/1992%EB%85%84" title="1992년">1992년</a>에는 최고 회의 의장 등을 역임하였다. <a href="/wiki/1993%EB%85%84" title="1993년">1993년</a> 대통령제 도입에 따른 선거에서 당선하여 제5대 대통령이 되었다. <a href="/wiki/1998%EB%85%84" title="1998년">1998년</a>에 대통령직에서 물러났다. 그 후 <a href="/wiki/2001%EB%85%84" title="2001년">2001년</a> <a class="mw-disambig" href="/wiki/%EC%82%AC%ED%9A%8C%EB%AF%BC%EC%A3%BC%EB%8B%B9" title="사회민주당">사회민주당</a> 의원을 맡았다. <a href="/wiki/2006%EB%85%84" title="2006년">2006년</a> <a href="/wiki/6%EC%9B%94" title="6월">6월</a>, 연립내의 내분에서 총리직을 사임했다.</p>
<p><a href="/wiki/2010%EB%85%84" title="2010년">2010년</a> <a href="/wiki/6%EC%9B%94_26%EC%9D%BC" title="6월 26일">6월 26일</a>, <a href="/wiki/%EB%B9%8C%EB%89%B4%EC%8A%A4" title="빌뉴스">빌뉴스</a>에서 <a href="/wiki/%EC%A0%84%EB%A6%BD%EC%84%A0%EC%95%94" title="전립선암">전립선암</a>으로 사망했다<sup class="reference" id="cite_ref-brazauskas_1-1"><a href="#cite_note-brazauskas-1">[1]</a></sup>.</p>
