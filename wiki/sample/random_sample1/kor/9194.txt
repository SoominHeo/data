<p><b>알레산드로 파르네세 일 조바네</b>(<span style="font-size: smaller;"><a href="/wiki/%EC%9D%B4%ED%83%88%EB%A6%AC%EC%95%84%EC%96%B4" title="이탈리아어">이탈리아어</a>:</span> <span lang="it" xml:lang="it">Alessandro Farnese il Giovane</span>, <a href="/wiki/1520%EB%85%84" title="1520년">1520년</a> <a href="/wiki/10%EC%9B%94_5%EC%9D%BC" title="10월 5일">10월 5일</a> - <a href="/wiki/1589%EB%85%84" title="1589년">1589년</a> <a href="/wiki/3%EC%9B%94_2%EC%9D%BC" title="3월 2일">3월 2일</a>)는 <a href="/wiki/%EC%9D%B4%ED%83%88%EB%A6%AC%EC%95%84%EC%9D%B8" title="이탈리아인">이탈리아인</a> <a href="/wiki/%EC%B6%94%EA%B8%B0%EA%B2%BD" title="추기경">추기경</a>이자 외교관, 예술의 후원자인 동시에 수집가이다. 그는 <a href="/wiki/%EA%B5%90%ED%99%A9_%EB%B0%94%EC%98%A4%EB%A1%9C_3%EC%84%B8" title="교황 바오로 3세">교황 바오로 3세</a>(또한 이름도 같은 알레산드로 파르네세)의 손자이자 <a href="/wiki/1547%EB%85%84" title="1547년">1547년</a>에 살해당한 초대 <a class="new" href="/w/index.php?title=%ED%8C%8C%EB%A5%B4%EB%A7%88_%EA%B3%B5_%EB%AA%A9%EB%A1%9D&amp;action=edit&amp;redlink=1" title="파르마 공 목록 (없는 문서)">파르마 공</a> <a href="/wiki/%ED%94%BC%EC%97%90%EB%A5%B4_%EB%A3%A8%EC%9D%B4%EC%A7%80_%ED%8C%8C%EB%A5%B4%EB%84%A4%EC%84%B8" title="피에르 루이지 파르네세">피에르 루이지 파르네세</a>의 아들이다.</p>
<h2><span class="mw-headline" id=".EC.9D.BC.EB.8C.80.EA.B8.B0">일대기</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%95%8C%EB%A0%88%EC%82%B0%EB%93%9C%EB%A1%9C_%ED%8C%8C%EB%A5%B4%EB%84%A4%EC%84%B8_(%EC%B6%94%EA%B8%B0%EA%B2%BD)&amp;action=edit&amp;section=1" title="부분 편집: 일대기">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p><a href="/wiki/%EB%B0%9C%EB%A0%8C%ED%83%80%EB%85%B8" title="발렌타노">발렌타노</a>(지금의 <a href="/wiki/%EB%B9%84%ED%85%8C%EB%A5%B4%EB%B3%B4_%ED%98%84" title="비테르보 현">비테르보 현</a>)에 태어난 그는 <a href="/wiki/%EB%B3%BC%EB%A1%9C%EB%83%90" title="볼로냐">볼로냐</a>에서 공부했으며 파르마 교구의 행정관으로 임명되었다. <a href="/wiki/1534%EB%85%84" title="1534년">1534년</a> <a href="/wiki/12%EC%9B%94_18%EC%9D%BC" title="12월 18일">12월 18일</a> 그는 교황 바오로 3세에 의해 산탄젤로 직함의 <a href="/wiki/%EC%B6%94%EA%B8%B0%EA%B2%BD" title="추기경">부제급 추기경</a>으로 서임되었다. 알레산드로 파르네세는 <a href="/wiki/%EC%8B%A0%EC%84%B1_%EB%A1%9C%EB%A7%88_%EC%A0%9C%EA%B5%AD" title="신성 로마 제국">신성 로마 제국</a> 교회의 부대법관, 티볼리의 장관, <a class="mw-redirect" href="/wiki/%EC%82%B0%ED%83%80_%EB%A7%88%EB%A6%AC%EC%95%84_%EB%A7%88%EC%A1%B0%EB%A0%88_%EB%8C%80%EC%84%B1%EC%A0%84" title="산타 마리아 마조레 대성전">산타 마리아 마조레 대성전</a>의 주임 사제, <a class="mw-redirect" href="/wiki/%EC%84%B1_%EB%B2%A0%EB%93%9C%EB%A1%9C_%EB%8C%80%EC%84%B1%EC%A0%84" title="성 베드로 대성전">성 베드로 대성전</a>의 주임 사제, <a href="/wiki/%EC%95%84%EB%B9%84%EB%87%BD" title="아비뇽">아비뇽</a>의 행정관 등 기타 다른 많은 관직과 성직록을 받았다. <a href="/wiki/1536%EB%85%84" title="1536년">1536년</a>에 그는 몬레알레의 주교가 되었으며 <a href="/wiki/1552%EB%85%84" title="1552년">1552년</a>에는 그곳에 <a href="/wiki/%EC%98%88%EC%88%98%ED%9A%8C" title="예수회">예수회</a> 대학을 세웠다. 1538년에는 마사의 교구장, 1553년에는 투르와 카오르의 대교구장이 되었다. <a href="/wiki/1580%EB%85%84" title="1580년">1580년</a>에는 베네벤토의 교구장, 오스티아의 주교급 추기경, <a href="/wiki/%EC%B6%94%EA%B8%B0%EA%B2%BD%ED%9A%8C%EC%9D%98" title="추기경회의">추기경회의</a>의 수석 사제 등이 되었다.</p>
<p>또한 그는 <a class="mw-redirect" href="/wiki/%EA%B5%90%ED%99%A9%ED%8A%B9%EC%82%AC" title="교황특사">교황특사</a>가 되어 끊임없이 서로 적대한 <a class="mw-redirect" href="/wiki/%EC%8B%A0%EC%84%B1_%EB%A1%9C%EB%A7%88_%EC%A0%9C%EA%B5%AD%EC%9D%98_%EC%B9%B4%EB%A5%BC_5%EC%84%B8" title="신성 로마 제국의 카를 5세">신성 로마 제국의 카를 5세</a>와 <a class="mw-redirect" href="/wiki/%ED%94%84%EB%9E%91%EC%8A%A4%EC%9D%98_%ED%94%84%EB%9E%91%EC%88%98%EC%95%84_1%EC%84%B8" title="프랑스의 프랑수아 1세">프랑스의 프랑수아 1세</a>를 중재하여 두 사람 사이에 평화를 가져왔다. <a href="/wiki/1546%EB%85%84" title="1546년">1546년</a>에 그는 <a href="/wiki/%EC%8A%88%EB%A7%90%EC%B9%BC%EB%8D%B4_%EB%8F%99%EB%A7%B9" title="슈말칼덴 동맹">슈말칼덴 동맹</a>을 배척한 카를 5세를 지원하기 위해 교황군을 이끌었다. 1580년의 <a href="/wiki/%EC%BD%98%ED%81%B4%EB%9D%BC%EB%B2%A0" title="콘클라베">콘클라베</a>에서 그는 교황직의 강력한 후보자가 되었지만 낙선하고 말았다. 한편 건축 사업에 몰두하였던 파르네세 추기경은 <a href="/wiki/%EB%A1%9C%EB%A7%88" title="로마">로마</a>에 <a class="new" href="/w/index.php?title=%EC%A0%9C%EC%88%98_%EC%84%B1%EB%8B%B9&amp;action=edit&amp;redlink=1" title="제수 성당 (없는 문서)">제수 성당</a>, <a class="new" href="/w/index.php?title=%EB%B9%8C%EB%9D%BC_%ED%8C%8C%EB%A5%B4%EB%84%A4%EC%84%B8&amp;action=edit&amp;redlink=1" title="빌라 파르네세 (없는 문서)">빌라 파르네세</a>, <a href="/wiki/%ED%8C%8C%EB%A5%B4%EB%84%A4%EC%84%B8_%EA%B6%81%EC%A0%84" title="파르네세 궁전">파르네세 궁전</a>과 같은 건물들을 짓거나 성당들의 보수를 지휘하였다.</p>
<p>수집가이기도 했던 알레산드로 파르네세는 개인적으로 고대 이래의 로마 조각물들을 모아 정리하는 작업을 실행하였다. 이것들은 <a class="new" href="/w/index.php?title=%EB%B6%80%EB%A5%B4%EB%B4%89-%ED%8C%8C%EB%A5%B4%EB%A7%88_%EA%B0%80&amp;action=edit&amp;redlink=1" title="부르봉-파르마 가 (없는 문서)">부르봉-파르마</a> 왕들이 상속받았다가 넘어간 뒤 오늘날에는 대부분이 <a href="/wiki/%EB%82%98%ED%8F%B4%EB%A6%AC" title="나폴리">나폴리</a>에 있다. 그는 또한 살아있는 예술가들의 훌륭한 후원자이기도 했다. 그는 <a class="mw-redirect" href="/wiki/%ED%8B%B0%EC%B9%98%EC%95%84%EB%85%B8_%EB%B2%A0%EC%B2%BC%EB%A6%AC" title="티치아노 베첼리">티치아노</a>, <a class="mw-redirect" href="/wiki/%EB%AF%B8%EC%BC%88%EB%9E%80%EC%A0%A4%EB%A1%9C" title="미켈란젤로">미켈란젤로</a>, <a class="mw-redirect" href="/wiki/%EB%9D%BC%ED%8C%8C%EC%97%98%EB%A1%9C" title="라파엘로">라파엘로</a>가 그린 중요한 그림들을 소장하였다.</p>
<p>알레산드로 파르네세는 <a href="/wiki/1589%EB%85%84" title="1589년">1589년</a>에 예순아홉 살의 나이에 선종하였으며, 시신은 제수 성당의 중앙 <a class="mw-disambig" href="/wiki/%EC%A0%9C%EB%8C%80" title="제대">제대</a> 앞에 묻혔다.</p>
