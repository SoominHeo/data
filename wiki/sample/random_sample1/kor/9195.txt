<p><b>알레스 호드리구 지아스 다 코스타</b> (Alex Rodrigo Dias da Costa, 흔히 <b>알레스</b> (Alex) 로 알려짐, <a href="/wiki/1982%EB%85%84" title="1982년">1982년</a> <a href="/wiki/6%EC%9B%94_17%EC%9D%BC" title="6월 17일">6월 17일</a>, <a href="/wiki/%EB%8B%88%ED%85%8C%EB%A1%9C%EC%9D%B4" title="니테로이">니테로이</a> ~) 는 <a href="/wiki/%EB%B8%8C%EB%9D%BC%EC%A7%88" title="브라질">브라질</a>의 전 <a href="/wiki/%EC%B6%95%EA%B5%AC" title="축구">축구</a> 선수로, 포지션은 <a href="/wiki/%EC%88%98%EB%B9%84%EC%88%98" title="수비수">중앙 수비수</a>이다. 그는 피지컬적인 힘과 슈팅 파워로 알려져 있으며, 그 특징으로 "탱크" (De Tank) 라는 별명을 얻었다.<sup class="reference" id="cite_ref-Tank_2-0"><a href="#cite_note-Tank-2">[2]</a></sup></p>
<p>2004년 <a href="/wiki/%ED%94%84%EB%A6%AC%EB%AF%B8%EC%96%B4%EB%A6%AC%EA%B7%B8" title="프리미어리그">프리미어리그</a> 클럽 <a href="/wiki/%EC%B2%BC%EC%8B%9C_FC" title="첼시 FC">첼시</a>에 영입된 그는 네덜란드 클럽 <a href="/wiki/PSV_%EC%97%90%EC%9D%B8%ED%8A%B8%ED%98%B8%EB%B2%88" title="PSV 에인트호번">PSV 에인트호번</a>으로 임대되었고, 그곳에서 3번의 <a href="/wiki/%EC%97%90%EB%A0%88%EB%94%94%EB%B9%84%EC%8B%9C" title="에레디비시">에레디비시</a> 우승을 거두었다. 2007년 여름, 알레스는 마침내 <a href="/wiki/%EB%84%A4%EB%8D%9C%EB%9E%80%EB%93%9C" title="네덜란드">네덜란드</a>에서의 3년 생활에 종지부를 찍고 첼시 스쿼드에 합류하였다.<sup class="reference" id="cite_ref-3"><a href="#cite_note-3">[3]</a></sup> 첼시 시절, 그는 <a class="new" href="/w/index.php?title=FA%EC%BB%B5_2008-09&amp;action=edit&amp;redlink=1" title="FA컵 2008-09 (없는 문서)">2009년 FA컵</a>과 <a class="new" href="/w/index.php?title=%EC%B2%BC%EC%8B%9C_FC_2009-10_%EC%8B%9C%EC%A6%8C&amp;action=edit&amp;redlink=1" title="첼시 FC 2009-10 시즌 (없는 문서)">2010년 리그 FA컵 "더블"</a>을 획득하였고, <a href="/wiki/2008_UEFA_%EC%B1%94%ED%94%BC%EC%96%B8%EC%8A%A4%EB%A6%AC%EA%B7%B8_%EA%B2%B0%EC%8A%B9%EC%A0%84" title="2008 UEFA 챔피언스리그 결승전">2008년 UEFA 챔피언스리그 결승전</a>에도 진출하였다.</p>
<p>2003년과 2008년 사이, 알레스는 <a href="/wiki/%EB%B8%8C%EB%9D%BC%EC%A7%88_%EC%B6%95%EA%B5%AC_%EA%B5%AD%EA%B0%80%EB%8C%80%ED%91%9C%ED%8C%80" title="브라질 축구 국가대표팀">브라질 축구 국가대표팀</a> 17경기를 뛰었고, <a class="new" href="/w/index.php?title=2007%EB%85%84_%EC%BD%94%ED%8C%8C_%EC%95%84%EB%A9%94%EB%9D%BC%EC%B9%B4&amp;action=edit&amp;redlink=1" title="2007년 코파 아메라카 (없는 문서)">2007년 코파 아메라카</a> 우승과 <a href="/wiki/2003%EB%85%84_CONCACAF_%EA%B3%A8%EB%93%9C%EC%BB%B5" title="2003년 CONCACAF 골드컵">2003년 CONCACAF 골드컵</a> 준우승을 거두었다.</p>

<h2><span class="mw-headline" id=".ED.81.B4.EB.9F.BD_.EA.B2.BD.EB.A0.A5">클럽 경력</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%95%8C%EB%A0%88%EC%8A%A4_%ED%98%B8%EB%93%9C%EB%A6%AC%EA%B5%AC_%EC%A7%80%EC%95%84%EC%8A%A4_%EB%8B%A4_%EC%BD%94%EC%8A%A4%ED%83%80&amp;action=edit&amp;section=1" title="부분 편집: 클럽 경력">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<h3><span class="mw-headline" id=".EC.82.B0.ED.88.AC.EC.8A.A4">산투스</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%95%8C%EB%A0%88%EC%8A%A4_%ED%98%B8%EB%93%9C%EB%A6%AC%EA%B5%AC_%EC%A7%80%EC%95%84%EC%8A%A4_%EB%8B%A4_%EC%BD%94%EC%8A%A4%ED%83%80&amp;action=edit&amp;section=2" title="부분 편집: 산투스">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>알레스는 브라질의 <a href="/wiki/%EC%82%B0%ED%88%AC%EC%8A%A4_FC" title="산투스 FC">산투스</a>에서 경력을 시작하였는데, 그는 2002년에 데뷔하여 시즌 막판에 클럽의 중앙 기둥이 되었다. 알레스는 1-1로 비긴 <a href="/wiki/%ED%94%8C%EB%A3%A8%EB%AF%B8%EB%84%A8%EC%8B%9C_FC" title="플루미넨시 FC">플루미넨시</a>와의 경기에서 리그 경기에 처음으로 출전하였다. 데뷔전을 치른지 한달도 되지 않아, 알레스는 <a href="/wiki/CR_%EB%B0%94%EC%8A%A4%EC%BF%A0_%EB%8B%A4_%EA%B0%80%EB%A7%88" title="CR 바스쿠 다 가마">바스쿠 다 가마</a>와의 2002년 9월 18일 경기에서 산투스 첫 골을 득점하였다. 여러 차례 인상적인 활약을 펼친 후, 알레스는 선발 라인업 명단에 포함되었다. 그의 첫 시즌인 2002 시즌에 산투스는 1968년 이후 처음으로 <a class="mw-redirect" href="/wiki/%EB%B8%8C%EB%9D%BC%EC%A7%88_%EC%84%B8%EB%A6%AC_A" title="브라질 세리 A">브라질 세리 A</a> 타이틀을 획득하였다.</p>
<p>2003 시즌, 알레스는 34경기에 출전하였고, 수비로 기용되었지만, 무려 9골을 득점하였다. 또한, 알레스는 산투스가 <a href="/wiki/%EC%BD%94%ED%8C%8C_%EB%A6%AC%EB%B2%A0%EB%A5%B4%ED%83%80%EB%8F%84%EB%A0%88%EC%8A%A4_2003" title="코파 리베르타도레스 2003">코파 리베르타도레스</a>의 결승전 진출에 힘을 보탰으나, 산투스는 아르헨티나의 <a href="/wiki/%EB%B3%B4%EC%B9%B4_%EC%A3%BC%EB%8B%88%EC%96%B4%EC%8A%A4" title="보카 주니어스">보카 주니어스</a>를 상대로 합계 1-5로 패하였다. 코파 리베르타도레스 결승 2차전, 알레스는 1-3으로 패한 이 경기에서 산투스의 만회골을 득점하였다.</p>
<p>알레스는 2004년에 명성있는 <a href="/wiki/PSV_%EC%97%90%EC%9D%B8%ED%8A%B8%ED%98%B8%EB%B2%88" title="PSV 에인트호번">PSV 에인트호번</a>과 첼시의 스카우트 <a class="new" href="/w/index.php?title=%ED%94%BC%EC%97%90%ED%8A%B8_%EB%8D%94_%ED%94%BC%EC%84%9C&amp;action=edit&amp;redlink=1" title="피에트 더 피서 (없는 문서)">피에트 더 피서</a>에 의해 <a href="/wiki/%ED%94%84%EB%A6%AC%EB%AF%B8%EC%96%B4%EB%A6%AC%EA%B7%B8" title="프리미어리그">프리미어리그</a> 클럽 <a href="/wiki/%EC%B2%BC%EC%8B%9C_FC" title="첼시 FC">첼시</a>와 계약하였으나, <a href="/wiki/%EA%B3%A0%EC%9A%A9_%ED%97%88%EA%B0%80%EC%A0%9C" title="고용 허가제">고용 허가</a>를 얻는데 어려움을 겪은 그는 네덜란드 <a href="/wiki/%EC%97%90%EB%A0%88%EB%94%94%EB%B9%84%EC%8B%9C" title="에레디비시">에레디비시</a> 클럽 PSV 에인트호번으로 <a class="new" href="/w/index.php?title=%EC%9E%84%EB%8C%80_(%EC%B6%95%EA%B5%AC)&amp;action=edit&amp;redlink=1" title="임대 (축구) (없는 문서)">임대</a>되었다.<sup class="reference" id="cite_ref-4"><a href="#cite_note-4">[4]</a></sup></p>
<h3><span class="mw-headline" id="PSV_.EC.97.90.EC.9D.B8.ED.8A.B8.ED.98.B8.EB.B2.88_.EC.9E.84.EB.8C.80">PSV 에인트호번 임대</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%95%8C%EB%A0%88%EC%8A%A4_%ED%98%B8%EB%93%9C%EB%A6%AC%EA%B5%AC_%EC%A7%80%EC%95%84%EC%8A%A4_%EB%8B%A4_%EC%BD%94%EC%8A%A4%ED%83%80&amp;action=edit&amp;section=3" title="부분 편집: PSV 에인트호번 임대">편집</a><span class="mw-editsection-bracket">]</span></span></h3>

<p>알레스의 첼시와 PSV 에인트호번 이중 계약으로 알레스는 3년간 PSV 에인트호번으로 임대되었고, 첼시는 €1M의 바이아웃 조항을 쓸 수 있게 되었다. 알레스는 이후 3년간 PSV 에인트호번 선수로 활약하게 되었고, <a href="/wiki/%EC%97%90%EC%9D%B8%ED%8A%B8%ED%98%B8%EB%B2%88" title="에인트호번">에인트호번</a> 클럽에서 활약하면서 4번 유니폼을 사용하였다. PSV에서 활약하면서, 그는 대포알 같은 프리킥과 강력한 피지컬 능력으로 "탱크" (De Tank) 라는 별명을 얻었다.<sup class="reference" id="cite_ref-Tank_2-1"><a href="#cite_note-Tank-2">[2]</a></sup></p>
<p>그는 <a class="new" href="/w/index.php?title=%EB%84%A4%EB%8D%9C%EB%9E%80%EB%93%9C_%EC%B6%95%EA%B5%AC_2004-05&amp;action=edit&amp;redlink=1" title="네덜란드 축구 2004-05 (없는 문서)">2004-05 시즌</a>에 공식적으로 PSV 에인트호번에 합류하여 팀이 <a href="/wiki/%EC%97%90%EB%A0%88%EB%94%94%EB%B9%84%EC%8B%9C" title="에레디비시">에레디비시</a> 타이틀과 <a href="/wiki/KNVB%EC%BB%B5" title="KNVB컵">KNVB컵</a> 타이틀을 획득하여 국내 더블을 이룰 수 있게 하였다. <a href="/wiki/UEFA_%EC%B1%94%ED%94%BC%EC%96%B8%EC%8A%A4%EB%A6%AC%EA%B7%B8_2004-05" title="UEFA 챔피언스리그 2004-05">챔피언스리그</a>에서, 알레스는 <a href="/wiki/AC_%EB%B0%80%EB%9E%80" title="AC 밀란">밀란</a>과의 준결승전까지 PSV 에인트호번이 진출하는데 중요한 역할을 하였다. (PSV 에인트호번은 홈경기에서 3-1로 이겼으나, 이전에 <a href="/wiki/%EC%8A%A4%ED%83%80%EB%94%94%EC%98%A4_%EC%A3%BC%EC%84%B8%ED%8E%98_%EB%A9%94%EC%95%84%EC%B0%A8" title="스타디오 주세페 메아차">산 시로</a>에서 0-2로 패하면서 원정 다득점 원칙에 따라 결승 진출이 좌절되었다.) PSV 에인트호번은 준결승으로 진출하는 길목에 <a href="/wiki/AS_%EB%AA%A8%EB%82%98%EC%BD%94_FC" title="AS 모나코 FC">모나코</a>와 <a href="/wiki/%EC%98%AC%EB%9E%AD%ED%94%BC%ED%81%AC_%EB%A6%AC%EC%98%B9" title="올랭피크 리옹">리옹</a>을 꺾었다.</p>
<p><a class="new" href="/w/index.php?title=%EC%97%90%EB%A0%88%EB%94%94%EB%B9%84%EC%8B%9C_2005-06&amp;action=edit&amp;redlink=1" title="에레디비시 2005-06 (없는 문서)">2005-06 시즌</a>, 알레스는 PSV 에인트호번에서 에레디비시 2연패를 달성하였다. 비록 <a class="new" href="/w/index.php?title=%ED%94%BC%EC%97%90%ED%8A%B8_%EB%8D%94_%ED%94%BC%EC%84%9C&amp;action=edit&amp;redlink=1" title="피에트 더 피서 (없는 문서)">피에트 더 피서</a>와 첼시 수석 스카우트 <a class="new" href="/w/index.php?title=%ED%94%84%EB%9E%91%ED%81%AC_%EC%95%84%EB%A5%B4%EB%84%A4%EC%84%BC&amp;action=edit&amp;redlink=1" title="프랑크 아르네센 (없는 문서)">프랑크 아르네센</a>은 클럽의 수비 문제를 해결하기 위해 알레스를 복귀시켜야 한다고 주장하였으나, 첼시는 그를 복귀시키지 않았다.<sup class="reference" id="cite_ref-times_5-0"><a href="#cite_note-times-5">[5]</a></sup> 그 대신, <a class="mw-redirect" href="/wiki/%EC%A3%BC%EC%A0%9C_%EB%AA%A8%EB%A6%AC%EB%89%B4" title="주제 모리뉴">주제 모리뉴</a> 첼시 감독은 알레스의 자리에 <a class="new" href="/w/index.php?title=%ED%95%A0%EB%A6%AC%ED%8A%B8_%EB%B6%88%EB%9D%BC%EB%A3%A8%EC%A6%88&amp;action=edit&amp;redlink=1" title="할리트 불라루즈 (없는 문서)">할리트 불라루즈</a>를 영입하였다.<sup class="reference" id="cite_ref-times_5-1"><a href="#cite_note-times-5">[5]</a></sup> 그 결과, 알레스는 <a class="new" href="/w/index.php?title=%EC%97%90%EB%A0%88%EB%94%94%EB%B9%84%EC%8B%9C_2006-07&amp;action=edit&amp;redlink=1" title="에레디비시 2006-07 (없는 문서)">2006-07</a> 시즌에 PSV 에인트호번에서의 3년차를 보냈다. 그는 PSV 에인트호번에서 에레디비시 3연패를 거두었는데, 팀은 최종전에 드라마틱하게 <a href="/wiki/AFC_%EC%95%84%EC%95%BD%EC%8A%A4" title="AFC 아약스">아약스</a>와 <a href="/wiki/AZ_%EC%95%8C%ED%81%AC%EB%A7%88%EB%A5%B4" title="AZ 알크마르">AZ</a>를 모두 따돌렸다. 알레스는 <a href="/wiki/%EC%95%84%EC%8A%A4%EB%84%90_FC" title="아스널 FC">아스널</a>과의 <a class="new" href="/w/index.php?title=UEFA_%EC%B1%94%ED%94%BC%EC%96%B8%EC%8A%A4%EB%A6%AC%EA%B7%B8_2006-07_%ED%86%A0%EB%84%88%EB%A8%BC%ED%8A%B8%EC%A0%84&amp;action=edit&amp;redlink=1" title="UEFA 챔피언스리그 2006-07 토너먼트전 (없는 문서)">UEFA 챔피언스리그 16강전</a>에서 후반에 양쪽 모두의 득점을 기록하였다. 팀 골키퍼 <a href="/wiki/%EC%97%90%EC%9A%B0%EB%A0%90%EB%A5%98_%EA%B3%A0%EB%A9%94%EC%8A%A4" title="에우렐류 고메스">에우렐류 고메스</a>를 넘기는 <a href="/wiki/%EC%9E%90%EC%B1%85%EA%B3%A8" title="자책골">자책골</a>을 넣은 뒤 8분을 남긴 상황에서 헤딩골을 득점하여 스스로를 구제하고, PSV 에인트호번의 8강행에 결정적인 역할을 하였다.<sup class="reference" id="cite_ref-6"><a href="#cite_note-6">[6]</a></sup> 그는 나중에 부상당하는 바람에 <a href="/wiki/%EB%A6%AC%EB%B2%84%ED%92%80_FC" title="리버풀 FC">리버풀</a>과의 <a class="new" href="/w/index.php?title=UEFA_%EC%B1%94%ED%94%BC%EC%96%B8%EC%8A%A4%EB%A6%AC%EA%B7%B8_2006-07_%ED%86%A0%EB%84%88%EB%A8%BC%ED%8A%B8%EC%A0%84&amp;action=edit&amp;redlink=1" title="UEFA 챔피언스리그 2006-07 토너먼트전 (없는 문서)">UEFA 챔피언스리그 8강전</a> 출전이 좌절되었고, 팀은 합계 0-4로 패해 떨어졌다.</p>

<h4><span class="mw-headline" id="2007-08_.EC.8B.9C.EC.A6.8C">2007-08 시즌</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%95%8C%EB%A0%88%EC%8A%A4_%ED%98%B8%EB%93%9C%EB%A6%AC%EA%B5%AC_%EC%A7%80%EC%95%84%EC%8A%A4_%EB%8B%A4_%EC%BD%94%EC%8A%A4%ED%83%80&amp;action=edit&amp;section=5" title="부분 편집: 2007-08 시즌">편집</a><span class="mw-editsection-bracket">]</span></span></h4>

2008년, <a href="/wiki/%EB%A7%A8%EC%B2%B4%EC%8A%A4%ED%84%B0_%EC%9C%A0%EB%82%98%EC%9D%B4%ED%8B%B0%EB%93%9C_FC" title="맨체스터 유나이티드 FC">맨체스터 유나이티드</a>전을 앞두고 몸을 푸는 <a href="/wiki/%ED%81%B4%EB%A1%9C%EB%93%9C_%EB%A7%88%EC%BC%88%EB%A0%90%EB%A0%88" title="클로드 마켈렐레">클로드 마켈렐레</a>와 알레스</div>

<p>네덜란드 언론 <i><a href="/wiki/%EB%8D%94_%ED%85%94%EB%A0%88%ED%9D%90%EB%9D%BC%ED%94%84" title="더 텔레흐라프">더 텔레흐라프</a></i>와의 인터뷰에서, 알레스는 첼시가 그의 고용 허가를 획득하는 단계에 있어서, 다음 시즌에 첼시에 합류할 확률이 매우 높다고 확인하였다.<sup class="reference" id="cite_ref-7"><a href="#cite_note-7">[7]</a></sup> 알레스는 이후, 첼시가 그의 고용 허가를 발급 받는 것을 완료하였고, 다음 시즌을 첼시 선수로 시작할 수 있다고 확인하였다.</p>
<p>2007년 8월 2일, 알레스는 고용 허가를 발급받았고, 개인 조항에 합의를 본 후 메디컬을 통과하며 첼시로의 이적을 완료하였고, <a href="/wiki/%EB%9F%B0%EB%8D%98" title="런던">런던</a> 클럽과 3년 계약을 체결하였다.</p>
<p>2007년 8월 14일, 알레스는 <a class="mw-redirect" href="/wiki/%EC%A3%BC%EC%A0%9C_%EB%AA%A8%EB%A6%AC%EB%89%B4" title="주제 모리뉴">주제 모리뉴</a> 감독과 함께 잉글랜드 언론들이 참석한 기자 회견에 참석하였고, 여기서 그는 등번호 33번을 받았다. 8월 19일, 그는 1-1로 비긴 <a href="/wiki/%EB%A6%AC%EB%B2%84%ED%92%80_FC" title="리버풀 FC">리버풀</a>전에서 <a href="/wiki/%ED%94%8C%EB%A1%9C%EB%9E%91_%EB%A7%90%EB%A3%A8%EB%8B%A4" title="플로랑 말루다">플로랑 말루다</a>와 85분에 교체 출전하며 잉글랜드 무대에 데뷔하였다. 그는 0-2로 패한 <a href="/wiki/%EC%95%A0%EC%8A%A4%ED%84%B4_%EB%B9%8C%EB%9D%BC_FC" title="애스턴 빌라 FC">애스턴 빌라</a> 원정경기와 4-0으로 이긴 <a href="/wiki/%ED%97%90_%EC%8B%9C%ED%8B%B0_AFC" title="헐 시티 AFC">헐 시티</a>와의 <a class="mw-redirect" href="/wiki/%ED%92%8B%EB%B3%BC_%EB%A6%AC%EA%B7%B8_%EC%BB%B5" title="풋볼 리그 컵">풋볼 리그 컵</a> 경기에도 출전하였다.<sup class="reference" id="cite_ref-8"><a href="#cite_note-8">[8]</a></sup></p>
<p>2007년 10월 20일, 그는 <a href="/wiki/%EB%AF%B8%EB%93%A4%EC%A6%88%EB%B8%8C%EB%9F%AC_FC" title="미들즈브러 FC">미들즈브러</a>전에서 27m 프리킥을 성공시키며 첼시에서의 첫 득점을 기록하였다.<sup class="reference" id="cite_ref-9"><a href="#cite_note-9">[9]</a></sup> 그는 <a href="/wiki/%EB%A1%9C%EC%84%BC%EB%B3%B4%EB%A5%B4%EA%B7%B8_BK" title="로센보르그 BK">로센보르그</a>와의 11월 28일 챔피언스리그 경기에서 또다시 프리킥 골을 성공시켰다.<sup class="reference" id="cite_ref-10"><a href="#cite_note-10">[10]</a></sup> 그는 이후 4-4로 비긴 <a href="/wiki/%EC%95%A0%EC%8A%A4%ED%84%B4_%EB%B9%8C%EB%9D%BC_FC" title="애스턴 빌라 FC">애스턴 빌라</a>와의 경기에서 <a href="/wiki/%EC%95%88%EB%93%9C%EB%A6%AC_%EC%86%81%EC%B2%B8%EC%BD%94" title="안드리 솁첸코">안드리 솁첸코</a>의 패스를 받아 <a href="/wiki/%EC%8A%A4%ED%83%AC%ED%8D%BC%EB%93%9C_%EB%B8%8C%EB%A6%AC%EC%A7%80" title="스탬퍼드 브리지">스탬퍼드 브리지</a>에서 통산 3호골을 득점하였다.<sup class="reference" id="cite_ref-11"><a href="#cite_note-11">[11]</a></sup></p>
<h4><span class="mw-headline" id="2008-09_.EC.8B.9C.EC.A6.8C">2008-09 시즌</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%95%8C%EB%A0%88%EC%8A%A4_%ED%98%B8%EB%93%9C%EB%A6%AC%EA%B5%AC_%EC%A7%80%EC%95%84%EC%8A%A4_%EB%8B%A4_%EC%BD%94%EC%8A%A4%ED%83%80&amp;action=edit&amp;section=6" title="부분 편집: 2008-09 시즌">편집</a><span class="mw-editsection-bracket">]</span></span></h4>
<p>알레스는 <a href="/wiki/%ED%94%84%EB%A6%AC%EB%AF%B8%EC%96%B4%EB%A6%AC%EA%B7%B8_2008-09" title="프리미어리그 2008-09">2008-09 시즌</a>동안 1군에서 많은 출장 기회가 주어졌다. 그는 주전 멤버였던 <a href="/wiki/%ED%9E%88%EC%B9%B4%EB%A5%B4%EB%91%90_%EC%B9%B4%EB%A5%B4%EB%B0%9C%EB%A5%98" title="히카르두 카르발류">히카르두 카르발류</a>가 시즌 대부분 기간을 부상으로 결장하면서 기회를 받았다. 그는 2008년 11월 1일, <a href="/wiki/%EC%84%A0%EB%8D%9C%EB%9E%9C%EB%93%9C_AFC" title="선덜랜드 AFC">선덜랜드</a>전에서 득점하며 첼시의 프리미어리그 1,000호골 주인공이 되었다.<sup class="reference" id="cite_ref-12"><a href="#cite_note-12">[12]</a></sup> 알레스는 이전에 <a href="/wiki/PSV_%EC%97%90%EC%9D%B8%ED%8A%B8%ED%98%B8%EB%B2%88" title="PSV 에인트호번">PSV 에인트호번</a> 시절에 감독이었던 <a href="/wiki/%EA%B1%B0%EC%8A%A4_%ED%9E%88%EB%94%A9%ED%81%AC" title="거스 히딩크">휘스 히딩크</a>가 첼시 사령탑을 맡고 나서 모든 경기에 선발로 출전하였고, <a href="/wiki/%EC%BD%94%EB%B2%88%ED%8A%B8%EB%A6%AC_%EC%8B%9C%ED%8B%B0_FC" title="코번트리 시티 FC">코번트리 시티</a>와의 <a class="new" href="/w/index.php?title=FA%EC%BB%B5_2008-09&amp;action=edit&amp;redlink=1" title="FA컵 2008-09 (없는 문서)">FA컵</a> 8강 원정 경기에서 득점하였다.<sup class="reference" id="cite_ref-13"><a href="#cite_note-13">[13]</a></sup></p>

<p>스탬퍼드 브리지에서 열린 <a href="/wiki/%EB%A6%AC%EB%B2%84%ED%92%80_FC" title="리버풀 FC">리버풀</a>과의 챔피언스리그 8강 2차전에서, 알레스는 27m 거리에서 프리킥으로 득점하며 경기를 2-2로 균형을 맞추었다. 경기는 4-4로 종료되어 첼시가 합계 7-5로 승리하며 준결승전에 진출하였으나,<sup class="reference" id="cite_ref-14"><a href="#cite_note-14">[14]</a></sup> 첼시는 준결승전에서 <a href="/wiki/FC_%EB%B0%94%EB%A5%B4%EC%85%80%EB%A1%9C%EB%82%98" title="FC 바르셀로나">바르셀로나</a>와의 판정시비 끝에 탈락하였다.<sup class="reference" id="cite_ref-15"><a href="#cite_note-15">[15]</a></sup> 5월 10일, 알레스는 <a href="/wiki/%EB%94%94%EB%94%94%EC%97%90_%EB%93%9C%EB%A1%9C%EA%B7%B8%EB%B0%94" title="디디에 드로그바">디디에 드로그바</a>의 프리킥을 헤딩골로 연결하였고, 이 <a href="/wiki/%ED%94%84%EB%A6%AC%EB%AF%B8%EC%96%B4%EB%A6%AC%EA%B7%B8" title="프리미어리그">프리미어리그</a> 경기에서 첼시는 아스널을 4-1로 박살냈다.<sup class="reference" id="cite_ref-16"><a href="#cite_note-16">[16]</a></sup> 이 골은 아스널을 상대로한 그의 두번째 골이었다.</p>
<h4><span class="mw-headline" id="2009-10_.EC.8B.9C.EC.A6.8C">2009-10 시즌</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%95%8C%EB%A0%88%EC%8A%A4_%ED%98%B8%EB%93%9C%EB%A6%AC%EA%B5%AC_%EC%A7%80%EC%95%84%EC%8A%A4_%EB%8B%A4_%EC%BD%94%EC%8A%A4%ED%83%80&amp;action=edit&amp;section=7" title="부분 편집: 2009-10 시즌">편집</a><span class="mw-editsection-bracket">]</span></span></h4>
<p>알레스는 <a class="new" href="/w/index.php?title=%EC%9B%94%EB%93%9C_%ED%92%8B%EB%B3%BC_%EC%B1%8C%EB%A6%B0%EC%A7%80&amp;action=edit&amp;redlink=1" title="월드 풋볼 챌린지 (없는 문서)">월드 풋볼 챌린지</a> 토너먼트의 <a href="/wiki/AC_%EB%B0%80%EB%9E%80" title="AC 밀란">밀란</a>전에서 프리시즌 첫 경기를 치르었다.<sup class="reference" id="cite_ref-17"><a href="#cite_note-17">[17]</a></sup></p>
<p>2009년 10월 13일, 알레스는 <a href="/wiki/%EC%B2%BC%EC%8B%9C_FC" title="첼시 FC">첼시</a>와 4년 재계약을 체결하면서, <a href="/wiki/%EC%8A%A4%ED%83%AC%ED%8D%BC%EB%93%9C_%EB%B8%8C%EB%A6%AC%EC%A7%80" title="스탬퍼드 브리지">스탬퍼드 브리지</a>에 2013년 여름까지 남게 되었다.<sup class="reference" id="cite_ref-18"><a href="#cite_note-18">[18]</a></sup></p>
<p>알레스는 2009년 8월 15일, 허벅지 부상을 당하며 시즌 초 경기에 결장하였다.<sup class="reference" id="cite_ref-19"><a href="#cite_note-19">[19]</a></sup> 그는 2009년 10월 28일, <a href="/wiki/%EB%B3%BC%ED%84%B4_%EC%9B%90%EB%8D%94%EB%9F%AC%EC%8A%A4_FC" title="볼턴 원더러스 FC">볼턴 원더러스</a>와의 리그컵 경기에서 시즌 첫 선발출전을 하였고, 90분 풀타임을 소화하였다. 그는 2009년 12월 16일, <a href="/wiki/%ED%8F%AC%EC%B8%A0%EB%A8%B8%EC%8A%A4_FC" title="포츠머스 FC">포츠머스</a>와의 경기에서 시즌 첫 <a href="/wiki/%ED%94%84%EB%A6%AC%EB%AF%B8%EC%96%B4%EB%A6%AC%EA%B7%B8" title="프리미어리그">프리미어리그</a> 선발 출전을 하였다.<sup class="reference" id="cite_ref-20"><a href="#cite_note-20">[20]</a></sup></p>
<p>2010년 3월 13일, 그는 4-1로 이긴 <a class="mw-redirect" href="/wiki/%EC%9B%A8%EC%8A%A4%ED%8A%B8_%ED%96%84_%EC%9C%A0%EB%82%98%EC%9D%B4%ED%8B%B0%EB%93%9C_FC" title="웨스트 햄 유나이티드 FC">웨스트 햄 유나이티드</a>전에서 시즌 스탬퍼드 브릿지 경기 첫 골을 득점하며 리드를 가져다주었다.<sup class="reference" id="cite_ref-21"><a href="#cite_note-21">[21]</a></sup></p>
<h4><span class="mw-headline" id="2010-11_.EC.8B.9C.EC.A6.8C">2010-11 시즌</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%95%8C%EB%A0%88%EC%8A%A4_%ED%98%B8%EB%93%9C%EB%A6%AC%EA%B5%AC_%EC%A7%80%EC%95%84%EC%8A%A4_%EB%8B%A4_%EC%BD%94%EC%8A%A4%ED%83%80&amp;action=edit&amp;section=8" title="부분 편집: 2010-11 시즌">편집</a><span class="mw-editsection-bracket">]</span></span></h4>

<p>히카르두 카르발류가 <a href="/wiki/%EB%A0%88%EC%95%8C_%EB%A7%88%EB%93%9C%EB%A6%AC%EB%93%9C_CF" title="레알 마드리드 CF">레알 마드리드</a>로 이적하면서, 알레스는 2010-11 시즌을 <a href="/wiki/%EC%A1%B4_%ED%85%8C%EB%A6%AC" title="존 테리">존 테리</a>와 함께 첼시의 주전 센터백으로 시작하였다. 2010년 10월 3일, 그는 <a href="/wiki/%EC%8A%A4%ED%83%AC%ED%8D%BC%EB%93%9C_%EB%B8%8C%EB%A6%AC%EC%A7%80" title="스탬퍼드 브리지">스탬퍼드 브리지</a>에서 <a href="/wiki/%EC%95%84%EC%8A%A4%EB%84%90_FC" title="아스널 FC">아스널</a>을 상대로 22m 프리킥을 성공시키며, 2-0 승리에 쐐기를 박았다. 10월 5일, 그는 2-0으로 승리한 아스널과의 일요일 경기에서 넓적다리 부상을 당하며 최장 3주아웃 판정을 받았고, <a href="/wiki/%EC%9D%B4%EB%9E%80_%EC%B6%95%EA%B5%AC_%EA%B5%AD%EA%B0%80%EB%8C%80%ED%91%9C%ED%8C%80" title="이란 축구 국가대표팀">이란</a>과 <a href="/wiki/%EC%9A%B0%ED%81%AC%EB%9D%BC%EC%9D%B4%EB%82%98_%EC%B6%95%EA%B5%AC_%EA%B5%AD%EA%B0%80%EB%8C%80%ED%91%9C%ED%8C%80" title="우크라이나 축구 국가대표팀">우크라이나</a>와의 친선경기를 앞둔 브라질 국가대표 명단에서 빠졌다.<sup class="reference" id="cite_ref-22"><a href="#cite_note-22">[22]</a></sup> 2010년 10월 30일, 알레스는 첼시에 복귀하여 2-1로 이긴 <a href="/wiki/%EB%B8%94%EB%9E%99%EB%B2%88_%EB%A1%9C%EB%B2%84%EC%8A%A4_FC" title="블랙번 로버스 FC">블랙번 로버스</a> 전에서 풀타임 출전을 하였다. 2010년 12월 1일, 알레스는 무릎 부상으로 3달 결장이 불가피하게 되었다. 부상에서 재활 도중, <a href="/wiki/%EC%B9%B4%EB%A5%BC%EB%A1%9C_%EC%95%88%EC%B2%BC%EB%A1%9C%ED%8B%B0" title="카를로 안첼로티">카를로 안첼로티</a> 감독은 알레스가 1월 말까지 무릎 수술로부터 완전히 회복하지 않을 경우 수비수를 한명 영입하는 것을 고려할 것이라 하였고, 결국 첼시는 <a href="/wiki/%EB%8B%A4%EB%B9%84%EB%93%9C_%EB%A3%A8%EC%9D%B4%EC%8A%A4" title="다비드 루이스">다비드 루이스</a>를 벤피카로부터 영입하였다.<sup class="reference" id="cite_ref-23"><a href="#cite_note-23">[23]</a></sup> 2011년 3월 중순까지, 알레스는 1군 훈련에 복귀하였다.<sup class="reference" id="cite_ref-24"><a href="#cite_note-24">[24]</a></sup></p>
<p>부상 복귀 후, 2011년 4월 9일, 그는 1-0으로 이긴 <a href="/wiki/%EC%9C%84%EA%B1%B4_%EC%95%A0%EC%8A%AC%EB%A0%88%ED%8B%B1_FC" title="위건 애슬레틱 FC">위건 애슬레틱</a>전에 복귀하였고, 그는 75분에 <a href="/wiki/%ED%8C%8C%EC%9A%B8%EB%A3%A8_%ED%8E%98%ED%97%A4%EC%9D%B4%EB%9D%BC" title="파울루 페헤이라">파울루 페헤이라</a>와 교체되었다. 2011년 5월 8일, 그는 <a href="/wiki/%EB%A7%A8%EC%B2%B4%EC%8A%A4%ED%84%B0_%EC%9C%A0%EB%82%98%EC%9D%B4%ED%8B%B0%EB%93%9C_FC" title="맨체스터 유나이티드 FC">맨체스터 유나이티드</a>와의 시즌을 좌우하는 경기에서 선발로 출전하여 실점에 영향을 주는 실책을 범하였고, 하프 타임에 다비드 루이스와 교체되었으며, 다음 2011년 5월 15일, <a href="/wiki/%EB%89%B4%EC%BA%90%EC%8A%AC_%EC%9C%A0%EB%82%98%EC%9D%B4%ED%8B%B0%EB%93%9C_FC" title="뉴캐슬 유나이티드 FC">뉴캐슬 유나이티드</a>와의 스탬퍼드 브리지 경기에서 풀타임을 소화하며, 2-2로 비긴 이 경기에서 82분에 헤딩골을 넣었다. <a href="/wiki/%EC%97%90%EB%B2%84%ED%84%B4_FC" title="에버턴 FC">에버턴</a>과의 리그 최종전에서, 알레스는 <a href="/wiki/%EC%A1%B4_%ED%85%8C%EB%A6%AC" title="존 테리">존 테리</a>와 더불어 선발 출전하였으나, 70분에 페헤이라와 교체 아웃되었고, 팀은 <a href="/wiki/%EA%B5%AC%EB%94%94%EC%8A%A8_%ED%8C%8C%ED%81%AC" title="구디슨 파크">구디슨 파크</a>에서 0-1 패배를 당하였다. 이 시즌동안, 알레스는 20경기에 출전하여 모든 대회를 통틀어 2득점을 올렸다. 알레스는 인터뷰에서 유년 시절 리버풀 서포터였다고 진술한 것으로 인해 비난의 대상이 되었다.</p>
<h4><span class="mw-headline" id="2011-12_.EC.8B.9C.EC.A6.8C">2011-12 시즌</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%95%8C%EB%A0%88%EC%8A%A4_%ED%98%B8%EB%93%9C%EB%A6%AC%EA%B5%AC_%EC%A7%80%EC%95%84%EC%8A%A4_%EB%8B%A4_%EC%BD%94%EC%8A%A4%ED%83%80&amp;action=edit&amp;section=9" title="부분 편집: 2011-12 시즌">편집</a><span class="mw-editsection-bracket">]</span></span></h4>
<p>여름 이적시장에서, 알레스는 <a class="new" href="/w/index.php?title=%EC%95%84%EC%8A%A4%EB%84%90_FC-%EC%B2%BC%EC%8B%9C_FC_%EA%B2%BD%EC%9F%81&amp;action=edit&amp;redlink=1" title="아스널 FC-첼시 FC 경쟁 (없는 문서)">지역 라이벌</a> <a href="/wiki/%EC%95%84%EC%8A%A4%EB%84%90_FC" title="아스널 FC">아스널</a>의 이적 제의 대상이 되었다.<sup class="reference" id="cite_ref-25"><a href="#cite_note-25">[25]</a></sup> 2011년 8월 20일, <a class="mw-redirect" href="/wiki/%EC%9B%A8%EC%8A%A4%ED%8A%B8_%EB%B8%8C%EB%A1%9C%EB%AF%B8%EC%B9%98_%EC%95%A8%EB%B9%84%EC%96%B8_FC" title="웨스트 브로미치 앨비언 FC">웨스트 브로미치 앨비언</a>과의 홈 경기에서, 알레스의 수비 실책은 스트라이커 <a href="/wiki/%EC%85%B0%EC%9D%B8_%EB%A1%B1" title="셰인 롱">셰인 롱</a>이 경기 4분만에 득점하게 만들었다. 그러나 첼시는 막판에 2-1로 뒤집었다. 2011년 9월 21일, <a href="/wiki/%ED%92%80%EB%9F%BC_FC" title="풀럼 FC">풀럼</a>과의 리그컵 경기에서, 알레스는 49분에 <a href="/wiki/%EC%BC%80%EB%A6%BC_%ED%94%84%EB%A0%88%EC%9D%B4" title="케림 프레이">케림 프레이</a>에게 반칙을 한 것으로 인해 레드 카드를 받았고, 페널티킥으로 이어졌다. 그러나, <a href="/wiki/%ED%8C%8C%EC%9D%B4%ED%8C%80_%EC%B9%B4%EC%82%AC%EB%AF%B8" title="파이팀 카사미">파이팀 카사미</a>의 페널티킥은 백업 골키퍼 <a href="/wiki/%EB%A1%9C%EC%8A%A4_%ED%84%B4%EB%B6%88" title="로스 턴불">로스 턴불</a>의 선방에 가로막혔다. 경기는 첼시가 승부차기에서 4-3으로 이기며 다음 라운드에 진출하였다.<sup class="reference" id="cite_ref-26"><a href="#cite_note-26">[26]</a></sup> 2011년 11월, 그는 0-2로 패한 <a href="/wiki/%EB%A6%AC%EB%B2%84%ED%92%80_FC" title="리버풀 FC">리버풀</a>과의 리그컵 8강전에서 첼시 선수로써의 마지막 출전을 기록하였다. 그는 선발로 시작한 이 경기에서 페널티 에어리어의 핸드볼을 범하였고, 턴불은 다시 <a class="mw-redirect" href="/wiki/%EC%95%A4%EB%94%94_%EC%BA%90%EB%A1%A4" title="앤디 캐롤">앤디 캐롤</a>의 슛을 선방하였다.</p>
<p>2011년 12월 3일, 알레스는 이적 요청서를 제출하였고, 이는 승인되었다.<sup class="reference" id="cite_ref-27"><a href="#cite_note-27">[27]</a></sup> 그는 신임 감독 <a href="/wiki/%EB%A7%88%ED%81%AC_%ED%9C%B4%EC%8A%A4" title="마크 휴스">마크 휴스</a>가 취임한 <a class="mw-redirect" href="/wiki/%ED%80%B8%EC%A6%88_%ED%8C%8C%ED%81%AC_%EB%A0%88%EC%9D%B8%EC%A0%80%EC%8A%A4_FC" title="퀸즈 파크 레인저스 FC">퀸즈 파크 레인저스</a>의 제의 대상이었다. 그러나 이적은 양쪽 모두 협상에 합의점을 찾지 못하면서 결렬되었다; 퀸즈 파크 레인저스는 알레스의 주급 조건을 맞출 수 없었고, 알레스는 잉글랜드에서 더 이상 머물기를 원치 않았다.<sup class="reference" id="cite_ref-28"><a href="#cite_note-28">[28]</a></sup><sup class="reference" id="cite_ref-29"><a href="#cite_note-29">[29]</a></sup></p>

<p>2012년 1월 27일, 알레스는 £4.2M에 <a href="/wiki/%ED%8C%8C%EB%A6%AC_%EC%83%9D%EC%A0%9C%EB%A5%B4%EB%A7%B9_FC" title="파리 생제르맹 FC">파리 생제르맹</a>에 합류하며<sup class="reference" id="cite_ref-30"><a href="#cite_note-30">[30]</a></sup> 그의 전 감독인 카를로 안첼로티와 재회하였다. 그는 이적 사유에 대해 <a href="/wiki/%EC%95%88%EB%93%9C%EB%A0%88_%EB%B9%8C%EB%9D%BC%EC%8A%A4%EB%B3%B4%EC%95%84%EC%8A%A4" title="안드레 빌라스보아스">안드레 빌라스보아스</a> <a href="/wiki/%EC%B2%BC%EC%8B%9C_FC" title="첼시 FC">첼시</a> 감독이 충분한 출장 시간을 주지 못하였다고 공개적으로 진술하였다.<sup class="reference" id="cite_ref-31"><a href="#cite_note-31">[31]</a></sup> 그는 <a class="new" href="/w/index.php?title=%EC%82%AC%EB%AF%B8_%ED%8A%B8%EB%9D%BC%EC%98%A4%EB%A0%88&amp;action=edit&amp;redlink=1" title="사미 트라오레 (없는 문서)">사미 트라오레</a>가 떠나면서 공석이 된 등번호 13번을 받았다.</p>
<p>2012년 2월 12일, 알레스는 0-0으로 비긴 <a href="/wiki/OGC_%EB%8B%88%EC%8A%A4" title="OGC 니스">니스</a>와의 경기에서 PSG 데뷔전을 치르었는데, 그는 하프 타임에 <a class="mw-redirect" href="/wiki/%ED%81%AC%EB%A6%AC%EC%8A%A4%ED%86%A0%ED%94%84_%EC%9E%98%EB%A0%88" title="크리스토프 잘레">크리스토프 잘레</a>와 교체되어 출전하였다. 그 다음주인 2012년 2월 19일, 알레스는 몽펠리에와의 경기에서 27m 프리킥을 <a class="new" href="/w/index.php?title=%EC%95%88%EB%8D%B0%EB%A5%B4%EC%86%A1_%EB%A3%A8%EC%9D%B4%EC%8A%A4_%EC%A7%80_%EC%B9%B4%EB%A5%B4%EB%B0%9C%EB%A5%98&amp;action=edit&amp;redlink=1" title="안데르송 루이스 지 카르발류 (없는 문서)">네네</a>의 도움을 받아 PSG 이적 후 1호골을 득점하였고, 결과는 2-2 무승부였다. 2012년 4월 8일, 알레스는 <a href="/wiki/%EC%98%AC%EB%9E%AD%ED%94%BC%ED%81%AC_%EB%93%9C_%EB%A7%88%EB%A5%B4%EC%84%B8%EC%9C%A0" title="올랭피크 드 마르세유">마르세유</a>전에서 61분에 결승골을 득점하였고, 팀은 2-1로 이겼다. 2012년 11월 6일, <a href="/wiki/GNK_%EB%94%94%EB%82%98%EB%AA%A8_%EC%9E%90%EA%B7%B8%EB%A0%88%EB%B8%8C" title="GNK 디나모 자그레브">디나모 자그레브</a>와의 챔피언스리그 경기에서, 알레스는 16분에 하프 볼리슛으로 <a class="new" href="/w/index.php?title=%EC%9D%B4%EB%B0%98_%EC%BC%88%EB%B0%94&amp;action=edit&amp;redlink=1" title="이반 켈바 (없는 문서)">이반 켈바</a> 디나모 골키퍼의 손을 넘기는 슛으로 팀의 첫번째 골을 득점하였다.</p>
<p>2013년 8월 3일, <a href="/wiki/%EA%B0%80%EB%B4%89" title="가봉">가봉</a> <a href="/wiki/%EB%A6%AC%EB%B8%8C%EB%A5%B4%EB%B9%8C" title="리브르빌">리브르빌</a>의 <a class="new" href="/w/index.php?title=%EC%8A%A4%ED%83%80%EB%93%9C_%EB%8B%B9%EA%B3%B5%EC%A0%9C&amp;action=edit&amp;redlink=1" title="스타드 당공제 (없는 문서)">스타드 당공제</a>에서 열린 <a href="/wiki/FC_%EC%A7%80%EB%A1%B1%EB%8C%95_%EB%93%9C_%EB%B3%B4%EB%A5%B4%EB%8F%84" title="FC 지롱댕 드 보르도">보르도</a>와의 <a class="new" href="/w/index.php?title=%ED%8A%B8%EB%A1%9C%ED%8E%98_%EB%8D%B0_%EC%83%B9%ED%94%BC%EC%98%B9_2013&amp;action=edit&amp;redlink=1" title="트로페 데 샹피옹 2013 (없는 문서)">트로페 데 샹피옹</a> 경기에서 알레스는 경기 종료를 앞두고 찬 마지막 킥으로 2-1 결승골을 득점하였다.<sup class="reference" id="cite_ref-32"><a href="#cite_note-32">[32]</a></sup></p>
<h3><span class="mw-headline" id=".EB.B0.80.EB.9E.80">밀란</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%95%8C%EB%A0%88%EC%8A%A4_%ED%98%B8%EB%93%9C%EB%A6%AC%EA%B5%AC_%EC%A7%80%EC%95%84%EC%8A%A4_%EB%8B%A4_%EC%BD%94%EC%8A%A4%ED%83%80&amp;action=edit&amp;section=11" title="부분 편집: 밀란">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>이탈리아 세리에 A의 밀란이 프랑스 리그1 파리 생제르망의 브라질 출신 수비수 알렉스의 영입을 확정지었다. AC 밀란은 5일 공식 홈페이지를 통해 “브라질 출신의 수비수 알렉스를 영입했다”고 밝혔다.</p>
<h2><span class="mw-headline" id=".EA.B5.AD.EA.B0.80.EB.8C.80.ED.91.9C.ED.8C.80_.EA.B2.BD.EB.A0.A5">국가대표팀 경력</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%95%8C%EB%A0%88%EC%8A%A4_%ED%98%B8%EB%93%9C%EB%A6%AC%EA%B5%AC_%EC%A7%80%EC%95%84%EC%8A%A4_%EB%8B%A4_%EC%BD%94%EC%8A%A4%ED%83%80&amp;action=edit&amp;section=12" title="부분 편집: 국가대표팀 경력">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>2003년 7월 17일, 알레스는 <a href="/wiki/%EB%A9%95%EC%8B%9C%EC%BD%94_%EC%B6%95%EA%B5%AC_%EA%B5%AD%EA%B0%80%EB%8C%80%ED%91%9C%ED%8C%80" title="멕시코 축구 국가대표팀">멕시코</a>와의 <a href="/wiki/2003%EB%85%84_CONCACAF_%EA%B3%A8%EB%93%9C%EC%BB%B5" title="2003년 CONCACAF 골드컵">2003년 CONCACAF 골드컵</a> 경기에서 <a href="/wiki/%EB%B8%8C%EB%9D%BC%EC%A7%88_%EC%B6%95%EA%B5%AC_%EA%B5%AD%EA%B0%80%EB%8C%80%ED%91%9C%ED%8C%80" title="브라질 축구 국가대표팀">브라질 국가대표</a> 데뷔전을 치르었다. 그는 비록 <a class="new" href="/w/index.php?title=%EB%B8%8C%EB%9D%BC%EC%A7%88_U-23_%EC%B6%95%EA%B5%AC_%EA%B5%AD%EA%B0%80%EB%8C%80%ED%91%9C%ED%8C%80&amp;action=edit&amp;redlink=1" title="브라질 U-23 축구 국가대표팀 (없는 문서)">브라질 U-23 국가대표팀</a> 일원으로 대회에 참여하였으나, 정식 토너먼트였으므로, <a class="mw-redirect" href="/wiki/%EA%B5%AD%EC%A0%9C_%EC%B6%95%EA%B5%AC_%EC%97%B0%EB%A7%B9" title="국제 축구 연맹">FIFA</a> 출전 기록에 남게 되었다.<sup class="reference" id="cite_ref-33"><a href="#cite_note-33">[33]</a></sup></p>
<p>그는 <a href="/wiki/2006%EB%85%84_FIFA_%EC%9B%94%EB%93%9C%EC%BB%B5" title="2006년 FIFA 월드컵">2006년 FIFA 월드컵</a> 최종 엔트리 발탁을 대기하고 있었다. 그러나, 월드컵 후, 그는 <a href="/wiki/%EB%91%A5%EA%B0%80" title="둥가">둥가</a> 신임 국가대표팀 감독에 의해 차출되었다. 그는 결승전에서 숙적 <a href="/wiki/%EC%95%84%EB%A5%B4%ED%97%A8%ED%8B%B0%EB%82%98_%EC%B6%95%EA%B5%AC_%EA%B5%AD%EA%B0%80%EB%8C%80%ED%91%9C%ED%8C%80" title="아르헨티나 축구 국가대표팀">아르헨티나</a>를 3-0으로 대파한 브라질의 <a href="/wiki/2007%EB%85%84_%EC%BD%94%ED%8C%8C_%EC%95%84%EB%A9%94%EB%A6%AC%EC%B9%B4" title="2007년 코파 아메리카">2007년 코파 아메리카</a> 우승 주역이었다. 그는 부상으로 <a href="/wiki/2009%EB%85%84_FIFA_%EC%BB%A8%ED%8E%98%EB%8D%94%EB%A0%88%EC%9D%B4%EC%85%98%EC%8A%A4%EC%BB%B5" title="2009년 FIFA 컨페더레이션스컵">2009년 FIFA 컨페더레이션스컵</a> 참가가 불발되었다.</p>
<p>5월 11일, 그는 <a href="/wiki/2010%EB%85%84_FIFA_%EC%9B%94%EB%93%9C%EC%BB%B5" title="2010년 FIFA 월드컵">2010년 FIFA 월드컵</a>의 예비 엔트리 7인에 포함되어 있었다.<sup class="reference" id="cite_ref-34"><a href="#cite_note-34">[34]</a></sup></p>
