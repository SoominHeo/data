<h1 class="firstHeading" id="firstHeading" lang="ko">알렉사 인터넷</h1>
<div class="mw-body-content" id="bodyContent">

<div class="mw-content-ltr" dir="ltr" id="mw-content-text" lang="ko"><script>function mfTempOpenSection(id){var block=document.getElementById("mf-section-"+id);block.className+=" open-block";block.previousSibling.className+=" open-block";}</script><p><b>알렉사 인터넷 주식회사</b>(Alexa Internet Inc.)는 <a href="/wiki/%EB%AF%B8%EA%B5%AD" title="미국">미국</a>의 <a href="/wiki/%EC%95%84%EB%A7%88%EC%A1%B4%EB%8B%B7%EC%BB%B4" title="아마존닷컴">아마존닷컴</a> (위치: <a href="/wiki/%EC%8B%9C%EC%95%A0%ED%8B%80" title="시애틀">시애틀</a>, <a href="/wiki/%EC%9B%8C%EC%8B%B1%ED%84%B4_%EC%A3%BC" title="워싱턴 주">워싱턴 주</a>)의 <a href="/wiki/%EC%9E%90%ED%9A%8C%EC%82%AC" title="자회사">자회사</a> 로서 <a href="/wiki/%EC%BA%98%EB%A6%AC%ED%8F%AC%EB%8B%88%EC%95%84_%EC%A3%BC" title="캘리포니아 주">캘리포니아 주</a> 의 <a href="/wiki/%EC%83%8C%ED%94%84%EB%9E%80%EC%8B%9C%EC%8A%A4%EC%BD%94" title="샌프란시스코">샌프란시스코</a> 에 소재한다. 알렉사는 웹상에서 다른 <a href="/wiki/%EC%9B%B9%EC%82%AC%EC%9D%B4%ED%8A%B8" title="웹사이트">웹사이트</a>(예: <a href="/wiki/%EC%95%BC%ED%9B%84!" title="야후!">야후!</a>, <a href="/wiki/%EA%B5%AC%EA%B8%80" title="구글">구글</a>)들의 정보를 제공하는 웹사이트를 운영하는 기업으로 유명하다. 인기도 순위와 페이지 뷰, 툴바 기능을 서비스한다.</p>
