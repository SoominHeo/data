<p><b>알렉산더 데니스</b><sup class="reference" id="cite_ref-3"><a href="#cite_note-3">[3]</a></sup> (<span style="font-size: smaller;"><a href="/wiki/%EC%98%81%EC%96%B4" title="영어">영어</a>:</span> <span lang="en" xml:lang="en">Alexander Dennis</span>)는 <a href="/wiki/%EC%98%81%EA%B5%AD" title="영국">영국</a>의 자동차 메이커이다.</p>
<p><a href="/wiki/2004%EB%85%84" title="2004년">2004년</a>에 설립된 회사로 <a href="/wiki/%EC%98%81%EA%B5%AD" title="영국">영국</a> 및 <a href="/wiki/%EC%84%9C%EC%9C%A0%EB%9F%BD" title="서유럽">서유럽</a>의 버스 메이커 중에서 가장 규모가 크다. 또한 <a href="/wiki/%EC%9C%A0%EB%9F%BD" title="유럽">유럽</a>을 비롯해 <a href="/wiki/%EC%BA%90%EB%82%98%EB%8B%A4" title="캐나다">캐나다</a>, <a href="/wiki/%EC%A4%91%ED%99%94%EC%9D%B8%EB%AF%BC%EA%B3%B5%ED%99%94%EA%B5%AD" title="중화인민공화국">중화인민공화국</a>, <a href="/wiki/%EB%A7%90%EB%A0%88%EC%9D%B4%EC%8B%9C%EC%95%84" title="말레이시아">말레이시아</a>, <a href="/wiki/%EB%89%B4%EC%A7%88%EB%9E%9C%EB%93%9C" title="뉴질랜드">뉴질랜드</a>, <a href="/wiki/%EB%82%A8%EC%95%84%ED%94%84%EB%A6%AC%EC%B9%B4" title="남아프리카">남아프리카</a>, <a href="/wiki/%EB%AF%B8%EA%B5%AD" title="미국">미국</a>에서 사업을 하고 있으며 <a href="/wiki/2013%EB%85%84" title="2013년">2013년</a> 기준으로 <a href="/wiki/%EC%98%81%EA%B5%AD" title="영국">영국</a> 버스 시장의 34%의 점유율을 보이고 있다.<sup class="reference" id="cite_ref-4"><a href="#cite_note-4">[4]</a></sup> <a href="/wiki/2014%EB%85%84" title="2014년">2014년</a>에 <a class="new" href="/w/index.php?title=%EC%97%94_%EA%B8%80%EB%A1%9D&amp;action=edit&amp;redlink=1" title="엔 글록 (없는 문서)">엔 글록</a>과 <a class="new" href="/w/index.php?title=%EB%B8%8C%EB%9D%BC%EC%9D%B4%EC%96%B8_%EC%82%AC%EC%9A%B0%ED%84%B0&amp;action=edit&amp;redlink=1" title="브라이언 사우터 (없는 문서)">브라이언 사우터</a>가 공동으로 55%의 지분을 보유하게 되었다.<sup class="reference" id="cite_ref-5"><a href="#cite_note-5">[5]</a></sup></p>
<p><a href="/wiki/%EB%8C%80%ED%95%9C%EB%AF%BC%EA%B5%AD" title="대한민국">대한민국</a>의 경우 <a href="/wiki/%EA%B2%BD%EA%B8%B0%EB%8F%84" title="경기도">경기도</a>에서 <a href="/wiki/2014%EB%85%84" title="2014년">2014년</a> <a href="/wiki/12%EC%9B%94_9%EC%9D%BC" title="12월 9일">12월 9일</a>부터 <a href="/wiki/2%EC%B8%B5_%EB%B2%84%EC%8A%A4" title="2층 버스">2층 버스</a>인 <a class="new" href="/w/index.php?title=%EC%95%8C%EB%A0%89%EC%82%B0%EB%8D%94_%EB%8D%B0%EB%8B%88%EC%8A%A4_%EC%9D%B8%EB%B0%94%EC%9D%B4%EB%A1%9C500&amp;action=edit&amp;redlink=1" title="알렉산더 데니스 인바이로500 (없는 문서)">인바이로500</a><sup class="reference" id="cite_ref-6"><a href="#cite_note-6">[6]</a></sup>의 <a href="/wiki/%EB%B6%81%EC%95%84%EB%A9%94%EB%A6%AC%EC%B9%B4" title="북아메리카">북아메리카</a> 사양을 반입하여 직행좌석버스, 광역급행버스 노선에서<sup class="reference" id="cite_ref-7"><a href="#cite_note-7">[7]</a></sup> 1개월 정도 시범운행을 하였다.</p>
