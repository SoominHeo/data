<div class="mw-content-ltr" dir="ltr" id="mw-content-text" lang="ko"><script>function mfTempOpenSection(id){var block=document.getElementById("mf-section-"+id);block.className+=" open-block";block.previousSibling.className+=" open-block";}</script><p><b>알렉산더 스튜어트</b>(Alexander Stewart, <a href="/wiki/1214%EB%85%84" title="1214년">1214년</a> ~ <a href="/wiki/1283%EB%85%84" title="1283년">1283년</a>) 또는 <b>알렉산더 던도널드</b>(Alexander of Dundonald)는 <a href="/wiki/%EC%8A%A4%EC%BD%94%ED%8B%80%EB%9E%9C%EB%93%9C" title="스코틀랜드">스코틀랜드</a>의 귀족으로 제4대 <a href="/wiki/%EC%8A%A4%EC%BD%94%ED%8B%80%EB%9E%9C%EB%93%9C_%EC%99%95%EC%8B%A4_%EC%B5%9C%EA%B3%A0%EC%8B%9C%EC%A2%85%EA%B4%80" title="스코틀랜드 왕실 최고시종관">스코틀랜드 왕실 최고시종관</a>(High Steward of Scotland)을 역임했다. 두 아들 <a class="new" href="/w/index.php?title=%EC%A0%9C%EC%9E%84%EC%8A%A4_%EC%8A%A4%ED%8A%9C%EC%96%B4%ED%8A%B8_(5%EB%8C%80_%EC%8A%A4%ED%8A%9C%EC%96%B4%ED%8A%B8_%EC%8B%9C%EC%A2%85%EA%B4%80)&amp;action=edit&amp;redlink=1" title="제임스 스튜어트 (5대 스튜어트 시종관) (없는 문서)">제임스 스튜어트</a>와 <a class="new" href="/w/index.php?title=%EC%A1%B4_%EC%8A%A4%ED%8A%9C%EC%96%B4%ED%8A%B8_(%EB%B3%B8%ED%82%AC_%EA%B2%BD)&amp;action=edit&amp;redlink=1" title="존 스튜어트 (본킬 경) (없는 문서)">존 스튜어트</a>를 통해 <a class="mw-disambig" href="/wiki/%EB%A9%94%EB%A6%AC_%EC%8A%A4%ED%8A%9C%EC%96%B4%ED%8A%B8" title="메리 스튜어트">메리 스튜어트</a>와 <a class="new" href="/w/index.php?title=%EB%8B%A8%EB%A6%AC_%EA%B2%BD&amp;action=edit&amp;redlink=1" title="단리 경 (없는 문서)">단리 경</a> <a class="mw-disambig" href="/wiki/%ED%97%A8%EB%A6%AC_%EC%8A%A4%ED%8A%9C%EC%96%B4%ED%8A%B8" title="헨리 스튜어트">헨리 스튜어트</a>의 선조가 된다.</p>
<h2><span class="mw-headline" id=".EC.83.9D.EC.95.A0">생애</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%95%8C%EB%A0%89%EC%82%B0%EB%8D%94_%EC%8A%A4%ED%8A%9C%EC%96%B4%ED%8A%B8&amp;action=edit&amp;section=1" title="부분 편집: 생애">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>3대 시종관을 지낸 <a class="new" href="/w/index.php?title=%EB%B0%9C%ED%84%B0_%EC%8A%A4%ED%8A%9C%EC%96%B4%ED%8A%B8_(3%EB%8C%80_%EC%8A%A4%ED%8A%9C%EC%96%B4%ED%8A%B8_%EC%8B%9C%EC%A2%85%EA%B4%80)&amp;action=edit&amp;redlink=1" title="발터 스튜어트 (3대 스튜어트 시종관) (없는 문서)">발터 스튜어트</a>와 베토크의 아들로, 외가쪽으로는 <a class="new" href="/w/index.php?title=%EB%8D%98%EC%BC%88%EB%93%9C_%EC%99%95%EA%B0%80&amp;action=edit&amp;redlink=1" title="던켈드 왕가 (없는 문서)">던켈드 왕가</a>의 후손이었다. 외할머니 마조리에는 <a class="new" href="/w/index.php?title=%EB%8D%B0%EC%9D%B4%EB%B9%84%EB%93%9C_1%EC%84%B8&amp;action=edit&amp;redlink=1" title="데이비드 1세 (없는 문서)">데이비드 1세</a>의 서자인 <a class="new" href="/w/index.php?title=%EB%85%B8%EC%84%AC%EB%B2%8C%EB%A6%AC%EC%95%84&amp;action=edit&amp;redlink=1" title="노섬벌리아 (없는 문서)">노섬벌리아</a> <a href="/wiki/%EB%B0%B1%EC%9E%91" title="백작">백작</a> 헨리의 딸이었다.</p>
<p>그는 <a href="/wiki/1246%EB%85%84" title="1246년">1246년</a> 아버지 발터 사후 시종관직을 이어받아 <a href="/wiki/1283%EB%85%84" title="1283년">1283년</a>까지 재직했다. 시종관직은 아들 <a class="new" href="/w/index.php?title=%EC%A0%9C%EC%9E%84%EC%8A%A4_%EC%8A%A4%ED%8A%9C%EC%96%B4%ED%8A%B8_(5%EB%8C%80_%EC%8A%A4%ED%8A%9C%EC%96%B4%ED%8A%B8_%EC%8B%9C%EC%A2%85%EA%B4%80)&amp;action=edit&amp;redlink=1" title="제임스 스튜어트 (5대 스튜어트 시종관) (없는 문서)">제임스 스튜어트</a>에게로 이어졌다.</p>
