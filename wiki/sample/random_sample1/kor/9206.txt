<p><b>알렉산더 해밀턴</b>(Alexander Hamilton, <a href="/wiki/1755%EB%85%84" title="1755년">1755년</a> 혹은 <a href="/wiki/1757%EB%85%84" title="1757년">1757년</a><sup class="reference" id="cite_ref-1"><a href="#cite_note-1">[1]</a></sup> <a href="/wiki/1%EC%9B%94_11%EC%9D%BC" title="1월 11일">1월 11일</a> ~ <a href="/wiki/1804%EB%85%84" title="1804년">1804년</a> <a href="/wiki/7%EC%9B%94_12%EC%9D%BC" title="7월 12일">7월 12일</a>)은 <a href="/wiki/%EB%AF%B8%EA%B5%AD" title="미국">미국</a>의 <a href="/wiki/%EB%B2%95%EB%A5%A0%EA%B0%80" title="법률가">법률가</a>이자 <a class="mw-redirect" href="/wiki/%EC%A0%95%EC%B9%98%EC%9D%B8" title="정치인">정치인</a>, 재정가, <a class="mw-redirect" href="/wiki/%EC%82%AC%EC%83%81%EA%B0%80" title="사상가">정치 사상가</a>이다. <a href="/wiki/%EB%AF%B8%EA%B5%AD_%EA%B1%B4%EA%B5%AD%EC%9D%98_%EC%95%84%EB%B2%84%EC%A7%80" title="미국 건국의 아버지">미국 건국의 아버지</a>(Founding Fathers) 중 한 명으로 꼽히며, <a href="/wiki/1787%EB%85%84" title="1787년">1787년</a> <a class="mw-redirect" href="/wiki/%EB%AF%B8%EA%B5%AD_%ED%97%8C%EB%B2%95" title="미국 헌법">미국 헌법</a>의 제정에 공헌했다. 초대 워싱턴 정부 시절 <a class="new" href="/w/index.php?title=%EC%9E%AC%EB%AC%B4%EB%B6%80_(%EB%AF%B8%EA%B5%AD)&amp;action=edit&amp;redlink=1" title="재무부 (미국) (없는 문서)">재무부</a> 장관으로 재직했다. 알렉산더 해밀턴은 카리브해의 <a class="mw-redirect" href="/wiki/%EB%84%A4%EB%B9%84%EC%8A%A4" title="네비스">네비스</a> 섬에서 <a href="/wiki/%EC%8A%A4%EC%BD%94%ED%8B%80%EB%9E%9C%EB%93%9C" title="스코틀랜드">스코틀랜드</a>인 아버지와 <a href="/wiki/%EC%9C%84%EA%B7%B8%EB%85%B8" title="위그노">위그노</a>의 핏줄을 이어받은 어머니에게 태어났다. <a href="/wiki/%EC%A1%B0%EC%A7%80_%EC%9B%8C%EC%8B%B1%ED%84%B4" title="조지 워싱턴">조지 워싱턴</a>과 뜻을 같이 하는 <a class="mw-redirect" href="/wiki/%EC%97%B0%EB%B0%A9%EC%A3%BC%EC%9D%98%EC%9E%90" title="연방주의자">연방주의자</a>로 <a href="/wiki/%ED%86%A0%EB%A8%B8%EC%8A%A4_%EC%A0%9C%ED%8D%BC%EC%8A%A8" title="토머스 제퍼슨">토머스 제퍼슨</a>을 비롯한 <a class="new" href="/w/index.php?title=%EB%B0%98_%EC%97%B0%EB%B0%A9%EC%A3%BC%EC%9D%98%EC%9E%90&amp;action=edit&amp;redlink=1" title="반 연방주의자 (없는 문서)">반 연방주의자</a>와 대립하였다. 특히 <a href="/wiki/%ED%86%A0%EB%A8%B8%EC%8A%A4_%EC%A0%9C%ED%8D%BC%EC%8A%A8" title="토머스 제퍼슨">토머스 제퍼슨</a>과 많이 초기 재정 정책으로 대립했는데, 해밀턴이 독립전쟁 때의 빚을 모든 주가 공평하게 나눠 내자고 한 반면에 제퍼슨은 자신 주의 빚은 자신 주가 갚으라고 하였다. 해밀턴의 기반 지역이었던 <a class="mw-redirect" href="/wiki/%EB%89%B4_%EC%9E%89%EA%B8%80%EB%9E%9C%EB%93%9C" title="뉴 잉글랜드">뉴 잉글랜드</a>는 공업 위주 지역이라 빚이 많았는데, 결국 수도를 북부인 뉴욕에서 남부와 북부 경계선 즈음에 위치한 워싱턴 D.C로 옮기고 모든 빚을 나눠 갚는다는 절충안을 내놓았고 제퍼슨이 이를 수용하게 된다. 제퍼슨과는 <a href="/wiki/1791%EB%85%84" title="1791년">1791년</a> <a class="new" href="/w/index.php?title=%EB%AF%B8%ED%95%A9%EC%A4%91%EA%B5%AD_%EC%A0%9C1_%EC%9D%80%ED%96%89&amp;action=edit&amp;redlink=1" title="미합중국 제1 은행 (없는 문서)">미국 제1 은행</a> 설립 당시에도 큰 마찰을 빚었는데, 제퍼슨은 은행 설립에 반대했으나 해밀턴은 수정헌법을 근거로 설립을 주장했고, 당시 대통령 워싱턴이 해밀턴의 주장을 들어줌으로서 아메리카 합중국 은행을 설립하게 된다. 이 당시 제퍼슨은 "<a class="new" href="/w/index.php?title=%EC%99%95%EB%8B%B9%EC%A3%BC%EC%9D%98&amp;action=edit&amp;redlink=1" title="왕당주의 (없는 문서)">왕당주의</a>"라며 강도 높게 비판했다. 워싱턴이 정계에서 물러난 뒤 연방주의자당은 애덤스 파와 해밀턴 파로 나뉘게 되고, 결국 <a href="/wiki/1800%EB%85%84" title="1800년">1800년</a> 대통령 선거에서 제퍼슨에게 대통령을 내주고 의회마저 장악당하게 된다.</p>

<h2><span class="mw-headline" id=".EC.BA.90.EB.A6.AC.EB.B9.84.EC.96.B8.EC.97.90.EC.84.9C.EC.9D.98_.EC.96.B4.EB.A6.B0_.EC.8B.9C.EC.A0.88">캐리비언에서의 어린 시절</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%95%8C%EB%A0%89%EC%82%B0%EB%8D%94_%ED%95%B4%EB%B0%80%ED%84%B4&amp;action=edit&amp;section=1" title="부분 편집: 캐리비언에서의 어린 시절">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>알렉산더 해밀턴은 <a href="/wiki/%EB%A6%AC%EC%9B%8C%EB%93%9C_%EC%A0%9C%EB%8F%84" title="리워드 제도">리워드 제도</a> 네비스 섬의 수도인 찰스타운에서 태어났다. 네비스는 영국 서인도 제도 중에 하나이다. 해밀턴은 프랑스 위그노 후손인 레이첼 포셋과 스코틀랜드 출신인 제임스 해밀턴 사이에서 태어났다.</p>
<p>해밀턴의 어머니는 어린 해밀턴을 데리고 <a href="/wiki/%EB%B2%84%EC%A7%84_%EC%A0%9C%EB%8F%84" title="버진 제도">버진 제도</a>에 있는 <a href="/wiki/%EC%84%B8%EC%9D%B8%ED%8A%B8%ED%81%AC%EB%A1%9C%EC%9D%B4_%EC%84%AC" title="세인트크로이 섬">세인트크로이 섬</a>으로 이주했는데, 당시 세인트크로이는 덴마크의 지배를 받고 있었다. 해밀턴의 출생년도가 1757년인지 1755년인지는 확실하지 않다. 해밀턴이 북아메리카에 도착한 이후 대부분의 역사적 증거들은 해밀턴이 1757년에 태어났다는 사실을 보여준다. 하지만 1930년대 덴마크에서 해밀턴의 캐리비언 제도에서의 어린 시절에 대해 처음 발간된 기록에서는 1755년생으로 보고 있다. 해밀턴은 <a href="/wiki/13%EA%B0%9C_%EC%8B%9D%EB%AF%BC%EC%A7%80" title="13개 식민지">13개 식민지</a>에 처음 도착했을 때 출생년도를 1757년으로 기입했다.</p>
<h2><span class="mw-headline" id=".EA.B5.90.EC.9C.A1">교육</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%95%8C%EB%A0%89%EC%82%B0%EB%8D%94_%ED%95%B4%EB%B0%80%ED%84%B4&amp;action=edit&amp;section=2" title="부분 편집: 교육">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>1772년 가을에 해밀턴은 <a href="/wiki/%EB%89%B4%EC%A0%80%EC%A7%80_%EC%A3%BC" title="뉴저지 주">뉴저지 주</a> 엘리자베스타운에 있는 그래머 스쿨인 엘리자베스타운 아카데미에 도착한다.</p>
<h2><span class="mw-headline" id=".EC.9E.AC.EB.AC.B4.EC.9E.A5.EA.B4.80">재무장관</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%95%8C%EB%A0%89%EC%82%B0%EB%8D%94_%ED%95%B4%EB%B0%80%ED%84%B4&amp;action=edit&amp;section=3" title="부분 편집: 재무장관">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p><a href="/wiki/1789%EB%85%84" title="1789년">1789년</a> 9월 11일 조지 워싱턴 대통령은 초대 재무장관으로 해밀턴을 지명한다. <a href="/wiki/1795%EB%85%84" title="1795년">1795년</a> 1월 마지막 날 해밀턴은 사임한다. 의회의 구성과 활동을 필두로 미국 정부의 대부분이 이 5년 동안 구축된다.</p>
<h2><span class="mw-headline" id=".EA.B2.B0.ED.88.AC">결투</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%95%8C%EB%A0%89%EC%82%B0%EB%8D%94_%ED%95%B4%EB%B0%80%ED%84%B4&amp;action=edit&amp;section=4" title="부분 편집: 결투">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<div class="rellink boilerplate seealso"><img alt="" data-file-height="128" data-file-width="128" height="16" src="//upload.wikimedia.org/wikipedia/commons/thumb/e/ec/Crystal_Clear_app_xmag.svg/16px-Crystal_Clear_app_xmag.svg.png" srcset="//upload.wikimedia.org/wikipedia/commons/thumb/e/ec/Crystal_Clear_app_xmag.svg/24px-Crystal_Clear_app_xmag.svg.png 1.5x, //upload.wikimedia.org/wikipedia/commons/thumb/e/ec/Crystal_Clear_app_xmag.svg/32px-Crystal_Clear_app_xmag.svg.png 2x" width="16"/> <a href="/wiki/%EC%97%90%EB%9F%B0_%EB%B2%84%EC%99%80_%EC%95%8C%EB%A0%89%EC%82%B0%EB%8D%94_%ED%95%B4%EB%B0%80%ED%84%B4%EC%9D%98_%EA%B2%B0%ED%88%AC" title="에런 버와 알렉산더 해밀턴의 결투">에런 버와 알렉산더 해밀턴의 결투</a> 문서를 참고하십시오.</div>
<p>알렉산더 해밀튼은 <a href="/wiki/%EC%97%90%EB%9F%B0_%EB%B2%84" title="에런 버">에런 버</a>로부터 <a href="/wiki/1804%EB%85%84" title="1804년">1804년</a> <a href="/wiki/7%EC%9B%94_11%EC%9D%BC" title="7월 11일">7월 11일</a>에 결투를 신청받았다. 결투로 아들을 잃은 적이 있는 알렉산더 해밀턴의 입장에서는 결투를 원치 않았으나 자신의 명예 때문에 억지로 결투를 승낙했다. 그 당시 뉴욕주의 법은 결투를 금지하고 있었기 때문에 알렉산더 해밀턴과 애런 버는 강을 건너서 결투가 허가된 지역인 뉴저지 주의 위호켄 외곽의 허드슨 강변 바위절벽 아래로 이동한 뒤 그 곳에서 결투를 벌였다. 결과는 애런 버가 승리하였고 패배한 알렉산더 해밀턴은 다음날 사망했다.</p>
<h2><span class="mw-headline" id=".EC.82.AC.EC.83.81">사상</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%95%8C%EB%A0%89%EC%82%B0%EB%8D%94_%ED%95%B4%EB%B0%80%ED%84%B4&amp;action=edit&amp;section=5" title="부분 편집: 사상">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>해밀턴은 영국의 귀족주의를 열렬히 찬미하고 숭배하였으며, 민주주의가 국민들의 변덕과 감정적이고 충동적인 분위기에 의해 좌우되는 통치라고 이해하였다.<sup class="reference" id="cite_ref-autogenerated1_2-0"><a href="#cite_note-autogenerated1-2">[2]</a></sup><sup class="reference" id="cite_ref-3"><a href="#cite_note-3">[3]</a></sup> 동료인 <a href="/wiki/%EC%A0%9C%EC%9E%84%EC%8A%A4_%EB%A7%A4%EB%94%94%EC%8A%A8" title="제임스 매디슨">제임스 매디슨</a>과 달리, <a href="/wiki/%EB%AF%B8%EA%B5%AD_%EC%83%81%EC%9B%90" title="미국 상원">미국 상원</a>을 국민의 대표가 아니라, 귀족의 대표라고 보았다.<sup class="reference" id="cite_ref-autogenerated1_2-1"><a href="#cite_note-autogenerated1-2">[2]</a></sup></p>
