<p><b>알렉산드르 리호라비치 루카셴코</b>(<span style="font-size: smaller;"><a href="/wiki/%EB%B2%A8%EB%9D%BC%EB%A3%A8%EC%8A%A4%EC%96%B4" title="벨라루스어">벨라루스어</a>:</span> <span lang="be" xml:lang="be">Алякса́ндр Рыго́равіч Лукашэ́нка</span> <small>알략산드르 리호라비치 루카셴카</small> <small><a href="/wiki/%EC%9C%84%ED%82%A4%EB%B0%B1%EA%B3%BC:IPA" title="위키백과:IPA">IPA</a>: </small><span class="IPA" title="IPA 발음">[alʲaˈksandr rɨˈɣoravʲɪtʃ ɫukaˈʂɛnka]</span>, <span style="font-size: smaller;"><a href="/wiki/%EB%9F%AC%EC%8B%9C%EC%95%84%EC%96%B4" title="러시아어">러시아어</a>:</span> <span lang="ru" xml:lang="ru">Алекса́ндр Григо́рьевич Лукаше́нко</span> <small>알렉산드르 그리고리예비치 루카셴코<sup>[<a href="/wiki/%EC%9C%84%ED%82%A4%EB%B0%B1%EA%B3%BC:%EB%9F%AC%EC%8B%9C%EC%95%84%EC%96%B4%EC%9D%98_%ED%95%9C%EA%B8%80_%ED%91%9C%EA%B8%B0" title="위키백과:러시아어의 한글 표기">*</a>]</sup></small>, <span style="font-size: smaller;"><a href="/wiki/%EB%AC%B8%ED%99%94%EC%96%B4" title="문화어">문화어</a>:</span> 알렉싼드르 그리고리예비치 루까쉔꼬, <a href="/wiki/1954%EB%85%84" title="1954년">1954년</a> <a href="/wiki/8%EC%9B%94_30%EC%9D%BC" title="8월 30일">8월 30일</a> ~ )는 <a href="/wiki/%EB%B2%A8%EB%9D%BC%EB%A3%A8%EC%8A%A4" title="벨라루스">벨라루스</a>의 초대(현직) <a href="/wiki/%EB%8C%80%ED%86%B5%EB%A0%B9" title="대통령">대통령</a>이다. 2010년 대선에서 4선에 성공하고 최근 2015년 대선에서 5선에 성공하여 총 26년 장기 집권을 이어가고 있다.</p>
<p>그는 집단농장 관리인 출신의 무명이었으나 <a class="mw-redirect" href="/wiki/%EC%86%8C%EB%B9%84%EC%97%90%ED%8A%B8_%EC%97%B0%EB%B0%A9" title="소비에트 연방">소비에트 연방</a>의 <a class="mw-redirect" href="/wiki/%EC%86%8C%EB%B9%84%EC%97%90%ED%8A%B8_%EC%B5%9C%EA%B3%A0%ED%9A%8C%EC%9D%98" title="소비에트 최고회의">소비에트 최고회의</a> 의원에 선출되었으며 1991년에 열린 최고회의에서 벨라루스(당시 <a href="/wiki/%EB%B2%A8%EB%A1%9C%EB%A3%A8%EC%8B%9C_%EC%86%8C%EB%B9%84%EC%97%90%ED%8A%B8_%EC%82%AC%ED%9A%8C%EC%A3%BC%EC%9D%98_%EA%B3%B5%ED%99%94%EA%B5%AD" title="벨로루시 소비에트 사회주의 공화국">벨로루시 소비에트 사회주의 공화국</a>) 출신으로는 유일하게 소련의 해체를 반대하여 눈길을 끌기도 하였다.</p>
<p>벨라루스가 소련에서 분리·독립한 후에는 반부패 운동가로 유명해졌으며 독립 이후에 새로 제정된 벨라루스 헌법을 통해 처음 치러진 1994년 대통령 선거에서 압도적인 지지로 당선, 벨라루스의 제2대 대통령에 취임하였다. 그의 당선은 <a href="/wiki/%EB%9F%AC%EC%8B%9C%EC%95%84" title="러시아">러시아</a>와의 관계 강화 제시, 강력한 부패 척결 의지와 민족주의 주장 등에 힘입은 것이다.</p>
<p>그러나 대통령에 취임한 이후에는 차츰 독재로 흘러가기 시작하였는데, 1996년에 임기를 연장하여 2001년에 재선되었다. 2004년에는 다시 헌법을 개정해 초대 대통령의 연임 제한을 폐지한 데 이어서 2006년과 2010년에 열린 대통령 선거에서 다시 압도적인 지지로 당선되었다.</p>
<p>1999년 12월 8일 러시아와 벨라루스 양국이 조인한 <a href="/wiki/%EB%9F%AC%EC%8B%9C%EC%95%84-%EB%B2%A8%EB%9D%BC%EB%A3%A8%EC%8A%A4_%EA%B5%AD%EA%B0%80_%EC%97%B0%ED%95%A9_%EC%B0%BD%EC%84%A4_%EC%A1%B0%EC%95%BD" title="러시아-벨라루스 국가 연합 창설 조약">러시아-벨라루스 국가 연합 창설 조약</a>이 조인되면서 2000년 1월 <a href="/wiki/%EB%9F%AC%EC%8B%9C%EC%95%84-%EB%B2%A8%EB%9D%BC%EB%A3%A8%EC%8A%A4_%EA%B5%AD%EA%B0%80_%EC%97%B0%ED%95%A9" title="러시아-벨라루스 국가 연합">러시아-벨라루스 국가 연합</a> 초대 최고 국가 평의회 의장에 취임하였다. 1997년에는 <a href="/wiki/%EB%8C%80%ED%95%9C%EB%AF%BC%EA%B5%AD" title="대한민국">대한민국</a>을 방문해서 <a href="/wiki/%EA%B9%80%EC%98%81%EC%82%BC" title="김영삼">김영삼</a> 대통령과 정상 회담을 열기도 했다. 독재와 벨라루스의 인권 탄압으로 인해 <a href="/wiki/%EC%9C%A0%EB%9F%BD" title="유럽">유럽</a>과 <a href="/wiki/%EB%AF%B8%EA%B5%AD" title="미국">미국</a>으로부터 "유럽 최후의 독재자"라는 비난을 받고 있다. 최근에는 공식 행사에서 4살된 자신의 막내 아들을 자주 데려가는 것을 보고 일부에선 그가 아들에게 권력을 물려줄 것이란 추측이 나오고 있다.<sup class="reference" id="cite_ref-1"><a href="#cite_note-1">[1]</a></sup></p>
<p>최근의 보도에 따르면, 루카셴코는 고민을 가지고 있으며 <a href="/wiki/%EB%B2%A8%EB%9D%BC%EB%A3%A8%EC%8A%A4" title="벨라루스">벨라루스</a>가 언젠가는 <a href="/wiki/%EC%9A%B0%ED%81%AC%EB%9D%BC%EC%9D%B4%EB%82%98" title="우크라이나">우크라이나</a>의 반정부 시위처럼 독재가 붕괴될 위기에 처할까봐, 약간 두려워하는 증상을 보이고 있다고 한다.<sup class="reference" id="cite_ref-2"><a href="#cite_note-2">[2]</a></sup> 최근에 들어서서 <a href="/wiki/%EC%9A%B0%ED%81%AC%EB%9D%BC%EC%9D%B4%EB%82%98" title="우크라이나">우크라이나</a>의 반정부 시위를 비판하기도 하였다.<sup class="reference" id="cite_ref-3"><a href="#cite_note-3">[3]</a></sup></p>
