<h1 class="firstHeading" id="firstHeading" lang="ko">알렉세이 안토노프</h1>
<div class="mw-body-content" id="bodyContent">
<div id="siteSub">위키백과, 우리 모두의 백과사전.</div>
<div id="contentSub"></div>
<div class="mw-jump" id="jump-to-nav">
					이동:					<a href="#mw-head">둘러보기</a>, 					<a href="#p-search">검색</a>
</div>
<div class="mw-content-ltr" dir="ltr" id="mw-content-text" lang="ko"><script>function mfTempOpenSection(id){var block=document.getElementById("mf-section-"+id);block.className+=" open-block";block.previousSibling.className+=" open-block";}</script><p><b>알렉세이 이노켄티예비치 안토노프</b> (<span style="font-size: smaller;"><a href="/wiki/%EB%9F%AC%EC%8B%9C%EC%95%84%EC%96%B4" title="러시아어">러시아어</a>:</span> <span lang="ru" xml:lang="ru">Алексей Иннокентьевич Антонов</span>, <a href="/wiki/1896%EB%85%84" title="1896년">1896년</a> <a href="/wiki/9%EC%9B%94_9%EC%9D%BC" title="9월 9일">9월 9일</a> ~ <a href="/wiki/1962%EB%85%84" title="1962년">1962년</a> <a href="/wiki/6%EC%9B%94_16%EC%9D%BC" title="6월 16일">6월 16일</a>)는 <a class="mw-redirect" href="/wiki/%EC%86%8C%EB%B9%84%EC%97%90%ED%8A%B8%EA%B5%B0" title="소비에트군">소비에트군</a>의 장군으로, <a href="/wiki/%EC%A0%9C2%EC%B0%A8_%EC%84%B8%EA%B3%84_%EB%8C%80%EC%A0%84" title="제2차 세계 대전">제2차 세계 대전</a>에서의 공로로 <a class="extiw" href="https://en.wikipedia.org/wiki/Order_of_Victory" title="en:Order of Victory">승리 훈장</a>을 수여받았다.</p>
<p><a href="/wiki/1896%EB%85%84" title="1896년">1896년</a>, 현 <a class="mw-redirect" href="/wiki/%EB%B2%A8%EB%A1%9C%EB%A3%A8%EC%8B%9C" title="벨로루시">벨로루시</a>의 <a href="/wiki/%ED%9D%90%EB%A1%9C%EB%93%9C%EB%82%98" title="흐로드나">흐로드나</a> (Hrodna) 에서 차르 체제 아래의 포병 장교의 아들로 태어났다. 1921년 <a class="mw-redirect" href="/wiki/%ED%94%84%EB%A3%AC%EC%A0%9C_%EA%B5%B0%EC%82%AC_%EC%95%84%EC%B9%B4%EB%8D%B0%EB%AF%B8" title="프룬제 군사 아카데미">프룬제 군사 아카데미</a> 를 졸업하고, <a href="/wiki/%EB%9F%AC%EC%8B%9C%EC%95%84_%EB%82%B4%EC%A0%84" title="러시아 내전">러시아 내전</a> 당시에 <a href="/wiki/%EB%B6%89%EC%9D%80_%EA%B5%B0%EB%8C%80" title="붉은 군대">붉은 군대</a> 측으로 참전하였다. 1938년에는 프룬제 육군사관학교의 교관이 되었다.</p>
<p>1941년, 소비에트 남서부 전선과 남부 전선의 참모장이 되었고, 이듬해 소비에트 군의 참모 차장 및 작전 과장이 되었다. 그의 임무는 다른 장교와 협력하여 군사 상황을 <a href="/wiki/%EC%9D%B4%EC%98%A4%EC%8B%9C%ED%94%84_%EC%8A%A4%ED%83%88%EB%A6%B0" title="이오시프 스탈린">스탈린</a>에게 알리는 것이었다.</p>
<p>1944년, 안토노프는 수석 대변인이 되어 <a href="/wiki/%EC%96%84%ED%83%80_%ED%9A%8C%EB%8B%B4" title="얄타 회담">얄타 회담</a>과 <a href="/wiki/%ED%8F%AC%EC%B8%A0%EB%8B%B4_%ED%9A%8C%EB%8B%B4" title="포츠담 회담">포츠담 회담</a> 양 회담에 출석하였다. 얄타 회담에서 그는 교통망에 대한 폭격으로 <a class="new" href="/w/index.php?title=%EC%A0%9C2%EC%B0%A8_%EC%84%B8%EA%B3%84_%EB%8C%80%EC%A0%84_%EB%93%9C%EB%A0%88%EC%8A%A4%EB%8D%B4_%ED%8F%AD%EA%B2%A9&amp;action=edit&amp;redlink=1" title="제2차 세계 대전 드레스덴 폭격 (없는 문서)">드레스덴</a> 공습을 유도하였고, 연합군이 소비에트 연방을 어떻게 지원할 수 있는지에 대하여 말하였다.</p>
<p>전후 안토노프는 자카프카스 군사 지구의 부사령관과 사령관을 역임하였고, 1955년에는 <a href="/wiki/%EB%B0%94%EB%A5%B4%EC%83%A4%EB%B0%94_%EC%A1%B0%EC%95%BD_%EA%B8%B0%EA%B5%AC" title="바르샤바 조약 기구">바르샤바 조약 기구</a> 통합군의 참모 총장을 겸임하였다. 1962년, 모스크바에서 사망할 때까지 이 직위를 유지하였다.</p>
<p>1956년 안토노프는 유명한 발레리나 <a href="/wiki/%EC%98%AC%EA%B0%80_%EB%A0%88%ED%8E%98%EC%8B%A0%EC%8A%A4%EC%B9%B4%EC%95%BC" title="올가 레페신스카야">올가 레페신스카야</a>와 재혼하였다.</p>
