<p><b>알렉스 게레로</b>({{|Al|ex|Alexander Guerrero}}, <a href="/wiki/1986%EB%85%84" title="1986년">1986년</a> <a href="/wiki/12%EC%9B%94_20%EC%9D%BC" title="12월 20일">12월 20일</a> ~ )는 <a href="/wiki/%EC%BF%A0%EB%B0%94" title="쿠바">쿠바</a>출신의 야구 선수이자, <a href="/wiki/%EC%9D%BC%EB%B3%B8" title="일본">일본</a> <a href="/wiki/%EC%84%BC%ED%8A%B8%EB%9F%B4_%EB%A6%AC%EA%B7%B8" title="센트럴 리그">센트럴 리그</a> <a class="mw-redirect" href="/wiki/%EC%A3%BC%EB%8B%88%EC%B9%98_%EB%93%9C%EB%9E%98%EA%B3%A4%EC%8A%A4" title="주니치 드래곤스">주니치 드래곤스</a>의 선수이다.</p>
<p>쿠바를 탈출한 후 <a href="/wiki/2013%EB%85%84" title="2013년">2013년</a>에 <a href="/wiki/%EC%95%84%EC%9D%B4%ED%8B%B0" title="아이티">아이티</a>로 망명한다. <a href="/wiki/10%EC%9B%94_22%EC%9D%BC" title="10월 22일">10월 22일</a>에는 <a href="/wiki/%EB%A1%9C%EC%8A%A4%EC%95%A4%EC%A0%A4%EB%A0%88%EC%8A%A4_%EB%8B%A4%EC%A0%80%EC%8A%A4" title="로스앤젤레스 다저스">로스앤젤레스 다저스</a>와 4년 총액 2,800만 달러에 계약을 맺는다.</p>
