<p><b>알렉스 퍼거슨</b> 경(<span style="font-size: smaller;"><a href="/wiki/%EC%98%81%EC%96%B4" title="영어">영어</a>:</span> <span lang="en" xml:lang="en">Sir<sup class="reference" id="cite_ref-.EA.B8.B0.EC.82.AC.EC.9E.91.EC.9C.84_2-0"><a href="#cite_note-.EA.B8.B0.EC.82.AC.EC.9E.91.EC.9C.84-2">[2]</a></sup> Alex Ferguson</span> <small>앨릭스 퍼거슨<sup>[<a href="/wiki/%EC%9C%84%ED%82%A4%EB%B0%B1%EA%B3%BC:%EC%98%81%EC%96%B4%EC%9D%98_%ED%95%9C%EA%B8%80_%ED%91%9C%EA%B8%B0" title="위키백과:영어의 한글 표기">*</a>]</sup></small>, <a href="/wiki/%EB%8C%80%EC%98%81_%EC%A0%9C%EA%B5%AD_%ED%9B%88%EC%9E%A5" title="대영 제국 훈장">CBE</a><sup class="reference" id="cite_ref-.ED.9B.88.EC.9E.A5_3-0"><a href="#cite_note-.ED.9B.88.EC.9E.A5-3">[3]</a></sup>, <a href="/wiki/1941%EB%85%84" title="1941년">1941년</a> <a href="/wiki/12%EC%9B%94_31%EC%9D%BC" title="12월 31일">12월 31일</a> ~ )은 <a href="/wiki/%EC%98%81%EA%B5%AD" title="영국">영국</a> <a href="/wiki/%EC%8A%A4%EC%BD%94%ED%8B%80%EB%9E%9C%EB%93%9C" title="스코틀랜드">스코틀랜드</a> 출신의 전 <a href="/wiki/%EC%B6%95%EA%B5%AC_%EA%B0%90%EB%8F%85" title="축구 감독">축구 감독</a>이다. 1986년부터 2013년까지 <a href="/wiki/%EB%A7%A8%EC%B2%B4%EC%8A%A4%ED%84%B0_%EC%9C%A0%EB%82%98%EC%9D%B4%ED%8B%B0%EB%93%9C_FC" title="맨체스터 유나이티드 FC">맨체스터 유나이티드 FC</a>의 감독을 맡았다.</p>
<p>선수 시절, <a href="/wiki/%EB%8D%98%ED%8E%8C%EB%A6%B0_%EC%95%A0%EC%8A%AC%EB%A0%88%ED%8B%B1_FC" title="던펌린 애슬레틱 FC">던펌린 애슬래틱</a> 및 <a href="/wiki/%EB%A0%88%EC%9D%B8%EC%A0%80%EC%8A%A4_FC" title="레인저스 FC">레인저스</a> 등 스코틀랜드 클럽에서 <a href="/wiki/%EA%B3%B5%EA%B2%A9%EC%88%98" title="공격수">공격수</a>로 활약하였으며 1965-66 시즌에 던펌린 애슬래틱 소속으로 득점왕을 하였다. 선수 생활 마지막엔 코치로서 활동하였고, <a class="new" href="/w/index.php?title=%EC%9D%B4%EC%8A%A4%ED%8A%B8_%EC%8A%A4%ED%84%B8%EB%A7%81%EC%85%94_FC&amp;action=edit&amp;redlink=1" title="이스트 스털링셔 FC (없는 문서)">이스트 스털링셔</a>와 <a href="/wiki/%EC%84%B8%EC%9D%B8%ED%8A%B8_%EB%AF%B8%EB%A0%8C_FC" title="세인트 미렌 FC">세인트 미렌</a>에서 감독직을 역임하였다. 이후 <a href="/wiki/%EC%95%A0%EB%B2%84%EB%94%98_FC" title="애버딘 FC">애버딘</a>에 취임하며, 스코틀랜드 리그 3회 우승, <a href="/wiki/%EC%8A%A4%EC%BD%94%ED%8B%80%EB%9E%9C%EB%93%9C_%EC%BB%B5" title="스코틀랜드 컵">스코틀랜드 컵</a> 4회 우승, 1983년 <a href="/wiki/UEFA_%EC%BB%B5%EC%9C%84%EB%84%88%EC%8A%A4%EC%BB%B5" title="UEFA 컵위너스컵">UEFA 컵위너스컵</a> 우승을 하며 감독으로서 큰 성공을 거두었다. 이후 전임 감독이었던 조크 스테인이 사망하자, <a href="/wiki/%EC%8A%A4%EC%BD%94%ED%8B%80%EB%9E%9C%EB%93%9C_%EC%B6%95%EA%B5%AC_%EA%B5%AD%EA%B0%80%EB%8C%80%ED%91%9C%ED%8C%80" title="스코틀랜드 축구 국가대표팀">스코틀랜드</a> 대표팀에 선임되어 <a href="/wiki/1986%EB%85%84_FIFA_%EC%9B%94%EB%93%9C%EC%BB%B5" title="1986년 FIFA 월드컵">1986년 FIFA 월드컵</a>에 참가하였다.</p>
<p>1986년 11월 <a class="mw-redirect" href="/wiki/%EB%A7%A8%EC%B2%B4%EC%8A%A4%ED%84%B0_%EC%9C%A0%EB%82%98%EC%9D%B4%ED%8B%B0%EB%93%9C" title="맨체스터 유나이티드">맨체스터 유나이티드</a>에 취임하였고, 26년 재임 중 <a href="/wiki/%ED%94%84%EB%A6%AC%EB%AF%B8%EC%96%B4%EB%A6%AC%EA%B7%B8" title="프리미어리그">프리미어리그</a> 우승 13회, <a href="/wiki/FA%EC%BB%B5_(%EC%9E%89%EA%B8%80%EB%9E%9C%EB%93%9C)" title="FA컵 (잉글랜드)">잉글랜드 FA컵</a> 5회, <a href="/wiki/UEFA_%EC%B1%94%ED%94%BC%EC%96%B8%EC%8A%A4%EB%A6%AC%EA%B7%B8" title="UEFA 챔피언스리그">UEFA 챔피언스리그</a> 2회 우승 등 총 38회의 트로피를 들어올렸다.<sup class="reference" id="cite_ref-4"><a href="#cite_note-4">[4]</a></sup> 영국 축구 역사상 최초로 <a href="/wiki/1999%EB%85%84" title="1999년">1999년</a> 맨체스터 유나이티드가 <a href="/wiki/%ED%8A%B8%EB%A0%88%EB%B8%94" title="트레블">트레블</a>(리그 우승, FA컵 우승, UEFA 챔피언스리그 우승을 동시에 이루는 것)을 달성 후, 이에 따른 공로를 인정받아 기사작위에 서임되었다.<sup class="reference" id="cite_ref-5"><a href="#cite_note-5">[5]</a></sup> 2010년 12월 19일, <a href="/wiki/%EB%A7%B7_%EB%B2%84%EC%A6%88%EB%B9%84" title="맷 버즈비">맷 버즈비</a>의 기록을 넘으면서 가장 오랫동안 맨체스터 유나이티드에 재임한 감독으로 기록되었다. 2012-13 시즌을 마지막으로 팀을 리그 우승시키며 은퇴하였다.</p>

<h2><span class="mw-headline" id=".EC.96.B4.EB.A6.B0_.EC.8B.9C.EC.A0.88">어린 시절</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%95%8C%EB%A0%89%EC%8A%A4_%ED%8D%BC%EA%B1%B0%EC%8A%A8&amp;action=edit&amp;section=1" title="부분 편집: 어린 시절">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>알렉스 퍼거슨은 1941년 12월 31일, <a href="/wiki/%EC%8A%A4%EC%BD%94%ED%8B%80%EB%9E%9C%EB%93%9C" title="스코틀랜드">스코틀랜드</a> <a href="/wiki/%EA%B8%80%EB%9E%98%EC%8A%A4%EA%B3%A0" title="글래스고">고번</a>에 있었던 할머니 집에서 태어났다. 조선소에서 일하던 아버지 알렉산더 비튼 퍼거슨과 어머니 엘리자베스 하디 사이에서 태어난 그는, 이후 667 고번 로드에 위치한 공동 주택에서 자랐는데 철거되기 전까지 부모님과 동생인 마틴과 함께 살았다.</p>
<p>브룸런 로드 초등학교, 고번 고등학교를 다녔으며<sup class="reference" id="cite_ref-6"><a href="#cite_note-6">[6]</a></sup><sup class="reference" id="cite_ref-7"><a href="#cite_note-7">[7]</a></sup> 유년시절, <a href="/wiki/%EB%A0%88%EC%9D%B8%EC%A0%80%EC%8A%A4_FC" title="레인저스 FC">레인저스</a>를 응원했다.<sup class="reference" id="cite_ref-8"><a href="#cite_note-8">[8]</a></sup><sup class="reference" id="cite_ref-9"><a href="#cite_note-9">[9]</a></sup></p>
<h2><span class="mw-headline" id=".EC.84.A0.EC.88.98_.EA.B2.BD.EB.A0.A5">선수 경력</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%95%8C%EB%A0%89%EC%8A%A4_%ED%8D%BC%EA%B1%B0%EC%8A%A8&amp;action=edit&amp;section=2" title="부분 편집: 선수 경력">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>퍼거슨의 선수 경력은 <a href="/wiki/%ED%80%B8%EC%8A%A4_%ED%8C%8C%ED%81%AC_FC" title="퀸스 파크 FC">퀸스 파크 FC</a> 소속으로 16살 때 <a href="/wiki/%EA%B3%B5%EA%B2%A9%EC%88%98" title="공격수">공격수</a>로 데뷔하였다.<sup class="reference" id="cite_ref-10"><a href="#cite_note-10">[10]</a></sup> 퍼거슨은 당시 자신의 첫 경기를 회고하며 "악몽"이었다고 밝혔다.<sup class="reference" id="cite_ref-nightmare_11-0"><a href="#cite_note-nightmare-11">[11]</a></sup> 스트랜레어를 상대로서 퀸스 파크 FC는 한 골을 득점하였으나 2-1로 패배하였다. 퍼거슨이 선수로서 잘 알려진 경기는 <a href="/wiki/1959%EB%85%84" title="1959년">1959년</a> <a href="/wiki/%EB%B0%95%EC%8B%B1_%EB%8D%B0%EC%9D%B4" title="박싱 데이">박싱 데이</a> 매치 때 <a class="new" href="/w/index.php?title=%ED%80%B8_%EC%98%A4%EB%B8%8C_%EC%82%AC%EC%9A%B0%EC%8A%A4_FC&amp;action=edit&amp;redlink=1" title="퀸 오브 사우스 FC (없는 문서)">퀸 오브 사우스</a>와의 원정 경기에서 7-1 대패한 경기에서 퀸스 파크 선수로서 유일하게 득점하였다.<sup class="reference" id="cite_ref-12"><a href="#cite_note-12">[12]</a></sup></p>
<p>퀸스 파크 선수로서 31경기에 출장해 20골을 기록한 퍼거슨은 <a href="/wiki/1960%EB%85%84" title="1960년">1960년</a>, 원치않던 <a href="/wiki/%EC%84%B8%EC%9D%B8%ED%8A%B8_%EC%A1%B4%EC%8A%A4%ED%86%A4_FC" title="세인트 존스톤 FC">세인트 존스톤</a>으로 이적하였다. 이적 후에도 꾸준히 골을 기록하였으나 적응 문제로 클럽에게 이적을 계속해서 요구했다. 퍼거슨은 다른 클럽으로 이적을 원했고 심지어 <a href="/wiki/%EC%BA%90%EB%82%98%EB%8B%A4" title="캐나다">캐나다</a>로 이주까지 고려했었다.<sup class="reference" id="cite_ref-13"><a href="#cite_note-13">[13]</a></sup> 그럼에도 세인트 존스톤은 퍼거슨과의 계약을 추진하였으나 퍼거슨은 <a href="/wiki/%EB%A0%88%EC%9D%B8%EC%A0%80%EC%8A%A4_FC" title="레인저스 FC">레인저스</a>를 상대로 <a href="/wiki/%ED%95%B4%ED%8A%B8%ED%8A%B8%EB%A6%AD" title="해트트릭">해트트릭</a>을 기록하는 놀라운 활약을 보였고 <a href="/wiki/1964%EB%85%84" title="1964년">1964년</a> 여름, <a href="/wiki/%EB%8D%98%ED%8E%8C%EB%A6%B0_%EC%95%A0%EC%8A%AC%EB%A0%88%ED%8B%B1_FC" title="던펌린 애슬레틱 FC">던펌린</a>으로 이적하여 아마추어 선수가 아닌 정규 선수로 계약하게 되었다.</p>
<p>1964-65 시즌 개막 후, 던펌린은 스코틀랜드 리그에서 강세를 보이면서 스코틀랜드 컵 결승까지 올랐으나, 퍼거슨은 세인트 존스톤과의 리그 경기에서 저조한 경기를 보여 결승전에서 출전하지 못했다. 던펌린은 결승에서 <a href="/wiki/%EC%85%80%ED%8B%B1_FC" title="셀틱 FC">셀틱</a>에게 3-2 패배를 당했고 그 이후, 리그 또한 1점 차로 우승에 실패하였다. 1965-66 시즌, 퍼거슨은 51경기에 출전하여 45골을 기록하였고 31골을 기록한 셀틱의 조 맥브라이드와 함께 리그 득점 랭킹 상위권에 올랐다.<sup class="reference" id="cite_ref-14"><a href="#cite_note-14">[14]</a></sup></p>
<p>이후 퍼거슨은 당시 최고 이적료인 6만 5천 파운드에 <a href="/wiki/%EB%A0%88%EC%9D%B8%EC%A0%80%EC%8A%A4_FC" title="레인저스 FC">레인저스</a>로 이적하였다. 퍼거슨은 1969년 스코틀랜드 컵 결승을 패배를 최악의 경기로 회고하였는데,<sup class="reference" id="cite_ref-blame_15-0"><a href="#cite_note-blame-15">[15]</a></sup> 당시 경기에서 <a href="/wiki/%EC%85%80%ED%8B%B1_FC" title="셀틱 FC">셀틱</a>의 주장인 빌리 맥닐을 전담했던 그는 마지막 골을 허용하고 말았고, 그 이후로부터 1군 대신 유소년 팀 소속으로 뛰게 되었다.<sup class="reference" id="cite_ref-junior_16-0"><a href="#cite_note-junior-16">[16]</a></sup> 동생 마틴 퍼거슨에 의하면 퍼거슨은 매우 분노하며 준우승 메달을 던졌다고 밝혔다.<sup class="reference" id="cite_ref-losersmedal_17-0"><a href="#cite_note-losersmedal-17">[17]</a></sup> 또한 <a href="/wiki/%EA%B0%80%ED%86%A8%EB%A6%AD" title="가톨릭">가톨릭</a>을 믿던 아내 캐시와 결혼하자 <a href="/wiki/%EA%B0%9C%EC%8B%A0%EA%B5%90" title="개신교">개신교</a> 팀이었던 레인저스는 퍼거슨을 차별하기 시작했다.<sup class="reference" id="cite_ref-18"><a href="#cite_note-18">[18]</a></sup> 이후 퍼거슨은 자신의 자서전에서,<sup class="reference" id="cite_ref-19"><a href="#cite_note-19">[19]</a></sup> 자신이 레인저스에 입단할 때 구단에서 아내의 종교에 대해 매우 껄끄러워했기에, 부득이 구단을 떠날 수 밖에 없었고, 때문에 컵 결승전에서 실수를 저지르는 악영향을 끼쳤다고 밝혔다.</p>
<p>그 해 10월, <a class="mw-redirect" href="/wiki/%EB%85%B8%ED%8C%85%EC%97%84_%ED%8F%AC%EB%A0%88%EC%8A%A4%ED%8A%B8_FC" title="노팅엄 포레스트 FC">노팅엄 포레스트</a>는 퍼거슨과 계약하길 원했으나,<sup class="reference" id="cite_ref-signforest_20-0"><a href="#cite_note-signforest-20">[20]</a></sup> 아내가 잉글랜드으로 가길 원하지 않았기 때문에 <a href="/wiki/%ED%8F%B4%EC%BB%A4%ED%81%AC_FC" title="폴커크 FC">폴커크</a>로 이적하였다. 입단 후 선수 겸 코치로서 활약을 하였으나 존 프렌티스 감독 부임 후 퍼거슨을 코치직에서 해임하였고, 이에 퍼거슨은 <a href="/wiki/%EC%97%90%EC%96%B4_%EC%9C%A0%EB%82%98%EC%9D%B4%ED%8B%B0%EB%93%9C_FC" title="에어 유나이티드 FC">에어 유나이티드</a>로 이적을 요구. <a href="/wiki/1974%EB%85%84" title="1974년">1974년</a> 에어 유나이티드에서 은퇴하였다.</p>
<h2><span class="mw-headline" id=".EC.B4.88.EA.B8.B0_.EA.B0.90.EB.8F.85_.EA.B2.BD.EB.A0.A5">초기 감독 경력</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%95%8C%EB%A0%89%EC%8A%A4_%ED%8D%BC%EA%B1%B0%EC%8A%A8&amp;action=edit&amp;section=3" title="부분 편집: 초기 감독 경력">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<h3><span class="mw-headline" id=".EC.9D.B4.EC.8A.A4.ED.8A.B8_.EC.8A.A4.ED.84.B8.EB.A7.81.EC.85.94">이스트 스털링셔</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%95%8C%EB%A0%89%EC%8A%A4_%ED%8D%BC%EA%B1%B0%EC%8A%A8&amp;action=edit&amp;section=4" title="부분 편집: 이스트 스털링셔">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>1974년 6월, 퍼거슨은 감독으로선 젊은 나이인 32살의 나이로 <a class="new" href="/w/index.php?title=%EC%9D%B4%EC%8A%A4%ED%8A%B8_%EC%8A%A4%ED%84%B8%EB%A7%81%EC%85%94_FC&amp;action=edit&amp;redlink=1" title="이스트 스털링셔 FC (없는 문서)">이스트 스털링셔</a>에 취임하였다. 주급 40파운드의 임시 계약직이었다.<sup class="reference" id="cite_ref-21"><a href="#cite_note-21">[21]</a></sup> 이 때부터 엄격한 규율을 강조한 것으로 알려졌는데, 당시 클럽의 공격수였던 바비 맥컬리는 "누가 와도 두렵지 않았는데, 퍼거슨은 처음부터 무서웠다"고 털어놓기도 했다.<sup class="reference" id="cite_ref-22"><a href="#cite_note-22">[22]</a></sup></p>
<p>그 해 10월, <a href="/wiki/%EC%84%B8%EC%9D%B8%ED%8A%B8_%EB%AF%B8%EB%A0%8C_FC" title="세인트 미렌 FC">세인트 미렌</a>으로부터 감독 제의를 받았다. 반면 이스트 스털링셔는 리그 하위권이었으나 퍼거슨은 <a class="new" href="/w/index.php?title=%EC%A1%B0%ED%81%AC_%EC%8A%A4%ED%85%8C%EC%9D%B8&amp;action=edit&amp;redlink=1" title="조크 스테인 (없는 문서)">조크 스테인</a> 감독의 조언을 받고, 더 큰 클럽인 세인트 미렌으로 옮기기로 결심하였다.</p>
<h3><span class="mw-headline" id=".EC.84.B8.EC.9D.B8.ED.8A.B8_.EB.AF.B8.EB.A0.8C">세인트 미렌</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%95%8C%EB%A0%89%EC%8A%A4_%ED%8D%BC%EA%B1%B0%EC%8A%A8&amp;action=edit&amp;section=5" title="부분 편집: 세인트 미렌">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>세인트 미렌에 취임한 퍼거슨은 1978년까지 팀을 이끌면서 <a href="/wiki/%EC%8A%A4%EC%BD%94%ED%8B%80%EB%9E%9C%EB%93%9C_%ED%92%8B%EB%B3%BC_%EB%A6%AC%EA%B7%B8_2%EB%B6%80" title="스코틀랜드 풋볼 리그 2부">스코틀랜드 2부</a> 하위 팀이었던 팀을 완전히 탈바꿈시킨다. 관중 천 명 넘던 팀에서 1977년에 <a href="/wiki/%EC%8A%A4%EC%BD%94%ED%8B%80%EB%9E%9C%EB%93%9C_%ED%92%8B%EB%B3%BC_%EB%A6%AC%EA%B7%B8_1%EB%B6%80" title="스코틀랜드 풋볼 리그 1부">1부 리그</a> 우승, 빌리 스타크, 토니 피츠패트릭 등 유망주를 발굴하여 공격적인 축구를 선보였다. 당시 팀 평균 연령은 19세였으며 주장이었던 피츠패트릭은 20살이었다.</p>
<p>1977년, 퍼거슨은 애버딘으로부터 감독 지원을 거절당한 바 있다. 당시 감독에 빌리 맥닐 감독이 선임되었으나, 1년 뒤 <a href="/wiki/%EC%85%80%ED%8B%B1_FC" title="셀틱 FC">셀틱</a>으로 떠나 공석이 되자, 퍼거슨이 취임하게 되었다.<sup class="reference" id="cite_ref-23"><a href="#cite_note-23">[23]</a></sup></p>
