<p><b>알렌테주 지방</b>(<span style="font-size: smaller;"><a href="/wiki/%ED%8F%AC%EB%A5%B4%ED%88%AC%EA%B0%88%EC%96%B4" title="포르투갈어">포르투갈어</a>:</span> <span lang="pt" xml:lang="pt">Alentejo</span>, <a class="mw-redirect" href="/wiki/IPA" title="IPA">IPA</a>: ɐlẽˈtɛʒu)은 포르투갈의 중남부에 위치한 지방이다. 이름의 어원인 "Além-Tejo"는 "<a href="/wiki/%ED%83%80%EA%B5%AC%EC%8A%A4_%EA%B0%95" title="타구스 강">타구스 강</a> 넘어" 혹은 "타구스 강 건너"의 의미를 지니고 있다. 이 지방은 포르투갈의 다른 지방과 타구스 강을 경계로, 남쪽의 <a href="/wiki/%EC%95%8C%EA%B0%80%EB%A5%B4%EB%B8%8C_%EC%A7%80%EB%B0%A9" title="알가르브 지방">알가르브 지방</a>과의 경계까지 뻗어있다. 알렌테주는 <a href="/wiki/%EC%82%AC%EB%A5%B4%EB%8D%B0%EB%83%90" title="사르데냐">사르데냐</a>나 <a href="/wiki/%EC%BD%94%EB%A5%B4%EC%8B%9C%EC%B9%B4" title="코르시카">코르시카</a> 등지에서도 발견되는 <a class="new" href="/w/index.php?title=%EB%8B%A4%EC%84%B1%EC%9D%8C%EC%95%85&amp;action=edit&amp;redlink=1" title="다성음악 (없는 문서)">다성음악</a> 형식의 민속합창단으로 알려져있다.</p>
<p>주요 도시로는 <a href="/wiki/%EB%B2%A0%EC%9E%90_(%ED%8F%AC%EB%A5%B4%ED%88%AC%EA%B0%88)" title="베자 (포르투갈)">베자</a>, <a class="new" href="/w/index.php?title=%EC%84%B8%EB%A5%B4%ED%8C%8C&amp;action=edit&amp;redlink=1" title="세르파 (없는 문서)">세르파</a>, <a class="new" href="/w/index.php?title=%EC%8B%9C%EB%84%A4%EC%8A%A4&amp;action=edit&amp;redlink=1" title="시네스 (없는 문서)">시네스</a>, <a href="/wiki/%EC%97%90%EB%B3%B4%EB%9D%BC" title="에보라">에보라</a>, <a class="new" href="/w/index.php?title=%EC%97%98%EB%B0%94%EC%8A%A4&amp;action=edit&amp;redlink=1" title="엘바스 (없는 문서)">엘바스</a>, <a href="/wiki/%ED%8F%AC%EB%A5%B4%ED%83%88%EB%A0%88%EA%B7%B8%EB%A5%B4" title="포르탈레그르">포르탈레그르</a>가 있다. 전통적으로 이어져온 지방이며, 오늘날엔 포르투갈의 7개 <a href="/wiki/%ED%86%B5%EA%B3%84%EC%A7%80%EC%97%AD%EB%8B%A8%EC%9C%84%EB%AA%85%EB%AA%85%EB%B2%95" title="통계지역단위명명법">통계지역단위명명법</a> 2단계 지역으로 분류된다. 과거 <a class="new" href="/w/index.php?title=%EB%A6%AC%EC%8A%A4%EB%B3%B4%EC%95%84%EC%9D%B4%EB%B0%9C%EB%A5%B4%EB%91%90%ED%85%8C%EC%A3%BC_%EC%A7%80%EB%B0%A9&amp;action=edit&amp;redlink=1" title="리스보아이발르두테주 지방 (없는 문서)">리스보아이발르두테주 지방</a>으로 구분되던 <a class="new" href="/w/index.php?title=%EB%A0%88%EC%A7%80%EB%A6%AC%EC%95%84%EB%91%90%ED%85%8C%EC%A3%BC&amp;action=edit&amp;redlink=1" title="레지리아두테주 (없는 문서)">레지리아두테주</a>(Lezíria do Tejo)는 오늘날엔 지역통계분류단위 3단계에 해당하는 준지방으로 알렌테주에 속한다.</p>
