<p><b>알로르스타르</b>(<span style="font-size: smaller;"><a href="/wiki/%EB%A7%90%EB%A0%88%EC%9D%B4%EC%96%B4" title="말레이어">말레이어</a>:</span> <span lang="ms" xml:lang="ms">Alor Setar</span>)는 <a href="/wiki/%EB%A7%90%EB%A0%88%EC%9D%B4%EC%8B%9C%EC%95%84" title="말레이시아">말레이시아</a> <a href="/wiki/%ED%81%AC%EB%8B%A4_%EC%A3%BC" title="크다 주">크다 주</a>의 주도로, 면적은 666㎢, 인구는 217,368명(2005년 기준)이다.</p>
<p><a href="/wiki/%ED%83%80%EC%9D%B4" title="타이">태국</a> 국경에서 남쪽으로 약 45km, <a href="/wiki/%EB%B2%84%ED%84%B0%EC%9B%8C%EC%8A%A4" title="버터워스">버터워스</a>에서 북쪽으로 약 93km 정도 떨어진 곳에 위치하며 말레이시아 초대 총리인 <a href="/wiki/%ED%88%B0%EC%BF%A0_%EC%95%95%EB%91%98_%EB%9D%BC%EB%A7%8C" title="툰쿠 압둘 라만">툰쿠 압둘 라만</a>과 <a class="mw-redirect" href="/wiki/%EB%A7%88%ED%95%98%ED%8B%B0%EB%A5%B4_%EB%B9%88_%EB%AA%A8%ED%95%98%EB%A7%88%EB%93%9C" title="마하티르 빈 모하마드">마하티르 빈 모하마드</a> 전 총리가 태어난 곳이기도 하다. 2003년 12월 21일 특별시로 승격되었다.</p>
<h2><span class="mw-headline" id=".EA.B5.90.ED.86.B5">교통</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%95%8C%EB%A1%9C%EB%A5%B4%EC%8A%A4%ED%83%80%EB%A5%B4&amp;action=edit&amp;section=1" title="부분 편집: 교통">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>시에서 북쪽으로 15km 정도 떨어진 곳에 <a class="new" href="/w/index.php?title=%EC%88%A0%ED%83%84_%EC%95%95%EB%91%98_%ED%95%A0%EB%A6%BC_%EA%B3%B5%ED%95%AD&amp;action=edit&amp;redlink=1" title="술탄 압둘 할림 공항 (없는 문서)">술탄 압둘 할림 공항</a>이 있으며 <a href="/wiki/%EC%BF%A0%EC%95%8C%EB%9D%BC%EB%A3%B8%ED%91%B8%EB%A5%B4" title="쿠알라룸푸르">쿠알라룸푸르</a>를 연결하는 <a href="/wiki/%EB%A7%90%EB%A0%88%EC%9D%B4%EC%8B%9C%EC%95%84_%ED%95%AD%EA%B3%B5" title="말레이시아 항공">말레이시아 항공</a>과 <a class="new" href="/w/index.php?title=%ED%8C%8C%EC%9D%B4%EC%96%B4%ED%94%8C%EB%9D%BC%EC%9D%B4_%ED%95%AD%EA%B3%B5&amp;action=edit&amp;redlink=1" title="파이어플라이 항공 (없는 문서)">파이어플라이 항공</a>, <a href="/wiki/%EC%97%90%EC%96%B4%EC%95%84%EC%8B%9C%EC%95%84" title="에어아시아">에어아시아</a> 노선이 운행된다. 도심에는 기차역이 들어서 있으며 <a href="/wiki/%EC%9D%B4%ED%8F%AC" title="이포">이포</a>와 <a href="/wiki/%EC%BF%A0%EC%95%8C%EB%9D%BC%EB%A3%B8%ED%91%B8%EB%A5%B4" title="쿠알라룸푸르">쿠알라룸푸르</a>를 연결하는 철도 노선이 지나간다. 그 밖에 <a href="/wiki/%EC%BF%A0%EC%95%8C%EB%9D%BC%EB%A3%B8%ED%91%B8%EB%A5%B4" title="쿠알라룸푸르">쿠알라룸푸르</a>와 <a href="/wiki/%EC%8B%B1%EA%B0%80%ED%8F%AC%EB%A5%B4" title="싱가포르">싱가포르</a>, <a href="/wiki/%ED%83%80%EC%9D%B4" title="타이">태국</a>을 남북으로 연결하는 고속도로가 지나간다.</p>
<h2><span class="mw-headline" id=".EC.99.B8.EB.B6.80_.EC.97.B0.EA.B2.B0">외부 연결</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%95%8C%EB%A1%9C%EB%A5%B4%EC%8A%A4%ED%83%80%EB%A5%B4&amp;action=edit&amp;section=2" title="부분 편집: 외부 연결">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
