<p><b>알류샨 열도</b>(Aleutian Islands, <a href="/wiki/%EC%B6%95%EC%B9%98%EC%96%B4" title="축치어">축치어</a>로 섬을 뜻하는 "aliat"에서 유래됨)는 북태평양에 위치한 섬들로, 알래스카의 서남부, <a href="/wiki/%EC%95%8C%EB%9E%98%EC%8A%A4%EC%B9%B4_%EB%B0%98%EB%8F%84" title="알래스카 반도">알래스카 반도</a>의 끝에서 <a href="/wiki/%EB%9F%AC%EC%8B%9C%EC%95%84" title="러시아">러시아</a>의 <a href="/wiki/%EC%BA%84%EC%B0%A8%EC%B9%B4_%EB%B0%98%EB%8F%84" title="캄차카 반도">캄차카 반도</a>에 걸쳐 길이 약 1930km의 화산 <a href="/wiki/%EC%97%B4%EB%8F%84" title="열도">열도</a>이다. 대부분이 <a href="/wiki/%EB%AF%B8%EA%B5%AD" title="미국">미국</a>의 <a href="/wiki/%EC%95%8C%EB%9E%98%EC%8A%A4%EC%B9%B4_%EC%A3%BC" title="알래스카 주">알래스카 주</a>에 속하고 있지만, 최서단의 <a href="/wiki/%EC%BD%94%EB%A7%8C%EB%8F%84%EB%A5%B4%EC%8A%A4%ED%82%A4%EC%98%88_%EC%A0%9C%EB%8F%84" title="코만도르스키예 제도">코만도르스키예 제도</a>는 <a href="/wiki/%EB%9F%AC%EC%8B%9C%EC%95%84" title="러시아">러시아</a>에 속해있고 <a href="/wiki/%ED%99%98%ED%83%9C%ED%8F%89%EC%96%91_%EC%A1%B0%EC%82%B0%EB%8C%80" title="환태평양 조산대">환태평양 조산대</a>의 일부를 구성한다. 알래스카 산지의 연장으로 2,000m가 넘는 <a href="/wiki/%ED%99%94%EC%82%B0" title="화산">화산</a>이 많으며, 기후는 한랭습윤하고 <a href="/wiki/%EC%95%88%EA%B0%9C" title="안개">안개</a>가 많다. 이 부근은 세계 3대 어장의 하나로서 어류의 보고(寶庫)이다.<sup class="reference" id="cite_ref-.EA.B8.80.EB.A1.9C.EB.B2.8C_.EB.AF.B8.EA.B5.AD_1-0"><a href="#cite_note-.EA.B8.80.EB.A1.9C.EB.B2.8C_.EB.AF.B8.EA.B5.AD-1">[1]</a></sup></p>
