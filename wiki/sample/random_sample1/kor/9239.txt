<p><b>알리 바카르</b>(Ali Bakar, <a href="/wiki/1947%EB%85%84" title="1947년">1947년</a> <a href="/wiki/11%EC%9B%94_18%EC%9D%BC" title="11월 18일">11월 18일</a> ~ <a href="/wiki/2003%EB%85%84" title="2003년">2003년</a> <a href="/wiki/8%EC%9B%94_9%EC%9D%BC" title="8월 9일">8월 9일</a>)는 <a href="/wiki/%EB%A7%90%EB%A0%88%EC%9D%B4%EC%8B%9C%EC%95%84" title="말레이시아">말레이시아</a>의 전 <a href="/wiki/%EC%B6%95%EA%B5%AC" title="축구">축구</a> <a href="/wiki/%EA%B3%B5%EA%B2%A9%EC%88%98" title="공격수">공격수</a>였으며, 그의 형인 <a class="new" href="/w/index.php?title=%EC%9D%B4%EC%82%AC_%EB%B0%94%EC%B9%B4%EB%A5%B4&amp;action=edit&amp;redlink=1" title="이사 바카르 (없는 문서)">이사 바카르</a> 또한 축구 선수로 활동하였다.</p>
<p>1970년대 <a class="new" href="/w/index.php?title=%ED%8E%98%EB%82%AD_FA&amp;action=edit&amp;redlink=1" title="페낭 FA (없는 문서)">페낭 FA</a>에서 활약했으며, <a href="/wiki/1974%EB%85%84" title="1974년">1974년</a> 팀의 <a class="new" href="/w/index.php?title=%EB%A7%90%EB%A0%88%EC%9D%B4%EC%8B%9C%EC%95%84_%EC%BB%B5&amp;action=edit&amp;redlink=1" title="말레이시아 컵 (없는 문서)">말레이시아 컵</a> 우승 및 <a href="/wiki/1976%EB%85%84" title="1976년">1976년</a> 팀의 <a class="new" href="/w/index.php?title=%EC%95%84%EA%B0%80_%EC%B9%B8_%EA%B3%A8%EB%93%9C%EC%BB%B5&amp;action=edit&amp;redlink=1" title="아가 칸 골드컵 (없는 문서)">아가 칸 골드컵</a> 우승에 공헌하였다.</p>
<p>또한 1970년대 <a href="/wiki/%EB%A7%90%EB%A0%88%EC%9D%B4%EC%8B%9C%EC%95%84_%EC%B6%95%EA%B5%AC_%EA%B5%AD%EA%B0%80%EB%8C%80%ED%91%9C%ED%8C%80" title="말레이시아 축구 국가대표팀">말레이시아 축구 국가대표팀</a>의 주요 선수로 활약했으며, <a href="/wiki/1976%EB%85%84_AFC_%EC%95%84%EC%8B%9C%EC%95%88%EC%BB%B5" title="1976년 AFC 아시안컵">1976년 AFC 아시안컵</a>에 출전하였다. 또한 올림픽 국가대표팀으로도 활동하면서 <a href="/wiki/1972%EB%85%84_%ED%95%98%EA%B3%84_%EC%98%AC%EB%A6%BC%ED%94%BD" title="1972년 하계 올림픽">1972년 하계 올림픽</a> 본선 진출에 기여한 뒤 본선에도 뛰었으며, <a href="/wiki/1974%EB%85%84_%EC%95%84%EC%8B%9C%EC%95%88_%EA%B2%8C%EC%9E%84" title="1974년 아시안 게임">1974년 아시안 게임</a>에도 참가해 2차 라운드에서 <a href="/wiki/%EB%8C%80%ED%95%9C%EB%AF%BC%EA%B5%AD_%EC%B6%95%EA%B5%AC_%EA%B5%AD%EA%B0%80%EB%8C%80%ED%91%9C%ED%8C%80" title="대한민국 축구 국가대표팀">대한민국</a>을 상대로 결승골을 기록해 팀의 3-2 승리를 이끄는 등 팀이 동메달을 따는 데 기여했다.</p>
<p><a href="/wiki/2003%EB%85%84" title="2003년">2003년</a> <a href="/wiki/8%EC%9B%94_9%EC%9D%BC" title="8월 9일">8월 9일</a> <a href="/wiki/%EC%8B%B1%EA%B0%80%ED%8F%AC%EB%A5%B4" title="싱가포르">싱가포르</a>에서 열린 자선경기에 참가해 경기를 하던 도중 갑작스러운 <a class="mw-disambig" href="/wiki/%EC%8B%AC%EC%9E%A5%EB%A7%88%EB%B9%84" title="심장마비">심장마비</a>로 의식을 잃었으며, 병원에 도착하였을 때는 <a href="/wiki/%EB%8D%B0%EB%93%9C_%EC%98%A8_%EC%96%B4%EB%9D%BC%EC%9D%B4%EB%B2%8C" title="데드 온 어라이벌">이미 사망한 상태</a>였다.<sup class="reference" id="cite_ref-1"><a href="#cite_note-1">[1]</a></sup></p>
<p>사망 이후 시신은 고향인 <a href="/wiki/%ED%94%BC%EB%82%AD_%EC%A3%BC" title="피낭 주">피낭 주</a>에 매장되었다.</p>
