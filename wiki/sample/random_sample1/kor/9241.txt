<p><b>알리 아미니</b>(<a href="/wiki/1905%EB%85%84" title="1905년">1905년</a> <a href="/wiki/12%EC%9B%94_12%EC%9D%BC" title="12월 12일">12월 12일</a>~<a href="/wiki/1992%EB%85%84" title="1992년">1992년</a> <a href="/wiki/12%EC%9B%94_12%EC%9D%BC" title="12월 12일">12월 12일</a>)는 <a href="/wiki/%EC%9D%B4%EB%9E%80" title="이란">이란</a>의 정치가이다. <a href="/wiki/1961%EB%85%84" title="1961년">1961년</a>부터 이듬해까지 <a class="mw-redirect" href="/wiki/%EC%9D%B4%EB%9E%80_%EC%A0%9C%EA%B5%AD" title="이란 제국">이란 제국</a>의 <a class="new" href="/w/index.php?title=%EC%9D%B4%EB%9E%80%EC%9D%98_%EC%B4%9D%EB%A6%AC&amp;action=edit&amp;redlink=1" title="이란의 총리 (없는 문서)">총리</a>를 지냈다.</p>
<p><a href="/wiki/1905%EB%85%84" title="1905년">1905년</a> <a href="/wiki/%ED%85%8C%ED%97%A4%EB%9E%80" title="테헤란">테헤란</a>에서 태어났다.<sup class="reference" id="cite_ref-moin92_1-0"><a href="#cite_note-moin92-1">[1]</a></sup> <a href="/wiki/%EC%B9%B4%EC%9E%90%EB%A5%B4_%EC%99%95%EC%A1%B0" title="카자르 왕조">카자르 왕조</a>의 5대 <a href="/wiki/%EC%83%A4" title="샤">샤</a>였던 <a class="new" href="/w/index.php?title=%EB%AA%A8%EC%9E%90%ED%8C%8C%EB%A5%B4_%EC%95%97%EB%94%98_%EC%83%A4_%EC%B9%B4%EC%9E%90%EB%A5%B4&amp;action=edit&amp;redlink=1" title="모자파르 앗딘 샤 카자르 (없는 문서)">모자파르 앗딘 샤 카자르</a>의 외손자이다. 페르시아와 <a href="/wiki/%ED%94%84%EB%9E%91%EC%8A%A4" title="프랑스">프랑스</a>에서 교육을 받았으며 <a class="new" href="/w/index.php?title=%EA%B7%B8%EB%A5%B4%EB%85%B8%EB%B8%94_%EB%8C%80%ED%95%99%EA%B5%90&amp;action=edit&amp;redlink=1" title="그르노블 대학교 (없는 문서)">그르노블 대학교</a> 법대를 졸업한 후 <a class="mw-redirect" href="/wiki/%ED%8C%8C%EB%A6%AC_(%EB%8F%84%EC%8B%9C)" title="파리 (도시)">파리</a>에서 박사 학위를 받았다.</p>
<p>37세 나이에 <a class="new" href="/w/index.php?title=%EC%95%84%ED%9D%90%EB%A7%88%EB%93%9C_%EA%B0%80%EB%B0%A4&amp;action=edit&amp;redlink=1" title="아흐마드 가밤 (없는 문서)">아흐마드 가밤</a> 내각에서 처음 장관직을 지냈으며, 한때 <a href="/wiki/%EB%AA%A8%ED%95%98%EB%A7%88%EB%93%9C_%EB%AA%A8%EC%82%AC%EB%8D%B0%EA%B7%B8" title="모하마드 모사데그">모하마드 모사데그</a>의 <a class="new" href="/w/index.php?title=%EA%B5%AD%EB%AF%BC%EC%A0%84%EC%84%A0_(%EC%9D%B4%EB%9E%80)&amp;action=edit&amp;redlink=1" title="국민전선 (이란) (없는 문서)">국민전선</a>에 가담했다가 1952년 탈당했다.<sup class="reference" id="cite_ref-rubin80_2-0"><a href="#cite_note-rubin80-2">[2]</a></sup> <a href="/wiki/%ED%8C%8C%EC%A6%90%EB%A1%A4%EB%9D%BC_%EC%9E%90%ED%97%A4%EB%94%94" title="파즐롤라 자헤디">파즐롤라 자헤디</a> 내각에서 경제부장관을 지냈으며 1955년까지 유임되었다. 같은 해 그는 주미(駐美) 대사로 임명되어 3년간 재임했다.<sup class="reference" id="cite_ref-rubin80_2-1"><a href="#cite_note-rubin80-2">[2]</a></sup> 1950년대 들어 그는 총리 후보의 물망에 올랐으며 <a href="/wiki/1961%EB%85%84" title="1961년">1961년</a> 총리로 지명되었다. 그러나 이듬해 그의 총리직은 <a href="/wiki/%EB%AA%A8%ED%95%98%EB%A7%88%EB%93%9C_%EB%A0%88%EC%9E%90_%ED%8C%94%EB%9D%BC%EB%B9%84" title="모하마드 레자 팔라비">샤</a>의 측근이자 <a href="/wiki/%EB%B9%84%EB%A5%B4%EC%9E%94%EB%93%9C" title="비르잔드">비르잔드</a> 지방의 대지주인 <a href="/wiki/%EC%95%84%EC%82%AC%EB%8F%8C%EB%9D%BC_%EC%95%8C%EB%9E%8C" title="아사돌라 알람">아사돌라 알람</a>으로 교체되었다.</p>
<p>1970년대 들어, 70대 나이에 그는 이란 정치계에 복귀하도록 요구되었다. <a href="/wiki/%ED%8C%94%EB%9D%BC%EB%B9%84_%EC%99%95%EC%A1%B0" title="팔라비 왕조">팔라비 왕조</a>의 말기에 그는 황제의 조언자로 일하였다. 1979년 <a class="mw-redirect" href="/wiki/%EC%9D%B4%EB%9E%80_%EC%9D%B4%EC%8A%AC%EB%9E%8C_%ED%98%81%EB%AA%85" title="이란 이슬람 혁명">이란 이슬람 혁명</a>이 발생하자 <a href="/wiki/%ED%94%84%EB%9E%91%EC%8A%A4" title="프랑스">프랑스</a>로 망명하였다. 프랑스 파리에서 그의 87세 생일에 사망했다.<sup class="reference" id="cite_ref-moin92_1-1"><a href="#cite_note-moin92-1">[1]</a></sup></p>
