<h1 class="firstHeading" id="firstHeading" lang="ko">Delete (SQL)</h1>
<div class="mw-body-content" id="bodyContent">
<div id="siteSub">위키백과, 우리 모두의 백과사전.</div>
<div id="contentSub"></div>
<div class="mw-jump" id="jump-to-nav">
					이동:					<a href="#mw-head">둘러보기</a>, 					<a href="#p-search">검색</a>
</div>
<div class="mw-content-ltr" dir="ltr" id="mw-content-text" lang="ko"><script>function mfTempOpenSection(id){var block=document.getElementById("mf-section-"+id);block.className+=" open-block";block.previousSibling.className+=" open-block";}</script><p>구조화 질의어(<a href="/wiki/SQL" title="SQL">SQL</a>)에서, <b>DELETE</b> 문은 <a href="/wiki/%ED%85%8C%EC%9D%B4%EB%B8%94_(%EB%8D%B0%EC%9D%B4%ED%84%B0%EB%B2%A0%EC%9D%B4%EC%8A%A4)" title="테이블 (데이터베이스)">테이블</a> 또는 뷰에서 한개 이상의 행을 삭제한다. 하위 집합은 삭제에 대한 조건을 정의할 수 있으며, 정의하지 않으면 모든 행이 삭제된다.<sup class="reference" id="cite_ref-1"><a href="#cite_note-1">[1]</a></sup></p>

<p></p>
<h2><span class="mw-headline" id=".EC.82.AC.EC.9A.A9.EB.B2.95">사용법</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Delete_(SQL)&amp;action=edit&amp;section=1" title="부분 편집: 사용법">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p><code>DELETE</code> 문은 다음 구문을 따른다</p>
<dl>
<dd><b><code>DELETE</code></b> <code>FROM</code> <i>테이블_또는_뷰_이름</i> [<b><code>WHERE</code></b> 조건]</dd>
</dl>
<p><a href="/wiki/WHERE_(SQL)" title="WHERE (SQL)"><code>WHERE</code> 조건</a>에 맞는 모든 행은 테이블에서 삭제된다. <code>WHERE</code> 절을 생략하면, 모든 행을 삭제한다. <sup class="reference" id="cite_ref-2"><a href="#cite_note-2">[2]</a></sup><code>DELETE</code>문은 어떤 행이라도 리턴하지 않으며, <a class="new" href="/w/index.php?title=%EA%B2%B0%EA%B3%BC_%EC%A7%91%ED%95%A9&amp;action=edit&amp;redlink=1" title="결과 집합 (없는 문서)">결과 집합</a>을 발생시키지도 않는다.</p>
<p><code>DELETE</code> 문을 실행하는 것은 다른 테이블을 삭제하게끔 실행하는 <a href="/wiki/%EB%8D%B0%EC%9D%B4%ED%84%B0%EB%B2%A0%EC%9D%B4%EC%8A%A4_%ED%8A%B8%EB%A6%AC%EA%B1%B0" title="데이터베이스 트리거">트리거</a>가 발생할 수 있다. 예를 들면, 두 테이블이 <a href="/wiki/%EC%99%B8%EB%9E%98_%ED%82%A4" title="외래 키">외래 키</a>로 연결되어 있고 행이 참조된 테이블에서 삭제된다면, <a href="/wiki/%EC%B0%B8%EC%A1%B0_%EB%AC%B4%EA%B2%B0%EC%84%B1" title="참조 무결성">참조 무결성</a>이 유지되도록 참조하고 있는 테이블에도 공통적으로 삭제된다.</p>
<div class="rellink boilerplate seealso"><img alt="" data-file-height="128" data-file-width="128" height="16" src="//upload.wikimedia.org/wikipedia/commons/thumb/e/ec/Crystal_Clear_app_xmag.svg/16px-Crystal_Clear_app_xmag.svg.png" srcset="//upload.wikimedia.org/wikipedia/commons/thumb/e/ec/Crystal_Clear_app_xmag.svg/24px-Crystal_Clear_app_xmag.svg.png 1.5x, //upload.wikimedia.org/wikipedia/commons/thumb/e/ec/Crystal_Clear_app_xmag.svg/32px-Crystal_Clear_app_xmag.svg.png 2x" width="16"/> <a class="new" href="/w/index.php?title=%EC%A0%84%ED%8C%8C_%EC%83%81%EC%88%98&amp;action=edit&amp;redlink=1" title="전파 상수 (없는 문서)">전파 상수</a> 문서를 참고하십시오.</div>
<h2><span class="mw-headline" id=".EC.98.88.EC.A0.9C">예제</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Delete_(SQL)&amp;action=edit&amp;section=2" title="부분 편집: 예제">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<ul>
<li><i>pies</i> 테이블에서 <i>flavour</i> 열과 <i>Lemon Meringue</i> 열에서 행이 같으면 삭제한다:</li>
</ul>
<div class="mw-highlight mw-content-ltr" dir="ltr">
<pre>
<span class="k">DELETE</span> <span class="k">FROM</span> <span class="n">pies</span> <span class="k">WHERE</span> <span class="n">flavour</span><span class="o">=</span><span class="s1">'Lemon Meringue'</span><span class="p">;</span>
</pre></div>
<ul>
<li><i>trees</i> 테이블에서, <i>height</i> 값이 80보다 작으면 행을 삭제한다:</li>
</ul>
<div class="mw-highlight mw-content-ltr" dir="ltr">
<pre>
<span class="k">DELETE</span> <span class="k">FROM</span> <span class="n">trees</span> <span class="k">WHERE</span> <span class="n">height</span> <span class="o">&lt;</span> <span class="mi">80</span><span class="p">;</span>
</pre></div>
<ul>
<li><i>mytable</i>에서 모든 행을 삭제한다:</li>
</ul>
<div class="mw-highlight mw-content-ltr" dir="ltr">
<pre>
<span class="k">DELETE</span> <span class="k">FROM</span> <span class="n">mytable</span><span class="p">;</span>
</pre></div>
<ul>
<li><i>mytable</i>에서 where 조건에 서브쿼리를 사용해 행을 삭제한다:</li>
</ul>
<div class="mw-highlight mw-content-ltr" dir="ltr">
<pre>
<span class="k">DELETE</span> <span class="k">FROM</span> <span class="n">mytable</span> <span class="k">WHERE</span> <span class="n">id</span> <span class="k">IN</span> <span class="p">(</span><span class="k">SELECT</span> <span class="n">id</span> <span class="k">FROM</span> <span class="n">mytable2</span><span class="p">)</span>
</pre></div>
<ul>
<li><i>mytable</i>에서 값 목록을 사용하여 행을 삭제한다:</li>
</ul>
<div class="mw-highlight mw-content-ltr" dir="ltr">
<pre>
<span class="k">DELETE</span> <span class="k">FROM</span> <span class="n">mytable</span> <span class="k">WHERE</span> <span class="n">id</span> <span class="k">IN</span> <span class="p">(</span><span class="n">value1</span><span class="p">,</span> <span class="n">value2</span><span class="p">,</span> <span class="n">value3</span><span class="p">,</span> <span class="n">value4</span><span class="p">,</span> <span class="n">value5</span><span class="p">)</span>
</pre></div>
