<p><b>알브레히트 폰 프로이센 공작</b>(<span style="font-size: smaller;"><a href="/wiki/%EB%8F%85%EC%9D%BC%EC%96%B4" title="독일어">독일어</a>:</span> <span lang="de" xml:lang="de">Albrecht Herzog von Preußen</span>, <a href="/wiki/1490%EB%85%84" title="1490년">1490년</a> <a href="/wiki/5%EC%9B%94_17%EC%9D%BC" title="5월 17일">5월 17일</a> - <a href="/wiki/1568%EB%85%84" title="1568년">1568년</a> <a href="/wiki/3%EC%9B%94_20%EC%9D%BC" title="3월 20일">3월 20일</a>)은 <a href="/wiki/%EB%8F%85%EC%9D%BC" title="독일">독일</a>의 귀족으로, <a class="mw-redirect" href="/wiki/%ED%98%B8%EC%97%94%EC%B4%90%EB%A0%88%EB%A5%B8_%EC%99%95%EA%B0%80" title="호엔촐레른 왕가">호엔촐레른 왕가</a> 출신 초대 <a href="/wiki/%ED%94%84%EB%A1%9C%EC%9D%B4%EC%84%BC_%EC%99%95%EA%B5%AD" title="프로이센 왕국">프로이센</a> 공작이며, 37대 튜턴 기사단의 단장이었다. <a class="mw-disambig" href="/wiki/%EB%B8%8C%EB%9E%80%EB%8D%B4%EB%B6%80%EB%A5%B4%ED%81%AC" title="브란덴부르크">브란덴부르크</a><a href="/wiki/%EC%84%A0%EC%A0%9C%ED%9B%84" title="선제후">선제후</a> <a class="new" href="/w/index.php?title=%EC%95%8C%EB%B8%8C%EB%A0%88%ED%9E%88%ED%8A%B8_3%EC%84%B8_%EC%95%84%ED%82%AC%EB%A0%88%EC%8A%A4&amp;action=edit&amp;redlink=1" title="알브레히트 3세 아킬레스 (없는 문서)">알브레히트 3세 아킬레스</a>의 손자이자 <a href="/wiki/%EC%95%88%EC%8A%A4%EB%B0%94%ED%9D%90" title="안스바흐">안스바흐</a> 백작 <a href="/wiki/%ED%94%84%EB%A6%AC%EB%93%9C%EB%A6%AC%ED%9E%88_2%EC%84%B8_(%EB%B8%8C%EB%9E%80%EB%8D%B4%EB%B6%80%EB%A5%B4%ED%81%AC-%EC%95%88%EC%8A%A4%EB%B0%94%ED%9D%90-%EC%BD%9C%EB%A6%84%EB%B0%94%ED%9D%90)" title="프리드리히 2세 (브란덴부르크-안스바흐-콜름바흐)">프리드리히 2세</a>의 아들이었다. 그는 <a class="mw-redirect" href="/wiki/%EB%A3%A8%ED%84%B0" title="루터">루터</a>의 종교개혁을 공식으로 지지한 몇안되는 군주이며, <a href="/wiki/%ED%8F%B4%EB%9E%80%EB%93%9C" title="폴란드">폴란드</a>에서 최초로 <a class="mw-redirect" href="/wiki/%EC%BD%94%ED%8E%98%EB%A5%B4%EB%8B%88%EC%BF%A0%EC%8A%A4" title="코페르니쿠스">코페르니쿠스</a>의 지동설을 공식 지지한 소수의 군주였다. <a href="/wiki/%EB%A7%88%EB%A5%B4%ED%8B%B4_%EB%A3%A8%ED%84%B0" title="마르틴 루터">마르틴 루터</a>의 <a href="/wiki/%EC%A2%85%EA%B5%90_%EA%B0%9C%ED%98%81" title="종교 개혁">종교 개혁</a>을 지지한 알베르트는 <a href="/wiki/1527%EB%85%84" title="1527년">1527년</a> <a class="mw-redirect" href="/wiki/%ED%8A%9C%ED%8A%BC_%EA%B8%B0%EC%82%AC%EB%8B%A8" title="튜튼 기사단">튜튼 기사단</a> 그랜드 마스터직을 <a class="new" href="/w/index.php?title=%EB%B0%9C%ED%84%B0_%ED%8F%B0_%EC%B9%B4%EB%85%BC%EB%B6%80%EB%A5%B4%ED%81%AC&amp;action=edit&amp;redlink=1" title="발터 폰 카논부르크 (없는 문서)">발터 폰 카논부르크</a>에게 넘겨주고, 자신도 <a class="mw-redirect" href="/wiki/%EB%A1%9C%EB%A7%88_%EA%B0%80%ED%86%A8%EB%A6%AD" title="로마 가톨릭">로마 가톨릭</a>에서 <a href="/wiki/%EB%A3%A8%ED%84%B0%EA%B5%90" title="루터교">루터교</a>로 개종하였다.</p>
<p><a href="/wiki/%EB%8F%85%EC%9D%BC" title="독일">독일</a>의 귀족이자 <a href="/wiki/%ED%94%84%EB%A1%9C%EC%9D%B4%EC%84%BC_%EA%B3%B5%EA%B5%AD" title="프로이센 공국">프로이센 공국</a>의 초대 공작이며, <a class="mw-redirect" href="/wiki/%ED%8A%9C%ED%8A%BC%EA%B8%B0%EC%82%AC%EB%8B%A8" title="튜튼기사단">튜튼기사단</a>의 제37대 그랜드마스터를 역임하였다. <a href="/wiki/%ED%94%84%EB%A1%9C%EC%9D%B4%EC%84%BC" title="프로이센">프로이센</a> 정부의 창설자이자 초대 통치자이며, <a href="/wiki/%EB%A3%A8%ED%84%B0%EA%B5%90" title="루터교">루터교</a>를 국교로 삼은 최초의 국가 군주기도 하다. 기사이자 사제 신분이었던 그는 <a href="/wiki/1525%EB%85%84" title="1525년">1525년</a> <a class="mw-redirect" href="/wiki/%ED%94%84%EB%A1%9C%ED%85%8C%EC%8A%A4%ED%83%84%ED%8A%B8" title="프로테스탄트">프로테스탄트</a>로 개종하고, 그해 <a href="/wiki/2%EC%9B%94_10%EC%9D%BC" title="2월 10일">2월 10일</a> 폴란드 국왕이었던 외삼촌 <a href="/wiki/%EC%A7%80%EA%B7%B8%EB%AC%B8%ED%8A%B8_1%EC%84%B8" title="지그문트 1세">지그문트 1세</a>에 의해 <a href="/wiki/%ED%94%84%EB%A1%9C%EC%9D%B4%EC%84%BC" title="프로이센">프로이센</a> <a class="mw-disambig" href="/wiki/%EA%B3%B5%EC%9E%91" title="공작">공작</a>에 임명되었다. 그는 자신의 영지인 <a href="/wiki/%ED%94%84%EB%A1%9C%EC%9D%B4%EC%84%BC_%EA%B3%B5%EA%B5%AD" title="프로이센 공국">프로이센 공국</a>에 신앙의 자유 허용 및 세속국가화를 선언했으며, 독자적인 외교권과 군사 양성 등을 관철시켰다. 어머니를 통해 폴란드계 혼혈 독일인인 알브레히트는 독일의 제후들 중 <a href="/wiki/%EB%A7%88%EB%A5%B4%ED%8B%B4_%EB%A3%A8%ED%84%B0" title="마르틴 루터">마르틴 루터</a>의 <a href="/wiki/%EC%A2%85%EA%B5%90_%EA%B0%9C%ED%98%81" title="종교 개혁">종교 개혁</a>을 적극 지지하였고, <a href="/wiki/%ED%8F%B4%EB%9E%80%EB%93%9C" title="폴란드">폴란드</a>인 <a class="mw-redirect" href="/wiki/%EC%BD%94%ED%8E%98%EB%A5%B4%EB%8B%88%EC%BF%A0%EC%8A%A4" title="코페르니쿠스">코페르니쿠스</a>의 <a class="mw-redirect" href="/wiki/%EC%A7%80%EB%8F%99%EC%84%A4" title="지동설">지동설</a>에도 적극 지지하였다. 코페르니쿠스는 알베르트의 가신, 측근들을 직접 진료하기도 했다. 그는 새로 건국된 <a href="/wiki/%ED%94%84%EB%A1%9C%EC%9D%B4%EC%84%BC_%EA%B3%B5%EA%B5%AD" title="프로이센 공국">프로이센 공국</a>을 종교국이 아닌 세속 국가로 운영하였다. <a class="new" href="/w/index.php?title=%EC%BE%A8%EB%8B%88%ED%9E%88%EC%8A%A4%EB%B2%A0%EB%A5%B4%ED%81%AC_%EB%8C%80%ED%95%99&amp;action=edit&amp;redlink=1" title="쾨니히스베르크 대학 (없는 문서)">쾨니히스베르크 대학</a>의 설립자이기도 하다.</p>

title="알브레히트 3세 아킬레스 (없는 문서)">알브레히트 3세 아킬레스</a> 역시 형 프리드리히가 1471년 아들이 없이 죽자 선제후직을 차지한다.<sup class="reference" id="cite_ref-3"><a href="#cite_note-3">[3]</a></sup> 할아버지 알브레히트 아킬레스는 안스바흐 변경백작직을 아버지 <a href="/wiki/%ED%94%84%EB%A6%AC%EB%93%9C%EB%A6%AC%ED%9E%88_2%EC%84%B8_(%EB%B8%8C%EB%9E%80%EB%8D%B4%EB%B6%80%EB%A5%B4%ED%81%AC-%EC%95%88%EC%8A%A4%EB%B0%94%ED%9D%90-%EC%BD%9C%EB%A6%84%EB%B0%94%ED%9D%90)" title="프리드리히 2세 (브란덴부르크-안스바흐-콜름바흐)">프리드리히 2세</a>에게, 브란덴부르크 선제후직은 큰아버지 <a class="new" href="/w/index.php?title=%EC%9A%94%ED%95%9C_%ED%82%A4%EC%BC%80%EB%A1%9C&amp;action=edit&amp;redlink=1" title="요한 키케로 (없는 문서)">요한 키케로</a>에게 넘겨주었다. 하지만 그는 아버지 <a href="/wiki/%ED%94%84%EB%A6%AC%EB%93%9C%EB%A6%AC%ED%9E%88_2%EC%84%B8_(%EB%B8%8C%EB%9E%80%EB%8D%B4%EB%B6%80%EB%A5%B4%ED%81%AC-%EC%95%88%EC%8A%A4%EB%B0%94%ED%9D%90-%EC%BD%9C%EB%A6%84%EB%B0%94%ED%9D%90)" title="프리드리히 2세 (브란덴부르크-안스바흐-콜름바흐)">프리드리히 2세</a>가 다른 형제들의 쿠데타로 축출되는 바람에 아버지로부터 영지를 상속받지 못했고, <a class="mw-redirect" href="/wiki/%ED%8A%9C%ED%8A%BC_%EA%B8%B0%EC%82%AC%EB%8B%A8" title="튜튼 기사단">튜튼 기사단</a>의 단장이 되었다.</p>
<p>여러번 결혼한 그의 후처 중 안나 마리아는 그의 조카딸의 딸이 된다. <a class="new" href="/w/index.php?title=%EC%95%88%EB%82%98_%EB%A7%88%EB%A6%AC%EC%95%84&amp;action=edit&amp;redlink=1" title="안나 마리아 (없는 문서)">안나 마리아</a>는 알브레히트의 5촌 조카 엘리자베스의 딸이자 6년 연상인 친사촌형 <a class="new" href="/w/index.php?title=%EC%9A%94%EC%95%84%ED%9E%98_1%EC%84%B8_%EB%84%A4%EC%8A%A4%ED%86%A0%EB%A5%B4&amp;action=edit&amp;redlink=1" title="요아힘 1세 네스토르 (없는 문서)">요아힘 1세 네스토르</a>의 손녀딸이었다. 안나 마리아의 부모 역시 근친혼이었는데, 안나 마리아의 친정아버지 <a class="new" href="/w/index.php?title=%EB%B8%8C%EB%A3%AC%EC%8A%A4%EC%9C%84%ED%81%AC-%EB%A5%B4%EB%84%A4%EB%B6%80%EB%A5%B4%ED%81%AC&amp;action=edit&amp;redlink=1" title="브룬스위크-르네부르크 (없는 문서)">브룬스위크-르네부르크</a>의 에릭 1세는 알브레히트의 증조부 <a class="new" href="/w/index.php?title=%ED%94%84%EB%A6%AC%EB%93%9C%EB%A6%AC%ED%9E%88_1%EC%84%B8_(1371%EB%85%84)&amp;action=edit&amp;redlink=1" title="프리드리히 1세 (1371년) (없는 문서)">선제후 프리드리히 1세</a>의 딸이자 대고모인 세실의 친손자였다. 장인 에릭은 알브레히트에게는 6촌 형이 된다.<sup class="reference" id="cite_ref-4"><a href="#cite_note-4">[4]</a></sup></p>
<p>에릭은 브란덴부르크의 엘리자베스와 결혼한다. 에릭은 자기 친할머니 세실의 남동생인 외종조부 <a class="new" href="/w/index.php?title=%EC%95%8C%EB%B8%8C%EB%A0%88%ED%9E%88%ED%8A%B8_3%EC%84%B8_%EC%95%84%ED%82%AC%EB%A0%88%EC%8A%A4&amp;action=edit&amp;redlink=1" title="알브레히트 3세 아킬레스 (없는 문서)">알브레히트 3세 아킬레스</a>의 아들 <a class="new" href="/w/index.php?title=%EC%9A%94%ED%95%9C_%ED%82%A4%EC%BC%80%EB%A1%9C&amp;action=edit&amp;redlink=1" title="요한 키케로 (없는 문서)">요한 키케로</a>의 손녀이자, <a class="new" href="/w/index.php?title=%EC%9A%94%EC%95%84%ED%9E%98_1%EC%84%B8_%EB%84%A4%EC%8A%A4%ED%86%A0%EB%A5%B4&amp;action=edit&amp;redlink=1" title="요아힘 1세 네스토르 (없는 문서)">요아힘 1세 네스토르</a>의 딸과 결혼한다. 세실의 손자인 에릭과 세실의 남동생 알브레히트 3세 아킬레스의 증손녀인 엘리자베스는 7촌 숙질간이 된다.<sup class="reference" id="cite_ref-5"><a href="#cite_note-5">[5]</a></sup></p>
<p>장모가 되는 엘리자베스는 알브레히트의 4촌 형 <a class="new" href="/w/index.php?title=%EC%9A%94%EC%95%84%ED%9E%98_1%EC%84%B8_%EB%84%A4%EC%8A%A4%ED%86%A0%EB%A5%B4&amp;action=edit&amp;redlink=1" title="요아힘 1세 네스토르 (없는 문서)">요아힘 1세 네스토르</a>의 딸로 알브레히트보다 20살 연하인 5촌 조카딸이 된이다.</p>

<p><a class="new" href="/w/index.php?title=%EB%B8%8C%EB%9E%80%EB%8D%B4%EB%B6%80%EB%A5%B4%ED%81%AC-%EC%95%88%EC%8A%A4%EB%B0%94%ED%9D%90&amp;action=edit&amp;redlink=1" title="브란덴부르크-안스바흐 (없는 문서)">브란덴부르크-안스바흐</a>와 <a class="new" href="/w/index.php?title=%EC%BD%9C%EB%A3%B8%EB%B0%94%ED%9D%90&amp;action=edit&amp;redlink=1" title="콜룸바흐 (없는 문서)">콜룸바흐</a>, <a href="/wiki/%EB%B0%94%EC%9D%B4%EB%A1%9C%EC%9D%B4%ED%8A%B8" title="바이로이트">바이로이트</a>의 <a href="/wiki/%ED%9B%84%EC%9E%91" title="후작">후작</a> <a href="/wiki/%ED%94%84%EB%A6%AC%EB%93%9C%EB%A6%AC%ED%9E%88_2%EC%84%B8_(%EB%B8%8C%EB%9E%80%EB%8D%B4%EB%B6%80%EB%A5%B4%ED%81%AC-%EC%95%88%EC%8A%A4%EB%B0%94%ED%9D%90-%EC%BD%9C%EB%A6%84%EB%B0%94%ED%9D%90)" title="프리드리히 2세 (브란덴부르크-안스바흐-콜름바흐)">프리드리히 2세</a>의 셋째 아들이었던 알베르트는 청소년기에 <a href="/wiki/%EC%BE%B0%EB%A5%B8_%EB%8C%80%EC%84%B1%EB%8B%B9" title="쾰른 대성당">쾰른 대성당</a>에서 그리스도교적 규범을 배웠으며, 주로 <a href="/wiki/%EC%BE%B0%EB%A5%B8" title="쾰른">쾰른</a> 선제후 헤세의 헤르만 4세의 궁정에서 시간을 보냈다. 그는 헤세의 헤르만 4세로부터 <a href="/wiki/%EC%BE%B0%EB%A5%B8_%EB%8C%80%EC%84%B1%EB%8B%B9" title="쾰른 대성당">쾰른 대성당</a>의 캐논에 임명되었다. 그는 성당의 캐논직에 있었으나 <a href="/wiki/%EC%88%98%ED%95%99" title="수학">수학</a>과 <a href="/wiki/%EA%B3%BC%ED%95%99" title="과학">과학</a>에 관심이 있었고, 과학적 이론에 심취하여 그리스도교 교리에 대해 반박을 제기하기도 했다. 그의 수학, 과학에 대한 흥미는 다른 성직자들에게도 알려졌지만, 일부 성직자들의 지원을 받았다.</p>

브란덴부르크-안스바흐">게오르크 경건공</a>이 일으킨 <a href="/wiki/%EC%BF%A0%EB%8D%B0%ED%83%80" title="쿠데타">쿠데타</a>로 축출되면서, 별다른 영지를 받지 못한 알브레히트는 <a class="mw-redirect" <p><a href="/wiki/1519%EB%85%84" title="1519년">1519년</a> <a href="/wiki/12%EC%9B%94" title="12월">12월</a> <a href="/wiki/%EC%8B%A0%EC%84%B1_%EB%A1%9C%EB%A7%88_%EC%A0%9C%EA%B5%AD" title="신성 로마 제국">신성 로마 제국</a>의 황제 <a class="mw-disambig" href="/wiki/%EB%A7%89%EC%8B%9C%EB%B0%80%EB%A6%AC%EC%95%88_1%EC%84%B8" title="막시밀리안 1세">막시밀리안 1세</a>는 폴란드의 전쟁을 선언하였다. 그는 중남부 프로이센을 원하던 황제 <a class="mw-disambig" href="/wiki/%EB%A7%89%EC%8B%9C%EB%B0%80%EB%A6%AC%EC%95%88_1%EC%84%B8" title="막시밀리안 1세">막시밀리안 1세</a>를 설득하여 <a href="/wiki/1521%EB%85%84" title="1521년">1521년</a> 4년간의 휴전 협정을 체결케 하여 <a href="/wiki/%ED%8F%B4%EB%9E%80%EB%93%9C" title="폴란드">폴란드</a>와 <a href="/wiki/%EC%98%A4%EC%8A%A4%ED%8A%B8%EB%A6%AC%EC%95%84" title="오스트리아">오스트리아</a> 간의 갈등을 완화시켰다. 한편 <a href="/wiki/%EB%A7%88%EB%A5%B4%ED%8B%B4_%EB%A3%A8%ED%84%B0" title="마르틴 루터">마르틴 루터</a>의 종교개혁으로 종교 갈등이 발생했을 때 그는 작센 공 프리드리히 3세 등과 함께 몇 안되는 <a class="mw-redirect" href="/wiki/%EB%A3%A8%ED%84%B0" title="루터">루터</a> 지지자가 되었다. 그의 루터 지지 소식을 접한 <a href="/wiki/1522%EB%85%84" title="1522년">1522년</a> <a class="new" href="/w/index.php?title=%EC%95%88%EB%93%9C%EB%A0%88%EC%95%84%EC%8A%A4_%EC%98%A4%EC%A7%80%EC%95%88%EB%8D%94&amp;action=edit&amp;redlink=1" title="안드레아스 오지안더 (없는 문서)">안드레아스 오지안더</a>는 그에게 루터교 운동에 적극 참여해달라고 여러번 설득한다. 오지안더의 거듭된 설득으로 그는 루터교 운동에 가담하게 된다. 1522년 하순 알브레히트는 <a href="/wiki/%EB%B9%84%ED%85%90%EB%B2%A0%EB%A5%B4%ED%81%AC" title="비텐베르크">비텐베르크</a>를 방문하여 <a href="/wiki/%EB%A7%88%EB%A5%B4%ED%8B%B4_%EB%A3%A8%ED%84%B0" title="마르틴 루터">마르틴 루터</a>와 담론하였다. 루터는 그에게 <a class="new" href="/w/index.php?title=%EA%B0%80%ED%86%A8%EB%A6%AD_%EA%B5%90%EB%A6%AC&amp;action=edit&amp;redlink=1" title="가톨릭 교리 (없는 문서)">가톨릭 교리</a>에 얽매이지 말고 결혼할 것을 권고하고, 새로운 세속국가의 건설을 호소하였다. 알브레히트는 루터의 제안을 자신의 친척들과 일부 논의했지만 동의를 얻지 못했다. 루터는 새로운 세속국가의 건설은 조심스럽게 진행해야 한다며, 만약 프로테스탄트 교리를 채택한 기사는 교황 <a class="new" href="/w/index.php?title=%EC%95%84%EB%93%9C%EB%A6%AC%EC%95%88_4%EC%84%B8&amp;action=edit&amp;redlink=1" title="아드리안 4세 (없는 문서)">아드리안 4세</a>로부터 처벌을 받을 수 있다고 조언하였다.</p>
<h3><span class="mw-headline" id=".ED.94.84.EB.A1.9C.ED.85.8C.EC.8A.A4.ED.83.84.ED.8A.B8_.EA.B0.9C.EC.A2.85.EA.B3.BC_.ED.94.84.EB.A1.9C.EC.9D.B4.EC.84.BC_.EA.B3.B5.EA.B5.AD_.EA.B1.B4.EA.B5.AD">프로테스탄트 개종과 프로이센 공국 건국</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%95%8C%EB%B8%8C%EB%A0%88%ED%9E%88%ED%8A%B8_%ED%8F%B0_%ED%94%84%EB%A1%9C%EC%9D%B4%EC%84%BC_%EA%B3%B5%EC%9E%91&amp;action=edit&amp;section=4" title="부분 편집: 프로테스탄트 개종과 프로이센 공국 건국">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p><a href="/wiki/1525%EB%85%84" title="1525년">1525년</a> <a href="/wiki/2%EC%9B%94_10%EC%9D%BC" title="2월 10일">2월 10일</a> 알브레히트는 프로테스탄트로 개종과 동시에 기사이자 사제 신분에서 벗어났고, 결혼하여 자식을 둘 수 있게 된다. 알브레히트는 <a href="/wiki/1525%EB%85%84" title="1525년">1525년</a> <a href="/wiki/4%EC%9B%94_10%EC%9D%BC" title="4월 10일">4월 10일</a> 그는 외삼촌 <a class="mw-redirect" href="/wiki/%EC%A7%80%EA%B8%B0%EC%8A%A4%EB%AC%B8%ED%8A%B8_1%EC%84%B8_(%ED%8F%B4%EB%9E%80%EB%93%9C)" title="지기스문트 1세 (폴란드)">지기스문트 1세</a>로부터 새로운 영지인 <a href="/wiki/%ED%94%84%EB%A1%9C%EC%9D%B4%EC%84%BC" title="프로이센">프로이센</a>을 창설, 할당받고 <a href="/wiki/%ED%94%84%EB%A1%9C%EC%9D%B4%EC%84%BC" title="프로이센">프로이센</a> <a class="mw-disambig" href="/wiki/%EA%B3%B5%EC%9E%91" title="공작">공작</a>에 임명되었다. 초기에 폴란드의 왕 지기스문트는 그에게 공작 작위 수여를 거부했다. 그러나 <a href="/wiki/%EB%A7%88%EB%A5%B4%ED%8B%B4_%EB%A3%A8%ED%84%B0" title="마르틴 루터">마르틴 루터</a>의 막후 설득으로 공작 작위 수여식이 거행되었다. 그의 영지는 독일령인 서부 프로이센과 폴란드의 영향력하에 있던 동부 프로이센이었다. 또한 그는 루터파를 국교로 삼았다. 그는 바로 <a class="new" href="/w/index.php?title=%ED%8F%B4%EB%9E%80%EB%93%9C-%EB%A6%AC%ED%88%AC%EC%95%84%EB%8B%88%EC%95%84_%EC%97%B0%EB%B0%A9_%EC%99%95%EA%B5%AD&amp;action=edit&amp;redlink=1" title="폴란드-리투아니아 연방 왕국 (없는 문서)">폴란드-리투아니아 연방 왕국</a>의 봉신으로 충성을 맹약하였고, 프로이센 공작직은 알브레히트의 남자 후손들만이 계승할 수 있다는 법령을 신설하였다. 알브레히트는 자신의 신설 공작령에 새로운 도시인 <a href="/wiki/%EC%BE%A8%EB%8B%88%ED%9E%88%EC%8A%A4%EB%B2%A0%EB%A5%B4%ED%81%AC" title="쾨니히스베르크">쾨니히스베르크</a>(Königsberg, 왕의 도시라는 뜻)를 건설하고, 프로이센의 수도로 삼았다.</p>
<p>알브레히트는 폴란드-리투아니아 연방 왕국의 봉신이 되는 조건으로 자치권을 요구했다. 자체적인 군대 양성을 허용할 것과 폴란드 굴덴 외에 <a href="/wiki/%ED%94%84%EB%A1%9C%EC%9D%B4%EC%84%BC" title="프로이센">프로이센</a>의 자체적인 화폐 주조권, 그리고 폴란드의 속국이지만 독자적인 외교활동을 수용할 것을 <a class="mw-redirect" href="/wiki/%EC%A7%80%EA%B8%B0%EC%8A%A4%EB%AC%B8%ED%8A%B8_1%EC%84%B8_(%ED%8F%B4%EB%9E%80%EB%93%9C)" title="지기스문트 1세 (폴란드)">지기스문트 1세</a>에게 요구하여 허락받았다. 또한 프로이센의 지역 의회를 설치하여 주민 자치권을 허용하였다.</p>
<p><a href="/wiki/%ED%8F%B4%EB%9E%80%EB%93%9C" title="폴란드">폴란드</a>에서 쾨니히스베르크로 돌아온 뒤, 알브레히트는 자신의 <a class="mw-redirect" href="/wiki/%ED%94%84%EB%A1%9C%ED%85%8C%EC%8A%A4%ED%83%84%ED%8A%B8" title="프로테스탄트">프로테스탄트</a> 개종을 선언했다. 그가 재직했던 <a class="mw-redirect" href="/wiki/%ED%8A%9C%ED%8A%BC_%EA%B8%B0%EC%82%AC%EB%8B%A8" title="튜튼 기사단">튜튼 기사단</a>은 충격을 받았고, 기사들은 노골적인 반대입장과 개종 철회를 요구했다. 그러나 알브레히트는 <a href="/wiki/%EC%BE%A8%EB%8B%88%ED%9E%88%EC%8A%A4%EB%B2%A0%EB%A5%B4%ED%81%AC" title="쾨니히스베르크">쾨니히스베르크</a> 시민들의 인망을 얻었고, 개종 거부와 동시에 튜튼 기사단의 프로이센지부 단원들에게 <a href="/wiki/%EB%A3%A8%ED%84%B0%EA%B5%90" title="루터교">루터교</a>로 개종을 강요했다. 메메르트의 사령관 <a class="new" href="/w/index.php?title=%EC%97%90%EB%A6%AC%ED%9E%88_%ED%8F%B0_%EB%B8%8C%EB%9D%BC%EC%9A%B4%EC%8A%88%EB%B0%94%EC%9D%B4%ED%8A%B8-%EB%B3%BC%ED%8E%9C%EB%B7%94%ED%8E%A0&amp;action=edit&amp;redlink=1" title="에리히 폰 브라운슈바이트-볼펜뷔펠 (없는 문서)">에리히 폰 브라운슈바이트-볼펜뷔펠</a> 등 소수는 끝까지 <a class="mw-redirect" href="/wiki/%EB%A1%9C%EB%A7%88_%EA%B0%80%ED%86%A8%EB%A6%AD" title="로마 가톨릭">로마 가톨릭</a>을 유지하며, 개종 반대 입장을 고수하였다. 알브레히트의 개신교 개종 선언과 폴란드 왕국 제후 임명을 계기로, <a class="mw-redirect" href="/wiki/%ED%8A%9C%ED%8A%BC_%EA%B8%B0%EC%82%AC%EB%8B%A8" title="튜튼 기사단">튜튼 기사단</a>장 퇴임 후에도 그에게 충성을 맹세했던 <a class="mw-redirect" href="/wiki/%ED%8A%9C%ED%8A%BC_%EA%B8%B0%EC%82%AC%EB%8B%A8" title="튜튼 기사단">튜튼 기사단</a>의 대군행관(Großkomtur), 의무총감(Großspittler), 기사단 재무총감(Ordenstressler), 기사단 의복감(Ordenstrappier) 등 고위 지휘관들을 비롯한 많은 기사들이 신앙을 지키겠다며 알브레히트의 곁을 떠났다. 알브레히트가 사망할 당시 튜튼 기사단에는 55명의 기사만이 남아 있었다. 남아 있던 기사들도 알브레히트의 개종에 혐오감을 느끼고, 후임 그랜드 마스터인 <a class="new" href="/w/index.php?title=%EB%B0%9C%ED%84%B0_%ED%8F%B0_%EC%B9%B4%EB%A1%A0%EB%B6%80%EB%A5%B4%ED%81%AC&amp;action=edit&amp;redlink=1" title="발터 폰 카론부르크 (없는 문서)">발터 폰 카론부르크</a>에게, 프로이센은 실패한 국가라며 불만을 토로하였다.</p>
<p>독일의 북동부의 서부 프로이센은 <a href="/wiki/%EC%8B%A0%EC%84%B1_%EB%A1%9C%EB%A7%88_%EC%A0%9C%EA%B5%AD" title="신성 로마 제국">신성 로마 제국</a>의 영토였지만, 그는 공작 작위를 수여받는 대신, 폴란드의 국왕 <a class="mw-redirect" href="/wiki/%EC%A7%80%EA%B8%B0%EC%8A%A4%EB%AC%B8%ED%8A%B8_1%EC%84%B8_(%ED%8F%B4%EB%9E%80%EB%93%9C)" title="지기스문트 1세 (폴란드)">지기스문트 1세</a>의 봉신임을 공개선언함으로써 가톨릭을 지지하던 <a class="mw-redirect" href="/wiki/%ED%95%A9%EC%8A%A4%EB%B6%80%EB%A5%B4%ED%81%AC_%EC%99%95%EA%B0%80" title="합스부르크 왕가">합스부르크 왕가</a>로부터 자신의 영토를 지킬 수 있었다. <a class="mw-redirect" href="/wiki/%EC%A7%80%EA%B8%B0%EC%8A%A4%EB%AC%B8%ED%8A%B8_1%EC%84%B8_(%ED%8F%B4%EB%9E%80%EB%93%9C)" title="지기스문트 1세 (폴란드)">지기스문트 1세</a>는 자신이 가톨릭 신자인데다가 가톨릭 국가의 군주였는데도, 프로테스탄트로 개종한 조카 알브레히트를 옹호하였다. 그가 신성 로마 제국 황제가 아닌 폴란드-리투아니아 연방의 국왕 <a class="mw-redirect" href="/wiki/%EC%A7%80%EA%B8%B0%EC%8A%A4%EB%AC%B8%ED%8A%B8_1%EC%84%B8_(%ED%8F%B4%EB%9E%80%EB%93%9C)" title="지기스문트 1세 (폴란드)">지기스문트 1세</a>의 봉신이 된 것에는, 외삼촌-조카간이라는 개인적인 인연 외에 <a href="/wiki/%EB%A7%88%EB%A5%B4%ED%8B%B4_%EB%A3%A8%ED%84%B0" title="마르틴 루터">마르틴 루터</a>의 설득도 작용하였다. 바로 <a href="/wiki/%EC%8B%A0%EC%84%B1_%EB%A1%9C%EB%A7%88_%EC%A0%9C%EA%B5%AD" title="신성 로마 제국">신성 로마 제국</a> <a href="/wiki/%EC%B9%B4%EB%A5%BC_5%EC%84%B8" title="카를 5세">카를 5세</a>와 교황 <a class="mw-redirect" href="/wiki/%ED%81%B4%EB%A0%88%EB%A9%98%EC%8A%A4_7%EC%84%B8" title="클레멘스 7세">클레멘스 7세</a>는 이를 두고 '크라쿠프의 젖소 거래'라며 비웃었다.</p>
<p>알브레히트는 자신의 <a href="/wiki/%ED%94%84%EB%A1%9C%EC%9D%B4%EC%84%BC_%EA%B3%B5%EA%B5%AD" title="프로이센 공국">프로이센 공국</a>을 종교국이 아닌 세속 국가로 운영하였다. 그때까지 프로이센의 일부 지역에 남아 있던 프로이센 토속 신앙등에 대해서도 허용하였다. 기존의 유럽의 황제와 국왕, 제후들이 독실한 신앙심에 의해서 혹은 형식적으로나마 <a class="mw-redirect" href="/wiki/%EB%A1%9C%EB%A7%88_%EA%B0%80%ED%86%A8%EB%A6%AD" title="로마 가톨릭">로마 가톨릭</a>을 국교로 삼은 관례를 깬 것이었다. 그의 영향을 받아 <a href="/wiki/1526%EB%85%84" title="1526년">1526년</a> <a class="mw-redirect" href="/wiki/%ED%8A%9C%ED%8A%BC_%EA%B8%B0%EC%82%AC%EB%8B%A8" title="튜튼 기사단">튜튼 기사단</a>의 리보니아 지부장 <a class="new" href="/w/index.php?title=%EA%B3%A0%ED%83%80%EB%A5%B4%ED%8A%B8_%EC%BC%80%ED%8B%80%EB%9F%AC&amp;action=edit&amp;redlink=1" title="고타르트 케틀러 (없는 문서)">고타르트 케틀러</a> 역시 <a href="/wiki/%EC%BF%A0%EB%A5%BC%EB%9E%80%ED%8A%B8" title="쿠를란트">쿠를란트</a>의 공작이 되면서 자신의 영지를 세속 국가로 운영하게 된다.</p>
<h3><span class="mw-headline" id=".EC.A0.80.ED.95.AD.EA.B3.BC_.EC.9E.A0.EB.9E.80.ED.8A.B8_.EB.B0.98.EB.9E.80_.EC.82.AC.EA.B1.B4">저항과 잠란트 반란 사건</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%95%8C%EB%B8%8C%EB%A0%88%ED%9E%88%ED%8A%B8_%ED%8F%B0_%ED%94%84%EB%A1%9C%EC%9D%B4%EC%84%BC_%EA%B3%B5%EC%9E%91&amp;action=edit&amp;section=5" title="부분 편집: 저항과 잠란트 반란 사건">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>1522년 '튜턴 기사수도회' 대수도원장 알베르트 폰 호헨졸레른을 설득하여 루터파 운동에 가담하게 했다.</p>
<p>그의 개종 선언과 세속국가화에 대해서는 오랫동안 반발과 논란이 벌어졌다. 남아 있던 튜튼 기사단은 그가 가톨릭에서 프로테스탄트로 너무 쉽게 개종했다며 저항하였다. 그러던 중 <a href="/wiki/1525%EB%85%84" title="1525년">1525년</a> <a class="new" href="/w/index.php?title=%EC%9E%A0%EB%9E%80%ED%8A%B8&amp;action=edit&amp;redlink=1" title="잠란트 (없는 문서)">잠란트</a>에서 농민 시위가 발생했다. 귀족들의 과도한 세금 징수와 노동력 무급 착취, 갑작스러운 알브레히트의 개종 및 신 국가의 세속국가화, <a class="mw-redirect" href="/wiki/%ED%8A%9C%ED%8A%BC_%EA%B8%B0%EC%82%AC%EB%8B%A8" title="튜튼 기사단">튜튼 기사단</a>의 세속화 문제 등에 대한 불만, 알브레히트 및 프로이센 정부의 프로테스탄트 지지 등에 대한 불만이었다. 카이멘 출신 농부와 샤켄 출신의 여관 주인을 지도자로 하는 이들 반란군은 <a href="/wiki/%EC%BE%A8%EB%8B%88%ED%9E%88%EC%8A%A4%EB%B2%A0%EB%A5%B4%ED%81%AC" title="쾨니히스베르크">쾨니히스베르크</a> 시민들의 폭넓은 지지를 받았다. 농민군은 정부에 귀족들의 세금 강제 징수 폐지, 모든 형태의 귀족이 거두는 세금 폐지, 세금을 2마르크 대로 조정하고, 농민들에게는 1후페의 토지를 무상으로 나눠줄 것을 요구했다. 이들은 귀족들의 난폭함에 일으킨 저항이며 알브레히트에게 저항하는 것이 아니라고 밝혔다. 바로 알브레히트는 농민군과 회의를 개최하여 농민들의 의견을 수렴하였다. 현장에서 회의가 진행되었는데, 이 동안 농민군의 지도자 두 명은 반역죄로 처형되었다. 농민군의 일부 조건을 수용하면서 농민군은 자연 해산되었다. 이후 프로이센은 개신교의 지역이자 파벌주의가 만연한 지역으로 알려지게 되었다.</p>
<p><a href="/wiki/1526%EB%85%84" title="1526년">1526년</a> <a href="/wiki/3%EC%9B%94_1%EC%9D%BC" title="3월 1일">3월 1일</a> 알브레히트는 <a href="/wiki/%EB%8D%B4%EB%A7%88%ED%81%AC" title="덴마크">덴마크</a>의 공주 도로테아와 결혼하였다. 이는 정략결혼으로 <a href="/wiki/%EB%8D%B4%EB%A7%88%ED%81%AC" title="덴마크">덴마크</a>는 <a href="/wiki/%EC%8A%A4%EC%B9%B8%EB%94%94%EB%82%98%EB%B9%84%EC%95%84_%EB%B0%98%EB%8F%84" title="스칸디나비아 반도">스칸디나비아 반도</a> 내에 루터교도가 많은 국가이기도 했다. 알브레히트는 <a class="new" href="/w/index.php?title=%EB%B8%8C%EB%9E%80%EB%8D%B4%EB%B6%80%EB%A5%B4%ED%81%AC-%EC%95%88%EC%8A%A4%EB%B0%94%ED%9D%90&amp;action=edit&amp;redlink=1" title="브란덴부르크-안스바흐 (없는 문서)">브란덴부르크-안스바흐</a> <a href="/wiki/%ED%9B%84%EC%9E%91" title="후작">후작</a>인 형 <a class="mw-redirect" href="/wiki/%EA%B2%8C%EC%98%A4%EB%A5%B4%ED%81%AC_%ED%8F%B0_%EB%B8%8C%EB%9E%80%EB%8D%B4%EB%B6%80%EB%A5%B4%ED%81%AC-%EC%95%88%EC%8A%A4%EB%B0%94%ED%9D%90" title="게오르크 폰 브란덴부르크-안스바흐">게오르크 폰 브란덴부르크-안스바흐</a>의 도움을 받아, <a href="/wiki/%ED%94%84%EB%9E%91%EC%BC%84" title="프랑켄">프랑켄</a>과 상부 <a class="mw-redirect" href="/wiki/%EC%8A%90%EB%A0%88%EC%A7%80%EC%97%94" title="슐레지엔">슐레지엔</a>에 <a href="/wiki/%EB%A3%A8%ED%84%B0%EA%B5%90" title="루터교">루터교</a>회를 설립하였다. 브란덴부르크와 뉘른베르크에 프로테스탄트 교회의 설립을 추진한 그는 브란덴부르크, 뉘른베르크에 각각 개신교 교구 관리자를 두었다.</p>
<h3><span class="mw-headline" id=".EC.A0.95.EC.B9.98.2C_.EC.82.AC.ED.9A.8C.EA.B0.9C.ED.98.81">정치, 사회개혁</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%95%8C%EB%B8%8C%EB%A0%88%ED%9E%88%ED%8A%B8_%ED%8F%B0_%ED%94%84%EB%A1%9C%EC%9D%B4%EC%84%BC_%EA%B3%B5%EC%9E%91&amp;action=edit&amp;section=6" title="부분 편집: 정치, 사회개혁">편집</a><span class="mw-editsection-bracket">]</span></span></h3>

<p>즉위 직후부터 그는 농노들을 해방시켜 귀족들의 세력을 약화시켰다. 칙령을 내려 그는 모든 마을에 학교를 설립하게 하여 해방 농노와 농노 출신자 자녀들에게 학업의 기회를 부여하였다.</p>
<p>알브레히트는 루터교를 국교로 정함과 동시에 루터교적 가르침을 시민들에게 교육하였다. 알브레히트의 이러한 정책은 1510년 이후 60년간 <a class="mw-redirect" href="/wiki/%ED%94%84%EB%A1%9C%ED%85%8C%EC%8A%A4%ED%83%84%ED%8A%B8" title="프로테스탄트">프로테스탄트</a> 확산의 원인을 제공하였다. 그런데 가톨릭교도 및 이교도들에게는 세금을 많이 부과했다. <a href="/wiki/%EC%8B%A0%EC%84%B1_%EB%A1%9C%EB%A7%88_%EC%A0%9C%EA%B5%AD" title="신성 로마 제국">신성 로마 제국</a> 정부와 <a class="mw-redirect" href="/wiki/%EA%B0%80%ED%86%A8%EB%A6%AD%EA%B5%90%ED%9A%8C" title="가톨릭교회">가톨릭교회</a>는 알브레히트가 불법 행위를 자행한다며 비판하였다. 알브레히트는 <a href="/wiki/%ED%94%84%EB%A1%9C%EC%9D%B4%EC%84%BC" title="프로이센">프로이센</a> 지역 <a class="mw-redirect" href="/wiki/%EA%B0%80%ED%86%A8%EB%A6%AD%EA%B5%90%ED%9A%8C" title="가톨릭교회">가톨릭교회</a>의 토지와 재산을 몰수하여 지역 귀족들에게 나눠줌으로써 불만을 무마시켰다.</p>
<p><a href="/wiki/%ED%8F%B4%EB%9E%80%EB%93%9C" title="폴란드">폴란드</a>에서 최초로 <a class="mw-redirect" href="/wiki/%EC%BD%94%ED%8E%98%EB%A5%B4%EB%8B%88%EC%BF%A0%EC%8A%A4" title="코페르니쿠스">코페르니쿠스</a>의 지동설을 공식 지지하였다. <a class="mw-redirect" href="/wiki/%EC%BD%94%ED%8E%98%EB%A5%B4%EB%8B%88%EC%BF%A0%EC%8A%A4" title="코페르니쿠스">코페르니쿠스</a>는 <a href="/wiki/%ED%8F%B4%EB%9E%80%EB%93%9C" title="폴란드">폴란드</a> 본국의 외교관이자 의사로 알베르트의 측근들을 치료하기도 했다. 또한 알베르트는 새 달력 제정을 생각하여 <a href="/wiki/%ED%94%84%EB%A1%9C%EC%9D%B4%EC%84%BC" title="프로이센">프로이센</a> 내의 <a href="/wiki/%EB%A3%A8%ED%84%B0%EA%B5%90" title="루터교">루터교</a>도 <a href="/wiki/%EC%B2%9C%EB%AC%B8%ED%95%99" title="천문학">천문학</a>자들을 동원하여 코페르니쿠스 체계를 이용한 천문계산을 작성하도록 지지했다. 이 계산표는 로마 <a class="new" href="/w/index.php?title=%EA%B7%B8%EB%A0%88%EA%B3%A0%EB%A6%AC%EC%9A%B0%EC%8A%A4_%EA%B5%90%ED%99%A9&amp;action=edit&amp;redlink=1" title="그레고리우스 교황 (없는 문서)">그레고리우스 교황</a>의 달력 개혁에도 영향을 준다.</p>
<p><a href="/wiki/1544%EB%85%84" title="1544년">1544년</a> 그는 일부 반대에도 불구하고 사재를 투자하여 <a href="/wiki/%EC%BE%A8%EB%8B%88%ED%9E%88%EC%8A%A4%EB%B2%A0%EB%A5%B4%ED%81%AC" title="쾨니히스베르크">쾨니히스베르크</a>에 <a class="new" href="/w/index.php?title=%EC%BE%A8%EB%8B%88%ED%9E%88%EC%8A%A4%EB%B2%A0%EB%A5%B4%ED%81%AC_%EC%95%84%EC%B9%B4%EB%8D%B0%EB%AF%B8&amp;action=edit&amp;redlink=1" title="쾨니히스베르크 아카데미 (없는 문서)">쾨니히스베르크 아카데미</a>를 설립한다. 이는 <a href="/wiki/%EC%8B%A0%EC%84%B1_%EB%A1%9C%EB%A7%88_%EC%A0%9C%EA%B5%AD" title="신성 로마 제국">신성 로마 제국</a>내의 가톨릭 명문 대학 <a class="new" href="/w/index.php?title=%ED%81%AC%EB%9D%BC%EC%B9%B4%EC%9A%B0_%EC%95%84%EC%B9%B4%EB%8D%B0%EB%AF%B8&amp;action=edit&amp;redlink=1" title="크라카우 아카데미 (없는 문서)">크라카우 아카데미</a>에 대응하기 위해 설립한 것이다. 그의 쾨니히스베르크 대학은 독일 내에서 <a class="mw-redirect" href="/wiki/%EB%A7%88%EB%A5%B4%EB%B6%80%EB%A5%B4%ED%81%AC_%EB%8C%80%ED%95%99" title="마르부르크 대학">마르부르크 대학</a>에 이은 두 번째 루터교 대학이었다. 알브레히트는 독일어 교재와 프로테스탄트 교리 문답집의 인쇄 비용을 스스로 부담했다. 그는 <a class="new" href="/w/index.php?title=%EC%95%88%EB%93%9C%EB%A0%88%EC%95%84%EC%8A%A4_%EC%98%A4%EC%8B%9C%EC%95%88%EB%8D%94&amp;action=edit&amp;redlink=1" title="안드레아스 오시안더 (없는 문서)">안드레아스 오시안더</a>를 초빙하여 쾨니히스베르크 대학의 교수진의 한 사람으로 위촉한다. 1549년에는 천문학 서적의 번역과 발간에 투자하기도 했다. 1551년 안드레아스 헤스를 초빙하여 <a href="/wiki/%EC%BE%A8%EB%8B%88%ED%9E%88%EC%8A%A4%EB%B2%A0%EB%A5%B4%ED%81%AC" title="쾨니히스베르크">쾨니히스베르크</a> 성의 남쪽 성벽을 건설하였다. 그밖에 알베르트는 쾨니히스베르크의 공립(公立) 도서관<sup class="reference" id="cite_ref-6"><a href="#cite_note-6">[6]</a></sup>을 설립하였다.</p>
<p>1548년의 아우크스부르크 가신조 협정에 참석한 알브레히트는 당시 반 카를 5세파 군주의 한 사람이기도 했다. 프로테스탄트가 확산되면서 그의 영지 내에서는 루터파와 오시안더파의 신학 해석에 대해 의견이 갈라지면서 물리적 폭력으로 번졌다. 알브레히트는 폭력을 금지함과 동시에 오시안더파의 손을 들어주었다.</p>
<h3><span class="mw-headline" id=".EC.B5.9C.ED.9B.84">최후</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%95%8C%EB%B8%8C%EB%A0%88%ED%9E%88%ED%8A%B8_%ED%8F%B0_%ED%94%84%EB%A1%9C%EC%9D%B4%EC%84%BC_%EA%B3%B5%EC%9E%91&amp;action=edit&amp;section=7" title="부분 편집: 최후">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>그의 치세 후반에는 교리를 놓고 루터파와 오시안더파의 신학 해석 갈등이 계속되었다. 1552년 안드레아스 오시안더가 죽자 알브레히트는 파울로 스칼리츠라는 가명으로 작가, 여행가로 활동하던 <a class="new" href="/w/index.php?title=%EC%9A%94%ED%95%9C_%ED%8E%91%ED%81%AC&amp;action=edit&amp;redlink=1" title="요한 펑크 (없는 문서)">요한 펑크</a>를 프로테스탄트 지도자로 임명하였다. <a class="new" href="/w/index.php?title=%EC%9A%94%ED%95%9C_%ED%8E%91%ED%81%AC&amp;action=edit&amp;redlink=1" title="요한 펑크 (없는 문서)">요한 펑크</a>는 그에게 큰 영향을 행사했고, 알브레히트의 총애와 영향력을 이용해 공공 비용으로 상당한 재산을 취득한다. 또한 그에게 오시안더의 가르침에 대한 비판을 강요하기도 했다. 알브레히트의 생전에 이미 요한 펑크의 활동은 정치적, 종교적 분란을 제공했고, <a href="/wiki/1566%EB%85%84" title="1566년">1566년</a> <a href="/wiki/10%EC%9B%94_28%EC%9D%BC" title="10월 28일">10월 28일</a> 법원 음모 사건을 계기로 처형된다.</p>
<p>알베르트는 1526년 <a class="new" href="/w/index.php?title=%EB%8D%B4%EB%A7%88%ED%81%AC%EC%9D%98_%EC%99%95&amp;action=edit&amp;redlink=1" title="덴마크의 왕 (없는 문서)">덴마크의 왕</a> <a class="new" href="/w/index.php?title=%ED%94%84%EB%A0%88%EB%8D%B0%EB%A6%AC%ED%81%AC_1%EC%84%B8&amp;action=edit&amp;redlink=1" title="프레데리크 1세 (없는 문서)">프레데리크 1세</a>의 딸 <a class="new" href="/w/index.php?title=%EB%8D%B4%EB%A7%88%ED%81%AC%EC%9D%98_%EB%8F%84%EB%A1%9C%ED%85%8C%EC%95%84&amp;action=edit&amp;redlink=1" title="덴마크의 도로테아 (없는 문서)">덴마크의 도로테아</a>(1504 - 1547)와 결혼하여 4녀 2남을 두었지만 두 아들 프리드리히 알베르트는 2살, 알베르트는 1살때 죽었다. 1550년 브론스위크-르네부르크 백작 에릭의 딸 <a class="new" href="/w/index.php?title=%EC%95%88%EB%82%98_%EB%A7%88%EB%A6%AC%EC%95%84&amp;action=edit&amp;redlink=1" title="안나 마리아 (없는 문서)">안나 마리아</a>와 재혼하여 딸 엘리자베스와 아들 <a class="mw-redirect" href="/wiki/%EC%95%8C%EB%B8%8C%EB%A0%88%ED%9E%88%ED%8A%B8_%ED%94%84%EB%A6%AC%EB%93%9C%EB%A6%AC%ED%9E%88_%ED%8F%B0_%ED%98%B8%EC%97%94%EC%B4%90%EB%A0%88%EB%A5%B8" title="알브레히트 프리드리히 폰 호엔촐레른">알베르트 프리드리히</a>를 두었다. <a href="/wiki/%EC%8B%A0%EC%84%B1_%EB%A1%9C%EB%A7%88_%EC%A0%9C%EA%B5%AD" title="신성 로마 제국">신성 로마 제국</a> 국내의 정치, 종교 문제에 적극적이던 그는 시종일관 황제 <a href="/wiki/%EC%B9%B4%EB%A5%BC_5%EC%84%B8" title="카를 5세">카를 5세</a> 황제에 맞서 <a href="/wiki/%EB%A7%88%EB%A5%B4%ED%8B%B4_%EB%A3%A8%ED%84%B0" title="마르틴 루터">마르틴 루터</a>의 <a href="/wiki/%EC%A2%85%EA%B5%90_%EA%B0%9C%ED%98%81" title="종교 개혁">종교 개혁</a>을 적극 지지, 후원하기도 했다.</p>
<p>알브레히트는 많은 귀족, 학자, 프로테스탄트 지도자들과 서신을 주고받았으며, 그 중 많은 편지들이 20세기에도 전해졌다. <a href="/wiki/1568%EB%85%84" title="1568년">1568년</a> <a href="/wiki/3%EC%9B%94_20%EC%9D%BC" title="3월 20일">3월 20일</a> 역병(plague)에 걸려서 사망했다. 그가 병에 걸린지 16시간 만에 그의 재혼한 부인 <a class="new" href="/w/index.php?title=%EC%95%88%EB%82%98_%EB%A7%88%EB%A6%AC%EC%95%84&amp;action=edit&amp;redlink=1" title="안나 마리아 (없는 문서)">안나 마리아</a> 역시 역병에 감염되어 같은 날 사망하게 된다.</p>
<h3><span class="mw-headline" id=".EC.82.AC.ED.9B.84">사후</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%95%8C%EB%B8%8C%EB%A0%88%ED%9E%88%ED%8A%B8_%ED%8F%B0_%ED%94%84%EB%A1%9C%EC%9D%B4%EC%84%BC_%EA%B3%B5%EC%9E%91&amp;action=edit&amp;section=8" title="부분 편집: 사후">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>역병에 걸린 알브레히트는 <a class="new" href="/w/index.php?title=%EC%BE%A8%EB%8B%88%ED%9E%88%EC%8A%A4%EB%B2%A0%EB%A5%B4%ED%81%AC_%EC%84%B1%EB%8B%B9&amp;action=edit&amp;redlink=1" title="쾨니히스베르크 성당 (없는 문서)">쾨니히스베르크 성당</a> 내에 자신의 무덤을 설계해두었다. 알브레히트의 시신은 쾨니히스베르크 성당에 안치되었다.</p>
