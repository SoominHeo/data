<p><b>청년전사운동</b>(<span style="font-size: smaller;"><a href="/wiki/%EC%95%84%EB%9E%8D%EC%96%B4" title="아랍어">아랍어</a>:</span> <span lang="ar" xml:lang="ar">حركة الشباب المجاهدين‎</span> <small>하라카트 알샤바브 알무자히딘<sup>[<a href="/wiki/%EC%9C%84%ED%82%A4%EB%B0%B1%EA%B3%BC:%EC%99%B8%EB%9E%98%EC%96%B4_%ED%91%9C%EA%B8%B0%EB%B2%95/%EC%95%84%EB%9E%8D%EC%96%B4_%EC%8B%9C%EC%95%88" title="위키백과:외래어 표기법/아랍어 시안">*</a>]</sup></small>)은 <a href="/wiki/%EC%86%8C%EB%A7%90%EB%A6%AC%EC%95%84" title="소말리아">소말리아</a>의 테러 무장단체이다. 보통 <b>알샤바브</b>(<span lang="ar" xml:lang="ar">الشباب‎</span>)라고 줄여 부른다.</p>
<p><a href="/wiki/%EC%86%8C%EB%A7%90%EB%A6%AC%EC%95%84" title="소말리아">소말리아</a>는 1991년 당시 대통령 <a href="/wiki/%EC%8B%9C%EC%95%84%EB%93%9C_%EB%B0%94%EB%A0%88" title="시아드 바레">시아드 바레</a> 축출 이후, 20여년간 내전을 치뤄왔다. 이 과정에서 이슬람적 질서 회복을 가치로 두고 <a class="new" href="/w/index.php?title=%EC%9D%B4%EC%8A%AC%EB%9E%8C%EB%B2%95%EC%A0%95%EC%97%B0%ED%95%A9&amp;action=edit&amp;redlink=1" title="이슬람법정연합 (없는 문서)">이슬람법정연합</a>(<span lang="en" xml:lang="en">ICU</span>)이 출범하였다. 얄샤바브는 이 단체에 뿌리를 둔다. 알샤바브는 2010년 7월 11일 <a href="/wiki/%EC%9A%B0%EA%B0%84%EB%8B%A4" title="우간다">우간다</a> <a href="/wiki/%EC%BA%84%ED%8C%94%EB%9D%BC" title="캄팔라">캄팔라</a>에서 벌어진 자살폭탄 연쇄테러의 배후로 알려져 있다.<sup class="reference" id="cite_ref-al_1-0"><a href="#cite_note-al-1">[1]</a></sup></p>
<h2><span class="mw-headline" id=".EC.84.B1.ED.96.A5">성향</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%95%8C%EC%83%A4%EB%B0%94%EB%B8%8C_(%EC%86%8C%EB%A7%90%EB%A6%AC%EC%95%84)&amp;action=edit&amp;section=1" title="부분 편집: 성향">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>이 단체는 소말리아 과도정부, <a class="mw-redirect" href="/wiki/%EC%9C%A0%EC%97%94%ED%8F%89%ED%99%94%EC%9C%A0%EC%A7%80%EA%B5%B0" title="유엔평화유지군">유엔평화유지군</a>, <a href="/wiki/%EC%9C%A0%EC%97%94" title="유엔">유엔</a>에 대해 반감을 가지고 있다. 이념적으로는 극단적인 이슬람 경전 해석을 따라간다. 따라서, <a href="/wiki/%EC%9D%8C%EC%95%85" title="음악">음악</a> 금지, 국가 형벌에 <a class="new" href="/w/index.php?title=%EC%8B%A0%EC%B2%B4%EC%A0%88%EB%8B%A8%ED%98%95&amp;action=edit&amp;redlink=1" title="신체절단형 (없는 문서)">신체절단형</a>과 <a href="/wiki/%ED%88%AC%EC%84%9D%ED%98%95" title="투석형">투석형</a>의 포함을 요구한다.<sup class="reference" id="cite_ref-al_1-1"><a href="#cite_note-al-1">[1]</a></sup></p>
<h2><span class="mw-headline" id=".EA.B7.9C.EB.AA.A8">규모</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%95%8C%EC%83%A4%EB%B0%94%EB%B8%8C_(%EC%86%8C%EB%A7%90%EB%A6%AC%EC%95%84)&amp;action=edit&amp;section=2" title="부분 편집: 규모">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>소말리아 중남부에 세력을 두고 있는 이 단체는 <a href="/wiki/%EB%AA%A8%EA%B0%80%EB%94%94%EC%8A%88" title="모가디슈">모가디슈</a>에서 정부군과 매일 전투를 벌인다. <a href="/wiki/%EC%95%8C%EC%B9%B4%EC%97%90%EB%8B%A4" title="알카에다">알카에다</a>는 해당 단체를 지원하고 있으며, 이 과정에서 1998년 <a href="/wiki/%EC%BC%80%EB%83%90" title="케냐">케냐</a>·<a href="/wiki/%ED%83%84%EC%9E%90%EB%8B%88%EC%95%84" title="탄자니아">탄자니아</a> 미국 대사관 테러의 배후이자 <a href="/wiki/%EC%95%8C%EC%B9%B4%EC%97%90%EB%8B%A4" title="알카에다">알카에다</a> 최고위층인 <a class="new" href="/w/index.php?title=%ED%8C%8C%EC%A4%84_%EC%95%95%EB%91%98%EB%9D%BC_%EB%AA%A8%ED%95%98%EB%A9%94%EB%93%9C&amp;action=edit&amp;redlink=1" title="파줄 압둘라 모하메드 (없는 문서)">파줄 압둘라 모하메드</a>의 역할이 크다. <a href="/wiki/%EC%95%8C%EC%B9%B4%EC%97%90%EB%8B%A4" title="알카에다">알카에다</a>와 연계를 통해 <a href="/wiki/%EC%9D%B4%EB%9D%BC%ED%81%AC" title="이라크">이라크</a>, <a href="/wiki/%ED%8C%8C%ED%82%A4%EC%8A%A4%ED%83%84" title="파키스탄">파키스탄</a>, <a href="/wiki/%EC%95%84%ED%94%84%EA%B0%80%EB%8B%88%EC%8A%A4%ED%83%84" title="아프가니스탄">아프가니스탄</a> 수백명이 이 단체에 들어가 있다. 전투시 동원 가능한 전력은 3000~7000명 정도이다.<sup class="reference" id="cite_ref-al_1-2"><a href="#cite_note-al-1">[1]</a></sup></p>
