<p><b>알자스</b>(<span style="font-size: smaller;"><a href="/wiki/%ED%94%84%EB%9E%91%EC%8A%A4%EC%96%B4" title="프랑스어">프랑스어</a>:</span> <span lang="fr" xml:lang="fr">Alsace</span>, <span style="font-size: smaller;"><a href="/wiki/%EC%95%8C%EB%A0%88%EB%A7%8C%EC%96%B4" title="알레만어">알레만어</a>:</span> <span lang="gsw" xml:lang="gsw">Elsass</span>, <span style="font-size: smaller;"><a href="/wiki/%EB%8F%85%EC%9D%BC%EC%96%B4" title="독일어">독일어</a>:</span> <span lang="de" xml:lang="de">Elsass, Elsaß</span> <small>엘자스<sup>[<a href="/wiki/%EC%9C%84%ED%82%A4%EB%B0%B1%EA%B3%BC:%EB%8F%85%EC%9D%BC%EC%96%B4%EC%9D%98_%ED%95%9C%EA%B8%80_%ED%91%9C%EA%B8%B0" title="위키백과:독일어의 한글 표기">*</a>]</sup></small>)는 <a href="/wiki/%ED%94%84%EB%9E%91%EC%8A%A4" title="프랑스">프랑스</a> 북동부에 위치한 <a href="/wiki/%ED%94%84%EB%9E%91%EC%8A%A4%EC%9D%98_%EB%A0%88%EC%A7%80%EC%98%B9" title="프랑스의 레지옹">레지옹</a>으로 중심 도시는 <a href="/wiki/%EC%8A%A4%ED%8A%B8%EB%9D%BC%EC%8A%A4%EB%B6%80%EB%A5%B4" title="스트라스부르">스트라스부르</a>이며 면적은 8,280.20㎢, 인구는 1,815,488명(2006년 기준)이다.</p>
<p>남서쪽으로는 <a href="/wiki/%ED%94%84%EB%9E%91%EC%8A%88%EC%BD%A9%ED%85%8C" title="프랑슈콩테">프랑슈콩테</a>(현재의 <a href="/wiki/%EB%B6%80%EB%A5%B4%EA%B3%A0%EB%89%B4%ED%94%84%EB%9E%91%EC%8A%88%EC%BD%A9%ED%85%8C" title="부르고뉴프랑슈콩테">부르고뉴프랑슈콩테</a>), 서쪽으로는 <a href="/wiki/%EB%A1%9C%EB%A0%8C" title="로렌">로렌</a>과 접하며 북쪽과 동쪽으로는 <a href="/wiki/%EB%8F%85%EC%9D%BC" title="독일">독일</a>, 남쪽으로는 <a href="/wiki/%EC%8A%A4%EC%9C%84%EC%8A%A4" title="스위스">스위스</a>와 국경을 접한다. 2개 주(<a href="/wiki/%EB%B0%94%EB%9E%AD_%EC%A3%BC" title="바랭 주">바랭 주</a>, <a href="/wiki/%EC%98%A4%EB%9E%AD_%EC%A3%BC" title="오랭 주">오랭 주</a>)를 관할한다. 2016년 1월 1일을 기해 시행된 레지옹 개편에 따라 알자스와 샹파뉴아르덴, 로렌이 <a href="/wiki/%EA%B7%B8%EB%9E%91%ED%85%8C%EC%8A%A4%ED%8A%B8" title="그랑테스트">그랑테스트</a>(옛 이름 알자스샹파뉴아르덴로렌)로 합병되었다.</p>

<ul>
<li><a href="/wiki/842%EB%85%84" title="842년">842년</a> : <a class="new" href="/w/index.php?title=%EC%8A%A4%ED%8A%B8%EB%9D%BC%EC%8A%A4%EB%B6%80%EB%A5%B4%EC%9D%98_%EB%A7%B9%EC%84%B8&amp;action=edit&amp;redlink=1" title="스트라스부르의 맹세 (없는 문서)">스트라스부르의 맹세</a>, <a href="/wiki/%EC%84%9C%ED%94%84%EB%9E%91%ED%81%AC_%EC%99%95%EA%B5%AD" title="서프랑크 왕국">서프랑크 왕국</a>의 샤를과 <a href="/wiki/%EB%8F%99%ED%94%84%EB%9E%91%ED%81%AC_%EC%99%95%EA%B5%AD" title="동프랑크 왕국">동프랑크 왕국</a>의 루이가 맏형 <a href="/wiki/%EB%A1%9C%ED%83%80%EB%A5%B4_1%EC%84%B8" title="로타르 1세">로타르 1세</a>에 대항해 맺은 맹세.</li>
