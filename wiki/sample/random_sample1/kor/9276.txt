<p><b>알테라 주식회사</b>(<span lang="en" xml:lang="en">Altera Corporation</span>)는 <a class="mw-redirect" href="/wiki/%ED%94%84%EB%A1%9C%EA%B7%B8%EB%9E%98%EB%A8%B8%EB%B8%94_%EB%85%BC%EB%A6%AC_%EC%86%8C%EC%9E%90" title="프로그래머블 논리 소자">프로그래머블 논리 소자</a>의 선두 제조업체이다. 알테라는 기술<a class="mw-redirect" href="/wiki/%EC%A3%BC%EC%8B%9D%EC%8B%9C%EC%9E%A5" title="주식시장">주</a>중에서 <a class="new" href="/w/index.php?title=%EB%82%98%EC%8A%A4%EB%8B%A5-100&amp;action=edit&amp;redlink=1" title="나스닥-100 (없는 문서)">나스닥-100</a>에 속하고 <a href="/wiki/S%26P_500" title="S&amp;P 500">S&amp;P 500</a>에 포함되는 회사이다.</p>
<p>알테라는 프로그래머블 논리 소자를 발표한 초창기 선두업체 <a class="new" href="/w/index.php?title=%EC%8B%9C%EA%B7%B8%EB%84%A4%ED%8B%B1&amp;action=edit&amp;redlink=1" title="시그네틱 (없는 문서)">시그네틱</a>과 <a href="/wiki/%EB%AA%A8%EB%86%80%EB%A6%AC%EC%8B%9D_%EB%A9%94%EB%AA%A8%EB%A6%AC" title="모놀리식 메모리">모놀리식 메모리</a> 다음으로, 프로그래머블 논리 소자의 선두업체중 하나이다. 알테라가 개발한 다양한 기능들은 프로그래머블 칩기반 시스템(SOPC)을 가능하게 했다. 최근에는 임베디드 메모리, 임베디드 프로세서, 고속 트랜시버를 포함하고 있다. 130나노미터와 90나노미터 제품발표의 성공은 연구의 좋은 성과이다. 알테라의 <a class="new" href="/w/index.php?title=%EB%8B%88%EC%98%A4%EC%8A%A4_II&amp;action=edit&amp;redlink=1" title="니오스 II (없는 문서)">니오스 II</a>와 <a class="new" href="/w/index.php?title=%EB%8B%88%EC%98%A4%EC%8A%A4_%EC%9E%84%EB%B2%A0%EB%94%94%EB%93%9C_%ED%94%84%EB%A1%9C%EC%84%B8%EC%84%9C&amp;action=edit&amp;redlink=1" title="니오스 임베디드 프로세서 (없는 문서)">니오스</a> 소프트 코어 프로세서와 하드카피 II &amp; 하드카피 소자는 알테라의 시장점유율을 확장시켰고, 세계의 임베디드 프로세서와 각각의 구조화된 <a href="/wiki/%EC%A3%BC%EB%AC%B8%ED%98%95_%EB%B0%98%EB%8F%84%EC%B2%B4" title="주문형 반도체">주문형 반도체</a>를 알테라로 대체하였다. 주요한 경쟁사는 <a class="mw-redirect" href="/wiki/%EC%9E%90%EC%9D%B4%EB%A7%81%EC%8A%A4" title="자이링스">자이링스</a>, <a href="/wiki/%EB%9E%98%ED%8B%B0%EC%8A%A4_%EC%84%B8%EB%AF%B8%EC%BB%A8%EB%8D%95%ED%84%B0" title="래티스 세미컨덕터">래티스 세미컨덕터</a>, <a class="new" href="/w/index.php?title=%EC%95%A1%ED%85%94&amp;action=edit&amp;redlink=1" title="액텔 (없는 문서)">액텔</a>, <a class="new" href="/w/index.php?title=%ED%80%B5%EB%A1%9C%EC%A7%81&amp;action=edit&amp;redlink=1" title="퀵로직 (없는 문서)">퀵로직</a>과 <a href="/wiki/%EC%95%84%ED%8A%B8%EB%A9%9C" title="아트멜">아트멜</a>이다.</p>
<p>알테라는 논리 회로의 설계와 시뮬레이션 기능을 지닌, 소프트웨어군 <a href="/wiki/%EC%BF%BC%ED%84%B0%EC%8A%A4_II" title="쿼터스 II">쿼터스 II</a>를 제공한다. 비록 그들의 소프트웨어군이 주요언어인 <a href="/wiki/VHDL" title="VHDL">VHDL</a>과 <a href="/wiki/%EB%B2%A0%EB%A6%B4%EB%A1%9C%EA%B7%B8" title="베릴로그">베릴로그</a>를 광범위하게 지원하지만, 알테라는 <a class="mw-redirect" href="/wiki/%EC%95%8C%ED%85%8C%EB%9D%BC_%ED%95%98%EB%93%9C%EC%9B%A8%EC%96%B4_%EA%B8%B0%EC%88%A0_%EC%96%B8%EC%96%B4" title="알테라 하드웨어 기술 언어">알테라 하드웨어 기술 언어</a>로 알려진 <a href="/wiki/%ED%95%98%EB%93%9C%EC%9B%A8%EC%96%B4_%EA%B8%B0%EC%88%A0_%EC%96%B8%EC%96%B4" title="하드웨어 기술 언어">하드웨어 기술 언어</a>의 개발자이다.</p>
<p>2015년 6월 1일, 인텔이 알테라를 주당 54달러, 167억 달러(약 18조 6천억원)에 인수했다고 공식 발표했다.</p>

<h2><span class="mw-headline" id=".EA.B2.BD.EC.9F.81.EC.82.AC">경쟁사</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%95%8C%ED%85%8C%EB%9D%BC&amp;action=edit&amp;section=9" title="부분 편집: 경쟁사">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<ul>
<li>알테라의 주 경쟁사는 <a class="mw-redirect" href="/wiki/%EC%9E%90%EC%9D%B4%EB%A7%81%EC%8A%A4" title="자이링스">자이링스</a>와 <a href="/wiki/%EB%9E%98%ED%8B%B0%EC%8A%A4_%EC%84%B8%EB%AF%B8%EC%BB%A8%EB%8D%95%ED%84%B0" title="래티스 세미컨덕터">래티스 세미컨덕터</a>이다.</li>
<li><a href="/wiki/FPGA" title="FPGA">FPGA</a> 부분에서 주요한 경쟁사는 <a class="new" href="/w/index.php?title=%EC%95%A1%ED%85%94&amp;action=edit&amp;redlink=1" title="액텔 (없는 문서)">액텔</a>과 <a class="new" href="/w/index.php?title=%ED%80%B5%EB%A1%9C%EC%A7%81&amp;action=edit&amp;redlink=1" title="퀵로직 (없는 문서)">퀵로직</a>이다.</li>
</ul>
<p>CPLD와 전통적인 FPGA는 엄밀히 논리소자인 반면에, 큰 FPGA 회사의 새로운 소자는 고성능 <a href="/wiki/%EB%94%94%EC%A7%80%ED%84%B8_%EC%8B%A0%ED%98%B8_%EC%B2%98%EB%A6%AC" title="디지털 신호 처리">디지털 신호 처리</a> 응용에 근접하고 있다. 이러한 경향은 <a class="new" href="/w/index.php?title=%EC%A0%84%EC%9E%90_%EC%8B%9C%EC%8A%A4%ED%85%9C_%EB%8B%A8%EA%B3%84&amp;action=edit&amp;redlink=1" title="전자 시스템 단계 (없는 문서)">전자 시스템 단계</a> 설계라고 부른다. 이것은 완전히 새로운것은 아니지만, 지난 10년동안의 마케팅과 광고뿐만 아니라, 산업에서 실 사용자층에 의한 힘이다. 오늘날에 수백만 게이트 소자와 <a class="new" href="/w/index.php?title=%EC%B9%A9_%EC%8B%9C%EC%8A%A4%ED%85%9C&amp;action=edit&amp;redlink=1" title="칩 시스템 (없는 문서)">칩 시스템</a> 구조와, IP를 조합하는 고수준 합성 도구는 복잡한 시스템을 완벽하게 구현할 수 있다. 그래서 미래에 알테라와 자이링스같은 벤더는 <a href="/wiki/%ED%85%8D%EC%82%AC%EC%8A%A4_%EC%9D%B8%EC%8A%A4%ED%8A%B8%EB%A3%A8%EB%A8%BC%ED%8A%B8" title="텍사스 인스트루먼트">텍사스 인스트루먼트</a>나 <a href="/wiki/%EC%95%84%EB%82%A0%EB%A1%9C%EA%B7%B8_%EB%94%94%EB%B0%94%EC%9D%B4%EC%8A%A4" title="아날로그 디바이스">아날로그 디바이스</a>같은 회사처럼 성장할 것이다.</p>
<h2><span class="mw-headline" id=".EC.8B.A4.EC.A0.81">실적</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%95%8C%ED%85%8C%EB%9D%BC&amp;action=edit&amp;section=10" title="부분 편집: 실적">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>2006년 12월 29일 실적 발표 결과, 2006년 매출은 12.9억 달러, 영업이익은 3억 108만 달러를 기록하였다. <sup class="reference" id="cite_ref-1"><a href="#cite_note-1">[1]</a></sup> 2007년 3사분기에는, 2007년 2사분기보다 1% 감소한, (사분기) 매출 3억 1580만 달러를 기록하였다.</p>
