<p><b>알폰소 12세</b>(Alfonso XII, <a href="/wiki/1857%EB%85%84" title="1857년">1857년</a> <a href="/wiki/11%EC%9B%94_28%EC%9D%BC" title="11월 28일">11월 28일</a> ~ <a href="/wiki/1885%EB%85%84" title="1885년">1885년</a> <a href="/wiki/11%EC%9B%94_25%EC%9D%BC" title="11월 25일">11월 25일</a>)는 스페인의 국왕이자 1차 스페인 공화국을 통치했던 수반이다. 왕정 복고가 이뤄지면서 등극한 첫 왕이다. 본명은 알폰소 프란치스코 페르난도 피오 후안 데 마리아 델라 콘쳅시온 그레고리오 펠라요 데 보르본 이 보르본(Alfonso Francisco Fernando Pío Juan de María de la Concepción Gregorio Pelayo de Borbón y Borbón)이다.</p>
<p><a href="/wiki/%EC%9D%B4%EC%82%AC%EB%B2%A8_2%EC%84%B8" title="이사벨 2세">이사벨 2세</a>의 아들이었으며 전해진 바에 따르면 아버지는 <a class="mw-redirect" href="/wiki/%EC%B9%B4%EB%94%94%EC%A6%88_%EA%B3%B5%EC%9E%91_%ED%94%84%EB%9E%80%EC%8B%9C%EC%8A%A4%EC%BD%94" title="카디즈 공작 프란시스코">카디즈 공작 프란시스코</a>이다. 하지만 부계 혈통은 오랫동안 논란의 여지로 남아 있었으며 일부에서는 혼외 태생을 주장하는 입장도 있다. 가장 유력한 부계 혈통으로는 왕실 군대를 지휘하던 엔리케 푸이그몰토나 토레의 공작이던 프란시스코 세라노이다.</p>

<h2><span class="mw-headline" id=".EC.83.9D.EC.95.A0">생애</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%95%8C%ED%8F%B0%EC%86%8C_12%EC%84%B8&amp;action=edit&amp;section=1" title="부분 편집: 생애">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<h3><span class="mw-headline" id=".EC.B6.94.EB.B0.A9">추방</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%95%8C%ED%8F%B0%EC%86%8C_12%EC%84%B8&amp;action=edit&amp;section=2" title="부분 편집: 추방">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>이사벨 여왕은 <a href="/wiki/1868%EB%85%84" title="1868년">1868년</a> 혁명으로 인해 강제 출국을 감행하게 되면서 당시 어렸던 알폰소 12세와 함께 <a class="mw-redirect" href="/wiki/%ED%8C%8C%EB%A6%AC_%EC%8B%9C" title="파리 시">파리</a>로 피신했다. 그곳에 머물면서 다시 어린 왕자는 <a href="/wiki/%EB%B9%88" title="빈">빈</a>으로 가 공부하게 됐다. <a href="/wiki/1870%EB%85%84" title="1870년">1870년</a> <a href="/wiki/6%EC%9B%94_25%EC%9D%BC" title="6월 25일">6월 25일</a> 부유한 스페인 귀족들이 여왕에게 찾아와 계속된 복고를 요청하게 되면서 왕자는 다시 여왕의 곁으로 돌아와 알폰소 12세라는 왕호를 갖게 됐다.</p>
<p>정식으로 스페인에 돌아오지는 못했으므로 그는 <a href="/wiki/%EC%98%81%EA%B5%AD" title="영국">영국</a>으로 떠나 왕실 군학교에서 군 관련 공부를 시작했다. <a href="/wiki/1874%EB%85%84" title="1874년">1874년</a> <a href="/wiki/12%EC%9B%94_1%EC%9D%BC" title="12월 1일">12월 1일</a> 그는 자신을 따르던 추종자들에 대한 확답으로 자신이 유일한 스페인 왕국의 혈통이며 적법자임을 발표했다. 그해 말 시민 전쟁으로 왕명을 받들던 군이 승리하면서 그는 스페인으로 돌아올 채비를 했다.</p>
<h3><span class="mw-headline" id=".EA.B7.80.EA.B5.AD">귀국</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%95%8C%ED%8F%B0%EC%86%8C_12%EC%84%B8&amp;action=edit&amp;section=3" title="부분 편집: 귀국">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>새로운 왕 알폰소 12세는 <a href="/wiki/%EB%A7%88%EB%93%9C%EB%A6%AC%EB%93%9C" title="마드리드">마드리드</a>에 <a href="/wiki/%EB%B0%94%EB%A5%B4%EC%85%80%EB%A1%9C%EB%82%98" title="바르셀로나">바르셀로나</a>, <a href="/wiki/%EB%B0%9C%EB%A0%8C%EC%8B%9C%EC%95%84" title="발렌시아">발렌시아</a>를 거쳐 입성했다.(1875년) 이듬해 왕과 대치하던 카를로스 주의자와의 분쟁으로 기수에 서 있던 돈 카를로스 공작을 죽이고 왕권을 다졌다.</p>
<p><a href="/wiki/1878%EB%85%84" title="1878년">1878년</a> <a href="/wiki/1%EC%9B%94_23%EC%9D%BC" title="1월 23일">1월 23일</a> 그는 사촌지간이던 안토니 공작의 딸 마리아 메르세데스 공주와 결혼했지만 그녀는 6개월 만에 세상을 떠났다. 그해 말에는 타라코나 출신의 한 젊은 노동자가 왕을 저격하려다 저지당했다.</p>
<h2><span class="mw-headline" id=".EC.9E.AC.ED.98.BC.EA.B3.BC_.EC.9E.90.EB.85.80">재혼과 자녀</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%95%8C%ED%8F%B0%EC%86%8C_12%EC%84%B8&amp;action=edit&amp;section=4" title="부분 편집: 재혼과 자녀">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p><a href="/wiki/1879%EB%85%84" title="1879년">1879년</a> <a href="/wiki/11%EC%9B%94_29%EC%9D%BC" title="11월 29일">11월 29일</a> 알폰소는 상당히 먼 친척 관계던 <a href="/wiki/%EC%98%A4%EC%8A%A4%ED%8A%B8%EB%A6%AC%EC%95%84_%EB%8C%80%EA%B3%B5_%EC%B9%B4%EB%A5%BC_%ED%8E%98%EB%A5%B4%EB%94%94%EB%82%9C%ED%8A%B8" title="오스트리아 대공 카를 페르디난트">오스트리아 대공 카를 페르디난트</a>의 딸 <a href="/wiki/%EC%98%A4%EC%8A%A4%ED%8A%B8%EB%A6%AC%EC%95%84%EC%9D%98_%EB%A7%88%EB%A6%AC%EC%95%84_%ED%81%AC%EB%A6%AC%EC%8A%A4%ED%8B%B0%EB%82%98" title="오스트리아의 마리아 크리스티나">마리아 크리스티나</a>와 결혼했다. 신혼여행에서 돌아오는 찰나에도 한 요리사가 왕 부부를 사살하려 시도하였다.</p>
<p>두 사람 사이에서 태어난 자녀는 다음과 같다.</p>
<ul>
<li>아스투리아스의 공주, 메르세데스(<a href="/wiki/1880%EB%85%84" title="1880년">1880년</a> <a href="/wiki/9%EC%9B%94_11%EC%9D%BC" title="9월 11일">9월 11일</a> ~ <a href="/wiki/1904%EB%85%84" title="1904년">1904년</a> <a href="/wiki/10%EC%9B%94_17%EC%9D%BC" title="10월 17일">10월 17일</a>) : 부르봉 왕가의 샤를 왕자와 결혼</li>
<li>스페인의 마리아 테레사 공주(<a href="/wiki/1882%EB%85%84" title="1882년">1882년</a> <a href="/wiki/11%EC%9B%94_12%EC%9D%BC" title="11월 12일">11월 12일</a> ~ <a href="/wiki/1912%EB%85%84" title="1912년">1912년</a> <a href="/wiki/9%EC%9B%94_23%EC%9D%BC" title="9월 23일">9월 23일</a>) : 바이에른의 왕자 페르디난드와 결혼</li>
<li><a href="/wiki/%EC%95%8C%ED%8F%B0%EC%86%8C_13%EC%84%B8" title="알폰소 13세">알폰소 13세</a>(<a href="/wiki/1886%EB%85%84" title="1886년">1886년</a> <a href="/wiki/5%EC%9B%94_17%EC%9D%BC" title="5월 17일">5월 17일</a> - <a href="/wiki/1941%EB%85%84" title="1941년">1941년</a> <a href="/wiki/2%EC%9B%94_28%EC%9D%BC" title="2월 28일">2월 28일</a>) : 스페인 국왕(<a href="/wiki/%EB%B0%94%ED%85%90%EB%B2%A0%EB%A5%B4%ED%81%AC%EC%9D%98_%EB%B9%85%ED%86%A0%EB%A6%AC%EC%95%84_%EC%97%90%EC%9A%B0%ED%97%A4%EB%8B%88%EC%95%84" title="바텐베르크의 빅토리아 에우헤니아">바텐베르크의 빅토리아 에우헤니아</a>와 결혼)</li>
</ul>
<h2><span class="mw-headline" id=".EC.A3.BD.EC.9D.8C">죽음</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%95%8C%ED%8F%B0%EC%86%8C_12%EC%84%B8&amp;action=edit&amp;section=5" title="부분 편집: 죽음">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<div class="thumb tleft">
<div class="thumbinner" style="width:302px;"><a class="image" href="/wiki/%ED%8C%8C%EC%9D%BC:Alfonso_XII_of_Spain_Mausoleum.jpg"><img alt="" class="thumbimage" data-file-height="2000" data-file-width="4741" height="127" src="//upload.wikimedia.org/wikipedia/commons/thumb/0/0f/Alfonso_XII_of_Spain_Mausoleum.jpg/300px-Alfonso_XII_of_Spain_Mausoleum.jpg" srcset="//upload.wikimedia.org/wikipedia/commons/thumb/0/0f/Alfonso_XII_of_Spain_Mausoleum.jpg/450px-Alfonso_XII_of_Spain_Mausoleum.jpg 1.5x, //upload.wikimedia.org/wikipedia/commons/thumb/0/0f/Alfonso_XII_of_Spain_Mausoleum.jpg/600px-Alfonso_XII_of_Spain_Mausoleum.jpg 2x" width="300"/></a>
<div class="thumbcaption">
<div class="magnify"><a class="internal" href="/wiki/%ED%8C%8C%EC%9D%BC:Alfonso_XII_of_Spain_Mausoleum.jpg" title="실제 크기로"></a></div>
알폰소 12세를 기리는 기념비(마드리드)</div>
</div>
</div>
<p>1885년 28세의 젊은 나이에 그는 <a href="/wiki/%EA%B2%B0%ED%95%B5" title="결핵">결핵</a>으로 생을 마감했다. 아주 어린 나이에 재위했음에도 그는 통치에 있어 노련함을 잃지 않았으며 사람을 판단하는 능력이 뛰어났기에 추방의 상황에도 다시 돌아와 고국의 왕위를 이어나갈 수 있었다. 또한 사망한 해까지 국민들을 직접 찾아다니며 기근과 질병에 힘겨워하던 백성들에 다가갔다. 당시에는 스페인 전역에 <a href="/wiki/%EC%BD%9C%EB%A0%88%EB%9D%BC" title="콜레라">콜레라</a>가 유행한데다 <a href="/wiki/%EC%A7%80%EC%A7%84" title="지진">지진</a>으로 모든 농토와 소규모 촌락이 황폐화된 상태였다.</p>
<p>그는 통치에 있어 어느 쪽의 의견에도 치우치지 않았으며 이용되지 않기 위해 자신의 권력을 철저히 지킨 군주였다. 짧은 통치 기간 동안 말년의 자연재해를 빼고서는 모든 상황이 안정돼 국내는 물론 대외 경제로 인한 부가 축적됐으며 왕실 재정이 풍족해졌다. 미국 스페인 전쟁을 겪으면서도 스페인이 모든 혁명세력에 초토화되지 않은 이유는 그가 생전에 재구축했던 행정 및 군사제도 때문이라고 여겨진다.</p>
