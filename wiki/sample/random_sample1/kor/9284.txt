<p><b>알프레도 스트로에스네르 마티아우다</b>(<span style="font-size: smaller;"><a href="/wiki/%EC%8A%A4%ED%8E%98%EC%9D%B8%EC%96%B4" title="스페인어">스페인어</a>:</span> <span lang="es" xml:lang="es">Alfredo Stroessner Matiauda</span> <small>알프레도 스트로에스네르 마티아우다<sup>[<a href="/wiki/%EC%9C%84%ED%82%A4%EB%B0%B1%EA%B3%BC:%EC%8A%A4%ED%8E%98%EC%9D%B8%EC%96%B4%EC%9D%98_%ED%95%9C%EA%B8%80_%ED%91%9C%EA%B8%B0" title="위키백과:스페인어의 한글 표기">*</a>]</sup></small><a href="/wiki/1912%EB%85%84" title="1912년">1912년</a> <a href="/wiki/11%EC%9B%94_3%EC%9D%BC" title="11월 3일">11월 3일</a> ~ <a href="/wiki/2006%EB%85%84" title="2006년">2006년</a> <a href="/wiki/8%EC%9B%94_16%EC%9D%BC" title="8월 16일">8월 16일</a>)는 <a href="/wiki/%ED%8C%8C%EB%9D%BC%EA%B3%BC%EC%9D%B4" title="파라과이">파라과이</a>의 정치인이다.</p>
<h2><span class="mw-headline" id=".EC.9D.BC.EC.83.9D">일생</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%95%8C%ED%94%84%EB%A0%88%EB%8F%84_%EC%8A%A4%ED%8A%B8%EB%A1%9C%EC%97%90%EC%8A%A4%EB%84%A4%EB%A5%B4&amp;action=edit&amp;section=1" title="부분 편집: 일생">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p><a href="/wiki/%EB%8F%85%EC%9D%BC" title="독일">독일</a> 이민자의 아들로 태어났으며(그의 집안 성씨는 원래 Strößner 또는 Strössner로 표기했었다), <a href="/wiki/%EC%9C%A1%EA%B5%B0" title="육군">육군</a>에 들어가 여러 전쟁에서 공을 세워 <a href="/wiki/1951%EB%85%84" title="1951년">1951년</a> 육군총사령관이 되었다. <a href="/wiki/1954%EB%85%84" title="1954년">1954년</a> 군부 <a href="/wiki/%EC%BF%A0%EB%8D%B0%ED%83%80" title="쿠데타">쿠데타</a>를 일으켜 <a class="new" href="/w/index.php?title=%ED%8E%98%EB%8D%B0%EB%A6%AC%EC%BD%94_%EC%B0%A8%EB%B2%A0%EC%8A%A4&amp;action=edit&amp;redlink=1" title="페데리코 차베스 (없는 문서)">페데리코 차베스</a> <a class="new" href="/w/index.php?title=%ED%8C%8C%EB%9D%BC%EA%B3%BC%EC%9D%B4%EC%9D%98_%EB%8C%80%ED%86%B5%EB%A0%B9&amp;action=edit&amp;redlink=1" title="파라과이의 대통령 (없는 문서)">대통령</a>을 축출하고 <a class="new" href="/w/index.php?title=%EC%BD%9C%EB%A1%9C%EB%9D%BC%EB%8F%84_%EB%8B%B9_(%ED%8C%8C%EB%9D%BC%EA%B3%BC%EC%9D%B4)&amp;action=edit&amp;redlink=1" title="콜로라도 당 (파라과이) (없는 문서)">콜로라도 당</a>의 대통령 후보로 단독 출마하여 당선되었다.</p>
<p>이후 빈민층과 농민을 회유하면서 동시에 부유층의 이익을 옹호하였으며, 경제개발과 반공정책을 추진하였다. 집권 초기에는 경제개발도 어느 정도 성과를 거두고, 물가도 안정되어 많은 지지를 받았다. <a href="/wiki/%EB%B8%8C%EB%9D%BC%EC%A7%88" title="브라질">브라질</a> 국경에 있는 대규모 댐 건설 사업에 참여하여 전력난을 해소하는 성과를 오렸으며, 자신의 이름을 딴 <a class="mw-redirect" href="/wiki/%ED%91%B8%EC%97%90%EB%A5%B4%ED%86%A0%ED%94%84%EB%A0%88%EC%8B%9C%EB%8D%B4%ED%85%8C%EC%8A%A4%ED%8A%B8%EB%A1%9C%EC%97%90%EC%8A%A4%EB%84%A4%EB%A5%B4" title="푸에르토프레시덴테스트로에스네르">푸에르토프레시덴테스트로에스네르</a>라는 도시까지 건설하였다.</p>
<p>그러나 장기집권이 계속되면서 반대자에 대한 가혹한 인권탄압과 권력기반 유지를 위한 막대한 비용 지출로 국제적인 비난을 받게 되었다. 헌법을 개정하여 장기집권을 실현하여 연속으로 여러 차례 대통령 선거에서 당선되었으며, <a href="/wiki/1988%EB%85%84" title="1988년">1988년</a> 8번째로 당선되었다.</p>
<p>그러나 <a href="/wiki/1989%EB%85%84" title="1989년">1989년</a> 군부 쿠데타로 쫓겨났으며, <a href="/wiki/%EB%B8%8C%EB%9D%BC%EC%A7%88" title="브라질">브라질</a>로 망명하여 그 곳에서 여생을 보냈다. 파라과이를 무려 35년간이나 철권 통치한 그는 외국 언론에 의해 세계 10대 독재자로 선정되기도 하였다. 망명 후 법정에서 살인죄로 기소되었으나, 파라과이로의 소환은 실현되지 못한 채 <a href="/wiki/2006%EB%85%84" title="2006년">2006년</a> 브라질에서 사망하였다.<sup class="reference" id="cite_ref-1"><a href="#cite_note-1">[1]</a></sup> 사망 후 파라과이에서는 그의 유해 송환을 거부하여 그는 브라질의 공동묘지에 묻혔다.</p>
