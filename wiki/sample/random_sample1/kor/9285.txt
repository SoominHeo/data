<p><b>알프레드 자리</b>(Alfred Jarry, <a href="/wiki/1873%EB%85%84" title="1873년">1873년</a> <a href="/wiki/9%EC%9B%94_8%EC%9D%BC" title="9월 8일">9월 8일</a> ~ <a href="/wiki/1907%EB%85%84" title="1907년">1907년</a> <a href="/wiki/11%EC%9B%94_1%EC%9D%BC" title="11월 1일">11월 1일</a>)은 <a href="/wiki/%ED%94%84%EB%9E%91%EC%8A%A4" title="프랑스">프랑스</a>의 <a href="/wiki/%EC%8B%9C%EC%9D%B8" title="시인">시인</a>·<a href="/wiki/%EA%B7%B9%EC%9E%91%EA%B0%80" title="극작가">극작가</a>다.</p>
<p><a href="/wiki/%EB%A7%88%EC%98%8C_%EC%A3%BC" title="마옌 주">마옌 주</a> <a href="/wiki/%EB%9D%BC%EB%B0%9C_(%EB%A7%88%EC%98%8C_%EC%A3%BC)" title="라발 (마옌 주)">라발</a>에서 태어났다. <a href="/wiki/%EB%A0%8C_(%EB%8F%84%EC%8B%9C)" title="렌 (도시)">렌</a>의 관립 중학교에서 공부하는 동안에 소극(笑劇) &lt;폴란드인&gt;을 써서 물리교사를 희롱했다. <a href="/wiki/1891%EB%85%84" title="1891년">1891년</a> <a href="/wiki/%ED%8C%8C%EB%A6%AC_(%ED%94%84%EB%9E%91%EC%8A%A4)" title="파리 (프랑스)">파리</a>로 상경하여 <a class="new" href="/w/index.php?title=%EB%A7%88%EB%A5%B4%EC%85%80_%EC%8A%88%EC%9B%8C%EB%B8%8C&amp;action=edit&amp;redlink=1" title="마르셀 슈워브 (없는 문서)">마르셀 슈워브</a>(Marcel Schwob, 1867~1905)나 <a href="/wiki/%EB%A0%88%EB%AF%B8_%EB%93%9C_%EA%B5%AC%EB%A5%B4%EB%AA%BD" title="레미 드 구르몽">레미 드 구르몽</a>과 교제하며 시작(詩作)에 전념하는 한편, <a class="mw-redirect" href="/wiki/%EB%A4%BC%EB%84%A4_%ED%8F%AC" title="뤼네 포">뤼네 포</a>와도 알게 되어 &lt;폴란드인&gt;을 개작, &lt;유뷔왕&gt;이란 이름으로 <a href="/wiki/1896%EB%85%84" title="1896년">1896년</a> 제작극장(制作劇場)에서 상연하였다. 야비한 언사와 대담한 연출, 강렬한 풍자 때문에 이 상연은 스캔들이 되었으나, 반역정신의 덩어리같은 자리는 &lt;쇠사슬에 묶인 유뷔&gt;를 비롯한 일련의 유뷔물로 보수파와 맞섰다.</p>
