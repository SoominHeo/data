<h1 class="firstHeading" id="firstHeading" lang="ko">알프레트 폰 슐리펜</h1>
<div class="mw-body-content" id="bodyContent">

<div class="mw-content-ltr" dir="ltr" id="mw-content-text" lang="ko"><script>function mfTempOpenSection(id){var block=document.getElementById("mf-section-"+id);block.className+=" open-block";block.previousSibling.className+=" open-block";}</script><p><b>알프레트 폰 슐리펜 백작</b>(<span lang="De" xml:lang="De">Alfred Graf von Schlieffen</span>, <a href="/wiki/1833%EB%85%84" title="1833년">1833년</a> <a href="/wiki/2%EC%9B%94_28%EC%9D%BC" title="2월 28일">2월 28일</a> <a href="/wiki/%EB%B2%A0%EB%A5%BC%EB%A6%B0" title="베를린">베를린</a>에서 출생 ~ <a href="/wiki/1913%EB%85%84" title="1913년">1913년</a> <a href="/wiki/1%EC%9B%94_4%EC%9D%BC" title="1월 4일">1월 4일</a> <a class="new" href="/w/index.php?title=%EC%97%90%EB%B2%A4%EB%8B%A4&amp;action=edit&amp;redlink=1" title="에벤다 (없는 문서)">에벤다</a>에서 사망)은 <a href="/wiki/%EB%8F%85%EC%9D%BC_%EC%A0%9C%EA%B5%AD" title="독일 제국">독일 제국</a>의 <a class="new" href="/w/index.php?title=%EC%9C%A1%EA%B5%B0_%EC%9B%90%EC%88%98&amp;action=edit&amp;redlink=1" title="육군 원수 (없는 문서)">육군 원수</a>였다. <a class="mw-redirect" href="/wiki/%EC%A0%9C1%EC%B0%A8_%EC%84%B8%EA%B3%84%EB%8C%80%EC%A0%84" title="제1차 세계대전">제1차 세계대전</a> 초반에 독일군의 작전 계획인 "<a class="mw-redirect" href="/wiki/%EC%8A%90%EB%A6%AC%ED%8E%9C_%EC%9E%91%EC%A0%84" title="슐리펜 작전">슐리펜 작전</a>"의 고안자로 알려져 있다.</p>
<h2><span class="mw-headline" id=".EC.83.9D.EC.95.A0">생애</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%95%8C%ED%94%84%EB%A0%88%ED%8A%B8_%ED%8F%B0_%EC%8A%90%EB%A6%AC%ED%8E%9C&amp;action=edit&amp;section=1" title="부분 편집: 생애">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>1833년 2월에 베를린에서, 프로이센 육군 소장의 아들로서 태어났다. 1854년에 군에 입대했고, 1863년에 <a class="new" href="/w/index.php?title=%EC%B0%B8%EB%AA%A8%EB%B3%B8%EB%B6%80&amp;action=edit&amp;redlink=1" title="참모본부 (없는 문서)">참모본부</a>에서 근무한다. 1866년 <a class="mw-redirect" href="/wiki/%EB%B3%B4%EC%98%A4%EC%A0%84%EC%9F%81" title="보오전쟁">보오전쟁</a> 때에는 대위, 1870년~1871년의 <a class="mw-redirect" href="/wiki/%EB%B3%B4%EB%B6%88%EC%A0%84%EC%9F%81" title="보불전쟁">보불전쟁</a>에 참모 장교(<a href="/wiki/%EC%86%8C%EB%A0%B9" title="소령">소령</a>)으로서 참전했다. 그 후 근위 우란연대장이나 참모본부 근무를 맡는다. 1884년에 참모본부 국장에 취임했다. 1888년 참모본부 차장, 1891년에 <a href="/wiki/%EC%95%8C%ED%94%84%EB%A0%88%ED%8A%B8_%ED%8F%B0_%EB%B0%9C%EB%8D%94%EC%A0%9C" title="알프레트 폰 발더제">알프레트 폰 발더제</a>의 뒤를 이어서 참모총장이 되었다. 1903년에 <a class="new" href="/w/index.php?title=%EC%83%81%EA%B8%89%EB%8C%80%EC%9E%A5&amp;action=edit&amp;redlink=1" title="상급대장 (없는 문서)">상급대장</a>으로 승진했다.</p>
<p>1905년에 가상적국 <a href="/wiki/%EB%9F%AC%EC%8B%9C%EC%95%84_%EC%A0%9C%EA%B5%AD" title="러시아 제국">러시아 제국</a>과 <a href="/wiki/%ED%94%84%EB%9E%91%EC%8A%A4" title="프랑스">프랑스</a>에 대한 작전 계획인 "<a class="mw-redirect" href="/wiki/%EC%8A%90%EB%A6%AC%ED%8E%9C_%EC%9E%91%EC%A0%84" title="슐리펜 작전">슐리펜 작전</a>"을 고안했다. 이 계획은 <a class="mw-redirect" href="/wiki/%EB%8C%80_%EB%AA%B0%ED%8A%B8%EC%BC%80" title="대 몰트케">대 몰트케</a>나 발더제의 기본 계획을 구체화한 것이었다. 러시아와 프랑스 양국과의 이정면 전쟁을 피하기 위해, 개전 후 전력을 다해서 단기간 프랑스 공략을 노렸고, 이어서 철도 수송을 구사해서 부대를 동쪽으로 수송해서 남은 적 러시아를 공격한다는 계획이었다. 이 계획을 실현하기 위해, 슐리펜 이후의 독일군은 이동가능한 <a class="new" href="/w/index.php?title=%EC%A4%91%ED%8F%AC&amp;action=edit&amp;redlink=1" title="중포 (없는 문서)">중포</a>의 배치나, 수송부대를 중심으로 하는 병참의 충실에 힘을 쏟았다. 이 계획은 제1차 세계대전 초기에 변경된 형태로 실행되지만, 슐리펜의 계획과는 달리 독일군의 진격은 <a class="mw-redirect" href="/wiki/1%EC%B0%A8_%EB%A7%88%EB%A5%B8_%EC%A0%84%ED%88%AC" title="1차 마른 전투">마른 회전</a>에서 좌절되었고, 이후는 슐리펜이 상정하지 않았던 <a href="/wiki/%EC%B0%B8%ED%98%B8%EC%A0%84" title="참호전">참호전</a>·<a href="/wiki/%EC%B4%9D%EB%A0%A5%EC%A0%84" title="총력전">총력전</a>으로 옮겨지게 된다. 슐리펜은 군사령관이라기보다도 작전 이론가의 성격이 강했다.</p>
<p>슐리펜 작전 발표 다음해인 1906년에 예비역으로 편입되었다. 그 후에도 개인적으로 슐리펜 작전의 개정을 계속했고, 1911년에 <a class="mw-redirect" href="/wiki/%EC%9B%90%EC%88%98_(%EA%B5%B0%EC%9D%B8_%EA%B3%84%EA%B8%89)" title="원수 (군인 계급)">원수</a>로 진급했지만, 순수하게 명예적인 조치였다. 제1차 세계대전 발발하기 전인, 1913년에 <a href="/wiki/%EB%B2%A0%EB%A5%BC%EB%A6%B0" title="베를린">베를린</a>에서 사망했고, 군인 묘지에 매장되었다. 묘소에는 <a href="/wiki/%ED%99%A9%EC%A0%9C" title="황제">황제</a> <a href="/wiki/%EB%B9%8C%ED%97%AC%EB%A6%84_2%EC%84%B8" title="빌헬름 2세">빌헬름 2세</a>으로부터 헌화가 있었다.</p>
<p>슐리펜의 유언으로서, 슐리펜 작전과 관련된 "나에게 강한 우익을!(Macht mir den rechten Flügel stark!)"말이 널리 알려져 있다. 슐리펜이 프랑스 공격 계획에서 <a href="/wiki/%EB%B2%A8%EA%B8%B0%EC%97%90" title="벨기에">벨기에</a>를 통과하는 독일군 우익을 가장 중시하여 항시 이 말을 했던 것은 사실이지만, 주치의인 로프스 <a href="/wiki/%EA%B5%B0%EC%9D%98%EA%B4%80" title="군의관">군의관</a>의 회고록에 따르면, 병석에 있는 슐리펜은 <a href="/wiki/%EA%B5%B0%EC%82%AC" title="군사">군사</a>나 <a href="/wiki/%EC%97%AD%EC%82%AC" title="역사">역사</a>, <a href="/wiki/%EC%A0%95%EC%B9%98" title="정치">정치</a>, 가족 등을 갈피를 잡지 못하고 말하는 상태였고, 실제로 "최후의 말"이라는 것은, 자신의 병상을 냉정하게 분석한 "작은 원인이 큰 결과를 부른다(Kleine Ursachen, große Wirkungen)"이라는 것이었다고 한다.</p>
