<p><a href="/wiki/%EB%A1%9C%EB%A7%88_%EC%A0%9C%EA%B5%AD" title="로마 제국">로마 제국</a> 시대에 지금의 알헤시라스에는 포르투스 알부스(Portus Albus), 카에타리아(Caetaria), 율리아 트라두크타(Iulia Traducta)라는 도시가 있었다. 그 후 <a href="/wiki/711%EB%85%84" title="711년">711년</a>에 스페인을 침공한 <a href="/wiki/%EB%AC%B4%EC%96%B4%EC%9D%B8" title="무어인">무어인들</a>이 알헤시라스를 세웠고 알헤시라스는 "초록빛 섬"을 뜻하는 <a href="/wiki/%EC%95%84%EB%9E%8D%EC%96%B4" title="아랍어">아랍어</a> 단어인 "알자지라 알카드라"(<span style="font-size: smaller;"><a href="/wiki/%EC%95%84%EB%9E%8D%EC%96%B4" title="아랍어">아랍어</a>:</span> <span lang="ar" xml:lang="ar">الجزيرة الخضراء, al-Jazirah al-Khadra'</span>)에서 유래되었다. <a href="/wiki/1035%EB%85%84" title="1035년">1035년</a>부터 <a href="/wiki/1058%EB%85%84" title="1058년">1058년</a>까지 알헤시라스에는 <a href="/wiki/%ED%83%80%EC%9D%B4%ED%8C%8C" title="타이파">타이파</a>라고 하는 독립 국가가 있었다.</p>
<p><a href="/wiki/1344%EB%85%84" title="1344년">1344년</a>에 <a href="/wiki/%EC%B9%B4%EC%8A%A4%ED%8B%B0%EC%95%BC_%EC%99%95%EA%B5%AD" title="카스티야 왕국">카스티야 왕국</a>의 <a href="/wiki/%EC%95%8C%ED%8F%B0%EC%86%8C_11%EC%84%B8" title="알폰소 11세">알폰소 11세</a>가 알헤시라스를 정복했지만 <a href="/wiki/1368%EB%85%84" title="1368년">1368년</a>에 무어인들이 알헤시라스를 정복했다. 이 때 <a href="/wiki/%EA%B7%B8%EB%9D%BC%EB%82%98%EB%8B%A4_%EC%99%95%EA%B5%AD" title="그라나다 왕국">그라나다 왕국</a>의 무함마드 5세가 알헤시라스를 파괴했고 결국 알헤시라스는 폐허로 남게 된다. 그 후 <a href="/wiki/%EC%8A%A4%ED%8E%98%EC%9D%B8_%EC%99%95%EC%9C%84_%EA%B3%84%EC%8A%B9_%EC%A0%84%EC%9F%81" title="스페인 왕위 계승 전쟁">스페인 왕위 계승 전쟁</a>이 한창이던 <a href="/wiki/1704%EB%85%84" title="1704년">1704년</a>에 영국군에게 포로로 잡힌 <a href="/wiki/%EC%A7%80%EB%B8%8C%EB%A1%A4%ED%84%B0" title="지브롤터">지브롤터</a> 출신 스페인 이민자들이 알헤시라스를 재건했으며 <a href="/wiki/1906%EB%85%84" title="1906년">1906년</a>에 이 곳에서 <a class="new" href="/w/index.php?title=%EC%95%8C%ED%97%A4%EC%8B%9C%EB%9D%BC%EC%8A%A4_%ED%9A%8C%EC%9D%98&amp;action=edit&amp;redlink=1" title="알헤시라스 회의 (없는 문서)">알헤시라스 회의</a>가 열렸다.</p>
