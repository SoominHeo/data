<h1 class="firstHeading" id="firstHeading" lang="ko">암모니아수</h1>
<div class="mw-body-content" id="bodyContent">

<div class="mw-content-ltr" dir="ltr" id="mw-content-text" lang="ko"><script>function mfTempOpenSection(id){var block=document.getElementById("mf-section-"+id);block.className+=" open-block";block.previousSibling.className+=" open-block";}</script><p><b>암모니아수</b>, 또는 <b>수산화 암모늄</b>(Ammonium hydroxide)은 <a href="/wiki/%EC%95%94%EB%AA%A8%EB%8B%88%EC%95%84" title="암모니아">암모니아</a>의 수용액을 일컫는 말이다. <a href="/wiki/%EC%95%94%EB%AA%A8%EB%8B%88%EC%95%84" title="암모니아">암모니아</a> 용액에는 NH<sub>4</sub>OH <a href="/wiki/%EB%B6%84%EC%9E%90" title="분자">분자</a>가 관찰되지 않고 분리할 수도 없기 때문에 엄밀하게는 수산화 암모늄이라는 용어는 잘못된 것이나, 많은 곳에서 사용되고 있다.</p>

<h2><span class="mw-headline" id=".EC.84.B1.EC.A7.88">성질</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%95%94%EB%AA%A8%EB%8B%88%EC%95%84%EC%88%98&amp;action=edit&amp;section=1" title="부분 편집: 성질">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>암모니아수에는 <a href="/wiki/%EC%95%94%EB%AA%A8%EB%8B%88%EC%95%84" title="암모니아">NH<sub>3</sub></a> + <a href="/wiki/%EB%AC%BC" title="물">H<sub>2</sub>O</a>와 NH<sub>4</sub>OH 의 중간형태의 구조를 가진 <a href="/wiki/%EB%B6%84%EC%9E%90" title="분자">분자</a>가 존재한다고 여겨진다. <a href="/wiki/%EC%95%94%EB%AA%A8%EB%8B%88%EC%95%84" title="암모니아">암모니아</a>가 <a href="/wiki/%EB%AC%BC" title="물">물</a>에 녹으면서 다음과 같은 평형이 생성된다.<sup class="reference" id="cite_ref-sajun_1-0"><a href="#cite_note-sajun-1">[1]</a></sup></p>
<dl>
<dd>
<dl>
<dd><a href="/wiki/%EC%95%94%EB%AA%A8%EB%8B%88%EC%95%84" title="암모니아">NH<sub>3</sub></a> + <a href="/wiki/%EB%AC%BC" title="물">H<sub>2</sub>O</a> ⇄ NH<sub>4</sub><sup>+</sup> + OH<sup>-</sup></dd>
</dl>
</dd>
</dl>
<p>위 반응의 <a href="/wiki/%EC%9D%B4%EC%98%A8%ED%99%94_%EC%83%81%EC%88%98" title="이온화 상수">이온화 상수</a>는 25<a href="/wiki/%EC%84%AD%EC%94%A8" title="섭씨">°C</a>에서 1.77×10<sup>-5</sup>이다.<sup class="reference" id="cite_ref-mcgraw_2-0"><a href="#cite_note-mcgraw-2">[2]</a></sup></p>
<p><a href="/wiki/%EC%95%94%EB%AA%A8%EB%8B%88%EC%95%84" title="암모니아">NH<sub>3</sub></a> + <a href="/wiki/%EB%AC%BC" title="물">H<sub>2</sub>O</a>분자는 <a href="/wiki/%EC%95%94%EB%AA%A8%EB%8B%88%EC%95%84" title="암모니아">암모니아</a> <a href="/wiki/%EB%B6%84%EC%9E%90" title="분자">분자</a>와 <a href="/wiki/%EB%AC%BC" title="물">물</a> <a href="/wiki/%EB%B6%84%EC%9E%90" title="분자">분자</a>가 <a href="/wiki/%EC%88%98%EC%86%8C_%EA%B2%B0%ED%95%A9" title="수소 결합">수소 결합</a>으로 결합되어 있는 형태를 가지며, <a href="/wiki/%EB%85%B9%EB%8A%94%EC%A0%90" title="녹는점">녹는점</a>은 -79.0<a href="/wiki/%EC%84%AD%EC%94%A8" title="섭씨">°C</a>이다. 암모니아수에는 2<a href="/wiki/%EC%95%94%EB%AA%A8%EB%8B%88%EC%95%84" title="암모니아">NH<sub>3</sub></a> + <a href="/wiki/%EB%AC%BC" title="물">H<sub>2</sub>O</a>태의 분자도 존재하며, <a href="/wiki/%EB%85%B9%EB%8A%94%EC%A0%90" title="녹는점">녹는점</a>은 -78.8<a href="/wiki/%EC%84%AD%EC%94%A8" title="섭씨">°C</a>이다.<sup class="reference" id="cite_ref-mcgraw_2-1"><a href="#cite_note-mcgraw-2">[2]</a></sup> gd</p>
<h2><span class="mw-headline" id=".EC.A0.9C.EB.B2.95">제법</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%95%94%EB%AA%A8%EB%8B%88%EC%95%84%EC%88%98&amp;action=edit&amp;section=2" title="부분 편집: 제법">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p><a href="/wiki/%EC%95%94%EB%AA%A8%EB%8B%88%EC%95%84" title="암모니아">암모니아</a>를 <a href="/wiki/%EB%AC%BC" title="물">물</a>에 녹인다. 1<a href="/wiki/%EB%B6%80%ED%94%BC" title="부피">부피</a>의 <a href="/wiki/%EB%AC%BC" title="물">물</a>에 670<a href="/wiki/%EB%B6%80%ED%94%BC" title="부피">부피</a>의 <a href="/wiki/%EC%95%94%EB%AA%A8%EB%8B%88%EC%95%84" title="암모니아">암모니아</a>가 녹을 수 있다. <a href="/wiki/%EC%95%94%EB%AA%A8%EB%8B%88%EC%95%84" title="암모니아">암모니아</a>가 <a href="/wiki/%EB%AC%BC" title="물">물</a>에 녹는 과정에서 열이 나기 때문에 식혀줄 필요가 있다. 이 과정에서 <a class="mw-redirect" href="/wiki/%EA%B3%B5%EA%B8%B0_(%EA%B8%B0%EC%B2%B4)" title="공기 (기체)">공기</a> 중의 <a href="/wiki/%EC%9D%B4%EC%82%B0%ED%99%94_%ED%83%84%EC%86%8C" title="이산화 탄소">이산화 탄소</a>가 약간 녹아들어가서 약간의 <a href="/wiki/%ED%83%84%EC%82%B0%EC%97%BC" title="탄산염">탄산염</a>이 생성되나 일반적으로는 이를 제거하지 않는다.<sup class="reference" id="cite_ref-sajun_1-1"><a href="#cite_note-sajun-1">[1]</a></sup></p>
<h2><span class="mw-headline" id=".EC.9A.A9.EB.8F.84">용도</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%95%94%EB%AA%A8%EB%8B%88%EC%95%84%EC%88%98&amp;action=edit&amp;section=3" title="부분 편집: 용도">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>암모니아수는 주로 다음과 같은 용도로 사용된다.<sup class="reference" id="cite_ref-sajun_1-2"><a href="#cite_note-sajun-1">[1]</a></sup></p>
<ul>
<li>침전제, <a href="/wiki/%EC%99%84%EC%B6%A9_%EC%9A%A9%EC%95%A1" title="완충 용액">완충 용액</a> 등의 시약</li>
<li>의류의 얼룩 제거, 모직물 세탁</li>
<li>국소 자극제, 흥분제, 제산제, 중화제 등의 의약품</li>
</ul>
<h2><span class="mw-headline" id=".EB.B3.B4.EA.B4.80.EC.83.81.EC.9D.98_.EC.A3.BC.EC.9D.98">보관상의 주의</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%95%94%EB%AA%A8%EB%8B%88%EC%95%84%EC%88%98&amp;action=edit&amp;section=4" title="부분 편집: 보관상의 주의">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p><a href="/wiki/%EC%95%94%EB%AA%A8%EB%8B%88%EC%95%84" title="암모니아">암모니아</a>가 증발될 수 있고 <a class="mw-redirect" href="/wiki/%EA%B3%B5%EA%B8%B0_(%EA%B8%B0%EC%B2%B4)" title="공기 (기체)">공기</a> 중에 존재하는 <a href="/wiki/%EC%9D%B4%EC%82%B0%ED%99%94_%ED%83%84%EC%86%8C" title="이산화 탄소">이산화 탄소</a> 등의 <a href="/wiki/%EC%82%B0_(%ED%99%94%ED%95%99)" title="산 (화학)">산</a>성 <a href="/wiki/%EA%B8%B0%EC%B2%B4" title="기체">기체</a>를 흡수하기 때문에 <a href="/wiki/%EA%B3%A0%EB%AC%B4" title="고무">고무</a>, <a href="/wiki/%ED%94%8C%EB%9D%BC%EC%8A%A4%ED%8B%B1" title="플라스틱">플라스틱</a>, <a href="/wiki/%EC%9C%A0%EB%A6%AC" title="유리">유리</a> 등의 마개로 막아서 보관하여야 한다. <a href="/wiki/%EC%98%A8%EB%8F%84" title="온도">온도</a> 상승과 함께 <a href="/wiki/%EC%95%94%EB%AA%A8%EB%8B%88%EC%95%84" title="암모니아">암모니아</a>의 분압이 상승하여 폭발할 수 있으므로 30<a href="/wiki/%EC%84%AD%EC%94%A8" title="섭씨">°C</a> 이하의 찬 곳에서 보관해야 한다. 그 밖의 주의사항은 <a href="/wiki/%EC%95%94%EB%AA%A8%EB%8B%88%EC%95%84" title="암모니아">암모니아</a>의 것을 따른다.<sup class="reference" id="cite_ref-sajun_1-3"><a href="#cite_note-sajun-1">[1]</a></sup></p>
