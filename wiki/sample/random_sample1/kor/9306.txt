<p><b>전미 여객 철도공사</b>(<span style="font-size: smaller;"><a href="/wiki/%EC%98%81%EC%96%B4" title="영어">영어</a>:</span> <span lang="en" xml:lang="en">National Railroad Passenger Corporation</span>, 이하 줄여서 <b>암트랙</b>(<span style="font-size: smaller;"><a href="/wiki/%EC%98%81%EC%96%B4" title="영어">영어</a>:</span> <span lang="en" xml:lang="en">Amtrak</span>)은 <a href="/wiki/%EB%AF%B8%EA%B5%AD" title="미국">미국</a> 전 지역 역에 여객철도 운송업을 하는 준공영 기업으로 암트랙이란 낱말은 (<span style="font-size: smaller;"><a href="/wiki/%EC%98%81%EC%96%B4" title="영어">영어</a>:</span> <span lang="en" xml:lang="en">American</span>)과 (<span style="font-size: smaller;"><a href="/wiki/%EC%98%81%EC%96%B4" title="영어">영어</a>:</span> <span lang="en" xml:lang="en">track</span>)의 합성어로 본사는 <a href="/wiki/%EC%9B%8C%EC%8B%B1%ED%84%B4_D.C." title="워싱턴 D.C.">워싱턴 D.C.</a>에 위치해 있다.</p>

<h2><span class="mw-headline" id=".EA.B0.9C.EC.9A.94">개요</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%95%94%ED%8A%B8%EB%9E%99&amp;action=edit&amp;section=1" title="부분 편집: 개요">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p><a href="/wiki/%EB%AF%B8%EA%B5%AD" title="미국">미국</a>의 경우 <a class="mw-disambig" href="/wiki/%EA%B5%AD%EC%9C%A0_%EC%B2%A0%EB%8F%84" title="국유 철도">국유 철도</a>가 존재 했던 시대가 없었고 전 국토의 철도 네트워크는 <a class="mw-disambig" href="/wiki/%EC%82%AC%EC%B2%A0" title="사철">사철</a> 회사로 구성 되었다. <a class="mw-redirect" href="/wiki/%EC%A0%9C2%EC%B0%A8_%EC%84%B8%EA%B3%84%EB%8C%80%EC%A0%84" title="제2차 세계대전">제2차 세계대전</a> 이후 <a href="/wiki/%ED%95%AD%EA%B3%B5" title="항공">항공</a>이나 <a href="/wiki/%EC%9E%90%EB%8F%99%EC%B0%A8" title="자동차">자동차</a> <a class="mw-redirect" href="/wiki/%EC%88%98%EC%86%A1" title="수송">수송</a>에 의해, <a href="/wiki/%EB%AF%B8%EA%B5%AD" title="미국">미국</a>의 철도 여객 수송량은 감소된 결과 <a href="/wiki/1960%EB%85%84%EB%8C%80" title="1960년대">1960년대</a>에 많은 철도 회사가 여객 영업의 폐지를 단행했다. 철도 여객 수송을 유지하기 위해 각 지역의 철도 회사의 여객 수송 부문을 통합한 전국 일원적인 조직으로 <a href="/wiki/1971%EB%85%84" title="1971년">1971년</a>에 설립했다.</p>
<p>형태는 <a href="/wiki/%EC%9D%BC%EB%B3%B8" title="일본">일본</a> <a href="/wiki/JR" title="JR">JR</a> 여객 회사와 <a class="mw-redirect" href="/wiki/%EC%9D%BC%EB%B3%B8_%ED%99%94%EB%AC%BC%EC%B2%A0%EB%8F%84" title="일본 화물철도">일본 화물철도</a>를 역전한 형태가 되어 있기 때문에 자기 부담의 노선은 <a href="/wiki/%EB%85%B8%EC%8A%A4%EC%9D%B4%EC%8A%A4%ED%8A%B8_%EC%BD%94%EB%A6%AC%EB%8D%94" title="노스이스트 코리더">노스이스트 코리더</a>로 불리는 <a href="/wiki/%EB%B3%B4%EC%8A%A4%ED%84%B4" title="보스턴">보스턴</a>과 <a href="/wiki/%EB%89%B4%EC%9A%95" title="뉴욕">뉴욕</a>에서 <a href="/wiki/%EC%9B%8C%EC%8B%B1%ED%84%B4_D.C." title="워싱턴 D.C.">워싱턴 D.C.</a>과 <a href="/wiki/%EB%AF%B8%EC%8B%9C%EA%B0%84_%EC%A3%BC" title="미시간 주">미시간 주</a>의 일부 구간으로, 그 외의 지구에서는 각 화물 철도 회의 선로를 빌려 자기 부담의 여객 <a href="/wiki/%EC%97%B4%EC%B0%A8" title="열차">열차</a>를 운행하는 형태가 되었다. 또한 <a href="/wiki/%EB%8F%84%EC%8B%9C" title="도시">도시</a>권의 여객 열차의 일부의 운행 위탁을 하청받고 있다.</p>
<p>발족 직후에 일어난 <a class="mw-redirect" href="/wiki/%EC%98%A4%EC%9D%BC_%EC%87%BC%ED%81%AC" title="오일 쇼크">오일 쇼크</a>는 석유류 상승에 수반하는 차나 항공기의 연료가 증가 되면서 철도 수송에 있어서 유리했던 시기도 있지만 <a href="/wiki/1980%EB%85%84%EB%8C%80" title="1980년대">1980년대</a>의 <a href="/wiki/1978%EB%85%84" title="1978년">1978년</a>에 제정된 항공 규제 완화법에 의해 <a href="/wiki/%EB%AF%B8%EA%B5%AD" title="미국">미국</a> 남서부를 중심으로 <a class="mw-redirect" href="/wiki/%EC%82%AC%EC%9A%B0%EC%8A%A4_%EC%9B%A8%EC%8A%A4%ED%8A%B8_%ED%95%AD%EA%B3%B5" title="사우스 웨스트 항공">사우스 웨스트 항공</a>이 세력을 늘리고 있으며 세계 규모의 <a href="/wiki/%EC%A0%80%EA%B0%80_%ED%95%AD%EA%B3%B5%EC%82%AC" title="저가 항공사">저가 항공사</a>의 성공과 <a class="mw-redirect" href="/wiki/%EB%B9%84%EC%A6%88%EB%8B%88%EC%8A%A4_%EB%AA%A8%EB%8D%B8" title="비즈니스 모델">비즈니스 모델</a>의 확립에 의해 중장거리 도시간 노선을 중심으로 철도의 이용객을 감소하는 결과를 가져왔다. 이용이 침체했기 때문에 철도 운행 기술도 세계적으로 높다고 해도 경영 상태가 어려워 지기 시작해 <a href="/wiki/%EB%AF%B8%EA%B5%AD" title="미국">미국 정부</a>와 운행 지역의 일부의 주로부터 재정 원조로 유지하고 있다.</p>
<p><a href="/wiki/2001%EB%85%84" title="2001년">2001년</a> <a href="/wiki/9%C2%B711_%ED%85%8C%EB%9F%AC" title="9·11 테러">9·11 테러</a> 이후 미국인의 대부분이 민간 항공기에 의한 이동을 피해 암트랙의 항공기 이외의 이동 수단을 선택했기 때문에, 일시적으로 실적은 회복을 보였지만, <a href="/wiki/2000%EB%85%84" title="2000년">2000년</a>부터 <a href="/wiki/%EC%B2%A0%EB%8F%84_%EC%82%AC%EA%B3%A0" title="철도 사고">철도 사고</a>등 중대 사고를 빈번히 발생 하면서 그 마저 도시를 여객 철도 사업 그 자체의 폐지되고 있다. 또한 일부 역에서 대량의 <a href="/wiki/%EC%93%B0%EB%A0%88%EA%B8%B0" title="쓰레기">쓰레기</a>와 <a href="/wiki/%EB%82%99%EC%84%9C" title="낙서">낙서</a>가 빈번히 보여져 그 환경의 저하 되었고 그로 인해 여객 수요가 감소되는 등 심각한 문제를 안고 있다.</p>
<h2><span class="mw-headline" id=".EC.97.B4.EC.B0.A8_.EC.84.A4.EB.B9.84.EC.99.80_.EC.9D.B4.EC.9A.A9.EC.9E.90">열차 설비와 이용자</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%95%94%ED%8A%B8%EB%9E%99&amp;action=edit&amp;section=2" title="부분 편집: 열차 설비와 이용자">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>기본적으로 지정석에서 예약을 넣을 필요가 있지만 자유석도 있다. 덧붙여 창구에서 구입할 때 신분 증명서의 제시가 필요하다.</p>
<p><a href="/wiki/%EB%85%B8%EC%8A%A4%EC%9D%B4%EC%8A%A4%ED%8A%B8_%EC%BD%94%EB%A6%AC%EB%8D%94" title="노스이스트 코리더">노스이스트 코리더</a> 및 <a href="/wiki/%EC%BA%98%EB%A6%AC%ED%8F%AC%EB%8B%88%EC%95%84_%EC%A3%BC" title="캘리포니아 주">캘리포니아 주</a>의 3개의 노선을 제외한 전 노선에 <a href="/wiki/%EC%B9%A8%EB%8C%80%EC%B0%A8" title="침대차">침대차</a>와 좌석차로 구성되어 있다. 열차에 따라서는 비즈니스 석, 식당차(다이나), 라운지, <a class="mw-redirect" href="/wiki/%EC%B9%B4%ED%8E%98" title="카페">카페</a>, 짐차, 합조차로 편성되어 있다. 또한 저소득층과 <a href="/wiki/%EC%B2%A0%EB%8F%84_%EB%8F%99%ED%98%B8%EC%9D%B8" title="철도 동호인">철도 동호인</a>을 비롯해 시간에 여유가 있는 관광객과 장년층과 노년층이 많이 이용한다.</p>
<p>침대차의 경우 손님과 좌석차의 손님 대우의 경우 상당한 차이가 있다. 예를 들면 침대차의 샤워를 좌석차의 손님이 사용하지 못한 반면 손님이 소프트 드링크를 무제한으로 이용할 수 있으나 좌석차 손님의 경우 별도로 <a href="/wiki/%EB%8B%AC%EB%9F%AC" title="달러">달러</a>를 지불해야 한다. 그리고 좌석차 손님의 경우 <a href="/wiki/%EB%B2%A0%EA%B0%9C" title="베개">베개</a>만 제공한다.</p>
<p><a href="/wiki/%EB%85%B8%EC%8A%A4%EC%9D%B4%EC%8A%A4%ED%8A%B8_%EC%BD%94%EB%A6%AC%EB%8D%94" title="노스이스트 코리더">노스이스트 코리더</a>의 경우 암트랙의 주요 노선에 속하지만 <a href="/wiki/%EB%B2%84%EC%8A%A4" title="버스">버스</a>와 <a href="/wiki/%EB%AF%B8%EA%B5%AD" title="미국">미국</a> 항공과 경쟁을 하고 있으며 비즈니스 승객의 획득에 힘을 쓰고 있다. <a href="/wiki/%EC%BA%98%EB%A6%AC%ED%8F%AC%EB%8B%88%EC%95%84_%EC%A3%BC" title="캘리포니아 주">캘리포니아 주</a> 3개의 노선의 경우(<a href="/wiki/%EC%BA%90%ED%94%BC%ED%86%A8_%EC%BD%94%EB%A6%AC%EB%8D%94" title="캐피톨 코리더">캐피톨 코리더</a>, <a href="/wiki/%EC%83%8C%EC%99%80%ED%82%A8%EC%8A%A4_(%EC%97%B4%EC%B0%A8)" title="샌와킨스 (열차)">샌와킨스</a>, <a href="/wiki/%ED%8D%BC%EC%8B%9C%ED%94%BD_%EC%84%9C%ED%94%84%EB%9D%BC%EC%9D%B4%EB%84%88" title="퍼시픽 서프라이너">퍼시픽 서프라이너</a>) 경영적으로 적자에 불구하고 폭넓은 손님층으로 일정한 승객수를 확보하고 있다. <a href="/wiki/2008%EB%85%84" title="2008년">2008년</a> <a href="/wiki/8%EC%9B%94" title="8월">8월</a> 현재 무인역에서 자동 열차 안내 시스템의 도입과 <a href="/wiki/%EC%BA%98%EB%A6%AC%ED%8F%AC%EB%8B%88%EC%95%84_%EC%A3%BC" title="캘리포니아 주">캘리포니아 주</a>에서 운영하는 <a href="/wiki/BART" title="BART">BART</a>, <a href="/wiki/%EC%B9%BC%ED%8A%B8%EB%A0%88%EC%9D%B8" title="칼트레인">칼트레인</a> <a href="/wiki/%EC%A7%80%EC%97%AD" title="지역">지역</a> 교통 기관과 제휴하고 있다.</p>
<p><a href="/wiki/2005%EB%85%84" title="2005년">2005년</a> <a href="/wiki/6%EC%9B%94_10%EC%9D%BC" title="6월 10일">6월 10일</a> 통계로 역수는 528개로 가장 많은 역은 <a href="/wiki/%EB%89%B4%EC%9A%95" title="뉴욕">뉴욕</a>이며 연간 800만명을 이용했다.</p>
