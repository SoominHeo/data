<p><b>압두 디우프</b>(<a href="/wiki/%EC%9B%94%EB%A1%9C%ED%94%84%EC%96%B4" title="월로프어">월로프어</a>: Abdu Juuf, <a href="/wiki/1935%EB%85%84" title="1935년">1935년</a> <a href="/wiki/9%EC%9B%94_7%EC%9D%BC" title="9월 7일">9월 7일</a><sup class="reference" id="cite_ref-PS_1-0"><a href="#cite_note-PS-1">[1]</a></sup> ~ )는 <a href="/wiki/%EC%84%B8%EB%84%A4%EA%B0%88" title="세네갈">세네갈</a>의 정치인으로 1981년부터 2000년까지 재임했던 <a href="/wiki/%EC%84%B8%EB%84%A4%EA%B0%88" title="세네갈">세네갈</a>의 2대 <a href="/wiki/%EB%8C%80%ED%86%B5%EB%A0%B9" title="대통령">대통령</a>이다. 국방장관, 대통령 총무장관, 계획·공업장관, 총리 등을 지내고 <a href="/wiki/1980%EB%85%84" title="1980년">1980년</a> 전임 대통령 상고르가 5번째 임기가 끝나기 전에 사임하자, <a href="/wiki/1980%EB%85%84" title="1980년">1980년</a> <a href="/wiki/12%EC%9B%94_31%EC%9D%BC" title="12월 31일">12월 31일</a>부터 그 다음날 <a href="/wiki/1981%EB%85%84" title="1981년">1981년</a> <a href="/wiki/1%EC%9B%94_1%EC%9D%BC" title="1월 1일">1월 1일</a>까지 세네갈 대통령 권한대행으로 1일간 임시 승계하였으며 <a href="/wiki/1981%EB%85%84" title="1981년">1981년</a> <a href="/wiki/1%EC%9B%94_1%EC%9D%BC" title="1월 1일">1월 1일</a>부터 이듬해 <a href="/wiki/1982%EB%85%84" title="1982년">1982년</a> <a href="/wiki/2%EC%9B%94_1%EC%9D%BC" title="2월 1일">2월 1일</a>까지 세네갈 공화국 제2대 대통령 재임하였고 <a href="/wiki/1982%EB%85%84" title="1982년">1982년</a> <a href="/wiki/%EA%B0%90%EB%B9%84%EC%95%84" title="감비아">감비아</a>와 <a href="/wiki/%EC%84%B8%EB%84%A4%EA%B0%90%EB%B9%84%EC%95%84" title="세네감비아">세네감비아</a> 연방 공화국이 성립되자 연방정부 대통령이 되었다.</p>
<p><a href="/wiki/1983%EB%85%84" title="1983년">1983년</a> <a href="/wiki/2%EC%9B%94" title="2월">2월</a> 대통령 선거에서 본격 당선되었고, <a href="/wiki/1988%EB%85%84" title="1988년">1988년</a> <a href="/wiki/2%EC%9B%94" title="2월">2월</a> 대통령 선거에서 본격 재선되었다. <a href="/wiki/1993%EB%85%84" title="1993년">1993년</a> <a href="/wiki/3%EC%9B%94" title="3월">3월</a> 실질적 3선 대통령이 되었다. <a href="/wiki/1985%EB%85%84" title="1985년">1985년</a>부터 <a href="/wiki/1986%EB%85%84" title="1986년">1986년</a>까지는 <a href="/wiki/%EC%95%84%ED%94%84%EB%A6%AC%EC%B9%B4_%ED%86%B5%EC%9D%BC_%EA%B8%B0%EA%B5%AC" title="아프리카 통일 기구">아프리카 통일 기구</a>(OAU)의 의장을 맡았다.</p>
<p>그는 평화적인 권력 승계와 <a class="new" href="/w/index.php?title=2000%EB%85%84_%EC%84%B8%EB%84%A4%EA%B0%88_%EB%8C%80%ED%86%B5%EB%A0%B9_%EC%84%A0%EA%B1%B0&amp;action=edit&amp;redlink=1" title="2000년 세네갈 대통령 선거 (없는 문서)">2000년 세네갈 대통령 선거</a>에서 <a href="/wiki/%EC%95%95%EB%91%98%EB%9D%BC%EC%98%88_%EC%99%80%EB%8D%B0" title="압둘라예 와데">압둘라예 와데</a>에게 패배한 뒤 기꺼이 떠난것으로 잘 알려져 있다. 그는 2003년부터 <a href="/wiki/%ED%94%84%EB%9E%91%EC%BD%94%ED%8F%AC%EB%8B%88" title="프랑코포니">프랑코포니</a>의 사무총장을 하고 있다.</p>
<h2><span class="mw-headline" id=".EC.B4.88.EA.B8.B0_.EC.82.B6">초기 삶</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%95%95%EB%91%90_%EB%94%94%EC%9A%B0%ED%94%84&amp;action=edit&amp;section=1" title="부분 편집: 초기 삶">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>디우프는 세네갈 <a href="/wiki/%EB%A3%A8%EA%B0%80_(%EC%84%B8%EB%84%A4%EA%B0%88)" title="루가 (세네갈)">루가</a>에서 <a href="/wiki/%ED%92%80%EB%9D%BC%EC%9D%B8" title="풀라인">풀라인</a> 어머니와 <a href="/wiki/%EC%84%B8%EB%A0%88%EB%A5%B4%EC%A1%B1" title="세레르족">세레르족</a> 아버지의 아이로 태어났다. 그는 <a href="/wiki/%EC%83%9D%EB%A3%A8%EC%9D%B4_(%EC%84%B8%EB%84%A4%EA%B0%88)" title="생루이 (세네갈)">생루이</a>에 있는 리세 파이더르브 (Lycée Faidherbe)에 초등학교와 중등학교를 다녔으며, <a href="/wiki/%EB%8B%A4%EC%B9%B4%EB%A5%B4" title="다카르">다카르</a> 대학교와 그 후 <a href="/wiki/%ED%8C%8C%EB%A6%AC_(%ED%94%84%EB%9E%91%EC%8A%A4)" title="파리 (프랑스)">파리</a> <a class="mw-redirect" href="/wiki/%EC%86%8C%EB%A5%B4%EB%B3%B8_%EB%8C%80%ED%95%99%EA%B5%90" title="소르본 대학교">소르본 대학교</a>에서 법을 공부했다. 디우프는 1959년에 학교를 졸업했다.<sup class="reference" id="cite_ref-2"><a href="#cite_note-2">[2]</a></sup></p>
