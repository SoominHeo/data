<h1 class="firstHeading" id="firstHeading" lang="ko">압둘라 빈 압둘 카디르</h1>
<div class="mw-body-content" id="bodyContent">
<div id="siteSub">위키백과, 우리 모두의 백과사전.</div>
<div id="contentSub"></div>
<div class="mw-jump" id="jump-to-nav">
					이동:					<a href="#mw-head">둘러보기</a>, 					<a href="#p-search">검색</a>
</div>
<div class="mw-content-ltr" dir="ltr" id="mw-content-text" lang="ko"><script>function mfTempOpenSection(id){var block=document.getElementById("mf-section-"+id);block.className+=" open-block";block.previousSibling.className+=" open-block";}</script><p><b>압둘라 빈 압둘 카디르 문시</b>(Abdullah bin Abdul Kadir Munshi(Munsyi), <a href="/wiki/1796%EB%85%84" title="1796년">1796년</a> - <a href="/wiki/1854%EB%85%84" title="1854년">1854년</a>)<sup class="reference" id="cite_ref-1"><a href="#cite_note-1">[1]</a></sup> 또는 <b>문시 압둘라</b>(Munshi Abdullah)는 <a href="/wiki/%EB%A7%90%EB%A0%88%EC%9D%B4_%EB%B0%98%EB%8F%84" title="말레이 반도">말레이 반도</a>의 <a href="/wiki/%EC%9D%B8%EB%8F%84" title="인도">인도</a>계 고전 <a href="/wiki/%EC%9E%91%EA%B0%80" title="작가">작가</a>이다. <a class="mw-redirect" href="/wiki/%EB%A7%90%EB%9D%BC%EC%B9%B4" title="말라카">말라카</a>에서 태어난 유명한 <a href="/wiki/%EC%8B%B1%EA%B0%80%ED%8F%AC%EB%A5%B4" title="싱가포르">싱가포르</a>의 <a class="new" href="/w/index.php?title=%EB%AC%B8%EC%8B%9C&amp;action=edit&amp;redlink=1" title="문시 (없는 문서)">문시</a>였으며<sup class="reference" id="cite_ref-a_2-0"><a href="#cite_note-a-2">[2]</a></sup>, <a href="/wiki/%EB%A9%94%EC%B9%B4" title="메카">메카</a> 순례를 위해 여행하던 도중 성지에 도착하지 못하고 58세의 나이로 당시 <a href="/wiki/%EC%98%A4%EC%8A%A4%EB%A7%8C_%EC%A0%9C%EA%B5%AD" title="오스만 제국">오스만 제국</a>령이던 <a href="/wiki/%EC%95%84%EB%9D%BC%EB%B9%84%EC%95%84_%EB%B0%98%EB%8F%84" title="아라비아 반도">아라비아 반도</a>의 <a href="/wiki/%EC%A7%80%EB%8B%A4" title="지다">지다</a>에서 죽었다.<sup class="reference" id="cite_ref-3"><a href="#cite_note-3">[3]</a></sup><sup class="reference" id="cite_ref-4"><a href="#cite_note-4">[4]</a></sup> 그를 칭하는 '문시(Munshi)'라는 말은 '스승'이나 '교육자'를 의미한다.</p>
<p>문시 압둘라는 <a class="new" href="/w/index.php?title=%ED%95%98%EB%93%9C%EB%9D%BC%EB%AF%B8&amp;action=edit&amp;redlink=1" title="하드라미 (없는 문서)">하드라미</a>(<a href="/wiki/%ED%95%98%EB%93%9C%EB%9D%BC%EB%A7%88%EC%9A%B0%ED%8A%B8" title="하드라마우트">하드라마우트</a> 아랍계 사람) 상인의 증손자이며<sup class="reference" id="cite_ref-5"><a href="#cite_note-5">[5]</a></sup>, <a class="mw-redirect" href="/wiki/%ED%83%80%EB%B0%80%EC%9D%B8" title="타밀인">타밀인</a> 및 <a class="mw-disambig" href="/wiki/%EB%A7%90%EB%A0%88%EC%9D%B4%EC%9D%B8" title="말레이인">말레이인</a>의 혈통에도 속한다.<sup class="reference" id="cite_ref-6"><a href="#cite_note-6">[6]</a></sup> 민족 및 종교적 배경 때문에 말레이인들에게 말레이 지역에서 태어난 말레이 혼혈 엘리트 계층을 의미하는 '자위 프라나칸(<span style="font-size: smaller;"><a href="/wiki/%EB%A7%90%EB%A0%88%EC%9D%B4%EC%96%B4" title="말레이어">말레이어</a>:</span> <span lang="ms" xml:lang="ms">Jawi Peranakan</span>)'으로 불리기도 한다. 그는 아버지가 밟았던 경로를 따라서 <a href="/wiki/%EC%98%81%EA%B5%AD" title="영국">영국</a> 및 <a href="/wiki/%EB%84%A4%EB%8D%9C%EB%9E%80%EB%93%9C" title="네덜란드">네덜란드</a> 식민지 관료들을 위해 통번역과 <a href="/wiki/%EC%96%B8%EC%96%B4" title="언어">언어</a> 교사 일을 담당하였다.</p>
<p>문시 압둘라는 당시까지의 말레이계 고전 작가들 중 가장 교양이 있는 사람이었으며<sup class="reference" id="cite_ref-a_2-1"><a href="#cite_note-a-2">[2]</a></sup>, <a href="/wiki/%EC%9E%90%EC%9C%84_%EB%AC%B8%EC%9E%90" title="자위 문자">말레이어의 문자</a>를 혁신하였고<sup class="reference" id="cite_ref-b_7-0"><a href="#cite_note-b-7">[7]</a></sup>, 말레이 현대 문학의 아버지로 평가받고 있다.<sup class="reference" id="cite_ref-8"><a href="#cite_note-8">[8]</a></sup> 주요 작품으로는 <a href="/wiki/%EC%9E%90%EC%84%9C%EC%A0%84" title="자서전">자서전</a>인 《<a class="new" href="/w/index.php?title=%ED%9E%88%EC%B9%B4%EC%96%8F_%EC%95%95%EB%91%98%EB%9D%BC&amp;action=edit&amp;redlink=1" title="히카얏 압둘라 (없는 문서)">히카얏 압둘라</a>(Hikayat Abdullah)》, <a href="/wiki/%ED%81%B4%EB%9E%80%ED%83%84_%EC%A3%BC" title="클란탄 주">클란탄</a>으로의 여행을 기록한 《<a class="new" href="/w/index.php?title=%ED%82%A4%EC%82%AC_%ED%94%8C%EB%9D%BC%EC%95%BC%EB%9E%80_%EC%95%95%EB%91%98%EB%9D%BC_%ED%81%AC_%ED%81%B4%EB%9E%80%ED%83%84&amp;action=edit&amp;redlink=1" title="키사 플라야란 압둘라 크 클란탄 (없는 문서)">키사 플라야란 압둘라 크 클란탄</a>(Kisah Pelayaran Abdullah ke Kelantan)》, 성지 메카로의 순례를 기록한 마지막 작품인 《<a class="new" href="/w/index.php?title=%ED%82%A4%EC%82%AC_%ED%94%8C%EB%9D%BC%EC%95%BC%EB%9E%80_%EC%95%95%EB%91%98%EB%9D%BC_%ED%81%AC_%EB%A9%94%EC%B9%B4&amp;action=edit&amp;redlink=1" title="키사 플라야란 압둘라 크 메카 (없는 문서)">키사 플라야란 압둘라 크 메카</a>(Kisah Pelayaran Abdullah ke Mekah)》 등이 있다. 압둘라의 작품은 후대의 말레이 작가들에게 영감의 원천이 되었으며, 고전 말레이 문학 시기에서 현대 말레이 문학 시기로 이행하는 초기 단계의 이정표가 되었다.<sup class="reference" id="cite_ref-b_7-1"><a href="#cite_note-b-7">[7]</a></sup></p>
<p>압둘라는 또한 당시 말레이 정치체계 '크라자안(Kerajaan, 왕국)'에 대한 열정적인 비평가이기도 했다. 작품 《키사 플라야란 압둘라 크 클란탄》에는 말레이 지도자들에 대한 충고와 영국식 통치 체계와 말레이식 통치 체계의 비교가 담겨 있다.<sup class="reference" id="cite_ref-9"><a href="#cite_note-9">[9]</a></sup> 압둘라는 크라자안 체제가 말레이인의 사회 개선에 장애가 되어 말레이인에게 해롭다고 논했다. 압둘라는 말레이 술탄이 이기적이고, 신민을 인간이라기보다는 동물에 가깝게 대할 정도로 그들에 대해 무관심하다고 평하고 있다.</p>
<p>말레이 사회에서 <a href="/wiki/%EA%B7%BC%EB%8C%80%EC%84%B1" title="근대성">근대성</a>과 '탁월해지기 위해 애쓰는(striving for excellence)' 인간의 개념은 압둘라의 생각과 그의 신랄한 크라자안 비평에 뿌리를 두고 있다고 해도 과언이 아니다. 크라자안 하에서 말레이인들은 <a href="/wiki/%EA%B5%90%EC%9C%A1" title="교육">교육</a> 기회를 박탈당하고 억압받기 쉽다. 교육받지 못한 사람은 부당한 대우에 대해 의문시할 능력도 갖지 못하고, 결국 자신의 처우를 개선하는 변화를 위해 주도적으로 행동할 수 없게 된다. 압둘라의 비평은 과장되어 있을 수도 있지만, 전혀 근거가 없는 것도 아니었다. 그는 실제로 말레이 <a href="/wiki/%EC%A0%80%EB%84%90%EB%A6%AC%EC%8A%A4%ED%8A%B8" title="저널리스트">저널리스트</a>의 효시격인 인물로 여겨진다.</p>
