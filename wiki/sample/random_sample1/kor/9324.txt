<p><b>앙골라 민족 해방 전선</b>(<span style="font-size: smaller;"><a href="/wiki/%ED%8F%AC%EB%A5%B4%ED%88%AC%EA%B0%88%EC%96%B4" title="포르투갈어">포르투갈어</a>:</span> <span lang="pt" xml:lang="pt">Frente Nacional de Libertação de Angola</span>, FNLA)은 <a href="/wiki/%EC%95%99%EA%B3%A8%EB%9D%BC" title="앙골라">앙골라</a>의 <a href="/wiki/%EA%B8%B0%EB%8F%85%EA%B5%90_%EB%AF%BC%EC%A3%BC%EC%A3%BC%EC%9D%98" title="기독교 민주주의">기독교 민주주의</a> <a href="/wiki/%EC%9A%B0%EC%9D%B5" title="우익">우익</a> 정당이며 과거 반군단체였다. 1956년 콩고족을 중심으로 콩고 왕국의 부흥을 목적으로 결성된 앙골라 북부 인민 동맹(UPNA)이 전신이며, <a href="/wiki/%EC%95%99%EA%B3%A8%EB%9D%BC_%EB%82%B4%EC%A0%84" title="앙골라 내전">앙골라 내전</a> 당시 <a class="mw-redirect" href="/wiki/%EC%95%99%EA%B3%A8%EB%9D%BC_%EC%99%84%EC%A0%84_%EB%8F%85%EB%A6%BD_%EB%AF%BC%EC%A1%B1_%EB%8F%99%EB%A7%B9" title="앙골라 완전 독립 민족 동맹">앙골라 완전 독립 민족 동맹</a>(UNITA)과 <a href="/wiki/%EB%AF%B8%EA%B5%AD" title="미국">미국</a>, <a href="/wiki/%EB%82%A8%EC%95%84%ED%94%84%EB%A6%AC%EC%B9%B4_%EA%B3%B5%ED%99%94%EA%B5%AD" title="남아프리카 공화국">남아프리카 공화국</a>, <a href="/wiki/%EC%A4%91%ED%99%94%EC%9D%B8%EB%AF%BC%EA%B3%B5%ED%99%94%EA%B5%AD" title="중화인민공화국">중화인민공화국</a>을 비롯한 <a href="/wiki/%EC%95%99%EA%B3%A8%EB%9D%BC" title="앙골라">앙골라</a>의 <a href="/wiki/%EC%9A%B0%EC%9D%B5" title="우익">우익</a> 반군을 지원하는 국가들의 지원을 받으면서 <a class="mw-redirect" href="/wiki/%EC%95%99%EA%B3%A8%EB%9D%BC_%EC%9D%B8%EB%AF%BC_%ED%95%B4%EB%B0%A9_%EC%9A%B4%EB%8F%99" title="앙골라 인민 해방 운동">앙골라 인민 해방 운동</a>(MALA)과 <a href="/wiki/%EC%86%8C%EB%A0%A8" title="소련">소련</a>군에 맞써 싸웠다.</p>
<p>앙골라 독립 후도 <a class="new" href="/w/index.php?title=MPLP&amp;action=edit&amp;redlink=1" title="MPLP (없는 문서)">MPLP</a>와 맞써 싸우기 위한 FNLA, <a class="mw-redirect" href="/wiki/UNITA" title="UNITA">UNITA</a> 연합 사이의 내전은 계속되었지만, MPLA 정부는 북부에 지반을 가진 FNLA 및 카빈다 영토 해방 전선(FLEC)를 적극적으로 공격하는 한편 FNLA와 FLEC의 후견인이었다. 1978년 <a href="/wiki/%EC%9E%90%EC%9D%B4%EB%A5%B4" title="자이르">자이르</a>의 <a href="/wiki/%EB%AA%A8%EB%B6%80%ED%88%AC_%EC%84%B8%EC%84%B8_%EC%84%B8%EC%BD%94" title="모부투 세세 세코">모부투 세세 세코</a> 정권이 FNLA와 FLEC을 지원했고 1979년 프랑스가 앙골라와의 국교를 정상화하면서 후원자를 잃은 FNLA의 세력은 크게 감소했다. 1990년대부터 무력 투쟁을 포기하고 정당 활동 노선으로 돌아서면서 FNLA도 합법적인 정치 노선을 걷게 된다.</p>
