<p><b>앙드레 드 그라세</b> (<span style="font-size: smaller;"><a href="/wiki/%ED%94%84%EB%9E%91%EC%8A%A4%EC%96%B4" title="프랑스어">프랑스어</a>:</span> <span lang="fr" xml:lang="fr">Andre De Grasse</span>; <a href="/wiki/1994%EB%85%84" title="1994년">1994년</a> <a href="/wiki/11%EC%9B%94_10%EC%9D%BC" title="11월 10일">11월 10일</a> ~ )는 <a href="/wiki/%EB%B0%94%EB%B2%A0%EC%9D%B4%EB%8F%84%EC%8A%A4" title="바베이도스">바베이도스</a>계 <a href="/wiki/%EC%BA%90%EB%82%98%EB%8B%A4%EC%9D%B8" title="캐나다인">캐나다인</a> 남자 <a class="mw-redirect" href="/wiki/%EC%9C%A1%EC%83%81" title="육상">육상</a> <a href="/wiki/%EB%8B%A8%EA%B1%B0%EB%A6%AC_%EB%8B%AC%EB%A6%AC%EA%B8%B0" title="단거리 달리기">단거리 종목</a> 선수이다.</p>
<p><a href="/wiki/%EB%B8%8C%EB%9D%BC%EC%A7%88" title="브라질">브라질</a> <a href="/wiki/%EB%A6%AC%EC%9A%B0%EB%8D%B0%EC%9E%90%EB%84%A4%EC%9D%B4%EB%A3%A8" title="리우데자네이루">리우데자네이루</a>에서 열린 <a href="/wiki/2016%EB%85%84_%ED%95%98%EA%B3%84_%EC%98%AC%EB%A6%BC%ED%94%BD" title="2016년 하계 올림픽">2016년 하계 올림픽</a>에서 200m, 100m, 4×100 m 계주에서 각각 은메달과 동메달 2개를 거머줬다. 그리고 NCAA 남자 선수권에서 200m 캐나다 신기록을 썼다. 드 그라세는 <a href="/wiki/%EC%A4%91%EA%B5%AD" title="중국">중국</a> <a href="/wiki/%EB%B2%A0%EC%9D%B4%EC%A7%95_%EC%8B%9C" title="베이징 시">베이징</a>에서 열린 <a href="/wiki/2015%EB%85%84_%EC%84%B8%EA%B3%84_%EC%9C%A1%EC%83%81_%EC%84%A0%EC%88%98%EA%B6%8C_%EB%8C%80%ED%9A%8C" title="2015년 세계 육상 선수권 대회">2015년 세계 육상 선수권 대회</a>에서 100m와 4×100 m 계주에서 동메달을 땄다.</p>
<p>드 그라세는 캐나다 역사상 처음으로 올림픽 육상종목에서 메달을 3개를 땄다. 올림픽에서 메달 2개를 딴 <a href="/wiki/%EB%8F%84%EB%84%88%EB%B2%88_%EB%B2%A0%EC%9D%BC%EB%A6%AC" title="도너번 베일리">도너번 베일리</a>와 <a href="/wiki/%ED%8D%BC%EC%8B%9C_%EC%9C%8C%EB%A6%AC%EC%97%84%EC%8A%A4" title="퍼시 윌리엄스">퍼시 윌리엄스</a>의 기록을 깼다. 앞으로의 장래가 촉망받는 선수이다.</p>

<h2><span class="mw-headline" id=".EC.83.9D.EC.95.A0">생애</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%95%99%EB%93%9C%EB%A0%88_%EB%93%9C_%EA%B7%B8%EB%9D%BC%EC%84%B8&amp;action=edit&amp;section=1" title="부분 편집: 생애">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p><b>앙드레 드 그라세</b>는 고등학교 시절에 육상 선수였고, 26세때 캐나다로 이민간<sup class="reference" id="cite_ref-6"><a href="#cite_note-6">[6]</a></sup> <a class="mw-redirect" href="/wiki/%ED%8A%B8%EB%A6%AC%EB%8B%88%EB%8B%A4%EB%93%9C%ED%86%A0%EB%B0%94%EA%B3%A0" title="트리니다드토바고">트리니다드토바고</a>출신의 그의 어머니 <b>베벌리 드 그라세</b> (Beverley De Grasse)와 십대때 캐나다로 이민간 <a href="/wiki/%EB%B0%94%EB%B2%A0%EC%9D%B4%EB%8F%84%EC%8A%A4" title="바베이도스">바베이도스</a> 출신의 그의 아버지 <b>알렉스 웨이스</b> (Alex Waithe) 사이에서 태어났다.</p>
<p>앙드레 드 그라세는 <a href="/wiki/%EC%98%A8%ED%83%80%EB%A6%AC%EC%98%A4_%EC%A3%BC" title="온타리오 주">온타리오 주</a> <a class="new" href="/w/index.php?title=%EB%A7%88%ED%81%AC%ED%97%98_(%EC%98%A8%ED%83%80%EB%A6%AC%EC%98%A4_%EC%A3%BC)&amp;action=edit&amp;redlink=1" title="마크험 (온타리오 주) (없는 문서)">마크험</a>에서 자랐다. 성녀 마르그리트 부르주아 가톨릭 초등학교에 딸린 유치원에서 2학년까지 재학했다. 그리고 그의 어머니는 테레사 가톨릭 초등학교로 전학시켰다. 그리고 그곳에서 8학년까지 재학했다. 또한 미카엘 맥기브니 신부 가톨릭 아카데미와 밀리켄 밀스 고등학교에 다녔다.</p>
<p>드 그라세는 어머니로 인해 <a class="mw-redirect" href="/wiki/%EB%A1%9C%EB%A7%88_%EA%B0%80%ED%86%A8%EB%A6%AD" title="로마 가톨릭">로마 가톨릭</a>을 믿게되었다. 그리고 팔에 "hope"라는 단어를 세겼다.</p>
<p>드 그라세는 11학년에 육상을 시작하였다. 그는 농구바지를 입고 컨버스 신발을 신고 스타팅 블록 없이 연습했다. 그의 첫 100m는 10.9초 였다.<sup class="reference" id="cite_ref-7"><a href="#cite_note-7">[7]</a></sup> 그리고 <a href="/wiki/1984%EB%85%84_%ED%95%98%EA%B3%84_%EC%98%AC%EB%A6%BC%ED%94%BD" title="1984년 하계 올림픽">1984년 하계 올림픽</a> 400m 계주 동메달리스트 토니 샤프로부터 코칭받게 되었다. 커피빌 커뮤니티 칼리지에 재학중이다가 <a href="/wiki/2015%EB%85%84" title="2015년">2015년</a> 3학년때 <a href="/wiki/%EB%AF%B8%EA%B5%AD" title="미국">미국</a>의 <a href="/wiki/%EC%84%9C%EB%8D%98%EC%BA%98%EB%A6%AC%ED%8F%AC%EB%8B%88%EC%95%84_%EB%8C%80%ED%95%99%EA%B5%90" title="서던캘리포니아 대학교">서던캘리포니아 대학교</a>로 옮겼다.</p>
