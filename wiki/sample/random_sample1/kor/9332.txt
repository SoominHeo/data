<p><b>앙리 5세</b>(Henry V, <a href="/wiki/1820%EB%85%84" title="1820년">1820년</a> <a href="/wiki/9%EC%9B%94_29%EC%9D%BC" title="9월 29일">9월 29일</a> ~ <a href="/wiki/1883%EB%85%84" title="1883년">1883년</a> <a href="/wiki/8%EC%9B%94_24%EC%9D%BC" title="8월 24일">8월 24일</a>)는 <a href="/wiki/%ED%94%84%EB%9E%91%EC%8A%A4" title="프랑스">프랑스</a>의 왕세자로 국왕 <a href="/wiki/%EC%83%A4%EB%A5%BC_10%EC%84%B8" title="샤를 10세">샤를 10세</a>의 손자이자, 베리 공작 <a class="new" href="/w/index.php?title=%EC%83%A4%EB%A5%BC_%ED%8E%98%EB%A5%B4%EB%94%94%EB%82%AD&amp;action=edit&amp;redlink=1" title="샤를 페르디낭 (없는 문서)">샤를 페르디낭</a>의 아들이었다. 왕세자가 되기 이전의 직책인 샹보르 백작으로 불린다. <a class="mw-redirect" href="/wiki/%EB%B6%80%EB%A5%B4%EB%B4%89_%EC%99%95%EC%A1%B0" title="부르봉 왕조">부르봉 왕조</a>의 마지막 정통 후계자였다.</p>
<h2><span class="mw-headline" id=".EC.83.9D.EC.95.A0">생애</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%95%99%EB%A6%AC_5%EC%84%B8&amp;action=edit&amp;section=1" title="부분 편집: 생애">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>어려서 아버지 <a class="new" href="/w/index.php?title=%EC%83%A4%EB%A5%BC_%ED%8E%98%EB%A5%B4%EB%94%94%EB%82%AD&amp;action=edit&amp;redlink=1" title="샤를 페르디낭 (없는 문서)">샤를 페르디낭</a>을 잃고 할아버지 <a href="/wiki/%EC%83%A4%EB%A5%BC_10%EC%84%B8" title="샤를 10세">샤를 10세</a>의 잠정적 후계자가 되었다. 또한 자식이 없던 <a class="mw-redirect" href="/wiki/%EB%A7%88%EB%A6%AC_%ED%85%8C%EB%A0%88%EC%A6%88_%EC%83%A4%EB%A5%BC%EB%A1%9C%ED%8A%B8" title="마리 테레즈 샤를로트">마리 테레즈 샤를로트</a>가 그를 특별히 귀여워했다고도 한다. 그러나 전제정치를 펴던 샤를 10세는 <a href="/wiki/1830%EB%85%84" title="1830년">1830년</a> 7월 혁명으로 퇴위 압력에 직면했다. 그러나 샤를 10세는 당시 5세의 어린 나이였던 손자인 앙리에게 자리를 양위하고 오를레앙 공작 루이 필리프를 섭정공으로 하도록 계획했다. 그러나 <a class="mw-redirect" href="/wiki/%EB%A3%A8%EC%9D%B4_%ED%95%84%EB%A6%AC%ED%94%84" title="루이 필리프">루이 필리프</a> 자신이 정변을 일으켜 왕위에 오름으로서 그는 할아버지와 함께 피신해야 했다.</p>
<p><a href="/wiki/1870%EB%85%84" title="1870년">1870년</a> <a href="/wiki/%EB%82%98%ED%8F%B4%EB%A0%88%EC%98%B9_3%EC%84%B8" title="나폴레옹 3세">나폴레옹 3세</a>가 퇴위하자 그는 다시 왕위계승권을 주장하였다. 프랑스의 일부 정치인은 입헌군주제를 하는 조건으로 왕위를 주는 협상안을 제시했으나 그가 <a class="mw-redirect" href="/wiki/%ED%94%84%EB%9E%91%EC%8A%A4_%EB%8C%80%ED%98%81%EB%AA%85" title="프랑스 대혁명">프랑스 대혁명</a>이후의 성과를 부정함으로써 왕위에 오르지 못하였다. 후계자 없이 사망하여 <a href="/wiki/%ED%94%84%EB%9E%91%EC%8A%A4" title="프랑스">프랑스</a>의 <a class="mw-redirect" href="/wiki/%EB%B6%80%EB%A5%B4%EB%B4%89_%EC%99%95%EC%A1%B0" title="부르봉 왕조">부르봉 왕조</a> 직계는 단절되고 말았다.</p>
