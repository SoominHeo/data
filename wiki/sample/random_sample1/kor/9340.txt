<h2><span class="mw-headline" id=".EC.9D.B4.EB.A6.84">이름</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%95%99%EC%8A%A4_%EC%95%84%EB%A5%B4%ED%89%81&amp;action=edit&amp;section=1" title="부분 편집: 이름">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p><a href="/wiki/%EC%95%84%EB%AA%85" title="아명">원명(原名)</a>은 <b>한스 하인리히 에른스트 하르퉁</b>(Hans Heinrich Ernst Hartung)이며 나중에 <b>앙스 아르퉁</b>(Hans Hartung)으로 <a href="/wiki/%EA%B0%9C%EB%AA%85" title="개명">개명(改名)</a>하였다.</p>
<h2><span class="mw-headline" id=".EC.83.9D.EC.95.A0">생애</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%95%99%EC%8A%A4_%EC%95%84%EB%A5%B4%ED%89%81&amp;action=edit&amp;section=2" title="부분 편집: 생애">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<h3><span class="mw-headline" id=".EC.83.9D.EC.95.A0_.EC.B4.88.EA.B8.B0">생애 초기</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%95%99%EC%8A%A4_%EC%95%84%EB%A5%B4%ED%89%81&amp;action=edit&amp;section=3" title="부분 편집: 생애 초기">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>그는 <a href="/wiki/%EB%8F%85%EC%9D%BC_%EC%A0%9C%EA%B5%AD" title="독일 제국">독일 제국</a> <a href="/wiki/%EC%9E%91%EC%84%BC_%EC%A3%BC" title="작센 주">작센 주</a> <a href="/wiki/%EB%9D%BC%EC%9D%B4%ED%94%84%EC%B9%98%ED%9E%88" title="라이프치히">라이프치히</a>에서 출생하였고 그의 아버지는 <a href="/wiki/%EB%82%B4%EA%B3%BC" title="내과">내과</a> <a href="/wiki/%EC%9D%98%EC%82%AC" title="의사">전문의</a> 및 <a href="/wiki/%EC%9D%98%ED%95%99" title="의학">의학</a><a href="/wiki/%EC%84%9D%EC%82%AC" title="석사">석사</a>였다.</p>
<h3><span class="mw-headline" id=".ED.95.99.EB.A0.A5">학력</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%95%99%EC%8A%A4_%EC%95%84%EB%A5%B4%ED%89%81&amp;action=edit&amp;section=4" title="부분 편집: 학력">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<ul>
<li><a href="/wiki/1924%EB%85%84" title="1924년">1924년</a> <a href="/wiki/%EB%9D%BC%EC%9D%B4%ED%94%84%EC%B9%98%ED%9E%88_%EB%8C%80%ED%95%99%EA%B5%90" title="라이프치히 대학교">바이마르 공화국 작센 주 라이프치히 대학교 철학과</a> 전퇴</li>
<li><a href="/wiki/1925%EB%85%84" title="1925년">1925년</a> <a href="/wiki/%EB%9D%BC%EC%9D%B4%ED%94%84%EC%B9%98%ED%9E%88_%EB%8C%80%ED%95%99%EA%B5%90" title="라이프치히 대학교">바이마르 공화국 작센 주 라이프치히 대학교 미술사학과</a> 중퇴</li>
<li><a href="/wiki/1926%EB%85%84" title="1926년">1926년</a> <a class="extiw" href="https://en.wikipedia.org/wiki/Dresden_Academy_of_Fine_Arts" title="en:Dresden Academy of Fine Arts">바이마르 공화국 작센 주 드레스덴 미술학교</a> 중퇴</li>
</ul>
<h3><span class="mw-headline" id=".EC.9D.BC.EC.83.9D.EA.B3.BC_.EC.97.85.EC.A0.81">일생과 업적</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%95%99%EC%8A%A4_%EC%95%84%EB%A5%B4%ED%89%81&amp;action=edit&amp;section=5" title="부분 편집: 일생과 업적">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>일찍이 추상적 표현에 천부적 소질을 가지었던 그는 <a class="extiw" href="https://en.wikipedia.org/wiki/Tachisme" title="en:Tachisme">타시슴</a> <a href="/wiki/%EB%AF%B8%EC%88%A0" title="미술">아트</a> <a class="mw-disambig" href="/wiki/%EC%B6%94%EC%83%81%ED%99%94" title="추상화">추상화</a>의 선구자로 일컬어지며 다이내믹한 선의 움직임과 리드미컬한 구성이 두드러진 작품들을 양산하여 <a class="mw-disambig" href="/wiki/%EC%B6%94%EC%83%81%ED%99%94" title="추상화">추상화</a> 분야의 대표적 <a href="/wiki/%ED%99%94%EA%B0%80" title="화가">화가</a>가 되었다.</p>
<p><a href="/wiki/1926%EB%85%84" title="1926년">1926년</a> <a href="/wiki/%ED%94%84%EB%9E%91%EC%8A%A4" title="프랑스">프랑스</a> <a href="/wiki/%ED%8C%8C%EB%A6%AC_(%ED%94%84%EB%9E%91%EC%8A%A4)" title="파리 (프랑스)">파리</a>에서 <a class="mw-disambig" href="/wiki/%EC%B6%94%EC%83%81%ED%99%94" title="추상화">추상화</a><a href="/wiki/%ED%99%94%EA%B0%80" title="화가">가</a>로 화단에 첫 등단하였고 이듬해 <a href="/wiki/1927%EB%85%84" title="1927년">1927년</a> <a href="/wiki/%EB%8F%85%EC%9D%BC" title="독일">바이마르 공화국</a> <a href="/wiki/%EC%9E%91%EC%84%BC_%EC%A3%BC" title="작센 주">작센 주</a> <a href="/wiki/%EB%93%9C%EB%A0%88%EC%8A%A4%EB%8D%B4" title="드레스덴">드레스덴</a>에서 본격 <a class="mw-disambig" href="/wiki/%EC%B6%94%EC%83%81%ED%99%94" title="추상화">추상화</a><a href="/wiki/%ED%99%94%EA%B0%80" title="화가">가</a> 활약을 하기 시작하였다.</p>
<p>이후 그는 <a href="/wiki/1929%EB%85%84" title="1929년">1929년</a>에 <a href="/wiki/%EB%8F%85%EC%9D%BC" title="독일">바이마르 공화국</a> <a href="/wiki/%EB%B0%94%EC%9D%B4%EC%97%90%EB%A5%B8_%EC%A3%BC" title="바이에른 주">바이에른 주</a> <a href="/wiki/%EB%AE%8C%ED%97%A8" title="뮌헨">뮌헨</a>에서 <a href="/wiki/%EC%8A%A4%EC%9B%A8%EB%8D%B4" title="스웨덴">스웨덴</a> 태생의 <a href="/wiki/%EB%85%B8%EB%A5%B4%EC%9B%A8%EC%9D%B4" title="노르웨이">노르웨이</a> <a class="mw-redirect" href="/wiki/%EC%97%AC%EC%9E%90" title="여자">여성</a> <a class="mw-disambig" href="/wiki/%EC%B6%94%EC%83%81%ED%99%94" title="추상화">추상화</a><a href="/wiki/%ED%99%94%EA%B0%80" title="화가">가</a> <a class="extiw" href="https://en.wikipedia.org/wiki/Anna-Eva_Bergman" title="en:Anna-Eva Bergman">아나 에바 베르망 아르퉁</a>(Anna-Eva Bergman Hartung)과 결혼하여 <a href="/wiki/%EB%85%B8%EB%A5%B4%EC%9B%A8%EC%9D%B4" title="노르웨이">노르웨이</a> <a href="/wiki/%EC%98%81%EC%A3%BC%EA%B6%8C" title="영주권">영구거주권</a>을 취득하였으며 이듬해 <a href="/wiki/1930%EB%85%84" title="1930년">1930년</a> <a href="/wiki/%EB%85%B8%EB%A5%B4%EC%9B%A8%EC%9D%B4" title="노르웨이">노르웨이</a> <a href="/wiki/%EC%98%81%EC%A3%BC%EA%B6%8C" title="영주권">영구거주권</a>을 포기하고 <a href="/wiki/%EC%8A%A4%EC%9B%A8%EB%8D%B4" title="스웨덴">스웨덴</a> <a href="/wiki/%EC%98%81%EC%A3%BC%EA%B6%8C" title="영주권">영구거주권</a>을 취득하였으나 1년 후 <a href="/wiki/1931%EB%85%84" title="1931년">1931년</a> <a href="/wiki/%EB%8F%85%EC%9D%BC" title="독일">바이마르 공화국</a> <a href="/wiki/%EC%9E%91%EC%84%BC_%EC%A3%BC" title="작센 주">작센 주</a> <a href="/wiki/%EB%93%9C%EB%A0%88%EC%8A%A4%EB%8D%B4" title="드레스덴">드레스덴</a>에서 <a class="extiw" href="https://en.wikipedia.org/wiki/Anna-Eva_Bergman" title="en:Anna-Eva Bergman">아나 에바 베르망 아르퉁</a>(Anna-Eva Bergman Hartung)과 이혼하여 <a href="/wiki/%EC%8A%A4%EC%9B%A8%EB%8D%B4" title="스웨덴">스웨덴</a> <a href="/wiki/%EC%98%81%EC%A3%BC%EA%B6%8C" title="영주권">영구거주권</a>을 포기하였다.</p>
<p>1년 후 <a href="/wiki/1932%EB%85%84" title="1932년">1932년</a> <a href="/wiki/7%EC%9B%94_20%EC%9D%BC" title="7월 20일">7월 20일</a>에는 <a href="/wiki/%ED%94%84%EB%9E%91%EC%8A%A4" title="프랑스">프랑스</a> <a href="/wiki/%ED%94%84%EB%9E%91%EC%8A%A4_%EC%9C%A1%EA%B5%B0" title="프랑스 육군">육군</a> <a href="/wiki/%ED%95%98%EC%82%AC" title="하사">하사</a> 임관하여 <a href="/wiki/%ED%94%84%EB%9E%91%EC%8A%A4" title="프랑스">프랑스</a> <a href="/wiki/%ED%94%84%EB%9E%91%EC%8A%A4_%EC%99%B8%EC%9D%B8%EB%B6%80%EB%8C%80" title="프랑스 외인부대">육군 외인부대</a> 정훈부사관으로 보임되었고 이듬해 <a href="/wiki/1933%EB%85%84" title="1933년">1933년</a> <a href="/wiki/%EC%8A%A4%ED%8E%98%EC%9D%B8" title="스페인">스페인</a> <a href="/wiki/%EC%98%81%EC%A3%BC%EA%B6%8C" title="영주권">영구거주권</a>을 취득하여 <a href="/wiki/%EC%8A%A4%ED%8E%98%EC%9D%B8" title="스페인">스페인</a> <a href="/wiki/%EB%A9%94%EB%85%B8%EB%A5%B4%EC%B9%B4_%EC%84%AC" title="메노르카 섬">메노르카 섬</a>에 거주하였으며 1년 후 <a href="/wiki/1934%EB%85%84" title="1934년">1934년</a> <a href="/wiki/%EC%8A%A4%ED%8E%98%EC%9D%B8" title="스페인">스페인</a> <a href="/wiki/%EC%98%81%EC%A3%BC%EA%B6%8C" title="영주권">영구거주권</a>을 포기하였고 <a href="/wiki/%ED%8C%8C%EB%A6%AC_(%ED%94%84%EB%9E%91%EC%8A%A4)" title="파리 (프랑스)">프랑스 파리</a>로 귀환하여 <a href="/wiki/%EB%82%98%EC%B9%98_%EB%8F%85%EC%9D%BC" title="나치 독일">나치 독일</a> <a href="/wiki/%EC%8B%9C%EB%AF%BC%EA%B6%8C" title="시민권">시민권자</a> 신분의 <a href="/wiki/%ED%94%84%EB%9E%91%EC%8A%A4" title="프랑스">프랑스</a> <a href="/wiki/%ED%94%84%EB%9E%91%EC%8A%A4_%EC%9C%A1%EA%B5%B0" title="프랑스 육군">육군</a> <a href="/wiki/%EA%B5%B0%EC%9D%B8" title="군인">군인</a> 신분으로 <a class="mw-disambig" href="/wiki/%EC%B6%94%EC%83%81%ED%99%94" title="추상화">추상화</a><a href="/wiki/%ED%99%94%EA%B0%80" title="화가">가</a> 분야에 종사하다가 <a href="/wiki/1939%EB%85%84" title="1939년">1939년</a> <a href="/wiki/9%EC%9B%94_1%EC%9D%BC" title="9월 1일">9월 1일</a> <a href="/wiki/%EC%A0%9C2%EC%B0%A8_%EC%84%B8%EA%B3%84_%EB%8C%80%EC%A0%84" title="제2차 세계 대전">제2차 세계 대전</a>이 일어나자 <a href="/wiki/%ED%94%84%EB%9E%91%EC%8A%A4_%EC%99%B8%EC%9D%B8%EB%B6%80%EB%8C%80" title="프랑스 외인부대">프랑스 육군 외인부대</a> 정훈부사관으로 <a href="/wiki/%EC%A0%9C2%EC%B0%A8_%EC%84%B8%EA%B3%84_%EB%8C%80%EC%A0%84" title="제2차 세계 대전">2차 대전</a>에 참전하였고 <a href="/wiki/%EC%A0%9C2%EC%B0%A8_%EC%84%B8%EA%B3%84_%EB%8C%80%EC%A0%84" title="제2차 세계 대전">전란</a> 참전 중이던 이듬해 <a href="/wiki/1940%EB%85%84" title="1940년">1940년</a>에 <a href="/wiki/%EC%8A%A4%ED%8E%98%EC%9D%B8" title="스페인">스페인</a>의 <a href="/wiki/%EC%A1%B0%EA%B0%81%EA%B0%80" title="조각가">조각가</a> <a class="extiw" href="https://en.wikipedia.org/wiki/Julio_Gonz%C3%A1lez_(sculptor)" title="en:Julio González (sculptor)">훌리오 곤살레스 이 페이세르</a>(Julio González i Pellicer)의 <a href="/wiki/%EB%94%B8" title="딸">외동딸</a> 로베르타 공잘레즈 아르퉁(Roberta González Hartung)과 재혼한 것에 이어 이듬해 <a href="/wiki/1941%EB%85%84" title="1941년">1941년</a>에는 <a href="/wiki/%ED%94%84%EB%9E%91%EC%8A%A4" title="프랑스">프랑스</a> <a href="/wiki/12%EC%9B%94_9%EC%9D%BC" title="12월 9일">12월 9일</a>을 기하여 <a href="/wiki/%ED%94%84%EB%9E%91%EC%8A%A4" title="프랑스">프랑스</a> <a href="/wiki/%ED%94%84%EB%9E%91%EC%8A%A4_%EC%9C%A1%EA%B5%B0" title="프랑스 육군">육군</a> <a href="/wiki/%EC%A4%91%EC%82%AC_(%EA%B5%B0%EC%82%AC)" title="중사 (군사)">중사</a> 예편하였고 이듬해 <a href="/wiki/1946%EB%85%84" title="1946년">1946년</a> <a 연립공화국</a> <a href="/wiki/%EC%8B%9C%EB%AF%BC%EA%B6%8C" title="시민권">시민권</a>을 포기와 동시에 <a href="/wiki/%ED%94%84%EB%9E%91%EC%8A%A4" title="프랑스">프랑스</a> <a href="/wiki/%EC%8B%9C%EB%AF%BC%EA%B6%8C" href="/wiki/1947%EB%85%84" title="1947년">1947년</a> <a href="/wiki/%EA%B0%9C%EC%8B%A0%EA%B5%90" title="개신교">개신교</a>(<a href="/wiki/%EA%B0%90%EB%A6%AC%EA%B5%90" title="감리교">감리회</a>)에 귀의하였다.</p>
<p><a href="/wiki/1952%EB%85%84" title="1952년">1952년</a>에는 21년 전 <a href="/wiki/%EC%9E%91%EC%84%BC_%EC%A3%BC" title="작센 주">독일 작센 주</a> <a href="/wiki/%EB%93%9C%EB%A0%88%EC%8A%A4%EB%8D%B4" title="드레스덴">드레스덴</a>에서 이혼하였던 1번째 前 부인 <a class="extiw" href="https://en.wikipedia.org/wiki/Anna-Eva_Bergman" title="en:Anna-Eva Bergman">아나 에바 베르망 아르퉁</a>(Anna-Eva Bergman Hartung)을 <a href="/wiki/%ED%8C%8C%EB%A6%AC_(%ED%94%84%EB%9E%91%EC%8A%A4)" title="파리 (프랑스)">프랑스 파리</a>에서 우연히 다시 만나며 이후 그녀와 5년간의 열정 어린 해후를 나누다가 <a href="/wiki/1957%EB%85%84" title="1957년">1957년</a> <a class="extiw" href="https://en.wikipedia.org/wiki/Anna-Eva_Bergman" title="en:Anna-Eva Bergman">아나 에바 베르망 아르퉁</a>(Anna-Eva Bergman Hartung)과 전격 재결합하였다.</p>
<p>일생 동안 두 번 결혼을 하였지만 슬하에 자녀가 없었던 그는 <a class="extiw" href="https://en.wikipedia.org/wiki/Tachisme" title="en:Tachisme">타시슴</a> <a href="/wiki/%EB%AF%B8%EC%88%A0" title="미술">아트</a>의 선구자로 일컬어지는 <a class="mw-disambig" href="/wiki/%EC%B6%94%EC%83%81%ED%99%94" title="추상화">추상화</a><a href="/wiki/%ED%99%94%EA%B0%80" title="화가">가</a>이다.</p>
