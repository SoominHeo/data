<p><b>앙카라</b>(<span style="font-size: smaller;"><a href="/wiki/%ED%84%B0%ED%82%A4%EC%96%B4" title="터키어">터키어</a>:</span> <span lang="tr" xml:lang="tr">Ankara</span>, <span style="font-size: smaller;"><a href="/wiki/%EB%AC%B8%ED%99%94%EC%96%B4" title="문화어">문화어</a>:</span> 앙까라)는 <a href="/wiki/%ED%84%B0%ED%82%A4" title="터키">터키</a>의 <a class="mw-redirect" href="/wiki/%EC%88%98%EB%8F%84_(%EC%82%AC%ED%9A%8C)" title="수도 (사회)">수도</a>이다. <a href="/wiki/%EC%9D%B4%EC%8A%A4%ED%83%84%EB%B6%88" title="이스탄불">이스탄불</a>에 이어 두 번째로 큰 도시이며 <a href="/wiki/%EC%95%99%EC%B9%B4%EB%9D%BC_%EC%A3%BC" title="앙카라 주">앙카라 주</a>의 주도이기도 하다. 인구는 4,587,558명이고(<a href="/wiki/2014%EB%85%84" title="2014년">2014년</a> 통계), <a href="/wiki/%EC%95%99%EC%B9%B4%EB%9D%BC_%EC%A3%BC" title="앙카라 주">앙카라 주</a>의 인구는 5,150,072명(<a href="/wiki/2015%EB%85%84" title="2015년">2015년</a> 통계)이다.</p>
<p>앙카라는 터키의 초대 대통령 <a href="/wiki/%EB%AC%B4%EC%8A%A4%ED%83%80%ED%8C%8C_%EC%BC%80%EB%A7%90_%EC%95%84%ED%83%80%ED%8A%80%EB%A5%B4%ED%81%AC" title="무스타파 케말 아타튀르크">무스타파 케말 아타튀르크</a>의 정치적 중심지로, 1923년 <a href="/wiki/%EC%98%A4%EC%8A%A4%EB%A7%8C_%EC%A0%9C%EA%B5%AD" title="오스만 제국">오스만 제국</a>이 멸망하고 <a class="mw-redirect" href="/wiki/%ED%84%B0%ED%82%A4_%EA%B3%B5%ED%99%94%EA%B5%AD" title="터키 공화국">터키 공화국</a>이 세워지자, <a href="/wiki/%EC%9D%B4%EC%8A%A4%ED%83%84%EB%B6%88" title="이스탄불">이스탄불</a>을 대신해 수도로 지정됐다. 앙카라는 공무원이 많이 거주하는 도시이지만, <a class="new" href="/w/index.php?title=%EC%95%84%EB%82%98%ED%86%A8%EB%A6%AC%EC%95%84_%EC%A7%80%EC%97%AD&amp;action=edit&amp;redlink=1" title="아나톨리아 지역 (없는 문서)">아나톨리아 지역</a>의 상공업의 중심지이며, 도로와 철도 교통의 중심지다.</p>
<p>앙카라에는 중동전문대학(Hacettepe Üniversitesi)와 앙카라대학(Ankara Üniversitesi)이 있다. 또 국립도서관, 고고학박물관, 민속학박물관과 국립극장, 대통령 관현악단이 앙카라에 있다.</p>
<p>앙카라가 수도가 된 이후의 개발로 옛 도심지였던 울루스(Ulus)와 신 개발지대인 예니세히르(Yenisehir)로 구분된다. 울루스는 로마·비잔틴·오스만 양식의 옛 건물과 좁은 도로로 대표되는 데 반해 예니세히르에는 넓은 도로, 호텔, 극장과 아파트 건물이 들어차 더 현대도시다운 모습을 하고 있다. 정부 청사와 외국 공관도 예니세히르에 소재해 있다.</p>

<h2><span class="mw-headline" id=".EC.97.AD.EC.82.AC">역사</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%95%99%EC%B9%B4%EB%9D%BC&amp;action=edit&amp;section=1" title="부분 편집: 역사">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<h3><span class="mw-headline" id=".EA.B3.A0.EB.8C.80.EC.82.AC">고대사</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%95%99%EC%B9%B4%EB%9D%BC&amp;action=edit&amp;section=2" title="부분 편집: 고대사">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>앙카라는 <a class="mw-redirect" href="/wiki/%EA%B8%B0%EC%9B%90%EC%A0%84_2000%EB%85%84" title="기원전 2000년">기원전 2000년</a>부터 <a class="new" href="/w/index.php?title=%EA%B8%B0%EC%9B%90%EC%A0%84_1700%EB%85%84&amp;action=edit&amp;redlink=1" title="기원전 1700년 (없는 문서)">기원전 1700년</a>까지 히타이트 문명의 지배하에 있었다. 기원전 1000년 경에는 프리기아인들이 이 곳에 정착했고, 프리기아의 수도인 고르디움이 현재 앙카라 주의 남서쪽에 위치한 포랏르에 있었다. 앙카라는 고대에 <a href="/wiki/%ED%94%84%EB%A6%AC%EA%B8%B0%EC%95%84" title="프리기아">프리기아</a>인들의 풍요한 도시였으나 후에 지진으로 고르디움은 파괴되었다. <a class="mw-redirect" href="/wiki/%EB%AF%B8%EB%8B%A4%EC%8A%A4%EC%9D%98_%EC%86%90" title="미다스의 손">미다스의 손</a> 신화로 유명한 <a href="/wiki/%EB%AF%B8%EB%8B%A4%EC%8A%A4" title="미다스">미다스 왕</a>이 <a href="/wiki/%ED%94%84%EB%A6%AC%EA%B8%B0%EC%95%84" title="프리기아">프리기아</a>의 왕이었다.</p>
<p><a href="/wiki/%ED%94%84%EB%A6%AC%EA%B8%B0%EC%95%84" title="프리기아">프리기아</a>의 풍습은 <a href="/wiki/%EB%A6%AC%EB%94%94%EC%95%84" title="리디아">리디아</a>와 페르시아의 통치 기간에도 계승됐다. 프리기아의 풍습은 로마 시대 때까지 계속됐다. <a href="/wiki/%ED%8E%98%EB%A5%B4%EC%8B%9C%EC%95%84_%EC%A0%9C%EA%B5%AD" title="페르시아 제국">페르시아 제국</a>의 왕도(王道)가 이곳을 지나갔다. 중앙 아나톨리아 지방의 주권은 페르시아 제국이 기원전 330년 알렉산더 대왕에게 패배하기 전까지 <a href="/wiki/%ED%8E%98%EB%A5%B4%EC%8B%9C%EC%95%84_%EC%A0%9C%EA%B5%AD" title="페르시아 제국">페르시아 제국</a>이 갖고있었다. <a class="mw-redirect" href="/wiki/%EC%95%8C%EB%A0%89%EC%82%B0%EB%8D%94_%EB%8C%80%EC%99%95" title="알렉산더 대왕">알렉산더 대왕</a>은 <a class="new" href="/w/index.php?title=%EA%B3%A0%EB%A5%B4%EB%94%94%EC%9B%80&amp;action=edit&amp;redlink=1" title="고르디움 (없는 문서)">고르디움</a>을 통해 앙카라에 입성했고 짧은 기간만 머물렀다. <a href="/wiki/%EA%B8%B0%EC%9B%90%EC%A0%84_323%EB%85%84" title="기원전 323년">기원전 323년</a> <a class="mw-redirect" href="/wiki/%EC%95%8C%EB%A0%89%EC%82%B0%EB%8D%94_%EB%8C%80%EC%99%95" title="알렉산더 대왕">알렉산더 대왕</a>이 사망하자, 마케도니아 제국은 뿔뿔히 나뉘어서 안티고노스 1세가 앙카라 지역을 차지한다,</p>
<p>기원전 300년 경에는 <a href="/wiki/%EC%95%84%EB%82%98%ED%86%A8%EB%A6%AC%EC%95%84" title="아나톨리아">아나톨리아</a> 지방의 북동부에 위치한 <a href="/wiki/%ED%8F%B0%ED%86%A0%EC%8A%A4" title="폰토스">폰토스</a>가 무역거점으로 삼기위해서 이 지역에 진출했다. 그리고 앙카라의 이름의 기원이 된 안키라(Áγκυρα, <a href="/wiki/%EA%B7%B8%EB%A6%AC%EC%8A%A4%EC%96%B4" title="그리스어">그리스어</a>로 닻)라는 이름이 생겨났다.</p>
<h3><span class="mw-headline" id=".EC.BC.88.ED.8A.B8_.EC.8B.9C.EB.8C.80">켈트 시대</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%95%99%EC%B9%B4%EB%9D%BC&amp;action=edit&amp;section=3" title="부분 편집: 켈트 시대">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>그 후 <a href="/wiki/%EA%B8%B0%EC%9B%90%EC%A0%84_232%EB%85%84" title="기원전 232년">기원전 232년</a>경 갈라티아 지방에 정착한 <a class="mw-redirect" href="/wiki/%EC%BC%88%ED%8A%B8" title="켈트">켈트</a>인의 일파인 <a class="new" href="/w/index.php?title=%ED%85%8D%ED%86%A0%EC%82%AC%EA%B2%8C%EC%8A%A4&amp;action=edit&amp;redlink=1" title="텍토사게스 (없는 문서)">텍토사게스</a>(라틴어 Tectosages)인들의 중심지가 되었다.</p>
<h3><span class="mw-headline" id=".EB.A1.9C.EB.A7.88_.EC.8B.9C.EB.8C.80">로마 시대</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%95%99%EC%B9%B4%EB%9D%BC&amp;action=edit&amp;section=4" title="부분 편집: 로마 시대">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p><a href="/wiki/%EA%B8%B0%EC%9B%90%EC%A0%84_189%EB%85%84" title="기원전 189년">기원전 189년</a>에는 로마의 집정관 '그나이우스 만리우스 불소'가 앙카라를 점령하여 갈라티아인들에 대한 군사활동의 근거지로 삼았다. <a href="/wiki/%EA%B8%B0%EC%9B%90%EC%A0%84_63%EB%85%84" title="기원전 63년">기원전 63년</a>에는 <a class="mw-redirect" href="/wiki/%ED%8F%BC%ED%8E%98%EC%9D%B4%EC%9A%B0%EC%8A%A4" title="폼페이우스">폼페이우스</a>가 다른 텍토사게스 영토와 함께 한 수장 아래 두어 한동안 자치가 이루어지다가 <a href="/wiki/%EA%B8%B0%EC%9B%90%EC%A0%84_25%EB%85%84" title="기원전 25년">기원전 25년</a>에 <a href="/wiki/%EC%95%84%EC%9A%B0%EA%B5%AC%EC%8A%A4%ED%88%AC%EC%8A%A4" title="아우구스투스">아우구스투스</a> 황제가 앙카라를 <a href="/wiki/%EB%A1%9C%EB%A7%88_%EC%A0%9C%EA%B5%AD" title="로마 제국">로마 제국</a>의 갈라티아주 수도로 삼았다.</p>
<p>이때 앙카라의 주민구성은 그리스인, <a href="/wiki/%EC%9C%A0%EB%8C%80%EC%9D%B8" title="유대인">유대인</a>, 로마인, 로마화된 켈트인들을 포함하고 <a href="/wiki/%EA%B7%B8%EB%A6%AC%EC%8A%A4%EC%96%B4" title="그리스어">그리스어</a>가 사용되었으나 그리스화된 헬레니즘 도시가 되지는 않은 상태였다. <a href="/wiki/19%EB%85%84" title="19년">기원후 19년</a>경 그리스인 지리학자 <a href="/wiki/%EC%8A%A4%ED%8A%B8%EB%9D%BC%EB%B3%B8" title="스트라본">스트라본</a>은 앙카라를 도시가 아니라 요새라고 불러 그리스·로마 도시 수준의 공공시설 등을 갖추지 못하였다고 암시하고 있다.</p>
<p>비문이나 주화에 남겨진 기록으로 유추해 볼 때 당시 앙카라의 문화는 켈트 바탕에 로마의 사상과 관습이 얹힌 형태였다고 여겨진다. <a href="/wiki/150%EB%85%84" title="150년">150년</a>경에야 진정한 헬레니즘이 싹트기 시작한다. <a href="/wiki/%EA%B8%B0%EB%8F%85%EA%B5%90" title="기독교">기독교</a>가 북쪽이나 서북쪽으로부터 전파된 시기는 이르면 <a href="/wiki/1%EC%84%B8%EA%B8%B0" title="1세기">1세기</a> 정도로 생각된다. 하지만 <a href="/wiki/192%EB%85%84" title="192년">192년</a> 처음 기록이 남아있는 안키라 교회를 사도 <a class="mw-redirect" href="/wiki/%EB%B0%94%EC%9A%B8%EB%A1%9C" title="바울로">바울로</a>가 세웠다는 증거는 없고 그가 갈라티아 북부에 다녀갔다는 증거도 없다. 단지 <a href="/wiki/%EA%B0%88%EB%9D%BC%EB%94%94%EC%95%84%EC%9D%B8%EB%93%A4%EC%97%90%EA%B2%8C_%EB%B3%B4%EB%82%B8_%ED%8E%B8%EC%A7%80" title="갈라디아인들에게 보낸 편지">갈라디아인들에게 보낸 편지</a>를 통해서 갈라디아교회와 사도 바울로간의 관계를 짐작할 수 있다.</p>
<p>앙카라의 위상은 콘스탄티노플이 로마 제국의 국제 도시가 되고 나서야 가까운 지리조건 덕에 크게 올라갔다. 그 후 중세에도 앙카라는 여전히 중요한 도시로의 위치를 고수하였다. <a href="/wiki/%EB%B9%84%EC%9E%94%ED%8B%B0%EC%9B%80_%EC%A0%9C%EA%B5%AD" title="비잔티움 제국">비잔티움 제국</a>의 도시였을 때는 페르시아인들과 아랍인들의 공격을 받았다.</p>
<h3><span class="mw-headline" id=".ED.8A.80.EB.A5.B4.ED.81.AC_.EC.A0.9C.EA.B5.AD_.EC.8B.9C.EB.8C.80">튀르크 제국 시대</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%95%99%EC%B9%B4%EB%9D%BC&amp;action=edit&amp;section=5" title="부분 편집: 튀르크 제국 시대">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p><a href="/wiki/1073%EB%85%84" title="1073년">1073년</a>에는 <a class="mw-redirect" href="/wiki/%EC%85%80%EC%A3%BC%ED%81%AC_%ED%8A%80%EB%A5%B4%ED%81%AC" title="셀주크 튀르크">셀주크 튀르크</a>에게 점령을 당하였다. <a href="/wiki/%EC%8B%AD%EC%9E%90%EA%B5%B0" title="십자군">십자군</a> 원정을 온 툴루즈의 레이몽 4세가 <a href="/wiki/1101%EB%85%84" title="1101년">1101년</a> 셀주크 튀르크를 몰아냈으나 비잔틴 제국은 앙카라를 지배할 만한 힘이 없어 그 후 셀주크 튀르크와 다른 경쟁 세력들은 앙카라를 두고 서로 싸웠다.</p>
<p><a href="/wiki/1356%EB%85%84" title="1356년">1356년</a>에는 <a href="/wiki/%EC%98%A4%EC%8A%A4%EB%A7%8C_%EC%A0%9C%EA%B5%AD" title="오스만 제국">오스만 제국</a>의 2대 술탄인 <a href="/wiki/%EC%98%A4%EB%A5%B4%ED%95%9C_1%EC%84%B8" title="오르한 1세">오르한 1세</a>가 앙카라를 정복하였다. <a href="/wiki/%ED%8B%B0%EB%AC%B4%EB%A5%B4_%EC%A0%9C%EA%B5%AD" title="티무르 제국">티무르 제국</a>의 <a href="/wiki/%ED%8B%B0%EB%AC%B4%EB%A5%B4" title="티무르">티무르</a>가 아나톨리아 원정 때 앙카라를 포위 공격해 <a href="/wiki/1402%EB%85%84" title="1402년">1402년</a> 빼앗았다. 그러나 <a href="/wiki/1403%EB%85%84" title="1403년">1403년</a>에 앙카라는 다시 오스만의 지배하에 돌아와서 <a href="/wiki/%EC%A0%9C1%EC%B0%A8_%EC%84%B8%EA%B3%84_%EB%8C%80%EC%A0%84" title="제1차 세계 대전">제1차 세계 대전</a> 때까지 오스만 제국이 지배하고 있었다.</p>
<p>1차 대전의 종반 무렵, 현재의 <a href="/wiki/%ED%84%B0%ED%82%A4" title="터키">터키</a> 지방은 오스만 술탄의 지배하에 있었으나 <a href="/wiki/%EA%B7%B8%EB%A6%AC%EC%8A%A4" title="그리스">그리스</a> 군의 침공을 받고 있는 상태였다. 터키 민족주의 지도자 케말 파샤(후의 <a class="mw-redirect" href="/wiki/%EC%BC%80%EB%A7%90_%EC%95%84%ED%83%80%ED%8A%80%EB%A5%B4%ED%81%AC" title="케말 아타튀르크">케말 아타튀르크</a>)는 그의 저항운동 본부를 <a href="/wiki/1919%EB%85%84" title="1919년">1919년</a> 앙카라에 세웠다(<a class="mw-redirect" href="/wiki/%ED%84%B0%ED%82%A4_%EB%8F%85%EB%A6%BD%EC%A0%84%EC%9F%81" title="터키 독립전쟁">터키 독립전쟁</a> 참조). <a href="/wiki/1923%EB%85%84" title="1923년">1923년</a> <a href="/wiki/%ED%84%B0%ED%82%A4" title="터키">터키 공화국</a>이 수립되고 앙카라가 <a href="/wiki/%EC%9D%B4%EC%8A%A4%ED%83%84%EB%B6%88" title="이스탄불">이스탄불</a>(당시 <a href="/wiki/%EC%BD%98%EC%8A%A4%ED%83%84%ED%8B%B0%EB%85%B8%ED%8F%B4%EB%A6%AC%EC%8A%A4" title="콘스탄티노폴리스">콘스탄티노폴리스</a>) 대신 터키의 수도로 정해졌다.</p>
<h3><span class="mw-headline" id=".ED.84.B0.ED.82.A4_.EA.B3.B5.ED.99.94.EA.B5.AD_.EC.8B.9C.EB.8C.80">터키 공화국 시대</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%95%99%EC%B9%B4%EB%9D%BC&amp;action=edit&amp;section=6" title="부분 편집: 터키 공화국 시대">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>앙카라가 <a class="mw-redirect" href="/wiki/%ED%84%B0%ED%82%A4_%EA%B3%B5%ED%99%94%EA%B5%AD" title="터키 공화국">터키 공화국</a>의 수도가 된 이후, 구시가지는 <a href="/wiki/%EC%9A%B8%EB%A3%A8%EC%8A%A4" title="울루스">울루스</a>(Ulus)로, 신시가지는 예니셰히르로 불리게 되었다. 로마시대, 비잔틴시대, 오스만 제국 시대의 유적들과 오래된 시장, 옛 관공서 등은 울루스에, 대로, 신식 호텔, 극장, 쇼핑몰, 신식 <a class="mw-redirect" href="/wiki/%EA%B4%80%EA%B3%B5%EC%84%9C" title="관공서">관공서</a>, <a href="/wiki/%EB%8C%80%EC%82%AC%EA%B4%80" title="대사관">대사관</a> 등은 <a href="/wiki/%ED%81%AC%EC%A6%90%EB%9D%BC%EC%9D%B4" title="크즐라이">크즐라이</a>(Kızılay)를 중심으로 하는 신시가지에 위치하게 됐다. 그 이후, 앙카라는 <a class="mw-redirect" href="/wiki/%ED%84%B0%ED%82%A4_%EA%B3%B5%ED%99%94%EA%B5%AD" title="터키 공화국">터키 공화국</a>의 <a href="/wiki/%EC%88%98%EB%8F%84" title="수도">수도</a>로서 눈부신 발전을 이룩했다. 1924년에는 앙카라의 인구는 35,000명 밖에 되지 않았다. <a href="/wiki/1950%EB%85%84" title="1950년">1950년</a>에는 286,781명이 사는 도시가 되었고, <a href="/wiki/2014%EB%85%84" title="2014년">2014년</a>에는 5,150,072명의 인구가 사는 대도시가 되었고, 현재 <a href="/wiki/%ED%84%B0%ED%82%A4" title="터키">터키</a>에서 두 번째로 인구가 많은 도시가 됐다.</p>
<p><a href="/wiki/2015%EB%85%84" title="2015년">2015년</a> <a href="/wiki/10%EC%9B%94_10%EC%9D%BC" title="10월 10일">10월 10일</a> 현지시각 오전 10시 4분에(EEST) <a href="/wiki/%EC%95%99%EC%B9%B4%EB%9D%BC_%EC%A4%91%EC%95%99%EC%97%AD" title="앙카라 중앙역">앙카라 중앙역</a> 앞에서 <a href="/wiki/2015%EB%85%84_%EC%95%99%EC%B9%B4%EB%9D%BC_%ED%8F%AD%ED%83%84_%ED%85%8C%EB%9F%AC" title="2015년 앙카라 폭탄 테러">테러</a>가 발생했다. <a href="/wiki/2015%EB%85%84_%EC%95%99%EC%B9%B4%EB%9D%BC_%ED%8F%AD%ED%83%84_%ED%85%8C%EB%9F%AC" title="2015년 앙카라 폭탄 테러">이 테러</a>로 102명이 사망하고, 400명이 넘는 사람들이 다쳤다. 이 테러는 <a href="/wiki/%ED%84%B0%ED%82%A4_%EA%B3%B5%ED%99%94%EA%B5%AD%EC%9D%98_%EC%97%AD%EC%82%AC" title="터키 공화국의 역사">터키 공화국의 역사</a>상 최악의 인명피해를 낸 테러였다.<sup class="reference" id="cite_ref-1"><a href="#cite_note-1">[1]</a></sup></p>

<h2><span class="mw-headline" id=".EA.B8.B0.ED.9B.84">기후</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%95%99%EC%B9%B4%EB%9D%BC&amp;action=edit&amp;section=7" title="부분 편집: 기후">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>앙카라는 <a href="/wiki/%EC%A4%91%EC%95%99%EC%95%84%EB%82%98%ED%86%A8%EB%A6%AC%EC%95%84_%EC%A7%80%EC%97%AD" title="중앙아나톨리아 지역">중앙아나톨리아 지역</a>, 즉 내륙에 위치하다보니 <a href="/wiki/%EB%8C%80%EB%A5%99%EC%84%B1_%EA%B8%B0%ED%9B%84" title="대륙성 기후">대륙성 기후</a>를 띈다. 여름에는 덥고 건조한 날씨가, 겨울에는 춥고 눈이 많이 온다. 주로 가을과 봄에 비가 내린다. 앙카라는 고지대에 위치하다보니 여름에는 낮에 덥고 건조하지만, 밤에는 시원하다. 연간 강수량은 402mm 이고 1월 평균기온은 0.4°C, 7월 평균기온은 23.6°C이다.</p>

<h2><span class="mw-headline" id=".EC.9D.B8.EA.B5.AC">인구</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%95%99%EC%B9%B4%EB%9D%BC&amp;action=edit&amp;section=8" title="부분 편집: 인구">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<div class="thumb tleft">
<div class="thumbinner" style="width:177px;"><a class="image" href="/wiki/%ED%8C%8C%EC%9D%BC:Ankara_at_night.jpg"><img alt="" class="thumbimage" data-file-height="426" data-file-width="449" height="166" src="//upload.wikimedia.org/wikipedia/commons/thumb/b/ba/Ankara_at_night.jpg/175px-Ankara_at_night.jpg" srcset="//upload.wikimedia.org/wikipedia/commons/thumb/b/ba/Ankara_at_night.jpg/263px-Ankara_at_night.jpg 1.5x, //upload.wikimedia.org/wikipedia/commons/thumb/b/ba/Ankara_at_night.jpg/350px-Ankara_at_night.jpg 2x" width="175"/></a>
<div class="thumbcaption">
<div class="magnify"><a class="internal" href="/wiki/%ED%8C%8C%EC%9D%BC:Ankara_at_night.jpg" title="실제 크기로"></a></div>
앙카라 광역권 위성사진</div>
</div>
</div>
<p><a href="/wiki/1923%EB%85%84" title="1923년">1923년</a> <a href="/wiki/%ED%84%B0%ED%82%A4" title="터키">터키</a>의 수도가 될 당시 앙카라는 원래 인구 50만 명을 수용할 수 있는 도시로 계획이 되었다. <a href="/wiki/1927%EB%85%84" title="1927년">1927년</a> 앙카라의 인구는 75,000명에 불과했지만, <a href="/wiki/1950%EB%85%84%EB%8C%80" title="1950년대">1950년대</a>에 들어서면서 지방에서 많은 사람들이 이주하여 인구가 폭증하기 시작하였다. <a href="/wiki/2013%EB%85%84" title="2013년">2013년</a>에는 5,045,083명(남자 2,507,525명, 여자 2,537,558명)으로 집계되었다.<sup class="reference" id="cite_ref-Turkey_pop_2007_3-0"><a href="#cite_note-Turkey_pop_2007-3">[3]</a></sup></p>
<h2><span class="mw-headline" id=".EA.B2.BD.EC.A0.9C">경제</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%95%99%EC%B9%B4%EB%9D%BC&amp;action=edit&amp;section=9" title="부분 편집: 경제">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<div class="thumb tleft">
<div class="thumbinner" style="width:177px;"><a class="image" href="/wiki/%ED%8C%8C%EC%9D%BC:A_view_of_the_BDDK_building,_Atat%C3%BCrk_Avenue_in_Ankara.jpg"><img alt="" class="thumbimage" data-file-height="2048" data-file-width="1536" height="233" src="//upload.wikimedia.org/wikipedia/commons/thumb/9/92/A_view_of_the_BDDK_building%2C_Atat%C3%BCrk_Avenue_in_Ankara.jpg/175px-A_view_of_the_BDDK_building%2C_Atat%C3%BCrk_Avenue_in_Ankara.jpg" srcset="//upload.wikimedia.org/wikipedia/commons/thumb/9/92/A_view_of_the_BDDK_building%2C_Atat%C3%BCrk_Avenue_in_Ankara.jpg/263px-A_view_of_the_BDDK_building%2C_Atat%C3%BCrk_Avenue_in_Ankara.jpg 1.5x, //upload.wikimedia.org/wikipedia/commons/thumb/9/92/A_view_of_the_BDDK_building%2C_Atat%C3%BCrk_Avenue_in_Ankara.jpg/350px-A_view_of_the_BDDK_building%2C_Atat%C3%BCrk_Avenue_in_Ankara.jpg 2x" width="175"/></a>
<div class="thumbcaption">
<div class="magnify"><a class="internal" href="/wiki/%ED%8C%8C%EC%9D%BC:A_view_of_the_BDDK_building,_Atat%C3%BCrk_Avenue_in_Ankara.jpg" title="실제 크기로"></a></div>
카바클르데르에 위치한 터키 은행감독청(BDDK) 빌딩</div>
</div>
</div>
<p>앙카라는 중요한 상업·공업도시이며 주변의 농업 지대에게는 중요한 시장이다.</p>
<p>역사적으로 <a href="/wiki/%EC%95%99%EA%B3%A0%EB%9D%BC_%EC%97%BC%EC%86%8C" title="앙고라 염소">앙고라 염소</a>털로 만든 모헤어, 앙고라 토끼 털로 만든 앙고라 울을 생산하는 섬유산업이 발달했었다. 또한, <a class="new" href="/w/index.php?title=%EC%A4%91%EC%95%99_%EC%95%84%EB%82%98%ED%86%A8%EB%A6%AC%EC%95%84_%EC%A7%80%EB%B0%A9&amp;action=edit&amp;redlink=1" title="중앙 아나톨리아 지방 (없는 문서)">중앙 아나톨리아 지방</a>에서 포도가 많이 생산되었기 때문에 와인 생산지로도 유명했다. 특히, 카바클데레 와인(Kavaklıdere wine)이 유명하다.</p>
<p>공업에서는 <a class="new" href="/w/index.php?title=TAI&amp;action=edit&amp;redlink=1" title="TAI (없는 문서)">TAI</a>(Turkish Aerospace Industries), MKE, 아셀산(ASELSAN), 하벨산(Havelsan), 로켓산(Roketsan), 누롤 마키나(Nurol Makina) 등의 항공산업 및 방위산업 관련 회사들의 본사가 위치하여있다. 이들 업체의 수출은 계속해서 늘어나고 있다. 또한, 독일의 버스, 트럭 제조 회사인 <a href="/wiki/MAN_SE" title="MAN SE">MAN SE</a>의 공장도 위치하여있다. 또한, 앙카라에 위치한 <a href="/wiki/%EC%95%99%EC%B9%B4%EB%9D%BC_%EB%8C%80%ED%95%99%EA%B5%90" title="앙카라 대학교">앙카라 대학교</a>, 중동공과 대학교, 빌켄트 대학교 등에서 배출한 인재들이 꾸준히 산업에 투입되고 있다.</p>
<h3><span class="mw-headline" id=".EC.87.BC.ED.95.91">쇼핑</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%95%99%EC%B9%B4%EB%9D%BC&amp;action=edit&amp;section=10" title="부분 편집: 쇼핑">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<div class="thumb tleft">
<div class="thumbinner" style="width:222px;"><a class="image" href="/wiki/%ED%8C%8C%EC%9D%BC:ANKAmall.JPG"><img alt="" class="thumbimage" data-file-height="1536" data-file-width="2048" height="165" src="//upload.wikimedia.org/wikipedia/commons/thumb/c/cc/ANKAmall.JPG/220px-ANKAmall.JPG" srcset="//upload.wikimedia.org/wikipedia/commons/thumb/c/cc/ANKAmall.JPG/330px-ANKAmall.JPG 1.5x, //upload.wikimedia.org/wikipedia/commons/thumb/c/cc/ANKAmall.JPG/440px-ANKAmall.JPG 2x" width="220"/></a>
<div class="thumbcaption">
<div class="magnify"><a class="internal" href="/wiki/%ED%8C%8C%EC%9D%BC:ANKAmall.JPG" title="실제 크기로"></a></div>
<a href="/wiki/%EC%95%99%EC%B9%B4%EB%AA%B0" title="앙카몰">앙카몰</a>의 전경</div>
</div>
</div>
<p>외국인 관광객들이 많이 찾는 시장은 <a href="/wiki/%EC%9A%B8%EB%A3%A8%EC%8A%A4" title="울루스">울루스</a>에 위치한 츠크르크츠라르 요쿠수(<i>Çıkrıkçılar Yokuşu</i>)로 주로 <a class="mw-redirect" href="/wiki/%EC%96%91%ED%83%84%EC%9E%90" title="양탄자">양탄자</a>, 가죽 제품 등을 판매한다. 바클크라르 시장(Bakırcılar Çarşısı)은 장식품, 카펫, 골동품 등으로 유명한 시장이다. 앙카라성 근처에도 <a href="/wiki/%ED%96%A5%EC%8B%A0%EB%A3%8C" title="향신료">향신료</a>, <a href="/wiki/%EA%B1%B4%EA%B3%BC" title="건과">말린 과일</a>, <a class="mw-redirect" href="/wiki/%EA%B2%AC%EA%B3%BC%EB%A5%98" title="견과류">견과류</a> 등을 판매하는 시장이 있다. 또한, <a class="new" href="/w/index.php?title=%EC%8A%A4%ED%9E%88%EC%98%88&amp;action=edit&amp;redlink=1" title="스히예 (없는 문서)">스히예</a> 광장 주변에는 매일 전통시장이 열린다.</p>
<p>현대적인 쇼핑몰은 <a href="/wiki/%ED%81%AC%EC%A6%90%EB%9D%BC%EC%9D%B4" title="크즐라이">크즐라이</a>나 <a href="/wiki/%EC%B0%AC%EC%B9%B4%EC%95%BC" title="찬카야">찬카야</a>의 투나르 힐미 에비뉴에서 주로 볼 수 있다. 크즐라이역 주변에는 크즐라이 쇼핑몰이 있는데 푸드코드, 스타벅스, 옷가게가 입점해있다. <a class="new" href="/w/index.php?title=%EC%95%84%ED%8A%B8%EB%A6%AC%EC%9B%80_%EB%AA%B0&amp;action=edit&amp;redlink=1" title="아트리움 몰 (없는 문서)">아트리움 몰</a>(Atrium Mall) 옆에 바로 옆에 있는 <a class="new" href="/w/index.php?title=%EC%95%84%ED%83%80%EC%BF%A8%EB%A0%88_%ED%83%80%EC%9B%8C&amp;action=edit&amp;redlink=1" title="아타쿨레 타워 (없는 문서)">아타쿨레 타워</a>의 꼭대기 층에는 <a class="mw-redirect" href="/wiki/%EB%A0%88%EC%8A%A4%ED%86%A0%EB%9E%91" title="레스토랑">레스토랑</a>이 있다. <a class="new" href="/w/index.php?title=A%C5%9ET%C4%B0%EC%97%AD&amp;action=edit&amp;redlink=1" title="AŞTİ역 (없는 문서)">AŞTİ역</a> 주변에는 아르마다 쇼핑 몰이 위치하고 있다. 또한, <a href="/wiki/%EC%BC%80%EC%B9%98%EC%99%B8%EB%A0%8C" title="케치외렌">케치외렌</a>의 에트릭 지역에 대형 슈퍼마켓, 오락실, H&amp;M 등이 입점해있는 <a class="new" href="/w/index.php?title=%EC%95%88%ED%83%80%EB%A0%88%EC%8A%A4_%EC%87%BC%ED%95%91%EB%AA%B0&amp;action=edit&amp;redlink=1" title="안타레스 쇼핑몰 (없는 문서)">안타레스 쇼핑몰</a>이 위치하고 있다.</p>
<p><a class="new" href="/w/index.php?title=%EC%95%84%EB%A5%B4%EB%A7%88%EB%8B%A4&amp;action=edit&amp;redlink=1" title="아르마다 (없는 문서)">아르마다</a>, <a class="new" href="/w/index.php?title=CEPA&amp;action=edit&amp;redlink=1" title="CEPA (없는 문서)">CEPA</a>, 켄트파크 쇼핑 몰이 고속도로 변에 있고, 갈레리아, 알자디움, 골디온이 위밋쾨이에 위치한다. H&amp;M, <a class="mw-redirect" href="/wiki/ZARA" title="ZARA">ZARA</a> 등의 브랜드가 입점해 있는 <a href="/wiki/%EC%95%99%EC%B9%B4%EB%AA%B0" title="앙카몰">앙카몰</a>은 앙카라에서 가장 큰 쇼핑몰로 <a class="new" href="/w/index.php?title=%EC%95%84%ED%81%AC%EC%BE%A8%ED%94%84%EB%A4%BC%EC%97%AD&amp;action=edit&amp;redlink=1" title="아크쾨프뤼역 (없는 문서)">아크쾨프뤼역</a> 바로 앞에 위치해 있다.</p>
<h2><span class="mw-headline" id=".EC.A0.95.EC.B9.98">정치</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%95%99%EC%B9%B4%EB%9D%BC&amp;action=edit&amp;section=11" title="부분 편집: 정치">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>앙카라는 정치적으로 이슬람주의 보수당인 <a href="/wiki/%EC%A0%95%EC%9D%98%EA%B0%9C%EB%B0%9C%EB%8B%B9_(%ED%84%B0%ED%82%A4)" title="정의개발당 (터키)">정의개발당</a>(AKP, Adalet ve Kalkınma Partisi) <a href="/wiki/%EC%BC%80%EB%A7%90%EC%A3%BC%EC%9D%98" title="케말주의">케말주의</a> 중도좌파인 <a href="/wiki/%EA%B3%B5%ED%99%94%EC%9D%B8%EB%AF%BC%EB%8B%B9" title="공화인민당">공화인민당</a>(CHP, Cumhuriyet Halk Partisi) 민족주의 극우인 <a class="new" href="/w/index.php?title=%EB%AF%BC%EC%A1%B1%EC%A3%BC%EC%9D%98%EC%9E%90%EC%9A%B4%EB%8F%99%EB%8B%B9&amp;action=edit&amp;redlink=1" title="민족주의자운동당 (없는 문서)">민족주의자운동당</a>(MHP, Milliyetçi Hareket Partisi)가 의석을 나누어 갖고 있다. 선거구 수는 25개이다. 2014년 선거 결과 <a href="/wiki/%EA%B3%B5%ED%99%94%EC%9D%B8%EB%AF%BC%EB%8B%B9" title="공화인민당">공화인민당</a>은 2개의 의석을 차지하고 있으나, 앙카라에서 가장 인구가 많은 <a href="/wiki/%EC%B0%AC%EC%B9%B4%EC%95%BC" title="찬카야">찬카야</a>(Çankaya)에 의석을 가지고 있다. <a class="new" href="/w/index.php?title=%EB%AF%BC%EC%A1%B1%EC%A3%BC%EC%9D%98%EC%9E%90%EC%9A%B4%EB%8F%99%EB%8B%B9&amp;action=edit&amp;redlink=1" title="민족주의자운동당 (없는 문서)">민족주의자운동당</a>은 2개의 의석을 갖고 있다. 전반적으로 <a href="/wiki/%EC%A0%95%EC%9D%98%EA%B0%9C%EB%B0%9C%EB%8B%B9_(%ED%84%B0%ED%82%A4)" title="정의개발당 (터키)">정의개발당</a>이 많은 의석을 차지하고 있다. 앙카라는 <a href="/wiki/%EC%9D%B4%EC%A6%88%EB%AF%B8%EB%A5%B4" title="이즈미르">이즈미르</a>나 <a href="/wiki/%EC%9D%B4%EC%8A%A4%ED%83%84%EB%B6%88" title="이스탄불">이스탄불</a>과는 달리 정치적으로 보수적인 경향을 보인다. 하지만, <a href="/wiki/2013%EB%85%84_%ED%84%B0%ED%82%A4_%EB%B0%98%EC%A0%95%EB%B6%80_%EC%8B%9C%EC%9C%84" title="2013년 터키 반정부 시위">2013년 터키 반정부 시위</a> 때 앙카라에서도 <a href="/wiki/%EC%9D%B4%EC%8A%A4%ED%83%84%EB%B6%88" title="이스탄불">이스탄불</a>, <a href="/wiki/%EC%9D%B4%EC%A6%88%EB%AF%B8%EB%A5%B4" title="이즈미르">이즈미르</a> 못지 않은 격렬한 시위가 발생했고, 다수의 사상자도 있었다.<sup class="reference" id="cite_ref-4"><a href="#cite_note-4">[4]</a></sup></p>
<p>메리흐 괵첵(Melih Gökçek)이 1994년부터 현재까지 앙카라의 시장으로 재임 중이며, 터키 복지당(RP, Refah Partisi)에서 선행당(FP, Fazilet Partisi)으로 그리고 <a href="/wiki/%EC%A0%95%EC%9D%98%EA%B0%9C%EB%B0%9C%EB%8B%B9_(%ED%84%B0%ED%82%A4)" title="정의개발당 (터키)">정의개발당</a>으로 당적을 옮겼다. <a class="new" href="/w/index.php?title=%EB%A9%94%EB%A6%AC%ED%9D%90_%EA%B4%B5%EC%B2%B5&amp;action=edit&amp;redlink=1" title="메리흐 괵첵 (없는 문서)">메리흐 괵첵</a>은 <a href="/wiki/1994%EB%85%84" title="1994년">1994년</a> 선거를 시작으로 1999년, 2004년, 2009년까지 당선되었으며, 2014년 선거에서도 당선되면서 5선 시장이 되었다. <a class="new" href="/w/index.php?title=%EB%AF%BC%EC%A1%B1%EC%A3%BC%EC%9D%98%EC%9E%90%EC%9A%B4%EB%8F%99%EB%8B%B9&amp;action=edit&amp;redlink=1" title="민족주의자운동당 (없는 문서)">민족주의자운동당</a>에서 <a href="/wiki/2009%EB%85%84" title="2009년">2009년</a> 보수 정치인인 <a class="new" href="/w/index.php?title=%EB%A7%8C%EC%88%98%EB%A5%B4_%EC%95%BC%EB%B0%94%EC%89%AC&amp;action=edit&amp;redlink=1" title="만수르 야바쉬 (없는 문서)">만수르 야바쉬</a>(Mansur Yavaş)를 내새웠으나 낙선했다. <a class="new" href="/w/index.php?title=%EB%A9%94%EB%A6%AC%ED%9D%90_%EA%B4%B5%EC%B2%B5&amp;action=edit&amp;redlink=1" title="메리흐 괵첵 (없는 문서)">메리흐 괵첵</a>은 2009년 선거에서 1% 차이로 승리했으며, <a class="new" href="/w/index.php?title=%EB%A7%8C%EC%88%98%EB%A5%B4_%EC%95%BC%EB%B0%94%EC%89%AC&amp;action=edit&amp;redlink=1" title="만수르 야바쉬 (없는 문서)">만수르 야바쉬</a>는 이를 조직적인 부정 선거라고 주장했다. 만수르 야바쉬는 터키 최고선거관리위원회에 제소했으나 기각당했고, 그 후에 <a href="/wiki/%EC%9C%A0%EB%9F%BD_%EC%9D%B8%EA%B6%8C_%EC%9E%AC%ED%8C%90%EC%86%8C" title="유럽 인권 재판소">유럽 인권 재판소</a>에 이를 다시 재소했다. 메리흐 괵첵이 5번이나 당선됐지만, <a href="/wiki/2009%EB%85%84" title="2009년">2009년</a> 선거는 상당수의 유권자들이 <a class="mw-redirect" href="/wiki/%EB%B6%80%EC%A0%95%EC%84%A0%EA%B1%B0" title="부정선거">부정선거</a>로 인식하고 있다.<sup class="reference" id="cite_ref-5"><a href="#cite_note-5">[5]</a></sup></p>

<p><a href="/wiki/%EC%95%99%EC%B9%B4%EB%9D%BC_%EC%A7%80%ED%95%98%EC%B2%A0" title="앙카라 지하철">앙카라 지하철</a>과 대부분의 버스는 EGO(The Electricity, Gas, Bus General Directorate)<sup class="reference" id="cite_ref-6"><a href="#cite_note-6">[6]</a></sup>에서 운영한다. 대다수의 버스는 <a href="/wiki/%EA%B5%90%ED%86%B5%EC%B9%B4%EB%93%9C" title="교통카드">교통카드</a>만을 이용해서 사용이 가능하지만, 일부 개인이 운영하는 버스에서는 현금으로만 탑승이 가능하다. 현재 앙카라에는 <a href="/wiki/%EA%B2%BD%EC%A0%84%EC%B2%A0" title="경전철">경전철</a>인 앙카라이(Ankaray, A1)와 M1, M2, M3까지 3개의 지하철이 운행중이며, M4가 건설중이다. 또한, 쉔테페와 예니마할레역을 연결하는 <a class="mw-redirect" href="/wiki/%EC%BC%80%EC%9D%B4%EB%B8%94%EC%B9%B4" title="케이블카">케이블카</a>도 운행 중이다.</p>
<p><a href="/wiki/%EC%95%99%EC%B9%B4%EB%9D%BC_%EC%A4%91%EC%95%99%EC%97%AD" title="앙카라 중앙역">앙카라 중앙역</a>(Ankara Railway Station)은 <a href="/wiki/%ED%84%B0%ED%82%A4" title="터키">터키</a> 철도 교통에서 매우 중요한 위치를 차지한다. 터키 국영철도에서 앙카라에서 이스탄불, 에스키쉐히르, 시바스 등 전국 각지로 여객 서비스를 제공한다. 또한, <a class="new" href="/w/index.php?title=%EC%8B%A0%EC%9E%94%EC%97%AD&amp;action=edit&amp;redlink=1" title="신잔역 (없는 문서)">신잔역</a>(Sincan)에서부터 <a class="new" href="/w/index.php?title=%EC%B9%B4%EC%95%BC%EC%89%AC%EC%97%AD&amp;action=edit&amp;redlink=1" title="카야쉬역 (없는 문서)">카야쉬역</a>(Kayaş)까지 <a href="/wiki/%EA%B4%91%EC%97%AD%EC%B2%A0%EB%8F%84" title="광역철도">광역철도</a>도 운행중이다. <a href="/wiki/2009%EB%85%84" title="2009년">2009년</a> <a href="/wiki/3%EC%9B%94_13%EC%9D%BC" title="3월 13일">3월 13일</a>에 앙카라와 에스키쉐히르를 잇는, 2011년 8월 23일에는 앙카라와 코냐간, 2014년 7월 25일에는 앙카라와 <a href="/wiki/%EC%9D%B4%EC%8A%A4%ED%83%84%EB%B6%88" title="이스탄불">이스탄불</a>을 3시간 30분만에 연결하는 <a class="mw-redirect" href="/wiki/%EA%B3%A0%EC%86%8D%EC%97%B4%EC%B0%A8" title="고속열차">고속열차</a>인 <a href="/wiki/YHT" title="YHT">YHT</a>(Yüksek Hızlı Tren)가 각각 개통되었다.<sup class="reference" id="cite_ref-7"><a href="#cite_note-7">[7]</a></sup></p>
<p>앙카라는 또한 국토의 중앙에 위치하여 있기에 버스 교통에서도 중요한 위치를 차지한다. <a href="/wiki/%EB%B2%84%EC%8A%A4_%ED%84%B0%EB%AF%B8%EB%84%90" title="버스 터미널">버스 터미널</a>인 AŞTİ(Ankara Şehirlerarası Terminal İşletmesi)는 거의 모든 노선을 갖추고 있다.</p>
<p>앙카라의 공항은 시 북서쪽에 위치한 <a href="/wiki/%EC%97%90%EC%84%BC%EB%B3%B4%EC%95%84_%EA%B5%AD%EC%A0%9C%EA%B3%B5%ED%95%AD" title="에센보아 국제공항">에센보아 국제공항</a>이 있다. 아타튀르크 국제공항이나 <a href="/wiki/%EC%82%AC%EB%B9%84%ED%95%98_%EA%B4%B5%EC%B2%B8_%EA%B5%AD%EC%A0%9C%EA%B3%B5%ED%95%AD" title="사비하 괵첸 국제공항">사비하 괵첸 국제공항</a>에 비해 노선수가 적지만 터키 국내선과 <a class="mw-redirect" href="/wiki/%ED%94%84%EB%9E%91%ED%81%AC%ED%91%B8%EB%A5%B4%ED%8A%B8" title="프랑크푸르트">프랑크푸르트</a>, <a href="/wiki/%EB%AA%A8%EC%8A%A4%ED%81%AC%EB%B0%94" title="모스크바">모스크바</a> 등 유럽 주요 도시를 연결하는 노선이 있다.</p>

<h2><span class="mw-headline" id=".EC.8A.A4.ED.8F.AC.EC.B8.A0">스포츠</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%95%99%EC%B9%B4%EB%9D%BC&amp;action=edit&amp;section=14" title="부분 편집: 스포츠">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<div class="thumb tright">
<div class="thumbinner" style="width:222px;"><a class="image" href="/wiki/%ED%8C%8C%EC%9D%BC:Ankara_Arena_Genel_G%C3%B6r%C3%BCn%C3%BCm.JPG"><img alt="" class="thumbimage" data-file-height="2122" data-file-width="3184" height="147" src="//upload.wikimedia.org/wikipedia/commons/thumb/1/1d/Ankara_Arena_Genel_G%C3%B6r%C3%BCn%C3%BCm.JPG/220px-Ankara_Arena_Genel_G%C3%B6r%C3%BCn%C3%BCm.JPG" srcset="//upload.wikimedia.org/wikipedia/commons/thumb/1/1d/Ankara_Arena_Genel_G%C3%B6r%C3%BCn%C3%BCm.JPG/330px-Ankara_Arena_Genel_G%C3%B6r%C3%BCn%C3%BCm.JPG 1.5x, //upload.wikimedia.org/wikipedia/commons/thumb/1/1d/Ankara_Arena_Genel_G%C3%B6r%C3%BCn%C3%BCm.JPG/440px-Ankara_Arena_Genel_G%C3%B6r%C3%BCn%C3%BCm.JPG 2x" width="220"/></a>
<div class="thumbcaption">
<div class="magnify"><a class="internal" href="/wiki/%ED%8C%8C%EC%9D%BC:Ankara_Arena_Genel_G%C3%B6r%C3%BCn%C3%BCm.JPG" title="실제 크기로"></a></div>
<a href="/wiki/%EC%95%99%EC%B9%B4%EB%9D%BC_%EC%95%84%EB%A0%88%EB%82%98" title="앙카라 아레나">앙카라 아레나</a></div>
</div>
</div>
<p><a href="/wiki/%ED%84%B0%ED%82%A4" title="터키">터키</a>의 타지역처럼 <a href="/wiki/%EC%B6%95%EA%B5%AC" title="축구">축구</a>가 가장 인기 있는 스포츠이다. 1923년 창단된 <a href="/wiki/%EA%B2%90%EC%B8%A8%EB%A0%88%EB%A5%B4%EB%B9%84%EB%A5%BC%EB%A6%AC%EC%9D%B4_SK" title="겐츨레르비를리이 SK">겐츨레르비를리이 SK</a>가 앙카라를 연고지로 하는 축구팀으로 <a href="/wiki/%EC%89%AC%ED%8E%98%EB%A5%B4%EB%A6%AC%EA%B7%B8" title="쉬페르리그">쉬페르리그</a>에서 활동중이다. <a href="/wiki/%EA%B2%90%EC%B8%A8%EB%A0%88%EB%A5%B4%EB%B9%84%EB%A5%BC%EB%A6%AC%EC%9D%B4_SK" title="겐츨레르비를리이 SK">겐츨레르비를리이 SK</a>는 <a href="/wiki/1987%EB%85%84" title="1987년">1987년</a>과 <a href="/wiki/2001%EB%85%84" title="2001년">2001년</a>에 <a href="/wiki/%ED%8A%80%EB%A5%B4%ED%82%A4%EC%98%88_%EC%BF%A0%ED%8C%8C%EC%8A%A4" title="튀르키예 쿠파스">튀르키예 쿠파스</a>의 우승팀이었다. 또한, <a href="/wiki/1910%EB%85%84" title="1910년">1910년</a> 창단된 <a href="/wiki/MKE_%EC%95%99%EC%B9%B4%EB%9D%BC%EA%B7%80%EC%A5%90" title="MKE 앙카라귀쥐">MKE 앙카라귀쥐</a>는 가장 오래된 축구팀이지만 현재는 <a class="new" href="/w/index.php?title=TTF_2._%EB%A6%AC%EA%B7%B8&amp;action=edit&amp;redlink=1" title="TTF 2. 리그 (없는 문서)">TTF 2. 리그</a>에 소속되어있다. <a href="/wiki/MKE_%EC%95%99%EC%B9%B4%EB%9D%BC%EA%B7%80%EC%A5%90" title="MKE 앙카라귀쥐">MKE 앙카라귀쥐</a>는 <a href="/wiki/1972%EB%85%84" title="1972년">1972년</a>과 <a href="/wiki/1981%EB%85%84" title="1981년">1981년</a> <a href="/wiki/%ED%8A%80%EB%A5%B4%ED%82%A4%EC%98%88_%EC%BF%A0%ED%8C%8C%EC%8A%A4" title="튀르키예 쿠파스">튀르키예 쿠파스</a>의 우승팀이었다. <a class="new" href="/w/index.php?title=%EA%B2%90%EC%B8%A8%EB%A0%88%EB%A5%B4%EB%B9%84%EB%A5%B4_B%ED%8C%80&amp;action=edit&amp;redlink=1" title="겐츨레르비르 B팀 (없는 문서)">겐츨레르비르 B팀</a>과 <a href="/wiki/%ED%95%98%EC%A0%9C%ED%85%8C%ED%8E%98_SK" title="하제테페 SK">하제테페 SK</a>도 <a class="mw-redirect" href="/wiki/%EC%89%AC%ED%8E%98%EB%A5%B4_%EB%A6%AC%EA%B7%B8" title="쉬페르 리그">쉬페르 리그</a>에 소속되었던 적이 있다. 위의 모든 팀들은 <a class="new" href="/w/index.php?title=%EC%95%99%EC%B9%B4%EB%9D%BC_19_%EB%A7%88%EC%9C%BC%EC%8A%A4_%EC%8A%A4%ED%83%80%EB%94%94%EC%9B%80&amp;action=edit&amp;redlink=1" title="앙카라 19 마으스 스타디움 (없는 문서)">앙카라 19 마으스 스타디움</a>을 홈 구장으로 한다.<sup class="reference" id="cite_ref-8"><a href="#cite_note-8">[8]</a></sup> 또한 <a class="new" href="/w/index.php?title=%EC%98%A4%EC%8A%A4%EB%A7%8C%EC%8A%A4%ED%8F%AC%EB%A5%B4&amp;action=edit&amp;redlink=1" title="오스만스포르 (없는 문서)">오스만스포르</a>도 앙카라를 홈으로 하는 팀으로 2010년까지 쉬페르리그에 소속되어 있었다. 홈 구장은 예니켄트 아사쉬 스타디움으로 신잔에 위치한다.</p>
<p>또한, <a class="new" href="/w/index.php?title=%EC%8B%A0%EC%9E%94&amp;action=edit&amp;redlink=1" title="신잔 (없는 문서)">신잔</a>을 홈으로 하는 <a class="new" href="/w/index.php?title=%EB%B6%80%EA%B7%B8%EC%82%AC%EC%89%AC%EC%8A%A4%ED%8F%AC%EB%A5%B4&amp;action=edit&amp;redlink=1" title="부그사쉬스포르 (없는 문서)">부그사쉬스포르</a>, 에티메스구트를 홈으로 하는 에티메스구트 쉐케르스포르, 예니마할레를 홈으로 하는 튀르크 텔레콤스포르, 찬카야를 홈으로 하는 앙카라 데미르스포르, 케치외렌의 케치외렌귀쥐, 케치외렌스포르, 푸르사클라르스포르, <a class="new" href="/w/index.php?title=%EB%B0%94%EB%A3%B8%EC%8A%A4%ED%8F%AC%EB%A5%B4&amp;action=edit&amp;redlink=1" title="바룸스포르 (없는 문서)">바룸스포르</a>, <a class="new" href="/w/index.php?title=%EC%95%8C%ED%8A%BC%EB%8B%A5&amp;action=edit&amp;redlink=1" title="알튼닥 (없는 문서)">알튼닥</a>을 홈으로 하는 <a class="new" href="/w/index.php?title=%ED%8E%98%ED%8A%B8%EB%A1%A4_%EC%98%A4%ED%94%BC%EC%8B%9C_%EC%8A%A4%ED%8F%AC%EB%A5%B4&amp;action=edit&amp;redlink=1" title="페트롤 오피시 스포르 (없는 문서)">페트롤 오피시 스포르</a> 같은 군소 팀들도 존재한다.</p>
<p>또한, <a href="/wiki/%EB%86%8D%EA%B5%AC" title="농구">농구</a> 역시 인기 있는 스포츠이다. 앙카라에는 <a href="/wiki/%EC%95%99%EC%B9%B4%EB%9D%BC_%EC%95%84%EB%A0%88%EB%82%98" title="앙카라 아레나">앙카라 아레나</a>를 홈으로 하는 튀르크 텔레콤과 TOBB 스포츠 홀을 홈으로 하는 카사 테드 콜레즈리레르 두 개의 농구팀이 있다.</p>
<p>배구에서는 할크뱅크 앙카라가 통산 6회 터키 남자 배구 리그(Türkiye 1. Voleybol Ligi) 우승을 차지한 강팀이다.</p>
<p><a class="mw-redirect" href="/wiki/%EC%95%84%EC%9D%B4%EC%8A%A4_%EC%8A%A4%EC%BC%80%EC%9D%B4%ED%8C%85" title="아이스 스케이팅">아이스 스케이팅</a>과 <a href="/wiki/%EC%95%84%EC%9D%B4%EC%8A%A4%ED%95%98%ED%82%A4" title="아이스하키">아이스하키</a> 경기도 <a class="new" href="/w/index.php?title=%EC%95%99%EC%B9%B4%EB%9D%BC_%EB%B2%84%EC%A6%88_%ED%8C%8C%ED%85%8C%EB%8B%88_%EC%82%AC%EB%9D%BC%EC%9D%B4&amp;action=edit&amp;redlink=1" title="앙카라 버즈 파테니 사라이 (없는 문서)">앙카라 버즈 파테니 사라이</a>(Ankara Buz Pateni Sarayı)에서 열린다.</p>
<p><a href="/wiki/1980%EB%85%84%EB%8C%80" title="1980년대">1980년대</a> 이후로는 <a class="mw-redirect" href="/wiki/%EC%8A%A4%EC%BC%80%EC%9D%B4%ED%8A%B8_%EB%B3%B4%EB%93%9C" title="스케이트 보드">스케이트 보드</a>장도 많이 만들어졌다.</p>
<p><a href="/wiki/2012%EB%85%84" title="2012년">2012년</a> 완공된 <a class="new" href="/w/index.php?title=THF_%EC%8A%A4%ED%8F%AC%EC%B8%A0_%ED%99%80&amp;action=edit&amp;redlink=1" title="THF 스포츠 홀 (없는 문서)">THF 스포츠 홀</a>에서는 <a class="new" href="/w/index.php?title=%ED%84%B0%ED%82%A4_%ED%95%B8%EB%93%9C%EB%B3%BC_%EC%89%AC%ED%8E%98%EB%A5%B4_%EB%A6%AC%EA%B7%B8&amp;action=edit&amp;redlink=1" title="터키 핸드볼 쉬페르 리그 (없는 문서)">터키 핸드볼 쉬페르 리그</a>(Türkiye Hentbol Süper Ligi)와 여자 핸드볼 리그(Türkiye Kadınlar Hentbol Süper Ligi)가 개최된다.</p>
<h2><span class="mw-headline" id=".EA.B5.90.EC.9C.A1">교육</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%95%99%EC%B9%B4%EB%9D%BC&amp;action=edit&amp;section=15" title="부분 편집: 교육">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<h3><span class="mw-headline" id=".EB.8C.80.ED.95.99.EA.B5.90">대학교</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%95%99%EC%B9%B4%EB%9D%BC&amp;action=edit&amp;section=16" title="부분 편집: 대학교">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>앙카라는 <a href="/wiki/%EC%9D%B4%EC%8A%A4%ED%83%84%EB%B6%88" title="이스탄불">이스탄불</a>과 더불어 <a class="mw-redirect" href="/wiki/%EB%8C%80%ED%95%99%EA%B5%90" title="대학교">대학교</a>가 모여 있는 도시로 유명하다.</p>
<div class="div-col columns column-width" style="-moz-column-width: 20em; -webkit-column-width: 20em; column-width: 20em;">

<h2><span class="mw-headline" id=".EA.B3.B5.EC.9B.90">공원</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%95%99%EC%B9%B4%EB%9D%BC&amp;action=edit&amp;section=18" title="부분 편집: 공원">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>앙카라에는 도시 곳곳에 크고 작은 공원들이 있다. <a href="/wiki/%EC%9A%B8%EB%A3%A8%EC%8A%A4" title="울루스">울루스</a>에 위치한 <a href="/wiki/%EA%B2%90%EC%B8%A8%EB%A6%AD_%EA%B3%B5%EC%9B%90" title="겐츨릭 공원">겐츨릭 공원</a>(Gençlik Parkı)에는 놀이공원과 호수가 있어서 주말에 많은 사람들이 찾는다. 크즐라이에 위치한 귀벤 공원(Güven Park)에는 시미트와 꽃을 파는 행상들이 많이 있으며, 아타튀르크의 동상이 있다. 카바크르레데에 위치한 쿨루 공원(Kuğulu Park)에는 중국 정부에서 선물받은 백조와 흑조, 오리 등이 살고 있다. 알튼 공원(Altın Park)는 넓은 공원과 호수가 있으며, 배를 탈 수 있다. 앙카라 외곽지역인 에르야만에는 야경과 호수가 아름다운 괵수 공원(Göksu Park)가 있다.</p>
<p>또한, 앙카라 아레나 주변에는 한국전쟁에서 전사한 터키 군인을 기리기 위한 <a href="/wiki/%ED%95%9C%EA%B5%AD_%EA%B3%B5%EC%9B%90" title="한국 공원">한국 공원</a>(Kore Bahçesi)이 있다.</p>
<p>겐츨릭 공원은 1952년부터 <a href="/wiki/1976%EB%85%84" title="1976년">1976년</a>까지 100 <a href="/wiki/%ED%84%B0%ED%82%A4_%EB%A6%AC%EB%9D%BC" title="터키 리라">터키 리라</a>의 뒷면에 그려져있었다. 또한, 시내에 <a class="new" href="/w/index.php?title=%EC%95%84%ED%83%80%ED%8A%80%EB%A5%B4%ED%81%AC_%EC%98%A4%EB%A5%B4%EB%A7%8C_%EC%B9%98%ED%94%84%ED%8A%B8%EB%A6%AC%EC%9D%B4&amp;action=edit&amp;redlink=1" title="아타튀르크 오르만 치프트리이 (없는 문서)">아타튀르크 오르만 치프트리이</a>(Atatürk Orman Çiftliği)라는 <a class="mw-redirect" href="/wiki/%EC%95%84%ED%83%80%ED%8A%80%EB%A5%B4%ED%81%AC" title="아타튀르크">아타튀르크</a>가 생전에 농장일을하던 농원이 있는데, 이 곳에는 <a href="/wiki/%EB%8F%99%EB%AC%BC%EC%9B%90" title="동물원">동물원</a>과 농장, 레스토랑 등이 있다. 또한, <a class="mw-redirect" href="/wiki/%EC%95%84%ED%83%80%ED%8A%80%EB%A5%B4%ED%81%AC" title="아타튀르크">아타튀르크</a>가 태어난 <a href="/wiki/%EA%B7%B8%EB%A6%AC%EC%8A%A4" title="그리스">그리스</a> <a href="/wiki/%ED%85%8C%EC%82%B4%EB%A1%9C%EB%8B%88%ED%82%A4" title="테살로니키">테살로니키</a>에 위치한 생가를 완벽하게 복원해 놓은 집도 있다. <a href="/wiki/%EB%86%8D%EC%9E%A5" title="농장">농장</a>인 치프트릭(Çiftlik)에 방문하는 <a class="mw-redirect" href="/wiki/%EA%B4%80%EA%B4%91%EA%B0%9D" title="관광객">관광객</a>들은 농장에서 생산되는 치즈, 전통 맥주, 아이스크림 등을 맛 볼수 있으며, 주변에는 카페와 레스토랑들이 있다.</p>

<h2><span class="mw-headline" id=".EC.83.81.EC.A7.95.EB.AC.BC">상징물</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%95%99%EC%B9%B4%EB%9D%BC&amp;action=edit&amp;section=19" title="부분 편집: 상징물">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<h3><span class="mw-headline" id=".EC.95.99.EA.B3.A0.EB.9D.BC_.EA.B3.A0.EC.96.91.EC.9D.B4">앙고라 고양이</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%95%99%EC%B9%B4%EB%9D%BC&amp;action=edit&amp;section=20" title="부분 편집: 앙고라 고양이">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<div class="rellink relarticle mainarticle"><img alt="" data-file-height="128" data-file-width="128" height="16" src="//upload.wikimedia.org/wikipedia/commons/thumb/e/ec/Crystal_Clear_app_xmag.svg/16px-Crystal_Clear_app_xmag.svg.png" srcset="//upload.wikimedia.org/wikipedia/commons/thumb/e/ec/Crystal_Clear_app_xmag.svg/24px-Crystal_Clear_app_xmag.svg.png 1.5x, //upload.wikimedia.org/wikipedia/commons/thumb/e/ec/Crystal_Clear_app_xmag.svg/32px-Crystal_Clear_app_xmag.svg.png 2x" width="16"/> 이 부분의 본문은 <a href="/wiki/%ED%84%B0%ED%82%A4%EC%8B%9C_%EC%95%99%EA%B3%A0%EB%9D%BC" title="터키시 앙고라">터키시 앙고라</a>입니다.</div>
<div class="thumb tright">
<div class="thumbinner" style="width:222px;"><a class="image" href="/wiki/%ED%8C%8C%EC%9D%BC:Turkish_Angora_in_Ankara_Zoo_(AO%C3%87).JPG"><img alt="" class="thumbimage" data-file-height="826" data-file-width="1017" height="179" src="//upload.wikimedia.org/wikipedia/commons/thumb/7/76/Turkish_Angora_in_Ankara_Zoo_%28AO%C3%87%29.JPG/220px-Turkish_Angora_in_Ankara_Zoo_%28AO%C3%87%29.JPG" srcset="//upload.wikimedia.org/wikipedia/commons/thumb/7/76/Turkish_Angora_in_Ankara_Zoo_%28AO%C3%87%29.JPG/330px-Turkish_Angora_in_Ankara_Zoo_%28AO%C3%87%29.JPG 1.5x, //upload.wikimedia.org/wikipedia/commons/thumb/7/76/Turkish_Angora_in_Ankara_Zoo_%28AO%C3%87%29.JPG/440px-Turkish_Angora_in_Ankara_Zoo_%28AO%C3%87%29.JPG 2x" width="220"/></a>
<div class="thumbcaption">
<div class="magnify"><a class="internal" href="/wiki/%ED%8C%8C%EC%9D%BC:Turkish_Angora_in_Ankara_Zoo_(AO%C3%87).JPG" title="실제 크기로"></a></div>
2012년 1월 <a class="new" href="/w/index.php?title=%EC%95%99%EC%B9%B4%EB%9D%BC_%EB%8F%99%EB%AC%BC%EC%9B%90&amp;action=edit&amp;redlink=1" title="앙카라 동물원 (없는 문서)">앙카라 동물원</a>의 앙고라 고양이</div>
</div>
</div>
<p>앙카라는 세계적으로 유명한 <a href="/wiki/%EA%B3%A0%EC%96%91%EC%9D%B4" title="고양이">고양이</a> 품종인 <a href="/wiki/%ED%84%B0%ED%82%A4%EC%8B%9C_%EC%95%99%EA%B3%A0%EB%9D%BC" title="터키시 앙고라">터키시 앙고라</a>(Ankara kedisi)가 살던 지방이다. <a href="/wiki/%ED%84%B0%ED%82%A4%EC%8B%9C_%EC%95%99%EA%B3%A0%EB%9D%BC" title="터키시 앙고라">터키시 앙고라</a>는 주로 희고, 부드러운 긴 털을 갖고 있지만, 오늘날에는 품종 개량을 통해 검은털이나 붉은털을 가진 <a href="/wiki/%ED%84%B0%ED%82%A4%EC%8B%9C_%EC%95%99%EA%B3%A0%EB%9D%BC" title="터키시 앙고라">터키시 앙고라</a>도 많다. <a href="/wiki/%ED%84%B0%ED%82%A4%EC%8B%9C_%EC%95%99%EA%B3%A0%EB%9D%BC" title="터키시 앙고라">터키시 앙고라</a>는 <a href="/wiki/%ED%8E%98%EB%A5%B4%EC%8B%9C%EC%95%88" title="페르시안">페르시안 고양이</a>나 <a class="new" href="/w/index.php?title=%ED%84%B0%ED%82%A4%EC%8B%9C_%EB%B0%B4&amp;action=edit&amp;redlink=1" title="터키시 밴 (없는 문서)">터키시 밴</a>과도 가까운 품종이다. 눈은 파랑색, 녹색, 황색인 경우도 있지만 간혹가다 한쪽 눈과 다른쪽 눈의 색이 다른 <a class="mw-redirect" href="/wiki/%EC%98%A4%EB%93%9C%EC%95%84%EC%9D%B4" title="오드아이">오드아이</a>도 있다. 귀는 크고 눈은 아몬드 모양으로 생겼다.</p>
<h3><span class="mw-headline" id=".EC.95.99.EA.B3.A0.EB.9D.BC_.ED.86.A0.EB.81.BC">앙고라 토끼</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%95%99%EC%B9%B4%EB%9D%BC&amp;action=edit&amp;section=21" title="부분 편집: 앙고라 토끼">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<div class="rellink relarticle mainarticle"><img alt="" data-file-height="128" data-file-width="128" height="16" src="//upload.wikimedia.org/wikipedia/commons/thumb/e/ec/Crystal_Clear_app_xmag.svg/16px-Crystal_Clear_app_xmag.svg.png" srcset="//upload.wikimedia.org/wikipedia/commons/thumb/e/ec/Crystal_Clear_app_xmag.svg/24px-Crystal_Clear_app_xmag.svg.png 1.5x, //upload.wikimedia.org/wikipedia/commons/thumb/e/ec/Crystal_Clear_app_xmag.svg/32px-Crystal_Clear_app_xmag.svg.png 2x" width="16"/> 이 부분의 본문은 <a class="new" href="/w/index.php?title=%EC%95%99%EA%B3%A0%EB%9D%BC_%ED%86%A0%EB%81%BC&amp;action=edit&amp;redlink=1" title="앙고라 토끼 (없는 문서)">앙고라 토끼</a>입니다.</div>
<div class="thumb tright">
<div class="thumbinner" style="width:172px;"><a class="image" href="/wiki/%ED%8C%8C%EC%9D%BC:Joey_Giant_Angora_Buck.jpg"><img alt="" class="thumbimage" data-file-height="1728" data-file-width="2304" height="128" src="//upload.wikimedia.org/wikipedia/commons/thumb/3/35/Joey_Giant_Angora_Buck.jpg/170px-Joey_Giant_Angora_Buck.jpg" srcset="//upload.wikimedia.org/wikipedia/commons/thumb/3/35/Joey_Giant_Angora_Buck.jpg/255px-Joey_Giant_Angora_Buck.jpg 1.5x, //upload.wikimedia.org/wikipedia/commons/thumb/3/35/Joey_Giant_Angora_Buck.jpg/340px-Joey_Giant_Angora_Buck.jpg 2x" width="170"/></a>
<div class="thumbcaption">
<div class="magnify"><a class="internal" href="/wiki/%ED%8C%8C%EC%9D%BC:Joey_Giant_Angora_Buck.jpg" title="실제 크기로"></a></div>
<a class="new" href="/w/index.php?title=%EC%95%99%EA%B3%A0%EB%9D%BC_%ED%86%A0%EB%81%BC&amp;action=edit&amp;redlink=1" title="앙고라 토끼 (없는 문서)">앙고라 토끼</a></div>
</div>
</div>
<p><a class="new" href="/w/index.php?title=%EC%95%99%EA%B3%A0%EB%9D%BC_%ED%86%A0%EB%81%BC&amp;action=edit&amp;redlink=1" title="앙고라 토끼 (없는 문서)">앙고라 토끼</a>(Ankara tavşanı)는 앙카라 주변에 살던 길고, 부드러운 털을 가진 <a href="/wiki/%ED%86%A0%EB%81%BC" title="토끼">토끼</a>이다. <a class="new" href="/w/index.php?title=%EC%95%99%EA%B3%A0%EB%9D%BC_%ED%86%A0%EB%81%BC&amp;action=edit&amp;redlink=1" title="앙고라 토끼 (없는 문서)">앙고라 토끼</a>는 가장 오래된 <a href="/wiki/%EA%B0%80%EC%B6%95" title="가축">가축</a>용 토끼 중에 하나인데, <a class="mw-redirect" href="/wiki/%EC%A4%91%EC%84%B8%EC%8B%9C%EB%8C%80" title="중세시대">중세시대</a>에 프랑스 왕실의 애완용 <a href="/wiki/%ED%86%A0%EB%81%BC" title="토끼">토끼</a>로 유명하다. <a href="/wiki/%EB%AF%B8%EA%B5%AD" title="미국">미국</a>에는 20세기 초에 처음으로 도입됐다. <a class="new" href="/w/index.php?title=%EC%95%99%EA%B3%A0%EB%9D%BC_%ED%86%A0%EB%81%BC&amp;action=edit&amp;redlink=1" title="앙고라 토끼 (없는 문서)">앙고라 토끼</a>는 털인 <a href="/wiki/%EC%95%99%EA%B3%A0%EB%9D%BC%ED%84%B8" title="앙고라털">앙고라 울</a>로 유명한데, <a href="/wiki/%EC%95%99%EA%B3%A0%EB%9D%BC%ED%84%B8" title="앙고라털">앙고라 울</a>은 희고 길며 가벼워서 의복 재료에 많이 쓰인다.</p>
<p><a class="new" href="/w/index.php?title=%EC%95%99%EA%B3%A0%EB%9D%BC_%ED%86%A0%EB%81%BC&amp;action=edit&amp;redlink=1" title="앙고라 토끼 (없는 문서)">앙고라 토끼</a>를 사육하는 가장 큰 이유는 <a href="/wiki/%EC%95%99%EA%B3%A0%EB%9D%BC%ED%84%B8" title="앙고라털">앙고라 울</a>을 얻기 위해서다. 대부분의 <a class="new" href="/w/index.php?title=%EC%95%99%EA%B3%A0%EB%9D%BC_%ED%86%A0%EB%81%BC&amp;action=edit&amp;redlink=1" title="앙고라 토끼 (없는 문서)">앙고라 토끼</a>는 온순하지만, 주의 깊게 다뤄주어야 한다. 여름에는 털을 짧게 깎기도 하는데, 긴 털 때문에 여름에 열사할 가능성이 있기 때문이다.</p>
