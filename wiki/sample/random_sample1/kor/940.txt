<p><a href="/wiki/%EB%AC%BC%EB%A6%AC%ED%95%99" title="물리학">물리학</a>의 <a href="/wiki/%EC%96%91%EC%9E%90%EC%97%AD%ED%95%99" title="양자역학">양자역학</a>에서 <b>EPR 역설</b>(EPR Paradox)은 물리량의 측정 문제를 제기한 정교한 <a class="mw-redirect" href="/wiki/%EC%82%AC%EA%B3%A0%EC%8B%A4%ED%97%98" title="사고실험">사고실험</a>이다. <a href="/wiki/1935%EB%85%84" title="1935년">1935년</a> <a href="/wiki/%EC%95%8C%EB%B2%A0%EB%A5%B4%ED%8A%B8_%EC%95%84%EC%9D%B8%EC%8A%88%ED%83%80%EC%9D%B8" title="알베르트 아인슈타인">아인슈타인</a>(Einstein)과 <a class="new" href="/w/index.php?title=%EB%B3%B4%EB%A6%AC%EC%8A%A4_%ED%8F%AC%EB%8F%8C%EC%8A%A4%ED%82%A4&amp;action=edit&amp;redlink=1" title="보리스 포돌스키 (없는 문서)">포돌스키</a>(Podolsky) 및 <a class="new" href="/w/index.php?title=%EB%84%A4%EC%9D%B4%EC%84%A0_%EB%A1%9C%EC%A0%A0&amp;action=edit&amp;redlink=1" title="네이선 로젠 (없는 문서)">로젠</a>(Rogen)은 양자역학이 완전한 물리 이론이 아님을 보이기 위해 이 역설을 발표했으며, "EPR"은 그들 이름의 머릿글자를 딴 것이다. "Can Quantum-Mechanical Description of Physical Reality Be Considered Complete?"이라는 제목의 논문을 통해 나왔다.<sup class="reference" id="cite_ref-1"><a href="#cite_note-1">[1]</a></sup></p>
<p>이는 양자역학의 주류 해석인 <a href="/wiki/%EC%BD%94%ED%8E%9C%ED%95%98%EA%B2%90_%ED%95%B4%EC%84%9D" title="코펜하겐 해석">코펜하겐 해석</a>의 문제 제기에서 시작한다. 코펜하겐 해석은 어떤 상태를 <a class="new" href="/w/index.php?title=%EC%B8%A1%EC%A0%95_%EB%AC%B8%EC%A0%9C_(%EB%AC%BC%EB%A6%AC%ED%95%99)&amp;action=edit&amp;redlink=1" title="측정 문제 (물리학) (없는 문서)">측정</a>할 때, 측정함과 동시에 그 계는 측정에 해당하는 <a class="new" href="/w/index.php?title=%EA%B3%A0%EC%9C%A0%EC%83%81%ED%83%9C&amp;action=edit&amp;redlink=1" title="고유상태 (없는 문서)">고유상태</a>로 붕괴해 버린다는 것이다. <a class="mw-redirect" href="/wiki/%EA%B5%AD%EC%86%8C%EC%84%B1_%EC%9B%90%EB%A6%AC" title="국소성 원리">국소성 원리</a>에 의해 양자역학의 측정 결과는 (빛의 속도로 바로 갈 수 있는 것보다) 멀리 떨어진 곳에서도 측정할 수 있다는 것이다.</p>

<h2><span class="mw-headline" id=".EC.82.AC.EA.B3.A0.EC.8B.A4.ED.97.98_.EB.82.B4.EC.9A.A9">사고실험 내용</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=EPR_%EC%97%AD%EC%84%A4&amp;action=edit&amp;section=1" title="부분 편집: 사고실험 내용">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>EPR(Einstein, Podolsky, Rogen) 사고실험은 다음과 같다.</p>
<p>어떠한 A라는 상태와 B라는 상태가 <a href="/wiki/%EC%96%91%EC%9E%90_%EC%96%BD%ED%9E%98" title="양자 얽힘">얽힘</a> 계를 생각하자. 이 상태는 하나의 <a class="mw-redirect" href="/wiki/%ED%95%B4%EB%B0%80%ED%86%A0%EB%8B%88%EC%95%88" title="해밀토니안">해밀토니안</a>에 따라 진행한다. 이 A와 B가 빛의 속도로 도달하지 못할 만큼 멀리 떨어지게 된다. 이제 양자계의 한 부분 A를 측정하자. 여기에서 이루어진 측정의 결과가, 어떤 보존 법칙 때문에, 멀리 떨어진 다른 부분 <b>B</b>의 물리적 상태를 결정한다. A의 측정은 <a href="/wiki/%EA%B5%AD%EC%86%8C%EC%84%B1%EC%9D%98_%EC%9B%90%EB%A6%AC" title="국소성의 원리">비국소적</a>으로 작용하여, EPR 실험 결과는 다음의 <a class="mw-disambig" href="/wiki/%EC%9D%B4%EB%B6%84%EB%B2%95" title="이분법">이분법</a>을 낳는다.</p>
<ol>
<li>멀리 떨어진 곳에서 행해진 실험의 결과를 즉각적으로 알게 되는 일이 발생한다.</li>
<li>양자역학은 불완전한 이론으로, <b>B</b>에서 일어난 특수한 현상을 설명할 수 없으며 추가적인 변수를 필요로 한다.</li>
</ol>
<p>원래 이 사고실험은 <a class="new" href="/w/index.php?title=%EC%96%91%EC%9E%90%EC%97%AD%ED%95%99%EC%9D%98_%EB%B6%88%EC%99%84%EC%A0%84%EC%84%B1&amp;action=edit&amp;redlink=1" title="양자역학의 불완전성 (없는 문서)">양자역학의 불완전성</a>을 보이기 위해 고안되었으나 실제 <a class="extiw" href="https://en.wikipedia.org/wiki/Bell_test_experiments" title="en:Bell test experiments">실험 결과</a>는 <a href="/wiki/%EC%88%A8%EC%9D%80_%EB%B3%80%EC%88%98_%EC%9D%B4%EB%A1%A0" title="숨은 변수 이론">숨은 변수 이론</a>이 실제 실험결과와는 상반됨을 보였다.</p>
<p>아인슈타인은 양자역학을 실재를 묘사하는 완전한 이론으로서 받아들이지 않았으며, 목숨이 다할 때까지 양자론적 측정 결과들에 대해 "신이 주사위를 던지지 않는" 해석을 찾아내기 위해 노력하였다.</p>
