<div class="mw-content-ltr" dir="ltr" id="mw-content-text" lang="ko"><script>function mfTempOpenSection(id){var block=document.getElementById("mf-section-"+id);block.className+=" open-block";block.previousSibling.className+=" open-block";}</script><p><b>앤드루스 나카하라</b>(Andrews Nakahara)는 <a href="/wiki/1983%EB%85%84" title="1983년">1983년</a> <a href="/wiki/3%EC%9B%94_12%EC%9D%BC" title="3월 12일">3월 12일</a>생의 <a href="/wiki/%EC%9D%BC%EB%B3%B8%EA%B3%84_%EB%B8%8C%EB%9D%BC%EC%A7%88%EC%9D%B8" title="일본계 브라질인">일본계 브라질인</a>인 종합격투기 선수다. 가라테를 베이스로 하고 있으며 <a href="/wiki/2008%EB%85%84" title="2008년">2008년</a> <a href="/wiki/4%EC%9B%94_29%EC%9D%BC" title="4월 29일">4월 29일</a> 드림2에서 <a class="mw-redirect" href="/wiki/%EC%82%AC%EC%BF%A0%EB%9D%BC%EB%B0%94_%EC%B9%B4%EC%A6%88%EC%8B%9C" title="사쿠라바 카즈시">사쿠라바 카즈시</a>와 데뷔전에서 데뷔전으로서는 선전을 했지만 아쉽게 지게 됐다. <a href="/wiki/2008%EB%85%84" title="2008년">2008년</a> <a href="/wiki/9%EC%9B%94_23%EC%9D%BC" title="9월 23일">9월 23일</a> 드림6에 출전하여 리저브매치의 상대인 <a href="/wiki/%EC%9C%A4%EB%8F%99%EC%8B%9D_(%EA%B2%A9%ED%88%AC%EA%B8%B0_%EC%84%A0%EC%88%98)" title="윤동식 (격투기 선수)">윤동식</a>을 2R 초반에 제압했다. <a href="/wiki/2009%EB%85%84" title="2009년">2009년</a> <a href="/wiki/4%EC%9B%94_5%EC%9D%BC" title="4월 5일">4월 5일</a>에는 <a href="/wiki/%EC%98%A4%EC%95%BC%EB%A7%88_%EC%8A%8C%EA%B3%A0" title="오야마 슌고">오야마 슌고</a> 선수를 더욱 더 상승된 기량으로 1R에 제압했다.</p>
