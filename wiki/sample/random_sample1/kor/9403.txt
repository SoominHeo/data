<p><b>앤드류사쿠스 몽골리안시스</b>(Andrewsarchus mongoliensis, <i>Andrews</i> + <a class="mw-redirect" href="/wiki/%EA%B7%B8%EB%A6%AC%EC%8A%A4_%EC%96%B4" title="그리스 어">그리스 어</a> <i>ἀρχός</i>, "지배자")는 <a href="/wiki/%EC%97%90%EC%98%A4%EC%84%B8" title="에오세">에오세</a> 시기와 <a href="/wiki/%EC%98%AC%EB%A6%AC%EA%B3%A0%EC%84%B8" title="올리고세">올리고세</a> 전반기에 살았던 <a href="/wiki/%ED%8F%AC%EC%9C%A0%EB%A5%98" title="포유류">포유류</a> <a class="mw-redirect" href="/wiki/%EC%9A%B0%EC%A0%9C%EB%AA%A9" title="우제목">우제목</a>에 속하는 동물이었다. 현재의 <a class="mw-disambig" href="/wiki/%EC%97%BC%EC%86%8C" title="염소">염소</a>, <a href="/wiki/%EC%96%91" title="양">양</a>에 가까운 동물이었으나 육식성이었다. 완벽한 전신골격은 거의 발견되지 않았으나 주로 사체를 먹었을 것으로 추정하고 있다.</p>
<h2><span class="mw-headline" id=".EA.B0.9C.EC.9A.94">개요</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%95%A4%EB%93%9C%EB%A5%98%EC%82%AC%EC%BF%A0%EC%8A%A4&amp;action=edit&amp;section=1" title="부분 편집: 개요">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>앤드류사쿠스는 커다란 길쭉한 코와, 날카로운 이빨 그리고 납작한 볼을 가진 네발 짐승으로, 길쭉한 코와 날카로운 이빨은 다른 짐승의 뼈를 부수기 위해서 라고 생각된다. 발톱은 없고 발굽이 있으며, 속도가 느린편이라 시체를 주로 먹었을 추정된다. 앤드류사쿠스의 화석은 주로 <a href="/wiki/%EB%AA%BD%EA%B3%A8" title="몽골">몽골</a>과 <a href="/wiki/%EC%A4%91%EA%B5%AD" title="중국">중국</a> 북부 <a href="/wiki/%EB%A7%8C%EB%A6%AC%EC%9E%A5%EC%84%B1" title="만리장성">만리장성</a> 이북 지역에서 발견되었으며, 첫 발견자인 채프먼 앤드류의 이름을 따서 앤드류사쿠스라 이름붙였다.</p>
<p>앤드류사쿠스는 아직 몇 개의 해골밖에 알려지지 않아서, 이 동물이 포식자였는지, 혹은 이 동물이 하이에나 같은 청소부 역할을 했을지에 대해서 논쟁이 붙고 있다. 두개골의 길이는 1m이고, 몸길이는 3m 가량으로 추정된다. 앤드류사쿠스는 육식성이었으나, 그와 가까운 친척이던 <a class="mw-redirect" href="/wiki/%EC%95%84%EB%A9%94%EB%A6%AC%EC%B9%B4_%EB%8C%80%EB%A5%99" title="아메리카 대륙">아메리카 대륙</a>의 <a class="new" href="/w/index.php?title=%EC%98%A4%EB%A0%88%EC%98%A4%EB%8F%88%ED%8A%B8&amp;action=edit&amp;redlink=1" title="오레오돈트 (없는 문서)">오레오돈트</a>는 초식동물이었다. 현재의 <a class="mw-disambig" href="/wiki/%EC%97%BC%EC%86%8C" title="염소">염소</a>와 <a href="/wiki/%EC%96%91" title="양">양</a>에 가까운 근연종으로, <a class="mw-disambig" href="/wiki/%EC%97%BC%EC%86%8C" title="염소">염소</a>와 <a href="/wiki/%EC%96%91" title="양">양</a>의 직계 조상인 <a class="new" href="/w/index.php?title=%EC%98%A4%EB%A0%88%EC%98%A4%EB%8F%88%ED%8A%B8&amp;action=edit&amp;redlink=1" title="오레오돈트 (없는 문서)">오레오돈트</a>와 공통 조상에서 갈려나간 종이었다.</p>
<h3><span class="mw-headline" id=".EC.8B.9D.EC.84.B1">식성</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%95%A4%EB%93%9C%EB%A5%98%EC%82%AC%EC%BF%A0%EC%8A%A4&amp;action=edit&amp;section=2" title="부분 편집: 식성">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>앤드류사쿠스는 날카로운 송곳니와 넓고 튼튼한 어금니를 갖고 있었다. 주로 육식성이었으나 작은 동물을 직접 사냥하기도 했고, 죽은 동물의 사체, 즙이 많은 나뭇잎과 나무줄기, 산<a href="/wiki/%EB%94%B8%EA%B8%B0" title="딸기">딸기</a> 종류, 곤충의 유충과 땅을 기어다니는 벌레류, 작은 <a href="/wiki/%EC%84%A4%EC%B9%98%EB%A5%98" title="설치류">설치류</a>와 <a href="/wiki/%EB%8F%84%EB%A7%88%EB%B1%80" title="도마뱀">도마뱀</a> 등을 씹어먹었다.</p>
<p>얼굴 앞부분에 난 윗송곳니는 턱 밖으로 나와 있으며, 안쪽으로 구부러져 있고, 어금니는 고기를 잘 씹도록 칼날과 같은 형태를 띄고 있었다. 위턱의 어금니는 상당히 넓었는데, 뼈를 부술 수 있을 수준이었다. 기타 이빨은 곰의 이빨과 비슷하다.</p>
