<p><b>앨런 볼</b>(<span style="font-size: smaller;"><a href="/wiki/%EC%98%81%EC%96%B4" title="영어">영어</a>:</span> <span lang="en" xml:lang="en">Alan Ball</span>, <a href="/wiki/1957%EB%85%84" title="1957년">1957년</a> <a href="/wiki/5%EC%9B%94_13%EC%9D%BC" title="5월 13일">5월 13일</a> ~ )은 <a href="/wiki/%EB%AF%B8%EA%B5%AD" title="미국">미국</a>의 <a href="/wiki/%EA%B0%81%EB%B3%B8%EA%B0%80" title="각본가">각본가</a>이자 <a href="/wiki/%EC%98%81%ED%99%94_%EA%B0%90%EB%8F%85" title="영화 감독">영화 감독</a> · <a href="/wiki/%EC%98%81%ED%99%94_%ED%94%84%EB%A1%9C%EB%93%80%EC%84%9C" title="영화 프로듀서">프로듀서</a>이다. 영화 《<a href="/wiki/%EC%95%84%EB%A9%94%EB%A6%AC%EC%B9%B8_%EB%B7%B0%ED%8B%B0" title="아메리칸 뷰티">아메리칸 뷰티</a>》와 TV 드라마 《<a href="/wiki/%ED%8A%B8%EB%A3%A8_%EB%B8%94%EB%9F%AC%EB%93%9C" title="트루 블러드">트루 블러드</a>》의 각본을 써 이름을 알렸다. TV 시리즈 《<a href="/wiki/%EC%8B%9D%EC%8A%A4_%ED%95%8F_%EC%96%B8%EB%8D%94" title="식스 핏 언더">식스 핏 언더</a>》를 제작하기도 했다.</p>
<p><a href="/wiki/%EC%A1%B0%EC%A7%80%EC%95%84_%EC%A3%BC" title="조지아 주">조지아 주</a>의 <a href="/wiki/%EB%A7%A4%EB%A6%AC%EC%97%90%ED%83%80_(%EC%A1%B0%EC%A7%80%EC%95%84_%EC%A3%BC)" title="매리에타 (조지아 주)">매리에타</a>에서 태어나 <a href="/wiki/%EC%A1%B0%EC%A7%80%EC%95%84_%EB%8C%80%ED%95%99%EA%B5%90" title="조지아 대학교">조지아 대학교</a>와 <a href="/wiki/%ED%94%8C%EB%A1%9C%EB%A6%AC%EB%8B%A4_%EC%A3%BC%EB%A6%BD_%EB%8C%80%ED%95%99%EA%B5%90" title="플로리다 주립 대학교">플로리다 주립 대학교</a>에서 연극예술을 전공했고, 1980년 졸업했다.</p>
