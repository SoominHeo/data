<p><b>앨릭스 촐라</b>(Alex Chola, <a href="/wiki/1956%EB%85%84" title="1956년">1956년</a> <a href="/wiki/6%EC%9B%94_5%EC%9D%BC" title="6월 5일">6월 5일</a> ~ <a href="/wiki/1993%EB%85%84" title="1993년">1993년</a> <a href="/wiki/4%EC%9B%94_27%EC%9D%BC" title="4월 27일">4월 27일</a>)는 <a href="/wiki/%EC%9E%A0%EB%B9%84%EC%95%84" title="잠비아">잠비아</a>의 전 <a href="/wiki/%EC%B6%95%EA%B5%AC" title="축구">축구</a> 선수 및 축구 <a class="mw-disambig" href="/wiki/%EA%B0%90%EB%8F%85" title="감독">감독</a>이었다.</p>
<p><a href="/wiki/%EC%BD%A9%EA%B3%A0_%EB%AF%BC%EC%A3%BC_%EA%B3%B5%ED%99%94%EA%B5%AD" title="콩고 민주 공화국">콩고 민주 공화국</a> 출생으로 <a href="/wiki/1972%EB%85%84" title="1972년">1972년</a> 콩고 민주 공화국의 <a class="new" href="/w/index.php?title=%EC%86%94%EB%B2%A0%EB%82%98_FC&amp;action=edit&amp;redlink=1" title="솔베나 FC (없는 문서)">솔베나 FC</a>에서 데뷔했으며, 이후 <a href="/wiki/1974%EB%85%84" title="1974년">1974년</a> 잠비아의 <a href="/wiki/%EC%9D%80%EB%8F%8C%EB%9D%BC" title="은돌라">은돌라</a>로 이주한 뒤 <a class="new" href="/w/index.php?title=%EB%AC%B4%ED%92%80%EB%A6%AC%EB%9D%BC_%EB%B8%94%EB%9E%99%ED%92%80&amp;action=edit&amp;redlink=1" title="무풀리라 블랙풀 (없는 문서)">무풀리라 블랙풀</a>에 입단해 활약하였다. 그 뒤 <a href="/wiki/1980%EB%85%84" title="1980년">1980년</a> <a class="new" href="/w/index.php?title=%ED%8C%8C%EC%9B%8C_%EB%8B%A4%EC%9D%B4%EB%82%98%EB%AA%A8%EC%8A%A4_FC&amp;action=edit&amp;redlink=1" title="파워 다이나모스 FC (없는 문서)">파워 다이나모스 FC</a>로 이적했으며, <a href="/wiki/1985%EB%85%84" title="1985년">1985년</a>까지 활약한 뒤 은퇴를 선언하였다. 또한 <a href="/wiki/1975%EB%85%84" title="1975년">1975년</a> <a href="/wiki/%EC%9E%A0%EB%B9%84%EC%95%84_%EC%B6%95%EA%B5%AC_%EA%B5%AD%EA%B0%80%EB%8C%80%ED%91%9C%ED%8C%80" title="잠비아 축구 국가대표팀">잠비아 축구 국가대표팀</a>에 정식으로 데뷔한 뒤, <a href="/wiki/1985%EB%85%84" title="1985년">1985년</a>까지 활약하며 잠비아 국가대표팀 역사상 <a href="/wiki/%EA%B3%A0%EB%93%9C%ED%94%84%EB%A6%AC_%EC%B9%98%ED%83%88%EB%A3%A8" title="고드프리 치탈루">고드프리 치탈루</a> 다음으로 많은 득점 기록을 보유하였다.</p>
<p>은퇴 이후 <a href="/wiki/1985%EB%85%84" title="1985년">1985년</a>부터 <a href="/wiki/1990%EB%85%84" title="1990년">1990년</a>까지 친정팀인 파워 다이나모스에서 감독 생활을 했으며, <a href="/wiki/1992%EB%85%84" title="1992년">1992년</a>부터 다시 파워 다이나모스의 감독을 맡아 그 해 '올해의 감독상'을 수상하기도 했다. 그 뒤 <a href="/wiki/1993%EB%85%84" title="1993년">1993년</a> 고드프리 치탈루가 이끄는 잠비아 국가대표팀 수석 코치에 선임되었다.</p>
<p><a href="/wiki/1993%EB%85%84" title="1993년">1993년</a> <a href="/wiki/4%EC%9B%94_27%EC%9D%BC" title="4월 27일">4월 27일</a> <a href="/wiki/%EC%84%B8%EB%84%A4%EA%B0%88_%EC%B6%95%EA%B5%AC_%EA%B5%AD%EA%B0%80%EB%8C%80%ED%91%9C%ED%8C%80" title="세네갈 축구 국가대표팀">세네갈</a>과의 <a href="/wiki/1994%EB%85%84_FIFA_%EC%9B%94%EB%93%9C%EC%BB%B5_%EC%95%84%ED%94%84%EB%A6%AC%EC%B9%B4_%EC%A7%80%EC%97%AD_%EC%98%88%EC%84%A0" title="1994년 FIFA 월드컵 아프리카 지역 예선">1994년 FIFA 월드컵 아프리카 지역 예선</a>을 치르기 위해 <a class="new" href="/w/index.php?title=%EC%9E%A0%EB%B9%84%EC%95%84_%EA%B3%B5%EA%B5%B0&amp;action=edit&amp;redlink=1" title="잠비아 공군 (없는 문서)">잠비아 공군</a> 소유의 비행기인 <a class="new" href="/w/index.php?title=%EB%93%9C_%ED%95%98%EB%B9%8C%EB%9E%9C%EB%93%9C_%EC%BA%90%EB%82%98%EB%8B%A4_DHC-5_%EB%B2%84%ED%8E%84%EB%A1%9C&amp;action=edit&amp;redlink=1" title="드 하빌랜드 캐나다 DHC-5 버펄로 (없는 문서)">드 하빌랜드 캐나다 DHC-5 버펄로</a>를 타고 이동하던 도중 <a class="mw-redirect" href="/wiki/%EA%B0%80%EB%B4%89_%EA%B3%B5%ED%99%94%EA%B5%AD" title="가봉 공화국">가봉 공화국</a>의 <a href="/wiki/%EB%A6%AC%EB%B8%8C%EB%A5%B4%EB%B9%8C" title="리브르빌">리브르빌</a> 근방의 영해 상공에서 <a href="/wiki/%ED%95%AD%EA%B3%B5_%EC%82%AC%EA%B3%A0" title="항공 사고">항공 사고</a>가 발생했으며, 촐라를 포함한 <a href="/wiki/1993%EB%85%84_%EC%9E%A0%EB%B9%84%EC%95%84_%EC%B6%95%EA%B5%AC_%EA%B5%AD%EA%B0%80%EB%8C%80%ED%91%9C%ED%8C%80_%ED%95%AD%EA%B3%B5%EA%B8%B0_%EC%B0%B8%EC%82%AC" title="1993년 잠비아 축구 국가대표팀 항공기 참사">탑승객 30명 전원이 그 자리에서 사망</a>하였다.<sup class="reference" id="cite_ref-1"><a href="#cite_note-1">[1]</a></sup></p>
<p>사망 이후, <a href="/wiki/%EB%A3%A8%EC%82%AC%EC%B9%B4" title="루사카">루사카</a>에서 촐라의 이름을 딴 '앨릭스 촐라 도로'가 세워졌다.</p>
