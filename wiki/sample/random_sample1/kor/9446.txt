<h1 class="firstHeading" id="firstHeading" lang="ko">앨버트 루툴리</h1>

<p><b>앨버트 루툴리</b>(Albert Lutuli, <a href="/wiki/1898%EB%85%84" title="1898년">1898년</a>경 ~ <a href="/wiki/1967%EB%85%84" title="1967년">1967년</a> <a href="/wiki/7%EC%9B%94_21%EC%9D%BC" title="7월 21일">7월 21일</a>)는 <a href="/wiki/%EB%82%A8%EC%95%84%ED%94%84%EB%A6%AC%EC%B9%B4_%EA%B3%B5%ED%99%94%EA%B5%AD" title="남아프리카 공화국">남아프리카 공화국</a>의 정치인이자 교사로, <a href="/wiki/1952%EB%85%84" title="1952년">1952년</a>부터 <a href="/wiki/1967%EB%85%84" title="1967년">1967년</a>까지 <a href="/wiki/%EC%95%84%ED%94%84%EB%A6%AC%EC%B9%B4_%EB%AF%BC%EC%A1%B1%ED%9A%8C%EC%9D%98" title="아프리카 민족회의">아프리카 민족회의</a> 의장을 지내는 동안 당시 남아프리카 공화국의 소수 백인 정권에 대한 반대 운동을 전개했으며 <a href="/wiki/1960%EB%85%84" title="1960년">1960년</a> 아프리카 민족 회의의 발전과 <a href="/wiki/%EC%95%84%ED%8C%8C%EB%A5%B4%ED%8A%B8%ED%97%A4%EC%9D%B4%ED%8A%B8" title="아파르트헤이트">아파르트헤이트</a> 반대 운동을 주도한 공로로 <a href="/wiki/%EB%85%B8%EB%B2%A8_%ED%8F%89%ED%99%94%EC%83%81" title="노벨 평화상">노벨 평화상</a>을 수상했다.</p>
<p><a href="/wiki/1898%EB%85%84" title="1898년">1898년</a>경 <a href="/wiki/%EB%82%A8%EB%A1%9C%EB%94%94%EC%A7%80%EC%95%84" title="남로디지아">남로디지아</a>에서 <a class="mw-redirect" href="/wiki/%EC%A0%9C7%EC%9D%BC_%EC%95%88%EC%8B%9D%EC%9D%BC_%EC%98%88%EC%88%98_%EC%9E%AC%EB%A6%BC%EA%B5%90%ED%9A%8C" title="제7일 안식일 예수 재림교회">제7일 안식일 예수 재림교회</a> 선교사 존 버니언 루툴리(John Bunyan Lutuli)와 음토냐 구메데(Mtonya Gumede)의 셋째 아들로 태어났다. 아버지가 죽은 이후 그와 그의 어머니는 <a href="/wiki/%EB%82%A8%EC%95%84%ED%94%84%EB%A6%AC%EC%B9%B4_%EA%B3%B5%ED%99%94%EA%B5%AD" title="남아프리카 공화국">남아프리카 공화국</a> <a href="/wiki/%EC%BD%B0%EC%A4%84%EB%A3%A8%EB%82%98%ED%83%88_%EC%A3%BC" title="콰줄루나탈 주">콰줄루나탈 주</a>에 있는 마을인 그로트빌(Groutville)에서 마틴 루툴리(Martin Luthuli)와 함께 살았다.</p>
<p>그는 감리교 교회에서 세례를 받았으며 <a href="/wiki/1920%EB%85%84" title="1920년">1920년</a> 정부에서 받은 장학금으로 애덤스 대학에서 고도의 교원 양성 학교 스탭으로 참가했는데, 당시 애덤스 대학 총장이던 Z. K. 매슈즈(Z.K. Mathews)가 그를 가르쳤다. 그의 어머니가 애덤스 대학과 금전적인 협력을 했기 때문에 포트헤어 대학교로부터 장학금 지급을 거절당했다. <a href="/wiki/1928%EB%85%84" title="1928년">1928년</a> 아프리카 교원 협회 서기관에 올랐으며 <a href="/wiki/1933%EB%85%84" title="1933년">1933년</a>에는 회장에 오르게 된다. 그는 선교사로도 일했다.</p>
<p><a href="/wiki/1933%EB%85%84" title="1933년">1933년</a> 부족장이 그에게 부족장을 맡아줄 것을 제안했다. 그는 이 제안을 받아들일 지를 놓고 2년 동안 망설이다가 <a href="/wiki/1936%EB%85%84" title="1936년">1936년</a> 초에 이 제안을 받아들였고 <a href="/wiki/1952%EB%85%84" title="1952년">1952년</a> 정부에 의해 폐지될 때까지 부족장을 지냈다.</p>
<p><a href="/wiki/1944%EB%85%84" title="1944년">1944년</a>에는 <a href="/wiki/%EC%95%84%ED%94%84%EB%A6%AC%EC%B9%B4_%EB%AF%BC%EC%A1%B1%ED%9A%8C%EC%9D%98" title="아프리카 민족회의">아프리카 민족회의</a>에 참가했으며 <a href="/wiki/1945%EB%85%84" title="1945년">1945년</a>에는 콰줄루나탈 주 위원회 위원, <a href="/wiki/1951%EB%85%84" title="1951년">1951년</a>에는 위원장으로 선출되었다. <a href="/wiki/1952%EB%85%84" title="1952년">1952년</a>에는 다른 아프리카 민족 회의 위원장들과 함께 당시 정부에서 만든 흑인 차별 법률을 비폭력인 방법으로 무시하자는 내용의 캠페인을 벌였고 그 해에 아프리카 민족 회의 의장으로 취임했다. <a href="/wiki/1956%EB%85%84" title="1956년">1956년</a> 아프리카 민족 회의는 "자유 헌장"을 제정했고 그는 <a href="/wiki/1960%EB%85%84" title="1960년">1960년</a> <a href="/wiki/%EB%85%B8%EB%B2%A8_%ED%8F%89%ED%99%94%EC%83%81" title="노벨 평화상">노벨 평화상</a>을 수상하게 된다. 그러나 <a href="/wiki/1967%EB%85%84" title="1967년">1967년</a> 7월 자택 근처에서 당한 사고로 치명적인 상처를 입었고 그로 인해 사망하고 만다.</p>
