<p><b>앨버트 조지프 스캔런</b>(Albert Joseph Scanlon, <a href="/wiki/1935%EB%85%84" title="1935년">1935년</a> <a href="/wiki/10%EC%9B%94_10%EC%9D%BC" title="10월 10일">10월 10일</a> ~ <a href="/wiki/2009%EB%85%84" title="2009년">2009년</a> <a href="/wiki/12%EC%9B%94_22%EC%9D%BC" title="12월 22일">12월 22일</a>)은 <a href="/wiki/%EC%9E%89%EA%B8%80%EB%9E%9C%EB%93%9C" title="잉글랜드">잉글랜드</a>의 전 <a href="/wiki/%EC%B6%95%EA%B5%AC" title="축구">축구</a> <a href="/wiki/%EA%B3%B5%EA%B2%A9%EC%88%98" title="공격수">공격수</a>였다.</p>
<p><a href="/wiki/%EB%A7%A8%EC%B2%B4%EC%8A%A4%ED%84%B0_%EC%9C%A0%EB%82%98%EC%9D%B4%ED%8B%B0%EB%93%9C_FC" title="맨체스터 유나이티드 FC">맨체스터 유나이티드 FC</a>의 공격수였던 <a class="new" href="/w/index.php?title=%EC%B0%B0%EB%A6%AC_%EB%AF%B8%ED%8A%BC&amp;action=edit&amp;redlink=1" title="찰리 미튼 (없는 문서)">찰리 미튼</a>의 조카이며, 유소년 시절부터 뛰어난 재능을 보여 <a href="/wiki/1950%EB%85%84" title="1950년">1950년</a>부터 맨체스터 유나이티드의 유소년팀에 입단해 활약하였다. 이후 <a href="/wiki/1953%EB%85%84" title="1953년">1953년</a>년과 <a href="/wiki/1954%EB%85%84" title="1954년">1954년</a> 연속으로 <a class="new" href="/w/index.php?title=FA_%EC%9C%A0%EC%8A%A4%EC%BB%B5&amp;action=edit&amp;redlink=1" title="FA 유스컵 (없는 문서)">FA 유스컵</a>에서 팀이 우승을 차지하는 데 공헌했으며, <a href="/wiki/11%EC%9B%94_20%EC%9D%BC" title="11월 20일">11월 20일</a> <a href="/wiki/%EC%95%84%EC%8A%A4%EB%84%90_FC" title="아스널 FC">아스널 FC</a>와의 경기로 프로 무대에 데뷔하였다. 그 뒤 팀이 1955-56 시즌과 1956-57 시즌 2연속 리그 우승에 오르는 데 기여했으나, 스캔런은 두 번 모두 우승 메달을 받지 못했다.</p>
<p>이후 <a href="/wiki/1958%EB%85%84" title="1958년">1958년</a> <a href="/wiki/2%EC%9B%94_5%EC%9D%BC" title="2월 5일">2월 5일</a> <a href="/wiki/%EB%B2%A0%EC%98%A4%EA%B7%B8%EB%9D%BC%EB%93%9C" title="베오그라드">베오그라드</a>에서 열린 <a class="mw-redirect" href="/wiki/%EC%B8%A0%EB%A5%B4%EB%B2%A0%EB%82%98_%EC%A6%88%EB%B2%A0%EC%A6%88%EB%8B%A4" title="츠르베나 즈베즈다">레드 스타 베오그라드</a>와의 원정 경기를 끝내고 <a href="/wiki/%EB%A7%A8%EC%B2%B4%EC%8A%A4%ED%84%B0" title="맨체스터">맨체스터</a>로 복귀하던 도중 재급유를 위해 <a href="/wiki/%ED%95%AD%EA%B3%B5%EA%B8%B0" title="항공기">항공기</a>가 <a href="/wiki/%EB%AE%8C%ED%97%A8" title="뮌헨">뮌헨</a>을 경유한 뒤, 항공기가 이륙에 실패하여 <a href="/wiki/%EB%AE%8C%ED%97%A8_%EB%B9%84%ED%96%89%EA%B8%B0_%EC%B0%B8%EC%82%AC" title="뮌헨 비행기 참사">뮌헨 비행기 참사</a>로 알려진 <a href="/wiki/%ED%95%AD%EA%B3%B5_%EC%82%AC%EA%B3%A0" title="항공 사고">항공 사고</a>가 발생하였다. 당시 스캔런은 <a href="/wiki/%EB%B3%B4%EB%B9%84_%EC%B0%B0%ED%84%B4" title="보비 찰턴">보비 찰턴</a> 등과 함께 생존했던 선수 중 한 명이었으며, 사고 당시 다리가 부러지고 <a class="mw-redirect" href="/wiki/%EB%91%90%EA%B0%9C%EA%B3%A8" title="두개골">두개골</a>이 골절되었으며 <a href="/wiki/%EC%BD%A9%ED%8C%A5" title="콩팥">신장</a>이 손상되는 중상을 입었다. 하지만 1958-59 시즌을 앞두고 기적적으로 부상에서 복귀해 시즌 종료 때까지 16골을 득점하면서 팀을 리그 2위로 이끌었다.</p>
<p>그 뒤 <a href="/wiki/1960%EB%85%84" title="1960년">1960년</a> 11월 1만 8천 <a class="mw-redirect" href="/wiki/%EC%98%81%EA%B5%AD_%ED%8C%8C%EC%9A%B4%EB%93%9C" title="영국 파운드">파운드</a>에 <a href="/wiki/%EB%89%B4%EC%BA%90%EC%8A%AC_%EC%9C%A0%EB%82%98%EC%9D%B4%ED%8B%B0%EB%93%9C_FC" title="뉴캐슬 유나이티드 FC">뉴캐슬 유나이티드 FC</a>로 팀을 옮겼으나, 부진을 거듭하여 <a href="/wiki/1962%EB%85%84" title="1962년">1962년</a> 2월 하부 리그 소속이었던 <a href="/wiki/%EB%A7%81%EC%BB%A8_%EC%8B%9C%ED%8B%B0_FC" title="링컨 시티 FC">링컨 시티 FC</a>로 이적하였다. 이후 <a href="/wiki/1963%EB%85%84" title="1963년">1963년</a> 4월부터 <a href="/wiki/%EB%A7%A8%EC%8A%A4%ED%95%84%EB%93%9C_%ED%83%80%EC%9A%B4_FC" title="맨스필드 타운 FC">맨스필드 타운 FC</a> 소속으로 뛰었으며, <a href="/wiki/1966%EB%85%84" title="1966년">1966년</a> <a class="new" href="/w/index.php?title=%EB%B2%A8%ED%8D%BC_%ED%83%80%EC%9A%B4_FC&amp;action=edit&amp;redlink=1" title="벨퍼 타운 FC (없는 문서)">벨퍼 타운 FC</a>로 옮긴 뒤 은퇴를 선언하였다.</p>
<p>은퇴 이후 맨체스터 인근에 있는 <a class="new" href="/w/index.php?title=%EC%BD%9C%EA%B2%8C%EC%9D%B4%ED%8A%B8%ED%8C%8C%EB%AA%B0%EB%A6%AC%EB%B8%8C&amp;action=edit&amp;redlink=1" title="콜게이트파몰리브 (없는 문서)">콜게이트파몰리브</a> 공장에서 <a href="/wiki/%EA%B2%BD%EB%B9%84%EC%9B%90" title="경비원">경비원</a>으로 일했으며, <a href="/wiki/2007%EB%85%84" title="2007년">2007년</a> <a href="/wiki/5%EC%9B%94_13%EC%9D%BC" title="5월 13일">5월 13일</a> 과거 팀 동료였던 <a href="/wiki/%EB%B9%8C_%ED%8F%B4%ED%81%AC%EC%8A%A4" title="빌 폴크스">빌 폴크스</a>와 함께 2006-07 시즌 우승을 차지한 맨체스터 유나이티드에 <a href="/wiki/%ED%94%84%EB%A6%AC%EB%AF%B8%EC%96%B4%EB%A6%AC%EA%B7%B8" title="프리미어리그">프리미어리그</a> 우승 트로피를 전달하였다. 그 뒤 <a href="/wiki/2008%EB%85%84" title="2008년">2008년</a> <a href="/wiki/5%EC%9B%94_21%EC%9D%BC" title="5월 21일">5월 21일</a> <a href="/wiki/%EB%9F%AC%EC%8B%9C%EC%95%84" title="러시아">러시아</a> <a href="/wiki/%EB%AA%A8%EC%8A%A4%ED%81%AC%EB%B0%94" title="모스크바">모스크바</a>에서 열린 맨체스터 유나이티드 FC와 <a href="/wiki/%EC%B2%BC%EC%8B%9C_FC" title="첼시 FC">첼시 FC</a>와의 <a href="/wiki/2008_UEFA_%EC%B1%94%ED%94%BC%EC%96%B8%EC%8A%A4%EB%A6%AC%EA%B7%B8_%EA%B2%B0%EC%8A%B9%EC%A0%84" title="2008 UEFA 챔피언스리그 결승전">2007-08 UEFA 챔피언스리그 결승전</a>을 관전하였으며, 이 때 뮌헨 비행기 참사의 생존자인 보비 찰턴, <a class="new" href="/w/index.php?title=%ED%95%B4%EB%A6%AC_%EA%B7%B8%EB%A0%89&amp;action=edit&amp;redlink=1" title="해리 그렉 (없는 문서)">해리 그렉</a> 또한 같이 관전하였다.</p>
<p>이후 <a href="/wiki/2009%EB%85%84" title="2009년">2009년</a> <a href="/wiki/10%EC%9B%94_21%EC%9D%BC" title="10월 21일">10월 21일</a> <a href="/wiki/%EC%8B%A0%EC%9E%A5%EB%B3%91" title="신장병">신장병</a> 및 <a href="/wiki/%ED%8F%90%EB%A0%B4" title="폐렴">폐렴</a>으로 <a href="/wiki/%EA%B7%B8%EB%A0%88%EC%9D%B4%ED%84%B0%EB%A7%A8%EC%B2%B4%EC%8A%A4%ED%84%B0_%EC%A3%BC" title="그레이터맨체스터 주">그레이터맨체스터 주</a>의 <a href="/wiki/%EC%86%94%ED%8D%BC%EB%93%9C" title="솔퍼드">솔퍼드</a>에 있는 <a class="new" href="/w/index.php?title=%EC%86%94%ED%8D%BC%EB%93%9C_%EA%B5%AD%EB%A6%BD%EC%9D%98%EB%A3%8C%EC%9B%90&amp;action=edit&amp;redlink=1" title="솔퍼드 국립의료원 (없는 문서)">솔퍼드 국립의료원</a>에서 치료를 받은 뒤, 2달동안 투병 생활을 하다가 <a href="/wiki/12%EC%9B%94_22%EC%9D%BC" title="12월 22일">12월 22일</a> 사망했다.<sup class="reference" id="cite_ref-1"><a href="#cite_note-1">[1]</a></sup> <a href="/wiki/2010%EB%85%84" title="2010년">2010년</a> <a href="/wiki/1%EC%9B%94_13%EC%9D%BC" title="1월 13일">1월 13일</a> <a class="new" href="/w/index.php?title=%EC%9C%84%EC%8A%A4%ED%8A%B8&amp;action=edit&amp;redlink=1" title="위스트 (없는 문서)">위스트</a>에서 장례식이 거행되었으며, 맨체스터 유나이티드 동료였던 보비 찰턴, 해리 그렉, <a href="/wiki/%EB%B9%8C_%ED%8F%B4%ED%81%AC%EC%8A%A4" title="빌 폴크스">빌 폴크스</a> 및 과거 맨체스터 유나이티드 선수였던 <a href="/wiki/%EB%8D%B0%EB%8B%88%EC%8A%A4_%EB%A1%9C" title="데니스 로">데니스 로</a>, <a class="new" href="/w/index.php?title=%EB%85%B8%EB%B9%84_%EC%8A%A4%ED%8B%B8%EC%8A%A4&amp;action=edit&amp;redlink=1" title="노비 스틸스 (없는 문서)">노비 스틸스</a>를 포함해 300여명의 관계자들이 참석했다.</p>
