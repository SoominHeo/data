<p><b>야나 올렉산드리우나 클로치코바</b>(<span style="font-size: smaller;"><a href="/wiki/%EC%9A%B0%ED%81%AC%EB%9D%BC%EC%9D%B4%EB%82%98%EC%96%B4" title="우크라이나어">우크라이나어</a>:</span> <span lang="uk" xml:lang="uk">Яна Олександрівна Клочкова</span>, <a href="/wiki/1982%EB%85%84" title="1982년">1982년</a> <a href="/wiki/8%EC%9B%94_7%EC%9D%BC" title="8월 7일">8월 7일</a> 태어남)는 <a href="/wiki/%EC%9A%B0%ED%81%AC%EB%9D%BC%EC%9D%B4%EB%82%98" title="우크라이나">우크라이나</a>의 <a href="/wiki/%EC%88%98%EC%98%81" title="수영">수영</a> 선수이다. <a href="/wiki/%ED%81%AC%EB%A6%BC_%EB%B0%98%EB%8F%84" title="크림 반도">크림 반도</a>의 <a href="/wiki/%EC%8B%AC%ED%8E%98%EB%A1%9C%ED%8F%B4" title="심페로폴">심페로폴</a>에서 태어났다. <a href="/wiki/%EC%98%AC%EB%A6%BC%ED%94%BD" title="올림픽">올림픽</a>에서 <a href="/wiki/%EA%B8%88%EB%A9%94%EB%8B%AC" title="금메달">금메달</a> 4개를 포함해 통산 5개의 메달을 따냈다.</p>
<p><a href="/wiki/%EC%8B%9C%EB%93%9C%EB%8B%88" title="시드니">시드니</a>에서 열린 <a href="/wiki/2000%EB%85%84_%ED%95%98%EA%B3%84_%EC%98%AC%EB%A6%BC%ED%94%BD" title="2000년 하계 올림픽">2000년 하계 올림픽</a>에서 200미터 개인혼영과 400미터 개인혼영 금메달을 따냈다. <a href="/wiki/%EC%95%84%ED%85%8C%EB%84%A4" title="아테네">아테네</a>에서 열린 <a href="/wiki/2004%EB%85%84_%ED%95%98%EA%B3%84_%EC%98%AC%EB%A6%BC%ED%94%BD" title="2004년 하계 올림픽">2004년 하계 올림픽</a>에서는 같은 두 종목에서 다시 금메달을 따냈다. 그리하여 올림픽에서 같은 종목 둘을 2연패한 최초의 여자 수영 선수가 되었다.</p>
<p>세계 선수권 대회에서 10번의 우승 기록이 있으며 400미터 개인혼영에서는 단수로(short course)와 장수로(long course) 풀 양쪽 다에서 세계 기록 보유자이다.</p>
