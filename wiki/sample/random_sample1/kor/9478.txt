<p><b>야마다 아키요시</b> (<span style="font-size: smaller;"><a href="/wiki/%EC%9D%BC%EB%B3%B8%EC%96%B4" title="일본어">일본어</a>:</span> <span lang="ja"><span lang="ja"><ruby><rb>山田顕義</rb> <rp>(</rp><rt>やまだ あきよし</rt><rp>)</rp></ruby></span></span>, <a href="/wiki/1844%EB%85%84" title="1844년">1844년</a> <a href="/wiki/11%EC%9B%94_18%EC%9D%BC" title="11월 18일">11월 18일</a> ~ <a href="/wiki/1892%EB%85%84" title="1892년">1892년</a> <a href="/wiki/11%EC%9B%94_11%EC%9D%BC" title="11월 11일">11월 11일</a>) 는 <a href="/wiki/%EC%A1%B0%EC%8A%88_%EB%B2%88" title="조슈 번">조슈</a> <a href="/wiki/%EB%B2%88%EC%82%AC" title="번사">번사</a>, <a href="/wiki/%EC%9D%BC%EB%B3%B8" title="일본">일본</a>의 <a class="mw-redirect" href="/wiki/%EC%A0%95%EC%B9%98%EC%9D%B8" title="정치인">정치인</a>, 육군 군인이다. 육군 중장까지 올랐으며, 내각에서 공부경(工部卿), 내무경, 사법경, <a href="/wiki/%EC%A0%9C1%EC%B0%A8_%EC%9D%B4%ED%86%A0_%EB%82%B4%EA%B0%81" title="제1차 이토 내각">제1차 이토 내각</a> ~ <a href="/wiki/%EC%A0%9C1%EC%B0%A8_%EB%A7%88%EC%93%B0%EC%B9%B4%ED%83%80_%EB%82%B4%EA%B0%81" title="제1차 마쓰카타 내각">제1차 마쓰카타 내각</a>의 사법대신을 지냈다.</p>
<h2><span class="mw-headline" id=".EC.83.9D.EC.95.A0">생애</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%95%BC%EB%A7%88%EB%8B%A4_%EC%95%84%ED%82%A4%EC%9A%94%EC%8B%9C&amp;action=edit&amp;section=1" title="부분 편집: 생애">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p><a href="/wiki/%EB%A9%94%EC%9D%B4%EC%A7%80_%EC%8B%9C%EB%8C%80" title="메이지 시대">메이지 시대</a>의 군인으로서 <a href="/wiki/%EB%A9%94%EC%9D%B4%EC%A7%80_%EC%9C%A0%EC%8B%A0" title="메이지 유신">유신</a>시절 <a class="mw-redirect" href="/wiki/%EB%A9%94%EC%9D%B4%EC%A7%80_%EC%A0%95%EB%B6%80" title="메이지 정부">메이지 정부</a>에 기여함과 동시에, 근대 일본의 법전 편찬의 임무를 맡았으며, <a href="/wiki/%EB%8B%88%ED%98%BC_%EB%8C%80%ED%95%99" title="니혼 대학">니혼 대학</a> (일본법률학교) 과 <a href="/wiki/%EA%B3%A0%EC%BF%A0%EA%B0%80%EC%BF%A0%EC%9D%B8_%EB%8C%80%ED%95%99" title="고쿠가쿠인 대학">고쿠가쿠인 대학</a>(國學院大學) 의 설립에 관여하였다.</p>
