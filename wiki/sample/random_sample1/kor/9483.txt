<h1 class="firstHeading" id="firstHeading" lang="ko">야마자키 스스무</h1>
<div class="mw-body-content" id="bodyContent">
<div id="siteSub">위키백과, 우리 모두의 백과사전.</div>
<div id="contentSub"></div>
<div class="mw-jump" id="jump-to-nav">
					이동:					<a href="#mw-head">둘러보기</a>, 					<a href="#p-search">검색</a>
</div>
<div class="mw-content-ltr" dir="ltr" id="mw-content-text" lang="ko"><script>function mfTempOpenSection(id){var block=document.getElementById("mf-section-"+id);block.className+=" open-block";block.previousSibling.className+=" open-block";}</script><p><b>야마자키 스스무</b>(山崎烝, 1833? ~ 1868년 2월 6일(게이오 4년 1월 3일))는 <a href="/wiki/%EC%9D%BC%EB%B3%B8" title="일본">일본</a> <a href="/wiki/%EC%8B%A0%EC%84%BC%EA%B5%AC%EB%AF%B8" title="신센구미">신센구미</a>의 대원으로, 감찰역을 맡았다. 야마자키는 <a href="/wiki/%EC%98%A4%EC%82%AC%EC%B9%B4" title="오사카">오사카</a> 출신의 <a class="mw-redirect" href="/wiki/%EB%A1%9C%EB%8B%8C" title="로닌">낭인</a>이었으며, 가토리류 봉술의 달인이었다. 1863년에 야마자키는 공식적으로 신센구미에 들어가게 되었고, <a href="/wiki/%EA%B3%A4%EB%8F%84_%EC%9D%B4%EC%82%AC%EB%AF%B8" title="곤도 이사미">곤도 이사미</a>에 의해 <a class="new" href="/w/index.php?title=%EC%8B%9C%EB%A7%88%EB%8B%A4_%EA%B0%80%EC%9D%B4&amp;action=edit&amp;redlink=1" title="시마다 가이 (없는 문서)">시마다 가이</a>와 함께 후에 <a href="/wiki/%EC%9D%B4%EC%BC%80%EB%8B%A4%EC%95%BC_%EC%82%AC%EA%B1%B4" title="이케다야 사건">이케다야 사건</a>을 일으키게 되는 상황을 조사하게 된다.<sup class="reference" id="cite_ref-1"><a href="#cite_note-1">[1]</a></sup></p>
<p><a href="/wiki/%EA%B5%90%ED%86%A0" title="교토">교토</a>에 있을 때 그의 직업은 의사였으며, <a href="/wiki/%EB%8B%88%EC%8B%9C%ED%98%BC%EA%B0%84%EC%A7%80" title="니시혼간지">니시혼간지</a>로 본부를 옮기기 전의 초창기 신센구미의 의술 분야를 담당하기도 했다.</p>
<p>야마자키는 그의 삼촌에게 전수받아 문학과 무술에 뛰어났다고 전해지며, 그의 학문적 소양은 신센구미가 고위 간부와 정치를 얘기할 때 필수적이었다고 한다.</p>
<p>1865년에 야마자키는 곤도와 <a class="mw-redirect" href="/wiki/%ED%9E%88%EB%A1%9C%EC%8B%9C%EB%A7%88" title="히로시마">히로시마</a>에 동행하게 된다. 곤도와 함께 야마자키는 <a href="/wiki/%EC%A1%B0%EC%8A%88_%EB%B2%88" title="조슈 번">조슈 번</a>을 조사하기 위해 다음 해까지 돌아오지 않았다고 한다.</p>
<p>야마자키는 <a class="new" href="/w/index.php?title=%EB%A7%88%EC%93%B0%EB%AA%A8%ED%86%A0_%EB%A3%8C%EC%A4%80&amp;action=edit&amp;redlink=1" title="마쓰모토 료준 (없는 문서)">마쓰모토 료준</a>에게 의술을 배웠다. 마쓰모토에 따르면 야마자키는 무뚝뚝하고, 온화하고 친절한 사람이었다고 한다.</p>
<p>1867년에, 야마자키는 신센구미의 나머지 무사들과 함께 <a href="/wiki/%ED%95%98%ED%83%80%EB%AA%A8%ED%86%A0" title="하타모토">하타모토</a>가 되었다.</p>
<p>1868년 <a href="/wiki/%EB%B3%B4%EC%8B%A0_%EC%A0%84%EC%9F%81" title="보신 전쟁">보신 전쟁</a> 중에 야마자키는 심각하게 다치게 되고 1868년 1월 13일에 죽게 된다. 정확한 사망 장소는 알 수 없다. <sup class="reference" id="cite_ref-2"><a href="#cite_note-2">[2]</a></sup></p>
<p>야마자키는 곤도 이사미와 <a href="/wiki/%ED%9E%88%EC%A7%80%EC%B9%B4%ED%83%80_%EB%8F%84%EC%8B%9C%EC%A1%B0" title="히지카타 도시조">히지카타 도시조</a>의 전적인 신임을 얻었고, 죽을 때까지 신센구미에서 가장 믿을만 한 대원 중 한 명으로 여겨졌다.</p>
<h2><span class="mw-headline" id=".EB.AF.B8.EB.94.94.EC.96.B4.EC.97.90.EC.84.9C.EC.9D.98_.EC.95.BC.EB.A7.88.EC.9E.90.ED.82.A4">미디어에서의 야마자키</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%95%BC%EB%A7%88%EC%9E%90%ED%82%A4_%EC%8A%A4%EC%8A%A4%EB%AC%B4&amp;action=edit&amp;section=1" title="부분 편집: 미디어에서의 야마자키">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p><a class="new" href="/w/index.php?title=%ED%94%BC%EC%8A%A4%EB%A9%94%EC%9D%B4%EC%BB%A4_%EC%BF%A0%EB%A1%9C%EA%B0%80%EB%84%A4&amp;action=edit&amp;redlink=1" title="피스메이커 쿠로가네 (없는 문서)">피스메이커 쿠로가네</a>, <a class="new" href="/w/index.php?title=%EB%B0%94%EB%9E%8C%EC%9D%98_%EB%B9%9B&amp;action=edit&amp;redlink=1" title="바람의 빛 (없는 문서)">바람의 빛</a> 등에 나왔다. 1999년에 영화 <a href="/wiki/%EA%B3%A0%ED%95%98%ED%86%A0" title="고하토">고하토</a>에도 나온 바 있다. 야마자키는 <a href="/wiki/%EB%8B%8C%EC%9E%90" title="닌자">닌자</a>가 아니었음에도 불구하고, <a href="/wiki/%EB%B0%94%EB%9E%8C%EC%9D%98_%EA%B2%80%EC%8B%AC" title="바람의 검심">바람의 검심</a>에서의 <a class="mw-redirect" href="/wiki/%ED%95%9C%EB%83%90" title="한냐">한냐</a>를 포함한 <a href="/wiki/%EC%95%A0%EB%8B%88%EB%A9%94%EC%9D%B4%EC%85%98" title="애니메이션">애니메이션</a>이나 <a href="/wiki/%EB%93%9C%EB%9D%BC%EB%A7%88" title="드라마">드라마</a> 등에서는 주로 훌륭한 닌자로 묘사된다. 또한, <a href="/wiki/%EC%9D%80%ED%98%BC" title="은혼">은혼</a>의 야마자키 사가루 역시 야마자키에서 모티브를 따온 것이다.</p>
