Say! JUMP</a>의 최연장자 이다.</p>

<ul>
<li><a href="/wiki/2001%EB%85%84" title="2001년">2001년</a> <a href="/wiki/9%EC%9B%94_23%EC%9D%BC" title="9월 23일">9월 23일</a>, 쟈니즈 사무소의 오디션을 보고 레슨생으로서 입소. <a href="/wiki/12%EC%9B%94" title="12월">12월</a>, <a href="/wiki/%EC%9E%90%EB%8B%88%EC%8A%A4_%EC%A3%BC%EB%8B%88%EC%96%B4" title="자니스 주니어">쟈니즈 Jr.</a> 유닛·Ya-Ya-yah가 결성되어 멤버가 됐다.</li>
<li><a href="/wiki/2002%EB%85%84" title="2002년">2002년</a> <a href="/wiki/5%EC%9B%94_15%EC%9D%BC" title="5월 15일">5월 15일</a>에 Ya-Ya-yah로서, <a href="/wiki/%EC%9D%BC%EB%B3%B8%EB%B0%A9%EC%86%A1%ED%98%91%ED%9A%8C" title="일본방송협회">NHK</a> 애니메이션 《<a href="/wiki/%EB%8B%8C%EC%9E%90%EB%B3%B4%EC%9D%B4_%EB%9E%80%ED%83%80%EB%A1%9C" title="닌자보이 란타로">닌타마 란타로</a>》의 주제가 〈용기 100%/세계가 하나로 될 때까지〉로 CD 데뷔. <a href="/wiki/%EC%95%BC%EC%98%A4%ED%86%A0%EB%A9%94_%ED%9E%88%EC%B9%B4%EB%A3%A8" title="야오토메 히카루">야오토메 히카루</a>가 가입할 때까지는 아카마 나오야와 메인 보컬을 맡았다.</li>
<li><a href="/wiki/2005%EB%85%84" title="2005년">2005년</a> <a href="/wiki/4%EC%9B%94" title="4월">4월</a>에 호리코시 고등학교 입학.</li>
<li><a href="/wiki/2007%EB%85%84" title="2007년">2007년</a> <a href="/wiki/9%EC%9B%94_21%EC%9D%BC" title="9월 21일">9월 21일</a>부터 <a class="mw-redirect" href="/wiki/%ED%97%A4%EC%9D%B4!_%EC%84%B8%EC%9D%B4!_%EC%A0%90%ED%94%84" title="헤이! 세이! 점프">Hey! <li><a href="/wiki/2008%EB%85%84" title="2008년">2008년</a> <a href="/wiki/2%EC%9B%94_17%EC%9D%BC" title="2월 17일">2월 17일</a>에 호리코시 고등학교 졸업.</li>
<li><a href="/wiki/2012%EB%85%84" title="2012년">2012년</a> <a href="/wiki/1%EC%9B%94_2%EC%9D%BC" title="1월 2일">1월 2일</a> 개최한 Hey! Say!
