<p><i><b>Epilogue</b></i>는 <a href="/wiki/2010%EB%85%84" title="2010년">2010년</a> <a href="/wiki/3%EC%9B%94_9%EC%9D%BC" title="3월 9일">3월 9일</a> 발매된 <a href="/wiki/%EC%97%90%ED%94%BD%ED%95%98%EC%9D%B4" title="에픽하이">에픽하이</a>의 스페셜 음반이다. 전 소속사 <a class="mw-redirect" href="/wiki/%EC%9A%B8%EB%A6%BC_%EC%97%94%ED%84%B0%ED%85%8C%EC%9D%B8%EB%A8%BC%ED%8A%B8" title="울림 엔터테인먼트">울림 엔터테인먼트</a>와 현 소속사 <a href="/wiki/%EB%A7%B5%EB%8D%94%EC%86%8C%EC%9A%B8" title="맵더소울">맵더소울</a>이 합병하여 발매하고, <a href="/wiki/DJ_%ED%88%AC%EC%BB%B7" title="DJ 투컷">DJ 투컷</a>이 군복무로 빠져 2인조로 활동한 첫 음반이다. <a href="/wiki/DJ_%ED%88%AC%EC%BB%B7" title="DJ 투컷">DJ 투컷</a>의 군 복무에 따라 <a href="/wiki/%ED%83%80%EB%B8%94%EB%A1%9C" title="타블로">타블로</a>가 작곡한 신곡과 함께 기존에 만들었던 미공개곡을 수록하였다. 이 음반 이후로 에픽하이로서의 활동은 <a href="/wiki/2012%EB%85%84" title="2012년">2012년</a>까지 중단하고, <a href="/wiki/2013%EB%85%84" title="2013년">2013년</a>에 컴백 할 예정이라고 밝혔다.<sup class="reference" id="cite_ref-second_3-0"><a href="#cite_note-second-3">[3]</a></sup> 하지만 에픽하이는 2012년 10월에 정규 7집으로 컴백했다.</p>

<h2><span class="mw-headline" id=".EB.B0.B0.EA.B2.BD.28back0.29">배경(back0)</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Epilogue_(%EC%97%90%ED%94%BD%ED%95%98%EC%9D%B4%EC%9D%98_%EC%9D%8C%EB%B0%98)&amp;action=edit&amp;section=1" title="부분 편집: 배경(back0)">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<h3><span class="mw-headline" id=".EC.9D.8C.EB.B0.98_.EC.9E.91.EC.97.85">음반 작업</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Epilogue_(%EC%97%90%ED%94%BD%ED%95%98%EC%9D%B4%EC%9D%98_%EC%9D%8C%EB%B0%98)&amp;action=edit&amp;section=2" title="부분 편집: 음반 작업">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p><a href="/wiki/2009%EB%85%84_10%EC%9B%94" title="2009년 10월">2009년 10월</a>, <a href="/wiki/DJ_%ED%88%AC%EC%BB%B7" title="DJ 투컷">DJ 투컷</a>이 입대함에 따라, 에픽하이는 2년 공백을 가지게 되었다. <a href="/wiki/11%EC%9B%94_13%EC%9D%BC" title="11월 13일">11월 13일</a>, <a href="/wiki/%EC%9C%A0%ED%8A%9C%EB%B8%8C" title="유튜브">유튜브</a>와 맵더소울닷컴을 통해 자신들의 솔로 음반을 비롯한 <a href="/wiki/MYK" title="MYK">MYK</a>의 데뷔 음반 소식을 전한 뒤 차후 활동 없이 에픽하이의 음반이 여러 형식으로 새로 나올 수 있다고 밝혔다.<sup class="reference" id="cite_ref-update_4-0"><a href="#cite_note-update-4">[4]</a></sup> 원래 DJ 투컷 없이는 활동을 하지 않는다고 결심했었으나 군복무 중이던 DJ 투컷의 권유로 스페셜 음반을 제작하기 시작하였고 이후 <a href="/wiki/%ED%83%80%EB%B8%94%EB%A1%9C" title="타블로">타블로</a>는 100곡에 가까운 B 사이드 곡 중 몇 곡을 골라 작업하기 시작하였다. 추가적으로 타이틀 곡 "Run"과 몇 곡을 작곡하였다.<sup class="reference" id="cite_ref-5"><a href="#cite_note-5">[5]</a></sup> 공개적으로는 별 다른 활동없이 <a href="/wiki/%ED%83%80%EB%B8%94%EB%A1%9C" title="타블로">타블로</a>와 <a href="/wiki/%EB%AF%B8%EC%93%B0%EB%9D%BC_%EC%A7%84" title="미쓰라 진">미쓰라 진</a>은 타 가수 음반 작업 및 O.S.T에 참여하거나 콘서트 및 게스트로 참여하였고 <a href="/wiki/2010%EB%85%84_1%EC%9B%94" title="2010년 1월">2010년 1월</a>에는 <a href="/wiki/%ED%94%84%EB%9E%91%EC%8A%A4" title="프랑스">프랑스</a>에서 열린 국제 음악축제인 《미뎀 2010》에 참여하였다.<sup class="reference" id="cite_ref-6"><a href="#cite_note-6">[6]</a></sup><sup class="reference" id="cite_ref-7"><a href="#cite_note-7">[7]</a></sup></p>
<p>이후 전 소속사인 <a class="mw-redirect" href="/wiki/%EC%9A%B8%EB%A6%BC_%EC%97%94%ED%84%B0%ED%85%8C%EC%9D%B8%EB%A8%BC%ED%8A%B8" title="울림 엔터테인먼트">울림 엔터테인먼트</a>와의 <a href="/wiki/%EB%A7%B5%EB%8D%94%EC%86%8C%EC%9A%B8" title="맵더소울">맵더소울</a>의 합병을 알리며 <a href="/wiki/1%EC%9B%94_31%EC%9D%BC" title="1월 31일">1월 31일</a>, 프랑스에서 돌아온 에픽하이는 <a href="/wiki/2010%EB%85%84_3%EC%9B%94" title="2010년 3월">2010년 3월</a>에 스페셜 음반을 발매한다고 발표하였다.<sup class="reference" id="cite_ref-8"><a href="#cite_note-8">[8]</a></sup><sup class="reference" id="cite_ref-9"><a href="#cite_note-9">[9]</a></sup> 또한 간접적으로 <a href="/wiki/TBNY" title="TBNY">TBNY</a>의 <a href="/wiki/%EC%96%80%ED%82%A4" title="얀키">얀키</a>가 참여함을 밝혔다.<sup class="reference" id="cite_ref-10"><a href="#cite_note-10">[10]</a></sup> 트랙리스트가 공개되면서 6번 트랙 〈비늘〉에 참가함이 밝혀졌다. <a href="/wiki/2%EC%9B%94_6%EC%9D%BC" title="2월 6일">2월 6일</a>, <a href="/wiki/DJ_%ED%88%AC%EC%BB%B7" title="DJ 투컷">DJ 투컷</a>의 휴가를 알리면서 투컷 또한 스페셜 음반에 참여함을 간접적으로 알렸다.<sup class="reference" id="cite_ref-11"><a href="#cite_note-11">[11]</a></sup> 다음 날엔 음반에 대해 4집 <i><a href="/wiki/Remapping_the_Human_Soul" title="Remapping the Human Soul">Remapping the Human Soul</a></i>과 비슷할 것이라고 알렸다.<sup class="reference" id="cite_ref-12"><a href="#cite_note-12">[12]</a></sup> 이어 <a href="/wiki/%ED%8A%B8%EC%9C%84%ED%84%B0" title="트위터">트위터</a>를 통해 작업하는 모습을 올렸으며 <a href="/wiki/2%EC%9B%94_11%EC%9D%BC" title="2월 11일">2월 11일</a>, 음반의 마무리 작업 중임을 알리며 음반의 제목은 ‘Epilogue’이며 변동없이 <a href="/wiki/2010%EB%85%84_3%EC%9B%94" title="2010년 3월">2010년 3월</a>에 발매될 것이라고 밝혔다.<sup class="reference" id="cite_ref-13"><a href="#cite_note-13">[13]</a></sup></p>
<p><a href="/wiki/2%EC%9B%94_18%EC%9D%BC" title="2월 18일">2월 18일</a>, 녹음 작업을 마무리하였으며 믹싱 작업을 앞두고 현재 음반의 자켓을 디자인 중이라고 밝혔다.<sup class="reference" id="cite_ref-14"><a href="#cite_note-14">[14]</a></sup><sup class="reference" id="cite_ref-15"><a href="#cite_note-15">[15]</a></sup> 또한 맵더소울도 리뉴얼 중이라고 알렸다. <a href="/wiki/2%EC%9B%94_21%EC%9D%BC" title="2월 21일">2월 21일</a>에는 음반의 녹음을 끝마쳤다.<sup class="reference" id="cite_ref-16"><a href="#cite_note-16">[16]</a></sup></p>
<p>이어 <a href="/wiki/%ED%83%80%EB%B8%94%EB%A1%9C" title="타블로">타블로</a>는 트위터를 통해 <i>Epilogue</i>의 발매일이 <a href="/wiki/3%EC%9B%94_9%EC%9D%BC" title="3월 9일">3월 9일</a>로 정해졌다고 밝혔고<sup class="reference" id="cite_ref-17"><a href="#cite_note-17">[17]</a></sup> 다음 날, <a href="/wiki/%EB%AF%B8%EC%93%B0%EB%9D%BC_%EC%A7%84" title="미쓰라 진">미쓰라 진</a>은 현재 믹싱 작업과 뮤직비디오 촬영 중임을 알렸다.<sup class="reference" id="cite_ref-18"><a href="#cite_note-18">[18]</a></sup> <a href="/wiki/2%EC%9B%94_25%EC%9D%BC" title="2월 25일">2월 25일</a>에는 음반 작업 영상 및 티저 영상을 유튜브를 통해 공개하였다. 영상에서 기존에 밝힌 바와 같이 “기존 음반에 넣으려 했던 곡들 중 컨셉이 겹치는 몇 몇 곡들을 미완성인 채로 배제하였다가 이번 음반을 통해 완성하여 공개한다”고 밝혔다.<sup class="reference" id="cite_ref-19"><a href="#cite_note-19">[19]</a></sup></p>
<h3><span class="mw-headline" id=".ED.99.9C.EB.8F.99">활동</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Epilogue_(%EC%97%90%ED%94%BD%ED%95%98%EC%9D%B4%EC%9D%98_%EC%9D%8C%EB%B0%98)&amp;action=edit&amp;section=3" title="부분 편집: 활동">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>음반을 작업하면서 <a href="/wiki/%ED%83%80%EB%B8%94%EB%A1%9C" title="타블로">타블로</a>와 <a href="/wiki/%EB%AF%B8%EC%93%B0%EB%9D%BC_%EC%A7%84" title="미쓰라 진">미쓰라 진</a>은 여러 활동을 전개하였는데 <a href="/wiki/2010%EB%85%84" title="2010년">2010년</a> <a href="/wiki/2%EC%9B%94_11%EC%9D%BC" title="2월 11일">2월 11일</a>에는 에픽하이가 프로듀싱을 맡고 있는 <a href="/wiki/%EC%9D%B8%ED%94%BC%EB%8B%88%ED%8A%B8" title="인피니트">인피니트</a>라는 신인 보이 밴드를 공개하였다.<sup class="reference" id="cite_ref-infinite_20-0"><a href="#cite_note-infinite-20">[20]</a></sup> 에픽하이는 인피니트와 함께 <a class="mw-redirect" href="/wiki/%EC%82%BC%EC%84%B1" title="삼성">삼성</a>의 <a href="/wiki/%EB%91%90%EA%B7%BC%EB%91%90%EA%B7%BC_Tomorrow" title="두근두근 Tomorrow">두근두근 Tomorrow</a> 캠페인의 일환으로 개최되고 있던 <a href="/wiki/2010%EB%85%84_%EB%8F%99%EA%B3%84_%EC%98%AC%EB%A6%BC%ED%94%BD" title="2010년 동계 올림픽">2010년 동계 올림픽</a>에 <a href="/wiki/2010%EB%85%84_%EB%8F%99%EA%B3%84_%EC%98%AC%EB%A6%BC%ED%94%BD_%EB%8C%80%ED%95%9C%EB%AF%BC%EA%B5%AD_%EC%84%A0%EC%88%98%EB%8B%A8" title="2010년 동계 올림픽 대한민국 선수단">대한민국 선수단</a>을 응원하는 동영상을 만들기도 했다.<sup class="reference" id="cite_ref-infinite_20-1"><a href="#cite_note-infinite-20">[20]</a></sup> 이어 에픽하이가 <a href="/wiki/%EB%AF%B8%EA%B5%AD" title="미국">미국</a>의 언론인 <a href="/wiki/CNN" title="CNN">CNN</a>의 "토크 아시아(Talk Asia)"에 출연하였다. 이는 <a href="/wiki/%EB%B0%95%EC%A7%80%EC%84%B1" title="박지성">박지성</a>, <a href="/wiki/%EB%B9%84_(%EA%B0%80%EC%88%98)" title="비 (가수)">비</a>, <a href="/wiki/%EC%9D%B4%EB%B3%91%ED%97%8C" title="이병헌">이병헌</a> 등 <a href="/wiki/%EB%8C%80%ED%95%9C%EB%AF%BC%EA%B5%AD" title="대한민국">대한민국</a> 출신 인물에 이은 출연으로써, <a href="/wiki/%ED%83%80%EB%B8%94%EB%A1%9C" title="타블로">타블로</a>는 “CNN이 한국 음악의 깊이와 다양성을 알고 큰 관심을 보여줘서 기뻤다. 전 세계인들에게 우리만의 힙합을 대표해 전할 수 있는 이 기회, 과분한 영광이었다”라고 소감을 밝혔다.<sup class="reference" id="cite_ref-21"><a href="#cite_note-21">[21]</a></sup> 본 인터뷰는 <a href="/wiki/4%EC%9B%94_21%EC%9D%BC" title="4월 21일">4월 21일</a> 밤 CNN을 통해 방송된다고 알렸다.<sup class="reference" id="cite_ref-22"><a href="#cite_note-22">[22]</a></sup></p>
<p><a href="/wiki/2%EC%9B%94_26%EC%9D%BC" title="2월 26일">2월 26일</a>에는 트랙리스트를 발표하였다.<sup class="reference" id="cite_ref-23"><a href="#cite_note-23">[23]</a></sup> 음반의 자켓은 타블로가 사진을 찍고 미쓰라 진이 글을 썼다고 밝혔다.<sup class="reference" id="cite_ref-24"><a href="#cite_note-24">[24]</a></sup> 이후 <a href="/wiki/2%EC%9B%94_28%EC%9D%BC" title="2월 28일">2월 28일</a>, 믹스 작업 마무리에 임박했으며 뮤직비디오 또한 촬영을 마쳤고 현재 편집 상태임을 알리며 이어 많은 활동을 할 예정이라고 알렸다.<sup class="reference" id="cite_ref-25"><a href="#cite_note-25">[25]</a></sup> <a href="/wiki/3%EC%9B%94_2%EC%9D%BC" title="3월 2일">3월 2일</a>에는 이번 음반이 모든 곳에서 판매되며, <a href="/wiki/%EB%A7%B5%EB%8D%94%EC%86%8C%EC%9A%B8" title="맵더소울">맵더소울닷컴</a> SHOP 메뉴에서는 메뉴 업데이트로 인하여 판매하지 않는다고 사전 공지를 하였다.<sup class="reference" id="cite_ref-26"><a href="#cite_note-26">[26]</a></sup> 또한 앨범 자켓을 공개하였다.</p>
<p><a href="/wiki/2%EC%9B%94_25%EC%9D%BC" title="2월 25일">2월 25일</a>, <a href="/wiki/%EB%AF%B8%EC%93%B0%EB%9D%BC_%EC%A7%84" title="미쓰라 진">미쓰라 진</a>이 <a href="/wiki/%EA%B9%80%EC%A7%84%ED%91%9C_(%EA%B0%80%EC%88%98)" title="김진표 (가수)">김진표</a>와 함께 <a href="/wiki/%EC%97%A0%EB%84%B7" title="엠넷">엠넷</a>의 새로운 프로그램을 맡았다고 밝히고<sup class="reference" id="cite_ref-27"><a href="#cite_note-27">[27]</a></sup>, 3월 1일부터 출연하였다.</p>
<p><a href="/wiki/3%EC%9B%94_3%EC%9D%BC" title="3월 3일">3월 3일</a> 새벽에 타블로가 타이틀 곡은 "Run"이라고 밝혔다.<sup class="reference" id="cite_ref-28"><a href="#cite_note-28">[28]</a></sup> 또한 <a class="new" href="/w/index.php?title=%EC%97%A0%EA%B5%B0&amp;action=edit&amp;redlink=1" title="엠군 (없는 문서)">엠군</a>, <a href="/wiki/%EC%9C%A0%ED%8A%9C%EB%B8%8C" title="유튜브">유튜브</a>, <a href="/wiki/%EC%95%84%ED%94%84%EB%A6%AC%EC%B9%B4TV" title="아프리카TV">아프리카</a> 이 세 동영상 사이트에 <a href="/wiki/%EC%9D%B8%ED%94%BC%EB%8B%88%ED%8A%B8" title="인피니트">인피니트</a>의 한 멤버인 엘(L)을 주인공으로 하고 다른 멤버들이 출연한 타이틀 곡의 뮤직비디오 티져 영상을 동시에 공개하였다. 뮤직비디오의 티져 영상은 세 편으로 나뉘어 공개될 예정이며 음반 발매 당일 날 《<a href="/wiki/%EC%9C%A0%ED%9D%AC%EC%97%B4%EC%9D%98_%EC%8A%A4%EC%BC%80%EC%B9%98%EB%B6%81" title="유희열의 스케치북">유희열의 스케치북</a>》 녹화를 시작으로 활동을 시작한다고 밝혔다.<sup class="reference" id="cite_ref-29"><a href="#cite_note-29">[29]</a></sup> 당일 오전부터 트위터를 통해 음반이 마스터링 작업에 돌입했음을 알렸고 음반 발매 후부터 송 노트를 게시하겠다고 밝혔다.<sup class="reference" id="cite_ref-30"><a href="#cite_note-30">[30]</a></sup> 오후에는 티저 첫 번째 편인 <a href="/wiki/%ED%83%80%EB%B8%94%EB%A1%9C" title="타블로">타블로</a> 편을 공개하였다.<sup class="reference" id="cite_ref-31"><a href="#cite_note-31">[31]</a></sup> 다음 날, 두 번째 "미쓰라 진 편"이 나왔다.<sup class="reference" id="cite_ref-second_3-1"><a href="#cite_note-second-3">[3]</a></sup> 마지막 3월 5일에 세 번째 티저인 "Run" 뮤직비디오 티저와 함께 노래의 일부분을 공개했다.<sup class="reference" id="cite_ref-32"><a href="#cite_note-32">[32]</a></sup></p>
<p><a href="/wiki/%EB%84%A4%EC%9D%B4%EB%B2%84" title="네이버">네이버</a> 웹툰 《싸우자 귀신아》의 작가인 임인스는 "R U N"이라는 제목의 글에 에픽하이 새 음반 소개 만화를 그렸다.<sup class="reference" id="cite_ref-33"><a href="#cite_note-33">[33]</a></sup> 또한 <a href="/wiki/%EB%A7%B5%EB%8D%94%EC%86%8C%EC%9A%B8" title="맵더소울">맵더소울</a>에는 이 만화를 <a href="/wiki/%EC%98%81%EC%96%B4" title="영어">영어</a>로 번역한 티저 만화가 올라왔다.<sup class="reference" id="cite_ref-34"><a href="#cite_note-34">[34]</a></sup><sup class="reference" id="cite_ref-35"><a href="#cite_note-35">[35]</a></sup> 또한 절친한 사이로 알려진 <a href="/wiki/%EC%BD%94%EC%9A%94%ED%83%9C" title="코요태">코요태</a>의 <a href="/wiki/%EB%B9%BD%EA%B0%80" title="빽가">빽가</a>는 에픽하이의 포스터 촬영을 맡았다.<sup class="reference" id="cite_ref-36"><a href="#cite_note-36">[36]</a></sup></p>
<p><a href="/wiki/3%EC%9B%94_9%EC%9D%BC" title="3월 9일">3월 9일</a> 발매 당일 새벽에 타이틀곡 "Run"의 싱크가 어긋난 불법 뮤직비디오가 유출되어 파장을 일으켰다.<sup class="reference" id="cite_ref-mv_37-0"><a href="#cite_note-mv-37">[37]</a></sup> 그 영향으로 에픽하이는 공식 뮤직비디오를 다소 일찍 공개했고<sup class="reference" id="cite_ref-38"><a href="#cite_note-38">[38]</a></sup>, 타블로와 미쓰라 진은 뮤직비디오가 유출된데에 격앙된 표현으로 실망감을 감추지 않았다.<sup class="reference" id="cite_ref-39"><a href="#cite_note-39">[39]</a></sup><sup class="reference" id="cite_ref-40"><a href="#cite_note-40">[40]</a></sup></p>
<p>발매 다음 날 3월 10일에는 어떠한 해외 프로모션 없이 <a href="/wiki/%EB%AF%B8%EA%B5%AD" title="미국">미국</a> <a class="mw-redirect" href="/wiki/%EC%95%84%EC%9D%B4%ED%8A%A0%EC%A6%88" title="아이튠즈">아이튠즈</a> "힙합/랩 음반 차트"에서 1위를 하기도 하였다.<sup class="reference" id="cite_ref-41"><a href="#cite_note-41">[41]</a></sup> 또한 <a href="/wiki/%EB%89%B4%EC%A7%88%EB%9E%9C%EB%93%9C" title="뉴질랜드">뉴질랜드</a> 아이튠즈 1위, <a href="/wiki/%EC%BA%90%EB%82%98%EB%8B%A4" title="캐나다">캐나다</a> 아이튠즈 3위, 한터차트 1위 등 세계 곳곳의 차트에서 발매 하루 만에 상위권에 들었다.<sup class="reference" id="cite_ref-42"><a href="#cite_note-42">[42]</a></sup> <a href="/wiki/3%EC%9B%94_12%EC%9D%BC" title="3월 12일">3월 12일</a>에 <a href="/wiki/%EB%AE%A4%EC%A7%81%EB%B1%85%ED%81%AC" title="뮤직뱅크">뮤직뱅크</a>를 통해 공중파에 복귀하였다.<sup class="reference" id="cite_ref-43"><a href="#cite_note-43">[43]</a></sup> 이어 《<a href="/wiki/%EC%9C%A0%ED%9D%AC%EC%97%B4%EC%9D%98_%EC%8A%A4%EC%BC%80%EC%B9%98%EB%B6%81" title="유희열의 스케치북">유희열의 스케치북</a>》에 출연하였고<sup class="reference" id="cite_ref-44"><a href="#cite_note-44">[44]</a></sup> 곧바로 《<a href="/wiki/%EC%87%BC!_%EC%9D%8C%EC%95%85%EC%A4%91%EC%8B%AC" title="쇼! 음악중심">쇼! 음악중심</a>》, 《<a class="mw-redirect" href="/wiki/%EC%9D%B8%EA%B8%B0%EA%B0%80%EC%9A%94" title="인기가요">인기가요</a>》 등 방송 3사 음악 프로그램에 출연하며 본격적으로 활동하기 시작하였다.<sup class="reference" id="cite_ref-45"><a href="#cite_note-45">[45]</a></sup></p>
<p><a href="/wiki/3%EC%9B%94_13%EC%9D%BC" title="3월 13일">3월 13일</a>, 에픽하이는 대한민국 힙합 웹사이트 <a href="/wiki/%ED%9E%99%ED%95%A9%ED%94%8C%EB%A0%88%EC%9D%B4%EC%95%BC" title="힙합플레이야">힙합플레이야</a>의 10주년을 맞아 열리는 힙플쇼(Hiphop Playa Show, 힙합플레이야 공연)에 <a class="mw-redirect" href="/wiki/%EC%8A%88%ED%94%84%EB%A6%BC_%ED%8C%80" title="슈프림 팀">슈프림 팀</a> 등과 1차 출연진으로 공개되었다.<sup class="reference" id="cite_ref-46"><a href="#cite_note-46">[46]</a></sup> 또한 <a href="/wiki/2010%EB%85%84" title="2010년">2010년</a> <a href="/wiki/3%EC%9B%94_20%EC%9D%BC" title="3월 20일">3월 20일</a> <a href="/wiki/%EC%9C%A0%ED%8A%9C%EB%B8%8C" title="유튜브">유튜브</a>가 개최한 《유튜브 뮤직데이》에 헤드라이너로 참여하였다. <a href="/wiki/4%EC%9B%94_11%EC%9D%BC" title="4월 11일">4월 11일</a>에는 <a href="/wiki/%EB%A6%AC%EB%B0%94%EC%9D%B4%EC%8A%A4" title="리바이스">리바이스</a>가 주최하는 <i>Keep The Faith</i> 콘서트에 <a href="/wiki/%EC%9C%A4%EB%AF%B8%EB%9E%98" title="윤미래">윤미래</a> 등과 참여하였다.<sup class="reference" id="cite_ref-47"><a href="#cite_note-47">[47]</a></sup> 이어 공중파 방송에 출연하며 <a href="/wiki/4%EC%9B%94_17%EC%9D%BC" title="4월 17일">4월 17일</a>에 <a href="/wiki/%ED%9E%99%ED%95%A9%ED%94%8C%EB%A0%88%EC%9D%B4%EC%95%BC" title="힙합플레이야">힙합플레이야</a> 10주년 쇼에 헤드라이너로 참여하였다. <a href="/wiki/%EB%AF%B8%EC%93%B0%EB%9D%BC_%EC%A7%84" title="미쓰라 진">미쓰라 진</a>을 제외하고 <a href="/wiki/%ED%83%80%EB%B8%94%EB%A1%9C" title="타블로">타블로</a>는 아내 <a href="/wiki/%EA%B0%95%ED%98%9C%EC%A0%95" title="강혜정">강혜정</a>의 출산으로 인해 5월 초까지 휴식을 가졌다.<sup class="reference" id="cite_ref-48"><a href="#cite_note-48">[48]</a></sup> <a href="/wiki/5%EC%9B%94_2%EC%9D%BC" title="5월 2일">5월 2일</a>, 타블로가 득녀를 하고 복귀한 후<sup class="reference" id="cite_ref-49"><a href="#cite_note-49">[49]</a></sup> 활동은 기존에 녹화한 음악 프로그램과 예능 프로그램에 출연하였으며 주로 라디오 프로그램 게스트로 참여하였다. 이후 <a href="/wiki/5%EC%9B%94_26%EC%9D%BC" title="5월 26일">5월 26일</a>, 타블로는 트위터를 통해 <i>Epilogue</i> 활동 마감을 알렸다.<sup class="reference" id="cite_ref-50"><a href="#cite_note-50">[50]</a></sup></p>
<h2><span class="mw-headline" id=".EC.84.B1.EA.B3.BC">성과</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Epilogue_(%EC%97%90%ED%94%BD%ED%95%98%EC%9D%B4%EC%9D%98_%EC%9D%8C%EB%B0%98)&amp;action=edit&amp;section=4" title="부분 편집: 성과">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p><i>Epilogue</i>는 한터차트에 의하면 2010년 한 해 동안 총 18,000장이 팔려 2010년 32위를 기록했다.<sup class="reference" id="cite_ref-51"><a href="#cite_note-51">[51]</a></sup> 가온차트에 의해 집계된 판매량은 26,000장으로, 순위는 50위를 기록하였다.<sup class="reference" id="cite_ref-52"><a href="#cite_note-52">[52]</a></sup> 다른 음반 활동과는 달리 활동 도중에 <a href="/wiki/%ED%83%80%EB%B8%94%EB%A1%9C%EC%9D%98_%EC%8A%A4%ED%83%A0%ED%8D%BC%EB%93%9C_%EB%8C%80%ED%95%99%EA%B5%90_%ED%95%99%EB%A0%A5_%EC%9C%84%EC%A1%B0_%EB%85%BC%EB%9E%80" title="타블로의 스탠퍼드 대학교 학력 위조 논란">큰 악재</a>가 터져 별다른 수상이나 성과는 거두지 못하였다.</p>
<h2><span class="mw-headline" id=".EC.9D.8C.EB.B0.98_.EC.A0.95.EB.B3.B4">음반 정보</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Epilogue_(%EC%97%90%ED%94%BD%ED%95%98%EC%9D%B4%EC%9D%98_%EC%9D%8C%EB%B0%98)&amp;action=edit&amp;section=5" title="부분 편집: 음반 정보">편집</a><span class="mw-editsection-bracket">]</span></span></h2>

<h3><span class="mw-headline" id=".EB.B6.80.ED.81.B4.EB.A6.BF">부클릿</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Epilogue_(%EC%97%90%ED%94%BD%ED%95%98%EC%9D%B4%EC%9D%98_%EC%9D%8C%EB%B0%98)&amp;action=edit&amp;section=7" title="부분 편집: 부클릿">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>가사집 및 음반 정보 등 11장으로 이루어져 있으며 <a href="/wiki/%ED%83%80%EB%B8%94%EB%A1%9C" title="타블로">타블로</a>가 직접 촬영한 8장의 사진들과 <a href="/wiki/%EB%AF%B8%EC%93%B0%EB%9D%BC_%EC%A7%84" title="미쓰라 진">미쓰라 진</a>이 작성한 글이 수록되어 있다.</p>
<h3><span class="mw-headline" id=".EB.9D.BC.EC.9D.B4.EB.84.88_.EB.85.B8.ED.8A.B8">라이너 노트</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Epilogue_(%EC%97%90%ED%94%BD%ED%95%98%EC%9D%B4%EC%9D%98_%EC%9D%8C%EB%B0%98)&amp;action=edit&amp;section=8" title="부분 편집: 라이너 노트">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>아래는 음반 발매 당일부터 <a href="/wiki/%ED%83%80%EB%B8%94%EB%A1%9C" title="타블로">타블로</a>가 <a class="external text" href="http://twitter.com/blobyblo" rel="nofollow">자신의 트위터</a> 를 통해서 트랙 순으로 공개한 <i>Epilogue</i>의 라이너 노트이다. (일부는 인터뷰 중 밝힌 내용 포함.)</p>
<ul>
<li><b>서랍 &amp; Run</b>: 준비, 질주, 숨돌리기, 예측하지 못한 정지, 재출발 등 인생을 달리기에 비유하여 달리는 느낌을 음악으로 표현하고 싶었다고 밝혔다. 또한 〈서랍〉은 여태 에픽하이가 발매해왔던 타이틀곡 및 후속곡의 제목을 두 글자의 단어로 담아냈다.</li>
<li><b>바보</b>: 사랑 앞에 작아지는 사람들이 갈수록 흔하지 않은 세상 속에서 바보의 순수함을 잃지 못하는 사람들을 위한 노래이며, 심플한 리듬 위주 편곡을 하였다고 밝혔다. 다른 인터뷰에서는 <a href="/wiki/%ED%95%84%ED%84%B0_(%EC%9D%8C%EC%95%85%EA%B0%80)" title="필터 (음악가)">필터</a>를 보고 썼다고 밝혔다.<sup class="reference" id="cite_ref-epep_53-0"><a href="#cite_note-epep-53">[53]</a></sup></li>
<li><b>Wordkill</b>: 5집 <i><a href="/wiki/Pieces,_Part_One" title="Pieces, Part One">Pieces, Part One</a></i> 작업할 때 쓴 곡. 타블로가 당시 어떤 악플을 본 후 작곡하였으며 <i><a href="/wiki/Lost_Maps" title="Lost Maps">Lost Maps #001</a></i>에 수록된 데모 버전과는 달리 훅을 바꿨다.<sup class="reference" id="cite_ref-epiloguehp_54-0"><a href="#cite_note-epiloguehp-54">[54]</a></sup></li>
<li><b>Blossom</b>: <a href="/wiki/%EC%9D%B4%ED%84%B0%EB%84%90_%EB%AA%A8%EB%8B%9D" title="이터널 모닝">이터널 모닝</a>의 멤버로서 작업한 연주곡이며 피아노를 치고 녹음해서 왜곡시켜 동양적인 느낌을 담고 싶었다고 밝혔다.</li>
<li><b>비늘</b>: 유일한 이번 음반의 랩 피쳐링 곡이며, 과거에 타블로와 미쓰라 진, 그리고 얀키가 했었던 〈<a href="/wiki/%EB%B9%84%EC%9D%98_%EB%9E%A9%EC%86%8C%EB%94%94" title="비의 랩소디">비의 랩소디</a>〉의 후속작이라고 밝혔다.</li>
<li><b>잡음</b>: 라디오 주파수 잡음을 녹음해서 리듬으로 꾸몄으며 작곡보다 편곡에 집중한 곡이라고 밝혔다. 애당초 타블로의 랩 파트가 있었으나 미쓰라 진이 잘한 나머지 포기하였으며 또한 다음엔 이와같이 "생활의 소리"들로만 한 곡을 완성할 생각이라고 밝혔다.<sup class="reference" id="cite_ref-epiloguehp_54-1"><a href="#cite_note-epiloguehp-54">[54]</a></sup></li>
<li><b>Over</b>: 항상 최고여야 하는 사람들에 대한 안타까움으로 만든 곡이다. 또한 해외 팬과 적극적으로 교류하기 위하여 <a href="/wiki/%EC%98%81%EC%96%B4" title="영어">영어</a>로 만들었다.<sup class="reference" id="cite_ref-epep_53-1"><a href="#cite_note-epep-53">[53]</a></sup></li>
<li><b>Coffee</b>: 한 시트콤의 오프닝곡으로 쓰려다 우울하다고 퇴짜를 맞았고, 또 <a href="/wiki/Pieces,_Part_One" title="Pieces, Part One">5집</a>에 수록하려다가 〈우산〉과 겹쳐 수록하지 못했다.</li>
</ul>
<h3><span class="mw-headline" id=".EB.AE.A4.EC.A7.81.EB.B9.84.EB.94.94.EC.98.A4">뮤직비디오</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Epilogue_(%EC%97%90%ED%94%BD%ED%95%98%EC%9D%B4%EC%9D%98_%EC%9D%8C%EB%B0%98)&amp;action=edit&amp;section=9" title="부분 편집: 뮤직비디오">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>타이틀 곡 "Run"의 뮤직비디오는 <a class="new" href="/w/index.php?title=%EC%9F%88%EB%8B%88%EB%B8%8C%EB%A1%9C%EC%8A%A4&amp;action=edit&amp;redlink=1" title="쟈니브로스 (없는 문서)">쟈니브로스</a>의 홍원기 감독이 맡았으며 <a href="/wiki/%EC%97%90%ED%94%BD%ED%95%98%EC%9D%B4" title="에픽하이">에픽하이</a>를 비롯한 같은 소속사 후배 그룹인 <a href="/wiki/%EC%9D%B8%ED%94%BC%EB%8B%88%ED%8A%B8" title="인피니트">인피니트</a>가 출연하였다. <a href="/wiki/3%EC%9B%94_5%EC%9D%BC" title="3월 5일">3월 5일</a>, 뮤직비디오의 티저 영상을 공개하며 음반 발매일이자 뮤직비디오 공개일이었던 <a href="/wiki/3%EC%9B%94_9%EC%9D%BC" title="3월 9일">3월 9일</a>, 정각 무렵 뮤직비디오가 한 사이트를 통해 사전 유출되어 예정보다 빨리 공개하였다.<sup class="reference" id="cite_ref-mv_37-1"><a href="#cite_note-mv-37">[37]</a></sup> 사막 한 가운데에 밴드 장면에서 인피니트와 함께 <a href="/wiki/%ED%83%80%EB%B8%94%EB%A1%9C" title="타블로">타블로</a>, <a href="/wiki/%EB%AF%B8%EC%93%B0%EB%9D%BC_%EC%A7%84" title="미쓰라 진">미쓰라 진</a>이 나온다. 또한 인피니트의 엘이 주인공을 맡았으며 출생에서부터 성인이 된 이후의 모습을 1인칭 시점으로 보여준다.<sup class="reference" id="cite_ref-55"><a href="#cite_note-55">[55]</a></sup></p>
<p>뮤직비디오의 시작과 함께 출생부터 성장을 하는 모습과 함께 달리는 모습이 번갈아 나오며, 탈선 후 오토바이를 타게 되고 여자친구를 만난다. 그러나 오토바이 사고와 다른 남자에게 여자친구를 잃게 되고, 이후 복서로서 열심히 훈련하기 시작한다. 마지막 장면에선 신발끈을 오른쪽만 묶는 복선과 함께 왼쪽 다리에 의족을 찬 채 달리는 모습을 보여주며 사고로 다리를 잃었음을 보여준다. 계속해서 훈련하는 모습이 나오며 끝난다.</p>
<h2><span class="mw-headline" id=".ED.8F.89.EA.B0.80">평가</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Epilogue_(%EC%97%90%ED%94%BD%ED%95%98%EC%9D%B4%EC%9D%98_%EC%9D%8C%EB%B0%98)&amp;action=edit&amp;section=10" title="부분 편집: 평가">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p><a href="/wiki/%EB%84%A4%EC%9D%B4%EB%B2%84" title="네이버">네이버 뮤직</a>은 한국대중음악상 선정위원단과 네티즌 선정위원단의 평가로 별점 5.5점을 부여했다. 한국대중음악상 선정위원단은 담백하고 색깔 뚜렷한 작품이라 평했으며, 네티즌 선정위원단은 <a href="/wiki/DJ_%ED%88%AC%EC%BB%B7" title="DJ 투컷">DJ 투컷</a>의 공백이 느껴졌다고 평했다.<sup class="reference" id="cite_ref-naver_1-1"><a href="#cite_note-naver-1">[1]</a></sup> 음악 리뷰 사이트 이즘은 부드럽지만 몇 년 동안 에픽하이가 보여준 모습 그대로라며 식상하다는 평을 내렸다.<sup class="reference" id="cite_ref-izm_2-1"><a href="#cite_note-izm-2">[2]</a></sup> 그 외에도 에픽하이 특유의 모습을 보여줬지만 기존의 음악을 답습하고 너무 안정적이면서 질린다는 평도 있다.<sup class="reference" id="cite_ref-56"><a href="#cite_note-56">[56]</a></sup></p>
