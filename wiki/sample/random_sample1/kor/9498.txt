<p><b>야스다 쇼타</b> (<span style="font-size: smaller;"><a href="/wiki/%EC%9D%BC%EB%B3%B8%EC%96%B4" title="일본어">일본어</a>:</span> <span lang="ja"><span lang="ja"><ruby><rb>安田章大</rb> <rp>(</rp><rt>やすだしょうた</rt><rp>)</rp></ruby></span></span>, 1984년 9월 11일 ~ )는 <a href="/wiki/%EC%9D%BC%EB%B3%B8" title="일본">일본</a>의 <a href="/wiki/%EB%B0%B0%EC%9A%B0" title="배우">배우</a>이자, <a href="/wiki/%EA%B0%80%EC%88%98" title="가수">가수</a>이다. <a class="mw-redirect" href="/wiki/%EC%9E%90%EB%8B%88%EC%A6%88_%EC%82%AC%EB%AC%B4%EC%86%8C" title="자니즈 사무소">자니즈 사무소</a>에 소속의 <a href="/wiki/%EC%B9%B8%EC%9F%88%EB%8B%88%E2%88%9E" title="칸쟈니∞">칸쟈니∞</a>의 멤버이다. <a href="/wiki/%ED%9A%A8%EA%B3%A0_%ED%98%84" title="효고 현">효고 현</a> <a href="/wiki/%EC%95%84%EB%A7%88%EA%B0%80%EC%82%AC%ED%82%A4_%EC%8B%9C" title="아마가사키 시">아마가사키 시</a> 출신. 애칭은 <b>야스(ヤス)</b>, <b>쇼짱(章ちゃん)</b>, <b>얏상(やっさん)</b> 등. <a href="/wiki/%EC%95%84%EB%9D%BC%EC%8B%9C" title="아라시">아라시</a> 멤버 <a href="/wiki/%EC%95%84%EC%9D%B4%EB%B0%94_%EB%A7%88%EC%82%AC%ED%82%A4" title="아이바 마사키">아이바 마사키</a> 와 사이가 좋다.</p>

<h2><span class="mw-headline" id=".EC.95.BD.EB.A0.A5">약력</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%95%BC%EC%8A%A4%EB%8B%A4_%EC%87%BC%ED%83%80&amp;action=edit&amp;section=1" title="부분 편집: 약력">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<ul>
<li><a href="/wiki/1997%EB%85%84" title="1997년">1997년</a> <a href="/wiki/9%EC%9B%94" title="9월">9월</a>, <a href="/wiki/%EC%95%84%EC%9D%B4%EB%B0%94_%EB%A7%88%EC%82%AC%ED%82%A4" title="아이바 마사키">아이바 마사키</a>(현재 <a href="/wiki/%EC%95%84%EB%9D%BC%EC%8B%9C" title="아라시">아라시</a>로 당시 쟈니즈 주니어)의 팬이었던 누나가 자신의 마음대로 이력서를 보냈다. 야스다 본인은 내키지 않았지만 "어차피 합격하지 않을테니까 가지 않아도 돼"라는 어머니의 말에 반항심을 느껴 오디션을 보게 되었고, <a class="mw-redirect" href="/wiki/%EC%9E%90%EB%8B%88%EC%A6%88_%EC%82%AC%EB%AC%B4%EC%86%8C" title="자니즈 사무소">쟈니스 사무소</a>에 입소하게 된다.</li>
<li>입소 동기로는 <a href="/wiki/%EB%8B%88%EC%8B%9C%ED%82%A4%EB%8F%84_%EB%A3%8C" title="니시키도 료">니시키도 료</a>, <a href="/wiki/%EB%A7%88%EB%A3%A8%EC%95%BC%EB%A7%88_%EB%A5%98%ED%97%A4%EC%9D%B4" title="마루야마 류헤이">마루야마 류헤이</a>, <a class="mw-redirect" href="/wiki/%EC%98%A4%EC%BF%A0%EB%9D%BC_%EB%8B%A4%EB%8B%A4%EC%9A%94%EC%8B%9C" title="오쿠라 다다요시">오쿠라 다다요시</a>가 있다.<sup class="reference" id="cite_ref-1"><a href="#cite_note-1">[1]</a></sup></li>
