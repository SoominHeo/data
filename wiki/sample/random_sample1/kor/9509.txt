<p><b>야인시대</b>(野人時代)는 <a class="mw-disambig" href="/wiki/SBS" title="SBS">SBS</a>에서 <a href="/wiki/2002%EB%85%84" title="2002년">2002년</a> <a href="/wiki/7%EC%9B%94_29%EC%9D%BC" title="7월 29일">7월 29일</a>부터 <a href="/wiki/2003%EB%85%84" title="2003년">2003년</a> <a href="/wiki/9%EC%9B%94_30%EC%9D%BC" title="9월 30일">9월 30일</a>까지 총 124부작 방송한 김두한의 일대기를 그린 <a class="mw-redirect" href="/wiki/%EB%8C%80%ED%95%98%EB%93%9C%EB%9D%BC%EB%A7%88" title="대하드라마">대하드라마</a>이다. 극본은 <a href="/wiki/%EC%9D%B4%ED%99%98%EA%B2%BD_(%EC%9E%91%EA%B0%80)" title="이환경 (작가)">이환경</a> 작가, 연출은 <a href="/wiki/%EC%9E%A5%ED%98%95%EC%9D%BC" title="장형일">장형일</a> 감독이다.</p>
<p>최고 시청률 57.1%<sup class="reference" id="cite_ref-2"><a href="#cite_note-2">[2]</a></sup>, 평균 30%대 등 드라마로서 이례적으로 <a href="/wiki/%EB%A7%9E%EC%9D%B4%EB%B0%A9" title="맞이방">맞이방</a>에서 시청하는 풍경, 방영시간에 거리에 사람이 없다 하여 귀가 시계로 불리며 <a href="/wiki/%EB%8C%80%ED%95%9C%EB%AF%BC%EA%B5%AD" title="대한민국">대한민국</a>은 물론 <a href="/wiki/%EC%BD%94%EB%A6%AC%EC%95%84%ED%83%80%EC%9A%B4" title="코리아타운">로스앤젤레스의 코리아타운</a> 등 <a class="mw-redirect" href="/wiki/%EC%8B%A0%EB%93%9C%EB%A1%AC" title="신드롬">신드롬</a>을 일으켰으며<sup class="reference" id="cite_ref-3"><a href="#cite_note-3">[3]</a></sup><sup class="reference" id="cite_ref-4"><a href="#cite_note-4">[4]</a></sup><sup class="reference" id="cite_ref-5"><a href="#cite_note-5">[5]</a></sup><sup class="reference" id="cite_ref-6"><a href="#cite_note-6">[6]</a></sup> 방영 동안 방영 후에도 부천 야인시대세트장은 관광지가 되기도 하였다.<sup class="reference" id="cite_ref-7"><a href="#cite_note-7">[7]</a></sup> 또한, <a href="/wiki/2004%EB%85%84" title="2004년">2004년</a> <a href="/wiki/%EB%AA%BD%EA%B3%A8" title="몽골">몽골</a>에서도 방송되어 김두한 역을 연기했던 <a href="/wiki/%EB%B0%B0%EC%9A%B0" title="배우">배우</a> <a href="/wiki/%EC%95%88%EC%9E%AC%EB%AA%A8" title="안재모">안재모</a>는 <a class="mw-redirect" href="/wiki/%EC%82%AC%ED%9A%8C%EC%A3%BC%EC%9D%98%EC%9E%90" title="사회주의자">사회주의자</a>인 <a href="/wiki/%EB%82%98%EC%B0%A8%EA%B8%B4_%EB%B0%94%EA%B0%80%EB%B0%98%EB%94%94" title="나차긴 바가반디">나차긴 바가반디</a> <a href="/wiki/%EB%AA%BD%EA%B3%A8" title="몽골">몽골</a> <a href="/wiki/%EB%8C%80%ED%86%B5%EB%A0%B9" title="대통령">대통령</a>으로부터 국빈초청을 받기도 했으며 주로 러시아 문화의 영향을 받아왔던 <a href="/wiki/%EB%AA%BD%EA%B3%A8" title="몽골">몽골</a>에서 <a class="mw-redirect" href="/wiki/%ED%95%9C%EA%B5%AD_%EB%93%9C%EB%9D%BC%EB%A7%88" title="한국 드라마">한국 드라마</a>가 방송되는 기폭제가 되었다.<sup class="reference" id="cite_ref-8"><a href="#cite_note-8">[8]</a></sup><sup class="reference" id="cite_ref-9"><a href="#cite_note-9">[9]</a></sup> <a href="/wiki/%EC%9D%BC%EB%B3%B8" title="일본">일본</a> <a href="/wiki/KNTV_(%EC%9D%BC%EB%B3%B8%EC%9D%98_%EB%B0%A9%EC%86%A1%EC%82%AC)" title="KNTV (일본의 방송사)">KNTV</a>에서 방송과<sup class="reference" id="cite_ref-10"><a href="#cite_note-10">[10]</a></sup> <a href="/wiki/%EC%A1%B0%EC%84%A0%EB%AF%BC%EC%A3%BC%EC%A3%BC%EC%9D%98%EC%9D%B8%EB%AF%BC%EA%B3%B5%ED%99%94%EA%B5%AD" title="조선민주주의인민공화국">조선민주주의인민공화국</a>에서도 인기를 끌었다.<sup class="reference" id="cite_ref-11"><a href="#cite_note-11">[11]</a></sup></p>

<h2><span class="mw-headline" id=".EC.9D.B4.EB.AA.A8.EC.A0.80.EB.AA.A8">이모저모</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%95%BC%EC%9D%B8%EC%8B%9C%EB%8C%80&amp;action=edit&amp;section=27" title="부분 편집: 이모저모">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<ul>
<li>당초 주말 특별기획드라마로 예정되었으나 작가 이환경이 집필해 왔던 KBS 1TV <a href="/wiki/%EC%A0%9C%EA%B5%AD%EC%9D%98_%EC%95%84%EC%B9%A8" title="제국의 아침">제국의 아침</a>이 주말에 방영되고 있던 탓인지<sup class="reference" id="cite_ref-12"><a href="#cite_note-12">[12]</a></sup> 월화드라마 편성으로 바뀌었다.</li>
<li>2003년 12월 열린 제 16회 한국방송작가상(2002년 중후반기부터의 내용 위주) 후보에 오르기도 했지만 "한번 작가상을 받은 사람은 다시 받을 수 없다"는<sup class="reference" id="cite_ref-13"><a href="#cite_note-13">[13]</a></sup> 규정 때문에 탈락했다.</li>
</ul>
