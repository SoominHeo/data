<p><b>야코프 미하일로비치 스베르들로프</b>(<span style="font-size: smaller;"><a href="/wiki/%EB%9F%AC%EC%8B%9C%EC%95%84%EC%96%B4" title="러시아어">러시아어</a>:</span> <span lang="ru" xml:lang="ru">Я́ков Миха́йлович Свердло́в</span>, <a href="/wiki/1885%EB%85%84" title="1885년">1885년</a> <a href="/wiki/6%EC%9B%94_3%EC%9D%BC" title="6월 3일">6월 3일</a> ~ <a href="/wiki/1919%EB%85%84" title="1919년">1919년</a> <a href="/wiki/3%EC%9B%94_16%EC%9D%BC" title="3월 16일">3월 16일</a>)은 <a class="mw-redirect" href="/wiki/%EC%86%8C%EB%B9%84%EC%97%90%ED%8A%B8_%EC%97%B0%EB%B0%A9" title="소비에트 연방">소비에트 연방</a>의 혁명가, 정치가이다. <a href="/wiki/%EB%9F%AC%EC%8B%9C%EC%95%84_%EC%82%AC%ED%9A%8C%EB%AF%BC%EC%A3%BC%EB%85%B8%EB%8F%99%EB%8B%B9" title="러시아 사회민주노동당">러시아 사회민주노동당</a>(РСДРП)과 <a href="/wiki/%EC%86%8C%EB%A0%A8_%EA%B3%B5%EC%82%B0%EB%8B%B9" title="소련 공산당">소련 공산당</a>(КПСС)의 당원이었으며, <a href="/wiki/1917%EB%85%84" title="1917년">1917년</a>부터 <a href="/wiki/1919%EB%85%84" title="1919년">1919년</a>까지 <a class="new" href="/w/index.php?title=%EC%A0%84%EB%9F%AC%EC%8B%9C%EC%95%84_%EC%86%8C%EB%B9%84%EC%97%90%ED%8A%B8_%EB%8C%80%ED%9A%8C_%EC%A4%91%EC%95%99%EC%A7%91%ED%96%89%EC%9C%84%EC%9B%90%ED%9A%8C&amp;action=edit&amp;redlink=1" title="전러시아 소비에트 대회 중앙집행위원회 (없는 문서)">전러시아 소비에트 대회 중앙집행위원회</a> 의장을 지냈다.</p>
