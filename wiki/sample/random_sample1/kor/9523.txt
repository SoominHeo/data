<h2><span class="mw-headline" id=".EC.9D.8C.EC.95.85_.EA.B5.90.EC.9C.A1">음악 교육</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%95%BC%ED%81%AC_%EC%A1%B4&amp;action=edit&amp;section=1" title="부분 편집: 음악 교육">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>Jacques Zoon 은 <a href="/wiki/%EC%95%94%EC%8A%A4%ED%85%8C%EB%A5%B4%EB%8B%B4" title="암스테르담">암스테르담</a>의 Sweelinck 음악원 에서 Koos Verheul 와 Harrie Starreveld 에게서 사사하였고 <a href="/wiki/%EC%BA%90%EB%82%98%EB%8B%A4" title="캐나다">캐나다</a>의 Bannf 아트센터에서 Geoffrey Gilbert 과 András Adorján 밑에서 수학하였다.</p>
<h2><span class="mw-headline" id=".EC.98.A4.EC.BC.80.EC.8A.A4.ED.8A.B8.EB.9D.BC_.EC.97.B0.EC.A3.BC.EC.9E.90">오케스트라 연주자</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%95%BC%ED%81%AC_%EC%A1%B4&amp;action=edit&amp;section=2" title="부분 편집: 오케스트라 연주자">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>네덜란드 청소년 오케스트라의 단원이었던 Jacques Zoon 은 그 후 네덜란드 국립 유스 오케스트라와 유럽연합 유스 오케스트라에서 <a href="/wiki/%ED%81%B4%EB%9D%BC%EC%9A%B0%EB%94%94%EC%98%A4_%EC%95%84%EB%B0%94%EB%8F%84" title="클라우디오 아바도">클라우디오 아바도</a>와 <a href="/wiki/%EB%A0%88%EB%84%88%EB%93%9C_%EB%B2%88%EC%8A%A4%ED%83%80%EC%9D%B8" title="레너드 번스타인">레너드 번스타인</a> 같은 지휘자와 함께 활동을 시작한다. 그 후 <a class="new" href="/w/index.php?title=%EC%95%94%EC%8A%A4%ED%85%8C%EB%A5%B4%EB%8B%B4_%ED%95%84%ED%95%98%EB%AA%A8%EB%8B%88_%EA%B4%80%ED%98%84%EC%95%85%EB%8B%A8&amp;action=edit&amp;redlink=1" title="암스테르담 필하모니 관현악단 (없는 문서)">암스테르담 필하모니 관현악단</a>과 <a class="mw-redirect" href="/wiki/%ED%97%A4%EC%9D%B4%EA%B7%B8_%ED%95%84%ED%95%98%EB%AA%A8%EB%8B%89%EC%9D%98_%EA%B4%80%ED%98%84%EC%95%85%EB%8B%A8" title="헤이그 필하모닉의 관현악단">헤이그 필하모닉의 관현악단</a>의 수석 플루티스트를 역임하였다. <a href="/wiki/1988%EB%85%84" title="1988년">1988년</a>에서 <a href="/wiki/1994" title="1994">1994</a>까지, 지휘자 <a href="/wiki/%EB%A6%AC%EC%B9%B4%EB%A5%B4%EB%8F%84_%EC%83%A4%EC%9D%B4" title="리카르도 샤이">리카르도 샤이</a> 밑에서 세계 제일의 오케스트라로 꼽히는 <a class="mw-redirect" href="/wiki/%EC%BD%98%EC%84%B8%EB%A5%B4%ED%8A%B8%ED%97%A4%EB%B3%B4_%EA%B4%80%ED%98%84%EC%95%85%EB%8B%A8" title="콘세르트헤보 관현악단">콘세르트헤보 관현악단</a>의 수석 플루티스트로 활동한다. 그 후 <a href="/wiki/%EB%B2%A0%EB%A5%BC%EB%A6%B0_%ED%95%84%ED%95%98%EB%AA%A8%EB%8B%88_%EA%B4%80%ED%98%84%EC%95%85%EB%8B%A8" title="베를린 필하모니 관현악단">베를린 필하모니 관현악단</a>에서의 수석과 <a class="new" href="/w/index.php?title=%EC%9C%A0%EB%9F%BD_%EC%B2%B4%EC%9E%84%EB%B2%84_%EC%98%A4%EC%BC%80%EC%8A%A4%ED%8A%B8%EB%9D%BC&amp;action=edit&amp;redlink=1" title="유럽 체임버 오케스트라 (없는 문서)">유럽 체임버 오케스트라</a>의 수석 연주자로 <a href="/wiki/%ED%81%B4%EB%9D%BC%EC%9A%B0%EB%94%94%EC%98%A4_%EC%95%84%EB%B0%94%EB%8F%84" title="클라우디오 아바도">클라우디오 아바도</a>, <a class="new" href="/w/index.php?title=%EB%B2%A0%EB%A5%B4%EB%82%98%EB%A5%B4%ED%8A%B8_%ED%95%98%EC%9D%B4%ED%8C%85%ED%81%AC&amp;action=edit&amp;redlink=1" title="베르나르트 하이팅크 (없는 문서)">베르나르트 하이팅크</a>와 <a href="/wiki/%EB%8B%88%EC%BD%9C%EB%9D%BC%EC%9A%B0%EC%8A%A4_%EC%95%84%EB%A5%B4%EB%86%8D%EC%BF%A0%EB%A5%B4" title="니콜라우스 아르농쿠르">니콜라우스 아르농쿠르</a> 같은 지휘자들 밑에서 활동하였다. <a href="/wiki/1989%EB%85%84" title="1989년">1989년</a>에서 <a href="/wiki/1997%EB%85%84" title="1997년">1997년</a> 사이 <a href="/wiki/%EB%B3%B4%EC%8A%A4%ED%84%B4_%EA%B5%90%ED%96%A5%EC%95%85%EB%8B%A8" title="보스턴 교향악단">보스턴 교향악단</a>의 수석으로 <a class="mw-redirect" href="/wiki/%EC%84%B8%EC%9D%B4%EC%A7%80_%EC%98%A4%EC%9E%90%EC%99%80" title="세이지 오자와">세이지 오자와</a>와 함께하였다. 현재 <a href="/wiki/%ED%81%B4%EB%9D%BC%EC%9A%B0%EB%94%94%EC%98%A4_%EC%95%84%EB%B0%94%EB%8F%84" title="클라우디오 아바도">클라우디오 아바도</a>가 이끄는 루체른 페스티발 오케스트라와 모차르트 오케스트라의 수석주자로 활동하고 있으며 전 세계에서 <a href="/wiki/%EB%8F%85%EC%A3%BC" title="독주">독주</a>, <a href="/wiki/%EC%8B%A4%EB%82%B4%EC%95%85" title="실내악">실내악</a>, <a class="mw-redirect" href="/wiki/%EC%98%A4%EC%BC%80%EC%8A%A4%ED%8A%B8%EB%9D%BC" title="오케스트라">오케스트라</a> 연주자로서 활발한 연주를 펼치고있다. 음반으로는 <a class="mw-redirect" href="/wiki/%EB%8F%84%EC%9D%B4%EC%B9%98_%EA%B7%B8%EB%9D%BC%EB%AA%A8%ED%8F%B0" title="도이치 그라모폰">도이치 그라모폰</a>, <a href="/wiki/EMI" title="EMI">EMI</a>, <a href="/wiki/%ED%95%84%EB%A6%BD%EC%8A%A4" title="필립스">필립스</a>, <a class="new" href="/w/index.php?title=Decca&amp;action=edit&amp;redlink=1" title="Decca (없는 문서)">Decca</a>, <a class="new" href="/w/index.php?title=Chandos&amp;action=edit&amp;redlink=1" title="Chandos (없는 문서)">Chandos</a>, 보스턴 레코드 등의 음반사에서 그의 연주를 들을 수 있다. 최근 <a href="/wiki/%ED%81%B4%EB%9D%BC%EC%9A%B0%EB%94%94%EC%98%A4_%EC%95%84%EB%B0%94%EB%8F%84" title="클라우디오 아바도">클라우디오 아바도</a> 와 함께 <a class="mw-redirect" href="/wiki/%EB%8F%84%EC%9D%B4%EC%B9%98_%EA%B7%B8%EB%9D%BC%EB%AA%A8%ED%8F%B0" title="도이치 그라모폰">도이치 그라모폰</a> 음반사에서 <a class="mw-redirect" href="/wiki/%EB%AA%A8%EC%B0%A8%EB%A5%B4%ED%8A%B8" title="모차르트">모차르트</a> 플루트 협주곡을 녹음하였다.</p>
<h2><span class="mw-headline" id=".EA.B5.90.EC.9C.A1.EC.9E.90">교육자</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%95%BC%ED%81%AC_%EC%A1%B4&amp;action=edit&amp;section=3" title="부분 편집: 교육자">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p><a href="/wiki/1989%EB%85%84" title="1989년">1989년</a> 부터 <a href="/wiki/1994%EB%85%84" title="1994년">1994년</a>까지 <a href="/wiki/%EB%A1%9C%ED%85%8C%EB%A5%B4%EB%8B%B4" title="로테르담">로테르담</a> 음악원의 교수로 재직하였고, 그 후 <a href="/wiki/%EB%AF%B8%EA%B5%AD" title="미국">미국</a>의 인디아나 대학에서 <a href="/wiki/1994%EB%85%84" title="1994년">1994년</a>부터 <a href="/wiki/1997%EB%85%84" title="1997년">1997년</a>까지, 보스턴 대학과 뉴잉글랜드 음악원에서 <a href="/wiki/1997%EB%85%84" title="1997년">1997년</a>부터 <a href="/wiki/2001%EB%85%84" title="2001년">2001년</a>까지 교수로 재직하였다. <a href="/wiki/%EB%8F%85%EC%9D%BC" title="독일">독일</a>의 베를린 음악대학에서 가르친 후 <a href="/wiki/2002%EB%85%84" title="2002년">2002년</a> 부터 현재까지 <a href="/wiki/%EC%8A%A4%EC%9C%84%EC%8A%A4" title="스위스">스위스</a>의 <a href="/wiki/%EC%A0%9C%EB%84%A4%EB%B0%94_%EA%B5%AD%EB%A6%BD_%EA%B3%A0%EB%93%B1%EC%9D%8C%EC%95%85%EC%9B%90" title="제네바 국립 고등음악원">제네바 국립 고등음악원</a>의 교수로 재직 중이며 <a href="/wiki/%EC%8A%A4%ED%8E%98%EC%9D%B8" title="스페인">스페인</a>의 마드리드 퀸 소피아 음악학교에서 실내악과 교수로 재직중이다.</p>
<h2><span class="mw-headline" id=".EC.95.85.EA.B8.B0">악기</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%95%BC%ED%81%AC_%EC%A1%B4&amp;action=edit&amp;section=4" title="부분 편집: 악기">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>나무 플루트로 연주하는 것으로 잘 알려진 Jacques Zoon 은 그가 직접 헤드(취구)를 제작하여 연주한다 . 바로크 시대의 곡들은 직접 만든 고전 플루트로 연주하며 플루트 제작자인 William S. Haynes 는 고전 플루트의 취약점을 개선한 Zoon 모델 이라 불리는 플루트를 개발하였다.</p>
