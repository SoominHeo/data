<p><b>얀 블록하위선</b>(<span style="font-size: smaller;"><a href="/wiki/%EB%84%A4%EB%8D%9C%EB%9E%80%EB%93%9C%EC%96%B4" title="네덜란드어">네덜란드어</a>:</span> <span lang="nl" xml:lang="nl">Jan Blokhuijsen</span>, <a href="/wiki/1989%EB%85%84" title="1989년">1989년</a> <a href="/wiki/4%EC%9B%94_1%EC%9D%BC" title="4월 1일">4월 1일</a> ~ )은 <a href="/wiki/%EB%84%A4%EB%8D%9C%EB%9E%80%EB%93%9C" title="네덜란드">네덜란드</a>의 <a href="/wiki/%EC%8A%A4%ED%94%BC%EB%93%9C_%EC%8A%A4%EC%BC%80%EC%9D%B4%ED%8C%85" title="스피드 스케이팅">스피드 스케이팅</a> 선수이다. 2007년에 국가대표로 발탁되어 <a href="/wiki/%EC%95%8C%ED%81%AC%EB%A7%88%EB%A5%B4" title="알크마르">알크마르</a>에서 열린 <a class="new" href="/w/index.php?title=2007%EB%85%84_%EC%9C%A0%EB%9F%BD_%EC%8A%A4%ED%94%BC%EB%93%9C_%EC%8A%A4%EC%BC%80%EC%9D%B4%ED%8C%85_%EC%84%A0%EC%88%98%EA%B6%8C%EB%8C%80%ED%9A%8C&amp;action=edit&amp;redlink=1" title="2007년 유럽 스피드 스케이팅 선수권대회 (없는 문서)">유럽 주니어 선수권대회</a>에서 금메달 1개, 은메달 2개, 동메달 3개를 획득했고, <a href="/wiki/2010%EB%85%84_%EB%8F%99%EA%B3%84_%EC%98%AC%EB%A6%BC%ED%94%BD" title="2010년 동계 올림픽">2010 밴쿠버 동계 올림픽</a>에서 <a href="/wiki/2010%EB%85%84_%EB%8F%99%EA%B3%84_%EC%98%AC%EB%A6%BC%ED%94%BD_%EC%8A%A4%ED%94%BC%EB%93%9C%EC%8A%A4%EC%BC%80%EC%9D%B4%ED%8C%85_%EB%82%A8%EC%9E%90_%ED%8C%80_%EC%B6%94%EC%9B%94" title="2010년 동계 올림픽 스피드스케이팅 남자 팀 추월">팀 추월</a> 동메달을 획득했다. 이후 세계 선수권대회에서 다수의 메달을 획득했고, <a href="/wiki/2012%EB%85%84_%EC%84%B8%EA%B3%84_%EC%A2%85%EB%AA%A9%EB%B3%84_%EC%8A%A4%ED%94%BC%EB%93%9C_%EC%8A%A4%EC%BC%80%EC%9D%B4%ED%8C%85_%EC%84%A0%EC%88%98%EA%B6%8C_%EB%8C%80%ED%9A%8C" title="2012년 세계 종목별 스피드 스케이팅 선수권 대회">2012 헤이렌베인 세계 종목별 선수권대회</a>에서 팀 추월 부문 금메달을 획득하고 2014 유럽 선수권대회에서 종합 우승했다. 같은 해에 <a href="/wiki/%EC%86%8C%EC%B9%98" title="소치">소치</a> <a href="/wiki/2014%EB%85%84_%EB%8F%99%EA%B3%84_%EC%98%AC%EB%A6%BC%ED%94%BD" title="2014년 동계 올림픽">동계 올림픽</a>에서 5000m 종목 은메달을 획득하였다.</p>

class="new" href="/w/index.php?title=%EC%9E%90%EC%9C%84%ED%8A%B8%EC%8A%A4%ED%95%98%EB%A5%B4%EB%B0%94%EC%9A%B0%EB%8D%94&amp;action=edit&amp;redlink=1" title="자위트스하르바우더 (없는 문서)">자위트스하르바우더</a> 출신으로 어릴 때
