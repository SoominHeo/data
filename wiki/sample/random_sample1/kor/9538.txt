<p><b>얀 안나 휘마르 쾰레만스</b> (<span style="font-size: smaller;"><a href="/wiki/%EB%84%A4%EB%8D%9C%EB%9E%80%EB%93%9C%EC%96%B4" title="네덜란드어">네덜란드어</a>:</span> <span lang="nl" xml:lang="nl">Jan Anna Gumaar Ceulemans</span>, <a href="/wiki/1957%EB%85%84" title="1957년">1957년</a> <a href="/wiki/2%EC%9B%94_28%EC%9D%BC" title="2월 28일">2월 28일</a> ~ )는 <a href="/wiki/%EB%B2%A8%EA%B8%B0%EC%97%90" title="벨기에">벨기에</a>의 전직 <a href="/wiki/%EC%B6%95%EA%B5%AC" title="축구">축구</a> 선수로, 포지션은 <a href="/wiki/%EB%AF%B8%EB%93%9C%ED%95%84%EB%8D%94" title="미드필더">공격형 미드필더</a>였다.</p>
<p><a href="/wiki/1974%EB%85%84" title="1974년">1974년</a> <a href="/wiki/%EB%A6%AC%EB%A5%B4%EC%84%9C_SK" title="리르서 SK">리르서 SK</a> 소속으로 데뷔했고 <a href="/wiki/1978%EB%85%84" title="1978년">1978년</a>부터 <a href="/wiki/1991%EB%85%84" title="1991년">1991년</a>까지 <a class="mw-redirect" href="/wiki/%ED%81%B4%EB%9F%BD_%EB%B8%8C%EB%A3%A8%EA%B2%8C_KV" title="클럽 브루게 KV">클럽 브루게</a> 소속으로 뛰면서 팀의 <a class="mw-redirect" href="/wiki/%EB%B2%A8%EA%B8%B0%EC%97%90_%EB%A6%AC%EA%B7%B8" title="벨기에 리그">벨기에 리그</a> 4회 우승(1980년, 1988년, 1990년, 1992년)과 벨기에 컵 2회 우승(1986년, 1991년), 벨기에 슈퍼컵 5회 우승(1980년, 1986년, 1988년, 1990년, 1991년)에 공헌했다.</p>
<p><a href="/wiki/1977%EB%85%84" title="1977년">1977년</a>부터 <a href="/wiki/1991%EB%85%84" title="1991년">1991년</a>까지 <a href="/wiki/%EB%B2%A8%EA%B8%B0%EC%97%90_%EC%B6%95%EA%B5%AC_%EA%B5%AD%EA%B0%80%EB%8C%80%ED%91%9C%ED%8C%80" title="벨기에 축구 국가대표팀">벨기에 대표팀</a>에서 뛰는 동안 96경기에 출전하여 23득점을 기록했으며 그가 세운 대표팀 출전 기록은 벨기에 대표팀 최다 출전 기록으로 남아 있다. <a href="/wiki/UEFA_%EC%9C%A0%EB%A1%9C_1980" title="UEFA 유로 1980">UEFA 유로 1980</a>과 <a href="/wiki/1982%EB%85%84_FIFA_%EC%9B%94%EB%93%9C%EC%BB%B5" title="1982년 FIFA 월드컵">1982년 FIFA 월드컵</a>, <a href="/wiki/UEFA_%EC%9C%A0%EB%A1%9C_1984" title="UEFA 유로 1984">UEFA 유로 1984</a>, <a href="/wiki/1986%EB%85%84_FIFA_%EC%9B%94%EB%93%9C%EC%BB%B5" title="1986년 FIFA 월드컵">1986년 FIFA 월드컵</a>, <a href="/wiki/1990%EB%85%84_FIFA_%EC%9B%94%EB%93%9C%EC%BB%B5" title="1990년 FIFA 월드컵">1990년 FIFA 월드컵</a>에서 벨기에 대표로 출전했으며 특히 UEFA 유로 1980에서는 벨기에가 준우승을 차지하는 데에 공헌했고 1986년 FIFA 월드컵에서는 벨기에가 4위를 차지하는 데에 공헌했다. <a href="/wiki/2004%EB%85%84" title="2004년">2004년</a> 3월 <a href="/wiki/%ED%8E%A0%EB%A0%88" title="펠레">펠레</a>가 선정한 현존하는 최고의 축구 선수 목록인 <a href="/wiki/FIFA_100" title="FIFA 100">FIFA 100</a>에 선정되었다.</p>
