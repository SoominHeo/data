<p><b>양각도국제호텔</b>(羊角島國際飯店, <span style="font-size: smaller;"><a href="/wiki/%EB%AC%B8%ED%99%94%EC%96%B4" title="문화어">문화어</a>:</span> 량각도국제려관, <span style="font-size: smaller;"><a href="/wiki/%EC%98%81%EC%96%B4" title="영어">영어</a>:</span> <span lang="en" xml:lang="en">Yanggakdo International Hotel</span>)은 <a href="/wiki/%EC%A1%B0%EC%84%A0%EB%AF%BC%EC%A3%BC%EC%A3%BC%EC%9D%98%EC%9D%B8%EB%AF%BC%EA%B3%B5%ED%99%94%EA%B5%AD" title="조선민주주의인민공화국">조선민주주의인민공화국</a>에서 가장 큰 호텔이자 호화스러운 호텔이다. <a href="/wiki/%ED%8F%89%EC%96%91%EC%A7%81%ED%95%A0%EC%8B%9C" title="평양직할시">평양직할시</a> <a href="/wiki/%EC%A4%91%EA%B5%AC%EC%97%AD" title="중구역">중구역</a> 동흥동 <a href="/wiki/%EC%96%91%EA%B0%81%EB%8F%84" title="양각도">양각도</a>에 위치하고 <a href="/wiki/%ED%8F%89%EC%96%91%EC%A7%81%ED%95%A0%EC%8B%9C" title="평양직할시">평양직할시</a>의 중심지에서 남동쪽으로 4km에 위치해 있다.</p>

<h2><span class="mw-headline" id=".EC.97.B0.ED.98.81">연혁</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%96%91%EA%B0%81%EB%8F%84%EA%B5%AD%EC%A0%9C%ED%98%B8%ED%85%94&amp;action=edit&amp;section=1" title="부분 편집: 연혁">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p><a href="/wiki/1985%EB%85%84" title="1985년">1985년</a> <a href="/wiki/3%EC%9B%94_23%EC%9D%BC" title="3월 23일">3월 23일</a>에 <a href="/wiki/%ED%94%84%EB%9E%91%EC%8A%A4" title="프랑스">프랑스</a> 회사와 공동으로 지었고, <a href="/wiki/1990%EB%85%84" title="1990년">1990년</a> <a href="/wiki/5%EC%9B%94_20%EC%9D%BC" title="5월 20일">5월 20일</a> 완공했으나, <a href="/wiki/1995%EB%85%84" title="1995년">1995년</a>에 개관했다.</p>
<h2><span class="mw-headline" id=".EB.B0.B0.EA.B2.BD">배경</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%96%91%EA%B0%81%EB%8F%84%EA%B5%AD%EC%A0%9C%ED%98%B8%ED%85%94&amp;action=edit&amp;section=2" title="부분 편집: 배경">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>수신을 수용 외에도 층은 <a href="/wiki/%EC%A1%B0%EC%84%A0%EB%AF%BC%EC%A3%BC%EC%A3%BC%EC%9D%98%EC%9D%B8%EB%AF%BC%EA%B3%B5%ED%99%94%EA%B5%AD" title="조선민주주의인민공화국">조선민주주의인민공화국</a>의 통화 세트, 엽서 및 편지, 서양 가격에 기본 상품의 구매를 제공한다. 이 바이며, 서점 현지 영어 신문, <a href="/wiki/%ED%8F%89%EC%96%91%EC%8B%A0%EB%AC%B8" title="평양신문">평양신문</a>, 논문과 <a href="/wiki/%EA%B9%80%EC%9D%BC%EC%84%B1" title="김일성">김일성</a>, <a href="/wiki/%EA%B9%80%EC%A0%95%EC%9D%BC" title="김정일">김정일</a>, <a href="/wiki/%EA%B9%80%EC%A0%95%EC%9D%80" title="김정은">김정은</a>의 사진 백업 문제를 포함한 <a href="/wiki/%EC%A1%B0%EC%84%A0%EB%AF%BC%EC%A3%BC%EC%A3%BC%EC%9D%98%EC%9D%B8%EB%AF%BC%EA%B3%B5%ED%99%94%EA%B5%AD" title="조선민주주의인민공화국">조선민주주의인민공화국</a>의 독서 자료를 보유하고 있는 지역의 역사의 <a href="/wiki/%EC%A1%B0%EC%84%A0%EB%AF%BC%EC%A3%BC%EC%A3%BC%EC%9D%98%EC%9D%B8%EB%AF%BC%EA%B3%B5%ED%99%94%EA%B5%AD" title="조선민주주의인민공화국">조선민주주의인민공화국</a> 이해를 설명되지 않은, 아동 도서 및 자료이다.</p>
<p>호텔의 부지는 원래 9,000 평방 미터 아홉 홀 골프 코스를 포함했다. <a href="/wiki/2011%EB%85%84" title="2011년">2011년</a> 골프 코스는 중국 투자의 건강 단지가 건설 될 수 있도록 공간을 확보하기 위해 폭파되었다. 또한 <a class="new" href="/w/index.php?title=%EC%96%91%EA%B0%81%EC%84%AC&amp;action=edit&amp;redlink=1" title="양각섬 (없는 문서)">양각섬</a>에 위치한 호텔의 부지 옆에, 평양국제영화회관이다. 평양국제영화제의 개폐 의식은 여기에서 이루어진다.</p>
<p>양각도국제호텔은 <a href="/wiki/%EC%A1%B0%EC%84%A0%EB%AF%BC%EC%A3%BC%EC%A3%BC%EC%9D%98%EC%9D%B8%EB%AF%BC%EA%B3%B5%ED%99%94%EA%B5%AD" title="조선민주주의인민공화국">조선민주주의인민공화국</a>의 투어에 표준 정지, 그리고 그래픽 소설 평양에 등장했다.</p>
<h2><span class="mw-headline" id=".EC.8B.9C.EC.84.A4">시설</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%96%91%EA%B0%81%EB%8F%84%EA%B5%AD%EC%A0%9C%ED%98%B8%ED%85%94&amp;action=edit&amp;section=3" title="부분 편집: 시설">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>지상 170m, 47층짜리 초고층 건물인데 최상층에 대해서는 48층설이 존재하지만, 이는 지상 47층, 지하 1층 건물이라 층수를 합쳐서 부른 것이라는 견해가 있다.</p>
<p>식사-객실 하나, 둘, 메인 연회장과 일본, 중국과 한국음식 식당 - 객실 : 회전하는 레스토랑뿐만 아니라, 손님에게 발행한 호텔 가이드는 호텔 2층에 네 더 레스토랑이 포함되어 있음을 나타낸다.</p>
<p>5층 오프 호텔 투숙객 한계고, 또한 엘리베이터 패널의 버튼이 없다. 지하는 볼링장을 포함, 풀룸, 수영장, 이발소, 카지노 및 마사지 클럽 독점적으로 여성 직원들과 중국 기업에 의해 실행된다.</p>
<p>객실 1001개. 침대 1963개. 꼭대기는 층 전체가 회전하는 식당으로 평양 시내를 조망할 수 있다. 지하에는 카지노도 있다.</p>
