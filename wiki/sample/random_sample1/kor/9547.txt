<h2><span class="mw-headline" id=".EC.84.B1.EC.A7.88">성질</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%96%91%EA%B3%B5_(%EB%AC%BC%EB%A6%AC%ED%95%99)&amp;action=edit&amp;section=1" title="부분 편집: 성질">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p><a href="/wiki/%EB%B0%98%EB%8F%84%EC%B2%B4" title="반도체">반도체</a> 결정에서는 "주위의 <a href="/wiki/%EC%9B%90%EC%9E%90%EA%B0%80%EB%9D%A0" title="원자가띠">원자가띠</a>가 순서대로 양공이 떨어져서 새로운 전공이 생긴다."라는 과정을 차례대로 반복하여 결정내부에서 돌아다닐수 있어서 마치 "양의 전하를 가진 전자"와 동일한 행동을 해서 <a href="/wiki/%EC%A0%84%EA%B8%B0%EC%A0%84%EB%8F%84%EB%8F%84" title="전기전도도">전기전도도</a>에 기여한다. 그리고 주위의 원자가띠가 아니고 <a class="new" href="/w/index.php?title=%EC%A0%84%EB%8F%84%EC%A0%84%EC%9E%90&amp;action=edit&amp;redlink=1" title="전도전자 (없는 문서)">전도전자</a> (<a class="mw-redirect" href="/wiki/%EC%9E%90%EC%9C%A0%EC%A0%84%EC%9E%90" title="자유전자">자유전자</a>)와 원자가띠 사이의 에너지 준위 차이로 인하여 큰 에너지를 열이나 빛으로 방출해 전류의 <a class="mw-disambig" href="/wiki/%EC%BA%90%EB%A6%AC%EC%96%B4" title="캐리어">캐리어</a>의 존재는 소멸한다. 이것을 <b>캐리어의 재결합</b>이라고 부른다.</p>
<p>양공은 <a class="mw-redirect" href="/wiki/%EC%9E%90%EC%9C%A0%EC%A0%84%EC%9E%90" title="자유전자">자유전자</a>와 비슷하게 전하를 옮기는 캐리어로 행동할 수 있다. 양공에 의한 <a class="new" href="/w/index.php?title=%EC%A0%84%EA%B8%B0_%EC%A0%84%EB%8F%84%EC%84%B1&amp;action=edit&amp;redlink=1" title="전기 전도성 (없는 문서)">전기 전도성</a>을 <b>P형</b>이라고 한다. 반도체에 <a class="new" href="/w/index.php?title=%EC%96%B5%EC%85%89%ED%84%B0&amp;action=edit&amp;redlink=1" title="억셉터 (없는 문서)">억셉터</a>를 <a href="/wiki/%EB%8F%84%ED%95%91" title="도핑">도핑</a>하면 <a href="/wiki/%EC%9B%90%EC%9E%90%EA%B0%80%EB%9D%A0" title="원자가띠">원자가띠</a>가 <a class="mw-redirect" href="/wiki/%EC%97%B4_%EC%97%90%EB%84%88%EC%A7%80" title="열 에너지">열 에너지</a>에 의해서 <a class="new" href="/w/index.php?title=%EC%96%B5%EC%85%89%ED%84%B0_%EC%A4%80%EC%9C%84&amp;action=edit&amp;redlink=1" title="억셉터 준위 (없는 문서)">억셉터 준위</a>에 <a class="mw-disambig" href="/wiki/%EC%B2%9C%EC%9D%B4" title="천이">천이</a>되어서 양공의 농도가 높은 반도체를 <a href="/wiki/P%ED%98%95_%EB%B0%98%EB%8F%84%EC%B2%B4" title="P형 반도체">P형 반도체</a>라고 부른다.</p>
<p>일반적으로 양공의 <a class="new" href="/w/index.php?title=%EB%93%9C%EB%A6%AC%ED%94%84%ED%8A%B8_%EC%9D%B4%EB%8F%99%EB%8F%84&amp;action=edit&amp;redlink=1" title="드리프트 이동도 (없는 문서)">드리프트 이동도</a> (단순히 <a class="new" href="/w/index.php?title=%EC%9D%B4%EB%8F%99%EB%8F%84&amp;action=edit&amp;redlink=1" title="이동도 (없는 문서)">이동도</a>)는 자유 전자보다 작고 <a href="/wiki/%EA%B7%9C%EC%86%8C" title="규소">실리콘</a>결정중에는 전자의 1/3정도 된다. 그리고 이로써 결정되는 <a class="new" href="/w/index.php?title=%EB%93%9C%EB%A6%AC%ED%94%84%ED%8A%B8_%EC%86%8D%EB%8F%84&amp;action=edit&amp;redlink=1" title="드리프트 속도 (없는 문서)">드리프트 속도</a>는 각각의 전자나 양공이 가지는 속도가 아니고 평균적인 속도인것에 주의할 필요가 있다.</p>
<p><a href="/wiki/%EC%9B%90%EC%9E%90%EA%B0%80%EB%9D%A0" title="원자가띠">원자가띠</a>의 맨 위에는 <a href="/wiki/%EC%97%90%EB%84%88%EC%A7%80" title="에너지">E</a>-<a href="/wiki/%ED%8C%8C%EC%88%98" title="파수">k</a> 공간상에서 형상이 다른 복수의 밴드가 <a class="mw-redirect" href="/wiki/%EC%B6%95%ED%87%B4" title="축퇴">축퇴</a>되고, 여기에 대응해서 양공의 밴드도 <a class="new" href="/w/index.php?title=%EC%9C%A0%ED%9A%A8%EC%A7%88%EB%9F%89&amp;action=edit&amp;redlink=1" title="유효질량 (없는 문서)">유효질량</a>이 다른 <b>무거운 양공</b> (<span lang="en" xml:lang="en">heavy hole</span>)과 <b>가벼운 양공</b>(<span lang="en" xml:lang="en">light hole</span>)의 밴드로 나뉜다. 또한 <a href="/wiki/%EC%8B%A4%EB%A6%AC%EC%BD%98" title="실리콘">실리콘</a>의 스핀-궤도 상호작용 (<span lang="en" xml:lang="en">spin–orbit coupling</span>)이 적은 원소에서 스핀 궤도 스프릿트 오프밴드 (스핀 분열밴드)도 에너지적으로 가까워서 (Δ=44 meV) 독립적으로 논의하는 것은 그만큼 어려워진다. <a class="new" href="/w/index.php?title=%EC%9D%B4%EB%8F%99%EB%8F%84&amp;action=edit&amp;redlink=1" title="이동도 (없는 문서)">이동도</a>가 매우 중요한 <a href="/wiki/%EB%B0%98%EB%8F%84%EC%B2%B4_%EC%86%8C%EC%9E%90" title="반도체 소자">반도체 소자</a>에서는 결정의 일그러짐을 도입하여 원자가띠 정상의 축퇴를 품는 동시에 양자 준위를 바꿔서 가벼운 전공을 주로 이용해 <a class="mw-redirect" href="/wiki/%ED%8F%AC%EB%85%BC" title="포논">포논</a> (<span lang="en" xml:lang="en">phonon</span>) 산란이나 캐리어의 실제 유효질량의 감소를 도모하는 경우가 있다.</p>
<h2><span class="mw-headline" id=".EC.96.B4.EC.9B.90">어원</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%96%91%EA%B3%B5_(%EB%AC%BC%EB%A6%AC%ED%95%99)&amp;action=edit&amp;section=2" title="부분 편집: 어원">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>양공을 의미하는 영명 "<span lang="en" xml:lang="en">hole</span> <small>홀<sup>[<a href="/wiki/%EC%9C%84%ED%82%A4%EB%B0%B1%EA%B3%BC:%EC%98%81%EC%96%B4%EC%9D%98_%ED%95%9C%EA%B8%80_%ED%91%9C%EA%B8%B0" title="위키백과:영어의 한글 표기">*</a>]</sup></small>"은 구멍을 뜻하고, 마치 전자가 채워져 있어야 할 곳에 전자가 없는 것을 "구멍"으로 비유한 것이다. <a href="/wiki/%ED%99%80_%ED%9A%A8%EA%B3%BC" title="홀 효과">홀 효과</a>의 “홀” (<span lang="en" xml:lang="en">Hall</span>)은 인명으로, 양공과는 관계없다.</p>
