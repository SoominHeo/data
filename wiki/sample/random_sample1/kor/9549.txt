<p><b>양동원</b>(梁棟原, <a href="/wiki/1987%EB%85%84" title="1987년">1987년</a> <a href="/wiki/2%EC%9B%94_5%EC%9D%BC" title="2월 5일">2월 5일</a> ~ )은 <a href="/wiki/%EB%8C%80%ED%95%9C%EB%AF%BC%EA%B5%AD" title="대한민국">대한민국</a>의 <a href="/wiki/%EC%B6%95%EA%B5%AC" title="축구">축구</a> 선수이다. 현재 <a href="/wiki/K%EB%A6%AC%EA%B7%B8_%EC%B1%8C%EB%A6%B0%EC%A7%80" title="K리그 챌린지">K리그 챌린지</a> <a href="/wiki/%EC%84%B1%EB%82%A8_FC" title="성남 FC">성남 FC</a> 소속이며, 포지션은 <a href="/wiki/%EA%B3%A8%ED%82%A4%ED%8D%BC" title="골키퍼">골키퍼</a>이다.</p>

<h2><span class="mw-headline" id=".EC.B6.95.EA.B5.AC.EC.9D.B8_.EA.B2.BD.EB.A0.A5">축구인 경력</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%96%91%EB%8F%99%EC%9B%90&amp;action=edit&amp;section=1" title="부분 편집: 축구인 경력">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<h3><span class="mw-headline" id=".ED.81.B4.EB.9F.BD_.EA.B2.BD.EB.A0.A5">클럽 경력</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%96%91%EB%8F%99%EC%9B%90&amp;action=edit&amp;section=2" title="부분 편집: 클럽 경력">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>2005년 <a href="/wiki/%EB%8C%80%EC%A0%84_%EC%8B%9C%ED%8B%B0%EC%A6%8C" title="대전 시티즌">대전 시티즌</a>에 입단하였지만 <a href="/wiki/%EC%B5%9C%EC%9D%80%EC%84%B1" title="최은성">최은성</a>에 밀려 2011년 <a href="/wiki/%EC%88%98%EC%9B%90_%EC%82%BC%EC%84%B1_%EB%B8%94%EB%A3%A8%EC%9C%99%EC%A6%88" title="수원 삼성 블루윙즈">수원 삼성</a>으로 이적하기 전까지 리그 13경기 출전에 그쳤다. 수원에서의 첫 시즌인 2011 시즌 3경기에 출전에 그쳤지만, 2012년엔 <a href="/wiki/2012%EB%85%84_%ED%95%98%EA%B3%84_%EC%98%AC%EB%A6%BC%ED%94%BD" title="2012년 하계 올림픽">런던 올림픽</a>에 차출된 <a href="/wiki/%EC%A0%95%EC%84%B1%EB%A3%A1" title="정성룡">정성룡</a>을 대신하여 리그 11경기에 출전하며 프로 데뷔 후 시즌 가장 많은 리그 경기에 출전하였다.</p>
<p>2014년 주전으로 활약하기 위해 수원의 재계약 제안을 거부하고 <a href="/wiki/%EA%B0%95%EC%9B%90_FC" title="강원 FC">강원 FC</a>로 이적하였고,<sup class="reference" id="cite_ref-1"><a href="#cite_note-1">[1]</a></sup> 2014 시즌 리그 16경기에 출전하였다. 시즌 종료 후 군 복무를 위해 <a href="/wiki/%EC%83%81%EC%A3%BC_%EC%83%81%EB%AC%B4_%EC%B6%95%EA%B5%AC%EB%8B%A8" title="상주 상무 축구단">상주 상무</a>에 입단하였다.</p>
