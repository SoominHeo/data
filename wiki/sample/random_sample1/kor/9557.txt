<p><b>양상쿤</b>(<small><a href="/wiki/%EC%A4%91%EA%B5%AD%EC%96%B4" title="중국어">중국어</a></small> <small><a href="/wiki/%EA%B0%84%EC%B2%B4%EC%9E%90" title="간체자">간체</a>:</small> <span lang="zh-hans" href="/wiki/%ED%95%9C%EC%96%B4_%EB%B3%91%EC%9D%8C" title="한어 병음">병음</a>:</small> <font face="Lucida Sans Unicode">Yáng Shàngkūn</font>, <small><a href="/wiki/%ED%95%9C%EA%B5%AD_%ED%95%9C%EC%9E%90%EC%9D%8C" title="한국 한자음">한자음</a>:</small> 양상곤, <a href="/wiki/1907%EB%85%84" title="1907년">1907년</a> <a href="/wiki/5%EC%9B%94_25%EC%9D%BC" title="5월 25일">5월 25일</a> ~ <a href="/wiki/1998%EB%85%84" title="1998년">1998년</a> <a href="/wiki/9%EC%9B%94_14%EC%9D%BC" title="9월 14일">9월 14일</a>)은 <a href="/wiki/%EC%A4%91%ED%99%94%EC%9D%B8%EB%AF%BC%EA%B3%B5%ED%99%94%EA%B5%AD" title="중화인민공화국">중국</a>의 <a href="/wiki/%EC%A0%95%EC%B9%98%EA%B0%80" title="정치가">정치가</a>이자 <a href="/wiki/%EA%B5%B0%EC%9D%B8" title="군인">군인</a>(예비역 중화인민공화국 지상군 <a href="/wiki/1%EC%84%B1_%EC%9E%A5%EA%B5%B0" title="1성 장군">준장</a>)이다. <a class="mw-redirect" href="/wiki/%EC%B6%A9%EC%B9%AD" title="충칭">충칭</a> <a href="/wiki/%EC%A7%81%ED%95%A0%EC%8B%9C" title="직할시">직할시</a> <a class="mw-redirect" href="/wiki/%ED%89%81%EB%82%9C_%ED%98%84" title="퉁난 현">퉁난 현</a>에서 태어났다. <a class="mw-redirect" href="/wiki/%EC%A4%91%ED%99%94%EC%9D%B8%EB%AF%BC%EA%B3%B5%ED%99%94%EA%B5%AD_%EC%A3%BC%EC%84%9D" title="중화인민공화국 주석">국가 주석</a>과 <a href="/wiki/%EC%A4%91%EA%B5%AD%EA%B3%B5%EC%82%B0%EB%8B%B9_%EC%A4%91%EC%95%99%EC%A0%95%EC%B9%98%EA%B5%AD" title="중국공산당 중앙정치국">당중앙 정치국</a> 위원, <a class="new" href="/w/index.php?title=%EC%A4%91%ED%99%94%EC%9D%B8%EB%AF%BC%EA%B3%B5%ED%99%94%EA%B5%AD%EC%9D%98_%EC%A4%91%EC%95%99%EA%B5%B0%EC%82%AC%EC%9C%84%EC%9B%90%ED%9A%8C&amp;action=edit&amp;redlink=1" title="중화인민공화국의 중앙군사위원회 (없는 문서)">중앙군사위원회</a> 부주석을 역임하였다. <a href="/wiki/%EC%9E%A5%EC%A0%95" title="장정">장정</a>에 참여했다.</p>
<p>동생 <a class="new" href="/w/index.php?title=%EC%96%91%EB%B0%94%EC%9D%B4%EB%B9%99&amp;action=edit&amp;redlink=1" title="양바이빙 (없는 문서)">양바이빙</a>(楊白冰)과 함께 <a class="new" href="/w/index.php?title=%EC%96%91%EC%9E%90%EC%9E%A5_(%EC%9D%B8%EB%AC%BC)&amp;action=edit&amp;redlink=1" title="양자장 (인물) (없는 문서)">양자장</a>(楊家將:양씨 집안의 장군)으로 불리며, <a class="mw-redirect" href="/wiki/%EC%9D%B8%EB%AF%BC%ED%95%B4%EB%B0%A9%EA%B5%B0" title="인민해방군">인민해방군</a> 내에서 막강한 영향력을 행사했다.</p>

<h2><span class="mw-headline" id=".EC.83.9D.EC.95.A0.EC.99.80_.EA.B2.BD.EB.A0.A5">생애와 경력</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%96%91%EC%83%81%EC%BF%A4&amp;action=edit&amp;section=1" title="부분 편집: 생애와 경력">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<h3><span class="mw-headline" id=".EC.83.9D.EC.95.A0_.EC.B4.88.EB.B0.98">생애 초반</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%96%91%EC%83%81%EC%BF%A4&amp;action=edit&amp;section=2" title="부분 편집: 생애 초반">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<div class="thumb tleft">
<div class="thumbinner" style="width:152px;"><a class="image" href="/wiki/%ED%8C%8C%EC%9D%BC:Yang_Shangkun.jpg"><img alt="" class="thumbimage" data-file-height="533" data-file-width="400" height="200" src="//upload.wikimedia.org/wikipedia/commons/thumb/6/61/Yang_Shangkun.jpg/150px-Yang_Shangkun.jpg" srcset="//upload.wikimedia.org/wikipedia/commons/thumb/6/61/Yang_Shangkun.jpg/225px-Yang_Shangkun.jpg 1.5x, //upload.wikimedia.org/wikipedia/commons/thumb/6/61/Yang_Shangkun.jpg/300px-Yang_Shangkun.jpg 2x" width="150"/></a>
<div class="thumbcaption">
<div class="magnify"><a class="internal" href="/wiki/%ED%8C%8C%EC%9D%BC:Yang_Shangkun.jpg" title="실제 크기로"></a></div>
청년 시절</div>
</div>
</div>
<p><a href="/wiki/%EC%93%B0%EC%B4%A8_%EC%84%B1" title="쓰촨 성">쓰촨 성</a> <a class="mw-redirect" href="/wiki/%EC%B6%A9%EC%B9%AD" title="충칭">충칭</a> 출신으로, <a href="/wiki/1926%EB%85%84" title="1926년">1926년</a>에 <a href="/wiki/%EC%A4%91%EA%B5%AD%EA%B3%B5%EC%82%B0%EB%8B%B9" title="중국공산당">중국공산당</a>에 입당하였고, 당에 의해 소련에 파견되어 <a class="mw-redirect" href="/wiki/%EB%AA%A8%EC%8A%A4%ED%81%AC%EB%B0%94_%EC%A4%91%EC%82%B0%EB%8C%80%ED%95%99" title="모스크바 중산대학">모스크바 중산대학</a>에 유학하였다. 1931년 중국에 돌아와서 1933년 <a href="/wiki/%EB%A7%88%EC%98%A4%EC%A9%8C%EB%91%A5" title="마오쩌둥">마오쩌둥</a>과 <a href="/wiki/%EC%A3%BC%EB%8D%94" title="주더">주더</a>의 <a class="mw-redirect" href="/wiki/%EC%9E%A5%EC%8B%9C_%EC%86%8C%EB%B9%84%EC%97%90%ED%8A%B8" title="장시 소비에트">장시 소비에트</a>에 합류하였다. 친소련 성향의 지도부 "<a href="/wiki/28%EC%9D%B8%EC%9D%98_%EB%B3%BC%EC%85%B0%EB%B9%84%ED%82%A4" title="28인의 볼셰비키">28인의 볼셰비키</a>"의 일원으로 당 중앙위원 후보로 선출되었다. 이들은 기존 지휘부였던 마오쩌둥과 주더 대신 <a href="/wiki/%ED%99%8D%EA%B5%B0" title="홍군">홍군</a>을 지휘하였으나 기존의 게릴라전략을 버리고 정규전을 펼치다 1935년 <a href="/wiki/%EA%B5%AD%EB%AF%BC%ED%98%81%EB%AA%85%EA%B5%B0" title="국민혁명군">국민당군</a>의 제5차 토벌전에서 궤멸적 타격을 입었다. 이 패배때문에 중국공산당은 <a href="/wiki/%EC%9E%A5%EC%A0%95" title="장정">장정</a>을 결의했고, 양은 장정 중에 열린 <a href="/wiki/%EC%AD%8C%EC%9D%B4_%ED%9A%8C%EC%9D%98" title="쭌이 회의">쭌이 회의</a>에서는 "28인의 볼셰비키"를 벗어나 마오를 지지하여 마오가 다시 당권을 장악하는 데 기여했다.</p>
<p><a class="mw-redirect" href="/wiki/%EC%A0%9C2%EC%B0%A8_%EA%B5%AD%EA%B3%B5%ED%95%A9%EC%9E%91" title="제2차 국공합작">제2차 국공합작</a> 이후 항일 전쟁기에는 당중앙 북방국 부서기와 서기를 거쳤고, <a href="/wiki/%ED%99%94%EB%B2%A0%EC%9D%B4" title="화베이">화베이</a>에서 게릴라전을 지도했다. <a class="mw-redirect" href="/wiki/%EA%B5%AD%EA%B3%B5%EB%82%B4%EC%A0%84" title="국공내전">해방 전쟁 시기</a>에 중앙군사위원회 비서장으로서 군의 고급 간부에게 명령을 전달하는 역할을 맡았다.</p>
<h3><span class="mw-headline" id=".EC.A4.91.ED.99.94.EC.9D.B8.EB.AF.BC.EA.B3.B5.ED.99.94.EA.B5.AD_.EC.88.98.EB.A6.BD_.EC.9D.B4.ED.9B.84">중화인민공화국 수립 이후</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%96%91%EC%83%81%EC%BF%A4&amp;action=edit&amp;section=3" title="부분 편집: 중화인민공화국 수립 이후">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>건국 이후, <a href="/wiki/1949%EB%85%84" title="1949년">1949년</a>부터는 당의 일상 업무를 처리하는 <a href="/wiki/%EC%A4%91%EA%B5%AD%EA%B3%B5%EC%82%B0%EB%8B%B9_%EC%A4%91%EC%95%99%EC%84%9C%EA%B8%B0%EC%B2%98" title="중국공산당 중앙서기처">중국공산당 중앙서기처</a> 주임을 지냈고, <a href="/wiki/1956%EB%85%84" title="1956년">1956년</a>에 <a href="/wiki/%EB%8D%A9%EC%83%A4%EC%98%A4%ED%95%91" title="덩샤오핑">덩샤오핑</a>이 <a href="/wiki/%EC%A4%91%EA%B5%AD%EA%B3%B5%EC%82%B0%EB%8B%B9_%EC%A4%91%EC%95%99%EC%9C%84%EC%9B%90%ED%9A%8C_%EC%B4%9D%EC%84%9C%EA%B8%B0" title="중국공산당 중앙위원회 총서기">당 총서기</a>에 취임하면서 업무적으로 밀접한 관계를 가지게 됨으로써 그의 신뢰를 얻게 된 것으로 보인다.</p>
<p><a href="/wiki/1965%EB%85%84" title="1965년">1965년</a>경에 중앙서기처 주임에서 해임되어 <a href="/wiki/%EA%B4%91%EB%91%A5_%EC%84%B1" title="광둥 성">광둥 성</a> 당위원회 서기로 격하되었고 <a class="mw-redirect" href="/wiki/%EB%AC%B8%ED%99%94%EB%8C%80%ED%98%81%EB%AA%85" title="문화대혁명">문화대혁명</a>이 시작되면서, <a href="/wiki/%ED%8E%91%EC%A0%84" title="펑전">펑전</a>(彭眞), <a class="new" href="/w/index.php?title=%EB%A3%A8%EB%94%A9%EC%9D%B4&amp;action=edit&amp;redlink=1" title="루딩이 (없는 문서)">루딩이</a>(陸定一), <a class="new" href="/w/index.php?title=%EB%A4%84%EB%A3%A8%EC%9D%B4%EC%B9%AD&amp;action=edit&amp;redlink=1" title="뤄루이칭 (없는 문서)">뤄루이칭</a>(羅瑞卿) 등과 함께 "양상쿤 반혁명집단"으로 몰리게 되었다. 또 <a href="/wiki/%EB%A7%88%EC%98%A4%EC%A9%8C%EB%91%A5" title="마오쩌둥">마오쩌둥</a>으로부터 "양상쿤은 <a href="/wiki/%EC%9C%84%EC%97%B0" title="위연">위연</a>이다"라는 비판을 받고 실각하였다.</p>
<h3><span class="mw-headline" id=".EA.B5.AD.EA.B0.80.EC.A3.BC.EC.84.9D.EC.A7.81_.EC.9E.AC.EC.9E.84.EA.B3.BC_.EC.B5.9C.ED.9B.84">국가주석직 재임과 최후</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%96%91%EC%83%81%EC%BF%A4&amp;action=edit&amp;section=4" title="부분 편집: 국가주석직 재임과 최후">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p><a href="/wiki/1978%EB%85%84" title="1978년">1978년</a>에 복권되어, <a href="/wiki/1982%EB%85%84" title="1982년">1982년</a>에는 당 관료에 재임명되었다. 덩샤오핑과 함께 <a href="/wiki/1989%EB%85%84" title="1989년">1989년</a>의 <a class="mw-redirect" href="/wiki/%EC%B2%9C%EC%95%88%EB%AC%B8_%EA%B4%91%EC%9E%A5" title="천안문 광장">천안문 광장</a>의 민주화 시위를 군에 의한 진압에 대해 책임이 있는 것으로 알려져 있다. <sup class="정리_필요 noprint">[<a href="/wiki/%EC%9C%84%ED%82%A4%EB%B0%B1%EA%B3%BC:%EC%B6%9C%EC%B2%98_%EB%B0%9D%ED%9E%88%EA%B8%B0" title="위키백과:출처 밝히기">출처 필요</a>]</sup> 그는 형식적으로 국가주석직에 있었지만 실권을 쥐고 있던 <a href="/wiki/%EB%8D%A9%EC%83%A4%EC%98%A4%ED%95%91" title="덩샤오핑">덩샤오핑</a>과 갈등관계에 있었다. 더구나 군부 내에 그의 지지자가 많은 것 역시 <a href="/wiki/%EB%8D%A9%EC%83%A4%EC%98%A4%ED%95%91" title="덩샤오핑">덩샤오핑</a>에게 걸림돌이 되었다.</p>
<p><a class="mw-redirect" href="/wiki/%EC%9D%B8%EB%AF%BC%ED%95%B4%EB%B0%A9%EA%B5%B0" title="인민해방군">인민해방군</a>에 있어서 그의 영향력이 너무나 컸기 때문에, 당 지도자를 <a href="/wiki/%EC%9E%A5%EC%A9%8C%EB%AF%BC" title="장쩌민">장쩌민</a>으로 교체시키기 위해서 <a href="/wiki/1992%EB%85%84" title="1992년">1992년</a>에 덩샤오핑에 의해 정치국원의 자리로부터 제거된다. 양상쿤과 그의 동생 양바이빙은 군 내부에 너무나 많은 지지자와 동지가 있었기 때문에, 그의 사직으로 군의 최고위직의 절반이 퇴역하게 된 것은 잘 알려진 사실이다. 그리고 <a href="/wiki/%EC%A4%91%EA%B3%B5%ED%8C%94%EB%8C%80%EC%9B%90%EB%A1%9C" title="중공팔대원로">중공팔대원로</a> 중의 한 명이기도 하다.</p>
