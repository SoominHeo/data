<p><b>양시칠리아의 테레사 크리스티나</b>(Teresa Cristina Maria de Bourbon-Duas Sicilias, <a href="/wiki/1822%EB%85%84" title="1822년">1822년</a> <a href="/wiki/3%EC%9B%94_14%EC%9D%BC" title="3월 14일">3월 14일</a> ~ <a href="/wiki/1889%EB%85%84" title="1889년">1889년</a> <a href="/wiki/12%EC%9B%94_28%EC%9D%BC" title="12월 28일">12월 28일</a>)는 <a href="/wiki/%EB%B8%8C%EB%9D%BC%EC%A7%88_%EC%A0%9C%EA%B5%AD" title="브라질 제국">브라질 제국</a> <a href="/wiki/%ED%8E%98%EB%93%9C%EB%A3%A8_2%EC%84%B8_(%EB%B8%8C%EB%9D%BC%EC%A7%88)" title="페드루 2세 (브라질)">페드루 2세</a>의 황후다.</p>
<p><a href="/wiki/%EC%96%91%EC%8B%9C%EC%B9%A0%EB%A6%AC%EC%95%84_%EC%99%95%EA%B5%AD" title="양시칠리아 왕국">양시칠리아 왕국</a>의 <a href="/wiki/%ED%94%84%EB%9E%80%EC%B2%B4%EC%8A%A4%EC%BD%94_1%EC%84%B8_(%EC%96%91%EC%8B%9C%EC%B9%A0%EB%A6%AC%EC%95%84)" title="프란체스코 1세 (양시칠리아)">프란체스코 1세</a>와 그의 두 번째 왕비 <a class="new" href="/w/index.php?title=%EC%8A%A4%ED%8E%98%EC%9D%B8%EC%9D%98_%EB%A7%88%EB%A6%AC%EC%95%84_%EC%9D%B4%EC%9E%90%EB%B2%A8%EB%9D%BC&amp;action=edit&amp;redlink=1" title="스페인의 마리아 이자벨라 (없는 문서)">스페인의 마리아 이자벨라</a> 사이에서 태어났다. <a href="/wiki/%ED%8E%98%EB%A5%B4%EB%82%9C%EB%8F%84_7%EC%84%B8" title="페르난도 7세">페르난도 7세</a>의 왕비 <a href="/wiki/%EC%96%91%EC%8B%9C%EC%B9%A0%EB%A6%AC%EC%95%84%EC%9D%98_%EB%A7%88%EB%A6%AC%EC%95%84_%ED%81%AC%EB%A6%AC%EC%8A%A4%ED%8B%B0%EB%82%98" title="양시칠리아의 마리아 크리스티나">양시칠리아의 마리아 크리스티나</a>는 그녀의 언니이다. 테레사의 아버지 프란체스코 1세는 일찍 세상을 떠났고 어머니 마리아 이자벨라는 그녀에게 관심을 기울이지 않았다.</p>
<p><a href="/wiki/1842%EB%85%84" title="1842년">1842년</a> <a href="/wiki/9%EC%9B%94_4%EC%9D%BC" title="9월 4일">9월 4일</a> 테레사는 네 살 연하의 페드루와 결혼했다. 페드루는 아름답지 않은 외모에, 당시로서는 결혼적령기가 지난 테레사를 보고 실망했다. 그러나 테레사는 배려심이 깊고 지적인 여성이었고 예술에 재능이 있었으며 두 사람은 비슷한 분야에 대한 관심과 공통된 취미를 가지고 있었다. 그녀는 브라질에 시집오면서 예술가와 학자들을 데려왔고, 고고학 연구를 후원하기도 했다. 테레사는 남편을 헌신적으로 내조했고 자신의 의무를 충실히 이행했으며 사람들에게 존경받았다. 부부간의 애정은 열정적인 것은 아니었지만 서로를 존중했다. 페드루 2세는 아내 외에도 몇 명의 애첩이 있었지만 가정은 단란했다.</p>
<p><a href="/wiki/1889%EB%85%84" title="1889년">1889년</a> <a href="/wiki/11%EC%9B%94_15%EC%9D%BC" title="11월 15일">11월 15일</a> 쿠데타로 왕정이 폐지된 사건은 그녀에게 큰 충격을 안겨주었다. 유럽으로 향하는 배 안에서 그녀는 갑자기 건강이 악화되었고 <a class="mw-redirect" href="/wiki/%ED%8F%AC%EB%A5%B4%ED%86%A0" title="포르토">포르토</a>에 도착했을 땐 이미 손쓸 방도가 없었다. <a href="/wiki/12%EC%9B%94_28%EC%9D%BC" title="12월 28일">12월 28일</a> 그녀는 심장발작으로 죽었고 브라질 사람들은 황후의 죽음을 애도했다. 테레사의 유해는 <a class="mw-redirect" href="/wiki/%EB%B8%8C%EB%9D%BC%EA%B0%84%EC%82%AC_%EC%99%95%EA%B0%80" title="브라간사 왕가">브라간사 왕가</a>의 영묘에 묻혔다가 <a href="/wiki/%ED%8E%98%ED%8A%B8%EB%A1%9C%ED%8F%B4%EB%A6%AC%EC%8A%A4" title="페트로폴리스">페트로폴리스</a>에 이장되었다. 테레사가 생전에 수집한 예술품들은 테레사 마리아 크리스티나 컬렉션이라고 명명되어 <a class="mw-redirect" href="/wiki/%EC%9C%A0%EB%84%A4%EC%8A%A4%EC%BD%94" title="유네스코">유네스코</a> <a href="/wiki/%EC%84%B8%EA%B3%84%EA%B8%B0%EB%A1%9D%EC%9C%A0%EC%82%B0" title="세계기록유산">세계기록유산</a>에 등재되어 있다.</p>
