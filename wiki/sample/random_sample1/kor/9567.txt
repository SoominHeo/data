<p><b>양조</b>(釀造)는 주로 <a href="/wiki/%EA%B3%A1%EB%AC%BC" title="곡물">곡물</a>에서 얻은 <a href="/wiki/%EB%85%B9%EB%A7%90" title="녹말">녹말</a>을 <a href="/wiki/%EB%AC%BC" title="물">물</a>에 담근 뒤, <a href="/wiki/%ED%9A%A8%EB%AA%A8" title="효모">효모</a>로 <a href="/wiki/%EB%B0%9C%ED%9A%A8" title="발효">발효</a>시켜 <a href="/wiki/%EC%88%A0" title="술">술</a>을 만드는 과정이다. 양조는 <a class="mw-redirect" href="/wiki/%EA%B8%B0%EC%9B%90%EC%A0%84" title="기원전">기원전</a> 6천년 경부터 시작되었으며, <a href="/wiki/%EA%B3%A0%EB%8C%80_%EC%9D%B4%EC%A7%91%ED%8A%B8" title="고대 이집트">고대 이집트</a>에서 양조 기술을 사용하였다는 증거가 발견되었고, 현존하는 문헌 중 가장 오래된 것으로 알려진 <a href="/wiki/%EC%88%98%EB%A9%94%EB%A5%B4" title="수메르">수메르</a>의 몇몇 문헌에서 다양한 양조 방법에 대한 설명이 발견되었다.<sup class="reference" id="cite_ref-1"><a href="#cite_note-1">[1]</a></sup><sup class="reference" id="cite_ref-2"><a href="#cite_note-2">[2]</a></sup><sup class="reference" id="cite_ref-3"><a href="#cite_note-3">[3]</a></sup> 양조는 양조업자에 의해 <a class="new" href="/w/index.php?title=%EC%96%91%EC%A1%B0%EC%9E%A5&amp;action=edit&amp;redlink=1" title="양조장 (없는 문서)">양조장</a>에서 이루어지며, <a class="mw-redirect" href="/wiki/%EC%96%91%EC%A1%B0%EC%97%85" title="양조업">양조업</a>은 대부분의 <a href="/wiki/%EC%84%9C%EC%9C%A0%EB%9F%BD" title="서유럽">서유럽</a> 국가들의 경제의 상당 부분을 차지한다.</p>
<p>양조의 기본 재료는 <a href="/wiki/%EB%AC%BC" title="물">물</a>, <a href="/wiki/%EB%B0%9C%ED%9A%A8" title="발효">발효</a>시켜 <a href="/wiki/%EC%95%8C%EC%BD%94%EC%98%AC" title="알코올">알코올</a>로 변환시킬 수 있는 <a href="/wiki/%EB%85%B9%EB%A7%90" title="녹말">녹말</a> (주로 <a class="mw-redirect" href="/wiki/%EB%A7%A5%EC%95%84" title="맥아">맥아</a> 등에서 채취), 발효 과정을 일으키는 <a href="/wiki/%ED%9A%A8%EB%AA%A8" title="효모">효모</a>, <a href="/wiki/%ED%99%89" title="홉">홉</a>과 같은 <a href="/wiki/%EC%A1%B0%EB%AF%B8%EB%A3%8C" title="조미료">조미료</a> 등이다.<sup class="reference" id="cite_ref-alabev.com_4-0"><a href="#cite_note-alabev.com-4">[4]</a></sup> 부차적으로 녹말을 얻는 <a class="new" href="/w/index.php?title=%EB%B6%80%EA%B0%80%EB%AC%BC&amp;action=edit&amp;redlink=1" title="부가물 (없는 문서)">부가물</a>로 <a href="/wiki/%EC%98%A5%EC%88%98%EC%88%98" title="옥수수">옥수수</a>, <a href="/wiki/%EC%8C%80" title="쌀">쌀</a>, <a href="/wiki/%EC%84%A4%ED%83%95" title="설탕">설탕</a> 등이 사용되며,<sup class="reference" id="cite_ref-beer-brewing.com_5-0"><a href="#cite_note-beer-brewing.com-5">[5]</a></sup> <a href="/wiki/%EC%95%84%ED%94%84%EB%A6%AC%EC%B9%B4" title="아프리카">아프리카</a>에서는 <a href="/wiki/%EC%A1%B0_(%EC%8B%9D%EB%AC%BC)" title="조 (식물)">조</a>, <a href="/wiki/%EC%88%98%EC%88%98" title="수수">수수</a>, <a href="/wiki/%EC%B9%B4%EC%82%AC%EB%B0%94" title="카사바">카사바</a> 뿌리가, <a href="/wiki/%EB%B8%8C%EB%9D%BC%EC%A7%88" title="브라질">브라질</a>에서는 <a href="/wiki/%EA%B0%90%EC%9E%90" title="감자">감자</a>가, <a href="/wiki/%EB%A9%95%EC%8B%9C%EC%BD%94" title="멕시코">멕시코</a>에서는 <a href="/wiki/%EC%9A%A9%EC%84%A4%EB%9E%80" title="용설란">용설란</a>이 시용된다.<sup class="reference" id="cite_ref-BeerHunter.com_6-0"><a href="#cite_note-BeerHunter.com-6">[6]</a></sup></p>
<p>양조가 이루어지는 데에는 <a class="new" href="/w/index.php?title=%EC%A0%9C%EB%A7%A5%EC%95%84&amp;action=edit&amp;redlink=1" title="제맥아 (없는 문서)">제맥아</a>, <a class="new" href="/w/index.php?title=%EC%A0%9C%EB%B6%84&amp;action=edit&amp;redlink=1" title="제분 (없는 문서)">제분</a>, 담금, 끓임, <a href="/wiki/%EB%B0%9C%ED%9A%A8" title="발효">발효</a>, <a href="/wiki/%EC%88%99%EC%84%B1" title="숙성">숙성</a>, <a href="/wiki/%EC%97%AC%EA%B3%BC" title="여과">여과</a>, <a href="/wiki/%ED%8F%AC%EC%9E%A5" title="포장">포장</a> 등의 몇 가지 과정을 거친다. 빌효는 따뜻하게 하는 것, 차게 하는 것, 자생적으로 하는 것의 세 가지 방법이 있는데, 과정에 따라 <a href="/wiki/%EC%97%90%EC%9D%BC" title="에일">에일</a>, <a href="/wiki/%EB%9D%BC%EA%B1%B0" title="라거">라거</a>, <a class="new" href="/w/index.php?title=%EB%9E%8C%EB%B9%84%ED%81%AC&amp;action=edit&amp;redlink=1" title="람비크 (없는 문서)">람비크</a> 등의 다른 <a href="/wiki/%EC%88%A0" title="술">술</a>이 만들어진다. 발효는 <a class="new" href="/w/index.php?title=%EC%96%91%EC%A1%B0%EC%9E%A5&amp;action=edit&amp;redlink=1" title="양조장 (없는 문서)">양조장</a>에서 개폐되거나 밀폐된 용기에서 이루어지며, 부차적으로 <a class="new" href="/w/index.php?title=%EC%88%A0%ED%86%B5&amp;action=edit&amp;redlink=1" title="술통 (없는 문서)">술통</a>이나 <a class="mw-redirect" href="/wiki/%EC%88%A0%EB%B3%91" title="술병">술병</a>에서 이루어지는 경우도 있다.</p>
<p><a href="/wiki/%EC%88%A0" title="술">술</a> 이외에도 <a href="/wiki/%EA%B0%84%EC%9E%A5" title="간장">간장</a>이나 <a href="/wiki/%EC%8B%9D%EC%B4%88" title="식초">식초</a> 등을 담그는 것 또한 양조라고 하는데, <a href="/wiki/%ED%8F%AC%EB%8F%84%EC%A3%BC" title="포도주">포도주</a>나 <a href="/wiki/%EC%82%AC%EA%B3%BC%EC%A3%BC" title="사과주">사과주</a>처럼 <a href="/wiki/%EA%B3%BC%EC%9D%BC" title="과일">과일</a>에 압력을 가하여 용액을 추출하는 것이나 <a class="mw-redirect" href="/wiki/%EB%B2%8C%EA%BF%80%EC%88%A0" title="벌꿀술">벌꿀술</a>처럼 <a href="/wiki/%EB%B2%8C%EA%BF%80" title="벌꿀">벌꿀</a> 자체만을 이용하는 것은 포함되지 않는다.</p>
