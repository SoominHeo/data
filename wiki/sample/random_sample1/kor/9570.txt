<p><b>양줘나</b>(<b>楊卓娜</b>, <b>양탁나</b>, 본명은 <b>양나</b>(<b>楊娜</b>), <a href="/wiki/%EC%95%84%EB%AA%85" title="아명">아명(兒名)</a>은 <b>양줘링</b>(<b>楊卓玲</b>, <b>양탁령</b>), <b>양리나</b>(<b>楊麗娜</b>, <b>양여나</b>, <b>양려나</b>), <a href="/wiki/%EC%98%81%EC%96%B4" title="영어">영문</a>명은 <b>Griselda Yeung</b>, <a href="/wiki/1977%EB%85%84" title="1977년">1977년</a> <a href="/wiki/11%EC%9B%94_10%EC%9D%BC" title="11월 10일">11월 10일</a> ~ )는 <a href="/wiki/%ED%99%8D%EC%BD%A9" title="홍콩">중화인민공화국 홍콩 특별행정구</a>의 여성 <a class="mw-redirect" href="/wiki/%EC%98%81%ED%99%94_%EB%B0%B0%EC%9A%B0" title="영화 배우">영화배우</a>, <a href="/wiki/%EB%B0%B0%EC%9A%B0" title="배우">텔레비전 연기자</a>이다.</p>
<h2><span class="mw-headline" id=".EC.83.9D.EC.95.A0">생애</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%96%91%EC%A4%98%EB%82%98&amp;action=edit&amp;section=1" title="부분 편집: 생애">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>그녀의 원적지는 <a href="/wiki/%EC%A4%91%ED%99%94%EC%9D%B8%EB%AF%BC%EA%B3%B5%ED%99%94%EA%B5%AD" title="중화인민공화국">중화인민공화국</a> <a href="/wiki/%EA%B4%91%EB%91%A5_%EC%84%B1" title="광둥 성">광둥 성</a> <a href="/wiki/%EC%9E%94%EC%9E%A5_%EC%8B%9C" title="잔장 시">잔장 시</a> <a href="/wiki/%EC%B8%A0%EC%B9%B8_%EA%B5%AC" title="츠칸 구">츠칸 구</a>이다.</p>
<p>그녀는 <a href="/wiki/%EC%A4%91%ED%99%94%EC%9D%B8%EB%AF%BC%EA%B3%B5%ED%99%94%EA%B5%AD" title="중화인민공화국">중화인민공화국</a> <a href="/wiki/%EA%B4%91%EB%91%A5_%EC%84%B1" title="광둥 성">광둥 성</a> <a href="/wiki/%EC%9E%94%EC%9E%A5_%EC%8B%9C" title="잔장 시">잔장</a> 외가에서 출생하였으며 <a href="/wiki/%EA%B4%91%EC%A0%80%EC%9A%B0_%EC%8B%9C" title="광저우 시">중화인민공화국 광둥 성 광저우</a>와 <a href="/wiki/%EC%A4%91%ED%99%94%EC%9D%B8%EB%AF%BC%EA%B3%B5%ED%99%94%EA%B5%AD" title="중화인민공화국">중화인민공화국</a> <a href="/wiki/%EA%B4%91%EB%91%A5_%EC%84%B1" title="광둥 성">광둥 성</a> <a href="/wiki/%EC%9E%94%EC%9E%A5_%EC%8B%9C" title="잔장 시">잔장 시</a> <a href="/wiki/%EC%B8%A0%EC%B9%B8_%EA%B5%AC" title="츠칸 구">츠칸 구</a>에서 잠시 유년기를 보낸 적이 있는 그녀는 <a href="/wiki/1983%EB%85%84" title="1983년">1983년</a> <a href="/wiki/%ED%99%8D%EC%BD%A9" title="홍콩">홍콩</a>에 이주하여 <a href="/wiki/%ED%99%8D%EC%BD%A9" title="홍콩">홍콩</a> 본가에서 성장하였다.</p>
<p>그녀는 <a href="/wiki/2001%EB%85%84" title="2001년">2001년</a> <a class="extiw" href="https://zh.wikipedia.org/wiki/%E9%A6%99%E6%B8%AF%E5%B0%8F%E5%A7%90%E7%AB%B6%E9%81%B8" title="zh:香港小姐競選">미스 홍콩 선발대회</a>에서 입상하여 첫 데뷔하였고 <a href="/wiki/2002%EB%85%84" title="2002년">2002년</a> <a href="/wiki/%ED%85%94%EB%A0%88%EB%B9%84%EC%A0%84_%EB%93%9C%EB%9D%BC%EB%A7%88" title="텔레비전 드라마">텔레비전 드라마</a>에서 <a href="/wiki/%EB%B0%B0%EC%9A%B0" title="배우">연기자</a> 데뷔하였으며 <a href="/wiki/2003%EB%85%84" title="2003년">2003년</a> <a class="mw-redirect" href="/wiki/%ED%99%8D%EC%BD%A9_%EC%98%81%ED%99%94" title="홍콩 영화">홍콩 영화</a> 《심조주걸륜(尋找周杰倫)》의 단역으로 <a class="mw-redirect" href="/wiki/%EC%98%81%ED%99%94_%EB%B0%B0%EC%9A%B0" title="영화 배우">영화배우</a> 데뷔하였다.</p>
<p>그녀의 여동생은 <a href="/wiki/%ED%99%8D%EC%BD%A9" title="홍콩">홍콩</a>의 <a class="mw-redirect" href="/wiki/%EC%98%81%ED%99%94_%EB%B0%B0%EC%9A%B0" title="영화 배우">영화배우</a> 겸 <a href="/wiki/%EA%B0%80%EC%88%98" title="가수">가수</a>인 <a href="/wiki/%EC%96%91%EC%9D%B4" title="양이">양이</a>(楊怡)이다.</p>
