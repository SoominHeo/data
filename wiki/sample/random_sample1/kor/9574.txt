<h1 class="firstHeading" id="firstHeading" lang="ko">양쪽성</h1>
<div class="mw-body-content" id="bodyContent">
<div id="siteSub">위키백과, 우리 모두의 백과사전.</div>
<div id="contentSub"></div>
<div class="mw-jump" id="jump-to-nav">
					이동:					<a href="#mw-head">둘러보기</a>, 					<a href="#p-search">검색</a>
</div>
<div class="mw-content-ltr" dir="ltr" id="mw-content-text" lang="ko"><script>function mfTempOpenSection(id){var block=document.getElementById("mf-section-"+id);block.className+=" open-block";block.previousSibling.className+=" open-block";}</script><p><a href="/wiki/%ED%99%94%ED%95%99" title="화학">화학</a>에서 <b>양쪽성</b> 물질은 <a href="/wiki/%EC%82%B0_(%ED%99%94%ED%95%99)" title="산 (화학)">산성</a>과 <a href="/wiki/%EC%97%BC%EA%B8%B0" title="염기">염기성</a> 모두로 반응하는 것을 말한다. <a href="/wiki/%EC%95%84%EB%AF%B8%EB%85%B8%EC%82%B0" title="아미노산">아미노산</a>, <a href="/wiki/%EB%8B%A8%EB%B0%B1%EC%A7%88" title="단백질">단백질</a>, <a href="/wiki/%EB%AC%BC" title="물">물</a> 등이 해당한다. <a href="/wiki/%EC%95%84%EC%97%B0" title="아연">아연</a>, <a href="/wiki/%EC%A3%BC%EC%84%9D_(%EC%9B%90%EC%86%8C)" title="주석 (원소)">주석</a>, <a href="/wiki/%EC%95%8C%EB%A3%A8%EB%AF%B8%EB%8A%84" title="알루미늄">알루미늄</a>, <a href="/wiki/%EB%B2%A0%EB%A6%B4%EB%A5%A8" title="베릴륨">베릴륨</a> 등은 양쪽성 <a href="/wiki/%EC%82%B0%ED%99%94%EB%AC%BC" title="산화물">산화물</a>을 만든다. 여기서는 <a class="new" href="/w/index.php?title=%EB%B8%8C%EB%A2%B4%EC%8A%A4%ED%85%8C%EB%93%9C-%EB%A1%9C%EC%9A%B0%EB%A6%AC%EC%9D%98_%EC%82%B0-%EC%97%BC%EA%B8%B0&amp;action=edit&amp;redlink=1" title="브뢴스테드-로우리의 산-염기 (없는 문서)">브뢴스테드-로우리의 산-염기</a>에 의해 <a href="/wiki/%EC%82%B0_(%ED%99%94%ED%95%99)" title="산 (화학)">산</a>을 다른 물질에게 H<sup>+</sup>를 주는 물질, <a href="/wiki/%EC%97%BC%EA%B8%B0" title="염기">염기</a>를 다른 물질로부터 H<sup>+</sup>를 받아들이는 물질로 생각한다.</p>
<h2><span class="mw-headline" id=".EC.96.91.EC.AA.BD.EC.84.B1_.ED.99.94.ED.95.A9.EB.AC.BC">양쪽성 화합물</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%96%91%EC%AA%BD%EC%84%B1&amp;action=edit&amp;section=1" title="부분 편집: 양쪽성 화합물">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>양쪽성 화합물은 염기성 화합물과 반응하여 <a class="new" href="/w/index.php?title=%EC%A7%9D%EC%82%B0&amp;action=edit&amp;redlink=1" title="짝산 (없는 문서)">짝산</a>이 되고 산성 화합물과도 반응하여 <a class="new" href="/w/index.php?title=%EC%A7%9D%EC%97%BC%EA%B8%B0&amp;action=edit&amp;redlink=1" title="짝염기 (없는 문서)">짝염기</a>가 되는 화합물을 일컫는데, 대표적인 예로 H<sub>2</sub>0, Al(OH)<sub>3</sub> 등이 있다.</p>
<p>예를 들어 산화 아연(<a href="/wiki/%EC%95%84%EC%97%B0" title="아연">Zn</a><a href="/wiki/%EC%82%B0%EC%86%8C" title="산소">O</a>)은 용매(여기서는 H<sub>2</sub>0가 용매이다.)의 pH 값에 따라</p>
<p>산성 용액에서는</p>
<dl>
<dd>ZnO + 2H<sup>+</sup> → Zn<sup>2+</sup> + H<sub>2</sub>O</dd>
</dl>
<p>염기성 용액에서는</p>
<dl>
<dd>ZnO + H<sub>2</sub>O + 2OH<sup>-</sup> → [Zn(OH)<sub>4</sub>]<sup>2-</sup></dd>
</dl>
<p>로 이온화된다.</p>
<p>양쪽성 화합물의 예는 그밖에도 많다. 가장 대표적인 경우가 <a href="/wiki/%EB%AC%BC" title="물">물</a>이다.</p>
<p>염기:</p>
<dl>
<dd>H<sub>2</sub>O + HCl → H<sub>3</sub>O<sup>+</sup> + Cl<sup>−</sup></dd>
</dl>
<p>산:</p>
<dl>
<dd>H<sub>2</sub>O + NH<sub>3</sub> → NH<sub>4</sub><sup>+</sup> + OH<sup>−</sup></dd>
</dl>
<p><a href="/wiki/%EC%88%98%EC%82%B0%ED%99%94_%EC%95%8C%EB%A3%A8%EB%AF%B8%EB%8A%84" title="수산화 알루미늄">수산화 알루미늄</a> 역시 양쪽성을 가진다.</p>
<p>염기:</p>
<dl>
<dd>Al(OH)<sub>3</sub> + 3HCl → AlCl<sub>3</sub> + 3H<sub>2</sub>O</dd>
</dl>
<p>산:</p>
<dl>
<dd>Al(OH)<sub>3</sub> + NaOH → NaAl(OH)<sub>4</sub></dd>
</dl>
<h2><span class="mw-headline" id=".EC.96.91.EC.AA.BD.EC.84.B1_.EC.9D.B4.EC.98.A8">양쪽성 이온</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%96%91%EC%AA%BD%EC%84%B1&amp;action=edit&amp;section=2" title="부분 편집: 양쪽성 이온">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>한편 이온중에도 양쪽성을 띠는 물질이 있는데 이런 이온을 양쪽성 이온이라고 한다.</p>
<p>가장 대표적인 예로 HSO<sub>4</sub><sup>-</sup>가 있다.</p>
<p>HSO<sub>4</sub><sup>-</sup>는 pH 7이상의 용매 속에서</p>
<p>HSO<sub>4</sub><sup>-</sup> + H<sup>+</sup> → H<sub>2</sub>SO<sub>4</sub> (알짜이온 반응식)</p>
<p>로 반응하는데 여기서 HSO<sub>4</sub><sup>-</sup>는 수소이온을 받아들였으므로 염기로 작용하지만</p>
<p>pH 7이하의 용매 속에서는</p>
<p>HSO<sub>4</sub><sup>-</sup> + OH<sup>-</sup> → SO<sub>4</sub><sup>2-</sup> + H<sub>2</sub>O (알짜이온 반응식)</p>
<p>로 반응하게 되어 여기서는 HSO<sub>4</sub><sup>-</sup>가 수소이온을 내보내게 되어 산으로 작용한다.</p>
