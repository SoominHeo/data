<p><b>양효진</b>(梁孝眞, <a href="/wiki/1989%EB%85%84" title="1989년">1989년</a> <a href="/wiki/12%EC%9B%94_14%EC%9D%BC" title="12월 14일">12월 14일</a> ~ )은 <a href="/wiki/%EB%8C%80%ED%95%9C%EB%AF%BC%EA%B5%AD" title="대한민국">대한민국</a>의 여자 <a href="/wiki/%EB%B0%B0%EA%B5%AC" title="배구">배구</a> 선수이고 <a class="mw-redirect" href="/wiki/%EC%88%98%EC%9B%90_%ED%98%84%EB%8C%80%EA%B1%B4%EC%84%A4_%ED%9E%90%EC%8A%A4%ED%85%8C%EC%9D%B4%ED%8A%B8" title="수원 현대건설 힐스테이트">수원 현대건설 힐스테이트</a> 소속의 센터이다. 스파이크 높이는 287 cm, 블로킹 높이는 280cm이다.</p>

<p>양효진은 <a href="/wiki/%EB%B6%80%EC%82%B0%EA%B4%91%EC%97%AD%EC%8B%9C" title="부산광역시">부산광역시</a> 출신으로 초등학교 4학년 때 신장이 165cm였고, 부산여중 3학년 때 186cm까지 성장하였다.<sup class="reference" id="cite_ref-1"><a href="#cite_note-1">[1]</a></sup></p>
<h2><span class="mw-headline" id=".ED.94.84.EB.A1.9C_.EC.84.A0.EC.88.98_.EC.8B.9C.EC.A0.88">프로 선수 시절</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%96%91%ED%9A%A8%EC%A7%84&amp;action=edit&amp;section=2" title="부분 편집: 프로 선수 시절">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p><a href="/wiki/%EB%82%A8%EC%84%B1%EC%97%AC%EC%9E%90%EA%B3%A0%EB%93%B1%ED%95%99%EA%B5%90" title="남성여자고등학교">남성여고</a>를 졸업하고 <a href="/wiki/2007%EB%85%84" title="2007년">2007년</a>에 <a class="mw-redirect" href="/wiki/NH%EB%86%8D%ED%98%91_2007~2008_V-%EB%A6%AC%EA%B7%B8#.EC.8B.A0.EC.9D.B8_.EB.93.9C.EB.9E.98.ED.94.84.ED.8A.B8" title="NH농협 2007~2008 V-리그">신인 드래프트</a>에서 1라운드 4순위로 지명받아서<a class="mw-redirect" href="/wiki/%EC%88%98%EC%9B%90_%ED%98%84%EB%8C%80%EA%B1%B4%EC%84%A4_%ED%9E%90%EC%8A%A4%ED%85%8C%EC%9D%B4%ED%8A%B8" title="수원 현대건설 힐스테이트">현대건설</a>에 입단했다. 프로 데뷔 시즌 정규 리그에 양효진은 그 해 신인들 중에서는 가장 많은 득점(308점)을 기록했고, 블로킹 감각도 뛰어나 전체 3위(세트당 0.57개)를 차지했다.<sup class="reference" id="cite_ref-2"><a href="#cite_note-2">[2]</a></sup> 그러나 그 해 우승을 차지한 GS칼텍스의 센터 <a href="/wiki/%EB%B0%B0%EC%9C%A0%EB%82%98" title="배유나">배유나</a>에게 밀려 신인왕 수상에는 실패했다. 남성여고 시절 시간차 공격만을 시도하여 속공이 부족하고, 키에 비해 체중이 너무 적게 나간다는 것이 단점으로 지목되기도 한다.<sup class="reference" id="cite_ref-.EC.8B.A0.EC.9D.B8.EC.99.95_3-0"><a href="#cite_note-.EC.8B.A0.EC.9D.B8.EC.99.95-3">[3]</a></sup> 이동 공격은 아직 능숙하지 않지만 본인이 적극적으로 시도하려고 하기에 상당히 희망적이며 속공도 점차 위력적으로 되어가고 있다. 드래프트에서 앞순위는 아니었으나 센터부분에서 <a href="/wiki/%EA%B9%80%EC%84%B8%EC%98%81_(%EB%B0%B0%EA%B5%AC_%EC%84%A0%EC%88%98)" title="김세영 (배구 선수)">김세영</a>과 <a href="/wiki/%EA%B9%80%ED%98%9C%EC%A7%84_(%EB%B0%B0%EA%B5%AC_%EC%84%A0%EC%88%98)" title="김혜진 (배구 선수)">김혜진</a>과 더불어 단연 두각을 나타내고 있는 장신 센터이다.</p>
<p>양효진은 <a href="/wiki/2008%EB%85%84_%ED%95%98%EA%B3%84_%EC%98%AC%EB%A6%BC%ED%94%BD" title="2008년 하계 올림픽">베이징 올림픽</a> 최종 예선 국가대표로 선발되어 <a href="/wiki/%ED%83%9C%EB%A6%89%EC%84%A0%EC%88%98%EC%B4%8C" title="태릉선수촌">태릉선수촌</a>에 입촌했다.</p>
<p><a href="/wiki/2013%EB%85%84" title="2013년">2013년</a> <a href="/wiki/1%EC%9B%94_26%EC%9D%BC" title="1월 26일">1월 26일</a> <a href="/wiki/%EC%84%B1%EB%82%A8%EC%8B%A4%EB%82%B4%EC%B2%B4%EC%9C%A1%EA%B4%80" title="성남실내체육관">성남실내체육관</a>에서 열린 <a class="mw-redirect" href="/wiki/%EC%84%B1%EB%82%A8_%ED%95%9C%EA%B5%AD%EB%8F%84%EB%A1%9C%EA%B3%B5%EC%82%AC_%ED%95%98%EC%9D%B4%ED%8C%A8%EC%8A%A4_%EC%A0%9C%EB%8B%88%EC%8A%A4" title="성남 한국도로공사 하이패스 제니스">도로공사</a>와의 원정경기에서 40득점을 올리면서 팀 승리에 기여했는데, 40득점은 2012년 12월 30일 흥국생명과의 경기에서 기록한 개인 최다 득점 26점을 경신한 것이다.<sup class="reference" id="cite_ref-4"><a href="#cite_note-4">[4]</a></sup></p>
