<p><b>어도비 드림위버</b>(Adobe Dreamweaver)는 <a href="/wiki/%EB%A7%A4%ED%81%AC%EB%A1%9C%EB%AF%B8%EB%94%94%EC%96%B4" title="매크로미디어">매크로미디어</a>가 개발한 <a href="/wiki/%EC%9B%B9_%EA%B0%9C%EB%B0%9C" title="웹 개발">웹 개발</a> <a href="/wiki/%EC%9D%91%EC%9A%A9_%EC%86%8C%ED%94%84%ED%8A%B8%EC%9B%A8%EC%96%B4" title="응용 소프트웨어">응용 프로그램</a>이며, 현재는 2005년에 매크로미디어를 인수하였던 <a href="/wiki/%EC%96%B4%EB%8F%84%EB%B9%84_%EC%8B%9C%EC%8A%A4%ED%85%9C%EC%A6%88" title="어도비 시스템즈">어도비 시스템즈</a> 소유이다.<sup class="reference" id="cite_ref-1"><a href="#cite_note-1">[1]</a></sup></p>
<p>드림위버는 <a href="/wiki/%EB%A7%A5_OS" title="맥 OS">맥</a>과 <a class="mw-redirect" href="/wiki/%EB%A7%88%EC%9D%B4%ED%81%AC%EB%A1%9C%EC%86%8C%ED%94%84%ED%8A%B8_%EC%9C%88%EB%8F%84" title="마이크로소프트 윈도">윈도</a> <a href="/wiki/%EC%9A%B4%EC%98%81_%EC%B2%B4%EC%A0%9C" title="운영 체제">운영 체제</a>를 둘 다 지원한다. 최근에 나온 버전들은 <a class="mw-redirect" href="/wiki/CSS" title="CSS">CSS</a>, <a href="/wiki/%EC%9E%90%EB%B0%94%EC%8A%A4%ED%81%AC%EB%A6%BD%ED%8A%B8" title="자바스크립트">자바스크립트</a>, 다양한 <a class="mw-redirect" href="/wiki/%EC%84%9C%EB%B2%84_%EC%82%AC%EC%9D%B4%EB%93%9C_%EC%8A%A4%ED%81%AC%EB%A6%BD%ED%8C%85" title="서버 사이드 스크립팅">서버 사이드 스크립팅</a>, 그리고 <a href="/wiki/ASP.NET" title="ASP.NET">ASP.NET</a>, <a href="/wiki/%EC%96%B4%EB%8F%84%EB%B9%84_%EC%BD%9C%EB%93%9C%ED%93%A8%EC%A0%84" title="어도비 콜드퓨전">콜드퓨전</a>, <a class="mw-redirect" href="/wiki/%EC%9E%90%EB%B0%94_%EC%84%9C%EB%B2%84_%ED%8E%98%EC%9D%B4%EC%A7%80" title="자바 서버 페이지">자바 서버 페이지</a>, <a href="/wiki/PHP" title="PHP">PHP</a> 등의 <a href="/wiki/%ED%94%84%EB%A1%9C%EA%B7%B8%EB%9E%98%EB%B0%8D_%EC%96%B8%EC%96%B4" title="프로그래밍 언어">프로그래밍 언어</a> 및 <a href="/wiki/%EC%86%8C%ED%94%84%ED%8A%B8%EC%9B%A8%EC%96%B4_%ED%94%84%EB%A0%88%EC%9E%84%EC%9B%8C%ED%81%AC" title="소프트웨어 프레임워크">프레임워크</a>과 같은 <a href="/wiki/%EC%9B%94%EB%93%9C_%EC%99%80%EC%9D%B4%EB%93%9C_%EC%9B%B9" title="월드 와이드 웹">웹</a> 기술에 대한 지원을 포함하고 있다. 현재는 <a href="/wiki/%EC%96%B4%EB%8F%84%EB%B9%84_%ED%81%AC%EB%A6%AC%EC%97%90%EC%9D%B4%ED%8B%B0%EB%B8%8C_%ED%81%B4%EB%9D%BC%EC%9A%B0%EB%93%9C" title="어도비 크리에이티브 클라우드">어도비 크리에이티브 클라우드</a>에서 무료로 다운로드하여 사용할 수 있다.</p>

<h2><span class="mw-headline" id=".EA.B8.B0.EB.8A.A5">기능</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%96%B4%EB%8F%84%EB%B9%84_%EB%93%9C%EB%A6%BC%EC%9C%84%EB%B2%84&amp;action=edit&amp;section=1" title="부분 편집: 기능">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>드림위버는 <a href="/wiki/%EC%9C%84%EC%A7%80%EC%9C%84%EA%B7%B8" title="위지위그">위지위그</a> <a href="/wiki/%ED%94%84%EB%A0%88%EC%8A%A4%ED%86%A0_(%EB%A0%88%EC%9D%B4%EC%95%84%EC%9B%83_%EC%97%94%EC%A7%84)" title="프레스토 (레이아웃 엔진)">프레스토</a> 기반의 편집기이며 사용자에 의해 HTML 코드의 상세한 정보를 숨길 수 있으며, 이로써 코드를 잘 모르는 사람들이 웹 페이지와 사이트를 만들 수 있게 도와 준다.</p>
