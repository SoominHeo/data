<p><b>어도비 콜드퓨전</b>(Adobe ColdFusion)은 1995년 제레미 앨라이어와 조셉 앨라이어가 만든 상업적인 <a href="/wiki/%EA%B3%A0%EC%86%8D_%EC%9D%91%EC%9A%A9_%ED%94%84%EB%A1%9C%EA%B7%B8%EB%9E%A8_%EA%B0%9C%EB%B0%9C" title="고속 응용 프로그램 개발">RAD</a> 플랫폼이다. 최초 개발 의도는 간단하게 데이터베이스와 HTML 페이지를 연동하기 위해 설계되었는데 2.0버전부터는 <a href="/wiki/%ED%86%B5%ED%95%A9_%EA%B0%9C%EB%B0%9C_%ED%99%98%EA%B2%BD" title="통합 개발 환경">IDE(통합개발환경)</a>와 <a href="/wiki/%EC%8A%A4%ED%81%AC%EB%A6%BD%ED%8A%B8_%EC%96%B8%EC%96%B4" title="스크립트 언어">스크립트 언어</a>를 포함한 개발 플랫폼으로 거듭나게 되었다. <a href="/wiki/%EC%96%B4%EB%8F%84%EB%B9%84_%EC%8B%9C%EC%8A%A4%ED%85%9C%EC%A6%88" title="어도비 시스템즈">어도비 시스템즈</a>에서 제공하고 있는 현재 버전의 콜드퓨전은 기업업무시스템과 <a href="/wiki/%EB%A6%AC%EC%B9%98_%EC%9D%B8%ED%84%B0%EB%84%B7_%EC%95%A0%ED%94%8C%EB%A6%AC%EC%BC%80%EC%9D%B4%EC%85%98" title="리치 인터넷 애플리케이션">RIA</a> 개발을 위한 통합된 기능을 제공하고 있다. 콜드퓨전은 주로 <a href="/wiki/PHP" title="PHP">PHP</a>와 <a href="/wiki/%EC%95%A1%ED%8B%B0%EB%B8%8C_%EC%84%9C%EB%B2%84_%ED%8E%98%EC%9D%B4%EC%A7%80" title="액티브 서버 페이지">ASP</a>와 비교되곤 한다.</p>

<h2><span class="mw-headline" id=".EA.B0.9C.EC.9A.94">개요</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%96%B4%EB%8F%84%EB%B9%84_%EC%BD%9C%EB%93%9C%ED%93%A8%EC%A0%84&amp;action=edit&amp;section=1" title="부분 편집: 개요">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>콜드퓨전은 1995년 얼래어(Allaire)의 제품으로 조셉과 제레미 형제에 의해 발표되었다. 2001년 앨라이어는 <a href="/wiki/%EB%A7%A4%ED%81%AC%EB%A1%9C%EB%AF%B8%EB%94%94%EC%96%B4" title="매크로미디어">매크로미디어</a>와 합병되었고 다시 2005년 <a href="/wiki/%EC%96%B4%EB%8F%84%EB%B9%84_%EC%8B%9C%EC%8A%A4%ED%85%9C%EC%A6%88" title="어도비 시스템즈">어도비 시스템즈</a>로 인수되었다.</p>
<p>콜드퓨전 MX 7 엔터프라이즈 에디션부터는 게이트웨이 인터페이스를 통해 <a href="/wiki/%EB%8B%A8%EB%AC%B8_%EB%A9%94%EC%8B%9C%EC%A7%80_%EC%84%9C%EB%B9%84%EC%8A%A4" title="단문 메시지 서비스">SMS</a>나 <a href="/wiki/%EC%9D%B8%EC%8A%A4%ED%84%B4%ED%8A%B8_%EB%A9%94%EC%8B%A0%EC%A0%80" title="인스턴트 메신저">인스턴트 메신저</a>와 같은 비동기 이벤트처리가 다룰 수 있는 기능이 추가되었다.</p>
<p>콜드퓨전은 주로 <a href="/wiki/%EC%9B%B9%EC%82%AC%EC%9D%B4%ED%8A%B8" title="웹사이트">웹사이트</a>나 <a href="/wiki/%EC%9D%B8%ED%8A%B8%EB%9D%BC%EB%84%B7" title="인트라넷">인트라넷</a>에서 데이터처리를 위해 사용되며 <a href="/wiki/SOAP" title="SOAP">SOAP</a>이나 <a href="/wiki/%EC%9B%B9_%EC%84%9C%EB%B9%84%EC%8A%A4" title="웹 서비스">웹 서비스</a>, <a href="/wiki/%EC%96%B4%EB%8F%84%EB%B9%84_%ED%94%8C%EB%9E%98%EC%8B%9C" title="어도비 플래시">플래시</a> 리모팅과 같은 원격 서비스를 생성하기 위해 사용되기도 한다. 특히 클라이언트에서 사용하는 <a class="mw-redirect" href="/wiki/%EC%96%B4%EB%8F%84%EB%B9%84_%ED%94%8C%EB%A0%89%EC%8A%A4" title="어도비 플렉스">플렉스</a>기술과 잘 어울리는 서버 쪽 기술이다.</p>

<h2><span class="mw-headline" id="CFML">CFML</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%96%B4%EB%8F%84%EB%B9%84_%EC%BD%9C%EB%93%9C%ED%93%A8%EC%A0%84&amp;action=edit&amp;section=3" title="부분 편집: CFML">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>콜드퓨전의 특징적인 기능 중 하나가 스크립트 언어인 CFML(ColdFusion Markup Language)이다. CFML은 <a href="/wiki/%EC%95%A1%ED%8B%B0%EB%B8%8C_%EC%84%9C%EB%B2%84_%ED%8E%98%EC%9D%B4%EC%A7%80" title="액티브 서버 페이지">ASP</a>나 <a class="mw-redirect" href="/wiki/%EC%9E%90%EB%B0%94_%EC%84%9C%EB%B2%84_%ED%8E%98%EC%9D%B4%EC%A7%80" title="자바 서버 페이지">JSP</a>,<a href="/wiki/PHP" title="PHP">PHP</a>와 기능적으로 비교되곤 하는데, 문법적인 측면에서는 <a href="/wiki/HTML" title="HTML">HTML</a>에 가까운 언어이다. "콜드퓨전"은 "CFML"을 같은 의미로 인식되곤 하지만 콜드퓨전 외에도 <a href="#.EB.8C.80.EC.B2.B4.ED.95.A0_.EC.88.98_.EC.9E.88.EB.8A.94_.EC.84.9C.EB.B2.84_.ED.99.98.EA.B2.BD">추가적인 CFML 애플리케이션 서버</a>가 제공되고 있다. CFML의 다른 기능은 <a href="/wiki/%EB%8B%B7%EB%84%B7_%ED%94%84%EB%A0%88%EC%9E%84%EC%9B%8C%ED%81%AC" title="닷넷 프레임워크">닷넷 프레임워크</a> 환경이나 이미지 처리에서 유사하거나 좀 더 강력하게 동작한다.</p>
<p><br/>
HTML의 각 요소에 대응되는 콜드퓨전 태그는 "CF"를 접두어를 추가하고 HTML에서 해석될 내용을 표기한다.</p>
<h3><span class="mw-headline" id="Hello_world_.ED.94.84.EB.A1.9C.EA.B7.B8.EB.9E.A8"><a href="/wiki/Hello_world_%ED%94%84%EB%A1%9C%EA%B7%B8%EB%9E%A8" title="Hello world 프로그램">Hello world 프로그램</a></span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%96%B4%EB%8F%84%EB%B9%84_%EC%BD%9C%EB%93%9C%ED%93%A8%EC%A0%84&amp;action=edit&amp;section=4" title="부분 편집: Hello world 프로그램">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>&lt;cfoutput&gt; 는 변수나 다른 컨텐츠를 외부로 출력하는 시작 태그이다.</p>

<h2><span class="mw-headline" id="CFScript">CFScript</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%96%B4%EB%8F%84%EB%B9%84_%EC%BD%9C%EB%93%9C%ED%93%A8%EC%A0%84&amp;action=edit&amp;section=5" title="부분 편집: CFScript">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>콜드퓨전은 CFML뿐 아니라 <a href="/wiki/%EC%95%A1%EC%85%98%EC%8A%A4%ED%81%AC%EB%A6%BD%ED%8A%B8" title="액션스크립트">액션스크립트</a>나 <a href="/wiki/%EC%9E%90%EB%B0%94%EC%8A%A4%ED%81%AC%EB%A6%BD%ED%8A%B8" title="자바스크립트">자바스크립트</a>처럼 사용할 수 있는 내장된 스크립트 언어인 CFScript도 사용할 수 있다.</p>
<p><br/>
핵심 엔진을 비롯한 기본 기능과 내장된 <a href="/wiki/%EC%8A%A4%ED%81%AC%EB%A6%BD%ED%8A%B8_%EC%96%B8%EC%96%B4" title="스크립트 언어">스크립트 언어</a>(CFScript)는 C를 기반으로 작성되었고 플러그인 모듈은 자바로, 문법은 HTML과 유사하게 만들어졌다.</p>

<h3><span class="mw-headline" id=".EC.BD.9C.EB.93.9C_.ED.93.A8.EC.A0.84_3.1">콜드 퓨전 3.1</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%96%B4%EB%8F%84%EB%B9%84_%EC%BD%9C%EB%93%9C%ED%93%A8%EC%A0%84&amp;action=edit&amp;section=8" title="부분 편집: 콜드 퓨전 3.1">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>버전 3.1에서는 <a href="/wiki/%EC%86%94%EB%9D%BC%EB%A6%AC%EC%8A%A4_(%EC%9A%B4%EC%98%81_%EC%B2%B4%EC%A0%9C)" title="솔라리스 (운영 체제)">솔라리스</a> 운영체제도 지원되기 시작했다. 콜드 퓨전 스튜디오는 화면미리보기 기능과 HTML문법검사기능이 추가되었다.</p>
<h3><span class="mw-headline" id=".EC.BD.9C.EB.93.9C.ED.93.A8.EC.A0.84_4">콜드퓨전 4</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%96%B4%EB%8F%84%EB%B9%84_%EC%BD%9C%EB%93%9C%ED%93%A8%EC%A0%84&amp;action=edit&amp;section=9" title="부분 편집: 콜드퓨전 4">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>"콜드 퓨전"의 명칭이 간단하게 "콜드퓨전"으로 변경되었다. 아마도 저온 핵융합 방식을 나타내는 Cold fusion과 구분하기 위함이기도 했다. 이때부터 콜드퓨전은 인트라넷 개발 등 해외에서 많이 사용되었다.</p>
<h3><span class="mw-headline" id=".EC.BD.9C.EB.93.9C.ED.93.A8.EC.A0.84_4.5">콜드퓨전 4.5</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%96%B4%EB%8F%84%EB%B9%84_%EC%BD%9C%EB%93%9C%ED%93%A8%EC%A0%84&amp;action=edit&amp;section=10" title="부분 편집: 콜드퓨전 4.5">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>버전 4.5부터는 자바 객체를 직접 호출하는 기능이 추가되어 시스템 명령을 실행하고 J2EE 서버와 직접 통신할 수 있게 되었다.</p>
<h3><span class="mw-headline" id=".EC.BD.9C.EB.93.9C.ED.93.A8.EC.A0.84_5">콜드퓨전 5</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%96%B4%EB%8F%84%EB%B9%84_%EC%BD%9C%EB%93%9C%ED%93%A8%EC%A0%84&amp;action=edit&amp;section=11" title="부분 편집: 콜드퓨전 5">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>앨라이어를 <a href="/wiki/%EB%A7%A4%ED%81%AC%EB%A1%9C%EB%AF%B8%EB%94%94%EC%96%B4" title="매크로미디어">매크로미디어</a>에서 인수한 이후 첫 번째 출시한 제품이다. The last to be legacy coded for a specific platform.</p>
<h3><span class="mw-headline" id=".EC.BD.9C.EB.93.9C.ED.93.A8.EC.A0.84_MX_.28.EB.B2.84.EC.A0.84_6.29">콜드퓨전 MX (버전 6)</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%96%B4%EB%8F%84%EB%B9%84_%EC%BD%9C%EB%93%9C%ED%93%A8%EC%A0%84&amp;action=edit&amp;section=12" title="부분 편집: 콜드퓨전 MX (버전 6)">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>2000년 이전부터 앨라이어는 코드네임 "Neo"를 시작했다. 이 프로젝트는 콜드퓨전 서버를 <a href="/wiki/%EC%9E%90%EB%B0%94_(%ED%94%84%EB%A1%9C%EA%B7%B8%EB%9E%98%EB%B0%8D_%EC%96%B8%EC%96%B4)" title="자바 (프로그래밍 언어)">자바</a> 기반으로 다시 작성하는 것이었다. 자바 런타임 환경에서 동작하기 때문에 이식성과 서버상 보안을 강화할 수 있었다. 지금은 어도비 라이브사이클 개발팀에 있고 당시 소프트웨어 개발책임자였던 Damon Cooper(<a class="external free" href="http://www.dcooper.org)가" rel="nofollow">http://www.dcooper.org)가</a> 프로젝트 초기 책임을 맡았다.</p>
<p>2001년 1월 16일 앨라이어는 매크로미디어의 합병소식을 전했다. 매크로미디어에서 개발팀을 이어받아 콜드퓨전 5.0을 발표했다. 버전 5까지는 "콜드퓨전"의 이름을 유지했다. 2002년 매크로미디어는 자사 브랜드를 강조하며 조금 달라진 이름의 제품으로 콜드퓨전 MX(6.0)을 발표했다. 콜드퓨전 MX는 완전하게 <a href="/wiki/%EC%9E%90%EB%B0%94_%ED%94%8C%EB%9E%AB%ED%8F%BC,_%EC%97%94%ED%84%B0%ED%94%84%EB%9D%BC%EC%9D%B4%EC%A6%88_%EC%97%90%EB%94%94%EC%85%98" title="자바 플랫폼, 엔터프라이즈 에디션">J2EE</a> 플랫폼을 기반으로 다시 만들어졌다. 콜드퓨전 MX는 또한 플래시 리모팅을 바탕으로 <a class="mw-redirect" href="/wiki/%EB%A7%A4%ED%81%AC%EB%A1%9C%EB%AF%B8%EB%94%94%EC%96%B4_%ED%94%8C%EB%9E%98%EC%8B%9C" title="매크로미디어 플래시">매크로미디어 플래시</a>와 통합된 개념으로 설계 되었다.</p>
<p>콜드퓨전 MX의 출시와 함께 CFML도 <a href="/wiki/%EA%B0%9D%EC%B2%B4_%EC%A7%80%ED%96%A5_%ED%94%84%EB%A1%9C%EA%B7%B8%EB%9E%98%EB%B0%8D" title="객체 지향 프로그래밍">객체 지향 프로그래밍</a> 기반으로 변모하게 되었다.</p>
<h3><span class="mw-headline" id=".EC.BD.9C.EB.93.9C.ED.93.A8.EC.A0.84_MX_7">콜드퓨전 MX 7</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%96%B4%EB%8F%84%EB%B9%84_%EC%BD%9C%EB%93%9C%ED%93%A8%EC%A0%84&amp;action=edit&amp;section=13" title="부분 편집: 콜드퓨전 MX 7">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>2005년 2월 7일 콜드퓨전 7.0이 출시되면서 제품명을 "매크로미디어 콜드퓨전 MX 7"으로 바꾸었다. CFMX 7에는 플래시 기반과 XForm 기반, 웹폼과 리포트 빌더가 추가되어 <a href="/wiki/PDF" title="PDF">PDF</a>뿐 아니라 FlashPaper, RTF, <a href="/wiki/%EB%A7%88%EC%9D%B4%ED%81%AC%EB%A1%9C%EC%86%8C%ED%94%84%ED%8A%B8_%EC%97%91%EC%85%80" title="마이크로소프트 엑셀">Excel</a>로 변환을 지원하였다. 어도비 PDF 출력기능을 바탕으로 어떤 HTML 페이지라도 프린트에 적합하게 변환하는 기능을 제공하게 되었다. 엔터프라이즈 에디션에는 <a href="/wiki/%EA%B2%8C%EC%9D%B4%ED%8A%B8%EC%9B%A8%EC%9D%B4" title="게이트웨이">게이트웨이</a>가 추가되어 IM, SMS, 파일변경검사(Directory Watcher), 비동기 실행 등 HTTP 요청 외의 서비스에 대해 인터랙션이 가능하게 되었다. XML에 대한 지원을 바탕으로 네이티브 스키마 검사 기능을 포함할 수 있게 되었다.</p>
<p>코드명 "Mystic"이었던 콜드퓨전 MX 7.0.2에서는 어도비 플렉스 2와 협업할 수 있는 기능이 추가되었다.</p>
<h3><span class="mw-headline" id=".EC.96.B4.EB.8F.84.EB.B9.84_.EC.BD.9C.EB.93.9C.ED.93.A8.EC.A0.84_8">어도비 콜드퓨전 8</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%96%B4%EB%8F%84%EB%B9%84_%EC%BD%9C%EB%93%9C%ED%93%A8%EC%A0%84&amp;action=edit&amp;section=14" title="부분 편집: 어도비 콜드퓨전 8">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>2007년 7월 30일 <a href="/wiki/%EC%96%B4%EB%8F%84%EB%B9%84_%EC%8B%9C%EC%8A%A4%ED%85%9C%EC%A6%88" title="어도비 시스템즈">어도비 시스템즈</a>에서는 기존 제품명에서 "MX"를 뺀 <b>콜드퓨전 8</b>을 공개했다. 코드명 "Scorpio"(황도십이궁에서 8번째 별자리로 콜드퓨전 제품의 여덟 번째 버전이라는 의미)라는 이름으로 베타 테스트를 진행하였으며 <a href="/wiki/%EC%96%B4%EB%8F%84%EB%B9%84_%EC%8B%9C%EC%8A%A4%ED%85%9C%EC%A6%88" title="어도비 시스템즈">어도비 시스템즈</a>에서 기대했던 5,000명보다 훨씬 많은 전 세계 14,000여명의 개발자가 테스터로 참여하였다. 콜드퓨전 개발팀은 보스톤과 매사추세츠, 인도의 방갈로르 개발자로 구성되었다.</p>
<p>추가된 기능으로 <a class="mw-redirect" href="/wiki/%EC%96%B4%EB%8F%84%EB%B9%84_%EC%96%B4%ED%81%AC%EB%A1%9C%EB%B1%83" title="어도비 어크로뱃">어도비 어크로뱃</a> 폼과 통합된 기능을 제공하는 CFPDFFORM 태그를 비롯해 이미지를 조작하는 기능과 <a href="/wiki/%EB%8B%B7%EB%84%B7_%ED%94%84%EB%A0%88%EC%9E%84%EC%9B%8C%ED%81%AC" title="닷넷 프레임워크">닷넷 프레임워크</a>와 통합된 기능, 이전에 매크로미디어 브리즈로 알려졌던 웹 기반 협업 솔루션인 Adobe Acrobat Connect를 이용해 동적인 프레젠테이션을 만들어낼 수 있는 CFPRESENTATION 태그가 추가되었다. 그리고 엔터프라이즈 에디션 버전에서는 서버 모니터링 기능이 내장된 콜드퓨전 관리자가 추가되었다. 콜드퓨전 8은 <a href="/wiki/%EB%A6%AC%EB%88%85%EC%8A%A4" title="리눅스">리눅스</a>, <a class="mw-redirect" href="/wiki/OS_X" title="OS X">OS X</a>, <a class="mw-redirect" href="/wiki/%EC%9C%88%EB%8F%84_%EC%84%9C%EB%B2%84_2003" title="윈도 서버 2003">윈도 서버 2003</a>등 다양한 <a href="/wiki/%EC%9A%B4%EC%98%81_%EC%B2%B4%EC%A0%9C" title="운영 체제">운영 체제</a>에서 사용할 수 있었다.</p>
<p>콜드퓨전 8에 추가된 다른 기능으로는 <a href="/wiki/Ajax" title="Ajax">Ajax</a> 위젯, 파일 압축 조작(CFZIP), <a class="mw-disambig" href="/wiki/%EB%A7%88%EC%9D%B4%ED%81%AC%EB%A1%9C%EC%86%8C%ED%94%84%ED%8A%B8_%EC%9D%B5%EC%8A%A4%EC%B2%B4%EC%9D%B8%EC%A7%80" title="마이크로소프트 익스체인지">마이크로소프트 익스체인지</a> 서버 통합기능(CFEXCHANGE), <a href="/wiki/CAPTCHA" title="CAPTCHA">자동가입방지(CAPTCHA)</a> 생성기능을 포함한 이미지 조작(CFIMAGE), 멀티 쓰레딩, 개별 애플리케이션 설정, <a class="mw-redirect mw-disambig" href="/wiki/%EC%95%A0%ED%85%80" title="애텀">Atom</a>이나 <a href="/wiki/RSS" title="RSS">RSS</a> 피드에 대한 지원, 리포팅 기능 강화, 강력한 암호화 라이브러리, 배열 및 구조 향상, 향상된 데이터베이스 인터랙션, 성능 향상, PDF 조작 및 병합 기능(CFPDF), 디버깅, <a href="/wiki/%EC%95%84%ED%8C%8C%EC%B9%98_%EB%8D%94%EB%B9%84" title="아파치 더비">아파치 더비</a>와 같은 내장 데이터베이스 지원, <a href="/wiki/ECMA%EC%8A%A4%ED%81%AC%EB%A6%BD%ED%8A%B8" title="ECMA스크립트">ECMA스크립트</a>에 좀 더 충실해진 CFSCRIPT와 같은 것들이 포함되었다.</p>
<p>콜드퓨전 애플리케이션 개발을 위해서는 주로 <a href="/wiki/%EC%96%B4%EB%8F%84%EB%B9%84_%EB%93%9C%EB%A6%BC%EC%9C%84%EB%B2%84" title="어도비 드림위버">어도비 드림위버</a> CS4나 매크로미디어 HomeSite 5.x, CFEclipse, <a class="mw-disambig" href="/wiki/%EC%9D%B4%ED%81%B4%EB%A6%BD%EC%8A%A4" title="이클립스">이클립스</a>등 다양한 도구를 사용할 수 있다. 새로운 콜드퓨전 8의 기능을 위한 패치를 제공하고 있다.</p>
<h3><span class="mw-headline" id=".EC.96.B4.EB.8F.84.EB.B9.84_.EC.BD.9C.EB.93.9C.ED.93.A8.EC.A0.84_9">어도비 콜드퓨전 9</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%96%B4%EB%8F%84%EB%B9%84_%EC%BD%9C%EB%93%9C%ED%93%A8%EC%A0%84&amp;action=edit&amp;section=15" title="부분 편집: 어도비 콜드퓨전 9">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>콜드퓨전 9(코드명 Centaur)는 2009년 10월 5일 출시되었다. CF9의 새로운 기능은 다음과 같다.</p>
<ul>
<li>사용자가 사전에 정의한 함수(User Defined Function, UDF)와 콜드퓨전 컴포넌트(CFC)를 CFScript에서 사용할 수 있다.</li>
<li>명확한 "로컬" 변수의 유효범위를 사용해 더 이상 함수의 앞부분에서 로컬 변수를 요청할 필요가 없다.</li>
<li>CFC를 위해 암시적인 게터/세터 구현.</li>
<li>"init"나 CFC에서 동일한 이름의 함수를 호출해서 암시적인 생성자를 만든다.</li>
<li>예외 처리를 위해 추가된 CFFinally 태그와 흐름 제어를 위한 CFContinue 태그.</li>
<li>하이버네이트를 기반으로 <a href="/wiki/%EA%B0%9D%EC%B2%B4_%EA%B4%80%EA%B3%84_%EB%A7%A4%ED%95%91" title="객체 관계 매핑">객체 관계 매핑</a>(Object-relational mapping,ORM) 데이터베이스 통합 지원.</li>
<li>Server.cfc 파일에 onServerStart, onServerEnd 메소드 적용.</li>
<li><a class="mw-redirect" href="/wiki/%EC%96%B4%EB%8F%84%EB%B9%84_%ED%94%8C%EB%A0%89%EC%8A%A4" title="어도비 플렉스">어도비 플렉스</a>와 어도비 에어와 긴밀한 통합 지원.</li>
<li>마이크로소프트의 주요 제품인 워드, 엑셀, 셰어포인트, 익스체인지, 파워포인트와의 통합 지원.</li>
<li>메모리 관리자나 가상 파일 시스템을 통해 메모리상에서 컨텐츠를 다룰 수 있는 기능 제공.</li>
<li>공개된 서비스에 대해 접근 제어, 보안, 서버내부 함수처리 등을 통한 보호기능 제공.</li>
</ul>
<h4><span class="mw-headline" id=".EC.96.B4.EB.8F.84.EB.B9.84_.EC.BD.9C.EB.93.9C.ED.93.A8.EC.A0.84_.EB.B9.8C.EB.8D.94">어도비 콜드퓨전 빌더</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%96%B4%EB%8F%84%EB%B9%84_%EC%BD%9C%EB%93%9C%ED%93%A8%EC%A0%84&amp;action=edit&amp;section=16" title="부분 편집: 어도비 콜드퓨전 빌더">편집</a><span class="mw-editsection-bracket">]</span></span></h4>
<p>어도비 콜드퓨전 빌더(코드명 "Bolt")는 콜드퓨전 애플리케이션을 만들기 위해 활용할 수 있는 새로운 이클립스 기반 개발 IDE이다. 현재 버전 1.0 출시되고 있으며 이후 업데이트 버전이 추가적으로 배포되었다.</p>
<p>이 도구에 포함된 기능은 다음과 같다.</p>
<ul>
<li><a href="/wiki/%EA%B0%9D%EC%B2%B4_%EA%B4%80%EA%B3%84_%EB%A7%A4%ED%95%91" title="객체 관계 매핑">객체 관계 매핑</a>(ORM) 자동 설정</li>
<li>애플리케이션 코드 생성</li>
<li>서버 관리</li>
<li>이클립스 프레임워크를 쉽게 확장</li>
<li>CFML, HTML, Javascript, CSS 구문 하이라이트 기능 지원</li>
