<p><b>어드밴스트 마이크로 디바이시스 주식회사</b>(Advanced Micro Devices, Inc.) 또는 <b>AMD</b>는 <a href="/wiki/%EB%AF%B8%EA%B5%AD" title="미국">미국</a>의 <a class="mw-redirect" href="/wiki/%EC%A7%91%EC%A0%81%ED%9A%8C%EB%A1%9C" title="집적회로">집적회로</a> 제조사로, <a href="/wiki/%EC%BA%98%EB%A6%AC%ED%8F%AC%EB%8B%88%EC%95%84_%EC%A3%BC" title="캘리포니아 주">캘리포니아 주</a> 서니베일에 위치하고 있다. <a href="/wiki/1969%EB%85%84" title="1969년">1969년</a> <a href="/wiki/%ED%8E%98%EC%96%B4%EC%B0%A8%EC%9D%BC%EB%93%9C_%EB%B0%98%EB%8F%84%EC%B2%B4" title="페어차일드 반도체">페어차일드 반도체</a> 출신들이 주축이 되어 세워진 AMD는, <a href="/wiki/%EC%9D%B8%ED%85%94" title="인텔">인텔</a>에 이어 두 번째로 큰 <a class="mw-redirect" href="/wiki/X86_%EC%95%84%ED%82%A4%ED%85%8D%EC%B2%98" title="X86 아키텍처">x86 아키텍처</a> 호환 프로세서 제조사이며 <a href="/wiki/%ED%94%8C%EB%9E%98%EC%8B%9C_%EB%A9%94%EB%AA%A8%EB%A6%AC" title="플래시 메모리">플래시 메모리</a> 분야에서도 주도적인 위치에 서 있다. 2008년 7월에 <a class="new" href="/w/index.php?title=%ED%97%A5%ED%84%B0_%EB%A3%A8%EC%9D%B4%EC%B8%A0&amp;action=edit&amp;redlink=1" title="헥터 루이츠 (없는 문서)">헥터 루이츠 박사</a>(Dr. Héctor Ruiz)로부터 <a class="new" href="/w/index.php?title=%EB%8D%94%ED%81%AC_%EB%A7%88%EC%9D%B4%EC%96%B4&amp;action=edit&amp;redlink=1" title="더크 마이어 (없는 문서)">더크 마이어</a>가 CEO 및 회장직을 이어받았다. 더크 메이어는 불도저 문제와 <a href="/wiki/%ED%80%84%EC%BB%B4" title="퀄컴">퀄컴</a>에 Aderno 매각 등 문제로 이사회에서 해임되고, 로리 리드가 새로운 CEO가 되었다. 현재 CEO 및 회장은 리사 수 박사이다.</p>
<p>AMD는 <a href="/wiki/%EC%95%A0%EC%8A%AC%EB%A1%A0" title="애슬론">애슬론</a>, <a href="/wiki/%EC%95%A0%EC%8A%AC%EB%A1%A0_64" title="애슬론 64">애슬론 64</a>, <a href="/wiki/%EC%98%B5%ED%85%8C%EB%A1%A0" title="옵테론">옵테론</a>, <a href="/wiki/%ED%8A%9C%EB%A6%AC%EC%98%A8_64" title="튜리온 64">튜리온 64</a>, <a href="/wiki/%EC%85%88%ED%94%84%EB%A1%A0" title="셈프론">셈프론</a>, <a href="/wiki/%EB%93%80%EB%A1%A0" title="듀론">듀론</a> 등의 x86-호환 프로세서로 잘 알려져 있다. 또한 AMD는 <a href="/wiki/%EC%A7%80%EC%98%A4%EB%93%9C" title="지오드">지오드</a>와 같은, 노트북에 쓰이기 쉬운 낮은 클럭의 모바일용 <a href="/wiki/%EC%A4%91%EC%95%99_%EC%B2%98%EB%A6%AC_%EC%9E%A5%EC%B9%98" title="중앙 처리 장치">CPU</a>도 제작하고 있다. <a href="/wiki/2006%EB%85%84" title="2006년">2006년</a> <a href="/wiki/6%EC%9B%94_24%EC%9D%BC" title="6월 24일">6월 24일</a>, AMD는 54억 달러에 <a class="mw-redirect" href="/wiki/ATI" title="ATI">ATI</a>를 인수 합병하기로 합의했다.</p>
<p>또, <a href="/wiki/2007%EB%85%84" title="2007년">2007년</a>에는 <a class="mw-redirect" href="/wiki/%ED%8E%98%EB%84%98" title="페넘">페넘</a>을 출시했으며, <a href="/wiki/%ED%8E%98%EB%84%98_II" title="페넘 II">페넘 II</a>, <a href="/wiki/%EC%95%A0%EC%8A%AC%EB%A1%A0_II" title="애슬론 II">애슬론 II</a> 등의 모델들이 출시하였다. <a href="/wiki/2011%EB%85%84" title="2011년">2011년</a> CPU와 GPU를 하나의 칩셋에 통합한 가속 처리 장치인 <a href="/wiki/AMD_%EA%B0%80%EC%86%8D_%EC%B2%98%EB%A6%AC_%EC%9E%A5%EC%B9%98" title="AMD 가속 처리 장치">1세대 APU 라노</a>와 새롭게 설계된 <a href="/wiki/%EB%B6%88%EB%8F%84%EC%A0%80_(%EB%A7%88%EC%9D%B4%ED%81%AC%EB%A1%9C%EC%95%84%ED%82%A4%ED%85%8D%EC%B2%98)" title="불도저 (마이크로아키텍처)">불도저</a>를 출시, <a href="/wiki/2012%EB%85%84" title="2012년">2012년</a>에는 <a href="/wiki/%EB%B6%88%EB%8F%84%EC%A0%80_(%EB%A7%88%EC%9D%B4%ED%81%AC%EB%A1%9C%EC%95%84%ED%82%A4%ED%85%8D%EC%B2%98)" title="불도저 (마이크로아키텍처)">불도저</a>에서 약간 개선된 <a href="/wiki/%ED%8C%8C%EC%9D%BC%EB%93%9C%EB%9D%BC%EC%9D%B4%EB%B2%84_(%EB%A7%88%EC%9D%B4%ED%81%AC%EB%A1%9C%EC%95%84%ED%82%A4%ED%85%8D%EC%B2%98)" title="파일드라이버 (마이크로아키텍처)">파일드라이버</a>와 <a href="/wiki/AMD_%EA%B0%80%EC%86%8D_%EC%B2%98%EB%A6%AC_%EC%9E%A5%EC%B9%98" title="AMD 가속 처리 장치">2세대 APU 트리니티</a>를 선보였다. 또 <a href="/wiki/2013%EB%85%84" title="2013년">2013년</a>에는 <a href="/wiki/AMD_%EA%B0%80%EC%86%8D_%EC%B2%98%EB%A6%AC_%EC%9E%A5%EC%B9%98" title="AMD 가속 처리 장치">2세대 APU 트리니티</a>에서 <a href="/wiki/%EC%A4%91%EC%95%99_%EC%B2%98%EB%A6%AC_%EC%9E%A5%EC%B9%98" title="중앙 처리 장치">CPU</a>와<a href="/wiki/%EA%B7%B8%EB%9E%98%ED%94%BD_%EC%B2%98%EB%A6%AC_%EC%9E%A5%EC%B9%98" title="그래픽 처리 장치">GPU</a>의 클럭을 높인 <a href="/wiki/AMD_%EA%B0%80%EC%86%8D_%EC%B2%98%EB%A6%AC_%EC%9E%A5%EC%B9%98" title="AMD 가속 처리 장치">3세대 APU 리치랜드</a>를 출시하고 <a href="/wiki/%ED%8C%8C%EC%9D%BC%EB%93%9C%EB%9D%BC%EC%9D%B4%EB%B2%84_(%EB%A7%88%EC%9D%B4%ED%81%AC%EB%A1%9C%EC%95%84%ED%82%A4%ED%85%8D%EC%B2%98)" title="파일드라이버 (마이크로아키텍처)">파일드라이버</a>기반의 기존제품에서 동작클럭만 증가시킨 <a href="/wiki/AMD_FX" title="AMD FX">FX-9XXX시리즈</a>를 출시한다. <a href="/wiki/2014%EB%85%84" title="2014년">2014년</a>에는 <a href="/wiki/AMD_%EA%B0%80%EC%86%8D_%EC%B2%98%EB%A6%AC_%EC%9E%A5%EC%B9%98" title="AMD 가속 처리 장치">3세대 APU 리치랜드</a>에서 <a href="/wiki/%EC%A4%91%EC%95%99_%EC%B2%98%EB%A6%AC_%EC%9E%A5%EC%B9%98" title="중앙 처리 장치">CPU</a>개선과 <a class="new" href="/w/index.php?title=Graphic_Core_Next&amp;action=edit&amp;redlink=1" title="Graphic Core Next (없는 문서)">GCN</a>으로의 <a href="/wiki/%EA%B7%B8%EB%9E%98%ED%94%BD_%EC%B2%98%EB%A6%AC_%EC%9E%A5%EC%B9%98" title="그래픽 처리 장치">GPU</a>세대 교체가 이루어진 <a href="/wiki/%EC%8A%A4%ED%8C%80%EB%A1%A4%EB%9F%AC_(%EB%A7%88%EC%9D%B4%ED%81%AC%EB%A1%9C%EC%95%84%ED%82%A4%ED%85%8D%EC%B2%98)" title="스팀롤러 (마이크로아키텍처)">스팀롤러</a> 아키텍처기반 <a href="/wiki/AMD_%EA%B0%80%EC%86%8D_%EC%B2%98%EB%A6%AC_%EC%9E%A5%EC%B9%98" title="AMD 가속 처리 장치">4세대 APU 카베리</a>가 출시되었다.</p>
<p>AMD는 (x86 기반) 개인용 컴퓨터 CPU 시장에서 <a href="/wiki/%EC%9D%B8%ED%85%94" title="인텔">인텔</a>의 유일한 중대 라이벌이다. <a href="/wiki/ATI_%ED%85%8C%ED%81%AC%EB%86%80%EB%A1%9C%EC%A7%80%EC%8A%A4" title="ATI 테크놀로지스">ATI</a>를 2006년 흡수 합병한 이후에 AMD와 경쟁사인 <a href="/wiki/%EC%97%94%EB%B9%84%EB%94%94%EC%95%84" title="엔비디아">엔비디아</a>는 구별된 그래픽 처리 장치(GPU) 시장을 지배해오고 있다.<sup class="reference" id="cite_ref-1"><a href="#cite_note-1">[1]</a></sup></p>

<h2><span class="mw-headline" id=".EC.97.AD.EC.82.AC">역사</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%96%B4%EB%93%9C%EB%B0%B4%EC%8A%A4%ED%8A%B8_%EB%A7%88%EC%9D%B4%ED%81%AC%EB%A1%9C_%EB%94%94%EB%B0%94%EC%9D%B4%EC%8B%9C%EC%8A%A4&amp;action=edit&amp;section=1" title="부분 편집: 역사">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>어드밴스트 마이크로 디바이시스는 1969년 5월 1일에 <a href="/wiki/%ED%8E%98%EC%96%B4%EC%B0%A8%EC%9D%BC%EB%93%9C_%EB%B0%98%EB%8F%84%EC%B2%B4" title="페어차일드 반도체">페어차일드 반도체</a>의 동료들 중 7명과 함께 <a class="mw-disambig" href="/wiki/%EC%A0%9C%EB%A6%AC_%EC%83%8C%EB%8D%94%EC%8A%A4" title="제리 샌더스">제리 샌더스</a>가 설립하였다.<sup class="reference" id="cite_ref-2"><a href="#cite_note-2">[2]</a></sup><sup class="reference" id="cite_ref-3"><a href="#cite_note-3">[3]</a></sup></p>
<p>1969년 9월, AMD는 캘리포니아 주 <a href="/wiki/%EC%83%8C%ED%83%80%ED%81%B4%EB%9E%98%EB%9D%BC_(%EC%BA%98%EB%A6%AC%ED%8F%AC%EB%8B%88%EC%95%84_%EC%A3%BC)" title="샌타클래라 (캘리포니아 주)">샌타클래라</a>의 임시 거처에서 캘리포니아 주 <a href="/wiki/%EC%84%9C%EB%8B%88%EB%B2%A0%EC%9D%BC_(%EC%BA%98%EB%A6%AC%ED%8F%AC%EB%8B%88%EC%95%84_%EC%A3%BC)" title="서니베일 (캘리포니아 주)">서니베일</a>로 이전하였다.<sup class="reference" id="cite_ref-4"><a href="#cite_note-4">[4]</a></sup> 즉시 고객 기반을 확보하기 위해 AMD는 처음에 페어차일드와 <a href="/wiki/%EB%82%B4%EC%85%94%EB%84%90_%EC%84%B8%EB%AF%B8%EC%BB%A8%EB%8D%95%ED%84%B0" title="내셔널 세미컨덕터">내셔널 세미컨덕터</a>가 설계한 마이크로칩의 <a class="new" href="/w/index.php?title=2%EC%B0%A8_%EA%B3%B5%EA%B8%89%EC%9B%90&amp;action=edit&amp;redlink=1" title="2차 공급원 (없는 문서)">2차 공급원</a>이 되었다.<sup class="reference" id="cite_ref-directory_5-0"><a href="#cite_note-directory-5">[5]</a></sup><sup class="reference" id="cite_ref-6"><a href="#cite_note-6">[6]</a></sup> AMD는 먼저 로직 칩 생산에 집중하였다.<sup class="reference" id="cite_ref-7"><a href="#cite_note-7">[7]</a></sup></p>
<p>1969년 11월 회사는 1970년에 판매를 시작한 <a href="/wiki/4%EB%B9%84%ED%8A%B8" title="4비트">4비트</a> <a href="/wiki/%EC%A7%91%EC%A0%81_%ED%9A%8C%EB%A1%9C" title="집적 회로">MSI</a> <a href="/wiki/%EC%8B%9C%ED%94%84%ED%8A%B8_%EB%A0%88%EC%A7%80%EC%8A%A4%ED%84%B0" title="시프트 레지스터">시프트 레지스터</a>인 Am9300을 제조했으며 이것이 AMD 최초의 제품이다.<sup class="reference" id="cite_ref-svh_8-0"><a href="#cite_note-svh-8">[8]</a></sup><sup class="reference" id="cite_ref-9"><a href="#cite_note-9">[9]</a></sup> 또, 1970년, AMD는 최초의 사유 제품인 Am2501 로직 카운터를 생산했으며, 이 제품은 매우 성공적이었다.<sup class="reference" id="cite_ref-lojek220_10-0"><a href="#cite_note-lojek220-10">[10]</a></sup><sup class="reference" id="cite_ref-history_11-0"><a href="#cite_note-history-11">[11]</a></sup> 1971년에 가장 많이 팔린 제품은 Am2505이며, 이는 당시 가장 빠른 <a href="/wiki/%EA%B3%B1%EC%85%88%EA%B8%B0" title="곱셈기">곱셈기</a>였다.<sup class="reference" id="cite_ref-lojek220_10-1"><a href="#cite_note-lojek220-10">[10]</a></sup><sup class="reference" id="cite_ref-rodengen41_12-0"><a href="#cite_note-rodengen41-12">[12]</a></sup></p>
<p>1971년, AMD는 <a href="/wiki/64%EB%B9%84%ED%8A%B8" title="64비트">64비트</a> 양극 RAM인 Am3101을 기점으로 <a class="mw-disambig" href="/wiki/RAM" title="RAM">RAM</a> 칩 시장에 뛰어들었다.<sup class="reference" id="cite_ref-rodengen41_12-1"><a href="#cite_note-rodengen41-12">[12]</a></sup><sup class="reference" id="cite_ref-13"><a href="#cite_note-13">[13]</a></sup> 그 해에 AMD는 상당한 양의 선형 집적 회로의 판매량을 기록했고 그 해까지 회사의 총 판매 소득은 4,600,000 달러에 달했다.<sup class="reference" id="cite_ref-lojek220_10-2"><a href="#cite_note-lojek220-10">[10]</a></sup><sup class="reference" id="cite_ref-14"><a href="#cite_note-14">[14]</a></sup></p>
<p>AMD는 1972년 9월 공개 상장했다.<sup class="reference" id="cite_ref-directory_5-1"><a href="#cite_note-directory-5">[5]</a></sup><sup class="reference" id="cite_ref-15"><a href="#cite_note-15">[15]</a></sup><sup class="reference" id="cite_ref-16"><a href="#cite_note-16">[16]</a></sup></p>
