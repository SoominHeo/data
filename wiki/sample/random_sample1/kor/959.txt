<p><b>1996년 대한민국 FA컵 (공식 대회명칭 : 제1회 FA CUP 축구대회)</b>은 <a href="/wiki/%EB%8C%80%ED%95%9C%EC%B6%95%EA%B5%AC%ED%98%91%ED%9A%8C" title="대한축구협회">대한축구협회</a>에서 주관한 컵 대회인 <a href="/wiki/FA%EC%BB%B5_(%EB%8C%80%ED%95%9C%EB%AF%BC%EA%B5%AD)" title="FA컵 (대한민국)">FA컵</a>의 원년 대회이다. 전신인 <a href="/wiki/%EC%A0%84%EA%B5%AD%EC%B6%95%EA%B5%AC%EC%84%A0%EC%88%98%EA%B6%8C%EB%8C%80%ED%9A%8C" title="전국축구선수권대회">전국축구선수권대회</a> (2001년 통합)를 포함할 시 51번째 대회이다. 프로 구단 이외에 실업, 대학, 아마추어 성인 구단이 모두 참여하여 <a href="/wiki/1996%EB%85%84" title="1996년">1996년</a> 대한민국 성인축구 최강팀을 가리기 위해 개최되었다.</p>

<h2><span class="mw-headline" id=".EB.B3.B8.EC.84.A0_.EA.B2.BD.EA.B8.B0_.EA.B2.B0.EA.B3.BC">본선 경기 결과</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=FA%EC%BB%B5_1996&amp;action=edit&amp;section=1" title="부분 편집: 본선 경기 결과">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>우승상금 3,000만 원을 두고, 본선은 <a href="/wiki/K%EB%A6%AC%EA%B7%B8" title="K리그">K리그</a> 9개 팀, <a href="/wiki/%EC%A0%84%EA%B5%AD%EC%8B%A4%EC%97%85%EC%B6%95%EA%B5%AC%EC%97%B0%EB%A7%B9%EC%A0%84" title="전국실업축구연맹전">실업축구연맹전</a> 전·후반기 우승팀(<a href="/wiki/%EC%9D%B4%EB%9E%9C%EB%93%9C_%ED%91%B8%EB%A7%88_%EC%B6%95%EA%B5%AC%EB%8B%A8" title="이랜드 푸마 축구단">이랜드</a>, <a href="/wiki/%EC%83%81%EB%AC%B4_%EC%B6%95%EA%B5%AC%EB%8B%A8" title="상무 축구단">상무</a>) 및 전반기 준우승팀(<a href="/wiki/%EA%B3%A0%EC%96%91_KB%EA%B5%AD%EB%AF%BC%EC%9D%80%ED%96%89_%EC%B6%95%EA%B5%AC%EB%8B%A8" title="고양 KB국민은행 축구단">국민은행</a>) 등 총 3개 팀, 대학축구팀 중 4개 팀이 참가하여 <a href="/wiki/%EC%8B%B1%EA%B8%80_%EC%97%98%EB%A6%AC%EB%AF%B8%EB%84%A4%EC%9D%B4%EC%85%98_%ED%86%A0%EB%84%88%EB%A8%BC%ED%8A%B8" title="싱글 엘리미네이션 토너먼트">녹아웃 토너먼트</a> 방식으로 열렸다.</p>
<h3><span class="mw-headline" id="16.EA.B0.95.EC.A0.84">16강전</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=FA%EC%BB%B5_1996&amp;action=edit&amp;section=2" title="부분 편집: 16강전">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>본선 16강전은 <a href="/wiki/1996%EB%85%84" title="1996년">1996년</a> <a href="/wiki/11%EC%9B%94_30%EC%9D%BC" title="11월 30일">11월 30일</a> 또는 <a href="/wiki/12%EC%9B%94_1%EC%9D%BC" title="12월 1일">12월 1일</a>에 치러졌다. 이 중 <a href="/wiki/K%EB%A6%AC%EA%B7%B8" title="K리그">K리그</a> 팀끼리 격돌한 <a class="mw-redirect" href="/wiki/%EC%9A%B8%EC%82%B0_%ED%98%84%EB%8C%80" title="울산 현대">울산 현대</a>와 <a class="mw-redirect" href="/wiki/%EC%95%88%EC%96%91_LG_%EC%B9%98%ED%83%80%EC%8A%A4" title="안양 LG 치타스">안양 LG 치타스</a>의 경기는 <a href="/wiki/%EC%9A%B8%EC%82%B0%EA%B3%B5%EC%84%A4%EC%9A%B4%EB%8F%99%EC%9E%A5" title="울산공설운동장">울산공설운동장</a>에서, 그 외 경기들은 <a href="/wiki/K%EB%A6%AC%EA%B7%B8" title="K리그">K리그</a> 팀의 연고지 경기장에서 치러졌다.</p>
