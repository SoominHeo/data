<h1 class="firstHeading" id="firstHeading" lang="ko">어린</h1>
<div class="mw-body-content" id="bodyContent">

<p><b>어린</b>(魚鱗, <span lang="en" xml:lang="en">fish scale</span>)은 <a href="/wiki/%EB%AC%BC%EA%B3%A0%EA%B8%B0" title="물고기">물고기</a>의 피부를 덮고 있는 <a href="/wiki/%EB%B9%84%EB%8A%98" title="비늘">비늘</a>이다. 어류는 대부분 피부가 비늘로 덮여 있으며, 비늘은 종마다 그 크기나 모양, 구조가 모두 천차만별이다. <a class="new" href="/w/index.php?title=%EC%83%88%EC%9A%B0%EA%B3%A0%EA%B8%B0&amp;action=edit&amp;redlink=1" title="새우고기 (없는 문서)">새우고기</a>나 <a href="/wiki/%EA%B1%B0%EB%B6%81%EB%B3%B5%EA%B3%BC" title="거북복과">거북복</a>처럼 단단하고 뻣뻣한 판 모양 비늘이 있는가 하면 <a href="/wiki/%EC%9E%A5%EC%96%B4" title="장어">장어</a>나 <a href="/wiki/%EC%95%84%EA%B7%80" title="아귀">아귀</a>처럼 비늘 크기가 너무 작아 보이지 않거나 아예 비늘이 없는 종도 있다.</p>
<p><a href="/wiki/%EC%97%B0%EA%B3%A8%EC%96%B4%EB%A5%98" title="연골어류">연골어류</a>(<a href="/wiki/%EC%83%81%EC%96%B4" title="상어">상어</a> 및 <a href="/wiki/%EA%B0%80%EC%98%A4%EB%A6%AC" title="가오리">가오리</a>류)는 판금 모양 비늘로 덮여 있으며 이를 <b>순린</b>(楯鱗, <span lang="en" xml:lang="en">placoid scales</span>)이라 한다. <a href="/wiki/%EA%B2%BD%EA%B3%A8%EC%96%B4%EB%A5%98" title="경골어류">경골어류</a>는 <a href="/wiki/%EC%97%B0%EC%96%B4" title="연어">연어</a>나 <a href="/wiki/%EC%9E%89%EC%96%B4" title="잉어">잉어</a> 무리의 <b>원린</b>(圓鱗, <span lang="en" xml:lang="en">cycloid scales</span>), <a href="/wiki/%EB%86%8D%EC%96%B4" title="농어">농어</a>무리의 <b>즐린</b>(櫛鱗, <span lang="en" xml:lang="en">ctenoid scales</span>), <a class="mw-redirect" href="/wiki/%EC%B2%A0%EA%B0%91%EC%83%81%EC%96%B4" title="철갑상어">철갑상어</a> 및 <a href="/wiki/%EB%8F%99%EA%B0%88%EC%B9%98" title="동갈치">동갈치</a>에게서 보이는 <b>경린</b>(硬鱗, <span lang="en" xml:lang="en">ganoid scales</span>)으로 나뉜다. 일부 어종은 어린 대신 <a class="new" href="/w/index.php?title=%EC%9D%B8%EA%B0%91&amp;action=edit&amp;redlink=1" title="인갑 (없는 문서)">인갑</a>으로 피부가 덮여 있는 것도 있고, <a href="/wiki/%EB%93%9C%EB%A0%81%ED%97%88%EB%A6%AC" title="드렁허리">드렁허리</a> 따위는 비늘이나 딱지가 없이 피부가 드러나 있다.</p>
<p>어린은 어류의 <a href="/wiki/%ED%94%BC%EB%B6%80%EA%B3%84%ED%86%B5" title="피부계통">피부계통</a>의 일부로서 <a class="mw-redirect" href="/wiki/%EC%A7%84%ED%94%BC" title="진피">진피</a>층의 <a href="/wiki/%EC%A4%91%EB%B0%B0%EC%97%BD" title="중배엽">중배엽</a>에서 생성되는 점에서 <a class="new" href="/w/index.php?title=%ED%8C%8C%EC%B6%A9%EB%A5%98_%EB%B9%84%EB%8A%98&amp;action=edit&amp;redlink=1" title="파충류 비늘 (없는 문서)">파충류 비늘</a>과 구분된다.<sup class="reference" id="cite_ref-1"><a href="#cite_note-1">[1]</a></sup> 어류의 어린 생성 <a href="/wiki/%EC%9C%A0%EC%A0%84%EC%9E%90" title="유전자">유전자</a>와 같은 유전자가 <a href="/wiki/%ED%8F%AC%EC%9C%A0%EB%A5%98" title="포유류">포유류</a>의 털과 이빨 생성에 관여한다. 연골어류의 순린은 척추동물의 이빨과 구조적 상동기관이기 때문에 피부에 돋은 이빨이라는 뜻에서 <b>피치</b>(皮齒, <span lang="en" xml:lang="en">dermal denticles</span>)라고도 한다. 경골어류의 비늘도 이빨과 비슷한 구조일 것이라는 설이 제기된 바 있지만, 아마 연골어류와는 다른 조직에서 유래하는 것으로 추측된다.<sup class="reference" id="cite_ref-2"><a href="#cite_note-2">[2]</a></sup> 대부분의 어류는 슬라임성 점액질 층이 어린 위를 덮고 있다.</p>

<h2><span class="mw-headline" id=".EC.88.9C.EB.A6.B0">순린</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%96%B4%EB%A6%B0&amp;action=edit&amp;section=1" title="부분 편집: 순린">편집</a><span class="mw-editsection-bracket">]</span></span></h2>

<p><b>순린</b>(楯鱗, <span lang="en" xml:lang="en">placoid scales</span>) 또는 방패비늘은 <a href="/wiki/%EC%83%81%EC%96%B4" title="상어">상어</a>, <a href="/wiki/%EA%B0%80%EC%98%A4%EB%A6%AC" title="가오리">가오리</a> 등 <a href="/wiki/%EC%97%B0%EA%B3%A8%EC%96%B4%EB%A5%98" title="연골어류">연골어류</a>에게서 나타난다. 구조적으로 척추동물의 <a class="mw-redirect" href="/wiki/%EC%9D%B4%EB%B9%A8" title="이빨">이빨</a>과 상동기관이기 때문에 피부에 돋은 이빨이라는 뜻에서 <b>피치</b>(皮齒, <span lang="en" xml:lang="en">dermal denticles</span>)라고도 한다. 비늘 중앙에 <a href="/wiki/%EC%B9%98%EC%88%98" title="치수">치수</a>가 있어 혈액을 공급받고 그 주위를 원뿔형으로 <a href="/wiki/%EC%83%81%EC%95%84%EC%A7%88" title="상아질">상아질</a> 층이 덮고 있다. 이것이 <a class="mw-redirect" href="/wiki/%EC%A7%84%ED%94%BC" title="진피">진피</a> 위의 직사각형 기저판 위에 올라가 있고, 그 위 가장 바깥쪽을 <a href="/wiki/%EB%B2%95%EB%9E%91%EC%A7%88" title="법랑질">법랑질</a>이 덮고 있다. 순린은 물고기가 자란다고 같이 크기가 커지지 않으며, 대신 그 수가 많아진다.</p>

<h2><span class="mw-headline" id=".EA.B2.BD.EA.B3.A8.EC.96.B4.EB.A5.98">경골어류</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%96%B4%EB%A6%B0&amp;action=edit&amp;section=2" title="부분 편집: 경골어류">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<h3><span class="mw-headline" id=".EC.9B.90.EB.A6.B0">원린</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%96%B4%EB%A6%B0&amp;action=edit&amp;section=3" title="부분 편집: 원린">편집</a><span class="mw-editsection-bracket">]</span></span></h3>

<h3><span class="mw-headline" id=".EC.A6.90.EB.A6.B0">즐린</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%96%B4%EB%A6%B0&amp;action=edit&amp;section=4" title="부분 편집: 즐린">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
