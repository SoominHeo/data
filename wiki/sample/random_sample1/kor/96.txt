<p><b>bzip2</b>는 <a href="/wiki/%EB%B2%84%EB%A1%9C%EC%9A%B0%EC%A6%88-%ED%9C%A0%EB%9F%AC_%EB%B3%80%ED%99%98" title="버로우즈-휠러 변환">버로우즈-휠러 변환</a> 기반의 <a class="new" href="/w/index.php?title=%EC%95%95%EC%B6%95_%EC%95%8C%EA%B3%A0%EB%A6%AC%EC%A6%98&amp;action=edit&amp;redlink=1" title="압축 알고리즘 (없는 문서)">압축 알고리즘</a> 및 <a href="/wiki/%EC%95%95%EC%B6%95_%EC%86%8C%ED%94%84%ED%8A%B8%EC%9B%A8%EC%96%B4" title="압축 소프트웨어">압축 소프트웨어</a>이다. 줄리안 시워드(<span lang="en" xml:lang="en">Julian Seward</span>)가 개발하였으며, <a href="/wiki/1996%EB%85%84" title="1996년">1996년</a> 7월에 0.15 버전을 처음 공개했다. 소프트웨어는 <a href="/wiki/%EC%98%A4%ED%94%88_%EC%86%8C%EC%8A%A4" title="오픈 소스">오픈 소스</a>이며, <a href="/wiki/BSD_%ED%97%88%EA%B0%80%EC%84%9C" title="BSD 허가서">BSD 허가서</a>와 비슷한 라이선스를 갖는다.</p>
<p>bzip2는 <a href="/wiki/Gzip" title="Gzip">gzip</a>와 비슷하게 파일 하나만을 압축하며, 압축한 파일에는 보통 .bz2 확장자가 붙고 따라서 <a href="/wiki/Tar_(%ED%8C%8C%EC%9D%BC_%ED%8F%AC%EB%A7%B7)" title="Tar (파일 포맷)">tar</a>와 함께 사용하는 것이 일반적이다.</p>
<h2><span class="mw-headline" id=".EC.95.95.EC.B6.95_.ED.9A.A8.EC.9C.A8">압축 효율</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Bzip2&amp;action=edit&amp;section=1" title="부분 편집: 압축 효율">원본 편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>bzip2는 <a href="/wiki/%EB%B2%84%EB%A1%9C%EC%9A%B0%EC%A6%88-%ED%9C%A0%EB%9F%AC_%EB%B3%80%ED%99%98" title="버로우즈-휠러 변환">버로우즈-휠러 변환</a>을 써서 자주 반복되는 문자열을 같은 문자열로 변환한 다음 <a class="new" href="/w/index.php?title=MTF_%EB%B3%80%ED%99%98&amp;action=edit&amp;redlink=1" title="MTF 변환 (없는 문서)">MTF 변환</a>, <a href="/wiki/%ED%97%88%ED%94%84%EB%A7%8C_%EB%B6%80%ED%98%B8%ED%99%94" title="허프만 부호화">허프만 부호화</a>를 차례대로 적용하는 구조이다. bzip2는 <a href="/wiki/Gzip" title="Gzip">gzip</a>이나 <a class="mw-disambig" href="/wiki/ZIP" title="ZIP">ZIP</a>에 비해 대체로 압축률이 좋지만 비교적 느리다.</p>
<p>bzip2의 이전 버전 bzip은 블록 정렬을 한 다음 <a href="/wiki/%EC%82%B0%EC%88%A0_%EB%B6%80%ED%98%B8%ED%99%94" title="산술 부호화">산술 부호화</a>를 사용했지만, 특허 문제 때문에 bzip2에서는 이 방식을 쓰지 않는다.</p>
