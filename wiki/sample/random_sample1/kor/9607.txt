<p>《<b>언더커버 보스</b>》(<i>Undercover Boss</i>)은 <a href="/wiki/%EC%98%81%EA%B5%AD" title="영국">영국</a> <a href="/wiki/%EC%B1%84%EB%84%90_4" title="채널 4">채널 4</a>에서 제작·방송된 대기업 <a class="mw-redirect" href="/wiki/%EC%B5%9C%EA%B3%A0_%EA%B2%BD%EC%98%81%EC%9E%90" title="최고 경영자">최고 경영자</a>(CEO)들이 자신들의 회사 일용직 사원으로 취업하여 진행되는 <a href="/wiki/%EB%AA%B0%EB%9E%98%EC%B9%B4%EB%A9%94%EB%9D%BC_(%EB%B0%A9%EC%86%A1_%EC%A0%9C%EC%9E%91_%EA%B8%B0%EB%B2%95)" title="몰래카메라 (방송 제작 기법)">몰래카메라</a> 형식의 리얼리티 프로그램이다. <a href="/wiki/%EC%B1%84%EB%84%90_4" title="채널 4">채널 4</a>를 통해 <a href="/wiki/2009%EB%85%84" title="2009년">2009년</a>부터 현재까지 방송하고 있다.<a href="/wiki/%EB%AF%B8%EA%B5%AD" title="미국">미국</a>판은 <a class="mw-redirect" href="/wiki/CBS_(%EB%AF%B8%EA%B5%AD_%EB%B0%A9%EC%86%A1%EC%82%AC)" title="CBS (미국 방송사)">CBS</a>를 통해 <a href="/wiki/2010%EB%85%84" title="2010년">2010년</a> <a href="/wiki/2%EC%9B%94_7%EC%9D%BC" title="2월 7일">2월 7일</a>부터 <a href="/wiki/4%EC%9B%94_11%EC%9D%BC" title="4월 11일">4월 11일</a>까지 시즌 1을 방송했으며, 같은 해 <a href="/wiki/9%EC%9B%94_26%EC%9D%BC" title="9월 26일">9월 26일</a>부터 시즌 2를 방송하고 있다.<br/>
<a href="/wiki/%EB%8C%80%ED%95%9C%EB%AF%BC%EA%B5%AD" title="대한민국">대한민국</a>에서는 <a href="/wiki/%EB%AC%B8%ED%99%94%EB%B0%A9%EC%86%A1" title="문화방송">MBC</a>를 통해 미국판을 <a href="/wiki/2010%EB%85%84" title="2010년">2010년</a> <a href="/wiki/7%EC%9B%94_25%EC%9D%BC" title="7월 25일">7월 25일</a>부터 <a href="/wiki/9%EC%9B%94_26%EC%9D%BC" title="9월 26일">9월 26일</a>까지 <a href="/wiki/%EB%B0%95%EB%AA%85%EC%88%98" title="박명수">박명수</a> 진행의 《<b>언더커버 보스 : 회장님은 위장 취업중</b>》이라는 명칭으로 매주 <a href="/wiki/%EC%9D%BC%EC%9A%94%EC%9D%BC" title="일요일">일요일</a> 오후 11시 35분<sup class="reference" id="cite_ref-1"><a href="#cite_note-1">[1]</a></sup>(<a href="/wiki/%ED%95%9C%EA%B5%AD_%ED%91%9C%EC%A4%80%EC%8B%9C" title="한국 표준시">KST</a>)에 방송되었다. 시즌 2는 <a href="/wiki/2010%EB%85%84" title="2010년">2010년</a> <a href="/wiki/11%EC%9B%94_21%EC%9D%BC" title="11월 21일">11월 21일</a>부터 <a href="/wiki/2011%EB%85%84" title="2011년">2011년</a> <a href="/wiki/2%EC%9B%94_27%EC%9D%BC" title="2월 27일">2월 27일</a>까지 매주 <a href="/wiki/%EC%9D%BC%EC%9A%94%EC%9D%BC" title="일요일">일요일</a> 오후 11시 50분 (<a href="/wiki/%ED%95%9C%EA%B5%AD_%ED%91%9C%EC%A4%80%EC%8B%9C" title="한국 표준시">KST</a>)에 방송되었다. 시즌 3는 당초 <a href="/wiki/2011%EB%85%84" title="2011년">2011년</a> <a href="/wiki/6%EC%9B%94" title="6월">6월</a>에 방송될 예정이었으나 <a href="/wiki/2011%EB%85%84" title="2011년">2011년</a> <a href="/wiki/12%EC%9B%94" title="12월">12월</a> 이후로 연기되었다. <a href="/wiki/%EC%A0%80%EC%9E%91%EA%B6%8C" title="저작권">저작권</a> 문제로 다시보기가 지원되지 않는다.</p>
