<p><b>얼룩바다뱀</b>(annulated sea snake)은 <a href="/wiki/%EC%BD%94%EB%B8%8C%EB%9D%BC%EA%B3%BC" title="코브라과">코브라과</a> <a class="new" href="/w/index.php?title=%EB%A8%B9%EB%8C%80%EA%B0%80%EB%A6%AC%EB%B0%94%EB%8B%A4%EB%B1%80%EC%86%8D&amp;action=edit&amp;redlink=1" title="먹대가리바다뱀속 (없는 문서)">먹대가리바다뱀속</a>의 일종이다.<sup class="reference" id="cite_ref-ITIS_2-0"><a href="#cite_note-ITIS-2">[2]</a></sup> 학명은 <b>히드로피스 키아노킨크투스</b>(<span lang="la" xml:lang="la"><i>Hydrophis cyanocinctus</i></span>). <a href="/wiki/%EC%9D%B8%EB%8F%84%EC%96%91" title="인도양">인도양</a>, <a href="/wiki/%ED%8E%98%EB%A5%B4%EC%8B%9C%EC%95%84_%EB%A7%8C" title="페르시아 만">페르시아 만</a>, <a href="/wiki/%EC%9D%B4%EB%9E%80" title="이란">이란</a>, <a href="/wiki/%ED%8C%8C%ED%82%A4%EC%8A%A4%ED%83%84" title="파키스탄">파키스탄</a>, <a href="/wiki/%EC%9D%B8%EB%8F%84" title="인도">인도</a>, <a href="/wiki/%EC%8A%A4%EB%A6%AC%EB%9E%91%EC%B9%B4" title="스리랑카">스리랑카</a>, <a href="/wiki/%EB%B0%A9%EA%B8%80%EB%9D%BC%EB%8D%B0%EC%8B%9C" title="방글라데시">방글라데시</a>, <a href="/wiki/%EB%AF%B8%EC%96%80%EB%A7%88" title="미얀마">미얀마</a>, <a class="mw-redirect" href="/wiki/%ED%83%9C%EA%B5%AD" title="태국">태국</a>, <a href="/wiki/%EB%A7%90%EB%A0%88%EC%9D%B4%EC%8B%9C%EC%95%84" title="말레이시아">말레이시아</a>, <a href="/wiki/%ED%95%84%EB%A6%AC%ED%95%80" title="필리핀">필리핀</a>, <a href="/wiki/%ED%8C%8C%EB%82%98%EC%9D%B4_%EC%84%AC" title="파나이 섬">파나이 섬</a>, <a href="/wiki/%ED%95%9C%EA%B5%AD" title="한국">한국</a>, <a href="/wiki/%EC%9D%BC%EB%B3%B8" title="일본">일본</a>, <a href="/wiki/%EC%86%94%EB%A1%9C%EB%AA%AC_%EC%A0%9C%EB%8F%84" title="솔로몬 제도">솔로몬 제도</a>, <a href="/wiki/%EB%82%A8%EC%A4%91%EA%B5%AD%ED%95%B4" title="남중국해">남중국해</a>, <a href="/wiki/%EB%8F%99%EC%A4%91%EA%B5%AD%ED%95%B4" title="동중국해">동중국해</a>, <a class="mw-redirect" href="/wiki/%EC%82%B0%EB%8F%99%EB%B0%98%EB%8F%84" title="산동반도">산동반도</a>, <a class="mw-redirect" href="/wiki/%EB%9E%B4%EC%98%A4%EB%8B%9D" title="랴오닝">랴오닝</a>, <a class="mw-redirect" href="/wiki/%EB%89%B4%EA%B8%B0%EB%8B%88" title="뉴기니">뉴기니</a>에 이르는 넓은 지역에 분포한다.</p>
