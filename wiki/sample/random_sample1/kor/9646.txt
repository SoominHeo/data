<h1 class="firstHeading" id="firstHeading" lang="ko">에덴의 동쪽 (소설)</h1>
<div class="mw-body-content" id="bodyContent">
<div id="siteSub">위키백과, 우리 모두의 백과사전.</div>
<div id="contentSub"></div>
<div class="mw-jump" id="jump-to-nav">
					이동:					<a href="#mw-head">둘러보기</a>, 					<a href="#p-search">검색</a>
</div>
<div class="mw-content-ltr" dir="ltr" id="mw-content-text" lang="ko"><script>function mfTempOpenSection(id){var block=document.getElementById("mf-section-"+id);block.className+=" open-block";block.previousSibling.className+=" open-block";}</script><p><b>에덴의 동쪽</b>(East Of Eden)은 <a href="/wiki/%EB%AF%B8%EA%B5%AD" title="미국">미국</a>의 작가 <a href="/wiki/%EC%A1%B4_%EC%8A%A4%ED%83%80%EC%9D%B8%EB%B2%A1" title="존 스타인벡">존 스타인벡</a>이 쓴 <a href="/wiki/1952%EB%85%84" title="1952년">1952년</a>의 소설이다. <a href="/wiki/%EC%BA%98%EB%A6%AC%ED%8F%AC%EB%8B%88%EC%95%84_%EC%A3%BC" title="캘리포니아 주">캘리포니아 주</a> <a class="new" href="/w/index.php?title=%EC%83%90%EB%A6%AC%EB%82%98%EC%8A%A4&amp;action=edit&amp;redlink=1" title="샐리나스 (없는 문서)">샐리나스</a>를 배경으로, <a href="/wiki/%EC%95%84%EC%9D%BC%EB%9E%9C%EB%93%9C" title="아일랜드">아일랜드</a> 이주민인 해밀튼 가와 동부에서 온 트라스크 가, 두 가문의 몇 대에 걸친 역사를 그린 작품이다. 제목은 <a href="/wiki/%EA%B5%AC%EC%95%BD%EC%84%B1%EA%B2%BD" title="구약성경">구약성경</a> <a href="/wiki/%EC%B0%BD%EC%84%B8%EA%B8%B0" title="창세기">창세기</a>에서 <a class="mw-redirect" href="/wiki/%EC%B9%B4%EC%9D%B8" title="카인">카인</a>이 동생 <a class="mw-redirect" href="/wiki/%EC%95%84%EB%B2%A8_(%EA%B5%AC%EC%95%BD)" title="아벨 (구약)">아벨</a>을 죽이고 에덴의 동쪽으로 도망쳤다는 구절에서 붙여졌다. <a href="/wiki/1955%EB%85%84" title="1955년">1955년</a> <a href="/wiki/%EC%97%98%EB%A6%AC%EC%95%84_%EC%B9%B4%EC%9E%94" title="엘리아 카잔">엘리아 카잔</a> 감독, <a href="/wiki/%EC%A0%9C%EC%9E%84%EC%8A%A4_%EB%94%98" title="제임스 딘">제임스 딘</a> 주연의 영화로도 제작되었다.</p>
