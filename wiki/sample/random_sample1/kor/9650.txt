<h2><span class="mw-headline" id=".EA.B0.9C.EC.9A.94">개요</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%97%90%EB%8F%84%EA%B0%80%EC%99%80_%EC%BD%94%EB%82%9C&amp;action=edit&amp;section=1" title="부분 편집: 개요">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>「일본 경찰의 구세주」, 「헤이세이(平成)시대의 <a href="/wiki/%EC%85%9C%EB%A1%9D_%ED%99%88%EC%A6%88" title="셜록 홈즈">홈즈</a>」라고 불리며, 동쪽의 고교생 탐정으로 불리던 <b>쿠도 신이치</b>는 <a href="/wiki/%EA%B2%80%EC%9D%80_%EC%A1%B0%EC%A7%81" title="검은 조직">검은 조직</a>에 의해 아직 개발 중인 독약 <a href="/wiki/APTX4869" title="APTX4869">APTX4869</a>를 먹게 되어 어린 아이로 퇴화 하였다. 어려서부터 알고 지내던 이웃 아가사 히로시 박사집에 간 신이치는 모리 란의 아버지이며 탐정인 <a href="/wiki/%EB%AA%A8%EB%A6%AC_%EC%BD%94%EA%B3%A0%EB%A1%9C" title="모리 코고로">모리 코고로</a>의 사무실에서 생활하며 사건을 맡다보면 검은조직의 내막을 밝혀낼수도 있을 것이라는 박사의 충고를 받아들여, 어려서부터 알고 지내던 <a href="/wiki/%EB%AA%A8%EB%A6%AC_%EB%9E%80" title="모리 란">모리 란</a>의 집에서 테이탄 초등학교 1학년 B반을 다니고, 코난으로 정체를 숨긴 채 함께 지내고 있다. 여러가지 사건을 보기 좋게 해결하면서 <a href="/wiki/%ED%95%98%EC%9D%B4%EB%B0%94%EB%9D%BC_%EC%95%84%EC%9D%B4" title="하이바라 아이">하이바라 아이</a>나 <a href="/wiki/%ED%95%AB%ED%86%A0%EB%A6%AC_%ED%97%A4%EC%9D%B4%EC%A7%80" title="핫토리 헤이지">핫토리 헤이지</a>등의 도움을 받아 <a href="/wiki/%EA%B2%80%EC%9D%80_%EC%A1%B0%EC%A7%81" title="검은 조직">검은 조직</a>의 음모를 쫓고 있다.</p>
<p><a href="/wiki/%EA%B0%90%EA%B8%B0" title="감기">감기</a>가 걸린 상태에서, <a class="mw-redirect" href="/wiki/%EB%B9%BC%EA%B0%88" title="빼갈">바이칼</a>을 마시면 일시적으로 몸이 원래대로 돌아온다는 내용이 극장판을 포함해 두 번 나왔었다. 그리고 하이바라가 바이칼의 성분을 참고해 만든 해독제로 원래대로 돌아간 적도 있다. 드라마판 제2기에서는 바이칼과 다른 술을 혼합하여 만든 <a href="/wiki/%EC%BC%80%EC%9D%B4%ED%81%AC" title="케이크">케이크</a>를 우연히 먹고 원래대로 돌아간 적도 있다. 하지만 몸에서 오는 느낌의 부담감은 크고, 감기에 걸린 상태나 마찬가지여서 해독제의 효과가 떨어져 코난이 되려는 시간이 다가오면 호흡이 거칠어지고, 가슴을 세게 누르는 듯한 압박감이 밀려온다. 이는 드라마나 애니메이션, 만화에서도 동일하다.</p>
<p>생일은 <a href="/wiki/5%EC%9B%94_4%EC%9D%BC" title="5월 4일">5월 4일</a>. 아버지는 <a href="/wiki/%EB%AA%85%ED%83%90%EC%A0%95_%EC%BD%94%EB%82%9C%EC%9D%98_%EB%93%B1%EC%9E%A5%EC%9D%B8%EB%AC%BC_%EB%AA%A9%EB%A1%9D#.EC.BF.A0.EB.8F.84_.EC.9C.A0.EC.82.AC.EC.BF.A0" title="명탐정 코난의 등장인물 목록">쿠도 유사쿠</a>이며 어머니는 <a href="/wiki/%EB%AA%85%ED%83%90%EC%A0%95_%EC%BD%94%EB%82%9C%EC%9D%98_%EB%93%B1%EC%9E%A5%EC%9D%B8%EB%AC%BC_%EB%AA%A9%EB%A1%9D#.EC.BF.A0.EB.8F.84_.EC.9C.A0.ED.82.A4.EC.BD.94" title="명탐정 코난의 등장인물 목록">쿠도 유키코</a>이다.</p>
<h2><span class="mw-headline" id=".EC.BD.94.EB.82.9C.EC.9D.98_.EC.A0.95.EC.B2.B4">코난의 정체</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%97%90%EB%8F%84%EA%B0%80%EC%99%80_%EC%BD%94%EB%82%9C&amp;action=edit&amp;section=2" title="부분 편집: 코난의 정체">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>코난이 쿠도 신이치라는 것을 아는 인물은 <a href="/wiki/%EC%95%84%EA%B0%80%EC%82%AC_%ED%9E%88%EB%A1%9C%EC%8B%9C" title="아가사 히로시">아가사 히로시</a>, <a class="mw-redirect" href="/wiki/%EB%AF%B8%EC%95%BC%EB%85%B8_%EC%95%84%EC%BC%80%EB%AF%B8" title="미야노 아케미">미야노 아케미</a> (<a class="mw-redirect" href="/wiki/%EA%B2%80%EC%9D%80%EC%A1%B0%EC%A7%81" title="검은조직">검은조직</a>의 멤버 진의 총격 직후에 사망), <a class="mw-redirect" href="/wiki/%EC%BF%A0%EB%8F%84_%EC%9C%A0%EC%82%AC%EC%BF%A0" title="쿠도 유사쿠">쿠도 유사쿠</a> (아버지), <a class="mw-redirect" href="/wiki/%EC%BF%A0%EB%8F%84_%EC%9C%A0%ED%82%A4%EC%BD%94" title="쿠도 유키코">쿠도 유키코</a> (어머니), <a href="/wiki/%ED%95%AB%ED%86%A0%EB%A6%AC_%ED%97%A4%EC%9D%B4%EC%A7%80" title="핫토리 헤이지">핫토리 헤이지</a>, <a href="/wiki/%ED%95%98%EC%9D%B4%EB%B0%94%EB%9D%BC_%EC%95%84%EC%9D%B4" title="하이바라 아이">하이바라 아이</a>, <a class="mw-redirect" href="/wiki/%EA%B2%80%EC%9D%80%EC%A1%B0%EC%A7%81" title="검은조직">베르무트</a>, <a class="mw-redirect" href="/wiki/%ED%98%BC%EB%8F%84_%EC%97%90%EC%9D%B4%EC%8A%A4%EC%BC%80" title="혼도 에이스케">혼도 에이스케</a>, <a href="/wiki/FBI_(%EB%AA%85%ED%83%90%EC%A0%95_%EC%BD%94%EB%82%9C)" title="FBI (명탐정 코난)">아카이 슈이치</a>이다. 또한 극장판이나 특별판까지 포함시킬 경우 <a class="mw-redirect" href="/wiki/%EA%B4%B4%EB%8F%84%ED%82%A4%EB%93%9C" title="괴도키드">괴도키드</a>, 극장판 6기 <a class="mw-redirect" href="/wiki/%EB%AA%85%ED%83%90%EC%A0%95_%EC%BD%94%EB%82%9C_%EA%B7%B9%EC%9E%A5%ED%8C%90_6%EA%B8%B0_-_%EB%B2%A0%EC%9D%B4%EC%BB%A4%EA%B0%80%EC%9D%98_%EB%A7%9D%EB%A0%B9" title="명탐정 코난 극장판 6기 - 베이커가의 망령">명탐정 코난 극장판 6기 - 베이커가의 망령</a>에서 등장한 히로키 (한국명: 전우, Noah's Ark 노아의 방주 직후에 사망), 극장판 13기 <a class="mw-redirect" href="/wiki/%EB%AA%85%ED%83%90%EC%A0%95_%EC%BD%94%EB%82%9C_%EA%B7%B9%EC%9E%A5%ED%8C%90_13%EA%B8%B0_-_%EC%B9%A0%ED%9D%91%EC%9D%98_%EC%B6%94%EC%A0%81%EC%9E%90" title="명탐정 코난 극장판 13기 - 칠흑의 추적자">명탐정 코난 극장판 13기 - 칠흑의 추적자</a>에서 등장한 <a class="mw-redirect" href="/wiki/%EA%B2%80%EC%9D%80%EC%A1%B0%EC%A7%81" title="검은조직">검은조직</a>의 멤버인 <a class="mw-redirect mw-disambig" href="/wiki/%EC%95%84%EC%9D%B4%EB%A6%AC%EC%89%AC" title="아이리쉬">아이리쉬</a> (극 중에서 검은조직 키안티 (캔티)의 총격에 사망) 그리고 <a href="/wiki/%EB%A3%A8%ED%8C%A1_3%EC%84%B8" title="루팡 3세">루팡 3세</a>도 알고 있다. (<a class="mw-redirect" href="/wiki/%EB%A3%A8%ED%8C%A13%EC%84%B8_vs_%EB%AA%85%ED%83%90%EC%A0%95_%EC%BD%94%EB%82%9C" title="루팡3세 vs 명탐정 코난">루팡3세 vs 명탐정 코난</a> 참조).</p>
<ul>
<li>극장판 <a class="mw-redirect" href="/wiki/%EB%A3%A8%ED%8C%A13%EC%84%B8_vs_%EB%AA%85%ED%83%90%EC%A0%95_%EC%BD%94%EB%82%9C_THE_MOVIE" title="루팡3세 vs 명탐정 코난 THE MOVIE">루팡3세 vs 명탐정 코난 THE MOVIE</a> 편에서는 앨런 스미시라는 사람이 죽기 전에 정체가 뭐냐? 라고 묻자 고등학생 쿠도 신이치 (남도일)라고 답하기는 했으나, 앨런은 쿠도 신이치 (남도일)가 어떤 인물인지 모르므로 제외했다.</li>
</ul>
<p><a href="/wiki/FBI_(%EB%AA%85%ED%83%90%EC%A0%95_%EC%BD%94%EB%82%9C)" title="FBI (명탐정 코난)">FBI (명탐정 코난)</a> 대원인 <a class="mw-redirect" href="/wiki/%EC%A1%B0%EB%94%94_%EC%8A%A4%ED%84%B8%EB%A7%81" title="조디 스털링">조디 스털링</a>은 코난의 정체를 모르지만 그들 앞에서 무리한 연기를 한 적이 없기 때문에 약간 들켜도 상관없다고 생각하는 모습을 볼 수 있다.</p>
<p>또 작아진 이후에 <a href="/wiki/%EB%AA%A8%EB%A6%AC_%EB%9E%80" title="모리 란">모리 란</a>에게 자신의 정체를 밝히려고 몇 번 시도했지만 그럴 때마다 방해받는다. 혼도 에이스케 이외에는 스스로 정체를 밝히려고 하지 않는다.</p>
<h2><span class="mw-headline" id=".EC.B6.94.EB.A6.AC.EB.B2.95">추리법</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%97%90%EB%8F%84%EA%B0%80%EC%99%80_%EC%BD%94%EB%82%9C&amp;action=edit&amp;section=3" title="부분 편집: 추리법">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>주로 셜록 홈즈와 왓슨 박사를 염두에 두고 있으며, 증거품 수집 및 파기와 거기서 연관된 사건 개요의 추궁이 추리의 중심이다. 특히 증거품에 관한 잡다하고 상세한 지식에서 나온 설명이나 지적이 자주 나타나는 까닭에 주변 사람들을 놀라게 하는 경우가 많다. 한편으로는 범죄자 및 관계자의 심리 추측도 잘 하지만, 그것은 어디까지나 사건 추리의 보조 수단이나 관계자의 말에 대한 추측 및 의문에 한정되어서 광범위한 관계자의 인물상이나 성장 환경에까지 관련된 활동은 하지 않는 경우가 많다. 다만 신빙성 없는 정보나 증거에도 의미를 부여하여 사건과의 관계성을 밝히려는 무리수도 가끔 있다.</p>
<h2><span class="mw-headline" id=".EB.AA.85.EB.8C.80.EC.82.AC">명대사</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%97%90%EB%8F%84%EA%B0%80%EC%99%80_%EC%BD%94%EB%82%9C&amp;action=edit&amp;section=4" title="부분 편집: 명대사">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>만화에서 명대사는 “내 이름은 코난, 탐정이죠…(江戸川コナン…探偵さ…)”였고, TV애니메이션과 극장판에서는 “진실은 언제나 하나!(真実はいつもひとつ)”이다. 단 한 번 345화에서 코난, 탐정이다! (江戸川コナン…探偵だ!) 라는 멋진 대사를 날려주기도 했다.</p>
<p>또한 애니메이션 오프닝에서는 매회 나레이션이 있으며 회를 거듭할수록 발전되어 나온다. 참고로 어느 오프닝 나레이션에서는 "끊어진 하나의 진실을 간파하는 겉모습은 아이, 두뇌는 어른 그 이름은 명탐정 코난(たった一つの真実見抜く、見た目は子供、頭脳は大人、その名は名探偵コナン!)"이라는 대사가 나온다.</p>
<p>극장판에서는 여러 버전으로 나레이션이 나오며 매 나레이션마다 끝날 때 "몸은 작아졌어도 두뇌는 같은, 미궁이 없는 명탐정. 진실은 언제나 하나.(小さくなっても頭脳は同じ 迷宮なしの名探偵 真実はいつもひとつ! )"라는 대사가 나온다.</p>
<p>나레이션 외 여러 장면을 통해 명대사 또는 명언이 등장하며 신이치로서 말하는 대사의 경우, 소노코가 시기할 정도다. 참고로 명탐정 코난은 사랑하는 이들의 취향에 따라서 명대사의 범위가 다양하게 설정될 수 있다.</p>

<p>쿠도 신이치는 테이탄 <a href="/wiki/%EA%B3%A0%EB%93%B1%ED%95%99%EA%B5%90" title="고등학교">고등학교</a> 2학년 B반 추리 광팬으로 그 중 <a href="/wiki/%EC%85%9C%EB%A1%9D_%ED%99%88%EC%A6%88" title="셜록 홈즈">셜록 홈즈</a>의 팬이다. 뛰어난 실력의 고교생 탐정으로 세간에 알려져 있으며 신장은 174cm 이다. 체중 58kg 주소는 <a href="/wiki/%EB%8F%84%EC%BF%84_%EB%8F%84" title="도쿄 도">도쿄 도</a> 베이커 2번가 21번지이다. <a class="new" href="/w/index.php?title=%EC%9B%90%EC%96%B4&amp;action=edit&amp;redlink=1" title="원어 (없는 문서)">원어</a> 수준으로 <a href="/wiki/%EC%98%81%EC%96%B4" title="영어">영어</a>를 할 수 있다. <sup class="reference" id="cite_ref-1"><a href="#cite_note-1">[1]</a></sup></p>
<p>고등학생이 되고 나서 부모님이 사는 샌프란시스코로 향하는 비행기 안에서 발생한 살인 사건을 해결한 것을 계기로 경찰에게 주목을 받게 된다.</p>
<p>성격은 냉정하게 보이지만, 실은 오기 있고 질투심이 깊다. 특히 란과 관련된 일이면 어린 아이처럼 무모한 행동을 하는 것도 적지 않다. 하지만 어떤 일이 있더라도 단념하지 않고 강한 의지를 가지고 있다. 또한 자신과 관련되지 않는 사건이라도 흥미 위주로 관여하는 안 좋은 버릇을 가지고 있다.</p>
<p>그러다가 몸이 작아져서 코난이 된 후에는 키는 95cm 체중은 18kg으로 줄어들었다.</p>
<p>테이탄 <a href="/wiki/%EC%B4%88%EB%93%B1%ED%95%99%EA%B5%90" title="초등학교">초등학교</a> 1학년 B반에 재학하고 있으며 <a href="/wiki/%EB%AA%A8%EB%A6%AC_%EB%9E%80" title="모리 란">모리 란</a>의 집에서 얹혀산다 얼굴을 알고 있는 란에게 의심을 가지지 않도록 하기 위해서 신이치의 아버지인 <a class="mw-redirect" href="/wiki/%EC%BF%A0%EB%8F%84_%EC%9C%A0%EC%82%AC%EC%BF%A0" title="쿠도 유사쿠">쿠도 유사쿠</a>의 안경을 쓴 것을 시작으로 일상적으로 안경을 쓰게 되었다. 하지만 시력 역시 신이치 때의 그대로이므로 도수가 없다. 이후 체력이 아이 수준에 맞게 떨어지지만 성격이나 버릇 그리고 두뇌나 기술은 변함이 없기 때문에 그 두뇌로 수많은 사건을 해결하고 있<a class="mw-redirect" href="/wiki/%EC%BF%A0%EB%8F%84_%EC%9C%A0%EC%82%AC%EC%BF%A0" title="쿠도 유사쿠">다만</a>한 어린 아이의 신체이기 때문에 <a class="mw-redirect" href="/wiki/%ED%83%90%EC%A0%95" title="탐정">탐정</a>으로서 해야 할 일에 제한이 되기 때문에 아가사 히로시가 만든 발명품인 시계형 마취총이나 나비 넥타이형 음성 변조기 추적장치가 달려 있는 안경, 귀걸이형 휴대전화 <a class="mw-redirect" href="/wiki/%ED%83%90%EC%A0%95" title="탐정">에</a> 의적으로 부족한 부분을 보완하고 있다.</p>
<p>탐정이기 때문에 작아져도 사건이 터지면 본능적으로 수사에 돌입하고 이것을 못마땅히 여기는 모리 코고로에게 매번 맞는다. 대부분 알밤 정도로 그치지만 극장판 5기 천국으로의 카운트 다운에서는 모리 코고로의 추리가 틀렸음을 지적했다가 모리에게 멱살을 잡혀 던져지기까지 했다.</p>
<p>몸이 작아지고 나서 코난의 모습으로는 사건의 진상을 말해도 아무도 상대해 주지 않기 때문에 코난의 사정을 모르는 사람들의 경우는 시계형 마취총으로 마취시킨 후 나비 넥타이형 변성기로 그 사람의 목소리를 낸다. 코난의 정체를 알고 있는 사람의 경우에는 그 사람에게 부탁을 하여 복화술로 진상을 이야기를 한다. 기본적으로 <a href="/wiki/%EB%AA%A8%EB%A6%AC_%EC%BD%94%EA%B3%A0%EB%A1%9C" title="모리 코고로">모리 코고로</a>를 마취시켜 그의 목소리로 사건을 해결을 하며 코고로가 없을 경우에는 <a href="/wiki/%EC%8A%A4%EC%A6%88%ED%82%A4_%EC%86%8C%EB%85%B8%EC%BD%94" title="스즈키 소노코">스즈키 소노코</a>, <a class="new" href="/w/index.php?title=%EC%95%BC%EB%A7%88%EB%AC%B4%EB%9D%BC_%EB%AF%B8%EC%82%AC%EC%98%A4_%EA%B2%BD%EB%B6%80&amp;action=edit&amp;redlink=1" title="야마무라 미사오 경부 (없는 문서)">야마무라 미사오 경부</a>를 마취총으로 재워서 목소리를 빌리거나 <a href="/wiki/%EC%95%84%EA%B0%80%EC%82%AC_%ED%9E%88%EB%A1%9C%EC%8B%9C" title="아가사 히로시">아가사 히로시</a> 박사에게 부탁하여 복화술을 쓴다.(혼도 에이스케의 정체를 몰랐을 때는 그의 앞에서 이런 방법으로 추리하는 것을 꺼려했다.)</p>
<p>어떨 때에는 코고로나 <a href="/wiki/%EB%A9%94%EA%B5%AC%EB%A0%88_%EC%A5%AC%EC%A3%A0" title="메구레 쥬죠">메구레 쥬죠</a>, <a class="mw-redirect" href="/wiki/%EC%9A%94%EC%BD%94%EB%AF%B8%EC%A1%B0_%EC%82%B0%EA%B3%A0" title="요코미조 산고">요코미조 산고</a>에게 힌트를 주려고 하는 경우도 있다. 또 전화로 란에게 사건의 진상을 이야기해, 란에게 전해 달라고 하는 경우도 있고, 엄마인 유키코에게 진상을 말하고 이야기한 적도 있다. 그리고 휴대폰 메일로 쿠도 신이치로서 진상을 알리거나 모리 코고로로서 알리거나, 에도가와 코난으로서 알린 경우가 있다. 스스로 진상을 추리해서 직접 말할 때도 있다. 이 경우는 범인과 코난이 1 대 1로 대립되는 경우이다. 특히 검은 조직 간부이거나 괴도 키드와 맞서는 경우나 극장판에서는 거의 매번 그런다.</p>
<p><a href="/wiki/%EC%86%8C%EB%85%84_%ED%83%90%EC%A0%95%EB%8B%A8_(%EB%AA%85%ED%83%90%EC%A0%95_%EC%BD%94%EB%82%9C)" title="소년 탐정단 (명탐정 코난)">소년 탐정단 (명탐정 코난)</a>이 있을 때는 그들에게 속임수의 힌트가 되는 말을 하게 하는 형식으로 도움을 받는다.</p>
<p>자신이 한 추리를 이야기할 때 코난인 상태에서 이야기를 하면 주변의 어른들에게 관심을 끌기 위해서 "어라라~?이상하네~?(あれれ～?おっかしいぞぉ～?)라고 말을 한다. 또한 어른들도 모르는 지식을 무심코 이야기해서 주변에서 이상하게 여길 경우 "신이치에게서 들었다. "라고 하거나 "텔레비전을 통해 알았다. "는 식으로 넘어간다. 한편 너무 적극적으로 추리에 가담할 때에는 란에게 신이치라고 의심받을 빌미를 제공하기도 한다.</p>
<p>피아노 소나타 월광 살인 사건에서는 범인의 자살을 저지하지 못하고 죽음에 이르게 한다. 이 사건은 코난에게 마음의 상처가 되었으며 그것을 회상할 때는 슬픈 눈으로 "추리로 범인을 추적하여 죽음으로 이르게 하는 것은 살인자와 다를 게 없다(推理で犯人を追い詰めて死なせてしまっては、殺人者と変わらない」との事)."라며 마음속으로 독백을 한다.</p>
<p>소꿉 친구인 <a href="/wiki/%EB%AA%A8%EB%A6%AC_%EB%9E%80" title="모리 란">모리 란</a>과 소꿉친구나 연애 관계이지만 솔직하지는 못하다. 코난이 신이치라고는 꿈에도 생각하지 않은 란이 「정말 좋아해」(<span style="font-size: smaller;"><a href="/wiki/%EC%9D%BC%EB%B3%B8%EC%96%B4" title="일본어">일본어</a>:</span> <span lang="ja">だーい好き</span> <small>다이스키<sup>[<a href="/wiki/%EC%9C%84%ED%82%A4%EB%B0%B1%EA%B3%BC:%EC%9D%BC%EB%B3%B8%EC%96%B4%EC%9D%98_%ED%95%9C%EA%B8%80_%ED%91%9C%EA%B8%B0" title="위키백과:일본어의 한글 표기">*</a>]</sup></small>)라고 자신에게 고백했고때 문에 원래대로 돌아가면 란에게 고백하려고 했다. 일시적으로 신이치의 몸으로 돌아왔을때, 자신의 아빠가 엄마에게 프로포즈한 같은 장소에서 본인도 고백하려고 했으나, 사건에 연루되어서 어떻게든 해결됐지만 해결과 동시에 코난으로 돌아가 버려 어쩔 수 없이 고백도 못 하게 되었다. 마음속으론 란을 그리워한다. 또한 자신이 다른 사람에 비해 란에게 솔직만큼은 못한 편인 것을 알고 있고, 자신이 더 솔직해졌으면 좋겠다고 생각한다. 란의 눈물을 싫어하며, 설령 란이 자신을 더 이상 좋아해 주지 않게 된다고 해도 자신은 란이 슬픈 것은 싫다고(애니메이션 270화) 이야기하는 것을 보아 란을 무척 생각하고 있다.</p>
<p>항상 코난이 아닌 신이치인 채로 란에게 좋아한다는 고백을 하길 원했고, '홈즈의 묵시록' 에피소드에서 란에게 "넌 성가신 사건이야, 쓸데없는 감정이 마구 섞여서 설령 내가 홈즈라고 해도 무리일 거야. 좋아하는 여자의 마음을 정확하게 알아내는 건 말야. 러브는 0(Love is Zero)이라고? 웃기지 마! 0은 모든 것의 시작. 거기서 시작하지 않으면 아무것도 생기지 않고, 달성할 수도 없어!"라는 대사와 함께 고백했다.</p>
<p>초등학교 1학년이었을 때는 부끄러워하여 란에게 "모리(毛利)"라고 불렀으며 자신에게도 "쿠도 군(工藤くん)"이라고 부르게 한 시기가 있었지만, 유치원에서의 한 사건을 계기로 서로의 이름을 부르게 되었고, 지금까지 "모리"가 아닌 "란"으로 부르고 있다. 동급생인 스즈키 소노코 역시 "소노코"라고 칭하고 있지만, 직접적으로 부른다기보다는 코난인 상태에서 속마음으로 '소노코 저 자식...'등의 말을 하거나, '소노코 녀석이랑 가는 거야?'와 같은 간접적인 어투를 더 많이 사용한다. 또한 위의 사건을 계기로 항상 등하교를 같이 하게 되어, 학교 친구들로부터 등교 시 교실에 들어서면 '부부 등장~'과 같은 놀림을 매번 받는다.</p>
<p>란에게는 "귀찮은 사건이 있기 때문에 해결할 때까지는 돌아갈 수 없다.(やっかいな事件に関っているから解決するまで帰れない)"라고 하지만, 란이 걱정하지 않게 가끔 나비 넥타이형 음성 변조기로 신이치의 목소리를 내어 란에게 전화를 하고 있다.</p>
<p><a href="/wiki/%EC%B6%95%EA%B5%AC" title="축구">축구</a>를 좋아하여, 중학교 때까지 축구부에 소속해 있었다. 그 기술은 <a class="mw-redirect" href="/wiki/J%EB%A6%AC%EA%B7%B8_%EB%94%94%EB%B9%84%EC%A0%84_1" title="J리그 디비전 1">J리그</a>에 스카우트될 만한 실력이지만, 신이치는 어디까지나 운동신경을 기르기 위한 것이기 때문에 그 뒤로 축구를 그만두었다. 란은 만약 계속 축구를 했다면, 축구 스타가 되어 있을 것이라고 한다. 이 때문에 범인이 도주하려고 할 때는 발 밑에 떨어져 있는 것을 축구 공처럼 발로 차서 상대를 제압하거나 움직임을 멈추기도 한다. 좋아하는 <a href="/wiki/%EC%B6%95%EA%B5%AC" title="축구">축구</a> 선수는 레이 커티스이다. 소년 탐정단과 놀 때는 야구를 많이 하지만, 본인은 축구를 하고 싶기 때문에 불만을 가지고 있다.(종종 야구 할 때 발을 사용하기도 한다.)</p>
<p><a href="/wiki/%EC%85%9C%EB%A1%9D_%ED%99%88%EC%A6%88" title="셜록 홈즈">셜록 홈즈</a>의 광팬으로 그를 "세계 최고의 탐정"으로 평가한다. 가장 좋아하는 말은 "불가능한 것을 제외하고 남는 것이 아무리 믿을 수 없는 것 일지라도 그것이 진실이다.(<i>When you have eliminated the impossible, whatever remains, however improbable, must be the truth.</i>)"라는 홈즈의 추리론이다.</p>
<p>자신의 취향에 맞는 이야기, 특히 홈즈 관련 이야기가 한번 나오면 일방적으로 자기 이야기만 해 버린다. 첫 화에서도 모처럼 유원지에 놀러 갔음에도 불구하고, 홈즈 이야기만 하여 란이 섭섭해한 적도 있다. 그 때문에 란에게서 "추리광(推理オタク)" 소리를 듣는다.</p>
<p>엄청난 음치로, 신이치에게 음악 수업은 골칫거리이다. 이것은 코난이 되어서도 변하지 않으며 그것 때문에 란에게 정체를 들킬 뻔한 일까지 있었다. 그러나 음감으로만 따지면 란보다 훨씬 뛰어나 절대 음감을 가지고 있다. 이에 대해 일본의 음향 연구소 소장인 <a class="new" href="/w/index.php?title=%EC%8A%A4%EC%A6%88%ED%82%A4_%EB%A7%88%EC%93%B0%EB%AF%B8&amp;action=edit&amp;redlink=1" title="스즈키 마쓰미 (없는 문서)">스즈키 마쓰미</a>(<a class="extiw" href="https://ja.wikipedia.org/wiki/%E9%88%B4%E6%9C%A8%E6%9D%BE%E7%BE%8E" title="ja:鈴木松美">鈴木松美</a>)<sup class="reference" id="cite_ref-2"><a href="#cite_note-2">[2]</a></sup>는 “이 경우는 소리를 능숙하게 낼 수 없는 <a href="/wiki/%EC%9D%8C%EC%B9%98" title="음치">음치</a>이며 소리는 확실히 들을 수 있는 <a href="/wiki/%EC%9D%8C%EC%B9%98" title="음치">음치</a>다.”라고 하였다. 이것은 '스트라디바리우스의 불협화음(전주곡), (후주곡)'에서 알 수 있다. 또한 홈즈의 영향을 받아 바이올린도 배우기도 하였으며 잘 연주할 수 있다. 소리는 확실히 들을 수 있는<a href="/wiki/%EC%9D%8C%EC%B9%98" title="음치">음치</a>인 덕에 블랙 임팩트 조직의 손이 닿는 순간 편에서는 도청기로 들리는 소리만 듣고도 수평 대향 엔진이 내는 소리임을 맞히고 차종까지 맞히는 능력을 선보였다.</p>
<p>연예계에는 별로 흥미가 없는 것 같지만 어떤 사건을 계기로 알게 된 <a class="new" href="/w/index.php?title=%ED%88%AC_%EB%AF%B9%EC%8A%A4&amp;action=edit&amp;redlink=1" title="투 믹스 (없는 문서)">투 믹스</a>의 <a class="mw-redirect" href="/wiki/%EB%8B%A4%EC%B9%B4%EC%95%BC%EB%A7%88_%EB%AF%B8%EB%82%98%EB%AF%B8" title="다카야마 미나미">다카야마 미나미</a><sup class="reference" id="cite_ref-3"><a href="#cite_note-3">[3]</a></sup>로부터 CD를 받고 나서는, 노래방에 갈 때마다 그 노래를 부르고 있다고 한다.</p>
<p>신이치인 상태에서 자신을 지칭할 때는 "나(<b>オレ</b>)"이나 어른을 상대로 말하거나 추리를 할 때는 "저(僕)"로 지칭한다. 또한 코난인 상태에서 자신을 지칭할 때는 "나(<b>オレ</b>)"이나 란 등 아이 행세를 하지 않으면 안 되는 상대에게는 "저(<b>僕</b>)"라고 지칭한다. 자주 하는 말은 "바보(バーロ)"이다. 기본적으로 <a href="/wiki/%ED%91%9C%EC%A4%80%EC%96%B4" title="표준어">표준어</a>로 이야기를 하지만, 친한 사람과의 일상 대화에서는 자기만의 특유 표현법으로 말할 때가 있다.</p>
<p>첫사랑의 추억 살인사건 편(회상)에서 쿠도 신이치는 레몬파이를 좋아하는 것으로 나온다.</p>

<h3><span class="mw-headline" id=".EC.97.90.EB.8F.84.EA.B0.80.EC.99.80_.EC.BD.94.EB.82.9C.EC.9D.98_.ED.98.B8.EC.B9.AD">에도가와 코난의 호칭</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%97%90%EB%8F%84%EA%B0%80%EC%99%80_%EC%BD%94%EB%82%9C&amp;action=edit&amp;section=7" title="부분 편집: 에도가와 코난의 호칭">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>란, 아유미, 미츠히코, <a href="/wiki/%ED%82%A4%EC%82%AC%ED%82%A4_%EC%97%90%EB%A6%AC" title="키사키 에리">키사키 에리</a>, <a href="/wiki/%ED%86%A0%EC%95%BC%EB%A7%88_%EC%B9%B4%EC%A6%88%ED%95%98" title="토야마 카즈하">토야마 카즈하</a>, 경찰 관계자, 제임스 블랙은 “코난 군(コナン君)”, 코고로는 “코난(コナン)” 또는 “꼬마(ボウズ)!”로 부른다. 겐타는 “코난(コナン)”이라고 부르고 <a href="/wiki/%EC%8A%A4%EC%A6%88%ED%82%A4_%EC%86%8C%EB%85%B8%EC%BD%94" title="스즈키 소노코">스즈키 소노코</a>는 코난을 “코난 군(コナン君)”, “안경잡이 꼬마(メガネの ガキンチョ)”라고 부르며 고바야시 선생은 “에도가와 군(江戸川君)”이나 “코난 군(コナン君)”으로 부른다. <a href="/wiki/%ED%95%AB%ED%86%A0%EB%A6%AC_%ED%97%A4%EC%9D%B4%EC%A7%80" title="핫토리 헤이지">핫토리 헤이지</a>는 주변에 사람들이 있을 때조차도 “쿠도(工藤)”라고 하다가 당황해하더니 “꼬마(ボウズ)”, “코난 군(コナン君)” 등으로 고쳐 부른다. 코난의 정체를 아는 사람끼리 있을 때에는 거의 원래 이름인 “쿠도(工藤)”, “쿠도군(工藤君)”으로 부른다.</p>
<p>코난의 정체를 아는 하이바라는 "쿠도 군(工藤君)"이라고 부르지만 사람들이 있을 때는 “에도가와 군(江戸川君)”으로 고쳐 부른다. 아가사 박사는 코난과 전화를 할 때나 단둘이 있을 때는 “신이치(新一)”나 “신이치 군(新一君)”이라고 하며, 란이나 소년 탐정단이 있을 때에는 “코난 군(コナン君)”이라고 한다. 어머니인 쿠도 유키코는 단 둘이 있을 때는 “신이치(新一)”나 “신짱(新ちゃん)”이라고 하며, 남들과 있을 때는 "코난 쨩(コナンちゃん)"이라고 부른다.</p>
<p>역시 코난의 정체를 알고 있는 베르무트는 "은총알(Silver Bullet)"이나 쿨 가이(Cool Guy)로 부르며 조디 스털링은 “쿨 키드(Cool kid)”, "멋진 꼬마"라고 부른다. 아카이 슈이치는 "꼬마(ボウヤ)" <a class="mw-redirect" href="/wiki/%EA%B4%B4%EB%8F%84_%ED%82%A4%EB%93%9C" title="괴도 키드">괴도 키드</a>는 “탐정군(探偵君)” “명탐정(名探偵)” "꼬마탐정"이라고 부른다. 신이치로 변장한 상태에서 코난을 지칭할 때는 안경잡이 꼬마"(メガネのボウズ)"나 "코난 군(コナン君)"이라고 한다.</p>
<h3><span class="mw-headline" id=".EC.BF.A0.EB.8F.84_.EC.8B.A0.EC.9D.B4.EC.B9.98.EC.9D.98_.ED.98.B8.EC.B9.AD">쿠도 신이치의 호칭</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%97%90%EB%8F%84%EA%B0%80%EC%99%80_%EC%BD%94%EB%82%9C&amp;action=edit&amp;section=8" title="부분 편집: 쿠도 신이치의 호칭">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>란은 “신이치(新一)”라고 부르며 코고로는 “탐정 꼬마(探偵ボウズ)”나“신이치(新一)”라고 부른다. 에리는 “신이치 군(新一君)” 아가사 박사는 “신이치 군(新一君)”이나 “신이치(新一)”라고 부르며 소노코나 메구레 경부는 “신이치 군(新一君)”이나 “쿠도군(工藤君)”으로 부른다. 하이바라 아이나 카즈하는 “쿠도군(工藤君)”, 쿠도 유키코는 “신이치(新一)”나 “신짱(新ちゃん)”으로 부르며 헤이지는 “쿠도(工藤)”라고 부른다. 아유미는 “신이치 오빠(新一お兄さん)”, 미츠히코는 “신이치 씨(新一さん)”라고 부른다. 덧붙여 란이나 다른 사람 앞에서 코난은 신이치(자신)를 “신이치 형(新一兄ちゃん)”이라고 부른다.</p>
<h3><span class="mw-headline" id=".EC.97.90.EB.8F.84.EA.B0.80.EC.99.80_.EC.BD.94.EB.82.9C.EC.9D.98_.EB.B3.B5.EC.9E.A5">에도가와 코난의 복장</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%97%90%EB%8F%84%EA%B0%80%EC%99%80_%EC%BD%94%EB%82%9C&amp;action=edit&amp;section=9" title="부분 편집: 에도가와 코난의 복장">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>코난이 되고 나서 초반에는 대부분 정장에 빨간색의 나비넥타이를 입었으나 회를 거듭할수록 다양한 복장을 하게 된다.<sup class="reference" id="cite_ref-4"><a href="#cite_note-4">[4]</a></sup></p>
<h2><span class="mw-headline" id=".EC.A3.BC.EC.9A.94_.EC.A0.95.EB.B3.B4">주요 정보</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%97%90%EB%8F%84%EA%B0%80%EC%99%80_%EC%BD%94%EB%82%9C&amp;action=edit&amp;section=10" title="부분 편집: 주요 정보">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<ul>
<li>에도가와 코난이라는 이름의 여권은 없다. 즉, 에도가와 코난이라는 이름이 호적에 등재되어 있지 않기 때문에 여권을 만들수가 없다. 따라서 사건의 무대는 일본 국내에만 한정된다. 하지만 예외적으로 코난이 하이바라에 의해 해독제를 먹고 일시적으로나마 신이치로 다시 돌아올 수 있게 되어 영국으로 출국하여 사건을 해결하는 에피소드가 생겨나게 되었다. 또한 <a class="mw-redirect" href="/wiki/%EB%AA%85%ED%83%90%EC%A0%95_%EC%BD%94%EB%82%9C_%EA%B7%B9%EC%9E%A5%ED%8C%90_6%EA%B8%B0_-_%EB%B2%A0%EC%9D%B4%EC%BB%A4%EA%B0%80%EC%9D%98_%EB%A7%9D%EB%A0%B9" title="명탐정 코난 극장판 6기 - 베이커가의 망령">명탐정 코난 극장판 6기 - 베이커가의 망령</a>에서는 19세기 영국을 재현해 놓은 게임 속 공간에서 미션을 수행하는 장면도 나왔다. 그리고 <a class="mw-redirect" href="/wiki/%EB%A3%A8%ED%8C%A13%EC%84%B8_vs_%EB%AA%85%ED%83%90%EC%A0%95_%EC%BD%94%EB%82%9C" title="루팡3세 vs 명탐정 코난">루팡3세 vs 명탐정 코난</a> 특별판에서는 베스파니아 왕국으로 란을 구출하러 가는 장면도 나왔다.</li>
<li><a class="mw-redirect" href="/wiki/%EC%9A%94%EB%AF%B8%EC%9A%B0%EB%A6%ACTV%EB%B0%A9%EC%86%A1" title="요미우리TV방송">요미우리 TV</a> 본사앞에는 에도가와 코난의 동상이 건립되고 있다. 또한 돗토리 현에 위치한 작가의 고향에서는 코난의 동상이 세워져 있는 '코난대교'와 코난을 소재로 한 전시물이 전시되어 있는 '아오야마 고쇼 후루사토관'이 조성되어 있다.</li>
<li>코난이 음성변조기를 이용해 내세우는 탐정역은 모리 코고로, 스즈키 소노코, 아가사 박사, 야마무라 경부 등등이다. 그런데 코난과 신이치가 동일 인물이라는 걸 아는 아가사 박사는 마취총에 의한 기절이 아닌 복화술로 사건을 해결한다. 딱 한 번 <a href="/wiki/%ED%95%AB%ED%86%A0%EB%A6%AC_%ED%97%A4%EC%9D%B4%EC%A7%80" title="핫토리 헤이지">핫토리 헤이지</a>를 쓴 적이 있는데, 중간에 헤이지가 깼지만 안 깬 척 하면서 계속 듣고 있었고, 이 일로 헤이지는 코난의 정체가 신이치임을 알게 된다.</li>
<li>극장판 <a class="mw-redirect" href="/wiki/%EB%AA%85%ED%83%90%EC%A0%95_%EC%BD%94%EB%82%9C_%EA%B7%B9%EC%9E%A5%ED%8C%90_8%EA%B8%B0_-_%EC%9D%80%EB%B9%9B_%EB%82%A0%EA%B0%9C%EC%9D%98_%EB%A7%88%EC%88%A0%EC%82%AC" title="명탐정 코난 극장판 8기 - 은빛 날개의 마술사">은빛 날개의 마술사</a>에서 코고로에게 들어가야 할 마취바늘이 비행기동체가 흔들리는 바람에 실수로 <a href="/wiki/%ED%82%A4%EC%82%AC%ED%82%A4_%EC%97%90%EB%A6%AC" title="키사키 에리">키사키 에리</a>에게 들어가고 말았다. 어쩔수 없이 코난은 최초로 에리를 탐정역으로 이용했다. 에리 목소리를 내려고 했을때 실수로 소노코의 목소리가 나왔으며, 이때 마취바늘에 잠들지 않았던 소노코도 깜짝 놀랬다. 코난은 겨우 에리의 목소리로 추리를 해서 사건을 해결했다.</li>
<li>단행본 25권 (TV방송판 188화)에서는 범인의 권총으로 총격을 당해 좌측 복부 관통에 중상을 입었다. 수술했을 때에 혈액이 부족했지만 <a href="/wiki/%EB%AA%A8%EB%A6%AC_%EB%9E%80" title="모리 란">란</a>의 수혈로 구사일생하게 된다. 참고로 신이치(코난)와 란의 혈액형이 같고 이 점은 하이바라도 잘 알고 있다.</li>
<li>야구를 할 때 축구의 버릇이 몸에 베어 있어서인지 다리를 이용할 때도 있다. 또 TV게임을 잘 못해 <a href="/wiki/%EC%86%8C%EB%85%84_%ED%83%90%EC%A0%95%EB%8B%A8_(%EB%AA%85%ED%83%90%EC%A0%95_%EC%BD%94%EB%82%9C)" title="소년 탐정단 (명탐정 코난)">탐정단 일행</a>에게 비웃음을 사기도 한다. 그 때문인지, 코고로나 란이 잠든 후에 거실에서 몰래 연습할 때도 있다.</li>
<li>극중에서는 <a class="extiw" href="https://ja.wikipedia.org/wiki/W21S" title="ja:W21S">W21S</a> 휴대폰을 사용하지만, 극장판 <sup class="reference" id="cite_ref-5"><a href="#cite_note-5">[5]</a></sup>에서는 공중 전화나 「귀걸이형 휴대 전화」를 사용하며, 현재에 이르러서는 거의 쓰이지 않는다. 최근에는 <a href="/wiki/%EC%86%8C%EB%8B%88_%EC%97%91%EC%8A%A4%ED%8E%98%EB%A6%AC%EC%95%84" title="소니 엑스페리아">소니 엑스페리아</a> 기종을 사용하고 있다. 한국판 13기에서 코난이 스마트폰으로 통화하는 모습이 나왔다.</li>
<li>청력과 기억력이 뛰어나다는것이 증명되는 사례는 블랙 임팩트 조직의 손이 닿는 순간에서 나왔다 독특하고 불규칙적인 공회전 소리와 낮게 깔리는 배기음 만을 듣고도 수평대향 엔진임을알고 그종류의 엔진을 쓰는 차종류 (왜건, <a href="/wiki/%EC%8A%A4%EB%B0%94%EB%A3%A8" title="스바루">스바루</a>, <a href="/wiki/%ED%8F%AC%EB%A5%B4%EC%89%90" title="포르쉐">포르쉐</a>)까지 완벽히 알아맞힌다.</li>
<li><b>쿠도 신이치</b>라는 이름의 유래는 다음과 같이 성인 쿠도는 <a class="new" href="/w/index.php?title=%ED%83%90%EC%A0%95_%EC%9D%B4%EC%95%BC%EA%B8%B0&amp;action=edit&amp;redlink=1" title="탐정 이야기 (없는 문서)">탐정 이야기</a>(<a class="extiw" href="https://ja.wikipedia.org/wiki/%E6%8E%A2%E5%81%B5%E7%89%A9%E8%AA%9E" title="ja:探偵物語">探偵物語</a>)에 나오는 쿠도 슌사쿠(<b>工藤</b>俊作)에서 파생되었으며 이름인 신이치는 일본의 명탐정이었던 '이와이 신이치' 에서 파생된 것이다.</li>
<li><b>에도가와 코난</b>이라는 이름은 <a href="/wiki/%EC%97%90%EB%8F%84%EA%B0%80%EC%99%80_%EB%9E%80%ED%8F%AC" title="에도가와 란포">에도가와 란포</a>와 <a class="mw-redirect" href="/wiki/%EC%95%84%EC%84%9C_%EC%BD%94%EB%84%8C_%EB%8F%84%EC%9D%BC" title="아서 코넌 도일">코난 도일</a>의 이름에서 생각해 낸 것이다.</li>
</ul>
<h2><span class="mw-headline" id=".EC.84.B1.EC.9A.B0.EC.A7.84_.EB.B0.8F_.EB.93.9C.EB.9D.BC.EB.A7.88_.EB.B0.B0.EC.97.AD">성우진 및 드라마 배역</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%97%90%EB%8F%84%EA%B0%80%EC%99%80_%EC%BD%94%EB%82%9C&amp;action=edit&amp;section=11" title="부분 편집: 성우진 및 드라마 배역">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>일본어판의 경우 <a href="/wiki/%ED%83%80%EC%B9%B4%EC%95%BC%EB%A7%88_%EB%AF%B8%EB%82%98%EB%AF%B8" title="타카야마 미나미">타카야마 미나미</a>가 에도가와 코난과 어릴 적 쿠도 신이치 목소리를 더빙하고 있으며 , <a href="/wiki/%EC%95%BC%EB%A7%88%EA%B5%AC%EC%B9%98_%EC%BA%87%ED%8E%98%EC%9D%B4" title="야마구치 캇페이">야마구치 캇페이</a>가 중학생 이후의 쿠도 신이치는 목소리를 더빙하고 있다. 한국어 더빙판에서는 에도가와 코난의 경우 1기에서 <a href="/wiki/%EC%B5%9C%EB%8D%95%ED%9D%AC" title="최덕희">최덕희</a>가 맡았으며 2기부터 <a href="/wiki/%EA%B9%80%EC%84%A0%ED%98%9C_(%EC%84%B1%EC%9A%B0)" title="김선혜 (성우)">김선혜</a>가 맡고 있다. 쿠도 신이치는 <a href="/wiki/%EA%B0%95%EC%88%98%EC%A7%84_(%EB%82%A8%EC%9E%90_%EC%84%B1%EC%9A%B0)" title="강수진 (남자 성우)">강수진</a>이 처음부터 현재까지 더빙을 맡고 있다. 영어 더빙판은 에도가와 코난역을 <a class="new" href="/w/index.php?title=%EC%95%A8%EB%A6%AC%EC%8A%A8_%EB%B9%85%ED%86%A0%EB%A6%B0&amp;action=edit&amp;redlink=1" title="앨리슨 빅토린 (없는 문서)">앨리슨 빅토린</a>(<a class="extiw" href="https://en.wikipedia.org/wiki/Alison_Viktorin" title="en:Alison Viktorin">Alison Viktorin</a>)이, 쿠도 신이치역을 <a class="new" href="/w/index.php?title=%EC%A0%9C%EB%A6%AC_%EC%A3%BC%EC%9B%B0&amp;action=edit&amp;redlink=1" title="제리 주웰 (없는 문서)">제리 주웰</a>(<a class="extiw" href="https://en.wikipedia.org/wiki/Jerry_Jewell" title="en:Jerry Jewell">Jerry Jewell</a>)이 맡았다.</p>
<p>드라마 스페셜에서는 <a href="/wiki/%EC%98%A4%EA%B5%AC%EB%A6%AC_%EC%8A%8C" title="오구리 슌">오구리 슌</a><sup class="reference" id="cite_ref-6"><a href="#cite_note-6">[6]</a></sup>(<a href="/wiki/2006%EB%85%84" title="2006년">2006년</a>)과 <a href="/wiki/%EB%AF%B8%EC%A1%B0%EB%B0%94%ED%83%80_%EC%A4%80%ED%8E%98%EC%9D%B4" title="미조바타 준페이">미조바타 준페이</a>(<a href="/wiki/2011%EB%85%84" title="2011년">2011년</a>)가 쿠도 신이치 역을 맡았으며, <a class="new" href="/w/index.php?title=%ED%9B%84%EC%A7%80%EC%82%AC%ED%82%A4_%EB%82%98%EC%98%A4&amp;action=edit&amp;redlink=1" title="후지사키 나오 (없는 문서)">후지사키 나오</a>(<a class="extiw" href="https://ja.wikipedia.org/wiki/%E8%97%A4%E5%B4%8E%E7%9B%B4" title="ja:藤崎直">藤崎直</a>)(2006년)가 에도가와 코난 역<sup class="reference" id="cite_ref-7"><a href="#cite_note-7">[7]</a></sup>을 맡았다.</p>
<h3><span class="mw-headline" id=".EB.A7.88.EC.9D.8C_.EC.86.8D.EC.9D.98_.EB.8F.85.EB.B0.B1_.EC.B2.98.EB.A6.AC">마음 속의 독백 처리</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%97%90%EB%8F%84%EA%B0%80%EC%99%80_%EC%BD%94%EB%82%9C&amp;action=edit&amp;section=12" title="부분 편집: 마음 속의 독백 처리">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>일본어 더빙판에서는 코난의 대사 및 마음속으로 하는 독백 장면의 대사를 코난의 목소리를 맡은 <a href="/wiki/%ED%83%80%EC%B9%B4%EC%95%BC%EB%A7%88_%EB%AF%B8%EB%82%98%EB%AF%B8" title="타카야마 미나미">타카야마 미나미</a>가 전부 처리한다. 그런데다가 애니메이션에서 신이치의 모습으로 등장하는 장면이 적기 때문에 신이치의 목소리를 담당하는 <a href="/wiki/%EC%95%BC%EB%A7%88%EA%B5%AC%EC%B9%98_%EC%BA%87%ED%8E%98%EC%9D%B4" title="야마구치 캇페이">야마구치 캇페이</a>는 자신의 차례가 적은 것을 한탄하고 있다.</p>
<p>그러나 한국어 더빙판에서는 코난이 마음속으로 독백을 하는 장면의 대사를 신이치의 목소리를 맡은 <a href="/wiki/%EA%B0%95%EC%88%98%EC%A7%84_(%EB%82%A8%EC%9E%90_%EC%84%B1%EC%9A%B0)" title="강수진 (남자 성우)">강수진</a>이 처리하며, 영어 더빙판에서도 신이치의 목소리를 맡은 <a class="new" href="/w/index.php?title=%EC%A0%9C%EB%A6%AC_%EC%A3%BC%EC%9B%B0&amp;action=edit&amp;redlink=1" title="제리 주웰 (없는 문서)">제리 주웰</a>(<a class="extiw" href="https://en.wikipedia.org/wiki/Jerry_Jewell" title="en:Jerry Jewell">Jerry Jewell</a>)이 처리한다. 그렇게 함으로써 에도가와 코난의 정체가 쿠도 신이치라는 것을 더욱 강조하는 것을 볼 수 있다.</p>
<h2><span class="mw-headline" id=".EA.B0.81_.EC.96.B8.EC.96.B4.ED.8C.90.EB.B3.84_.EC.9D.B4.EB.A6.84">각 언어판별 이름</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%97%90%EB%8F%84%EA%B0%80%EC%99%80_%EC%BD%94%EB%82%9C&amp;action=edit&amp;section=13" title="부분 편집: 각 언어판별 이름">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>쿠도 신이치의 경우 한국어 더빙판에서는 <b>남도일</b><sup class="reference" id="cite_ref-8"><a href="#cite_note-8">[8]</a></sup>이라는 이름으로 나오며, 해외판에서는 <b>지미 쿠도</b>(Jimmy Kudo)라는 이름으로 나온다.</p>
<p>에도가와 코난의 경우 한국어 더빙판에서는 그냥 <b>코난</b><sup class="reference" id="cite_ref-9"><a href="#cite_note-9">[9]</a></sup>으로 나오며, 해외판에서는 <b>코난 에도가와</b>(Conan Edogawa)라는 이름으로 나온다.</p>
