<p><b>에두아르두 두스 헤이스 카르발류</b> (Eduardo dos Reis Carvalho, <a href="/wiki/1982%EB%85%84" title="1982년">1982년</a> <a href="/wiki/9%EC%9B%94_19%EC%9D%BC" title="9월 19일">9월 19일</a>, <a class="new" href="/w/index.php?title=%EB%AF%B8%EB%9E%80%EB%8D%B8%EB%9D%BC&amp;action=edit&amp;redlink=1" title="미란델라 (없는 문서)">미란델라</a> ~), 혹은 줄여서 <b>에두아르두</b> (Eduardo) 는 <a href="/wiki/%ED%8F%AC%EB%A5%B4%ED%88%AC%EA%B0%88" title="포르투갈">포르투갈</a>의 축구 선수로, 현재 잉글랜드 <a href="/wiki/%ED%94%84%EB%A6%AC%EB%AF%B8%EC%96%B4%EB%A6%AC%EA%B7%B8" title="프리미어리그">프리미어리그</a>의 <a href="/wiki/%EC%B2%BC%EC%8B%9C_FC" title="첼시 FC">첼시</a>에서 <a href="/wiki/%EA%B3%A8%ED%82%A4%ED%8D%BC" title="골키퍼">골키퍼</a>로 활약하고 있다.</p>

<h2><span class="mw-headline" id=".ED.81.B4.EB.9F.BD_.EA.B2.BD.EB.A0.A5">클럽 경력</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%97%90%EB%91%90%EC%95%84%EB%A5%B4%EB%91%90_%EB%91%90%EC%8A%A4_%ED%97%A4%EC%9D%B4%EC%8A%A4_%EC%B9%B4%EB%A5%B4%EB%B0%9C%EB%A5%98&amp;action=edit&amp;section=1" title="부분 편집: 클럽 경력">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>에두아르두는 1부리그에서 데뷔하기 위해 20대 중반까지 기다려야 했다. 그는 <a href="/wiki/SC_%EB%B8%8C%EB%9D%BC%EA%B0%80" title="SC 브라가">SC 브라가</a> 유소년팀 출신으로, 그는 2006-07 시즌, <a href="/wiki/%ED%94%84%EB%A6%AC%EB%A9%94%EC%9D%B4%EB%9D%BC%EB%A6%AC%EA%B0%80" title="프리메이라리가">포르투갈 리그</a> 데뷔전을 치루었고, 이후 6개월간 <a href="/wiki/SC_%EB%B2%A0%EC%9D%B4%EB%9D%BC%EB%A7%88%EB%A5%B4" title="SC 베이라마르">SC 베이라마르</a>에 임대되었었다.</p>
<p>그 다음 시즌, 에두아르두는 또다시 <a href="/wiki/%EB%B9%84%ED%86%A0%EB%A6%AC%EC%95%84_FC" title="비토리아 FC">비토리아 FC</a>로 임대되었다. 그는 <a class="new" href="/w/index.php?title=%EC%B9%B4%EB%A5%BC%EB%A3%A8%EC%8A%A4_%EC%B9%B4%EB%A5%B4%EB%B0%94%EC%95%8C&amp;action=edit&amp;redlink=1" title="카를루스 카르바알 (없는 문서)">카를루스 카르바알</a>가 이끄는 팀의 주축 선수가 되어 초대 포르투갈 리그 컵 우승을 거미쥐었고, 그에 따라 비토리아는 <a class="mw-redirect" href="/wiki/UEFA_%EC%BB%B5" title="UEFA 컵">UEFA 컵</a> 진출권을 획득하였다. 그는 <a href="/wiki/%EC%8A%A4%ED%8F%AC%EB%A5%B4%ED%8C%85_CP" title="스포르팅 CP">스포르팅 CP</a>와의 결승전에서 환상적인 활약을 펼치며, 승부차기에서 세 차례 선방하였다.</p>
<p>에두아르두는 2008-09 시즌 브라가로 복귀하여 리그 30경기에 모두 출장하였으며, <a class="mw-redirect" href="/wiki/UEFA_%EC%BB%B5" title="UEFA 컵">UEFA 컵</a> 16강에 진출하였다. 브라가는 이듬해 리그 최고의 성적인 준우승을 거두었으며, 그는 고작 20골을 실점하여 최고의 골키퍼가 되었다.</p>
<p><a href="/wiki/2010%EB%85%84" title="2010년">2010년</a> <a href="/wiki/7%EC%9B%94_7%EC%9D%BC" title="7월 7일">7월 7일</a>, 그는 <a href="/wiki/%EC%9D%B4%ED%83%88%EB%A6%AC%EC%95%84" title="이탈리아">이탈리아</a> <a href="/wiki/%EC%84%B8%EB%A6%AC%EC%97%90_A" title="세리에 A">세리에 A</a>의 <a href="/wiki/%EC%A0%9C%EB%85%B8%EC%95%84_CFC" title="제노아 CFC">제노아 CFC</a>와 €4M의 가격에 4년 계약을 체결하였고, 클럽을 떠난 <a href="/wiki/%EB%A7%88%EB%A5%B4%EC%BD%94_%EC%95%84%EB%A9%9C%EB%A6%AC%EC%95%84" title="마르코 아멜리아">마르코 아멜리아</a>의 자리를 대체하였다.<sup class="reference" id="cite_ref-Genoa_1-0"><a href="#cite_note-Genoa-1">[1]</a></sup> 브라가는 제노바가 선수를 매각할 시 25%의 이적료를 받도록 계약하였다.<sup class="reference" id="cite_ref-transferFee_2-0"><a href="#cite_note-transferFee-2">[2]</a></sup></p>
<p>에두아르두는 그의 <a href="/wiki/%EC%84%B8%EB%A6%AC%EC%97%90_A" title="세리에 A">세리에 A</a> 첫 시즌 모든 경기에 출장하였고, 팀은 중위권으로 시즌을 마감하였다. 2011-12 시즌을 앞두고 <a href="/wiki/ACF_%ED%94%BC%EC%98%A4%EB%A0%8C%ED%8B%B0%EB%82%98" title="ACF 피오렌티나">ACF 피오렌티나</a>에서 <a href="/wiki/%EC%84%B8%EB%B0%94%EC%8A%A4%ED%8B%B0%EC%95%B5_%ED%94%84%EB%A0%88%EC%9D%B4" title="세바스티앵 프레이">세바스티앵 프레이</a>가 이적함에 따라 그는 주전 자리를 잃었다. <a href="/wiki/2011%EB%85%84" title="2011년">2011년</a> <a href="/wiki/7%EC%9B%94" title="7월">7월</a>, 그는 포르투갈의 <a href="/wiki/SL_%EB%B2%A4%ED%94%BC%EC%B9%B4" title="SL 벤피카">SL 벤피카</a>로 임대되었다. 리스본은 이 계약으로부터 시즌 종료 후, 선수 구매 권한을 가지게 되었다.</p>
<h2><span class="mw-headline" id=".EA.B5.AD.EA.B0.80.EB.8C.80.ED.91.9C_.EA.B2.BD.EB.A0.A5">국가대표 경력</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%97%90%EB%91%90%EC%95%84%EB%A5%B4%EB%91%90_%EB%91%90%EC%8A%A4_%ED%97%A4%EC%9D%B4%EC%8A%A4_%EC%B9%B4%EB%A5%B4%EB%B0%9C%EB%A5%98&amp;action=edit&amp;section=2" title="부분 편집: 국가대표 경력">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p><a href="/wiki/UEFA_%EC%9C%A0%EB%A1%9C_2008" title="UEFA 유로 2008">UEFA 유로 2008</a> 종료 후, <a href="/wiki/%ED%8F%AC%EB%A5%B4%ED%88%AC%EA%B0%88_%EC%B6%95%EA%B5%AC_%EA%B5%AD%EA%B0%80%EB%8C%80%ED%91%9C%ED%8C%80" title="포르투갈 축구 국가대표팀">포르투갈 축구 국가대표팀</a>의 신임 감독 <a class="new" href="/w/index.php?title=%EC%B9%B4%EB%A5%BC%EB%A3%A8%EC%8A%A4_%ED%80%98%EC%9D%B4%EB%A1%9C%EC%8A%A4&amp;action=edit&amp;redlink=1" title="카를루스 퀘이로스 (없는 문서)">카를루스 퀘이로스</a>는 <a href="/wiki/2010%EB%85%84_FIFA_%EC%9B%94%EB%93%9C%EC%BB%B5" title="2010년 FIFA 월드컵">2010년 FIFA 월드컵</a> <a href="/wiki/%EB%AA%B0%ED%83%80_%EC%B6%95%EA%B5%AC_%EA%B5%AD%EA%B0%80%EB%8C%80%ED%91%9C%ED%8C%80" title="몰타 축구 국가대표팀">몰타</a>와 <a href="/wiki/%EB%8D%B4%EB%A7%88%ED%81%AC_%EC%B6%95%EA%B5%AC_%EA%B5%AD%EA%B0%80%EB%8C%80%ED%91%9C%ED%8C%80" title="덴마크 축구 국가대표팀">덴마크</a>와의 예선전을 앞두고 발탁되었고, <a class="new" href="/w/index.php?title=%ED%81%84_(1975%EB%85%84)&amp;action=edit&amp;redlink=1" title="큄 (1975년) (없는 문서)">큄</a>의 백업 골키퍼가 되었다.</p>
<p>그는 <a href="/wiki/2009%EB%85%84" title="2009년">2009년</a> <a href="/wiki/2%EC%9B%94_11%EC%9D%BC" title="2월 11일">2월 11일</a>, <a href="/wiki/%ED%95%80%EB%9E%80%EB%93%9C_%EC%B6%95%EA%B5%AC_%EA%B5%AD%EA%B0%80%EB%8C%80%ED%91%9C%ED%8C%80" title="핀란드 축구 국가대표팀">핀란드</a>전에서 60분 선발출장하며 데뷔전을 치루었다. 이 경기에서 1-0으로 승리하였고, 잔여 월드컵 예선경기에 모두 참여하였다. 그는 <a href="/wiki/%EB%82%A8%EC%95%84%ED%94%84%EB%A6%AC%EC%B9%B4_%EA%B3%B5%ED%99%94%EA%B5%AD" title="남아프리카 공화국">남아프리카 공화국</a>에서 열린 <a href="/wiki/2010%EB%85%84_FIFA_%EC%9B%94%EB%93%9C%EC%BB%B5" title="2010년 FIFA 월드컵">2010년 FIFA 월드컵</a> 본선에서도 선발출장하여 <a href="/wiki/%EC%BD%94%ED%8A%B8%EB%94%94%EB%B6%80%EC%95%84%EB%A5%B4_%EC%B6%95%EA%B5%AC_%EA%B5%AD%EA%B0%80%EB%8C%80%ED%91%9C%ED%8C%80" title="코트디부아르 축구 국가대표팀">코트디부아르</a>, <a href="/wiki/%EC%A1%B0%EC%84%A0%EB%AF%BC%EC%A3%BC%EC%A3%BC%EC%9D%98%EC%9D%B8%EB%AF%BC%EA%B3%B5%ED%99%94%EA%B5%AD_%EC%B6%95%EA%B5%AC_%EA%B5%AD%EA%B0%80%EB%8C%80%ED%91%9C%ED%8C%80" title="조선민주주의인민공화국 축구 국가대표팀">북한</a>, <a href="/wiki/%EB%B8%8C%EB%9D%BC%EC%A7%88_%EC%B6%95%EA%B5%AC_%EA%B5%AD%EA%B0%80%EB%8C%80%ED%91%9C%ED%8C%80" title="브라질 축구 국가대표팀">브라질</a>과의 조별리그전에서 무실점을 기록하였다. 그러나 그는 <a href="/wiki/%EC%8A%A4%ED%8E%98%EC%9D%B8_%EC%B6%95%EA%B5%AC_%EA%B5%AD%EA%B0%80%EB%8C%80%ED%91%9C%ED%8C%80" title="스페인 축구 국가대표팀">스페인</a>과의 16강전 경기에서 <a href="/wiki/%EB%8B%A4%EB%B9%84%EB%93%9C_%EB%B9%84%EC%95%BC" title="다비드 비야">다비드 비야</a>에게 1골을 헌납하여 0-1로 패하였다.</p>
<p>그는 클럽에서 교체 선수로 전락함에 따라, 그는 국가대표 주전 골키퍼 자리를 <a href="/wiki/%ED%9B%84%EC%9D%B4_%ED%8C%8C%ED%8A%B8%EB%A6%AC%EC%8B%9C%EC%9A%B0" title="후이 파트리시우">후이 파트리시우</a>에게 내주었다. 두 골키퍼는 모두 <a href="/wiki/UEFA_%EC%9C%A0%EB%A1%9C_2012" title="UEFA 유로 2012">UEFA 유로 2012</a> 예선에서 5경기씩 출장하였고, 포르투갈은 본선에 진출하였다.</p>
