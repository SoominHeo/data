<p><b>에드워드 릴리 스테티니어스 2세</b>(<span lang="en" xml:lang="en">Edward Reilly Stettinius, Jr.</span>, <a href="/wiki/1900%EB%85%84" title="1900년">1900년</a> <a href="/wiki/10%EC%9B%94_22%EC%9D%BC" title="10월 22일">10월 22일</a> - <a href="/wiki/1949%EB%85%84" title="1949년">1949년</a> <a href="/wiki/10%EC%9B%94_31%EC%9D%BC" title="10월 31일">10월 31일</a>)는 <a href="/wiki/1944%EB%85%84" title="1944년">1944년</a>부터 <a href="/wiki/1945%EB%85%84" title="1945년">1945년</a>까지 <a class="mw-redirect" href="/wiki/%ED%94%84%EB%9E%AD%ED%81%B4%EB%A6%B0_%EB%A3%A8%EC%8A%A4%EB%B2%A8%ED%8A%B8" title="프랭클린 루스벨트">프랭클린 루스벨트</a> 및 <a href="/wiki/%ED%95%B4%EB%A6%AC_S._%ED%8A%B8%EB%A3%A8%EB%A8%BC" title="해리 S. 트루먼">해리 S. 트루먼</a> 대통령 당시의 미국의 국무부 장관이었다.</p>
<p>그는 <a href="/wiki/%EC%8B%9C%EC%B9%B4%EA%B3%A0" title="시카고">시카고</a>에서 태어났고 <a href="/wiki/1920%EB%85%84" title="1920년">1920년</a>까지 <a class="new" href="/w/index.php?title=%ED%8F%BC%ED%94%84%EB%A0%88%ED%8A%B8_%ED%95%99%EA%B5%90&amp;action=edit&amp;redlink=1" title="폼프레트 학교 (없는 문서)">폼프레트 학교</a>를 다녔고, 그 후 <a href="/wiki/1924%EB%85%84" title="1924년">1924년</a>까지 <a href="/wiki/%EB%B2%84%EC%A7%80%EB%8B%88%EC%95%84_%EB%8C%80%ED%95%99%EA%B5%90" title="버지니아 대학교">버지니아 대학교</a>를 다녔다. <a href="/wiki/1926%EB%85%84" title="1926년">1926년</a> <a href="/wiki/%EC%A0%9C%EB%84%88%EB%9F%B4_%EB%AA%A8%ED%84%B0%EC%8A%A4" title="제너럴 모터스">제너럴 모터스</a>의 부사장 존 리 프래트의 보좌관이 되었고, <a href="/wiki/1931%EB%85%84" title="1931년">1931년</a> 그 자리를 물려받았다. <a href="/wiki/%EC%A0%9C%EB%84%88%EB%9F%B4_%EB%AA%A8%ED%84%B0%EC%8A%A4" title="제너럴 모터스">제너럴 모터스</a>에서 그는 구조조정 프로그램을 개발하는 일을 맡았고, 이것으로 그는 <a class="mw-redirect" href="/wiki/%ED%94%84%EB%9E%AD%ED%81%B4%EB%A6%B0_%EB%A3%A8%EC%8A%A4%EB%B2%A8%ED%8A%B8" title="프랭클린 루스벨트">프랭클린 루스벨트</a>와 접촉하였다. 그는 <a class="mw-redirect" href="/wiki/%ED%94%84%EB%9E%AD%ED%81%B4%EB%A6%B0_%EB%A3%A8%EC%8A%A4%EB%B2%A8%ED%8A%B8" title="프랭클린 루스벨트">프랭클린 루스벨트</a>를 위하여 <a class="new" href="/w/index.php?title=%EA%B5%AD%EA%B0%80%EB%B6%80%ED%9D%A5%EB%B6%80&amp;action=edit&amp;redlink=1" title="국가부흥부 (없는 문서)">국가부흥부</a>에서 잠시 일하였다. <a href="/wiki/1934%EB%85%84" title="1934년">1934년</a> 그는 <a class="new" href="/w/index.php?title=%EB%AF%B8%EA%B5%AD_%EC%B2%A0%EA%B0%95_%ED%9A%8C%EC%82%AC&amp;action=edit&amp;redlink=1" title="미국 철강 회사 (없는 문서)">미국 철강 회사</a>에 들어가 상급 관리자가 되었다가 <a class="mw-redirect" href="/wiki/%ED%94%84%EB%9E%AD%ED%81%B4%EB%A6%B0_%EB%A3%A8%EC%8A%A4%EB%B2%A8%ED%8A%B8" title="프랭클린 루스벨트">프랭클린 루스벨트</a>가 미국의 대통령으로 당선되자 생산관리부(the Office of Production Management)의 감독자로서 행정부에 참여하도록 요청받았다. 2년 뒤, 동맹국들에게 <a class="mw-redirect" href="/wiki/%EB%AC%B4%EA%B8%B0_%EB%8C%80%EC%97%AC%EB%B2%95" title="무기 대여법">무기 대여법</a>에 의한 원조의 책임자(무기대여국장)가 되었고, 이는 <a href="/wiki/1943%EB%85%84" title="1943년">1943년</a> 국무부 차관이 될 때까지의 직책이었다. <a href="/wiki/1944%EB%85%84" title="1944년">1944년</a> <a href="/wiki/11%EC%9B%94" title="11월">11월</a> 당시 국무부 장관 <a href="/wiki/%EC%BD%94%EB%8D%B8_%ED%97%90" title="코델 헐">코델 헐</a>의 건강 악화로 장관직을 승계하였다.</p>
<p>스테티니어스는 <a href="/wiki/1945%EB%85%84" title="1945년">1945년</a> <a href="/wiki/6%EC%9B%94_26%EC%9D%BC" title="6월 26일">6월 26일</a> 공식적으로 설립된 <a href="/wiki/%EC%9C%A0%EC%97%94" title="유엔">유엔</a>의 핵심 인물이자 의장이었다. 또한 <a href="/wiki/%EC%96%84%ED%83%80_%ED%9A%8C%EB%8B%B4" title="얄타 회담">얄타 회담</a>, <a href="/wiki/%EB%8D%A4%EB%B2%84%ED%84%B4%EC%98%A4%ED%81%AC%EC%8A%A4_%ED%9A%8C%EC%9D%98" title="덤버턴오크스 회의">덤버턴오크스 회의</a>에서 중요한 역할을 하였다. 얄타 회담을 앞두고 그는 부하직원에게 한국이 도대체 어디에 있는 나라인지 아느냐고 묻기도 하였다. <sup class="reference" id="cite_ref-1"><a href="#cite_note-1">[1]</a></sup></p>
<p>국무부장관으로서 스테티니어스는 <a href="/wiki/%ED%95%80%EB%9E%80%EB%93%9C" title="핀란드">핀란드</a>에서 발견된 러시아의 암호 책을 <a href="/wiki/%EC%86%8C%EB%A0%A8" title="소련">소련</a>에 돌려주는 결정을 하였다. 이것은 러시아의 전송문을 해독하려는 미국의 노력을 방해한 것이었다. 그 전송문의 상당수는 나중에 해독되었는데 이것은 미국 정부의 고위 관리에까지 소련의 세력이 폭넓게 침투해 있음을 알려 주었다. 그가 이렇게 행동한 이유는 아직도 명확하지 않다. 그 이후에 스테티니어스는 국무부 장관직을 사임하고 초대 주 유엔의 미국 대사가 되었다. <a href="/wiki/1946%EB%85%84" title="1946년">1946년</a> <a href="/wiki/6%EC%9B%94" title="6월">6월</a> 그는 이 자리도 사임하고, 소련과의 긴장을 완화하는 수단으로서 <a href="/wiki/%EC%9C%A0%EC%97%94" title="유엔">유엔</a>을 활용하는 것에 반대하는 <a href="/wiki/%ED%95%B4%EB%A6%AC_S._%ED%8A%B8%EB%A3%A8%EB%A8%BC" title="해리 S. 트루먼">해리 S. 트루먼</a>에 대하여 비판하였다. <a href="/wiki/1946%EB%85%84" title="1946년">1946년</a>∼<a href="/wiki/1947%EB%85%84" title="1947년">1947년</a> <a class="mw-redirect" href="/wiki/%EB%B2%84%EC%A7%80%EB%8B%88%EC%95%84%EB%8C%80%ED%95%99%EA%B5%90" title="버지니아대학교">버지니아대학교</a> 총장을 역임하였다.</p>
