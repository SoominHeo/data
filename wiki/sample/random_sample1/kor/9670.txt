<p><b>에드워드 파머 톰슨</b>(Edward Palmer Thompson, 1924년 2월 3일 – 1993년 8월 28일)은 <a href="/wiki/%EC%98%81%EA%B5%AD" title="영국">영국</a>의 <a href="/wiki/%EC%97%AD%EC%82%AC" title="역사">역사</a>가, <a href="/wiki/%EC%9E%91%EA%B0%80" title="작가">작가</a>, <a href="/wiki/%EC%82%AC%ED%9A%8C%EC%A3%BC%EC%9D%98" title="사회주의">사회주의</a>자, 평화운동가이다. 18세기 말에서 19세기 초엽에 걸쳐 <a href="/wiki/%EC%98%81%EA%B5%AD" title="영국">영국</a>에서 풍미하였던, 이상을 급히 실현하려는 <a class="mw-redirect" href="/wiki/%EC%82%AC%ED%9A%8C%EC%9A%B4%EB%8F%99" title="사회운동">사회운동</a>을 주제로 한 노작인 『영국노동계급의 형성』<sup class="reference" id="cite_ref-1"><a href="#cite_note-1">[1]</a></sup>으로 최고 유명하고 <a href="/wiki/%EC%9C%8C%EB%A6%AC%EC%97%84_%EB%AA%A8%EB%A6%AC%EC%8A%A4" title="윌리엄 모리스">윌리엄 모리스</a>의 전기인 『윌리엄 모리스』<sup class="reference" id="cite_ref-2"><a href="#cite_note-2">[2]</a></sup>와 <a href="/wiki/%EC%9C%8C%EB%A6%AC%EC%97%84_%EB%B8%94%EB%A0%88%EC%9D%B4%ED%81%AC" title="윌리엄 블레이크">윌리엄 블레이크</a>를 연구한 책<sup class="reference" id="cite_ref-3"><a href="#cite_note-3">[3]</a></sup>을 인행하였으며, 다작한 언론인이자 수필가이고 심지어 『더 시카오스 페이퍼스』<sup class="reference" id="cite_ref-4"><a href="#cite_note-4">[4]</a></sup>를 위시한 <a href="/wiki/SF_(%EC%9E%A5%EB%A5%B4)" title="SF (장르)">공상과학소설</a>과 시집을 출판하기도 하였다. <a href="/wiki/%EA%B3%B5%EC%82%B0%EB%8B%B9" title="공산당">공산당</a> 지식인의 일원이고 1956년 <a class="mw-redirect" href="/wiki/%EC%86%8C%EB%B9%84%EC%97%90%ED%8A%B8_%EC%97%B0%EB%B0%A9" title="소비에트 연방">소비에트사회주의공화국연방</a>군의 <a href="/wiki/%ED%97%9D%EA%B0%80%EB%A6%AC" title="헝가리">헝가리</a> 침공에 대처를 놓고 탈당하게 되었지만, 1950년대 말기 영국에서의 제일차 <a class="mw-redirect" href="/wiki/%EC%8B%A0%EC%A2%8C%EC%9D%B5" title="신좌익">신좌익</a>에서 중요한 역할을 맡았다. 에드워드 파머 톰슨은 1964년에서 1970년과 1974년에서 1979년에 <a href="/wiki/%EB%85%B8%EB%8F%99%EB%8B%B9_(%EC%98%81%EA%B5%AD)" title="노동당 (영국)">노동당</a> 행정부를 상대하는 기개 높고 의기가 당당한 좌익・사회주의 논객이었고 1980년대에는 <a href="/wiki/%EC%9C%A0%EB%9F%BD" title="유럽">구주</a>에서 세계 평화와 인류의 생존을 위협하는 핵의 생산ㆍ저장ㆍ사용 금지를 골자로 한 반핵운동을 앞장서서 이끈 지식인이었다.</p>
