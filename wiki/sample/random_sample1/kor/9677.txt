<p><b>에디르네</b>(<span style="font-size: smaller;"><a href="/wiki/%ED%84%B0%ED%82%A4%EC%96%B4" title="터키어">터키어</a>:</span> <span lang="tr" xml:lang="tr">Edirne</span>, <span style="font-size: smaller;"><a href="/wiki/%EA%B7%B8%EB%A6%AC%EC%8A%A4%EC%96%B4" title="그리스어">그리스어</a>:</span> <span lang="el" xml:lang="el">Αδριανούπολη</span>, <span style="font-size: smaller;"><a href="/wiki/%EB%B6%88%EA%B0%80%EB%A6%AC%EC%95%84%EC%96%B4" title="불가리아어">불가리아어</a>:</span> <span lang="bg" xml:lang="bg">Одрин</span>)는 <a href="/wiki/%ED%84%B0%ED%82%A4" title="터키">터키</a>의 가장 서쪽에 있는 <a href="/wiki/%EB%8F%84%EC%8B%9C" title="도시">도시</a>이다. <a href="/wiki/%EA%B7%B8%EB%A6%AC%EC%8A%A4" title="그리스">그리스</a>와 <a href="/wiki/%EB%B6%88%EA%B0%80%EB%A6%AC%EC%95%84" title="불가리아">불가리아</a> 국경근처의 도시로 터키의 <a href="/wiki/%EC%97%90%EB%94%94%EB%A5%B4%EB%84%A4_%EC%A3%BC" title="에디르네 주">에디르네 주</a>의 주도이다. 영어권에서는 <b>아드리아노플</b>로 알려져 있다.</p>
<h2><span class="mw-headline" id=".EC.A7.80.EB.A6.AC.EC.A0.81_.EC.9C.84.EC.B9.98">지리적 위치</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%97%90%EB%94%94%EB%A5%B4%EB%84%A4&amp;action=edit&amp;section=1" title="부분 편집: 지리적 위치">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>북서쪽으로 <a href="/wiki/%EB%B6%88%EA%B0%80%EB%A6%AC%EC%95%84" title="불가리아">불가리아</a>, 서쪽으로 <a href="/wiki/%EA%B7%B8%EB%A6%AC%EC%8A%A4" title="그리스">그리스</a>, 남서쪽으로 <a href="/wiki/%EC%97%90%EA%B2%8C_%ED%95%B4" title="에게 해">에게 해</a>와 면해 있는 에디르네 주의 중간에 있다. 그리스와 불가리아 국경 근처로 툰자 강과 메리츠 강의 합류 지점에 있다.</p>
<p><a href="/wiki/%EB%9F%B0%EB%8D%98" title="런던">런던</a>과 <a href="/wiki/%EC%9D%B4%EC%8A%A4%ED%83%84%EB%B6%88" title="이스탄불">이스탄불</a>을 잇는 철도가 지나며, 간선도로로 중앙 유럽, <a href="/wiki/%EC%9D%B4%EC%8A%A4%ED%83%84%EB%B6%88" title="이스탄불">이스탄불</a>과 연결되는 교통의 요지이다.</p>
<h2><span class="mw-headline" id=".EC.97.AD.EC.82.AC">역사</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%97%90%EB%94%94%EB%A5%B4%EB%84%A4&amp;action=edit&amp;section=2" title="부분 편집: 역사">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>역사적으로 에디르네는 <a href="/wiki/%EC%95%84%EB%82%98%ED%86%A8%EB%A6%AC%EC%95%84" title="아나톨리아">아나톨리아</a>와 발칸 반도 사이의 주요통로가 지나는 관계로 전략적인 요충지로 여겨져 전쟁이 끊이질 않았다. <a href="/wiki/%ED%8A%B8%EB%9D%BC%ED%82%A4%EC%95%84" title="트라키아">트라키아</a>의 부족들이 최초로 정착하여 살았고 일찍부터 <a href="/wiki/%EB%A1%9C%EB%A7%88_%EC%A0%9C%EA%B5%AD" title="로마 제국">로마 제국</a>에 편입되었다. <a href="/wiki/125%EB%85%84" title="125년">125년</a> 황제 <a href="/wiki/%ED%95%98%EB%93%9C%EB%A6%AC%EC%95%84%EB%88%84%EC%8A%A4" title="하드리아누스">하드리아누스</a>에 의해 재건되고 확장되어, 그의 이름을 따서 <b>하드리아노폴리스</b> 또는 <b>아드리아노폴리스</b>로 개명되었다. 로마 속주 <a href="/wiki/%ED%8A%B8%EB%9D%BC%ED%82%A4%EC%95%84" title="트라키아">트라키아</a>의 수도로 번창했으며 <a href="/wiki/371%EB%85%84" title="371년">371년</a> <a href="/wiki/%EB%B0%9C%EB%A0%8C%EC%8A%A4" title="발렌스">발렌스</a> 황제가 <a href="/wiki/%EC%95%84%EB%93%9C%EB%A6%AC%EC%95%84%EB%85%B8%ED%8F%B4%EB%A6%AC%EC%8A%A4_%EC%A0%84%ED%88%AC" title="아드리아노폴리스 전투">아드리아노폴리스 전투</a>에서 고트족과 싸우다 전사하였다. 그후에도 계속해서 <a href="/wiki/%EB%B9%84%EC%9E%94%ED%8B%B0%EC%9B%80_%EC%A0%9C%EA%B5%AD" title="비잔티움 제국">비잔티움 제국</a>의 중요한 도시이자 부수도로 남아있다가 <a href="/wiki/10%EC%84%B8%EA%B8%B0" title="10세기">10세기</a>에 <a class="mw-redirect" href="/wiki/%EB%B6%88%EA%B0%80%EB%A5%B4_%EC%A1%B1" title="불가르 족">불가르 족</a>에게 함락되었고, <a href="/wiki/%EC%A0%9C4%EC%B0%A8_%EC%8B%AD%EC%9E%90%EA%B5%B0" title="제4차 십자군">제4차 십자군</a>등 라틴인들에게 2차례의 심각한 약탈을 당했다.</p>
<p><a href="/wiki/1360%EB%85%84" title="1360년">1360년</a>과 <a href="/wiki/1362%EB%85%84" title="1362년">1362년</a> 두 차례에 걸쳐 <a href="/wiki/%EC%98%A4%EC%8A%A4%EB%A7%8C_%EC%A0%9C%EA%B5%AD" title="오스만 제국">오스만 제국</a>의 <a class="mw-redirect" href="/wiki/%EB%AC%B4%EB%9D%BC%EB%93%9C_1%EC%84%B8" title="무라드 1세">무라드 1세</a>의 침공으로 오스만의 세력으로 완전히 넘어갔고 <a href="/wiki/1369%EB%85%84" title="1369년">1369년</a>에 지금과 같은 이름으로 바뀌었다. 이후 오스만 제국의 수도로 오스만의 유럽 침략의 발판이 되었다. 15세기에 들어서 오스만 제국의 수도이자 행정·상업·문화 중심지로서 번영을 누렸고 <a href="/wiki/1453%EB%85%84" title="1453년">1453년</a> <a href="/wiki/%EC%BD%98%EC%8A%A4%ED%83%84%ED%8B%B0%EB%85%B8%ED%8F%B4%EB%A6%AC%EC%8A%A4%EC%9D%98_%ED%95%A8%EB%9D%BD" title="콘스탄티노폴리스의 함락">콘스탄티노폴리스의 함락</a>으로 수도의 자리를 <a href="/wiki/%EC%9D%B4%EC%8A%A4%ED%83%84%EB%B6%88" title="이스탄불">이스탄불</a>에 내주었다.</p>
<p>근대에 이르러 수차례 전쟁을 거치면서 외국의 지배를 받고 파괴되어 점차 쇠퇴해갔다. <a href="/wiki/1829%EB%85%84" title="1829년">1829년</a>과 <a href="/wiki/1878%EB%85%84" title="1878년">1878년</a>에는 <a href="/wiki/%EB%9F%AC%EC%8B%9C%EC%95%84_%EC%A0%9C%EA%B5%AD" title="러시아 제국">러시아 제국</a>에게 점령되었으며, <a href="/wiki/1913%EB%85%84" title="1913년">1913년</a> <a href="/wiki/%EC%A0%9C1%EC%B0%A8_%EB%B0%9C%EC%B9%B8_%EC%A0%84%EC%9F%81" title="제1차 발칸 전쟁">제1차 발칸 전쟁</a> 중에는 <a href="/wiki/%EB%B6%88%EA%B0%80%EB%A6%AC%EC%95%84" title="불가리아">불가리아</a>에 점령되었고 다시 터키로 넘어갔다. <a href="/wiki/1920%EB%85%84" title="1920년">1920년</a> <a href="/wiki/%ED%84%B0%ED%82%A4_%EB%8F%85%EB%A6%BD_%EC%A0%84%EC%9F%81" title="터키 독립 전쟁">터키 독립 전쟁</a> 중에는 <a href="/wiki/%EA%B7%B8%EB%A6%AC%EC%8A%A4" title="그리스">그리스</a>에 넘어갔으나, <a href="/wiki/1922%EB%85%84" title="1922년">1922년</a> 결국 다시 터키로 넘어왔다.</p>
