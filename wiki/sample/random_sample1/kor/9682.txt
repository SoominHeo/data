<p><b>에런 제임스 램지</b> (<span style="font-size: smaller;"><a href="/wiki/%EC%98%81%EC%96%B4" title="영어">영어</a>:</span> <span lang="en" xml:lang="en">Aaron James Ramsey</span>, <a href="/wiki/1990%EB%85%84" title="1990년">1990년</a> <a href="/wiki/12%EC%9B%94_26%EC%9D%BC" title="12월 26일">12월 26일</a>, <a href="/wiki/%EC%9B%A8%EC%9D%BC%EC%8A%A4" title="웨일스">웨일스</a> <a href="/wiki/%EC%BC%80%EC%96%B4%ED%95%84%EB%A6%AC" title="케어필리">케어필리</a> ~ )는 <a href="/wiki/%EC%9B%A8%EC%9D%BC%EC%8A%A4" title="웨일스">웨일스</a>의 축구 선수이자 국가대표팀 주장으로, 현재 <a href="/wiki/%ED%94%84%EB%A6%AC%EB%AF%B8%EC%96%B4%EB%A6%AC%EA%B7%B8" title="프리미어리그">프리미어리그</a>의 <a href="/wiki/%EC%95%84%EC%8A%A4%EB%84%90_FC" title="아스널 FC">아스널</a> 소속이다. 이적 한 <a href="/wiki/%EC%84%B8%EC%8A%A4%ED%81%AC_%ED%8C%8C%EB%B8%8C%EB%A0%88%EA%B0%80%EC%8A%A4" title="세스크 파브레가스">세스크 파브레가스</a>를 대체 할 것으로 기대받았다. 유소년 팀에서부터 뛰던 <a class="mw-redirect" href="/wiki/%EC%B9%B4%EB%94%94%ED%94%84_%EC%8B%9C%ED%8B%B0" title="카디프 시티">카디프 시티</a>에서 데뷔했으며 1군 팀에서 16경기를 뛰었다. <a href="/wiki/2008%EB%85%84" title="2008년">2008년</a> <a class="mw-redirect" href="/wiki/%EC%95%84%EC%8A%A4%EB%84%90" title="아스널">아스널</a>로 이적하는데 이적료는 알려지지 않았지만 약 5백만 파운드로 추정된다.<sup class="정리_필요 noprint">[<a href="/wiki/%EC%9C%84%ED%82%A4%EB%B0%B1%EA%B3%BC:%EC%B6%9C%EC%B2%98_%EB%B0%9D%ED%9E%88%EA%B8%B0" title="위키백과:출처 밝히기">출처 필요</a>]</sup></p>

<h2><span class="mw-headline" id=".ED.81.B4.EB.9F.BD_.EA.B2.BD.EB.A0.A5">클럽 경력</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%97%90%EB%9F%B0_%EB%9E%A8%EC%A7%80&amp;action=edit&amp;section=1" title="부분 편집: 클럽 경력">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<h3><span class="mw-headline" id=".EC.B4.88.EA.B8.B0_.EA.B2.BD.EB.A0.A5">초기 경력</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%97%90%EB%9F%B0_%EB%9E%A8%EC%A7%80&amp;action=edit&amp;section=2" title="부분 편집: 초기 경력">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>축구를 배우기 이전에는 학교에서 <a href="/wiki/%EB%9F%AD%EB%B9%84" title="럭비">럭비</a> 선수로 활동했었다. 축구로 방향을 바꾼 이후 지역 팀인 <a class="mw-redirect" href="/wiki/%EC%B9%B4%EB%94%94%ED%94%84_%EC%8B%9C%ED%8B%B0" title="카디프 시티">카디프 시티</a>에 입단했다. <a href="/wiki/%EB%9F%B0%EB%8D%98" title="런던">런던</a>으로 떠나기 전까지 부모님과 형과 함께 살았다. 웨일스어와 영어를 모두 사용할 수 있다.</p>

<h4><span class="mw-headline" id="2013-14_.EC.8B.9C.EC.A6.8C">2013-14 시즌</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%97%90%EB%9F%B0_%EB%9E%A8%EC%A7%80&amp;action=edit&amp;section=13" title="부분 편집: 2013-14 시즌">편집</a><span class="mw-editsection-bracket">]</span></span></h4>
<p>프리시즌 좋은 경기력을 펼치면서 <a class="mw-redirect" href="/wiki/%EC%95%84%EC%8A%A4%EB%82%A0" title="아스날">아스날</a> 팬들에게 기대감을 안겼다. 비록 시즌 초반이긴 하지만 <a class="mw-redirect" href="/wiki/%EC%9E%89%EA%B8%80%EB%9E%9C%EB%93%9C_%ED%94%84%EB%A6%AC%EB%AF%B8%EC%96%B4%EB%A6%AC%EA%B7%B8" title="잉글랜드 프리미어리그">리그</a> 4경기 2골, <a href="/wiki/UEFA_%EC%B1%94%ED%94%BC%EC%96%B8%EC%8A%A4%EB%A6%AC%EA%B7%B8" title="UEFA 챔피언스리그">챔피언스리그</a> 3경기 3골을 기록하며 <a href="/wiki/%EC%98%AC%EB%A6%AC%EB%B9%84%EC%97%90_%EC%A7%80%EB%A3%A8" title="올리비에 지루">올리비에 지루</a>와 함께 팀 내 득점 1위를 달리고 있다. <a href="/wiki/%EB%A7%88%ED%8B%B0%EC%99%B8_%ED%94%8C%EB%9D%BC%EB%AF%B8%EB%8B%88" title="마티외 플라미니">마티외 플라미니</a>가 <a class="mw-redirect" href="/wiki/%EC%95%84%EC%8A%A4%EB%82%A0" title="아스날">아스날</a>에 영입되면서 엄청난 활동량을 갖춘 두 미드필더가 아스날의 상승세를 이끌고 있다. 11-12 시즌에 지적되었던 단점들을 개선했는데 이는 부정확한 패스와 경기 템포 조절 실패, 그리고 드리블에 자신감이 없던 부분이었다. 12-13 시즌 <a href="/wiki/%EB%AF%B8%EC%BC%88_%EC%95%84%EB%A5%B4%ED%85%8C%ED%83%80" title="미켈 아르테타">미켈 아르테타</a>가 수행하던 <a class="mw-redirect" href="/wiki/%EB%94%A5-%EB%9D%BC%EC%9E%89_%ED%94%8C%EB%A0%88%EC%9D%B4%EB%A9%94%EC%9D%B4%EC%BB%A4" title="딥-라잉 플레이메이커">딥-라잉 플레이메이커</a> 역할을 맡으면서 87.3%의 정확도였던 패스 정확도를 88.2%까지 끌어올렸으며, <a href="/wiki/%EB%AF%B8%EC%BC%88_%EC%95%84%EB%A5%B4%ED%85%8C%ED%83%80" title="미켈 아르테타">미켈 아르테타</a>가 부재할 때엔 그가 평균적으로 기록하던 91% 정도의 패스 성공률을 기록하였다. 부상 이후 경기 템포 조절과 미숙한 드리블 실력은 <a href="/wiki/%EC%95%84%EB%A5%B4%EC%84%BC_%EB%B2%B5%EA%B1%B0" title="아르센 벵거">아르센 벵거</a> 감독이 좌우 윙어로 기용하면서 아스날의 템포에 적응 시키는 동시에 드리블에 자신감을 가져오게 하였다. <a href="/wiki/%ED%86%A0%EB%A7%88%EC%8B%9C_%EB%A1%9C%EC%8B%9C%EC%B8%A0%ED%82%A4" title="토마시 로시츠키">토마시 로시츠키</a>때와 마찬가지로 윙어로 변신하는데는 실패하였지만, 미드필더에서의 자신감을 향상시킨 동시에, 탈압박 능력을 기르는데는 큰 도움이 되었다. <a class="mw-redirect" href="/wiki/%EC%9E%89%EA%B8%80%EB%9E%9C%EB%93%9C_%ED%94%84%EB%A6%AC%EB%AF%B8%EC%96%B4%EB%A6%AC%EA%B7%B8" title="잉글랜드 프리미어리그">리그</a>에서의 <a href="/wiki/%EC%84%A0%EB%8D%9C%EB%9E%9C%EB%93%9C_AFC" title="선덜랜드 AFC">선덜랜드 AFC</a>와의 경기에서는 111회의 터치를 기록하였고, 경기 내 양팀 통틀어 최다 패스인 88개의 패스를 하였으며 그 중 81개를 성공시키면서 92%의 성공률을 보였다. 또한 <a href="/wiki/2014%EB%85%84" title="2014년">2014년</a> <a href="/wiki/5%EC%9B%94_18%EC%9D%BC" title="5월 18일">5월 18일</a> 영국 <a href="/wiki/%EC%9B%B8%EB%B8%94%EB%A6%AC_%EC%8A%A4%ED%83%80%EB%94%94%EC%9B%80" title="웸블리 스타디움">웸블리 스타디움</a>에서 열린 <a class="mw-redirect" href="/wiki/%EC%9B%A8%EC%8A%A4%ED%8A%B8%ED%96%84_%EC%9C%A0%EB%82%98%EC%9D%B4%ED%8B%B0%EB%93%9C" title="웨스트햄 유나이티드">웨스트햄 유나이티드</a>와의 2013-14 <a class="mw-redirect" href="/wiki/%EC%9E%89%EA%B8%80%EB%9E%9C%EB%93%9C_FA%EC%BB%B5" title="잉글랜드 FA컵">잉글랜드 FA컵</a> 결승전에서 연장 후반 4분 스코어 2:2상황에서 역전 결승골을 뽑아내며 팀의 9년 무관을 끊는데 큰 공헌을 하였다.</p>

id="cite_ref-.EC.98.AC.ED.95.B4.EC.9D.98_.EC.9C.A0.EB.A7.9D.EC.A3.BC_20-0"><a href="#cite_note-.EC.98.AC.ED.95.B4.EC.9D.98_.EC.9C.A0.EB.A7.9D.EC.A3.BC-20">[20]</a></sup> 전직 웨일스 감독 <a href="/wiki/%EA%B2%8C%EB%A6%AC_%EC%8A%A4%ED%94%BC%EB%93%9C" title="게리 스피드">게리 스피드</a>는 램지를 웨일스의 주장으로 지명하였으며 <a href="/wiki/2011%EB%85%84" title="2011년">2011년</a> <a href="/wiki/3%EC%9B%94_26%EC%9D%BC" title="3월 26일">3월 26일</a>, <a href="/wiki/%EC%9E%89%EA%B8%80%EB%9E%9C%EB%93%9C" title="잉글랜드">잉글랜드</a>전부터 완장을 차고 출전하였다. 이것은 20세 90일로 기존 22세 135일로 마이크 잉글랜드가 1964년에 가지고 있던 기록을 갈아치우며 역대 최연소 웨일스 주장이 된 것이다.<sup class="reference" id="cite_ref-.EB.9E.A8.EC.A7.80_.EC.A3.BC.EC.9E.A5_21-0"><a href="#cite_note-.EB.9E.A8.EC.A7.80_.EC.A3.BC.EC.9E.A5-21">[21]</a></sup> <a href="/wiki/2011%EB%85%84" title="2011년">2011년</a> <a href="/wiki/5%EC%9B%94_27%EC%9D%BC" title="5월 27일">5월 27일</a>, <a href="/wiki/%EB%B6%81%EC%95%84%EC%9D%BC%EB%9E%9C%EB%93%9C" title="북아일랜드">북아일랜드</a>와의 경기에서 주장으로써 첫 골을 기록했다. <a href="/wiki/2012%EB%85%84" title="2012년">2012년</a> <a href="/wiki/10%EC%9B%94" title="10월">10월</a>, 대표팀이 연패를 기록하자, 분위기 쇄신차원에서 <a class="mw-disambig" href="/wiki/%EC%95%A0%EC%8A%90%EB%A6%AC_%EC%9C%8C%EB%A6%AC%EC%97%84%EC%8A%A4" title="애슐리 윌리엄스">애슐리 윌리엄스</a>가 새 주장으로 임명된다. 그러나 램지가 커리어에서 다시 주장에 복귀할 가능성은 매우 높다.</p>
<h3><span class="mw-headline" id="2012.EB.85.84_.ED.95.98.EA.B3.84_.EC.98.AC.EB.A6.BC.ED.94.BD_.EC.B6.95.EA.B5.AC"><a href="/wiki/2012%EB%85%84_%ED%95%98%EA%B3%84_%EC%98%AC%EB%A6%BC%ED%94%BD_%EC%B6%95%EA%B5%AC" title="2012년 하계 올림픽 축구">2012년 하계 올림픽 축구</a></span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%97%90%EB%9F%B0_%EB%9E%A8%EC%A7%80&amp;action=edit&amp;section=15" title="부분 편집: 2012년 하계 올림픽 축구">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>웨일스를 대표한 5명의 선수로 <a href="/wiki/%EC%98%81%EA%B5%AD" title="영국">영국</a> 단일 팀에 참가한 올림픽 8강전이었던 <a href="/wiki/%EB%8C%80%ED%95%9C%EB%AF%BC%EA%B5%AD" title="대한민국">대한민국</a>전에서 페널티킥으로 3회 상대했다. 전반 2차례 <a href="/wiki/%EC%A0%95%EC%84%B1%EB%A3%A1" title="정성룡">정성룡</a>을 상대로 1회 성공, 1회 실축했고 승부차기에서는 <a href="/wiki/%EC%9D%B4%EB%B2%94%EC%98%81" title="이범영">이범영</a>을 상대로 성공했다. 그러나 영국 단일 팀은 승부차기 끝에 패했다.</p>
