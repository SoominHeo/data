<p><b>에른스트 폰 프라이베르크</b> (Ernst von Freyberg, <a href="/wiki/1958%EB%85%84" title="1958년">1958년</a> <a href="/wiki/10%EC%9B%94_26%EC%9D%BC" title="10월 26일">10월 26일</a> ~ )는 <a href="/wiki/%EC%A0%9C%EB%84%A4%EB%B0%94" title="제네바">제네바</a>에서 출생한 <a href="/wiki/%EB%8F%85%EC%9D%BC" title="독일">독일</a>의 <a class="new" href="/w/index.php?title=%EA%B2%BD%EC%98%81%EC%9D%B8&amp;action=edit&amp;redlink=1" title="경영인 (없는 문서)">경영인</a>이자 <a class="mw-redirect" href="/wiki/%EC%9D%80%ED%96%89%EA%B0%80" title="은행가">은행가</a>이다. 2013년 2월 15일부터 <a class="new" href="/w/index.php?title=%EC%A2%85%EA%B5%90_%EC%82%AC%EC%97%85_%ED%98%91%ED%9A%8C&amp;action=edit&amp;redlink=1" title="종교 사업 협회 (없는 문서)">종교 사업 협회</a>(IOR, 통칭 바티칸 은행)의 총재로 임명되어 현재 총재직을 수행하고 있다.<sup class="reference" id="cite_ref-1"><a href="#cite_note-1">[1]</a></sup></p>
<p>에른스트 폰 프라이베르크는 <a href="/wiki/%EA%B8%B0%ED%98%BC" title="기혼">기혼</a>이고,<sup class="reference" id="cite_ref-2"><a href="#cite_note-2">[2]</a></sup> <a href="/wiki/%EB%AA%B0%ED%83%80_%EA%B8%B0%EC%82%AC%EB%8B%A8" title="몰타 기사단">몰타 기사단</a>의 회원이다.</p>
