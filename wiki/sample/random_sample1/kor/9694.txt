<p><b>에리스</b>(임시 명칭 2003 UB<sub>313</sub>)는 <a class="mw-redirect" href="/wiki/%EC%82%B0%EB%9E%80_%EB%B6%84%ED%8F%AC%EB%8C%80" title="산란 분포대">산란 분포대</a>에서 발견된 <a href="/wiki/%EC%99%9C%ED%96%89%EC%84%B1" title="왜행성">왜행성</a>으로 <a href="/wiki/%EC%BA%98%EB%A6%AC%ED%8F%AC%EB%8B%88%EC%95%84_%EC%A3%BC" title="캘리포니아 주">캘리포니아 주</a> <a href="/wiki/%ED%8C%94%EB%A1%9C%EB%A7%88_%EC%B2%9C%EB%AC%B8%EB%8C%80" title="팔로마 천문대">팔로마 천문대</a>의 <a href="/wiki/%EC%B2%9C%EB%AC%B8%ED%95%99%EC%9E%90" title="천문학자">천문학자</a> <a href="/wiki/%EB%A7%88%EC%9D%B4%ED%81%B4_E._%EB%B8%8C%EB%9D%BC%EC%9A%B4" title="마이클 E. 브라운">마이클 E. 브라운</a>이 발견하였다. 2006년 발견 당시에 <a class="mw-redirect" href="/wiki/%ED%97%88%EB%B8%94_%EB%A7%9D%EC%9B%90%EA%B2%BD" title="허블 망원경">허블 망원경</a>의 관측 결과를 토대로 추산한 지름은 약 2400km로 <a href="/wiki/%EB%AA%85%EC%99%95%EC%84%B1" title="명왕성">명왕성</a>보다 약간 더 컸다.</p>
<p>에리스의 크기가 밝혀지자 <a href="/wiki/%ED%83%9C%EC%96%91%EA%B3%84" title="태양계">태양계</a>의 <a class="mw-redirect" href="/wiki/%EC%97%B4_%EB%B2%88%EC%A7%B8_%ED%96%89%EC%84%B1" title="열 번째 행성">열 번째 행성</a>으로 인정해야 한다는 주장도 나왔다. 그러나 비슷한 크기의 천체가 계속 발견되면서 <a href="/wiki/%EA%B5%AD%EC%A0%9C%EC%B2%9C%EB%AC%B8%EC%97%B0%EB%A7%B9" title="국제천문연맹">국제천문연맹</a>은 <a href="/wiki/%ED%96%89%EC%84%B1" title="행성">행성</a>의 정의를 보다 엄밀하게 할 필요성을 느꼈고, <a href="/wiki/2006%EB%85%84" title="2006년">2006년</a> <a href="/wiki/8%EC%9B%94_24%EC%9D%BC" title="8월 24일">8월 24일</a> 통과된 새 정의에 의해 에리스는 <a href="/wiki/%EB%AA%85%EC%99%95%EC%84%B1" title="명왕성">명왕성</a> 등과 함께 "<a href="/wiki/%EC%99%9C%ED%96%89%EC%84%B1" title="왜행성">왜행성</a>"으로 분류되었다.</p>
<p>태양에서 에리스까지의 거리를 a×10<span style="display:none">^</span><sup>n</sup> 형식으로 표현하면(km 기준) 102×10<span style="display:none">^</span><sup>8</sup> km 정도 된다.</p>

<p>에리스는 <a href="/wiki/2005%EB%85%84" title="2005년">2005년</a> <a href="/wiki/1%EC%9B%94_5%EC%9D%BC" title="1월 5일">1월 5일</a>, <a href="/wiki/%EB%A7%88%EC%9D%B4%ED%81%B4_E._%EB%B8%8C%EB%9D%BC%EC%9A%B4" title="마이클 E. 브라운">마이클 E. 브라운</a>, <a class="new" href="/w/index.php?title=%EC%B0%A8%EB%93%9C_%ED%8A%B8%EB%A3%A8%EC%A7%88%EB%A1%9C&amp;action=edit&amp;redlink=1" title="차드 트루질로 (없는 문서)">차드 트루질로</a>, <a class="new" href="/w/index.php?title=%EB%8D%B0%EC%9D%B4%EB%B9%84%EB%93%9C_%EB%9D%BC%EB%B9%84%EB%85%B8%EC%9C%84%EC%B8%A0&amp;action=edit&amp;redlink=1" title="데이비드 라비노위츠 (없는 문서)">데이비드 라비노위츠</a>로 이뤄진 팀이 <a href="/wiki/2003%EB%85%84" title="2003년">2003년</a> <a href="/wiki/10%EC%9B%94_23%EC%9D%BC" title="10월 23일">10월 23일</a> 촬영된 사진에서 발견했다. 발견 사실은 <a href="/wiki/2005%EB%85%84" title="2005년">2005년</a> <a href="/wiki/7%EC%9B%94_29%EC%9D%BC" title="7월 29일">7월 29일</a>, 다른 <a href="/wiki/%ED%95%B4%EC%99%95%EC%84%B1_%EB%B0%94%EA%B9%A5_%EC%B2%9C%EC%B2%B4" title="해왕성 바깥 천체">해왕성 바깥 천체</a>인 <a class="mw-redirect" href="/wiki/(136108)_2003_EL61" title="(136108) 2003 EL61">(136108) 2003 EL<sub>61</sub></a>, <a href="/wiki/%EB%A7%88%EC%BC%80%EB%A7%88%EC%BC%80_(%EC%99%9C%ED%96%89%EC%84%B1)" title="마케마케 (왜행성)">마케마케</a>와 함께 발표되었다. 이 연구 팀은 이미 여러 대형 천체인 <a href="/wiki/50000_%EC%BD%B0%EC%98%A4%EC%95%84" title="50000 콰오아">50000 콰오아</a>, <a href="/wiki/90482_%EC%98%A4%EB%A5%B4%EC%BF%A0%EC%8A%A4" title="90482 오르쿠스">90482 오르쿠스</a>, <a href="/wiki/90377_%EC%84%B8%EB%93%9C%EB%82%98" title="90377 세드나">90377 세드나</a>등을 발견하기도 했다.</p>
<p><a href="/wiki/%ED%8C%94%EB%A1%9C%EB%A7%88_%EC%B2%9C%EB%AC%B8%EB%8C%80" title="팔로마 천문대">팔로마 천문대</a>의 48인치 Samuel Oschin <a href="/wiki/%EB%B0%98%EC%82%AC_%EB%A7%9D%EC%9B%90%EA%B2%BD" title="반사 망원경">반사 망원경</a>의 일상 관측 업무가 연구팀에 할당되었다. 느린 이동 속도로 인해 사진 속 천체는 바로 발견되지 않았다. 연구팀은 자동 화상 검색 소프트웨어를 사용, 시간당 1.5 <a href="/wiki/%EC%B4%88_(%EA%B0%81%EB%8F%84)" title="초 (각도)">각초</a>보다 느리게 움직이는 천체를 제거하여 오인식 가능성을 줄였다. 1.75 arcsec/hr의 느린 속도로 움직이는 90377 세드나가 발견되자, 연구팀은 예전 자료를 다시 뒤져 각운동 속도가 느린 천체를 다시 점검하였다. 2005년 1월 재검사에서 에리스의 느린 운동이 발견되었다.</p>
<p>2005년 10월, 위성 <a href="/wiki/%EB%94%94%EC%8A%A4%EB%85%B8%EB%AF%B8%EC%95%84_(%EC%9C%84%EC%84%B1)" title="디스노미아 (위성)">디스노미아</a>의 존재가 밝혀졌다. 과학자들은 이를 이용하여 에리스의 질량을 알아내려 하고 있다.</p>
<h2><span class="mw-headline" id=".EC.9D.B4.EB.A6.84">이름</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%97%90%EB%A6%AC%EC%8A%A4_(%EC%99%9C%ED%96%89%EC%84%B1)&amp;action=edit&amp;section=2" title="부분 편집: 이름">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<div class="thumb tright">
<div class="thumbinner" style="width:222px;"><a class="image" href="/wiki/%ED%8C%8C%EC%9D%BC:Eris_Antikensammlung_Berlin_F1775.jpg"><img alt="" class="thumbimage" data-file-height="600" data-file-width="560" height="236" src="//upload.wikimedia.org/wikipedia/commons/thumb/7/73/Eris_Antikensammlung_Berlin_F1775.jpg/220px-Eris_Antikensammlung_Berlin_F1775.jpg" srcset="//upload.wikimedia.org/wikipedia/commons/thumb/7/73/Eris_Antikensammlung_Berlin_F1775.jpg/330px-Eris_Antikensammlung_Berlin_F1775.jpg 1.5x, //upload.wikimedia.org/wikipedia/commons/thumb/7/73/Eris_Antikensammlung_Berlin_F1775.jpg/440px-Eris_Antikensammlung_Berlin_F1775.jpg 2x" width="220"/></a>
<div class="thumbcaption">
<div class="magnify"><a class="internal" href="/wiki/%ED%8C%8C%EC%9D%BC:Eris_Antikensammlung_Berlin_F1775.jpg" title="실제 크기로"></a></div>
에리스 (기원전 520년 경)</div>
</div>
</div>
<p>에리스는 그리스 신화의 분쟁과 불화의 여신인 <a href="/wiki/%EC%97%90%EB%A6%AC%EC%8A%A4_(%EC%8B%A0%ED%99%94)" title="에리스 (신화)">에리스</a>(<span style="font-size: smaller;"><a href="/wiki/%EA%B7%B8%EB%A6%AC%EC%8A%A4%EC%96%B4" title="그리스어">그리스어</a>:</span> <span lang="el" xml:lang="el">Ἔρις</span>)의 이름을 땄다.<a class="external autonumber" href="http://astrogeology.usgs.gov/HotTopics/index.php?/archives/211-2003-UB313-named-Eris.html" rel="nofollow">[1]</a> 에리스란 이름이 <a href="/wiki/2006%EB%85%84" title="2006년">2006년</a> <a href="/wiki/9%EC%9B%94_13%EC%9D%BC" title="9월 13일">9월 13일</a> 공식 결정되기까지 오랜 기간 2003 UB<sub>313</sub>라는 임시 이름이 쓰였다. 공식 이름 결정이 오래 걸린 것은 에리스가 <a href="/wiki/%ED%96%89%EC%84%B1" title="행성">행성</a>의 이름을 가질 것인지 <a href="/wiki/%EC%86%8C%ED%96%89%EC%84%B1" title="소행성">소행성</a>이나 다른 <a href="/wiki/%EC%99%9C%ED%96%89%EC%84%B1" title="왜행성">왜행성</a>의 이름을 가질 것인지가 우선 정해졌어야 하기 때문이다<sup class="reference" id="cite_ref-1"><a href="#cite_note-1">[1]</a></sup> <a href="/wiki/2006%EB%85%84" title="2006년">2006년</a> <a href="/wiki/8%EC%9B%94_24%EC%9D%BC" title="8월 24일">8월 24일</a> 국제 천문 연맹이 행성의 정의를 바꾸면서 에리스를 <a href="/wiki/%EC%99%9C%ED%96%89%EC%84%B1" title="왜행성">왜행성</a>으로 지정하면서 곧 공식 이름도 주어졌다<sup class="reference" id="cite_ref-_2-0"><a href="#cite_note--2">[2]</a></sup> 이 이름에는 에리스의 정체에 대한 천문학 공동체 사이에 벌어진 분쟁을 얼마간 반영하고 있다.<sup class="reference" id="cite_ref-namechange1_3-0"><a href="#cite_note-namechange1-3">[3]</a></sup></p>
<p>이전까지 언론 등에서는 다음 비공식 이름으로 언급되기도 했다.</p>
<ul>
<li>"릴라"(Lila) - 발견을 발표했던 웹 페이지의 <a href="/wiki/URL" title="URL">URL</a> 중에 포함되었던 "planetlila"에서 생긴 오해로, 마이클 E. 브라운이 당시 얻은 딸 릴라(Lilah)에서 왔다.</li>
<li>"제나"(Xena) - 연구팀 내에서 사용되던 비공식 이름으로 인기 TV 시리즈였던 《<a class="new" href="/w/index.php?title=%EC%97%AC%EC%A0%84%EC%82%AC_%EC%A0%9C%EB%82%98&amp;action=edit&amp;redlink=1" title="여전사 제나 (없는 문서)">여전사 제나</a>》에서 왔다.</li>
</ul>
<p>이 이름들은 국제 천문 연맹에 제안되지는 않았다. 브라운은 <a href="/wiki/%ED%8E%98%EB%A5%B4%EC%84%B8%ED%8F%AC%EB%84%A4" title="페르세포네">페르세포네</a>(이미 소행성 <a class="new" href="/w/index.php?title=399_%ED%8E%98%EB%A5%B4%EC%84%B8%ED%8F%AC%EB%84%A4&amp;action=edit&amp;redlink=1" title="399 페르세포네 (없는 문서)">399 페르세포네</a>에 쓰였다)를 주장하기도 하였으나, 결국 에리스란 이름을 제안하였으며 <a href="/wiki/2006%EB%85%84" title="2006년">2006년</a> <a href="/wiki/9%EC%9B%94_13%EC%9D%BC" title="9월 13일">9월 13일</a> 공식 채택되었다.<sup class="reference" id="cite_ref-4"><a href="#cite_note-4">[4]</a></sup></p>
