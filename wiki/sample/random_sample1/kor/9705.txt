<p><b>에무카에 정</b>(<span style="font-size: smaller;"><a href="/wiki/%EC%9D%BC%EB%B3%B8%EC%96%B4" title="일본어">일본어</a>:</span> <span lang="ja" xml:lang="ja">江迎町</span>)은 <a href="/wiki/%EC%9D%BC%EB%B3%B8" title="일본">일본</a> <a href="/wiki/%EB%82%98%EA%B0%80%EC%82%AC%ED%82%A4_%ED%98%84" title="나가사키 현">나가사키 현</a> 북부에 있는 <a href="/wiki/%EA%B8%B0%ED%83%80%EB%A7%88%EC%93%B0%EC%9A%B0%EB%9D%BC_%EA%B5%B0" title="기타마쓰우라 군">기타마쓰우라 군</a>의 정이다. 면적은 32.07km²이고 인구는 <a href="/wiki/2010%EB%85%84" title="2010년">2010년</a> <a href="/wiki/2%EC%9B%94_1%EC%9D%BC" title="2월 1일">2월 1일</a> 기준으로 5,838명이다. <a href="/wiki/2010%EB%85%84" title="2010년">2010년</a> <a href="/wiki/3%EC%9B%94_31%EC%9D%BC" title="3월 31일">3월 31일</a>에 <a href="/wiki/%EC%8B%9C%EC%B9%B4%EB%A7%88%EC%B9%98_%EC%A0%95" title="시카마치 정">시카마치 정</a>과 함께 <a href="/wiki/%EC%82%AC%EC%84%B8%EB%B3%B4_%EC%8B%9C" title="사세보 시">사세보 시</a>에 편입되었다.</p>

<h2><span class="mw-headline" id=".EC.A7.80.EB.A6.AC">지리</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%97%90%EB%AC%B4%EC%B9%B4%EC%97%90_%EC%A0%95&amp;action=edit&amp;section=1" title="부분 편집: 지리">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>나가사키 현 본토 지역의 최북부, 기타마쓰우라 반도의 북부를 정역으로 하고 <a href="/wiki/%EC%82%AC%EC%84%B8%EB%B3%B4_%EC%8B%9C" title="사세보 시">사세보 시</a> 시가지에서 북서쪽으로 약 20km 떨어진 곳에 위치한다. 정역은 에무카에 강이 동쪽에서 서쪽에 횡단하고 북서부는 반도로 돌진한 에무카에 만에 접한다. 북부·남부는 구릉지가 펼쳐져 있다.</p>
<h3><span class="mw-headline" id=".EC.9D.B8.EC.A0.91.ED.95.9C_.EC.9E.90.EC.B9.98.EC.B2.B4">인접한 자치체</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%97%90%EB%AC%B4%EC%B9%B4%EC%97%90_%EC%A0%95&amp;action=edit&amp;section=2" title="부분 편집: 인접한 자치체">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<ul>
<li><a href="/wiki/%EC%82%AC%EC%84%B8%EB%B3%B4_%EC%8B%9C" title="사세보 시">사세보 시</a></li>

<li><a href="/wiki/%ED%9E%88%EB%9D%BC%EB%8F%84_%EC%8B%9C" title="히라도 시">히라도 시</a></li>
<li><a href="/wiki/%EA%B8%B0%ED%83%80%EB%A7%88%EC%93%B0%EC%9A%B0%EB%9D%BC_%EA%B5%B0" title="기타마쓰우라 군">기타마쓰우라 군</a> <a href="/wiki/%EC%8B%9C%EC%B9%B4%EB%A7%88%EC%B9%98_%EC%A0%95" title="시카마치 정">시카마치 정</a>·<a href="/wiki/%EC%82%AC%EC%9E%90_%EC%A0%95" title="사자 정">사자 정</a></li>

<h2><span class="mw-headline" id=".EC.97.AD.EC.82.AC">역사</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%97%90%EB%AC%B4%EC%B9%B4%EC%97%90_%EC%A0%95&amp;action=edit&amp;section=3" title="부분 편집: 역사">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>에무카에는 예부터 교통의 요충지이며 <a href="/wiki/%EC%97%90%EB%8F%84_%EC%8B%9C%EB%8C%80" title="에도 시대">에도 시대</a>에는 <a href="/wiki/%ED%9E%88%EB%9D%BC%EB%8F%84_%EB%B2%88" title="히라도 번">히라도 번</a>의 본진이 놓여 히라도 가도의 <a href="/wiki/%EC%8A%88%EC%BF%A0%EB%B0%94" title="슈쿠바">역참 마을</a>로서 번창했다.</p>
<ul>
<li>1940년 4월 1일 - 에무카에 촌이 정으로 승격해 에무카에 정이 되었다.</li>
</ul>
<h2><span class="mw-headline" id=".EC.82.B0.EC.97.85">산업</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%97%90%EB%AC%B4%EC%B9%B4%EC%97%90_%EC%A0%95&amp;action=edit&amp;section=4" title="부분 편집: 산업">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>기타마쓰우라 반도에서는 반도내의 각 지역에서 석탄이 생산되어 호쿠쇼 탄전으로서 번창했다. 그 때문에 정내에도 여러 개의 탄광이 있었지만 1960년대에 모두 폐광해 현재에는 농업이 중심이다.</p>
