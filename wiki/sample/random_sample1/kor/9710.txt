<p><b>에밀리아노 사파타 살라사르</b>(<span style="font-size: smaller;"><a href="/wiki/%EC%8A%A4%ED%8E%98%EC%9D%B8%EC%96%B4" title="스페인어">스페인어</a>:</span> <span lang="es" xml:lang="es">Emiliano Zapata Salazar</span>, <a href="/wiki/1879%EB%85%84" title="1879년">1879년</a> <a href="/wiki/8%EC%9B%94_8%EC%9D%BC" title="8월 8일">8월 8일</a>~<a href="/wiki/1919%EB%85%84" title="1919년">1919년</a> <a href="/wiki/4%EC%9B%94_10%EC%9D%BC" title="4월 10일">4월 10일</a>)는 <a href="/wiki/%ED%8F%AC%EB%A5%B4%ED%94%BC%EB%A6%AC%EC%98%A4_%EB%94%94%EC%95%84%EC%8A%A4" title="포르피리오 디아스">포르피리오 디아스</a>의 <a href="/wiki/%EB%8F%85%EC%9E%AC" title="독재">독재</a> 정권에 대항하여 <a href="/wiki/1910%EB%85%84" title="1910년">1910년</a> 시작한 <a href="/wiki/%EB%A9%95%EC%8B%9C%EC%BD%94_%ED%98%81%EB%AA%85" title="멕시코 혁명">멕시코 혁명</a>의 지도자였다. 그는 <a href="/wiki/%EB%A9%95%EC%8B%9C%EC%BD%94" title="멕시코">멕시코</a>의 국민적 영웅 중 하나로 칭송된다. 전국적으로 도시와 길거리 그리고 심지어 주거 개발 프로젝트와 통화에 이르기까지 그의 이름은 폭넓게 사용된다. 그의 파란만장한 삶은 <a href="/wiki/1952%EB%85%84" title="1952년">1952년</a> <a href="/wiki/%EC%97%98%EB%A6%AC%EC%95%84_%EC%B9%B4%EC%9E%94" title="엘리아 카잔">엘리아 카잔</a> 감독, <a href="/wiki/%EB%A7%90%EB%A1%A0_%EB%B8%8C%EB%9E%80%EB%8F%84" title="말론 브란도">말론 브란도</a> 주연의 영화 <a href="/wiki/%ED%98%81%EB%AA%85%EC%95%84_%EC%9E%90%ED%8C%8C%ED%83%80" title="혁명아 자파타">혁명아 자파타</a>에 잘 나타나 있다.</p>
