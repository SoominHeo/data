<p><b>에밋 킹</b>(Emmit King, <a href="/wiki/1959%EB%85%84" title="1959년">1959년</a> <a href="/wiki/3%EC%9B%94_24%EC%9D%BC" title="3월 24일">3월 24일</a> ~ )은 전 <a href="/wiki/%EB%AF%B8%EA%B5%AD" title="미국">미국</a>의 <a class="mw-redirect" href="/wiki/%EB%8B%A8%EA%B1%B0%EB%A6%AC%EB%8B%AC%EB%A6%AC%EA%B8%B0" title="단거리달리기">단거리달리기</a> 선수로 2번이나 <a href="/wiki/%ED%95%98%EA%B3%84_%EC%98%AC%EB%A6%BC%ED%94%BD" title="하계 올림픽">하계 올림픽</a>(<a href="/wiki/1984%EB%85%84" title="1984년">1984년</a>과 <a href="/wiki/1988%EB%85%84" title="1988년">1988년</a>)을 위하여 미국 릴레이 팀의 멤버로 선발되었지만 나가지 않았다.</p>
<p><a href="/wiki/%EC%95%A8%EB%9D%BC%EB%B0%B0%EB%A7%88_%EC%A3%BC" title="앨라배마 주">앨라배마 주</a> <a href="/wiki/%EB%B2%A0%EC%84%9C%EB%A8%B8_(%EC%95%A8%EB%9D%BC%EB%B0%B0%EB%A7%88_%EC%A3%BC)" title="베서머 (앨라배마 주)">베서머</a>에서 태어난 킹은 <a href="/wiki/1983%EB%85%84_%EC%84%B8%EA%B3%84_%EC%9C%A1%EC%83%81_%EC%84%A0%EC%88%98%EA%B6%8C_%EB%8C%80%ED%9A%8C" title="1983년 세계 육상 선수권 대회">1983년 세계 육상 선수권 대회</a>에서 100m 경주 <a href="/wiki/%EB%8F%99%EB%A9%94%EB%8B%AC" title="동메달">동메달</a>을 획득하기로 가장 잘 알려졌다.</p>
<p>그는 <a href="/wiki/1986%EB%85%84" title="1986년">1986년</a> <a href="/wiki/6%EC%9B%94_17%EC%9D%BC" title="6월 17일">6월 17일</a> <a href="/wiki/%ED%94%8C%EB%A1%9C%EB%A6%AC%EB%8B%A4_%EC%A3%BC" title="플로리다 주">플로리다 주</a> <a href="/wiki/%ED%83%AC%ED%8C%8C" title="탬파">탬파</a>에서 열린 대회에서 100m의 개인 전력 10.4초를 세웠다.</p>
<p><a href="/wiki/%EC%95%A8%EB%9D%BC%EB%B0%B0%EB%A7%88_%EB%8C%80%ED%95%99%EA%B5%90" title="앨라배마 대학교">앨라배마 대학교</a>에서 수학하는 동안에 키은 10.15초와 함께 <a href="/wiki/1983%EB%85%84" title="1983년">1983년</a> NCAA 100m 국내 챔피언이 되었다.</p>
