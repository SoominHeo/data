<p><b>에버랜드 리조트</b>(Everland Resort)는 <a href="/wiki/%EB%8C%80%ED%95%9C%EB%AF%BC%EA%B5%AD" title="대한민국">대한민국</a> <a href="/wiki/%EA%B2%BD%EA%B8%B0%EB%8F%84" title="경기도">경기도</a> <a href="/wiki/%EC%9A%A9%EC%9D%B8%EC%8B%9C" title="용인시">용인시</a> <a href="/wiki/%ED%96%A5%EC%88%98%EC%82%B0_(%EA%B2%BD%EA%B8%B0)" title="향수산 (경기)">향수산</a> 자락에 있는 <a href="/wiki/%ED%85%8C%EB%A7%88%ED%8C%8C%ED%81%AC" title="테마파크">테마파크</a>, 워터파크, 동물원 등을 갖춘 대단위 리조트이다.</p>
<p><a href="/wiki/1976%EB%85%84" title="1976년">1976년</a> <b>용인자연농원</b>(龍仁自然農園)으로 처음 개장한 이래<sup class="reference" id="cite_ref-1"><a href="#cite_note-1">[1]</a></sup> <a href="/wiki/%EB%8C%80%ED%95%9C%EB%AF%BC%EA%B5%AD" title="대한민국">대한민국</a>의 테마파크 중 가장 큰 규모를 자랑한다.<sup class="reference" id="cite_ref-2"><a href="#cite_note-2">[2]</a></sup> 주요 시설물로는 <a href="/wiki/%ED%85%8C%EB%A7%88%ED%8C%8C%ED%81%AC" title="테마파크">테마파크</a> <i>에버랜드</i>, 워터파크 ‘<a href="/wiki/%EC%BA%90%EB%A6%AC%EB%B9%84%EC%95%88_%EB%B2%A0%EC%9D%B4" title="캐리비안 베이">캐리비안 베이</a>′, 숙박시설 <i>홈브리지</i> 등이 있다. <a href="/wiki/1996%EB%85%84" title="1996년">1996년</a>에 에버랜드로 BI를 변경하였고, <a href="/wiki/2006%EB%85%84" title="2006년">2006년</a>에 에버랜드 리조트로 BI를 다시 변경하였다.<sup class="reference" id="cite_ref-3"><a href="#cite_note-3">[3]</a></sup></p>
<p>기업집단 <a href="/wiki/%EC%82%BC%EC%84%B1%EA%B7%B8%EB%A3%B9" title="삼성그룹">삼성</a>의 계열사인 <a href="/wiki/%EC%A0%9C%EC%9D%BC%EB%AA%A8%EC%A7%81" title="제일모직">삼성에버랜드㈜</a>가 운영하여 왔으며, 2014년 7월에 운영사인 삼성에버랜드가 제일모직㈜으로 상호를 변경함에 따라<sup class="reference" id="cite_ref-4"><a href="#cite_note-4">[4]</a></sup> 에버랜드 리조트는 기존 <b>삼성에버랜드㈜</b>에서 <b>제일모직 (주)에버랜드 리조트</b>로 상호를 변경하였다. 제일모직 (주)에버랜드 리조트는 기존 <b>제일모직㈜</b>에서 <b>삼성물산 (주)에버랜드 리조트</b>로 상호를 변경하였다.</p>
<p><a href="/wiki/2013%EB%85%84" title="2013년">2013년</a> <a href="/wiki/4%EC%9B%94_26%EC%9D%BC" title="4월 26일">4월 26일</a> <a href="/wiki/%EC%9A%A9%EC%9D%B8_%EA%B2%BD%EC%A0%84%EC%B2%A0" title="용인 경전철">용인 경전철</a>(에버라인)이 <a href="/wiki/%EB%B6%84%EB%8B%B9%EC%84%A0" title="분당선">분당선</a> <a href="/wiki/%EA%B8%B0%ED%9D%A5%EC%97%AD" title="기흥역">기흥역</a>부터 <a href="/wiki/%EC%A0%84%EB%8C%80%C2%B7%EC%97%90%EB%B2%84%EB%9E%9C%EB%93%9C%EC%97%AD" title="전대·에버랜드역">전대·에버랜드역</a>까지 개통되었다.<sup class="reference" id="cite_ref-5"><a href="#cite_note-5">[5]</a></sup> 또한 <a href="/wiki/2013%EB%85%84" title="2013년">2013년</a> <a href="/wiki/11%EC%9B%94_30%EC%9D%BC" title="11월 30일">11월 30일</a> <a href="/wiki/%EB%B6%84%EB%8B%B9%EC%84%A0" title="분당선">분당선</a>이 <a href="/wiki/%EB%A7%9D%ED%8F%AC%EC%97%AD" title="망포역">망포역</a>에서 <a href="/wiki/%EC%88%98%EC%9B%90%EC%97%AD" title="수원역">수원역</a>까지 연장 개통 되었다.<sup class="reference" id="cite_ref-6"><a href="#cite_note-6">[6]</a></sup> 따라서 <a href="/wiki/%EA%B2%BD%EB%B6%80%EC%84%A0" title="경부선">1호선</a> <a href="/wiki/%EC%88%98%EC%9B%90%EC%97%AD" title="수원역">수원역</a>에서 에버랜드를 대중교통으로 가장 빠르게 가려면 <a href="/wiki/%EB%B6%84%EB%8B%B9%EC%84%A0" title="분당선">분당선</a>을 타고 <a href="/wiki/%EA%B8%B0%ED%9D%A5%EC%97%AD" title="기흥역">기흥역</a>에서 하차하여 <a href="/wiki/%EC%9A%A9%EC%9D%B8_%EA%B2%BD%EC%A0%84%EC%B2%A0" title="용인 경전철">용인 경전철</a>로 환승 후 <a href="/wiki/%EC%A0%84%EB%8C%80%C2%B7%EC%97%90%EB%B2%84%EB%9E%9C%EB%93%9C%EC%97%AD" title="전대·에버랜드역">전대·에버랜드역</a>에서 하차하면 된다. 이렇게 가면 수원역에서 에버랜드 까지 약 50분 소요된다. 수원에서 66번이나 66-4번 버스를 타고 에버랜드로 갈 수도 있지만, 수원시내와 신갈, 용인시내를 통과하면서 소요시간이 오래걸리는 편이다. 또한, <a href="/wiki/%EA%B0%95%EB%82%A8%EC%97%AD" title="강남역">강남역</a>에서는 5002번 버스를, <a href="/wiki/%EC%9E%A0%EC%8B%A4%EC%97%AD" title="잠실역">잠실역</a> 또는 <a href="/wiki/%EA%B0%95%EB%B3%80%EC%97%AD" title="강변역">강변역</a>, <a href="/wiki/%EC%88%98%EC%84%9C%EC%97%AD" title="수서역">수서역</a>, <a href="/wiki/%EA%B0%80%EB%9D%BD%EC%8B%9C%EC%9E%A5%EC%97%AD" title="가락시장역">가락시장역</a>에서는 5700번 버스를 타면 에버랜드로 올 수 있다. <a href="/wiki/%EC%84%9C%ED%98%84%EC%97%AD" title="서현역">서현역</a>에서는 1500-2번 버스가 운행되고 있지만, 2018년 하반기에 <a href="/wiki/%EC%88%98%EC%9D%B8%EC%84%A0" title="수인선">수인선</a>이 <a href="/wiki/%EB%B6%84%EB%8B%B9%EC%84%A0" title="분당선">분당선</a>과 직결 운행이 되면, <a href="/wiki/%EA%B2%BD%EC%9D%B8%EC%84%A0" title="경인선">1호선</a> <a href="/wiki/%EC%9D%B8%EC%B2%9C%EC%97%AD" title="인천역">인천역</a>에서 <a href="/wiki/%EC%88%98%EC%9D%B8%EC%84%A0" title="수인선">수인선</a>을 타고 <a href="/wiki/%EB%B6%84%EB%8B%B9%EC%84%A0" title="분당선">분당선</a> <a href="/wiki/%EA%B8%B0%ED%9D%A5%EC%97%AD" title="기흥역">기흥역</a>에서 하차하여, <a href="/wiki/%EC%9A%A9%EC%9D%B8_%EA%B2%BD%EC%A0%84%EC%B2%A0" title="용인 경전철">용인 경전철</a>로 환승 후 <a href="/wiki/%EC%A0%84%EB%8C%80%C2%B7%EC%97%90%EB%B2%84%EB%9E%9C%EB%93%9C%EC%97%AD" title="전대·에버랜드역">전대·에버랜드역</a>에서 하차하면 된다. 이렇게 가면 인천역에서 에버랜드 까지 약 2시간 소요된다.</p>
