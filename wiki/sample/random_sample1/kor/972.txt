<p><b>FC 빅토리아 플젠</b>(<span style="font-size: smaller;"><a href="/wiki/%EC%B2%B4%EC%BD%94%EC%96%B4" title="체코어">체코어</a>:</span> <span lang="cs" xml:lang="cs">FC Viktoria Plzeň</span>)은 <a href="/wiki/%ED%94%8C%EC%A0%A0" title="플젠">플젠</a>을 연고로 하는 <a href="/wiki/%EC%B2%B4%EC%BD%94" title="체코">체코</a>의 축구 클럽이다. 이 팀은 <a href="/wiki/1911%EB%85%84" title="1911년">1911년</a>에 창단되었으며, 현재 <a class="mw-redirect" href="/wiki/%EA%B0%90%EB%B8%8C%EB%A6%AC%EB%88%84%EC%8A%A4%EB%A6%AC%EA%B0%80" title="감브리누스리가">감브리누스리가</a>에 출전하고 있다.</p>
<p>1970-71 <a class="new" href="/w/index.php?title=%EC%B2%B4%EC%BD%94%EC%8A%AC%EB%A1%9C%EB%B0%94%ED%82%A4%EC%95%84_%EC%BB%B5&amp;action=edit&amp;redlink=1" title="체코슬로바키아 컵 (없는 문서)">체코슬로바키아 컵</a>에서 준우승을 차지하면서 다음 시즌에 열린 <a class="mw-redirect" href="/wiki/UEFA_%EC%BB%B5_%EC%9C%84%EB%84%88%EC%8A%A4%EC%BB%B5" title="UEFA 컵 위너스컵">UEFA 컵 위너스컵</a>에 출전했다. 한편 우승 팀인 <a href="/wiki/FC_%EC%8A%A4%ED%8C%8C%EB%A5%B4%ED%83%80%ED%81%AC_%ED%8A%B8%EB%A5%B4%EB%82%98%EB%B0%94" title="FC 스파르타크 트르나바">FC 스파르타크 트르나바</a>는 <a href="/wiki/UEFA_%EC%B1%94%ED%94%BC%EC%96%B8%EC%8A%A4%EB%A6%AC%EA%B7%B8" title="UEFA 챔피언스리그">유러피언컵</a>에 출전했다.</p>
<p>2009-10 <a class="new" href="/w/index.php?title=%EC%B2%B4%EC%BD%94_%EC%BB%B5&amp;action=edit&amp;redlink=1" title="체코 컵 (없는 문서)">체코 컵</a>에서 우승을 차지하면서 <a href="/wiki/UEFA_%EC%9C%A0%EB%A1%9C%ED%8C%8C%EB%A6%AC%EA%B7%B8_2010-11" title="UEFA 유로파리그 2010-11">UEFA 유로파리그 2010-11</a>에 출전했고 2010-11 <a class="mw-redirect" href="/wiki/%EA%B0%90%EB%B8%8C%EB%A6%AC%EB%88%84%EC%8A%A4%EB%A6%AC%EA%B0%80" title="감브리누스리가">감브리누스리가</a>에서 처음으로 우승을 차지하면서 <a href="/wiki/UEFA_%EC%B1%94%ED%94%BC%EC%96%B8%EC%8A%A4%EB%A6%AC%EA%B7%B8_2011-12" title="UEFA 챔피언스리그 2011-12">UEFA 챔피언스리그 2011-12</a>에 처음 출전했다.</p>
<h2><span class="mw-headline" id=".EC.97.AD.EB.8C.80_.ED.81.B4.EB.9F.BD_.EB.AA.85.EC.B9.AD">역대 클럽 명칭</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=FC_%EB%B9%85%ED%86%A0%EB%A6%AC%EC%95%84_%ED%94%8C%EC%A0%A0&amp;action=edit&amp;section=1" title="부분 편집: 역대 클럽 명칭">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
