<p><b>에반 본</b>(<span lang="en" xml:lang="en">Evan Bourne</span>)은 <a href="/wiki/%EB%AF%B8%EA%B5%AD" title="미국">미국</a>의 <a class="mw-redirect" href="/wiki/%ED%94%84%EB%A1%9C%EB%A0%88%EC%8A%AC%EB%9F%AC" title="프로레슬러">프로레슬러</a>이다. <a href="/wiki/WWE" title="WWE">WWE</a>에서 활동하며 현재 <a href="/wiki/%EB%A7%81_%EB%84%A4%EC%9E%84" title="링 네임">링 네임</a>은 '에반 본'이다. <a href="/wiki/WWE" title="WWE">WWE</a>와 계약하기 전에는, <a class="mw-disambig" href="/wiki/IWA" title="IWA">IWA</a>, <a class="mw-redirect" href="/wiki/%EB%A7%81_%EC%98%A4%EB%B8%8C_%EC%98%A4%EB%84%88" title="링 오브 오너">링 오브 오너</a>, <a class="mw-redirect" href="/wiki/%EB%93%9C%EB%9E%98%EA%B3%A4_%EA%B2%8C%EC%9D%B4%ED%8A%B8" title="드래곤 게이트">드래곤 게이트</a> <a class="new" href="/w/index.php?title=Society_X_promotion&amp;action=edit&amp;redlink=1" title="Society X promotion (없는 문서)">Society X promotion</a>, <a class="mw-disambig" href="/wiki/TNA" title="TNA">TNA</a> 등에서 활동 하였다. 또한 에번 본은 높은 <a class="new" href="/w/index.php?title=%EC%8A%88%ED%8C%85_%EC%8A%A4%ED%83%80_%ED%94%84%EB%A0%88%EC%8A%A4&amp;action=edit&amp;redlink=1" title="슈팅 스타 프레스 (없는 문서)">슈팅 스타 프레스</a>와 같은 고공 비행 기술로 유명하다.</p>

<h2><span class="mw-headline" id=".ED.99.9C.EB.8F.99_.EC.82.AC.ED.95.AD">활동 사항</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%97%90%EB%B2%88_%EB%B3%B8&amp;action=edit&amp;section=1" title="부분 편집: 활동 사항">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<h3><span class="mw-headline" id=".EC.B4.88.EA.B8.B0">초기</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%97%90%EB%B2%88_%EB%B3%B8&amp;action=edit&amp;section=2" title="부분 편집: 초기">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>에번 본은 고등학교 시절, <a href="/wiki/%EB%A0%88%EC%8A%AC%EB%A7%81" title="레슬링">레슬링</a>팀에 있었다<sup class="reference" id="cite_ref-slam_2-0"><a href="#cite_note-slam-2">[2]</a></sup>. 특히, 그는 고등학교 3학년 때, <a href="/wiki/%EB%AF%B8%EC%A3%BC%EB%A6%AC_%EC%A3%BC" title="미주리 주">미주리 주</a> <a href="/wiki/%EC%84%B8%EC%9D%B8%ED%8A%B8%EB%A3%A8%EC%8B%9C%EC%95%84" title="세인트루시아">세인트루시아</a>에 있는 <a class="new" href="/w/index.php?title=GCW&amp;action=edit&amp;redlink=1" title="GCW (없는 문서)">GCW</a>에서 프로모션을 하고, 3개월 간 훈련을 하였다. 훈련 후, <a class="new" href="/w/index.php?title=GCW&amp;action=edit&amp;redlink=1" title="GCW (없는 문서)">GCW</a>에서 <a href="/wiki/2000%EB%85%84" title="2000년">2000년</a> <a href="/wiki/10%EC%9B%94_20%EC%9D%BC" title="10월 20일">10월 20일</a> <a href="/wiki/%EB%AF%B8%EC%A3%BC%EB%A6%AC_%EC%A3%BC" title="미주리 주">미주리 주</a>에서 18세 미만인 사람이 레슬링 자격을 받은 최초의 사람이 되었다<sup class="reference" id="cite_ref-3"><a href="#cite_note-3">[3]</a></sup>.</p>
<p><a href="/wiki/2003%EB%85%84" title="2003년">2003년</a>에, 에번 본(이 때부터, Matt라는 이름을 사용)은 'Shamrock'이라는 기구의 <a class="new" href="/w/index.php?title=GCW&amp;action=edit&amp;redlink=1" title="GCW (없는 문서)">GCW</a>에 등록 되었다. 그 후, 에번 본은 <a class="new" href="/w/index.php?title=%EB%B9%8C%EB%A6%AC_%EB%A7%A5%EB%8B%90&amp;action=edit&amp;redlink=1" title="빌리 맥닐 (없는 문서)">빌리 맥닐</a>과 <a class="mw-redirect" href="/wiki/%ED%83%9C%EA%B7%B8_%ED%8C%80" title="태그 팀">태그 팀</a>을 하게 되었다. 하지만, <a href="/wiki/%EC%95%85%EC%97%AD_(%ED%94%84%EB%A1%9C%EB%A0%88%EC%8A%AC%EB%A7%81)" title="악역 (프로레슬링)">악역</a>인 <a class="new" href="/w/index.php?title=Nikki_Strychnine&amp;action=edit&amp;redlink=1" title="Nikki Strychnine (없는 문서)">Nikki Strychnine</a>와 <a href="/wiki/%EB%8C%80%EB%A6%BD_(%ED%94%84%EB%A1%9C%EB%A0%88%EC%8A%AC%EB%A7%81)" title="대립 (프로레슬링)">대립</a>을 하게 된다<sup class="reference" id="cite_ref-4"><a href="#cite_note-4">[4]</a></sup>.</p>
<h3><span class="mw-headline" id="IWA">IWA</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%97%90%EB%B2%88_%EB%B3%B8&amp;action=edit&amp;section=3" title="부분 편집: IWA">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>에번 본은 <a class="new" href="/w/index.php?title=2003%EB%85%84%EC%97%90&amp;action=edit&amp;redlink=1" title="2003년에 (없는 문서)">2003년에</a> <a class="mw-disambig" href="/wiki/IWA" title="IWA">IWA</a>에 데뷔를 한다. 이 때부터 새로운 <a href="/wiki/%EB%A7%81_%EB%84%A4%EC%9E%84" title="링 네임">링 네임</a>인 '맷 사이달'을 사용하게 된다. 그는 <a href="/wiki/2004%EB%85%84" title="2004년">2004년</a> <a href="/wiki/1%EC%9B%94_17%EC%9D%BC" title="1월 17일">1월 17일</a> <a class="mw-disambig" href="/wiki/IWA" title="IWA">IWA</a>에서 라이트 헤비웨이 챔피언십 타이틀을 얻는다<sup class="reference" id="cite_ref-5"><a href="#cite_note-5">[5]</a></sup>. 하지만 같은 해 <a href="/wiki/6%EC%9B%94_26%EC%9D%BC" title="6월 26일">6월 26일</a>, 라이벌인 <a class="new" href="/w/index.php?title=Delirious&amp;action=edit&amp;redlink=1" title="Delirious (없는 문서)">Delirious</a>에게 빼앗긴다<sup class="reference" id="cite_ref-6"><a href="#cite_note-6">[6]</a></sup>. 하지만, 그는 <a class="new" href="/w/index.php?title=Daizee_Haze&amp;action=edit&amp;redlink=1" title="Daizee Haze (없는 문서)">Daizee Haze</a>와 <a class="mw-redirect" href="/wiki/%ED%83%9C%EA%B7%B8_%ED%8C%80" title="태그 팀">태그 팀</a>을 짜서, 다시 되찾는다.</p>
<p>그 후, 그는 <a class="mw-redirect" href="/wiki/%EC%97%90%EC%9D%B4%EC%A0%9C%EC%9D%B4_%EC%8A%A4%ED%83%80%EC%9D%BC" title="에이제이 스타일">에이제이 스타일</a>과 <a href="/wiki/%EB%8C%80%EB%A6%BD_(%ED%94%84%EB%A1%9C%EB%A0%88%EC%8A%AC%EB%A7%81)" title="대립 (프로레슬링)">대립</a>을 하고, 또한 <a href="/wiki/CM_%ED%8E%91%ED%81%AC" title="CM 펑크">CM 펑크</a>, <a class="mw-redirect" href="/wiki/%ED%81%AC%EB%A6%AC%EC%8A%A4_%EC%82%AC%EB%B9%88" title="크리스 사빈">크리스 사빈</a>, <a class="new" href="/w/index.php?title=%EB%84%A4%EC%9D%B4%ED%8A%B8_%EC%9B%B9&amp;action=edit&amp;redlink=1" title="네이트 웹 (없는 문서)">네이트 웹</a>과 같은 사람들과 직면하게 된다. <a href="/wiki/2005%EB%85%84" title="2005년">2005년</a> <a href="/wiki/9%EC%9B%94_24%EC%9D%BC" title="9월 24일">9월 24일</a>, 그는 <a class="new" href="/w/index.php?title=Ted_Petty_Invitational&amp;action=edit&amp;redlink=1" title="Ted Petty Invitational (없는 문서)">Ted Petty Invitational</a> 토너먼트에서 이긴다. 토너먼트에서, 그는 <a class="new" href="/w/index.php?title=El_Generico&amp;action=edit&amp;redlink=1" title="El Generico (없는 문서)">El Generico</a>, <a class="new" href="/w/index.php?title=Tyler_Black&amp;action=edit&amp;redlink=1" title="Tyler Black (없는 문서)">Tyler Black</a>, <a class="new" href="/w/index.php?title=Sabin&amp;action=edit&amp;redlink=1" title="Sabin (없는 문서)">Sabin</a>을 패배시키고, 마지막에 <a class="new" href="/w/index.php?title=Kevin_Steen&amp;action=edit&amp;redlink=1" title="Kevin Steen (없는 문서)">Kevin Steen</a>과 <a class="new" href="/w/index.php?title=Arik_Cannon&amp;action=edit&amp;redlink=1" title="Arik Cannon (없는 문서)">Arik Cannon</a>를 만나지만 결국 이긴다<sup class="reference" id="cite_ref-7"><a href="#cite_note-7">[7]</a></sup>. 그 후, 그는 <a class="mw-disambig" href="/wiki/IWA" title="IWA">IWA</a>에 조금 등장하고, 마지막으로 <a href="/wiki/2007%EB%85%84" title="2007년">2007년</a> <a href="/wiki/8%EC%9B%94" title="8월">8월</a>에 등장한다.</p>
<h3><span class="mw-headline" id="TNA_.282004_.7E_2005.29">TNA (2004 ~ 2005)</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%97%90%EB%B2%88_%EB%B3%B8&amp;action=edit&amp;section=4" title="부분 편집: TNA (2004 ~ 2005)">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>그는, <a class="mw-disambig" href="/wiki/TNA" title="TNA">TNA</a>에서 경기를 하기 시작했는데, 그는 <a class="new" href="/w/index.php?title=X_Division&amp;action=edit&amp;redlink=1" title="X Division (없는 문서)">X Division</a> <a class="new" href="/w/index.php?title=Gauntlet_for_the_Gold&amp;action=edit&amp;redlink=1" title="Gauntlet for the Gold (없는 문서)">Gauntlet for the Gold</a> 경기에 참여 하였다<sup class="reference" id="cite_ref-8"><a href="#cite_note-8">[8]</a></sup>. 하지만 그는 패배하고, 그는 <a class="new" href="/w/index.php?title=%EC%9E%90%EB%B2%84&amp;action=edit&amp;redlink=1" title="자버 (없는 문서)">자버</a>로 전락한다.</p>

<p>그는 <a href="/wiki/2006%EB%85%84" title="2006년">2006년</a> <a href="/wiki/5%EC%9B%94" title="5월">5월</a> '드래곤 게이트'에서 <a href="/wiki/%EC%9D%BC%EB%B3%B8" title="일본">일본</a> 순회를 하게 된다<sup class="reference" id="cite_ref-11"><a href="#cite_note-11">[11]</a></sup>. 그 후, <a class="mw-redirect" href="/wiki/Syxx-Pac" title="Syxx-Pac">Syxx-Pac</a>과 매니저를 사이에 두고 <a href="/wiki/%EB%8C%80%EB%A6%BD_(%ED%94%84%EB%A1%9C%EB%A0%88%EC%8A%AC%EB%A7%81)" title="대립 (프로레슬링)">대립</a>을 하게 된다<sup class="reference" id="cite_ref-12"><a href="#cite_note-12">[12]</a></sup><sup class="reference" id="cite_ref-13"><a href="#cite_note-13">[13]</a></sup>.</p>

<h2><span class="mw-headline" id=".EA.B0.9C.EC.9D.B8_.EC.83.9D.ED.99.9C">개인 생활</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%97%90%EB%B2%88_%EB%B3%B8&amp;action=edit&amp;section=10" title="부분 편집: 개인 생활">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>그의 부모님은 둘다 선생님이다.<sup class="reference" id="cite_ref-37"><a href="#cite_note-37">[37]</a></sup> 그는 <a href="/wiki/%EC%84%B8%EC%9D%B8%ED%8A%B8%EB%A3%A8%EC%8B%9C%EC%95%84" title="세인트루시아">세인트루시아</a>에 있는 <a class="new" href="/w/index.php?title=Parkway_West_High_School&amp;action=edit&amp;redlink=1" title="Parkway West High School (없는 문서)">Parkway West High School</a>을 졸업하였으며, <a href="/wiki/2001%EB%85%84" title="2001년">2001년</a>에는, <a href="/wiki/%EB%AF%B8%EC%A3%BC%EB%A6%AC_%EB%8C%80%ED%95%99%EA%B5%90" title="미주리 대학교">미주리 대학교</a>에 입학하여, <a href="/wiki/%EB%A7%88%EC%BC%80%ED%8C%85" title="마케팅">마케팅</a>을 공부하였지만, <a class="mw-redirect" href="/wiki/%ED%94%84%EB%A1%9C_%EB%A0%88%EC%8A%AC%EB%9F%AC" title="프로 레슬러">프로 레슬러</a>로 되기로 결정한다.<sup class="reference" id="cite_ref-38"><a href="#cite_note-38">[38]</a></sup> 그는 레슬링을 계속 하였고, 대학은 1주일에 3번정도 간다. 그는 <a href="/wiki/2005%EB%85%84" title="2005년">2005년</a> <a href="/wiki/%ED%95%99%EC%82%AC" title="학사">학사</a> 학위를 따고, 마케팅 쪽에서 일하다가 결국 <a class="mw-redirect" href="/wiki/%ED%94%84%EB%A1%9C%EB%A0%88%EC%8A%AC%EB%9F%AC" title="프로레슬러">프로레슬러</a>가 되었다.</p>
