<p><b>FC 샤흐테르</b>(<span style="font-size: smaller;"><a href="/wiki/%EC%B9%B4%EC%9E%90%ED%9D%90%EC%96%B4" title="카자흐어">카자흐어</a>:</span> <span lang="kk" xml:lang="kk">Шахтёр Футбол Клубы</span>)는 <a href="/wiki/%EC%B9%B4%EC%9E%90%ED%9D%90%EC%8A%A4%ED%83%84" title="카자흐스탄">카자흐스탄</a>의 <a href="/wiki/%EC%B6%95%EA%B5%AC" title="축구">축구</a> 클럽으로 현재 <a href="/wiki/%EC%B9%B4%EC%9E%90%ED%9D%90%EC%8A%A4%ED%83%84_%ED%94%84%EB%A6%AC%EB%AF%B8%EC%96%B4%EB%A6%AC%EA%B7%B8" title="카자흐스탄 프리미어리그">카자흐스탄 프리미어리그</a>에서 활동하고 있다. 샤흐테르는 <a href="/wiki/1962%EB%85%84" title="1962년">1962년</a>에 <a href="/wiki/%EC%B9%B4%EC%9E%90%ED%9D%90%EC%8A%A4%ED%83%84" title="카자흐스탄">카자흐스탄</a> 중앙에 위치한 <a class="mw-redirect" href="/wiki/%EC%B9%B4%EB%9D%BC%EA%B0%84%EB%94%94" title="카라간디">카라간디</a>를 연고지로 하여 창단되었다. <a href="/wiki/%EC%B9%B4%EC%9E%90%ED%9D%90%EC%8A%A4%ED%83%84" title="카자흐스탄">카자흐스탄</a> 독립 이전에는 <a class="mw-redirect" href="/wiki/%EC%86%8C%EB%B9%84%EC%97%90%ED%8A%B8_%EC%97%B0%EB%B0%A9" title="소비에트 연방">소비에트 연방</a>에서 리그에 참가하였다. 독립 이후의 최고 성적은 1995시즌과 2007시즌의 3위이다.</p>
<p><a href="/wiki/2008%EB%85%84" title="2008년">2008년</a> <a href="/wiki/9%EC%9B%94_19%EC%9D%BC" title="9월 19일">9월 19일</a>, 샤흐테르와 <a class="new" href="/w/index.php?title=FC_%EB%B3%B4%EC%8A%A4%ED%86%A0%ED%81%AC&amp;action=edit&amp;redlink=1" title="FC 보스토크 (없는 문서)">보스토크</a>는 승부조작으로 <a href="/wiki/%EC%B9%B4%EC%9E%90%ED%9D%90%EC%8A%A4%ED%83%84_%ED%94%84%EB%A6%AC%EB%AF%B8%EC%96%B4%EB%A6%AC%EA%B7%B8" title="카자흐스탄 프리미어리그">카자흐스탄 프리미어리그</a>에서 퇴출되었고, 감독과 코치진은 60개월간 자격 중지를 받았다. <a href="/wiki/2008%EB%85%84" title="2008년">2008년</a> <a href="/wiki/10%EC%9B%94_2%EC%9D%BC" title="10월 2일">10월 2일</a>, <a class="new" href="/w/index.php?title=%EC%B9%B4%EC%9E%90%ED%9D%90%EC%8A%A4%ED%83%84_%EC%B6%95%EA%B5%AC_%ED%98%91%ED%9A%8C&amp;action=edit&amp;redlink=1" title="카자흐스탄 축구 협회 (없는 문서)">카자흐스탄 축구 협회</a>는 결정을 재고하여, 샤흐테르는 승점 9점 차감되고, 보스토크는 강등되었다. 보스토크의 남은 경기는 모두 0-3으로 처리되었다.</p>
