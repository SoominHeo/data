<h1 class="firstHeading" id="firstHeading" lang="ko">에센 타이시</h1>
<div class="mw-body-content" id="bodyContent">
<div id="siteSub">위키백과, 우리 모두의 백과사전.</div>
<div id="contentSub"></div>
<div class="mw-jump" id="jump-to-nav">
					이동:					<a href="#mw-head">둘러보기</a>, 					<a href="#p-search">검색</a>
</div>
<div class="mw-content-ltr" dir="ltr" id="mw-content-text" lang="ko"><script>function mfTempOpenSection(id){var block=document.getElementById("mf-section-"+id);block.className+=" open-block";block.previousSibling.className+=" open-block";}</script><p><b>에센 칸</b>(也先汗, ? ~ <a href="/wiki/1455%EB%85%84" title="1455년">1455년</a>)또는 <b>에센 타이시</b>(也先太師)는 <a href="/wiki/%EC%98%A4%EC%9D%B4%EB%9D%BC%ED%8A%B8" title="오이라트">오이라트</a>의 지도자이자 <a href="/wiki/%EB%B6%81%EC%9B%90" title="북원">북원</a>의 대칸(재위 <a href="/wiki/1453%EB%85%84" title="1453년">1453년</a> ~ <a href="/wiki/1454%EB%85%84" title="1454년">1454년</a>)으로 이전 오이라트의 지도자던 토곤의 아들이다. 휘는 <b>오이라트 에센</b>(<small><a class="mw-redirect" href="/wiki/%ED%95%9C%EA%B5%AD_%ED%95%9C%EC%9E%90" title="한국 한자">한국 한자</a>: </small>衛拉特 也先 <small>위랍특 야선</small>)이었다.</p>
<h2><span class="mw-headline" id=".EC.83.9D.EC.95.A0">생애</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%97%90%EC%84%BC_%ED%83%80%EC%9D%B4%EC%8B%9C&amp;action=edit&amp;section=1" title="부분 편집: 생애">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p><a href="/wiki/%EC%98%A4%EC%9D%B4%EB%9D%BC%ED%8A%B8" title="오이라트">오이라트</a>부의 지도자는 칸이라는 직위대신 타이시(太師)라는 칭호를 사용하였다. 타이시라는 칭호는 <a href="/wiki/%EB%AA%BD%EA%B3%A8_%EC%A0%9C%EA%B5%AD" title="몽골 제국">몽골 제국</a> 시대 군사령관이라는 의미였다. 오이라트부 지도자가 실권을 장악하였음에도 칸의 지위에 오르지 못한 이유는 오이라트부는 <a href="/wiki/%EC%B9%AD%EA%B8%B0%EC%A6%88_%EC%B9%B8" title="칭기즈 칸">칭기즈 칸</a>의 직계가 아니었기 때문이었다. 설령 이들이 칸이라는 호칭을 사용한다고 하드라도, 몽골의 유목민들이 그를 칸으로 인정해주지 않았는데, 이를 칭기즈 칸의 통치원리(chinggisid principle)라고 하였다.</p>
<p><a href="/wiki/1428%EB%85%84" title="1428년">1428년</a> 토곤과 손을 잡은 <a class="mw-redirect" href="/wiki/%EC%BF%A0%EB%B9%8C%EB%9D%BC%EC%9D%B4" title="쿠빌라이">쿠빌라이</a> 가문의 토크토아부카가 동몽골의 칸이었던 <a href="/wiki/%EC%95%84%EC%9E%90%EC%9D%B4_%EC%B9%B8" title="아자이 칸">아자이 칸</a>을 살해하여 전 몽골 지역을 사실상 통합하고 허수아비 대칸인 <a class="mw-redirect" href="/wiki/%ED%83%80%EC%9D%B4%EC%8A%A8_%EC%B9%B8" title="타이슨 칸">타이슨 칸</a>이 되었다.</p>
<p><a href="/wiki/1439%EB%85%84" title="1439년">1439년</a> 토곤이 죽자 뒤를 이어 오이라트의 지도자가 되어 토곤이 착수한 <a class="mw-redirect" href="/wiki/%EC%B0%A8%EA%B0%80%ED%83%80%EC%9D%B4_%ED%95%9C%EA%B5%AD" title="차가타이 한국">차가타이 한국</a>에 대한 영토 공략에 박차를 가해 <a class="mw-disambig" href="/wiki/%EC%9D%BC%EB%A6%AC" title="일리">일리</a>와 <a class="mw-redirect" href="/wiki/%ED%88%AC%EB%A5%B4%ED%8C%90" title="투르판">투르판</a>, <a class="mw-redirect" href="/wiki/%ED%95%98%EB%AF%B8" title="하미">하미</a>를 평정하고, 현재의 <a class="mw-redirect" href="/wiki/%EC%8B%A0%EA%B0%95%EC%9C%84%EA%B5%AC%EB%A5%B4_%EC%9E%90%EC%B9%98%EA%B5%AC" title="신강위구르 자치구">신강위구르 자치구</a> 대부분을 차지하였다.</p>
<p>동시에 <a class="mw-redirect" href="/wiki/%EC%A3%BC%EC%B9%98_(%EB%AA%BD%EA%B3%A8)" title="주치 (몽골)">주치</a> 가문의 <a href="/wiki/%ED%82%B5%EC%B0%A8%ED%81%AC_%EC%B9%B8%EA%B5%AD" title="킵차크 칸국">킵차크 칸국</a>에 대한 공략도 서둘러서 그 영토의 일부를 편입시켜 에센 시대 오이라트의 영토는 서쪽으로는 <a href="/wiki/%EB%B0%9C%ED%95%98%EC%8B%9C_%ED%98%B8" title="발하시 호">발하시 호</a>에서 <a href="/wiki/%EB%B0%94%EC%9D%B4%EC%B9%BC_%ED%98%B8" title="바이칼 호">바이칼 호</a>, 동쪽으로는 <a href="/wiki/%EB%A7%8C%EC%A3%BC" title="만주">만주</a>에 이르는 광대한 지역이었다.</p>
<p><a href="/wiki/1406%EB%85%84" title="1406년">1406년</a> <a href="/wiki/%EB%AA%85%EB%82%98%EB%9D%BC" title="명나라">명나라</a>의 <a href="/wiki/%EC%98%81%EB%9D%BD%EC%A0%9C" title="영락제">영락제</a>는 몽골 부족들에게 조공무역을 허락하였는데, 마시(馬市)라는 형태로 교역을 하여 영종까지 관례화되었다. 명나라는 이들로부터 말과 가축 등 그 부산물을 수입하고, 비단 등의 의류와 식량 등을 수출하였다.</p>
<p>처음에는 50명 정도의 사절단 규모가 에센 때에 이르러 3,000명까지 늘어났고, 주변 위구르의 상인들까지 가세하여 무역량이 늘어나고 밀무역도 성행하였다.</p>
<p>이에 심각한 문제를 겪던 명나라는 오이라트 부족에 대한 무역을 제한하였고 <a href="/wiki/1448%EB%85%84" title="1448년">1448년</a> 사례감 왕진은 실제 인원에 대한 조공무역만 허용했으며, 말 값도 오이라트가 제시한 가격의 20%만 지급하였다.</p>
<p>이에 불만을 품은 오이라트는 정통제 14년(<a href="/wiki/1449%EB%85%84" title="1449년">1449년</a>) 명나라 변방인 <a class="mw-redirect mw-disambig" href="/wiki/%EC%82%B0%EC%8B%9C%EC%84%B1" title="산시성">산시성</a> <a class="mw-redirect" href="/wiki/%EB%8B%A4%ED%89%81" title="다퉁">다퉁</a>(大同)으로 침입해 일명 <a href="/wiki/%ED%86%A0%EB%AA%A9%EC%9D%98_%EB%B3%80" title="토목의 변">토목의 변</a>(土木之變) 또는 <a class="mw-redirect" href="/wiki/%ED%86%A0%EB%AA%A9%EB%B3%B4%EC%9D%98_%EB%B3%80" title="토목보의 변">토목보의 변</a>(土木堡之變)을 일으켰는데 이때 명나라의 황제인 <a class="mw-redirect" href="/wiki/%EC%98%81%EC%A2%85_(%EB%AA%85)" title="영종 (명)">영종</a>(정통제)을 사로잡았다.</p>
<p>에센은 포로로 잡은 영종이 협상에 아무런 영향을 주지 못하자 아무런 조건 없이 <a href="/wiki/1450%EB%85%84" title="1450년">1450년</a>에 명나라 조정에 송환했고 <a href="/wiki/1452%EB%85%84" title="1452년">1452년</a> 타이슨 칸이 자신을 공격하자 이를 빌미로 살해한 뒤 에센은 동몽골 황족에 대한 대대적인 숙청 작업에 들어갔다.</p>
<p>황족 중 어머니가 오이라트 출신이 아닌 경우에는 모두 죽여 버렸으며 에센은 사람만 죽인 것이 아니라, 동몽골이 가지고 있던 거의 대부분 심지어는 기록과 문서 그리고 족보 등도 거의 대부분 소실되어 버렸다.</p>
<p><a href="/wiki/1453%EB%85%84" title="1453년">1453년</a> 직접 대칸의 자리에 올라 최초로 <a href="/wiki/%EC%B9%AD%EA%B8%B0%EC%A6%88_%EC%B9%B8" title="칭기즈 칸">칭기즈 칸</a> 가계가 아닌 오이라트 계통의 칸이 탄생했으나 오래 못 가 <a href="/wiki/1454%EB%85%84" title="1454년">1454년</a> 부하에게 살해당했다.</p>
<p>에센의 피살과 함께 오이라트 부도 급속히 와해되기 시작하여 오이라트는 몽골의 서쪽으로 물러났고 이후 오이라트는 서쪽에서 새롭게 세력을 형성함으로써 나중에 몽골 고원을 다시 장악하게 되는 <a href="/wiki/%EC%A4%91%EA%B0%80%EB%A5%B4_%EC%B9%B8%EA%B5%AD" title="중가르 칸국">준가르 제국</a>의 등장을 준비하고 있었다.</p>
