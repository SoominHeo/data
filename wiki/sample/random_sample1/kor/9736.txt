<p><b>에스타디오 마리오 알베르토 켐페스</b>(<span lang="es" xml:lang="es">Estadio Mario Alberto Kempes</span>)은 <a href="/wiki/%EC%95%84%EB%A5%B4%ED%97%A8%ED%8B%B0%EB%82%98" title="아르헨티나">아르헨티나</a> <a href="/wiki/%EC%BD%94%EB%A5%B4%EB%8F%84%EB%B0%94_%EC%A3%BC_(%EC%95%84%EB%A5%B4%ED%97%A8%ED%8B%B0%EB%82%98)" title="코르도바 주 (아르헨티나)">코르도바 주</a> 샤토 카레라스에 있는 종합 경기시설이다. <a href="/wiki/%EC%BD%94%EB%A5%B4%EB%8F%84%EB%B0%94_(%EC%95%84%EB%A5%B4%ED%97%A8%ED%8B%B0%EB%82%98)" title="코르도바 (아르헨티나)">코르도바 시</a>에선 10 <a href="/wiki/%ED%82%AC%EB%A1%9C%EB%AF%B8%ED%84%B0" title="킬로미터">km</a> 정도 떨어져 있다. 보통 에스타디오 코르도바(<span style="font-size: smaller;"><a href="/wiki/%EC%8A%A4%ED%8E%98%EC%9D%B8%EC%96%B4" title="스페인어">스페인어</a>:</span> <span lang="es" xml:lang="es">Estadio Córdoba</span>)나 <b>에스타디오 (올림피코) 샤토 카레라스</b>(<span lang="es" xml:lang="es">Estadio (Olímpico) Chateau Carreras</span>)라고 부른다.</p>
<p>경기장은 <a href="/wiki/1978%EB%85%84_FIFA_%EC%9B%94%EB%93%9C%EC%BB%B5" title="1978년 FIFA 월드컵">1978년 FIFA 월드컵</a>을 위해 <a href="/wiki/1976%EB%85%84" title="1976년">1976년</a> 지어졌다. 다른 아르헨티나의 경기장들처럼 경기장 전체에 좌석이 설치되어 있는 것은 아니지만, 공식적인 수용인원은 46,083명이다. 이 스타디움은 <a href="/wiki/1978%EB%85%84_FIFA_%EC%9B%94%EB%93%9C%EC%BB%B5" title="1978년 FIFA 월드컵">1978년 FIFA 월드컵</a>당시 맹활약으로 <a href="/wiki/%EC%95%84%EB%A5%B4%ED%97%A8%ED%8B%B0%EB%82%98_%EC%B6%95%EA%B5%AC_%EA%B5%AD%EA%B0%80%EB%8C%80%ED%91%9C%ED%8C%80" title="아르헨티나 축구 국가대표팀">아르헨티나 축구 국가대표팀</a>이 이 월드컵에서 우승하게 한 일등공신인 <a href="/wiki/%EB%A7%88%EB%A6%AC%EC%98%A4_%EC%BC%90%ED%8E%98%EC%8A%A4" title="마리오 켐페스">마리오 켐페스</a>의 이름을 기념하기 위해 지금의 이름으로 개명되었다.</p>
<p>코르도바를 연고로 하는 세 축구팀, <a class="mw-redirect" href="/wiki/%ED%83%80%EC%98%88%EB%A0%88%EC%8A%A4_%EB%8D%B0_%EC%BD%94%EB%A5%B4%EB%8F%84%EB%B0%94" title="타예레스 데 코르도바">타예레스 데 코르도바</a>, <a href="/wiki/%EC%9D%B8%EC%8A%A4%ED%8B%B0%ED%88%AC%ED%86%A0_%EC%95%84%ED%8B%80%EB%A0%88%ED%8B%B0%EC%BD%94_%EC%84%BC%ED%8A%B8%EB%9E%84_%EC%BD%94%EB%A5%B4%EB%8F%84%EB%B0%94" title="인스티투토 아틀레티코 센트랄 코르도바">인스티투토 아틀레티코 센트랄 코르도바</a>, <a href="/wiki/CA_%EB%B2%A8%EA%B7%B8%EB%9D%BC%EB%85%B8" title="CA 벨그라노">벨그라노</a>는 별도로 클럽의 경기장을 갖고 있지만 많은 관중이 예상되는 빅 매치의 경우 이 경기장을 사용한다.</p>
<p>이 곳은 <a href="/wiki/1978%EB%85%84_FIFA_%EC%9B%94%EB%93%9C%EC%BB%B5" title="1978년 FIFA 월드컵">1978년 FIFA 월드컵</a> 이외에도 <a href="/wiki/1987%EB%85%84" title="1987년">1987년</a> <a href="/wiki/%EC%BD%94%ED%8C%8C_%EC%95%84%EB%A9%94%EB%A6%AC%EC%B9%B4" title="코파 아메리카">코파 아메리카</a>과 <a href="/wiki/2001%EB%85%84" title="2001년">2001년</a> <a href="/wiki/FIFA_U-20_%EC%9B%94%EB%93%9C%EC%BB%B5" title="FIFA U-20 월드컵">FIFA U-20 월드컵</a> 시에도 경기장으로 사용되었다.</p>
