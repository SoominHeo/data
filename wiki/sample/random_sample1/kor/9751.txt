<h2><span class="mw-headline" id=".EC.97.AD.EC.82.AC">역사</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%97%90%EC%96%B4_%EC%8B%9C%EC%95%94&amp;action=edit&amp;section=1" title="부분 편집: 역사">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p><a href="/wiki/1965%EB%85%84" title="1965년">1965년</a> <a href="/wiki/9%EC%9B%94_15%EC%9D%BC" title="9월 15일">9월 15일</a> 밸런스 에어 시암(<span style="font-size: smaller;"><a href="/wiki/%EC%98%81%EC%96%B4" title="영어">영어</a>:</span> <span lang="en" xml:lang="en">Varan Air-Siam</span>)으로 설립해 회사명의 일부 출자자의 유래는 경영자인 <a href="/wiki/%ED%83%80%EC%9D%B4" title="타이">타이</a> 왕족의 한 사람에게 유래했다.</p>
<p><a href="/wiki/1970%EB%85%84" title="1970년">1970년</a> <a href="/wiki/2%EC%9B%94" title="2월">2월</a> <a href="/wiki/%EB%8D%94%EA%B8%80%EB%9F%AC%EC%8A%A4_DC-4" title="더글러스 DC-4">더글러스 DC-4</a> 기종을 도입해 <a href="/wiki/%EB%B0%A9%EC%BD%95" title="방콕">방콕</a> ~ <a href="/wiki/%ED%99%8D%EC%BD%A9" title="홍콩">홍콩</a>간 화물 노선을 취항했다. <a href="/wiki/1971%EB%85%84" title="1971년">1971년</a> <a href="/wiki/3%EC%9B%94" title="3월">3월</a>에 <a href="/wiki/%EB%B0%A9%EC%BD%95" title="방콕">방콕</a> ~ <a href="/wiki/%ED%99%8D%EC%BD%A9" title="홍콩">홍콩</a> ~ <a href="/wiki/%EB%8F%84%EC%BF%84" title="도쿄">도쿄</a> ~ <a href="/wiki/%ED%98%B8%EB%86%80%EB%A3%B0%EB%A3%A8" title="호놀룰루">호놀룰루</a> ~ <a href="/wiki/%EB%A1%9C%EC%8A%A4%EC%95%A4%EC%A0%A4%EB%A0%88%EC%8A%A4" title="로스앤젤레스">로스앤젤레스</a> 국제선을 개설하고 <a href="/wiki/%EC%9D%BC%EB%B3%B8" title="일본">일본</a>에 취항을 시작했다. 개설 당시 <a href="/wiki/%EB%8D%94%EA%B8%80%EB%9F%AC%EC%8A%A4_DC-8" title="더글러스 DC-8">더글러스 DC-8</a>로 운항했다. 지상 업무의 경우 <a href="/wiki/%EC%9D%BC%EB%B3%B8%ED%95%AD%EA%B3%B5" title="일본항공">일본항공</a>과 제휴했다. <a href="/wiki/1971%EB%85%84" title="1971년">1971년</a>에 4편으로 운항하고 있었다. 그 후 <a href="/wiki/1972%EB%85%84" title="1972년">1972년</a> 운항을 거쳐 <a href="/wiki/1975%EB%85%84" title="1975년">1975년</a>부터 <a href="/wiki/1976%EB%85%84" title="1976년">1976년</a>까지 <a href="/wiki/%EB%B0%A9%EC%BD%95" title="방콕">방콕</a> ~ <a href="/wiki/%ED%99%8D%EC%BD%A9" title="홍콩">홍콩</a> ~ <a class="mw-disambig" href="/wiki/%ED%9B%84%EC%BF%A0%EC%98%A4%EC%B9%B4" title="후쿠오카">후쿠오카</a> ~ <a href="/wiki/%ED%98%B8%EB%86%80%EB%A3%B0%EB%A3%A8" title="호놀룰루">호놀룰루</a> ~ <a href="/wiki/%EB%A1%9C%EC%8A%A4%EC%95%A4%EC%A0%A4%EB%A0%88%EC%8A%A4" title="로스앤젤레스">로스앤젤레스</a> 국제선 노선도 운항했다. 기종은 <a href="/wiki/%EB%A7%A5%EB%8F%84%EB%84%AC%EB%8D%94%EA%B8%80%EB%9F%AC%EC%8A%A4_DC-10" title="맥도넬더글러스 DC-10">맥도넬더글러스 DC-10</a>과 <a href="/wiki/%EB%B3%B4%EC%9E%89_747" title="보잉 747">보잉 747</a> 기종으로 운항했다.</p>
<p>기존 시세보다 싼 가격의 항공권 이 인기를 끌었으나 <a href="/wiki/1975%EB%85%84" title="1975년">1975년</a> <a href="/wiki/1%EC%9B%94" title="1월">1월</a> <a href="/wiki/%ED%83%80%EC%9D%B4" title="타이">타이</a> 정부가 정책으로 국제선을 운항하는 회사를 하나하기로 결정했기 때문에 경영이 악화되기 시작했다.<sup class="reference" id="cite_ref-1"><a href="#cite_note-1">[1]</a></sup> <a href="/wiki/1976%EB%85%84" title="1976년">1976년</a> <a href="/wiki/12%EC%9B%94_29%EC%9D%BC" title="12월 29일">12월 29일</a>에 자금난으로 인해 갑자기 운항을 중단했다. 이후 운항을 재개가 불가피 하자 <a href="/wiki/1977%EB%85%84" title="1977년">1977년</a> <a href="/wiki/1%EC%9B%94_14%EC%9D%BC" title="1월 14일">1월 14일</a> 마침내 최종 운항 수 없게 <a href="/wiki/%ED%83%80%EC%9D%B4" title="타이">타이 정부</a>에 <a class="mw-redirect" href="/wiki/%ED%83%80%EC%9D%B4%ED%95%AD%EA%B3%B5" title="타이항공">타이항공</a>에 합병을 요청했다. 같은 해 <a href="/wiki/2%EC%9B%94_3%EC%9D%BC" title="2월 3일">2월 3일</a>에 <a href="/wiki/%ED%83%80%EC%9D%B4" title="타이">타이</a> 교통 운행 면허가 취소 되면서 거액의 빚을 남기게 되었다.</p>
