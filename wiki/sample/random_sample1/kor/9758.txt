<p><b>에어린에어</b> (<span style="font-size: smaller;"><a href="/wiki/%ED%94%84%EB%9E%91%EC%8A%A4%EC%96%B4" title="프랑스어">프랑스어</a>:</span> <span lang="fr" xml:lang="fr">Société Airlinair</span>)는 <a href="/wiki/%EC%97%90%EC%96%B4%ED%94%84%EB%9E%91%EC%8A%A4" title="에어프랑스">에어프랑스</a>의 지분에 등록된 자회사이자 <a href="/wiki/%ED%94%84%EB%9E%91%EC%8A%A4" title="프랑스">프랑스</a>의 지역 항공사로 <a href="/wiki/1998%EB%85%84" title="1998년">1998년</a>에 설립한 항공사로 <a href="/wiki/%ED%8C%8C%EB%A6%AC_%EC%98%A4%EB%A5%BC%EB%A6%AC_%EA%B3%B5%ED%95%AD" title="파리 오를리 공항">파리 오를리 공항</a>과 <a href="/wiki/%EC%83%9D%ED%85%8D%EC%A5%90%ED%8E%98%EB%A6%AC_%EA%B5%AD%EC%A0%9C%EA%B3%B5%ED%95%AD" title="생텍쥐페리 국제공항">생텍쥐페리 국제공항</a>을 <a href="/wiki/%ED%97%88%EB%B8%8C_%EA%B3%B5%ED%95%AD" title="허브 공항">허브 공항</a>을 사용한다. 에어린 에어는 주로 <a href="/wiki/%ED%94%84%EB%9E%91%EC%8A%A4" title="프랑스">프랑스</a>의 국내선 노선과 <a href="/wiki/%EC%9C%A0%EB%9F%BD" title="유럽">유럽</a>의 국제선 노선을 취항 했으나 <a href="/wiki/2013%EB%85%84" title="2013년">2013년</a>에 <a href="/wiki/%EB%A0%88%EC%A7%80%EC%98%A4%EB%82%A0" title="레지오날">레지오날</a>, <a href="/wiki/%EB%B8%8C%EB%A6%BF_%EC%97%90%EC%96%B4" title="브릿 에어">브릿 에어</a>가 합병하면서 <a href="/wiki/HOP!" title="HOP!">HOP!</a>이 설립됐다.</p>
