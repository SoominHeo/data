<p><b>에어백</b>(Airbag)은 <a href="/wiki/%EC%9E%90%EB%8F%99%EC%B0%A8" title="자동차">자동차</a>에서 <a href="/wiki/%EC%82%AC%EA%B3%A0" title="사고">사고</a>가 났을 때 부상률을 낮추기 위해 <a href="/wiki/%EC%9E%90%EB%8F%99%EC%B0%A8" title="자동차">자동차</a>에 설치된 공기주머니이다. <a href="/wiki/%EC%A2%8C%EC%84%9D" title="좌석">좌석</a>의 전면이나 측면에 설치된다. 공기주머니가 처음에는 숨어있었던 것이 <a href="/wiki/%EC%9E%90%EB%8F%99%EC%B0%A8" title="자동차">자동차</a>가 <a href="/wiki/%EC%82%AC%EA%B3%A0" title="사고">사고</a>로 <a class="mw-disambig" href="/wiki/%EC%B6%A9%EA%B2%A9" title="충격">충격</a>을 받게 되면 공기주머니가 공기주머니를 터트려주는 센서에 의해 터지게 된다. <a href="/wiki/%ED%98%84%EB%8C%80%EC%9E%90%EB%8F%99%EC%B0%A8" title="현대자동차">현대자동차</a>와 <a href="/wiki/%EA%B8%B0%EC%95%84%EC%9E%90%EB%8F%99%EC%B0%A8" title="기아자동차">기아자동차</a>의 경우, 최근 출시되는 신차에 어드밴스드 에어백을 적용하고 있다. 하지만 어드밴스드 에어백이 있는 <a href="/wiki/%EC%9E%90%EB%8F%99%EC%B0%A8" title="자동차">자동차</a>라도 <a href="/wiki/%EC%9E%90%EB%8F%99%EC%B0%A8" title="자동차">자동차</a>에 탔을 때에는 먼저 <a href="/wiki/%EC%95%88%EC%A0%84%EB%9D%A0" title="안전띠">안전띠</a>부터 바르게 착용해야 하며, 탑승자가 <a href="/wiki/%ED%82%A4" title="키">키</a>가 작은 <a href="/wiki/%EC%96%B4%EB%A6%B0%EC%9D%B4" title="어린이">어린이</a>일 때는 카시트를 반드시 뒷좌석에 하여 태워주어야 한다.</p>
