<p><b>에어브리지 에어 카고</b>(<span style="font-size: smaller;"><a href="/wiki/%EC%98%81%EC%96%B4" title="영어">영어</a>:</span> <span lang="en" xml:lang="en">AirBridgeCargo Airlines</span>, <span style="font-size: smaller;"><a href="/wiki/%EB%9F%AC%EC%8B%9C%EC%95%84%EC%96%B4" title="러시아어">러시아어</a>:</span> <span lang="ru" xml:lang="ru">Авиакомпания</span>)는 <a href="/wiki/%EB%9F%AC%EC%8B%9C%EC%95%84" title="러시아">러시아</a>의 화물 항공사로 본사는 <a href="/wiki/%EB%9F%AC%EC%8B%9C%EC%95%84" title="러시아">러시아</a> <a href="/wiki/%EB%AA%A8%EC%8A%A4%ED%81%AC%EB%B0%94" title="모스크바">모스크바</a>에 위치해 있으며 <a href="/wiki/2003%EB%85%84" title="2003년">2003년</a>에 설립했다. 또한 사용하고 있는 <a href="/wiki/%ED%97%88%EB%B8%8C_%EA%B3%B5%ED%95%AD" title="허브 공항">허브 공항</a>으로 <a href="/wiki/%EB%AA%A8%EC%8A%A4%ED%81%AC%EB%B0%94" title="모스크바">모스크바</a>에 위치한 <a href="/wiki/%EC%85%B0%EB%A0%88%EB%A9%94%ED%8B%B0%EC%98%88%EB%B3%B4_%EA%B5%AD%EC%A0%9C%EA%B3%B5%ED%95%AD" title="셰레메티예보 국제공항">셰레메티예보 국제공항</a>이 있으며 계열사는 <a href="/wiki/%EB%9F%AC%EC%8B%9C%EC%95%84" title="러시아">러시아</a>의 화물 항공사이고, 모기업인 <a href="/wiki/%EB%B3%BC%EA%B0%80_%EB%93%9C%EB%84%A4%ED%8F%AC%EB%A5%B4_%ED%95%AD%EA%B3%B5" title="볼가 드네포르 항공">볼가 드네포르 항공</a>이 있다.</p>

<h2><span class="mw-headline" id=".EC.97.AD.EC.82.AC">역사</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%97%90%EC%96%B4%EB%B8%8C%EB%A6%AC%EC%A7%80_%EC%B9%B4%EA%B3%A0_%EC%97%90%EC%96%B4&amp;action=edit&amp;section=1" title="부분 편집: 역사">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>화물과 전세 운항 경험이 있는 <a href="/wiki/%EB%B3%BC%EA%B0%80_%EB%93%9C%EB%84%A4%ED%8F%AC%EB%A5%B4_%ED%95%AD%EA%B3%B5" title="볼가 드네포르 항공">볼가 드네포르 항공</a>에서 정기 화물편 부문을 분리해 <a href="/wiki/2003%EB%85%84" title="2003년">2003년</a>에 설립했다. <a href="/wiki/2004%EB%85%84" title="2004년">2004년</a> <a href="/wiki/1%EC%9B%94" title="1월">1월</a> <a href="/wiki/%EC%95%8C%EB%A6%AC%ED%83%88%EB%A6%AC%EC%95%84_%ED%95%AD%EA%B3%B5" title="알리탈리아 항공">알리탈리아 항공</a>에서 양수하여 2대의 <a href="/wiki/%EB%B3%B4%EC%9E%89_747" title="보잉 747">보잉 747-200F</a>에서 1대를 사용하여 같은 해 <a href="/wiki/5%EC%9B%94" title="5월">5월</a>부터 <a href="/wiki/%EB%A3%A9%EC%85%88%EB%B6%80%EB%A5%B4%ED%81%AC" title="룩셈부르크">룩셈부르크</a>에서 <a href="/wiki/%EB%AA%A8%EC%8A%A4%ED%81%AC%EB%B0%94" title="모스크바">모스크바</a>와 <a href="/wiki/%EB%85%B8%EB%B3%B4%EC%8B%9C%EB%B9%84%EB%A5%B4%EC%8A%A4%ED%81%AC" title="노보시비르스크">노보시비르스크</a>를 거쳐 <a class="mw-redirect" href="/wiki/%EB%B2%A0%EC%9D%B4%EC%A7%95" title="베이징">베이징</a>과 <a class="mw-redirect" href="/wiki/%EC%83%81%ED%95%98%EC%9D%B4" title="상하이">상하이</a>를 잇는 정기 화물 노선 운항을 시작했다. 같은 해 <a href="/wiki/10%EC%9B%94" title="10월">10월</a>에 기종이 추가 도입 되었다. <a href="/wiki/2005%EB%85%84" title="2005년">2005년</a>에 1대의 <a href="/wiki/%EB%B3%B4%EC%9E%89_747" title="보잉 747">보잉 747</a> 화물기가 추가로 도입 되면서 <a href="/wiki/%ED%99%8D%EC%BD%A9" title="홍콩">홍콩</a>과 <a class="mw-redirect" href="/wiki/%EB%82%98%EA%B3%A0%EC%95%BC" title="나고야">나고야</a> 노선에 취항하기 시작했다. <a href="/wiki/2012%EB%85%84" title="2012년">2012년</a> <a href="/wiki/2%EC%9B%94" title="2월">2월</a>에 <a href="/wiki/%EB%B3%B4%EC%9E%89_747" title="보잉 747">보잉 747-8F</a>를 1대 도입해 취항하고 있다.</p>
