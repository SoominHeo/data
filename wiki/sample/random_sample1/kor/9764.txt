<p><b>에어트랜 에어웨이스</b>(<span style="font-size: smaller;"><a href="/wiki/%EC%98%81%EC%96%B4" title="영어">영어</a>:</span> <span lang="en" xml:lang="en">AirTran Airways</span>)는 <a href="/wiki/%EB%AF%B8%EA%B5%AD" title="미국">미국</a>의 <a class="mw-redirect" href="/wiki/%EC%A0%80%EB%B9%84%EC%9A%A9_%ED%95%AD%EA%B3%B5%EC%82%AC" title="저비용 항공사">저비용 항공사</a>로 <a href="/wiki/%ED%95%98%EC%B8%A0%ED%95%84%EB%93%9C_%EC%9E%AD%EC%8A%A8_%EC%95%A0%ED%8B%80%EB%9E%9C%ED%83%80_%EA%B5%AD%EC%A0%9C%EA%B3%B5%ED%95%AD" title="하츠필드 잭슨 애틀랜타 국제공항">하츠필드 잭슨 애틀랜타 국제공항</a>을 <a href="/wiki/%ED%97%88%EB%B8%8C_%EA%B3%B5%ED%95%AD" title="허브 공항">허브 공항</a>으로 삼고 있으며 <a href="/wiki/%EB%AF%B8%EA%B5%AD" title="미국">미국</a> 국내선을 매일 750회 이상 운행하고 있으며 세계에서 <a href="/wiki/%EB%B3%B4%EC%9E%89_717" title="보잉 717">보잉 717-200</a>을 가장 많이 운영하고 있다.</p>

<h2><span class="mw-headline" id=".EC.97.AD.EC.82.AC">역사</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%97%90%EC%96%B4%ED%8A%B8%EB%9E%9C&amp;action=edit&amp;section=1" title="부분 편집: 역사">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p><a href="/wiki/1992%EB%85%84" title="1992년">1992년</a> <a href="/wiki/6%EC%9B%94" title="6월">6월</a> 밸류제트로 설립 했으며 <a href="/wiki/10%EC%9B%94" title="10월">10월</a> 운항을 시작해 <a href="/wiki/1994%EB%85%84" title="1994년">1994년</a> <a href="/wiki/8%EC%9B%94" title="8월">8월</a> 현재의 사명으로 변경했다. <a href="/wiki/1997%EB%85%84" title="1997년">1997년</a> <a href="/wiki/7%EC%9B%94" title="7월">7월</a> 같이 <a class="mw-redirect" href="/wiki/%EC%A0%80%EB%B9%84%EC%9A%A9_%ED%95%AD%EA%B3%B5%EC%82%AC" title="저비용 항공사">저비용 항공사</a>인 항공 사고를 일으킨 밸류제트와 합병했다. 여기에 "reverse merger"란 기술이 사용되고 밸류제트가 회사명을 변경하고 유지하는 방식이였다. 당시 사고에 의하여 가치 제트 항공의 고객 이미지 저하와 경영 부진에 되었기 때문에 취해진 조치였다. <a href="/wiki/1997%EB%85%84" title="1997년">1997년</a> 기존의 선착순 전석 자유석을 재차, 좌석 지정이 시작되었다. 또한 종래의 전석 이코노미 클래스를 비즈니스 클래스 좌석이 도입되었다. <a href="/wiki/1999%EB%85%84" title="1999년">1999년</a> <a href="/wiki/9%EC%9B%94" title="9월">9월</a> 세계 최초로 <a href="/wiki/%EB%B3%B4%EC%9E%89_717" title="보잉 717">보잉 717</a> 항공기의 납입을 받아 이듬해 <a href="/wiki/10%EC%9B%94" title="10월">10월</a>부터 운영을 시작했다. <a href="/wiki/2000%EB%85%84" title="2000년">2000년</a> <a href="/wiki/12%EC%9B%94" title="12월">12월</a> 처음 국제선 사업으로 <a href="/wiki/%EC%95%A0%ED%8B%80%EB%9E%9C%ED%83%80" title="애틀랜타">애틀랜타</a>발 <a href="/wiki/%EB%B0%94%ED%95%98%EB%A7%88" title="바하마">바하마</a> 노선이 개설되었다. <a href="/wiki/2003%EB%85%84" title="2003년">2003년</a>, <a href="/wiki/%EB%8D%B4%EB%B2%84" title="덴버">덴버</a>, <a href="/wiki/%EB%9D%BC%EC%8A%A4%EB%B2%A0%EC%9D%B4%EA%B1%B0%EC%8A%A4" title="라스베이거스">라스베이거스</a>, <a href="/wiki/%EB%A1%9C%EC%8A%A4%EC%95%A4%EC%A0%A4%EB%A0%88%EC%8A%A4" title="로스앤젤레스">로스앤젤레스</a>에 <a href="/wiki/%EC%95%84%EB%A9%94%EB%A6%AC%EC%B9%B4" title="아메리카">아메리카</a> 횡단 운항을 시작하면서 규모가 확장하기 시작 했으며 같은 해 <a href="/wiki/7%EC%9B%94" title="7월">7월</a> 100대의 <a href="/wiki/%EB%B3%B4%EC%9E%89_737" title="보잉 737">보잉 737</a>를 주문했다. <a href="/wiki/2006%EB%85%84" title="2006년">2006년</a> <a
