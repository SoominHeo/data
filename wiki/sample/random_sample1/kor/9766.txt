<p><b>에어포트 나리타 호</b>(<span style="font-size: smaller;"><a href="/wiki/%EC%9D%BC%EB%B3%B8%EC%96%B4" title="일본어">일본어</a>:</span> <span lang="ja" xml:lang="ja">エアポート成田, エアポートなりた</span>)는 <a class="mw-redirect" href="/wiki/%EB%8F%99%EC%9D%BC%EB%B3%B8_%EC%97%AC%EA%B0%9D_%EC%B2%A0%EB%8F%84" title="동일본 여객 철도">동일본 여객 철도</a>가 운행하는 <a href="/wiki/%EC%86%8C%EB%B6%80_%EC%BE%8C%EC%86%8D%EC%84%A0" title="소부 쾌속선">소부 쾌속선</a> - <a href="/wiki/%EC%9A%94%EC%BD%94%EC%8A%A4%EC%B9%B4_%EC%84%A0" title="요코스카 선">요코스카 선</a> 구간을 운행하는 쾌속 열차 중 <a href="/wiki/%EB%82%98%EB%A6%AC%ED%83%80_%EA%B3%B5%ED%95%AD%EC%97%AD" title="나리타 공항역">나리타 공항</a>발착의 쾌속 열차에게 부여된 명칭이다. 덧붙여 예전에는 나리타 공항발 쾌속 열차라면 이 명칭을 사용해왔으나 현재는 나리타 공항 - 도쿄 - 구리하마 구간을 운행하는 쾌속 열차에만 이 명칭을 사용하고 있다.</p>
<p><a href="/wiki/%EC%9A%94%EC%BD%94%EC%8A%A4%EC%B9%B4_%EC%84%A0" title="요코스카 선">요코스카 선</a>에서는 보통 열차로 취급되므로, <b>쾌속 에어포트 나리타</b>가 아닌, <b>보통 에어포트 나리타</b>라고 안내된다.</p>

<h2><span class="mw-headline" id=".EC.9A.B4.ED.96.89_.ED.98.84.ED.99.A9">운행 현황</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%97%90%EC%96%B4%ED%8F%AC%ED%8A%B8_%EB%82%98%EB%A6%AC%ED%83%80_%ED%98%B8&amp;action=edit&amp;section=1" title="부분 편집: 운행 현황">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<ul>
<li>기본적으로는, <a href="/wiki/%EC%86%8C%EB%B6%80_%EC%BE%8C%EC%86%8D%EC%84%A0" title="소부 쾌속선">소부 쾌속선</a> - <a href="/wiki/%EC%9A%94%EC%BD%94%EC%8A%A4%EC%B9%B4_%EC%84%A0" title="요코스카 선">요코스카 선</a>의 일부이다.</li>
<li>나리타 공항발 도쿄/구리하마 행 열차는 이 명칭으로 안내되지 않고 단지 "쾌속"으로만 안내된다.</li>
<li><a href="/wiki/%EA%B0%80%EC%8B%9C%EB%A7%88_%EC%84%A0" title="가시마 선">가시마 선</a>의 <a class="mw-redirect" href="/wiki/%EA%B0%80%EC%8B%9C%EB%A7%88_%EC%8B%A0%EA%B6%81%EC%97%AD" title="가시마 신궁역">카시마 신궁</a>행 열차 및 <a href="/wiki/%EC%86%8C%EB%B6%80_%EB%B3%B8%EC%84%A0" title="소부 본선">소부 본선</a> <a class="mw-disambig" href="/wiki/%EB%82%98%EB%A3%A8%ED%86%A0_%EC%97%AD" title="나루토 역">나루토</a> 행 열차를 병결하는 열차도 존재하지만(모두 <a href="/wiki/%EC%82%AC%EC%BF%A0%EB%9D%BC_%EC%97%AD_(%EC%A7%80%EB%B0%94_%ED%98%84)" title="사쿠라 역 (지바 현)">사쿠라</a>에서 분리의 과정을 거친다.), 현재는 대폭적인 감편에 의해서 1일 1편씩에 머무른다.</li>
<li>이전에는 운행 차량인 <a href="/wiki/%EB%8F%99%EC%9D%BC%EB%B3%B8_%EC%97%AC%EA%B0%9D%EC%B2%A0%EB%8F%84_E217%EA%B3%84_%EC%A0%84%EB%8F%99%EC%B0%A8" title="동일본 여객철도 E217계 전동차">E217계</a>에 자동 방송 장치가 장착되지 않았으며, <a class="new" href="/w/index.php?title=%EC%B0%A8%EB%82%B4_%EB%B0%A9%EC%86%A1&amp;action=edit&amp;redlink=1" title="차내 방송 (없는 문서)">차내 방송</a>은 <a class="new" href="/w/index.php?title=%EC%B0%A8%EC%9E%A5&amp;action=edit&amp;redlink=1" title="차장 (없는 문서)">차장</a>이 직접했다. 영어 안내 방송의 경우는 차장이 소지하는 카세트 테이프를 재생하는 방식이었으며, 그나마 차장에 따라서 하지 않기도 했다. 그러다가 E217계의 제어장치 갱신에 맞추어 자동 방송 장치가 설치되어 이 문제는 해결되었다.</li>
<li><a href="/wiki/%EC%9D%BC%EB%B3%B8%EA%B5%AD%EC%9C%A0%EC%B2%A0%EB%8F%84_113%EA%B3%84_%EC%A0%84%EB%8F%99%EC%B0%A8" title="일본국유철도 113계 전동차">113계</a>로 운행되었을 무렵부터 측면 행선지 표시기에 "에어포트 나리타"와 병기 되고 있다.</li>
</ul>
<dl>
<dt>현재 투입되고 있는 차량</dt>

<h2><span class="mw-headline" id=".EC.97.B0.ED.98.81">연혁</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%97%90%EC%96%B4%ED%8F%AC%ED%8A%B8_%EB%82%98%EB%A6%AC%ED%83%80_%ED%98%B8&amp;action=edit&amp;section=3" title="부분 편집: 연혁">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<ul>
<li><a href="/wiki/1991%EB%85%84" title="1991년">1991년</a> <a href="/wiki/3%EC%9B%94_19%EC%9D%BC" title="3월 19일">3월 19일</a> : <a class="new" href="/w/index.php?title=%EB%82%98%EB%A6%AC%ED%83%80%EC%97%AD&amp;action=edit&amp;redlink=1" title="나리타역 (없는 문서)">나리타</a> - <a href="/wiki/%EB%82%98%EB%A6%AC%ED%83%80_%EA%B3%B5%ED%95%AD%EC%97%AD" title="나리타 공항역">나리타 공항</a> 구간 개통으로 운행 개시. 다만, <a class="new" href="/w/index.php?title=%EC%8B%9C%EA%B0%81%ED%91%9C&amp;action=edit&amp;redlink=1" title="시각표 (없는 문서)">시각표</a>나 측면 행선지 표시기에 "에어포트 나리타"라고 표기될 때까지는 시간이 걸림.</li>
<li><a href="/wiki/1993%EB%85%84" title="1993년">1993년</a> <a href="/wiki/8%EC%9B%94_1%EC%9D%BC" title="8월 1일">8월 1일</a> : <a class="new" href="/w/index.php?title=%EC%97%98_%ED%8A%B9%EA%B8%89&amp;action=edit&amp;redlink=1" title="엘 특급 (없는 문서)">엘 특급</a> "아야노 호"(あやめ)가 격하되어, 일부 열차에 <a class="new" href="/w/index.php?title=%EA%B0%80%EC%82%AC%EB%A7%88_%EC%84%A0&amp;action=edit&amp;redlink=1" title="가사마 선 (없는 문서)">가사마 선</a> 가시마 신궁행 열차와 연결을 함.</li>
<li><a href="/wiki/1995%EB%85%84" title="1995년">1995년</a> <a href="/wiki/12%EC%9B%94_1%EC%9D%BC" title="12월 1일">12월 1일</a> : 특별 쾌속 운행 폐지.</li>
<li><a href="/wiki/2001%EB%85%84" title="2001년">2001년</a> <a href="/wiki/12%EC%9B%94_1%EC%9D%BC" title="12월 1일">12월 1일</a> : <a href="/wiki/%EC%8B%9C%EC%8A%A4%EC%9D%B4_%EC%97%AD" title="시스이 역">시스이</a>를 정차역에 추가.</li>
<li><a href="/wiki/2002%EB%85%84" title="2002년">2002년</a> <a href="/wiki/12%EC%9B%94_1%EC%9D%BC" title="12월 1일">12월 1일</a> : 나리타 공항발 열차는 이 명칭을 사용하지 않게 되었으며, "에어포트 나리타"라는 명칭은 나리타 공항행 열차에만 한정됨.</li>

<li><a href="/wiki/2009%EB%85%84" title="2009년">2009년</a> <a href="/wiki/3%EC%9B%94_14%EC%9D%BC" title="3월 14일">3월 14일</a> : 소부 본선 나루토행 편성 연결 열차를 신설.</li>

<h2><span class="mw-headline" id=".ED.8A.B9.EB.B3.84_.EC.BE.8C.EC.86.8D_.EC.97.90.EC.96.B4.ED.8F.AC.ED.8A.B8_.EB.82.98.EB.A6.AC.ED.83.80_.ED.98.B8">특별 쾌속 에어포트 나리타 호</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%97%90%EC%96%B4%ED%8F%AC%ED%8A%B8_%EB%82%98%EB%A6%AC%ED%83%80_%ED%98%B8&amp;action=edit&amp;section=4" title="부분 편집: 특별 쾌속 에어포트 나리타 호">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>임시 열차로서 <a href="/wiki/1991%EB%85%84" title="1991년">1991년</a>에 나리타 공항역이 개업하기 전부터 <a href="/wiki/1995%EB%85%84" title="1995년">1995년</a>까지 오후나 - 나리타 공항 구간을 "에어포트 나리타"라는 이름의 특별 쾌속이 운행되었다. 처음 운행했을 때는 도쿄에서 발착했으며, 나중에 요코스카 선 오후나까지 연장되었다. 도중의 정차역은 도쓰카, 히가시토쓰카, 호도가야, 요코하마, 신카와사키, 니시오이, 시나가와, 신바시, 도쿄 , 긴시초, 후나바시, 지바, 쓰가, 요쓰카이도, 나라타였다. 투입된 차량은 다른 요코스카 선 - 소부 쾌속선 열차와 같은 113계로 11량편성이었으며 헤드 마크를 게시했던 시기도 있었다. 또한, 지금의 "쾌속 에어포트 나리타"에서 사용되고 있는 E217계의 행선 안내판 롤지에도 "'특별 쾌속 에어포트' 나리타 나리타 공항" "'특별 쾌속 에어포트' 나리타 도쿄", "'특별 쾌속 에어포트 나리타' 오후나"가 표기되어 있다.</p>
