<h2><span class="mw-headline" id=".EA.B2.BD.EB.A0.A5">경력</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%97%90%EC%9A%B0%EB%8D%B0%EB%A5%B4_%EC%B9%B4%EB%A7%88%EB%9D%BC&amp;action=edit&amp;section=1" title="부분 편집: 경력">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<h3><span class="mw-headline" id=".EC.9C.A0.EB.85.84.EC.8B.9C.EC.A0.88">유년시절</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%97%90%EC%9A%B0%EB%8D%B0%EB%A5%B4_%EC%B9%B4%EB%A7%88%EB%9D%BC&amp;action=edit&amp;section=2" title="부분 편집: 유년시절">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>동 에우데르 카마라 대주교는 <a href="/wiki/%EB%B8%8C%EB%9D%BC%EC%A7%88" title="브라질">브라질</a> 북동부 <a href="/wiki/%EC%84%B8%EC%95%84%EB%9D%BC_%EC%A3%BC" title="세아라 주">세아라 주</a>의 수도 <a href="/wiki/%ED%8F%AC%EB%A5%B4%ED%83%88%EB%A0%88%EC%9E%90" title="포르탈레자">포르탈레자</a>(Fortaleza)에서 <a href="/wiki/%EC%B4%88%EB%93%B1%ED%95%99%EA%B5%90" title="초등학교">초등학교</a> 교사인 어머니와 지방<a href="/wiki/%EC%8B%A0%EB%AC%B8" title="신문">신문</a> <a href="/wiki/%EA%B8%B0%EC%9E%90_(%EC%A7%81%EC%97%85)" title="기자 (직업)">기자</a>였던 아버지 사이에서 태어났다. 그는 열세명의 형제자매 중 열두째였다. <a class="new" href="/w/index.php?title=%EC%9D%98%EB%A3%8C%EC%8B%9C%EC%84%A4&amp;action=edit&amp;redlink=1" title="의료시설 (없는 문서)">의료시설</a>이 없어서 <a class="new" href="/w/index.php?title=%ED%9B%84%EB%91%90%EC%97%BC&amp;action=edit&amp;redlink=1" title="후두염 (없는 문서)">후두염</a>으로 다섯 형제자매를 잃는 불행을 겪기도 하였으나, 부모로부터 이웃에 대한 자비와 작문실력을 물려받았다.</p>
<h3><span class="mw-headline" id=".EA.B5.90.ED.9A.8C.EC.97.90.EC.84.9C.EC.9D.98_.ED.99.9C.EB.8F.99">교회에서의 활동</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%97%90%EC%9A%B0%EB%8D%B0%EB%A5%B4_%EC%B9%B4%EB%A7%88%EB%9D%BC&amp;action=edit&amp;section=3" title="부분 편집: 교회에서의 활동">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>동 에우데르 카마라 대주교는 지혜로운 부친의 영향으로 진로를 설계하였다. 카마라 대주교의 아버지는 카마라 대주교가 어렸을 때 <a href="/wiki/%EC%84%B1%EB%8B%B9" title="성당">성당</a>에 나가지는 않았지만, <a href="/wiki/%EC%82%AC%EC%A0%9C" title="사제">사제</a>는 하느님과 이웃을 위해서 살아야 한다는 것은 알고 있는 분이었다. 카마라 대주교의 <a href="/wiki/%EB%B3%B5%EC%9D%8C%EC%84%9C" title="복음서">복음서</a> 묵상집인 《카마라와 함께 하는 복음》(황병렬 옮김, <a class="new" href="/w/index.php?title=%EC%83%9D%ED%99%9C%EC%84%B1%EC%84%9C&amp;action=edit&amp;redlink=1" title="생활성서 (없는 문서)">생활성서</a> 刊)에 의하면 카마라 대주교의 부친은 아들에게 이렇게 말했다.</p>

<p>그래서 카마라는 아버지가 말하는 <a href="/wiki/%EC%82%AC%EC%A0%9C" title="사제">사제</a>가 되어야겠다고 생각하여 신학교에 진학했다. 신학교 시절부터 <a href="/wiki/%ED%95%99%EB%AC%B8" title="학문">학문</a>, <a class="mw-redirect" href="/wiki/%EC%9E%91%EB%AC%B8" title="작문">작문</a>, <a class="mw-redirect" href="/wiki/%EC%9B%85%EB%B3%80" title="웅변">웅변</a> 실력이 모두 뛰어날 만큼 재능이 많았던 그는 <a href="/wiki/1921%EB%85%84" title="1921년">1921년</a> <a class="mw-redirect" href="/wiki/%EB%A1%9C%EB%A7%88_%EA%B0%80%ED%86%A8%EB%A6%AD_%EA%B5%90%ED%9A%8C" title="로마 가톨릭 교회">로마 가톨릭 교회</a> <a href="/wiki/%EC%82%AC%EC%A0%9C" title="사제">사제</a>가 된 후에 자신의 재능을 <a href="/wiki/%EA%B5%90%ED%9A%8C" title="교회">교회</a>를 위해 발휘하였다. <a href="/wiki/%EB%A6%AC%EC%9A%B0%EB%8D%B0%EC%9E%90%EB%84%A4%EC%9D%B4%EB%A3%A8" title="리우데자네이루">리우데자네이루</a> 교구 보좌주교였던 그는 지역이 넓어서 지역 <a href="/wiki/%EA%B5%90%ED%9A%8C" title="교회">교회</a>들간의 의사소통이 어려운 문제를 해결하기 위해 브라질 주교회의(CNBB)를 결성하였다. 브라질 주교회의는 <a class="mw-redirect" href="/wiki/%EC%84%B8%EA%B3%84%EC%84%B1%EC%B2%B4%EB%8C%80%ED%9A%8C" title="세계성체대회">세계성체대회</a>(1955년)와 <a href="/wiki/%EC%A0%9C2%EC%B0%A8_%EB%B0%94%ED%8B%B0%EC%B9%B8_%EA%B3%B5%EC%9D%98%ED%9A%8C" title="제2차 바티칸 공의회">제2차 바티칸 공의회</a>(<a href="/wiki/1962%EB%85%84" title="1962년">1962년</a>-<a href="/wiki/1965%EB%85%84" title="1965년">1965년</a>)결성을 주도하였다. 세계성체대회는 카마라 대주교가 가난한 사람들을 위해 헌신한 계기가 되었다. 성체대회가 끝난뒤 <a href="/wiki/%ED%94%84%EB%9E%91%EC%8A%A4" title="프랑스">프랑스</a> 로마 가톨릭 교회의 게를리어 <a href="/wiki/%EC%B6%94%EA%B8%B0%EA%B2%BD" title="추기경">추기경</a>이 "당신의 뛰어난 재능을 가난한 이들을 위해서 쓸 생각을 왜 하지 않습니까? 빈부의 격차는 창조주에 대한 모욕입니다."라면서 <a href="/wiki/%EC%84%B1%EC%A7%81%EC%9E%90" title="성직자">성직자</a>로서의 양심을 일깨웠던 것이다.</p>
<h3><span class="mw-headline" id=".EA.B0.80.EB.82.9C.ED.95.9C_.EC.82.AC.EB.9E.8C.EC.9D.84_.EC.9C.84.ED.95.9C_.EC.82.B6">가난한 사람을 위한 삶</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%97%90%EC%9A%B0%EB%8D%B0%EB%A5%B4_%EC%B9%B4%EB%A7%88%EB%9D%BC&amp;action=edit&amp;section=4" title="부분 편집: 가난한 사람을 위한 삶">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<h4><span class="mw-headline" id=".ED.8C.8C.EB.B2.A8.EB.9D.BC.EC.97.90.EC.84.9C.EC.9D.98_.EC.82.AC.ED.9A.8C.EB.B3.B5.EC.A7.80">파벨라에서의 사회복지</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%97%90%EC%9A%B0%EB%8D%B0%EB%A5%B4_%EC%B9%B4%EB%A7%88%EB%9D%BC&amp;action=edit&amp;section=5" title="부분 편집: 파벨라에서의 사회복지">편집</a><span class="mw-editsection-bracket">]</span></span></h4>
<p>카마라 대주교는 빈민지역인 <a href="/wiki/%ED%8C%8C%EB%B2%A8%EB%9D%BC" title="파벨라">파벨라</a>의 주민들을 위해 일하기 시작했다. 공공주택을 짓고, <a href="/wiki/%ED%85%94%EB%A0%88%EB%B9%84%EC%A0%84" title="텔레비전">텔레비전</a>, <a href="/wiki/%EC%8B%A0%EB%AC%B8" title="신문">신문</a>, <a href="/wiki/%EB%9D%BC%EB%94%94%EC%98%A4" title="라디오">라디오</a>에 출연하여 가난한 사람들을 위한 모금에 동참할 것을 요구했던 것이다.</p>
<h4><span class="mw-headline" id=".EB.B6.88.ED.8F.89.EB.93.B1.ED.95.9C_.ED.98.84.EC.8B.A4.EC.97.90_.EB.88.88.EB.9C.A8.EB.8B.A4.">불평등한 현실에 눈뜨다.</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%97%90%EC%9A%B0%EB%8D%B0%EB%A5%B4_%EC%B9%B4%EB%A7%88%EB%9D%BC&amp;action=edit&amp;section=6" title="부분 편집: 불평등한 현실에 눈뜨다.">편집</a><span class="mw-editsection-bracket">]</span></span></h4>
<p>하지만 동 에우데르 카마라 대주교가 사회복지사업으로는 가난이 해결되지 않는다는 것을 알게 되는 것은 그렇게 오랜 시간이 걸리지 않았다. '브라질 사회는 국민의 10퍼센트밖에 안 되는 부자들은 <a href="/wiki/%EB%8F%85%EC%9D%BC" title="독일">독일</a>의 32배나 되는 넓은 토지를 차지하는 반면에, 대다수의 <a href="/wiki/%EB%AF%BC%EC%A4%91" title="민중">민중</a>들은 <a href="/wiki/%EC%86%8C%EC%9E%91%EC%9D%B8" title="소작인">소작인</a>도 아닌 <a href="/wiki/%EB%86%8D%EC%97%85" title="농업">농업</a>노동자로 전락해서 비참하게 살아가는 매우 <a href="/wiki/%ED%8F%89%EB%93%B1" title="평등">평등</a>하지 못한 사회'라는 것을 사회선교를 하면서 깨달은 것이다. 하지만 <a href="/wiki/%EB%AF%B8%EA%B5%AD" title="미국">미국</a>등의 <a href="/wiki/%EB%8B%A4%EA%B5%AD%EC%A0%81_%EA%B8%B0%EC%97%85" title="다국적 기업">다국적 기업</a>들과 결탁한 군사<a href="/wiki/%EB%8F%85%EC%9E%AC" title="독재">독재</a>정권과 <a class="mw-disambig" href="/wiki/%EC%96%B8%EB%A1%A0" title="언론">언론</a>은 이러한 현실에 전혀 관심이 없었다. 더 부끄러운 사실은 <a href="/wiki/%EA%B5%90%ED%9A%8C" title="교회">교회</a>가 부자와 권력자들과 결탁하여, <a href="/wiki/%EC%98%88%EC%96%B8%EC%9E%90" title="예언자">예언자</a>적인 목소리를 전혀 내지 않고 있다는 것이었다.<sup class="reference" id="cite_ref-3"><a href="#cite_note-3">[3]</a></sup> 그래서 카마라 대주교는 가난은 하느님의 창조질서에 어긋나는 것이며, <a href="/wiki/%EB%AF%BC%EC%A4%91" title="민중">민중</a>들이 인간으로서의 최소한의 삶과 <a href="/wiki/%EC%9D%B8%EA%B6%8C" title="인권">인권</a>을 존중받으면서 사는 세상을 만드는 것이 <a href="/wiki/%EB%B3%B5%EC%9D%8C" title="복음">복음</a>화라는 자신의 <a href="/wiki/%EC%8B%A0%ED%95%99" title="신학">신학</a>을 실천하기 위해 사회선교에 헌신하였다. <a href="/wiki/%EA%B5%90%ED%99%A9" title="교황">교황</a> <a class="mw-redirect" href="/wiki/%EC%9A%94%ED%95%9C_%EB%B0%94%EC%98%A4%EB%A1%9C_2%EC%84%B8" title="요한 바오로 2세">요한 바오로 2세</a>도 브라질 북동부 올린다와 레시페 교구의 대주교로 임명하여 카마라 대주교의 사회선교를 도왔다.</p>
<h4><span class="mw-headline" id=".ED.96.89.EB.8F.99.E2.80.A4.EC.A0.95.EC.9D.98.E2.80.A4.ED.8F.89.ED.99.94.EC.9A.B4.EB.8F.99">행동․정의․평화운동</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%97%90%EC%9A%B0%EB%8D%B0%EB%A5%B4_%EC%B9%B4%EB%A7%88%EB%9D%BC&amp;action=edit&amp;section=7" title="부분 편집: 행동․정의․평화운동">편집</a><span class="mw-editsection-bracket">]</span></span></h4>
<p>카마라 대주교 당시 브라질 사회는 빈부의 격차가 매우 극심하였다. 인구의 10%도 안되는 엘리트 특권층들은 국민총생산량의 60퍼센트를 차지하여 부유하게 사는 반면에, 전체 인구의 절반이나 되는 4천 5백만 명은 36크루제리오(약 8달러)도 안 되는 돈으로 한 달을 살고 있었다. 그래서 카마라 대주교는 <a href="/wiki/%EB%AF%BC%EC%A4%91" title="민중">민중</a>을 계몽하여 자존감(자신을 소중하게 여길 줄을 아는 의식. 자존감을 가진 사람은 자신을 소중히 여기는만큼 이웃도 소중하게 여길 수 있다. 이는 자기만 사랑하고, 다른 사람은 사랑할 줄을 모르는 <a class="mw-disambig" href="/wiki/%EC%9D%B4%EA%B8%B0%EC%A3%BC%EC%9D%98" title="이기주의">이기주의</a>와는 다르다.)을 갖게 하였다. 그 실례로 동 에우데르 카마라는 <a class="mw-redirect" href="/wiki/%EC%B2%AD%EC%86%8C%EB%B6%80" title="청소부">청소부</a>에게 정말 부끄럽게 여겨야 하는 것은 일하느라 거칠고 더러워진 당신의 <a href="/wiki/%EC%86%90" title="손">손</a>이 아니라, 자기중심적인 태도라고 말해주었다. <sup class="reference" id="cite_ref-4"><a href="#cite_note-4">[4]</a></sup> 특권층과 정부에 대해서는 브라질 사회는 평등하지 못한 사회라고 비판하며 <a href="/wiki/%EC%98%88%EC%96%B8%EC%9E%90" title="예언자">예언자</a>적인 목소리를 내었다. 그는 <a class="mw-redirect" href="/wiki/%EB%A1%9C%EB%A7%88_%EA%B0%80%ED%86%A8%EB%A6%AD_%EA%B5%90%ED%9A%8C" title="로마 가톨릭 교회">로마 가톨릭 교회</a>내 <a class="mw-disambig" href="/wiki/%EC%A7%84%EB%B3%B4" title="진보">진보</a>적인 평신도들의 모임을 지원하였으며, 지역공동체와 성인교육프로그램을 실시하였다. 또한 로마 가톨릭 교회내의 진보적인 <a href="/wiki/%EC%84%B1%EC%A7%81%EC%9E%90" title="성직자">성직자</a>들을 단결하게 하여, 하느님의 말씀으로 불평등한 사회를 바로 잡고자 하였다.</p>
<h4><span class="mw-headline" id=".EA.B6.8C.EB.A0.A5.EC.9E.90.EB.93.A4.EA.B3.BC_.ED.8A.B9.EA.B6.8C.EC.B8.B5.EC.9D.98_.ED.83.84.EC.95.95">권력자들과 특권층의 탄압</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%97%90%EC%9A%B0%EB%8D%B0%EB%A5%B4_%EC%B9%B4%EB%A7%88%EB%9D%BC&amp;action=edit&amp;section=8" title="부분 편집: 권력자들과 특권층의 탄압">편집</a><span class="mw-editsection-bracket">]</span></span></h4>
<p>엘리트 특권층들과 권력자들은 <a href="/wiki/%EC%98%88%EC%96%B8%EC%9E%90" title="예언자">예언자</a>적인 목소리를 내는 카마라 대주교를 미워하여,‘<a class="mw-redirect" href="/wiki/%EC%82%AC%ED%9A%8C%EC%A3%BC%EC%9D%98%EC%9E%90" title="사회주의자">사회주의자</a>’, ‘국가전복자’라고 모함하면서 탄압하였다. 온갖 방법으로 빈민가에서의 목회를 방해하였으며, 살해위협을 하기까지 하였다.</p>
<h4><span class="mw-headline" id=".ED.95.98.EB.8A.90.EB.8B.98.EC.9D.80_.EA.B0.80.EB.82.9C.ED.95.9C_.EC.82.AC.EB.9E.8C.EB.93.A4_.EA.B0.80.EC.9A.B4.EB.8D.B0_.EA.B3.84.EC.8B.A0.EB.8B.A4.">하느님은 가난한 사람들 가운데 계신다.</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%97%90%EC%9A%B0%EB%8D%B0%EB%A5%B4_%EC%B9%B4%EB%A7%88%EB%9D%BC&amp;action=edit&amp;section=9" title="부분 편집: 하느님은 가난한 사람들 가운데 계신다.">편집</a><span class="mw-editsection-bracket">]</span></span></h4>
<p>하지만 카마라 대주교는 주님은 가난한 사람들안에 계신다는 믿음을 실천하기 위해 결코 <a href="/wiki/%EB%AF%BC%EC%A4%91" title="민중">민중</a>들과 같이 사는 목회방식을 포기하지 않았다. 그는 행동하는 <a href="/wiki/%EC%84%B1%EC%A7%81%EC%9E%90" title="성직자">성직자</a>였다. 그 실례로 카마라 대주교는 <a href="/wiki/1965%EB%85%84" title="1965년">1965년</a> 방 세 개짜리 서민 <a href="/wiki/%EC%95%84%ED%8C%8C%ED%8A%B8" title="아파트">아파트</a>로 이사하였으며, 주교관을 개방하여 도움이 필요한 사람들은 언제든지 드나들 수 있게 하였다. 신학교도 빈민가로 옮겨서 신학생들이 자신들이 살아가는 사회의 진짜 모습을 볼 수 있게 하였다. <a href="/wiki/1985%EB%85%84" title="1985년">1985년</a> <a class="mw-redirect" href="/wiki/%EB%A1%9C%EB%A7%88_%EA%B0%80%ED%86%A8%EB%A6%AD_%EA%B5%90%ED%9A%8C" title="로마 가톨릭 교회">로마 가톨릭 교회</a>의 <a class="mw-redirect" href="/wiki/%ED%95%B4%EB%B0%A9%EC%8B%A0%ED%95%99" title="해방신학">해방신학</a>에 대한 회의적 입장<sup class="reference" id="cite_ref-5"><a href="#cite_note-5">[5]</a></sup>으로 <a href="/wiki/%EB%8C%80%EC%A3%BC%EA%B5%90" title="대주교">대주교</a>직을 보수적인 후임자에게 맡기고 은퇴해야 했던 그는 <a href="/wiki/1999%EB%85%84" title="1999년">1999년</a> 레시페 교구에서 90세를 일기로 별세하였다. 하지만 그의 말과 행동이 일치하는 삶은 <a href="/wiki/1973%EB%85%84" title="1973년">1973년</a> <a class="mw-redirect" href="/wiki/%EB%85%B8%EB%B2%A8%ED%8F%89%ED%99%94%EC%83%81" title="노벨평화상">노벨평화상</a> 수상이 거론될 정도로 세계적으로 큰 영향을 주었다.</p>
<h2><span class="mw-headline" id=".EC.96.B4.EB.A1.9D">어록</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%97%90%EC%9A%B0%EB%8D%B0%EB%A5%B4_%EC%B9%B4%EB%A7%88%EB%9D%BC&amp;action=edit&amp;section=10" title="부분 편집: 어록">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<blockquote class="toccolours" style="float:none; padding: 10px 15px 10px 15px; display:table;">
<p>“<a href="/wiki/%EC%84%B8%EA%B3%84" title="세계">세계</a> 인구의 20퍼센트가 80퍼센트의 것을 먹어치우고, 80퍼센트가 20퍼센트를 가지고 끼니를 때웁니다. 그리고 미친 듯이 <a class="new" href="/w/index.php?title=%EA%B5%B0%EB%B9%84%EA%B2%BD%EC%9F%81&amp;action=edit&amp;redlink=1" title="군비경쟁 (없는 문서)">군비경쟁</a>에 소비하는 돈은 이 <a href="/wiki/%EC%A7%80%EA%B5%AC" title="지구">지구</a>의 빈곤을 없애고도 남을만한 거액입니다. 여러분께 제안합니다. 미움, <a href="/wiki/%ED%8F%AD%EB%A0%A5" title="폭력">폭력</a>을 없애는데 힘을 모아주십시오. 세계 <a href="/wiki/%EB%B9%88%EA%B3%A4" title="빈곤">빈곤</a>을 없애는데 참여하자고 간절히 청하고 싶습니다.”(<a href="/wiki/1989%EB%85%84" title="1989년">1989년</a> <a href="/wiki/10%EC%9B%94_2%EC%9D%BC" title="10월 2일">10월 2일</a> <a class="mw-redirect" href="/wiki/%EC%84%9C%EC%9A%B8" title="서울">서울</a> 제44차 <a class="mw-redirect" href="/wiki/%EC%84%B8%EA%B3%84%EC%84%B1%EC%B2%B4%EB%8C%80%ED%9A%8C" title="세계성체대회">세계성체대회</a> 당시 에우데르 카마라 대주교가 <a class="mw-redirect" href="/wiki/%EC%B2%9C%EC%A3%BC%EA%B5%90%ED%9A%8C" title="천주교회">천주교회</a> 잡지 《<a class="new" href="/w/index.php?title=%EC%83%9D%ED%99%9C%EC%84%B1%EC%84%9C&amp;action=edit&amp;redlink=1" title="생활성서 (없는 문서)">생활성서</a>》와 나눈 인터뷰)</p>
</blockquote>
<blockquote class="toccolours" style="float:none; padding: 10px 15px 10px 15px; display:table;">
<p>“내가 가난한 이들에게 먹을 것을 주면 사람들은 나를 성인(聖人)이라 부르고, 내가 가난한 이들은 왜 먹을 것이 없는지 물으면 사람들은 나를 <a class="mw-redirect" href="/wiki/%EA%B3%B5%EC%82%B0%EC%A3%BC%EC%9D%98%EC%9E%90" title="공산주의자">공산주의자</a>라고 부른다.”</p>
</blockquote>

<h2><span class="mw-headline" id=".EC.B6.9C.EC.B2.98">출처</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%97%90%EC%9A%B0%EB%8D%B0%EB%A5%B4_%EC%B9%B4%EB%A7%88%EB%9D%BC&amp;action=edit&amp;section=12" title="부분 편집: 출처">편집</a><span class="mw-editsection-bracket">]</span></span></h2>

<h2><span class="mw-headline" id=".EA.B0.81.EC.A3.BC">각주</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%97%90%EC%9A%B0%EB%8D%B0%EB%A5%B4_%EC%B9%B4%EB%A7%88%EB%9D%BC&amp;action=edit&amp;section=13" title="부분 편집: 각주">편집</a><span class="mw-editsection-bracket">]</span></span></h2>

<li id="cite_note-4"><span class="mw-cite-backlink"><a href="#cite_ref-4">↑</a></span> <span class="reference-text">《카마라와 함께 읽는 복음》p.14-15</span></li>
