<p><b>에우로페</b>(<span style="font-size: smaller;"><a href="/wiki/%EA%B7%B8%EB%A6%AC%EC%8A%A4%EC%96%B4" title="그리스어">그리스어</a>:</span> <span lang="el" xml:lang="el">Ευρωπη</span>)는 <a href="/wiki/%EA%B7%B8%EB%A6%AC%EC%8A%A4_%EC%8B%A0%ED%99%94" title="그리스 신화">그리스 신화</a>에 등장하는 여성이다. <a href="/wiki/%EB%8C%80%EB%A5%99" title="대륙">대륙</a> <a href="/wiki/%EC%9C%A0%EB%9F%BD" title="유럽">유럽</a>의 이름과 <a href="/wiki/%EC%9C%84%EC%84%B1" title="위성">위성</a> <a href="/wiki/%EC%9C%A0%EB%A1%9C%ED%8C%8C_(%EC%9C%84%EC%84%B1)" title="유로파 (위성)">유로파</a>의 이름은 그녀에게서 따온 것이다.</p>

<h2><span class="mw-headline" id=".EA.B0.80.EA.B3.84">가계</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%97%90%EC%9A%B0%EB%A1%9C%ED%8E%98&amp;action=edit&amp;section=1" title="부분 편집: 가계">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>문헌들에 따르면 에우로페의 가족에 관해서는 세부적으로 약간씩 차이난다. 그러나 그녀는 페니키아인이며 <a href="/wiki/%EC%9D%B4%EC%98%A4_(%EC%8B%A0%ED%99%94)" title="이오 (신화)">이오</a>의 후손이다. 이오는 제우스의 사랑을 받은 전설적인 요정으로 제우스가 아내 <a href="/wiki/%ED%97%A4%EB%9D%BC" title="헤라">헤라</a>의 질투를 피하기 위해 그녀를 암소로 변하게 하였다.</p>
<p>에우로페는 <a href="/wiki/%ED%8B%B0%EB%A0%88" title="티레">티레</a>의 <a href="/wiki/%ED%8E%98%EB%8B%88%ED%82%A4%EC%95%84" title="페니키아">페니키아</a> 왕 <a href="/wiki/%EC%95%84%EA%B2%8C%EB%85%B8%EB%A5%B4" title="아게노르">아게노르</a>와 여왕 <a href="/wiki/%ED%85%94%EB%A0%88%ED%8C%8C%EC%82%AC" title="텔레파사">텔레파사</a>의 딸이였다고 언급된다. 그러나 <a href="/wiki/%ED%98%B8%EB%A9%94%EB%A1%9C%EC%8A%A4" title="호메로스">호메로스</a>의 <i><a href="/wiki/%EC%9D%BC%EB%A6%AC%EC%95%84%EC%8A%A4" title="일리아스">일리아스</a></i>에 따르면 그녀는 <a href="/wiki/%ED%8F%AC%EC%9D%B4%EB%8B%89%EC%8A%A4_(%EC%8B%A0%ED%99%94)" title="포이닉스 (신화)">포이닉스</a>의 딸이다.<sup class="reference" id="cite_ref-1"><a href="#cite_note-1">[1]</a></sup> 그녀에게는 남자 형제가 있었는데 <a href="/wiki/%EC%B9%B4%EB%93%9C%EB%AA%A8%EC%8A%A4" title="카드모스">카드모스</a>와 <a href="/wiki/%ED%82%AC%EB%A6%AD%EC%8A%A4" title="킬릭스">킬릭스</a>가 그들이었다. (몇몇 전승에서는 포이닉스도 그 형제들에 포함된다.)</p>
<h2><span class="mw-headline" id=".EC.A0.84.EC.84.A4">전설</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%97%90%EC%9A%B0%EB%A1%9C%ED%8E%98&amp;action=edit&amp;section=2" title="부분 편집: 전설">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>제우스가 하루는 꽃을 따러 나온 에우로페의 아름다움에 반했다. 제우스는 하얗고 멋진 황소로 변해서 그녀에게 접근했고 그녀를 등에 업고 바다로 들어가 <a href="/wiki/%ED%81%AC%EB%A0%88%ED%83%80" title="크레타">크레타</a>로 납치해서 그녀와 정을 통한다. 한편 아게노르는 딸이 행방불명이 되자 아들들을 불러 여동생을 찾아오기 전에는 돌아오지 말라고 엄명을 내렸다. 이에 카드모스와 킬릭스는 동생을 찾아 사방을 헤매게 되는데 카드모스는 <a href="/wiki/%EC%95%84%ED%8F%B4%EB%A1%A0" title="아폴론">아폴론</a>의 도움으로 <a href="/wiki/%ED%85%8C%EB%B2%A0_(%EA%B7%B8%EB%A6%AC%EC%8A%A4)" title="테베 (그리스)">테베</a>에 정착하여 도시를 세우고, 그리스에 문자를 가져왔다고 한다. 킬릭스는 <a class="mw-redirect" href="/wiki/%EC%86%8C%EC%95%84%EC%8B%9C%EC%95%84" title="소아시아">소아시아</a>의 <a href="/wiki/%ED%82%AC%EB%A6%AC%ED%82%A4%EC%95%84" title="킬리키아">킬리키아</a>의 왕이 되어 그의 이름이 나라 이름이 되었다. 포이닉스는 세 번째 형제로 기록되기도 한다.</p>
<p>크레타에 도달한 후에 제우스는 에우로페에게 <a href="/wiki/%ED%97%A4%ED%8C%8C%EC%9D%B4%EC%8A%A4%ED%86%A0%EC%8A%A4" title="헤파이스토스">헤파이스토스</a>가 만들어 준 목걸이를 선물했고, <sup class="reference" id="cite_ref-2"><a href="#cite_note-2">[2]</a></sup> 에우로페는 크레타에서 여왕이 되었고 제우스와의 사이에서 세 아들을 낳았는데 그 아들들의 이름은 <a href="/wiki/%EB%AF%B8%EB%85%B8%EC%8A%A4" title="미노스">미노스</a>, <a href="/wiki/%EB%9D%BC%EB%8B%A4%EB%A7%8C%ED%8B%B0%EC%8A%A4" title="라다만티스">라다만티스</a>, <a href="/wiki/%EC%82%AC%EB%A5%B4%ED%8E%98%EB%8F%88" title="사르페돈">사르페돈</a>이다.</p>

<h2><span class="mw-headline" id=".EC.9C.A0.EB.9F.BD.EC.9D.98_.EC.9D.B4.EB.A6.84">유럽의 이름</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%97%90%EC%9A%B0%EB%A1%9C%ED%8E%98&amp;action=edit&amp;section=3" title="부분 편집: 유럽의 이름">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>그녀의 이름은 <a class="mw-redirect" href="/wiki/%EC%8A%A4%ED%8A%B8%EB%9D%BC%EB%B3%B4" title="스트라보">스트라보</a><sup class="reference" id="cite_ref-3"><a href="#cite_note-3">[3]</a></sup>와 같은 그리스 고대 지리학자들에 의해 <a href="/wiki/%EC%9C%A0%EB%9F%BD" title="유럽">유럽</a> 대륙의 이름으로 차용되었다. 이 이름은 <a href="/wiki/%EA%B7%B8%EB%A6%AC%EC%8A%A4%EC%96%B4" title="그리스어">그리스어</a> <i>Ευρώπη</i>에서 비롯되어 거의 모든 유럽어족에서 사용되었다.</p>
