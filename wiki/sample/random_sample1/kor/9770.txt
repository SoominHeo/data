<p><b>에이노 아브라모비치 라흐야</b>(<span style="font-size: smaller;"><a href="/wiki/%ED%95%80%EB%9E%80%EB%93%9C%EC%96%B4" title="핀란드어">핀란드어</a>:</span> <span lang="fi" xml:lang="fi">Eino Abramovich Rahja</span>; <span style="font-size: smaller;"><a href="/wiki/%EB%9F%AC%EC%8B%9C%EC%95%84%EC%96%B4" title="러시아어">러시아어</a>:</span> <span lang="ru" xml:lang="ru">Э́йно Абра́мович Ра́хья</span>: 1885년 6월 20일 - 1936년 4월 26일)는 <a href="/wiki/%ED%95%80%EB%9E%80%EB%93%9C" title="핀란드">핀란드</a>계 <a href="/wiki/%EB%9F%AC%EC%8B%9C%EC%95%84" title="러시아">러시아</a>의 정치인이다.</p>
<p>1903년 <a href="/wiki/%EB%9F%AC%EC%8B%9C%EC%95%84_%EC%82%AC%ED%9A%8C%EB%AF%BC%EC%A3%BC%EB%85%B8%EB%8F%99%EB%8B%B9" title="러시아 사회민주노동당">러시아 사회민주노동당</a>에 입당했고, 이후 분당 사태 때 <a href="/wiki/%EB%B3%BC%EC%85%B0%EB%B9%84%ED%82%A4" title="볼셰비키">볼셰비키</a> 쪽에 가담했다. 1917년 <a class="new" href="/w/index.php?title=7%EC%9B%94_%EC%A0%95%EA%B5%AD&amp;action=edit&amp;redlink=1" title="7월 정국 (없는 문서)">7월 정국</a>에서 볼셰비키가 패배하자 레닌을 핀란드로 도피시키기도 했다. <a href="/wiki/%ED%95%80%EB%9E%80%EB%93%9C_%EB%82%B4%EC%A0%84" title="핀란드 내전">핀란드 내전</a> 당시 라흐야는 <a class="mw-redirect" href="/wiki/%EC%A0%81%ED%95%80%EB%9E%80%EB%93%9C" title="적핀란드">적핀란드</a> 측에서 가장 유능한 군사 지휘관이었다. 내전이 적군의 패배로 끝나자 <a class="mw-redirect" href="/wiki/%EB%9F%AC%EC%8B%9C%EC%95%84_SFSR" title="러시아 SFSR">러시아 SFSR</a>로 도피하여 거기서 여생을 보내며 <a href="/wiki/%EB%B6%89%EC%9D%80_%EA%B5%B0%EB%8C%80" title="붉은 군대">붉은 군대</a>에서 군단장 등을 지냈다.</p>
<p>1920년대 초 <a href="/wiki/%EA%B7%B8%EB%A6%AC%EA%B3%A0%EB%A6%AC_%EC%A7%80%EB%85%B8%EB%B9%84%EC%98%88%ED%94%84" title="그리고리 지노비예프">그리고리 지노비예프</a>와 정치적으로 가까이 지내다가, 1927년 종파행위를 이유로 <a class="new" href="/w/index.php?title=%ED%95%80%EB%9E%80%EB%93%9C_%EA%B3%B5%EC%82%B0%EB%8B%B9&amp;action=edit&amp;redlink=1" title="핀란드 공산당 (없는 문서)">핀란드 공산당</a> 중앙위원회에서 제명을 당했다. 1936년 <a href="/wiki/%EC%95%8C%EC%BD%94%EC%98%AC" title="알코올">알코올</a> 중독으로 인한 궤양으로 사망했고 <a class="mw-redirect" href="/wiki/%EC%95%8C%EB%A0%89%EC%82%B0%EB%93%9C%EB%A5%B4_%EB%84%A4%ED%94%84%EC%8A%A4%ED%82%A4_%EC%84%B1%EB%8B%B9" title="알렉산드르 네프스키 성당">알렉산드르 네프스키 성당</a>에 묻혔다.</p>
