<h1 class="firstHeading" id="firstHeading" lang="ko">에이레 누어</h1>
<div class="mw-body-content" id="bodyContent">
<div id="siteSub">위키백과, 우리 모두의 백과사전.</div>
<div id="contentSub"></div>
<div class="mw-jump" id="jump-to-nav">
					이동:					<a href="#mw-head">둘러보기</a>, 					<a href="#p-search">검색</a>
</div>
<div class="mw-content-ltr" dir="ltr" id="mw-content-text" lang="ko"><script>function mfTempOpenSection(id){var block=document.getElementById("mf-section-"+id);block.className+=" open-block";block.previousSibling.className+=" open-block";}</script><p><b>에이레 누어</b>(<span style="font-size: smaller;"><a href="/wiki/%EC%95%84%EC%9D%BC%EB%9E%9C%EB%93%9C%EC%96%B4" title="아일랜드어">아일랜드어</a>:</span> <span lang="ga" xml:lang="ga">Éire Nua</span>)는 "새로운 <a href="/wiki/%EC%95%84%EC%9D%BC%EB%9E%9C%EB%93%9C" title="아일랜드">아일랜드</a>"라는 뜻으로, <a href="/wiki/%EC%95%84%EC%9D%BC%EB%9E%9C%EB%93%9C_%EA%B3%B5%ED%99%94%EA%B5%AD%EA%B5%B0_%EC%9E%84%EC%8B%9C%ED%8C%8C" title="아일랜드 공화국군 임시파">아일랜드 공화국군 임시파</a>(PIRA)와 <a class="mw-redirect" href="/wiki/%EC%8B%A0%ED%8E%98%EC%9D%B8%EB%8B%B9" title="신페인당">신페인당</a>이 1970년대에서 1980년대 초까지 주장했던 <a href="/wiki/%EC%97%B0%EB%B0%A9%EC%A0%9C" title="연방제">연방제</a>식 <a class="new" href="/w/index.php?title=%ED%86%B5%EC%9D%BC_%EC%95%84%EC%9D%BC%EB%9E%9C%EB%93%9C&amp;action=edit&amp;redlink=1" title="통일 아일랜드 (없는 문서)">아일랜드 통일안</a>이다.</p>
<p>에이레 누어에 따르면 <a href="/wiki/%EB%B6%81%EC%95%84%EC%9D%BC%EB%9E%9C%EB%93%9C" title="북아일랜드">북아일랜드</a>가 영국의 지배로부터 해방되어야 함은 물론이고, 현재 존재하는 <a class="mw-redirect" href="/wiki/%EC%95%84%EC%9D%BC%EB%9E%9C%EB%93%9C_%EA%B3%B5%ED%99%94%EA%B5%AD" title="아일랜드 공화국">아일랜드 공화국</a> 역시 1922년 영국에 의해 강제된 <a href="/wiki/%EC%98%81%EA%B5%AD-%EC%95%84%EC%9D%BC%EB%9E%9C%EB%93%9C_%EC%A1%B0%EC%95%BD" title="영국-아일랜드 조약">영애조약</a>에 의해 건국된 것이므로 해체되어야 한다. 그리고 아일랜드의 <a href="/wiki/%EC%95%84%EC%9D%BC%EB%9E%9C%EB%93%9C%EC%9D%98_%EC%A7%80%EB%B0%A9" title="아일랜드의 지방">전통적인 네 지방</a>(<a href="/wiki/%EC%96%BC%EC%8A%A4%ED%84%B0" title="얼스터">얼스터</a>, <a href="/wiki/%EB%A8%BC%EC%8A%A4%ED%84%B0" title="먼스터">먼스터</a>, <a href="/wiki/%EB%A0%8C%EC%8A%A4%ED%84%B0" title="렌스터">렌스터</a>, <a href="/wiki/%EC%BD%94%EB%85%B8%ED%8A%B8" title="코노트">코노트</a>)가 연방주체가 되는 연방국가를 이루고 아일랜드 섬 중앙의 <a href="/wiki/%EC%95%A0%EC%8A%AC%EB%A1%A0_(%EC%95%84%EC%9D%BC%EB%9E%9C%EB%93%9C)" title="애슬론 (아일랜드)">애슬론</a>에 중앙의회를 설치한다.<sup class="reference" id="cite_ref-1"><a href="#cite_note-1">[1]</a></sup></p>
<p>1980년대 이후 에이레 누어는 내부적으로 인기를 잃어 거의 폐기되었다. 그러나 PIRA가 무장투쟁을 포기한 뒤에도 <a class="new" href="/w/index.php?title=%EA%B3%B5%ED%99%94%EC%A3%BC%EC%9D%98_%EC%8B%A0%ED%8E%98%EC%9D%B8&amp;action=edit&amp;redlink=1" title="공화주의 신페인 (없는 문서)">공화주의 신페인</a>과 <a class="new" href="/w/index.php?title=%ED%94%BC%EC%96%B4%EB%84%88_%EC%97%90%EC%9D%B4%EB%A0%8C&amp;action=edit&amp;redlink=1" title="피어너 에이렌 (없는 문서)">피어너 에이렌</a>, <a href="/wiki/%EC%95%84%EC%9D%BC%EB%9E%9C%EB%93%9C_%EC%97%AC%EC%84%B1%ED%8F%89%EC%9D%98%ED%9A%8C" title="아일랜드 여성평의회">아일랜드 여성평의회</a>는 에이레 누어를 지지하고 있다.</p>
