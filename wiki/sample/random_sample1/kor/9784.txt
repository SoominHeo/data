<p><b>에이버리 브런디지</b>(Avery Brundage, <a href="/wiki/1887%EB%85%84" title="1887년">1887년</a> <a href="/wiki/9%EC%9B%94_28%EC%9D%BC" title="9월 28일">9월 28일</a> ~ <a href="/wiki/1975%EB%85%84" title="1975년">1975년</a> <a href="/wiki/5%EC%9B%94_8%EC%9D%BC" title="5월 8일">5월 8일</a>)는 <a href="/wiki/%EB%AF%B8%EA%B5%AD" title="미국">미국</a>의 운동선수였고 스포츠 공무원, 미술품 수집가이며 <a class="mw-redirect" href="/wiki/%EB%B0%95%EC%95%A0%EC%A3%BC%EC%9D%98%EC%9E%90" title="박애주의자">박애주의자</a>였으나 그는 현재까지도 미국 올림픽 위원회의 위원 때와 <a href="/wiki/%EA%B5%AD%EC%A0%9C_%EC%98%AC%EB%A6%BC%ED%94%BD_%EC%9C%84%EC%9B%90%ED%9A%8C" title="국제 올림픽 위원회">국제 올림픽 위원회</a>(IOC)의 위원장이었을 때 했던 결정으로 많은 비판을 받고 있기도 하다.</p>
<h2><span class="mw-headline" id=".EC.83.9D.EC.95.A0">생애</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%97%90%EC%9D%B4%EB%B2%84%EB%A6%AC_%EB%B8%8C%EB%9F%B0%EB%94%94%EC%A7%80&amp;action=edit&amp;section=1" title="부분 편집: 생애">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p><a href="/wiki/%EB%94%94%ED%8A%B8%EB%A1%9C%EC%9D%B4%ED%8A%B8" title="디트로이트">디트로이트</a> 출생으로 <a class="mw-redirect" href="/wiki/%EC%9D%BC%EB%A6%AC%EB%85%B8%EC%9D%B4_%EB%8C%80%ED%95%99%EA%B5%90_%EC%96%B4%EB%B0%94%EB%82%98-%EC%83%B4%ED%8E%98%EC%9D%B8" title="일리노이 대학교 어바나-샴페인">일리노이 대학교 어바나-샴페인</a>에서 <a href="/wiki/%ED%86%A0%EB%AA%A9%EA%B3%B5%ED%95%99" title="토목공학">토목공학</a>을 전공했고 <a href="/wiki/1909%EB%85%84" title="1909년">1909년</a>에 졸업했다. 몇 년 후 그는 <a href="/wiki/%EC%8B%9C%EC%B9%B4%EA%B3%A0" title="시카고">시카고</a> 등지에서 토목공학과 관련된 일을 하는 에이버리 브런디지 사를 세웠으며 <a href="/wiki/1947%EB%85%84" title="1947년">1947년</a>까지 운영했는데 그의 개인사는 대학교에서 이룬 것과 관련돼있다.</p>
<p>브런디지는 모든 스포츠에 다재다능하기도 해서 <a href="/wiki/%EC%8A%A4%ED%86%A1%ED%99%80%EB%A6%84" title="스톡홀름">스톡홀름</a>에서 열린 <a href="/wiki/1912%EB%85%84_%ED%95%98%EA%B3%84_%EC%98%AC%EB%A6%BC%ED%94%BD" title="1912년 하계 올림픽">1912년 하계 올림픽</a> 때 <a class="mw-redirect" href="/wiki/%EA%B7%BC%EB%8C%80_5%EC%A2%85" title="근대 5종">근대 5종</a>과 <a class="mw-redirect" href="/wiki/10%EC%A2%85_%EA%B2%BD%EA%B8%B0" title="10종 경기">10종 경기</a>에 출전해서 각각 6위와 16위를 차지했다. 그는 또한 미국 체육 MVP상(US national all-around title)을 1914년, 1916년, 1918년에 받았다.</p>

<p>1928년 <a class="new" href="/w/index.php?title=%EC%95%84%EB%A7%88%EC%B6%94%EC%96%B4%EC%9A%B4%EB%8F%99%EC%97%B0%ED%95%A9&amp;action=edit&amp;redlink=1" title="아마추어운동연합 (없는 문서)">아마추어운동연합</a>(AAU)의 회장이 되고 다음해인 <a href="/wiki/1929%EB%85%84" title="1929년">1929년</a>에는 <a href="/wiki/%EB%AF%B8%EA%B5%AD_%EC%98%AC%EB%A6%BC%ED%94%BD_%EC%9C%84%EC%9B%90%ED%9A%8C" title="미국 올림픽 위원회">미국 올림픽 위원회</a>(USOC)의 회장이 되었다. 또 <a href="/wiki/1930%EB%85%84" title="1930년">1930년</a>에는 <a class="mw-redirect" href="/wiki/%EA%B5%AD%EC%A0%9C%EC%9C%A1%EC%83%81%EA%B2%BD%EA%B8%B0%EC%97%B0%EB%A7%B9" title="국제육상경기연맹">국제육상경기연맹</a>(IAAF)의 부회장이 되었다. 1936년에 열린 <a href="/wiki/1936%EB%85%84_%ED%95%98%EA%B3%84_%EC%98%AC%EB%A6%BC%ED%94%BD" title="1936년 하계 올림픽">올림픽</a>에서는 <a class="mw-redirect" href="/wiki/%EC%95%84%EC%8A%88%EC%BC%80%EB%82%98%EC%A7%90" title="아슈케나짐">유대계 독일인</a>이 배제되었기 때문에 이번 대회를 보이콧하자는 움직임이 일기도 했지만 USOC 회장이었던 브런디지는 그런 움직임을 전부 부정했다. 그리고 선수들에게 보이콧 행동을 부추겼던 <a class="new" href="/w/index.php?title=%EC%96%B4%EB%8B%88%EC%8A%A4%ED%8A%B8_%EB%A6%AC_%EC%96%91%ED%82%A4&amp;action=edit&amp;redlink=1" title="어니스트 리 양키 (없는 문서)">어니스트 리 양키</a>가 <a href="/wiki/%EA%B5%AD%EC%A0%9C_%EC%98%AC%EB%A6%BC%ED%94%BD_%EC%9C%84%EC%9B%90%ED%9A%8C" title="국제 올림픽 위원회">국제 올림픽 위원회</a> 이사회에서 추방되었고 브런디지가 그 후임이 되었다. 브런디지는 후에 독일의 <a href="/wiki/%EC%95%84%EB%8F%8C%ED%94%84_%ED%9E%88%ED%8B%80%EB%9F%AC" title="아돌프 히틀러">아돌프 히틀러</a>의 <a class="mw-redirect" href="/wiki/%EB%82%98%EC%B9%98" title="나치">나치</a> 체제를 옹호하는 발언을 해 이것이 <a href="/wiki/%EB%8F%85%EC%9D%BC" title="독일">독일</a>의 지지를 얻어서 미국우선위원회로부터 제명되었다.</p>
<p><a class="mw-redirect" href="/wiki/%EC%A0%9C2%EC%B0%A8_%EC%84%B8%EA%B3%84%EB%8C%80%EC%A0%84" title="제2차 세계대전">제2차 세계대전</a> 중에 IOC 위원장이었던 <a href="/wiki/%EC%95%99%EB%A6%AC_%EB%93%9C_%EB%B0%94%EC%98%88%EB%9D%BC%ED%88%AC%EB%A5%B4" title="앙리 드 바예라투르">앙리 드 바예라투르</a>가 사망한 후, 브런디지는 IOC 부회장이 되었다. 이때 위원장은 <a href="/wiki/%EC%8B%9C%EA%B7%B8%ED%94%84%EB%A6%AC%EB%93%9C_%EC%97%90%EB%93%9C%EC%8A%A4%ED%8A%B8%EB%A3%80" title="시그프리드 에드스트룀">시그프리드 에드스트룀</a>이었지만 <a href="/wiki/1952%EB%85%84" title="1952년">1952년</a>이 은퇴한 후에 브런디지가 후임이 되었다. IOC 위원장 재임중에 어떤 형태로도 프로선수가 참가하는 것을 강하게 반대했지만 이 태도는 차츰 스포츠계에서, 또 다른 IOC 위원도 반대하는 형국이 되어버렸다. 이러한 변화는 몇 번이고 논의를 한 것처럼 사건을 일으켰기 때문이다. 그 결과로 <a href="/wiki/1972%EB%85%84_%EB%8F%99%EA%B3%84_%EC%98%AC%EB%A6%BC%ED%94%BD" title="1972년 동계 올림픽">1972년 동계 올림픽</a>에 참가한 <a href="/wiki/%EC%98%A4%EC%8A%A4%ED%8A%B8%EB%A6%AC%EC%95%84" title="오스트리아">오스트리아</a>의 <a class="new" href="/w/index.php?title=%EC%B9%B4%EB%A5%BC_%EC%8A%90%EB%9E%80%EC%B8%A0&amp;action=edit&amp;redlink=1" title="카를 슐란츠 (없는 문서)">카를 슐란츠</a>가 프로란 이유로 경기에 참석하지 못했다. 또 <a href="/wiki/1968%EB%85%84_%ED%95%98%EA%B3%84_%EC%98%AC%EB%A6%BC%ED%94%BD" title="1968년 하계 올림픽">1968년 하계 올림픽</a>에서는 <a class="mw-disambig" href="/wiki/%ED%86%A0%EB%AF%B8_%EC%8A%A4%EB%AF%B8%EC%8A%A4" title="토미 스미스">토미 스미스</a>와 <a href="/wiki/%EC%A1%B4_%EC%B9%BC%EB%A1%9C%EC%8A%A4" title="존 칼로스">존 칼로스</a>가 시상식에서 블랙파워 행동을 찬성하는 의미에서 주먹을 불끈 쥐고 들어올린 사건도 있었다. 브런디지는 이에 대해서 두 선수를 선수촌에서 추방시키고 출장정지를 처분했다.</p>
<p>브런디지의 이름을 세계에 알린 사건은 <a href="/wiki/1972%EB%85%84_%ED%95%98%EA%B3%84_%EC%98%AC%EB%A6%BC%ED%94%BD" title="1972년 하계 올림픽">1972년 하계 올림픽</a>에서 열린 사건이 결정적이었는데 <a href="/wiki/9%EC%9B%94_5%EC%9D%BC" title="9월 5일">9월 5일</a> 일어난 <a href="/wiki/%EA%B2%80%EC%9D%80_9%EC%9B%94%EB%8B%A8" title="검은 9월단">검은 9월단</a>이 <a href="/wiki/%EC%9D%B4%EC%8A%A4%EB%9D%BC%EC%97%98" title="이스라엘">이스라엘</a> 선수들을 인질로 잡고 선수 및 감독 11명이 살해당한 <a href="/wiki/%EB%AE%8C%ED%97%A8_%EC%98%AC%EB%A6%BC%ED%94%BD_%EC%B0%B8%EC%82%AC" title="뮌헨 올림픽 참사">뮌헨 참사</a> 후에 브런디지는 올림픽을 계속한다는 결정을 내렸다. 브런디지에게 쏟아진 비난은 엄청났지만 그 후에 올계속된 올림픽에서 뮌헨을 떠난 선수는 거의 없었다. 9월 5일에 열릴 경기는 전부 연기되었고 다음날 올림픽 경기장에서 8만관중과 3천명의 선수가 모여서 추도식을 열었다. 많은 청중들의 분노에 찬 소리가 소용돌이 치던 중, 브런디지는 추모연설에서 살해당한 이스라엘 선수에 대한 언급은 한마디도 없었고 올림픽의 강함만 강조했다.</p>
<p>브런디지는 <a href="/wiki/%EC%95%84%ED%8C%8C%EB%A5%B4%ED%8A%B8%ED%97%A4%EC%9D%B4%ED%8A%B8" title="아파르트헤이트">아파르트헤이트</a> 정책을 이유로 <a href="/wiki/%EB%A1%9C%EB%94%94%EC%A7%80%EC%95%84" title="로디지아">로디지아</a>가 올림픽에서 제외 되는 것을 반대해 강한 반발을 불어일으켰다. 뮌헨 참사 이후, 브런디지는 이스라엘선수가 살해당한 것과 로디지아선수에게 참가를 허용한것을 두고 비판이 쏟아졌다. 뮌헨 올림픽 후에 IOC 위원장에서 물러났고 현재까지 IOC 위원장을 맡은 유일한 미국인이다.</p>
<p>1975년 <a href="/wiki/%EC%84%9C%EB%8F%85" title="서독">서독</a> <a href="/wiki/%EA%B0%80%EB%A5%B4%EB%AF%B8%EC%8A%88%ED%8C%8C%EB%A5%B4%ED%85%90%ED%82%A4%EB%A5%B4%ED%97%A8" title="가르미슈파르텐키르헨">가르미슈파르텐키르헨</a>에서 사망했다.</p>
