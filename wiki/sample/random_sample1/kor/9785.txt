<p><b>에이브러햄 로빈슨</b>(<span style="font-size: smaller;"><a href="/wiki/%EB%8F%85%EC%9D%BC%EC%96%B4" title="독일어">독일어</a>:</span> <span lang="de" xml:lang="de">Abraham Robinso(h)n</span> <small>아브라함 로빈존<sup>[<a href="/wiki/%EC%9C%84%ED%82%A4%EB%B0%B1%EA%B3%BC:%EB%8F%85%EC%9D%BC%EC%96%B4%EC%9D%98_%ED%95%9C%EA%B8%80_%ED%91%9C%EA%B8%B0" title="위키백과:독일어의 한글 표기">*</a>]</sup></small>, <span style="font-size: smaller;"><a href="/wiki/%ED%9E%88%EB%B8%8C%EB%A6%AC%EC%96%B4" title="히브리어">히브리어</a>:</span> <span lang="he" xml:lang="he">אַבְרָהָם רוֹבִּינְזוֹן</span> <small>아브라함 로빈존</small> <small><a href="/wiki/%EC%9C%84%ED%82%A4%EB%B0%B1%EA%B3%BC:IPA" title="위키백과:IPA">IPA</a>: </small><span class="IPA" title="IPA 발음">[avraˈham ˈrobinzon]</span>, <span style="font-size: smaller;"><a href="/wiki/%EC%98%81%EC%96%B4" title="영어">영어</a>:</span> <span lang="en" xml:lang="en">Abraham Robinson</span> <small>에이브러햄 로빈슨<sup>[<a href="/wiki/%EC%9C%84%ED%82%A4%EB%B0%B1%EA%B3%BC:%EC%98%81%EC%96%B4%EC%9D%98_%ED%95%9C%EA%B8%80_%ED%91%9C%EA%B8%B0" title="위키백과:영어의 한글 표기">*</a>]</sup></small> <small><a href="/wiki/%EC%9C%84%ED%82%A4%EB%B0%B1%EA%B3%BC:IPA" title="위키백과:IPA">IPA</a>: </small><span class="IPA" title="IPA 발음">[ˈeɪbɹəhæm ˈrɒbɪnsən]</span>, 1918–1974)은 <a class="mw-redirect" href="/wiki/%EB%B9%84%ED%91%9C%EC%A4%80%ED%95%B4%EC%84%9D%ED%95%99" title="비표준해석학">비표준해석학</a>을 창시한 수학자이다.</p>
<h2><span class="mw-headline" id=".EC.83.9D.EC.95.A0">생애</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%97%90%EC%9D%B4%EB%B8%8C%EB%9F%AC%ED%96%84_%EB%A1%9C%EB%B9%88%EC%8A%A8&amp;action=edit&amp;section=1" title="부분 편집: 생애">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>당시 <a href="/wiki/%EB%8F%85%EC%9D%BC_%EC%A0%9C%EA%B5%AD" title="독일 제국">독일 제국</a>에 속해 있었던 <a href="/wiki/%EB%B0%94%EC%9A%B0%EB%B8%8C%EC%A7%80%ED%9D%90" title="바우브지흐">바우브지흐</a>(오늘날 <a href="/wiki/%ED%8F%B4%EB%9E%80%EB%93%9C" title="폴란드">폴란드</a>에 소속)의 <a href="/wiki/%EC%9C%A0%EB%8C%80%EC%9D%B8" title="유대인">유대인</a> 가정에 태어났다. 로빈존 가족의 본명은 독일어식 철자인 <span style="font-size: smaller;"><a href="/wiki/%EB%8F%85%EC%9D%BC%EC%96%B4" title="독일어">독일어</a>:</span> <span lang="de" xml:lang="de">Robinsohn</span> <small>로빈존<sup>[<a href="/wiki/%EC%9C%84%ED%82%A4%EB%B0%B1%EA%B3%BC:%EB%8F%85%EC%9D%BC%EC%96%B4%EC%9D%98_%ED%95%9C%EA%B8%80_%ED%91%9C%EA%B8%B0" title="위키백과:독일어의 한글 표기">*</a>]</sup></small>이었으나, 이후 1940년에 영어식으로 "h"를 생략하였다.</p>
<p>로빈존 가족은 <a href="/wiki/%EC%8B%9C%EC%98%A8%EC%A3%BC%EC%9D%98" title="시온주의">시온주의</a>를 지지하였으며, 1933년에 로빈슨은 15세의 나이에 당시 <a href="/wiki/%EC%98%81%EA%B5%AD_%EC%9C%84%EC%9E%84%ED%86%B5%EC%B9%98%EB%A0%B9_%ED%8C%94%EB%A0%88%EC%8A%A4%ED%83%80%EC%9D%B8" title="영국 위임통치령 팔레스타인">영국 위임통치령 팔레스타인</a>의 <a href="/wiki/%EC%98%88%EB%A3%A8%EC%82%B4%EB%A0%98" title="예루살렘">예루살렘</a>으로 이민하였다. <a href="/wiki/%EC%98%88%EB%A3%A8%EC%82%B4%EB%A0%98_%ED%9E%88%EB%B8%8C%EB%A6%AC_%EB%8C%80%ED%95%99%EA%B5%90" title="예루살렘 히브리 대학교">예루살렘 히브리 대학교</a>에서 학사 학위를 수여받았다.</p>
<p><a href="/wiki/%EC%A0%9C2%EC%B0%A8_%EC%84%B8%EA%B3%84_%EB%8C%80%EC%A0%84" title="제2차 세계 대전">제2차 세계 대전</a>이 발발할 당시 로빈슨은 <a href="/wiki/%ED%94%84%EB%9E%91%EC%8A%A4" title="프랑스">프랑스</a>에 있었는데, 기차와 발품을 통해 겨우 <a href="/wiki/%EC%98%81%EA%B5%AD" title="영국">영국</a> <a href="/wiki/%EB%9F%B0%EB%8D%98" title="런던">런던</a>으로 탈출하였다. 이 과정은 로빈슨이 독일 여권을 갖고 있었고, 자세한 프랑스 지도를 갖고 있었기 때문에 프랑스군의 의심을 받아 특히 힘들었다. 런던에 도착한 로빈슨은 <a href="/wiki/%ED%94%84%EB%9E%91%EC%8A%A4_%EA%B3%B5%ED%99%94%EA%B5%AD_%EC%9E%84%EC%8B%9C_%EC%A0%95%EB%B6%80" title="프랑스 공화국 임시 정부">프랑스 공화국 임시 정부</a> 공군의 일원이 되었다. 로빈슨은 수학 능력 덕택에 직접 전투에 참전하지 않았고, 대신 전투기 날개의 <a href="/wiki/%EA%B3%B5%EA%B8%B0%EC%97%AD%ED%95%99" title="공기역학">공기역학</a>을 연구하였다.</p>
<p>전후 1946년에 <a class="new" href="/w/index.php?title=%ED%81%AC%EB%9E%9C%ED%95%84%EB%93%9C_%EB%8C%80%ED%95%99%EA%B5%90&amp;action=edit&amp;redlink=1" title="크랜필드 대학교 (없는 문서)">크랜필드 대학교</a>(Cranfield University) 공기역학과 강의원(senior lecturer)이 되었고, 같은 해 <a href="/wiki/%EC%98%88%EB%A3%A8%EC%82%B4%EB%A0%98_%ED%9E%88%EB%B8%8C%EB%A6%AC_%EB%8C%80%ED%95%99%EA%B5%90" title="예루살렘 히브리 대학교">예루살렘 히브리 대학교</a>에서 석사 학위를 수여받았다. 1949년에 <a href="/wiki/%EB%9F%B0%EB%8D%98_%EB%8C%80%ED%95%99%EA%B5%90" title="런던 대학교">런던 대학교</a>에서 박사 학위를 수여받았다.</p>
<p>1951년에 <a href="/wiki/%ED%86%A0%EB%A1%A0%ED%86%A0_%EB%8C%80%ED%95%99%EA%B5%90" title="토론토 대학교">토론토 대학교</a> 응용수학과 교수가 되었고, 1957년에 <a href="/wiki/%EC%95%84%EB%B8%8C%EB%9D%BC%ED%95%A8_%ED%94%84%EB%A0%9D%EC%BC%88" title="아브라함 프렝켈">아브라함 프렝켈</a>의 자리를 채워 <a href="/wiki/%EC%98%88%EB%A3%A8%EC%82%B4%EB%A0%98_%ED%9E%88%EB%B8%8C%EB%A6%AC_%EB%8C%80%ED%95%99%EA%B5%90" title="예루살렘 히브리 대학교">예루살렘 히브리 대학교</a> 수학과장이 되었다. 1962년에는 <a href="/wiki/%EC%BA%98%EB%A6%AC%ED%8F%AC%EB%8B%88%EC%95%84_%EB%8C%80%ED%95%99%EA%B5%90_%EB%A1%9C%EC%8A%A4%EC%95%A4%EC%A0%A4%EB%A0%88%EC%8A%A4" title="캘리포니아 대학교 로스앤젤레스">캘리포니아 대학교 로스앤젤레스</a>의 수학·철학과로 옮겼고, 1967년에는 <a href="/wiki/%EC%98%88%EC%9D%BC_%EB%8C%80%ED%95%99%EA%B5%90" title="예일 대학교">예일 대학교</a>로 옮겼다.</p>
<p>1974년에 <a href="/wiki/%EC%B7%8C%EC%9E%A5%EC%95%94" title="췌장암">췌장암</a>으로 사망하였다.</p>
