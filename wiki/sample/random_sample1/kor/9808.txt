<p><b>에타 중간자</b>와 <b>에타 프라임 중간자</b>는 <a href="/wiki/%EC%9C%84_%EC%BF%BC%ED%81%AC" title="위 쿼크">위 쿼크</a>와 <a href="/wiki/%EC%95%84%EB%9E%98_%EC%BF%BC%ED%81%AC" title="아래 쿼크">아래 쿼크</a>, <a href="/wiki/%EA%B8%B0%EB%AC%98_%EC%BF%BC%ED%81%AC" title="기묘 쿼크">기묘 쿼크</a>가 섞여 만들어진 중성 <a class="new" href="/w/index.php?title=%EC%9C%A0%EC%82%AC%EC%8A%A4%EC%B9%BC%EB%9D%BC&amp;action=edit&amp;redlink=1" title="유사스칼라 (없는 문서)">유사스칼라</a> <a href="/wiki/%EC%A4%91%EA%B0%84%EC%9E%90" title="중간자">중간자</a>의 일종이다. 다른 유사스칼라 <a class="new" href="/w/index.php?title=%EC%BF%BC%EC%BD%94%EB%8A%84&amp;action=edit&amp;redlink=1" title="쿼코늄 (없는 문서)">쿼코늄</a>(중성 중간자)도 첨자를 붙여 "에타 중간자"라고 부른다. 예를 들어 η<sub>c</sub>는 c<span style="text-decoration:overline;">c</span>의 중간자다.</p>

</math></span><img alt="\pi ^{{0}}{}'" aria-hidden="true" class="mwe-math-fallback-image-inline" src="https://wikimedia.org/api/rest_v1/media/math/render/svg/e2201d292bc13c2fd180bf7ca443992f15cefb1d" style="vertical-align: -0.338ex; width:3.098ex; height:2.676ex;"/></span>라는 이름으로 예측하였다.<sup class="reference" id="cite_ref-1"><a href="#cite_note-1">[1]</a></sup> 에타 중간자는 <a href="/wiki/1961%EB%85%84" title="1961년">1961년</a>에 <a href="/wiki/%EB%AF%B8%EA%B5%AD" title="미국">미국</a>의 <a class="mw-redirect" href="/wiki/%EB%A1%9C%EB%A0%8C%EC%8A%A4_%EB%B2%84%ED%81%B4%EB%A6%AC_%EC%97%B0%EA%B5%AC%EC%86%8C" title="로렌스 버클리 연구소">로렌스 버클리 연구소</a>의 <a class="new" href="/w/index.php?title=%EB%B2%A0%EB%B0%94%ED%8A%B8%EB%A1%A0&amp;action=edit&amp;redlink=1" title="베바트론 (없는 문서)">베바트론</a>에서 <a class="mw-redirect" href="/wiki/%ED%8C%8C%EC%9D%B4%EC%98%A8" title="파이온">파이온</a>과 <a href="/wiki/%ED%95%B5%EC%9E%90" title="핵자">핵자</a>를 충돌하며 발견되었다.<sup class="reference" id="cite_ref-2"><a href="#cite_note-2">[2]</a></sup> 에타 프라임 중간자는 <a href="/wiki/1964%EB%85%84" title="1964년">1964년</a>에 <a href="/wiki/%EB%AF%B8%EA%B5%AD" title="미국">미국</a>의 <a class="mw-redirect" href="/wiki/%EB%A1%9C%EB%A0%8C%EC%8A%A4_%EB%B2%84%ED%81%B4%EB%A6%AC_%EC%97%B0%EA%B5%AC%EC%86%8C" title="로렌스 버클리 연구소">로렌스 버클리 연구소</a><sup class="reference" id="cite_ref-3"><a href="#cite_note-3">[3]</a></sup> 와 <a class="new" href="/w/index.php?title=%EB%B8%8C%EB%A3%A9%ED%97%A4%EC%9D%B4%EB%B8%90_%EC%97%B0%EA%B5%AC%EC%86%8C&amp;action=edit&amp;redlink=1" title="브룩헤이븐 연구소 (없는 문서)">브룩헤이븐 연구소</a><sup class="reference" id="cite_ref-4"><a href="#cite_note-4">[4]</a></sup> 에서 동시에 발견되었다. 맵시 에타는 <a href="/wiki/1980%EB%85%84" title="1980년">1980년</a>에 두 공동연구 (Partridge et al.<sup class="reference" id="cite_ref-5"><a href="#cite_note-5">[5]</a></sup> 과 Himel et al.<sup class="reference" id="cite_ref-6"><a href="#cite_note-6">[6]</a></sup>)에서 독립적으로 발견하였다.</p>
<h2><span class="mw-headline" id=".ED.8A.B9.EC.84.B1">특성</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%97%90%ED%83%80_%EC%A4%91%EA%B0%84%EC%9E%90&amp;action=edit&amp;section=2" title="부분 편집: 특성">원본 편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>에타 중간자와 에타 프라임 중간자의 <a href="/wiki/%EC%A7%88%EB%9F%89" title="질량">질량</a>의 차는 <a href="/wiki/%EC%BF%BC%ED%81%AC" title="쿼크">쿼크</a> 모형의 예측 (0) 보다 크다. 이를 <b>에타-에타 프라임 문제</b>라고 한다. 이 현상은 <a href="/wiki/%EC%88%9C%EA%B0%84%EC%9E%90" title="순간자">순간자</a>로 설명할 수 있다.</p>
