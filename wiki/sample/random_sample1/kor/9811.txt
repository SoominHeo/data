<p><b>에토레 마요라나</b>(<span style="font-size: smaller;"><a href="/wiki/%EC%9D%B4%ED%83%88%EB%A6%AC%EC%95%84%EC%96%B4" title="이탈리아어">이탈리아어</a>:</span> <span lang="it" xml:lang="it">Ettore Majorana</span>, <a href="/wiki/1906%EB%85%84" title="1906년">1906년</a> <a href="/wiki/8%EC%9B%94_5%EC%9D%BC" title="8월 5일">8월 5일</a> ~ <a href="/wiki/1959%EB%85%84" title="1959년">1959년</a> 이후에 사망 추정)는 <a href="/wiki/%EC%9D%B4%ED%83%88%EB%A6%AC%EC%95%84" title="이탈리아">이탈리아</a>의 <a href="/wiki/%EB%AC%BC%EB%A6%AC%ED%95%99%EC%9E%90" title="물리학자">물리학자</a>이다. <a href="/wiki/%EC%A4%91%EC%84%B1%EB%AF%B8%EC%9E%90" title="중성미자">중성미자</a>의 질량을 연구하였으나, 젋은 나이에 갑자기 실종되었다. 주요 업적으로는 <a class="new" href="/w/index.php?title=%EB%A7%88%EC%9A%94%EB%9D%BC%EB%82%98_%EB%B0%A9%EC%A0%95%EC%8B%9D&amp;action=edit&amp;redlink=1" title="마요라나 방정식 (없는 문서)">마요라나 방정식</a>(Majorana equation)과 <a class="new" href="/w/index.php?title=%EB%A7%88%EC%9A%94%EB%9D%BC%EB%82%98_%EC%9E%85%EC%9E%90&amp;action=edit&amp;redlink=1" title="마요라나 입자 (없는 문서)">마요라나 입자</a>의 예측이 있다.</p>
<h2><span class="mw-headline" id=".EC.83.9D.EC.95.A0">생애</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%97%90%ED%86%A0%EB%A0%88_%EB%A7%88%EC%9A%94%EB%9D%BC%EB%82%98&amp;action=edit&amp;section=1" title="부분 편집: 생애">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p><a href="/wiki/%EC%8B%9C%EC%B9%A0%EB%A6%AC%EC%95%84" title="시칠리아">시칠리아</a>의 <a href="/wiki/%EC%B9%B4%ED%83%80%EB%8B%88%EC%95%84" title="카타니아">카타니아</a>에서 태어났고, 어려서부터 <a href="/wiki/%EC%97%94%EB%A6%AC%EC%BD%94_%ED%8E%98%EB%A5%B4%EB%AF%B8" title="엔리코 페르미">엔리코 페르미</a>의 <a href="/wiki/%EB%A1%9C%EB%A7%88_%EB%9D%BC_%EC%82%AC%ED%94%BC%EC%97%94%EC%B0%A8_%EB%8C%80%ED%95%99%EA%B5%90" title="로마 라 사피엔차 대학교">로마 라 사피엔차 대학교</a> 연구소 조수로서 일했다.</p>
<p><a href="/wiki/1923%EB%85%84" title="1923년">1923년</a>에 <a href="/wiki/%EB%A1%9C%EB%A7%88_%EB%9D%BC_%EC%82%AC%ED%94%BC%EC%97%94%EC%B0%A8_%EB%8C%80%ED%95%99%EA%B5%90" title="로마 라 사피엔차 대학교">로마 라 사피엔차 대학교</a>에 입학하여 공학을 연구하였으나, <a class="mw-redirect" href="/wiki/%EC%97%90%EB%B0%80%EB%A6%AC%EC%98%A4_%EC%84%B8%EA%B7%B8%EB%A0%88" title="에밀리오 세그레">에밀리오 세그레</a>의 권고로 <a href="/wiki/1928%EB%85%84" title="1928년">1928년</a>에 <a href="/wiki/%EB%AC%BC%EB%A6%AC%ED%95%99" title="물리학">물리학</a>으로 전공을 바꿨다. <a href="/wiki/1928%EB%85%84" title="1928년">1928년</a>에 학부생 시절에 첫 물리학 논문을 발표하였다. 이는 오늘날 <a class="new" href="/w/index.php?title=%ED%86%A0%EB%A8%B8%EC%8A%A4-%ED%8E%98%EB%A5%B4%EB%AF%B8_%EB%AA%A8%ED%98%95&amp;action=edit&amp;redlink=1" title="토머스-페르미 모형 (없는 문서)">토머스-페르미 모형</a>으로 불리는데, 이는 <a class="new" href="/w/index.php?title=%EB%9F%AC%EC%9B%B0%EB%A6%B0_%ED%86%A0%EB%A8%B8%EC%8A%A4&amp;action=edit&amp;redlink=1" title="러웰린 토머스 (없는 문서)">러웰린 토머스</a>(<span style="font-size: smaller;"><a href="/wiki/%EC%98%81%EC%96%B4" title="영어">영어</a>:</span> <span lang="en" xml:lang="en">Llewellyn Thomas</span>)가 거의 동시에 같은 내용의 논문을 발표했기 때문이다. <a href="/wiki/1929%EB%85%84" title="1929년">1929년</a>에 졸업하였다.</p>
<p>졸업후 마요라나는 <a href="/wiki/%EB%AC%BC%EB%A6%AC%ED%95%99" title="물리학">물리학</a> 연구를 계속했으나, 대부분 주목받지 못했다. <a href="/wiki/1932%EB%85%84" title="1932년">1932년</a>에 <a href="/wiki/%EB%A1%9C%EB%9F%B0%EC%B8%A0_%EA%B5%B0" title="로런츠 군">로런츠 군</a>의 무한 차원 표현에 대한 논문을 저술하였고, 또한 <a class="mw-redirect" href="/wiki/%EC%9D%B4%EB%A0%8C_%ED%80%B4%EB%A6%AC" title="이렌 퀴리">이렌 퀴리</a>와 <a href="/wiki/%ED%94%84%EB%A0%88%EB%8D%B0%EB%A6%AC%ED%81%AC_%EC%A1%B8%EB%A6%AC%EC%98%A4%ED%80%B4%EB%A6%AC" title="프레데리크 졸리오퀴리">프레데리크 졸리오퀴리</a>의 실험 결과가 <a href="/wiki/%EC%96%91%EC%84%B1%EC%9E%90" title="양성자">양성자</a>와 비슷한 질량을 가진 중성 입자에 의한 것이라고 제안하였다. <sup class="reference" id="cite_ref-1"><a href="#cite_note-1">[1]</a></sup> 이는 오늘날 <a href="/wiki/%EC%A4%91%EC%84%B1%EC%9E%90" title="중성자">중성자</a>의 존재에 대한 최초의 예측으로 여겨진다. 그러나 마요라나는 (<a href="/wiki/%EC%97%94%EB%A6%AC%EC%BD%94_%ED%8E%98%EB%A5%B4%EB%AF%B8" title="엔리코 페르미">엔리코 페르미</a>의 권고를 무시하고) 이를 출판하지 않았고, <a href="/wiki/%EC%A4%91%EC%84%B1%EC%9E%90" title="중성자">중성자</a>의 발견에 대한 공로는 <a href="/wiki/%EC%98%81%EA%B5%AD" title="영국">영국</a>의 <a href="/wiki/%EB%AC%BC%EB%A6%AC%ED%95%99" title="물리학">물리학자</a>인 <a href="/wiki/%EC%A0%9C%EC%9E%84%EC%8A%A4_%EC%B1%84%EB%93%9C%EC%9C%85" title="제임스 채드윅">제임스 채드윅</a>에게 가게 되었다.</p>
<p><a href="/wiki/1933%EB%85%84" title="1933년">1933년</a> 초에 마요라나는 <a href="/wiki/%EC%9D%B4%ED%83%88%EB%A6%AC%EC%95%84" title="이탈리아">이탈리아</a>를 떠나 독일 <a href="/wiki/%EB%9D%BC%EC%9D%B4%ED%94%84%EC%B9%98%ED%9E%88" title="라이프치히">라이프치히</a>에서 <a href="/wiki/%EB%B2%A0%EB%A5%B4%EB%84%88_%ED%95%98%EC%9D%B4%EC%A0%A0%EB%B2%A0%EB%A5%B4%ED%81%AC" title="베르너 하이젠베르크">베르너 하이젠베르크</a>와 같이 연구를 시작하였고, 또 <a href="/wiki/%EC%BD%94%ED%8E%9C%ED%95%98%EA%B2%90" title="코펜하겐">코펜하겐</a>에서 <a href="/wiki/%EB%8B%90%EC%8A%A4_%EB%B3%B4%EC%96%B4" title="닐스 보어">닐스 보어</a>도 만났다. 그러나 마요라나는 <a href="/wiki/1933%EB%85%84" title="1933년">1933년</a> 가을에 급성 <a href="/wiki/%EC%9C%84%EC%97%BC" title="위염">위염</a>으로 앓아누우면서 다시 이탈리아로 돌아올 수 밖에 없었다. 이후 마요라나는 4년 동안 집 밖을 거의 떠나지 않았고, 논문도 몇 편 발표하지 않았다.</p>
<p><a href="/wiki/1937%EB%85%84" title="1937년">1937년</a>에 마요라나는 <a class="new" href="/w/index.php?title=%EB%82%98%ED%8F%B4%EB%A6%AC_%EB%8C%80%ED%95%99%EA%B5%90&amp;action=edit&amp;redlink=1" title="나폴리 대학교 (없는 문서)">나폴리 대학교</a> 이론 물리학 정교수로 임용되었다. <a href="/wiki/1938%EB%85%84" title="1938년">1938년</a> <a href="/wiki/3%EC%9B%94_25%EC%9D%BC" title="3월 25일">3월 25일</a>에 마요라나는 <a class="new" href="/w/index.php?title=%EB%82%98%ED%8F%B4%EB%A6%AC_%EB%AC%BC%EB%A6%AC%ED%95%99_%EC%97%B0%EA%B5%AC%EC%86%8C&amp;action=edit&amp;redlink=1" title="나폴리 물리학 연구소 (없는 문서)">나폴리 물리학 연구소</a>의 소장이었던 <a class="new" href="/w/index.php?title=%EC%95%88%ED%86%A0%EB%8B%88%EC%98%A4_%EC%B9%B4%EB%A0%90%EB%A6%AC&amp;action=edit&amp;redlink=1" title="안토니오 카렐리 (없는 문서)">안토니오 카렐리</a>(<span style="font-size: smaller;"><a href="/wiki/%EC%9D%B4%ED%83%88%EB%A6%AC%EC%95%84%EC%96%B4" title="이탈리아어">이탈리아어</a>:</span> <span lang="it" xml:lang="it">Antonio Carrelli</span>)에게 편지를 보냈는데, 편지에는 자신이 번복할 수 없는 결정을 내렸으며, 만약 자신이 실종될 경우 가족과 친구들에게 사과하는 내용이 담겨 있었다. <a href="/wiki/3%EC%9B%94_27%EC%9D%BC" title="3월 27일">3월 27일</a>에 마요라나는 <a href="/wiki/%ED%8C%94%EB%A0%88%EB%A5%B4%EB%AA%A8" title="팔레르모">팔레르모</a>에서 <a href="/wiki/%EB%82%98%ED%8F%B4%EB%A6%AC" title="나폴리">나폴리</a>로 가는 배에 탑승하였고, 자신의 삼촌이자 물리학자였던 <a class="new" href="/w/index.php?title=%ED%80%B4%EB%A6%AC%EB%85%B8_%EB%A7%88%EC%9A%94%EB%9D%BC%EB%82%98&amp;action=edit&amp;redlink=1" title="퀴리노 마요라나 (없는 문서)">퀴리노 마요라나</a>(<span style="font-size: smaller;"><a href="/wiki/%EC%9D%B4%ED%83%88%EB%A6%AC%EC%95%84%EC%96%B4" title="이탈리아어">이탈리아어</a>:</span> <span lang="it" xml:lang="it">Quirino Majorana</span>)에게 의미심장한 내용의 편지를 보낸 뒤 실종되었다. 그러나 <a href="/wiki/1997%EB%85%84" title="1997년">1997년</a>에 <a href="/wiki/%EB%B2%A0%EB%84%A4%EC%88%98%EC%97%98%EB%9D%BC" title="베네수엘라">베네수엘라</a>의 한 정비공이 <a href="/wiki/1955%EB%85%84" title="1955년">1955년</a>에 자신의 정비소에서 차량 수리를 의뢰하고 함께 사진을 찍은 남자가 에토레 마요라나였다고 주장했다. 이 주장의 진위 여부를 확인하기 위해 <a href="/wiki/%EC%9D%B4%ED%83%88%EB%A6%AC%EC%95%84" title="이탈리아">이탈리아</a> 경찰이 정비공이 가진 사진을 분석했고, 사진 속의 남자가 에토레 마요라나가 맞음을 확인하면서 그가 적어도 <a href="/wiki/1955%EB%85%84" title="1955년">1955년</a>까지는 생존해있었음이 밝혀졌다.</p>
