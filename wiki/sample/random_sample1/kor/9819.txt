<p><b>에틸렌다이아민테트라아세트산</b>(ethylenediaminetetraacetic acid, EDTA, 에틸렌 다이아민 테트라 아세트산<sup class="reference" id="cite_ref-1"><a href="#cite_note-1">[1]</a></sup>)은 <a class="mw-redirect" href="/wiki/%EC%9C%A0%EA%B8%B0%ED%99%94%ED%95%A9%EB%AC%BC" title="유기화합물">유기화합물</a>의 일종이다. 화학식은 C<sub>10</sub>H<sub>16</sub>N<sub>2</sub>O<sub>8</sub>이다. 여섯 자리 <a href="/wiki/%EB%A6%AC%EA%B0%84%EB%93%9C" title="리간드">리간드</a>로 작용할 수 있으며 <a href="/wiki/%EA%B8%88%EC%86%8D" title="금속">금속</a> <a href="/wiki/%EC%9D%B4%EC%98%A8" title="이온">이온</a>과 결합하여 <a href="/wiki/%EC%B9%B4%EC%9D%B4%EB%9E%84%EC%84%B1" title="카이랄성">카이랄성</a>을 가진 <a href="/wiki/%ED%82%AC%EB%A0%88%EC%9D%B4%ED%8A%B8" title="킬레이트">킬레이트</a> 화합물을 만든다. EDTA는 <a href="/wiki/%EA%B8%88%EC%86%8D" title="금속">금속</a> <a href="/wiki/%EC%9D%B4%EC%98%A8" title="이온">이온</a>을 중심으로 하는 팔면체의 여섯 꼭짓점에 동시에 <a class="mw-redirect" href="/wiki/%EB%B0%B0%EC%9C%84%EA%B2%B0%ED%95%A9" title="배위결합">배위</a>할 수 있으며, 그 결과 중심 <a href="/wiki/%EA%B8%88%EC%86%8D" title="금속">금속</a>은 <a href="/wiki/%EB%A6%AC%EA%B0%84%EB%93%9C" title="리간드">리간드</a>에 의해 둘러싸여지게 된다. 따라서 EDTA는 특정 <a href="/wiki/%EA%B8%88%EC%86%8D" title="금속">금속</a> <a href="/wiki/%EC%9D%B4%EC%98%A8" title="이온">이온</a>에 대하여 강한 친화력을 가진다. <a class="mw-redirect" href="/wiki/%EB%B0%B1%EC%83%89" title="백색">백색</a> <a href="/wiki/%EA%B2%B0%EC%A0%95" title="결정">결정</a> 또는 분말로 존재한다. 245℃에서 분해된다. <a class="mw-redirect" href="/wiki/%EB%B6%84%EC%9E%90%EB%9F%89" title="분자량">분자량</a>은 292.24이고, <a href="/wiki/%EB%B0%80%EB%8F%84" title="밀도">밀도</a>는 0.86g/cm<sup>3</sup>이다. 사<a href="/wiki/%EC%96%91%EC%84%B1%EC%9E%90" title="양성자">양성자</a><a href="/wiki/%EC%82%B0_(%ED%99%94%ED%95%99)" title="산 (화학)">산</a>으로 작용할 수 있다.</p>

<h2><span class="mw-headline" id=".ED.95.A9.EC.84.B1">합성</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%97%90%ED%8B%B8%EB%A0%8C%EB%8B%A4%EC%9D%B4%EC%95%84%EB%AF%BC%ED%85%8C%ED%8A%B8%EB%9D%BC%EC%95%84%EC%84%B8%ED%8A%B8%EC%82%B0&amp;action=edit&amp;section=1" title="부분 편집: 합성">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>EDTA는 <a href="/wiki/%EC%97%90%ED%8B%B8%EB%A0%8C%EB%8B%A4%EC%9D%B4%EC%95%84%EB%AF%BC" title="에틸렌다이아민">에틸렌다이아민</a>, <a href="/wiki/%ED%8F%BC%EC%95%8C%EB%8D%B0%ED%95%98%EC%9D%B4%EB%93%9C" title="폼알데하이드">폼알데하이드</a>, <a href="/wiki/%EB%AC%BC" title="물">물</a>, 그리고 <a href="/wiki/%EC%82%AC%EC%9D%B4%EC%95%88%ED%99%94_%EB%82%98%ED%8A%B8%EB%A5%A8" title="사이안화 나트륨">사이안화 나트륨</a>으로부터 합성된다.</p>
<h2><span class="mw-headline" id=".ED.99.9C.EC.9A.A9">활용</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%97%90%ED%8B%B8%EB%A0%8C%EB%8B%A4%EC%9D%B4%EC%95%84%EB%AF%BC%ED%85%8C%ED%8A%B8%EB%9D%BC%EC%95%84%EC%84%B8%ED%8A%B8%EC%82%B0&amp;action=edit&amp;section=2" title="부분 편집: 활용">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>EDTA는 매우 넓은 범위에서 여러가지 용도로 사용된다. 대표적인 활용의 예는 다음과 같은 것이 있다.</p>
<ul>
<li>EDTA는 욕조세제의 첨가물이기도 한데, 이는 EDTA가 <a href="/wiki/%EB%B9%84%EB%88%84" title="비누">비누</a>의 음이온 성분과 반응하여 욕조에 끼는 때의 주성분인 침전을 형성하는 <a href="/wiki/%EC%B9%BC%EC%8A%98" title="칼슘">Ca</a><sup>2+</sup><a href="/wiki/%EC%9D%B4%EC%98%A8" title="이온">이온</a>과 반응하기 때문에 침전의 용해도를 증가시켜 때가 씻겨질 수 있게 하기 때문이다.</li>
<li>EDTA는 <a href="/wiki/%EB%AC%BC" title="물">물</a> 속에 들어있는 <a href="/wiki/%EA%B8%88%EC%86%8D" title="금속">금속</a> <a href="/wiki/%EC%9D%B4%EC%98%A8" title="이온">이온</a>을 제거하는 데도 이용된다.</li>
<li>EDTA는 <a href="/wiki/%EB%82%A9" title="납">Pb</a><sup>2+</sup><a href="/wiki/%EC%9D%B4%EC%98%A8" title="이온">이온</a>과 강한 친화성을 가지기 때문에 <a href="/wiki/%EB%82%A9" title="납">납</a>중독의 치료제로도 사용되어 왔다.<sup class="reference" id="cite_ref-2"><a href="#cite_note-2">[2]</a></sup></li>
<li>EDTA와 <a href="/wiki/%EC%B2%A0" title="철">철</a> <a href="/wiki/%EC%9D%B4%EC%98%A8" title="이온">이온</a>의 착물은 <a href="/wiki/%EC%8B%9D%EB%AC%BC" title="식물">식물</a>에 <a href="/wiki/%EC%B2%A0" title="철">철</a> 성분을 공급하는 데 이용되기도 한다.</li>
<li>EDTA로 식용유에 존재하는 <a class="mw-redirect" href="/wiki/%EC%82%B0%ED%99%94" title="산화">산화</a>반응의 <a href="/wiki/%EC%B4%89%EB%A7%A4" title="촉매">촉매</a>인 <a href="/wiki/%EA%B8%88%EC%86%8D" title="금속">금속</a> <a href="/wiki/%EC%9D%B4%EC%98%A8" title="이온">이온</a>을 제거하여 식용유가 썩는 것을 방지할 수 있다.</li>
<li><a href="/wiki/%EC%83%9D%ED%99%94%ED%95%99" title="생화학">생화학</a>이나 <a href="/wiki/%EB%B6%84%EC%9E%90%EC%83%9D%EB%AC%BC%ED%95%99" title="분자생물학">분자생물학</a>에서는 <a href="/wiki/DNA" title="DNA">DNA</a>나 <a href="/wiki/%EB%8B%A8%EB%B0%B1%EC%A7%88" title="단백질">단백질</a>에 관련된 실험을 할 때, 시료가 <a href="/wiki/%ED%9A%A8%EC%86%8C" title="효소">효소</a>로 인하여 손상되는 것을 막기 위해 <a href="/wiki/%ED%9A%A8%EC%86%8C" title="효소">효소</a>가 필요로 하는 <a href="/wiki/%EA%B8%88%EC%86%8D" title="금속">금속</a> <a href="/wiki/%EC%9D%B4%EC%98%A8" title="이온">이온</a>을 제거할 목적으로 EDTA를 사용한다.</li>
<li>EDTA는 <a href="/wiki/%ED%98%88%EC%95%A1" title="혈액">혈액</a>의 응고방지제로도 이용된다.</li>
<li><a href="/wiki/%EB%B6%84%EC%84%9D%ED%99%94%ED%95%99" title="분석화학">분석화학</a>에서는 EDTA를 주로 <a href="/wiki/%EA%B8%88%EC%86%8D" title="금속">금속</a> <a href="/wiki/%EC%9D%B4%EC%98%A8" title="이온">이온</a>의 분석, 분리, 제거 등에 활용한다.</li>
<li>EDTA는 완충용액의 성분으로 사용될 수 있다.</li>
</ul>
