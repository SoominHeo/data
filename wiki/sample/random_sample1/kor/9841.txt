<p><b>엔비디아</b>(<span lang="en" xml:lang="en">NVIDIA</span>, <span style="font-size:small"><a href="/wiki/%EB%82%98%EC%8A%A4%EB%8B%A5" title="나스닥">나스닥</a>: <a class="external text" href="http://quotes.nasdaq.com/asp/SummaryQuote.asp?symbol=NVDA&amp;selected=NVDA" rel="nofollow"><b>NVDA</b></a></span>)는 <a href="/wiki/%EC%BB%B4%ED%93%A8%ED%84%B0" title="컴퓨터">컴퓨터</a>용 <a href="/wiki/%EA%B7%B8%EB%9E%98%ED%94%BD_%EC%B2%98%EB%A6%AC_%EC%9E%A5%EC%B9%98" title="그래픽 처리 장치">그래픽 처리 장치</a>와 멀티미디어 장치를 개발, 제조하는 회사이다.</p>
<p><a href="/wiki/%EC%97%91%EC%8A%A4%EB%B0%95%EC%8A%A4" title="엑스박스">엑스박스</a>와 <a href="/wiki/%ED%94%8C%EB%A0%88%EC%9D%B4%EC%8A%A4%ED%85%8C%EC%9D%B4%EC%85%98_3" title="플레이스테이션 3">플레이스테이션 3</a>과 같은 비디오 게임기에 그래픽 카드 칩셋을 공급하였으며 <a href="/wiki/2005%EB%85%84" title="2005년">2005년</a> <a href="/wiki/12%EC%9B%94_14%EC%9D%BC" title="12월 14일">12월 14일</a>, 엔비디아는 ULI를 인수했다. 이 회사는 ATI 칩셋에 쓰이는 사우스 브릿지를 생산하고 있었다. 본사는 <a href="/wiki/%EC%BA%98%EB%A6%AC%ED%8F%AC%EB%8B%88%EC%95%84_%EC%A3%BC" title="캘리포니아 주">캘리포니아 주</a>의 샌타클라라에 있다.</p>

<h2><span class="mw-headline" id=".EC.A3.BC.EC.9A.94_.EC.A0.9C.ED.92.88">주요 제품</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%97%94%EB%B9%84%EB%94%94%EC%95%84&amp;action=edit&amp;section=1" title="부분 편집: 주요 제품">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>엔비디아의 주요 제품에는 "<a href="/wiki/%EC%A7%80%ED%8F%AC%EC%8A%A4" title="지포스">지포스</a>"(GeForce) 시리즈 브랜드로 유명한 그래픽 카드 칩셋과 "<a href="/wiki/%EC%97%94%ED%8F%AC%EC%8A%A4" title="엔포스">엔포스</a>"(nForce) 브랜드로 알려진 메인보드 칩셋이 있다. 이밖에 컴퓨터 그래픽 전문가를 위해 만든 그래픽 카드 칩셋 <a class="mw-redirect" href="/wiki/%EC%BF%BC%EB%93%9C%EB%A1%9C" title="쿼드로">쿼드로</a>(Quadro) 시리즈와 <a href="/wiki/%EA%B3%A0%EC%84%B1%EB%8A%A5_%EC%BB%B4%ED%93%A8%ED%8C%85" title="고성능 컴퓨팅">고성능 컴퓨팅</a>용 카드인 <a href="/wiki/%EC%97%94%EB%B9%84%EB%94%94%EC%95%84_%ED%85%8C%EC%8A%AC%EB%9D%BC" title="엔비디아 테슬라">테슬라</a>(Tesla)시리즈도 있다.</p>
<h2><span class="mw-headline" id=".EC.97.AD.EC.82.AC">역사</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%97%94%EB%B9%84%EB%94%94%EC%95%84&amp;action=edit&amp;section=2" title="부분 편집: 역사">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p><a class="new" href="/w/index.php?title=Jen-Hsun_Huang&amp;action=edit&amp;redlink=1" title="Jen-Hsun Huang (없는 문서)">Jen-Hsun Huang</a>, <a class="new" href="/w/index.php?title=Chris_Malachowsky&amp;action=edit&amp;redlink=1" title="Chris Malachowsky (없는 문서)">Chris Malachowsky</a>, <a class="new" href="/w/index.php?title=Curtis_Priem&amp;action=edit&amp;redlink=1" title="Curtis Priem (없는 문서)">Curtis Priem</a>은 1993년 1월 회사를 세웠으며, 1993년 4월 캘리포니아에서 사업을 시작했다. 리바 계열 그래픽 칩셋을 만들기까지 이 회사는 잘 알려져 있지 않았다. 1999년 <a href="/wiki/%EB%82%98%EC%8A%A4%EB%8B%A5" title="나스닥">나스닥</a>에 상장하였고, 그 해 5월에 천만 번째 그래픽 칩셋을 생산하였다. 2000년, 부두 시리즈로 잘 알려진 <a href="/wiki/3dfx" title="3dfx">3dfx</a> 사를 인수하였다. 엔비디아는 실리콘 그래픽스 같은 OEM 회사들과 친한 관계를 맺기 시작했으며 2002년 2월, 1억 번째 그래픽 칩셋을 생산했다.</p>
<p>현재 엔비디아와 <a class="mw-redirect" href="/wiki/ATI" title="ATI">ATI</a>는 많은 컴퓨터에 사용되고 있는 그래픽 칩셋을 생산하고 있다. 엔비디아의 주 제품인 지포스 시리즈는 <a href="/wiki/1999%EB%85%84" title="1999년">1999년</a> <a href="/wiki/%EC%A7%80%ED%8F%AC%EC%8A%A4_256" title="지포스 256">지포스 256</a>으로 시작되었다. 현재 <a href="/wiki/%EC%A7%80%ED%8F%AC%EC%8A%A4" title="지포스">지포스</a> 시리즈는 데스크톱, 지포스 Go 시리즈는 노트북용으로 사용되고 있다. 한편, PDA 등의 장치를 위한 고포스 시리즈도 있다.</p>
<p>자체 공장이 없기 때문에 칩 제작은 <a href="/wiki/IBM" title="IBM">IBM</a>, <a href="/wiki/ST%EB%A7%88%EC%9D%B4%ED%81%AC%EB%A1%9C%EC%9D%BC%EB%A0%89%ED%8A%B8%EB%A1%9C%EB%8B%89%EC%8A%A4" title="ST마이크로일렉트로닉스">ST마이크로일렉트로닉스</a>, <a href="/wiki/TSMC" title="TSMC">TSMC</a> 등의 회사에서 한다. 이로 인해 공급과 수요의 물량에 차이가 나기도 한다.</p>
<p>2016년 8월 기준으로 서버시장에 진출한데 이어 자율주행차 전용 반도체를 공개하며 인공지능기술과 관련된 사업영역을 점점 확대하고 있다.</p>
<h3><span class="mw-headline" id=".EC.9E.A5.EC.B9.98_.EB.93.9C.EB.9D.BC.EC.9D.B4.EB.B2.84.EC.9D.98_.EC.9D.B4.EB.A6.84">장치 드라이버의 이름</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%97%94%EB%B9%84%EB%94%94%EC%95%84&amp;action=edit&amp;section=3" title="부분 편집: 장치 드라이버의 이름">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>엔비디아 그래픽 카드는 오래된 장치부터 최신의 장치까지 두루 지원할 수 있는 통합형 <a href="/wiki/%EC%9E%A5%EC%B9%98_%EB%93%9C%EB%9D%BC%EC%9D%B4%EB%B2%84" title="장치 드라이버">장치 드라이버</a>를 제공하는 데에 애쓰고 있다. 처음에는 한동안 데토네이터(Detonator)라는 이름을 사용하였으나, 포스웨어(ForceWare)를 거쳐 지금은 지포스(GeForce)라는 이름을 사용한다. 드라이버가 업데이트됨에 따라, 특히 <a class="mw-redirect" href="/wiki/%EC%9C%88%EB%8F%84_%EB%B9%84%EC%8A%A4%ED%83%80" title="윈도 비스타">윈도 비스타</a>의 출현으로 오래된 장치들 가운데 대다수를 지원하지 않는 경우가 많아졌다. 고객들이 최근의 <a href="/wiki/%EA%B7%B8%EB%9E%98%ED%94%BD_%EC%B9%B4%EB%93%9C" title="그래픽 카드">그래픽 카드</a>를 구입할 것을 유도하려는 의도가 깔려 있다는 의견이 있다.</p>
<h3><span class="mw-headline" id=".EB.85.BC.EB.9E.80">논란</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%97%94%EB%B9%84%EB%94%94%EC%95%84&amp;action=edit&amp;section=4" title="부분 편집: 논란">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
