<h1 class="firstHeading" id="firstHeading" lang="ko">엔비디아 아이온</h1>
<div class="mw-body-content" id="bodyContent">

<p><b>엔비디아 아이온</b>(Nvidia Ion)은 저가격 휴대용 컴퓨터의 <a href="/wiki/%EB%A9%94%EC%9D%B8%EB%B3%B4%EB%93%9C" title="메인보드">메인보드</a>용으로 설계한 <a href="/wiki/%EC%97%94%EB%B9%84%EB%94%94%EC%95%84" title="엔비디아">엔비디아</a> 제품 라인이다. <a href="/wiki/%EA%B7%B8%EB%9E%98%ED%94%BD_%EC%B2%98%EB%A6%AC_%EC%9E%A5%EC%B9%98" title="그래픽 처리 장치">그래픽 처리 장치</a>와 <a href="/wiki/%EC%B9%A9%EC%85%8B" title="칩셋">칩셋</a>을 함께 사용한다. 현재 아이온은 1세대와 2세대 제품이 있다.</p>
<h2><span class="mw-headline" id=".EA.B0.9C.EC.9A.94">개요</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%97%94%EB%B9%84%EB%94%94%EC%95%84_%EC%95%84%EC%9D%B4%EC%98%A8&amp;action=edit&amp;section=1" title="부분 편집: 개요">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>엔비디아 아이온 제품에는 지포스 9M 시리즈 MCP79MX (지포스 9400M G GPU 내장) 칩셋, DDR3-1066 또는 DDR2-800 SDRAM, <a href="/wiki/%EC%9D%B8%ED%85%94_%EC%95%84%ED%86%B0" title="인텔 아톰">인텔 아톰</a> 프로세서를 포함한다. 원래는 <a href="/wiki/%EB%84%B7%EB%B6%81" title="넷북">넷북</a>이나 <a class="new" href="/w/index.php?title=%EB%84%B7%ED%86%B1&amp;action=edit&amp;redlink=1" title="넷톱 (없는 문서)">넷톱</a> 장치용으로 설계된 <a class="new" href="/w/index.php?title=Pico-ITXe&amp;action=edit&amp;redlink=1" title="Pico-ITXe (없는 문서)">Pico-ITXe</a> 메인보드에 기반을 두었다. 2009년 2월에 <a href="/wiki/%EB%A7%88%EC%9D%B4%ED%81%AC%EB%A1%9C%EC%86%8C%ED%94%84%ED%8A%B8" title="마이크로소프트">마이크로소프트</a>는 <a class="mw-redirect" href="/wiki/%EC%9C%88%EB%8F%84_%EB%B9%84%EC%8A%A4%ED%83%80" title="윈도 비스타">윈도 비스타</a>에 맞추어 아이온 기반 플랫폼을 인증하였다.<sup class="reference" id="cite_ref-1"><a href="#cite_note-1">[1]</a></sup> 소형 아이온 기반 컴퓨터들은 2009년 중순에 출시되었다. 아이온 시스템은 <a href="/wiki/DirectX" title="DirectX">DirectX</a> 10.0, <a href="/wiki/OpenGL" title="OpenGL">OpenGL</a> 3.3과 호환된다. 그러므로 <a class="new" href="/w/index.php?title=VDPAU&amp;action=edit&amp;redlink=1" title="VDPAU (없는 문서)">VDPAU</a>와 <a href="/wiki/%EC%97%94%EB%B9%84%EB%94%94%EC%95%84_%ED%93%A8%EC%96%B4%EB%B9%84%EB%94%94%EC%98%A4" title="엔비디아 퓨어비디오">3세대 퓨어비디오 HD</a> 기능과 더불어 사실적인 7.1 서라운드 음향의 <a href="/wiki/1080p" title="1080p">1080p</a> 고선명 비디오를 재생할 수 있다.<sup class="reference" id="cite_ref-2"><a href="#cite_note-2">[2]</a></sup> 아이온 시스템은 <a href="/wiki/OpenCL" title="OpenCL">OpenCL</a>과 더불어 <a class="mw-redirect" href="/wiki/%EC%BF%A0%EB%8B%A4" title="쿠다">쿠다</a> (엔비디아의 <a href="/wiki/GPGPU" title="GPGPU">GPGPU</a> 기술)도 사용할 수 있다. ION-LE 기반 시스템은 동일한 기본 하드웨어를 공유하지만 비스타 및 DirectX 10 지원은 배제되어 있다.</p>
