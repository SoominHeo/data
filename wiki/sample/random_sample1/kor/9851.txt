<h1 class="firstHeading" id="firstHeading" lang="ko">엘라하</h1>
<div class="mw-body-content" id="bodyContent">
<div id="siteSub">위키백과, 우리 모두의 백과사전.</div>
<div id="contentSub"></div>
<div class="mw-jump" id="jump-to-nav">
					이동:					<a href="#mw-head">둘러보기</a>, 					<a href="#p-search">검색</a>

<div class="mw-content-ltr" dir="ltr" id="mw-content-text" lang="ko"><script>function mfTempOpenSection(id){var block=document.getElementById("mf-section-"+id);block.className+=" open-block";block.previousSibling.className+=" open-block";}</script><p><b>엘라하</b>(<span style="font-size: smaller;"><a href="/wiki/%EA%B3%A0%EB%8C%80_%EC%95%84%EC%9D%BC%EB%9E%9C%EB%93%9C%EC%96%B4" title="고대 아일랜드어">고대 아일랜드어</a>:</span> <span lang="sga" xml:lang="sga">Elatha</span>, <span style="font-size: smaller;"><a href="/wiki/%EC%95%84%EC%9D%BC%EB%9E%9C%EB%93%9C%EC%96%B4" title="아일랜드어">아일랜드어</a>:</span> <span lang="ga" xml:lang="ga">Ealadha</span>)는 <a href="/wiki/%EC%95%84%EC%9D%BC%EB%9E%9C%EB%93%9C_%EC%8B%A0%ED%99%94" title="아일랜드 신화">아일랜드 신화</a>에 나오는 <a href="/wiki/%ED%8F%AC%EB%AA%A8%EB%A5%B4" title="포모르">포모르</a>의 군주이다.</p>
<p>엘라하는 금발을 기른 미남자로 묘사된다. <a class="new" href="/w/index.php?title=%EB%8B%AC%EB%B0%94%ED%9D%90&amp;action=edit&amp;redlink=1" title="달바흐 (없는 문서)">달바흐</a>의 아들이며 <a href="/wiki/%ED%88%AC%EC%95%84%ED%95%98_%EB%8D%B0_%EB%8B%A4%EB%82%9C" title="투아하 데 다난">투아하 데 다난</a>의 <a href="/wiki/%EC%97%90%EB%A6%AC%EC%9A%B0" title="에리우">에리</a>와의 사이에 <a href="/wiki/%EB%B8%8C%EB%A0%88%EC%8A%A4" title="브레스">브레스</a>를 낳았다. 엘라하는 은으로 만든 배를 타고 바다를 건너 에리를 찾아와 아이를 갖게 했는데, 이때 금발의 젊은 청년의 모습을 하고 금으로 만든 옷을 입고 금으로 만든 <a href="/wiki/%ED%86%A0%ED%81%AC_(%EC%9E%A5%EC%8B%A0%EA%B5%AC)" title="토크 (장신구)">토크</a> 다섯 개를 끼고 있었다고 한다.</p>
<p><a href="/wiki/%ED%8B%B0%EB%9F%AC%EC%9D%B4_%EB%B2%8C%ED%8C%90_%EC%A0%84%ED%88%AC" title="티러이 벌판 전투">제2차 티러이 벌판 전투</a> 당시 <a href="/wiki/%EB%8B%A4%EA%B7%B8%EB%8B%A4_%EB%AA%A8%EB%A5%B4" title="다그다 모르">다그다</a>의 수금 <a href="/wiki/%EC%9A%B0%EC%96%B4%EB%84%88" title="우어너">우어너</a>를 포모르인들이 훔쳐 갔는데, 이때 엘라하가 우어너를 맡고 있었다.</p>
<p>엘라하는 <a href="/wiki/%EB%B8%8C%EB%A0%88%EC%8A%A4" title="브레스">브레스</a>의 아버지로 포모르라는 것이 통설이지만, <a href="/wiki/%EC%97%90%ED%9D%90%EB%88%84" title="에흐누">에흐누</a>(<a class="mw-redirect" href="/wiki/%EB%A3%A8_%EB%9D%BC%EB%B8%8C%EB%8B%A4" title="루 라브다">루</a>의 어머니)와의 사이에 <a href="/wiki/%EB%8B%A4%EA%B7%B8%EB%8B%A4_%EB%AA%A8%EB%A5%B4" title="다그다 모르">다그다</a>, <a class="mw-redirect" href="/wiki/%EC%98%A4%EA%B7%B8%EB%A7%88" title="오그마">오그마</a>를 낳았다고도 한다. 에흐누는 포모르 여자이기에 이 말이 맞다면 다그다와 오그마는 순혈 포모르가 된다. 이것은 매우 혼란스럽다. 어쩌면 포모르와 투아하 데 다난 사이의 분쟁은 구세대의 신들이 신세대의 신들로 교체되는 과정일 수도 있다.</p>
