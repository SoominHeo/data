<p><b>엘리슨 오니즈카</b>(Elison Shoji Onizuka, 鬼塚承次, <a href="/wiki/1946%EB%85%84" title="1946년">1946년</a> <a href="/wiki/6%EC%9B%94_24%EC%9D%BC" title="6월 24일">6월 24일</a> ~ <a href="/wiki/1986%EB%85%84" title="1986년">1986년</a> <a href="/wiki/1%EC%9B%94_28%EC%9D%BC" title="1월 28일">1월 28일</a>)는 <a href="/wiki/%EB%AF%B8%EA%B5%AD" title="미국">미국</a>의 우주 비행사이자 군인이다.</p>
<p><a href="/wiki/%ED%95%98%EC%99%80%EC%9D%B4_%EC%A3%BC" title="하와이 주">하와이 주</a>에서 <a href="/wiki/%EC%9D%BC%EB%B3%B8%EA%B3%84_%EB%AF%B8%EA%B5%AD%EC%9D%B8" title="일본계 미국인">일본계</a> 2세로 태어났으며, <a href="/wiki/%EC%BD%9C%EB%A1%9C%EB%9D%BC%EB%8F%84_%EB%8C%80%ED%95%99%EA%B5%90" title="콜로라도 대학교">콜로라도 대학교</a>를 졸업한 후 <a href="/wiki/1970%EB%85%84" title="1970년">1970년</a>에 <a href="/wiki/%EB%AF%B8%EA%B5%AD_%EA%B3%B5%EA%B5%B0" title="미국 공군">미국 공군</a>에 입대하였다. 그는 <a href="/wiki/1978%EB%85%84" title="1978년">1978년</a>에 우주 비행사로 뽑혔으며, <a href="/wiki/1985%EB%85%84" title="1985년">1985년</a> <a href="/wiki/1%EC%9B%94_24%EC%9D%BC" title="1월 24일">1월 24일</a>에 <a class="new" href="/w/index.php?title=STS-51-C&amp;action=edit&amp;redlink=1" title="STS-51-C (없는 문서)">STS-51-C</a> 디스커버리 미션을 수행하여 미국 최초로 아시아계 우주인이 되었다. 하지만, 이듬해인 <a href="/wiki/1986%EB%85%84" title="1986년">1986년</a> <a href="/wiki/1%EC%9B%94_28%EC%9D%BC" title="1월 28일">1월 28일</a>, <a href="/wiki/STS-51-L" title="STS-51-L">STS-51-L</a> 미션에 우주왕복선 전문 엔지니어로 <a href="/wiki/%EC%B1%8C%EB%A6%B0%EC%A0%80_%EC%9A%B0%EC%A3%BC%EC%99%95%EB%B3%B5%EC%84%A0" title="챌린저 우주왕복선">챌린저 우주왕복선</a>에 탑승했으나 <a href="/wiki/%EC%B1%8C%EB%A6%B0%EC%A0%80_%EC%9A%B0%EC%A3%BC%EC%99%95%EB%B3%B5%EC%84%A0_%ED%8F%AD%EB%B0%9C_%EC%82%AC%EA%B3%A0" title="챌린저 우주왕복선 폭발 사고">폭발 사고</a>로 사망했다.</p>
