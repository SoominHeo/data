<h1 class="firstHeading" id="firstHeading" lang="ko">엘리오 비토리니</h1>
<div class="mw-body-content" id="bodyContent">
<div id="siteSub">위키백과, 우리 모두의 백과사전.</div>

<div class="mw-content-ltr" dir="ltr" id="mw-content-text" lang="ko"><script>function mfTempOpenSection(id){var block=document.getElementById("mf-section-"+id);block.className+=" open-block";block.previousSibling.className+=" open-block";}</script><p><b>엘리오 비토리니</b>(Elio Vittorini, <a href="/wiki/1908%EB%85%84" title="1908년">1908년</a> <a href="/wiki/7%EC%9B%94_23%EC%9D%BC" title="7월 23일">7월 23일</a> ~ <a href="/wiki/1966%EB%85%84" title="1966년">1966년</a> <a href="/wiki/2%EC%9B%94_12%EC%9D%BC" title="2월 12일">2월 12일</a>)는 <a href="/wiki/%EC%9D%B4%ED%83%88%EB%A6%AC%EC%95%84" title="이탈리아">이탈리아</a>의 <a href="/wiki/%EC%86%8C%EC%84%A4%EA%B0%80" title="소설가">소설가</a>이다. <a href="/wiki/%EC%8B%9C%EC%B9%A0%EB%A6%AC%EC%95%84_%EC%84%AC" title="시칠리아 섬">시칠리아 섬</a> <a class="mw-redirect" href="/wiki/%EC%8B%9C%EB%9D%BC%EC%BF%A0%EC%82%AC" title="시라쿠사">시라쿠사</a> 시(市)에서 출생하였다. 공업학교를 졸업한 후 북부 <a href="/wiki/%EC%9D%B4%ED%83%88%EB%A6%AC%EC%95%84" title="이탈리아">이탈리아</a>로 이주하여 공장 기사의 생활을 하면서 첫 작품 &lt;소시민&gt;(1931)을 발표하였다. 또한 영미 문학의 번역에 손을 댔고 특히 <a class="mw-disambig" href="/wiki/%EB%A9%9C%EB%B9%8C" title="멜빌">멜빌</a>이나 <a class="mw-redirect" href="/wiki/%ED%8F%AC%ED%81%AC%EB%84%88" title="포크너">포크너</a>로부터 크게 영향을 받았다. 전쟁 중에는 지하조직에 참여, 저항 운동을 계속하여 <a href="/wiki/%ED%8C%8C%EC%8B%9C%EC%A6%98" title="파시즘">파시즘</a> 타도에 중요한 역할을 다하였다. 그동안의 경험이 독자적 문체를 낳게 하는 계기가 되어, &lt;시칠리아에서의 대화&gt;(1941)는 <a class="mw-redirect" href="/wiki/%ED%8C%8C%EB%B2%A0%EC%84%B8" title="파베세">파베세</a>의 작품과 더불어 <a class="new" href="/w/index.php?title=%EB%84%A4%EC%98%A4%EB%A6%AC%EC%96%BC%EB%A6%AC%EC%A6%98_%EB%AC%B8%ED%95%99&amp;action=edit&amp;redlink=1" title="네오리얼리즘 문학 (없는 문서)">네오리얼리즘 문학</a>의 모체를 이루었다. 전후에는 &lt;포리테크니코&gt;, &lt;메나보아&gt;지(誌) 등의 주필이 되어 <a class="new" href="/w/index.php?title=%EC%82%AC%EB%A5%B4%ED%8A%B8%EB%A5%B4_%ED%97%A4%EB%B0%8D%EC%9B%A8%EC%9D%B4&amp;action=edit&amp;redlink=1" title="사르트르 헤밍웨이 (없는 문서)">사르트르 헤밍웨이</a> 등과 긴밀한 연락을 취하면서 <a href="/wiki/%EC%9D%B4%ED%83%88%EB%A6%AC%EC%95%84" title="이탈리아">이탈리아</a> 문학계의 지도적 위치를 차지하였다. 그가 키우지 않은 전후(戰後)작가는 없다고 할 정도로 지도적 활동을 하였다. 대표작에는 &lt;인간과 비인간&gt;(1945), &lt;붉은 카네이션&gt;(1948) 등이 있다.</p>
