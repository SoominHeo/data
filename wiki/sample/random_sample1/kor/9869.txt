<p><b>엘리자베스 돌</b>(<span lang="en" xml:lang="en">Mary Elizabeth Hanford "Liddy" Dole</span>, <a href="/wiki/1936%EB%85%84" title="1936년">1936년</a> <a href="/wiki/7%EC%9B%94_29%EC%9D%BC" title="7월 29일">7월 29일</a> ~ )은 <a href="/wiki/%EB%AF%B8%EA%B5%AD" title="미국">미국</a>의 여성 정치인이다.</p>
<p><a href="/wiki/%EB%85%B8%EC%8A%A4%EC%BA%90%EB%A1%A4%EB%9D%BC%EC%9D%B4%EB%82%98_%EC%A3%BC" title="노스캐롤라이나 주">노스캐롤라이나 주</a> 출신이며, <a href="/wiki/%EB%93%80%ED%81%AC_%EB%8C%80%ED%95%99%EA%B5%90" title="듀크 대학교">듀크 대학교</a> 졸업 후 <a href="/wiki/%ED%95%98%EB%B2%84%EB%93%9C_%EB%8C%80%ED%95%99%EA%B5%90" title="하버드 대학교">하버드 대학교</a>로 진학하여 교육대학원에서 교육학 석사와 <a href="/wiki/%ED%95%98%EB%B2%84%EB%93%9C_%EB%A1%9C%EC%8A%A4%EC%BF%A8" title="하버드 로스쿨">하버드 로스쿨</a>에서 법무박사(J.D.) 학위를 취득했다. 정부에서 일하다가 상원의원 <a href="/wiki/%EB%B0%A5_%EB%8F%8C" title="밥 돌">밥 돌</a>과 만나게 되어 <a href="/wiki/1975%EB%85%84" title="1975년">1975년</a> 그의 두 번째 부인으로 결혼했다. 남편의 정치 활동을 돕는 동시에 <a href="/wiki/1983%EB%85%84" title="1983년">1983년</a> ~ <a href="/wiki/1987%EB%85%84" title="1987년">1987년</a> 연방 교통부 장관, <a href="/wiki/1989%EB%85%84" title="1989년">1989년</a> ~ <a href="/wiki/1990%EB%85%84" title="1990년">1990년</a> 연방 노동부 장관으로 재직했다. <a href="/wiki/1991%EB%85%84" title="1991년">1991년</a> ~ <a href="/wiki/1999%EB%85%84" title="1999년">1999년</a> 미국 <a class="mw-redirect" href="/wiki/%EC%A0%81%EC%8B%AD%EC%9E%90%EC%82%AC" title="적십자사">적십자사</a> 총재로 활동했다. <a href="/wiki/1996%EB%85%84" title="1996년">1996년</a> 남편 밥 돌이 공화당 대통령 후보로 나서자 남편의 대통령 선거 운동을 도왔으나, 적십자사 총재의 신분으로 적극적인 활동을 하지는 않았다. 미국의 유명 여성 정치인으로 <a href="/wiki/2000%EB%85%84" title="2000년">2000년</a> 대통령 선거에서 유력한 후보로 거론되었으나, 선거에 나서지 않았다. <a href="/wiki/2002%EB%85%84" title="2002년">2002년</a> <a href="/wiki/%EB%85%B8%EC%8A%A4%EC%BA%90%EB%A1%A4%EB%9D%BC%EC%9D%B4%EB%82%98_%EC%A3%BC" title="노스캐롤라이나 주">노스캐롤라이나 주</a>에서 연방 상원의원으로 당선되어, 이미 상원의원으로 활동한 바 있는 남편에 이어 미국에서는 최초로 부부가 모두 상원의원 선거에 본인의 힘으로 당선되는 기록을 세웠다. <a href="/wiki/2008%EB%85%84" title="2008년">2008년</a> 재선에서 <a class="mw-redirect" href="/wiki/%EB%AF%B8%EA%B5%AD_%EB%AF%BC%EC%A3%BC%EB%8B%B9" title="미국 민주당">민주당</a> 후보 <a class="new" href="/w/index.php?title=%EC%BC%80%EC%9D%B4_%ED%97%A4%EC%9D%B4%EA%B1%B4&amp;action=edit&amp;redlink=1" title="케이 헤이건 (없는 문서)">케이 헤이건</a>에게 패해 낙선했다.</p>
