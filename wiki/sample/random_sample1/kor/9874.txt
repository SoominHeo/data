<p><b>엘리자베스 앤 워런</b> (<span style="font-size: smaller;"><a href="/wiki/%EC%98%81%EC%96%B4" title="영어">영어</a>:</span> <span lang="en">Elizabeth Ann Warren</span><sup class="reference" id="cite_ref-2"><a href="#cite_note-2">[2]</a></sup>, <a href="/wiki/1949%EB%85%84" title="1949년">1949년</a> <a href="/wiki/6%EC%9B%94_22%EC%9D%BC" title="6월 22일">6월 22일</a><sup class="reference" id="cite_ref-autogenerated1_3-0"><a href="#cite_note-autogenerated1-3">[3]</a></sup>~)은 <a href="/wiki/%EB%AF%B8%EA%B5%AD" title="미국">미국</a>의 <a class="mw-redirect" href="/wiki/%ED%95%99%EC%9E%90" title="학자">학자</a>이자 현재 <a href="/wiki/%EB%AF%BC%EC%A3%BC%EB%8B%B9_(%EB%AF%B8%EA%B5%AD)" title="민주당 (미국)">민주당</a> 소속의 <a href="/wiki/%EB%A7%A4%EC%82%AC%EC%B6%94%EC%84%B8%EC%B8%A0_%EC%A3%BC" title="매사추세츠 주">매사추세츠 주</a> <a class="mw-redirect" href="/wiki/%EC%83%81%EC%9B%90%EC%9D%98%EC%9B%90" title="상원의원">상원의원</a>을 역임하고 있는 <a class="mw-redirect" href="/wiki/%EC%A0%95%EC%B9%98%EC%9D%B8" title="정치인">정치인</a>이다. 그녀는 <a href="/wiki/%ED%95%98%EB%B2%84%EB%93%9C_%EB%A1%9C%EC%8A%A4%EC%BF%A8" title="하버드 로스쿨">하버드 로스쿨</a>에서 주로 <a href="/wiki/%ED%8C%8C%EC%82%B0%EB%B2%95" title="파산법">파산법</a>을 강의하였다. 저명한 법학자로서, 그녀는 <a href="/wiki/%EC%83%81%EB%B2%95" title="상법">상법</a>에서 가장 많이 인용되는 학자들 중 하나이기도 하다.<sup class="reference" id="cite_ref-4"><a href="#cite_note-4">[4]</a></sup> 그녀는 강력한 <a href="/wiki/%EC%86%8C%EB%B9%84%EC%9E%90%EB%B3%B4%ED%98%B8" title="소비자보호">소비자보호</a> 운동 지지자로서 <a class="new" href="/w/index.php?title=%EC%86%8C%EB%B9%84%EC%9E%90%EC%9E%AC%EC%A0%95%EB%B3%B4%ED%98%B8%EB%B6%80&amp;action=edit&amp;redlink=1" title="소비자재정보호부 (없는 문서)">소비자재정보호부</a>의 제안과 창설에 큰 영향을 끼쳤다. 그녀는 또한 대중적으로 알려진 많은 논문과 저서를 썼으며, 미국 경제와 가계 재정과 관련해서 많은 인터뷰를 받기도 하였다.</p>
<p>1997년 <a href="/wiki/%EB%AF%B8%EA%B5%AD_%EC%9D%98%ED%9A%8C" title="미국 의회">미국 의회</a>가 설치해 운영한 <a class="new" href="/w/index.php?title=%ED%8C%8C%EC%82%B0%EC%A1%B0%EC%82%AC%EC%9C%84%EC%9B%90%ED%9A%8C&amp;action=edit&amp;redlink=1" title="파산조사위원회 (없는 문서)">파산조사위원회</a>의 고문을 지냈고, 1998년에는 <a class="new" href="/w/index.php?title=%EB%82%B4%EC%85%94%EB%84%90_%EB%A1%9C_%EC%A0%80%EB%84%90&amp;action=edit&amp;redlink=1" title="내셔널 로 저널 (없는 문서)">내셔널 로 저널</a>이 '미국의 가장 영향력 있는 여성 법률가 50인'으로 선정했다.</p>
<p><a class="mw-redirect" href="/wiki/2008%EB%85%84_%EA%B8%88%EC%9C%B5_%EC%9C%84%EA%B8%B0" title="2008년 금융 위기">2008년 금융 위기</a>가 오자, 그녀는 이후 <a class="new" href="/w/index.php?title=%EB%B6%80%EC%8B%A4%EC%9E%90%EC%82%B0%EA%B5%AC%EC%A0%9C_%ED%94%84%EB%A1%9C%EA%B7%B8%EB%9E%A8&amp;action=edit&amp;redlink=1" title="부실자산구제 프로그램 (없는 문서)">부실자산구제 프로그램</a>이 되는 <a class="new" href="/w/index.php?title=%EB%B6%80%EC%8B%A4%EA%B4%80%EB%A6%AC%EC%9C%84%EC%9B%90%ED%9A%8C&amp;action=edit&amp;redlink=1" title="부실관리위원회 (없는 문서)">부실관리위원회</a> 고문을 역임하였다. 이후 그녀는 <a href="/wiki/%EB%B2%84%EB%9D%BD_%EC%98%A4%EB%B0%94%EB%A7%88" title="버락 오바마">버락 오바마</a> 밑에서 <a href="/wiki/%EB%AF%B8%EA%B5%AD_%EB%8C%80%ED%86%B5%EB%A0%B9%EC%8B%A4" title="미국 대통령실">대통령 직속 고문</a>과 가계 재정과 관련하여 <a href="/wiki/%EB%AF%B8%EA%B5%AD_%EC%9E%AC%EB%AC%B4%EB%B6%80" title="미국 재무부">재무부 최고 고문</a>을 역임하였다. 2000년대 말에는 정치적으로 중요 인물로 부상하여 <a class="new" href="/w/index.php?title=%EB%82%B4%EC%85%94%EB%84%90_%EB%A1%9C_%EC%A0%80%EB%84%90&amp;action=edit&amp;redlink=1" title="내셔널 로 저널 (없는 문서)">내셔널 로 저널</a>이나 <a href="/wiki/%ED%83%80%EC%9E%84_100" title="타임 100">타임 100</a>에 선정되기도 하였다.</p>
<p>2011년 9월에 워런은 <a href="/wiki/%EA%B3%B5%ED%99%94%EB%8B%B9_(%EB%AF%B8%EA%B5%AD)" title="공화당 (미국)">공화당</a> 소속의 <a class="new" href="/w/index.php?title=%EC%8A%A4%EC%BD%A7_%EB%B8%8C%EB%9D%BC%EC%9A%B4_(%EC%A0%95%EC%B9%98%EC%9D%B8)&amp;action=edit&amp;redlink=1" title="스콧 브라운 (정치인) (없는 문서)">스콧 브라운</a> 현 매사추세츠 주 상원의원에 대항하는 선거의 후보로 출마하였다. 그녀는 <a href="/wiki/2012%EB%85%84" title="2012년">2012년</a> <a href="/wiki/11%EC%9B%94_6%EC%9D%BC" title="11월 6일">11월 6일</a>에 선거에서 승리하였고, 매사추세츠 주 최초의 여성 상원의원이 되었다. 그녀는 <a class="new" href="/w/index.php?title=%EC%83%81%EC%9B%90_%EB%85%B8%EB%A0%B9%ED%99%94%EB%B0%A9%EC%A7%80%EC%9C%84%EC%9B%90%ED%9A%8C&amp;action=edit&amp;redlink=1" title="상원 노령화방지위원회 (없는 문서)">상원 노령화방지위원회</a>와 <a class="new" href="/w/index.php?title=%EC%83%81%EC%9B%90_%EC%9D%80%ED%96%89%EC%A3%BC%EA%B1%B0%EB%8F%84%EC%8B%9C%EC%9C%84%EC%9B%90%ED%9A%8C&amp;action=edit&amp;redlink=1" title="상원 은행주거도시위원회 (없는 문서)">상원 은행주거도시위원회</a>, <a class="new" href="/w/index.php?title=%EC%83%81%EC%9B%90_%EB%B3%B4%EA%B1%B4%EA%B5%90%EC%9C%A1%EB%85%B8%EB%8F%99%EC%9C%84%EC%9B%90%ED%9A%8C&amp;action=edit&amp;redlink=1" title="상원 보건교육노동위원회 (없는 문서)">상원 보건교육노동위원회</a>에 소속되어있다.</p>
<p>그녀는 민주당 내 진보 세력의 상징으로 여겨지고 있으며<sup class="reference" id="cite_ref-5"><a href="#cite_note-5">[5]</a></sup><sup class="reference" id="cite_ref-6"><a href="#cite_note-6">[6]</a></sup>, <a href="/wiki/2016%EB%85%84_%EB%AF%B8%EA%B5%AD_%EB%8C%80%ED%86%B5%EB%A0%B9_%EC%84%A0%EA%B1%B0" title="2016년 미국 대통령 선거">2016년 대선</a>에서 민주당내 주요 후보로 부상하고 있다. 하지만 워런은 현재까지 2016년 대선 불출마를 고수하였고, 끝내 민주당 경선에도 출마하지 않았다.<sup class="reference" id="cite_ref-nightmare_7-0"><a href="#cite_note-nightmare-7">[7]</a></sup><sup class="reference" id="cite_ref-coulddefinitely_8-0"><a href="#cite_note-coulddefinitely-8">[8]</a></sup><sup class="reference" id="cite_ref-notgoingtorun_9-0"><a href="#cite_note-notgoingtorun-9">[9]</a></sup> 선거인단 선거에서 <a href="/wiki/%EB%B6%88%EC%B6%A9%EC%8B%A4%ED%95%9C_%EC%84%A0%EA%B1%B0%EC%9D%B8%EB%8B%A8" title="불충실한 선거인단">불충실한 선거인단</a>의 표 2표를 얻었다.</p>
<h2><span class="mw-headline" id=".EC.A0.80.EC.84.9C">저서</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%97%98%EB%A6%AC%EC%9E%90%EB%B2%A0%EC%8A%A4_%EC%9B%8C%EB%9F%B0&amp;action=edit&amp;section=1" title="부분 편집: 저서">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>2004년 엘리자베스 워런은 딸 아멜리아 워런 티아기와 함께 <a class="new" href="/w/index.php?title=%EB%A7%9E%EB%B2%8C%EC%9D%B4%EC%9D%98_%ED%95%A8%EC%A0%95:_%EC%99%9C_%EC%A4%91%EC%82%B0%EC%B8%B5_%EC%97%84%EB%A7%88%EC%99%80_%EC%95%84%EB%B9%A0%EB%93%A4%EC%9D%80_%ED%8C%8C%EC%82%B0%ED%95%98%EB%8A%94%EA%B0%80&amp;action=edit&amp;redlink=1" title="맞벌이의 함정: 왜 중산층 엄마와 아빠들은 파산하는가 (없는 문서)">맞벌이의 함정: 왜 중산층 엄마와 아빠들은 파산하는가</a>라는 책을 저술했다. 미국은 맞벌이가 외벌이보다 50%나 더 버는데, 한국은 겨우 15% 더 번다고 조사되었다.<sup class="reference" id="cite_ref-10"><a href="#cite_note-10">[10]</a></sup> 그런데도 미국의 맞벌이는 외벌이 보다 더 많이 파산을 한다면서, 그 원인은 무엇인지에 대해 쓴 책이다.</p>
<p>하버드 로스쿨 파산법 교수 시절 <a class="new" href="/w/index.php?title=%EC%86%8C%EB%B9%84%EC%9E%90_%ED%8C%8C%EC%82%B0_%ED%94%84%EB%A1%9C%EC%A0%9D%ED%8A%B8&amp;action=edit&amp;redlink=1" title="소비자 파산 프로젝트 (없는 문서)">소비자 파산 프로젝트</a>의 일환으로 파산한 2,220명을 인터뷰했다. 그 결과 이들의 대부분은 신용카드를 긁어댄 청년도, 기댈 데 없이 곤궁한 노인도 아니라는 사실을 발견했다. 최악의 재정난에 처한 이들은 놀랍게도 자녀가 있는 맞벌이 가정이었다.<sup class="reference" id="cite_ref-11"><a href="#cite_note-11">[11]</a></sup> 연구 결과, 맞벌이 부부 중 한명이 결혼, 임신, 출산 등으로 실직하면 가계가 급속도로 위축돼 가계 <a href="/wiki/%ED%8C%8C%EC%82%B0" title="파산">파산</a> 및 <a class="new" href="/w/index.php?title=%EC%8B%A0%EC%9A%A9%EB%B6%88%EB%9F%89%EC%9E%90&amp;action=edit&amp;redlink=1" title="신용불량자 (없는 문서)">신용불량자</a>로 전락하는 경우가 많은 것으로 나타났다.<sup class="reference" id="cite_ref-12"><a href="#cite_note-12">[12]</a></sup></p>
<p>맞벌이를 하면 외벌이 보다 소득이 2배 가까이 증가해, 그만큼 많은 소비를 여유있게 하게 된다. 그러다가 갑자기 한명이 실직을 하면, 바로 그 즉시 소비량이 줄어야 하는데도 불구하고, 한번 늘어난 씀씀이가 줄어들기는 매우 힘들고, 따라서 수많은 가정이 파산을 한다는 것이다.</p>
