<p><b>엘리자베스 테일러</b> 여사(<span style="font-size: smaller;"><a href="/wiki/%EC%98%81%EC%96%B4" title="영어">영어</a>:</span> <span lang="en" xml:lang="en">Dame Elizabeth Taylor</span>, <a href="/wiki/%EB%8C%80%EC%98%81_%EC%A0%9C%EA%B5%AD_%ED%9B%88%EC%9E%A5" title="대영 제국 훈장">DBE</a>, <a href="/wiki/1932%EB%85%84" title="1932년">1932년</a> <a href="/wiki/2%EC%9B%94_27%EC%9D%BC" title="2월 27일">2월 27일</a>~<a href="/wiki/2011%EB%85%84" title="2011년">2011년</a> <a href="/wiki/3%EC%9B%94_23%EC%9D%BC" title="3월 23일">3월 23일</a>)는 <a href="/wiki/%EC%98%81%EA%B5%AD" title="영국">영국</a>과 <a href="/wiki/%EB%AF%B8%EA%B5%AD" title="미국">미국</a>의 국적을 가진 <a href="/wiki/%EB%B0%B0%EC%9A%B0" title="배우">배우</a>이다. <b>리즈 테일러</b>(Liz Taylor)라는 애칭으로도 불린다. 아역 배우로 시작하여 성인기까지, 그녀는 연기력과 글래머, 아름다움, 그리고 보라빛 눈동자로 대중을 사로잡았다. 여덟 번에 걸친 결혼과 목숨을 잃을 뻔한 병마와 수십년동안 활동해 온 사회사업가로서, AIDS 홍보와 연구, 치료의 전문가로서 수 많은 이들의 주목을 받았다. 테일러는 <a href="/wiki/%EC%95%84%EC%B9%B4%EB%8D%B0%EB%AF%B8_%EC%97%AC%EC%9A%B0%EC%A3%BC%EC%97%B0%EC%83%81" title="아카데미 여우주연상">아카데미 여우주연상</a>을 2회 수상했으며, 할리우드 황금기의 가장 위대한 영화스타로 손꼽히고 있다. 아메리카 필름 연구소는 여성 레전드 목록 일곱 번째에 그녀를 이름을 올렸다.</p>
<p><a href="/wiki/1999%EB%85%84" title="1999년">1999년</a> <a href="/wiki/%EB%8C%80%EC%98%81_%EC%A0%9C%EA%B5%AD_%ED%9B%88%EC%9E%A5" title="대영 제국 훈장">대영 제국 훈장 2등급</a>(DBE, 작위급 훈장)을 받았고, <a href="/wiki/2011%EB%85%84" title="2011년">2011년</a> <a href="/wiki/3%EC%9B%94_23%EC%9D%BC" title="3월 23일">3월 23일</a> 네 명의 자녀들이 지켜보는 가운데서 80세에 세상을 떠났다.<sup class="reference" id="cite_ref-1"><a href="#cite_note-1">[1]</a></sup></p>

<h2><span class="mw-headline" id=".EC.83.9D.EC.95.A0">생애</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%97%98%EB%A6%AC%EC%9E%90%EB%B2%A0%EC%8A%A4_%ED%85%8C%EC%9D%BC%EB%9F%AC&amp;action=edit&amp;section=1" title="부분 편집: 생애">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<h3><span class="mw-headline" id=".EC.96.B4.EB.A6.B0.EC.8B.9C.EC.A0.88">어린시절</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%97%98%EB%A6%AC%EC%9E%90%EB%B2%A0%EC%8A%A4_%ED%85%8C%EC%9D%BC%EB%9F%AC&amp;action=edit&amp;section=2" title="부분 편집: 어린시절">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>엘리자베스 로즈먼드 테일러는 영국에 거주했던 미국인 부모 프랜시스 렌 테일러 (1897–1968)와 사라 비올라 웜브롯 (1895–1994) 사이에서 1남 1녀 중 막내로 <a href="/wiki/%EB%9F%B0%EB%8D%98" title="런던">런던</a>의 <a href="/wiki/%ED%96%84%ED%94%84%EC%8A%A4%ED%8B%B0%EB%93%9C" title="햄프스티드">햄프스티드</a>에서 태어났고, 당시 집은 런던 북서쪽 교외에 있는 햄프스티드 가든 서버브 지역의 8 와일드우드로였다.<sup class="reference" id="cite_ref-2"><a href="#cite_note-2">[2]</a></sup><sup class="reference" id="cite_ref-3"><a href="#cite_note-3">[3]</a></sup><sup class="reference" id="cite_ref-4"><a href="#cite_note-4">[4]</a></sup> 테일러의 오빠인 하워드 테일러는 1929년 태어났다.<sup class="reference" id="cite_ref-taraborrelli2006_5-0"><a href="#cite_note-taraborrelli2006-5">[5]</a></sup> 미국인이었지만 영국의 <a href="/wiki/%EC%86%8D%EC%A7%80%EC%A3%BC%EC%9D%98" title="속지주의">속지주의</a> 국적법에 의해 영국에서 태어난 그녀에게는 자동으로 영국 국적이 함께 부여되었다. 그녀의 부모는 원래 <a href="/wiki/%EC%BA%94%EC%9E%90%EC%8A%A4_%EC%A3%BC" title="캔자스 주">캔자스 주</a> 출신이었는데, 아버지는 미술상이었고, 어머니는 원래 "사라 소던"이라는 무대명을 가진 연극 배우로 <a href="/wiki/1926%EB%85%84" title="1926년">1926년</a> <a href="/wiki/%EB%89%B4%EC%9A%95" title="뉴욕">뉴욕</a>에서 결혼을 하면서 은퇴했다. 이름에서 첫 두 이름은 할머니인 엘리자베스 매리 로즈먼드 테일러를 기념하기 위해 지은 것이다.</p>
<p>3세 때, 테일러는 바카니와 함께 발레 레슨을 받았다. 7세 때 그녀의 가족들은 <a href="/wiki/%EC%A0%9C2%EC%B0%A8_%EC%84%B8%EA%B3%84_%EB%8C%80%EC%A0%84" title="제2차 세계 대전">제2차 세계 대전</a>을 피하기 위해 미국으로 귀국하였다. <a href="/wiki/1939%EB%85%84" title="1939년">1939년</a> 그녀의 엄마가 아이들을 데리고 먼저 뉴욕으로 갔고, 그녀의 아버지는 런던에 남아서 사업을 마무리하고 11월 뉴욕에 도착했다.<sup class="reference" id="cite_ref-6"><a href="#cite_note-6">[6]</a></sup> 그들은 <a href="/wiki/%EC%BA%98%EB%A6%AC%ED%8F%AC%EB%8B%88%EC%95%84_%EC%A3%BC" title="캘리포니아 주">캘리포니아 주</a> <a href="/wiki/%EB%A1%9C%EC%8A%A4%EC%95%A4%EC%A0%A4%EB%A0%88%EC%8A%A4" title="로스앤젤레스">로스앤젤레스</a>에서 정착을 했는데, 그 곳은 당시 사라의 가족들이 있는 곳이었다.</p>
<p>칼럼니스트 헤더 호퍼의 시선을 사로잡았던 것이 인연이 되어, 부유한 영국 사업가이자 치버 코든의 약혼자였던, <a class="mw-redirect" href="/wiki/%EC%9C%A0%EB%8B%88%EB%B2%84%EC%84%A4_%ED%94%BD%EC%B2%98%EC%8A%A4" title="유니버설 픽처스">유니버설 픽처스</a>의 회장이자 대주주인 안드레아 베런스를 소개 받았다. 베런스는 사라에게 엘리자베스를 코든에게 보여야 한다고 주장했다. 코든은 엘레자베스의 아름다움에 매료되었고, 그녀는 카메라를 잘받아 외모를 돋보이게 하는 그런 눈동자를 가지고 있었다.<sup class="reference" id="cite_ref-7"><a href="#cite_note-7">[7]</a></sup> <a class="mw-redirect" href="/wiki/%EB%A9%94%ED%8A%B8%EB%A1%9C-%EA%B3%A8%EB%93%9C%EC%9C%88-%EB%A9%94%EC%9D%B4%EC%96%B4" title="메트로-골드윈-메이어">메트로-골드윈-메이어</a>(줄여서 MGM)는 곧 이 영국소녀에게 관심을 나타냈지만, 제작자인 존 콘시다인과 비공식적인 오디션을 본 후에 테일러가 노래를 잘 하지 못했다는 이유로 계약을 하는데는 실패했다. 그러나 <a href="/wiki/1941%EB%85%84" title="1941년">1941년</a> <a href="/wiki/9%EC%9B%94_18%EC%9D%BC" title="9월 18일">9월 18일</a> 6개월 계약으로 주급 $100에 계약을 했다.</p>
<p>그녀는 9살의 나이로 《<i>There's One Born Every Minute</i>》에 데뷔를 했고, 이것이 그녀의 유일한 유니버설 픽처스의 작품이었다.</p>
<h3><span class="mw-headline" id=".EC.95.84.EC.97.AD_.EB.B0.B0.EC.9A.B0">아역 배우</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%97%98%EB%A6%AC%EC%9E%90%EB%B2%A0%EC%8A%A4_%ED%85%8C%EC%9D%BC%EB%9F%AC&amp;action=edit&amp;section=3" title="부분 편집: 아역 배우">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<div class="thumb tright">
<div class="thumbinner" style="width:182px;"><a class="image" href="/wiki/%ED%8C%8C%EC%9D%BC:Taylor,_Elizabeth_10.jpg"><img alt="" class="thumbimage" data-file-height="1600" data-file-width="1271" height="227" src="//upload.wikimedia.org/wikipedia/commons/thumb/b/b6/Taylor%2C_Elizabeth_10.jpg/180px-Taylor%2C_Elizabeth_10.jpg" srcset="//upload.wikimedia.org/wikipedia/commons/thumb/b/b6/Taylor%2C_Elizabeth_10.jpg/270px-Taylor%2C_Elizabeth_10.jpg 1.5x, //upload.wikimedia.org/wikipedia/commons/thumb/b/b6/Taylor%2C_Elizabeth_10.jpg/360px-Taylor%2C_Elizabeth_10.jpg 2x" width="180"/></a>
<div class="thumbcaption">
<div class="magnify"><a class="internal" href="/wiki/%ED%8C%8C%EC%9D%BC:Taylor,_Elizabeth_10.jpg" title="실제 크기로"></a></div>
1942년경 아역 배우 시절의 테일러</div>

<p>10살 때인 <a href="/wiki/1942%EB%85%84" title="1942년">1942년</a> 《<i>There's One Born Every Minut</i>e》의 단역으로 영화 데뷔했다. 유니버설 픽처스와 계약을 통해 그녀는 《명견 랫시의 귀로》(<i>Lassie Come Home</i>)의 영화화를 위한 배역을 구하던 MGM과 7년 계약을 맺는다. MGM과의 첫 작품은 <a href="/wiki/%EC%83%AC%EB%9F%BF_%EB%B8%8C%EB%A1%A0%ED%85%8C" title="샬럿 브론테">샬럿 브론테</a>의 소설을 영화화한 <a href="/wiki/20%EC%84%B8%EA%B8%B0_%ED%8F%AD%EC%8A%A4" title="20세기 폭스">20세기 폭스</a>의 1944년작 《<a href="/wiki/%EC%A0%9C%EC%9D%B8_%EC%97%90%EC%96%B4" title="제인 에어">제인 에어</a>》로, 그녀는 헬렌 번즈 역을 맡았다. 이후 그녀는 다시 MGM사의 영화를 위해 영국으로 돌아갔으며 같은 해에 <a href="/wiki/1944%EB%85%84" title="1944년">1944년</a> 《도버의 하얀 절벽》(<i>The White Cliffs of Dover</i>)을 찍었다. 그녀를 스타덤에 올려준 것은 12살 때 찍은 MGM의 영화 《녹원의 천사》(<i>National Velvet</i>)에서 맡은 벨벳 브라운 역으로, 그녀가 맡은 벨벳 브라운은 그랜드 내셔널에서 우승하기 위해 애마를 길들이는 소녀였다. 또한 이 영화는 <a class="mw-redirect" href="/wiki/%EB%AF%B9%ED%82%A4_%EB%A3%A8%EB%8B%88" title="믹키 루니">믹키 루니</a>와 영국의 신인 <a class="mw-redirect" href="/wiki/%EC%95%88%EC%A0%A4%EB%9D%BC_%EB%9E%9C%EC%8A%A4%EB%B2%A0%EB%A6%AC" title="안젤라 랜스베리">안젤라 랜스베리</a>를 스타로 만들어주었으며, <a href="/wiki/1944%EB%85%84" title="1944년">1944년</a> 12월 개봉이후 엄청난 성공을 거두었고, 테일러의 삶을 영원히 바꾸어 놓았다. 그녀의 일생에서 발생된 많은 건강 문제들은 이때 《녹원의 천사》를 촬영하던 중 입은 척추 부상으로 야기된 것이다.</p>
<p>《녹원의 천사》은 박스오피스에서 4백만 달러를 벌어들였으며, 그녀는 3만 달러로 인상된 연봉으로 장기계약에 서명했다. 이 영화로 얻은 성공을 이어가기 위해 테일러는 또 다른 동물 영화 《용감한 래시》(<i>Courage of Lassie</i>)에 참여하였으며, 1946년 《용감한 래시》의 성공으로 더 인상된 $1,500의 보너스와 주급 $750의 조건으로 계약을 체결했다. <a href="/wiki/1947%EB%85%84" title="1947년">1947년</a> <a class="mw-redirect" href="/wiki/%EC%9B%8C%EB%84%88%EB%B8%8C%EB%9D%BC%EB%8D%94%EC%8A%A4" title="워너브라더스">워너브라더스</a>의 작품인 《아버지와 인생을》(<i>Life With Father</i>)에서는 매리 스키너의 역을 맡았으며, 같은 해 《신시아》(<i>Cynthia</i>)에서는 신시아 비숍 역을 맡아서 연기를 했다. 이듬 해 <a href="/wiki/1948%EB%85%84" title="1948년">1948년</a>에는 《쥬디와의 데이트》(<i>A Date with Judy</i>)에서는 캐롤 프링글 역을 맡아서 연기를 했고, 수전 프랙킷 역을 맡아서 한 《줄리아 미스비헤이비어》( <i>Julia Misbehaves</i>)에서도 성공을 거두었다. 테일러는 꾸준한 흥행제조기라는 명성을 얻음과 동시에 "원샷 리즈"(One-Shot Liz, 한번에 장면을 찍어내는 그녀의 능력을 가리키는 말)라는 별명을 얻어 전도 유망한 경력을 쌓았다. 아미의 역할을 한 《<a class="mw-redirect" href="/wiki/%EC%9E%91%EC%9D%80%EC%95%84%EC%94%A8%EB%93%A4" title="작은아씨들">작은아씨들</a>》(<i>Little Women</i>, 1949)은 그녀의 아역으로는 마지막 배역이었다. <a href="/wiki/1948%EB%85%84" title="1948년">1948년</a> 10월 퀸매리호를 타고 영국으로 가서 《공모자》(<i>Conspirator</i>)를 찍었다. 이 영화에서 그녀는 최초로 성인 역을 맡았다.</p>
<h3><span class="mw-headline" id=".EC.84.B1.EC.9D.B8_.EC.97.B0.EA.B8.B0.EC.9E.90">성인 연기자</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%97%98%EB%A6%AC%EC%9E%90%EB%B2%A0%EC%8A%A4_%ED%85%8C%EC%9D%BC%EB%9F%AC&amp;action=edit&amp;section=4" title="부분 편집: 성인 연기자">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<div class="thumb tleft">

<p>다른 아역 배우들과는 달리, 테일러는 쉽게 성인 연기자로 옮겨갔다.<sup class="reference" id="cite_ref-death_8-0"><a href="#cite_note-death-8">[8]</a></sup> <a href="/wiki/1949%EB%85%84" title="1949년">1949년</a> 《공모자》가 나오기 전에 《<a href="/wiki/%ED%83%80%EC%9E%84_(%EC%9E%A1%EC%A7%80)" title="타임 (잡지)">타임</a>》은 기사를 통해 그녀를 ‘엄청난 가치를 지닌 보석, 진정한 스타 <a href="/wiki/%EC%82%AC%ED%8C%8C%EC%9D%B4%EC%96%B4" title="사파이어">사파이어</a>’라고 찬사를 보냈으며, <a class="mw-redirect" href="/wiki/%ED%97%90%EB%A6%AC%EC%9A%B0%EB%93%9C" title="헐리우드">헐리우드</a>에서는 <a class="new" href="/w/index.php?title=%EB%AA%BD%EA%B3%A0%EB%A9%94%EB%A6%AC_%ED%81%B4%EB%A6%AC%ED%94%84%ED%8A%B8&amp;action=edit&amp;redlink=1" title="몽고메리 클리프트 (없는 문서)">몽고메리 클리프트</a>나 <a class="mw-redirect" href="/wiki/%EC%BB%A4%ED%81%AC_%EB%8D%94%EA%B8%80%EB%9D%BC%EC%8A%A4" title="커크 더글라스">커크 더글라스</a> 그리고 <a href="/wiki/%EC%97%90%EB%B0%94_%EA%B0%80%EB%93%9C%EB%84%88" title="에바 가드너">에바 가드너</a>와 같은 차세대 스타로 손꼽았다.<sup class="reference" id="cite_ref-9"><a href="#cite_note-9">[9]</a></sup> 그 영화는 박스오피스에서는 실패했지만, 남 몰래 공산주의 스파이와 결혼한 21살짜리 연기를 한 16살의 테일러는 비평가들 사이에서 성숙한 연기로 칭찬을 받았다. 주당 2,000 달러의 새로운 급여조건 하에서 최초로 찍은 《만취》(1950)는 은막의 아이돌 <a class="new" href="/w/index.php?title=%EB%B0%B4_%EC%A1%B4%EC%8A%A8&amp;action=edit&amp;redlink=1" title="밴 존슨 (없는 문서)">밴 존슨</a>과 짝을 맞춘 영화이지만, 비평가들과 박스오피스에서 모두 실패를 했다. 또한 그 영화는 새롭게 깨달은 그녀의 관능미를 과시하는데도 도움이 되지 못했다.</p>
<p>박스오피스에서 성공한 최초의 작품은 로맨틱 코미디로 <a href="/wiki/%EC%8A%A4%ED%8E%9C%EC%84%9C_%ED%8A%B8%EB%A0%88%EC%9D%B4%EC%8B%9C" title="스펜서 트레이시">스펜서 트레이시</a>와 <a class="mw-redirect" href="/wiki/%EC%A1%B0%EC%95%88_%EB%B2%A0%EB%84%B7" title="조안 베넷">조안 베넷</a>과 호흡을 맞춰 케이 뱅크스역을 맡아 성인 연기를 한 1950년작 《<a class="new" href="/w/index.php?title=%EC%8B%A0%EB%B6%80%EC%9D%98_%EC%95%84%EB%B2%84%EC%A7%80&amp;action=edit&amp;redlink=1" title="신부의 아버지 (없는 문서)">신부의 아버지</a>》이었다. 이 영화는 후속 편 《아버지의 작은 유산》(1951)을 내놓게 했으며, 같이 연기를 했던 스펜서 트래이스가 “지루하고, 지루했고 .... 지루했다”고 평한 작품이었다. 박스오피스에서는 비교적 성공을 거두었지만, 드라마틱한 배우로서의 경력을 쌓게 한 것은 테일러의 다음 영화였다. <a href="/wiki/1949%EB%85%84" title="1949년">1949년</a>말 테일러는 조지 스트븐스의 영화 《<a class="mw-disambig" href="/wiki/%EC%A0%8A%EC%9D%80%EC%9D%B4%EC%9D%98_%EC%96%91%EC%A7%80" title="젊은이의 양지">젊은이의 양지</a>》를 찍기 시작했다. 이 영화가 <a href="/wiki/1951%EB%85%84" title="1951년">1951년</a> 개봉되자, 테일러는 조지 이스트먼(클리프트)와 그의 가난하고 임신한 여공 앨리스 트립(셸리 윈터스) 사이에 끼어든 망가진 사교계 명사 안젤라 비커스의 연기로 환호를 받았다. 이 영화는 비평가들이 다음 50년 영화 역사를 통틀어 평판을 얻은 고전이라는 평과 같이 그녀의 연기 경력에 있어서 전환점이 되었다. <a class="mw-redirect" href="/wiki/%EB%89%B4%EC%9A%95%ED%83%80%EC%9E%84%EC%A6%88" title="뉴욕타임즈">뉴욕타임즈</a>의 A.H. 웨일러는 "엘리자베스의 풍부하고, 아름다운 연기는 그녀의 경력에서 최고의 노력이다."라고 썼으며, 박스오피스 평론가들은 "테일러양은 <a href="/wiki/%EC%95%84%EC%B9%B4%EB%8D%B0%EB%AF%B8%EC%83%81" title="아카데미상">아카데미상</a>감이다."라고 언급했다.</p>

<p>테일러는 그 당시 자기에게 배정된 역할에 점점 불만을 가지기 시작했다. 《맨발의 공작부인》이나 《크라이 투마로우》와 같은 작품에서 주연을 맡기를 원했지만, MGM은 그녀의 역할을 <a class="mw-disambig" href="/wiki/%EC%B9%B4%EB%A9%94%EC%98%A4" title="카메오">카메오</a>나 기억에 남지 않은 영화들, 《<i>Callaway Went Thataway</i>》(1951), 《최고의 사랑》(1952), 《<a href="/wiki/%EC%95%84%EC%9D%B4%EB%B0%98%ED%98%B8" title="아이반호">아이반호</a>》(1952), 《모든 걸 다 가진 소녀》 (1953), 《호걸 브롬멜》 등에 배역으로 제한했다. 그녀는 아이반호에서는 로웨나 아가씨 역을 맡기를 원했지만, 그 역은 <a class="new" href="/w/index.php?title=%EC%A1%B0%EC%95%88_%ED%8F%B0%ED%8B%B4&amp;action=edit&amp;redlink=1" title="조안 폰틴 (없는 문서)">조안 폰틴</a>에게 돌아갔다. 테일러에게 주어진 역은 레베카 역이었다. 테일러가 첫 아이를 임신했을 때, MGM은 업무 시간을 두 시간 더 늘여 《모든 것을 다 가진 소녀》를 찍도록 그녀를 다그쳤다. 그녀의 임신이 더 진행되기 전에 새로운 영화를 하나 더 찍을 속셈이었다. 테일러는 돈이 필요하다는 사실이 슬펐다. 그때 그녀는 새집을 막 샀고, 두 번째 남편인 마이클 와일딩과 그녀의 아기 때문에 일은 꽤 빡빡했다. 테일러에게 역이 배당되기는 했지만, 그녀의 임신으로 인해 《코끼리 걸음마》(1954년)는 거절당해야 했다. 테일러보다 20살이 더 많지만 닮은 점이 많은 <a href="/wiki/%EB%B9%84%EB%B9%84%EC%96%B8_%EB%A6%AC" title="비비언 리">비비언 리</a>는 배역을 가졌고, 로케이션을 찍기 위해 <a class="mw-redirect" href="/wiki/%EC%8B%A4%EB%A1%A0" title="실론">실론</a>으로 갔다. 영화를 찍는 동안 비비언 리는 신경쇄약으로 고생했고, <a href="/wiki/1953%EB%85%84" title="1953년">1953년</a> 1월 첫 아이 마이클 와일딩 주니어를 출산한 후 테일러가 그 역을 대신 맡게 되었다.<sup class="reference" id="cite_ref-10"><a href="#cite_note-10">[10]</a></sup></p>
<p>테일러의 다음 영화 《랩소디》(1954)는 지루한 사랑 이야기로 좌절을 맛보게 된다. 테일러는 기질이 강한 바이올리니스트(비토리오 개스맨)와 진지한 젊은 피아니스트(존 에릭슨)과 사랑에 빠진 아름답고, 부유한 소녀 루이스 루란트 역을 했다.</p>
<p>《코끼리 걸음마》와 《랩소디》 다음에 찍은 테일러의 4기 영화인 《호걸 브롬멜》에서는 세련된 패트리샤 부인으로 캐스팅되어, 스튜어트 그랜저와 연기를 펼쳤다. 《내가 본 마지막 파리》 (1954)는 이전 작보다는 약간 나아졌으며, 《숙취》를 같이 찍은 밴 존슨과 같이 연기를 펼쳐다. 헬렌 엘스워스 윌리의 배역은 <a class="new" href="/w/index.php?title=%EC%A0%A4%EB%8B%A4_%ED%94%BC%EC%B8%A0%EC%A0%9C%EB%9F%B4%EB%93%9C&amp;action=edit&amp;redlink=1" title="젤다 피츠제럴드 (없는 문서)">젤다 피츠제럴드</a>의 것에 기반을 둔 것이며, 비록 둘째 아이를 임신했지만, 테일러는 12개월만에 찍는 네 번째 영화에 정성을 쏟았다. 박스오피스에서 약간의 성공을 거두긴 했지만, 그녀는 여전히 더 나은 배역을 원했다.</p>
<h3><span class="mw-headline" id="1955.E2.80.931979">1955–1979</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%97%98%EB%A6%AC%EC%9E%90%EB%B2%A0%EC%8A%A4_%ED%85%8C%EC%9D%BC%EB%9F%AC&amp;action=edit&amp;section=5" title="부분 편집: 1955–1979">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p><a class="new" href="/w/index.php?title=%EC%A1%B0%EC%A7%80_%EC%8A%A4%ED%8B%B0%EB%B8%90%EC%8A%A4&amp;action=edit&amp;redlink=1" title="조지 스티븐스 (없는 문서)">조지 스티븐스</a> 감독의 <a href="/wiki/1956%EB%85%84" title="1956년">1956년</a> 서사영화 《자이언트》에서 <a href="/wiki/%EB%A1%9D_%ED%97%88%EB%93%9C%EC%8A%A8" title="록 허드슨">록 허드슨</a>이나 <a href="/wiki/%EC%A0%9C%EC%9E%84%EC%8A%A4_%EB%94%98" title="제임스 딘">제임스 딘</a>의 상대역으로 보다 중요한 역할을 맡게 된 이후 테일러는 4년 연속 아카데미상 여우주연상 후보에 노미네이트된다. <a href="/wiki/1957%EB%85%84" title="1957년">1957년</a> 《레인트리 카운티》에서는 <a class="new" href="/w/index.php?title=%EB%AA%BD%EA%B3%A0%EB%A9%94%EB%A6%AC_%ED%81%B4%EB%A6%AC%ED%94%84%ED%8A%B8&amp;action=edit&amp;redlink=1" title="몽고메리 클리프트 (없는 문서)">몽고메리 클리프트</a>의 상대역을 맡았고, <a class="new" href="/w/index.php?title=1958&amp;action=edit&amp;redlink=1" title="1958 (없는 문서)">1958</a>년 《양철 지붕위의 고양이》에서는 <a href="/wiki/%ED%8F%B4_%EB%89%B4%EB%A8%BC" title="폴 뉴먼">폴 뉴먼</a>의 상대역으로, <a href="/wiki/1959%EB%85%84" title="1959년">1959년</a> 《지난 여름 갑자기》에서는 몽고메리 클리프트와 <a class="new" href="/w/index.php?title=%EC%BA%90%EC%84%9C%EB%A6%B0_%ED%96%85%EB%B2%88&amp;action=edit&amp;redlink=1" title="캐서린 햅번 (없는 문서)">캐서린 햅번</a> 그리고 <a href="/wiki/%EB%A8%B8%EC%8B%9C%EB%94%94%EC%8A%A4_%EB%A7%A4%EC%BC%80%EC%9E%84%EB%B8%8C%EB%A6%AC%EC%A7%80" title="머시디스 매케임브리지">머시디스 매케임브리지</a>와 호흡을 맞춰 연기를 했고, <a href="/wiki/1960%EB%85%84" title="1960년">1960년</a> 《버터필드8》에서는 당시 남편이었던 <a href="/wiki/%EC%97%90%EB%94%94_%ED%94%BC%EC%85%94" title="에디 피셔">에디 피셔</a>와 함께 주연을 맡았다.</p>
<p>1996년 테일러는 당시 남편 <a href="/wiki/%EB%A6%AC%EC%B2%98%EB%93%9C_%EB%B2%84%ED%84%B4" title="리처드 버턴">리처드 버턴</a>과 함께 출연한 《누가 버지니아 울프를 두려워하랴?》에서 마사 역으로 <a href="/wiki/%EC%95%84%EC%B9%B4%EB%8D%B0%EB%AF%B8%EC%83%81" title="아카데미상">아카데미상</a>에서 여우주연상을 수상하며 생애 두 번째 아카데미상 수상을 했다.<sup class="reference" id="cite_ref-Parish344_11-0"><a href="#cite_note-Parish344-11">[11]</a></sup></p>

<h3><span class="mw-headline" id=".EA.B2.B0.ED.98.BC">결혼</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%97%98%EB%A6%AC%EC%9E%90%EB%B2%A0%EC%8A%A4_%ED%85%8C%EC%9D%BC%EB%9F%AC&amp;action=edit&amp;section=7" title="부분 편집: 결혼">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>테일러는 8차례 7명의 남편과 결혼을 했다.</p>
<ul>

<p><a href="/wiki/1971%EB%85%84" title="1971년">1971년</a> 테일러는 40세의 젊은 나이로 할머니가 되었다. 그녀의 임종 시 4자녀들과 10명의 손자 그리고 4명의 증손자가 곁을 지켰다.<sup class="reference" id="cite_ref-death_8-1"><a href="#cite_note-death-8">[8]</a></sup></p>
<h2><span class="mw-headline" id=".EC.82.AC.EB.A7.9D">사망</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%97%98%EB%A6%AC%EC%9E%90%EB%B2%A0%EC%8A%A4_%ED%85%8C%EC%9D%BC%EB%9F%AC&amp;action=edit&amp;section=9" title="부분 편집: 사망">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>테일러는 말년에 여러 건강 문제를 가지고 있었다.<sup class="reference" id="cite_ref-12"><a href="#cite_note-12">[12]</a></sup> <a href="/wiki/2004%EB%85%84" title="2004년">2004년</a>에는 만성 <a href="/wiki/%EC%8B%AC%EB%B6%80%EC%A0%84" title="심부전">심부전</a>을 앓고 있다는 보고가 있었고, <a href="/wiki/2009%EB%85%84" title="2009년">2009년</a>에는 누수관을 교체하는 심장수술을 했다. 2011년 2월에는 만성 심부전에 대한 새로운 증상으로 치료를 위해 세다스-시나이 메디컬 센터에 입원을 했다.</p>
<p><a href="/wiki/2011%EB%85%84" title="2011년">2011년</a> <a href="/wiki/3%EC%9B%94_23%EC%9D%BC" title="3월 23일">3월 23일</a> <a href="/wiki/%EB%A1%9C%EC%8A%A4%EC%95%A4%EC%A0%A4%EB%A0%88%EC%8A%A4" title="로스앤젤레스">로스앤젤레스</a>에 있는 세다스-시나이 메디컬 센터에서 4명의 자녀들이 지켜보는 가운데 향년 80세를 일기로 세상을 떠났다.<sup class="reference" id="cite_ref-death_8-2"><a href="#cite_note-death-8">[8]</a></sup></p>

<h2><span class="mw-headline" id=".EC.88.98.EC.83.81_.EA.B2.BD.EB.A0.A5">수상 경력</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%97%98%EB%A6%AC%EC%9E%90%EB%B2%A0%EC%8A%A4_%ED%85%8C%EC%9D%BC%EB%9F%AC&amp;action=edit&amp;section=11" title="부분 편집: 수상 경력">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>엘리자베스 테일러는 <a href="/wiki/1960%EB%85%84" title="1960년">1960년</a>의 《버터필드8》과 <a href="/wiki/1966%EB%85%84" title="1966년">1966년</a>작 《누가 버지니아 울프를 두려워하랴?》에서 아카데미 여우주연상을 2번 수상하여 루이스 레이너, 비트 데이비스, 올리비아 드 하빌랜드, 비비언 레인, 잉글리드 버그만, 글렌다 잭슨, 재인 폰다, 샐리 필드, 조디 포스터, 그리고 힐러리 스왠크 등과 함께 2회 수상자 반열에 올랐다. 1992년에는 AIDS와 싸우는 작품으로 <a href="/wiki/%EC%A7%84_%ED%97%88%EC%88%84%ED%8A%B8_%EB%B0%95%EC%95%A0%EC%83%81" title="진 허숄트 박애상">진 허숄트 박애상</a>을 수상하기도 했다.</p>
