<p><b>엘빈 브루노 크리스토펠</b>(<span style="font-size: smaller;"><a href="/wiki/%EB%8F%85%EC%9D%BC%EC%96%B4" title="독일어">독일어</a>:</span> <span lang="de" xml:lang="de">Elwin Bruno Christoffel</span>, <small><a href="/wiki/%EC%9C%84%ED%82%A4%EB%B0%B1%EA%B3%BC:IPA" title="위키백과:IPA">IPA</a>: </small><span class="IPA" title="IPA 발음">[ˈɛlvɪn ˈbruːnɔ krɪˈstɔfəl]</span>, 1829~1900)은 <a href="/wiki/%EB%8F%85%EC%9D%BC" title="독일">독일</a>의 수학자이자 물리학자이다. <a href="/wiki/%EB%AF%B8%EB%B6%84%EA%B8%B0%ED%95%98%ED%95%99" title="미분기하학">미분기하학</a>의 기초를 마련하였다.</p>
<h2><span class="mw-headline" id=".EC.83.9D.EC.95.A0">생애</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%97%98%EB%B9%88_%EB%B8%8C%EB%A3%A8%EB%85%B8_%ED%81%AC%EB%A6%AC%EC%8A%A4%ED%86%A0%ED%8E%A0&amp;action=edit&amp;section=1" title="부분 편집: 생애">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>1829년 11월 10일 <a href="/wiki/%EB%85%B8%EB%A5%B4%ED%8A%B8%EB%9D%BC%EC%9D%B8%EB%B2%A0%EC%8A%A4%ED%8A%B8%ED%8C%94%EB%A0%8C_%EC%A3%BC" title="노르트라인베스트팔렌 주">노르트라인베스트팔렌 주</a> <a class="new" href="/w/index.php?title=%EB%AA%AC%EC%83%A4%EC%9A%B0&amp;action=edit&amp;redlink=1" title="몬샤우 (없는 문서)">몬샤우</a>(<span style="font-size: smaller;"><a href="/wiki/%EB%8F%85%EC%9D%BC%EC%96%B4" title="독일어">독일어</a>:</span> <span lang="de" xml:lang="de">Monschau</span>)에서 태어났다. 크리스토펠의 아버지는 옷감 상인이었다.</p>
<p><a href="/wiki/%EC%BE%B0%EB%A5%B8" title="쾰른">쾰른</a>에서 <a href="/wiki/%EA%B9%80%EB%82%98%EC%A7%80%EC%9B%80" title="김나지움">김나지움</a>을 졸업하였고, 1850년에 <a href="/wiki/%EB%B2%A0%EB%A5%BC%EB%A6%B0" title="베를린">베를린</a>으로 상경하여 <a href="/wiki/%EB%B2%A0%EB%A5%BC%EB%A6%B0_%ED%9B%94%EB%B3%BC%ED%8A%B8_%EB%8C%80%ED%95%99%EA%B5%90" title="베를린 훔볼트 대학교">베를린 훔볼트 대학교</a>에서 <a href="/wiki/%ED%8E%98%ED%84%B0_%EA%B5%AC%EC%8A%A4%ED%83%80%ED%94%84_%EB%A5%B4%EC%A3%88_%EB%94%94%EB%A6%AC%ED%81%B4%EB%A0%88" title="페터 구스타프 르죈 디리클레">페터 구스타프 르죈 디리클레</a> 밑에서 공부하였다. 1856년에 전기에 대한 논문으로 박사 학위를 수여받았다.</p>
<p>이후 다시 몬샤우로 귀향하여 3년을 보냈다. 1859년에 베를린으로 돌아와 <a href="/wiki/%ED%95%98%EB%B9%8C%EB%A6%AC%ED%83%80%EC%B9%98%EC%98%A8" title="하빌리타치온">하빌리타치온</a>을 마치고, <a href="/wiki/%EB%B2%A0%EB%A5%BC%EB%A6%B0_%ED%9B%94%EB%B3%BC%ED%8A%B8_%EB%8C%80%ED%95%99%EA%B5%90" title="베를린 훔볼트 대학교">베를린 훔볼트 대학교</a> 연구원(<span style="font-size: smaller;"><a href="/wiki/%EB%8F%85%EC%9D%BC%EC%96%B4" title="독일어">독일어</a>:</span> <span lang="de" xml:lang="de">Privatdozent</span>)이 되었다. 1862년에 <a href="/wiki/%EC%B7%A8%EB%A6%AC%ED%9E%88_%EC%97%B0%EB%B0%A9_%EA%B3%B5%EA%B3%BC%EB%8C%80%ED%95%99%EA%B5%90" title="취리히 연방 공과대학교">취리히 연방 공과대학교</a> 교수가 되었다. 1869년에 귀국하여 <a href="/wiki/%EB%B2%A0%EB%A5%BC%EB%A6%B0_%EA%B3%B5%EA%B3%BC%EB%8C%80%ED%95%99%EA%B5%90" title="베를린 공과대학교">베를린 공과대학교</a> 교수가 되었다.</p>
<p>1871년에 <a href="/wiki/%ED%94%84%EB%A1%9C%EC%9D%B4%EC%84%BC" title="프로이센">프로이센</a>은 <a href="/wiki/%ED%94%84%EB%A1%9C%EC%9D%B4%EC%84%BC-%ED%94%84%EB%9E%91%EC%8A%A4_%EC%A0%84%EC%9F%81" title="프로이센-프랑스 전쟁">프로이센-프랑스 전쟁</a>에서 승리하여, <a href="/wiki/%EC%8A%A4%ED%8A%B8%EB%9D%BC%EC%8A%A4%EB%B6%80%EB%A5%B4" title="스트라스부르">스트라스부르</a>를 포함한 <a href="/wiki/%EC%95%8C%EC%9E%90%EC%8A%A4-%EB%A1%9C%EB%A0%8C" title="알자스-로렌">알자스-로렌</a> 지방을 프랑스로부터 점령하였다. 프로이센의 지배 아래 <a href="/wiki/%EC%8A%A4%ED%8A%B8%EB%9D%BC%EC%8A%A4%EB%B6%80%EB%A5%B4_%EB%8C%80%ED%95%99%EA%B5%90" title="스트라스부르 대학교">스트라스부르 대학교</a>는 독일식 대학교로 새로 개교하였으며, 크리스토펠은 1872년에 <a href="/wiki/%EC%8A%A4%ED%8A%B8%EB%9D%BC%EC%8A%A4%EB%B6%80%EB%A5%B4_%EB%8C%80%ED%95%99%EA%B5%90" title="스트라스부르 대학교">스트라스부르 대학교</a> 교수가 되었다.</p>
<p>1894년에 스트라스부르 대학교에서 은퇴하였다. 크리스토펠은 평생 독신이었으며, 1900년 3월 15일에 사망하였다.</p>
