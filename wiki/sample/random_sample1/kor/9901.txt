<h1 class="firstHeading" id="firstHeading" lang="ko">여러 언어에서의 파랑과 초록의 구별</h1>
<div class="mw-body-content" id="bodyContent">
<div id="siteSub">위키백과, 우리 모두의 백과사전.</div>
<div id="contentSub"></div>
<div class="mw-jump" id="jump-to-nav">
					이동:					<a href="#mw-head">둘러보기</a>, 					<a href="#p-search">검색</a>
</div>
<div class="mw-content-ltr" dir="ltr" id="mw-content-text" lang="ko"><script>function mfTempOpenSection(id){var block=document.getElementById("mf-section-"+id);block.className+=" open-block";block.previousSibling.className+=" open-block";}</script><div class="thumb tright">
<div class="thumbinner" style="width:382px;"><a class="image" href="/wiki/%ED%8C%8C%EC%9D%BC:Linear_visible_spectrum.svg"><img alt="" class="thumbimage" data-file-height="115" data-file-width="605" height="72" src="//upload.wikimedia.org/wikipedia/commons/thumb/d/d9/Linear_visible_spectrum.svg/380px-Linear_visible_spectrum.svg.png" srcset="//upload.wikimedia.org/wikipedia/commons/thumb/d/d9/Linear_visible_spectrum.svg/570px-Linear_visible_spectrum.svg.png 1.5x, //upload.wikimedia.org/wikipedia/commons/thumb/d/d9/Linear_visible_spectrum.svg/760px-Linear_visible_spectrum.svg.png 2x" width="380"/></a>
<div class="thumbcaption">
<div class="magnify"><a class="internal" href="/wiki/%ED%8C%8C%EC%9D%BC:Linear_visible_spectrum.svg" title="실제 크기로"></a></div>
<a href="/wiki/%EA%B0%80%EC%8B%9C%EA%B4%91%EC%84%A0" title="가시광선">가시광선</a>의 <a href="/wiki/%EC%8A%A4%ED%8E%99%ED%8A%B8%EB%9F%BC" title="스펙트럼">스펙트럼</a>을 어떤 이름으로 부르는지는 문화에 따라 다르다. 현대 유럽어에서는 대략 520–570 nm 영역을 "녹색"(green)이라 하지만, 언어에 따라서는 450–530 nm 정도("파랑/초록")나 530–590 nm 정도("초록/노랑") 정도를 녹색이라 부르기도 한다.</div>
</div>
</div>
<p>여러 <a href="/wiki/%EC%96%B8%EC%96%B4" title="언어">언어</a> 가운데에는 <b><a href="/wiki/%ED%8C%8C%EB%9E%91" title="파랑">파랑</a>과 <a href="/wiki/%EC%B4%88%EB%A1%9D" title="초록">초록</a>을 구별</b>하는 언어도 있고, 구별하지 않는 언어도 있다. 또, 짙은 파랑과 옅은 파랑을 농도나 채도만 차이나는 같은 파란색이 아닌, 별개의 색으로 여기는 언어도 있다.</p>
<p><a class="new" href="/w/index.php?title=%EB%B8%8C%EB%A0%8C%ED%8A%B8_%EB%B2%8C%EB%A6%B0&amp;action=edit&amp;redlink=1" title="브렌트 벌린 (없는 문서)">브렌트 벌린</a>과 <a class="new" href="/w/index.php?title=%ED%8F%B4_%EC%BC%80%EC%9D%B4&amp;action=edit&amp;redlink=1" title="폴 케이 (없는 문서)">폴 케이</a>의 <a href="/wiki/1969%EB%85%84" title="1969년">1969년</a> 연구 〈Basic Color Terms: Their Universality and Evolution〉에 따르면, 한 언어에서 <a class="mw-redirect" href="/wiki/%ED%8C%8C%EB%9E%80%EC%83%89" title="파란색">파란색</a>과 <a class="mw-redirect" href="/wiki/%EC%B4%88%EB%A1%9D%EC%83%89" title="초록색">초록색</a>을 구별하기 전까지는 <a href="/wiki/%EA%B0%88%EC%83%89" title="갈색">갈색</a>, <a class="mw-redirect" href="/wiki/%EC%9E%90%EC%A3%BC%EC%83%89" title="자주색">자주색</a>, <a class="mw-redirect" href="/wiki/%EB%B6%84%ED%99%8D%EC%83%89" title="분홍색">분홍색</a>, <a class="mw-redirect" href="/wiki/%EC%A3%BC%ED%99%A9%EC%83%89" title="주황색">주황색</a>과 <a href="/wiki/%ED%9A%8C%EC%83%89" title="회색">회색</a>을 구별하는 낱말이 나타나지 않는다. 색을 나타내는 낱말이 발전되는 과정은 그 연구 결과에 따르면 <a class="mw-redirect" href="/wiki/%ED%95%98%EC%96%80%EC%83%89" title="하얀색">하얀색</a>/<a href="/wiki/%EA%B2%80%EC%A0%95" title="검정">검정색</a>(또는 밝음/어두움), <a class="mw-redirect" href="/wiki/%EB%B9%A8%EA%B0%84%EC%83%89" title="빨간색">빨간색</a>과 <a class="mw-redirect" href="/wiki/%EC%B4%88%EB%A1%9D%EC%83%89" title="초록색">초록색</a>/<a class="mw-redirect" href="/wiki/%EB%85%B8%EB%9E%91%EC%83%89" title="노랑색">노랑색</a>의 구별로부터 시작된다.</p>
<p>많은 언어에는 파란색과 초록색을 구별하는 낱말이 없고, 둘을 하나로 일컫는 낱말만 존재한다.</p>
