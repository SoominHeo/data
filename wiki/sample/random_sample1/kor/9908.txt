<p><b>여순항 포위</b>(<span style="font-size: smaller;"><a href="/wiki/%EC%9D%BC%EB%B3%B8%EC%96%B4" title="일본어">일본어</a>:</span> <span lang="ja" xml:lang="ja">旅順攻囲戦</span> <small>료준 코이센<sup>[<a href="/wiki/%EC%9C%84%ED%82%A4%EB%B0%B1%EA%B3%BC:%EC%9D%BC%EB%B3%B8%EC%96%B4%EC%9D%98_%ED%95%9C%EA%B8%80_%ED%91%9C%EA%B8%B0" title="위키백과:일본어의 한글 표기">*</a>]</sup></small>, <span style="font-size: smaller;"><a href="/wiki/%EB%9F%AC%EC%8B%9C%EC%95%84%EC%96%B4" title="러시아어">러시아어</a>:</span> <span lang="ru" xml:lang="ru">Оборона Порт-Артура</span>)는 <a href="/wiki/%EB%9F%AC%EC%9D%BC_%EC%A0%84%EC%9F%81" title="러일 전쟁">러일 전쟁</a> 중기부터 후기까지 일어난 최대의 전투로 전략적 요충지이던 <a href="/wiki/%EB%A7%8C%EC%A3%BC" title="만주">만주</a>의 여순항을 둘러싸고 벌어진 지상전과 해상전이다.</p>
<p>당시 여순항 요새는 <a href="/wiki/%EC%B2%AD%EC%9D%BC_%EC%A0%84%EC%9F%81" title="청일 전쟁">청일 전쟁</a> 당시 <a href="/wiki/%EC%97%AC%EC%88%9C_%EC%A0%84%ED%88%AC" title="여순 전투">여순 전투</a>에서 단 하루만에 일본군에게 함락당했으나 <a href="/wiki/%EB%9F%AC%EC%9D%BC_%EC%A0%84%EC%9F%81" title="러일 전쟁">러일 전쟁</a> 당시에는 야전 축성의 달인이라 불렸던 <a href="/wiki/%EB%A1%9C%EB%A7%8C_%EC%BD%98%EB%93%9C%EB%9D%BC%EC%B2%B8%EC%BD%94" title="로만 콘드라첸코">로만 콘드라첸코</a> 소장에 의해 난공불락 요새로 탈바꿈하여 각종 포와 M1893 <a href="/wiki/%EB%A7%A5%EC%8B%AC%EA%B8%B0%EA%B4%80%EC%B4%9D" title="맥심기관총">맥심기관총</a>까지 갖추고 있었다.</p>
<p>이 전투에서 500kg의 포탄을 쏠 수 있는 대규모 280mm 포와 조명 <a href="/wiki/%EA%B3%A1%EC%82%AC%ED%8F%AC" title="곡사포">곡사포</a>, <a href="/wiki/%EB%A7%A5%EC%8B%AC%EA%B8%B0%EA%B4%80%EC%B4%9D" title="맥심기관총">맥심기관총</a>, <a href="/wiki/%EB%B3%BC%ED%8A%B8_%EC%95%A1%EC%85%98" title="볼트 액션">볼트 액션</a> 소총, 가시 철조망, 전기 담장, 아크 램프 서치, 라디오 신호 등 갖가지 기술도 이용되었다.</p>

<h2><span class="mw-headline" id=".EB.B0.B0.EA.B2.BD">배경</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%97%AC%EC%88%9C%ED%95%AD_%ED%8F%AC%EC%9C%84&amp;action=edit&amp;section=1" title="부분 편집: 배경">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p><a href="/wiki/1904%EB%85%84" title="1904년">1904년</a> <a href="/wiki/%EB%9F%AC%EC%9D%BC_%EC%A0%84%EC%9F%81" title="러일 전쟁">러일 전쟁</a>이 발발하자 당시 러시아군이 주둔하던 <a href="/wiki/%EB%82%A8%EB%A7%8C%EC%A3%BC" title="남만주">남만주</a> 일대는 전쟁터가 되었다. 그 중 한 곳이 여순항으로 그곳은 러시아의 태평양 함대가 주둔한 유일한 극동 부동항으로 전략적 요충지이기도 했다.</p>
<p>러시아군은 <a href="/wiki/%EC%95%84%EB%82%98%ED%86%A8%EB%A6%AC_%EC%8A%A4%ED%85%8C%EC%85%80" title="아나톨리 스테셀">아나톨리 스테셀</a> 총장이 이끄는 5만 명의 러시아군과 약 16척 정도의 함대와 승무원들을 갖추고 있었고 여순항의 요새 주변에는 174 고지, <a class="mw-redirect" href="/wiki/203_%EA%B3%A0%EC%A7%80" title="203 고지">203 고지</a> 등의 언덕이 있어 방어를 더욱 강력하게 하였고 <a class="mw-redirect" href="/wiki/%EB%A1%9C%EB%A7%8C_%EC%BD%98%ED%8A%B8%EB%9D%BC%EC%B2%B8%EC%BD%94" title="로만 콘트라첸코">로만 콘트라첸코</a> 소장의 주도로 1.5km의 방어 라인에 콘크리트 요새와 기계 포상 설치 및 참호 연결을 하게 되었다.</p>
<p>이에 일본군은 <a href="/wiki/1904%EB%85%84" title="1904년">1904년</a> <a href="/wiki/7%EC%9B%94_30%EC%9D%BC" title="7월 30일">7월 30일</a> <a href="/wiki/%EB%85%B8%EA%B8%B0_%EB%A7%88%EB%A0%88%EC%8A%A4%EC%BC%80" title="노기 마레스케">노기 마레스케</a> 남작이 지휘하는 일본군 육군 제3군 9만 명과 포병의 지원을 받아 여순항으로 진격했다.</p>
<h2><span class="mw-headline" id=".EC.A0.84.ED.88.AC_.EA.B3.BC.EC.A0.95">전투 과정</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%97%AC%EC%88%9C%ED%95%AD_%ED%8F%AC%EC%9C%84&amp;action=edit&amp;section=2" title="부분 편집: 전투 과정">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<h3><span class="mw-headline" id=".EA.B3.A0.EC.95.84_.EA.B3.A0.EC.A7.80_.EC.A0.84.ED.88.AC">고아 고지 전투</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%97%AC%EC%88%9C%ED%95%AD_%ED%8F%AC%EC%9C%84&amp;action=edit&amp;section=3" title="부분 편집: 고아 고지 전투">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p><a href="/wiki/1904%EB%85%84" title="1904년">1904년</a> <a href="/wiki/8%EC%9B%94_7%EC%9D%BC" title="8월 7일">8월 7일</a> ~ <a href="/wiki/8%EC%9B%94_19%EC%9D%BC" title="8월 19일">8월 19일</a> 일본군은 120mm의 포로 포격을 시작했다. 일본군 함대는 해안 포격에 참여하며 북동부 지역에 있는 약 604m의 2개 군대가 고립된 고아 고지를 공격했다.</p>
<p>그 고지는 심한 방어 태세를 갖추진 않았지만 급경사였고 러시아군이 설치한 여러 장애물들이 있어 진격하는 데 많은 어려움이 있었다.</p>
<p><a href="/wiki/8%EC%9B%94_7%EC%9D%BC" title="8월 7일">8월 7일</a> 아침부터 야간까지 두 개 언덕을 향해 포병 공격을 시작하였으나 주변 강의 범람과 집중 호우로 많은 병사들이 익사하고 예상치 못한 러시아군의 기관총 공격으로 큰 피해를 냈다. <a href="/wiki/8%EC%9B%94_8%EC%9D%BC" title="8월 8일">8월 8일</a> 노기 마레스케 남작이 다시 포병 공격을 하여 러시아의 <a href="/wiki/%EB%B0%A9%ED%98%B8%EC%88%9C%EC%96%91%ED%95%A8" title="방호순양함">방호순양함</a> <a href="/wiki/%EB%85%B8%EB%B9%84%ED%81%AC_(%EB%B0%A9%ED%98%B8%EC%88%9C%EC%96%91%ED%95%A8)" title="노비크 (방호순양함)">노비크</a>가 화재로 큰 피해를 내고 20시간에 걸친 전투 끝에 <a href="/wiki/8%EC%9B%94_9%EC%9D%BC" title="8월 9일">8월 9일</a> 일본군은 고지를 함락시켰다.</p>
<p>이 전투에서 일본군 1230명의 사상자를 냈고 러시아군은 여순항에 갇힌 태평양 함대가 위험하게 되었다. 이에 <a href="/wiki/%EB%B9%8C%ED%97%AC%EB%A6%84_%EB%B9%84%ED%8A%B8%EA%B2%8C%ED%94%84%ED%8A%B8" title="빌헬름 비트게프트">빌헬름 비트게프트</a> 제독이 탈출을 시도해 <a href="/wiki/%EB%B8%94%EB%9D%BC%EB%94%94%EB%B3%B4%EC%8A%A4%ED%86%A0%ED%81%AC" title="블라디보스토크">블라디보스토크</a>로 가려했으나 <a href="/wiki/8%EC%9B%94_10%EC%9D%BC" title="8월 10일">8월 10일</a> <a href="/wiki/%ED%99%A9%ED%95%B4_%ED%95%B4%EC%A0%84_(1904%EB%85%84)" title="황해 해전 (1904년)">황해 해전</a>에서 일본의 <a href="/wiki/%EB%8F%84%EA%B3%A0_%ED%97%A4%EC%9D%B4%ED%95%98%EC%B9%98%EB%A1%9C" title="도고 헤이하치로">도고 헤이하치로</a> 제독에게 모두 격침당하고 전사하게 된다.</p>

<h3><span class="mw-headline" id="174_.EA.B3.A0.EC.A7.80_.EC.A0.84.ED.88.AC">174 고지 전투</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%97%AC%EC%88%9C%ED%95%AD_%ED%8F%AC%EC%9C%84&amp;action=edit&amp;section=4" title="부분 편집: 174 고지 전투">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p><a href="/wiki/1904%EB%85%84" title="1904년">1904년</a> <a href="/wiki/8%EC%9B%94_13%EC%9D%BC" title="8월 13일">8월 13일</a> 노기 장군은 늑대 고지에서 러시아군의 정찰 기구 격추에 실패하고 러시아군의 포병들이 조정 부족이 심각한 것을 알고 가능한 빨리 여순항으로 병력을 내려가기 위해 174 고지를 공격하기로 했다.</p>
<p>일본군은 여순항 요새 수비대의 항복을 요구하는 전문을 보냈으나 러시아군은 즉각 거부하고 <a href="/wiki/8%EC%9B%94_19%EC%9D%BC" title="8월 19일">8월 19일</a> 새벽 일본군은 공격을 시작했다. 이 전투에서 러시아군은 이전 <a href="/wiki/%EB%82%A8%EC%82%B0_%EC%A0%84%ED%88%AC" title="남산 전투">남산 전투</a>를 이끌었던 트레차코프 대령의 동시베리아 연대가 맡았다.</p>
<p>이전 남산 전투에서와 마찬가지로 트레차코프 대령을 끝까지 후퇴를 거부하며 참호에서 첫 번째 방어 라인을 지켰으나 다음날 8월 20일 원군이 오지 않아 병력의 절반이 전사, 부상 또는 탈영을 해 혼란에 빠지고 그 사이 일본군의 대공세로 고지는 함락된다.</p>
<p>이 전투에서 일본군인 1800명이 사상당했고 러시아군은 1000명이 사상당했으며 <a href="/wiki/8%EC%9B%94_24%EC%9D%BC" title="8월 24일">8월 24일</a> 노기 장군은 174 고지의 동부와 서부에 걸쳐 있는 웬타이 계곡으로 병력을 파견했으나 <a href="/wiki/8%EC%9B%94_25%EC%9D%BC" title="8월 25일">8월 25일</a> 대패로 끝나게 된다. 그 시기 <a href="/wiki/%EC%98%A4%EC%95%BC%EB%A7%88_%EC%9D%B4%EC%99%80%EC%98%A4" title="오야마 이와오">오야마 이와오</a> 장군은 <a href="/wiki/%EC%9A%94%EB%8F%99_%EC%A0%84%ED%88%AC" title="요동 전투">요동 전투</a>에서 <a class="mw-redirect" href="/wiki/%EC%95%8C%EB%A0%89%EC%84%B8%EC%9D%B4_%EC%BF%A0%EB%A1%9C%ED%8C%8C%ED%8A%B8%ED%82%A8" title="알렉세이 쿠로파트킨">알렉세이 쿠로파트킨</a> 사령관이 이끄는 러시아군을 격파한다.</p>
<h3><span class="mw-headline" id=".ED.8F.AC.EC.9C.84">포위</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%97%AC%EC%88%9C%ED%95%AD_%ED%8F%AC%EC%9C%84&amp;action=edit&amp;section=5" title="부분 편집: 포위">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>노기 장군은 여순항 요새에 대한 직접적인 공격보다는 요새 아래의 콘크리트 벽을 폭파시키고 참호와 터널을 만드는 데 집중해 1만 6000명 이상의 병력과 대포를 강화했는데 거대한 11인치 곡사포들을 곳곳에 설치했다.</p>
<p>그 사이 여순항 요새의 수비대들은 동료 임원들 간의 협력 부족과 <a href="/wiki/%EC%B0%A8%EB%A5%B4" title="차르">차르</a>에 대한 불평, <a href="/wiki/%EA%B4%B4%ED%98%88%EB%B3%91" title="괴혈병">괴혈병</a>과 신선한 음식의 부족으로 인한 <a href="/wiki/%EC%9D%B4%EC%A7%88" title="이질">이질</a>을 겪게 된다. 한편 노기 장군은 동쪽에 있는 <a class="mw-redirect" href="/wiki/203_%EA%B3%A0%EC%A7%80" title="203 고지">203 고지</a>에 관심을 돌리게 되는데 그곳은 여순항에 주둔한 러시아 함대를 한눈에 내려다 볼 수 있는 전략적 요충지였다.</p>
<p>9월 중순 일본군은 8km의 참호 공사를 끝내고 203 고지로 병력을 이동시켰으나 러시아군의 포격과 기관총 공격으로 많은 사상자만 냈다. 끝없는 노기 장군의 시도 끝에 일본군은 3500명의 사상자를 냈고 <a href="/wiki/10%EC%9B%94_29%EC%9D%BC" title="10월 29일">10월 29일</a> 노기 장군은 <a href="/wiki/%EB%A9%94%EC%9D%B4%EC%A7%80_%EC%B2%9C%ED%99%A9" title="메이지 천황">메이지 천황</a>의 생일을 맞아 다시 공격을 시작했으나 결국 124명의 장교와 3611명의 병사들만 전사하게 되었다.</p>
<p>겨울에 일본군은 280mm 곡사포로 추가 증원해 배치했는데 그 시기 러시아군의 <a href="/wiki/%EB%B0%9C%ED%8A%B8_%ED%95%A8%EB%8C%80" title="발트 함대">발트 함대</a>가 이동하고 있다는 정보가 들려 노기 장군은 지체할 시간 없이 203 고지를 공격하기로 결정했다.</p>

<p>일본군의 여러 차례에 걸친 공격으로 203 고지에 관심을 가지지 않던 <a href="/wiki/%EC%95%84%EB%82%98%ED%86%A8%EB%A6%AC_%EC%8A%A4%ED%85%8C%EC%85%80" title="아나톨리 스테셀">아나톨리 스테셀</a> 총장과 <a class="mw-redirect" href="/wiki/%EB%A1%9C%EB%A7%8C_%EC%BD%98%ED%8A%B8%EB%9D%BC%EC%B2%B8%EC%BD%94" title="로만 콘트라첸코">로만 콘트라첸코</a> 소장은 즉각 방어를 강화하고 가파른 측면에 거대한 벽을 쌓고 방형 포루로 보호했으며 다른 고지의 요새로도 참호를 연결했다.</p>
<p>러시아군이 방어 태세를 끝내고 203 고지를 난공불락 요새로 만들고 발트 함대가 인도양을 통과할 시기 <a href="/wiki/1904%EB%85%84" title="1904년">1904년</a> <a href="/wiki/11%EC%9B%94_26%EC%9D%BC" title="11월 26일">11월 26일</a> 일본군은 다시 공격을 시작했다. 요새에 대한 정면 돌파를 시도하지만 수비대의 포격과 기관총으로 인해 4000명의 사상자를 냈지만 비공식적으로는 그 두 배에 달할 것으로 추측한다.</p>
<p><a href="/wiki/11%EC%9B%94_28%EC%9D%BC" title="11월 28일">11월 28일</a> 오전 8시 30분에 대규모 포병 지원으로 다시 일본군은 고지를 공격해 280mm의 포까지 쐈지만 러시아군 수비대의 잘 싸여진 대열과 수류탄, 기관총 공격으로 큰 피해를 입었다. <a href="/wiki/12%EC%9B%94_5%EC%9D%BC" title="12월 5일">12월 5일</a> 오전 10시 30분을 가해 다시 대공세를 펼쳐 포격 지원으로 주변 두 개의 언덕을 함락시킨 뒤 17시 일본군은 무사히 203 고지를 함락시키는 데 성공한다.</p>
<p>그러나 이 전투에서 노기 장군의 무리한 작전으로 노기 장군의 두 아들을 포함해 엄청난 피해를 냈다.</p>

<p>203 고지를 함락시켜 유리한 위치에서 여순항을 내려다 볼 수 있게된 일본군은 280mm의 포와 500kg의 철갑탄, 곡사포 등으로 공격을 시작해 <a href="/wiki/1904%EB%85%84" title="1904년">1904년</a> <a href="/wiki/12%EC%9B%94_5%EC%9D%BC" title="12월 5일">12월 5일</a> 여순항에 정박해 있던 배수량 1만 960톤의 <a class="new" href="/w/index.php?title=%ED%8F%B4%ED%83%80%EB%B0%94_(%EB%9F%AC%EC%8B%9C%EC%95%84_%EC%A0%84%ED%95%A8)&amp;action=edit&amp;redlink=1" title="폴타바 (러시아 전함) (없는 문서)">폴타바</a>가 대폭발을 일으키며 침몰했다.</p>
<p>그리고 <a href="/wiki/12%EC%9B%94_7%EC%9D%BC" title="12월 7일">12월 7일</a> 배수량 1만 2900톤의 전함 <a href="/wiki/%EB%A0%88%ED%8A%B8%EB%B9%84%EC%9E%94_(%EC%A0%84%ED%95%A8)" title="레트비잔 (전함)">레트비잔</a>이 포탄을 8발이나 맞아 뷔렌 제독이 부상을 입고 사실상 고철로 변해버렸으며 <a href="/wiki/12%EC%9B%94_9%EC%9D%BC" title="12월 9일">12월 9일</a> <a href="/wiki/%ED%8F%AC%EB%B2%A0%EB%8B%A4_(%EC%A0%84%ED%95%A8)" title="포베다 (전함)">포베다</a>와 <a class="new" href="/w/index.php?title=%EB%9F%AC%EC%8B%9C%EC%95%84_%EC%A0%84%ED%95%A8_%ED%8D%BC%EB%A0%88%EC%8A%A4%EB%B9%84%ED%8A%B8&amp;action=edit&amp;redlink=1" title="러시아 전함 퍼레스비트 (없는 문서)">퍼레스비트</a>, 순양함 <a class="new" href="/w/index.php?title=%EB%9F%AC%EC%8B%9C%EC%95%84_%EC%88%9C%EC%96%91%ED%95%A8_%ED%8C%94%EB%9D%BC%EB%8B%A4&amp;action=edit&amp;redlink=1" title="러시아 순양함 팔라다 (없는 문서)">팔라다</a>, <a class="new" href="/w/index.php?title=%EB%9F%AC%EC%8B%9C%EC%95%84_%EC%88%9C%EC%96%91%ED%95%A8_%EB%B0%94%EC%96%80&amp;action=edit&amp;redlink=1" title="러시아 순양함 바얀 (없는 문서)">바얀</a> 등이 파괴되었다. 곧이어 배수량 1만 960톤의 전함 <a class="new" href="/w/index.php?title=%EB%9F%AC%EC%8B%9C%EC%95%84_%EC%A0%84%ED%95%A8_%EC%84%B8%EB%B0%94%EC%8A%A4%ED%86%A0%ED%8F%B4&amp;action=edit&amp;redlink=1" title="러시아 전함 세바스토폴 (없는 문서)">세바스토폴</a>도 파괴당했으며 몇 척의 소함정을 제외한 4척의 전함과 2척의 순양함, 그 외 10척의 각종 함정들이 모두 격침당해 버렸다.</p>
<p>이 포격은 전함들 외에도 조선소와 시가지까지 이어져 사실상 함정의 수리가 불가능해졌고 시가를 공황상태로 몰아넣었으며 일본군은 항구 밖 광산에서 순양함 <a class="new" href="/w/index.php?title=%EC%9D%BC%EB%B3%B8_%EC%88%9C%EC%96%91%ED%95%A8_%EB%8B%A4%EC%B9%B4%EC%82%AC%EA%B3%A0&amp;action=edit&amp;redlink=1" title="일본 순양함 다카사고 (없는 문서)">다카사고</a>를 잃었다. 이후 <a href="/wiki/1905%EB%85%84" title="1905년">1905년</a> 1월 여순항에서 탈출한 세바스토폴의 선장 <a href="/wiki/%EB%8B%88%EC%BD%9C%EB%9D%BC%EC%9D%B4_%ED%8F%B0_%EC%97%90%EC%84%BC" title="니콜라이 폰 에센">니콜라이 폰 에센</a>이 항복했고 이때 침몰한 여러 함정들을 일본군에게 인양되어 수리된 후 일본군의 함정으로 재탄생하게 되었다.</p>
<h3><span class="mw-headline" id=".ED.95.AD.EB.B3.B5">항복</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%97%AC%EC%88%9C%ED%95%AD_%ED%8F%AC%EC%9C%84&amp;action=edit&amp;section=8" title="부분 편집: 항복">편집</a><span class="mw-editsection-bracket">]</span></span></h3>

<div class="thumbinner" style="width:222px;"><a class="image" href="/wiki/%ED%8C%8C%EC%9D%BC:Nogi_and_Stessel.jpg"><img alt="" class="thumbimage" data-file-height="693" data-file-width="905" height="168" src="//upload.wikimedia.org/wikipedia/commons/thumb/1/19/Nogi_and_Stessel.jpg/220px-Nogi_and_Stessel.jpg" srcset="//upload.wikimedia.org/wikipedia/commons/thumb/1/19/Nogi_and_Stessel.jpg/330px-Nogi_and_Stessel.jpg 1.5x, //upload.wikimedia.org/wikipedia/commons/thumb/1/19/Nogi_and_Stessel.jpg/440px-Nogi_and_Stessel.jpg 2x" width="220"/></a>
<div class="thumbcaption">
<div class="magnify"><a class="internal" href="/wiki/%ED%8C%8C%EC%9D%BC:Nogi_and_Stessel.jpg" title="실제 크기로"></a></div>
항복한 직후 <a href="/wiki/%EB%85%B8%EA%B8%B0_%EB%A7%88%EB%A0%88%EC%8A%A4%EC%BC%80" title="노기 마레스케">노기 마레스케</a>(중앙 왼쪽)과 <a href="/wiki/%EC%95%84%EB%82%98%ED%86%A8%EB%A6%AC_%EC%8A%A4%ED%85%8C%EC%85%80" title="아나톨리 스테셀">아나톨리 스테셀</a>(중앙 오른쪽)과 그들의 휘하 장교들</div>
</div>
</div>
<div class="thumb tleft">
<div class="thumbinner" style="width:222px;"><a class="image" href="/wiki/%ED%8C%8C%EC%9D%BC:Port_Arthur_from_Gold_Hill.jpg"><img alt="" class="thumbimage" data-file-height="1693" data-file-width="2437" height="153" src="//upload.wikimedia.org/wikipedia/commons/thumb/8/84/Port_Arthur_from_Gold_Hill.jpg/220px-Port_Arthur_from_Gold_Hill.jpg" srcset="//upload.wikimedia.org/wikipedia/commons/thumb/8/84/Port_Arthur_from_Gold_Hill.jpg/330px-Port_Arthur_from_Gold_Hill.jpg 1.5x, //upload.wikimedia.org/wikipedia/commons/thumb/8/84/Port_Arthur_from_Gold_Hill.jpg/440px-Port_Arthur_from_Gold_Hill.jpg 2x" width="220"/></a>
<div class="thumbcaption">
<div class="magnify"><a class="internal" href="/wiki/%ED%8C%8C%EC%9D%BC:Port_Arthur_from_Gold_Hill.jpg" title="실제 크기로"></a></div>
러시아의 제1태평양 함대, 후에 일본군에게 인양되어 일본 전함으로 개조되었다가 반환</div>
</div>
</div>
<p>203 고지의 함락과 제1태평양 함대의 괴멸은 여순 전투의 종막이나 마찬가지였고 이후 수주일에 걸쳐 다른 고지에서의 저항이 이어졌지만 여순 요새 사령관 <a href="/wiki/%EC%95%84%EB%82%98%ED%86%A8%EB%A6%AC_%EC%8A%A4%ED%85%8C%EC%85%80" title="아나톨리 스테셀">아나톨리 스테셀</a> 중장은 사실상 전의를 상실해버렸다. 203 고지 부근의 포대들에서는 수비대가 <a href="/wiki/12%EC%9B%94_6%EC%9D%BC" title="12월 6일">12월 6일</a> 철수를 시작했고 자연히 주변의 요새들이 고립되어 버렸다.</p>
<p><a href="/wiki/12%EC%9B%94_7%EC%9D%BC" title="12월 7일">12월 7일</a>, 스테셀 중장은 여순 요새 사령부에서 작전 회의를 소집했으나 뚜렷한 대책이 없었고 203 고지의 함락으로 주요 포대와 보루들이 사실상 일본군의 정확한 포격 유도 하에 무차별 포격을 당하게 생겼고 부상병들이 속출했다.</p>
<p>여기에 일본 제3군은 공병대가 땅굴을 파고 들어가 폭약을 장착, 보루를 기초 부분부터 파괴하는 전술을 구사했고 이제 러시아군은 지하에서부터 속수무책으로 공격을 받게 되었으며 <a href="/wiki/12%EC%9B%94_15%EC%9D%BC" title="12월 15일">12월 15일</a> <a class="mw-redirect" href="/wiki/%EB%A1%9C%EB%A7%8C_%EC%BD%98%ED%8A%B8%EB%9D%BC%EC%B2%B8%EC%BD%94" title="로만 콘트라첸코">콘트라첸코 소장</a>이 전사하면서 여순 요새의 방위력은 더욱 약화되었다.</p>
<p>공병과 포병의 긴밀한 협조 하에 그 동안 숱한 일본군을 살상한 이룡산의 포대가 <a href="/wiki/12%EC%9B%94_28%EC%9D%BC" title="12월 28일">12월 28일</a> 함락됨으로써 스테셀 중장은 절망에 빠져버렸으며 <a href="/wiki/12%EC%9B%94_31%EC%9D%BC" title="12월 31일">12월 31일</a>에는 송수산의 보루와 포대들이 일본군에게 함락되었고 마침내 <a href="/wiki/1905%EB%85%84" title="1905년">1905년</a> <a href="/wiki/1%EC%9B%94_1%EC%9D%BC" title="1월 1일">1월 1일</a>, 15시를 기해 스테셀 중장은 항복을 결심했다.</p>
<p>그 시각에도 전투는 계속되고 있었고 주요 포대와 보루가 함락된 러시아군은 서서히 밀리고 있었다. 더 이상의 저항은 막대한 희생자만을 낼 뿐이라는 것이 스테셀 중장의 생각이었다. 그는 <a href="/wiki/%EC%98%81%EC%96%B4" title="영어">영어</a>로 항복문서를 작성하게 했다. 당시 국제 공용어는 <a href="/wiki/%ED%94%84%EB%9E%91%EC%8A%A4%EC%96%B4" title="프랑스어">프랑스어</a>였지만 일본인에게는 영어가 공용어라는 것이 일반 상식이었기 때문이다.</p>
<p>이 항복문서를 든 견습사관 마르첸코는 일본군 제1 보병사단 2연대의 병사들과 조우했고 곧 항복교섭이 시작되어 일본측 대표로는 <a class="new" href="/w/index.php?title=%EC%9D%B4%EC%A7%80%EC%B9%98_%EA%B3%A0%EC%8A%A4%EC%BC%80&amp;action=edit&amp;redlink=1" title="이지치 고스케 (없는 문서)">이지치 고스케</a> 소장이, 러시아측 대표로는 참모장 레이스 대령이 선임되었다. 장소는 수사영이 되었고 마침내 <a href="/wiki/1905%EB%85%84" title="1905년">1905년</a> <a href="/wiki/1%EC%9B%94_2%EC%9D%BC" title="1월 2일">1월 2일</a>, 16시 35분 항복 협상을 마무리 지었다.</p>
<h2><span class="mw-headline" id=".EA.B2.B0.EA.B3.BC_.EB.B0.8F_.EC.98.81.ED.96.A5">결과 및 영향</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%97%AC%EC%88%9C%ED%95%AD_%ED%8F%AC%EC%9C%84&amp;action=edit&amp;section=9" title="부분 편집: 결과 및 영향">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<div class="thumb tleft">
<div class="thumbinner" style="width:202px;"><a class="image" href="/wiki/%ED%8C%8C%EC%9D%BC:Russia_stamp_Siege_of_Port_Arthur_2004_10r.jpg"><img alt="" class="thumbimage" data-file-height="1148" data-file-width="922" height="249" src="//upload.wikimedia.org/wikipedia/commons/thumb/8/8b/Russia_stamp_Siege_of_Port_Arthur_2004_10r.jpg/200px-Russia_stamp_Siege_of_Port_Arthur_2004_10r.jpg" srcset="//upload.wikimedia.org/wikipedia/commons/thumb/8/8b/Russia_stamp_Siege_of_Port_Arthur_2004_10r.jpg/300px-Russia_stamp_Siege_of_Port_Arthur_2004_10r.jpg 1.5x, //upload.wikimedia.org/wikipedia/commons/thumb/8/8b/Russia_stamp_Siege_of_Port_Arthur_2004_10r.jpg/400px-Russia_stamp_Siege_of_Port_Arthur_2004_10r.jpg 2x" width="200"/></a>
<div class="thumbcaption">
<div class="magnify"><a class="internal" href="/wiki/%ED%8C%8C%EC%9D%BC:Russia_stamp_Siege_of_Port_Arthur_2004_10r.jpg" title="실제 크기로"></a></div>
2004년 여순항 포위전 100주년을 기념해 발행된 러시아 우표</div>
</div>
</div>
<p>러시아군은 총 3만 5000명의 병력으로 전투를 시작해 12,000명의 사상자를 냈지만 이 중 전사자는 3,000여명에 불과했고 그나마 대부분은 203 고지 전투 이후에 집중적으로 늘어난 것이니 말이다.</p>
<p>반면 <a href="/wiki/%EB%85%B8%EA%B8%B0_%EB%A7%88%EB%A0%88%EC%8A%A4%EC%BC%80" title="노기 마레스케">노기 마레스케</a> 중장의 제3군은 최초 10만의 병력으로 시작해 속속 들이 보충된 병력까지 합쳐 총 13만에 이르는 대군을 투입해 155일간의 전투를 치렀으나 전술적인 변화없이 무모하기 그지없는 공세의 반복 결과 일본 제3군은 전사 1만 5400명을 포함해 총 6만 212명이라는 엄청난 수의 사상자를 냈다.</p>
<p>이는 전군의 절반에 가까운 수치로 러시아군의 5배가 넘는 막대한 손실이었고 특히 장교의 손실이 엄청나 전투 종료 시까지 온전히 살아남은 이들이 겨우 10명에 불과했다는 점은 크나큰 충격이었다.</p>
<p>이후 노기 장군이 여순항을 떠난 후 <a href="/wiki/%EC%98%A4%EC%95%BC%EB%A7%88_%EC%9D%B4%EC%99%80%EC%98%A4" title="오야마 이와오">오야마 이와오</a> 장군이 남은 제3군의 1만 2000명의 병력을 이끌고 <a href="/wiki/%EB%B4%89%EC%B2%9C_%EC%A0%84%ED%88%AC" title="봉천 전투">봉천 전투</a>를 치렀고 러시아군은 이 전투의 패배로 <a href="/wiki/%EC%93%B0%EC%8B%9C%EB%A7%88_%ED%95%B4%EC%A0%84" title="쓰시마 해전">쓰시마 해전</a>과 <a href="/wiki/%ED%8F%AC%EC%B8%A0%EB%A8%B8%EC%8A%A4_%EC%A1%B0%EC%95%BD" title="포츠머스 조약">포츠머스 조약</a>, <a class="mw-disambig" href="/wiki/%ED%94%BC%EC%9D%98_%EC%9D%BC%EC%9A%94%EC%9D%BC" title="피의 일요일">피의 일요일 사건</a>으로 이어지게 된다.</p>
