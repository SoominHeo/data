<p><b>역학</b>(力學, <span style="font-size: smaller;"><a href="/wiki/%EB%AC%B8%ED%99%94%EC%96%B4" title="문화어">문화어</a>:</span> 력학, <span style="font-size: smaller;"><a href="/wiki/%EA%B3%A0%EB%8C%80_%EA%B7%B8%EB%A6%AC%EC%8A%A4%EC%96%B4" title="고대 그리스어">고대 그리스어</a>:</span> <span lang="grc" xml:lang="grc">Μηχανική</span>)은 <a href="/wiki/%EB%AC%BC%EB%A6%AC%ED%95%99" title="물리학">물리학</a>의 한 분야로, <a class="mw-redirect" href="/wiki/%ED%9E%98_(%EB%AC%BC%EB%A6%AC%ED%95%99)" title="힘 (물리학)">외력</a>을 받고 있는 <a href="/wiki/%EB%AC%BC%EC%B2%B4" title="물체">물체</a>의 정지 또는 <a href="/wiki/%EC%9A%B4%EB%8F%99_(%EB%AC%BC%EB%A6%AC%ED%95%99)" title="운동 (물리학)">운동</a> 상태를 설명하고 예측하는 <a class="mw-redirect" href="/wiki/%EC%9E%90%EC%97%B0_%EA%B3%BC%ED%95%99" title="자연 과학">자연 과학</a>이다.<sup class="reference" id="cite_ref-1"><a href="#cite_note-1">[1]</a></sup> 역학에는 여러 하위 분야가 있으며, 크게 보아 <a href="/wiki/%EA%B3%A0%EC%A0%84%EC%97%AD%ED%95%99" title="고전역학">고전역학</a>과 <a href="/wiki/%EC%96%91%EC%9E%90%EC%97%AD%ED%95%99" title="양자역학">양자역학</a>으로 구분할 수 있다.<sup class="reference" id="cite_ref-ReferenceA_2-0"><a href="#cite_note-ReferenceA-2">[2]</a></sup></p>

<div class="rellink boilerplate seealso"><img alt="" data-file-height="128" data-file-width="128" height="16" src="//upload.wikimedia.org/wikipedia/commons/thumb/e/ec/Crystal_Clear_app_xmag.svg/16px-Crystal_Clear_app_xmag.svg.png" srcset="//upload.wikimedia.org/wikipedia/commons/thumb/e/ec/Crystal_Clear_app_xmag.svg/24px-Crystal_Clear_app_xmag.svg.png 1.5x, //upload.wikimedia.org/wikipedia/commons/thumb/e/ec/Crystal_Clear_app_xmag.svg/32px-Crystal_Clear_app_xmag.svg.png 2x" width="16"/> <a href="/wiki/%EB%AC%BC%EB%A6%AC%ED%95%99%EC%9D%98_%EC%97%AD%EC%82%AC" title="물리학의 역사">물리학의 역사</a> 문서를 참고하십시오.</div>
<p>인류는 <a href="/wiki/%EC%84%A0%EC%82%AC_%EC%8B%9C%EB%8C%80" title="선사 시대">선사 시대</a>부터 이미 <a href="/wiki/%EB%B9%97%EB%A9%B4" title="빗면">빗면</a>, <a href="/wiki/%EB%B0%94%ED%80%B4" title="바퀴">바퀴</a>, <a href="/wiki/%EC%A7%80%EB%A0%88" title="지레">지레</a>, <a href="/wiki/%EB%8F%84%EB%A5%B4%EB%9E%98" title="도르래">도르래</a>와 같은 <a href="/wiki/%EB%8B%A8%EC%88%9C_%EA%B8%B0%EA%B3%84" title="단순 기계">단순 기계</a>를 <a href="/wiki/%EB%8F%84%EA%B5%AC" title="도구">도구</a>로 사용하여 왔다. 고대에 들어 여러 문화에서는 역학 지식을 이용한 여러 기술들을 발전시켰다. 고대 그리스의 <a href="/wiki/%EC%95%84%EB%A5%B4%ED%82%A4%EB%A9%94%EB%8D%B0%EC%8A%A4" title="아르키메데스">아르키메데스</a>와 같은 학자들은 <a href="/wiki/%EC%95%84%EB%A5%B4%ED%82%A4%EB%A9%94%EB%8D%B0%EC%8A%A4_%EB%82%98%EC%84%A0%EC%96%91%EC%88%98%EA%B8%B0" title="아르키메데스 나선양수기">아르키메데스 나선양수기</a><sup class="reference" id="cite_ref-3"><a href="#cite_note-3">[3]</a></sup> 나 <a href="/wiki/%EC%95%84%EB%A5%B4%ED%82%A4%EB%A9%94%EB%8D%B0%EC%8A%A4_%EA%B0%88%EA%B3%A0%EB%A6%AC" title="아르키메데스 갈고리">아르키메데스 갈고리</a>와 같은 기계를 만들기도 하였다.<sup class="reference" id="cite_ref-4"><a href="#cite_note-4">[4]</a></sup></p>
<p>12 세기의 아랍 과학자인 <a class="mw-redirect" href="/wiki/%EC%95%8C%ED%95%98%EC%9D%B4%ED%83%90" title="알하이탐">알하이탐</a>이 <a href="/wiki/%EA%B4%91%ED%95%99" title="광학">광학</a>을 연구하면서 <a href="/wiki/%EA%B3%BC%ED%95%99%EC%A0%81_%EB%B0%A9%EB%B2%95" title="과학적 방법">과학적 방법</a>을 도입하여 관찰을 바탕으로한 과학 이론을 발전시켰고<sup class="reference" id="cite_ref-5"><a href="#cite_note-5">[5]</a></sup>, <a href="/wiki/%EA%B0%88%EB%A6%B4%EB%A0%88%EC%98%A4_%EA%B0%88%EB%A6%B4%EB%A0%88%EC%9D%B4" title="갈릴레오 갈릴레이">갈릴레오 갈릴레이</a>는 <a class="mw-redirect" href="/wiki/%EC%88%98%EB%A6%AC_%EB%AA%A8%ED%98%95" title="수리 모형">수리 모형</a>(數理 模型)을 이용하여 물체의 운동을 예측하였다.<sup class="reference" id="cite_ref-6"><a href="#cite_note-6">[6]</a></sup> <a href="/wiki/%EC%95%84%EC%9D%B4%EC%9E%91_%EB%89%B4%ED%84%B4" title="아이작 뉴턴">아이작 뉴턴</a>이 <a class="mw-redirect" href="/wiki/%EA%B3%A0%EC%A0%84_%EC%97%AD%ED%95%99" title="고전 역학">고전 역학</a>을 정립하였고<sup class="reference" id="cite_ref-7"><a href="#cite_note-7">[7]</a></sup>, 이후 다양한 분야의 역학이 발전하게 되었다.</p>
<p>20세기 들어 고전 역학으로는 해결할 수 없는 현상들을 설명하기 위한 역학 이론들이 만들어졌는데, <a class="mw-redirect" href="/wiki/%EC%83%81%EB%8C%80%EC%84%B1_%EC%9D%B4%EB%A1%A0" title="상대성 이론">상대성 이론</a>과 <a class="mw-redirect" href="/wiki/%EC%96%91%EC%9E%90_%EC%97%AD%ED%95%99" title="양자 역학">양자 역학</a>과 같은 것들이 있다.<sup class="reference" id="cite_ref-8"><a href="#cite_note-8">[8]</a></sup></p>
<h2><span class="mw-headline" id=".EC.97.AD.ED.95.99.EC.9D.98_.EC.A2.85.EB.A5.98">역학의 종류</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%97%AD%ED%95%99_(%EB%AC%BC%EB%A6%AC%ED%95%99)&amp;action=edit&amp;section=2" title="부분 편집: 역학의 종류">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>역학에는 여러 하위 분야가 있으며, 크게 보아 <a href="/wiki/%EA%B3%A0%EC%A0%84%EC%97%AD%ED%95%99" title="고전역학">고전역학</a>과 <a href="/wiki/%EC%96%91%EC%9E%90%EC%97%AD%ED%95%99" title="양자역학">양자역학</a>으로 구분할 수 있다.<sup class="reference" id="cite_ref-ReferenceA_2-1"><a href="#cite_note-ReferenceA-2">[2]</a></sup> 보다 자세하게는 다음의 분야로 나눌 수 있다.<sup class="reference" id="cite_ref-9"><a href="#cite_note-9">[9]</a></sup></p>
<ul>
<li><a class="mw-redirect" href="/wiki/%EA%B3%A0%EC%A0%84_%EC%97%AD%ED%95%99" title="고전 역학">고전 역학</a>: <a href="/wiki/%EC%9B%90%EC%9E%90" title="원자">원자</a>보다 상대적으로 크고 <a class="mw-redirect" href="/wiki/%EB%B9%9B%EC%9D%98_%EC%86%8D%EB%8F%84" title="빛의 속도">빛의 속력</a>에 비해 아주 느리게 운동하는 물체의 운동을 다룬다.</li>
<li><a class="mw-redirect" href="/wiki/%EC%83%81%EB%8C%80%EC%84%B1_%EC%9D%B4%EB%A1%A0" title="상대성 이론">상대성 이론</a>: 빛의 속력과 비슷한 속력으로 움직이는 경우를 포함하여 모든 가능한 속력으로 움직이는 물체를 다룬다.</li>
<li><a href="/wiki/%EC%97%B4%EC%97%AD%ED%95%99" title="열역학">열역학</a>: <a href="/wiki/%EC%97%B4" title="열">열</a>, <a class="mw-redirect" href="/wiki/%EC%9D%BC_(%EB%AC%BC%EB%A6%AC)" title="일 (물리)">일</a>, <a href="/wiki/%EC%98%A8%EB%8F%84" title="온도">온도</a> 및 다수의 입자로 이루어진 <a href="/wiki/%EA%B3%84_(%EB%AC%BC%EB%A6%AC%ED%95%99)" title="계 (물리학)">계</a>의 통계적 성질을 다룬다.</li>
<li><a class="mw-redirect" href="/wiki/%EC%96%91%EC%9E%90_%EC%97%AD%ED%95%99" title="양자 역학">양자 역학</a>: 미시적 수준에서의 물질의 성질을 거시적 측정값과 연결시키는 이론의 집합이다.</li>
</ul>
<h2><span class="mw-headline" id=".EA.B3.A0.EC.A0.84_.EC.97.AD.ED.95.99">고전 역학</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%97%AD%ED%95%99_(%EB%AC%BC%EB%A6%AC%ED%95%99)&amp;action=edit&amp;section=3" title="부분 편집: 고전 역학">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<div class="rellink relarticle mainarticle"><img alt="" data-file-height="128" data-file-width="128" height="16" src="//upload.wikimedia.org/wikipedia/commons/thumb/e/ec/Crystal_Clear_app_xmag.svg/16px-Crystal_Clear_app_xmag.svg.png" srcset="//upload.wikimedia.org/wikipedia/commons/thumb/e/ec/Crystal_Clear_app_xmag.svg/24px-Crystal_Clear_app_xmag.svg.png 1.5x, //upload.wikimedia.org/wikipedia/commons/thumb/e/ec/Crystal_Clear_app_xmag.svg/32px-Crystal_Clear_app_xmag.svg.png 2x" width="16"/> 이 부분의 본문은 <a class="mw-redirect" href="/wiki/%EA%B3%A0%EC%A0%84_%EC%97%AD%ED%95%99" title="고전 역학">고전 역학</a>입니다.</div>
<p>고전 역학은 <a href="/wiki/%EC%95%84%EC%9D%B4%EC%9E%91_%EB%89%B4%ED%84%B4" title="아이작 뉴턴">아이작 뉴턴</a>의 <a href="/wiki/%EB%89%B4%ED%84%B4_%EC%9A%B4%EB%8F%99_%EB%B2%95%EC%B9%99" title="뉴턴 운동 법칙">운동 법칙</a>을 기반으로한 <a class="mw-redirect" href="/wiki/%EB%89%B4%ED%84%B4_%EC%97%AD%ED%95%99" title="뉴턴 역학">뉴턴 역학</a>에서 시작하여, <a href="/wiki/%EC%A1%B0%EC%A0%9C%ED%94%84%EB%A3%A8%EC%9D%B4_%EB%9D%BC%EA%B7%B8%EB%9E%91%EC%A3%BC" title="조제프루이 라그랑주">조제프루이 라그랑주</a>와 <a href="/wiki/%EC%9C%8C%EB%A6%AC%EC%97%84_%EB%A1%9C%EC%96%B8_%ED%95%B4%EB%B0%80%ED%84%B4" title="윌리엄 로언 해밀턴">윌리엄 로언 해밀턴</a> 등에 의해 발전된 역학이다. 고전 역학에서는 <a href="/wiki/%EB%AC%BC%EC%B2%B4" title="물체">물체</a>를 <a href="/wiki/%EC%A2%8C%ED%91%9C%EA%B3%84" title="좌표계">좌표계</a>에 표시할 수 있는 하나의 입자로 간주한다. 고전 역학은 일정한 질량을 갖는 물체에 어떠한 힘이 가해질 때 변화하는 <a href="/wiki/%EB%B3%80%EC%9C%84" title="변위">변위</a>에 대해 수리 모형을 사용하여 예측한다. <a href="/wiki/%EA%B0%88%EB%A6%B4%EB%A0%88%EC%98%A4_%EA%B0%88%EB%A6%B4%EB%A0%88%EC%9D%B4" title="갈릴레오 갈릴레이">갈릴레오 갈릴레이</a> 등의 과학자들은 고전 물리학이 성립되기 이전에 이미 수리 모형으로 물체의 운동을 예측하고 있었다. <a href="/wiki/%EC%95%84%EC%9D%B4%EC%9E%91_%EB%89%B4%ED%84%B4" title="아이작 뉴턴">아이작 뉴턴</a>은 수리 모형으로 <a href="/wiki/%EB%89%B4%ED%84%B4_%EC%9A%B4%EB%8F%99_%EB%B2%95%EC%B9%99" title="뉴턴 운동 법칙">뉴턴 운동 법칙</a>을 완벽히 기술하여 고전 역학을 완성하였다.</p>
<p>하지만 19세기 말, 20세기 초에 들어오면서 고전역학체계로 해석할 수 없는 현상들이 생겼는데 양자역학이 고전역학보다 적용범위가 더 넓었기 때문이었다.</p>
<p>더나아가 고전역학, 양자역학 다음으로 상대론적양자역학이 등장함에 따라 모든 세계를 다룰 수 있게 되었다.</p>

<p>뉴턴 운동 법칙은 물체의 운동을 다루는 세 개의 <a href="/wiki/%EB%AC%BC%EB%A6%AC_%EB%B2%95%EC%B9%99" title="물리 법칙">물리 법칙</a>으로, 고전 역학의 바탕을 이루고 있다.<sup class="reference" id="cite_ref-10"><a href="#cite_note-10">[10]</a></sup> 뉴턴의 운동 법칙은 다음과 같다.</p>
<ul>
<li><a href="/wiki/%EA%B4%80%EC%84%B1" title="관성">관성</a>의 법칙 : 물체의 <a href="/wiki/%EC%A7%88%EB%9F%89_%EC%A4%91%EC%8B%AC" title="질량 중심">질량 중심</a>은 외부 힘이 작용하지 않는 한 일정한 <a href="/wiki/%EC%86%8D%EB%8F%84" title="속도">속도</a>로 움직인다.</li>
<li><a href="/wiki/%EA%B0%80%EC%86%8D%EB%8F%84" title="가속도">가속도</a>의 법칙: 물체의 <a href="/wiki/%EC%9A%B4%EB%8F%99%EB%9F%89" title="운동량">운동량</a>의 시간에 따른 변화율은 그 물체에 작용하는 <a href="/wiki/%EC%95%8C%EC%A7%9C%ED%9E%98" title="알짜힘">알짜힘</a>과 (크기와 방향에 있어서) 같다.</li>
<li>작용과 반작용의 법칙: 물체 A가 다른 물체 B에 힘을 가하면, 물체 B는 물체 A에 크기는 같고 방향은 반대인 힘을 동시에 가한다.</li>
</ul>
<p>고전 역학에서 다루는 <a class="mw-redirect" href="/wiki/%EC%9B%90_%EC%9A%B4%EB%8F%99" title="원 운동">원 운동</a>, <a class="new" href="/w/index.php?title=%ED%83%80%EC%9B%90_%EC%9A%B4%EB%8F%99&amp;action=edit&amp;redlink=1" title="타원 운동 (없는 문서)">타원 운동</a>, <a href="/wiki/%ED%8F%AC%EB%AC%BC%EC%84%A0_%EC%9A%B4%EB%8F%99" title="포물선 운동">포물선 운동</a>, <a class="new" href="/w/index.php?title=%EC%8C%8D%EA%B3%A1%EC%84%A0_%EC%9A%B4%EB%8F%99&amp;action=edit&amp;redlink=1" title="쌍곡선 운동 (없는 문서)">쌍곡선 운동</a>, <a class="mw-redirect" href="/wiki/%EC%9E%90%EC%9C%A0%EB%82%99%ED%95%98" title="자유낙하">자유낙하</a>, <a class="mw-redirect" href="/wiki/%EC%A7%84%EB%8F%99%EC%9E%90" title="진동자">진동자</a> 등 여러 가지 종류의 운동은 결국 뉴턴의 운동 법칙으로 귀결된다.<sup class="reference" id="cite_ref-11"><a href="#cite_note-11">[11]</a></sup></p>
<h3><span class="mw-headline" id=".EB.8F.99.EC.97.AD.ED.95.99">동역학</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%97%AD%ED%95%99_(%EB%AC%BC%EB%A6%AC%ED%95%99)&amp;action=edit&amp;section=6" title="부분 편집: 동역학">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>동역학(Dynamics)은 4대역학(유체역학, 열역학, 고체역학, 동역학)중 하나로 물체 사이에 작용하는 힘과 물체의 운동과의 관계를 연구하는 학문이다. 동역학은 고등학교 물리1, 물리2의 역학의 상위호환이며 일반적으로 대학교(한국)에서는 동역학은 물리학보다는 기계공학에서 많이 쓰인다. 현대의 기계는 매우 복잡한 메커니즘으로 돌아가는데, 그 구동 과정에서 필연적으로 진동이 발생하게 되는데 이를 분석하고 제어하기 위한 동역학과 그의 심화 버전인 기계진동학이 있다.</p>

<h4><span class="mw-headline" id=".EC.9A.B4.EB.8F.99.ED.95.99">운동학</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%97%AD%ED%95%99_(%EB%AC%BC%EB%A6%AC%ED%95%99)&amp;action=edit&amp;section=8" title="부분 편집: 운동학">편집</a><span class="mw-editsection-bracket">]</span></span></h4>
<div class="rellink relarticle mainarticle"><img alt="" data-file-height="128" data-file-width="128" height="16" src="//upload.wikimedia.org/wikipedia/commons/thumb/e/ec/Crystal_Clear_app_xmag.svg/16px-Crystal_Clear_app_xmag.svg.png" srcset="//upload.wikimedia.org/wikipedia/commons/thumb/e/ec/Crystal_Clear_app_xmag.svg/24px-Crystal_Clear_app_xmag.svg.png 1.5x, //upload.wikimedia.org/wikipedia/commons/thumb/e/ec/Crystal_Clear_app_xmag.svg/32px-Crystal_Clear_app_xmag.svg.png 2x" width="16"/> 이 부분의 본문은 <a href="/wiki/%EC%9A%B4%EB%8F%99%ED%95%99" title="운동학">운동학</a>입니다.</div>
<p><a href="/wiki/%EC%9A%B4%EB%8F%99%ED%95%99" title="운동학">운동학</a>은 <a href="/wiki/%EC%9E%85%EC%9E%90" title="입자">입자</a>나 <a href="/wiki/%EB%AC%BC%EC%B2%B4" title="물체">물체</a> 또는 다수의 물체가 모여 이루어진 <a href="/wiki/%EA%B3%84_(%EB%AC%BC%EB%A6%AC%ED%95%99)" title="계 (물리학)">계</a>의 <a href="/wiki/%EC%9A%B4%EB%8F%99_(%EB%AC%BC%EB%A6%AC%ED%95%99)" title="운동 (물리학)">운동</a>을 다루는 <a class="mw-redirect" href="/wiki/%EA%B3%A0%EC%A0%84_%EC%97%AD%ED%95%99" title="고전 역학">고전 역학</a>의 하위 학문이다. 운동학은 운동의 양상만을 다루고 운동이 일어나는 원인에 대해서는 고려하지 않는다.<sup class="reference" id="cite_ref-12"><a href="#cite_note-12">[12]</a></sup><sup class="reference" id="cite_ref-13"><a href="#cite_note-13">[13]</a></sup><sup class="reference" id="cite_ref-14"><a href="#cite_note-14">[14]</a></sup></p>
<p>운동학은 <a href="/wiki/%EB%8F%99%EC%97%AD%ED%95%99" title="동역학">동역학</a>의 하위 분야로 다뤄지기도 한다. 이 경우엔 흔히 운동 기하학이라고도 불린다.<sup class="reference" id="cite_ref-15"><a href="#cite_note-15">[15]</a></sup> 운동학은 물체의 이동을 입자의 궤적이나 선 등으로 나타낸다. 물체의 이동에 따라 측정되는 <a href="/wiki/%EC%86%8D%EB%A0%A5" title="속력">속력</a>이나 <a href="/wiki/%EA%B0%80%EC%86%8D%EB%8F%84" title="가속도">가속도</a>등도 기하학적으로 나타낼 수 있다. 운동학은 <a href="/wiki/%EC%B2%9C%EC%B2%B4%EB%AC%BC%EB%A6%AC%ED%95%99" title="천체물리학">천체물리학</a>에서 <a href="/wiki/%EC%B2%9C%EC%B2%B4" title="천체">천체</a>의 움직임을 나타내는 것부터, <a href="/wiki/%EA%B8%B0%EA%B3%84%EA%B3%B5%ED%95%99" title="기계공학">기계공학</a>에서 부품의 움직임을 예측하거나 <a href="/wiki/%EB%A1%9C%EB%B4%87%EA%B3%B5%ED%95%99" title="로봇공학">로봇공학</a>에서 로봇팔의 움직임을 설계하는 일, 그리고 <a class="new" href="/w/index.php?title=%EC%83%9D%EC%B2%B4_%EC%97%AD%ED%95%99&amp;action=edit&amp;redlink=1" title="생체 역학 (없는 문서)">생체 역학</a>에서 <a class="mw-redirect" href="/wiki/%EA%B3%A8%EA%B2%A9" title="골격">골격</a>의 움직임을 설명하는 등 다양한 분야에서 쓰인다.<sup class="reference" id="cite_ref-16"><a href="#cite_note-16">[16]</a></sup></p>

<p>1788년 프랑스의 수학자이자 과학자인 <a href="/wiki/%EC%A1%B0%EC%A0%9C%ED%94%84%EB%A3%A8%EC%9D%B4_%EB%9D%BC%EA%B7%B8%EB%9E%91%EC%A3%BC" title="조제프루이 라그랑주">조제프루이 라그랑주</a>는 《해석역학》을 발표하여<sup class="reference" id="cite_ref-17"><a href="#cite_note-17">[17]</a></sup> 뉴턴 운동 법칙의 수리 모형을 발전시켜 <a href="/wiki/%EB%9D%BC%EA%B7%B8%EB%9E%91%EC%A3%BC_%EC%97%AD%ED%95%99" title="라그랑주 역학">라그랑주 역학</a>을 수립하였으며, 1833년 아일랜드의 <a href="/wiki/%EC%9C%8C%EB%A6%AC%EC%97%84_%EB%A1%9C%EC%96%B8_%ED%95%B4%EB%B0%80%ED%84%B4" title="윌리엄 로언 해밀턴">윌리엄 로언 해밀턴</a>은 라르랑주 역학을 토대로 고전 역학의 계를 운동량으로 이루어진 <a href="/wiki/%EC%9C%84%EC%83%81_%EA%B3%B5%EA%B0%84_(%EB%AC%BC%EB%A6%AC%ED%95%99)" title="위상 공간 (물리학)">위상 공간</a>으로 나타내는 해석 역학인 <a href="/wiki/%ED%95%B4%EB%B0%80%ED%84%B4_%EC%97%AD%ED%95%99" title="해밀턴 역학">해밀턴 역학</a>을 정리하였다.<sup class="reference" id="cite_ref-18"><a href="#cite_note-18">[18]</a></sup></p>

<h2><span class="mw-headline" id=".EC.83.81.EB.8C.80.EB.A1.A0.EC.A0.81_.EC.97.AD.ED.95.99">상대론적 역학</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%97%AD%ED%95%99_(%EB%AC%BC%EB%A6%AC%ED%95%99)&amp;action=edit&amp;section=15" title="부분 편집: 상대론적 역학">편집</a><span class="mw-editsection-bracket">]</span></span></h2>

1919년 5월 29일 <a class="mw-redirect" href="/wiki/%EC%95%84%EC%84%9C_%EC%97%90%EB%94%A9%ED%84%B4" title="아서 에딩턴">아서 에딩턴</a>이 관측한 개기일식의 사진. <a class="mw-redirect" href="/wiki/%EC%A4%91%EB%A0%A5_%EB%A0%8C%EC%A6%88" title="중력 렌즈">중력 렌즈</a> 현상이 관찰되어 <a class="mw-redirect" href="/wiki/%EC%9D%BC%EB%B0%98_%EC%83%81%EB%8C%80%EC%84%B1_%EC%9D%B4%EB%A1%A0" title="일반 상대성 이론">일반 상대성 이론</a>이 입증되었다. 사진에서 태양 주위에 보이는 띠 모양의 빛들이 중력 렌즈 현상이다.</div>

<p>고전 역학은 원자보다 크고 빛의 속력보다 매우 느린 물체의 운동에 대해서는 정확히 예측할 수 있으나, 빛이 속력과 가까운 물체의 운동에 대해서는 통용되지 않는다. 예를 들어 수백만 <a href="/wiki/%EB%B3%BC%ED%8A%B8" title="볼트">볼트</a>의 <a class="mw-redirect" href="/wiki/%EC%A0%84%EC%9C%84%EC%B0%A8" title="전위차">전위차</a>에 의해 생성된 <a href="/wiki/%EC%A0%84%EC%9E%90%EA%B8%B0%EC%9E%A5" title="전자기장">전자기장</a>에 놓인 <a href="/wiki/%EC%A0%84%EC%9E%90" title="전자">전자</a>의 운동은 고전 역학이 예측하는 바와 같이 무한정 힘에 비례하여 속력이 커지지는 않는다. 고전 역학의 운동 이론을 적용하면 전자의 운동 에너지가 네 배가 될 때 전자의 속력은 빛의 속력의 1.98 배에 이르러야 하나 실제로는 0.99 배로서 여전히 빛의 속력보다 작다. <a href="/wiki/%EC%95%8C%EB%B2%A0%EB%A5%B4%ED%8A%B8_%EC%95%84%EC%9D%B8%EC%8A%88%ED%83%80%EC%9D%B8" title="알베르트 아인슈타인">알베르트 아인슈타인</a>은 <a class="mw-redirect" href="/wiki/%EC%83%81%EB%8C%80%EC%84%B1_%EC%9D%B4%EB%A1%A0" title="상대성 이론">상대성 이론</a>을 수립하여 <a href="/wiki/%EC%8B%9C%EA%B3%B5%EA%B0%84" title="시공간">시공간</a> 자체가 절대적이지 않다는 설명으로 이러한 모순을 해결하였다. 이로써 현대 물리학은 <span class="mwe-math-element"><span class="mwe-math-mathml-inline mwe-math-mathml-a11y" style="display: none;"><math xmlns="http://www.w3.org/1998/Math/MathML">

<h3><span class="mw-headline" id=".ED.8A.B9.EC.88.98_.EC.83.81.EB.8C.80.EC.84.B1_.EC.9D.B4.EB.A1.A0">특수 상대성 이론</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%97%AD%ED%95%99_(%EB%AC%BC%EB%A6%AC%ED%95%99)&amp;action=edit&amp;section=16" title="부분 편집: 특수 상대성 이론">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<div class="rellink relarticle mainarticle"><img alt="" data-file-height="128" data-file-width="128" height="16" src="//upload.wikimedia.org/wikipedia/commons/thumb/e/ec/Crystal_Clear_app_xmag.svg/16px-Crystal_Clear_app_xmag.svg.png" srcset="//upload.wikimedia.org/wikipedia/commons/thumb/e/ec/Crystal_Clear_app_xmag.svg/24px-Crystal_Clear_app_xmag.svg.png 1.5x, //upload.wikimedia.org/wikipedia/commons/thumb/e/ec/Crystal_Clear_app_xmag.svg/32px-Crystal_Clear_app_xmag.svg.png 2x" width="16"/> 이 부분의 본문은 <a class="mw-redirect" href="/wiki/%ED%8A%B9%EC%88%98_%EC%83%81%EB%8C%80%EC%84%B1_%EC%9D%B4%EB%A1%A0" title="특수 상대성 이론">특수 상대성 이론</a>입니다.</div>
<p><a href="/wiki/1905%EB%85%84" title="1905년">1905년</a> 아인슈타인은 〈움직이는 물체의 전기역학에 대하여〉(독일어: Zur Elektrodynamik bewegter Körper)를 발표하였다.<sup class="reference" id="cite_ref-20"><a href="#cite_note-20">[20]</a></sup> 아인슈타인은 이 논문에서 서로 다른 속력을 갖는 물체와 관찰자 사이의 상대 운동에 대한 새로운 이론을 제시하였다. 이 논문에서는 <a href="/wiki/%EC%8B%9C%EA%B0%84" title="시간">시간</a>과 <a href="/wiki/%EA%B3%B5%EA%B0%84" title="공간">공간</a>이 기존의 고전 역학에서 다루는 것과 같은 정적인 독립 변수가 아니라 서로 결합된 <a href="/wiki/4%EC%B0%A8%EC%9B%90" title="4차원">4차원</a> <a class="mw-disambig" href="/wiki/%EB%B2%A1%ED%84%B0" title="벡터">벡터</a>인 <a href="/wiki/%EC%8B%9C%EA%B3%B5%EA%B0%84" title="시공간">시공간</a>으로서 다루어진다. 또한, <a href="/wiki/%EB%B9%9B" title="빛">빛</a>과 같은 질량이 없는 입자를 내보내는 물체와 관찰자가 서로 어떠한 상대 운동을 하던지 <a class="mw-redirect" href="/wiki/%EB%B9%9B%EC%9D%98_%EC%86%8D%EB%8F%84" title="빛의 속도">빛의 속력</a>은 이와 무관하게 늘 일정함을 밝혔다. 이 두 가지 사항에 따라 서로 다른 <a class="mw-redirect" href="/wiki/%EA%B4%80%EC%84%B1%EA%B3%84" title="관성계">관성계</a>에 놓인 물체와 관찰자 사이의 거리,시간 등의 측정값은 상대 운동의 속력에 따라 변할 수 있다는 결론에 도달하게 된다.</p>
<p>특수 상대성 이론의 예측은 이후 정밀한 실험을 통해 입증되었다.<sup class="reference" id="cite_ref-21"><a href="#cite_note-21">[21]</a></sup></p>
<h3><span class="mw-headline" id=".EC.9D.BC.EB.B0.98_.EC.83.81.EB.8C.80.EC.84.B1_.EC.9D.B4.EB.A1.A0">일반 상대성 이론</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%97%AD%ED%95%99_(%EB%AC%BC%EB%A6%AC%ED%95%99)&amp;action=edit&amp;section=17" title="부분 편집: 일반 상대성 이론">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<div class="rellink relarticle mainarticle"><img alt="" data-file-height="128" data-file-width="128" height="16" src="//upload.wikimedia.org/wikipedia/commons/thumb/e/ec/Crystal_Clear_app_xmag.svg/16px-Crystal_Clear_app_xmag.svg.png" srcset="//upload.wikimedia.org/wikipedia/commons/thumb/e/ec/Crystal_Clear_app_xmag.svg/24px-Crystal_Clear_app_xmag.svg.png 1.5x, //upload.wikimedia.org/wikipedia/commons/thumb/e/ec/Crystal_Clear_app_xmag.svg/32px-Crystal_Clear_app_xmag.svg.png 2x" width="16"/> 이 부분의 본문은 <a class="mw-redirect" href="/wiki/%EC%9D%BC%EB%B0%98_%EC%83%81%EB%8C%80%EC%84%B1_%EC%9D%B4%EB%A1%A0" title="일반 상대성 이론">일반 상대성 이론</a>입니다.</div>
<p>아인슈타인이 1915년 발표한 일반 상대성 이론은 특수 상대성 이론에 <a href="/wiki/%EC%A4%91%EB%A0%A5" title="중력">중력</a>을 포함시켜 일반화 한 것이다. 일반 상대성 이론에서는 중력을 <a href="/wiki/%EC%8B%9C%EA%B3%B5%EA%B0%84" title="시공간">시공간</a>의 <a href="/wiki/%EA%B3%A1%EB%A5%A0" title="곡률">곡률</a>로 해석한다.<sup class="reference" id="cite_ref-22"><a href="#cite_note-22">[22]</a></sup> 즉, 관찰자의 입장에서 보면 시공간 자체가 늘어나거나 줄어드는 것 처럼 보이게 된다. 아인슈타인은 이러한 이론을 바탕으로 <a class="mw-redirect" href="/wiki/%EC%A4%91%EB%A0%A5_%EB%A0%8C%EC%A6%88" title="중력 렌즈">중력 렌즈</a> 현상을 예측하였고, 1919년 5월 29일 <a class="mw-redirect" href="/wiki/%EC%95%84%EC%84%9C_%EC%97%90%EB%94%A9%ED%84%B4" title="아서 에딩턴">아서 에딩턴</a>이 개기일식을 관측하면서 이를 확인하였다. 아서 에딩턴은 이듬해인 1920년 자신의 관찰 결과가 일반 상대성 이론에 부합한다는 논문을 발표하였다.<sup class="reference" id="cite_ref-23"><a href="#cite_note-23">[23]</a></sup></p>
<h2><span class="mw-headline" id=".EC.97.B4.EC.97.AD.ED.95.99">열역학</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%97%AD%ED%95%99_(%EB%AC%BC%EB%A6%AC%ED%95%99)&amp;action=edit&amp;section=18" title="부분 편집: 열역학">편집</a><span class="mw-editsection-bracket">]</span></span></h2>

</div>
<p><a href="/wiki/%EC%97%B4%EC%97%AD%ED%95%99" title="열역학">열역학</a>은 거시적인 물체들 사이의 <a href="/wiki/%EC%97%B4" title="열">열</a> 에너지 전달을 <a href="/wiki/%EC%98%A8%EB%8F%84" title="온도">온도</a>와 관련지어 설명하는 학문이다. 열역학에서는 <a href="/wiki/%EB%AC%BC%EC%A7%88" title="물질">물질</a>을 구성하는 <a href="/wiki/%EB%B6%84%EC%9E%90" title="분자">분자</a>의 구조와 같은 미시적인 것에는 관심을 두지 않는 대신 거시적으로 <a href="/wiki/%EA%B3%84_(%EB%AC%BC%EB%A6%AC%ED%95%99)" title="계 (물리학)">계</a>의 상태를 온도, <a href="/wiki/%EB%B6%80%ED%94%BC" title="부피">부피</a>, <a href="/wiki/%EC%95%95%EB%A0%A5" title="압력">압력</a> 등의 변수로서 기술한다.<sup class="reference" id="cite_ref-24"><a href="#cite_note-24">[24]</a></sup></p>
<p>열역학에서 다루는 계를 <a href="/wiki/%EC%97%B4%EC%97%AD%ED%95%99_%EA%B3%84" title="열역학 계">열역학 계</a>라고 한다. 열역학 계는 우주 전체 가운데 지금 고찰하고 있는 현상에 관여하는 계만을 구분짓는 개념이다. 예를 들어 자동차 엔진의 운동을 생각할 때 엔진의 운동에 관여하는 에너지의 공급과 사용, 그리고 그에 따른 피스톤의 운동과 같은 것 만을 고려하여 하나의 계로 생각할 수 있을 것이다. 이렇게 설정된 계 이외의 나머지 우주는 환경또는 외부라고 한다.<sup class="reference" id="cite_ref-.EA.B9.80.EC.9E.AC.EC.9A.A9_25-0"><a href="#cite_note-.EA.B9.80.EC.9E.AC.EC.9A.A9-25">[25]</a></sup> 일반적으로 열역학 계 안에는 무수히 많은 입자들을 포함되므로 계 전체의 온도나 부피, 압력 등을 고려할 때, 입자 하나 하나의 상태를 관찰하는 것은 불가능하다. 따라서 열역학은 계 내부에 포함되는 입자들 총체에 대해 <a class="mw-redirect" href="/wiki/%ED%86%B5%EA%B3%84" title="통계">통계</a>적인 방법으로 기술하게 된다. 계 전체의 상태를 나타내는 변수로는 앞서 말한 온도, 부피, 압력 외에 <a href="/wiki/%EC%97%94%ED%8A%B8%EB%A1%9C%ED%94%BC" title="엔트로피">엔트로피</a>가 있다.<sup class="reference" id="cite_ref-26"><a href="#cite_note-26">[26]</a></sup> 이렇게 계의 상태를 통계적으로 살피는 역학을 <a class="mw-redirect" href="/wiki/%ED%86%B5%EA%B3%84_%EC%97%AD%ED%95%99" title="통계 역학">통계 역학</a>이라고 하는데, 열역학은 통계 역학의 대표적인 분야 가운데 하나이다.</p>
<p>열역학 계는 환경과 에너지와 물질을 주고 받는 조건에 따라 열린계, 닫힌계, 고립계로 나뉜다. 열린계는 물질과 에너지 모두를 환경과 주고 받을 수 있는 계이고, 닫힌계는 에너지와 일은 전달되지만 물질은 전달되지 않는 계이며, 고립계는 에너지와 물질 모두 전달되지 않는 계이다.<sup class="reference" id="cite_ref-27"><a href="#cite_note-27">[27]</a></sup> 일반적으로, 외부에서 계로 일과 에너지가 전달되면 계 내부의 에너지가 증가한다. 닫힌 계에서는 내부 물질의 에너지가 커져 온도가 올라가게 되고 열린계에서는 온도가 높은 상태에서 낮은 상태로 에너지와 물질이 이동하게 된다.<sup class="reference" id="cite_ref-.EA.B9.80.EC.9E.AC.EC.9A.A9_25-1"><a href="#cite_note-.EA.B9.80.EC.9E.AC.EC.9A.A9-25">[25]</a></sup></p>
<p>열역학 계 내에서 온도에 따른 물질과 에너지의 이동은 <a href="/wiki/%EC%97%B4%EC%97%AD%ED%95%99_%EB%B2%95%EC%B9%99" title="열역학 법칙">열역학 법칙</a>을 따른다. 열역학 법칙은 계 내에서 에너지는 일정하여 새롭게 생성되거나 파괴되지 않고 변환만이 가능하다는 것과, 계는 엔트로피가 최대인 상태일 때 열적 평형상태에 다다르게 된다는 것이다.<sup class="reference" id="cite_ref-28"><a href="#cite_note-28">[28]</a></sup> 흔히 무질서도라고 이해되는 엔트로피는 계의 상태를 나타내는 상태 <a href="/wiki/%ED%95%A8%EC%88%98" title="함수">함수</a>로서 계 내부의 모든 입자가 균등한 상태에 이를 때 최댓값을 갖는다. 예를 들어, 외부와 에너지와 물질이 차단된 고립계에 질량은 같고 온도가 0℃와 100℃로 다른 물을 놓아둔다면 이 두 물이 가장 균등하게 섞이는 상태, 즉 엔트로피가 최대인 50℃가 된다는 것이다. 우주 전체 역시 하나의 고립계로 생각할 수 있으므로 엔트로피가 최대인 상태로 진행하게 된다. 이 때문에, 열역학에 엔트로피 개념을 도입한 <a href="/wiki/%EB%A3%A8%EB%8F%8C%ED%94%84_%ED%81%B4%EB%9D%BC%EC%9A%B0%EC%A7%80%EC%9A%B0%EC%8A%A4" title="루돌프 클라우지우스">루돌프 클라우지우스</a>는 "우주의 에너지는 일정하나 우주의 엔트로피는 최댓값에 도달하려고 한다"라고 표현하였다.<sup class="reference" id="cite_ref-29"><a href="#cite_note-29">[29]</a></sup><sup class="reference" id="cite_ref-30"><a href="#cite_note-30">[주해 1]</a></sup></p>

<p>양자 역학은 <a href="/wiki/%EC%9B%90%EC%9E%90" title="원자">원자</a>이하의 물질들 사이에서 일어나는 상호 작용을 설명하는 역학 이론이다. 원자나 그보다 작은 여러 입자들의 움직임은 거시 세계를 대상으로 하는 고전 역학이 제시하는 물리 법칙이 적용되지 않는다. 20세기 이후 물리학자들은 이런 미시 세계에서 일어나는 상호 작용들을 설명하기 위한 이론을 개발하고 실험을 통해 입증하여 왔다. <a href="/wiki/%EC%96%91%EC%9E%90_(%EC%97%90%EB%84%88%EC%A7%80)" title="양자 (에너지)">양자</a>는 더 이상 나눌 수 없는 물리량의 기본 단위를 뜻한다. 고전 역학에서는 두 물체의 상호작용에 관여하는 에너지를 연속적이며 무한히 작게 나눌 수 있는 것으로 보았지만, 실제 자연에서 물질의 상호 작용에 관여하는 에너지는 일정 크기의 양자를 기준으로 그의 정수배로만 작용하게 된다.<sup class="reference" id="cite_ref-31"><a href="#cite_note-31">[30]</a></sup></p>
<p>고전 역학에서도 물질이나 전하 같은 입자로 취급할 수 있는 물리량은 양자로서 설명할 수 있다. 그러나, 고전 역학에서는 전자기 복사와 같은 에너지의 흐름은 연속되는 양으로서만 취급한다. 1900년 독일의 물리학자 <a href="/wiki/%EB%A7%89%EC%8A%A4_%ED%94%8C%EB%9E%91%ED%81%AC" title="막스 플랑크">막스 플랑크</a>는 <a class="mw-redirect" href="/wiki/%ED%9D%91%EC%B2%B4_%EB%B3%B5%EC%82%AC" title="흑체 복사">흑체 복사</a> 실험에서 온도가 높을 때 전자기파의 강도 분포가 고전 역학의 예측과 다른 이유를 에너지가 양자화 되어 전달되기 때문이라는 것을 밝혔다. 플랑크는 흑체 복사의 측정 자료와 일치하는 수리 모형을 제시하기 위하여 방출된 빛의 에너지가 훗날 <a href="/wiki/%ED%94%8C%EB%9E%91%ED%81%AC_%EC%83%81%EC%88%98" title="플랑크 상수">플랑크 상수</a>로 불리게 된 특정한 상수와 진동수를 곱한 값의 정수배로만 주어진다고 가정하였고, 이러한 가정은 여러 차례의 실험을 통하여 잘 들어맞는 것이 확인되었다.<sup class="reference" id="cite_ref-32"><a href="#cite_note-32">[31]</a></sup></p>
<p>이렇게 에너지의 전달을 설명하기 위해 시작된 양자 역학은 <a href="/wiki/%EC%9B%90%EC%9E%90" title="원자">원자</a>에 대한 연구가 진척됨에 따라 원자 이하의 물질들이 보이는 여러 현상을 설명하는 학문으로서 자리잡게 되었다. 1905년 아인슈타인은 광자가 양자화되어 있다는 것을 밝혔고<sup class="reference" id="cite_ref-33"><a href="#cite_note-33">[32]</a></sup>, 1913년 <a href="/wiki/%EB%8B%90%EC%8A%A4_%EB%B3%B4%EC%96%B4" title="닐스 보어">닐스 보어</a>는 수소의 방출 스펙트럼을 <a class="mw-disambig" href="/wiki/%EC%96%91%EC%9E%90%ED%99%94" title="양자화">양자화</a>된 전자의 에너지 준위로서 설명하였으며<sup class="reference" id="cite_ref-34"><a href="#cite_note-34">[33]</a></sup>, <a href="/wiki/%EB%A3%A8%EC%9D%B4_%EB%93%9C_%EB%B8%8C%EB%A1%9C%EC%9D%B4" title="루이 드 브로이">루이 드 브로이</a>은 운동하는 물체역시 파동으로서 서술할 수 있다는 것을 밝히고 <a href="/wiki/%EB%AC%BC%EC%A7%88%ED%8C%8C" title="물질파">물질파</a>를 도입하였다.<sup class="reference" id="cite_ref-35"><a href="#cite_note-35">[34]</a></sup></p>

<p>1927년 <a href="/wiki/%EB%B2%A0%EB%A5%B4%EB%84%88_%ED%95%98%EC%9D%B4%EC%A0%A0%EB%B2%A0%EB%A5%B4%ED%81%AC" title="베르너 하이젠베르크">베르너 하이젠베르크</a>는 <a href="/wiki/%EB%B6%88%ED%99%95%EC%A0%95%EC%84%B1_%EC%9B%90%EB%A6%AC" title="불확정성 원리">불확정성 원리</a>를 발표하였다. 운동하는 미시적 입자의 위치와 운동량을 동시에 측정할 수 없다는 불확정성 원리로 인해 원자 이하의 입자에 대한 관찰은 확률에 의할 수 밖에 없다는 것이 밝혀졌다.<sup class="reference" id="cite_ref-36"><a href="#cite_note-36">[35]</a></sup> 원칙적으로 위치의 불확정성과 운동량의 불확정성의 곱은 <a href="/wiki/%ED%94%8C%EB%9E%91%ED%81%AC_%EC%83%81%EC%88%98" title="플랑크 상수">플랑크 상수</a>보다 작아질 수 없다. 이에 따라 <a href="/wiki/%EC%97%90%EB%A5%B4%EB%B9%88_%EC%8A%88%EB%A2%B0%EB%94%A9%EA%B1%B0" title="에르빈 슈뢰딩거">에르빈 슈뢰딩거</a>는 원자핵에 묶인 전자의 움직임을 입자로서 설명하는 것을 버리고, 파동으로서 설명하는 <a href="/wiki/%EC%8A%88%EB%A2%B0%EB%94%A9%EA%B1%B0_%EB%B0%A9%EC%A0%95%EC%8B%9D" title="슈뢰딩거 방정식">슈뢰딩거 방정식</a>을 수립하였다. 슈뢰딩거 방정식은 하나의 전자가 원자핵 주변에서 점 또는 미소 공간에 존재할 확률을 제시한다.<sup class="reference" id="cite_ref-37"><a href="#cite_note-37">[36]</a></sup> 닐스 보어는 관찰자가 어떠한 방식으로 실험을 하는 지에 따라 물체가 입자 또는 파동의 성질을 보일 수 있다는 <a class="mw-redirect" href="/wiki/%EC%83%81%EB%B3%B4%EC%84%B1_%EC%9B%90%EB%A6%AC" title="상보성 원리">상보성 원리</a>로서 불확정성 원리를 설명하였다.<sup class="reference" id="cite_ref-38"><a href="#cite_note-38">[37]</a></sup></p>
<p>불확정성의 원리와 상보성 원리가 과학적 사실로 받아들여지게 되자, 과학자들은 이것이 우주의 실제와 어떠한 관련이 있는 지를 논의하였다. <a href="/wiki/%EC%BD%94%ED%8E%9C%ED%95%98%EA%B2%90_%ED%95%B4%EC%84%9D" title="코펜하겐 해석">코펜하겐 해석</a>은 <a href="/wiki/%EC%8A%88%EB%A2%B0%EB%94%A9%EA%B1%B0%EC%9D%98_%EA%B3%A0%EC%96%91%EC%9D%B4" title="슈뢰딩거의 고양이">슈뢰딩거의 고양이</a>와 같이 결정되어 있지 않는 사건은 관측자의 측정과 함께 하나의 사건으로 결정되며 다른 가능한 상태들은 더 이상 일어날 수 없는 일이 된다고 설명한다. 오늘날 양자 역학과 우주의 실제에 대해서는 일반적으로 코펜하겐 해석이 받아들여지고 있으나 <a href="/wiki/%EB%8B%A4%EC%84%B8%EA%B3%84_%ED%95%B4%EC%84%9D" title="다세계 해석">다세계 해석</a>과 같은 다른 해석들도 존재한다.<sup class="reference" id="cite_ref-39"><a href="#cite_note-39">[38]</a></sup></p>
