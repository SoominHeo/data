<p>《<b>연개소문</b>》(淵蓋蘇文)은 <a class="mw-disambig" href="/wiki/SBS" title="SBS">SBS</a>에서 방영하였고, 단재 <a href="/wiki/%EC%8B%A0%EC%B1%84%ED%98%B8" title="신채호">신채호</a>의 <a href="/wiki/%EC%A1%B0%EC%84%A0%EC%83%81%EA%B3%A0%EC%82%AC" title="조선상고사">조선상고사</a>를 근거로 한 <a href="/wiki/2006%EB%85%84" title="2006년">2006년</a> <a href="/wiki/7%EC%9B%94_8%EC%9D%BC" title="7월 8일">7월 8일</a> ~ <a href="/wiki/2007%EB%85%84" title="2007년">2007년</a> <a href="/wiki/6%EC%9B%94_17%EC%9D%BC" title="6월 17일">6월 17일</a>까지 총 100부작 <a class="mw-redirect" href="/wiki/%EB%8C%80%ED%95%98%EB%93%9C%EB%9D%BC%EB%A7%88" title="대하드라마">대하드라마</a>이다. 총제작비 400억 원의 <a href="/wiki/%EB%8C%80%ED%95%9C%EB%AF%BC%EA%B5%AD" title="대한민국">대한민국</a> 역사상 최대의 제작비를 들인 드라마. <a href="/wiki/%EA%B3%A0%EA%B5%AC%EB%A0%A4" title="고구려">고구려</a> 후기부터 고구려의 멸망까지 <a href="/wiki/%EC%97%B0%EA%B0%9C%EC%86%8C%EB%AC%B8" title="연개소문">연개소문</a>의 생애를 다루었다. 극본은 <a class="mw-disambig" href="/wiki/%EC%9D%B4%ED%99%98%EA%B2%BD" title="이환경">이환경</a> 작가 , 연출은 <a href="/wiki/%EC%9D%B4%EC%A2%85%ED%95%9C" title="이종한">이종한</a> 감독이다. 한편, 작가 이환경씨는 2003년 <a href="/wiki/%EC%95%BC%EC%9D%B8%EC%8B%9C%EB%8C%80" title="야인시대">야인시대</a> 이후 3년 만에 SBS로 돌아왔다. 아울러, <a href="/wiki/2006%EB%85%84" title="2006년">2006년</a> <a href="/wiki/9%EC%9B%94_30%EC%9D%BC" title="9월 30일">9월 30일</a>에는 추석 특집 프로그램으로 결방되었고 <a href="/wiki/2006%EB%85%84" title="2006년">2006년</a> <a href="/wiki/10%EC%9B%94_7%EC%9D%BC" title="10월 7일">10월 7일</a>에는 10분 앞당긴 8시 35분에 방영(25,27회)되었으며<sup class="reference" id="cite_ref-1"><a href="#cite_note-1">[1]</a></sup> 이 날 KBS 2TV <a href="/wiki/%EC%86%8C%EB%AC%B8%EB%82%9C_%EC%B9%A0%EA%B3%B5%EC%A3%BC" title="소문난 칠공주">소문난 칠공주</a>는 7시 55분부터 2회 연속(55~56회) 편성된 바 있었고 다음 날에는 축구 중계 편성으로 결방됐는데 <a href="/wiki/2006%EB%85%84" title="2006년">2006년</a> <a href="/wiki/9%EC%9B%94_2%EC%9D%BC" title="9월 2일">9월 2일</a>에 이어 두 번째(7시 50분부터 축구 중계 편성)로 결방되었으며 <a href="/wiki/2006%EB%85%84" title="2006년">2006년</a> <a href="/wiki/9%EC%9B%94_3%EC%9D%BC" title="9월 3일">9월 3일</a> 45~46회 연속 방송됐다.</p>
