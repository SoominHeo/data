<p><b>연방군 장군총감</b>(<span style="font-size: smaller;"><a href="/wiki/%EB%8F%85%EC%9D%BC%EC%96%B4" title="독일어">독일어</a>:</span> <span lang="de" xml:lang="de">Generalinspekteur der Bundeswehr; <b>GenInspBw</b></span> <small>게네랄인스펙토이어 데어 분데스베어<sup>[<a href="/wiki/%EC%9C%84%ED%82%A4%EB%B0%B1%EA%B3%BC:%EB%8F%85%EC%9D%BC%EC%96%B4%EC%9D%98_%ED%95%9C%EA%B8%80_%ED%91%9C%EA%B8%B0" title="위키백과:독일어의 한글 표기">*</a>]</sup></small>)은 <a href="/wiki/%EB%8F%85%EC%9D%BC_%EC%97%B0%EB%B0%A9%EA%B5%B0" title="독일 연방군">독일 연방군</a>의 최선임 장교이다. 그 명칭은 독일 전통의 참모총장 명칭인 <a href="/wiki/%EB%8C%80%EC%9E%A5%EA%B5%B0%EC%B0%B8%EB%AA%A8" title="대장군참모">장군참모</a>에서 유래했다.</p>
<p>역대 장군총감은 모두 <a class="mw-redirect" href="/wiki/4%EC%84%B1%EC%9E%A5%EA%B5%B0" title="4성장군">4성장군</a>이 부임했으며, <a class="mw-redirect" href="/wiki/%EB%8F%85%EC%9D%BC_%EC%97%B0%EB%B0%A9%EA%B5%AD%EB%B0%A9%EB%B6%80" title="독일 연방국방부">연방국방부</a> 내의 병력지도참모부(<span lang="de" xml:lang="de">Führungsstab der Streitkräfte</span>)의 장이었다. 장군총감은 평화시 군통수권을 가진 연방국방장관에게 직접 보고를 하는 군사자문관이다.</p>
<p>연방군 장군총감에게 딸린 직속 부하로는 연방군의 각 분과 - 즉 <a href="/wiki/%EB%8F%85%EC%9D%BC_%EC%9C%A1%EA%B5%B0" title="독일 육군">독일 육군</a>, <a href="/wiki/%EB%8F%85%EC%9D%BC_%ED%95%B4%EA%B5%B0" title="독일 해군">독일 해군</a>, <a href="/wiki/%EB%8F%85%EC%9D%BC_%EA%B3%B5%EA%B5%B0" title="독일 공군">독일 공군</a>의 장인 <a href="/wiki/%EC%9C%A1%EA%B5%B0%EC%B4%9D%EA%B0%90" title="육군총감">육군총감</a>, <a href="/wiki/%ED%95%B4%EA%B5%B0%EC%B4%9D%EA%B0%90" title="해군총감">해군총감</a>, <a href="/wiki/%EA%B3%B5%EA%B5%B0%EC%B4%9D%EA%B0%90" title="공군총감">공군총감</a>, 그리고 <a href="/wiki/%EC%A0%84%EB%A0%A5%EA%B8%B0%EB%B0%98%EA%B5%B0" title="전력기반군">전력기반군</a> 사령관과 <a href="/wiki/%EA%B5%AC%ED%98%B8%EC%97%85%EB%AC%B4%EA%B5%B0" title="구호업무군">구호업무군</a> 사령관이 있다.</p>
