<p><b>FK 믈라다볼레슬라프</b>는 <a href="/wiki/%EC%B2%B4%EC%BD%94" title="체코">체코</a>의 <a href="/wiki/%EC%B6%95%EA%B5%AC" title="축구">축구</a> 클럽으로 센트럴 보헤미안 지방의 <a href="/wiki/%EB%AF%88%EB%9D%BC%EB%8B%A4%EB%B3%BC%EB%A0%88%EC%8A%AC%EB%9D%BC%ED%94%84" title="믈라다볼레슬라프">믈라다볼레슬라프</a>를 연고로 하고 있다.</p>
<p>믈라다볼레슬라프는 <a href="/wiki/%EC%B2%B4%EC%BD%94" title="체코">체코</a>가 독립한 뒤부터 4부리그에서 시작하였고 1996-97시즌에 4부리그 우승을 통해 3부리그의 두 디비전 가운데 하나인 체코 풋볼리그에 올라가 한 시즌만에 2부리그로 승격하였다. 2부리그에서 한동안 보내던 믈라다볼레슬라프는 2003-04 시즌에 2부 리그 우승을 차지하며 <a class="mw-redirect" href="/wiki/%EA%B0%90%EB%B8%8C%EB%A6%AC%EB%88%84%EC%8A%A4_%EB%A6%AC%EA%B0%80" title="감브리누스 리가">감브리누스 리가</a>에 올라가게 되었다.</p>
<p>2008-09 시즌이 끝난 현재 믈라다볼레슬라프는 리그 6위를 차지하였다.</p>

<h2><span class="mw-headline" id=".EC.9C.A0.EB.9F.BD_.EB.8C.80.ED.9A.8C">유럽 대회</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=FK_%EB%AF%88%EB%9D%BC%EB%8B%A4%EB%B3%BC%EB%A0%88%EC%8A%AC%EB%9D%BC%ED%94%84&amp;action=edit&amp;section=1" title="부분 편집: 유럽 대회">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>클럽의 최고성적은 2005-06시즌의 <a class="mw-redirect" href="/wiki/%EA%B0%90%EB%B8%8C%EB%A6%AC%EB%88%84%EC%8A%A4_%EB%A6%AC%EA%B0%80" title="감브리누스 리가">감브리누스 리가</a> 2위로, 다음시즌에 <a class="mw-redirect" href="/wiki/UEFA%EC%BB%B5" title="UEFA컵">UEFA컵</a>에 진출하였다. 2006-07시즌 UEFA컵에서 1라운드에서 <a href="/wiki/%EC%98%AC%EB%9E%AD%ED%94%BC%ED%81%AC_%EB%93%9C_%EB%A7%88%EB%A5%B4%EC%84%B8%EC%9C%A0" title="올랭피크 드 마르세유">올랭피크 마르세유</a>를 합계 4-3으로 물리치고 조별리그에 진출하는 저력을 보여줬지만, 조별리그에서는 1승도 하지 못하고 탈락하였다.</p>
<p>다음 시즌인 <a href="/wiki/UEFA%EC%BB%B5_2007-08" title="UEFA컵 2007-08">UEFA컵 2007-08</a>에서 <a class="mw-redirect" href="/wiki/US_%ED%8C%94%EB%A0%88%EB%A5%B4%EB%AA%A8" title="US 팔레르모">팔레르모</a>를 합계 1-1에서 승부차기 4-2로 물리치고 다시 조별리그에 올라갔지만 4위를 기록하여 조별리그에서 또 다시 탈락하였다.</p>
