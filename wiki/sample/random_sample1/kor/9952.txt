<p><b>연제운</b>(<a href="/wiki/1994%EB%85%84" title="1994년">1994년</a> <a href="/wiki/8%EC%9B%94_28%EC%9D%BC" title="8월 28일">8월 28일</a> ~ )은 <a href="/wiki/%EB%8C%80%ED%95%9C%EB%AF%BC%EA%B5%AD" title="대한민국">대한민국</a>의 <a href="/wiki/%EC%B6%95%EA%B5%AC" title="축구">축구</a> 선수이며 포지션은 <a href="/wiki/%EC%88%98%EB%B9%84%EC%88%98" title="수비수">수비수</a>이다. 현재 <a href="/wiki/%EB%8C%80%ED%95%9C%EB%AF%BC%EA%B5%AD" title="대한민국">대한민국</a> <a href="/wiki/K%EB%A6%AC%EA%B7%B8_%ED%81%B4%EB%9E%98%EC%8B%9D" title="K리그 클래식">K리그 클래식</a> <a href="/wiki/%EC%84%B1%EB%82%A8_FC" title="성남 FC">성남 FC</a>에서 활약하고 있다.</p>

<h2><span class="mw-headline" id=".EC.B6.95.EA.B5.AC.EC.9D.B8_.EA.B2.BD.EB.A0.A5">축구인 경력</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%97%B0%EC%A0%9C%EC%9A%B4&amp;action=edit&amp;section=1" title="부분 편집: 축구인 경력">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<h3><span class="mw-headline" id=".EC.84.A0.EC.88.98_.EA.B2.BD.EB.A0.A5">선수 경력</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%97%B0%EC%A0%9C%EC%9A%B4&amp;action=edit&amp;section=2" title="부분 편집: 선수 경력">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p><a href="/wiki/%EC%84%B1%EB%82%A8_FC" title="성남 FC">성남 FC</a>의 유소년 팀인 <a href="/wiki/%ED%92%8D%EC%83%9D%EA%B3%A0%EB%93%B1%ED%95%99%EA%B5%90" title="풍생고등학교">풍생고등학교</a>를 졸업한 후 <a href="/wiki/%EC%84%A0%EB%AC%B8%EB%8C%80%ED%95%99%EA%B5%90" title="선문대학교">선문대학교</a>를 거쳐 2016 시즌을 앞두고 <a href="/wiki/%EC%84%B1%EB%82%A8_FC" title="성남 FC">성남 FC</a>에 입단하였다. 6월 12일 <a href="/wiki/%EC%A0%84%EB%B6%81_%ED%98%84%EB%8C%80_%EB%AA%A8%ED%84%B0%EC%8A%A4" title="전북 현대 모터스">전북 현대 모터스</a>와의 경기에서 교체 투입되며 데뷔전을 치렀다. 같은 해 7월 10일 <a href="/wiki/%EC%83%81%EC%A3%BC_%EC%83%81%EB%AC%B4_%EC%B6%95%EA%B5%AC%EB%8B%A8" title="상주 상무 축구단">상주 상무</a>와의 경기에서 데뷔골을 성공시켰다.<sup class="reference" id="cite_ref-1"><a href="#cite_note-1">[1]</a></sup></p>
