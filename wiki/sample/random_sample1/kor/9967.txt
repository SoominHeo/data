<p>《<b>열혈남아</b>》(<span style="font-size: smaller;"><a href="/wiki/%EC%98%81%EC%96%B4" title="영어">영어</a>:</span> <span lang="en" xml:lang="en">Cruel Winter Blues</span>)는 <a href="/wiki/2006%EB%85%84" title="2006년">2006년</a> <a href="/wiki/11%EC%9B%94_9%EC%9D%BC" title="11월 9일">11월 9일</a> 개봉한 <a href="/wiki/%EB%8C%80%ED%95%9C%EB%AF%BC%EA%B5%AD" title="대한민국">대한민국</a>의 <a href="/wiki/%EC%98%81%ED%99%94" title="영화">영화</a>이다. <a href="/wiki/%EC%9D%B4%EC%A0%95%EB%B2%94" title="이정범">이정범</a> 감독의 입봉작으로, <a href="/wiki/%EC%84%A4%EA%B2%BD%EA%B5%AC" title="설경구">설경구</a>, <a href="/wiki/%EC%A1%B0%ED%95%9C%EC%84%A0" title="조한선">조한선</a>, <a href="/wiki/%EB%82%98%EB%AC%B8%ED%9D%AC" title="나문희">나문희</a>가 출연했다. 같은 날짜에 개봉한 영화 《<a href="/wiki/%EC%82%AC%EB%9E%91%EB%94%B0%EC%9C%88_%ED%95%84%EC%9A%94%EC%97%86%EC%96%B4" title="사랑따윈 필요없어">사랑따윈 필요없어</a>》가 같은 제작사에서 제작된 영화라 논란이 일기도 했다. 또한, 흥행에는 실패했지만 이 작품으로 <a href="/wiki/%EB%82%98%EB%AC%B8%ED%9D%AC" title="나문희">나문희</a>는 <a href="/wiki/%EC%B2%AD%EB%A3%A1%EC%98%81%ED%99%94%EC%83%81" title="청룡영화상">청룡영화상</a> 여우조연상을 수상했다.</p>
