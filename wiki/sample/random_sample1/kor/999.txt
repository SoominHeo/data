<p><b>FM 방송</b>(<span style="font-size: smaller;"><a href="/wiki/%EC%98%81%EC%96%B4" title="영어">영어</a>:</span> <span lang="en" xml:lang="en">FM broadcasting</span>)은 <a href="/wiki/%EC%A3%BC%ED%8C%8C%EC%88%98_%EB%B3%80%EC%A1%B0" title="주파수 변조">주파수 변조</a>를 이용하는 방송이다. 과거 <a class="mw-redirect" href="/wiki/%EC%95%84%EB%82%A0%EB%A1%9C%EA%B7%B8_TV" title="아날로그 TV">아날로그 TV</a> 방송의 음성부도 해당된다. 본 항목에서는 FM <a href="/wiki/%EB%9D%BC%EB%94%94%EC%98%A4" title="라디오">라디오</a> 방송을 토대로 설명한다.</p>

<h2><span class="mw-headline" id=".EB.B0.A9.EC.86.A1_.EB.8C.80.EC.97.AD">방송 대역</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=FM_%EB%B0%A9%EC%86%A1&amp;action=edit&amp;section=1" title="부분 편집: 방송 대역">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>FM 라디오 방송 대역은 보통 전세계적으로 <a href="/wiki/%EC%B4%88%EB%8B%A8%ED%8C%8C" title="초단파">초단파</a> 87.5~108.0 MHz 사이에서 사용되고 있다.<sup class="reference" id="cite_ref-1"><a href="#cite_note-1">[1]</a></sup> 주파수 간격은 보통 전세계적으로는 0.1 MHz(100 kHz)씩 사용하며, 일부 국가에 따라 간격 차이가 있다. <a href="/wiki/%EB%8C%80%ED%95%9C%EB%AF%BC%EA%B5%AD" title="대한민국">대한민국</a>, <a href="/wiki/%EB%B6%81%EC%95%84%EB%A9%94%EB%A6%AC%EC%B9%B4" title="북아메리카">북아메리카</a> 및 <a href="/wiki/%EC%B9%B4%EB%A6%AC%EB%B8%8C_%ED%95%B4" title="카리브 해">카리브 해</a> 연안 지역에서는 0.2 MHz(200 kHz)씩 홀수 단위로 배당되며, <a href="/wiki/%EC%9D%B4%ED%83%88%EB%A6%AC%EC%95%84" title="이탈리아">이탈리아</a>에서는 0.05 MHz(50 kHz)씩 사용하기도 한다. <a class="mw-redirect" href="/wiki/%EA%B5%AC%EC%86%8C%EB%A0%A8" title="구소련">구소련</a> 및 과거 <a href="/wiki/%EB%8F%99%EA%B5%AC%EA%B6%8C" title="동구권">동구권</a> 국가에서는 65~74 MHz 대역에 0.03 MHz(30 kHz)씩 사용한 적이 있었으며, <a href="/wiki/%EC%9D%BC%EB%B3%B8" title="일본">일본</a>에서는 76~90 MHz 대역을 사용하고 있다.</p>
<h2><span class="mw-headline" id=".EB.B3.80.EC.A1.B0">변조</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=FM_%EB%B0%A9%EC%86%A1&amp;action=edit&amp;section=2" title="부분 편집: 변조">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<div class="rellink relarticle mainarticle"><img alt="" data-file-height="128" data-file-width="128" height="16" src="//upload.wikimedia.org/wikipedia/commons/thumb/e/ec/Crystal_Clear_app_xmag.svg/16px-Crystal_Clear_app_xmag.svg.png" srcset="//upload.wikimedia.org/wikipedia/commons/thumb/e/ec/Crystal_Clear_app_xmag.svg/24px-Crystal_Clear_app_xmag.svg.png 1.5x, //upload.wikimedia.org/wikipedia/commons/thumb/e/ec/Crystal_Clear_app_xmag.svg/32px-Crystal_Clear_app_xmag.svg.png 2x" width="16"/> 이 부분의 본문은 <a href="/wiki/%EC%A3%BC%ED%8C%8C%EC%88%98_%EB%B3%80%EC%A1%B0" title="주파수 변조">주파수 변조</a>입니다.</div>
<p>FM 방송은 <a href="/wiki/%EC%A3%BC%ED%8C%8C%EC%88%98_%EB%B3%80%EC%A1%B0" title="주파수 변조">주파수 변조</a>를 사용하는데, AM방송(<a href="/wiki/%EC%9E%A5%ED%8C%8C%EB%B0%A9%EC%86%A1" title="장파방송">장파방송</a>,<a href="/wiki/%EC%A4%91%ED%8C%8C%EB%B0%A9%EC%86%A1" title="중파방송">중파방송</a>,<a class="mw-redirect" href="/wiki/%EB%8B%A8%ED%8C%8C%EB%B0%A9%EC%86%A1" title="단파방송">단파방송</a>)에 비해 음질이 깨끗하며 기상 조건의 영향을 덜 받는 장점이 있다. AM채널이 20여개의 채널 밖에 두지 못한 반면, FM은 많은 주파수를 사용해서 채널 여유분이 많아 100여개 이상의 채널을 두는 것이 가능하다.</p>
<p>주파수 변조의 특성상 넓은 주파수 대역을 사용해야하기 때문에 <a href="/wiki/%EC%B4%88%EB%8B%A8%ED%8C%8C" title="초단파">초단파</a>이상의 주파수 대역을 사용한다. 넓은 주파수의 특성으로 음성주파수 대역이 50 Hz~15kHz으로 고음질의 방송과 스테레오 방송을 할 수 있다.</p>
<h2><span class="mw-headline" id=".EC.97.A0.ED.8D.BC.EC.8B.9C.EC.8A.A4">엠퍼시스</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=FM_%EB%B0%A9%EC%86%A1&amp;action=edit&amp;section=3" title="부분 편집: 엠퍼시스">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<div class="rellink relarticle mainarticle"><img alt="" data-file-height="128" data-file-width="128" height="16" src="//upload.wikimedia.org/wikipedia/commons/thumb/e/ec/Crystal_Clear_app_xmag.svg/16px-Crystal_Clear_app_xmag.svg.png" srcset="//upload.wikimedia.org/wikipedia/commons/thumb/e/ec/Crystal_Clear_app_xmag.svg/24px-Crystal_Clear_app_xmag.svg.png 1.5x, //upload.wikimedia.org/wikipedia/commons/thumb/e/ec/Crystal_Clear_app_xmag.svg/32px-Crystal_Clear_app_xmag.svg.png 2x" width="16"/> 이 부분의 본문은 <a class="new" href="/w/index.php?title=%EC%97%A0%ED%8D%BC%EC%8B%9C%EC%8A%A4&amp;action=edit&amp;redlink=1" title="엠퍼시스 (없는 문서)">엠퍼시스</a>입니다.</div>
<p>FM 방송으로 쓰는 주파수 변조는 변조 주파수가 높을 수록 잡음이 심한 경우가 생긴다. 이를 해결하기 위해 높은 변조 신호에서 변조를 강하게 하여 송신하고 역특성 회로로 원래 신호로 되돌려서 <a href="/wiki/%EC%8B%A0%ED%98%B8_%EB%8C%80_%EC%9E%A1%EC%9D%8C%EB%B9%84" title="신호 대 잡음비">신호 대 잡음비</a>를 개선하는 방식을 쓰는데 이를 엠퍼시스(Emphasis)라고 한다. 엠퍼시스는 송신측에서 변조하는 '프리엠퍼시스(pre-emphasis)'와 수신기에서 다시 되돌리는 '디엠퍼시스(de-emphasis)'가 있다.</p>
<p>대부분의 국가에서 엠퍼시스 값은 50μs이며, <a class="mw-redirect" href="/wiki/%EB%B6%81%EB%AF%B8" title="북미">북미</a>지역과 <a href="/wiki/%EB%8C%80%ED%95%9C%EB%AF%BC%EA%B5%AD" title="대한민국">대한민국</a>에서는 75μs이다. 모노와 스테레오 방송 모두 적용된다. 프리엠퍼시스와 디엠퍼시스의 값이 다른 경우 고음부 신호가 약간 왜곡되는 경우가 있다.</p>
<h2><span class="mw-headline" id="FM_.EC.8A.A4.ED.85.8C.EB.A0.88.EC.98.A4">FM 스테레오</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=FM_%EB%B0%A9%EC%86%A1&amp;action=edit&amp;section=4" title="부분 편집: FM 스테레오">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>FM 스테레오 방송은 FM 방송을 두가지 신호로 동시에 보내는 방식이다. <a href="/wiki/%EC%B4%88%EB%8B%A8%ED%8C%8C" title="초단파">초단파</a> 주파수가 가지는 장점인 넓은 전파의 폭을 이용하여 전면 부분의 입체 재생 효과를 낼 수 있다.</p>
<p>FM 스테레오 방송은 주로 <a class="new" href="/w/index.php?title=%ED%8C%8C%EC%9D%BC%EB%9F%BF_%ED%86%A4_%EB%B0%A9%EC%8B%9D&amp;action=edit&amp;redlink=1" title="파일럿 톤 방식 (없는 문서)">파일럿 톤 방식</a>(AM-FM 스테레오 방식)을 사용한다. 이 방식은 기존 모노 FM 수신기와 호환을 위해 나온 것으로 좌측(L),우측(R) <a href="/wiki/%EB%A7%88%EC%9D%B4%ED%81%AC%EB%A1%9C%ED%8F%B0" title="마이크로폰">마이크로폰</a>으로 입력된 두 신호를 변조한 합신호(L+R)와 차신호(L-R)를 하나의 전파에 담는다. 모노 방송으로 송출되는 합신호(L+R)의 대역폭은 0~15KHz, 스테레오 방송으로 송출되는 차신호(L-R)의 대역폭은 23~53kHz이며, 여기서 23~38 kHz,38~53kHz로 나뉘어 각각 L,R신호를 담는다. 그리고 가운데의 19 kHz 대역에 검토용으로 사용하는 <a class="new" href="/w/index.php?title=%ED%8C%8C%EC%9D%BC%EB%9F%BF_%EC%8B%A0%ED%98%B8&amp;action=edit&amp;redlink=1" title="파일럿 신호 (없는 문서)">파일럿 신호</a>를 사용한다.</p>
<p>파일럿 톤 방식의 장점은 기존 모노 수신기와 호환이 된다는 점이 있으나 모노 방송에 비해 전파 왜곡에 더 민감해서 잡음에 약하며, 부분적인 중첩 방식이기 때문에 분리도 면에서 10%정도로 손실되는 문제가 있다.</p>
<p>보통 2채널 스테레오 방식인데, 4채널 스테레오 방송을 1970년대 미국 중심으로 실용화할 예정이었으나 방식이 통일되지 않고, 높은 비용으로 무산되었다.</p>
<h2><span class="mw-headline" id="FM_.EB.8B.A4.EC.A4.91_.EB.B0.A9.EC.86.A1">FM 다중 방송</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=FM_%EB%B0%A9%EC%86%A1&amp;action=edit&amp;section=5" title="부분 편집: FM 다중 방송">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>FM 방송의 주파수 대역은 고음질과 스테레오 방송을 실시함에도 대역폭의 여유가 있기 때문에 이를 활용하여 별도의 음성 신호나 데이터를 중첩하여 방송하는 <a class="new" href="/w/index.php?title=FM_%EB%8B%A4%EC%A4%91_%EB%B0%A9%EC%86%A1&amp;action=edit&amp;redlink=1" title="FM 다중 방송 (없는 문서)">FM 다중 방송</a>같은 시스템도 가능하다.</p>
<p>FM 다중 방송은 <a href="/wiki/%ED%85%94%EB%A0%88%EB%B9%84%EC%A0%84" title="텔레비전">텔레비전</a>의 <a class="new" href="/w/index.php?title=%EC%9D%8C%EC%84%B1%EB%8B%A4%EC%A4%91%EB%B0%A9%EC%86%A1&amp;action=edit&amp;redlink=1" title="음성다중방송 (없는 문서)">음성다중방송</a>과 같은 원리이며 FM 방송의 주파수 대역폭중 57 kHz, 67 kHz, 76 kHz, 92kHz의 높은 대역에 보조 채널을 설정하고 여기에서 프로그램 안내, 뉴스 헤드라인, 교통 정보, 주식 정보, 무선 호출(페이징), 라디오 텍스트, 배경 음악, 날짜, 시각 등 여러가지 다양한 내용을 보낼 수 있다.<sup class="reference" id="cite_ref-2"><a href="#cite_note-2">[2]</a></sup></p>
<p>FM 다중 방송 방식은 크게 3가지가 있다.</p>
<ul>
<li><a class="new" href="/w/index.php?title=%EB%9D%BC%EB%94%94%EC%98%A4_%EB%8D%B0%EC%9D%B4%ED%84%B0_%EC%8B%9C%EC%8A%A4%ED%85%9C&amp;action=edit&amp;redlink=1" title="라디오 데이터 시스템 (없는 문서)">라디오 데이터 시스템</a>(RDS) : 57kHz의 대역을 사용한다. 전송 속도는 1187.5bps로 <a href="/wiki/%EC%9C%A0%EB%9F%BD" title="유럽">유럽</a>에서 주로 쓰이는 방식이다.</li>
<li><a class="new" href="/w/index.php?title=%EB%B3%B4%EC%A1%B0%EC%A0%81_%ED%86%B5%EC%8B%A0_%EC%97%85%EB%AC%B4_%ED%97%88%EA%B0%80&amp;action=edit&amp;redlink=1" title="보조적 통신 업무 허가 (없는 문서)">보조적 통신 업무 허가</a>(SCA) : 67kHz의 대역을 사용한다. <a href="/wiki/1955%EB%85%84" title="1955년">1955년</a> <a href="/wiki/%EB%AF%B8%EA%B5%AD" title="미국">미국</a>에서 시작한 방식으로 최초의 FM 다중 방송 방식으로 알려져 있다. <a href="/wiki/KBS_%EC%A0%9C3%EB%9D%BC%EB%94%94%EC%98%A4" title="KBS 제3라디오">KBS 제3라디오</a>가 <a href="/wiki/2000%EB%85%84" title="2000년">2000년</a>에 <a class="mw-redirect" href="/wiki/%EC%A4%91%ED%8C%8C_%EB%B0%A9%EC%86%A1" title="중파 방송">중파 방송</a> 전환 이전에 <a href="/wiki/1995%EB%85%84" title="1995년">1995년</a> <a href="/wiki/KBS_%EC%A0%9C1%EB%9D%BC%EB%94%94%EC%98%A4" title="KBS 제1라디오">KBS 제1라디오</a>의 <a href="/wiki/%ED%91%9C%EC%A4%80FM" title="표준FM">표준FM</a> 방송에서 SCA방식을 통한 FM 다중 방송으로 송출한 적이 있었다.</li>
<li><a class="new" href="/w/index.php?title=%EB%8D%B0%EC%9D%B4%ED%84%B0_%EB%9D%BC%EB%94%94%EC%98%A4_%EC%B1%84%EB%84%90&amp;action=edit&amp;redlink=1" title="데이터 라디오 채널 (없는 문서)">데이터 라디오 채널</a>(DARC) : 76kHz의 대역을 사용한다. 16kbps의 대용량으로 전송하며 <a href="/wiki/1986%EB%85%84" title="1986년">1986년</a> 일본 <a class="mw-redirect" href="/wiki/NHK" title="NHK">NHK</a>에서 실시한 방식이다. <a href="/wiki/1995%EB%85%84" title="1995년">1995년</a> <a href="/wiki/3%EC%9B%94_28%EC%9D%BC" title="3월 28일">3월 28일</a> <a href="/wiki/%EC%B9%B4%EC%8B%9C%EC%98%A4" title="카시오">카시오</a>에서 문자 데이터를 수신받아 화면에 내보내는 수신기인 '<a class="new" href="/w/index.php?title=%EC%B9%B4%EC%8B%9C%EC%98%A4_MR-1&amp;action=edit&amp;redlink=1" title="카시오 MR-1 (없는 문서)">카시오 MR-1</a>'을 내놓았다.<sup class="reference" id="cite_ref-.EB.B0.A9.EC.86.A1.EB.AF.B8.EB.94.94.EC.96.B4_3-0"><a href="#cite_note-.EB.B0.A9.EC.86.A1.EB.AF.B8.EB.94.94.EC.96.B4-3">[3]</a></sup></li>
</ul>
