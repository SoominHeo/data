<p><b>영대병원역</b>(嶺大病院驛)은 <a href="/wiki/%EB%8C%80%ED%95%9C%EB%AF%BC%EA%B5%AD" title="대한민국">대한민국</a> <a href="/wiki/%EB%8C%80%EA%B5%AC%EA%B4%91%EC%97%AD%EC%8B%9C" title="대구광역시">대구광역시</a> <a href="/wiki/%EB%82%A8%EA%B5%AC_(%EB%8C%80%EA%B5%AC%EA%B4%91%EC%97%AD%EC%8B%9C)" title="남구 (대구광역시)">남구</a> 대명로에 있는 <a href="/wiki/%EB%8C%80%EA%B5%AC_%EB%8F%84%EC%8B%9C%EC%B2%A0%EB%8F%84_1%ED%98%B8%EC%84%A0" title="대구 도시철도 1호선">대구 도시철도 1호선</a>의 <a href="/wiki/%EC%A0%84%EC%B2%A0%EC%97%AD" title="전철역">전철역</a>이다.</p>

<h2><span class="mw-headline" id=".ED.8A.B9.EC.A7.95">특징</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%98%81%EB%8C%80%EB%B3%91%EC%9B%90%EC%97%AD&amp;action=edit&amp;section=1" title="부분 편집: 특징">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p><a href="/wiki/%EB%8C%80%EA%B5%AC_%EB%8F%84%EC%8B%9C%EC%B2%A0%EB%8F%84_1%ED%98%B8%EC%84%A0" title="대구 도시철도 1호선">대구 도시철도 1호선</a>의 <a class="mw-disambig" href="/wiki/%EA%B5%90%EB%8C%80%EC%97%AD" title="교대역">교대역</a>과 <a href="/wiki/%ED%98%84%EC%B6%A9%EB%A1%9C%EC%97%AD" title="현충로역">현충로역</a>사이에 있으며, 교대역에서 오른쪽으로 꺾어 대명로가 시작되는 지점에 역이 위치하고 있다.</p>
<h2><span class="mw-headline" id=".EC.97.AD.EB.AA.85.EC.9D.98_.EC.9C.A0.EB.9E.98">역명의 유래</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%98%81%EB%8C%80%EB%B3%91%EC%9B%90%EC%97%AD&amp;action=edit&amp;section=2" title="부분 편집: 역명의 유래">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>인근에 <a href="/wiki/%EC%98%81%EB%82%A8%EB%8C%80%ED%95%99%EA%B5%90" title="영남대학교">영남대학교</a> 의료원·의학전문대학원이 위치하고 있고, 영대병원네거리 근처에 위치하고 있어서 역명이 정해졌다. 공사 중 역명은 남대구역이었다. <a href="/wiki/%EB%8C%80%EA%B5%AC_%EB%8F%84%EC%8B%9C%EC%B2%A0%EB%8F%84_2%ED%98%B8%EC%84%A0" title="대구 도시철도 2호선">대구 도시철도 2호선</a> <a href="/wiki/%EC%98%81%EB%82%A8%EB%8C%80%EC%97%AD" title="영남대역">영남대역</a>과 함께 <a href="/wiki/%EC%98%81%EB%82%A8%EB%8C%80%ED%95%99%EA%B5%90" title="영남대학교">영남대학교</a> 본 캠퍼스와 의대 캠퍼스가 모두 지하철로 연결된다. 출구는 3곳이다.</p>
<h2><span class="mw-headline" id=".EC.97.AD.EC.82.AC">역사</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%98%81%EB%8C%80%EB%B3%91%EC%9B%90%EC%97%AD&amp;action=edit&amp;section=3" title="부분 편집: 역사">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<ul>
<li><a href="/wiki/1997%EB%85%84" title="1997년">1997년</a> <a href="/wiki/11%EC%9B%94_26%EC%9D%BC" title="11월 26일">11월 26일</a> : <a href="/wiki/%EB%8C%80%EA%B5%AC_%EB%8F%84%EC%8B%9C%EC%B2%A0%EB%8F%84_1%ED%98%B8%EC%84%A0" title="대구 도시철도 1호선">대구 도시철도 1호선</a> 개통과 함께 영업 개시</li>
<li><a href="/wiki/2016%EB%85%84" title="2016년">2016년</a> <a href="/wiki/7%EC%9B%94_22%EC%9D%BC" title="7월 22일">7월 22일</a> : <a href="/wiki/%EC%8A%A4%ED%81%AC%EB%A6%B0%EB%8F%84%EC%96%B4" title="스크린도어">스크린도어</a> 설치</li>
</ul>
<h2><span class="mw-headline" id=".EC.97.AD_.EA.B5.AC.EC.A1.B0">역 구조</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%98%81%EB%8C%80%EB%B3%91%EC%9B%90%EC%97%AD&amp;action=edit&amp;section=4" title="부분 편집: 역 구조">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>2면 2선의 <a class="mw-redirect" href="/wiki/%EC%83%81%EB%8C%80%EC%8B%9D_%EC%8A%B9%EA%B0%95%EC%9E%A5" title="상대식 승강장">상대식 승강장</a>이 있는 지하 역이다. <a href="/wiki/%EC%8A%A4%ED%81%AC%EB%A6%B0%EB%8F%84%EC%96%B4" title="스크린도어">스크린도어</a>가 설치되어 있다.</p>
<p>대합실을 통해 반대편 승강장으로 이동 할 수 있다.</p>
<h3><span class="mw-headline" id=".EC.8A.B9.EA.B0.95.EC.9E.A5">승강장</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%98%81%EB%8C%80%EB%B3%91%EC%9B%90%EC%97%AD&amp;action=edit&amp;section=5" title="부분 편집: 승강장">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
