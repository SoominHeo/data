<p><b>영덕군</b>(盈德郡)은 <a href="/wiki/%EB%8C%80%ED%95%9C%EB%AF%BC%EA%B5%AD" title="대한민국">대한민국</a> <a href="/wiki/%EA%B2%BD%EC%83%81%EB%B6%81%EB%8F%84" title="경상북도">경상북도</a> <a class="mw-redirect" href="/wiki/%EB%8F%99%ED%95%B4%EC%95%88" title="동해안">동해안</a>에 있는 <a href="/wiki/%EA%B5%B0_(%ED%96%89%EC%A0%95_%EA%B5%AC%EC%97%AD)" title="군 (행정 구역)">군</a>이다. 어업이 발달하여 특산물로 <a href="/wiki/%EB%8C%80%EA%B2%8C" title="대게">대게</a>가 유명하다. 기타 복숭아, 송이, 물가자미, 황금은어 등의 특산물이 있고, 해수욕장으로 고래불, 장사, 대진해수욕장이 있으며, 유원지로는 달산 옥계계곡, 지품 오천솔밭 등이 있다. 주요관광지로 강구항, 삼사해상공원, 영덕어촌민속전시관, 영덕풍력발전단지, 영덕신재생에너지전시관, 신돌석장군유적지, 영해 <a class="new" href="/w/index.php?title=%EA%B4%B4%EC%8B%9C%EC%A0%84%ED%86%B5%EB%A7%88%EC%9D%84&amp;action=edit&amp;redlink=1" title="괴시전통마을 (없는 문서)">괴시전통마을</a>, <a class="new" href="/w/index.php?title=%EC%98%81%EB%8D%95%ED%96%A5%EA%B5%90&amp;action=edit&amp;redlink=1" title="영덕향교 (없는 문서)">영덕향교</a> 등이 있다. 군청 소재지는 <a href="/wiki/%EC%98%81%EB%8D%95%EC%9D%8D" title="영덕읍">영덕읍</a>이고, 행정구역은 1읍 8면이다.</p>

<h2><span class="mw-headline" id=".EC.97.AD.EC.82.AC">역사</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%98%81%EB%8D%95%EA%B5%B0&amp;action=edit&amp;section=1" title="부분 편집: 역사">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<ul>
<li><a href="/wiki/%EA%B3%A0%EA%B5%AC%EB%A0%A4" title="고구려">고구려</a> 시대에는 우시군 (于尸郡)이었다.</li>
<li><a href="/wiki/%EA%B3%A0%EA%B5%AC%EB%A0%A4" title="고구려">고구려</a> <a href="/wiki/%EC%9E%A5%EC%88%98%EC%99%95" title="장수왕">장수왕</a> 때에는 군 전체가 고구려에 속해 있었으나, <a href="/wiki/%EC%A7%84%ED%9D%A5%EC%99%95" title="진흥왕">진흥왕</a> 때 <a href="/wiki/%EC%8B%A0%EB%9D%BC" title="신라">신라</a>에 편입되었다.</li>
<li><a href="/wiki/%EC%8B%A0%EB%9D%BC" title="신라">신라</a> 시대에는 야시홀, 야성 (野城), 유린군 (有隣郡) 등으로 이름이 바뀌었다.</li>

<h2><span class="mw-headline" id=".EC.A7.80.EB.A6.AC">지리</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%98%81%EB%8D%95%EA%B5%B0&amp;action=edit&amp;section=2" title="부분 편집: 지리">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>경상북도 동북부에 위치하여 영남지방에 속하고 <a href="/wiki/%ED%8F%AC%ED%95%AD%EC%8B%9C" title="포항시">포항시</a>와 <a href="/wiki/%EC%9A%B8%EC%A7%84%EA%B5%B0" title="울진군">울진군</a>, <a href="/wiki/%EC%98%81%EC%96%91%EA%B5%B0" title="영양군">영양군</a>, <a href="/wiki/%EC%B2%AD%EC%86%A1%EA%B5%B0" title="청송군">청송군</a>과 경계를 이루고 있다. <a href="/wiki/%ED%83%9C%EB%B0%B1%EC%82%B0%EB%A7%A5" title="태백산맥">태백산맥</a>이 동남쪽으로 뻗어내리고, 칠보산과 팔각산이 형성되어 있으며, 병곡 영해평야를 이루고 향토의 젖줄인 오십천이 흐르고 있다. <sup class="reference" id="cite_ref-11"><a href="#cite_note-11">[11]</a></sup></p>
<h3><span class="mw-headline" id=".EA.B8.B0.ED.9B.84">기후</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%98%81%EB%8D%95%EA%B5%B0&amp;action=edit&amp;section=3" title="부분 편집: 기후">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>대체적으로 온후한 편이며, 여름 최고온도는 섭씨 38도이며, 겨울 최저온도는 영하 13.8도이다. 봄철은 동남풍, 겨울철은 서북풍이 심하게 불며 가을이 길고 봄이 짧은 것이 특징이다.</p>

<div class="rellink relarticle mainarticle"><img alt="" data-file-height="128" data-file-width="128" height="16" src="//upload.wikimedia.org/wikipedia/commons/thumb/e/ec/Crystal_Clear_app_xmag.svg/16px-Crystal_Clear_app_xmag.svg.png" srcset="//upload.wikimedia.org/wikipedia/commons/thumb/e/ec/Crystal_Clear_app_xmag.svg/24px-Crystal_Clear_app_xmag.svg.png 1.5x, //upload.wikimedia.org/wikipedia/commons/thumb/e/ec/Crystal_Clear_app_xmag.svg/32px-Crystal_Clear_app_xmag.svg.png 2x" width="16"/> 이 부분의 본문은 <a href="/wiki/%EC%98%81%EB%8D%95%EA%B5%B0%EC%9D%98_%ED%96%89%EC%A0%95_%EA%B5%AC%EC%97%AD" title="영덕군의 행정 구역">영덕군의 행정 구역</a>입니다.</div>
<p>영덕군의 행정 구역은 1읍 8면으로 구성되어 있다. 영덕군의 면적은 741.06㎢이다. 인구는 2012년말 기준 19,919세대 40,795명이고, 그 중 27.1%가 <a href="/wiki/%EC%98%81%EB%8D%95%EC%9D%8D" title="영덕읍">영덕읍</a>에 거주한다. 남성 19,486명, 여성 21,309명이다. 1960년대까지는 119,498명이었으나 이후 인구가 감소하였다. <sup class="reference" id="cite_ref-13"><a href="#cite_note-13">[13]</a></sup></p>

<h2><span class="mw-headline" id=".EA.B5.90.ED.86.B5">교통</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%98%81%EB%8D%95%EA%B5%B0&amp;action=edit&amp;section=7" title="부분 편집: 교통">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<h3><span class="mw-headline" id=".EB.8F.84.EB.A1.9C_.EA.B5.90.ED.86.B5">도로 교통</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%98%81%EB%8D%95%EA%B5%B0&amp;action=edit&amp;section=8" title="부분 편집: 도로 교통">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<div class="rellink boilerplate seealso"><img alt="" data-file-height="128" data-file-width="128" height="16" src="//upload.wikimedia.org/wikipedia/commons/thumb/e/ec/Crystal_Clear_app_xmag.svg/16px-Crystal_Clear_app_xmag.svg.png" srcset="//upload.wikimedia.org/wikipedia/commons/thumb/e/ec/Crystal_Clear_app_xmag.svg/24px-Crystal_Clear_app_xmag.svg.png 1.5x, //upload.wikimedia.org/wikipedia/commons/thumb/e/ec/Crystal_Clear_app_xmag.svg/32px-Crystal_Clear_app_xmag.svg.png 2x" width="16"/> <a href="/wiki/%EC%98%81%EB%8D%95%EA%B5%B0%EC%9D%98_%EB%86%8D%EC%96%B4%EC%B4%8C%EB%B2%84%EC%8A%A4" title="영덕군의 농어촌버스">:영덕군의 농어촌버스</a> 문서를 참고하십시오.</div>
<p><a href="/wiki/%EB%8B%B9%EC%A7%84%EC%98%81%EB%8D%95%EA%B3%A0%EC%86%8D%EB%8F%84%EB%A1%9C" title="당진영덕고속도로">당진영덕고속도로</a>가 있으며 <a href="/wiki/2009%EB%85%84" title="2009년">2009년</a> <a href="/wiki/12%EC%9B%94_18%EC%9D%BC" title="12월 18일">12월 18일</a>에 연장선인 (<a href="/wiki/%EB%82%99%EB%8F%99_%EB%B6%84%EA%B8%B0%EC%A0%90" title="낙동 분기점">낙동 분기점</a> ~ <a href="/wiki/%EC%98%81%EB%8D%95_%EB%82%98%EB%93%A4%EB%AA%A9" title="영덕 나들목">영덕 나들목</a>)이 착공하여 <a href="/wiki/2016%EB%85%84" title="2016년">2016년</a> <a href="/wiki/12%EC%9B%94" title="12월">12월</a>에 개통되었다. <sup class="reference" id="cite_ref-15"><a href="#cite_note-15">[15]</a></sup> 추후 <a href="/wiki/%EB%8F%99%ED%95%B4%EA%B3%A0%EC%86%8D%EB%8F%84%EB%A1%9C" title="동해고속도로">동해고속도로</a>와 직결 될 예정이다. 남북으로 <a href="/wiki/%EA%B5%AD%EB%8F%84_%EC%A0%9C7%ED%98%B8%EC%84%A0" title="국도 제7호선">국도 제7호선</a>과 동서로 <a href="/wiki/%EA%B5%AD%EB%8F%84_%EC%A0%9C34%ED%98%B8%EC%84%A0" title="국도 제34호선">국도 제34호선</a>이 관통한다. 예전에는 <a href="/wiki/%EA%B5%AD%EB%8F%84_%EC%A0%9C7%ED%98%B8%EC%84%A0" title="국도 제7호선">국도 제7호선</a>이 확장되지 않아 피서철에는 상습 지정체 구간이었으나, 확장 공사가 이루어져 교통 상황이 나아졌다. 시외버스편을 이용할 때에는 인근의 <a href="/wiki/%EC%95%88%EB%8F%99%EC%8B%9C" title="안동시">안동시</a>이나 <a href="/wiki/%ED%8F%AC%ED%95%AD%EC%8B%9C" title="포항시">포항시</a>을 찾는 것이 편리하다.</p>
<h3><span class="mw-headline" id=".EC.B2.A0.EB.8F.84_.EA.B5.90.ED.86.B5">철도 교통</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%98%81%EB%8D%95%EA%B5%B0&amp;action=edit&amp;section=9" title="부분 편집: 철도 교통">편집</a><span class="mw-editsection-bracket">]</span></span></h3>
<p><a href="/wiki/2017%EB%85%84" title="2017년">2017년</a> <a href="/wiki/10%EC%9B%94" title="10월">10월</a>에 <a href="/wiki/%EB%8F%99%ED%95%B4%EC%84%A0" title="동해선">동해선</a>이 영덕까지 개통되어 <a class="new" href="/w/index.php?title=%EB%82%A8%EC%A0%95%EC%97%AD&amp;action=edit&amp;redlink=1" title="남정역 (없는 문서)">남정역</a>, <a class="mw-disambig" href="/wiki/%EA%B0%95%EA%B5%AC%EC%97%AD" title="강구역">강구역</a>, <a class="new" href="/w/index.php?title=%EC%98%81%EB%8D%95%EC%97%AD&amp;action=edit&amp;redlink=1" title="영덕역 (없는 문서)">영덕역</a>에서 기차를 탈 수 있게 되어 <a href="/wiki/%EB%B6%80%EC%82%B0%EA%B4%91%EC%97%AD%EC%8B%9C" title="부산광역시">부산</a>이나 <a href="/wiki/%EB%8C%80%EA%B5%AC%EA%B4%91%EC%97%AD%EC%8B%9C" title="대구광역시">대구</a>, <a href="/wiki/%EC%84%9C%EC%9A%B8%ED%8A%B9%EB%B3%84%EC%8B%9C" title="서울특별시">서울</a>로의 접근이 더욱 편해진다. 2019년경 동해선이 <a href="/wiki/%EC%82%BC%EC%B2%99%EC%97%AD" title="삼척역">삼척역</a>까지 연장되면 <a class="new" href="/w/index.php?title=%EC%98%81%ED%95%B4%EC%97%AD&amp;action=edit&amp;redlink=1" title="영해역 (없는 문서)">영해역</a>과 <a class="new" href="/w/index.php?title=%EB%B3%91%EA%B3%A1%EC%97%AD&amp;action=edit&amp;redlink=1" title="병곡역 (없는 문서)">병곡역</a>에서도 영덕군의 북부 지역에서도 철도 이용이 용이해질 전망이다. 현재 영덕군에서 가장 가까운 역은 <a href="/wiki/%ED%8F%AC%ED%95%AD%EC%97%AD" title="포항역">포항역</a>이다. <a href="/wiki/%EA%B9%80%EC%B2%9C%EC%97%AD" title="김천역">김천역</a>에서 <a class="new" href="/w/index.php?title=%EC%98%81%EB%8D%95%EC%97%AD&amp;action=edit&amp;redlink=1" title="영덕역 (없는 문서)">영덕역</a>을 잇는 <a class="new" href="/w/index.php?title=%EA%B2%BD%EB%B6%81%EB%82%B4%EB%A5%99%EC%84%A0&amp;action=edit&amp;redlink=1" title="경북내륙선 (없는 문서)">경북내륙선</a>이 검토 중에 있다.</p>

<h2><span class="mw-headline" id=".EC.9D.98.EB.A3.8C">의료</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%98%81%EB%8D%95%EA%B5%B0&amp;action=edit&amp;section=12" title="부분 편집: 의료">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>영덕군에는 병원이 있으나 종합병원은 <a href="/wiki/%EC%98%81%ED%95%B4%EB%A9%B4" title="영해면">영해면</a>에 한 곳 있을 뿐이고, 그마저도 분만실이 없어 출산은 <a href="/wiki/%EC%95%88%EB%8F%99%EC%8B%9C" title="안동시">안동시</a>나 <a href="/wiki/%ED%8F%AC%ED%95%AD%EC%8B%9C" title="포항시">포항시</a>에서 하는 실정이다.</p>
<h2><span class="mw-headline" id=".EC.A0.95.EC.B9.98">정치</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%EC%98%81%EB%8D%95%EA%B5%B0&amp;action=edit&amp;section=13" title="부분 편집: 정치">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>적은 인구로 인해 국회의원 선거구는 <a href="/wiki/%EB%B4%89%ED%99%94%EA%B5%B0" title="봉화군">봉화군</a>, <a href="/wiki/%EC%98%81%EC%96%91%EA%B5%B0" title="영양군">영양군</a>, <a href="/wiki/%EC%9A%B8%EC%A7%84%EA%B5%B0" title="울진군">울진군</a>과 공유하는 상황이고, 과거에는 <a href="/wiki/%EC%B2%AD%EC%86%A1%EA%B5%B0" title="청송군">청송군</a>과 선거구를 공유한 적이 있었다.</p>
